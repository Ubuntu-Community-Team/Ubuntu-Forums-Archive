### Post by Richardcavell on 2009-05-29
You can use either.  I recommend the 64-bit one.  There's slightly more support for the 32-bit flavour, but for all the software that really counts, you can get 64-bit support.  There are theoretical performance problems because the 64-bit version uses more memory for integers and pointers and so on, but a new iMac is not going to run into trouble.

Richard

### Post by Richardcavell on 2009-05-30
Hi,

'i386' refers to the 32-bit version for Intel x86 processors.
'amd64' refers to the 64-bit version for Intel x86 processors.

If you want the 64-bit variety, download the amd64 version.  I'm using the 64-bit version on my MacBook right now, and it works fine.

Richard
