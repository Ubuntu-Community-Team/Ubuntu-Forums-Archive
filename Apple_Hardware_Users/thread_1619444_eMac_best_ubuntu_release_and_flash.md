### Post by Dukenukemx on 2010-11-13
I tried Debian out a few months ago, and I didn't like it.  Everything did work, and it didn't require me to hunt down obscure methods for getting Wifi, 3D, and power saving features working.  Unlike Ubuntu 10.04, which over a month of using it I ended up going straight back to Mac Leopard.  

There were other problems with Debian.  There was no FireFox on the repository, just a FireFox alternative.  Apparently, the Debian guys are extremely anal about what is "Open Source".  Using an obscure method, I was able to install an outdated version of FireFox.  Though, outdated meant you couldn't use some of FireFox's plugins.  

The second problem was performance.  It was so horribly slow that web browsing was nearly impossible.  Scrolling around in FireFox or other web browsers was an extreme pain.  The repository didn't have a lot of the apps I liked, or the apps were extremely outdated.  In the end, I went back to Mac OS X.

Though, I just recently installed Ubuntu 10.10, and was shocked to see how well it works.  Nearly everything worked after the installation, and flash video works.  Most flash video works, and even then somewhat slowly.  I had to fix 3D and battery meter, but it works.  

There are problems, like certain Compiz effects can tint the screen blue, and clicking on chat accounts does nothing, but these are problems I can live with.  Linux Mint is just a fancy UI on Debian, and unless it did a speed increase and updated the repository then I'm not interested.
