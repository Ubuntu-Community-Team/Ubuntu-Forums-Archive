### Post by shawnhcorey on 2010-12-03
> **riccardo said:**
> I got around this problem by choosing the 'alternate' image.  Is this a recognised problem??  Is there another work around??

Yes, it's a known problem.  The thing is that they want to have the same software on all versions of Ubuntu.  Because of the differences between i386 and PPCs, the PPC's image is too large for a standard CD.  There are two solutions:

1. But oversized CDs; they come in 800MB and 900MB sizes.

2. Use the Alternate CD image.  The only differences them between is the installer.  The Alternate CD uses a text-based installer, the Desktop CD uses the GUI one.

I've stopped using CDs; I use a LiveUSB for installation.  See:
