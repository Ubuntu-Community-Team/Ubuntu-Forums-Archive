```
-p "" is definitely the recommended way so that grub.efi auto-detects the prefix. Also efi does not use 

```

function load_video {
  insmod vbe
  insmod vga
  insmod video_bochs
  insmod video_cirrus
}

```it should be 

```

function load_video {
  insmod efi_gop
  insmod efi_uga
}

```Go to [https://wiki.archlinux.org/index.php/GRUB2#Bootloader_Installation_for_UEFI_systems](https://wiki.archlinux.org/index.php/GRUB2#Bootloader_Installation_for_UEFI_systems) for more info. I am not an Ubuntu user. I use Archlinux but the instructions are similar.
