### Post by slickymaster on 2015-05-15
No there isn't.

What you can do is to install it using PlayonLinux. Open the terminal and run```
sudo apt-get install playonlinux
```Once you have installed PlayonLinx, open it and type iTunes in the search-bar, select the version you want to install and follow the prompts.
