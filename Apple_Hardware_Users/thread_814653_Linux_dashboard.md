### Post by frog_pilot on 2008-06-01
There is a compiz-fusion-plugin: "widgetlayer". You can configure it with ccsm "CompizConfigSettingsManager" (maybe you have to install it first). You will find this under "System" > "Settings" > "ccsm".

You can setup screenlets (maybe you have to install it first) to work as widgets.
