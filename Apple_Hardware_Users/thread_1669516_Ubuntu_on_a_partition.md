### Post by Bucky Ball on 2011-01-17
With apple I think 'Boot Camp' is what you're looking for if intending to dual (or triple!) boot. ;)

[http://www.apple.com/support/bootcamp/](http://www.apple.com/support/bootcamp/)

[http://mactel-linux.sourceforge.net/wiki/Dual_Booting](http://mactel-linux.sourceforge.net/wiki/Dual_Booting)

Your other option is to download the Desktop Edition (10.04 LTS safest for the moment) and try Ubuntu off the disk. This is a bit slower than a full install but won't alter your system any and you'll see what it has to offer. You will probably need an ethernet cable as installing wireless drivers when you using the LiveCD is not a goer, but you *can* download the Gimp (fantastic app!) and Inkscape

### Post by srs5694 on 2011-01-18
Bucky Ball seems to have missed the fact that you're running a G4 (PowerPC) CPU; Boot Camp is for Intel only. There's [a sticky thread in this forum](http://ubuntuforums.org/showthread.php?t=427714) devoted to PowerPC basics, so you might want to read that.

FWIW, I've had various Linux distributions installed on a ca. 1999 iMac for at least a decade -- Yellow Dog, SuSE, Debian, and perhaps others. I've never tried Ubuntu, though. The machine's old enough that I fear to try a modern heavy-GUI distribution, so I'll probably stick with Debian until it dies. (I seldom use the computer for anything these days; it's just too old.) Several different booting methods exist -- or did exist; some are based on Mac OS Classic and so aren't relevant for a machine like yours, although others are still relevant. In any event, it's certainly possible to install Linux on a PowerPC Mac. I can't promise how well it'll work on your specific computer, but in general it works fine.
