### Post by jackiemcghee on 2008-09-01
Did you install pommed? 

It doesn't seem to be mentioned on the Macbook page you linked to, but it's mentioned in the Macbook Pro sheet at [https://help.ubuntu.com/community/MacBookPro#Function%20Keys](https://help.ubuntu.com/community/MacBookPro#Function%20Keys)
```
sudo apt-get install pommed
```
will get you going, and I've never needed to do anything beyond that.

Hope this helps
