### Post by grahammechanical on 2015-12-01
It is your choice. If you choose the Install Ubuntu option we will get 5 options or choices.

1) Install Ubuntu alongside. That is alongside other operating systems.
2) Erase disk & Install Ubuntu.
3) Encrypt the new Ubuntu installation for security
4) Use LVM with the new Ubuntu installation
5) Something else. Create and resize partitions yourself or choose multiple partitions for Ubuntu.

If you choose option 2 then it will do what it says it will do.; It will erase the disk and you will loose not just Windows 10 but also OSX 10.10.1. Do not be in any doubt about that.

If you do not see option 1 - Install alongside it will most likely mean that there is not enough space on the disk to install Ubuntu and there could be several reasons for that. So, take a screenshot of the partition layout from Windows and post it in your thread.

You may need to use option 5 - Something Else. And we will need to give you further advice if that is the situation.

In this link #4 image is out of date because it does not show options 3 & 4. But they will be there. 

[http://www.ubuntu.com/download/desktop/install-ubuntu-desktop](http://www.ubuntu.com/download/desktop/install-ubuntu-desktop)

Welcome. You can't install Ubuntu on a Mac with a regular USB as far as I know. You need a DVD or to [follow this]("https://help.ubuntu.com/community/How%20to%20install%20Ubuntu%20on%20MacBook%20using%20USB%20Stick"). Been trying to work through this with another user on another thread. ;)

Using a disk is recommended. Apart from that, go for it! You'll need to go on a learning curve to keep Windows, but you can have more than two OSs installed if you like. OSx, Ubuntu, Windows triple boot! :)
