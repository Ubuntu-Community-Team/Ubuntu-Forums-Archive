### Post by nathanid95 on 2009-12-05
Let me get this straight:
1: You burned a DMG of Ubuntu from your Mac to a CD/DVD.
2: You tried to boot from that disc on other PC's?

If this is right, there are several possible fixes.

a: Make sure you are using the proper startup method to boot from a disk, this is usually a BIOS option you access from the startup splash/loading screen.

b: Make sure those computers have compatible processors. Ubuntu will run on "PC, 64-Bit PC and Intel-based Mac architectures."

c: If all else fails, burn another disk. I personally recommend using the built-in Disk Utility
