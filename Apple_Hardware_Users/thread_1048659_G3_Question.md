### Post by mkvnmtr on 2009-01-24
What probably happened is you didn't have enough ram. My G4 came with 256Mb and that is not enough to run the live disk. This week I installed with the alt installer on a unit that had 192MB and got an install but could not really use it until the new ram arrived. You will use the bootloader yaboot not grub on ppc. If you install and don't like it it is no problem to put the mac disk in and reinstall mac as the only os. In fact installing and reinstalling will get to be a normal thing if you are like me and mess up your system once in a while. It is not a big deal and doesn't take long.
If you don't have 512 MG of ram you probably will not be happy with you installation.
