I installed Emerald, installed a few themes, but now i'm stuck. The overwhelming advice i've found via google has been to run it by pressing alt+f2 and type "emerald --replace"; however, being on a macbook, f2 also controls the brightness of my screen, so if i press alt+f2, it just makes the screen brighter.

As of now, if I want to use emerald themes, I have to type it into a terminal and keep the window open; otherwise, if I close the terminal session, window borders disappear completely.

### Post by _alex_ on 2008-06-14
You can press fn+alt+f2, which will bring up the "Run Application" dialog. However, for emerald to stick, you proably want to add it to the Sessions list in System > Preferences > Sessions.
