### Post by tiresia on 2009-03-07
Can you please tell more about your hardware? It's a PPC or an Intel? 
You can download the Ubuntu LiveCD and start from it pressing C
Open a Terminal and type```
gksudo nautilus
```
You can then back up what you need.

Alternatively, if you have a second Mac, you can connect the two computer via FW and start the second (not working) Mac pressing T (target modus) at start up.
