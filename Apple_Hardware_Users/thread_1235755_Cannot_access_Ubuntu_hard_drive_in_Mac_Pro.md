### Post by Jenkins1 on 2009-08-09
going by this [http://ubuntuforums.org/showthread.php?t=863012](http://ubuntuforums.org/showthread.php?t=863012) it appears that it is not very easy for a mac to read ext3 format. It would be easiest to copy any important info to a flash drive etc and then do a clean install. You could copy ubuntu onto an ntfs partition and then on to the mac. Although i would not be 100% if it would work doing the second method.
