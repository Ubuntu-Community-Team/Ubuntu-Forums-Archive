### Post by Lars Noodén on 2014-04-18
Lubuntu 14.04 supports PPC.  But that uses LXDE and I haven't tried to see if you could add Unity and the other Ubuntu-desktop components.  You can find Lubuntu here:

[http://cdimage.ubuntu.com/lubuntu/releases/14.04/release/](http://cdimage.ubuntu.com/lubuntu/releases/14.04/release/)

Either the Desktop or the Alternate image should work.  The Desktop is over-sized and won't fit on a CD, in case that matters.  There are also some install notes.  

[https://wiki.ubuntu.com/TrustyTahr/ReleaseNotes/Lubuntu#PPC](https://wiki.ubuntu.com/TrustyTahr/ReleaseNotes/Lubuntu#PPC)
[https://help.ubuntu.com/community/Lubuntu/Documentation/FAQ/PPC](https://help.ubuntu.com/community/Lubuntu/Documentation/FAQ/PPC)

You might need to add boot option to get started with your graphics card depending on the model.
