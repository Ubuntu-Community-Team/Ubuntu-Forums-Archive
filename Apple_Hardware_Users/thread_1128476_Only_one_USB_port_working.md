### Post by pmaciver on 2009-04-18
Actually I found a (not so technical) solution for this from other people that have had the same problem.

What you do is shut down the machine, remove the battery. Put it back in and start it up again, and hey presto, both ports work.

No idea why. But I have only had to do it once and it works all the time now even after shutdowns and restarts. 

But I don't think that this is specific to ubuntu, as the person that reported the solution was working with os x.

Anyway I thought that I would post the solution in case anyone has the same problem.
