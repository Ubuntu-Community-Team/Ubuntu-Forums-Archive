### Post by morphos on 2009-02-01
I've used an external HDD attached via Firewire to a Mac Mini successfully.  You use your OS to set EFI to the destination of your choice.  

You modify that choice by holding down the Option key during boot to get a manual choice in the loading of media.  

I would think that USB would be the same way.

### Post by smarmo on 2009-02-01
All you should have to do is hook up the external drive, boot from your installation disc (hold down "C"), and somewhere in the installation process it will ask on what volume you want to install OS X (you might have to click an "options" button or something). After you've installed, your computer will still boot into ubuntu. And if you want to boot into OS X, hook up the external drive, and hold down the "option" key during boot.
