### Post by umbrAtrorum on 2008-07-04
what exactly do you mean by features? the touchpad specialties? the special functions on the f1-f12 keys? look around on the forum - all those questions are answered here already. for a quick introduction on how to install ubuntu (and some clues what ubuntu does and does not support) see [https://help.ubuntu.com/community/MacBook](https://help.ubuntu.com/community/MacBook) . i hope that helps.

jaspeR

ps: if you are unsure whether ubuntu does what you want, you might consider a dual boot solution, having ubuntu installed alongside mac os x. i had some issues with that installation yesterday and found very competent help here. look up [this]("http://ubuntuforums.org/showthread.php?t=848754") thread for that. i have a macbook pro, but the basic installation plan will be pretty similar. i tried dual boot without rEFIt and had no fun at all. if you install rEFIt which takes 2 minutes to do, it is easy and safe to install ubuntu.
