### Post by mardose on 2011-05-31
In regards to my post above let me say that for the record I'm not a huge fan of Firefox on older systems. My personal recommendation would be to install Opera (slightly longer load times, but faster browsing in general) which should be simple because I believe Diamond-SLE has the original Ubuntu repositories enabled. If so, type:

sudo apt-get update 
*enter password, the default is "admin" I believe*
sudo apt-get install opera
