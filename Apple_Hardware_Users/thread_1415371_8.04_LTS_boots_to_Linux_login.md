### Post by linuxopjemac on 2010-02-27
so, you have an eMac PPC G4 1.25 with an ATI Radeon 9200 graphics card. It should be able to get some good resolutions and 3D-acceleration with this machine, according to others. Make a backup of your **working** Xorg.conf file, in case you want to get back to that later. 

```
sudo mv /etc/X11/Xorg.conf /etc/X11/Xorg.conf.bak

```
Then we make a new one and paste the required code into it:
```

sudo touch /etc/X11/Xorg.conf
```

and paste from [here]("http://mac.linux.be/content/g4-emac-radeon-compiz").

then CTRL-X and "y" to save.

This is the best Xorg.conf file for this machine, if it won't work, you can try other emac Radeon configuration files, which are less demanding [here]("http://mac.linux.be/content/cross-distribution-issues").
