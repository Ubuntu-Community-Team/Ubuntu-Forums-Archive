### Post by ronocdh on 2009-03-23
Are you running Ubuntu natively or in virtualization? An easy way to answer this question is, "Do you I have to restart my computer form OS X to get into Linux?" If so, then you're installed natively.

Basically, WINE is a great way to get Windows applications to run under Linux. You can still use this even on a Mac. If, however, you mean that you only have the Adobe software installer for OS X, then it'll be a bit tougher. I recommend installing it in OS X and running Ubuntu in virtualization from there, if it's really vital software for you to have.

Or perhaps your university offers protected downloads of it to students? That way you could choose which platform you want it for and download it for Windows XP and run in WINE.

### Post by ad_267 on 2009-03-25
> **Rownrie said:**
> How do I run Linux and OSX in virtualization?

You can install VirtualBox on OSX (or use any other virtualisation software like VMWare). Then you can create a new virtual machine and install Ubuntu onto it. There are plenty of guides on how to do this and it's really easy when you give it a go.

This would mean that you could run all your applications you need in OSX and then you can boot Ubuntu in a virtual machine in OSX without having to restart your computer. You can then easily switch between applications running in OSX and applications running on Ubuntu.

You can get an idea of what this does from the screenshots on the VirtualBox website: [http://www.virtualbox.org/wiki/Screenshots](http://www.virtualbox.org/wiki/Screenshots)
