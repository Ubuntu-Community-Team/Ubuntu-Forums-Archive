### Post by dngfng on 2010-08-30
Honestly - I wouldn't use BootCamp for this - rEFIt does a much better job of it.
At startup you will be able to chose between OS X and Ubuntu.

I do not see any real downsides in running Ubuntu nativly. Also I do not see why there
should be any problems with Compiz.

Here is a really well written tutorial on how to do this (it is for triple boot but you can just ignore the steps for Windows).
