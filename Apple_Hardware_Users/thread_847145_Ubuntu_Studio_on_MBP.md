### Post by russo.mic on 2008-07-02
So, I just wanted to throw this out there.

I've finally gotten around to installing Ubuntu Studio, and that has to have been the best thing I've ever done.  

I run a recording studio here in Baton Rouge, LA, and part of my business is mobile recording.  The problem has always been, however, getting a mobile setup that is reliable.  Presonus Firepods are an okay option, but trying to daisy chain more than one of them in Windows or OS X is always pretty shaky.  I've had it fail more times than work.  

As of 2 weeks ago, 3 firepod's running with freebob and Ardour as the host is our new mobile setup.  Presonus has said for a while now that it's possible to link up to 3 FP10s to have them run in tandem, but it's never worked well ATL ALL!  I can't seem to MAKE it lose sync in Linux.  I think it to be amazing that Presonus can't seem to make drivers to run there hardware as reliably as freebob.  I've compiled Ardour to run VST plugins, and find the whole JACK setup to be wonderful.  My only complaint, which isn't that big of a deal because of how I use ardour, is it would be nice to have AFF or OMF support to easily export track lists to Nuendo, which is what I use natively at the studio.  

I guess I really just wanted to say Ubuntu Studio on a MacBook Pro runs great, and thanks to the ubuntu studio guys and the community for making this kind of rig not only possible, but reliable and a good option for my business.  I've beel blown away.  KEEP UP THE GREAT WORK!!!

Russo
