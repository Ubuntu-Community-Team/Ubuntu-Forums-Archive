### Post by jisaac on 2010-12-05
Hi guys!

Just wanted to share my very first experience with Natty Narwhal using my iMac.

I have VERY good news related with our ATI cards!
- NO MORE BANDING issues using full resolution 2560x1440 and 24bit color depth.
- NO MORE TEARING with or without compiz activated!!! The VSync option from proprietary drivers finally works!!! :) Linux + ATI users, welcome to the modern age!!!
- The only annoying defect still present is that you have to reconfigure it for each session you want VSync to work :(

To install it I had to use the Alternate Install CD:
- Then boot using the "recovery mode" and adding "radeon.modeset=0 nomodeset" (because of the blank screen issue)...
- Then I installed the ATI proprietary drivers proposed by Ubuntu.

-> Wifi is working well (but I had to wired my iMac during the install).
-> Internal speakers: Audio sounds strange... not working well. It seems that only the trebles are working.
-> Headphones: Audio sounds well.
-> Internal microphone does not work.
-> The keyboard and the mouse stop working after the ATI Drivers installation!?!? (I had the same issue with 10.10).
-> Very low performance when playing videos with Totem and VSync turned on.
-> Unity does not work at all.
-> At this stage (Alpha 1) Classic Desktop is full of bugs, lot of crashes...

Let you know more as soon as possible...
