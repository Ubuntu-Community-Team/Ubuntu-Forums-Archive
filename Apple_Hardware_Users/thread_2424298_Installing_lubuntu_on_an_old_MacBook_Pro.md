### Post by guiverc on 2019-08-06
Sorry I'm heading to bed, but this maybe worth a read - [https://ubuntuforums.org/showthread.php?t=2326770](https://ubuntuforums.org/showthread.php?t=2326770)

Also telling us the release of Lubuntu may help, 19.04 uses LXQt (manual can be found at [https://manual.lubuntu.me]("https://manual.lubuntu.me/")), however Lubuntu 18.04 LTS uses the older LXDE and thus is different to the modern Lubuntu & the new manual won't help you.

I'd suggest using the 'check disk for defects' option first (I think of it as good insurance), then you can always select "Start Lubuntu" which lets you access the web, and use the Lubuntu on the install-media first before clicking "Install Lubuntu"  ([https://manual.lubuntu.me/1/1.3/installation.html](https://manual.lubuntu.me/1/1.3/installation.html)) *FYI: if using Lubuntu 18.04 it'll look different to the manual links I provide, but the functionality is the same*
