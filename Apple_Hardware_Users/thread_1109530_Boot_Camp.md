use the bootcamp assistant to shrink your OS X partition and add a FAT32 partition. After that, boot into the Ubuntu LiveCD, and start System>Adminstration>PartitionEditor and delete the FAT32 partition at the end of the disk. and apply. Then start the isntaller and choose to install to the largest free space.

There is more info about installing here:
[https://help.ubuntu.com/community/MactelSupportTeam/AppleIntelInstallation](https://help.ubuntu.com/community/MactelSupportTeam/AppleIntelInstallation)
