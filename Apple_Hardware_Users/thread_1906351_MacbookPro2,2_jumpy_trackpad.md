Hi, I just installed Ubuntu 11.10 on a MacbookPro2,2 (which was a challenge itself) and now almost everything seems to be working well.

The biggest annoyance right now is the trackpad seems to be very inaccurate and jumpy. Specifically, it seems to alternate between moving slow and quickly at very short intervals while I am mousing around. The effect is almost like the pointer is snapping to an invisible grid, which makes accurately hitting buttons very tedious.

The trackpad is smooth as silk in Mac OS.

Here's an image of what happens when I attempt to draw straight lines in gimp at various angles. All I'm doing is sliding my finger across the pad smoothly. The result is a staircase looking line if I draw diagonally which I think best illustrates the problem:
[ATTACH]210483[/ATTACH]

As you can see, I'm able to draw nearly perfect horizontal and vertical lines because the pointer is snapped in place, when I draw diagonal lines, all bets are off.

If I had to guess, it might have something to do with the fact that the trackpad on this model is in a wide aspect ratio, and so the driver is stretching the sensor area somehow which is introducing the jaggedness.

Anyone have any ideas on how to fix this?
