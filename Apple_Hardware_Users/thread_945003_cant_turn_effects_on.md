Right - it shows you how to install Ubuntu, but whether Compiz will work is a whole 'nother story. It has a lot to do with what access the Linux kernel has to your hardware. When you're in a VM, the kernel has to go through VMware and ultimately OS X's kernel, and somewhere in that translation the ability to run fancy 3D graphics is lost.
