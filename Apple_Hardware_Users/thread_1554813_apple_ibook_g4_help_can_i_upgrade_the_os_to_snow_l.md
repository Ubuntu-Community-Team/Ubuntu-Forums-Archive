### Post by liquidfunk on 2010-08-19
You have a number of options;

1) You buy Mac OS X Tiger from eBay.

2) You buy either a 512Mb RAM stick for the iBook G4 and install Mac OS X Leopard onto your iBook.

3) There are versions of Linux for the PowerPC architecture. (Debian, Arch, Yellow Dog).

To install Mac OS, insert the disk into the drive, turn the machine on, and hold down the C button. It will then boot off the drive.

I'd suggest (from the top menu) opening Disk Utility and formatting the drive you've got before you do it. 

You don't want to do an upgrade, a fresh install would be better.
