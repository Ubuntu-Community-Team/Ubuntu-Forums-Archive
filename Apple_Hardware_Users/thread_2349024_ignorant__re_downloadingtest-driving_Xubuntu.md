### Post by g33zr on 2017-01-10
Your MacBook can handle 64-bit, so use that. Also, I'd recommend downloading the ISO and burning it to a DVD, which you can boot live to test the distro.

A better way is to install VirtualBox, a cross-platform program, and run the downloaded ISO from there. You can even install Xubuntu or whatever other distro you might want to test in VB without dual-booting from your hard drive.
