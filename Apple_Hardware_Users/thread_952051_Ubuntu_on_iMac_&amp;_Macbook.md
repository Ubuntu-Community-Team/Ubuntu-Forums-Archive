Here is my config for my iMac Aluminium :

/dev/sda GUID Partition Table 500 Go
/dev/sda1 EFI FAT32 200 Mo (do not delet it !!!)
/dev/sda2 Mac OSX Leopard HFS+ 111 Go
/dev/sda3 ArchLinux ext3 30 Go
/dev/sda4 Ubuntu ext3 30 Go
/dev/sda5 Shared datas HFS+ 294 Go

So I have triple boot, I don't use a swap partition (rather swapfiles)
and all my datas can be accessed from the 3 OS :)

Hope that help !
