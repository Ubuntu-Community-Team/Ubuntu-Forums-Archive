### Post by retchid on 2011-11-04
if you want a system that you can just run off USB without installing on your hard drive
download unetbootin
go to linux puppy site
download puppy 5.28 ISO
install  puppy iso to usb using net bootin
boot from usb (bios)

this will just use the usb effectively a live usb  you can install software to and use as if it was an operating system

also mac puppy is good version of the above 

a 2gb sandisk usb (£4) is more than adequate - you can of course access your HD

30 seconds to boot allow you to try something different without installation or rubbish live usb/cd which dont really allow you to install software to them

Yes I am using it now - also runs spotify via WINE no problems (UNITY couldn't)
and look MAH no hard-drive

I will never ever go  back to booting from a HD on a laptop
