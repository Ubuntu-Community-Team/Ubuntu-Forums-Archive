### Post by slick666 on 2010-12-16
I don't know if you're going to be able to do that. OSX install is 17+GB and the installation script requires a minimum HD size of around 80GB. Additionally if you don't have a Mac computer you can't run it. Unlike Windows the EUSLA (End Use Software License Agreement) requires that the OS be executed on Apple branded hardware. Meaning that you can't install it or run it on hardware not sold by Apple. Apple computers also have hardware used to enforce this.

Sorry to be such a downer on your experiment but I don't think it can be done and it might be outside the scope of this forum.

If you're looking to try a couple things n the OS the easiest way to to go into the retail store and just sit down at a machine. As long as it's not too destructive you can pretty much try whatever.
