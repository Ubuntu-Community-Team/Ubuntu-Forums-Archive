### Post by Tylerjd on 2010-09-11
You should always keep MacOS installed on a *small* partition, as there are reports of Macs not booting up when Mac OS is not installed. Install Ubuntu per the instructions. You can use the bootcamp pane in system preferences in System Preferences in Mac OS to choose to boot up from "Windows" and it should automatically boot from linux.

If that doesn't work check you hard disk to make sure that it isn't dead. The flashing folder with the question mark is a sign of a dead HDD when Mac OS is installed.
