Hi,
Often people have asked me what the ubuntu or linux alternative is to rapidweaver. The reply they have received from elsewhere is usually a list of HTML editors or dev environments like Quanta Plus, Kompozer, Bluefish, etc.

The answer is in fact that Rapidweaver stands apart from these things because it is actually more of a Content Management System than a Web Development environment although it does have some powerful dev features.

In this case the more similar software would be: Wordpress, Joomla, Drupal etc. The advantage these CMSs have over Rapidweaver is that they can be accessed and used anywhere your website can be. In fact they are websites with built in editors and user permissions etc. They go far beyond what rapidweaver is capable of. The disadvantage is that Rapidweaver is much more - rapid. It has access to the powerful image manipulation etc of a mac and therefore is compelling as a CMS if you don't need to have many people adding content to the site.

I think there probably is a market for a linux program like Rapidweaver but I would suggest that as CMS like Joomla improve it will become unnecessary before it is finished.

Hope that is helpful to someone.
