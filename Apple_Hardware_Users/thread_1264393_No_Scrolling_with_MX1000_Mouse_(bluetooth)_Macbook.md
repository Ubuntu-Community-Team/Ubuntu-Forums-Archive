### Post by edmondt on 2009-09-13
I think I have found the problem from this bug report:

[https://bugs.launchpad.net/ubuntu/+source/linux/+bug/332443](https://bugs.launchpad.net/ubuntu/+source/linux/+bug/332443)


My bluetooth wasn't actually "working", even tho the bluetooth mouse connected through memory from OSX configuration I suspect.

Running this command:

```

sudo hciconfig hci0 reset
sudo /etc/init.d/bluetooth restart

```

Made bluetooth work again...

I added this command to /etc/rc.local

Restarted, then paired Mouse and everything works now... 

installed xbindkeys and added the following in my ~/.xbindkeyssrc:

```

"xte 'keydown Alt_L' 'key Left' 'keyup Alt_L'"
   b:8
"xte 'keydown Alt_L' 'key Right' 'keyup Alt_L'"
   b:9
"xte 'keydown Control_L' 'key c' 'keyup Control_L'"
   b:6
"xte 'keydown Control_L' 'key v' 'keyup Control_L'"
   b:7
"xte 'keydown Control_L' 'keydown Alt_L' 'key d' 'keyup Control_L' 'keyup Alt_L'"
   b:10"

```

Now back and Forward works, Side middle button shows desktop, Left middle scroll copy, Right middle scroll paste... 

Nice to know it can work just a little bit better than Windows7 and OSX eh? :)
