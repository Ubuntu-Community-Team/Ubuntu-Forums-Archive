### Post by eric-yorba on 2013-04-01
> **IanViator said:**
> I have just installed ubuntu on my macbook air, and everything works well, but is there any means by which an aperture photo library can be converted and installed on Shotwell? I would prefer keeping my metadata, projects, and so forth intact.

I'm by no means an Aperture expert, but I believe there's a setting to write metadata into the photo files.  If you have this set, Shotwell will pick up tags, titles, comments, etc.

Shotwell has no notion of projects, however there are events which can be used to group photos.
