### Post by rsgooch on 2010-08-24
Mac keyboard by changing the layout to USA Alternative international (former us_intl) The Apple aluminum with 10 key seems to work fine.  

Steps:
System=>Preferences=>Keyboard

Layouts tab

Keyboard model: Apple Aluminum Keyboard (ISO)

Add... button

country: United States

Variant:USA Alternative Alternative international (former us_intl)

Make sure that it is the first in the list using the move up and move down buttons

This seemed to get most of the buttons on my keyboard working.

Will work on the media keys next.
