### Post by kelvin.illa on 2010-11-01
This is what I did for my MacBook 2,1 on installation up until 10.04 back then. I don't really think the ideal procedure has changed:

If you have kept your OS X partition, skip this step:
Download the "[AppleUSBVideoSupport.]("http://www.mediafire.com/?81xtkqyttjt")" It's the number 2 hit on Google using the search term "AppleUSBVideoSupport", but I hear it's against Apple's EULA to distribute the driver. Put the file wherever you find convenient and most accessible.

Go to "Ubuntu Software Center." Use "isight" as a search term. Install "isight-firmware-tools". After the installation, it will prompt you for the location of your "AppleUSBVideoSupport." If you kept your OS X partition, you don't have to change anything (I suppose); otherwise, change the file location to the location of the file you just downloaded. MAKE SURE THE FILE LOCATION IS OF THE CORRECT SPELLING. Restart your computer.

additional info: [https://help.ubuntu.com/community/MactelSupportTeam/AppleiSight?action=show&redirect=AppleiSight](https://help.ubuntu.com/community/MactelSupportTeam/AppleiSight?action=show&redirect=AppleiSight)
