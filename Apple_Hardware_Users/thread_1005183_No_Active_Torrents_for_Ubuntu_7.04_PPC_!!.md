### Post by stream303 on 2008-12-08
7.04 was my favorite too, although I have been able to get the later versions running once the weirdo installer bugs were found out. :)

Since 7.04 is EOL'ed as far as updates go, if you like that version, I'd really recommend Debian Etch 4.0r5 "stable".  It still gets updates, has the older less-complex xorg.conf and other tricks for video configuring, plus you can run a newer kernel 2.6.24, which was a neat standout-feature for this "Etch-n-half" release.  Obviously it doesn't have the absolute latest software, but it is still more up to date than Feisty 7.04, and installation is a lot easier with no repo changes to deal with either.
