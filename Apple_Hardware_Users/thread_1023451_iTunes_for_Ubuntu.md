### Post by Master Chief on 2008-12-30
> **shizumasa14 said:**
> I am interested in switching from windows to Ubuntu.  I really want to keep iTunes though, for my iPod.  Is there a Ubuntu compatible version of iTunes?

To answer your question, like "hanzomon4" did earlier, no there is no official version of iTunes for Ubuntu, neither for any other Linux distribution. The reason for this is simple; Apple copyright and software patents.

You can however install iTunes in VirtualBox (no coverflow support in iTunes) or better install/use VMWarePlayer which offers coverflow support in iTunes and appears to be faster (at least on my Ubuntu installations).

Note: It might be a pain to install VMWarePlayer for you as new Ubuntu user, and that is why I would suggest to start with VirtualBox first, which to me is more end-user friendly, and this way get used to virtual machines and all terminology.
