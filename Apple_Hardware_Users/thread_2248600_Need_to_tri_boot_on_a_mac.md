### Post by d4m1r on 2014-10-16
1) OS X is installed by default.

2) Install Windows 8.1 through OS X (Boot Camp)

3) Install reFit

4) Install Ubuntu last

Multiple guides available on the internet, and any that has instructions along the above lines will work. You can also search my previous posts (via my user profile) where I've posted the basic instructions on how I got Windows, OS X, and Ubuntu all running perfectly on my 2012 Macbook Pro. Personal suggestion, install Windows and Ubuntu in MBR (BIOS mode) instead of UEFI like some guides will tell no....No need and much easier to get working and change things.
