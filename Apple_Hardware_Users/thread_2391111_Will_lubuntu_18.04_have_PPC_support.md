[Lubuntu 17.04 (Zesty Zapus) PowerPC Daily Build ISOs to No Longer Be Produced]("https://news.softpedia.com/news/lubuntu-17-04-zesty-zapus-powerpc-daily-build-isos-to-no-longer-be-developed-512659.shtml") is an article that gives some information regarding the non-availability of PPC downloads.

I seem to remember that there were too few testers with PPC hardware and no-one wanted to release an ISO that hadn't been properly tested. It looks like your installation is supported until April 2019 then you will need to use different hardware if you want to continue running Lubuntu.
