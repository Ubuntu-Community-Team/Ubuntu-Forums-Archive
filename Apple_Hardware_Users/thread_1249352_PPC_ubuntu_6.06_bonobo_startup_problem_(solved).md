[LEFT]How to fix ubuntu nautilus bonobo startup problem.
1.Download and install xubuntu 6.06 alternate
2.install ubuntu desktop, from synaptics package manager
3.install gnome-panel from synaptics 
4.logout then select gnome in the sesions of xubuntu login manager, then login
5.go to synaptics then update gdm "GNOME display manager" 
6.when done reboot, when you start gnome without an internet connection the bonobo and nautlius starts error free.
