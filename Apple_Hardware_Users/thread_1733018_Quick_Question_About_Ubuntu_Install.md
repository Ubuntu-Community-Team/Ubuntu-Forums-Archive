### Post by Anon7-2521 on 2011-04-19
Says it doesn't work for maverick. Here's the best way to do it: BACK EVERYTHING UP!!!!!

Put in the 10.10 CD click try ubuntu without installing. If you like, it use the GUI installer. You may wish to partition the drive in disk utility first, so as to preserve your OS drive, or you can do it in gparted. Once you've done this, the installation /should/ detect that OS X is installed. You can select "install alongside OS X" and it'll do everything for you. Make sure ALL your data is backed up in the event something goes wrong.
