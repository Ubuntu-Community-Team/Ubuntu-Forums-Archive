### Post by itismike on 2010-11-14
poopmonster666, you're gonna have to put in a little more effort than this. Maybe this isn't something you wish to tackle just now. If you're interested in learning more, this should get you started:

Download a live CD and test it out:
[http://www.ubuntu.com/desktop/get-ubuntu/download](http://www.ubuntu.com/desktop/get-ubuntu/download) (follow the instructions to create a bootable CD)
With the CD in the drive, reboot your Mac and hold down the 'c' key to make it boot from the CD.

Determine how compatible your Mac is with current versions of Ubuntu:
[https://wiki.ubuntu.com/MactelSupportTeam/CommunityHelpPages](https://wiki.ubuntu.com/MactelSupportTeam/CommunityHelpPages)

The most valuable resource is a carefully-phrased search:
[http://lmgtfy.com/?q=how+to+install+ubuntu+10.04+on+a+macintosh](http://lmgtfy.com/?q=how+to+install+ubuntu+10.04+on+a+macintosh)

Good luck!
