### Post by RealEmotionX on 2010-11-05
OKAY. SOLUTION AHEAD! Its not totally complete, but basic functionality can be achieved on a 300mhz iBook with the following settings.

FIRST
Install this distro
It's not as polished as Xubuntu, but you wont find a distro that will function as well on an 11 year old machine.
*[http://cdimage.debian.org/debian-cd/current/powerpc/iso-cd/debian-506-powerpc-xfce+lxde-CD-1.iso](http://cdimage.debian.org/debian-cd/current/powerpc/iso-cd/debian-506-powerpc-xfce+lxde-CD-1.iso)*

Right away, open terminal and run after using SU
*apt-get install network-tools*

After this, run 
*iwconfig*
this will show you which of your interfaces has a wireless capability, in my case it is eth1

After you know this information, punch this in!
*iwconfig eth1 essid YOUR_ESSID*

This will not work on encrypted networks. I do not know if I can get this to work with WPA or WEP and frankly dont care to. I lock my router down to Mac addresses, so its unnecessary. If you know how to fix this without breaking everything, go right ahead. Anything I tried would Make ETH1 stop functioning.

To make sure you are connected run
[I]dhclient eth2[I]
This may or may not be necessary. I don't pretend to know. I run it every time to check to see if my router's ip address appears. It does. :D 


Thank everyone for their help. If I find a graphical way to do all this that doesnt break Eth1, I will post it. 

I am posting this from my iBook.
