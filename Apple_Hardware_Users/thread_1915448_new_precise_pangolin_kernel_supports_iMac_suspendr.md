### Post by entangled on 2012-01-26
For anyone who cares about ubuntu suspend/resume on an iMac 
- the latest kernel 3.2.0-10 #18 is the first to work properly on my dual boot iMac 11,2 running precise pangolin alpha 1. Previous kernels have allowed suspend, but could not resume back to the correct display.
So far, precise has been well-behaved - no known problems on my iMac.
