### Post by tiresia on 2009-01-31
> **mikeym said:**
> Hi,

I've managed to install Grub on my G4 PPC over the boot from CD option and a kernel that wont boot.

The Bootloader for PPC is *not* GRUB but Yaboot.
Download the right Installer-CD for your architecture here
[https://wiki.ubuntu.com/PowerPCDownloads](https://wiki.ubuntu.com/PowerPCDownloads)

If you want to install Intrepid (8.10), be aware that the installer has a bug. Here is the solution for it
[http://ubuntuforums.org/showthread.php?t=964904](http://ubuntuforums.org/showthread.php?t=964904)
