### Post by entangled on 2016-04-01
Yes, it is definitely possible to install linux on iMac hardware. Some builds of ubuntu were bootable for install, but the current one seems to be ignored by the iMac bootloader. I use refind to manage booting but iMac must recognise it.

I had to attach a home-made dongle to the external display port socket to make the display go to the iMac screen. I've found that Archboot, Fedora and Manjaro all boot and install reliably. Runtime is more or less OK, quicker than osx, but I am not able to sleep and resume. Sleep works but resume results in a useless blank screen.
