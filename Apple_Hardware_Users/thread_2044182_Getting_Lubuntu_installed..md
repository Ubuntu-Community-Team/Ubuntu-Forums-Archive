### Post by darkapec on 2012-08-18
download lubuntu (i assume from another computer as that one probably does not have a cd burner)

burn lubuntu ISO to cd

insert cd into G3

restart G3 Holding the "C" button as soon as you hear the "chime noise"

system should now boot from cd, go ahead and follow the GUI instructions (they are pretty straight forward, post back if you need further help from there)

Also note that all data etc will be lost on the G3 if you install lubuntu so backup any data needed before installation.
