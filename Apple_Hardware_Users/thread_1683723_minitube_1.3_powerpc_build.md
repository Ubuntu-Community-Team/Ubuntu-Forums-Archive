### Post by sflinux on 2011-10-11
I just installed debian squeeze on a powerpc.  Found this thread and saw there is a new version of minitube 1.5.  I downloaded the source code.
"http://flavio.tordini.org/files/minitube/minitube-1.5.tar.gz"
# aptitude install build-essential qt4-dev-tools libphonon-dev phonon 
cd minitube
#qmake
#make
$ ./build/target/minitube

Voila, very cool.
