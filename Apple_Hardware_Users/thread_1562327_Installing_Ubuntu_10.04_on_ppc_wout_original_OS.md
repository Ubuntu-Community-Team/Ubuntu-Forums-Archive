### Post by shawnhcorey on 2010-08-27
If you use a CD, use the Alternative CD, not the Desktop CD.  The Desktop CD does not fit onto a standard CD.  The only difference between them (aside from their size) is the Alternative CD uses a text-based installer and the Desktop CD uses a GUI one.
