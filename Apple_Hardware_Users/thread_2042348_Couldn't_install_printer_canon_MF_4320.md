Hi,
the most efficient way to install a printer on a mac is using the cups interface.
enter in safari or in a different browser and type 127.0.0.1:631 then go to manage printers - add printer - select the printer basing on the connection (usb or via IP address if device is connected over the network) then select a printing driver if canon didn't release the driver for this printer for ubuntu or for mac you can use  a generic one.

If printer is pcl 5-6 or postscript mac and ubuntu contains generic pcl and postscript drivers.
Once added you will see printer in System Preferences - Printers.
