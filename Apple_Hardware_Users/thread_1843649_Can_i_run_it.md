### Post by rsavage on 2011-09-14
For PPC: You have loads of ram for ubuntu (you say 1.5GB on the sticky thread?). Your problem is the processor speed. The only way to tell if xubuntu is ok is to install it and see how it feels. If you have an ok video card then you'll probably be alright. If it is sluggish then install lubuntu. Lubuntu is the lightest, but you don't get round window corners, shadows, and transparency. 8GB is more than adequate for both.  Instructions for installing the latest versions of both are on the last page of the sticky thread (xubuntu is slightly easier). There is also a 10.04 live cd of xubuntu (although it is more resource hungry than 11.04), but you'll have to amend the link for 9.04 to find it.
 
For Intel: I think you'll be able to install what you like since you have a fast processor and plenty of ram, but I don't own one of those machines so can't really advise.

### Post by yokohama1970 on 2011-09-15
As for the Intel MacBook. I would stick to Mac OS 10.6 or 10.7.

As for Windows 7 & Ubuntu 11.04 Intel, you might want to check out Oracle/Sun Micro Systems VirtualBox for running both OS, within Mac OS. VirtualBox is very cool & runs on Intel-Based Macs (Not PowerPC). Plus, no Disk-Partitions, since they run Virtually.

As for Ubuntu PowerPC. As mentioned before, 500mhz G4 will be extremely slow, but should work with Ubuntu 10.04.3 LTS PowerPC. I always recommend LTS Distros, due to length of support (3 Years).

The good thing is you have a G4 Tower, so you have Hardware part swap options, which may help?
