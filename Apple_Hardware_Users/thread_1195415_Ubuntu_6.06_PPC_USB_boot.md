### Post by 3rdalbum on 2009-06-24
PowerPC-based Macintoshes cannot boot from USB. If you have one of the Macs with Firewire, you might be able to put an ordinary CD or DVD drive into a 5.25 inch Firewire drive enclosure, and boot from it... but I'm not sure if holding down the C key will work.
