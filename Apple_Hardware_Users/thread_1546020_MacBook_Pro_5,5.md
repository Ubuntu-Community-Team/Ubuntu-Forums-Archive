### Post by aviedw on 2010-08-04
Truthfully what i would do first is download the live cd first. Run that in your system and make sure everything is working correctly. Check the graphics, check memory usage, make sure the mouse works, make sure the network card works. Make sure both cores are being utilized. Just do the run of the mil stuff that you normally do on a system. Then if you feel comfortable with the results go ahead and dual boot or clean install ubuntu. I hope  that helps.
