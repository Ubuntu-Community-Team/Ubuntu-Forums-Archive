### Post by Richard85 on 2011-02-01
In firefox, 
preferences > advanced > Connection Settings > Check Automatic proxy configuration URL
and reload.
This allows firefox to access the web but safari is still not working.

---

### Post by Richard85 on 2011-02-01
In Safari,

Safari > Preferences > Advanced > Proxies - Change Settings > Uncheck HTTP and HTTPS and check Automatic Proxy Configuration.
OK
APPLY

This fixed the Safari issue but her Dropbox cannot connect to the internet. But atleast we got Safari to load webpages.
