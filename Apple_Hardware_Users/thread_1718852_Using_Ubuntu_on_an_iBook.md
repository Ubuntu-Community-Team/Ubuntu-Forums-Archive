I don't think there is a PowerPC netbook version available for the iBook I'm afraid. There are a number of "how to" threads posted on the forum so it's a good idea to follow one of them if you are new.  The instructions for the G3 and G4 machines will be slightly different I would expect so make sure you follow the right thread!
