### Post by Nwwmac on 2009-06-27
I am running Ubuntu 9 on VM Fusion on an aluminum iMac. It runs well - better than Win 7 beta - but my text is a bit fuzzy because the resolution of the second screen I'm using is not supported. 

The new 2nd screen (non iMac screen) has a suggested resolution of 2048 x 1152. It's a widescreen format, so I chose what I thought would be a close match, 1920 x 1080. This fills the screen and does not look terrible, but the text in Firefox is fuzzy enough to be noticeable. 

Is there a better resolution choice? Or will I get support for 2048 x 1152 by switching to ATI drivers? I'm using the default driver now. 

Suggestions for a Ubuntu newbie? Thanks!

---

### Post by Nwwmac on 2009-06-29
Best solution I came up with was to tweak the resolution down bit. There are resolutions with 16:9 and 16:10 ratios and I believe my ratio was off just a fraction, so it was not obvious when not viewing text.

 Hopefully a new version in a few months time will give me the precise setting I need. Until then I guess I will just have to be patient.
