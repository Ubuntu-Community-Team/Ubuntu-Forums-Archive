### Post by Rubi1200 on 2011-11-26
I've never used a Mac so cannot really answer the question of using the GParted CD.

However, altering partitions that are mounted is _never _a good idea.

If the GParted CD won't run what about another LiveCD that you can boot the Mac with?

Then you can install GParted on the LiveCD if you have an Internet connection and work with the partitions in a much safer way.

Of course, make backups before any kind of major changes!
