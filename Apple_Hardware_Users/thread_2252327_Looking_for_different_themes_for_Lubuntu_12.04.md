### Post by sudodus on 2014-11-11
The engine under the hood has many components in common with Ubuntu Server and standard Ubuntu. Those components (packages) update without a problem until April 2017. But all the Lubuntu specific program packages lack support and are vulnerable to security issues. Of course you can continue to run it, if you consider the security risks small. But we cannot recommend it.
[URL="http://ubuntuforums.org/showthread.php?t=2229730"]
Forums Staff recommendations on EOL Ubuntu releases, in particular 10.04 desktop.[/URL]

Lars Noodén is one of the experts and able to help you get your Powerbook G4 running with a current version of Lubuntu. I don't know if there are many versions/models of Powerbook G4, so I ask you to describe your computer with as many details as possible, starting with

- model (name, number)
- CPU
- RAM (size)
- graphics chip/card
- wifi chip/card

and then continue in dialogue with Lars.
