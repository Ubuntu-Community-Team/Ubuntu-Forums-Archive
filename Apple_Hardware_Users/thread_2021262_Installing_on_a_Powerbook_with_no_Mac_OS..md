### Post by gwjvan on 2012-07-09
Yes, hold c at startup.

It shouldn't matter that you have 9.04 installed. If you don't want to save anything from that hard drive, just select the option to automatically erase and use the whole hard drive when you run the installer.

It might not be necessary to identify what model you have until later on, if something goes wrong. However, here's a link to help [http://support.apple.com/kb/HT3065](http://support.apple.com/kb/HT3065)
