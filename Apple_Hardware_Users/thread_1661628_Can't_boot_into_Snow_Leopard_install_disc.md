### Post by DoubleClicker on 2011-01-06
i've never experienced that,  but there are some general troubleshooting steps you could try:

Frst try holding the C key down when you boot. Make sure it's down from before the boot tone untill after the boot splash shows on the screen.

If that doesn't work,  hold command+shift+P+R until you hear 2 boot tones. After that hold down the C key.  

I'm fairly sure that that should sallow you to boot into the snow leopard CD.    However, if you plan to dual boot I highly suggest using rEFIt.
