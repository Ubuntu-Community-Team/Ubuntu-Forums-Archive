### Post by employeeno5 on 2008-08-07
Recently on 500MHz G3 (also with only 512MB RAM) I had the PPC version of Hardy working surprisingly fast. In fact it seemed to work much faster than when I had tried Fiesty on the the same machine. 

I was originally going to install Xubuntu, but I couldn't find a PPC build for Hardy yet. Ubuntu worked out just fine though.

However, to keep things nimble I eventually ended up doing a server installation.
Then I installed xorg and Fluxbox and Xfce. I usually just use Fluxbox on it's own, but Xfce runs very well on it, even with some basic composting effects enabled.
