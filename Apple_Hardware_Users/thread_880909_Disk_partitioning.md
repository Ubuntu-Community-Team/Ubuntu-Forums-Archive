### Post by mkvnmtr on 2008-08-05
The live disk gives you the option to partition during install. On my IBook G4 I believe I used the disk utility on the Mac install disk to create Then a partition. The mistake I made was using too small a first partition. I did't realize how much I would like Ubuntu. Proceed slowly. You can go back up to the point you format the new partitions. If you are not sure press back until you are out of the installer and go on Firefox and ask on this forum. I got to that point about 3 times before I had the courage to ull the trigger. Back up every thing on your OSX. If your like me you might mess up a few times. If you have the Mac disks it is no big deal.

### Post by limasierra on 2008-08-05
If you want to make the partition before installing kubuntu use gparted a disk partitioning program. To run it you must start the live cd and then use the following command```
sudo gparted
```
