### Post by coolaj86 on 2010-06-20
System -> Preferences -> Keyboard Shortcuts
gnome-screenshot --area
alt-ctrl-4

Unfortunately there's a bug and you can't map alt-shift-4 as you would like.

### Post by JohnAtilano on 2010-06-20
System -> Preferences -> Keyboard Shortcuts

I used CMD+3 for "Take Screenshot"

I used Alt+CMD+3 for "Take Screenshot of a Window"

CMD key will actually say "Mod4" in the shortcut.
