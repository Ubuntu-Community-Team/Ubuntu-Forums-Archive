### Post by MisterGaribaldi on 2012-02-05
Apple only writes three versions of iTunes:

1. Mac OS X / PPC
2. Mac OS X / Intel
3. Windows

So, if you don't want to install via Wine, your options are to install it in Windows (dual boot) or use a Mac.
