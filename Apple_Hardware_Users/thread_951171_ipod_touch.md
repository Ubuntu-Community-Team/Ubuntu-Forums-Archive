### Post by stream303 on 2008-10-18
Here's one possible way without having access to itunes permanently:

Download FileZilla on the Ubuntu box.  This will pair up with the Air-Sharing app you purchase on the ipod via wireless and you can ftp the files between the two that way.  I'm not running air-sharing myself, so double-check that it will support your file formats.

Somehow, you'll need to establish an iTunes account to purchase air-sharing - perhaps trusted friend, family member, etc on their windows or mac box that can install itunes temporarily.  There might be other similar apps that allow for wireless file sharing on the ipod.

Once you have the account, and have some sort of wireless connectivity, you can download and install air-sharing on your ipod, and use FileZilla on the Ubuntu box to transfer the files.  If suppose you could purchase an inexpensive wireless router if you don't have one already just to do the file sharing.

Anyway, just some ideas...
