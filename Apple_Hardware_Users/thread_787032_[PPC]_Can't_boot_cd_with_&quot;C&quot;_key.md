### Post by stream303 on 2008-05-08
Thanks to Egruber, there might be a solution for those who can't seem to boot a CD while holding down the "C" key.

The problem may be that some are taking it so literally, that it might be better to modify the advice to the following:

**Very quickly after powering on, hold down the "C" key until you hear or see the system booting from CD.**

Seems that on some systems, holding down the C key *before* powering on may not actually work.
