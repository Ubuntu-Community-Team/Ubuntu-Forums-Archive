### Post by stream303 on 2008-04-26
When I upgraded from an Gutsy to Hardy RC, I manually edited my /etc/apt/sources.list file from the details here:

[http://www.xubuntu.com/news/hardy/ports](http://www.xubuntu.com/news/hardy/ports)

I then did
```
sudo apt-get update

sudo apt-get upgrade

sudo apt-get dist-upgrade
```

and it seemed to go ok.  I did notice that afterwards, in my Sources gui, that these "ubuntu-ports" appeared in my third-party tabs, and I made sure to UNcheck the ones in the main Ubuntu source tabs, as they point to the original archives.

Maybe this will work...
