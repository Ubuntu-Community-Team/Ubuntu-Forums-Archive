### Post by DGortze380 on 2008-12-01
download the ubuntu iso
open disk utility
Command + 'O' (or file "Open Disk Image")
Select the ubuntu iso
Highlight the ubuntu iso on the left panel of disk utility
Click burn
Select the slowest burn rate
Wait until it pops out of the drive
