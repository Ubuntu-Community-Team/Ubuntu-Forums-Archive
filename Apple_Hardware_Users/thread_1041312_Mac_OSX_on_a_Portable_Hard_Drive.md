### Post by cyberdork33 on 2009-01-16
> **coiax said:**
> The tutorial says keeping Mac OSX on a portable hard drive, how is this accomplished?
boot from the leopard installation dvd, tell it to install to the external disk, finish install process, done. hold alt/option on startup to select the disk to boot from.

> **coiax said:**
> Secondly, if I at a later date decide to go back to OSX (unlikely as it is), can I reverse the modifications of the bootloader and such?
yep, again boot the Leopard Install DVD, in the menus, start the disk utility application. Use it to completely repartition the drive, install to your desired partition.
