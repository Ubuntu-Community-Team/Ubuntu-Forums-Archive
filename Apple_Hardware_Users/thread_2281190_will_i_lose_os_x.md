### Post by coffeecat on 2015-06-05
I suggest you wait until you have hear from someone with recent MacOS experience before you think of doing a complete erase disk and install. For that reason I have moved your thread to ***Apple Hardware Users**.*

With recent versions of MacOS, you have no installation media to install from. You have to rely on network downloads and whatever utilities are built into the machine. See this Apple page:

[https://support.apple.com/en-gb/HT201376](https://support.apple.com/en-gb/HT201376)

They refer to a recovery system. If that is a hidden recovery partition, then a full erase and install of Ubuntu will likely erase that too. Linked Apple page about recovery here:

[https://support.apple.com/en-gb/HT201314](https://support.apple.com/en-gb/HT201314)

This states:

> Newer Macs include the ability to start up directly from an Internet-based version OS X Recovery. Your Mac automatically uses this feature when the Recovery System on the startup drive isn't available. For example, if your startup drive encounters an issue, or if your startup drive has been replaced or erased. 

I suggest you post more details of your Macbook. An experienced Apple user may be able to help you decide whether this is a &#8220;newer Mac&#8221;.
