### Post by dreamof3d on 2009-01-18
I have a Mac Pro 2-dual quads 2.66, 6 gig ram, nvidia 8800gt. I have all my hd bays full. Bay 1 is a 300gb 10000rpm raptor for OS X. Bay 2 1 TB partitioned for media,ntfs,fat32, and Hfs. and software archives. Bay 3 and 4 are set up as Widows 7 Ult 64, and Ubuntu Studio x64.They are 250 gb and 300 gb drives. I found the best error impossible way to install Ubuntu, or any Linux flavor is to use Mac's diskUtility and partition a dedicated drive for Linux as MBR and "freespace". Next, drop your cd/dvd in the superdrive, and shut down the Mac. I then open the case, and gently ease the non-Linux drives out a bit on their sleds so they can't be accessed. Boot your Mac holding the "c" key and follow the easy Ubuntu install instructions. Of course say rules apply to Fedora 10, OpenSuse 11.1 etc. perfect install every time. When Ubuntu has re-booted and installed updates, i shut her down, ease the drives back in, close the case, and reboot holding the option key. select the disk labled "windows" and you are good to go.

From experience of trying to find the perfect Linux, (Ubuntu) :) This is the only FAILSAFE way I've found if you don't want the install creating bogus 200 mb partitions on your Mac side, or windows for that matter. 
Installing Winows of any flavor? Just make sure you install on the proper drive, and using bootcamp is not at all necessary. Windows will not write a boot partition on any othe drives but it's install drive. So no need to back out drives for windoz. My dives are slightly different in size, so I know which is which. Dissabling the other drives for Linux leaves no room for error. Hope this helped. Very simple and very safe way to get Linux on a Mac Pro.

BTW dj-toonz, In Gnome, everything works perfect including the keyboard eject button, and the digital/optical out. Mine is pumped though my Insignia 5.1 surround reciever @ 120 watts per channel. I also installed KDE 4.2 so I can have different desktop pictures on my 28.5" and 22" Viewsonics. Eject button not working when switching desktop session to KDE. Other than that, you will be better than twice as fast as my fighter jet. Good Luck!!
