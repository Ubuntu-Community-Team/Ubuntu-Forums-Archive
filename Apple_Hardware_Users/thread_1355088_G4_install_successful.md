### Post by Donald1715 on 2009-12-14
After some research thru the forums here, I found the solution to installing Ubuntu on my PowerPC Mac G4.

I had tried at first to install Ubuntu 9.04 with a CD made on my PC, but it wouldn't boot. Then from info found in these forums, I downloaded edubuntu 6.06 ppc, on my PC, then burned the ISO file with the suggested program on my PC, then inserted the cd in my G4 and ... it worked. Booted fine, installation went smoothly, and edubuntu detected my LAN and everything NP.

I'm writing this now from the G4 and have done a little surfing the web and all is fine. I'm almost sad I'm giving this system to a friend, lol. But now that it's working well, I think it's a much better OS than what was on it originally, which was OSX Panther 10.3.5.

By the way, this G4 is a 400 mhz, 384 mb ram with a DVD  burner, an ATI Rage video card and two hard drives. A 20 gig with OSX Panther, and a 30 gig with Edubuntu 6.06

My next project is to get Ubuntu to run on my PC, but from what I've heard, there may be a problem with my Gforce video card.
