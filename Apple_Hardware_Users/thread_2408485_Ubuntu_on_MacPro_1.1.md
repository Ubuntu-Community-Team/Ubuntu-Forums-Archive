### Post by kevin160 on 2018-12-19
It is a very fiddly install, so if you are installing Linux for the  first time, I would suggest starting with a different computer if you  have one available. 

I used to run Ubuntu on my old pro 1,1.  It should still be possible.  You can't install with USB, so you'll either need to burn a CD/DVD or install to an HD using a different computer and then move that HD to the pro.  [This thread]("https://ubuntuforums.org/showthread.php?t=2287767") describes the method I've used in the past. 

You can also run Ubuntu in a virtual machine.  The Pro 1,1 can run Mac OS X El Capitan and VMware Fusion 8.5, both of which are reasonably current and useful.  

Keep in mind that machines from that era use ridiculous amounts of power.   You might use enough power over the course of a year to pay for a new i3.
