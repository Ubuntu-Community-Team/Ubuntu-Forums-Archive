---

### Post by alexmurray on 2009-07-21
If you run Ubuntu (or Windows for that matter) on your MacBook Pro, the Macbook Pro will automatically enable ONLY the discrete graphics controller, so you won't have to do anything yourself to enable it.

Unfortunately unless you boot via EFI (see [this thread]("http://ubuntuforums.org/showthread.php?t=995704") for more info), there is no way to use the onboard (non-discrete) graphics chip.
