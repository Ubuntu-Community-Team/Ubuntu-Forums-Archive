Is it the bcm4331 chip? (sudo lshw -C network | grep product)   If so, that's the same as the mini I have.  

I'm currently using the bcmwl-kernel-source package (the proprietary driver).  

I've found that the b43 driver (firmware-b43-installer + b43-fwcutter) works, but only in the 2.4 GHz band.  

No guarantees, but I often find that wifi problems on laptops are related to power-saving features, and turning that off tends to solve them.  You can check the current power management state of your wifi with the "iwconfig" command, and turn it off temporarily with (replace with your interface if different):

```
sudo iwconfig wlp2s0 power off
```

I assume that you're using tlp to manage power?  Do you find your wifi being okay when you are plugged into power?  If so, you can try changing the line in your /etc/default/tlp 

WIFI_PWR_ON_BAT=5

to 

WIFI_PWR_ON_BAT=1
