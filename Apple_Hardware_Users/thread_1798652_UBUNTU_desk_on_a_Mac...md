The easiest way IMHO to try it out without harming your system at all would be to download the [Ubuntu live CD]("http://www.ubuntu.com/download/ubuntu/download") and fire up your system and use the 'Try Ubuntu without installing' option.

Another viable option would be to download and install [VirtualBox]("http://www.virtualbox.org/wiki/Downloads") for MAC, install it and create a Virtual Machine of Ubuntu and see if you like it.
