### Post by prenuk on 2012-07-01
Try this website: [http://www.psychocats.net/ubuntu/virtualbox/](http://www.psychocats.net/ubuntu/virtualbox/)

You'll want to use VirtualBox, which is free Virtual Machine software. You can run Ubuntu in it. VirtualBox is open source and has a big community, so if you have any issues, you can just google them.  The link above is for installing Ubuntu in VirtualBox on windows, but it should be the same on a mac. Installation instructions for VirtualBox can be found on their website: [https://www.virtualbox.org/wiki/Downloads](https://www.virtualbox.org/wiki/Downloads)

Download the following to install virtual box.

[LIST]
[*]**VirtualBox 4.1.18 for OS X hosts** [ x86/amd64]("http://download.virtualbox.org/virtualbox/4.1.18/VirtualBox-4.1.18-78361-OSX.dmg")
[/LIST]
After that, the install instructions for ubuntu inside the VM should be the same as in the tutorial I mentioned at the top.

### Post by SirWhy on 2012-07-02
If you click on the 'Mount CD/DVD' box, it will give you an option to boot from a physical disk or an ISO. You can choose your ISO by using the second option and then just specifying the path to the ISO.
