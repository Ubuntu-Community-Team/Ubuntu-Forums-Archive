OSX doesn't run very well on VMware. If you use one of te modified versions that are around on the internet, it'll run very sloppy at most but definately not as suppozed.

I use Ubuntu on my server and PC at home. I also have a MBP running OSX with Ubuntu on a Virtual Machine. I was planning to use Ubuntu often when I bought it a year ago, but in practice I hardly use it except for some testing (that is, on my MBP!), because OSX is actually a sophisticated operating system that works very well on a laptop. 

Belief me: it's nothing like windows. Whereas on windows nothing ever works as it's suppozed to, OSX is actually an acceptable alternative for Ubuntu/Linux to my experience. Nice thing is that all command line applications are (mostly) the same. There's also some nice integration of software with the trackpad, and it can do a few hours longer with the batteries on OSX relative to Ubuntu.
