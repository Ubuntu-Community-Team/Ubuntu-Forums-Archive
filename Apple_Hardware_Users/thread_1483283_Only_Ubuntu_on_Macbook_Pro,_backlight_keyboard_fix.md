I have a Macbook Pro laptop that was running Snow Leopard 6.1 and decided that I would prefer having only a Linux system. So I reformatted the hdd and installed Ubuntu 10.04, which works perfectly... just as I had hoped. The only problem is that I no longer have any backlighting on the keyboard. Is there any way to restore the backlighting while keeping Ubuntu as the only operating system? Thanks.

### Post by lael on 2010-05-14
What Version of a Macbook Pro do you have?

There are a bunch of docs that might be able to help you [https://help.ubuntu.com/community/MacBookPro]("https://help.ubuntu.com/community/MacBookPro")

There is a package called 'pommed' that can help you with the keyboard backlight

```
sudo add-apt-repository ppa:mactel-support 
sudo apt-get install pommed
```
