### Post by urilabob on 2014-07-22
Thought I'd post this in case anyone else has similar issues
Symptoms :(
Main monitor works OK, external VGA monitor is detected OK (shows normally in 'Displays' system setting), but remains black (note: another monitor connected via DVI lead worked fine)
Solution 

[LIST]
[*] 	:KS 		
Go into Tools Icon (top right) - System Settings - Displays and clik on "Mirror Displays" button, wait a while with both screens black (I guess it's searching for a resolution that's compatible with both screens) then when it reappears, turn off the "Mirror Displays" button, and the external monitor magically works

	
[/LIST]
