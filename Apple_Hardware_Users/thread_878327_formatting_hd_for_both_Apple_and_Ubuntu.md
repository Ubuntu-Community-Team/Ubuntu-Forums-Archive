### Post by Macintosh Sauce on 2008-08-02
> **Neal167 said:**
> I have an external usb hd and want to *share* files from a friend's Apple computer and transfer them to my computer which is running Ubuntu.  How do I format it (FAT32 etc) so both computers can read/write to the USB hard drive?

I would use Disk Utility on the Mac to format it as MSDOS (FAT). There is an option in the Erase tab for it. That way both computers can read it easily.
