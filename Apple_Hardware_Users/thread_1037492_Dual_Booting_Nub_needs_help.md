### Post by mkvnmtr on 2009-01-12
It is best to have the Mac os installed first. Then boot the mac install disk. With Disk Utility on the install disk make the mac partition smaller giving yourself enough free space to install Ubuntu. Of course any time you work with partitions you should back up your important stuff before you start. After the mac partition is the size you want make sure you didn't format the free space and back out of the install disk. Start in your mac os to be sure everything is ok. If it is not just reinstall on the mac partition with the mac install disk.  Then when you are happy with mac put in the Ubuntu powerpc disk that you burned as an image. Restart holding down the "c"key. Dont let up the key until first the mac icon comes up and then the linux icon comes up. When both the icons are up and the little ball stops spinning you can click on the linux icon and then the little arrow. This should start you boot of the live disk. When you start to boot and have any other problems come back to the forum. Anything that can happen probably will but it is all really easy to work around.
