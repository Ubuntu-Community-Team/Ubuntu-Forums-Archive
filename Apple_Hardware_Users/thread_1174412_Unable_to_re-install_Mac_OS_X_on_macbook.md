### Post by arabxptyltd on 2009-05-30
I have Ubuntu 8.10 on my Macbook (4,1 or 5,1) full install, no Mac OS X. I'm very happy with but due to my other Macbook being dead, I need to re-install Mac OS/X.

When booting from Mac OS X I get the error:

Mac OS X cannot be installed on this computer.  
OK Restart

There are no other options, and simply no way to install.

Any help is greatly appreciated

### Post by cyberdork33 on 2009-05-30
you need to use disk utility on the mac install dvd to reformat your entire hard drive. This will make it available to install OS X to.

### Post by arabxptyltd on 2009-05-31
Thanks. All menu options are disabled. There is no way to access Disk Utility.  
Only specify [Ok] or [Reboot]

### Post by arabxptyltd on 2009-05-31
The Solution was to find the Mac OS X CD's that are paired with the Macbook.  I have two sets for two computers less then a year apart.

I had to use the correct CD's to get a successful boot and access to the Disk Utility.
