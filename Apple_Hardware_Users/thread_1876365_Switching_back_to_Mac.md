### Post by Hatsune Miku on 2011-11-06
**EVERYONE CALM DOWN, I GOT THIS**

I get this problem all the time, Its more or less from timing. There is a very simple fix, VERY simple.

Hold the OPTION key, and ONLY that key as soon as u press the power button the Mac Mini.

It will bring up Apples EFI/Open Firmware boot-loader. Insert the Mac OS X Install disc, It will display an icon after a bit of time has passed, use your Mouse/keyboard to select the Install Disc. Hit the "Enter" key, or Click on the Install Disc. Install OS X then lulz.
