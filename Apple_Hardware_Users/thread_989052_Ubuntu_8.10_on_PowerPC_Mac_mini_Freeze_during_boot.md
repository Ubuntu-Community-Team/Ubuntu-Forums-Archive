### Post by tiresia on 2008-11-21
If you see in the 'main' download directory, there is no desktop, i.e. LiveCD for Intrepid PowerCD
 [http://cdimage.ubuntu.com/ports/releases/8.10/release/](http://cdimage.ubuntu.com/ports/releases/8.10/release/)

So I guess, you downloaded the daily version here
[http://cdimage.ubuntu.com/ports/daily-live/20081117/](http://cdimage.ubuntu.com/ports/daily-live/20081117/)
But as you can see, it is still the same since a month, and I doubt, that we'll ever get one for Intrepid. I tried it few weeks ago, and it did not work for me - and I never tried to understand why, also because I prefere the alternate installer for its lightness. 

Therefore, if you want to install Ubuntu I would suggest you to download the alternate CD.
The Intrepid alternate installer CD has a small problem. It can't detect the CD-ROM correctly. 
Here is the solution: [http://ubuntuforums.org/showthread.php?t=964904](http://ubuntuforums.org/showthread.php?t=964904)

If you want just to try the system before installing it, download Ubuntu Hardy (8.04.01). It is the LTS Release
