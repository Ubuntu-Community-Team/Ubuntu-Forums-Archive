### Post by rsavage on 2011-12-06
As I understand it (and unless things have changed) then the answer is no.  Although, if you do a search you'll find people trying to do it through emulation (!).
 
You can't compile the code because the actual daemon (the clever bit) is proprietary.
 
You should be able to use dropbox through the website I think.  I haven't got dropbox installed on my windows machine, yet I am able to use the 'Basic uploader' and download files too.  It should be the same on PPC.  So all is not lost.
 
You can use Ubuntu One to sync files to other Ubuntu machines, Windows machines or your phone.
