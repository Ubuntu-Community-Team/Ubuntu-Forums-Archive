### Post by linuxopjemac on 2011-06-10
First thing to check is if you have a working internet connection. 

The second thing to check is if this is a temporary problem in the server repository. Do a 
```
sudo apt-get update
sudo apt-get upgrade
```

It is also likely that your /etc/apt/sources.list file is not correct.

Can you give us the output of:
```
cat /etc/apt/sources.list

```

sudo apt-get update
sudo apt-get upgrade
```you will be asked for your password. Enter your login password; there will be no visual feedback, then press the enter key.

As you are using 10.04, it would sem that if you had previously updated successfully, the deleted files (the "404 Not Found") should have already been replaced by update.
