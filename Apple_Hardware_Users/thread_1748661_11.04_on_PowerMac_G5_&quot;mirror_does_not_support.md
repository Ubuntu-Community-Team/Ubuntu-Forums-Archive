### Post by avtolle on 2011-05-04
It sounds to me you have run into the problem set out in [http://ubuntuforums.org/showthread.php?t=1741799](http://ubuntuforums.org/showthread.php?t=1741799) and you should take a look at it. I would grab the alternate iso, as suggested, then go for an expert cli install to be sure you can designate the correct mirror, then do ```
sudo apt-get update && sudo apt-get install xorg ubuntu-desktop
```after rebooting following the install. You might also need to load a working /etc/X11/xorg.conf file while at the CLI before a reboot after installing xorg and the meta-package. Good luck.

I've not upgraded yet, as it is my general policy to not do so for a month after a new release, so I've not personally tried this on 11.04. This procedure has worked for me starting with 9.04 forward.

### Post by keithmck on 2011-05-05
Thanks! That worked. In case anyone else is trying to do the same thing, here's what I did to get everything working:

0. Make sure the G5 is connected to the internet over ethernet.
1. Boot alternate installer iso.
2. At the prompt do a CLI install for PPC: ```
cli-powerpc64
```
3. Go through the installer normally.
4. When you're asked for a mirror for the repos, use: ```
ports.ubuntu.com
```
5. Finish installer and reboot.
6. Log in at the command line.
7. Install Ubuntu desktop using: ```
sudo apt-get update && sudo apt-get install xorg ubuntu-desktop
```
8. Reboot.



I also wanted to get the wireless to work, so to do that I did the following:

1. Open Terminal and install the wireless firmware: ```
sudo apt-get install b43-fwcutter
```
2. Reboot.
3. Have wireless.
4. Unplug the ethernet cord from the G5 and make the wireless router work again so your girlfriend can order dinner on her laptop.
