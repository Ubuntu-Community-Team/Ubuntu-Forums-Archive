### Post by jake57 on 2015-03-11
Hi there,

I seen this thread and figured I could relate, as I've tried multiple versions of Itunes on Wine. The truth is iTunes is just going to run lousy on Wine regardless of which version you choose. One of the versions I ran(an older one, I think 7 or 6) on my last desktop actually worked well enough to access my music, however the store was never accessible, nor were my own purchased videos. It was also hopeless when I tried to connect my ipod and transfer any files.

From what I've read on the subject, iTunes is one of the windows programs that Wine simply cannot run well. I do not think you're going to have much success in getting it to work like it should. If syncing hasn't been able to work for anyone, I doubt the store will even have a chance at being stable enough to use.

Good luck if you do decide to play around with iTunes some more, but in my opinion it really isn't worth the time trying to configure Wine to run it.
