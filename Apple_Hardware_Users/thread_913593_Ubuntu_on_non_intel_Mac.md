### Post by Trott on 2008-09-07
I have an iBook G3 (not clamshell) and am replacing the hard drive in the near future. I was wondering if it's possible to install ubuntu on a non intel mac.

---

### Post by 3rdalbum on 2008-09-08
Very possible, but you need the community-maintained PowerPC edition of Ubuntu. Get the latest community version, not the old official Ubuntu Edgy. You might find Xubuntu a better fit for the hardware, especially if your iBook has pretty much the stock amount of RAM, but I'm not sure that there's a Xubuntu disc for PowerPC any more.

### Post by mkvnmtr on 2008-09-08

### Post by jtappin on 2008-09-08
I had Kubuntu Hardy running on a G3 iBook 600MHz, with 384Mb, without too much problem.

One thing to note however is that replacing the HDD on those guys is not for the faint-hearted -- unfortunately I've lost the link to a very good web page that showed how to do it. With that, a whole morning and a set of tools borrowed from an electronics technician who builds satellites I was able to do it and end up with no bits left over.
