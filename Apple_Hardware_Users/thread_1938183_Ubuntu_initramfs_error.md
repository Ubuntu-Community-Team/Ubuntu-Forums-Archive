### Post by Jon Monreal on 2012-03-09
I've found quite a few threads on various sites complaining about this exact issue across various versions of Ubuntu, with one possible "fix": use [unetbootin]("http://unetbootin.sourceforge.net/") to create a bootable USB drive.

To save time if you wish to attempt this, you could always use CD burning software to make an ISO from the CD, then use the ISO with unetbootin (rather than downloading again).
