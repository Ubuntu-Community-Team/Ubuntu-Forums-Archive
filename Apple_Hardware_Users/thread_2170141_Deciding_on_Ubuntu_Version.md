### Post by bcschmerker on 2013-08-25
Actually, Ubuntu® 10.4.7-LTS Lucid Lynx™ is only supported for Server applications, as of August 2013.  Desktop and notebook applications should use 12.04.3-LTS Precise Pangolin™, if the equipment needs more stable software than is available for Ubuntu® 13.04 Raring Ringtail™ or 13.08b2 (scheduled for RTM October 2013 as 13.10) Saucy Salamander™.
