### Post by Rob_H on 2009-01-07
If you really want iTunes running, your best bet is virtualization of Windows using VirtualBox or VMWare or <insert your favorite here>. Make sure you pick one with USB 2.0 support. Alternatively, you might be able to get it running under WINE. I've never tried.

But if you're not married to iTunes specifically, I'd highly recommend Amarok for Linux. It's a KDE app, but runs just fine under GNOME in Ubuntu. I've been using it for years for podcasts and music. I think the portable MP3 device support is still being reworked in the newest version (v2.0), so you'll probably want to stick with v1.4, which is the version that's in the Ubuntu repos. Good luck.
