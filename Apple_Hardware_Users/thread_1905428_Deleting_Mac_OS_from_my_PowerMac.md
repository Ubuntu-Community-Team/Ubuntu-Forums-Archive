### Post by canhoto on 2012-01-06
Hi.

In the last few months I've been using Ubuntu (mostly Xubuntu) on my (upgraded) PowerMac G4. 
Despite a few and not very important limitations regarding software that doesn't exist for PPC (like Google's Picasa and GoogleEarth), I discovered a new computer, which can be fast and functional without remembering me all the time that it's an old computer.
Ubuntu really works.

I also discovered the wonders of open-source, and that's stupid to pay hundreds of dollars or euros with proprietary software when you have amazing open-source, free software.

I also find myself each day more against Apple's commercial strategy, forcing you to expensive hardware upgrades to be updated and letting you feel that if you don't have the latest cat you are not from this world, and that if you don't have the latest hardware you can't have the latest cat. Even if you have a great computer like the more recent G5's. Which are APPLE's G5's!

The only, tiny thing that keeps me from moving completely from Mac OS (apart from sentimental reasons and some respect for this great OS) is... OmniOutliner, which is a great piece of software with no equivalent on the Linux world.

Anyway, I found myself not using Mac OS at all on my PowerMac for weeks (even months). I only use it on my Macbook. And when I use it on my Power Mac I have always this feeling again, like if there was a red sign telling you "Hey, come on, give it up, your computer is garbage". It still works, but it has nothing to do with the speed and performance and software availability of Ubuntu. And the feeling of being still in the world of the leaving creatures that Ubuntu gives you.

So, I thought about using the 400 GB HD where I have Mac OS to:
- Install Ubuntu (maybe another release) or even another Linux distro, using 200 GB (the HD where I run Ubuntu has only 160 GB)
- Using the remaining about 200 GB for backups

So, I would like to ask:

- Do you think it is safe to do that?
- How should I do that safely?
- Am I a fool to do that or does someone have already done the same?

Thanks
