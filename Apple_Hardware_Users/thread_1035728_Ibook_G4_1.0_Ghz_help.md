### Post by McFrostey on 2009-01-10
i have the iMac G4 PowerPC, I would not recomend switching if you cant live without flash,linux games(xcept games based on the Quake3 engine in that case you have to compile them).

If you know you absoloutly want to install  Ubuntu 8.10 on a PPC donwload the iso... since Ubuntu doesnt officially support PPC the community picked up on Ubuntu's slack
download this ISO 


[http://cdimage.ubuntu.com/ports/releases/8.10/release/ubuntu-8.10-alternate-powerpc.iso](http://cdimage.ubuntu.com/ports/releases/8.10/release/ubuntu-8.10-alternate-powerpc.iso)

The CD your burning it to must be a CD-R and 700MB.

restart ur computer hold in C now the installation will work
