### Post by balloons on 2012-02-16
I decided to cross-post this call for testing to this subforum, as most folks running on apple hardware in here have limited support for the buttonless touchpads found on newer apple laptops in ubuntu. That's about to change!

A new driver is available in a ppa for testing that enables click draging and other multitouch operations. This support is hoping to go into precise, and could use some testing. **This does require you to be running precise**, but if your running it give it a try. At this point, the testing was supposed to be wrapping up tomorrow (Feb 17th), but I would appreciate feedback if you have a chance to test.

Please see this post for the details.
[SIZE="3"]**Note that you have to run the script on step 4! Apple hardware is not at the moment auto-recognized as being clickpads.**[/SIZE]
[http://www.theorangenotebook.com/2012/02/call-for-testing-clickpad.html](http://www.theorangenotebook.com/2012/02/call-for-testing-clickpad.html)
