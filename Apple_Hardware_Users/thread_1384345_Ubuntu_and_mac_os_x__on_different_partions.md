### Post by B_Free on 2010-01-18
Try [http://old-releases.ubuntu.com/releases/breezy/](http://old-releases.ubuntu.com/releases/breezy/). The live CD doesn't have an install option, so if you wanted to install it you need the alternative version. Many of the pit falls of other versions are not present in this older one. After all, your iMac is older and this Breezy OS was made for your machine. You can update this to a newer one if you decide you like it. You need at least 256 RAM for any distro.
