### Post by dansmith on 2009-06-26
As a long time Mac user who relies heavily on keyboard navigation, I've found myself unable to function well in an environment that requires me to track down the "Home" key in order to jump to the start of a line (I've got a Dell laptop, and "Home" is a tiny key squeezed on top of my numeric keypad.)  I had a terrible time finding good information about customizing this behavior, but I was finally able to make it work.  Here's my post, which might be of interest to other Mac users.

[http://ubuntuforums.org/showthread.php?p=7522446#post7522446](http://ubuntuforums.org/showthread.php?p=7522446#post7522446)

I should note that I've also used System->Preferences->Keyboard->Layouts->Layout Options->Alt/Win Key Behavior to use my "Alt" key (located where the Apple Command key lives) as "Control" and the Windows meta key as "Alt".
