### Post by TBABill on 2012-05-09
Once the driver is installed, which it appears it may already be, you can do as above and install again. A slight modification to the above may be ```
sudo apt-get install --reinstall bcmwl-kernel-source
```Once it finishes you can ```
sudo modprobe -r wl
``` Then ```
sudo modprobe wl
```If you don't get a connection, try to ```
sudo rfkill unblock all
```
