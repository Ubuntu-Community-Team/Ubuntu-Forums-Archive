### Post by stream303 on 2008-06-25
You can download Ubuntu PPC from here:

[https://wiki.ubuntu.com/PowerPCDownloads](https://wiki.ubuntu.com/PowerPCDownloads)

If you have enough ram, you can choose the live-cd desktop version.  If you don't have a lot of ram, and/or run into video difficulties, you can use the "alternate" version, which relies on text/ncurses for the installation.  (you still get a full gui afterwards).  I highly recommend the "alternate", unless you really need a live version.

You can also download a minimal network-install version, that also relies on text/ncurses for install, but downloads what it needs from the net with the very latest updates right from the start:

[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)

In this case, you'd want the 64-bit PPC version.

Hardy 8.04 is the latest, but you could try older versions as well if you run into trouble your first time out.  Because of your G5 iMac, I wouldn't go any lower than Feisty 7.04 due to thermal-fan issues which makes Dapper 6.06 a showstopper for instance.  And, if you have a 20" version with an nvidia card, you might run into a small screen-shift to the right with Hardy and Gutsy.  It doesn't bug me on my G5 iMac, but if it really drives you crazy, the answer is to run Feisty 7.04 - at least at first. :)

Want KDE or Xubuntu instead?  For a first-time install, I normally recommend sticking with Ubuntu, and then later downloading and switching to kubuntu-desktop or xubuntu-desktop from within an Ubuntu install.

So.... download and burn the image as an iso, to CD-R, at a low(ish) speed, and hold down the "C" key when booting up.

Whew!  To boil it down, I'd download the 8.04 Ubuntu "alternate", and see if you can deal with the minor screen issue, otherwise Feisty 7.04 Ubuntu "alternate" would be my second choice.
