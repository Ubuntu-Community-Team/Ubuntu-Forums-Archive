### Post by dsmoot on 2008-10-05
There are several links at the top of this forum to help get you started.  You need to read through them and see if you want to go through with this.  

Once you have the .iso image, you need to burn it to a CD using a tool that can burn an image file (I like the tool "burn" for OSX.

Once you have a bootable CD, you can test it by putting it in your drive and holding down "c" while rebooting.  You should be able to boot off the CD.  Just turn off power if you are not ready to install.

Read the FAQ's.  Download Refit.  Make a backup.  Verify your backup is bootable so you can always get back to a working system.  Then post again when you've done your homework.
