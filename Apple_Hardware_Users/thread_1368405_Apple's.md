### Post by tom4everitt on 2009-12-31
I would certainly go for an intel cpu. More and more programs don't come with ppc support anymore, and you can't install the latest os x on ppc. 

The intel processors are named either "dual core" or "dual 2 core". dual 2 core is 64 bit, whereas dual core is 32 bit (both are dual cores, obviously). 

The macbook and mac mini has not very good graphics support (except for the very latest of the latest). The macbook pro, imac and mac pro all has some sort of graphics card.
