Uh... I just used Bootcamp to install Windows XP (Home) using the 32Gig option (including the Mac drivers from the Leopard (1.5) disk... it's got a Windows installer for them on the Leopard disk) and then I installed Ubuntu 9.10 using the "Install within Windows" option from the 9.10 live disk to install from within Windows XP.  No hickups from Ubuntu on the install, either.

It works great!  I do have to use the Option key to select the Windows OS and then I choose Ubuntu instead of XP.

Haven't found anything that doesn't work yet.  Nor have I found any speedbumps, hangs, crashes or crawls.  Network, CDs, Internet, and USB mouse and drives work Ok.  I do need to get something to enable MP3 capability, but I understand that's normal for Ubuntu.

Oh yeah, it's a MacBook, 2.4GHz Core 2 Duo with 2 Gig of RAM and a 160Gig hard drive.  Got it refurbed from Apple.
