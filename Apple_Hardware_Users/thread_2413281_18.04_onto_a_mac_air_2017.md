### Post by savantelite on 2019-02-25
I am having the same problem on my macbook air 2015. On my prevous windows machine, I am given the option to rezize my storage for windows and linux during the gui startup. That has not been the case with my Mac Book Air running Mohave. I have spent time searching the web and found other how tos that had you make a partition with disk utility on Mac and then run the live USB and stick ubuntu there. But the problem is for me is Ubuntu can't even see the internal hard-drive. 

Apple has documentation and the program "Boot Camp Assistant" that works with Windows 10 ISO. In theory, one could make a partition with that, and once the partition is created, just run the live USB and place the Ubuntu there. But you still have to run some boot manager like rEFIT.  [http://refit.sourceforge.net/](http://refit.sourceforge.net/)
