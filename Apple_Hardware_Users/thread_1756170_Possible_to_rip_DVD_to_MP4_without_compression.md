### Post by handy on 2011-05-12
HandBrake should do the job for you.

You can set the size of the output file, you can choose between .mkv or .mp4, you can do a variety of other things as well.

It isn't in the standard Ubuntu repo's due to some library or whatever issue.

If you are genuinely interested in using it, 5 minutes of research should answer your questions re. Ubuntu installation.

I've been using it on Arch for quite some time, & it has proven itself to be an absolutely brilliant piece of software.

As good as it is, it is still being developed, which is a bonus.

I expect that it wouldn't have any problems converting .mkv to .mp4 .
