### Post by pbhedlund on 2012-05-28
> **whalogreg said:**
> What do I do from here?

So now we have identified the problem.  To completely remove the proprietary driver open Terminal and run this command

```
sudo apt-get purge bcmwl-kernel-source
```That should remove the above file. Otherwise do

```
sudo rm /etc/modprobe.d/blacklist-bcm43.conf
```Now try

```
sudo modprobe brcmsmac
```Wireless should now start and you should be able to connect to the network. If not, reboot and see if that works.
