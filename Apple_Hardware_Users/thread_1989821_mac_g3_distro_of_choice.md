### Post by MisterGaribaldi on 2012-05-30
Part of the problem with that sort of request is that really none of the "lighter" distros support PPC. I mean, I guess you could try something like Gentoo and have it only do a "lighter distro"-type of setup, but my suspicion is that would be 1000x more trouble than it'd be worth.

Debian PPC is pretty solid on older machines like that, to the extent that the hardware is still viable enough to support any more in the first place.

I used to have an old PowerMac G3 that I ran Debian on, and it was an even *older* system than you've got, and it worked well.

I'd recommend getting a little more RAM into that laptop, if you are able to do so. It would make it a lot easier to support.
