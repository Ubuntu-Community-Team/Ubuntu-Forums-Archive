### Post by ssam on 2009-01-07
USB is a standard. if a USB hub would not work with a mac then either the hub or the mac is not meeting the standard and would be allowed to use the usb logo.

i have never had any problem with any USB hub on a mac. you don't even have to worry about USB version 1 vs 2. the computer, hubs and devices all figure out the highest speed they can all support.
