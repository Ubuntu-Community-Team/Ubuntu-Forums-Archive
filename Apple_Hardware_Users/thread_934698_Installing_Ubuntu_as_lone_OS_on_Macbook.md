### Post by JetskiDude911 on 2008-10-03
As others have said, you should be able to just install Ubuntu on the whole hard drive, or use bootcamp to shrink down the OS X partition if you'd like to keep OS X on the machine. 

I did this once with 7.04 I think. Everything went fine except for some driver issues. I haven't tried this with 8.04, but I'm sure the driver support is much better at this point. 

If you decide to go back at any time, just pop your OS X disc in and boot from it.
