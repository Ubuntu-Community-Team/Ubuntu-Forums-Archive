### Post by factotum218 on 2012-08-01
I know, the thread is old. But I just finished up an hour ago and after doing some googling I wound up here. Now I run Xubuntu 12.04 on a G5 iMac PPC with 2 Gigs of ram.

I'll be honest, it's not to speedy. Not terrible, about as fast as OSX 10.4 but with better software options.

The nvidia card seems to hiccup a bit with the UI display (tabs don't show text sometimes, line numbers are replaced with a vertical grey area in Geany) but I've turned off copositing to help with window movement artifacts.

I wouldn't suggest running something like Eclipse on it. I would if I knew I wasn't going to have to close it ever again to wait through that launch time again. But I'm keeping my sofware choices slim.

If I where to do it again I might give Debian a shot for performance comparison, maybe setup openbox etc ala crunchbang. Otherwise I might have started with a minimal install and add xfce bits myself.

bf and bflegacy modules don't want to help, I've tried everything but I could not for the life of me get wireless working. Hopefully a usb wireless adapter will remedy that (yet to look into it).

I'm just happy to have it up and running for now. I'm hoping to take it to the office for coding and dev work. I hate typing on my laptop keyboard.
