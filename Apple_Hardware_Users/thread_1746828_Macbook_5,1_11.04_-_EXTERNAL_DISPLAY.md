### Post by alexmurray on 2011-08-08
The manual install simply means you need to install the binary nvidia driver after installing Ubuntu - make sure you've got the nvidia-current package installed (this installs the binary driver). Then start-up "NVIDIA X Server Settings" (nvidia-settings) and configure the external display from there.
