### Post by coffeecat on 2017-10-06
I found this firefox add-on:

[https://addons.mozilla.org/en-GB/firefox/addon/youtube-audio/?src=search](https://addons.mozilla.org/en-GB/firefox/addon/youtube-audio/?src=search)

Was that the sort of thing you were after?

Alternatively, this for Chrome browser:

[https://chrome.google.com/webstore/detail/audio-only-youtube/pkocpiliahoaohbolmkelakpiphnllog?hl=en](https://chrome.google.com/webstore/detail/audio-only-youtube/pkocpiliahoaohbolmkelakpiphnllog?hl=en)

I would imagine it would work on Chromium but, if not, you can download the deb installer for Chrome here:

[https://www.google.com/chrome/index.html](https://www.google.com/chrome/index.html)

That's assuming the old Macs are not so old that they are 32-bit. Chrome is only available for 64-bit systems.
