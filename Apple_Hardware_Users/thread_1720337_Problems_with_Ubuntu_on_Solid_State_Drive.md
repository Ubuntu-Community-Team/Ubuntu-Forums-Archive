Hello,

I recently "upgraded" my stock iMac 350 Slot Loader from the original Apple 7.8 Gig Hard disk to a Solid State 8 Gig hard disk. I would have rather had copied my old partition to the new drive, but instead I battled with the Apple internal CD Reader  , and it did not go so well. The alternate install CD had some issues reading, as did the standard install. In the end the Alternate CD decided to do an alternate , alternate and seemed to download most of what it needed while playing 20 questions along the way.

Now however when I start up some messages go by so quickly relating to disk state (errors?) I can not be sure what they say, and about half the time the hard disk is checked. Then that /temp or /tmp is not ready, before it gives a message about skipping profile relating to Firefox, then it boots completely.

Has anyone else run Ubuntu PPC from a solid state drive with similar or different results?

I believe this to be related to the solid state drive, which I checked thoroughly on another machine before installing it in the imac.



FYI there is a HUGE performance increase running Ubuntu on the SSD as opposed to the slow original Apple drive. No more clunky noises , and much less waiting, just as I had hoped.
