### Post by Penfold1971 on 2013-12-05
This is a wild shot, to do with the Folding at Home distributed computing project.

Here's my setup :

Machine with OS Ubuntu 12.04.3, now running F@H version 7.3.6. Machine runs headless.
**Intel iMac with F@HControl version 7.3.6 used to remotely monitor the F@HClient on the Ubuntu machine.**

Previously when all was F@H version 7.1.52, I could remotely (using F@HControl on my iMac) make the client pause or stop on the Ubuntu machine while I did a system restart (remotely via Terminal from the iMac) as necessary after particular software updates.

I understand that now that I should be able to "Finish" the client by right clicking the slot shown under under 'Folding slots' in the Status window of FAHControl. This works fine on the Ubuntu machine itself, but control-clicking the corresponding slot in F@HControl on my iMac doesn't work. I'd like to return the Ubuntu machine to its headless state to save watts, but in order to do so I'd want to be able to "Finish" remotely from the iMac.

Is there some sort of command to use in Terminal on my iMac, that will cause the F@H client on the Ubuntu machine to finish?

There used to be a thread here devoted to Folding, but I can't find it just now.

---

### Post by Penfold1971 on 2013-12-20
Thanks to a fellow folder, I've got the answer. It's easy.

I use the Mighty Mouse that came with the Intel iMac. In Sys Prefs, just allocate 'Secondary Button' to (say) the right button. Job Done.
