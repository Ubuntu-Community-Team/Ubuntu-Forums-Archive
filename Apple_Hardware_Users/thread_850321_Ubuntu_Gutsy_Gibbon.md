### Post by the8thstar on 2008-07-05
If you are using a virtual HD with Parallels, ONLY that HD (or file actually) is going to get formatted. In other words, it's a virtual format to allow Ubuntu to install on that file. 

Your real disk will not be erased, therefore you can use all the disk space for that operation.
