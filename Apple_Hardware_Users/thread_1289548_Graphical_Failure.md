### Post by alexmurray on 2009-10-12
Out of interest why are you efi booting? Basically since you are booting via efi both the 9400M and 9600M GT are available and X doesn't know which card to use - you need to have a section in your xorg.conf which specifies to use the nvidia driver, and to tell it which pci device (ie video card) to use:

```
Section "Device"
    Driver      "nvidia"
    BusID       "PCI:03:00:0" # for 9400M or
#   BusID       "PCI:02:00:0" # for 9600M GT
EndSection
```

See [here]("https://help.ubuntu.com/community/MacBookPro5-1_5-2/Jaunty#EFI") for more info on efi booting...

I'd suggest that instead you just boot via legacy bios emulation though...
