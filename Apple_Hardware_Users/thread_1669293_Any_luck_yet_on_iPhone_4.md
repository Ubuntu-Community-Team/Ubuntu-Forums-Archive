### Post by cyberey66 on 2011-01-18
> **edco76 said:**
> I know a few months ago no one had figured out how to add music to the new iphone with linux. Anyone have any luck on this yet?

As of now, not yet but close.  Keep an eye on libimobiledevice, [http://www.libimobiledevice.org/](http://www.libimobiledevice.org/) and libgpod [http://gtkpod.org/wiki/Libgpod](http://gtkpod.org/wiki/Libgpod).  They can read/write to the iphone 4, but the libraries used for syncing music are still implementing it.

There will be no need to jailbreak your iPhone when it is implemented.

As for now, the most painless way to add music is to using a virtual machine with windows and using this program which supports "drag and drop" copying of music, [http://www.copytrans.net/copytransmanager.php](http://www.copytrans.net/copytransmanager.php).  I'm really not a fan of iTunes, it's unusable if you have multiple computers and frequently want to add/remove songs/videos.
