### Post by overlord.gaurav on 2009-03-08
Never Winter Nights II works on Ubuntu under Wine.
To install wine, click on System > Administration > Synaptic Package Manager.
Browse through the list and select wine and click on apply.
Once the installation is complete.
Use [this]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=14492") page to fix issues in the game.
