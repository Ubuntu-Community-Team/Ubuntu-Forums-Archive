### Post by mfox on 2008-12-23
I don't think there's one distro that's any more compatible with MacOS files than another, since they all tend to use the same basic applications.  Look and feel is a different matter.  The ones that look and feel most like MacOSX out of the box are gOS 3 and DreamLinux.  The former is based on Ubuntu 8.04; I'm not sure about the latter.  But any distro can be customized to look like MacOSX by using MacOSX themes you can download. In general, you'll find the Gnome window manager to be more similar to the Mac than KDE, which is more Windows-like.  And they're all easy to network with Macs; there's a file you can download from the Ubuntu/Debian (and derivatives) repositories called Netatalk that makes this very easy.  It's also easy to access Mac files on the same computer that has a Linux installation.

But "easy" is relative; no Linux distro is as easy to set up as MacOSX.  It's not difficult, but there is a learning curve.
