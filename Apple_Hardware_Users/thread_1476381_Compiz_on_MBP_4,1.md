### Post by Mills00013 on 2010-05-08
Do you have the proprietary nvidia drivers? when you go to the hardware drivers menu inside system > administration, you should see two options for the nvidia drivers. you should install the one that says recommended. see if that fixes your problem...

If that doesnt work, try opening the appearance preference area and then go to visual effects and select the last option.
