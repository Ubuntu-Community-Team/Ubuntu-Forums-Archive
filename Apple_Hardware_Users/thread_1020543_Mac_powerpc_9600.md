### Post by tiresia on 2008-12-24
Even Ubuntu Intrepid can run on my PowerPc 7500.
I don't know if you ever installed Linux on an OldWorld PowerMac. You will always need a MacOS9 (or MacOS8, or MacOS7.5 - you can download the last one from the Apple WebSite ) in order to install BootX, the Kernel bootloader for this old machine. You can donwload it here:
[http://penguinppc.org/bootloaders/bootx/](http://penguinppc.org/bootloaders/bootx/)

So, yes, you can have Linux on those machines - and actually I like a lot to use it on my OldWorld PowerPc's. But I would buy one. It's a bit tricky to install it (yes, you need the alternate Installer) and the performance are not great. 
Definetely, Xubuntu will run better than Ubuntu - and of course, it's better if you use the CLI. I find that such machine can be a good small file/web server, if you need one. It depends what you want to do with.
Of course, you will be able to access the other HD on your system.
