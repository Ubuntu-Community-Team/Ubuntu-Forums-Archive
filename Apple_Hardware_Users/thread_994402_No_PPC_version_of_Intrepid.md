Also be aware that Gnome is going to be a bit heavy on that computer. An XFCE desktop (Xubuntu) might suit it better. 

But if you go that route, XFCE's file manager, Thunar, and XFDesktop, the icon manager are broken on the CD. If you search this forum you'll find my updated packages.

You may want to try Debian Lenny which has neither installer problems nor broken XFCE components, but then wireless setup is not seamless if you have an Airport One wireless card, and I think Debian is less pretty out of the box.

YMMV.
