### Post by mkvnmtr on 2009-03-02
Usually the solution is ram. I find to run Ubuntu I need 512Mb and then it is fast enough to work with. On my G4 Ibook I installed a 1Gb stick for a total of 1.25Gb and it is more than enough. If you open top or better htop if you have it installed you will see your memory usage. You should not be using swap much or you will be slow.
