### Post by steveo314 on 2008-12-08
I'm in the same hole. The linux kernel does not have support for MTPZ (Microsoft Zune). Any other mp3 player is supported. All you would be able to do is have a media player like amarok or songbird read what songs are on your Zune and populate the list. NO drap and drop. NO removal of songs. NO playback within linux. The only way to use it on linux is to install vmware. which is basically starting windows up within linux. ZUNE ONLY PLAYS WITH WINDOWS!!! :-(
