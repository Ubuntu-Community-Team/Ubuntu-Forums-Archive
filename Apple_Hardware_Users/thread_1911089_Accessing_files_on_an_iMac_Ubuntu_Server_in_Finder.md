### Post by CoffeeRain on 2012-01-19
Since it is a Mac, couldn't you just use the built-in Mac functions? Just look at the Sharing settings in System Preferences. You should be able to turn on Screen Sharing and File Sharing. File Sharing allows you to look at files on this computer as long as you are in the network. The Screen Sharing button should be next to the disconnect or connect button when you are trying to file share. If you click this, it acts like you are on that computer. :)
