### Post by Ir77 on 2012-02-18
This will probably be the issue :

The Natty LiveCD will not boot out of the box on MBP5,1 / MBP5,2 since nouveau fails on the 9600MGT (upstream bug report) - you can work around this by manually specifying the kernel command line option:

nouveau.noaccel=1 Some people have noticed that X wont start even with this setting unless the 9600Card is the selected card in macOS X here is the bug about it: [https://bugs.launchpad.net/ubuntu/+source/linux/+bug/787071](https://bugs.launchpad.net/ubuntu/+source/linux/+bug/787071)
