### Post by Johnsen332 on 2010-07-17
I have the same exact computer as you, and when I installed Ubuntu 10.04 (64 bit)  I had refit already installed. When I started my comp I saw the mac and linux partitions but only mac worked so I had to hold the option key during install and select the drive labeled Windows in order to load Ubuntu. Thinking I didn't need it I removed refit and tried to boot into Ubuntu, which didn't work, so I reinstalled it and it now works like a charm with no need to hold the option key. 

If you want to remove refit do this: 
1. On your Mac HD trash the efi folder
2. On Mac HD go to library > startupitems > and trash the refit blesser (or whatever it's called) 
3. Empty your trash and reboot twice (just to make sure) 

Let me know if this doesn't work or make any sense
