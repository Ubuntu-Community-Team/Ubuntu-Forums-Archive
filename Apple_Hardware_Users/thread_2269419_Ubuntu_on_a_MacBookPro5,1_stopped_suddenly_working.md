### Post by bapoumba on 2015-03-15
Just a few thoughts.

Natty is way beyond End Of Life, you would not get any security updates, better move to a supported release.

Next comment is that Apple hardware can have subtle variations according to release dates, so what works on one series may not on the next one. That is just the way it is.
I run Ubuntu on a late 2011 MBP in VirtualBox. No dual booting, to video issues, no fan problems, all these still get handled by Mac OS. Easy to set up, easy to install from the .iso file, no frustration :)

I'm not saying that you should do it this way, but i stopped pulling my hair out of dual booting.
