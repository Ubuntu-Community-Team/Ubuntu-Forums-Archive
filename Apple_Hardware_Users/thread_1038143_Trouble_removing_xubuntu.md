### Post by cyberdork33 on 2009-01-12
if you are passing the machine on, it probably the best idea to boot from the OSX installation DVD (hold alt during bootup until you see a menu). From there you can run Disk Utility and wipe your disk by repartitioning back to ne large partition. Then you can reinstall OS X from scratch and use BootCamp to install Windows if that is your plan.
