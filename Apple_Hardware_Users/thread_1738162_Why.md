### Post by pauljwells on 2011-04-24
I agree xcode is pretty nice :)

I'm writing mostly in Python with Fortran extensions, and that's so easy in Ubuntu, everything I need is in the repos and 2.7 is default.

On OSX, Python is a mess, with the mixed 64 and 32 bit stuff. OSX10.5 only has 2.5 and installing numpy, scipy and more obscure modules is a pain. Fortran compiling is also a pain. It's all possible, and easier than windows, but on ubuntu it just works so i can spend my time coding and not trying to get the tools to work!

Eclipse on OSX with PyDev isn't very stable in my experience either, esp on OSX10.6
