### Post by stream303 on 2008-08-08
If you are really running low on ram, or just want a very light ppc install, you may be interested in this:

[https://help.ubuntu.com/community/Installation/LowMemorySystems](https://help.ubuntu.com/community/Installation/LowMemorySystems)

While the intro describes Intel machines, I have used it to put Ubuntu ppc on several machines starting with nothing more than the ppc "server" install image, and then layering a small amount of xorg, xdm, xterm, etc on it as described.

You might even be interested in using a very minimal desktop like XFCE4 using these same techniques if super minimal window managers are a bit too spartan.
