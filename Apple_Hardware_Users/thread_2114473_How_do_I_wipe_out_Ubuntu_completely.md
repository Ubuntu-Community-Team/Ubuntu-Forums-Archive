### Post by bakelitedoorbell on 2013-02-10
> **phiboston said:**
> Does that mean that I have to reinstall my mac?

When you installed Ubuntu, did you set it up as a dual boot?  Or does your computer just have Ubuntu on it now?

If it's dual boot, you should back up (clone) your Mac partition first.  Hold the Option key while booting to choose a partition.

Then boot the computer using your Mac OSX install DVD, by putting in the DVD and turning the computer on while holding down the C key.  This will boot from the DVD instead of the hard drive.  Then erase the whole drive using Disk Utility.  Make sure you use the "Mac OS Extended" format.  After that you can reinstall OSX or restore your backup.
