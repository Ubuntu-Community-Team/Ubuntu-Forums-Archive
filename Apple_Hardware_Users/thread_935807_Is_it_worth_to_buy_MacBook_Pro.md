### Post by steam_engenius on 2008-10-02
If you just want to experience OS X and have the ability to run linux, then the low-end macbook would definitely suffice. I have a macbook and I keep OS X on it basically just so I can work with flash if I have to. The only justification I see for the macbook pro is if you're going to be doing some system-intensive work with OS X (audio/video editing namely, maybe gaming or something). If not, then I wouldn't say it's worth the extra $$$.
