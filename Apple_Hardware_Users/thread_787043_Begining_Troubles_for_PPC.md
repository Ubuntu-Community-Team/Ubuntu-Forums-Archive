### Post by ssam on 2008-05-08
There are 3 drivers for nvidia cards in linux.

nv, this is what you are using. its the old open source driver. it has no 3d accelleration, hence no compiz.

nvidia, the is the closed source propreitary driver from nvidia.  they only release it for x86 and x86-64. as it is closed source, no one can port it to powerpc. this has 3d accelleration.

nouveau. this is the new opensource driver. it will one day replace nv, but its not ready yet. it will have 3d acceleration.

If you are Linuxy enought that you know how to check source code out of a revision control system, compile stuff, and deal with broken X11, then you could give it a test. see [http://nouveau.freedesktop.org/wiki/](http://nouveau.freedesktop.org/wiki/) for details
