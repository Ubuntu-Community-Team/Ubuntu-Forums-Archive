### Post by Dougxdrums on 2009-09-23
I'll be honest with you.  I have had terrible luck dual booting Mac OS and Ubuntu.  I've done it twice and ended up wiping my hard drive and re-installing Mac OS onto my macbook (Thank god for time machine backups).  A few months ago I bought parallels desktop for mac, and haven't had a problem with it since I installed it.  It's a little expensive but it allows you to run Ubuntu inside of mac.  The only downside I have found is desktop effects will not work.  I believe there is even some open source software that will let you virtualize.  I think its called VMbox.
