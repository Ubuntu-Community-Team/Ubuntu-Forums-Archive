### Post by Dullstar on 2010-03-10
Through trial and error (it took a while to fix the results of a faulty set of directions :-P) I found out how to do it:

1.  Install rEFIt
2.  If it doesn't work, wait for the next version.  If it works, go to step 3.
3.  Install Linux - GRUB goes on Linux partition
4.  Install Windows - should work without problems - will update later.
