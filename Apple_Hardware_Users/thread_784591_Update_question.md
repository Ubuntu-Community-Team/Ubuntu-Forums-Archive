### Post by benanzo on 2008-05-06
It sounds like the packages were already downloaded on your Dell.  You can find out if you're set to download (but not install) updates automatically by going to System -> Administration -> Software Sources then select the "Updates" tab.  In the section for automatic updates I'll bet "Download all updates in the background" is selected on your Dell but not on your Mac.
