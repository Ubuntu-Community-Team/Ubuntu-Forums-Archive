### Post by Brightbelt on 2008-07-17
This is just a comment - for me, the latest batch of updates were getting stuck a lot. Really there were only 4 security updates and the rest were 'recommended' and had to do mostly with Evolution. 

When the install got stuck and I backed out, and I tried to go browse, I found that I lost my ability to browse online, even though my wireless showed to be active and healthy. Restarting fixed it every time.

I was able to install the updates a few at a time and finally they got done, but I might be able to save someone some time: the 
Linux-headers-image-26.24-19 update (the one with "image" in the name) seemed to be the one causing everything to get stuck. 

For me that was the fourth update down. The Main thing was, I think, that it really just needed to be installed last. The other linux-header updates probably worked as a foundation for it.
