Hi All,

Mozilla released [Firefox 29]("http://www.omgubuntu.co.uk/2014/04/firefox-29-released-whats-new-ubuntu"). Ubuntu built Firefox packages last week. Unfortunately the compiling of Firefox for PowerPC doesn't work at the moment.

**Firefox 29 build status** for the A-EON Live Remix distribution/Lubuntu 12.04.4: [launchpad.net]("https://launchpad.net/ubuntu/+source/firefox/29.0+build1-0ubuntu0.12.04.2/+build/5959936")

**Firefox 29 build status** for Lubuntu 14.04 LTS: [launchpad.net]("https://launchpad.net/ubuntu/+source/firefox/29.0+build1-0ubuntu0.14.04.2/+build/5959931")

Package build log for the A-EON Live Remix distribution/Lubuntu 12.04.4: [launchpad.net]("https://launchpad.net/ubuntu/precise/powerpc/+builds?build_state=all&build_text=&batch=75")

Package build log for Lubuntu 14.04 LTS: [launchpad.net]("https://launchpad.net/ubuntu/trusty/powerpc/+builds?build_state=all&build_text=&batch=75")

Cameron Kaiser has built [TenFourFox]("http://www.floodgap.com/software/tenfourfox/") (Firefox 29 for Mac OS X PowerPC) successfully: [oh-you-big-lug-have-some-australis-you.html]("http://tenfourfox.blogspot.de/2014/04/oh-you-big-lug-have-some-australis-you.html") and [tenfourfox-29-its-almost-here.html]("http://tenfourfox.blogspot.de/2014/04/tenfourfox-29-its-almost-here.html").

Since 24 doesn't end support until 31.2, I have 31a, 31b, 31.0  and 31.1 to get it right.
