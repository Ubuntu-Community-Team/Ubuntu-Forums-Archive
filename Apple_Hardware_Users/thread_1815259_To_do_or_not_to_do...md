### Post by pauljwells on 2011-08-01
DON'T DO IT!!

You need to keep an OSX partition to be able to install any firmware updates that Apple might release. You can do a minimal install ('customise' during install and de-select all the language and printer options etc.)

If you want a pure Ubuntu machine sell the mac and go for a PC with intel graphics. I did :-)
