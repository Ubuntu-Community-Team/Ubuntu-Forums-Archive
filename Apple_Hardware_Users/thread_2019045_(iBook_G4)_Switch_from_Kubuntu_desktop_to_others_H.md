### Post by Bucky Ball on 2012-07-07
Xubuntu uses xfce4. Install it, log out, change to 'xfce session' from the 'Session' pop-up.

Same for lxde (Lubuntu). Gnome not sure about but 'Gnome Classic' is in the Sessions pop-up of Ubuntu 12.04; you'll need to research for Gnome as things there are a bit fluid right now. ;)

Avoid installing another *buntu-desktop of anything: Be aware that this will install LOTS of packages and apps also which you will end up with in your app menus alongside the KDE stuff already in there. This is usually ok, if that's what you want, but can slow things down as it creates bloat and resource use and occasionally can cause conflicts. 

If you have enough disk space, maybe make a 'play' partition for installing other things to for experimentation/tweaking/breaking. It won't matter if that one breaks; you'll still have a stable (hopefully) Kubuntu install on another partition.

(Incidentally, FYI, there a heaps of DE you can use on Ubuntu/Linux, not just the default ones with the different flavours of Ubuntu; see Openbox, Fluxbox, Enlightenment, and more ...)
