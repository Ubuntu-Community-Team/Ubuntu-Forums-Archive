### Post by tiresia on 2009-01-12
Well, if you want, you can try following:
1. Insert the CD-Installer, and boot the computer holding CD
2. Choose to install Ubuntu on the USB Memory Stick
3. After the installation, start again the Computer. If you get again only to macOSX, try restarting and hold the Option-Key down. You should see a Linux Icon. Choose it and start from it.

Let me know, if you succed, and how fast (slow) is the system
