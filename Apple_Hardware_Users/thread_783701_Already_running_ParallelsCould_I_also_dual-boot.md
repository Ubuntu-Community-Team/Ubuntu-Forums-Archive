### Post by agim on 2008-05-06
It should work just fine. Parallels is hard on a processor, not the hard drive. The processor won't care that you have a partitioned hard drive. Once partitioned, for all intents and purposes, the second partition acts as an external hard drive.
So yeah, give it a shot.

Ubuntu, Windows and Leopard on the same computer, sounds interesting.
