### Post by cyberdork33 on 2009-02-17
the x86 normal ubuntu version will be the most compatible with most machines, but there are several versions of the Mac Mini, some have a PowerPC CPU and others have Intel CPUs. This will be the first determination you need to make as far as the architecture.

If Intel-based, there is an issue with booting what apple designates a "legacy OS" from USB or firewire. See the FAQ for more details on this and the architecture question.
