### Post by alexmurray on 2010-05-30
You need to disable the trackpad while typing - essentially whats happening is as you type you're brushing the touchpad with your hands and hence moving the mouse - its not Ubuntu's fault but yours :) - You should be able to do this by going to the Prefences -> Mouse menu and under that trackpad tab select 'Disable while typing' or something like that (writing this from memory from a windows machine).
