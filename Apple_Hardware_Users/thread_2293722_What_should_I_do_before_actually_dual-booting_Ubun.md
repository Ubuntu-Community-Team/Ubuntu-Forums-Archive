### Post by mystics on 2015-09-07
You should be able to test these in a live session. Try booting from the USB and select "Try Ubuntu Without Installing". From the live session, you should be able to test out the devices. 

For the WiFi, you may need to enable the driver using "Additional Drivers". Once enabled, make sure you can connect to the Internet.

For the webcam, it may just work. Just find something that will turn it on (sorry, I'm not sure what the default application(s) are with Ubuntu using Unity).

If they work within the live session, then you should be good to dual boot. Otherwise, post your problems and, hopefully, someone will know how to deal with them.
