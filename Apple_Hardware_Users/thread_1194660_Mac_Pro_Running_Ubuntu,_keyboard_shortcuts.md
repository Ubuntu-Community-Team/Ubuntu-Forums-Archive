### Post by tiagobt on 2009-06-22
In an Apple keyboard, the F2 key increases the screen brightness. To obtain the "real" F2 key, you have to type **Fn + F2**. To open the run application dialog, you have to type **Alt + Fn + F2**. I know this is probably too complicated for everyday use, so you might want to assign a different shortcut to the run application dialog.

For more information about Apple keyboards, visit the Wiki:
[https://help.ubuntu.com/community/AppleKeyboard](https://help.ubuntu.com/community/AppleKeyboard)

The Wiki also shows how to disable the behavior I explained.
