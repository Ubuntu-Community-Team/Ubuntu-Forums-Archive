One thing I don't like about the MBP keyboard is that it lacks a number of interesting keys: PgUp and Down, Home and End, mainly. Fortunately one can use the Fn + arrow keys combination to get those, but it would be nicer if I had *another* Fn key right by the arrow keys, so that I could do stuff using only one hand.

Anyone knows whether is possible to swap the right Alt key to Fn? xev doesn't even detect an event when the key is pressed so I believe it is a full firmware implementation, somewhere in the keyboard. I know the behaviour of Fn + F1-F12 keys can be changed though, so maybe there is hope to change other stuff about Fn.
