### Post by Yumi on 2015-02-13
Just installed Linux Mint as dual boot from USB last week. Used this guide "community.linuxmint.com/tutorial/view/1649". It is not Ubuntu, but Linux Mint is a fork of Ubuntu. A installation of Ubuntu without dual boot is straight forward.

Please pay attention if you want to do dual boot. Before installing You must set up partitions using OS X disk utility for Ubuntu. Upgrading OS X overwrote my grub, do any upgrade of OS X before installing.
