### Post by cyberdork33 on 2009-02-13
There has recently been a lot of activity on several keyboard layout bugs in launchpad. If you have an issue with your keyboard layout (this seems to mainly affect international layouts) such as, keys typing the wrong characters, no defined AltGr key, etc, please see this bug:

[https://bugs.edge.launchpad.net/bugs/67188](https://bugs.edge.launchpad.net/bugs/67188)

and specifically, get the script from this comment:
[https://bugs.edge.launchpad.net/ubuntu/+source/xkeyboard-config/+bug/67188/comments/61](https://bugs.edge.launchpad.net/ubuntu/+source/xkeyboard-config/+bug/67188/comments/61)

and provide some information to get this issue fixed.
