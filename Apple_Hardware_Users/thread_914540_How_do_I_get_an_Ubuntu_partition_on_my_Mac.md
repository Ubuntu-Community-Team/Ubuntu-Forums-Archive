### Post by Tlalock on 2008-09-09
Yes, it will be faster if it is installed on the HD as opposed to virtualization.  Assuming you have one of the current generation Macbooks, you can easily create a partition and then remove it later if you so choose.

Here is a link with detailed instructions on how to do what you want: [https://help.ubuntu.com/community/MacBook](https://help.ubuntu.com/community/MacBook)
