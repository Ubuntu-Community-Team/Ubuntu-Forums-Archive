### Post by DrMelon on 2010-08-15
Code::Blocks is easily extensible through plugins - and it also has a version for all other major operating systems, so if you find yourself needing an IDE on Mac OSX for example, you've got one that's familiar and ready to use. 
Its developement is also very active and it regularly releases new versions with more features.
Also, most guides for setting up certain libraries (SDL, OpenGL, BASS, etc) have a section for Code::Blocks.
