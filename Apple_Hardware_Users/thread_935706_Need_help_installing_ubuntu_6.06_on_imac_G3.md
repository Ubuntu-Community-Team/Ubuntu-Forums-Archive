### Post by crapple on 2008-10-02
Change this make a back up first:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.bak
```

```
Section "Module"
Load "i2c"
Load "bitmap"
Load "ddc"
[B]Disable "glx"
Disable "dri"[/B]
Load "extmod"
Load "freetype"
Load "int10"
Load "type1"
Load "vbe"
EndSection
```

[https://wiki.ubuntu.com/PowerPCKnownIssues#Blank%20screen%20on%20iMac%20G3]("https://wiki.ubuntu.com/PowerPCKnownIssues#Blank%20screen%20on%20iMac%20G3")
For a more detailed explanation.
