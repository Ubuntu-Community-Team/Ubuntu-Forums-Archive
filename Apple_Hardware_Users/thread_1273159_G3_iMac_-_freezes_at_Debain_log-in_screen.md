### Post by sweetleaf on 2009-09-25
Do you know about virtual consoles in Linux? Learning about them was very useful to me when I was struggling with GUI startup issues.

Press Ctl-Alt-F1 (or F2, F3, ..., F6) to access a CLI login screen. From there you can run commands like [FONT="Courier New"]top[/FONT] to try to figure out what's going on.

Then, press Ctl-Alt-F7 to return to the GUI.
