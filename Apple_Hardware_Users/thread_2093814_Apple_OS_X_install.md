### Post by iMac71 on 2012-12-14
> **anw0625 said:**
> Can I do a full install of ubuntu on my macbook pro it has OS X 10.6.8 on it now?  I don't want to do dual boot, just ubuntu.  Thanks
there shouldn't be particular problems: you'd have simply to:
1) boot your mac holding the ALT key with a live CD or a live DVD of your preferred Ubuntu release (Ubuntu 12.04 works well for me) in the Superdrive;
2) press the ENTER key when the keyboard icon appears at the bottom of the screen;
3) follow the on screen instructions.
This is the easiest way for obtaining a full Ubuntu install which replaces OSX.
