### Post by germanix on 2010-12-27
If you are asking for an Ubuntu ISO for the Mac?. You just use the normal Ubuntu ISO of your choice (I did and it works), downloaded from the Ubuntu Homepage. To boot the live CD from you Mac Mini you first have to go to System preferences, startup disk and choose start from the CD. As far as I know you will not need any file system drivers. You can format the drive with GParted to whatever you want.
Edit: It seems there may be an Mac ISO after all. I stand corrected. You can get it from here:
[http://cdimage.ubuntu.com/ports/rele...-amd64+mac.iso](http://cdimage.ubuntu.com/ports/rele...-amd64+mac.iso)
