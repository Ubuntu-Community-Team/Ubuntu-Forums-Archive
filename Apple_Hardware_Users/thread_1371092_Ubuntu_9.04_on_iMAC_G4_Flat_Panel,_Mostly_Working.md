### Post by redwoodguy on 2010-01-03
I had a 15" iMAC G4 flat panel (white ball) and it was running dog slow for browsing. It is my mom's computer. It has 256MB memory. 

I installed Ubuntu 9.04 for PPC. I wiped the OS/X completely off. It boots, it runs and it generally works. 

PLUSES
- It runs much faster than OS/X using Firefox on broadband. It is in the respectable category now, where it was in the unusable category running FF on OS/X. 
- The F-Spot application is a decent sub for iPHoto. Works nice. 

MINUSES
- Can't get any Youtube videos to play. I tried all kinds of install this and thats, and it just got buggered up. I see on one FAQ that there simply is no flash player for PPC. Hard to believe, but I guess so? 

-The Boot screen, or the splash screen which has the Ubuntu logo is weirdly distorted. It is pixellated and inverted colors, but the system boots fine, so no big deal. 

- Booting on the PPC seems quite slow compared to my Ubunto Quad Core, or my Ubuntu Netbook. 

All in all, it is much better now and Mom likes it!

-Did I mention you can't play any kind of Internet video?

### Post by Gen2ly on 2010-01-04
Yes, no flash player for PPC.  There is Gnash an open-source project for Flash but it's still young.  It does however run most YOUtube videos though.

Ubuntu is a bit heavy for that processor, but especially memory.  You can either upgrade your memory (512 MB min. I'd recommend) or choose a lighter-weight desktop.
