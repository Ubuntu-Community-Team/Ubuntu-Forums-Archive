### Post by buschi on 2008-05-20
hi!

you can set up a compose key. hitting <compose key>, then ~, then n will give you a ñ. system -- preferences -- keyboard -- layout options -- compose key position -- choose! ü is <compose key>, then ", then u.

alternatively, you can go a bit deeper and assign those characters to alt+n, alt+u etc. check the links on [http://www.thamnos.de/computer-corner/keyboard-layout/]("http://www.thamnos.de/computer-corner/keyboard-layout/") for that.

good luck,
sebastian.

### Post by Immortaly007 on 2009-05-08
Yay it worked for me as well :D éèáèëü etc!! (But i did have to remove the other keyboard layout from the list first).

So for those of you still wondering how to do it:
System - Preferences - Keyboard - Layout - Choose layout (for example VS) and then as variant choose VS international (with Dead Keys).
After you done that delete the other keyboard layouts from the list and voila!
