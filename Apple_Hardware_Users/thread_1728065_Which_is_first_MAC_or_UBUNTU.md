### Post by rg4w on 2011-04-13
I can't speak with confidence about most of those, but I'm pretty sure Linux had workspaces long before they found their way into OS X.  The Dock, however, is a holdover from NeXT, on which much of OS X is based (most of the APIs still use the "ns" prefix for "NeXTStep"), and I'd be surprised if there was anything in the Linux world like it prior to NeXT.

### Post by pauljwells on 2011-04-13
I seem to remember Apple being first with a lot of the file manager integration like playing mp3s and videos etc directly in the finder.

The top bar on the screen I'm pretty sure came originally from Gnome, although it's interesting to see the new Unity interface mimicking OS X by putting the app menus in there. A good thing in my opinion.

Another thing that you might not have considered is that OS X was the first OS to use the 3D graphics system to display the desktop, compiz came a long way afterwards.

Generally, Apple are pretty good at innovating nice GUI features
