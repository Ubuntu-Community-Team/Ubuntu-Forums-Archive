### Post by hajk on 2008-07-09
VMware Fusion runs in Mac OS X, allows 64-bit guests on 64-bit hosts (all recent Macs), and allows use of multiple processor cores (although only really useful with more than two cores available). None of the alternatives do, AFAIK.

Drawbacks of running in a VM: all VMs have limited graphics support, even when running on a host with a fine graphics card, so forget gaming; and the host tends to run hot with one or more VMs running (which stresses the processors) and, as a result, noisy on account of the fans blowing hard.
