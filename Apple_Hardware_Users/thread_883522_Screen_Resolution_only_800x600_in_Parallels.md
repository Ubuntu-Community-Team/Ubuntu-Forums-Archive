### Post by Euklid on 2008-08-08
Problem: It is not possible to set the screen resolution beyond 800X600 when running Ubuntu in Parallels. Same thing with Mandriva and others I guess!

Solution: Increase the Video Memory Size in Parallels (Double Click on Memory when vm is stopped) from 4MB to 16MB! Parallels is misleading here: It says 4MB is good for 1280x800!
