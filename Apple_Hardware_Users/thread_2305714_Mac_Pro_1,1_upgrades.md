### Post by Mario_C. on 2015-12-08
I still have a number of these machines at my company (many inactive). I have one set up with Ubuntu MATE (14.04.2) 64-bit, 32GB RAM, and an AMD5770 that was pulled from a newer Mac Pro  (5,1). It works great.

RAM modules for them are quite expensive to get now. We used Crucial memory modules, but the key thing is that the RAM must be **fully-buffered** DIMM (**FB-DIMM**). Check out [http://www.macvidcards.com](http://www.macvidcards.com) for PC video cards that have been flashed to work with it. I'm not sure about processor upgrade.
