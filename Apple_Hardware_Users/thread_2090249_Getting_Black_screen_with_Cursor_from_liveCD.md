### Post by trogdor1138 on 2012-12-06
Which disc image did you download and burn? Generally, only the special Mac version will boot in the manner you've described.

An easy way to check is to instead press and hold the 'option' key when you hear the boot chime. Hold this until the different boot options appear. If you see some discs labeled 'EFI boot' then you have the default desktop image. If you see only a disc labeled 'Windows' then you have the special Mac image.

If you know what EFI is and that's what you're trying to intentionally set up, then you do want the desktop image. If you have no idea what this is, I highly recommend using the Mac image instead for ease of use and installation.
