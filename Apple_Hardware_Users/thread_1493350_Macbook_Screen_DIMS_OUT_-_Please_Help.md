### Post by rat6fink on 2010-05-25
Try this:  System->Preferences->Screensaver->Power Management.  You should be under the "On AC Power" tab.  Uncheck the box that says "dim display when idle" When still in "Power Management" go to the "On Battery Power" tab, and you should see the same check box for dimming the display.  Uncheck that one too.  

That should do it.
