The one thing I could think of is that your keyboard is been detected differently during and after the installation.  If this is the problem you could try to pre-check your password spelling by writing it into the user name and, of course, erasing it afterwards (just to make sure Ubuntu is actually reading the same thing you think you're writing).

Os simple use the same word for both the user name and its password.  I know it's not a good security policy, but you can change it once you're logged in.
