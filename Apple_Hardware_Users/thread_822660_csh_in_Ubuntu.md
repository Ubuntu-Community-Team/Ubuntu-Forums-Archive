### Post by ibutho on 2008-06-08
csh and tcsh are available in the Ubuntu repostiories, so you can use the packaging tools to install them. You need to first enable the universe repository in your /etc/apt/sources.list or choose the option to show all installation sources in the Add/Remove software program.
