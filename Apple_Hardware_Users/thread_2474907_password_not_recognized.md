There is nothing different in the ubuntu way and all other Linux ways for accessing storage. They work the same.

If you are new to Linux, you shouldn't be running Ubuntu Server. You should be using one of the Ubuntu Desktop versions.  Gnome, KDE, Mate, XFCE, LXQt ... anything with a GUI.
