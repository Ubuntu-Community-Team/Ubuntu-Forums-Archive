### Post by hyperboloid on 2009-05-20
In Gnome I just go to System -> Preferences -> Power Management, where there is a checkbox for this. 

Another way (in Gnome) is to run gconf-editor from the command line, and go to apps -> gnome-power-manager -> buttons. 

Dunno if those tools are available in KDE, but there must be equivalents some where.
