### Post by DougInKY on 2010-12-03
I am a long time Linux user who had been somewhat away for the past year. I had been running Ubuntu on an old PC that I had but the motherboard finally gave up the dust a little over a year ago so I had been living with Linux running under Virtual Machines. Finally got tired of running Linux in a Virtual Machine on my iBook and my MacBook, so I decided to take action.  

I only have a 160gb hard drive in my iMac so it will stay with Linux on VMWare Fusion on it, but I have a 500gb hard drive in my MacBook Core 2 Duo,2 ghz (a BlackBook if you care).

I Decided to set my MacBook up as a dual boot machine, OS X and Ubuntu 10.4 LTS. Started reading yesterday and reinstalled OS X. Put Ubuntu on the top half of the Machine and feel like the world is right again. Still don't have multimedia running but the wifi, sleep function, printer via my wireless hub, etc are all running just fine. No regrets, in fact I am thrilled. Wish I had done this a long time ago.
