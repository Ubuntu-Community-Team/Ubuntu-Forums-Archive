### Post by guj4_n3b3sk4 on 2008-04-25
Well, try finding some players that can play .vro files so you don't bother with converting. VLC (that can be found for Win and I've used it on Mac, dunno if there's Linux version) for example can play .vro files. So, if you will watch that movies in Mac, you can download VLC media player for Mac, and watch it.

On the other side, if you want to bother with converting .vro to some other file format, here's the link of one converter that offers you many possibilities and file format's to convert .vro into:

[http://www.dvd-tool.com/vro-to-vob-mpeg.html]("http://www.dvd-tool.com/vro-to-vob-mpeg.html")
