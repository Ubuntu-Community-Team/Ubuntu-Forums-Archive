### Post by musinet on 2012-06-09
It seems you are trying to do a CD Boot. In this case, rEFIt shows an "Window Icon" representing the CD, non obstant it is a Ubuntu distro in there. :-)

Choose the "Windows" option and the CD Boot will start.

Please refer to this link, so there you will find out much valuable info: [https://help.ubuntu.com/community/DualBoot/MacOSX](https://help.ubuntu.com/community/DualBoot/MacOSX)
