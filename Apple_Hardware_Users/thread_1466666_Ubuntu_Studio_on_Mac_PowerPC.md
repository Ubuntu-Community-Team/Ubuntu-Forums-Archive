Ubuntu Studio is only available in Intel86 / AMD64, PowerPC.  You can install a std version of Ubuntu PPC and then add the Ubuntu Studio pkg manually using Synaptic. (Not the add/remove from the Gnome menu.)  

This will install all the applications and also give you the look & feel of Ubuntu Studio by adding the themes / wallpapers.  The only way you will notice it is not a real Ubuntu Studio is from the actual Ubuntu icon on the menu that has not changed to the blue one from Ubuntu Studio.

I did this when I had a PowerMac G5, and it even worked in 64-bit mode.
