### Post by robert shearer on 2010-02-26
you could try a minimal install to get a command line system and then add a desktop from there.
Depends on how comfortable you are with the command line and how much research you want to put in.

Minimal install images are here, scroll down for 32bitppc.

[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)

I have used this to install on a ppcg3/384MegRam and put the light 'Lubuntu' desktop on it.(lighter and faster than Xubuntu)
