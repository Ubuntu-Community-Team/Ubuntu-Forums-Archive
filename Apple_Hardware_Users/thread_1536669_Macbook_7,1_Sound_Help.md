### Post by labaom on 2010-07-23
Well you go in Alsa mixer like it says..

Now you would think if you move the slide things down it would get louder. That is what I thought. You have to move them up so the orange part of the slider disappears. So move them all up and unmute everything. 

Check mark surround speaker and default pcm and the one in between for headphone jack use but it will make the headphone jack red
