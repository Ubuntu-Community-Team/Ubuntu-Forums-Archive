### Post by mchladek on 2008-05-20
If you're looking to get back to a pure OS X installation, when you boot up from the OS X Install Disc you can open up Disk Utility from the menu bar up top.  I don't remember which heading it's under exactly, but it should be easy to find.  From Disk Utility, you can reformat your drive which will erase Ubuntu.

Edit:  And, of course, if you have any data on the drive, be sure to back it up.  Reformatting will erase everything.
