### Post by ajgreeny on 2010-12-16
I know nothing about mac file system permissions, but I suspect that you will be able to read those files and folders from ubuntu if you open them in nautilus with root permissions.  ```
gksudo nautilus
``` should open nautilus and allow you to read and copy files as needed.
