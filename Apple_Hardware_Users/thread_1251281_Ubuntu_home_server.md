### Post by turnerrw on 2009-08-31
No reason you should have any trouble here. Ubuntu shares folders using SAMBA. Mac OS X has native SAMBA support ... so you're shared Ubuntu folders will, to the Mac, appear on your LAN just like Windows shared folders. I do this all the time. My primary file server is Ubuntu, but my kids access it from Windows. No worries.
