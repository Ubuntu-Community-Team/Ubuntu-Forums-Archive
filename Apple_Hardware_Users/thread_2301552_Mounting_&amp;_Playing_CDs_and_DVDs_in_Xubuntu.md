### Post by Ed_Hughes on 2015-11-03
I have Xubuntu 15.10 on an imac with an external Apple USB Superdrive to play CDs and DVDs.
CDs - appear on desktop and autoplay using any application but when I try to eject them they are not shown as mounted on the context menu. I have to open File Manager or Disks to get an Eject option.
DVDs - No problem with normal formats which mount, autoplay and can be ejected from the desktop.
Restricted format DVDs - after following the instructiions here: [https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs](https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs) they can be played by opening in VLC or SMPlayer but they do not appear on the desktop or autoplay. Neither do they list in the File Manager and have to be ejected using Disks.
Sorry to mention this but all desktop flavours of Linux Mint, inclding XFCE mount and autoplay all CDs and DVDs.
