### Post by khelben1979 on 2008-08-10
I would like to use all the functions of the Logitech G15 keyboard together with Debian Etch on my laptop. How can fix this?

I have read on [gentoo Wiki]("http://gentoo-wiki.com/HARDWARE_Logitech_G15") to see what they have written on the matter and they have explained that one can do this:

keycode 176 = XF86AudioRaiseVolume
keycode 174 = XF86AudioLowerVolume
keycode 160 = XF86AudioMute
keycode 162 = XF86AudioPlay
keycode 144 = XF86AudioPrev
keycode 153 = XF86AudioNext
keycode 164 = XF86AudioStop

But since Debian Etch uses the X.org server I assume the above doesn't work.

Since this forum is mainly strived upon Ubuntu users and not Debian users I think that knowing how I can do it in Ubuntu might help me also.
