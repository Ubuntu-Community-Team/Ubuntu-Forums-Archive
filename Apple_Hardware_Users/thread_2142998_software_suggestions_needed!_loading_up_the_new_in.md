### Post by danield on 2013-05-09
-Google Chrome doesn't work on PPC :(
-Audacious doesn't sync with iPods last I checked.  You'll need Banshee or Rhythmbox for that.
-"has anyone used a wine emu of photoshop???"  Wine doesn't work on PPC.
-Task managers and status monitors are all fine on PPC as long as you don't need to monitor a battery.  I use Conky, though it's kind of a pain to set up.  The applets included with the Lubuntu taskbar are good.

### Post by rkmugen on 2013-05-10
If you need to run old PPC apps (designed for use on Mac OS 7.5 through Mac OS 9.0.4), you can attempt to compile or install Sheepshaver...

Though, you would need to obtain a usable "ROM" file to actually use the emulator and you'd also need access to a Mac OS installation disc (versions 8.x through 9.0.4, but nothing after 9.0.4).  Compatibility isn't perfect though, particularly for games.  There are a bunch of other issues with Sheepshaver, though if you really need to use Photoshop, I've managed to use Photoshop v3.0-6.0 on it.  Keep in mind though that I was using the Windows version of Sheepshaver.  The current -LINUX- version of Sheepshaver should work much in the same way however.

Installation/set-up details are here:  [http://www.emaculation.com/doku.php/ubuntu](http://www.emaculation.com/doku.php/ubuntu)
