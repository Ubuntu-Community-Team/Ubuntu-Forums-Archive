### Post by frog_pilot on 2008-05-24
What did you use for "linuxing" your G4.

I assume, you used ubuntu... :)

If you took an older version 6.06., 7.04, 7.10, you have to edit /etc/X11/xorg.conf. In 8.04 there is a graphical user interface to configure dual head.

In 8.04 look for "screen and graphics" in the menu or via alacarte ("main menu" in the settings section).

Dual head xorg.conf for any previous version could look like this:
```
#### DVI

	Option 		"ColorTiling" 	"1"

Section "Monitor"
    Identifier     "neovo"
    HorizSync       24.0 - 80.0
    VertRefresh     50.0 - 75.0
    Option         "DPMS"
EndSection

Section "Screen"
	Identifier	"TFT"
```

### Post by ssam on 2008-05-26
some apple computers have a restriction that only lets you use a second monitor as a mirror/clone of the first.

see [http://www.macworld.com/article/39969/2004/10/imacscreenspanning.html](http://www.macworld.com/article/39969/2004/10/imacscreenspanning.html) for talk of getting around the restriction.

on my powerbook (G4 titanium 15inch) in hardy, system->preferences->screen resolution works for detecting a second display. I think it helps a lot if you have an ATI graphics card, as these support xrandr 1.2.
