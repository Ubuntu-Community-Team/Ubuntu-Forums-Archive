### Post by ernsteiswuerfel on 2018-01-25
You could try Ubuntu MATE 16.04.3 ([klick]("http://cdimage.ubuntu.com/ubuntu-mate/releases/16.04.3/release/")). Some Mesa color bugs have been fixed in 12.0.x. As the point-releases of the long term supported versions update the graphics stack, it is possible that this will work on 16.04.3 now.

I am running Gentoo now on my G5, where this bug is fixed for good, mesa is much more recent (17.2.8). But Big Endian CPU graphics support always stays thrilling, so you may very well hit other graphics bugs. ;)
