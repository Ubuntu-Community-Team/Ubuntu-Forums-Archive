### Post by el.otro on 2009-06-24
If you are experiencing an input/output error, it is very likely a hardware problem, either your optical drive is not reading correctly, your cd is orrupted, or even your hdd might be the problem...I don't really have much experience with errors, I've been lucky to have no problems.  What it means is that the INPUT (from where the files are being copied) or the OUTPUT (where they're copied to) had an error.

However, pravindra, I'd recomend you first 

[LIST]
[*]try the option: "Check CD for Defects" on the CD installation menu
[*]then run memtest (I'm not sure about how much it has to do with that)
[*]try to run the liveCD and see if it runs correctly
[/LIST]
after that (which is a loooonggg process, specially memtest) you should know if it is a memory problem, if the CD you burned was the issue (which I'd be happy to hear), or at least you'll get some clues...That's what I'd do on that situation. And then record whatever significant messages you get, and someone more knoledgeble on the forum would probably know.

The best of lucks

### Post by mkvnmtr on 2009-06-24
The best way to get the ISO is to download the torrent. Less chance for error. Then burn with disk utility as slow as possible. Then if it boots up check the disk for errors. After the download you can or should check the I think they call it the md5 sum but on a torrent download I have never had to do it. Depending you may have trouble booting the disk when you have a good disk. If so post there are a number of ways to get it to boot or install. The 9.04 is working fine on my G4 Ibook but 8.04 and 8.10 might have been better. I trust that before long the bugs will be out of my 9.04 so that is what I would recommend.
