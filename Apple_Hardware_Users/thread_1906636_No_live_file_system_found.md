### Post by mpodroid on 2012-01-09
There's a bug in kernel 3.0 which prevents loading from CD.
Prepare a USB pen with the same ISO on it and boot with BOTH cd and USB pen

Check my post [here]("http://marpoz.blogspot.com/2012/01/ubuntu-on-macbook-pro-82.html")
