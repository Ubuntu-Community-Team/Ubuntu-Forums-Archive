### Post by phmadore on 2008-10-02
I'd say on any machine linux is more technically proficient in resource management, this is just a generalization, but I know it's why I intend to use ubuntu on my first ever mac, which is an older one I bought just today on Ebay. I've used Ubuntu on other systems... Anyway, for older macs, like mine, most everything is today unsupported, so the only best way to get a modern operating system that is always resource efficient is to go with linux, in my opinion. I'm sure there exist other options but I don't need to look into them. Further, another great advantage is that if you are using ubuntu on a pc, right, and then for some reason you decide to buy a mac, you won't have to deal with the whole problem that most people have when switching between, which is getting used to new controls and the entirely different setup of mac os. I don't know anything about mac os, really, and don't much intend to. I bought the ibook g3 on ebay so that I can take a clunker laptop to Iraq with me and still be able to do basic things without risking my much more expensive pc laptop. I've begun to like mac hardware, though I'm not sure I'll ever like their OS; even Itunes is a little too inefficient (but I love my ipod...) If I had more money I would have just bought a brand new mac air, and probably would have dual-booted, and I may still dual-boot, but anyway, this is all my uneducated opinion. Eventually I'll have about five different laptops and one desktop.
