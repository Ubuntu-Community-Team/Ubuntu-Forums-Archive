### Post by alexmurray on 2010-04-18
Ubuntu is based off of a completely different kernel than OSX (Linux vs BSD) so there are going to be inherent differences into how they handle hardware. In particular, Ubuntu (and Linux) is designed to be highly generic to support as many different platforms as possible, whereas OSX only has to support the limited range of Apple hardware, of which the OSX developers know intimately.

As a result OSX most likely has many different power-saving mechanisms built into it to handle the specific MacBook hardware, which Ubuntu does not. GPU switching definitely falls into this category but I'd say there is a lot more to it than just that. Ubuntu does use more power (simply because it isn't as aggressive in trying to save power) BUT for me atleast I still get enough battery life (just over 2 hours on my MBP5,1) to make it still worthwhile to use Ubuntu as my primary OS.
