I am trying to get keyboard retroillumination work on my machine but I have some trouble. It used to be OK in ubuntu with the mactel packages but I cannot get it to work the way I want in Arch.

Would you please dedicate 1 minute of your time to help a fellow?

Simply open a terminal then:

sudo kill `pidof gnome-power-manager`

then start it again with:

gnome-power-manager --verbose

Then hit the keyboard illumination keys (F5 and F6 on Macbookpro 5,5) and copy and paste here the corresponding code.
