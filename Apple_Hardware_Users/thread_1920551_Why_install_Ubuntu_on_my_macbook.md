### Post by Aearenda on 2012-02-05
Unless you have application software requirements that override all this, it seems to me that the operating system argument is mainly to do with principles - a free environment where you can find out what everything is doing if you want, and can diagnose and fix problems given the right skills, so giving back to the FLOSS community - or an increasingly controlled environment where one very wealthy company can change the entire landscape at a whim, and with no notice.

My experience of Ubuntu on a MacBook Pro 7,1 is that you pay a small price for that freedom - and the price is that not everything runs as smoothly as on OS X. Resuming for sleep, for example, is a little slower - and it flashes rubbish on the screen, briefly. It is necessary to tweak things to work properly, such as the brightness function keys, power management of the wifi hardware, and external displays. If you can live with that kind of thing, you gain freedom to theme it how you like, and so on.

In the end, everyone has a different take on this - I think you'll have to make up your own mind about what suits you!

### Post by 3rdalbum on 2012-02-05
You'll probably find that Linux is a lot more reliable.

Or, put it another way: If my experiences are worth going by, you'll probably have a filesystem crash every 18 months with Mac OS. If you use one of those programs that patches up the filesystem or claims to write a new filesystem non-destructively, then you'll get filesystem crashes every 3-6 months afterward.

On Linux... well, the only filesystem crash I've had was when the disk itself stopped working. And I'm using BTFS at the moment; if anything should crash it should be the experimental, in-heavy-development BTFS :-)

You may also simply find that Ubuntu is more exciting and easier to customise. Even with Unity. I don't know exactly what it is, but something about Ubuntu makes me just prefer Ubuntu over OS X.

In addition, the point for buying Mac hardware is that it is the only guaranteed stable way to run OS X, and the hardware and software are designed to mesh very tightly for optimal performance.  Once you take apart them apart, you start finding degradation in battery life, wireless connectivity etc.  You can tweak it to get things running close to what you had before under OS X, but it will never be better, IMHO.
