### Post by srt7300 on 2008-07-27
I just ran the Ubuntu Live CD on a Powerbook G4. 
A few random things I found helpful (this is not a complete step by step).

This link has links to the most recent Ubuntu versions for ppc mac.
[http://ubuntuforums.org/showthread.php?t=427714](http://ubuntuforums.org/showthread.php?t=427714)
The linked locations indicate which are for the mac powerpc, which is what you need. 

You can download the Ubuntu file (called an iso file) and burn it to a CD. Make sure you burn it as an ISO image and not just a regular file. Your CD burning program should have something to clearly indicate the option to burn an image or ISO image. 
You can do this on either mac or windows. ISO images don't care which platform you burn a CD from.

When you get to the Ubuntu desktop screen (after installing Ubuntu), in order to work with your mac files, you need to create a user that has the same UserID and GroupID as your current mac owner, and log into that user. I found that doing this stopped a whole set of seemingly unrelated problems I was having. 

I am working on a step by step for the ID thing, and other things I learned, but searching these terms should get you on the right path for now.
