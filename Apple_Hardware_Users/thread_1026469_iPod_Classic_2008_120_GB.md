### Post by stevenroose on 2009-01-02
I got the problem partially solved.

I succesfully added songs to the iPod using Hipo, but minding one important thing:
When adding songs with missing Tag Data (Artist: Unknown and Title: Unknown), the iPod failes to read them and concludes simply "No Music". Many mediaplayers, like Rhythmbox and Banshee index a file like "Nirvana - Smells like team spirit.mp3" with tags "Unknown - Unknown" as "Unknown Artist - Nirvana - Smells like team spirit", but the iPod doesn't!
So checkt all tracks with Unknown Artist en try to get their tags right.

But if you somewhere have a Windows pc near, I really recommand you [MediaMonkey]("http://mediamonkey.com")! The only reason why I use my home's Windows Desktop.
