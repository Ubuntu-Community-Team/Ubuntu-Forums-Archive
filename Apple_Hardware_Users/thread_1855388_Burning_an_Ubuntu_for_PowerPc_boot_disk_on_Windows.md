### Post by rsavage on 2011-10-06
Sorry to jump in and correct you hansdown, but G3s and G4s use the 32 bit version.  That link is also no good for powerpc.

Sylkain, you can use a windows machine to burn the cd.  Just set it to the lowest burn speed.  Also, before you do that you can make sure it has downloaded correctly by looking at checksums (I've never actually done this though).

For everything you ever wanted to know about powerpc and were afraid to ask there is this post [http://ubuntuforums.org/showthread.php?t=1798792](http://ubuntuforums.org/showthread.php?t=1798792) .  It tells you how to install lubuntu - probably the best thing for your machine.
