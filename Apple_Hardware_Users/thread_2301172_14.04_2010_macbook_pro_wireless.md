### Post by Rapacity on 2015-10-31
I got it working.

This is what I'm using now.

```
sudo modprobe -r b43 ssb
sudo modprobe ssb
sudo modprobe b43
sudo killall wpa_supplicant
sudo rfkill unblock all

```
