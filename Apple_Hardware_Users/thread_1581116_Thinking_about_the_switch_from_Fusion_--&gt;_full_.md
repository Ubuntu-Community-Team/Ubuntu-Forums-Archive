### Post by conundrumx on 2010-09-24
VMware (and I believe parallels as well, might be wrong) support DirectX, which is indeed 3D acceleration, but doesn't cover the methods and technologies used by Ubuntu.  I'd wager visual effects will work under a normal boot of Ubuntu, as opposed to a VM.  As far as performance from an Intel GMA...not so great.

The only reason I can think of to stick with OSX if you have a strong preference for Ubuntu is in the case you need access to OSX only applications.  The operating systems themselves are very similar functionally.
