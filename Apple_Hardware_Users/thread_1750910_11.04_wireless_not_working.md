### Post by Silmathoron on 2011-05-07
Okay. I guess that's good news (well sort of ^^). I think the driver is not really installed. What you can try is :
Press Alt + F2, write synaptic, press enter, enter your password and then search for broadcom in synaptic. You should find 5 packages. Select all those which are installed and remove them completely, close synaptic.
Then open a terminal and enter ```
sudo gedit '/etc/modprobe.d/blacklist.conf'
```
Check if you see b43 or b43legacy, if you do, remove them.
Then enter ```
sudo apt-get install bcmwl-kernel-source
```

Hope that'll work !
