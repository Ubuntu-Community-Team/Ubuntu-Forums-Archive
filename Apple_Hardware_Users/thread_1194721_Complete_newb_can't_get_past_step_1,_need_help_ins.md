### Post by tiagobt on 2009-06-23
The file you downloaded is correct (xubuntu-9.04-desktop-powerpc.iso). Now you have to burn the ISO image to a CD. Make sure you use the ISO burning mode (not a regular file burning mode). After that, restart your iMac with the CD inside. Hold down the "C" key as the computer powers on. When you hear the drive reading your CD, you can release the key. This should boot the live CD and let you install the new operating system.

For more information, check the links below (the instructions for Ubuntu and Xubuntu are pretty much the same):

[https://help.ubuntu.com/community/Installation/PowerPC](https://help.ubuntu.com/community/Installation/PowerPC)
[https://wiki.ubuntu.com/PowerPCFAQ](https://wiki.ubuntu.com/PowerPCFAQ)
