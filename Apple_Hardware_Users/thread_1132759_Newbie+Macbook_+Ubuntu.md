### Post by charliehorse55 on 2009-04-22
You will need to install refit initially, if only to sync the partition tables. After you have done that, you can uninstall. From then on the computer will automatically boot into mac, unless you hold the option key when booting. Holding option will show you a boot menu where you can choose either mac or linux. The intructions are provided in much more detail here:

[Apple Intel Installation]("https://help.ubuntu.com/community/MactelSupportTeam/AppleIntelInstallation")

--Evan
