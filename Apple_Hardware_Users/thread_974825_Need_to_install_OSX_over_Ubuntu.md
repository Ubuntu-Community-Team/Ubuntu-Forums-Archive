### Post by cyberdork33 on 2008-11-07
This is really not the place for this type of question since we are more apt to show how to put Ubuntu ON the Mac... but anyway...

Boot the OSX install disc (hold c on startup). In the menus, you can access DiskUtility. With DiskUtility you can repartition the entire drive with one (or maybe two... leaving space for a dual-boot install of Ubuntu) partition(s). Then you can exit back to the installer and it should allow you to install.
