### Post by yugnip on 2010-12-24
Check this link: [https://help.ubuntu.com/community/MacBook]("https://help.ubuntu.com/community/MacBook") and find your model. The instructions are fairly simple. For mac the main thing to remember is to install Grub on the Ubuntu partition. Also, to run Ubuntu without writing to the HD, wait for the menu after booting from the CD and choose 'live'.
