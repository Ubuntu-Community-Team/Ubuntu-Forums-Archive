### Post by mörgæs on 2011-10-22
Hi, welcome to the fora.

The guide is outdated, and yes, it is for dual boot. 

If you want a clean Buntu system, just boot from the USB stick (or CD) and choose 'use entire drive for Ubuntu' in the proces. Chances are that you only need pushing 'next, next, next, finish'. 

I can not tell if you should go for a 11.04 or 11.10. Best is to pick the one which works best in a live boot. 

Remember to have wired internet access while installing.
