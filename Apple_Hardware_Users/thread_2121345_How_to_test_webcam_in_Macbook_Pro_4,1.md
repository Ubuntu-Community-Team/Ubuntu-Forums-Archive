### Post by serpico007 on 2013-03-01
This is a post on my solution to help others.

I was wondering how to test the webcam in my latest install of 12.04 on my 4,1 macbook pro. I started searching any built in apps from the install and didn't find any. Then it came to me to go on Google+ and start a Hangout. The first time you do this it will ask to download and install a plugin and soon the webcam light turns green and your hangout starts. 

Hope this helps other users and saves them time from searching for an app to test their webcam.
