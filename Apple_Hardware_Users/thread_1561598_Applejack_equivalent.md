### Post by conundrumx on 2010-08-27
When you hold &#8984;+S during startup you're actually telling your Mac to enter single user mode, a feature which exists in all UNIX like OSes I know of.  What AppleJack does is simply attach itself to the login of root, so that you have immediate access to it.

Basically, Ubuntu has SUM the same way OSX does, and you can boot into it from GRUB ("Recovery mode"), but there is no script like AppleJack which will perform maintenance tasks for you.
