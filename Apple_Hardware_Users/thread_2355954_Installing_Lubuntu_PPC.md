### Post by katakaio on 2017-03-19
Hello mcmanuk,

Can you be more specific about the "command line" that you see when booting to the Lubuntu disc? (I'm assuming that you're using the [16.04 desktop image for PPC]("http://cdimage.ubuntu.com/lubuntu/releases/16.04.2/release/lubuntu-16.04-desktop-powerpc.iso"), but it would be great if you can confirm this.)

If your graphics are crashing with the desktop image (just a guess), you might consider installing with the [alternate image]("http://cdimage.ubuntu.com/lubuntu/releases/16.04.2/release/lubuntu-16.04-alternate-powerpc.iso"). The alternate images don't provide a "live" environment for you to explore, but they have a basic GUI for installing the OS that works when the graphics-heavy desktop installer doesn't.
