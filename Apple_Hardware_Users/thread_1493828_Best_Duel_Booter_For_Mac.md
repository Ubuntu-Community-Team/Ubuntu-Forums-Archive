### Post by paranoid_android on 2010-05-26
I triple boot my macbook and use refit and it works great, and it should give you a tux icon for booting ubuntu.  As far as I know there are no other options, but I could be wrong.  Also as far as using bootcamp, yes you can use it to help you install ubuntu but all it's really doing for you is partioning your hdd, you can just use disk utility to create a partition for ubuntu and refit will pick it up.  I think the disk utility route is the way to go bc if I remember correctly if you install linux via the bootcamp method refit gives you a windows icon.  Hope that helps.

### Post by sha.goyjo on 2010-05-26
AFAIK, rEFIt is currently your best option for running a linux/osx dual boot. You need something like rEFIt or bootcamp because you have to have a hybrid partition table, and you need a specialty boot manager like one of these to run it, and keep your partitions sync'd.
