### Post by Henry Higgins on 2008-10-25
Ubuntu has drivers for the old airport cards, which should intall automatically (they did on my iBook G3). If you have an airport extreme card, there's a bit of work to do (the drivers aren't included in the distro because of some rights issues, but you can easily get them on the Internet).

But since, as DRM free points out, there's no Flash plugin for PPC Linux, there's no point in installing Ubuntu on a PPC if you mainly want to watch YouTube videos.

### Post by stream303 on 2008-10-27
> **thetrystero said:**
>  As of now, I cannot watch any mpgs or avis because it's too jerky. even using firefox is a pain and it just slows to a crawl if i try to run thunderbird at the same time.

I think your main limitation regardless of OS is the ram.  Increasing it, if it would be cost-efficient, would be your best bet.

Much depends on what you do with it, although with Ubuntu PPC, you are going to be relying on 2D drivers etc, so I don't think Ubuntu will be able to wave the magic wand in the performance department, especially with only 320mb ram.

Unless you really desired to put Ubuntu on it for special reasons, I might be tempted to just live with it - maybe reinstall OSX 4, apply the latest combo-update (rather than doing incremental upgrades) to bring it up to 4.11, apply the security updates etc to start fresh, java, etc...

If Firefox is a tad slow, perhaps look into Camino (latest is 1.64) which is less resource hungry but uses the same Gecko rendering engine as Firefox....

If you can grab some extra memory cheaply, that would be the first thing I'd do...
