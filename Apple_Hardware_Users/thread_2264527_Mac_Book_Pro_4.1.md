### Post by grahammechanical on 2015-02-08
I have done many installs of Ubuntu for testing purposes and every time the default language for the installer is English and the default language for the operating system is English. There are of course different versions of English but we can set the keyboard layout to UK English and not US English. We do this during the installation process.

We also set the location so that the time settings are accurate for where we are living. I would be very surprised if setting the location to Paris or somewhere else in France would override the keyboard layout setting that we choose. Or the default language of the operating system.

I have also noticed that the web browser gets to know that I am in England and Google will give search results relative to my choices. So, I think you will have no problem accessing UK websites as you do now.

Do you have problems using OSX in English while you are living in France? Why do you expect Linux to be less capable than OSX? But you do need to understand that Linux is not a copy of OSX or of Windows. The Linux developers have their own ideas about how things are done. We have a variety of user interfaces to choose from. So, it is not impossible to find a user interface that suits us. All new things require some degree of learning.

I also recommend either Xubuntu or Lubuntu but not necessarily because of the age of your machine but because of your desire to put icons on the desktop. We can do that with Ubuntu but it is not obvious how it can be done. In Ubuntu we have icons of our most used applications in the Launcher on the left side of the screen. We also have the Dash which is like an overlay of the screen and on the Dash there are icons for our applications and recently used files.

It is a way of working that we can easily learn but if we do not wish to do that we can use Xubuntu or Lubuntu which may have a way of doing things that is more familiar to us.

### Post by oldrocker99 on 2015-02-08
):PDon't be concerned about your age;); I'm 66 and have been enjoyably using Ubuntu for 7 years. If you recall the Amiga, which was **fun** to use, you'll have as much fun, and even more, using Ubuntu. 

The default interface for Ubuntu is called Unity. A lot of people love it, and a lot of people dislike it. I'm personally running Ubuntu MATE, which is the desktop I fell in love with in 2008 (GNOME 2.3), and which was abandoned until an Argentinian hacker decided to update the source, and I'm in love with it.

There are also several other Ubuntu versions, Lubuntu (very lightweight and quick), Kubuntu (with KDE, arguably the best interface, and is VERY configurable), Ubuntu MATE (see above, and it is aleo pretty lightweight), and Xubuntu (with the similarly lightweight XFCE interface, which is somewhat more full-featured than LXDE), Edubuntu, for school-based educational purposes, Mythbuntu, for video work, and Ubuntu Studio, for multimedia content creation (Edubuntu, Ubuntu MATE, Mythbuntu, and Ubuntu Studio are not "official" versions, but they are allowed to use the *buntu name).

I prefer Ubuntu MATE because I think it has the best file manager (caja), is very full-featured, and is quick and easy to use, with top menus the way the Amiga had.

What you can do is download the various versions, burn a disk or (preferably) a 4GB USB drive, boot each one, and see which one you personally like. One of the many, many great advantages of Linux is that you have the **freedom** to use whichever desktop you want.
