### Post by dirt912 on 2011-04-13
I'm running the lates version of Maverick Meerkat on a macbook 2.1.  I had been using Lucid Lynx, but recently used the update manager to upgrade.

Whenever i try to use the built-in mic for any program (eg. skype, sound recorder, etc ...) nothing happens.

I have gotten the mic to work with skype on one occasion by going to:

system> preferences> sound

then under the hardware tab selecting "analog stereo input" from the "profile" drop menu

After that I was able to select "Internal Audio Analog Stereo" under the "Input" tab and turn up the mic volume.

By using this method I was able to get my mic to work, but then my speakers stopped working so i had to change the settings back.
