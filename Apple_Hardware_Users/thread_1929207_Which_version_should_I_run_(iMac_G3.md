### Post by Paddy Landau on 2012-02-21
Linux allows you to use a Live CD or, preferably, a Live USB (because a USB is faster). This lets you try it out without changing your system. (Instructions are on the [download page]("http://www.ubuntu.com/download/ubuntu/download").)

So, just try! Using a Live CD or Live USB, you will quickly find out whether or not it works on your computer. If it works, install it; if not, ask for help in getting it to work.

Ubuntu is great if you have at least 2Gb RAM. If you have a slow computer, go for Lubuntu.
