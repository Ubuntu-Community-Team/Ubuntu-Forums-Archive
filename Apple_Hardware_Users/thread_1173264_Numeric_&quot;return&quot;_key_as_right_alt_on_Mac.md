I've got a 2,1 MacBook with old keyboard layout, which means, that the button next to the right cmd key is the numeric keyboard return. I would like it to behave as right alt, which is absent in this layout. I know I propably need to change Xmodmap configuration, but i don't know, how to do it.

### Post by volanin on 2009-05-29
Easy!
Go to SYSTEM > PREFERENCES > KEYBOARD

Then choose the LAYOUTS tab.
There is a button down there called LAYOUT OPTIONS, click it!

Expand the KEY TO CHOOSE 3RD LEVEL options.
And select the option PRESS ENTER KEY ON KEYPAD TO CHOOSE 3RD LEVEL.

That's it!
