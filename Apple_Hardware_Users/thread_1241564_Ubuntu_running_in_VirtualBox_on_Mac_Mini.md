### Post by warreno on 2009-08-16
Officially geeking out now with Ubuntu Jaunty.

I threw it onto my Acer Aspire One netbook earlier this week, and after a few [hitches]("http://ubuntuforums.org/showthread.php?t=875887&page=2") (post #15) was pretty happy with the release, so when I discovered [VirtualBox]("http://www.virtualbox.org/") I figured why not. My home system is a Mini, but it's got the dual-core Intel chipset.

Currently Jaunty is downloading updates. But it works.

VirtualBox's documentation is a bit messy and needs some updating. It's not quite as easy as "Install this, then install that"; it needs work in some areas, especially since the emulator seems to have changed since the last time the PDF was written &#8212; that would be the PDF included on the Mac DMG file, not the docs available online. It would benefit HUGELY from screenshots.

Ah well.

Why? Because I do use Ubuntu on my netbook, and I want a sandbox that I can try things out on without having to worry about hosing my little cloud computer.

Also, I have a fantasy that I'll get VB for Ubuntu next, in the emulator, and use it to run WinXP. Then I'll get VB for XP, in the emulated emulator, and use it to run OSX. Because recursion is very amusing in an [MC Escher]("http://en.wikipedia.org/wiki/Ascending_and_Descending") kind of way.

Of course I won't; I don't have a quad core machine with 32 GB of RAM. But it is fun to think about.
