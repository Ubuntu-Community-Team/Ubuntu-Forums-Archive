### Post by whatthefunk on 2012-04-30
Linux programs wont work on an Apple.  Some programs, like Open Office, distribute Apple versions and so then they would work.  Kmail, a Linux specific KDE application, would not work on an Apple system.

EDIT: This got me curious and so Ive done some searching.  It is actually possible, you just have to tweak things a bit.

### Post by mode7 on 2012-04-30
Like whatthefunk said, certain projects provide OS X binaries. For instance, I use GIMP for OS X natively. I also use Gedit. A few other cross platforms programs I have run on OS X: Netbeans, Eclipse, QT Creator, Blender, Eduke32 :p, Google Chrome, ETC.

Also, several popular *nix programs have been made available via the Macports and Fink programs.
This includes KDE (Which KMail is a part of), although I cannot attest to it's stability.
[http://community.kde.org/Mac#Installing_KDE_on_Mac_OSX](http://community.kde.org/Mac#Installing_KDE_on_Mac_OSX)

Edit: As for files, it depends what you mean by "Ubuntu" files. Most common media file types can be read by both platforms. Mp3, etc..
