### Post by mevun on 2012-07-27
Disclaimer: I have not actually installed Ubuntu on my MacBookPro (and don't plan to), but I am planning to get a desktop for Ubuntu.  I did run Ubuntu in VirtualBox on my Mac for a short period of time.  My experience (with other Linux distributions) is mostly limited to software development.

Anyway, my rationale for getting Ubuntu is that free software downloads are easier with Linux distributions.  For example, I have MySQL (a database server) on my Mac, and it wasn't really very difficult to install after googling and finding a download site.  However, if you go to the Mac App store and search for MySQL or database, you will just find some relatively unpopular programs (like Bento) and some SQL utilities.  Relatively unpopular means that if you search online for questions related to that program, you won't find much help.

With the Linux distros that have repositories, it is simple to do apt-get, yum install, etc.  I know it's really not that hard to install popular open-source programs in Mac, but I just have higher confidence in the downloads for Linux because there is a larger user base for them compared to the Mac installation images.

I'm not willing to go Linux-only, because for general consumer purposes like browsing, office productivity (Pages, Numbers, Keynote), and iTunes, I am happy with those on the Mac.  Though maybe when I use Ubuntu (which is supposed to be very consumer-friendly), I will give up Mac OS X as I try the Linux equivalent programs.
