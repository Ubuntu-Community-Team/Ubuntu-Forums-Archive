### Post by pxwpxw on 2009-02-28
You need the Mac OSX DVD to set boot legacy, or another intel Mac with OSX bless function - MacBook as target , firewire connection. See the Intel Mac FAQ for the bless procedure to shorten the boot time. Would have been better to keep Mac OSX.
