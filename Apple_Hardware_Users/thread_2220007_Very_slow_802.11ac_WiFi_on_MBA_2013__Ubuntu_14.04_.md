### Post by varunendra on 2014-04-27
Welcome to the forums eugene11 !

As far as I know, 194 Mb/s speed is impossible with "bcmwl-kernel-source" driver. Unless it has been upgraded beyond my knowledge, it supports only a/b/g mode, not 'n' or higher, which means it will always be limited to 54 Mb/s at most (the limit of 'g' mode).

Did you check which driver was in use when you saw this speed? To check that -
```
lspci -nnk | grep -iA2 net
nm-tool
```
Actually only "nm-tool" is sufficient to show the current driver in use, the first command will also show exactly which card you have, so we can determine the best driver for it.
