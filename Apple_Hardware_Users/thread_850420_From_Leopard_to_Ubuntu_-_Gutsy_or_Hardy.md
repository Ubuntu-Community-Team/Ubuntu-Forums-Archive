### Post by _alex_ on 2008-07-06
Go with Hardy. However, I'd recommend a dualboot setup, that way you can have both OS X and Ubuntu :)
Check out the main guide at: [https://help.ubuntu.com/community/MacBook](https://help.ubuntu.com/community/MacBook)
You'll find all the details regarding wifi there. Also for a dualboot setup, the Macbook Pro guide offers a bit more info: [https://help.ubuntu.com/community/MacBookPro](https://help.ubuntu.com/community/MacBookPro)
