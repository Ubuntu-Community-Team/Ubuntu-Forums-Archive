### Post by tomreid on 2009-06-02
Hi

I am trying to load Ubuntu 9.04 onto a Powermac 4.1, but I cannot get started. 

I have read several times the instructions at [https://help.ubuntu.com/community/MactelSupportTeam/AppleIntelInstallation#Dual-Boot:%20Mac%20OSX%20and%20Ubuntu](https://help.ubuntu.com/community/MactelSupportTeam/AppleIntelInstallation#Dual-Boot:%20Mac%20OSX%20and%20Ubuntu)

I have backed my system up and installed Refit, but when I reboot the Mac, I get no new menu so I can only assume that the Refit software is not working properly. The text below that I copied from the Ubuntu help pages says that *"If you do not see the rEFIt boot menu when starting your Mac, it is not installed properly! *However there does not seem to be any help that I can find to tell me how to resolve the issue if it is not installed properly.

cheers

Tom.

[INSTALL rEFIt!]("http://refit.sourceforge.net/doc/c1s1_install.html") You don’t HAVE to use it once you get Ubuntu up and running, but until then, [INSTALL rEFIt!]("http://refit.sourceforge.net/doc/c1s1_install.html") You will need it after installing to sync partitions, once you have done that and Ubuntu works, then you can [uninstall it]("http://refit.sourceforge.net/doc/c1s3_remove.html"). You can even [burn a rEFIt CD]("http://refit.sourceforge.net/doc/c1s5_burning.html") if you prefer. If you do not see the rEFIt boot menu when starting your Mac, it is not installed properly!
