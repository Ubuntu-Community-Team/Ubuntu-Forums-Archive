### Post by ssam on 2008-10-19
it is probably possible, but it would be a very difficult task.

it is possible to install ubuntu from a running linux system using debootstrap, see  [https://help.ubuntu.com/8.04/installation-guide/powerpc/linux-upgrade.html](https://help.ubuntu.com/8.04/installation-guide/powerpc/linux-upgrade.html)

if you could get all the tools you neeed to work on mac os x then you could use that method.

however i think you would need a good understanding of linux/debian/ubuntu, mac os x and the differences between them. i have never heard of anyone doing this
