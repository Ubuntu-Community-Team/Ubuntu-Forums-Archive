Just so other people know, you can also have Ubuntu Studio on a PPC Macintosh (sort-of).  I used a Live Ubuntu 9.04 disc to install on my old PowerMac G5 1.8GHZ DP, it installed perfect. (only thing you need to do is install ATI drivers to get 3D acceleration)

Then just install the ubuntustudio-desktop package in Synaptic Pkg Manager. It will look like a PC version of Ubuntu Studio, except for the Applications menu has an Ubuntu icon and not a Ubuntu Studio icon. But you will have a Ubuntu Studio start-up/shutdown screens, wallpaper, etc.  And of course, you will have to look up the hacks to get Flash video playable.  I did not test out all the apps, just the basics, but they should work.
