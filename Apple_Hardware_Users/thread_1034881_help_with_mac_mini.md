### Post by mkvnmtr on 2009-01-09
Yes you can. If you have Panther I would guess it is a powerpc. I would use the disk utility on your Mac install disk to repartition making a smaller partition for mac. Leave the space for Ubuntu as free space. You should probably give it 15 or 20 GB. Then with the powerpc live disk you have downloaded restart holding down the "c" key. When you see the mac icon and the Linux icon come up you can let up on the key. Click on the linux icon you should boot up to the Ubuntu live disk. Use it a while and if you like it double click on the install icon. If you don't like it give the disk to a friend. If you have any problems booting or installing come back to the forum with any questions. It is pretty easy. I am dual booting the same two distros on my G4 Ibook.
