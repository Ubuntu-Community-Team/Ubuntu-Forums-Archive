### Post by zeroti on 2010-07-19
if your computer uses pbbuttonsd, you can try installing powerprefs if you don't have it already. it lets you set 'notap' while typing so that the mouse will move less. there are prob. ways to configure other set-ups for the same behavior.

there's always the possibility of upgrading to 10.04, that could solve some issues, but there's always the possibility that certain things might break or have to be configured. also, you could make sure you're up to date on upgrades to 9.10.
