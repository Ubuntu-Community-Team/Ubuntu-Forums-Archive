### Post by shawnhcorey on 2010-08-18
> **bmaxwell1970 said:**
> i dont think there is an alternate install for power pc, ill look some more.

There are two ISO images for a powerpc:  one is called the Desktop CD; the other is the Alternative CD.  To go [this page]("https://wiki.ubuntu.com/PowerPCDownloads") and choose the version you want.  When you switch to the new page, search for "Alternative install CD".

The Desktop CD will not fit onto a standard CD.

The difference between them is that the Desktop CD uses the GUI install program and the Alternative CD uses the test-based one.  Otherwise they're the same (except the Alternative CD is smaller).
