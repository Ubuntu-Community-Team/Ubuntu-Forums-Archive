### Post by rhythmiccycle on 2009-10-15
i don't know much about mac's,

but installing from the liveCD should work.

and if you are booted in windows then the wubi should work too

personally I think installing form the live CD will be better.

I haven't actually done it on a mac, but I did some thing similar...
if you install using wubi, then when you first boot, you'll get just the mac and windows os options, then if you pic windows, you'll get another boot option that will asks you ubuntu of windows.

if you install form the liveCD you should only get one prompt that has all the OS's.

also, I don't know all the technical details but when you install from wubi vs liveCD it partitions differently. And from personal experiences, liveCD install is more robust. For example, when I had a crash while installed in wubi, I had to do a check disk from with in windows, to be able to boot ubuntu again. But while installed from liveCD I've crashed a few times, and have never had any problems starting up again.
