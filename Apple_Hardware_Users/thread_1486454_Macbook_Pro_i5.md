### Post by newbie80 on 2010-05-17
I have mbp core i7 and have triple booted it with ubuntu lucid and win7. I do not see any problem with the installation so far except for couple of small issues (like for me xemacs is not working properly etc - there are alternatives for this) but most of the issues you have when you first install can be solved by following the instructions posted here [https://help.ubuntu.com/community/MacBookPro6-2/Lucid#LCD](https://help.ubuntu.com/community/MacBookPro6-2/Lucid#LCD). There is a link with instructions on how to install. There are also some specific instructions posted in one of the threads [http://ubuntuforums.org/showthread.php?t=1468240&page=4](http://ubuntuforums.org/showthread.php?t=1468240&page=4) - this is works with windows7 installed. i5 or i7 should not matter. You might also want to read the posted thread link carefully if you are doing only dual boot with lucid, as there seems to be some problem. I am not very sure about this but you may want to check on the threads more carefully about compatibility with SSD.
