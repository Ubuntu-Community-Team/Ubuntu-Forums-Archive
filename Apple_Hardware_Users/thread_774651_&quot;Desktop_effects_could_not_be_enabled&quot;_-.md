### Post by cyberdork33 on 2008-04-29
try installing fusion-icon and using that to control compiz.

```
sudo apt-get install fusion-icon
```

then ALT+F2 and run 'fusion-icon' you will get the control on the panel. You can add the fusion-icon command to your session to have it start automatically on startup.
