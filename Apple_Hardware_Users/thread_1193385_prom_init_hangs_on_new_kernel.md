It's probably just yaboot having trouble loading that 2.6.30 kernel, maybe the configuration, or that basic kernel version needing patching for ppc. 

It seems to be a lot smaller than the  2.6.26 (debian 501 lenny?) and it is a lot later version (ubuntu810 2.6.27, 904 2.6.28, 910 2.6.30).
The karmic kernel version  is currently 2.6.30 for intel, I dont know if the ppc karmic has caught up, maybe someone else has tried it. 

You could first try the ubuntu jaunty 904 ppc release kernel version 2.6.28
or for karmic(910) there is the daily build CD you could try booting to see how it goes.
[http://cdimage.ubuntu.com/ports/daily/current/](http://cdimage.ubuntu.com/ports/daily/current/)

karmic-alternate-powerpc.iso

Maybe you already tried all that.

Debian 500 2.6.26 seems to be very friendly to Macs both ppc and intel, although a bit short on the refinements.
