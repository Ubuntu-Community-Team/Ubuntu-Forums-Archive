### Post by Abir_Faisal on 2015-06-05
15.04 wouldn't install for some reason so I installed 14.04 LTS and upgraded to 15.04. I get a prompt telling me it can't find files. Turns out the same problem that prevents me from installing 15.04 is the same problem preventing me from being able to use kernel 3.19 and above. (3.19 is the kernel for 15.04) I tried using a USB mouse and keyboard but it completely disables the USB ports after loading the kernel.

Right now I'm using the 3.16 kernel from 14.10 it works fine. Keyboard works fine in grub its only after the kernel is loaded that it stops working. If I boot to 3.19 or 4.0 the USB ports and keyboard stop working. If I boot 3.16 it works fine. I'm not able to type the password for cryptsetup prompt in 3.19 or 4.0 but my keyboard works fine in 3.16 also I'm able to type the password from both the built in and external USB keyboard using kernel 3.16

Reason I want to use 3.19 and above is because I want native thunderbolt support and any new power saving features.

This problem is repeatable in both EFI and Bios modes. **I think its initramfs not loading the USB and keyboard drivers.**

Hardware: Macbook Pro 11,2
