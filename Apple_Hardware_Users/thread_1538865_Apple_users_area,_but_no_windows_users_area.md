### Post by Macskeeball on 2010-07-25
> **HelloIndustries said:**

I'd say i wasn't trying to stir-up debate, but that would be a lie.
As I understand it, this forum is not really about switching. It's about getting support for issues related to running Ubuntu on Apple **hardware**. Windows is not hardware, it's an OS. If you want support for running Linux in Windows, there's the virtualization forum (or General Help or whatever for wubi). If you want to discuss switching from Windows or Ubuntu on non-Apple hardware, you have all the other forums to choose from.

Specifically because of the marketshare of Windows, it is assumed. Ubuntu on Apple hardware has its own set of unique issues distinct from the broader PC market (EFI instead of BIOS, drivers for proprietary Apple hardware, older Macs using PowerPC instead of x86), which is why it gets its own special place.


> **Goolie said:**
> Isn't Mac OS X slightly based on linux?
No, no it is not. Linux is technically just a kernel, and Darwin (what OS X is built on top of) uses the Mach kernel and a BSD-subsystem. Linux is Unix-like, but Mac OS X is Unix-based.
