First I see that the above specs you can go with any flavor. The more lightweight ones would be Xubuntu, Lubuntu and Ubuntu Mate.

Then I would recommend the latest LTS, that is 16.04. There is no reason to install an older LTS version that you probably would have to replace in the near future. For a kind of server 16.04 is great and you get support till 2021. 
Therefore if all you need is Plex etc you could even go just with the server install.
