### Post by tiresia on 2008-09-15
Start your computer again and press immediately this four keys cmd+opt+O+F in order to access Open Firmware.
At the prompt run

```
devalias
```

You will see a list of all your devices and their open firmware path.
You must look for 'hd' (it is the alias for your hard disk) and its path.
It should be something like:
'device=/pci@f2000000/mac-io@17/ata-4@1f000/disk@0'
Note it (please pay attention to each details), and post it here.
Try:

```
boot hd:,\yaboot
```

If it does not work, try:

```
boot hd/disk@0:2,\yaboot
```

Let me know what happens

### Post by tiresia on 2008-09-22
In order to be sure that ubuntu detect correctly your HD run ```
ofpath /dev/sda2
```
You should get the *same* value. If so you can reinstall yaboot

```
 sudo yabootconfig -v 
```

If you can restart without problems, please mark the thread as 'solved'
