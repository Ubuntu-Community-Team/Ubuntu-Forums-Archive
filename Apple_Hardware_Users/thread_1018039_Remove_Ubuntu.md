### Post by Chrisj303 on 2008-12-21
Boot from your Ubuntu LIVE CD, then use the** partiton editor** to reformat the drive as either NTFS,FAT32 or HFS+ (anything that OS X can read)

Then reboot with the OS X install disc - and install OS X!


I also prefer OS X over Ubuntu, but like to keep a small partition for Ubuntu anyways (8GB is just fine). Maybe this is an option?

### Post by Captain- on 2008-12-27
Hold down the "C" key to boot from the OS X disc.  You can also try the ALT key to choose a boot up disk/OS.  Another possibility is, use the Remote Control if you have one, and hold the "menu" key after powering on, just prior to the boot sequence.  You can also hold down the left mouse button during boot to eject the disc.  Let us know if that helps.
