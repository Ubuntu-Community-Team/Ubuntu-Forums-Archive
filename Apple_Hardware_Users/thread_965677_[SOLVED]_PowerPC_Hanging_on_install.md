### Post by NYCNsciGuy on 2008-11-06
Hey!!  So when I did the install on my G4 iBook, I ran into the same problem you are describing.  At the prompt, if you enter "livecd-powerpc-nosplash" it should take you directly to the GUI where you can do the install.  (btw, check to make sure that I have the command correct...at the first prompt type "help", afterwards press the "Tab" key...this will bring up the list of possible boot command options!)

Hope this helps, and let me know if you still have trouble!
