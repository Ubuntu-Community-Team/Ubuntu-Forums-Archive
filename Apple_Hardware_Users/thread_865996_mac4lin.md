### Post by DJ_Peng on 2008-07-21
As it shows on page 6 of the [PDF documentation for Mac4Lin]("http://downloads.sourceforge.net/mac4lin/Mac4Lin_Documentation_2.pdf?modtime=1198008186&big_mirror=0"),

[LIST]
[*]Goto System > Preferences > Appearance. Click Install. Browse and choose the Mac4Lin Icon Theme .tar.gz file (wherever it is extracted).
[*]After the installation is confirmed click on 'Apply new theme'. If you need to make any changes/re-apply, click on Customize button and goto Icon page. Now select the Mac4Lin Icon theme to apply.
[/LIST]
The doc file includes screenshots if you need them and infra_red_dude is currently working on an update for Mac4Lin to bring it to version 1.0. You can get additional help from the [Mac4Lin topic]("http://ubuntuforums.org/showthread.php?t=555373") on these forums.
