yes you can install it.

You can run Ubuntu in a VM if you like. VirtualBox is free.

There is no risk when just booting the live CD. The user in the "horror story" you link to had a problem because he made the Ubuntu CD the boot volume. He didn't change it back to OSX. What he should have done was just reboot normally and hold C at bootup to boot from the CD. Even if the boot volume does get changed, you just have to hold the Option key on bootup to get a menu to choose a new boot volume.
