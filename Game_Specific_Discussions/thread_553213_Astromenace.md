USe the link above to find a ppa (personal package archive) which contains astromenace compiled for jaunty. You can add the sources to your /etc/apt/sources.list file and add the key by following links, I then drag and drop the text of the key (including -----BEGIN PGP PUBLIC KEY BLOCK-----
 and END...) to my desktop and add it through system:admin:software sources.
