### Post by quixote on 2010-04-26
You're using Ubuntu Netbook Remix?  (UNR)  I'm not familiar with a "Moblin Remix" so what I say may not apply.

Anyway, for future reference, the easiest way to do what you want is to use synaptic.  (I'm pretty sure it's installed by default in UNR.)  In synaptic, search for "firefox".  Many packages will come up, but choose the one labelled just "firefox" because that points to the most recent version.  It will also ask to install all the other programs (="packages") firefox depends on.  A firefox icon should appear in the Applications menu under Internet.

I thought firefox was installed by default on UNR.  I guess it isn't?

As for flash, also in synaptic, install "ubuntu-restricted extras".  That should pull in all the proprietary media-related stuff.  If you don't want it all, you just want flash, then install "flashplugin-installer" and "flashplugin-nonfree".  (The reason the user has to do this, rather than Canonical, is licensing issues from Adobe and the rest of them.)

I'd give firefox another try.  Nobody else has Adblock. :biggrin:
