### Post by Joe of loath on 2010-06-30
Although you want to start with Ubuntu, I highly suggest starting with Debian Stable. It's (as the name suggests) more stable, easier to customize and more lightweight. You wouldn't have to start with a memory-hogging Gnome GUI, you could start without a GUI and install a lighter one on top of it instead.

Yes, I know most of that can be done with Ubuntu, but it's too bloated for use as an appliance.
