### Post by snowpine on 2010-03-31
The only real difference is the user interface (netbook launcher vs. Gnome desktop). It's the same core system underneath (both are equally "Atom-optimized).

In previous versions of UNR, you could simply go to the System menu and choose Switch Desktop Mode to change to the classic Gnome Desktop. I think this feature was removed in 9.10, hopefully someone who has used 9.10 can tell you the new method. :)
