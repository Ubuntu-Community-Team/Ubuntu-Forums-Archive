### Post by aljoriz on 2010-01-03
UMR uses Moblin 2.0 which has a super fast boot time but is still relatively 'beta' so expect bugs here and there (no wifi/ethernet?).  It uses a social network type of gui where you can see tweets on the main page.

UNR 9.10 is very stable except for newly released netbooks, boot time is slower than UMR.  

Between the two I'd choose latter as it is more mature and community support is wider.

### Post by aljoriz on 2010-01-07
[How are UNR and UMR different?]("https://wiki.ubuntu.com/UbuntuOpenWeek/UbuntuMoblinRemix")
Ubuntu Netbook Remix was developed by Canonical well before Moblin. Canonical continues to develop and support UNR going forward. We are testing and supporting many hardware models out of the box with UNR and will continue to develop UNR past Karmic. UNR is quite popular, intuitive and well integrated in our platform.  
[LIST]
[*]UNR is well integrated with the Ubuntu platform (gnome, Ubuntu One, notifications, etc..) 
[*]UNR user interface is straight forward, simple to use 
[*]UNR has been shipping commercially for over 1 year, it's well tested and proven
[/LIST]
