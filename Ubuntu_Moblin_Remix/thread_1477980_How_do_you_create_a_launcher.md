### Post by cariboo on 2010-05-10
If you installed skype from a deb, you should have an icon in Internet. You can create a launcher by going to System->Main Menu.

Gnome is installed by default with Lucid UNE. You can select it from the login screen, just click your name and a menu bar at the bottom of the window will appear, you can select gnome there as well as a 2D desktop.
