### Post by Kixtosh on 2010-02-24
Actually, Bone, this is probably misleading. I have recently installed Ubuntu, and subsequently Xubuntu, onto two very old machines (see signature). Both of these were running W2K, for which extended support will terminate later this year (in July, IIRC). W2K was fine, initially, and mostly stable for my uncomplicated uses, but starting up becomes longer and longer as time goes by. So much so that it now requires almost five whole minutes to boot W2K completely on these machines, much the same as it does to boot Windows XP (Pro) completely on my relatively recent Toshiba Portégé R205 (R200 series), which is not very old in comparison (please note: ultra-portables always have far more limited specifications than standard laptops, so the Toshiba Portégé models are not speed demons by any measure).
 
Despite a huge difference in available resources, such as 256MB of RAM compared to 1.24GB or RAM, or a 400MHz Pentium II compared to 1.2GHz of Centrino Pentium M, both Ubuntu and Xubuntu boot up in roughly one third of the time required for XP on the newer Portégé R205. Usage is a different matter, nonetheless, since IE8 is still mostly faster and smoother than Firefox which I'm using on the older machines with Xubuntu, but overall, I would say that the system requirements of XP are totally misleading. W2K will run fine with 128MB or RAM, although 256MB is better, but XP doesn't even run properly with 1.24GB of RAM in some respects. A full 2GB of RAM should probably be considered the real life minimum for both XP and W7 (let's not even mention the "V" word).
 
Personally, I'm increasingly eager to install Xubuntu on the R205 fairly soon (probably Lucid Lynx 10.04 in May some time), including fingerprint reader support (if possible), to see how it stacks up. In the meantime, both Puppy Linux and Xubuntu perform very well (with some reservations) on machines with limited specifications when compared to XP in my opinion.

### Post by bone2006 on 2010-02-24
I've ran xubuntu, crunchbag, puppy linux and dsl.  I have to say I got the best performance out of crunchbag, but wasn't left with much of an OS.
