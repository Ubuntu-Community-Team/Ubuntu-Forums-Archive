### Post by snowpine on 2009-10-09
I am currently using Arch linux plus LXDE on my Dell Mini 9 netbook. It boots quickly and is stable; no complaints. Firefly seems kind of unnecessary; you can install Arch itself (set up exactly how you like it) quite easily using the Beginner's Guide:
[http://wiki.archlinux.org/index.php/Beginners_Guide](http://wiki.archlinux.org/index.php/Beginners_Guide)

In general I am wary of any project that's a "shortcut for installing Arch."
