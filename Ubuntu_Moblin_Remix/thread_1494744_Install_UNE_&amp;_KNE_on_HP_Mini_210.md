### Post by zoomy942 on 2010-05-27
you can install KDE and Gnome next to each other if you want.  problem is, KDE and Gnome have a lot of different dependencies so you'll end up with a lot of extra apps and stuff.  It gets messy.

What i would do is download the Kubuntu iso, make a live CD out of it and try it out.  IF it turns out you like it either wipe out gnome and install kde, or...  install them next to each other and pick which at boot.
