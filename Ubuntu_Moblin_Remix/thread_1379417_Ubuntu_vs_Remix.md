### Post by snowpine on 2010-01-16
A lot of misinformation in this thread. :)

Ubuntu Netbook Remix does **not** require an Atom processor (many netbooks such as the original Asus EEE do not have Atom processors) and its RAM requirements are identical to desktop Ubuntu, as you can see here:

[https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

As mentioned above, the only advantage to Netbook Remix is its special "launcher" user interface, which is optimized for viewing on small screens. I personally prefer a "regular" desktop on my netbooks, so I don't use the Netbook Remix myself (though I've certainly tried it).
