### Post by Sven6210 on 2010-01-06
I do agree with 'aljoriz'. I also tried the Ubuntu Moblin Remix and I consider it is still in a development phase. Therefore I would not use it in a productive environment yet.

It has some advantages such as the fast boot time. The new user interface is really new - nothing that I have seen before. Whether somebody likes it ir not is obviously a question of taste. I personally prefer the classical Ubuntu/Gnome user interface.

Ubuntu Netbook Remix based on 9.10 however is a very good product and does not need to hide behind Windows 7. I even convinced a friend who used Windows XP on his laptop to give UNR a try and after some days he quite liked it - even though he never used a Linux before in his life.

However, waht I need to admit, I needed to make some modifications in the system in order to make it run perfectly and supporting all hardware. Everything works now but it took me some time. I like spending this time as I see it as an opportunity to learn more about Linux, Ubuntu, Gnome, Alsa architecture etc. However, somebody who is not interested in these details, somebody who only wants to use a computer without spending time on hard- and software details should probably rather use Windows 7.

I personally will definitely not swap my Ubuntu Netbook Remix against any Windows. I like it too much, it's fast and it enables me to learn more about Ubuntu. And one day I might even use Ubuntu on my bigger machine which is currently still running under Windows XP.
