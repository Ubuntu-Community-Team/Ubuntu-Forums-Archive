### Post by qualtch on 2009-11-11
Hi and welcome to the forums!

There are various ways to do this:
( 1) Search option from the forums and Ubuntu.com website has a huge documentation and support site. ;) )

2) This link can be found from the [UNR download site.]("http://www.ubuntu.com/getubuntu/download-netbook") It's a nice and smooth way to install Ubuntu from USB stick and works the same way for Moblin Remix [https://help.ubuntu.com/community/Installation/FromImgFiles]("https://help.ubuntu.com/community/Installation/FromImgFiles")

3) You can also try using [UNetbootin]("http://unetbootin.sourceforge.net/") instead of the Win32 Disk Imager to install Ubuntu to your USB stick. The instructions are the same. It has an easy GUI to do it too.

Happy Ubuntuing! :)

### Post by Jackyboy86 on 2009-11-11
As Moblin remix is in the beta stages, i've got one bit of advice for you, as a newbie.
Don't bother.
Go get Ubuntu Netbook Remix (i'd suggest Jaunty Jackalope - there shouldn't be compatibility issues on the Lenovo S10, whereas theres currently hell on earth with the Graphics drivers in Karmic) and wallow in the glorious speed, the (relatively) stable platform, the sweet, beautiful interface, and wait the 6 months or so until all the moblin bugs are cleared up.
I've been Ubuntuing for years and been a complete geek for as long as I remember, and I still ended up giving up on Moblin.
On the Lenovo S10, it appears that the wireless isn't working out of the box yet, among other issues.
Get off windows (if thats what you're on), but head to something stable, and suitable for a noob.

Probably NOT what you wanted to hear (I hate the fact I can't get UMR to run properly) but its probably the best course.
