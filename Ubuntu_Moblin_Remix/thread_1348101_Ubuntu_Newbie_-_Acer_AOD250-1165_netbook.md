### Post by FerociousTwig on 2009-12-26
I just received a slightly newer version of your netbook. I have the AOD250-1950 or something like that. I am currently dual booting off of the 160 GB hard drive xp and Ubuntu 9.10 netbook remix. Everything is extremely smooth except for the internal microphone not working (though the mic jack works so if you plug in an external microphone you're golden) and the wifi needed a few packages to work.

I have not used the memory slot at all yet but I have an 8GB SD card I will try shortly. As for booting off of the memory slot, I have no idea. My initial guess would be no because that slot probably needs drivers to load to use it which are contained in the kernel... but I really dont know. Read up on the SDIO interface because if it's possible thats probably what you will be using.

### Post by aljoriz on 2010-01-01
To answer the thread starters post point by point

1.  Ubuntu Netbook Remix / Ubuntu has a live install feature.  It basically allows you to test ubuntu WITHOUT touchng anything on the hard disk.

2.  Most netbook support the following for boot:  Micro SD but be sure to go to the bios and change the boot order OR you can read the manual of your netbook on HOW to change the boot loader to USB by pressing a certain key.

3.  Windows 7 will run on recent netbooks provided it has the minimum requirement for 32bit 1 gig ram.  Why buy when you can go Open Source?  don't be afraid to get yourself wet by swimming.
