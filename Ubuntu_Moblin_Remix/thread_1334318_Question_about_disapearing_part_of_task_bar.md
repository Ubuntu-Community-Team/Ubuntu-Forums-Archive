### Post by pvfloripa on 2009-12-17
I figured out myself as well :)

For others who might have the same issue:

- Right-click on panel
- Choose "Add to Panel"
- Choose "Notification Area"

This should include (maybe you need to set it up) Skype, wireless network icon, volume control, etc.
