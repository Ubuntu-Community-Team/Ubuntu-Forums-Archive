### Post by tphive on 2010-11-07
I really must point out that the netbook versions of Ubuntu have NOT discontinued, but rather have been revamped under the name Ubuntu Netbook Edition (UNE). And is available from the main site. After having used UNE 10.10 however, I suggest against it in favor of UNE 10.04, mainly for a FAR superior sidebar. It, along with everything else can be found at this mirror server here: 

[http://ubuntu.media.mit.edu/ubuntu-releases/](http://ubuntu.media.mit.edu/ubuntu-releases/)

The folder '10.04.1' should have what you're looking for.

Depending on what the limiting factor is of the netbooks in question, there are many tweaks that can be made to optimize it's function. If you're using slow early Solid State Drives, there are several tweaks that can be done, things like vm.swappiness=10, noatime, turning journaling off, etc. If however you're using normal SATA platter drives, the distro alone might suit your fancy. You are even able to boot to the normal Ubuntu Desktop Edition if you so choose, with just the UNE install.
