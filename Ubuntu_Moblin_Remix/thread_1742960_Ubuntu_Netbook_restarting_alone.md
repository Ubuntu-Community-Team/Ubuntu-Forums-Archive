How have you got the dual boot setup, i.e. WUBI or from boot.

I had a similar problem when I used 10.10 so I had to either install ubuntu as a seperate system so no dual boot or through WUBI as a windows app. 10.04 and 10.10 do not like windows updates which sit in the system partition you know the 100 mb partition, as one its an NTFS partition and it is set to system for default which ubuntu and any other linux refer due to the partition actively running under a system configuration. 

ps check the netbook's vendors site for drivers and etc as the vendors will generally produce updates for the drivers and introduce compatibility for Linux.
