### Post by maybeway36 on 2007-12-22
The FreeNAS distro (FreeBSD-based) is shaping up pretty nice. I tries it out on an old computer of mine, and I could do pretty much everything I needed - format a disk with UFS, mount it, set up CIFS and SSH - via a Web browser.
My only complaint is that it refuses to install on my hard drive, but that will probably be fixed pretty soon. Fortunately, you can save your configuration on a DOS-formatted floppy.
This is a really cool idea for a distro, and it works really well. :)
