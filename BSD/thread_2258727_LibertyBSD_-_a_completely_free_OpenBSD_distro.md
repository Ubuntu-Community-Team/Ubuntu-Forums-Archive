### Post by Riley_Baird on 2014-12-30
[center][img]http://www.libertybsd.net/LibertyLogo2.svg[/img][/center]

OpenBSD is probably the most secure OS out there. However, OpenBSD contains firmware that does not come with source code. Furthermore, on first boot, many more non-free firmware files are downloaded.

Because of this, I decided to make an OpenBSD distro with these non-free bits removed. I call it LibertyBSD.

However, LibertyBSD will not be released until 3 BTC have been raised. You can help us get to this total by either making a donation or purchasing a pre-release CD.

For more information, see [http://www.libertybsd.net](http://www.libertybsd.net)

If you can't make a financial contribution, but still want to help, please upvote and comment on the Slashdot article so that it gets accepted: [http://slashdot.org/submission/4088331/openbsd-forked-to-remove-non-free-firmware](http://slashdot.org/submission/4088331/openbsd-forked-to-remove-non-free-firmware)
