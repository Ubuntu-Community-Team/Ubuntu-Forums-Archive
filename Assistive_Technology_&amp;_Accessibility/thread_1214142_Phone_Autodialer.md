### Post by schmidtbag on 2009-07-15
I created a shell script with Zenity and dtmfdial that will automatically dial a number on a phone.  All you have to do is type the phone number and put a phone up to the computer's speaker and press OK.  The program also has a contact list and has area code settings.

I don't think I programmed this to work with 1-800 numbers.  The help file included will tell you how to run this program without having an open terminal.
