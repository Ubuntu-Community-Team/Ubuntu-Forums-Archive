### Post by hareevs on 2010-03-03
Hi,
I created this little application which I would like to share with you. It allows you to run application by pressing buttons on your joystick or by moving the axis. It uses simple configuration file and can handle multiple joysticks.

It's written in Python and depends on python-pygame, but it should be installed automatically, if you use .deb package for installing.

You can download it from here: [COLOR=Blue][https://sourceforge.net/projects/jcont/](https://sourceforge.net/projects/jcont/)[/COLOR]

More information about usage can be found on it's man page.

I know that you can find some similar applications on the internet, but i wrote this application mainly for my personal use and I think that you can find it useful.

This tool is still in testing phase, so please e-mail me if you'll find a bug.
