### Post by BLTicklemonster on 2006-09-18
In my opinion:

I think it would of great value to have the live cd's capable of detecting and mounting any and all drives and partions. I say this because first of all, if the live cd could do it, then obviously Ubuntu would :) secondly because there are perhaps many times when a person may need to use something like *Ultimate Boot CD* but not actually have it, but they have a(n) ubuntu live cd sitting around. Let's say I have a hd that I can't boot to, and I want to load ubuntu on it, and at the last second, I begin to wonder if this is the hd I had lost a while back that had the wedding pictures with Elvis hanging out of a UFO? wouldn't it be nice if I could have a last second look see? 

I looked around for a bit for somewhere to post this, and this looked like as good a place as any.
