**What is eViacam?**

Enable Viacam (eViacam) is a mouse replacement software that moves the pointer as you move your head. It works on standard PCs equipped with a web camera.

**What's new for eViacam v1.7.2?**

This version fixes click generation in multi-monitor configurations, improves support for applications that do not recognise too fast click events, improves the configuration wizard and adds Lithuanian translation. See NEWS file for details.
  
**Where can I get it?**

Sources and Win32 binaries can be downloaded from:
[http://viacam.org](http://viacam.org)

To install on **Ubuntu** open a terminal and run:

```
sudo add-apt-repository ppa:cesar-crea-si/eviacam
sudo apt-get update
sudo apt-get install eviacam
```

Git repository:
[git://eviacam.git.sourceforge.net/gitroot/eviacam/eviacam]("git://eviacam.git.sourceforge.net/gitroot/eviacam/eviacam")
