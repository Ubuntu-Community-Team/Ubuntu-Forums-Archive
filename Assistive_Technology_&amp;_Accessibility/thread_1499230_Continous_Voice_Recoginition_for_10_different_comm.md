### Post by rao.nischal on 2011-07-30
You can try VEDICS.

With VEDICS, you can run any application installed in the system using the command "run" followed by the name if the application. 

It can also let you access anything visible on the screen by speaking out its name. It also lets you do window operations like close, minimize, maximize etc.

You can download VEDICS from [http://vedics.sourceforge.net](http://vedics.sourceforge.net).

Let me know if you face any problem.
