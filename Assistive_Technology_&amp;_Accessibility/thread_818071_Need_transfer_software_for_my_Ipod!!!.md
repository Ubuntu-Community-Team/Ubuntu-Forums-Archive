### Post by posburn on 2008-06-04
I personally like to use [gtkpod]("http://www.gtkpod.org/about.html") to handle my iPod; it's in the repositories, though I'm not sure that's the latest version.

It will do neat stuff like removing duplicate tracks, volume normalizing (I don't like to use my iPod's built-in Sound Check),playlist creation, etc.

Hope that helps!
