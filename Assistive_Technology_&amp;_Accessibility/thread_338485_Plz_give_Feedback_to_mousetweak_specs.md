### Post by frafu on 2007-01-14
Hello, 

I would like to let you know that I finished the first draft of the mousetweaks specification. For the moment, it contains 4 mousetweaks: 

1: accessing the contextual menu by a click&hold of the left mouse button (usually it is done with a right click)

2: dwell clicking: i. e.: software to do the various mouse clicks without the user having to press any button 

3: tremor reduction: smoothing the pointer movement of people that have a trembling hand

4: mouse gesture: triggering commands (actions) on the computer by doing determined gestures with the pointer

Any help, suggestion, remark, correction, addition that can be done to the mousetweaks specification are welcome. 

Have a nice day. 

frafu
