### Post by mister harbies on 2009-04-19
Hi there,

I am not really sure what you mean by software. However there are websites that will be able to help, in some way, like a dictionary. But this is not the same as learning from an actual tutor because so much is involved with learning Sign Languages i.e. grammar, facial expressions, direction of sign, etc.

It's not a simple transfer from one language to the other because Sign Language is not a written language like English. It is visual, so the rules are different. It is hard for something like software to be able to do. So many variables.

Perhaps you would like to clarify what you mean.

Mister Harbies
