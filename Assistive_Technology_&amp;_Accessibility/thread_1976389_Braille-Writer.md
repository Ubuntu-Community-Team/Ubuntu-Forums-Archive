**[COLOR=Lime][B]Sharada-Braille-Writer. **[/COLOR][/B]

Sharada braille  writer is a six key approach to producing print materials.  letters f,  d, s, j, k, l represent 1 2 3 4 5 6 of the braille dots respectively. By  pressing f and s together will produce letter k and like.  
**[COLOR=DarkGreen]How to install[/COLOR]**

Download the deb file from hear [http://code.google.com/p/sharada-braille-writer/](http://code.google.com/p/sharada-braille-writer/)
open it and install   
**[COLOR=Red]How to start using Sharada-Braille-Writer[/COLOR]**

**[COLOR=Navy]English[/COLOR]**

English  is the mode by default. braille contractions also can be managed with  this utility. In English, one can write contractions by pressing letter a  after writing the contracted letters. for example, press a after typing  c and it will write can and cd could etc. To make a Uppercase letter  Press "g" before typing the letter.  :guitar:
**[COLOR=Purple]Malayalam and Hindi[/COLOR]**

To select Malayalam alt+m to select Hindi alt+h for numerals alt+n and English alt+e. 
**Punctuations**

To mark any punctuation one should press semicolon first. example ;+dkl will produce "." 
**Navigation**

Ctrl+home  ctrl+end will help to move to the top and end of the file. Arrow keys  will help to navigate through text line by line and letter by letter.  ctrl+right arrow will help to navigate between words. r will start  reading to the end, and t will stop reading. One can change to an  ordinary typing mode by pressing 2 and go back to braille mode by  pressing 1 again.
