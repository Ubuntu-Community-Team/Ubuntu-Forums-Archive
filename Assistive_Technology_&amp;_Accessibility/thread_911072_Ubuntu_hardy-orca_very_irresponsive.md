### Post by JohnCarcinogen on 2009-03-24
Hi there,

I'm a little new to Linux and very new to Orca, but I was having some troubles myself with the app, so I did some searching and came up empty.

Fortunately, I did the next best thing - process of elimination - and found out that my firewall (Firestarter) was causing a lot of conflicts.

Try it out; if you use a firewall app, start it up, then start up Orca - run as usual. Then, kill the firewall app, and check on Orca to see if it improves.

Otherwise, you can always look at what else you're running to see if it's a problem.

Hope this helps!
