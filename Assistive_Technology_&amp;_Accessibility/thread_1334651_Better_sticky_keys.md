### Post by IAmNoodle on 2009-11-22
One of the few things that is better in Windows (I'm pretty much a convert) is Sticky keys. In Windows, it's easy to activate, use and keep track of, thanks to the icon in the system tray, with (for the benefit of those who don't know about it) the layout set like how the ctrl, alt, windows key and shift keys are on the keyboard. If you press shift, the icon would indicate you've pressed the shift key, for example. It means you can check if you press it, pause and forget if you've already pressed it. The feature in Ubuntu is far less intuitive. Since there's no icon to indicate what you've pressed or whether you've got it locked (press a key twice in a row), so you could find yourself in a small mess.

Is there an option I've missed that brings an icon up in, say, the top panel, so I can use and keep track of Sticky keys, or is this a feature that's yet to be written into the program?
