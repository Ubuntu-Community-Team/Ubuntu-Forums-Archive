Hello!

At the last meeting we decided to create a new mailing list focused on accessibility development. The existing list will continue as before but will be used for more general discussion and support questions.

This should make the existing list less intimidating to new posters by moving some of the more technical discussions, while providing a place where we can have long development debates  ;) 

You can sign up for the new list here: 
[https://lists.ubuntu.com/mailman/listinfo/ubuntu-accessibility-devel](https://lists.ubuntu.com/mailman/listinfo/ubuntu-accessibility-devel)

Bug mail will now be directed at the -devel list.

Sign up and join the development talks for Edgy+1!
