### Post by Bölvaður on 2008-03-16
It shouldn't be installed by default in the xfe version of ubuntu.
So you need to open up Synaptic Package Manager and download 'gnome-orca' then either type 'gnome-orca' in terminal to use it or enable it as a shortcut in the menu.

If you want it to start up everytime the computer does you should look for Sessions under Preferences in the menu and add 'gnome-orca'.

I dunno how the program works but that's the program you are talking about I think
