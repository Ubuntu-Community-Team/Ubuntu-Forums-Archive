### Post by earthpigg on 2011-04-25
looking at the [windows xp directions]("http://www.sos.mo.gov/wolfner/BARDinstructions.asp").

notice that she says to move the files inside the .zip to the device, not the .zip file itself.

it looks like you should be moving the files *extracted* from the .zip to the device, not the .zip file itself.

also:

> I have cleared everything off of the cartridge.

after deleting things, did you then empty the trash? 

space is not *actually* freed up until you 'empty trash'. all "deleting" something in Ubuntu actually does is move it to a folder on the device called ".Trash". Emptying the trash is when the actual deleting & freeing up space occurs. 

(It is the same on Windows and OS X, btw. Windows has a hidden folder called "recycled" instead of one called ".Trash"...)

### Post by startgame412 on 2011-07-03
You need to unzip the book and transfer it into the top level directory or the root directory of the cartridge. I have this service as well. What you have to do is right click the book in the zipped archive and right click where it says extract. It should extract in a folder of the same file name as the zip file. This is the folder that you put on the root or top directory of the cartridge. Hope this helps.
