### Post by cmauri on 2012-06-07
**What is eViacam?**

Enable Viacam (aka eViacam ) is a mouse replacement software that moves the pointer as you move your head. It works on standard PCs equipped with a web camera.

**What's new for eViacam v1.5.4 ?**

This release adds new translations (Russian and Portuguese) and fixes minor issues. See file NEWS for details.

**Where can I get it ?**

Sources and Win32 binaries can be downloaded from:
[http://viacam.org]("http://viacam.org")

To _install on Ubuntu_ open a terminal and run:
sudo add-apt-repository ppa:cesar-crea-si/eviacam
sudo apt-get update
sudo apt-get install eviacam

Git repository:
git://eviacam.git.sourceforge.net/gitroot/eviacam/eviacam
