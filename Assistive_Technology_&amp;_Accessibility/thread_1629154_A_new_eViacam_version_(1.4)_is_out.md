### Post by cmauri on 2010-11-23
Hello,

Just to say that recently we've published the new version 1.4 of Enable Viacam which includes important improvements for linux platforms. Specifically:

- New capture layer for linux platforms and camera controls dialogue. This new layer greatly improves the camera capture on linux platforms providing much better camera support and a configuration dialogue.
- Added configuration wizard
- Added documentation
- Added a new assisted pointer speed calibration method
- Added new gesture based click generation method (linux only)
- Added visual feedback during pointer actions (linux only)

Visit [http://viacam.org](http://viacam.org) for more information and downloads. Packages for several Ubuntu versions are also provided.

Enjoy!
