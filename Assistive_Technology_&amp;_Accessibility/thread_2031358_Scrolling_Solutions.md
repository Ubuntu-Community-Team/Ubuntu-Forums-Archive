### Post by MarjaE on 2012-07-21
I'd like to know what methods people prefer for scrolling:

1. Old-style scrollbars. I made them somewhat wider. I find these perfect for the mouse, but not so easy to use with the joystick.

2. Select the window and use the arrow keys.

3. Select the window and use some kind of scrollwheel. I used to use a scrollwheel, but it gave me severe hand pain.

4. Select the window and use special touchpad functions. Unfortunately, this requires a working touchpad with good drivers, which doesn't go haywire every time one's hand passes within a couple cm of the pad. this also requires a lot of dexterity.

Anything else? I'd like to see an easy option for joystick users.
