Small sample size here, just two 2.5" leftover laptop drives one 40G and the other 100G in Tiger Direct cases. One's IDE, the other SATA.  Both running oneiric ocelot and precise pangolin O.K.  on my recent desktop, notebook, and netbook.  My experience, if the pc will boot USB, the USB hard drives run with quite reasonable response.

BIG reason for the USB had drives is hopefully a buggy ubuntu ubiquity install will clobber the USB drive and not the main hard drive.  I have had one instance of installing oneiric ocelot alpha onto the USB hard drive clobbering the main 250G hard drive on the notebook, requiring a 4 DVD restore of Windoze...7 (my wife supports a couple web sites which are built with proprietary $$ software not ported to linux yet, if ever).

Another advantage is ubuntu image greater than 700 MB.  Put the USB hard drive on the tower which can handle huge iso's, install onto the USB hard drive then move the hard drive to the older and/or smaller pc.  Do note that grub gets confused a bit switching to different pc with different other hard drives but I've been able to handle that.

Options and choices.
