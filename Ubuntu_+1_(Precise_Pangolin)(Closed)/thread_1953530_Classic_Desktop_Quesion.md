### Post by keithpeter on 2012-04-06
Hello Quarkrad

The way I added launchers was to use window-alt and right-click 'add to panel'

Re-ordering launchers does not appear to work very well, so I just delete them (window-alt rightclick 'remove from panel') and add them in the order I wanted them

If anyone has a better method, I'd love to find out. Below is what I'm using on a single bottom panel in Gnome Classic
