### Post by grahammechanical on 2012-04-24
What kind of a keyboard do you have? On my keyboard there are 10 number keys. That is what limits the numbering of the icons in the launcher.

The idea is that we hold down the super key and then press a number key to launch the application that has that number.

If it was standard for keyboards to have keys numbered 11, 12, 13 and so on then what you are asking about might be possible. But there has not been any work done in 12.04 so that we can hold down the super key and press two number keys consecutively to launch icon numbered 11, 12, 13, and so on.

Of course, you could always install 12.04 and experiment and find out what is possible and what is not.
