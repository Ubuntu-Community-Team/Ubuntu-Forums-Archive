### Post by sloebs on 2012-04-02
Beta versions can be frustrating, but expected. 


[LIST=1]
[*]Grab the latest stuff until the bug squashers get to it:

```

sudo apt-add-repository ppa:ubuntu-audio-dev/alsa-daily
sudo apt-get update
sudo apt-get install alsa-hda-dkms

```
[*]Reboot.
[*]Go to *System Settings > Sound* and adjust settings, test speakers.
[/LIST]
