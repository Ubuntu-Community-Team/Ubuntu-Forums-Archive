### Post by 3rdalbum on 2012-01-25
Dear OP, Gnome 2 is dead and it's not coming back. In case you've been living under a rock for the last few months, you can install gnome-session-fallback to get an old-skool Gnome Panel session.

I'm curious about how Gnome Classic in Natty was "too different" for you to grasp. It's basically 10.10's Gnome 2, plus the global menu and overlay scrollbars - both of which are easily turned off. If these small modifications are too difficult, I'm afraid you might find it impossible to use computers after October this year. Maybe you should try working with Unity for long enough to get used to it? About a week should do the trick.

Computing is an industry which moves quickly and always has a lot of things changing. Ubuntu is a very fast-paced distribution. If you can't deal with things changing, then perhaps Ubuntu is not the correct distribution for you. You could try moving to Debian or another slow-paced distribution - but be aware that eventually the changes in the Linux desktop will filter through to even the slow-paced distros.

In short, you can buy some time, but you'll eventually have to adapt. Why not try now? Note that "moving back to Windows" is not an option either, as Windows 8 will have an entirely new, non-legacy desktop that you'll probably hate more than Unity :-)
