Hi,

My set up is:
2012 MacBook Pro i7 w/ 8GB RAM - running VMWare Fusion (4.1.1)


I have a vm image that runs Ubuntu 11.10 64-bit perfectly. I have since run the updater to bring it up to 12.04, which runs well, save for an irritating issue with "gstream" after the upgrade, where I cannot watch a streaming H.264 file in a browser.

So, I decided to create another VM with the same configuration options (4GB ram/ 3D = "ON"/ 40GB HDD space) and have downloaded 12.04 LTS beta 1 AMD64 ISO (the documentation leeds me to believe the x86 iso is 32bit only) 

The installation displays correctly, however, I experience a "black screen" in post-install after system restart. I have download the ISO from an alternate mirror and get the same result.

Is anyone else having a similar issue?

You should **not ever** use the "Easy Install" option unless the version of VMware that you are using supports the exact version of the client that you are trying to install.

"Easy Install" uses specific settings for the versions it supports and assumes that **you** know what you are doing when **you** select it. In the past it also pre-configured the Ubuntu client VMs in North America which was a real pain for those of use who required the same locale as the host system.

VMware will not fully support 12.04 for many months after official release if the previous releases of Ubuntu are any guide at all.
