### Post by Triality on 2012-04-01
Hi,
in the current beta 2 release, there is a problem with gnome-fallback (gnome-panel) using compiz/effects and minimizing windows.  I have 2 monitors, with each monitor having one gnome-panel showing the open windows for each display respectively. 

However, once I minimize any window on the right screen, the panel entry for that window jumps to the left window. If I minimize a window on the left monitor, it stays there. I have also observed this effect with compiz on Xubuntu 12.04 beta 2. Gnome-fallback without effects works as you would expect.  
EDIT: once you restore/maximize the window, it jumps to the right location on the right monitor.

Additionally, the UI is not consistent using gnome-fallback with effects. But I believe this was already noted here: [https://bugs.launchpad.net/ubuntu/+source/light-themes/+bug/933305](https://bugs.launchpad.net/ubuntu/+source/light-themes/+bug/933305) (Post #6)
