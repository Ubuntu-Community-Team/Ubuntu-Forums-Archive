### Post by neu5eeCh on 2012-03-18
> **livewire94 said:**
> I wanted to format and reinstall Ubuntu on my PC.  Was going to use Ubuntu 11.10 but since 12.04 LTS will be out in a couple months, I don't know if I should just install the beta.  

If I install Ubuntu 12.04 Beta 1 now.  Will I be able to update it to Beta 2 and RC and final LTS without formatting and reinstalling from the newer ISO's?

My advice? Do a fresh install of 12.04 Beta. Once it's installed:

sudo apt-get update

Then

sudo apt-get upgade

Once that's all done, you will have the latest version of 12.04. Just think of it as a rolling release until April comes around - then it becomes 12.04 LTS (and stays that way with regular maintenance updates); at which time Canonical starts working on **12.10 (Quantum Quokka)**.
