### Post by zombifier25 on 2012-04-02
You can disable the 'chameleonic' approach and apply a color yourself using CCSM or Ubuntu Tweak.
Install ccsm:
```
sudo apt-get install compizconfig-settings-manager
```Then open CompizConfig Settings Manager, choose Unity Plugin > Experimental > Background Color
