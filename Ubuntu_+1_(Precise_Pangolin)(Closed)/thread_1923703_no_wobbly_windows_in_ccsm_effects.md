You can try to reinstall compiz plugins.
```
sudo apt-get install --reinstall compiz-plugins compiz-plugins-default compiz-plugins-extra compiz-plugins-main compiz-plugins-main-default 

```It's there. Unless you compiled compiz from git and it failed to get sources and compile the plugins. Or if you're compiz/plugins from an alternative PPA.
