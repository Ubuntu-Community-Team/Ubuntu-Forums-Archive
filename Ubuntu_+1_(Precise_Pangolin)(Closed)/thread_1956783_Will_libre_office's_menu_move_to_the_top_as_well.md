### Post by ubuntu27 on 2012-04-11
Some day it will. Right now lo-menubar is ridden with bugs so it is not installed by default. Perhaps by 12.10? or 13.xx

If you want to test and report bugs, you are free to install it. :)


```
sudo apt-get install lo-menubar
```



To report a bug for Libreoffice's Global menu: lo-menubar, just press **ALT+F2**, and then type

```
ubuntu-bug lo-menubar
```

It will collect the necessary info and upload it to Launchpad. Remember to write what you were doing step by step, what happened, and what you were expecting to happen.


Good luck!
