### Post by mc4man on 2012-03-30
> I have found a way to remove the applications from dash. Press 'Alt+F2' to open the run dialogue. Type in 'gksu nautilus ...

If you don't want to see in Dash then just open the app's .desktop & add line I mentioned above.
Ex. (indicator-session-logout.desktop
```
[Desktop Entry]
Name=Log Out
TryExec=/usr/lib/indicator-session/gtk-logout-helper
Exec=/usr/lib/indicator-session/gtk-logout-helper --logout
Icon=system-log-out
Terminal=false
Type=Application
OnlyShowIn=Unity;
[COLOR="Blue"]NoDisplay=true[/COLOR]
Categories=System;
Version=1.0
X-Ubuntu-Gettext-Domain=indicator-session
```

Edit: while most that should be visible that weren't have been fixed if something you wish to see isn't then remove or edit that line - 
Ex. (gksu.desktop
```
[Desktop Entry]
Name=Root Terminal
Comment=Opens a terminal as the root user, using gksu to ask for the password
Encoding=UTF-8
Exec=gksu /usr/bin/x-terminal-emulator
Icon=gksu-root-terminal
StartupNotify=true
Terminal=false
X-MultipleArgs=false
Type=Application
Categories=GTK;Utility;TerminalEmulator;
[COLOR="Red"]NoDisplay=true[/COLOR]
X-Ubuntu-Gettext-Domain=gksu
```
