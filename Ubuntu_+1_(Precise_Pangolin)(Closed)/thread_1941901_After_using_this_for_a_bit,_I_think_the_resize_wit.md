### Post by kaldor on 2012-03-16
It's a really bad UI element, in my opinion. The overlay scrollbar is intended for scrolling- why does it need to resize the width of a window? I've accidentally resized my window many times when I just tried to scroll. There's no reason for it to exist, since I can just resize the window along the edges anyway. Is there any rationale for this?

It feels like a "just because it's cool" feature. Am I the odd one out in this, or do people agree? Might file a bug report.
