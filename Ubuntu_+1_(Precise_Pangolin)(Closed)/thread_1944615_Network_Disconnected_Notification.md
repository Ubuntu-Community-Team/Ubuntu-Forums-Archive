If anyone using Gnome Shell is having problems with the "Network Disconnected" notification not disappearing after coming out of Suspend mode, please go add to [this bug](https://bugs.launchpad.net/ubuntu/+source/gnome-shell/+bug/958492) so they will do something about it.

My experience is that the notification will disappear after the network is connected again, but if you ever click the button to never show it again, from then on it keeps showing it and doesn't disappear until you click on it.
