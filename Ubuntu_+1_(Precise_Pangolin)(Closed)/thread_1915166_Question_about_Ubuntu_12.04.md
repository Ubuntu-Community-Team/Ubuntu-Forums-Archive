### Post by grahammechanical on 2012-01-25
Surely the default desktop after log-in is the last used desktop. So, if we shut down from Unity 3D then we will log in into 3D unless we change it to something else. 

I am not using 12.04 at the moment and I only have Unity 3D and 2D installed but I am sure that the desktop I get is the last used desktop. It must surely work this way with Gnome shell or Gnome fall-back-mode.

I can tell you this that with 12.04 the option to change desktops is hidden. You have to click on the user name for the cog icon to appear on the password panel.

I have two users set up. One where the last used desktop is 3D. The other where the last used desktop is 2D. I am sure that I can used this to switch between 3D and 2D by just changing user. But I cannot test this out because I messed something up on my 12.04 install and I need to re-install.
