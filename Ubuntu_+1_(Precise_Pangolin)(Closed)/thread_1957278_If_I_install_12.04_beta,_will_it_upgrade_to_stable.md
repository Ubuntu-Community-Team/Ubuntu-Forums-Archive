### Post by Mark Phelps on 2012-04-12
Understand, that there is no capability to roll-back a version update, so if you encounter problems after updating, you're stuck!

And given that this is still in Beta testing, problems are likely.

You're better off waiting for the Release version, unless you want to spend your time troubleshooting problems.
