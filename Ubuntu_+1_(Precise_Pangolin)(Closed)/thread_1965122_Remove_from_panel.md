### Post by kansasnoob on 2012-04-25
> **megabuffen said:**
> I'm not using Unity.

If you're using 'gnome-panel' in a Gnome classic (no effects) session try pressing the Alt key and then right-clicking on the applet you wish to remove.

If you're using 'gnome-panel' in a  standard Gnome classic session try pressing both the Alt & Super keys at the same time then right-clicking on the applet you wish to remove.

The difference between "Gnome classic" and "Gnome classic (no effects)" is that the prior uses Compiz and the latter uses Metacity. I've had a lot of trouble with Compiz so I've focused on Metacity:

[http://ubuntuforums.org/showpost.php?p=11835822&postcount=389](http://ubuntuforums.org/showpost.php?p=11835822&postcount=389)

I did the same in Oneiric:

[http://ubuntuforums.org/showthread.php?t=1886799](http://ubuntuforums.org/showthread.php?t=1886799)

Note: The Super key is typically the one with the Windows logo.
