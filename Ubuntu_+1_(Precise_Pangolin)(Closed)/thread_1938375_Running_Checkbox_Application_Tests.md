### Post by grahammechanical on 2012-03-09
Hi

I, for almost two weeks, I have been trying off and on to run these Checkbox application testing suite as we were asked to do.

[http://testcases.qa.ubuntu.com/RunningCheckboxTestCases](http://testcases.qa.ubuntu.com/RunningCheckboxTestCases)

I just worked out why Checkbox failed to run the tests. It needs a basic unmodified beta install to work.

I am using a 12.04 that has been upgraded daily to Beta 1 and on it this suite of Checkbox Application Tests would go from the page where we select which tests to run to the page where the report  is submitted without running any of the tests.

I have just installed, after a bit of a struggle, 12.04 Beta 1 on a new partition and now I can run this suite of Checkbox Application Tests.

So, I would say that if anyone is serious, really serious about testing Ubuntu+1 that they do so on a fresh, basic, default install. I shall from now on.

Regards.
