I've noticed that there is not an option in Empathy preferences to have the messenger open on login. After looking at the startup applications and hearing about the new GNOME Login Sound disabled by default, I thought it might be a good idea for Ubuntu to have several other applications listed (but disabled) by default.

[IMG]http://img205.imageshack.us/img205/1935/startupapplications.png[/IMG]

For instance (all disabled by default of course):
[LIST]
[*]Empathy
[*]Gwibber
[*]Mail
[*]etc.
[/LIST]

While its possible to add startup applications on your own, it is not very user friendly for new/novice users. This also seems like an easy and consistent approach since the login sound is listed already.
