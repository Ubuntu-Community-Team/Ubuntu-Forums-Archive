### Post by dino99 on 2012-03-24
a few things:
- the kernel directly deals with X, so no needs it to disturb and confuse it.
- dont use "vanilla" driver, but the ones found in ubuntu archive or xorg-edgers ppa
- remove xorg.conf : sudo rm /etc/X11/xorg.conf

so to be concrete, from synaptic:

- purge everything related to : unity, compiz, jockey, nvidia
- add xorg-edgers ppa and update ([https://launchpad.net/~xorg-edgers/+archive/ppa](https://launchpad.net/~xorg-edgers/+archive/ppa))
- install nvidia-current 295.33 and xserver-xorg 1.12 (will be automatically done) and nvidia-settings (not the other proposed packages)
- install unity compiz jockey
- close synaptic and logout then reboot
- check that the nvidia-current driver is used: sudo jockey-gtk
- use nvidia-settings if needed (you should not)
