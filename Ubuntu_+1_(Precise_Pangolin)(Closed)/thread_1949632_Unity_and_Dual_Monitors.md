### Post by ubuntu27 on 2012-03-30
Sorry if I did not use the correct terminology. I am currently using my desktop in Japanese.

It is either called Display Settings or Monitor Settings.


Open the System Settings and click on "Display" (or maybe Monitor). It has an icon of a monitor with a Ruler.

If the window that appears let's you modify the display resolution, then you are at the right place. There should be an option to select ON/OFF for **"Sticky Edges"** which is bellow the "Launcher Placement"
