### Post by bcbc on 2012-04-10
If you're testing the 12.04 beta you should be prepared for failure, also file a bug if it fails, including your upgrade logs.

You should always backup your install prior to upgrading (\ubuntu\disks directory) so you can restore if something goes wrong. Also post in the Ubuntu+1 forum.

Since you just installed fresh, you might as well do a fresh 12.04 install when you're ready, but if you have a custom setup you don't want to lose, then migrating is convenient.

I don't know why your cursor jumps around unless you're hitting the touchpad while you type. There is a setting to switch off the mousepad while typing (although I believe it is known to cause issues on some computers). If the mouse cursor is just moving without stealing focus, then I don't know - haven't heard of a problem like that.
