### Post by shakabra on 2012-04-09
The extra columns can be done in nautilus preferences. 

In order to turn off thumbnail previews you could set the "Only smaller than" switch in nautilus options. This wouldn't allow for thumbnail previews in icon view.

### Post by fantab on 2012-04-09
Yes, it is possible to disable thumbnails in nautilus and it will disable thumbnail previews for all files and all views (if that is what you are looking for).

[LIST=1]
[*]Open Home Folder [nautilus]
[*]Edit - Preferences - Preview - Other Previewable Files - Show thumbnails - 
[*]Change the setting to NEVER. [No more thumbnails for all types of files].
[/LIST]
