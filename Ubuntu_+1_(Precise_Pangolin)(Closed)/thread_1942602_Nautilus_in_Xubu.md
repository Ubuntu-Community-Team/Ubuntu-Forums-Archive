### Post by ubuntu27 on 2012-03-18
I also use PCManFM in Xubuntu. Quite the fast beast, eh?

You can uninstall Nautilus, there should be no problem. (Are you still using UbuntuOne?)

The removal of Thunar should not pose a problem directly. Since you are using a Ubuntu Precise that is still in development, perhaps it might be a good idea to leave it alone. (Xubuntu-desktop "depends" on Thunar)



You can change the default file manager if you go to "Settings Manager", and then open "Preferred Applications"

Go to Utilities tab, and choose your favorite file manager from the list.
