### Post by Version Dependency on 2012-04-24
> **makitso said:**
> I know that AWN is not well supported so am a bit worried.

AWN is a dead project.  It hasn't been developed for awhile now and it's official forums are 99% spam postings, so any problems with it are not going to be fixed.  It's probably best you find a new dock.  

[Cairo Dock]("http://glx-dock.org/") (GLX Dock) and [Docky]("http://wiki.go-docky.com/index.php?title=Welcome_to_the_Docky_wiki") are both fine choices...and well maintained.  Cairo Dock even installs it's own custom session you can choose at login.
