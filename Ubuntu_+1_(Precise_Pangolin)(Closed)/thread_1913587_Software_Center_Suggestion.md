### Post by effenberg0x0 on 2012-01-22
Some things I'd like to see:

[LIST]
[*]More / better filters (Change / Add Categories).
[*]Invert indexing options (Relevance, Top Rated, A-Z, Date).
[*]Better / faster search.
[*]More information about each result on lists (there's a lot of free space at the right of each "line", yet we do have to click "more info" more than we should).
[*]Mass-Operations via checkboxes. Ex. More info in 10 apps opens 10 tabs.
[*]Return to the same position when pressing "back" and after clicking install.
[*]Add a couple tabs on the top/filters: Free Software, Paid Software, Media.
[*]Better description of each app. Some are OK, many have a single line that is not enough to describe what the thing does.
[*]List / Select available versions inside "More info". Allow you to download previous versions of software.
[*]Options for what to display:
[/LIST]
[INDENT]
[LIST=1]
[*]- Include libs (Yes/No) - Could reduce the number of search results hugely.
[*]- Show only Software that is not installed (Yes/No)
[/LIST]
[/INDENT]
[LIST]
[*]More info: Show launchpad bug reports for application
[*]Make it faster. I don't know if that will come from embedding C code in selected routines. Go ASM if you have to. It's too slow right now.
[/LIST]
Regards,
Effenberg
