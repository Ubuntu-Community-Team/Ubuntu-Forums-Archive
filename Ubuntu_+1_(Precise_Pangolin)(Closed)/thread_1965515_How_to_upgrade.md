### Post by kaldor on 2012-04-25
Have you been updating your OS as you're going? If so, you're already running the final release.

The development releases are simply milestones or checkpoints. Beta 2 with all updates applied is the same as Ubuntu 12.04 LTS.

To make sure you are up to date, run this command:

```
sudo apt-get update && sudo apt-get upgrade
```

After that, you're done.
