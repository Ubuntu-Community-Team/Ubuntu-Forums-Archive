### Post by kio_http on 2012-04-10
This system was originally a Kubuntu precise installation. In KDE 4.8.2 if I with the blur effect the entire UI will freeze and become unusable. To resolve this problem, I installed "driconf" and set "Enable limited fragment shader support on 915/945 to No. After doing the blurring works fine and KDE desktop effects are extremely fast.

However now after installing ubuntu-desktop, I realised that unity does not start unless the setting is on Yes!

This makes it difficualt to use bot unity and kde on the same system

My graphic card is an Intel Gma 950.

This is a view of the driconf setting. It seems that kde and unity want different settings. Prior to KDE SC 4.7.4, kde also worked with the "yes" option.
