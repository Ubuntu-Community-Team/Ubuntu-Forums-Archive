Yes, it is likely. Sometimes some required packages can lag behind others and this can result in some nasty surprises while updating. Keep checking for further updates, but it is not recommended that Partial Upgrades be used.
You are also using a beta version which is still in development (and subject to breakages). A more stable version (like 11-04 or 11-10, or even the last LTS release, 10-04) may be a better option for you.
