### Post by seawolf167 on 2012-03-02
You can install

```
sudo apt-get install dconf-tools
```

then hit [ALT][F2] and run 

```
dconf-editor
```

to get to the same configuration as the previous

```
gconf-editor
```

As for the list-type desktop, do you mean opening a window [via Nautilus] and changing the display to list or compact?
