### Post by tlcstat on 2012-04-12
Greetings,
This is a testing section. If you are not a tester, please don't replace your current operating system with 12.04 until the final release has been available for at least one month. 

It takes software and hardware to make an operating system useful. In this regard Precise is still broken. If you want to test and play around with this OS while in development then this part of the forum is for you. If you need a dependable OS that is fit for running the software that you depend on then stick with what you have or if you just want to upgrade install Oneiric from scratch (clean install). It's a little more effort but will result in a more stable system. 

I hate to be repetitive but here it is, when this Precise Pangolin testing area is no longer available on the forum then you can safely install the final release of the OS, but you cannot necessarily get all of your software running. An operating system that you are not accustomed to using is not nearly as aggravating as an operating system without the software needed to do a job. Since it is not probable that the many users of 12.04 have installed the over 73,000 software packages that are available, I would not take their word that 12.04 is stable enough to use. It's called "not enough information". 

I personally have 12.04 installed on a separate 160Gig HD and although nice, it is for the most part is unusable. Software(s) are like cogs. It only takes one broken cog to stop the wheel from turning.

A word to the wise should be sufficient.
