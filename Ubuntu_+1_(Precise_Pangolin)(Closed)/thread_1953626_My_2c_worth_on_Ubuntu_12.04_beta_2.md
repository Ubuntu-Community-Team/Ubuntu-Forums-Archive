### Post by terrykiwi83 on 2012-04-06
I was running 11.10 x64, for which it was pretty unstable at times. Windows blackening out and programs crashing.

I upgraded from within 11.10 using update-manager -d .. etc

Been running it now for a few days and I can honestly say I am very happy with it. I have had no crashes, and all the programs seem stable so far.

My only initial worry was whether the upgrade would murder my Banshee, but nope it left it there. I am a big banshee fan, never really bothered with Rythmbox.

Couple of other observations I have noticed:

[LIST]
[*]Programs start up way faster than on my 11.10
[*]The Privacy option in the System Settings (don't remember seeing that in 11.10).
[/LIST]


One problem I do have is relating to Audio. On 11.10 I was able to set up my audio for 5.1 (As I have the full setup). However in 12.04 when I try to change the mode to 5.1, only the front speakers work and not the rear, center or the sub.

All in all though am very happy with 12.04.

That is all :)
