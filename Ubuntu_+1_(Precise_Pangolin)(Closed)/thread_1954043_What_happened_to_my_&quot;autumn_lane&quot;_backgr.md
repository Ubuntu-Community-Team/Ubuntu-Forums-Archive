### Post by ubuntu27 on 2012-04-07
Wallpapers are now available in different packages with the Ubuntu version's name.

The recent update pulled the Ubuntu Precise's default wallpapers. What you want perhaps is the wallpapers from the current version (Oneiric Ocelot), so just install the following:

```
[sudo apt-get install ubuntu-wallpapers-oneiric]("apt://ubuntu-wallpapers-oneiric")
```



There are also ubuntu-wallpapers-lucid, ubuntu-wallpapers-maverick, ubuntu-wallpapers-natty, ubuntu-wallpapers-karmic  :guitar:


Enjoy the wallpapers! ^_^
