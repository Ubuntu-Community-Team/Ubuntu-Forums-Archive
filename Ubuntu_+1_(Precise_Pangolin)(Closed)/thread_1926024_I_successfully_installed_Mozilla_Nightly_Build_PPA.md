I'd expect people reading the +1 area to know how to add a PPA:

$ sudo apt-add-repository ppa:ubuntu-mozilla-daily/ppa
$ sudo apt-get update
$ sudo apt-get install firefox-trunk

(Also, thunderbird-trunk)
