### Post by Mr. Picklesworth on 2012-02-03
Are you adding a custom background (one you added with the + button) or using one of the presets? The preset backgrounds already have this field set so they don't expose the option. With a wallpaper you have added yourself, you should be able to change between Zoom, Tile, Stretch, etc. using a nearby select box.

We don't have a particularly shiny theme chooser at the moment, but Gnome Tweak Tool exposes separate options for your window theme, cursor theme and gtk theme. The default UI preferences exposes meta-themes, which are specific combinations of those three.

If you are concerned about having as many options as possible, I suggest poking around in dconf-editor.
