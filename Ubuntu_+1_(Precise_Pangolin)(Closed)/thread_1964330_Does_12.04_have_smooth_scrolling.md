In Chromium:

Type chrome://flags

### Post by godhika on 2012-04-24
Smooth scrolling only came very recently to Gnome or to be more precise to GTK3.4. As Precise is using a mix of different versions of Gnome components (for example Totem is still version 3.01) smooth scrolling is only avaible for certain programs (and only for touchpads). Afaik software store, nautilus, web (formerly known as epiphany) and maybe Gedit should have smooth scrolling.There might be some others. Hope that helps
