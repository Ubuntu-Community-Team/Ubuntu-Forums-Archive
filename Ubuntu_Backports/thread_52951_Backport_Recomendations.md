### Post by Velvet Elvis on 2005-07-29
I thought I'd mention a few packages that I've backported for my own use using apt-get build-dep>apt-get source>dpk-buildpackage that I think others would benifit from using.

[cssed](http://packages.ubuntu.com/breezy/editors/cssed) - The best CSS editor for linux I've ever used.  

[gqview-2.0.0-1](http://packages.ubuntu.com/breezy/graphics/gqview) - New major upstream release.  The use of the next generation of gtk file dialogs or whatever they are called make it much more practical to use gqview for moving sets of images between directories.

[gnomebaker 0.3+20050430-1](http://packages.ubuntu.com/breezy/gnome/gnomebaker) - Much less crashy but I'm still using k3b.
