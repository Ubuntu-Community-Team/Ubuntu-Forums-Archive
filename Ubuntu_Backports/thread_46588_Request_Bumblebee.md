### Post by bored2k on 2005-07-05
I think this is a great application for cataloguing basically anything. > Bumblebee is a simple and unpredictable general collections manager. It works with CSV text files, can export to html, and import file listings.
It allows great control over data, with regex search&replace, handy moving and adding columns and calling external programs. new versions also include imdb search for movies and several column types with special features.

[INDENT]Features[/INDENT]
    *  import / export CSV
    * export html
    * import complex file listings (UI fo ‘ls’ ;) )
    * search for titles in IMDB (also in batch mode)
    * regex search&replace
    * great freedom in mangling the columns
    * support several types of columns: text,number,check,decimal,list,description,url
    * inovative display of similar entities
