### Post by rkelly on 2005-11-13
One of my systems at home is used configured as a server and hence has no Gnome (or KDE for that matter) installed. Automatically mounting should be done with usbmount, but the version in Breezy is 0.0.12, which no longer works with udev version 0.060. usbmount 0.0.14 has been adapted to this udev version.

Seems to me that usbmount 0.0.14 would be a likely candidate for Breezy-backports.
