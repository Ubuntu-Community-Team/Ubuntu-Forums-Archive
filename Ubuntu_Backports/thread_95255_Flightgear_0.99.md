### Post by noigeaR on 2005-11-26
flightgear 0.9.9 isn't in dapper yet so it can't be backported. backporting is only the process of making packages that are already in the next unstable version (right now that's dapper drake) available in the current stable release (breezy badger).
take a look at the package list of dapper drake here [http://packages.ubuntu.com/dapper/](http://packages.ubuntu.com/dapper/)
as of today, flightgear is still listed as version 0.9.8-1
as soon as 0.9.9 is in dapper it should be easy to backport though (and i'd appreciate it if somebody would backport it then).
