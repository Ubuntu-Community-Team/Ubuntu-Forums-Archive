### Post by UbuWu on 2005-10-29
Sometime before release there is an upstream version freeze. From then on, no new versions of programs are included to avoid introducing new bugs and be able to concentrate on fixing existing bugs.

Btw. a package made with checkinstall is not good enough to be included in the repositories. There is a little more work needed. See here for more info: [https://wiki.ubuntu.com/MOTU](https://wiki.ubuntu.com/MOTU)
