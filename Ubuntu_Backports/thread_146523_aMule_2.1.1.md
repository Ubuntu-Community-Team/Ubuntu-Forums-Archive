### Post by Treviño on 2006-03-18
New version of **aMule**: ***2.1.1***. [www.amule.org](www.amule.org)

Many bugfixes, improved stability.

I've made an **ubuntu package** compiled using this string:
```
./configure --enable-optimize --enable-amule-daemon --enable-amulecmd --enable-webserver --enable-amule-gui --enable-wxcas --enable-alc && debuild binary
```
**Download** from: [B][amule_2.1.1-0ubuntu1_i386.deb]("http://italy.copybase.ch/blog/informatica/linux/uscito-amule-211/")

[/B]Bye!
