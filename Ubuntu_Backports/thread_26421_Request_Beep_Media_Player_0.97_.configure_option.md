### Post by skoal on 2005-04-12
The latest beep media player (0.97) doesn't seem to be configured with the '--with-xmms-eq' option.

You can read [here](http://www.sosdg.org/~larne/w/FAQ#Why_is_the_output_volume_so_low_with_the_new_equalizer.3F) for an explanation.

Configuring the build with this option enabled provides compatibility with the current XMMS equalizer behaviour.  Without this enabled in the build, the sound is somewhat muted and the presets don't provide similiar 'acoustics' as when using XMMS.

I guess it's a matter of taste for such a request, but I don't know where else to post this.

Thanks.
