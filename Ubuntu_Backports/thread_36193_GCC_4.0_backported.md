### Post by jdong on 2005-05-22
GCC 4.0 and its support libraries are now in Backports Staging. It was backported from Breezy.

Breezy has been migrating towards GCC4, and as a result, many packages in Breezy have been patched to work with this new compiler. I'd rather use the same compiler intended, or we may run into compatibility issues ;)
