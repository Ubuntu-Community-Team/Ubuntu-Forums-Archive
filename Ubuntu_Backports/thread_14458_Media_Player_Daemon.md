### Post by lt_kije on 2005-02-09
if you add the package name you're searching for (eg gmpc) right after [http://packages.debian.org](http://packages.debian.org) (ie packages.debian.org/gmpc), it'll search the debian repositories.

packages.debian.org/gmpc -> [http://packages.debian.org/cgi-bin/search_packages.pl?searchon=names&version=all&exact=1&keywords=gmpc](http://packages.debian.org/cgi-bin/search_packages.pl?searchon=names&version=all&exact=1&keywords=gmpc)
