Gaim-encryption is a plugin that allows to establish and encrypted converstion with someone you're talking to.
This is very useful when the sys-adms want to have a look at what you're saying ;)
In other words, if you use it in your office, you're not the system administrator and you're saying compromising things you better install gaim-encryption ;)
