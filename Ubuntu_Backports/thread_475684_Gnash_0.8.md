### Post by Tommy on 2007-07-11
> **squell said:**
> Would it be possible to backport Gnash 0.8 into feisty??

I saw that a backports request has been filed in the bug reporting system: [https://bugs.launchpad.net/gnash/+bug/120990/](https://bugs.launchpad.net/gnash/+bug/120990/)

If you have successfully built, installed, and run the package using prevu, I believe you can add your confirmation to the bug so it gets released. If it DOESN'T build, install, or run in prevu on your system you can add your comments about the error messages to help get them fixed.

(I have not yet tried to compile gnash using prevu myself.)

In either case you can subscribe to the bug so you will get a notice when Gnash progresses to Backports.
