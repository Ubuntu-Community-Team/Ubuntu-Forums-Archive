The source package for VDrift is in the Feisty repository ([http://packages.ubuntu.com/feisty/source/vdrift](http://packages.ubuntu.com/feisty/source/vdrift)), the binary packages are still to be built. 
You can build your own Edgy packages from the Feisty source package using Prevu ([http://www.ubuntuforums.org/showthread.php?t=269238](http://www.ubuntuforums.org/showthread.php?t=269238)). Builds and installs perfectly.
I don't think the backport team will let the packages be backported to Edgy before they are available in Feisty, but it shouldn't hurt to file a backport request at [http://launchpad.net/products/edgy-backports/](http://launchpad.net/products/edgy-backports/).
