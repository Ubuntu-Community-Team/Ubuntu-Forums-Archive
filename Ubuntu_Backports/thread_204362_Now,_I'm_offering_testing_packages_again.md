### Post by jdong on 2006-06-26
Just a quick announcement about requesting backports

Now, when a backported package is eligible (source package is present in Edgy and builds fine), I will offer as Launchpad bug attachments a set of testing debs. These are built on my machines using pbuilder and edgy sources in compliance with Backports policy, so they are as if the Ubuntu build cluster built it themselves.

That way, we can get more testing done without a lot of technical know-how on the tester's behalf.

And thanks to gdebi, it's easier than ever to install deb packages :)
