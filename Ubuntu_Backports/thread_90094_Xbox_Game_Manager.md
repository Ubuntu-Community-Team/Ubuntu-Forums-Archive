### Post by kalin on 2005-11-14
Have you tried [this]("http://xbgm.sourceforge.net/files.php#Debian")? It's on the xbgm site, in the downloads section.

If you want to download the deb package and install it without modifying the sources.list file you can find it here: [http://xbgm.sourceforge.net/debian/unstable/xbgmsharp_0.9.0-1_all.deb](http://xbgm.sourceforge.net/debian/unstable/xbgmsharp_0.9.0-1_all.deb).

Install using the following command (in the terminal, in the folder where you saved the file):

```

dpkg --install xbgmsharp_0.9.0-1_all.deb

```
