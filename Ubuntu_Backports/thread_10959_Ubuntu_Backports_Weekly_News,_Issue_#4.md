**[SIZE=3]Ubuntu Backports Weekly News, Issue #4[/SIZE]**
*01/12/2005*

**Project Description:**
The Ubuntu Backports project's goal is to provide a stable AND up-to-date Ubuntu Linux system by backporting desktop applications from Ubuntu's Development branches and Debian Sid to the latest stable release.
[B]
This week's recap:[/B]
A new "Promotion Discussion" model of determinining promotions was started this week. Let's see if that works out. We hit the 250 package mark this week, and also the 200MB barrier. This is turning out to be a big effort, but one well worth it.
