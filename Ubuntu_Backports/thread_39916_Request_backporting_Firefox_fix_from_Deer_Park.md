### Post by Jormundgand on 2005-06-07
On the Firefox Bugzilla, bug [263198]("https://bugzilla.mozilla.org/show_bug.cgi?id=263198") describes a bug with Firefox's handling of mouse scrolling over a division of text with CSS property overflow set to auto or scroll. It was reported by one of Ubuntu's own people. Bugs [260621]("https://bugzilla.mozilla.org/show_bug.cgi?id=260621") and [279117]("https://bugzilla.mozilla.org/show_bug.cgi?id=279117") report the same problem.

According to bug 263198 comment 2 the bug has been fixed in Firefox trunk build dated 2005-03-11. My request therefore is that the fix for this debilitating problem be backported into 1.0.4 as a bugfix. It may be possible to petition for this to be added to hoary-updates.

I've thrown it around for a while to no avail, and got no response from the Zen-like contingent of #ubuntu-devel.
