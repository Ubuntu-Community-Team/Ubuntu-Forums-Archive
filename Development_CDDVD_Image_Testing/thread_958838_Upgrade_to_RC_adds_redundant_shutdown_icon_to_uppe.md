I don't know if this is a bug (not a big deal to me), but it might look a little funny to new users.  I installed 8.04 in Virtualbox today and **immediately** upgraded it to 8.10 RC to see how things would go.  Everything went without a hitch.  But after it was finished, the original shutdown icon remained on the upper panel, with the addition of a secondary smaller one (which is to be used for fast account switching).  Not sure why they're both there instead of one replacing the other, considering the 8.04 test system was unmodified after install.  So it might be a bug...

I was presented with a dialog box telling me of the change and asking if I wanted to remove it.  If you didn't get that dialog box, you can always just right-click the old 'power' icon and choose to Remove from panel.
