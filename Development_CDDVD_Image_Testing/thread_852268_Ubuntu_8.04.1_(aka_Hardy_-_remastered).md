### Post by sartic on 2008-10-24
Thx 4 all your comments and testing! There will be no ftp dl from today.

Here r some quick tips to add or remove component fast from command line...

Remove compiz:

sudo apt-get remove compiz compizconfig-backend-gconf compiz-core libcompizconfig0 libdecoration0

Remove tracker:

sudo apt-get remove tracker libtracker-gtk0

Remove evolution+ekiga:

sudo apt-get remove evolution evolution-common evolution-data-server evolution-webcal
sudo apt-get remove contact-lookup-applet libedata-book1.2-2 libedata-cal1.2-6 libegroupwise1.2-13 libexchange-storage1.2-3 libgdata1.2-1 

Remove fspot:

sudo apt-get remove f-spot

Add thunderbird+gqview+gparted+frozen bubble+boot up manager:

sudo apt-get install thunderbird gqview gparted frozen-bubble bum

Add restricted extras:

sudo apt-get install ubuntu-restricted-extras

Add vlc+audacious+extras:

sudo apt-get install vlc audacious audacious-plugins-extra
