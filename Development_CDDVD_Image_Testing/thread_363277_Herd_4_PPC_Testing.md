### Post by grazie on 2007-02-16
Today I downloaded (rsync'ed actually) and installed both the Desktop and Alternate Xubuntu images for PPC. Although both installed, neither would boot for what looks like different reasons. If these were x86 builds I'm certain I wouldn't have had the same problems. However, with the escallation of commercial PPC support being no longer available, to all PPC support being completely dropped, I'm wondering whether what I am doing has any value. If PPC Ubuntu survives it'll be reliant upon a very small part of what is now quite a large community. In fact the largest Linux community. I have my doubts whether it will succeed.

Am I completely wasting my time testing and reporting bugs? I think am. If I report the tests I've done and the bugs I've found, will anyone be interested? I'm beginning to think not.

I returned to the Ubuntu distro around Christmas time as I was impressed by the work done by a small, but well focused Xubuntu team. At last there was a PPC distro that had loads going for it. I liked it so much I decided to be more actively involved in the Ubuntu community. I volunteered myself for testing in response to Henrik Omma's specific request for PPC testers. I now have doubts whether it's worth while continuing.

If anyone else is in the same position, I'd be very interested to know about it or to hear the views of anyone that cares to express them.
