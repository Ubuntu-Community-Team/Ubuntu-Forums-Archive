### Post by pochu on 2007-03-01
The purpose of the ISO Testing Team is to test the iso candidates, so you should test the CD as it is, if it starts, if it install to the hard disk, if it works fine... and report it on Launchpad (you can read the how-to).

The CDs we test are, mostly, the candidates to the different releases, such as the alpha candidates (herd 5 soon), RC, and Final. However, daily builds testing is also appreciated, but the candidates are more important.

If you have any other question, let us know :)
