### Post by kidders on 2007-05-15
Hi there,

In practical terms, you may find it easier to just copy hard disk dumps around, rather than rolling your own Ubuntu. (I'm assuming you're interested in finding a way to install Ubuntu on multiple boxes, without having to install & configure all the software you need, over and over.)

Obviously, using that approach means that all your servers would have to have the same CPU architecture, but that constraint would also apply if you made your own install CDs.
