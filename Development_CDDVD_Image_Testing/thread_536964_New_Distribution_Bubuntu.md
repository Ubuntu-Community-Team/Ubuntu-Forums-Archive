### Post by CarlosNYB on 2008-08-28
Well one good reason to make a spin-off is because you need to configure your machine in specific ways, and others are interested in similar customizations.  For example, Ubuntu Studio is optimized with a real-time kernel and with packages configured to work together for producing multimedia, sound, grahics, etc.  You don't need to download the iso for ubuntu studio to use the packages -- I prefered to install regular ubuntu and then download their kernel and effects/music packages.  Their efforts make it a bit easier to see what's out there for multimedia in the Ubuntu repositories, also.

Similarly, it seems many would be interested in a meta-package which could be coupled with the minimal install, something like a "damn small Ubuntu", even quicker and leaner than Xubuntu.
