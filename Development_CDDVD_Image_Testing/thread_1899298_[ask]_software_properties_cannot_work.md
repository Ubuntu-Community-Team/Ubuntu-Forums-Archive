I really don't see how that has anything to do with development cycle image testing.

lsb-release is a tool for reporting the Linux Standard Base version. See more at [https://wiki.linuxfoundation.org/en/LSB]("https://wiki.linuxfoundation.org/en/LSB")

If you are interested about creating your own installation images, you should start by looking at [Alternate CD]("https://help.ubuntu.com/community/InstallCDCustomization") and [LiveCD]("https://help.ubuntu.com/community/LiveCDCustomization")
