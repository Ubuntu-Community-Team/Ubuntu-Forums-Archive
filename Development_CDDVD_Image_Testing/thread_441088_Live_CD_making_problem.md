### Post by smartboyathome on 2007-05-12
When trying to add and remove programs in reconstructor, you have to go into the terminal and manually copy and paste your sources.list into there. Then, apt-get update && apt-get upgrade in that same terminal. Now, do not close the terminal yet! apt-get install your programs and you are ready to go! Be warned, though, make sure that you get all the programs on your disk, as once you finish, there is no way to edit it.

I'm pretty sure the chroot environment that terminal is running under doesn't support X, so you'll have to use a text based editor.  nano is pre-installed and works fine for me.

So you'll want to type:
nano /etc/apt/sources.list
within reconstructor's terminal, that'll edit the livecd's sources.list.  Then you can apt-get whatever apps you need.
