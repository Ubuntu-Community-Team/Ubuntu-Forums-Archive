### Post by Henrik on 2007-06-26
Martin Pitt has [announced]("https://lists.ubuntu.com/archives/ubuntu-devel-announce/2007-June/000310.html") that the main archive is now frozen for Tribe-2. We'll be looking to squash [this list of bugs]("https://bugs.launchpad.net/ubuntu/+bugs?field.milestone:Alist=469") before the Tribe-2 alpha. 

Testing will be coordinated via the [ISO test tracker]("https://isotesting.stgraber.org/"). If you want to participate in testing, please register (registrations from feisty were not carried over.) More information on testing and reporting found [here]("https://wiki.ubuntu.com/Testing/Community"). 

The first candidate images should be available today or tomorrow.

**Update:** The candidate ISO images are now available for testing, starting with 20070626.1
