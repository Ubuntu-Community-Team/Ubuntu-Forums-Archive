### Post by smartboyathome on 2009-02-22
Only legal issue is that if you plan to use unsupported packages (ie, packages that aren't in the repo or are changed), then you can't use *buntu in the name. The source code is provided for each program in Ubuntu's repo, so as long as thats provided then your good.
