### Post by RageOfOrder on 2007-08-26
I never got my synaptics touchpad working right, so I can't say for sure if this will help you, but I use evdev for my mouse and it doesn't seem to interfere with my touchpad.

```

Section "InputDevice"
     Identifier            "Mouse0"
     Driver                  "evdev"
     Option                 "evBits" "+1-2"
     Option                 "keyBits" "~272-287"
     Option                 "relBits" "~0-2 ~6 ~8"
EndSection

```
