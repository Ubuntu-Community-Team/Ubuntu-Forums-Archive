### Post by jshatch on 2007-12-28
I have been able to print on the MX310 using the Canon Pixma MP150 driver (CUPS).  Try that out.  I have done this both while connected via USB and over the network to a Windows machine.  Both work fine.

The only limitation I've seen so far is that you can't print at higher than 600x600 DPI.  YMMV.
