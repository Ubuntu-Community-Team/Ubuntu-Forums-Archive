### Post by hollywoodb on 2006-05-28
I'm running Xubuntu, which includes a newer version of Xfce which "feels" faster.  Resource-wise I'm about the same between breezy & dapper.  You might want to take a look at the Xubuntu route instead of default Gnome for an older system.

EDIT: If you like you can try Xfce on breezy before you upgrade:

```

sudo aptitude install xubuntu-desktop

```

then back out to login and select Xfce session

to remove xubuntu-desktop:
```

sudo aptitude remove xubuntu-desktop
```
