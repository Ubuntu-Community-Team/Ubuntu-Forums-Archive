It depends, some bios and motherboards, I guess, can monitor the heat and adjust the cpu fan speed accordingly (check your bios settings.) The same can be done with the power supply fan, so check this as well, located on the back of your computer near the fan if applicable. Mine has an automatic setting (computer controls the fan) or constant setting, stays the same, with the flick of the switch.

Installing the correct video drivers can ease the video card fan (another possible source of noise). If I install a fresh copy of a distro, like Ubunut with the stock nvidia driver "nv", the video card fan whirls away. As soon as I install the latest nvidia driver, all is fine unless I use a graphic intense program which will cause the fan to whirl.
