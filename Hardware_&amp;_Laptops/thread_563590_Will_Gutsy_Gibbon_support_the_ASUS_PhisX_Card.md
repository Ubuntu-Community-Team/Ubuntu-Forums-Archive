### Post by Pyre_Vulpimorph on 2008-03-22
Super Bump

I believe this is a valid question.  Is there any software whatsoever for Linux that can take advantage of the Ageia PhysX hardware?  Ageia's website says hardware acceleration will work ONLY on Windows XP (sucks for Vista adopters!) but Linux programmers have always proven resourceful.  Perhaps, now that Nvidia bought out Ageia, there is some greater possibility of PhysX drivers.

Personally I'm intrigued by the concept of advanced PPUs capable of utilizing PCIe 2.0 (and not just plain-vanilla PCI like current PhysX cards), but I think Nvidia killed it by promoting physics engines on their GP-GPUs.  Don't you think  dedicated PPUs should be allowed to take care of real-time physics and let the GPUs focus on what their built for--rendering graphics?

Physics Processing Units have vast potential, if only they were more widely supported--and we get some working Linux drivers!

EDIT:  PPUs would be useful for more than just gaming.  Think about the scientific applications!  Maybe I just bought into Ageia's propaganda, but physics processors don't deserve to die.
