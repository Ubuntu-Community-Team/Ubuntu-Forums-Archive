### Post by dabl on 2007-08-03
Envy has installed the new driver since 22 June -- that's what you need.  Get Envy here:

[http://albertomilone.com/nvidia_scripts1.html](http://albertomilone.com/nvidia_scripts1.html)

You want to download the file "envy_0.9.7-0ubuntu5_all.deb" which is about halfway down the page -- when it is downloaded, right-click it, choose "Install with Gdebi" and it will be installed. Then run it, and it will install the new driver for you.  :popcorn:
