### Post by Don Gatto on 2008-02-06
Hello,

As a blog ([http://www.bytebot.net/blog/archives/2007/03/29/which-ubuntu-for-a-core-2-duo](http://www.bytebot.net/blog/archives/2007/03/29/which-ubuntu-for-a-core-2-duo)) says, both amd64 and i386 versions of Ubuntu work on that processor well (I guess difference is nearly only in optimalisation). So the answer for the first two questions is 'yes, (k)Ubuntu is compatible'.

You can find nVidia drivers for Linux on their site. But you don't need to do that, since the nVidia driver is available through the Ubuntu repositories, which are a lot easier and safer to use - you could install them like any program for Ubuntu.

(By the way, you might be interested in the Kiwi Linux project, an Ubuntu-based system designed for Romanian and Hungarian users: [http://kiwilinux.org/kiwi/ro/](http://kiwilinux.org/kiwi/ro/) .)
