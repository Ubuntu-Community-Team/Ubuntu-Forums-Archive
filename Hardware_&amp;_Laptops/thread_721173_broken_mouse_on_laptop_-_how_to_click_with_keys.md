### Post by jken146 on 2008-03-11
Use keyboard shortcuts to get around.  Alt+F1 brings up the menu, Alt+F2 brings up a little box from which you can run a single command -- this is useful for launching applications.  Alt+Tab switches between open windows.  Alt+F4 closes the current window.  Use tab to move between links in web pages and space to activate them.

The terminal can be used to do pretty much anything you need too.
