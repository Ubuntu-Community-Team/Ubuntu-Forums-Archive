### Post by mr_niceguy on 2006-04-26
Ubuntu does not have a matching printer driver anyway. You're probably better off using [Turboprint]("http://www.turboprint.de/english.html"). If you set it up at the lowest resolution ("Draft" or 300 dpi) you can use their driver for free or else you can go to their website and pay for a key which enables you to print at higher resolutions.

A while back I did a quick script to install a free driver which Louis supplied the link to. Unfortunately many people (myself included) experience somewhat flaky support with the free driver but some people seem satisfied with it.
