### Post by AgenT on 2005-09-04
What do you mean which version? Which version of what exactly? Are you asking whether you should use Ubuntu or Kubuntu? 

You should of course try Hoary first since that is the newest stable release (Breezy is comming out very soon, but you will be able to upgrade to it without problems). 

If you have a fast internet connection, download the Ubuntu LiveCD for Hoary and try it out. This will let you test out Ubuntu without installing it. It will be much slower because every single program will run off the CD and not your hard drive, so do not take how slow into consideration when deciding on whether or not you should install Ubuntu.

And just so you know, your laptop is *not* old by any means. Only Windows is so annoyingly un-customizable. There are people running Linux on 300MHz computers fine. Ubuntu uses GNOME by default which requires, although a little less, similar hardware to a fresh XP install. But you can always customize it and even use something else.

Also, you can see how other people have installed different versions of Linux on your laptop here:
[Linux-Laptop Toshiba]("http://www.linux-on-laptops.com/toshiba.html")

Your laptop is listed only once, and the person wrote in Spanish. But there are other laptops similar to yours on there as well. Do note that usually you will have a much easier time than the people that are listed on Linux-Laptop because hardware support is always improving on Linux.
