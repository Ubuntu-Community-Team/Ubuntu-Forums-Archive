(Sorry for that.)  All I remember is this: I tried to install the **pcmcia-cs** package, but Ubuntu reported that the latest version of that package was already installed.  I believe I also tried to install or installed the **cardmgr** package.  I also ran the command **modprobe pcmcia**.  Let's see, what else did I do?  I guess that's it...if I remember anything else, I be sure to add it here.  I probably also rebooted my machine before it worked.
