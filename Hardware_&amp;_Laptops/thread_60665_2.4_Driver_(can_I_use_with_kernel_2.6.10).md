### Post by SickTwist on 2005-08-30
[QUOTE=Mr. Electric Wizard]I have a SMC9452TX Gigabit ethernet card that I am trying to install.
THe only drivers that were included for Linux were for 2.2 and 2.4.

No, you must use the kernel for which the driver was compiled. What happens if you just install the card and reboot the computer? Does the kernel detect the card?

After you have rebooted, run lspci in a terminal see if the ethernet card is detected. Also, you could run ifconfig to see if it is listed there as well. It would be called eth0 if it is the only ethernet card in your box.

Try "ifconfig -a".  If the interface is not "up" (i.e. has an IP address), ifconfig won't show it by default.  The -a means "show all".
