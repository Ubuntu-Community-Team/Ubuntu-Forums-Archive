### Post by tamoneya on 2008-04-08
looks like a fun build.  My only recommendation is that if you are not familar with linux that you set up a dual boot.  This allows you to pick which operating system you want to run( windows or linux).  The only reason i mention this now is because windows isnt as smart as linux when it comes to setting up dual boots.  It is far easier to install windows and then install linux.  Otherwise windows will overwrite the bootloader and linux will become unbootable.  While this isnt hard to fix it is better to avoid the problem if you can.
