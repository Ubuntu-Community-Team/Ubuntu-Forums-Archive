This morning I woke up and wanted to enable frequency scaling. By default I can not adjust the frequencies, it's on 'ondemand' by default - which does pretty nice by itself, but I wanted a little more control...

I looked around and found this simple execution does the trick:

```
sudo dpkg-reconfigure gnome-applets
```
