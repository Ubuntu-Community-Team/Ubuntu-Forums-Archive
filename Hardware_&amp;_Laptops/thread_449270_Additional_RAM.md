### Post by Matty2006 on 2007-05-20
This is from the Dell Website: [http://support.dell.com/support/edocs/systems/ins8200/en/sm_en/specs.htm#1103762]("http://support.dell.com/support/edocs/systems/ins8200/en/sm_en/specs.htm#1103762")

Memory Architecture
PC2100 DDR

Memory module connector
two user-accessible SODIMM sockets

Memory module capacities
128, 256, and 512 MB

Memory type
DDR SDRAM

Standard memory
128 MB

Maximum memory
1024 MB

Memory access time: clock speed
266 MHz

If you have 2 sticks of 256 then your going to need to replace these with two sticks of 512 to get 1G.
Or just swap one stick out for a 512 for a total of 768
-Cheers
