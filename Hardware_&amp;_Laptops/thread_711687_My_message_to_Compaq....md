### Post by nutz on 2008-02-29
I sent this to Compaq regarding the lack of available options for updating my system bios under Linux. Tell me what you think and if you have a comparable experience that is related to this I would like to hear about it.

> I recently tried to update the bios on my Presario F750us laptop with the software available on your website. I wanted the new bios because it provided enhanced power management support for my wireless nic. I do not use a Microsoft based operating system on my laptop and because of that was unable to install the bios update. I spoke with your technical support via the web chat interface to see if any options were available to me and they informed me that without having a Microsoft based operating system I would not be able to install any bios updates for this laptop. I informed them that I did have access to a floppy disk that had Microsoft DOS but they said that option would not work either. So to sum this up I am left with no support for this bios update unless I install a Microsoft based operating system on my laptop.

I believe this is an unfair business practice and would like to see some other option that does not require the installation of a Microsoft based operating system on the hard drive. Doesn't Compaq feel obligated to support their product in the absence of a Microsoft based operating system? If so then wouldn't they be obligated to express this condition or term when selling their products?

I realize this bios update is not required for the operation of this laptop but I feel discriminated against in the fact that I will no longer be able to receive bios updates because of the operating system I choose to use. Please provide your feedback on this as I genuinely like the product otherwise and would like to take advantage of future bios updates because they provide enhancements that I would like to benefit from in the same way users of a Microsoft based operating systems do.
