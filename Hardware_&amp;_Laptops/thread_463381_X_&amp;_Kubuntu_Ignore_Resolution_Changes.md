### Post by vigleik on 2007-06-03
What are you doing as root? You should never be root (well, almost never), but rather execute commands with sudo, or in the case of graphical programs, gksudo. So use "gksudo restricted-manager" instead. (The part about not doing things as root is different from most linux distros, and is a common mistake.) That should allow you to turn off the nvidia driver and get back a functional desktop.

Since you get the nvidia splash screen, it seems the nvidia driver is installed but doesn't handle the graphics card/monitor combination. Why, I don't know. I'm afraid you'll need help from someone who knows more about this than I do.
