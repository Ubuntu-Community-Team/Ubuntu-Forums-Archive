### Post by dthacker on 2007-03-10
Hi Psyphre, 
Time for hardware debugging!   You'll want to rule out the USB port and the keyboard, so try these steps.  

[LIST=1]
[*]move the keyboard to different USB ports on your computer, see if one port always locks up
[*]try a different keyboard, see if the problem goes away
[*]if possible, try a ps/2 keyboard or a ps2/USB adapter, see if the problem goes away. 
[/LIST]

See if those help you isolate the problem.  Good Luck!
