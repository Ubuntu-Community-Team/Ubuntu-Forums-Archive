What features are you looking for?  Do you want wifi?  RAID?  Anything else specific?  If you do not need any of these features, I might question the value in spending $200 on a mobo, as the extra $100 would be spent primarily on these features that you wont use.  I am using a Biostar T-Force 550 (Nvidia nforce 550 chipset) and paid under $100 for the board (no extra features I don't use... well, it does do raid 0/1/0+1 and I don't use that, but thats about it) and I couldn't be happier with it.  It's got more OC'ing control than any other board I've seen, and that includes tweaker boards at more than double the cost.

I'm not necessarily suggesting that you go with this board, but before you drop $200+ dollars onto a motherboard, be sure that the features you are paying for are features that you will actually use

EDIT: I know you said you don't need SLI, but this board is $120 after rebate on newegg... seems like a solid deal. [http://www.newegg.com/Product/Product.aspx?Item=N82E16813138033](http://www.newegg.com/Product/Product.aspx?Item=N82E16813138033)
