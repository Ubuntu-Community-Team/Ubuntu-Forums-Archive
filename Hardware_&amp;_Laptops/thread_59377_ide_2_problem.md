### Post by skoal on 2005-08-23
Did you ever have peripherals attached to the second IDE channel at one time or another, in Windows, Linux, or elsewhere, and it worked at one time?

If so, try swapping IDE ribbon cables with the first IDE channel and see if you're even able to boot?  I've bought new IDE cables before and they've been useless, so, it happens.

Make sure it's plugged in all the way (both at mobo and peripheral ends).  If a peripheral isn't keyed, you can tell which is pin1 by looking on green fab for "2" and/or (typically) the red stripe faces closest to the power connector for the peripheral.
