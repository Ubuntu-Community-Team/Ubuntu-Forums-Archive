### Post by timetrap on 2008-01-31
Welcome to LINUX!  It sounds like this may be your first time installing.  

You will need some information before you go much further(I am assuming that other things are not working also):

1. Graphics Card Model (Intel GMA950, GMA945, etc.)
2. Wireless LAN Card (Dell 1390 WLAN, etc.)
3. Wired NIC (Broadcom, Intel, etc)
4. Soundcard Model (Sound Blaster, etc.)

It is VERY important to know what hardware you currently have.  Without this information it may be very difficult to complete an install of any version of LINUX.

An easy way to obtain all of this is go to the support site for your laptop.  Then search by the service tag (on the bottom of your laptop) or search by model (Dell E1505, D630, etc.).  Find the hardware installed and write it down.

Once you get this information the forum can assist you further (or point you in the right direction).

You may even get really lucky with google.  An easy way to help yourself very quickly is to type the following phrase into google:

```
(<Laptop Model>)+[Ubuntu|Linux]
```

Replace <Laptop Model> with the actual model name (As discussed above). You will be suprised how many other people have the same problems.

Option "XkbModel" "pc105"
Option "XkbLayout" "us"
EndSection

Section "InputDevice"
Identifier "Configured Mouse"

Option "HorizScrollDelta" "0"
EndSection

Section "ServerLayout"
Identifier "Default Layout"
screen 0 "screen1" 0 0
Inputdevice "Generic Keyboard"
Inputdevice "Configured Mouse"
Inputdevice "Synaptics Touchpad"
EndSection
