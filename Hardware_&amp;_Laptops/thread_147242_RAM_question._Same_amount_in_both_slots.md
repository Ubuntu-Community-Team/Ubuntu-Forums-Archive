### Post by az on 2006-03-19
[QUOTE=weasel fierce]I just bought 512 MB of RAM, then realized that my existing slot only holds 256 MB.

On older boards which used SIMMs, you needed to add ram in matching pairs.  This is not true for sdram or ddram.  There is no performace hit, either, AFAIK.
