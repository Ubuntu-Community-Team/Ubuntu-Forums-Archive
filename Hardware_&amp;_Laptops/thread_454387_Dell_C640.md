### Post by berylmichael on 2007-05-25
HI!

Try reburning the Ubuntu image again but make sure that your burning program will burn the CD as an ISO image (see [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)). After that is completed, make sure that the download is verified as complete - the check sum is valid ( see [https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)). Once this is done, then you should be able to install Ubuntu.
