### Post by Keith_Beef on 2007-06-03
There, got it fixed.

Thanks to Heimo, I read the resolution HowTo, and changed the refresh rates.

```
sudo ddcprobe | grep monitorrange
monitorrange: 30-82, 50-75
```

Then use these figures to set the refresh rates in the /etc/X11/xorg.conf file.


Beef.
