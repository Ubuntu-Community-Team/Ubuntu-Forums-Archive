Hi, had the same problem when I plugged my Archos 504 in for the very first time. Not sure if it will work in your case but try going into the Settings on the Archos device itself, and make sure it is set to be detected as a "PC Hard Drive" and not a "Windows Media Device". If it is already set to "PC Hard Drive" and still nothing happens, try just toggling between the two settings, sometimes that can kick up a dialog box on the Archos for the USB connection asking you how you want to connect the device. If you see that come up, select "PC Hard Drive" and you should then see the device is mounted automatically on the desktop by default as A504 (hard drive icon). 
You can then use a file manager to drag and drop your videos, music, photos and files.
