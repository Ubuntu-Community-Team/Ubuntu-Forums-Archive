Hi Folks:

I"ve tried to install Ubuntu on an IBM 390x I've got here. The installer runs fine until it gets to the point where it wants to detect the network card. The installer reports that it cannot find the network card.

The network card is an IBM EtherJet pcmcia card and the network here is wired.

Any thoughts on a pcmcia card that will work or the posibility of using a USB wired network adapter?

Thanks everyone in advance for your thoughts and help!

### Post by rharris on 2005-12-14
Hi Folks:

I have found a solution to the problem. While it is not elegant, it is a KISS solution. After doing some research, I ended up at the following site:

[www.linux-usb.org/usbnet/](www.linux-usb.org/usbnet/)

I decided to give the D-Link DUB-E100 a try. After recieving the package from NewEgg I got it jacked in to the home network and tried installing Ubuntu again.

I am happy to say that the install went without a hitch, Ubuntu found the USB Network adapter with no problems at all and Ubuntu 5.10 lives happily on my laptop. The only very minor problem I have with the laptop now is the sound is'nt quite right but I'll figure it out.

Two computers down and one to go then I'll be windoze free!

Robert
