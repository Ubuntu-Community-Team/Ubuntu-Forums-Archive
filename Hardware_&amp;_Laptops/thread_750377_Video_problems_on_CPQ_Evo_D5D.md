### Post by bengray27 on 2008-04-09
Hello all, 
              I'm trying to install/run Ubuntu on a Compaq EVO D5D and have so far been unsuccessful.  I've tried both 7.04 & 7.10 and both get no video display when booted from the Live CD.  I can get to the point of the Ubuntu logo and progress bar after the boot menu but afterwards I get a black screen with vertical blue lines.  I don't think the PC has frozen up because I still get the Ubuntu login sound.  I am able to get video when I boot into safe graphics mode but if I install to the HDD, i get the same video problem again after initial reboot.  I'd really appreciate any help because I don't really like openSUSE all that much (that's so far the only OS besides WIN that has worked on this PC).  Thanks

System specs:
40 GB HDD
CPU - Intel P4 1.75GHz
640MB PC-133 RAM
Nvidia VANTA-16 AGP Card

### Post by bengray27 on 2008-04-22
Thanks to anyone that may have been looking into this. 
I solved my own problem.  The problem was the monitor that I was using was a 22" HP p1230 and by default, Ubuntu sets the resolution to the max that the monitor can handle.  Unfortunatley, my POS 16mb video card couldn't handle that big of a screen resolution.  Once I switched to a smaller 17" monitor, Ubuntu worked just fine.
