### Post by almkglor on 2007-06-08
Try searching for a tool called "915resolution".  915resolution can be installed via apt-get or synaptic.  You'll probably need to assign an unused mode with 1280x800.  Also, you may need to edit /etc/X11/xorg.conf to add a 1280x800 resolution.
