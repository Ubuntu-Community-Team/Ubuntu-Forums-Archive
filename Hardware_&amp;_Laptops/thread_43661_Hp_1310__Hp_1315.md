### Post by Zhukov on 2005-06-22
I've finally managed to make this printer work with everything.

Just follow this tutorial, except the final part about printing:

[https://wiki.ubuntu.com/HpPscHpPhotosmartSeriesAllInOnePrinters](https://wiki.ubuntu.com/HpPscHpPhotosmartSeriesAllInOnePrinters)



Instead:

Go to

[http://www.linuxprinting.org/show_printer.cgi?recnum=HP-PSC_1310](http://www.linuxprinting.org/show_printer.cgi?recnum=HP-PSC_1310)

and download the PPD.



Restart the cups system to see the new hpoj service by running
sudo /etc/init.d/cupsys restart 

In System->Administration->Printers double-click "New Printer"
Select "Use another printer by specifying a port:"
In the drop-down select the entry beginning with "PTAL".

Then, select "Install driver" and select your PPD file.


Now print a test page.


As you can see, both the scanner, and the printer work flawlessly.
