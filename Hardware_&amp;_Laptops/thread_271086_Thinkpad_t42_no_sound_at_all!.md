OK Jayne...don't worry too much! We've all been there...it can just be so overwehlming!
Let's take it one step at a time!
You'll need a terminal which you'll get by clicking on:
Applications > Accessories > Terminal..

A kind of "window" will open; that's called a terminal. It'll look something like this:

**th@luser:~$**

where "th" will be YOUR user name. Inside there and after the "$" symbol, you'll want to type the following:

```
sudo gedit /etc/apt/sources.list
```
At this point, you'll type your password in!

You can see this here:
[http://getautomatix.com/wiki/index.php?title=Installation&Itemid=38](http://getautomatix.com/wiki/index.php?title=Installation&Itemid=38)

sudo gedit /etc/apt/sources.list is called a "command line"; in this case, it'll prepare for the Automatix installation in your box.
