### Post by Dragonfire on 2006-12-13
Dell, HP/Compaq computers are heavily affected by this. that last update two days ago has had a very drastic effect on all computers using the texas instruments pci 1510 cardbus controller.

These computers will no longer recognize PCMCIA/PC cards. It is currently believed that during the software update these slots have been damaged. As the card slots do not work on Windows OS, or Ubuntu. The reason is currently unknown.

Will post update when new information is gathered.
