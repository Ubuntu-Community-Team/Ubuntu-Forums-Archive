### Post by H.E. Pennypacker on 2006-07-25
Here's the solution, before you try to suspend, go to Power Management, and enable "Suspend."  If it is already enabled, select something else from the same drop-down menu (let's say "Blank").  Then, go back to "Suspend," and when it asks you if you are sure you want to allow suspend, say yes, but don't check off the "don't ask me again" box.  

Do this every single time before you find a permanent solution.  I am still looking for a solution, and I still have to search launchpad to see if this has been reported before. 

By the way, it's possible that *you* only need to enable suspend once for it to always work, as opposed to my case.  See what works for you.  If it doesn't, we're in the same boat, and I am pretty sure this has been reported before, but I don't have the time to look it up.

Anyhow, stick to my solution in the mean time.
