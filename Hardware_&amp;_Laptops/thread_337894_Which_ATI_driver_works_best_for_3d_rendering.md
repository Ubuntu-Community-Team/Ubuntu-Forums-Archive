### Post by sloggerkhan on 2007-01-22
I am using the ATI open source driver and in the past have used the FGLRX closed source driver.

I have no problems running beryl with the open source driver and Armagetron seems to run fine. I think I might have gotten about 5-15% better performance in games from the closed source driver, though that's more of a guess than anything. Beryl seems a lot more stable with the open source drivers, though.

I think as long as you have a functional graphics card and your xorg.conf is configured correctly, you will be able to use 3-D acceleration.

I should edit my previous statement:
I have no issues with beryl, most games using open source driver on an x700.
I think you should use fglrx if you have an ATI card newer than an x800 or so.

2nd, some applications don't work as well with the open source driver as others. (For example, planshift crystal blue. (Texture problems)
