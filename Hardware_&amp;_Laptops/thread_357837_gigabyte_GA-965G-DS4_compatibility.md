I'm planning to buy a new motherboard gigabyte GA-965G for intel core 2 duo E6300.
I've chosen this mobo because it provides the intel 965G chipset with integrated graphic media accelerator X3000 where it's possible to run the intel open source drivers (these drivers should be supported by xorg starting from 7.2 release,  [http://xorg.freedesktop.org/wiki/ChangesForX11R72](http://xorg.freedesktop.org/wiki/ChangesForX11R72) ).

Almost all motherboards with G965 chipset are uATX form, while this gigabyte is ATX form. I found also two intel mobo ATX form with G965 chipset, DG965RY and DG965RY, but GA-965G has more features at about the same price.

I made some searches and I didn't found particular problems except something about realtek ALC888 audio chipset.

I would like to know if somebody have this motherboard and had found particular compatibility problems.
