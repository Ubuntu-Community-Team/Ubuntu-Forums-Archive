### Post by sas on 2006-05-29
Hi, one of the best things to do may be to try upgrading to Ubuntu 5.10 (see [https://wiki.ubuntu.com/BreezyUpgrade](https://wiki.ubuntu.com/BreezyUpgrade))

The .tar.gz extension means they're compressed files (like a .zip file on windows). If you click on the file it should open Archive Manager you can drag the files/folders from their to the desktop.

You can also type "tar xvzf filename.tar.gz" in the command prompt terminal.
