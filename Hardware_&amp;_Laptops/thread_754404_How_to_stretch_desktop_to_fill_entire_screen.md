### Post by rsf33 on 2008-04-13
I have an old Sony Vaio notebook, a PCG FX390 with an Intel 82815 graphics card and a 1024x768 display. The notebook came with Windows 2000 installed. When running Windows, if I set the screen resolution to 800x600, the desktop does not fill the screen--it is centered on the screen with big black borders around it. If I press Fn+F, the desktop stretches to fill the entire screen. This makes the image larger and easier for me to see. This is important to me because I'm visually impaired.

I installed Ubuntu 7.10 on this notebook, and everything is great if I use the display at 1024x768 resolution. If I set the screen resolution to 800x600, the desktop again does not fill the screen. Pressing Fn+F doesn't do anything when running Ubuntu.

Does anybody know how, when running Ubuntu,  to get the 800x600 image to stretch and fill the entire screen as it can in Windows? From the Intel website this facility appears to be called "panel fitting", but I don't know if the Ubuntu Intel video driver supports it or how to activate it if so.

I am using the "Intel experimental modesetting" video driver--which installed automatically when I installed Ubuntu 7.10 on the notebook.

Any help would be greatly appreciated. I tried using desktop zoom as a way to enlarge the image on the screen when in 1024x768 mode, but enabling visual effects causes all sorts of rendering problems (moving the mouse pointer leaves a trail of misplaced graphic image fragments).

Thank you,
Roger
