### Post by benanzo on 2007-05-11
Open a terminal and type:

```
glxinfo | grep direct
```

If you get:
```
direct rendering: Yes
```
Then your XServer is fine and we'll troubleshoot Beryl.

If you get:
```
direct rendering: No
```
Then your XServer is not setup correctly.

You can start troubleshooting the XServer by looking for errors in the logs:
```
cat /var/log/Xorg.0.log | grep EE
```
post the output back here.  What is the native resolution of your monitor?  Are you getting it with your current config?
