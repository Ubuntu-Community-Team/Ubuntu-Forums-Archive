Hello everyone,

Recently I bought a Brother DCP540CN printer, and when I tried to install it, I figured out that I can't install my printer through Administration->Printing dialog...

Anyway, I searched the forums and the internet, and found some great resources (below), then I gathered all needed information and wrote a simple bash script which only execute the needed commands (it also asks you to switch the printer off...)

So, I attached the script to this post.

For users who want to confirm that this script don't do any harm to their computer ([http://ubuntuforums.org/announcement.php?f=44](http://ubuntuforums.org/announcement.php?f=44)), you can read the post that I took the commands from: [http://ubuntuforums.org/showpost.php?p=2275292&postcount=31](http://ubuntuforums.org/showpost.php?p=2275292&postcount=31)

This script only intends for network installations (if someone needs a local installation, and if I have I time, I will write one for local installation).
It will install the printer (make sure to turn it off when asked), the scanner and the scan-key tool which enables you to scan from the machine to the computer without the computer...
