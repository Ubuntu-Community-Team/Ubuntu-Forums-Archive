### Post by RevThwack on 2007-09-18
If you're looking for dead easy, an ethernet modem like the  Actiontec GS002AN201 would be the way to go. Linux, Windows, and Mac all know how to talk to them since it's just sending normal ethernet traffic, so no configuration is needed to get the machine to work with them. The modems themselves are pretty easy to set up as well since it's usually just hitting an imbeded web site (think 192.168.1.1) and putting in the info for your ISP. You also get the added benifit of being able to run multiple computers or a whole network on them.

Oh yea, you can find the  Actiontec GS002AN201 on newegg.com. They have a few other ethernet modems on there if you don't like that one.
