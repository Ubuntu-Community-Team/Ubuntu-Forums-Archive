### Post by pytheas22 on 2008-04-14
Yeah, this should go way beyond the requirements to run Ubuntu with desktop effects.  If my Pentium 4/512 megabytes/onboard Intel 810 graphics card can do it, a machine with many more times the processing power and memory, and an nvidia card, should handle it without a problem.

Keep in mind that you'll need to install the nvidia closed-source drivers before you can run desktop effects; if it's now obvious how to do that when you first boot to Ubuntu, search the forums for directions.
