### Post by Fyda on 2006-10-05
Indeed! In fact, an extra modifier key increases the number of keyboard shortcuts you can set. :)

I think by default, the WinKey gets mapped to "Super". Amarok is one app that takes advantage of the Windows key by having its default global keyboard shortcuts set to "Win+Z" (Previous), "Win+X" (Play), "Win+C" (Pause), etc. (If you are familiar with Winamp 3's global shortcuts, the layout will be very familiar.) I'd argue that having that extra modifier key is better than trying to devise combos like Ctrl+Shift+something, or (heavens forbid) Ctrl+Shift+Alt+something, just to avoid a conflict with an existing shortcut. :D

But then again, I don't know. Maybe some people just really don't want that extra key where they expect Alt or Ctrl. It's totally their choice. Me, I'd find an Ubuntu sticker and make it into the Ubuntu Key instead! (Or maybe the Tux Key.) :D

PS. Sorry I can't give any useful advice on a laptop model -- I myself am looking for one, for university use and with decent Compiz/Beryl functionality, as well!

PPS. My friend uses a Toshiba Satellite of some sort, and while it does have a Windows key, it has only *one*, and it's placed in an inconvenient position at the top-right of the keyboard layout (meaning that most Win/Super combos have to be two-handed). If the user doesn't want a Windows key, then this might be an okay-ish compromise -- having a Windows key, but not having it get in the way.
