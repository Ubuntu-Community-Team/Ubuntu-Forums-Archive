### Post by Syke on 2007-06-12
Flashing a bios is a dangerous procedure. Any mistake can render the system quite dead.

Usually you need to create a DOS boot disk and run a flashing utility from the manufacturer using a bios image, also from the manufacturer.

Here's a good howto on creating a boot disk: [http://gentoo-wiki.com/HOWTO_Create_a_DOS_boot_disk](http://gentoo-wiki.com/HOWTO_Create_a_DOS_boot_disk)
