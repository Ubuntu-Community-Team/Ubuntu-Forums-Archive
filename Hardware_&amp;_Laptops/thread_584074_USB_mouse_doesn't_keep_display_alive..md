### Post by Kappity on 2007-10-20
Just upgraded to Gutsy, and still have one issue that's left over from Feisty.

I'm using an IBM Thinkpad A20M, an older model, probably from around 1999 or 2000, although I don't remember for sure.  For a mouse, it has the IBM trackpoint, which works pretty well.

However, I had an extra Apple USB Mighty Mouse lying around, so I plugged it in.  It works fine in most ways, right click, left click, scrolling, smooth cursor movement, etc.

One odd thing though.  Input from this mouse doesn't seem to count toward keeping the display awake.  I'll  be using it, and the display will suddenly blank.  Clicking the mouse doesn't bring it back.  I have to use the trackpoint or the keyboard to wake up the display again.  Not a big problem, but mildly annoying.  Anybody else run into this?

If I'm using the trackpoint instead of the mouse, or using the keyboard, that keeps the display alive.

Incidentally, disabling the screensaver or changing the power management settings doesn't prevent the display from blanking or going to sleep after just a couple of minutes.  That's a subject for another thread.  I mention it just in case somebody was going to suggest that as a solution.  There seems to be Thinkpad power management stuff that operates independently of the OS.
