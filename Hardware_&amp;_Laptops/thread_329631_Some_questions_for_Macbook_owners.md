Hello everyone---

I've spent this past deliberating which notebook I should get. I'm basically split between three options. 

Option 1: Get a Toshiba A105 (160gb, core 2 duo, card reader, s-video, etc) - $850
Option 2: Lenovo Thinkpad T60 (core 2 duo, 60gb, 512 ram) ~ $1100
Option 3: Macbook (core duo, 60gb, 512 ram) ~ $730

The toshiba has the whole works---tons of ram, hard drive space, a card reader. However, it doesn't feel too compact and I'm skeptical of the build quality. 

The thinkpad has great build quality, excellent linux support, and good weight. Simply too expensive though.

Which leads me to the macbook. I can get the first revision (core duo, not core 2 duo) for $883 w/2-yr purchase of applecare. I could probably sell the applecare on ebay for $150 so in the end, it'd be around $730 out of my wallet. 

I like the macbook's build quality, it's size (almost feels like a light book), and the ability to run all three major operating systems on it. The only gripe I have is that it only comes with 512mb RAM and upgrading it involves removing both sticks.

But anyways, here are some questions for macbook owners with ubuntu:

1) Since there's no right click button, one would need to bind another key to achieve this functionality. Is this easy to get used to after awhile or still a nuisance?

2) I've heard people have had heat issues with the core duo Macbooks with ubuntu. Have these problems been solved?

3) Do suspend, backlights, and any other hardware specific features work just as well in ubuntu as in osx?

Thank you
