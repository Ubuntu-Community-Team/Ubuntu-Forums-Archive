### Post by stinkypudding on 2006-09-24
I'm running Ubuntu on a B120, have no issues at all.  Even running Compiz on AIXGL, with the integrated graphics card.   The only issue I had setting up was with the Broadcom wireless card.  I used ndiswrapper to set it up.  Took me about an hour of research to get it set up correctly. There are plenty of helpful posts here to set up the wireless connection.

And I had never used linux before setting it up on this laptop.
