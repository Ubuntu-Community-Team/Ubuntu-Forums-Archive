### Post by Ricky_G on 2007-07-13
i own the Blue Snowball and tested it in Ubuntu using Audacity.  it does record, however it's a bit quiet.  Blue does have two different firmware updates to address volume.  one is for high gain and the other is for low gain. i mainly use mine to record band practice so i use the low gain firmware, which is probably why mine was quiet when i tested it.  if you're doing voice stuff you probably want to install the High gain firmware, im pretty sure that it comes with low gain out of the box.  problem is those updates are made for Windows.

the firmware can be downloaded here: [http://www.bluemic.com/update.php](http://www.bluemic.com/update.php)

### Post by cantubus on 2007-07-29
Do you mean USB microphone interface? Because numerous USB interfaces work in Ubuntu, including microphone. I tried my Snowball last night in 7.04, plugged it in, and in Audacity I selected it as an input source. Worked right away. I had previously updated the Snowball firmware for high gain under Windows, but since the firmware is stored in the hardware of the Snowball, it still works once connected to Ubuntu, so I was able to make excellent quality recordings. Next step is trying to get it to work in Ardour. Since all Ubuntu applications recognize the Snowball now, I can use it as a quality input for Skype, Gizmo, Ekiga, and other voice communications software.
