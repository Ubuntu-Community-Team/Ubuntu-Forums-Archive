### Post by dfreer on 2007-07-17
Well, in day to day task you are not going to see any difference really, as Ubuntu isn't that RAM heavy. It would increase your performance if you find you are maxing out your RAM, you can check by using the Gnome System Monitor and see how much RAM you are currently using.
