### Post by qrdl on 2007-05-04
You should check the xorg.conf

1. Find the section wuch corresponds to your videocard and check if Driver is set to "fglrx" (also check the xorg's log file. I guess there are some lines saying that it falls back to AIGLX)
2. If so you should turn off the AIGLX support, enter the foolowing lines in the end of xorg.conf

Section "ServerFlags"
          Option        "AIGLX"  "off"
EndSection

### Post by qrdl on 2007-05-04
Well the second videocard (the vesa one) does not affect anything. Look at the SreverLayout section - it uses screen with configured ati driver. You can safely leave it. The reason why you don't have direct rendering is that aiglx does not support fglrx, so just turn it off. In future if you decide to install beryl/compiz you should use xgl.
