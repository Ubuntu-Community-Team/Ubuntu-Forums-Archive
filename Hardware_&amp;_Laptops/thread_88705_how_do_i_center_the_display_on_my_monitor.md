### Post by michael_salcher on 2005-11-10
1) to adjust your monitor you should probably use the buttons on your monitor.
2) to execute commands as root you have to type "sudo foo" where "foo" the command that you want to execute. you can get a root shell by typing "sudo -s". remember, the password to use here is YOUR password. if you want to have a root password, you need to run "sudo passwd".
3) to install the nvidia driver, read [http://ubuntuguide.org/#installnvidiadriver](http://ubuntuguide.org/#installnvidiadriver)
