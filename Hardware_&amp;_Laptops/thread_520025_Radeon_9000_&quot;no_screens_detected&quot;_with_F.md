### Post by w4ett on 2007-08-07
The current fglrx  proprietary driver only works with the ATI 9500 cards/chipsets and above.  This is due to the fact that ATI has chosen to not support their legacy cards up to that chipset with linux drivers.  You will have to remove the modules from the attempted fglrx installation and use the xorg-driver-ati.

### Post by w4ett on 2007-08-07
Try this:
```

sudo apt-get remove xorg-driver-fglrx
```

then

```
sudo dpkg-reconfigure -phigh xserver-xorg
```

Select ati as your driver, and restart x. <ctrl><alt><backspace>
