### Post by az on 2006-02-09
A soft modem is a software modem - the manufactureres do not release the specs so that people can write open source drivers for it.  The manufactureres often release linux drivers themselves, but this is not ideal and so they are not included in the default install.

Visit linmodems.org, download and run scanmodem to see whan kind of modem you own.  There probably is a linux driver for it that you can get working.
