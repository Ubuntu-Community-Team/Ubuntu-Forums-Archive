Acer TravelMate 2480 Laptop Guide
This laptop works good under Ubuntu 7.04. The 3 common problems i've noticed with this laptop is the sound, video card, and wireless card. 

Sound:
Intel HDA
Driver Status: Installed and Enabled
Fix: 
Go to the mixer and turn on all devices and max the volume on 3D, Center, and PC Speaker.

Video
Intel 945
Driver Status: Installed and Enabled
Fix:
Resolution. Not really a huge problem but if you want your display to be clearer then install the 915resolution package. OpenGL also works quite nicely on this display.

Wireless:
Broadcom 4318
Driver Status: Not Installed and Disabled
Fix: 
You need the following: Ndiswrapper, and the windows drivers for your card (Acer Website) OR the script (somewhere on these fourms)
On the fourms there is a good guide on how to install them. I don't exactly know how to install them (I used a post on the fourms to install mine)

*Driver Status based on a fresh install of Ubuntu or Kubuntu.
