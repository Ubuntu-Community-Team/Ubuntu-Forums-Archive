### Post by geraldm on 2007-08-01
For the hardware problem portion, please check your BIOS options prior to booting
into Ubuntu.  See if there are options that you might be able to check/uncheck.
After booting, if there is a touchpad, try reducing the sensitivity.

You should be aware that there are applications that use strokes or gestures on the 
mouse to replace clicking.  It is unlikely that this kind of software would be installed, but
it is possible.  Such an application would work in concert with the window manager.

I have seen another thread mentioning the same problem, so I suspect the problem is there.
That was reported on feisty, with a notebook, I do not remember the manufacturer.
I would suggest looking around a bit, but file a bug report on the behavior if you do not 
encounter a solution or another similar bug report.  

Gerald
