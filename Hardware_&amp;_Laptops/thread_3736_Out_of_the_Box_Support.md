### Post by claymore on 2004-11-09
I know this doesn't really answer your question exactly, but the best way to make absolutely sure that things will work out of the box is by trying out the live CD.  It will give you a chance to load everything up, browse around the software a bit and make sure that everything is working well.

If you have a couple things that don't work perfectly, you can then come back and find out how to fix those problems before you are stuck in an installation that only mostly works.  With the how-to guides and package managers, you will probably find that you can get a lot of things working without resorting to that much manual editing.

Of course, some command-line work and manual configuring is a part of the linux experience.  With the ubuntu community, though, you will be able to get the help you need to find out HOW to do that configuring quickly.

If simplicity and hardware configuration is your biggest goal at this point, you may want to give MEPIS a try as well.  I don't like it as much as Ubuntu, but I have never had simpler install and configuration process before.
