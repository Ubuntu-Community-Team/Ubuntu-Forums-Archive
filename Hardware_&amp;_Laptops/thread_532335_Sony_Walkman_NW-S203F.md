### Post by redPirate on 2008-06-29
NW-E00X-Mp3 File Manager is the way to go.

You can get it from here:
[http://nwe00xmp3man.sourceforge.net/](http://nwe00xmp3man.sourceforge.net/)

The program requires Java to be installed so please install it first.

To run the program open a shell and type

$ java -jar [the full path to the NW-E00X....jar file]

It will provide you a GUI to do the transfers from local hard drive to MP3 player.
