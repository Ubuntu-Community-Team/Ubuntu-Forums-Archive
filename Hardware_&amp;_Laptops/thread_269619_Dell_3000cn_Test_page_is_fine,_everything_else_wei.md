### Post by mgmiller on 2006-10-02
Make sure you are setting up the printer correctly in Open Office.  It uses it's own printer settings.  Run the word processor and under the File menu run the Printer Settings.  Check that the settings for your printer are correct.  I believe it defaults to A4 paper, even if you have your system wide printer settings set to US letter, for example.  Make sure all the other settings for paper type and etc. are correct as well.

Good luck.
