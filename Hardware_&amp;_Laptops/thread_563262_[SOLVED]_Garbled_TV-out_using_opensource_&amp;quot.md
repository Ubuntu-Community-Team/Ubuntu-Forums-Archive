### Post by martin28 on 2007-09-30
Hi,

I have the same problem.  Unfortunately I don't have an answer for you.  The reason it doesn't work is because the Radeon driver does not support TV-out in feisty.  The ATI driver does support TV-out, but not for the 9200SE card.  So you're kinda screwed.

I've spent ages trying to find a solution as I have the same set-up.  The closest I came was a while ago when I found a page where someone had made some changes to the source code of   the Radeon driver and recompiled it to get TV-out working on Feisty.  The solution was way over my head and now I can't find it again.

You basically have three choices: switch to an earlier version of ubuntu, switch to a different video card, or learn enough linux to actually change the code of the radeon driver yourself.

Sorry, that's all I know.

Martin.

### Post by martin28 on 2007-09-30
actually, maybe i do have some good news.  seeing your post made me decide to have another go at this.  i found a post that worked for me.

[http://ubuntuforums.org/showthread.php?t=215763](http://ubuntuforums.org/showthread.php?t=215763)

this post tells you to go to a wiki that guides you through the set-up.  ithe step-by-step instructions are at:

[http://wiki.cchtml.com/index.php/Tvout](http://wiki.cchtml.com/index.php/Tvout)

it worked for me.  

a few things to point out:
what this does is it patches the open-source radeon driver so that it can do tv-out.  the way tv-out works is that it only supports one resolution which is 800x600.  if the resolution is anything other than 800x600 the tv is just blank.  however, for some stupid reason my primary monitor doesn't like 800x600 with tv-out enabled (without tv-out it is fine).  

so i changed the following lines in my xorg.config file:
Section "Monitor"
	Identifier	"SAMTRON"
	[B]HorizSync 30 - 81
	VertRefresh 56 - 75
[/B]	Option		"DPMS"
EndSection

I picked those values because i looked up my monitor documentation.

Then i changed the line:
SubSection "Display"
		Depth		24
		Modes		"1280x1024" "1152x864" "1024x768" "832x624" "800x600" "720x400" "640x480"

This puts it back to the original.

OK - so what does this do?

Well, allowing a bigger range of horizontal refresh rates also allows higher resolution on my primary monitor.  putting that second long list of resolutions back in does the same thing.

So i boot into a bigger resolution: 1230x1024.  this is good for working and internet etc, but the tv is black.  when i want to watch tv i go to the menu to change screen resolution.  system>preferences>screen resolution

i set resoltuion to 800x600 and my primary monitor looks awful and badly centered and i am missing half the screen etc etc, but the TV output is fine!  so then i watch a movie and when i'm done i change it back to a bigger resoltuion which kills tv-out but at least i can work on my primary monitor.

there's also a little tool to re-center the output and i know if i played around enough with my priomary monitor i could get it to display 800x600 correctly but i'm not going to do that now.  

instead i'm going to watch some movies!  :-)

hope this helps.
