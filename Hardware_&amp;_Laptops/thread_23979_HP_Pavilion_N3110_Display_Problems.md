### Post by t4m3 on 2005-04-04
Well, I fixed my own problem.  This time when I was running the dpkg-reconfigure xserver-xorg I selected the advance display set up as opposed to the 'medium' which I was selecting before.  Appearently those default values helped straighten out my monitor and its working perfectly now.
