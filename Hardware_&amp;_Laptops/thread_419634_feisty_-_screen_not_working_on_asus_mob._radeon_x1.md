### Post by chefweb on 2007-04-23
First you have to install the fglrx-package by doing: "sudo aptitude install xorg-driver-fglrx"
after that make as follows: "sudo dpkg-reconfigure -phigh xserver-xorg"
then select "fglrx"  .. after that, rebott. This should work...
