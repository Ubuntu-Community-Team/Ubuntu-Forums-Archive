### Post by frith on 2007-09-05
It sounds to me like you have your keyboard set up to use 'dead keys'.

For some international layouts, instead of a ' creating a ' character, it is part of a two key combination - eg  'c would produce a 'c' character with an accent on top.

This is the default mode for some layouts. To change it, you just go to System->Preferences->Keyboard then the Layouts tag and select 'U.S. English'. (You may have to add it with the 'Add button')

hope this helps.

Frith.
