### Post by mordred1 on 2006-11-23
Hi there!


I own a dell inspiron 9400 loaded with a geforce 7900gs and it appears i had the same problem as you.


I seem to have resolved it by using the nvidia-settings (see synaptic).
I set the refresh rate from auto to 60hz then I used the detect display button to re-detect my screen. After applying this, no more flickering for me.
Of course, I'll have to see if this works on the long run...

So in short:
1.Install the nvidia-settings
2.Go to X server display configuration.
Set refresh rate.
4.Detect Display
4Validate when prompted.
5.Apply

I hope it helps.
