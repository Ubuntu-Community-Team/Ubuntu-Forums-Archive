### Post by bgoody on 2005-11-28
Hi.  I have a dual boot system on a Toshiba and it will do that every time if I boot into Windows and then afterwards into Linux.  It seems that Windows speaks to the bios in a way that Linux can't.

The solution was to go into the bios setup and turn off the screen blanking there.  Once done, it works as long as you boot Linux each time.  If you go into Windows you have to reset it each time from the bios.
Brian
