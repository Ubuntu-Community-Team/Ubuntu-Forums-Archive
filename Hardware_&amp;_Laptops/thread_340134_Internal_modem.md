### Post by StefanoCole on 2007-01-17
Hello, go to the following link: [http://linmodems.org/]("http://linmodems.org/")
and download the scanModem utility. Follow the instructions to run it. It will generate a few files. One of them is called ModemData.txt, open it and read it carefully. It should contain information about your modem: model, brand, chipset and if a Linux driver is available for it.
