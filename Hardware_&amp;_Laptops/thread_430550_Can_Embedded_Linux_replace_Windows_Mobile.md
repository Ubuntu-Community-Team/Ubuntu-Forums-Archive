### Post by jjordan on 2007-05-03
Umm, no, well  maybe

There are several projects to put together a replacement for Windows Mobile on various hardware.  This is not an easy process.

There are some Embedded Linux based solutions for specific applications (the Tivo runs on Linux).

Do a web search for "Linux" and the mobile device you have to see the possibilities.

If you are looking for a PDA/ mobile phone and can wait a few months look at the OpenMoko project using the NEC1973.  This is an iPhone killer based on open source and a hardware platform built from the ground up to support it.
