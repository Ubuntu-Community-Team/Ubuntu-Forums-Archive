### Post by ramjet_1953 on 2007-05-09
I'm afraid that you have the least Linux friendly brand of printer available. 

Lexmark are notorious for being completely closed with their hardware and drivers, making it almost impossible for open source drivers to be written for their printers.

If you follow this link, you will see what I mean:

[http://www.linuxprinting.org/show_printer.cgi?recnum=Lexmark-Z611](http://www.linuxprinting.org/show_printer.cgi?recnum=Lexmark-Z611)

However, if you have an old PC lying around, you could always install WIndows on that and then use it as a print server for the Lexmark.
