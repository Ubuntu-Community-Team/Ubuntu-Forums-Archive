### Post by djdicbob on 2006-07-04
Hello ...  I have run into this problem numerous times with other flavors of linux and windoze...In my expeirence the issue is with the BIOS....some BIOS versions pass control to software or continue to use firware control of the power features.  

This is were it may be hard to pin point..some BIOS also have performance settings for processors that can be dependent upon variables like ac power settings and dc power settings.   

I found that if I turned off any processor maximizg features the fans quited down and turned off when I used suspend.  

I am not sure of the difference between suspend and hibernate are, but that could be something to look into as well!
