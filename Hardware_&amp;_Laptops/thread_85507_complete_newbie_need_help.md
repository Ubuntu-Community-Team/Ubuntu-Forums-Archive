I'll make the assumption that you downloaded the file in Windows, so you'll need this [util.]("http://www.pc-tools.net/win32/md5sums/")

### Post by dbott67 on 2005-11-03
Keep in mind that not every CD/DVD player is the same.  Some CD players are much better at reading data from 'burned' CDs, where others have some problems.

Check to make sure that there aren't any scratches, dirt, smudges, fingerprints or other crud on the CD that might make reading difficult.

The quickest solution would be to re-burn the ISO at 8x.

-Dave
