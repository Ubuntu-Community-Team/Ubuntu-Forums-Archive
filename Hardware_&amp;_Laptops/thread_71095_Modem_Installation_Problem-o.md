### Post by mlomker on 2005-10-02
> Annny way, I would appreciate any help given to me by anybody.


I'm assuming this is a software modem, aka windows modem.  The more expensive modems are 'real' modems and don't require a software driver, but these days most of them are software (DSP's).

Go to [linmodems]("http://www.linmodems.org") and download their modem detection script.  It'll tell you which driver to try.  You might have to spend $15 to buy a driver from [linuxant]("http://www.linuxant.com"), worse case scenario.
