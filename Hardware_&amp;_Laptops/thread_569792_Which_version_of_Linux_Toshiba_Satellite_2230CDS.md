### Post by jnorthr on 2007-10-07
Look here for ideas: [http://www.xubuntu.org/](http://www.xubuntu.org/)

You may need to use the alternate CD to install kubuntu as it uses a text terminal screen and is better to install when system memory is small. When installed, xubuntu will give you a graphical gui to work with. Be sure to make your swap partition big (600MB-ish) as the swap file will be used a lot when running big programs.

Good luck - your grandfather may fly yet   :KS
