I have a Fujitsu-Siemens-E8010-Laptop. I had to set the microphone-device to get the micro working.Here You find my setting at system-pane-menul - from German:

System / Settings (Einstellungen) / Audio /
Devices (Geraete) / Audioconference / Intel 82801DB-ICH4-MIC ADC

Hope this information will help some users of skype.
