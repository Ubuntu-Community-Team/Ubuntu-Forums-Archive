### Post by mowglix on 2007-03-15
I had the same problem trying to get Ubuntu 6.10 to use the 1280x768 resolution instead of stretching a non-widescreen one (1024x768).

I have an ATI Mobile Radeon X1600 card so I went to ATI's homepage to download the drivers and installer script. Installed them and before running aticonfig I backed up the original Xorg.conf file, then I typed the following in the terminal window:

sudo aticonfig --initial --lcd=center --resolution=0,1280x768 ,1024x768

Finally I restarted Gnome and tears flowed from my eyes when I saw that it worked :guitar: 

I hjope this is helpful for someone

### Post by Ingo88 on 2007-03-30
Really seems like you have been trying a lot :)

Here's the Device section from my xorg.conf. I guess you just have to change the Centermode option from "centermode" to "on". This is the only place in my xorg.conf where the Centermode option is present.

```
Section "Device"
	Identifier  "aticonfig-Device[0]"
	Driver      "fglrx"
	Option	    "VideoOverlay" "on"
	Option	    "OpenGLOverlay" "off"
	Option	    "Centermode" "on"
EndSection
```
