I would think the solution would be to open a terminal and type -
sudo gedit /etc/fstab   - and change the line
/dev/sda2 /media/Storage ntfs nls=iso8859-1,users,umask=000 0 0
to 
/dev/sda2 /media/Storage ntfs nls=iso8859-1,umask=000,ro 0 0
to match the ntfs partition that is getting mounted. :)
