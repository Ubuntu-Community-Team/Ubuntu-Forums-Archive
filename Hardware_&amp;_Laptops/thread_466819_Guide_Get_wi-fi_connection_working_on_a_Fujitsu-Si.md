### Post by bjarte on 2007-06-07
This is a guide specifically for Fujitsu-Siemens laptops with wi-fi card **Intel Pro/Wireless 2200BG** and BIOS version **R01-S0X**. The explains how to upgrade your BIOS to version **R01-S0Z**. The newer version of the BIOS allows you to turn on the wi-fi card in the BIOS settings, rather than using Windows software, which is the default.

This guide is based on my **Fujitsu-Siemens Amilo Pro V2000**, but should work on other laptops with the same wi-fi card and BIOS version.
