### Post by bmt on 2006-09-13
The 3490 is supported by the included sane in Dapper -- mine worked out of the box (Epson Perfection 3490 Photo).

If you have the 3490 document feeder, I think you need the Epson-provided gt-f520 plugin, but I don't have that hardware (document feeder).

Reference: [http://www.sane-project.org/sane-supported-devices.html]("Sane supported hardware") then go to the Search Engine.

### Post by noynac on 2006-09-13
I have an Epson 3490, and it works great. I did not install iscan-plugin-gt-f520. However, I did have to put a esfw52.bin firmware file on my hard drive and modify a config file to reference this file. Do a search of this forum on esfw52.bin and you will find instructions.
