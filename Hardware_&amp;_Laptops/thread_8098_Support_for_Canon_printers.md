> 
Section 23: Canon S200

23.0 See also the Canon S200 database page.

23.1 Is this printer supportted in free software?

    Maybe. You can try BJC-600 drivers without any color options: you should print only in grayscale at 360 dpi. Margins should be adjusted. You can use "alignmarins" or XPP.
23.2 Is this printer supported in commercial software?

    Yes. TurboPrint.
