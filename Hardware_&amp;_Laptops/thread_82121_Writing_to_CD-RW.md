### Post by drummer on 2005-10-25
Did you only drag the file onto the blank and eject?? you have to tell nautilus to burn the cd once you've put all the files you want on there.  In the nautilus window displaying the cdrw (Go > CD/DVD Creator) click "File > Write to disc..." and you'll get a cd write dialog that has some options and then writes the cd.
