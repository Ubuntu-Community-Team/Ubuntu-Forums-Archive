So my assumption is that the hardware test is ignoring the headphone jack.

So, given this, I figure if I can get linux to ignore the headphone jack, my internal speakers should work. I've checked alsamixer and kmix and neither seem to have a "Headphone Jack Sense" option. Any help would be appreciated.

### Post by moeFinley on 2008-01-07
The headphone jack sense option will only appear if the driver/hardware you have carries this feature.  Most cards don't.  You would think it makes sense they all have this option as all sound card detect when headphones are plugged in.
