### Post by Arktis on 2005-06-19
I've got a Radeon 9600 AGP card with the x.org v8.14.13 drivers installed and an Nvidia GeForce4 MX 4000 PCI card in the same P.C. Both have their own monitors obviously, and both have worked under Hoary. The problem I'm having is in getting them to play together as a dual setup so I can have a nice large desktop area.

With a fresh install, Hoary didn't like my 9600 card but set up my GeForce for me, so that my computer would end up using the 9600 during boot (as normal) and then switch over to the GeForce once X started. This would leave the previous monitor with a depressing black screen while I could still use the computer with the other one. So, after searching the forums, I set up the ati drivers and then set about mucking around with my xorg.conf. The result was that I got the 9600 working with X, but now the other display, the GeForce, has no picture at all. I've never really done this before, so I was hoping somebody might point me in the right direction.

I read something about TwinView and Xinerama but I'm pretty clueless as to how they work and how to set them up. TwinView is from nvidia, so it probably only works with their cards, so I assume Xinerama would be the way to go. I searched for it in synaptec and saw that I have libxinerama1 installed, but that's it...
