### Post by wthanna on 2008-01-14
Actually, a virtual machine is a great way to try out Ubuntu.  You won't be able to really check out the fancy 3D effects offered by Compiz or Beryl, but you will be able to add and remove programs, etc. and try them out for a period of time, and more fully explore Ubuntu than you would be able to using the live CD. Using the live CD, everything is gone as soon as you reboot (custom settings, installed programs, etc.) since you are running the entire operating system from a CD.  It also runs quite a bit slower than a virtual machine does.  Running from a live CD, programs have to be uncompressed as you run them, which brings quite a performance hit along with it. Once you have VMWare or another virtual machine program installed on your system, you can easily try out MANY linux distributions... other versions of windows, etc..  Virtual machines are awesome, actually..  My production mail server (CITADEL) which has several dozen very active users runs in a virtual machine full time, as well as an old Novell Netware server which used to run on old hardware. Virtual machines are great! I have no idea why anyone would discourage you from giving it a try.
