Did you happen to run "dpkg-reconfigure -phigh xserver-xorg" before?  You have a recent backup copy of your xorg.conf, xorg.conf.20080310220131.  Is that the original, working version?  If it is, you can use it again with

```
cd /etc/X11
sudo mv xorg.conf.20080310220131 xorg.conf
```
Then, either reboot or restart X server again with <Ctrl><Alt>Backspace.

### Post by taurus on 2008-03-10
Try the next one.

```
sudo mv /etc/X11/xorg.conf.1 /etc/X11/xorg.conf
```
Then restart X, <Ctrl><Alt>Backspace.

And if that one doesn't work, try the next one.

```
sudo mv /etc/X11/xorg.conf~ /etc/X11/xorg.conf
```
Then restart X, <Ctrl><Alt>Backspace.
