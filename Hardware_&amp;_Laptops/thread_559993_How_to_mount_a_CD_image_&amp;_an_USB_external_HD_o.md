### Post by logos34 on 2007-09-30
Not sure why the usb drive partition isn't automounting at bootup.  You could check system>prefs>removable drives and media>storage tab to see if "mount removable drives when hot-plugged" is enabled--maybe it's conflicting, so untick it.  Or add the UUID to fstab:
[B]
blkid /dev/sdb1[/B]

see if that helps.

For the ISO, try:

> /windows/"Documents and Settings"/Myself/"Mis documentos"/freudylacan.iso /media/psicoanalisis iso9660 loop,auto 0 0
EDIT: I just tried the same thing with an .iso on my ntfs partition and it doesn't want to mount, although it works with an .iso on my linux partition.  So either there's a problem with the path or ntfs-3g.

**Why not simply create a folder and copy the contents of the .iso to it?**
