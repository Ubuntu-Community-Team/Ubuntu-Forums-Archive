 from what it seems you are doing, you are running two screens and still trying to use TwinView.  This is impossible.  To use twinview, you need one (1) device section, one (1) screen section, and one (1) monitor section.  You add the 

```
Option "TwinView"
Option "TwinViewOrientation" "RightOf"
Option "Xinerama"
Option "Metamodes" "1280x1024,1280x1024; 1024x768,1024x768; 800x600,800x600"
Option "SecondMonitorHorizSync" "yourvalues"
Option "SecondMonitorVertRefresh" "yourvalues"
Option "RenderAccel"
Option "HWcursor"
```
