### Post by rdoolen on 2008-01-29
I run Gutsy on A P5010 and everythng works fine. Do you have 915rsolution installed? If not, you need to install it. 

I have periodically had video issues. I would recomend NOT using "Screens and Drivers" from the admin menu, that seemed flakey. Also, when you do get the video set properly, save a copy of xorg.conf. If you see a problem restore your backup.
