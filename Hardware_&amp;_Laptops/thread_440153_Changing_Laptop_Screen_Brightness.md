### Post by TomFumb on 2007-05-14
If you're using Ubuntu Feisty (I'm not sure about edgy, dapper, hoary, or xubuntu / kubuntu...) go to System / Preferences / Screensaver, then click on the 'Power Management' tab, there you can set the brightness defaults for different power states
