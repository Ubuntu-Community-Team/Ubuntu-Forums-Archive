### Post by Yellow Pasque on 2008-02-24
ardnew,
Generate the modeline with this:
```
cvt 1360 768 60.015Hz
```
Paste the result into the Monitor Section. Get rid of those other mode lines.

```
SubSection "Display"
        Modes        "1360x768"
    EndSubSection
```
-->
```
SubSection "Display"
       Depth         24 
       Virtual         1360   768
       Modes        "1360x768"
     EndSubSection
```
