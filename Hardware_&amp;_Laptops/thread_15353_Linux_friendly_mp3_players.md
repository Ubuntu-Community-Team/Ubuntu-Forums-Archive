### Post by ianbillmorris on 2005-02-15
I use an IRiver IHP120 (Now called an HP120) works as USB mass storage but does have a database you can use as well.  There is a java database manager avalable from [http://www.shredzone.net/projects/ifish/](http://www.shredzone.net/projects/ifish/)  but you don't actually need it as you can just browse around the filesystem and play MP3's from there.  (It also supports OGG!)

The newer IRiver Mp3 players do not seem to be USB Mass Storage complient although IRiver release UMS firmware some time after the product goes on sale for people who do not want to use there manager software it is a bit of a lottery so make sure there are UMS drivers avalable before you buy.

I've been using the Creative player for a week and a half now and it's fine.  I use it exclusively in my car (currently via cassette adapter until I upgrade the stereo later this year) so I can't comment on things like headphone performance.  Works great for me.
