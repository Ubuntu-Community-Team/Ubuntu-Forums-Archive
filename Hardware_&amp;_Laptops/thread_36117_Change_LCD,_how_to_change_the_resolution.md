In order for the Screen Resolution settings to give you the option of 1280x1024 - that resolution size has to be available for use in your /etc/X11/xorg.conf file - 

Could you paste the sections in your /etc/X11/xorg.conf file labeled: "Monitor" and "Screen" here so that I may see them?  I'm guessing that those two sections are preventing you from using the higher resolution.

(you can do this easily by opening a terminal window and typing  ```
gedit /etc/X11/xorg.conf
``` )
