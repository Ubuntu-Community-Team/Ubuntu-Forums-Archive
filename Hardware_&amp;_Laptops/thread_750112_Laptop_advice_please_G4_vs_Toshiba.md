### Post by Rocket2DMn on 2008-04-09
There is PowerPC support for Ubuntu, but you will probably get better support with the Toshiba laptop.  For half the price, it's going to be a better deal than the Mac, and since you're not interested in the Mac part anyway, you might as well skip it altogether.
You will get better support on other distributions using an i386 install (standard PC, not apple type) instead of PowerPC (older Apples without intel processors).
I'm not sure if iTunes recognizes FLAC, but since I suggest the Toshiba, it's not a problem for :)
I'm not really sure on the speed comparisons, but a 2.4GHz Celeron is not bad.  Remember that with Apple, you are paying a lot for the name and what I call "the shiny factor" - I think you can figure that one out.

So long story short, I recommend the Toshiba because you will receive better support for it in linux, it is a decent older laptop, and is much cheaper than the Apple - it's perfect for what you want to use it for.
