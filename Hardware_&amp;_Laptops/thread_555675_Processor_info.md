### Post by dabl on 2007-09-20
I think you're right on the Core Duo CPUs.

Here's Intel's main processor data site: [http://www.intel.com/products/processor/index.htm](http://www.intel.com/products/processor/index.htm)

To be a 64-bit processor, it has to have the "EM64T" feature.  All the dual cores EXCEPT the Core Duos advertise that feature, as far as I can tell.  None of the "Core Duo Desktop" CPUs appear to have the EM64T feature.
