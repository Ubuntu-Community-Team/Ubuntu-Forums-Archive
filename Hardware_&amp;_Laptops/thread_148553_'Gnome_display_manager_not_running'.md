### Post by Mustard on 2006-03-22
On a desktop machine I would suggest that you manually shrink the window using the display control buttons on your monitor that control alignment/size/warping of the screen, as well as brigthness, contrast etc, but I have no idea whether you have these on a laptop.

The 'Gnome display manager not running' message at shutdown sounds normal to me, if I recall my shutdown correctly.  If you have a gui, which you do, then GDM is definitely running.
