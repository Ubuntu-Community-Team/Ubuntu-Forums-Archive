### Post by jml on 2005-12-12
I agree with everything that has been written so far.  Ubuntu is a great distro for laptops.  It seems to support a huge amount of hardware.  Since you want to keep the price at about $1,000.00 you may have to make a few compromises.  Taking stock of just what you plan to do with your laptop will go a long way in making for a pleasant experience.  If you are into gaming, then you want a unit with accelerated 3D graphics ,(either ATI or Nvidia).  If you want to work/play with multimedia files (photos, video, audio) get a laptop with as big a hard drive you can afford.  If you plan to connect either a digital camera, or multimedia player to the laptop, try to find one with USB2 ports rather than the original USB 1 ports, they transfer files much faster. 

And now for the big one.  Wireless (Wi-Fi) access.  If you plan to use the laptop to connect to a wireless network, do your homework very carefully.  If all you need is unencrypted wireless access, pretty much any modern laptop will do.  If you plan to connect to wireless networks using WEP encryption, then you should be ok as well.  Ubuntu's networking tool supports WEP keys.  If you need to connect to a wireless network using WPA, (a more robust encryption tool) then you have to find out what the actual chip set the laptop you are planning to buy uses.  WPA requires the use of an application called WPA_Supplicant.  It does not support all wireless chip sets.  My own painful example is my Averatec 3250 laptop.  A pretty good unit all around, very compatible with Ubuntu,except for the wireless.  I need WPA encryption and the chipset that is installed (rt2500) is not supported.  Getting my connection up is a real pain.  So the only caution is the wireless.  If you don't need or plan to use wireless, then you should be home free.
