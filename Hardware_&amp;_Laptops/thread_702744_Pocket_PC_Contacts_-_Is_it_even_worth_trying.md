### Post by oldsoundguy on 2008-02-20
Outlook= WINNDOZE

Linux and windows are two different animals.  Only way to sync with Linux is to FLASH a Linux program on to the PDA .. and that can get very difficult currently as there is only ONE that looks to work, and that is Familiar Linux.  (Flashing will remove Windows so you have to save that to your hard drive "just in case".)  (and flashing has to be done on a SERIAL PORT from a WINDOWS machine, not the USB .. requires installing the old Hyper Terminal and configuring the port.)

The use of the PDA in the US has a lot to do with it .. With the exception of Palm (and there is a sych program for Palm in Linux as Palm is Linux friendly) .. allmost all other PDA's are running some form of Windows Mobile (I have 2003SE on the two I have and I wish I could get rid of it as the WiFi is busted in the program and will not work.  And, of course, Windows blames HP and HP blames Windows and 70 bucks later back to point A!

The iDoinks from Apple run a really strange locked up Unix based program.

When the US finally catches up with the UK and Japan in PDA usage, you will see improvements.
