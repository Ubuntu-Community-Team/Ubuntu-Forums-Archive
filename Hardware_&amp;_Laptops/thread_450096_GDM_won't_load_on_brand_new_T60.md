### Post by lord bacon on 2007-05-21
kk first of i recommend downloading the alternate iso which gives a text based installer (the x1400 video card you have in your computer does not work with the default drivers supplied by ubuntu)

once you do this you can have a shot at installing Ubuntu in text mode. 
i would suggest install windows first and set up your partitions along the lines of 20GB for windows, 40GB for linux and the rest as free space, just format and install the windows partition and create the others (do not format them)
once windows is installed format the free space as NTFS, this can be done from windows
Then put in the alternate Ubuntu CD and install, the first options should be pretty easy to understand, when you get to the partition setup things get a bit more complicated, you want to choose manual setup, select the 40GB partition and then under partition type (i think) select automatically set up this partition, or something along those lines.
once it is installed and the system is restarted you should be greeted with options 
namely something along the lines of 

Ubuntu 2.6 xxxx - generic
Ubuntu 2.6xxxx -generic (recovery)
memtest 64
Other operating systems
(if you have other operating systems installed they appear here)

select the recovery version of Ubuntu
input your username and password
now type into the console

sudo nano /etc/apt/sources.list

this will give you a screen with text on it (this is a simple text editior), (
you will also notice that every now and again there is a string of text that pops up, this is because X-Server is still crashing i think, but either way you can just imagine it isnt there) 
Now remove the ## from the start of any lines that have a repository link to them, (an example is deb [http://security-XXXXX](http://security-XXXXX)    so on and so forth)
once this is done press ctrl X sot exit and select y to save.
now type

sudo nano /etc/X11/xorg.conf

and add the following lines to the bottom of this page

Section "Extensions"
        Option  "Composite" "Disable"
EndSection

Section "ServerFlags"
        Option  "AIGLX" "off"
EndSection

now exit and save

now type in 

sudo apt-get update
sudo apt-get install linux-restricted-modules-$(uname -r) # (Okay if it is already installed)
sudo apt-get install xorg-driver-fglrx
sudo depmod -a

(note each new line is a new command that you type in once the other is finished)

once this is finished type in

sudo shutdown -r now

[/EDIT]

go into recovery mode again and type sudo aticonfig --inital

sudo shutdown -r now

[/EDIT]

hopefully you should see the nice Ubuntu screen on your next boot (make sure you choose the first option to boot from this time, not the recovery version)

if this happens load up log in and have a look around, if you want to get your laptop to actually suspend use this page and method 2 of installation (slightly harder)

[http://wiki.cchtml.com/index.php/Ubuntu_Feisty_Installation_Guide](http://wiki.cchtml.com/index.php/Ubuntu_Feisty_Installation_Guide)

Would like to point out i used the above article and many others (that i cant find) that help me set up my laptop,to write this help piece

Enjoy and tell us how u did
