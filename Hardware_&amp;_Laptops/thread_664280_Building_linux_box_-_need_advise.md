### Post by Yellow Pasque on 2008-01-11
- That looks like a hot/noisy build. I would go with Intel for a high-power build like this, even if it is a bit more expensive. That CPU is going to run hot under load.

- Seagate 7200.10's run hot/loud. WD3200AAKS would be better, Samsung 321KJ would be best.

- The Thermaltake PSU will probably be noisy and TT is notorious for lying about their specs. Corsair 450VX, Seasonic S12-430, or Antec NeoHE-430 would be better options.
