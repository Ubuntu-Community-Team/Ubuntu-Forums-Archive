### Post by taurus on 2007-12-25
Try

```
sudo mkdir /media/hdc1
sudo mount -t ext3 /dev/hdc1 /media/hdc1
df -h
```
And if you want to write to it, you need to change the ownership of /media/hdc1 from root to your login name.  _Assuming_ your login name is **juice**, do

```
sudo chown -R **juice**:**juice** /media/hdc1
ls -la /media
```

And of course, you can add an entry in /etc/fstab so it would be mounted automatically each time you boot Ubuntu.  Edit /etc/fstab

```
gksudo gedit /etc/fstab
```
and place a # in front of this line since you don't need it anymore.

```
#/dev/hdc        /media/cdrom1   udf,iso9660 user,noauto     0       0
```
Then, add this line to the end of it.

```
/dev/hdc1     /media/hdc1   ext3   defaults   0   1
```
Save it and reboot.
