### Post by az on 2005-07-25
I would run the default desktop on anything from a 350 MHz processor or greater.  Your mileage may vary.  If you find it too slow, you can run a leaner (more bare) system by installing the XFCE4 package and running the XFCE desktop instead of Gnome, or install Icewm and run that.

If you run icewm, also install the "menu" package to make your life easier.
