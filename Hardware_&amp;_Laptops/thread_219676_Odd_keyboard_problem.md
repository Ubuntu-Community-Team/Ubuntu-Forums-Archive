### Post by Sef on 2006-07-26
> But after sometime the keyboard seems to behave oddly (Both in Ubuntu and Windows)

1) You have a bad keyboard.

2) You're port (either usb or ps/2) is bad.

Borrow a keyboard from someone and use the same port.  Does the problem still exist?  If not, then you have a bad keyboard.

If it does, switch ports and see if the problem still exists. If it doesn't, then the problem is the port.
