### Post by roncrump on 2008-01-27
Hi,

I've just bought a Wacom Bamboo tablet and am trying to get it working as fully as possible on my laptop, running 64-bit Ubuntu 7.10.

Followed the How-to to get the drivers installed and all went well, so I have basic functionality.

However, I have a couple of minor issues at present:

(1) it seems to have disabled the scrolling on my synaptics touchpad, it still shows up as enabled if I look in the touchpad configuration tool, but it isn't functioning.

(2) in Inkscape, it was suggested that enabling the input devices was the way to go. So I set each of the settings for the stylus etc to "Screen" as recommended on the Ubuntu wiki. However, when I try to use the calligraphy tool it does not do what I expect. Pressing the lower button on the stylus instead of activating the tool so that a line is drawn, drags the page around. If I disable the wacom input devices I can draw with the calligraphy tool as long as I hold down the left button on my synaptics touchpad rather than the lower button on the stylus.

(3) on the Ubuntu wiki page for Wacom tablets, there is a screen shot which shows a pressure sensitivity button in inkscape for the calligraphy tool. I don't see this. Does it only become visible if you have set pressure curve settings for the tablet? Or is it an inkscape version issue?

Thanks.

Ron.
