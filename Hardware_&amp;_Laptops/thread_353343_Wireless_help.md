### Post by citizenofnowhere on 2007-02-04
If you are unsure about your wireless card being supported, you could try running the livecd to find out. A live cd is basically all of the ubuntu default installation that can run directly off a cdrom - you can also use the same disk to install later if you choose.

You can get one at the main ubuntu download page, just look for the one that says "live cd" or "live disk". 

Hardware support over the last few releases has increased quite drastically. There are still a handful of wireless cards that don't work by default - broadcom ones I think. But if it helps, I have a plain old intel 2200bg wireless and it works perfectly. 

If it doesn't work with the livecd it doesn't mean it won't work at all. You can use a windows driver (yes, really :)) with something called ndiswrapper (you will find a very brief how to on using it [here]("http://www.beginningubuntu.com/dapper_tips.html#Using_Ndiswrapper_to_get_wifi_working"), and a more complete one [here]("http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_install_Windows_Wireless_Drivers_.28Ndiswrapper.29")

All the best!
