### Post by astyanax on 2006-03-10
It all starts in you x config file: /etc/X11/xorg.conf
Back it up first and then have at it.

There's 4 areas you need to change (NOTE: these settings may differ from yours.  My card is a NVIDIA Geforce2 MX 400, so I just the nvidia drivers, which you have to install separately):

1.) You have to add the device for your video card using the driver for your video card
```
Section "Device" 
   	Driver          "nvidia" 
   	Identifier      "Device[1]" 
   	Screen 1 
	Option 			"NoLogo"
   	Option          "TVOutFormat" "SVIDEO" #"Composite" #or SVIDEO etc 
   	Option          "TVStandard" "NTSC-M" #"PAL-G" #or NTSC etc 
   	Option          "ConnectedMonitor" "Monitor[1]" 
   	BusID           "PCI:1:0:0" #adjust using 'lspci' or cat /proc/pci 
EndSection
```


2.) You have to add another monitor:
```
Section "Monitor" 
	Identifier 	"Monitor[1]" #TV 
	HorizSync 	30-50  # this will be specifc to your TV
	VertRefresh 	60      # this will be specifc to your TV
EndSection
```

3.) You have to add another Screen for the monitor:
```
Section "Screen" 
   	Device "Device[1]" 
   	Identifier "Screen[1]" 
   	Monitor "Monitor[1]" 
   	DefaultDepth 24 
        SubSection "Display" 
               Depth 	16
               Modes 	"800x600" 
       	EndSubSection    
       	SubSection "Display" 
               Depth 	24 
               Modes 	"800x600" 
       	EndSubSection    
EndSection
```

4.) You have to alter your ServerLayout (the section in bold and italic):
```
Section "ServerLayout"
	Identifier	"Simple Layout"
	Screen 0	"Screen[0]" 
	***Screen 1	"Screen[1]" RightOf "Screen[0]" ***
	InputDevice	"Mouse1" "CorePointer" 
	InputDevice	"Keyboard1" "CoreKeyboard" 
EndSection
```

This should help you set up your TVr.
