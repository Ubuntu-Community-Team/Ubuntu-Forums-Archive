Simple installation with my new hardware. Hopefully this can help someone with similar hardware.

Compaq Presario SR2150NX [ HP TS-XN0512RS ]
Intel Celeron D 356 3.33GHz, 512MB DDR2, 120GB SATAI,DVD±RW, 10/100 LAN, Modem

Video
EVGA 512-P2-N757-TR Geforce 8600 GT 512MB GDDR3 PCI Express x16 Video Card

Download and Install Ubuntu 7.10 beta (full release is October 2007)
[http://releases.ubuntu.com/7.10/](http://releases.ubuntu.com/7.10/)
I use ubuntu-7.10-beta-desktop-i386.iso

Install Updates using the Update Manager

Install Restricted nVidia Drivers using Adept

Install Kubuntu-Desktop using Synaptic

Install Restricted items using Adept (gets you quicktime, etc.)
In Adept, search for Quicktime and install the related Gstreamer.
[https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)

Install Wine using Adept

Install games using wine (Cedega made me pull my hair out, just use wine.)

This has worked perfectly for me and completely blows my XP out of the water. It runs like butter (very good).
