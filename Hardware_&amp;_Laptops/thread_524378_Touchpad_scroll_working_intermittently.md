### Post by ddrichardson on 2007-08-13
I assume you restarted X after altering the xorg.conf - but if you do a "man synaptics" there are a load of values that it accepts that aren't in some of the configuration programs.

The one I imagine you need is the:
```
Option RightEdge "xxxx"
```

On my synaptics touchpad I've got it set at 7400, so it uses only the dedicated scroll area and not the main part of the touchpad.

Once X is restarted by the way, you can use synclient in a terminal to alter settings on the fly and test them out.

### Post by ddrichardson on 2007-08-13
Its because you haven't restarted X - logout then do a CTRL+ALT+BACKSPACE to restart.

Any options are added to /etc/X11/xorg.conf under the input device entry for the touchpad. But like I say you can check them first with Synclient.
