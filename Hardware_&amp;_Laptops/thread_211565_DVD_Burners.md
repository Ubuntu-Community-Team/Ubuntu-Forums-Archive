I'm using a DVD±RW NEC ND-4570A DUAL Layer 16X DVD-/+ RW. (Dual layer not tested yet). 
Theres a sticky thread called [Hardware Compatibiliy List]("http://www.ubuntuforums.org/showthread.php?t=183664"), which you might have a look at.

When I search that thread for burner and DVD i got these:
Lite-On DVD+/- Double LAyer DVD Burner 
Samsung SH-W163A SATA DVD Burner [OK after tweaking]

I checked the Linux Hardware Compatibility HOWTO (for linux in general), but could not find info about specific DVD burners, only this: ["Most, if not all, ATAPI and SCSI DVD-ROM and writable DVD drives are supported"]("http://www.tldp.org/HOWTO/Hardware-HOWTO/dvd.html")
