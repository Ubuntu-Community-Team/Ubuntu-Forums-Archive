### Post by newlinux on 2007-04-04
A 6200 can definitely do 1920x1080 (1080p). It might be a little hard to find the right modeline/xorg settings, but I know it's been done. That card supports even higher resolutions than that...  Heck, the myth wiki says people using a 5200 have gotten 1920x1080...  [http://www.mythtv.org/wiki/index.php/Configuring_HDTV](http://www.mythtv.org/wiki/index.php/Configuring_HDTV)


 If you are not using XvMC I'd recommend a 6200. the processor needed to run 1080p is more of the problem... i have a 6200LE 128MB on board PCIe fanless card that works geat, but I don't have a 1080p set so I don't use it at that resolution.
