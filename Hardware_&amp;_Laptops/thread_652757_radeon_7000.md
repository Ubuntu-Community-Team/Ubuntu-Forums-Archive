### Post by glug101 on 2007-12-31
I'm getting 100fps from glxgears.  I'm not on a laptop, but I am using the same chipset in a desktop computer (Mobile AMD Athlon XP-M 2800+, 512meg RAM, Radeon 7000/VE).  I'll post my xorg.conf at the end in case that helps.  However, this is what actually helped me:

```
sudo apt-get install driconf
```

After installing the program, run it:

```
driconf
```

and then under the performance tab, change "Use HyperZ to boost performance" to "Yes".  If I turn it on I get 1000+ fps, with it off I get closer to 550 fps.  Now all I have to do is figure out what is wrong with my 3-D acceleration in VMware Player
