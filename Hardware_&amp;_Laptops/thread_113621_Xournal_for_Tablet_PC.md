### Post by chuvisco on 2006-01-06
Version 0.1.1 of Xournal, a notetaking program similar to Gournal, Jarnal, and MS Journal, was released yesterday.  From the Xournal  [website]("http://math.mit.edu/%7Eauroux/software/xournal/"), it "aims to provide superior graphical quality (subpixel resolution) and overall functionality."  

There isn't a .deb available, but it was easy enough to install following the instructions on the site.  It doesn't have pressure sensitivity, but the subpixel resolution is nice and there are several useful tools.  IMO, this is now the best Journal-like program available.

---

### Post by chuvisco on 2006-02-12
Version 0.2 was released on Jan. 29.  New features include a binary installer (install without compiling), PDF annotation, PS/PDF backgrounds, and eraser emulation via  pen (right mouse) button.

Anyone that has installed Ubuntu on a Tablet PC should definitely give this program a try.  Xournal is the application that finally replaced MS Journal for me.

[IMG]http://math.mit.edu/~auroux/software/xournal/screenshot.png[/IMG]

### Post by chuvisco on 2006-06-11
Version 0.2.2 has been released.  New features include "mapping of tools to stylus buttons, moving selection by drag-and-drop works across page boundaries, vertical space tool can move items to next page, can change color or thickness of pen strokes in selection, function to apply paper style to all pages, and others.  

Also, the eraser tip on my stylus was not working in previous versions, but is now working in 0.2.2.  This is becoming a very nice program.
