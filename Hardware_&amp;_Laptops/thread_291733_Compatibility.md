### Post by John.Michael.Kane on 2006-11-02
There have been user reports of overheating issues with the E1705 running ubuntu, I'm not sure if this matter has been adressed or not.

You can download the livecd which will allow you to access your hardware,and see what works. The livecd also has the option to install the OS should want too.

As for the printer most of the dell printers are made by lexmark which from my knowledge are not supported.

If you plan to by a linux supported printer have look here [http://www.linuxprinting.org/printer_list.cgi](http://www.linuxprinting.org/printer_list.cgi)
