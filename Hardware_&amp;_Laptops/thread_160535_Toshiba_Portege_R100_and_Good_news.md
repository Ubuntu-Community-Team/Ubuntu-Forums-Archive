### Post by Tom Tiger on 2006-04-15
No... this is not a question.  It is more a look it works :-)  So good news.
The reason I'm posting this is that I found several people who couldn't get the thing to work on Linux.... and I thought it would be fun to put a successtory in between.

A while back I got a Toshiba Portege R100 (R=Revolutionary series)  and this is a very nice, very very flat (as high as a RJ45 connector).  It came with XP pro.

But I wanted Linux.  So it was distro time....  

Knoppix (and I tried several 3.6 up to 4.02) wouldn't work.  The reason was that I was booting from a pcmcia cdrom and it failed to load the pcmcia modules, it drops down to a shell and stops. The R100 can't boot from USB (only USB floppy) so I could have made a bootflop but.... I decided to keep on testing distros (slow day at the office)  Mandrive 2005LE actually installs and works,  but looses the cdrom after install....  So Ubuntu 5.10,  it detected the pcmcdrom, activated the slot and installed, without any problem, both networkcards detected and working.  So  now I've got a pretty cool subnotebook running Ubuntu 5.10 :-)
