### Post by Seaman on 2006-08-09
Try first removing the new memory and putting one of your old memories in the memory slot which you put the new memory, so we can find out if it is

1) The actual amount of memory that is the cause of the problem.
or
2) The new memory itself that is the cause of the problem.
or
3) The memory slot that is the cause of the problem.
or
4) Something else that causes the problem.
