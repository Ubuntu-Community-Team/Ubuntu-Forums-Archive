### Post by dmitriyp13 on 2008-03-18
I've had a good experience with running VirtualBox on mediocre hardware.  I was even able to run 3D CAD on the guest OS (winXP).  However, as far as I know, all of the video card acceleration stuff is not passed to the guest OS so the CPU handles all video acceleration.  It can be quite taxing if you are working on very intensive stuff such as CAD (or games).  The most important thing to get is a fast processor.
