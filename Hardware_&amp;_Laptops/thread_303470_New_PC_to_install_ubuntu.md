### Post by ReaderRat on 2006-11-20
It is suggested that, up to 1 Gig RAM, you have a /swap  of 2 Gigs. Any more RAM, and you probably don't need a /swap. Yes, that is a good version (alternate-amd64), but you can run the regular LiveCD (desktop-amd64) with 1 Gig of RAM. You will be given a choice, when partitioning in the installer, to set up your hard drive partitions.
