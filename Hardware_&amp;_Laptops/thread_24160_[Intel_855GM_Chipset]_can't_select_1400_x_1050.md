try this

[http://www.celifornia.com/documents/dell700m.html#3._855resolution](http://www.celifornia.com/documents/dell700m.html#3._855resolution) 

and remember to change the permissions of startup script to excutable

use # chmod a+x startupscript

while in the /etc/init.d/ folder

if that doesn't work the problem probably is that 855resolution isn't running
before GDM runs
