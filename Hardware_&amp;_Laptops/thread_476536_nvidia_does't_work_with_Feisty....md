### Post by dreadlord_chris on 2007-06-17
first off, remove the following section from xorg.conf:
```

Section "DRI"
Mode 0666
EndSection

```

and add this section in it's place:
```

Section "Extensions"
  option "Composite" "Enable"
EndSection

```

Add these 2 options to the **Screen** section:
```

  option "AllowGLXWithComposite" "true"
  option "AddARGBGLXVisuals" "True"

```

Save the file and restart X (control-alt-backspace) - that should (hopefully) fix you up.
