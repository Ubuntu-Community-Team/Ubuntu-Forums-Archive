### Post by chele on 2006-02-28
Just install it on the HardDrive and then put it in the other machine. As the drivers on Linux are all modules loaded at boot time, it will just load different modules for the different hardware. There is not much that gets hard-wired during the installation.

I have done this when installing linux on bunch of PC's. It just worked. Mostly :-)
