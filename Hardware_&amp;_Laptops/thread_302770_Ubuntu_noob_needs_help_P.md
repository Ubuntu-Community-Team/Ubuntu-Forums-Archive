### Post by generice_genus on 2006-12-23
Hey, I'm having trouble with the HP photosmart 7450 card reader also. 

I can mount the drive following this procedure on Ubuntu 6.10 using gnome:
1. Insert card
2. open computer:// (Places->Computer)
3. Wait for for light on card reader to stop blinking
4. Double click on "HP Photosmart 7400"
5. A dialog window will now appear saying "opening HP Photosmart 7400, wait for this to finish and  
the file browser window will switch to the root directory (i.e. "/")
6. The card should now be mounted at /media/usbdisk so browse to this folder to access your card

I will now start a thread to try and figure out how to make the card reader handled like any other removable storage device in gnome.

EDIT: This thread is [http://ubuntuforums.org/showthread.php?p=1922637]("http://ubuntuforums.org/showthread.php?p=1922637")
