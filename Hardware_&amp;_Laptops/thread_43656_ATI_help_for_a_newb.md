### Post by Zeroedout on 2005-06-22
allright. from [http://jesusfranco.3qhost.com/main/?cat=3](http://jesusfranco.3qhost.com/main/?cat=3)
In order to compile the module we will need the kernel’s header. To do this open up your **Synaptic Package Manager **(System>Administration>Synaptic Package Manager) and search for “**linux-image**” take note of the one you have installed. I have **linux-image-386 version 2.6.10-7**. Now do a search for “**linux headers**” install the correct version. I installed **linux-headers-386 version 2.6.10-7** accept any additional packages. Install them and exit [b]Synaptic
[/b]

 make sure you only install the headers that belong to your kernel, Synaptic should stop you from installing incorrect ones but i guess it dosn't. So if you've already installed incorrect ones, uninstall them. Even though you have Kubuntu, you should still have synaptic, you CAN use kynaptic, but personally, i wouldn't. Then, once the right headers are installed ONLY, run the ati installer, and it should then build the module, move the module to the right location, and run sudo fglrxconfig in a terminal.
