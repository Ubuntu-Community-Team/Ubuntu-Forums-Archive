### Post by Zuph on 2007-04-18
Depending on what kind of usage your server is going to see, you might want to go with a Mini-ITX system.  ([http://www.mini-itx.com/](http://www.mini-itx.com/))  These are extremely small and power efficient computers.  I've never used one with Ubuntu (or any Linux), but they use common components, so you shouldn't have difficulty with it.

Obviously, you can't use this for anything resource intense or critical.  Additionally, the built in network card you get on these isn't going to be as good as a decent Intel PCI card.

You might also look at getting an old laptop off ebay with a busted screen.  Might fit the bill.  Just do a little research and make sure Ubuntu will run on it first.
