### Post by Topinambur on 2007-03-21
I am running Edgy on Asus G1. The only piece of hardware which does not work is the integrated webcam. I would mostly use it for Skype, but considering that Skype for Linux does not use the webcam anyway, it's not a significant loss. The sound with mic. works flowlessly after recompiling the latest Alsa drivers.
All ports (including the card reader) work immediately after installation. Nvidia adapter runs perfectly after installing the latest Nvidia driver. Dual display, as well as TV-out works, although requires some tweaking.
I was also quite happy to see that my Canon 30D digital camera is fully supported by default in Edgy. I can easily copy and edit RAW files - very impressive.
One small issue is the DVD-RAM drive. The manufacturer is Matshita, notoriously known for it's strongly coded RPC-2 firmware. If you're not going to watch DVD's from different regions, it's not a big deal. Otherwise, consider buying another DVD drive, which can be flashed with RPC-1 firmware (check [http://forum.rpc1.org/portal.php]("http://forum.rpc1.org/portal.php"))

In overall - IMHO - this is the laptop, built to run Linux!
