### Post by EXCiD3 on 2007-05-07
Actually the 7300 GS will run games pretty well. I have a 7600 in my laptop and can play all the latest games at medium-high graphics settings. Nvidia is a good route to go as ATI is not supporting their linux drivers near as well as Nvidia. A 7300 is a step down from what I have, but should be plenty good for playing games. 

The embedded graphics card in you have probably shares its memory with the rest of your computer's ram. A new graphics card will give you dedicated memory which is faster than normal ram. This ram increases frame rates, etc, because graphics intesive applications use the video card's memory and processor instead of your main memory and cpu. This will increase performance all around.

Beryl runs at ~60 fps on my graphics card. A 7300 should run Beryl great. The 7300 is a lesser know video card yet it will still give you plenty of performance. If you want to see card comparisons, google "graphics card comparisons". Tom's Hardware is a good place to look for these specs.
