> **tux74 said:**

maybe reset your bios and redo the settings, I have had (rarely) a problem with bios causing problems after a simple memory upgrade on some systems, I generally clear cmos then reload with fail safe settings, boot to see how its going, if all is well I go back into bios and manually configure to my liking.

If that does not work, then perhaps you got a bad stick of ram, try running the computer on 1 stick then the other. Does it run fine on 1 and not the other? does it run fine on either 1 seperately? If it runs fine on either stick seperately then perhaps you have incompatable ram types and need to match to the one you prefer.

Just some of the stuff I start looking at when I run into memory problems, hope that helps get you started.
