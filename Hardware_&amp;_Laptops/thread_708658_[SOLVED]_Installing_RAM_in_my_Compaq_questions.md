This first question is going to sound extremely newbish. I know that the speed of your RAM and your mobo are supposed to match. But I also know that one of them can be higher than the other. But which one?

I have some 444MHz laptop RAM (I get all kinds of good junk) and my laptop's RAM is 400MHz. Of course, with my luck, that wont work, will it?

Supposing it does work, I have a question about removing one of the sticks in my laptop. I've done RAM in desktops, but in this laptop, the sticks seem to be in there really tightly. I'm afraid to yank too hard without asking advice from someone with a Compaq. Is there a trick?

### Post by Existentialist on 2008-02-27
The RAM will be run at the speed of the slowest stick.  So you can use any RAM, within the size and type requirement of the motherboard, but it will all run at the 400MHz speed.

I don't know the specifics of your computer, but make sure there is no release mechanism such as a having to angle the ram a certain way before pulling too hard.
