### Post by xkaliburx on 2008-02-07
Well, I have a new Dell XPS M1330 laptop with a pretty much storyboard install.  It shipped with *cough* vista, so I simply started the laptop on it's 1st run with a Knoppix live cd, gparted the hard drive making 100gb avail for ubuntu.

Did the install, it recognized nearly everything.  The restricted drivers were needed for the NVidia video card and wireless but no problem at all.  I then installed thinkfinger, thinkfinger tools enabling the fingerprint scanner, lastly installing kopete which found the built in webcam.

The only unresolved issue I have is with the power dimming the sceen when idle.  Power savings are correct, but that is only on battery power, so I have had great success with it.

Good luck with the search.
