### Post by Rocket2DMn on 2008-04-01
Darkchef, that user was trying to install with the LiveCD, which does not work for everybody.  If you get a blank screen with the LiveCD, too, that's OK, you can install with the alternate install cd.  It is a text based installer without a live session, but is very easy to use.  You can get it at [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) by checking the box at the bottom for alternate cd.
Although you should be OK when you boot after install, if for some reason your screen is still blank, here is a little guide to reconfigure X - [http://ubuntuforums.org/showthread.php?t=690760](http://ubuntuforums.org/showthread.php?t=690760)
