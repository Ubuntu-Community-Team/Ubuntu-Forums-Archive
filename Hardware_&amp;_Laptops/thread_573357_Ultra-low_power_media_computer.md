### Post by subitul on 2007-11-24
Further searching around, I've found that 5W was very optimistic. What I have found is one of VIA's new mini-ITX (VB7001G) boards that consumes about 20W (then add a few more watts for a hard drive) when under full load and it claims to have the highest performance/power ratio. It's avalible for £72 inc. VAT which isn't a touch on the pre-assembled things (like TranquilPCs) even when you add in a power brick and a notebook hard drive.
So I think this is problem solved but if anyone else bright ideas then please add.
