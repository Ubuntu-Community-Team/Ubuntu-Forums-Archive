### Post by clean97gti on 2007-03-17
in your xorg.conf file under Subsection Display where it says "1280x800"
you need to add more resolutions. Add "1600x1200" "1600x1050" "1440x900" etc according to what your monitor can display.

in a terminal, type sudo gedit /etc/X11/xorg.conf and make the changes.
You might want to make a copy of the file before editing though in case it screws up X. You'll also need to restart X server, although I would just reboot the machine. Seems to come across fewer problems that way.
