### Post by PhoenixP3K on 2006-05-14
Ok I guess you got a USB device connected during the install. Perhaps the installer is trying to configure it or find a proper driver. You might try to unplug all USB devices connected during the install process. Then after the install is complete look for hardware compatibility with new drivers.
