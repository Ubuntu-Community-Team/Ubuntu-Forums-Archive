### Post by K0LO on 2006-07-08
Here's a couple of things that I'd try (if you haven't already):

1. Check the motherboard manufacturer's web site to see if there is a newer BIOS available.

2. Check the settings in the BIOS SETUP to see if the processor is being overclocked.

3. Check the memory settings in the BIOS SETUP to see if aggressive timings are being applied.

Good luck!

Mark

### Post by K0LO on 2006-07-08
If the PC is made by Compaq then you can go to Compaq's web site. You'll need to find the computer's model number (hopefully somewhere on the chassis) to look it up.

Before doing so, enter the BIOS SETUP and see if it displays the BIOS version number so you can write it down. If there is a newer version available on the web site there should also be instructions on how to update.

I know what you mean about a lack of BIOS options. I just got done working on my Mom's Compaq PC and there are so few configuration options available in their BIOSes. So I guess you can scratch my suggestions #2 and #3 off your list!

Mark
