### Post by jespdj on 2007-11-29
You need to do some research to learn about computer memory.
[http://en.wikipedia.org/wiki/Dynamic_random_access_memory](http://en.wikipedia.org/wiki/Dynamic_random_access_memory)

> 1. what is "ECC" or "non-ECC" (my MB can have either)
You don't need ECC memory for a normal PC. That's memory with special error correction bits which is mainly meant for servers. It's a lot more expensive than non-ECC memory.
> 2. what is "unbuffered: (vs. buffered I guess, and why can my MB only have unbuffered (and what types are unbuffered))
I don't know exactly. Your motherboard manual should tell you what kind you need.
> 3. what is "CAS latency"
This has to do with the access speed of the memory.
> 4. what is "PC(and then a number, like 2700 or 2100)"
This indicates the speed of the memory.
> 5. what is the difference between DDR, DDR2, DDR3, etc. and why can my MB only have DDR?
There are different kinds of DDR and a motherboard / chipset is normally made for one kind of memory.
> 6. How can I find out what exactly my current RAM is? (I want to get the exact same kind for maximum compatibility)
Open up your computer and look at the codes on the memory chips.
There are also programs which tell you exactly what type of memory is in your computer, for example [CPU-Z](http://www.cpuid.com/cpuz.php) (for Windows!).
