### Post by Kateikyoushi on 2006-10-21
You can install the cpufreq monitor applet for gnome. Then right click on the top panel add and select cpufreq monitor. Then you will be able to monitor you cpu tempreture on the gnome panel.

To install it copy this to the terminal.
```
sudo aptitude install gnome-cpufreq-applet
```
