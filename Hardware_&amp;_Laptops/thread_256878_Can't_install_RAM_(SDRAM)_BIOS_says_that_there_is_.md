### Post by pravuil on 2006-09-13
The problem might be a bad ram stick or a bad ram slot. Pull out one of the sticks and start the computer. If your computer recognizes it on boot then turn off the computer and swap the sticks in the same slot. Repeat. If it still recognizes the ram, change the slot and repeat the process a couple more times. If there is still no error make sure that both sticks are SDRAM. 

I know some RAM back then had to be the same type in order to work correctly. If your using SDRAM for both sticks it shouldn't matter.  I don't think I've ever had a problem with mixing different speeds though. Someone needs to call me on that. I haven't been in computer hardware for a while so I'm a little rusty with this.

### Post by az on 2006-09-13
If your mother board does not recognise ECC ram and the second stick is ECC, that may be your problem.

If the density of the chips on one stick is higher that the first, again, your motherboard may or may not ignore one.

I would reccommend trying to get two used 128 meg sticks instead - you should pay around five dollers each (including shipping) for that on ebay these days...
