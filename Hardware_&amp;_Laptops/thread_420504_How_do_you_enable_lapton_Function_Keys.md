### Post by ZeroXR on 2007-04-24
Supposedly, the laptop BIOS must be the newest version for the first step. The other part is more dependent on hardware compatibility with Ubuntu. I had a project Toshiba laptop that I had worked on and found that the Fn + F-keys didn't work to control brightness or anything else. The odd thing was my Dell laptop had full compatibility except for the "MediaDirect" key which was a direct command to launch Windows Media Center when I still had it on the Dell.

If you still have a Windows partition still, try updating the BIOS with the newest version. Other than that... it's all "luck" that is left.

### Post by nickdbliss on 2008-04-18
hi there. Well i was having the same problem but now it is solved on my Hp nx6325 hehe. Having flawless ubuntu experience now.:guitar:

I just wanted those keys for brightness +-... Although by simple pressing them wont do anything there is a lil trick into it. 

Here how it worked for me.

It was accidental though lol.Was pressing Ctrl+Alt+F6 then i could change the brightness with normal function keys ie fn+f9 or f10 and again switching back with Ctrl+alt+f7. Also i found it verified at this link [http://wiki.cchtml.com/index.php/HP](http://wiki.cchtml.com/index.php/HP)


"...LCD-brightness can only be changed in VT (Ctrl+Alt+F1 -> change brightness -> switch back to X works)...."

Hope it works for you as well. Good luck!!!
