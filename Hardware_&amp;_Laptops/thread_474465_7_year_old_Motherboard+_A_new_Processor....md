### Post by tgalati4 on 2007-06-15
I'm guessing the best you can do is to get a 1.4 GHz Celeron that will fit in a slot 1 converter to PGA370 (I only paid $8 for it).  The Celeron processors usually have lower cache, so in reality, a PIII with max RAM will probably have the same performance.

Choice: (1) Older processor with max ram:   ~1,000 Bogomips
            (2) Newer processor at double speed with smallish cache: ~1,000 Bogomips

Only testing will really tell, but since the bus is old (100 MHz) you are limited with data transport speed, so putting a faster processor will only help so much.

I would only replace it if the processor was fried.  Otherwise, max your memory as others have suggested and start with Dapper as it is quite stable on old hardware.

Good Luck.
