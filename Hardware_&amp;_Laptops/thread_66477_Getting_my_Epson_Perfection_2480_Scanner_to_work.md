### Post by danst0 on 2005-09-17
Everything is running smooth you just have to find the firmware file for your scanner on the install-cd.
For my scanner I found the firmware on [http://www.bot.unibas.ch/~pascal/epson2480.html](http://www.bot.unibas.ch/~pascal/epson2480.html) 

And follow the guidlines mentioned there (beside you do NOT have to install anything!)

Afterwards start xsane as a user from the terminal. Then you can see why it takes so long to start xsane up... because the scanner has to heat up first...;-)

Hope this is working for you as well!
