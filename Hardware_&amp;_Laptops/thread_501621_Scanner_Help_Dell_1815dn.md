### Post by geraldm on 2007-07-15
Dell's site has the firmware for the drivers as a download.
The scanner portion is not supported in sane.  
Best course is to read the manuals, and see if the ethernet interface has some
interactive interface that a script could generate a scan.  An *.html page or xml with
editable parameters would work well.  Novell is also selling the device, so it may
possibly have some OPENsuse package.
If you investigate the etnernet interface and it can generate scans please post some
information on what is needed.

Gerald
