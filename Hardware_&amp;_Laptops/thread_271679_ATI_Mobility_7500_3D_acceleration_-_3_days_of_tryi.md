### Post by thoffland on 2006-10-23
You can type

```
uname -r
```

or 

```
uname -a
```

And it will tell you. Then you just go into synaptic and look up your kernel number, but instead of 386, choose the 686. Get the headers too as you might need them later for program installs. 

If you didn't install it yourself, you probably have 386. There are some system req's though for the 686... I think more than PIII or something. You might want to look that up if your on an older machine.
