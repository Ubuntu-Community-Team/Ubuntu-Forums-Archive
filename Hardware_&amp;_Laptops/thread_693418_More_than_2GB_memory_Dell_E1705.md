### Post by Existentialist on 2008-02-11
Any issues with supporting over 1GB per slot on your laptop is going to be a hardware support issue.  According to Dell each of the 2 memory slots only supports 1GB per slot.   From the OS standpoint, if you are running a 32 bit OS you will only be able to address somewhere between 2.75-3.5GB of the 4GB.  The Intel T2500 is capable of running 64 bit OSs so if you do upgrade and find the RAM works you'll be fine as long as you run the AMD64 version of ubuntu.
