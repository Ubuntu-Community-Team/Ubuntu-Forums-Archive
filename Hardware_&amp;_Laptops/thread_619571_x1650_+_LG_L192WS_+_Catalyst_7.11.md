**Installing the ATI drivers properly**
First of all, you need to remove any old fglrx driver and disable the one provided by linux-restricted-modules.

Uninstall fglrx, if present:
```
sudo aptitude purge xorg-driver-fglrx ~nfglrx-kernel fglrx-amdcccle
```

Open [COLOR="DarkGreen"]/etc/default/linux-restricted-modules-common[/COLOR] with the text editor of your choice ([COLOR="Red"]as root[/COLOR]).
At the bottom:
```
DISABLED_MODULES="fglrx"
```

Now install the packages needed to build the kernel module and make .deb packages:
```
sudo aptitude install dpkg-dev module-assistant build-essential fakeroot dh-make debhelper debconf libstdc++5 linux-headers-generic
```

Build the Ubuntu 7.10 installation packages:
```
sh ati-driver-installer-7-11-x86.x86_64.run --buildpkg Ubuntu/7.10
```
(NOTE: This can not be done inside a directory path that includes spaces)

Install driver, kernel module source and Catalyse Control Center:
```
sudo dpkg -i fglrx-kernel-source_8.433-1_i386.deb xorg-driver-fglrx_8.433-1_i386.deb fglrx-amdcccle_8.433-1_i386.deb
```

Build and install the fglrx kernel module:
```
sudo m-a prepare,update
sudo m-a build,install fglrx-kernel
sudo depmod
```

Remove the packages, if you want to:
```
rm fglrx-amdcccle_8.433-1_i386.deb
rm fglrx-installer_8.433-1_i386.changes
rm fglrx-kernel-source_8.433-1_i386.deb
rm xorg-driver-fglrx-dev_8.433-1_i386.deb
rm xorg-driver-fglrx_8.433-1_i386.deb
```
