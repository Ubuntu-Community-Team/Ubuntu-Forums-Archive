### Post by zaziork on 2007-02-06
Well I finally found the solution (no thanks to anyone I asked here). Contrary to some, I thought I'd post how I fixed this so that anyone with the same problem in future may benefit.

It wasn't actually a problem specific to Saitek keyboards, nor indeed the Grub boot loader. It was a BIOS setting, which needed to be changed to allow USB keyboards to be recognised and operated by the BIOS rather than the OS. 

The actual setting differs according to the motherboard/bios/version you're using, so it's really just a case of hitting delete at boot to access the bios, then having a poke around to identify what you need to alter to reflect the aforementioned change.
