Well, I figured this one out on my own. I found a discussion about 915resolution. In hopes that 7.04 might have the command installed, I entered it and it told me that it wasn't installed but could be, so I downloaded it and installed it. The list mode showed I now had the 1440x900 mode. So I rebooted with 1440x900 as my default now.
