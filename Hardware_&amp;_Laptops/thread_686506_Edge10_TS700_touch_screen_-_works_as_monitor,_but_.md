### Post by DoctorMO on 2008-02-06
Serious help has arrived :-)

OK the first thing you have to do when your dealing with a new input device is check to see if anything appears in /dev/inputs/ for the device, for instance when you plug in a wacom tablet you get /dev/inputs/wacom as well as /dev/inputs/inputN (where n is the next available number)

As soon as you have that input node you can move onto configuring X11; until then you should concentrate on getting the usb driver working.

If the driver is a kernel driver than one of the things that might be useful is to send the source files you got to someone on the kernel usb hacking list since they can then hack it into the real kernel so when you upgrade your computer you won't loose support. At the moment when you install 3rd party kernel drivers you'll loose them on even a security upgrade so be careful to keep the sources somewhere safe just in case you need to install it again after an update.

Your using an old version of dohickey so it doesn't tell me what if any drivers are being used; not to worry.

Once you've told me if it's a kernel driver and if those input nodes exist I can help you further.
