I have an Alienware m7700 that I want to use Ubuntu on.  I started with 6.06 but I am now running with 6.10.  My problem is that Ubuntu is almost unusable with the touchpad.  I want to completely disable it.  I like using my Logitech Mx510 mouse.  The touchpad is extremely sensitive, especially with the mouse plugged in.  If I even touch the laptop to the sides of the touchpad, the cursor jumps.  With or without the mouse attached, I will get a mouse click.  I can be touching only the mouse and nothing else.  If I hover over a button or link, it will act like I clicked on it when I know for certain that I didn't.  The mouse is fine in Windows XP and on other PCs.  

Can anyone tell me how to disable the touchpad entirely?  Is it possible?

Thanks,

Fred

Update:  Found solution.  Amazingly I never knew that Fn - F10 would disable the touchpad on my laptop.  :)  Doh!
