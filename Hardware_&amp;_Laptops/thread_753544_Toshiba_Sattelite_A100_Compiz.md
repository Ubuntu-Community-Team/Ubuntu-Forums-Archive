### Post by Adreniline on 2008-04-12
Assuming your drivers are properly configured, this should add the 'Advanced Desktop Settings' to your** System > Administration** list:
> sudo apt-get install compizconfig-settings-manager

If your video card needs to be configured, see here: [https://help.ubuntu.com/community/BinaryDriverHowto/ATI](https://help.ubuntu.com/community/BinaryDriverHowto/ATI)
