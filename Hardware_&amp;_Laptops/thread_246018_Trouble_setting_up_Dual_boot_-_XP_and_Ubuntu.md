### Post by TheMatt on 2006-08-28
Hi everyone,
I am trying to set up a dual boot with XP and Ubuntu. I have XP installed, but I can't seem to install Ubuntu. I have 15 GB of free unpartitioned space I want to install it on. When I pop in the disk at boot, it reads it and then just boots into Windows. I would uninstall windows, but I cant because I only have a recovery disk that came with my computer, not an XP disk. Help would be much appreciated.

---

### Post by shunthemask on 2006-08-28
Have you set your cd-rom up as the default boot device?

---

### Post by slibuntu on 2006-08-28
shunthemask is right, i had to do the same thing, when the laptop turns on hit the key for the boot menu (usually F12 or Del) and make sure the laptop is trying to boot from the cd drive

---

### Post by TheMatt on 2006-08-28
Got it.
 
I set the CD drive as the first boot device in the BIOS but that didn't work. So during the boot up I pressed F12 and forced it to boot from the CD drive manually, which worked. Thanks for the help.
