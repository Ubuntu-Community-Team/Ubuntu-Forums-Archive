### Post by titi(ger) on 2008-01-07
I have setup 3 computers using this mainboard.
To setup Gutsy use the save graphics installation ( normal one doesn't work.)

Everything installs well and runs fine.( including nvidia 3d ...)

But there are some sound problems in games! If you start for example glest you'll get some crackling noises in the music. I think there are 2 soundchips onboard (normal sound and HD sound) and they get an IRQ problem or something like this.
There are no sound problems when playing music/watching films ... only the 3d-games have some sound trouble. ( tested glest/nexuiz)
