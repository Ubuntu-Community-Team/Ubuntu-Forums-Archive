### Post by kmslinux on 2007-10-14
Are the speakers unmuted and turned up? I know that's a stupid question, but you have to make sure... If they are, then read ahead. If you are logged in graphically as root, which you shouldn't be, then you probably would get sound. But, if you are logged in as a regular user, try this in the terminal:

```
jheronimus@jheronimus-laptop:~$ sudo chmod 666 /dev/dsp
jheronimus@jheronimus-laptop:~$ sudo chmod 666 /dev/mixer
```

That should work. If it doesn't, PM me. :)

### Post by SchwarzeKrause on 2007-10-15
I've made it!!!
```

sudo apt-get build-essential
wget ftp://ftp.alsa-project.org/pub/driver/alsa-driver-1.0.15rc1.tar.bz2
tar xvpjf alsa-driver-1.0.15rc1.tar.bz2
cd alsa-driver-1.0.15rc1
./configure --with-cards=hda-intel
make
sudo make install

```
