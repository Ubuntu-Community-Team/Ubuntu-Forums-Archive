### Post by rgilham on 2006-03-07
With the recent power problems in Cape Town South Africa a ups helps. The Meissner UPS are a local brand and rather than using the WinPower software which uses Java and therefore rather heavy weight you can install the nut tools (network ups utilities.) 

Configure the nut system as described in /usr/share/doc/nut/README.Debian but make the nut user a member of the dialout group.

The driver that the UPS needs to use is blazer

Enjoy!

knutclient is a nice gui under KDE
