### Post by daengbo on 2008-01-01
I recently wrote[ a blog about eeeXubuntu]("http://ibeentoubuntu.blogspot.com/2007/12/eeexubuntu-please-sir-may-i-have-some.html") and what a great idea it was. I mentioned laptop-specific distros as being a wonderful thing for laptop owners. See [http://www.fitzenreiter.de/averatec/index-e704.htm](http://www.fitzenreiter.de/averatec/index-e704.htm) The upkeep on these is huge, though, and it would be easier to just have a package that modified the base distro.

I ended up [putting some ideas about how it might work]("http://ibeentoubuntu.blogspot.com/2008/01/laptop-specific-packages.html") and a mock-up of the web form on my blog. I'd appreciate thoughts and criticism. I'm especially worried about package conflicts.

I haven't ever made a DEB package and haven't made an RPM in about six years, so I'll need to do some research, but I believe this can work and make installation for laptops easy. The mock-up is [here.]("http://picasaweb.google.com/daengbo/IBeenToUbuntu/photo?authkey=yaf_ADugHGc#5150407559497455154") if you don't want to bother visiting the blog.
