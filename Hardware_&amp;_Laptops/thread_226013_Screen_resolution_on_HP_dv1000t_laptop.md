### Post by bartlantz on 2006-08-02
I had this same trouble.  After searching the web for a solution and not getting anywhere I just ran this command:

apt-get install 915resolution

(actually I ran apt-get install 855resolution and apt-get selected the correct 915resolution package)
 
and that set up everything correctly without any help from me. I rebooted  my dv1000t and it now runs in 1280x768 perfectly.

perhaps you could back out your scripts and try this approach?

### Post by wammin on 2006-08-02
Yep, this eventually worked for me, too. I was a total Ubuntu newbie, and I originally tried "apt-get install 915resolution" but I couldn't figure out why apt-get couldn't find the package. Eventually I realized that I had to enable the Universe in my software settings. Wish somebody would have said that before.

Anyway, when you install 915resoloution using the package, it works great, without any additional config.
