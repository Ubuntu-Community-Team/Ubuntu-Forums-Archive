### Post by mcduck on 2008-03-07
I'd start by checking that all power and PATA/SATA cables are properly connected. If that doesn't help, the next thing to check would be that RAM sticks and graphics card are also properly in their slots. And if even that doesn't solve the problem the last thing would be to check that CPU cooler is still in it's place..

Edit: The actual beep codes depend on what BIOS your motherboard is using. At least for AWARD BIOS one long beep would mean that memory is not correctly installed.
