### Post by DirtDawg on 2005-01-29
Have you thought about trying the "live" CD? You burn it to a disk (an "iso" image), then put it in your CD Rom. When the computer reboots, you'll have Ubuntu desktop instead of Windows! There's no danger to your computer this way, no way to screw up your hard drive or erase windows files,  and you can "test drive" Ubuntu and see if everything runs OK. The downside is you cannot install anything when using a live CD, and anything you save will be lost as soon as you end the session. In addition, things will run slower because Ubuntu will be reading from the disk as opposed to reading from the Hard Drive (which is much faster).  

This is how I started with Linux because like you, I am not a programmer, but was curious about the whole Linux thing. On that note, I should also add that you may be suprised at how little programming you really need to know (ie: none). And you'll find the more you work with Linux, the more "programming" you'll start to pick up (not out of necessity, but rather because you may want to).

Hope this helps and welcome aboard!
