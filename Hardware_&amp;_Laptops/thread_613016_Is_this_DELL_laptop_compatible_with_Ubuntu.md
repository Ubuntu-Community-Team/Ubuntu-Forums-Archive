### Post by tschneiter on 2007-11-14
That computer should work reasonably well with ubuntu.
Some notes:
If possible, get an nVidia video card. ATI simply hasn't put forth the effort to get their open source drivers together, though I understand that headway is being made. I'd just stick with nVidia.

Also, I would go the intel route vs the AMD route. Simply because intel parts tend to work more effortlessly with linux (and intel has recently declared their devotion to open source). If the subsidiary of a company cannot publish open source drivers, would you expect its parent company to? (AMD > ATI).

On that same note, Stay away from dell-branded wireless cards. They'll work, but you'll need an extra step to make them do so. Your best bet is an intel-branded card. Besides that, everything should work just peachy. I'm on a 1405 here and loving it.

### Post by tschneiter on 2007-11-15
The dell 1520 model offers intel/nvidia solutions, though at a bit higher cost. 

Also, if you dont _need_ powerful graphics, or a 15" screen, you could go for the 14xx series are all great choices. I went for the 14" with an upgraded screen because it has great resolution (1440x900) and is a lot more portable than my previous 15". It has intel integrated graphics but I can run tux racer, compiz/etc. The only downside I've found to using the intel graphics is that there is an old problem with the driver wherein the frame buffer cannot be larger than some set size. Since I am using a secondary monitor, and I have it set up as an extended desktop, I cannot run compiz. I can, however, use xcompmgr to let me use avant window navigator, and I can still play 3d games.

Good luck!
