### Post by w4ett on 2007-07-15
Well...first thing I see is that u have a Via GFX chrome 9.......I believe those use the via or the unichrome driver for your video. Currently you have the vesa driver enabled. Try this:

sudo apt-get install xserver-xorg-driver-unichrome

sudo dpkg-reconfigure -phigh xserver-xorg
