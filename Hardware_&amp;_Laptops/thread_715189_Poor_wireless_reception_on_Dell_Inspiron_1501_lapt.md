### Post by buckwheat12n on 2008-03-04
I have a dual boot setup with WinXP on one partition and Ubuntu 7.10 on the other.  I get great reception with the wireless card while in Windows.  When I boot into Ubuntu I get about half the reception.  I can only go about half the distance away from the router that I can go while in Windows.  I am using the restricted driver and the wireless card is a broadcom.  Has anyone else experienced this?  Is using the restricted driver not the way to go?  Any help would be appreciated.

---

### Post by buckwheat12n on 2008-03-04
Ok, answered my own question.  I scrapped the restricted driver and installed via ndiswrapper.  Much better range now.
