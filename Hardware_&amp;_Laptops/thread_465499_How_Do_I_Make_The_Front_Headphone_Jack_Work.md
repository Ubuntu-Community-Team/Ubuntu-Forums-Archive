### Post by Seeker84 on 2008-03-23
So this worked for me.

I'm in gusty gibbon.  I double click the sound and make sure the following is selected.
Headphone
PCM
CD
PC Speaker
Capture

Restart comp. then go back to the same place and hit PCM Out Path & Mute.
An option opens up in the original dialog. in that tab, set it to post 3D.

It worked for me.
