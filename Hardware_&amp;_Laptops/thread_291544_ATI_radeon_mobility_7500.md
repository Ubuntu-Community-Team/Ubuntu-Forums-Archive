### Post by urbanus on 2006-11-02
I know how you feel. Been there done that. And found a solution. :-k 

Here are two pages you might want to visit if you haven't done that already:
fglrx: [https://help.ubuntu.com/community/BinaryDriverHowto/ATI](https://help.ubuntu.com/community/BinaryDriverHowto/ATI) :D 
radeon: [https://help.ubuntu.com/community/radeondriver](https://help.ubuntu.com/community/radeondriver)

HOW EVER!!!!! :-k 
To get the radeon driver to work you need to add "radeon" to the "/etc/modules" file.
BUT, if you have the "radeon" word in that file when using the fglrx you will get horrible GL "speed" and it will show as Mesa and not Ati. So make sure you haven't got i wrong ;) 

PS! Sometimes when one tries to get something to work one will make changes to the system that when you find the right tutorial it wont work. ](*,) 
Too new packages for instance. So a clean install when you find the right guide is recommended. :-D 

Hope this helps you! ;)
