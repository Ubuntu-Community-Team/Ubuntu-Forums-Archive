### Post by chewearn on 2007-12-13
Generally, you don't have to download and install hardware drivers for Ubuntu because the hardware drivers are already included (this is different from Windows).
Most of the time, only thing you need to install is Ubuntu itself, and nothing else.

Having said that, if you have a hardware that is unsupported by the linux kernel, then you might need to jump thru hoops to get it working.

In your case, the HP6710b laptop should work, but with some minor tweaks during installation:
[https://wiki.ubuntu.com/LaptopTestingTeam/HP6710b](https://wiki.ubuntu.com/LaptopTestingTeam/HP6710b)
