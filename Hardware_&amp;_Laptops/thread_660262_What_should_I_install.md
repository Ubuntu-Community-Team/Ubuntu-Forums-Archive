### Post by hari_home on 2008-01-06
40GB is too small for dual boot but you can do it. Windows with you applications can take as
much as 15GB quite easily. Good to allow 20GB each and external Harddrive or thumb drive for data.

I suggest ubuntu gutsy because it supports more hardware such as printers, scanners and wireless network better than others. User interface is fairly user friendly.

Still be prepared to kill a weekend doing this.

Install with wired connection  to network.

 Wireless may give you trouble.
If the native driver in the distribution does not work, your choices are to go to the wireless card vendor's website and get linux or windows driver (.inf file, .sys file) with ndiswrapper. If the
vendor support Linux it may work better. See my posting by searching for Toshiba A215 where I
posted my experience on A215-S7416.

Hope this helps.
