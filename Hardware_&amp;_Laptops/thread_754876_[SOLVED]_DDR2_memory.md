### Post by kutjara on 2008-04-15
> **sharks said:**
> I would like to upgrade to kingston 533 mhz ddr2 ram. I have a doubt that whether to upgrade to normal ram or value. What is value ram? Which is better?

Typically, value RAM doesn't quite meet the specifications of high-performance systems (such as hard-core gaming rigs), so it gets sold cheaper for use in systems that will not be too stressed.

If you're going to be using your PC for intensive gaming, graphics, or lots of memory intensive applications, it's probably worth spending a bit more on name-brand RAM. If you're a normal desktop user, value RAM is probably ok.

### Post by Adreniline on 2008-04-15
If it's something like corsair valueselect (in other words, a well known company), it's probably fine.  I'm using Corsair Valueselect, and it worked fine when I was gaming (its 667mhz paired, not 533mhz, just in case you're wondering).

If it's some unknown company, it'd be best to stay away (as the above post stated).

But if you watch for a few weeks, there are times when the higher end RAM will sell for the same (or very close to) the value RAM brands.  It's all about the timing (happened about two or three weeks after I bought my RAM).
