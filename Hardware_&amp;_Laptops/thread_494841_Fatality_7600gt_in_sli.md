I would install the nVidia drivers with Envy :[http://albertomilone.com/nvidia_scripts1.html](http://albertomilone.com/nvidia_scripts1.html) .... Then once you have the drivers installed you can enable Sli by opening a terminal and: 

```
nvidia-xconfig --sli=yes
```

I got my 7900 GT KO's running perfect this way...
