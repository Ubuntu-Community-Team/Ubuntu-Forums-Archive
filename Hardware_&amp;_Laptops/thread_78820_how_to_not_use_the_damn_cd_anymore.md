I want to be able to do apt-get without the blasted cd anymore. How can I do it?

---

### Post by aysiu on 2005-10-19
Type this in a terminal ```
sudo nano /etc/apt/sources.list
``` Type a # sign in front of the CD line (should be the first one).
Type Control-X
Type Y
Type Enter
That's it.
