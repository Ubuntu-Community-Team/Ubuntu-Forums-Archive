Clicking is definately a bad thing to be hearing.  Every drive I've ever encountered clicking either died completely or suffered serious data loss when the clicking ended.  You may want to stop trying to boot that drive if it has anything important on it and backup the data you need to a new/another drive.  Red X's in the default ubuntu theme usually have to do with file permissions.  If you're accessing parts of your drive you had setup with an existing install from the live cd, you're seeing those X's more likely than not due to ubuntu not having the rights to access those files since they were created by a different user (as far as the software is concerned).
