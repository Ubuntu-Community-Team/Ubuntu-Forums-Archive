### Post by allwin on 2007-01-21
Go into a terminal and type "ps -x" and you'll see them all.  There's a utility out there called BUM (Boot up manager) which allows you to weed them out.  Only trouble is...I don't know which are the good processes vs the bad ones.  You can also check out the "top" command.  I'm not on my Ubuntu system right now, but that helps you identify the running "processes" and so on.  Also in GNOME there is a system monitor application  which you can add to the panel which provides similar info.
