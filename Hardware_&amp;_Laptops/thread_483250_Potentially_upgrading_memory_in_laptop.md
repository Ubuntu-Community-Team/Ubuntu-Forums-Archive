### Post by joe.turion64x2 on 2007-06-24
This is your lappy [http://www.thinkwiki.org/wiki/Category:A21m](http://www.thinkwiki.org/wiki/Category:A21m) isn't it?

According to the specs given there the supported memory type is **Sodimm PC100** (PC133 works sometimes though). A 512MB PC133 stick costs here in Mexico about USD$100.

You installed extra memory on it, didn't you? Because it comes with 128MB only from stock. This laptops usually have only 2 memory slots so if you already have 2 128MB sticks you'd have to get rid of one of them at least. The only remaining thing here to know is what is the maximum amount of installed memory.

Joe.
