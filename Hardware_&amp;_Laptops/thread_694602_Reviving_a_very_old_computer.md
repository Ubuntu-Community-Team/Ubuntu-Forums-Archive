### Post by julian67 on 2008-02-12
+1 for Antix, should be fast, particularly if you give it a swap partition. Also look at Wolvix Hunter, slackware based, good hardware detection, excellent set of default applications, Xfce desktop and very easy to use and lightweight/fast.  Antix is based on Mepis so it's another Debian derivative, it will be quite familiar to an Ubuntu user. On my laptop it's much faster than Xubuntu Gutsy and also faster than Xubuntu Feisty.  Feisty was pretty decent for older stuff but Xubuntu Gutsy is far too heavy. Luckily Antix uses the Debian testing repos so the software is up to date (can't say the same about Feisty any more).
