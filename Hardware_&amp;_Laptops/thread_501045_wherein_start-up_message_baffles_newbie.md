### Post by lisati on 2007-07-14
This is part of the normal "stuartup" progress, roughly equivalent to DOS and Windoze "CHKDSK"/"SCANDISK" that help make sure that your file system is in a usable condition - you don't normally need to be too worried by it. The settings can be changed - come back to us if you need some help changing them.

### Post by w4ett on 2007-07-14
Ubuntu forces a file system check at  preset intervals to maintain system integrity.  Nothing to worry about, just normal system houskeeping!  :)
