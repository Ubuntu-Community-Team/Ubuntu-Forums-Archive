### Post by nehalem on 2004-12-19
Try to start a game but not from the menu.  The problem is, as I can best judge, when you start a game from the menu it makes a sound and many sound cards (like onboard ones) can't have gnome playing a sound and have a game try to use it at the same time.  So when a game starts, it can't find the sound device because it's in use.

Try to navigate to /usr/games and start a game from there and I'll bet sound will work.
