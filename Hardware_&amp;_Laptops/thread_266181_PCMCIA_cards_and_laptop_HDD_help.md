### Post by K.Mandla on 2006-09-26
My only suggestion would be to ...

1. If you're planning on using Windows at all, make sure the BIOS isn't freaked out by the size of the drive. Older laptops generally have [BIOS limits on hard drive size]("http://www.seagate.com/support/kb/disc/bioslmt.html"), and get feisty if you put in a drive that exceeds it.

2. Along similar lines, if you're planning on using Ubuntu with an old laptop and you need PCMCIA access, make sure you don't run into [this problem]("http://www.ubuntuforums.org/showthread.php?t=125334"). (But then again, how could you know you were having it until you had it ... ? :confused: )
