Or you could even just jump straight to Breezy. I've run Breezy with the Xubuntu desktop on a P2 with 128Mb, and while it wasn't the fastest machine alive, it worked fine. 

I think the processor, hard drive and memory you've got would be plenty to get 5.10 going (I looked up some stats for I4000's on ebay -- I'm assuming you have about a 650Mhz in that?). In my experience, if it can run XP without undue stress, it should handle Breezy fine.

That way you'll have a better idea if the problem you're seeing is hardware related, or if it was something that was ironed out in later versions.

My only other suggestion is to keep in mind that Live CD versions of Ubuntu can be finicky. What works in a full install might be clunky off the Live CD ... and vice-versa, sometimes.  Have fun! :mrgreen:
