### Post by quietglow on 2005-11-11
Just found this thread. I just bought a n610c to run as my lightweight laptop--I mainly use my desktop.

I did a clean install of Breezy and had no trouble with video recognition. If that xconf doesn't work I can post mine as well.

As far as configuring the lid close, you can do it the hard way by editing /etc/acpi/lid.sh and having it point to /etc/acpi/sleep.sh

BUT, you may want to install the Gnome power manager:

sudo apt-get install gnome-power-manager

It is much better than the default battery monitor and you can choose what you want to happen when graphically. Works very nicely.
