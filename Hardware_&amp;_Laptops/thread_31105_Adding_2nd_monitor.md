### Post by spartas on 2005-05-01
I would need to know more information about the monitor, as well as the computer you're using.  Is it a desktop or laptop, are you using VGA or a digital connector?

If you are using a laptop, check for a button to switch between displays.  I am currently using a Dell laptop on an external flat panel monitor, but had to press Fn+F8 (the CRT/LCD button) to have the screen show up on the external display.  Another side note, for me Fn+F8 does not work in GNOME, only in terminal mode (when the system first boots, or by manually switching over using Ctrl+Alt+F1 or any other text terminal screen).
