### Post by RobsterCraw on 2008-06-15
Sorry to dig up an old thread, but since nobody answered...

I've been using the Zonbu laptop for two months now, and am generally very pleased.  Bought it simply for use while travelling to check business e-mail, but discovered it really is a completely functional legitimate laptop.  I got it almost by accident, as the Asus 4GB Surf PC I first purchased had a defective mouse button, and no replacements were in stock, so I switched to the Zonbu.

For most casual (non-programmer/tweaker) PC users, it offers just about every bit of software you could want pre-installed.  If you need specialized Windows-only software (Quickbooks, etc.) you might have a problem.

In comparison to the Ubuntu 8.04 that I run on my main office PC and home PC, the Zonbu distro is better in some respects, worse in others.

Better in that so much stuff you would need is already there and does not require hunting down - DVD player, for example, no need to use Medibuntu (yah, I'm overseas, wink-wink).

Worse in that you don't[COLOR="Red"]*[/COLOR] have the ability to install other software and the immense resource library that Ubuntu offers through the Synaptic Package Manager.

[COLOR="Red"]*[/COLOR] Technically, not true.  You can give yourself permanent root privileges now that Zonbu has made the Developer package accessible.  With root privileges enabled, you should be able to install any Linux package.  I can't say that it works yet as I just made the modification yesterday and have not had time to play.

Zonbu pricing is an interesting thing; just about two weeks ago, their pricing structure changed radically.  Originally, you could get the laptop for $279 plus $15/month for two years for online storage... or buy it outright without online storage for $479 list, $399 street price, $349 one-time sale at TigerDirect (my lucky timing!).  Now, however, you can only buy it at the $479, no service-augmented lower pricing.  At $479, maybe not such an awesome deal.  Not bad, but not $349.

Would I buy it again?  Likely so... or I might be tempted to find an alternate cheap laptop and install Ubuntu if only for the sake of consistency between all my systems.

Zonbu now offers their distro for download, requires a DVD.  I did try to install it on an older desktop, but it got fouled up because initial boot requires an internet connection.  Without it, the machine got stuck in a "please connect" loop and would not properly re-boot at all.  I didn't have the patience to re-install as I was turning that machine into a cheapie media center, so I can't say how the Zonbu works on a more powerful machine.  That's regrettable as I find myself curious enough now to be searching the office closets for other abandoned PCs!

The one thing I would change with the Zonbu laptop itself, and searching for this is what brought me to this thread, is that it needs more RAM.  With 4500+ songs loaded, Banshee is slow to sort and now FireFox won't launch if Bashee is running because of the song database.  A bit of a surprise, but easily fixed as it uses standard memory as any other Everex laptop.  Warranty voided, of course, but why live in fear?  ;-)

Hope that wasn't a longer answer than asked for.

- Rob
