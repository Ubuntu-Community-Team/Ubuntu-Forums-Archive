### Post by irv on 2006-08-16
This is just to let others know that there is a problem when using a USB Hub when booting up Ubuntu 6.06 Linux. I can make this happen everytime.
I have a self powered USB Hub which I plug into my laptop so I can use a mouse, keyboard, printer, and 60gig external Hard Drive. If I have the hub pluged into the laptop the boot just hangs and will not boot. Even if I take an unplug everything from the hub it still will not boot. I need to unplug the hub before it will boot up.
After it is fully booted, I can then plug it in and everything works great. It finds the hard drive, mouse, keyboard, and printer. I thought I would post this in case someone has the same problem. I am thinking this is a kernel problem not Ubuntu. The Hub is a CyberPower USB 2.0 and the laptop is a no-name with a pentium M intel processor.
Irv
