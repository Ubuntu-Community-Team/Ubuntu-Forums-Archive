### Post by CWayne on 2007-10-03
I'm a recent convert to Linux, using Ubuntu Fiesty 7.04. I have a Toshiba A75 S229 laptop. I was given a live cd by a friend saying "try it, you'll like it". I ran the live cd for a couple of hours and read this forum for a week, then decided to go for the dual boot setup. During the install the system locked up 3 times, apparently damaging the MBR which forced me to forget dual booting with XP. I'm very glad I did!. After completing the install and updates I noticed the following improvements immediately.

1- The screen had a nice uncluttered look, not like XP.
2- The Processor ran much cooler. XP was causing it to run hotter and shut down on over temperature.
3- The system responded much quicker than before, appeared to be not as "bloated".
4- The Processor activity was very low compared to Windows, which always seemed to be very busy.
5- The system booted up and shut down much faster.
6- When forced to power off due to a "lock up" I rarely had any problenm resuming operations quickly with little or no loss of data (I try to save often).

I had no problems with any of the following.
1- USB mouse (Kensington wheel mouse) and other USB devices.
2- CD/DVD R/W Drive (Matshita DVD-RAM UJ-820S).
3- Display (RS300M AGP [Radeon Mobility 9100IGP]).
4- Sound System (IXP150 AC'97 Audio Controller by ATI Technologies Inc).
5- Firewire (IEEE 1394 TSB43AB21 IEEE-1394a-2000 Controller (PHY/Link)).
6- Wireless interface (AR5212 802.11abg NIC, Atheros Communications).
7- Touch Pad & buttons or keyboard.

Problems I did have was that initially Ubuntu would lock up often and I had no control.
I had to power off & back on, which Ubuntu handled with no complaints or problems.
I cut these lockups considerably by getting rid of the screensavers and battery alarm actions. I left the low battery notification active. I also got rid of any programs I had no intention of using.

What I have installed is "Deluge" a bittorrent program, which I think is very good.
Also gtkPod, which I haven't tried yet.
I plan to install AcetoneISO2 for ISO images and to work with "daa" files.
I'm also looking for the best program to handle "rar" files.
I haven't figured out a way to run Rosetta Stone for language study and Ship Navigation programs (I'm a sailor) which are written for Windows.
I don't yet know how to take care of the occasional "lockup" problem.
I don't know how to gain control of the system when this happens.

Any comments, advice or suggestions will be welcomed.
Cliff

### Post by James7 on 2007-10-03
Glad to hear you are having success with your Toshiba. I have a Toshiba laptop too and I feel it has been liberated by Ubuntu--it's a speed demon now and I really like it.

As for your question about RAR files:

1. if you go to System --> Administration --> Synaptic Package Manager and start that, and do a search for "rar", you can install "unrar (nonfree)" (I recommend the nonfree one, this will allow you to uncompress RAR files using the terminal.

2. OR.... apparently WinRAR actually works quite well under wine (you'd have to go again to Synaptic Package Manager, install wine, and then install WinRAR, see for details here:

[http://appdb.winehq.org/objectManager.php?sClass=application&iId=391]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=391")

Good luck with it all ! :)
