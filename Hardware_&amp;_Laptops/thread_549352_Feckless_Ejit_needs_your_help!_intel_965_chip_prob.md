### Post by Nick73 on 2007-09-12
Hi Ive fixed this (at least got 1200 x 800) by doing this

> [ Download xserver-xorg-video-intel_2.1.0-1ubuntu1_i386.deb ]
# sudo aptitude remove xserver-xorg-video-i810
# sudo dpkg -i xserver-xorg-video-intel_2.1.0-1ubuntu1_i386.deb
# sudo sed -i -e 's+vesa+intel+' -e 's+i810+intel+' /etc/X11/xorg.conf
# sudo /etc/init.d/gdm restart

I found it in this thread
> [http://ubuntuforums.org/showthread.php?t=494943&highlight=intel+965&page=5](http://ubuntuforums.org/showthread.php?t=494943&highlight=intel+965&page=5)

### Post by dabl on 2007-09-12
You have removed the wrong (-i810) driver and installed the intel video chip driver that was appropriate for your system -- congratulations!

Wobbly windows come from Beryl or Compiz-Fusion.  If you installed Feisty, I think Beryl is in the repos.  Prepare yourself for more "character-building" if you're going to do that.  But it is pretty cool once you get it working.  :)


BTW, downloading files for installation is NOT recommended for the noob!  There are 22,000+ packages in the repositories, including xserver-xorg-video-intel -- you need to build some proficiency with installing that way before you start grabbing things off the Internet. Linux is VERY different from Windows, in that regard.  ;-)

### Post by dabl on 2007-09-12
Actually, it's kind of a "chicken and egg" situation with the Command Line Interface (CLI) commands.  If you already know the command you're interested in, for example "fsck", then you enter ```
man fsck
``` and all the options for the fsck command will be listed on your console screen.  In other words, the "manual" is baked into the Linux system, and entering "man" and any command will give you the whole deal for that command.

Of course, for the noob who doesn't already know the commands, this is a fairly useless feature. :lolflag:


I bought the book "Beginning Ubuntu Linux" by Keir Thomas. :cool:
