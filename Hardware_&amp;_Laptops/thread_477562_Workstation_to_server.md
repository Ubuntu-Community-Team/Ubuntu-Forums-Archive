### Post by arvevans on 2007-06-18
Dennis

It depends on what you want to do.  Most of the Microsoft functionality is supported in Linux, along with some functions that Microsoft will not do.  However, Linux is not Microsoft.  The application programs to do your tasks are not the same applications that you are familiar with in MS.  For legal reasons it is not possible to make exact copies of a proprietary computing system's applications and then distribute them for free (Microsoft gets a bit upset about that).

If you need Word Processing, Spreadsheets and Presentation applications, then Linux has Open Office (Open office will also work on Microsoft).  For web browsing, there are a number of browsers (Firefox, Epiphany, etc.) that may already be familar to Microsoft users.  For server support, there is Apache, which is also used on many Microsoft server systems.

You will find that the Linux desktop is a bit different from the MS layout, but still functionally equivalent.  In Linux you have the choice of many different desktop handlers (Gnome, KDE, XFce, etc.)  New converts to Linux seem to do well with the standard Ubuntu, but you can also try XUbuntu (a very fast but abbreviated desktop manager) or KUbuntu, which is more like Windows than some of the others. 

Main thing to remember is that Linux is not just another version of Microsoft.  It is different in many ways, and will require a small learning curve to become familiar with it.  The end result will be a more powerful system, but you have to learn how to apply and make use of that power.

Your hardware list seems pretty standard.  Try downloading the Ubuntu Live CD and booting from it first.  This will give you the opportunity to play with Linux without actually installing it on your hard disk and can help you determine if Linux is for you.  This also lets you determine if Linux likes your hardware.

When you download the Ubuntu Live CD, you need to burn an "Image Copy" in order for it to be bootable.  You will also have to set your BIOS Settings so that the CD-ROM is first choice for booting or it will not see the CD to boot from it.

If you run into problems, please post here with a description of the problem and you will find many people who are willing to help.
