### Post by Gary.M on 2007-08-04
If you want to keep access to some Windows stuff, and unsupported printers and scanners fit this... what I did was install Vmware's Player V2. Install Windows XP (get the machine from easyvmx.com) and then install the printers there. My Canon bubblejet is shared from Kubuntu and available on my home network. I print to it from the XP Virtual machine when I want all the capability of the Windows driver instead of the limited output of the Linux one. I could also have connected directly to it via the virtual machine's USB 2.0 connections which I have used to access my unsupported Canon scanner.
