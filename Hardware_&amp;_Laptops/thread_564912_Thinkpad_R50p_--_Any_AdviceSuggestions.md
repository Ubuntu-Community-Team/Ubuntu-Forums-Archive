### Post by scapalexis888 on 2007-10-02
i have a ibm thinkpad x31, with somewhat old specs. Ubuntu runs a lot faster on this machine than windows xp did. Most of the function keys work (brightness, volume, monitor off, and think light). I am still trying to get bluetooth to work, also the page forward/backward keys. But over all, Ubuntu runs superbly on my ibm thinkpad and if you ever need some windows stuff, use VMWare (windows on vmware ran fast for a 1.4ghz processor). Ubuntu definitely runs nicely on thinkpads.
