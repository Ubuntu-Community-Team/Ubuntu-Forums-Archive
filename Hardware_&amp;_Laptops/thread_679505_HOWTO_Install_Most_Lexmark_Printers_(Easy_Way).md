### Post by chewit on 2008-01-27
Hi,

I have been using Ubuntu for about a month. Everything has worked well, apart from my Lexmark Printer. But finally I have got my Z640 printer working. I'm going to show you all how.

1) I downloaded the red hat driver from the Lxmark website.
2) Then I extracted the driver and then the hard part.
3) I extracted the .rpm file from the driver, then the clever part.
4) I converted the .rpm file to .deb.

You can download the files here (download them both):

[http://www.gamersarena.co.uk/z600cups_1.0-2_i386.deb](http://www.gamersarena.co.uk/z600cups_1.0-2_i386.deb)
[http://www.gamersarena.co.uk/z600llpddk_2.0-2_i386.deb](http://www.gamersarena.co.uk/z600llpddk_2.0-2_i386.deb)

(Files hosted on my server)

Install the two packages (install z600cups_1.0-2_i386.deb first and then install the second package), no need for the terminal. Install & then add your Lexmark printer. This driver should work with most Lexmark printers, especially for the Z600series.

Tell your friends about this thread, if they are struggling to get it to work.
