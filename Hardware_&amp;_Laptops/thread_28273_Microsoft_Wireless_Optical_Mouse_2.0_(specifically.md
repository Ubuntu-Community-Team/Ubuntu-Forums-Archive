### Post by teppic on 2005-07-28
I can confirm that the kernel is the problem. I just upgraded the Hoary kernel to the one from Breezy: I added the following line to /etc/apt/sources:

```
deb http://gb.archive.ubuntu.com/ubuntu breezy main restricted
```

I then ran 'apt-get update' and installed linux-image-k7. After a reboot into the new kernel (2.6.12-5), the mouse problem has gone away completely and it works fine.

This isn't ideal obviously - I don't want to be running an experimental kernel.
