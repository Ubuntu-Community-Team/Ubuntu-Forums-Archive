I've been trying to get my resolution to its native 1280x800 on my Dell Inspiron 710m.

I've followed all the suggestions in the threads using the 915resolution tool.
I'm using: "sudo 915resolution 73 1280 800" and everything is set properly. Once I reload the interface (ctrl-alt-shift-backspace) the resolution will switch to 1280x800 appropriately.

The issue arises when I reboot the system after adding the 915resolution command to run at startup, the 1280x800 resolution is set within 915resolution but the system will not let me switch to it. If I reload the interface it automatically switches to the proper 1280x800.

I'm not sure what to do here. Everything works perfectly but I have to reload the interface everytime I reset???

Any help would be appreciated.
