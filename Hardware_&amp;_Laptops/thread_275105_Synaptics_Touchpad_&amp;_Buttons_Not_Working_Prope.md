### Post by metiosarius on 2006-10-11
Well, I figured out what the problem was - and it was REALLY simple; and now im thinking **DUH!**

I just had to change the driver line to "synaptics" and all was fixed and everything is working how it should. :D

```

Section "InputDevice"
	Identifier	"Configured Mouse"
	Driver		"synaptics"
	Option		"CorePointer"
	Option		"Device"		"/dev/input/mice"
	Option		"Protocol"		"ExplorerPS/2"
	Option		"ZAxisMapping"		"4 5"
	Option		"Emulate3Buttons"	"true"
EndSection

```

Thanks for the help :)
