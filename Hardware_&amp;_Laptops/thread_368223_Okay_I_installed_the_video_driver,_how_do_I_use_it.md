### Post by Stephen Howard on 2007-02-23
After making a backup of xorg.conf, where it says:

[FONT="Courier New"][INDENT]Driver		"vesa"[/INDENT][/FONT]


change it to:

[INDENT][FONT="Fixedsys"]Driver      "sis"[/FONT][/INDENT]


Cross fingers and reboot.  If it doesn't work, you'll end up at a command line prompt, just revert back to the backup xorg.conf for the time being and tell us about it.

If it does work, here's a link to some [configuration options]("http://www.die.net/doc/linux/man/man4/sis.4.html") for the sis driver.
