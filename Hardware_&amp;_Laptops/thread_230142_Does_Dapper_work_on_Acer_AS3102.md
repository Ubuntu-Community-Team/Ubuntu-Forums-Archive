### Post by bagboy on 2006-08-17
Hi!   I'm running Dapper on an Acer AS3102 as I write this.  The internal wirless was recognized and configured during the install.  The Audio works just fine and the video identifies at ATI Radeon 200M even though it is actually an ATI Radeon 1100.  I'm relatively new to Ubuntu but so far it seems to be working fine.  There are some quirky messages at boot about PCI resources not be reallocated, but I haven't had time to research it yet.  I think there may be some messages about that.  All in all, it seems to be working fine.  I will say that I came to Ubuntu because Fedora Core 5 would NOT run on this laptop.  Strange error message about PCI BIOS Bug #81 and udev hanging the machine at boot.  Ubuntu installed cleanly and even with the PCI resource messages, it boots and runs well.  Hope that helps!
