### Post by AR_Kozz on 2007-06-30
I've got Edgy installed on my ps3. It works better than I expected. Video is a bit choppy though, because the RSX graphic accelerator is not accessible to an installed OS. I haven't installed any codecs so I haven't done much experimenting with that.

  I run it on a sdtv, and it looks surprisingly good. (the official YDL distro doesn't work good with sdtv) I've played some older games that can run without the RSX, and they look good.

you'll get 1600 x 1200 resolution easily if you have an 1080p HDTV or a HDCP-able monitor. 720p is 1440 x 900 resolution.

Unless they give linux some access to the RSX, my ps3 running linux won't really be much of a video player, but that's 
not much of a negative, because the ps3 with its native OS is OUTSTANDING for video, sound, and everything else. Including some fine upconversion, even of ps1 and ps2 games.

set up a usb hard drive to share between the ubuntu install and the sony XMB, and you can use the ubuntu install to do things like, say, edit or transcode video, then switch back to the XMB when it's time to view it.

I think I've seen what you read about swap on psubuntu. that thread made me scratch my head because ubuntu creates swap memory on the hard drive, by default, anyway... increasing the amount it creates could make it run faster, but I haven't needed to do anything like that just yet.

 finally the ps3 with XMB performs many functions of computers itself, including web browsing, storage and playback of music, video and image files... not to mention the games which, though still not a great many, are the best anywhere.
