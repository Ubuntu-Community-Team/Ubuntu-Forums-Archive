### Post by beefsprocket on 2005-11-02
You might want to run an md5 checksummer against any .iso files that you download in the future. Iso files from torrents are safe (don't need checking as long as you trust the tracker). Also, once you burn the iso, run your burner program's verify feature against the newly minted disc. It sounds like your media went bad -- the installer will choke up if it cannot perfectly copy data from the disc to your hdd.
