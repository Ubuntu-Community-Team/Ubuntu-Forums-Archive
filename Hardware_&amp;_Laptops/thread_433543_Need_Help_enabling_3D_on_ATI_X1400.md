i had similar problem

```
enigma@kubuntu:~$ fglrxinfo
Xlib:  extension "XFree86-DRI" missing on display ":0.0".
display: :0.0  screen: 0

```

what i did ,  i got rid of fglrx thing - means purge it from your system ( xorg-driver-fglrx, restricted modules ect.) ,
```
sudo aptitude purge
``` ( i am using ubuntu)

i also did cleanup in etc/X11/ and deleted all old xorg.conf files (to have clean config) , then i made

```
sudo dpkg-reconfigure xserver-xorg
```
and choosed vesa as driver, rebooted - so i achieved "clean without fglrx system" , 

you can turn off restricted manager(you dont need it) disabling it in system-preferences-sessions-start up programs

then i followed the how to to install alternative fglrx driver from ati


it is not good looking how to i know,  , but it worked for me :)
