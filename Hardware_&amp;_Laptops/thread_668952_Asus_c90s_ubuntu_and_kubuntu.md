### Post by gm75 on 2008-01-15
Hi everyone

ive just finished installing linux and windows dual boot on new c90s.  i have windows xp installed for games but havent been in it much since i got wine working for steam and starcraft.

i tried ubuntu first but found that wine does not run correctly in gnome for whatever reason.  the game would run but would not go over the taskbar so games could be played but you would have to look at your taskbar all the time.

next i tried kubuntu and found it to work surprisingly well.  the restricted driver worked perfectly for the video card.  basic Fn keys work, such as brightness and volume, but the wireless Fn does not work, wireless does work though just not the Fn key.  other than those the only thing that does not work is the built in camera and finger print reader.  the devices are recognized by the system but i cannot get it to work with any programs. guess its a matter of drivers??

all in all the 7.10 ubuntu and kubuntu both work without any further manipulation, beside the restricted driver install (which is painless).

ive also tested kubuntu alpha 3 but it has alot of problems.

*** do not upgrade the system for any reason.  as soon as i upgraded through the system tray icon the system was unusable.  tried several times and it only resulted in me formating and reinstalling

hope this helps anyone looking to buy a c90s
