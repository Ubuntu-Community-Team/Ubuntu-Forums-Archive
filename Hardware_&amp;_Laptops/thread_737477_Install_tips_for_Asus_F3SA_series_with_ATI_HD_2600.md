First Post!

Folks, I've had quite a time getting my new laptop squared away.  If you've got an Asus, particularly an F3Sa series with an ATI HD 2600, listen up!

1. Vista Disk Manager can resize partitions, including the current active partition!  Make room for Ubuntu using the Disk Management tool, incredibly easy.  Unlike everything else in Veestuh.

2. Ubuntu 7.10 will install just fine from CD

3.YOU NEED to get the 'Envy' Display Driver ute here: [http://www.albertomilone.com/nvidia_scripts1.html](http://www.albertomilone.com/nvidia_scripts1.html)

DO NOT attempt to manually configure the display driver yourself!

I haven't done anything with my wireless card yet; probably easier to make my self a very, very long ethernet cable.

Oh, and check the make for your HD; if it is a Hitachi, you REALLY want to go and get the Hitachi testing utility: [http://www.hitachigst.com/hdd/support/download.htm](http://www.hitachigst.com/hdd/support/download.htm)

My HD died after less than 72 hours of use, bad sectors all over the place.  Asus was good about the replacement, but they give you the same model, beware!

Hope this helps!
