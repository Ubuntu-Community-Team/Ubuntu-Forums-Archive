### Post by fredj on 2007-08-19
The older television type CRT computer monitors suffered from a problem known as burn in.
If you displayed the same image for a long time it would burn into the screen and couldn't be 
removed. The screen saver (which displays a continuously changing image) was developed
to avoid this problem.
LCD displays don't suffer from image burn in but they do have a related problem called image
persistence. If you display the same image for a long time then it will appear as a background
image. You should be able to remove it by displaying a white screen for at least 30 minutes.
