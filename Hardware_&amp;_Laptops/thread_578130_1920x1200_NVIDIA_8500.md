### Post by dfreer on 2007-10-16
> (EE) Failed to initialize GLX extension (Compatible NVIDIA X driver not found)
Sounds like you don't have your driver installed. This should take care of that, provided your system is in a consistant state after using envy and/or automatix.
```
sudo apt-get install nvidia-glx-new
```
After you do that, backup your /etc/X11/xorg.conf file, and modify it.
```
Section "Device"
Identifier "NVIDIA GeForce 8500 GT"
**Driver "nvidia"**
BusID "PCI:3:0:0"
VideoRam 512000
EndSection

```

Reboot. If x-server crashes, simply restore the backup instead of reconfiguring the X server. Make sure to copy that error log again, and once you can get online post it again. If it does work, then you can try using *sudo nvidia-settings* to change your resolution, or you can add the 1920x1200 mode directly in your xorg.conf file like so:
```
Depth **24**
**Modes "1920x1200" "1024x768" "800x600" "640x480"**
EndSubSection
EndSection
```

If you still have problems with the nvidia driver, try using the **nv** driver instead, and add that 1920x1200 resolution manually. As always, feel free to ask any questions you might have!
