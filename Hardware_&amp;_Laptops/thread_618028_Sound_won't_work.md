### Post by garnetrook on 2007-11-22
I have a gateway e-100m and I never got sound to work with Ubuntu -- I've only been using linux for a few months, but I've been trying for weeks and still nothing.  Last time I installed Linux Mint (ubuntu-based) it seemed to have the stac9200 drivers, but they still didn't work.

BUT! Sound has worked for me both with Fedora Core 8, and Pardus 2007.3, so I would try one of those!  Personally, I thought Fedora had terrible wireless (it looks like it's probably a temporary, new-release sort of issue) and was a bit bloated, but Pardus looks and runs great!  Plus Pardus has proprietary codecs (mp3, wav, etc) installed out of the box.

If you have 2 blank CD-Rs lying around, it'd definitely be worth it to LiveCD these guys just to see if you can hear the boot-up sound.
