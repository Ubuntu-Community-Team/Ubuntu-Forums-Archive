### Post by genbuntu on 2007-10-31
Hi

I finally found a workaround . Instead of restarting the computer one can change to terminal session and back to graphic session by first pressing ctrl+alt+F1 and then ctrl+alt+F7.  This should resume the 'garbled' display back to normal.
