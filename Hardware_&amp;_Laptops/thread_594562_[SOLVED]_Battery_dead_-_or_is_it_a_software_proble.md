### Post by kvonb on 2007-10-28
Let it run until it turns itself off, that will give you a real idea.

It might be that it has it's calculations wrong, mine does that sometimes.  I just ignore it, it seems to sort itself out eventually :).

You might have to turn off sleep/power down in power management first. otherwise it will sleep/shutdown when the power manager thinks the battery is too low.

It shouldn't hurt it, just don't run any software during the test.

PS Don't forget, running from the mains constantly will ruin your battery, always try to "exercise" the battery by using it until it's nearly dead, recharge, then disconnect the mains/charger and run it down again.
