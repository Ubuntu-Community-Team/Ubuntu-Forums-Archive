### Post by giosetti on 2007-01-23
Xubuntu will be the variant of choice for you since it works very well on machines with limited ressources.

I recommend you to goto [http://www.xubuntu.org/get#requirements]("http://www.xubuntu.org/get#requirements")

As you see you can install xubuntu with the alternate install. I have done this with a Latitude CPi 366  (128 RAM) as well as with a Latitude CP 300 (64 RAM). Installing isn't difficult. 

Don't install with the Desktop CD - it'll take ages. 

The only problem I have with the CPi 366 is that it freezes frequently, though not always, on booting. I have started a separate thread. The CP 300 boots impeccably.
