### Post by MistaED on 2005-10-24
I would say that 192mb chunk of ram will make Kubuntu/Ubuntu run fine. You should be able to run most programs you can with x86, the open source ones anyway like firefox, gaim, etc. Stuff like skype won't.

My experience with a Ubuntu Hoary Live CD and an Imac DV 400mhz /w 64MB was slow and painful, but this was probably due to the ram. I've noticed CPU isn't as much of a factor as the ram. If you find Kubuntu or Ubuntu to be too slow, I suppose you could try that Xubuntu which uses XFCE4 for the Desktop Environment. Try out a live CD first I guess.

Hope this helps a bit.
