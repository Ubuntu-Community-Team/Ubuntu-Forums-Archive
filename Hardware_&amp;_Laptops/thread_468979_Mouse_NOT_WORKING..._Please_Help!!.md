### Post by DagMan on 2007-06-10
can you post the contents of /etc/X11/xorg.conf

We're looking for this at the bottom in a section called **"ServerLayout"**
```
        InputDevice     "Configured Mouse"
```
And near the top a 
section lookimg something similar to this
```
Section "InputDevice"
        Identifier      "Configured Mouse"
        Driver          "mouse"
        Option          "CorePointer"
        Option          "Device"                "/dev/input/mice"
        Option          "Protocol"              "ImPS/2"
        Option          "ZAxisMapping"          "4 5"
        Option          "Emulate3Buttons"       "true"
EndSection

```

If you don't have those, add them and restart the graphics.  If you do, still post the contents of that file for other people to look at.
