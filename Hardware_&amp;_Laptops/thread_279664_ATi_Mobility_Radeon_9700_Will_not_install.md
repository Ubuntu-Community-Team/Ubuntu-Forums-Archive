### Post by matthew on 2006-10-18
I have that exact card working beautifully under Dapper. Here's how I did it.

Open a terminal and enter these one at a time.
```
sudo apt-get update 
sudo apt-get install linux-restricted-modules-$(uname -r)
sudo apt-get install xorg-driver-fglrx fglrx-control
```
Once the above packages are correctly installed, then do the following

```
sudo aticonfig --initial 
sudo aticonfig --overlay-type=Xv
```And finally
```
sudo dpkg-reconfigure xserver-xorg
```This will put you through a series of menu choices to set up your video card. Make sure and choose "fglrx" as the driver you want to use and when given a choice highlight the video resolution that is native to your laptop. When in doubt, choose the default options.

Then reboot.
