### Post by heavyboots on 2005-05-06
I accidentally built something similar to this up recently--same mobo at any rate, an AMD 3200 and a Maxtor SATA 300gb drive (planning on making a home theater system). I went with the Aria case, which is pretty decent and has a *lot* of modders fiddling with it.

The Ubuntu report is that while I got it to boot, I'm here trying to figure out how in hell to get the SATA recognized. So far, Knoppix, Gentoo and Ubuntu have all failed to recognize the Maxtor drive, which is posted way out on IDE channel 5--I may crack the case open in a second and see if I can at least get that down to 4.

Also, since my case has no floppy I haven't gotten around to doing the latest firmware upgrade for the mobo, which may help with this issue.

In any case, that's what I've got so far... I'll post back if I find someone who got the lil' sucker working.
