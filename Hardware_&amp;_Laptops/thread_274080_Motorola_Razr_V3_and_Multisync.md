### Post by elektronaut on 2007-01-06
I'm using a Motorola e770v, which is the budget version of the v3x. I can send files via bluetooth on the command line like this: 'gnome-obex-send -d 11:22:33:44:55:66 file' where the number has to be the phone's address which you can get via 'hcitool scan'. There's a bug in gnome-bluetooth in Edgy, otherwise the graphical method in nautilus would work. About the OTA SyncML support mentioned above: I'm using this with scheduleworld, which is a great possibility to sync with other stuff also, like evolution and Google calendar.
