### Post by mistywindow on 2006-05-16
I'm a total newbie. Yesterday I installed Ubuntu 5.10 to dual boot on my desktop and I'm delighted with it.

Now I'm trying to install it on my 15.4" screen laptop (Asus Z92U, AMD Turion). Unfortunately the default screen resolution is 1280 x 800 and the installation program doesn't recognise that.

I lose most of the lower part of the screen, except for the very bottom part which shows at the top! The selection will scroll down but the screen won't.

I can't see the choices to make for partitioning.

Is there a solution to this? A command I should enter at the beginning of the installation maybe.

I'm an experienced Windows user, but almost Linux illiterate.

---

### Post by mistywindow on 2006-05-16
Found the answer thanks.
Command line:
*linux vga=771*
