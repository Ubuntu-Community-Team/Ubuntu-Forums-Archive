### Post by Kateikyoushi on 2006-10-15
Try to reconfigure your X server with the following command.

```
sudo dpkg-reconfigure xserver-xorg
```

In case it still doesn't work after that post your xorg.conf by copying this line to a terminal, and the output back here.

```
cat /etc/X11/xorg.conf
```
