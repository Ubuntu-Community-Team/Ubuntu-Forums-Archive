### Post by gsporter on 2008-07-22
VIA doesn't offer much in the way of end user support.  I believe these boards are marketed throught 

iDOT  - [http://www.idot.com.tw/en/index.jsp](http://www.idot.com.tw/en/index.jsp)

However they only have Windoze drivers.  It runs xp fairly well for net appliance type application.

Also Walmart is marketing a computer based on either the PC3500 or the PC2500 (main differance I have found is support for SATA)
It use gOS which is based on Ubunutu and runs well.

see here: [http://www.thinkgos.com/](http://www.thinkgos.com/)

It runs well.  I also have used these boards for a number of IPCop fw, an Openfiler NAS boxes.

Not sure about max resolution. It uses shared RAM (video/system) so max it out on RAM.

I have not used the 12-32V 80W picoPSU-WI-32V specifically but heat should not be an issue as it is generated in the "external brick" transformer.   Need to check the wattage and number of connectors ie IDE, SATA , PCI-E

Good luck.  I am still looking for the PC3500G-H with the video out for a multimedia box!!!
