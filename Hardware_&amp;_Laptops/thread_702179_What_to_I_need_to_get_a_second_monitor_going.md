### Post by oldsoundguy on 2008-02-20
Think your choice will be replacing the on board with a card.  Have not researched PCI-e, but what you really need (for future expansion if for nothing else) is a DUAL DVI output or HDMI.   You can always dumb them down with adapters to run a standard ratio CRT, but should you ever decide to go for it (like I did) by getting a pair of wide screens and getting cinema displays, you will have the card installed already and the driver system will be there 
(BTW, found it exceptionally EASY to download and install ENVY on the Ubuntu machines and to use that to install the cards.  Then everything works.
Granted, it is so "unlinux" to use an installation program, but if it works, why not?
When I first started using linux, there wasn't such a thing as a package manager and there were no repositories for programs .. and that was NOT FUN to get a system talking. It was interesting but not fun and it took lots of time, energy and some hair loss.
