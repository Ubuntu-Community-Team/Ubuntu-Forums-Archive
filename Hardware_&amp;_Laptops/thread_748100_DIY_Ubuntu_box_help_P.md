### Post by insineratehymn on 2008-04-07
1) Yes, that motherboard would make a **mean** server board, in fact... its quite similar to the one I'm using on my server currently.

2) Case-built USB connectors + Old motherboards = A very scared me. You never know if/how those things will work. I've even seen them do damage to systems. If you got a new MB, use it with the case USB, if not... get a PCI USB hub, or an external USB port.

3) Cheap is Linus Torvald's middle name (I'll save all of you know-it-all's the [trouble]("http://en.wikipedia.org/wiki/Linus_Torvalds")). What I'm really saying is, you could probably run this system on a computer that would choke on XP.

4) I really like Antec's stuff.

EDIT:

BTW, a server is supposed to sound something like a jet taking off.
