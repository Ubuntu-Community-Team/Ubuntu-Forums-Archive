### Post by dave9191 on 2005-07-19
Lo is that localhost connection that the computer has to itself. One of the thing that its used for it letting you access services that you run on your machine like a web server through the localhost address. Im sure that its used for other stuff as well and should be left on. 

Dave

### Post by dave9191 on 2005-07-19
If you have not enabled any extra repositories and installed software from them, then you have nothing to worry about, the ubuntu team make a fairly secure distro. If you have installed extra software from outside what the ubuntu team put together then you might want to setup a firewall for that added bit of protection. But the lo device shouldnt be a security risk.
