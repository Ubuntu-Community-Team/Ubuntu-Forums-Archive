First, I apologize in advance if thios question is yet another thing that you can only do in Windoze, and you can't do in Linux.  

Second, my question is, is it possible to get the Middle Mouse Scroll Button working like in Windows, where you click upon it, and it comes up with those little up and down and side-to-side arrows and allow you to scroll by moving the mouse up and down, and not just using the scroll?  If so, how?

### Post by GreyFox503 on 2005-09-12
If you use Firefox, then type "about:config" in the address bar.  This will bring up many advanced settings.

At the top of the page where it says "filter" type "autoscroll" (that's what that feature is called).  You should see one line that says "general.autoScroll".

If it is not set to true, right-click on it and select "toggle".  This should change it to true and allow you to use autoscroll with the middle click.
