### Post by jjordan on 2007-05-08
It depends on the motherboard some will work with diffrent modules and some will not.  Is this a laptop?  Put the 1GB in DIMM A slot and see if it boots.  If it does not then remove the 256MB from Slot B and try again.  It is generally better to put the RAM in the first slot.  You probably will not be able to notice the difference between 1.25GB and 1GB.  

If this is a laptop make sure you have enough space before you try to hibernate.

Your swap partition will be fine unless you are set up to hibernate to it and it is smaller than your total RAM +a little.
