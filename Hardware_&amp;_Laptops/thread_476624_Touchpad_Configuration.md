### Post by nachotronics on 2007-06-18
Hello, you could try to install gsynaptics
```
sudo apt-get install gsynaptics
```
Its a configuration tool for Synaptics touchpads. After installing you can access it on System -> Preferences -> Touchpad or just running "gsynaptics" on a terminal window
