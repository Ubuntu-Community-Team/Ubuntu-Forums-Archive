### Post by dabl on 2007-09-06
GF 7900 GS is a great graphics card!  Mine does 17,000 fps in glxgears, before overclocking.

@bimmer, it all looks good except that PSU.  I tried an Antec 450W and it powered on exactly once.  Period.  I replaced it with a TruePower Trio 650, and never looked back.  I'm running 5 hard drives with it, and an overclocked Core 2 Extreme.  So, I'd recommend you go ahead and get a stronger PSU.  :)
