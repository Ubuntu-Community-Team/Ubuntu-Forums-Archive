### Post by jimrz on 2007-09-25
Sounds as though the problem is occuring while you are using windows. Is the machine dual boot? And, if so, does the same problem occur when using ubuntu? If it does not work in either your nic may have gone south. In windows I guess the first thing to do would be to go to Device Manager and check that there are no warnings on the device. If all appears ok, you might try re-installing the driver (or rolling it back if there has been a recent update) or, if still no luck, removing the device the rebooting and see if it works after windows re-installs the device.
