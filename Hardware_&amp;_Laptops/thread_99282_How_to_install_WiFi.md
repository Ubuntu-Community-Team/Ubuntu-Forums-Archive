check the output of this:

ifconfig -a 

If you have eth0 and eth1 there, then WLAN support is most probably already present, you just need to configure it. Ask guys at [Kubuntu Desktop support forum](http://ubuntuforums.org/forumdisplay.php?f=106) to suggest you a nice application in that case. :)

**Edit:** you may want to check [this thread](http://www.ubuntuforums.org/showthread.php?t=38942), as it already contains quite a few links to nice apps.
