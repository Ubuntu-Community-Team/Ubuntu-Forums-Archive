### Post by Rinias on 2007-02-26
There's nothing blaringly wrong (to me) with your xorg.conf... The only difference I have is that I have a "BusID" line in the device section...

What exactly is the problem? When the computer tries to start the graphics, does it not get there? If so, are you getting a blue screen asking you if you want to see the error log? If you are, then that log would be useful, especially any lines that start with (WW) or (EE).

I had a problem as well with my card (GF 6100) but I did a 

sudo dpkg-reconfigure -phigh xserver-xorg

and entered the values I wanted but with the "nv" driver. After that I went along and edited xorg.conf

sudo gedit /etc/X11/xorg.conf

or

sudo nano /etc/X11/xorg.conf 

(second option is if you have no graphics - after making the necessary changes, you do CTRL-x then Y then enter) and I changed just "nv" to "nvidia"

Worked well :)

Please post some more information (exact errors, etc) so that we can debug :)

Riri
