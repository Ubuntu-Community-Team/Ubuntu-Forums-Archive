### Post by kidders on 2007-02-09
Hi there,

It's been a long time since I used Mandriva/Mandrake, but if I remember correctly, it actually prompts you to configure hardware changes when it boots. Ubuntu doesn't do anything like that, but it should still be able to handle the switch ... provided Mandriva would've been able to.

Assuming the motherboards/CPUs aren't wildly different (eg different architectures), you should be able to get away with making some minor tweaks, like you mentioned. The only way to know for certain is to try it though.
