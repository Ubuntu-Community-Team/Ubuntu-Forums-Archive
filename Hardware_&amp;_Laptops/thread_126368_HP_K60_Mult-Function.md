### Post by jwh400 on 2006-02-06
I've read a lot of horror stories about printer configuration with Linux so I thought I'd share a success story. The HP K60 is a bit quirky to install and set up even with Windows and I had to go through the process twice with both Windows 2000 and XP but after doing a lot of lurking here I decided to try it. I've included a couple screenshots and the first shows everything I selected in the package manager. I'm not sure if I needed hpoj or not but I selected it and haven't had a problem so far. 
[[IMG]http://img24.imagevenue.com/loc109/th_601c4_Package_Manager.jpg[/IMG]](http://img24.imagevenue.com/img.php?loc=loc109&image=601c4_Package_Manager.jpg)
After the package install I then set up the printer which was the easiest process I've encountered for this model yet. As you can see, the printer was found and I set it up as "Local" and in the second step clicked "Install".
[[IMG]http://img144.imagevenue.com/loc57/th_87eae_Printer_Install.jpg[/IMG]](http://img144.imagevenue.com/img.php?loc=loc57&image=87eae_Printer_Install.jpg)
From the "Printers" window I right clicked on the K60 icon and selected "Properties" and succesfully printed a test page. In all, the process took about 10 minutes.
It was from all the posts here that taught me where to even begin doing this because I didn't have a clue since I'm new at using Linux. Thanks to everyone for guiding another newbie.\\:D/
