### Post by Stebbins on 2007-09-18
I have been unable to boot Ubuntu since trying to upgrade to feisty.  I can still access the root file system when I use run the "Rescue a Broken System" option on the Alternate CD.  What I would like to do is copy my important files to a flash drive and then reinstall Linux.  But I can't seem to access my USB drive from the rescue shell.  How can I mount my USB drive in rescue mode?

Thank you,
~Stebbins

EDIT:  The solution for this turned out to be really simple.  All I had to do was run Ubuntu off of a normal LiveCD.  This gave me read access to all of the files on my computer, and also automatically mounted my USB drive.  I was able to salvage my files with ease.
