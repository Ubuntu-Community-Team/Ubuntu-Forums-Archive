### Post by schallstrom on 2007-08-09
If you are using Gnome, which is default in Ubuntu, it would be the easiest way to go to *System -> Preferences -> Mouse* and then *Motion*.

If you want to do it the hard way, edit your */etc/X11/xorg.conf*, but I don't know the correct config option.

**Edit:** If you want to do it with xorg.conf, have a look here: [http://www.linuxquestions.org/questions/showthread.php?t=425552](http://www.linuxquestions.org/questions/showthread.php?t=425552)
