### Post by deadlikeoscar on 2007-06-08
First thing I would do is put the cd in (as long as it is the live cd that is the default download) and see how it works. If your video is screwy and not detected then you may need some assistance but if it loads up (you don't have to install it, it runs off the cd) then you know that your stuff should be supported out of the box.

To answer your questions a little better. The installation is said to be easier than Windows as long as you are installing Ubuntu to the whole hard drive. If you are looking to dual boot you are going to need to do some research but even that is okay. Ubuntu is very good at detecting your hardware. Like I said, if the live cd boots up fine then you can go ahead and click the install icon on the desktop and it will begin the installation. There are a lot of great guides on using Ubuntu. If you have trouble locating some I'm sure we can help you track some down. It's not as bad as you think, I would just play around and have fun. The only time Linux can be hard is when something doesn't work and I am actually thankful for those times because it helps me become a better Linux user.
