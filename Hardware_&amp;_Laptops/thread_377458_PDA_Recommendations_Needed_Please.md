### Post by Catsworth on 2007-03-06
Hey Guys,

First off, apologies if this is in the wrong place - it seemed like the most logical place to put it.

I'm looking to buy a PDA of some sort, it doesn't have to be too expensive/fancy (in fact cheap and cheerful is probably the order of the day).

My requirements, in no particular order, are:

1.  Should be able to easily transfer files between it and my Ubuntu devices (currently 1 laptop and 1 server on a wireless network, USB to the laptop would be fine).
2.  Should have capability of opening multiple file types (.html, .pdf, .txt) as I want to use it mainly as a reading platform so that I can tote technical papers around with me for reading wherever I am.  For this reason it would also be nice if it had a backlight and a good clear screen.
3.  Should have a calendar functionality, although this doesn't have to be anything too exciting, it would be nice if this could synch with Thunderbird but that's not too great an issue.
4.  A colour screen would be nice as this would make reading the various files that little bit easier.
5. Wi-Fi would be nice, although USB/Ethernet would be an acceptable alternative, if it's got Wi-Fi then it will need to support WPA so that it can connect to the wirless network.

On top of the above.  If it's capable of running a Linux OS then that's great, I would prefer to steer clear of Windows/MS products although I'm not totally Anti them.

I'd also be interested to know what sort of storage options are on the devices that are being recommended, for example do they take memory cards/sticks of some sort etc?

Thanks for the help guys.
