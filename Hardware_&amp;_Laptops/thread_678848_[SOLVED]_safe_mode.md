### Post by taurus on 2008-01-26
When you boot your machine, you would see a message from the BIOS.  Then, you have three seconds to press Esc to bring up GRUB menu.  The second option is the recovery mode which is what you want.  Once you boot into recovery mode, you can reconfigure your X server again with

```
dpkg-reconfigure -phigh xserver-xorg
```
or you can edit /etc/X11/xorg.conf directly if you wish.

```
nano -Bw /etc/X11/xorg.conf
```
Save the changes with <Ctrl>o and exit with <Ctrl>x.  Then, reboot your machine with

```
shutdown -r now
```
