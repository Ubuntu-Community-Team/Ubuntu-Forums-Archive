### Post by root.user1 on 2008-03-12
I ~do~ manage to kill Seagate drives... well, at least I did until I stopped buying them.  I only buy WD drives now.  I have a nice Raptor here and just bought a 650GB WD drive last week from newegg.com.

I'd suggest that you visit newegg.com and read the reviews on various drives, motherboards, and other peripherals.

I set my desktops up with drive trays and caddys - I can cold swap a physical drive and use the machine to boot another OS without dual booting.  I can also boot Linux up on the boot drive and have a Windows harddisk in another tray so I can peel DLL files and whatnot off when I'm tweaking Wine for something.  The removeable drives make the desktop computers so much more flexible.

Virtualbox works fine for me but I don't game in Windows.  I use Windows 2000 and run a couple of very expensive applications that I need.  With Windows 2000 I don't have to deal with Microsoft bothering me about when, where, & why I'm installing the Windows OS.  I have XP and Vista licenses here, but since Microsoft started treating everyone like thieves, I just use 2000 and Linux.  If you want freedom and your apps will run under Windows 2000, check eBay out for a copy.
