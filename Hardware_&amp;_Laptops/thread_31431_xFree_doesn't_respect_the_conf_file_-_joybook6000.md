### Post by BeBoxer on 2005-05-03
Two things to try. Just logging out and logging in does not restart the X server, I don't believe. When you are logged out, you can do Ctrl-Alt-Backspace to kill the X server. The gdm process will then restart it. I believe this is the fastest way to ensure that X is starting from scratch. A reboot would work as well, but that takes longer of course.

Second, there will be a log file in /var/log/Xorg.0.log (or some similar name). Check it, looking especially for any resolutions which it is rejecting for some reason. Maybe it doesn't like that modeline for some reason, and is falling back to a more default resolution.
