### Post by mirtux on 2005-02-01
Hi,
           maybe it is not useful for you because it's not regarding the laptop you want to buy, but i hope it can be useful.
 I own an Acer Travelmate 8005, with 1.8GHz Dothan Centrino processor, now working with Ubuntu 4.10. When i installed for the first time Ubuntu, i was coming from Fedora Core 3: it's a very nice distribution but i think it doesn't have, at least from my experience, a very good laptop support (maybe noone of the core developers has done an extensive trial with laptop); for example the throttling CPU was not working in automatic installation. 
         When i installed Ubuntu all things went smootly.... [img]http://www.ubuntuforums.org/images/smilies/eusa_clap.gif[/img] no problem regarding my laptop features. The only point (but it's not depending from the distribution) is that CPU throttling only works with **acpi-cpufreq **module and not with **speedstep-centrino **(native for Centrino) modules. I hope you have a good connection with Internet because the basic Ubuntu installation is not rich of packages..
      
      However now i'm a very happy Ubuntu laptop user... 
      
      Regards,
      MC
