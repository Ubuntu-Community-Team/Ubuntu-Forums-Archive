### Post by snowpine on 2008-01-31
Hi Benji,

1) The eee does not have a CD player... you can attach a USB CD player, or you can play ogg/mp3/etc. files
2) You can use removable media like a SD card or a USB drive for your data, this will free up more space for applications
3) maybe/maybe not... depends on what you need
4) the eee has between 2 and 8 gigs depending on which model, but you could always use a SD card or hook it up to an external drive if you have a lot of files. Or, check out something like google docs... all your files live on the internet, so you don't actually store them on the eee. 
5) the built in Xandros, I don't know... but if you install eeexubuntu, then obviously you would be using ubuntu's synaptic package manager

good luck! check out eeeuser.com for answers to all your questions...

sp
