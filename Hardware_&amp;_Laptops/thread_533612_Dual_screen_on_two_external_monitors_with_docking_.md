### Post by tseliot on 2007-08-24
type:
```
sudo nvidia-settings
```

get to X Server Display Configuration, and you will see a "Display" frame containing "Model", "Configuration", "Resolution".

Select "Configuration" and choose "Separate X Screen".

Then click on "Save to X Configuration File"

log out and press CTRL+ALT+Backspace (with the external monitor plugged in)

### Post by DiThi on 2007-08-24
Once you set the twinview configuration in nvidia-settings (not separate X servers as the previous post suggested):

- in nvidia-settings, save the xorg.conf to /tmp
- type in terminal:
```
cd /etc/X11
sudo cp xorg.conf xorg.conf.bak
sudo cp /tmp/xorg.conf .
```
- restart X

Once you have twinview when your window manager starts, it should pop the windows in one screen or the other one, not inbetween. Also maximizing works for either screen. When not maximized, you can move the windows freely between both desktops.

I don't know how it works in gnome, but in KDE:
- the panel is either in one desktop or the other, if you want both, add another panel
- you can set wallpapers for each screen separately

You have two advantage by being both screens the same size: if you restart the X server and only 1 screen is attached, it works as one screen. Also you don't have death zones (which are not a problem when windows maximize for each screen)
