I've only used external video on Dell laptops; on those machines the external video is set with a keytouch (Fn + F8 or whatever) and for me, if the external monitor and native screen resolutions didn't match, one won't show. In my case I had a 1600x1200 LCD running out to a 1280x1024 monitor, and they both wouldn't run together. It was only one or the other.

I don't know if I've been any help at all. Remember that your BIOS might determine if the video is piped to the external jack or not. :-k
