### Post by nmanglani on 2006-08-04
I'm running Ubuntu 6.0.6

My laptop (Dell Precision M60) would hang whenever I would start the computer on battery power. This occurred after I installed my wireless card using ndis wrapper and net manager. When I booted with the adapter plugged in it would work fine. I saw somebody had a similar problem and didn't have a reply for a solution and I couldn't find the thread again to reply.

I got around this issue by changing my wireless card settings to start and stop with "Application" rather than "Fn + F2/Application" in my system Bios. I hope this helps.
