### Post by DaveT on 2008-01-03
Very late reply but hopefully usefull to anybody else thinking about buying this laptop (aspire9300).
 
works well... eventually
 
problems
 
Xorg is a pain to set up with the screen, i have problems from time to time (generally after an update) getting my laptop graphics to play ball again (as usual i forget how i solved it last time), im using nvidia restriced drivers
 
The wireless card is a pain to get running also it has to use the restricted drivers for the broadcom 43xx family. ive known this card to crash the laptop while in use (probably my instalation error as its worked without issues on this install)
 
like i said with the above problems addressed, this machine works quite well, i have had the occasional freeze up but not since my last install, (which went smoother than usual)
 
sound has been an issue on previous installs (hd-intel/nvidia/realtek), i get the impression than Acer use a unique hardware configuration which trips up the linux install. (when this problem occurs the driver is working but no sound comes out of the speakers!)
 
As for the Acer HD partition setup, i really cant comment as i re installed Vista as soon as the laptop arrived (h8 all the bloatware that arrives on a pc thesedays) so any unusual partitions were removed, i noticed that there was a hiden partition but i assumed that was a typical OEM restore/backup partition.
 
i know i havent listed the solutions to the problems but it should point people in the right direction if they have any of the above issues
 
Dave
