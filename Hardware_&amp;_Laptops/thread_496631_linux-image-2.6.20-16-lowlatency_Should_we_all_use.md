### Post by tgoose on 2007-07-09
In simple terms, the low latency allows malware to lock up the computer more easily. That shouldn't be an issue, but just in case it is, it's better to run the default kernel for normal desktop use (low latency doesn't really bring any benefits for desktop use except for real-time work like audio or video editing, or controlling robots!)
