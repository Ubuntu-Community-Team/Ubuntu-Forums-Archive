### Post by Skidpad on 2007-06-12
Although I haven't seen one like this, your power supply's output could be reduced (bad power supply) and not providing its rated voltage/wattage.  The only faulty power supplies I've seen were completely dead - no output at all.

There should be a sticker on the power supply showing rated input & output; you can check the output with a multimeter, if you have one.

Is the boot time the only thing that is different when running on AC (versus DC battery) power?  File access, LCD brightness, anything else different once it boots up?

### Post by tgalati4 on 2007-06-12
With Ubuntu, you shouldn't need to boot up that often.  Put the machine on standby and you can go for several days or weeks between reboots.  So bootup time is less critical.  

With Windows, bootup time is important because you have to do it so often.
