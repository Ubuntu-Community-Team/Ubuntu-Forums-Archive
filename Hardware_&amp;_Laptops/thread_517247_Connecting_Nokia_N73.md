### Post by benyau on 2008-03-29
Do you have the miniSD card in the phone?
 
FYI, I got a new N73 yesterday.  It worked out of the box as a mass storage device, i.e. a usb disk.  Steps

1. Put the miniSD card into the phone.  My Nokia box came with a 128m miniSD.

2. Connect phone to computer using the given data cable.

3. N73 should detect the connection and ask to select a connection type.  choose "mass storage"

4. Ubuntu 7.10 automatically open a file manager pointing to the mass storage.  There should also be a disk icon appeared on the desktop.

Hope the above help.
