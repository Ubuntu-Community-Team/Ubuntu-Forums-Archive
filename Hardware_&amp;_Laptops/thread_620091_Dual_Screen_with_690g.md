### Post by mysticrider92 on 2007-11-26
According to [this site]("http://www.pcstats.com/articleview.cfm?articleid=2172&page=3") the ATi chip used in the 690G chipset supports dual monitors, but it can not use both DVI and HDMI at the same time. Since you are using LCD's, you could either run one analog and one digital, or find a cheap video card to drive the second.

Also, remember that while this is one of the best integrated chipsets, it is not powerful enough for any high-power 3D use (e.g. CAD, complex games, etc...).
