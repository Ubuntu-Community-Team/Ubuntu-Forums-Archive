### Post by ugm6hr on 2007-12-29
What are you hoping to do on that laptop?

I would recommend DSL: [http://damnsmalllinux.org/](http://damnsmalllinux.org/)

It is Fluxbox or JWM-based (very light), with a 2.4 kernel (also light).

Puppy will run on it, albeit not too quickly.

If you really want a Ubuntu-based OS: [http://www.psychocats.net/ubuntu/minimal#barebones](http://www.psychocats.net/ubuntu/minimal#barebones)
The ultra-light variation at the bottom of the page might be reasonable.  And perhaps Opera would run too (instead of Dillo).
You could make make easy addition of desktop control, file explorer etc with this: [http://lxp.sourceforge.net/](http://lxp.sourceforge.net/) (altho it does look like XP).
