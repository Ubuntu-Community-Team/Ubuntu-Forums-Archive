### Post by chewearn on 2008-03-20
I don't own any of these products, but I have borrowed iPod and PSP from buddies.  I was curious how well these devices worked in ubuntu.

The PSP will auto-mount in ubuntu (once you enabled the USB), then you can drag-drop media files via file manager.  These file will showed up in the PSP GUI and you can playback, no hassle.

The iPod, on the other hand, does not show the files that are drag-drop into it.  You need iTunes (or some equivalent hack) to get it to show up.  I find this retarted, but that's just me :)
