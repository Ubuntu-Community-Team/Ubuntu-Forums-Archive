### Post by archiesteel on 2006-01-24
My opinion? Go for the regular version. AMD64 works great, BUT you won't be able to view Quicktime (Sorenson) movies or see Flash anims, as there are no native 64-bit versions of these proprietary codecs/players, unless you set up a chroot. I used the 64-bit version for a while but eventually reinstalled the 386 version.
