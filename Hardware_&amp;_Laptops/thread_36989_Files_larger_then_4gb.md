### Post by nocturn on 2005-05-26
It should already support this.

The maximum size for a single file on Ext3 fs (the Ubuntu default) is 4 Terrabytes.
Reiser (what I'm using, is 210 PetaBytes).

This page is helpfull:
[http://www.novell.com/documentation/suse91/suselinux-adminguide/html/apas04.html](http://www.novell.com/documentation/suse91/suselinux-adminguide/html/apas04.html)
