---

### Post by Existentialist on 2008-01-25
According to Intel your board only supports up to 533.  That doesn't mean the 667 wont run in it necessarily, it may just clock it down to 533 for compatibility although this is no guarantee.  There is no benefit to you running 667 anyways as it would likely be clocked down, and it is best to run it at the same speed as the CPU's FSB.
