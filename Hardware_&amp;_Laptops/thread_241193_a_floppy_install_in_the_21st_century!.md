### Post by majesticturkey on 2006-08-22
There's a program called Rawrite which allows you to use a floppy to boot. I'm not 100% sure what your problem is, but it might be possible to read the floppy, and then read the CD-ROM from there.

If the computer has USB ports, then as the above poster mentioned, Dang Small Linux (unsure if I can say that word here :)) would be a great idea to put on a USB memory stick or flash drive, and boot from that, if the BIOS supports it. I think there are installations of Linux that can put the very bare essentials on from floppy, and you can use a network or the internet to work from there.
