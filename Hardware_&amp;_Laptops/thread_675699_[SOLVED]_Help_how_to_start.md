### Post by Yellow Pasque on 2008-01-23
You should install the 64-bit version if you have a 64-bit processor. The alternate install CD is a better option for newer graphics cards. Anyway...

The best way to install the Ubuntu fglrx driver is from the terminal: Press Ctrl+Alt+F1
Type:
> sudo apt-get install xorg-driver-fglrx
sudo aticonfig --initial -f --overlay-type=Xv
Then restart.

This guide might prove handy if you want to try the latest version of the ATI Catalyst drivers and use the 3D effects. Beware that they do not support AGP cards though.
[http://wiki.cchtml.com/index.php/Ubuntu_Gutsy_Installation_Guide](http://wiki.cchtml.com/index.php/Ubuntu_Gutsy_Installation_Guide)
