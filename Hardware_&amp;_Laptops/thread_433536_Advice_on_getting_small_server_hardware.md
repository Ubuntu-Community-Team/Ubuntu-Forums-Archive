### Post by [h2o] on 2007-05-05
I currently run my webserver (Ubuntu Server 6.06) on an "ordinary" computer (old desktop computer in a quite large case) at my parents house. Since their connection is quite bad compared to the one I got, and the fact that doing maintenance on it gets difficultdue to distance I am considering getting a new server and placing it at my place.
Problem is, I don't have any good spot for a large gray box that goes "wrmmmm-humm-humm" all day and night since I will probably get insane.

So, my questions is: Do you have any advice on what kind of hardware I could/should get that meets the following criteria:

1. Small - It should not be much larger than 30x30 cm. Although I am quite flexible on this particular criteria.

2. It should be quiet. Since it will be placed in my living room where I spend most of my time when I am not in school or asleep it needs to not drive me insane.

3. It must be able to run the following services:
  a) apache with php (or some other httpd)
  b) mysql
  c) ssh
  d) svn
Now, the traffic to and from the server is not very high. Average is about 1300 hits per day on the webserver.

4. Cheap - I am not made of money, so the cheaper the better. But this is the least important criteria.


I have looked at single board computers which look nice, but I am a bit unsure if the CPU power they possess are high enough for running all my services.

Any advice are much appreciated!

### Post by jespdj on 2007-05-05
If you're looking for silent components to build your own PC, have a look at [http://www.silentpcreview.com/](http://www.silentpcreview.com/)

On that website I learned about the [Antec P150](http://www.silentpcreview.com/article272-page1.html) case which I used to build my computer - it has some nice features to make it quiet, such as a system with rubber bands to suspend harddisks into etc.

It works great, my compiler is almost completely silent.

I have a Core 2 Duo E6600, 2 GB RAM, 320 GB Seagate SATA-II harddisk, DVD writer, Asus P5B Deluxe/WiFi motherboard and an Asus ATI X1600 passively cooled video card (I would have gotten an nVidia if I had known that ATI's Linux drivers are so crappy!).
