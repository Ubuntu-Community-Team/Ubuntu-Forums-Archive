### Post by GoodPanos on 2006-05-08
If you can find it on eBay, AmbiCom WL1100C-CF (Compact-Flash) is a good card.  It works right of the back with Ubuntu, no need to install or configure anything.

In brief it's a Compact-Flash (CF) card so you will need an adapter.  It's 802.11b and does not support WPA.

### Post by monomaniacpat on 2006-05-08
I use the Linksys WPC11 v.4 on open source drivers found [here](http://rtl8180-sa2400.sourceforge.net/). I even wrote a howto the other day, but it hasn't come up yet. I had a few problems along the way, but they were mostly incompetance related. It uses the RTL8180 chipset, so anything using that should be fine. Also, check out [this]("http://linux-wless.passys.nl/") site for a list of known linux-compatible cards.
