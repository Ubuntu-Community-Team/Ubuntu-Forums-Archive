### Post by djotaku on 2007-03-26
This is the first time I have a modern laptop and I intend to use it mostly when traveling.  Here's my question/scenario.

I'm at the airport using my laptop while I wait at the gate.  (Hopefully tethered to the mains)  Then they start boarding the plane.  I don't want to shutdown my CPU and then boot it back up when we read X thousand feet because that will waste time and thus battery power.  

So, which should I pick?  Hibernate or Standby?  

I read online and it seems that hibernate stores RAM to disk, but it will still have to go through the boot process.  However, it looks like standby is just a low power state, so it is still using power.  So which would you use?  Usually it's about a half hour to forty-five minutes between boarding the plane and getting to use CPU again.

Thanks,

### Post by swamytk on 2007-03-26
Hibernate is best suited, since it is safe to switch off the laptop while on move. It does not take much time to boot.

### Post by rara on 2007-03-26
I would recommend hibernate because in my experience, it has used less power. Standby just blacks out the screen and decreases power consumption by a small amount. Hibernate is nearly shut down, but probably uses just as much as a restart.
