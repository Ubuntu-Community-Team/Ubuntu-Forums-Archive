### Post by bbqsandwich on 2008-04-16
Hello,

I have an Acer Aspire 3680 laptop. It is 1.6 GHz, 1 GB RAM. I believe it has 64 MB video memory.

I recently hooked up a 19" widescreen monitor. I love it, but I think the 1440x900 resolution is putting a strain on the machine. It became slower until I disabled visual effects/compiz, and then it was OK but still slightly slower than when using the laptop's built-in screen.

Can anyone recommend any way to improve the performance, perhaps a system setting change? I would like to be able to use visual effects at this new higher resolution without sluggish performance.

I found a BIOS setting that appears to set the video memory to 128 MB using virtual memory, but it appeared to make no noticeable change & in fact I am not 100% sure what that setting really is (I like changing BIOS settings without understanding them, don't you?)

Any suggestions greatly appreciated! Thanks in advance!

### Post by Diabolis on 2008-04-18
virtual memory means that it will use hard drive space. Accessing a hard drive is a very slow process compared to ram memory. So even if you would get better graphics performance, accessing your hard drive would slow down your pc.

### Post by ckgreenman on 2008-04-18
> **bbqsandwich said:**
> Hello,

I have an Acer Aspire 3680 laptop. It is 1.6 GHz, 1 GB RAM. I believe it has 64 MB video memory.




First off,  so you know what video card is in the system?  Exactly Make/model?

Also One thing you can check is to open a term and run:
"glxinfo |grep -i direct"

That will tell you if Direct rendering in enable.  If it isn't, depending on your vid card, you might have to load additional drivers (maybe even proprietary) inroder to use the extended features of your card.
