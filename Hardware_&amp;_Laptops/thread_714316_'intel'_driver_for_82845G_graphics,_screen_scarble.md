### Post by DirtDawg on 2008-03-12
I believe the way to get the intel drivers is to install the intel drivers:
```
sudo apt-get install xserver-xorg-video-intel
```
then run the command:
```
sudo dpkg-reconfigure -phigh xserver-xorg
```
Answer the few questions it asks, then logout/login.
Unless I'm mistaken, there is no need to manually alter your xorg.conf file.

On a side note, I'm still using Feisty with the same graphics chip and I was really hoping the intel drivers were coming preinstalled and working correctly with Gutsy. I hope they do in the Hardy release. The i810 drivers are outdated and when I installed the intel drivers, they occasionally crashed on me.
