### Post by gray-squirrel on 2007-03-17
I'm in the process of installing a new power supply, an Ultra V DR Series 600W to replace an older 400W which has made noises yesterday morning which suggest that it should be replaced.

The problem I have is this: I have an Abit VA-20, which has an electrolyte capacitor almost standing right up against a socket where the power supply's 20-pin connector is supposed to be connected.  In turn, the 20-pin connector has a tab which pushes the capacitor over if an attempt to get the connector completely in the socket is made.  I managed to get it in completely, but the capacitor is leaning away from the socket.  At least there was no snap noise and it is still in place otherwise.  I'm very relucatant to turn the machine on because of the possibility of toasting my devices (not because of the power supply itself, but because of the motherboard not handling the electricity properly because of that one capacitor).

I am prepared to get another system board (with accompanying CPU), but I'd like to know before I shell out any money on that if things should still be okey and that there shouldn't be any problems with powering up the system.  I didn't think electrolyte capacitors could "move" like that, and a few months ago I accidentally dropped a screw onto the system board and thought that was the end of it, but the computer has worked flawlessly since then.

I thought of taking the tab off the 20-pin connector, but I couldn't figure out how to do it, but then again, if I did and something happened to the power supply I would not be able to return it in exchange for another.

### Post by berylmichael on 2007-03-17
As long as the capacitor isn't touching anything else, it should be fine. They can bend some and it won't hurt anything. The only issue is that where the capacitor is connected to the motherboard, the connection is broken (the solder joint isn't making good contact). Only a technician would be able to tell you this.
