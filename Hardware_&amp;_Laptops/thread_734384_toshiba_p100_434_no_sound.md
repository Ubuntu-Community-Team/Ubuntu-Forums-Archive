### Post by Fjatle on 2008-03-24
First of, there is no 7.1... There is an Ubuntu 7.10 though :)

Have seen in System-Preferences-sound and seen what the options is there?
It should recognize your sound card automatically  if it is set to do so...
Try to set the options there to "Alsa", and test if you get any sound.

If this dont work, you can try go to Programs-Accessories-Terminal and type alsamixer
Here you shuold have "master" and"PCM" turned up... if they dont, you use arrow keys to 
change your settings..
