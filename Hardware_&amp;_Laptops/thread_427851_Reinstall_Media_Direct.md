### Post by Tintinnabulation on 2007-04-30
Dell media direct is installed on a hidden partition.  Maybe you formatted this when installing ubuntu.

I don't think it's possible to preserve media direct when installing ubuntu because it changes the MBR unless you install ubuntu a special way.  See:
[http://ubuntuforums.org/showthread.php?t=321812&highlight=dell+media+direct](http://ubuntuforums.org/showthread.php?t=321812&highlight=dell+media+direct)
