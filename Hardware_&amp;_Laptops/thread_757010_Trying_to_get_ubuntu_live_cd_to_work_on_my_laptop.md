### Post by carrett on 2008-04-16
Try pushing F4 (or maybe it's alt-F4?) while it's booting up. This should show you some messages that might help figure out what's up.

Or, before you boot you should be able to edit the boot options, one of the defaults is quiet, another is splash. See if you can't remove those. I can't remember how to do it off the top of my head and don't have a CD with me, but I know it can be done. Will post more later if you can't figure it out before I get home.

If and when you do get some messages, post them here and we can try to figure it out. Good luck!
