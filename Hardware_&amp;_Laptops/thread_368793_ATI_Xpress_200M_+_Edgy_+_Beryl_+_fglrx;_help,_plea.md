### Post by twitchku on 2007-02-24
Excellent!! 
Thanks for the references guys, i've been going nuts trying to find a fix for that.
I ended up just setting beryl to run with:

LD_PRELOAD=/usr/lib/fglrx/libGL.so.1.2.xlibmesa beryl

Works great!

Thanks again.
