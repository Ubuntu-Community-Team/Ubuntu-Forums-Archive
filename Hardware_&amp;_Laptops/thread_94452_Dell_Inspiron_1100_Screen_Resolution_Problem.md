### Post by bodhi.zazen on 2006-11-24
FYI: I see this looks like an old thread, but I just installed on a Dell Inspiron 1100.

I had the same problem.

The solution was two fold:

1. Download and install [I1100A32.exe](http://ftp.us.dell.com/bios/I1100A32.exe) (AKA Update bios to A32 as suggested by [color=blue]mootsfox[/color]). I ran this on windows as wine is not on the live (install) CD. For further information see here: [Dell Drivers & Downloads](http://support.dell.com/support/downloads/download.aspx?c=us&l=en&s=gen&releaseid=R87514&SystemID=INS_PNT_CEL_1100&os=LNUX&osl=en&deviceid=5003&devlib=0&typecnt=1&vercnt=8&formatcnt=2&libid=1&fileid=114439)

2. I had to manually edit /etc/X11/xorg.conf

Look for the "monitor section"
Add these lines>         Identifier      "Generic Monitor"
        HorizSync 31.5 - 50.0
        VertRefresh 40-90
        Option "UseEdidFreqs" "1"
#       Option          "DPMS"

Then look for the "Screen section"
Default depth = 24
Modify the defaults to look like this:>         SubSection "Display"
                Depth           24
                Modes           **"1280x1024"** "1024x768" "800x600" "640x480"
        EndSubSection

My final monitor section looked like this:> Section "Monitor"
        Identifier      "Generic Monitor"
        HorizSync 31.5 - 50.0
        VertRefresh 40-90
        Option "UseEdidFreqs" "1"
#       Option          "DPMS"
EndSection

Then all I needed to do was re-start X (Ctrl-Alt-Backspace worked for me)

I attached the xorg.conf in full.

This is for Dapper, Edgy did not work so well on this laptop, the HD was in some type of constant activity....

HTH 8)
