### Post by bigbang on 2007-09-02
I've found a handy little program for inputting text through handwriting. It is called Cellwriter and it runs a dream in Ubuntu, its website even provides a  '.deb'.
I thought I might write this because I havent seen any solutions in the forum that are as accurate as this. The only downsides to this program is you have to write each letter in a box, the program needs to be trained to your handwriting and it is always open. However I'm sure these problems will be rectified soon, but for now the program is perfectly usable.
It has my personal recommendation to tablet pc users (even though I dont use one).
