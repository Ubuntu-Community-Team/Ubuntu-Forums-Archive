### Post by Existentialist on 2008-02-23
Any of the 8600 or up Nvidia cards have two monitor outputs that you can use.  You can disable the onboard video in BIOS most likely and run both monitors from that one card.  As far as I know, you can run one monitor off of each of the display adapters if you wanted, but I don't have any experience running a setup that way.  Setting both monitors up through the nvidia-settings gui once you have the drivers installed is pretty easy.

### Post by Existentialist on 2008-02-24
I don't really want to tell you that you need to spend more money, all I can speak for is the setups i have seen it work with.  It seems to me that the Nvidia support is much better, so running both monitors off of the one card you buy is probably easiest.  I have ran dual 1600x1200 monitors with 3d effects off of: 1 6800ultra, 1 8600gt, and 1  8800gt.  From what I have seen with gaming benchmarks, the 7300 series was not as powerful as the 6800's so I can't promise you anything with that.  Anything from the 7600gt up should be more powerful than the 6800ultra, so theoretically since I did it with the 6800ultra you could do it with any of these cards.
