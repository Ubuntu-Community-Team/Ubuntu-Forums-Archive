I am trying to format a Compact Flash card. Every time I try and write the partition table to disk the card gets automounted and the write operation can not be completed.  It is a usb compact flash adapter the device is /dev/sda.

### Post by kaptk2 on 2005-03-15
So after searching an spelling things correctly I found that if you go to Computer ---> Desktop Preferences ---> Removable Storage you can setup what gets mounted when. I just turned off all the automounting and was able to go on my merry way. Sorry for cluttering up the forums
