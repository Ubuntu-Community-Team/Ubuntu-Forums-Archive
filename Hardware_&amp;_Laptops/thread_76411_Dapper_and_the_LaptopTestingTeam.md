### Post by Burgundavia on 2005-10-15
As with Breezy, the LaptopTestingTeam is looking for people to help. This is not for those who cannot reinstall at every point release/RC/preview release and cannot find the time to file bugs.

So should you join the Laptop Testing Team? If you answer yes to the following questions, then do so.

1. You can reinstall at every point release
2. You can file bugs about each and every piece of non-working hardware in a default install.
4. You can follow up on the bugs reports you have filed.
3. You have the time to do the above. About 4 hours to start and about 1/2 hour/week.

If you answered NO to any of the above questions, then simply file your hardware here:
[https://wiki.ubuntu.com/HardwareSupportMachinesLaptops](https://wiki.ubuntu.com/HardwareSupportMachinesLaptops)

If you answered YES, then go here:
[https://wiki.ubuntu.com/LaptopTestingTeam](https://wiki.ubuntu.com/LaptopTestingTeam)

If you already have joined the team, please keep your template updated on the TestingTeam.
