### Post by allwin on 2007-03-15
You could always try out the LIVE CD.  You might be pleasantly surprised.  I have an old 500MHZ Pentium desktop here which runs well, even with Beryl (turned down to 16 bit color however), using Ubuntu and Gnome.  Memory might be a little problem for you, though.

There's another failrly spartan distro out there, SLAX, (slackware distro) which also is about the fastest and leanest one I've encountered, so much so it is feasible to run from the LiveCD alone fairly zippy.

Worth experimenting.  If it runs at all from the LIVECD, if you decide to install it, it will be generally faster still.
