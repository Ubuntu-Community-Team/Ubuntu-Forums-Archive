### Post by FreewheelinFrank on 2007-12-06
A new mouse seems to have fixed this problem for me- obviously popped a few too many zombies in HL2.

Other reports I linked too seem to be genuine issues with a click being falsely interpreted as a double click.

One method to check is to run this command in the terminal:

```
xev
```

It generates a list of output from the mouse. If a solitary click generates two entries for the same button, suspect a failing mouse.
