
then:

```

sudo ntfs-config

```

Tick the appropriate boxes and restart.

### Post by CValentine on 2007-08-23
The ntfs-config script will show only drives, that are not mounted when you run it, since otherwise they are in use by the normal non-writeable driver. So the best way is to install ntfs-config etc. and reboot before running it. Maybe you can also simply unmount them and run the script afterwards, haven't checked that...
