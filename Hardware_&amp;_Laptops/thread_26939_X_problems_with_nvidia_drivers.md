### Post by heimo on 2005-04-14
My guess it that autodetection is not working and you need HorizSync and VertRefresh lines - possibly Modeline (but not neccessarily).

This thread explains how to do it:
 [http://www.ubuntuforums.org/showthread.php?t=21984]("http://www.ubuntuforums.org/showthread.php?t=21984")

EDIT: Oh, and don't hesitate to ask if you can't get it working.

This is the part from log that tells current values (which seem too low / narrow)
 ```

(II) NVIDIA(0): Generic Monitor: Using default hsync range of 29.00-49.00 kHz
(II) NVIDIA(0): Generic Monitor: Using default vrefresh range of 0.00-60.00 Hz
```
