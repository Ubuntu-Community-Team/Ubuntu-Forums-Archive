### Post by aysiu on 2006-05-11
Use a live CD to test Ubuntu's hardware detection. If only one or two things are off (say, screen resolution), then go ahead and install Ubuntu. If three or more things are off (sound, internet, USB), forget it.

A live CD runs completely off the CD itself and your computer's RAM. It does not affect your hard drive.

If you're planning to dual-boot, I'd advise you to visit this website first:
[http://users.bigpond.net.au/hermanzone](http://users.bigpond.net.au/hermanzone)
