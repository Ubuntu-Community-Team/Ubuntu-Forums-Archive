Hi,
1st there's an error on this line that says: modprobe speedstep-centrino
obvious theres an extra, incorrect, 'modprobe'.

Usually theres no need to list the cpufreq_* modes. They will load automatically.
In my case i wont even have to mention the cpu frequency controller module. It's all done automatically. 

All I have done was install powernowd (a boot deamon/service) and ensure that it starts on boot time. 
An useful package too is cpufrequtils, not installed by default.
