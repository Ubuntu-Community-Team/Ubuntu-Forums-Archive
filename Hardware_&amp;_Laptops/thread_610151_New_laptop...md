### Post by maharbA on 2007-11-11
I'm no expert, but from what I've seen/personal experience, Dell's Inspiron Laptops work very well.

I'm writing this on an Inspiron 8600. I've had _no_ major problems. This particular machine has an ATI graphics card, so that was a pain, but you can get them with (I believe) either Intel or nVidia, so you can avoid that.

Seriously, everything is supported (new in Gutsy, the modem I've never used is supported) and the battery lasts forever. Even the blue Function keys worked out of the box -- so did the built-in volume keys (the play/pause. stop, back, fwd buttons only work in some apps, but all the default ones).

Ubuntu runs as easily on this laptop as on any desktop.
