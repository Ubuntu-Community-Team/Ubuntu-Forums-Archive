### Post by RandomJoe on 2007-01-18
Another recent Core 2 Duo purchaser.  And as the others said, it is _fantastic_.  Beats the pants off _anything_ else I've seen (mostly P4 systems, to be fair to AMD!) and so much cooler - mine runs 97-99F as well, even under heavy load it *might* get to 105, and that's with the stock Intel fan.

I looked at all four processors, and chose the E6600 too.  I wanted the 4MB cache, and the money they want to bump up to the 6400 (for 2MB) or 6700 (for 4MB) just isn't worth it - we're talking a whopping 260 MHz difference.  (As he thinks back to when that would have been a _doubling_ of the processor speed...! :rolleyes: )

Probably the biggest issue right now for Linux on these is to be sure what chipset your motherboard uses.  You can search these forums for more on the issue, but many boards using Intel chipsets are using a Jmicron controller for (IIRC) PATA ports, and the Jmicron support appears to be pretty flaky right now.  I bought a board using the nVidia nForce 4 chipset, and it's been fantastic.
