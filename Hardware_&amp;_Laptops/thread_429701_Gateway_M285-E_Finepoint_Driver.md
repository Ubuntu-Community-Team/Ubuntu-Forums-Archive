### Post by bb002 on 2007-07-05
Well. If you guys can compile I have this guy that I've been working on. I need more ppl testing it though to see how it works for other Tablets. This driver is for Fiesty and xorg 7.1. So your mileage may very, period.

[edit2]
I've attached some binaries in my next post. so check those if you don't want to compile but still follow these instructions until you get to the compile stage and install the binaries instead.
[/edit2]

First off the finepoint digitizers connect through a serial port and we need to set that up first. you need the setserial program installed :```
sudo apt-get install setserial
``` then we add this to the /etc/serial.conf file ```
/dev/ttyS0 uart 16550A port 0x06a8 irq 4 baud_base 38400 spd_normal skip_test
```

Now we need to add these lines to the /etc/X11/xorg.conf file ```
Section "InputDevice"
        Identifier "Tablet"
        Driver "fpit"
        Option "Device" "/dev/ttyS0"
        Option "AlwaysCore" "on"
        Option "InvertY"
        Option "MaximumXPosition" "12550"
        Option "MaximumYPosition" "7650"
        Option "MinimumXPosition" "400"
        Option "MinimumYPosition" "400"
        Option "Passive" "false"
        Option "SendCoreEvents"
EndSection

``` and this line to the serverlayout: ```
        InputDevice     "Tablet"
```

download this file [http://xorg.freedesktop.org/releases/X11R7.1/src/driver/xf86-input-fpit-X11R7.1-1.1.0.tar.bz2](http://xorg.freedesktop.org/releases/X11R7.1/src/driver/xf86-input-fpit-X11R7.1-1.1.0.tar.bz2) and extract it. Seeing as this is simpler than making a patch file and explaining how to use it (no clue how to make one anyhow). overwrite the file ./src/xf86Fpit.c with the attached xf86Fpit.c.tar.bz2.

there are a few packages you need installed to compile the driver ```
sudo apt-get install build-essential xorg-dev
```that *should* get all the dependencies.
now open a terminal and browse to where you extracted the driver and do ```
./configure --prefix=/usr
make
sudo make install
```


if all goes well you are now ready to reboot and enjoy your now working pen!
