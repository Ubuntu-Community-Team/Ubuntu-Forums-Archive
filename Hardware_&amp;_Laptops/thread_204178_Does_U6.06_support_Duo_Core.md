### Post by Hellkeepa on 2006-06-27
HELLo!

Once you've installed Ubuntu (which you should be able to do without any assistance, or at least I hope so. :P ), then all you need to do is to open "Synaptic":
System -> Administration -> Synaptic Package Manager

Then you just search for what you want to install, either via the search-feature or manually. Click on it, select "Mark for installation" for all the apps you want to install, then press "Apply".

Happy codin'!
