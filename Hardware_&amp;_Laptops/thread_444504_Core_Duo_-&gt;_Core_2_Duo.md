### Post by blackened on 2007-05-15
The Core Duo is Socket M, the Core 2 Duo is LGA 775. Two different interfaces, so no, they aren't interchangeable.

### Post by Syke on 2007-05-15
It's dependent on the laptop's motherboard. Core Duos can be upgraded to Core 2 Duos if the motherboard supports it. The Core 2 Duo is available in Socket M form, just like Core Duo.

Wiki confirms it: [http://en.wikipedia.org/wiki/Socket_M](http://en.wikipedia.org/wiki/Socket_M)
