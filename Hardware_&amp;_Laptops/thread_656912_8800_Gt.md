### Post by jespdj on 2008-01-03
The 8800 GT is so new that it isn't supported by the nVidia video driver that comes with Ubuntu 7.10. You need to install the newest nVidia driver.

You can do it yourself: get it [here](http://www.nvidia.com/object/linux_display_ia32_169.07.html) from nVidia, but it's probably easier to use [envy](http://albertomilone.com/nvidia_scripts1.html), which is a script to automatically install the latest video drivers.
