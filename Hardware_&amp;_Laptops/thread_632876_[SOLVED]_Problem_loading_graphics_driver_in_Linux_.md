### Post by Pandemic187 on 2007-12-06
I should also mention that I (almost) had Compiz Fusion working on debian, which required me to edit my xorg.conf file. The following is seen in my current xorg.conf, for Mint:
```
Section "Device"
	Identifier	"Generic Video Card"
	Driver		"vesa"
	BusID		"PCI:0:2:0"
EndSection
```
As I said, I almost had Compiz working fully in debian, except that for some reason it didn't want to load window decorations. The visual effects were working properly though, and my xorg.conf file actually showed the model of my graphics chip (Intel 945GM), and the driver under Section "Device" was indeed the i810 driver.
