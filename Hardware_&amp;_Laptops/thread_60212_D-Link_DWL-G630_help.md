### Post by firecat53 on 2005-08-28
I've got the DWL-G630(rev C!) working using the most recent versions of the Madwifi drivers and the most recent development version of wpa_supplicant. Make sure you check which hardware revision of your card that you have before you try anything because rev A and B use a different chipset, I believe. If you have Rev. C, check out the Madwifi page and the forums for several how-to's. Let me know if you need more help! It does work!!
