### Post by Martje_001 on 2007-05-13
What filesystem is it?
For FAT32 try:
```
sudo mount -t vfat /dev/sda1 /media/sda1
```
For NTFS
```
sudo mount -t ntfs /dev/sda1 /media/sda1
```
