### Post by az on 2005-07-10
If you run the installer, the information is on one of the help pages (F3, I think)

The minimum for installing is 32 megs, but you need at least 128 to run Gnome normally.  Otherwise it is very sluggish...

2.1 Gigs of HD space is the minimum.
