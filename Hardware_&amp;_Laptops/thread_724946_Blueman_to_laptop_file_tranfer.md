### Post by sleepingdragon on 2008-03-15
I had a similar problem, but I never found a decent solution for blueman. The only thing that seemed to work for me was to use gnome-obex-server to send the files. 

I use Thunar for file browsing and it allows you to configure custom actions for files, so for my files I use the command *gnome-obex-send* and it appears in my context menu as "Send via Bluetooth". It even supports multiple file selection - and you even get a pretty BT icon in the taskbar showing transfer status!
