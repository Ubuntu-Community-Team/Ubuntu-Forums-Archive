I have a Compaq Presario ---SR1103WM, to be exact. I had much trouble about 2 months ago. At that time, was only able to use the integrated graphics, but wanting to use my NVIDIA.

For details and the solution, see here:

[http://ubuntuforums.org/showthread.php?t=380790]("http://ubuntuforums.org/showthread.php?t=380790")

In order for me to get x to start when having onboard as primary, I had to remove the 

```
agpgart
intel_agp
```

entries created in /etc/hotplug/blacklist.

And the 

```
blacklist agpgart
blacklist intel_agp
```

entries in /etc/modprobe.d/blacklist.

Not including the above when setting the PCI (NVIDIA) as primary, would cause x to fail.


Then it was just a matter of getting the "Device", "Screen" and "Monitor" entries correct. 

I made more changes using (in KDE) Kmenu > Settings > Peripherals > Monitor & Display (this option was not there by default, but trying to guess and think it was in 'kde-guidance' package).
