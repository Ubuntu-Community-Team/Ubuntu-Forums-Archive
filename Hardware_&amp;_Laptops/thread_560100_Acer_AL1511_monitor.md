old message:
If you have this monitor and you get the out of range errors on the screen you may have to install with the Xubuntu server disk or the Edubuntu server disk.

Or if you try to add this monitor to an existing install you may have to install the xubuntu desktop before you use the monitor.

Installing with the Xubuntu server disk is the only way that I could get this monitor to work. 
It works quite well with Xubuntu or Edubuntu. I am very happy with it.

This monitor only operates at 800 X 600 H38 V60.
