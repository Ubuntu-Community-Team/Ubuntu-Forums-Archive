### Post by mblinux on 2006-10-27
Hello to everyone.
I have this old desktop pc: PII 266MHz on an ASUS P2L97 Rev. 1xx (chipset I think is i440LX), 3 ram modules(1x128MB, 2x64MB total: 256MB ram). Video card Matrox G200 Mystique. A noname ethernet card with realtek chip and a sound card. 8GB hard disk. 

This system was running for years win98SE. Now it refuses to install Ubuntu or Xubuntu. The installation using the alternative cd 6.06.1 Dapper and the new Edgy fails usually at 34%, no error messages, I just see the hdd led that is ON for ages (hours). I have also tried Gentoo, Dreamlinux. Only the Damn Small Linux distribution made it to run and install on hdd. 

Today I have decided to give it a try with WinXP Pro SP2 and everything worked fine. The installation took some time but now winXP runs just fine with Kaspersky suite and firefox 2.0. 

I was really surprised since I was always thinking that the best way to use old hardware is linux. 

All this without tweaking the linux installation or giving special installation orders and using the guided partitioning ( manual partitioning with / and swap partitions also failed). 

I'd really appreciate any suggestions. 
Thank you.
