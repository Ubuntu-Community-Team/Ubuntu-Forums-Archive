### Post by airtonix on 2006-10-25
Open up the "removable disks and media" manager from the ubuntu system menu -> preferences -> removable disks and media, then select the  media tab, then untick the  "play music files when connected" option.....

Next move to the disk manager from the ubuntu system menu -> administration -> disks, select the Ipod  in the left coloumn, then click on the 2nd partition and then click disable.....

Then click the format button....
