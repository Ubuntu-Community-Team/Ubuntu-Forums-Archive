### Post by taurus on 2007-02-21
Put your PCI graphic card in.  Then boot into recovery mode from from GRUB menu and at the prompt, type

```
dpkg-reconfigure xserver-xorg
```
If you are not sure about a question, just use the default and pick **vesa** as a driver for your graphic card.  Once X is running again, then you can install an ATI driver for it.

Reboot your machine with

```
shutdown -r now
```

[http://albertomilone.com/driver.html](http://albertomilone.com/driver.html)
