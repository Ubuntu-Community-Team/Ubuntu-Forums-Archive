### Post by marx2k on 2007-01-11
If youre going to start with a NEW computer, I would just buy parts and put it together. You might pay a little more (MAYBE) but usually you can get it for around the same prices as the major computer companies offer for complete systems.  If anything, think of it as you spending the extra 60-100$ on computer parts and not Windows :) 

newegg.com always has amazing deals on OEM parts for the computer.

You already have parts for the new computer (hard drive, some memory, video card (however good that video card is)), probably ethernet card, power supply - cannibalize it down to the CMOS battery and spare jumpers if youre not going to use it.

If not, get 80-pin IDE cables as that has helped me achieve higher UDMA speeds.

### Post by RandomJoe on 2007-01-12
No, no!  You need to get an _exact part number_ from someone - one of the manufacturer websites, or Compaq (who is HP now) - and get ONLY that part number.  There have been a wide variety of memory styles over the years, and you have to get exactly the style that motherboard is designed for.

If you go to one of the websites I listed, they have a "memory selector" right on the front page.  You select your manufacturer (Compaq), then the model line of computer (Deskpro), then the exact model (hopefully you can find something somewhere on the machine to tell you this).  The website will help you drill down to the specific model you have as long as you can find the info.  Then it will tell you what model numbers and sizes of memory you can put in there.  

For some machines (like my GX200s) they may just say nothing is available due to low demand, so you may need to try several vendors.

### Post by RandomJoe on 2007-01-14
This is why I (again) suggest going to those memory manufacturers' websites I listed before and try there.  If you can find sufficient identifying info on the computer to dig down to a specific model on their website, and they have memory that will work in it, they will tell you how much memory the machine can handle, any special configurations if required, and _exactly the part numbers_ you need to order for the size modules you want.

No, a memory module won't necessarily work just because it's identical physically to the existing ones.  Some manufacturers, and Compaq in particular - at least at one point in their past - used some reallly oddball memory cards.  I had a machine like this at one time, and tried a load of different "standard" modules without success until I got the correct ones for the machine.

There are a large number of parameters to memory modules - different voltages, timings, speeds, parity, chip configurations...  Varying even among modules that are otherwise physically identical.
