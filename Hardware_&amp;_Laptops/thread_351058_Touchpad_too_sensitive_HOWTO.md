### Post by jmpmjmpm on 2007-02-01
Hi all,

I had a problem with my touchpad misbehaving. The main problem was that the "tap" was way too sensitive and if you placed your finger on the touchpad it would interpret it as a "click" making for a nightmare web browsing experience. Although I had found several howto's on how to turn off the tap feature, I wanted it to work, and here is the solution.

It seems that Ubuntu uses a generic driver for the touchpad and the problem begins here, you need to make sure you have the correct driver installed so

```
sudo apt-get install xserver-xorg-input-synaptics
```

then you need to add the following to your xorg.conf file so

```
sudo nano /etc/X11/xorg.conf
```

then locate the section that looks like this

```
Section "InputDevice"
	Identifier "Synaptics Touchpad"
	Driver "synaptics"
	Option "SendCoreEvents" "true"
	Option "Device" "/dev/psaux"
	Option "Protocol" "auto-dev"
	Option "MaxTapMove" "10"
	Option "SHMConfig" "on"
EndSection
```

if it is not there then add it

then find the section that looks like this

```
Section "ServerLayout"
	Identifier	"Default Layout"
	Screen		"Default Screen"
	InputDevice	"Generic Keyboard"
	InputDevice	"Synaptics Touchpad"
	InputDevice	"Configured Mouse"
	InputDevice     "stylus" "SendCoreEvents"
	InputDevice     "cursor" "SendCoreEvents"
	InputDevice     "eraser" "SendCoreEvents"
EndSection
```

you see the "synaptics touchpad" entry? if it's not there add it.

Now you need to adjust the value that varies the sensitivity of the "tap", this is the option that looks like this

```
Option "MaxTapMove" "10"
```

the value 10 is what worked for me, but your mileage may vary, try in increments of 1 up from 0, with 0 not detecting the tap at all. You may want to experiment with the options "FingerHigh" and "FinerLow" also if your still having problems, although I did not need to do this.


you may also wish to install gsynaptics as a gui frontend to control some of the touchpad's properties. After install you can find it in system>preferences then click touchpad.

```
sudo apt-get install gsynaptics
```
