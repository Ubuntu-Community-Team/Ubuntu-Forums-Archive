I just bought a Logitech Precision gamepad for my laptop. It's similar to an old non-analog PSX controller. It's all digital, but it was only ten bucks and it works out of the box on Ubuntu. It and Qjoypad are all I need for emulators 'n such on Linux.

Yup, it's the USB one. It works fine for me. Cheap, too. If you need to, install the package "joystick" via apt/synaptic and use the calibration tools.
