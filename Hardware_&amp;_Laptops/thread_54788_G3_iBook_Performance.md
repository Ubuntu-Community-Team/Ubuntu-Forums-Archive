### Post by lol on 2005-08-07
I am running a Debian testing on a G3 400Mhz (powerbook, not ibook though). It is actually running much faster than Mac OS X, but don't expect too much: the HD is slow and the limited amount of memory makes it hard to run to many applications at the same time.

I strongly suggest running a lighweight environment, like xfce, rather than gnome on such a machine...
