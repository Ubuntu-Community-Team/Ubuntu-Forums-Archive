### Post by TeleTommy on 2006-11-21
The SIP Protocol (Session Initiation Protocol) exists to "bring two people together". It doesn't tell you to use a specific codec, that's why you can use it for video, for VoIP or several other things.

Systems like Yahhoo, ICQ are "closed source" that's why it's extremly difficult to use their video feature on linux - I don't think that's possible. Perhaps the next skype version does support video with linux. 

Maybe there is other software that I don't know. Perhaps some Java Software that you can use with windows and linux?
