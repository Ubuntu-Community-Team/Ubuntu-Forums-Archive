### Post by shad0w_walker on 2007-08-13
If its an acctual dial i think you might be out of luck, most of those real turning dials are physical volume controls that don't have any thing to do with the OS. If it works like a keyboard volume button i believe you can edit what channels you control using

System > Preferences > Sound

Should be able to deselect all the mixer channels in there and it wont change them if you knock it. If that doesn't work try right clicking on the volume control icon, select preferences and deselect them there.
