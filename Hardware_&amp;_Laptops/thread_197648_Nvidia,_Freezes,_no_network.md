### Post by davidjlloyd on 2006-06-16
Can I just say that I have beed playing with Linux on and off for about 10 years, to start with as a teenager just for the hell of it, trying to get X working, playing about with graphics stuff. Since then I have tried out the latest developmments every year or so to see how Desktop development is coming along (and if I can finally give up MS!)...and I have to say that finally Ubuntu does it for me! It (more or less) worked perfectly from the start! It would be great if other distros would catch on to the fact that only one CD with key apps is required. Excellent stuff Ubuntu!

Anyway, I have been playing with ubuntu for a few weeks, seeing in the new 6.06. I have had problems with random freezes (mouse still moves, no keyboard response) with only option to do a hard reset (no keyboard access to terminal). Also the network kept cutting out with no apparent way to restart it without a reboot. This of course is useless if your using Ubuntu as your desktop and is likely to eturn anyone back to MS eventually. 

I got the impression that this might be graphics related, I have an integrated Nvidia graphics chip on the NForce2 GT (XC Cube) and had to install this as per instructions on website. The graphics all seemed to work when not freezing so a few searches on the forums here highlighted the possibility that there may be BIOS issues. I turned off anything unneccessary in there, paying particular attention to the graphics settings and Ubuntu has been working great since. Not sure which setting it was particularly but it may have been todo with the onboard AGP socket (I'm not using it).

So if you have Nvidia hardware and random freezes, turn off all the unneccessary stuff in the BIOS! It appears to have fixed random network cut-outs too!

Now if anyone wants to get Sketchup and AutoCAD running on Linux (without a virtual PC and windows!!!) that would be great! Wine developers, keep up the good work, your getting there!
