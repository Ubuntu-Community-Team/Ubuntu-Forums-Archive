### Post by jackmc on 2007-06-08
I'd say you should be fairly well supported, it doesn't look like there is anything too extreme there! I'm new myself, I reccomend you get the live CD in there and see how it goes.

For me, it automatically connected to my network (wired, then wireless after entering details)

Networked printer took 5 mins max to set up - driver was built in, works perfectly

I can bluetooth stuff to and from my phone

CD/DVD burning is not difficult (I like K3b for this)

openoffice works well 99% of the time

cool photo tools are built in to organise your collection

Amarok is what I use for music - it supports my Ipod, and I like it.

Not sure on the specifics, but there is palm support built in.

These forums should be able to answer most of the questions you can come up with, none of my issues have gone unsolved :)

### Post by Dedoimedo on 2007-06-08
Hello,

You can boot into live CD session to get a feeling of the distro.

It will also give you an indication regarding your hardware (detection). If you are worried about programs - don't worry - you'll have everything you need and saw in Windows twice over.

You can also install virtualization software in Windows (like VMware Server) and try Linux as a virtual machine and see how it works. You'll be able to practice with all the little settings, command line, compilation etc without worrying of corruputing your real system. Then, when you're ready, you can install for real.
