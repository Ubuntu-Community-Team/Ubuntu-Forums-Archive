### Post by coskierken on 2008-02-01
Here is my fixes I found.  First, with ATI 8.01, xserver-glx is no longer needed, so purge it from your system.  I you want to set it up with no problems (at least I had none on my computer / Asus A8Jp AT X1700I) download Envy and use it to install the driver.  It worked great!  if you have not installed compiz config manager, do it.  You should be able to activate Custom Effects (right click / change desktop  background / visual effects / Custom.  

You should be able to do it now.  I downloaded Emerald and it is pretty good, also.  I recommend both packages together Compiz-Fusion / Emerald.  Another if you want it is arts manager for downloading Icon themes, backgrounds, etc.  Useful, but not needed.

Also, since the video is still BAD with Compiz active (still working on it), I would download Compiz Switch:

[http://forlong.blogage.de/article/pages/Compiz-Switch](http://forlong.blogage.de/article/pages/Compiz-Switch)

I am 64bit and had to make the install and not use the Deb installer (Failed).  It works pretty good and you can put an shortcut on the bar or desktop to turn off and on.  Not bad and useful, but could use some tweaking.
