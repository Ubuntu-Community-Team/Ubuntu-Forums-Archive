### Post by nicholaspaul on 2005-03-14
This time I found the answer myself. And it wasn't even Ubuntu related... go figure.
The problem was that I had to 'connect' the mouse using the button on the underside of the mouse. After pressing the button on the USB receiver, pressing the connect button connects the mouse. 

Well, I hope this helps anyone with the same problem.
