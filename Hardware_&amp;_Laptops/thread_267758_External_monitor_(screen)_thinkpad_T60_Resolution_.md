### Post by blx_286 on 2006-09-29
Here is what I did on an old Compaq to keep it from sleeping when I closed the lid. I can't tell ya which one worked, because I did them all.

Went into System> Preferences> Power and changed the power options there. Then in Terminal, I typed

```
xset -dpms
xset dpms force off
xset s off
setterm -blank 0 -powersave off -powerdown 0
killall gnome-power-manager
```

I also did

```
sudo gedit /etc/X11/xorg.conf
```

and commented the DPMS section out

```
Section "Monitor"
	Identifier	"Generic Monitor"
#	Option		"DPMS"
EndSection
```

Maybe that will help. Sorry, I didn't have any probs with the external flat screen I was using, so I can't help with that.
