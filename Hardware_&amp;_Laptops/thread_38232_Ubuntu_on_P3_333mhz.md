### Post by dbott67 on 2005-06-03
I've got Ubuntu 5.04 running on a P2-300 Toshiba Laptop with 256 MB RAM (8 GB Hard Drive).  The performance of Gnome is acceptable (I've had XP Pro on it, as well as KDE, XFCE, IceWM and Fluxbox).  Here's what I've noticed:

Although Fluxbox & IceWM run much faster than Gnome or KDE, once you load an application (like Firefox) the difference is barely noticable.  So, I would recommend running Gnome and just adding as much RAM as you can.

I've also installed Ubuntu on my work computer (P4-3.0 GHz, 512 MB RAM, dual boot with XP) and home computer (P3-600, 512 MB RAM, dual boot with XP Pro).  My P3-600 runs just fine with XP and Gnome --- not much of a performance difference in all honesty.

I've still got to keep MS around to run a few work-related apps (pcAnywhere, Frontpage and a couple of other custom apps), but I have been able to run for days at a time before I have to reboot into XP.

I've got Wine working with some apps, but if anyone has figured out how to get pcAnywhere working on Ubuntu, I'd be one step closer! :)
