### Post by heimo on 2005-08-15
I generated new modeline for 1024x768@75Hz using this page:
[http://www.sh.nu/nvidia/gtf.php]("http://www.sh.nu/nvidia/gtf.php")
 ```
 # 1024x768 @ 75.00 Hz (GTF) hsync: 60.15 kHz; pclk: 81.80 MHz
  Modeline "1024x768_75.00"  81.80  1024 1080 1192 1360  768 769 772 802  -HSync +Vsync
``` 

Copy paste to Monitor section of xorg.conf. If that doesn't help, change via (driver) to vesa and try again,
then you could try with vesa, but without the Modeline. It's possible that any change to xorg.conf can prevent starting X and force you to text-only command line (console). In that case, undo changes to xorg.conf and try restarting gdm (all neccessary steps in post I linked).

EDIT: Oh, and remove those 1x1 parts. No idea what they're doing, but those are definitely not needed.
