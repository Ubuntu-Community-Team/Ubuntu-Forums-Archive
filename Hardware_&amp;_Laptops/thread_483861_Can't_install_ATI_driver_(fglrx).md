sudo aptitude install fglrx-driver

or sudo aptitude install xorg-driver-fglrx

### Post by Prefader on 2007-06-25
There are 2 problems that I'm aware of with this graphics adapter and Ubuntu Feisty:

1) The driver version that works with this card ( 8.28.8 ) won't work on versions of Xorg after 7.1.  Feisty uses 7.2.
2) The driver doesn't seem to compile on Feisty's kernel.

Check [this thread]("http://ubuntuforums.org/showthread.php?t=432554&highlight=radeon+9000+mobility"), too - maybe one of the methods explained there will work for you.  I haven't tried them yet.  If all else fails, it's possible to get hardware acceleration using the open-source ati driver - if you need an xorg.conf to reference, I can supply you with mine.
