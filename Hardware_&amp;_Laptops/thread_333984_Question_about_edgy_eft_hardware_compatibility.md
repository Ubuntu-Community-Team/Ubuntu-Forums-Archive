Hi,

I would recommend checking with [LinuxPrinting.org]("http://openprinting.org/printer_list.cgi") before you buy a printer. If it says the printer works well, then search the forums for that printer name to make sure there aren't any widely known problems. I bought a laser printer from Samsung without checking first and it works only through USB and not through the network interface.
