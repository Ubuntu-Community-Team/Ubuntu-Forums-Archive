How to Install Ubuntu in Acer Aspire 5043WLMi

1.Install Ubuntu 6.10 Edgy Eft (Feisty kernel panic sound system)
2.Configure internet.
3.Make the automatic atualization
4.Install the video card  Wiki ATI: [http://wiki.cchtml.com/index.php/Ubuntu_Edgy_Installation_Guide](http://wiki.cchtml.com/index.php/Ubuntu_Edgy_Installation_Guide)
5.Install wireless driver: [http://ubuntuforums.org/showthread.php?t=197102](http://ubuntuforums.org/showthread.php?t=197102)
6. Install acer-acpi: [http://ubuntuforums.org/showthread.php?t=224349&highlight=acer_acpi](http://ubuntuforums.org/showthread.php?t=224349&highlight=acer_acpi)
7.Install Automatix: [http://www.guiaubuntupt.org/wiki/index.php?title=Ubuntu_edgy](http://www.guiaubuntupt.org/wiki/index.php?title=Ubuntu_edgy)
8.In Automatix, install video and sound codecs.Install the Automatix reade/write NTFS/FAT32 mounter, to avoid problems with PQSERVICE. 
9.Install beryl: [http://wiki.beryl-project.org/index.php/Install/Ubuntu/Edgy/XGL](http://wiki.beryl-project.org/index.php/Install/Ubuntu/Edgy/XGL)

Instalação 100% Ubuntu no Acer Aspire 5043WLMi

1.Instalar o Ubuntu 6.10 Edgy Eft
2.Configure a internet.
3.Fazer atualização automática de todos os pacotes
4.Instale a placa de vídeo pelo tutorial da Wiki ATI: [http://wiki.cchtml.com/index.php/Ubuntu_Edgy_Installation_Guide](http://wiki.cchtml.com/index.php/Ubuntu_Edgy_Installation_Guide)
5.Instale o driver da placa wireless
[http://ubuntuforums.org/showthread.php?t=197102](http://ubuntuforums.org/showthread.php?t=197102)
6.Depois, instale o acer-acpi para poder ligar a placa (luzinha acende e apaga)
[http://ubuntuforums.org/showthread.php?t=224349&highlight=acer_acpi](http://ubuntuforums.org/showthread.php?t=224349&highlight=acer_acpi)
7.Instale o Automatix e outros programas utilizando o Guia: [http://www.guiaubuntupt.org/wiki/index.php?title=Ubuntu_edgy](http://www.guiaubuntupt.org/wiki/index.php?title=Ubuntu_edgy)
8.Pelo Automatix, instale os programas que interessar. Essencial instalar os codecs de vídeo e som, DVD e etc. Também é essencial instalar o Automatix reade/write NTFS/FAT32 mounter, para evitar os problemas do Ubuntu com a partição de recuperação PQSERVICE. 
9.Instalar o beryl e as frescuras que estiver afim: [http://wiki.beryl-project.org/index.php/Install/Ubuntu/Edgy/XGL](http://wiki.beryl-project.org/index.php/Install/Ubuntu/Edgy/XGL)
