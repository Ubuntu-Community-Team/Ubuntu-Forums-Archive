### Post by zerofx on 2007-03-30
I actually resolved this eventually. There was one command that was featured in one howto, but not the other.

```
mkdir -p /usr/X11R6/lib/modules/dri
ln -s /usr/lib/dri/fglrx_dri.so /usr/X11R6/lib/modules/dri 
```

in addition to this command as well which I overlooked.

```
sudo ln -s /usr/lib/dri /usr/lib/xorg/modules/dri
```

The basic steps I took to resolve this were as follows:
-Install Uby
-Update Uby
-Follow **[this howto]("https://help.ubuntu.com/community/BinaryDriverHowto/ATI#head-d8c6fd05bce340dfc3ad483abf0e18997868540b")** up to and including the troubleshooting tips. I used the "Ubuntu Way" from the repos. Haven't tested with the bleeding from ATI's website.

And that's all really. Simple.
