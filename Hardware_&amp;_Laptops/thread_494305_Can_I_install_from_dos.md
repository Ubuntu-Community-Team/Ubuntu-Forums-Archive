Hi, I'd like to install xubuntu on an older Toshiba Portege 3110ct.  It's a small laptop, so the cd drive is external and I can't boot from it.  I was checking out Linux Laptops, and someone installed a version of debian using this method, here's a quote:

[I]Unfortunately it does not seem to be possible to boot from the 3110CT's external CDROM drive, so I took advantage of the pre-installed Windows and copied the contents of the debian/dists/potato/main/disks-i386/current/ on the CD to a new C:\debian\current directory under Windows.

Now the installer can be run from DOS. The only safe way to get into DOS is to restart the machine and then press F8 as it reboots. Choose "DOS command line Safe Mode" from the menu that appears. Trying to use a DOS shell under Windows causes all sorts of trouble.

Once in DOS, cd to the debian\current directory you made earlier and run the install.bat script.[/I]

The link and details can be found here:
