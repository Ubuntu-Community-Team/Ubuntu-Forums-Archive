### Post by taehher on 2007-06-18
Are you experiencing performance issues?  Most likely what is happening is that your ram is being used as a disk cache.  You shouldn't worry about manually clearing the cache, If applications need more ram then the kernel will automatically clear some of the disk cache. 
My free memory usually hovers around 10-15mb free and I have 2 gigs of ram.  So don;t worry, free memory is a wasted resource.
