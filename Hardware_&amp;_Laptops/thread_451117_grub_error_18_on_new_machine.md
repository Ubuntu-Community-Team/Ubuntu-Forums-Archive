### Post by matburton on 2007-05-23
I'm no expert but this is probably because the BIOS on the machine you're now using has labelled the drive differently compared to the other machine.

So Grub can't find where it's installed and hence error 18.

I'd wack the live CD in and reinstall Grub (you can do the same when you take the hard disk back)

Take a look at [[COLOR="Blue"]this post[/COLOR]]("http://ubuntuforums.org/showthread.php?t=24113"), I'd try the second method personally.

Hope that helps ;)
