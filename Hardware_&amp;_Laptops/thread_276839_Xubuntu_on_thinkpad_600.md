### Post by ssalman on 2006-10-13
I'm not sure if this is the same problem I had or not, but I had the same symptoms on a thinkpad x22. I found [this]("http://www.ubuntuforums.org/showthread.php?t=229485") thread which helped me figure out what was the problem.

anyway... when you boot, use the following option to force it not to use the ATI driver (if you have an ATI video card):

at the end of the boot options line add --->  "vesa"

That worked for me.
