### Post by PureRumble on 2007-04-06
Hi, and thanks for taking your time reading this. I am running Ubuntu Edgy. I have an ATI RADEON 9800 PRO 128 MB. My problem is that I can not change my screen resolution and/or refresh rate.

I go System >> Preferences >> Screen Resolution. The settings are currently 1280x1024 @ 75 Hz. I try to change resolution or refresh rate and confirm my changes. This dialog window pops up asking me if I want to keep the new settings or revert back to previous settings. But when I see this dialog window the resolution and refresh rate is still the same, namely 1280x1024 @ 75 Hz. Even if I click "keep", it will still stay the same.

I have installed ATI drivers using this guide [https://help.ubuntu.com/community/BinaryDriverHowto/ATI](https://help.ubuntu.com/community/BinaryDriverHowto/ATI)

I have also done the following.

```

mkdir -p /usr/X11R6/lib/modules/dri  
ln -s /usr/lib/dri/fglrx_dri.so /usr/X11R6/lib/modules/dri 

```

This was a troubleshooting step provided by the guide in case it didn't work. After that I wrote those commands, and restarted the Xserver, the resolution had changes to 1280x1024 @ 75 Hz and I was happy. But I did not know that I wouldn't be able to change them later.
