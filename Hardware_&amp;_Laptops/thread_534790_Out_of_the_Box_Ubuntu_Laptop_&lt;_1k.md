### Post by unixhead on 2007-08-26
I'd recommend going for Dell systems or Acer laptops. I've been installing Ubuntu on collegues' Travel Mates, I write this post using Aspire 3683. TM works fine out of the box, you only have to install 915resolution package to get 1280x800 by default.

Aspire 3683 has headphones issue in Gutsy but everything else works like a charm, including WiFi.

Given even with 2Gb of RAM (512 is not enough for Vista they run by default anyway) the price is lower than $1K, it is not a bad choice.

But sticking with ugly Dell laptops may be even better idea, they have a support and test the hardware. Supporting Dell gives them options to install Linux on more and more machines, by the way.

Hope this helps.
