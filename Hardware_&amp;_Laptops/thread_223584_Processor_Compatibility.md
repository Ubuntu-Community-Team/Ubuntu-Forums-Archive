### Post by Jagot on 2006-07-26
Linux can handle more processor architectures than any other operating system, and therefore can indeed handle AMD 64 bit processors, or indeed the dual core ones.

An iso specific to AMD 64 processors is available to download from the main site.

The only problem with the 64bit processors is that some packages are not yet produced for them, most notably is probably Adobe Flash.  There are workarounds, but it's a little bit more hassle.

### Post by cracker on 2006-07-27
Linux is capable of using just about any processor, and usually better than windows. If the x86_64 distribution doesn't have the packages you need, you can still use i386 kernels on Athlon 64s--they are 100% backward compatible with x86. The only thing I would stay away from are Intel Core Duo laptops, as a LOT of them seem to have issues with the cooling fans not working. This is not an issue with the processor itself, but with the motherboards being used to support them. As such, some Intel Core Duo machines will work perfectly fine under linux. The thing is, make sure you pay attention when choosing your motherboard, and any other critical hardware, like video cards, drive controllers, network cards, modems, etc., when building your computer. These pieces are where compatibility can be a problem.
