I think I've found the answer - I had to install foomatic-db-gutenprint. Not sure why it had disappeared. I see that Gutenprint is the new name for Gimp-print - maybe it somehow got removed during an update because of the name change?

Anyway - all working now. I hope this helps anyone else with the same problem.
