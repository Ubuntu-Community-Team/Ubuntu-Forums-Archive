I don't have time right now to check about drivers for this laptop but...
1. Hard drive rpm's will have zero effect on anything
2. If you download the "Live CD" of ubuntu you can boot up with it without causing any changes to your Hard drive. 

There are two main desktop versions "live" and "alternate" 
Live will allow you boot up into a running from the CD version of Ubuntu, if you like it once it has booted up you then choose the "install" icon on the desktop.
The alternate is more of a non-graphical installation without the "live" OS feature. It's not completely non-graphical but for a novice it might be more advanced than you'd appreciate.

Try the LIVE CD if you boot up into the Ubuntu desktop play with it for a while. If it runs well consider installing it.
