### Post by evilghost on 2007-01-23
Those with the EMU10K1 chipset work very well in Linux because they support 32-channel hardware mixing.  The Sound Blaster Live 5.1 series have the EMU10K1 chipset.

Game performance is increased when using this cards because it mitigates the need for software mixing.  Additionally I find that simultaneous sounds works perfectly without the need for dmix/etc.
