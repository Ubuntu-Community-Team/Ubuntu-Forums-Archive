### Post by kkallio on 2006-02-23
I have searched the forums and the web for help in setting up my Canon (BJC-4550) printer in Ubuntu with no luck. It just seems that several others have similar problems with various printers: the printer is not detected properly, maybe it is possible to send some data to the printer and get some response, but not really get it printing.

I already gave up with this and proceeded with other things - namely installing Kubuntu. Then, for my surprise, I noticed that the printer was detected and worked properly when I tried it in KDE. And since then it has also been working in Ubuntu, so kubuntu printer installer did something that fixed it.

There definitely is something screwed with Ubuntu printer detection mechanism. So if you have printer problems, just give it a try in Kubuntu first, it may solve the problems...
