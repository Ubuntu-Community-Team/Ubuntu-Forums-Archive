### Post by mocoloco on 2007-02-09
Here's what I would suggest to you.

1) Make sure the keyboard actually works :)
Can you get into your Bios screen? (As the computer's booting you usually press del, F12, etc.)

2) If yes to #1, is your keyboard irregular and hard for Ubuntu to detect?
Try another live CD, like say [Knoppix]("http://www.knoppix.org/") (also based on Debian) and see if your keyboard is detected and works.

3) If #1-2 is ok, Is the Ubuntu CD defective?
 At the boot menu of the CD you can check for defects on the disc (if you can arrow down to select this your keyboard obviously works)

4) If #1-3 are ok, try using the Ubuntu Alternate CD to install.  It's not graphical, but it's more robust, and the end result is the same OS.

By the way, is it a PS/2 or USB keyboard?
