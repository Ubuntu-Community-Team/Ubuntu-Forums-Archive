### Post by henriquemaia on 2006-10-04
[B]This issue is solved. I posted here for the record, if someone stumbles on the same problem.
[/B]
Recently, on my other computer, grub timer would just stuck there, without counting down. I could boot if I pressed esc and selected the kernel manually, but this was annoying me a bit.

After googling around for some answer, someone pointed out that this could be solved with changing the coin size motherboard battery. I did this (and also a motherboard flash reset) and set everything up again. Grub timer is working again.
