### Post by 67GTA on 2007-06-27
Look in SYSTEM>PREFERENCES>SCREEN RESOLUTION and see if it is listed in the drop down menu. If not, you can boot into recovery mode and run "sudo dpkg-reconfigure xserver-xorg" That will take you through the steps to set these options. Make sure you have your monitor specs beforehand. If you are unsure of an answer, just hit enter and leave it as default.
