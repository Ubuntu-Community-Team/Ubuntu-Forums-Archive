### Post by Rexbron! on 2006-07-12
Do you have a wireless router at home? That would be the easy way to test.

Otherwise, check System Settings > Network settings and see if a wireless device is detected.

Also, check Internet > Wireless Assistant to see if you can connect to any unsecured networks in you nieghbourhood.

Depending on the kernel, you make have to install the madwifi drivers to get ubuntu recognise the hardware. Though all the ubuntu kernels have detected my wireless card by default.

I have an Acer Aspire 3624 at home.
