### Post by cmon on 2007-11-10
Hi there. 

I have the same soundcard as you and I got my sound working by doing this:

1. Install the latest Alsa drivers following this guide: [https://help.ubuntu.com/community/HdaIntelSoundHowto](https://help.ubuntu.com/community/HdaIntelSoundHowto)

2. Installing this package : ```
sudo apt-get install linux-backports-modules-generic
```

3.  Reboot and enjoy the awesome sound of your laptop speakers :)
