### Post by 444 on 2008-02-16
First, you probably want k8 optimization, not k7 (k7 is Athlon XP).

As for performance, it will not improve it much, with most things you will not notice.

An example where it does matter: I was using Abiword to open a very big document on a 900mhz computer, and it was being slow and hogging 100% cpu. Optimizing it for my CPU instead of 386 helped with that noticeably.

So you have to be using something that requires CPU power to get benefit. And you don't need to convert everything, just the one program that needs the CPU.
