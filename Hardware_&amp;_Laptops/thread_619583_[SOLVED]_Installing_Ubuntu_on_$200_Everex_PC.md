### Post by aysiu on 2007-11-22
gOS is based on Ubuntu, so all you would have to do is get a Ubuntu Alternate CD and then type these commands into the terminal: ```
sudo apt-cdrom add
sudo apt-get update
sudo apt-get install ubuntu-desktop
```
