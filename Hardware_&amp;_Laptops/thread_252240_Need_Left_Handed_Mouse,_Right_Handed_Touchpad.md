### Post by burnt on 2006-09-12
Got this to work by doing the following:

-Under Preferences->Mouse I unchecked Left-Handed Mouse
-In my xorg.conf file I added
 Option "ButtonMapping" "3 2 1 6 7"
 under the Mouse Input Device Section.
-I left Touchpad Input Device Section alone.

I had to reboot to have the xorg.conf file reread.
