### Post by geoffsid on 2008-01-01
Hi

Yes exact same model, ordered online, free delivery, came next day . I am very pleased, only start up problem I have seen is that in XP the keyboard occasionally fails to initialize, a reboot rectifies so far. I haven't yet done any XP updates or SP2. Tuxracer causes XP to freeze if run fullscreen.
 Using Feisty 'cos already had CD and heard that Gutsy is quite broken. Wireless adapter is working well after compiling RT73 driver. Sound is working. No scroll facility from touchpad under linux.
 The only problem really is that you have to use the vesa graphics driver, the SiS M672 graphics chip supports 3D under XP and Vista but I can find no sign of a linux driver yet (new hardware?).
Battery runtime is 1hour 20mins. max.
 I considered an Asus EEE PC but came to the conclusion that that after buying an external drive etc. for the Asus I might as well have a real notebook.

PS: The vesa driver shows very faint lines across the desktop when using a light tint such as the default Feisty desktop  (Simple Ubuntu). But it doesn't bother me, just chose a desktop that doesn't show the effect.

Hope you find this useful.
