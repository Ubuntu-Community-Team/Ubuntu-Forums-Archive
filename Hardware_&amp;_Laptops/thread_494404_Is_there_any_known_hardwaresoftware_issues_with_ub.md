Welcome

first thing to do would be to download the "live cd" iso, burn it "as an image" to a disk, and try running it. this will give you the oppotunity to see how your hardware functions with ubuntu without actually making any changes to your system. the live cd runs ubuntu in ram from your cd rom drive, so it is a bit slower than the actual install,but is great for checking things out. if all works correctly on the live cd then you should have no issues when installed. if you have any problems/issues while running the live cd, post back here with specific questions and also list specifics of the hardware that you are using. nearly any hardware you might have can be made to run and there are plenty of people on these forums willing and able to help out.
