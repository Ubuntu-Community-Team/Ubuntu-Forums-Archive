Gung Hay Fa Choy! Turns out the problem was when I had reformatted the drive in Windows, the NTFS format had not completed correctly. I reformatted it as NTFS, disconnected the drive properly and mounted it using the mount command like so : mount -t  ntfs-3g /dev/sdb1 /media/(dir) 
and it worked!
