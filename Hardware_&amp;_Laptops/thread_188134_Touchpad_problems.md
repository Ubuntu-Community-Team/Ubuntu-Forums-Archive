### Post by givré on 2006-06-15
@hikeb : i finaly success in installing qsynaptics.
In your xorg.conf, in your synaptics section, just add the option  ```
Option "ShmConfig" "true"
```
It has to looks like something like that:
```
Section "InputDevice"
        Identifier "Synaptics Touchpad"
        Driver "synaptics"
        Option "SendCoreEvents" "true"
        Option "Device" "/dev/psaux"
        Option "Protocol" "auto-dev"
        Option "HorizScrollDelta" "0"
        Option "ShmConfig" "true"
EndSection
```
Restart X (Alt-Ctr-backspace or sudo /etc/init.d/gdm restart) and it should be ok.
See if it helps for your problem
