### Post by gollybegully on 2007-02-06
ok I have found a solution @ [http://forums.freestandards.org/read.php?25,251,473](http://forums.freestandards.org/read.php?25,251,473) if anyone is interested.  

Till suggested: "To fix the size problem check whether your printer has resolutions 360, 720, 1440 dpi, (multiples of 360 dpi) or 300, 600, 1200 dpi (multiples of 300 dpi). In the printer setup tool choose a model whose resolutions are of the same type of yours."

indeed when I switched to the ip4000 drivers instead it worked like a charm!
