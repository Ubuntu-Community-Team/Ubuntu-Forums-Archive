### Post by hajk on 2005-10-22
Well, you have to install the printer via the System/Administration/Printing menu, I would be highly surprised if a standard HP LaserJet printer wasn't supported.
You probably have two options: the original HP printing support, and the newer  HP (HPLIP) support -- this last bit runs on Breezy by default (I use it myself for another HP printer).
