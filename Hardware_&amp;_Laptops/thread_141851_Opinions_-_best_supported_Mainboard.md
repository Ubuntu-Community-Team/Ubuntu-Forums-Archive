### Post by Bagnaj97 on 2006-03-17
I would recommend an Asrock 939Dual SATA-2. It supports everything you wanted, including silence (Passive chipset cooler and supports Cool n Quiet). I'm using one and it's great. Only 100MBit Lan though. The good thing is that it has a "Future slot" so you can use AMD socket AM2 processors when they come out. It also supports PCI-E and AGP. Everything worked out of the box on Ubuntu Dapper (Haven't tried Breezy on it but it should be fine).
