### Post by pauljwells on 2007-10-17
I tried upgrading to Gutsy and had a lot of problems. With something like an OQO you're already setting yourself something of a challenge so it pays to run a little behind the bleeding edge.

At 6 months old 7.04 is still newer than OSX Tiger or Vista. I have a nice stable installation with Feisty and can help you with it. Gutsy doesn't really add anything that the OQO can use in practical terms; I would wait until it's a couple of months old and the bugs are ironed out. It's still beta in anyone else's book.

Incidentally, I am having some good experience with Puppy Linux. It,s not without its problems, but as it runs completely from RAM it is very fast. It certainly has a lot of promise for the OQO, which is dog slow, it has to be said.

### Post by pauljwells on 2007-11-09
Hi Anchor1te
I installed Ubuntu 7.04 from the text installer. I never got the built-in wifi working but my AT76_USB USB stick worked out of the box (but doesn't work in Gutsy, beware!).
Here is my xorg.conf - it gives you the 800x480 res and gets the tablet stylus working.
The only other change I made was to disable key repeats in keyboard preferences.

```

#

# package.
#
# If you have edited this file but would like it to be automatically updated
# again, run the following command:
#   sudo dpkg-reconfigure -phigh xserver-xorg

Section "Files"
	FontPath	"/usr/share/fonts/X11/misc"
	FontPath	"/usr/share/fonts/X11/cyrillic"
	FontPath	"/usr/share/fonts/X11/100dpi/:unscaled"
	FontPath	"/usr/share/fonts/X11/75dpi/:unscaled"
	FontPath	"/usr/share/fonts/X11/Type1"
	FontPath	"/usr/share/fonts/X11/100dpi"
	FontPath	"/usr/share/fonts/X11/75dpi"
	# path to defoma fonts
	FontPath	"/var/lib/defoma/x-ttcidfont-conf.d/dirs/TrueType"

Section "Module"
	Load	"i2c"
```
