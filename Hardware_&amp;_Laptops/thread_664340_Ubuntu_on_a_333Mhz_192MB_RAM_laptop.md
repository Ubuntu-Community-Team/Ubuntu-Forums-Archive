### Post by psyburn on 2008-01-11
Yes it will, however you will need to install it using the alternate install CD
Normal CD will hang on anything with less than 256MB RAM

You might want to try Xubuntu instead if you feel regular Ubuntu is too slow.

I've used all three on a 400 MHz 256MB
I've used Xubuntu and Ubuntu on the same 400 MHz with 128MB RAM and I prefer Xubuntu as it feels much snappier.
