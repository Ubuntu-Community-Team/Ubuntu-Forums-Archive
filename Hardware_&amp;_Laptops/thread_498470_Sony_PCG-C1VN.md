### Post by spargonaut on 2007-09-11
ii have the same lappy, and have reinstalled linux on it a number of times.
( hey, its a testing lappy, and i'm still learning.  i tend to keep hosing it. )

anyway, i've only installed vanilla debian or knoppix on it.
i've got debian etch installed on it right now.
i'm sure you can get ubuntu working on it, with minimal tweaking.

unfortunately, the easiest way i've found to install ( unless you can lay your hands on a sony pcmcia cd or dvd rom ), is to extract the HDD, which you need to replace anyway, install using a desktop, re-install the hdd, and reconfigure xorg.conf.

at this point, thats about as far as i've gotten.
( networking works right out of the box using a netgear 10/100 pcmcia card )
i just picked it back up a couple weeks ago, and i can really only play with it in my spare time.
next is sound and camera.

i realize this post is a couple months old, but if you want any config files, lemme know.
