### Post by kirys on 2007-11-02
Yeah it worked well on feisty with little work.

Now it seems that i made it works for gutsy too (after a lot of experiments....)

here is how I've done (it is the result of a trial and error, so i cannot guarantee that works on other laptop too, and I'm not sure the procedure make any sense ^_^) :

1) i booted the laptop without the external monitor
2) i've run "sudo dpkg-reconfigure xserver-xorg", i've configured manually the graphic board to use the nvidia drivers on the specs of the lcd monitor of the laptop (just to reset all changes i did on the xorg.conf
3) i've run "sudo nvidia-xconfig -twinview"
4) i've run "sudo nvidia-xconfig -dynamic-twinview"
5) i've rebooted the laptop, it does configure the monitor well but with some erratic twinview behaviour (like dual monitor but the monitor of the laptop is off so i've half screen visible :p...)
6) I've fixed the behaviour using nvidia-settings (i want the laptop monitor off while using the external one ^_^)

Hope that the procedure works also for you.
