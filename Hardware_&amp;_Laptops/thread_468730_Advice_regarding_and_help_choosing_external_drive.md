### Post by ngaio on 2007-06-09
You can buy external cases, e.g. Vantec, that have both USB 2.0 and Firewire.  Firewire uses a lot less CPU than USB, and it has error correction protocols built in.  You can also daisy-chain them, if you like to do that in future.  However on the down side I don't care much for the fiddly 4 pin interface.  If you have an express card slot, investigate eSATA -- it's a lot faster.  Vantec has USB & eSATA combo drives.
