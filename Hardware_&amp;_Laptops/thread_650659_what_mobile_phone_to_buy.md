### Post by jackhynes on 2007-12-30
Okay so I've got my phone and so far I've been blown away by it. It's simply amazing!

The only problem is it's not particularly Linux compatible. The plug and play works but unmounting doesn't always seem to work and results in file errors when trying to play on the phone. Converting videos etc. can be done using a series of programs including mplayer, but I was a wimp and booted Windows to run a program called 'Super' which is built on a lot of Linux apps.

The music quality is excellent and the 3.5mm headphone jack is great for plugging into Hi-Fis. The wireless capability is what really sells it for me. You get broadband speeds without paying any extra for an all inclusive data plan. The problem is for that is that you have to be within range of a signal but as I spend a good portion of the day at home it's great.

All in all it's a great phone but is nearly completely unsupported in Linux.
