Dell monitors are decent.  you should be able to get a lot of monitor for $500.  If you restrict it to dell II would get a nice 24' like this one: [http://accessories.us.dell.com/sna/products/Monitors/productdetail.aspx?c=us&l=en&s=dhs&cs=19&sku=320-6095](http://accessories.us.dell.com/sna/products/Monitors/productdetail.aspx?c=us&l=en&s=dhs&cs=19&sku=320-6095).
