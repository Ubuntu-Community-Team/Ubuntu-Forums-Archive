### Post by dabl on 2007-08-31
I built my rig last fall, for Linux.  I chose an Intel motherboard even though it was considerably more expensive than the alternatives.  I think it was a smart move because:

- Integrated HDA sound system works great -- no need for a separate sound card
- BIOS is up-to-speed with all current technology drive types
- Intel makes their BIOS flash updates available for Linux as well as Windows

not to mention that I wanted the Core 2 Extreme CPU, and no issues about compatibility of the CPU and motherboard.  :lolflag:

Then just pick an Nvidia card -- they all work as far as I can tell.
