### Post by rivalarrival on 2007-09-11
Short answer: No, it didn't do it.

Long Answer: Probably. :)

How far in the boot process does it get?  What errors do you get? 

If you can't get to the bios settings at the very beginning, you've got a hardware problem that the software just can't cause.

If it's something about "non system disk" or "missing operating system" then yeah, ubuntu MIGHT have something to do with it, but you'd still be able to boot to the ubuntu installation disk or a windows installation disk, or any other boot disk. 

If you really can't boot to anything, you've got a hardware problem, not an ubuntu problem.
