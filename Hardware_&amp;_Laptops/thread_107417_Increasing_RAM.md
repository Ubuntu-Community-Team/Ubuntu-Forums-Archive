### Post by kingsidy on 2005-12-23
you can actually install the ram yourselve. it is pretty easy. Just open wherever the ram goes in your laptop and remove the old one in case it is using all slots and snap the new one in. I think that the BIOS will detect the change in ram and adjust accordingly. I have done it a couple of time on laptops dual booting linux without any problems. The BIOS will say that it detected a new value of ram and it will change it automatically i think. It did not affect any of the laptop at all except for the improved performance. At least 1 gig is the way to go. I got rid of my swap in ubuntu and my pagefile in windows. Run excellent.
