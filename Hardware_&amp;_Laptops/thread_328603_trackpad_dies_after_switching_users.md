### Post by theonlykman on 2007-05-16
I can also confirm this behavior. A usb mouse does work so its not that big of an issue for me. But for other people wanting to use my laptop it is. 

Also, the problem doesn't occur if I log off and a different user logs in. It only happens when switching users while still logged in. Based on this I sort of came up with a workaround, kinda. 

Instead of directly switching users, try locking your screen and from the resulting password dialog hit the "switch users" button. For me, doing this, the mousepad does freeze.

Its not that much more inconvenient, but that still leaves this problem to be fixed.
Hope that helps
