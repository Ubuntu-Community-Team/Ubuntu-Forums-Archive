Hi,
  I am running an Iiyama Vision Master Pro 510 monitor off an Nvidia GeForece FX5200 graphics card and am using the Nvidia binary drivers. I have noticed that there is flickering at the bottom and top left of the screen, particularly noticeable when you drag any window containing plenty of white areas (e.g. an xterm) over the areas concerned. I have used a

sudo dpkg-reconfigure xserver-xorg

to try and get around this by creating a new /etc/X11/xorg.conf file. My default screen resolution is 1600x1200 and I use 24-bit colour depth. The monitor is attached using the BNC inputs, but I wouldn't think this would make much difference. 

I have used the values for the limits of horizontal and vertical sync ranges from the Iiyama technical reference, but am thinking that maybe a more conservative setting for these values is necessary. Please see my xorg.conf file below.

Any suggestions as to how to get rid of this flickering will be much appreciated!
