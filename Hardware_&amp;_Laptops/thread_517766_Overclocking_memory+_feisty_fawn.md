### Post by fredj on 2007-08-05
Any overclocking has to be done in the bios. Windows doesn't have any overclocking facilities
built in and neither does Ubuntu or any other operating system. If you can't do it in the Bios then
you can't do it.

---

### Post by fredj on 2007-08-05
Any windows program you have that allows overclocking does so because it is changing the 
bios, it is not changing windows. The memory speed is always set in the bios. The speed gain
between DDR333 and DDR400 is so small so why bother.
