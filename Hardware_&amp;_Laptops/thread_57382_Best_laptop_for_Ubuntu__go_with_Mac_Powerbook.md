[QUOTE=oneafrikan]Hey all

Quick question for ya, which I'd love to know your thoughts to:

Gareth[/QUOTE]
 I have a PC desktop dual booting WinXP and Ubuntu.  The PC spends 95% of it's time in Ubuntu.

I also have a 12" iBook dual booting OSX and Fedora Core 4.  The iBook spends 95% of it's time in OSX.  Why?  Because there is no linux support for Airport Extreme.  The only way I can network with FC4 on the iBook is with an ethernet cable.  And that goes for Ubuntu as well.

Also, package selection on linux-ppc is far more limited.  If you're used to Hoary on x86, prepare to run in to a few things on ppc that aren't in the repos.

I never faced any horror stories with installs.  Both Ubuntu Hoary and Fedora Core 4 installed without much hassle on the iBook.  You'll just have to reinstall OSX beforehand to resize the partitions, as Ubuntu can't do that after the fact like it can on Windows.

So here's my recommendation:  my iBook is a great machine.  I'd go for the iBook and just run OSX.  If you want to run Ubuntu as the primary OS, use an x86-based laptop.  It'll be easier and more things will work.

### Post by xmm0 on 2005-08-18
Get a thinkpad.  Heck, if cost is a factor go e-bay, but make sure it has the warranty.  Verify this, the e-bay listing should have the model and type as well as the serial.  Type that into [http://www-307.ibm.com/pc/support/site.wss/pageRedirect.do?lang=en_US&page=pewselect&brand=root&doctype=&subtype=Cat](http://www-307.ibm.com/pc/support/site.wss/pageRedirect.do?lang=en_US&page=pewselect&brand=root&doctype=&subtype=Cat)
and check it.  If its good, then go for it.  Thinkpad is incredibly supported under linux, with very few exceptions.  Heck, even the ACPI things work like sleep and hibernate, which I have not seen on any other laptop I've owned or played with. (one of my professors, however, got ACPI working on his Gateway tablet...... I'm yet to see it.) The support is fantastic amongst the thinkpad linux crowd.  Anyhow take a look at [www.thinkwiki.org](www.thinkwiki.org) and forum.thinkpads.com .   BTW, I recently went to the DefCon hacker convention lugging my Gateway 7422gx around (HEAVY) and noticed that Thinkpads were in style in a big way.   Most running linux, if that says anything.

Support for linux on the iBook isn't so great, all my buddies with them (all linux geeks) run OS X only.  They don't want to dual boot from an OS that has total hw support to one that doesn't.  I admit, they are sexy, but when it comes down to it, you'd be better off looking at a x86 machine.

PS: get a thinkpad

### Post by JELaVallee on 2005-08-20
I've been running Ubuntu Hoary on an IBM Thinkpad T43p for a couple of weeks now and it's been a dream.  Prior to Ubuntu, I also ran SuSE 9.2 Pro (v. stable but rather dated packages), SuSE 9.3 Pro (not as stable, had a lot of wireless issue and GNOME 2.10 was kinda clobbed together tho not as bad as RedHat), and CentOS4 (functional, but not all that satisfying w/r/t wireless hardware support) on this machine.  Ubuntu just worked and I've alway been a bigger fan of apt-get/deb over yast/rpm.

I also dropped the licensing fee on VMWare Workstation 5 ([http://www.vmware.com](http://www.vmware.com)) and have that setup to run Win2kPro on a 3Gb virtual partition.  That's great for having win32 gimme apps like Adobe Creative Suite, MS Office, and non-3D game support (mmm, Starcraft...).

With this machine (as with most notebooks) I highly recommend getting the gtkwifi panel applet (v 1.8.1 is excellent) for managing wireless configs across varied networks... I still have some issues here, but have atleast been able to get WEP40-128 stable.

Hope this helps,
