### Post by Icarosaurus on 2007-06-26
Sorry for late reply...
I hope you have already solved your problem...
I have got this wireless mouse...worked automatically for me...
But I removed my old ps2 mouse leaving xorg.conf as it was (I see you kept your wired mouse active, adding your wireless one).
So my Xorg.conf looks like this:

```
Section "InputDevice"
	Identifier	"Configured Mouse"
	Driver		"mouse"
	Option		"CorePointer"
	Option		"Device"		"/dev/input/mice"
	Option		"Protocol"		"ImPS/2"
	Option		"ZAxisMapping"		"4 5"
	Option		"Emulate3Buttons"	"true"
	Option		"Resolution"		"1600"
	Option		"SampleRate"		"6700"
EndSection
```

I just added "resolution" and "SampleRate" options, fitting my laser mouse capabilities..
bye!

### Post by brazzmonkey on 2007-06-27
i forgot about this topic...
i eventually made it work using a slightly different configuration :
```
Section "InputDevice"
	Identifier "Wireless Mouse"
	Driver "mouse"
	Option "SendCoreEvents" "true"
	Option "Protocol" "auto"
	Option "ZAxisMapping" "4 5"
	Option "Emulate3Buttons" "true"
	Option "Device" "/dev/input/mice"
	#Option "Name" "062a:0000"
	#Option "CorePointer"
EndSection
```
