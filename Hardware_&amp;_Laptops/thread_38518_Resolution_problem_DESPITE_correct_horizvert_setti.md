### Post by skoal on 2005-05-31
You might want to check the contents of /var/log/Xorg.0.log.  Look for these sections to give you a better idea of what's happening:

to see what the driver is actually using for hsync/vsync (here's mine):
(II) NVIDIA(0): Generic Monitor: Using hsync range of 30.00-107.00 kHz
(II) NVIDIA(0): Generic Monitor: Using vrefresh range of 48.00-120.00 Hz

and the validated modes under this line:
(**) NVIDIA(0): Validated modes for display device CRT-1:
(**) NVIDIA(0):      Default mode "1280x1024": 157.5 MHz, 91.1 kHz, 85.0 Hz
(**) NVIDIA(0):      Default mode "1024x768": 94.5 MHz, 68.7 kHz, 85.0 Hz
[...]

* You might want to try adding these 2 lines to your 'xorg.conf' file (under the Device section), since you have an Nvidia card:

Option          "UseEdidFreqs" "off"
Option          "IgnoreEDID" "on"

and see what modes are validated after that.

** Are you sure the mode 1152x768 is actually supported by your monitor?  Make those entries 1280x1024 (or higher) instead and see if any other resolutions show up.
