### Post by reckless2k2 on 2007-09-07
flashing the bios is usually never a bad thing unless it's a beta. even then, it would really depend on research and any possible conflicts. i had a machine that i had to downgrade the bios for certain feature sets not available in the current version. if and only if i needed to use a certain feature set then i could downgrade. 

long and short was that new bios didn't see processor upgrade. older bios would recognize it. when i upgraded the processor above a certain range, i had to downgrade the bios to except the higher processor speed. 

so all in all it's really feature/hardware specific. typically, a new flash is always a good thing.
