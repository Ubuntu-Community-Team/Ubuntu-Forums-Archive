### Post by geraldm on 2007-07-15
man xorg.conf
Look in the DISPLAY SUBSECTION for the keyword "Virtual"
after that give the x dimension and then the y dimension.
It is set for each depth in use.
use "Viewport" to set the view to your choice area.

As usual, back up the xorg.conf file so you have it to return to if
the new edit did not work as you wanted it to.

Gerald
