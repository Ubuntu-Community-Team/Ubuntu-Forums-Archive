### Post by PhilippWollermann on 2005-07-08
Hi!

Which monitor do you use and which connection (VGA, DVI, notebook display) is between monitor and graphics card? :-)

If you're still using the F700P (as mentioned in your xorg.conf), maybe a modeline can help to reduce that shaking ... you could try one of these web-based modeline calculators or you could use the "ati" driver again and look through the xorg logfile (I guess the modeline it uses is written to the logfile :)).

Arghh, third edit. ;) Switching to the old "ati" driver should be no problem. Just insert this into your xorg.conf:

```
Section "Device"
        Identifier      "my_ati"
        Driver          "ati"
        BusID           "PCI:1:0:0"
EndSection
```

Then change the "Device" entry in the "Screen" section to "my_ati".

Philipp
