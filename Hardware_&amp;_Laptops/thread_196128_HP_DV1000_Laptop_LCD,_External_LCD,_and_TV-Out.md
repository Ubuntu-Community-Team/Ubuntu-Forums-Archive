	SubSection "Display"
		Depth		32

	EndSubSection
```Have you tried to add a 1680x1080 mode to the default screen?
```

		Modes		"1680x1080" "1280x768"
	EndSubSection
```The X server should try to use the optimal resolution automatically via DDC. Maybe you're able to change the resolution manually using xrandr or the gnome gui resolution manager after switching screens using Fn+F4, if it doesn't happen automatically?
