### Post by logos34 on 2008-03-19
The live cd needs 256mb ram to run, but sometimes even that is not enough (or maybe graphics is stealing some).  Tell him to try the text-mode alternate install cd.  Make sure the Bios boot order is 1. cdrom 2. hard drive 3. removable/floppy.  Or just physically disconnect the floppy drive from the cable.
