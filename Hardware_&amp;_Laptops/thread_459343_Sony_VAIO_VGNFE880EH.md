Using a version of Wubi that works with Vista I was able to install Feisty Fawn onto the laptop. After a grub17 error and replacing the files it finally worked. Function keys work sort of. 

Pretty good, got Beryl running as well and started installing some applications I frequently use. However, I soon noticied my build in microphone doesn't work. Though it's detected as HDA Intel.
