While these are good ideas, they don't change the actual amount of heat coming out from the computer, but they will make the computer more quiet. Opening the window will of course cool your room down, but there may be a few things you can do with your computer to make it generate less heat.

Does your CPU support frequency scaling? If so, then changing the settings for this will surely help alot. Also, some graphics cards supports changing the core frequency, which may help. 
If you use Desktop effects, try to turn these off.
Also, you can try to have a minimal set of programs and services running.

If this is not enough, then there are more drastic things you can do: remove unnecessary harddrives, underclock the CPU and if possible undervolt it as well. But at this level, things are a bit technical... :)

### Post by Soybean on 2007-09-07
You could also try disabling or disconnecting unused peripherals. Some of them produce a surprising amount of heat. There was a thread a while ago about a built-in webcam that was getting really hot, even when not in use, for example.

Also, you said this was a problem while you were using the computer... but if you're ever away from it long enough for the screensaver to activate, switching to the "Blank Screen" screensaver can significantly improve your heat production on some hardware. I'm not sure what it is about some of the screensavers that's so difficult, but they can really make some computers heat up.
