### Post by noone on 2005-08-29
Hi

I found a way to fix the black screen on Xserver-startup with the Ati Radeon X700 (mobile)

Just add:

Option "MonitorLayout" "LVDS,AUTO"

to the Card-Section in xorg.conf

maybe that helps some people...

Then it looks like:

```
Section "Device"
	Identifier	"ATI Technologies, Inc. Radeon Mobility X700 (RV410)"
	Driver		"ati"
	BusID		"PCI:1:0:0"
	Option		"MonitorLayout" "LVDS,AUTO"
``` 


But there is another little problem with the new config.
i get from time to time a "Display changed: LCD off" message.

If anybody knows, how to fix it, please tell me.

bye
