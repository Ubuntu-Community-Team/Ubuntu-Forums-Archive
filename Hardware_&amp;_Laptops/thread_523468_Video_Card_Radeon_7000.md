### Post by w4ett on 2007-08-12
> **chadwicktr said:**
> Sorry, I am still learning all the ins and outs of ubuntu. But from what I can tell from system monitor it looks like 64mb of my 768.

How would I change this? I've searched and googled but haven't found anything yet...

The radeon 7000 chipset/cards are NOT  supported by the proprietary fglrx video driver (fglrx supports the 9500 series and above) so the open source xorg driver 'ati' is the one you should be using. (I use the 9200 series and am in the same boat as you)  You might want to verify your bios settings to ensure that the shared memory settings are properly enabled.  

768 MB of system memory is plenty for Ubuntu, so that is not an issue.  I think I've read that your laptop has some issues due to the fact it has memory permanently installed on the motherboard ....have you tested your memory using memtest on the live cd?  If not, now would be the time to do so to verify it's integrity. 

Also let's make sure you are running the correct driver.  From the terminal type:
```

cat /etc/X11/xorg.conf
```

and post the output.
