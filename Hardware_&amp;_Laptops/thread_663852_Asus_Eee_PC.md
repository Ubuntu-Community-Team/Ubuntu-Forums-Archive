### Post by BrendanM on 2008-01-10
I don't own an EEE PC, but my understanding is that it comes with Open Office, which is mostly compatible with MS Office formats (the fonts might get messed up occassionally or something like that). You could also reformat the drive and install Windows XP and Microsoft Office, but then you'd have to buy a copy of WinXP and Office. Unless your Uni has some kind of special educational discount deal.

### Post by michaelzap on 2008-01-10
I don't own one either...yet. Apparently there will be a new version in a couple months or so with more storage space, a faster processor, and no cooling fan for total silence so I'm going to wait and see. And in theory the next generation will be able to run Windows XP (ew).

The current version runs a custom distro of Xandros Linux, so OpenOffice (and AbiWord, Gnumeric, etc.) run on it fine. So you should get about the same level of Office compatibility that you do from Ubuntu in general. That means that some PowerPoint presentations may look a little screwy, but in general you shouldn't have any major problems.

And yes you can reformat these and put Gutsy on them as well.
