### Post by Perfex on 2007-08-23
Well there are alot of people looking for working printers in linux , the HP Deskjet 4100 I just purchased at Walmart for my mothers pc running fiesty works great.
Just thought I would share the experience 

1. asuming you have the ink cartridges installed, etc. plug everything in usb, power
2. Download the Self-Extracting Archive With Installer @ [http://hplip.sourceforge.net/downloads.html](http://hplip.sourceforge.net/downloads.html)
3. Right click the .run file , open it, run it
4. it will go through a long installation process , it will promt you too restart
5. go to system , preferences ,HPLIP Tool box, follow the promts on setting up the printer 
(it's pretty self explanitory )
6. You should be set to print & scan
Note I printed a Document she had been working on in openoffice on some cardstock, used the highest settings, printed gorgeous  

Note that the scan button does not work on the printer it self, though you can scan using Xsane Image scanner, also you can set up everything else, view ink levels etc. via the HPLIP toolbox.

**Note the HPLIP in the synaptic wont work with the printer**

Good luck
