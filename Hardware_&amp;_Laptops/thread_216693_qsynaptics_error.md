### Post by stevieb on 2006-07-16
Got it running, but what ended up working was uninstalling "xfree86-driver-synaptics" and rebooting.  then (with no mouse!-make sure you hotkey a terminal session- i use yakuake) then sudo apt-get install xserver-xorg-input-synaptics.

if anyone has any caveats to add to this, please do.  i'm a newbie and this might actually be a dangerous thing to do!
