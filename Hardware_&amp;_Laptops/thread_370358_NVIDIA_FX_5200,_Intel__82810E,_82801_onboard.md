1. Ubuntu installation (Ubuntu 6.10  w/kernel 2.6.17-10-generic)

3. NVIDIA drivers installation with *envy*. **Monitor is connected to onboard output**.
After this just reconnect a monitor to PCI card. Everything is great.

**System log** /var/log/Xorg.0.log

### Post by Linux_oid on 2007-03-09
I'm using TV-out (S-video). Computer is connected to TV set. Resolution is limited to 
1024x768 by NVIDIA. Update manager is on. Notifications about updating are so annoying so I press update and X server doesn't start next day.

1. Disconnect TV.
2. Connect monitor to "PC out" on card.
3. Restart (reboot) computer.
4. BIOS setting > set AGP > turn computer off.
5. Connect monitor to onboard "PC out".
6. Turn computer on. X server doesn't start.
7. Press Ctrl+Alt+F1 > log in > start  ***envy* ** (I'm using *envy* for NVIDIA drivers installation)> Install NVIDIA.
8. After awhile you have black screen with blinking cursor. Press Alt+F1 to see messages.
9. After installation is finished go BIOS settting > set PCI > save. Turn computer off. Connect monitor to card's "PC out".
10. Done.
