### Post by John the train on 2007-04-23
I've just installed Xubuntu Dapper on my Dell Latitude ( 192MB RAM, 300Mhz P2 processor ) and it's looking good. I used the alternate CD as Slipperhead recommends, as I understand it the alternate CD has a lower RAM threshold for install than the live CD. If you're DLing the alternate CD the Xfce desktop in Xubuntu is generally considered less RAM hungry than Gnome, hence my personal choice and - I assume - BoneKracker's recommendation.
