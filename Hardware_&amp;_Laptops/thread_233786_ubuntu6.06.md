### Post by cstudent on 2006-08-10
If the Ubuntu 6.06 Cd booted up to where you see the desktop you should have 2 icons. One says examples and the other says install. To install you click on the install icon and follow the setup wizard instructions.

If you have anything on your Ubuntu 5.10 disk you want to save, like music or documents, etc., you better back them up first to a CD or something. They will be wiped out when you install Ubuntu 6.06.
