### Post by windowsrefund on 2006-01-12
Hello everyone,

I thought it would be interesting to start a thread on keyboards. For some time now, I've been looking for a new keyboard that meets the following requirements and haven't had much success:

- Should work fine with GNU/Linux
- Should "feel" great
- No need to install proprietary software (drivers) in order to get it to work

Even better if...
- Small (82 keys)
- ALT key on both left and right side
- CTRL key on both left and right side
- No useless windoze key

So far I have this to say about the products I've seen..

Deck Keyboard 
[http://deckkeyboards.com](http://deckkeyboards.com)

I just got received this in the mail yesterday and tried it out last night. Despite all the great reviews I read while researching this product, trust me! It feels HORRIBLE. The keys do not feel stable and seem rather "high" meaning you have to press the keys down quite a bit before they actually make their mark. Other annoyances include the fact that some keys are in odd positions and are sized differently depending on where they're located. For example, the "1" key is much larger than the "2", "3", "4" (and so on) keys. I believe this design decision was made for cosmetic reasons since this key appears to be the same size of the backspace key that sits in the same location on the opposite side of the board. The end result is that it is very easy to mistakenly hit the "1" key when attempting to hit ESC. Moving on, the ENTER key is annoyingly small. The BACKSPACE key is still very small (even though it's the same size as "1"). There is no ALT or CTRL key on the right hand side (something I planned on addressing by remapping the DEL and INS keys). Overall, all the keys feel a bit small and just feel horrible when typing. I will be returning this keyboard ASAP.


Happy Hacker Keyboards 
[http://shop.store.yahoo.com/pfuca-store/index.html](http://shop.store.yahoo.com/pfuca-store/index.html)

Are these people smoking crack? The keys have been placed in the strangest layout I've seen so far! The ~,',|, and \ keys have been placed in the upmost right-hand corner. No direction keys (although some models appear to have been built with a very sloppy looking attempt at placing these keys where the HHKB logo is normally placed). ALT keys appear to exist on both sides but alot of valuable space is also taken up by useless MAC keys. Overall, I can't see buying this thing considering it's high price and key locations.

DAS Keyboard
[http://www.daskeyboard.com/](http://www.daskeyboard.com/)

OK, the most obvious thing about this board is that it has no key markings. But the reviews are saying that this board has an incredible "feel" to it due to the fact that each key is individually weighted. I bought one of these for my buddy and haven't heard anything bad (yet). However, he doesn't type much so he may not have fully realized any of the board's benefits. I probably would have picked myself up one already but hesitate since it is a full size board (numberic keypad, function keys, and all). 

Logitech Boards (wireless)

These are the best "feeling" boards I've ever typed on but I know I'm going to end up banging my head against the wall if I try to get any of these working under GNU/Linux. What a shame.

So how about it guys? What's else is out there?

Best,

Adam
