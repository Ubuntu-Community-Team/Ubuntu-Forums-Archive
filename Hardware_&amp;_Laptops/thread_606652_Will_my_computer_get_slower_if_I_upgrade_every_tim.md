### Post by adam.tropics on 2007-11-08
The current version is 7.10. As far as I know, you can upgrade to that, but, and someone correct me if I am wrong, using the update manager to do that will require two upgrades, first to Feisty, then to Gutsy. Seems like a waste of downloading to me. The best bet would be to back up your data and such, and then download the xubuntu Alternate install cd....[see here]("http://www.xubuntu.org/get")....you have enough memory to install from that but apparently now the regular cd.

I am not sure anyone can say for sure your machine will 'always' be able to run the latest xubuntu for example, as over time, requirements are bound to change, but for now at least you should be fine. Plus, there's nothing to say that you have to upgrade...if your machine performs the tasks you need, why worry about it!
