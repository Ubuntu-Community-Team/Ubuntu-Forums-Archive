### Post by bcardarella on 2008-01-24
My laptop has a numerical keypad on the right hand side. (17" monitor... so I guess they decide to squeeze more keys onto the system) Anyway, the issue I'm having is when I try to use SHIFT along with the END or HOME keys which are on the numerical keypad. (END = 1, HOME = 7).

Pressing SHIFT and either END or HOME usually allows me to highlight and entire row from the cursor position. But now it is simply printing the number of the keypad. This is simulating the action of having NumLock on.... very annoying.

Is there any way to turn this off? I'm hoping that this is just a key mapping issue and not built into the hardware.

### Post by zilupe on 2008-03-06
System - Preferences - Layout Options - Miscellaneous compatibility options
Check "Shift with numpad keys works as in MS Windows".
