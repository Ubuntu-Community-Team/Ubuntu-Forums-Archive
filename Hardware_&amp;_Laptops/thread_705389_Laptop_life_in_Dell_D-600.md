### Post by acranox on 2008-02-24
A few searches will reveal many people have problems with decreased battery life.  But you can alleviate about 90% of the issues by turning on the various power-saving features that probably aren't enabled by default.  Make sure laptop-mode it turned on, but disable the hard drive spin-down.    Set up cpu frequency scaling.  You can monitor the rate of discharge of the battery, which is helpful to allow you to monitor changes.  You can also enable power saving of the wireless card.
You've just got to start configuring the computer, it takes a little TLC...and time.
