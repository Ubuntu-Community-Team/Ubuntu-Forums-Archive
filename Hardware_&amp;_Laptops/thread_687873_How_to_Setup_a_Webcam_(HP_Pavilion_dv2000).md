### Post by Agent ME on 2008-02-04
Try getting and trying the program 'cheese' (its a mac photobooth clone). It seems to work with a lot of webcams, including ones that only support v4l2 drivers.

(You can get cheese by going to Applications -> Add/Remove, and entering it in the search bar. When it comes up, check the box next to it and press apply changes.)

If the webcam doesn't work then, you'll probably have to find a driver for it.
