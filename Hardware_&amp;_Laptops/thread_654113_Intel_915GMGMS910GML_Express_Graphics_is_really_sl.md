```

        Identifier      "Generic Video Card"
        Driver          "i810" <------------ chnage me to "intel"
        BusID           "PCI:0:2:0"

```

oh wait shouldn't you have a modules section like this?

```
Section "Module"
	Load		"glx"
	Load		"GLcore"
	Load		"v4l"
EndSection
```

I would try adding the modules sections first
