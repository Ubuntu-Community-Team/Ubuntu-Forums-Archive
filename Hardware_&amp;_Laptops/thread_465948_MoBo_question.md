### Post by IntuitiveNipple on 2007-06-06
Based on experience I'd say your problem is around 
> after carefully cleaning the CPU heat sink and fan I applied more thermal paste
The BIOS code is executed by the CPU, so if the CPU can't start then you'll not get the Power-On Self-Test.

I'm not sure how much you know about thermal paste so I'll mention a few issues just-in-case.
[LIST]
[*]Remove the heatsink
[*]Remove the CPU
[*]Clean the heatsink and CPU so no old paste remains
[*]Apply a very small amount of paste to the CPU
[*]Spread it evenly with the blade of a box-cutter knife
[*]The paste should be so thin it is translucent and you can just make out the markings on the chip
[*]There should be no air gaps, streaks, or bubbles
[*]Replace the CPU ensuring the pins are seated firmly and the CPU is pressed down before clamping it
[*]Replace the heatsink taking care to place it evenly without twisting or leaning so the paste isn't disturbed
[*]Clip the heatsink clamps into place
[/LIST]

This basic procedure with special attention to the details has saved the day on numerous system builds for me.
