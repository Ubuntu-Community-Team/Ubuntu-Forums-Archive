Hi all,
Today I was asked to install Kubuntu Edgy 6.10 on a PC based on an ASUS P5P800 SE Motherboard. At first I cannot start the installation process (I use the Alternate CD), receiving a "CD mount error", so I changed DVD reader. This time installation process started, but was painfully slow. So I decided to download and browse the motherboard manual, discovering that the motherboard presents two peripheral management modes , a so-called "Enhanced mode" and a legacy mode. The first one permits to manage 4 IDE peripherals + 2 SATA peripherals; the legacy one permits only 4 IDE. I switched to Legacy (I have only IDE peripherals) and all worked well.... Anyone has better ideas ?
