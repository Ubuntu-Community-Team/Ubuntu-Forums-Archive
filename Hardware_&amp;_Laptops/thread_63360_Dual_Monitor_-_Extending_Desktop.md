### Post by mlord on 2005-09-09
[QUOTE=lamar_air]My notebook does have an ATI Radeon video card.  How do I go about doing this simpler way?  Please give me whatever detail you can because i'm sort of new to using linux.[/QUOTE]

There's no nice GUI for this one yet, except maybe (?) in the ATI binary-only drivers (avoid those, unless you're really a glutton for punishment and just *have* to have the fastest 3D videogame performance on the block).

So you have to become your own expert.  Save a spare copy of the X11 configuration file (/etc/X11/xorg.conf), and then edit the original file and experiment with these options:

Section "Device"
        Driver "ati"
        ...
       Option "MonitorLayout" "LVDS, CRT"
       Option "MergedFB" "true"
       Option "CRT2Position" "RightOf"
       Option "MetaModes" "1024x768-1024x768"
       ...
If your external monitor is NOT plugged in before you start up Ubuntu, then you must also provide more info to X11 about the capabilities of the monitor you will be plugging in.  Look these up in the monitor's documentation, or risk frying your monitor.
        ...
        Option          "CRT2HSync"       "31.0-80.0"
        Option          "CRT2VRefresh"  "60.0-76.0"
        ...

Again, you MUST read the manpage a few zillion times until you begin to understand what those options are doing.  Then you can tweak them as needed until things work how you want them to.   Use this command in a text window:

   man radeon

Cheers!
