### Post by ibzter on 2005-09-18
if your using gnome, it's possible to get there features by installning the gnome-bluetooth and obexserver packages. you will also need bluez-pin for device pairing. after installing these packages you will have a "Bluetooth File Sharing" applet available in Applications -> System Tools. after you have paired your devices you will be able to share your files between bluetooth enabled devices.(if you're running kubuntu the same things can be found in the kde-bluetooth package)
