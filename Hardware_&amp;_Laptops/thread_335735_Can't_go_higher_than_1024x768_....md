### Post by Enverex on 2007-01-10
Open /etc/X11/xorg.conf in some editor, whatever you like using like gedit, nano, etc and then paste it here.

EDIT: Damn too slow.

Anyway, see those lines:
        Modes      "1024x768" "800x600" "640x480"

Well you need to edit them to add the modes you want, say....
        Modes      "1600x1200" "1024x768" "800x600" "640x480"

and then save the file back as the xorg.conf and restart X (ctrl+alt+backspace).
