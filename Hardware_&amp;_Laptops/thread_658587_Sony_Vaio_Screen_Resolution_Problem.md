Hello

I have a Sony Vaio laptop (model  PCG-982L). It has a Pentium III chip. My problem is that since I upgraded from Dapper to Gutsy my screen resolution is stuck at 640 X 480 (with the refresh rate at 60Hz). This means that I now have a 2 1/2 inch black border around my screen. When I attempted to adjust this in my Screen Resolution Preferences I found that 640 X 480 is the only option in the drop down menu. 

I'm no expert but I think the problem might be that the proper driver for my video card is not installed. Unfortunately, I do not have the user manual to the laptop;  I have no idea what kind of video card it has, and have not had any luck finding this info using google. 

Interestingly, when I recently tried out the new gOS linux distro (which I heard is really Ubuntu in different clothing) I had no problem with the screen resolution.

### Post by Will2Power on 2008-01-04
Problem solved. I just followed these steps:

sudo apt-get install xserver-xorg-video-intel 

Then edit your /etc/X11/xorg.conf and change the

Driver "i810" 

to

Driver "intel" 

You can edit the xorg.conf from the command line using

sudo nano /etc/X11/xorg.conf
