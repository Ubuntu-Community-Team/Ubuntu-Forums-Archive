### Post by abovett on 2005-12-30
Try using the "System Monitor" (under Applications -> System Tools), or open a terminal window and run "top". Either of these will show you what processes are running and how much CPU time they are using etc. You may find that when the system is running slow there's a process using all the CPU time or something.
