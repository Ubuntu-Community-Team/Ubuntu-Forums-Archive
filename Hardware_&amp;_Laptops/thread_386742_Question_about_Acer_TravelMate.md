### Post by jjordan on 2007-04-04
Take a look for your model on this page:

[https://wiki.ubuntu.com/LaptopTestingTeam/Acer](https://wiki.ubuntu.com/LaptopTestingTeam/Acer)

There is a similar model listed.  No one can guarantee that every hardware device on your laptop is going to operate without some configuration effort.  Laptops come in a mind boggeling array of models and sub-models.  Even within a given model there are options and manufacturers frequently change hardware specifications without telling anyone.  This is not a problem only in the Linux/ FOSS world.  The way this community improves and expands is by people trying things, figuring things out and contributing.  A good first step is to set up a dual boot system, so that you can use that other operating system while you are testing the the Linux waters.  

I use a dual boot with a reliable version of Kubuntu on one partition and a Beta version on another.

I recommend you install the Beta of Kubuntu 7.04 and give it a try.  It is fairly easy to setup a dual boot and even though 7.04 is still in Beta I believe it already supports Laptops better than 6.10.

Before you start; check Windows drive, delete all temp files and defrag.  That will make it much easier to resize.

-j

### Post by jjordan on 2007-04-04
The Live CD will be slowwwww. But take heart it will still give you an idea of what will work on the first try.  The live CD tries to detect everything and set it up but is not always as successful as an installation.  

Some people like Gnome, Some people like KDE.  For me the consistent interface and variety of applications available outweigh the percieved performance hit.  Some people like vanilla, some like chocolate.  Using one does not lock you out of the other, there are a couple of Gnome programs that do not have effective counter parts in KDE, Gparted comes to mind.

The Gnome KDE debate will go on forever, I hope.  There is plenty of space in FOSS for both desktops.  My real favorite is GnuStep but I am not sure that will ever be ready for prime time.
