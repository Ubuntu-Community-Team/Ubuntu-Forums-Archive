### Post by l:x on 2007-02-16
Hello Analgins,

If you want to install Ubuntu on a Pentium 200Mhz with only 32MB RAM, you can't install a full Gnome Desktop Environment, neither can you install a KDE one.

You'll have to think about Xubuntu (Xfce Window Manager) wich is meant to be a light-weight WM for low-end machines.

There are other light-weight alternatives, of course. (Like Fluxbox, Openbox, ...) You've distro's that are Ubuntu based, but use a WM other than Gnome, KDE or Xfce (like Fluxbuntu that uses Fluxbox).

Just search around on the internet and keep us posted about your progress!
