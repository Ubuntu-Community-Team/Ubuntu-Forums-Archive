### Post by alephnull on 2006-10-19
According to this post:
[http://www.ubuntuforums.org/showthread.php?t=221174&highlight=monitorlayout](http://www.ubuntuforums.org/showthread.php?t=221174&highlight=monitorlayout)
TwinView is only for nVidia. Since you have the Intel 945G, you'd want to use Xinerama to get the dualhead monitor config working, although I also have a D620 with the Intel video chipset and have had no success thus far.
You should be able to switch displays using <fn><f8> on the laptop keyboard, and you can remove the port replicator from the docking station so that you can actually open the laptop lid to do this. Not an elegant solution by any means, but this is what I currently do to allow use of my external monitor.
I'm currently using a default system-generated xorg.conf; FWIW, here is my xorg.conf that does NOT (currently!) work for the dual head configuration (but should, according to different posts I've read):
