### Post by watchitman on 2004-12-12
This is a work around. Go to 'Computer > Desktop Preferences > Sound' and uncheck  'Enable sound server startup.' If you want your system event sounds back, recheck the the box. It's stupid but it works. Maybe one of these days there will be a proper sound subsystem on Linux that all developers use (as opposed to trying to write their own or whatever it is they do). In the meantime we have to do nonsense like this.
