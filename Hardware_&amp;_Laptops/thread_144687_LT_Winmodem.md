### Post by boon on 2006-03-15
The key resource you need to access is:

linmodems.org  

Essentially, all the information is there. Take some time to inspect the site, but if it is unclear to you, do not panic -  proceed.

If I could summarise the steps you need to take:

1. Download and run the scanmodem utility. If you only have access to the internet via windows you will need to download the file via windows and then mount your windows partition and copy across the file to your ubuntu partition.

2. Once you have the output from scanmodem, join the mailing list for an explanation of what to do next. 

3. Post your queries to linmodems maillist in plain text - html or richtext will be rejected without explanation.

4. Report back here on your progress (or lack therof) during this process - we will help I hope.
