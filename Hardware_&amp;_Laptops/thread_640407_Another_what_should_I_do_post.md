### Post by ilrudie on 2007-12-14
As for the things you need/want:
Playing MP3s and DVD's is 100% possible... Search the forums for great info on getting it setup.
 
I'm not sure about Netflix Instant or what it requires.  If its a windows only client you can always try it in WINE.
 
BelArc is a nifty tool and there are ways to get all that information and more in Ubuntu but I am not aware of a program that wraps it all up in a neat belarc style bundle.  I don't know what CrapCleaner does but most likely you wont need it... Ubuntu uses the apt package manager which cleans up after itself nicely and will inform you if you have extra packages that are nolonger required when you uninstall programs.  
 
Burning audio cd's is a breeze.  Serpentine is built in but there are tons of other burning tools that you can try and see what you like.
 
Convert flac audio should also be no problem as flac support is built into Ubuntu.  Flac and Ogg are the two prefered formats for SoundJuicer which is the default cd ripping program.
 
Online backups would depend on the site and if it requires some special client side programs.  If you only want to back up a few things you can probably just use FTP to get them to the site.
 
Making PDFs is also easy.  The easiest way is to install the cups-pdf package.  You can then set up a PDF printer that most any program in ubuntu will be able to use.
 
I hope this encourages you to try out Ubuntu and good luck with the switch.

### Post by popch on 2007-12-16
On the hardware side:

The Walmart $200 PC is called 'green' for its low power consumption. It uses a VIA board. Some members of the Ubuntu forums have bought the machine and appear to be pleased.

When energy consumption really is an issue, you should either buy hardware which does not consume any power at all when turned off or you should use a power strip which cuts off all power when the computer is not in use.

And, of course, you should turn off your computer and the periphery when not in use.
