### Post by Wattos on 2007-06-17
when you start your machine press F2 or del which will bring up your bios. In the bios settings go to boot settings (or something similar) and change the boot priorities to CD rom and then HDD. then save and exit.  

The easiest way to do dual boot with grub is to install first WIndows XP on one partition and then Ubuntu on another partition. If you install ubuntu later on, ubuntu will take care of the dual boot
