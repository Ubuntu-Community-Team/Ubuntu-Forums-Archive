### Post by opscure on 2008-02-18
You will need to configure your xorg.conf file inside of /etc/X11/.  You can find a template for 3 monitor setup here:
[http://terrencemiao.com/Webmail/msg00928.html]("http://terrencemiao.com/Webmail/msg00928.html")
and the man page for configuring your xorg.conf file here:
[http://linux.die.net/man/5/xorg.conf]("http://linux.die.net/man/5/xorg.conf")

Hope it helps.
