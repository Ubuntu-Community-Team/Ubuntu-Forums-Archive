### Post by pseudo-random on 2008-04-08
It's all about BIOS settings. Some older MOBO's won't support booting off USB.
Some do but not by default so you'll need to enable it.
Some do but they have the hard-drive or CD-rom as priority one and two.
You need to enter the BIOS and change the settings.
