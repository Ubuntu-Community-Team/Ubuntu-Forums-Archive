### Post by ExBuM on 2008-01-04
I'm trying to install the 32-bit version of ubuntu onto my laptop (a Dell Inspiron 8000). I can see the loading bars and stuff when I'm booting from CD but when I select "Start or Install Ubuntu," nothing shows up. I know it's there because I can hear the bongo music and when I pressed ctrl + alt + del, up, and then enter, my laptop turned off. I checked the CD for defects with that first menu and it said everything was okay. 

I then tried starting from the Safe Graphics mode and then everything worked. So is it safe to install or do I have to do something first to make sure that I can see stuff when Ubuntu is installed?

### Post by Miotch on 2008-01-04
I had that happen with my desktop. It should be just fine to install, but just for kicks, when you put the install CD in, there is an F-button that corresponds with VGA. Hit that, and select some settings you know work with your laptop. 

Your laptop might not be configured to take the default resolution that the CD is told to use.

That worked for me, at least.
