### Post by gilgongo on 2007-10-23
Have you tried opening a terminal and typing "alsamixer" in there? That should bring up one of the most revolting user interfaces you are ever likely to see this side of 1985, but it might enable you to turn the sound back on. If you see "MM" anywhere, that means "mute" so it the "M" again to turn that back on. Use the arrow keys to move the volumes up. It escape to save the setting and quit.
