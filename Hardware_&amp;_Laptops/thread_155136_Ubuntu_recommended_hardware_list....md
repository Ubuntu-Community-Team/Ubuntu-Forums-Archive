### Post by DoubleClicker on 2006-04-04
There is a difference between supported hardware and recommended hardware.  support hardware means if "you use it, it will work".  Recommended hardware means, "of all the choices out there, these are the ones that will work the best."

I would like to know configurations that, I know if I bought, it would "just work" without any need for tweeking or editing config files, and would take full advantage of the hardware.
