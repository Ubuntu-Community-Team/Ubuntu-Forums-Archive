### Post by jjordan on 2007-05-14
Skype under Linux has no video support.  Wengophone is available for Windows, Mac, and Linux and has video.  I use it from my Linux box to my wifes MAC.  Best of all it is FOSS and standards based.  Download the latest release candidate from the openwengo website it works much better than the one in the repos and you don't really have to install, just unzip and run the script.

WengoPhone can interoperate with AIM, MSN and a few others but just chat.  

I use my Creative Live Cam Notebook Pro with it.
