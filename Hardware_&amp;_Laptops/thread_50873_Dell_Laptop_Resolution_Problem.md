### Post by javiadip on 2005-07-21
try opening xorg.conf file and adding the resolutions there

to open a xorg.conf 

```
sudo pico /etc/X11/xorg.conf
```

try adding this to the end of the file where you see the configuration of your 800x600 resolution

```


SubSection "Display"
Depth     4
Modes    "1024x768"
EndSubSection


```

try switching the Depth number to 1,4, 8 or 15. also make your DefaultDepth is set as 24
