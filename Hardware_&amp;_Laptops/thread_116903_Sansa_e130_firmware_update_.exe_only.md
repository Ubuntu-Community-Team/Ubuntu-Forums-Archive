### Post by chammi on 2006-05-07
I think the easiest thing to do is just to:

1. put it into Mass Storage Mode
2. copy the exe file to it
3. take it to a friend/neighbor/relative who has WinXP, 
4. move the exe file onto their desktop 
5. and then run it on that machine. 

I don't think the firmware ties you to a particular computer, but I don't play DRM music anyway, so I don't know.  Sounds just like Micro$oft to somehow get system info (harddrive s/n, XP product key, etc) from the computer that you are installing the firmware from and only let you copy music to the player from THAT computer....but luckily, I don't think it's that sneaky (yet).
