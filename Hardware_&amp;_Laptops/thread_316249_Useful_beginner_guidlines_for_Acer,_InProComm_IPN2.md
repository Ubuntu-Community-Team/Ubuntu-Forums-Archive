### Post by abh83 on 2006-12-10
This is not a HowTo but rather a collection of useful links and tips that I, as a complete beginner to Ubuntu and Linux in General, used as a guideline.

I hope this will help the Acer Aspire user and especially those who have the 1520 series. However, this will also be useful to people with a Linksys InProComm IPN2220 wireless card, a WPA-encrypted Wifi access point and a nVidia GeForce FX Go graphics card.

I have an Acer Apsire 1520 (1524WLMi/1525WLMi):

> AMD Athlon 64 processor 3700+
nVidia GeForce FX Go 5700 64MB
TSSTcorpCD/DVD-Dual drive (+/-RW)
512MB DDR RAM
802.11 b/g wireless LAN: Linksys InProComm IPN2220 Wifi


After installing Ubuntu 6.06 - Dapper Drake - Gnome almost everything seemed to be working just fine. However, i was only able to connect to the internet via Ethernet Cable (due to my wireless router and encryption issues) and the graphics card was running at a lower resolution.

My DVD drive, synaptic Touchpad, 4 usb ports, usb external ntfs harddisk, usb typhoon optical 3 button mouse and all the other hardware is running flawlessly!
