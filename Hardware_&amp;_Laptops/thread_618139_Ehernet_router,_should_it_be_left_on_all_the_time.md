### Post by KOld Iron on 2007-11-21
Buying a router certainly has some advantages, maybe even for just one computer:
- router logs in at your internet provider automatically after power on (it stores all the configuration as said before)
- it normally has a built-in firewall (you may want to check with the particular product that it really has one though)
- it provides DHCP, which means you do not have to worry about static IP configuration of your laptop ethernet interface
- you could connect a printer via ethernet, which means the printer does not have to be connected to your laptop directly (better mobility)

If you buy one, however, I would consider buying one with a WLAN access point included, because you get all the features just mentioned and increased mobility (since you have a laptop and assuming is has already a WLAN interface built-in).

All of this is of course not really necessary, but nice, when you have it :-)
