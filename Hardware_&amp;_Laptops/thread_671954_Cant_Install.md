### Post by mac.ryan on 2008-01-19
Nope, but it happened to me in the past that the default settings of the live CD where such that my computer would not boot properly with them.

During the text-mode installation (only the installation is text-mode, the installed version of ubuntu has graphics!) you will be prompted for more configuration options. Once finished the installation you will boot from the HD and you should find yourself in the graphic environment.

At least this was how I worked around similar problems with my Dell XPS.
