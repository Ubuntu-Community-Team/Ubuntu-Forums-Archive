### Post by ArcticChicken on 2007-12-03
I'm sure you figured this out, but I thought I'd reply in case it's helpful for anyone else.

Within the xorg.conf file, the "BusID" entry belongs in the Device section for your video card (or rather the video card you're interested in targeting and defining, if more than one is present).  To translate the numeric sequence given by lspci into the BusID entry you will need to change the period to a colon and you can leave out any leading zeros.  In your case, that would render "02:0a.1" into a line that reads:

BusID "PCI:2:a:1"

Perhaps an easier way to figure out what BusID entry you need is to run:

Xorg -scanpci

Note that this command will fail if X is still running.  That said, the output should give you the exact string of text you need to use for the BusID (e.g. "1:1:0").
