Hi, I'm sure this is more of a hardware issue than an Ubuntu issue.... anyhow, I just put together a Frankenstein box with parts and pieces from many different sources.  I had the Ubuntu Server install cd.  The CD booted fine, the install went fine, the first reboot into the new system went fine.  However, on the second reboot - the BIOS will not detect the HD.

### Post by NobleSavage on 2006-12-23
Fixed my problem.  The HD 160GB was too big for the controler or BIOS or something.  Put in an old 40G and it works fine.  :)
