### Post by endersshadow on 2006-11-04
Here's how I fixed it:

```
sudo apt-get remove --purge 915resolution
```

Then I restarted X. Then...

```
sudo apt-get install 915resolution
sudo 915resolution -l

```

And here was my problem...I was using 7e.  I saw that 5a was a res that I wasn't goingto use (1600x1200, 32bits), so then I did this:

```
sudo 915resolution 5a 1400 1050 32
```

At this point, I restarted X AND IT WORKED! I checked my /etc/default/915resolution file, and it was set up for auto, so I then rebooted to test it.  It went back to its old hand of 1280x1024 and blurriness.  So, I manually put in 5a, 1400, 1050, and 32 for the default config file, respectively.  I then restarted, and voila...it works :-D
