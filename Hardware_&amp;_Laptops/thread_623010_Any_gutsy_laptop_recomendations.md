### Post by tlivingd on 2007-11-28
I'm running the lenovo y410 150ish gb hard drive and 2gb memory 1.66mhz dual core.  dual booting with vista.  
this is my first linux/ubuntu, install ever.  and it hasn't been too difficult.  My sound doesn't work yet but we're working on that (search for lenovo y410)  and i had an issue with WEP authorization with my wireless network with ubuntu (windoze was fine) switched over to WPA1 and found it to work great, (maybe an issue if you log into coffee shops however.  it's more secure too (couldn't get the gf's computer to log onto a WPA2 network though so 1 it was for me.)
oh and the touch pad speed/acceleration was goofy but 3 simple lines in some file fixed that (not sure of file but again i'm new to this).  (search for alps touch pad.) The lenovo y410 actually runs Vista Aero pretty well too, if you plan on keeping it on the machine. though I haven't put any system intensive tasks in front of it.  the machine hardware feels better nicer keyboard with no flex, than my compaq armada and e500 laptops i had before (both business class machines) and this is supposed to be consumer/home office level. and the screen hinges seem better on this than on both aformentioned compaq's.
