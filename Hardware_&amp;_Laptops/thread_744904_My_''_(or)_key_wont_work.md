For any keyboard layout issues, it is important to use System/Preferences/Keyboard to change the settings.

If you add the Keyboard Indicator applet (right click on panel, "Add to panel..."), you can view visually the current layout, and see for each key which character is mapped.

Unless it is a bug in the keyboard layouts, it is most probably a laptop bought in another country, and the keys are labeled in a different way (change layout to fix).
