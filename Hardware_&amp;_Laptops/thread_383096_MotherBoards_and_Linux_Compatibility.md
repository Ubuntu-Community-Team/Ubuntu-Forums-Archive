### Post by daou on 2007-03-13
It's almost an art trying to figure out whether hardware will work on a Linux distro. For example, some motherboard might work with Ubuntu but not with Fedora (or vice versa). Best thing to do is to google the motherboard with some keywords like "problem" "linux" "compatible" "ubuntu" etc. See if people are having problems. If you run into the same problem a lot of times, then it's probably not smart to buy that MB, especially if there is no workaround. Also, don't buy the newest and flashiest MB on the market. Usually compatibility comes a little late for Linux (if ever).

There are some sites that try to collect compatibility info for Linux (ex.[ http://www.linuxcompatible.org/compatibility.html).]("http://www.linuxcompatible.org/compatibility.html")
