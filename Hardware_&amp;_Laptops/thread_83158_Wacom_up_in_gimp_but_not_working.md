### Post by 23meg on 2005-10-28
Is yours a Xinerama multimonitor setup? If so, try entering the 

```
xsetwacom set cursor gimp on
xsetwacom set stylus gimp on
xsetwacom set eraser gimp on
``` 

commands and on startup and launching GIMP afterwards.

Also, AFAIK the ISDV4 device is meant to be forced in tablet pc digitizers only. See if removing the "Option "ForceDevice" "ISDV4" " lines from your xorg.conf helps.
