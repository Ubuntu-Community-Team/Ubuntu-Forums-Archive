### Post by banewman on 2007-11-23
You've probably used the whole hd for ubuntu - yes?
Windows - being the unfriendly people they are - won't recognize linux file systems.
Put the live cd from ubuntu in and reboot - there's a prog called "gparted" that will let you resize ubuntu and make some free space for windows.
Having said that a dualboot system will be so much easier if you install windows first because it overwrites the mbr and won't recognize the ubuntu install if it's already there. :)
