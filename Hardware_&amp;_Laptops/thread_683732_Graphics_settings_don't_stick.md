### Post by Yellow Pasque on 2008-02-26
I think you're very close, you just need to add the second screen to your server layout and attach the proper screen to the proper screen section.

I'm trying to figure out how you're laying this out. Which monitor is connected to which card? Which card/monitor is the primary one and what position is the other one going to be relative to it? (right, left, above, below)

For example, if you have the GeForce3 connected to the 1600x1200 and you want it to be the primary monitor, and the GeForce 5 conneted to the 1440x900 and to the right of the primary, then your serverlayout would look like:

```
Section "ServerLayout"
	Identifier	"Default Layout"
  screen 0 "screen1" 0 0
  screen 1 "Default Screen" RightOf "screen0"
	Inputdevice	"Generic Keyboard"
	Inputdevice	"Configured Mouse"
EndSection
```

Also, please run this command for me:
```
lspci | grep VGA
```
