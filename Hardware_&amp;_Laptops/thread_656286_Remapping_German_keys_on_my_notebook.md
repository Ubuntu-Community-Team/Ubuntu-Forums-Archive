### Post by jespdj on 2008-01-02
To get rid of the dead keys, go to System / Preferences / Keyboard, select the Layouts tab, and install and activate a different layout (for example US Default). Do you have a German keyboard? Maybe you should then select the German layout there.

You can modify shortcuts with System / Preferences / Keyboard Shortcuts; there you might be able to map those special keys you mentioned.

I am using US International (with dead keys) and these mappings already work (I didn't change anything):

Alt Gr + q = ä
Alt Gr + p = ö
Alt Gr + y = ü
Alt Gr + s = ß
