### Post by cookies on 2007-05-28
Go to Applications > Accessories > Terminal and type this in:

```
sudo nautilus
```

It'll ask for a password, this is your password. Now you can edit that file, But beware, you are able to edit essential system files, too. Back up the file you are trying to edit.
