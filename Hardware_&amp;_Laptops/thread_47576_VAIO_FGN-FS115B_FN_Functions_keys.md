### Post by NikoC on 2005-11-03
The function key support for Sony is not included in the driver pack, you also have to download and install the utilities which are available on the sony website.

[http://www.vaio-link.com/downloads/register/register.asp?l=en](http://www.vaio-link.com/downloads/register/register.asp?l=en)

This is the url where you can fill in your serial number which is on the back of your portable. When you proceed you can download and install several features like the drivers and utilities for the function keys.
