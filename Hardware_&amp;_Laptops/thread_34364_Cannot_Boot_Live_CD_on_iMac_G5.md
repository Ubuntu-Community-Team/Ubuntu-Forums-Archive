### Post by khad on 2005-05-14
Troubleshoot:

1. Did you download the image from Ubuntu or was it a CD you received from Canonical?
2. If downloaded, did you burn it on your iMac?
3. If yes, did you use Disk Utility to burn the image?
4. If yes, that is the problem.

Solution:

1. Download Firestarter FX ([http://www.versiontracker.com/dyn/moreinfo/macosx/19058](http://www.versiontracker.com/dyn/moreinfo/macosx/19058)) and use that to burn the ISO image.
2. Smile big.

If this did not help, let me know.
