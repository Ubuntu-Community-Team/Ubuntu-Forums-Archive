### Post by Mondoman on 2007-11-08
Lars, first check to see if you already have that feature available in your printer driver.  For example, I'm using the CUPS software driver with my HP printers.  CUPS is most likely already installed in your system (you can search for "CUPS" in System/Administration/Synaptic Package Manager to check).  When I want to print multiple pages per physical page, I just use the File/Print window to bring up the printing settings, and on the "Page Setup" tab  the Layout section has a "Pages per side" choice, where you can change the default of 1 to 2, 4, etc.
