### Post by sisseck on 2007-04-28
Well dont i feel stupid.... found a post sugesting the use of the nvidia-settings command and it worked like a charm.

So here is my guide so far, install the protected driver, then do this
```
sudo nvidia-settings
```
now here is the "tricky" part, i was only able to select 1280x768 but if you select the advanced option you can select a pan area to 1280x800 this at least worked for me. The screen resolution shows 1280x768 but you can see the difference after applying the pan area.

Here is a transcript of my entire xorg.conf file, hope others will find it usefull and thanks again Gypsy for helping me out.
