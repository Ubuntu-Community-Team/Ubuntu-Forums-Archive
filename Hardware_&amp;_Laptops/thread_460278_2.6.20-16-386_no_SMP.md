Hello all, it seems as though the 2.6.20-16-386 kernel does not support Multiple processors.

I installed Feisty on someone's laptop and I tried installing the -386 kernel.  Well after then the System Monitor showed only one CPU.  I knew this was wrong as the PC is a Pentium Dual Core.

I then went back to the -16-generic and the System Monitor showed two CPUs then.

Anyone else have this problem?

I gather the -386 kernel don't support SMP?

Thanks.

### Post by rufius on 2007-05-31
This is not an error per se. The 386 kernel has no optimization flags beyond what the 386 would support and the 386 never had support for SMP. If anything, you would want to install the 686 kernel on this individual's laptop. That said, the generic kernel should be fine, but 386 isn't appropriate for any system with more than one logical core/processor.
