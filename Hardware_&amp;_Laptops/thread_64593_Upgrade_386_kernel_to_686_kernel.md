### Post by Scanner on 2005-09-11
Is there a guide that details how to upgrade the stock 386 kernel to the 686 kernel?  If this can just be done from synaptic which files to do I need to install?  I am running a Compaq Evo N180 notebook that is fully up to date with a gig of RAM and would like to be using all of it.

### Post by mlomker on 2005-09-11
Yup, you can do it through Synaptic.  Just do a search on linux-image and you'll get a list of kernels.

---

### Post by heimo on 2005-09-11
You need to install linux-686 package and reboot. Do not uninstall 386 version, you will have choice to use it if 686 doesn't work for some reason.
