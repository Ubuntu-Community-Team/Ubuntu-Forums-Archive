### Post by hypeiv on 2005-07-19
Good Idea trying the live cd. I have a radeon 9500 and some VIA onboard sound and never had any problems with that combination.

As far as dual booting the installer cd is set up to allow you to set up the dual boot. Since linux will require its own partition it even has tools to help you resize an existing ntfs partition to make room. Its pretty straight forward but be careful b/c anytime you are messing with partitions there is a chance you may really mess something up so its always a good idea to have back ups. On my main pc I only use ubuntu and on my laptop I dual boot but it had windows xp on it before I installed ubuntu. When I boot up I can pick between the two OS's.
 
once you're set up check out [www.ubuntuguide.org](www.ubuntuguide.org) for tips on getting things set up the way you like.
