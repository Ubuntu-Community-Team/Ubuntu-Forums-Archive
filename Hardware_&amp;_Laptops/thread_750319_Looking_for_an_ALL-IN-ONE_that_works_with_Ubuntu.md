### Post by rjs987 on 2008-04-11
I really like the HP Photosmart c6280 that I bought last week. Upgraded HPLIP to the latest version and now I can do more on this in Linux than I ever could with my separate Canon ip3000 and Visioneer 6200 usb scanner in Windows. The ip3000 was only partially supported in Linux. I ended up using the CUPS driver for a bjc7000 to get somewhat reliable document prints but was unable to any acceptable photo printing at all. Visioneer is essentially not supported on Linux.

I set up the HP AIO to connect on my network rather than the usb cable and HPLIP automatically found it right away (even faster than Windows ever found any new hardware). I can scan using Kooka, I can print great looking documents from any application, I can even print borderless (full bleed) photos using digiKam, I can copy directly from the scanner to the printer, I can print photos directly from most flash memory cards, and on and on. Very easy setup and very easy to use IMHO.
