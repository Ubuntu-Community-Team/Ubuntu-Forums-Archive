As for the SATA hard drives, I've never had one fail, but I believe Ubuntu will tell you at boot.

### Post by Zyzzyx on 2006-09-16
For the RAID, there might be an audio setting on the mobo or controller to simply have the system beep when there's a failure. At least that's what I'm used with the limited server RAID experience I have. Or, there might be something available for software notification. I really doubt that you'll end up having to reboot to see if a drive has failed.
