### Post by eivind on 2005-10-08
First, try this to see if there are any informative log messages:

Alt+F2

(or Alt+F3 or F4)

During a normal install, some of the virtual terminals shown by pressing Alt+F-something show what is happening. Try this first, and if you can see anything at all, this may be the first step towards remedy...

Good luck!
