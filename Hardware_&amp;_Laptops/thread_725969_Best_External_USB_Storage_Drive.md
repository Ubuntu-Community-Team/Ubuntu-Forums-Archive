### Post by ugm6hr on 2008-03-16
Most USB drives work fine.

I have a Toshiba (3.5") and a WD Passport (2.5").  Both just need plugging in.

Only thing is that most (excluding WD) HD's are NTFS format, so they just need reformatting to ext3.

### Post by michaelaly on 2008-03-16
Any external USB drive should work just fine in Ubuntu, I've plugged several different types in and had no problems. The only thing I can think of is some drives come with automatic backup software that may only run on windows, but that won't prevent you from using the hard drive.
