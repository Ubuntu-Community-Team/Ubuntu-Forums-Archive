### Post by one_stinky_bum on 2007-01-18
Get a tablet that uses a Wacom digitizer, like the Levono and IBMs and you should be fine. Check out the linux wacom project to see what's supported and what's not. I had a gateway M280 and support wasn't great. I finally got the tablet/finepoint digitizer to work, but it didn't work as well as it did in windows. I would say that windows tablet support is much better than Ubuntu.

If only we had a tablet PC centered version, or for that matter a linux tablet pc distro.:( 

Hope this helps.
