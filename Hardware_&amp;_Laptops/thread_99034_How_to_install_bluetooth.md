### Post by ltmon on 2005-12-04
Hi,

I did mine with a bluetooth USB dongle on a fairly standard desktop.  You may have different issues with a laptop inbuilt bluetooth.

First off, the basics are described here: [https://wiki.ubuntu.com/BluetoothSetup](https://wiki.ubuntu.com/BluetoothSetup).  Try this, and write back if you have issues.

Next, you need to install a frontend to bluetooth services.  On gnome (Ubuntu) this is in a package called "gnome-bluetooth".  On KDE (kubuntu) it's "kde-bluetooth".  I use kde, and found KDE bluetooth fairly stable and intuitive.  I don't know much about gnome bluetooth.
