### Post by steevc on 2007-04-18
I'm looking at upgrading my aging PC (specs below) to something more up to date. I think it has to be dual core these days and the reviews I've seen say the Intel chips are winning on performance/price.

My plan is to replace the motherboard+CPU+memory that is currently housed in my Antec Aria case. This will impose some limits, but I'd rather not replace stuff that's working fine.

I'm not a big games player, so don't need the ultimate in graphics, but I won't object to reasonable graphics for the kids to play games and to get the latest 3D desktop stuff. I mainly want improved performance for things like converting video/audio files and a generally more responsive system. I like to run things like Folding@home so the system is likely to be running full whack most of the time. I'll only overclock if this is not going to cause heat problems or make the system much noisier. I want the system to be reasonably quiet, but don't expect total silence.

I'd prefer hardware that is going to work well with Ubuntu rather than the cutting edge. This will be a Linux only PC so Windows features are irrelevant.

Here's what I think I need:
[LIST]
[*]mATX motherboard - I'd like one with everything on-board if possible. I need LAN (not wireless) and Firewire (optional as I have a PCI card). Don't need anything too fancy in the sound department. 5.1 is enough. Needs to support 2 ATA devices (HD and DVD). I don't have any SATA yet. At least 2 PCI slots.
[*]Core 2 Duo - probably a 6300. I know the 6320 is coming soon, but depends on the relative prices
[*]Either 1 or 2GB memory. I get by with 768MB now, but a bit more may help. Am I right in thinking that a pair of DIMMS will be quicker than one big one?
[*]Maybe a better CPU cooler if the Intel one is either too tall to fit the Aria or is going to be noisy
[*]Maybe a graphics card if MB graphics is going to have driver issues. Nothing too flash. Preferably passively cooled.
[/LIST]

My budget is around UKP250

Update: Some further reading suggests that the 4300 C2D may be worth a look as the performance penalty may not be much and it may run cooler. I've just seen the 4400 on ebuyer for a few pounds less (they may not have updated the all yet), so that could be ideal. I may compromise on using my old case and get a bigger one so I can fit a quiet fan.
