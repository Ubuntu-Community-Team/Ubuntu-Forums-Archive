### Post by JGJones on 2008-01-21
Sadly we can't have the same for power plugs unless you have a recent Apple laptop (those MagSafe power plugs are a life saver, it's shocking that no-one thought of it earlier!)

As for wired ethernet, I actually go wireless for this one - I only use wired for desktop/servers.

Broken laptops - ie dodgy screens etc - can still be used - ie in office we have a Dell laptop which have a broken switch for closing display (it switch it off) - it get stuck and need taking apart - so in a word, the laptop cannot be closed down otherwise you're stuck with a blank screen.

So my solution was to turn it into a CD cover design/printer station (we have a CD/DVD duplicator in the office for printing out a few hundred's of DVD's for clients so the laptop is used for making cover design and printing it onto DVD's via CD cover printer (printer is a Print Factory 2 / PF2).

So if a laptop is broken in that it can't be used laptop-wise, but can be used in other ways - ie broken screen, plug it into a monitor or headless server or something.
