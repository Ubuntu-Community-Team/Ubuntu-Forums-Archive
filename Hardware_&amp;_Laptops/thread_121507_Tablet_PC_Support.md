### Post by alfcateat on 2006-01-28
Hi All,
   I have a bit of support setup for my Gateway M275 tablet PC. So far, I have been able to get the stylus working with Linux. The only thing I see lacking (well, of course besides handwriting input) has been right-clicking. I was able to get it working using the wacom utilities. If you do a search on ubuntuforums for m275, you'll see one thread which listed quite a bit of info about just getting tablet pc support with an acer notebook. I hope this is helpful. If you need a pointer directly to the thread, please let me know.
    Appearently, X just treats your pen input device just like another mouse. I had to manually edit the file /etc/X11/xorg.conf. FWIW, here's what mine finally ended up with [COLOR="Red"](text in red is the tablet pc functionality parts...)[/COLOR]. I know one part will be machine specific, and that is the /dev/ttyS1 pointer for my tablet input device. This is using an old serial type connection internally. From what I understand, most tablet PC's use USB connections to the input hardware.

Cheers,
John
