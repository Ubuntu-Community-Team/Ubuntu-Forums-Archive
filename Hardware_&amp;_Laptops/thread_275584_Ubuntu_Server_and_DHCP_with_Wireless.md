### Post by mark_g on 2006-10-11
Not all WLAN cards are currently supported by Ubuntu. You best google for the "card name + linux" to see if it works. If it's supported, it's very easy to set up.
Of those cards that don't work normally, you could try to use the ndiswrapper (which uses the card's windows drivers), but that's not always easy. So best is to ensure it'll work by searching the web for it.
