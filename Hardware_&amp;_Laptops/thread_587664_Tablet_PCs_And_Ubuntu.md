### Post by oofnik on 2007-10-23
I have a Lenovo x60 tablet. I use it to take all my notes in college, and I love it.
Unfortunately I've started my note base in a Windows-only program that doesn't like wine (EverNote), so I'm stuck using Windows, at least for this semester. Previously I had been using Ubuntu for two years.
There's a [website]("http://luke.no-ip.org/x60tablet/") that is dedicated to getting all of the hardware of the x60 working properly. It has been a great help, and I hope to be back in Ubuntu next semester!

The only problems I've experienced are that the screen rotation won't work properly with Beryl running (a bug in the intel i810 driver and xrandr, I think), and suspend sometimes mysteriously fails and I have to reboot. But other than that, the tablet input works, the thumbprint reader works (!!), and so does almost everything else with very few problems. It's awesome.
