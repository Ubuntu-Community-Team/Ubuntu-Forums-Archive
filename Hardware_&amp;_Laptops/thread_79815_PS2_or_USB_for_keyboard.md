### Post by metoo on 2005-10-21
None, if everything works as expected.

In rare cases there can be problems during boot to detect USB keyboards properly. Then, if the system hangs during boot, you cannot use the keyboard. I had such an issue a few month ago, where the system hang at boot with a non fatal error, just asked to hit CRTL-D, but would not react to the USB keyboard, but continued booting fine, after I attached a PS/2 keyboard and hit the key-sequence on that. And that was when I had enabled the USB DOS compatibility in the BIOS.

Even so, I use USB keyboards on all of my 3 PCs.
It is really unlikely that you encounter any issues using an USB keyboard.

It's more the question if future PCs will still have a PS/2 interface. So if you want to buy an expensive keyboard, make sure it can be connected to USB.

BTW, the key reading rate can be much higher on USB Keyboards. That's why gamers like to use them (more under W OS). For normal use there is just no difference.
