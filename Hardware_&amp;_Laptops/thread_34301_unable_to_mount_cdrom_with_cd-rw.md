### Post by SWAT on 2005-05-14
If you have dual-boot; Boot into Windows and check if the drive correctly reads/recognizes the CDRW.
If you don't have dual-boot; Take out your CD/DVD drive and put it in a computer with Windows installed. Then check if the drive correctly reads/recognizes the CDRW.
