### Post by Cybolic on 2005-05-14
I've been having nothing but problems with the 7 series of drivers from Nvidia, and so I suggest downloading and installing the latest of the 6 series, [release 6629](http://www.nvidia.com/object/linux_display_ia32_1.0-6629.html).

You need to install it from from a "real" terminal, not an x-terminal, so log out from Gnome or KDE, switch to terminal 1 by pressing Ctrl+Alt+F1, and log in as your normal user, and type the following to exit X:```
sudo /etc/init.d/gdm stop
```[SIZE=1] (this might be kdm or xdm instead of gdm, if you're not using the Gnome login manager)[/SIZE]

then install the kernel headers so you can compile the driver:```
sudo apt-get install linux-headers
```[SIZE=1]This will tell you to select a version, select the version matching your running kernel (you can type "uname -a" to see the kernel-version)[/SIZE]

Now install the driver:```
sudo sh /path/to/NVIDIA-Linux-x86-1.0-6629-pkg1.run
```
Then, to make sure the driver is loaded:```
sudo modprobe nvidia
```
And last, restart X:```
sudo /etc/init.d/gdm restart
```

Hope this helps.
