### Post by James Wilford on 2006-08-06
Hi,

I'd like some advice on building a new system to run Kubuntu 6.06. The primary uses will be browsing, web development, some MP3 encoding, etc. I also want to be able to run VMware Server with decent performance. I'm not a hardcore gamer, but might occasionally want to play a bit of Half Life 2 under WinXP as well.

A major priority for me is low noise. I want the system to be virtually inaudible from anything over 2 or 3 feet away. So far I'm thinking of the following components:

**Case** - [Antec Solo]("http://www.bjorn3d.com/read.php?cID=937") - this is the only part I've made a final decision on. However it doesn't seem to be available anywhere in the UK for at least 2 weeks, so I've got plenty of time to decide on the rest. 
**CPU** - AMD Athlon 64 3000+ socket 939 Venice
**Graphics card** - probably a GeForce 6200 based board, with TV out and DVI. I'll go for a fanless design for low noise.
**HDD** - Samsung Spinpoint 300Gb SATA
**Memory** - 1Gb DDR400 - not sure if 2 * 512Mb is better than a 1Gb stick, or which brand to get, suggestions welcome.
**Optical** - 2 drives, probably Liteon but suggestions welcome.
**Memory card reader** - 3.5" internal black USB reader.

So the big decision is the motherboard. My requirements are that Cool'n'Quiet and onboard temp/fan sensors must work in Linux, ACPI must work, the SATA chipset must work out of the box, and it needs to have enough USB headers to handle the 2 front USB ports on the case plus the internal card reader. 

I've been reading lots of reviews and postings trying to decide on VIA vs. nVidia chipsets. I've read that VIA is more Linux-friendly, but nForce4 performs better, and it does seem to be a popular choice on this forum. I'm considering these 3 ASUS boards at the moment:

**[ASUS A8V]("http://www.asus.com/products.aspx?l1=3&l2=15&l3=68&model=478&modelmenu=1")** - VIA KT800 Pro chipset.
**[ASUS A8V-XE]("http://www.asus.com/products.aspx?l1=3&l2=15&l3=143&model=1043&modelmenu=1")** - VIA K8T890 chipset, similar to A8V but with PCI-E
**[ASUS A8N5X]("http://www.asus.com/products.aspx?l1=3&l2=15&l3=207&model=596&modelmenu=1")** - nForce 4 chipset, PCI-E. More expensive than the others, and has a northbridge fan which I'd have to replace with a big heatsink, to get rid of the noise.

I'm wondering if the A8N5X might be overkill for my needs, but I have read reports of it working perfectly under Ubuntu, so that's a plus. I'd be interested to hear any experiences with the VIA chipsets. I'm very confused on the whole VIA vs. nVidia issue at the moment.

I think I've written enough for now, any advice would be greatly appreciated.

James

### Post by John.Michael.Kane on 2006-08-06
James Wilford the specs seem fine.I built my box around 64 3000+ socket 939 with 1gig ram,and used a biostar tforce 939. it's fast for what i do,however if your going to be doing heavy dev work more then one-two windows open alot of compliing i would op for an x2-3800 it would allow the work to be off loaded between the two procesors.also 2x512Mb is fine if the board supports dual channel though you can get away with a single 1gig stick with would allow you to add another 1gig stick later should you need 2x1-gig.as for the Optical i'm using an NEC ND-3550A.
