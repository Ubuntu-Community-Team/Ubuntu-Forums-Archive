### Post by behemot on 2007-01-26
I successfully installed xUbuntu (breezy) on my softmoded xbox.

[http://sourceforge.net/project/downloading.php?groupname=xbox-linux&filename=xUbuntu-0.9.0-602.iso&use_mirror=superb-east](http://sourceforge.net/project/downloading.php?groupname=xbox-linux&filename=xUbuntu-0.9.0-602.iso&use_mirror=superb-east)

I was unable to set it up on F, I had to use game partion E.

The script failed to correctly setup network so I had to fix /etc/network/interfaces manually.

Another problem was German keyboard layout.  This I was unable to fix as of now.

When I dist-upgraded it to dapper, it completely crashed Ubuntu.

If someone has some success runnig Ubuntu on xbox, please post your experiences.
