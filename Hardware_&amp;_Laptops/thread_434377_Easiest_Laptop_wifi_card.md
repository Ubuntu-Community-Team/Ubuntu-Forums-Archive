### Post by mmatt on 2007-05-06
A quick google: [http://linux-wless.passys.nl/index.php](http://linux-wless.passys.nl/index.php)

No doubt you've already found that site. I moved to linux already having a Belkin 54g wireless card, with it's broadcom chipset. Although there are reverse engineered 'native' drivers for this card, which run at 11 Mb/s, I'm using the ndiswrapper driver which acts as a wrapper around a window's driver so that you can use it in linux. I have to say it works very well on my Dell inspiron 5100.

This wasn't always the case, however, in SuSE 10.0 it was an extreme hassel to make this setup work with WPA. I would choose a wireless card with native linux support, like the ones mentioned in the site above.
