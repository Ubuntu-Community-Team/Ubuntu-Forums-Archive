### Post by Death_Sargent on 2007-05-14
you need to install and configure the fglrx driver

you can try this

[http://ubuntuforums.org/showthread.php?t=438008&highlight=install+fglrx](http://ubuntuforums.org/showthread.php?t=438008&highlight=install+fglrx)

but i recomend this unless you use eye candy
[http://wiki.cchtml.com/index.php/Ubuntu_Edgy_Installation_Guide#C](http://wiki.cchtml.com/index.php/Ubuntu_Edgy_Installation_Guide#C)

then try this if you like eyecandy (beryl and stuff)

Basically just 

```
sudo gedit /etc/X11/xorg.conf
```

edit it so compositing is off by default

```
Section "Extensions"
        Option  "Composite" "0"
EndSection

Section "ServerFlags"
        Option  "AIGLX" "off"
EndSection
```

**NOTE:** be sure to set composite to 0 so you can use xgl to get eyecandy. If you wish to use 3d games and such under eye candy look into this [http://ubuntuforums.org/showthread.php?t=176636](http://ubuntuforums.org/showthread.php?t=176636)

install the driver

```
sudo apt-get update
sudo apt-get install linux-restricted-modules-$(uname -r) #Okay if it is already installed
sudo apt-get install xorg-driver-fglrx
```

configure the xorg and driver

```
sudo aticonfig --initial
```

finalize this safely

```
sudo aticonfig --overlay-type=Xv
```

and reboot
