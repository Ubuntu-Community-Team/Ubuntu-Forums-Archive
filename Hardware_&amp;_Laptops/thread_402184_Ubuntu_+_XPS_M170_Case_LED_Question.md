Pretty sure the OP was referring to "Quickset" and it's ability to change LED color/brightness from within Windows. I think I have what he wants:


[http://linux.dell.com/libsmbios/download/libsmbios/libsmbios-0.13.10/](http://linux.dell.com/libsmbios/download/libsmbios/libsmbios-0.13.10/)

download the tar.gz , 

./configure && make && sudo make all
sudo modprobe dcdbas
sudo bins/dellLEDCtl -h
sudo bins/dellLEDCtl -i
sudo bins/dellLEDCtl -z1 1 -z2 2 -z3 2 -l 7


Dell has a pretty big Linux depo now that they started putting Ubuntu on Notebooks.
