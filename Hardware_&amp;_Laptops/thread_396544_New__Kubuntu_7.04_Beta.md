### Post by mikewhatever on 2007-03-29
Good luck with the testing. By no means a Linux guru, I can only offer some help on Firefox. Type about:config in the address bar, then type backspace in the filter. Change the value to 0 and restart Firefox. The backspace should be working.

Does it only scroll too fast in Firefox? I suspect that's the touchpad/mouse configuration, so look there.

The software you are looking for is mostly security related. Most Ubuntu users do not use neither AV, nor anti-spyware. [http://psychocats.net/ubuntu/security](http://psychocats.net/ubuntu/security)
Others install an AV of their choice, CLAM, Avast or AVG. Clam should be in the repositories, therefore the easiest to install. AVG and Avast now have deb installation files, so should not be too hard to. [http://free.grisoft.com/doc/5390/lng/us/tpl/v5#avg-anti-virus-free](http://free.grisoft.com/doc/5390/lng/us/tpl/v5#avg-anti-virus-free)
