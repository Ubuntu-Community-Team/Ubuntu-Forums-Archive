### Post by LetterRip on 2007-12-19
Hi all, here are the issues I've found with my Everex XT5000T - which I'm migrating to Ubuntu full time (Asta la Vista Vista...)

No sound from primary speakers - was solved with the help of this thread
[http://ubuntuforums.org/showthread.php?t=506303](http://ubuntuforums.org/showthread.php?t=506303)


Volume adjustment via volume wheel - while I can adjust volume via Fn-F6 (mute) Fn-F10 (Volume down) and Fn-F11 (Volume up) I cannot adjust the volume via the wheel on the front of the device.

Brightness adjustment via Fn-F8 and Fn-F9 
Alas unsolved as of currently. In Windows those shortcuts lower and raise screen brightness respectively.

Wireless enable/disable via a switch on the front of the laptop - the switch turns off its LED indicating that the wireless is off but it remains on.  In Windows I can use it to turn off wireless if I want to save additional power, or am in an area where wireless is not allowed.

Reduced Power button - In windows there is a button on the device  that sets the processors to half speed - this usually allows the fan to shut down allowing me to work in relative quiet.  Alas it doesn't work under ubuntu for me.

Touchpad - even after adding a recommended configuration utility (TouchPad) and making the required change to my x11 config file, I'm still unable to use the touchpad for clicking (actually it seems it does work occassionally at random since links will open when i attempt to scroll the pointer around), the scroll area remains inactive(ie I can't use the side for scrolling documents and webpages up and down), and I can't find a way to increase the touchpad velocity (ie in windows I can scroll the entire viewing area with a single finger stroke across the pad - whereas in Ubuntu the pointer moves about the same distance as my finger did - so it takes a lot of strokes to move from one side of the screen to the other).

I'd also like to request a few items to be installed by default - Gnome SysInfo; NTFS Configuration Tool.

Thanks for the great distribution,

and if anyone has advice to solve the remaining issues I'd appreciate it.

Tom M.
