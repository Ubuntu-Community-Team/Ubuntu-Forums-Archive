I am just writing an review about video card support in linux and especialy the professional video cards like FireGL. Sow i am verry interested if it is possible to get this videocard to work. I have allready done some research to FireGL's and other cards like that. Sow i know that the FireGL GPU is the same as the Radeon 8500. In the recent Ati driver, support for the FireGL 8800 is removed. The latest driver that supports the FireGL 8800 is "ATI Proprietary Linux Driver Version 8.28.8". Sow i think you need to try that one [LINK]("https://a248.e.akamai.net/f/674/9206/0/www2.ati.com/drivers/linux/ati-driver-installer-8.28.8.run").
Besides that you could try the opensource radeon driver that is present in xorg. This is original an radeon driver but the firegl has the same core (R200) as an radeon 8500 witch is supported by the opensource radeon driver.

Would you like to try these options?
