I have a slightly simpler wireless mouse and keyboard set up and it works great with the normal USB interface, it was already install when I set up ubuntu though so I dont know much about configuring it after that if its any different at all, should work on startup without ne drivers, however there is one problem, if you use the grub bootloader to dual boot windows and linux the wirless (at least through the USB) is not supported at that point so if you wish to select any OS aside from your default you will need a normal keyboard handy
