### Post by Sutekh on 2006-02-23
Not a stupid question at all.  

The LiveCD is a prety good indicator of how well your hardware will go with the default installation of Ubuntu.


[QUOTE=helpless]Is this just because I am running it from a live cd or will I encounter the same problems with a full install to the hard disk? I really need to check some files on my hard disk etc.[/QUOTE]
Bottom line is that if you couldn't see your other drives with the LiveCD, you may need to do some configuring when you install Ubuntu.  

You could try do this when using the LiveCD for practice, using the instructions here [Mounting Windows Partitions in Ubuntu - by ]("http://www.psychocats.net/linux/mountwindows.php")[aysiu]("http://www.ubuntuforums.org/member.php?u=21941")

It's not as crazy as it may look.  Basically the link will help you identify the device name of your other hard drives, and show you how to 'mount' them, allowing you access to them.



As for the wireless, I'm afraid this may be more of an issue.  What sort of wireless card is it?  Try searching the forum for the make/model to see if people have configured the same or similar cards in Ubuntu
