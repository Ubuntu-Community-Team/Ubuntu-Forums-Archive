### Post by tturrisi on 2007-05-22
Both!
Some ram, such as pc2700 is backward compatable w/ pc 2100, meaning the ram will run at 2100 if that's what the mb & cpu requires.  But you could,'t use pc 4200 and expect it to run at 2100.  These are usually stated on the ram box or the ram's spec sheet.
