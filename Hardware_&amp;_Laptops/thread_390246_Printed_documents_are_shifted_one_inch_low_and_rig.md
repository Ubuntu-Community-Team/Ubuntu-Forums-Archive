### Post by samichx on 2008-01-19
Sorry if this reply seems a bit late the clx3160 works great. After installing the Samsung Unified Drivers (from the web, never from the disk included) it prints and scans well. Network print works about the same with autodetect behind a router.

Note: Be very careful to install the Unified drivers. Change the sh using a text editor from /bin/sh or /bin/dash to bash when in k/ubuntu. Then use sudo in console, not sudo su, not from the run dialogue. Otherwise some of the files from random programs may call root their owner. This is no a samsung thing so much as a user/Linux complication we should all be familiar with.
