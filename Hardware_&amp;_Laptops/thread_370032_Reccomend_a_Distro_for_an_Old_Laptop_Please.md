### Post by Dylnuge on 2007-02-25
An External CD Drive would work, you could also use a smaller distro like DamnSmallLinux or Gentoo Minimal that would load onto a floppy (beware, Gentoo requires a manual install from the terminal). Ubuntu's LiveCD could be loaded from a external CD Drive as well. I recommend downloading a distro like Xubuntu set to use Xface, or installing Xface at the soonest possible moment.

Linux will be faster then Windows, because there are no "rouge processses." Most Windows systems get bogged down with millions of applications, from IM to Skype to SQL to Spyware, trying to load immediatly. These stay loaded. Linux won't have too many applications unless you install them.

However, be aware that linux is different! You can't use all of your windows apps, you need to find new ones. It looks and feels different

Welcome to linux.
