### Post by bward1 on 2007-01-28
Yea it will.  The amd64 got it's name when amd first came out with the 64 bit desktop processor.  This will now install on any 64 bit desktop processor whether it be amd or intel.  Most intel processors now such as the core 2 duo have EM64T so they too can run in 64 bit.  Though i'm sure intel isn't very happy the linux community still calls it amd64.
