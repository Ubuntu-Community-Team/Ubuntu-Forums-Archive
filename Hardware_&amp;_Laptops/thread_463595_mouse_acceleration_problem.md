### Post by Pru on 2007-06-06
I just found this:
[http://gentoo-wiki.com/HOWTO_Counter-Strike_1.6_With_WINE#Xset_-_remove_mouse_acceleration](http://gentoo-wiki.com/HOWTO_Counter-Strike_1.6_With_WINE#Xset_-_remove_mouse_acceleration)

It says that if you use
```
xset m 0 0
```
it will disable mouse acceleration but it will reset when you log out
