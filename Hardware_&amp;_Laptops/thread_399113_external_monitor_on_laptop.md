### Post by NikoC on 2007-04-02
Hmmz, the shortcut doesn't work on my lappy what does work is the following:
alt+f2 --> type nvidia-settings
hook up the second monitor
goto X-server Display Configuration in the nvidia-settings and do 'Detect displays'
Now both displays should appear
I enabled the second one via TwinView (the option that doesn't require a X-restart)
Then you merely have to select whether for example you want to use them next to each other, or the new one as a clone of your primary screen, etc...
To disable them I just get back into Nvidia-settings and disable the second monitor.

Good luck
