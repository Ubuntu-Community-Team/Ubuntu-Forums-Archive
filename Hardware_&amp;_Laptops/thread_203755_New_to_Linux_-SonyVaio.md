### Post by coffeecat on 2006-06-26
I'm new to this particular forum, but I've been using various flavours of Linux for about a year now. And I'm posting from Ubuntu Dapper running sweetly on my Sony Vaio VGN-FS215B - wirelessly too, and with WPA. 8) A few words of encouragement then and one big cautionary comment.

I have not experienced the fan issue on this particular Vaio. I've not done any proper monitoring, but the fan comes on occasionally seemingly no more and no less than when I run another distro/OS on it. (I've set it up as a quad boot with Fedora 5, SuSE 10.1 and Windows.) The live/install CD detected all my hardware and ran the install well, with the exceptions I mention below. It setup the correct 1280x800 video resolution without any fiddling about on my part. I do encourage you to try out Linux, and Ubuntu in particular. After only a year I now rarely go into Windows, and can foresee a completely Windows-free future.

The cautionary notes. The special Fn+function keys combinations do not work ootb. Windows has special software drivers for them. I found a howto for Breezy but I got multiple errors when attempting to compile one of the scripts in Dapper. I'll be posting a help myself for this soon.

The biggest problem I came across was caused either by the weird partition layout with a hidden first Windows recovery partition, or something to do with the partition table itself. I won't go into details now, but the only way I could get Ubuntu or Fedora to install (but strangely SuSE was OK) was to completely wipe and reformat the whole hd and start again. Not something you want to do if you're unsure about partitioning, but don't let me put you off. Googling around, I seem to be the only person in the whole Universe who's come across this issue. Perhaps it's something to do with this particular model of the Vaio. I don't know. I've seen reports of hassle-free installations onto other models of Vaio.

So, a few questions. Which model Vaio is yours? Which install CD do you have, live or the alternate one? If you've got the live CD, have you tried running it live yet? Do you want to set up a dual-boot with Windows, or just an Ubuntu single-boot?

Oh, and perhaps you could post the partion layout on your Sony, and whether or not you've got Windows partitioning software.
