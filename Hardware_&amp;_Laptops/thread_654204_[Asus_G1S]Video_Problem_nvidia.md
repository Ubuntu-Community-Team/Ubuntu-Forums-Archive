### Post by krakopince on 2008-01-03
This how I did : 

Download the bios file frome asus support website :

[BIOS FILE]("http://dlsvr01.asus.com/pub/ASUS/nb/G1S/G1S300AS.zip")

Save it on my hard disk (C:\) at root

Extract it 

Reboot

Hit F2 at Asus splash screen

In the bios there is a section "Easy flash"

Choose the bios file you previously extract

Go flash ... 

Enjoy ^^

I must admit flashing from Hard Drive is not the best option, I think it is best to do it from an USB Key.

WARNING : Flashing your bios can cause a lot of damage to your system if it is done wrong blablabla, I'm not responsible blablabla ^^
