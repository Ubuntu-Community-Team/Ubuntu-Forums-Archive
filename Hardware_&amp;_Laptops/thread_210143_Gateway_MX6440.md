### Post by gotfoo on 2006-07-06
I just want to let people know that my Gateway MX6440 (see sig for specs)  laptop works with all versions of 'buntu.

I have settled on Xubuntu as my main system and I have added in GNOME and KDE apps and libs as needed.

The monitor is a 15.4 Widescreen and I didn't have to tweak a thing to get it to render correctly.

This [Wireless BCM4318 AirForce Card]("http://www.ubuntuforums.org/showthread.php?t=190177") workes perfectly each time just used the **bcmwl5a.inf** instead.

External USB Hard-Drive and Samsung Media Player YP-T7z connect and work fine.

External USB mouse works. But it did not like my Micro$oft wireless mouse.:rolleyes: 

The extra Function keys work (dim screen adjust volume)

I added on K3B Cd Kreator and the DVD/CD Burner works.

The only issue I've had was with Ubuntu not auto-starting the wifi connection.:confused: 

In short the only thing I had to configure by hand was the wifi card. 

I went through several installations of Ubuntu, Kubuntu and Xubuntu to see which one I liked the best and I got the wifi card working each time.

This laptop was not my choice because it is replacement for my HP ZE4560 which died (see Best Buy Warranty) but so far Ubuntu works the BEST.=D>
