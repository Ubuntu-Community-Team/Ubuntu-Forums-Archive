### Post by K.Mandla on 2006-02-11
You should get the option to boot either Windows or Linux when you restart the machine. If you see a short message labeled GRUB and a three-second timer ticking down, hit the escape key quickly. That will give you options for the boot.

I installed Windows first and Ubuntu second, so I get the GRUB menu by default. Come to think of it, I've never installed it in the other direction; I don't even know what happens in that case. :confused: 

As far as installing your wireless, try these pages. ...

[https://wiki.ubuntu.com/WifiDocs](https://wiki.ubuntu.com/WifiDocs)
[https://wiki.ubuntu.com/WifiDocs/Ndiswrapper?action=show&redirect=HowToSetUpNdiswrapper](https://wiki.ubuntu.com/WifiDocs/Ndiswrapper?action=show&redirect=HowToSetUpNdiswrapper)
[https://wiki.ubuntu.com/NetworkCards](https://wiki.ubuntu.com/NetworkCards)
