### Post by slipperhead on 2008-04-05
i dont know about the latest ubuntu, but i had dapper 6.06 running on a toshiba 320 cdt, 96mb ram (plus swap), so i am assuming it would be the same.

if you can press f6 (i think) and add vga=0x314 (changes the resolution) to the end of the line of boot options this should let you see enough to get the install done

if you get it installed you can then edit the menu list to make this permanent
