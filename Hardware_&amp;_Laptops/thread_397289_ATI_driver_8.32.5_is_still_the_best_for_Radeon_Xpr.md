### Post by scotty64 on 2007-03-30
If you havea laptop with a Radeon Xpress 200M chipset, I can recommend to use the 8.32.5. driver from ATI. This was the first driver that supports the on board memory via sideport extension and it IMHO still the best one to use. The two drivers released after this one only introduced new issues for me and did not enhance the performance or stability:
Issues I encountered:
8.34.8 -> striped screen and sometimes hangs when shutting down X. Certain video applications crashed X.
8.35.5 -> the above issue PLUS: The screen goes blank and flickers when trying to shut down X, although ATI has listed this bug as fixed in the release notes of this driver.
So: Stay with the first driver and enjoy DRI, Video and a clean X shutdown.
