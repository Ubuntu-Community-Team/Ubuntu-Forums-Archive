### Post by mjwood0 on 2006-02-23
Not to question anyone's integrity, but why should we advocate bypassing security measures placed on "public" computers?

While I would be frustrated running windows at school, there are reasons that it's installed and therefore, it seems like you should install Ubuntu on your own computer.  It's a huge security problem for the admins.

Picture this -- you run from the Live CD and mount a FAT32 drive which is usually write / read protected by the MS OS installed.  Since you mounted it under Linux, you could change any system file.  I'm not trying to give ideas here, but from an admin point of view they wouldn't want you doing it even if it WAS for educational purposes.
