### Post by schmolch on 2008-01-14
You have to use "sudo" to run the editor with superuser rights, as in "sudo gedit".

About the onenote files, i would be very suprised if there is a way to read them directly in linux since onenote is only available in windows and MS does not really like giving others access to their formats.

What you can probably do is convert your onenote files to pdf under windows (i just assume onenote can do that, i never used it) and then open them in Linux with xournal.
Xournal is the best notetaking program for linux, not as good as onenote though (just assuming again but its a pretty safe bet since tablet-programs are very rare and new in Linux).
Xournal can open PDFs and you can annotate them, you can not edit them though.

Another program you will find usefull if you decide to stick with linux is cellwriter. Its a onscreen keyboard with handwriting recognition, again not as good as in windows but pretty usefull in my opinion.

Im using a Tablet myself (X60T) so if you have any further questions, just shoot.
