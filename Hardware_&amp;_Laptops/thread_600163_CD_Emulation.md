### Post by MrFSL on 2007-11-02
if you have a network connection you will actually want to remove the CD from being looked for at all since the network files will be more up to date. - You would not want to copy the files on the CD to your disk.

You can remove or adjust Software sources by going to:

System > Administration > Software Sources
