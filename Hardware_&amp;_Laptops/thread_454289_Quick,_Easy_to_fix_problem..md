### Post by Kobalt on 2007-05-25
The configuration of your X server is altered. Reconfigure it this way : press Ctrl+Alt+F1 and log in, to access a command line. From there, enter : ```
sudo dpkg-reconfigure xserver-xorg
```
Answer the questions (leave the default choice when you don't know) and reboot : ```
sudo reboot
```
It should help to get your graphical interface back.
