### Post by rcatman on 2006-10-19
Hey. I had trouble dealing with my webcam in Ubuntu aswell. I'm still new to a lot of the linux universe and I continue to find new walls but luckily these forums help me climb them. 

Check out this [thread]("http://www.ubuntuforums.org/showthread.php?t=143732&highlight=webcam").

VLC is great to record some video and save, but I'd open up synaptic (or whatever package manager you use, i'm assuming you're in gnome) and search 'webcam'. 

I wanted to find a tool to upload an image at an interval. So..I installed webcam but couldn't figure out the config file. Then I persisted and installed camorama and I love it! Its graphical and you can see the webcam live. I'd look around in the repositories to see if theres anything you can find.
