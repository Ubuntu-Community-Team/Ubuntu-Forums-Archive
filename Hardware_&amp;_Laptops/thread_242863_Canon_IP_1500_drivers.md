### Post by DoctorMO on 2006-08-25
You should be able to use the printer from the windows xp machine as a shared printer (assuming your talking about two networked computers, one windows xp the other ubuntu) share the printer on the windows side, and using cups you should be able to configure it to connect to the smb printer (a windows shared printer is over the smb network)

If your talking about the ip1500 connected to the ubuntu machine directly then you will need to either 1) complain to canon about their lack of linux support, 2) help gutenprint make ip1500 printer drivers, 3) pay $40 for turboprint drivers.
