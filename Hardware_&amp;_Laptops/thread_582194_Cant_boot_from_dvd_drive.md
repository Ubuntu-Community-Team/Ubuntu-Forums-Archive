### Post by stapler123 on 2007-10-19
One question first as I'm un familiar with the ubuntu dvd, why can't you just install it from the live cd? That is all I have ever used and thought that ubuntu only came in sizes that fit on cd.

Well it's not the same exactly on every computer.   Generally you are looking for the boot order in the bios, but some of the older computers i've used don't let you choose which of the cd drives to boot from. If messing with that bit doesn't get you anywhere I would probably open up the case and see how it's set up. If the cd drive and the dvd drive are both on one channel, I would put the dvd drive in the cd drives place. Making sure to set the jumper in the back to master.
