### Post by StefanoCole on 2006-02-08
Go to [http://linmodems.org/]("http://linmodems.org/") and download the ScanModem tool. It is a script. Follow the instructions to run it. It generates a file named ModemData.txt. Read it, it contains info about your modem and drivers availability. If you have problems understanding this file post its contents to the forum. Someone will help you understanding it.
Also you can read the following wiki: [https://wiki.ubuntu.com/DialupModemHowto]("https://wiki.ubuntu.com/DialupModemHowto")

Bye, Stefano
