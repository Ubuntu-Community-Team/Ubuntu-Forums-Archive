  I've found that most laptops will activate the TV output at power on if something is connected (not a reboot, but power-on). 

If you want to be able to turn this on and off through software, I'm not sure if there is a frontend to the i810 driver that can be changed on the fly, but if you look at the man page (type "man i810") there is info on setting up a TV as output - see the "Monitor Layout" option....
