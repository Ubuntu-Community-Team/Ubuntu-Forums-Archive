### Post by jsandys on 2006-12-19
Follow the steps in this thread: [http://www.ubuntuforums.org/showthread.php?p=1773624](http://www.ubuntuforums.org/showthread.php?p=1773624)

Instead of 
Screen       1   "Second Screen" RightOf "Main Screen"
you can use
Screen       1   "Second Screen" Relative "Main Screen" 0 0
to get the screens to overlap.

Also see [https://help.ubuntu.com/community/NvidiaTVOut](https://help.ubuntu.com/community/NvidiaTVOut) for info on how to set up the Screen section for the TV, the screen resolution needs to be 640X480.

If you have a Nvidia graphics card use the Nvidia drivers from Ubuntu and use Twinview.
