### Post by xinel on 2004-11-15
Try this:

If in xorg go to xorg.conf and in the device section type in
Option "NvAGP" "0"

reboot

If using xfree86 go to the config, and in the device section type in
Option "NvAGP" "0"

reboot

It turns off AGP but u don't get stalls and crashes

Hope that helps

- XineL
