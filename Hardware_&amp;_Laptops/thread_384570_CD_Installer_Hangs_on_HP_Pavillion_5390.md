### Post by ramjet_1953 on 2007-03-15
When you burnt the iso of the LiveCD, what speed did you burn it at?

Any speed faster than 4 X for ANY iso image is asking for trouble.

Also for burning ANY iso you need to use high quality media. Low cost media is OK for most things but NOT iso's.

If the above is OK, you may need to download and run the Alternate CD. Some systems don't like the graphical interface of the LiveCD.

Don't worry, though, the Alternate CD is still quite intuitive and shouldn't cause you any grief.
