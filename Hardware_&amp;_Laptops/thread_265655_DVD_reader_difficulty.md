### Post by mjpatey on 2006-09-26
I'd suggest if you're uncomfortable setting these things up yourself, just try running Automatix or Easy Ubuntu.  Both of these will install all kinds of media codecs and other extremely useful options automatically.

Go here...

[http://getautomatix.com/wiki/index.php?title=Installation&Itemid=38](http://getautomatix.com/wiki/index.php?title=Installation&Itemid=38)

and look under the "Automatic Automatix Installer" section.  Where it says "Open a terminal," go to your Desktop, and click Applications --> Accessories --> Terminal.  Then type what they say to type, and you're good to go!

Helpful hint:  You can copy and paste the text from the web page into the Terminal.  Select the text line you want in the browser, then CTRL-C.  But in Terminal, paste isn't CTRL-V... it's CTRL-**SHIFT**-V.
