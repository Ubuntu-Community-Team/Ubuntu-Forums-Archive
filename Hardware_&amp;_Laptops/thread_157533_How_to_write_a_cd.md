### Post by taurus on 2006-04-09
There are a few of them that would let you copy files to your CD-R.  However, k3b is probably the best of them out there.  So, use apt-get to install it if you don't already have it on your system...
```

sudo apt-get install k3b

```
Then, it should be in Applications -> Video...

### Post by hajk on 2006-04-09
If you have the standard Ubuntu Gnome desktop, then just inserting a blank CD-R will open a Nautilus window for burning a CD (all very graphical), and a question will pop up about burning an audio or a data CD, etc.
