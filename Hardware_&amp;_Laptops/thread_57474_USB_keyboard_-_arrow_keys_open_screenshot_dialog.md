### Post by kjkrum on 2005-08-16
I just installed Hoary and attached a USB keyboard on a KVM switch.  Pressing insert/home/pgup/pgdn/delete/end or any of the arrow keys brings up the save screenshot dialog.  All I found was this unanswered question from nine months ago describing the exact same problem.

[http://ubuntuforums.org/archive/index.php/t-29114.html](http://ubuntuforums.org/archive/index.php/t-29114.html)

---

### Post by kjkrum on 2005-08-17
I found the solution...

Under Keyboard Shortcuts, screenshot defaults to "Screen" (PrintScreen, I assume).  Clearing it fixes this problem.
