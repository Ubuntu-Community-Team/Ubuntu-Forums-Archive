I've upgraded the memory on my desktop computer, adding a brand new 128MB RAM stick, however, I don't notice any improvement on its performance. What command or utility can I use to verify how much memory is installed on this computer?

---

### Post by Jason_25 on 2006-04-23
I always use 
```

free

```
You should always run memtest before booting into an OS to check stability after a ram upgrade.  It also shows memory bandwidth too.  With that said, 128 MB of ram isn't much of an upgrade.
