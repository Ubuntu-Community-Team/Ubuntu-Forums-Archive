The default Ubuntu disk has a minimum of 192 megs of ram.  I don't know if it gives you a warning if you don't have enough. The alternative install cd will install on computers with less than 192 megs, and I can personally say that 128 megs is no problem, but I don't know how far that will go.  

With so little memory, you will likely see quite a bit of slowness when using most modern window managers.  I recommend installing something like Damn Small Linux ([http://www.damnsmalllinux.org/)](http://www.damnsmalllinux.org/)).  This distro is very fast, looks nice, and has most of the software that you will need.  It ran beautifully out of the box on my old p233 Thinkpad with 80 meg of ram. (with the exception of the cardbus wireless adapter I bought).

If you still don't want to give up on Ubuntu, I might suggest running a memory check from the install disc.

Hope this helps!
