### Post by Monkus on 2006-09-28
Hi, I can give you a quick lowdown.

Ubuntu is what?
Ubuntu is a Linux Distribution (a form of linux based on Debian, another form)

So, what's Kubuntu
Ubuntu with the KDE window manager (read Desktop) instead of the Gnome manager that comes with Ubuntu

Edubuntu?
Ubuntu for Kids

Xubuntu?
Ubuntu with the Xfe window manager (read Desktop) which is much lighter and will run smoother on slower machines.

What's Breezy? What's Dapper?
Breezy and Dapper are different versions (read, 1.0/2.0/3.0 and so on) of Ubuntu. Dapper LTS is 6.06 and the new one coming in October is called Edgy Eft (all named after animals, similar to Mac Tiger/Leopard etc.)

I hope that answers your quesitons. I too am quite the noob but feel my education growing so very quickly with this incredible community here on the forums. Take care and enjoy your experience.

### Post by bswilson on 2006-09-28
**Kubuntu** is a user friendly operating system based on KDE, the K Desktop Environment. With a predictable 6 month release cycle and part of the Ubuntu project, Kubuntu is the GNU/Linux distribution for everyone.

The **Edubuntu** Linux distribution brings the spirit of Ubuntu to schools, through its customised school environment. The current version of Edubuntu is aimed at classroom use, and future versions of Edubuntu will expand to other educational usage, such as university use.

**Xubuntu** is a complete GNU/Linux based system with an Ubuntu base. It's lighter, and more efficient than Ubuntu with GNOME or KDE, since it uses the Xfce Desktop environment, which makes it ideal for old or low-end machines, as well as thin-client networks.

The names **Breezy**, **Dapper**, **Edgy**, etc. are the code names for the releases that are made.  [Read more here]("http://www.ubuntu.com/ubuntu/releases").
