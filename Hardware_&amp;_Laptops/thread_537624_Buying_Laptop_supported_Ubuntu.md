### Post by Alexis Phoenix on 2007-08-29
I wouldn't get a Dell laptop if it has an expresscard slot.  Apparently there is an acpi bug in the bios (on the Inspiron 1501 at least) that means that hotplugging expresscards doesn't work on Linux (you have to boot with the card plugged in instead).  I know this because I have one of these and had to get an expresscard to be able to have a firewire connection (something theyv'e stopped putting in them).  Incidentally, (I've read that) winXP and Vista don't see this bug, since XP doesn't understand expresscards anyway so they "just work" and Vista has some kind of fix.

Also with Dell, the new ones all seem to have ATI graphics cards - there are issues getting 3d desktop effects to work with these - although you can use xgl (supposedly - it doesn't work for me), it takes a lot of extra cpu time.  The ati driver doesn't support direct 3d accelleration in the very new cards, and ATI's driver (fglrx) doesn't support the compiz extensions to X.

I chose Dell partly because they are going to be supporting Linux by supplying laptops with it pre-installed.  Looks like they've got some mileage to go yet.  I wouldn't have another one.  I'd probably get Lenovo (ThinkPad) or a Vaio (cuz they're cute, and the jog-dial is so funky!) if I had any money.

OTOH, if you don't need to use an expresscard, or don't mind having to switch on the laptop with the card in the slot, and don't need firewire, and you're not interested in fancy desktop effects, Dell gives you a heck of a lot of "bang" for your "buck" (even if the keyboard's a bit odd and screen gives bad reflections), in my case a dual core 64bit Turion with 1GB ram, 120GB hard disk (I think) and a battery that gives 7 hours use, for less than £500.
