### Post by Muis24 on 2007-03-05
If you aren't able to boot from USB at all, you can always try to install Ubuntu *in* Windows, using install.exe. Check this for more details: [https://wiki.ubuntu.com/install%2eexe](https://wiki.ubuntu.com/install%2eexe) . There are some useful links on the bottom of the page. **REMEMBER**: install.exe is still in an experimental phase, but it's working already (at least on my laptop ;)), however, it still needs some work.
