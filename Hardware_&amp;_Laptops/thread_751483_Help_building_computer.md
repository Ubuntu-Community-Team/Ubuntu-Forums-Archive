That looks like a pretty good setup, but I would suggest that you go with a case that has a powersupply already on it with fans (that is what I did couple of weeks ago) and it worked out great for me.  Just my opinion, oh and that the same type of RAM I got and they are great but one problem is 32 bit OS only sees up to 3.5 gigs so you might want to go with Win XP 64 bit if possible and Ubuntu 8.04 64 bit if possible.
