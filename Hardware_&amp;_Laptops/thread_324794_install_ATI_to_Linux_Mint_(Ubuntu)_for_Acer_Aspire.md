***_Solution:_***

follow this steps:

[LIST]
[*]sudo apt-get update
[*]sudo apt-get install linux-restricted-modules-$(uname -r)
[*]sudo apt-get install xorg-driver-fglrx
[*]sudo depmod -a
[/LIST]

[LIST]
[*]add in: /etc/modules       the line: fglrx
[/LIST]

[LIST]
[*]sudo ln -s /usr/lib/dri /usr/lib/xorg/modules/dri
[/LIST]
