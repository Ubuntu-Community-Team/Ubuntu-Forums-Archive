### Post by jjordan on 2007-05-08
I have used a very wide variety of external HDs and never had any issues.  The one thing to be aware of is that if the drive has something like a pushbutton to backup, the button probably will not work without substantial effort.  Normally the software provided to actually perform the backup will not run under Linux.  There are many great ways to automate backups under Linux for $free. If price is important it is usually much less expensive to buy an external case seperately and put whatever HD size you want into it. You can sometimes double or triple capacity for the same price.
