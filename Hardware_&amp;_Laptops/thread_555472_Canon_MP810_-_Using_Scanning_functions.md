### Post by Thirsty Ferret on 2007-09-20
Hi everyone

I have recently bought a Canon Pixma MP810 printer / scanner / copier. I'm very pleased with it as it prints really well and (on Windows) the scanning is great.

I managed to install the printer on Ubuntu Feisty using the Pixma iP4200 drivers (as recommended on the OpenPrinting website here - [http://openprinting.org/show_printer.cgi?recnum=Canon-PIXMA_MP810](http://openprinting.org/show_printer.cgi?recnum=Canon-PIXMA_MP810) ), and the printing seems great on Ubuntu, but as yet I've been unable to find a way to use the scanner functions on Ubuntu. It seems it's not supported by SANE, or at least does not appear on this list: [http://www.sane-project.org/sane-mfgs.html#Z-CANON](http://www.sane-project.org/sane-mfgs.html#Z-CANON)

I think this all-in-one works slightly differently to other scanners I've seen, in that it seems you set up the scan on the printer itself, using the controls on the top, then it does the scan and finally transfers the image to the computer (Windows XP at least) at the end. When I tried performing this when on Ubuntu, nothing happened on the computer and the printer stalled before it started scanning while it was looking for a computer to transfer it to.

I've checked all the Canon regional websites (Americas, Europe/Middle East, Asia, Oceania) and none of them seem to carry Linux drivers for this specific printer. I've filed a request for Linux drivers on the Canon site but I don't think one guy asking for them is going to solve much :P This is the Canon Europe download page for the MP810 drivers - [http://software.canon-europe.com/products/0010398.asp](http://software.canon-europe.com/products/0010398.asp)

In summary, I'd like to make use of the scanner functions of the Canon Pixma MP810 in Ubuntu. Thanks in advance for your help! If you need any more info or copy-and-pastes please let me know.

Chris
