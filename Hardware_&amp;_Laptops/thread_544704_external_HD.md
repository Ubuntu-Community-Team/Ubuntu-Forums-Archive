```
### Post by merlinus on 2007-09-06
You might try adding this line to the end of /etc/fstab:

/dev/sdb1 /media/Beast ntfs-3g defaults,locale=en_US.UTF-8,uid=1000,gid=1000,umask=0002 0 2

But do this only if you want it mounted every time you boot ubuntu.

Also, enter

```

id

```to make sure that your uid and gid =1000.  If not, change those parameters accordingly.

Use this to edit the file:

```

gksudo gedit /etc/fstab

```

Then restart.

-merlin
```
