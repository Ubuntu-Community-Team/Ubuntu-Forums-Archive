### Post by Nulifier on 2007-08-29
I tried two things and now it works

put the switch in the left side (where it has a picture of a computer, this took me about an hour to get)

then go to System > Prefrences > Keyboard
click on the layout options tab and under Alt/Win Key behavior select Super is mapped to the Win-keys

if that doesn't work go to the layout tab and choose a more applicable keyboard layout (I am using the logitech internet keyboard layout)

hope this helps
