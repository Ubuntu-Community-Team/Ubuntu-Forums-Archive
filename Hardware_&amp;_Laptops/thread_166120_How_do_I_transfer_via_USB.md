### Post by bernardfrancois on 2006-04-26
Probably this can be done, but what I would do is use the network cards to transfer the files. I suppose both computers have an ethernet card with an RJ-45 connector...

If you don't have a network already, you can use a crossed UTP cable.
If both pc's have Ubuntu, you need to setup a NFS (network file system); if one computer has Windows, you need to setup a SMB share.
You can search the [wiki](http://wiki.ubuntu.com) for more information on how to do this.
