### Post by Tautoa on 2007-09-15
I'm using a Motorola L7, which I've never had a problem with.
I can send ringtones, pictures, videos, games, whatever, through a USB cable, it can use MP3s as ringtones, although they have to be stored on the phone, rather than the MicroSD card, so you might want to think about lowering the bitrate. Also, it doesn't seem to handle variable bitrate MP3s, or any with a bitrate of more than 196kbps. But that's never caused me any problems :)

Only problem is, if you're using the L6 and not getting signal, chances are the L7 will be the same...
