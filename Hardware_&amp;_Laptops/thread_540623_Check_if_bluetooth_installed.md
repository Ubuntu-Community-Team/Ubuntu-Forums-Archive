### Post by CRCarl on 2007-09-02
There is a whole set of bluetooth utilities - 

```
bluez-btsco           bluez-hcidump         bluez-pin
bluez-cups            bluez-pan             bluez-sdp
bluez-firmware        bluez-passkey-gnome   bluez-utils
bluez-gnome           bluez-pcmcia-support  
```

You need bluez-utils to do a basic probe.
