```

```

### Post by taurus on 2007-11-22
You should boot into windows and shut it down cleanly or at least eject your external harddrive cleanly so you won't see that warning again when you try to mount it from Ubuntu.  Otherwise, you can mount it from a terminal with

```
sudo mount -t ntfs-3g /dev/sdb1 /media/sdb1 -o force
ls -la /media/sdb1
```
