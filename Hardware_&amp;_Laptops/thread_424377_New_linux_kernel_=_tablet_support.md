### Post by Zuph on 2007-04-26
Ubuntu on the tablet isn't QUITE there yet, even with this new Kernel.  Reasons:

There's no way to log in without a password, so you have to set up an automatic login.  This isn't terribly secure.

There is no handwriting recognition software out there.

If you don't mind these two things, then the passive pen-active tablet thing works darn near perfectly already.
