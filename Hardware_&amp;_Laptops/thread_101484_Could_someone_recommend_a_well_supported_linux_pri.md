### Post by SickTwist on 2005-12-10
I would like to echo previous sentiments by stating that you will have great success with HP printers. HP printer drivers are included with Ubuntu and it is easy as pie to set up a new printer.

I have personally tested Ubuntu with the following Hewlett-Packard printers:
[LIST]
[*]Deskjet 612C
[*]Deskjet 672C (connected to Windows machine, Ubuntu prints to it via LAN)
[*]Deskjet 940C
[*]Deskjet 3840
[*]OfficeJet Pro 1170Cse
[*]PSC 1310 (connected to Windows machine, Ubuntu prints to it via LAN. I had to turn off "enable bidirectional support" in the printer settings on the Windows box in order for the printer to work correctly with Ubuntu. Specific directions to fix this issue are [here]("http://ubuntuforums.org/showthread.php?t=10582").)
[/LIST]

### Post by disturbed1 on 2005-12-11
These are the printers I've used with Ubuntu (Warty, Breezy, not Hoary yet)

Samsung ML-1740
Epson R200
Canon i960, i850, s820, BJC2200
HP PCS750, 5750, 8100.

The HP's have the best output in Ubuntu for me. The Samsung laser works just as well in Linux as in Windows. I was 100% unsatisfied with the Epson support, even when I compiled the newest versions of Gimp Print. I did manage to get it to print on CDs, but the quality was horrible.

The Canon printers *worked* but I used a paid driver.

My vote would be for just about any HP printer. HP also makes Linux printer drivers, and they even have information on their website.
