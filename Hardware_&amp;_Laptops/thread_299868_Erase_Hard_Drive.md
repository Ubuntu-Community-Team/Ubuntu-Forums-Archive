### Post by testube_babies on 2006-11-14
Ubuntu doesn't take up very much space.  The minimum system requirement is 2GB, and I rarely find myself using more than 10 in any one installation.  If you're really ready to take the dive, then the installation CD gives you an option to completely erase your hard disk and install Ubuntu.

### Post by doobit on 2006-11-14
When you are using the live CD or the alternate install CD (which I recommend) you will be told how big your hard drive is and asked what you want to do with it. Since this appears to be your first time doing it, I recommend just letting the CD install using the entire hard drive. It needs to be at least 4 GB for Ubuntu and can be a little less for Xubuntu. If it's only a 2GB or smaller CD, then you should get a bigger one, or use DamnSmall Linux or Puppy Linux.

### Post by AgenT on 2006-11-14
Here are some official sources and it all depends on what type of install you wish to do. Desktop install is the easiest but requires the most from your hardware in terms of minimum system requirements to install.

Taken from official install page:
> Desktop CD  The desktop CD allows you to try Ubuntu without changing your computer at all, and at your option to install it permanently later. This type of CD is what most people will want to use. You will need at least 192MB of RAM to install from this CD.
Here is the alternate CD install which is more difficult than Desktop install because it asks you questions and makes you select options that some people may not know how to answer - the big advantage is that the system requirements are smaller:
> Alternate install CD  The alternate install CD allows you to perform certain specialist installations of Ubuntu. It provides for the following situations:[LIST]
[*]creating pre-configured OEM systems;
[*]setting up automated deployments;
[*]upgrading from older installations without network access;
[*]LVM and/or RAID partitioning;
[*]installing GRUB to a location other than the Master Boot Record;
[*]installs on systems with less than about 192MB of RAM.[/LIST]More information (for alternate install):
> **Memory and Disk Space Requirements**

                                    You must have at least 32MB of memory and 190MB of hard disk space for a minimal installation of the base system. For the full default installation, you must have at least 128MB of memory and 2GB of hard disk space.
