### Post by heimo on 2005-04-23
To my eye everything looks quite good. There's a small chance that wrong mode is selected, which gives too high refresh rate for your monitor. I'd try something like:

 ```
VertRefresh	70-85
```

You probably don't need that modeline in the Monitor section, but it should be fine.

If that doesn't help you could try (just to find out if it makes difference), using vesa driver instead of ati. If that makes no difference, change it back to ati and drop DefaultDepth to 16 (once again, just to get more information).

This is quite odd. Was this message:
 > display warning that refresh rate is out of its capability... 
monitors own message? It'd suggest that wrong mode was selected. The example abowe limits maximum refresh rate. You can also try setting it to one specific value:
 ```
VertRefresh	75
```

Or:
 ```
VertRefresh	85
```

If you keep the modeline, try changing its name from '1024x768_85.00' to  '1024x768'.

Hopefully this helps.


EDIT: Oh, one more thing, uncomment (for now)
Load    "glx"
to:
#Load    "glx"

and check this thread:
[http://www.ubuntuforums.org/showthread.php?t=22496](http://www.ubuntuforums.org/showthread.php?t=22496)
