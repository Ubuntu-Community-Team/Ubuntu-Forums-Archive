### Post by mcduck on 2005-09-06
[QUOTE=tseliot]
2)And I don't have dual channel RAM (DUAL DDR) . Mine is only DDR. Would this be ok?[/QUOTE]
Dual Channel RAM only means that if you have two sticks of RAM, both the same size, they both are used in separate channels to double the overall speed. It is a feature of the chipset/motherboard, and uses just normal DDR sticks.

So if you buy a mobo supporting this, you should also make sure you buy two sticks of RAM. So instead of buying for example one 1GB stick you should get 2*512MB sticks..
