### Post by chewearn on 2008-01-14
You need to check the motherboard manual, which will usually mention any combination that will or will not work.  As far as I know, most recent mobo, like from the past few years, no longer has the problem when you need to match RAM sticks in pairs.

I have a system with one 512MB and one 1GB, and it works.  You could just stick the RAM in there, check if the BIOS detect all 3 sticks properly (i.e. total memory detect is as expected), then run memtest for a couple of hours.
