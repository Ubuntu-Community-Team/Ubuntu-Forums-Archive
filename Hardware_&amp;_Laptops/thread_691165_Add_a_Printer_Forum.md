### Post by pmshirey on 2008-02-08
Actually, it seems like you have no clue what you are talking about. (No offense.) With UBUNTU 7.10 and higher this is very easy with the add new printer software. (system/administrating/printing/new-printer/) Let me state the easy steps.

[LIST=1]
[*]select connection
[*]Select Manufacturer
[*]Select Model
[*]Enter printer info
[*]Enter Password
[/LIST]

Let me note that after you select the Model, it selects a driver for you. This almost always works unless you are a newbie.:lolflag:
