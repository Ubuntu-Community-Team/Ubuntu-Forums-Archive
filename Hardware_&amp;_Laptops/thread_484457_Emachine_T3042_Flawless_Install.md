I have been running UBUNTU ever since I bought "Linux Magazine" and received the DVD install of 6.06 LTS back in Aug of 2006 (Best $11 I ever spent. I have had 6 successful installs. 

THIS UBUNTU 7.04 installation was launched from the CD iso package burnt in XPPRO with a freeware iso burner.

The system booted right up and had no problems recognizing the on board video which is an Nvidia chipset. 

THE SYSTEM IS STOCK with 256mb of RAM so I didnt think I'd be able to take advantage of and 3D effects like BERYL.

After I ran a chkdsk /f on the drive I resized with Gparted which is included with 7.04 then reboooted and started the install process. I continued to try and engage Nvidia restricted drivers through outside of the terminal because I have had to recover failed graphics configurations through the terminal in the past and it's some-what time consuming. 

After only 4 days with the system up I received all the latest updates. After the latest kernerl update I went into enable "desktop effects" through the menu this time instead of the boxs being greyed out I had the option to enable. I pressed OK and it prompted me to download the correct Nvidia driver and the rest was streamlined.

I posted this because I never added any memory to the system and the on-board Nvidia that came with this e-machine rockis! If you see an old AMD emachine at a garage sale you might want to rescue it and load a nice copy of UBUNTU.

After hours of playing with beyl I refined my repositories and added all my fav internet radio staTIONS TO MY XMMS VIA STREAMCAST.
