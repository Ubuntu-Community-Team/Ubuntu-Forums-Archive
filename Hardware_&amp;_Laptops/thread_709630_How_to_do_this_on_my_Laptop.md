### Post by Fire_Chief on 2008-02-27
You don't need to save any drivers from your Vista system for the Linux system (except *maybe* the network drivers). They are totally different and mostly not compatible. When you install Ubuntu, it will attempt to automatically load drivers for all the hardware it recognizes. After the install finishes, if you have any non-working hardware, then it can be addressed on a per device basis.
