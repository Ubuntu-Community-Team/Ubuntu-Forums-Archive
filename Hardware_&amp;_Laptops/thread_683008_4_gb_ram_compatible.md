### Post by chrisdeckard on 2008-01-30
You most likely are running 32-bit Windows Vista which will not use more than about 3.2GB of RAM.  It's a 32-bit limitation.  I don't know, but it sounds like your Vista experience rating might be different because of the way that Vista and the memory controller are interacting.  My thought is that because you have so much RAM, and Vista can't utilize it, that it might not be running as efficiently.  You may also not have matched memory pairs, which could significantly affect your memory throughput.

BTW, the Vista experience rating is stupid, I wouldn't rely on that.

As for Ubuntu, you should run the 64-bit version to take full advantage of the 4GB of memory.  There is no reason why installing Ubuntu would affect your installation of Windows.
