### Post by adampyre on 2007-06-05
Can you identify the source of the clicking sound? A fan or your HD. I would also attempt to try booting with minimal peripherals, remove any and all cards and boot up with your monitor attached to th eon board adaptor. If you still have any issues, are you sure that your RAM is compatible with the board? On new egg, it looks like this boards RAM specs are: 

Number of Memory Slots  	2×184pin + 2×240pin
Memory Standard 	DDR 400/DDR2 533
Maximum Memory Supported 	2GB

Does your ram match these specs?
