### Post by lazychris2000 on 2008-05-29
I am currently running Hardy on my C840 with almost no problems.  Check my post on tuxmobil for more info [here]("http://www.angelfire.com/wv/xmorgan/C840/C840-hardy.html").  The only problem that I had is installing the nvidia drivers.  As long as you don't install them, you should be fine.  Even suspend/hibernate work out of the box, as well as processor speed scaling (if you have a p4m).
I have an atheros wireless card, which works out of the box.  I have not tested the modem, so I cannot guarantee it will work.  I would definately download a copy of it and try the live cd to make sure everything is going to work that you need before you install it and find out that something or other is not fully supported.
