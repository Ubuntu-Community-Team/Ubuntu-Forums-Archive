### Post by tommcd on 2006-12-22
You can have windows and ubuntu coexist. If the laptop comes with ubuntu preinstalled, however, I don't know if windows can be installed without erasing ubuntu unless there is free space available for that purpose. If you want ubuntu and windows, perhaps that system 76 people can leave a partition available for you to install windows on. Contact them here:
[http://system76.com/information.php?info_id=9&osCsid=2adda1eb75e3bfbf1f8d0d84805b7229](http://system76.com/information.php?info_id=9&osCsid=2adda1eb75e3bfbf1f8d0d84805b7229)
If all else fails, you can install windows, then reinstall ubuntu and have ubuntu partition the hard drive as you want. In any case, if you install windows after installing ubuntu you will have to reinstall grub, either fom the live or alternate CD, or from a super grub disk:
[http://users.bigpond.net.au/hermanzone/p15.htm#Re-install_Grub_with_Live_CD](http://users.bigpond.net.au/hermanzone/p15.htm#Re-install_Grub_with_Live_CD)
[http://users.bigpond.net.au/hermanzone/p15.htm#reinstallgrub](http://users.bigpond.net.au/hermanzone/p15.htm#reinstallgrub)
[http://users.bigpond.net.au/hermanzone/SuperGrubDiskPage.html](http://users.bigpond.net.au/hermanzone/SuperGrubDiskPage.html)
I don't own a system 76 PC, but if I were in the market for a laptop I would probably buy from them because you would know that all your hardware would work in linux.
I assume they are in the USA. Shoot them an email and ask them.
