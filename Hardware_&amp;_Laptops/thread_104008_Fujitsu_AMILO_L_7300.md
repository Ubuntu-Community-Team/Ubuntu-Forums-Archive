### Post by java_java_and_more_java on 2008-04-16
RC stands for Release Candidate and is the version that comes after the beta releases and before the final release, For ubuntu the RC is usually scheduled for release a week before the final version.

If you are not in a hurry to install you could wait for the release version which is on the 24th of April.
