### Post by oldsoundguy on 2008-04-21
On the sound. I got 5.1 digital feed from my Creative card, but it is dodgy at best.  Awaiting a replacement card to verify IF it is the program or the card.  
My experience with ON BOARD sound has been sufficient that the majority of time, If someone  wants reliability and a bit more control, I suggest replacing same with a CARD.  You do NOT have to use Creative .. Turtle Beach makes a great card, also! (they have 3rd party Linux support!)

As to the display, you could try using Envy (a third party program avail: [http://albertomilone.com/nvidia_scripts1.html](http://albertomilone.com/nvidia_scripts1.html)) to install your NVida driver package and see if that changes things.  I have NVidia on all 3 of my Ubuntu boxes and used the program to install on all of them.  
It is driving a super 22" HP wide screen cinema display on the main computer without missing a beat (sharing with a Windows box via a DVI KVM). and driving 18" NEC LCD's on the other two. Also have a KVM and a Windows box on one of those set ups!
