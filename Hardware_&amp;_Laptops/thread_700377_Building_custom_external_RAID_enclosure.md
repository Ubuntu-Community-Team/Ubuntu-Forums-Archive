### Post by edieguez on 2008-02-18
I've been researching external RAID solutions like the SANS DIGITAL MS2UT External RAID Subsystem or the Thermaltake Muse R-Duo RAID but all of them leaving me wanting. So I am considering the problem of building my own external RAID enclosure - this is something I have never done before so I am wondering if anyone has done something similar. Let me describe the general parameters of the features I'd like to have:

1. Like the SANS DIGITAL solution, I want a box with two SATA HDD in a RAID 1 configuration. I'd like the box to support hotswapping HDD to easily rebuild an array after one HDD failure. I'd like the box to have one eSATA port that would allow another computer to connect to the box and access the internal HDDs in a transparent fashion - it should work for Linux and Windows without having to fiddle with drivers.

2. I'd like the box to have 802.11g so I can mount the box on my homework and have it behave like an NAS that my laptops can access over SAMBA as if it were an attached HDD. This part is relatively easy since I've done it before using an old machine.

3. I'd like the box to have an internal DVD burner to allow portable backup of the data stored in the RAID 1 array for off-site backup. This will eventually be upgraded to a Blue-Ray burner once prices come down.

4. I'd like the box to run on Linux but use dedicated hardware RAID - no software solutions. I'm hoping a hardware solution will be transparent to Linux and avoid problems with drivers, etc. An example is the NVIDIA Media Shield RAID hardware that I believe is transparent to Linux and Windows.

Has anyone designed/priced a similar device? For me, the difficult part right now is #1. I do not know how to get an internal HDD to look like an external HDD to another machine. Any help would be appreciated.
