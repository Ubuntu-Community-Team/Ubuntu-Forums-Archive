### Post by Artemis3 on 2007-12-31
[quote="www.xubuntu.org"]To run the Desktop CD (LiveCD + Install CD), **you need 128 MB RAM to run** or 192 MB RAM to install. **The Alternate Install CD only requires you to have 64 MB RAM.**[/quote]

Xubuntu Alternate install will let you install it fine. The 192 requirement is to use the Desktop (livecd) install.

If you think XFCE is too slow, another thing you can do is use the [Minimal CD](https://help.ubuntu.com/community/Installation/MinimalCD) and then install the light enviroment of your choice, such as icewm.
