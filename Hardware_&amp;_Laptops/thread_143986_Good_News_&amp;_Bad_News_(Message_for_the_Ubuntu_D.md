### Post by edal on 2006-03-13
OK, first the good bit. A big thank you to the Ubuntu developers who worked over the weekend after the password bug was found. It's appreciated guys, thanks.

Now, I've been an Ubuntu user for about six months and one thing about this distro is driving me CRAZY!!! When I run my Sony Vaio under MS Windows XP I can get about two hours fifteen minutes battery life, this is with the screen brightness at 50%. Under Ubuntu with the same conditions I get one hour five minutes before the warning, exactly half the battery life that Windows gives me.

This is madness. There must be some way for the Ubuntu developers to sort out the battery life problem. There are countless messages about this on the board but nothing gets done. It would therefore be appreciated if the developers could use the late release of the next version to get a decent battery life under Linux.

Thanks guys, I knew you could do it.
