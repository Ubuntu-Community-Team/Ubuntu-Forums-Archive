### Post by zenkoanlife on 2006-05-11
Sounds like overheating to me. My only suggestion would be to not only make sure all the fans work, but that the air has a nice path through the system, tidy up the cables and such... or maybe upgrade the existing fans or install additional fan(s) if the case allows.
