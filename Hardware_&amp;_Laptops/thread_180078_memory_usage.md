### Post by Bazon on 2006-05-21
AFAIK that's a design question:
Ubuntu nearly always uses nearly all the memory (as I see in my gnome System Monitor applet). 

And that is not bad: What is the use of ununsed memory: Nothing! It's useless! ;)
Therfore it's usefull to use the full amount of memory. :)

(Of course it's an important question how the memory is given to processes, but in most cases you can trust the sysem here...)
