### Post by Fyre2012 on 2007-09-08
I have the same monitor (AL1916W), and here's what I did to fix it:

I adjusted /etc/X11/xorg.conf to have the display section's 24bit depth mode to have 1440x900 as the first available resolution:

 SubSection "Display"
                Depth   24
                Modes           [COLOR="Red"]"1440x900"[/COLOR]     "1280x1024"     "1280x960"      "1152x921"      "1152x864"      "1024x768"      "832x624"       "800x600"     "640x480"
        EndSubSection
EndSection


Once that was done, (and after a restart) I had the problem with the desktop being bigger than the screen. Turns out I just had to press the 'menu' button on the monitor itself, navigated to the "Analog / Digital" selector and switched it from Analog to Digital.  *poof* and like magic it worked =)

I have a Radon 9600 Pro with a DVI output, but I couldn't get the install CD to load up properly when using it the DVI cable (black screen, invalid input), so I had to use the VGA to to the install (which was text based cuz I did Raid 0, but that's another story)

I know it's an older thread, but I figured I'd offer my success story =)

Great monitor!

### Post by TubaTodd on 2007-09-08
I had a similar problem with a 22" Acer monitor not allowing me to use 1680x1050. I spent several hours trying to get it to work. Here is what fixed it. Be sure you have xserver-xorg-video-intel driver installed. Next, change your xorg.conf to read...

```

Section "Device"
	Identifier	"Generic Video Card"
	Driver		"intel"
	BusID		"PCI:0:2:0"
EndSection

Section "Monitor"
	Identifier	"Generic Monitor"
	Option		"DPMS"
EndSection

Section "Screen"
	Identifier	"Default Screen"
	Device		"Generic Video Card"
	Monitor		"Generic Monitor"
	DefaultDepth 24
	SubSection "Display"
		Depth		15
		Modes		"1440x900" "1280x1024" "1152x864" "1024x768" "832x624" "800x600" "720x400" "640x480"
	EndSubSection
	SubSection "Display"
		Depth		16
		Modes		"1440x900" "1280x1024" "1152x864" "1024x768" "832x624" "800x600" "720x400" "640x480"
	EndSubSection
	SubSection "Display"
		Depth		24
		Modes		"1440x900" "1280x1024" "1152x864" "1024x768" "832x624" "800x600" "720x400" "640x480"
	EndSubSection
EndSection
```

Now use the "Screen Resolution" application to make sure the proper resolution is selected. I don't know if this is right or wrong, but I commented out ALL of the other resolutions besides the one I wanted. The Screen Resolution tool still allows me to select other choices though.
