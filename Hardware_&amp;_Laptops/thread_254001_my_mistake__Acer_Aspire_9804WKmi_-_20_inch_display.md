### Post by daller on 2006-09-17
The 20,1 inch screen is no problem!

I'm running 1680x1050 (on a 15,4", though!)

The HD setup depends on what you want!

If I were you, I wouldn't consider dual-booting, but if you really need Windows, then do one of the following: (Based on what you like!)

Option 1:
 - Harddisk1: Windows
 - Harddisk2: Ubuntu

Option 2:
 - Harddisk1: Windows and Ubuntu
 - Harddisk2: Files to be used on both platforms (Format it as ext3 - a linux-partition, and install the ext-drivers in windows!)

Option 3: (my favourite)
 - Harddisk1: Ubuntu
 - Harddisk2: /home
