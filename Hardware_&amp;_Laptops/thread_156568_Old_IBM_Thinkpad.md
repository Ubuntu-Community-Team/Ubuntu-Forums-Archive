### Post by spd106 on 2006-04-07
I have a laptop of a similar vintage running ubuntu. See [here]("https://wiki.ubuntu.com/LaptopTestingTeam/Kapok3300C"). 

You will be rather hard pushed with only 64 MB though. Maybe try finding some cheap replacements on ebay. You gave the name of the wifi router (wrt54g) not the card, have a look [here]("https://wiki.ubuntu.com/HardwareSupportComponentsWirelessNetworkCards") for some support.

Try default install first to see if it works. It may be better to then try the server install and add Xubuntu after as the desktop. As a last thought, I have found dapper alphas to be much more responsive than breezy.
