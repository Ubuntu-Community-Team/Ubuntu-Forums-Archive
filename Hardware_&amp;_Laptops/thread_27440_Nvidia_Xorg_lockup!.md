### Post by alastair on 2005-04-16
There does not appear to be anything obviously wrong in the log (or the conf).

I have seen X lockup with NVidia on some machines though - and it turned out to be AGP. You could :

a) Try upgrading to a newer NVidia driver
b) Check the system logs after a lockup (/var/log/syslog)
c) Try using the "NvAGP" option to change AGP support - built-in, linux driver - or NO AGP.

But the AGP line is just a guess.
