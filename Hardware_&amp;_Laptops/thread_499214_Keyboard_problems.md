Hi,

K-menu->System Settings->select 'Regional & Languages-> and then select 'Keyboard Layout'.

Now you can use the drop down list and select your keyboard model, or if you do not find exact match ..go for the 'Generic 105-key (Intl) PC.

### Post by rafelprohens on 2008-01-08
I've just solved my problem with the keyboard.
Now all is clear and understandable!

I recall what the problem was:
the keys: Left-Right-Up-Down arrows, PgUp, PgDown, Del, Insert, Home and End do not work properly. All of them do the same: they open a new window looking for the snapshot program. But, if I unplug the keyboard for a while and the plug it again, then they work properly!!

The keybooard that I was using was an old one with not an USB plug and, to adapt it to my laptop, I was using an adapter connector.
THIS WAS THE PROBLEM!!! Now, I know it!

How I have arrived to this conclusion? Because I bought a new keyboard with a USB connector to plug it directly to my laptop and then all worked fine!!

I hope this message could help to somebody, sometime.
