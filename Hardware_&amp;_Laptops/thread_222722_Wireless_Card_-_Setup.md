### Post by tturrisi on 2006-07-25
1. run Synaptic. System > Administration > Synaptic
1.a. enable universe repositories via Synaptic Settings Menu > Repositories
2. install linux-restricted-modules for your kernel.  Can be found in Synaptic at Base System Restricted.
3. use net-admin to setup connection.
