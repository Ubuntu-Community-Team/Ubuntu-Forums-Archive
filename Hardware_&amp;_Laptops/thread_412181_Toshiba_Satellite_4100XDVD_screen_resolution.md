### Post by smonteith on 2007-04-23
Yes I found the solution.

Change the default colour depth from 24 to 16.


Here is the section from my configuration:

Section "Screen"
Identifier "Default Screen"
Device "Trident Microsystems Cyber 9525"
Monitor "Generic Monitor"
DefaultDepth 16

The last line above is what I changed.

### Post by fishymark27 on 2007-06-24
You need to edit your xorg.conf file.

Type:

sudo mousepad /etc/X11/xorg.conf 

to get access to it, then scroll down until you see the screen section
I would make a back up of the xorg.conf fil before you make any changes

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup

Then if it all goes pear shaped you can copy it back from the command line.
