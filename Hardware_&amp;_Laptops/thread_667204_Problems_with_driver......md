### Post by jackflap on 2008-01-14
Two ways you can try:

Go to command line and type:

sudo apt-get install xorg-driver-fglrx

Or:

Go to System -> Administration (or Preferences?) -> Synaptic Package Manager

And install the xorg-driver-fglrx package through there
