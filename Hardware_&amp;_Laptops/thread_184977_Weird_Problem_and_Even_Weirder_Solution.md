Hay,

I've been using Dapper Drake since Flight 5 on and off. Every time I tried it I had a funny problem, it would crash randomly, sometimes right after I had loged in, several times a day, without any error messages or hints in the log files. For a while I've wanted to upgrade my RAM from 512 MB to 1GB, so yesterday I found a good deal on RAM, brought it home, installed it, and Dapper has been running smoothly ever since. :D What I'm guessing was the problem was that my RAM had something wrong with it that Breezy could handle but Dapper could not. So, for anyone who's having this problem, I would suggest trying a nother stick of RAM if you have it.
