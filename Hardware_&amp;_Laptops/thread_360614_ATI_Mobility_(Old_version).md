### Post by Benitez on 2007-02-13
[SIZE="3"]Sorry, I'm kinda n00b here also, but I did have some problems with this also so I can provide *some* help...

Firstly, we need to figure out what type of ATI card you have. This command will look at what you have in the PCI bus and then look for what is connected to the VGA section. Type in a terminal window: ```
sudo lspci | grep VGA
```

This should give you something like this (from my laptop):
```
01:00.0 VGA compatible controller: ATI Technologies Inc Radeon Mobility M6 LY

```

Hopefully it is a card that is supported (either by the proprietary drivers or by the FLOSS drivers). Also, type in this command to view your xorg server configuration. This might help us. So type in: ```
sudo vim /etc/X11/xorg.conf
```

We'll go from there. Do you need 3D acceleration support, or is the video just totally *not* working with the laptop?

--Benitez

[SIZE="3"] Hmm... try one more thing... Enter the command:
```
glxinfo | grep rendering
```

If you get an output of something like: **direct rendering: Yes**, then you effectively have 3D capability. 

Looking at your xorg.conf file gives me this however:
```

Section "Screen"
  Identifier "Default Screen"
  Device "ATI Technologies, Inc. Rage Mobility P/M AGP 2x"
  Monitor "Generic Monitor"
  DefaultDepth 24

...

    SubSection "Display"
      Depth 16
      Modes "1024x768"
    EndSubSection
    SubSection "Display"
      Depth 24
      Modes "1024x768"
    EndSubSection
EndSection
```

This reveals that you have 24-bit color. It might be that you need to simply comment out the last subsection that has Depth 24 with hashes (#) and restart X. This will force X to use a lower color bitrate, which may allow you to run 3D graphics a *bit* faster (I'm not too sure).

The reason you need to do this is due to limited video memory. Basically having a larger color depth requires more video RAM, leaving less with which to do fancy things such as 3D. Really the only way to remedy this then would be to get a newer, better laptop :( .
