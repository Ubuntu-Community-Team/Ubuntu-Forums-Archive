### Post by ivansv on 2007-02-06
doesn't make a difference, the bios is the same for both win and linux although vendors of laptops generally ship with win preinstalled and the bios setup corrupts some things in linux, for example acpi enabled corrupts sound in linux so at install time you have to install with acpi=off or you'll have nosound.
most vendors have upgrades for various things in  their machines, i just upgraded the bios on my dell desktop, dell has a great pgrade setup site.
you can fix the bios for linux but its complicated. to fix acpi the DSDT setup has to be fixed and in some cases the kernel has to be recompiled, way beyond the knowledge of most users or their inclination.
