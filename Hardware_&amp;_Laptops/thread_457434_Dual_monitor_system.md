### Post by Tilo on 2007-05-31
I've used two and more monitors on one system a lot - and have run into problems with either
ATI or NVidia cards.    

WIth Nvidia I don't like that they need custom (not open source) drivers, and you need to re-install the driver after you upgrade your kernel -- that is painful every single time.

With ATI, some newer cards have problems with the ATI-drivers.  

My advise is to check that the specific card which you plan to buy is fully supported.
I'd go for a non-Nvidia card if possible, with two DVI connectors (to save your slots).

Here's a completely different thought: you can use Synergy [http://synergy2.sourceforge.org/](http://synergy2.sourceforge.org/)
to use your laptop as a second interface - and use the same mouse/keyboard as your desktop.

no extra cost involved.. 

   Tilo

P.S. I'm using two DELL 2405FPW and one Samsung 214T, connected to two NVIdia Cards.

### Post by Limpan on 2007-11-16
I'd make this post as a note to all that might have the same question as i did.

I've been running my dual monitor system for about 9 months now and have had few to none problems. I only had to set up the correct resolutions with the nVidia-driver-configuration-thingy. It's quite easy and you can have resolutions where only one monitor is used, for example when i play NeverWinter Nights only my left monitor is used and I am then able to play it in full screen mode. Works like a charm and if you are reasonably used to Ubuntu you'll get it to work.

I now use Gutsy and Compiz-Fusion works like a charm.
