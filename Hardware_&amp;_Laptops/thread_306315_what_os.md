### Post by skymt on 2006-11-24
All the *boxen are similar, but most have a few features the others don't. Fluxbox has tabbed windows and a simple panel, Openbox has flexible keybindings, pipe menus, and tons of speed. Blackbox is what most others are based on (Openbox 3 is written from scratch), so it doesn't really have too many distinctions (it's been a while since I've tried it, so feel free to contradict me if something's changed).

If you want to use your laptop to try things and learn, consider installing Gentoo. It will be a challenge, but the install guide is comprehensive and well-written, and the community is generally supportive (if a bit elitist). Even if it doesn't work out for you, I can guarantee you'll have learned something about Linux.
