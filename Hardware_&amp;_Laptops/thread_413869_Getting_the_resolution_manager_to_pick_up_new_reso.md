### Post by heimo on 2007-04-19
Maybe try a custom modeline, mimic one of these:

[http://www.nvnews.net/vbulletin/showthread.php?t=89554](http://www.nvnews.net/vbulletin/showthread.php?t=89554)
```
 Modeline "1360x768" 85.500 1360 1424 1536 1792 768 771 777 795 +Hsync +Vsync
ModeLine     "My768p" 74.2 1216 1353 1393 1646 678 705 710 750 +hsync +vsync

```[http://www.gustavobarbieri.com.br/xorg.conf-wxga_sony_bravia_40](http://www.gustavobarbieri.com.br/xorg.conf-wxga_sony_bravia_40)
```
        Mode    "WXGA_1360x768"      # vfreq 60.015Hz, hfreq 47.712kHz
                DotClock        85.500000
                HTimings        1360 1424 1536 1792
                VTimings        768 771 777 795
                Flags   "+HSync" "+VSync"
        EndMode
```
