### Post by jeffus_il on 2008-03-29
You should have a device section looking something like this:
> Section "Device"
        Identifier      "Intel Corporation 82945G/GZ Integrated Graphics Controller"
        Boardname       "intel"
        Busid           "PCI:0:2:0"
        Driver          "intel"
        Screen  0
EndSection


Try running ```
sudo dpkg-reconfigure -phigh xserver-xorg
```
in a terminal.
