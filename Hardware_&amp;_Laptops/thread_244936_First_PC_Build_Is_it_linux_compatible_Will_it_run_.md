### Post by Belyel on 2006-08-28
I run Ubuntu on my AMD64 machine, and have run both the i386 and AMD64 kernels successfully.  For the true speed you will need to run the AMD64 kernel, but to be quite honest, the AMD64 kernel is a pain in the neck if you are installing and working with a lot of software, as many packages do not come precompiled for AMD64.  That said, the AMD64 support is getting exponentially better and you can find How-tos for most software on these forums.  I don't know how the AMD64 kernel handles dual-core, but I do know that the smp kernel handles them quite well.

For XGL, if you go with the i386 kernel, you can use [this guide]("http://www.ubuntuforums.org/showthread.php?t=148860&highlight=compiz.real%3A+composite+extension") for your nVidia chipset.  The built-in graphics card should be sufficient.  If you go with the 64 bit kernel, use [this guide]("http://ubuntuforums.org/showthread.php?t=131659&highlight=AMD64+Xgl").  Both seem to have good positive feedback.

As for your hardware question:  If the processor of your computer is the brain, the power supply is the heart.  While you can run a body or a computer with a weak heart, sacrafices may need to be made when two or more power hungry components need the same life source.  I would go with a 400W power source that is a "true" 400W, not the cheap PSU built in to the case.  Antec or Enermax make excellent PSUs.

Welcome to the Linux community!
