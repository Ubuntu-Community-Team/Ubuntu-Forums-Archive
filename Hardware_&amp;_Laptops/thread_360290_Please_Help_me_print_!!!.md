### Post by wickstopher on 2007-02-13
Try installing Printers (foomatic GUI printer configuration tool) through the Applications > Add/Remove tool.  It will show up in your Applications > System Tools Menu.  Reconfigure any printer you have added through that.  Worked very well for me with my HP deskjet 3845.
