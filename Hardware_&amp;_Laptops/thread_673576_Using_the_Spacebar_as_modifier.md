I found a product, The [Matias Half Keyboard ]("http://www.matias.ca/halfkeyboard/") and wondered if, in software, it would be simple to either:

a: Use a keyboard map to do this. (change the spacebar to a meta)
or
b: write a driver to do the same thing.

I could see definite advantages of this style, however, I would make one change. make the switch run for the entire keyboard eg. switching the entire keyboard right/left instead of just the left side. One other thing would be able to use it over any keyboard layout. (post assignment?)  I use Dvorak, but could see it being used mainly for Qwerty. I use computers in vehicles (while stopped of course) and could use this more ergonomically than twisting to access the keyboard with both hands.
