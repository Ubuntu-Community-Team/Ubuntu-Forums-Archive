### Post by mertle on 2007-04-04
I also have an ATI r300 and /finally/ got Beryl to work, so I'll try to help where I can. I'm using the open source 'ati' driver with built in AIGLX, but I have also used the 'radeon' driver with the same success.

Firstly, I can already see you have a few things in your xorg.conf that need to be removed. You have two 'Device' sections (one listing 'radeon' as your driver, the other using 'fglrx') and two 'Extensions' sections, both with compositing set to 0.

Secondly, you've installed fglrx, which is a pain in the butt, in my experience, so I stick with the default ati or radeon driver. When you installed fglrx, it overwrote some files which you need to be able to revert back to the default driver, so you will probably need to reinstall the default drivers that comes with Ubuntu for ATI.

I think you can revert to the former drivers by  removing the package "xserver-xorg-video-all", then reinstalling it, but I'm not exactly sure- don't wanna tell ya to do something if it might screw up X. There's guide in the Ubuntu wiki (can't find it at the moment, but I know I saw it) that tells ya how to revert back to your pre-fglrx drivers. Beryl's wiki may have an answer, too, if not here in these forums.
