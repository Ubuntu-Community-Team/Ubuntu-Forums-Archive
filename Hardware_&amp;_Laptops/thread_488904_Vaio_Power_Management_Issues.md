On my Sony Vaio SZ330PB the machine will suspend  if I ask it to by closing the lid, using the quit applet or if set up to do so pressing the power button.   On AC power it will suspend after a time as set in the power applet.  However on battery power the screen will go dark but the machine does not suspend.

The screen will dim automatically when removing the power cord, adjusting the sliders in the power applet or the function keys may be used to adjust brightness as well.  On resume from suspend full brightness is restored no matter what settings are.  However when hibernating the machine on battery power, the reduced brightness level is restored.  The fix that replaces sh with bash in various scripts is in use.

The machineis run using only the Intel 945GM video chip.

I should be thankful that these issues are of limited scope as many around here can't get function keys, suspend or hibernate to work.

Any help on these issues would be appreciated.
