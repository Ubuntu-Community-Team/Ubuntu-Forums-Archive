### Post by Rhubarb on 2007-11-03
It sounds to me like you need to update your BIOS.
As you didn't specify the model (it's an A6000 series), I can't be sure if it'll fix up the boot with usb mouse in problem.

It's worth a try anyway.  Have a look at [http://www.asus.com/index.aspx](http://www.asus.com/index.aspx)
and click on the "download" button there on the page
Enter in "Laptop" for the 1st box, A6000 for the 2nd box, and the laptop model for the 3rd box.
Click on search, then click on the BIOS, it's got a brief description of the BIOS update there.

You may have to install windows (yeah I know it's horrible, but you only need to use it for 1 minute to install the new BIOS).
Alternatively you might be able to get the BIOS image installed in freedos (there's a few howto's of how to do that here in the forums somewhere).

Cheers,
