### Post by Bartender on 2007-11-02
The alternate install would probly be a better choice than the LiveCD version.
I tried installing Ubuntu 7.04 to a PC with 190 MB RAM a couple of days ago.  It took 20 minutes to get the desktop.  Clicked on "Install" and it just sat there for another ten minutes.  Optical drive spinning like crazy but no HDD activity.
Turned it off, grabbed a Xubuntu alternate install CD, installation still a little slow but no problems.  If you go with Xubuntu you'll want to get acquainted with the App Launcher.  Otherwise it's likely your friend may decide that Xubuntu is "too hard".
It was just a coupla days ago - how did I do that anyway?  :)
RC on either panel, choose "Add", choose Launcher, then go to "Applications" in upper left corner and start AppFinder.  Look thru the list.  Let's say you want a shortcut to GIMP.  Find GIMP, drag/drop it to the left-hand side of the App Launcher window (under "New Item") and let it go.  Then use the "minus" sign to get rid of the "New Item" so that only GIMP is there in the left-hand column.  Then hit OK.  
I think that's all you need to make the launcher functional.
This was with Xubuntu 6.10.  Unless things have changed since then creating shortcuts on the desktop is not as easy as one would expect.
EDIT: If you have broadband installing GNOME later is easy.  Go into Synaptic, find gnome-desktop, install.  Then pick your poison by using the "Sessions" option during startup.  But you'll lose the speed advantage from the lighter weight Xfce desktop environment.
