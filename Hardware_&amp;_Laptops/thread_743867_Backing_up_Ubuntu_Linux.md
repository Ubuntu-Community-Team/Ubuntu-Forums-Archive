### Post by chewearn on 2008-04-03
I use the following method to back-up the OS (not data; for data, I have another method):

1. Have two identical (HD1 and HD2), or nearly identical harddisks.
2. Install ubuntu on HD1 and HD2.
3. Use HD1; keep HD2 in storage.
4. Before making a major change (like distribution upgrade), copy partitions from HD1 to HD2, using GParted LiveCD.
5. Then, do the changes on HD2, and keep HD1 in storage.
