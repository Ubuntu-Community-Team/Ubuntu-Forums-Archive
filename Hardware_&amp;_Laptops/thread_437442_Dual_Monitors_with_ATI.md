### Post by Ziox on 2007-05-08
Make a backup of the file:
```
sudo cp /etc/X11/xorg.conf_backup
```

Next, regenerate xorg.conf file
```
sudo dpkg-reconfigure xserver-xorg
```
choose either "ati" or "fglrx" as the driver, depending on which driver you have installed.

Then check the link "Dual Monitors" in my signature, and choose either "Big Desktop" (for "fglrx" driver) or "MergedFB" (for "ati" driver)
