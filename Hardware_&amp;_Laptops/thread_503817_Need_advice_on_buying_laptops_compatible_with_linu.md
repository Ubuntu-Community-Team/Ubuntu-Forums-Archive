### Post by ali4949 on 2007-07-19
If you are looking for high quality in laptops, nothing beats Thinkpads. I don;t mean style and looks, I mean being robust and be able to withstand a lot of abuse. I am using a refurbished Thinkpad T30 for the last 3 years and it was atleast 2 yrs old when i bought it and I dont have any complains. The older Thinkpads have always been very compatible with debian based distros, as i used to have T23 also and it worked fine. I can;t say anything about the built in wifi cards as I never have had one built in, but the ethernet card is detected just fine. be careful of the fingerprint reader as I am not sure if it always works if at all under ubuntu ( although Lenovo is supposed to be shipping Suse certified laptop for the high end business customers soon) so if it works under one distro, more than likely  you can make it work with some tweaking under ubuntu also. As far as the video card goes, i am able to run compiz on a 16MB video card (LOL.. although it runs very slow so i dont enable that feature) so if you have any of the newer card you should be fine.
If you have $2000 budger go with Thinkpad, you cant go wrong

### Post by phenest on 2007-07-19
100% compatible? Kinda takes the fun out of installing Linux. Seriously, I just bought a Philips X56 (Twinhead H12Y): Intel Core 2 Duo 1.83Ghz, 1Gb RAM, Intel 3945ABG wireless, Realtek 8139C ethernet, 56K modem, SATA hdd, dvd-rw drive, synaptics touchpad, 12.1" 1280x800 screen on an Intel GMA945 video integrated (up to 224Mb shared), O2-Micro 4 in 1 card reader, Express card slot, 3 USB 2.0 ports.

The fun I had trying to install ANY distro of Linux! 2 drivers crashed, so I couldn't even start the Live CD. The current 2.6.20...-generic kernel plays havoc on the Core 2 Duo CPU so I'm temporarily using the -386 kernel. I had to use a PIO driver instead of the MIMO driver to get the wired ethernet working. The card reader don't work at all, but give it time and I'm sure the driver will be fixed by someone (if it hasn't already).

And it still works better than having XP or Vista on this laptop!

Not very helpful post this, perhaps, but it does demonstrate that you can get Linux working on almost any machine if you want it to.
