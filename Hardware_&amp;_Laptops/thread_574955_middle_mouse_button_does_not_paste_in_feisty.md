Hi everyone,

Since I upgraded from edgy to feisty, my middle mouse button is completely ignored. It is a very annoying problem since (as I assume just like many other Linux/unix users)  I'm using the middle mouse button paste functionality very often (e.g. in terminals). Also in firefox I use it often to open a link in a new tab. 

Using the left and right buttons simultaneously is not a workaround I prefer. 

After many hours of search I found out that the bug is related to a kernel bug (click [here]("http://https://bugs.launchpad.net/ubuntu/+source/linux-source-2.6.20/+bug/83571"))
I'm using 2.6.20-15-generic and 2.6.20-16-generic (preferably 15, since battery monitoring is also buggy under 16). I wonder if someone already found and tried out a fix for this very annoying and in fact embarrassing problem? Or, if some people have a middle mouse button fully active under both mentioned kernels. I'm really amazed that there are no posts on this topic because it is so annoying. Maybe I'm one of the few exceptions for whom it doesn't work. Anyhow, this is a bug that shouldn't have been there. It was already know many months before the release of feisty and that makes it a little embarrassing, and frustrating as well.
