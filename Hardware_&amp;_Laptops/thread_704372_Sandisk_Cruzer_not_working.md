### Post by ajgreeny on 2008-02-22
It had probably been removed from a Vista system without using the "Remove safely" icon.  This seems to upset the file system on usb drives in Vista, and prevent them from mounting in ubuntu (and probably other linux distros.  If it happens again just use it in Vista again and remove it properly to reset whatever goes wrong if you don't do that.  There is then no need to reformat, or so I have read.
