### Post by Haywood on 2007-10-31
Hi,

I am running Gusty on an IBM T-30.  The log in sound plays fine, as well as the new mail wav that I have set up through Thunderbird.  The issue is when I log out, the log out wav does not play.  This was not an issue when I was running Feisty.  Under Sound Preferences (Devices Tab) Sound Events has ALSA selected and the Music and Movies is set to Autodetect.  Any help would be greatly appreciated.

### Post by Haywood on 2007-12-18
Thanks to all who veiwed this thread...
For those who still may have this problem, I found the answer at:
[http://ubuntuforums.org/showthread.php?t=583452](http://ubuntuforums.org/showthread.php?t=583452)
Install esound through the package manager.

### Post by thebestofall007 on 2007-12-29
i had this problem too. this not only works on IBM T-30, but it also worked on my acer 5610z laptop. i changed all of the sound  settings in the sound preferences menu to the alsa option instead of autodetect.  this cleared the problem up.  thanks for the post!!!! spread the word on this, because I've seen this floating around different forums also.
