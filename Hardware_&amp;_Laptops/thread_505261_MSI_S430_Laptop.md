### Post by Heliix on 2007-07-20
hello,
I'm currently running ubuntu edgy on MSI S271, if that counts ;-)
I ran into several 'problems':
1) the screen resolution was bad after default installation, I have to download and install the correct driver (..ehm.. which one it was? don't remember. ask google like I did that time)
2) when I plugged in the headphones, the sound from master speakers wasn't automatically switched of. this, too, was solved, when I installed a newer (2.6.22-7-generic) kernel
3) acpi makes some problems, I'm not able to suspend to ram/or to disk and I'm not able to turn the fan on manually or to change the temperature at which it is switched on. 
thus, my average working temperature is between 45-50degC even if I only browse the web and read some stuff. 

generally, I can't recommend these laptops, I choosed the MSI 271 because it is damn light&portable (12.1' screen, ~2kg), pretty fast (1GB RAM, 64bit Athlon processor) and still not expensive. well, it was worth that price, but..
