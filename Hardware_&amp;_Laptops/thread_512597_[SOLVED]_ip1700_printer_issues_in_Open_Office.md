### Post by cid222 on 2007-08-04
Hi,

I have the same problem you described where I can print from firefox, but I can't actually print pdfs from Document Viewer. It must be something with the specific application, because when I opened the pdf file through xpdf, it was able to print. So try opening pdfs with a different viewer.
Also, I printed using the lpr command by just typing:

lpr nameoffile.pdf 

On the terminal and that worked as well. I wouldn't doubt it if this worked with other files.
