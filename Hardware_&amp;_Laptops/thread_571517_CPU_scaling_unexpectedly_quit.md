### Post by lusp00ky on 2007-10-11
Remove these lines from your /etc/modules:

```
speedstep-centrino
cpufreq_conservative
cpufreq_ondemand
cpufreq_powersave
cpufreq_stats
cpufreq_userspace
```

And reboot. When the graphical login comes up, press Ctrl + Alt + F1, and login in the text mode, using your username and password, and type:

```
# sudo modprobe acpi-cpufreq
# sudo modprobe <module>
```

For each <module>: cpufreq_conservative, cpufreq_ondemand, cpufreq_powersave, cpufreq_stats, cpufreq_userspace, one by time.

Later, 'exit' the terminal and press Ctrl + Alt + F7, and login in the graphical mode, and see if CPU frequency scaling is working.

If not, open up a terminal and post here the ouput of the following commands:

```
# lsmod
# dmesg

```
