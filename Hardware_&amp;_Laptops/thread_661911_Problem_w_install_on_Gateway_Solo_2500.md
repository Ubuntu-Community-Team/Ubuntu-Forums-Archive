### Post by AfterBerner on 2008-01-08
Howdy,

I am an absolute nube to Xubuntu.  I finally got the Gateway solo 2500 to install Feisty Faun (3-1/2 months ago) and haven't been able to get back to tweak it to my liking, until today.  However, I had a problem right after I got it installed and it has not gone away.  This laptop has 128 MB ram and a Celeron 333 processor.  I'm using a Netgear EA101 usb network adapter to patch into broadband internet access.  The problem I'm having is that the whole computer locks up after a few minutes of downloading anything - web pages, patches to linux, files from online sites.  The computer screen locks up, the mouse refuses to respond as do any of the keys.  At this point all I can do is hold down the power button and restart the system.  I don't think this problem is entirely related to internet downloads, however, as I had it lock up on me when I was trying to type a large document.  The problem is, it happens more frequently when I'm on the internet getting files, surfing pages, or trying to update Xubuntu.  This poor old laptop can still run win98 (I still have it on a separate HD) but it was dog slow.  Xubuntu, as the only os on the HD, is lightning fast on this machine, but is unstable as I've mentioned.  Any and all help that anyone can give me would be so greatly appreciated.

Thanks in advance.

### Post by AfterBerner on 2008-01-17
Well, I tried something drastic and it worked.  I installed Gutsy Gibbon over the Feisty Faun installation and I have NOT had the problem since.  There must have been something changed between Faun and Gibbon.  I hope this helps anyone else with this problem.
