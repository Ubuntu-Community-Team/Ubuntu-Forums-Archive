### Post by s_p_a_r_k_y on 2006-11-19
linux wont let you eject a cd unless it is unmounted. So by right clicking and hitting eject on the desktop you unmount it, and it should be able to eject it by the button then. if the drive is actually damaged and not opening correctly, there is a paperclip sized hole that can be used. Find a paperclip and push one end into that tiny little hole to eject it.
