### Post by alastair on 2005-04-25
Xorg.0.log looks OK, but this :

postfix/master[7537]: fatal: could not find any loopback addresses

from the syslog is suspicious. Loopback is "lo" and you will have problems if it is not present.

Switch to a VT (CTRL+ALT+F1) and login, then :

ifconfig -a

You should have a "lo" device :

lo  Link encap:Local Loopback
....

If not, check /etc/network/interfaces for :

# The loopback network interface
auto lo
iface lo inet loopback
