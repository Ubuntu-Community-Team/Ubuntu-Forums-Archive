### Post by Guspaz on 2008-07-08
Running Kubuntu 8.04, there are no 610 drivers, and Canon's MP150/MP160 drivers are 32-bit only (so don't work).

TurboPrint's drivers for the MP470 are also only available as part of the beta of 2.0 (trial only), and those are also only 32-bit.

I'm installing the previous version of TurboPrint, which only supports the MP460; hopefully that's close enough.

EDIT: Nope, the MP460 driver only prints a certain percentage of the page and then aborts. The percentage printed is dependent on how much data is being sent, so more of a 300DPI page prints than a 600DPI page.

Since I only wanted to print a movie ticket from Cineplex, that was fine; it printed enough of the page so that the ticket came out. Still, if you have an MP470 and a 64-bit processor, you'll apparently need to wait until they release TurboPrint 2.0 in order to get proper print support.
