### Post by snipe122 on 2006-05-25
as far as I know the keys to switch between monitors dont work under ubuntu.

Furthermore I think you need to activate TwinView to see sth. on your beamer or 2nd monitor at all. At least thats the way you do it with nVidia cards if you run the binary nvidia-driver. Try to modify you xorg.conf like this: (but backup before)

Section "Device"
	Identifier	"NVIDIA Corporation NV31M [GeForce FX Go5600]" / keep your line
	Driver		"nvidia" /keep yours as well
	BusID		"PCI:1:0:0" /keep yours
	Option		"UseFBDev"		"true" /keep
	Option		"TwinView"		"true" /add
	Option		"MetaModes" "1024x768,1024x768" /add
	Option 		"TwinViewOrientation" "Clone" /add
EndSection


but be sure to install the nvidia-glx driver first, dont use the nv driver that comes with ubuntu by default... which nvidia-card do you have?

### Post by snipe122 on 2006-05-27
As far as I know you can only clone to one device... whether you have two monitors cloned or a monitor and a tv. For a tv you can add the option "ConnectedMonitor" with the parameters "CRT, TV" for example...

maybe this is useful for you:

[http://download.nvidia.com/XFree86/Linux-x86/1.0-8178/README/appendix-g.html](http://download.nvidia.com/XFree86/Linux-x86/1.0-8178/README/appendix-g.html)

all options are described there.


Maybe you get a black screen because your computer-monitor is the 2nd device now. the console is usually on the first device.
