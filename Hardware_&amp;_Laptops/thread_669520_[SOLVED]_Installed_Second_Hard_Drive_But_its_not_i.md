### Post by logos34 on 2008-01-16
fstab section:
[http://www.psychocats.net/ubuntu/mountwindows#fstab](http://www.psychocats.net/ubuntu/mountwindows#fstab)

try ntfs-config again...toggle the enable write support, because it's suppose to bring up a dialog box where you enter the mountpoint, then it will add a line to fstab.

Or manually create a mountpoint and add an fstab entry like the others.  e.g.:

**sudo mkdir /mnt/sdb1**

**gksudo gedit /etc/fstab**

add this:

> /dev/sdb1 /mnt/sdb1 ntfs-3g defaults,locale=en_GB.UTF-8 0 1

If you want to use the UUID #, find it with either command:

blkid

ls -l /dev/disk/by-uuid
