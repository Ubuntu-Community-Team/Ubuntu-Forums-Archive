I've got a white 2GB Surf and it's great. Only one issue I've had with it though. That version uses a different repository to the 4GB and you can't install ksmserver and kicker from this one, meaning you can't activate the full KDE desktop mode. Which is a pain as, much as I like the default desktop, there are times when only a full desktop will do.

But that's easy to get around. All you need to do is edit your /etc/apt/sources.list and change every reference to 700 to 701. Then run sudo apt-get update, followed by sudo apt-get install ksmserver kicker. Once this is done you can change the repository back. You should now be able to get to the Full Desktop option from the Shutdown menu.

Thought I'd post that here as I spent ages finding how to do that the other day!
