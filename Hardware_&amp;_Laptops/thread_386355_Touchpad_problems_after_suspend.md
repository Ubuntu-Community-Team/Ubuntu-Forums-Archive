### Post by Kenty on 2007-04-25
I'm having the exactly same problem. I wished it would have been solved on Feisty, but it had not. 
I haven't found a solution for this, but found a work around after getting this problem. 

[LIST=1]
[*]System -> Quit
[*]Switch User
[*]Login as the same user again
[/LIST]

This work around is silly, but it makes touchpad fully work again.
