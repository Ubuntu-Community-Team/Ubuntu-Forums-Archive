### Post by davidNYC on 2005-06-18
Hello, all:

Thanks for all your input.  It turns out that the solution is simply not to be lazy, but to actually copile the patch and run it, following exactly the instructions in:


[http://katie-and-rob.org/katie/linux_sony_vaio_tr3.html](http://katie-and-rob.org/katie/linux_sony_vaio_tr3.html) 

I added to the xorg.conf the modeline and the horizontal and vertical timings; now the screen is a gorgeous 1280x1024.

To ensure that this comes up each time, I also copied 1280patch to the /etc/init.d directory, and put a link to it as 021280patch in the rcS.d directory.  The programs there are run every time the system boots, so there is no further need to play with it.

Might a thing like this be included in the default Ubuntu install?

David
