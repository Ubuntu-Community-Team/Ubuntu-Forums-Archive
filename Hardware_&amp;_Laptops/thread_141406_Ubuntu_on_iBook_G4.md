### Post by jwbaker on 2006-03-09
Ubuntu works great on an iBook.  However there are some things you might want to keep in mind.  Linux just barely works with the Airport Extreme.  See [http://bcm43xx.berlios.de/](http://bcm43xx.berlios.de/)  If you want an iBook model that Just Works With Linux (TM), get an iBook G3, with a plain old Airport, instead.  Or if you don't care about the wireless that's also fine.
