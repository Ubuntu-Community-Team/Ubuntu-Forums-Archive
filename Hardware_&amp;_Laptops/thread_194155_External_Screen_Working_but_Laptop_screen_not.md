### Post by ewtesterman@cox.net on 2006-08-16
first

reboot in to recovery mode

next

cp /etc/X11/xorg.conf /etc/X11/xorg.conf-secondary

this will create a copy of the settings for the secondary, you ll find that helpful when you go to add it back later.

now 

dpkg-reconfigure xserver-xorg.conf

you ll need to choose the appropriate drivers for your card, to play it safe I would go with the default recommendations (optimise it later with proprietary drivers just get working for now). You should be fine with defaults all the way through, let it auto-scan your stuff. (Note: I have an ati radeon in a Pavillion, auto-scanning the monitor would make the reconfigure freeze in a black screen, reboot repeat the previous step but do not auto-scan the monitor if that occurs). 

Reboot

shutdown -r now

You should be in business at this point.

It is never recommended to do an install using two monitors at the same time. It seems most installer will only configure x for 1 of them, in your case the secondary. So go find a thread to install your proprietary drivers.
