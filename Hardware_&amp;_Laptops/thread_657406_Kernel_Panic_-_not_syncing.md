### Post by Yellow Pasque on 2008-01-03
Can you explain what you did when you "unmounted the memories"?

> There's also a message on the first screen about "no 80 conductor cable is...This is a message from the BIOS telling you that you're using an old 40-pin IDE cable without ground wires between each conductor. It won't cause errors, but it may slow down your hard disk if it has to keep resending data that's being corrupted. If there are a lot of retransmissions, the system may downgrade the maximum speed of the bus to limit the retransmissions.
