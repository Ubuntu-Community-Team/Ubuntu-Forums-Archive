### Post by Joe_CoT on 2006-11-06
Check with the [Laptop Testing Team](https://wiki.ubuntu.com/LaptopTestingTeam) to see how they're doing. Other than that, if you see a particular one you want, search the forums to see how it's doing (or google in general -- if it runs in gentoo, it'll usually run in Ubuntu).

I for one am using an MS-1013, which works great, but is a little outdated at this point. The MSI whitebooks seem to fare pretty well, as do the Asus ones.

### Post by plexi50 on 2006-11-06
I have been using Dapper on my Dell M140 for over a year. I detected and installed every bit of hardware without issue, power use is excellent (around 3 hours with the standard battery) and wireless (ipw2200) is flawless. The only downside is the Intel 945 graphics, I wish I had Nvidia. They no longer make this model, but I think it is rebranded as 1405 (with the 1505) having a bigger screen. I can't recommned this combo enough.
