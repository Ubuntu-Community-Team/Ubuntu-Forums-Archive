### Post by jellylogix on 2008-06-24
Well Quickplay is a  windows application that was only made to work with windows. As far as i know, there is no easy way to configure what the buttons do, as it is probably hard coded into the bios (knowing that the computer knows what to do when you press the button while the computer is off.) The BIOS is probably coded to understand what the buttons do, including turning the computer on, and then running the Quickplay application when windows has fully launched. If the quickplay application is missing when it goes to run it, it silently returns an error and gives up. Therefore, unless you know how to somehow change the hardcoding in the BIOS to run a seperate application, your gonna have to have 2 dead buttons.
