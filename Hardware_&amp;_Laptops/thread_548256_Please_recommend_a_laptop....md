### Post by rybu on 2007-09-14
If you want to ensure no compatibility problems, be aware there's a bug in the NVIDIA drivers.  It only shows its ugly face if you're using compiz (desktop effects "on" in gnome).  It's an avoidable problem but it'll be a problem until NVIDIA releases a bug fix.  This is planned but the timetable is unclear.  BTW, this should only effect you if you want to use Ubuntu 7.10, or if you significantly modify earlier versions of Ubuntu.

Bug report: [https://bugs.launchpad.net/ubuntu/+source/linux-restricted-modules-2.6.22/+bug/130325](https://bugs.launchpad.net/ubuntu/+source/linux-restricted-modules-2.6.22/+bug/130325)
