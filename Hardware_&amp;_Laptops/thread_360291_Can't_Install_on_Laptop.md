### Post by Benitez on 2007-02-13
[SIZE="3"]I had a similar issue with Xubuntu. To make a long story short, I had Breezy installed, borked the installation sometime later, and then decided to upgrade to Edgy. The reason I mention this is because I used the alternate installer for Breezy and I know it worked; however, Edgy's alternate installed did *not*.

Try this, since this was how I got Edgy to work:

Firstly, you'll need a fast internet connection, since that will make it bearable. Otherwise leave it on overnight and it should work. Simply download the Minimal CD installation from **[here]("https://help.ubuntu.com/community/Installation/MinimalCD")** and then try running it as you would the alternate install. The Minimal CD install will install the bare minimum and then download the rest of the source from the Ubuntu servers.

The *really* nice thing about this is that it will give you the choice of which *buntu install you want: Xubuntu, Ubuntu, Edubuntu, or Kubuntu.

Personally, I'd pick Xubuntu for a nice, fast system or Ubuntu for a full-fledged desktop system or fast laptop. But that's just me :) 

Anyway hope this helps. Please feel free to post up if you get massive epic win with this install or if it is just massive epic fail.

--Benitez
