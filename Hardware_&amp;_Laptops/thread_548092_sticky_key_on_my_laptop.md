### Post by PeterF on 2007-10-10
When running the graphical interface (¨X¨)  the command xmodmap should be able to remap your stuck key, I don't know a way to do it without running "X".
To see how it works lookup the command manual in the terminal with: ```
man xmodmap
```.
A sample to replace the caps-lock can be found [here](http://www.terminally-incoherent.com/blog/2007/08/02/remapping-the-caps-lock-key/).
