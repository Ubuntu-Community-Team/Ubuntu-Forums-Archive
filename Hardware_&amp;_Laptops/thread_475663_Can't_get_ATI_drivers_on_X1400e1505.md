HI Folks,

I've been trying to get my new Dell e1505 running with the *lastest* ATI drivers. I've gone through the install steps outlined here: [http://wiki.cchtml.com/index.php/Ubuntu_Edgy_Installation_Guide#C](http://wiki.cchtml.com/index.php/Ubuntu_Edgy_Installation_Guide#C)

steve@Redstone:~$ sudo depmod -ae
steve@Redstone:~$ sudo echo fglrx >> /etc/modules

bash: /etc/modules: Permission denied


then if I do another fglrxinfo command the same "Mesa GLX Indirect" stuff appears showing that the ATI drivers did not run/load.

I've been through this and various trials with the same results. 

For reference (see below) is a copy of my Xorg.conf file. 

I'm at a loss here and would really like to get Beryl running under Ubuntu and my e1505. The video card should be more and able to handle the 3D stuff (does under Vista just fine). In fact the laptop runs great and I also have the full resolution of 1680 x 1050. Other than the ATI drives WiFi and Bluetooth run fine under Ubuntu 7.04.


Any suggestions would be greatly appreciated.
