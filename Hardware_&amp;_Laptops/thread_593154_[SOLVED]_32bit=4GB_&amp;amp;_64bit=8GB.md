### Post by PmDematagoda on 2007-10-26
Ubuntu 64 bit and other 64 bit OSes can support a very large amount of RAM which includes 8Gb whereas 32 bit OSes support only a maximum of about 4Gb.

So if you used 64 bit you can use all your RAM(Considering that you have 8Gb of RAM). If you used 32 bit OSes then only a maximum of about 4Gb may be registered.

A 32 bit OS can work on 8Gb of RAM without many hitches. The only major problem faced would be the fact that only about 4Gb of the 8Gb of RAM would be registered.
