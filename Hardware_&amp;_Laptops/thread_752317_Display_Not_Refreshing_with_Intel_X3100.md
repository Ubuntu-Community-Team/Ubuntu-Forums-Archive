### Post by prshah on 2008-04-11
> **gavinjb said:**
> 
Section "Device"
	Identifier	"Intel Corporation Mobile GM965/GL960 Integrated Graphics Controller"
	Driver		"intel"
	BusID		"PCI:0:2:0"
EndSection


Your intel installation look fine. Maybe you can add the line ```
Option "UseFBDev"  "true"
``` in the above section, and see if it makes a difference:

Eg>
```

Section "Device"
	Identifier	"Intel Corporation Mobile GM965/GL960 Integrated Graphics Controller"
	Driver		"intel"
	BusID		"PCI:0:2:0"
[color=red]	Option		"UseFBDev"		"true"[/color]
EndSection

```
