### Post by spirilis on 2005-04-10
This is really silly, but have you restarted your machine since you made the changes?

I was struggling with my 2005FPW widescreen monitor for almost an hour until I realized that doing a "Log Out" doesn't actually kill/restart the X server, so I was making changes to xorg.conf and thinking they weren't making any difference when I logged out/logged back in... because I was never actually restarting the X server.
Hitting CTRL-ALT-BACKSPACE (the quick way) or just rebooting the system (the long way) ensures X restarts with the new settings.

If that fails to solve the problem, inspect your /var/log/Xorg.0.log file (X server's logfile) for any clues...

### Post by nad on 2005-04-10
Have you tried lowing your refresh rate to 60Hz to see if you have additional resolutions available at Computer -> System Configuration -> Screen Resolution ?

Everything else looks okay except that Dell lists this monitors' HSync @ 31-80Hz. You may need to change this in the config file. If you do and the only resolution you wish to use is 1600x1200, change the VertRefresh to 60.

Dan M
