### Post by breakaway on 2007-01-28
Launch a terminal, and type

[/code][breakaway@FC6Server ~]$ cat /proc/cpuinfo[/code]

This will generate output similar to:

```

processor       : 0
vendor_id       : GenuineIntel
cpu family      : 15
model           : 1
model name      : Intel(R) Celeron(R) CPU 1.80GHz
stepping        : 3
cpu MHz         : 1792.761
cache size      : 128 KB
fdiv_bug        : no
hlt_bug         : no
f00f_bug        : no
coma_bug        : no
fpu             : yes
fpu_exception   : yes
cpuid level     : 2
wp              : yes
flags           : fpu vme de pse tsc msr pae mce cx8 apic mtrr pge mca cmov pat pse36 clflush dts acpi mmx fxsr sse sse2 ss ht tm up
bogomips        : 3587.59
```

That's how you find out what CPU you have. 

And 

```
[breakaway@FC6Server ~]$ cat /proc/meminfo
```

Will tell you how much memory you have (look for "memtotal")
