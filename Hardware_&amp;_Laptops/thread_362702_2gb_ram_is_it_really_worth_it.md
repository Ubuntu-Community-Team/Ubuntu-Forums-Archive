### Post by ramjet_1953 on 2007-02-15
An easy way to check on your RAM usage is to right-click on either the top or bottom panel and add the Hardware Monitor applet. (Not the Hardware Sensors Monitor).
Then just choose "Memory" as the device you want to monitor, when configuring it.

### Post by ltmon on 2007-02-15
I have 2Gig, running Kubuntu 6.10.

The only times I use more than 1Gig are when running virtual machines (which I do a lot of) and when I'm pulling silly tricks such as mounting /tmp on RAM.

Other times might be running application servers, or very memory hungry apps (Eclipse can get that way with a lot of plugins).

Day to day desktop use will usually never come close to using this amount.
