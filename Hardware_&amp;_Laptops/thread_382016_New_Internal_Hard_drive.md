### Post by matburton on 2007-03-11
Id suggest it may not be appearing because it isn't yet formated and doesn't have any partitions.

If you want to put windows on do it first, It'll save a lot of hassle.

Then use the ubuntu live cd to resize the windows partitions and install a new version of ubuntu.

This might seem stupid but theres method behind the madness!
[http://ubuntuforums.org/showthread.php?t=35087]("http://ubuntuforums.org/showthread.php?t=35087")

You can use this method to tar up your external ubuntu install and plonk it on top of the fresh install.

Hey presto, just remember that you might have to fix grub after! I'm sure the menu.lst will be wrong after.

Hope that helps!
Oh and leave ubuntu on your external drive in case things go wrong ;)
