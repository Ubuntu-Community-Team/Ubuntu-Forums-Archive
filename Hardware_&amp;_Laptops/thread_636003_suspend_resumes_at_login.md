### Post by NilsE on 2007-12-09
Go into the Configuration Editor under system tools and find Apps > gnome-power-manager then select lock and see if use_screensaver_settings is checked.  If so uncheck it and you should be OK.

Or turn off screen saver lock in screen saver preferences
