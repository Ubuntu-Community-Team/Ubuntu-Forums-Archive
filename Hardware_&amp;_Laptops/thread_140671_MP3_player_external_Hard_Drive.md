### Post by welsh_spud on 2006-03-06
Do you want to put music on it? You can use a program called Gnomad2 which I use for my Zen Micro

```
sudo apt-get install gnomad2
```

if you don't like that program you can try one called neutrino

```
sudo apt-get install neutrino
```

As for it being a removable drive, have a look in /media when in Nautilus. It might be hiding in there :)
