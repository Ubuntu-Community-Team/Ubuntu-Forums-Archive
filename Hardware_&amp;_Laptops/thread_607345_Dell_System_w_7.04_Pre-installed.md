### Post by RobertGloverJr on 2007-11-11
I have a Dell Inspiron 1420n which came with Ubuntu 7.04 pre-installed.  It works perfectly.  I destroyed it by installing 7.10 on top of it, but all I had to do to fix it was press the ESC during start-up to bring up a menu which had an option to restore the laptop to the factory settings.  Doing that reinstalled ubuntu 7.04 and everything was perfect again.

  I got every upgrade possible on the 1420n, which I think it a good idea since in a year the difference in price will be forgotten but I'll still have the superior hardware.

   There is no bloatware. It's a standard ubuntu 7.04 installation except that Dell fixed all the problems that a true plain vanilla install would have regarding drivers and such specific to the Inspiron.
