### Post by aysiu on 2007-04-06
That's weird. There's this line in the /etc/fstab: ```
/dev/hdc1 /media/momwindows ntfs-3g defaults,locale=en_US.UTF-8 0 0
``` But the drive does not appear to be mounted. Can you paste these commands in and post back the output? ```
sudo mkdir /media/momwindows
``` If this gives you an error saying the directory already exists, that's okay. Just keep going. ```
sudo mount -t ntfs /dev/hdc1 /media/momwindows -o nls=utf8,umask=000
df -h
cd /media/momwindows
ls
```

### Post by aysiu on 2007-04-06
In the future, a Knoppix CD makes the mounting/recovery process a lot easier (all drives and partitions appear as icons on the desktop you can just click to mount and browse), and whether you use Knoppix or Ubuntu, you do *not* need to install Linux or set up a dual boot to rescue data off Windows. The live CD is sufficient.
