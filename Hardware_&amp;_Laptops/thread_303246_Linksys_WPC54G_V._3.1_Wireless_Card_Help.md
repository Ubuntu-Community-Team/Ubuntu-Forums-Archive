### Post by gustolove on 2006-11-20
if you open a console and type "iwconfig" and then hit enter what does it display?

if it says "no network adapter with wireless extensions" <-- then the card is not detected properly

but if it displays a card then u just need to install an easy to use wireless manager

in the console you would want to type:

sudo apt-get install kwirelessmonitor
