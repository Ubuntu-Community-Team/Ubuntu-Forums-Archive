I've got a 64-bit Dapper system that's been stable for a long time. A recent reboot picked up the 2.6.17-12 generic SMP kernel.  I note that with that kernel USB storage devices are not recognized via any of the interfaces I've got.  If I boot back to 2.6.17-11, they all work fine.

I haven't been messing with the system at all lately in terms of tweaking configs or anything. I can boot back and forth between the two kernels now, and with no other changes the USB flash readers either work (-11) or don't work (-12).
