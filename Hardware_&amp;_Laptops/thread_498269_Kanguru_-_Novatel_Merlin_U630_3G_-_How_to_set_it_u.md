### Post by Carlos Santiago on 2007-07-11
I build a DEB package to setup the Novatel Merlin U630 PCMCIA Card (Wireless UMTS / 3G Modem) to connect to Portuguese ISP Kanguru ([www.kanguru.pt](www.kanguru.pt)). Working on Ubuntu 7.04 (Feisty).

It requires setserial, which is available on install CD.

After install, it requires the restart of udev service (either with /etc/init.d/udev restart, or with a reboot)
