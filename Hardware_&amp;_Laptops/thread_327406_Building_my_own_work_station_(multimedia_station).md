### Post by rejser on 2006-12-31
they often use pipe cooling, like and old air-chilled motorcycle engine. 
This works fine doing what you need, not the best system for playing higher demanding games.
The power-supply is external and need no fans.
Only thing that sounds should be the hard-drive or the dvd.
A shuttle box most often come with motherboard so that you only need to fit a cpu, ram and hard -drive to run it.
