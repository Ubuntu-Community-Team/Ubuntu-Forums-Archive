### Post by jglen490 on 2007-05-22
That model has a P100, or something of that vintage.  If 16MB is the maximum RAM for that machine, it's going to be tough running anything except a really light distro that uses a window manager vice any kind of GUI.

I had a similar IBM laptop, a TP365XD (P120/72MB/3.2GB).  I had to find a BIOS update so that I could install the 64MB RAM module.  Perhaps Toshiba has a similar deal.  If not, you might look at Puppy Linux or Damn Small Linux as a starting point.  Even with that, a sub-1GB drive might present challenges.

But give it a try, you may be pleasantly surprised :o !!
