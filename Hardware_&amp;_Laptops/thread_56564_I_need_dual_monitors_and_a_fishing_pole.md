### Post by invalid on 2005-08-13
Hiya,
 
 If you are using the nvidia driver (which it seems you are) you can take advantage of the builtin TwinView option.

Just make your Device section look like this:

```
Section "Device"
	Identifier	"NVIDIA Corporation NV31 [GeForce FX 5600]"
	Driver		"nvidia"
	BusID		"PCI:1:0:0"
	Option		"TwinView"	"on"
	Option		"MetaModes"	"1280x1024, 1280x1024; 1024x768, 1024x768"
	Option		"SecondMonitorHorizSync"	"30-85"
	Option		"SecondMonitorVertRefresh"	"50-160"
	Option		"TwinViewOrientation"		"RightOf"
EndSection
```

Simply add those last 5 Option lines in, and restart X (or reboot your system).

Before you do this, here is what you should do for backing up your config file:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.orig
```

Now, if something happens to break, type:
```
sudo cp /etc/X11/xorg.conf.orig /etc/X11/xorg.conf
```
and restart X (or reboot your system)

Or you can simply reconfigure your xserver with the command:
```
sudo dpkg-reconfigure xserver-xorg
```
This will create a new xorg.conf file for you.

I hope this helps,
