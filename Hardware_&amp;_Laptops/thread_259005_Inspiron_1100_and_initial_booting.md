Have you found and used the method of burning the image which is 'Burn Image'? This is very different from creating any form of ISO CD. The downloaded file (.iso) is an image of an ISO cd, and it is necessary to apply the image without any adaption, a bit like papering a wall, use the image 'as is'. The facility for this will not usually be one of the usual consumer method options, such as create CD, copy CD, create data cd etc etc. It will often be found in one of the menus, called something like image burn, or similar. Choose a speed of burn wich is small compared to the apparent capability of your machine. If I use the full rated speed, I get image failures (I find half my 'maximum' speed is good),  each byte can be important, and if you have an option to check the burned image accuracy, it is useful to choose that also.

If you have doubts about your CD drive, or your machine, test with a purchased bootable CD, such as your original windows CD or Dell CD etc (?) 

Your machine will need to have its bios set to accept booting from the CD drive first before the hard drive boots. Can be done easily. test if needed - OK as long as machine sucessfully boots from *any* cd.

The options you mention - the 'ISO options' are NOT what you need. Nor is the 'bootable CD' option!
All of theses add something or change something to your file, which is not what you look for.

You need a more advanced sort of option where the CD image you know you have, is burned without question straight to the blank CD, warts and all, whether it happens to be an iso image or not!

'Image burn' is similar to what you are looking for.
