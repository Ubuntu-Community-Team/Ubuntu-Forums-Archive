### Post by Grue on 2005-12-01
I've had Ubuntu installed on my laptop for the last eight months and I really like it. What's surprised me is the bad performance I get. GNOME is unbearable so for a long time I used XFCE4. It ran okay I guess, but still was very slow when opening applications or the likes. I thought it might be an install issue so I did a complete wipe when Breezy came out and installed on a freshly formated partition. 
GNOME was still so slow that I felt I couldn't really use it, so I installed the kubuntu-desktop package. Things ran okay (way better than GNOME for some reason), with Firefox as the exception. 

A description of the laptop is [here](http://www.theiling.de/acer.html).

I know both GNOME and KDE can be a bit sluggish at times, but it's the system in general that seems to be at a crawl.

There aren't any heat problems and all the hardware seems to work as expected. My best guess is that the GFX driver is somehow not working as it should, but I don't know...

A cold-boot takes around 3 - 4 minutes(!) That can't be right.
