### Post by I922sParkCir on 2007-06-04
This solved it for me on a Wacom Graphire4. I did the suspend fix
> 
Every suspend/resume cycle drops the wacom, so create the file /etc/acpi/resume.d/20-setserial.sh and add the following (modify settings for your device if necessary):

 #!/bin/bash
 /bin/setserial /dev/ttyS0 port 0x0200 irq 5 autoconfig

Then, go ahead and run a:

sudo chmod +x /etc/acpi/resume.d/20-setserial.sh


Make sure your xorg.conf  includes:>  Load "wacom"

Like this:

> Section "Module"
	Load	"i2c"
	Load	"bitmap"
	Load	"ddc"
	Load	"dri"
	Load	"extmod"
	Load	"freetype"
	Load	"glx"
	Load	"int10"
	Load	"type1"
	Load	"vbe"
	Load	"wacom"
EndSection

I hope this work for you.
-Jai Ubuntu 7.04
