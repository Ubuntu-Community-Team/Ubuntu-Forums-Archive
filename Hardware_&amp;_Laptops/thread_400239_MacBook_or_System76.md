### Post by alanphil on 2007-04-03
It really comes down to what you want to do with the laptop. Do you need OS X? With the Macbook running OS X, you can install and run many Unix applications. Look at the apple website for additional information (or google). You could also run Parallels and have Ubuntu running at the same time as OS X. You would want to buy lots of RAM if you go this route.

The System 76 machines are supposed to be really great. My friend bought one a few months ago and is very happy with the laptop. If you don't need OS X and specific applications that run on OS X (Photoshop, iPhoto, iTunes, etc), then I think this would be route to take!
