### Post by AgenT on 2005-07-13
While this is an old topic, there is one other solution to the problem of using the ML-1740 laser printer. It is possible to use the ML-1210 drivers (foomatic) that come with Ubuntu by default. No installation is required. This has also been tested to work via print server (lpd).

One quick tip: use the USB connection as there have been reports of drivers not working when using the non-USB method.
