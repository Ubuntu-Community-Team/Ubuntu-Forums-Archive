### Post by K.Mandla on 2006-02-28
Is it possible? Probably yes. It would be tricky, though.

The bottom line is, is your brother going to be happy with it? It sounds like the laptop is in rough shape as it is, and if there's a chance he won't appreciate the time and work it takes to get it going. ...

I think the best first step is to download the Live CD and see if you can get that going. Make sure it will run OK, then see if he likes it. Try the Live CD on another computer if you have to.
