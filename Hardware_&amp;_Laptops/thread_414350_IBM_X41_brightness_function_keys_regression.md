Hi.
I've recently aquired an IBM X41 laptop, and decided to install Ubuntu 6.10 on it. I was surprised to find out that (almost) everything worked as designed. My volume function keys worked, and my LCD brightness keys worked (with the OSD indication). It was good.

I have installed (clean installation) Ubuntu 7.04 on that same laptop, and I'm quite happy, especially with the Fn keys, except that brightness OSD behavior has regressed.
Brightness changes using the Fn keys, either up or down, affect the brightness of the display, but although the OSD status window appears, its status remains unaffected. This happens with and without XGL.

This regression is introduced in 7.04, compared to 6.10
