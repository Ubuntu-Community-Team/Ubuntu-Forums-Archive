### Post by yigal.weinstein on 2006-02-15
Partial answer to my own problem.  

hp-toolbox claims I have no ink in "cartridge 15" the black cartridge.

Put a new one in, and I now have black printing.

However I was able to print in black in XP with the same printer with the "empty" cartridge.

Possible answers,

1. XP defaults to using the color cartridge to print black so that it was capable of printing black documents.

2.  The linux monitor is not as finely tuned as the Windows monintor.  The cartridge is not empty but linux thinks it is and therefore nothing comes out.

These are the only possibilities I can think of.
