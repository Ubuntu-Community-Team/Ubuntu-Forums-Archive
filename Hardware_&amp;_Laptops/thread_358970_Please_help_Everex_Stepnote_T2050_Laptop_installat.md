### Post by tgalati4 on 2007-02-20
According to your link:

"Everex recommends Windows Vista Home Premium"

Maybe the BIOS has a non-windows boot detector.  As a safety feature, of course.

Did you try searching the Everex user forum?  Try booting using the "Alternate Install Disk"  This will take you to a text-based menu system where at least you can see what is going on.  For a quicker solution try Damn Small Linux (50MB), it will show you the boot progress so you can see where the hang is occuring.  The Ubuntu Live CD is on quiet mode so you can't see what is happening.  Looks pretty, but not helpful.

If you do get DSL running it will be Damn Fast.

Remember:  "Everex recommends Windows Vista Home Premium"

Why would you want to change from that (Microsoft paid for) endorsement?
