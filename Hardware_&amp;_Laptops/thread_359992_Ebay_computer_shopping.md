### Post by dannyboy79 on 2007-02-13
i would not buy that system from ebay! if I were you I would buy all the components from tigerdirect.com and build it yourself. you can save money. the only reason for buying a computer is normally for support reasons and it appears that you're not buying the computer with an os so you really don't need support. there is a great barebone system that match what you want already at tigerdirect.com right now for only. then just add the other components. http://www.tigerdirect.com/applications/category/category_slc.asp?page=3&Nav=|c:336|&Sort=3&Recs=10

as far as which motherboard, go here and read what hardware does'nt have problems [https://wiki.ubuntu.com/HardwareSupport](https://wiki.ubuntu.com/HardwareSupport)
 the only real concern is the video, it appears that you want an nvidia, great choice, no support issues there, then the sound, then the ide controller (you don't want the jmicron controller).
