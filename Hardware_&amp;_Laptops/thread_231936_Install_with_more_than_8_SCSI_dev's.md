### Post by the_billness on 2006-08-08
I have noticed during a server install that if you have more than 8 SCSI drives connected 9-N will not be accessable. I have noticed this in Ubuntu as well as Knoppix. So if you've built a RAID with more then 8 SCSI drives connected and the install fails you now might have a reason why. I'm not sure if this is a bug or a limitation of the OS.
