### Post by navilon on 2005-09-01
do you have a blank cd in the drive? if so you must write data to the CD it before you can mount it.

If you want to burn cds, I would suggest either [xcdroast](http://www.xcdroast.org) or [K3b](http://k3b.kd3.org).
Open up the terminal (Gnome->System Tools->Terminal) and type one or both of the commands

```
sudo apt-get install k3b
sudo apt-get install xcdroast
```
