### Post by axcairns on 2007-04-29
I managed to get 1440x900 going on the Via chip so no need for the old Nvidia card.

I came across a howto on these forums which showed how to use the gtf tool to generate modeline entries for the monitor section of xorg.conf. All you do is specify horizontal resolution, vertical resolution and vertical refresh as command line parameters and it generates the right modeline entry. I ran gtf for each of the recommended resolutions in my monitor user guide then pasted them into xorg.conf. I then explicitly referenced these modes in the Modes lines in the Screen section.
