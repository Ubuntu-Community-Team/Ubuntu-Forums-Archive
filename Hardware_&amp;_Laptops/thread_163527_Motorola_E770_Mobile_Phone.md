### Post by zugu on 2006-04-21
I recently aquired a Motorola E770 mobile phone. WindowsXP and Ubuntu both detect it and there is no need for drivers. However, in order to perform any data transfer through the USB interface, I need to run some Motorola software that comes with the telephone, called Motorola Phone Tools. However, this software runs on WinXP *ONLY*. This was a big disappointment for me. I wrote an email to Motorola, telling them how dissatisfied I am with their discriminating politics.

I am interested in some open source Linux-compatible applications that could allow me to make data transfers between my Ubuntu PC and the phone.

Any feedback is welcome.
