### Post by daou on 2006-10-24
That is a difficult question to answer. If the device is just for storing data and the computer recognizes it as memory (like a USB memory stick), you should be able to access and download whatever data is inside. However, if it is some sort of specialized piece of hardware, it might need its own drivers. Tell you what, why don't you download and burn Ubuntu CD (if you don't have one) and run the LiveCD (no installation necessary). Once Ubuntu is running from the LiveCD, you can try plugging the device into a USB port and see whether Ubuntu recognizes it.

If Ubuntu doesn't recognize it, it doesn't necessarily mean it won't be able to do it with a little bit of fiddling. It might be hard, though.
