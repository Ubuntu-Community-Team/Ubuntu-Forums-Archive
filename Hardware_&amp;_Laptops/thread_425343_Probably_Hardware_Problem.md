### Post by jespdj on 2007-04-28
It could be that your CPU or memory is getting too hot. The memory could also be damaged. It's hard to tell exactly...

Make sure your computer is adequately cooled, maybe have a look inside and remove dust etc. (with the computer off ofcourse!), then switch it on with the case open, make sure that all the fans are working. Make sure it stays cool and do the test again. What's the CPU temperature when the computer is working hard?
