### Post by troutrou on 2004-11-25
In the end,  turns out that Ubuntu did miracles with my ATI Rage 128 Pro.

If you do a search in the APPLE /PPC forum, you will find a thread about this card. Turns out that it was commonly used on Apple products !

THe miracle ? Well, Ubuntu activated 3D automatically during the install apparently !

It was already enabled. I tried a 3D game that uses OpenGL ('foobillard', found in Synaptic), and no doubt, 3D is working ! :-)

However this is no passport for successful 3D gaming, since the only 2 games I was really interested in, a couple flight simulators, don't work ! :o(((
