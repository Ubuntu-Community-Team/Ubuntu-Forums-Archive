### Post by jespdj on 2007-12-11
You can install Powertop to see what is using the most power and to get tips for saving power. It's in the Ubuntu repository, so you can install it with
```
sudo apt-get install powertop
```
For more info, see [http://www.lesswatts.org/projects/powertop/](http://www.lesswatts.org/projects/powertop/)
