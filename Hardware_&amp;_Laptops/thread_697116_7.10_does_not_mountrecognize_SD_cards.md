### Post by taurus on 2008-02-14
Is it fat32/vfat?  Try to see if you can mount it from a terminal.

```
sudo mkdir /media/sdd1
sudo mount -t vfat /dev/sdd1 /media/sdd1 -o umask=000
df -h
```

---

### Post by blakdogg on 2008-02-14
Thanks, it worked. The filesystem was vfat.
