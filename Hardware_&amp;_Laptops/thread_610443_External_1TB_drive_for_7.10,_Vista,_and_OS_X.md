### Post by helfrez on 2007-11-12
If it's a NAS is should be distro agnostic by definition..However, I use a HP Mediavault that I got on sale at Compusa one week for $199. It supports up to 1.2TB with GigE. Works great for small office because it supports accessing shares via samba(cifs), nfs, ftp, and via web browser. It also shows up via Upnp as media streaming compatible.

If you have extra pcs laying around, build yourself one, otherwise, there are several consumer nas solutions on the market, buffalotech makes high end units with very high capacities.
