### Post by zilebune on 2007-09-22
> **zeeshan4it said:**
> Ok great and firefox plug in java runtime and flash both were working fine on that?

I'm writing this on an HP nx6325 laptop with AMD Turion 64 X2 running Ubuntu Feisty.
Firefox is running just fine, the same for Thunderbird. I have Sun's JRE installed along with the browser plugin, of course and they are performing flawlessly! The Flash plugin is also running smoothly(I saw these were your big concerns).

Please note that you shouldn't choose your future computer based only on the CPU vendor or architecture. Today's dual core CPUs are quite similar  regarding real-life performance, so looks to me like you are choosing between two brands and doing this you are very influenced by their marketing policy.

My advice (want it or not) is to put these PCs side by side and check their components against the Ubuntu Compatibility list to see if they are supported. This way when you install Ubuntu on the new hardware it will work (mostly) out of the box, smooth and clear. 

I think this is what you are looking for but pardon me if I was wrong.
