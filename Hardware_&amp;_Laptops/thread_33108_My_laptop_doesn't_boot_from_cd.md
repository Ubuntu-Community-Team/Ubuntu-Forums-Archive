### Post by gadgetwizard1 on 2005-05-10
I too have a laptop that won't boot from CD. It's an IBM240.

In fact it has no CD or Floppy device.

There is a section in the Ubuntu forum that explains how to install Ubuntu by copying the files to a windows partition and then using loadlin to install Ubuntu.

However, i have still to find someone who can run the live CD iso from a windows partition instead of the CDrom.

I did manage to put the ISO of the Knoppix Live distro on to the top level (C:) of a windows partion (Fat) and then use a grub loader to either run Windows or to run the ISO as though it was a CDrom.

This works very well and has no effect on the Laptop at all. I also save the Knoppix configuration to the hard drive and configured a /home directory so that i did not need to reconfigure everything each time a loaded the 'live' Knoppix.

I wish i could do this with Ubuntu, but as i say i have yet to find a way to run the 'live' cd from hard disk.
