### Post by heimo on 2005-08-12
Couple points to check: Power saving, you probably don't want the screen to go black after 10 minutes - that setting is probably near screensaver settings.

Signal output: By default most laptops don't feed output signal. To toggle between laptops screen, laptop + output and output signal only modes you use Fn key and key with "screen" symbol on it (two rectangles), its some of the function (F1,F2...) keys.

If you can't test projector now, try to reserve some time before the actual presentation to test that everything works.
