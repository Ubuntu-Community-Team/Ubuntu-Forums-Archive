You should perhaps type also (I am pretty sure this is not you problem but just in case):
ifconfig eth1 up
to enable the eth1 interface (type ifconfig eth1 to see if it is up or down to be sure it is OK).
If you still get the same error messages that means that no dhcp server is running or it has a problem (so this would mean that your Access point does not work properly). If you want test this you can set one up on another computer on your lan and see if "dhclient eth0" works correctly (you perhaps have this working I guess).
