### Post by drogoh on 2005-07-29
You'll notice the biggest hit with something like Firefox.  I run Ubuntu on an old Gateway 333MHz with 192M of PC100 and it was able to handle the default install but I was swapped out nearly 64M with Gnome and Gaim running.  Firefox just put it in the ground.

I switched to Fluxbox from Gnome because Flux isn't as heavy and my memory usage dropped like an anvil.  I'm currently running about 13-25M into swap opposed to 64-128M with a couple of Eterms, beep-media-player, Firefox and Gaim.  Just don't expect it to be as nimble as a machine maybe half as better.
