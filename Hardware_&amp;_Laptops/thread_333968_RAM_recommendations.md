### Post by Asix on 2007-01-08
Getting an additional 256 MB of RAM should be good for you, unless you set up your swap file too small during installation. It should be at least 256 MB (512 MB is enoguh for the majority of systems). So if you made it too small, you could make it a little bit bigger.
When it comes to your intended printer, just take a look at the manufacturer's web site and see if there is a Linux driver for it. Ubuntu Device Database is still not of a tremendous help right now, but as the time goes, it will become more and more useful.
Good luck.
