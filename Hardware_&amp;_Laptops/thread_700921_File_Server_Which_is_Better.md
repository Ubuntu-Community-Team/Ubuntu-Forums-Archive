### Post by Existentialist on 2008-02-19
They will both do the job just fine.  The benefit of linux is that you have a lot more control over what services are running and consuming your resources.  You can run it without any graphics if you would like and therefore it will consume a lot less system resources.  Or even run Xubuntu which runs xfc, a lightweight desktop environment.  Iptables is also useful if you don't have the server behind a router or firewall for filtering mac addresses or ips.  I guess a big thing is whether or not you intend to use the computer for anything else, or if it is purely a server, and what kind of hardware you plan to run it with.
