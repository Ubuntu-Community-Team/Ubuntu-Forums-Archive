### Post by victorgreen on 2007-08-10
There are ati drivers around- theyre not the greatest but they function. 

That wifi card is supported in the latest kernel release I believe. Otherwise, Intel has release drivers but you'd have to compile them yourself. 

Everything else should work just fine. If it uses the intel 3945 wifi card then it uses the chipset that accompanies it which is pretty well supported by now. Im using the Intel 965 chipset (1generation newer) and had to do some serious fiddling to get wifi and sound to work. 

Have fun with Ubuntu on your new laptop!
