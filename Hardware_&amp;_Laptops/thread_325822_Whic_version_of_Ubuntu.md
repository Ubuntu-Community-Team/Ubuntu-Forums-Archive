### Post by fredj on 2006-12-26
The normal installation of Dapper will only detect one processor and if you install the
Dapper SMP kernel it will detect both processors but it will not work with Intel wireless
chipsets and certain Nvidia graphics chips. Install Edgy 6.10 this has an SMP kernel by
default and works better with recent notbooks.
