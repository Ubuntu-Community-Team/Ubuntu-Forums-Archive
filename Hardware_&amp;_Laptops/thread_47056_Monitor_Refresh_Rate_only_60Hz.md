### Post by frodon on 2005-07-07
ok

If you've not made a backup of xorg.conf :
1- Boot on linux
2- when startup is done press Ctrl+Alt+F1 (go to virtual terminal)
3- login
4- type ```
sudo vi /etc/X11/xorg.conf
```
5- with arrows go to the monitor section and put the old parameters : when the cursor is where you want to modify the text press "c" then "w" it will allow you to modify text so modify the parameter to the old value (press "u" to undo wrong modifications). When you've finished to modify xorg.conf press "Esc" to quit editing mode and after ":" then "w" then "q" then "enter" to save and quit (":" then "q" to quit without saving).
6- Reboot your system

if you have a backup of the xorg.conf follow these steps :
1- Boot on linux
2- when startup is done press Ctrl+Alt+F1 (go to virtual terminal)
3- login
4- type ```
sudo cp your_backup_file /etc/X11/xorg.conf
```
5- Reboot your system

Well at this step you've restored your system.

First, **always** always make a backup of your xorg.conf file before modify it : ```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup
```So with a backup file it's really easy to restore your xorg.conf.
