### Post by David Haas on 2005-06-02
Madpilot--I have no expperience with the Epson Stylus C86, but the C82 worked well for me for 3 years before it died. Unfortunately, I don't see a PPD file for it in Ubuntu. The last stylus model listed at /usr/share/cups/model/gimp-print/4.2 is the one for the C84: escp2-c84.ppd.gz. It's easier to set up a printer that's all set to work with Ubuntu Hoary. I can't answer your question about borderless full-bleed printing. Shouldn't the salesperson know about this? In general, I'm more impressed with my new HP Deskjet 5740 than with the Epson Stylus C82, but it may not be a fair comparison, because printers are getting better--and cheaper. 
David
