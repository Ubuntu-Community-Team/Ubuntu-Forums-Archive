### Post by onaka_the_kaka on 2007-05-14
After a lot of sweat and tears I've got TV-out on my HP Omnibook 6100 which features a Mobility M6 LY card. 

The trick for me was to first use this guide for 3d-acceleration and stuff:
[http://ubuntuforums.org/showthread.php?t=246746](http://ubuntuforums.org/showthread.php?t=246746)

I then had to enter /etc/X11/xorg.conf and enable the SWcursor option:
Option		"SWcursor" "on"

After I did this I installed atitvout using the command "sudo aptitude install atitvout"

Now finally I had to first disable screen output(on my laptop that was Fn+F5 and then I run the command:
sudo atitvout -f t

To force s-video output....A bit of work so I hope there's an easier way to do this ..
