### Post by az on 2007-04-19
The desktop version is a live cd which boots a live system from the cd.  The entire OS is running on the disk.  The installation copies the filesystem onto your hard drive.

The alternate installer just is a repository of packages.  It's installer installs each package individually onto the hard drive.


You end up with exactly the same thing.  The alternate installation takes more than twice as long (since it goes through each package one by one and the live cd just plops the whole thing in one shot.)
