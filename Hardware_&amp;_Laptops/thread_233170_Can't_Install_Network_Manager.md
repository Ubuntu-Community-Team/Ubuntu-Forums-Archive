### Post by Jagot on 2006-08-09
Yep - go for the updates first.  That may help.  

When you have done the updates, try updating the repository package information before trying to install, so:

```
sudo aptitude update
sudo aptitude install network-manager-gnome
```

To get your sources.list:

```
sudo gedit /etc/apt/sources.list
```

Then copy and paste the contents of that file here.  Before you do that, try downloading the updates first and doing what I've said.
