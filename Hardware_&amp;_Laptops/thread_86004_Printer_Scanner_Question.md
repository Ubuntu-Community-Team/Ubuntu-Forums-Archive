### Post by Joe_lurker on 2005-11-04
I have a HP OfficeJet 7310 All In One.  It has a build in print-server so it can run on a network.  It works great!  Two sided printing, scanning, I haven't tryed to fax from Ubutnu with it though.

HP has made drivers available with the PTAL project.  You will need to install the hpoj package.
