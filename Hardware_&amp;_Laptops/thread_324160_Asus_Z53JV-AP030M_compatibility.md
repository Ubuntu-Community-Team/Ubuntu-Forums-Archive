### Post by generice_genus on 2006-12-23
A good way to test is to download the installation CD and boot, this provides a usable ubuntu linux environment so you can try before you "buy" i.e. install

 In order to say whether certain things will work people will need to know what particular types of hardware you have e.g. the Make and Model no. particularly for the wi-fi card.

For your nvidia card you can install the driver following the instructions on
[https://help.ubuntu.com/community/BinaryDriverHowto/Nvidia](https://help.ubuntu.com/community/BinaryDriverHowto/Nvidia)

### Post by lothar_m on 2006-12-23
i've bought this laptop a few days ago.

so far heres my experience....

i've setup my system in dual boot (edgy + xp pro) without any problem what so ever. everything went fine.

in edgy i've ran automatix to setup a bunch of stuff (including nvidia drivers).

working stuff...

video card
audio card
wireless (intel card)
compiz (requires heavy tweaking - mine isn't completely functional yet)


now the bad part..... 

i've got no camera (built in - [http://ubuntuforums.org/showthread.php?t=451200](http://ubuntuforums.org/showthread.php?t=451200) - requires kernel 2.6.18 or higher)
i've got no microphone (built-in)
haven't tested memory card reader yet (ricoh model)
sound from headphones seems to require some tweaking with the sound driver (there were sucess cases reported).

so far thats it....

by the way... one last note.... as far as i know the asus Z53J series is merely a rebranded F3JC series. so whatever works with f3jc models will also work with the z53j.
