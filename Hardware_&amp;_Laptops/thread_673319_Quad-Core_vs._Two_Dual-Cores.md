### Post by Tweenk on 2008-01-20
1. Not that many games use multiple cores efficiently, because they are not paralellized (is that a word?). I know that Supreme Commander does that, but I don't know if any other games do. The older ones I'm certain don't.
2. Windows XP won't run on quad core systems, because the standard license is for 2 cores only. You would have to use Vista to play your Windows games, which will harm your performance. I have the impression that dual cores running XP have better game performance than quad cores running Vista, but I must admit that my opinion is highly uninformed (i.e. I have never seen the Aero interface, because I don't know anybody with a machine that could run it).
