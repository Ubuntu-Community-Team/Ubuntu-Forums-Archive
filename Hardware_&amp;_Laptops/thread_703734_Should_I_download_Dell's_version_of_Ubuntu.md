### Post by jespdj on 2008-02-22
I have the standard Ubuntu 7.10 installed on my XPS M1330 (when I bought this laptop there wasn't a Dell-specific version available yet). (See my signature for the exact specifications).

Everything works out-of-the-box with the standard version, except the internal microphone, but I found a fix for that here: [bug #147682](https://bugs.launchpad.net/dell/+bug/147682)

So you don't really need the Dell-specific version. I don't know if there are any extras on the Dell-specific version that are worth it.
