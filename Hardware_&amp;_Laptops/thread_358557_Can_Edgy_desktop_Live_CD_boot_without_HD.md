### Post by h6w on 2007-02-11
Hi,

To answer your title question, yes.  The Live CD boots without touching the hard disk so therefore this sounds like a hardware issue to me.  If XP is having problems, too, then there's likely to be something wrong with the hardware.

Try downloading the Ultimate Boot CD from [www.ultimatebootcd.com](www.ultimatebootcd.com) on another machine, burn it, and wack it into your CD drive.  Run a few of the tests especially a memory test.

Cheers,
Tudor.
