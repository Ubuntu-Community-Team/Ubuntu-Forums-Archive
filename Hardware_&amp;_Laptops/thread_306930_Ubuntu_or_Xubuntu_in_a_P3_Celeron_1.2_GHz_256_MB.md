### Post by CoolHand on 2006-11-25
I think it depends on how your mother is with computers.  If you do the entire config for her either should work.  Personally I prefer Xubuntu.  I think gnome has been too crippled in the attempts to make it easier but for a new user (if your mother is a new user) it may be easier to go with Ubuntu and gnome.  There is a performance gain with Xubuntu but I don't think it is a huge difference so maybe the best advice is to try each and see which you like best, or she likes best.

### Post by motang on 2006-11-25
Well I shall be honest with you, Xubuntu didn't support my hardware (my Voodoo 3000 video card) as much as Ubuntu did.  So for that reason I really don't like it too much but, Xfce is much faster on that system that I tried Xubutu on, and since it is a older laptop I would really recommend that you use the desktop environment that isn't resource intensive.  The faster and smoother the system runs the more you mother is going to like it, if it's slow and sluggish then she is going to ask you to restore it to it's original status (and we wouldn't want that, not really go back to Win98 or Win XP).  I hope this helps you out, as both desktop environments are sufficient for what she is going to be using the laptop for.
