### Post by MrFSL on 2007-11-02
> can i use it to connect two computers and transfer files?
can i use one to connect a printer to?

The network interfaces are networking devices. They can be used to setup networks between other network equipment.

If you would like to connect 3 computers together, using the computer with two network interfaces as a "hub" or server of sorts you can do that with a bit of configuration.  If you have a network printer (one that has a network interface of it's own) you could connect it to one of these network interfaces and print to it.

(I should mention you will need to use crossover cables to connect them together and not standard Ethernet cables.)

One configuration that I was eluding to in my previous post is that of a server. You could connect on network interface to your ISP such as a cable modem, and use the other network interface for a private network of your own. This server could provide services, act as a firewall, monitor traffic, limit access to the internet... and much more.

Hopefully this is much more helpful.
