### Post by kyleabaker on 2007-10-21
Ok, so the problem was something with a bunch of backup copies that were made by something. I deleted:
xorg.conf.1
xorg.conf.2
xorg.conf.3
xorg.conf.4
xorg.conf.5
xorg.conf.6
xorg.conf.7
xorg.conf.8
xorg.conf.9
xorg.conf.10
xorg.conf.11
xorg.conf.failsafe
xorg.conf.failsafe.1
xorg.conf.failsafe.2
xorg.conf.failsafe.3
xorg.conf.failsafe.4
xorg.conf.failsafe.5
xorg.conf.failsafe.6
xorg.conf.failsafe.7

and it suddenly started working after I restarted the x server. Now compiz-fusion is working again. So I guess if anyone has similar problems to mine, try making a backup of the current one that you saved and then deleting the others and restart x.
