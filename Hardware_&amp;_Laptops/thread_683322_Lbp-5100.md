### Post by TyphoonX on 2008-02-01
> I'm wondering if there's any way to make it work without having to restart it every day...

For Kubuntu
```
kdesu kate /etc/rc.local
```
For Ubuntu
```
gksu gedit /etc/rc.local
```

Add this to the file then save and exit
```
/etc/init.d/ccpd start
```
