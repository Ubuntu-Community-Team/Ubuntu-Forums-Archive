### Post by glug101 on 2005-10-10
[QUOTE=matthewlmcclure]For another data point, I see this problem on Fedora Core 3 running kernel-2.6.12-1.1378_FC3.[/QUOTE]

*point*

Ok, since this is happening on multiple versions of linux (particularly since it's happening on radically different versions of linux like RedHat/RPM based Fedora, and on Debian based Ubuntu) that earlier posts were correct.  This definately does NOT look like a silly problem with a particular package but is instead a kernel problem. :(
