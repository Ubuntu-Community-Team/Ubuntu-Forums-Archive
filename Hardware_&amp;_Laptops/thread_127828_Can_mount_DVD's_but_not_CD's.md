### Post by bluevoodoo1 on 2006-02-10
system > preferences > removable drives and media

make sure they're all checked in removable media.

also there's a program called 'powertweak' that let's you control the settings of things like this to a greater degree. I know one of the options is 'check for media' (when inserted a disc into the drive). you could get it with synaptic. Run it at the command line with "gpowertweak"

[http://powertweak.sourceforge.net/](http://powertweak.sourceforge.net/)

Some other thoughts: Is autoplay enabled? DMA? do you have the 'hal' package?
