### Post by tommcd on 2006-11-24
Just a guess, but have a look at the section in xorg.conf about the keyboard in the origonal file:

Section "InputDevice"
	Identifier	"Generic Keyboard"
	Driver		"kbd"
	Option		"CoreKeyboard"
	Option		"XkbRules"	"xorg"
	Option		"XkbModel"	"pc105"
	Option		"XkbLayout"	"gb"
EndSection
----------------------------
and in the new one with high-rez graphics:

Section "InputDevice"
	Identifier	"Generic Keyboard"
	Driver		"kbd"
	Option		"CoreKeyboard"
	Option		"XkbRules"	"xorg"
	Option		"XkbModel"	"pc104"
	Option		"XkbLayout"	"uk"
--------------------------
See how th new one is"pc104" and XkbLayout "uk" and the old is "pc105" and "gb" respectively. Perhaps you could just edit the new keyboard section so it is like the old one that worked. Back up the new xorg.conf first though for safe keeping.
