### Post by n.aggel on 2007-08-19
THANK YOU!!!! for anyone with the same problem::

Goto Applications -> Sound and Video -> Volume Control and unmute the side speakers!!!

{for reason i don't know the speakers are consedered to be side speakers.....

PS:: if you want to use the laptop shortcut keys to control the volume then go to volume control applet {next to date and time} right click, select preferences and then select side speakers to be tracked for control!
