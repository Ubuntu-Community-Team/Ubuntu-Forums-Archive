### Post by sdennie on 2008-03-18
The Dell XPS m1330 is a great linux laptop.  They don't offer all options with the pre-loaded Ubuntu model so I ended up buying one with Vista (though, the first time I booted the machine the Ubuntu 7.10 disk was in the CD drive so, the machine wasn't subjected to Vista).  Depending on how you configure a m1330, it can be a bit pricey but, it's very light, performance is excellent and battery life is fantastic.  It essentially "just works" out of the box but, you can tweak it a bit if you are so inclined and dramatically increase battery life and performance while on AC power.  I actually don't think I'll ever buy a machine from anyone but Dell after seeing how well everything works with linux on this machine.

Strabes, he's already bought the laptop, and the Inspiron is a 64-bit AMD with Intel graphics.  You're a little late!  :)

However, I have this exact notebook and its working fine with only minor tweaking.  My wifi works great after enabling the restricted drivers for it (it prompted me to do this the first time I booted Ubuntu 7.10.)  The only problem I have with wifi is that I never get a full signal (usually around 70-80%), but this is known issue with broadcom wifi cards.

All the other hardware (touchpad, webcam, etc.) seem to work perfectly.  I had some trouble getting Visual Effects to work, but I was able to get it working fine with 

```
sudo apt-get install xserver-xgl
```

Wobbly windows, 3D cube, etc. work great now!

I don't know much about configuring wifi, so you may want to do a new post on the networking forum.  However, I can verify that wifi will work fine on the 1526.  I just boot it up and boom, it recognizes the card, connects to the network and runs fine.

One last piece of advice:  When you get it all working correctly, take good notes and back up your files so you want have to go through all the trouble if and when you have to reformat or upgrade to the next version.
