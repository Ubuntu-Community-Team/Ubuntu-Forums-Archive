### Post by Mack1 on 2006-09-19
Sounds like a reading problem in the drive itself, not a bios problem or whatever.

I had these problems with a badly burned kubuntu-install-cd.
Once i used another cd-drive problems were all gone.

You might want to try that?

I wouldn't bet on it being a bios problem.

You might try to calculate the md5 code, see if it is the correct one. if it is and you still cannot get the drive to read the cd correctly, try another cd drive.

good luck!
