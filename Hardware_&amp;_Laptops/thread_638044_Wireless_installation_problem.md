### Post by dsauxier on 2007-12-11
Have you tried the Restricted Drivers Manager (assuming you're on Gutsy Gibbon 7.10)?

The broadcom drivers are supposed to be there, but for full functionality you need to enable it through the Restricted Drivers Manager.  It will probably require you to download additional software in order to do this, so you'll need to have a hard-wired connection while it does that.
