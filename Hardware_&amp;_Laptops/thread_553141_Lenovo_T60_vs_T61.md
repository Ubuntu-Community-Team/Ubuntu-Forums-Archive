### Post by rybu on 2007-09-17
I have a T61p 6457-5KU.  So far pretty much everything works.  The only real exception is that the NVIDIA driver for our cards has a bug in it.  The bug isn't terribly bad in that it seems to only show itself when you use a 3d-accellerated app together with Compiz (the "desktop effects" in Gnome).  Turning desktop effects off has been the best solution so far.  NVIDIA says the next release of the driver should have a fix for this problem.

This wiki contains a distilled version of everything I know on how to set up Ubuntu on this machine:

[http://www.thinkwiki.org/wiki/Installing_Ubuntu_7.10_%28Gutsy_Gibbon%29_Tribe_5_on_a_ThinkPad_T61#Video](http://www.thinkwiki.org/wiki/Installing_Ubuntu_7.10_%28Gutsy_Gibbon%29_Tribe_5_on_a_ThinkPad_T61#Video)

I'm pretty happy with my setup.  Let me know if you need to know anymore more specific.
