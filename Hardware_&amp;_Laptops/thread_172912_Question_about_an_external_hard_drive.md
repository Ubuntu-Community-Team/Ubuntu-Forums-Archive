### Post by ajgreeny on 2006-05-09
How is the external drive formatted?  If it's NTFS it will only be read and not write as linux can not (yet) write to NTFS partitions.  If the drive is empty and has no needed files on it ,or can be backed up somewhere else temporarily, you could reformat it to fat32 and it will be read and write in both windows and linux.
