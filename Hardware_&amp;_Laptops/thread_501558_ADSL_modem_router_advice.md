### Post by coffeecat on 2007-07-15
**Any** adsl modem router should work with **any** operating system. The OS has nothing to do with the modem router. If you find one that stipulates Windows, this is probably referring to a configuration wizard software that is used as a simple way of setting up the device. So long as you get a modem router that is configurable via a browser interface, you can set it up using Windows, Linux or MacOS.

I doubt whether you'll be able to find a one-port router. It would be pointless if the router gave you NAT and NAT is a useful part of your security.

[Here's a good selection]("http://www.ebuyer.com/customer/search/index.html?rb=0&strSearch=&bolShowAll=true&intStoreID=8&intCatID=109&intSubcatUID=1146&bolShowAll=true") of wired modem routers. If you are thinking of getting wireless, you'll be able to find Ebuyer's selection easily enough.

Any of those should work with Linux. The only possible problem I can think of is if one comes with old firmware that can't cope with IPv6 addressing. Search the forums for IPv6 if you haven't heard of this. It should be less of a problem now that Vista comes with IPv6 enabled by default.

**Edit:** Just to add - personally I would avoid the D-Link devices. I'd go for a Netgear, Linksys or Zoom if I was buying.
