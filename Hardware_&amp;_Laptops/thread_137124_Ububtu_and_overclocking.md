### Post by flyingbrass on 2006-02-27
[QUOTE=WildTangent]I have my 2.4 ghz Celeron overclocked to 2.8 ghz, and it seems stable as a rock. The only way for you to find out is to try it. However, you will void your warranty, so as long as you're prepared to do that, then go ahead.[/QUOTE]
I've also been running a 2.4 Celeron at 2.8 for well over a year with no problems.  A few adjustments were required to make it 100% stable.  In my case, bumping up the memory voltage a notch was the key.  I also experimented with other voltages and cooling.

If you want an OC'd box to be rock solid it is important to do some testing.  In Windows I used the most heat-generating torture test in Prime95.  If that will run for 24 hours without crashing, you're pretty stable.  I don't know if Prime95 is available for Linux.  Memtest86 is, which is good for testing memory.
