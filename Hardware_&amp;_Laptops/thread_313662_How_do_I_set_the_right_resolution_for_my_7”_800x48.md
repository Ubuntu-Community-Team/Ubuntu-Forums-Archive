### Post by an.echte.trilingue on 2006-12-06
I am assuming this is your only screen, the command you need is 

```
sudo dpkg-reconfigure xserver-xorg
```

It will run a wizard of sorts that will walk you through configuring your screen as needed.  You should not have to edit config files by hand unless you are trying to set up a dual-head (multiple monitor) displays.

As a note to help you find better documentation in the future: in linux, generally any howtos more than 2 years old are dated.  The tutorial you linked to is from 1998, and linux has come a long way since then.  Ubuntu does not use XFree86 but rather a fork of XFree86 called xorg since XFree86 changed their license to something not compatible with the GPL (and they are now in the process of dying off entirely).

Good Luck!
-mat

EDIT:
> Also can I get my display broken if I use the wrong settings?

** YES! Be Careful**
