### Post by Jonathan Métillon on 2007-01-18
Thank you MTzeon! as I said before you answer, I disabled dri and glx:

```
Section "Module"
    Load    "i2c"
    Load    "bitmap"
    Load    "ddc"
#    Load    "dri"
    Load    "extmod"
    Load    "freetype"
#    Load    "glx"
    Load    "int10"
    Load    "type1"
    Load    "vbe"
EndSection
```

And now I can launch OpenOffice applications. But I have no screensaver and my displays will keep displaying all night long. Any fix or workaround, like energy saving mode that can replace the screensaver?

Also, as I said, I had a badalloc error but I fixed that by increasing video memory, as you can read [here]("http://ubuntuforums.org/showthread.php?t=297507").

```
Section "Device"
    ...
    VideoRam    262144
    Option          "CacheLines" "8192"
    ...
EndSection
```
