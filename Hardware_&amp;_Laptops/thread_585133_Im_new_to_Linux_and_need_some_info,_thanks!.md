### Post by hauger on 2007-10-21
Wow, that's a large, large question.  I've been using Linux on and off for years now, and once even made the switch completely for a while before converting back to a dual-boot setup.

As for hardware compatibility problems, I've found most of the time that any compatibility problems usually only occur with the small "obscure" hardware devices, such as web cams and wireless.  Right now I'm having a sound issue, but usually things can be resolved with some detective work and help from the community.

File compatibility seems to be to be quite high.  No real problems.

The best way to learn about linux is to download and burn a "live" CD, throw it into your drive, boot up to the CD (the OS will run right off the CD, abeit slower), and tool around with stuff.  See if your hardware works.

Enjoy.
Rob.
