### Post by tonym on 2005-07-02
You shouldn't need to reinstall - at boot time the hardware is detected and appropriate drivers loaded so the new hardware should be handled automatically.  There might be problems with some bits - I've had problems with on-board network devices.  Try everything out and only start worrying if something doesn't work - even then its probably only a case of installing a driver rather than a rebuild.
