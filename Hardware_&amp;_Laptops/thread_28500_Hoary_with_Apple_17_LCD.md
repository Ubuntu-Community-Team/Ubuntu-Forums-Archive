I'm having a couple problems with Hoary and my Apple 17 LCD.

Hardware:
Dell PowerEdge 420SC (Intel HT P4 2.8/1GB RAM)
ATI Radeon 9800 Pro 256MB (DVI)
Apple Studio Display 17 LCD (using Apple ADC->DVI converter)
HD#1: Windows XP
HD#2: Ubuntu 5.04 with X.Org

Problem #1:
I can only get the vesa driver to work.  It works fine and displays at the full 1280x1024 resolution.  The problem is that I can't use the ati or fglrx drivers.  Both result in a totally blank screen.  When I hook it up to my 20" Apple LCD, it works great (and looks fantastic).  But that screen is for my G5.  Has anyone managed to get hardware accelerated drivers to work with an Apple 17 LCD?  What should I investigate?  I'm at a loss for solving this part of the problem.

Problem #2:
With the vesa driver enabled, I never get the display to sleep.  I set the power management options to power the display down after 30 minutes but it stays on.  The screen is blanked but it is powered.  Again, this works fine with the 20 LCD.

Any help is greatly appreciated.  I know that the hardware combo can work because it does so perfectly when the Dell runs Windows XP (1280x1024 HW ATI accelerated + power management).
