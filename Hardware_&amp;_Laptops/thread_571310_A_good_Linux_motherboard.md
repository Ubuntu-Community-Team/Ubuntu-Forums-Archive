### Post by Tekno_Cowboy on 2007-10-21
I'm running that board with a 4200+ EE and it works great with Ubuntu 7.10, and pretty good in 7.04.
If you get this board be sure to update to the latest BIOS before you try putting GNU/Linux on it. At some point they added HPET to the BIOS and it makes Ubuntu happy. Make sure the HPET is set to 32/64-bit to match the version you are installing.
I have had some troubles with Gentoo based distros, but it works great with Ubuntu.
One problem I am having is that my optical out on the board is having issues, but I'm sure that's just a setting issue I need to work out. I had no sound in 7.04 untill I updated ALSA, since the ALSA that shipped with 7.04 didn't support this MB.

Hope this helps!
