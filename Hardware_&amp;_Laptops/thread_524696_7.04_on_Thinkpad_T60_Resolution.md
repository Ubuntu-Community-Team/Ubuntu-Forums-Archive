Hello,

I have the same model Thinkpad. Mine uses the ATI X1300 card. I've had the best luck installing Ubuntu 7.04 using the **Alternate CD** vs the Live CD. The Alt CD will ask you all the resolutions your card supports and you just select them from a list. Once done you will be all set. :guitar:

Also, once in the Ubuntu environment, just update your system then run the restricted ATI drivers listed under: SYSTEM>Administration>Restricted Drivers Manager and you will have acceleration enabled.

Enjoy,

### Post by rabbit83 on 2007-08-13
There's an easier way: run
```
sudo dpkg-reconfigure xserver-xorg
```.

You can leave all settings untouched except the resoultions - choose the display's native.
