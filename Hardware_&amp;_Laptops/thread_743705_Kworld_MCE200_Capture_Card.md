### Post by Slushdoot on 2008-04-03
MythTV is a complete system for recording and watching recordings using an analog or DVB tuner card.  Many cards are supported, though I can't speak to that individual card.

While it doesn't specifically include the drivers, the Mythbuntu control centre (sudo apt-get install mythbuntu-control-centre) lets you simply select a card from a dropdown list which it then automatically configures for you.  If you don't want to use MythTV this probably isn't the solution for you.

Check out the MythTV wiki to read all about the system itself and loads of supported hardware.
