### Post by rasec on 2006-07-14
I tried fedora, and it seemed to work ok. But since I've been using debian for a while I decided to try out ubuntu. I originally had debian on my powerbook, but one of the packages in sid screwed up gnome which prevented my from using firefox under kde, so I decided to give dapper a shot and I've been using every since (well except for that brief encounter with fc5).

Fedora is nice and everything, and my first exposure to linux was with redhat 4 or so, but once I gave apt a shot, yum seemed like something too little too late. I have to admit though, I do prefer fedora's installer over dapper's because it provides me with more configuration options. But I also like how dapper's installation cd doubles as a livecd with a gui, fedora's only provides a cli/chroot rescue mode environment last time I checked.
