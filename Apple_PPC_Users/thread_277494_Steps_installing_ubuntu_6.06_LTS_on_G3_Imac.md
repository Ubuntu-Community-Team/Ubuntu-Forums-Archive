### Post by GREGMACKENZIE on 2006-10-18
Hi,

Yeah, I haven't found the dapper installer very friendly either. The shipit CD does not work, you can't boot an imac g3 crt from it. You need the text based installer which can be had from the website. If your on dialup though, as I am, downloading it is not an option.

However as the previous poster noted, the 5.10 breezy shipit cd will work, as will the hoary 5.0.4 shipit cd.

It appears impossible to "upgrade" using the cds in sequence. After I installed hoary I put in the breezy CD, and while it did pop up with the "upgrade manager" and a few things were updated, the repositories essentialy remain hoary. I had to put the 5.10 cd in and install from that to fully get breezy.

Breezy is pretty cool and will get you going with the Ubuntu experience.

Greg
