If you want to have an OS on the external drive choose OS X for that because it will be much easier to boot than Linux (meaning from the external drive)

And if you want to install OS X you have to format the drive as HFS+ (and i would also include the Journaling and Case-Sensitive options)

### Post by 3rdalbum on 2007-06-12
You can have OS X running inside a window on Ubuntu, (it's a program called Mac-on-Linux) but if you can't write commands then you might struggle to get it running. That's okay; I don't use Mac-on-Linux.

You can have OS X on one hard drive or one partition, and Linux on the other; and be able to choose which to start up when you turn on the computer.
