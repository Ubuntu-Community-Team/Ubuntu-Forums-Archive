Do as the message states. In a terminal
```
$ sudo dpkg --configure -a
```
How did you do your update? I find the Update Manager often doesn't work or causes problems. Better to use to Synaptic or Aptitude to get updates/upgrades, etc. Doesn't take much to learn.
