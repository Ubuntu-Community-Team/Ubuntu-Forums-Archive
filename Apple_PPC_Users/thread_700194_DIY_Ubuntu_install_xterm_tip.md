### Post by stream303 on 2008-02-18
I recently installed a minimal cli system, and then selectively added back in X-window components for my own unique setup.  I followed the docs here, even though they it seems to be pc-based at the start.  I just installed the server image, but declined to actually install the dns or lamp options - there are other ways to get a cli box up.

[https://help.ubuntu.com/community/Installation/LowMemorySystems](https://help.ubuntu.com/community/Installation/LowMemorySystems)

If you are new to doing this, one thing you might come across is that on ppc, the fonts for the xterm terminal can be very small.  Even if you abandon xterm for something a bit nicer, initially you can make xterm much more pleasant by altering the font size.

Just drag your mouse over the xterm window, hold CTRL-RIGHT CLICK, and change the size to something Large or Huge.  Ah!

This isn't anything really new, but if you are a newcomer to using xterm, it'll keep you from going blind. :)
