### Post by toojays on 2005-02-09
The link above is only relevant for the iBook G4 and some AlBooks (i.e. Apple laptops with Radeon 9200 or greater). Those laptops only got sleep support recently, and it has not made it into Linus's kernel yet.

The iBook G3 has had sleep support in the kernel for some time now. I'm not sure what the status of that is with respect to Ubuntu, but you should be able to get it to work somehow. If you can't get the information you need here, the Debian PowerPC list might be a good place to try.
