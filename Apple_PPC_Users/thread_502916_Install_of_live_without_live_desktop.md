The live cd is strictly that, Live.  If you desire to install Ubuntu without the need of first booting up into the Gnome desktop, you will need to use the Alternate Install CD.  

Ubuntu 7.04 Feisty Fawn PowerPC install cd's can be found [here]("http://cdimage.ubuntu.com/ports/releases/7.04/release/").

> **tcrroadie said:**
> The live cd is strictly that, Live.  If you desire to install Ubuntu without the need of first booting up into the Gnome desktop, you will need to use the Alternate Install CD.  

Since when? There is an Install icon on the desktop, which you can use to install Ubuntu to your hard disk.

Note that on PowerPC you will need an empty partition (or one you don't mind losing); the installer cannot resize HFS partitions. There are howtos on this forum for resizing partitions manually, but make sure you back up everything first!
