### Post by rondackcpu on 2007-06-06
I can only answer for Thunderbird since I don't use the others.  In your home folder, click on "View" and select "show hidden folders"; then find and open ".mozilla-thunderbird"; then find a file with a random string as its name followed by ".default"; open it; find "mail"; open it; find the user account you want; open it; inside will be ascii files called "inbox", "sent", etc.  Copy what ever of those you are interested in saving.  Back up one step and choose other accounts, if any.  From these ascii files, you can copy the whole thing, or open with a text editor and save individual email.

Or just copy the whole "mail" folder.
