### Post by slux on 2005-05-22
I'm sorry but you're not going to be able to run those games on Ubuntu PPC because there are only versions for the x86 processor architecture available. It might be worth it to mail the relevant developers asking for a PPC port (which might be only a compile away since both games have already been released on PPC for OS X) but I wouldn't hold my breath.

Quake 3 will be GPLed pretty soon and then you'll be able to play that one. Meanwhile, there are a few first person shooters that you can play: Cube, Nexuiz (very soon to be released) and of course Quake 1 & Quake 2.
