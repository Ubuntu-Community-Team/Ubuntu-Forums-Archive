### Post by ixus_123 on 2006-10-13
ubuntu runs pretty well on my iBook Clamshell - 366mhs G3, 320mb ram.

Sleep works on lid closed.  XFCE is lots faster than Gnome & I prefered this for teh clean look & to help free up screen real-estate as I only have 800x600 resolution.

If you want really speedy you can install blackbox window manager - it's good if you only want to use a specific app when you log in as you'll be able to make full use of memory & cpu for that rather than desktop eye candy.

I did the normal Ubuntu install & installed the other desktops after.

There was a split screen issue on my installl but this is easily fixed.

The only other problem I had was that I couldn't get manula partitioning to work - a problem with the yaboot partition so I just went for teh auto mode.

I couldn't tell you what power management is like - my battery is lucky to last 30 mins these days ao is always plugged into the mains.  Currently the iBook is enjoying a new lease of life as an Ubuntu webserver

### Post by Jose R on 2006-10-13
iBook G3, 600mHz, 640ram.

Like you I had 10.3.9 on my iBook, but it ran so slow I stopped using it.  Installed Ubuntu, liked it, was a little snappier.  Tried to do the MOL thing, got it working, but it slowed everything down.  So I wiped the HD clean and did a clean install of Ubuntu.

I really like the look and feel of the gnome DE.  The transparency effect is cool, but the minute I have too many apps open things slow down considerably.

So, once I resolve some issues with java and download a few more apps I want, I will go back to Xubunu for the speed.  Or, maybe even tried fluxbox.
