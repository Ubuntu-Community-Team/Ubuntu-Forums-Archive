### Post by ipixel on 2005-05-21
I'm a bit disappointed, that nobody seems to have an answer to this one... I thought this surely would be a simple one, and that someone would come back to me within minutes with the usual "Oh, just use XYZ."

After *MUCH* searching through Google, and the forums here, it seems that there are 2 ways of entering international characters:

1) Using a keyboard layout that supports your character.

This option *should* be the most intuitive and 'standard', however, as there are NO PROGRAMS that can show us what the keyboard layouts are, and what key combinations trigger which characters, this is usually ignored by most Linux users.

2) Using a CONTROL+NUMBER combination, based on the character's UNICODE number.

I just read at another thread, that *supposedly*, if we type "control + CODE" - where CODE is the character's Unicode code, we will get that character - regardless of the keyboard layout being used. I will try that later, when I'm using Kubuntu again (at office right now). If anyone else can report on the usability of this technique, please do!!!
