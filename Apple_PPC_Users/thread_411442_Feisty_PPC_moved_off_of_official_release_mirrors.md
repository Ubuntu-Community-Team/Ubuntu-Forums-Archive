### Post by DJ_Max on 2007-05-03
Does anyone know why the SPARC architecuture  is still official,  when hardly anyone uses it?

The PPC user base in general has always been small, no matter the OS. The premire user was Apple (I know about IBM, but hardly anyone uses it as a desktop, let alone a server, besides corporations.) but now their using x86. One of the other reasons for ditching PPC support, is for the simple fact, there are too many variations between IBM/Motorola/Freescale models (POWER, POWER6xx, POWER2, POWER3, etc..)

To be honest, there's not many reasons to keep PPC, besides most schools use iMacs, but than again, most schools with iMacs use OS X (some still use 9....) and see no reason to change.\

As long as there's a strong community effort (the entire Fedora project is community driven) Ubuntu PPC should be here to stay. Though, don't count on the Playstation 3 to keep PPC going, it's sales are horrible.
