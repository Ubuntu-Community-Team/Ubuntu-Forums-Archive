### Post by junglepig on 2007-10-28
It can be done. I've been running Feisty instead of Jaguar on my Beige G3 for months. It is rock solid. 
Installation is tricky if it is a beige. For Beige you'll need a utility called BootX. After installation you'll need to transfer the new ramdisk image and kernel to the SystemFolder/Linux kernels folder on your Mac OS 9 startup disk. There are several articles on the web that can get you going if you really want to do it.

I tried the update to Gutsy once (From the Ubuntu Gutsy Alternate Install CD image), but it didn't go smoothly because I ran out of disk space on the 4 gig drive :(. I'll try again when I have time. Feisty's all good.
