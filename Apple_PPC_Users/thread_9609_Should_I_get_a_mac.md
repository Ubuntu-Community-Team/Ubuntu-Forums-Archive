If you get a new mac, get it for the OSX OS which is an absolutely wonderful closed system.  And if you have the HD space, *do* partition and put Linux on it, but I suspect that if it is supposed to be a Linux-first machine, you may be better off getting an x86 machine.

The disadvantages of the Apple Hardware -- the Linux support while good, isn't as good as the x86 hardware.  Things like Games, Flash Players, etc, aren't there (yet).

Advanatge:  OSX.  OSX may be resource heavy and hard to customize, but it is absolutely gorgeous, and the system is really good (being BSD based stable, etc.) and the hardware used in OSX integrates really well with few hang ups.

If I didn't have this old G3 Pismo gathering dust and i had to buy a new laptop, I would get something from the x86 world and put Linux on it in a partition.
 :-) 

Have fun!
