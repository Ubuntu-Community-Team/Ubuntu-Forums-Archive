### Post by Skia_42 on 2006-07-21
Make sure that when you are burning the boot disk that you are doing it slowly (1X - 4X). If you burn it at faster speeds there is a higher chance of mistakes being written. This makes a big difference when you are booting from the disk especially in Laptops.
