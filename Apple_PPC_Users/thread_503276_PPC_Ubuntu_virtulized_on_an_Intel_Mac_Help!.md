### Post by schmod on 2007-07-17
Quick answer:  No.  You can't.

Long answer:  If you could,  you wouldn't want to.  Efficently emulating a PowerPC on an Intel chip was, for quite a long time considered to be the holy grail of processor emulation.  Apple's Rosetta technology is the only thing that even comes *close* to doing it well, and I doubt you'd be able to use it to run a virtualized copy of Linux.

If you've got a fast processor, PearPC might offer some help to you, and there's always SheepShaver (although I would tend to doubt that you'd be able to get Ubuntu going in SheepShaver). If you actually get it to work, anticipate a HUGE performance hit (SheepShaver's quoted as running at 1/8 native speed).

You'd be better off virtualizing x86 Ubuntu, which is a relatively easy task these days.

There's absolutely no reason you'd want to do this.
