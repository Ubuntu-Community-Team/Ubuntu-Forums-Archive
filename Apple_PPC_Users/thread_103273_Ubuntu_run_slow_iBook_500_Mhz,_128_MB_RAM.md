### Post by BabaFree on 2005-12-13
If your not too set on using gnome, you could try the XFCE desktop (or maybe it's xcfe, I always get it confused :)  It using a whole lot less resources than gnome.  One drawback, though, is that you can't have files on the desktop.  You have to run programs from the menu.  I had an old computer I put Ubuntu on for my in-laws and gnome would make simply crawl.  With Xfce it is very usable.  You can download it through Synaptic and choose it at login under sessions.  Hope this helps,
