### Post by stream303 on 2008-01-21
Ah, this default catch may be related to what I've been reading about an 8Gb partition limitation (obviously with a 20gb drive it would have more than one partition in use).

When you upgraded the hard drive, the limitation was broken, and now default-catch shows up.

Maybe you can find a firmware update to get past this, or you'll have to manually partition the drive with an 7.5GB limitation (just to be safe) in one of the partitions - I'm not familiar with it, so I'm just feeding back some googling....
