### Post by Uta on 2006-07-22
Target Mode--
1. Apple says that you should unplug all other FireWire devices from both computers.

2. Make sure that the Target Mac is turned off. (The Host doesn't need to be turned off.)Insert DVD in Host machine.

3. Use a FireWire (6-pin to 6-pin) cable and connect the two Macs.

4. Start up the Target Mac while you hold down the "t" key. You can release the t as soon as the FireWire icon appears

You will be able to install with the host's DVD drive into the machine with the broken DVD drive.

If you need more info, since this is not really a Linux question I would suggest you google Mac OS X Target Mode. Good Luck!
