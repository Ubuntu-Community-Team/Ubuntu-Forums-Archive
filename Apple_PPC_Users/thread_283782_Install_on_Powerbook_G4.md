### Post by calum on 2006-10-24
Yes, it's quite possible.  You'll need to create a separate hard disk partition to install Ubuntu on, though... I /think/ the Ubuntu installer can do this for you now without trashing your OS X installation, but you really, really want to back everything up first just in case.

(If the Ubuntu installer can't do it, you can do it by booting from your OS X install disk and running Disk Utility-- but that will definitely trash your OS X install in the process, and you'll need to reinstall it afterwards.)
