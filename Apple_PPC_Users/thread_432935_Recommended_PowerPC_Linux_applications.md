### Post by stmiller on 2007-05-04
Some of my fav's:

banshee (for music)
grip (ripping CDs)
digikam (for photos)
VLC (playing back flash video, windows media, playing DVDs, and most any video formats)
K3B (burning CDs/DVDs)
Frozen-Bubble (addictive game)
Audacity with all the ladspa plugins and such

And you do not have to download .deb files. Just type in a terminal
sudo apt-get install grip

and the program will install. Or use the program synaptic to pick programs from a detailed searchable list.
