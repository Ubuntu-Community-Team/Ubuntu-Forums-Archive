### Post by handylinux on 2007-01-10
> **Anavrin229 said:**
> i got a imac running on mac os 9.2 and i cant get it to run ubuntu from a CD. can someone sugest a linux to use and also a site to download it to a disc plz

It sounds like you have downloaded an Ubuntu CD ISO file, and burned it to a CD. Is it the PowerPC Ubuntu (not the i86)? Have you made a CD correctly from it? See [this thread]("http://www.ubuntuforums.org/showthread.php?t=332928") for discussion.

If it's the correct Ubuntu CD, and it's been created properly, and your iMac has enough RAM to run Ubuntu, the CD should boot your iMac -- unless there's something wrong with the CD drive. Test the drive with a bootable Mac OS CD.

EDIT: Depending on what model iMac you have, you may run into the "8GB limit" on the hard disk. I see you've posted on a thread where that is discussed. Haven't tried installing on any older Macs with that problem, so don't know how it works. [Low End Mac]("http://www.lowendmac.com/imacs/index.shtml") is a good resource for information about older Macs; see "G3 iMac profiles" (assuming that's what you have).

### Post by samden on 2007-01-10
I have recently installed Linux on my iMac (fairly medium spec - 500Mhz and 192Mb RAM). A few pointers:

The Ubuntu live cd probably won't run, depending on how much RAM you have. You may find however that the Xubuntu live cd will run (Xubuntu is basically a version of Ubuntu that is a bit more lightweight). I run Xubuntu and like it much more that Ubuntu anyway (I have tried the Ubuntu live cd on another machine and it didn't feel as good as Xubuntu, but that was just my opinion).

If the Xubuntu live cd will not run, you will need to use the alternative install cd for either Ubuntu or Xubuntu.

Some people say they had no problems installing, others (like me) have had weeks of tearing their hair out! You don't know till you start. But after using Xubuntu on my iMac I can definitely recommend it as good for this machine.

Have a browse around the Ubuntu website and see the options for cds to download (follow the "ubuntu" link at the top of this page). There is a "Xubuntu" link on the Ubuntu home page - follow that for Xubuntu.

You WILL have some problem at some stage, don't give up when it seems to be turning to custard. First, do a search of existing posts, both through the search option on this site and through Google - it is likely that someone else has had the same problem previously and the answer already exists somewhere. If you can't find the answer on the net, just post a question here and there'll be plenty of people interested in helping.

Happy installing!
