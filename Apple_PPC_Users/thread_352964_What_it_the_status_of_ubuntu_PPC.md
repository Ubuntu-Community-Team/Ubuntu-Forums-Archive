### Post by 3rdalbum on 2007-02-05
Also, any showstopper bugs in the PowerPC build will not stop the release of the x86 and x86_64 versions, but showstoppers in the x86 version (for instance) will stop the 64-bit release and probably stop the PowerPC release too.

No security updates, either, probably; but I figure that since Linux is pretty secure to begin with, and PPC Linux is a "niche inside a niche", you're well protected even without constant security updates.

Dapper and Edgy will still get security updates, but Feisty won't.
