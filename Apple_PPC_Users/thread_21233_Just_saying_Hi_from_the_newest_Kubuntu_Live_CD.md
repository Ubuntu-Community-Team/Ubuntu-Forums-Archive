### Post by spaceballl on 2005-03-21
Hey all,

Just writing to say hi. I'm on the newest version of the Kubuntu LiveCD (KDE version of Ubuntu) on my iBook G4. Basically, it looks great! Everything is very snappy here, and I'm loving the overall experience. I'm running on Konquerer right now. It's no Firefox, but it seems to be working great.

While there's no replacement for Apple's iLife suite and many other OS X niceities, I really feel like if PPC linux supported Apple's airport extreme cards, I would run some sort of linux distro instead of OSX for 50% of the time. Well, here's to hoping that happens at some point.

In any case, for those of you who haven't given Kubuntu, a shot, check out the LiveCD. Very nicely done. I prefer Gnome overall, but I hadn't seen the newest version of KDE for a while and it is definitely some pretty stiff GNOME competition. Okay that's all :-).

-Kevin
