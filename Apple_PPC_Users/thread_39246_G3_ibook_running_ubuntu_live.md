 the file you downloaded (about 700 megs) is a disk image.. to burn it you need to use a program like nero and find a menu option that says 'Burn ISO' or 'Burn disk image'.

If you are using OSX, the program you can use is 'disk utility'. I think it's in Applications > Utilities > Disk Utility. (I don't have a mac myself.. just used ones at school to dl and burn linux CDs :p ).
Select Images > Open, open the live CD file to burn, then click the burn icon in the toolbar (there are some screen shots [HERE]("http://www.mrbass.org/mac/dvdburn/#1") to help)

Hope you get the live CD working (and like it enough to install Ubuntu :))
