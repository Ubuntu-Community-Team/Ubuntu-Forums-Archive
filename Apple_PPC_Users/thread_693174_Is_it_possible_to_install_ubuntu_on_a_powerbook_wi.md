### Post by dabs on 2008-04-20
Sorry to bring this thread back from the dead, but I am (was) having the exact same problem described above..

Try holding cmd+F1 (the brightness down button, no need for fn) on boot:  This forces the use of an external screen.

(Tip from **shortl** of osxhints, all credit to him!)

I gave up and moved on a while ago but now at least anyone else with the same problem can figure it out.
