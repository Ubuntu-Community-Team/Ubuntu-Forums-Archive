### Post by eidex on 2006-10-18
problems with ppc platform (i have an ibook g4):

-no flash support (i know there is gnash and some other pgrogramm, but gnash is still in development and has problems with many sites/videos/sounds)

-laptop runs very hot, fan control is not installed by default (dapper, edgy), so if you install it this is the first thing to do to protect your book (search for therm_adt746x), even with installed fan module the fan starts very often, its annoying because with osx you only hear it if you do cpu intensive work.

-i have problems with ibmjava on some websites, there is no sun java, you may not encounter that problem

-airport is annoying too, at home it works perfectly with encrypted network, public wlans often make problems, i can connect to them but it only works for 1-5 minutes, then i have to reconnect (by shell script, networkmanager cant reconnect)

i never regreted installing dapper or edgy but i still nead osx for some tasks (now that compiz works on my ibook i think i will spend more time with linux again)

greetings
