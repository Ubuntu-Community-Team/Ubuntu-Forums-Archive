### Post by anurse on 2007-02-16
Hi,

I use ubuntu as a daily work machine. This includes all the standard stuff (web browsing, e-mail, writing, spread sheets). I don't want to sound like a zealot but I also like the FOSS approach. It seems more fair; more equal. I don't want to argue philosophy, but the idea of using something in which a community plays an important role, in which there is participation and mutual help, is also something I like. It reminds me, a bit, of the bad old days when Mac was going to die and we huddled together in user groups. I like Mac and still use it. For my perhaps modest uses, I don't think Linux is in any way superior to Mac OS. On the other hand, for what I do, Mac is not superior to Linux.

### Post by ssam on 2007-02-19
> **min_max9000 said:**
> Thanks for the tips. Where is a good repository of trustworthy Linux apps. For mac I was used to Version-Tracker or macupdate, but I am not aware of a Linux type service? Can someone point me to one? tnx.

Applications -> Add/Remove

its about as easy as it can be to install new software in ubuntu.

if you want a more advanced interface try

System -> Administration -> Synaptic package manager.

also look at System -> Administration -> Software sources, and make sure all the ubuntu repositories are enabled.

occasionally you might want something that is not the ubuntu repositories, or a newer version. in this case it usually takes a bit more work to install the program. feel free to ask if this happens.
