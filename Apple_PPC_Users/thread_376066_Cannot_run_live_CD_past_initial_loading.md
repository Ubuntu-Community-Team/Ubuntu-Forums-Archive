### Post by mike_hore on 2007-03-06

Hi Kerttie,

The Dapper installation is broken on G5 iMacs.  This issue has been around a long time -- if you search this forum for "G5" you should find all the gory details.  It can be worked around, but it's a hassle.  I recommend using Edgy instead -- it installs fine.  Afterwards you'll need to do a few steps to get the fans to be quiet and the screen resolution right, but in the end it should all work.

Cheers,  Mike.
