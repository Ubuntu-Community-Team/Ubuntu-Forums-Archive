### Post by ZoiaGuyver on 2007-03-23
I did a install last night on my G4 450, 768mb ram, 80gb HD with both Edgy and Fiesty, i had similar probs but i dont know if this solution will work for you it may.

Unplug the Tower with the tower unplugged push the power button to clear any remaining power, open the side panel and look at the bottom of the board (closest to the chasis) for the PVRAM battery (its about half the size of a AA battery). If you look just to the right of it, you should see a little grey button. Push and hold that for around 10 secs then let go this will force all default settings in PVRAM to be restored. 
Plug the system back in (and if possible make sure you have a active connection to the net, the system will need a active connection to resync the date and time). 

Power up with the Ubuntu/Kubuntu CD in the drive while holding down the "c" key, it should then start to boot from the CD.

Ive read a lot of problems with the Mac G4's being badly synced clocks, dead/dying pvram battereries or custom OF (Open Firmware) settings being the cause of failures to install Ubuntu and its varients.

Hope this helps.

Ubuntu rocks on the G4's :D
