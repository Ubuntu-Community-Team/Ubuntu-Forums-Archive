### Post by BladeMelbourne on 2008-04-14
Gnash doesn't work more often than not (it's pretty immature).

Many of us use Firefox plugins (eg Download Helper) to download Flash videos (flv files) from sites like youtube, then open them using Videolan Client (VLC).

The Totem player in Hardy can apparently search and play Flash videos from youtube without using a web browser (although I haven't tried this yet).

Others use Mac on Linux to boot their Mac partition within the linux desktop. Performance is just bearable on a G4 1.42 GHz.

Skype is another such program available for Mac PPC and Linux i386, but not Linux PPC.
