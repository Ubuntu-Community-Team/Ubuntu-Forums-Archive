### Post by Synchro on 2006-08-02
This seemed like an obvious thing to want to do, but it doesn't 'just work'.

There have been various threads about booting Ubuntu from external Firewire HDs. However, none of them are very conclusive, or contain incomplete information. They are also mostly out of date, applying to PPC hardware. On Intel Macs, not only do you have a new CPU, but also one with completely different firmware and boot device support, including BootCamp.

On my MacBook, I booted from a 6.06 server install CD, formatted the external FW HD, then ran the installer. No problems were reported (and I didn't see yaboot mentioned at all), but when I try to boot from it (it shows up in BootCamp), I get:

"No bootable device -- insert boot disk and press any key"

I've not seen this message mentioned in any of the other firewire booting threads, nor have I found discussion of booting from firewire on Intel Macs.

Anyone have a definitive howto for this?

Thanks.
