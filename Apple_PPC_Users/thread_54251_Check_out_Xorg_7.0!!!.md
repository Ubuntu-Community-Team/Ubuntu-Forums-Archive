### Post by autocrosser on 2005-08-04
As I hear it, Breezy will use Xorg 7.0--ATI card users rejoice!!!

Part of the spec upgrade is--
1.  Added DMA support to radeon(4) for Render and Xv operations
2.  Numerous ATI driver updates from the GATOS project, including TV input support :smile: 
3.  Experimental DRI support for Radeon 9500 and above  \\:D/ 
4.  Tiled framebuffer support for radeon(4)
5.  ATI Theater 200 video decoder support
6.  Individual extensions may be enabled or disabled on the command line using the -extension flag
7.  Updated PCI scanning

Sounds like a MUCH better time for all!!!

[http://xorg.freedesktop.org/wiki/ChangesSince68](http://xorg.freedesktop.org/wiki/ChangesSince68)
