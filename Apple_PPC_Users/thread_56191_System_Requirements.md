### Post by paulg on 2005-08-14
Hey Freddie, I think I can help you out since I have a similar situation. I just installed Ubuntu PPC on the iMac previous to the version you have (G3 333MHz, tray load, 96MB ram etc.). I have 32 MB more than you but I do not think that is much of an advantage and doubt you would have any trouble running a desktop as I find it runs quite well. The Gnome window manager, although very pretty, was hitting the swap file too often. I found a handy little HowTo on the wiki:

[https://wiki.ubuntu.com/LowEndSystemSupport](https://wiki.ubuntu.com/LowEndSystemSupport)

XFCE is not as sleek as Gnome but as the author says I definitely find it a lot more 'snappy'. Good luck!

~pAul.

P.S. Gnome was usable but I so far prefer the performance boost in XFCE.
