### Post by jtbrookreson on 2008-01-27
I found some pages online that helped me (I googled "Xubuntu on Ubuntu). It seemed to answer my questions. I installed Xubuntu and relogged in, and everything works fine. It is almost "cleaner." I can feel the machine not working as hard. So far I like it. I will report any bugs that I might find.

But my suggestions to anyone out there with a G3 500mhz that wants Xubuntu, first install Ubuntu (6.04 seems to be the best right now), and then in terminal type this command

sudo apt-get install xubuntu-desktop

the rest is easy....
