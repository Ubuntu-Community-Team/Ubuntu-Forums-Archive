### Post by 3rdalbum on 2007-04-21
Basically:

New-world Powermac (Old-worlds might still work, albeit with a lot more effort), new enough so that it came with USB ports.

256 megabytes of RAM

CD-ROM drive

2 gigabytes of hard disk space if you want to install.

### Post by 3rdalbum on 2007-04-22
System Requirement for Xubuntu:

128 megabytes of RAM (192 megabytes if you want to use the Desktop CD)
Newworld Powermac
2 gigabytes of hard disk space for installation (note: you must create a non-partitioned space, or delete an existing hard drive partition, before you can install)
