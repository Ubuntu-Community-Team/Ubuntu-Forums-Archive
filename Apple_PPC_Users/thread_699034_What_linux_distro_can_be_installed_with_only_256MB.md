### Post by stream303 on 2008-02-17
> **Support the 2nd said:**
> I want to put this on a 433mhz G4.

Although this seems pc-centric, there are good notes here about low-memory systems, although I'd try a normal Ubuntu install and see if you can live with it or not.

[https://help.ubuntu.com/community/Installation/LowMemorySystems](https://help.ubuntu.com/community/Installation/LowMemorySystems)

You might also consider doing an "alternate" image install which is text-based for installation, rather than a live-cd Desktop version.  If you can live with a cli-only version, the  server install would fly right along.

Burn some here as an iso image, and let us know how it goes:  (Just hold down the "C" key to boot)

Here's the usual download locations:
[http://www.ubuntu.com/getubuntu/downloadmirrors](http://www.ubuntu.com/getubuntu/downloadmirrors)

or you can try the ports

[http://cdimage.ubuntu.com/ports/releases/](http://cdimage.ubuntu.com/ports/releases/)
