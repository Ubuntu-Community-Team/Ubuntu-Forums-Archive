### Post by stream303 on 2008-03-16
I'm a bit suspicious about the ram swapping/additions too.  Have you tried using the  "alternate" install images, and use just the original ram?  Without the firmware upgrade, some macs have a hard time with additional and/or 3rd-party ram.

Check
[http://www.everymac.com/](http://www.everymac.com/)

to identify your machine(s), and see what the original and maximum ram amounts are.  Perhaps play around with using less ram at first, and then add ram to see if the machine still crashes.
