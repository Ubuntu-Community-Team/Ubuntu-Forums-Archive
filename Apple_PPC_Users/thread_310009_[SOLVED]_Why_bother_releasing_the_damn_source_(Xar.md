### Post by 3rdalbum on 2006-11-30
When you build a piece of software intending it to only run on x86, you tend to make coding decisions that end off fixing it to that one architecture.

The converse is true - when you build software intending for it to build on multiple architectures, it will then tend to run on everything.

Releasing the source code does have one big advantage: The community might be able to fix the platform-specific bits of the code, so it will build and run on all arches.
