### Post by stmiller on 2006-08-05
Yes should be. Make sure it is connected and powered on, then run the installer as normal from your computer. When it gets to the part about partitioning the hard drive, make sure it is pointing to the firewire drive to install there. You MIGHT have to hold down the option key when booting (every time) to pick the firewire drive when it's all done. Though I'm not sure.
