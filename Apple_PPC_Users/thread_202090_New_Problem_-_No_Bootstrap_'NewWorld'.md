What ubuntu CD are you installing from: live desktop, server, or alternate?
What I would have done if I was you is to create free disk space in the disk, then let ubuntu automatically create ALL the partitions it needs using the free disk space. If you do that the installer will create the newworld boostrap, root, and swap partitions for you. There are some people with special needs that really have to create partitions by hand, but for many of us the automatic partitioning is perfectly acceptable.

The trick is that when installing from the live desktop CD you have to let the installer automatically create all the partitions ubuntu needs. If you really want to create ubuntu partitions by hand then you should be using the "alternate" ubuntu CD because it has a more flexible (and less buggy) installer.
