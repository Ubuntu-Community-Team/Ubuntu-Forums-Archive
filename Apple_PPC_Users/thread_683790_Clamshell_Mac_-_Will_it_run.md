### Post by stream303 on 2008-02-02
> **H3retic said:**
> Great!  Would it work even faster if I put a fluxbox interface on top of it?  The fluxbuntu forums aren't much help, so is it possible?

Yes, this is the perfect box for a minimal gui system that doesn't reply on a desktop at all, just old-school xorg and a simple window manager like fluxbox.  (although fluxbox is not set up properly in Gutsy!  Argh.  For a first install, I'd try Feisty to get your feet wet)

See the reference to the docs page on how to roll your own Ubuntu in this thread:

[http://ubuntuforums.org/showthread.php?t=685324](http://ubuntuforums.org/showthread.php?t=685324)

HOWEVER, when all is said and done, the apps you choose to run on it might still run slowly regardless of how fine-tuned your homebrew "desktop" is.  So instead of running Gedit as a text editor, you'd probably want to run leafpad instead.  You'll be using the terminal a lot, and a lighter weight alternative to the gnome terminals might be rxvt or just hanging out in the standard xterm.  Firefox is likely to run slowly.  

You'll have to evaluate yourself to see if you are up to all the manual configuration.  For some it is a lot of fun, for others it is just too much pain, and not being able to run the more bloated apps out today might result in a desktop that is not as functional as you'd want it to be.
