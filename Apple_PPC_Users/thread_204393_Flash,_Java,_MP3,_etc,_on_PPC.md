### Post by LettuceandPickles on 2006-07-03
[QUOTE=Rondonjin]All I need to know is are there MPlayer plugins, MP3 codecs, Xine codecs, Flash, Sun Java for PPC Linux?[/QUOTE]

Flash is almost impossible/worthless in Linux on a Macintosh (at least pre-Intel.)

If you need Flash, make sure the Mac has at least 512MB of RAM and install Tiger.  If you're used to Windows there shouldn't be any OSS objections to Mac OS X, and there's nothing wrong with OS X, all that stuff just works. (Unless this is an original iMac lacking FireWire, if it is, make sure you have at least 256 MB of RAM and install Panther.)
