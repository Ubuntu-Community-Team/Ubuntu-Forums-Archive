### Post by Auria on 2007-04-01
I have not tested this, but it may work:

Insert mac OS X install disk. Boot on it until you get to installer. Search the menus at the top of the screen, there should be a way to access disk utility. From there, you can reformat your hard drive in a format OS X will understand (or just erase the buntu partition, i'm not sure) then hopefully you can install
