### Post by Ptero-4 on 2006-05-15
Konqueror and firefox shrink the pics on-screen to fit them on the size of the browser window. The funny colours are a known bug in breezy's color palette implementation, which applies a yellow layer that shouldn't apply, fortunatelly it doesn't affect the pics themselves only the viewer window (your web browser). Expanding them with the zoom control takes this layer off the screen.
