### Post by digs on 2007-06-17
The installer might not be able to install yaboot into an external drive. I know previous ubuntu's wouldn't do it - not sure about current ones. Check out [http://www.debian.org/releases/stable/powerpc/install](http://www.debian.org/releases/stable/powerpc/install) to manually install yaboot.

It's a pain in the *** with debian: [http://www.howtoforge.com/boot_debian_from_external_firewire](http://www.howtoforge.com/boot_debian_from_external_firewire)
