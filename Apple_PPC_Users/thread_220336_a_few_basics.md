### Post by avtolle on 2006-07-21
Thank you. Before proceeding, you should look at some of the threads concerning various issues others have had when upgrading memory. It appears that on occasion, adding additional memory which is "unmatched" causes boot up problems, etc. 

Now, as to trying to answer at least part of one of your original questions:

Even though OpenOffice Writer is a resource hog, it allows files to be saved in many formats, including .doc for Word. Thus, I would recommend that one (it is part of the default Ubuntu install). 

As to the install itself, I recommend the memory upgrade before installing, at least if you want to use Ubuntu with its default Gnome Desktop Environment. Also, as to the install itself, I suggest that you obtain the Alternate Install CD by download and then burnining the.iso as an image to CD. This seems to be the better way to go, and presents fewer issues than use of the Desktop CD with the Graphic Installer.

### Post by avtolle on 2006-07-21
My understanding: unmatched memory means, e.g., two banks of RAM; one, 64 mb; one, 128 mb. There has been mention of this in a few posts on threads dealing w/boot problems, etc., on G3s in particular. Sorry, not in a position to retrieve the specific one(s) right now. There was also a suggestion on at least one thread that whether there had been an update to the firmware made a difference. One poster, who had none of the recurring issues discussed in the thread, i.e., his system just worked after install, had updated his firmware, and had "unmatched" memory, while another, who was having many of the same issues contained in the link I posted earlier, had not, and had "unmatched" memory issues. Didn't mean to cause you concern by the earlier posts, but felt you needed to be advised of some issues dealt with by some posters with similar hardware. I have a G4 Cube, 450 mh, 20G HD, and upgraded memory to 576 mb, BTW. Not any display, etc. problems or boot problems others w/different systems have had.

### Post by RHTopics on 2006-07-21
Robin,

If you are not sure yet about installing Ubuntu, then you may want to consider trying Ubuntu by using the Live CD.  That will let you try Ubuntu and see if you like it.  I think you will like it.:) 

The Live CD is also known as the Desktop version.  The CD provided through Ubuntu's free ShipIt program is your best option for a more trouble free experience.  It took about 4 weeks for the CDs to arrive after I requested them.

There is a problem with the Live CD and iMac G3 booting to a blank screen.  Solutions to fix this problem can be found by searching this forum.

To use the Live CD, you definitely want to have at least 256MB RAM installed.  The more the better.  I have used Kingston's Value Ram (128MB) and a PNY memory chip (256MB) in my iMac without problems.
