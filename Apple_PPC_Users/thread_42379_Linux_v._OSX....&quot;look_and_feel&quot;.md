### Post by msgyrd on 2005-06-17
I use OSX on a 12in powerbook and linux on a full sized desktops and here are my impressions:

OS X is a much more integrated system, things play nice together and feel right when using them in conjunction with one another.  What I don't like is multitasking in OSX.  Sure, it's partly my screen size that limits the dock size, but still, I enjoy the (native)multiple desktops on linux.  For productivity, I'm better off in KDE or Gnome, because I can more easily access all of my open projects documents.  Expose works nice if you have a handful of things open, but open up 10 or more windows and expose is showing you thumbnails that are difficult to make out and not much better than just cycling through them like on windows.  I like multiple desktops because I can just cycle desktops and see my documents in full size 3 or 4 at a time.

For general use, I prefer OSX, the eye candy and smoothness of the system are part of what I enjoy about computing, and linux is still lacking. If I'm programming or wanting to run servers, linux is where it's at, but if I want to browse the web, upload pictures from my camera, view web based media, and such, my powerbook is what I grab. Also, since OSX offers the power of unix, I don't feel such a need for linux on my Apple machine as I did on my desktops. However, if the airport extreme ever became supported, I would certainly try linux on the Mac.

Linux is a tank, Ubuntu is a nicely painted tank, but my powerbook still compares as a BMW.

### Post by Chunga on 2005-06-21
I use Mac OS X almost all of the time now for one reason: Spotlight.  Let me just say that it is my opinion that the Finder has gone way downhill over the past five years.  It seems to be getting progressively worse with each major OS X release.  Fortunately there is almost no need to deal with the Finder at all.  With Tiger, at any time you can just press command-space and type in some keywords for what you need and it will always find it.  It is pretty much making the hierarchical structure of file systems obsolete--I might as well just put all my files in one big folder for all it matters.  Apple really did Spotlight right, and I can see some big improvements for it still to come.  It can only get better.

I understand that similar desktop searching tools are in the works for Linux, which will be great and something big to be looking forward to.  When such systems are mature enough I'll be able to use Linux as my full-time desktop again.

Personally, I don't have a problem with the dock.  Some people really hate it.  It's considerably different than your standard task bar, but I guess I've had enough experience with both that I could use either equally well.

On virtual desktops, I personally don't like them.  It doesn't bother me that I have a lot of windows overlapping, and I usually do.  The difference is that OS X's expose makes activating any given window on a screen full of windows much faster than without expose.  This is just how I personally like to work--I won't argue that this is faster than using virtual desktops.  There may be tools for Linux with expose functionality, but I haven't looked into it.

As far as the "look and feel," aqua versus anything else isn't a big concern of mine.  I'm neutral.  I consider aqua as just a theme.  It neither increases nor decreases my productivity (which is the most important part) as far as I can tell.  It would be nice if Mac OS X had a supported way to change themes, if only for a bit of variation.

There are theme managers for OS X, and there is plenty of free 3rd party software available that will make OS X extremely customizable (see [ResExcellence](http://www.resexcellence.com/)).  There is also at least one very good, free [virtual desktop tool](http://desktopmanager.berlios.de/) if that is your cup of tea.  Bottom line: Mac OS X is extremely customizable (see [this screeny](http://www.resexcellence.com/snapshots/snaps_html/06-10-05_Spursman.shtml) ), but the only major benefits that I currently see (as a user) in favor of Mac OS X is Expose and Spotlight.  As soon as those two features are as functional on Linux as they are on OS X, there would be no reason why Linux couldn't play a more significant role as my main desktop.
