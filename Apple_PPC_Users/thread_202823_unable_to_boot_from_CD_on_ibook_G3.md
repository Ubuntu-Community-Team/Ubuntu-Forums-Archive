### Post by hbpro on 2006-06-24
I downloaded ubuntu 6.06 live cd ppc version and used the alt (option) key during ibook startup to select CD. The CD worked fine and ubuntu was up and running on my ibook. I tried running openoffice also and it went fine.

Then I clicked the "install" option on the ubuntu desktop to install ubuntu on my ibook. The system then hanged and stopped responding. So after waiting for 10-15 mins, I had to manually shutdown the machine and restart it.

But after that the system stopped responding to any keys during boot up. I tried alt (option), c, apple+alt+o+f, apple+alt+p+r and many others that I was able to find on various forums. But the system responds to none of these keys. The system responds only to apple+v keys for the verbose mode at boot up. I have not set any security password at boot up. After the initial beep at power on, the system immediately shows apple logo and boots up with mac os x.

I have no idea what has happenned as the system worked for the first time but now it has stopped responding to any keys at boot up and so I am not able to boot from CD. I have a G3 ibook with 192MB RAM. 

On some forums it was informed that I remove additional RAM and then try to reset the PRAM. But even that is not working. I am unable to reset the PRAM as well. 

Please advise what can I do to boot my system from ubuntu CD again.
