### Post by sulobanks on 2007-02-11
The first place would be here:
[http://pcmcia-cs.sourceforge.net/ftp/SUPPORTED.CARDS](http://pcmcia-cs.sourceforge.net/ftp/SUPPORTED.CARDS)

That will tell you what cards actually work with the pcmcia system.  Make sure to pay attention to chipset rather than simply the model number.  Frequently, the differences between revisions for a particular model involve new chipsets.  As far as WPA is concerned, I'm pretty sure that will be a card-by-card issue.  I haven't gotten it to work with airport extreme, but I know of some others who have, so WPA must work to some degree on ppc arch.
