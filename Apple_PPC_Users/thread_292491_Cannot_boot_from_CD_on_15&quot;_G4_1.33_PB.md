### Post by linear on 2006-11-05
There are a variety of things you can try:

1. Hold down the option key at startup, see if the edubuntu CD shows up as a bootable disk.

2. Burn the iso to a disk again, making sure that you burn as an image to the disk. Also burn at a slow speed (4x or less).

3. Try the alternate install disk. Works better on some machines.

4. Try a Breezy CD. If you can boot and install from that, do a dist-upgrade.
