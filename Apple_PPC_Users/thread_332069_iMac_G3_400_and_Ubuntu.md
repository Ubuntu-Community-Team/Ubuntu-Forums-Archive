### Post by bodycoach2 on 2007-01-07
On an iMac G3 400, I'd recommend Xubuntu over Ubuntu, or when Fluxbuntu comes out for PPC, go with that.

Like the link says, the LiveCD just doesn't work well. Use the alternate install CD, and make sure you burn it at a very slow speed. The slowest you can set. That seems to fix many install  problems on older iMac, especially anything under 800 MHz.

Let us know how it goes.
