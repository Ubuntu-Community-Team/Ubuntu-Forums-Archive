Acrobat Reader causes errors with dpkg on my x86 computer - so it's not worth installing. Evince (Document Viewer) which comes with Ubuntu, is much better (or you could try installing xpdf).

If you're talking about the program that used to be called Acrobat Exchange, then sorry - Adobe doesn't make one for Linux, full-stop. You can actually make PDFs through OOo and Scribus, if that's any help.
