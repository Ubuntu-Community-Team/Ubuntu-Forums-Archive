### Post by slux on 2004-12-02
The great thing about GNU/Linux is that not only can you do all those things, there are options for doing them in other ways too. Take window management for example. It does not end with GNOME or KDE, there's a wide variety of different window managers with different concepts of how the desktop ought to be. You mentioned the OS X eyecandy. Maybe you'd like to check out [e17](http://www.rasterman.com/) which is a fully themeable WM using 3D acceleration (the new version is still a work in progress though).

My desire for freedom is one reason for my personal usage of GNU/Linux, the point that azz mentioned. But I've always found the environment preferable to others in many other ways as well. Even to OS X. Saying Linux is a Civic and OS X is not entirely appropriate in my opinion. I don't like car analogies, but the situation is more like choosing between a Ferrari and a Lamborghini. Both are great in different ways...
