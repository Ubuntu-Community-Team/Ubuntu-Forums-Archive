### Post by erlyrisa on 2006-02-05
**Well - it's official** *i'm stupid*
all I had todo to fix this was disablle webcache-ing.
-It seems that konqueror tries to open files like they were local if you have your cache settings set to cache all the time. - if Any one else was having trouble with this then here is the info.
change : tools:HTML settings: Cache Policy, click : use cache if possible. - your web browsing will now be as per normal.
