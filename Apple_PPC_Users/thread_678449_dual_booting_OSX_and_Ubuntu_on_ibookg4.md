### Post by stmiller on 2008-01-25
boot camp and vmware do not work on PowerPC.

You can dual boot OS X and Ubuntu on an iBook. The best thing is to start from scratch- so back up ALL data, then start the OS X installer and make two partitions before installing OS X. You can access the OS X disk utility from the top menu at the beginning of the OS X install.

Then let OS X install in one of the partitions, then after the OS X install boot the Ubuntu CD and let it install in the remaining partition.

The Ubuntu installer will automatically look for the free partition and will not touch your OS X partition.
