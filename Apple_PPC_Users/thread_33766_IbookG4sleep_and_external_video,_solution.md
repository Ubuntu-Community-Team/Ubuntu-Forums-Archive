 I modified a xorg.conf file and i add some options. The sleep function work to add a option:
Option		"AGPMode"		"4"
in  Section "Device" where is my driver  Radeon.
Ref:
[http://ubuntuforums.org/showthread.php?t=25264&page=4&pp=10](http://ubuntuforums.org/showthread.php?t=25264&page=4&pp=10)

The external video work when i add this section:

Section "Device"
        Identifier      "radeon_clone"
        Driver          "radeon"
        Option          "DDCMode"       "true"
        Option          "CloneMode"     "1024x768"
        Option          "MetaModes"     "1280x854-1024x768 1024x768-1024x768 800x600-800x600"
        Option          "CRT2HSync"     "31.5-80"
        Option          "CRT2VRefresh"  "56.3-75"
EndSection
but I have to disable usefbdev and  uses mergedfb. If you use option usefbdev external video don't work.
Ref: [http://lists.debian.org/debian-powerpc/2005/01/msg00222.html](http://lists.debian.org/debian-powerpc/2005/01/msg00222.html)
