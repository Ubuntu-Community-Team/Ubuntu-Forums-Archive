### Post by ssam on 2006-09-03
since dapper (6.06) there has been a 'desktop' and 'alternate installer' disk.

the desktop disk is the live cd, but now it also includes an installer. the alternate cd is just the text mode installer (useful for older hardware, or non standard setups).

if you want a live cd get the desktop cd.
