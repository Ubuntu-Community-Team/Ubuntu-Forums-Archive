### Post by DirtDawg on 2008-01-09
I have a lil' red imac that's probably about the same age. Ubuntu does run, albeit a little slow, so I would strongly recommend using an "alternative install" cd which uses a text interface (but is still easy to install). I would also recommend using a lighter desktop than Gnome like Xfce or IceWM (I currently use IceWM which is lightning fast but a real pain to configure). 

I have actually installed "vanilla" Debian on my imac and it runs a bit faster than Ubuntu, although it can be more difficult to setup and tweak at times.

You should also be aware that there is no way to install Flash, codecs are hit and miss, Java can be difficult to install, and 3D acceleration is probably a no-go. Even with all those disadvantages, I am still of the opinion that Linux is a much better solution than OSX or OS9.

Good Luck
