### Post by stmiller on 2007-05-02
Yes there is an official feisty powerpc release. There are about 5 threads on this forum which say 'where is the powerpc release?' so please search for those threads. I'd recommend feisty over anything previous. The entire 'long term' thing is really just for x86 computers. Powerpc machines benefit from the newer kernels (sleep, wireless, other things) in feisty.

### Post by BryannMelvin on 2007-05-03
I run 606 LTS on a g3 ibook dualusb
This DOES have security updates available for a longer term BUT has some bugs particularly around sleep

Kubuntu 606 seems to do better than regular Ubuntu however

If you want later versions of some software available that is a problem

I HAVE compiled my own versions (newer) of Scribus KOffice GraphicsMagick and others that are not available for dapper.

If it's a personal use computer you are probably better off with Feisty however if it is one of a set of production use machines where you will add and tailor apps to do WORK I recommend. using Kubuntu 606 with the gui clock set at UTC. (hovering will give local time) Gnome occasionally screws the clock when left to sleep too long.on my g3 ibooks.,

I've been using 606 with my own compiled versions of Graphics and Office software since 606 was released,
on both pc and ppc.
