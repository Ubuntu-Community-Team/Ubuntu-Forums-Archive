you can use target mode. you will find instructions here: [http://docs.info.apple.com/article.html?artnum=58583](http://docs.info.apple.com/article.html?artnum=58583)  

*** IMPORTANT: don't forget to update firmware if necessary BEFORE installing os x. PS some firmware updates can only be done from os 9.1  

if you haven't already installed ubuntu; you'll be in better shape. 

1. partition your drive with disk utility found on the os x  install disk.
2. leave one partition as free space for the install of ubuntu after you install os x. that way yaboot  will know where other os is located (as i recall, it's best to make the frees space partition the first partition).  
3.  you should be able to select the target disk as the one to install on.  install os x.

good luck
