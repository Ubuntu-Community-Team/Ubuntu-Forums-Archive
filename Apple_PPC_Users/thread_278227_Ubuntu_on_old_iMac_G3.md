### Post by 3rdalbum on 2006-10-16
The Ubuntu Server distribution, which is **text-only**, should run on it. You can use an Alternative CD or the specific Server CD.

Otherwise, you could upgrade the RAM. I believe Xubuntu Dapper can be run on 128 megs. Unless you're VERY confident with pulling apart and putting computers back together, you should take the computer to an Apple centre (or someone who is). There are only two RAM slots, capable of 128 megs each, and the CPU must be removed to access the lower one.
