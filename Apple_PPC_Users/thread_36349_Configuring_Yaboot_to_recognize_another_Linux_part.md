### Post by pvz on 2005-05-23
At the yaboot-promt, do a Tab to see other boot options. You have the following lines in your config: 
# This entry automatically added by the Ubuntu installer for an existing
# Linux installation on /dev/hda3
label=hda3-Linux
which means yaboot noticed your debian install on /dev/hda3.
so after you da a tab at the yaboot promt, it should list hda3-Linux as an option. Type that at the promt, enter, and boot into Debian.. You can change the name of the label into label=debian or something. Don't forget to run sudo ybin after you made any changes in your yaboot.conf.
