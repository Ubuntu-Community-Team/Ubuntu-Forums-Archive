### Post by stream303 on 2008-01-26
I found the following book interesting since it was written by an author who has some PPC experience and has some examples including responses from a ppc iBook.  Although it comes nowhere near what we see in the forums, I did find the book interesting and useful.  It concentrates on the LTS 6.06.x Dapper release, and I found many things useful even though I don't run Dapper.

**Hacking Ubuntu - Serious Hacks, Mods, and Customizations**
by Neal Krawetz
Wiley Publishing Inc, 
ISBN 978-0-4701-0872-7
Copyright 2007

If I were to go back to Dapper now, I'd pick up the "alternate" **6.06.2** install for my desktops from the mirrors:
[http://www.ubuntu.com/getubuntu/downloadmirrors](http://www.ubuntu.com/getubuntu/downloadmirrors)

For me, it's not the end-all, be-all book on Ubuntu, but no book is.  Might be worth a look especially if you are running on a low-memory system.
