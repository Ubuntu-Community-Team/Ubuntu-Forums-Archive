Also, is this a CD that you burnt yourself or one you requested from Shipit? If it's one you burnt yourself, check that you burnt it as a disc image, just as a single file on a data CD.

### Post by gfer66 on 2007-04-24
Remember that you need at least 256 MB of RAM Memory to run Ubuntu. If your B&W G3 has less, you must use the [alternate install CD]("http://cdimage.ubuntu.com/ports/releases/7.04/release/ubuntu-7.04-alternate-powerpc.iso").
