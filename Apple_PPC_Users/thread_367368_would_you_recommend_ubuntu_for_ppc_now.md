### Post by maggot_brain on 2007-02-22
While I would wholeheartedly and unreservedly recommend Ubuntu on X86, the same can't be said for PPC.  Ubuntu has dropped official support for PPC and the mac keyboard layouts for the x-window system are broken in 6.10 (Edgy). Personally I would recommend Debian Etch, which I am currently using, Ubuntu is based on Debian.  You can share data between OSX and Linux by using a HFS non-journalled partition if I recall correctly.  Also if you have the latest OSX you can resize partitions within it to create space for Linux.  If Debian seems a bit daunting for you, Fedora (basically Redhat) and OpenSuse (basically Suse) do up to date PPC distributions.  I've tried OpenSuse PPC but found it to be incredibly slow on an iBook G3 700 mhz.  I've not tried Fedora though.  The Debian Etch install is in my opinion no more difficult than the Ubuntu alternate cd text mode install so give it a go.

I hope this is of some help!

Ananda

### Post by 3rdalbum on 2007-02-22
The implications of it not being supported by Canonical: No paid support, no security updates. You still get new versions thanks to the community team (and because Canonical still do compile for PowerPC to help the team). PowerPC versions might be released a little while after the x86 version, just like with Xubuntu.

Airport is workable with Ubuntu, and although I don't know if Apple's preferred Bluetooth chipset is supported, I *do* know that general Bluetooth support is included in Ubuntu.

Yep, it's possible to access the OS X partition from within Ubuntu; don't worry.

As an aside, there is a Mac-specific Linux distribution that is currently being developed called Copland - yes, this is my own project :-)   When it's released I'll post the announcement to this forum, so stay tuned! I'm sure Debian will be a better choice at the moment, due to stability, but before long I'm hoping that Copland will become a good choice due to better features, and concentration on what Mac users want and need.

### Post by arbulus on 2007-04-04
I have Ubuntu Edgy dual booted on my iMac G5, and it runs wonderfully.  The only problems that I have are no Adobe products at all (Flash, Acrobat), and I don't think Java apps run either, but I may be wrong.  Anyway, it's been really great and it was a lot of fun getting it set up, and for most things, I would recommend it.  However, because the official support for PPC is being dropped, and there is just generally hit or miss compatablity on PPC in general, then I would typically say no.  If you have a spare PC or don't mind setting up a dual boot on a PC you have, I would say go for that.  Simply because from what I've found in my searches trying to find fixes etc, apparently Linux on a Mac is just an oddity.  You can't use Wine, you don't have alot of multimedia support, and video drivers just don't exist.  I found that nVidia does not nor will they have accelerated video drivers for PPC Linux.  Which is a shame.  So that means boggy 3D graphics (my screen savers are absolute resource hogs!), and no 3D desktop (Beryl or Compiz).  

So, I guess my conclusion would be, if you don't mind the lack of multimedia support and aren't really concerned with high level graphics, then go for it.  Especially if you want to run a machine as a file/print server or web server.  If that's the case then I would recommend it wholeheartedly, but I just wanted to add the caveat so that you can know what you're getting into.
