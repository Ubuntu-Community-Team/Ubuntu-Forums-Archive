### Post by pauljwells on 2006-11-15
There are some posts already about single boot ubuntu on macintels. I haven't tried it myself (still waiting for my macbook pro to arrive :D )

However, you CAN keep your machine single boot ubuntu, if you first install OSX and use a program called Carbon Copy Cloner to make an exact image of your OSX install on an external firewire drive.

Whenever you need to update firmware (which isn't very often: I've done it only once and I'm on my sixth mac) you can plug in the firewire drive and hold down the option key when powering up to get the mac OS selection menu.

Once you make the firewire copy you can erase the machine HD and start your ubuntu install...

Good Luck
