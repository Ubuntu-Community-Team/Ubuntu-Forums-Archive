### Post by jaradronald on 2006-10-17
Since it's a G3 you may want to try using the Ubuntu 5.10 Install Image. Some of the modular support for G3 harward drivers has been removed from 6.06 (to focus more on the G4 & G5 support). I have a G3 Powerbook (Old World Wallstreet) that is running 5.10 beautifully.

It looks like there's no Power PC Distrib of Edubuntu 5.10, but you may try going with 5.10 and then you can install the Educational Software Packages on top of the base system. The standard install will take up 2 GB of your hard drive, so you may prefer to perform a "server" install (or base system install without a GUI) and then you can use apt-get to get the xubuntu package and whatever educational X Apps you want.

I'd agree with finding a cheap upgrade for your Hard Drive (even 10 GBs should be sufficient), as well as maxing out your systems RAM capacity. Both are so cheap (if you can find ones that work with your system) that it's well worth the investment.
