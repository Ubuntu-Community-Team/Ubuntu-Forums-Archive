### Post by guidop on 2007-12-12
Xubuntu 7.04 works OK on my 400 MHz iMac DV with 640 MB RAM.  It has just enough processor throughput to play DVD's.  I can play some flash ads with the gnash plugin, but it really slows the browser.  I have not been able to get YouTube videos to work.

A clean install using the Feisty (7.04) PPC alternate install CD should get you a working desktop.
If you only need to boot Linux or Linux + OS X, this should work best.

On my iMac, I performed a server instal using a Dapper (6.06.1) PPC alternate install CD to be able to partition my hard drive and still boot OS9.  I changed my /etc/apt/sources list, did a dist-upgrade to Feisty, installed the XFCE desktop - but had to manually edit the /etc/X11/xorg.conf file to get X to work.

This thread might also help:
[http://ubuntuforums.org/showthread.php?t=495347]("http://ubuntuforums.org/showthread.php?t=495347")


Also, it's recommended that you burn the install CD at the slowest possible speed.  It seems to help when installing on machines with older optical drives.
