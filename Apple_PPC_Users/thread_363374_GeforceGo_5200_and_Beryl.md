### Post by grazie on 2007-02-16
To use Beryl you need a card that supports hardware-accelerated rendering and a matching driver. There are no PPC accelerated drivers. So no, you can't really run Beryl... no great loss IMHO. However, it also means you can't run Google Earth either (even if there was a client), which is a great shame.
