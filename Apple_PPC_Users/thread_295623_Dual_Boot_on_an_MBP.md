### Post by frog_pilot on 2008-03-28
first read this: [Announcement:Intel Mac Users Please Read First ]("http://ubuntuforums.org/showthread.php?t=185305#1") its sticky in this section.

Ubuntu on intelmac is the same as ubuntu on every x86 platform. You have to use the standard i386-version of ubuntu.

As a matter of course you can create different boot-volumes on your internal harddrive and you can not only run OSX and Ubuntu, you can also run windows on these machines.

I think bootcamp is the usual  bootmanager for intelmacs. It will give you options to boot into windows or OSX and you can also boot in ubuntu, if its necessary.

[Link]("http://wiki.ubuntuusers.de/Apple_Computer") to a german howto.
