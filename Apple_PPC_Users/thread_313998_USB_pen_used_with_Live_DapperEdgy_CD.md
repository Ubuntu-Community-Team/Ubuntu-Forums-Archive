### Post by Uta on 2006-12-06
I boot my Live PPC Dapper CD using the "persistent" method to enable my USB pen drive, so my changes, documents, drivers, downloads etc, can be saved. I used the same USB pen via the persistent boot method and it worked perfectly with the Live PPC Edgy CD. Good way to try out the Edgy CD with an instant working wireless connect!
