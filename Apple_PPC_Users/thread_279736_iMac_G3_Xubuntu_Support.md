### Post by DirtDawg on 2006-10-19
> **r6mile said:**
> So the normal Xubuntu LiveCD doesn't work on G3's? Do I have to download the Alternate CD?

Not exactly. The live CD will not run very well on an older computer and can lock-up or freeze. The alternate CD has a text based inistallation (instead of GUI based) that requires far less resources and will be a much more smooth install.

So, yes you can use the live CD to install Xubuntu on your G3, but the alternate CD is recommended. :KS
