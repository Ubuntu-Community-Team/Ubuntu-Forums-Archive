### Post by por100pre1 on 2007-10-26
The **Advanced Desktop Effects Settings** is a must see for desktop effects:

```
sudo aptitude install compizconfig-settings-manager
```

[**Deluge**]("http://deluge-torrent.org/News:Portal") is a great BitTorrent client:

```
sudo aptitude install deluge-torrent
```

**Startup Manager** is a Grub and USplash screen configuration tool:

```
sudo aptitude install startupmanager
```

Not sure if they are available for Apples :(
