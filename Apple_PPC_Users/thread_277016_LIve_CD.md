### Post by meng on 2006-10-13
Desktop CD is the live CD, it boots to a graphical desktop which you can play around with, then start a GUI installer that installs Ubuntu to the hard drive.

Alternate CD goes straight into a text-based installer, which installs Ubuntu to the hard drive (same end result).
