### Post by chudleydog on 2005-05-06
Doing the tiger upgrade (boot from CD, follow the prompts), results in dearly beloved Apple overwriting the boot loader and the blasted machine now boots straight into OS X (G4 Titanium Powerbook) (previously dual boot Ubuntu / OS X).

---

### Post by khc on 2005-05-06
Hold the option key when you boot, then choose to boot from linux, then run 'sudo ybin'
