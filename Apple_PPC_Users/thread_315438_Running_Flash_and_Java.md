### Post by stream303 on 2006-12-09
At this stage, video card manufacturers aren't releasing specs nor binary blobs for the PPC platform to run 3D effects or intense graphics.  Who knows what the future will bring, but it doesn't appear likely.  The same goes for flash.

What we do have is a solid 2D, non-flash PPC distribution.  Unfortunately, 3D is capturing the interests of users and developers to the point where PPC is considered boring.  Add to that the lack of PPC sales and you have a platform headed for being a port for enthusiasts instead of the mainstream.

That being said, there is still a LOT of value in the current Dapper and Edgy PPC distributions.
