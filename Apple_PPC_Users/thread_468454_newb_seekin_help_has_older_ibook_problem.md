### Post by DirtDawg on 2007-06-09
Is there a reason you're using 5.10? That's a very old version and whatever problems you are experiencing may have been addressed in more recent versions.

I would strongly recommend you try [Xubuntu]("http://www.xubuntu.org/") at version 6.06 (Dapper) or later. I also recommend you use the "Alternate Install" CD, which uses a text-based installer and will likely run much smoother.

Good luck.
