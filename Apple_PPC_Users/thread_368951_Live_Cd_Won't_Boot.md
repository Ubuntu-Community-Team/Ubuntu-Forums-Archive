### Post by 3rdalbum on 2007-02-24
Checklist:

1. Are you sure you've got the Live CD for PowerPC? If not, download/order the one for PPC.
2. When you insert the CD that you've burnt, does it appear on the desktop? If not, try burning at a lower speed or onto a better quality disc.
3. When you double-click the disc, does it open up with a number of files and folders, or just one file with a .iso extension? If it's just one file, then you've burnt it incorrectly - you must tell your CD burning software to burn it *as a disc image*.
4. Are you holding down the C key as you start up? If not, then do it.
5. If that doesn't work, try holding down Command-Option-O-F as you start up. When the command-line appears, wait a few seconds and then type "boot cd".
