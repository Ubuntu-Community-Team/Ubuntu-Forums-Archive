### Post by stmiller on 2006-09-04
I run ubuntu on my iBook G4. It is EXCELLENT. Everything works, even the volume/brightness/eject keys at the top of the keyboard. Sleep works, wireless works if you follow the steps for the broadcom card. 

And make sure to install the network-manager. Makes it as easy to join access points as in OS X. Just click the icon and choose the network you want.

After booting to the gnome desktop, ubuntu is only using 84MB of memory. So yes, it is much faster than OS X.

And mac-on-linux works very well, if you read how to configure that. 

The downside is that there is no flash for PPC Linux, so you cannot watch you tube, etc. And I have found that mp4 video does not playback well in PPC Linux. AAC playback does work fine, however.

There IS a realplayer for PPC Linux which works well. And most windows media type content plays with mplayer.
