### Post by colinhayes on 2007-11-08
Is 10.4 still on any partition in your computer?  If so, boot that up, insert the cd, and try launching the installer from there.  It (should) reboot into the cd and start the installation process.  At least it did for me. 

Of course after you install you'll have to reboot holding down option, boot into linux and then run ybin if you want your old bootloader back.
