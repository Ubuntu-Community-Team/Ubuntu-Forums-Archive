### Post by tiiim on 2005-01-04
i have!


I way prefer Ubuntu to YDL 4. YDL was a nice distro and all and it inspired me to get back into Linux. However Ubuntu detected by monitor and graphics card perfectly. In YDL i had to run X in "safe mode" to manually sort it out even then the res and colours were lower. Also YDL (at time of writting) also switched the sound modules to another system the only draw back to this was there is no sound at all in YDL! (it is a documented common bug too!) While when Ubuntu installed i heard the friendly sounds of the drums then when logged in the nice loading sound which i knew instantly Unbuntu was going to be very nice.

I used be a KDE fan and not like GNOME, but the new GNOME desktop has now made me rethink my whole desktop and now loving GNOME.

YDL is RPM based eveb though very supported you still have the ol' missing files probs while debs are far superior. 

Even though YDL is great IMHO i think Ubuntu wins i had min probs with Ubuntu on my system while YDL came with issues (some still not fixed) and so for someone moving from say, OS X, Ubuntu will jump out and grab more than the soundless YDL desktop.

I think the Ubuntu developers have done a superb job on the PPC side and keep it up and you will still start stealing people from the YDL camp as word gets around. YDL 4 didnt make a huge impact on the Linux world while Ubuntu has exploded off to a great start... so excellent.

Obviously this is my experience on my computer others may not have such a please Ubuntu experience as i had. But Ubuntu is deff one of the best Linux distro's i tried so far and being Debian based its brining the power of a great distro yet so far away from the 'common user' yet Ubuntu brings the advance usage and user friendly side all in one place. IMHO Ubuntu steals the show easily. Im not saying this as a Ubuntu users the  reason is cos' i was (REMEMBER) a YDL user before i tried Ubuntu and it won me hands down. =D> 

YDL has a lot of work to catch up with Ubuntu the only thing that stops Ubuntu is YDL large user base but if Ubuntu carries on the PPC side of Ubuntu will grow.

### Post by chascon on 2005-01-06
I agree Ubuntu-ppc is better than most others at figuring out hardware and configuring.  I found ydl4 very slow on my 400mhz iMac slot loading machine.  Ubuntu is faster, perhaps even with accel disabled.  ydl 4 configured fine for me on this version (I've had issues with previous).  I think ubuntu-ppc has more software too.  I remember being frustrated at how few browsers were available for ydl4.
But what I think makes ubuntu better is their attitude.  ydl4 is commercial based in your face.  If you want to test their betas for them you have to be part of a pay club.  Imagine having to pay to put your hardware at risk!  They should be paying you.  The final releases are restricted from public release untill a time they deem it ok to release it publically.  Greedy?  On this matter I'm with the debian crowd (they do not do this).  I'd be using sarge right now if ubuntu didn't have xsane working properly (and helping out reporting some bugs that are going to make it into debian anyway).  Debian does have tons of software going for it though.  And ubuntu is debian anyway, so I'm fine with that.

### Post by netsec on 2005-01-14
I have used YDL 3 and 4.  YDL 3 worked great, and YDL 4 has the issues others have discussed.  If there is one thing that I like about YDL over Ubuntu (and I think I will be sticking with Ubuntu) is that it detects and tries to set up both the wireless and ethernet connections.  If it fails to get a link, it just says it failed.  

I have yet to get my wireless card activated on Ubuntu and I assume it is because I do not have an access point at home.  Seems logical, but when I boot up I am told that there is no wireless card present.  YDL at least tried to find a signal even when I was using my wired connection.
