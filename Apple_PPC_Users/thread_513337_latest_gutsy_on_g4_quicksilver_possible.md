### Post by ajmctaggart on 2007-07-30
I understand one's desire to run the most bleeding edge available, however, it's in your best interest to run a feisty install (alt disk is best) if you plan on using this machine.  If you just wanna play, then I would keep going with Gutsy.  But, I really think Feisty will give you a more usable install.  Remember what is called 3rd Alpha is optimization on the x86 platform, meaning that PPC optimizations probably hasn't even begun on Gutsy.  I could honestly say that it is more like compatible not optimizing...


On another note, see how far you get with the alt install disk, these can be found by googling 

UBUNTU PPC FAQ

The link should be on the 2nd page (of your screen...)

good luck,
