### Post by joshuapurcell on 2006-09-07
Are problems that come up on the new Intel-based Apple systems really similar in any way to using Ubuntu on the older Power-based computers? To put it another way: Do Intel-based Mac owners use a kernel that is similar to the x86-based install but just with a different application to interact with the firmware, or is the kernel more like the PPC version used on older Macs? I have to think Intel-based Macs use the x86 software, and therefore have x86 problems.

I know the Intel-based Apple systems use a type of firmware (which is similar to older Macs), but other than that the hardware is very similar to existing x86 systems. I assume that these new Macs use a kernel and associated packages that are similar (if not identical) to the ones available on the regular x86 Ubuntu install CD, but with a different type of bootloader. Let me know if this isn't the case.

At the same time, it's worth pointing out that there are other Power-based systems (like Genesi's ODW and other computers) that benefit from having a forum that focuses on supporting this type of architecture.

If what I'm saying is right, then would it make more sense to have the Intel-based Mac users look to the x86 sub-forums for support? Should this sub-forum start supporting two different architectures, both of which are using different Ubuntu software?

Throw all this out the window if PPC-Mac users and Intel-Mac users both use the same exact Ubuntu software... that would at least give one relevant thing in common to the users here.
