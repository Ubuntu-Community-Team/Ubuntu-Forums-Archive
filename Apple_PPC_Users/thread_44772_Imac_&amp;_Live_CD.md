### Post by jdp on 2005-06-27
I've booted it just about flawlessly on my 350MHz Slotloader w/ 256MB RAM.  It was as simple as using 'boot cd:,\install\yaboot' in Open Firmware because the OF settings OS X made didn't like to boot the CD by just holding 'c' at startup.  If you haven't, you probably want to install the latest firmware update too, as that can solve many problems on old iMacs, esp. if you ever want to install OS X.
