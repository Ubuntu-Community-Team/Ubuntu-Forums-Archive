### Post by Rinnan on 2006-06-19
I hope this isn't too far off-topic, on my Dual USB G3, everything works now.  It never hung, but it didn't sleep correctly or wake.  Now that works perfectly.  Only two problems left:  On install one must do manual editing of the xorg.conf file (HorizSync must be added) in order to get it to do the native 1024x768, and there is some corruption on buttons of various kinds (any push button might have streaks of color and distortion running through it, seems to effect only greyed buttons and is fixed on mouseover.  These are relatively minor problems compared to the full set of problems I had on Dapper release, so I'm very, very happy with Dapper on PPC.  Other than those two, the only other problems are problems that effect all Linux PPC, like no beagle :(
