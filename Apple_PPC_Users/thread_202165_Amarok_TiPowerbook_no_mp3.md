### Post by N8K99 on 2006-06-23
It is solved. I found this link deep within the wiki. [http://packages.ubuntu.com/dapper/libs/libxine-extracodecs](http://packages.ubuntu.com/dapper/libs/libxine-extracodecs) 

download the package and then using the right click you should have an option to install the package in kde. 

Voila! now, I can listen to my mp3s no matter which OS I am running!
