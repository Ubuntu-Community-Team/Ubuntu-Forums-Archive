### Post by 3rdalbum on 2007-01-17
No, sorry. You'll have to gather some developers together and write one yourself.

EDIT: There are emulators (SheepShaver can run Mac OS 9, PearPC can run OS X), but you still need to run the operating system emulated, and you also need a copy of a New-world Macintosh's ROM. How you get that onto Linux without losing the ROM file's resource fork is beyond me :-)
