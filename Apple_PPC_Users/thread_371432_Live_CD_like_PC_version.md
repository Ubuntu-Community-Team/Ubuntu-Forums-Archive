### Post by tgalati4 on 2007-02-27
Hold the C key until the apple logo shows up.  You will hear the disk spinning as it is read.  The 6.06.1 works well on a G4 15" 1 Ghz powerbook.  Wireless networking works, sound switches work, brightness works.  Lost 2 finger scroll, but I think you can use edge scroll instead.  

I use Tiger 10.4.8 with XDarwin and Desktop Manager.  I have 8 desktops set up.  One of them runs XDarwin and inside of that I have 4 Xwindow's sessions running various Gnome programs.  Works pretty well and you can keep iTunes!

I have found Tiger to be bullet proof over the past 2 years.  I only need to reboot once every few weeks to defrag the memory and restore performance.  There is some slowing down as memory gets ragged.

Good luck.
