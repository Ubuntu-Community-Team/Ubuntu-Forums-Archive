### Post by FierceBob on 2006-08-05
Cobbwebb -

First, let me say that I am new to Ubuntu (and not extremely experienced with Linux in general), but I do know Macs and OS X.

I think you're going to have problems with trying to run OS X inside of Linux inside of a VM on that machine.  The Mac OS expects to see very specific hardware, and if you're running inside a VM it won't see what it wants.

I believe that a dual-boot system is possible, though I don't know whether using the Linux boot manager is going to be the best way to go.

Sorry I can't give you advice on how to accomplish what you want, but you will save yourself a lot of grief by not trying to use the VM option.
