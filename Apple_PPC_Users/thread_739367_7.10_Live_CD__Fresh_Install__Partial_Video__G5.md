### Post by stream303 on 2008-03-30
> **MightyWhighty said:**
> I have the same issue with my screen wrapping on the right side.  Was wondering what steps you took to 'back the driver down' to feisty's

Take a look at the bug report I filed:

[https://bugs.launchpad.net/ubuntu/+bug/189844](https://bugs.launchpad.net/ubuntu/+bug/189844)

I did it in an unusual way at first.  I reinstalled Feisty and grabbed the file
**nv_drv.so**
from this directory:
**/usr/lib/xorg/modules/drivers**

and copied it to a usb-key.  Then I installed Gutsy, and when it was finished, I copied the old nv_drv.so file back into the same directory on Gutsy.

I also made sure that the permissions were just like the original.

Or you could pick up that file manually from here - just be sure to use the PowerPC version:

[http://packages.ubuntu.com/feisty/x11/xserver-xorg-video-nv](http://packages.ubuntu.com/feisty/x11/xserver-xorg-video-nv)

The problem is that if any updates come down the line, it will be overwritten so there is a way to grab the old file, pin it, and it won't be updated.  I don't think that at this stage of the game there will be any updates, so I'm just keeping a copy of the driver handy to copy back over just in case.

It's not the elegant way to do it, but it works. :)
