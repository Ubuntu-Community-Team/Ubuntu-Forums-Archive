### Post by DirtDawg on 2006-09-28
Kubuntu will probably not work. You should try [Xubuntu]("http://www.xubuntu.org/"). It uses XFCE instead of KDE, but it's tailor made for small machines. Even then, with 64MB of RAM you're cutting it close.

### Post by anurse on 2006-09-28
I agree that text based install (alternate CD) is the way to go, but I'd go for something even less that XFC. You could try something like the ICE WM and be careful with the programmes you want to run. A programme that uses a fair amount of memory (say, like Open Office or Firefox) might work but it will be very slow.
