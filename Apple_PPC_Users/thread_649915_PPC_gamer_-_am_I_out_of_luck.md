There's probably even fewer games on PPC linux than mac PPC :( you can always run them through an emulator but it will most likely be way to slow. ### Post by stmiller on 2007-12-26
> **Cienna said:**
> 
i looked for emulators and find many, but none of them seem to run on PPC. i've read something about that being because PPC has native multiboot (?) but i've also read it's dog slow, which obviously isn't going to help me. 


Anything trying to run x86 code on PowerPC will be very slow, because it is having to translate every line of code on the fly from x86 instruction to PPC. And there will be no 3D support in any emulation- not enough to play any 3D game, anyway.

If you have an ATI card in your Powermac, the Linux games should work fine (neverball, etc.) but I know this perhaps doesn't help. :(
