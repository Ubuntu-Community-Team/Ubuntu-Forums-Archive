### Post by guidop on 2008-01-05
I think the problem is that you are trying to boot an OS X install disk for an Intel processor machine (MacBook) on a PowerPC machine (PowerBook G4).  The OS X Leopard (10.5) install DVD will boot both architectures, but I don't believe the Tiger (10.4) disks will.

I think you will need to get your hands on an OS X install disk for PowerPC to be able to boot your machine.  Do you have the original CD or DVD install disks that came with your PowerBook?  Failing that, you might still be able to find a left-over or used **retail** version Tiger install disk for less than the cost of the new Leopard version.
