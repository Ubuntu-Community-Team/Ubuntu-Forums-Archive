### Post by stmiller on 2006-09-20
Okay the good and bad:

 If your powerbook has nvidia video chip, sleep/suspend will NOT work. If you have any ati chip, it WILL work. I realise this might be the make it or break it aspect.

It is fairly easy to set up a dual boot system. And mac-on-linux will let you boot up your OS X half in Ubuntu. So you can run/access it the same as if you were booted into OS X. Though sound doesn't work that well. So don't think you can run Reason or anything like that in MOL. But MOL speed is fast, and MOL doesn't take hardly any CPU at all. 

Wireless: wireless works with the broadcom driver (airport extreme), though a few short steps of extracting the apple driver firmware is required to getting it to work in ubuntu. Make sure to install gnome-network-manager then you can choose access points from an icon at the top right of the screen, similar to OS X. No configuration needed.

As far as applications, the ubuntu servers have LOTS of stuff already. And you can browse things to install by starting the program synaptic.

Or browse to see what is available here:

[http://packages.ubuntu.com/](http://packages.ubuntu.com/)

There is LOTS of stuff. And with synaptic, it's a simple check box of what you want, then 'apply' to install.

Of course what you decide to do is completely personal. OS X is very nice indeed. Good luck!
