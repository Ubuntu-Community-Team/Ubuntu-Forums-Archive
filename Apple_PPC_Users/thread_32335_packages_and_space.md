### Post by ssam on 2005-05-07
you could probably fit the basic install in 2 or 3 gigs. but if you plan to install some more apps, and store any documents you might wan to use atleast 5 gigs.

you will also need a swap (virtual memory) partition. this is generally recomended to be between 1 and 2 times the size of your ram. 512mb is probably enough in most situations.
