### Post by slux on 2005-01-19
I'm running Ubuntu on a '99 Clamshell iBook with a 300MHz G3 and it's working just fine. (well, it would on a 300MHz x86 machine as well I guess)

I don't think it's that slow at all, I can run OpenOffice.org and everything just fine.

The RAM amount is crucial though. I used to have 96MB on it and it did work but things go very swappy and especially using OpenOffice.org is a major PITA. 128 might help a bit but in my experience that is still a little low for a GNOME desktop. I've currently got a 256MB DIMM installed on it and that's what makes all the difference. It's amazing that these machines shipped with 32MB when they were new and can still run the latest GNOME desktop very well.

In theory the system's supposed to be a dream when it comes to hardware support on Linux, power management and all but after installing ubuntu and switching to 2.6 with it there have been some glitches I've not been able to fix yet. The battery life's pretty good for a laptop this old, I guess many of the clamshells will have almost worn out batteries but I'm still getting around 4h at best from mine.

Even old Apple hardware does tend to be a bit expensive, I've noticed. I'm sure you'd get a similarly powered x86 laptop cheaper. I guess Apple users don't like to let go of their systems that easily. ;)

### Post by Viro on 2005-01-19
[QUOTE=matthewrevell]What's Dri?

Is it speedy enough to use as a second machine?[/QUOTE]
 DRI is the Direct Rendering Infrastructure, which basically allows 3D acceleration in Linux. DRI doesn't work on newer cards, but it should work find on the Clamshell iBook.

As for the speed, that's variable and it highly depends on what you're expecting in a machine. For my needs, OpenOffice is still slow even on the Powerbook 12" 1.33 GHz with 1.25 GB of RAM. It takes a long time to load, and I basically don't have the patience to wait for it. That's why I use Abiword, but that's another issue altogether.

For the work that I do, writing loads of simulation code in Java and C, I find my machine just 'bearable'. If you're doing light work and don't expect fast responses from your machine, YMMV. You might even be satisfied with the iBook G3 300 Mhz.

Apart from ebay, I don't know anywhere else to buy second hand macs. Sorry.
