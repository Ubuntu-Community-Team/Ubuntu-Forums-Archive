### Post by ajmctaggart on 2007-07-13
No, not at all...

Here's the deal- if you're laptop's cd drive is broke, you could either use firewire target disk mode of CERTAIN newer macs to use that mac's cd drive as an external firwire cd drive ( I personally don't know why this works on some, and not on others but it does)

The other BETTER solution is to find a friend WITH a mac laptop that has a working cd drive (powerpc based of course) install ubuntu w/o any specifics for that machine, i.e. dont go setup custom resolutions or install any support for your' friends laptop.  Then just move the HD back into your machine.

I have a Powerbook Pismo and a Titanium Powerbook, the Pismo won't boot off of CD cause I've hacked it up a bit (don't know what happened there) so I installed onto my HD of choice via Tibook, then swapped it back into the Pismo, worked like a charm!

Just an idea, Im not so good w/ Network installs etc...

But Debian does explain how to grab these 4 files and start the network install, prettty straightforward if you google "Install Debian from Network" that should work for you.

Good Luck
