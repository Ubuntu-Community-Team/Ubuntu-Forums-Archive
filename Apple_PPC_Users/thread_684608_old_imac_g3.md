### Post by Midwest-Linux on 2008-02-02
I have a old IMAC, there are different types. Some of these older IMACS have only like 32 MB RAM and you really need to upgrade the RAM before installing Ubuntu. I have both the tray loader and slot loader.




The Tray loader usually has SODIMM RAM (which is the same RAM as older Windows type laptops uses) and you need to open up the computer, and it can be quite tricky.

[http://docs.info.apple.com/article.html?artnum=43012](http://docs.info.apple.com/article.html?artnum=43012)

[http://docs.info.apple.com/article.html?artnum=43013](http://docs.info.apple.com/article.html?artnum=43013)




If you have the newer type (slot loader) you can easily replace the RAM (which is the more common PC 100 type) on the bottom hatch of the computer.

[http://docs.info.apple.com/article.html?artnum=95144](http://docs.info.apple.com/article.html?artnum=95144)



If you are not sure what G3 IMAC you have (there are different types) ...start here.

[http://www.welovemacs.com/apotimflscme.html](http://www.welovemacs.com/apotimflscme.html)


I have replaced RAM in both machines and installed Ubuntu 7.04 using the ALTERNATE TEXT INSTALLER PPC version. Just remember to burn it into a ISO using the lowest speed available. 


I have found that a minimum of 256 MB is needed to run Ubuntu, more the better. While PC100 memory is fairly inexpensive, SODIMM's are fairly reasonable..though a bit more money. Check E-Bay for some good deals.
