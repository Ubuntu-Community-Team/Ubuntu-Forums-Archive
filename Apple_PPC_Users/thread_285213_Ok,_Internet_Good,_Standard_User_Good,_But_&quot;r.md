### Post by 3rdalbum on 2006-10-27
Probably the easiest thing for you to do now is reinstall, and this time use the normal install rather than the Expert install.

Specifying a root password during the Alternative CD install is a bad idea unless you're a Linux expert. You only get that option when you do an expert install. Just install Ubuntu again straight over your Ubuntu partition, using the ordinary easy install mode.

By the way, Adobe have not made PowerPC Linux versions of Flash and Acrobat Reader. Ubuntu comes with a PDF reader and a video player, but I recommend also installing GXine and Mplayer.
