### Post by 3rdalbum on 2006-12-20
Yep, ISO files are exactly the same format whether they are intended to be used with PowerPC or with x86. (or MIPS, Sparc, ARM, etc...)

You might be getting confused with disk image formats that are used by the Macintosh operating system. Most of the time when you see a disk image that's meant to be mounted within Mac OS, it's in .dmg format.
