Hello there and welcome! There is a kernel in Linux, as well as in OS X and other operating systems. What I think you are referring to is the terminal. While there is a terminal in windows, the linux terminal is really advanced with lots of features. There are several shells to choose from when using the terminal, the most common (I think) is called Bash. Note that this type of terminal also exists in OS X.

The kernel is the part of the OS that communicates with the hardware. So it contains drivers, information about filesystems and so on. On top of the kernel we have an X server, which handles graphics. On top of this one can have a desktop suite, such as Gnome or KDE.
Gnome and KDE in turn, have lots of programs such as browsers, office programs and so on. Some of these programs are specific to Gnome/KDE, while others can be used independently of desktop.

The components I have mentioned also have their counterparts in other OSes, but one thing that makes Linux stick out is that it's very modular. So you can choose all these parts, but it may be difficult to change if you're not an advanced user.

My advice is to start by testing the programs that come with Ubuntu. Also, have a look in Synaptic to see what stuff is available by simply clicking! Then, check out ubuntuguide.org for many great tips!

Good luck!
