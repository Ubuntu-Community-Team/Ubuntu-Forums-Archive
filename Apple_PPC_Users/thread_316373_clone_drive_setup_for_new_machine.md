### Post by mike3k on 2006-12-10
You can create an image of the entire drive with Disk Utility (at least in 10.4; I'm not sure if it works in earlier systems). In the disk utility window, select the main disk icon (disk0), *NOT* one of the partitions on that disk and click new image. It will create an image of the entire drive with all partitions. You can then restore that image.
