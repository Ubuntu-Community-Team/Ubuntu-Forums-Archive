### Post by DjKritical on 2007-06-12
Well it's hard to guess exactly what is causing this,

However here are a few things for you to try:

- Try and press CTRL+ALT+F1 ... this should bring you to a Virtual Console, from here you should be able to execute commands.

- Try and boot into Single User mode, this will log you in as root and you can start checking logs and such...

-  Check that capslock still works, generally if capslock stops working you're probably going to need a reboot
