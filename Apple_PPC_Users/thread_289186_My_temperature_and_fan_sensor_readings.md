Ok, so I have Edgy working fine on my 20" rev-AG5 iMac - although had to manually load a power module to control fans.

At any rate, I'm keeping a close eye on fan speed, cpu temp, etc so I added the cpu frequency scaling applet, and also the Gnome "sensor-applet" (universe repository) to the top bar.

**Warning:** I had to install the power module manually *before* trying to add the sensor-applet - otherwise synaptic would just hang forever)

[http://www.ubuntuforums.org/showthread.php?t=284508](http://www.ubuntuforums.org/showthread.php?t=284508)

**At idle, the system just rests with the fans at 1500 rpm, 52C cpu temp (70F ambient room temp), at 900 mhz.**

Bringing up Gimp and trying to load the system does make all three of these monitoring points change, but I haven't really stress tested the system yet.

Not sure if I want to. :)  But at least I can keep an eye on things.
