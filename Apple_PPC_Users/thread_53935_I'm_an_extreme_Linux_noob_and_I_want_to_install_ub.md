### Post by dave9191 on 2005-08-02
That depends what you mean by "that hard". It can be very easy if you install it and everything works, but if you have to start fixing things yourself you might start running into trouble depending on your computer background. What machine you have also makes a difference as different machines have diffrent out of the box support. 

But more importantly, data on your computer. You should never start to experiment without having a backup of your data first. If you have to start repartioning you will need a full backup. If you start resizing partions there is always chance of data loss. It probably is possible to do it without data loss, but if you are new to this you might end up messing things up. USB hard drives and memory sticks are very cheap these days. It is well worth the inverstment. Even if you are not going to mess around wtih linux. Backups should be done regulary. I do one every day. Anything might happen, and hard disks are known to fail at the worst of times.
