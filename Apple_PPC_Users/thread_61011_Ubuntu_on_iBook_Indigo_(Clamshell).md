### Post by Rxke on 2005-09-11
Hi, Matei,

Never tried FluxBox, so don't know the answer, XFce works great though.

As for running it closed, it is possible, but surely not a thing you want to try. Cooling is partly via the keyboard, so you'd end up with a unstable system in the best case, hardware destruction in the worst.

Now... for other uses... Believe it or not but I use a clam every day. One additional usage is that it serves as a wireless connectionpoint for a set of non networked cmputers throughout the house. If I want those computers to have webacces, I just plug in an ethernet cable, using FireStarter firewall, and bingo, network connection (via iBooks Airport)

not blazingly fast, but more than sufficient.

### Post by SepheeBear on 2005-09-11
Fluxbox works great in Hoary PPC. Im using it with an iBook Graphite 466 w/ 192MB. It is ***much*** faster than the default GNOME desktop. GNOME was pretty much click-and-wait on this machine. Fluxbox made it usable again. XFCE Terminal is a good replacement for gnome-terminal since it launches a little bit faster.

I switched from using XFCE because of a video issue where the desktop would turn to green after playing video. That doesnt happen in fluxbox.
