### Post by stmiller on 2007-05-01
Adobe won't even make flash for x86_64 Linux, so for ppc linux it's going to be a stretch, unfortunately.

I do want to mention that you CAN watch flash videos (youtube) in ppc linux by doing the following:

- Make sure gnash is installed to avoid 'no-flash' redirects
- Use a bookmark from this page: [http://1024k.de/bookmarklets/video-bookmarklets.html](http://1024k.de/bookmarklets/video-bookmarklets.html)
and you can download the embedded flash video file (.flv) from youtube
- Open that .flv file with VLC. Enjoy your youtube.
