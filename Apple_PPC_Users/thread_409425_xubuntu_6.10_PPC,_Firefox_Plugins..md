### Post by Auria on 2007-04-14
The answer is unfortunately very simple : you can't

The makers of these only release i386 binaries, and since its not Open Source we cant recompile it

the only possibility left is to use open-source alternatives, like Gnash for flash. However it doesnt work very well, for me the only thing it plays about correctly is ads :( work is being done though

there is also swfdec, its an open-source flash player oriented at youtube. it plays youtube, which gnash doesn't, but i IIRC Gnash is still better overall

For Shockwave and Real, i don't know of anything that can play them (doesnt mean it doesnt exist, though)

### Post by 3rdalbum on 2007-04-15
There is a Realplayer for PowerPC, but only a few people have actually had success with it. Even if you do get it working, it does not play Realmedia files that are too old; so you may be out of luck there.

In theory, you can run the i386 version of Realplayer for Linux in Qemu; this is called "user-mode emulation", but I have not found anyone who has actually done this. If you attempt it and succeed, please send me a PM!
