### Post by effraie on 2006-09-07
Hello,

two french ubuntu-ppc users have just set up an ubuntu-ppc repository here : [http://mental-ppc.tuxfamily.org](http://mental-ppc.tuxfamily.org) (only french for now, soon in english too, other languages translators are welcome)

you can browse the rep here : [http://mental-ppc.tuxfamily.org/dists](http://mental-ppc.tuxfamily.org/dists)

Aim of the rep is to provide packages for ubuntu wich are not in official ppc repositories.

to use it, edit /etc/apt/sources.list and add :

```
#Mental-ppc
deb http://mental-ppc.tuxfamily.org/dists dapper-mppc all 
#deb-src http://mental-ppc.tuxfamily.org/dists dapper-mppc all 
```

ther are just few packages available now, but we hope increase them soon, help is welcome.
