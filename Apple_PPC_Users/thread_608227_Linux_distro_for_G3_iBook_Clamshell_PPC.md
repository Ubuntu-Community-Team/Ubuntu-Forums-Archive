On a system with so little ram a custom install and would probably be best. If you want later features you need more ram to run a 2.6 kernel. 
If you stay with 64 mb Slackingtosh or debian with a 2.4 kernel are probably the way to go, both are easy to do minimal installs. Even then you would need to run a window manager to have anything like a nice user experience.
 The problem is if you a new linux user using a laptop with an old kernel you may struggle to set it up with things like wifi, suspend.etc, if possible at all.
If I was you I'd get more ram. Install ubuntu 6.06 from the alternate cd, even then run a window manager. learn a little then upgrade.
If I was  you and choose to stay with 64mb  Then I,d do a debian net install and use this forum along with the debian forum to find any information you need to set up your system. This is the best place to for powerpc linux (apart from maybe gentoo, though gentoo it's not a new user  distro).  95% of info here  is relevant  to a debian install.  
Re: Slackintosh i always found it easy to install on a desktop, but although fast and maybe the best and most suitable for low memory systems. If your a new user, the set up  you need may be beyond you.

64mb is not a lot, just my 2c
