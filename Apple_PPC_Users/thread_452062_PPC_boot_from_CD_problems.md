In Nero, did you use the option "Burn a Disc Image" in StartSmart?

That's the way to do it; don't "open" the ISO and don't just burn it as a solitary file.

I think the 400MHz iMac's CD drives are okay, but you might want to try burning to a "music" CD-R or even a CD-RW for greater reliability.

### Post by Bleyze on 2007-05-27
I had the same problem trying to Boot the Ubuntu Live CD on my Mac Mini. I'd hold down the "C" key and it would still boot off the Hard disk. I downloaded it 4 times assuming it was a corruption somewhere.

In the end I tested the OS X CD that came with the Mac. I couldn't boot off that either. The problem was my keyboard. The keyboard had the standard green socket on it and was connected with a USB to Keyboard socket adapter. Replacing that keyboard with a (borrowed) keyboard with a built in USB connector on it allowed me to hold down the "C" key and boot off the CD and install.
