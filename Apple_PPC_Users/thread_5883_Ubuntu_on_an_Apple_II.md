### Post by slux on 2004-11-23
If you are referring to the truly ancient real Apple II, then certainly not. 

If you mean one of the Apple Macintosh II computers, the answer for Ubuntu specifically is still no (and I believe also for the original Macintosh II which has a 68020 processor)  but at least some of the Macintosh II series computers do work with the m68k version of Debian which Ubuntu is based on. And on a m68k computer you'd hardly notice the difference between Debian and Ubuntu anyway.

You can get more information at [http://www.debian.org/ports/m68k/](http://http://www.debian.org/ports/m68k/) 

If you decide to install Debian on an old 68k-based mac, you'll get a modern operating system on a very, very slow machine. It's usable but don't expect to be running GNOME as your desktop, I've ran NetBSD on a Quadra 840AV which is the fastest 68k based Mac ever made and it wasn't a speed demon. Running modern software on old a bit exotic hardware can still be fun though. :)
