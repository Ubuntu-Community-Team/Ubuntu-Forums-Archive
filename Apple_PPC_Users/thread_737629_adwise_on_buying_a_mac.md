### Post by adelahunty on 2008-03-27
Older Macs, such as the eMac, are very different to PCs, as the processor is different: they use a PowerPC (or PPC, for short) instead of an Intel-based one.  This means that you can't just pick up a normal distribution of Linux (or Windows, come to that), and use it on your Mac.  You must find a PPC version, and there aren't many of these about.  Fortunately, there are community run versions of Ubuntu / Kubuntu / Xubutu, as well as other distributions, such as OpenSuse and Fedora (although this is quite old now).

As ever, it depends a lot on what you want from your install... :)  99% of the time, there is really little difference between a PPC Mac with Ubuntu, and any other install of Ubuntu in all it's various flavours.  I wish there was a PPC Mac version of Sun's Java JDK available for programming - personally, I cannot stand the IBM version I have to use whenever I do any Java on a Mac running Linux.  Other than that, there is very little difference as a Mac user.

With regard to the hardware, I have a friend running Ubuntu Feisty on a machine very similar to the one you are looking at.  It runs perfectly fine, with no problems at all (as far as I can tell, anyhow).  The one thing you may want to look into is the size of the hard drive.  On all the iMac and eMac machines, they require taking the machine to pieces if you want to upgrade it.  It isn't difficult, but can take some time.

The one thing about Mac machines such as iMacs and eMacs is that while they rarely break down, when they do break, you can't just go to the shops and get a spare (unless it is the drive or the memory which has broken).  The spares can be difficult to find, although eBay makes matters easier.

Having said that, I have only had one Mac break in the past six years, and it was a new Intel mac Mini - I still use my old grey iBook to run Xubuntu, and have recently helped a couple of friends set up some Blue and White G3 PowerMacs.  Rock solid machines... :)  If the price is good, then you should be very happy with it, and should have little problem in setting it all up and running Ubuntu.
