### Post by Tommy on 2007-08-03
> **sethdotcom said:**
> I have an old imac g3 400mhz 768 mb of ram, 2 sticks. I removed the 512mb stick and noticed that my computer was faster using only the 256mb stick. Could there be a problem with the 512mb stick that makes my computer slower? or do you think it is just me?

It is possible -- there may not be anything "wrong" with the 512mb memory, but it may have different characteristics from the other one, which could be slowing things down. I haven't been able to confirm whether it's true on that particular machine, but on some machines it makes a huge difference to have the memory "matched" in characteristics. 

I know on some old iMacs one memory slot is really hard to reach, but you might try swapping around and seeing if you see the same effect with just the 512mb stick.

Alternatively, you could go ahead and buy two of whatever the maximum that model can use (maybe 512mb each, maybe larger) and try replacing both at the same time. I suspect you will have better performance, even if there is nothing wrong with your two existing RAM modules.

Good luck!
