### Post by slux on 2005-04-18
> 
Why not just run it on OS X?
[http://fink.sourceforge.net/](http://fink.sourceforge.net/)

Because OS X is not completely free software and it would defeat the purpose of running free software. (I'm assuming the use of "free software" implies the reasons are ideological instead of strictly practical).

My reasons for using Linux on the Mac are here:

1. Package management.

 Going to a hundred different places to download a mixed set of software and then proceeding to install it (or in some cases of course just dragging to the app folder) is *not* as good as being able to click for what you want in Synaptic and then watch it automatically install *and* getting automatic updates for the complete system.

And no, Fink certainly isn't the same thing. It only has limited package selection and none of it applies to the native OS X apps.

2. Excellent selection of free software applications that fit right into my desktop.

Fink etc. have many of these but running thru Apple X11 doesn't seem ideal and they don't play nice with the rest of the desktop. Essentials such as AmaroK are still missing. I also quite like GNOME and there's no point running OS X and then not using aqua on it.

3. Performance

GNU/Linux is somewhat faster.

4. Familiar envinronment

It's hard to say how much this effects my perception but the fact is that I've been using Linux almost exclusively for at least a solid six years now. I know the system very well and while I try to keep an open mind it is still a factor.

5. Openness

I can fiddle with literally everything on Linux and do it thru the command line. There are things that are off limits on OS X as far as I know.

6. last but not least: Ideology

I like the free software ideals and am on this boat not only because I like the superior applications but also because I agree to what FSF has to say on the ethicality of proprietary software. There happen to be entirely pragmatic benefits to this as well and I'm reaping them every day I'm using a free system. As RMS says, [Pragmatic Idealism](http://www.gnu.org/philosophy/pragmatic.html).
