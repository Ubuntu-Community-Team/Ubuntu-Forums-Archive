### Post by RHTopics on 2006-07-10
Using a new feature for Ubuntu 6.06, you can install apps and have them available the next time you boot up.  It is called *Live CD persistence*.

Below is the link to the Ubuntu Community Documentation that describes how to use this feature:

[https://help.ubuntu.com/community/LiveCDPersistence](https://help.ubuntu.com/community/LiveCDPersistence)

The document describes how to initiate the feature for PCs when booting up.  For Macs,  type the following on the **boot:** prompt

```
live persistent
```
