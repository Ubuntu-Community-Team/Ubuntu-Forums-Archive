### Post by tcrroadie on 2007-07-03
I confirm what stmiller said.  If you plan on giving Ubuntu Feisty a try on your iMac, I would recommend a dual boot installation along side of OS X.  If you go with a dual boot, you will need to reinstall OS X first and create an extra "Free Space" partition for Ubuntu during the OS X installation.  This is done by using the "Disk Utility" tool that is included on your OS X installation DVD.  

You can download Ubuntu 7.04 Feisy Fawn live and alternate install cd's from the link below. 

[Ubuntu 7.04 Feisty Fawn Downloads]("http://cdimage.ubuntu.com/ports/releases/feisty/release/")

I also recommend reading the [Ubuntu PowerPCFAQ Wiki]("https://wiki.ubuntu.com/PowerPCFAQ").  The wiki should answer most of your questions.
