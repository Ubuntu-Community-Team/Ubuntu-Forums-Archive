### Post by TravisNewman on 2005-04-26
2. Probably. Depends on the hardware. You can also install prelink in Ubuntu which will speed up loading considerably. I THINK OSX uses this, but I'm not sure.
3. That's exactly it. mp3, avi, etc, etc, are all hindered by copyrights and therefore cannot be distributed without paying licensing fees or having some other agreement with the copyright holders. You can install the capability to play these files after you've installed Ubuntu.
4. Installing Ubuntu to hard drive will help tremendously, and when you've installed you can also install the binary drivers for your video card (also restricted by copyrights/patents), which will increase the fluidity of the interface. OSX has a very very nice, tight interface, and Ubuntu's (or anyone elses at this time) isn't quite up to OSX's standards. However, the interface is only one part of the entire experience, and I think once you've installed Ubuntu and can really play with it more, you'll see what I mean.
