### Post by ajgreeny on 2006-06-15
It may not appear in your menu's for some reason but if you just press alt+F2 you will get a run dialogue where you simply type "kmail" without the quotes, and up it should pop.

Kmial is just part of the KDE pim package, and is very like thunderbird in many ways though it does have a few different and added functions.  You can even use it to filter out spam like Mailwasher in windows by setting up a pop3 filter for messages above 1Kb in size.  All will then be listed and you can chose to download those you want, leave them on the server or, delete from the server without downloading them at all.
