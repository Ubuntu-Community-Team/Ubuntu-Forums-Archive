### Post by DJ_Max on 2005-09-29
Firmware is basically a program hardcoded into hardware. So, as the name suggests, it's in between *hard*ware and *soft*ware.

If you need a firmware update, Apple.com should have it, as thats where I upgraded mines from. It's called OpenFirmware. But I've never upgraded my CPU. Check where you are buying the part from.
