### Post by atoponce on 2006-07-08
Ubuntu blows *BSD out of the water on all 3 counts!

1) Hardware recognition is the best that I have seen on any operating system.  During our local LUG install fests, we install Ubuntu on most machines brought in, because we have the least amount of issues with it.

2) Ubuntu is far more user friendly on many terms.  Hacking, servers, everyday use, etc.  Things are just very simple, and easy to use.  If you can't figure it out, the OS is laid out in a very plain and straight forward manner, so it is easy to work with when asking for help.

3) The only distro I know that has more software repositories than Ubuntu is Debian, and it isn't far ahead.  If you ever use *BSD for any amount of time, you will begin to seriously miss the amount of software available at your fingertips.
