### Post by Auria on 2007-10-21
1) What CD version did you download? (for a beginner version 7.04 is probably the best one)
2) Older G3 computers need a special procedure, be sure to check the PPC FAQ (it's a sticky on these forums)
3) You need to burn the CD and boot from it, you can't double-click it from mac OS. Use e.g. disk utility (at least on OS X) to burn the contents of the disk image to a CD, then insert it in your drive, reboot while holding 'c' or, if 'c' doesn't work for you, 'alt'. Then you will boot into ubuntu and you can install from there. 

Be careful not to tell the installer to take the whole disk if you want to keep mac OS! Also, before partitionning, it is recommended to backup important data. (it will not partition unless you tell it to, don't be afraid, just booting the CD won't hurt.)
