### Post by onehotcutey on 2006-07-02
Elive is made for x86.  To install it for PPC you have to compile it from source, which can be obtained from get-e.org.

This Howto is great place to start.

[http://ubuntuforums.org/showthread.php?t=97199&highlight=e17+source](http://ubuntuforums.org/showthread.php?t=97199&highlight=e17+source)

I compiled it about two weeks ago just for the most up to date version, and it still works wonderfully.

You will have to install the *-dev packages from synaptic to compile though.
