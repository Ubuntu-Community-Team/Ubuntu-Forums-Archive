### Post by n00bWillingToLearn on 2006-07-13
If you are really deseperate, WINE can be run on PPC though Qemu. Qemu is an emulator, much like VirtualPC, but it has one feature that sets it apart in that you can run a linux application compiled for x86 on a PPC without having to emulate the entire operating system. So you wouldn't need to buy Windows, and you wouldn't be wasting memory on Windows, like you would running the entire OS in Qemu but the windows applications are still going to run just as slowly, and use up just as much CPU, as they would in virtualPC. If you have never tried running VirtualPC on a G3, don't, it really is painfully slow.
