### Post by Ptero-4 on 2006-03-02
IIRC. It's impossible to set linux to run an external monitor in that iBook model. I know b/c I used to own one of those. In MacOS and OSX it's possible though. And one other thing. The iBook 500MHz only supports video mirroring not multihead. There's a hack to activate multihead support on the iBooks but it won't work with ATI Rage cards (read: iBook 500MHz) it only works in ATI Radeon (iBook 600MHz or later models).
