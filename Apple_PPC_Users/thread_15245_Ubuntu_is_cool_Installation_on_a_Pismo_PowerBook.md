Hello everyone!

This is my very first post. I discovered Ubuntu last week and I gave it a try. So far my experience is very good considering that Ubuntu is in early development stage.

For the record, I'd like to add another successful installation on a PowerBook. I succesfully installed Ubuntu Warty on:

PowerBook FireWire (Pismo)
FastMac G4 Upgrade [Motorola G4 7410 @ 550 MHz, 1 Mb L2 Cache (2.5:1)]
1 GB RAM [2-2-2 from Transintl.com]
60GB Hitatchi HD 5400RPM 8MB Buffer ATA/6 [HTS548060M9AT00]
Original AirPort Card (Not AirPort Extreme)
+ All the rest original specs of the Pismo ([http://tinyurl.com/5xdo3](http://tinyurl.com/5xdo3))

I can use only my Built-in Ethernet port to get network access, and after searching the forums high+low I found out that Apple's AirPort cards are not supported yet.

I also tried to install Warty on an external FireWire HD without any success, I added this to the IdeaPool section of the Wiki.

I just want to hear people's opinions about running an OS other than MacOS on Apple hardware.

Also what do you do? I mean do you use your Linux boxes for fun or work?

Kudos to the Ubuntu Development Team, I guess the next release, Hoary, is gonna be kick-ass class  \\:D/

### Post by fu. on 2005-02-22
hi jml75,

i suggest you definitely **try** Ubuntu (or any other Linux flavor for Apple Hardware) **before** **removing** MacOS X.

That's why i want to see FireWire support on the current Linux distros (including Ubuntu). I want to Boot from a FireWire HD and try the new OS leaving my internal HD untouched. Thing is that we're still waiting for this, no Linux distro that i've tried so far provides decent FireWire support to install + boot from a FW HD (not even YellowDog that provides a special image for installation on FW drives). FW support is wacky yet  ](*,) 

now regarding speed + resources, it all depends on what you use your machine for. In most cases Panther (MacOS X 10.3.x) is much faster than the current Linux distros for PPC machines plus the GUI + all the graphics are way better hands down.

On the other hand Linux can get fast if you start tweaking it (and gaining knowledge about it) as you work with it. It's a great replacement for Wintel machines but i'm not so sure if the same is true for Apple Hardware yet.

Also take a look at this article:

[So you own a Mac, why run Linux?](http://www-106.ibm.com/developerworks/library/l-pmac.html)

HTH

### Post by Down8ve on 2005-02-22
I have warty running on my almost-6-year-old Lombard.  The Linksys card was detected during installation, as it gave me a choice of wireless or ethernet being my default.

OS X was stable, but too slow.  I reinstalled OS 9, but the fonts were so ugly and the software so out-of-date that I could not stand it!  Web browsing via Explorer, Wamcom (sp?) and Mozilla 1.2 was terrible.

I tried the latest hoary seed two days ago, and while it had some great improvements there were also too many bugs to run on a day-to-day machine.  I suggest  folks use the warty release until hoary is ready.  By the way, hoary looks to be amazing!  How nice it is for these folks to offer such a nice system for free.

Be sure to plug in your wireless pcmcia cards prior to installation.
