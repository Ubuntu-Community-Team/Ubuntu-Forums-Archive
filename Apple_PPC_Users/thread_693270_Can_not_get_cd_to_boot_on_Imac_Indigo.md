### Post by ssam on 2008-02-11
first thing to try is switch off the mac, and boot it with the mouse button held down.

if that does not work you can eject it with a straightned paper clip. poke it in the right corner of the cd slot, it does not need to go in very far.

### Post by stream303 on 2008-02-11
Some quickie tips:

Burn the iso (not just copy the file) at the slowest speed possible, ie 1x, 2x etc.  Use good quality cd media.  If the mouse-button hold doesn't work, try booting holding down the "C" key.

Take a look at these helpful docs as you go:

[https://wiki.ubuntu.com/PowerPCKnownIssues](https://wiki.ubuntu.com/PowerPCKnownIssues)

and

[https://wiki.ubuntu.com/PowerPCFAQ](https://wiki.ubuntu.com/PowerPCFAQ)

Let us know how it goes!

### Post by VCF on 2008-02-11
Have no fear, it will work with 6.06 (I got it working from the live CD)  just be mindful of the blank screen issue on the PowerPCKnownIssues stream303 mentioned.  Again it is at:

[https://wiki.ubuntu.com/PowerPCKnownIssues#head-cae29299476585f042f0185bea1d51b9372722c8](https://wiki.ubuntu.com/PowerPCKnownIssues#head-cae29299476585f042f0185bea1d51b9372722c8)

If you are having trouble with 6.10, start with 6.06 LTS and upgrade to 6.10 (be very careful upgrading beyond that).  

Do you have 256Mb+  memory?  More memory sure helps.

I also found I had to swap out my CDRW drive as it had problems reading those burnt disks (even despite burning them really really slooooooow)  But it had problems with other Mac disks as well, so I knew if was a H/W problem.
