### Post by Midwest-Linux on 2007-09-13
I decided to try Yellowdog 5.0 on my G3, after downloading SIX ISOs. Thats right six CD ISOs, then burning each one to disk as a ISO at 4 X speed. That took well over a hour. I was ready to install Yellowdog on my Mac G3 PPC. Installation actually took over two hours. So finally after the last CD, it said to pop the CD out and reboot, now comes Yellowdog...right? Well no. it did not boot, in fact all that came up was the Mac bootloader and there was no Yellowdog.

 So then I turned off the computer and re-inserted Disc One eventally I found a choice between a new install and a update. So I checked off the update, 20 minutes later, it said to put in Disc Four so I did and then it said to eject and reboot. Right now its been several hours already on this. So I rebooted, and then yet again it comes up with the Mac bootloader, yet again no Yellowdog. So i head for the Yellowdog forums, I go to register, it says my email address is invalid. OK I give another email address and yep..that was invalid too.

 This has to be absolutely  the worst linux distro I have run across to date...and there are not very many...but seriously save yourself some time, bandwith,and of course the six CDs that will be wasted ...stay away from this dog. I can't belive they release a PPC distro that cannot be booted on a PPC after a ungodly installation time, and a distro that takes up SIX CDs and the support? There isn't any.

 Back to Ubuntu 7.04 PPC Text alternate installer.....

### Post by tgalati4 on 2007-09-14
The PPC processor is decent, and with enough RAM, they are decent, low-cost machines.  However, only Tiger really makes all of the hardware work like it should.  I had problems with Dapper on a powerbook, G4, 1 GHz machine.  I'm not sure if Feisty is any better since Canonical dropped support for PPC shortly after Dapper.  Feisty may have fixed some problems, but I'm satisfied with Tiger and using XDarwin to remotely log into my other Ubuntu machines.

The bottomline--use an operating system that works best with your hardware.  Anything else is a constant struggle.
