### Post by grazie on 2007-02-14
You certainly can boot with both USB and firewire devices if it's enabled in the kernel. However, after quickly looking at a standard Ubuntu config file, it doesn't look like they're set on by default. This would be mean you'd need to build your own kernel with these options turned on.
