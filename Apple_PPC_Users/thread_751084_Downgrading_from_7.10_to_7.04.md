### Post by stream303 on 2008-04-10
Downgrading a whole distro is a drag.  However if you still want to do it, just reinstall by booting with the 7.04 disk you have.

I'm not sure if  you have set up a dual-boot scenario, but if you have the machine dedicated solely to Ubuntu, when it comes time to partition, you can just let it "use the whole disk".

If you get 7.04 installed, you may want to make sure that you have the backports-repository checked in your software sources.  This should give it a little more life.

Or you can hang in here for a couple of weeks when Hardy goes to official release status.  It seems to be shaping up nicely.
