### Post by acorn22 on 2006-05-28
Ok I have got it! Someone at the apple DB helped me :)

Here is the solution:

"power on computer,
hold CMD,ALT,O-F
Wait until some prompt appears
type:
reset-nvram (hit enter)
reset-all (hit enter)
The PB will reboot

Again try to start by holding the ALT key or by holding the C key.

If you don't know your password, use this program to turn it off. apps>utils>open firmware password. If you don't have it, google it or something. Then download it and run it. follw the instructions."

Happy Linux Booting!:cool:
