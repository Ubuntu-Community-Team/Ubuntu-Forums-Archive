### Post by HandyMac on 2006-11-18
I can't speak to the comparison of OS X with Ubuntu, as I'm only just learning about the latter (and a little disappointed to learn that apparently PPC support is going to be dropped now I've begun). But I've set up hundreds of Macs in the last 18 years.

Though OS X 10.4 'Tiger' will run on G3 Macs, I find they're more comfortable with 10.3.9. You can save considerable space by customizing the install, leaving out unneeded printer drivers, language resources, etc.; and after installing everything, by stripping out unneeded language resources (again, in updates and third-party apps) and unneeded code architectures (G4, G5, Intel) with the [Monolingual]("http://www.versiontracker.com/dyn/moreinfo/macosx/13031") utility (use with care, it can do serious damage -- I believe I've read that some Adobe apps won't run if unneeded languages are removed). 

If speed is still an issue, I've heard that Unsanity's [Shadowkiller]("http://www.unsanity.com/haxies/shadowkiller") will help on slow Macs (haven't tried it myself). I've set up several ~350MHz models with OS X, and found them quite acceptable  -- though above comments that Ubuntu is faster are probably correct. It's apples and oranges, really; depends what you want.
