### Post by Rob_Frohne on 2005-12-29
Hi,

I am looking for more tips to make kubuntu easier for me to migrate to.  I'm coming from OS X.  Here are the ones I have come across so far.

1)  Install mouseemu using Adept (universe repository I think)
2)  In System Settings -> Regional & Accessibility -> Keyboard Shortcuts -> Modifier Keys, select the Mac options.
3)  In all the applications you use use the Settings -> Configure Shortcuts, set the command keys to do what they used to.
4)  In System Settings ->  Regional & Accessibility -> Keyboard Shortcuts -> General Shortcuts, set the shortcuts you use to match what you are used to.
5)  Use the same applications where they are available.  For example, I use Thunderbird on OS X and now I use it on kubuntu.  I copied my Profile folder from OS X to kubuntu and it works the same as it did on OS X.

What other good ideas can you add?

Thanks,

Rob

### Post by tokyovigilante on 2005-12-29
If you're using the Composite extension, install skippy-xd, its the closest thing to Expose for KDE/Gnome. It uses the XDamage and Composite extensions to render via the GPU comparably to Quartz 2D.

 If you're not using Composite, then plain skippy will work similarly, but without the live update a la Expose.
