### Post by Adrian on 2006-01-30
[QUOTE=fannymites]Easiest way is to apt-get install kubuntu-desktop.  Next time you get to the login screen select session and choose either kde or gnome.[/QUOTE]

I'd recommend using **aptiude** instead of **apt-get** when installing the kubuntu-desktop meta-package. This makes it possible to remove the KDE desktop again by doing a
```
aptitude remove kubuntu-desktop

```
