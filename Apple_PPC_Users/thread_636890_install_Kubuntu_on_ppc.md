### Post by Auria on 2007-12-10
Burn the image as CD slowly (usually 4X... do not butn the image as a file on the CD but its contents as the CD, burning apps should let you do that.). Insert the disk in your CD drive, reboot the computer, hold down the C key to boot from CD (if that doesn't work try 'alt')

Be sure to check the PPC FAQ (it's a sticky) as 7.10 has a few nasty bugs in it and you will likely need to follow a procedure to fix them. Alternatively version 7.04 should not contain these bugs.

Hope it helps :)
