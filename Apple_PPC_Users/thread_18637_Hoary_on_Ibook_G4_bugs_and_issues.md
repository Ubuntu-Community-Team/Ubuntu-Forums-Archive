### Post by Pineault on 2005-03-07
This is a cross-post from Hoary forum
Hi all, don't know if this should be posted here or on development forum, but wanted to share problems and bugs that prevent effective use of Ubuntu on Ibook G4 before hoary's release:
1. Sleep: as of yet kernel 2.6.10-4-powerpc, sleep will cause kernel panic and force user into hard reboot, a number of threads and bugs have been filed on this issue, among others
[http://ubuntuforums.org/archive/index.php/t-5760.html](http://ubuntuforums.org/archive/index.php/t-5760.html)
[https://bugzilla.ubuntu.com/show_bug.cgi?id=6977](https://bugzilla.ubuntu.com/show_bug.cgi?id=6977)

2. Wifi, since airport extreme will not in the near future be supported, the solution is to use a USB wireless device, like the D-Link DWL-122, to use this, one has to run a script, the device is unstable, connection fails constantly, and there is no integration in the default network app.
Again this has been discussed in various threads such as this one
[http://ubuntuforums.org/showthread.php?t=5660](http://ubuntuforums.org/showthread.php?t=5660)

Resolving these two issues would make the ubuntu Ibook an amazing linux laptop...

any ideas as to how and when these issues will be resolved.

This thread could also be used to add other Ibook G4 known bugs and problems related to the upcoming hoary release.
