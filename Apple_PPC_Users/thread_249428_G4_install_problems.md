### Post by tdecarlo on 2006-09-02
When you restart the computer, make sure you hold down the <option> key from the moment it shuts down.  Don't let go of it until the menu pops up to select your operating system.  If only your Mac OS shows up, then you didn't burn the image correctly.  If a CD icon appears that says "windows," that's your Linux disc.  Select it and your computer will boot into Linux.

### Post by 3rdalbum on 2006-09-04
I don't know how you've burnt the discs, but they must be burnt with Toast as a "Disc Image". If you just dragged the ISO file onto the blank disc, or if you opened up the ISO and dragged the files from it onto the blank disc, then you've made it incorrectly.

To answer one of your questions, every Linux distro requires that you burn a disc image correctly onto a disc. If you don't have Toast, and if there are no free image burning programs for your version of Mac OS, I suggest you order some CDs from ShipIt.

### Post by kulath on 2006-09-05
Quode,

I burned four CDs that failed to work, it was only the fifth that was OK, so it can easily be that the burn is not perfect. Fortunately, I was using Toast, which has a verify function, so the faulty discs had failed to verify. Unfortunately, I think that the free disc burning mechanisms do not provide a verify function.

Anyway, here is what you do, I am sorry that the installation instructions on the site are so bad.

1) Burn the CD in accordance with the instructions.
2) Boot into Mac OS in the normal way.
3) Insert the CD into your CD drive.
4) After some scratching around while the CD is read, the CD should appear on your desktop. If it does not, then something is wrong with the CD, so you need to burn another.
5) Open the CD in the finder by double-clicking on the CD icon. If this does not work, then there is something wrong with the burn.
6) Choose Restart... from the apple menu.
7) Immediately when the 'Bong' sounds, press and hold the c key on the keyboard. It should now boot into the CD installer.
