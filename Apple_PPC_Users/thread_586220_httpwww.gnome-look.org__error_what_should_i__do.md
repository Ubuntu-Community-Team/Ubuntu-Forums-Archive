### Post by 5-HT on 2007-10-22
You can unpack them to ~/.themes on a per-user basis or /usr/share/themes for a system-wide install. The error you're getting from the theme manager is probably due to trying to install the unpacked themes. If you want to install via the theme manager, it'll want the archives--no need to unpack.
