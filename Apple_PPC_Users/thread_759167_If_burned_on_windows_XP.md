### Post by BladeMelbourne on 2008-04-18
Of course you can.

1) Download an Ubuntu power pc ISO to your Windows XP PC. You probably want to download the alternative installer ISO image.
2) Use a program like [ISO Recorder]("http://isorecorder.alexfeinman.com/isorecorder.htm") to burn the image to CD.
3) Put the burnt CD into the Mac and hold down the C key when the Mac boots up.

### Post by stream303 on 2008-05-16
I don't know what mac you are trying to install to, but be sure that you are burning at a low enough speed for your mac's cdrom to deal with it.  Many of us burn at 2x or 4x for install images anyway.

Be sure you are using a CD-R, and NOT a CDRW.  Has been known to cause problems.

On some machines, you may have better luck by powering up the unit first, and very quickly afterwards holding down the -C- key, and hold it down long enough until you hear or see the system being booted by the cdrom.

### Post by Elephex on 2008-06-17
I actually just did this same thing!

1.) Download the ISO file from whatever server you find it on.
2.) I used Nero on my WinXP machine to burn the image (it then showed up on WinXP as a hybrid disc)
3.) I put the CD in my g4 TiBook
4.) Hold "C" while starting it up
5.) Choose your install!


If you are having problems with it not booting from the CD, try reburning a CD. The problem could lie with the CD not having the correct files on it. Make sure that you select "Burn IMAGE" when doing all this. You want to make a CD of the OS, not a CD of the image of the OS (does that make sense?)

Hope it helps in any fashion.
