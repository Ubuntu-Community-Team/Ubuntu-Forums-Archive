### Post by mrbass on 2005-05-22
I use a lite-on 16X dvd burner in a bytecc case (oxford 911 chipset) with firewire.  It works great under os x with patchburn (both panther and tiger).  Linux I've only used it a couple of times but works fine.   One small caveat though....to upgrade the firmware must attach it to an internal IDE cable on a windows box.  I have pretty recent firmware so I doubt I'll need to upgrade it again but just in case I do that would be the necessary step.
