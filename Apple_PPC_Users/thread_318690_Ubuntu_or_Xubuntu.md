### Post by doobit on 2006-12-14
> **meng said:**
> If you like XFCE, go with Xubuntu. Any programs that can be run in GNOME or KDE can also be run in XFCE (but you may be asked to install certain GNOME/KDE libraries in order to do so).

I really like XFCE. It may have been less eye-candy once, but the most recent releases can be customized nicely and be very beautiful and functional. Even Thunar, the file manager, has become full-featured. GTK2 programs run beautifully in XFCE. 

Xubuntu itself starts out with fewer packages in the base install than Ubuntu, so that's something to consider. If you want all of the Ubuntu packages, but also want to run the XFCE Window manager system, then you might want to install Ubuntu first and then add the Xubuntu-desktop.

### Post by meng on 2006-12-14
Yes, openoffice, gimp and firefox are applications.
Xfce, GNOME and KDE are desktop environments.
Linux applications can run in any environment, although some rely on specific libraries installed by default (e.g. in GNOME) but readily installed even if you use another environment.

### Post by meng on 2006-12-14
If you install with ethernet plugged in, the installer will set up your ethernet connection for you automagically. Otherwise you may have to fiddle with it post-install to get it working. I see no significant security risk by installing while it is plugged in.
