### Post by stream303 on 2008-01-19
For me, I'm starting to think that Network Manager in Gutsy is just not ready for prime time.

On both my iBook and iMac, I had to actually remove Network Manager to get my wired networks working properly.  I still had a gui network interface after I removed Network Manager and reconfigured it manually via the gui interface.

This may not be your problem, and I freely admit that I have ZERO wireless experience, since I don't run wireless.
