### Post by sulobanks on 2006-06-04
The short answer is that Skype isn't available for linux ppc.  However, you could use something like qemu to run Skype for Windows or you could check out Ekiga, which reportedly can communicate with Skype users.  I haven't tried either of these myself yet, since I haven't had a need for something like Skype.  I tried qemu to run Win 98 on Breezy and got just about everything but networking to function.  Maybe it's improved by now.
