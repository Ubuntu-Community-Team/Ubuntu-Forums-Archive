### Post by adelahunty on 2008-03-27
I have installed a lot of Xubuntu dists recently on iBooks, for some odd reason.  Karma, I guess....

From my recent experiences, I can tell you the following about installing with discs:

1.  Feisty seems to install with no problems.  Use the live or alternate install discs, and you're laughing.  Depending on your RAM, I would say this is a better option.
2.  Gutsy takes a bit of work, but will get there.
3.  Hardy, for love or money, I cannot get onto a clamshell.  I know others have, but not this soldier.  Life is too short to keep trying, I think! :)

My recommendation is to get an installation on the iBook, using an alternate disc - Feisty should be fine.  Then use the upgrade manager to check for new dist upgrades (there will be) using the -c command-line switch at a terminal.  I have never had any problems with this at all.  And if you wait a few weeks, the non-developmental version of Hardy should also be available.

That's my experience, anyhow.  Your mileage may of course vary...
