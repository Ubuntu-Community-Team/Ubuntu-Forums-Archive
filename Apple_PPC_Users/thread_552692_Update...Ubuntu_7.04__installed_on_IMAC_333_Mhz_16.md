### Post by Midwest-Linux on 2007-09-16
UPDATE
IMAC 333 Mhz 161 MB ram 6 gig Hard drive. Ubuntu 7.04 successfully installed. Uses the 144 pin SO Dimm (Note: 144 pin SO Dimm for PC laptops WILL work in the IMAC!)

This is a continuation of another thread I started called "Can the firmware be upgraded without a disc? " [http://ubuntuforums.org/showthread.php?t=552437](http://ubuntuforums.org/showthread.php?t=552437) 

I am only starting this new thread because it might help others, who are considering buying a IMAC to put Ubuntu on. Use the alternate text based installer for Ubuntu 7.04. If your IMac only has 32 MB or 64 MB of Ram, you can upgrade and you should upgrade the Ram before installing Ubuntu. Mine only had 64 MB with the swap drive. I used 128 MB SO Dimm ram from a old defunct PC laptop I had lying around and from another spare 64 MB  SO Dimm i had lying around.

Well I installed Ubuntu 7.04 using the alternate text based installer. I put in the Ubuntu disc in the CD Rom, shut the machine down and restarted it. I did not have to press C to make the disc boot. 
Well it took awhile to install but I think thats due to a slow CDROM more than anything. 

Important, if it seems to hang around 6% when installing, thats normal, just leave it and it will continue to install and 15 to 20 minutes for just that 6% is not unreasonable.

Anyway, it installed OK. It runs a little slow, but it works. I'm sure some additional ram will take care of that. I am listening to streaming audio after I installed the additional (G-Streamer) codecs in the add/remove section (be sure to click the top to all available install). Thats what I wanted it for to listen to streaming audio in another part of the house. Now if I can configure this for wireless internet, that would be really great

 Just a note to all who are installing Ubuntu on a Mac, after install if you find you do not have internet access. Simply click the little network icon on top and click wired connection and as long as you have your network wire connected to the LAN jack you should be good to go. I also have to do that with my other Mac (which also has ubuntu 7.04) which is a G3 non IMac, its a very minor issue and just a small step. But it needs to be noted, especially with noobies. 

I didn't realize that all IMacs  are New World, thats good to know as i just got another off of E-Bay with no HD but that one is a 400 Mhz machine with 128 Ram...though I think those might be the more common 168 pin versions. 

From what I understand so far, while I Macs are New World, there are  different types of New World. One uses the 144 pin PC 100 SO Dimm's and the other type uses the more common 168 pin PC 100 Dimms. The version I have uses the 144 pin SO Dimms with a upper limit of 256 MB, Ubuntu 7.04 should work OK on that. I'm using 161 MB ram. More modern IMACS, wil have a upper limit of 512 MB for one model, but others will have a upper limit of 1 GB.

To clear up the confusion about what IMACs contain what type of Ram, read this
[http://docs.info.apple.com/article.html?artnum=58007](http://docs.info.apple.com/article.html?artnum=58007)


While the 144 pin SO dimms cost more, they are not that expensive. Again, the common 144 pin SO DIMMS ones used in PC laptops will work in the IMACS as long as the PC range is the same like PC 100 or PC 133.
