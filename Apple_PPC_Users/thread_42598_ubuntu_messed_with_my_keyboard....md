Do you have a laptop?  By default Ubuntu is set up so you have to press Fn+(function key) for the brightness, volume etc (which happens to be the way I like it-- I have OSX set up that way too).  

You can change this by running the 'powerprefs' GUI, although I don't know if that's on the Live CD.  If it's not, you'll have to do a 'man pbbuttonsd' to find out where the configuration file is stored that'll let you change this.
