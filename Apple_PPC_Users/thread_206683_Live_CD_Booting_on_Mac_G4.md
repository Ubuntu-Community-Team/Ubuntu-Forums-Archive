### Post by keyshawn on 2006-07-01
Fran,

As described on one of the download mirror pages ( [http://ftp.ussg.iu.edu/linux/ubuntu-releases/6.06/](http://ftp.ussg.iu.edu/linux/ubuntu-releases/6.06/)  ), the desktop edition is live, which means it won't install on your computer automatically. You do not need to modify the desktop CD to run it 'live.' 

If you have a PPC computer, then on that download page, select the link that has PPC and desktop in the name. 

More information (Basics) are in the help guide, found at [https://help.ubuntu.com/ubuntu/desktopguide/C/index.html](https://help.ubuntu.com/ubuntu/desktopguide/C/index.html) 

If you have any questions that are not in there and have not found your solutions in the community help site (has great info !) ( [https://help.ubuntu.com/community](https://help.ubuntu.com/community) )or would prefer to ask others directly, you can post them in the forum. 

Hope to see you around here and good luck !

Regards,
