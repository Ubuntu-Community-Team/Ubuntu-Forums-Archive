### Post by peter_m on 2005-12-02
Watchout, iMAC 333 nead a ROM update (or firmware update can't remember) to see the 256MB memory modules.  Otherwise they won't see the 256 at all and the largest acceptable module will be only 128MB. To install the update, you need Mac OS 8.5.x or 9.x.x . Best to try while MAC OS is still installed. I dont think you can use anything larger then 2x 256 for a total of 512MB.
