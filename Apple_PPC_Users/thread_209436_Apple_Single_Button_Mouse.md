### Post by Ptero-4 on 2006-07-05
It is emulated with f11 and f12. You can use mouseemu (still experimental) which provides an easier way of configuring mouse button emulation, or you can search in the forum, there's a thread on how to get the buttons to be fn+alt and fn+ctrl.
