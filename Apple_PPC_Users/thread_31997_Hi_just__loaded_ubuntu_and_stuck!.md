### Post by ssam on 2005-05-05
hello

welcome to ubuntu.

linux is getting better and better at making everything you need available in a graphic user interface, but unfortunatly not everything done yet.

the printer should be easy to set up in.

system -> administration -> printing.

dual booting should have been set up by default. you press 'L' for Linux and X for mac os X, at boot time.

you might want to try Mac-on-Linux (MOL). it lets you run mac os x either in a window on you linux desktop, or on a virtual screen. (i think you switch between mac and linux with Crtl-Alt-F7 and Ctrl-Alt-F8). the trouble is that you will need to use a command line to set it up.

do you find any combination of coloured text on coloured background easier to read? i know some people find white on black easier than black on white. or that some people can read better on a blue background.

where do you live, there might be a local guru who can come and help you set up.

sam
