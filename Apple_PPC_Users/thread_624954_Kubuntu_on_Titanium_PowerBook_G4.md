The broadcom one is the Airport Express card in newer macs. And there have been major fixes so broadcom works great in Gutsy as some have reported.

### Post by stmiller on 2007-11-29
One big difference is that in Linux powernowd supports scaling the CPU down on these Powerbooks automatically. So it increases the cpu speed when you need it, and turns it down when you don't. OS X Tiger does not support this feature for some reason. With OS X it's either set to high or low and remains fixed there.
