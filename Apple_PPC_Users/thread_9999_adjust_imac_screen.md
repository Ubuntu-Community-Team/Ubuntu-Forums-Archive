### Post by chascon on 2005-01-04
you can use xvidtune to adjust but I've never used it.
One trick I discovered is to simply reboot into mac OS (OS X in my case) and adjust screen tilt and centering there.  It seems that any adjusting done in OS X relfects a little more in linux upon reboot.  I don't know if xvidtune handles tilting but mac OS X does.

OS course, try xvidtune if you don't have a dual-boot, if you don't want to reboot, or just know xvidtune.

I reboot every one in a while anyway.
