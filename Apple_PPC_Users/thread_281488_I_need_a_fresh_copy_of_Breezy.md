### Post by bodycoach2 on 2006-10-22
I've loaded Xubuntu 6.06.1 on two iMac G3, 333 Mhz, 160 mg, 6gig HD. One is Strawberry (pink), and one is Blueberry (blue). These are the Bondi style macs. 

Xubuntu works perfectly on them. Everything works, except wireless. I usually use a Belkin USB 54g Wireless Network adapter on my Ubuntu or Xubuntu installations, as it works out-of-the-box. But, not on Macs.

When the ppc version comes out, try nUbuntu (fluxbox version). Or, you could use the install CD, load on the server installation (text only, no GUI), and install Openbox, Fluxbox, or any of the other slim GUI's. Xubuntu should work good enough though.
