### Post by 3rdalbum on 2007-04-21
Just as Macs can't run Windows, Macs can't run the x86 version of Ubuntu; since x86 is the most popular version, it's the one that comes with the book. However, there is a PowerPC version which will work.

[http://cdimage.ubuntu.com/ports/releases/7.04/beta/](http://cdimage.ubuntu.com/ports/releases/7.04/beta/)

Give it a few days, the site is getting hammered right now because there's just been a new Ubuntu release.

If you want to install onto an external hard disk, that shouldn't be a problem - it will probably appear as an option in the installer.
