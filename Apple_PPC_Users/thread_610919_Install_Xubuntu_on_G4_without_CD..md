That should not stop you.
Macos9 should be able  do the same job, only difference is you will need to resize your macos9 partition from the installer, unless you happen to have a spare partition already for ubuntu to use.
If you have a network connection you can download the Netboot installer images as explained in my guide and try booting with open firmware to see if the installer is going to work (do a trial run and see what the install partitioner shows you)
But If you have the Macos9 installer disk it would be best to reinstall Macos9 is one partition leaving a second partition free for ubuntu installer (if you had another mac to install from).
