### Post by ssam on 2005-09-24
macromedia could probably make one very easily, but they dont seem to think it is worth it. email them or pester them. or pester sites that are flash only.

the gpl flash plays are a long way behind the official one, it is a lot of work to implement, and there are not very many people who need it.

if you are a web developer you might want to look at svg as an alternative. soon it should be able to do everything that flash can do and more.

if you really need flash you'll need to run mac os x in mol, you an x86 version in qemu (probably hard to do.), if you have an x86 linux machine on you network you could use ssh and x forwarding.
