### Post by mjh007 on 2006-03-24
Welcome to the forums.

I would suggest going to [http://www.ubuntu.com/download]("http://www.ubuntu.com/download") and trying out the live cd. It gives you an idea of what Ubuntu is like.

Download the iso (in your case the PowerPC version) and burn it to a CD. Use Nero etc.

Boot using the CD and you should be able to test out Ubuntu without modifying your system.

If you like it, you can download an install iso and go for the real thing.

P.S. I don't know much about Macs, but I think you might be better off using a simple Desktop environment like XFCE instead of the default GNOME of Ubuntu as you might not have much RAM on your system.
