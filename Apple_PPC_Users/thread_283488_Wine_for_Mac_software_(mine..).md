### Post by kidders on 2006-10-24
Hi there,

Modern Macs run on a variant of BSD, which is too closely related to Linux to need emulation, really. Depending on the app you want to use, compiling it from source on your Ubuntu box is often all you need to do to have it run natively on Linux ... and of course, the other way around, if you want a Linux app to run on your Mac.
