### Post by ssam on 2007-04-12
the alpha and beta cds have installed and updated fine.

i think there will be less mirrors with powerpc packages, if you previously used a mirror you might manually need to change your sources file.

the build server builds packages for sparc, powerpc, ia64, i386 and amd64. i dont see why they would turn powerpc off. so if there is a security fix i dont see why a powerpc package would not be built.

however if there was a powerpc specific security (or serious) bug, it would be very low priority for canonical staff to fix it. if someone else made a patch i don't see why they would not include it.
