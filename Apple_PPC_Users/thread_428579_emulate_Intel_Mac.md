### Post by 3rdalbum on 2007-05-01
Senuxis, are you on a Macintosh or a PC?

Sheepshaver and PearPC are both PowerPC emulators; the former for OS 9, the latter for OS X, but you will need a Mac OS CD and possibly the Mac OS ROM file from a proper installed system. I believe Qemu can boot up Linux Kernel 2.4 with its PowerPC emulation, but you won't be able to do a lot with it.
