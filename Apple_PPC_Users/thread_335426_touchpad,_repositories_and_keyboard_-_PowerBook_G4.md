### Post by dshan on 2007-01-11
Scrolling using the PowerBook's trackpad (i.e. two finger scrolling) was only made available in the final (1.67GHz) models as I recall. If you have an earlier model trackpad scrolling is not supported by the hardware. Tapping to click on the other hand has been implemented for a long time however, though I seem to recall reading somewhere it's not supported by Ubuntu (I could be wrong about that). 

The option key is equivalent to the alt key in Ubuntu (and Windows for that matter) so opt + e for Edit menu in Terminal and gedit, etc. There is no del key on Apple laptops - use fn + delete to forward delete. Home is fn + <left arrow> and end is fn + <right arrow>. pgUp is fn + <up-arrow> and pgDn is fn + <down-arrow>.

The repositories should point to Edgy if that's what you installed, use Applications -> System Tools->Sysinfo to check (if that doesn't work, it's broken on x86 maybe ppc too, try uname -a in Terminal).
