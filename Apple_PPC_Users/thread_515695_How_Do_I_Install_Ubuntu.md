### Post by tcrroadie on 2007-08-03
If you plan on dual booting OS X along with Ubuntu Feisty, you will need to reinstall OS X first and create another free partition for your Ubuntu Feisty install during your OS X installation.  I took this approach 2 months ago when I installed Feisty Fawn on my iBook G4 and it worked great.  

Just wanted to give you a heads up on this, so you don't accidentally hose your OS X installation when trying to install Feisty using the partition manager on your Ubuntu Feisty Live/Alternate install cd. 

Good luck.
