I'm not sure I follow what you mean about "Mac-tel" boxen.  I assume you are currently running everything on MacOS 9?  Classic apps are dead and buried as far as Apple is concerned, Steve announced that about this time last year... so if those programs are an absolute must, I'd get MacOS 9 running on that used G4 and I wouldn't mess with it.

My fiancee runs Mac OS 10.3.9 on an old Blue and White with a 500MHz G4 upgrade card... it works great, and it runs old MacOS 9 apps like a champ, so your employers old G4 will get you through.

As for switching over:  I've been running various versions of Linux on PowerPC for a few years now.  There is a lot of hype out there touting that it as the greatest thing ever, but that depends on what your priorities are.  And the dark side that no one seems to mention when a person such as yourself asks about switching is this:  PowerPC Linux is NOT the same as x86 Linux.  I've run everything from Slackware to Ubuntu and many in-betweens on x86 hardware and had hardly any major problems... and I've run everything from YellowDog to Ubuntu on my PowerPC hardware... and there are several problems you will face.

First, a lot of the "compatibility" that Linux users tout comes from x86-only projects like Wine.  You won't be able to install certain software because of the processor.  Without Wine you can't trick Flash into running, Java support is second-tier, and if you program in Mono there are no Windows.Forms.  And not even counting apps that will not run on the hardware, you will encounter the problem of instability.  PPC versions have much smaller installed user bases and aren't usually as robust.  I've switched to Ubuntu because I was just so tired of either not having anything current (YellowDog) or having to beta-test and patch everything myself (Gentoo.)

What I'm saying is this: If you switch to Linux, just jump over to x86 completely.  Run your old classic apps on the G4 with a KVM(+audio) switch and set up a SMB network between the machines for file sharing.  It works beautifully for me...

[QUOTE=Lost-in-North-Dakota]So, I can get a used G4 tower from my employer for about $100 (I don't know if it is a Yikes or Sawtooth).
