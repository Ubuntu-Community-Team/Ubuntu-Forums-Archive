### Post by gear_head on 2008-04-04
I can't really imagine why someone would want to do harm to linux installations, or Mac for that matter. Seems they're all going after Microsoft because of the inherent flaws in their programming.

All Linux is built off of Unix which has always been relatively more secure in general, and has been supported worldwide by millions of programmers many of whom have contributed to improved security in linux over many years. So I'm pretty sure you won't absolutely NEED AV, but if it brings you peace of mind it would be worth it. Though presently there seems to be little to no threat.

The root user account is the only account that has rights to literally everything in the system, so if you're not logged in as root, it's virtually impossible to install or access the more sensitive hardware or software resources.

I've also noticed recently the advent of SELinux (security enhanced linux) which keeps an eye out for potential security risks. It even warns you when you are logged in as "root", which I thinks is pretty cool.

Hope this helps.
