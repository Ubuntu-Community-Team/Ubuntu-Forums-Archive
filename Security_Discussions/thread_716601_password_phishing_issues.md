### Post by erginemr on 2008-03-06
Hello and welcome to the Ubuntu forum.

If you mean phishing by having visited a fake web-mail / bank / etc. site, then you may take a look at:
[http://www.opendns.com/](http://www.opendns.com/)

It is basically a DNS server that promises a faster surfing and keeps track of blacklisted malware / phishing sites.

I have been using OpenDNS for some time on both Windows and Ubuntu, It is reliable and doesn't hamper Internet speed (at least in my case).

For more comments from other fellow forum users:
[http://ubuntuforums.org/showthread.php?t=487520](http://ubuntuforums.org/showthread.php?t=487520)
[http://ubuntuforums.org/showthread.php?t=543659](http://ubuntuforums.org/showthread.php?t=543659)

And for a howto on the configuration of OpenDNS:
[https://www.opendns.com/start?device=ubuntu](https://www.opendns.com/start?device=ubuntu)
[http://ubuntu-tutorials.com/2007/03/14/how-to-setup-opendns-on-ubuntu/](http://ubuntu-tutorials.com/2007/03/14/how-to-setup-opendns-on-ubuntu/)
