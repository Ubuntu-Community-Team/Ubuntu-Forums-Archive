### Post by p_quarles on 2008-03-01
Oh, so this server belongs to you? This isn't anything to worry about, security wise. Like you said, the certificate has expired. Renew it and that will take care of the error message. 

If you want to be extra cautious and make sure that no one is spoofing the server (extremely unlikely in this situation), just make sure the IP address is correct. Run:
```
host www.stockton.co.za
```
This should spit back the numerical IP address of the server. If it's correct, all is well.
