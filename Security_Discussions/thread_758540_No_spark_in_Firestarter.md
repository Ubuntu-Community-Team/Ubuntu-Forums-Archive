### Post by TenPlus1 on 2008-04-18
Ubuntu is pretty safe out-of-the-box and the only reason you would need Firestarter is if you are either going to set Pc access for networks or allow certain ports to be used...  If you dont have a network, take it off...

As for eth1, it may be a wireless card that hasn't been configured, but so long as your internet connection is working fine, just make sure Firestarter preferences do not point to eth1 on any page...
