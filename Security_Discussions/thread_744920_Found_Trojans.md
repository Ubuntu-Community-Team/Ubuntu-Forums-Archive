### Post by 3rdalbum on 2008-04-04
It's possible for a Windows virus to detect the Linux partition and install a payload into it, completely bypassing the Linux security system. But to my knowledge, no viruses do this. Ordinary Windows viruses pose no threat to your Linux system.

That's no reason not to protect your Windows system, as new viruses are being written every day, and there could be one that merely destroys all data on all connected drives.
