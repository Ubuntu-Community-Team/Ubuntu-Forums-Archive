### Post by HermanAB on 2008-02-15
Simple:
[http://fsf.org](http://fsf.org)

It sounds counter intuitive, but the best way to prevent piracy is to claim copyright, then give it away.  This way, the economic value of the code is destroyed and it is turned into a commodity and you make money by selling your services instead.  

Since you are the expert that knows your code best, people will gladly pay you to do things for them.  You don't need to sell the code.

For example, people pay me big bucks to install and configure systems and write device drivers.
