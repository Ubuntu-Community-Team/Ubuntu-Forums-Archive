### Post by HermanAB on 2008-02-25
Those are likely web sites trying to find browser exploits.  If you were running Windows, then you should be worried.  With Linux, things like that are just an annoyance, but nothing much to worry about.

I've been using Linux since 1998 and haven't been hacked yet.  The important thing is to use LONG passwords for everything - more than 8 characters.
