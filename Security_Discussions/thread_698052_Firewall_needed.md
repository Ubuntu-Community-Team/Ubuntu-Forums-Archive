### Post by MasterJS on 2008-02-15
Well, it is recommended only for servers but if you feel like putting the work in getting all the ports set up right, I'll tell you. See, Ubuntu already has a firewall. Its called iptables. The thing is, you need a GUI to be able to manipulate it. That such GUI would have to be FireStarter.

### Post by emarkd on 2008-02-15
As MasterJS said above, you already have a firewall and all ports are closed by default.  If you need to open some ports, it's a bit difficult to edit the settings, so most people install Firestarter to help control it.  It's in Synaptic.
