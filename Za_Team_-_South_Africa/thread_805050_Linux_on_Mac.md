### Post by chris_nava on 2008-05-23
Yes.
If it's an Intel based processor (made in the last 3 years) you can use bootcamp and install Hardy.
If it's a PPC based processor (older) then you'll need to install Gutsy.

I had a bit of trouble getting wireless working on my PPC based laptop.  You may want to read up on that before attempting the install. (As you will need a few files from your MacOS install on a thumb drive.)
