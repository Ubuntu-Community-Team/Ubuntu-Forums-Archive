### Post by Sho on 2006-02-28
- Konversation 0.19 has been accepted into (K)Ubuntu Dapper at the end of January. It is part of the default installation as the primary fully-featured IRC client of that distribution.

- Konversation is not a library, and it's not a language interpreter ;-). The app is essentially isolated from the rest of the tree. Dependencies have not changed in a meaningful way (we do depend on KDE 3.4 now, which is met by Breezy).

- Konversation 0.19 is a major bugfix and enhanced functionality release. A large number of known issues from Konversation 0.18 have been fixed, and many features requested by the user community have been implemented. A detailed changelog is available on our [website](http://konversation.kde.org/) and over at KDE-Apps.org: [http://www.kde-apps.org/content/show.php?content=9927](http://www.kde-apps.org/content/show.php?content=9927)

- The 0.19 release, about a month old now, has been very successful for the project. No critical bugs have been reported so far, and distributions like OpenSuSE 10.1 have begun including us in our default installation selection.
