### Post by lisa-m on 2005-12-07
[http://snackamp.sourceforge.net/](http://snackamp.sourceforge.net/)
 i would like to have this software backported, it's a very powerful mediaplyer and has so many features it'll put rythmbox, beep-media-player, xmms and many others to shame... 

Features

    * Plays mp3 and Ogg/Vorbis/Wav (or any snack compatible file).
    * Plays mp3 and ogg streams either by specifying the url or using a web browser, including podcasted files.
    * Fully integrated media manager, including tag editors, built for very large media collections. Fast indexing and cataloging of tracks.
    * Create, save and restore conventional play lists of individual songs.
    * Unique AutoPlaylist feature, allows you to flag folders (albums) of tracks to be included in the playlist. You may then sort/shuffle in many different ways.
    * Cross platform, runs on any platform the snack 2.2 extension runs on include Windows 9x, NT, 2000, Macintosh, Unix, Linux. On Windows taskbar icon provides pause and other options.
    * Built in Web server to allow remote control of player and serve HTML user manual. Also acts as a Shoutcast/Icecast media server to media clients to allow you to share your media. Provides a powerful web interface to allow selection of desired tracks.
    * Uses enhanced Amp MP3 and Ogg Vorbis decoders built into snack, high decode quality and low CPU utilization.
    * Auto level adjustment and gapless playback of files, does not modify file itself (useful for files encoded using different encoders) .
    * Decodes ID3V2 and ID3V1 tags and displays them when available.
    * Writes ID3V2/V1 and Ogg tags, including existing tag preservation and Unicode support.
    * Automatically clean up mangled names and ID3 tags from the APL manager.
    * Powerful sorting/shuffling capabilities for the playlist, supports metadata fields. User can save sorting configurations.
    * Rename misnamed files directly from playlist.
    * Uses metakit database for fast access.
    * Can monitor folders for incoming music files (e.g. while ripping).
    * User definable hotkeys.
    * Database includes Track Ratings, Last Play times and many more fields for sorting, filtering, and exporting.
    * Built-in support for LIRC/WinLRIC remote controls
    * Support for album art, including searching google for images and dragging them to folders.
    * Support for attached devices such as mp3 players and key fobs, will auto mount as required.
    * Color and Font schemes may be saved and recalled
    * Freedb query for album tagging (both discid generation and text searching) (new)

latest linux version is snackAmp-3.1.2
