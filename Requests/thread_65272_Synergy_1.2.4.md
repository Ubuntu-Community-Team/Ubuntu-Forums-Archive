I'm pretty new to the ubuntu scene, screwing around with my laptop for the past few months but I noticed one program I really think is helpful that is quite old in the Ubuntu package repository: Synergy. The version is currently 1.0.14 which was released back in August of 2003. It seems that the author has been updating his program since January of this year, and has made quite a few advances/bug fixes. It would be really helpful to have an up-to-date version of this program.

About the program: Synergy is a program that, simply put, turns TCP/IP into a KM switch. It allows for two computers of different operating systems (MacOSX/Windows/Unix variants) to use the same keyboard and mouse by using whatever network the computers are attached to. It's quite simple (no encryption) but is a quite amazing program.
