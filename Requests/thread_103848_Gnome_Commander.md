### Post by prizrak on 2005-12-14
It's a nice little file manager, there is an older version in the repositories right now but it has some issues. The issue I experienced was that while I get to choose custom applications to handle certain file types (such as .ogg or .avi) it doesn't actually start them when trying to open the file. It gives "Application Undefined" error. I grabbed the latest version from [http://www.nongnu.org/gcmd/](http://www.nongnu.org/gcmd/) took the .rpm and installed it with alien, the install worked and even my custom apps were remmembered. The issue is also gone. Nothing is dependent on it (AFAIK) so I don't think it would be a very hard backport. However this is a very low priority one as you CAN use the .rpm so if you decide not to backport it I won't flame you :)
