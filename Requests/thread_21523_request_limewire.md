### Post by Technoviking on 2005-03-22
Another way to install LimeWire is to convert the LimeWire rpm via alien:
```
sudo alien LimeWireLinux.rpm
``` 
Then install the created /.deb file
```
sudo dpkg -i linuxwirelinux.deb
``` 
The advantage of doing it this way is that your icons are automatically created.

Mike
