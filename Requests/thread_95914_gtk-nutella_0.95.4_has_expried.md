### Post by Turtle.net on 2005-11-30
Hi,
I really wanted to update gtk-gnutella ... so I updated it myself
First go download the .deb file you want at [http://sourceforge.net/project/showfiles.php?group_id=4467&package_id=4479&release_id=373073](http://sourceforge.net/project/showfiles.php?group_id=4467&package_id=4479&release_id=373073)

I personnaly chose the GTK1 because I want something with the less CPU consumption.

after that it's really simple:
```
 $ sudo dpkg -i GTK1_gtk-gnutella_0.96.0-0_i386.deb
```

Now I can run gtk-gnutella 0.96, and I can even uninstall it through synaptic if I want ;)
