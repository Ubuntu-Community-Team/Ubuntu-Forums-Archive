### Post by Galoot on 2006-01-27
[Backup Manager]("http://www.backup-manager.org/") version 0.5.8 is in the universe/admin repositories.

It is a simple and easily configurable command line backup tool written in bash and perl suitable for desktop use. Aside from perl, its only dependency is gettext for localization. It's simply an actively developed script. (The most recent version is 03-Jan-2006.)

[Version 0.6]("http://www.backup-manager.org/releases/06/") adds many new features and is "the first stable version" according to the author.> It gathers features and improvements of more than a year of developement. In this release, several backup methods are available: tarballs, incremental tarballs, mysql databases, svn repositories and even generic methods. Multiple upload methods are also supported: FTP, SSH and RSYNC. Automatic burning of DVD/CDR media is possible and external hook scripts can be defined for customizing the backup process. The program is fully translated in 5 languages: French, German, Spanish, Italian and Vietnamese. This release comes with the text version of the user guide, which is also available in PDF and in HTML in the website. Every bug reported in previous versions has been closed.

Backup Manager 0.6.1 is a bugfix for 6.0 and can be found [here]("http://www.backup-manager.org/releases/061/").
