### Post by Vincentu on 2005-09-28
Not sure if this should be a backport request or an update request for Breezy :
[mmm-mode]("http://mmm-mode.sourceforge.net/") is an emacsen (read GNU-Emacs/XEmacs) elisp package needed to use multiple major modes (think editing jsp, php, html with embedded javascript or css, etc.).
The current version in Hoary and Breezy is [0.4.7-3]("http://packages.ubuntu.com/cgi-bin/search_packages.pl?searchon=names&version=all&exact=1&keywords=mmm-mode").
That version is badly broken: font-lock (colors) works but [indentation does not]("http://www.emacswiki.org/cgi-bin/emacs-fr/HtmlModeDeluxe").
The "new" version ([0.4.8]("http://packages.debian.org/changelogs/pool/main/m/mmm-mode/mmm-mode_0.4.8-2/changelog") released [2004-06-16]("http://sourceforge.net/project/showfiles.php?group_id=8658")) is now in [Debian unstable and testing]("http://packages.qa.debian.org/m/mmm-mode.html").
I thought that Breezy would include the 0.4.8-2 (or at least 0.4.8-1) but the actual pool shows just the old broken 0.4.7-3 like in Hoary.
Maybe I should address that request to the maintainer of that Universe package in Breezy, but I was unable to find a contact to the right person.
Anyway thanks a lot for the fantastic work so far.
