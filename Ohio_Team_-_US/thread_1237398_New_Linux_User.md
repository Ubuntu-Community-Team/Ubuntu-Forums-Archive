### Post by jpeddicord on 2009-08-11
GIMP currently only supports RGB color profiles -- meaning you can only work with images designed for digital display. You can print RGB images, but they won't look as exactly as they do on screen. When you use CMYK, you're using the color profile designed for print, so colors come out more vibrant on paper. I personally have never used CMYK as I don't do much printing, but there are some photo people out there who need it.
