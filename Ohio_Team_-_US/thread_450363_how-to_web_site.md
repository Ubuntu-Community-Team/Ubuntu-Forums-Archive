I am currently working on a "how-to" website for Ubuntu. It has 3 goals.

1. Help new users find the FOSS programs they need to make Ubuntu useable.

2. Help them install Ubuntu and the software they need.

3. Raise awareness for FOSS in general.

I am working on a database for the programs. It will include information from the each program's homepage and Wikipedia. It will also let users know what platform the program can be used with (Linux, Apple, Windows). Then there will be room for users comments.
