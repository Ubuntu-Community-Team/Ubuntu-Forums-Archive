### Post by runningwithscissors on 2007-10-31
I have 2GB of memory and I don't think I've ever needed to use my swap partition.

The only time it was used was when I loaded an _entire_ Gentoo install into RAM because I needed to compile x86 binaries for a program and the coss compiler was giving me trouble.

Great if you can buy 16GB and all, but I don't think it'll be used much by the machine. (Or maybe it will, I know that programs for the Apple PC love to chomp on memory)
