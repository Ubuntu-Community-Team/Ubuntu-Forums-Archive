### Post by yoasif on 2007-04-25
I'd say it's either a proxy or bad DNS server that is causing this issue.

Of course, the easiest way to check for sure is to check the hosts file or setting the DNS manually to something like opendns, and disabling proxies. 

Spyware for Mac OS X is a rare thing, and since Mac users have to authenticate to install apps that change system configurations like this, it's rather unlikely, unless your friend is installing some weird random app that does this.

No amount of security can stop stupid/careless users.
