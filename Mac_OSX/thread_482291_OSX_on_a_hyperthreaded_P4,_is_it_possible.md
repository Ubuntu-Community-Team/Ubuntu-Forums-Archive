### Post by iamhugeinjapan on 2007-06-23
1) yes

2) OS X is designed to only install on  Apple computers. It's not designed to be run on any machine you want. However certain enterprising people have hacked it to install (illegally) on various non-Apple computers. I don't feel it's appropriate to give you any pointers on that here though, so you'll need to do your own investigation.

I have heard talk of VMware images of OS X working on other computers via VMware Player/Server/Workstation, but I've never seen it myself. I'm not sure what the legal issues with that are.
