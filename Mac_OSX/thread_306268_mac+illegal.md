### Post by aysiu on 2006-11-24
If you read the [**Mac OS X user license agreement**](http://images.apple.com/legal/sla/macosx104.pdf), you'll see that you are licensed to install it on only Apple computers.

So if your country honors both parties adhering to license agreements, then, yes, it is illegal to install Mac OS X on a non-Apple computer.
