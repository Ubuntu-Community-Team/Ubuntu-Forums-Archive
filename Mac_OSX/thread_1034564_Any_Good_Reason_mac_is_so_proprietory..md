### Post by Frak on 2009-01-09
> **emshains said:**
> it only runs on mac hardware, so there is no way you could run it on any kind of PC.

Mac OS X is, on the hardware access level, just a modified version of BSD that runs on x86 hardware. It has standard computer parts, with the exception of the EFI chip, which is not *always* included on motherboards. There's no magical Mac hardware, even the PPC Mac's used IBM-PC hardware.
