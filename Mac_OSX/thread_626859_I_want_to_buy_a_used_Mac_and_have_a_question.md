### Post by Tipo on 2007-11-29
If you clear the HD, you would need a copy of an Operating System on a CD somewhere in order to restore it.  You *could* just boot it up (if it already has an OS on it) and delete all the files, but this could be a pain seeing as users and admins would be different - I'd recommend doing a clean install.

Age of the mac is kind of murky, a lot of older macs can run the latest OS... Compatibility is based on the processor speed. Leopard will work on Macs with G4 processors that are 867 MHz or faster (although if the processor is that slow, I would recommend installing Tiger on the machine).

More Leopard specs available here: [http://www.apple.com/macosx/techspecs/]("http://www.apple.com/macosx/techspecs/")
