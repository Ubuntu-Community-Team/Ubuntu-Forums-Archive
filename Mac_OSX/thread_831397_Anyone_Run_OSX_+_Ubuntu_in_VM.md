### Post by shawnansasio on 2008-07-01
To Get see you mac home folder go to System Prefences > Sharing > Enable windows sharing.
Then in VMware Go to Places > Network > Then if you don't see your mac there go to Windows Network > Click on your mac then type your username and password.
