I think you need [Tinker Tool]("http://mac.softpedia.com/get/Utilities/TinkerTool.shtml"). I've got a slightly dated version of it, and it certainly has a tab for adjusting fonts. Right up to 288 point, which should be big enough. :wink:

I haven't tried to adjust any fonts, because the version I've got I installed under Tiger and this is the first time I've opened it in Leopard and it's telling me to upgrade. But all the reports I've heard of Tinker Tool are good.

I really don't know whether you can do this without a third-party app, but tell me - how did you adjust your desktop fonts in Leopard? I've looked all over System Preferences and can't find where you do it.
