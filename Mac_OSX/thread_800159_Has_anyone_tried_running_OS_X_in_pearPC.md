### Post by Incense on 2008-05-19
I ran Tiger on it a couple years ago. It took hours to install OS X, and once it was installed everything was very slow (remember the program is emulating the PowerPC chip) and networking didn't work at all. So while it was fun to open TextEdit, and watch safari not load pages, it really didn't feel worth it by the time I was done. I hear that Panther works better, but I only had a copy of PPC Tiger. (Yeah you need a PPC copy of OSX, the intel version will not work). If you really want to run OS X on linux, I would recommend installing ubuntu on your PPC Mac, then running [Mac On Linux]("http://mac-on-linux.sourceforge.net/"). THAT runs VERY well.
