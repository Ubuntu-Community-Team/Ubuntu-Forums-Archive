Hey everybody,

I just created a useful search plugin for GetDeb.net for use within Firefox Search Engines.

getdeb.net is a nice website that provides prepackaged .deb archives. It can sometimes come in handy.

Just rename the text file to "getdebsearch.src" and add it (and an icon) to /usr/lib/firefox-3.0.5/searchplugins folder

replace "firefox-3.0.5" with your version of firefox

There is an official one from GetDeb, but it lacks an icon. [http://www.getdeb.net/getdeb_search.xml](http://www.getdeb.net/getdeb_search.xml)
