### Post by nexzzt on 2016-09-26
While creating the recording rule (or you can edit it after), you have to go to the "Schedule Options" menu, then the "Schedule Filters" menu, then select "This Channel" (tick the box). Haven't found an easier way yet. If anyone has found an easier way, please share. Would really appreciate it.

EDIT: Just found that if you using the program guide or program finder, when you press select/enter on the desired listing, you get the following options:

Record this showing
Record all showings
Record one showing (this episode)
Record all showings (this channel)
Edit recording rule

The second last option should do what you need.

Cheers
Ben
