### Post by rhpot1991 on 2008-08-28
You would need to transcode them first.  Over on wiki.mythtv.com there was an automatic commercial flag removal script, but I was never happy with the results.  Because things can go so wrong I always felt the user should be involved when removing commercials.  So you can either do them by hand or get the automatic script into place, do either of them and then use mythexport.
