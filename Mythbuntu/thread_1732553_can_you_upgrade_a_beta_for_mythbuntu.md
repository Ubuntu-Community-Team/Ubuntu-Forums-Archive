### Post by klc5555 on 2011-04-18
Generally with all the Ubuntus, if you've installed early a beta or release candidate of version "X", Update Manager will indicate and update each package as a later one is released on the march toward version "X". By the release date, this fairly automatic package-by-package updating will have resulted in your total installation being at the same release-level version "X" as everyone else who installed just the final release version. And like everyone else, your installation will continue to be kept current by Update Manager thereafter.

So no fancy apt-get or synaptic work required at all. No reinstall either.
