### Post by mal on 2009-11-29
I was able to get my 9.10 diskless clients to power down properly with the following changes to /opt/ltsp/i386/etc/init.d/mythbuntu-diskless-client:

Change line 36 and 38 from:

```
mkdir -p /var/run/sendsigs.omit.d/
...
pidof nbd-client > /var/run/sendsigs.omit.d/nbd-client
```

To:

```
mkdir -p /lib/init/rw/sendsigs.omit.d/
...
pidof nbd-client > /lib/init/rw/sendsigs.omit.d/nbd-client
```

This was discussed [in this thread]("http://ubuntuforums.org/showthread.php?t=1157560").

Hope this helps.

Mal
