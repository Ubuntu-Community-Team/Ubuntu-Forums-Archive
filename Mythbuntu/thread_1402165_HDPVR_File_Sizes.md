### Post by Chuffinora on 2010-02-08
Does a set-top box settings affect file sizes for an HDPVR?

I just swapped out an SA3250 STB for a SA4250 box,  with no changes to my MythTV system (other than modifying the firewire channel change script).  My recording size has gone from 1.5GB per hour to 3.9GB per hour?

To my eye I can see no quality differences in the recordings.  Are there any settings I should adjust on the SA4250 box?

---

### Post by neutron68 on 2010-02-08
Yes, the STB output resolution will effect the HD-PVR file size.

For example, maybe your old STB was set to output 1280x720 (720p) and maybe the new box is set to output 1920x1080 (1080i)?  The 1080i signal has more information in it than the 720p signal does, thus a bigger file.
