Simple question, is the Hauppauge HD-PVR device supported in ubuntu?  If not are there plans on adding the required driver etc?

---

### Post by novellahub on 2008-09-30
It is not supported under a standard Mythbuntu install.  The future version of Mythtv 0.22 adds support.  You can use it by enabling the Mythv trunk builds:

[http://www.mythbuntu.org/auto-builds](http://www.mythbuntu.org/auto-builds)

Then you will have to install the driver using the instructions from here:

[http://www.mythtv.org/wiki/index.php/Hauppauge_HD-PVR](http://www.mythtv.org/wiki/index.php/Hauppauge_HD-PVR)
