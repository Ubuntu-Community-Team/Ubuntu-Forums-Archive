### Post by azmyth on 2010-02-16
Yes if using the GUI based synaptic.  So say you want to do an upgrade, hit the "mark all upgrades" icon. On the left hand side where it says custom filters select this. One of the choices will be "marked changes" select this. You'll then see what's going to be changed.
