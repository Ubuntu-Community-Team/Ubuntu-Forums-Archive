Then go ahead and
```

```

After all is done give it a few test runs, see the wiki if you are unfamiliar with its usage: [https://help.ubuntu.com/community/MythExport_old](https://help.ubuntu.com/community/MythExport_old)

Then when all is said and done leave a comment on the bugs mentioned above.

It is worth noting that this is the same code that has been on the Mythbuntu Testing PPA for months now, so if you are running that code this should be a nice easy test.
