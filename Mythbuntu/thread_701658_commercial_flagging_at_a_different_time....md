### Post by foxbuntu on 2008-02-20
You could create a cron job and a script to use mythcommflag, however from inside mythtv there is no option for this. You can however set the CPU usage to low and user jobs to 1 in mythtv-setup (Backend Setup) and that will make sure to keep the CPU usuage low for playback.
