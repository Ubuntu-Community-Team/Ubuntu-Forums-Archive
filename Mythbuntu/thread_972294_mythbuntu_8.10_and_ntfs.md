### Post by spoky99 on 2008-11-05
If you want use a ntfs partition disk for save record with mytbuntu you must install ntfs tools by synaptic and give write permission to the ntfs drive otherwise when you aren't in able to select the tv into mythtv (after a few second of freshing  mythtv return to the menu without see the tv)
The problem is that without the ntfs tools you could not give the write permission in the ntfs folder and you aren't in able to change the group to mythtv.
