Is there a way to set a value for the audio sync delay? On my system the audio seems be 200mS out all the time and it is a PITA to set it every time we watch a recording or livetv.

### Post by AlecJ on 2010-06-15
The setting is in General Playback page 1/9

make sure "Use video as timebase" is ticked

If you see the sync go out during playback, skip back a few seconds. This allows a bigger buffer and time to correct the sync.
