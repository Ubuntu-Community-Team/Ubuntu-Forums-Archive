### Post by AlecJ on 2009-05-31
go to..Setup/Media settings/Videos settings/General settings in the frontend

then type in the path to your new drives, using a colon : for the different paths.
I use a USB drive to store my movies.

/media/Lacie/vidoes:/var/lib/mythtv/videos      etc

The same can be done with Music in the music setting menu.
