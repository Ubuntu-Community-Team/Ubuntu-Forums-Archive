### Post by thatguruguy on 2010-12-09
> **Crowie said:**
> Just out of interest how did you go about getting your metadata for the games.  In particular the backgrounds and cover.  Did you have to do each game separately or is there a script lurking about somewhere.  Still using 0.23.1.

One difference between .23 and .24 is that mythgame in .24 includes the ability to scan the internet for metadata.  However, it only pulls game information, the box cover and a screenshot.  It doesn't pull fanart (the background).

Certain games don't have box covers available.  In the case of aquaria, I just used google to find a screenshot I liked to use as fanart, and placed it in the fanart folder.  I created the box cover myself in gimp from a couple different images, and placed it in the cover folder.  Then I manually pointed the metadata of the game to the images I had stored.
