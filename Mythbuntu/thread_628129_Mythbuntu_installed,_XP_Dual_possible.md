### Post by uopjohnson on 2007-12-01
You can use a partition manager to change the size of your current partitions and then install windows.  You would then need to reload grub as the bootloader because windows replaces it without asking.  If you don't have any recordings and you haven't done a lot of setup I would highly recommend starting from scratch.  Install windows xp first, leaving most of your drive un partitioned.  Then install mythbuntu.  This is much much easier if you don't have a lot of experience.   If you don't want to go this route you should probalby post in the main install forum because you are more likely to find someone to help you with the specifics there.
