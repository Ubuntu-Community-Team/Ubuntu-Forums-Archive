### Post by dibuntux on 2011-12-24
Hi everybody, just a recap for using the internal UPnP server of myth.

I just set up on my Android phone (LG Optimus One, mod with HW acceleration & JIT on, OC to 720 MHz) a nice UPnP client, BubbleUPnP, that works well with myth. Audio is done via internal player (ok mp3 & flac, no Ogg Vorbis on Android 2.2) & video is done externally. I used MX Player with soft codec for Arm V6. I'm obtaining 10-20 fps on standard DVB recordings. Other videos, well depends on the kind. I'm just asking to much from my phone.

One problem I researched around is videos settings for UPnP. It all depends on the directory set in Setup-Media Settings-Video Settings-General Settings:Dir that hold Videos. This is becouse myth 0.24 does not support Storage Group. So you can only have one location set for videos.

Using UPnP you can easily setup a frontend using a box like WD Live TV and watch even LiveTV: you just have to set a recording first, no channel zapping.

Hope it helps. Merry Xmas to all!
