### Post by maddentim on 2012-11-16
I had the same problem as you. with it only registering 1 dvb. I found the solution for me to be the card number in my /etc/modprobe.d/ file. The suggestion was for my file to have in it:
```
options saa7164 card=8
```

but I really needed to have card 7:
```
options saa7164 card=7
```

I got my clue from:
[http://linuxtv.org/hg/v4l-dvb/file/tip/linux/Documentation/video4linux/CARDLIST.saa7164](http://linuxtv.org/hg/v4l-dvb/file/tip/linux/Documentation/video4linux/CARDLIST.saa7164)

I must have gotten an older card (even though I just bought it!)
