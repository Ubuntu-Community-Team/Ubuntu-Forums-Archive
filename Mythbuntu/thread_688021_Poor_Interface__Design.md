Ok,

When I am selecting items on the menu I use the Enter Key, now assigned to the BIG OK button on my remote.

Oddly enough when I view the guide during live TV pressing the Enter button or BOG OK button MyhtTV wants to record the show.

I would much rather the Enter key or BIG OK button select the channel in the guide for me to watch it and the record button setup a recording and sure if Im trying to select a show that is in the future with the OK button then I wold be prompted with a recording option. 

Any ideas how I can change this behavior?

### Post by ridshack on 2008-02-04
I got the answer on what I thought was a dead post.

It's a setting in mythfrontend utils/setup > setup > tv settings > programme guide. On the 2nd page of that you'll see 'Use select to change channel in the programme guide'. Make sure that's ticked. When that's set move all the way to 'finish' to store the setting change.

Thanks to justinh2!
