### Post by jregan on 2012-06-04
Bought the app.  Very attractive UI, simple to use, my kids have an easier time using it to control the frontend/choose their shows than using the frontend.  

Streaming works well with 12.04 (though it needs a lot of CPU!).  Neat to show off recordings to coworkers from the iPad.

Remote control a lot easier to use than setting up LIRC, and a lot more information packed.

All in all, the new API is very cool if it means more applications like this.
