Is upgrade for existing mythbuntu users still an option or will a fresh install (xubuntu + mythtv) be required?

The packages will be removed from 17.04 and later, so you'll still be able to upgrade to 16.04. And the only packages that are getting removed are the Mythbuntu specific packages, so you'll still be able to upgrade after that you just won't have the Mythbuntu packages anymore (Mythbuntu-control-centre, mythbuntu-desktop, etc). MythTV will still be in the repos and the PPA will still be available (which you should really be running from anyway)

### Post by gottabefoss on 2017-01-10
I had no success doing an in-place upgrade from 14.04 to 16.04. Something broke every time I tried.

So I did a fresh install of Lubuntu 16.04, added the mythbuntu 0.28 repository, and installed mythtv and mythweb. Everything went smoothly. I had no problems whatsoever restoring my mythtv database and recordings from backup.

So as long as the mythbuntu repository keeps going, not having the mythbuntu distribution in and of itself isn't such a big deal. The repository mythtv install works really well.
