### Post by ridgeland on 2010-01-02
I hate navigating the setting interface without a mouse.
To have the mouse in the backend you have to turn the mouse on in the frontend.  really.
In the frontend there is an option box:
Hide Mouse Cursor in MythTV 
I think it's in 
Utilities/Setup -> Setup -> Appearance
The default is to have the mouse OFF - nicer when viewing a show.
