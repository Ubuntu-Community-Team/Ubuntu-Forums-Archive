### Post by zymba on 2008-09-25
I have this case and I am using a Logitech Harmony 550 remote with the bundled IR receiver.

I followed this guide to get it working

[http://ubuntuforums.org/showthread.php?t=779674](http://ubuntuforums.org/showthread.php?t=779674)

But also be aware not every remote will work with this receiver. Apparently, only MCE remotes work with it. The Logitech can be configured as an MCE so I had no problem generating my own hardware.conf.
