### Post by newlinux on 2008-02-06
I have not tried this, but I would guess it would be possible. In mythvideo you could setup .ppt or .pps files to be associated with impress so that it launches impress (ooimpress -show to start off in show mode) when you encounter those files. Then you could probably use LIRC (or at least lircmd) to go back and forth in the slide show , and to exit impress via remote. I don't know if ooimpress works with LIRC, but between lircmd and irexec you should be able to go forth and back and exit.
