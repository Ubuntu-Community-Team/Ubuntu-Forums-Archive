### Post by doudar on 2010-01-03
I really liked the idea of mythpywii but lacking volume control was a show stopper for my wife so I quickly hacked this version together. The changes are "B"+tilt now control volume, and "A"+"B"+tilt control the playback speed. Not much debugging has been done on it at all. 

Just take this script (attached) and drop it in place  of your existing script. If you need to install from scratch, the original author has instructions on his page [http://www.benjiegillam.com/mythpywii-installation/](http://www.benjiegillam.com/mythpywii-installation/) - just use this script instead of the one on his page. If you have any problems feel free to post them here.
