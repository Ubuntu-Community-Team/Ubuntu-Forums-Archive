### Post by whatthefunk on 2013-01-11
Which package manager do you use?  In most, you can turn off automatic updates.  For example, if you use Synaptic, go to Settings > Repositories.  In the Software Sources window that opens, go to the Updates tab and uncheck "Check for updates."
