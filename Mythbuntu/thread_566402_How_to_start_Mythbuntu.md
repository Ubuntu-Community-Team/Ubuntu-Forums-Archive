### Post by lmicu on 2008-12-31
> **Plexus81 said:**
> *never used linux before*

I've just installed ubuntu, then upgraded it to 7.10 -> installed mythbuntu.

But how do I start the media center inside ubuntu?

Well in order to have MythTv (because that is what I assumed you wanned in the first place) you will have to install it also by typing:

sudo apt-get install mythtv (or sudo apt-get install mythtv-frontend)


and then you will be able to start it from the menu or from a terminal by typing "mythtv"

(I hope this helps)


PS: If you are trying to start Mythbuntu then you will have to open it from System-Administration
