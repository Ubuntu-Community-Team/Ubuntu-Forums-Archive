### Post by murchball on 2008-05-15
I've got a Harmony 880 working with Mythbuntu. The first step is to get your existing remote to work with Myth. You didn't mention this, but I'm guessing that you have a Hauppauge remote. Get this to work first before trying to get the 880 to work. 

I believe that when you add the myth box to your Harmony setup, you need to set it up as a media center pc and then there is an option for MythTV. You then need to choose the remote that you want the 880 to mimic (the Hauppauge?). 

The limitation and strength of the Harmony software is that it's all wizard driven. Easy to set up initially, but it can be hard to customize. There is an open source alternative, I haven't tried it yet, but it looks very interesting. []("http://www.engadgethd.com/2008/05/05/concordance-enables-logitech-harmony-programming-in-unix-linux/")
