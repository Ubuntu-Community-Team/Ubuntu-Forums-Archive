### Post by sanderd17 on 2011-08-04
There are no big differences between 32-bit and 64-bit if you have only 3GB of RAM or less. Once you have more than 3 GB of RAM, than 64-bit has an advantage.

Most apps (certainly all popular ones like VLC and XBMC) work on 64-bit and 32-bit. I haven't found any maintained app that doesn't work on 64-bit.

### Post by thatguruguy on 2011-08-05
If you process a lot of videos, there is a marked increase in speed if you use 64-bit, due to improved multi-threading.

I have dolphin-emu installed on my mythbox, and there is a significant improvement in perfromance when using the 64-bit version.
