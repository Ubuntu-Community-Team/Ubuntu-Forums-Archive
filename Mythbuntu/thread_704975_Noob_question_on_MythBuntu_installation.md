### Post by newlinux on 2008-02-22
You can install all the other apps with mythbuntu, and they'll still be around after upgrades. Pretty much anything you can install in ubuntu you can install in mythbuntu.

As to whether you are better off starting off with desktop ubuntu and putting mythtv on that - if you are pretty familiar with Myth and ubuntu, you probably don't need the extra ease that comes with a mythbuntu install - I'd be inclined to go that way (I've never actually installed mythbuntu, I run the myth packages on an ubuntu desktop) - but others might have different opinions. You really can't lose either way... If you like the regular desktop and know you will use a lot as such, start with that...
