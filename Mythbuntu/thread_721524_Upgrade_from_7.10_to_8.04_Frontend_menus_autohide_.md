Yesterday I upgraded to 8.04 using the command as listed in the above sticky.  I'm wasn't home while it ran, so i'm not sure about any install errors.  When the frontend loads, the menu only displays for a second or so, then all the graphics disappear.  If I navigate the menu the items show up again for a second, then hide again.  This wasn't a problem in 7.10.  Let me know what info to post to help track down the cause.

EDIT: Changing to a different menu style seems to fix it.  The default (mythbuntu) is the one that autohides, but MythCenter seems fine.

EDIT again: MythCentre doesn't respond to exit [Esc] command to leave the frontend.

YET ANOTHER EDIT!  Seems that the ESC problem was that in the setup, if the exit command 'default' is used, you can't exit from the top menu.  if you manually set it, it shows up.
