### Post by esullivan on 2009-02-11
I have an LCD TV in my bedroom.  I was thinking about creating a MythTV box for a PVR, that I could use on that TV (VGA connection).  

My question is, is there a device that I can connect to my living room TV via HDMI or component connection that will work off the MythTV box via network?  Also could I use BOTH at the same time watching two different shows?

---

### Post by nickrout on 2009-02-11
You can conceivably run two front ends from two video cards from the same box. However there are practical difficulties, such as controlling the lounge frontend instance with a remote control when the computer is in the bedroom.

Far better to run myth as it is designed, with a combined frontend/backend at one tv and a remote frontend at the other tv, with ethernet between them.

### Post by Caps18 on 2009-02-12
If you want to watch different shows, what was recommended is correct.

I have my one Mythbuntu box hooked up to three HDTVs and use remote IR extenders around the house.  I'm the only person that uses it though, but it is cool being able to use a HDMI splitter and have the same thing playing downstairs as well as upstairs.
