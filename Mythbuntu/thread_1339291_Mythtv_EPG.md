### Post by SiHa on 2009-11-27
I have a remote key bound to the "live TV in Guide" jumppoint. Pressing this will bring up the guide, where you can select a channel to watch.
I think there is also a checkbox in the setup somewhere (TV settings -> Program Guide?)"Select in guide changes to Live TV" or someting similar.

### Post by Neon Dusk on 2009-11-27
Are you using Mythtv 0.22?

Channel changing in the epg is broken in the version of mythtv that comes with mythbuntu 9.10 (you used to be able to press M to view the channel) - see [http://svn.mythtv.org/trac/ticket/7406](http://svn.mythtv.org/trac/ticket/7406). As SiHa says there is an option in the frontend setup to use select to view live tv or you could install a newer version of 0.22 using [http://www.mythbuntu.org/auto-builds](http://www.mythbuntu.org/auto-builds)
