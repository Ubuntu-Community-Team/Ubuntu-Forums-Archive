### Post by volkswagner on 2008-05-08
> **bergqvistjl said:**
> Can I install Mythbuntu without a TV Tuner card installed, and if i can, can i install one later?

You do not need a tuner.  You can add one later.  It is easiest to have it installed and detected during the OS install.  You should think about if your current machine will be the server.  If it will not be the server than installing mythbuntu now may not be best.  The first install on your network needs to be the master backend (server).

You can install mythbuntu now as a master backend or master backend with frontend.  You can then later add machines to the network.  You can add a frontend only and install a tuner in the master.  You can add a slave backend/frontend and intall the tuner in the new machine.  

A tuner is only required to watch tv.  You can use other media funtions.

Happy Mythbunting.
