### Post by Flecko on 2009-11-29
I had the same problem with 9.10. In 9.04 you could pick the seperator character for channels such as "_" or "." but it appears that this option is gone in 9.10.

What I did was run Mythtv-Backend-Setup from the Services menu and then edit the channel names manually. Pain in the butt, but only has to be done once.

Best of luck!
