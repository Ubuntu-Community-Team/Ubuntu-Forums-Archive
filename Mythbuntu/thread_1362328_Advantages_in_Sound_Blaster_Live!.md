### Post by pu15e on 2009-12-23
If you're an audiophile with a great ear for quality, use the Live. Otherwise, keep it for something else down the track ;-)

In other words, the Live is likely to throw a *slightly* better quality sound. Unless you have some crazy audio gear, though (in which case you'll be using connectors the Live doesn't have), you'll likely never hear the difference...

The Value version didn't have any really redeeming features, at any rate. I actually have one in my desktop machine, but only for use in my gaming instance of XP so I can use it's gameport with my old MS Sidewinder Pro F/F. The rest of the card is disabled. It's completely disabled in the other OS's the PC boots as the P5e mainboard already has a fairly decent audio solution...
