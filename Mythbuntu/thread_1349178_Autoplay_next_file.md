### Post by SiHa on 2009-12-08
> In Mythtv .22 pressing 'i' over a folder will give you a popup box
asking if you want to play all videos in the folder....or if it doesn't play them in the order you want, you can go into the 'Edit Metadata' menu, there's a option where you can specify a file to always autoplay next.
Unfortunately, in the Mythbuntu Default theme, it gets truncated to 'File to always...'.

It would be a bit tedious, but if there's you're going to watch them often it might be worth it.
