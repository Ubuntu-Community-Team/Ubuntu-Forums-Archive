### Post by jb5 on 2009-07-18
I don't know if you've considered this, but.....

I had a problem with some of the screen areas, frames, text sizes etc. for a theme I used at the time, and in the end edited the appropriate .xml file.

It took a bit of trial and error to get it how I wanted it, but got there in the end. (You need to quit the frontend and then restart to see the changes).

Also, after an update I then had to re-establish my version of the .xml file. (Used a little bash script to automate this process).

So you could start by decreasing the font-sizes directly in the weather xml file for the theme you want to use. 

(The first adjustments were the hardest for me as I wasn't exactly sure which xml file to edit, but as there is a weather-ui.xml file that should make things a little easier. Make a backup first though!)

EDIT - Use the text sizes from the blue theme as your starting point.
