### Post by wd5iyt on 2008-04-18
I just submitted a bug on an issue I had installing multiple Mythbuntu systems at home.

I built one machine using the "Alternate Install" CD.  It worked fine.  The second machine I built with the Live CD.  When I tried to NFS mount between them, I discovered the UID/GID for the "mythtv" user was different between the two installs.

This means you can't mix systems built with the Live CD and ones built with the Alternate Install in the same mythtv system without some hacking.

This ought to be fixed as it would confuse a new user, especially one not familiar with how UID's and GID's work with NFS.
