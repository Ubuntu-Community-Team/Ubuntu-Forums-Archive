### Post by tgm4883 on 2014-08-20
Not sure why you would look at ubuntuupdates.org, I don't know if there is any affialiation with Ubuntu/Canonical, but it surely isn't the best place to look for this information. 

The best place to look for what MythTV updates the Mythbuntu team provides would be the Mythbuntu website (specifically [http://www.mythbuntu.org/repos](http://www.mythbuntu.org/repos) ). You can read all the text or simply glance at the graphic to see what versions of MythTV get built for a specific release. This is also one of the reasons that ubuntuupdates.org is bad, as it says 0.27 is available for raring, which is technically true, however it no longer receives any of the 0.27 updates (this may be the only case where something being technically true isn't the best kind of true).

The second best place to look for what MythTV updates are available for a particular release would be [https://launchpad.net/~mythbuntu](https://launchpad.net/~mythbuntu) where you are able to dive into the PPA for a particular version and see what releases (and which package versions) are available. This appears to be the same place that ubuntuupdates.org pulls its info from, buy why go there when you can get the same info from the source. 

There has been a 0.27 repo for trusty for since trusty started development, so I'm not sure why ubuntuupdates.org says there isn't (again, maybe another reason to not go there). As for the testing repo, it is true that there isn't a trusty or utopic repo for that. I've just pushed a change that should fix that.
