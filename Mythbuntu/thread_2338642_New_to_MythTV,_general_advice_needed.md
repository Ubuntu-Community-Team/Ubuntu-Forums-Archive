### Post by uteck on 2016-10-02
The Mythbuntu Control Center makes setting up things easy.  So setting it up to work with other front ends, or setting up a secondary backend is just a click of the button.

I prefer using Kodi for the frontend since it has more apps for streaming from online services, but the Myth frontend is a lot easier to set up recordings from.  With Kodi I have to login to the Myth webpage to setup recordings, but since I don't record much anymore it is not a hassle.
