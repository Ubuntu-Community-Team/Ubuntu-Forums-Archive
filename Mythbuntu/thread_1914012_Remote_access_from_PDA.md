### Post by wyliecoyoteuk on 2012-01-23
Have you installed the Mythmote app?

once you have done that, in the Mythtv frontend, go to setup>general> last page, tick "enable network remote".
then go to settings in Mythmote on the phone and create a new location profile with the ip address of the PC and the port number (default is 6546).
Or if you want to use the web interface, make sure that MythWeb is installed.
