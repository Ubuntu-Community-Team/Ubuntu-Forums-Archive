### Post by Hayesio on 2008-11-27
A default installation of Mythbuntu will install all the MythTV plugins like MythVideo, MythDVD, MythMusic etc.  These plugins give you the functionality you are asking for.  Mythbuntu will work fine without a tv tuner - you just cant what TV.  Also, if you just want the video/music/DVD function, you dont need a backend.  However, if you intend on setting up the TV later, i'd recomment installing the backend from the start anyway.
