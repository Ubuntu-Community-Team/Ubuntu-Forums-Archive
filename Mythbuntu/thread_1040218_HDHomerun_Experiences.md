### Post by murchball on 2009-01-15
I don't have any experience with the HDHomeRun on satellite, but mine work great for capturing HD with my antenna or Clear QAM over cable. From what I've read, satellite doesn't transmit clear QAM, so the HDHomeRun doesn't sound like the right solution unless you're planning on using an antenna. The only option for recording HD from satellite is analog component capture with the Hauppauge HD-PVR. Support for this is available in the development builds, but it's not for beginners yet.
