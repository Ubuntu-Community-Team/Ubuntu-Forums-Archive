### Post by superm1 on 2012-04-17
Hi Folks,

We're about to spin the candidate images for Mythbuntu 12.04 LTS.  Rather than find lots of problems after launch, it would be amazing if we could get some community testing now for any last minute issues that need to be fixed in the ISO images.

It is my opinion (and that of the other Mythbuntu developers) that these images are quite stable at this point.  I'm running 12.04 in production myself.

12.04 includes MythTV 0.25 and the mythbuntu repositories MCC plugin built in automatically.

So if you are looking for an excuse to upgrade, now is the time!

You can fetch images from: [http://cdimages.ubuntu.com/mythbuntu/daily-live/current/](http://cdimages.ubuntu.com/mythbuntu/daily-live/current/)

If you find problems, please report them on launchpad at [http://bugs.launchpad.net/mythbuntu](http://bugs.launchpad.net/mythbuntu)

If you did happen to test, feel free to report whether you did or didn't have problems at the ISO tracker at:
[http://iso.qa.ubuntu.com/qatracker/milestones/204/builds](http://iso.qa.ubuntu.com/qatracker/milestones/204/builds)

There are two Mythbuntu sections there.

MythTV 0.25 is available for earlier releases of Ubuntu as well at the Mythbuntu repos ([http://mythbuntu.org/repos](http://mythbuntu.org/repos)), so if you want to upgrade other machines to MythTV 0.25 but not necessarily Mythbuntu 12.04, you can do that too.

Thanks everyone!
