### Post by declanmullen on 2011-09-03
I've got 4 physical tuners in my combined mythtv 0.24.1 frontend/backend system. I'm sucessfully recording shows via them.

However in live tv I cannot change between the tuners. I've been trying in vain via the Change Input command ie 'c' (is that the correct command or should I be using another one ?). While watching live tv and while no shows are being recorded, when I press 'c' the onscreen text comes up telling me which tuner/input I'm currently using, however I don't know how to use it to switch to one of the other tuners/inputs. The onscreen display doesn't list any of the other inputs.

What am i doing wrong ?

Thanks,
Declan

### Post by klc5555 on 2011-09-03
I think the "c" command (to change to the next available tuner) became obsolete after Mythtv 0.20.2 or thereabouts. It makes things a bit complicated with multiple backend/frontend systems. The normal way is to use "m" to pull down the onscreen menu, where "switch inputs" should list all the currently available tuners.
