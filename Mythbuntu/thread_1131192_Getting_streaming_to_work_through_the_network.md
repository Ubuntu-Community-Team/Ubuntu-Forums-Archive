### Post by bb_145 on 2009-04-20
Mythweb might be what you are looking for.  If you installed it, it can be reached at the server address /mythweb  (A password needs to be set up first). (e.g. [http://192.168.2.14/mythweb](http://192.168.2.14/mythweb))

More info here:

[http://www.mythtv.org/wiki/MythWeb](http://www.mythtv.org/wiki/MythWeb)

It can be easily installed through the Mythbuntu control panel (including setting up the password)

The other way to access recordings from another computer is Mythtv player

[http://www.sudu.dk/mythtvplayer/index.php](http://www.sudu.dk/mythtvplayer/index.php)

I find this works quite well
