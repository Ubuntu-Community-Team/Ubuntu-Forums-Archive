### Post by jb5 on 2010-10-18
Not sure if this is a solution to your particular symptoms, but I found that when I had the '    *    ' key defined in my licrc file (~/.lirc/mythtv)

it caused all sorts of problems, when trying to use the remote.

If I redefine the key; first in lircd.conf```
         star                        0x0037
```

and then in my licrc file```
begin
    remote = NOVA-T500
    prog = mythtv
    button = star
    config = F6
    repeat = 0
    delay = 0
end
```

Then after a COLD boot, the remote functions properly in mythtv.

Maybe worth a shot?
