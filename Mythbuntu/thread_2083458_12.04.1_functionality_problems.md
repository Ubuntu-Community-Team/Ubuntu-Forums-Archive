### Post by Addanc on 2012-11-24
The start-up problem where the router isn't powered appears to be an Ubuntu issue; viz if the DHCP network configuration dialogue doesn't happen the loop-back address used between back and front-ends cannot be used; presumably the network stack isn't being enabled because of the lack of configuration; **the solution** is to set the network IP address statically.
