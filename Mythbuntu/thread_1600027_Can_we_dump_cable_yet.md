Netflix will not work in linux.  If you really wanted it you could do something with a VM running windows.  Hulu does work in linux, but flash is a cpu hog.

### Post by iitywygms on 2010-10-20
I use boxee.  It has an application called clicker.  Clicker has comedy central as a source.  The shows are about a week later than the actual shows date, but so what.  I do not have to use a "browser" it just streams like it was a tv channel.
The adobe player is a resource hog, my cpu is maxed when it is being used.  You need a pretty good processor if you plan on using it.
I suggest visiting the boxee forums.  Lots of info and stuff there concerning streaming and flash.
As far as the quality goes.  It looks like sd tv to me when viewing it.  It is not high def, but the quality is fine.  Sort of like watching a movie that has been encoded with divx, or the like.
Just try boxee.  If you don't like the quality, uninstall it.
I must give props to the myth developers, without them, none of this would be possible.
