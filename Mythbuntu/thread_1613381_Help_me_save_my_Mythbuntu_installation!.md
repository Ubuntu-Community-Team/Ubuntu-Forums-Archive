### Post by David Grigor on 2010-11-05
Here in the US,you can get a bluetooth dongle for about $6. Then get a off-brand wii remote ( $20 ) to use with it. Setup is very easy and doesn't require any fiddling with lirc.  Only downside is that if you also use multiple frontends the wii remote can connect to the wrong one so best to only use on 1 machine.
