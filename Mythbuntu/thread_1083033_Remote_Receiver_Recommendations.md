### Post by Boppy3125 on 2009-03-03
I started with the same remote.  I had a PVR 150 in the computer and its remote, so I just set up the Harmony as the PVR150 and everything was cool on Mythtv.  When I moved the computer and my other HT equipment into a cabinet, I got the Hot Link Pro at SmartHome for about a hund and it worked flawlessly.  The cool thing about the Hot Link Pro is that the sensor cable plugs in with RCA, so a Male-Female RCA extender would allow adding some additional distance if needed.  (I think I have that right, if it is important I can check at home.)
