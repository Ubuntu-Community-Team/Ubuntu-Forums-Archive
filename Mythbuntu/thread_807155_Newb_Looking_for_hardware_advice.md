### Post by tgm4883 on 2008-05-25
This is what you need to know
[http://www.mythbuntu.org/requirements](http://www.mythbuntu.org/requirements) 

If you want a well supported tuner, the PVR-150 (single tuner) or PVR-500 (dual tuner) are well supported with hardware encoders.

If you are looking for HD tuners then you will want something like the pcHDTV 5500 (internal single HD tuner) or the HDHomerun (external via ethernet dual HD tuner).

If you are planning on using a cable box and have an HD cable box and live in the USA, you can use firewire to stream.  You will get unencrypted channels.

If you want to get all channels you subscribe to, you can get the new Hauppauge HD PVR.  It should be shipping soon and is reportedly to have linux support pretty soon.
