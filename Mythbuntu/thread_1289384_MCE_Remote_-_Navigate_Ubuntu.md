### Post by KillerKiwi on 2009-10-12
A while ago I create a small python app that maps remote keys to fake x key pushs

[http://code.google.com/p/jkeys/]("http://code.google.com/p/jkeys/")

I mostly use the code to allow my 4 years old to play snake with a joypad ;)

there is also irexec if you wont to map a remote key at any time to a command
[http://manpages.ubuntu.com/manpages/karmic/man1/irexec.1.html]("http://manpages.ubuntu.com/manpages/karmic/man1/irexec.1.html")
I use irexec to make the power button on the remote restart mythfrontend
