### Post by mrand on 2008-11-04
[http://packages.ubuntu.com/intrepid/ubuntu-desktop](http://packages.ubuntu.com/intrepid/ubuntu-desktop)

shows that ubuntu-desktop does not require/depend-on items like open office or pidgin.  This means that it is up to system configuration - or more exactly, which tool (apt-get vs. aptitude vs. synaptic) is used and how it is configured.  Some will automatically install recommended items, while others will not.  I'm kinda glad they don't - some recommended lists for meta packages can be quite large.
