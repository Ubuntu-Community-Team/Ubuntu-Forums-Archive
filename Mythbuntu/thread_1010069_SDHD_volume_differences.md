I was able to set the volumes for my SD recording profiles to match my HD volumes (which I'm not able to change).


Utilities/Setup >> Setup >> TV Settings >> Recording Profiles >> MPEG-2 Encoders >> <choose profile> >> <fourth page - Volume %>
