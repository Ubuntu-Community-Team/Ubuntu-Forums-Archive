### Post by jwrudolf on 2008-06-22
Thanks for the help.  I did some more digging and you are right there are like a million versions of that remote.  I discovered the one that comes with the PVR-150 MCE kit works with the Windows Media Center Remote (New version) option in mythbuntu (using the lirc_mceusb2).  I had tried it before and couldn't get it going but I tried it again last night and now it works.  Go figure.  Anyway , anyone with the pvr-150 mce kit use the Windows Media Center Remote option.
