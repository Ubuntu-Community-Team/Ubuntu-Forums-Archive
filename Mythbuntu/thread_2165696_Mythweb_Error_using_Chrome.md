I am trying to use Chrome on my WinXP to access MythWeb. I am trying to view a movie. I see the list of movies and then click on it to play it and I get the following error message:

No webpage was found for the web address: [http://10.10.10.10/mythweb/video/stream?Id=1](http://10.10.10.10/mythweb/video/stream?Id=1)

As cited above, I have two Chrome media player plugins enabled.


I can also add that when I use Opera (v12.16) as my webbrowser on the same PC, Mythweb works. It serves up my *.m4v movie, and it streams it perfectly.

For those of you that use Opera, I had to modify Opera's preferences (Main Menu|Settings|Preferences|Upper TAB - Advanced|Left Column - Downloads) 
I Added MIME TYPE video/x-m4v to use VLC plugin.
