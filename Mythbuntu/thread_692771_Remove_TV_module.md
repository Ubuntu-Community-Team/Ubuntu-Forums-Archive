Definitely. I actually had this same question a couple of days ago (I'm canceling my cable TV, so don't want to waste space/time on "Watch TV" and "View Recordings")

I haven't done any big menu changes yet, and I'm still in the "messing around" phase, but MythTV's menu system is pretty simple/customizable. Here are two links that helped me learn a lot:

[http://www.myhdbox.com/mythtips/2006/04/tip-2-how-to-customize-mythtvs-menus.php](http://www.myhdbox.com/mythtips/2006/04/tip-2-how-to-customize-mythtvs-menus.php)
[http://www.mythtv.org/wiki/index.php/Menu_theme_development_guide](http://www.mythtv.org/wiki/index.php/Menu_theme_development_guide)

(the second one is a very good reference guide)

And don't forget: always make a backup of the files before you edit them, in case you screw them up!

P.S. No need to switch to Mythdora. While I haven't tried it (and can't compare it to Mythbuntu), the menu customizing is entirely up to MythTV, which is independent of distro.
