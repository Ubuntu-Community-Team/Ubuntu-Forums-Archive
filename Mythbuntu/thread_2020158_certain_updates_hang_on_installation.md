### Post by jim09 on 2012-07-07
Certain updates hang on installation like samba and some made from the MCC.  It sees that if you are installing them from the gui a dialog box doesnt appear that requires a response.  If you install the updates from a terminal with apt-get they proceed and you can enter a response and the installation continues.  Using apt-get is a work around if doing a clean install but you might not know
in cases what package to install.
