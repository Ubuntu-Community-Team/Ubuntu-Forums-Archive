### Post by uncle hammy on 2010-01-14
Just beware, you will want to read up on smb.conf because the default settings will likely cause you some permissions issues with files created then accessed on your Windows machines.

I found that adding these lines to my shares section seemed to produce the best all round results....

force user = <MYUSER> <--- enter your own user here
force group = mythtv
inherit permissions = yes
directory mask = 07775

note, I use "MYUSER" on ALL my machines, both Linux & Windows.  I also set the group to mythtv, because I use these shares for my music, videos, which I set up within mythtv.

Also, you may want to read through this to enable name resolution support within your whole network (so you can "ping "LinuxMachine", rather than by IP)....

[http://ubuntuforums.org/showthread.php?t=405662](http://ubuntuforums.org/showthread.php?t=405662)
