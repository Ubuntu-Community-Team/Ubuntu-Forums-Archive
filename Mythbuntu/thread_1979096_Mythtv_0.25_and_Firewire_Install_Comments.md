[LEFT]Did not see these Googling so a few Mythbuntu 12.04 install/setup comments.


1) New Ivy Bridge 3770k build with Asus board P8Z77-V PRO and ATI 6670.  Using Firewire
(added pci based FW card) works out of the box with my Motorola DCT 6412,
should work with other similar STB's.  No need for plugreport, mpeg testing, or
6200ch channel changer as mentioned at [http://www.mythtv.org/wiki/FireWire](http://www.mythtv.org/wiki/FireWire) and
elsewhere.

<REMOVED>

4) If you change your "Storage Directories" make sure you have read and write permissions on the new directories, otherwise live tv etc. won't start.
