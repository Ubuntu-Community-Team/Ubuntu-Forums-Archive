Yes, 10.04 can use any NVIDIA version.

If you want to stick with 9.10, enable the autobuilds repository.  It has the same support that 10.04 has to use an NVIDIA version independent of the myth build.

[http://www.mythbuntu.org/auto-builds](http://www.mythbuntu.org/auto-builds)
