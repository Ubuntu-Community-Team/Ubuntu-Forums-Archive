### Post by PhilWig on 2011-05-04
Peter,

1.  Sorry if I misunderstand but you seem to be suggesting that if you adopt ME TV you will need a DVB receiver but not if you use MythTV.  You will need one if you want to pick up DVB with any recorder application.  Once the bug has bitten though you'll want more than one, even with multi-rec facility of Myth.

2.  There are many Myth compatible remotes available and some DVB cards come with one, but you'll find it easier to start without one and simply use the keyboard.  A wireless keyboard works great and you might find it a good option.
