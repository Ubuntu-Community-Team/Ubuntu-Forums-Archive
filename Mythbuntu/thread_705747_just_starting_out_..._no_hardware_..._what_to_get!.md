### Post by stevanbt on 2008-02-24
Hi,
I don't know what to recommend really, the mythtv site does have some recommended hardware specs - [http://www.mythtv.org/docs/mythtv-HOWTO-3.html#ss3.1]("http://www.mythtv.org/docs/mythtv-HOWTO-3.html#ss3.1")

Our combined frontend/backend box has the following spec:-
Antec Fusion Case
AMD X2 4200+
2Gig Memory
hauppauge Nova-t 500
2 x 500 gig drives (I need to install LVM really)
Gigabit ethernet installed in our house
Mythbuntu 7.10

The case fans are pretty quiet and it seems to run cool (I installed the stock cpu cooler which isn't all that quiet).

It took about three nights to get everything installed and running, then another few nights to fiddle with a couple of things - one or two left to do, but they're trivial so there's no rush.

I also run a stock Ubuntu 7.10 machine with Mythtv frontend (this is my main pc so it's reasonable spec AMD X2 5000+, 2Gig memory etc).

Hope this helps, Steve.
