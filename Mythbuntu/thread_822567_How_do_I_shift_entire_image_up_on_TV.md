### Post by ab1301 on 2008-06-09
There are a few different ways.  First, make sure the resolution is set properly.  I can't help with this, as setting up the resolution will be different according to what your hardware is.  If you think this is the problem, look up how to set the resolution properly for your hardware.

Next, you can adjust the appearance in Mythtv by going into settings > setup.  From there, there are two options to adjust your display.  One uses little arrows to crop and displace the Mythtv screen.  I forget what the name of this menu option is, but its a descriptive name, so you should be able to find it.  The other option is to go into "appearance" on the setup menu.  From there, you can enter numbers to displace and crop the size of the Mythtv screen.  To me, this option seems a little more reliable and predictable than using the arrows.  Either way, this method will allow you to resize the Mythtv screen, and move it into the proper location.  When viewing tv, myth will use the same screen size for the tv.

Finally, if you just want to adjust the picture while watching tv, you can use go into the menu while watching, select "manual zoom mode" (or something like that, I can't remember exactly), and use the up and down arrows to move the picture up and down.
