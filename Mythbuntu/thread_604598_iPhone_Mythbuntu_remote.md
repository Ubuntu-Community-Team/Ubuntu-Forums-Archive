### Post by Almighty on 2007-11-06
I don't know how many Mytbuntu users own an iPhone, but I found a very interesting app that allows you to control your Myth box using your iPhone via a web interface. The app requires no additional setup than what it originally calls for in the Mythdora instructions.

Set up files and instructions are found [here](http://kylestoneman.com/out/remote.tar.gz) and here is the [ blog](http://www.legatissimo.info/node/355) post.


EDIT :I should also add, this should work with any web enabled phone. The formatting is just for the iPhone.
