### Post by thatguruguy on 2010-11-30
Gotcha.  I had that issue when I first updated to .24 from .23.1.

Go into synaptic and run the updates from there.  It will show you which programs are creating inconsistencies which are preventing the update.  In my case, I had to remove a couple things that were stuck on the .23.1 version and install the .24 versions.  Since then, I've had no problems with the daily builds.
