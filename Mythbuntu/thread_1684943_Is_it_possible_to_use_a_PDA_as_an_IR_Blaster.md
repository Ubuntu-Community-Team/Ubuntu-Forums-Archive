I know its common to use an IR Blaster to control sat receivers (in my case Dish Network) but is it possible to plug a PDA into a Myth backend and have it control the sat?

I have a Dell x50v with an IR transmitter built into it that has been collecting dust for awhile now. It would be nice to put it to good use instead of buying an IR Blaster. I used it as a remote in the past and it worked ok but I'd like Myth to command it to issue commands to my sat receiver. Is this possible?

The x50v has both wifi and bluetooth so I'm not limited to just usb.
