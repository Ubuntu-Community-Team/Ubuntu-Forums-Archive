### Post by ian dobson on 2010-09-24
Hi,

I think the key combinations used are defined in the keybindings. I think you can chage these bindings through the mythfrontend.

Maybe have a look here:- [http://www.mythtv.org/docs/mythtv-HOWTO-11.html#ss11.1](http://www.mythtv.org/docs/mythtv-HOWTO-11.html#ss11.1)

Edit: If you have mythweb installed you can edit the key bindings through the setup menu.
