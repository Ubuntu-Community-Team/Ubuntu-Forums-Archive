### Post by SilkBC on 2008-05-27
Greetings!

I have just installed Mythbuntu 8.04 on some new hardware that has an NVidia graphics card.  I installed it while connected to my 32" LCD TV.

When it boots, I am able to see the console display and Mythbuntu splash screen no problem, but when it gets to launching X, I just get a blank screen and a message from the TV that says "Signal is not compatible with the input".

I am suspecting that I need to add the TV resolutions an/or refresh rates as indicated by the user manual, but when I open up '/etc/X11/xorg.conf', all I see for the "monit", "screen", etc. sections are "Configured Monitor", "Configured mouse", etc.

I supposed I could manually edit this file but I would rather use the system tools to do it.  Running 'dpkg-reconfigure xserver-xorg' only asks questions about the keyboard, which I do not care about.

If I try to run 'Xorg -configure', it tells me that there is already a server active for display 0.

My questions:

  1.) How would I boot and not have X run?
  2.) What might be the best way to input the supported resolutions for my TV?

TIA.

-SilkBC

Looks like it isn't sending the proper video signal. If you are in North America (along with some other countries) you are probably looking for NTSC-M. Check the man pages for your driver (which I assume is the proprietary nVidia driver) for an option similar to TVFormat or TVSignal.

Unfortunately I have an ATI card so I can't help you much there--but I think the idea is similar :)

Good luck!
