### Post by SteveGodfrey on 2008-12-03
Why.

I leave the media centre logged in so people at home can watch tv etc. However I also use the box a lot remotely and use NX for a remote connection. Using NX stops the remote from launching MythTV.

Solution.

If I log into Myth locally using a different username when I NX with my account it won't mess up the local login. I thought using the MythTV account would be a good solution! What I've done is created an account called TV and I'll leave that logged in locally.
