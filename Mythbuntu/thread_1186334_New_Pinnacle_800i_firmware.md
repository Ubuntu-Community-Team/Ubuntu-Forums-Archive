Guys FYI, the mythtv page was just updated.  There is a new firmware that is required if you use v4l-dvb tree in order to get the pinnacle 800i and firmware dvb-fe-xc5000 to work.  You have to load the new dvb-fe-xc5000-1.6.114.fw

This is confirmed now working in Ubuntu Jaunty 9.04 with mythbuntu and mythtv.
