### Post by laffinet on 2009-09-01
How did you cancel the recordings ?

I'm assuming you scheduled a recurring recording.

Go to upcoming recordings, highlight the one you want to cancel, hit enter. Select "edit options".
The top line should show you how this "show" is recorded (once a week, every day etc.). Set this to "do not record this programme", then scroll down to "save these settings" and you should be all done.
