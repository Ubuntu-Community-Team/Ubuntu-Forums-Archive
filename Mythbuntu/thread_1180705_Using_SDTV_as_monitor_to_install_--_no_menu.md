### Post by silverdulcet on 2009-06-07
Its very similar to a live desktop of the Ubuntu Desktop cd. When you let it boot into the ubuntu user it should load the desktop, where among other things you can use the Applications>Settings menu to adjust the resolution as well as test the functionality of your hardware prior to installing. When you are ready, there is a "Install" icon that will allow you to do a graphical install from the desktop. Pay close attention to the partitioning menu when it offers to automatically partition, or let you choose which partitions to use.
