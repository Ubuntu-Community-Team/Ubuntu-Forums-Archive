### Post by blackoper on 2009-06-02
it streams what is in the mythvideo's directory. you'd need to go into mythfrontend's setup section and get to the mythvideo section. from there you specify the directory. I had to reboot my system for the new settings to take effect (upnp server runs while mythfrontend is active)
