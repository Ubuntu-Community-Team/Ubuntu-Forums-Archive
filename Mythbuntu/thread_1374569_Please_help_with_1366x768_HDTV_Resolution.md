Something odd going on here.  I have a screen with the same native res as yours. Despite this it reports 1280x720 (IIRC)to the video card via the HDMI connection, and something different (can't remember) via the VGA.  Whichever I use, HDMI or VGA, the nvidia card/driver selects the required resolution, and the screen is ALWAYS full.  I believe this is because the TV will upscale whatever it gets to it's own native resolution internally.

So, the screen is "full" but if the data you are displaying doesn't fill it, then mythfrontend setup has values you can tweak to adjust it. However, as the video playback settings default to 100% I don't know why you aren't getting a full screen.  Maybe you can set it to greater than 100%.  The aspect ratio problems can be fixed here too, as x and y are independently adjustable.

One thing, I remember that when using the VGA, and cycling through the zoom modes with the 'w' key, it was 4:3 that produced the correct aspect ratio on the 16:9 screen.  Weird but true. This setting can be made the default zoom level in the settings too.
