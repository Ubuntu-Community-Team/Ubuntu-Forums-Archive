### Post by DutchLoki on 2008-06-25
Is it the satellite setup itself or the nova-s tunercard driver installation you need?

The setup for an satellite card (DVB-S card) is for the most part the same so you can pick any tutorial about an dvb-s setup. Possibly you use a motor or switch to watch more than one satellite so you might need some extra info about the configuration of diseqc to control these. 

In the past i've used tutorials for my satellite setup and found several using the technotrend cards(the nova-s is a rebranded technotrend card), althoug this does nog matter. I've used german and belgian guides from which i can find the urls if you can read these languages.
