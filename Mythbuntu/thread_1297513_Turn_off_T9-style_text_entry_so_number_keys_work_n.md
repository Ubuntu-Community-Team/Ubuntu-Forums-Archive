### Post by mgrusin on 2009-10-21
Hey, I have a really dumb problem I hope someone can help with.

I'm using a wired keyboard to control MythTV.  It works fine, except that for any dialog in mythfrontend, the number keys don't just work.  They show a quick T9-style inline menu of other characters that each number could bring up (e.g. if I press '2', I get "[abc2]" and each time I hit '2' it moves to the next choice.)

This would be nice if I was using a remote, but I'm not. I've looked through the setup dialogs and I can't find the menu option to turn this off.  Anyone know where it is?

Thanks VERY much! -Mike.

### Post by mgrusin on 2009-10-22
Sure enough, you're right! Thanks for the information. I'm very surprised that the devels didn't put in an option to turn this on and off, since the rest of Myth is so uber-customizable, but it does appear to be the case.

I found a semi-workaround: turn on "Use line edit virtual keyboards" in setup/appearance.  Then when you're in a text box, you can turn on the keyboard with [enter], and type normally.  But it's a bit of a PITA since [enter] doesn't then actually enter the information - you need to tab to the next button.

While digging, I found that this won't be an issue in the next version of the UI, which is something to look forward to:

[http://www.gossamer-threads.com/lists/mythtv/users/380726]("http://www.gossamer-threads.com/lists/mythtv/users/380726")
