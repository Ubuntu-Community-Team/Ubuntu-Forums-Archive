### Post by theophile on 2009-03-26
I believe that moving forward, Xorg no longer needs the xorg.conf file but if one is present, it will take the settings in there. I think that without the xorg.conf, a lot of "autoconfiguring" takes place.

### Post by superm1 on 2009-03-26
If you however want an xorg.conf you can run:

```
sudo dpkg-reconfigure xserver-xorg
```
