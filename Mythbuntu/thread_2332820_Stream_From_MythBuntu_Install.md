### Post by ajgreeny on 2016-08-04
I have never used mythbuntu so have no idea whether it has a dlna server as a default installation, but have a look to see if it has **minidlna** installed and if so you will certainly have a server to stream multimedia already available.

If it is not already there install **minidlna** and configure it according to the info at [https://help.ubuntu.com/community/MiniDLNA](https://help.ubuntu.com/community/MiniDLNA).

I find it works very well from my Xubuntu 14.04 streaming to my LG Smart TV, and I use it more now than plexmediaserver, which I also have to stream to other non-dlna enabled TVs using a set-top box with rarflix side loaded as plex client.

I can not help you with streaming to the web; I prefer to keep all that sort of thing local.
