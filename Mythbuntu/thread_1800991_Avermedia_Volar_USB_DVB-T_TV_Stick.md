### Post by itlarson on 2011-07-10
I used one of these on the mythtv system I built for my parents.  It seems to pull in a pretty good signal with over-the-air digital television.  Originally I used a Hauppaug pcie tuner, but it didn't seem to pull in a good signal, so I returned it.  I've heard that much of the electronics available today is built assuming it will be used with cable TV, and doesn't work well with OTA signals, which can be weaker.  The Avermedia tuner has DVB-T right in it's name though, so it should be optimised for broadcast television.
