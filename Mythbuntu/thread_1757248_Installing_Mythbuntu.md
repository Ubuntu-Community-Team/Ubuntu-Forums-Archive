### Post by Dan_R on 2011-05-13
As a novice, and if you're using this computer primarily as a Mythbox, do the install now.

If you want to share files over your network with a Windows based computer, go with the Samba install. Install VNC if you want to be able to view the actual desktop from another PC remotely, controlling the mouse etc. If you will ever have another computer used as a mythtv frontend to play movies and music etc from your backend you are setting up, install the MythTV service.
