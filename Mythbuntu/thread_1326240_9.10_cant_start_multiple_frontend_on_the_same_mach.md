### Post by kja999 on 2009-11-14
In myth 0.21 when importing music, I would start a new frontend on the same box so I could watch TV or whatever. Since 0.22 the frontend now recognises there is already a frontend open and just switched to it instead of giving a new session.

Has anyone else noticed you can't start multiple frontends on the same machine ?

Is there a way to stop this behaviour ? ...  or more fundamentally, how can I get out of the music import screen while its running ?

### Post by kja999 on 2009-11-15
Cheers Ian, this makes it a super simple workaround.


For anyone who is interested:
save the existing launch script - 
sudo cp /usr/bin/mythfrontend /usr/bin/mythfrontend.orig

then alter the file -
sudo vi /usr/bin/mythfrontend

commenting out the first line by adding a # at the start, so it reads like this - 
#pidof mythfrontend.real 2>&1 >/dev/null && wmctrl -a "MythTV Frontend" 2>/dev/null && exit 0
