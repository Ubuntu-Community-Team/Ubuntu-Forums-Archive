### Post by nickrout on 2009-11-07

> Yes the upnp server.  What do you mean it should be there already?  It doesn't know where my music is, since I have never had myth before.

The directory you have your music in for mythmusic will be shared via upnp. By default in mythbuntu it is /var/lib/mythtv/music.

If you want to change the directory navigate in the frontend to 

Utilities/Setup|Setup|Media Settings|Music Settings|General Settings

and change it there
