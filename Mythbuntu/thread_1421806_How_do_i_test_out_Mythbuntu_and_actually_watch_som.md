### Post by larsolav on 2010-03-04
I believe the live CD only has the front-end option on it. I don't think it is possible to use a live CD bootup to set up the TV tuner cards as this is done in the back-end setup. In mythtv there really is no difference between watching live TV with recording (Live TV is basically almost simultaneously playback of a recording).
