### Post by aseriesoftubes on 2008-04-15
I don't have this particular card, but I do have some similar cards (PVR-150 and 500, both of which have s-video input). It should be somewhat easy to set up two separate input connections when configuring the backend. If I were you, I'd read the Mythbuntu manual [here]("http://www.mythbuntu.org/documentation/mythbuntu_installation.pdf") (5.4MB PDF). The information you're interested in is around page 86, but I'd read the entire configuration section and make sure you're up for it before installing. :)

---

### Post by aseriesoftubes on 2008-04-15
I should also mention that once the input connections are properly set up, you can switch between them in the frontend with a single key press.
