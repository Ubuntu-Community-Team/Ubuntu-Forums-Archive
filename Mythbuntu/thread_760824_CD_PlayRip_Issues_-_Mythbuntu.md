### Post by wyliecoyoteuk on 2008-09-22
Hi, this may be a little late for you, but I had the same problem.
If you have it set to play the CD automatically, you MUST wait for the player to start, and then exit it before you can rip the CD.
If you don't, and it crashes, it will not rip until you eject the CD, and then wait for it to start etc.

In setup, you can disable CD autoplay. but I think you still need to wait for the CD to spin up before selecting import.
