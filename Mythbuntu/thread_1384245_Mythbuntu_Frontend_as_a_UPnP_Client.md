### Post by ahhughes on 2010-01-20
The answer is NO!

There is a workaround - djmount can mount a UPnP AV (DNLA) server as part of the filesystem (which can be added to the frontend library). So it *could* work - but there is some serious usablity issues here when you consider how easy this is done on the ps3, astone, asus on or Western Digital Live.....
