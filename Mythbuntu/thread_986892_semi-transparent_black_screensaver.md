### Post by florin on 2008-11-19
I want to setup a screensaver that would make the screen black, but with something like 15...25% transparency. The effect would be like the blank screensaver, but semi-transparent.

The reason for this is obvious. I want to emulate the behavior of many media players which after a period of inactivity reduce the contrast and/or the brightness of the image but do not blank it altogether. So the user sees that the system is still up, but the screen is not toasted by a high-contrast and/or high-brightness fixed image. E.g. the PS3 does that.

I think it's a very good idea from a usability point of view.
The standard blank screensaver leaves doubt whether the system is still up or not (from a usability perspective there should be no doubt, even at first sight).
No screensaver is bad for the TV.
Other screensavers (non-blank non-fixed image) do not feel appropriate for a media center machine.

The problem is, how to do it? How to make the screen black and assign some transparency to the screensaver?
