### Post by Toby.Mills on 2008-04-09
I have a very unique opportunity next week in that my partner is going to be away on business for the whole week.
This generally only happens once a year so its my only real opportunity for serious 'scheduled maintenance' on my Myth and Asterisk systems, including software / hardware upgrades and hardware cleaning etc.

I'm currently running an old version of knoppmyth on a dedicated backend and mythbuntu 7.10 on several frontends.
The system has to be bulletproof, one missed program and my life isn't worth living :)

Its been very stable for the last year with only one problem when I ran out of disk space on the backend.

Anyway, I want to pick brains on my best approach.

Should I install 7.10 final on the server as well and wait for the final version of 8.04.
Or should I go for 8.04 beta and then upgrade to final when it comes out in a few weeks.

My big concern is that if I go with 7.10, then doing an upgrade while my partner is here is quite a big risk to take if anything goes wrong or breaks.
If I install 8.04 beta then I have all week to get it going, but if anything breaks in between when she gets back and when the final version comes out, then that could be just as bad.

Are there any issues with the upgrade path from 7.10 to 8.04 final (I want to do it all while the system is running so no cd reinstalls etc).
Is it possible to upgrade 8.04beta to the final version without having to reinstall from CD.

Ideally I want to make all my frontends diskless as well, so my preference is probably to go to 8.04 beta and then make the short jump to 8.04 final as soon as it comes out.

Any comments or suggestions?

### Post by Mrbbad on 2008-04-10
My thoughts on this would be that if you are fine with 7.10 (ish) right now, everything is working well and all.  Just leave it.

My personal experience with 8.10 is that it's rather unstable, mostly just the backend (and ESPECIALLY when it comes to watching/using TV), but I've had random database corruptions, etc. too.

So unless you are willing to take the pain that might come with the beta, just wait until it's more stable.

Just my 2 cents from my own experience.  I'm up to 3 reinstalls.  All hair-pullers.  My friend who uses MythBuntu is also up to about 3 reinstalls as well.
