### Post by managementboy on 2009-04-03
hi, mythtv has no application to make skins, you have to edit XML files and generate PNGs to create a new "theme". 

check [http://www.mythtv.org/wiki/Theme_development_guide](http://www.mythtv.org/wiki/Theme_development_guide) for a developer guide to the new 0.22 version.
