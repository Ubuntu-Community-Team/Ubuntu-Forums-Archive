### Post by foxbuntu on 2008-04-01
Freddie,

Edit /etc/lirc/hardware.conf

in the "Modules" Section of the IR Receiver

Change the loading modules to:

```
lirc_dev lirc_i2c
```

Then save it exit...and do the following:

```
sudo modprobe lirc_i2c
```

Then:

```
dmesg | grep lirc
```

You Should get something like:

```
lirc_dev: IR Remote Control driver registered, at major 61 
lirc_i2c: chip found @ 0x71 (Hauppauge IR (PVR150)) 
lirc_dev: lirc_register_plugin: sample_rate: 10
```

If this works then do this:

```
sudo /etc/init.d/lirc restart && sudo udevcontrol reload_rules
```

now check for:

```
/dev/lirc0
```

If this is still not working post any errors and also include your hardware.conf as well.
