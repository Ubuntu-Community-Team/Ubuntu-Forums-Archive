### Post by Geoffke on 2011-12-14
If you the owner of an iPhone, another type of  smart-phone or even an  iPad, it could come in handy to have them to act as a remote control for  your MythTV installation based on Ubuntu (this works on MythBuntu  too!).

[IMG]http://www.geoffke.be/images/nieuws/mythtv_remote_1.png[/IMG]

The original code was written by Kyle Stoneman in 2007, but since the code was a bit outdated I have rewritten the entire code to meet the current and future standards.

You can download the remote control here: [http://www.geoffke.be/nieuws/17/](http://www.geoffke.be/nieuws/17/)
Instructions on how to install everything are added. If you have any remarks, don't hesitate to post them! 

I hope this is usefull to other MythTV users and I hope others will use this code to build upon...
