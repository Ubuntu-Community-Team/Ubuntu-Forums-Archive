### Post by andrewc6l on 2008-03-20
Basically, the backend is the recorder and the frontend is the playback. You can have multiple frontends talking to one backend, but you don't need to - you can also install both the frontend and backend on one box.

   Andrew

### Post by volkswagner on 2008-03-20
Sure You need at least one backend.  This backend can be on the same machine as a frontend.

The backend is the software/hardware that houses the database and tuners.  The frontend is the interface for scheduling and viewing media.
