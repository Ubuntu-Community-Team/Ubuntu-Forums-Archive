### Post by ian dobson on 2009-07-18
Hi,

If you have a DVB-c,s,t card you can enable "multi-rec" for the card in mythtv-setup. This will allow you to record several channels from the same multiplex.

### Post by ian dobson on 2009-07-19
Hi,

As I've already posted you need to use DVB cards to use the multirec function. These are tuners for digital TV and not analog. If you want to record several analog channels at the same time you need several tuners. Maybe look if you can get another PVR150 or a PVR500 with is a dual PVR150.

Regards
Ian Dobson

### Post by newlinux on 2009-07-20
Generally. if you have *n* analog tuners, you can record *n* analog programs at once. So if you hve 5 analog tuners you can record 5 analog programs at once. At some number of tuners your disk speed may be the problem if they are all recording to the same disk.
