### Post by OffAxis on 2007-10-23
> The password problem has been fixed
alright.

> I installed mythtv from the repos on a fresh gusty 32bit.
If you're doing a fresh install the best thing would be to use a mythbuntu .iso proper. [http://mythbuntu.org]("http://mythbuntu.org")
instead of messing around with the package installs. The install wizard for mythbuntu couldn't be easier.

> If I purge mysql and mythtv and reinstall will that work or will there be a config file left behind that make new install have the same problems?
The mysql server stores its users in a mysql table. This table is written or destroyed during an install or removal.
So, if you're just doing a default mysql installation, then no; no config file is left.
As for mythtv, you know where the mysql password for it is located (/etc/mythtv/mysql.txt) and that's just paired with whatever account you manually create in mysql for mythtv after the reinstall.
But I would recommend the mythbuntu iso. It totally streamlines and simplifies the installation.
