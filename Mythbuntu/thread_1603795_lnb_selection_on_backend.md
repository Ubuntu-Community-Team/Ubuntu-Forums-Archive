### Post by simonnev on 2010-10-29
Not one for giving up, seems like I found the answer myself..
when trying to set up the lnb and saving using the keyboard and merely pressing the finish button it would not save the selection every time. (very hit and miss approach) i.e. show unconnected again on the card input.
However by pressing alt+f after selecting the input (lnb in my case)it then shows lnb page again, then press escape then alt+f it saves every-time.
