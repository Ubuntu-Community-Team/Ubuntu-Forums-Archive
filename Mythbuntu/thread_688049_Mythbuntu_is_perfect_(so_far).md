### Post by Caps18 on 2008-02-04
I was very impressed with the Mythbuntu setup and install.  It was the smoothest linux install I have ever had (out of 10 or 15 versions over the last 10 years).

It even found, downloaded and installed the nvidia driver, without me having to compile anything.  My M$ remote even worked as advertised (I may need to change/add some buttons.).  

I have this as my HDTV tuner on my 120" projector system and it works great so far.  I've only played with it a little, but it is working really good so far.

The only question I have so far is if there is some way to turn off the fading channel/show descriptions.  During the fade away, my system slows down.  But that is probably more of a MythTV question.

Thanks for the hard work in creating this software.  I like it a lot.  

(I might have to look into programming a new theme.  I had an idea a while ago for a simplistic/modern looking one.)
