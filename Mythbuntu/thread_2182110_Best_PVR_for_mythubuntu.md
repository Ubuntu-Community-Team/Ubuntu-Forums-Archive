### Post by Lizel on 2013-10-20
I believe you are asking about what video capture device to use?

If you want HD in mpeg2, you can't get there from here.

You can capture SD mpeg2 with a hauppauge 150 or similar pci cards (or a usb device) from the composite out of a set top box.  If it is an HD box, these cards will capture HD channels but it is going to be in SD letterbox - black bars at top and bottom.

For real HD you need something like the Hauppauge Colossus PCIe 1x card.  It will capture the component output of the set top box as mpeg4.  

Watch out for the latest single tuner Bell HD boxes (6141?).  They do not have component output.  Not sure about composite.

If you want component you have to pick up a used 6131 HD box on ebay or Kijiji.  It has both component and composite.
