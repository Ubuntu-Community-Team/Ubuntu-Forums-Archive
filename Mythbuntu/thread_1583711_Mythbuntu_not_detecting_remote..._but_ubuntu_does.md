### Post by LowSky on 2010-10-06
have you tirend reinstalling LIRC from synaptic and picking the correct remote from the list?

You should have a few files for lirc. You should have one file in you home folder that point to every applicaiton that will access lirc

its named .lircrc

it will have a list like so

```

#Custome lircrc generated via mythbuntu-lirc-generator
#All application specific lircrc files are within ~/.lirc
include ~/.lirc/mythtv
include ~/.lirc/mplayer
include ~/.lirc/xine
include ~/.lirc/mplayer
include ~/.lirc/vlc
include ~/.lirc/xmame
include ~/.lirc/xmess
include ~/.lirc/totem
include ~/.lirc/elisa
```



then the file  ~/.lirc/mythtv

```


# LIRCRC Auto Generated by Mythbuntu Lirc Generator
# Author(s): Mario Linociello, Nick Fox, John Baab
# Created for use with Mythbuntu
begin
    remote = mceusb_haupauge
    prog = mythtv
    button = Stop
    config = Escape
    repeat = 0
    delay = 0
end


```
