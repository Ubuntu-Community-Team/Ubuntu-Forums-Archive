### Post by theSeekerr on 2010-01-19
Thanks to the migration to the new MythUI library, basically all the old themes are gone.

The default theme, Terra, is a bit odd - you might be more comfortable with the Mythbuntu theme, it features traditional 2-pane vertical lists.

Yes, the views are part of the theme. The only "complete" themes I'm aware of for 0.22 are Terra, Graphite, MythBuntu and blue-abstract. The last two are the most like the themes in 0.21. There's a theme contest which ends in a few weeks time which will hopefully produce some more themes to choose from.
