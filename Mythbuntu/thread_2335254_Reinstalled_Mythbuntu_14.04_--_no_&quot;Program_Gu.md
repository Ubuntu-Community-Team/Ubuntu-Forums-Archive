I'd applied the various updates that were offered, but they did not cure the problem. Then a day or two ago I was offered the update to 16.04, but that was a disaster, so I saved my recordings to an external drive and ran mythconverg_backup.pl and copied the resulting files to the external drive as well. Then I tried installing 14.04.5 from scratch but the installation failed -- something involving xfsettingsd -- so I installed 14.04.2. I can watch live TV, but there is no longer a Program Guide (or whatever it was called) on the menu, and when I press "I" while viewing I get "Unknown." BTW, when 14.04 *was* working -- although flakily -- I often found that the program title displayed in the recording list was correct but the description belonged to another program -- not one that I had recorded at all.
