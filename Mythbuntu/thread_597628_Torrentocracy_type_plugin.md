### Post by ickyb0d on 2007-10-31
Although I never used torrentocracy, I currently use [Torrentflux](http://en.wikipedia.org/wiki/Torrentflux) as a web based torrenting system in conjunction with MythTV.  So far it's worked out pretty well - aside from having to go to the video manager to load in any new videos into the database.

If you're interested in using this, it should be in the default repos.  "sudo apt-get install torrentflux" should work.
