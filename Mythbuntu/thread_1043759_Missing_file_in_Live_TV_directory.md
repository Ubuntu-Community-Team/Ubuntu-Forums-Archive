### Post by slack42 on 2009-01-19
I got it to work again. I went back into backend setup and then rescanned for channels and noticed that I wasn't getting a lock on any channels. I don't know why but I think the driver for the tuner card got messed up some how. I didn't know how to fix that so I just turned off the machine and left it until today because I had no idea how to deal with it.

Today when I started it up again it just started working again. So I'm guessing that turning the box on and off again did it. I had tried a reboot before but it was still not working. I don't know why but the shut down and clean start up must have got the driver for the card working again.

Don't know if this will help anyone but I thought it best to post how it started working again just in case it helps someone.
