### Post by tmcgary on 2010-10-10
zepfan,

this link [http://www.greenhughes.com/content/installing-boxee-mythbuntu](http://www.greenhughes.com/content/installing-boxee-mythbuntu) outlines a way to add buttons to launch various programs from the mythtv front end 'home page'. Additionally if you search in the mythbuntu forum on how to integrate hulu desktop and mythbuntu a user outlined a method similar to this link. I have (and I'm sure many others have as well) used this method of adding program executes to the front end and they work pretty well.

Your idea is similar, but i get the sense that you would want to have this be this to be the 'home page' once the HTPC was booted. To that end I think you would be talking about making a new skin/theme for mythtv/myhtbuntu with these buttons as the default options on this new themes opening page. I have no experience with themes. 

Best of luck!
