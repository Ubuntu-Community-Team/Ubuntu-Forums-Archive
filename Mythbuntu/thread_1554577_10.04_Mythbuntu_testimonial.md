### Post by Calmor on 2010-08-16
Figured I'd drop by and give my testimonial about my Mythbuntu system.

I've had this system running for about two years now.  I started with 8.04 and migrated to 8.10.  I stayed there, running JY Avenard's patches, until this past weekend when I upgraded to 10.04.

As another change, I've decided to start using my Myth machine as a main desktop machine as well.  It's by far the fastest machine in the house, and it doesn't make sense to use an 8 year old PC when this one sits largely unusued when I need a PC.

I backed up the home folder, created a new home partition to make future upgrades easier, and installed 10.04.  Hardly a hitch in the install process.

I had previously used firewire via a set-top box that worked occasionally, but have since gotten rid of cable.  I now have a Hauppauge HVR-1600 and an antenna.  I live close enough to see the transmitting antenna, and the picture is stable.  HVR-1600 support (for the ATSC tuner at least) worked out of the box.

For the best part, I have a dual-headed nvidia card - no idea what model anymore.  But, I set the TV up on the DVI output as X screen 0 and an LCD on the VGA output as X screen 1.  Using the restricted drivers, I set them up as separate screens without Xinerama (to allow VDPAU to work).

The menu, all desktop icons, etc all load on screen 1, the Mythfrontend runs on screen 0.  The firefly mostly works out of the box too, though there are some nagging glitches (every other press of volume up turns the volume down unless you hit another key between - makes it hard to turn the volume up).  I still have my old config saved somewhere, so I'll restore.  

(Speaking of - does anyone know if there's a launchpad issue or somewhere else that I can support by providing my configuration??)

I now can use the machine as both, at the same time.  Firefly changes channels, volume, etc.  I can work at the other screen at the desk at the same time with minimal effect on the TV viewing.  Installation of the packages I need, like CUPS, VirtualBox, and OpenOffice completed the story (and it was nice to pick the things I wanted too).  Due to lack of desk space, I had to move the printer and scanner to the old machine across the room, but it's a small price to pay.

All in all, I'd say this is probably my favorite PC configuration, ever.  Rock solid, ultra-useful, and cost efficient (no need to buy a new PC).

Thanks to all on the Mythbuntu team for making 10.04 a solid product!
