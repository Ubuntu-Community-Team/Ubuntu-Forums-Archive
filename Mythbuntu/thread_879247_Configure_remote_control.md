### Post by LarryJ2 on 2008-08-17
I added a "repeat" to my volume controls so that if you hold the button down, the volume on screen indicator smoothly moves right or left. (from my Streamzap  /.mythtv/lircc file)

```
begin
    remote = Streamzap_PC_Remote
    prog = mythtv
    button = VOL_DOWN
    config = [
    repeat = 2
    delay = 0
end

begin
    remote = Streamzap_PC_Remote
    prog = mythtv
    button = VOL_UP
    config = ]
    repeat = 2
    delay = 0
end
```

Also I made the >> and << buttons jump to the commercial deletion breaks.
```
# >> Skip to next commercial break marker
begin
   remote = Streamzap_PC_Remote
   prog = mythtv
   button = >>
   config = z
   repeat = 0
   delay = 0
end

# <<  is skip to previous commercial break marker
begin
   remote = Streamzap_PC_Remote
   prog = mythtv
   button = <<
   config = q
   repeat = 0
   delay = 0
end
```
