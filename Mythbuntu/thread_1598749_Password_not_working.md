### Post by alien878 on 2010-10-18
Maybe I'm pointing out the obvious, but the password to do an upgrade/update is your own userID password.  The same one you use to do an sudo.  I.e. If you are logged in as userx, then you enter userx's password.

If you've lost that (quite possible as mythbuntu logs you in without entering a password), then one of the two methods in the link I posted earlier can be used to reset it.
