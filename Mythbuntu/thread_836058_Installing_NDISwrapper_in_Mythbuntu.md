### Post by ianmajor12 on 2008-06-24
I've tried MythBuntu in several different network configurations.

It's best to login to the live CD get the networking working and then install MythBuntu. The install (even the live frontend) tests the connection to the backend.

I suppose if it was an all in one install, you could enter 127.0.0.1 to get past the screen.
