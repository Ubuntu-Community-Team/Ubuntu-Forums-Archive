### Post by tgm4883 on 2008-11-05
For standard definition video, a single wireless link (backend hardwired) should be OK.  For HD video, you will need it hardwired.  

Running wires is a pain, but a pain only once if done correctly.  The best way to do it is have it done when the house is getting built.  If this is an after the fact problem, then running them under the floor for first floor stuff or through the ceiling and down the wall for second floor stuff would be your next best option.  After that, perhaps running a flat cat5e cable under carpet, and the last option would be 802.11n wireless, although that really is a last ditch effort.
