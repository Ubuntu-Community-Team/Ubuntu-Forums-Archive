### Post by ubuntme on 2008-02-05
utilities / edit keys / TV frontend

Add the required key to CHANUPDATE

You may need an alternative button to carry out the current function, however, my remote has an small 'enter' key mapped to enter, and a large 'ok' key mapped to return.  So ok changes the channel, enter enters the recording options.
