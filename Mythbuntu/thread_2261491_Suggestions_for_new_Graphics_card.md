### Post by lukeiamyourfather on 2015-01-19
Is this for a very old television? Practically all televisions made in the last decade have HDMI which is ubiquitous on modern graphics cards. If you must have native S-Video output there are still cards around with it though they are very old like this one.

[http://www.newegg.com/Product/Product.aspx?Item=N82E16814129108](http://www.newegg.com/Product/Product.aspx?Item=N82E16814129108)

A better option in my opinion would be to get a modern graphics card (better performance, more efficient, supported drivers) and if needed convert it to S-Video with an adapter like this one.

[http://www.amazon.com/Tmvel-PETHCSP-Composite-S-video-Converter/dp/B00GGFY6LW](http://www.amazon.com/Tmvel-PETHCSP-Composite-S-video-Converter/dp/B00GGFY6LW)
