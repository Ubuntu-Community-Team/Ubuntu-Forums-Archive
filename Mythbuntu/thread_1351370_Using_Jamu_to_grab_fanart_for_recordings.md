In 0.22 you can use jamu to get fanart for the "Watch Recorgings" screen, which is great. I live in Finland and often times the titles in EPG are not an exact match to the ones in thetvdb. I know that you can get around this by adding series name overrides to jamu.conf. The problem that I'm having is that many show/movie titles include punctuation marks, like colons and parenthesis.

For example:
Mentalist (K13)
Movie: Pirates of the Caribian

In the first example, (K13) simply indicates the PG-rating of the show. The second example just has a "movie:" prefix in front of the actual title. When you add an override to jamu.conf for a title that includes these punctuation marks, fanart doesn't get downloaded.

Anyone know a workaroud for this problem?
