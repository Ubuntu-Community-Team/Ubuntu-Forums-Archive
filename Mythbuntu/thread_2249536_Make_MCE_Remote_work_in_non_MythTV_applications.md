### Post by tgalati4 on 2014-10-23
Open up Keyboard Shortcuts in Control Center.  Look for the Volume Down, press that in the dialog box with mouse, then press the Volume Down key and see if a keycode shows up.  If so, then the remote should work with all applications that have Volume Down.
