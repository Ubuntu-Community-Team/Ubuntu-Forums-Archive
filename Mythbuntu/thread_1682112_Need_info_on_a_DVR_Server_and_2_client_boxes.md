Minimum you would need two boxes.

1 Mythtv-backend box that would house all your storage, media, tuners, etc

1 frontend system for each TV, you could put a frontend on the backend box for one of your systems.

You would need 1 tuner for each channel that you want to watch/record at the same time. I've got two myself, but rarely watch livetv. I would venture that most mythtv users set up many schedules and watch prerecorded content.
