### Post by klc5555 on 2012-04-07
Mythtv is not the correct software for this. Mythtv is designed primarily to be a PVR, to record and play back TV. It does have some video playback capability, but the TV scheduler, database, and recorder that is the mythtv backend must be configured before anything else in mythtv will work.

The correct software for what you want to do is either XBMC, if you want a complete media center solution for video playback, or possibly just a standalone video player like VLC, Mplayer, or Xine.

Therefore, rather than install Mythbuntu, which is Mythtv specific, you would do better installing one of the regular Ubuntu variants, like Xubuntu, and then install XBMC, VLC, Mplayer, or Xine as you may prefer.

### Post by klc5555 on 2012-04-07
This is a bit beyond the scope of this particular forum, which is limited to the installation and use of Mythtv on the various flavors of Ubuntu Linux. 

However, you probably received no replies to your XBMC posting because that message was not clear nor complete. In particular the operating system being used on the computer, and, if this operating system was in fact Linux, the distribution and version of Linux being used was not mentioned.

If you choose to use some flavor of Ubuntu Linux as the operating system on your computer for the purpose of watching downloaded videos, you have several options. Mythtv is probably the least satisfactory solution for this because it is intended for TV, assumes you have a television recorder card on or connected to your computer, and is complicated to set up and configure. If you wish to use it anyway, you should read the Mythtv installation manual at: [http://www.mythtv.org/wiki/User_Manual:Index](http://www.mythtv.org/wiki/User_Manual:Index)

If you have installed some version of Ubuntu Linux on your computer (Ubuntu, Xubuntu, or other) as your operating system, and you wish to try XBMC, you will not be burning XBMC to disk, as it seems you did before, but rather install it directly using a guide such as: [http://wiki.xbmc.org/index.php?title=HOW-TO:Install_XBMC_on_Ubuntu/HOW-TO_1](http://wiki.xbmc.org/index.php?title=HOW-TO:Install_XBMC_on_Ubuntu/HOW-TO_1)

Or, finally, if you have installed some flavor of Ubuntu as your operating system, you may install any of the simpler standalone video players previously mentioned (VLC, Mplayer, Xine, or others) directly from the download repositories that Ubuntu uses. And any of these standalone players will be satisfactory for watching downloaded videos. 

Good luck.
