### Post by Caps18 on 2008-11-12
Last night I troubleshooted a problem that I was having.  I have enough hard drive space available, but MythTV wouldn't create the file when recording shows.  I thought that it had to be a database issue, but before rebuilding that, I decided to use the watch live TV feature and found out that the signal strength of the TV stations was from 0%-17%.  One of the wires had come loose connecting the antenna and that solved the problem.

I think it would be a good idea to add a error message saying that the signal strength was too low (Or if there was no disk space it would say that).  Now it just says "File not found.", instead of giving the reason that it had problems making the file.

I'm using 7.10, so it might be fixed in version 8.
