### Post by neutron68 on 2009-10-28
Mythbuntu upgraded pretty smoothly over the Net by the command line "sudo update-manager -d". :D

After the upgrade and reboot, the first thing I noticed is that, Ubuntu 9.10 askes me to log in every time as user eric.  It didn't do this before the upgrade.  Before, Mythtv launched and no login was required.
Should I report this as a bug?  How do I report bugs?

The next thing I noticed is that when I navigate to a program that was previously recorded, I can use the arrow keys to select it and I can cause it to play back by hitting the ENTER key.  But, I cannot use the right-arrow key and get a menu that will allow me to delete the program.  It's like it doesn't know how to respond to the right (or left) arrow any more?!   Should I report this as a bug?  How do I report bugs?

I guess I could also report that I have a Hauppauge HD-PVR which I have been playing with in Windows XP.  I went into the Mythtv backend configuration and set up the HD-PVR as an h.264 device.  It is now in my capture card list.
I was unable to swtich to it in the WATCH TV function.  I believe you are supposed to be able to switch between video sources with the Y key during the WATCH TV function.  My system stayed on the HD-HOMERUN tuner and didn't switch to the HD-PVR.

### Post by neutron68 on 2009-10-29
I found a workaround for the login problem.

I went to the Ubuntu desktop, did a right-click and chose Applications, System, then Login Screen.  I changed the setting to automatically login user eric at startup.  Now, Mythbuntu 9.10 behaves like my previous 9.04 install did.

Eric
