I believe this is different from the General (Advanced) section [Utilities/Setup -> Setup -> TV Settings -> General (4th page labeled Advanced)].  This section says that it does not affect the scheduler and will only let recordings start or finish late if it doesn't have any sort of conflict.  Anyway, I set these to zero just to be sure.

Where are the default settings for the scheduler so new recordings won't be forced to start one minute early and one minute late?

### Post by garethiandavies on 2009-01-05
you will find this option in the recording priorities > set recording priorities menu.  

There used to be an independent one option in mythweb for schedules set up in there, though that appears not to be there anymore.  Perhaps the mythfrontend menu now covers mythweb configuration too in this instance.

PS:  I have checked, and yes - the priorities menu now configures mythweb for you too.
