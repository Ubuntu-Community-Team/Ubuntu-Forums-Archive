### Post by movieman on 2009-01-30
I had that problem yesterday, getting two Myth frontends and a couple of terminals starting whenever I logged in! I closed all the windows, quit out of the desktop and selected the 'save windows' tickbox (or whatever it's called in the dialog box that asks if you want to log out, reboot, etc). Then I logged back in, quit again and cleared that tickbox, logged back in and finally had a clean desktop with only one frontend.

To be honest, it's a pretty awful user interface; there should be a menu option somewhere to save the current desktop rather than a tickbox that stays ticked until you exit without it ticked.
