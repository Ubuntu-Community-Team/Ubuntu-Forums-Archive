### Post by volkswagner on 2008-08-09
Yes install vista first.  If you wont be storing much on vista and are short on disk space 20gigs would suffice.  Depending on your vista install medium, you can create the small partition first before install.  This can be using the mythbuntu live cd, or vista.  If you only have a system restore disk, you will need to install vista than shrink the partition using the live cd, or within vista.

If you are creating separate partitions for recordings and such 10gigs is plenty for / on mythbuntu.  My current / and /home total less than 5 gigs consumed for mythbuntu 7.10.
