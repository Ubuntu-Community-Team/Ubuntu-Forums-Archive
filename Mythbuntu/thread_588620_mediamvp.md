### Post by superm1 on 2007-11-15
It shouldn't be any different than setting up with a standard box for a remote frontend.

Open MCC, enable MySQL server.

Open up mythtv-setup and make sure both IP addresses in the general area are set to be the ip address of your network card.

Reboot.
