Today I got a message for update. There seemed to be alot of mythtv 0.21 updates so I gladely accepted. The problem is now that I can´t get acces to my mythvideo menu. ### Post by murbanci on 2008-03-12
You have to uninstall the old mythdvd from symantec package manager and then mark mythvideo for upgrade and everything will work.  The problem is because the 0.21 version of mythvideo combines mythvideo and mythdvd.  Hope this helps.

---

### Post by axelsvag on 2008-03-12
Thank you very much. I went into synaptic choosed mythvideo and then choosed upgrade for mythvideo. Synaptic suggested that mythdvd had to be removed and I pressed apply. After a restart of the frontend the mythvideo worked again. The problem is solved. By the way how can I put the SOLVED tag in the title?
