### Post by movieman on 2009-11-09
Is there any way to configure MythTV so that recordings override live TV? It's annoying when we accidentally leave it on live TV and then miss three or four recordings even though no-one is watching anything.

---

### Post by SiHa on 2009-11-10
If you don't have the 'Allow LiveTV to move recordings' box checked, I thought it should display an on-screen prompt that it's going to change the channel to make the recording.
Failing that, in 9.10 there's the Live TV Idle Timeouot option, which will prompt to exit Live TV after a set period of inactivity. I have mine set to 60 minutes.
