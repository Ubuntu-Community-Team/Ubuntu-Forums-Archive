### Post by tgm4883 on 2009-08-21
There will be an installer slideshow in Ubuntu 9.10 (think things like "did you know", tips, good plugins to use, etc). This will be displayed after the user has inputted all selections while the installation is happening. What I'm looking for is what info we should provide in this area. I've included some guidelines below in case anyone wants to do a page on their own.

> * Write in the present (no "Ubuntu will" or "Ubuntu would"). Not just present tense, but with an awareness of what is going on around the slideshow. We can assume the user is running a default Ubuntu session as he installs it, and also that he has chosen to install Ubuntu willingly; that person is now an Ubuntu user. The sales-person voice should not exist. If you start to hear that voice, file a bug immediately.

* Avoid the car instruction manual voice where possible. That is, try not to tell people where specific features are, but what they can expect to find. This is because the slideshow won't follow our users everywhere, so we have to answer a different question: "What next?" If we fixate energy on pressing buttons, it will not stick :)

* Write from the perspective of us (people, probably) explaining Ubuntu and free software to the immediate end user; the person behind the screen. Think of it as enthusing with a friend about this great new thing they have started to explore, making sure they know what they need to know to really enjoy it.

### Post by NautiusMaximus on 2009-08-25
How about pointing out some of the places where it's likely the user is going to want to change the default settings?

There are a huge number of settings in Mythbuntu. The average user probably doesn't have a clue what half of them are. For many settings, it's probably perfectly OK to assume that the default settings are what you want.

However, there are places where the user probably wants to change the default settings. Either because they refer to something that is very specific to someone's particular hardware or TV viewing habits, or because the default settings are just plain daft.

To give an example, after a considerable amount of swearing and tearing out of hair, I finally discovered that the reason why I couldn't play DVDs was that there is a setting to enable or disable DVD playback, which is set to disabled by default. I'm sure I'm not the only user who felt the need to change that setting. Pointing out in advance where things like that are would make the whole process of setting up the machine a lot easier.

### Post by Slate8 on 2009-08-25
Here are some of the quick, leading tips I would have found useful when first installing Mythbuntu:

**Did you know:**

You can can stream live and recorded TV around your house if you install Mythbuntu on multiple computers.

Mythbuntu can use a dual tuner DVB card as 10 tuners with "MultiRec".

Mythbuntu supports subtitle and interactive (aka red button) functionality on DVB streams.

Mythbuntu comes with the MythWeb plugin for web browser based scheduling and configuration. Set something to record when you aren&#8217;t at home.

The MythVideo plugin can play pretty much any video format such as DVD ISOs and MKVs and has great IMDB integration

But.

The slideshow has quite a few references like:

"For more information have a look at the MythTV Wiki: [[COLOR="Blue"]http://www.mythtv.org/wiki[/COLOR]](http://www.mythtv.org/wiki)
