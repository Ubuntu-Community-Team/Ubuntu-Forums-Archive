### Post by Chris Musampa on 2010-02-14
You want to change
REMOTE_DEVICE="/dev/lirc0"

to 

REMOTE_DEVICE="/dev/input/event5"

then restart lirc.

You may find that the event number changes sometimes after a reboot so it is worth setting up a udev rule, see about halfway down the page here:
[http://www.mythtv.org/wiki/Hauppauge_WinTV_Nova-T_500_PCI](http://www.mythtv.org/wiki/Hauppauge_WinTV_Nova-T_500_PCI)
