### Post by itlarson on 2011-07-17
I'm using a DViCO FusionHDTV7 Dual Express.  The only glitch with it is it doesn't show the signal strength for live TV, but it records fine.  It seems to do OK with OTA television, which isn't true for all tuners, many of which assume a cable signal.  I was wondering if you could get it in Australia (I'm in the U.S.), But Divco's website gives a list of [Australian retailers]("http://www.fusionhdtv.co.kr/eng/whereToBuy.aspx").
