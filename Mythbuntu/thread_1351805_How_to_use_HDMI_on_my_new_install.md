### Post by Caysho on 2009-12-11
You should be able to use the HDMI out as a primary display port, so the output from the motherboard should just work.
Whether the display mangles it (overscan, reports incorrect resolution etc) is another matter.  

Check the TV manuals.  I know LG and Samsung have "Just Scan" options which present the picture without any post processing, which is what I am using.  Have a look at the [pixel mapping wiki]("http://pixelmapping.wikispaces.com/").
