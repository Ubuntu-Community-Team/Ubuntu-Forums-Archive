### Post by AKADAP on 2009-10-30
After upgrading to 9.10, the custom menus I added to the main menu are gone. If I go to "Applications>Settings>Main Menu", they are all there, and look correct, but they do NOT appear in the menus.

For example: I added an electronics menu to the main menu that has a link to a schematic capture program. "Applications>Electronics" just isn't there, so I have no way of starting the schematic capture program. If I look in "Applications>Settings>Main Menu", the Electronics menu is there along with the link to the schematic capture program.

### Post by AKADAP on 2009-11-01
(this entry may not go well, Ubuntu forums has learned a new trick of infinitely reloading every page. I have to hit the stop button to be able to read anything. I don't know what will happen when I submit this post.)

Because I had installed Ubuntu first then installed Mythbuntu as a package on top of it, Gnome used to be my default window manager. Previous updates had left this alone. 9.10 on the other hand arbitrarily decided that it should switch the default to the window manager that comes with Mythbuntu. 9.10 also comes with a select box on the login page that was in the muted colors of the new color scheme, so it took me a while to notice it was there. I switched back to Gnome, and my menus were back.
