### Post by laga on 2007-11-01
Hi,

I'd recommend upgrading the system memory to at least 512M. 256M will likely work OK, but I've found that it works better with 512. If you run into issues, eg with swapping, try setting your theme to Iulius.

You're not gonna record much with 40G of disk space, but you probably knew that already.

Wrt a tuner card: a PVR 150 should be OK, read up on "IR blaster" - that's what you use to control the Set Top Box.

Your VGA chipset will likely be OK.

So, yeah - you're pretty much good to go. Good luck :)
