### Post by MickSulley on 2014-06-28
Just downloaded 14.04 and did a clean install.  After install Mythbuntu opens up and looks OK, but the underlying desktop is set to a resolution so fine that I cannot read it, the characters are so small they are not properly formed, so even a magnifying glass does not help.  This means that I cannot do anything.

I have a ASUS GeForce 210 Nvidia 1Gb video card.  I tried the install selecting GeForce video driver and generic driver, both give the same result.  

Any ideas how to overcome this?

### Post by itlarson2 on 2014-06-28
Try unchecking "custom dpi settings" in settings manager->appearance->fonts.  Appearance has a shirt for an icon, fonts is the third tab, and custom dpi settings is the bottom line of the tab.

---

### Post by lordviperz on 2014-06-30
I just installed a new system with the same card from MSI. I had the same problem. I had to set the DPI settings down to 90 from 96 and now the screen looks fine.
