### Post by Boppy3125 on 2009-07-20
If I am reading you question correctly, I think the answer is that it can be made as a diskless frontend.  I thought I read that in the MythBuntu 8.04 release notes.  A quick google also shows minimyth as a distro for network booting a diskless frontend.  I think there is also a project for a bootable CD that is a fully functional frontend.  I don't believe a true Thin Client is there yet due to the decoding and display requirement.  Some are using the newer ION based machines to use as a lower power frontend, still with enough video horsepower, but I am not sure how well that is all working.
