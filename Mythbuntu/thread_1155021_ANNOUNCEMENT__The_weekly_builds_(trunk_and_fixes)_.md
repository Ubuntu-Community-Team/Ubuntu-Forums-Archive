### Post by tgm4883 on 2009-05-10
The Weekly builds PPA (both -fixes and -trunk) as well as the testing PPA have changed locations (moved) and are no longer available at the old locations.  This is because launchpad now allows us to contain all the PPA's in a single team rather than across several teams.  We have a single package available that you can install to activate these PPA's.  It will prompt you about which you want to activate.  See the following links for details.

[http://www.mythbuntu.org/testingandreporting](http://www.mythbuntu.org/testingandreporting)

[http://www.mythbuntu.org/auto-builds](http://www.mythbuntu.org/auto-builds)
