### Post by purduephotog on 2010-04-18
When invoking tightVNC, you must specify the screen number apparently with the version of VNC installed with mythbuntu 10.04 B2 (and then upgraded).

so, IP:1 (for first display), :2 for second (or 5901, 5902).

### Post by purduephotog on 2010-04-18
apt-get install x11vnc

Reboot.

Start vncserver :2
connect via tight vnc to IP:2 (or IP:5902)

Start /usr/bin/myth-control-center from command line

Enable VNC
Add vnc password
reboot system

connect via tight VNC on port 0

Works.
