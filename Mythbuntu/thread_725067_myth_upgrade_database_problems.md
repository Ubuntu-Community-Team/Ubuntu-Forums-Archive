### Post by nasha on 2008-03-16
```
optimize_mythdb.pl
```
This is a common script that is used to, funnily enough, optimize the database tables! You pick up many things when you dig deep enough :)

Myth username for the db is MythTV, and the password is randomly generated by MythBuntu, unless you changed it. As far as i know there is no 'root username' for the db, as there isnt actually a 'root' account in ubuntu. The sudo command is used to give users root privileges.

### Post by Nikas on 2008-03-16

### Post by txcrackers on 2008-03-16
> **nasha said:**
> ```
optimize_mythdb.pl
```
As far as i know there is no 'root username' for the db, as there isnt actually a 'root' account in ubuntu. The sudo command is used to give users root privileges.
Yes, and no - there is no "root" user *password* set on the database, so you can login to mysql as the root (admin) user. (This is normally not considered to be a very good practice, by the way.)

This is **not** the same as the "root" system account - which does exist, but in Ubuntu land doesn't have the password set (but you can).
