### Post by PaulRSte on 2013-01-26
When switching to the workspace running mythtv using Ctrl/Alt/arrow, the Mythtv screen judders repeatedly for 3 or 4 seconds before settling down. The icon for that workspace also shows a series of smaller icons inside it.  I've tried this with mythtv in either workspace 1 or 2 and the result is the same.  Switching to non mythtv workspaces does not seem to cause an issue but this has only been tested with the desktop and Firefox.

I am currently running ubuntu 12.4.1 with mythtv 0.25.  This problem only started happening after a run of update manager about 2 weeks ago.  A further run of update manager has not changed this.

Thanks - Paul

### Post by PaulRSte on 2014-01-16
This did go away for a very short perioed and returned after another update. After upgrading to 12.10 the issue has cleared and feels more solid.  If you are going to go this route from 12.04 LTS, be aware that the 13.04 upgrade disables workspaces [http://askubuntu.com/questions/260510/how-do-i-turn-on-workspaces-why-do-i-only-have-one-workspace-in-13-04]("http://askubuntu.com/questions/260510/how-do-i-turn-on-workspaces-why-do-i-only-have-one-workspace-in-13-04") so any autologin should be disabled before upgrading just in case you can't exit from the auto logged in app. e.g. mythtv 0.26.
