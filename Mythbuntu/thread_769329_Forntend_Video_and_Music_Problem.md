### Post by apauna on 2008-04-28
I think I have figured it out. At least this worked for me.

Opened fstab on my myhtfrontend and mounted music and videos via samba.

Here is my fstab:

```
# /etc/fstab: static file system information.
#

# /dev/sda5
UUID=ce2b9cf7-802b-4fe1-889f-5c5fd2528a7b none            swap    sw              0       0
/dev/scd0       /media/cdrom0   udf,iso9660 user,noauto,exec,utf8 0       0
# main-myth.local:/var/lib/mythtv/music  /var/lib/mythtv/music  nfs  suid,dev,exec  0  0

\\main-myth.local\music  /var/lib/mythtv/music  smbfs uid=1000,dir_mode=0776,file_mode=0776,credentials=/root/.smbpasswd,gid=1000  0  0
\\main-myth.local\videos  /var/lib/mythtv/videos  smbfs uid=1000,dir_mode=0776,file_mode=0776,credentials=/root/.smbpasswd,gid=1000  0  0
``` 

Here is the .smbpasswd file:
```

username=<samba user goes here>
password=<password goes here>
```

I placed this in root with read/write to root read only to root group and no access to others
```
chmod 640 /root/.smbpasswd
```
