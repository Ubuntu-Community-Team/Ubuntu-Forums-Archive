### Post by khPWXxF on 2014-06-14
[COLOR=#000000]Are you using nvidia graphics?
I found both 8300 and 8400 are unstable with drivers 311.  Downgrade to 304.
[/COLOR][COLOR=#000000]
```
sudo apt-get install nvidia-304
```[/COLOR][COLOR=#000000]
This driver is more stable than 311 but not as good as the 295 shipped with 12.04. Limited testing with VDPAU slim and VDPAU normal works fine with SD recordings but the HD demo in setup (man trudging through snow) flickers a bit but it was 100% solid with 295.

However, there are nvidia improvements in 0.27.1 which may change things completely.
