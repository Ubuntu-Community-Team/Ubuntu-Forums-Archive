### Post by ianhaycox on 2008-11-17
A question about the width and height for nuvexport on UK DVB-S transmissions.

I have a MythTv setup using a Hauppauge WinTV-HVR4000(Lite) DVB-S/S2 to record UK TV programmes from Astra 2D. This all works well but to watch programmes on my TV I use nuvexport to export to Xvid onto a USB key then plug the key into the TV which plays Xvid files.

The question is what width and height sizes should I use during the export to retain the best quality. File size isn't a big issue.

The excellent page at [http://www.mythtv.org/wiki/index.php/Nuvexport](http://www.mythtv.org/wiki/index.php/Nuvexport) suggests using 624 as the width for widescreen 16:9 broadcasts but seems to imply it is different for UK broadcasts.

For example, running nuvexport shows,

```

You have chosen to export 2 episodes:

  1. 6940 (BBC HD):
     Mon Nov 17 11:02:00 2008 (Mon Nov 17 11:02 AM) x  (1.8182:1)
  2. Apparitions
     (Thu Nov 13 10:00 PM) 720x576 MPEG2 (16:9)

```

The first is an HD broadcast. What width to use ?

The second is an SD broadcast. Do I use 720 as the width and take the suggested height (which is less than 576) ?
