### Post by tgm4883 on 2008-03-20
We would like to announce the availability of Mythbuntu 8.04 beta.

Please see [www.mythbuntu.org/hardybeta](www.mythbuntu.org/hardybeta) for more information about this release.

This Mythbuntu release is based off the MythTV 0.21 release. It will not work with a backend or frontend from the Mythbuntu 7.10 release. You will need to either switch those machines to the Gutsy backports, or upgrade them to 8.04.

To upgrade to Mythbuntu 8.04 from 7.10, please open a terminal and type

```
sudo update-manager -c -d
```

It is very important that if you discover any bugs with the upgrade that they get filed early. If too many people wait, issues will be much harder to discover.

We would like to get in as much testing as possible, especially with bugs that are in MythTV. The MythTV developers are very responsive to bug reports, so if you discover a bug with MythTV in this release, please file it.
If the bug contains a crash report generated by the Ubuntu apport service, file it with launchpad. If it doesn't, file it with [http://svn.mythtv.org](http://svn.mythtv.org).

If you would like to get support from other users in this forum, please be sure to include a [hardy] or [8.04] tag in your post title to differentiate your post from the 7.10 stable release.
