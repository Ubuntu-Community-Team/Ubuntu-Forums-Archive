### Post by tgm4883 on 2008-10-30
Mythbuntu has the same repos as Ubuntu, so you will receive security updates as well as all other updates from the ubuntu repos.  However, if you use weekly builds, these are only available for the current new release.  ie, they will stop for Hardy as of today.
