### Post by kpl388 on 2009-05-26
I had the same problem. I was able to fix it by using irrecord to generate a new lircd.conf files. I think the remote has a different toggle bit mask than the one listed in the default MCE Remote (new version Phillips) settings. You can get away with just replacing the top portion of lircd.conf.mceusb file with that generated by irrecord. The section I replaced is below. Then, each remote code has to be changed from 0x00007xxx to 0x00001xxx. I believe this is an issue with 64-bit processors, but I'm not positive. Then run mythbuntu-lirc-generator and mythbuntu-lircrc-generator. Good luck!

begin remote

  name        mceusb2
  bits           13
  flags RC6|CONST_LENGTH
  eps            30
  aeps          100

  header       2661   863
  one           435   439
  zero          435   439
  pre_data_bits   24
  pre_data       0x1BFF82
  gap          106043
  toggle_bit_mask 0x8000
  rc6_mask    0x100000000
