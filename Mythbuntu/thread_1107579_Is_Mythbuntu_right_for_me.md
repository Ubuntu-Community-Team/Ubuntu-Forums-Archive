### Post by lgiordano on 2009-03-27
From one noob to another...Mythtv is great. At the same time, its as frustrating as it is rewarding. 

I've built two Mythtv boxes. The first one was with spare parts (like you're thinking of doing). The second one was made from scratch with new, quiet, and lower power consumption parts. I have it sitting in my living room and on 24hrs.

You'll be able to use it as a print server as well. Mythbuntu is basically a stripped down Ubuntu with Mythtv installed for you.

There's a Windows app called WinMyth: [http://winmyth.sourceforge.net/]("http://winmyth.sourceforge.net/") that can be used to watch your recordings on windows pc's. I only tried it for a few mins as it was lagging a bit over my wireless G network. You can easily set up windows file sharing on Mythbuntu and just watch the video files directly in media player. You can even boot to the Mythbuntu live CD in a Windows computer and use it a frontend to watch your recordings, all while using Mythtv.

Yes, you can gain access to your Mythtv box online. More info [here]("http://www.mythtv.org/wiki/MythWeb").

My advice to you is to do as much research as you can. Then, do more research.

### Post by SiHa on 2009-03-27
> Is the community responsive and willing to help dumb n00bs?
Very. 
I only dived into Linux / Myth about a year ago, and I've found the Ubuntu community to be extremely helpful.

DOS with a GUI is quite a good description. If you can remember back to the days of DOS ans were happy enough with a CLI then, you'll be OK with Linux, most likely. I was daunted initially with the though of having to compile lots of stuff, but I think I've had to do it twice. As lgiordano says, Linux can be very frustrating, but on balance, Mythtv is worth the struggle.

Just one bit of advice. Once you have Mythtv setup working as it should**. Leave it alone**. Particularly the Backend machine.

As you've already got the hardware, I'd say give it a go and enjoy the learning experience.
