### Post by uncle hammy on 2010-11-06
First off, yes, I am aware 0.24 is not officially released, and there is an inherent risk of things like this happening.

MythMusic plugin is installed on all my systems.  However, there is no trace of it anywhere in the frontend.  No "Play Music" in Media Library, no "Music Settings" under "Settings", no "Music Settings" under "Settings>Media Settings".....Nothing.

It was definitely present with build 27009, however with the 2 builds since then, it's gone.  I even double checked on a seldom used frontend this morning running 29009.  It was there, upgrade, gone. 

Anyone else running 0.24 noticed this?  I haven't been able to find anything relating to it in the bug reports, mailing lists, etc.
