### Post by utar on 2010-08-29
Sorry I meant 0.23 stock or autobuilds?

Have you read [http://www.mythtv.org/wiki/MythVideo_.22_Transition_Guide#Metadata_Lookup](http://www.mythtv.org/wiki/MythVideo_.22_Transition_Guide#Metadata_Lookup)

Basically if you press "w" when you have selected a file Myth should automatically download the metadata.  If it doesn't work if you press "m" and select metadata from the menu you can manually type in a movie reference.
