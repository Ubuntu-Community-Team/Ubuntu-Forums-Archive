### Post by uMuppet on 2008-12-16
Go into Settings -> Video Manager

Find the move you want to get data for press Menu, download IMDB data.  If you have alot of movies it might pay to run an automatic script to do this for you.  There are a few [here]("http://mythkiwi.com/index.php/remository?func=select&id=3")
