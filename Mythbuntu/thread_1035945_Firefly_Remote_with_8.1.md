### Post by Dragonflyoh on 2009-01-10
I recently updated to 8.1 and am trying to get my Firefly remote to run with Mythtv. I have used the Myth Control center and that does not have the Firefly listed. I downloaded the Infrared Remote Control software which has the Firefly listed. The program recognizes my USB RF receiver device but after I install the remote does not work. I tried irw and it is loking for an input but nothing shows up as I press the buttons. I have looked and found nothing with 8.1 and the Firefly remote. Any assistance would be appreciated.

---

### Post by Dragonflyoh on 2009-01-11
I found the file home/name/.lirc/mythtv generated by the installer did not have any information about the remote control keys. I copied the keys from  a Lirc config file that was modified by Ryan Schmitz  and it works great. A couple of key to change to my liking but I now have my Firefly remote working.
