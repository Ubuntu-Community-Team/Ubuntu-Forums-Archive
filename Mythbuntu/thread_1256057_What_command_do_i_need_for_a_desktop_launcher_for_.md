### Post by novellahub on 2009-09-04
I just did this yesterday.  Right click on the XFCE desktop and there should be a option of adding a new launcher. Click that and a dialog box comes up. In the command field, if you start typing myth and it should suggest mythvfrontend as one of the options.  Choose that and pick a icon (There is a Mythtv icon as one of the choices).  Then you should have shortcut (Launcher) on the desktop.
