### Post by funkydan2 on 2009-01-04
From a quick look at Amahi's [Features](http://www.amahi.org/features/gallery) it doesn't seem that these tools are really aiming at doing the same kinds of things 'out of the box' (though you could setup the same features from either starting point).

Mythbuntu is focussed towards making a Home Theatre PC (or a network of such things) whereas Amahi seems to be focussed towards making a home network server (with things like backup, file searching, media streaming - but not HTPC).

Of course, if Amahi looks like it does everything you want *except* MythTV, I'm sure you could easily install MythTV on that Fedora based system. Alternatively, if TV recording/wathing (MythTV) is what you're main goal is, and the other things are peripheral, then Mythbuntu (or Mythdora) might be a better starting point.
