### Post by uteck on 2014-07-05
If you do upgrade, look to see if you can use Mythbuntu Control Center to update the MythTV version first then update the system to 12.04, that way they database does not jump forward a few versions which might cause a problem.

### Post by lordviperz on 2014-07-07
For what its worth, I just finished building a new system because I was running 10.04 on an older system. WHile the 10.04 system still worked, I was like you and wanted to upgrade to the latest. So I built the new system for 2 main reasons. One was to have a system I knew would run 14.04 and future releases and second to test the waters of an upgrade from 10.04 to 14.04. So I simply copied all my recordings over to the new system and restored the 10.04 database (MythTV 23) to the new 14.04 system (MythTV 27) and it worked just fine. No errors.
