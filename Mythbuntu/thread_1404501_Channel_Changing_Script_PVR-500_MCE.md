### Post by ian dobson on 2010-02-14
Hi,

OK what you need to do it write a simple script that calls the chchange script with the missing parameters.

Something like:-

```

#!/bin/bin
/usr/local/bin/chchange.sh REMOTE_NAME  $1

```

Change the text REMOTE_NAME to the name of your remote. The $1 is the parameter sent by mythtv. When you've created the script test it with scriptname 1234 where 1234 is the channel number you want.

Regards
Ian Dobson
