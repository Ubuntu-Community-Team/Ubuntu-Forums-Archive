### Post by volkswagner on 2008-08-31
If you want to do anything inside mythtv, 

[http://www.mythtv.org/wiki/index.php/Main_Page]("http://www.mythtv.org/wiki/index.php/Main_Page")

Most questions can be found above.

To get you started you will want to set up ripping music and where to store it.

The default location is /var/lib/mythtv/music
You will need to choose how you want it ripped what file format, ie: mp3 or choose Flac if you want lossless.


I don't think mythv rips the album art so if that is important you may need to find the settings to let it add album art or use an alternative ripper via the desktop.
