### Post by funkydan2 on 2008-09-18
There's probably a few ways you could do this.

1.) From within mythtv, mark out the sections of the recording you want to keep, transcode the file to remove those sections and then either burn to DVD or copy the raw file to a disk somewhere.
2.) Open up the MPEG file in some sort of video editing program, chop up the bits you want and away you go.

I don't think there's a way to, from a remote computer, just download the middle bits out of the recording, but it is very easy to edit recordings.
