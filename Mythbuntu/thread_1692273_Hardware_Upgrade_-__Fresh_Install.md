### Post by LowSky on 2011-02-22
All that work upgrading and you're going to keep a PVR-150? There are newer cards with much better support. But I'm guessing you need analog/blaster support for a Satellite TV box.

I support the idea "if it works, why change". But I like to tinker so I'll upgrade to a RC the first chance I get.


There is a compromise, switch to LinHES. Its' based on Arch Linux so its a rolling release, so legacy stuff is gone when its not needed and upgrades are usually smoother. Of course this is only an option if its strictly a mythtv box... personally I use Ubuntu with the mythtv packages, and use my PC as an all-in-one server, desktop, media player.
