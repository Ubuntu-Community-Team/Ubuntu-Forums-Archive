### Post by gareththered on 2007-10-13
When I insert a DVD into my Mythbuntu machine the 'Play DVD' options appears in MythTV followed by the File Manager opening to show the DVD contents.
Is there a way to stop this happening?

### Post by stevetv on 2007-10-13
yep.  in the frontend settings.  setup > media settings > dvd settings > general settings > on dvd insertion

select "do nothing".

you probably know, but the mythtv wiki has lots of usage information including a users manual...  you can find it here [http://www.mythtv.org/wiki/index.php/Main_Page](http://www.mythtv.org/wiki/index.php/Main_Page)

### Post by superm1 on 2007-10-13
Well in this case, if you were on beta, this was resolved at RC.  There is an xfce setting to control what happens when DVDs are inserted.
