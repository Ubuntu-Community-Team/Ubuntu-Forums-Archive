### Post by brianafischer on 2010-08-20
For some clarification on how to do this:

1. Install this deb [http://www.mythbuntu.org/files/mythbuntu-repos.deb](http://www.mythbuntu.org/files/mythbuntu-repos.deb)
2. Update the system with Update Manager or sudo apt-get update
3. run "sudo dpkg-reconfigure mythbuntu-repos" from a terminal

OR

System -> Administration -> Mythbuntu Control Centre (MCC)
Select the "Repositories" tab on the right pane.
Select 0.23.1 and "PPA"
Click Apply

[IMG]http://mythbuntu.org/files/start_mcc_gnome.png[/IMG]
