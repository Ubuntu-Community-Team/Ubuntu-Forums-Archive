### Post by mike_c2 on 2015-09-04
I just did a clean install of mythbuntu 14.04.2 applied all updates and then had a problem with listings not loading into the guide. I have two Schedules Direct Subscribed Lineups and neither one would populate during setup. After trying to setup the lineup several times without success, I loaded 14.02.2 again and applied all updates. I still could not get either subscribed lineup to load.**Resolution**:I loaded 14.02.2 a third time and did not apply any updates and went right into Mythtv setup. This time I was able to get both of my subscribed lineups to populate and all of the listings loaded in the guide. When I was all done with the Mythbuntu setup, I applied all updates and everything is working now.**Bug?**It seems like there is a bug in one of the updates that gets applied that prevents the Subscribed Lineups from populating properly. If updates are applied after the schedules direct subscribed lineup has been completed, this issues does not create problems.**Thoughts? Should I file a bug?**I hope this helps others, I was quite frustrated until I figured out a work around.
