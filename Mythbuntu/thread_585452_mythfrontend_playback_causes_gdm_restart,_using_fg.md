### Post by laga on 2007-10-22
Hi,

your X config looks good now, I suppose. Your current problem is related to your TV tuner (you probably knew that already ;)).

For DVB, I'd usually recommend kaffeine for testing. But that's a KDE app which will pull in a lot of dependencies-  not sure if you like that. Come to think of think, they should all be removed again if you remove kaffeine.

Anyways, there's also the dvb-utils package which comes with a few command line utilities like tzap and scan (or was it ""dvbscan"?) Not exactly user friendly, but a good way to find out if your card is working.

Maybe xine works too, but I'm not sure how to set that up for DVB. I believe it needs a channel.conf file which can be created using dvb-utils ;)
