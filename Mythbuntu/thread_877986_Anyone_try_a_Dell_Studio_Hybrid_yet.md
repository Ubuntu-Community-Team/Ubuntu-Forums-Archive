### Post by theacoustician on 2008-08-02
Dell has been showing off [this "green" PC]("http://www.dell.com/content/products/productdetails.aspx/desktop-studio-hybrid?c=us&cs=19&l=en&s=dhs&ref=dthp") for a while now, but this week is the first time I can recall actually seeing it on sale. You can get it with up to a T9500, 4GB RAM, Blu-Ray, and X3100.  It has DVI and HDMI outs and TosLINK, so in theory, this could make a sweet little front end.  Since theory doesn't always line up with reality, if you happen to grab one, post up your review here.
