### Post by lmcboy on 2009-04-12
Hello,

What can I do to get a DVD from Mytharchive that will be playable in my DVD player?  After a few frustrations (including old lockfiles causing mytharchive to fail silently, and needing to install "unstripped" variants of some support packages), I finally got mytharchive to run successfully.  However, the DVD it produced was only playable on my Macbook Pro.

I then tried copying the ISO image over to the Macbook, and burning it to disc using the OS X program "Disk Utility".  This DVD is playable on my computers (Mac, Windows, and Ubuntu), but not on my relatively new Sony DVD player, which says "Playback prohibited by area restrictions".  This player is region 1.  I presume the Mytharchive DVD is region-free, and indeed my Macbook recognizes it as such.

Is there anything I can do to get a burned DVD that will play on my Sony DVD player?  One final note: we have played burned DVDs on this player before, which were created with the Mac program "iDVD".  The blank DVDs I'm using are all from the same batch.  So I don't think the problem is inherent to either my DVD player, or my batch of blank DVDs.
