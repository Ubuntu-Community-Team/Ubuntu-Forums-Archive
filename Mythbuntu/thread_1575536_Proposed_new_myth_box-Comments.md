### Post by LowSky on 2010-09-18
tmcgary that list of processors is for 3D High Def... no need to worry about that, I really doubt you will be playing those. As long as the video card can run HDCP and 1080p you should be fine.

Dont use onboard if you dont have to. It will waste RAM resources and onboard graphics tend to run hotter when used. It might effect system performance.

most mATX cases accept a normal ATX power supply, that corsair is a really good choice.. stick with it if you can.
 I have the 550 Watt version and I'm kicking myself for not buying the 750 Watt option. Just remember the more parts the more watts you will need.

If you want to go much smaller in case size look at the options from Antec, they have some really nice cases and the company is known for great power supplies included with their cases.
Just make sure the case you use can accept full sized PCI cards if you want to use that HVR-1600.

The HDhomerun is a great external option, but if your feeling you want an internal card, the HVR-2250 works great and its a half sized PCI card so you can try a smaller case with it.
