Had another go of setting up lirc (without sucess):

Used the Mythbuntu Control Centre's Infrared section to enable Lirc support. This installed the Lirc software.

The Contol Centre supplies the choice of Winfast PVR2000 and Winfast TV2000/XP, neither seem to correspond to my DTV2000DS/Y04G0051. Neither worked for my remote, so had a go at setting Lirc up manually:

Created following /etc/lirc/hardware.conf (based upon [http://linuxtv.org/wiki/index.php/Leadtek_WinFast_DTV2000DS#Remote_Control_Support_.282.29]("http://linuxtv.org/wiki/index.php/Leadtek_WinFast_DTV2000DS#Remote_Control_Support_.282.29") recommendation):
```
# /etc/lirc/hardware.conf

REMOTE="Leadtek dtv2000ds y04g0051"
REMOTE_MODULES=""
REMOTE_DRIVER="devinput"
REMOTE_DEVICE="/dev/input/event4"
REMOTE_LIRCD_CONF="/etc/lirc/lircd.conf"
REMOTE_LIRCD_ARGS=""

#Chosen IR Transmitter
TRANSMITTER="None"
TRANSMITTER_MODULES=""
TRANSMITTER_DRIVER=""
TRANSMITTER_DEVICE=""
TRANSMITTER_LIRCD_CONF=""
TRANSMITTER_LIRCD_ARGS=""

#Enable lircd

#Don't start lircmd even if there seems to be a good config file
#START_LIRCMD="false"

#Try to load appropriate kernel modules
LOAD_MODULES="true"

# Default configuration files for your hardware if any
LIRCMD_CONF=""

#Forcing noninteractive reconfiguration
#If lirc is to be reconfigured by an external application
#that doesn't have a debconf frontend available, the noninteractive
#frontend can be invoked and set to parse REMOTE and TRANSMITTER
#It will then populate all other variables without any user input
```

### Post by BishopPell on 2011-09-07
I managed to get further with ubuntu 11.04 by registering a keymap for the NEC protocol using ir-keytable. I was then able to use irw correctly. By default the af9015 has no keymap - rc_empty. So based on a blog I found I mapped to rc-leadtek-y04g0051 and suddenly things started working.

[http://forum.xbmc.org/showthread.php?t=101151](http://forum.xbmc.org/showthread.php?t=101151)

I put this in /ectc/rc.local


/etc/init.d/lirc stop 
/usr/bin/ir-keytable -c
/usr/bin/ir-keytable -p NEC -w /lib/udev/rc_keymaps/rc-leadtek-y04g0051
/etc/init.d/lirc start

Irw works now but I am still having issues with xbmc but I am hoping that's just xbmc config or keymap config.
