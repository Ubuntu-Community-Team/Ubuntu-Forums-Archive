I think if you have no schedules Myth automatically breaks everything down into 1 hour block.  So if you want to record something like that, you need to goto schedule recordings and pick the next block (8-9PM) and tell it to record.

I'm not sure if Mythtv has some kind of timed recording.  I think there is an assumption that your using it for TiVo type DVR functionality that uses schedule information.
