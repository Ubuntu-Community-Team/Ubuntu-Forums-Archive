### Post by zipperback on 2009-08-29
YouTube.com changed the way that videos were streamed not too long ago, and it caused some problems with third party applications. 

I know that Video DownloadHelper ( [https://addons.mozilla.org/en-US/firefox/addon/3006](https://addons.mozilla.org/en-US/firefox/addon/3006) )  stopped working for a little while but the developers resolved the issue and it works again, so I suspect that your application hasn't been updated to handle the new YouTube streams.

Here is the version history page that talks a little about the youtube change.

[http://www.videohelp.com/tools/Video_DownloadHelper/version-history](http://www.videohelp.com/tools/Video_DownloadHelper/version-history)

I realize it's not the same application that you are using but perhaps you might find the information helpful.
