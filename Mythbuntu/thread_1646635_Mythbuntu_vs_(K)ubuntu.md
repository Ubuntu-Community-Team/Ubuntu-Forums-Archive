### Post by williammanda on 2010-12-17
> **drvik said:**
> Hi all,

New to ubuntu and enjoying the ease of installation and setup of my mythtv system... I just wanted to confirm something.... is mythbuntu with kubuntu-desktop exactly the same as installing kubuntu with mythtv + mythtv control centre???

Yes but there are pro's and con's for each method. The Mythbuntu route will give you a slimmed down OS aimed at the HTPC vs the Kubuntu or ubuntu + mythtv will give you a full OS with HTPC functionality. Also the Mythbuntu will consume less resources. So it is up to you what preference you want.

### Post by David Grigor on 2010-12-17
I don't see any advantage to install seperately if this machine is going to be dedicated to mythtv. If your planning to use as a desktop as well there may be an advantage to using another distro.  I use mythbuntu for a dedicated machine to keep as simple and re-creatable as possible.
