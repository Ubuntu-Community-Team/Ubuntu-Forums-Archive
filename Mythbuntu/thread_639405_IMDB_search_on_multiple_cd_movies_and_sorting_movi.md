### Post by daengbo on 2007-12-13
1) You can rename the Movie (not the file) in the video manager, taking out the CD1 part. There is a place to specify if a file should be played next. Make sure that the cd2 file is chosen (it should be). Finally, set the cd2 "Browsable" option to off. Your movie should appear only once, be discoverable through IMDB, and play automatically.

2) I have all my TV programs in a TV folder with parental permissions set at 2 so they don't appear in the main list. When scrolling through movies, I use the normal view. When I want to watch a TV program, I switch to folder view and parental level 2, then view the TV folder. There are WAY too many episodes to go through any other way, regardless.
