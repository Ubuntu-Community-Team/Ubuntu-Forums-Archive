### Post by toddk111 on 2009-02-04
I'm using MythTV in Ubuntu 8.04 64bit (a fresh install due to a hard drive failure).  Whenever I start the frontend, Myth changes the theme.  I've made several attempts to change to a theme that I want but it will only stay in that theme for one or two times.  After that...every time I load Myth, it starts in a different theme.  It just keeps randomly choosing a theme.  Does anyone know how to fix this problem?

Thanks,
Todd

---

### Post by freak42 on 2009-02-04
there is a checkbox in mythfrontend
utilities-> setup -> appearance
that if checked will select a random theme on each boot.

---

### Post by toddk111 on 2009-02-04
That worked.  Simple and easy to overlook.  Thanks!
