### Post by david_f1976 on 2008-11-18
The AMD version is fine for any 64-bit processor. I have an Intel processor with AMD64 bit version running with no problem.

I mainly went down this route just to future-proof really. Whether you see a performance difference will depend on what else you're using it for other than a backend. For most everyday tasks I don't notice any perceptible performance difference.
