### Post by nelsonmuntz on 2008-06-03
I'm in the process of setting up a Mythbuntu box at present but cannot seem to get Xv working, and hence Live TV and DVD playback is awful.

I'm using an ATI Radeon HD 3450 card with the latest proprietary drivers from AMD's website. I've tried 

    aticonfig --overlay-type=Xv

as well as adding the following into the Module section in xorg.conf:

    Load   "xv"

but xvinfo continues to return 'no adaptors present'. 

Are there any known issues with the card I'm using perhaps?

Any suggestions on other things to try most appreciated!

### Post by nelsonmuntz on 2008-06-04
Thanks superm1. I wasn't aware of the log file.

Having browsed the log file, it turns out my problem is the fact I have an AVIVO based chipset and hence have to use the "TexturedVideo" option instead of "VideoOverlay". Strangely enough, I found mention of this parameter elsewhere and tried it unsuccessfully, but must have done something else wrong at the time. It's working now.

Thanks again! Now, onto my next tribulation! ;)
