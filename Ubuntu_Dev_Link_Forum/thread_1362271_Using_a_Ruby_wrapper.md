### Post by Ben Wright on 2009-12-28
In [http://github.com/tenderlove/nfc/blob/master/README.rdoc](http://github.com/tenderlove/nfc/blob/master/README.rdoc) you can find the instructions to install the wrapper. Specifically:
```
[FONT=monospace]
[/FONT]sudo port install libnfc
sudo gem install nfc

```

Thanks.
