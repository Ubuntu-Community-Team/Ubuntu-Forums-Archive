### Post by scu-ba-de-buntu on 2012-02-29
I am not aware of a portable geany, however the following will help under a limited linux user account:

If you change the config settings in your home directory, by copying and pasting all the files as suggested by [http://www.geany.org/Download/Extras#colors]("http://www.geany.org/Download/Extras#colors") for whatever editor theme you want, it will be able to highlight commands correctly *mostly*. I tired this and although it does not appear to work immediately, you can got to 'Document'->'Set Filetype', select the appropriate file-type and then highlighting will be turned on for the duration of the instance.
