### Post by SevenMachines on 2010-01-18
some of the 'hands on packaging' sessions involve downloading source to carry out some sort of packaging task on it. usually this is deliberately chosen to be quite a small package to keep those download times low. 
there can be a number of things you will need and can install before-hand (build-essentials and that kind of thing)
see [https://wiki.ubuntu.com/PackagingGuide/HandsOn](https://wiki.ubuntu.com/PackagingGuide/HandsOn)
You can also look at some of the similar [past sessions]("https://wiki.ubuntu.com/UbuntuDeveloperWeek/Previous") to see how you can be prepared so that actual downloading during a session is minimal, but the 'Hands On' link above shows what is usually needed
