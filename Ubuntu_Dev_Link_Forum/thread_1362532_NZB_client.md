### Post by Emanuele_Z on 2009-12-23
Hi all,

both as an experiment and for fun, I developed a command line version of an NZB client.
The name is **myNZB**; it is released under the GPL v3 and apart the sources you can have the precompiled i386 and amd64 packages.
Home page is:
[http://mynzb.youlink.org](http://mynzb.youlink.org)
It has been written in C++ and it should be ok.
If you want to test it, to use it or just to make any questions about the C++ source code (some paradign I've used etc etc..) please feel free to do it...
This software is useful to me... I hope it'll be useful to others as well.

Have fun! :D
