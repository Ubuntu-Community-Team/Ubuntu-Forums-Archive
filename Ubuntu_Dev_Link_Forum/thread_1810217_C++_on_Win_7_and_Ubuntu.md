### Post by Mr. Shannon on 2011-07-22
As far as I know SDL is fully cross platform.  For an object oriented library for making games consider SFML, also cross platform.

When/if you go to 3D just make sure to use OpenGL and not DirectX.  By the way, both SDL and SFML are good ways to get an OpenGL context.  NOTE: I'v only read about this stuff.
