### Post by wolfger on 2008-03-12
So many reasons, so little time:
It's free.
It can surf websites without fear of picking up viruses.
It comes pre-packaged with most of the software you'll need: no need to go out and buy other stuff.
Installing new software via Synaptic (or Adept) Package Manager is even easier than installing new software on Windows.
If you want to uninstall something completely, it's not only possible to do so, it's easy to do so.
Software updates are painless.
Can run with Vista-like features on older computers that Vista won't run on.
If you find a bug, you can actually talk to a developer about getting it fixed.
