### Post by RSLxH on 2008-03-01
Dreamlinux 3.0 RC1 is available for download.

This is a fix up release with some minor improvements, while we get tha pace again to killmost of the bugs and launch the final 3.0 at 29th of March.

Some fixes:

- Video card detection now more robust. Wehen the correct driver is not detected, the configurator assumes a vesa driver. If some card still fails to load X, please let us know.

- Fixes in some Dreamlinux Installer routines. Now correct identifying ext3 and ext2 filesystems. USER DEFINED feature added to the type of directory to be kept on installation. Routine to partitioning the entire disk started but commented out due to shortage of time to test. It will be fully operational on the official version.

- Fixed the problem with Iceweasel add-ons.

- MadWifi drivers included. For more details see andrefelipe's post about.

- Laptop-Detect routine detects notebooks and starts the laptop-mode service only when a notebook is running Dreamlinux, not Desktop computers anuymore.

- Fixes to the root user home to make it compliant to the normal user setup. So, no more green screens when using root.

- Several graphics setups and fixes.

- Addition of the Color-Folders feature.

- Gnome shutdown routine replaced by in-house script to prevent freezing.

- Other minor fixes.

To be added/fixed to the final release:

- DLI to add the capability to automatically partition, format and install to the entire hard disk.

- MkDistro Live Remaster updated.

- Addition of MDistro Easy Remaster.

- Kernel 2.6.24 series.

- Improved Network connection.

- Improvements in the videocards hardware detections and configuration.

- New Easy Install applications.

- Addition of new wifi drivers. 

- Full set of background screens by andrefelipe and ruud kuin. 

- Possible DMA bug fixes to some specific bus like jmicron and others.

Ya, and sexy too ;)
