### Post by freduardo on 2008-06-15
Hi,

Some suggestions:

If you're going to put it on a desktop system, I'd suggest using Debian testing over stable. Testing has a lot more newer packages and, despite the name, it's very stable imho.

Also, since you have a fast internet connection, you can download a netinstall or businesscard iso. This will only provide a base install from the disc and get the rest of the files you need off the mirror of your choice. (netinstalls also have a GUI).

Good luck, have fun and I hope you'll like Debian as much as I/we do :)
