### Post by odiseo77 on 2008-05-20
> **Lod said:**
> The pro of Ubuntu is the easy installation of the nvidia proprieraty (?) driver and multimedia codecs. Is this the same in Debian?

Depends on what you define easy, but I find it pretty easy. As for the codecs, all you need is the debian-multimedia repository, and as for the nvidia driver, it might be a bit trickier since lenny's repos doesn't provide it from some time now, but in general terms, all you have to do is to shut down your X server and install the driver previously downloaded from nvidia.com, run nvidia-xconfig, start your X server, and you're done.

IMO, Debian has an undeserved bad reputation of being hard, when it's not.

As for me, I installed Debian some years ago, and I haven't looked back (currently running Debian, Ubuntu and Arch).

I think it gets this because while it is likely to "work" out of the box, it is not really optimized. If you're willing to tweak things (and I am), then it's fine. Setting up powernowd in Debian is not hard, but the procedure will look very intimidating to new users ("enable sysfs? what the heck?").
