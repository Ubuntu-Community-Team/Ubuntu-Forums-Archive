### Post by TheFu on 2019-09-24
It doesn't have anything to do with remastering anything, but **sftp** is the easiest, most secure, answer I know for transferring files between Unix and anything else.  There are clients for every OS with a network stack.

In 25+ yrs as a Unix/Linux admin, I've never needed to re/master a distro. Almost any problem that includes an installation can be solved quickly using some other, more flexible, method.  That isn't meant to say there aren't some, highly specific tasks where having an installable distro couldn't be useful.
