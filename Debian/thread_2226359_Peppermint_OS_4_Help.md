### Post by Frogs Hair on 2014-05-28
The code based 13.04 and there very little info on Peppermint forums about upgrading. If it is possible I believe you would have to wait for a new version of Peppermint. Since the Ubuntu 13.04 repository is closed that leads me to think Peppermint uses different repositories and you could find out with the following command. 

  ```
grep ^ /etc/apt/sources.list /etc/apt/sources.list.d/*
```
