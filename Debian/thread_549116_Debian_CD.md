### Post by Pancetilla on 2007-09-16
> **dptxp said:**
> How does Debian feel and fare as compared to Ubuntu, especially when Ubuntu is Debian based ?

It's more or less the same experience for a desktop user like me. Packages are not up-to-date in etch, but you can always upgrade to lenny; I stay in etch because I don't care about new versions of programs myself, but it's your choice...Lenny is more or less Ubuntu 7.04, AFAIK.

Most of the fixes that work for Ubuntu work for Etch too, so you can always get help here in ubuntuforums, if you wish (I do), you may miss some tool or two, but it's just one apt-get away. The repos are great (unofficial repos too) and the community is big. 

You can change the default GNOME theme, install a few things and think you're in Ubuntu. 

I imagine if you're expert user you may notice some different things, but I could not notice any difference when doing the usual "casual" jobs (surf the web, watch some videos, do some tweaking).

### Post by deserthowler on 2007-09-16
> **dptxp said:**
> How does Debian feel and fare as compared to Ubuntu, especially when Ubuntu is Debian based ?

This is my experience with both of them.

Debian feels lighter and not so bloated.  Just feels "crisper" to me.  Debian isn't as customized as Ubuntu.  You can remove more packages than with Ubuntu and not break the install.  Debian testing is quite stable and offers improvement without having to reinstall or upgrade the distro every 6 months or backport.  Debian also seems to be more flexible.

Ubuntu has an easier to use system and has more programs available through the extra repositories. Ubuntu has an awesome community and a lot more howto and Wiki information.

YMMV:)
