### Post by liveforfunnow on 2007-03-21
I tried MEPIS 6.5 Beta7 a while ago. It is nice:

1) Nvidia driver opt-in!
this is nice. at the liveCD bootup, you can choose to use the Nvidia driver right away. this saves me time, because with Ubuntu, I have to use 6.06 in safe graphics mode, then boot to recovery, then install drivers, then update 

2) CODEC support!
very nice implementation. XviD, DivX, MP3, .avi, you name it, MEPIS can play it. no need for Automatix.

3) BERYL BY DEFAULT
self-explanatory. awesome, pretty, and stable.

just my $.02
