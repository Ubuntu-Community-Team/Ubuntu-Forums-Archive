### Post by Xian on 2023-07-16
This is a functionality that needs to be added/enabled.

You will need to install the Gnome Extension Manager
[https://flathub.org/apps/com.mattjakeman.ExtensionManager](https://flathub.org/apps/com.mattjakeman.ExtensionManager)

Then find an extension that fits your likes.
For example: [https://extensions.gnome.org/extension/4337/desktop-icons-neo/](https://extensions.gnome.org/extension/4337/desktop-icons-neo/)
