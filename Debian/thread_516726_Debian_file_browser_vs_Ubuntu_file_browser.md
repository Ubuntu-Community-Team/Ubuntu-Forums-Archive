### Post by bennyj on 2007-08-04
> **fd9_ said:**
> I don't like the Debian file browser at all, with just the file menu.  I'm much more comfortable with the way the Ubuntu file browser is set up. How can I configure my Debian file browser to act like the one in Ubuntu?

open file browser click edit>preferences  then select the behavior tab and then check ALWAYS OPEN BROWSER IN WINDOWS.
