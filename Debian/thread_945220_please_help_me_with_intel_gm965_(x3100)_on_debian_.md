The package you need to install is xserver-xorg-video-intel.  It wasn't included in the original release of etch, but was a part of the etch and a half release.  However, I would highly recommend installing Lenny.  While its the testing release as of this moment, it is going to be released as stable sometime soon(soon for debian, at least). ### Post by tommcd on 2008-11-02
If you want to upgrade to Lenny, edit your /etc/apt/sources.list and change every instance of 'etch' to 'lenny'. Then run from terminal:
```
aptitude update
aptitude full-upgrade
```
If all goes well you should then have upgraded your system to Lenny. 

Or is you would prefer, install Lenny from the Lenny beta 2 CD:
[http://www.debian.org/devel/debian-installer/](http://www.debian.org/devel/debian-installer/)
Use the 1st install CD from the i386 set. I have been running Lenny from the beta 2 install for some time now, and it is very good.
