### Post by handy on 2006-11-06
I'm running **gNewSense** on my 2nd machine & typing this from it.

2.6.15-27-386 kernel

I have a separate **/home**  partition so the install was quick & kept my settings.

The uninitiated (like me) would say it's just Dapper without ***any*** non-free code contamination.

All my hardware was found; my nVidia GeForce4 Ti 4600 has to live with the **nv** driver's, which were installed correctly by default. As per sound.

My LAN connection was automatic, as was internet, after IPv6 was switched off in Firefox, I could browse the web (nothing new there), I still had to prepend my dns server addresses in dhclient.conf. Both of those problems are due to my router/modem.

My LAN connected LJ-5 printer setup as per usual = quickly & simply using supplied Ubuntu drivers.

I don't use wireless, but I would be interested to see how it handles wireless?
