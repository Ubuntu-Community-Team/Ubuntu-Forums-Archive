### Post by dptxp on 2007-12-26
We have a new release of Sidux.

Release notes for &#917;&#961;&#969;&#962; xmas

This update release of sidux 2007-04.5 "&#917;&#961;&#969;&#962;" concentrates on additional hardware support and stability improvements for wireless and DVB devices, combined with newly available hardware support for various ATi graphics cards in interaction with the new X.org 7.3/ xrandr 1.2 based X detection.

A new feature is the combined DVD ISO for kde-full, which supports i386 and amd64 architectures and avoids the hard size limit of CD-R media while trying to retain a slick package list. Further efforts have been invested into consolidating classical and advanced USB installation methods, kernel stabilisation and the live persistency framework. Various updates, fixes and adaptions have been applied to large parts of the package list.

As a special christmas present for our non english/ german speaking users, sidux 2007-04.5 "&#917;&#961;&#969;&#962;" ships an xdelta to add initial localisations for all currently supported languages. 

Details at [http://sidux.com/Article376.html](http://sidux.com/Article376.html)
