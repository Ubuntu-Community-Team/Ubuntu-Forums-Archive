### Post by starplex on 2011-12-16
You will need a script from here [http://www.2shared.com/file/eZSpENE3/install_update_texlive.html](http://www.2shared.com/file/eZSpENE3/install_update_texlive.html) or as an attachment. Brought to you by [http://alexkrispin.wordpress.com/2011/08/16/update-a-script-to-automate-the-installation-of-texlive-2011-on-ubuntu/](http://alexkrispin.wordpress.com/2011/08/16/update-a-script-to-automate-the-installation-of-texlive-2011-on-ubuntu/)

1. Open terminal, go to the directory where you downloaded the script and make it executable with ```
chmod +x install_update_texlive.sh
```
2. Execute the script with ```
./install_update_texlive.sh
```. A window will pop up. The choices there are self explanatory.

**[COLOR="Red"]~3 GB[/COLOR] of data will be downloaded**

3. After a short while a window appear with different settings. Again, self explanatory. Choose your languages, packages. I chose to download all of it so there are no hiccups later. Click next.

4. All you have to do now is wait. For the impatient there is a progress bar.
