### Post by eddier on 2012-08-28
During a search for some themes/skins (winamp/xmms/Audacious) I was amazed at some the incredibly convoluted instructions that are flying around describing how to install the themes.

Firstly the Themes must be .wsz types. The .wal type just do not work with the Linux version of winamp (aka Audacious).If anyone knows different then please let us know.:p

TO install,start Audacious (in winamp mode) and drag the .wsz folder on top of Audacious and release--job done!
