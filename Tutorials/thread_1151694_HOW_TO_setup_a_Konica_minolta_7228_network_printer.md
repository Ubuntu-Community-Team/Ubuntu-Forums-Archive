### Post by unavenged on 2009-05-07
This is a quick guide on how to set up the Konica minolta 7228 network printer in Ubuntu.
Hope this works for you as i couldn't find any other HOW TO's 

1) Download the [HP-LaserJet_5L-ljet4.ppd]("http://www.linuxprinting.org/ppd-o-matic.cgi?driver=ljet4&printer=HP-LaserJet_5L&show=0")

2) Goto "System"->"Administration"->"Printing".

3) Click new.

4) select AppSocket/HP JetDirect and type in the host IP and click forward.

5) Select provided PPD file, then browse to file and select it and click forward.

6)Type the name you want to use like "Network Printer" and click apply and print test page
