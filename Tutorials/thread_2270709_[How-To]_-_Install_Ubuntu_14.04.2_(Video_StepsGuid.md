Below are **the steps** that I did in the installation:

1. Download the version of Ubuntu that you need from [http://www.ubuntu.com/download/](http://www.ubuntu.com/download/)

2. Boot the .iso file in the VM.

3. Select Install Ubuntu.

4. Ensure the computer has enough space and able to connect to the internet.

5. Select Erase Disk and Install Ubuntu

6. Choose your timezone

7. Choose your credentials.

8. Now it starts installing. After awhile, it will ask you to Restart. Click Restart Now

9. Finish
