### Post by BC59 on 2012-01-15
To install the newest LibreOffice version the 3.5.0 beta2 you have to do it manually, since the official PPA is not containing yet this version:

First uninstall the previous LibreOffice version:
[http://www.libreoffice.org/get-help/installation/linux/#De-Installation_of_a_Previous_Version_of_Community-Supplied_LibreOffice_for_Linux](http://www.libreoffice.org/get-help/installation/linux/#De-Installation_of_a_Previous_Version_of_Community-Supplied_LibreOffice_for_Linux)

Then download LibreOffice .deb:
[http://www.libreoffice.org/download/pre-releases/](http://www.libreoffice.org/download/pre-releases/)

And finally follow the instructions to install it from here:
[http://www.libreoffice.org/get-help/installation/linux/](http://www.libreoffice.org/get-help/installation/linux/)

And my opinion about version 3.5.0
With version 3.4.4 to open a document with Writer, was taking 23 sec. 
With version 3.5.0 for the same document takes 10 sec.
They did a great job.

Problems with the beta version:
The view of the fonts is not perfect and the icons of the Office applications disappeared, so you have to find new ones.
