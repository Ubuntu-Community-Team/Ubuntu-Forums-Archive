### Post by akarkor on 2010-04-27
I would to inform that I have just now uploaded an extension form chrome/chromium browsers. The extension is based on similar extension which I have made for CZ community. After more request from my friends, I have just now finished the ubuntuforums.org version.

What is included ?

Quick access to threads. I have implemented most of threads. The sorting is not the same as in forum, but I have respected my design.

Quick access to TOP sites with sources of apps

Access to Bisigi project , to each section

Newest wallpapers from NexusDesktop separated by categories

RSS sources from recommended blogs + Digg.com

Access to all services of chrome

Task Manager

Bookmarks

Simple Notepad

Bookmarks Shortcuts (in first run in bookmarks shortcuts you can see only your non categorized bookmarks, clicking on << customize >> you can manage look of page, colors, font and you can select folder from your bookmarks and using this folder the bookmarks shortcuts manager should create a shortcuts. Shortcuts starting with numbers following letters. Usage hold together tab key with shortcut key. Example:
tab+1
tab+d
etc..)
