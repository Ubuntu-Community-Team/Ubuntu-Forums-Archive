### Post by linuxaddix on 2011-10-18
i found the easiest way to install plymouth themes and boot manager themes and here it is:
install Instructions:
Open Synaptic Package Manager/or software center and Add repository:
ppa:ingalex/super-boot-manager
open terminal:
sudo apt-get install "super-boot-manager"
its really easy 
for plymouth installs click enable then click on themes tab find your pick and double click it 
then click on set default
then select the number of the picture
and apply changes

next go to grub/burg installer and click install
then go to boot manager and click install
reboot your computer
when boot menu comes up press f2 to select what you want your boot menu to look like 
and when u boot up youll see the picture you picked loading.
for different os you need to do another install of superboot manager in there but you only need to set the plymouth theme
enjoy
