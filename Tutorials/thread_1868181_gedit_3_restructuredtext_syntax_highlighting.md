### Post by rplantz on 2011-10-23
I found a way to get syntax highlighting for restructuredtext (docutils) when I was using gedit 2: [http://mail.gnome.org/archives/gedit-list/2011-May/msg00012.html](http://mail.gnome.org/archives/gedit-list/2011-May/msg00012.html).

This went away when I upgraded to Oneiric Ocelot, which uses gedit 3. However, I got it back when I placed the same file in
```

@bob-desktop:~/.local/share/gtksourceview-3.0/language-specs$ ls
restructuredtext.lang

```
