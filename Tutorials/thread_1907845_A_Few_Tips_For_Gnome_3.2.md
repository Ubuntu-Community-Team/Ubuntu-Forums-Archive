I'm on it for quite a while and I could share some stuff which I think worth sharing.

so.. At first I didn't like it for being too..empty..but since linux is customizable...it turned out being something simpler than anything else I've used

-> edited windows title bar clicking actions like,
right click -> minimize windows
middle click -> menu

(advanced settings: gnome-tweak-tool)

-> Disabled some effects for gnome 3..

/usr/share/gnome-shell/js/ui/windowManager.js

find these lines end edit them like this:  const WINDOW_ANIMATION_TIME = 0

                                                                       const DIM_TIME = 0

                                                                       const UNDIM_TIME = 0

-> Got Docky as dock and took it to panel mood so when i minimize or maximize windows, they go on left side of it while it keeps my most used bookmarks on right side of it. I think it's quite a  good feature for making the desktop look unique.

and with docky, I enabled hiding mode as window dodge..so it works kinda like ubuntu's left side panel.it hides itself when a window gets near it... looks better for me on bottom of the screen instead of left side of it because of the symmetry

-> gnome-shell theme is default and gtk application and window themes are zukitwo.

as i said before when these are all combined, it's simply the simplest :P
