### Post by Mazehero55 on 2007-05-23
Just finished the new site so I figured I would post it up here and see what people think. 

It's a project to create open source commercial grade RPG games for Linux. Games as good as Fable or Final Fantasy or nwn. Because I'm sick of seeing a new RPG coming out for windows and feeling envious for it. I want to create games to make Windows users drool over. 

The first game is set in Middle Eastern setting and will be complete with towns cities and it's own language that I'm slowly putting together called Porah. 
It's still in the planning stages and I think it would be a good time to get the open source community involved. And I've also been getting sick of that empty forum. lol
If anyone has ideas or would like to contribute some concept art that would be great. I want to get as much of the community involved as possible. I plan on once the development has started to have contests so anybody can add something to the project.

Hope you enjoy everything. I'm soon going to upload some concept art that I have made so that some people can get an idea of the art style that the game is going to have.

Also if you don't like the idea, and you feel you must let yourself be heard . Please flame me here and not on the main forum, as I want to keep it constructive as possible. 

Thank you


[http://proj-kilo.sourceforge.net](http://proj-kilo.sourceforge.net)
