### Post by cogadh on 2007-09-19
You don't need to use WineDoors to get Steam installed, you just need Wine. WineDoors does make it easier to install some things, but it doesn't do anything that you can't already do with normal Wine.

The problem with having a standard repository for games is licensing and Ubuntu's strict package requirements policy. However, both of those could be worked past if you could get developers to agree to participate in a Steam-like platform on Ubuntu. In concept it sounds great, but I think the logistics of it would be hard to overcome.
