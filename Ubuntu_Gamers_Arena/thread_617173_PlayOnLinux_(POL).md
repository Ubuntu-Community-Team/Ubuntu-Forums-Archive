Hello

I'm sure some of you already know about POL but we recently added an english forum and we lack .. hmm well, english people ^^

PlayOnLinux is a script written in bash with a layout written in python, and it uses Wine ( [http://www.winehq.org/](http://www.winehq.org/) ).

Well, PlayOnLinux is like an interface for Wine. It's updated frequently and the support forum is quite fast to answer if you have any problem.
So If you don't know, or don't even wanna know how to configure Wine, POL is the way. It'll do it for you.

Everyone can create scripts for games that POL doesnt support yet. We already have scripts for great games like Warcraft 3, the Jedi Knight series, World of Warcraft, Guild Wars, Prey and some Steam Games ...

Here's PlayOnLinux link if you're interested : [http://www.playonlinux.com/en/](http://www.playonlinux.com/en/)

I'm using it to play Guild Wars and it work just great. See ya

### Post by Laibcoms on 2009-02-13
> **ELD said:**
> Why have play on linux and work for linux? Seems extremely annoying if i have to load two seperate programs that do the same thing for games and other software.

Although if i understand correctly the program basicaly changes Wines settings for each game right?

Here's the explanation: [http://wiki.winehq.org/PlayOnLinux](http://wiki.winehq.org/PlayOnLinux)

**A prefix's matter**

 When you install some applications with Wine or some other Third-Tools all your file are saved in the same prefix  
[[IMG]http://www.playonlinux.com/doc/images/wine_en.jpg[/IMG]]("http://www.playonlinux.com/doc/images/wine_en.jpg") 
In PlayOnLinux we use different prefix for all application, so we can do every specified configuration for each game 
[[IMG]http://www.playonlinux.com/doc/images/playonlinux_en.jpg[/IMG]]("http://www.playonlinux.com/doc/images/playonlinux_en.jpg") 
 
**Manage Wine Version**

 You have see in wine appdb's a software with a platinum rank for an older wine version and now it's ranked as Garbage ?  
Where is the problems ? in PlayOnLinux you can assign a version of Wine for each software ! 
 
**Plugin**

 PlayOnLinux Plugin's what is it ? 
It's a plugin to add something in PlayOnLinux like "Capture" (by GNU_Raziel use it for record your game (Fraps-like)), "[TransgamingCedega]("http://wiki.winehq.org/TransgamingCedega")" (by Tinou, use it to import installed game with cedega in PlayOnLinux)), "[DosBox]("http://wiki.winehq.org/DosBox")" (by Tinou) ... and other !


----------------------
I was formulating the same thing a few days ago before I discovered PlayOnLinux.  As I was experimenting with more and more games to run under Linux via WINE (for our Internet Cafe based on Ubuntu), I came to that problem.  PlayOnLinux did it - separate prefix for each application.

So far, I am very satisfied with PlayOnLinux.
