### Post by fakie_flip on 2007-04-18

> most of those games are .run or .package so there is really no need to create a .deb
it installs just fine.
on the other hand if you put here some games that are only available in "source" or "rpm" ....

debs are better than run and package files because they are easier to manage and uninstall. I know some games installed from run files come with uninstall scripts but very few that I have seen. debs usually add a game launcher to your menu too to save you some time from doing it. It's nice to be able to see and manage all your software installed from Synaptic. It will never be overwritten if the package manager knows it's there.
