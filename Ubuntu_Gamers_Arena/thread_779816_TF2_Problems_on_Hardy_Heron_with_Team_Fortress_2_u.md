### Post by zspace on 2008-09-13
> A recent update to the Steam application added some in-game community functions that do not work properly in Wine. These functions need to be disabled in order to play TF2.

   1.
      Launch Steam and click on the File menu
   2.
      Select “In-game tab”
   3.
      Uncheck “Enable Steam Community In-Game”

Team Fortress 2 is now installed and should run normally. If you find the game crashes or performs poorly, reduce the game's Model Detail settings to “Low”. You can then increase it in increments until you get an acceptable balance of performance and quality.

from [http://gaming.gwos.org/doku.php/wine:team_fortress_2](http://gaming.gwos.org/doku.php/wine:team_fortress_2)

that might be the problem, could be something more though.
