### Post by Old_Holborn on 2008-05-13
Instructions on how to get the Linux 32bit autopackage for the new development release Oolite v1.71.2, as well as an exhaustive change log have been posted in [[COLOR="RoyalBlue"]_Oolite Bulletins Forum_[/COLOR]]("http://aegidian.org/bb/viewtopic.php?p=50275#50275").

The 64bit users may build the v1.71.2 source code (or go for the bleeding edge), following the instructions posted [[COLOR="RoyalBlue"]_here_[/COLOR]]("http://aegidian.org/bb/viewtopic.php?p=51719#51719"). This guide is also valid for the 32bit users.

NOTE: Oolite v1.71.2 is stable enough to be considered a "development release". However, maintenance is still going on, following to users' feedback and therefore cannot be included in the official repositories yet, where only "stable releases" reside.
