### Post by elmy on 2009-01-03
There is a way to get it working in wine.

Install the Visual Basic 6 runtime drivers and the Direct X 9 runtime drivers and then the game will run.

I used winetricks to assist with the install of the VB6 and DirectX9 runtime packages.

Once installed the game installs and runs in Wine with no issues at all.
Battle Of Yavin also works. 

I'm running it on a HP Compaq NC8000 laptop with Ubuntu 8.10. and it works nicely.

The enhancement Master_Syrus mods mentioned on [www.bruneras.com](www.bruneras.com) also work but because of the additional texture renderings (and wine) you need a bit more balls under the keyboard to run it than what I have with my NC8000
