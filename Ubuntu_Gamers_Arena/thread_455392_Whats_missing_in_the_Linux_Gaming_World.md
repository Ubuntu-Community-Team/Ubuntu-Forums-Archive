### Post by tretle on 2007-05-26
This thread is an offshoot of "Whats missing in the Linux Gaming World?" as that thread describes, there are los of good native games but there are some genres which seem to be overlooked. This specific thread was set up with the poll to try and get a quick synopsis of what game genres gamers would like to see more/better games developed for. 

If the genre you want is not listed in the poll feel free to add it as a comment or simply use the comment system to discuss the state of developement for certain game genres in linux.  

There are eight poll choices, you can unfortunately only choose one so if you have multiple choices just put in the one you would want to see the most. The eight choices are not specific but rather massive in terms of enterpretation. Try and visit [http://en.wikipedia.org/wiki/Video_game_genres](http://en.wikipedia.org/wiki/Video_game_genres) before making a decision to make sure it is what you think it is.
