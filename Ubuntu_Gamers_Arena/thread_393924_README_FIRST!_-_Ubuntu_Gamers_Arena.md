_What is Ubuntu Gamers Arena?_
Ubuntu Gamers Arena is a website that are only dedicated for gaming and game related subjects on Ubuntu. You'll find alot of gaming guides, game related guides, game servers and gaming news.

_Where can I find this site?_
Right here; [Ubuntu Gamers Arena](http://gaming.gwos.org/news.php)

_Why are you doing this?_
There's many reason for it, 1) One of the drawbacks on using Ubuntu/Linux is that gaming is far behind contra other OS's, we really want to make an afford to promote and help people to an easy way to find games and play them on Ubuntu/Linux. 2) We're not happy with the currents dedicated linux game sites so we decided to start our own which are talking to the Ubuntu audience. 3) We are linux gaming enthusiasts and really like to share our experience with other users. 4) We want/trying to wake the game developers to making games for linux and/or taking a serious look at Linux as a gaming platform.

_What else do you have?_
Another project we have running parallel with Ubuntu Gamers Arena is Ubuntu Game List which contains over 300 game entries, many links to diffrent game developers, links to diffrent gaming tools. The list have been often updated now in nearly 2 years. You can find it here;  [http://gaming.gwos.org/doku.php/games:start](http://gaming.gwos.org/doku.php/games:start)
It will in the near future be merge with Ubuntu Gamers Arena so they become one.

_What is this forum for?_
This forum is for feedback on what's happening on Ubuntu Gamers Arena, if you find any errors in our guides or something that is incorrect. If you have a good idea or a great game howto you want to share on Ubuntu Gamers Arena, or if you have some game news or game related news. Critiques is welcome (as long it's constructive and in a good manner), or if you just wanna say how great Ubuntu Gamers Arena is ^_^
Also if you have any gaming guide requests (native game that is).
