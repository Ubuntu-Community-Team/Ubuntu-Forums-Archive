### Post by MG&amp;TL on 2014-09-03
The LAMP stack is the more well-known of the two, and most documentation is aimed at it, rather than XAMPP. If I were you, I'd go with LAMP - the documentation [here]("https://help.ubuntu.com/community/ApacheMySQLPHP") is very good.

As for Drupal vs anything else, IMO they're all much of a muchness. Pick one, try it, see if you like it. If you're at a school though, most of my previous educational establishments have used Moodle as their website / course environment, so I'd definitely try that out.
