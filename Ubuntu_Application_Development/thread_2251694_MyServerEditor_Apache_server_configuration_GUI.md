### Post by abdulla4 on 2014-11-06
New updated version of MyServerEditor has been uploaded to sourceforge

Apache server configuration  is more easy now with My server Editor.

 Prerequisites
 MyServerEditor should run with no problem either in 32 bit or 64 bit os
    but if you run a 64 bit os you should have java 32 bit installed and this includes Windows and Linux.
    Windows7 or above.
    java 8 or above and this includes Windows and Linux.
    Not tested on AMD processors nor mac os.


Setting MyserverEditor database
Myservereditor data base is the file responsible of compatibility with
Apache server version.

By default Myservereditor comes with a database that is compatible with Apache 2.4 version.
You may check Your Apache server version and download the right version  from files. Myservereditor data base has the extension of db.
To upload a data base to Myservereditor go to Advanced settings
> Upload data base




                 **Features**
