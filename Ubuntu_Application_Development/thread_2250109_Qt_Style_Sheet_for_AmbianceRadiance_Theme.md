### Post by maxter on 2014-11-27
install Ambiance/Radiance theme
install qt4-qtconfig
in qt4-qtconfig set Gui Style to GTK+

Voillà.. all you qt apps will use the theme of your desktop.
this is obviously for all the DE that use gtk (lxde, cinnamon, gnome, xfce ecc.)
