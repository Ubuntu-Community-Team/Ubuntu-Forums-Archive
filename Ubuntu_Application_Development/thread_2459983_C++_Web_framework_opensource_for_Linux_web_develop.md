### Post by TheFu on 2021-03-30
Last time I used C++ for web stuff was in the 1990s.

Since then, it has all been perl, ruby, python.  Programmer productivity is so much higher with scripted languages when compared to compiled languages - even JSP died out ... er ... mostly.  If only we could get JBOSS to die, die, die.

If I were creating a custom web server, then I'd definitely use C++ or Rust or Go (compiled), but we just use nginx and when it is time to scale, we'll use a few plackup/starman controllers to spin up or down more webapps as needed.
