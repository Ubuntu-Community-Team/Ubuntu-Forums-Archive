[FONT=arial][SIZE=4]I am amazed by the tremendous resources offered by the Linux communities.[/SIZE][/FONT]

[FONT=arial][SIZE=4]However,due to the memory burden of my occupation, I can't memorize the commands and options of Linux system.[/SIZE][/FONT]

[FONT=arial][SIZE=4]
[/SIZE][/FONT][FONT=arial][SIZE=4]Plan A: Make and share Good Mindmaps in this forum

[FONT=arial][SIZE=4]             There are so many veteran Linux users in this forum.[/SIZE][/FONT]
[FONT=arial][SIZE=4]             Sharing your versions of linux command mindmaps will help the users learn faster.
             Besides, mindmaps may help the application developers organize their project more efficiently and have better command&option design.
