### Post by sudodus on 2013-01-30
Welcome to the Ubuntu Forums :-)

I'm not sure what you want, but I have two guesses.

1. The guest user id (or guest account) in Ubuntu. It is there by default, when you install Ubuntu. Guest is very limited, it can write onto the internal drive, but it will be removed after log out. Guest is not allowed to read the private files of the ordinary user accounts.

2. A kiosk operating system. You can download Webconverger (which is free for private use, but there is an introductory advertisement). This system only allows running the web browser, and all data are reset at logout/reboot/poweroff.
