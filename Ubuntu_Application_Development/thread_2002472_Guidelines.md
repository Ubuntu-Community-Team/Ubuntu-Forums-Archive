### Post by mhall119 on 2012-06-12
Welcome to the Ubuntu Application Developer Forum. Here you will find other developers who are targeting Ubuntu as a platform for their applications.  This forum is about writing applications for Ubuntu, not for developing Ubuntu itself.  Please read the following guidelines to help keep the discussions here helpful and productive.

[LIST]
[*]While you are encouraged to discuss the development and deployment of your application, this forum should not be used for advertising

[*]You can use this forum to seek advice on design,development, promotion and deployment of your application

[*]Questions about how to do something technical or what tools to use should be asked on AskUbuntu.com ([http://www.askubuntu.com/questions/ask?tags=application-development](http://www.askubuntu.com/questions/ask?tags=application-development))

[*]For general programming help and advice, we encourage you to use the “Development & Programming” forums ([http://ubuntuforums.org/forumdisplay.php?f=310](http://ubuntuforums.org/forumdisplay.php?f=310))

[*]For general help and discussions about using Ubuntu, please use the “General Help” forums ([http://ubuntuforums.org/forumdisplay.php?f=331](http://ubuntuforums.org/forumdisplay.php?f=331))

[*]When asking about submitting your app to the Software Center, please specific if you application is open source or proprietary, as they have different paths for review and deployment

[*]Common Desktop APIs can be found here: [http://developer.ubuntu.com/resources/platform/api/](http://developer.ubuntu.com/resources/platform/api/)

[*]Information about Unity and how to integrate your application can be found here: [http://developer.ubuntu.com/resources/platform/unity/](http://developer.ubuntu.com/resources/platform/unity/)

[*]Information about packaging your application can be found here: [http://developer.ubuntu.com/publish/my-apps-packages/](http://developer.ubuntu.com/publish/my-apps-packages/)

[*]See [http://developer.ubuntu.com/community/](http://developer.ubuntu.com/community/) for more ways to engage with Ubuntu application developers like you
[/LIST]

App Developer Forum moderators are: Michael Hall (mhal119) and David Planella (dpm)
