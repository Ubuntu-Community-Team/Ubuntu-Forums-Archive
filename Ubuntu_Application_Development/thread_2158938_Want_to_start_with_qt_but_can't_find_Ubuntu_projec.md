Did you open Qt Creator by searching for "Ubuntu SDK" in the dash?  There are 2 QtCreator app launchers, once for vanilla QtCreator, the other is called "Ubuntu SDK" and features the QtCreator logo with a smaller Ubuntu circle-of-friends emblem on it.  You need to launch it using that "Ubuntu SDK" entry for it to load the Ubuntu SDK plugins.
