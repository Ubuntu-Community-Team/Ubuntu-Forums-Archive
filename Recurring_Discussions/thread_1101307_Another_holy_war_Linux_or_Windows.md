### Post by Johnsie on 2009-03-20
Get virtualbox for Ubuntu and install Windows in that or dual boot Windows and Linux. There's no point locking yourself out of the vast collection of good software that is available for Windows, especially if you have a valuable Windows license. If you delete Windows then you are simply throwing money away...  So dual boot or use a virtual machine. Don't shoot yourself in the foot by restricting yourself to just the Linux stuff when your computer is capable of so much more.

Certainly doing the VirtualBox thing would make you feel less tied. I've been using Ubuntu for about 4 years now and most of that time was spent using only Ubuntu, but then I realised I was missing out on a lot of good stuff. There are some great applications that simply aren't available for Linux and vice-versa.
