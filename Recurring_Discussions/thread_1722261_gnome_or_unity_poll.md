### Post by ElSlunko on 2011-04-05
Used 10.10 on my desktop in tandem with 11.04 on my laptop, ended up switching my desktop to 11.04 and use the unity interface.

My desktop experience is 90% the same. Instead of using super + space to bring up synapse, I just user super for Dash.

Instead of Docky, I just use the Unity Dock.

All my recent files, gwibber updates, program shortcuts are a few keystrokes away.

A few bugs + the ability to change the orientation/look of the dock and we're golden. Oh and make dash look better, I mean it has room for improvement.

I used global menu already so that's not new. I thought the window controls while maximised would be awkward but they're not (Except when you want to close a maximised window behind an on-maximised window).

Compiz is full of win, feels so smooth. The default animations are nice. Dash is fast. Really feels polished.

Not sure if having the window title auto-hide thing was really necessary.

Hmmm what else. Guess that's it. There's very little difference between my 10.10 workflow and 11.04 workflow. Unity simply replaced a bunch of the apps I used and installed anyways.

### Post by gnomeuser on 2011-04-10
> **3Miro said:**
> Poll makes no sense. Unity is Gnome, in 11.04 it is the exact same Gnome as in 10.10.

Unity is nothing more or less than a top panel, dock and new menu for Gnome.

And a different set of defaults, a global menu implementation, Application Indicators, the lack of proper PackageKit support and naturally a different notification implementation.

Yes Unity is GNOME (in the sense that it uses parts of the GNOME platform) but it diverges heavily even from core platform choices. It is GNOME in much the same way XFce or MeeGo (netbook) is GNOME.
