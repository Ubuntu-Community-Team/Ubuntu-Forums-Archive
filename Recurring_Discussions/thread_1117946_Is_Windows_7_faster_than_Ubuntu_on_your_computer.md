### Post by NightwishFan on 2009-04-08
Now regarding speed. Is this log in speed? Program launching? Copying files? CPU speed? Without any real hardware disparity device performance should really be similar.

As for software, most of Windows "speed" is mostly an illusion. For example, Windows Vista has a fast log-in. It all relies on active preloading, which causes the disk to be used more often. This is not such a bad thing, but it means that when you get a desktop, it is not really fully logged on, and it is working to log on before you even type your password.

Xp did this a lot more, and with a lot less regard for power usage. It also made the hard disk noisy, especially in the old days before devices managed themselves well.

I would have to say, the real definition of speed is how the software is written to use multi-core enhancements, and advanced paging. If Linux software is not written to take advantage of this, then software on Windows that does will become faster.
