### Post by Merk42 on 2009-11-12
If you're solely running Linux, no need for antivirus

As far as 'how long until one is invented' that's hard to say.

However, Linux has a lot of precautions set up before a program makes changes to your computer.  So firstly a virus couldn't install itself (you have to allow it), second, a virus couldn't make changes to your computer (you have to allow it)
