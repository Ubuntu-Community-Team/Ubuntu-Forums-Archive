### Post by dniMretsaM on 2011-10-09
KDE. Super customizable. It can be set up to be like Windows (i.e. if your church is switching from Windows, this would be a great option), or OSX (again, great for switching), or completely different from every other DE out there. However, if the computers are relatively low-end, you might want to look toward a lighter desktop (LXDE comes to mind). I, run it on a 9-ish year old computer and it runs nicely, though. So it's not too bad.

EDIT: I see in your poll that you have GNOME 3 and GNOME Classic together. Just so you know, Classic is an Ubuntu 11.04 only thing. It's still GNOME 2, not GNOME 3. You might be thinking of Gnome-Fallback (which is GNOME 3 programs/libraries made to look like GNOME 2). Also, you might want to look at [MATE Desktop Environment](https://github.com/Perberos/Mate-Desktop-Environment) which is a fork of GNOME 2 if you decide that a GNOME 2-style DE would be best.

Because I often need to switch between multiple windows, Gnome 2 works pretty well for me, but Unity does not.
