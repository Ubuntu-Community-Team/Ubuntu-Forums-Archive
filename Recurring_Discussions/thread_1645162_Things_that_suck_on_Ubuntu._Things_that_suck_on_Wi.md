### Post by hantms on 2010-12-14
..and the final verdict for business.


**Ubuntu**

* Firefox is broken. The address bar history and search box stop working numerous times per day. The address bar just doesn't show history, and an evil little ghost in the Google search box highlights all characters every second, so the next character you type erases everything.  This bug has existed for years.  Compiz probably involved somehow.

* Video playback is jittery.  Especially when Compiz is not turned off.

* Skype doesn't release new features for Linux.  Result: Cannot use Ubuntu during conference calls.

* USB audio devices (head sets, etc) need to be manually selected in the Sound panel before they start working.  In Windows they start working just by themselves, and Skype also remembers them.  (This is already better than in the last Ubuntu version, where I had to run the command line tool alsamixer every single time. Progress.)

* Extending your desktop on a second monitor or projector is ridiculous. In Windows you hit one key.  In Ubuntu with the NVidia panel it's too many clicks to count.  (System->Preferences->Monitors->Yes I want to use the ($*&##@^ NVidia panel->Detect Displays->Select the second display->Configure->Twinview->Ok->Apply->Yes I want to keep these settings-> Close.   Twice a day.

* I get system messages on a black terminal screen when starting up and shutting down.  I get the text CLI login before the graphic shell starts. 

* Exchange support in Evolution: Not happening. At. All.   All kinds of mobile devices do manage this.

* For company e-mail I use Outlook through a nice little XP VM through VirtualBox.  This actually works great. Except that VirtualBox doesn't want to update to a new version, not sure why. Repository system not as easy as it appears. 

* Every single day: *"Apt Authentication issue -- Problem during package list update. The package list update failed with a authentication failure. This usually happens behind a network proxy server."*  I am not ever behind a proxy server.

* Audio: Not as loud as in Windows on the same computer.  (Lenovo T410i). My older Lenovo T61 also had this.

* Video editing: Nothing comes even close to the one included (well, free separate download of Windows Essentials) with Windows. 

* iPod support is iffy. Just about works for MP3s, but transferring video and pictures doesn't happen, or at least I haven't figured it out yet.

* Ubuntu One has a weird interface. You never know what it is doing. And the website doesn't allow uploading multiple files.

* All kinds of chat and social networking BS built into the same toolbar thingie used for Ubuntu One. I think the Twitter one even stopped working.  Are people actuall using these for Facebook, Twitter and chat? 

**Windows 7**

* Boots up quickly to login screen. Then after logging in I sit there for a full 3 minutes looking at the Welcome message.  Weird. (Happens since joining the company domain.)

* Need to actively fight to keep Lenovo bloatware off it. Incredible load of crap. 

* Uses 2Gb of memory to just boot up and show the desktop. Which actually is okay, the thing has 6 Gb or so, memory is cheap and my company pays for it.

**Mac**

* No center mouse button.  Screw that. I am NOT using something where I cannot open a link in a new browser tab with one mouse click. 

* Terrible usability in general. Wrist hurts just using it for 2 hours or so. Forced to use it in the Teleconference Room because boss thinks it's classy.  It's not. It's a P.O.S. with a confused user interface.


**Verdict**

There you have it:  All operating systems suck.  Whenever I get totally fed up with one I switch to another, until that one sends me over the edge.  A vicious cycle of doom.

Still there should be a winner.  And the winner is...:  That incredibly lean, mean, easy to use, no frills Windows XP VM!  Long may it live. Only drawback is logging out and logging in as Admin to install something. But if you do that it's safe as houses. And it uses just 70 MB for the OS after booting up, after turning unneeded services off. It's brilliant.

### Post by Hur Dur on 2010-12-31
Three complaints about Ubuntu:

1.) It is heavy and bloated. For me, if GNU/Linux uses over 192mb on default settings idle, then it is way too heavy. I should note that I am more of a minimalist, and love low-resource OS. As for bloat, it simply has too much stuff on it. In my opinion, it should come with ALSA, OSS, X, GNOME, a music player, a video player, a text editor, and a web browser, then let the user install what they want after that. I tried installing Ubuntu on a 10GB partition. After installing Fluxbox, I had about 600mb of space left. Whereas Archbang Linux on the same 10GB partition left me with about 8GB of space, and everything I need for everyday use.

2.) User friendliness. I know that Ubuntu is supposed to be user friendly, and open Windows users to the world of Linux, but there is a point when it is too user friendly. If say a user wanted to move from Ubuntu to TinyCore or Arch, they would most likely have no idea what they are doing, unless they did some research beforehand.

3.) It's kind of buggy when out of default settings. I tried had Fluxbox on it, no desktop environment. It bugged out to where right click just brought up the create new option. There was no exit option, or shutdown option. Couldn't access the terminal, so I had to manually shut it down. Minor, but never had this problem in other distros.

Three complaints about Windows:

1.) Inability to decide what desktop environment or window manager you want. You have a weak computer, and want to run Windows Vista/7? Too bad, you're stuck with slow speed. There is no IceWM. There is no JWM. There is no Fluxbox. There is very little in what you can do in terms of optimizing Windows.

2.) Slowdown. I can keep Linux up for weeks at a time without slowdown. Windows stays up for 2 days and it runs at half the speed it did when I first booted it.

3.) Security. Windows has UAC, but asking you if you want to run a program is not security, it's just a nuisance, and most users turn it off. Linux, on the other hand, has hierarchy.
