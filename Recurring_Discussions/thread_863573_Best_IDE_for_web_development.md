### Post by themusicwave on 2008-07-18
I've never had any success getting clean code out of a WYSIWYG editor.  I think by definition they are for people who don't care about the code.

Generally, if you want clean code you have to write it yourself.  This is partly true because what you define as "clean" is different from what the people who made the editor define as clean.  

I would recommend Blue fish.  It isn't a WYSIWYG editor, but it does have tools to generate code for things like forms for you.  

It will also do syntax high lighting to make your life less painful.n
