### Post by toasty_ghosty on 2009-01-21
Hello

I'll try to answer this group of questions the best I can. First and foremost I have used Windows (who hasn't), owned a Mac, and use just Linux now for almost 2 years now.

> So, I am writing that post just to ask THE COMMUNITY for suggestion...
which is the best (stable) option then?

1- just Ubuntu (I can accept it as I need some of my Windows applications),

2- Ubuntu+Windows (I am usind patitions on my Dell, so both C:Ubuntu and C;Windows see my files on D:, although I have to show the path for Ubuntu each time... when I want to open for example an .doc file from Openoffise I get the answer that there is no such a file, then just showing the path it works..)

3- Just Mac (Apple aplications)

4- Mac +Ubuntu

Regarding 1, Ubuntu is quite stable depending on what all you have installed (beta software etc). When comparing it to my Mac experience, I feel that Ubuntu is actually more stable. I was having issues with stability when using certain application (Photoshop).

2. I am a gamer by heart and therefor, ran a duel boot machine for a while. Only used Windows for gaming. I never really had any issues with this. It might be a little more time consuming when installing. But once again, I never had any issues.

3. Just Mac, well, I use only Ubuntu for a reason. I found I did not like Apple software that well, or just that nearly all software I was using (Firefox, Thunderbird) was available in Linux. Also, the price was never nice. Always paying quite a bit more. The worst part was there were issues when trying to run software that had been ported over to Mac. I bought Age of Empires from Apple. Never ran. And WoW actually runs better in Wine then it did on the Mac.

4. As far as you can tell, I am pro-Linux. However, if you like Macs and they seem to fit you and fulfill you needs then by all means use it, but through my experiences, Ubuntu seems to fit the bill perfectly.

### Post by Brightbelt on 2009-01-21
Hi,
  I own an intel iMac and have Ubuntu installed with a dual-boot. Windows is almost completely phased out of my digital life, Hallelujah. But I'll also give you a straight answer here and it won't be popular here but here goes:

1-If you go with only Ubuntu, you are cutting your digital life's options off right at the start. Sure, there are things which you do now and Ubuntu may bit the bill presently, but a computer is a lot about growth -- having room for expansion as new technologies and as new software programs emerge. 
  Okay Yes,... with Wine or Crossover, Linux gives you some ways of having your applications, but in my experience, you ALMOST ALWAYS settle for older versions of software - in some cases, REALLY old versions. And the newer the technology (take onCD printing, for example), the more hoops you have to jump through when you use Linux. 
    So if there are new things which emerge down the road which spark your interest, you can only look on from a distance if you are using only Ubuntu. Sure, Linux will get there too,...just count on arriving 1 or 2 years later down the road, 
   And I've said nothing about configuring newer hardware yet.....

2- Macs are expensive, but remember what you're getting in terms of software...programs iMovie and Garageband would easily add $75-200 extra each if you had to buy them separately. You do get a more seamless union between software and hardware with Apple. 

3 - I dual-boot with Ubuntu on my Mac for fun, period. But my case is special in a way. I do music professionally, so I need quite a few pro applications to run flawlessly every day. These include Disk Label, Pro Logic 8,  Flash CS3 and Dreamweaver CS3. Just for one example, my flash site file won't even open in the version which my Crossover supports (Studio MX 2004) because that version is too old.  

4- In my experience, Ubuntu is stable, but it can break like any other system, depending upon what you install on it. 

5 - Ironically, after all this, I still enjoy using Ubuntu and I work with Crossover at times as though I'm running scientific experiments. I've come close to getting my whole Linux system designed so that it would actually meet my present needs.....close, that is.

Good Luck !!

### Post by MikeTheC on 2009-01-24
Apple is a company with a leader and a specific focus.

Linux is anything but.

Linux has many technological advantages, but it depends entirely upon what you need advantages in. Frankly, when it comes to the "digital lifestyle", let alone doing professional graphics and media work, hands-down Mac OS X is better suited.

What I live for -- and what I suspect many others do as well -- is the future fulfillment of the present potential for parity and then superiority of native F/OSS Linux apps with their commercial counterparts. Inkscape, Gimp, Scribus, Banshee/Amarock/Rhythmbox, F-Spot and BlueFish are in no way on par with, respectively, Illustrator, Photoshop, InDesign/QuarkXPress, iTunes, iPhoto and Dreamweaver. And, as has been said here many times already is that there is nothing like iLife for Linux (or, really, for Windows either).

So, again, what's important to you and what are your priorities?
