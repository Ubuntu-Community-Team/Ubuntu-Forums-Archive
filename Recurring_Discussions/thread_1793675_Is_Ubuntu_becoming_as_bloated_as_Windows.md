### Post by cariboo on 2011-06-29
> **brawnypandora0 said:**
> Bloat contributes to system slowdowns.

Yes it does, but what you are calling bloat has nothing to do with the way your system runs. In a Linux a program that isn't running doesn't take any resources. What makes Ubuntu run slow is all the added enhancements that make it easy for anyone to setup and use.

To get a true idea of what system resources are being used, open a terminal and type:

```
top
```

System monitor adds it's own overhead when it is running, so you don't get a true picture of what is going on in your system.
