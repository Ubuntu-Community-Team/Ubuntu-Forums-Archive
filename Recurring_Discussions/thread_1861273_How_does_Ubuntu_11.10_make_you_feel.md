### Post by MarjaE on 2011-10-15
Ubuntu 11.10 makes me angry because the basic choices are Gnome 3, and Unity, and both use features I'm trying to avoid. So right now I'm left either sticking with 11.04, which I've tweaked somewhat, or going back to 10.10 [backing up right now!], or switching to Xubuntu, Kubuntu, or another Linux entirely.

I used to like Ubuntu.

I used to feel like people respected other users and other users' needs. I don't feel like that any more. For the past few months, some users have been insulting other users over Unity, Gnome 3, and the like. And that's not good, and that's not something I think a reporting system can really deal with. We need the basic respect that different users have different needs from the desktop environment, and that these needs are legitimate. Now from my perspective, it looks like there are far more insults from Unity users toward Gnome 2 users, and people insult us, and people rub it in with comments about how this is the way Ubuntu is going, the decisions have been made, our voices don't count, and so on.

It's really a very hostile forum these days.
