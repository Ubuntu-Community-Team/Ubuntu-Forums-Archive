### Post by Son of William on 2009-04-07
You can try KDE through a Kubuntu live disk just like you can try Gnome on Ubuntu's live disk. 

An easier way to try both, however, is with virtual machines.  Sun's Virtualbox is my preferred vm application.  Just install Kubuntu on one, Ubuntu on another, another flavor of Linux on a third, etc.  

If you have the RAM and hard drive space (and they are cheap now so why not) you can run the vms at the same time and compare the different desktop environments side by side.
