### Post by McRat on 2010-07-02
When most O/S's install, they look at the hardware and install accordingly.

So normally a fresh install is less problematic.  Not sure if Ubuntu is different, but I have always done a fresh install.

I had wireless issues with 64-bit that were fixed by using 32-bit.  Even with Windows, 32-bit has wider support.

64-bit is the future, and it's here today, but if you are looking for the most trouble-free installs, 32-bit is perhaps the best choice.
