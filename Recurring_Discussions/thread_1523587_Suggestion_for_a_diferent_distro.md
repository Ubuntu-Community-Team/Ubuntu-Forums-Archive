### Post by NightwishFan on 2010-07-03
I suggest the following (any or all)

Fedora: Innovative and uses bleeding edge technology and security features. If I did not like Ubuntu's release schedule (LTS etc) I would be using it.

Debian: Universal OS. It will run on anything with anything. If you like the stock upstream stuff it is great too.

OpenSUSE: Best graphical configuration around, and nice innovation with the desktop experience. Arguably the best KDE.
