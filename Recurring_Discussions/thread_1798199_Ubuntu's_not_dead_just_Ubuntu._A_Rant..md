### Post by Madspyman on 2011-07-06
While Canonical may own the rights to the name Ubuntu, it's becoming increasingly clear that Ubuntu is less like Ubuntu than many of its derivatives. 

Other Ubuntu based distro's put a proper spin on it that feel like what the future of Ubuntu could have been if Mark Shuttleworth hadn't stepped down and shifted his focus toward product design.

His obsession with unnecessary design change has gotten in the way of him doing anything truly innovative. Nothing in Unity didn't come from Gnome Shell or OS X. Streamlining something doesn't have to mean a complete overhaul, especially if it's an unoriginal one. 

Now I don't use Mint, but I have used it and it seemed pretty streamlined while keeping the things that made Ubuntu a hit originally in focus. It puts the button on the UI without letting cluttered wiz bangy effects get in the way of the functionality. Where as Unity is starting to make Ubuntu look like a bad Michael Bay film. Is an overlay scrollbar even a feature?

Don't get me wrong I still use Ubuntu, just not Unity. I love the Ubuntu repositories, and Ubuntu is great for building a custom experience from a minimal install CD. That's the only reason I'm still using it, and I recommend customizing your own version of Ubuntu to anyone. So I guess sometimes an overhaul can be the answer. 

Running the software you need is more important than a graphics intensive wiz bang interface anyway. Flashy OS design needs to start taking a back seat, while making the UI functionality the main concern not branding.                                      

I honestly wish they'd retire the name Ubuntu and Call it Unity from here on out, Ubuntu has a reputation it's current incarnation can't live up to. 

It's hard recommend Ubuntu to anyone anymore when Ubuntu doesn't even remember what Ubuntu is. Although it seems they've inspired others to pick up where they left off, and that seems a lot more like Ubuntu to me.

What are some of the Ubuntu respin do you feel is closer to Ubuntu than Ubuntu itself?
