### Post by ubuntu27 on 2008-06-17
> **powerpleb said:**
> What does GNOME stand for, if anything?

From the Wikipedia:

The name “GNOME” was proposed as an acronym of **GNU Network Object Model Environmen**t by Elliot Lee, one of the authors of ORBit and the Object Activation Framework.[citation needed] It refers to GNOME’s original intention of creating a distributed object framework similar to Microsoft’s OLE.[6] This no longer reflects the core vision of the GNOME project, and the full expansion of the name is now considered obsolete. As such, some members of the project advocate dropping the acronym and re-naming “GNOME” to “Gnome”
