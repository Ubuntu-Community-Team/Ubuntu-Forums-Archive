### Post by Sublime Porte on 2009-08-08
> Is Python as easy as Visual Basic?

Depends on what you really consider VB to be. If you mean the language itself, then yes, I think Python is much easier, more logical and teaches better programming skills. If you mean the VB-style of programming, with a point-click GUI creator, that you just fill in a few lines of code to "make it all work", then no.

Although this doesn't necessarily provide the most thorough comparison between languages, you can compare their "hello world" implementations. Microsoft provides this listing for Visual Basic:

```
Imports System
Public Module modmain
   Sub Main()
     Console.WriteLine ("Hello World using Visual Basic!")
   End Sub
End Module
```

And this is the Python implementation:

```
print "Hello World using Python!"
```
