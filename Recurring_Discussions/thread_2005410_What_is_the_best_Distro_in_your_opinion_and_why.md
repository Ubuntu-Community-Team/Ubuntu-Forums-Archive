### Post by yeehi on 2012-06-17
I vote for [Trisquel]("http://trisquel.info/en").  

The current version is 5.5 Brigantia.

I like it because, unlike Ubuntu, it is a fully free distro. It is actually endorsed by the Free Software Foundation. (It doesn't get any money for this!) 

It is a Ubuntu based distribution, using the Linux-libre kernel. Being a Ubuntu based means it benefits from all the developers who work on Debian / Ubuntu. 

It installs like a dream and has great people in the Trisquel support community. 

Try it!
