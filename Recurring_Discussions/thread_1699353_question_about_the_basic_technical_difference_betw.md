### Post by JRV on 2011-03-03
Linux does not try to be a one size fits all operating system.

Instead of one big system that tries to do everything Linux is modular.

There are many different desktop environments you can use:

Ubuntu uses gnome.
Xubuntu uses xfce.
Lubuntu uses lxde.
Kubuntu uses kde.
and there are more jwm and icewm come to mind.

And you have a choice of file managers:

Nautilus is used by Ubuntu, there is also thunar, and pcmanfm.

In Linux the graphic user environment is optional.  Everything can be done from the command line.  The graphic programs are front ends for command line programs.

The graphic sub-system is called xorg and it is setup as a server/client so it is a true multi user system.

I hope this helps explain it.
