### Post by CharmyBee on 2009-07-30
WINDOWS programs.

CDBurnerXP Pro, IMGBurn and Infrarecorder are my favorite CD image burning/creating software.
ISOBuster is a gem for ripping off of trouble discs.
CDex is a great audio ripper too.

WinImage, although shareware, I find very essential to manipulate disk images.

Media Player Classic for WAV and MIDI files. Only let VLC associate with video files with pesky modern codecs (avi, wmv, flv, mp4, ogv, etc). ffdshow tryouts for codecs on anything but VLC.

Mozilla-based browser + Noscript is a must. You can also use a firewall. Comodo Personal Firewall is a good free one.

Also, adware/malware removal tools are essential, should you be in the event encounter them (and if you aren't careful and cautious, you will). Spybot Search & Destroy is a must have, as with Hijack This and MalwareBytes Malware scanner. 

There will be bloating temporary files too, so don't forget CCleaner. Piriform's defraggler tool is useful also. NTFS is not journaled like ext3, so you will eventually maintain the drive and defrag it now and then.

5. DVD burning - CDBurner XP, IMGburn or Nero.

11. MS Office 2007
