### Post by houseworkshy on 2012-10-11
Mint is the easiest for a newbie to linux who is used to Windows, looks familliar and most of the setup has been done.  Probably not the debian version as it rolls so they'll have to do things to it sometimes.  

If they don't mind doing a *few *things at the begining for low maintainence Mepris is good as it is based on debian stable and only uses the last KDE they think is bug free.  Configuring KDE is easy and it can be made to look and behave like anything, gnome2 or windows if they want.  Other gui's are in the repros anyway. The duel password system is differant to sudo but windows has something similar so it won't be a new concept.  If that's not too much, once set up that's usually it,  bar updating, hardly ever *need* to get techy again.  Not bleeding edge but works, huge repository, and changes only very very slowly.  Still uses synaptic which would be new to a windows user but fairly straighforward, they'll have to learn some sort of package management system whatever linux they use, synaptic is many peoples favorite gui installer anyway, me included.
Almost stable debian on a plate, the setting up has been done.

There are quite a few which are good for first timers, often something like debian or ubuntu based but with bits done already.  The most important thing for newbies, after the first wow, is probably stability, a system which just works.  Having to dive into the command line to mend something in week one is a real turn-off.   It also helps if the song remains the same for a while,  meaning whatever is learned continues to be valid for a long time.  One wants to drive around a bit before being forced into engineering. 

If they happen to like any of the Ubuntu GUI's after testing live,  the Ubuntu lts or something based on it,  is still a good choice after it's settled in ( .1 or later ) , five years is better than three.  Steer them away from the intermediate one's unless they are embrionic geeks though, six months is never enough to get them issue free enough for application users who just want a working machine and dramatic changes are the bill of fare.
