### Post by jfed on 2011-09-02
The fact that you mention games makes me curious. Do you mean like Steam games? Team Fortress 2, Portal, Call of Duty, Assassin's Creed, and other head-liners? 

Because A, you're not going to be able to run most of those easily with your hardware, and B, it's not easy to get steam games working on Linux easily.

You could play things like Quake, and there are a few rinky-dink-waste-an-hour games in the repositories...but otherwise you need Windows to game.

Sucks, I know.

For the other things you've listed, I'd definitely start with something like Ubuntu, or perhaps Linux Mint. Each distro has its merits, whose to say you have to commit to one? Try them all out!

Also, you can run Windows and Linux/Ubuntu side by side, and choose which one you want to use at startup. Look into doing something like that.
