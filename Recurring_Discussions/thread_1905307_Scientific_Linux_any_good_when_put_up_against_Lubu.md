### Post by snowpine on 2012-01-06
Scientific Linux is a great choice if you're looking for older, stable, well-tested software. It is roughly comparable to Ubuntu 10.04 LTS or Debian Stable in terms of age-of-packages. I run it on my work computer. :)

If you are looking for newer software in an RPM-based distro then Fedora is a better choice.

I think there is a Live CD available if you'd like to take it for a test drive...

Lightweight distros like Crunchbang especially would though. But full-blown distros would be fast as heck too, and UNIX OSes would be the same way.
 
Crunchbang and Scientific or Fedora would probably be a better way to start off than PC-BSD or OpenIndiana.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Good reasons for Crunchbang.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
You're trying to restore an old PC.
You're trying to make a new one snappier, faster, and more responsive than with the Windows 7 that normally comes installed on it if bought at say, Office Depot or something.
You're building a new PC yourself and want the quickest possible OS that uses the least amount of resources possible with a fully functional desktop. Puppy's also good for that, just not as good as Crunchbang.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
For Scientific.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
You want to run Red Hat, but can't afford and/or don't want to pay for it.
You want an OS with stable, well-tested software, although it would be slightly older.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
For Fedora.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
You want to run a Red Hat-based OS that is cutting-edge with the newest software and features.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 
UNIX is better if you're really hardcore.
