### Post by chessnerd on 2010-05-10
The Unix build is still having some issues (I'm not so sure they should have called it a "Beta" but they did) while Opera 10.53 on Windows 7 has offered a fast, reliable browsing experience for me. I've been using Opera for well over a year now and as long as its around I'm not going back. 

Is 10.5 the fastest browser in the world? Possibly. There are dozens of benchmarks and Opera 10.5 wins in most of them, but loses in others (typically coming in second to Chrome). 

However, once you learn how to use Opera, it really is much faster than any browser (even Chrome and Chromium). Here's what I mean:

Opera's mouse gestures let me quickly move about on the Internet. Back, forward, new tab, close tab and the like are much faster as gestures than as buttons now that I know how to use them. 

Opera Sync makes it SO much easier for me to deal with using multiple computers and makes setup on another machine instantaneous as my bookmarks, Speed Dial, history, and searches are added and synchronized automatically. 

Speed Dial is great because it lets me easily go to one of my top web sites when I open a new tab, and it creates keyboard shortcuts for all of the Speed Dial pages.

Opera Turbo helps speed things up over at my aunt's because she uses a Verizon 3G hotspot for her Internet. And, for those who have dial-up, Turbo is a must.

Opera's Downloads page is very nice. It gives just the right amount of information, lets you easily manage multiple downloads, and it even includes a Pause/Start button so you can stop a download and come back to it later. The Feed Manager is the best I've ever used.

Some call these features unnecessary because they never use them, but I do and I think they're great. I don't use every feature (I've yet to even touch "Opera Unite") but they are easy to ignore if you don't find them useful.

Honestly, when it comes to speed, there really isn't that much of a difference between Opera 10.5 and Chrome 5. If you honestly like the feel of Chrome better, then use Chrome. Use what you like. I like Opera, so that's what I use.

### Post by vaiocomputer on 2010-05-10
Opera is AWESOME!  It's one of my favorite browsers, along with Firefox and Chrome.  By the "benchmarks" alone, Opera is generally the fastest, next to Chrome 5 beta, by a <10% margin, which does not translate exactly to your browsing experience, I would say Opera is more reliably faster than Chrome.  Opera uses less memory due to it not having a new process per tab/extension, uses DNS prefetching (also in the Chrome betas), doesn't store hundreds of megabytes of cache to slow it down, and it is just faster on the different computers I have installed it in than Chrome, which on some can start lagging.
