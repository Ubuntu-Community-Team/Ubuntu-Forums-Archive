As for WiFi FreeBSD has wpa_supplicant inbuilt so I just followed the wireless section in the handbook to get wireless working ([http://www.freebsd.org/doc/handbook/network-wireless.html]("http://www.freebsd.org/doc/handbook/network-wireless.html")). Also actually make sure your wireless chipset is supported. Here is my wireless chipset in my laptop which works fine in FreeBSD:
```

andrew@andrew-laptop> dmesg | grep iwn0

```
