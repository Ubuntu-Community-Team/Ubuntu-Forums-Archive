Well everything might not work well for everyone. Ubuntu is the most user friendly Linux at least for me. Switching from Windows is not easy, but if you choose to, you should have the courage to learn something new. Just a bit patience, and most probably you'll love the way things go in Linux.

Everything installed in Linux doesn't go the Windows way in the Program Files (or a custom location), instead it is installed in the /usr/bin folder and mostly can be accessed via the Applications menu in the related sub category.

To search for programs you can simply type in a terminal
