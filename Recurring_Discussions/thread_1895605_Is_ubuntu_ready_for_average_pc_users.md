### Post by aysiu on 2011-12-15
I think there are only two major requirements for Ubuntu being ready for an average user:

1. The user has to come to it with an open mind. It can't be everything was "better" in Windows when Ubuntu's really just different from Windows.

2. A power user has to install and configure a working LTS version and tell the average user to automatically install updates but never to upgrade the entire OS.
