### Post by Aqualung on 2009-04-03
Here's something coming from my experience--a seasoned Windows veteran--with Ubuntu so far:

1. If you're sufficiently computer savvy, you should be able to handle root access in Ubuntu.

2. If, on the other hand, you're not sufficiently computer savvy, Ubuntu/Linux is not for you.

Hence an advice for those who attempt to think outside Windows by toying w/Ubuntu: do yourselves a favor and acquire root access. Ubuntu will suddenly emerge as more interesting. Asking for a password anytime one wants to blow a fart, and smashing one's head on the wall of file and folder permissions is no way to treat potential users, however savvy or dumb they may be. And yes, Ubuntu does that *every step of the friggin way*--that is, even for routine and minor moves.

This should also count as a suggestion for Ubuntu devs.: There is no way in hell that Ubuntu/Linux will ever be anything but a niche ultrageek product in its current state. If you are attempting to steal customers from Microsoft, disabling root access is definitely the wrong way to go.
