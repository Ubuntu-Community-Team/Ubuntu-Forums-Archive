### Post by mb_webguy on 2009-04-14
The links others have provided are excellent sources for an answer to your question, but I'll just add a few reasons why I personally use Linux.

[LIST=1]
[*]**It's free.**  Whereas Windows and Mac OS X cost $150 or more per licensed copy, Linux is completely free of charge.  You can install it on as many machines as you want without paying a dime.
[*]**The software is free.**  Software for nearly every task imaginable is freely available, and extremely easy to install through software repositories.
[*]**It's secure.**  The security model used by Linux is simply better than that of Windows.  Even if Linux were to become as popular as Windows, and more malware developers began targeting Linux systems, Linux users would have only a small fraction of the problems with viruses, trojans, and other malware than Windows users.
[*]**It's non-proprietary.**  No one can tell you how you can or can't use Linux.  There's no such thing as a Linux EULA.  There's no such thing as vendor-lock in the Linux world.  You're free to choose any software that works for you, and use it however you like.  Because it's free and open source, you can even modify it if you want to.
[*]**It's highly customizable.**  Unlike Windows, which essentially only gives you the option of changing the color of windows and the image used as your desktop background, Linux is extremely configurable and customizable.  Numerous desktop environments and window managers are available, and most are themselves highly customizable with different themes.
[*]**It can be installed on almost any hardware.**  Linux can run on your 10-year-old desktop gathering dust in the corner, or your brand-new laptop.  Linux (or some variation thereof) even runs many home appliances and consumer electronic devices.  So if you have some old hardware laying around, you can put it to use cheaply and easily with Linux.
[*]**It's a community effort.**  Rather than being dependent upon a single corporation for support, Linux is supported by a large community of enthusiastic user-developers.  Bugs tend to be noticed, reported, and fixed much more quickly than with proprietary software.  And if you need help, many knowledgeable individuals volunteer their time to forums like this one -- which means knowledgeable support 24 hours a day, rather than support from a part-time tech support guy reading from a cue card.
[/LIST]
