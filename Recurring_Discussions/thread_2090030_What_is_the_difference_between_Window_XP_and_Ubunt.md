### Post by ivotkl on 2012-12-01
I've used both quite often for around 2 years. I've been using Windows since 95 came out, using Ubuntu since 2009 although not that often at the beginning as I was getting used to it.

I' ll point you out some differences I've found with the years of usage (and some I've read as well):
- HDD is handled differently.
- There are nearly no viruses, though you can obviously still get hacked or led to entering harmful codes on terminal (an equivalent for Bill's command-line). I have, however, installed an antivirus as people I live with use windows and I don' t want that to be infected.
- There' s no such think as disk fragmentation, as far as I've read.
- A lot of distributions are user-friendly if you don' t want to stick with us (although I've tried around 5 of them and decided this is the one that suits best my needs).
- Most software is free-software and/or respects GPL license.
- You still have some proprietary drivers software (which I don' t like, but have to use on a few components my laptop has) in case there is an incompatibility list.
- You can modify operating system as much times as you want and it will still be legal (recommend you to read GPL License agreement).

### Post by agillator on 2012-12-01
What kind/level of differences are you looking for?

Windows is proprietary. When a new version comes out you must spend money for it. Ubuntu is free. When a new version comes out you simply install it at no cost.

If/when an error is found Windows is updated when Microsoft feels like it and gets around to it. Linux/Ubuntu, on the other hand, is updated almost immediately (and a new version of Ubuntu is released every six months). 

Linux was designed for networking from the start. MSDOS and Windows were not. The networking was added and, in my opinion, is problematic.

Linux is, in general, more secure than Windows. 

Linux is, again in general and in my opinion, more reliable than Windows. I have left a Linux computer running for well over a year without having to reboot it. When I did use windows that would have been impossible. My friends who use windows today still find that impossible.

Windows has more games available for it than Linux/Ubuntu does. In general there may be more software available for Windows, but anyone seems to be able to find software to do anything they wish on Linux/Ubuntu.

There is a philosophical difference between the two. A Windows program tends to do everything from start to end. This means you are effectively limited to what the programmer expects/foresees/thinks. Linux software tends to do only one thing but do it very,very well. The user is expected to tie programs together (pipes) to accomplish the total task. This allows things the programmers never foresaw or planned on to be easily accomplished. Obviously this is not 100% true; office suites are basically the same regardless of system.

The real power of Linux lies in the command line, readily available to the user. Windows command line use is very limited and I get the impression Microsoft would really rather not have it available at all. 

Obviously this is written by someone who has not used windows with any regularity in years and is quite happy with Linux/Ubuntu. A Windows user would give you a different list.

### Post by black veils on 2012-12-02
main differences for a newbie:

- you can change the graphical user interface (desktop environment). it doesnt just change the appearance though, there are different options and default software available. some desktops are lighter, some are heavier.

- the system is generally more customisable than Windows.

- not having to bother heavily with anti-virus and defragging.

- applications are usually installable via an app centre, instead of searching websites.

- "Driver support
some hardware problems Manufacturers of computer hardware don't always (yet) release drivers for Linux or publicize full specifications.
So, drivers are sometimes unavailable for the most recent high-end graphics cards — some are reverse-engineered from Windows drivers by volunteers. Some are also released free of charge, but under restrictive licenses.
Sadly, the same problem occurs with many low-end WiFi cards.
However, the vast majority of computers with standard hardware work out of the box with main Linux distributions. Common hardware such as USB keys or photo cameras never are a problem."

- "Gaming
gaming Hard-core computer gamers, you might find that the Linux gaming world is less professional and out-of-the-box than under Windows: the large game developing companies are only slowly getting interested in the free software world, which means that many well-known titles do not work on Linux, or need special restrictive emulators to work.
There is a very active free software game development community though, and they have produced quite a few high-quality games, both 2D and 3D, that are free software. You can head to the Ubuntu gaming forum to find more information."

a really good resource for newbie info is [this ]("http://getgnulinux.org/linux/#.ULdDbhGFKf0.facebook")
