In all fairness, there's not a LOT that MS Windows can do that Ubuntu can not, but there are some niche apps and hardware that need MS Windows to work.

### Post by JohnWesleyMethodist on 2009-07-13
> **Oceanik said:**
> For those who are using Windows and Ubuntu what do you think is missing/not working well in Ubuntu that you can do in Windows (apart from gaming)?

 It's not the big apps incompatibility with Ubuntu that's a problem, because Ubuntu has replacements for them all.

 It's the small apps that are problematic, that you just don't see replacements for on Ubuntu. Some guy somewhere writes a program that you find really handy and there is no replacement.

 Also of course games. I have added games from Add/Remove but most of them are buggy and crash, or lock up Ubuntu or whatever. Yikes!

 Actually one more major annoyance is security programs in Ubuntu are awful! Now I know security is not a huge concern on Linux, but I still would rather lock my door at night then trust my neighbors to play nice.

 On my Mother's laptop with Win 2000 I have Avast! Free which has a beautiful GUI and it's fully automated to update and scan. I also have Panda Anti-Rootkit which searches the computer for Rootkits and removes them if it finds them. It's 100% automated and has a GUI.

 On Ubuntu you have to run RKHunter in Terminal and then you get files that it says are suspicious and then the program ends and you don't know what to do next. Then you are supposed to download files and compare the Hash in the new file to the one on the drive. Then you have to update RKHunter so that it ignores those files which sometimes means editing RKHunter.conf. Pain in the backside!

 How about Firestarter? I want to set the outgoing data to restricted to limit what goes out to the internet. On Windows Firewalls that feature is included automatically. In Firestarter if you enable it and want to run xyz program that needs to connect to the internet, then you have to scour the internet to find which port that program needs in order to allow it access. Windows Firewalls automatically pop up and say: "Program xyz wants to access the internet on port xyz. Is this okay?".

 Oh yeah, more... How about Hardy - which is supported until 2011 - with it's non-updated repositories! If you want a newer version of program xyz, you can't get it unless you manually find it yourself on the internet which is a "security risk" to install according to Ubuntu. But you are lucky if the new version of the program is available in .deb format because if not only an expert can install the program. This is not user friendly which is ultimately Ubuntu's goal.

As a relatively new user (approx 2 years) I have found Ubuntu has so much going for it, its hard to say anything bad about it! I much prefer it over Windows and OSX. 

Ubuntu start up and shut down times are incredible, lack of virus concerns, amazing desktop effects, easy to use interface, and just about every kind of software is available... except for a few items. 

Unfortunately these few items are absolute showstoppers, limiting my ability to support Ubuntu as a complete replacement for a Windows or MAC

[LIST]
[*]Tomtom GPS Software

[/LIST]


[LIST]

[LIST]
[*]IM/Video Conferencing

(Pidgin, Empathy, Skype, Yahoo, Netmeeting)
