### Post by HermanAB on 2010-11-08
Howdy,

You are starting off from a Microsoft FUD fallacy.  The dominant OS is Linux.  There literally are billions of Linux devices out there - mostly cell phones, routers, servers and mainframes.  It is true that Windows is dominant on the desktop, but that is only a small part of the total market.  So, yes, MS has a lot to fear from Linux.  Linux can potentially wipe MS out if there is a shift in the market - e.g. if everyone would embrace online work instead of desktop work methods.
