### Post by snowpine on 2011-07-16
Flash Player is not a Linux distro, it is closed-source "proprietary" software from Adobe that has its own [hardware requirements]("http://www.adobe.com/products/flashplayer/systemreqs/"):

> Intel Pentium 4 2.33GHz, AMD Athlon 64 2800+ or faster processor (or equivalent)
512MB of RAM

Flash Player aside, there are plenty of Linux distros that will run on a Pentium 3 with 128mb of RAM (including Ubuntu). Basically you are looking at a CLI-only server install; you can use it for educational networking projects: file server, printer sharing, firewall, torrent slave, etc.
