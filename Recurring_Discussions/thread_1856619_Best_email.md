### Post by Basher101 on 2011-10-08
I have never been using any email clients on any OS so far. Then i tried Evolution but somehow i had troubles setting it up at all, then all my emails got just downloaded (so my inbox folder on the website was empty) and it did not show the notification when i got new emails. A few weeks later i just saw thunderbird, liked the icon and just installed it for a try...in a matter of seconds i got it set up, i got notifications and it just copies over the already inbox mails from the website (thus the ones already inbox on the website **_stay_** there) and only receives the new ones (as long as it is running in the background). I can only recommend it.
