Okay, you're being pretty vague here so again until you can provide a little bit more information I'll give you some basic stuff to get you started.

First make sure you've downloaded the correct iso from : [http://www.ubuntu.com/download/ubuntu/download](http://www.ubuntu.com/download/ubuntu/download)

After that burn it to a disc. Boot it , and "test the disks integrity" to make sure that the disc was created properly. 

Once you have verified that you can boot to the liveCD and install Ubuntu 11.10. Obviously removing the other install is a choice, or dual booting with it (just in case) it might be a good idea to dual boot at first.

Also when I said you had hardware problems, are you sure your hardware isn't failing. Not what type of hardware you have. If you still have issues after re-downloading the new iso I would definitely start questioning hardware, or the media you are burning it to.
