Hi guys. =]

It's been a while since I've been on here as well as a while since I've used Ubuntu sadly. I hope that none of you mind me sharing this, but I am really excited to spread the world and help support this project.

While browsing Wikipedia learning about Linux, BSD, and other Unix-like operating systems, I somehow stumbled across an operating system that is coded purely in C with some C++ programming as well. _It's designed to replicate and replace Windows operating system, by performing better performance, and also being compatible with Windows programs, applications, hardware, and software._

This operating system really interests me as I know it will interest you, but it really hurt me when _it was documented on Wikipedia about this operating system that it's labeled as a "hobby software," actually explained on Wikipedia on its page only because it's not very well known, lacks funding, and doesn't have many developers.._ =(

I can't believe Wikipedia really labeled this project as a "hobby software" because of those reasons. After checking the software out, reading & learning about it, and checking out its screenshots, I was impressed. You will be too. _It's been developed since the late 1990's. It looks and functions like Windows but without the security holes._

_I am trying to expose it as much as I can to get the word out to help this project. This is an open source operating system so if you support open source please help me spread the word! =] Hopefully some C & C++ programmers will find this thread and be interested in help programming it, as well as some people will find it and donate to the project. I know I'm going to be donating sometime this week._ I *really want to see this project take off.*

**Whether or not you're interested in this operating system, one thing is clear, and that is that this project has done what many operating systems have tried to do & are still working on.** As you'll see in the screenshots, this operating system got the interface of Windows completely down as well as compatibility within its own unique coding, and it also uses WINE to help be even more compatible, though still not 100%. _Even as it progresses and hopefully develops more through all of our help one day, it can not only become a real free alternative to Windows, but help developing operating systems such as & including Ubuntu help shape its interface and make it Windows-compatible as well._ =]

Nevertheless, here is detailed information & links.

**The operating system is 100% open source and desperately needs help.** **_Please contribute by helping its development by assisting with C & C++ programming; please contribute by donating money to the project (every little bit helps, I'm not donating much as I can't afford much); and/or please contribute by spreading the word like I'm doing - support open source by spreading the word, and help many open source programs that aim to provide alternative solutions to Windows develop further by/through supporting this operating system, in which is developing remarkable integrated Windows-compatibility (software, hardware, as well as interface and user-functionality) technology._** It's been developed by dedicated individuals since the late 1990's and in 2010 is still only in alpha though it's done a lot as you can see and looks great.
