### Post by blueshiftoverwatch on 2010-01-18
Firefox has the NoScript extension, which makes managing Javascript extremely fast and simple. As well as a plethora of other security related addons. Managing Javascript effectively (ie, blocking all scripts that aren't absolutely necessary) is probably one of the most important things you can do to secure any browser. Other browers will let you do it, but only (as far as I know) NoScript for Firefox makes it convenient enough that you won't mind doing it. A sketchy website can't run malicious code if your blocking it.
