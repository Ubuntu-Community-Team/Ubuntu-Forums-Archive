### Post by Version Dependency on 2012-02-17
Another way to execute "xkill" (other than adding a custom launcher) is adding it to Nautilus Scripts.  Go to /.gnome2/nautilus-scripts/ and create a file called xkill.sh.

```

#!/bin/sh
xkill
```Save the file and make it executable.  Now, right click on the desktop and you should see a menu called Scripts...and a submenu that lists xkill.sh as an option.

One way to build a MATE Ubuntu without Unity or Gnome 3 is to start with the Minimal CD. Then:

sudo apt-get install python-software-properties
sudo add-apt-repository "deb [http://packages.mate-desktop.org/repo/ubuntu](http://packages.mate-desktop.org/repo/ubuntu) oneiric main"
sudo apt-get update
sudo apt-get install mate-archive-keyring
sudo apt-get update
sudo apt-get install mate-core
sudo apt-get install gdm

will give you a working bare-bones system.
