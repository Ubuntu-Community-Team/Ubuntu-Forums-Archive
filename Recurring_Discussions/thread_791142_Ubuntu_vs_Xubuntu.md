### Post by 4Orbs on 2008-05-19
I have gone through an experience similar to rage-a-w in that I started with the standard Ubuntu and considered it really swell. So I went the whole shebang and upgraded to Ubuntu Studio and thought it was even more swell. Then after a few weeks I was overtaken by boredom one day and decided to see what Xubuntu was all about.. with no intention of abandoning UbuStudio or the Gnome desktop, not a chance, nada, no way. To make a long story short, I am now grooving on Xubuntu and will add the studio packages one at a time as-needed. Xubuntu is an excellent foundation for implementing my own specific preferences in software choices and desktop customization. It's everything I need and nothing I don't need (unless I choose to add fluff). There is nothing wrong with Gnome or KDE if you like lots of things pre-configured and ready to go immediately after installing. But Xubuntu is slick, light and uncluttered. That makes it better IMHO. Also, there is a noticeable difference in xfce as an addition to the Ubuntu install (slower) compared to a clean install all by itself (faster). I have re-installed standard Ubuntu a few times now, and each time find myself longing for Xubuntu and the Thunar file manager in particular. Ubuntu is a touring bike, Xubuntu is a motocrosser. Ubuntu is baggy swimming trunks, Xubuntu is a speedo. Ubuntu is a Cadillac, Xubuntu is a 'vette. oops, I forgot to make a long story short.
