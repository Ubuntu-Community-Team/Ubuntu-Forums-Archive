### Post by Giant Speck on 2009-04-10
[B]_Giant Speck's Guide to Telling the Differences Among FUD, Opinion and Fact._

[/B]The Merriam-Webster online dictionary defines a fact as "a piece of information presented as having objective reality."  A fact is a piece of information that is known to be real and true.  The following are examples of what are and what are not facts:[INDENT][SIZE=1]**FACT:  **Microsoft signed an agreement with Novell in order to work closely together to improve their software's ability to interact with other software.
**NOT A FACT:  **Microsoft signed an agreement with Novell in order to destroy Linux.

**FACT:  **Microsoft released Windows Vista as a replacement for Windows XP.
**NOT A FACT:  **Microsoft intentionally released a crippled operating system as a replacement for Windows XP in order to lower consumer confidence and raise expectations for Vista's replacement.
[/SIZE][/INDENT]FUD is an acronym that stands for fear, uncertainty and doubt.  Generally, FUD refers to the spread of disinformation as a competitive weapon.  FUD is usually identified by falsehoods, opinions, and misinformed points.  The following are examples of what is and what is not FUD:[INDENT][SIZE=1]**FUD:  **Linux (or specific distro) can do everything that Windows can.[/SIZE]
[SIZE=1]**NOT FUD:  **Linux (or specific distro) can do most things that Windows can, as well as things that Windows cannot.  Some examples are...[/SIZE]

[SIZE=1]**FUD:  **When using Windows, you have to use resource-crippling anti-virus and anti-spyware software, as well as firewalls.  Without these programs, you are at a constant risk for attack.[/SIZE]
[SIZE=1]**NOT FUD:  **Responsible browsing and downloading, and using a limited user account can be just as effective as using anti-malware software.  While Windows is prone to malware, anti-malware software is not an absolute necessity.[/SIZE]
[/INDENT]Next, we come to opinions.  Opinions are personal beliefs and judgments that rest on grounds insufficient to produce complete certainty.  Usually, opinions present no actual fact and can sometimes assume that what is true for the person making the opinion is true for all.  Here are examples of opinions:


[LIST]
[*][SIZE=1]Windows (or Linux) sucks.[/SIZE]
[*][SIZE=1] Windows (or Linux) crashes all the time.[/SIZE]
[*][SIZE=1] Windows (or Linux) is to blame for my hardware problems.[/SIZE]
[*][SIZE=1] Windows (or Linux) is to blame for my software problems.[/SIZE]
[*][SIZE=1] Windows (or Linux) is a virus.[/SIZE]
[*][SIZE=1] Linux users are communists.[/SIZE]
[*][SIZE=1] Windows users are sheep.[/SIZE]
[/LIST]
 
As a bonus, I will explain something I like to call "misguiding and misplaced anecdotes," or MMAs.  MMAs can normally be found in technical help threads.  MMAs almost never offer solutions and aren't very helpful to the user in need of help. Here is an example:[INDENT][SIZE=1]**USER 1:  **Ubuntu 8.10 doesn't install correctly on my computer.
**USER 2:  **I don't see what your problem is, I installed Ubuntu 8.10 on all four of my computers, and I didn't experience problems.[/SIZE][/INDENT]
