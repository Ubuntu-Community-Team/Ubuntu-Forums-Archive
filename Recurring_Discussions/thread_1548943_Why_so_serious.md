### Post by andymorton on 2010-08-09
I've never used a Mac so I can't comment on that but I prefer Linux  over Windows for the following reasons - 

1. Start-up and shut-down times are so much faster. The desktop appears, I click on the web browser icon, it opens immediately and I'm online. With Windows I'd have to wait for what seems like an age. Ubuntu shuts down within five seconds whereas Windows can take over a minute. 

2. The level of customisation. I can make Ubuntu look, sound and feel however I want it to. 

3. Plenty of free, open-source software available within a few clicks of the mouse. 

4. I don't have to worry about viruses, spyware etc. 

5. I'm not putting any money in Gates's and Ballmer's pockets. I don't want to sound like a rabid Microsoft hater but it is a horrible company with terrible business practices. 

6. An excellent level of support, i.e. this forum and various other sources. It's not often I have a problem with Ubuntu but if I do then I know that help is out there. 

7. Much less pressure on system resources. 

8. The level of control I have over my computer. Windows seems to say 'I'm going to do this whether you want me to or not'. Ubuntu says 'I'll only do something if you tell me to.' I like that. 

There;s probably many more reason that I'll think of later. 

The only downside to using Linux is that it won't run some specialist scientific software which I often need and the open-source alternatives aren't up to the mark. However, with that exception I can't do anything with Windows that I can't do with Linux and (as stated above) there are many advantages of using Linux. 

As for the question, Why so serious?, I think for many people using open source isn't just about software. It's also a political point whereby they believe in the ideas outlined in the 'Ubuntu philosophy' and reject the capitalist model of Microsoft and other proprietary companies.

### Post by murderslastcrow on 2010-08-09
Well, you're right that more experienced users are the ones who find the true benefits.

As such, every time I'm on Windows or OS X for school, it feels slow and clunky compared to my setup at home (even when I was using KDE 4).

I can't just Alt-Click windows to move them in OS X or Windows without getting third party add-ons, which don't work the same way. The speed is just unrivaled, as well.

Also, you can reproduce any GUI available easily in Linux. This, along with the fact that any Windows applications I would like to use run in Wine anyway, I'm left asking, "why would I want to use Windows or OS X?"

They are good operating systems, but when it comes down to it, I can do all the same stuff, faster, and with more helpful features in Linux, for free. Any impulse to use the other platforms would have to be entirely application-based, not OS-based.

Also, it's important to note that, while many open source applications have been ported to other another OS, unless it's all KDE 4 stuff, many applications are a pain to install on Windows/OS X without some sort of virtualization. Macports does an okay job, but anyone using Windows and wanting Banshee, Exaile, Shotwell, Openshot, etc. is gonna' be in the dark. Also, when you add up all the other benefits of using Linux over the other two (speed-wise and manageability), it just starts to make a lot of sense.
