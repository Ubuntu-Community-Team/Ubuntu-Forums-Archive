From FAQ Section on website:
> Is ReactOS legal?

Yes. It's fully legal.

Developers have not looked at Windows® source code. They have used the public documentation of Windows® OSes. They have made several tests to understand how Windows® works. In fact, ReactOS does the same things Windows® does, but not exactly in the same way, because they haven't the same source code. All code in ReactOS is under the GNU GPL (General Public License).

Why don't you help develop Wine/Linux instead?


This is a very common FAQ, so it's placed in User FAQ also (and technical answer is given in Developer FAQ).

ReactOS works very closely with Wine, and thus both projects actually benefit from each other. We have several developers in both the WINE and ReactOS projects that work on cross-compatibility issues between the two projects.

It is our view that Linux + Wine can never be a full replacement for Microsoft(R) Windows(R). ReactOS has the potential for a much higher degree of compatibility - especially for Microsoft(R) Windows(R) drivers - which WINE does not address.

### Post by dvlchd3 on 2010-05-12
It seems like another attempt to create an open source OS...not Linux, not FreeBSD, not OpenSolaris, but an OS written from scratch to be just like Windows, but without the overhead of Microsoft's proprietariness. (I know, not a real word...)

Assuming the code is stable and mature, the only issue I see is, why the hell would anyone use a "fake" Windows?  Its sort of like the last US election, a fake Democrat, or a real Democrat.  The real one won...

Anyway, I'm not sure I see it is a noble venture to attempt to write a FOSS version of Windows, however, I can't see it catching on very well at all.  There are already several GUIs for Linux that attempt to look just like Windows, in order to help users convert.  KDE and Gnome are not hard to use, and tend to be easier, after a user gets used to them.  I really think FOSS developers should spend more time creating better alternatives to Windows only software, rather than wasting time creating a fake Windows...

Just my opinion :)
