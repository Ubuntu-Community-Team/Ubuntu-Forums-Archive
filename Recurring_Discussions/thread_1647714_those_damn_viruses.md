### Post by mcduck on 2010-12-18
> **khelben1979 said:**
> I would recommend that you use anti-virus with your Ubuntu system any way, since Windows viruses on your system can spread and make harm to others which is using Windows.

In my opinion, that's the biggest reason for using anti-virus with Linux. Maybe doing a scan once a month would be enough..

I have to disagree with this, actually. :)

The viruses don't work on Linux, so they can't spread on their own. Of course you might accidentally get an infected file from somewhere, and then pass the file to some Window user. But at that point that windows user's antivirus program will detect it. If it doesn't, the guy already has some serious troubles, being using a non-functional antivirus software.. ;)

Of course if you run an e-mail server for a large company or or something like that, then it might make sense to scan all the files on the server instead of wasting the resources of all individual work stations. Which is pretty much the only reason why I'd ever run an antivirus program on a Linux machine. (The most popular antivirus app, ClamAV, actually is intended for this use, not for desktop computers.)

Of course everyone is free to do what they want, but I just see it the way that scanning for windows viruses on desktop Linux systems doesn't actually make any difference.
