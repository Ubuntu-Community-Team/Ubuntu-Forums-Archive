+1

We are many people (volunteers) who are happy to help. But we need details about the computer and a specific problem to solve. If you have many problems, start with the most basic one, solve that problem and go on to the next one. Feel free to ask about details, where to find more information, alternative ways to do things etc.

If you can switch your computer between classic BIOS and UEFI (in the BIOS menu system), you can easily install Ubuntu in BIOS mode, but you have to switch that setting of BIOS/UEFI when you switch between Ubuntu and Windows. The next step is to learn how to change some settings in UEFI mode, so that Ubuntu can co-exist with Windows (in different or the same drive). This is possible in most but not all computers. If the UEFI system conforms to the standard, it is possible, but some vendors have made systems that make it really hard (or impossible) to run anything else than Windows. For example, you should switch off secure boot and fast boot. Windows should not hibernate (not even semi-hibernate) for it to work well. And you need a current 64-bit Ubuntu (or Lubuntu, Xubuntu, Kubuntu) iso file and burn it to a DVD or flash it to a USB pendrive and install from that into an internal or external mass storage device (HDD, SSD, pendrive). USB 3 drives can be quite fast, and give you a nice system. See this link, and links from it, for example about UEFI.

[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)

[https://help.ubuntu.com/community/UEFI](https://help.ubuntu.com/community/UEFI)
