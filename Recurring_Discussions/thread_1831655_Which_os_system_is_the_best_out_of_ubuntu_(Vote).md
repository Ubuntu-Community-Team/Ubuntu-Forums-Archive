### Post by TheNosh on 2011-08-24
> **edwardpatch1 said:**
> 1 more day left to vote then i will make a document about it so i can show my friend which one is the best :)

I don't think your data will be very reliable if you collect it from an Ubuntu website. Ubuntu users tend to like Ubuntu.

I joined this forum when I used Ubuntu. I haven't been happy with a release since 9.04. I'm not quite sure if Ubuntu's quality has changed or if my standards have changed. Either way, out of the options you gave, my vote these days is Windows 7. It's fast, rich in features, and I've never had an update break it.

I like Linux for different reasons, such as customisation. I like that I can make most of my computer's interface fit the way I want it to work, rather than how someone else decided it should work. That conflicts with stability, though. Too many interchangeable parts and options leads to greater difficulty in making it work together.

I like Windows for stability and Linux for customisability. Unfortunately, these days, Ubuntu is not a distribution I find myself willing to deal with.
