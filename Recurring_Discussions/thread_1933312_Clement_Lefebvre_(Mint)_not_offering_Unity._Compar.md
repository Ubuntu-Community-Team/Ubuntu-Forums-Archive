2. In reading about this Unity controversy, I've noticed that former Mac users seem a LOT more open to it than former Windows users (of which I am one).  And I got to thinking, that Linux converts are going to come from really only two places.... they are going to be former Mac users, or former Windows users.
As a disaffected Windows user doing research on just WHICH Linux distro to migrate to, Ubuntu stood out at first, because it was one of the few distros I had actually heard about without any previous research.
But then Mint stood out- an extra layer of bug fixes HAS to be a good thing.  Plus, I'm more fond of green than brown!  So Mint has been a natural for me.  (Still, I've strongly considered Ubuntu Studio and Xubuntu, and may still end up running one of them)

Mint uses the Ubuntu repositories.... programs that run on Ubuntu run on Mint.  And that's a biggie, because coming from Windows, I'm used to having a wide variety of software programs to run... and as a musician, I need special software programs.  The fact that there are Ubuntu and Mint compatible programs is important to me.
And I think the fact that these two biggest Linux distros are compatible when it comes to writing programs is ALSO going to be SUPER IMPORTANT to programmers writing all kinds of useful programs.
This is HUGE.  We can argue about desktops all day long, but in the end, software availability is a make or break thing for an operating system.  I was interested in a fast, small OS in the early 90's that was an alternative to Windows (GeoWorks aka GEOS), and in many ways was superior to Windows.  It was small and fast, and the Motif interface was very easy to use.  But although it attracted attention, it never got to the place of critical mass where very many people were writing programs for it.   By 2000, it was pretty much dead.

Anyway, I see it as a win-win situation for both companies if they maintain software compatibility, but go in different desktop directions.  Mint obviously has more attraction to disaffected Windows users, but it seems former Mac Users are attracted to Ubuntu, and even to Unity.  I'm sure lots of kids get started on Macs at home or at school, then get out on their own, and can't afford to upgrade to the next generation Mac!  (Especially if they are musicians or artists!)
What are they going to do???  Run Ubuntu!!!!  (or maybe Pear OS)
And disaffected Windows users like me (I never COULD afford a Mac) are going to continue to be attracted to Mint's conservative approach.  And by appealing to different demographics, they aren't really competing with each other... but with software compatibility, and both of them growing.... WE MAY SOON REACH A THRESHOLD WHERE EVERY PIECE OF HARDWARE YOU BUY IS UBUNTU/MINT COMPATIBLE!!!!:D:D:D  AND COMES WITH DRIVERS.... or maybe will get the drivers loaded to the repository, or whatever.... but at some point companies will just bite the bullet and make their stuff Linux/Ubuntu/Mint compatible.  And anybody writing a major program will port it to Linux/Ubuntu/Mint!
4.  Somewhere recently, a Mint user pointed out that a lot of Ubuntu's numbers come from hits on their repositories.  But to the repositories, a Mint user LOOKS like an Ubuntu user... and so does every other Ubuntu-based distro user.  So that 20 million figure for Ubuntu users actually includes all users of Ubuntu based distros too.  So FWIW, minus Linux Mint's numbers from Ubuntus... but it doesn't really matter, because they aren't REALLY competing--- they are competing for "converts" from different places, but mutually encouraging programmers to write compatible programs.
5. Back when I was a GeoWorks user, I saw how emotionally attached I got to the interface.... and how much I HATED Windoze!
