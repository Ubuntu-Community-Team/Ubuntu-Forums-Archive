### Post by tagra123 on 2008-07-24
Not going to repeat good points mentioned above.

One thing that could help attitudes is for programs to backwards compatible. For instance I used a few programs that backward compatibility was not considered. I write some commercial software and if the new version could not use data or files created using the precious version -- it would be a hard sell. DVDStyler and Getlive come to mind although there are other too.

I think UBUNTU is easier to install than Windows already but I have compatible hardware.

Software loyalty is also an issue -- if you have software that you use everyday and there isn't a version for linux / ubuntu and it wont run in wine then that could be prohibative.

Wine could be GUI'ed up a little to make it a little more like windows. For instance a menu item like synaptic but labeled "Install Windows Program" might make the transition easier for a new user.

One of the biggest things I think would help is if the Video, Modem, Printer, etc... worked in the previous version that it will work in the next version. I've seen some printer compatibility broken in the last two years. This again is similar to backwards compatibility.

File sharing has come a long way.

XServer -- Someone who has done this right is Puppy Linux. When the video doesn't work you are given a "real" option to get a working display -- not a blank screen or cryptic message. I've used linux for a while now and know to just work with the xorg.conf file, but someone new or not as determined to get it to work through sheer will will give up. A blank screen after installing is unacceptable -- period.

### Post by hotrod6657 on 2008-08-03
I think the first step is to build a fleet of computers to sell at stores and online (like newegg).  I have no idea how hard it is to get places like best buy and office max to sell your stuff but smaller scale stores might be an option too.

Then, with these computers:

Install the latest stable version of Ubuntu.

Make sure that all of the hardware is compatible and works correctly on these machines.  That doesn't mean that they have to work perfectly from the get go but spend time figuring out how to get them to work right before packing them up.  And try to test thoroughly enough to uncover problems that new users might experience and develop solutions.

Dress it up.  Like was said, pay special attention to the visual department and have these boxes set up to have some of the "trick" effects enabled already.  Part of the dress up process means being willing to leave, or at least offer an alternative to, the brown theme.  If you give the OS a handful of themes, brown, a dark theme, primary color themes i think the general public will be more impressed by the looks.  Which, let's face it, is a very big part of being satisfied with your computer.

Get around the legal / ethical issues surrounding the codecs or make it easy for them to be downloaded without the new user trying very hard.  I know they're all in one package but i mean having the system prompt the new user to download them from the packet manager when they first start using the computer.  Anything to make it as easy as possible.

Finally, sell them for almost the cost of the hardware.  This will make them way cheaper than the windows and mac alternatives and make people a little more willing to try something new because it means saving money.  Then, with any money that is "profit" put that all in to the development fund as a donation to help facilitate even more improvements.  That way selling ubuntu boxes doesn't become a business and spoil all that i love about it.  It just helps to make the whole thing grow.

That's basically where i see that going.  It won't make Ubuntu the #1 OS by far, but it will get the word out there and once you have a lot of users who aren't super tech savvy using Ubuntu their friends will start to see it and get curious.  That brings them either to the online download, or to the stores to buy a new box.

I actually hope something like that happens.  It would be fun to help pick out the hardware and work on the systems.  I'll be keeping my ears open... haha (as if this is something that will happen very quickly)
