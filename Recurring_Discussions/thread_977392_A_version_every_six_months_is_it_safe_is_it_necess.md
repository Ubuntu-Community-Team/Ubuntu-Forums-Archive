### Post by wribeiro on 2008-11-10

Ok, everything would be fine if, after reinstalling all programs, my system stay stable, what is not happening:
- Intrepid only boots if a key is pressed on bootsplash (unless I use acpi=noirq);
- Keyboard fails sometimes when Mozilla accesses my gmail account;
- Nautilus is very much slow when showing large folders, specially using list view option (more than 3 minutes to show 1700 files);

Maybe it's time to make stronger versions and spend more time testing and solving bugs before releasing it. Users would be happier if, after ten months or a year, instead of six months, they could upgrade their systems having much less problems than what is happening nowadays.

I'm almost decided to downgrade my system to 8.04 or install Debian, it is not an easy decision, but maybe the only option I have.

### Post by aysiu on 2008-11-10
I've been through 8 releases, and I think your reasoning is flawed.

It is possible to have a stable release after 6 months, and Ubuntu has done it in the past. They don't do it every time, but they do do it sometimes. So if you get a release that's stable, stick with it as long as you can if you're worried about instability.

Also, there are the .1 versions. So there's 8.04 that's released when it's released, and then there's 8.04.1, which has critical fixes since the official release date.

Dapper (6.06) was pushed back two months (it was originally scheduled for an April release), and that's supported until next June.

Bottom line: you either find Ubuntu stable or you don't. Some find it stable. Others find it unstable. Still others find certain releases stable and others unstable. If you have a stable release of Ubuntu, stick with it. If you haven't found one, use something else (Debian, for example).

### Post by Ozor Mox on 2008-11-12
wribeiro, your thinking is flawed because you assume that when a version of Ubuntu is released every 6 months, the previous versions are all abandoned. In fact, bug fixes and updates continue for 3 years after an LTS release and 18 months after a regular release.
