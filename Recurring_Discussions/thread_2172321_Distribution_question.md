Welcome. This is a question that has been asked many time before. Low spec in order: Xubuntu, Lubuntu, then onto Puppy and a galaxy of others. You can go just for window managers with a system running Fluxbox, Openbox ... etc. Lowest spec would be a server like system with no desktop environment at all and working from terminal/CLI.
