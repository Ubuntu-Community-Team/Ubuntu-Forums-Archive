### Post by mrgnash on 2008-11-08
> **zoomy942 said:**
> So i ran that command (i'm a he..LOL) and it installed fine, but now how do i change to KDE?  It's not listed on the sessions section of the front page.

```
sudo pico /usr/share/xsessions/kde.desktop
```

Paste the following into the file:

```
[Desktop Entry]
Encoding=UTF-8
Name=GNOME

Exec=/usr/bin/startkde
```

Save, exit, log out and KDE should now be available from GDM. Note: I don't have KDE installed, so this is largely educated guesswork based on previous experience.
