### Post by 3rdalbum on 2011-10-21
> **NModern said:**
> I was afraid of "MAC" and now "Linux" has become just like it.

Er... how so? Because it has global menus that work differently to the ones in Mac OS, and a combined application launcher and window switcher that is bar-shaped and works differently to the one in Mac OS?

Unity is really nothing like Mac OS X's user interface. It's chalk and cheese.

> [SIZE=4]**Please create a normal classic desktop.**[/SIZE]

Ahh, well, we used to have one of those. People complained about long-standing bugs in it. We used to get a lot of flak because the classic desktop was ugly. Sure, it was easy to bolt on lots of different things, but it lacked some really nice window management features, context-sensitive drag 'n' drop, and a search system that returns more than zero results.

If you want this piece of software, it's still available in the gnome-session-fallback package. Changed a little, but still available.

I don't mind you being scared of the Macintosh, but the last thing I want is for you to be scared of change. I'd encourage you to try using Unity for a week. If you still don't like Unity after that time, then I'm sure you'll love Gnome Shell; heck, I love both.
