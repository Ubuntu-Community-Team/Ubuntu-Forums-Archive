If it is only Youtube then it is very easy, even now you don't need flash for youtube. On firefox you can install the flashvideoreplacer which works much better than flash, totem has a flash plugin that works on Youtube (though you can't control the resolution), there is minitube, Smplayer and Umplayer both have Youtube viewers. Indeed why would one even use flash for Youtube when there are so many better options?

So Youtube is a non issue, but there is a lot of media content that require flash (I don't mean porn, check out BBC or any of your favorite news site) and it is not only video (soundcloud, voice in google-translator, for examples).

To answer OP's question. I am not sure you can install pepper flash in Chromium because it is distributed with Chrome. If you only need flash 11.2 then why not just use Firefox?
