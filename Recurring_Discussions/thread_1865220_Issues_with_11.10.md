> **cobemk said:**
> And yes I still havent found any solution on how to get MY drives as shortcuts on THE DESKTOP.

You've managed to hide your question really well I must say under that huge block of text, but I found it!! :D

Install Ubuntu-Tweak and run it. Under Tweaks => Dekstop Icon Settings, you can find a few checks, tick the one that sits next to "Show mounted volumes on desktop".. There are a few other checks there that could be helpful for you..

And in general, if you don't want to use Ubuntu-Tweak, you can simply drag a location (from the location bar in nautilus) to the desktop while holding both CTRL and SHIFT down to create a shortcut for it on the desktop. You can make endless shortcuts this way, and rename them as you wish.
