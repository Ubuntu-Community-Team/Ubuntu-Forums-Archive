### Post by dtfinch on 2010-06-09
Thunderbird 3 for me seemed to have buggy/glitchy defaults (it's been a month so I don't remember specifics) and took some time to configure to run smoothly like version 2 did. There is still the slow font rendering issue, noticeable when scrolling the message list. And the new search seems unreliable, and a few previously working buttons (like showing remote images) seem to be no-op/stubs. I waited almost 3 years for it to come out, but I think it needs another year to stabilize and get everything that used to work working again.

Evolution seems to work better than it used to, though I can't remember all the specific problems that made me strongly dislike it in the past. There is still an issue with the leave messages on server and delete after X days, where it fails to actually delete anything, which can be a showstopper if you check your pop email from multiple systems. Another issue is that I'd like to be able to make the message list font smaller, but I've found no way to do so. Since my need to check email from multiple places has gone down, I've been content to leave it as the default since my last install.
