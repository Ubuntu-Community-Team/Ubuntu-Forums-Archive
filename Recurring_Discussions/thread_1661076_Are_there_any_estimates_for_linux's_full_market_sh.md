### Post by HermanAB on 2011-01-06
If you troll the Linux for Devices type web sites, you'll note that there are around 350 million Linux devices made each year.  Assuming that they last around 5 years, that alone amounts to about 1.8 billion devices.  Add to that another billion or so Linux cell phones, servers and desktops and the Linux installed base should be around 3 billion devices.

According to the Microsoft data on their marketing slides,  there are about 1 billion Windows devices in service.
