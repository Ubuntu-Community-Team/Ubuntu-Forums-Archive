### Post by snowpine on 2008-11-09
A few things to realize about Windows XP:
1) It's almost 8 years old (Xubuntu 8.10 was released this month)
2) It requires only 64mb of ram (Xubuntu 8.10 requires 192mb)
3) Every hardware manufacturer tests their hardware extensively for Windows (with Xubuntu, it can be hit-or-miss)
4) Microsoft tried to discontinue XP earlier this year, but lots of people really, really like it. :)

So I don't think it's really a fair comparison--if you're going to compare Xubuntu 8.10 with anything, compare it with Vista.

It might be worth trying Xubuntu 8.04 Hardy Heron on your computer. 8.04 is a stable long term support release, so there's the possibility it may run better on your hardware. If you want a really fast OS designed specifically for older hardware, try SliTaz, Puppy, DSL, etc...

### Post by 2Perfect on 2008-11-13
I did some tests for a technical presentation on Ubuntu vs. Vista. I submitted the topic before doing any real research; I just expected Ubuntu to be a possible alternative, or perhaps even better than Vista.

The setup was a fresh install of Ubuntu 8.04, a fresh install of Windows Vista home premium, and a non-fresh Windows XP on an old laptop. Times tested included boot up, login, shutdown, firefox load time with no tabs/extensions, firefox load time with tabs/extensions, hardware load time (USB storage). WinXP on this machine wasn't a fresh install, but might even be better than a fresh install. I've heavily modified the startup programs, services, etc. and barely have anything on it other than C++ and Java. 

Ubuntu was a little faster than Vista in all tests except for the USB load time (it was slower than vista for some reason). Anyway, just for fun, I tested times on Win XP and it turned out to be faster than both Vista and Ubuntu.

I ran the same tests with a non-fresh install of Ubuntu and Vista and the times didn't point toward one OS. For example Boot and Firefox load for Ubuntu was way faster than Vista, but Shutdown and Desktop load time was slower. So really it depends on what you have on your system, but on the default setup, Ubuntu is faster.
