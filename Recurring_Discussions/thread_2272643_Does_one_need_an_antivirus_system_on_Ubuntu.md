### Post by Rob Sayer on 2015-04-08
If you download files and share them with windows users (or you have a windows partition) you may want to have an on demand av scanning program like clamtk.  But you certainly don't need an av program running in the background like you do in windows.

It's not 100% accurate to say that viruses just target windows.  There are also iOS viruses, but not many.

The thing with linux is partly the very small user base.  It's also because the permission levels in unix/linux work properly, which they certainly don't in windows.  This makes it much harder to install a virus.

Note that just because you don't get viruses in linux does NOT in any way mean you cannot be hacked.  You still need a good password for a start.  This is why when noobs on forums are asking how to set up a password free login the more experienced linux geeks tend to panic.

In fact using some app or even OS thinking it'll make you immune to viruses and hacks is the worst thing you can do re. security.  I know a couple of windows users whose email password was hacked or phished ... probably the latter ... and they can't understand why their email is full of crap because they were using Norton Utilities.

I'd go so far as to say that if someone has that sort of mentality, they won't be secure with *any* OS.
