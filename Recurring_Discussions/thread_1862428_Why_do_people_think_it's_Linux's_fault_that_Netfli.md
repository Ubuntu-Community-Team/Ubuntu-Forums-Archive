### Post by aysiu on 2011-10-17
> **ki4jgt said:**
> When I posted a comment about the fact that it was fully capable of running Netflix's UI, but that Linux didn't want to hide things and take away control of the system from the user (by closing the source code) like other systems That's actually irrelevant. Linux can run closed source applications (Skype, Dropbox, Opera, Amazom MP3 Downloader, etc.). If there were a native Linux Silverlight plugin (including DRM), as opposed to Moonlight, then Netflix would run fine on Linux. There isn't an application approval board ("Oh, that has closed source or DRM? Well, we won't let it run on Linux"). If anyone creates an application for Linux, it'll run on Linux.
