### Post by samalex on 2010-09-03
Honestly they're all crap.  It used to be I'd only trust WD or IBM, but in my years of using computers I've seen no one vendor stand ahead of the others in quality or stability.  My suggestion to anyone building a new computer is to get two HD's the same size but by different vendors or at least different models because you don't want to get two HD's in the same batch that may have the same flaw.  If one is backing-up the other you don't want both dieing for the same reason.
