### Post by LeoSolaris on 2008-03-15
Well... right now Linux in general is still building itself up into a real competitor for Windows and Mac. It does not have the polish the big two have in terms of desktops, but it is getting there. The eyecandy is lighter and prettier than the main two, so at least that is a plus.

The software however is not in Ubuntu's control entirely. They do rely on upstream for polish and productivity. And until Game developing companies start producing games for Linux (or even Mac for that matter, they are just starting to catch up.) games will be lacking.   

In the end, what do you get gratis from Ubuntu that isn't done better (and more expensively) in Windows or Mac? It's the level of customization. You can try your hand at programing your own tools and plugins etc. It will take a while to learn, but it can be done.

Linux is still developing and has not yet really come into it's own as far as normal user productivity. It's a work in progress. As an OS it is far younger than Windows and Mac, but in that limited time, it has sprouted rather fast.

I may be new, but when I research, I research! I have had my eye on Linux for years, but have held off till now.

### Post by WalmartSniperLX on 2008-03-15
It depends on what you are looking for as a user. Linux offers the freedom that doesn't come with proprietary software. However, you trade it off for commercial support and application variety. I honestly don't need any support especially if I have to pay for it. Most people who use linux are independent and/or use it for evaluation or freedom for development. 

I've used Ubuntu for several months (6.06-6.10) and switched to Fedora and have been using Fedora ever since. I don't have any other OS and have been using Linux as a primary OS happily for almost a year now. So for someone like me, Ubuntu is good for a lot of things, especially since I don't use anything else. 

Sure Windows and Mac can do everything Linux can do, and more. But, the freedom that is a given with Linux IMO is worth much more than whatever MS and Mac have over Linux.

Edit: For the last few months I haven't been anything more than your average computer user. I don't develop and I don't do anything professional. I use Linux as a primary OS to get thru life and for entertainment. It works great for me :D

Edit2: Honestly Linux will never change its major course. There will be a day when it is more mainstream and will have easier, and possibly, more attractive interfaces, more support, better software, and compatibility. However, It will never be a direct competition for Mac or Windows. Linux is an entirely different world. If you don't see the point to Ubuntu/Linux, then you may never see a point. If you feel like Mac and Windows have more to offer then use Mac and/or Windows. Linux was designed for freedom, not competition.

### Post by popch on 2008-03-15
What do you mean, 'What is Ubuntu good for'? Ubuntu is a package containing an OS and a fairly large number of complete applications.

Hence, your question translates to 'what are an OS and applications good for'. You need those to make a computer perform any function more involved than a letterweight or a doorstop.

Once you realise that there are 'products' which can satisfy the needs of most computer users, you could actually ask what some of the overpriced equivalents are for, such as Windows or MS Office.

Ask any of the large computer manufacturers who start selling budget PCs with Ubuntu or other variants of Linux what Ubuntu is for. 

> **edmondt said:**
> When I think Windows I think of the selection of software and hardware support.

That's aptly put. Think of the selection of hardware that is supported by Windows and compare it with the selection of hardware Linux in general and Ubuntu in particular supports. There is considerable overlap, but they are not identical.

Ubuntu=total control. :)

Windows can be such a drag with maintenance, so can linux if something goes wrong.

At least with linux, you have some control and ability to solve your situation with your own resources and as an online community, Linux content is voluminous. You can have tiny linux variants that exceed at being small and offereing the most features.

Getting that old machine working with linux is a real possibility.  

You may have a solution for an XP install that works as well, but is it as well **supported and documented** as the equivalent linux solution to the same problem?

What is Ubuntu good for?  It's good having ownership of a system that is essential for your daily needs, the container of very private information, the access point to the 'World', which is not always a nice place! :)

One word, control...
