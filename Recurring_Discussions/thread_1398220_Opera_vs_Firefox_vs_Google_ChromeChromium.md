### Post by I_luv_saber on 2010-07-13
Opera, hands down.

I have problems with this line of thinking.

First of all, speed is great. Faster speeds are greater... but only to a certain extent. There is a saturation point at which increasingly fast speeds have diminishing returns in terms of factors in making a browser choice.

If I can get up and make myself dinner while waiting for a page to load in one browser, and the other browser will load the page almost instantly, then obviously I would pick the browser almost instantly. But if two different browsers load pages almost instantly and one happens to scientifically load pages faster and the difference in page load speed is almost imperceptible to the human eye, then I see no compelling reason to switch to a "faster" browser that is .00001 seconds faster. I can't tell the difference between .00001 seconds and .00002 seconds.

This is similar to the whole "iOS has three times as many apps as Android does, so iOS is better" fallacy. If iOS had 300 apps and Android had 100 apps, maybe that would be true (even then probably not), but at 300,000 and 100,000, who cares? I have not looked at 1000 different apps, let alone 100,000 apps. I have about ten apps total installed on my phone. As long as I can get the apps I want, I don't care if there are 200,000 more available on another platform.

Same deal with browser speed. If I can get the speed I want on Firefox, I don't care if I can shave a fraction of a second of the page loading time in another browser. That isn't a compelling reason to switch.

Add to that that Chrome (for me at least) has generally been pretty fast at loading pages but occasionally randomly hangs on some pages, and then I have to reload to get the page to actually load (this has happened to me in OS X, Windows, and Ubuntu). Any gains I might have gotten in speed in Chrome immediately are lost if I have to reload the page to get it to load completely.

I don't think any browser is empirically "better" than any other browser, but I use Firefox because it behaves the way I want it to. I can easily configure via about:config so that new tabs will load to the far right and not just next to whatever I have loaded, and if I undo a close tab, I know it'll restore to exactly where it was before (if you install the Chrome extension to load new tabs on the right, any undo you do will pop it to the right, regardless of where it used to be when you closed it).
