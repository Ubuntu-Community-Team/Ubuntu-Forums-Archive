At the begining of this post I wrote about how I discovered scripts, if your a newbie like me you might enjoy this.
1.)Open a terminal, and type;
        cd /usr/bin
2.) then type;
        vim fu
note that vim is a command editor and fu may be replaced by anything you would like
3.) copy paste this in

#! /bin/bash

clear
echo "Hello your name!"

echo "We are:" 
date
echo

echo
cal

echo "uptime:"
uptime
echo

echo "who is logged on and what are they doing"
w
echo

echo "Snapshot of current processes"
ps
echo

echo "CPU Temperature"
sensors
echo "Your hard drive looks like this:"
echo
df -h /
echo
du -hs ~/Documents
du -hs ~/Pictures
du -hs ~/Downloads
echo

echo "Ipv4 network (Address Resolution Protocol)"
arp -v
echo

4.) save and quit (esc button, :wq)
note the esc button changes it from insert mode
the ":wq" means write quit

5.) chmod +x fu
note this makes the file executable
// fu is whatever you named your file

6.) type fu or whatever your command is, in the terminal
  and you get calander, system temp, disk usage etc...
one command to sum up everything.
