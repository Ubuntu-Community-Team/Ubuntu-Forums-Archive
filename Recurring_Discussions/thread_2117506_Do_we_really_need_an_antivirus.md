### Post by Paqman on 2013-02-21
> **ale88.. said:**
> What can a malware do without the password ?

Malware tends to exploit weaknesses in the software. What most will try to do is use an exploit to take on elevated privileges so that they can act as root without having to supply a password. The way to defend against this is to make sure your software is up to date, so that any known exploits are patched. Adding an antivirus package would do very little to improve security if all the software is fully up to date.
