A Linux forum seems like a funny place to ask whether or not to switch back to Linux.

Everything aside, I believe a dual boot would be your best option. There's always [Wubi]("http://wubi-installer.org/") if you aren't too technically inclined and just want to do an easy install, although I wouldn't recommend it if you're willing to go through with the more complicated dual-boot process.

At any rate, I've done the same for years. Windows is only on my HDs for gaming. I simply find Linux to be superior for everything else.

The nice thing with Ubuntu is that you can experiment. If you do try Kubuntu, you can install the Ubuntu packages and have Gnome if you decide you don't like it.

At any rate, I believe that Ubuntu is, at least, reasonably stable. At the same time, it has great support (as is evidenced by this forum). Simply put, I believe that unless you're trying to run some very critical applications, Ubuntu or one of its derivatives should be fine for you. Clearly, many other users agree with this sentiment. People wouldn't go from Windows to Ubuntu if it didn't offer drastically better stability.

If you need help, remember that this forum will always be here, and many will be waiting to help you.
