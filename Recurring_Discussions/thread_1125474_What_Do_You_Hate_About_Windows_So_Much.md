### Post by donovan1983 on 2009-04-14
I don't hate Windows. It works great for many people. I just personally can't stand using it or maintaining it. I want a system that I can customise extensively and that I don't have to fight with. At the same time, I want something that just works. Being up-to-date certainly helps too.

My latest version of Windows is XP Professional and it shows its age very badly. As much as various Linux systems' hardware support is put down, more stuff works out of the box for me in Linux than Windows. I don't enjoy having to create custom, hardware specific installation CDs for Windows XP that includes such rudimentary things as SATA drivers. Of course each system requires slightly different drivers so I have to create new CDs for each. And even if I slipstream the latest service pack into it I'm still stuck downloading hundreds of megabytes of updates. Been there, done that far too many times over the years.

Vista is admittedly a lot better and Windows 7 seems to further improve upon it. They are leagues better than XP. Windows 7 has an excellent UI that both combines the best ideas from various operating systems and can teach those other ones a few lessons at the same time. Driver installation is improved over XP and the system is overall more responsive. But I don't like legacy features like drive letters, the primitive shell (PowerShell is interesting, but it's no bash), its slow recognition of USB devices being plugged in, no way to put your home directory on a separate partition upon installation, and a still very kludgy and primitive file system layout (and the registry is just an awful thing).

I will not complain about the security in Windows. It is one of my pet peeves that security is a constant complaint about Windows. I have never, ever gotten infected by a virus in Windows. I rarely used a firewall nor antivirus software. A properly patched system just using the Windows automatic updates is not all that vulnerable. Most security problems in Windows are caused by what sits between the chair and the keyboard. The XP security model is seriously flawed (running as administrator by default, etc.) and Vista's approach a bit too extreme (Windows 7 addresses this) but not using Internet Explorer, not clicking on random email attachments, and not installing software from shady sources is enough to keep your system clean. I know, I did it for years.

The same user who downloads "biginternetworm.exe" and runs it in Windows will likely happily download "deletemyhomedirectory" and run it in Linux. Except Windows Vista or Windows 7 (and maybe even XP SP2 or above) would likely warn the user before running it, while Linux won't since it's fully within its permission system. And for those who respond that the user would have to do a "chmod +x" or something first, so long as the MIME type was correct on the server it was downloaded from the correct modes will be already set. No system can fully save the user from themselves.
