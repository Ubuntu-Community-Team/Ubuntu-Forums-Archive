]n example, Let's suppose one can never connect to the internet using wifi. Assuming the postee did any research, The post should be entitled: No internet, Wifi, chipset & computer make.

If everyone posted like this, then all related post would be in the same category, organized by problem, hardware/software and then system. This would make it so much easier to find solutions to problems. This would also reduce the number of repeat/duplicate post. There is so much that can be done with organized information.
