### Post by SeijiSensei on 2012-01-11
Moonlight does not include the proprietary "digital-rights management" code that Microsoft includes in Silverlight.  Netflix uses that DRM software to control access to its streams.  So the short answer is no.

Amazon's Instant Video service uses Flash and plays just fine in Linux.
