### Post by Jestersage on 2009-07-18
First off, I know everyone are already familiar with that "Linux is not Windows" Mantra. I accept that. However, what about MacOSX? Can it be the ideal Mac replacement?

Personally, I think it can. Aside from sharing underlining similar architecture, I also find that KDE, and to some extend GNOME, is similar to Mac's UI. Also, Mac seems to be getting increasingly paranoid, to the fact that it feels like I am sudoing every single time.

Another thign I notice is that many Windows users I know say that they do not like MacOSX's UI... which is, to be frank, very similar to KDE

The only issue is that the instruction given must be in GUI, which is the real challenge (as most instruction in Ubuntu is given in text command, even though there is a GUI alternative)

Thus, I propose an experiment as follow:
a) Get somethign that is fairly cheap but powerful (I scored a good P4 HT-64 w/1G for 100, and a Sempron w/512 for 30)
b) Load up Ubuntu Studio/64 Studio, with KDE if possible. Load up a good theme
c) Get everything working!
d) Send it to someone who is used to Mac.
e) When providing assistance, must never go to terminal (and if it does, experiment is an auto fail) Synaptic is acceptable, however.

### Post by ahmatti on 2009-07-20
I actually moved from using Linux on work computer to Mac after last year and I couldn't be happier :) In principle I'd like to use Ubuntu, but in practise better support for commercial software is what makes me more efficient with a Mac and that is what counts when you do your work. I work in research and I have to admit that I need MS office for my work, also a lot of apps that have Linux alternative work better on a Mac (everything Java based e.g Matlab, Maple). Because OSX is based on BSD I also the full power of *nix system and can use my favorite open source stuff such as R and Emacs etc. Also I need the compatibilty with Windows using colleagues and I find better on a Mac.

At home I still use Ubuntu almost exclusively and if I did just programming projects and web surfing + Latex documents I would be more efficient with Ubuntu.
