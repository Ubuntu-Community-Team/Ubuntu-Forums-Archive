### Post by Welly Wu on 2012-09-30
I am using GNOME 3.5.4 late tonight. I will use it for this upcoming week to learn more about its new features and I hope that I will be able to update to GNOME 3.6.0 sometime in the near future.

GNOME has a few annoying features. I have to constantly use the top left hot corner to access my windows or applications or virtual desktops. The other annoying feature is that the notifications require me to click on each one to get rid of my notifications.

I removed apparmor-notify which caused me a lot of grief. It constantly told me about another apparmor denial message in Mozilla Firefox. So, I removed it and I am back to a clean desktop environment. GNOME is much easier to use now without being bombarded by notifications.

GNOME is GNOME. I don't see a lot of new and useful features that stand out other than the top left and bottom right hot corners. This reminds me of Windows 8. There is no start button or orb or icon to show me the menu system. GNOME 3.5.4 is like a tablet interface for desktop users. I hate that garbage on my Linux desktop. I know that I am going to upgrade to Microsoft Windows 8 Pro 64 bit, but that will be inside a VM Ware Workstation 9 64 bit guest virtual machine that I can power off.

My Zinio Reader 4 software application works properly. Thank goodness. I read a lot of digital magazines so this is an important software application that must work all of the time for me.

To be quite honest and truthful with the community, I just don't see GNOME growing on me at any time in the future. It's fundamentally flawed for a desktop environment designed for tablet form factors rather than traditional PCs. There's too much visual eye candy that doesn't do much useful stuff. It takes effort and work to scroll through the long list of installed software applications or apps. Switching virtual desktops is requires that I use the top left hot corner and lots of mousing around and more mouse clicks.

GNOME is buggy. It has not crashed on me yet, but there is a long list of known issues that still need to get fixed. For the most part, I don't notice these bugs or regressions.

GNOME is ugly. It is not pretty and it is not beautiful. It's functional without the form.

These are just some of the random thoughts and impressions that I have about GNOME tonight. We'll see how I feel tomorrow morning after I wake up from sleep. I have a feeling that it is going to be a very long week ahead while using GNOME.
