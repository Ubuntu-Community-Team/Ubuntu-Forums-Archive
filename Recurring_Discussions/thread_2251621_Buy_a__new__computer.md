### Post by TheFu on 2014-11-05
Any APU-based system will not provide great results for general purpose computing.
These are generally fine for watching videos, since the video decoding happens in hardware using the video drivers.  Flash will not be great. Firefox with too many windows will not be great.

I'd find a refurb from Microcenter if I needed something cheap - any Core i5 will be a better system. Heck, there are Core i3 systems for less money (but they don't include a monitor).  Used 20" monitors should be $50-70.

If you want a video-watching machine - just for h.264, get a raspberry Pi for $60 or .... 

Or if you want a more-normal desktop [http://www.pricewatch.com/price/computer_systems_with_os/i3-550](http://www.pricewatch.com/price/computer_systems_with_os/i3-550) can provide some ideas.

Is there a specific reason you'd like an all-in-one?  Don't want to allow upgrades?

### Post by TheFu on 2014-11-06
Guess I wasn't clear.  All-in-one systems charge a premium for the form factor. Nothing wrong with that, there are just more cost effective solutions.  If an all-in-one is required, fine.  Just intended to point out that for THE SAME MONEY, a much more useful, general purpose, suitable for desktop AND videos, machine could be had.

I've had a few APU-based systems over the years. Watching a movie on one right now AMD-E350.  It is older and without the GPU and drivers, it would never handle hidef.  It doesn't handle flash well and it doesn't do most desktop things well.

Watching youtube is listed as a requirement.  I wouldn't get any APU system for that. Others have different opinions. In the next few years, we will see new video formats with much higher resolutions (4K). Why risk not being able to handle those with an under powered box?
