### Post by Vaphell on 2012-11-25
k/x/l/ubuntu are versions of ubuntu with different desktop environments slapped on top. These desktop environments vary in terms of bling and features 
ubuntu = unity (ubuntu team's own creation)
kubuntu = kde
xubuntu = xfce
lubuntu = lxde
you can also use gnome shell.
from lightest to heaviest
lxde > xfce > unity/gnome/kde

It is possible to have all these desktop environments available on a single installation, installers merely set up their default desktop environment. Installing another DE is a matter of adding few packages from the standard software center.

how the DEs look and function - you can go to youtube, there is plenty of clips cshowcasing them.
Pretty much all DEs are themeable 
eg check
gnome-look.org
kde-look.org
xfce-look.org
and google in general

### Post by Aaron Christianson on 2012-12-01
Sorry! I forgot about this thread. If you're still out there, This netbook should be able to handle any linux distro and provide reasonable performance, though you will want to stick with a 32bit, given the RAM limitations (64bit takes twice the ram for everything). Even the heavier distros (like normal Ubuntu) should be reasonably fast at these specs, though you may notice a little lag here and there. Linux Mint seems like an interesting choice, and everyone says that the its new Cinnamon desktop is very fast. CrunchBang is also a good option, and the community around it is fantastic.

There are much lighter distros out there which you could try if you are very persistent and technically inclined, but they are not generally considered to be newb friendly. Gentoo can be optimized to the hilt, but it is generally considered to be the most difficult distribution to install and maintain (both of these things, high performance and difficulty, come from that fact that you compile everything from source locally).
