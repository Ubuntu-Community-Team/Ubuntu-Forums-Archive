### Post by Paqman on 2012-10-13
Depending on what your criteria for "Ready for the desktop" are: both yes and no.

Is Linux suitable for a wide range of desktop use cases? Yes.

Is Linux suitable for every single use case that Windows is? No. There are some specific gaps in the software catalogue, which is hardly surprising really.

So, bit of a pointless question IMO. "The desktop" covers so many diverse uses that it's impossible to say anything coherent.
