### Post by p_quarles on 2008-02-19
Moved to Recurring Discussions, since this comes up a lot and boils down to personal preference. Yes, the support for Kubuntu on these forums is fine. KDE isn't the main focus of the Ubuntu project, so it lacks some of the eyecandy features of the GNOME version. It's easy to try it out to see if you like it. Just a simple terminal command:
```
sudo apt-get install kubuntu-desktop
```
You can then load a KDE session from the options menu at the login screen.

### Post by aysiu on 2008-02-19
Yes, it is worth trying out Kubuntu, but the support is not equally good. The vast majority of documentation and user support is for Ubuntu, not Kubuntu. Nevertheless, you should be able to use Kubuntu just fine. It's a matter of good support versus great support.
