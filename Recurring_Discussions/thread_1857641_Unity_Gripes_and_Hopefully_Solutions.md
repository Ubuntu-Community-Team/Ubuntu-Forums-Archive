A lot of these may be customizable. But I've crashed enough operating systems, and accidentally destroyed my previous Ubuntu installation, trying to fix problems. So I am a lot less eager to customize the desktop environment than I used to be.

Both the scroll bars and the side panel are supposed to pop up when needed. But anything that's supposed to pop up when needed is going to behave erratically. Second, the side panel. Same basic issue.

Third, right now I'm running Firefox. I can access the Firefox preferences, but I'm not sure how to get from here to the Unity preferences to see if there is any way to disable the side panel.

In fact, I'm at a loss as to how to fix anything in Unity w/o acccessing the Ubuntu preferences. Or how to open the other software I was planning to use. I was hoping to get some work done in Unity to get some perspective, but I can't figure out how to get the work done.

The upper left of the screen opens the shortcuts menu. It allows me to search for files to find the ones I'm working on. Unfortunately, I often have two files with the same name, so I need to know the file-system location too. It sets media and internet apps as the default options. I only rarely use the media apps included.

I'm not sure how to switch between apps in Unity without using the sidebar. And frankly, I'd rather not have to use anything that pops up like that. It effectively makes the left side of the screen unusable because it might pop up.

The sidebar includes several apps. I use three of them if we count the software center. After opening my Home Folder - I actually keep most of my documents on another partition, and relatively few in my home folder - I can change the preferences for the folder windows. But I still can't change the preferences for the shortcuts menu or the annoying sidebar.

I can open two of the files I was planning to work with. But it's difficult to switch between the two files to compare them and keep them consistent.

So... right now, I have a lot of specific problems in Unity, and I don't have the solutions I'd expected to find. I can't find Unity preferences or any equivalent, to turn off the sidebar, to enable a less aggravating way of switching between open documents, or to put the folders I need in the shortcuts menu.

### Post by MarjaE on 2011-10-11
Okay. I'd like to offer some suggestions:

NO ANIMATION!

Having the sidebar pop up is a major issue. Either allow users to get rid of the sidebar, or allow users to keep it in place at all times, or allow users to switch settings to make it visible or invisible. Any of these would work better than having it appear and disappear when the mouse wanders there, especially with the delay and stutter involved.

No Accidental Maximization.

Having windows maximize whenever we try to move them out of the way is bad enough. Having windows maximize and having the sidebar is just asking for trouble. It's important to let users keep their windows and the sidebar out of each others' way. Apple can get away with its dock, which many users put on one or the other side, because Apple doesn't maximize windows willy-nilly.

Program Menus and Unity and Ubuntu Menus shouldn't get in each others' way.

Most Ubuntu users seem to be familiar with Gnome and Windows which put program menus at the top of each window. It keeps program menus out of the way of system and desktop environment menus. I think MacOS and Unity are more elegant here, but MacOS keeps the Apple Menu in the corner. Unity puts the shortcuts menu there, and makes the Unity desktop menus disappear once anything else has been opened.

I for one need a window-switcher like the Gnome Dock.
