### Post by kherring7383 on 2011-11-05
As with many, I too didn't like Unity either in 11.04 so I stuck with Gnome 2. But now that I've installed 11.10, I find that Unity real isn't that bad if you customize the icon size and transparency with the Compiz plug in. For me personally, I use Docky at the bottom of the screen for all my favorite apps and have customized the Launcher for access to system apps like Compiz, Gparted, Ubuntu-Tweak tool and others I commonly use. If you still want a gnome menu, check out Cardapio and add to the Launcher. Then you can access all your apps from a Gnome menu.
