### Post by kaldor on 2011-10-16
> **judderwocky said:**
> Any suggestions will be considered. Thanks.

Grab [_Scientific Linux_]("http://www.scientificlinux.org/") 6.1. It's a debranded and free clone of Red Had Enterprise Linux intended for use in production environments. You'll need to learn a little bit,  but in the end it might be worth it. Software will be slightly outdated, though. Don't let the name fool you; it's only named that because it's used in CERN and FERMI's labs.

Other option is to use Ubuntu LTS. If you need stability then use the latest LTS. Right now, that would be [_Ubuntu 10.04.3_]("http://releases.ubuntu.com/lucid/"). 

The normal Ubuntu releases shouldn't be used if you require absolute stability. If you're dealing with important stuff, then use one of the options I listed above.
