### Post by rosencrantz on 2011-05-02
I agree that Linux for beginners can be a bit steep, although Ubuntu has increased the standards of user-friendliness a lot.
However, a lot of what you complain about is along the lines
'Ubuntu is not intuitive', i.e. not like Windows. This is never going to change, and for good reason. 
Take software installation. With a Windows administrator account, you can install software from whatever source with a simple double click.
On Ubuntu and other Linux distributions, software is primarily installed from a central repository through the Software Centre, where you should find applications for every need of the standard user (I think you can also get VLC there, from the partner repositories).
Sometimes, of course, you need a package not in the official repositories. In that case Ubuntu expects you to be a slightly more advanced user (which makes sense) and you'll have to tackle downloading and installing .debs or extracting and compiling tar.gz packages.
The same goes for all the password prompts. Each time you edit something that's not in your home directory you are expected to show your credentials. This might be a bit cumbersome, but it makes Linux incredibly safe - ever wondered why no one uses virus protection around here?
In summary: this system is not aimed at nerds. Stick to the defaults, they are no less intuitive than Windows would be for beginners. And take some time to play around - the solution to all standard problems shouldn't be more than a mouse click away.
