AFAIK, Microsoft's license requires that you have Windows Pre-installed. However, when I tried the student upgrade option from 32-bit XP to 64-bit 7, at one point I accidentally overwrote my XP partition, and for a time I had no XP to upgrade from. So, I believe that if you boot from the upgrade disk, and use the custom/clean-install option, you'll probably not even need a windows previously installed. Of course, you'll have to be willing to violate the license. BTW, there is a problem if you choose to electronically download the 64-bit student upgrade and try to unpack/burn this image yourself from anything other than 64-bit Vista or 7; a simple Google solved this problem for me.
