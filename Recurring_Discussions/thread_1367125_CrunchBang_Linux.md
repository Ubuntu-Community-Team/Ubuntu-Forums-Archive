### Post by XubuRoxMySox on 2009-12-29
Xubuntu Karmic runs faster than Crunchbang on one of my computers, but it's the other way around on a different computer! Mileage depends on hardware.

Crunchbang is awesome. There's also a "Crunchbang Lite" which is "lighter" simply because it has fewer applications installed by default. Openbox is stark and minimal, but versatile, powerful, and beautiful in Crunchbang.

It's a hardware thing...

Try a Crunchbang CD and enjoy!

### Post by snowpine on 2009-12-29
CrunchBang is a fantastic distro; it's my favorite "out of the box" Linux experience. (If I have the time, I prefer building my own from a minimal install.) The application selection is very thoughtful, and all the codecs are pre-installed, so it's really "plug and play."

It is based on 9.04, so all of the applications are 6 months older than 9.10, not a good choice if you need the latest features, but a great choice if you like stable, well-tested apps. 

Firefox is Firefox no matter which distro you use.

### Post by snowpine on 2009-12-29
> **~sHyLoCk~ said:**
> How is fluxbuntu different than crunchbang? [besides the fact that one uses fluxbox and the other openbox] Which one is lighter?

Fluxbuntu is a dead distro (latest version is 7.10 release candidate) whereas CrunchBang has a 9.04 release and a very active community.

Fluxbuntu was a bit "lighter" in its default configuration, because it lacked certain features we take for granted today, like Firefox and Network Manager. :)

In terms of Openbox vs. Fluxbox, my understanding is that the difference is negligible (maybe 2mb at the most).

In my opinion, the best "successor" to Fluxbuntu is AntiX, which uses Fluxbox (and/or IceWM) and is based on SimplyMEPIS/Debian Testing.
