### Post by pissedoffdude on 2012-10-11
It's going into the direction of a philosophical question! LOL

Haha, it is a pretty tough question to answer.  From what I understand, a shell is a user interface.  For example, bash and openbox (command line & gui) are shells for the linux operating system because they allow the user to interact with the linux kernel.  So then, from my understanding, that makes window managers and desktop environments GUI shells that depend on X (at least in the case of linux!)

Now, where it gets more confusing is the difference between desktop environments and window managers.  Basically, I understand it as desktop environments DEPEND on window managers, file managers, and other libraries, like KDE depends on its window manager, Kwin, etc.  In the case of Unity, it depends on compiz.  But Unity doesn't have its own libraries, file manager, etc.  Instead, it depends on GNOME and its libraries, so that's why it's a shell for GNOME, rather than an independent desktop environment.

### Post by grahammechanical on 2012-10-11
Unity is one of those things that is greater than the sum of its parts. The purpose is to have a unified (hence the name Unity) user experience across a range of computer type devices.

This is why it is describe as a technology.

Drill down into that web page and you get this:

[http://unity.ubuntu.com/projects/unity/]("http://unity.ubuntu.com/projects/unity/")

and this:

[http://unity.ubuntu.com/projects/appindicators/]("http://unity.ubuntu.com/projects/appindicators/")

and this

[http://unity.ubuntu.com/projects/notifyosd/]("http://unity.ubuntu.com/projects/notifyosd/")

As the first page says - it is a range of technologies. We now have mobile/cell phones that have hardware that is as powerful or more powerful than a desktop PC.

See this:

[http://www.ubuntu.com/devices/android]("http://www.ubuntu.com/devices/android")

Many people are using computers without being aware that the device they are using is computer technology. And they do not care that they know little or nothing about the device other than it looks good and does what they want.

In the same way that PC designations have little meaning to the millions using Android devices. So, if Ubuntu is going to be used in these types of computer devices then old style desktop PC language and designations will not have any meaning to the many people (hopefully) using these Ubuntu devices.
