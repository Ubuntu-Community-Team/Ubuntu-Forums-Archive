Try xcompmgr.  It's a really simple compositing manager.  On cards that won't run Compiz I've been able to get Gnome-Do and other compositing working.  To install, do this:
```
sudo aptitude install xcompmgr
```
Then, go to System->Preferences->Startup Applications.  Add an application with Name "Compositing Manager", Command "xcompmgr", and Comment "Shiny!"  That should make it work for you.
