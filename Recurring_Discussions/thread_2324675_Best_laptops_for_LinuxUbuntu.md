I'd like five dollars for every discussion similar to this I've seen here. Not support and has been asked a zillion times previously, let me count the ways. ;)

Your main issues are generally wireless and video. Find out what the machine you are interested in is using then check how other Ubuntu users are going with it. The only other thing that can be a problem is installing because of manufacturer BIOS tweaks to do with UEFI. Most, if not all, of these can be overcome with some massaging and tweaking.

Do a web search for the machine and append 'ubuntu' to the end of it. Do a search for machines that are compatible with Lubuntu then post what you think might suit you here for feedback rather than 'what machines work with this OS?'. For insance, you say:

> I am looking for an extremely high quality laptop to put Ubuntu on.

If the extent of your 'search' is posting this thread you are reducing you chances considerably.

The simple answer to what high-spec machine works with Lubuntu: probably hundreds. What suits your requirements is more to the point. As I say, there are a ton of posts and websites discussing questions which don't have a definitive answer, like this one. There is no 'best' laptop for any OS. There is a laptop that the OS works well on and suits your needs, and only you know what that machine is. 

Plenty of info here and out there already to start digging through. The first step would be to write down what you want to do with the machine. (If you are intending to surf the net, watch a few movies and check emails, the specs you have nominated are killing a mosquito with an elephant gun and a waste of $$$.) 

Best laptops for Lubuntu? I have Lubuntu on a RPi. Not a laptop, but it runs faultlessly. Does that make it the best? Lubuntu is designed to run on old, low-spec machines and up. It will run on just about anything. The wireless card, GPU and BIOS a machine uses should be your main concern IMHO.

Good luck.
