### Post by LaCone on 2011-01-13
Linux Newbie Point Of View

Hi.
I am a linux newbie, for most of the time, I'm in windows world, not as an expert, just a regular average user, my linux experience began when the windows world start getting invaded by all kind of malware, and I have been attacked so severely that I have to reformat and reinstalled my windows system, did I abandon windows completely and migrate to linux? I wish I did, but I didn't, why? here's the story:

1. First thing of all, I have to decide, the choice. in Linux world, to choose, can give a newbie a headache, examples: .deb based vs .rpm based; gnome vs KDE; rolling release vs interval release; 6 month fixed interval release vs no fixed interval release; upgrade vs fresh 
install; and finally which distribution is the best for me.

2. so to make thing uncomplicated, I search for, a distro that have made top ten in distrowatch, and as user friendly as possible, so i choose ubuntu.

3. because I have such a slow and expensive internet connection, downloading a 700mb iso files is not an option, no friends of mine interested in linux so no one to borrow, so shipit from ubuntu is my choice.:D

4. my ubuntu cd came, I installed, dual boot with windows xp, I'm happy, well, not for long, you see, linux is so much different when you talk about applications, all applications are in some place they called, a repository, and each distribution also each version of the 
distribution have different repository, and its frozen, that's mean if you choose distribution x version y, and it has firefox version z, you will stuck with firefox version z until you upgrade the distribution version!

5. ok, I'll try to live with that, so I explored the repo, and get interested in a 3D game, I click apply, it need to downloads a long list of dependencies as big as 200mb, ok, I'll give it a try, now remember, I have a slow and expensive internet connection, and I can't go to some internet cafe since they don't install the same linux os I have, and even they do, I don't know how to get all the dependencies, copy it, and installed to my system, maybe you the geek know how to do this, but I'm just a newbie, right?

6. so I hit the ok button, and cross my finger, and after several hour, it get downloaded half of it, and get stuck, because the internet connection went down once, why there is no download auto resume for synaptic at all ?!

7. A DVD repo! that's sound a good news! so I ordered one set of them, I get six DVD repo! OMG!

8. I carefully follow the tutorial how to make it work, and it work! but it was painfull! to install a lausy game, I have to insert, eject,and reinsert the DVD repos more than two dozens times!, because all the dependencies are all sooo scattered! I think I'll break my dvd rom long 
before I even finished the installation!

9. I need to copy the repo! yap, there is a tutorial for that, so I did, and I end up wasting 40Gb of my hdd for that because I have to copy all six DVD repo to make it work!


Now, don't get me wrong here, I don't hate linux, I love it, I allways follow the news, just in case there is some news I've been waiting for, what is that? here it is:

1. a linux distro that came with a stand alone installer for each of application, like windows or mac, not the repo system.

2. if linux stays with repo system, at least linux can frezee the os, but not the apps in the repo, so we can stay with the linux version that already work with our hardware and still be able to keep the apps up to date.

3. if linux stays with frozen repo system, at least please make it organized so all interconnected dependencies are fit in one DVD, so we don't get our dvd rom broken, when trying to install apps off line.

The reason behind my wish list is this questions:
1. If windows is so bad, and expensive, why it still leads the dekstop market share?
2. If linux is so great, and free, why it still only less than 2% in dekstop market share?

### Post by matt_symes on 2011-01-13
Hi

The only real problem you have is a slow expensive internet connection. For that you need to lobby your telco.

I know that is no help for you, but there are work arounds like the DVD set you have. 40GB is a tiny amount on most new hard drives.

The repository is designed to be a central, safe and secure way of installing software. This is different than others, where the software you can download can have all kinds of malware inside it. The repository model is being copied by others as the main distribution channel for software.

You need to choose the best OS for your situation and circumstances. If the small amount of hassle you have to put up with using the DVD means more to you than having a safe, secure and free operating system packed full with software, then maybe another operating system should be your first choice.
