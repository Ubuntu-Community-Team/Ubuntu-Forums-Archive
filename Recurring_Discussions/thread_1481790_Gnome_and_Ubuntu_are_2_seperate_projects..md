### Post by MCVenom on 2010-05-13
> **ubunterooster said:**
> Except that Removes package ubuntu-desktop. No problem; I'll just add that back and... Why did it also bring back package evolution :?:
Ubuntu-desktop is a metapackage; it's nothing itself but it has all the software in a default Ubuntu install listed as its dependencies. You remove something in that list and apt thinks you've done the equivalent to cutting the limbs off of the ubuntu-desktop 'package'. So it removes that 'package', seeing as how it's useless now. But since all ubuntu-desktop ever was a glorified list, it doesn't harm your computer one bit to remove it. Meanwhile, if for some reason you must have that 'package' on your system, you gotta take everything that comes with it, period.
