### Post by aysiu on 2009-10-09
Well, I use Windows at work, and it reminds me why I use Windows at work... and Linux at home.

Arguments about what operating system is "better" or "superior" don't interest me (and frankly also seem pointless).

An operating system is a tool, and I will use the most appropriate tool for the job. Since my job involves using Filemaker Pro, pivot tables in Excel (sorry but the data pilot in OpenOffice doesn't cut it), Microsoft OneNote, and Adobe Acrobat Pro, Linux is just not a sensible option. And I don't get running Windows as a virtual machine inside Linux. If I need to use Windows programs, I'm going to use Windows. That makes the most sense.

But at home, I use a web browser, an email client, maybe Skype, and GIMP occasionally. No need for Windows then. Linux is far more appropriate. And I don't even need a full-blown laptop or desktop. A netbook works fine for me as a primary (only, actually) computer.

As for the RAM issue, I've tried out Windows 7 release candidate in a virtual machine with 512 MB of RAM, and it was super-snappy... not as snappy as OpenBox or IceWM, but certainly as snappy as Gnome or KDE in 512 MB of RAM.

I dig Linux. But I don't dig Windows bashing.
