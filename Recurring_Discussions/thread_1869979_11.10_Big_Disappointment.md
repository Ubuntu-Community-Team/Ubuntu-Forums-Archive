### Post by jereece on 2011-10-26
I was using 10.10 and had been upgrading for several years.  Somehow Ubuntu got corrupt so I did a fresh install with 11.10.  So far I am not liking it at all.

1.  The Classic Gnome menu system is gone.  So I followed the instructions on this form to bring it back.  It is not the same because System is still not there.  Plus you have to manually select Classic Gnome every time you reboot or you will have the new system.

2.  I hate the min-max-close buttons on the left.  Before I used a different theme to move it to the right.  In 11.10 it appears that Ubuntu has done away with themes.  I even searched for it and the only item resulting is wallpaper.  There used to be a set of theme's but that's gone.

3.  Evolution is gone.  I exported all my stuff so I have to have Evolution.  Fortunately I did find that in Software Center but it would have been nice to have this installed by default.

4.  I don't like how Ubuntu hides things with no hint there's something hidden.  For example, when I opened Firefox it took me a while to figure out how to close the window.  I did not realize the were no min-max-close buttons are hidden under the top bar.  I finally moved my cursor over the top bar and it changed to the window bar.  I also don't like how Ubuntu hides open windows under the launch icon for that program.    

I am finding it really difficult and frustrating to navigate 11.10.  So I am thinking of moving to Mint.  But if anyone has any fixes for the above problems, I am willing to listen.

Thanks,
Jim
