### Post by Paddy Landau on 2010-09-03
> **Karlof said:**
> But I have the feeling not only from this query that ex windows users are not really welcome, and that despite the overt encouragement to change to linux os,s from windows, you sometimes don,t make it that easy, and welcoming.
I'm sorry you feel that way, and it certainly wasn't intended. I myself am an ex-Windows user.

Windows users are more than welcome. The philosophy behind Linux is freedom, which means we believe in your freedom to choose any operating system you want -- including Windows.

If you read the posts in this forum, you'll find lots of help for people wishing to leave Linux and return to Windows, or people wishing to use both Windows and Linux.

> **Karlof said:**
> I installed the dreaded AVG but it does not show anywhere under applications is this another linux ploy lol or am I being stupid what did I do wrong
A Linux ploy? :lol:

I've never installed a Windows antivirus on Linux (actually, I've never installed any kind of antivirus on Linux). I can't comment on how to use AVG.

As previously posted, it's pointless; it's like putting mole traps into a goldfish bowl. It's just not necessary.

If you really, really want a Windows antivirus, then I suggest you:

[LIST]
[*]Start Synaptic (System > Administration > Synaptic Package Manager)
[*]Uninstall AVG
[*]Install ClamAV
[/LIST]
ClamAV is designed for Linux. It's useful only if you wish to prevent passing Windows viruses onto other people, e.g. if you run a mail server or if you often forward junk emails to friends. If you don't do those, then you don't need an antivirus. Period.
