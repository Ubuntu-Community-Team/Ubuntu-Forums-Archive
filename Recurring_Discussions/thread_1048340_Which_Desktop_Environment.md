### Post by Joshuwa on 2009-01-23
I'd say try them all. Trying just one, and not liking it, could end up causing you to become frustrated and not like Arch, rather than disliking the desktop environment.

I'd also suggest trying KDE 4.x *and* KDE 3.5. They're both KDE, but behave different enough that it is possible to despise one and enjoy the other.

Openbox is nice. If you're going to Arch for the lean, minimal, and totally customized setup, it just could be a good match.

### Post by SomeGuyDude on 2009-01-24
I run Compiz by itself. Imagine all the eye-candy with OpenBox's simplicity. :) Specific perks?

2) Light on the system generally (FAR less than GNOME did).
3) Great menu system if you install it.
4) Pretty easy to configure.
5) Uses GTK, which I prefer.

Sure it's slightly more CPU intensive, but I'm not complaining.
