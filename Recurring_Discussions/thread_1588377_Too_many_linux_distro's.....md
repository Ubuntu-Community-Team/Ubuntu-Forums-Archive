### Post by 3Miro on 2010-10-22
Great diversity and natural selection, this is the power of Linux. If you fix things to 5 - 6 distros, you will get stagnation, diversity opens the doors to innovation. Furthermore, distributions always share code and ideas, if an idea or code is good, it doesn't matter if it first appears in Ubuntu, Fedora or Obscure Penguin OS, it will soon get implemented everywhere. It is true that there is a lot of waste in the sense that many ideas aren't that good and don't catch on, but everyone is given the same fair chance.

The number 5 - 6 is too small. Code is not very hard to consolidate, however, philosophy is much harder. Starting with the commercial interests of RedHat, Novel, Mandriva and Canonical that cannot be reconciled, then you have national interest of projects like Pardus, then you have community based distros like Debian, Gentoo, Arch and Slackware with orthogonal philosophies; consolidation is practically impossible. The best you can do is what you have now, several large families and a bunch of smaller independent distros.
