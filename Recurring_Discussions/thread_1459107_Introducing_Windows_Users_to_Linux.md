### Post by chessnerd on 2010-04-21
I'll give a +1 to Linux Mint. It's elegant, easy to use, and functions similar to Windows in several ways (menu button in the bottom left, notification area in the bottom right, a menu search like in Vista/7). It also has some nice utilities, like its update manager, that are easy to understand.

My only qualm about Mint is that it does have higher system requirements than some distros, so if you are working on older computers (say, less than 512 RAM) you might want to go with a different desktop environment. 

For this, I would suggest LXDE, which is also Windows-like and uses very little RAM. Lubuntu 10.04 looks somewhat promising as an LXDE distro. I've also heard that PC Linux OS is good for LXDE.

Otherwise, I think Mint would be a good choice.

Using Linux for your project is a great choice. You don't have to worry about licensing, it's a solid and reliable system, and it comes with a lot of good software. Oh, and it's also free, that helps too :D. Good luck with your project, I hope you're able to help people.
