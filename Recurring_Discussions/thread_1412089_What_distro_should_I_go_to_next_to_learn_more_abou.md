### Post by squilookle on 2010-02-20
I won't touch Gentoo with a ten foot barge pole. I tried it several times, on several computers and always found that both the live cd and install cd images were broken, and I couldn't even get started. I've also heard of other problem in the community, but I don't know the details. 

Slackware is a great choice for learning more about Linux, and a great distro in general. The only reason I didn't stick with it was package management. Simple but time consuming to install and update anything. 

Arch is a great choice. I've found it to be perfectly stable, I learnt alot during the install, including how to edit alot of the config files. 

The package manager is excelent, one of the best I have used in Linux.
