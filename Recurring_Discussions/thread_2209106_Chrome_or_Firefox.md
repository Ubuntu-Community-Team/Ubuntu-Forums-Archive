### Post by GSF1200S on 2014-03-05
I trust Mozilla five times more than Google, and thats reason enough for me to use Firefox. The addons for Firefox are more able to modify the UI, Firefox can be made to use much less screen space, and Firefox uses much much less memory because it is not multithreaded. I have 16GB of memory so that really doesnt matter I guess. Chrome is a good browser. I dont like its UI personally, but it works well.

Im just holding my breath until Google pulls the plug on Mozilla's income. Once Firefox becomes unusable, Ill switch to Uzbl or Opera out of spite. That day will suck though- Ive been using Firefox since early version 2...

I might be cynical or ignorant here. I think after a while it seems some people just have to hate on something thats established. It doesnt matter how good or bad it is- it has to be hated on. Firefox had a good many reasons to hate it- it was slow, slow, not very stable, bloated, slow and slow. Chrome crushed it on speed, and Mozilla changed Firefox to compete. That was good. Unfortunately, Firefox is eternally stuck with this reputation even though now speed differences are inperceivable. An exception is startup time, at least when not using an SSD drive. 

I honestly dont understand why Firefox has so thoroughly nose-dived in popularity. Its a great web browser...
