### Post by ninjaaron on 2011-01-06
I like Windows 7. It's alright if you uninstall the bloat and defrag regularly. The Aero desktop environment is slick looking and functional, and all the latest and greatest proprietary software comes out there first. I think it's a fine system. Only a few things I don't like: The need for anti-virus software, the somewhat more demanding system maintenance needs, and just when you think you've zapped all the shipped ad-ware, you get another ad for some sort of system maintenance tool. I also prefer the customization options in Ubuntu, but Windows is fairly customizable as well.

I did delete my Windows partition recently on my computer because I never used it after the Ubuntu Flash Plug-in became usable for more than YouTube. It wasn't because it was bad, just cause I didn't need it, and I booted it so rarely that when I did, I would have to download tons of updates all the time, so it got to the point that it was completely useless.

But I still like it. I like most operating systems I've tried, except Damn Small Linux and Slax.

By the way, Ubuntu has nearly as many security pop-ups as Windows, though not so ugly. Tight security is a good thing.
