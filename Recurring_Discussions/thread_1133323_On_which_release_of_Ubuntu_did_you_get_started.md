### Post by phrostbyte on 2009-04-24
I've "used" Ubuntu since Warty 4.10, but I didn't install it until Dapper 6.06. Ubuntu was really exiciting at the time and was quickly gaining popularity in the FOSS community because the promise of an easy to use / install Debian. At the time you had really two choices, an easy to use but inflexible distro (ie Fedora, Mandrake), or a flexible but hard to install uber l33t distro (gentoo [was really popular at the time], or Debian). Ubuntu was really pushing the idea that they will merge the two, be easy to use, but still have the power of Debian. I think they nailed that.

I've seen Ubuntu release after release and I still think Dapper was the biggest as far as improvements in a single release.
