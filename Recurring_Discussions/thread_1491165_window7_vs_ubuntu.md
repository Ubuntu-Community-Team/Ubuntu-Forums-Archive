### Post by crysis on 2010-05-29
i dont know why and in what situation u are asking this. but if u are building a new pc and looking for a new os, then u might give ubuntu a try. it's free and u could save some money. if it doesn't work out then get win 7 home premium later on.

if u r a new pc user or have never tried linux urself, then it's best that u go for win 7 directly. OEM versions of windows are cheap and u can get one easily.


another big consideration is software. windows has better software support. if having good and professional quality software like Microsoft office, OCR software, high-end image or audio-video editing/printing/publishing, Design and engineering software like those from Autodesk, or u like playing latest games then go for windows as there is no linux alternative for them. but u probably dont need these unless u r a university student or working on a company project. 

if u are a beginner level software developer/student then u might feel more comfortable in windows's modular and maintenance free design and with Microsoft Visual Studio.  

if u r a home user and want to use computer for email/internet,  word processing for home/school, book-keeping, play cd/dvd, editing/viewing personal photo/movies. and ur computer usage is routine like, doing predefined specific things only, then windows 7 is probably an overkill, try ubuntu first.
