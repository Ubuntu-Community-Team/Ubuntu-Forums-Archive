### Post by BrokenKingpin on 2009-09-23

I use Linux for the following reasons:
- It installs all my hardware drivers out of the box
- It has most of the software I need out of the box (my prefered browser, IM, Office software, etc)
- It is fast and stable
- It is FREE

Those are just a few reasons. That being said I still use Windows for some development. I wouldn't worry about what other people say, just use whatever you see best fits the task at hand.

### Post by moonfleet on 2009-09-23
I don't see any reason not to dual boot if you have the disk space. You already paid for your XP, so deleting it isn't going to hurt Microsoft whatsoever. May as well keep it, and if you don't want to support MS, don't buy anything else from them. Almost all the major software that runs on Linux has versions that run on Windows too. So, just make your XP installation as "free" as you can. You can easily stop almost all the services that call home to MS, if you want to go that far.

For what it's worth, I keep a dual boot installation in case I might need to do something with my laptop that I might not be able to with Ubuntu. Pretty rare, but it's possible. Laptops have a lot of proprietary, non-standard stuff.
