### Post by slooksterpsv on 2010-01-17
Linux, like all OSes, do have security concerns.

Here's what kind of protection I run on my computer:

NONE

I just did a clean install of Karmic Koala, installed the apps I needed (especially adblockplus for firefox) and I'm set. I'm not worried about viruses, spyware, rootkits, intrusion, etc. Granted I'm behind a secondary firewall so intrusion would be difficult (router). All in all,

Windows = Freak out over security
Linux = Do your updates you're good - no av, no advanced firewall, no antispyware, etc.
Linux Server = You should know how to protect your server before installing a server.
Mac's = Require antivirus and antispyware (apple recommends running two at least) - e.g. McAfee and Avast (run both, both residential at the same time). So Macs you have to secure down like Windows and "Freak Out"
