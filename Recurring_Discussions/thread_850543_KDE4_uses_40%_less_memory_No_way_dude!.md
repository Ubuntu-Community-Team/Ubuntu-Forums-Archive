### Post by jacksaff on 2008-07-05
You miss the point.
Kde4 uses Qt4 which in general uses less memory than Qt3 and it also has more functionality shared between apps so less stuff needs to be loaded multiple times. Run a bunch of kde4 apps in kde4 and then do the same with kde3 for kde3 and see what happens. Each extra kde4 app will have less to load (in general, some will have new functionality, making them bigger) and will load faster than their kde3 counterpart.
The default kde4 desktop has more going on than kde3 and there isn't any sensible way of comparing them. 
You also need to account for the fact that kde3 has had several years of polish and cruft removal.
