### Post by houseworkshy on 2012-04-25
It depends on what you do.  Keep it simple and add as required is probably best.  
Also set up what is there already "sudo ufw default deny" "sudo ufw enable" entered in the terminal are good starts, this gets the firewall running.  
Bung in a virus checker too, the need isn't pressing for linux yet but one will be shareing files like as not.
[https://help.ubuntu.com/](https://help.ubuntu.com/)  is a very good start, learn the basics and it has apt;url links ( these initialise installers ) scattered through it so you can put in some things you like the look of as you go.
