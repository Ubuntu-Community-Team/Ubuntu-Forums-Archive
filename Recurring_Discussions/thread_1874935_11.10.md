### Post by chrissywissy on 2011-11-04
I like it. The Unity desktop takes some getting used to, but I'd had used that in 11.04 and appreciate the tweaks it's had.
 
It's slow to boot (and shutdown) as some others have reported, but once up and running it seems more responsive than Windows. I only have a relatively old slimline HP desktop, with 1GB RAM and a Turion ML34 processor, but it seems ok in use. No difference on my machine between Unity 3D and 2D.
 
Must say it's a step too far for my old Satellite Pro laptop (512MB), so I'm keeping Debian 6 on that.
