### Post by Ahunt on 2010-06-22
> **areteichi said:**
> I remember reading someone's post on this forum that Chrome contains a feature which tracks user data and sends to Google. Do Chromium developers get rid of that feature or is it also in Chromium as well?

The definitive answer to this was posted on the Chromium blog on 2 June 2010 [In The Open, For RLZ]("http://blog.chromium.org/2010/06/in-open-for-rlz.html") which says:

"Chromium will also continue to exist as it always has, without any RLZ library included. And, you can still download a Google Chrome with no RLZ behavior at [www.google.com/chrome](www.google.com/chrome). But now that RLZ is open, Google Chrome distributed through promotional means will include this open-source implementation of RLZ."
