### Post by SunnyRabbiera on 2008-07-18
I find the linux directory easy to figure out, I had a helpful (though outdated) guide on how linux directories worked and they put it in windows terms for the beginner...
/bin = binaries, similar to "program files" in windows.
/home = home directory, similar to my documents in windows.
/lib = libraries of each application, sometimes binaries are found here
/share = shared files and folders... simple enough
/media = the media drives on the computer
/opt = optional
/tmp = temporary files
/sys = system, similar to the "windows" directory
/dev = development
/mnt = mount
