### Post by Armor Nick on 2011-06-13
This is a post to Linux extremists. I know not everyone uses Linux like it's a religion, but there's a lot of blog posts and the like where people make 'unbiased' comparisons of Ubuntu and Windows. However, I've used all major operating systems so far and every single one of them has its pros and cons. Therefore, I have decided to make a post debunking common Windows myths, as a counter-strike to the many similar Linux posts.

**Windows is expensive!**
That&#8217;s true. However, how many of us have ever payed directly for our Windows license? Even if you ignore pirated copies, you still get Windows with your pc. And by the time the support for the current version ends, it&#8217;s time to buy a new computer anyway.

**You pay extra for your Windows license when you buy a pc!**
That&#8217;s also true. However, even though Linux is free, a Linux pc still averages more expensive than a Windows pc. And believe me, I&#8217;ve been to a lot of online Linux stores and I&#8217;ve compared the prices to comparable Windows computers and they cost a considerable amount more.

[B]But we like to support the community!
[/B]
Wait, what?! I&#8217;d rather pay more money for something that Just Works &#8482;. The amount of money you add to a Linux computer is a lot more than the cost of a Windows license.

**Windows takes huge amounts of memory**
This one is actually not true. I&#8217;ve been doing a lot of tests myself comparing the memory usage of Linux to Windows and where Ubuntu 10.10 uses ~650 MB of RAM, Windows 7 uses ~750 MB of RAM. I admit that it&#8217;s 100 MB more on average. However, it&#8217;s only 100 MB more. Of course, that&#8217;s my minimum. Some hackers have gone so far as to get the memory usage of Windows 7 (dubbed Tiny7) down to ~170 MB of RAM. And yes, I realize that Linux itself goes 10 MB or less but really, you still need some functionality as well.


**Windows software is too expensive!**
People often point fingers at Microsoft Office and Photoshop when bringing up this issue. They don&#8217;t seem to know (or ignore) that most open source software has a Windows version. And there&#8217;s even some software on Windows that&#8217;s Windows-only. Imagine that! Not everyone on the operating system with the world&#8217;s largest market share wants to pay for their applications!

**Blue screens!**
I still hear quite a lot of these and I really don&#8217;t know why. It&#8217;s been ages since I&#8217;ve last seen a blue screen. The only blue screens I&#8217;ve seen happened on Windows XP systems. And I hate that ugly piece of bugs.

**Windows has viruses!**
Yeah, but it has the largest market share and is by definition the largest target. Other than that, I have to think hard on when I saw my last virus. I think it was at the time when Windows XP (I hate that thing!) was the dominant Windows. Nowadays, there&#8217;s the Windows equivalent of sudo and the Windows Firewall no longer sucks. Also, Microsoft released their very own Anti-virus application that doesn&#8217;t take Norton amounts of RAM. (yes, Norton amounts of RAM :p)

**Windows is bloated!**
So yeah, Windows comes with a lot of functionality by default. However, it&#8217;s possible to turn off any Windows feature whenever you want. I myself remove Windows Media Player first chance I get. On the other hand, Linux isn&#8217;t bloated, they say? Well, Ubuntu comes with a Video editor, a full-featured office suite and image editor? Mac comes with a Video editor, photo editor, music editor and more? Wow, they&#8217;re more bloated than Vista!

**Windows is too restrictive!**
This is an argument I once read on the Gnome blogs. In an article about the removal of a configuration option. That&#8217;s right, Gnome is constantly &#8216;simplifying&#8217; things, and Ubuntu&#8217;s Unity doesn&#8217;t allow you to move the Launcher. Apple doesn&#8217;t even let you choose your hardware. So how restrictive is Windows? It puts all the advanced options in an advanced tab. seriously, there&#8217;s not a single thing in Windows that you can&#8217;t change, if you&#8217;ve got the time to find where they put the button.

**Linux/Mac has this and that feature!**
In my eyes, this is a feature itself. People like simple things. Virtual desktops, wobbly windows, exposé and others like it actually make the desktop more complex. But even if you simple must have these features, you&#8217;re free to install them yourself. You can mix and match your perfect desktop. You can even install a different window manager if you really want!

**Conclusion**
Again, let me stress that I don't particularly dislike Ubuntu. I sort of like the Gnome Shell and Unity really has its appeal. I just like Windows 7 *better*. 

I could give any number of reasons why Windows is better than Linux, and similar to many so-called comparisons I could pull statistics out of the air, but that would be a load of rubbish. All long-living operating systems have their uses, and none is an objective way better than another.

I will end with the immortal words originally quoted by TotalBiscuit:
> [It's okay to not like things, but don't be a **** about it. It's okay to not like things, but don't be a **** about the things you don't like.]("http://www.youtube.com/watch?v=DDZZj5S7o-w")
