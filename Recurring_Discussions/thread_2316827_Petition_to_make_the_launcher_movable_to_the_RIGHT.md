### Post by rdu2 on 2016-03-11
First of I honestly get that it is an viable design choice to not make the launcher movable to the bottom, because that would defeat the purpose of a launcher - users who want to place it at the bottom are better served with a desktop environment that uses a windows style task bar.

But as it has been mentioned before the ubuntu button that used to be the reason for the launcher being on the left side has since been placed on the launcher itself. see [http://askubuntu.com/questions/33605/can-i-move-the-unity-launcher](http://askubuntu.com/questions/33605/can-i-move-the-unity-launcher)

For another user in need of this feature see also [http://ubuntuforums.org/showthread.php?t=2316646&highlight=launcher+right+side](http://ubuntuforums.org/showthread.php?t=2316646&highlight=launcher+right+side)

Personally I have the exact same problem as the user asking [[this]("http://askubuntu.com/questions/379198/how-to-move-ubuntu-12-04-launcher-to-the-right")] question - a two monitor setup where the right monitor is the primary display thus placing the launcher in the *middle* of my workspace, making it nearly impossible to „find” the launcher items with the mouse because I am used to place the cursor at the screen edge with one swift move and then navigate it to the item I need.

Setting the launcher to reside on the secondary screen is not an option, because I use the secondary screen only 50% of the time so the launcher changes location every time I unplug the secondary screen which is really confusing. Plus the secondary screen is about twice the size of the primary (notebook) screen thus the space on secondary is too valuable to clutter it with the launcher. Thirdly I have the primary screen in front of me and the secondary to the side so the launcher is just too far away from the action if it is all the way to the left. Finally having the launcher on the periphery device rather than on the actual „computer“ just makes no sense in terms of workspace-logic.

I think my workspace setup is fairly common - for example everyone using workspaces with provided monitors at the library of my university *has* to have the secondary (provided) screen to the left. It just seems absurd to prevent everyone who works like that from using ubuntu efficiently.
