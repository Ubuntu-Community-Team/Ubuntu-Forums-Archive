### Post by FFighter on 2008-05-18
What I like in Linux-based OS (specially Ubuntu):
 * The fact that GUI is detached from the kernel
 * The fact you can use any shell you would like
 * bash
 * The simple I/O interface
 * The fact that is is free and open, so you can dig and learn if you have the time (usually I haven't but it is great to have this possibility).


What I like about Ubuntu:
 * Works out of the box
 * Convention over configuration (defaults, only customize if you need/want to)
 * Easy for people coming from Windows to get used to the POSIX world.
 * Good alternative to Windows

What do I do for a living?
R.: I'm a web developer, deseloper-like. I like to design the interfaces, invent the concept, choose the color pallete and also code the software, decide on the architecture, model the objects and data... make the thing work and so on. 

Can I get work done on Ubuntu?
 * Of course. BUT I do have to run XP in VM ... see below:

What do I like about Windows?
 * Works out of the box
 * Flash CS3
 * Photoshop CS3
 * DirectX 10 (hey, I like gaming!)

The point of this post? Well, I think Linux shouldn't try to be Windows, but should try to learn from Windows success. Windows is a great OS after all (XP is pretty stable. I have had many crashes with Hardy in the last few days (sound related, mainly, it seemed to have stabilized, I think becouse of the last updates. But it did made me wonder on many things regarding Linux's stability)

So, Linux should become a good commercial software platform. I'm not a FOSS zealot, I like Free and Open Source software, of course, but I think that good software if commercial, should be bought. Why not?

I really miss the Flash IDE, Photoshop, Illustrator and other creative, multimedia-intesive applications (3ds Max, anyone?). Ok, games I can play on my XBOX 360, but DirectX also allows for an abstraction for applications too, not only games per se. I wonder why such initiatives couldn't be adopted in the Linux world. Maybe Canonical could fund something like this with enough interest from the community?

In my view, the big software corporations maybe, such as Adobe, could start developing/porting their products to Ubuntu, taking Ubuntu as a model. Then, the community could "port" it to other distros, since Ubuntu is the most user-friendly and the only distro that could compare to Windows in the out-of-the-boxness factor :)

What do you think?
