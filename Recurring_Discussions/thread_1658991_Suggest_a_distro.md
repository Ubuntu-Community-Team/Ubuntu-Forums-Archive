### Post by Hur Dur on 2011-01-03
SliTaz is definitely recommended. It's lightweight, it uses around 20mb of RAM idle. It doesn't look ancient though, like many other featherweight distros. I find the default look for it very appealing. As far as packages go, it's repositories are not as large as Debian's or Arch's, but for a small distro, it has quite a few packages in the repository, over 1,500 I believe. 

Other than SliTaz, I recommend Slackware, Gentoo, Arch, and if you really want to learn about Linux, Tiny Core. You have to do everything yourself, all it supplies you with is a GUI and a package manager. No sound system, no drivers, no flash, no web browsing, no login manager, nothing but an app viewer, terminal, and a control panel with a few tools. You also may want to check out Austrumi. I heard it is an excellent distro, but I haven't had a chance to try it out yet.

Debian is awesome. Although you asked for a distro that is not Debian or Debian-based, it actually teaches you a lot about Linux to do a Business Card net install. You have to install everything yourself, including drivers, X, ALSA, OSS, terminal, etc. It is also very light on resources if you want it to be.
