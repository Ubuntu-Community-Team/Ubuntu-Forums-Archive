### Post by jualin on 2008-07-10
KDE programs work in Gnome and Gnome programs work in KDE. The difference is just the desktop environment and the default programs that come with it. The main difference noticeble is the user interface.

### Post by jualin on 2008-07-10
Also KDE is suppose to consume more resources than Gnome. And many people argue that Gnome is more stable, however you should pick the one that you like the most. For Windows users, KDE is more user friendly since it looks pretty similar to Windows. 
From Ubuntu (Gnome) you can install KDE by installing the "kubuntu-desktop" package and from Kubuntu (KDE) you can install Gnome by installing the "ubuntu-desktop" package. After you do this you will have the choice to pick which one you want to use in the log in Window under "Sessions". Hope this helps!

### Post by Canis familiaris on 2008-07-10
Both are desktop enviroments. i.e. the GUI you see in the desktop. 
Yes KDE programs runs in GNOME and GNOME programs run in KDE, though in these case a little slower than in the native desktop environment but acceptable.
Both are very good desktop environments. I suggest you try both of them before deciding what to use. I prefer GNOME because I prefer simplicity but your opinion will differ.
