### Post by lcman on 2011-10-12
So, here's the thing-ever since Firefox 5 has been updated to 6 & 7, Firefox has become really unusable for me! It wouldn't load pages at all sometimes!

I believe that this is some kind of memory issue or something. When I type an address and press enter, let alone loading it, it won't even try to resolve that domain! :confused:

However,if I close Firefox and reopen it, the website might load! Note the word "might" because sometimes, I'll have to close and reopen a couple times like this morning when I was in a hurry and really had to check my email and had to use Google chrome instead because damn Firefox decided that it won't load the Gmail website!

EVERYTHING works- chrome, Opera, Safari, IE and Firefox <= 5.0 but Firefox 7 and 6 won't and I guess future releases will be the same.

So, I send a couple of support requests to Mozilla, opened threads in their support platform but they never fixed that. I got like 80+ people that clicked "I have this problem too" in my thread but Mozilla decided that they'd just ignore that issue for the greater good. I also saw a lot more threads about the issue... Go figure!

Now, the only reason I've been using Firefox is because I have a couple of addons that I really love and can't live without:

1. Lastpass
2. Read it later
3. Unplug
4. Firebug
5. Epub catalog

So, I searched around and found most of these addons for the Google chrome browser. I got Lastpass, Postponer for Read it later and a couple for the others that I haven't tried yet!

So, basically, I've decided not to use Firefox anymore even if they fix the problem because this really made me go mad! Instead, I'm going to use the more stable and better Google chrome because it has many extensions that make it as good as / even better than FF.

Oh btw0, I have this problem on both my Windows 7 and Ubuntu 11.04 machine.

oh btw1, I'm not asking for help, just sharing something! :popcorn:
