### Post by colin.p on 2010-07-25
I have a dedicated computer upstairs for downloading. I use Transmission for quick downloads, and Deluge for queuing up several. Both torrent programs download to the same folder, and only run from midnight to 9am.

I access that computer from my laptop downstairs running 10.04 using remote desktop viewer. If I'm in win 7, I access using 10.04 in VB, again using remote desktop. Quick and easy and saves me from staggering up the stairs.

As far as which one I prefer, I find both have their strengths and weaknesses, the reason I use both.
