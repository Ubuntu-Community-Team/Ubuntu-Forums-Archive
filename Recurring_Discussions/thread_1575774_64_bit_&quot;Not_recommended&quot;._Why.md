### Post by dmizer on 2010-09-16
Mostly because for desktop use there are a few things that don't work without some tinkering, including (but not limited to) browser flash support. Some hardware (wireless) also may not have 64bit drivers available for Linux.

Yes, there are ways to work around these minor shortcomings, but for a new user who's first install of Linux is Ubuntu these problems can send them away from Linux forever. With a few exceptions, 32bit works just as well as 64bit so there's no reason to risk it.

While there are good reasons to use 64bit, for a new user those reasons mean nothing when they can't watch youtube videos. This may seem trivial to you, but it really can make the difference between a user who sticks with Ubuntu and a user who gives up and goes back to Windows or Mac.
