### Post by acelin on 2008-05-01
Ok, so I just spent several hours using Kubuntu 8.04. I have never used KDE for that long, and my experience was limited to Freespire 2.0 last year. 

I really liked the widget feature- very impressive. They looked good for the most part; I just wish the opacity was adjustable.

That is where it ended. 

The panel was ok, but when i set it on small so it didnt take up so much space, the clock widget was only half visible. The K logo would also disappear. 

The menu was awful. It is like the Vista menu + utter worthlessness. Who designed this? It was a pain to use, and very slow. Give me back Gnome!!!

Now about internet. Imagine my surprise that a file browser is also the internet browser- just worthless. Imagine someone using it for the first time, a complete computer noob? I quickly downloaded Firefox in Adept- which did work perfectly. Very quick and easy installation.

Really, K is completely user unfriendly. The file manager looked like Windows ME, and the start menu was a cheap rip off of vista. 

It was a step backwards in time for me. I felt like I was using a computer that was made in the early 90's, only with a few things looking a little nicer. 

It made me feel ashamed to have downloaded Kubuntu, then have wasted a disc on it. 

This is my opinion and experience. 

My conclusion: That we need to step it up. Stuff like this cant happen. 

We need to make Ubuntu the most friendly OS in every way, and our last step is the GUI- and I think it is the biggest step. 

We need tighter looking file managers, intuitive interfaces, unique designs, flexible designs.

We need programs built in that work and are friendly. Ubuntu has already accomplished this. Kubuntu is nowhere close.

We need the full package.
