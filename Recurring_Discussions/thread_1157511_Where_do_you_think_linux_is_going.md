### Post by k2t0f12d on 2009-05-21
> **claymater said:**
> Hey guys, I was thinking about it the other day while I was on my moms macbook, That "Well Macs are kinda "creative" based, and windows are sorta games and "working" based""... What is linux trying to do?

You say *linux*, which by context I take to mean some nebulous idea of the complete GNU+Linux operating system with a desktop configuration including a windows manager and some applications. That isn't at all like *Windows* or *Mac*, because after assembling the constituent parts, a GNU+Linux operating system is not driving any single person's or group or person's vision. Neither Windows nor Mac provide anything like the range of useful applications that can be had with free software, but offer binary compatibility with a very small handful of high-profile programs, usually with additional purchase required.

So there is no way to answer your question in this context, although you can guess at what individual users or distributions are trying to make. i.e. Ubuntu drives a user-friendly experience starting with installation and all the way through to a complete and mature desktop, but this is not what GNU+Linux "is".

GNU+Linux provides the building blocks with which *any* vision can be pursued, not just the wanky PR vomit peddled by a few powerful corporations.

> **claymater said:**
> I know its open source, and is trying to create a free desktop environment.. but is that it?Freedom means the ability to pursue any vision without requiring permission (or where granting permission up front is the status quo). That isn't just *it*. That's *everything*, because everything else follows.

> **claymater said:**
> So where do you think linux is headed?Different software projects that contribute to GNU+Linux develop at different speeds, for different reason, with different leaderships. Further, any of those things can change at any time, and usually do so in an emergent, evolutionary way. GNU+Linux benefits from this by becoming an even more versatile platform with which to build a vision.

> **claymater said:**
> And where do you see linux going?Linux is a kernel program. Linux can be used to drive almost any type of device specifically because it distributes with source, is promiscuous with other code, and is not limited in scope by specialization (although specialized versions can and do exist). I see Linux development continuing unabated until 1) proprietary interests pervert the law to the degree that free software development is no longer possible, or 2) the paradigm shifts and a better computing design succeeds it.
