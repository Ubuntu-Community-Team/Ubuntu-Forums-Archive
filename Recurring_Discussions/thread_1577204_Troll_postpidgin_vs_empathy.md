### Post by murderslastcrow on 2010-09-18
Yeah, the main issue is that Empathy just doesn't get in my face with new messages, which I need it to. I'm not going to click twice to get to something that used to be automatic.

Why should I? Really, think about this from a practical mindset. That's a ton of work for something that used to just happen.

I'm fine with that being the default, but why isn't the option even there? Could it be that difficult to program automatic popping up a new message?

But yeah, so long as Empathy can do all Pidgin can from a practical standpoint, I'm fine with it. But why use something that's inconvenient? My parents have constantly complained about Empathy and asked for something else. People don't like that behavior, they like to be interrupted when their IM is on and set to Online.

But yes, Empathy is still very nice, I don't doubt that. I used it for a week before I gave up on the clicking twice every time I got a new message from a conversation I'd closed. It just doesn't mesh well with me.

That's my main issue with Gnome applications in general. They aren't always as flexible as they probably should be. I don't think any default will satisfy everyone, but if I can tweak it to work the best for me, that's what I want.
