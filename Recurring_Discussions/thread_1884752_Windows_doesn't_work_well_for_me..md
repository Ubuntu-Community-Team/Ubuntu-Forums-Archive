### Post by Gawains Green Knight on 2011-11-21
I haven't had the displeasure of having to really use windows in about 15 years, until today when someone asked me to edit a word file with equations and "track changes". Expecting trouble if I used libreoffice, especially with the equations, I fired up the windows partition on my work computer for the first time to give it a whirl.

I was shocked at how bad windows is. I haven't used it in a while, and everyone raves about how good windows 7 is, but it was torture having to run it. I felt physically drained afterwards.

My main problems were the fact that things routinely don't work. Repeating the same sequence of actions faithfully can result in completely different behaviors (computers aren't logical any more? In fact when things don't work windows will ask you to repeat the same actions again!) Annoyingly unhelpful 'help' windows randomly pop up. Lots of updates and programs just turn themselves on randomly. The computer was slow and failed to correctly detect hardware, on a new machine were everything just worked with Ubuntu. The desktop was cluttered after opening 2 windows, with no workspace switcher. Middle mouse button doesn't work in windows (didn't realise how important this was until it was gone).

I just needed to rant.....
