### Post by tomasrey88 on 2009-08-04
Depends. If you have a dual boot Win/Ubuntu HD, then yes, get antivirus. My setup is dual boot Win/Ubuntu with no internet access for the Win98 partition because it is unsupported and will probably die from viruses in a minute on the web. Then, before I use any files in Win98, I scan it for viruses with ClamAV in Ubuntu. Since Ubuntu Linux is relatively (don't run stuff in root, don't run server if you're a noob, run desktop) impervious to viruses, then scanning for viruses in Ubuntu for the Windows partition is the perfect set-up. As long as you're not looking at porn files or using shareware/freeware in windows, you should be able to keep Windows virus-free.

Some people like to use antivirus to keep from accidentally forwarding files/emails to friends with a virus on it.
