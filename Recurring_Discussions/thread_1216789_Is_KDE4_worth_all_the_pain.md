### Post by beartham on 2009-07-18
Does anybody get any work done with KD4 and plasma? I have been a Linux user for more than ten years, Red Hat, Suse, Fedora, and now Kubuntu (Jaunty on AMD 64). I recently gave up on Fedora 8, but it was KDE3 and the desktop wasn't the problem. In fact KDE3 was a superior desktop manager, far better than Gnome or any Windows platform. Why couldn't we quit while we were ahead?

Now the desktop is the number 1 problem because of KDE4 plasma. Why do we have to spend so much energy on cosmetics? Are we competing with Microsoft and Apple?

Today my box slowed to a crawl when I started running an XML parser, but I noticed that it was only running about 4% of the CPU with some intermittant disk sleeping. **Plasma was using 97%**. Finally it froze the machine. When I rebooted, I closed all applications, and Plasma was still running 97% CPU. I had 4 hidden panels, so I removed one of them (on the right), and the problem went away.

Would anybody like to start an anti-cosmetic revolt?:frown:
