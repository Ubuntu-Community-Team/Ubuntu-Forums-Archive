### Post by dusdus on 2010-03-30
Hi there!

Please, I want to know if I'm the only one experiencing this or not. Maybe I'm biased or something...
I've been using Firefox 3.5, 3.6 and the pre-release, available in some PPA, and in my opinion they all sucked. I've also been tweaking firefox, using addons, the topic in this forum, using vacuum with sqlite3, fidling around in about:config, and so on. Firefox is slow, using lots of memory, has an ugly font (why not simply using system fonts anyway?!) and I really don't understand why it's a standard application in ubuntu.
I've been using firefox in a 32- and a 64-bit environment.
My conclusions:
[LIST]
[*]Flash is even more horrible in firefox than in other browsers: slow, buffering while it's not needed, cracky sound and flashing video (no that's not a good thing, even when it's a flash-file)
[*]startup time could be reduced by using sqlite3's vacuum, but more than that...nope
[*]using RAM for firefox's profile was a slight improvement, but still it's slow.
[*]it renders like a granny on a highway: the webpages are even waiting for the browser to rethink the page (again) 
[/LIST]

I'm using chromium-browser now, which is outperforming firefox by miles.
My advice:
```

sudo apt-get install chromium-browser

```
NOW!


ok, finally got this off my chest, back to ubuntu...
