### Post by kaibob on 2010-06-16
> **coolbrook said:**
> and doing it with a USB connection to my laptop.  Simple Scan is pretty decent. Nevertheless, please feel free to discuss alternatives or scanner application preferences.
There are not all that many choices, and the choice depends, in part, on your needs. Xsane is probably the most full-featured but the interface can be pretty confusing. If you have an Epson scanner, you can download image scan from the avasys site. If you only want to scan to PDF's, have a look at gscan2pdf, which is in the repositories. I believe there are some sane frontend's in the repositories, but I don't have much experience with them. Finally, you can write  a shell script using the command-line scanimage utility. 

I use a shell script with scanimage for daily scanning and image scan when I need to change a lot of settings. I used to rely entirely on xsane and liked it once I learned the program.
