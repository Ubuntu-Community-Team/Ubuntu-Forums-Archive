### Post by Old_Grey_Wolf on 2010-12-03
In my opinion, for a lot of people that are not computer enthusiasts it is one of the best choices. It is fairly easy to install. If you have problems, the forum is active and a good place to get help. If you have hardware that is not compatible out-of-the-box with Ubuntu, there are many sources for help to fix most problems. If you want the bleeding edge software you can include development Personal Package Archives (PPAs) in your sources list. If you want fairly new software you can upgrade to the new release every 6 months. If you are less concerned with new software you can keep any release for the 18 months it is supported. If you really don't care about how old the software is and want fewer upgrades and potentially more stability you can use the Long Term Support (LTS) releases that are supported for 3 years.

### Post by Spice Weasel on 2010-12-14
> **uberbuntufan64y said:**
> Whats a busybox?

It's an alternative to the GNU userland utilities and commands (used in Ubuntu, Fedora, most common distributions.) Busybox is less common but is used in Alpine Linux, Damn Small Linux and TinyCore Linux. It's quite different and smaller than the GNU utils (for example, -ash lacks a lot of the features of bash, and free can only display the available memory in bytes).

Basically, it's a different but similar in principle set of commands that you can use on a terminal.
