Don't know the exact name or if everyone agrees it is an OS. But I would say the Amiga OS on my Amiga 2000. Nothing needed to configure, no questions "Are you sure?" accept this, accept that etc....

So simple. Insert diskette with the program you want and go.

C64, was ok, but having to play the cassette tape just to start it up.....

DOS I liked and hated. Damn having ~7MB of free memory but still having to fight with it to free a kB of memory it needed for EMS memory or the likes.

Win95/98(SE)/ME, fairly Ok, liked ME the most, had the least of troubles for me, though a lot others might disagree.

Win2000/XP/ I first liked XP and still do in some ways, it is stable, nice interface. Switching to XP-64 was the last straw, since my TV card did not work because the vendor refuses to publish a driver. To be fair, this is not entirely Microsofts fault, but I feel it is the nature of these company's to force you to switch to a "better" version. And so I did.

So now Ubuntu because it does everything I want in a lot of user friendly ways so I can concentrate on getting something done in an efficient way, while still leaving the option to "play" with it. 
What might be the most I like about it is that it evolves, and people are doing their best to get it user friendly or otherwise improved. So its not the OS only, but the framework/community behind it.
