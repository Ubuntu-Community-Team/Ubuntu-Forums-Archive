### Post by AngusH on 2010-11-06
Hi, you might like to take a look at this [http://www.psychocats.net/ubuntu/kdegnome](http://www.psychocats.net/ubuntu/kdegnome)

1) I'm not sure if kubuntu has the software center but if it does it will have the same progs and if it doens't you will be able to get it/something close to it.
