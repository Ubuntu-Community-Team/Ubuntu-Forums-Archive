### Post by NoaHall on 2009-11-16
You don't need anti-virus in Linux-based Os's (Ubuntu, etc) because they haven't got any major viruses in the wild. Any bug hole found in the code is quickly(compared to Windows) seen and fixed. This is because anyone can see the source code, and check things out. The worst thing that can happen is using some weak closed source or unknown software and the user runs it as root, allowing for major damage to be done. If you are running a server, you'd only need it to make sure Windows doesn't get infected(which will only happen really if you use it as a email server)
