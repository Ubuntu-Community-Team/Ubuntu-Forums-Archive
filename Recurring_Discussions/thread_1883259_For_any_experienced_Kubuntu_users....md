### Post by JayKay3OOO on 2011-11-19
My favourite thing about KDE is the standard set of dekstop widgets that enables you to make KDE work the way you want. I always miss folder view from other operating systems. MS has something similar in the task bar, but it's not the same as having a scrollable window on the desktop that seems to hardly impact computer performance.

I've come to love some of the standard applications such as Amarok music player dophin file manager and dragon player. Kword is also great for essays and such. 

Dolphin has now matured to a point where it no longer crashes all the time. 

My laptops wireless actually works now, something that was broken in a previous release. 

The standard KDE menu (that I don't use) is actually quite fast now and a joy to use. . 

I used to think KDE looked a bit disjointed, but I'm glad to say that now I think it does seem to flow a bit more and applications seem to 'fit' more in KDE that they used to.

A fresh install looks and feels a lot like Windows which is great for people who may be using Kubuntu on another machine instead of paying for another licence which will make the transition a lot easier. For all those who are bored of this then it's simply 5 minutes of widget tweaking and panel changing to get everything the way you want.

It's interesting too see things that KDE has done for ages in Windows such as the plus/minus for selecting icons (tick/un-tick in windows).

But why do I use Kubuntu KDE? The Kubuntu standard setup is quite good and the software sources have everything I ever need.

To me KDE is the perfect desktop environment at the moment for everyday use. Enough flash to make it look fairly nice to look at without being distracting and light enough to run fast and smooth even on my 2.6GHZ single core 2GB RAM laptop. You might scoff at it for using 795mb RAM when I only have 1 browser window open, system monitor, one folder, skpe and Amarok open, but as RAM is so cheap it hardly matters. The standard must be getting on for 3 or 4 by now.

I've used all the Graphical Interfaces and I always seem to come back to KDE. I'm glad it exists and I enjoy using my computer because of it.
