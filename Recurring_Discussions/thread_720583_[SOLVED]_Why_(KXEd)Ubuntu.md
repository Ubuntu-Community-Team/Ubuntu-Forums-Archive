### Post by Gadren on 2008-03-10
I like Kubuntu basically because I like KDE more than Gnome and I find it better to have an OS built with the intention of using KDE instead of one which uses Gnome.  Having tried both Ubuntu with KDE, and Kubuntu, I've found the latter to feel more "integrated" -- and it's that feeling of everything being integrated which is why I like KDE in the first place.

Nothing against Gnome of course (minus some of the anti-customization decisions by some of its developers ;) ), but Kubuntu feels somewhat fresher and more customizable, like I'm on the forefront of development, whereas Gnome feels to me more like an object which is useful but not as adaptable.
