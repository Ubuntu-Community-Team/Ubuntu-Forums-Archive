It is now at the point when I offer an employee a new computer, they refuse.  I remember when the same guys would fight over who got the new computers.

I imagine as I use Linux more and more, I'll start to see more and more of it's limitations.  But so far with a "box of parts" machine, Ubuntu is more likely to run on it that Win7 Pro in my limited experience.  Even "Engineered For Win7" machines aren't guaranteed to work off the shelf.
