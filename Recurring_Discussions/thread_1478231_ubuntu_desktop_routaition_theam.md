### Post by murderslastcrow on 2010-05-09
Here, I'll actually answer your question- yes, you get that with compiz that comes with Ubuntu. However, if you wanna' configure some basic options, go the Ubuntu Software and search for Simple CompizConfig Settings Manager (typing in simple compiz should do the trick). This will automatically install the regular, somewhat complicated configuration program, both accessible from System/Preferences.

I also suggest installing the extra plugins to get the most out of it, since some stuff has been moved to that package in Lucid. Just look for compiz-fusion-plugins-extra in the Software Center as well. After that, you're all set for playing around all you want! :D Enjoy.
