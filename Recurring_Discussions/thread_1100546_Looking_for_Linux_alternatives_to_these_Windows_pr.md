### Post by Temposs on 2009-03-19
I think it should be obvious that things designed for Windows will be done best on Windows. 

MSSQL is designed by Microsoft and works best with their products. If you want great database management, try a database designed for Linux or at least designed to be cross-platform, like MySQL or a number of others. 

Likewise for gaming, there are games designed for Windows and games designed for Linux. The biggest computer game companies go for the biggest market when they design their games, so the market leans heavily toward Windows. If you want to play games on Linux, there are plenty of good ones designed for Linux, though they won't be as well developed as the bleeding edge games designed by the big companies. Just check some threads on games to play in the forums.

I think the reason you don't see WYSIWYG web development as advanced as on Windows is that most web developers on Linux don't prefer to use WYSIWYG. It's not really a good way to develop websites in the long run. It creates messy markup that is not too easy to look through manually. If you want an easy way to develop a website, you should try a Content Management System(CMS) like Drupal or Wordpress or Joomla. They make web development very easy, and it's the prevailing way to make a website fast and easy.
