### Post by mcduck on 2009-04-08
> **jwaclawsky said:**
> I am running Ubuntu 8.04. Are viruses a problem that I need to deal with? (I have become paranoid from a virus laden Microsoft world) If so is there a virus detection/clean up package available that I can install. Thanks.

No, it isn't. There are practically no Linux viruses around, and the ones that exist are either proof-of-concepts not running wild, or target certain outdated software versions.

As long as you keep your system updated and only install software from trusted sources you are safe without any virus protection. Actually all Linux virus scanners mainly scan for Windows viruses, and are intended for use on corporate e-mail servers and such purposes to protect Windows computers. :D

Unless you install some server software you are also safe without having a firewall.

And there isn't any adware or spyware around either.
