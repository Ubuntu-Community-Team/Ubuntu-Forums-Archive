### Post by 3rdalbum on 2011-10-24
> **scooper77515 said:**
> I installed 11.10 but don't like the layout, large icons in sidebar, and how you have to dig for everything.

You don't "have to dig for everything". Use the search bar or put the icons you most need on the Launcher (sidebar).

Unity takes a little while to get used to if you've been using Gnome 2 for a long time, but the payoff is worth it. I wouldn't bet on Gnome 2 being around in the future.

> Is there an easy way to make the layout look like 10.10?

I assume you're talking about Gnome Classic, because 10.10 Netbook Edition looks almost exactly like 11.10.

Assuming you haven't already used Google or this forum's search engine to find one of nearly a hundred topics mentioning this already...

Install gnome-session-fallback. Log out. Click on the little cog next to your name and choose Gnome Classic. Log in. This is a similar layout to ordinary Gnome Classic - no Ubuntu modifications as in earlier versions of Ubuntu. To add items to the panel, Alt-right-click.
