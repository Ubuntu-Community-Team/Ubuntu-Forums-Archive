### Post by Cavalryman on 2008-01-08
As long as Linux is open source software, there *will* be multiple versions because various developers will modify the code to suit their particular needs. The problem with the suggestion is that there is no such thing as "best Linux" or even "best" features because what is best depends upon what you need. I think Ubuntu has done a marvelous job of combining the features that "most ordinary users" will find useful. Devotees of some other versions of Linux describe Ubuntu as "bloatware" while people who like some of the commercial versions think Ubuntu should have more features. That's the beauty of open source software. Personally, I like it easy like Ubuntu, but that's just what suits my needs.
