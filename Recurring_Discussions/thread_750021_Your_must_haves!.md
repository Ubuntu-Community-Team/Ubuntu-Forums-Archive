### Post by Baby Boy on 2008-04-09
Codecs, Players, Emulator. You can find descriptions for those by doing search under *Applications->Add/Remove...*

As for my recommendations:
Amarok 
aMule
Audacity
FrostWire
K3B

The good thing about Ubuntu is that you have mostly all necessary programs already installed so all you need to do is find the drivers that might be missing and install the codecs. Everything else is a matter of personal preference - you might want to try the Art programs like Blender or Inkscape (GIMP is already installed) or install some games.

### Post by Foster Grant on 2008-04-13
I'm adding a must-have.

Ubuntu Tweak, which in its next version (0.3.0) may be renamed GNOME Tweak.

[IMG]http://i176.photobucket.com/albums/w186/Foster_Grant/Screenshot-1.png[/IMG]

Nice. Very useful.

[http://ubuntu-tweak.com/](http://ubuntu-tweak.com/)

Has an APT-GET (Synaptic, for you recent converts :) ) repository; for information on how to add, see the Downloads page.

For you new-to-Linux folks who are uncomfortable with the command line: open Synaptic (use your password), go to Settings->Repositories-->Third-Party Software, click the Add button and put at least the first 'deb' line (the whole line, starting with 'deb') into the text-entry box.

(NOTE: I am not the developer, I don't know the guy's name, and I'd like to thank the new edition of *Linux Format* for pointing me in that direction. :) )
