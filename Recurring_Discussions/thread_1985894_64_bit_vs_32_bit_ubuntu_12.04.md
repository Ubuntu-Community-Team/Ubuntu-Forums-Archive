### Post by oldfred on 2012-05-24
Moved to recurring discussions as it has been answered so many times.

Ubuntu was finally going to make 64 bit the recommendation as most computers support it, but found about 25% of users had 32bit systems. So they still recommend 32bit for those old systems.

They assume if you have a new system you know you need 64 bit.
Many threads in forum on reason to use 64bit, a few very special and old apps have not been updated to 64 bit and might be the only special case where 32 bit would be a choice. 

While pae will use your 8GB, it is less efficient than 64bit.
