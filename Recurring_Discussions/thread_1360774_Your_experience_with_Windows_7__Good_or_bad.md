### Post by dmengo on 2009-12-21
I posted this question on another Linux forum to get some feedback.  Has anyone had any good or bad experiences with Windows 7 yet?  
 
I'm currently running Windows 7 at home on two laptops, a desktop, and numerous virtual machines that run off a Hyper-V server. I never buy the expensive retail versions and instead ordered the OEM versions which are less expensive.  (The Hyper-V server product is free and runs on a T100 PowerEdge server.)  Up until now, I've been primarily using the Ultimate versions of Windows 7 on all my computers and the trial Enterprise version for the virtual machines, mostly for testing.
 
So far I've been really impressed with Windows 7. Very stable and works well even on older hardware.  Windows 7 doesn’t seem to have the performance issues that plagued Windows Vista. It certainly doesn’t thrash the hard drive the same way Windows Vista did.  It looks like they've corrected the issues with SuperFetch slowing down the system.
 
The new Windows deployment and imaging tools make working with Windows 7 far easier than previous versions of Windows. I've done a few backups, installations, and restores for other people that went well and were finished in a few hours. I like being able to create a customized build for someone and then immedately grab a backup image in case something goes wrong later.  The Windows Easy Transfer tool makes migrating user profiles really easy.  I haven't encountered a single problem with this tool.
 
I haven't tested the BitLocker or AppLocker features yet. The HomeGroup feature works well and was easy to setup.  Overall my experience using Windows 7 has been positive for the most part.  No crashes, blue screens, or applications freezing.  :)
 
Has anyone else had any positive experiences with Windows 7 so far?
