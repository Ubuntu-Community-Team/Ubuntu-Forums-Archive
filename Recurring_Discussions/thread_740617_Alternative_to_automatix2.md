### Post by ubuntu27 on 2008-03-30
To answer your question directly.

There is an alternative to Automatix called [EasyUbuntu]("http://easyubuntu.freecontrib.org/"). I have never tried, but ti claims to make it easer to install codecs.

Also, I want to tell you that since UBuntu 7.10, you only need to install a package called "ubuntu-restricted-extras" to install all the major codecs, Java, and SUN's Flash.
You can install it using Synaptic Package Manager, the Add/Remove in Application menu, and of course with the terminal

```
sudo apt-get install ubuntu-restricted-extras
```

Things has become easer. We do not need Automatix anymore. ;)

PS: WIndows _out of the box_ dosn't have all the codecs eather.

Also, if you want to learn more about r[estricted Codecs and Formats]("https://help.ubuntu.com/community/RestrictedFormats"). Visit the following page:

[https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)
