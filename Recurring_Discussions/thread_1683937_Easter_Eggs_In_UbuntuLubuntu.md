### Post by WorMzy on 2011-02-08
Running "gegls from outer space" in GNOME's run dialog brings up a space invaders-esque game. Similarly, running "free the fish" will unleash Wanda the GNOME fish.

If right-click gnome-panel and choose Properties, then right-click three times on a tab, you get another tab with a gegl waving at you.

Open a terminal and run "apt-get -h", and you'll get some help info followed by "This APT has Super Cow Powers.", if you run "apt-get moo" you'll get an ascii cow with the message "Have you mooed today?".

Similarly, "aptitude -h" gives "This aptitude does not have Super Cow Powers.", and "aptitude moo" gives "There are no Easter Eggs in this program.". If you increase the verbosity of that command (with -v flags), you get different messages. e.g. "aptitude -vv moo" gives "Didn't I already tell you that there are no Easter Eggs in this program?".
