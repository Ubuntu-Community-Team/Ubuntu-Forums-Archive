### Post by happysmileman on 2009-05-09
The main differences between distros is basically how you install software (Different package managers, package formats, repos and packages available, and in Gentoo for example all software gets compiled from source), and what the default settings are.

If you install/remove the right software it's basically the same in pretty much every other way (but having different repos means it may be hard to get the same versions of all the software, some distros offer updates more often than others)

There's also some specialised software written for some distros other than package managers (OpenSuse has it's own software for configuring hardware and system settings)

### Post by starcannon on 2009-05-09
They are both excellent.
Fedora uses RPM packages, and Ubuntu uses Deb packages.
They are both linux, both can ultimately achieve exactly the same goals, though they may do some things differently, and goal x may be easier to achieve on one or the other.

The question of "better" is silly, they are both Linux, the best distro is the one that gets your goals achieved in the most efficient manner, and as such is highly dependent upon personal tastes, work flow, and countless other "human" elements that no distro can account for in every regard.
