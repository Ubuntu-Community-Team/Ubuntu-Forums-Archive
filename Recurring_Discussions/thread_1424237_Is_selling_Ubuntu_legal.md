### Post by Jive Turkey on 2010-03-07
In some countries, some of the code in the restricted repositories and medibuntu repositories is considered to have valid patent claims on it.  Additionally, some countries have restrictions on the use of encryption.  Selling an assembled computer with the restricted packages installed, and or file system encryption might not be 100% legal.

The CD images on their own would be fine in the US AFAIK.  However if you were to modify them to have the medibuntu codecs installed by default, selling them, or even having them might violate the DMCA.
