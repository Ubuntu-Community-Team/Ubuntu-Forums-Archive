### Post by deadflowr on 2013-10-23
If you want a more stable environment then LTS is the way to go.
If you want the latest(or in some cases, newer to Ubuntu) versions of software, then the current release is the way.

LTS releases have what are known as point releases, which offer an updated hardware stack.
The newer hardware stack includes a newer Linux kernel, which offers better support for newer hardware and fixes for others.And the newer hardware stack includes an updated graphics stack, same thing like the kernel it too offers better support.

What the LTS point releases don't offer are upgrades to the applications available.
So most packages tend to be older, though they do get security fixes.
Some packages get upgraded to the newest version regardless of what version of Ubuntu you run. Firefox and Thunderbird being most prominent. 
but they are the exception and not the rule.

So it comes down to either using an overall newer system, or one that is mainly older,  yet built for stable long term usage.
