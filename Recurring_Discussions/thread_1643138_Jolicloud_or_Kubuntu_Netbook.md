### Post by toupeiro on 2010-12-12
You do not HAVE to be connected to the internet, but the desktop is HTML5 and the concept of that desktop is it is a roaming desktop that moves from jolicloud machine to jolicloud machine pulling applications with you.  What isn't supported in that scenario is when you apt-get an application that is not compatible with the HTML 5 cloud desktop.  The underlying OS is ubuntu, and so if its in a default repo, the upstream testing qualifies it, but it is not necessarily going to work WITH the jolicloud features, and jolicloud will not support applications it hasn't written their wrappers for.  Just be aware of that, and always look in their cloud software center first for apps.  If they are not there, install them if you would like them and access them via the underlying OS menuing system.

I use Citrix on jolicloud just fine.  It's not supported by jolicloud, but it works without a hitch...
