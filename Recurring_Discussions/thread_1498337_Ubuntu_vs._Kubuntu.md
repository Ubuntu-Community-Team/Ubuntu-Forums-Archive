### Post by kaldor on 2010-05-31
```
sudo apt-get install kubuntu-desktop
```

The above command will install KDE alongside of GNOME so that you can switch between GNOME and KDE at login if you so desire. Kubuntu isn't the best choice for KDE implementation, it's mainly a KDE showcase. 

If you want to see a "powered by KDE" distro and not a "Distro with KDE", try openSUSE 11.2!

As for software, I think that KDE stuff is often more feature rich than the others. Just my observation though.
