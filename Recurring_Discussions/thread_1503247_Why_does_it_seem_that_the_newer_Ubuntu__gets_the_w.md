### Post by daldude on 2010-06-06
Why does it seem that each newer release of Ubuntu seems to get worse? They add some new features but remove other features that are still desired.

9.04 seemed to be the best version because it still had the old style of adding programs where you could select several programs to install at once but starting with 9.10 you had to add one program at a time and then wait for it to finish installing and then start over again and search for another program to install, why did they think this way was better? It's a hassle to install one program at a time when you have multiple programs to install after a reinstall or upgrade.
In 9.04 you had some control over system event sounds but with 10.04 you can only either turn off system event sounds or turn them on and and you have very limited control of what sounds play, either default or a limited amount of sounds to choose from and you can only assign a few events to sounds unlike MAC and Windows which   allows you to assign any event or operation to a sound and to pick and choose which one you want to assign sounds to. You can't even change the "Login Screen Ready" Sound in 10.04 but you could with 9.04.
There is a reason Windows and MAC OS has system sounds, because people like to assign events to sounds.

9.04 allowed you to have  choice of several Login Screens but 10.04 you have no choice and are stuck with that boring plain login screen.


With 9.04 you could have customized version of Ubuntu like Ubuntu Muslim Eddition or a Christan Eddition with customozed Login Screens, Mouse Pointers and Even the icons at the top of the screen on the Panel were costomized. 10.04 has some customizeation but not as complex as 9.04.

9.10 had that anoying Video bug with ATI cards that made the desktop black with no icons on it if you had Visual Effects enabled but it did support digital out on my Motherboard and when you moved a windows playing a movie you still saw the movie playing but with 9.04 you just saw and outline of the window.

10.4 still supports Digital Out on my motherboard and shows the contents of dragged windows and media players plus added a program to purchase music right from Ubuntu and added the social network access from one window but the last 2 could easily be added via a new program.

The issue is that Canonical is trying to appeal to too many people by trying to automate everything for everyone within the storage budget that a single LiveCD allows. If you do not like this, then perhaps you should try a distribution that does the opposite. My preference is Gentoo Linux:

[http://www.gentoo.org/](http://www.gentoo.org/)

There seem to be a significant number of people here that are happy with Arch Linux, although I do not believe you will receive the same level of flexibility from it:

[http://www.archlinux.org/](http://www.archlinux.org/)

Despite that, Arch Linux is still good, although my suggestion is that try Gentoo Linux. Should you try a new disbribution, you will want to backup your system first. Also, although Gentoo Linux's documentation focuses on installations from minimal installation discs, you can do an installation of Gentoo Linux from inside of Ubuntu and it is my understanding some some Gentoo Linux users even use Ubuntu LiveCDs to do installations of Gentoo Linux. You will likely have to substitute Firefox for [COLOR=SeaGreen]links[/COLOR] (you will love doing that) and possibly [COLOR=SeaGreen]gparted[/COLOR] for [COLOR=SeaGreen]fdisk[/COLOR], but you likely can install that software in Ubuntu via apt-get, even if you are running a LiveCD.
