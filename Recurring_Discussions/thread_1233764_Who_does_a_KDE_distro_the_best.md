the same with me.

I have tried other distro's with KDE but I still stay with Kubuntu.

- Mandriva: has a nice controlcenter for setting up the system to the users needs. But: the package management has some issues with dependencies and the overall system has some stability issues too.
Mandriva uses its own Networktool which is quite good. Browser: Firefox and Konqueror

- opensuse: Yast (the packagemanager/systemcontrolcenter...) is really powerful and needs some time for the user to get used to it. Especially the one-click-install stuff - I have never understood this. 
But in general opensuse is quite good, but a little bit slow. Browser: Firefox and Konqueror

- Fedora 11 with KDE: is really nice (as every Fedora) but focuses more on development and so on - so you have to (un-)install lots of apps to get in a good shape for private/average use.
good: Fedora uses the Gnome-Networkmanager which works much better than the KNetworkmanager of KDE
Browser: Konqueror

- Kubuntu: just has the standard KDE things and no controlcenter. Everything relies on the KDE tools which makes it a little bit complex but in general it is quite good and you also can manage things to your needs.
Kubuntu uses the KNetworkmanager which has its problems with WLAN. So maybe installing the Gnome Networkmanager would be good...
Browser: Konqueror

in my experience Kubuntu works best for me personally because it is the most stable and fastest Distro (on my computer) among the mentioned above. One downside compared to the Gnome-Ubuntu: Kubuntu uses KPackagekit for (un-)installing Softwarepackages. KPackagekit sometimes has problems to find things via its search engine and it also crashes sometimes. The best way to install packages is therefore via Terminal and apt-get install...
Another one is that Kubuntu uses Konqueror as the default browser. So if you want a different one, because Konqueror sometimes crashes and has some problems in rendering websites too, you have to install one. 
The software repos are the same as in Ubuntu.
