Regarding the freedom and all the stuff - well we have to make some compromise - we cannot all be RMS. When you buy a CD the tracks will most likely be in a proprietary format - RMS won't approve but he is perfect - we are not (not yet at least). And where do you draw the line between doing what is right and what is comfortable (like using that handy proprietary format mentioned above)? Here is a thought of my own: I know that source code is available and it can be inspected but most of us use the precompiled binaries. Most of us don't inspect the source code so, on that matter, we have to trust the .deb package submission system to be safely enough not to allow some malware or other bad stuff. Not to mention the fact that a compiler tool can include some viruses in the final result of the compilation even if the source code is clean. What I mean is that this is a compromise we all cope with it - we don't inspect the Ubuntu source code - instead we trust that Ubuntu developers have compiled the binaries in a secure manner. The issue of trust is raised - you have the source code but not checking it may be like using a source close proprietary software - so from a practical point of view it doesn't matter if the software is closed or open source - only the moral issue remains but it cannot stand if you do not compile Ubuntu yourself since you cannot tell based on your own experience that Ubuntu source code is what you have as a compiled OS or otherwise. Just give it a thought - you have the freedom of an open-source ecosystem but are you using it?  

I cannot see what is wrong with trying to charge a fee for some of the services the company behind Ubuntu provides. Even if the fee was comparable with the one charged by Microsoft you are still in advantage due to the fact that the source code is available. You know that having a good product does not mean you can sell it also. The world mouth is not enough anymore - you have to invest in a minimal marketing setup like a website, community forum, free CD/DVD's etc. Free doesn't always means good - what is the point if no one hears about your stuff?
