### Post by 3Miro on 2011-11-07
What you are experiencing is normal.

11.04 is build on top of Gnome 2 and GTK2, both of which have been around for ages. 11.10 uses Gnome 3 and GTK3, it is normal to experience more problems on such transition. Also, 11.04 has now been around for over half a year and people have been building updates all the time. Give 11.10 some more time for the bugs to get fixed (file bug reports and such).

When it comes to the LTS, I do feel like Canonical is one release behind. 10.10 offered almost nothing new, it should have been the first Untiy release with 11.04 making the move to Gnome 3 and GTK3. Then 11.10 should have been the release to polish the final set of features before the LTS.

We can see a good 12.04 as is, but I feel it will take some time after the initial release to get there.
