### Post by Giant Speck on 2010-07-13
> **samstreet101 said:**
> What doesn't sit well out of interest? Any specifics? I suppose it does already come with windows'like buttons..

1. Chromium doesn't honor system font size settings.  The interface text (tabs, bookmarks, address bar, buttons, etc.) is too big.  I haven't been able to figure out how to change this.

2. Chromium doesn't have built-in autoscrolling.  I have to use a third-party extension to get autoscrolling capability, and even then, it doesn't work on every website.

3. Chromium doesn't feel native, even with "Use GTK Theme" selected.  All that does is change the colors to more-closely match system colors.  The only other thing it does is use system icons.

4. vBulletin's WYSIWYG post editor doesn't work in Chromium.

The only compliment I have for Chromium is that Flash and Java seem to work better in Chromium than they do in any other browser in Linux that I've tried.
