### Post by OrangeCrate on 2008-12-08
A fair comparison of Evolution vs Thunderbird, would not be Thunderbird alone, but Thunderbird with the Lightening extension installed. Both would be good PIMs.

Personally, I have a lot of experience with TBird, and little with Evolution. I could certainly recommend a closer look at Thunderbird/Lightening combo.

[https://addons.mozilla.org/en-US/thunderbird/addon/2313](https://addons.mozilla.org/en-US/thunderbird/addon/2313)

Or, if you would prefer to keep Thunderbird and a calendar function separate, you could use Sunbird along side Thunderbird. The Sunbird package is in the repositories.

[http://www.mozilla.org/projects/calendar/](http://www.mozilla.org/projects/calendar/)

### Post by rick08 on 2008-12-08
I myself prefer to use evolution because of the extra features, such as calender and task management.  I also found evolution to be easier to set up than thunderbird.  It really is just preference though.  If you only need email, then use thunderbird, but if you want email plus a little more without having to install other plugins etc., then use evolution.
