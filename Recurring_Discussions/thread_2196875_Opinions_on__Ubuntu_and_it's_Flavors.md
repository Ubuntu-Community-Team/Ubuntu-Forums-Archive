### Post by BrokenKingpin on 2014-01-09
I use Xubuntu. I migrated to Xfce after Gnome2 died and have been very happy with it. Xubuntu for me strikes that perfect balance between speed, simplicity, and usability. My desktop looks great, and runs fast!

Kubuntu/KDE is pretty nice as well, but still always end up back on Xubuntu. I like the design of the KDE desktop, and a lot of the principals behind it, but just find it a bit too buggy and hard to configure the way I like it. I try it every few releases, and like it more each time.

Lubuntu/LXDE is great if you need a very slim/fast distro, but find Xubuntu a lot easier to configure, and isn't that much slower than Lubuntu.

I don't really like Ubuntu/Unity. My major issue Ubuntu/Unity is the integrated Amazon crap. Sure, you can probably remove it, but I don't even want to have to deal with it. Unity is fine, but find Xfce more usable for how I work.
