Given that I recently converted to Arch Ill throw in my two cents.  

Aside from your dislike of command line environments, all of your requirements were pretty much an exact description of Archlinux.  It's known for the fantastic documentation in the arch wiki, it's extremely lightweight so it boots fast (even on an old Pentium III mine boots in under 20 seconds), and if you know what youre doing you can install the latest drivers for any kind of setup you might have.

Some drawbacks though are that you really do need to know what youre doing and you need to be comfortable with the command line.  The wiki is very thorough but you usually wont get any help in the forums if you dont understand one of the wiki articles.

Bottom line, if you want a OS that just "works" with minimal effort, then arch and slackware arent for you.
