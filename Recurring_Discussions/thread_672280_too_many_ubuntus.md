### Post by mikedep333 on 2008-01-19
Tons of distros have all the different desktop environments. In Ubuntu it is just like in those other distros; you can always install the other desktop environments later. The difference is that Ubuntu makes those other desktop environments both installable and previewable by default, and with the appropriate administration tools. You can only have one environment per CD because CDs are small. It's not like Ubuntu really consists of separate distros, it's just different default software on a CD.

The author compared the different variations of Ubuntu to the different editions of Vista and states that this all confuses novice users. But there is only one version of Ubuntu for novice users: regular Ubuntu.

I do agree that one DVD with (gnome) Ubuntu, Kubuntu, Xubuntu, etc would be nice. I think a 3rd party does that though.
