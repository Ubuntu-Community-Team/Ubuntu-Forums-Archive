It's not the Suse default. It's not the 4.1 default either (this is the 4.1 default [http://www.notmart.org/images/plasmakde41.png]("http://www.notmart.org/images/plasmakde41.png")) Either way, changing Plasma themes is ridiculously easy in 4.1.

You right click on the desktop and choose 'Desktop Settings'. Click 'New Themes' which launches a small window that lets you browse through Plasma themes on KDE Look. Click the 'install' button beside the theme to install it, then go back to the desktop settings screen to select it. Check this link for screenshots/video of hoe to do this ( [http://aikurn.wordpress.com/2008/07/01/itty-bitty-tip-how-to-change-the-plasma-theme/]("http://aikurn.wordpress.com/2008/07/01/itty-bitty-tip-how-to-change-the-plasma-theme/") )
