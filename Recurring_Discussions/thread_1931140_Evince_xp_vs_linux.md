### Post by HeroOfCanton on 2012-02-24
Microsoft has access to proprietary hardware information that linux developers typically do not. Plus most companies make drivers for Windows while leaving linux support on the community to figure out on their own. It's not uncommon for programs to be more efficient in Windows just based on interaction with the hardware.

That said, through proper setup and configuration my Xubuntu setup runs faster than Windows ever did, looks much better, allows multiple workspaces, and increases my efficiency due to a personalized layout. I can afford a little extra weight by the individual programs thanks to the choices the OS itself gives me.

If you try customizing linux to run how YOU want it increases in productivity can be huge, and isn't that what really matters? Not which one takes more RAM to run program X. Just my 2 cents...
