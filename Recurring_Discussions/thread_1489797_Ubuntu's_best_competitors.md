### Post by HermanAB on 2010-05-21
Linux is Linux is Linux...

The better you know Linux the more you realize that the perceived differences between distros are very superficial and unimportant.  In other words, if you think that one distro is better than another, it is only because you still don't quite know what you are doing and need to RTFM some more.

Anyhoo, the older distros still have better wizards than Ubuntu - which is nice for those of us that don't like to RTFM. So Ubuntu still needs a few more years to catch up with Fedora, Mandriva and Suse in the gee whizz pointy clicky department, but not that it matters much.
