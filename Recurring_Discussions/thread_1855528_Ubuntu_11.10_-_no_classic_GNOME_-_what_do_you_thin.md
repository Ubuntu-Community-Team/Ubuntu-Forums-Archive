### Post by Copper Bezel on 2011-10-06
> **ubun2geek said:**
> GNOME 2.x is better than GNOME 3 in a few respects such as customization, which I already stated. What is the difference between classic GNOME and GNOME 2.x? I use them interchangeably. OOPS. :)

Gnome 3 actually includes some rather advanced new customization features via extensions. Crazy customization toys will be available as the system advances. 

Saying "classic Gnome" is usually going to be read as the "Ubuntu Classic" non-Unity Gnome 2.0 session in Ubuntu Natty (which, as you know, uses Compiz in place of Metacity and some other tweaks and extra applets and is thus not "pure" Gnome.) The Unity session in Natty is *also* Gnome 2.0, just again running Compiz and with the Gnome panel replaced by the Unity Compiz plugin. 

Gnome 3 includes the new Shell interface and the fallback session, which is equivalent to a Gnome 2.0 environment with a better panel and a few applets left out. The fallback session is also compatible with Compiz.

So essentially, there are two possible shells in Gnome 2.0 (one that ships with Gnome in addition to Unity)  and three in 3.0 (two that are provided by Gnome, in addition to Unity.) It's best to specify both the Gnome version and the shell, as they're not strictly dependent on each other.
