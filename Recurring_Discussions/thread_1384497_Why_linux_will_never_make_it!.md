WHY LINUX WILL NEVER MAKE IT AS A MAINSTREAM OPERATING SYSTEM!
 

 I've been fooling with Linux off and on now for over 10 years.  At times I get to a point that I want to use something besides Windows and I'll download a distro or purchase one off of Ebay because of the time involved in the download just to have a look see at something different.  I'll connect a spare hard drive to my computer for the install to make sure my Windows OS doesn't get corrupted, been there, done that one!!!   
 

 Most Linux installs are, for the most part, hands off, ie, there are very few problems during the installation.  I've found this to be true with Open Suse, Fedora, Red Hat, Ubuntu,  Mepis, etc and probably others as well that I haven't played with. I was immediately impressed with the ease of installing addons, ie, Flash Player, Filezilla and a few others but when it came to sharing drives across a Windows network it choked, BIG TIME!  After posting several messages asking for help in the Ubuntu forum and searching online I finally found what might have been the fix.  Trouble is the “fix” was almost a page full of Samba/Linux command line crap JUST to be able to share files on a SECOND drive in a Ubuntu computer???  Nah, think I'll go back to my Suse install instead so Ubuntu “bit the dust” after this fiasco!  That and Ubuntu wouldn't play well with a multiboot computer running both Windows and Ubuntu on different drives.  Remove the Windows drive and Ubuntu won't even boot and again even though Grub was installed only on the Ubuntu drive, the fix for this was more command line crap!??
 

 So the down side of Suse is that I've installed 11.1 probably a half dozen times only to get pissed off and disconnect that drive after it fails to install Flash Player on EVERY installation!  Filezilla went in but only after a great deal of trouble.  That and yesterday I tried it again only to find that there are NO drivers available from Suse for my HP 6500 wireless printer and after hunting for a half and hour I found what was suppose to be drivers from HP, again command line crap, that failed so I got no way to print from the Suse install????  Then I tried to playback a music video that I had downloaded only to find that Banshee didn't have the proper codecs and it was going to take another act of God to get them loaded, so here we go again!  BACK TO WINDOWS!
 

 And I won't even go into trying to install Unreal Tournament and Quake III!  I can't say how much time I've wasted trying to get those two to run and this has been over the past 8 or so years and I've NEVER gotten either to work???

 At this point I'd have to say that Linux should be considered as a “hobby” OS only, not as a replacement for Windows as most people will NOT put up with these problems that seem to happen on a daily basis!
