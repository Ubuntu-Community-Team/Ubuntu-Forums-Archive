### Post by majikins on 2008-12-01
contacts sidebar - shows the address book as sidebar with a search function so that you can add/edit your contacts.
Gmailui - Add concepts inspired by GMail to Thunderbird, like archiving mail and powerful message searching across all folders. Supports tag searching. All those lovely keyboard shortcuts work their magic in Thunderbird.
lookout - those pesky outlook formated e-mails that come through as winmail.dat get interpreted properly inline.
lightning - full blown calender and task utility.  Accept and create meetings as with outlook.
nostalgy - move between folders, copy and move messages via keyboard shortcuts.
quotecollapse - when you subscribe to mailing lists, some threads become really long and ongoing replies can make reading the mail tedious. Quotecollapse solves this by cleaning up past replies and focussing on what was posted by the sender.

Goto [https://addons.mozilla.org/en-US/thunderbird/](https://addons.mozilla.org/en-US/thunderbird/) - and search each one to get more indepth info.

Cheers
