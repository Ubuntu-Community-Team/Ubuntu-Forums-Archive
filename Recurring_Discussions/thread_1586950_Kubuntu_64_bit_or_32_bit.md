### Post by 3rdalbum on 2010-10-02
If you are running 32-bit now and have no other reason to reinstall Ubuntu, then don't bother doing it yet. Wait until you already need to reinstall anyway, and then just install 64-bit.

If this is your first Ubuntu install, then get 64-bit, don't bother with 32-bit.

64-bit is faster for video processing and other kinds of number-intensive operations. If you just surf the web and do word processing you won't notice a difference.
