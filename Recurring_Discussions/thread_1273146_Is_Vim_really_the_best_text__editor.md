### Post by ibuclaw on 2009-09-23
You can always use:
```
vim -y
```
To go into "easy" mode. (Always on INSERT mode)

To switch between INSERT and REPLACE, press **SPACE**+**INSERT**
To switch to Ex mode, press **Ctrl**+**L** followed by the :command you want to use (ie: for saving/quitting application).

It seems to be the more sane way to use it for users who don't want to have to be constantly press 'i' to input some text.
