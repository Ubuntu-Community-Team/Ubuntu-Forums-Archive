It's normal human behavior to identify with/approve of the (perceived) small, scrapy, underdog. Consult any basic psychology text.
If it's viewed as a competition, the larger / more powerful / wealthier / more successful player will usually be viewed unfavorably.

### Post by monkeybrain20122 on 2013-12-08
OK, I have a copy of Ubuntu 13.10 installed in an external hard disk which I use as a portable by booting off different machines. I anticipate some situations where the host machine may not have the specs to run Unity smoothly and I also heard that remote desktop doesn't play too well with 3d desktop  on the server side (though I never use it and chances are I wouldn't use Ubuntu on the server side). So I did
```
sudo apt-get install lxde
sudo apt-get install gnome-session-flashback
```

Now I can switch between Unity, Lxde, openbox, gnome classic (or whatever it is called), gnome classic no effect just by logging in and out.  It can't be easier (imagine doing that with Windows or Mac) I really don't understand the angst and gnashing of the teeth. No one forces you to use anything.

Edited: make sure do not install lubuntu-desktop, for that would pull in a lot of stuffs that you don't need, just lxde.
