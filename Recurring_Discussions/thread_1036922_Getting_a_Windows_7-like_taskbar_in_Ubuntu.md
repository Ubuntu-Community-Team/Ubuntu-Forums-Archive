### Post by weblordpepe on 2009-01-26
Just to get back on topic. The first poster asked how to emulate the feature of the Windows 7 taskbar. 

The taskbar is an improvement but it doesn't really address the main wonky problems of navigation in Windows. It just kinda...keeps compatibility.

Basically what this user wants is:[LIST]
[*]Launchers to be able to keep the same icon for when the program is running

[*]Right-clicking a launcher/running program reveals the same list of common tasks for that program, regardless of if its running or not. E.g. recently used files.

[*]No text labels.
[/LIST]

Now as far as opinions go - I think it stinks having no text labels. I think its a very good idea keeping the same icons for running programs and shortcuts though - where its specified. The notification tray is silly. If anything, programs which use the notification area should use their launcher icon instead of sitting in that area. Or...the notification area should have dormant icons, which could possibly indicate when their program is running.

How to do it:

This functionality could easily be done by creating a special gnome tray applet which holds icons/launchers. When an application is run out of that tray applet, it could hijack the icon's perception of the notification area. The special gnome applet could tell the running program 'hey your notification area icon should be -HERE-' which points to its launcher icon.

Difficult for a power user to create though, but easy for a programmer willing to break/fork from existing standards.

Bumping this with some new info I didn't see posted reading through the rest. I landed in this thread looking to get some of the better features of the new Aero taskbar.

If you open up the Compiz Settings Manager and check *"Window Previews"* under Extras then you'll get a preview of open windows when you hover over them in the task bar. It's not as functional as Windows 7's as there's no close button and it doesn't bring the window to the front for a preview but it's better than nothing.
