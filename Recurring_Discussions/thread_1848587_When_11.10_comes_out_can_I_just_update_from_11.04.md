### Post by grahammechanical on 2011-09-22
Yes. Update manager should give you notice that a new distribution upgrade is available. All you have to do is click. But I suggest that you wait two or three weeks until the rush has died down. Otherwise the download/upgrade will be slow.

I have done this a few times. Another way is to download a 11.10 iso image and burn it to disk. Then hold the shift key down to prevent the live CD from loading and you will get a message offing to update from the CD. If you are connected to the internet then the latest updates will be done at the same time. This can sometimes be quicker as a lot of the files will come from the CD.

Regards.

### Post by 3Miro on 2011-09-23
An upgrade would require conversion of all settings from Gnome 2 to Gnome 3. This is not trivial and can go wrong on so many levels.

It is OK to try an upgrade, but backup all important files and be ready with an install CD to make a full re-install.

If I were you and I were to run into trouble with the upgrade, I wouldn't even bother posting on the help forum, just make a clean install. The help forum usually gets clogged with people having all sorts of trouble and getting help can be hard. Clean install is so much easier.
