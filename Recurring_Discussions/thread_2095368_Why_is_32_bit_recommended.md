### Post by oldfred on 2012-12-16
Actually now the issue is more that 32bit causes problems for new systems.

The 32 bit  does not work with any new Windows 8 (pre-installed) computer or some Windows 7 systems that have UEFI installed.

Only the 64 bit currently has the UEFI install option and if Windows is UEFI you have to install Ubuntu in UEFI.
