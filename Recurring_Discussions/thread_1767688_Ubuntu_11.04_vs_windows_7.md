### Post by egervari on 2011-05-26
I don't think you can compare them easily. It depends on what you're doing.

If you are a ruby developer, it's linux all the way. Programming in Ruby/Rails on windows is a disaster. Even when you have it working, the tests run like 4-10x slower than they do on Linux.

Linux has problems with less than common drivers though. I have an FA-66 hardware audio box for professional mixing and it's supported amazingly well on Windows - just as good as a SoundBlaster card. Even though I can get it work on Linux, I have lots of problems every few hours - sound just cuts out for no reason. Also, having to manually setup your FA-66 is a massive pain in the ***, and having to manually start qJackCtl just to hear sound when you boot up is also dumb. Windows wins.

When it comes to games, windows wins. Same with photoshop. I use both types of programs from time-to-time, so I am totally dependent on Windows somewhat.

Ubuntu 11.04 also comes with lots of bugs and weird problems. When 10.04 or 10.10 came out, I would have said Linux was fast, stable and had low issues... but that's not true with 11.04. You have minimize/maximize buttons disappearing with compwiz all the time... and it gets worse and worse as you use it. You are forced to use 'metacity --replace'. Even though this is a known issue for a really, really long time... the 11.04 distribution STILL hasn't fix it.

I also can't use Unity - not because it has clear deficiencies or is strange, but because I get all sorts of graphic-related problems, it's slower, etc. It's also hard to edit your launcher bar too. Try putting RubyMine and IntelliJ IDEA on the side bar at the SAME time - it's impossible to figure out. Ubuntu Classic is still far superior. I am not the only one who has issues with it - everyone I know running Ubuntu has had the same problems. It should never have been put in the release. I know people who are not hard-core users or programmers that were so frustrated with it - and they are lost as a result.

Despite all of that, Firefox and other browsers all run so much better. Generally file operations are much more efficient. Multitasking is superior. It utilizes ram better. And it's free.

There's no clear victor, but generally, I wouldn't discount windows until 11.04 gets its act together.
