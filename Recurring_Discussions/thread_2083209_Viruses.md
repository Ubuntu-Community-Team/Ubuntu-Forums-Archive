### Post by fballem on 2012-11-11
It's less of an issue with Linux than it is with Windows. There are a three major reasons:

1) The market is a lot smaller. Windows and Mac are (currently) a lot more popular.

2) Linux is inherently built for multi-user. The permissions that a user has are limited. Windows was originally built with the assumption that the user had control of the whole machine. Windows has gotten tighter over the past years, but the issues are still there.

3) Linux is open source. It is a lot harder to put a virus into Linux, since the software is so widely (and so quickly) reviewed. Microsoft is a closed-source shop, so the code is not as widely used.
