### Post by cogadh on 2008-04-23
What comes with Windows when you install it? Nothing but Windows. No hardware specific drivers, no firmware, no DVD decoders, no Flash, no Java, no MP3 codecs, no commercial software. How is that any different from Linux distros only providing just free software? Actually, it is much better than what Windows offers since most distros already come with a full office suite preinstalled, as well as most of the most commonly used "regular user" applications. After installing Windows you still need to download and install all the drivers for your hardware (usually from multiple websites or disk sources), install all your commercial applications and configure you entire system. With Ubuntu and most modern Linux distros, you just need to run your particular package manager, select the software, drivers, etc. you want and install it. Yes, certain hardware is more difficult to configure than others, but with every successive release, that is changing. When I first started using Linux well over 10 years ago, it barely had a usable GUI, let alone support for any of my hardware. Now I can install it and have a fully functional GUI-based system in under a half an hour. With Windows, I might have a functional OS in about a half hour, but I'm going to spend the rest of the day downloading and installing everything I need to get a *fully* funstional system.

The only thing that makes Windows more prevalent than Linux is the fact that nearly all system manufacturers preinstall it on the desktops they sell. That in turn makes software developers only want to develop for the OS that the vast majority of users are going to have. It has nothing to do with the free software zealots demanding only free software, it is just simple business sense. With major system manufacturers like Asus and Dell beginning to sell machines with Linux preinstalled and demanding that the hardware manufacturers start supporting Linux actively, the market will begin to change. It certainly won't be quick and I doubt we will see Microsoft dethroned by Linux anytime soon, but it will happen and has already started to happen.
