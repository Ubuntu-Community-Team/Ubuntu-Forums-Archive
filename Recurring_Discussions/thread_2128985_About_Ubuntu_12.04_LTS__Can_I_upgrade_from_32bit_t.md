### Post by pretty_whistle on 2013-03-24
I currently have Ubuntu 12.04 LTS 32bit but want to install the 64 bit version (I bought a new computer which is a 64bit).  Is it ok to do this or would things mess up?

---

### Post by ManamiVixen on 2013-03-25
Yes, it's ok. Canonical actually insists on the 64bit version, but it will have to be a fresh install. You cant upgrade from 32bit to 64bit with the Ubuntu CD/ISO. So it will have to be a reinstall/fresh install on the new PC.

---

### Post by dcstar on 2013-03-27
There is no direct "upgrade", you will have to back up all of your data and settings and manually restore them on the fresh 64-bit install.

The forums are full of detailed threads on how to do this.
