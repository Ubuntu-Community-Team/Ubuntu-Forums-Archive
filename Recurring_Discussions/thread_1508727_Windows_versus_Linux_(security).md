### Post by da burger on 2010-06-13
As the article says both have flaws (no one is perfect). But I would generally say linux is safer. Just a few reasons of the top of my head.
1)The root account is not given to a user by default in most distros. This means system files are untouchable by a virus unless it infects root.
2)There is a greater diversity in systems designs. This means a virus has to be able to infect them all to do major damage.
3)Linux is open source so there is a colossal community of developers and bug reporters to catch and patch vulnerabilities.
4)Linux installers are not often executables. A .deb cannot do anywhere near as much damage as a .exe.
5)The more popular linux distros have a package managment systems. These mean that by default you are only installing trusted packages.

Finally what the article said about linux not being an attractive target is rubbish. Which would you rather infect, slow desktop computers that each effect one person. Or servers that give you huge computing power, internet speed and each effect thousands of people. Because most servers run linux.
