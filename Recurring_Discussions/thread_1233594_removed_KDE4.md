### Post by Jimleko211 on 2009-08-06
One thing KDE is not is simple.  You pretty much gotta know what you're doing and looking for to get what you want.  Do keep in mind that it's the popular opinion of the GNU/Linux community that Kubuntu is the WORST KDE implemenation.  If you want to try KDE on a better system, try it on Debian, Fedora, Mandriva, or Arch.

I've tried Openbox, and it's a lot faster than pretty much anything I've tried before, but it takes some getting used to.  To load a program (unless you have keyboard shortcuts, which I never found out how to do) you gotta minimize your program if it's fullscreen and then launch it.  And I never did like the buttons and grey look of everything (though I only used the bundled themes).

Good luck with however you decide to go ^_^

### Post by XubuRoxMySox on 2009-08-06
I removed it too, after a trial run. To be fair, it was Kubuntu-desktop, not pure KDE. But I didn't really like all the K-applications much, although Konqueror is exquisite and powerful. Still, it felt like "AOL for Linux" for lack of a better way to put it. But the main thing was how much slower it was. My system (which is older anyway) got bogged down in it.

I'm using [LXDE]("http://lxde.org") now (available in Ubuntu repos) and it has the basic "look" and familiarity of KDE with none of the weight and bloat. Everything runs much faster and the interface is so simple that anyone can sit down at my computer and just use it effortlessly even if all they know id Windows or Mac.
