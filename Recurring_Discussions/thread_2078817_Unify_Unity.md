### Post by bogobojca on 2012-10-31
I'd like to propose two changes to Unity on the Brainstorm, but I don't know whether they are valid ideas to be proposed there.
The first thing is adding/tranferring the Shutdown menu to Dash, which is meant to be 'the central point' of the desktop environment. I see it as 5-6 icons corresponding to the items of the current menu (shutdown, restart, suspend, logout, lock, help) in the third line of Dash Home. This would bring us one step closer to Dash's desktop omnipotence.
The second thing is a little trickier: redirecting application's notification area menu to it's launcher quicklist. For example VLC shows up in the notification area with its big orange icon decomposing top bar's monochromatic color design. Moreover theres two of those icons on the screen now and this defies the idea of unity in Unity. 
I'm no programmer in any sense, but I can imagine that someone can write a piece of code that brings notification area API into the launcher.
PS. wouldn't it be nice if there was the workspace name displayed on the top bar when no window is active?
