### Post by compmodder26 on 2011-05-25
The difference is in the Desktop Environment that each uses, as well as the default programs that come with them.  Kubuntu uses KDE, Xubuntu uses XFCE, and Ubuntu uses Gnome.  Do some research on these three desktop environments and you'll see the difference.

Also, Ubuntu is the original distribution out of the three.  The other two (Kubuntu and Xubuntu) are spin-offs of Ubuntu that use the different desktop environments I listed above.

And recently Lubuntu was added to the mix.  It uses the LXDE desktop environment.

### Post by jkurtisr32 on 2011-05-25
> **H A NAZIR said:**
> Please tell me the difference between these three terms.

Xubuntu: Uses XFCE desktop environment, and is packaged with minimal software. Can be used on older computers. Will run really quickly on newer computers. Will have to install a few things in order to have the same functionality as Ubuntu distro.

Ubuntu: GNOME desktop environment. Has everything that you need in a desktop, especially for beginners. Has pretty legit functionality right out of the box.

Kubuntu: Uses K Desktop Environment (KDE). More of a "heavyweight" distribution, as it comes with a lot of packages installed, and is (from what I hear) really customizable. I say "heavyweight" because it is still MUCH lighter on the system than Win 7, or something like that.

Hope this helps in your decision.
