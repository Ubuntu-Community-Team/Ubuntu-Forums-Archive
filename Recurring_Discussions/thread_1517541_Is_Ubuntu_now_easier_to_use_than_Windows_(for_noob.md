### Post by aysiu on 2010-06-25
How do you define a "noob" (isn't that a derogatory term, by the way?)?

Is this someone who has *never* used a computer? Or is this someone who is just new to Linux?

If it's someone who has never used a computer, Ubuntu would probably be easier to use than Windows. Applications, Places, and System are clearly labeled (what does "Start" do?), and there is a separate button just to exit. The system tray isn't cluttered up. The software center is (*shock, gasp*) for managing both the installation and removal of software.

In Windows, however, you use Programs and Features only to remove programs. Most of the time you install programs from a CD or a downloaded .exe file.

That said, I don't know many people who have never used a computer. If this so-called "noob" is just a new user to Linux and has used Windows all her life, obviously Ubuntu will be more difficult, since the familiar will almost always be easier.

If you want Ubuntu to be easier for a new user, here are the prerequisites: [list][*]The new user has to be open-minded.[*]An experienced Ubuntu user should install and configured Ubuntu for this new user, whether this experienced Ubuntu user is an OEM preinstalling it or just a tech-savvy friend setting it up in-house.[*]The new user can have zero dependence on programs that have no Linux equivalent or port. For example, if this new user needs Netflix streaming, Microsoft OneNote, Adobe CS5, and AutoCAD, forget it. Why bother? If, however, she is able to use Thunderbird, Firefox, GIMP, OpenOffice, Audacity, and Songbird on Windows, she'll be fine using those same programs on Linux (well, Songbird's defunct, but at least she won't need iTunes and can probably get along fine with Rhythmbox, Banshee, or AmaroK.[/list] It makes sense to be mindful of those prerequisites. They certainly don't apply to everyone, but they are also not an impossible set of circumstances. Also note that nothing mentioned has anything to do with the quality of the software Ubuntu comes with. It all comes down to compatibility, setup, and an open mind.

That's not to say there aren't problems with Ubuntu in terms of usability. But I use Windows every day at work, and there are problems with Windows, too (with Windows 7 I've already encountered the black screen of death, printer spooler needing manual restart, Outlook calendars suddenly disappearing, and a Synaptics Touchpad update breaking the touchpad functionality). The Ubuntu usability issues I can think of off the top of my head are desktop icons appearing on top of each other, resume from sleep taking a long time to reconnect to wireless, and dragging a file to a system folder giving you access denied instead of prompting you for sudo credentials.

No system is perfect. I use Windows 7, Mac OS X Leopard, and Ubuntu Lucid regularly, and they all have their benefits and quirks. I know plenty of Mac users who have problems. And I know plenty of Windows users who have problems. I think computers are flawed and fun machines, and nothing is ever going to be a panacea for usability concerns.
