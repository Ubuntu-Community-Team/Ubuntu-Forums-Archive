### Post by sanderd17 on 2011-11-04
What tools do you want to use?

If it's a simple site, all platforms will do.

If you need to use Windows tools like ASP, then Windows is a better platform.

If you use more open-source tools like PHP and MySQL, then Linux is better.

Linux can also be customised a lot further than Windows.

### Post by Paqman on 2011-11-04
Is the whole stack you're going to be using identical otherwise? The standard Linux server is LAMP (Linux/Apache/MySQL/PHP) but you can have virtually the same stack on Windows (WAMP). Obviously there's a big difference between a WAMP or LAMP setup and IIS. So there's a lot more to it than just Linux or Windows.

Certainly LAMP servers are very common, and very easy to set up  on Linux and you'll find tons of documentation for them. But if your skills are with Windows server packages then you might find that easier.
