### Post by fbkarsdorp on 2010-05-20
Its all about flexibility, its interactive nature. Because it is written in E-Lisp, you can tune it into whatever you want it to be. Why is it so good for programming? Here's a list:

1. the buffers and frames: no other editor I know of can do as much as emacs can with multiple buffers and frames. This is especially handy coding in C for example where you have multiple frames open (header file, implementation file, terminal, compiler etc.)

2. take programming in Python. You can integrate a python interpreter into your emacs allowing you to (like in idle but much stronger and robust) send code directly to the interpreter. 

3. Start learning a bit of E-lisp and you will be able to write your own plugins, little tweaks etc etc. Doing this gives you the feeling that the editor slowly becomes your own.

4. Last but nut least: for Lisp, yeah... Lemondor fame is but a hack away....
