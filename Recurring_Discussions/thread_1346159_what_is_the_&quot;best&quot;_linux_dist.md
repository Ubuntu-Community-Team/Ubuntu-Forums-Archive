### Post by julianb on 2009-12-04
This post is sent from a machine running tinycorelinux, a great option for those who don't have a lot of extra processor power / RAM / hard drive space to burn.

using eee pc, 1gb ram, 4gb hard disk (solid state disk)

tiny core linux makes big strides towards the "it just works" functionality but it has a ways to go before it matches ubuntu-with-GNOME in that regard.

Standard Ubuntu is one of the most user-friendly OSes anywhere, and works with a wide variety of software+hardware. but if you want a super fast OS or you want to save ram/hard disk space, tiny core linux might be for you.
