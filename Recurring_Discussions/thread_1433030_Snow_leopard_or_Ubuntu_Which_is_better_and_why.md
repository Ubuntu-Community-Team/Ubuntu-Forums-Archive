### Post by kaldor on 2010-03-18
> **schwabdale said:**
> Please post... viruses...Stability...speed.. ect

I use both Snow Leopard and Ubuntu on a day-to-day basis. I'll give you my opinions.

OS X overview:

1) Both are UNIX. Well, Ubuntu (Linux) is a UNIX clone, but they both work very similarly down under. Snow Leopard even uses Bash ("linux commands") on the terminal.

2) Mac is based on FreeBSD, so it is inherently stable. But, parts of it were closed-source; meaning Apple changed some things, opened up security issues via applications (such as the old Safari hack issue) and it's not *as* secure as FreeBSD originally was. It's still a load safer than Windows, though. Just turn off Administrator by default and enable Root like on Ubuntu.

3) Some things about OS X are very different from Ubuntu. For example, you can't rightclick>cut. You must drag and drop in order to cut/paste. To rename a file, you need to select it and then press Return(enter). The UI is also very different, but quite easy to learn. I prefer the dock+global menu to GNOME/KDE/etc.

4) Runs best on Apple hardware. Big problem there. It'll cost 1000+ to fully enjoy the OS.

5) Proprietary software is very *very* available. It's almost as good as Windows for what the common person would need. If openoffice doesn't work, you have MS Office available. You also have Photoshop CS4 available.

6) The Apple hardware is great. I love it. 

7) Easy to network Ubuntu + OS X due to a lot of utilities



Snow leopard is much faster. My MacBook has the same specs as my HP laptop; Snow Leopard takes less than 10 seconds to load up, and shuts off in exactly 3 seconds. That's fast. 

Snow Leopard is better than Ubuntu for getting up and running; essentially out of the box 100%. But, if you find a good computer preinstalled with Ubuntu, they are pretty equal.

Overall, it comes to preference, cost, needs. I consider OS X just another distro.
