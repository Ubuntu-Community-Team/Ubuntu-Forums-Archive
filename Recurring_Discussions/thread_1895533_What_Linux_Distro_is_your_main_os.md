### Post by ninjaaron on 2011-12-15

I did manage, after much pain and suffering, to install Ubuntu One on my Arch machine, so I can have my work on my netbook and my "desktop" without any fuss.  I heart Ubuntu One.

I like both Arch and Ubuntu 11.10 very much, though, as you may have heard, they are somewhat different.  Debian is cool in it's own way (and using Sid almost makes it like a cross between Ubuntu and Arch), but the font rendering is too horrible to look at.  It could become my main OS if they ever make any easy way to install Ubuntu patches to freetype.  Maybe not though.  Arch is too cool for school, and it's packages are basically patch-free, so you get updates almost as soon as they appear upstream, way sooner than any other distro, even when using unstable or development versions, and it's rock-solid if you pay attention to what you're doing.

### Post by N00b-un-2 on 2012-01-04
Of course there are non GNU/Linux distros.  There are many BSDs.  Chrome, Android, Splashtop, Boxxee, etc. are all Linux based operating systems that are non traditional and may or may not utilize GNU tools.  iOS is not Linux but does in fact utilize many GNU tools in development.  Sony PS3 and Nintendo Wii utilize highly modified proprietary operating systems but they are based on the Linux kernel.  There are even Windows API based distros, such as ReactOS.  There are so many more choices out there than just Windows, OSX and Ubuntu (but why would you want to use anything but the best?)
