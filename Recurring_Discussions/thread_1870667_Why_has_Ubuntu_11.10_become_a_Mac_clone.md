### Post by jwbrase on 2011-10-28
> **LinuxFan999 said:**
> I don't think Ubuntu has become a Mac clone. There are quite a few differences. For example, Mac OSX is mainly colored white and gray, instead of black, brown, and purple.
The only similarity is the fact that both have a dock.

But when the only similarity is the most significant similarity (and using a dock instead of a taskbar has *very* significant workflow implications), you can't just blow off accusations of copycatting by saying "it's the only similarity".

That said, as I understand (I've used the Unity dock much more recently than the OS X dock, so I'm going by hearsay), the OS X dock is a bit better implemented than the Unity dock and so manages to avoid what is probably Unity's biggest blunder (As I understand, the OS X dock actually gives separate dock icons to separate running instances of a program, as well as giving a separate icon to the program's launcher, whereas Unity does not). I still prefer a taskbar myself.

To answer the question asked in the thread title: They're copycatting Mac because everybody thinks the Mac has a good user interface. It's not just Canonical that's made this mistake. The default settings on Win7 have the taskbar behaving as a dock (and make the same mistakes as Unity). The only differences are that the taskbar defaults to being on the bottom instead of the side (Mostly a cosmetic difference for a dock), and that the worst snafu's in Win7's implementation can be configured away.

Anyways, I agree with keithpeter that you might want to check out XFCE (the interface Xubuntu uses, but you don't have to install Xubuntu to use it. It's in the repositories, so you could use it on top of your existing Ubuntu install). It's what I'll most likely be going with once 10.04 hits end-of-life, unless Gnome 3's fallback mode really shapes up in the next year and a half.
