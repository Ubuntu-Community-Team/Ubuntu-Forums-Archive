### Post by BrokenKingpin on 2011-11-25
That graph is not very representative at all. The Mint line includes all the various mint releases/spins, as they are all hosted on the same website (which is what DistroWatch bases those numbers on). Ubuntu's derivatives are all on separate sites. So for this to be an accurate comparison you would have to add all of Ubuntu's officially supported derivatives together (which would destroy the Mint numbers).

Not to mention that Mint is 99% Ubuntu anyways... so Canonical must be doing something right.

Mint has only surpassed Ubuntu recently, and they have yet to move from Gnome2. When they switch to Gnome3 you will most likely see things change a bit (the same as when Ubuntu moved from Gnome2).
