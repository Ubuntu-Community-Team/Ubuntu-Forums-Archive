### Post by Merk42 on 2011-09-30
> **BobW said:**

The following is for 11.10 and above (if you're using 11.04 you can just choose "Classic" from the login screen)
If you don't want Unity you have 3 options:
[LIST]
[*]Install GNOME-Shell from the software center and choose that at the login screen. This is the default interface for GNOME3 if your hardware supports it.
[*]Install gnome-session-fallback from the software center and choose that at the login screen. This is the default interface for GNOME3 if your hardware does not support GNOME Shell
[*]Use a different DE (KDE, XCFE, etc) If you choose this it may just be easier to install Kubuntu Xubuntu, etc.
[/LIST]
