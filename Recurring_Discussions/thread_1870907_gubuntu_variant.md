### Post by mcduck on 2011-10-28
> **narcisgarcia said:**
> When will be available again an (g)Ubuntu flavour with Gnome as default desktop environment?

Ubuntu project has passed from having Gnome as main desktop environment (and KDE, XFCE, LXDE as alternatives), to kick Gnome from normal/easy installations.

Ubuntu still has Gnome as it's default desktop environment. Ubuntu 11.10 runs Gnome 3 desktop, only replacing the Gnome Shell with Unity.

Making a version with the old Gnome 2 wouldn't make any sense, it's not supported by it's developers any more and as all the various programs move to using services, toolkits and other features of Gnome 3 a distro using Gnome 2 would simply end being insecure, outdated, an most importantly unable to run the latest version of any program.
