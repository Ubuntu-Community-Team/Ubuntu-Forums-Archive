### Post by aysiu on 2010-02-12
*Cannot* is an absolute term in this context that has no meaning.

What is the likelihood that a tornado will show up in San Francisco and destroy the Golden Gate Bridge? I'm pretty sure every day, as I pass by it on the way to work, I'm going to see the GGB and not see it destroyed by a tornado.

Is it true that the GGB **cannot** be destroyed by a tornado? That's a silly statement. Of course it can be. It's just highly unlikely.

But safety in computing is not an odds game, thankfully. Ubuntu is reasonably safe out of the box. If you read the security sticky EssexEagle linked to, you can find ways to harden security even more (encryption, AppArmor, etc.). No matter what you do, though, realize that social engineering can happen on any platform, no matter how "secure" it is, because social engineering involves tricking the user, and if the user holds the key to the door, it doesn't matter how strong that door is.

Actually yes you can. Gnome-Look.org users were recently the targets of malware hidden in a Gnome theme pack and a screen saver. It's currently very rare for Linux to be infected or compromised in this way. However it seems criminals are now testing the Linux waters to see just how vulnerable it's desktop users are.

The truth that none of want to admit is that we're just as vulnerable as Windows users. If not more so. Most Linux users don't bother with anti-virus software. Most Linux users don't think twice before running an installer from a trusted source with root privileges. Firefox was also recently used as a vector of attack against those using open source software.

This is a wake up call to open source software and Linux users. We have the attention of the criminals who would like to exploit our complacency and arrogance.

Remember Linux is sold pre-installed on around a 3rd of netbooks. It's also sold pre-installed on desktops and laptops from companies like system76, Zareason and Dell (when dell bother to offer a desktop).

So when installing software from a source other than the distros repositories, take measures to be sure there aren't any hidden dangers. ClamAV and ClamTK make a very good free of charge, non-cripple ware anti-virus solution.
