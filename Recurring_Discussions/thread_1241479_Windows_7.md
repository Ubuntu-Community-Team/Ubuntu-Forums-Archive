### Post by gletob on 2009-08-16
RTM stands for release to manufacturing.  This means that the version on technet is what has been sent to OEMs like Acer, Lenovo, HP, Fujitsu, Dell, and others.  The retail version may be the slightest bit modified but the RTM is extremely close if not identical.

BTW: I'm using the RTM release of Windows 7 right now.  It's very nice, I've had no issues with it whatsoever.  In my opinion it does a much better job than XP with many tasks.
