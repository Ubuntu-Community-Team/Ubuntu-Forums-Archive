### Post by alexpotter on 2011-10-20
I haven't used Windows 7 for over a year now and because I run a low spec laptop Ubuntu really takes advantage of it. For instance, Intel (my GPU manufacturer) refused to make Aero drivers for Windows and under Ubuntu I can use Compiz (much better anyway)

Also the Ubuntu community is very helpful with problems (that's why these places exist) and Canonical releases new software nearly every day and new Ubuntu releases every 6 months. 

One downside may be gaming, but because I have a PS3 I do all of the hardcore gaming on there and things like Minecraft on my PC.

Another thing I like is customisation. If you don't like a certain part of Ubuntu, just swap it out for a different one! For example, if you don't like the Unity interface, you can install GNOME 3 instead. Windows doesn't allow you to do this as much

Also if my system breaks I don't have to resort to torrent downloads or forking over £70 for a new copy of Windows - I can download a free copy of Ubuntu from the website

The software that's installed is a big bonus. Ubuntu comes with things like Firefox, Backup utilities, photo managers and E-mail right out of the box. Also, the Software Centre makes it really easy to install good software without the threat of viruses/malware. Windows hardly comes with anything. :)

Well I can say this...I've been using Ubuntu and other distros for quite some time. My laptop was dying and I wanted a cheap replacement. Got a great deal on a Toshiba netbook with Win 7 starter. That's why I recommend Linux to others. Fast, free, easy to install and use, plenty of apps that continue to improve. 
 
Use it long enough to get good at it and you'll realize the pains you feel trying to install and use it are just part of the learning curve, much like you would encounter if you were brand new to Windows.
