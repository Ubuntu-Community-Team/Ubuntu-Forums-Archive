### Post by aysiu on 2009-08-07
I think they both will win, which basically means Google will win.

Google is the huge profit-maker for Mozilla Firefox, so Firefox isn't really direct competition.

Chrome is just Google's way of saying "Hey, we want things as speedy as possible so people are more likely to use web applications, because we know if they use web applications, they're most likely to use *our* web applications (GMail, Google Docs, Google Calendar, Google Voice)."

As Chrome becomes more refined, it'll encourage other browsers (Firefox, Safari, Opera... even, Internet Explorer) to refine their JavaScript engines and page rendering times, which will still be good for Google. It's all a win-win for Google.

Sure, Google'd love it if people used Chrome, but a lot of people who use Firefox also still use Google and GMail, which means more advertising views and click-throughs, which means more revenue for Google.

If Google really viewed Firefox as competition in a zero-sum game, they'd find some kind of way to strong-arm Mozilla into not making Google the default search engine in the search bar (not sure how they'd do it, but if you have a lot of money and some lawyers, you usually can win somehow--see Pirate Bay trial for more details).

### Post by aysiu on 2009-08-07
In terms of sheer marketshare numbers, I think it will really depend on how the Google Chrome OS turns out.

If Google uses its corporate muscle to get major OEMs to not only **install** Chrome OS on netbooks but also **promote** the Chrome OS as well, then obviously more people will be using Chrome, because it will be the default browser (some Firefox diehards would install Firefox anyway, but most people stick with defaults).

The success of Chrome OS on the netbook will be entirely a matter of marketing and mindshare, not technical capability. Over the past two years, Microsoft has successfully ruined Linux's weak play for netbooks via preinstallation. Who was going to promote netbook Linux? Ubuntu? Xandros? Linpus? gOS? They just don't have the resources to compete with Microsoft in terms of advertising to the general public and coercion of OEMs. Where was Asus' "It's better with Linux" campaign?

You see how bitterly Microsoft and Apple are fighting it out with the whole "I'm a Mac, I'm a PC" ad campaigns. If only technical merit mattered, Microsoft wouldn't pay Dell and HP to plaster "_____ recommends Windows Vista for everyday computing" all over their websites.

Google isn't dumb. It also has, like Microsoft, billions of dollars at its fingertips.

You can see already that they're launching advertising for Google Docs over Microsoft's online Office. If Google puts serious advertising money into promoting Chrome OS over Windows 7, just enough people may buy into this whole netbook-as-netbook concept and not think of netbooks as just cheaper laptops, which will mean:

1. Firefox will still be a big player
2. Chrome will be a big player as well
3. Linux will get a lot more hardware support
4. More people using the Cloud, which will make for more Google bashing (over privacy issues) on these forums than the Microsoft bashing we currently see too much here
