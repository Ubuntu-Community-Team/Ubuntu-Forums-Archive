### Post by malspa on 2010-05-20
> **MacUntu said:**
> Chromium is close to 400,000 lines of code - you did a code review and can assure me there's no bad Google voodoo hidden? ;)

No, of course I did not.  Perhaps you're right and there's some Google hidden in Chromium, but I doubt it, and I'm not losing any sleep worrying about it, either.  It's all open source, so I figure that someone would find it if it was there.

My point, of course, is that they are not the same, but many people use the names Chromium and Chrome interchangeably.  Not everyone realizes that Chromium and Google Chrome are not the same web browser.  For example, in Linux I have **chromium-browser** installed, but I also have **google-chrome-unstable** installed in one distro and **google-chrome-beta** in another. 

More info: 

[http://news.softpedia.com/news/Google-Chrome-vs-Chromium-Understanding-Stable-Beta-Dev-Releases-and-Version-No-140060.shtml](http://news.softpedia.com/news/Google-Chrome-vs-Chromium-Understanding-Stable-Beta-Dev-Releases-and-Version-No-140060.shtml)

> Google Chrome vs Chromium

Another common question concerns the actual differences between Chromium and Google Chrome. In practical terms, they are minor. The short version, Google Chrome is Chromium with a different logo and a slightly different spelling. This is what Google Chrome adds to Chromium builds: 
- the Chrome logo and other branding changes;
- an auto-updater which downloads and installs the latest Chrome version from the respective channel;
- an option, enabled by default, to send anonymous usage statistics to Google;
- code to send stats like installs and country info back to Google, no option to disable it;

I got java working by just running these three commands:

$ sudo apt-get install sun-java6-jre sun-java6-plugin

$ sudo ln -sf /usr/local/jre1.6.0_16/lib/i386/libnpjp2.so /usr/lib/chromium-browser/plugins/

$ sudo rm -rf /usr/lib/chromium-browser/plugins/libjavaplugin_oji.so

went to this website and it works
[http://www.java.com/en/download/help/testvm.xml](http://www.java.com/en/download/help/testvm.xml)
