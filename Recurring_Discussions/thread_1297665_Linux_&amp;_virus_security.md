### Post by smileyguy on 2009-10-22
Does Linux (or Ubuntu) not suffer from viruses because it is inherently secure, or is it just because virus creators don't create many (or any) viruses for it?

---

### Post by Nerd King on 2009-10-22
A number of reasons:

1. *nix systems are generally more secure because of the way they handle user priveliges and file permissions.
2. There are fewer people on *nix systems (though this can be countered by the fact that they are often more interesting targets, ie a webserver holding credit card details is more useful than joe bloggs gaming PC)
3. Generally *nix users are more tech-savvy so social engineering is less likely to work.
