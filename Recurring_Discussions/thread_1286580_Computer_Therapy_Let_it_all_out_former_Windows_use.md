### Post by Eisenwinter on 2009-10-09
I don't like Windows as a main operating system, but I think it has great hardware support, and is generally reliable for certain tasks.

I still use Windows for music production, and sound editing in general, it has unmatched support for sound editing applications.

It still does crash every now and then, but hey, I've learned to always save my work in time, mostly ;)

So yeah, I don't hate Windows. I don't love it, but don't hate it.

I prefer to use Linux as my main operating system, as I find it more secure, and that it gives me more control over what I'm doing. 

I also find problem solving to be easier in Linux, because I have easy access to the deepest parts of the system, if I wish to have this access.

As far as this thread goes, I wouldn't describe it as "computer therapy". It's simply stating your opinion regarding Windows as an operating system platform.
