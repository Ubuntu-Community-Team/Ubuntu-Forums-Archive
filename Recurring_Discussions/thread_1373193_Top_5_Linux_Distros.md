### Post by Dark Aspect on 2010-01-05
> **gutterslob said:**
> 
Arch
Tried it recently, with mostly good results.....great package management and rolling release model, but has given me small problems (mostly hal related)


Did you add ck-launch-session in your .xinitrc file? That fixs most hal related problems. Also you need to add hal in the rc.conf array?

**.xinitrc**

```
exec *ck-launch-session* gnome-session
```

**rc.conf**

```
DAEMONS=(syslog-ng !network netfs crond *hal* networkmanager fam alsa)
```
