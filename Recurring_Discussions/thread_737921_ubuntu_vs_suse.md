### Post by k2t0f12d on 2008-03-29
> **malluda said:**
> i looked in suse site where there is a dvd installation 4.1GB.
Does ubuntu have dvd installation?

The openSUSE DVD is installation **only** media.  That is, it does not boot into a usable desktop environment, but instead immediately begins a very user-friendly native installation.  I have found Ubuntu DVD installation media; however, using the DVDs do not offer any advantages.

When you download the DVD media, you are getting a set version of every software package on the disc.  As soon as you install you will get the same basic installation that you can get with a LiveCD, and have to do updates anyway.  The only reason to have the DVD is if you have no internet at all and *any* version of the software packages is better then none at all.

> **malluda said:**
> will ubuntu be complete if i choose cd installation?

You will get the same installation with either the DVD or the CD, that is if you can even find out where to download a DVD iso for Ubuntu.

> **malluda said:**
> why suse is 4GB whereas ubuntu only 700?

I use openSUSE KDE LiveCd (~700mb) to install openSUSE.  It does the same installation that the DVD does.  Since I use the online repositories, having the DVD doesn't offer me any advantages.  openSUSE also offers GNOME LiveCDs.

> **malluda said:**
> what additions does suse have versus ubuntu ?

openSUSE and Ubuntu have substantively the same software.  openSUSE may handle non-free software differently from Ubuntu and uses a completely different package management system.  Its administration is almost totally controllable from the YaST2 gui program (meaning very little to no dependence on using the console).
