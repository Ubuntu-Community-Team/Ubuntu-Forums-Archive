### Post by dca on 2008-06-26
Because the source code is freely available to use, edit, redistribute, etc, etc unlike Microsoft products where everything is prorietary and closed source.  This goes for the kernel itself and the other utilities and apps.  Open source gives you the ability to create whatever you want.  You can take the linux kernel, add a bunch of open source utils, and create a server oriented distro.  You can take the Linux kernel, tweak the heck out of it, add a bunch of little open source projects, and install it on a cell phone.
 
The funny thing about 'Leo' is they really didn't have to do anything.  All the hardware (very small list, look at Apple's product line) was laid out in front them.  No variation, no ability to swap WiFi cards or other components when defective and replace w/ generic.  Heck, MS has over 20k paid programmers and they still can't create an OS that doesn't require reinstallation every six mos....

There is only one 'Linux.' And that is the kernel at the heart of you Operating System. The parts that make up all these different distributions are the different extra bits of software that make it look or behave a certain way. Comparing Vista/XP to OpenSuse/Ubuntu is a bit silly, as Vista is simply a newer version of Windows as compared to XP. The reason there are all these different distros is because the heart of free software and GNU/Linux is freedom and the ability to choose whatever software you would like to run on top of your kernel.
