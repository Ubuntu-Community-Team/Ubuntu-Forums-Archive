Installing applications for example. How anyone can say Googling for a download, finding the right download, downloading it and running through an install wizard is easier than typing the application name in the Software Center and clicking install is baffling.

Hardware support. Seriously? Has anyone here ever tried installing Windows on a blank PC? What you end up with is a device manager list of hardware with no drivers. I've seen Windows installs where I've needed motherboard drivers(!), graphics drivers, ethernet drivers, wifi drivers, ADSL modem drivers, 3G dongle drivers, printer drivers and sound drivers. Yes things have improved a little in Win7 but not by a lot. Let's compare Apples to Apples here. If you buy Ubuntu pre-installed, like you'd buy Windows pre-installed, hardware support isn't an issue.

As for command line stuff. Yesterday I spent several hours trying to fix a Windows install that had suffered some filesystem corruption. I needed to fiddle with registry entries, use the recovery console on the CD and run obscure commands at the command prompt. and I still can't get the automatic updates service to work. The average user would have just spent £60 to get Windows reinstalled at a shop. We do fail epically on these forums when we paste someone an apt-get line rather than pointing them to the software centre, we need to stop doing that if we want this "you need to use the command line to use Ubuntu" myth to disappear. 

There are valid reasons why Ubuntu isn't quite up to scratch for some users. For example, Blu-ray playback, or running Windows games, or Windows apps. But again, if you gave the same user a Mac (the pinnacle of usability apparently) then your Windows games and applications aren't gonna work there either - and Macs seem fit for the common user.

When it comes down to it, if you gave a pre-installed Ubuntu PC with the codecs etc. to the average Joe and walked away, they'd be fine with it. They'd be able to check Facebook and play Farmville. Many of the "problems" that get listed are because the user either doesn't want to learn at all, or they try to enforce a Windows mentality onto Ubuntu (e.g. Googling for an app installer or trying to install iTunes rather than use Rhythmbox). 

Arguably we could do more on the first login to educate the user, or offer the option of printing the Ubuntu Manual on first login if they have a printer.

So, yes. If Windows is ready for the average user, and Mac is ready for the average user then I believe Ubuntu definitely is.
