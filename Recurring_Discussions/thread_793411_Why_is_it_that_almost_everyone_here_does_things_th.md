### Post by aysiu on 2008-05-13
Terminal commands are handy for several reasons: [list][*]The same command works in Gnome, Xfce, KDE, IceWM, Fluxbox, etc. Point-and-click instructions are graphical environment-specific and may even change from version to version within the same environment. [*]Commands can be copied and pasted. Point-and-click instructions must be followed. [*]Commands give useful-for-diagnosis error messages if the command fails. [*]This is a text-based forum.[/list]
