### Post by JKyleOKC on 2012-10-31
Not quite the same, Snowpine, but close.

Xubuntu has a different set of application programs than does Ubuntu; adding XFCE to a Ubuntu install doesn't change the word processor from LibreOffice to Abiword, for example. The add-ons in Xubuntu are selected to be "lighter" in their use of resources, so will be somewhat snappier than the Ubuntu ones. However the difference is so small that it usually goes unnoticed.

We who use Xubuntu rapidly learn that answers given by the Ubuntu folk (who are of course the majority) often fail to directly apply to our Xubuntu systems. Configuration files are in different places, the menus are different, and so on. Thus Ubuntu unmodified is probably the simplest for newcomers, since most responses to cries for help will apply directly to it. Ubuntu plus XFCE would be the second simplest, but responses would have to be evaluated more closely. Xubuntu is most likely to get not-quite-valid responses, but enough of us who use it are around to help counteract this.

Any of the three, though, would be a good choice. And I'll add my welcome here...

### Post by Peripheral Visionary on 2012-11-04
Ubuntu "with Xfce tied on" wouldn't have Abiword and Gnumeric and other "lightweight" applications in place of the heavier LibreOffice and other stuff.

**The "xubuntu-desktop" package adds Xfce _plus_ all the other stuff that comes on a standard Xubuntu CD! ** Including the default themes and artwork and settings and such.  And all the applications that come on a Xubuntu CD.

**If you just want to try out the Xfce desktop environment, install only Xfce - not "xubuntu desktop."**
