### Post by Hagar Delest on 2011-03-02
Sun had announced that it would set up a foundation but it has never been done. Since Oracle has a rather bad image in the open source world, some of the OOo community decided to fork after Oracle bought Sun.

OOo is not obsolete and there is no conflict of interest leading to the fork (it's the opposite: LibO founders have been asked to quit the community council since they wanted to set up another community).

LibO wants to be more open source and protect the code by the multiplication of the contributions (in OOo, the coders give their code to Sun/Oracle).

Now, the switch depends on your use of the software. Personally, there are some differences that make LibO unusable for me so I stick to OOo. Up to the user to chose the best software for his needs.
