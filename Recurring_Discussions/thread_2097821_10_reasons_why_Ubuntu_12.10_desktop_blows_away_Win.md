[ this article ]("http://www.theregister.co.uk/2004/10/22/security_report_windows_vs_linux/") convinced me to load my first Linux system

it's a little dated and Windows has done much in UAC to tighten up . but fundamental concerns remain . Windows was developed from the old IBM 5150 DOS system -- which was made to be modified as desired by the software developer and/or user . Linux is derived from the UNIX system -- fundamentally designed for commercial use with security built in from the start

in the world today one must really pause and reflect on the Real Purpose of Software . Some of it is for "data mining" : gathering information about the people who are using that software 

from what I read Linux is more aligned with the FSF -- free software concept -- wherein software is developed by the community and shared by the community . the source is required to be open so that it can be inspected and improved . which means we have a much better chance to know what the software is really doing ... Cheers to Mr. Stallman .

maybe I wear my tin foil hat too much . but I read an awful lot about "Big Data" and "Clouds" , and every day hacking, --

it's a Scary World out there in cyberspace 

interestingly it may be the mercantilist system demand to protect 'intellectual property' that finally pushes security into software . e/commerce doesn't seem to provide enough motive to induce the change that is needed 

keep an eye on UEFI . that is where a software audit has to begin .

### Post by irv on 2012-12-26
Here is my list of 10 reasons why Ubuntu 12.04 or 12.10 with Unity desktop blows away Windows.

> 1. Better interface.
2. It just runs. I don't have to keep rebooting it because of lockups and slowdowns.
3. It runs faster on my older laptop. Windows runs much slower and programs load slower also.
4. Windows needs virus protection which slows it down. I don't worry about virus' or malware in Ubuntu.
5. Windows does not have a Dash home where I can run lens.
6. I can find things much faster on Unity desktop. I find the search much better in Ubuntu.
7. For the most part I find programs running under Ubuntu much better then Windows programs. There are a few windows programs I like better, but I can run them under Wine or a Virualbox.
8. I love the fact I can do so much at the command prompt.
9. Much easier to install software.
10. And last but not least, the support is so much better. This forum for one and the Linux community in general is just great. It seems I can always find an answer to a problem with Ubuntu, where windows users and MS are not that helpful.
