### Post by p_quarles on 2008-04-23
KDE4 is not really ready for anyone but the KDE die-hards at this point, I think. 4.1 is still the target "finished" release, so it might be worth holding off one's judgment until then. 

As for it being anything like Vista: I don't get that. "Too much like Windows" is the charge that's been levelled against KDE for a while now and -- quite frankly -- it's an extremely surface perception. While KDE has never been shy about taking good ideas from Windows, it is really quite a bit different in most ways. The most notable thing, in my view, is its ability to essentially simulate shell piping in a graphical environment: KDE's ability to embed its core apps into pretty much anything else is quite impressive. Configure Kate the way you like it, and it's the same when you open up Quanta Plus or KDevelop. 

KDE4, specifically, is a much-needed new release. Some of the code in 3.X is messy enough that it's shown some pretty persistent bugs. The applications that have fully-working releases for KDE4, so far, seem like much cleaner versions of what is in principle the same software.
