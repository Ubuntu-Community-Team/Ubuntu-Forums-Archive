### Post by Madspyman on 2011-08-06
If you really need a portable computer with great specs a laptop is a better choice than a netbook. The problem is that less expensive netbooks appear to do more, but don't have the power to run some of the software compatible with the OS installed on it. 

Tablets are different then a netbook because the apps on the tablet are created for mobile use and work better than trying to install software on a netbook that it's specs of it can't handle. 

Overall the tablet experience is designed to be a better one in that you usually can't install anything on it that it's specs can't handle. Also the software is easily available (same for a Linux netbook, but not a windows netbook which are usually what average consumers buy). Plus the fact that you don't need to set it down to use it, it isn't bulky to carry around, and it has a great form factor for mobile gaming.

Basically, netbooks are small laptops that can awkwardly run software usually designed for larger laptops. Tablets are touchscreen mobile devices built to run mobile software designed to provide a mobile friendly experience.
