### Post by mörgæs on 2010-02-16
There are many kinds of Ubuntu to choose from. Ubuntu is the main distribution with Gnome graphical user interface. KDE as used in Kubuntu looks a little more like Windows. Xubuntu is the lightweight member of the family.

There are lots of less-known Ubuntu distributions (in short: Distros) :
[http://en.wikipedia.org/wiki/List_of_Linux_distributions#Ubuntu-based](http://en.wikipedia.org/wiki/List_of_Linux_distributions#Ubuntu-based)

You can try each of them as a live CD, if you want to take a look before you decide.
