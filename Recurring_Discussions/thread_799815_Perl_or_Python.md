### Post by pmasiar on 2008-05-19
Your feeling is right, Python is (slowly) replacing Perl, but many old hands (some are around here) who know Perl, do not see why. Reason is: because both Perl and Python are very close in features, and readability counts only for people not using Perl daily. 

IMHO time spent on learning Python is well spent, Python is becoming more popular, even if Perl might have bigger codebase. many people (including me) left Perl because code is just not readable, after you did not used Perl for a year. Python is more forgiving in this sense.

Try to learn Python in a weekend, if Python clicks, implement it in Python. If not, do it in Perl.

See also [http://learnpython.pbwiki.com/PythonTricks](http://learnpython.pbwiki.com/PythonTricks)
