### Post by mcduck on 2009-08-10
Why? Because not everybody uses computer for gaming. Some of us actually do work with them. ;)

I play my games on consoles, and earn my living with my computer. With that background it's absolutely meaningless if Windows games can be run on Linux or not, I just want the OS that allows me to do my work in the most efficient way and minimum amount of time wasted in maintaining the OS itself.

You really shouldn't expect *any* Windows software to run on Linux. Programs made for one operating system aren't even supposed to be usable in any other operating sysrtem. You can't run Linux or OSX programs on Windows, you can't run Windows or Linux programs on OSX, and the fact that you are able to get even some Windows programs running on Linux should be considered as a nice bonus instead of something you'd expect to work.
