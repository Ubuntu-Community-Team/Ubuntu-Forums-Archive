### Post by Zorael on 2008-06-17
There is no answer to the question of which of them is "best". When wolves have swallowed the sun and the moon, when the stars have been snuffed and the earth trembles at the oncoming storm of Ragnarök, Gnomelings and KDElings will still fight over which is better.

To install it, just grab the **kubuntu-desktop** package in Synaptic or through terminal commands.
```
$ sudo aptitude install kubuntu-desktop
```
It will install KDE as well as the standard Kubuntu applications; Amarok, K3b, Kaffeine, etc.

If you *only* want KDE (and as such, miss out on the whole "desktop environment"), just grab **kdebase-bin** instead.
```
$ sudo aptitude install kdebase-bin
```


...or just burn a live CD of Kubuntu and reboot into it. :>
