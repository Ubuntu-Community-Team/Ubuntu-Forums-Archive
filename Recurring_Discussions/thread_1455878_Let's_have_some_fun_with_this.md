### Post by e4uforums on 2010-04-16
Once compiz is installed, open a terminal and type:

```
ccsm
```This will bring up compiz settings manager.

On this screen there is a checkbox you can use to enable the desktop cube.  There is also a checkbox for animations.  Check the box next to animations, then click on the word animations itself.  This will allow you to adjust the effects and behavior of windows.

Have some fun with it and play around with the settings.

### Post by Naggobot on 2010-04-16
I also have the cube working, a good starting point is to have 4 desktops. This can be adjusted from the bottom right corner (where there are normally 2 desktops visible on the switcher). Right click -> preferences -> set columns 4 and rows 1. 

Also you need to set "rotate cube" from ccsm, otherwise you have cube which does not rotate :roll:. 

Also with Compiz you can drag windows from one desktop to another over the cube edge which is a handy feature when you get familiar with it. 

With 9.10 you also can set multiple backgrounds (i.e. back desktop background switches automatically as a slide show). I do not know which OS had this feature first or which OS has  and this but I think it is cool. Admittedly there is very little practical use for it but I still like it and use it to rotate through a set of images (even though some times it tends to overlap multiple images).
