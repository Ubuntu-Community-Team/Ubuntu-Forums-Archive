### Post by Quarkrad on 2011-09-19
I'm a newbie although I've been using Ubuntu since 8.10. A few months ago I had to get a new pc and the beast was 64bit.  To be honest, in day-to-day use I have not had any issues - I was a bit nervous in that I was not sure if there were  applications that I used that would not work in a 64bit environment. However, I installed a clean 64bit version (10.10) from scratch and it has been great.  I did not load 11.04 because I do not like Unity - I have a large monitor and do not like my Desktop looking like a Netbook Desktop - but I doubt if there is any difference in 11.04 64bit to 10.10 64bit.  In terms of performance I have no idea because I would have to have the same set up running 32 bit.  What I can say is I have had no issues - it is the same robust,fast , reliable OS.  Load it up and enjoy!
