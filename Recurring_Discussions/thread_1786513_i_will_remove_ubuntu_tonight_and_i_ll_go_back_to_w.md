### Post by walt.smith1960 on 2011-06-19
I always multi boot Windows & various Linux flavors.  The surest way to have problems is to put all your eggs in one basket. I haven't started Windows in months but sure as shootin' if I deleted it tomorrow morning I'd need it  tomorrow afternoon#-o.  Disk space is pretty cheap(unless you have an SSD) and I don't make the partitions larger than needed.  Use Linux when you feel like and if you need Windows, start Windows.  When I was exclusively Windows I had 2 installs; one used only for important stuff with minimal online exposure and another install for everything else.  Image the everyday partition when fresh and when (not if) that installation craps the bed, delete the partition and restore a fresh image.  Keep your data files in a partition separate from the OS.
