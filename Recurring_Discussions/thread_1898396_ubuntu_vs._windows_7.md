### Post by guyver_dio on 2011-12-23
Windows 7 is BIG, gigs compared to 700mb for ubuntu (ubuntu plus all the linux programs I use installed it's only a little bigger than windows on it's own). It takes SO much longer to install windows 7 than ubuntu (even if you connect to the net and download updates during the ubuntu installation). In my experience it takes much longer for windows to boot up (ubuntu is probably about 4-5 times faster).

Windows also uses alot of resources just idling, though with unity ubuntu is starting to use more too but then take into account you really need antivirus software and maybe a firewall installed with windows so that really starts effecting performance (especially if you use bloatware like norton). 

Performance in terms of speed (like opening programs etc..) is a tough area. I'd say a fresh install of windows, they are probably tied, windows is very responsive in the beginning and I've found ubuntu 11.10 to be a little less responsive than my previous experiences with ubuntu. However windows after a few months of use starts to lag up a bit, and the time you need to wait for windows to become usable at startup gets longer and longer. Ubuntu is responsive and usable within seconds of starting up, no matter how much you've used it (unless you've told it to startup alot of programs at startup but then that's really your own fault). Windows also starts to lag up if you've had it running constantly for several days, it's not very good at handling resources.

The only real upside to windows I feel is compatibility, everything is made for it before any other OS if any other OS at all but that really hasn't got anything to do with the OS at all, that's just because of market share. If all you swapped was ubuntu's and windows market share, I think ubuntu would do a much better job than windows has and certainly be much more respected. 

Go read lots of reasons why people like windows over ubuntu, it's always more games are available and run much better, lot more software available, all hardware is supported etc... Those aren't really comments about the OS itself, that's just the nature of having most of the world using the operating system.
