### Post by evilsoup on 2012-10-19
The only thing I'll uninstall will be the shopping lens, I think.

I'll install all the codecs, flash, etc. The GIMP, VLC, guake, get-iplayer (and probably RTMPdump), mplayer, ffmpeg, handbrake, ps3mediaserver.

There's an excellent Firefox add-on, 'open with', which allows me to right-click youtube links and open them in VLC (works much better than flash ever did, for videos). Plus the usual Adblock, noscript, etc.

The one I would definitely recommend to everyone is [easystroke]("https://apps.ubuntu.com/cat/applications/easystroke/"), I always hate using other people's computers without my mouse gestures.

I've got a few old windows games, so wine too.

There are also a few commandline utilities, but I can't remember which ones are installed by default.
