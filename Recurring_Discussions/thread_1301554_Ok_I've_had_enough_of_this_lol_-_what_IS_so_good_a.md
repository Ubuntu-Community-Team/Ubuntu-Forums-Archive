The new control panel, homegroup, wifi, tweaks to UAC and libraries are also very nice. This isnt
In conclusion I would say that Windows 7 is worth it if you can get it cheap (like me) but if you have to pay full price I would stick with XP or Vista if you want a Microsoft OS.

Windows 7 isn't all smooth sailing though and has it's faults (poor install experience, not as secure as Ubuntu, isn't all that different to Vista, expensive.)

For now Windows 7 is my default OS as Ubuntu doesn't seem to run well on my computer (ATI graphics...) and until ATI sorts out their driver on Linux I am perfectly happy running Windows 7.
