### Post by witeshark17 on 2009-12-04
IMO, the finding that the windows registry is a central point of security failure is the most important. More than enough repeated events involving registry edits that cannot be stopped and actually target security software support this main point. AFAIK, attempts to make the registry behave as "stealthy" have only wasted system resources without resulting in a real, long lasting defence. No other OS uses such a system core.
