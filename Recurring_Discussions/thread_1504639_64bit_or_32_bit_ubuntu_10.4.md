### Post by LowSky on 2010-06-08
I can't believe how much confusion there still is with 64bit processors, they have been availible since 2003.


Everything should be fine. Except for Adobe Flash, you will need the 64Bit alpha version from [HERE]("http://download.macromedia.com/pub/labs/flashplayer10/libflashplayer-10.0.45.2.linux-x86_64.so.tar.gz")

All you will need to do extract the file and add it to the hidden folder named **~/.mozilla/plugins** folder, which can be found in your ** /home** folder
