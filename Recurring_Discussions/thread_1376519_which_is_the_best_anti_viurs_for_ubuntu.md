### Post by Project PWNED on 2010-01-09
There's no need for virus scanners in Ubuntu; you cannot infect yourself with Windows viruses. There are virus scanners for Linux (Ubuntu), but mainly for people hosting SMB shares or mail servers, where Windows clients interact with the Linux system and a virus scanner would be needed to check the files.

### Post by ooVoh9em on 2010-01-09
You do not need to run an Anti-Virus on your Ubuntu desktop to increase your security. Most of these products will run under Linux natively, but they simply scan for known Windows-based threats. 

If you are not running service where data is filtered through your computer before it reaches Windows clients, then it is not in your best interest to run an Anti-Virus on a *nix-based system.

Ubuntu also comes shipped with no Internet-facing services, so you also do not need to run a firewall on your desktop unless you install some sort of server software. 

If you would like more information about security-related topics please follow a few of these useful links: 

- [http://ubuntuforums.org/showthread.php?t=510812](http://ubuntuforums.org/showthread.php?t=510812)

- [https://help.ubuntu.com/community/Security](https://help.ubuntu.com/community/Security)
