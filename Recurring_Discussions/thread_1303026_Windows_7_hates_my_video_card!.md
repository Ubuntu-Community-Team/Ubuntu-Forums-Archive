### Post by cascade9 on 2009-10-29
That 'something in vista' isn't really in vista. Its a screenshot of some system specs, but thats for onboard video. 

Dedicated RAM = how much the BIOS has set to be used all the time by video. Shared RAM = how much more can be used by the video, if it needs to, but can also be used as system RAM. 

As for what NoaHall was talking about, the name varies depending on who made the graphics chips. ATI calls it HyperMemory, nVidia calls it TurboCache.
