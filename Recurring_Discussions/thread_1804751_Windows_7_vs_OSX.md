### Post by Mr. Shannon on 2011-07-16
I'm in a good position to know which OS I like since I'v use them all.  I ran windows XP for years, then switched to Ubuntu last Christmas and at work I use Mac OS X Leopard (I think).

As far as the graphical user interface (gui) is concerned I prefer Windows (not 7) and Ubuntu Classic.  I'm not the biggest fan of the Dock.

For software it would again have to be Windows or Linux.  Windows has good backwards compatibly (at least until recently).  While in Linux most of my software is free so having to get a new version of the software is not a problem.  I don't really know how well Mac OS X software is backwards compatible.  Of course, assuming all of the open source software that I use on Linux is available for the other OS's then this is mostly a mute point.

Now for the meat of the OS.  I do most of my work with a text editor (vim), a terminal, MATLAB/Mathematica, and a web browser (I may have to use MS Word and Excel next semester).  Since Mac OS X is Unix based, the terminal is the same as Linux in many ways.  So for the majority of what I do I think I would rather have Mac than Windows.

It comes down to if someone where to offer to give me a PC with Windows or a Mac (for some reason Linux was not allowed to be installed on these computers) I would probably choose the Mac as long as I did not have to buy software for it.  Even though I don't care for its gui (except for the Expose feature).  I just have to many bad memory's trying to figure out how to set up a development environment on Windows.  I was new to programming at that point though.
