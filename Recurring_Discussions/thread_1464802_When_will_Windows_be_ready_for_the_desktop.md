### Post by chessnerd on 2010-04-29
(You can substitute "Ubuntu/Linux" for "Windows" and "Ubuntu 10.04" for "Windows 7" if you want to read this as talking about "The Year of Linux on the Desktop." I've written it to work both ways.)

For a long time people have wondered about when Windows would FINALLY be ready for the "average user." Since Windows first came out it has had a hard time really catching on. Sure, users here and there could pick it up and use it effectively, but others would be confused, frustrated, and could even lose data after frying their installs.

Now, I think 2010 is the long prophesied "Year of Windows on the Desktop." Windows has a long history of not pulling through in this regard, but the most recent release, Windows 7, shows a lot of promise.

Windows 7 is reinventing the Windows UI and changing the way that users are asked to look at Windows. Old slogans and brands are out, new ones are in. The way that users interact with Windows is different when using Windows 7. It's easier, faster, and simpler than previous versions.

And Windows 7 didn't just get a face-lift. Compared to its predecessor it's much more stable and shouldn't get the same level of flack. It offers better hardware support and boasts fewer crashes and failures. I, myself, have used Windows 7 for some time and I can tell you that it's pretty solid. At first, the UI changes confused me and I didn't like them, but I've grown to know and appreciate the changes despite the fact that the changes make it feels a bit more Mac-ish than previous versions.

All in all, Windows 7 could (maybe) lead to a break through for Windows and allow users to see that Windows has true potential.

I tell you: this is the year of Windows on the Desktop!
