### Post by prshah on 2010-10-01
> **Bear number one said:**
> I was under the impression that any less than 4gb ram would not be recognised to its full potential by 64bit. Have I got this wrong

Yep, you HAVE got it wrong. You can use 64bit with less than 4GB ram provided you have 64-bit capable hardware (Eg, Core 2 Duo CPU).

I use 64-bit on a C2D1.6/2GB RAM and it runs superbly. There is no specific performance improvement in day-to-day usage, but it really shines when doing any CPU intensive video/audio/graphics encoding.

In fact, in day-to-day usage, 64-bit's memory usage is slightly over that of 32-bit, but the difference is not significant.

Summary: if you have capable hardware and over 2GB ram (not a REQUIREMENT, just a suggestion), please use 64bit, ESPECIALLY SO if you work with video/audio/graphics creation and/or editing.
