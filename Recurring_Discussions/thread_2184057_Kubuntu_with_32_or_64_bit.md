### Post by deadflowr on 2013-10-27
Cool, it's the recurring discussion of 32 bit vs 64 bit with a twist.

The recommendations that are on the various Ubuntu and Ubuntu-based pages of using 32 bit, is mainly for users who do not know what arch they have. 32 bit should run on most any system, where as 64 bit won't.

If you know what you have and it is capable of running 64 bit, then don't worry about the recommendation.

FYI, I run Kubuntu with 64 bit and it runs awesome.
