When I started with Ubuntu...

1) There was no wireless support to speak of. I think you had to use command line utilities to associate with wireless networks and you couldn't easily switch between them. Today: network-manager allows you to easily use any supported wireless card to automatically connect to wireless networks in range. It supports all methods of authentication. Click click and done!

2) There was no boot screen for Ubuntu. Just a bunch of text scrolling. Very scary looking. Today: I'd like to see at least a little info of what's going on in the background!

3) If you wanted to install video drivers you would have to go to the ATi/nVidia site and install the drivers yourself and manually. Then you would have to do it again if there was a kernel upgrade. Today: you get a pop up saying there are drivers available: click here to install them. It retrieves and installs them automatically!

4) If you wanted to play an audio or video file that didn't have a decoder installed you'd just get weird symptoms or error messages. Today: it tells you it needs a codec, searches and installs it for you!

5) You couldn't use the internet for video. Flash lagged two versions behind that of other platforms. Today you pretty much get the same experience everyone else does!

6) Back then, you'd be lucky if you found a web cam that worked with Linux. Today most web cams work, out-of-box and automatically. Every system I try, the web cam works. Hooray for UVC!

7) If you typed a command into the terminal you might get something like "bad command or filename". Today: it says program / command not installed, install it by using sudo apt-get install command!

8) To use the best refreshrate and resolution for your monitor you would sometimes have to manually edit xorg.conf, today there is NO xorg.conf!

9) If a program crashed or behaved badly, there was a big barrier to new users to report that bug. Today all you have to do is click "send crash report" and it will automatically formulate a bug report for your app with all the related logs and attachments. All you have to do is describe the bug!

10) There wasn't really any feasible way for regular users to do video editing on Ubuntu. Today there are multiple emerging products. One is even being included by default in the next release (Pitivi)!

--

These are just things from off the top of my head. The Ubuntu desktop has progressed enormously since I started using it and I was happy with it back then! Slow incremental progress can sometimes make us forget that there is any. You just seem to have lucked out with your hardware. Get some decent supported hardware and you'll notice that Ubuntu is really quite awesome.
