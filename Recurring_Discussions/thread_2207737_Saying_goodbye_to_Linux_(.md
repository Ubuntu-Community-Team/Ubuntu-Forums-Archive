### Post by Profetak on 2014-02-25
First of all, I love Linux. I&#8217;m an Esperanto speaker and supporter, and one of the nicknames of Esperanto is &#8220;the Linux of languages&#8221;. Both Esperanto and Linux share some ideological points in common: freedom, cooperation, sharing, respecting, etc. I went to an Esperanto congress recently - try to imagine the scene I found: hundreds of people from all over even from different countries, a great number of vegetarians, gays, mystical people, spiritualists, atheists, alternative people, weird people, people that believe in extraterrestrials, and PEOPLE THAT USE LINUX! :p All of them speaking this engineered artificial language created by a nerd more than a hundred years ago. I really felt at home! :D
 
I may be exaggerating but not kidding. I&#8217;ve seen many esperantists&#8217; laptops running Linux. And whenever we went to a restaurant with a group of esperantists, almost half of them were vegetarians! One night we even made a vegan dinner! And to go out at night, many opted to go to gay bars! I guess these ideologies strongly correlate. That&#8217;s why I love Linux so much! :D
 
I&#8217;ve been using (trying to use) Linux for a couple of years (my PC came from factory with Mandriva, then I tried Ubuntu and Linux Mint). Unfortunately, I&#8217;ve been having many problems. :( For example:
- In my work as a teacher, I receive and have to make many files in doc, docx and ppt. LibreOffice almost always messes up the formatting. 
- I couldn't see the preview of my photographs in high resolution. So if I wanted to find a specific photo in a folder containing 300 hundred photos, I'd have to open one by one.
- Whenever I recorded sounds with Audacity, the speed of the sound recorded was always in slow motion.
- The quality of jpeg files originated in any program I tried was very poor (at least compared to files saved in Windows Paint).
- If I tried to watch videos in some specific websites, the system crashed and the only solution was reboot. So I simply couldn't use some favorite websites.
- I formatted my usb stick with Ubuntu and it made it unusable. The only solution was to format it in Microsoft Windows using a friend's computer.
- Most programs installed in Wine didn&#8217;t work properly. Some didn&#8217;t work at all. 
- Ubuntu default web browser, Firefox, refused to enter my yahoo account. I had to intall other browser.
- As Google Drive is still not available for Linux, we have the option of using something like SyncDrive to synchronize our files in the Google cloud. But was it slow!  
- I bought a Xerox Printer. No way to make it work! I had some options: patiently search for &#8220;a few&#8221; more hours for a solution, return the printer or reinstall Windows. :/
- Ok! I decided to install Windows on VirtualBox - only that I had to spend some more hours learning how to do that. After I got Windows installed, there was no way VirtualBox would recognize the USB port - more hours searching and no solution. :/
- Skype in Unity is hell! On Microsoft Windows, after you minimize Skype, it continues running on the background and it's always there on the taskbar near the clock. Only that Unity doesn't have such a taskbar and Skype vanishes but continues working! So I couldn't actually use it.
- I thought: &#8220;Ok! I'll install a taskbar in Unity.&#8221; Only that it was very difficult to use. :/
- I thought: &#8220;Ok! I'll install a Cinnamon desktop and have a Windows-looking environment.&#8221; Only that it made my computer really slow. :/
- I thought: &#8220;Ok! I'll uninstall Cinnamon!&#8221; Only that I tried a lot but couldn't. :/
- Ok, I&#8217;ll reset Ubuntu. &#8211; I typed some commands I found that were supposed to reset Ubuntu, but it made it worse. Programs started crashing and if I opened the Cinnemon desktop, the screen would freeze and I had to reboot. :/
- I thought: &#8220;Ok! I'll format my computer!&#8221; Only that I couldn't. The machine wouldn't recognize the installation CD, and the initial screen to access the BIOS setup (which was always there when Windows was running) had disappeared after I installed Ubuntu. So whenever I rebooted, or turned on the PC, Ubuntu would start first thing, without giving me the chance to access the installation CD or access the BIOS. ](*,)
- So, at this point in time I was with a slow crashing OS. I couldn&#8217;t fix it, and I couldn&#8217;t format my computer &#8211; it was practically a useless machine. I was about to take my computer to a professional to have it fixed. But then somebody told me I had to actually open the computer&#8217;s system box, and remove the CPU battery for some minutes and this would reset the BIOS. Never in my life had I opened a computer! :frown: But in a moment of despair, I did it. When I turned on the PC again, the BIOS setup screen reappeared (thank God!), so I could boot from the installation CD and get rid of Ubuntu (very sad but true). 
 
I NEVER THOUGHT I'D FEEL SO RELIEVED UPON SEEING THE SYMBOL OF MICROSOFT ON MY COMPUTER SCREEN. :frown:
 
I made a sincere effort to use Linux for some years and I&#8217;ve always enthusiastically told everyone around me to do the same. But with my experience, can I spread the word to everyone that Linux is really as good as enthusiasts say? I&#8217;d like to, but I don&#8217;t know. 
Ok, I recognize that all those problems I faced may have simple solutions (but I couldn&#8217;t find them). Or maybe I messed up and caused some of those problems. Maybe I could have fixed most problems if had searched for more hours, but why should it take hours? I can&#8217;t afford spending hours on every problem I face. I have to admit: I had far less problems with Windows. But as I read somewhere, maybe &#8220;life is too short to use Linux&#8221;. 
 
That&#8217;s sad. :cry: I&#8217;m very fond of Linux. I know I&#8217;ll use it again one day... but not so soon.

### Post by tgalati4 on 2014-02-25
Linux does have a lot of rough edges.  Not unlike Esperanto.  If Esperanto was so great, we would all be speaking it now.  But, it does have merit and there is a thriving community around it.  This is similar to linux.

You will encounter dead ends in linux.  Trying to get a Xerox printer that relies exclusively on Windows processing is difficult if not impossible to install in linux.  Using WINE is a last resort.  You need to find replacement programs and workflows in linux.  Otherwise you will go crazy trying to use WINE for every program that you are familiar with in Windows.  

My stepson is a teacher and he uses a Mac (his own), but the entire school district uses outdated Windows machines.  So I hear his frustrations.  Linux won't solve school politics.

Dual-boot is a viable option.  You don't speak Esperanto all the time do you?  Boot into Windows when you need to use it, use linux to preserve your sanity.

Going full-time linux is not unlike becoming a full vegan.  It's a commitment that requires some research and hard work.
