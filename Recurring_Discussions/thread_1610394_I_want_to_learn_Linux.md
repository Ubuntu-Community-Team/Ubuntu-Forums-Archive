### Post by Triblaze on 2011-12-13
As many people have said, just use it and explore it. I've found the best thing for learning is a sense of Linux curiosity. Don't be afraid of your computer, see what it can do for you, and what you can do for it.
First, make sure it operates on the basics necessary for daily use. Internet, a word processor, a music player. Once that's settled, you've got a good foothold and can comfortably use it. From there, every so often, try something new. If you think "it'd be nice if I could do this", or "I wish I had something like this", go look it up and find out how to do it.
You'll eventually pick up how to do things naturally. Like the CLI. Ubuntu is pretty good at not needing it too much for the beginning user, but you learn to love it. It's a great tool. There are a lot of commands to learn, but these two are arguably the most important for someone like you: "apropos" and "man". apropos followed by a key word will find all commands that have to do with that key word, along with a brief description. Say you want to look for a command that lets you copy files. Enter: apropos copy. It'll give you a list of all commands that have to do with copying, including cp, which is probably what you're looking for.
The other command, man, is short for manual. It's exactly what it sounds like. Enter "man" followed by a command name to read the manual for that command. When you're done reading, press q to quit.

It just takes some time. You'll learn it soon enough. Though one is never really done learning.
