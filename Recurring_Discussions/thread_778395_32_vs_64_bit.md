### Post by mivo on 2008-05-01
> **NightwishFan said:**
> Some links for any interested. Also both flash and java work on my 64bit Ubuntu.

I generally value my own observations higher than third-party feelings on something I have experienced on my own system. More people should follow that guideline. ;)

There is no 64-bit version of Java Webstart, and there is no 64-bit Java browser plugin from Sun. (The free alternatives do not work with all Java applets.) You can use a 32-bit browser and the 32-bit plugins, and while that worked for me in Feisty and Gutsy, it failed to work in Hardy. For me and some others. It does work for some users, however.

Keep in mind that I used 64-bit versions of the last three Ubuntu releases. 32-bit Hardy isn't any slower for me. It uses about 10-15% less memory, though that doesn't matter for me. I'd prefer to use the 64-bit version, even if only out of principle, but for the time being the 32-bit version is more convenient and hassle-free for me. After my experience with trying to get the 32-bit Java Webstart (it's about javaws, not Java in general) to run in 64-bit Hardy, I'd not currently recommend it to someone who doesn't have 4 GB or more and doesn't do video editing and intense number crunching stuff. Your mileage may vary, of course, and the decision should be individual and based on one's software needs. I only "objected" to the statement that 64-bit is "overall faster", because that isn't my experience (on my system, with the software I use).
