In my experience, it's problems that have come from upstream (Ubuntu 11.10) all the issues I've had with Lisa are the same as I had with Oneiric Ocelot - REALLY irritating backlight issues, log out/shut down failures, there where more but it didn't stay more than the week. I ran it from a netbook, a desktop and a Mac (intel) and got different issues with each. 

I wouldn't say it was unstable - just VERY buggy in the MGSE GUI - applications going missing in different workspaces, random lagging of menus, sometimes serious slowdown for no real reason,I also couldn't turn my bottom panel back on once I turned it off - had to restart. plus all the same problems that Ubuntu has - high power usage for laptops, all kinds of backlight issues.

I like the MGSE, I think it's a really good idea to get people used to gnome 3.2, but it's still not ready - not their fault, gnome 3 isn't really 'there' yet, but it will be!

I don't want this to become a mint V Ubuntu thing - that happens all to often, their basically the same anyway - you can install MGSE in ubuntu, you can install Unity on mint. There is no way round it, mint 12 is so close to 11.10, that it get most of the bugs also. Mint 11 is much less buggy than 11.04, mint 12 is currently more buggy than 11.10 - it's just all the new tech - new generation kernel, new generation DE, it's just getting used to it, both will move swiftly forward, but I don't use either for a main Desktop workstation at the moment.
