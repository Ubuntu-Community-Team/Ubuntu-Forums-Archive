### Post by wellofsouls on 2008-05-13
**Konqueror** :

Pros :
1) Better standards support (passes Acid2 with stable version, score higher in Acid3)
2) Stricter against non-standards-compliant sites (it may be a disadvantage in the short term, but it's better for the whole web in the long run, so kudos to Konqueror)
3) Loads pages faster, especially with latest KDE4 versions
4) great integration with KDE via kio KParts, making it not just a great web browser, but also a great file manager, file viewer/editor, media viewer, etc.
5) Looks great with KDE
6) consumes little resource (on top of KDE)

Cons :
1) It's part of KDE, so you have to install the core KDE files to use it, which are a LOT.
2) the KDE4 version has some annoying bugs (like the proxy bug)

**Firefox** :

Pros :
1) It's not dependent on KDE.
2) It has thousands of addon extensions of all kinds (but consumes a lot of resource and can cause strange problems if you install too much of them).

Cons :
1) It has worse standards support (doesn't pass Acid2 with stable version, scores less in Acid3)
2) It's less strict on non-standards-compliant sites
3) It loads pages slower
4) It's downright fugly.

Of course **Opera** is better than them both :KS
