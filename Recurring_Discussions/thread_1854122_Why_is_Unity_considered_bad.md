### Post by Copper Bezel on 2011-10-03
Yeah, Kubuntu shouldn't change your login screen unless you tell it to (it'll ask whether you want to use GDM or KDM,) but Xubuntu alters the branding in GDM itself. I don't think that happens if you install the non-branded XFCE, however.

The negativity toward Unity is nothing more or less than what you've already read in all those numerous web pages and threads you mentioned. = ' Long-time users have favorite features that aren't a part of Unity's plan. Ubuntu generally, beyond Unity, has been making a lot of little changes to make things work more smoothly for *most* people, and there are enough of those little changes that a majority of users seem to find themselves irritated or inconvenienced by one or two of them. Add to that the fact that Unity is very unlike other desktop Linux interfaces in that it has a very specific and well-defined workflow that doesn't really allow for any customization, and there's a different worm for everyone in the same apple.

But there are threads enough here on that.
