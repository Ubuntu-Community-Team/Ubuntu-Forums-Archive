### Post by kevin11951 on 2010-12-19
I know most people have gotten over the market share obsession, but I thought people would like to know that Linux has broken the 2% wall...

According to the Wikipedia usage stats (the most generic website I know next to Google), we have a usage share of 2.04%

As of 1 Oct 2010 - 16 Oct 2010:

```

**Operating System	Requests	Percentage**
Windows			3,774,721	84.42%
Mac			332,412		7.43%
Linux			91,155		2.04%

```

And Ubuntu has the most market share of the Linux's:

```

**Operating System	Requests	Percentage**
Ubuntu			33,628		0.75%
Android			20,885		0.47%
SUSE			3,088		0.07%
Fedora			2,695		0.06%
Debian			2,200		0.05%
Gentoo			849		0.02%
Red Hat			608		0.01%
CentOS			513		0.01%

```

Edit: [http://stats.wikimedia.org/wikimedia/squids/SquidReportOperatingSystems.htm](http://stats.wikimedia.org/wikimedia/squids/SquidReportOperatingSystems.htm)

### Post by Dr. C on 2010-12-19
The Linux share is 2.04% this can be broken down further as 

GNU / Linux 1.57%
Android Linux 0.47%

If you wish to eliminate mobile from the equation then GNU / Linux is 1.65%. 1.57%/(1-.047). Where 4.7% is the mobile OS percentage. 

By the way Wikimedia is a very accurate source since the websites in total are generic, and are not biased towards a particular OS. One of the problems with the [Net Applications]("http://marketshare.hitslink.com/") data (0.93% GNU / Linux) is that it does not identify websites that only work with Microsoft Windows or Microsoft Windows and Mac OS X in their analysis, thereby creating a possible bias against GNU / Linux.  On the other hand sites such as [W3Schools]("http://www.w3schools.com/browsers/browsers_os.asp") (5% Linux including Android?) are biased in favor of GNU / Linux since GNU/ Linux is far more popular among web developers and computer programmers than the general population
