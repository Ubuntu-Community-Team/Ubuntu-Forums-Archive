### Post by Ozor Mox on 2008-08-26
> Ubuntu users mainly use apt-get or aptitude to install things where Kubuntu users use adept.

I don't think this is correct. Ubuntu uses Synaptic and Kubuntu uses Adept. Both are front ends (GUIs) to apt, so both Ubuntu and Kubuntu use apt and aptitude. Everything else is right though.

To answer the OP, the only difference is the choice of default desktop environment/window manager and applications. Whether you install Ubuntu, Kubuntu, Xubuntu, Ubuntu Studio etc. all of the other versions, desktops and applications are available in the repositories.
