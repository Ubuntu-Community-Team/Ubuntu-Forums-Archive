### Post by mb_webguy on 2008-08-25
I use it for general computing because it's fast, free, secure, stable, and highly configurable.  I can set up my computer to look and function exactly how I want it, rather than how someone in Redmond thinks it should.  I don't have to worry about viruses or malware (except the stray script exploit in the web browser, which is both easily avoided and easily fixed).  If something *does* happen to go wrong, I can fix it without feeling I'm having to fight the OS to get things done. 

And also because I like to tinker.
