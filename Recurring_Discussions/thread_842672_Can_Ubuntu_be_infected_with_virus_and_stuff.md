### Post by LaRoza on 2008-06-27
Can it? Technically, anything is possible.

Is is remotely possible? No. The worst that could happen is that the **user** wipes the system using root priveleges, which is "hard" to do, but some people manage it.

The good thing about Ubuntu (and Linux in general) is that it is not only secure, it isn't targetted. If something does happen, using a live disk to retrieve your configurations is easy and reinstalling is typically simple to do (but that would be unheard of).
