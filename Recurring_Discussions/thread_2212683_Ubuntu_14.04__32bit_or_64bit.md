Sure, there's lots of advantages to 64-bit over 32-bit.

Advantages: [http://ubuntuforums.org/showthread.php?t=1979923](http://ubuntuforums.org/showthread.php?t=1979923)

jacksonOne advantage is that 64-bit systems support more memory. In theory a 32 bit system can recognize 2**32 (4294967296) bytes or 4096 MB, but a 64-bit system can address 2**64 bytes (about 16 exbibytes). Some argue that speed is another advantage, but that is debatable since most applications are 32-bit and will remain so for some time.
