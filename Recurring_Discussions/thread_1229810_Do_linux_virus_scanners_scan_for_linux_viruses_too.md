### Post by bodhi.zazen on 2009-08-03
The short answer is no. None of that is possible. The threads on desktop launchers are very flawed an misinformed.

You are obviously coming from windows and have good questions and much to learn about Linux.

The only way for those desktop launchers to run is if you yourself install and run them, you install untrusted applications from untrusted sources (social engineering) or if your coputer is already compromised, in which case game is already over and there are a thousand ways or more for damage to occur.

I suggest you read : [Ubuntu Security - Ubuntu Forums]("http://ubuntuforums.org/showthread.php?t=510812")

As well as the staff responses in those desktop threads.
