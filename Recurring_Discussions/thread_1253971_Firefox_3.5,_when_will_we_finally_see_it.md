+1

Shiretoko ***is*** Firefox 3.5 just branded differently in order to lessen confusion between the default firefox and itself.

To get Shiretoko recognized as Firefox in the userstring, just type **about:config** in the address bar and then click the top item (general.useragent.extra.firefox) and set the string with the word Firefox in stead of Shiretoko.  Leave the version number alone.
