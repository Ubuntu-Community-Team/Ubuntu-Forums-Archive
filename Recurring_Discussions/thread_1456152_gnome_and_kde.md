### Post by renkinjutsu on 2010-04-16
They're essentially just a DE (Desktop Environment) for you to do everyday tasks comfortably. A typical linux distro is made up of a "stack" of software
```

Additionally installed modules
DE
X server
shell
kernel
```
Goes something like that, i think.

Anyway, if you don't have an X server installed, you land yourself in a Virtual terminal.. just a black screen with a login prompt that might look something like this, with a blinking cursor ```
Ubuntu Karmic
Kernel 2.6.sldfsldkf
localhost login:_
```

After install X server and xorg or whatever, you can start X which basically allows you to have graphics/GUI/etc. This is a bare bones install and all you will see is a black screen (with maybe a mouse) and you can't really do anything. That's when a DE (*or a wm [window manager] for those people that don't like bloat*) comes in.. it adds a wallpaper so you don't have to look at a black screen that does nothing. It draws icons on the desktop and provides all the programs you need, like file managers (like explorer on windows or Finder on mac). In gnome, this will be Nautilus and in KDE it'll be Dolphin.
