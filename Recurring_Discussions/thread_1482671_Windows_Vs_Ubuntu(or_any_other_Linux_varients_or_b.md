### Post by Paqman on 2010-05-13
> **Spiritof76 said:**
>  Windows wouldn't be Windows without the graphical interface. I suppose it would be DOS


Windows hasn't been built on top of DOS for yonks. Everything from Win2000 onwards, IIRC.

As for the OP, Windows and desktop Linux distros are both of about similar complexity. They face pretty much the same technical challenges, and both tackle them well, although the approach differs in many cases. In some areas Linux is more competent (eg: security, memory management, TCP/IP) and in some Windows has the edge (eg: sound). Which is simpler? Neither, really.

At the end of the day, an OS is a pretty toothless animal without apps, so you have to consider them as well. At that point the question gets so broad and subjective that it's difficult to say anything much definitively.
