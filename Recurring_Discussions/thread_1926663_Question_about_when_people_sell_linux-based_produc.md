Not really any legal issues involved. GNU/Linux is an operating system published, for use, under the terms of the GPL; therefore free to use and with an openly published source code.

Applications can be written to run in the GNU/Linux environment and licensed to users under whatever terms the application creator chooses, GPL, open-source, freeware, payware or whatever. The operating system license has no legal relevance to the application license at all.
