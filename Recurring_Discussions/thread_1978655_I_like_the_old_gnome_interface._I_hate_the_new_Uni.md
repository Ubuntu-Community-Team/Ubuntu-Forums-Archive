### Post by zombifier25 on 2012-05-12
Lucky for you, the classic session is only an apt-get away:
```
sudo apt-get install gnome-panel
```
With this, you can get your favorite desktop back while still using the latest Ubuntu.
P.S. Do some research next time before whining :)

EDIT: Read here to make Classic Session look like that of in the old version: [http://www.omgubuntu.co.uk/2012/03/gnome-classic-in-ubuntu-12-04-its-like-nothing-ever-changed/](http://www.omgubuntu.co.uk/2012/03/gnome-classic-in-ubuntu-12-04-its-like-nothing-ever-changed/)
