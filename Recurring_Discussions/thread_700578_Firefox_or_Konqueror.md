### Post by lian1238 on 2008-02-18
"Konqueror is your file manager, web browser and universal document viewer." is what you see when you run Konqueror, and that's what it is. I like Konqueror more than firefox for browsing.

What I like about Konqueror (other than the fact that it can be used as a file browser):
You can divide the window, horizontally or vertically, then divide the divided ones and so on. So you can browse two (same or different) webpages/folders side by side.
You can press shift+down or up and let the page scroll. In firefox, it's the middle mouse click but stops scrolling on click.

I only use firefox for 2 things: youtube and yahoomail
In youtube, the video control buttons are misplaced and hard to click.
And the "all-new Yahoo! Mail" doesn't support Konqueror.

You'll find what you like more depending on your usage. :]

### Post by doorknob60 on 2008-02-18
If you don't already have it (It comes with Kubuntu but not Ubuntu), run this in a terminal (Applications>Accessories>Terminal):
```
sudo apt-get install konqueror
```
That will install Konqueror and all the KDE stuff that it needs :)
