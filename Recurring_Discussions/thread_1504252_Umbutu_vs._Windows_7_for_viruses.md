### Post by wilee-nilee on 2010-06-07
Most of the nastyware I use this term because there are different types, are set to run in a .exe environment. But some will like rootkits work across all platforms. Norton is a well known AV so the people who write this stuff, to disable it, without you even noticing it. Norton would be the last I would use as well of any of the really well known ones that are included as free trial when you buy a MS setup, they have big red bulls-eyes on them.

The best thing a MS user can do is use only a limited account for daily activities, not get caught by social engineering ie popups and others like email attachments...etc, and use a browser with a flashblock like noscript in FF. There are other addons in FF like wot, ghostery, adblockplus,and better privacy that keep the popups and information gathered minimized, and provide site information. 

I use avast, malwarebytes, and superantispyware on my MS setup and have never had a problem. But I only use the MS setup really to understand it to help others, I use open source most of the time.

This nastyware can be picked up by open source but will have no negative impact, but can be passed on to a MS setup with email and other ways. Really the limited account is suggested to keep about 90% of this stuff out, but you don' want to rely on this as any truism. Anytime there is root access or admin access your in danger of getting something which may gather data or key log or just destroy the setup. This is why open source generally is set to need a password for root access and even here there are limitations as to what you can do.

This only covers some things there are people on the forum who are knowledgeable about this stuff and hopefully you will get answers from them. I would in the end say never use Internet explorer ever, except to install a better browser because of the addon capability. IE is used to get updates, but should be set to save no cookies or any information, and a cleaner like ccleaner should be used to get out what you can. There is stuff though that can only be removed from the safe mode as well. Even when a MS setup says everything is removed it isn't.
