### Post by snowpine on 2011-02-12
SliTaz is my favorite ultra-lightweight distro. Puppy and TinyCore are also very popular. If you choose a lightweight distro, it won't come with a full suite of applications in the default install. Therefore if you want, for example, a spreadsheet application, you can simply install Gnumeric or OpenOffice Calc. I am 100% certain you can install either of these apps in SliTaz. :)

I guarantee you that Ubuntu will never compete with the 30mb SliTaz in the very small niche market of ultra-light operating systems for very old hardware. The absolute lightest install of Ubuntu (a CLI-only minimal install with no GUI or applications) has higher system requirements than a full GUI install of SliTaz. Ubuntu developers would need to remove over 90% of Ubuntu's features to make it fit in 30mb; not going to happen. :(

The great thing about Linux is that there are 100s of distros for all needs. Ubuntu cannot be all things to all people. They choose to forge ahead as a user-friendly distro for modern hardware. Certainly this approach seems to be working as Ubuntu is the #1 Linux distro in the world, while SliTaz is down at #46 according to distrowatch.com.

I will also add that software can not solve hardware problems. In some cases, the best use of one's time is to simply upgrade the hardware, whether this means upgrading one's RAM and hard drive, or retiring the old computer and purchasing something new. Where I live (New York) I regularly browse Craigslist and see Pentium 4's with 512mb of ram for under $100. Such a setup is easily capable of running a full-featured distro like Ubuntu (and would FLY! with something like SliTaz or Puppy).
