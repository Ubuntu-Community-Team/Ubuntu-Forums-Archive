### Post by fd9_ on 2008-07-24
Here are 12 things that I think would make Ubuntu more user friendly *out of the box* (that's the key phrase here):

1) Installing any program that's not in the repositories should prompt me to add the software source,  along with a English explanation of what exactly that means. Usability always comes first, security second (similar to the idea of opensuse 1-click install)

2) List of recently installed apps at my finger tips, from the gnome menu

3) List of commonly used apps at my finger tips, from the gnome menu

4) Better search bar that immediately appears on my gnome panel, finds results based not just on file name, but application name (ie, combine "tracker" and "run application" into one)

5) Prompt me if I want to put a desktop shortcut whenever I install a new program

6) I just installed an application. Where did it go? Tell me.

7) Moving files to root-access locations should not need the command line, prompt me for my password in a GUI. Same with opening root-access files.

8 ) Installing scripts (.sh files) should not need the command line

9) screenlets, compiz-manager, and a dock should come pre-installed and integrated better (really)

10) Synaptic should refresh repository list automatically when it needs to, don't ask me to do it

11) Rename commonly used, pre-installed programs so that they're easily recognizable (ie, Transmission should be Transmission Bittorrent Client)  (idea from [http://contentconsumer.com/2008/04/27/is-ubuntu-useable-enough-for-my-girlfriend/](http://contentconsumer.com/2008/04/27/is-ubuntu-useable-enough-for-my-girlfriend/) )

12) And finally, last but not least, when I first install Ubuntu, give me a nice "Hello new ubuntu user" dialog that tell me what apps are pre-installed, what they are for, and where I can find them so I don't have to go searching for them myself.

Again, this stuff should come default "out of the box" from a fresh installation.

Your thoughts?
