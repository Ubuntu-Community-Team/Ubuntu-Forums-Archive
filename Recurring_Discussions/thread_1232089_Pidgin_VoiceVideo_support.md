### Post by Johnsie on 2009-08-05
I've been waiting on voice/video support on Gaim/Pidgin since 2005. I wouldn't hold my breath. This is one of my number one annoyances with Ubuntu. The IM 'equivalents' lack some of the major features that make instant messaging fun. 

Skype is probably the best choice if you want A/V support on Linux but Skype for Linux is not as good as the Windows client. AMSN does video over MSN but is very buggy. Apparently GYachi works with Yahoo, but last time I tried that it wasn't compatible with the Yahoo chatrooms due to not supporting the captchas.

If you're mainly using your computer to video conference with friends then you're probably better off using Windows for now as that's where the quality IM's seem to be. The problem is that the major networks are using propriety protocols, so any software developed for Linux by third parties will struggle to be compatible with the multimedia features of the networks.
