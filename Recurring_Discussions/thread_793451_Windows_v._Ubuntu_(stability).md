### Post by roderick on 2008-05-13
You thought Linux was for gaming... sorry, but you were misinformed.

It was a server first and foremost and a desktop only recently. 

Gaming is way out there. The only real Gaming for it is the Free OSS games and a handful of native ports.

Wine is just now in it's first release candidate despite being around for 15 years, and can run many Windows programs with caveats - see [http://appdb/winehq.org](http://appdb/winehq.org) for details. It is not 100% rock solid. MANY programs will not run now and maybe never.

Linux is a valid Desktop and Server, but gaming needs some vendor support (in making native ports) before you can call it a Gaming system.

As for obfuscated commands... jump down off the pedestal. If you never used windowsbefore, you'd never know those old DOS commands either. As with every new system/hardware/electronics, there are commands/switches/buttons to learn. Invest in a dummies guide (just like you would for any unfamiliar subject like photography, taxidermy, etc) and learn the new system.

Good luck.
