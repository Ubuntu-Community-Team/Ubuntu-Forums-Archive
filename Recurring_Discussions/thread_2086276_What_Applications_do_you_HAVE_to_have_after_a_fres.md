### Post by Aaron Christianson on 2012-11-21
On Arch:

can't live without:
sudo - safer and more convenient than su
alsa-utils - need sound control
xorg - handy to have a graphical environement
firefox - moar internetz
vim - no explanation required. No computer is usable without it.
herbstluftwm - manual tiling WM, 100% configured and controlled from shell commands.
laptop-mode-tools - vastly improves battery life.
acpi - got to be able to get info about the battery
pm-utils - suspend, hibernate, etc.
zsh - more powerful shell than bash
mplayer - plays media from the command line. even been using it as my music player lately.
dmenu - ultra lightweight application launcher
wicd - connect to wifi with minimal fuss.
rxvt-unicode - terminal of choice

Productivity:
asciidoc - A markdown language that converts into several major document formats. need it + vim to do my work
libreoffice-writer+jre7-openjdk - don't need the whole suit, just this. LaTeX isn't quite where I need it for my field of study. Libreoffice is a bit clunckier, but it provides all the functionality that I need.
bibletime+xiphos - hard to be a biblical studies student without bible software

almost always have, but sometimes takes a while to get around to installing:
google-chrome - better for flash. worse for doing anything useful.
ffmpeg - sometimes I like to encode media
tmux - tile the console + other nice things
w3m - console web-browser with framebuffer support for images
yaort - a nice wrapper for pacman + AUR
gimp - it's gimp
scrot - take screenshots from the command line
alsi - display system info with ascii art
htop - like top, but nicer
vifm - console file manager with vim keybindings
transmission-cli - torrent client

On Ubuntu:

can't live without:
vim
laptop-mode tools

productivity:
asciidoc
bibletime+xiphos

Anything else is optional or pre-installed. I often remove as many things from Ubuntu as I install.
