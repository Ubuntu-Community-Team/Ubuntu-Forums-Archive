### Post by lovinglinux on 2010-07-13
I have been using 64bit since I upgraded my CPU, when Karmic was released. Lucid works fine, but not as good as when I bought the CPU.

Recently I became disappointed with the lack of Tracemonkey support on Firefox 3.x 64bit (I could swear it was enabled by default) and with the fact that Adobe dropped the support for 64bit flash. Both applications work fine tho.

Anyway, yesterday I decided to re-install Lucid to test how the 32bit would behave on my system. I don't know if it is a coincidence, but my system looks much snappier than the 64bit.

Kwin cube works without any performance degradation, even when running Windows 7 on a VM. I even have Aero on Windows and the RAM being used was cut by half.  The system doesn't choke when saving a VM and Firefox is scoring almost the same as Swiftfox on Peacekeeper benchmark. Chrome and Opera scored a little bit less tho.

There are a lot of complains in the forums about the little warning next to the 64bit version download on Ubuntu site. I must say that I understand now. Perhaps is my machine or my previous Ubuntu installation that was defective, but from what I have seen so far, I'm not going back to 64bit any time soon. 

I would like to know what are your feelings about Ubuntu 64bit and what architecture you are using?
