### Post by aysiu on 2011-08-07
I use Mac OS X, Ubuntu, and Windows 7 all three regularly.

As far as I can tell, the latest versions of all OSes are quite similar in terms of requiring authentication. Windows does *not* require a password for authentication by default (just additional clicks) but it can be easily configured to require a password.

All three platforms consider system time to be an administrative change.

In both Mac OS X and Ubuntu, you can check for system updates without additional authentication, but you do need to authenticate to actually install the updates.

I agree with the posters earlier who said it's far more important who's using the system than the system itself. If a user isn't installing system updates, being careful about what sites she visits and software she installs, and isn't aware of what phishing is, then her computer is far more vulnerable (regardless of operating system) than someone who is the opposite in all those regards.

Considering how many people there are who don't know even the basics of computer security, it makes about as much sense to argue about operating system security as it does to argue about who has the strongest door lock if half the people with houses open the lock for anyone who knocks on the door.
