### Post by NightwishFan on 2010-08-30
I would go with 64-bit with anything over 1gb. Under that you might notice more performance just because of the few mb of RAM you save using 32-bit. Also, with that low of RAM you will use swap heavily, so ensure you have a swap file. Xubuntu probably would run better, but not by much, so if you are used to Gnome you can go with that. I recommend either.

I would do this tweak to improve performance. Though most think that using SWAP will slow things down, it will help a RAM starved machine a great deal. This will make the kernel prefer to swap earlier and often.
```
sudo su
echo 'vm.swappiness=100' >> /etc/sysctl.conf
exit
```
