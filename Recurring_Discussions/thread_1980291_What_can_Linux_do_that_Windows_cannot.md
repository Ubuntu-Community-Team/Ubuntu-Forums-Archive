### Post by BlacqWolf on 2012-05-15
Linux can do anything. It can play games, or be productive. The only real division between what you can do in Linux or Windows is the lack of commercial apps or games.

Linux is easily customizable (don't like a theme or desktop environment? You can change it.)

You can do games or be productive - there are many productivity apps or games (though relatively few games) available from the repositories or other sources. 

There's simplified app management and research - you can search trusted repositories for apps and this way be sure of their integrity, then search the web for other source if you can't find what you need. (more risky and harder, as it is in Windows, but widens your search.)

There's also many ways of doing things, to fit those of different preferences. You can use the terminal, or you can easily do common tasks like installing a program from a graphical package manager with similar functionality as a command-line app.

If there's something you want that isn't available in Linux, there's always running a virtual machine or dual-booting. Ubuntu and other distros give the option of dual-booting during install, with minimal user interaction of this step required. (and a tool for importing Windows files and settings, if you have a Windows installation.) You can also use the simplified WUBI installer, for easily installing/removing Ubuntu like any other Windows application, with dual-boot-like functionality.

### Post by Simian Man on 2012-05-15
The concept of an OS "doing things others can't" doesn't really make a whole lot of sense.  From an end-user perspective, any OS just runs programs written for it.  That's all it does.  So Windows can run Windows programs and Linux can run Linux programs.  The fact that there are different programs for one system or another doesn't really reflect on the quality of that system.

From a technical point of view, different operating systems have different features and designs, but any modern operating system has sufficient features for the vast majority of computer users.

### Post by LillyDragon on 2012-05-15
Second that lack of BSOD! w00t! I may have only had two of those my whole life, but still, it's always jarring when it happens, especially when I see it on other people's laptops. xD

Also, some of my reasons for loving Linux!

1. Smarter MMU, translating into a smaller footprint on RAM usage and almost no writes to swap space unless absolutely necessary. Seems better on my hard-disk's life, IMO.

2. Your choice of Window Managers, so if you don't like your OS's official artwork and UI direction, use another WM! (I'm a Xubuntu use right now.) Wish it was that easy on Windows 7, I'm no fan of that start menu bar or the window borders, and don't feel like forking out for Window Blinds just to change that.

3. The threat of viruses is still there, so commonsense is still needed while browsing the Internet, but at least I don't have worry about spyware, malware, worms, trojans, and adware like I do on Windows; I can actually breath while doing a typical on Google. Even the threat of viruses on Linux is pretty slim, given it's relatively small market-share, but that's no license to be stupid! Overall, my computing experience is far less riddled with fear and gray hairs, I can just focus on using my system for crying out loud.

4. If a program locks up, it doesn't slow down the rest of the system with it, and if the WM crashes, it returns to the terminal and automatically restarts. Windows still can't do that. ;_;

5. Comes packed with Firefox, or at least a variant of it. Why doesn't Windows give me the choice between Internet Explorer or Firefox from the start? It's so annoying having to use IE for even that one time to download FF!

6. A universal command-line that can change anything you want about your system from one place. On Windows I'm stuck sifting through tick boxes and options just to find what I'm looking for no matter what, it's DOS terminal feels useless outside the use of command-line-based programs.

7. I don't need a third-party program to have virtual desktops, transparent windows or docks.

8. My little cousin can't "break it" when he uses my system. :P

9. I can have the safety net of a limited user account while still also possessing super user privileges at my password prompt. On Windows, I'm stuck switching users just to install *one* program before logging back onto my normal account. With Linux, even basic system administration is less flow-breaking.

10. Already comes packed with most of the drivers my system needs. This is also true for Windows, but it's usually missing the video, audio and Ethernet drivers obviously. Most Linux distros will go the extra mile to include those too, so all I have left to do is install my printer and scanner drivers.

11. Better support for all of my controllers, especially the console ones running through USB or printer-port adapters, since the drivers they need to interface with the OS is already included in the Linux kernel. I'm stuck looking for third-party drivers on Windows, usually, and it really sucks if the drivers work through direct-input. *shudders at poor compatibility with favorite games*

12. Some WMs are easy to customize and skin to your liking, I love how easy it is to skin my panels on Xfce, and the option to use GTK2 themes!

Now that's just outside of all my software needs, which is debatable on any OS. But I already have Linux equivalents to all my favorite programs if they don't run through WINE, or are already multi-platform, like Firefox and GIMP. That's not even all the reasons why I love Linux, but this one post is long enough already. xD
