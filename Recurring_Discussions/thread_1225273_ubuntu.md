### Post by starcannon on 2009-07-28
> **necromonger said:**
> is ubuntu 64 bit or any 64 bit os ready for prime time?
 
I've run the 64bit version of 8.04 for over 3 months, it was a bit more difficult, but doable. I think that its getting very close to being ready for almost everyone. Stick it on another partition or spare hdd and try it for awhile; I keep a separate /home so it was just an install away to go back to 32bit for me. I only run 2gb of ram, so for me I noticed no real meaningful performance improvements. Like others have said, your mileage may vary, some get along famously, some have a hell of a time making it work, and others, like me, are indifferent. 
 
Ready for primetime is at times a very loaded question. It depends on who is using it, and what the expectation is. I would bet that if an experienced 64bit Ubuntu installer had set things up for me, and just handed me the computer ready to go, I'd have loved it.
