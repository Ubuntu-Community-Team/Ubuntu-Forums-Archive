### Post by yetiman64 on 2011-12-17
I took a serious dislike to Unity in 11.04, but with Oneiric I turned that around considerably and now find it very usable/productive (in as much as my personal usage of the system can be called "productive").

I must say I had to "tame" a few design features first though,

1. Disabled the global menu (I detest the idea of using it on a desktop with a larger monitor).
2. Disabled the overlay scrollbars (I don't like them, but admit they function much better in 11.10 then when I used them in 11.04)
3. Allowed "all" programs to show in the Notification area (suits my usage better)
4. Sent the "Sidebar" (Launcher) to the bottom of the screen (once again much more "usable" on a widescreen) using the "Ubuntu Unity Plugin Rotated" plugin installed with the launchpad ppa for it.

There were still a few complaints from me, leftovers from my days using Gnome2, which I have managed to sidestep by installing and using AWN dock. I regularly switch back to the "old school" menus with it, though I'm slowly adapting to using Dash and search lenses etc. AWN also has a simple launcher where custom launchers can be made available to the user (Note the way I have AWN setup chews up about 130 MB RAM - not light at all). It also displays all the necessary hardware indicators I need to monitor that previously I could set up with applets in the gnome2 panels but can't in Unity. Having AWN set up allows for much easier customization of compiz settings particularly when the Unity plugin seizes up most of the display (I use the cube and expo wall extensively).

I do like to be able to control a system from the desk with just the mouse as much as possible (just my preference) and as such find the design of Unity restrictive, my final tweak was to install easystroke mouse gesture recognition software. Nearly every keystroke/key combination or command needed can be set up in it (not the keys for the tty terminals though, found that out the hard way ](*,)). Currently I have 147 gestures programmed in with many many more to go.

Oneiric + Unity + My Tweaks, working nicely now for me. Though the introduction of Unity initially prodded me into looking into other distros for a while, the urgency to change over no longer exists for me. Yes there are bugs (no new release is "stable" imo) but the situation with Oneiric seems to be a vast improvement over Natty.
