### Post by WA_Garrett on 2009-03-29
[http://en.wikipedia.org/wiki/List_of_Linux_computer_viruses]("http://en.wikipedia.org/wiki/List_of_Linux_computer_viruses")

You don't need anti-virus protection on Linux like you do with a windows box. I don't have an antivirus on my Linux partition because I really don't need it. You are more susceptible to social engineering than you are to Windows like virus attacks on Linux. Common sense is all you need to protect your self against malware on Linux.

### Post by kestrel1 on 2009-03-29
As above, no real need for AV prog under Linux, however if you have access to a Windows partition & you swap files between each you could check any you transfer from Linux via the web or email to Windows. You can try clamAV for this:```
sudo apt-get install clamav
```
