### Post by Cabs21 on 2010-01-18
It is usually a matter of preference. Yes Ubuntu and Debian are similar since Ubuntu was created from Debian but it is not exactly the same.  There are many people who think Debian is a better distro then Ubuntu and vise versa.  This is the same with Mint and Ubuntu.  They are similar but different in many ways.  It all comes down to what you like and prefer.  Ubuntu is a great distro to get your feet wet in the Linux pool.  The beauty of all this is that you can try them for free and with no cost to your system (Live CD).  So if you are curious about using Mint then make a Live CD and give it a try.
