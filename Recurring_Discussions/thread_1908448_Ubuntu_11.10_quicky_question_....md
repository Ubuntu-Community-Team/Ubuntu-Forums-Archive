[QUOTE=CDSH]I would like to stick with the one that is on 10.04 natively - if possible. At least until I can get used to something else.[/QUOTE]
Well, to be clear, you're using it; it's just different in this version. There are tweaks like the one BC59 suggested to make it look more like the older one. 

To clarify, the old environment was the Gnome 2 environment. Ubuntu started using the new, custom Unity interface as a part of the Gnome environment, and Gnome changed dramatically in the upgrade to Gnome 3. If you install Gnome Shell, you can see the current version of Gnome's own default interface, but it's not any closer to Gnome 2 than Unity is.
