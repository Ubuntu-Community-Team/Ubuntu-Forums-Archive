### Post by jerome1232 on 2008-11-30
The difference is more in 'look and feel' than anything else. You can install Gnome based apps in KDE and KDE apps in Gnome. (The problem is you might have to download a few hundred  megabytes of KDE libraries to run the one megabyte KDE program you want in Gnome.)

Since you want to figure out the difference you can actually have both installed at the same time and choose which one you want to load when you log in.

To install Kubuntu on a Gnome install:
```
sudo aptitude install kubuntu-desktop
```

Now you can choose which one you want by pressing f10 before you log in and selecting the session you want.

Both use the same base, but they have two different desktop environments. Ubuntu uses Gnome and Kubuntu uses KDE. KDE 3.5 is considered stable, whereas KDE 4 is still a "work in progress".

### Post by JohnFH on 2008-11-30
The above links should help to answer your question, but I'll give you my opinion since I use Kubuntu all the time.  I switched to Ubuntu from Windows about a year ago and quickly moved to Kubuntu because it allows the desktop to be customized a lot more.  Generally speaking, the KDE desktop as well as KDE applications have a lot more options that you can change to configure things.  Also Kubuntu does look better than Ubuntu (I think so anyway) and I love having all the control to adjust various things.  However, that's also a downside because it can be a bit confusing at times to remember where some settings are.  Also Ubuntu is more popular than Kubuntu (probably just because it's the default release) and quite often I've seen forum answers that don't help me because I use KDE instead of Gnome.   So then if you are deciding between the two I would recommend sticking with Ubuntu and Gnome for now even though I use KDE and Kubuntu myself and love it.  Sticking with Ubuntu and Gnome will make the learning process a lot easier at the start.
