### Post by khelben1979 on 2010-06-26
I would suggest that you just choose the file catalog / and then click on OK for the [ClamTK anti-virus]("http://en.wikipedia.org/wiki/Clamtk#Graphical_interfaces") to start searching and I'm sure it's gonna take a while. :)

Also, make sure to tick into preferences that viruses should be placed in quarantine, otherwise it won't do it as default.

There is no need to use another anti-virus solution on a Ubuntu Linux system in my opinion, just learn how to use it and you will find that it will be worth your time using it.
