> **parat26 said:**
> ):PHi guys,
 
Windows is a part of my life, because i using windows for 10-11 years.This summer i decided to take a look at another operating system. I decided to take a look at Linux(Ubuntu), at my second laptop. 
Always good not to experiment on your luxury horse but on a work horse.
 
> 
At the summer i have many problems with Ubuntu, for example it keeps freezing 3-4 times per **day**. 

In most cases this is due to hardware issues, not due to switching from one OS to another OS. And if this happens in the summer, the temperature migth get to high which causes to freeze your system (overheating cpu???)
 
> 
Also i can't run almost nothing with Wine, and i am really disappointed because there is actually a few games at Linux.
Linux is good for gaming. I've tried various games for Linux and never had any bad experience with Linux (or Ubuntu) with gaming. Regardless what sandrd17 says, Linux has made good progress about gaming.
 
Wine is a tool which creates a Windows environment on your Linux system. There are a lot of programs which runs good enough with Wine. It is not a holy blessing in replacing Windows. If you want to use Windows programs, you migth consider installing VirtualBox and install Windows as a virtual machine. However, there are very good Linux replacements for Windows programs.
 
> 
 I was wondering whats the best idea. Returning to Windows or keeping the Linux?
That is totally up to you! If you think you can do more with Windows, stay with Windows. If you think that Linux might be more of a help for you, try various distro's such as Fedora, Linux Mint, OpenSuse, Debian and many more (see [http://distrowatch.com/](http://distrowatch.com/)). 
 
 
Now your question about a Task Manager. Ubuntu (which i use) has a sort of task manager called System Monitor (System > Administration > System Monitor). But when you "my computer freezes....", do you mean only a program or your whole system? If it is your whole system which freezes, then a taskmanager won't help you either! Only a reboot will help!
