WTFPL means that anybody can do absolutely anything they want with it, and they don't have to give you any credit. But this is not all bad - read on.

Let's assume that there are two similar products using your WTFPL code:
[LIST]
[*]Both products will be better because of your code
[*]Your code will not cause either product's price to rise (due to competition between the two)
[*]You may, or may not be credited for each product
[*]Each project may, or may not contribute to your code
[*]People using either product will be happy because the product is better, and they didn't have to pay a penny
[/LIST]

If you licensed your code under the GPL instead, the above mentioned products may not want to include GPL code because:
[LIST]
[*]It forces them to release any alterations to the code under the GPL and
[*]The GPLv3 license forbids inclusion of GPL code in commercial applications - this can obviously make things difficult for companies wanting to use it
[/LIST]

So basically what it boils down to is: Are you willing to force companies to release any alterations to the code under the same license at the risk of them not using the code at all? Remember: If they don't use your code, their product will not be as good, and their users will not be as happy.

### Post by schauerlich on 2009-10-12
> **Penguin Guy said:**
> Er, would you mind telling me what exactly the point *is* then?

I don't mean to speak for Bachstelze or Theo, but I believe their point was thus:

The stated point of the GPL is to guarantee users and developers freedom to access the source code of the programs that they use. The BSD license allows others to modify their code and redistribute it under a different license. If one were to take code under a BSD license and redistribute it as GPL, the original maintainers of the BSD code no longer have the ability to incorporate those changes back into their own project, which is BSD licensed code. This has the same effect as if the code had been forked and made proprietary, which is what the GPL is supposed to protect against.
