### Post by Ctrl-Alt-F1 on 2010-10-02
> **ki4jgt said:**
> So what's on the other ttys? Or are there any?

They're just vertual terminals that you can use.  It's the same as using gnome-terminal except it doesn't require a desktop environment :)  It can be useful for troubleshooting things.  In linux you could be logged into all of them at the same time as the same (or different) users.

Some distributions use them for stuff.  Almost always the graphical user interface is on 7, and often (like in the case of OpenSuse) number 1 is used for system stuff.
