### Post by OffAxis on 2008-05-06
Here's a recent article comparing linux desktops:
[http://www.informationweek.com/news/software/linux/showArticle.jhtml?articleID=207200145](http://www.informationweek.com/news/software/linux/showArticle.jhtml?articleID=207200145)

> can you guys give me a name of a ubuntu that comes with byrel/compiz and kde4 inside ?

as perlluver pointed out, kubuntu-remix natively has the kde4 desktop, but you can always download the package and install it beside your ubuntu (gnome) or kubuntu(kde 3.5) desktop.

```
sudo aptitude install kde4
```

or, the kubuntu desktop version of kde4.
```
sudo aptitude install kubuntu-kde4-desktop
```

Here's a listing of the contents:

[http://packages.ubuntu.com/hardy/misc/kde4](http://packages.ubuntu.com/hardy/misc/kde4)
