### Post by linuxyogi on 2013-09-25
Hi,
I started using Ubuntu from 8.04. Then I installed 9.04, 9.10, 10.04, Xubuntu 12.04. 
After that I moved to other distros not because didn't like Ubuntu but because I wanted to try other distros. Some of the distros I tried includes openSUSE, Fedora, PCLINUXOS, Manjaro, PCBSD .... etc.

_Likes_

Easy installation of proprietary graphics driver.

Option for configuring DSL and mobile broadband via network manager.

Huge repository.

_Dislikes_

_**Extremely buggy**_

After trying lots of distros I am tired of distro hopping now. So I decided to come back to Ubuntu. 

Visual effects is something that I am not very obsessed with so I decided to install Lubuntu 13.04.

The installation process was as usual but the OS is I am sorry to say extremely buggy. I was getting like 

5 crashes every hour. I have never used Unity so I have no knowledge about it but Ubuntu itself has become 

very unstable. 

The other distros that I have used are not bug free but still not as buggy as 13.04. I am writing this only because I want Ubuntu to get better.

The ideal distro for me is the one which is rolling & stable at the same time. (LMDE sometime breaks after installing updates).

_Improvements_

**Primary : Please make it stable.** 

The proprietary graphics drivers are included in distros like Manjaro & PCLINUXOS. It will be awesome if Ubuntu can do the same in future releases.  

I read somewhere that Ubuntu is going to become a rolling release. If stability can be maintained despite being a rolling release then rolling is the best way to go.

What are your likes and dislikes ? 

What are the improvements/features you will like to see in Ubuntu ?
