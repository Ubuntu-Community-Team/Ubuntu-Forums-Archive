### Post by TBABill on 2012-01-23
Yes/No/Don't Know/Yes/Yes
 
In order, those are the answers as best I know. If you have specific application requirements that cannot be substituded for with a Linux replacement, you may find yourself disappointed. You can try to use Wine to help run some Windows apps, but your mileage will vary. 
 
Best to keep in mind that Linux is not Windows, nor does it try to be Windows. It is a completely separate, independent operating system with its own software available (literally tens of thousands) to do most anything you can imagine. However, if a specific Windows program is required (like iTunes) you will have to find a workaround or stick to Windows when necessary. 
 
You can also dual boot with Linux, meaning you can boot into either Windows or Linux as you choose. Or you can run one in a virtual window.
