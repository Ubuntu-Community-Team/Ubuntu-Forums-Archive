### Post by Dragonbite on 2009-12-08
I'm mixed regarding the KDE and Gnome variants

For KDE:[LIST][*]I like Quanta for web development. Pretty quick to open (unline Eclipse), provides GUI view (not quite WYSIWYG, but closer.. plus split screen is just sweet) and the publish function to move everything when done.[*]DigiKam edges out F-Spot on some features but I can live with F-Spot if I have to[*]Auto-changing wallpapers, because I like a number of them and get bored easily[*]Okular makes working with PDFs easy.  For example, I scanned in some chinese characters written down from my work's scanner. It scans it into a PDF.  To test it out, in Okular it looked like I could select portions of it and copy-and-paste it into a document.  In Ubuntu (Gnome) I had to pull it into Gimp instead.  But Gimp doesn't have the same search feature.[*]Glassy (I'm a sucker for shiny sometimes)[*]My HP Scanner has only worked in KDE, not with Gnome's Xsane (it drives me "INsane!").   Ihaven't tried with 2009 versions of Ubuntu yet to see if that has been resolved yet though.[*]KDE looks better overall, very smooth and more uniform (with QT apps)[/LIST]

For Gnome:[LIST][*]Menu layout makes it easier to get around than the slab-menu of KDE.  The slab isn't too bad when you get used to it, but the menu bar of Gnome provides one less level to have to go through[*]Cheese!  I know, but it is difficult to find a webcam program that fits in KDE other than Kopete, which has never worked with my webcam though Cheese has for a while now[*]Emapthy/Pidgin are better Instant Messengers than Kopete usually[*]I haven't had my Sound crash in Gnome while in KDE I get that error message about it having to fall back to the default (Pulise Audio).. happens with Intel integrated sound cards but doesn't seem to have a problem with Gnome (sound still works in KDE, but the alert is annoying)[*]Music players play MP3s.. I have yet to get Amarok to play them (let alone play MP3s using the free Fluendo Codec I have).[*]Evolution is a usable PIM program, Kontact is alright but I have to do more tweaking to get it the way I like[*]Monodevelop, because I work in .NET at work I like the idea of being able to convert that experience to a Linux-friendly platform (if I ever get around to it).[*]there are more GTK apps for Linux than Qt[*]I have had problems with getting wirless working in KDE while not so bad in Gnome.  The latest is running Fedora KDE and since it detected the wireless card out-of-the-box this is a moot point if Kubuntu and Ubuntu would use the same Braodcom open sourced drivers.[/LIST]

That's just my thoughts between them. Of course apps from one side of the "fence" can run on the other side but they never quite look like they fit in.
