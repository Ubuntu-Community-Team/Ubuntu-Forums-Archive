### Post by Meep3D on 2010-03-22
It's easy, you just need an actual business model - like Red Hat, MySQL, Mozilla or one of the many successful open source efforts.

There is a belief that if you somehow open-source your code you'll suddenly become more successful, which is simply untrue.  Making something and giving it away will only work if you are actually able to monetize some other aspect of your product.

Some things are, unfortunately, harder to monetize under a FOSS model than in a commercial environment - professional grade tools (graphics, video and sound editors), much user-centric software and most games.

Some software can only really be developed (profitably) under a closed, commercial, model though.
