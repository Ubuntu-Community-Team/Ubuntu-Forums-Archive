### Post by 3rdalbum on 2012-12-30
I don't like Flash more than anyone else, but there seems to be this misconception of what Flash is.

A lot of people think Flash is just way of putting streaming video on the Internet, a'la Youtube. The same people think that, now that HTML 5 includes a <video> tag, there's no more use for Flash.

That's so wrong.

Flash is really a full multimedia authoring platform. Sure, you can use it for streaming video, but you can also write full multimedia productions, games, interactive database-driven websites, and animations with it. For the entire scope of what Flash can do, HTML 5 is no replacement. In fact, there's nothing available on the internet right now that can replicate Flash's capabilities.

Correction: Maybe Java. But with much, MUCH more work by developers. Plus, Java is even worse for security holes than Flash. Load speed and stability on Java is horrible, whereas Flash is quick to load and decently stable.

If you really want Flash to "ride into the sunset" then you'd better wait for HTML 6 to be drafted, and hope it contains four times as much functionality as HTML 5. Knowing the W3C, we'll probably have a workable draft of HTML 6 some time in 2015...
