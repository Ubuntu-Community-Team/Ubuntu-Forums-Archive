[FONT=Times New Roman][SIZE=3] [/SIZE][/FONT]
[FONT=Times New Roman][SIZE=3]2.)[/SIZE]    [SIZE=3]Make an easy search feature like that on the Windows Vista/7 start menu, as linux will have much more unfamiliar software available.[/SIZE][/FONT]
[FONT=Times New Roman][SIZE=3] [/SIZE][/FONT]

[FONT=Times New Roman][SIZE=3] [/SIZE][/FONT]
[FONT=Times New Roman][SIZE=3]4.)[/SIZE]    [SIZE=3]Make a graphical overhaul – no more ugly brown and orange. It has it’s retrospective fans – but in comparison to Windows and OS X, our system appears dull and without gloss.[/SIZE][/FONT]
[FONT=Times New Roman][SIZE=3] [/SIZE][/FONT]
[FONT=Times New Roman][SIZE=3]5.)[/SIZE]    [SIZE=3]Emphasize the vastly superior amount of choice, and create a system menu that makes installing programs easier, and allows for the discovery of new and exciting software the user usually would have to pay top dollar for in windows.[/SIZE][/FONT]
[FONT=Times New Roman][SIZE=3] [/SIZE][/FONT]
[FONT=Times New Roman][SIZE=3]6.)[/SIZE]    [SIZE=3]Include a basic antivirus for free; sure, it may be redundant (in most people’s opinion, not mine), but safety is a concern in most Windows user’s minds, and they won’t know the difference between Windows and Unix executables.[/SIZE][/FONT]

This also means finding benefactors who will develop drivers so that *nix isn’t behind in drivers and can compete.[/SIZE][/FONT]
[FONT=Times New Roman][SIZE=3] [/SIZE][/FONT]
[FONT=Times New Roman][SIZE=3]8.)[/SIZE]    [SIZE=3]Sell something. It doesn’t have to be an OS – it can be PCs, merchandising, whatever. We need to be able to fund lots of things in the community like codecs to include with each OS.[/SIZE][/FONT]
