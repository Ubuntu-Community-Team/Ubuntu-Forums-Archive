### Post by wmcbrine on 2009-10-20
There's no such thing as "too easy". You ought to learn the broad concepts first before you get bogged down in petty details, and Python is perfect for that. It's also powerful enough that, yes, you may not find a need for other languages, though of course you *should* learn some. But you might eventually be motivated to learn C (or whatever) if you want to write something really fast -- that's not Python's strong suit.

For me, Python is actually the *end* (for now) of a long journey through many languages, and I don't think I'd fully appreciate it if it were my first and only language. But I feel sorry for anyone starting from scratch with C, even though it's a beautiful language in its own way.
