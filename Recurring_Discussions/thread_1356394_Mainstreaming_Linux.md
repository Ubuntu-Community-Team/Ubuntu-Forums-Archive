### Post by cariboo on 2009-12-19
Most problems are fairly easy to solve, they have been answered ten times over here in the forums. The biggest problem is that you have to learn how to use Ubuntu, just like you learned how to use Windows.

I have seen many people trying to solve a problem from a Windows point of view, there is a fellow a couple of threads up complaining there is no task manager. There is something similar called System Monitor. Most of the tools people are used to are available in Ubuntu, they just have different names.

So far the only new hardware I have had a problem with is a usb wireless device, that said Windows only on the box. I can get it to work on 32-bit versions using ndiswrapper, but the included 64-bit Windows drivers just don't work with 64-bit Ubuntu. I don't have any 64-bit Windows versions installed as the main os, so I haven't been able to check how it works with Windows, performance on 32-bit Windows systems is OK, but nothing to rave about.

The main point of this is, take time to learn the system, just because you've used Windows for several years, doesn't mean that the knowledge will transfer to different operating systems. My advice is to use a dual boot system for at least 6 months, or until you feel really comfortable using Ubuntu, by then if you have used Ubuntu almost exclusively, most problems will seem pretty trivial.
