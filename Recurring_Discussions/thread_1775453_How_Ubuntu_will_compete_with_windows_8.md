### Post by Copper Bezel on 2011-06-05
I see the concern in that Ubuntu is going to look "dated" to many Windows users. After all, Unity is fairly traditional-looking - Gnome 2 with a dock at a blush, and Gnome 2 already looked like Windows XP. Canonical specifically chose that conservative style over the more ultramodern Gnome Shell, however, and frankly, the same subset of intermediate users who balk at Unity on the Ubuntu side are going to be flipping out at Windows 8 on the Windows side. 

From the few minutes of video we've seen, Windows 8 does look to be very accessible for relatively computer-illiterate people while having a lot of potential for more advanced users nonetheless - it's a good strategy for a UI, and it takes a lot of risks that even Gnome Shell doesn't. 

I'd rather just appreciate it on its own merits, though. It's the interface we've come to associate with tablet Linux, combined with a tiling window manager and the neat little app widgets Microsoft actually came up with all on their own for Phone 7, which no one else really has. It's pretty cool. It's legitimately a UI that gets out of the way.

How does Linux compete with that? Well, the Compiz Grid plus a Scale corner plus turning off window decorations comes fairly close. = )

### Post by aysiu on 2011-06-06
How *will* Ubuntu try to compete? By doing the same thing it's always done--added new software, tweaked the user interface, hoped to make things more "user-friendly."
