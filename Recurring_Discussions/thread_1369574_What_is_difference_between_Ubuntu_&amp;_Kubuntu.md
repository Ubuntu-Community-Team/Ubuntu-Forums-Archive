### Post by sandy8925 on 2010-01-01
Ubuntu and Kubuntu are the same thing - they are both Ubuntu systems. Ubuntu uses Gnome as it's desktop environment and Kubuntu uses KDE as it's desktop environment. Both Gnome are KDE are user friendly and have awesome features. 

However, KDE 4 is a bit resource intensive and to run it smoothly you'd need to have atleast around 512 MB RAM. KDE also has many cool desktop widgets and lots of eyecandy.

### Post by gosalia on 2010-01-01
As said earlier Ubuntu and Kubuntu are both products of Conical Limited. Ubuntu has Gnome as its graphical desktop interface whilst Kubuntu uses KDE. Are you aware of the terms? Check out these images if your are not. [Gnome]("http://i.i.com.com/cnwk.1d/i/tr/contentPics/techrepublic_gnome_default_desktop.jpg") and [KDE]("http://pillateunlinux.files.wordpress.com/2008/11/1-kubuntu504-1.png").

See the difference for yourself.

By the way, the intension upon making Gnome was to have a user-friendly interface for Macintosh users as it uses two panels and has dock capabilities. Whilst KDE is more for Windows users as it has a start menu and only one bottom panel just like Windows.
