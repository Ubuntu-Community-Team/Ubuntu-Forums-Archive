### Post by voteforpedro36 on 2008-08-23
Let's see here...

Firefox: Good browser, fits in with GTK themes good. It loads fast enough, but is a memory hog sometimes, especially if I'm looking at high-quality jpg's on the web.

Opera: Another really good one, it doesn't fit in with GTK at all, it being QT and all. I love it, but I have a hard time using it now that I'm on a "looks matter" attitude, and while styles change some things, they don't change the menu colors and QTConfig usually just makes me screw up things worse.

Iceweasel: It's Firefox rebranded by Debian, Flash support is even worse, although I didn't try to install Flash 10 when I was using it.

Safari: No Clue

IE: Not good when I used it on Windows, slow, not configurable when it comes to looks (just like Windows), although I haven't used 7

Midori: Could be a great browser, I like it a lot, but it's buggy now.

Epiphany: Not bad, probably better w/Webkit, but I don't wanna pull in GNOME and try.

HotJava: wut?

Netscape: Never used it.

Konqueror: Didn't use it for long, but it's not bad, but it doesnt have the features of Firefox/Opera.

For me, Opera is the best browser, although I'm using Firefox, due to the QTConfig issue I have.

### Post by mcduck on 2008-08-24
Firefox. During the years I have found a couple of extensions that make my browsing so much easier that I wouldn't want to live without them. And some extensions that make my work just as much easier.

(Adblock+, Tab Mix Plus, Web Developer Toolbar, Stylish and Linkification)

So I'm not "overloading" FF with all kinds of features, just using a couple of nice extensions that work on background, and then enable the web developer toolbar when I need it.

I know that Opera can do adblocking, and I've understood custom per-page CSS code is possible as well, but they aren't as easily and quickly done as with firefox & these extensions. Opera definitley doesn't provide the kind of tabbed browsing I can acchieve with TMP. Linkification I could live without, but it's so small that I only remember even using it when I use some other browser and start finding badly made links on web pages.. :D

Oh yes, I sometimes also use Links2. ;)
