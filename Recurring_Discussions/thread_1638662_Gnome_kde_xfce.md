### Post by idi0tf0wl on 2010-12-06
These are Desktop Environments (DE), or Window Managers. They're responsible for how your windows (get it?) and how you get you your stuff. If you installed vanilla Ubuntu, chances are you're using the GNOME desktop environment. KDE is more like the way Windows works. Xfce is GNOME but smaller and less effective. Other, more distinctive environments exist, and you can install and use any of these environments (a mild proficiency in Google-fu should do the trick) without disrupting the filesystem (i.e. your stuff).

That's the beauty of Linux and the open source world: you can do what you want! Good luck!

### Post by WinterMadness on 2010-12-06
since youve said you dont know much about computers, ill try and make this as simple as possible:

its pretty much the way your computer looks. If you were on windows 7, you see the taskbar at the bottom, and the start menu, the way the windows look in general that is windows desktop environment (dunno if it has an actual name)

If you look at mac OSX youll see it has the white bar on top, and that panel at the bottom as the way to get around the OS in a graphical way.

Linux has this as well, but we have options (Linux isnt the only os to have options either)

We can choose several different DEs but it almost always comes down to KDE vs Gnome. Gnome is more mac like, KDE is more windows like... Kubuntu and Ubuntu have no differences under the hood. Think of a desktop environment(in regards to using kde or gnome on the same os) as two cars with different style bodies, but the same engines etc.
