To o/p:
CLI Companion is a tool aimed at making the terminal easier to use: it's a GUI that displays a list of commands and an embedded terminal under it. The application comes with a list of commonly used commands by default, each having a short description and if you want to find out more about a certain command, simply right click it and select "Help". This will display the "man" (manual) for the selected command.

### Post by rg4w on 2010-12-08
Respectfully:

Linux != Windows

But also:

Ubuntu != Terminal

Yes indeed, much of the power of Unix-based systems lies in the command line.  I enjoy it on Linux, I enjoy it on my Mac, and I laugh every time I have to use the crappy not-updated-in-more-than-a-decade DOS junk in Windows.

The command line has its place.  But that place is not in the daily habits of the new user, who will be the majority very soon as Ubuntu continues to expand into consumer markets.

The OP wasn't asking about how to make Ubuntu look like Windows.  On the contrary, in my reading he noted how much he likes Ubuntu, just not the command line.  

He's not alone.  Outside of Linux experts, most feel that way.  We can love Ubuntu and still love those days when we don't have to open Terminal just to get a day's work done.

There's a reason Ubuntu ships with a GUI.  There's a reason all GUIs have menus:  so you don't have to memorize lists of commands.

I'm very grateful for the helpful folks here who have taught me, and continue to teach me, the things I can do with my Ubuntu system using the command line.  It is a very powerful part of Linux, and one all developers and serious users can appreciate.

But at the same time I can fully appreciate how so many newcomers to Ubuntu would prefer to be able to get through the day with the GUI alone.

As Ubuntu gains ever more users, its installed base will more and more come to reflect the demographics of the general gene pool.  This is a good thing, but it also means that the command line will become ever more clearly recognized as a tool for experts and admins, and not something every Ubuntu user is expected to master.

So please, let's exercise a little patience with questions like this.  It's not the first time we've seen it, and we'll only be seeing it more and more until the OS matures to the point that the command line is no longer needed to do common tasks.
