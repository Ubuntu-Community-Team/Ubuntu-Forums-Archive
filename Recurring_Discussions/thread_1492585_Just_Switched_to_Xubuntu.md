I can confirm this. 

A vanilla install of Xfce on slimmed-down Ubuntu uses only 94 MB of RAM at boot. LXDE used about 87 MB at boot. I don't think 7 MB is a huge difference and I wouldn't call that "being a hog." However, Xubuntu's default configuration used closer to 160 MB.
