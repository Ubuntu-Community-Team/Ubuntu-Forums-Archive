### Post by seeker5528 on 2011-04-02
> **wolfen69 said:**
> Unity doesn't look like a mac to me.

Until we get "Minority Report" type UI's, we are limited to a finite set of UI's involving a mouse, keyboard, and screen. Of course there will be imitation.

The system in Minority Report that was controlled by the power glove was not *that* advanced, it was a limited function system created specifically for sifting through videos/images/documents in a way that is not so far removed from the way you might sift through images and documents you had laid out on your desk in the physical world. Any additional relevant data that was not part of the original data set was collected by somebody on a system using a keyboard and mouse, that was then copied to an external storage device, which they then carried to the other system and plugged in so the data was available to the person with the glove.

The technology *is* there. Multi-touch/gesture control keeps advancing and using something like the Kinect camera to view and help convert/interpret physical motion to input seems like the next logical extension to that.

But even with a limited function device you may have a variety of functions that then require menus, launchers, task management, etc... and when you need to input text, that still doesn't do away with the need for a keyboard, holding your hands in the air and typing on nothing, I think, will be a difficult adjustment for many people even if you have an onscreen keyboard showing a real time visual representation of the finger positions and key presses and that wouldn't technically be doing away with the keyboard any way, you would just be trading a physical keyboard for a virtual one.

Voice input could eliminate the *need* for a keyboard, but voice technology has been around for a long time and while it is great in the car, for use in automated help lines, etc... people don't seem to find it all that attractive on their computer outside of dictation and accessibility for people with physical limitations.
