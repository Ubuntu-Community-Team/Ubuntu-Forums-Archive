I guess by now you'll know how to open an app with admin privilages, but I'll explain it again.

Let's say you need to edit a file in /var

You have a few options.

You do everything in the terminal using

```
sudo nano /var/file
```

You could open a text editor using gksudo and then use the text editor to open the file.

```
gksudo gedit
```

Or you could use Nautilus (the file manager) in root mode.

```
gksudo nautilus
```

You can add the last two things to the Applications menu ("system -> preferences -> main menu").
