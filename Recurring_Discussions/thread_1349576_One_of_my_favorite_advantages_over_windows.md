### Post by SeanBlader on 2009-12-08
Seems simple, but snap-to screen edge from Compiz is one of my favorite features of Linux in that it allows me to make use of all of my screen fairly efficiently. Unfortunately with Windows, there's no such built-in feature, which means if I want to use all my screen real estate I have to be very precise in moving my windows right to the corner of the screen. I almost never like to have my windows maximized so I can click around them, so the simple thing of compiz having a snap-to screen edge is something that I really like having on my laptop where screen real estate is precious.

Who else has something in Linux besides it being free that they like over Windows?

### Post by MicrosoftFan on 2009-12-09
> **argosreality said:**
> To be fair, while cmd allowed you to batch commands I dont think it had the capability to pipe output.
 
Yes it does. It has the same Unix redirection operators. "|" sends the output of the command before to the input of the command after in the same way.
 
[http://technet.microsoft.com/en-us/library/bb490982.aspx](http://technet.microsoft.com/en-us/library/bb490982.aspx)
 
Examples:
```

help | more
 
dir | sort
 
sc query | find "RUNNING" /c

```
(Edit: last one is a useless command but you get the idea)
