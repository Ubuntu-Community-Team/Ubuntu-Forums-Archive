### Post by coffeecat on 2011-06-30
> **orky7 said:**
> i am using 10.04 and firefox-4 released a way back. so why do not they upgrade it to 4, instead we have to install by other means. whats the logic.

The logic is that enterprises will use LTS releases in preference to ordinary releases - 10.04 is a LTS. Enterprises need stability, not the latest and shiniest. Enterprises do not want version upgrades of applications because that involves staff training costs. The difference in the user interface between Firefox 3 and 4 is significant. You may find it easy to adapt - as do I - but employees in enterprises who have no particular interest in computers and operating systems need to be trained to the new interface.

The choice for the ordinary non-enterprise user is to use the latest version of Ubuntu if they want more up-to-date application versions, or stick with LTS releases for stability - which includes older versions of software. Or use ppas for later versions of applications in LTS versions of Ubuntu - and risk a potential trade-off in stability.

Because as a rule, Ubuntu does not upgrade any of it's applications with new major versions that introduces new features or extensive code changes, only with security and stability patches. 

For example, if you have Firefox 3.6.17, then you will get the 3.6.18, then 3.6.19 and so on, but not Firefox 4, which is a major upgrade. 

Why? Because of stability. 

As coffeecat explained, many users rely on the stability of Ubuntu, specially if they have chosen an LTS release. Upgrading to Firefox 4 could lead to many problems. 

Some time ago, Mozilla stopped providing support for Firefox 3.0 and the UbBuntu developers had no other option other than upgrading everyone to 3.6. They took some time testing the new version to make sure the upgrade wouldn't compromise anyone's system stability, but a lot of problems arose when they did the actual upgrade. 

So, what I mentioned in the Mega Thread, that you quoted above, is that Mozilla will stop providing support for Firefox 3.6 soon, so there won't be any option other then upgrading everyone to the latest stable release, which is currently version 5.0. However, there is no reason to rush such upgrade, as long Mozilla continue to provide security and stability patches for Firefox 3.6. The leap from Firefox 3.6 to 4 or 5 is huge. If users are already complaining about the upgrade from 4 to 5, imagine what will happen when they get the upgrade from 3.6 to 4. If you want the latest release, you can use a mozillatem ppa, which is provided by the Ubuntu developers and it is easy to install.
