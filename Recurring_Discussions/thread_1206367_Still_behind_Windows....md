### Post by Teamgeist on 2009-07-07
It is a seperate programmm and it is meant to be.

Ubuntu upgrade policy does not allow any other updates than securit updates.

The devs were wise enough to put FF3.5 in universe (as it was still a development version when Jaunty came out) so people can easily upgrade now.

The version (still called Shiretoko) that is in Universe was updated yesterday and is a build from July 1st. The day Firefox 3.5 came out.

So it in fact is FF3.5's final Version.

To have FF3.5 system wide as your default browser just run 

```
sudo ln -f -s /usr/bin/firefox-3.5 /usr/bin/firefox
``` in a Terminal.

Have fun with it.
