### Post by darkod on 2013-02-24
If the desktop environment is all that is bothering you (Unity), you can simply install another DE. Just google something like "how to install another desktop environment on ubuntu12.04" and it should give you plenty of options/tutorials.

For example, one thing people use is Gnome Classic, I think the actual package name is gnome-panel (google can also help you with that). After you install it, on the login page if you click the ubuntu logo next to the username, it will offer Gnome Classic as option in a lit that will also contain Ubuntu (the default Unity).

Select the classic, enter your password and that's it. If you ever wish to, you can log in with unity too. I think it remembers the last option used, so you wouldn't need to select classic on every boot if you use it regularly. Only the first time.

Some people use the MATE desktop, XFCE, etc. You can also find more details on those DEs on google, and screenshots.
