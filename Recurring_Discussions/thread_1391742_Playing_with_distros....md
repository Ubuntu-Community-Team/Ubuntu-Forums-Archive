### Post by RedSquirrel on 2010-01-28
> **Marvin666 said:**
> So far the poll results are Fedora, Debian, Mint, and Gentoo. Arch, and Slackware are out due to lack of gui installer/configuration.

Gentoo does not have a graphical installer.

If Arch and Slackware are out, then Gentoo is definitely out. For Gentoo, there is no installer at all. The installer is **you**. You will manually run the commands to install Gentoo. See the [Handbook]("http://www.gentoo.org/doc/en/handbook/") to get some idea of the task you would be facing.

Slackware does have a GUI installer of sorts. It should be fairly straightforward for you to get through, especially if you let the installer install every package in the default package sets (which the installer will say is the "recommended" action). There is an official Slackware package for Xfce, by the way. If you're looking for a break from Ubuntu, I believe Slackware would be a good choice. There will be a learning curve, but you will at least be able to say, "Whoa! This is different.". You'll want to read the various README, HOWTO documents on the Slackware mirrors. It would probably be worthwhile to browse the [Slackware forum]("http://www.linuxquestions.org/questions/slackware-14/") (especially some of the sticky threads there).

I'm not really a big fan of Arch, but their Beginner's (Beginners'? :P) Guide is very good. It might seem like a daunting task to install Arch, but that's mostly due to the level of detail provided in the guide. It doesn't really take that long to complete and there is an "installer". You will be left with a command-line system when you complete your Arch install. However, adding the X Window System and your favourite DE/WM is straightforward and well-documented.

Having said all that, if you _really_ don't want to use the command line  for installation/configuration/maintenance, then Slackware/Arch/Gentoo  are not for you.

Debian or Fedora might be worth a try. They don't have a steep learning curve and they would be at least somewhat different from Ubuntu (especially Fedora, for obvious reasons).
