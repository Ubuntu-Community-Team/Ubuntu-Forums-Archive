### Post by RiceMonster on 2010-01-05
It's fantastic. PackageKit has a nice clean interface, and if you're familiar with apt, you won't have any trouble learning yum.

When you install it, I recommend you add the [rpmfusion](http://rpmfusion.org/) repository so you can get support for mp3s and a few other things:
```
su -c 'rpm -Uvh http://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-stable.noarch.rpm http://download1.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-stable.noarch.rpm'
```

Also, it's very up to date, so for many people this a good thing, but it also means you may run into some bugs. So that's something to watch out for.

Fedora 12 is pretty good.

The Package manager (yum) is better than Ubuntu's (apt-get) in the command line, but Synaptic and the Software store in Ubuntu makes better GUI interfaces.

The applications are more "bleeding edge", for example my Fedora 12 is running Mozilla Thunderbird 3.? beta right now. It works nicely but it is a bit of a dissapointment when I return to Ubuntu or Windows and realize the stable version is only 2.?

I have a Broadcom wireless card in my laptop and Fedora 12 is the first and only distro I have tried that has detected and was able to use my wireless out-of-the-box with nothing downloaded or fwcutter or anything.  It uses open source firmware [[COLOR="Blue"]OpenFWWF[/COLOR]]("http://www.ing.unibs.it/openfwwf/").

It's pretty stable, and the bug reporting feature makes it very easy to report the bug, but the bugs I've encounter have not taken the system down or anything.  Most I've had is my Firefox browser closing on me.

Fedora gives you 3 (I think) options.. [LIST=1]
[*]the all-inclusive, pick-and-install DVD
[*]the LiveCD with Gnome
[*]the LiveCD with KDE
[/LIST]

I always end up with the LiveCDs because my laptop doesn't have a DVD drive and I do like to try it (usually only once) before installing it.

Oh, or there is the LiveUSB, so you don't have to burn a CD.  I've been trying to use these but openSUSE just doesn't play nice on USB drives so I gave up for a while.  I should start that practice again and save some money.

I usually download the ISO file to my desktop and use uNetBootin (available for Windows and Linux, Ubuntu's runs fine) to install from the ISO file to the USB drive. Too bad the installation DVDs are usually > 4 GB and my 8GB stick died so all I have are 4GB ones now.

The other reason I don't go with DVDs is you don't get the "try before you buy" and it just takes so darn long to download (I have a slow DSL connection)... like 15 hours or something.
