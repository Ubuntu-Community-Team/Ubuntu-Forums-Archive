### Post by 3Miro on 2011-05-08
> **karimruan said:**

Since you seem to know so much about this (an area I am only able to review from a user perspective), any information you can throw my way would be appreciated.

Discussion of the UI is fine. You can talk about Unity, however, you cannot talk about a generic Gnome as Gnome2 and Gnome3 have a very different UI (the window manager and UI of Gnome3 are called Gnome-shell). Gnome2 to Gnome-shell is at least as big of a change as Gnome2 to Unity. Many of us like Gnome2 but not Gnome-shell and I for one would much rather use Unity than Gnome-shell.

When the Gnome2 developers decided to discontinue the project, Canonical had three options. They could either accept the new drastically different interface of Gnome-shell or change the Ubuntu DE (i.e. go for XFCE or KDE) or they could make their own interface. They decided to make their own thing which would definitely appeal to a wider audience than Gnome-shell.

If you want to write a blog about this, you should test Gnome-shell yourself. Fedora 15 comes with Gnome3 + Gnome-shell and it is in beta stage of development, depending on your video card you may be able to just run that from a LiveCD. Arch has Gnome-shell in its official stable repositories. There is a way to test Gnome-shell (just the interface) on Ubuntu 10.10 and 11.04, I think there is a version of GS in the Software Center. You can install just the interface part of Gnome-shell or you can even install Gnome3, however, you should note that installing Gnome3 on 11.04 will break Unity as Unity is still using Gnome2.

XFCE and LXDE interfaces are very similar to Gnome2, you have panels with applets and so on. KDE is somewhat different. While you still have the regular panels, there is very interesting integration between the panels (kde-panel), the window manager (kwin) and the desktop space (called plasma). KDE has a number of widgets that can live on both the desktop and the panel and give you all sorts of goodies. For example the menu button can be placed right on the desktop, or you can have files and folders right on the panel. KDE is something worth trying (especially if you like customizing).
