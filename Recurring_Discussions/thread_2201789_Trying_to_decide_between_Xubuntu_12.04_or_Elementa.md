### Post by Bucky Ball on 2014-01-31
Replace 'user' with your username.

Scenario: disk1 = /home/user directory inside your / partition (your operating system partition). /mnt/data/Documents = a separate data partition mounted in the /mnt directory. 

So this:

```
ln -s /mnt/data/Documents /home/user/Documents


```... will put a symlink (or shortcut) to the directory /mnt/data/Documents in your /home/user directory inside /. The syntax is this (replace with your details)

```
ln -s /path/to/target /name/of/symlink
```

Hope that makes it a little clearer. ;) If you link XP also, there is no need to have that partition so big as the operating system is all that will be on there (XP only needs about 14Gb). Same with Elementary (this symlink line will work identically in that).
