### Post by Mark Phelps on 2011-07-26
Ubuntu is a general-purpose Linux distro.  It is not designed to be a multi-media server, although it CAN do that if you load up enough apps.

One issue you're likely to run into in Ubuntu is the lack of drivers for specific hardware.  And, although this can also be true of MS Windows, with Win7's driver legacy modes, this is not so much a problem.

Unfortunately, I don't use Ubuntu as an HTPC, so the only thing I've read about is MythTV -- which is designed to be a DVR replacement.  The link to info on it is below:

[http://www.mythtv.org/detail/mythtv]("http://www.mythtv.org/detail/mythtv")

Perhaps other folks who DO use Ubuntu as an HTPC can give you a more definitive answer.

### Post by Copper Bezel on 2011-07-26
The only HTPC I've owned ran Ubuntu, but I understand that a lot of folks would say that an HTPC that can't run Netflix is missing the point. I do think that having a fully functional OS on your TV has some advantages over using a completely dedicated system, with little disadvantage.

If you're comfortable with Ubuntu, it's not going to give you any real problems as an HTPC that Windows 7 won't. If you're more comfortable with Win 7, you're not going to get any special super powers by using Ubuntu. At default, it's not really any lighter than Win 7; you can *make* it lighter, quite a *lot* lighter, but I don't know what the real benefit is to that if you're just planning to use it for media anyway, and it's not going to be worth the effort if you aren't comfortable using Ubuntu to begin with.

Please don't encourage people to install outdated software with massive security holes. Windows 7 isn't a great operating system but it's better than XP by a long shot.

iPlayer works on Linux, doesn't it? I thought I once saw some instructions on the BBC website for it. Digital TV cards work in Linux - not all, but some of them. You can watch digital TV in MythTV and use it as a PVR.

As for Bluray, you can't "play" Bluray discs but I don't have any trouble ripping them and then playing the main movie from the hard disk.

Use whatever suits you, of course. If you think Windows is going to give you a better HTPC experience, use it. But a lot of people use XBMC, Boxee or MythTV (Mythbuntu really; it's easier to configure than straight MythTV) as a HTPC/media center.

The other thing you could consider is a couple of boxes for your TV. You can get network media playback boxes that can stream video files from your main computer and also record digital TV to its hard disk. They're not too expensive. You can also buy Bluray players for less than $200. It might be easier and cheaper to just get those two different devices.
