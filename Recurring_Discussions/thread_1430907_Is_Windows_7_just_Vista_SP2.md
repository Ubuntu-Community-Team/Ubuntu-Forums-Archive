### Post by Giant Speck on 2010-03-22
Windows Vista (then called Longhorn) was originally meant to be a minor step between Windows XP and Blackcomb, which was supposed to be Microsoft's next major operating system (and what is now known as Windows 7).

Longhorn was meant to be released in 2003 and Blackcomb was meant to be released in 2005.  However, as time progressed, more and more features that were slated for Blackcomb were actually added to Longhorn instead, thus pushing back the release date.  Because developers also had to work on updates for Windows XP and Windows Sever 2003 after the 2003 "summer of worms", Longhorn's development was delayed.

Because of this, in 2004, Microsoft announced that it had to revise its plans.  Blackcomb was scrapped, and the development of Longhorn was begun anew.  This time, instead of being built upon Windows XP's source code, Longhorn was built upon Windows Server 2003 SP1, which fixed flaws that were exploited by some of the major viruses that arose in 2003.  Many of the planned features (including WinFS) were either dropped or postponed.  Windows Vista was finally released to manufacturers in November 2006.

Windows 7's development was not a reaction to the reception to Windows Vista; rather, it was an underlying project that was already in development before Windows Vista was even released.  However, this certainly does not mean that they didn't learn from the disaster that was Windows Vista.  Vista certainly did have a large impact on the development of Windows 7.
