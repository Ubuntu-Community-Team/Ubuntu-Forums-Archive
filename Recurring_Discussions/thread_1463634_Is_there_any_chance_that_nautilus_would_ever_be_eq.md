### Post by MaxIBoy on 2010-04-28
No, it will never be equal, in the sense that it will never be exactly the same. If you want it actually the same, you'll need to use Windows. WINE has an explorer.exe but it isn't targeted at looking like Windows 7.


As for specific complaints:

You probably don't need to compress most ISOs, because they tend to contain compressed data anyway (i.e. if you compressed a movie, music, or software ISO, the result would likely take up more space instead of less.) File-roller (the archive manager program) is used to thinking of ISOs as compressed archives, so when you open it with file-roller offers to extract it, not to compress again. Imagine what would happen if you opened a zip file with file-roller and it offered to put it inside another zip archive!

Adding new features to Nautilus is as simple as installing or writing a plugin. There are probably plugins to do whatever you want. It's not up to the developers of Nautilus to decide which ones should be installed by default though. Maybe you should be complaining about the package maintainers for your distro, instead.

EDIT: To sort images by resolution is pretty trivial to accomplish using a shell script that invokes graphicsmagick or something like that. If you can write it in a shell script, then it can easily be rewritten as Nautilus plugin. Just follow this guide: [http://www.linux.com/archive/feature/114134?page=3](http://www.linux.com/archive/feature/114134?page=3) to write some Python code to invoke graphicsmagick, and you have a quick-and-dirty solution. Of course, if you're not a programmer I'm not saying you need to do this, but the tools are there if you want them. But that's a nifty enough feature that I may do it myself at some point.

Well, I think this is a great time to voice my disappointment with the Windows 7 file explorer and my praise for the Nautilus file manager.

[ for Windows ]

First off, at the very top of my list, is the fact that Windows 7 has abandoned "Explore" in the context menu.  Try it.. in XP or Vista, right-click and pick Explore.  Now try and do that in Windows 7.  This is an absolute showstopper for me.  If I can't get to a tree FAST and move around EASILY, you're all done right there.  What the heck were they thinking?

Second, there are so many new "shortcut" items in the explore-less explorer, that they are actually in the way now.  I can't get rid of all the "pretty" and get a clean explorer.  This, to me, is also a showstopper.

Customizations are getting fewer, harder to find, and they don't work.  If you've tried to customize the Explorer, you certainly know what I mean.  Need I say more on this?  I think not.

Why, oh why on earth, would I *want* to categorize things like photos,  by resolution, in my FILE manager.  That's what a picture manager is  for.  If they would stop trying to roll fifty features in to one place, perhaps we could navigate with meaning.

And last, all these "pretty" new features that help first-time users and newbies classify, categorize, search and find, and "help" them do things, are now just fluff.  They are big, bulky, clunky, and playing catch-up with Mac.  It's almost embarrassing - but thankfully, not for me.

[ for Nautilus ]

Oh look, there is my tree, exactly how it should look.  Oh wait, I can change it to look how I want it to look, when I want, as often as I want.  Did I miss something?  It doesn't matter if I did, I can learn more later and do more later.  I'm not only allowed to do that, I'm encouraged to do it, and share my results with others.  Gee.  What a concept.

Clean. Clean. Clean.  *clapping*  Look at the clarity of it all.

Right-click an .iso file, write to CD.  Isn't this how it *should be?

It interfaces with file compression, default programs, other drives and volumes where I want to see them, and it doesn't complain when I ask it to make large changes all at once.  Oh, and the progress bars show real progress, not just a heartbeat to pacify me.

Zoom controls, connect to server, bookmarks, extra pane.  What do you mean it is feature-starved?

Panels with a thousand wonderful add-ins - when and where YOU want them.

[ My conclusions ]

Why would I *want* it to be equivalent to Windows 7 file explorer?  That's taking 4 or 5 steps backward sacrificing performance in favor of "pretty".  If I had wanted to eat McDonalds, I would have lined up with the herd.
