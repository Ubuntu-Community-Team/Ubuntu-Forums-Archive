### Post by lovinglinux on 2009-05-06
I didn't tested Vista SP2 or SP3, but I feel Windows 7 is much better than Vista when first released. Not enough for me tho.

One thing in particular I think was a big improvement is the firewall, which seems to be a decent one now and easy to configure. It has a nice feature that allow you to disable rules, instead of changing it's properties or deleting the rule completely in order to temporarily allow/block a service.

I don't like the way things are organized, but I guess just need to get familiarized with it. Nevertheless, I was able to find everything I needed using the help tool.

Installation took only 18 minutes to complete and all my hardware was properly recognized and installed out-of-the-box, including my TV card, which required a driver hunting on previous versions.

It seems to run fine on my system with Aero enabled (P4 3.06 Mhz HT, 2Gb DDR2 ram, nVidia 7300 GT card, rated 4.0 with the built-in Windows performance tool), but I get FPS problems running America's Army game. Nevertheless, I always had to tweak the game settings, the gfx settings and system on XP and Vista to make it work properly. Didn't have time to do it yet.

Microsoft still insist on leaving several useless services running by default. Memory usage ranged from 26% to 30% after boot. 

The interface is beautiful and with nice little features. I like the new tray layout.

I still need to test a lot of things, but so far it looks good. One thing I would like to know is how it will behave 6 months from now of regular usage? Probably terribly, like any other Windows OS.
