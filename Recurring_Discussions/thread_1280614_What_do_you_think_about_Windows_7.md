### Post by ve4cib on 2009-10-02
I've been using it at work for the past several months.  It's okay, but I've been plagued by a few small bugs that make me feel like I'm using Ubuntu 5.04 all over again...

1- Aero flickers.  Moving windows around or typing too fast (apparently) makes sections of my screen start to flicker.  Dragging another window over the affected area seems to stop it temporarily.  But then it comes back later.  It's like using Beryl from a few years back.  If I had a better GPU in this machine it might work better though.

2- I cannot install Windows Updates.  Or rather it's tried to automatically install the same update (some service pack for Visual C++) but it fails every time.  So now I have this ginormous queue of "Reboot to install this update" requests.  And every time I shut down the computer it tries to install the update again.  The problem is, because I essentially have an "Installation in progress" I cannot install other software, like the latest version of Tortoise SVN.  Which, as you can imagine, is an incredible pain in the ***.

3- I can't figure out how to un-map Win+Space from showing the desktop.  I want to use Win+Space to show Launchy, but Windows seems to have decided that I have to be able to peek at my desktop (which is a pretty useless feature -- the desktop doesn't change much for me.  Peeking at windows, great.  Desktop, not so much.)

The new version of Windows Explorer (new to me anyway, this may have been around in Vista too) is a vast improvement over the XP-era one; you can have bookmarks in the sidebar, just like Nautilus and Thunar.  How grand!

KDE4 for Windows works reasonably well under Win7.  I've never used KDE4 under any other version of Windows, so I don't know how many of the limitations are due to KDE and how many are due to Windows, but it's kind of nice being able to use Dolphin to browse files.  Despite the improvements to Win Explorer it's still by far the crappiest file browser out there in my experience.

My problems are likely due to my using the Beta/RC versions, and not the "stable" final one though.

Overall I'd say the experience is mediocre.  Windows 7 is not going to convert me from Linux (big surprise from someone posting on this forum).  It's got its nice features, and it's bad ones.  Some of the eye-candy (particularly the mouse-tracking glow on the Taskbar icons) I expect to be copied by the Gnome and KDE devs in the coming months.

Now, for what most people use their computers for I don't think it is even necessary to use Windows.
