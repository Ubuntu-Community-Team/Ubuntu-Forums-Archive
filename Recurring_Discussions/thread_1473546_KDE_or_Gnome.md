### Post by yyyc186 on 2010-05-08
> **Neutron1998 said:**
> Is it possible to install the KDE environment on Ubuntu 10.04 after installation?  I'm still new to Linux so please forgive me.  :)

One of the never ending problems with the Ubuntu packaging community is that they NEVER bundle KDE and Gnome together on the same distribution, despite endless requests for it to happen and despite other distros doing the same.  OpenSuSE lets you install both from the same DVD.  Actually they might even have a few other desktops as well, but OpenSuSE is a different world from Ubuntu.

For some in-explicable reason, the Ubuntu camp decided to standardize on Gnome (a desktop fit for when programmers lived in caves and ate their young) instead of the more robust KDE desktop.  I can understand the push.  Unless you are a Linux expert, you cannot really install or modify anything under Gnome.  Something as simple as adding a font for your documents required locating a dark lord maige and begging for the list of crytpic text files (plural) one had to hack along with the magic invocations (plural) one had to issue to get the font recognized by the system.  On KDE it's a menu option.

The major downside for anyone trying to install a second Ubuntu desktop is that it requires _massive_ download bandwidth.  While _some_ people in the world have a cable modem and ISP which provides unlimited download bandwidth, others have satellite, dial up, or wireless broad band.  Each of those enforce either a daily or a monthly download limit.  When you choose to install the second desktop, you just cannot do it cleanly with any CD provided by the Ubuntu community.  You end up doing it via one of the package management tools and downloading roughly another 700 meg.  If you are on a satellite connection with 120Meg/day DL limit, you're pretty much hosed.  A dial up user will see their phone tied up for days, and a wireless broadband user near the edge of their packet limit will simply get a monthly statement for a few thousand dollars.

The install procedure for Ubuntu needs to go to a multi-disk distro so a user can install both KDE and Gnome, then learn just how little Gnome has to offer.
