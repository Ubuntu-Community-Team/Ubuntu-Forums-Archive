### Post by qyot27 on 2010-10-22
> **Wild_Wes said:**
> I only use windows when i game because when you surf with it you are more likely to get a virus. I hardly game though.:KS
'More likely' is a condition of user preparedness, actually.  Except for a nasty hiccup this past weekend, I'd been virus- and malware-free for 3 or 4 years or so, just due to my basic security regimen (which is based more on passive security than active, so it isn't something I have to constantly think about).  It also helps to know how to use Safe mode effectively or even using Ubuntu to nuke junk from orbit.  An ounce of prevention is worth a pound of cure, as they say.

I dual boot XP and Ubuntu, have since 2005 (save a few releases due to a graphical issue I didn't realize how to fix for two years...fairly sure it appeared with 6.10 and I didn't find the solution to it until 8.10).  I also use my parents' iMac for a few tasks, so I've used both Tiger and Snow Leopard rather frequently - much more frequently in the latter's case*.

*I'll admit here, I kind of cheat - the two or three things I use it for are DVD burning (which is done through the native OSX Disk Utility) and converting stuff to MPEG-2 using HCenc or Xvid using mencoder through Wine.  The Wine stuff actually means I'm using OSX like Unix, so it's not different at all compared to when I use Wine under Ubuntu.  The reason for Snow Leopard's heavier usage is I finally got Wine properly installed after they'd updated to it.  Prior to that I didn't use that computer for transcoding.
