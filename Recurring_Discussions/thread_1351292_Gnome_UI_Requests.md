### Post by Hypnoz on 2009-12-10
I use Ubuntu 9.04 and Gnome as my daily work env (I'm a linux admin so it was a no brainer).

After a while of using it, I have a list of some things I'd like to see in Gnome that would improve my user experience. Any discussion on a request is open.

_**Ubuntu + Gnome Suggestions**_
- Double click on top left icon closes window (helps with windows transition)
- Can drag with right mouse button to pull up context menu in new location (to choose between copy/paste)
- In a folder, "List View" should have some functionality from "Compact View", such as...
_____+ Can highlight many files in a window by drawing a box over them
_____+ Shift to select a list of files, Ctrl+shift to select a lower item, Shift to additionally select a list of items below that item.
For instance, you have a long list of files, and want to select 10, skip 5, select 20, skip 7, select 50. Select top, shift select #10, Ctrl-select #15, shift select #35, ctrl-select #42, shift select #92.
- Properties of a folder/folders should show how many files + folders, instead of just "items"
- Can't unlock and drag Network Manager icon on top dock
- Can't unlock and drag Pidgin icon on top dock
- When laptop is in sleep, and you plug it into a docking station with the lid closed, then wake it with the docking station power button, it should default output to the external monitor, and no display on laptop LCD. THIS IS A BIG FEATURE REQUEST FOR ME.
- "Open with..." window should have apps ordered alphabetically, maybe with 2-3 most popular, or good suggestions based on file type, on top.
- When a dialogue opens to save a file, and I start typing the filename, it opens a little box at the bottom right for searching files. I constantly have to click inside the text field box
- When I have a folder open, in the "Go To:" bar I can type smb://servername/ but I can't just type //servername/
- 100% zoom on list view and 100% zoom on compact view are completely different. 100% zoom on list view is really huge.
- Sometimes I save a picture to the desktop and the icon is HUGE, and I have to "stretch" it back down to normal.

Thank you for a great OS that I feel comfortable using every day at work!
