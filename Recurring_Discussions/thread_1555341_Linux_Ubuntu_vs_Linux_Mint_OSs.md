### Post by Nick_Jinn on 2010-08-19
> **daksai3 said:**
> if thats all there is to it, then ur right when u say that they are the same. but i still have to point out that ubuntu has a better support group and that it releases updates faster earlier since mint makes its own upgrades from ubuntu's (right??).

Well....If you use Mint you can use the Ubuntu forums. The exact same advice that will work for Ubuntu will work for Mint 99% of the time.....You might be surprised how many people HERE use Mint rather than Ubuntu. If somebody uses Mint you dont have to worry about your Ubuntu advice not working. Even the pathways in the CLI are the same. You can use the exact same commands. They even use the Ubuntu repositories for almost everything so there wont be a conflict in code for most fixes. 

So yeah, if you use Ubuntu you will get the new releases sooner than if you use Mint....There is a downside to this however.....Ever notice how there are thousands of bug reports and stability issues when a new release comes out? When Mint releases their very first download you are not getting the same thing that comes on the Ubuntu ISO....you are getting the Ubuntu ISO with all the fixes and updates from the first few weeks or longer of trouble shooting. The Mint ISO is always going to be more updated than the Ubuntu ISO, but the Ubuntu ISO is going to be released first. The Mint ISO isnt created until a lot of the bug fixes come in, so its going to be more functional out of the box before you install updates.....useful if you dont have broadband.

I also think it makes a better live CD since the live CD will already play DVDs.....something Ubuntu will not do, even if you install restricted extras. I have dual optical drives which is nice. With enough swap you can use it full time.

I much prefer the default Mint theme to the Ubuntu theme, but I usually change themes anyway. I like Greenhart and either graphite or clearlooks-black. 



Ubuntu is going to get that new release that much sooner, so if you absolutely must be on the cutting edge Ubuntu will give it to you first....However, the Ubuntu ISO is going to have all the bugs of the initial release and the Mint release is going to be more polished and updated once its released, its going to play MP3s and CDs right out of the box without messing with Medibuntu (which some people have various difficulties with), and has other proprietary drivers and codecs that Ubuntu omits for various legal reasons....The makers of Mint live in Ireland where certain laws that Ubuntu is concerned with dont apply. Its technically not legal to use Mint in some countries, which might be a positive selling point to a lot of people.

### Post by HoKaze on 2010-08-19
> **daksai3 said:**
> what about boot times and load times. which uses the hardware better? which efficiently uses computer resources?
As the underlying system is pretty much identical, the main (gnome) version of Mint and Ubuntu should both have very similar efficiency. The main differences between Mint and Ubuntu tend to be choice of software, multimedia, a few more drivers, different defaults, different themes and that's pretty much it.
The differences are a lot larger when you take some of the other Mint versions into account (KDE, LXDE, Fluxbox, etc) as they tend to vary a bit more from their closest Ubuntu equivalents (Kubuntu, Lubuntu, etc) but again, almost all of the core beneath the surface is the same.
