### Post by Greyed on 2008-07-18
Speed could be a placebo but memory is a matter of hard numbers.  When I last tried Opera 9.5 vs. Firefox 3 on WindowsXP and KUbuntu Firefox was running a consistent 10Mb higher on usage over Opera on the same set of pages.  On my game machine (1Gb of RAM) I don't worry about it too much.  On my 256Mb laptop or in a VM 10Mb could be significant.  

However the main reason that I stick with Firefox is neither philosophical or technical but personal.  I've gotten used to manipulation tabs in Firefox and to my knowledge there's no quick way to set Opera up to do the same.  The irony being that Opera was my first tabbed browser and where I learned the efficiency of using tabs extensively.  ;)
