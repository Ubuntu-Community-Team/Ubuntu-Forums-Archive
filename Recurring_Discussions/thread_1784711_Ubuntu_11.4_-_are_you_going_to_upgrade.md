### Post by AeroZak on 2011-06-17
I love the new Unity.  It's fun, looks great, and adds many new features.  However, it does use up more resources which bug some people I know.

One installation I did (on an ancient computer) didn't have the hardware to support it, so it couldn't run it.

Honestly, it feels nearly the exact same, but with many cool features that weren't there before!  At first, I wasn't very fond of it, but after a little while, I got addicted to it!

Overall, as long as your computer can run it, and you have a semi-decent computer that will be running Ubuntu, it will be fun and great!
