### Post by artsci2 on 2009-11-02
I like to install my OS's keeping them totally separate. I use a different hard drive for each OS and a third one for Data files.

For example disk 1 has Ubuntu, disk 2 windows, disk 3 is for data.
Disconnect disk 2 and 3, install Ubuntu on disk 1 using the "use entire disk" option.
Then disconnect disk 1 and connect disk 2. Install windows.
Finally re-connect all 3 disks. When you start the computer use the BIOS "select boot device" key. (eg right after starting, a message says "press F9 to select boot device") then choose the disk with the operating system yopu want to boot.

This has saved me a lot of trouble compared to when I tried to setup dual boot with gRUB. Especially when changing hardware etc.
