### Post by anoop999 on 2010-02-15
You can run any application on any desktop (e.g. you can run Synaptic in KDE). However, GNOME programs use a particular set of shared libraries and KDE applications use a different set of applications, so running a KDE application in GNOME (or vice versa) uses more memory and is potentially slower and less stable. They also use different window managers and menu programs, and come with a different set of default applications.

If you like, you can install multiple desktop environments and choose a particular one when you login!
