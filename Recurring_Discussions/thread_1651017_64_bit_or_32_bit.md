### Post by sikander3786 on 2010-12-22
First if not already sure, you need to make sure your computer is 64-bit capable ;-)

Secondly, there is no real advantage of 64-bit over 32-bit. You won't even feel the difference unless doing some compression jobs or video encoding. The daily use apps run the same at least for me on both 32-bit and 64-bit.

And now 32-bit can support RAM greater than 3GB by using PAE Kernel so that is not an advantage any more.

But, the flash issues etc that seemed to lie with 64-bit previously have mostly been solved now so for 64-bit, that is a plus.

Sometimes some 64-bit hardware is reported not working with 32-bit Ubuntu (very rare) so what I would recommend is to burn a Live CD/USB of both and test them on your system in Try mode. Test all the hardware and install the one that performs better.

Other than that, I think 32-bit vs 64-bit is extensively covered on Google.

Install Ubuntu in 64 bit. Especially with that hardware. Gimp works fine, as well as everything else. Despite popular belief, 64 bit support is actually very good in Ubuntu.
