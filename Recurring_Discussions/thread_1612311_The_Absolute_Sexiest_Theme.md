### Post by Rasa1111 on 2010-11-04
To install the AWN dock, with Lucido style available.. 

 Simply follow these instructions~

 it is a simple matter of pasting 2 commands into the terminal. 
[http://www.webupd8.org/2010/06/awn-lucido-gets-its-own-ppa.html](http://www.webupd8.org/2010/06/awn-lucido-gets-its-own-ppa.html)

If you do not already have AWN installed, Skip the first command. 
And only use the last 2. 

 If you do have AWN, the first command will remove it. 

I will repost the commands required to install it, here. 

Add the PPA~

```
sudo add-apt-repository ppa:awn-testing/ppa && sudo apt-get update

```

Install AWN with Lucido~ 

```
sudo apt-get install avant-window-navigator-trunk avant-window-navigator-data-trunk python-awn-trunk awn-settings-trunk awn-applets-python-core-trunk python-awn-extras-trunk awn-applets-python-extras-trunk awn-applets-c-core-trunk awn-applets-c-extras-trunk

```

 Once that is finished, 
Youll find AWN under "Accessories" menu. 
as {Avant Window Navigator]. 

 Now, If you want to install the AWN new hope theme, once you have AWN installed, 
That is as easy as going into the AWN preferences,
going to the "Themes" tab, Click "Install"
 Navigate to jurialmunkeys New Hope AWN theme, and click 'open'.
 And it will be installed in your AWN themes.

@Ranger_Joe, you do follow the monthly Screenshots threads in the Community Cafe, don't you?...
[http://ubuntuforums.org/showthread.php?t=1610583](http://ubuntuforums.org/showthread.php?t=1610583)

Your distinction of a panel vs. a dock isn't right as you can easily remove both the separator and the space from gnome-panel. Now that docks are beginning to add their own applets like a clock or weather report the lines are being blurred, but a dock is an interface for controlling the launching and switching of applications, it may or may not be stand-alone. A panel is a parent widget for holding children widgets, of which a dock may be one. You can add DockbarX or Talika Applet to gnome-panel, they both provide dock functions.

Here is my current desktop running Xfce4, xfce4-panel and Talika Applet (via xfce4-xfapplet-plugin and gnome-applets). You see that I can launch/minimize/maximize my browser, terminal and file manager directly from Talika, or any other application that I "pin" to the panel, just like a dock, but I can still have my clock and main menu all on the same panel. Talika allows you to switch the application's standard icon with a custom one as well...

Your distinction of a panel vs. a dock isn't right as you can easily remove both the separator and the space from gnome-panel. Now that docks are beginning to add their own applets like a clock or weather report the lines are being blurred, but a dock is an interface for controlling the launching and switching of applications, it may or may not be stand-alone. A panel is a parent widget for holding children widgets, of which a dock may be one. You can add DockbarX or Talika Applet to gnome-panel, they both provide dock functions.

Here is my current desktop running Xfce4, xfce4-panel and Talika Applet (via xfce4-xfapplet-plugin and gnome-applets). You see that I can launch/minimize/maximize my browser, terminal and file manager directly from Talika, or any other application that I "pin" to the panel, just like a dock, but I can still have my clock and main menu all on the same panel. Talika allows you to switch the application's standard icon with a custom one as well...

No, Imake pannels transparent editing gnome-panel.rc, in themes like equinox the name of the file is diferent, but its always inside the theme folder in gtk-2.0 or in gtk-2.0/apps, you have to look for a line...
```
bg_pixmap[NORMAL] = "images/panel_bg.png"
```
