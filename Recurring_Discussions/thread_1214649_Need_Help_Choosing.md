### Post by dangerjunkie2002 on 2009-07-16
Hi,

I'd recommend Ubuntu 9.04 (Jaunty) which will give you the standard Ubuntu desktop. I find it much better than 8.10 (Intrepid) on my machines. I've installed it on nearly a dozen machines and I've had no dramas getting it to work. The worst thing I've had to do was go to preferences and set the screen resolution on one machine.

If you've been a long time Windows user you may want to install the "kubuntu-desktop" package (really easy, just select it in the package manager and it all gets put on automagically). You can then try the KDE desktop when you log in to see if you like that better.

Cheers,
Paul.
