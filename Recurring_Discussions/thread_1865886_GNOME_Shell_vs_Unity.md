### Post by screaminj3sus on 2011-10-20
I like unity, but for me gnome-shell is all around better. It simply has the following advantages:

Better customization: Gnome-shell is much more themeable and customizable than unity's shell. Being written with javascript and css there are various themes and extensions available.

Faster. On my machine's anyway gnome-shell feels noticeably faster and more responsive. The effects seem smoother and more consistant. 

More polished. The look and feel seems more polished and coherent. Gnome-shell's empathy integration and notifications run circles around unity's indicators functionality wise.

### Post by 3rdalbum on 2011-10-21
Oh geez, this is horrible.

Horrible because I have to say that I like both very, very equally.

Gnome Shell has some nice features, and it looks great. It's super-responsive and neatly designed.

However, Unity's feature list suits me better. I love dragging and dropping files onto the exact program I want in the launcher. I also love the extra space I get on my monitor when using Unity; Gnome's almost-useless top bar plus an almost-useless (when maximised) window titlebar just makes me feel claustraphobic.

Oh, and I love launching programs from the keyboard.

I love both desktop environments. Gnome could implement the drag 'n' drop and add more space-saving features, and tweak their Alt-F2 so it can do searching of programs and files like Unity's, and I would never look back.
