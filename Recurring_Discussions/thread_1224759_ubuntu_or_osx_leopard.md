### Post by MikeTheC on 2009-07-27
Well, it depends on what you require for being a productive user of technology.

Myself, I have a 13" MBP running Leopard, and will soon run Snow Leopard once it comes out in September.

Linux is a fine OS -- and Ubuntu is my favorite distro -- but when it comes to doing desktop publishing, graphics work, and compatibility for school, Mac OS X is the better choice. Well, that plus I own a few iPods (main unit is an iPod Touch) and that effectively requires Mac OS X.

Now I know this question has been asked a zillion times, but why would you bother buying a Mac if you're not going to run Mac OS X on it? I totally understand if you plan to run Linux *in addition to* Mac OS X, but not *in lieu of* it.
