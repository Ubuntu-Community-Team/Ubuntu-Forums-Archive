### Post by doas777 on 2010-05-24
heres one example. 

I work for company X which uses an Linux server for somthing. We get a new peice of hardware for which there are no drivers, so we write one. but we don't want to have to redeploy it every kernel update, and and we're feeling altruistic, so we contribute the code the the kernel team, and they integrate it in. 

in this case, everyone gets paid, including me, and the folks that do the kernel (they work for the linux foundation, a non-profit).

keep in mind, most of the contributions to the kernel have been made by people of organizations that just provided that one patch.
