LXDE, XFCE, and KDE have a menu, if the only problem you have with Ubuntu is the Desktop environment there are Ubuntu versions that use these Desktop environments, they are called Lubuntu, Xubuntu and Kubuntu respectively.

You could install the packages called lubuntu-desktop or xubuntu-desktop or kubuntu-desktop on top of what you have but you would end with lots applications that duplicate the purpose, so trying them first from a Live CD/USB and doing a clean installation is a better idea. In my struggle to simply get app icons to appear on the panel while the app is running ([see post)]("http://ubuntuforums.org/showthread.php?t=1873519"), I stumbled upon ccsm (mentioned by dFlyer in 4th post), which makes it easy to set a reasonable launcher icon size.

sudo apt-get install compizconfig-settings-manager
ccsm
Navigate to Desktop | Ubuntu Unity Plugin | Experimental tab, and change 'Launcher icon size' to as low as 32 px.

Play with other settings as you like, but BEWARE! One change that may take effect is a complete disappearance of Unity! Wanna be alarmed? This'll do it.
