### Post by RetchingRabbit on 2009-03-17
Relax. You are safer using Ubuntu with nothing added than using Windows with a full compliment of security software. There are no known "viruses" or other exploits for Ubuntu at this time. While it may be theoretically possible to build an exploit, it will likely require a high degree of "social engineering" to make it work. 
If you are behind a router, you are as safe as anyone can be, and still be connected to the 'net.

Besides, most of the "anti-virus" software that will run on Linux only detects Windows exploits....so why bother?
