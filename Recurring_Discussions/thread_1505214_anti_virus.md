### Post by cariboo on 2010-06-08
With a default installation, you have nothing to worry about, there are no ports listening. The best way to check for open ports is to use a service like [URL="http://nmap-online.com/"]Nmap Online.
[/URL] 
As far as I know there are no Linux viruses in the wild. The biggest security problem is the user, he/she is susceptible to social engineering, downloads packages/programs from untrusted sources, and surfs unsafe web sites. 

The main thing you need for a secure system is common sense.
