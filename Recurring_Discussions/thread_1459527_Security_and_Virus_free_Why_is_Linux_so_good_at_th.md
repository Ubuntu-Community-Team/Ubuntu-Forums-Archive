### Post by aysiu on 2010-04-21
I think there are two tempting extremes you should be careful to avoid.

The one extreme is believing that Linux is technically in the same security situation as Windows except for being a smaller target and that if everyone was using Linux, there would be exactly the same degree of malware prevalence.

The other extreme is believing that just because Linux is better set up security-wise that it is invincible and that you don't need to employ good security common sense practices.

Worse yet, regardless of what extreme you lean toward, is believing that so-called "antivirus" is good protection against malware on Windows or Linux.

Computer security is a lot like house or apartment security. Is it ever possible to make a house or apartment *totally* invincible? No, not really. You could make it so that only a nuclear warhead or volcanic eruption, tornado, or comet collision could get into your house. But that's going a bit overboard, and your house wouldn't look very good or even be very easy for you to enter. You can get a thicker window that's harder to break, but some thief will find a way into the window if she's really determined. You can live in an area that has a lower crime rate. But even in those areas, crime still occurs. You can get a great alarm system, but if you yourself are stupid and leave the door open and the alarm system turned off, then anyone can just walk in.

Exactly the same deal with computers and operating systems.

Some security breaches are just because the person trying to break is so determined to break in she eventually just does. Some are just getting in easily through a vulnerability. Some take advantage of general bad setups. Some trick the user into installing them.

What you get with most Linux distros is a basically good security set-up. You can harden it more if you're paranoid (Google *AppArmor* and *NoScript* for more details), but it's probably fine for most people. It isn't a guarantee against ever being compromised, but living in a bubble is also not a guarantee you won't ever die.
