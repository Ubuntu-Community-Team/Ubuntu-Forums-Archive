### Post by mbgaski on 2010-08-09
As with most things "It depends."

Do you already have a Windows license?

If not, how much money are you looking to spend?  Ubuntu is free. Windows 7 is not.

Do you play a lot of computer games?  If so, Linux in general isn't going to be for you.

Do you have an iPod or significant amounts of iTunes DRM'd music?  If so, Linux isn't your optimal solution there.

Now, if you want to same some money, like an open OS, have a mostly non-DRM'd media collection, etc, then Ubuntu is a very, very good OS.  Particularly if you're into programming and development.  Things have really polished up in the last 18 months or so.

FWIW, my main desktop system is running Ubuntu 10.04.  I have slightly slower (though with a faster vid card) Windows XP system that I use exclusively for gaming.
