### Post by guiverc on 2019-11-14
If you're talking about UA Linux ([https://ualinux.com/en/ubuntu-gamepack](https://ualinux.com/en/ubuntu-gamepack)) then it's neither Ubuntu, nor an official flavor of Ubuntu. Ubuntu flavors can be found by going to [https://ubuntu.com/download/flavours](https://ubuntu.com/download/flavours)

If you want to get started on Ubuntu, or *official* flavors of Ubuntu - I'd suggest starting with a real flavor of Ubuntu instead of a distribution based on Ubuntu.

  I have no experience with UALinux, so I don't know how different it is, but please be aware you're not using Ubuntu, nor a 'flavor' of Ubuntu  (and if it's what I'm suggesting, it's UA Linux with the "*Ubuntu Gamepack*" added).
