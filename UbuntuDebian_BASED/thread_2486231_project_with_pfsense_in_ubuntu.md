### Post by TheFu on 2023-04-26
pfsense is BSD-based. It has nothing to do with Ubuntu/Linux other then it is also a Unix-like OS.

Visit the pfsense project and follow their instructions.  For play, running it inside a VM is fine.  It is not a good security practice to use a VM for a WAN router, however.  WAN routers need dedicated hardware.
