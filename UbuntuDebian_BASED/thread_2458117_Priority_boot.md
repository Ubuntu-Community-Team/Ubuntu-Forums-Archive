### Post by CelticWarrior on 2021-02-16
Welcome.

[B]No, it isn't possible and no Windows media installation does that either.
[/B]The boot process is always managed by the motherboard's firmware, BIOS (old) or UEFI (new). No bootable installation media of any OS or other software can override those BIOS/UEFI settings.

Now I suggest you ask your real question which is why your other than Windows media doesn't boot in the same computer where Windows media boots (BIOS/UEFI is set to to boot external media first). And the answer to that question is: The media isn't bootable or was incorrectly made and boots only in BIOS or in UEFI mode and you're trying to use it in the system where it doesn't boot.
