### Post by buzzingrobot on 2014-08-24
Burn an boot an Elementary ISO in live mode.

Elementary is known for its consistent adherence to its own design and interface standards, it uses its own window manager, its own file manager, as well as several applications created or adopted by the Elementary team. As such, it differs from many (most?) Ubuntu derivatives that simply add an existing interface created elsewhere.

The current release is based on Ubuntu 12.04 and is showing some age.  A new release based on 14.04 is currently in beta.
