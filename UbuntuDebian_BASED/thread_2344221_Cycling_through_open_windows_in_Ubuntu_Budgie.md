Hi,

  the odd alt-tab behaviour is a known issue upstream.  Its on the planned list during/or after the upcoming budgie desktop v11.

As a workaround we've introduced skippy-xd as a recommended install - if you fire-up budgie-welcome you can install in the recommendations page.

pressing CTRL+Tab gives a neat expose style view.  Its probably the best we can achieve until upstream development resolves the alt-tab issue.

Hope that helps
