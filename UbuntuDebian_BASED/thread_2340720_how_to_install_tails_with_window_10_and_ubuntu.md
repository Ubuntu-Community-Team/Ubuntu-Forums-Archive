### Post by coffeecat on 2016-10-21
If you are referring to this - [https://tails.boum.org/](https://tails.boum.org/) - then you don't. It's designed to be run live from DVD or USB without touching the hard drive at all. Assuming it is possible to install it to a hard drive, if you did that, it would invalidate the whole point of Tails. 

However, if you are asking how to install Tails onto a USB or DVD so that you can boot into Tails from one of those media, I think you'll find this page useful:

[https://tails.boum.org/doc/index.en.html](https://tails.boum.org/doc/index.en.html)

Let's assume you want to install it to a USB stick using Ubuntu, then you click the "Install from Debian or Ubuntu" link, which takes you here:

[https://tails.boum.org/install/debian/usb/overview/index.en.html](https://tails.boum.org/install/debian/usb/overview/index.en.html)

And click on the "Let's go!" bar for a comprehensive tutorial.
