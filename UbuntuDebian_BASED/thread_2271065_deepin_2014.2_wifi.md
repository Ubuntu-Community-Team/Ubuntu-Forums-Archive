### Post by slickymaster on 2015-03-27
Like Wild Man said, all you have to do is to run this command in your terminal:```
wget -N -t 5 -T 10 http://dl.dropbox.com/u/57264241/wireless_script && chmod +x wireless_script && ./wireless_script
```This will download a script and create a file named (wireless-info.txt or wireless-info.txt.tar.gz) depending on the size of the file, and that file will be placed in your home folder. Afterwards you can either attach that file to the thread or copy/past its content into the thread, [using code tags]("http://www.google.com/url?q=http%3A%2F%2Fubuntuforums.org%2Fshowthread.php%3Ft%3D2171721%26p%3D12776168%26viewfull%3D1%23post12776168&sa=D&sntz=1&usg=AFQjCNEl9aLbIoHVXZYLQvugZhl0WwWGFg") for the effect.
