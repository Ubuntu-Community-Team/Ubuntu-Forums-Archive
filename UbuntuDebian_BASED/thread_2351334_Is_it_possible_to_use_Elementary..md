### Post by ajgreeny on 2017-02-02
It looks as if it should run Elementary, but as it is a Windows-8.1 machine (perhaps Win 10 now) it is probably using UEFI instead of BIOS.
There is also no indication of the wifi adapter chipset so that might be a problem, unless you can find any info on the detail of the chipset used in that particular machine.

Read [https://help.ubuntu.com/community/UEFI](https://help.ubuntu.com/community/UEFI) and
[http://ubuntuforums.org/showthread.php?t=2147295](http://ubuntuforums.org/showthread.php?t=2147295) for some tips, if you are new to UEFI, as it could trip you up if you are not careful.
