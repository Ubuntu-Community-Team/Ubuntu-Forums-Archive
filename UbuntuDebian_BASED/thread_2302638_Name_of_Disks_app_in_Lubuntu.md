### Post by Bucky Ball on 2015-11-12
*Thread moved to **Ubuntu/Debian BASED**.*

If you are using the Ubuntu repositories on any distro and the app in question is in that repo it should be called the same thing. Look in Synaptic Package Manager for whatever app it is you're after. 

It is called gnome-disk-utility. You'll find it in Synaptics/Software Centre/install it using a terminal.

```
sudo apt-get install gnome-disk-utility
```
