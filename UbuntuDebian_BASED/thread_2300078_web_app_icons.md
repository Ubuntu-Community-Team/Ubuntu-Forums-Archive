### Post by mail-kariver on 2015-10-23
You are probably using shortcuts from Chrome. When Chrome does not have the icon in its cache, (i.e not a Chrome Webstore app) it generates an icon with the first letter of the site's title and the leading favicon color. Sadly, this also happens on Windows and all other OSes who handle Chrome/Chromium, even Chromebooks and Chromium OS.

For YouTube, the Chrome App that matched Youtube.com was available somehow and used. Please use Chrome Webstore rather than [COLOR=#252525][FONT=sans-serif]&#8801;>[/FONT][/COLOR]Tools>Create shortcut. For my part, I only have two icons causing problems, the two only icons I did not get from Webstore.
