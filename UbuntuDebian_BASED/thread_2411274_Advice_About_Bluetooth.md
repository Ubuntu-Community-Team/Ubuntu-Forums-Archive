### Post by TheFu on 2019-01-28
I don't know anything about Zorin, but 12.4 sounds really old and unsupported.

If you want to use bluetooth, hardware is required on both sides of the connection.  Many wifi cards include bluetooth, mainly in laptops. 

A "dongle" is short for a small USB attached device. I think BT is a peer-to-peer connection and that a compatible dongle connected to any linux computer would be both client and server capable. I've seen them for $5-$10.

You might want to seriously consider whether using bluetooth is a good idea at all.  There are well-known security problems with all versions.  I've been hacked though a bluetooth connection with a 100% patched Linux install.
