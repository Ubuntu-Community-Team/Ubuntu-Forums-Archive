So...first let me get 2 things out of the way.  1) I'm actually not looking to stick Ubuntu on this, but rather the non-GUI version of Kali Linux.  With the specs on this system, I have no intention of running a GUI, so listed "system requirements" do not apply..  (Depending on my options, I might just add the Kali repos to an Ubuntu Server install and call it a day.)  2) This question is about the Insignia Flex 11.6" tablet.  However, this system is basically a laptop with a detachable keyboard, not a tablet.  It's also very, very different on the inside from the 8" tablet with the same name.  Please note this difference, because I've asked this question elsewhere and got a response pertaining to the 8" tablet that I am about 98% sure isn't relevant.

First question: will this thing run Linux?  As in, at all?  Any distro?  I ask because it uses "secure boot" and apparently lacks any option to disable it.

Second question: if no, then is WUBI a potential workaround to this?  By which I mean, can a windows 10 system use WUBI and if so, will booting Ubuntu in this manner not load Windows?  I ask because this system has liminted memopry that cannot be upgraded and if WUBI runs simultaneously with Windows, that's a no-go.  (I used WUBI once a decade ago and honestly don't remember.)

Third question: the internal wifi card is a Broadcom model with 802.11n support.  Does the kernel in Kali, or if you don't know, the kernel in Ubuntu, currently support this out-of-the-box?  I would seriously like to avoid the third-party driver headaches I've had to deal with on Broadcom cards before.

Last question:  Would Linux support the detachable keyboard?  I never run the system without it attached and very, very seldom use the touchscreen (and even then only to scroll a web page, nothing more) so it is critical that the hardware keyboard remain functional.  It uses a very strange and probably very proprietary connector so I want to be certain it'll work.

### Post by Jon_Thomas on 2017-01-15
I know this is an older thread, but would thought I would respond since I just got this same tablet.

I have Ubuntu 16.10 installed and running off a USB-3 Sandisk Cruizer flash drive.  I currently have Windows 10 home on it, plus have Phoenix OS installled to the Windows partition.

Yes, it takes bit to load the Unity desktop, and yes, it is slow at times, but it was more an exercise to see if it would even work.  The WiFi works real well. If I can get the battery icon to work and get touch working, this may just get Ubuntu installed in place of Windows in the coming months. Also, the keyboard works fine for me.
