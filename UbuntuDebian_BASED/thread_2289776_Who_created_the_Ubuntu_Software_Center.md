### Post by coffeecat on 2015-08-07
For anyone reading this thread and experiencing difficulty with the Software Centre captcha:
[SIZE=3][b]
Ubuntu Software Centre captcha -  how to avoid.[/b][/SIZE]

If you see this window in Ubuntu Software Centre (click on the thumbnail to view image):

[ATTACH=CONFIG]263702[/ATTACH]

And if you already have an Ubuntu One account, simply click on the &#8220;Log-in with my existing account&#8221; link, and you will be taken to this window (click on the thumbnail to view image):

[ATTACH=CONFIG]263701[/ATTACH]

Where you will be able to log into your Ubuntu One account and continue to use whichever Software Centre service requires login. **If you are logged into the forum, you already have an Ubuntu One account and complaining about the captcha here is pointless. **

If you are a guest viewing this thread and do not have an Ubuntu One account, simply go to this link to create your Ubuntu One account without the inconvenience of a captcha:

[https://login.ubuntu.com/](https://login.ubuntu.com/)

You can then use the &#8220;Log-in with my exisiting account&#8221; link in the Software Centre.

**Footnote:** The volunteer forum staff agree that the captcha currently used in the Software Centre is near-impossible, if not completely impossible, and needs attention. Since we do not administer Ubuntu One, there is nothing we can do about it. Ranting about this issue on the forum is not helpful and is not welcome here.
