### Post by TheFu on 2015-10-27
Welcome to the forums.

Think this question belongs in a non-Ubuntu forum.

Don't think you can do any upgrade since chromexium is based on 14.04. There isn't any direct upgrade path for anyone from 14.04-->15.10. 

I wouldn't attempt to switch distros at all. You'll be left with unsupported, unpatched, packages. You could attempt to do the upgrades following the debian release method. I've never tried that on an ubuntu-based system.  [https://www.howtoforge.com/how-to-upgrade-debian-squeeze-to-wheezy](https://www.howtoforge.com/how-to-upgrade-debian-squeeze-to-wheezy) is an example method. Clearly all the sources will need to be ubuntu. This is very far off the "reservation"-**use at your own risk**.

You can wipe the other OS and install a fresh 15.10, probably. Be certain to review the release notes and check for any specific issues with the hardware.  I know there are complexities with my chromebook to run Ubuntu. 15.10 only has 9 months of support, so think carefully before going there. I'll be on 14.04 until June-ish - a few months after 16.04 is released. It will be a fresh install and I'll move my settings and data over manually.

If you have 15G storage free, could install 15.10 next-to the current OS.
