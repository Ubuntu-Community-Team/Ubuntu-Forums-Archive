### Post by coffeecat on 2016-11-10
> **ErmenegildoRos said:**
> The problem is that it doesn't recognize the user password after the installation process.


I have no experience with GM Linux, bit since it is based on Ubuntu my guess is that the installer is the same. If so, there are only two reasons I can think of:

1 - The ISO is corrupted. It's worth checking that.

2 - (More likely) - the password that you are using after installation is not what you think you set up in the installer. Make sure you haven't put caps lock on by mistake, or swap caps and lower case around in case you had the caps lock on when installing. The installer will warn you about that, but it's worth checking anyway. 

Failing all that, try resetting the password with this howto:

[http://www.psychocats.net/ubuntu/resetpassword](http://www.psychocats.net/ubuntu/resetpassword)
