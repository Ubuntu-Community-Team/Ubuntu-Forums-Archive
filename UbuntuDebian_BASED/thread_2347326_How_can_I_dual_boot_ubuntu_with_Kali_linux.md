### Post by yancek on 2016-12-23
Dual booting different distributions involves the same process whether you are using different Linux systems or Linux and another OS.  You need to create unallocated space if you have none currently and during the installation of Kali, you would need to select that space for your partitions.  If you have Ubuntu installed using UEFI, you will need to install Kali UEFI also.  Since Kali is a very poor choice as a computer Desktop system, I suppose you are interested in learning computer forensics.  No other reason to install/use it.  If that's the case, you would probably be better off creating a bootable flash drive with Kali on it as explained at their site below which would eliminate corrupting your current system.

   [http://docs.kali.org/downloading/kali-linux-live-usb-install](http://docs.kali.org/downloading/kali-linux-live-usb-install)
