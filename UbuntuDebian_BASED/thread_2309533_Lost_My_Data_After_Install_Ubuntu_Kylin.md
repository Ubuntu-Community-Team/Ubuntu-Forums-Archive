Welcome. Please note: ***General Help*** provides support for the recognised Ubuntu flavours: Ubuntu, Kubuntu, Xubuntu, Edubuntu, Lubuntu, UbuntuGnome, Ubuntu Studio, Mythbuntu and Ubuntu Mate.

Why do you think your data is lost? The error message "Filesystem read-only" could be telling you, as you tried to wipe Windows, that the operation couldn't be performed as Windows is hibernating. This could have damaged the Windows boot loader only and your data would still be where it was. 

When you try to boot the install CD, what 'panic' do you get? Please be specific about the message. 

How did you create the CD? Where did you download Ubuntu from and which release? What were you wanting to do? Wipe the drive and install Ubuntu? It sounds like you had valuable data on the drive. Did you do a backup prior to diving in to this?
