### Post by ansari.ibrahim1 on 2016-12-10
Katoolin is used to install Kali Linux tools, so I don't think it's causing the issue.. Kali isn't a very user-friendly Linux distribution as it's mainly used for ethical hacking and other sorts of penetration testing which is why I'd recommend you stick to Ubuntu. If you're getting the PXE mof, that means you haven't flashed the ISO properly. Try flashing an Ubuntu ISO using these instructions:
[https://www.ubuntu.com/download/desktop/create-a-usb-stick-on-ubuntu](https://www.ubuntu.com/download/desktop/create-a-usb-stick-on-ubuntu)
