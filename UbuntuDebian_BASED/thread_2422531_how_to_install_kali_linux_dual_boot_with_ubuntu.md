@pratap73, I notice that you have posted all your threads so far in the New to Ubuntu sub-forum which is "The perfect place to post for your Ubuntu support if you are new to Linux." (The sub-forum strapline.) If you are new to Linux, you might want to reconsider installing Kali, which can be a challenge even for experienced Linux users. 

From the [Kali Linux documentation](https://docs.kali.org/introduction/should-i-use-kali-linux):

> 
**Is Kali Linux Right For You?**

As the distribution’s developers, you might expect us to recommend that everyone should be using Kali Linux. The fact of the matter is, however, that Kali is a Linux distribution specifically geared towards professional penetration testers and security specialists, and given its unique nature, it is **NOT** a recommended distribution if you’re unfamiliar with Linux or are looking for a general-purpose Linux desktop distribution for development, web design, gaming, etc.
