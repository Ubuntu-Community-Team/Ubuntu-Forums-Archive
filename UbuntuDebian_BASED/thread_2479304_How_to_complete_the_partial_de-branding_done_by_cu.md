### Post by Frogs Hair on 2022-09-20
I don't know where the Ubiquity installer stores the images , but they are run from scripts created for each Ubuntu flavor , I suspect these would have to understood and altered for the new distro. When people were really interested in creating Ubuntu based distros some years ago, two tools for this were remastersys and Pinguy Builder. They maybe current forks of these projects. [ Pinguy Builder]("https://sourceforge.net/projects/pinguy-os/files/ISO_Builder/") seems to be missing dependencies on current Ubuntu builds and remastersys is no longer maintained. Good Luck !
