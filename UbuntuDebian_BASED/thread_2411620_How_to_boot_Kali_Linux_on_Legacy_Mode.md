### Post by yancek on 2019-02-01
Booting in UEFI mode is dependent upon your hardware.  When you boot the computer, look in the BIOS setup for any reference to UEFI.  IF there is none then you can't use UEFI.
Best  resource for using Kali is their web site, very well documented at the link below.  I hope you do not plan to use Kali as a basic Desktop system as it is not designed for that, also explained at the link below.  Do you currently have any operating system installed?  If so, what is it?
