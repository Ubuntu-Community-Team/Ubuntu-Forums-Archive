### Post by guiverc on 2021-05-29
Ubuntu 14.04 LTS reached it's end of 5 years of *standard support* life in 2019-April (see [https://fridge.ubuntu.com/2019/05/02/ubuntu-14-04-trusty-tahr-reached-end-of-life-on-april-25-2019-esm-available/](https://fridge.ubuntu.com/2019/05/02/ubuntu-14-04-trusty-tahr-reached-end-of-life-on-april-25-2019-esm-available/)).

Lubuntu 14.04 LTS [being a *flavor* had only 3 years of support]("https://fridge.ubuntu.com/2016/02/18/ubuntu-14-04-4-lts-released/"), thus reached it's EOL in 2017-April 

I'd suggest moving to a supported release of Lubuntu (esp. if using *amd64* or an architecture that is supported), otherwise a partially supported release like 18.04 (which still gets updates for some packages).

No GUI component of Lubuntu 14.04 gets ESM security updates, so if you continue using 14.04 ESM I'd suggest switching desktops.
