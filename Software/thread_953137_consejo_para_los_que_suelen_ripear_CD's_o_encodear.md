### Post by Kantier on 2008-10-19
si usan hardy, bájense el paquete de LAME para intrepid ([http://packages.ubuntu.com/intrepid/lame](http://packages.ubuntu.com/intrepid/lame) )

¿por qué? simple, la versión para hardy es la 3.97 mientras que la versión para intrepid es la 3.98

la diferencia más importante es que la v3.98 usa un algoritmo nuevo que provee una mejor relación bitrate/calidad , en la 3.97 hay que activar el algoritmo a mano y creo que ningún programa gráfico toma en cuenta eso.

fuente: [http://wiki.hydrogenaudio.org/index.php?title=Lame](http://wiki.hydrogenaudio.org/index.php?title=Lame)
> The --vbr-new switch enables the new VBR mode (no longer needed with LAME 3.98, as it is now the default VBR mode). In terms of quality, --vbr-new appears to be better than the old model, but reports of artifacts when using the new model do exist. Despite these possible issues, --vbr-new is currently recommended over the default VBR mode due to both the speed and quality increases afforded by the new algorithm.
