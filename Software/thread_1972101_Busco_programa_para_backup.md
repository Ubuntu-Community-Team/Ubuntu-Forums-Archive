### Post by biale on 2012-05-05
Para resguardar archivos de usuario cualquier compresor te sirve, solo evitaría el formato "rar". O incluso un simple copiar y pegar...

Pero no es lo mismo una partición NTFS completa sobre la que Windows luego pueda volver a arrancar. Podrías terminar con una partición 100% ok pero totalmente inutil para hacer arrancar a Win. 

Te sugiero estudiar los procedimientos de recuperación de Windows y sus limitaciones. Y estos difieren según las versiones (WXP, WVista-W7, etc)
