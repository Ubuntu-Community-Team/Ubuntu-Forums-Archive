### Post by AlexDDR on 2009-08-30
mi solucion fué

desinstalar el paquete firefox-3.5 (el que te pone el feo nombre shiretoko)

instalar el paquete abrowser-3.5 ( que instala una versionde firefox sin marca, pero que permite tener instalado el firefox 3.0 sin conflictos de directorio con la version 3.5)

instalar el paquete "totem-mozilla" con ese puedes escuchar la fmdos por ejemplo desde la pagina, ya que es totem el que gestiona el fujo de WMA con que funcionan muchas de esas paginas

que tambien se podria con el siguiente comando port ejemplo, y tambien crear un acceso directo para nisiquiera tener que abrir el navegador para escuchar la radio on line

totem "mmsh://streaming.iberoamerican.cl/fmdos?MSWMExt=.asf"


saludos
