Si la maquina esta aun en garantia, para no perderla la recomendacion es dejar la particion para instalacion intacta porque, en general, si verifican que no tiene el SO original rechazan el reclamo y hacen caer el compromiso de su parte.

Si el tema garantia no juega y no se quiere volver atras con el SO preinstalado, se puede borrar tranquilamente esa particion y llegado el caso de querer volver a usar ese otro SO se lo instala desde una imagen ISO (que uno procura por su cuenta, ya sea copiando el contenido de la particion a borrar o de otro origen).
