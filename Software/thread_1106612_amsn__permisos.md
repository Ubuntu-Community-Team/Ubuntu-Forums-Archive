### Post by josecuervo86 on 2009-03-25
Para obtener perimisos anda a la terminal, y pone:
```
sudo nautilus
```

Eso te va a abrir un explorador de archivos, pero con permisos root y ahi podes modificar lo que se te antoje (con cuidado!)
