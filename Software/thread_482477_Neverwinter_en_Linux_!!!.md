### Post by jpmorelli on 2007-06-27
> **sdm_cacto said:**
> Eso.. como hicste?? posteate unos screens... alto game

Super fácil, podés usar la instalación que tenés desde Windows para ahorrarte bajar 1.1Gb de resources que son los mismos que están en WIndows ( yo los bajé igualmente porque quería arrancar con todo nuevito, aunque obvio te podés traer los saved games que tenías.
La página del tutorial es esta:

[http://nwn.bioware.com/downloads/linuxclient.html]("http://nwn.bioware.com/downloads/linuxclient.html")

Crearse una cuenta ( es gratis y no mandan nada de spam )
Después bajás el cliente en español ( o el idioma que prefieras )

[http://files.bioware.com/neverwinternights/updates/linux/nwspanish129.tar.gz]("http://files.bioware.com/neverwinternights/updates/linux/nwspanish129.tar.gz")

y te bajás también los binarios:

[http://files.bioware.com/neverwinternights/updates/linux/nwclient129.tar.gz]("http://files.bioware.com/neverwinternights/updates/linux/nwclient129.tar.gz")

Según si bajaste los resources o copiaste desde Windows extraes todo en el directorio donde quieras el juego.
ültimo paso, ya termina... bajás el update a la última versión cliente:
[http://nwn.bioware.com/support/patch.html]("http://nwn.bioware.com/support/patch.html")
(Según la versión que instalaste ( español, alemán, inglés, etc ) y lo descomprimís sobre lo anterior sobreescribiendo los archivos que te diga que ya están. En este momento la última actualización del cliente es a la 1.68 en español

Vás a la carpeta donde instalaste y corrés el archivo ./fixinstall y ./nwn  y listo !!!

Si tenés problemas con el sonido (como yo los tuve, que se entrecorta) hacés:

cd ~/nwn/lib
mv libSDL-1.2.so.0 libSDL-1.2.so.0.bak ( no lo borré por si acaso y le cambié el nombre por .bak )

Te va a pedir la clave del juego y luego aranca todo y anda per-fec-to !!! :p

Suerte !

PD: Igualmente si no se entendió en el primer link que dejé explica todo lo mismo partiendo desde una instalación de Windows o desde cero, lo loco es que partiendo de cero ni siquiera hace falta tener los CD de NeverWinterNights. :popcorn:
