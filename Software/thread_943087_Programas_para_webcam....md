### Post by guisheca on 2008-10-10
Y mirá, te tiro el único que conozco (supongo que habrán muchos mas) y que uso, se llama "cheese"

```
sudo aptitude install cheese
```

Después lo abrís desde Aplicaciones>Gráficos>Chesse

Es simple pero sirve para sacar fotos y grabar videos, además tiene la opción de agregarle algunos efectos a las fotos que saca.
Espero te sirva, saludos.
