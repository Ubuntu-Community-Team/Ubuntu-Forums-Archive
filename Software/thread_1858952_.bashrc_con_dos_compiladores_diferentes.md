### Post by thegreeneyes on 2011-10-13
Y bueno, por si a alguien le interesa más posibilidades respecto a tener diferentes versiones de un mismo código instalado y accesibles:

[http://modules.sourceforge.net/](http://modules.sourceforge.net/)

por ejemplo, esto permitiría:


Here is an example of loading a module on a Linux machine under bash. 
                 % module load gcc/3.1.1
                 % which gcc
                 /usr/local/gcc/3.1.1/linux/bin/gcc
Now we'll switch to a different version of the module 
                 % module switch gcc gcc/3.2.0
                 % which gcc
                 /usr/local/gcc/3.2.0/linux/bin/gcc
And now we'll unload the module altogether 
                % module unload gcc
                % which gcc
                gcc not found
