### Post by gmunioz on 2009-07-17
Se recomienda desactivar JavaScript en Firefox 3.5

Se ha descubierto un exploit, problema de seguridad, para Mozila Firefox 3.5 

Para el que no existe hasta ahora, parche oficial disponible

El mismo permite la ejecución de código, si un usuario visita una página web especialmente manipulada. 

Esto puede ser utilizado por atacantes para ejecutar código arbitrario con los privilegios bajo los que 

se ejecuta el navegador

Si bien este fallo de seguridad ha sido publicado con detalles para ser usado en entornos Windows,

con modificaciones puede ser usado para ejecutarse código en cualquier otra plataforma. 

El fallo en concreto esta en el procesador de código JavaScript del navegador.

Publicado en:
[http://www.hispasec.com/unaaldia/3917](http://www.hispasec.com/unaaldia/3917)
