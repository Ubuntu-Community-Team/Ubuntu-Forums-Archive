Pero otra buena noticia que encontré es que se puede instalar JDownloader desde un repositorio:
Pegar esto en la terminal:
sudo add-apt-repository ppa:jd-team/jdownloader
Y luego:
sudo apt-get update && sudo apt-get install jdownloader
Súper fácil.
