### Post by aledruetta on 2009-07-24
creo que tenés que instalar algunos paquetes:
```
sudo aptitude install ubuntu-restricted-extras libdvdcss2
```
y quizá alguno más, pero podés probar con esos dos.

### Post by aledruetta on 2009-07-24
Uno es ubuntu-restricted-extras y el otro es la librería libdvdcss2.
Podés instalarlos juntos o separados.

Ahora, si no tenés instalados los repositorios de Medibuntu no vas a encontrar esa librería en los repositorios oficiales de Ubuntu.

En caso de tener que instalarlos:

[https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)
