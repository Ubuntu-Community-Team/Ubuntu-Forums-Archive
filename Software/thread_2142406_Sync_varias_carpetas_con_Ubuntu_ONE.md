### Post by guillermolisi on 2013-05-05
Una posible solucion de compromiso es que sincronices con UbuntuOne desde Ubuntu para /home/<tu_usuario> y desde Windows para "Mis documentos".

No es muy practico pero funciona.

Otra posible alternativa, que no verifique si es efectiva, seria montar el recurso compartido de Mis Documentos en un directorio dentro de tu /home/<usuario>. Tambien podria reemplazarse el montaje por un enlace simbolico (simil Acceso Directo de Windows),
