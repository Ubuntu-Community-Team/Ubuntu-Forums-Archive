### Post by por100pre1 on 2007-11-22
GCompris es educativo:

```
sudo aptitude install gcompris gcompris-sound-es
```

Planet Penguin Racer, un pingüino que se desliza por la nieve a gran velocidad:

```
sudo aptitude install planetpenguin-racer planetpenguin-racer-extras
```
