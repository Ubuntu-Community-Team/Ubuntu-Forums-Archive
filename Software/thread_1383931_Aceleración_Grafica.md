### Post by guillermolisi on 2010-01-17
Con chips de video VIA no podras lograr, por ahora y desde hace un buen tiempo, aceleracion grafica. El problema es que VIA desarrollo drivers para Linux que no funcionan bien y hace años que estan ahi y nada hicieron para solucionar el tema. Tampoco alguien se tomo el trabajo de lograr drivers funcionales porque VIA no abre su tecnologia lo suficiente como para encarar tal proyecto con algun exito. Hacer ingenieria inversa es un tremendo trabajo.
 
Recomendaciones por el momento: Si podes, cambia la placa por una ATi o nVidia. Si no podes, resignate a usar driver VESA sin aceleracion.
