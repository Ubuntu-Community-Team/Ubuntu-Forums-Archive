tienes que bajar la version para linux ;D , y descomprimirla a una carpeta cualquiera, y despues con el boton derecho  sobre un archivo que se llama gridwars (ese es el ejecutable, si fuese windows seria .EXE) ir a propiedades y verificar en la pestaña permisos que este marcado "permitir ejecutar como un programa", despues cierras eso y al mismo archivo le das doble click y listo

despues de eso puedes crear un lanzador en el escritorio o ejecutar directamente desde el navegador de archivos nautilus


SALUDOS

PD: No olvides apretar F3 durante el juego para cambiar el "grid"
