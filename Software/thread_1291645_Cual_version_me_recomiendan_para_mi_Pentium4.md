### Post by ramiro_md on 2009-10-15
Mirá si vas a hacer lo que te sugerí te dejo un link al manual de instalación de Debian Lenny (la actual versión estable): [http://man-es.debianchile.org/ ]("http://man-es.debianchile.org/"). Para instalar un gnome o un xfce "básico" tendrías que hacer lo siguiente a grandes rasgos:
1) **INSTALAR SOLO SISTEMA BASE DE DEBIAN.**
2) Instalar el sistema de X (x-windows-system-core creo que ese es el paquete).
3) Instalar gnome-core o xfce-core.

Así tendrías un sistema base con un entorno de escritorio básico para poder aprovechar de una mejor manera tus recursos de hardware. Obviamente luego vas a poder agregar programas para complementar tu sistema.

Un saludo y suerte.
