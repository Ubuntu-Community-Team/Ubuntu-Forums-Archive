Hola,

al aprender a compilar se me empezaron a quedar paquetes, y quiero compartirlos; la mejor forma es subirlo o crear un repositorio.


Bueno opte por lo ultimo

editamos nuestro source.list
```

sudo gedit /etc/apt/sources.list
```



> # mamalibre
deb [http://mamalibre-repo.orgfree.com/](http://mamalibre-repo.orgfree.com/) karmic/i386/


# mamalibre
deb [http://mamalibre-repo.orgfree.com/](http://mamalibre-repo.orgfree.com/) lucid i386


agregamos al final, la versión que tengamos y cerramos.
luego 

```
sudo apt-get update
```

hasta ahora solo tengo dos paquetes, iré buscando si tengo mas y los agregare.

plugins ares - para clientes como giftoxic
