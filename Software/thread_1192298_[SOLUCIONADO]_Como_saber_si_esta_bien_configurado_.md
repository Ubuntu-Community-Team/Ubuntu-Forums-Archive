El xorg.conf de Jaunty es más simple ya que la ultima versión del servidor X reconoce todo automáticamente.

Si quieres ver si queda bien, entonces tienes que revisar el archivo /var/log/Xorg.0.log y ver que opciones se activan, como el compozite para que funcione compiz.
