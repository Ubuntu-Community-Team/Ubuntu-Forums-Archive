### Post by matuu! on 2009-06-18
Para grabación multipista te recomiendo Ardour, es excelente. Para utilizarlo tenés que utilizar JACK, por lo que deberías instalar también qjackctl, para administrarlo en forma visual.
Para utilizarlo semi-profesionalmente tendrías que correr el kernel realtime, pero para empezar podés hacerlo con un kernel que no sea de baja latencia (uno común, para decirlo de alguna manera). Si queres probar alguna distribución que posea este kernel, probá ubuntustudio o Musix (la versión 2 está basada en Debian)
Si te interesa, hay un tuto en español de ardour en este [link]("http://www.musix.org.ar/wiki/index.php/Audio-Edicion-Es").
Espero te sirva.
Un abrazo.
