### Post by Hei Ku on 2009-04-05
Instalaste el paquete unrar o unrar-free?

rar no es un formato linux, asi que no viene por defecto. De hecho, ningun SO lo trae por defecto. Siempre instalas algun programa para manejarlo, y linux no es la excepcion.

### Post by nemodot on 2009-04-06
A solo un click de distancia:

[apt://rar]("apt://rar") Soporte para crear archivos rar

[apt://unrar]("apt://unrar") Soporte para descomprimir archivos rar
