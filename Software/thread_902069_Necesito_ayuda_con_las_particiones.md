BIENVENIDO!! :)
La forma mas simple de cambiar las particiones es desde el CD. Arrancas Ubuntu desde el CD. Vas a **Sistema>Administracion>Editor de Particiones**.
Si tenes Ubuntu ya instalado, primero hay que desactivar el SWAP. Hace click derecho en la particion **linux-swap** y elegi **Desactivar Intercambio**.
Despues podes borrar alguna de las particiones de Ubuntu o Ubuntu Studio y cambiar el tamaño de la otra, o borrar las 2 e instalar todo de cero. Si dejas una particion sin formato, cuando instales podes seleccionar una opcion que dice algo asi como **Usar el mayor espacio libre disponible**, para no tener que particionar de nuevo.
SUERTE!
