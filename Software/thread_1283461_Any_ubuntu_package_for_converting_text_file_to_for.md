### Post by Calmor on 2009-10-05
As pablo.s said, OpenOffice will create an HTML file.  (File, Save As, then choose HTML as a file type.)

It won't create the prettiest looking HTML code behind, but it will look ok in most browsers if the text is simple.

I recommend testing it out in your target browser/software before distributing it, though.

Otherwise, you could use a text to HTML converter, available online for free, without download.
