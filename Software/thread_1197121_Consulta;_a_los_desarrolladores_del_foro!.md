Este foro esta echo con vbulletin, un soft pago para foros, tenes alternativas libres y gratuitas al mismo como phpBB tambien. Pero en todo caso si lo que queres es hacer tu propio sitio y no gastar nada en un hosting, usando ubuntu solo necesitas instalar los paquetes:

Apache2
MySQL
PHP5
PhpMyAdmin

Con eso tendrias lo basico, y con solo poner tu sitio web en /var/www ya tenes un sitio web andando(para ingresar usas tu IP en el navegador).

En fin hay muchos tutoriales para hacer un webserver, y no siempre es necesario saber programacion para poder tener tu propio sitio web, con un poco de maña ya esta todo, asi que si queres tener tu foro como este podes usar vBulletin, o si no queres gastar un peso, tenes phpBB y muchos mas.

### Post by guillermolisi on 2009-06-25
Bueno, ya que hubo una respuesta por el lado de la plataforma, voy a hacer uno por el lado de las herramientas (que tal vez es parte de las respuestas que esperas).

Te voy a mencionar las herramientas que uso actualmente, ya sea para HTML puro o combinado con Javascript y PHP:

**KompoZer**
Para que te ubiques, es como el Dreamweaver pero con muchisimos bugs que a veces exasperan cuando lo usas.

**Quanta Plus**
Excelente herramienta, sobre todo si vas a usar PHP

**Blue Fish Editor**
Otra excelente herramienta para PHP

Para imagenes tenes Gimp.
