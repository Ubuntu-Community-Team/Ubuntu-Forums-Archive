### Post by pablo.s on 2009-09-18
> **Vero1 said:**
> El problema que tuve con los datos se debió a que grabé mal yo el CD y ya que estamos, se debe grabar como Datos o como .iso?

La carpeta se llama .evolution
(en fonética: punto evolution)
porque se guarda en tu /home
como datos de usuario.
El origen de que tenga un punto
antes del nombre viene de la
función que tiene colocar un
punto anteponiendo el nombre
del archivo o directorio para hacerlo
oculto. Esto implica que si ocultás
un directorio, la manipulación
accidental por parte del usuario 
se hace muy improbable (aunque
no imposible)
El formato .iso viene de la terminación
de un formato de imágen de disco
que sigue ciertas normas internacionales.
Por lo general se usa como:
```
backup.iso
```
siendo .iso la nomenclatura que
identifica el tipo de archivo.
