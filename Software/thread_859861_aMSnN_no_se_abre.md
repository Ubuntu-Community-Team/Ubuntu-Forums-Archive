### Post by marianom on 2008-07-15
Primer paso para debuggear es correrlo desde consola a ver si tira un error más informativo.
Sería:
1- Abri una consola (Aplicaciones-Accesorios-Terminal)
2- ejecuta el programa (generalmente con escribir el nombre del programa y darle enter, estaría)

Espera a ver si sale algún error y contanos aca cual es.
