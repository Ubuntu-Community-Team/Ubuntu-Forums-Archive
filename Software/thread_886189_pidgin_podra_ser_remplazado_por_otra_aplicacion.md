### Post by ubuntu27 on 2008-08-10
Hola a todos,

Empathy[1] va a ser parte de GNOME 2.24.
El ubuntu desktop team considera usarlo en vez de pidgin para Intrepid como el cliente mensajero predeterminado. Si estas usando intrepid, por favor prueba empathy y reporta errores a launchpad [2]. Empathy puede ser instalado corriendo synaptic y intalando el paquete empathy o usando el siguiente comando "sudo apt-get install empathy".

Si experimentas un error o fallo, mire a [3] antes de reportar

[1] [http://live.gnome.org/Empathy](http://live.gnome.org/Empathy)

[2] [https://bugs.launchpad.net/ubuntu/+source/empathy](https://bugs.launchpad.net/ubuntu/+source/empathy)

[3] [http://live.gnome.org/Empathy/Debugging](http://live.gnome.org/Empathy/Debugging)


Encontre esa informacion en este thread:

[http://ubuntuforums.org/showthread.php?t=885548](http://ubuntuforums.org/showthread.php?t=885548)
