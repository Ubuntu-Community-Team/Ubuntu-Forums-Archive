### Post by asterix77 on 2010-06-14
Move Media Player es software propietario diseñado únicamente para  windows y mac. De hecho **Move Networks **ha recibido  financiamiento de Microsoft, por lo que dudo que diseñen un reproductor  para gnu-linux. Lo que se me ocurre es que instales firefox para windows  con wine y le agregues los plugins de flash, quizá también de  silverligth y por último instales el reproductor de Move Networks. U  otra, si tienes una máquina potente, instalar una máquina virtual de  windows con virtualbox y lo hagas funcionar desde allí.

Revisa el siguiente post a ver si te ayuda.

[http://ubuntuforums.org/showthread.php?t=1375252](http://ubuntuforums.org/showthread.php?t=1375252)

Saludos
