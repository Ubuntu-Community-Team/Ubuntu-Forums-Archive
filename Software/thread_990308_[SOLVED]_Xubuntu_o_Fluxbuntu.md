### Post by guillermolisi on 2008-11-22
En la wiki de LXDE ([http://wiki.lxde.org/en/Ubuntu](http://wiki.lxde.org/en/Ubuntu)) encontre esto que puede serte util y confirma que podes usarlo en forma alternativa al XFCE o cualquier otro desktop grafico sin tener que desinstalar:
```
**Install from APT repository**

 
[LIST]
[*]
[LIST]
[*] Special note:
[/LIST]
 
[/LIST]
 LXDE desktop has been added to Ubuntu intredpid repositories. No editing of sources are needed. Simply install the cli-only from the server, alt, or net installer, then sudo apt-get install lxde will install with GDM 

We provide packages for LXDE in our own [Launchpad PPA]("https://launchpad.net/%7Elxde/+archive") repository maintained by [michael-r]("https://launchpad.net/%7Emichael-r") and [paulliu]("https://launchpad.net/%7Epaulliu"). Please add those lines to your /etc/apt/sources.list, or using Synaptic, and add this repository. 
  deb [http://ppa.launchpad.net/lxde/ubuntu](http://ppa.launchpad.net/lxde/ubuntu) hardy main
 deb-src [http://ppa.launchpad.net/lxde/ubuntu](http://ppa.launchpad.net/lxde/ubuntu) hardy main  
Packages for Ubuntu Gutsy are also provided. Just replace "hardy" with "gutsy". For Intrepid try using the hardy repositories. 
After adding our repository, execute the following commands. 
 sudo apt-get update
sudo apt-get install lxde
Done! 
 **[[edit]("http://wiki.lxde.org/en/index.php?title=Ubuntu&action=edit&section=3")] Configuration for Login Managers**

 **[[edit]("http://wiki.lxde.org/en/index.php?title=Ubuntu&action=edit&section=4")] GDM or KDM**

 No manual configuration is needed. Just select LXDE from the available sessions listed by the display manager. If you don't see LXDE, restart your gdm or kdm, or reboot. 
 **[[edit]("http://wiki.lxde.org/en/index.php?title=Ubuntu&action=edit&section=5")] **


```
