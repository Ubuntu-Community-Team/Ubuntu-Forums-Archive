Si el uso que le vas a dar al editor de audio no es muy exigente, digamos, no es profesional, audacity va bien y se puede instalar desde Synaptic o con ```
sudo apt-get install audacity
``` o ```
sudo aptitude install audacity
```

Ardour seguramente es lo mejor, pero para mí era demasiado sofisticado. Audacity es bien simple.
