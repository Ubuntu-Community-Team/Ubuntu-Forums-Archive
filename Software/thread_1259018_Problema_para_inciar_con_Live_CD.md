### Post by gmunioz on 2009-09-05
Hola gui...:

1- Wubi, es una aplicación que corre desde MS Windows, 

insertando el cd en una sesión del mismo y tiene la

misión de instalar Ubuntu, como sistema operativo

distinto de MS Windows, pero dentro de este y sobre

su sistema de archivos, perdiendo muchas de sus 

funcionalidades.

2- Si quieres iniciar desde el cd, una sesión live, 

debes ingresar al setup de la bios de tu ordenador,

según la bios puede ser presionando repetidas veces

al comienzo del booteo escape - suprimir - F10 y 

configurar como primer dispositivo de booteo a la

unidad de cd/dvd, guardar los cambios, insertar el cd

e iniciar desde el.
