### Post by thegreeneyes on 2011-10-13
jejeje, bueno, yo me respondo...

encontré que abriendo una consola, puedo definir varias configuraciones (Edit -> Profile o Edit -> Profile Preference). Allí puedo definir el background y los colores de las letras para cada profile, de modo que lo que haré será abrir tabs con diferente profile para aquellas terminales que quiera usar comunicándome de manera remota al servidor.

Todavía no sé cómo abrir las aplicaciones del otro servidor de modo que no se confundan con las de mi pc, pero ... veré.
