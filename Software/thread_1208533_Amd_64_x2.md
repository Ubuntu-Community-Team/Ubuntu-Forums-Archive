### Post by guillermolisi on 2009-07-09
He leido [en este thread]("http://ubuntuforums.org/showthread.php?t=987244&highlight=concurrency+booting"), en Ingles, que a varios les ha funcionado pero a muchos no. Es decir, hay quienes han observado ganancia de un par de segundos a varios y otros que no han logrado cambio alguno.

Con HH y versiones anteriores no funciona y hasta es peligroso usarlo ya que hay cierto tema no resuelto con HAL que podria convertir el sistema en inutilizable.

La opcion a usar efectivamente es "shell" y no es recomendable para servidores.

Para poder contar con registros de las pruebas hay un utilitario que se puede usar llamado "bootchart"

```
sudo apt-get install bootchart
```

Luego de iniciar pueden ver los resultados en /var/log/bootchart
