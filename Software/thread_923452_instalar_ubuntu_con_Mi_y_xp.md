### Post by N4zgu1 on 2008-09-18
Tienes dos opciones:

Wubi: Instalar ubuntu dentro de Winows sin hace particiones ni nada
(esto no es recomendable, pero te sirve para probar ubuntu)

Dual boot: Puedes
1) Redimensionar la partición de Windows y hacer espacio para ubuntu o
2) Formatear windows y instalar ubuntu y después windows

Puedes hacer todo esto desde el livecd con "gparted"
