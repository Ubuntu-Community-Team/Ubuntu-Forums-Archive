### Post by sajnox on 2009-04-26
Asegurate de tener instalados los paquetes:

compiz-fusion-plugins-main
compiz-fusion-plugins-extra

Como saberlo? Hacelo por la via facil:
```

sudo apt-get install compiz-fusion-plugins-extra compiz-fusion-plugins-main
```

Y luego fijate si andan los efectos que te faltan
