### Post by nopersona on 2009-06-06
Eso es problema del tema de iconos que estás usando. Busca la aplicación en el tema

```
/home/tu_usuario/.icons/nombre_del_tema
```

y busca la carpeta apps. Luego busca otro icono agradable y los reemplazas. No te olvides de revisar los distintos tamaños, 32x32 64x54, etc.
