### Post by BcRich on 2011-03-23
Hi 
is this what you are looking for? This software comes with ubuntu studio...
**Video**


[LIST]
[*][PiTiVi]("http://en.wikipedia.org/wiki/PiTiVi") - a video editing program (not included in latest release)
[*][Kino]("http://en.wikipedia.org/wiki/Kino_%28software%29") - a non-linear digital video editor
[*]Stopmotion - a stop-motion animation movie creator
[*][VLC media player]("http://en.wikipedia.org/wiki/VLC_media_player") - a media player
[*][Xjadeo]("http://en.wikipedia.org/w/index.php?title=Xjadeo&action=edit&redlink=1") - a simple video player that gets sync from JACK transport
[/LIST]
 ** Graphics**

 
[LIST]
[*]Agave - a color scheme generator
[*][Blender]("http://en.wikipedia.org/wiki/Blender_%28software%29") - a 3D animation program
[*][Enblend]("http://en.wikipedia.org/w/index.php?title=Enblend&action=edit&redlink=1") - an image compositing program
[*][FontForge]("http://en.wikipedia.org/wiki/FontForge") - a typeface (font) editor program
[*][GIMP]("http://en.wikipedia.org/wiki/GIMP") - a raster graphics editor
[*][Hugin]("http://en.wikipedia.org/wiki/Hugin_%28software%29") - photo stitching and HDR merging program
[*][Inkscape]("http://en.wikipedia.org/wiki/Inkscape") - a vector graphics editor
[*][Scribus]("http://en.wikipedia.org/wiki/Scribus") - a desktop publishing application
[*][Synfig]("http://en.wikipedia.org/wiki/Synfig") - a 2D vector graphics and timeline-based animation program
[/LIST]
