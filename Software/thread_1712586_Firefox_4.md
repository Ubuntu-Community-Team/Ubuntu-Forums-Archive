### Post by EnriqueK on 2011-03-23
Cada vez que hay un cambio de versión, hay varias extensiones que no son soportadas, lo mejor es esperar un  tiempo a la espera que las extensiones hayan migrado y   que la distro asuma el soporte.
Cuando se tienen dos versiones de FF, ambas usan las mismas configuraciones, lo que a la larga puede causar conflictos.
Para mi , si de probar se trata, lo mejor es descargar la versión portable, la descomprimes en una carpeta cualquiera y la corres mediante Wine ya que se trata de una versión para windows, de esta manera no se interfiere con lo que se temga instado ya que crea una nueva carpeta de configuraciones
