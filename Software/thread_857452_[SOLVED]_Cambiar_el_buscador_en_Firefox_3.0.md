### Post by VitaLiNux on 2008-07-12
Prueba dándole a la pestañita que tiene el ícono de google, ahí aparecen otros buscadores!

Le das a la última opción ¨Administrar buscadores¨(en el mío es ¨Manage search engines...)Donde tienes la opción de agregar más buscadores si los que están ahí no te sirven!:)

### Post by Mauro22 on 2008-07-12
Valucha!


1) Iniciar firefox como root (sudo) ya que tiene que escribir en /

2) Entrar a [http://mycroft.mozdev.org/google-search-plugins.html](http://mycroft.mozdev.org/google-search-plugins.html)

3) Clickear el link a google.com.ar que mas te guste.

3) Elegir la opcion de Empezar a utilizar inmediatemente

4) Comproba que se haya agregado

5) Listo! El buscador esta disponible para todos los usuarios.
