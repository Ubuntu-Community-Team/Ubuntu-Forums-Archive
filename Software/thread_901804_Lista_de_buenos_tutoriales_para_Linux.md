### Post by marianom on 2008-08-26
Ahora con un poco más de tiempo les resumo que se pueden encontrar:

1- Advanced Bash Scripting Guide: 802 páginas, basicamente: si leés este libro entero y practicás y aún así no aprendes bash, no lo vas a aprender nunca más (lo digo por experiencia :))
2- Command Line Tools Summary: 125 páginas donde tenés relevado los más importantes comandos para tu consola
3- Linux Filesystem Hierarchy: 149 páginas. ¿Qué es /usr? ¿y para qué esta /opt? ¿Cual es la función de /etc? Las respuestas en este tutorial.
4- Linux Advanced Routing Traffic Control HOWTO: 93 páginas. El título como que lo dice todo no?
5- Iptables Tutorial: 434 páginas sobre como entender y gestionar IpTables... hay vida más allá de firestarter
6- The Linux System Administrators Guide: 166 páginas. Un vuelo de pájaro sobre las cosas que tenés que entender para que puedas poner en tu CV que sabés como administrar un linux.
7- Securing Optimizing Linux The Ultimate Solution: 876 páginas. Que te cascoteen el rancho toda la noche, vos lo más divertido adentro.
