Send2Imageshack es un script que nos permite enviar nuestras imágenes a ImageShack de una manera muy sencilla.

Modo de uso:

0. Instalar python-pycurl (sudo aptitude install python-pycurl).
1. Descomprimir y copiar o enlazar a ~/.gnome2/nautilus-scripts
2. Seleccione los archivos que desea cargar.
3. Elegir sendToImageshack de la lista de scripts.
4. Una vez terminada la carga de la imágenes serán proporcionados los links hacia las mismas.

[DESCARGA]("http://gnome-look.org/content/show.php/Send2Imageshack?content=100952")

 Fuente: IH
