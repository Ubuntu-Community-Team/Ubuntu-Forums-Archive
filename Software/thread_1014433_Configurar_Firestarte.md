Firestarter *no* bloquea sino que permite que el usuario configure el firewall de Linux de una forma amigable (en lugar de las primitivas de configuracion de iptables).

Entiendo que el juego necesita acceder a Internet, con lo cual seria importante saber que puertos necesitas habilitar de entrada y salida tanto a Internet como entre las dos maquinas (porque la cliente debe poder pasar a Internet por el gateway).
