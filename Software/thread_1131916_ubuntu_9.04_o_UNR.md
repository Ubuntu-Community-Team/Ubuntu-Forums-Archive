### Post by pablo.s on 2009-04-21
Hola: El escritorio de la UNR se puede
desactivar, es solo un front-end que incluso
para menos conflicto se puede desinstalar
el paquete que lo contiene.
Yo recomiendo dos opciones:
La primera es UNR, es una versión con el
núcleo optimizado para los procesadores
Atom de Intel para bajo consumo de energia.
La segunda es una distro derivada de
Ubuntu llamada Crunchbang Linux EEE,
viene muy austera, con gestor OpenBox,
trae lo justo y necesario para una netbook y
para desargarla es muy liviana, alrededor
de 470 MB.
No te recomiendo (para una netbook) la
instalación de Jaunty porque no está
preparada para este tipo de ordenadores,
sino para escritorios y notebooks mas
grandes.
Saludos.

### Post by pablo.s on 2009-04-21
Si, CrunchBang está muy bien hecha,
he probado muchas distro para netbooks
y la dos unicas que no apestan son la UNR
y la de CrunchBang, que es un derivado de
Ubuntu, o sea debajo de todo lo que dice
CrunchBang, si le quitas los diez o quince
paquetes de marca es una Ubuntu.
Las que apestan (a mi parecer) son:
--Easy Peasy, porque es muy pesada para
  este tipo de maquinas, mala elección de
  GNOME como gestor de escritorio, mala
  elección de paquetes.
--EEEbuntu, porque tiene errores de
  usabilidad, no tiene el kernel optimizado,
  mala elección de paquetes.
--Xandros para EEE, porque arranca rapido
  pero tiene software de dudoso paradero,
  calienta las maquinas demasiado, porque
  viene de Xandros.
