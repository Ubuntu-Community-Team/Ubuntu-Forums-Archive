### Post by [P]oli on 2008-07-23
```

/usr/bin/compiz.real (core) - Fatal: No manageable screens found on display :**0.0**

```

Según esta línea, compiz no reconoce tu placa de video/monitor (más seguramente la placa), 

```

Section "ServerLayout"
	Identifier	"Default Layout"
  screen 0 "Default Screen" **0 0**
EndSection

```

Me llama la atención esta parte del xorg.conf, ya que compiz busca **0.0**, cuando xorg.conf muestra **0 0**, no sé si tendrá algo que ver, pero probaría de cambiarlo. Antes de cambiarlo asegurate de hacer un backup
> **Hei Ku said:**
> 
Primero hacemos el backup:
```

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.orig

```

Para volver esto atrás, si hay problemas:
```

sudo cp /etc/X11/xorg.conf.orig /etc/X11/xorg.conf

```

Ahora modificar tu xorg.conf:
```

sudo gedit /etc/X11/xorg.conf

```


Si ésto no lo soluciona, ni idea :S
