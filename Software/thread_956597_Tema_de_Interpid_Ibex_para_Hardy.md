### Post by sdennie on 2008-10-23
Podés bajar la nueva tema y artwork con:

```

sudo sh -c "echo deb http://ppa.launchpad.net/kwwii/ubuntu hardy main >> /etc/apt/sources.list"
sudo sh -c "echo deb-src http://ppa.launchpad.net/kwwii/ubuntu hardy main >> /etc/apt/sources.list"
sudo apt-get update
sudo apt-get install gtk2-engines-murrine human-theme ubuntu-artwork

```

Despues, debés tener una tema que se llama "New Human" que es la de Ibex.
