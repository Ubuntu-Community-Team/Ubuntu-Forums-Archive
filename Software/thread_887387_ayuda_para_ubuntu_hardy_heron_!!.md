### Post by faktorqm on 2008-08-12
Si, todo el mundo lo tiene asi. La configuracion "tipica" por decirlo de alguna manera, es windows pelado sin AV, el juego de turno, drivers de video, directx, team speak y msn. En el resto del disco, ubuntu con TODO. lo que vas a tener que hacer, es lo siguiente, si tenes ubuntu solo instalado, tirar el windows al final del disco "achicando" la particion de ubuntu un toque (podes usar gparted) y capaz vas a tener que reinstalar el grub. Sino, borrar todo, instalar windows, luego ubuntu. el /home lo pusiste en una particion aparte? si hiciste eso, directamente reinstalas ubuntu utilizando el mismo nombre de usuario, instalas los programas (podes usar dselect) y automaticamente tenes todo andando. Salu2!
