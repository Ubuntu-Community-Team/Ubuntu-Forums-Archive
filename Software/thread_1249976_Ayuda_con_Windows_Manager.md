### Post by guisheca on 2009-08-26
Si no estoy equivocado ubuntu carga al inicio el window-manager que estaba al finalizar. O sea, basta con que, al finalizar la sesión tengas metacity en ejecución.
Para correrlo:

```
metacity --replace
```

Si eso no funciona una solución sería que pongas el comando que te puse arriba para que se ejecute al inicio.
