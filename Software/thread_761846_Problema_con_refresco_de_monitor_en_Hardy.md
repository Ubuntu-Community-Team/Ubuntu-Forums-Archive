### Post by sajnox on 2008-04-21
Es muy comun que eso pase, para solucionarlo la forma mas facil es la siguiente (no prometo que funcione pero generalemente si)

Abris una terminal y haces lo siguiente

1)Backup del xorg
```

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.back
```

2) Reconfiguracion del xorg

```
sudo dpkg-reconfigure xserver-xorg
```

3) Reinicias las X apretando ctrl + alt + backspace

Probalo y nos contas si funciono.

Saludos

### Post by faktorqm on 2008-04-26
Agarra tu xorg.conf, borra la parte que dice 

```

Section "Monitor"
        Identifier      "Configured Monitor"
EndSection

```

y pone esta:

```

Section "Monitor"
        Identifier      "Configured Monitor"
        HorizSync       30.0 - 56.0
        VertRefresh     50.0 - 100.0
        Option         "DPMS"
EndSection

```
