### Post by guillermolisi on 2011-01-30
Contanos como hiciste la instalacion ya que eso nos pondra en situacion para entender que problemas pueden estar jugando y de ahi las posibles soluciones.

Tambien seria de ayuda informacion sobre tu maquina, cantidad de RAM, particiones del disco rigido, placa de video, etc.

Con gusto te puedo indicar como conseguir un CD original de 10.04 pero dado el tiempo transcurrido desde su lanzamiento tendras una gran cantidad de actualizaciones que aplicar (usando conexion a Internet) para contar con cantidad de arreglos al dia. Para evitar esto lo mas recomendado es bajar una imagen ISO, que vendra con la gran mayoria de esas actualizaciones, quemar a baja velocidad un CD, revisar su integridad usando el hash de MD5, y (dependiendo como instalaste) reinstalar o instalar de cero.

### Post by Pan0ramix on 2011-02-01
Puede ser que el usuario no haya instalado sino probado Ubuntu 10.04, es una alternativa.
De todos modos sin mas información no podría guiarse para hallar una solución. 
 
Le recomendaría iniciar de nuevo la instalación "de cero", esto es, borrando todo lo que tenga en el disco (inclusive si tiene un windows precargado). Luego podrá instalarlo desde Ubuntu si asi lo prefiere.
 
Todo debería comenzar con obtener la ultima versión del 10.04 con las actualizaciones que indica Guillermo. El lugar de descarga es [acá]("http://releases.ubuntu.com/releases/lucid/"). Escoge la versión que mas se ajuste a tu necesidad.
