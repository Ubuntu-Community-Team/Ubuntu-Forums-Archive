### Post by rafaelca on 2009-08-14
Seguro que más de uno se preguntara "¿Cómo hago para ver los vídeos rápidos?".

Bueno estuve buscando y gracias a 'FacilWare"; encontre una aplicación FABULOSA.

La aplicación se llama así: UBUNTU-RESTRICTED

Esta además de traer muchas cosas interesantes,arregla e instala TODO lo necesario para tú ubuntu.

Si eres de los que bajan todo manual aquí te dejo el programa.

Para instalarlo basta con irnos a "Sistema->Administración->Gestor de paquetes Sypnaptic"

Todas las dudas me dicen,aunque no creo que tengan ;).

Es sólo instalar y nada más.
