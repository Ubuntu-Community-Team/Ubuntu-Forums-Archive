Una alternativa para utilizar esa version de Flash es usar una maquina virtual con Win.

Eso significa contar con cierta cantidad de memeoria RAM para evitar que el sistema no se arrastre.

Uso VirtualBox y VMware con maquinas que tienen 2Gb RAM y procesador de doble nucelo y anda mejor que en una maquina real.

No tenes aceleracion grafica pero para Flash no importa.

Si sera solo para uso personal, es decir no necesitas monitorear desde otro equipo, mi recomendacion va por VBox (tenes una version OpenSource - OSE y una privativa. La primera no tiene acceso directo a dispositivos USB, la segunda si).
