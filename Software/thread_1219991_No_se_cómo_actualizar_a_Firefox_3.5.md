### Post by gmunioz on 2009-07-23
Hola rub....:

Simplemente:

Desinstala Firefox.

Pulsa con el boton derecho en Aplicaciones

Elige editar menus

Navega hasta Internet - Shiretoko

Pulsa el boton derecho, propiedades

Cambia el nombre por Firefox

Cambia el comentario por Web Browser

Cierra

Pulsa Aplicaciones

Navega a Internet - Firefox boton derecho

Elige añadir este lanzador al panel o al escritorio

según tus deseos.
