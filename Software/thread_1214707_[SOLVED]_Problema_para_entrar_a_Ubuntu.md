O sea, tenias un disco con los dos sistemas operativos (dual boot) y reinstalaste Windows solamente, correcto ?

Si es asi, Windows "pisa" la informacion de arranque que grabo oportunamente Ubuntu y por eso no te lo ofrece como alternativa para inicar la maquina.

Hay varias formas: Una es con una utilidad llamada Super Grub Disk, que tal vez para tu caso sea la mas recomendada.

En este thread tenes un caso similar al tuyo. Leelo y despues pregunta lo que necesites:

[http://ubuntuforums.org/showthread.php?t=1149624&highlight=super+grub](http://ubuntuforums.org/showthread.php?t=1149624&highlight=super+grub)
