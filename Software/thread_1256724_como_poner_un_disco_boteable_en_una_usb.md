Hola tsu....:

*"....pero que no en unetbootin tiene solo para un limitado numero de versiones o si tiene para acer booteable cualquier cd..."* ?????????? 

Esta aplicación, consistente en un solo archivo, a ejecutar como root

en cualquier distribución GnuLinux o desde Windows  (archivo.exe), muestra

una pantalla en la cual eliges la imagen iso de la distribución GnuLinux

que mas te plazca, para portarla a un dispositivo usb, el cual hace 

booteable, para que la distribución inicie del mismo, si la imagen iso es

de un cd/dvd para instalar, iniciará para instalar, si es de un cd/dvd 

live, iniciará la sesión live.

dispositivo usb con una distribución Linux y no para crear cds booteables
