### Post by Carlos C on 2010-02-08
Si tienes una tarjeta nvidia debes tener un xorg.conf configurado para esa tarjeta, por lo que obviamente no puedes usar una configuración para Intel, eso debes corregirlo.

El parámetro "Device" de tu "Section Screen" debe ser igual al parámetro "identifier" de tu "Section Device". Eso es lo que aparece en el enlace que te dieron, lo que ocurre es que en ese caso la tarjeta es una Intel.

Otro enlace que te puede servir:

[http://blog.etxea.net/index.php/2007/01/12/como-configurar-el-monitor-acer-1916ws-en-gnulinux-ubuntu](http://blog.etxea.net/index.php/2007/01/12/como-configurar-el-monitor-acer-1916ws-en-gnulinux-ubuntu)

La lógica del archivo no es tan compleja es cosa de fijarse bien ;).

Para generar un xorg.conf limpio puedes utlizar el comando:
```
sudo X :1 -configure
```
Se creará un archivo ~/xorg.conf.new. Debes copiarlo en /etc/X11 y renombrarlo a xorg.conf:
```
sudo cp ~/xorg.conf.new /etc/X11/xorg.conf
```
Con eso puedes trabajar con un archivo nuevo y añadir los parámetros que se te sugieren.
