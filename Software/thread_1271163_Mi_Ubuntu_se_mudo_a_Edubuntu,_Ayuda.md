### Post by josecuervo86 on 2009-09-20
Estas seguro que bajaste el cd de Ubuntu y no el de Edubuntu? No hay razón para que cambie de uno a otro.

De cualquier manera, lo mas sencillo en tu caso es abrir una terminal y escribir:

```
sudo apt-get install ubuntu-desktop
```

Lo que si, te advierto que esta opcion te va a instalar un monton de aplicaciones de Ubuntu ademas de las que ya tenes de Edubuntu, y los menus van a sufrir las consecuencias (muchos iconos). Por mi parte te recomendaria, si es que recien lo instalas, que vuelvas a reinstalarlo.

Te repito, no hay razón para que cambie de Ubuntu a Edubuntu, ya que el software que trae uno, dista mucho del otro. Yo primero me fijaria que el cd que bajaste sea realmente de Ubuntu.
