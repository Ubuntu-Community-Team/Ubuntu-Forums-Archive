### Post by juanman on 2008-06-24
El problema es que necesitás el plugin de shockwave (que es distinto al de flash), y no tiene versión para linux. Hay una peticion online para presionar a Adobe, pero dudo que de frutos ya que solo tiene 600 votos en 2 años y la "formal carta" solo dice: "WE NEED SHOCKWAVE IN LINUX!!!" :P
Ademas de que es una tecnología que se usa muy poco.
Entonces, soluciones:
- Instalar wine -> Instalar Firefox para windows (tambien puede ser el IE si sos masoquista :P) -> Instalar el plugin de shockwave desde la pagina [http://www.adobe.com/shockwave/download/index.cgi?Lang=Spanish](http://www.adobe.com/shockwave/download/index.cgi?Lang=Spanish)
- No jugar a los juegos de Cartoon Network que requieran Shockwave, y buscar paginas alternativas como minujuegos.com
