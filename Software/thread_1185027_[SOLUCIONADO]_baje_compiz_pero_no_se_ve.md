### Post by nopersona on 2009-06-11
Instalaste el gestor de preferencias?

```
sudo apt-get install compizconfig-settings-manager
```

Luego lo encuentras en sistema>prefrencias

También instala fusion icon para activarlo y desactivarlo con mayor comodidad.

Saludos.
