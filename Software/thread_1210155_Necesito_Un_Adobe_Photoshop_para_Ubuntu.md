Hola PelaOlinux Adobe PhotoShop No corre en Linux pero Ubuntu tienes un Excelente Programa de Diseno se llama **GIMP** que se encuentra en Aplicaciones>Graficos>Gimp.

Ahora hay una opcion de instalar **WINE** es una reimplementación de la API de Win16 y Win32 para sistemas operativos basados en Unix. Permite la ejecución de programas para MS-DOS, Windows 3.11, Windows 95, Windows 98, Windows ME, Windows NT, Windows 2000, Windows XP y Windows Vista.Con Wine podrias instalar Adobe Photoshop en Ubuntu.

_**P.D.:**_ No estoy deacuerdo con la utilizacion de Programas para Windows mediante  Wine, te recomiendo leer este Link donde se discute en el Foro la utilizacion de dichos programas mediante Wine.

**Hasta donde llegar con WINE**:
[http://ubuntuforums.org/showthread.php?t=1191124](http://ubuntuforums.org/showthread.php?t=1191124)
