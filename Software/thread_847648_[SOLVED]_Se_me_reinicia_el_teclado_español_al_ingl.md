### Post by santiagoward2000 on 2008-07-03
Hola!
A mi me pasaba lo mismo cuando usaba XGL. Yo lo solucioné poniendo que corra esta comando cuando iniciaba:
```
setxkbmap -model pc105 -layout es -variant basic 
```
Mi teclado es español de España. Si necesitas para español Latinoamericano sería:
```
setxkbmap -model pc105 -layout la -variant basic 
```
Saludos!

### Post by sajnox on 2008-07-05
Si las teclas estan en otro orden probablemente sea por la cantidad de teclas, te paso la configuracion de mi xorg.

```
Section "InputDevice"
	Identifier	"Generic Keyboard"
	Driver		"kbd"
	Option		"CoreKeyboard"
	Option		"XkbRules"	"xorg"
	Option		"XkbModel"	"pc105"
	Option		"XkbLayout"	"es"
EndSection
```
