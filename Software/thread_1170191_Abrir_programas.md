### Post by BoyOfDestiny on 2009-05-26
No se como usar esta programa, pero aquí hay tutorías en español en la pagina de aircrack
[http://aircrack-ng.org/doku.php?id=tutorial&DokuWiki=0b802504e616b6cee9672204e50b428f#spanish](http://aircrack-ng.org/doku.php?id=tutorial&DokuWiki=0b802504e616b6cee9672204e50b428f#spanish)

Normalmente, solamente poner el nombre de la programa en el &#8220;Terminal&#8221;

por ejemplo

Abre el terminal
Aplicaciones &#8594; Accesorios &#8594; Terminal
pone:

```
gedit
```

y la programa se abra

para instrucciones, pone:
```
man gedit
```

o 

```
man aircrack
```
