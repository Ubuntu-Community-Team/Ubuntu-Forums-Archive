### Post by totoloco on 2008-04-28
Como generar imágenes abstractas, muy bueno para fondos de escritorio.

Para imágenes abstractas de este tipo (Peter de Jong):

[http://www.complexification.net/gallery/machines/peterdejong/](http://www.complexification.net/gallery/machines/peterdejong/)
[http://flickr.com/photos/56529747@N00/sets/72157600305970586](http://flickr.com/photos/56529747@N00/sets/72157600305970586)

Se puede instalar el Fyre:
[http://fyre.navi.cx/](http://fyre.navi.cx/)

sudo apt-get install fyre

--------------------------------------

Para las del tipo Flame Fractal:

[http://www.tutorialsinmaths.com/Fractals/freeflamefractals.htm](http://www.tutorialsinmaths.com/Fractals/freeflamefractals.htm)
[http://students.washington.edu/bitsed/qosmic/images/qosmic1.3-screen1.jpg](http://students.washington.edu/bitsed/qosmic/images/qosmic1.3-screen1.jpg)

Se puede instalar Qosmic:
sudo apt-get install qosmic

El apophysis funciona para win y es libre tb:
[http://www.apophysis.org/index.html](http://www.apophysis.org/index.html)

----------------------------------
Para una de las dos instalaciones, no me acuerdo cual, se necesitan los source de getdeb.net

Agregar a /etc/apt/sources.list:

gksudo mousepad /etc/apt/sources.list
deb [http://ubuntu.org.ua/](http://ubuntu.org.ua/) getdeb/

Saludos
