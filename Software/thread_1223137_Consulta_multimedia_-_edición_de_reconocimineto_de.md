Para cambiarle los "datos" a el tema yo hice esto: abrí la canción desde el Amarok y le hice click derecho y puse Edit Track details y en la pestaña "Tracks" le puedo cambiar el título, genero, álbum, etc, etc...

### Post by aledruetta on 2009-07-25
Yo uso EasyTag. Es muy buena aplicación, porque te permite modificar todos los datos de los mp3, pero además busca en una base de datos en Internet y te completa los datos que no conocés.
```
sudo aptitude install easytag
```
