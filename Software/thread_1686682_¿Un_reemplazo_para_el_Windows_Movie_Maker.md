### Post by sebastianabate on 2011-02-14
Probá con el OpenShot ([www.openshot.org]("http://www.openshot.org")) que a mi gusto es lo más parecido al WMM, es muy fácil de usar y además tiene muchas opciones avanzadas por si más adelante querés hacer algo un poco más complicado.
Si tenés Ubuntu 10.10 fijate que está en los repositorios, pero si querés la última versión agregate el repositorio oficial de los desarrolladores e instalalo
```

sudo add-apt-repository ppa:openshot.developers/ppa
sudo apt-get update && sudo apt-get install openshot
```
