### Post by alfplayer on 2010-03-27
Probablemente es causado por el sistema de sonido PulseAudio. Este tipo de problemas disminuyeron en las versiones posteriores de Ubuntu. Se podría intentar resolver deshabilitando PulseAudio para que quede funcionando solo ALSA, aunque esto puede traer otros problemas.
