### Post by gefarion on 2008-06-10
Hola, para tenerlo en español simplemente instala el siguiente paquete:

openoffice.org-l10n-es

para tener el diccionario:

myspell-es

En resumen:
sudo apt-get install openoffice.org-l10n-es myspell-es
