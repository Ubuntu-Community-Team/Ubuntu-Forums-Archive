### Post by juancarlospaco on 2010-01-25
No Tint2 es un mini-panel (o micro-panel xD ) que no incluye el menu de Gnome.

Se usa cuando no se usa un DE (gnome,KDE), con Gestores de ventanas como IceWM, OpenBox, etc.

Aun asi se puede usar otras cosas para acceder a los programas, como [Kupfer]("apt:/kupfer"), [gRun]("apt:/grun"), [WBar]("apt:/wbar"), etc.
