Pero se me ocurren un par de ideas: una, que actualices el genisoimage a la versión de Intrepid, donde se supone que se corrigió el bug; otra, que pruebes con [este]("ftp://ftp.ubuntulinux.org/ubuntu/pool/universe/g/graveman/")[0], el [graveman]("http://graveman.tuxfamily.org/index.php?l=e")[1], que es un poco viejito pero usa otros paquetes para el proceso y tiene algunas opciones más que Brasero.
