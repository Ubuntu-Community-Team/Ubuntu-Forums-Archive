### Post by RafaelG on 2011-07-12
A800xl;

Prueba con las siguientes instrucciones:

1- Entra a la terminal y ejecuta **sudo apt-get install timidity**

2- Dentro de TuxGuitar, anda a Herramientas >Preferencias, te vas a  la seccion de Sonido y fijate que la opcion que dice "**Puerto Midi**"va a  estar marcada la opcion ***Midi Through Port-0 #0*** (o alguna parecida), en  este caso cambiala y selecciona la que dice ***TiMidity port 0 [128:0] ***(o parecido). Avisanos si  funciono!
