### Post by guillermolisi on 2008-11-12
El asunto con Shockwave, win e IE4Linux es asi:

Shockwave existe solamente para entorno Win. Podria ser que funcione bajo Wine pero no se si con Wine te funciona el IE, por ejemplo (nunca lo probe ni necesite hacerlo ya que para navegar me alcanza y sobra con Ubuntu y FFox/Konqueror).

IE4Linux es un port de IE (o algo asi) para entornos Linux y por lo tanto no puede "beneficiarse" (si cabe el termino) con los servicios de Shockwave.

Edit: IE4Linux es un add-on/plugin para FireFox
Edit II: Me corrijo. No es un add-on/plug-in, es un clon de IE para Wine. En todo caso habria que instalarlo en Wine y luego probar si el Shockwave lo reconoce como IE en un entorno Win y se instala.
