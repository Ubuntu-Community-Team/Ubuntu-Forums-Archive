Las opciones q tenes (o las q conozco) son:
- Instalar [esta extension]("http://extensions.services.openoffice.org/project/ImpressRunner") de OpenOffice para q abra los PPS automaticamente. Es media lerda la carga del OpenOffice pero es la opcion ideal si queres usar soft libre (instala la version 3 q carga un poquito mas rapido).
- Instalar el MS PowerPoint 2003 Viewer que esta perfectamente soportado por wine. Es un freeware (no libre pero si gratuito), lo bajas de [aca]("http://www.microsoft.com/downloads/details.aspx?familyid=428D5727-43AB-4F24-90B7-A94784AF71A4&displaylang=es")
- Usar la extension Open IT Online de Firefox, q te da la opcion al abrir algun documento de usar alguno de los visores Online (te recomiendo el de Zoho)
