### Post by Ghost|BTFH on 2010-03-31
Es posible que desee desactivar Compiz en Sistema -> Preferencias -> Apariencia en la pestaña "efectos visuales"

Compiz veces puede causar problemas con diferentes tipos de reproducción de vídeo - incluyendo Java.
