### Post by juanman on 2008-10-13
Para linux nativo hay algunas opciones, aunq no tan buenas como el virtual dj...

Te recomiendo: 

mixxx: ta en los repositorios, es simple y trae varios skins (me gusta el nCut)... En mi pc saltan los temas un poco con esta ultima version; creo q con un kernel de baja latencia debe funcionar mejor...

ultramixer: no es libre, la version full funciona por 1 hora y luego hay q reabrir el programa, pero la version free es muy buena igual. La podes bajar de [http://www.ultramixer.com/download.html]("http://www.ultramixer.com/download.html") . Necesita java y se instala con next, next...

Si no te alcanzan, parece q el virtual dj funciona con wine, aunq te come la CPU... [http://appdb.winehq.org/objectManager.php?sClass=version&iId=13450]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=13450")

Para mas programas e info para DJs en linux, te paso algunos links: [http://sound.condorow.net/](http://sound.condorow.net/)
[http://www.openjay.org/](http://www.openjay.org/)

Saludos
