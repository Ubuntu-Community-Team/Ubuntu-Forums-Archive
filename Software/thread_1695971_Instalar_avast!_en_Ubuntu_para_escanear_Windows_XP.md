### Post by guillermolisi on 2011-02-28
TRK posee dos actualizaciones a saber:

[LIST]
[*]De los motores y lista de virus
[*]Del mismo TRK
[/LIST]

Para el caso de motores y lista de virus, no se guardan porque esta pensado para correr siempre la ultima version disponible liberada para cada antivirus.

Para el segundo caso, solo se guardan las actualizaciones si es un pendrive o equivalente (RAM SSD, por ejemplo) y en el manual de TRK estan las indicaciones de como proceder para esto. Creo que no es posible sobre un CD regrabable, pero habria que verificarlo contra la documentacion de TRK.
