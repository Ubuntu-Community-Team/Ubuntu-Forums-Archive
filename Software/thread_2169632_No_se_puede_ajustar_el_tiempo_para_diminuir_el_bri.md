### Post by papibe on 2013-08-22
Hola,

Existe una opción para evitar que se disminuya el brillo. Lamentablemente yo tengo mi sistema en inglés, pero estoy seguro que se lo vas a poder traducir:
```
System Settings -> Brightness and Lock -> Dim screen to save power
```
Desmarca esa última opción.

Ojalá te sirva,
Saludos.
