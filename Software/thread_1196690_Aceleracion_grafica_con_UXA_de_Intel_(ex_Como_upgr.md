### Post by gmunioz on 2009-06-24
Hola Pablo:

Fuera de este tópico, (upgrade del kernel) pero vinculado a 

la última consulta, tengo entendido que:

Los problemas entre Jaunty y las gráficas de Intel son una 

deficiente vinculacion del actual nucleo 2.6.28 Kernel Mode 

Setting (KMS) de Intel con la nueva aceleracion UXA.

Si bien la solucion ha de venir con el driver de Intel 2.7.99.1, 

según lo expresado por el desarrollador de Canonical Bryce Harrington,

menciona tambien que los usuarios de Jaunty tienen la ventaja de poder

instalar el kernel 2.6.30 para resolver el problema. 

Sugiere, luego de instalado el kernel,  activar el modulo UXA en el 

archivo xorg.conf, que esta desactivado por defecto. 

Para activar UXA, se debería agregar Option AccelMethod" "UXA" dentro 

de la sección "Device" del archivo /etc/X11/xorg.conf


```
Section "Device"
        Identifier    "Configured Video Device"
        # ...
        Option        "AccelMethod" "UXA"
EndSection
```


Y para el caso de que la gráfica este dentro de una lista negra 

de Compiz, habría que agregar dentro del archivo 

/etc/xdg/compiz/compiz-manager la línea

```
SKIP_CHECKS=yes
```
