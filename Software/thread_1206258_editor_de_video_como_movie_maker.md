### Post by zhelo on 2009-07-07
> [**AvideMUX**]("http://avidemux.berlios.de/"): Quizá uno de los editores de vídeo más conocidos en Ubuntu Linux que también está disponible para BSD, Mac OS X y Microsoft Windows bajo licencia GNU GPL.
[**CinePaint**]("http://www.cinepaint.org/"): Su función principal es el retoque de películas fotograma a fotograma aunque ofrece funcionalidades adicionales como HDR o High Dinamyc Range para dar un aspecto más brillante a las imágenes.
[**Cinelerra**]("http://cinelerra.org/"): Un editor de vídeo que permite la captura, composición y edición de pistas de audio y vídeo.
[**FFmpeg**]("http://ffmpeg.mplayerhq.hu/"): La base principal de muchos de los editores de vídeo. Mediante el uso de la línea de comandos podremos editar o convertir nuestros vídeos a otros formatos.
[**Jahshaka**]("http://jahshaka.org/"): Un editor de vídeo con funcionalidades para añadir efectos en tiempo real. Su principal característica es la velocidad.
[**KDEnlive**]("http://www.kdenlive.org/"): El editor de vídeo de KDE que ofrece edición de múltiples pistas de vídeo y audio.
[**Kino**]("http://www.kinodv.org/"): Quizá el más completo de los editores de vídeo que nada tiene que envidiar a aplicaciones comerciales. Ofrece una excelente integración con la tecnología IEEE-1394 para capturar e integrar múltiples vídeos desde la videocámara.
[**LiVES**]("http://lives.sourceforge.net/"): Un sencillo y excelente editor de vídeo con el que no tendrás que preocuparte de aspectos como los formatos o la resolución de la película.
[**Slideshow Creator**]("http://slcreator.sourceforge.net/"): Aunque no es exactamente un editor de vídeo tal y como pueden ser el resto de los programas mencionados permite una serie de posibilidades para convertir tu colección de fotos en un DVD de calidad con efectos y banda sonora.
[**Vivia**]("http://vivia-video.org/"): Disponible para Linux y Windows, este editor de películas DV muestra una apuesta calara por la sencillez a la hora de modificar nuestras películas.
