### Post by Tosh78 on 2009-10-04
No deberias tener problemas con ninguna de las dos versiones.
Con respecto al disco, lo ideal seria tener una particion para swap, una para el sistema operativo ( / ) y otra para tus archivos y cosas (/home). El sistema ext4 es bastante mejorcito que el NTFS, pero sin embargo debes saber que si queres acceder a tus datos desde Windows no vas a poder. En ese caso, podes probar con algun programita que hay por ahi dando vueltas (que nunca me funciono a mi) o directamente NTFS.
