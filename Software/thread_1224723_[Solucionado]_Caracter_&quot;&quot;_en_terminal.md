### Post by pagondel on 2009-07-30
Yo edite el titulo ;).
Y conozco ese símbolo como tubería o pipe (en ingles), y recibe este nombre ya que tal tubería es utilizado para controlar el flujo de información, como por ejemplo cuando se utiliza para filtrar la salida de un comando con el comando grep:

```
lspci | grep VGA
```

EDIT: [Wikipedia en inglés](http://en.wikipedia.org/wiki/Pipe_(character))
