Please don't use the PPA on Ubuntu 16.04 LTS. Follow the instructions on the PPA page to ppa-purge it. I would have removed those packages completely except they need to be present for ppa-purge to work correctly.

There will be no further updates (security or bugfixes) to those PPAs for 16.04 LTS. There are multiple known bugs with the PPA on 16.04 LTS.

You have a choice between LTS or latest.
