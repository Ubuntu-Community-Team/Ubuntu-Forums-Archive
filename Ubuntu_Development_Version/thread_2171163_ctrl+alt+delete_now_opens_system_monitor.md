A long standing 'bug' in compiz has been fixed so now instead of a log out it opens sys monitor on the process tab.
If one prefers the log out then can be set back by removing both the command & binding in either gsettings or ccsm
(org.compiz.integrated in dconf-editor, command 21, run-command-21, in ccsm it's command 20

Edit: note that this can only affect new installs & or new user accounts
