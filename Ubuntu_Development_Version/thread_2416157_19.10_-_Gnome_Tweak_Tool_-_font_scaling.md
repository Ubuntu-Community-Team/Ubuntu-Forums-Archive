### Post by #&amp;thj^% on 2019-04-06
I'm very rusty with GUI's lately but if you don't mind using the terminal.
For the scaling factor this one :
```

gsettings reset org.gnome.desktop.interface text-scaling-factor
```
For my old eyes I use:
```
gsettings set org.gnome.desktop.interface text-scaling-factor 1.03

```

**NOTE: To get/see the current configuration for each settings replace "reset" with "get" .**
example:
```
gsettings get org.gnome.desktop.interface text-scaling-factor
1.0

```
