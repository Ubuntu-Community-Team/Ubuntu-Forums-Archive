### Post by TDK800 on 2013-03-05
There are currently two solutions for digitally visually signing PDF documents in Ubuntu that have to be manually set up:

JsignPDF:
[http://sourceforge.net/projects/jsignpdf/](http://sourceforge.net/projects/jsignpdf/)

OpenSignPDF:
[http://opensignature.sourceforge.net/english.php](http://opensignature.sourceforge.net/english.php)

I'm thinking it would be a great feature for businesses if this functionality would be supported by default in Ubuntu. Those unable to set it up themselves are forced to print the pdf, sign it, scan it vs Windows users who have Adobe products available which support signing.
