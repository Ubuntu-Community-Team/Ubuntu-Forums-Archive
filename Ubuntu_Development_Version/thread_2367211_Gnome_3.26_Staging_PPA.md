### Post by amano on 2017-07-27
The Gnome 3.26 staging PPA is starting populating: [https://launchpad.net/~gnome3-team/+archive/ubuntu/gnome3-staging/+packages?field.name_filter=&field.status_filter=published&field.series_filter=artful](https://launchpad.net/~gnome3-team/+archive/ubuntu/gnome3-staging/+packages?field.name_filter=&field.status_filter=published&field.series_filter=artful)

So you can track the current progress. Mutter, Gnome-Shell, gnome-session, gsd, g-c-c and gdm are on 3.25.4, tracker is on 1.99 already.

Nautilus is still on 3.25.1, but has support for the new tracker enabled. It is also missing a new fix from artful proposed, so it shows that a newer version is available there.

Please don't add the PPA just yet until jbicha tells you to do so. You might break your Gnome session. Or just try some not so risky things like the new gnome control center layout.

But it is nice to track the progress. A big yay to Jeremy ;)
