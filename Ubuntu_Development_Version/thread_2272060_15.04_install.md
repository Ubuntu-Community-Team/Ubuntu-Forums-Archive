### Post by Elfy on 2015-04-03
>  I would like to make 14.04 the 'default' selection in grub. 

Boot into 14.04

Reinstall grub there. 

```

 sudo grub-install /dev/sda
 sudo update-grub
```

Anything else should then be extra's.

### Post by oldfred on 2015-04-03
Grub remembers where you installed. You can use spacebar to uncheck all choices and the the reinstall location should be empty/blank. Check before & after.

 #To see what drive grub2 uses see this line   - grub-pc/install_devices:
sudo debconf-show grub-pc # for BIOS with grub-pc 
It will show drive model & serial number
to see drive info
sudo lshw -C Disk -short 

    sudo grub-probe -t device /boot/grub

   #to get grub2 to remember where to reinstall on updates:
sudo dpkg-reconfigure grub-pc or dpkg-reconfigure grub-efi-amd64
#Enter thru first pages,spacebar to choose/unchoose drive, enter to accept, do not choose partitions
[http://ubuntuforums.org/showthread.php?t=2189643](http://ubuntuforums.org/showthread.php?t=2189643)
