### Post by grahammechanical on 2014-01-24
If we are using Ubuntu Trusty Tahr + Unity then we can install gnome-session-flashback. We then get two additional options; Gnome Flashback (with Compiz) and Gnome flashback (with Metacity). Which I think is more compatible with Ubuntu Unity than Gnome Shell Classic which is a Gnome 3 shell extension and so, in my opinion, more suited to Ubuntu Gnome.Regards.
