### Post by grumblebum2 on 2014-04-16
I see this when the compiz **place windows**  plugin is not enabled.
Use compizconfig-settings-manager(ccsm) to enable it. Look in "window management".
If it is already enabled try changing placement mode from "smart" to "centered".
I use and prefer centered placement anyway.
