### Post by Alistair_Grant on 2015-02-02
I'm running Ubuntu 14.10 with the 3.19-rc6 kernel on a Dell XPS 13 9333 (3.19-rc7 is compiling as I write this).  My experience has been that 3.19 is more stable than either 3.16 or 3.18, primarily due to the fixes in the xHCI stack.  If you're on a Haswell based PC with USB ports controlled by xHCI (as opposed to eHCI), then you could well benefit from running the newer kernel (but don't forget that any bug reports back to Canonical need to be reproduced on the official Ubuntu kernel).

Cheers,
Alistair
