### Post by jbicha on 2017-09-07
> **MasterNetra said:**
> I know it's early still but apparently Gnome Tweak isn't installed by default in 17.10 will this change by final release?

No. The Ubuntu Desktop Team is firm that GNOME Tweaks will not be installed by default in 17.10.

While there are some very useful tweaks in Tweaks, there are also other tweaks that are difficult for Canonical to support.

Similarly, there are only two GNOME Shell extensions in 'main' that are officially supported: the Dock and the Indicators and both are enabled by default. If you want to install other extensions, feel free; they just aren't supported by Canonical.
