### Post by P-I H on 2018-04-08
You manage desktop icons in gnome-tweaks.
I personally don't want to have icons on the desktop and move these to the home directory.
To use them I click the Show Applications button and then right click the icon and select Add to Favorites. The icon will then be placed and clickable in Ubuntu Dock.
