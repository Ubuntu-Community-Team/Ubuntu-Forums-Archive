### Post by VMC on 2013-01-25
I do it a little differently. I use my own syslinus.cfg and ldlinux.sys and just replace all the other files with whats inside of the ISO. I don't have to reformat my flash drive.

At some point they changed the pointer to vmlinuz to vmlinuz.efi. So my current syslinux.cfg file looks like this:

```
DEFAULT live
LABEL live
kernel /casper/vmlinuz.efi
append noprompt cdrom-detect/try-usb=true file=/cdrom/preseed/ubuntu.seed boot=casper initrd=/casper/initrd.lz quiet splash --
```

Both syslinux and ldlinx files are located on the root.

That way it just boots the distro straight away.
