### Post by grahammechanical on 2013-04-23
Why don't you install Raring in another partition and test it for yourself. What you are asking about is a matter of individual perception. I have been using Raring for months and updating daily but 2 weeks ago I put in a fresh install and I perceive it to be much faster than my daily updated Raring Install. Boot time is much quicker. I do not perceive "laggy" at all. A lot depends upon individual setup and hardware. RAM speed. Hard drive speed. SSD is said to be very fast. 

Libreoffice is at version 4 and a lot of work has been done by the developers. They certainly fixed the little flaw of the file open dialog not remembering the last location you opened a file from.

Memory usage has dropped by 15-20% in my machine compared to 12.10. All the work being done to get Ubuntu on mobile devices is filtering down to desktop ubuntu.

But remember, this: If you install 13.04 you will have to upgrade to 13.10 in six months time because 13.04 only has 9 months support. As, does 13.10. From now on anyone installing an interim release will have to upgrade every six months. Anyone who does not like doing that should stick to 12.04 or 12.10.
