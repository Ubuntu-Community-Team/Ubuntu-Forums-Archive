### Post by The Cog on 2015-03-18
No. In theory you don't need to install - the updates will bring you to exactly the release version.
In practice I do usually do a fresh install when the final release comes out, but I think I am probably just wasting my time. The only things that might be a problem are default configurations changing - e.g. late artwork backgrounds etc. A pre-existing user configuration may not pick up the new default settings. You are so close to the proper release now that problems are very unlikely.

### Post by grahammechanical on 2015-03-18
The code name for 15.04 while it is under development is vivid vervet. It has separate repositories from 12.04 and 14.04 and 14.10. Those vivid software repositories will be the same repositories for any installation of 15.04 that is made with the final released version.

An install of vivid vervet at any time during the 26 weeks development period will become Ubuntu 15.04 by the end of April this year by doing regular updates. Preferably, daily updates.

Regards.

P.S. Anyone who installs 15.05 after release day will still need to tick the box "Install updates" while the installation is taking place. For there will be updates available.
