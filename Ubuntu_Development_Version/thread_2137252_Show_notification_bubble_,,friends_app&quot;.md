### Post by rsrocha on 2013-04-20
Open dconf Editor ando go to:

com - canonical - friends

Change the "notifications" value to "all"

You can also change the interval to display notifications.

I think the dev team should place these settings available in the System Settings so the user could easily change the friends notifications behaviour.

In 13.04 all your facebook/twitter feeds are in the dash on the social pane. You dont need any external app to show the feeds like gwibber did. On the social dash just right click on any notification and it will show them in detail with the link to facebook or twitter page.
