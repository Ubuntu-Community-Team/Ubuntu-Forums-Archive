### Post by heir4c on 2013-11-27
FYI:
When I install Trusty, there is a different setting in Nautilus (I don't now if this was in 13.10 but not in 13.04).
Via the menu-bar and than clicking "files" I go to Preferences and the "Behavior" Tab. 
There under 'Executalble Text Files' is the default setting now the second option checked. (see screenshot below).

When I doubleclick a executable file, just gedit opens the file and show the text. 
Before I get a window with 4 options in to choose.
I changed that now to "Ask each time" and now I get the window as before.
