### Post by cigtoxdoc on 2015-07-23
I have taken my two desktop machines and upgraded from 15.04 to 15.10 Ubuntu-Gnome.  Main reason for doing this (besides going against all warnings to the contrary) was to be able to run Evolution 3.16.3 (much improved over 3.16.0).  PCs involved are identical, based on the Intel 4770K processor with Haswell graphics.  The most serious problem was loading the latest version of Ubuntu Tweak so I could set file associations to the PDF Studio Pro software I use for reading and modifying PDF files.  So far, everything is stable.  I am using the Gnome Classic desktop on both PCs.
