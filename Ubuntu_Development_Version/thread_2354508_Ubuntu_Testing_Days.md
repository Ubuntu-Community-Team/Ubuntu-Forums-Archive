### Post by elopio on 2017-03-03
Hello!

Every friday is a testing day in the Ubuntu Community. We host a live chat on [http://ubuntuonair.com/](http://ubuntuonair.com/) to show a cool free software project that's in active development, we talk to the developers of these projects and we show tools to contribute to this project, with a little focus on testing.

Today will be more relaxed than usual, because we will be talking with Alan Pope about games. Basically, we'll be playing games and pretend that we are working :p

So [http://ubuntuonair.com/](http://ubuntuonair.com/) today at 16:00 UTC. That's in 45 minutes.

For more info about testing days, and links to past sessions: [https://wiki.ubuntu.com/Testing/UbuntuTestingDay](https://wiki.ubuntu.com/Testing/UbuntuTestingDay)

pura vida.
