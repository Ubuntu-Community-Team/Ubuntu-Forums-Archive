Heads up for any one running 'newer' hardware. This is a dual-boot with Win10 machine-Ubuntu Dev. Version

on about 'Secure Boot Violation' and then the 'no operating system found blah blah blah' check your system settings.

asked me to enter a code to confer it was me who initiated the change for Legacy blah blah and enter. 
All is good and I am a 'HappyCamper' again with my Ubuntu dual boot with Win10:D:D

### Post by oldfred on 2016-01-31
UEFI & BIOS updates have always reset to defaults.
So if you have made any changes to settings you have to redo those changes.

My old BIOS system had a lot of settings that I changes and after first update to BIOS, I had to recreate settings. So I took photos to make it easier to change them the next time.

My newer UEFI Asus motherboard posts a summary of changes when you go to same them. So much easier to manage settings.
