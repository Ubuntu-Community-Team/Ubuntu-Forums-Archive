### Post by mcellius on 2020-04-16
For that to work, I think, you have to use Gnome Tweaks.  Under Appearance, choose a dark shell (e.g. Yaru Dark), and it will apply the dark theme to the notifications and to the settings in the upper right corner.  (I don't know if this requires an extension - user themes - to be installed first.  I think at one time it did, but I don't know about now.)
