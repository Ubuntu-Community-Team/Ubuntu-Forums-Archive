### Post by grahammechanical on 2012-11-14
Look at System Monitor. Remember, Raring Ringtail is not a new version of Ubuntu. It is based upon 12.10 and is slowly, package by package changed into 13.04 as those packages are developed. My log in screen still says, 'Ubuntu 12.10.' This is not an error. It is the way of things when we use the development branch this early in the cycle.

An example of this is the fact that Raring had the 12.10 Linux kernel of 3.5 when we first stated using it. Now it has Linux kernel 3.7. Whereas, 12.04 will only get the Linux kernel 3.5 at the end of January.
