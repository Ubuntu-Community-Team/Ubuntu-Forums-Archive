### Post by mc4man on 2013-05-10
> **EgoGratis said:**
> And found another gem. Clicking backspace doesn't go one level up anymore...

Surprise, surprise - you can actually change that in nautilus quite easily.
(replaces current up dir. binding with  backspace key

 ```
gedit  ~/.config/nautilus/accels
```

Add this to a new line at bottom, don't be tempted to make it look like the other lines..
```
(gtk_accel_path "<Actions>/ShellActions/Up" "BackSpace")
```

restart nautilus or log out/in
