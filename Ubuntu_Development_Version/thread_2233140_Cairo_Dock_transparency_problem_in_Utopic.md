### Post by Cavsfan on 2014-07-08
Finally nailed it. When I purged the files and re-installed them the same dock appeared with the the custom items I had added to it.
So, I knew there was a config file I didn't delete.
Found this: [https://help.ubuntu.com/community/CairoDock](https://help.ubuntu.com/community/CairoDock) and added the ppa.
At the bottom I clicked on recurrent problems and it led me to this:
[http://www.glx-dock.org/ww_page.php?p=Recurrents%20problems&lang=en](http://www.glx-dock.org/ww_page.php?p=Recurrents%20problems&lang=en)
And deleted the folder **~/.config/cairo-dock/** 

And when I purged it I got this command from that page **sudo apt-get remove cairo-dock* --purge**

I had to edit the source line and enter trusty as there is no ppa for utopic yet.
But it still pulled in the cairo dock from the regular repositories because it had the later version.
