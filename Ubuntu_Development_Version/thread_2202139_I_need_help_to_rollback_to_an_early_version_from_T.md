### Post by grahammechanical on 2014-03-24
Ever since I started running the development branch I have always had a version of Ubuntu as a fallback OS. Up until now it has been 12.04. Soon I will be installing 14.04 as the fallback. If stuff I need does not setup in the development branch but it works in the stable/established Ubuntu that I am using as fallback then I am not so frustrated.

It is not a solution to this request but Ventrical has done extensive testing of the Btree file system (btrfs) as a means of rolling back an upgrade.  With the btrfs file system it is possibly to upgrade 12.04 to 14.04 and then rollback to 12.04. We seem to be a long way away from btrfs becoming the default UBuntu file system but if it ever happens an we get a decent GUI btrfs snapshot management utility then requests like this will be answered.
