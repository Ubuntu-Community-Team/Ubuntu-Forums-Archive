### Post by jbicha on 2017-09-08
The new design is resizable. It's faster to switch between panels. It doesn't have the problem with panel names being cut off at awkward places ( Region & Lan guage ).

As the [changelog]("https://launchpad.net/ubuntu/+source/gnome-control-center/1:3.25.92.1-0ubuntu1") points out, you can try out the old version by running *gnome-control-center-alt* . It's not possible to run both versions simultaneously.

I expect that the old version will be dropped before Ubuntu 17.10's release. It's only provided for testing and comparison now.
