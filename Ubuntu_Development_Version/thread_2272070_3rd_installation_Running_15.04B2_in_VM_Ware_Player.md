### Post by MikeMecanic on 2015-04-03
Reinstall 14.04.2 LTS alone

Testing Vivid Vervet in the free version of **VM Ware** (VMware® Player) **7.1.0 build-2496824**.  This installation was a prefect one: no error at all.  The only bug was that my computer does not support Intel VT. I try several options in VM Ware to install the AMD64 version but my mobo is too hold (2011).  So I download the 32 bit version of 15.04 (INTEL X86).  Works just fine, a bit more slower but it run smooth..

We must install the free version of VM Ware here: 
[https://my.vmware.com/web/vmware/free#desktop_end_user_computing/vmware_player/7_0](https://my.vmware.com/web/vmware/free#desktop_end_user_computing/vmware_player/7_0)

Install VMware Player Tools  according to this:  [http://kb.vmware.com/selfservice/microsites/search.do?language=en_US&cmd=displayKC&externalId=1022525](http://kb.vmware.com/selfservice/microsites/search.do?language=en_US&cmd=displayKC&externalId=1022525)


Just follow the default set-up in the terminal by answering or typing yes or no and press enter. That's it.  According to the bios setup that you have: Download 15.04 AMD or Intel x86 and install it in VM Ware.

This installation works fine and a video is available for installation purposes.  Made the post-installation without any problem.

All the best,
