All I did was...

```
vindsl@Zuul:~$ gksudo gedit /usr/src/nvidia-304-304.117/nv-acpi.c
```



To find the location of your 'nv-acpi.c' file, try...

```
$ whereis nv-acpi.c

```



[COLOR="#FF0000"]Comment[/COLOR]-out this section...

```
**[COLOR="#FF0000"]/*[/COLOR]**   if (pNvAcpiObject->notify_handler_installed)
**[COLOR="#FF0000"]* [/COLOR]**   {
**[COLOR="#FF0000"]* [/COLOR]**       NV_ACPI_OS_WAIT_EVENTS_COMPLETE();
**[COLOR="#FF0000"]*[/COLOR]**
**[COLOR="#FF0000"]* [/COLOR]**       // remove event notifier
**[COLOR="#FF0000"]*  [/COLOR]**      status = acpi_remove_notify_handler(device->handle, ACPI_DEVICE_NOTIFY, nv_acpi_event);
**[COLOR="#FF0000"]* [/COLOR]**   }
**[COLOR="#FF0000"]*/[/COLOR]**
```


Save.

Install 3.13-rc5.

Reboot.

Simple pimple!  ;)
