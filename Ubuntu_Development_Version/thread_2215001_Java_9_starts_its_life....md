### Post by slickymaster on 2014-04-04
Apparently, and according to Oracle, Java 9 and 10 will tackle big data, multi-language interoperability, cloud and mobile and will be shipped in 2015 and 2017 respectively. For the Java Development Kit (JDK) 10 or after, there's the intention of a unified type system that turns everything into objects and means no more primitives.

Here's a good read on it: [Forget about JavaOne. Let's talk about the feature list for Java 9]("http://www.theserverside.com/news/2240205870/Forget-about-JavaOne-Lets-talk-about-the-feature-list-for-Java-9")
