### Post by grahammechanical on 2013-12-22
Download a daily image ISO and install it. Any issues at this stage will most likely be with the installer program and not with the installed Ubuntu. Once Trusty Tahr is installed it is really quite stable. The intention is for daily images to be stable.

At this moment I have two issues with my install of Trusty. There is an intermittent bug that prevents the password field from accepting typed characters. I just click the keyboard layout indicator in the top panel and then I am able to type my password. The second is the appearance of a crash notification once the desktop loads. I close the dialog and away I go using Trusty Tahr without any issues. Which I think is remarkable work considering that a couple of years ago I would not have expected an alpha stage development release to be stable at all.

Regards.
