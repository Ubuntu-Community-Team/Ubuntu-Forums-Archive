### Post by grahammechanical on 2013-09-15
From the Appearance utility you can go back into all System Settings and in Software and Updates>Additional Drivers tab you can change video driver. Here are some commands

reset compiz
```
 dconf reset -f /org/org/compiz
```
reset unity
```
setsid unity
```
and
```
unity --reset-icons
```

might need sudo.
