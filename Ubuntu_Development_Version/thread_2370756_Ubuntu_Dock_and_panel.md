### Post by again? on 2017-09-15
> What will be included in Ubuntu is a full Dash to Dock, preconfigured according to the Ubuntu style, 
but with most settings not exposed. I’d like to remark that it is not a fork but an upstream collaboration with the Ubuntu Desktop team. 
Users will be able to install Dash to Dock to get access to the additional configurability, 
or chose the additional layer of quality assurance provided by the Ubuntu distribution.
[https://micheleg.github.io/dash-to-dock/release/2017/09/06/new-release-v61-gnome-3.26.html](https://micheleg.github.io/dash-to-dock/release/2017/09/06/new-release-v61-gnome-3.26.html)
