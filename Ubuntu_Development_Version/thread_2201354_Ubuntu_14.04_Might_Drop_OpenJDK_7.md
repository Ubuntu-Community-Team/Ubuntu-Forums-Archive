[TD]          **[FONT=verdana][SIZE=2][COLOR=black]Friday, January 24, 2014[/COLOR][/SIZE][/FONT]**:         [FONT=verdana][SIZE=2][COLOR=black]  After reports emerged that Ubuntu distributions might implement the  most recent version of GRUB, in another news, Canonical is skimming  through the necessity of removing the OpenJDK package in Ubuntu 14.04  LTS. Developers at Ubuntu are regularly sifting through the packages  that will stay and those that could be removed in the upcoming  distribution. It must be noted that the release of Ubuntu 14.04 LTS  (Trusty Tahr) is still pretty far ahead.[/COLOR][/SIZE][/FONT]

[FONT=verdana][SIZE=2][COLOR=black]“The foundations team have  been reviewing whether we should continue to provide OpenJDK as the  default Java implementation in Ubuntu; this package is quite a resource  drain on both the foundations and security teams from a SRU perspective  so demotion of openjdk7 to universe is being considered. Most Java-ish  things in main can be built using gcj; however Tomcat7 is not going to  work with gcj so would end up being demoted as well,” Ubuntu developer  James Page was quoted as saying. Canonical, therefore, won't have to  keep an eye on any security problems that need fixing in case OpenJDK is  not included by default in Trusty.    

Meanwhile, following  Canonical’s forking of the GNOME Control Centre many of the issues that  previously held back the upgrade have since been removed, giving a go  ahead to GNOME 3.10. The change will also mean that Ubuntu’s GNOME spin  will also ship with the newer desktop shell installed as default. Ubuntu  distributions might implement the most recent version of GRUB, still in  beta stage in Trusty Tahr. Canonical doesn't usually pick packages that  are still in the Beta stages but it just might make an exception for  the GRUB 2.02.

### Post by grahammechanical on 2014-01-24
I think that is is more accurate to point out that Grub 2.02 is beta as far as Ubuntu developers are concerned.  Ubuntu 14.04 will not be getting a beta version of Grub but a tested Grub 2.02. The particular package is named grub 2.02~beta2-2.  So, the beta designation is on the Ubuntu side of the name. Grub 2.02 is in the Trusty Tahr Proposed Respository for the purpose of being tested before it becomes part of Ubuntu 14.04.          Regards
