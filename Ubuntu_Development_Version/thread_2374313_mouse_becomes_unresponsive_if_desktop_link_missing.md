### Post by Vasu_Muppalla on 2017-10-14
If you create a link to an application on the desktop and later remove the app, the link remains on the desktop.

If you try to access the desktop link in any way, even with a right click to remove it, plasma stops responding. Well, the mouse at least- the keyboard still works so you can alt-F2 and logout.

Workaround is to delete the link file from command line.
