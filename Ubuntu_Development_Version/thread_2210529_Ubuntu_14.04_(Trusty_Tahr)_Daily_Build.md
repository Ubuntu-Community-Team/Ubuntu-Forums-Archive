### Post by grahammechanical on 2014-04-17
If it is not, then ask for your money back. :)

Think about it. The final release image of Ubuntu 14.04 is just a daily build. Except for the first few days at the start of the Trusty Tahr development cycle of 26 weeks there have been daily builds of the ISO image. The image that people will be downloading as 14.04 is the final daily build. Development work will now switch to the next release.

The above comment is true when applied to Interim releases but as 14.04 is an LTS release there is a slight difference. To keep 14.04 up to date with the latest Linux hardware stack there will be another 14.04 ISO image build in six months time - 14.04.1. Over the next two years there will be four builds of the 14.04 image. 14.04.1; 14.04.2; 14.04.3 and 14.04.4. These builds are called "point" releases. Six months later 16.04 LTS will be released.

And just in case you are wondering, your present install of 14.04 will also be updated with that newer code during the next two years.

Regards.
