### Post by Mateusz Stachowski on 2014-03-10
> **sacridex said:**
> Same question again.
Feature Freeze has already happened, is either one of qt5.2 or qtcreator3.0 to be expected in Tahr?

Both will happen in Trusty. Timo Jyrinki filled for FFe (Feature Freeze exception) which was granted by Steve Langasek.

[https://bugs.launchpad.net/ubuntu/+source/qtbase-opensource-src/+bug/1278329](https://bugs.launchpad.net/ubuntu/+source/qtbase-opensource-src/+bug/1278329)

He posted about that in ubuntu-devel mailing list: [Qt 5.2.1 final push]("https://lists.ubuntu.com/archives/ubuntu-devel/2014-February/038114.html").

> Thanks to everyone involved in fixing the numerous issues in the stackand apps, and welcome to join for the remaining testing / bug filing /
bug fixing at [https://bugs.launchpad.net/bugs/+bugs?field.tag=qt5.2](https://bugs.launchpad.net/bugs/+bugs?field.tag=qt5.2)


For those able to join the push, upgrade instructions are available at
[https://launchpad.net/~canonical-qt5-edgers/+archive/qt5-beta2](https://launchpad.net/~canonical-qt5-edgers/+archive/qt5-beta2) Note!
You will need to use the ppa-purge tool eventually, since it will not
be possible to upgrade from the PPA to the eventual archive versions.
