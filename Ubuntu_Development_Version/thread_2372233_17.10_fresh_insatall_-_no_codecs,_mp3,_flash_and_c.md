### Post by SeijiSensei on 2017-09-22
I used to install Flash using the "flashplugin-installer" package.  That works for Firefox but not for Chromium.  To install Flash for both browsers, I now enable the "partners" repository in /etc/apt/sources.list and use the "adobe-flashplugin" package instead.

For codecs, etc., I typically install SMPlayer and mpv.
```
sudo apt-get install smplayer mpv
```
You should then be able to watch most everything in SMPlayer.
