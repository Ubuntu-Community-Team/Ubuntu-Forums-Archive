### Post by wildmanne39 on 2018-03-08
You do not want to do a partial upgrade it almost always brakes your system. You probably received the partial upgrade offer because of a repository server issue, wait a while and try again or try another server but I do not recommend ever doing a partial upgrade, it is something that is done sometimes when testing the developmental version of ubuntu but it can still cause issues then as well and is not done very often even when testing the developmental version. 

I see you are running 18.04 but I still would wait.
