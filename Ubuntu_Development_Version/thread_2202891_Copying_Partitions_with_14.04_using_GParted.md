### Post by grahammechanical on 2014-01-31
> [COLOR=#000000]is there any problem updating 14.04 Alpha or Beta to a stable version once programs and PPAs are set up?[/COLOR]

Just do daily updates. You may find that the PPAs are for saucy and not trusty. The applications will install and run but it may be best to disable the PPAs after the programs are installed. and then use trusty PPAs when they are available. If you can install programs through the Trusty Software Centre then you will get versions that are already in the Trusty repositories. Unless you prefer the very latest versions of programs.

In regards to the other stuff that you are thinking about. I cannot advise you. No experience of doing that.
