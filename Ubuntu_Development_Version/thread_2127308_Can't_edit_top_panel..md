### Post by tad1073 on 2013-03-19
As far as removing "Ubuntu Desktop" from the panel, install gnome-tweak-tool from the software center, open it, click on "Desktop" and un-check "Have file manager handle the desktop". And as far as customizing the top panel, it is not possible unless you install gnome-fallback but I have had problems with customizing the panel.
