### Post by grahammechanical on 2013-07-19
Ubuntu 13.10 is very much like Ubuntu 13.04 as regards the system settings utilities. Nothing has changed. But in using 13.10 you may experience issues because 13.10 is under development. In system settings look for Appearance. That is where the background image is changed and it is the same in Ubuntu 13.04 and Ubuntu 12.10 and even Ubuntu 12.04.

If we select a static (unchanging background) then that image will appear as the background to the log in screen. If we select a rotating background (changes throughout the day) the the log in background image will be the standard Ubuntu background image which at the moment is the same image as in 13.04.
