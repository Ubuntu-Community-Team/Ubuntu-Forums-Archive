### Post by grahammechanical on 2015-01-06
I have not read your post. I do not have the time to work my way through posts that are difficult read.

Please remember that this is a user forum. This is not the place to make suggestions for improvements to Ubuntu. It is much too late to change things in the next version of Ubuntu (15.04). The decisions have already been taken.

The Unity user interface reached version 7 with the release of 14.04 and it was still at version 7 with the release of Ubuntu 14.10. And 15.04 will have Unity 7 as will 15.10 and 16.04. There are no plans to radically redesign the Unity user interface. Instead the intention is to give Ubuntu the Unity 8 user interface, which is Unity 7 re-written to run on the Mir display server that is being used in Ubuntu for phones and tablets. That stated intention is to give 16.04 users a login option of either Unity 7 running of the Xserver and the Compiz compositor or Unity 8 running on Mir. So, I think that it is safe to say that the design of Unity is here to stay.

Every six months or so there is the Ubuntu Online Summit. Users as well as developers can set up sessions to discuss whatever they want. It may be that a developer will become interested in an idea that someone puts forward and offers to work on the code. May be not. Check out some of the sessions.
