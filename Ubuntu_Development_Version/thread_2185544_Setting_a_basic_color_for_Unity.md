Hello,

Unity has no option to set the basic color for Dash, Notification bubbles, etc.
The launcher color can be altered with CCSM, but thats it.
In my opinion there should be a color chooser in the System Settings -> Appearance.
This should set the color for the Launcher, Dash, Notifcation Bubbles.
At the moment I'm using the Nylon Rainbow background image in 13.10 and the notifications are green, which looks horrible...
There is the very decent look of the top bar and the nice dark gray Ambient theme and then an ugly green bubble pops up when I want to change the volume...

Since Trusty will be an LTS version, it should be perfect. ;)
I Think it would be best if anyone could choose the desired color for himself.
But at least an option to set it to dark gray to fit into the Ambient theme would be enough I guess.
(Maybe light gray for Radiance theme)

### Post by grahammechanical on 2013-11-03
Perfect is not a requirement for an LTS. In my opinion the more choice the user has the less perfect the OS becomes. Unintended consequences can result when choices clash.  Is 13.10 different from 13.04 or 12.10 or 12.04 in this respect? These kinds of desires are often classed as "wish list."I remember when we had the choice to disable the Unity Plugin and many people choose to do so and then lost the desktop. We no longer have that choice. People still remove Unity without installing an alternative desktop before hand, with preditable results.We have Ubuntu with less and less choice and we have Kubuntu with plenty of choice (so I hear people say).
