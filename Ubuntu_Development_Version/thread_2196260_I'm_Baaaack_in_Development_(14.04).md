### Post by craig10x on 2014-01-05
@Hazzabin: The method i use to upgrade to the next version (whether in development or the final) is that IF you only have 1 ubuntu installed on your system, and you run the iso of the next version (for example let's say you are in 13.10 right now and would like to upgrade to 14.04 development) then run 14.04 daily build and the iso installer should offer 2 automatic options and 1 is to totally wipe out your 13.10 and install 14.4 and it will also offer you the option of UPGRADING to 14.04 (retaining most of your installed apps, settings and your data such as pictures, musics, doc, etc...).

THAT is what i have been using...when done in terminal or using software updater, it takes so much longer because it has to go to the internet for files...when done using the live iso installer, the files are RIGHT THERE ON THE DISC...so it only takes usually about 25 minutes or so (like a clean install) I find it works very well, is fast and brings over the majority of stuff (a few items will be lost but easy to add back quickly)...

Still, back up you important data on an external drive before you do it, of course...;)

If you have more then one ubuntu installed, the iso installer will not offer that option...then the only way you can do it is with software updater or terminal methods...
