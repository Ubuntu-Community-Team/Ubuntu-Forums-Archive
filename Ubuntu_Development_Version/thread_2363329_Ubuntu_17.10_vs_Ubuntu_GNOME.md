### Post by kansasnoob on 2017-06-09
There will be no Ubuntu GNOME 17.10. Ubuntu GNOME users of 17.04 or 16.04 will eventually be upgraded to Ubuntu once the default DE changes to GNOME. I suspect the soonest we'll see what the "new" Ubuntu will look like is sometime in August. Last I looked there still were Ubuntu GNOME 17.10 daily images but eventually they'll cease to exist.
