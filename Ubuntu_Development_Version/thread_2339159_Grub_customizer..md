There is a how to wiki in my signature that tells how to do it (In the green link the wiki is located in the 1st post). If you only have one system on your pc or laptop, it might not be worth the time.
But, when I install a new version of Ubuntu, the default grub takes 3-5 minutes to update and when a new kernel gets installed, it usually goes through update-grub 3 times.

So, if it's customized, it's takes 1 second to update. Who's got that much time for grub to update? Not me for sure.
I've been doing this for about 5 years and never had any problems.
