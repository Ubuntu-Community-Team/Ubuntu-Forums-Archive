### Post by ian-weisser on 2013-08-04
If you don't know how to do that, then you probably should not be using the pre-release (13.10) version of Ubuntu.
The pre-release version is intended for skilled testers willing to investigate breakage and submit bug reports. Data loss and failure-to-boot, while rare, is possible.

Desktop environments (Gnome, Unity, KDE, XFCE, etc.) can be added or removed in the package manager.
Simply add/remove the appropriate -desktop package.
For example, add Unity by installing the ubuntu-desktop package.
