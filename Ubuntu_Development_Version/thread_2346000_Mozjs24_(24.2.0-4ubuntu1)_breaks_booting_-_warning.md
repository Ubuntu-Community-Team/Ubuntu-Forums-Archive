### Post by harry332 on 2016-12-10
This is a warning to those that use proposed repo (zesty) at the moment.
Do not update mozjs24 to the version 24.2.0-4ubuntu1. Keep the working version 24.2.0-3.1ubuntu1.
Updating will result to booting failure with no tty's.
Only recovery mode from grub menu works.

If you already updated and broke your system, do this:
From recovery menu, choose "open network" and "go to root console".
From the console, do the command:
sudo apt-get install libmozjs-24-0v5=24.2.0-3.1ubuntu1
then answer "y" to do the downgrading.

Those using gnome-shell have mozjs24 installed.
