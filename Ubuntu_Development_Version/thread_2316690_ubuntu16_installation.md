It's great to see users install and comment on the latest development release, especially when it's an LTS which will be supported for the next five years. However, feedback posted here, which is directed at the developers of Ubuntu and it's many flavours, will certainly never reach them. If you think you have found a bug and want to help then you need to take some time by researching whether you are experiencing a problem that is already known or if it isn't then reporting that bug through the recognised channels, i.e. [Launchpad]("https://launchpad.net/").

As you're seeing a problem with the Turkish language it may be up to you to provide the required feedback in order for the issues to be fixed as most of us here won't be seeing such problems being speakers of more widely used languages than Turkish.

Please take a look at [ReportingBugs]("https://help.ubuntu.com/community/ReportingBugs") on the Ubuntu Help Wiki. I think the section [Reporting non-crash hardware and desktop application bugs]("https://help.ubuntu.com/community/ReportingBugs#Reporting_non-crash_hardware_and_desktop_application_bugs") is the most relevant to what you are seeing..
