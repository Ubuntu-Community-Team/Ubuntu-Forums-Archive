### Post by oldrocker99 on 2018-03-22
Near the end of installation, the system deletes all the package files which installed the software, and are no longer needed once the system is installed.

In other words, it's normal.

And "AMD64" is a naming convention, named after the Athlon64, the first commercial 64-bit processor. 32-bit programs are named "i386," after the Intel 386 chip, the first commercial 32-bit processor. AMD64 can be installed on any 64-bit system. you installed the right one.
