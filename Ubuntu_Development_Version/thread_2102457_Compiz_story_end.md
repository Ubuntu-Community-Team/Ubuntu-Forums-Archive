### Post by Stinger on 2013-01-09
I've been digging round a bit and as cariboo907 pointed out and [here on Phoronix]("http://www.phoronix.com/scan.php?page=news_item&px=MTI1MTM"), it seems that compiz isn't in such a bad shape after all but you have to use the  [ppa:smspillaz/compiz-experimental ]("https://launchpad.net/~smspillaz/+archive/compiz-experimental") to get all the improvents.

As you can see in this [article from Phoronix dated December 26, 2012]("http://www.phoronix.com/scan.php?page=news_item&px=MTI2MTY"), Ubuntu 12.10 Now Unredirects Fullscreen Windows.

This was done in Mutter back in 2011 as you can [read about here]("http://www.phoronix.com/scan.php?page=news_item&px=OTg1Mg")
> Unredirect override-redirect fullscreen windows, such as full-screen 3D games to avoid any performance impact [Adel; #597014]

I found this on the [KDE blogs from 24 Aug 2008 by Lubos Lunak]("http://blogs.kde.org/2008/08/24/kwins-composite-performance-part-ii")
> Oh well. Second improvement is unredirecting fullscreen windows

Unredirecting is not the whole truth as you can see in this [benchmark made by Owen Taylor back in June 2011]("http://blog.fishsoup.net/2011/06/13/benchmarking-compositor-performance/"), but if you use full-screen 3D it will improve performance.
The Compositing manager benchmark tool is Owen's own creation, [Xcompbench]("http://git.fishsoup.net/cgit/xcompbench").
