### Post by grahammechanical on 2013-11-07
A few minutes ago I installed through the software centre and I could not find anything in the dash to launch it so I tried unity 8 in the terminal and I got: compiz (core) - Error: Failure to load plugin 8. I never got Unity 8 loading in saucy either.

Update: When we install unity 8 shell through software centre we get unity8; unity8-fake-env; unity8-private; unity8-autopilot; libunity-core-6.0.8. That last package may have already been present. So, all the necessary stiff should be there (I think)

Now, it gets interesting for I now have two network manager icons. The usual one and a slim menu version with on/off sliders for wired and wireless connections. I also notice in the Dash>Applications two icons for System Settings. The usual System Settings and another one that is clearly a mobile version. At it "home" page it has, among other things, "About this phone." If we are moving through the options will find find a back buttom by clicking just above the bottom border and sliding upwards.

If we keep the Dash at Dash Home and filter by Applications then we see a load of what I think are Ubuntu mobile apps. I am not sure if they are installable.

Update 2: First step to installing these mobile apps is to login to our Ubuntu One account with the Touch System Settings. Second step is to try to install something. And it failed, I think due to trusty not understanding click packages.

Anyway, it seems that Dash>Applications is where our desktop applications are to be found. And Dash>fliter by applications is where the mobile platform apps are to be found for installation. I have yet to successfully install one.
