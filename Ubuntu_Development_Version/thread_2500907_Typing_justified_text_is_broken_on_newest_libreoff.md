### Post by Wise Ferret on 2024-09-15
After last update, when typing a paragraph aligned to "justified" and reaching approximately the last third of the line, the line starts to stretch while typing the first letter in a word and un-stretch back when typing a space. This makes it difficult to type justifieded text. On former versions, each line would be justified only after the user finished typing it and is typing the next line.
If you experience the same bug, please confirm it in the [bug report]("https://bugs.documentfoundation.org/show_bug.cgi?id=162977") I created.
