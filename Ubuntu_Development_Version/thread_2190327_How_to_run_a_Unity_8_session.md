### Post by grahammechanical on 2013-11-27
We can install Unity 8 shell. It is in the software centre but what you seem to be asking for will not be available for many months. In fact until 14.10 or later.

[http://www.youtube.com/watch?v=j8RDj-SQE1U&feature=c4-overview&list=UUm7OifwnZoMCChidCJZQruQ](http://www.youtube.com/watch?v=j8RDj-SQE1U&feature=c4-overview&list=UUm7OifwnZoMCChidCJZQruQ)

Ubuntu 14.04 will be released using Unity 7. There is talk of giving users an option to install Unity 8 but I do not think it will be any different to installing Unity 8 Shell.

This is Mark Shuttleworth's keynote address at the recent vUDS.

[http://summit.ubuntu.com/uds-1311/meeting/22027/intro-by-jono-bacon-keynote-by-mark-shuttleworth/](http://summit.ubuntu.com/uds-1311/meeting/22027/intro-by-jono-bacon-keynote-by-mark-shuttleworth/)

Unity 8 is designed to run on Mir. So, Unity 8 will not be part of a Ubuntu distribution until Mir replaces Xserver, LightDM and Compiz and the goal for that is between the release of 14.04 and the release of 15.04. And Mir+Unity 8 will not be backported to 14.04 even in a point release.
