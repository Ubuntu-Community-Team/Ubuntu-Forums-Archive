### Post by zika on 2013-11-26
We all know that worm is always working because we can not easily see what is in -proposed repository. What is going to be broken etc. OK, we could just do update/upgrade with proposed enabled and see what is on the menu. But would it be nicer to have it all without editing /etc/apt/sources.list often? Yes. After some roaming I've found that page was already written with (almost) all tools needed and I just had to fill some gaps and... So, to share that with You I give You: [https://wiki.ubuntu.com/Testing/EnableProposed](https://wiki.ubuntu.com/Testing/EnableProposed) ... I just want to thank all users from [https://wiki.ubuntu.com/Testing/EnableProposed?action=info](https://wiki.ubuntu.com/Testing/EnableProposed?action=info) that made this possible.
That is even more important this opens many new possibilities in working with PPAs and it will surely make my testing (once I do embark in TT i.e. in U+1 again) much more comfortable. This is not something edgers type, it's been available for quite some time but now I've found nice way to share that with You and not to hurt my laziness...
