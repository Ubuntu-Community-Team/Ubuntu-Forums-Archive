### Post by grahammechanical on 2015-07-10
If we have problems with our operating system then upgrading to a newer version of Ubuntu may not necessarily fix those problems. And it many cases it does not fix those problems. A fresh install would. But there is something to consider whether upgrading or doing a fresh install:

Newer versions of Ubuntu come with newer proprietary video drivers which often lack support for older video adapters. I know this because my video card is no longer supported by the latest proprietary video drivers that come with 15.10, which I am using. In my opinion, it is always best to revert to using the open source video driver before upgrading. And if doing a fresh install to not tick the box "install third party software" as that will bring in a proprietary video driver."

The video driver is loaded long before LightDM tries to do its stuff. The problem does not seem to be down to the login screen manager because changing to GDM did not fix it. I am still thinking it is a video driver problem.

What was it that made you think that Nouveau was not installed? Grub menu Advanced options for Ubuntu>Recovery Mode>Resume should get us to a working desktop using an open source video driver called llvmpipe. Does it?

Regards.
