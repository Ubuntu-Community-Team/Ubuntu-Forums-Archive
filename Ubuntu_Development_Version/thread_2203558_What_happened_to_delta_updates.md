### Post by slooksterpsv on 2014-02-04
With all the updates and upgrades every 6 months, what ever happened to delta patches/upgrades? For those that don't know what I mean - they're patches that update only parts of the code/files that need to be updated.

Example:

I have a 500MB file and a newer version comes out with only 2 lines changed in the code/added/removed/whatever. Instead of downloading that 500MB file again it downloads a delta patch that just updates the file where it needs updated at.

It may not have been Canonical/Ubuntu talking about delta updates, but I thought they were. Smaller downloads for updates and upgrades.

### Post by grahammechanical on 2014-02-04
There is another virtual Ubuntu Developer Summit (vUDS) coming up. If Delta updates are not on the agenda, then we can safely say in answer to your thread title, "Nothing." At least for another six months.

For more than a year now attention has been focused on the Ubuntu Mobile platform. Work is still being done on that but attention is slowly switching to converging the Ubuntu Mobile platform with the Ubuntu Desktop platform. So, attention is elsewhere.

Ubuntu for phones has image updates and not apt-get updates as on the desktop. with phones and tablets there is a need to reduce data download sizes. I wonder if Delta would be a handy technology for image updates. I have seen some speculation that image updates may in future come to the desktop. A lot of things are up for re-evaluation.

We can also keep an eye on the Ubuntu On-Air calendar and if there is a Ubuntu Engineering session we get the opportunity to use IRC to ask questions. 

Regards.
