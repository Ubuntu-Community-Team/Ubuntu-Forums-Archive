### Post by dawgfan16 on 2014-04-11
System just had me upgrade and it went tremendously well. There have been no issues and my system has been running pretty snappy. I have installed several DE's to try and Cinnamon is my preferred desktop other than the default Unity.
This has been one of the smoothest and problem free testing periods I have experienced.
Granted, my Linux experience is limited to the last 7 months but I have tried so many distros in that time and have learned so much that I feel like it has been longer.
I have completely removed Windows from all of the desktops and laptops in the house and I am so glad that I made the change.
Freedom to have a system look and work the way you want or need it to is so refreshing.
When the Final Release comes out next week, I plan to start over with a fresh install instead of upgrading so I can rid myself of all of the excess I have put on my current laptop that I use for testing and make it my go to system. Then I will convert one of the other laptops laying around as my new test sytem.
I must say that I think the devs can be proud of the work they put into making such a stable release.

Dawgfan16
