### Post by grahammechanical on 2014-03-02
No. Read about apt-get

[https://help.ubuntu.com/community/AptGet/Howto](https://help.ubuntu.com/community/AptGet/Howto)

Synaptic Manager, Software Updater and Ubuntu Software Centre run the apt-get commands. I find that Software Updater presents information about the packages in a more easily understood way but running the commands in a terminal sometimes gives more useful error information. Synaptic will identify if a package is ready to be updated by marking the package with a grey icon and updating that package may resolve a problem.
