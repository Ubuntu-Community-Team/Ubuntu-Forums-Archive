### Post by ventrical on 2014-02-21
I have been doing some of my usual experiments with swapping drives and USB sticks  on different systems to check the portability of Ubuntu and I notice that now Ubuntu (trusty) is more correctly determining  graphical adapter drivers during the boot sequence and not borking machines as it was if a swap was made from an install that was done on an older PC (Dell D) with Intel Graphics to a machine that had nVidia or  ATi.

 Basically, it appears that  a hard install is now jockeying graphical drivers as if it were a startup .iso!! which I think is good news and streamlines Ubuntu even more.

  Trying not to speak too soon .. but great work on this devs :)
