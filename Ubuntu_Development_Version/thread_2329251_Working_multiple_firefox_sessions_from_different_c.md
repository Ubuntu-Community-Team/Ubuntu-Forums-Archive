### Post by ventrical on 2016-06-29
I discovered  that multiple open sessions of firefox can be run from different containers from the Xapps tool. So this process allows to run apps from different containers concurrently. I am assuming this will be a model for the Unity8 desktop default which is a real bonus.

You can try the test yourself if you have a working unity8 desktop. Just install firefox to two different containers.  Go to Xapps and open one instance from one container and then go to another container that has firefox installed and open that session.
