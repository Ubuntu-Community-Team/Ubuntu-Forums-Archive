### Post by P-I H on 2019-04-03
There is a gnome extension.

**Transparent Top Bar by zhanghai **
Bring back the transparent top bar when free-floating in GNOME Shell 3.32.
This basically comes from the feature implementation removed in GNOME Shell 3.32, and I modified the code a bit to make it an extension. Enjoy!
Extension Homepage [https://github.com/DreaminginCodeZH/gnome-shell-extension-transparent-top-bar](https://github.com/DreaminginCodeZH/gnome-shell-extension-transparent-top-bar)
