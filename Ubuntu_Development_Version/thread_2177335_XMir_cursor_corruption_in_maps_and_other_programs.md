### Post by Mateusz Stachowski on 2013-09-28
This [bug]("https://bugs.launchpad.net/xmir/+bug/1216708") has been reported already and the devs think it's related to buggy X software cursor and is restricted to Nouveau. However there is a second [bug]("https://bugs.launchpad.net/xmir/+bug/1211060") report for this and the person that reported it uses Intel graphics. If anyone with Intel or Radeon graphics card experiences this cursor corruption on XMir it would be highly apprieciated for reporting it in that second mentioned bug.

The easiest way to reproduce it is to open one of this maps in your web browser:

Google Maps
OpenStreetMap
Bing Maps

and hover over it to see if the cursor displays correctly.
