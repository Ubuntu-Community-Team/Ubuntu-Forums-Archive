### Post by P-I H on 2018-08-11
My Logitech RX 1000 started to behave strange. Instead of single click I sometimes got a double click.
Bought a new Steelseries Rival 110, but it had the same problem.
Changed to an old Microsoft Laser Mouse 6000, which behaves OK.
Learnt that there are mouse settings in three places, Settings/Device/Mouse &Touchpad, Universal Access and gnome-tweaks/Keyboard &Mouse.
Manged to get the Rival 110 to work acceptable, but not perfect, by changing settings in all these places.

Perhaps related to the dpi values
Rival 110, 7200 dpi
RX 1000, 1000 dpi
Laser Mouse 6000, 800 dpi
