I am using lm-sensors, fancontrol and psensors. Amdgpu includes a new sensor.

This is the etc/fancontrol I'm using. But the values are of course different depending on the motherboard.
```
# Configuration file generated by pwmconfig, changes will be lostINTERVAL=10
DEVPATH=hwmon0=devices/pci0000:00/0000:00:02.0/0000:01:00.0 hwmon1=devices/platform/it87.552

FCTEMPS=hwmon1/device/pwm2=hwmon1/device/temp2_input
FCFANS=hwmon1/device/pwm2=hwmon1/device/fan2_input
```
