That only works for certain keyboards. I don't have the quick settings for my illuminated keyboard, but my illuminated keyboard has its own button to change its brightness. The new feature is more intended for keyboards that do not have their own brightness controls.


> **Claus7 said:**

5) How do you use the App center? I tried to install in a fresh minimal installation gnome shell extensions for example. I typed extension and by choosing debian I get the following error: AppLocalizations.delegate failed to load...

Like the older Ubuntu Software and GNOME Software apps, the App Center only shows apps that provide AppStream metadata. The gnome-shell-extensions package does not provide this, in part because it's assumed that you can install GNOME Shell Extensions a different way. In general, I recommend GNOME Shell Extension Manager.

But for this specific usecase, installing the package with synaptic or with apt in a terminal is good too.
