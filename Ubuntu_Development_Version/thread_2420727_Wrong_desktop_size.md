### Post by P-I H on 2019-06-10
Found what caused the change.
I clicked the "Adjust for TV" setting in "Settings" to see what happened. I didn't recognize any change, but after boot I saw that the desktop was smaller.
Clicked the "Adjust for TV" setting again and the desktop size is back to normal.

Tried this again and this time the desktop size changed, perhaps I didn't noticed that last time.
However the user interface is a little bit strange. To revert back I have to close and open "Settings" and then click the "Adjust for TV" setting again.
