### Post by Vaphell on 2014-04-15
just for kicks i wrote a script that allows for easy selection (guified checklist) of launchers that need to be copied to desktop. It uses yad (because zenity has been trash in last few years) which yad needs to be installed from ppa [https://launchpad.net/~webupd8team/+archive/y-ppa-manager](https://launchpad.net/~webupd8team/+archive/y-ppa-manager)

By default script lists all .desktop files in /usr/share/applications but you can narrow it down by providing a string of chars that needs to exist anywhere in the file name.

```
#!/bin/bash

pattern=$1

apps=/usr/share/applications
dest=$( awk -F'=' '/XDG_DESKTOP_DIR/ { print $2; }' ~/.config/user-dirs.dirs )
eval "dest=$dest"

yad_opt=( --geometry 800x600
          --list --checklist --multiple
          --column="select" --column='file' --column='name'
          --print-column=2 --separator='' )

yad_data=()

for f in "$apps"/*"$pattern"*.desktop
do
    name=${f##*/}; name=${name%.*}
    desc=$( awk -F'=' '/^Name=/ { print $2; exit }'  "$f" )
    yad_data+=( '' "$name" "${desc//&/and}" )
done

while read n
do
    f=$n.desktop
    cp -t "$dest" "$apps/$f" && chmod 700 "$dest/$f" && echo "$f copied to $dest"
done < <( yad "${yad_opt[@]}" "${yad_data[@]}" )
```
