### Post by grahammechanical on 2016-01-07
> Is Snappy Personal (desktop) going to be a rolling release?

It all depends on what we define as "rolling."

It seems that Ubuntu Personal will be based on Snappy Ubuntu Core. And Snappy Ubuntu Core, like the Ubuntu phone, has transactional updates. Ubuntu Phone gets Over the Air (OTA) updates every 6 weeks or so. Would that qualify the Ubuntu phone as being a rolling release? People would argue over something like that.

When I was experimenting with personal_x86.img (which called itself Snappy Ubuntu Desktop Next) when we logged into tty1 to update the OS it updated from the rolling/edge channel. So, updating Personal would bring in the latest daily build image.

I am guessing that when Ubuntu Personal is offered for normal use it will update similar to the Ubuntu Phone. The install will be set to update from the stable channel and will get updates/upgrades at a regular interval. It will not have the present 6 monthly release cycle.

I also think that it will be possible for experimenters & testers to set Personal to update from a channel that will bring in a daily build of the OS. Similar to what is spoken of here.

[https://developer.ubuntu.com/en/start/ubuntu-for-devices/image-channels/](https://developer.ubuntu.com/en/start/ubuntu-for-devices/image-channels/)

Ubuntu Personal will be updated/upgraded in a way that is very different to the present method used on what might now be called Ubuntu Traditional. I think that this much is certain.

Regards.
