### Post by grahammechanical on 2016-03-10
Is Xubuntu still working? If there was any irretrievable harm done you would have noticed it by now.

That Partial Upgrade offer is something we get from time to time when we are using the development version. The dialog lists 4 reasons why we may get that offer. The fourth reason is "Normal changes of a pre-release version of Ubuntu."

We do not recommend accepting a partial upgrade because in the past we experienced breakages. I experienced breakage when I accepted a partial upgrade when I first started using the development release. I have not accepted a partial upgrade since.

Here is a related topic

[http://ubuntuforums.org/showthread.php?t=2316479](http://ubuntuforums.org/showthread.php?t=2316479)

It is all a matter of opinion. A few months ago I found Xenial Xerus boring so I enabled the proposed repository to get a bit of excitement but Xenial is still boring.

The point is, we should expect to reinstall if we are using the development version. We should not be running the development version as our only OS. We should have our data on a separate partition. We should not expect applications installed through a PPA to work on the development version or even to be available. Likewise with software downloaded from some web site.

When I reinstall the development version it is to easily clear up some mess that I have created. Oh, and another thing. The development version has crash reporting enabled by default. Expect to see crash reports. We can either report the crash or ignore it. They often go away when the developers push out a fix.

regards.
