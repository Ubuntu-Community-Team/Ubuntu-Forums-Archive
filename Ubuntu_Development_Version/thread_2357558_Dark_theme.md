### Post by jbicha on 2017-04-03
Install gnome-themes-standard and switch your theme to Adwaita. (Adwaita is GNOME's default theme.)

Then use something like GNOME Tweak Tool to turn on Global Dark Theme.

Another option in 17.04 is greybird-gtk-theme, which is very similar to Adwaita Dark (it also respects the Global Dark Theme settings) because it's heavily based on Adwaita Dark.
