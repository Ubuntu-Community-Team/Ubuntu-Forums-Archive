### Post by grahammechanical on 2016-03-05
We do not find Ubuntu 16.04 on Ubuntu.com because it has not yet been released. The scheduled release day is 21st April.

[https://wiki.ubuntu.com/XenialXerus/ReleaseSchedule](https://wiki.ubuntu.com/XenialXerus/ReleaseSchedule)

We can install its development version (Xenial Xerus). Images are built daily and are to be found here

[http://cdimage.ubuntu.com/ubuntu/daily-live/current/](http://cdimage.ubuntu.com/ubuntu/daily-live/current/)

You most likely need xenial-desktop-amd.iso. There are also daily built images for flavours of Ubuntu. If you are interested.

Regards.
