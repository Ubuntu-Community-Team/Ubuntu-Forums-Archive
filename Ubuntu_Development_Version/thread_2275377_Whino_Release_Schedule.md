Whino Release Schedule

1. April 28th/2015  Super tool chain tool box base files to establish
a container to put the tool chain in when it is ready to be released.

2. May 10th/2015  Mir, xMir and xOrg meta-patch library files to unify
all current terminal servers.

3.May 20th/2015  SystemD/Upstart pre install/upgrade sniffer to jockey the most suitable init for any given machine.

4.June 2nd/2015   New Siesta app files uploaded for testing. Siesta will attempt to run micro kernels concurrently in terminals with new mini-desktops across various form factors.

5.July 30th/2015  After long break more base files and pre-alpha iso release of Whino

6.August 10th/2015 Release of new flavour, Ubuntu Cli, for Ubuntu afficianados.;)

7.August 20th/2015 Release of  Whino Alpha 2 for all variants.  

8.September 3rd/2015 Experimental Unity9 to be release for testing. UDC.

9.Sept 15/2015 Release of Whino Beta 1.Also .. the long awaited release for testing of Ubuntu Rain networking concept. If you think Ubuntu cloud was sexy, wait until you try Unity in the Rain. One cloud can produce trillions of drops of rain. As so as the network can be expandable when it is condensed so it can be condensed when it is expanded. Rain networking  will revolutionize computer networking as we know it.

10.Sept 30th/2015 Whole bunch of other stuff :)

11.Oct. 5th/2015 Release of Whino Beta 2

12. Oct 10th/2015 Feature freeze.

13. Oct 23/2015 Final Release.
