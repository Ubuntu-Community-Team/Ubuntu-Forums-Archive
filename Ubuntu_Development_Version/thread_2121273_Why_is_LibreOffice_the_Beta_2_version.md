### Post by Budovi on 2013-03-01
No, it is not that weird. Every software has some sort of development cycle, starting from development and ending with stable release. Usually, latest release candidate (RC) is released as stable version, maybe with changed version number.

Numbering of development versions differs. LibreOffice had released beta releases for 4.0.0 as "4.0.0 beta2" for example, followed by "4.0.0 RCx" and then RC3 actually became stable 4.0.0

Ubuntu Raring has outdated beta2 version in repository.
