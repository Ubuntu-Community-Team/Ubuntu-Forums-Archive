If you are running Ubuntu Gnome, all you have to do is install ubuntu-desktop, and you will be have a choice at login to go to Unity or Gnome. If you're running stock Ubuntu w/Unity, all you have to do is install ubuntu-gnome-desktop, and you will be able to choose a Gnome Shell session. Quite easy, either way.
