Unless I put them there manually, all my ppas are in the folder /etc/apt/sources.list.d/ listed under names such as
google-chrome.list. and so forth.
But the concept is the same.
you would run the above nano command, but extend the file path from just /etc/apt/sources.list to /etc/apt/sources.list.d/some-ppa.list.
The lines in the file are the same as they would look in the regualr sources.list file.
So as suggested you would just add a # to the beginning of the line to comment the line out, making the system skip it/ignore it.
