### Post by kilzzz on 2018-03-05
Not sure where to post this bug but I ran into some problems installing the Nvidia drivers. I am running the Ubuntu Mate 64 bit version of Bionic.

I tried to install the 340 version through the Additional Drivers, it failed.
I opened up synaptic and tried but it failed. I tried again but opened the details while it was installing. It was failing because of overwrite issues of the libraries.
I opened up a terminal and used apt and the -f option to force the overwrite. Not something I like to do, but from experience a failed video driver most of the time borks the system on reboot. I dreaded reinstalling everything I had done up to that point. 

So far the 340 Nvidia driver is working fine after that and its been a few days, Bionic is nice and very stable for an Alpha. No usual automatic bug reporting started, so I am not sure where to report it.

Only 1 bean? Looks like the forum was reinstalled at some point since I left after Unity was released.
