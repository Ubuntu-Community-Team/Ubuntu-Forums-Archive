> **userhv said:**
> Why is python 2 still supported in version 20.04? He believed that it would be removed.

"Will Python 2  still be supported for the whole 20.04 cycle at the end of the day"?
I would use the word tolerated rather than supported, and tolerated mostly for pragmatic reasons as some application developers for packages considered important, even with years of advanced notice, simply never did the work to convert to python3 (the same old argument of prioritizing the itch for a new features or bug fixes over working in the background on replacing code that worked with new code that did the same thing). So if something absolutely positively needs python2 it will be possible to use it, but you may need to jump through additional hoops as many python2 packages will not longer be directly available in 20.04
