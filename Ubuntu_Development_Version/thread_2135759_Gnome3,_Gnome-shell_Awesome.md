### Post by kurt18947 on 2013-04-16
> **sp-1070 said:**
> First off yes get of kde and go to gnome-shell because it just works sweeter or nicer or something i cant really define.
No i have not looked at extensions i will now though :P so post some you like most.


[LIST]
[*]Alternative Status Menu - fixes the oversight by gnome devs. to show separate suspend and shutdown/restart menu items.
[*]Lock Keys - Mandatory IMO for keyboards that don't have numlock and capslock indicators.  Wireless keyboards and some portables.
[*]Weather
[*]Places - a quick means to access file manipulations.  There is another which is not yet available for gnome 3.6 called all-in-one-places which I think is a little better.
[*]Activities Configurator - I'm not 100% certain about this one.  It has some nice features but I'm not sure if it can produce unintended consequences.
[*]Not a gnome extension but I find Tint2 panel from the repositories very useful.  It includes a hidden .config file in each user's /home folder to set things autohide, size, transparency etc.  There is an extension to add a lower panel with a desktops and windows switcher that I haven't installed.  The last I knew, it did not offer autohide which is useful on smaller screens.
[*]WorkspaceBar - adds a click-on-it workspace switcher to the upper bar.
[*]Remove Accessibility - hides the accessibility icon.
[*]There are a number of choices to add more traditional app menus if you want.  For now I just use the default launcher and the search function.
[/LIST]

Those are my choices for now. 13.04 shows 24 pages of extensions at 10/page, have fun.

### Post by montag dp on 2013-04-16
> **dino99 said:**
> Seems that gnome is following the unity designWasn't Gnome first?  Anyway, it is pretty similar, but Gnome takes care of the things I don't like about Unity:

-It's very quick and easy to switch applications.  Just click the Super key and all your windows are shown in an overview.  It's much better for me than switching with Unity's side bar thing, which IMO is clunky to use when you have many windows open, and especially with multiple instances of the same application.  Actually, I like the Gnome way better than the old taskbar way in most cases, which is one reason why I'm thinking about switching from KDE.
-The launcher also shows applications sorted by category, like a traditional menu, while still having the search functionality of the Dash in Unity (and without any adware!).  So you have the best of both worlds: a quick search-driven method to open programs fully integrated with a menu.  KDE has similar functionality, although it is more menu-oriented first instead of search-oriented.

I suppose those are the main things.  One thing I do like about Unity is the global menu at the top bar. In fact, with my current KDE I have a top bar with global menu + bottom bar with task manager.  I like the global menu because it saves vertical space on my laptop.
