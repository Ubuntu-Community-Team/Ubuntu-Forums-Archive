### Post by grahammechanical on 2015-10-16
We will know more if we join the Ubuntu Online Summit - 3rd to 5th November.

[http://summit.ubuntu.com/](http://summit.ubuntu.com/)

I expect Ubuntu 16.04 to be a traditional Ubuntu built from debian packages and updated package by package using apt-get. It will have the Unity 7 UI. I am not so certain that there will be a traditional Ubuntu 16.10 or a 17.04. It all depends on how the development of Ubuntu Snappy Personal goes and whether it is intended to be released as an ISO image for user installation in the traditional manner or just as images for OEMs to use for pre-installing Ubuntu Personal onto laptops.

And then there is Ubuntu Pocket Desktop. This was mentioned in someone's blog post but apart from the web site holding the images I cannot find any information as to what it actually is and what it is for.

I did download it and put it on a USB stick. It is built on vivid code and has a partition set up similar to Ubuntu Personal but without a boot loader. So, it did not load. It is a very small file size. It will fit on a 1GB USB stick whereas Personal needs at least a 10GB stick. But images are still being built. And there are X86 images.

[http://cdimage.ubuntu.com/ubuntu-pd/vivid/daily-preinstalled/current/](http://cdimage.ubuntu.com/ubuntu-pd/vivid/daily-preinstalled/current/)

It seems that Canonical has several experimental projects on the go. Any of which could go far or could cease to exist over the next year or two. That is the nature of Research and Development.

Regards.
