### Post by go2dell on 2010-03-18
THANKS for your efforts in helping to make Ubuntu better!  Since Lucid will be an LTS release, squashing bugs is especially important this time around.

The page describing known problems and how to report further bugs with the Alpha 3 release is [here]("http://www.ubuntu.com/testing/lucid/alpha3").  When Beta 1 is released later today, March 18, the page for that will be [here]("http://www.ubuntu.com/testing/lucid/beta1").

It appears the problem you experienced has already been reported as [Bug #538304]("https://bugs.launchpad.net/ubuntu/+source/adium-theme-ubuntu/+bug/538304").  Please check that bug to see if it describes your experience and add any information you have that might be helpful.  If that isn't your bug, then please [report a new bug]("http://help.ubuntu.com/community/ReportingBugs").

Have fun testing!
