### Post by zengeos on 2009-12-17
My initial impressions

The good:

Wireless networking seems to be significantly improved. At least the signal strength in Lucid reads about 30% stronger than in Karmic.

Most programs I have tried work ok.

After downloading the daily 12/16 version I was able to install to my hard drive fine.

Boot time is noticeably decreased...abut 25-30% in my estimation. However, it is not as smooth/seamless as Karmic's boot progression.

The X.Org video drivers actually work pretty well in the default install for Lucid and allow for some screen effects.

Now for some bads/issues.

This is related to the 12/16 build, NOT the Alpha release, so I have not reported through the bug report system.

ATI flgrx driver will not install. Once I downloaded and activated it I got various messages that my resolution was not configured and, several other warnings, upon clicking through those, I ended up with a black screen.

The backlight on my Acer 6530 laptop is not turned on in Lucid; it is turned on by default in Karmic. Please set the backlight to on in the initial install.
