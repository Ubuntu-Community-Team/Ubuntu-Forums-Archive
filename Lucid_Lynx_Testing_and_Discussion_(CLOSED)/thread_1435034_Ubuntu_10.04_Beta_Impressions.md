### Post by chessnerd on 2010-03-21
I'm making this post while testing 10.04 via LiveCD and so far it's worked well. I was worried that I might not like the large number of aesthetic and functional changes that Canonical has made to Ubuntu, thinking I might jump ship to a different distro, but I found Lucid to be, overall, a good OS.

I apologize for the length...

Here are my impressions: 

**Interface** - My greatest fear when it came to Lucid was that it would have a terrible interface. Here is what I thought...

* Window buttons, everyone is talking about them and I was concerned. Having them on the left still seem like a bad idea, but in the half hour that I've been testing I've already gotten used to them. Not a big deal.

* I don't really care for the default desktop background, but that's easy to change.

* I like the overall look-and-feel of the system, although I prefer the &#8220;Radiance&#8221; theme to the default &#8220;Ambiance.&#8221;

* I was pleased with the seamless Mint-like look of the window boarder and the menu bar. It looks very professional. But I was *very* impressed with the fact that many of the default applications allow you to grab and move the window by grabbing the part of the menu bar that doesn't have menus. It really makes the window boarder and menu bar feel like they are truly integrated, although it would be nice is this was the case with all of the applications and not just some of them.

* I don't care for the fact that there is now an Evolution icon. I never use Evolution and generally work to remove it as thoroughly as I can. Thankfully the icon seems easy to get rid of via Synaptic. Also, Empathy is still integrated into the indicator applet. I'm sure this is great for users that keep these applications, but I do find it frustrating that they are getting deeply integrated into the Ubuntu system when I don't use them.

* There is now only one workspace by default. I find this a little disappointing, but the problem is easy to rectify. My only qualm is that users new to Linux might not know how to set up with capability which was one of the first things that impressed me about Linux...

My grade for the interface: B-

**Applications** - The GIMP wasn't the only thing missing from the default install...

* I don't use the GIMP very often, so I don't mind that it is no longer present by default (although I will probably install it), but I was surprised that there was no painting program at all installed by default (unless you count OpenOffice Drawing). Still, this isn't a huge issue.

* Most of the games are now gone by default. Only Solitaire, Mahjongg, Mines,  Sudoku, and Gnometris (now called Quadrapassel) survived from the old versions, but I am integrated by the inclusion of a game called &#8220;gbrainy&#8221; which looks to be a trivia/puzzle game suite.

* There is now a video editor included by default and it happens to be Pitivi, which I already have as my editor under Karmic. While Pitivi is the most stable editor I've used in Linux, it is not very feature rich, but should be sufficient for most casual users.

* Still no default Flash support...

* The Software Center is looking really good now. It is much faster than it was in Karmic (even on a LiveCD) which I'm very pleased about that. I like the Featured Applications link and the new categories. There is now a category for installing Fonts and one for Themes and Tweaks. I'll be spending some time in those categories when I install Lucid this April...

My grade for applications: B+

**Stability and Performance** - You can't gage this too well from a LiveCD, and this is just the beta, but here are some things I noticed.

* No problems with the boot-up for me. Everything loaded fine.

* Boot up time was good for a LiveCD. I'm guessing it would be really good on a full install.

* Memory usage seems a bit better than on Karmic.

* Programs like Firefox, OpenOffice, Nautilus, and such all have worked fine.

* All the panel applets crashed when I tried to look at the Workspace Switcher preferences, but all of them were able to be reloaded successfully, the system recognized this as an error and offered to let me send a bug report, and the bug report process was pretty straight forward using Launchpad (evidently this bug is already being dealt with). Not every user is going to have an account, however, so while this is a good system it isn't perfect and I doubt most casual users are going to bother with it.

* The beta seems a more stable than the last alpha (on my hardware). The Alpha 3 had some issues with this machine's monitor. Basically, it refused to work with the LCD monitor, but did work when I hooked up an old CRT I just happened to have nearby. Don't know why, but I'm glad to see that that issue was fixed.

My grade for stability and performance: C+

Overall, I give Lucid a solid B. The interface, while different, seems to be an overall improvement. The default applications seem good. And while stability is still a bit of an issue, it's no worse than Karmic and will likely get better by April 29. 

When Lucid comes out in I'll be installing it on all of my machines and will not be switching to Fedora as I thought I might.

If you want to see a good piece of color palette, look no further than the hints of it for the website mockups! Now THAT is what I'd like to see in the desktop. WTH? Aim for white! Use orange as an accent! Stop crapping on about 'lightware' and make it LIGHT!

**Metacity theme. **

Good ideas in there but only just passable implementation. I really dig being able to drag by the menubar as well as the title. Scrollbars are ok. I don't care whether the buttons are on the left or right, I've used OS X and Windows frequently in the past, so both ways feel totally natural to me and frankly I have no idea why people are getting their panties in a twist over it. 

The buttons are a nice touch, i do like the inset frame around them but it often looks totally wrong when there is only a minimize button available (try making a USB startup disk, or installing) -- Canonical, don't let tiny things like this go unsolved! They make an impact on presentation and first impressions, they do. Also, why do I get no visual feedback when I'm pressing any of the titlebar buttons on a focused window (in particular the close button) -- don't let this go! It's so easy to fix and makes an impact on how enjoyable the desktop feels to use.

The aliasing on the corners too, is especially noticeable with this new metacity theme, particularly if you look down at the bottom corners. It may sound like I'm being pedantic, but this is aimed at ALL computer users, not just Linuxers who know this kind of thing can be changed, no? Tiny little rough edges like this will impact impressions. Don't let them go!

Don't!

**GTK & icons.**

******' EEWWWW. Why not just keep the old Human theme and make the menubars dark? It's the same engine, so how hard can this be?

As for the specifics... scrollbars are okay. Too round and too fat for my tastes, but they're okay. Buttons oveall are WAY borked, they have no sense of depth, the gradient is overdone and they just look very very rough. The old Human theme from 9.10 really shone in this area and this is a massive step back. Look at the Nautilus toolbar. Ugly as hell. On that subject, must we insist on having half the Nautilus window taken up with useless actions? Get rid of the main toolbar, you don't even need it!

I do love the icons in the panel. I'd get rid of the Firefox/Help shortcuts, though. And why do we have a purple desktop next to all these orange folders? I mean, yeah, I get it, but is it not obvious that these colours are clashing? Make the folders more pale or something, maybe even... manilla coloured! Totally simple fix and would make it look so much better.

Having one desktop -- that's fine by me. But if there's only one desktop, what's the desktop applet still doing in the panel?? New people aren't even gonna know what the hell it is. It doesn't do anything if you click on it and the tooltip doesn't tell you jack squat about why it's there or what you can do with it.

**Other little nice things. **

Trying to switch on Desktop Effects immediately informed me that I had to fetch the Nvidia driver. Ace! (Even though it failed to actually fetch them, giving me a 'file not found' error)

GIMP is dead. I use GIMP pretty much every day, but this makes sense to me. For all its usefulness, it does have a learning curve akin to climbing a brick wall with crutches and so I understand and support the decision to remove it.

The software centre is really rather nice now, actually. First impressions are very good. If only they could find some way to kill off the sidebar, and integrate it in some more aesthetically pleasing manner. 

Plugging in my iPhone let me browse all the music tracks on it. Not even OS X or Windows will let me do that. Outstanding!

Booting up straight into gdm for the liveCD boot and THEN asking you what you want to do is very, very nice.

**Little bad things. **

No wireless support for my NIC. Sigh. 

Where's Plymouth? Is that little animation of the dots working what Plymouth is meant to be? Didn't Fedora have this awesome burning blue planet or something?

Empathy is crap. Put Pidgin back in! It doesn't matter that Empathy is on its way to greater things if Pidgin is already there!

Yo, I can't read the text on my active app in the task list at the bottom of my screen for the bright skin tone behind it. Fix this!

How do I search my computer for apps and applets? Oh crap, I can't!  Seriously, how hard is this to address? Chuck in GNOME-Do or Launchy and add a shortcut to the panel so first-timers know it's available. Come on, Windows and OS X have had built-in-search-everything for years now. Linux has got it too, so why isn't it here?

Multimedia -- I still can't play DVDs or watch TV. Don't give legality issues as an excuse, give us a link to libdvdcss and a fair warning that what we're doing might be illegal in our country. It's better than nothing. At least this way people will know it's possible!

**Overall.**

Really, I'm surprised to say my first impressions are actually on the positive side. My biggest problems are that even with so very much potential in the new palette and branding, the first glance at the desktop gives the impression of being really lifeless, sludgy and dark -- pretty much the antithesis of what was proposed in the rebranding!

**tl;dr.**

Works ok, seems good, but is fugly and unprofessional-looking. C.
