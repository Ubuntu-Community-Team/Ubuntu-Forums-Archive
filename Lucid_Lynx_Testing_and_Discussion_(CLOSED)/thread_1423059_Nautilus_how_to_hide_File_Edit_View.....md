I use them every day...:p

If you want to save space, have you tried globalmenu? It is a gnome-panel applet that removes the GTK menus from every window, and places them in the panel instead. It saves space - it is vaguely similar to the way Mac OS X works.

It doesn't work with Java apps, or stuff like Firefox (but you can get an addon to hide the FF menu anyway).

In the end, I think you'll realise it makes almost no difference whatsoever that your windows have a few pixels dedicated to a menu bar. With keyboard accelerators, they can make you much more productive. I gave up trying to be minimalistic about it a while ago.
