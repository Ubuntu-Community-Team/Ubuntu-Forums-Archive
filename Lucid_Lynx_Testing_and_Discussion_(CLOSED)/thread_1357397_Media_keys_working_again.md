### Post by phenest on 2009-12-17
There was an issue regarding media keys not working on keyboards with some computers, mine being a Dell Precision M90 laptop. This was found to be connected to enabling 'show the pointer when the control key is pressed' in Mouse Preferences in Gnome. After testing alpha 1, I have discovered this now works perfectly.

[Launchpad bug]("https://bugs.launchpad.net/gnome-settings-daemon/+bug/267495")
