### Post by reiki on 2010-03-24
Karmic is still my primary OS. I can't bring myself to run beta1 as primary, but I have to say that 10.04 beta1 is working better for me "out of the box" than Karmic does after months of settling in and tweaking. 

I had to use the alternate CD to do the install as the live CD just didn't boot all the way to a desktop, but once installed, everything just seems to work. The included ATI driver is working fine for me so I haven't even bothered with installing the catalyst driver. I installed Ubuntu Restricted Extras and that got Flash and Java working just fine. 

I understand the whole premise behind Ubuntu's open source stance, but I STILL maintain that there should be an easy-to-follow guide to getting normal stuff to work (like firefox WITH flash and java for example). 

A new user should be able to install Ubuntu and begin using it for normal activities. Browsing the web, watching a store-bought DVD, etc. It's OK if the open source concept is maintained in the base install, but then the new user needs a guide for how to EASILY install the capabilities they've come to expect. All in one place. Easy to look up what they're trying to do. Little to no command line work. 

It's awesome that if you have more skill, you can make Ubuntu dance and sing, but a new user needs a more immediate and less technical experience. 

wow... way more ramble than I intended, but.... it is what it is. :)
