> **grubdude said:**
> The upgrade cannot be done through Alpha2 though, correct? You still need to download the new iso and install, yes?

No.

Just make sure you're update manager says everything is up to date.

system->admin->update manager

There is nothing special you have to do to "update to" alpha 3. As long as youv'e been installing updates since you installed alpha 2 you "have" alpha 3.
Alpha 3 is basically just an image of the current files and updates. If you've been updating regularly you have the same thing. Although still recomended to format/reinstall final version In april 29 if you've been updating from early alphas as they can leave some old files not used (kernels mostly that you can manually remove)
