### Post by cariboo on 2010-01-29
I'm having the same problem this morning. To change the server you are downloading from go to System->Administration->Software Sources, and Select Download From:, select Other from the drop down, In the Choose a Download Server you will see all the Country codes.

I usually just click Select Best Server and let it ping all the servers, and select the one with the lowest ping time. This doesn't work all the time, as some of the servers with the best ping times aren't fully up to date.
