The theme updates for Lucid also included a 'leet white-on-purple scheme for the Terminal. OK, whatever.

Anyway, it also included a "Legacy" profile of black text on white background which Ubuntu's been using for the terminal for a long time.

In the latest update (today, and possibly earlier), choosing File: Open tab: "Legacy" doesn't open a black-on-white tab. It just opens a white-on-purple tab.

You have to go to Edit: Profile preferences: Colors and deselect "Use colors from system theme" and reset the colors to black-on-white.

Obviously, this should be already set.

Anybody else encounter this?
