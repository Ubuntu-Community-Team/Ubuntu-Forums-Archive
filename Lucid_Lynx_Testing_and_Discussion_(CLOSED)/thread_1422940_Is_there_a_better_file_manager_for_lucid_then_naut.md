### Post by BwackNinja on 2010-03-06
I'd say that nautilus is the most complete gtk file manager, though pcmanfm2 is looking good so far (still not at the level nautilus is). A lot of why nautilus doesn't seem up to par is that it was originally intended as a spatial file manager, and does a really good job at that, although ubuntu and others ship it in browser mode by default. Nautilus is now getting more love in browser mode because the gnome devs have decided that browser mode will be default for gnome 3.

About your other question, the easiest way to find your gtk+ version is to type:
```
apt-cache policy libgtk2.0-0
```
 in a terminal.
