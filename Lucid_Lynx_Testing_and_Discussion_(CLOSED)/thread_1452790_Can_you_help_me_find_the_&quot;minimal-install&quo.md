### Post by psusi on 2010-04-12
"minimal-install" is a bit of a misnomer.  What you have found is the netboot installer, which downloads all current packages from the archive to install them, rather than having the packages already on the cd to install.  Since the packages are downloaded when you install, the installer does not need updated throughout the development process.
