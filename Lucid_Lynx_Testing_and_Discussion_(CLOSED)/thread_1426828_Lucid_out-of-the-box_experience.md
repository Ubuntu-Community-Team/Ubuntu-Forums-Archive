### Post by splicerr on 2010-03-10
So I downloaded the Lucid Daily Live CD and I'm currently running it. I must say I'm really impressed with the out-of-the-box experience. All my hardware works and the Desktop looks really slick with the new theme,  the transparent gnome-terminal and the desktop effects. 

But there's one thing that I don't understand. When I opened a H.264 encoded movie with Movie Player it asked me to search for codecs but it didn't find any and the movie didn't play. Sure after I enabled "universe" and "multiverse" the codecs were found. But if you are new to Ubuntu you probably don't know about this. 

So wouldn't it be more logical when the search for codecs fails to ask the user if he or she wants to enable the "universe" and "mulitverse" repositories? Surely that would greatly improve the  out-of-the-box experience for people new to Ubuntu and I'm sure I'm not the only one who have thought about this. So why isn't this implemented yet?
