### Post by MacUntu on 2009-11-09
Continuing the (now moved) Karmic thread [0] with a bit of info on the state of ATA TRIM Mark Lord posted at OCZ Forum [1]:

[QUOTE="Mark Lord"]It's currently in limbo. Matt, who first worked on it, hasn't updated his creaky patch set since April. In the meanwhile, Chris Mason and Christoph Hellwig have added TRIM support to btrfs in the current mainline kernel tree. I don't know if the ATA helper for it is there yet, though. Christoph was also prototyping XFS support for it at one point, but the early versions were horrid.

I think activity will pick up again in a few weeks, and definitely this winter.[/QUOTE]

Unfortunately "this winter" doesn't sound like we'll see it in 2.6.33 which maybe will be Lucid's kernel of choice. So no TRIM support for the upcoming LTS release? What a pity! :(
