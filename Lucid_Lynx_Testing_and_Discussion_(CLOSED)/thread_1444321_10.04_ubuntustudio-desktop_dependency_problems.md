### Post by dino99 on 2010-04-05
when such issues comes up, the better is first to verify repo:
 no conflict wirh different releases,
 no third repo,
 no ppa,
so remove or comment everything that is not legal lucid repo
then clean the system: clean, autoclean, autoremove
remove/purge ubuntustudio

update
sudo apt-get install -f
reinstall ubuntustudio

if errors are still there, well its probably that packages are not all yet updated onto server. (select main server to be sure)
so be patient a little bit, its Monday, let devs push their work.
