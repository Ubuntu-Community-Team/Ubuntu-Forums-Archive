For an interface and tooltips shown in db you can install 
PulseAudio Mixer Applet from ppa:v-geronimos/ppa
See [https://launchpad.net/~v-geronimos/+archive/ppa](https://launchpad.net/~v-geronimos/+archive/ppa)

[http://code.google.com/p/gnome-pulse-applet/](http://code.google.com/p/gnome-pulse-applet/)
