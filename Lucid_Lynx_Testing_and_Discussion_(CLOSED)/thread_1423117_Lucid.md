### Post by markbuntu on 2010-03-06
If you want a stable Lucid you should wait a few weeks after the release. That is when a lot of the hardware bugs are found and fixed. The testing community is big but not big enough to test every possible combination of hardware and software. 

If you have or can create a sepearate partition you can test lucid on please do and please please report any bugs you encounter. This will go a long way to ensuring that lucid will work on your hardware.

Lucid is fairly stable right now but subject to many many updates all the time ~100 or so every few days. If you have a limited internet connection this would not be a good time to get into lucid.
