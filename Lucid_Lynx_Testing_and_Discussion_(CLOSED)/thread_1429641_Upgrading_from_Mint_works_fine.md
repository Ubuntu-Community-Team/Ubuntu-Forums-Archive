### Post by nanog on 2010-03-14
The mint menu can be added to ubuntu with simple apt-get command:

[http://code.google.com/p/ubuntu-system-panel/wiki/APT_Synaptic_Repository](http://code.google.com/p/ubuntu-system-panel/wiki/APT_Synaptic_Repository)


IMO, the only advantage of running mint is automatic installation of codecs. This is fine for a one-man hobby distro but is illegal in the USA and not something that canonical can do.

Another recurring problem with mint is that distribution upgrades fail *badly* and are unsupported by mint.  I install ubuntu on my boxes exactly ***ONCE***. Any debian distro that breaks the dist-upgrade path seems exclusively targeted at "reinstall my distro(s) every few months" hobbyists.
