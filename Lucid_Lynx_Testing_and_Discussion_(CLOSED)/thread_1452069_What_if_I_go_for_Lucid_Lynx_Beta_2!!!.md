### Post by drs305 on 2010-04-11
If you install it now and continue to get updates you will have the new version when it is officially released.

However, Lucid is still in Beta and there are still daily updates. I would wait for at least the Release Candidate version to be made available. There isn't a big rush.

If you want to get as close to the actual release as possible, without dealing with the slow connections on release day, install the "daily build" a day or two before the scheduled release. That will be about as close as you can get without waiting for the official release.

Of course, best of all, just wait for a few days *after* the official release and see what the non-tester results are like as many more users install Lucid on their computers.
