### Post by Nesaskewatch on 2010-04-16
I am not sure if this has already been discussed, but I got into a big mess by ignorantly installing grub to my xp partition while upgrading from 9.10 to 10.04.  During the upgrade process, if one mouses over the the check boxes for where to install the new grub, a pop up information box tells the user to install grub to *all* of the entries if one is unsure where to put it.  This might perhaps be changed to include some sort of warning to never install grub to your windows partitions!  If one accidentally does what I did, see here; [http://articles.techrepublic.com.com/5100-10878_11-6031733.html](http://articles.techrepublic.com.com/5100-10878_11-6031733.html)  There are a number of good solutions there for a damaged windows boot loader.  In my case I ran fixboot.exe from the recovery console and that repaired the corrupted boot sector.

Cheers!
