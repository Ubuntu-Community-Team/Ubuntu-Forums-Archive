### Post by dcstar on 2010-04-01
I finally "bit the bullet" and mounted my 9.04 /home partition onto my 10.04 test system and here are a few things that may be of assistance to anyone else taking this path:

[LIST=1]
[*]The "Indicator Applet" needed to be manually added to the top RHS panel
[*]I had to install the **powernowd** package so my CPU did not stay at 100% when niced processes were running (like BOINC)
[*]Using the Update Manager "Upgrade" function will still overwrite any "Pinned" packages you have in your system....
[*]The transparency setting in the Gnome terminal had to be reset
[*]When I copied my existing /etc/fstab entries 10.04 would not boot because of the "usbfs" entry I previous used for VMware USB functionality
[/LIST]
Apart from those the overall setup seems to be performing ok.

All I have to do now is wait for VMware Server 1.0.10 to be compatible with the kernels shipped with this release......
