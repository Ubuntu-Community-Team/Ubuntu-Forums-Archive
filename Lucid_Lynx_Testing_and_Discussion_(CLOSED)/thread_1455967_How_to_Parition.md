### Post by Merlins on 2010-04-16
Hi, (apologies if you already know this, but...) if you want to play with other operating systems, you could install a Virtual Machine environment like Sun Virtualbox.  This is in the Ubuntu repositories I think.  I am beginner too and found it very easy to install and then set up a Windows environment within it.  The virtual environment is stored as a straight forward file, so its easy to delete, backup and so on.
