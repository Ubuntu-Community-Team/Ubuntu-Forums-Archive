### Post by exploder on 2010-01-23
I happened to find a wallpaper that looks good with the default Ubuntu artwork and would like to talk about this a little. Here is a screen shot so you can see for yourself.

[http://img37.imageshack.us/img37/1595/screenshotmi.png](http://img37.imageshack.us/img37/1595/screenshotmi.png)

This wallpaper was made by a community member and it is semi transparent so you can make it appear any color you like simply by changing the color of the desktop. I changed the desktop color to bright orange to get the effect you see in the screen shot.

What I like about this wallpaper is that it looks very good with the brown used in the default theme, it's not too busy and has a nice professional look to it. It's a nice bonus too that you can change the color if you change your theme. 

The default Ubuntu theme is pretty nice but it is very difficult to find wallpaper that really looks good with it. I thought this wallpaper or something along these lines might be worth mentioning because the color and style look so good with the rest of the defaults. I do not want to start the usual artwork argument thread that always appears with every new release. I am just making a suggestion because I really 
think this wallpaper looks very attractive with the rest of the default artwork. 

The logo might need some refinement if this was to be used as a default, it does kind of go along with the two panel look though, just the shading is a little to the blue side. I am just throwing this idea out there because in my opinion it looks nice.

Edit: Here is the thread where I found the wallpaper.

[http://ubuntuforums.org/showthread.php?t=978038&page=3](http://ubuntuforums.org/showthread.php?t=978038&page=3)
