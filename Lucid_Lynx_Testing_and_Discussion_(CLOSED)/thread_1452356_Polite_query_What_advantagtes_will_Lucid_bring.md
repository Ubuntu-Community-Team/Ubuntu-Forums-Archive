### Post by ranch hand on 2010-04-12
The main thing is that this is another LTS.  It is working as well as can be expected (or a little better) at this point.

This is intended for ext4.  It will work on ext3.

If you upgrade you will get ext3 and a lot of left  over stuff from your old install.

You can "change" ext3 to ext4.  I have done this and it is not the same nor worth it in my experience.  A fresh install on ext4 is well worth it.

You will have the problem of antique left over grub-legacy files if upgrading.

If I were you (I manly use 9.04 and 10.04-testing but still call 8.04 my Main) I would hold off for a little bit, maybe a month, to let the new LTS settle in.  I would back up your files while waiting.  I would then do a clean install.

This system, on ext4, just blows 8.04 out of the water.  There have been a lot of changes in the last 2 years, most for the better.

You still have a year of support for 8.04 so I would not be too fast to switch.

One thing I would do is install the package debfoster on your 8.04.  It is a package that maintains a list of all your installed packages (see synaptic for details).  This can make getting every thing you may want back on your new installation very easy (at least if you keep the list with your backed up files).
