### Post by jtniehof on 2010-01-01
With each release of Ubuntu, some people encounter frustrating problems of the sort "X doesn't work" or "my network doesn't work." These problems make it hard to get anything done on the machine, including submitting a good bug report. Fortunately the live CD images provide a very easy and relatively safe way to test on real hardware, so we can find and fix these bugs before release.

So I'd like to encourage everyone to do a quick test as each Lucid alpha and beta is released, and record their findings. I've set up a [wiki page]("https://wiki.ubuntu.com/jtniehof/LucidTesting") to coordinate the effort. Please record your results there, organized by your network/video hardware (as listed by lspci). If you have problems, link any applicable bug reports, forum posts, etc. The page is under my namespace on the wiki simply because this is not an "official" testing group; feel free to edit it to clarify instructions as well as adding test results. (It's rather barebones at the moment.)

If you're already involved in testing efforts, thank you, and please keep it up! This wiki page is meant to encourage people new to testing to try their hardware and record the results in an organized fashion, so we can produce excellent bug reports and find fixes.
