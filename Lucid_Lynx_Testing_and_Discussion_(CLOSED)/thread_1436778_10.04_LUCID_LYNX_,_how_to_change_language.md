### Post by coffeecat on 2010-03-23
Two comments:

If you're a newbie to Linux as well as to Ubuntu, you might be better off using the latest stable version of Ubuntu - 9.10, Karmic Koala. Since 10.04 (Lucid) is still in beta you will have inevitable bugs to deal with as well as your unfamiliarity with the OS. But if you have experience of other Linux distros, then ignore this comment and have fun with Ubuntu!

The language problem you've encountered might be a bug - I don't know - but try this. Go to System > Administration > Language Support. You shouldn't need to install the French language packs but click on the Install/Remove Languages button anyway, highlight French and tick the Translations and Spellchecking and writing aids boxes and click apply to make sure you've got everything you need. To change settings to French, I had to go to the 'Text' tab and change that before I could select French in the Language tab. I didn't test this because my spoken French is very rusty and I didn't want to change my desktop to something I might have difficulty understanding (:wink:) but give it a try.
