### Post by shafin on 2010-03-01
This surely deserves a thread of its own.
From the project page: "GNOME Activity Journal (formerly known as GNOME Zeitgeist) is a tool for  easily browsing and finding files on your computer. It keeps a  chronological journal of all file activity and supports tagging and  establishing relationships between groups of files."
We normally remember which files we worked on at any given time, activity journal tries to leverage this.It's set to be an integral part of gnome 3. Also recently there has been some mockups suggesting it to integrated in nautilus.

***Installation:***
                 Install it's PPA by adding           **ppa:zeitgeist/ppa**           to your system's Software Sources. then install gnome activity journal from synaptic/software center.


***Running:***
Applications&gt;Accessories&gt;Activity Journal.
