### Post by howefield on 2010-03-21
Did the plugins folder exist or did you create it to move the plugin in to it ?

If you created it, try renaming the folder plugin, without the "s"

Or you could try moving it to /usr/lib/mozilla/plugins with a terminal.

```
sudo mv  ~/.mozilla/plugins/name of file /usr/lib/mozilla/plugins
```

Substitute "name of file" with the actual file name.
