### Post by jpmelos on 2009-10-31
Hello, everyone.

I believe a nice feature to have on Lucid Lynx is something like "Commands to be Executed on Restart/Shut Down".

For example, that would be useful if you have a software that, for some reason, won't let you change the folder it works on. So, say it will always save in a subfolder of your home directory, but you have a different partition to keep your stuff. So, would be very useful to be able to run some "rsync" command everytime the computer restarts/shuts down... That's because sometimes you just forget you have that stuff on a folder different from the default.

Me, for example, I lost some files because they were in such situation, and I fresh installed Karmic Koala and forgot to copy them to the partition. Lucky me, that were not that important to cause any bigger damage.

I apologize if that feature is already implemented, but I couldn't find it. In such case, could you tell me where it is?

Thank you very much!
