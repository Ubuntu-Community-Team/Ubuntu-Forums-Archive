### Post by 3rdalbum on 2010-03-13
I would not dist-upgrade from a stable release to a development release. I tried it before during the Karmic cycle and it was a disaster.

Do a clean install of Lucid. You don't need to start from the Alpha 3 image; there are daily live builds being done all the time. Give it 24 hours before downloading it, though, to ensure it has the current 'mountall' and 'plymouth' packages.
