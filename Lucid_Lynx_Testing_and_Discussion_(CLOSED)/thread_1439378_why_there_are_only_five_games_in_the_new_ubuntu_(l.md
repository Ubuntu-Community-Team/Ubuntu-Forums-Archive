### Post by amano on 2010-04-29
Swell Foop and Same Gnome are the same game. Swell Foop is the updated version which had a name change (Gnometris changed to Quadrapassel btw). They are just different revisions of the same game thus they conflict with each other (as does Totem 2.28 and 2.30...).

The reason for the name changes is that Gnome programs shouldn' use "Gnome" in their names (it's Nautilus and not "GnomeExplorer", it is Totem and not "XineGnome") to get some coherence within the Gnome stack.
