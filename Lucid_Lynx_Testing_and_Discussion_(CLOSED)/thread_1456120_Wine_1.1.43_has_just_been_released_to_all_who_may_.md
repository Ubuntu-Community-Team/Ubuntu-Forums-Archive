### Post by Reiger on 2010-04-16
But there is a .deb repository for latest WINE: [http://www.winehq.org/download/deb](http://www.winehq.org/download/deb)

To start using the repository in Lynx you can use: ```
sudo apt-add-repository ppa:ubuntu-wine/ppa && sudo apt-get update
```
