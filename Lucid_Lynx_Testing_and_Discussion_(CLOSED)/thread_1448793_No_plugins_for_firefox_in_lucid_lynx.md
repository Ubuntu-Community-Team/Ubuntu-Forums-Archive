### Post by plucky on 2010-04-07
> **2cool4u92 said:**
> Hi,I'm using Lucid lynx beta 1. I'm not able to view videos in video sharing sites like youtube. Firefox informs me that there are missing plugins,but then says no suitable plugins were found. Is there any solution to this ?

Install **Ubuntu-restricted-extras** which will include flash plugin.

```
sudo apt-get install ubuntu-restricted-extras
```

or use **synaptic package manager** or **software centre** to install it.

Good Luck
