### Post by Seren on 2010-04-08
The current Lucid version is 1.6.5 (same as in Karmic).

This is the new "stable" release and should probably be included in Lucid : the main reason being that people with different major versions can't play with each other on the multiplayer server.

If the game is not upgraded for Lucid, users staying on LTS will be stuck on the old server for a few years and cut from most players. (including windows players who will probably upgrade to 1.8).

There is already an upgrade-request :
[https://bugs.launchpad.net/ubuntu/+source/wesnoth/+bug/557696](https://bugs.launchpad.net/ubuntu/+source/wesnoth/+bug/557696)

And I invite interested people or packagers to contribute.

(I am not sure that Lucid can sync from experimental debian repository at this point..)

Edit :

Ok apparently 1.8 won't be included in Lucid Lynx, not enough time, and it is too late after "feature freeze". It will probably be possible to get it through backports repos after Lucid release.
