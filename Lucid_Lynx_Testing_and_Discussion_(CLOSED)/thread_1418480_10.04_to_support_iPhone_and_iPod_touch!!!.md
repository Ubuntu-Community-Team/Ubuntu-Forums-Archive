### Post by Islington on 2010-03-01
Here is what you do: 

1. Add the repo, in a terminal:

```
sudo add-apt-repository ppa:pmcenery/ppa
```
```
sudo aptitude update 
```

2. Install necessary packages:

```
sudo aptitude install gvfs gvfs-backends gvfs-bin gvfs-fuse libgvfscommon0 ifuse libgpod-dev libgpod-common libimobiledevice-utils libimobiledevice0 python-imobiledevice libplist++1 libplist-utils python-plist libusb-1.0-0 libusb-1.0-0-dev libusbmuxd1 usbmuxd 
```

3. Configure Fuse (used to interface)

```
 gksudo gedit /etc/fuse.conf 
```

Change ```
#user_allow_other
``` to ```
user_allow_other
``` (delete the asterisk so it is no longer commenting.

4. &#8220;System&#8221; -> &#8220;Administration&#8221; -> &#8220;Users and Groups&#8221;, click on the little key at bottom to unlock it for making changes. Then, select your username (<islington> for me) and click on &#8220;Manage Groups&#8221;.

Lookout for &#8220;fuse&#8221; group, double-click  it. Tick the checkbox next to your name in the new popup window. Click on OK and close all dialogs. You&#8217;re now in the &#8220;fuse&#8221; group.

5. log out and in again

6. Mount the device by doing
```
sudo mkdir /media/ipod 
```
Give yourself the correct permissions so that you can edit this folder you have just created by using chmod or by running ```
gksudo nautilus
``` browse to /media and permissions are in the properties when you right click on the folder.
Mount using:```
ifuse /media/ipod/ 
```

7. Getting the UUID
make sure  /media/ipod/iTunes_Control/Device/ exists
```
lsusb -v | grep -i iSerial
``` for UUID
copy that 40 character long code.

8. Ipod-read-sysinfo-extended
```
ipod-read-sysinfo-extended <copied UUID> /media/ipod 
```

This should generate a file named iTunes_Control/Device/SysInfoExtended.

Make sure it&#8217;s not empty and should be a big plist (XML file) with a bunch of info.

9. Unmount 
```
fusermount -u media/ipod
```
Reboot.

10. Plug the iPod/iPhone, you should see it appear on the desktop and it should now be:
    1. Mounted as-is in Rhythmbox and you can add music files. 
    2. Mounted in gtkpod but you&#8217;ll have to mount using &#8220;ifuse /media/ipod&#8221;.


Guide adapted from here: [http://www.taranfx.com/sync-iphone-linux](http://www.taranfx.com/sync-iphone-linux)
