### Post by autocrosser on 2010-03-20
Current Nautilus update as of 19-3-2010 has a missing header file in nautilus.desktop (usr/share/applications)--If you update, you will get a segfault & nautilus non-start. I have two installs, so I rebooted to my older install & used the nautilus.desktop file from it (no update done today) to get Nautilus working again.....

Bug Report:  [https://bugs.launchpad.net/ubuntu/+source/nautilus/+bug/542423](https://bugs.launchpad.net/ubuntu/+source/nautilus/+bug/542423)
