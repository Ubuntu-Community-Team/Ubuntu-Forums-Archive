### Post by ranch hand on 2009-12-09
Yes, 3 years support on the desktop and 5 on the server.

Yes, they try to be pretty conservative with the LTS release.  It will have some tweeks here and there and a litmited number of new things.

Part of the problem with 9.10 is that they threw in a lot of new stuff so that they could be used in 10.04.

I think is likely to be great.  We are already using the 2.6.32 kernel (it is what is going in the final).  I am sure we will get  lot of updates on it but still, we are using it.

There will only be 3 Alpha releases and 2 betas.  As more people test on the betas, having 2 will, hopefully, get a bigger variety of hardware used in testing.

Most of the folks still having trouble with 9.10 are using boxes that really never saw testing.  The 2 betas should help that a lot.
