### Post by flipper9 on 2010-03-21
Updates are contstantly coming out.  When you run Update-Manager, you are getting the latest updates available.  If you run Update-Manager today, you are getting Beta 1 (and possibly some updates that didn't exist when Beta 1 was initially released.  If you run it in 2 months, you'll get the "official release" + whatever else has been updated.

Remember that an "alpha 3" or "beta 1" designation refers more to the actual "burned" CD-ROM and not what is on your computer.  The names are just "snapshots" in time. If you were to run Update-Manager in a couple days, your system would be upgraded to something between the Beta 1 CD-ROM and the Beta 2 CD-ROM because you are getting updates that were released after the CD-ROM.

It'd be nice if the "About Ubuntu" menu would reflect some "easier to understand" information.  Let's say that the string is updated after each milestone.  That way, someone could goto that menu and would feel good that they have updated successfully.
