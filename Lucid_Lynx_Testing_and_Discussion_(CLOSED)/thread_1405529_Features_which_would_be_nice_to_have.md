### Post by SKLP on 2010-02-12
So I've looked a bit at the proposed stuff for lucid, and these are some (IMO) interesting things:

[0] Multi-arch ( [https://blueprints.launchpad.net/ubuntu/+spec/foundations-lucid-multiarch-support](https://blueprints.launchpad.net/ubuntu/+spec/foundations-lucid-multiarch-support) ) 

[1] btrfs support ( [https://wiki.ubuntu.com/FoundationsTeam/BtrfsSupport](https://wiki.ubuntu.com/FoundationsTeam/BtrfsSupport) )

[2] /tmp as tmpfs ( [https://blueprints.launchpad.net/ubuntu/+spec/server-karmic-tmp-as-tmpfs](https://blueprints.launchpad.net/ubuntu/+spec/server-karmic-tmp-as-tmpfs) )

[3] automatically growing swap  files rather than static partitions ( [https://wiki.ubuntu.com/FoundationsTeam/Specs/SwapFile](https://wiki.ubuntu.com/FoundationsTeam/Specs/SwapFile) )

-----

[0] has been deferred for lucid,

[2] and [3] are currently blocked due to [https://blueprints.launchpad.net/bugs/386554](https://blueprints.launchpad.net/bugs/386554)

[1] seems to have been a bit stuck atm. but would be nice to have for lucid ;-)


So what are your favorite proposed features?
