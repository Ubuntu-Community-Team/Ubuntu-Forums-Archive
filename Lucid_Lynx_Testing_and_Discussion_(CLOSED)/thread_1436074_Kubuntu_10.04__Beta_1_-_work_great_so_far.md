### Post by 3Miro on 2010-03-22
I downloaded and installed Kubuntu 10.04 beta 1 and installed in dual-boot with my Ubuntu 9.10. When I say Ubuntu I should note that I have Gnome, KDE and XFCE installed and I often switch between them.

I expected problems with the beta, but actually I now have a fully functioning fully usable system. I was pleasantly surprised. Except for a few minor annoyances (when I am installing packages, the window that asks for the password pops up behind all other windows; installing kmplayer just gives the front end and you have to manually install mplayer to be able to play anything, the startup screen says Ubuntu instead of Kubuntu ... )

Huge improvements, Firefox works very well right now, overall the entire system is much faster and smoother, activities and different wallpapers per desktop seem to work now, more options to customize in the appearance menu, improved apps and widgets, and so on ...

It seems, however, that some things will not get fixed for now. We are still stuck with the horrible package manager and there is still the occasional sound bug. 

It was really nice to see Kubuntu working well. For me Gnome + Compiz is out of date and Gnome + Metacity is inferior to XFCE. Right now, if one wants a smooth and shiny modern DE for Linux, one has to go with KDE, but until now KDE + Ubuntu didn't go very well together. That is until now.
