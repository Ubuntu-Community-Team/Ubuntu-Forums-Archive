### Post by autocrosser on 2010-02-17
The problem is that W-P is not under any development any more--I got it updated to the .5x version during Jaunty testing (so it would just be a applet instead of in the tray area) & it needs to be rebuild with several of the new libraries. I've tried to bring it forward myself, but have not been able to make it work. 

The other problem will be with Gnome Shell & applets--W-P won't work at all with GS--I was able to get the older .4x version to work for a while, but ran into library problems with it also......Right now I'm using Wally---even with all the extra weight of the KDE libraries it pulls in--it seems to work much better than W-P ever did---just haven't been able to auto-start it yet.

What is needed is a new wallpaper switcher that will auto-start with ease & be light on system use...I don't have a answer for that one yet........
