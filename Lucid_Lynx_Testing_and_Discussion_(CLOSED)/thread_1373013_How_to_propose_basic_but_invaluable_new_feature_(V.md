### Post by Pipps on 2010-01-05
Hi Friends

May I suggest that Ubuntu is currently missing something which in my humble opinion is absolutely invaluable.

It is a very simple feature. It is a way of controlling the sound volume using a keyboard button combined with a flick of the mouse scroll-wheel.

I am aware that currently it is possible to click on the volume icon on the panel, and when the pop-up box appears, use the mouse-wheel turn the volume up or down.

However, I must say that it is significantly quicker, easier, more intuitive to be able to simply adjust the volume by holding down a key such as ALT and using the mouse-wheel to scroll up or down, to change the volume.  

I can confidently say this, because I have been doing it for some time whenever I have used XP. In Windows, this invaluable but incredibly small tool is called [VolumeMouse]("http://www.nirsoft.net/utils/volumouse.html"). 

I have spoken to the VolumeMouse developer, but he admits to have no experience in Linux programming. So is this something which the Ubuntu community could implement? It should surely be relatively straightforward to implement an equivalent in Linux? Could it work on the existing panel pop-up interface?

It would make an brilliant yet very small addition to Lucid in April this year, and be an invaluable feature of Ubuntu for the future!

Please let me know what you think.

Best wishes

Pipps
