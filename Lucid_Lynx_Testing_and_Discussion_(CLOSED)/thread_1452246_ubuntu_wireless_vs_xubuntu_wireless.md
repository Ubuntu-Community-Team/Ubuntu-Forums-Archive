### Post by wilee-nilee on 2010-04-11
Xubuntu is running the same network manager it should work. You can also just install the Xubuntu desktop in the Ubuntu one and choose between either at login.

Having both though will put some extra packages in both desktops like 2 screensavers so be ready for this. You can also remove the Ubuntu desktop and install the Xubuntu desktop.

So here is a site that is a great help in general. It shows the how to remove the Ubuntu and Kubuntu desktop removal lines for a pure XFCE which is basically xubuntu. This is for removing in 9.10 so don't use it, it is just a example of how a desktop is completely removed to be replaced by another.
