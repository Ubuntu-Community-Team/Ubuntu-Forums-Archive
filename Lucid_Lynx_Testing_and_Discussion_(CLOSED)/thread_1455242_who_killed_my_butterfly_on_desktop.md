### Post by coffeecat on 2010-04-15
> **sartic said:**
> Where can i find wallpaper width butterfly from beta?

Boot up with the Beta live CD and have a look in /usr/share/backgrounds in the live session. You'll find it there and you'll be able to copy it to a USB drive.

I've just had a look using a daily image live CD from about Alpha2 and Butterfly.jpg is not the only one that's been removed by the updates. MistyMorning.jpg was another pleasant one that's gone now, so it's worth saving the lot in case there are others you might want.

Alternatively, if you've still got Karmic installed somewhere you can find Butterfly and MistyMorning in the same place in Karmic. The removed images seem to be Karmic ones.
