### Post by Ibidem on 2010-02-18
Grub2. 
In /boot/grub/grub.cfg:
```
function recordfail {
  set recordfail=1
  if [ -n ${have_grubenv} ]; then if [ -z ${boot_once} ]; then save_env recordfail; fi; fi
}
```
There must be something screwed up, if it doesn't recognize that.
Edit the command list, delete the first line (says:
```
recordfail
```
then the important stuff)
If it boots, you will need to figure out how to make it permanent; otherwise, there's nothing permanently changed.
