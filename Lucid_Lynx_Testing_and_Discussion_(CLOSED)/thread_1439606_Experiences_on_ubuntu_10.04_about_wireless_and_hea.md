### Post by 3rdalbum on 2010-03-27
Please be more specific with your wireless issue. Saying "Realtek wireless" doesn't help much, because there are about a dozen different Realtek chipsets out there (mine doesn't have any issues from 9.10 onwards).

The output of the 'lspci' and 'lsusb' commands would be a help. Ubuntu 10.04's kernel version is the same as 9.10's (2.6.32) with some backported bits from 2.6.33, so wireless drivers will probably be the same versions.

EDIT: Also, Ubuntu 10.04 is in beta, and there are still daily live builds being made. You can always download one of those and try it out. Even the live CD should tell you if those problems are solved or not.
