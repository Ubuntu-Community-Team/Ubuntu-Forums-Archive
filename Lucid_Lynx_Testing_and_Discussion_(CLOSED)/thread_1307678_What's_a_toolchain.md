### Post by autocrosser on 2009-11-01
I already have my sources.list set for lucid--generally I run sources from the last & next releases for about the first week or so--that way I get the first backported stuff from the last release whilst I wait for the first new stuff.

Base-files are in: 5.0.0ubuntu8 (lucid)
libc6 is in: 2.10.1-3ubuntu1 (lucid)
binutils is in: 2.20-1ubuntu3 (lucid)
build-essential is in: 11.4 (lucid)
deb-helper is in: 7.4.3ubuntu1 (lucid)
dkms is in: 2.1.0.1-0ubuntu1 (lucid)
dpkg-dev is in: 1.15.4ubuntu2 (lucid)
fakeroot is in: 1.12.4ubuntu1 (lucid)
gnome-utils is in: 2.28.1-0ubuntu1 (lucid)

So, "most" of what's needed is already there--there are several more that I didn't post--the tools are waiting....................;)
