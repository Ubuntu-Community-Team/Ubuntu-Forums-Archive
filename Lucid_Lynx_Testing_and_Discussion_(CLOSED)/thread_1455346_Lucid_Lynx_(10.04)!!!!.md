### Post by roachk71 on 2010-04-16
I've read reviews on several tech sites, which (so far) look very appealing...

But as I've posted here so many times (and am certainly not alone in this practice,) I usually like to wait a few months after release before upgrading to any new version. This gives the developers and package maintainers enough time to work out major security and stability issues, and to eliminate show-stopping bugs *(those of us using Ubuntu when 6.04 was supposed to be released _painfully_ remember why its development and testing had to be extended, making it 6.06 instead.)* :-x

Also, there are quite a few packages which I must build from source; should there be major C or other library changes it may not be possible, so the wait is well worth it. 

In other words, *it really does pay to be patient*. :cool:

It's much too early, in my humble opinion, to conclusively decide whether or not to install Lucid. This is why I chose "Undecided" as my poll answer.
