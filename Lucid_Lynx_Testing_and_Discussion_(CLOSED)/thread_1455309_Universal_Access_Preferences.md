### Post by FuturePilot on 2010-04-15
Is there any way to remove the Universal Access Preferences icon from the panel? If I try it also removes the speaker icon and the envelope icon which I would like to keep.

---

### Post by Didius Falco on 2010-04-15
Alt-F2

```
gnome-keyboard-properties
```

under accessibility uncheck the box that says "Accessibility features can be toggled with keyboard"

This has happened to me enough times that I've added it to my "Useful Commands" personal help file. :rolleyes:
