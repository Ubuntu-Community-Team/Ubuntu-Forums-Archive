### Post by JohnJackson on 2010-01-11
There have been some changes in gnome-control-center, mostly for the worse in my opinion. The interface tab is now gone from the appearance dialog, meaning you can't change the appearance of icons in menus or change the position of text and icons in the toolbar. This is better suited to a "tweak" tool according to the devs. Also, backgrounds are now displayed individually in separate monitors instead of them being stretched. This change I'm not that bothered about as I rarely use more than one monitor. Note that you can't set different backgrounds as yet. 

The other change I noticed was the UI in the users and groups applet, which (as yet) doesn't look better to me.

Link to changelog and bug pages: [https://lists.ubuntu.com/archives/lucid-changes/2010-January/003172.html](https://lists.ubuntu.com/archives/lucid-changes/2010-January/003172.html)
