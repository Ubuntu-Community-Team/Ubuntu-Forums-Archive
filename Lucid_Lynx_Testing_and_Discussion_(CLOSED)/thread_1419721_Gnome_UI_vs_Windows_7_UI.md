### Post by Lorag on 2010-03-02
As far as i know there's nothing yet that acts like aero snap, but you can set windows to one half of the screen using compiz' grid plugin and a short-cut, i.e. strg+alt+"6" or "4" on your numpad.

> **Garyu said:**
> 2) The bottom panel with only icons showing, the ability to "sticky" icons to this panel, and a list of window thumbnails showing up when hovering over an icon which is running.

Dockbarx is able to do all that (with some of Compiz already existing features): [http://gnome-look.org/content/show.php/DockbarX?content=101604](http://gnome-look.org/content/show.php/DockbarX?content=101604) Have fun!

### Post by krazyd on 2010-03-02

Yup, KDE already does everything you mention (and more!) :D

There are at least 3 plasma-widgets which do something similar to what you want. The closest to Win7 is probably [Smooth Tasks]("http://www.kde-look.org/content/show.php/Smooth+Tasks?content=101586") (which is a fork of STasks). Under the description, click the link to STasks to see just how Win7-ish it can look.

A couple of other awesome dock-like task managers are [Fancy Tasks]("http://www.kde-look.org/content/show.php/Fancy+Tasks?content=99737") and [Daisy]("http://www.kde-look.org/content/show.php/Daisy?content=102077").

Kubuntu really isn't that buggy anymore, give it a shot..
