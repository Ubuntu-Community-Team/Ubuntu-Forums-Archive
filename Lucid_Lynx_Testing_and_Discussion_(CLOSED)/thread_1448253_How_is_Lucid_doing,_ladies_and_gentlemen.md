### Post by Scott82 on 2010-04-06
For me, ATI graphics drivers are working fine, but only if i install them before doing ANY other updates. basically here's what i had to do to get them to work:

Install from Beta Disc.
Install fglrx (it'll install some deps).
reboot
open a terminal
sudo aticonfig --initial
reboot
graphics now work
install updates (not entirely necessary, but it might be annoying to devs to get bug reports on things that have been fixed just because someone doesn't install updates).
