### Post by jfernyhough on 2010-04-25
Gloobus acts as a quick viewer for files so you can see a file without loading the full application (which can be useful for large apps like OpenOffice or GIMP which take a while to load).

The elementary-project's nautilus is patched so that pressing space will trigger Gloobus Preview rather than opening the file normally.
