### Post by solitaire on 2010-03-22
check out the folder:
"/usr/share/backgrounds/"
and
"/usr/share/gnome-background-properties"

Think those are the 2 main folders you should look at.

Basically you create a folder in the first directory and name it. in there create a ".xml" file with the same name as the folder.

Look at the xml files in the other folders to see the syntax required.
