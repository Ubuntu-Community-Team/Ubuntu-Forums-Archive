### Post by seeker5528 on 2010-02-10
> **mörgæs said:**
> By the way, is a netboot the same as a "minimal installation"?

With the netboot you are booting from an image stored elsewhere on the network, so you want the image to be as small as possible.

Just enough to boot, access the network/internet, and run the installer.

It's the same installer as the alternate installer (AKA text mode installer AKA Debian installer).

If you want to do a minimal installation that's an option, there are pre-canned choices for deskotop and multiple types of servers. You also have the option to select any additional packages you want to install.
