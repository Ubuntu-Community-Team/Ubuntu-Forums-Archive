### Post by bean123 on 2010-03-22
I've written a guide on how to archive this, it supports UEFI, Apple EFI and PCBIOS firmware:

[http://code.google.com/p/burg/wiki/QuadBoot](http://code.google.com/p/burg/wiki/QuadBoot)

It also contains info on how to install BURG on Apple EFI platform. It's very simple, just download the latest burg-efi64 and burg-themes packages from [http://code.google.com/p/burg/downloads/list](http://code.google.com/p/burg/downloads/list), extract to /EFI/burg directory, and rename burg_apple.cfg as burg.cfg. It'd be recognized by rEFIt automatically. In the BURG menu, you can use 't' hotkey to open a theme selection menu and try other themes at runtime. Here are some screenshots:

[http://code.google.com/p/burg/wiki/Screenshots](http://code.google.com/p/burg/wiki/Screenshots)
