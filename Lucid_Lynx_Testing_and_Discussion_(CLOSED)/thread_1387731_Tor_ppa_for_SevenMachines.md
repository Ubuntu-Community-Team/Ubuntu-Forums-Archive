### Post by SevenMachines on 2010-01-22
Tor is now available in both stable and experimental versions at deb.torproject.org, see [http://www.torproject.org/docs/debian.html.en](http://www.torproject.org/docs/debian.html.en)
If your using my ppa you should switch to the official repository.
```
deb http://deb.torproject.org/torproject.org karmic main
```or
```
deb http://deb.torproject.org/torproject.org experimental-karmic main
```I might still do ubuntu development release versions of the experimental tor''s but only if theres problems using the proper torproject repositories during the development release cycle.
