### Post by QIII on 2010-03-19
If you've been using apt-get, use apt-get.  If you've been using aptitude, use aptitude.

Although they touch back to the same base, they behave differently in some subtle ways.

Logging and dependencies, for instance.

I've used apt-get for years, so I haven't switched.  I've never had the dependency issues that are supposed to be a problem.
