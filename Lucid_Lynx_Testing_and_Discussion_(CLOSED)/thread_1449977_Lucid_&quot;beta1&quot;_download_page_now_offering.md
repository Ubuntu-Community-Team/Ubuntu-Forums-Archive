The updates** are** already in the servers and **have been for a whole week now** (most of them)...

If you run lucid (freshly installed or upgraded from another ubuntu release at any stage of development) and have installed all updates that have been available since you installed, you are already using beta2.

Beta2 is just a snapshot of the development that contains all the packages that were in the repos when the beta2 freeze came into effect on april 1st, plus perhaps a few packages that have been updated during the beta2 freeze to solve some more serious bugs that may have been encountered while testing the candidate iso files.

Did you expect to get a big chunk of updates when beta2 is officially released? Who would have thested those packages if they were kept secret from the testers (=us)?
You'll probably get pretty many updates **after** beta2 is released, but those will be packages that are newer than the ones on the beta2 isos, that the package maintainers have kept out of the repos because of the beta2 freeze.
