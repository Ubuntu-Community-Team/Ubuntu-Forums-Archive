### Post by lonniehenry on 2010-03-24
The compiz settings problems are probably because of using the nouveau driver.  It doesn't do desktop effects.  I assume you have a nvidia or ati card and didn't install non-open drivers. If you want effects, then install the hardware drivers using system>administration>hardware drivers.
