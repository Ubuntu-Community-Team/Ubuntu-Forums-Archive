You can add multiple instances of the quick access widget, then point them to the folder or network view you want, and give each one a different icon, name, etc...

If you want other types of icons than you can get using widgets, I think you have to right click the back ground, select 'Desktop Activity Settings', then change the type of activity to 'Folder View', assuming it wasn't already set to 'Folder View'.
