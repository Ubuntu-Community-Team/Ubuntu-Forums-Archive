### Post by rogerpittman on 2010-04-06
The default font in Lucid is "Sans".

- Right click on your desktop and select "Change Desktop Background" from the context menu.
- In the "Appearances Preferences" window, select the "Fonts" tab. This will show you what fonts are currently being used for applications, documents, window titles etc.
