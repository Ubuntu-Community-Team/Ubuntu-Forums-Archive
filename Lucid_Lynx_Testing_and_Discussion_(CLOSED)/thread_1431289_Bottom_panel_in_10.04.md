> **Vidayo said:**
> More like Microsoft Windows does?

Gnome is essentially Windows split onto two bars.  You can rearrange things to look and behave almost exactly like Windows in a few minutes.  The truth of the matter is the top bar is 80% wasted space, and most of the cruft there can be safely ditched.

The top bar makes sense in OSX with it's unified file menu concept in terms of you have space and you might as well use it, and since the file menu doesn't exist in main windows you don't really lose anything.  However on Gnome where the OSX file menu is is essentially a glorified Start button.

Although visually Lucid is very *cough* inspired *cough* by OSX, functionality wise it's nothing like it.
