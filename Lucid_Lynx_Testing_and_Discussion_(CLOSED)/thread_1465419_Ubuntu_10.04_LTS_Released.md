### Post by 23meg on 2010-04-29
Ubuntu 10.04 LTS (development codename Lucid Lynx), the third Long Term Support release of Ubuntu, has been released.

[SIZE="3"]**Please read the following before downloading / upgrading!**[/SIZE]

[list] [*] [Release announcement]("https://lists.ubuntu.com/archives/ubuntu-announce/2010-April/000133.html") on the [ubuntu-announce mailing list]("https://lists.ubuntu.com/mailman/listinfo/ubuntu-announce")
[*] [Release Notes]("http://www.ubuntu.com/getubuntu/releasenotes/1004")
[*] [Technical Overview]("http://www.ubuntu.com/getubuntu/releasenotes/1004overview")[/list]

[SIZE="3"]**Upgrades**[/SIZE]

If you're using Ubuntu 9.10, or have been testing Lucid before release, you can upgrade without having to perform a fresh install. Please read [http://www.ubuntu.com/getubuntu/upgrading](http://www.ubuntu.com/getubuntu/upgrading) for pertinent information.

[SIZE="3"]**Downloads**[/SIZE]

Please prefer using torrents, and [zsync]("https://help.ubuntu.com/community/RsyncCdImage"). If you have downloaded a development milestone release previously, zsync will let you only download the difference between it and the final image, saving lots of bandwidth and time.
