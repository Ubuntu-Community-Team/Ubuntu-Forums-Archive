### Post by yogesh.girikumar on 2010-02-03
Hi,


       It is always better to wait for the final release or at least the release candidate. There is no guarantee that it will be safe to do an upgrade even while it is in beta release. There will usually be problems with proprietary drivers almost always. It takes time to sort the issues out and fix them. Right now it's not safe to upgrade to Lucid at all, unless of course you do it for experimentation or if you're simply interested in checking it out and working on it, in which case, you can try to install it as a virtual machine on **Virtual box** which is a FOSS and you can download it for free.


       And speaking of hardware support, dell machines usually have good support for Ubuntu. They get certified by canonical for Ubuntu compatibility. But it's not wise to try Lucid Lynx on your machine yet. It's good to wait for bugs to be fixed and other hardware related compatibility issues to be sorted out. If I were you, I'd stick to Karmic for a bit longer.
