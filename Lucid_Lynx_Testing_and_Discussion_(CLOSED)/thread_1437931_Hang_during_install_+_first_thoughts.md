### Post by sigixv on 2010-03-24
I didnt file into launchpad, lack of time.

I installed with usb flash disk, installation went fine until the "computer has to reboot" message. The message box disappeared after clicking/hitting enter but that was it. Mouse remained responsive, it just didnt reboot. After resetting the machine, everything seemed fine though.


First thoughts:

1. Language and region settings: Please do something about this! I don't see the reason why i have to file all these settings during install when on first boot they are not implemented. Or so it seems, only partly implemented. Keyboard setting is good and the calendar is in my native language. All else is not and english is defaulted until i decide otherwise. Not a problem for me, but might be a let down for first time users.

2. Color & themes: overall not bad, classy first impression but two remarks.
Window controls on the left side... I'm not a fan. And what's more, the order is unlogic or at least unfamiliar. I tested it, and first time users experienced this as an extra addition to the "alien feel" they have with linux. So i say: right hand side, minimize maximize close...
And then a cosmetic thing: the iconset in use becomes quite chaotic and very ugly with the use of both yellow and purple in the same menus. Perhaps better to keep these yellow or make the purple a more gray shade


Hope this was a useful comment

keep up the good work
