### Post by mc4man on 2010-03-10
Several ways, a few
nautilus -> edit -> preferences -> media

in terminal
```
nautilus-file-management-properties
```

Edit: as of last nights live cd there wasn't an option to enable File Management in edit menus -> Preferences anymore - had to create one using the above as the launch command
