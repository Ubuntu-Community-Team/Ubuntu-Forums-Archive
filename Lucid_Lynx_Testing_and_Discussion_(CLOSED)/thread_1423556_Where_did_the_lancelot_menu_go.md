### Post by Reiger on 2010-03-07
Aptitude to the rescue!
```
sudo aptitude search lancelot
p   liblancelot-dev                                                      - development headers for liblancelot
i   liblancelot0                                                         - library that contains all UI widgets and layouts used in the Lancelot project
i   plasma-widget-lancelot                                               - addons for KDE 4 Plasma - lancelot widget
```
