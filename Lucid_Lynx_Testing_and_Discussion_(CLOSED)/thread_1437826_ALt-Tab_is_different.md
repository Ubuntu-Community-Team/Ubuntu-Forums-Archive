### Post by uBeer on 2010-03-24
It's by design as you see only see the window that is selected by Alt-Tab. Instead of darkening all the other windows they decided it would be nice to have them become totally transparent instead.

I actually like it, because the darkening of the other screens actually took my focus instead of the window that was about to be selected...
