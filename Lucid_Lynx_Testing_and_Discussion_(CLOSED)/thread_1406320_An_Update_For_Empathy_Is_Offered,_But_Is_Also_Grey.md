I do not know where you are getting  this from.  If it is Update Mangler quit using it and check synaptic (which is more dangerous than apt).

Click on "status" (bottom left of screen) and then, straight up from there you will see "installed upgradable".

Left click on empathy and then on "mark for upgrade".  This should give you an error saying that it can't be upgraded because it needs some things that are not installed (empathy-common and something with nautilus that I can't remember).  Install them and then try it again.

This should, when you hit the apply button, give you a long list of stuff that is to be removed.  If it all relates to empathy you should be safe to go on.

If you go to (in synaptic) Settings>Preferences and check all but the very last box there you will have the best gui tool for updates there is.

Update Mangler should just be forgotten with stable versions.  On testing versions it is just a wreck waiting to happen.
