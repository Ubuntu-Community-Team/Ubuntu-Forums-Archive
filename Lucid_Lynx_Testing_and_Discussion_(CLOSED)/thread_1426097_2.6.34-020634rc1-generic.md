### Post by moviemaniac on 2010-03-28
I rarely use the ubuntu mainline ppa - I usually compile my own versions - it only takes 3/4 of an hour from downloading the source to the rebooted machine with the new kernel installed and in general it works better (and you can remove all the unneeded stuff from the config).
