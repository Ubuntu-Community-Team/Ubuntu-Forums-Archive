Sure - just go to [http://launchpad.net/](http://launchpad.net/) and start an account, if you haven't already.

After you have the account, read through this guide on how to file a good (meaning one that will best help the devs diagnose and fix the problem) bug report and file it.

[https://help.ubuntu.com/community/ReportingBugs](https://help.ubuntu.com/community/ReportingBugs)

Keep an eye on it and provide any additional information that is asked for and you'll not only be solving it for yourself, you'll be solving it for many others.

Thanks for taking the time to do this - this kind of thing is really the whole point of being a beta tester.

Regards,

Didius

P.S. Since Lubuntu hasn't officially been added to the Ubuntu family (it's a customized offshoot which has changed some of the underlying architecture) you'd probably get more knowledgeable help from the Lubuntu site. You'll find a lot of links on this site:

[http://lubuntu.net/](http://lubuntu.net/)

Start with the **About** page.

Also, here is the Lubuntu branch of Launchpad: [https://launchpad.net/~lubuntu-desktop]("https://launchpad.net/%7Elubuntu-desktop")
