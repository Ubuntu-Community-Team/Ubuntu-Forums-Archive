### Post by Didius Falco on 2010-04-15
The package freeze is past for 10.04, but you can get the deb file for it directly from the gnome ftp site:

[http://ftp.debian.org/debian/pool/main/g/gnome-icon-theme/](http://ftp.debian.org/debian/pool/main/g/gnome-icon-theme/)

Just be aware, if it does cause any problems (unlikely, but still...) the Ubuntu devs won't offer any support for it. You'd have to file bug reports with the gnome devs.
