### Post by blackened on 2009-03-15
Most of the time when Compiz disables something it is due to a keyboard shortcut conflict. If you disabled the cube plugin, then you should be able to re-enable the workspace switcher plugin.

Open Compiz Config Settings Manager, under Desktop, enable Viewport Switcher.

If this doesn't work, then double-check the settings under "General Options", "Desktop Size" tab.
