[QUOTE=Steve S.]Seriously?  No one has any ideas how to fix this one?  Seemed like it might be a simple fix, one that I just didn't know...[/QUOTE]

I seem to remember seeing something relevant to this once, looking through various settings - let's see... Aha! Thought so!

Go to System->Preferences->Sessions, then check "Automatically save changes to session". I *think* that's what you're after - I've never used Hoary so I can't say for sure, but Breezy had a similar prompt on logout that used the "save changes" wording.

Hope that helps!
