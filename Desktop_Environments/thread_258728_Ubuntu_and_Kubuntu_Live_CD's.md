### Post by aysiu on 2006-09-16
Sorry--you can't.

The live CDs do not have the packages for the desktop environment. They have only *build-essential* and *ndiswrapper*.

If you want to install a desktop environment off CD, you'll need the **Alternate CD**, not the **Desktop CD**.
