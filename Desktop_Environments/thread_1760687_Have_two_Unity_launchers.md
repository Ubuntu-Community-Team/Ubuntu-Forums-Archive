### Post by potat on 2011-05-17
I don't think there's a way to do this directly using Unity as of now. However, check out AWN (avant-window-navigator). It provides a dockbar that can be used in addition to Unity. You could configure it to have all the launchers you want and remove all the launchers from Unity so that open programs showed up there.

Hope this helps.
