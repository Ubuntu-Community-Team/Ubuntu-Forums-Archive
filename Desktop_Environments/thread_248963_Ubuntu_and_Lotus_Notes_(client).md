### Post by NetworkGuy on 2006-09-01
This thread is a great howto on installing the Linux version of Lotus Notes.  The same version that is ported for RH & SUSE will work for Ubuntu.

[http://ubuntuforums.org/showthread.php?t=222492](http://ubuntuforums.org/showthread.php?t=222492)

Read the entire thread as a CUPS issue stops Notes from loading at first, but the fix is in there (you need a default printer defined in CUPS)

As to getting the version..  I don't believe you can get the client for free anymore, but if you have a friend at one of your clients, they can probably sign into their passport account and get the .zip file for you.
