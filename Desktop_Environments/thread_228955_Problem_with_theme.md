### Post by ardchoille on 2006-08-04
That is a problem with the theme itself. What you showed in your screenshot is a problem with the Metacity theme you are using - Metacity is the default window manager in gnome. You can try and fix it yourself by going into the theme's xml file. It will be in /usr/share/themes/<theme name>/metacity-1/* but I would suggest contacting the author and informing them of the problem.. they may be able to send you a fix.
