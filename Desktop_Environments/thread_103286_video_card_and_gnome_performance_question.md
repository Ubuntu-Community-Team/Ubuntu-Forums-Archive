### Post by cow_racer on 2005-12-13
Can the speed of X improve by using a better video card?  Current I have a Nvidia Geforce with 32MB.  X is fast enough, but I would like it to be faster.   Would getting a new video card with 64MB or 128 MB memory improve X performance for me?

---

### Post by RAOF on 2005-12-13
If you're already running the nvidia-glx drivers, the short answer is: not much.

The longer answer is: in some time (a year or two) the X org guys will have a working GPU accellerated X server.  At this point, a faster videocard with more ram will prove beneficial.  For the moment, there's not a lot of X accelleration going on, really.  Your 32MB geforce should be fine.
