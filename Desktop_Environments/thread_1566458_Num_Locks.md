### Post by DonEsteban on 2010-09-02
Do you really need NumLock on *before login*? If not, try the solution in the first paragraph of that page. If you do, just edit the file **/etc/gdm/Init/Default **as described in the second paragraph. Hardy and Gutsy were two (rather oldish) [Ubuntu Releases.]("http://en.wikipedia.org/wiki/Ubuntu_%28operating_system%29#Releases") Synaptic is the (graphical interface to the) ubuntu package manager: System -- Administration -- Synaptic Package Manager. 
Good luck!
