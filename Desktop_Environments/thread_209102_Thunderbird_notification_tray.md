### Post by msak007 on 2006-07-04
I was looking for the same thing myself the other day and found this: 
[URL="http://moztraybiff.mozdev.org/"]
http://moztraybiff.mozdev.org/[/URL]

Not sure why it's not in the official Thunderbird extensions site, but I've read that it takes forever to get an extension approved and added. Anyway, I installed it and it works as advertised. The only thing I didn't like about it is that there's no option to alter Thunderbird's behavior so that "closing" the application by clicking on the X minimized it to the tray, as I'm used to with all apps that have the minimize to tray feature built-in. Instead, you have to minimize it to get it to the tray. A minor quibble, but it's nice to have the minimize to tray feature. IMHO this is a feature that should be built-in by default.
