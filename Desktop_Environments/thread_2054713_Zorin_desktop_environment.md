### Post by 2F4U on 2012-09-08
Zorin is a Linux distribution on their own and, as far as I can see, the Zorin desktop is not available in Ubuntu repositories. So, either install Zorin or add their repositories and install the desktop that way. The last option is much likely to cause problems, which you have to solve on your own.
