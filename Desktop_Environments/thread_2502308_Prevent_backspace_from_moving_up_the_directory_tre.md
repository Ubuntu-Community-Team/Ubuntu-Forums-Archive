Feel free to use a different file manager.  There are about 20 of them for your consideration. Nautilus isn't the ONLY file manager in the world.
I've been using **pcmanfm** when I need a file manager for a few years.  Mainly, I use it when drag-n-drop is needed into a few programs with terrible File--> Open dialogs.
