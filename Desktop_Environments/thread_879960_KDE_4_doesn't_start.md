### Post by kernco on 2008-08-04
I have been having trouble using KDE 4 on my laptop.  I have been trying on different versions from a beta of 4.0 to the 4.1 release.  kdm works fine, but after the KDE splash screen ends, it seems that kdm crashes and I end up back at the login screen.  I never get to an actual desktop.  I have asked about this on #kde and they said it sounds like a driver problem or that it's a problem with the distribution I'm using.  I am skeptical about it being a driver problem because I use Gnome and KDE 3 without any problems.  The strangest thing about this, though, is that if I use the Kubuntu 8.04 KDE 4 Live CD, KDE 4 works fine and I can use it.  Once I install it to the hard drive, though, it stops working.  My laptop's motherboard is Via K8M890 with an integrated graphics chip, S3G DeltaChrome.  I'm using the openchrome driver.
