### Post by wasd591 on 2009-12-20
Let's try going to the main themes folder.
Goto my computer (Places > My Computer) and click *My Filesystem*, then click on home. If you don't see a folder called .themes, press ***Ctrl + H***. If you still don't see it then reply. If you do, double click and delete the folder with the theme inside it. Then restart and try installing it again.
