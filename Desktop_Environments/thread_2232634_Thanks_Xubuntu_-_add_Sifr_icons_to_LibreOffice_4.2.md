### Post by slickymaster on 2014-07-03
Hi Ghune, welcome to the forums.

To install and enable Sifr Icon theme, open your terminal window, hitting Super+T keys for the effect and run the following:```
sudo apt-get install libreoffice-style-sifr
```After that, open LibreOffice, click on Tools > Options > View and change the icon style to Sifr.
