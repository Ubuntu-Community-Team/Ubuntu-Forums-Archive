### Post by Steepike7 on 2011-10-11
I'm a bit torn between Gnome 3 and Unity. I've been using Gnome 3 in Fedora 15, but I kind of like what I'm seeing with 11.10 and I'm thinking of switching back. One thing I like is the absence of icons or a launcher on the desktop in Gnome 3. Initially, I didn't care for it so much but after using it for a while I've become used to it and don't really want a launcher on the desktop.

Is there a way to remove this launcher from Unity, or would I have to go with Gnome 3 to get it so that programs and the like only show up when I mouse into the top left corner?

Thanks, guys.

### Post by Copper Bezel on 2011-10-11
If you don't have it installed, get the package compizconfig-settings-manager. Once you open it, in the Unity plugin options, you can configure the Launcher Reveal Mode to show the launcher only when you hit the upper left corner (Shell style) or not at all and change the behavior to "autohide."

### Post by grahammechanical on 2011-10-11
You can use the same Unity plug-in to make the Launcher background transparent. The icons will still show. There is a slider that sets opacity. In the same way you can set the top panel transparent. It is in the experimental tab. With the Launcher set to auto-hide you get a screen that is all background image except for some small text at the top right and left.
