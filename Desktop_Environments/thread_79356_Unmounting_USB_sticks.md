### Post by skar42 on 2005-10-20
I tend to leave the stick in place for a minute or so before removing. I have had the odd case of corruption. But nothing to worry about. The data will write to the drive when it has a free cycle. Most of the time this is done almost immediately.

### Post by YanH on 2005-10-20
I tend to umnount USB devices on my machine just to be on the safe side. 

However if you are trying to explain this to non computer people then there is no difference between the way things work in Ubuntu and the windoze world. If you simply pull a memory stick out of a windoze box then it pops up a warning dialog telling you that you should have stopped it first. There is an icon on the taskbar that allows you to 'Unplug or Eject Hardware'.
