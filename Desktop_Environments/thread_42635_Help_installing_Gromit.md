### Post by SGC on 2005-06-18
Download the debian package from here:
[http://packages.debian.org/stable/x11/gromit](http://packages.debian.org/stable/x11/gromit)

And install it from the terminal with the following command:
dpkg -i gromit_20041213-3_i386.deb

Make sure to change the name of the package , if you are using non 1386 architecture
