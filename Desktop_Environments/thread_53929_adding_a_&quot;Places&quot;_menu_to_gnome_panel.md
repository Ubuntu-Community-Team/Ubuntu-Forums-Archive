### Post by mavicdog on 2008-01-09
Better late than never I guess. You have to add a bookmark in nautilus for the folder you want to add to the places menu.
From the help:
6.6.18.1.&#8195;Adding a Bookmark

To add a bookmark, open the folder or location that you want to bookmark, then choose Places &#9656; Add Bookmark.

If you are using a Nautilus browser window, choose Bookmarks &#9656; Add Bookmark.
