I am a great lover of Wine and free and Open Source software make for Windows :lolflag:

It starts also great under KDE4!

Or not really Open Source but free.

Why :

In poor countries and difficult environment for financial reason (schools, associations, etc.), the old computer need only a few resource from them needed by Linux excepted using different more experimental systems like Basic Linux, Small Dawn Linux, or Puppy Linux etc.

So I find good that very efficient solutions appears like KDE4 to give access to unlimited professional abilities, but I continue to use MINIMAL SYSTEMS in Windows environment where possible, and, in Linux, in Wine.

Different brilliant applications do the same or better than extremely complex programs :

- [http://members.ozemail.com.au/~nulifetv/freezip/freeware/edxor.htm](http://members.ozemail.com.au/~nulifetv/freezip/freeware/edxor.htm) (under 15 kByte)
- [http://www.movsd.com/thegun.htm](http://www.movsd.com/thegun.htm) (under 10 kByte)
[http://www.nakka.com/soft/npop/index_eng.html](http://www.nakka.com/soft/npop/index_eng.html) (best email client under 100 kByte zipped)
(it seems that the author has also a minimalist "web able tablet application" like within "Windows Meeting", an application missed until now in our Linux environment)
this program is great because it works without changes from windows and from wine and you can take it on a USB stick, no, it is to big: a floppy disk is enough! try to do it with your actual client to take it with you in your pocket between home and office or on travel!
npop can be combined with Lem to see HTML-emails correctly. Lem works also fine in wine and is of the smallest HTML viewer in the Linux environment (193 kByte zipped see [http://npopqfr.free.fr/?Consulter/Le_format_HTML](http://npopqfr.free.fr/?Consulter/Le_format_HTML) and the difficult to see download on the page [http://npopqfr.free.fr/download/lem.zip](http://npopqfr.free.fr/download/lem.zip) )
- [http://www.goosee.com/evewe/](http://www.goosee.com/evewe/) (a marvellous picturing application under 100 kbyte able to cover the sky with most amazing drawings, this means without practical size limit! designed by the creator of Puppy Linux himself, Barry Kauler, a great linuxian!)
- [http://offbyone.com/offbyone/](http://offbyone.com/offbyone/) may be the world's smallest and fastest web browser with full HTML 3.2 support. It is a completely self-contained, stand-alone 1.2 MB application with no dependencies on any other browser or browser component!

I like to use also the Windows FileZilla version because it works on both side without change: in Windows and the same version at the same place in Wine from the place at the Windows partition. the brilliant effect is all the pointer stay usable!

As text processor, I like [http://www.angelicsoftware.com/en/angel-writer.html](http://www.angelicsoftware.com/en/angel-writer.html) . it works fine in both windows and wine.

As PDF-viewer [http://blog.kowalczyk.info/software/sumatrapdf/](http://blog.kowalczyk.info/software/sumatrapdf/) free and Open Source (1 MByte zipped)

As Commander and compress tool of course [http://www.7-zip.org/](http://www.7-zip.org/) what else!

As dictionary one of the free 190 dictionaries from [http://www.freelang.com/dictionnaire/index.html](http://www.freelang.com/dictionnaire/index.html)  :guitar: 

As phone I did try [http://www.wengophone.com/index.php/mp_download_wp_win](http://www.wengophone.com/index.php/mp_download_wp_win) . it works in Wine and the webcam starts but I have no wengophony partner so I did not really test it! And it is not really a minimalistic solution: it requires 20 Mbyte room on the disk.
I suppose that the best minimalist way is always [http://gphone.sourceforge.net/](http://gphone.sourceforge.net/) in linux!

Same for presentation, It did use [http://www.pcwelt.de/downloads/unternehmenssoftware/buero-anwendungen/19536/](http://www.pcwelt.de/downloads/unternehmenssoftware/buero-anwendungen/19536/) in Windows in the very small Cygwin Version! Notice please that Basic Linux presents Magic Point as minimalist freehand drawing system! Beautiful! But I don't know if Basic Linux knows a way to save the made drawings... And with Eve you can do all what you wish perfectly without limit with only 100 kByte ;)

What I am missing?
