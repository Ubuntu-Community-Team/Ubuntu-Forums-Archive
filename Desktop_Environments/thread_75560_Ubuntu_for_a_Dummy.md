### Post by Stormy Eyes on 2005-10-14
[QUOTE=BobInIdaho]C: drive (windows xp) uses fat32.[/QUOTE]

Thanks. I was able to determine that by looking at your FDISK output; I just had to figure out how best to explain the next step.

Step 1: Create a mount point for your windows drive...

```
mkdir /media/windows
```

Step 2: Alter /etc/fstab to mount your windows drive automatically...

```
sudo gedit /etc/fstab
```

Paste the following line on a new line at the end of the file:

```
/dev/hda1 /media/windows vfat umask=000 0 0
```

Save your changes and close gedit. Then run **sudo mount /media/windows** to mount your Windows partition. The next time you reboot, Ubuntu should mount it for you.
