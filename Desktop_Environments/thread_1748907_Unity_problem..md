### Post by ivanovnegro on 2011-05-04
You have to enable Unity again, you need Unity for Ubuntu, at least for this session.
Type in a terminal:

```
unity --reset
```.

If you dont like the interface, logout, choose your user name and at the bottom choose Ubuntu Classic, that will boot your computer without the Unity interface.
