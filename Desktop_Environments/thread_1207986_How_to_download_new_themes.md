### Post by cph05a on 2009-07-08
For gtk and metacity themes:
download the theme which will be a .tar.gz file but don't extract it. Then go to System > Preferences > Apearance and click Install. Then find the .tar.gz and you should have your theme.

For Emerald/Beryl themes:
make sure emerald is installed.
download the theme which will be a .emerald. Go to System > Preferences > Emerald Theme Manager and click install, then find your theme.
