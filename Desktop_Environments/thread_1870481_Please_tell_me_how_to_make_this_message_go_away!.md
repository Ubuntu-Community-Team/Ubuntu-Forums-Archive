### Post by Cope57 on 2011-10-27
It is normal for the message to appear for the applications that you have running.  It is so the application could be dragged to another desktop/workspace if one desires.

If one has a different application running, it would also be mentioned in the message.

To remove the messages, move cursor off the workspaces windows, and use your keyboard to switch between desktops/workspaces. Ctrl+Alt+Left arrow and Ctrl+Alt+Right arrow should work with no modifications.
