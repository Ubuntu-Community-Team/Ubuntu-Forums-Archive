To install KDE, install the **kubuntu-desktop** package from Synaptic, or by using **aptitude** or **apt-get**. If you use aptitude, you will be able to unistall it easily if you're not happy with it. Type this in a terminal:
```

sudo aptitude install kubuntu-desktop
```

To separate the desktop environments, check out these threads:
[http://www.ubuntuforums.org/showthread.php?t=123969](http://www.ubuntuforums.org/showthread.php?t=123969)
[http://www.ubuntuforums.org/showthread.php?t=59455](http://www.ubuntuforums.org/showthread.php?t=59455)
