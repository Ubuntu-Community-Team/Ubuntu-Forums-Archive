### Post by fluffington on 2006-06-09
All I can think of is boot-admin, which is part of the [GNOME System Tools](http://www.gnome.org/projects/gst/). Ubuntu includes the package gnome-system-tools (it's in main and part of the default install), but doesn't include boot-admin. Here's the [bug report](https://launchpad.net/distros/ubuntu/+source/gnome-system-tools/+bug/34302) that explains why.
