### Post by urukrama on 2008-02-12
Xcompmgr only provides shadows and fading menus; for transparency you can use it in combination with transset. For more info on xcompmgr see [here]("http://ubuntuforums.org/showthread.php?t=75527"). In the Openbox guide linked to in my signature transset and xcompmgr are briefly discussed as well.

Wobbly windows won't work, but if you like 3d desktop switching, you can install 3ddesk.

If you want to have something like the expose feature, try skippy. If you want moving applications to create waves/ripples, have a look at xdesktopwaves.

I believe AWN works with xcompmgr, though I have never attempted to try it. Alternatively, you can use something like wbar.
