### Post by bagel on 2006-02-20
Are you talking about gmail-notify?

The command line for starting it is: 
```
gmail-notify
```

Easiest way to install it:
```
sudo apt-get install gmail-notify
```

I have used it for some time, but now I like the gmail-notification plugin for firefox better.
