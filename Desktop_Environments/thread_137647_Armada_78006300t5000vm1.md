### Post by zxee on 2006-02-28
Don't know what graphics card is in your Armada, but it might not be a driver issue. Check out the search link here: [http://www.ubuntuforums.org/search.php?searchid=4059099](http://www.ubuntuforums.org/search.php?searchid=4059099)
and look at your xorg.conf file found at /etc/X11/xorg.conf
you may just need to edit that file to provide the screen resolution options you want. If you open your xorg.conf file with a text editor (gedit) you should be able to scroll down and toward the end of the file see what resolutions are configured. Hope that helps.
