### Post by lolpenguin on 2012-08-19
GNOME 2.x or GRUB legacy??
I assume you want GNOME 2.x
First, there is Unity 2D. It works well on lower-end computers. If you like Unity, then use this. Click on the Ubuntu logo in the login screen then choose Ubuntu 2D.

If you don't like Unity, and want the GNOME Panel, there are two options.
Install the GNOME Fallback Session.
```
sudo apt-get install gnome-session-fallback
```
This looks like the old GNOME, but is GTK3 and you have to Alt + Right Click to access panel and applet options.

Or you can install MATE, a fork of GNOME 2. This is probably unnecessary though. [http://mate-desktop.org]("http://www.mate-desktop.org")
