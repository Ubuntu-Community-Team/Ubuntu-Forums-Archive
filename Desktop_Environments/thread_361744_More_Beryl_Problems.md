### Post by Softpedia on 2007-02-18
Hi,

type this command in a terminal:

*sudo dpkg-reconfigure -phigh xserver-xorg*

choose the nvidia driver from the list... hit enter... and then choose your resolution (1280x1024 i guess) and hit enter to finish the wizard.

then type this command:

*sudo nvidia-xconfig -add-argb-glx-visuals*

Restart the X server with **CTRL+ALT+BACKSPACE** ... re-login and Beryl should work!

### Post by Softpedia on 2007-02-18
I've just reinstalled my Ubuntu today.... and I've decided to install the NVIDIA drivers from the official site with the Envy script. Bad idea :) So I've just read your comments and I tried to run Beryl to see if mine works... Guess what? I doesn't work properly.... it doesn't show the window bars :)

So here is how I've solved the problem:

1. Type in a terminal the following line:

*wget -O- --quiet [http://nvidia.limitless.lupine.me.uk/ubuntu/root@lupine.me.uk.gpg](http://nvidia.limitless.lupine.me.uk/ubuntu/root@lupine.me.uk.gpg) | sudo apt-key add -*

2. Add the following repository to your repository sources... 

*deb [http://nvidia.limitless.lupine.me.uk/ubuntu](http://nvidia.limitless.lupine.me.uk/ubuntu) edgy stable*

3. Type in a terminal the following commands, one by one:

[I]sudo apt-get update
sudo apt-get upgrade
sudo apt-get install nvidia-glx
sudo dpkg-reconfigure -phigh xserver-xorg
sudo nvidia-xconfig -add-argb-glx-visuals[/I]

4. Restart your X server with CTRL+ALT+BACKSPACE and then re-login... and Beryl it's working very well :)

P.S.: When you run the *sudo dpkg-reconfigure -phigh xserver-xorg* command... you should see this wizard I was talking about earlier:
