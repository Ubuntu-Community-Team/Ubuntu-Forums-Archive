### Post by ZBlue on 2010-08-07
For the icon in the top panel you can right click on the shortcut, select "Properties" and then click on the picture box on the left. This should open up the input box where you can point to any icon of your choice for the shortcut.

Not sure if this will affect the icon in the applications menu :)
