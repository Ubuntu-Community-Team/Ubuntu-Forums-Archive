### Post by vigalance on 2008-07-17
How do you add a program to auto-start at boot-up with KDE?

---

### Post by pluviosity on 2008-07-17
It's a bit more complicated than in Gnome, but still manageable.

1) In Konqueror, open /home/username/.kde/Autostart, where username is your username.  

2) Right click and select Create New --> Link to Application.

3) Change the name to what the app you are having start up for your reference (eg. Amarok)

4) Under the Application tab, type the command for the program you want to start up (eg. amarok).

5) Click OK, and you should be set.

To stop this program from autostarting in the future, just go back to the folder and delete the link.
