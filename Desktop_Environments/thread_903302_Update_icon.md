### Post by mocoloco on 2008-08-28
Just FYI, I would recommend you always install all updates, since they often fix bugs and security issues.  But if you really want to keep some packages from updating:
System -> Administration -> Synaptic.
Search for and select one of the packages.
Click Package -> Force Version.
Select your current version.  It shouldn't offer those updates to you anymore.
