### Post by Landshark on 2011-01-11
Hello. I am running Ubuntu 10.04 64-bit using e16 as a windows manager. One application -- Firefox -- runs fine, but it appears maximized and with no window border.

When I attempt to resize the window, the big Gnome X cursor appears as opposed to the standard one. I can't resize the window or set an e16 border. No settings are remembered for this application.

Any thoughts?

Edit: Figured it out. ALT + right-click on the window. Go to window size. Select Full size / Normal. That will reduce the size of the window and return your border.

Selecting "Max Size Toggle" is what will create the situation that I encountered. It's a bug in e16, because once it's full size e16 no longer controls the window.

e16 rocks, but it can be fussy.
