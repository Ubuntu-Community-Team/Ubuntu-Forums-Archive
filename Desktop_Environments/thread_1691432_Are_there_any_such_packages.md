### Post by deconstrained on 2011-02-20
The answer is all in OpenOffice.

You can create equations in OpenOffice Formula and then copy/paste them into OpenOffice Draw, but Formula, while it's somewhat easy to use, doesn't use LaTeX (the universal standard in mathematical markup languages) but its own special syntax (why on earth the developers would set it up this way I do not know). 

If you don't feel like learning some B.S. single-purpose technical markup language you can always just use the UI for inserting and editing the bits and pieces of an equation.

Good luck!
