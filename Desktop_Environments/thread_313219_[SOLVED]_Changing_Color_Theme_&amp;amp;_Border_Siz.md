### Post by IYY on 2006-12-05
You can do all of that by editing the XML file called metacity-theme-1.xml. It will be located in the specific theme's directory, for example 

```
~/.themes/Redmond/metacity-theme-1/metacity-theme-1.xml
```

The file is pretty straight forward to understand...
