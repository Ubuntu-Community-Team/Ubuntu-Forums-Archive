### Post by kpkeerthi on 2006-06-26
1. Install network-manager-gnome
2. From the terminal, type ** nm-applet** to start the applet. (This is required only once. Subsequent boots auto starts the applet.) 
3. Click on the applet icon in tray to view all available networks.
