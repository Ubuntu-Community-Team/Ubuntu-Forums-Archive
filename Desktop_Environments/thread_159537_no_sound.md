### Post by gingermark on 2006-04-14
Sorry you decided to give up. Like any OS, Linux does take some setting up depending on your hardware.

I have a different motherboard, but the same Realtek Ac97 codec. As my card uses nVidia nForce, I installed their driver from their website, and set the sound system to OSS. Sound then worked fine.

A quick search of the forum for "realtek ac97 sound" threw up several threads, including [this one](http://ubuntuforums.org/showthread.php?t=150420&highlight=realtek+ac97+sound) which talks about installing Realtek's drivers. It's a tad more complicated.

Because there are so many Linux distros about drivers can be problematic if the manufacturer doesn't permit them to be distributed in a distro's native format. But it's quite do-able.

If you decide to give it another shot there is a specific forum for Hardware problems.

All the best.
