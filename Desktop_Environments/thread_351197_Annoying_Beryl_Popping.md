### Post by hise0001 on 2007-02-01
It might be that when pressing the key combination focus is going to the desktop and then when done focus is going back to the previous window.

There is a beryl setting for animation to occur when a window takes focus.  Look at:

Visual Effects -> Animations -> Focus Tab and make sure "Animation" is set to None instead of Wave.

--Hise
