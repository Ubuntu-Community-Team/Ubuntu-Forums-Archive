### Post by buzzingrobot on 2013-11-07
Open a terminal and execute this: "sudo nvidia-xconfig".  The xorg.conf file will be created.  If one already exists, it will be backed up.

I've found that in an annoying number of cases, the Nvidia installer fails to run nvidia-xconfig.
