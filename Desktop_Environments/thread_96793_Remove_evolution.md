### Post by jlx on 2005-11-29
ubuntu-desktop is a metapackage. It's only use to ensure you have all the needed packages for the functionality a desktop user should have (email, multimedia, browser, office suite, etc). You can remove it and your system will not suffer. There are always alternatives for the predefined applications.
