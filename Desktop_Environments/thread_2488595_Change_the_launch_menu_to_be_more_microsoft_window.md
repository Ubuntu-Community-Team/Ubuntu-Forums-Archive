### Post by Holger_Gehrke on 2023-07-10
If it's possible, it should be a Gnome Shell Extension. Since I'm using XUBuntu myself - which uses a different desktop environment and is more like Windows 7 in appearance of the menu - I'm not quite sure how you install those. I think you need the package gnome-shell-extension-manager for that. A look at [https://extensions.gnome.org](https://extensions.gnome.org) shows one named 'Applications-Menu' which might be close to what you're looking for.

### Post by Tadaen_Sylvermane on 2023-07-10
For the vanilla Gnome Ubuntu you can get an extension called ArcMenu. With a few other tweaks to some hotkeys and disable hotcorners via dconf-editor you can get pretty close. It has a number of menu layouts. One like Win 11 as well. The only thing I haven't found is how to move the bar up top to the bottom. But it's not something I need so I haven't really put any effort into finding that.
