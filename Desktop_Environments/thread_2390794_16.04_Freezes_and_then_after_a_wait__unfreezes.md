### Post by cruzer001 on 2018-05-04
Last 100 lines is good.

tail -100 **pathtofile** > **nameoflog**

example:
```
tail -100 /var/log/kern.log.1 > kernel_log
```
You now have a file in your Home directory called "kernel_log".

An easy way to post this log is to install pastebinit:
```
sudo apt install pastebinit
```
and then
```
cat ~/kernel_log | pastebinit
```
Then wait for it and (still in terminal) it will return a link.  Just post the link.

[https://help.ubuntu.com/community/Pastebinit](https://help.ubuntu.com/community/Pastebinit)
