### Post by blk_jack on 2009-10-24
Unfortunately there is no way of doing this without modifying the source code yourself.  I was able to get notify-osd working without a gnome-panel and focus follow, though.

That being said, you might want to look at focus-follow (key in gconf, google it).  While it doesn't display the notice on both monitors, it WILL display the notice on the monitor that currently has the focused window.
