### Post by Krytarik on 2011-03-10
This keeps on bugging me too. However I just recently re-enabled the minimize function.

If you are using Compiz, you can disable it for all windows with its "Window Rules" plugin:
- "System -> Preferences -> CompizConfig Settings Manager -> Window Management -> Window Rules"
- enter "any" into the field "Non minimizable windows"
