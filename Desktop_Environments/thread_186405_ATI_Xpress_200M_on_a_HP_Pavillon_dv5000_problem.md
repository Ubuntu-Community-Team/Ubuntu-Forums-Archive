### Post by Charon on 2006-06-02
I had the exact same problem with my laptop (a HP nx6125) and the solution was pretty easy :

go into synaptic and search for the restricted kernel modules for the current kernel. right-click on it and force synaptic to reinstall them. reboot and that should be it.

it worked for me and i have read a few threads around here where people had the same problem and got it working with this too.

[edit]
before doing this you should already have the fglrx-drivers installed.

did i understand it right that X doesn't start at all with them on his laptop?
[/edit]
