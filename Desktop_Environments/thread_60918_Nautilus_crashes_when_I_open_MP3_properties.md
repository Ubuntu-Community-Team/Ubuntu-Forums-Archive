### Post by xaque on 2005-08-29
I figured out a workaround. If you disable previewing files in Nautilus, it won't crash anymore. Disabling audio files only fixed OGG, apparently MP3 and AAC are considered to be in the "Other Previewable Files" category.

EDIT: This doesn't always fix it... it seems to be about 50/50 whether it will crash or not now.
