### Post by Zeike on 2010-07-25
The easiest way to do this I've found is to use the "wallpaper" plugin in compiz, which easily lets you set different wallpaper for each desktop.  But in order to get it to work you have to disable nautilus from using the desktop (so no desktop icons).
