### Post by amunimanghi on 2006-05-30
Yes, you can. Right click the file(s) and select 'Stretch Icon'. After, you will see a knob on each corner of the icon. Stretch it as you like or shrink it. If you dont like it, there is a option called 'Restore Icon's Original Size'.
