### Post by icanfly0307 on 2009-01-23
I don't think you can get everything under the Desktop folder to show on the Desktop in KDE. However, depending on the version you're running, you could try the Folder View widget. It'll let you view the contents of any directory that you specify.

If you want the traditional icons, just drag and drop them onto the desktop. Hope this helps. :)

Release candidates are out now and easy to install. Kde4.2 final is due next week.
