### Post by DuckHook on 2016-05-03
Canonical assumes that LTS users value stability above all else. Xenial still has plenty of bugs (just look through these forums) so the default is to not upgrade from Trusty until the first point release (16.04.1) in about 3 month's time. There are workarounds to force an install now, but it is not recommended.
