I'm new to Ubuntu and have a printer problem.
My setup is a 4 computer home network, originally all Windows OS connected thru a Siemens print server which is then connected to the router for internet access. My business laptop still runs XP pro, and a desktop still runs Win ME.  The other two computers are a desktop and an Acer laptop both running Ubuntu.
The first issue... the Ubuntu machines, using Samba, can only print if the XP Pro laptop is connected to the network and turned on.
The second issue... the Ubuntu computer print jobs, sent to the printer prior to turning on the XP Pro laptop, will not print and cannot be cancelled.
Can anyone lead me in the right direction to clear up these two issues?
