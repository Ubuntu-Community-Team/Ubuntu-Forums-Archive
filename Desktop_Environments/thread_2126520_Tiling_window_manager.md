### Post by MG&amp;TL on 2013-03-17
I use DWM on a regular basis, which is awesome (no pun intended!) and really configurable (you have to recompile the source code to configure it, so in a sense it is ultimately configurable if you can configure it at all) and I find it speeds up my workflow. I've tried others, but I find that DWM suits me just fine.

They're all...ah...*different*, so I suggest you just pick one at random that isn't too crazy and just see if you like the tiling concept.
