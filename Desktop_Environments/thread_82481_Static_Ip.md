### Post by Dr. Nick on 2005-10-26
Click System->Administration->Networking then hit properties on your lan card and check the Configuration blank.

This will modify your /etc/network/interfaces file. You can open that file to see how you could set it manually if you were ever without a GUI and needed to get it done
