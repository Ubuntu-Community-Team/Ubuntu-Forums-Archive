### Post by paul2005 on 2006-03-11
Just select a new theme and click close, that should change theme for you. If you want to download a new theme from the web (art.gnome.org for example) you can save it to your home directory and then click Install theme to open it from there.

### Post by paul2005 on 2006-03-11
I think that's separate, the themes just change window borders, icons etc... not the wallpaper. You can change wallpaper by right clicking on the desktop and selecting a new one.
