Under windows, ctrl+tab would swap the active tab in firefox. However, under KDE it swaps between active desktops. 
The only similar shortcut I can see is ctrl+[ and ctrl+], however neither of these works.

So my question is, what shortcut can I use to swap tabs, and if none, how can I set one? I wouldn't mind losing the ctrl+tab to do this, as I don't make much use of multiple desktops (dual screen means I have plenty of space anyway).

### Post by aysiu on 2006-01-15
Alt-F2
```
kcontrol
```

Go to Regional/Accessibility > Keyboard Shortcuts

Near the top, you'll see some command already assigned to Control-Tab and Control-Shift-Tab.

Unassign those commands. Firefox should then work fine.
