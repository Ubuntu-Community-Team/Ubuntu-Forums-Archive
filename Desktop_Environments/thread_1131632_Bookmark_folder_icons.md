### Post by jesterthejedi on 2009-05-16
GNOME FOLDER BUG: The way I fix this is ALT+F2 type in "gksu nautilus" (your password) navigate to /usr/share/icons hit CTRL+F, type "inode" Select all with mouse then hold down "SHIFT" and press "DELETE". If your updates install the Human or Gnome-Icon-Theme you may have to repeat this step to get drag and drop back and places back.
