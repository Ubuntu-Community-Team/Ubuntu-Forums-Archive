### Post by Ramses de Norre on 2006-06-12
Yes that does install all packages that would be installed from the disc image. You'll be asked whether you want to use gdm (gnome) or kdm (kde) for the login screen and I think the splash is set to kubuntu too (you can change it with sudo update-alternatives --config usplash-artwork.so).
