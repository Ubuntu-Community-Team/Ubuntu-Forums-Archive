### Post by drummer on 2006-02-01
Check out the gdm themes (in System > [can't remember the menu, second in the list] > Login Manager) and click the 'graphical login' tab. Whether the user list displays or not is dependent on the theme. You can also download new themes from gnome-look.org and install them in that same tab (click 'install theme').
