### Post by mlomker on 2005-09-07
DJ is correct.  If you'll never use something again then you can use complete removal.  I add/remove kernels all of the time tweaking my system and the complete removal will get rid of the /lib/modules files along with the kernel itself...leaving the system a lot cleaner.
