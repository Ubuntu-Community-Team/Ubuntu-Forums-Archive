### Post by weblars on 2005-05-10
Hi!

If your desklet uses weather data from weather.com you can find the station code for your city at [http://www.aspnetresources.com/tools/locid.aspx](http://www.aspnetresources.com/tools/locid.aspx)

Ljubljana, Slovenia has the code SIXX0002.


I'm using the iWeather desklet and it works fine for me.
