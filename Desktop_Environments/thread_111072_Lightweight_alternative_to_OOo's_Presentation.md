Maybe this will help you out for the creation of presentations.

### Post by Adrian on 2006-01-04
[QUOTE=art]You get a lot of KDE entries in GNOME menu...[/QUOTE]

That will not happen if you just install a single KDE program (and not the entire Kubuntu environment). Of course you will get that program in your menu, but I guess that is what you want anyway :)

The problem using KDE programs in Gnome is...

* They depend on KDE libraries, so they eat more system resources. Usually not a problem for any modern system (since we're not speaking of large amounts of RAM or CPU power here). The KDE libraries will be loaded when you decide to start your program, so you don't have to worry about them wasting resources.

* They don't look like Gnome programs by default (even though you can adjust them to look very similar). Some people find this annoying. Some people don't.

A lot of people are using Amarok and K3b in Gnome, and those are KDE applications (hence, if you use any of those programs, the KDE libs already exist on your system). If it's good, use it. I use some Gnome programs in KDE, and that is the same thing really.
