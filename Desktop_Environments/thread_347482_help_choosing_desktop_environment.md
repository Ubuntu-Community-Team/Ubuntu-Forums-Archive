### Post by taurus on 2007-01-27
If you already have Gnome running on your machine, you can install both XFce4 (Xubuntu) or fluxbox with

```
sudo aptitude update
sudo aptitude install xubuntu-desktop
sudo aptitude install fluxbox
```
Now, you can choose which window manager to use in the Options (bottom left corner) at the GUI login screen.

### Post by trojanek on 2007-01-28
Since you said that you like a text-based environment, why not try one?
[wmii]("http://wmii.suckless.org/") is a great one. It does support GTK+ and Qt, so you can still run all of your graphical programs. It's one that I used to use before I switched back to FVWM. Speaking of . . .
[FVWM]("http://www.fvwm.org/") is another nice window manager. It can look like and behave like whatever you want. It is easily the most customizable window manager that I have ever touched.
