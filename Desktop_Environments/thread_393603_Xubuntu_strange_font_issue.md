### Post by limbert65 on 2007-03-25
I'm running Xubuntu 6.10 with Xfce on a Gateway 450ROG laptop.  Today, on inserting a music CD, the system locked up solid (first time it's ever done that), and I had to force-restart.  After restarting, I found that the system fonts were all screwed up.  The system menu, desktop icons, and window title bar fonts were all messed up, and the spacing on the desktop icons had become very large.  I fixed the system menu font size, the desktop icon font size, and the window title bar font size simply by changing their values in the desktop preferences and window manager preferences.  However, two problems remain.

1)  The desktop icons are still spaced very far apart, and I can't make the spacing tighter.

2)  In only certain applications, the application's menu fonts are very large.  In Firefox, for example, the font in menus is very large, even though the web page text looks fine.  Also, the font in the address bar and the search bar is too large.  It's the same if you bring up the Google web page; the font that page's search bar is way too large.  Changing the font settings in Firefox's preferences changes only the web page text, but not the menu and address bar text.  The same goes for Mozilla Sunbird.  The text in the menus and in the headers for the different sections of the main window is way too large, to the point that it doesn't fit in the space provided.  I also installed Epiphany, just to see what its fonts were like, and at first everything seemed normal (its menus look fine), but when I searched for something in Google, the search text in the search bar was too large, and the text of the results page was tiny.  Increasing the font size for the page text made that text look normal, but also made the text in the search bar even larger at the same time.

Any suggestions on how to straighten out this frustrating problem would be much appreciated.
Thanks!
