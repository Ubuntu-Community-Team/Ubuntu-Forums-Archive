### Post by sikander3786 on 2011-06-13
In Unity, you can press <Super> key to bring up the Dash and then search for the programs. Or press <Super> + A and you'd see the Applications Dash. In the right corner of the search box, you would see an arrow which when clicked, lets you choose the categories like 'Accessories', 'Multimedia' etc.

You can have custom Menus as well. Cardapio is a good alternative.

[http://ubuntu4beginners.blogspot.com/2011/06/cardapio-menu-applet-for-unity-launcher.html](http://ubuntu4beginners.blogspot.com/2011/06/cardapio-menu-applet-for-unity-launcher.html)

Or even, there is a Classic Menu AppIndicator for Unity.

[http://ubuntu4beginners.blogspot.com/2011/06/classic-menu-appindicator-for-unity.html](http://ubuntu4beginners.blogspot.com/2011/06/classic-menu-appindicator-for-unity.html)

### Post by coffeecat on 2011-06-13
> **bill-lancaster said:**
> System settings brings up a panel called "Control Centre"  which contains "Main Menu" with many more application than will fit in the Launcher.

How can I access the "Main Menu" ?

The "Main Menu" entry in the control centre is actually the main menu editor for the classic gnome desktop. You can log into the classic desktop from the login screen but that control centre item is of no use in the Unity desktop. It's a pity it's called "Main Menu". That's confusing. It would have been better if they had called it "Main Menu Editor".

If you like to see a categorised application menu, right-click on the "Applications" icon near the bottom of the launcher - it's the grey one with the + sign behind what looks to me like a magnifying glass. If you click on one of the categories you still get the dash, but filtered for the category.
