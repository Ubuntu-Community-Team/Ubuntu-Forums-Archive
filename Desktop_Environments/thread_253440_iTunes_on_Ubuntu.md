### Post by hotani on 2006-09-08
Sadly, there is no all-in-one iTunes replacement on Ubuntu. You can try the Wine route, but I think even then it is not fully functional. It may play and organize but I don't think it will be able to update your iPod.

Here is what I've found in my search for a native linux iTunes replacement:

**Amarok**
Amarok gets rave reviews by many linux users, but coming from Mac I find it counter-intuitive and a general pain to use. It uses a tree system for browsing rather than the Artist | Album | Library method of iTunes and others. While this may be fine for some, I'd rather the iTunes way. It will see an iPod, but has little or no support for playlists. As for listening to music, it requires a save action whenever a playlist is updated which some (me) may feel is an unnecessary step. Basically, I hate Amarok. I don't care about the glitzy features, and the only reason I'm using it at home is because it is the only player with an EQ.

**Banshee**
Banshee is attempting to be an iTunes replacement, but is missing the browse feature (No Artist | Album boxes above Library) which, after using it for about 2 minutes I realized how much of a necessity this feature can be. It connects to iPods, can see playlists but won't do smart playlists. So Banshee is out.

**Rhythmbox**
Since all I want to do, most of the time is actually listen to music in the background, this is by far the best player out there and the most promising "iTunes replacement." It has the browse windows, it has a logical drag + drop playlist setup. For playing music, this is the best app yet, but it is not yet a full iTunes replacement. Rhythmbox cannot organize your music, because it won't edit the files. You can view file information, but can't edit it. This is where you need to go back to one of the other players for organization like Amarok. iPod support is there, but not fully supported - Last time I tried I believe it had problems seeing the playlists. 

**GTK Pod**
This app is also counter-intuitive, difficult to use, and should have you shouting obscenities in no time. However, it is the only app available that will fully support the iPod. It will do smart and regular playlists and allow updating of any kind on the iPod. I can't wait for this stuff to be implemented in Rhythmbox so I can  get rid of gtkpod.
