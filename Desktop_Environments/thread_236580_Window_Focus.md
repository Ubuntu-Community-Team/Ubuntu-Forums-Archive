### Post by maniacmusician on 2006-08-15
that's kind of weird...i dont about new features or anything, but in xfce, the focus feature is in Applications > Settings > Window Manager Settings, then the "Focus" tab. the only options for focus are "click to bring into focus" or "focus follows mouse"

well actually there is an option at the bottom of that window that seems relevent: Raise on Click. the option is "raise window when clicking inside an application window". in mine the box is checked. So that is probably what you're looking for. but if you don't have xfce, i dunno where it would be.
