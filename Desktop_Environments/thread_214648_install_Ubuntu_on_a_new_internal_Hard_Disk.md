### Post by ncappel1 on 2006-07-12
When the harddrive is correctly mounted and recognized by the computer, you should have no problem at all installing Ubuntu on the new one. During the install process, when the step comes up to partition the drives, simply choose to reformat the new drive and install. The install routine will do everything automatically.

for a very good site documenting all of this:
[http://users.bigpond.net.au/hermanzone/](http://users.bigpond.net.au/hermanzone/)
