Ermm.. The best I can do is:

If you right-click on the sidebar and click New Folder, you can then set the URL of the folder. So you can have several different folders viewable in the sidebar just a click away.

But I don't think you can see other partitions like that, or anything.. Also, it doesn't change automatically.

So it's a bit useless for what you want, so I guess you'll just have to stick with it I'm afraid :/
