### Post by H.E. Pennypacker on 2006-06-29
Nautilus Actions is something that modifies your context menu.  A context menu is when you right-click a file, a menu shows up showing several things, including "Open."  Nautilus Actions allows you to add your own commands to the context menu, such as, for example, opening a file as root so you don't have to go to the terminal and ask for permission there.  In other words, you don't have to go to the terminal, and for example, type in sudo gedit /etc/var/logs.list.  Instead, you can browse to /etc/var/logs.list, right click logs.list, and do "Open as Root."

To add Nautilus Actions to System > Preferences, install Nautilus Actions first.  It will add itself to the list.
