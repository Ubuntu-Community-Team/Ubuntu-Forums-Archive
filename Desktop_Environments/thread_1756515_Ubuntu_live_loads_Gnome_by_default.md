### Post by blauendonau on 2011-05-12
Hello!

As I recall, if you don't have the necessary hardware to run Unity, Ubuntu will default to "classic" GNOME. Perhaps your computer is not very new?

You can switch between the two desktop environments by logging out and selecting a DE from the menu at the login screen. The underlying operating system is the same, the desktop is just the stuff "on top", so programs should function properly under both environments.
