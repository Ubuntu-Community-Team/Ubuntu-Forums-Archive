### Post by Breambutt on 2011-03-09
Both have rather simple solutions.

As I understood this, the login screen background cannot be changed in the current version because the whole desktop system was supposed to be upgraded to the latest version but it didn't work / wasn't ready / something. It was doable before and will be doable again.

Here's an easy solution for that: [http://www.ubuntu-tennessee.org/tutorials/205/](http://www.ubuntu-tennessee.org/tutorials/205/)

As for the terminal, just open a terminal > edit > profiles > edit default (or create new) > ... you'll find an alpha channel slider somewhere over there.
