### Post by deadflowr on 2015-08-09
Super + W is scale, so for that you can use ccsm (compizconfig-settings-manager)
and go to the scale plugin (in window management section) and in the main tab Appearance change the speed there.

Don't really know what you mean, but that's how you can change the speed of those animations...
