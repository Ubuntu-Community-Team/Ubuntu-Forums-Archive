### Post by Sef on 2006-05-17
Two main ways:

1) Through restricted formats:

[https://wiki.ubuntu.com/RestrictedFormats]("https://wiki.ubuntu.com/RestrictedFormats")

An easy way to install them is to copy and paste the commands instead of typing them all out.  The copy and paste commands are under Edit on the menu bar.

2) Use Automatix: 

[http://ubuntuforums.org/showthread.php?t=138405&highlight=automatix]("http://ubuntuforums.org/showthread.php?t=138405&highlight=automatix")

Note: Only available for Breezy at this time.  There will be a version for Dapper when Dapper goes stable.
