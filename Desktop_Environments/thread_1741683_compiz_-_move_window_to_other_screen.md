### Post by Krytarik on 2011-04-28
Maybe you are talking about the "Destop Wall" plugin?:
"CompizConfig Settings Manager -> Desktop Wall -> Bindings (tab) -> Move with window within wall (subsection)"

Of course, you may have to translate some names into German if you have it that way. ;-)

Greetings.

### Post by Krytarik on 2011-04-28
Then maybe "Put To Next Output" of the "Put" plugin?:
"CompizConfig Settings Manager -> Window Management -> Put -> Bindings (tab) -> Put To Next Output (keyboard shortcut)"
