### Post by roaldz on 2007-08-08
yeah, that's simple. I don't use that software, and if I use that combo, I get the same blinkin "-".
You should try hitting CTRL+ALT+F1-F7. These are shortcuts to switch between tty's. You have 7 tty's enabled (by default) to work with. F1-F6 are commandline terminals, and F7 normally contains your graphical desktop (the X server). These can be quite usefull when troubleshooting X problems.
