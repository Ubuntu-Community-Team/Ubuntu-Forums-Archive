### Post by badactress on 2007-04-23
Puppy linux should run happily on your old P1.. you can get it from [http://www.puppylinux.org]("http://www.puppylinux.org"). I have it installed on a USB stick so I can use the same o/s at work and work & it's a fantastic linux distro.

DSL [http://www.damnsmalllinux.org/]("http://www.damnsmalllinux.org/") would also run perfectly, however the default applications in DSL are very basic so it needs a bit of work on the user's part to get it looking good & to have more modern applications. Still a brilliant operating system though & well worth a look..

You could check SLAX too.. [http://www.slax.org/]("http://www.slax.org/") It looks and feels more like a normal, modern operating system and might just about be ok on yr system. It booted fine from a live CD on my old laptop with 64Mb RAM & 400mhz cpu. Ran pretty slowly but I'm sure if I'd installed it to the hard drive it would've been a lot faster..

Hope this helps!

### Post by PurplePenguin on 2007-04-23
Great suggestions!  I'd like to add, though, that if you run Puppy Linux, you'll be running as root the whole time.  Some people might prefer to run something else as a system's main OS.  Puppy's a great rescue OS, though - it's really easy to boot from USB or a disc (into memory, freeing the drive), mount drives and make fixes.

---

### Post by badactress on 2007-04-23
Hi Purplepenguin,

You're completely right. Having to run Puppy as root makes it totally unsuitable as a main operating system. For that reason I'd take it off my list of recommendations & that's such a shame as out of the three I mentioned it's by far my favourite!
