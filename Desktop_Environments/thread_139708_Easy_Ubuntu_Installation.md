### Post by deeminter on 2006-03-04
You need to first download an "iso" image, then burn that image to a "CD-R".

Then once you have the "iso" image on a cd, you need to restart your computer and set up your computer's "BIOS" to boot from "CDROM".

Save the change (1st Boot device = CDROM) by pressing (usually) F10.

Allow the computer to reboot , then when the Ubuntu boot screen appears, follow the instructions.

You might want to go to a web site that gives more detailed instructions on how to install Ubuntu 5.10, and print out the instructions incase you need the later.

Hopefully the above will get you started in the right direction.
