I don't know if you have discussed this topic before but I think it would be nice if all opinions where in one Thread.

Since Unity is still new I believe there's plenty of room to throw in some ideas.
Here's what I'd like to see from future versions of Unity:

Customisation. And I'm not talking just about the colours, the icons, the size. I'm talking about dynamic shaping, positioning and functions.
I want to have the option to turn it into a wheel and place it at a random spot on screen or even link it to a declared shape in a wallpaper I just downloaded that make some pretty funny effects when I pass my cursor above it. Fixing the random cursor bug would also be nice.
I want it to have multiple functions instead of being just a luncher.
In addition I want easy tools to customise it and for my crappy laptop to run it just fine. I want it to provide me with the option to cycle wallpapers stored on my PC or right of the internet. I also want it to provide dynamic themes and the tools to create some of my own.
