### Post by mal on 2006-06-03
I just performed a clean install of Dapper, but kept my old /home partition.  This all seemed to work well, except that the power button has not appeared on the panel in the top right hand corner of the screen.  Also, the "System/Quit" icon is still the old "exit door" icon.

I can add the "Quit" applet to the panel, but this is also the old icon and can't be placed on the right-hand end of the panel next to the clock, like the standard Dapper button.

Any ideas how to get the new power button icons in the right places.

Mal

### Post by mal on 2006-06-03
OK - I think I have figured out the answer to my own question.

It seems that the new install started with "Clearlooks" selected as the theme instead of "Human" which I had selected in Breezy.  This obviously made a few things look different.  When I changed back to "Human" the correct icons appeared.  I still didn't get back the power switch on the panel, but I was able to add it, just not in the correct position.

Mal
