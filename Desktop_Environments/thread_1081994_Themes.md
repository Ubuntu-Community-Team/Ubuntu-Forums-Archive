### Post by .Maleficus. on 2009-02-27
There are a couple of ways.

1.  Take the theme you downloaded (do not extract it, keep it as a .tar.gz) and drag it onto the Theme Manager.

2.  Extract the themes to your .themes directory.  Then, open up the theme manager and apply it.

### Post by stumbleUpon on 2009-02-27
You can also right click on desktop > change desktop background > theme tab, then click on install and point to the theme archive that you downloaded. This will install the theme. Select it to use it.
