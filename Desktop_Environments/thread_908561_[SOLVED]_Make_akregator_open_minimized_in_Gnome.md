### Post by 67GTA on 2008-09-02
I prefer akregator for RSS feeds. I added it to sessions so it would start at boot, but the window opens every time. Is there any way to make it start minimized to the panel at boot like it does in KDE?

---

### Post by 67GTA on 2008-09-02
I answered myself. Running ```
akregator --help
``` in a terminal gave me ```
--hide-mainwindow
```
