If you use Compiz, you have to change cursor using CCSM (Advanced Desktop Effects Settings) [COLOR="Blue"]"General Options"[/COLOR] section, [COLOR="Blue"]"Cursor Theme"[/COLOR] (you can use ALT-F2 and type CCSM)

If you leave blank the line, the cursor should set to default preferences.
