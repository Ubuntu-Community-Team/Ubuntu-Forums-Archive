### Post by Nomad64 on 2005-11-01
The answer is both yes and no...Yes, because the hard drive is being used, therefor causing it to age and eventually die. No, because it doesn't matter what you are doing to use your HDD (installing an OS, copying files, etc.), it will eventually be used to the point where it dies.

This is why server-grade computers have multipal hard drives setup in RAID arrays. So if one HDD fails (which *will* happen) they can just replace it, and the data can be regained.
