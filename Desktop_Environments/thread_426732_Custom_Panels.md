### Post by Belathor on 2007-04-28
Hey!

Do a search in synaptic ( system -> administration -> synaptic ) for murrine. Install the package that shows up (should have a 0.52 in it). Then, go to [here]("http://www.gnome-look.org/index.php?xsortmode=high&page=0"). Download whatever gtk2 theme you like to your desktop. Then open up the gnome theme manager (system -> preferences -> themes) and drag the downloaded file into the theme manager's window OR click "Install Themes" and navigate to the file you want. I hope that helps...
