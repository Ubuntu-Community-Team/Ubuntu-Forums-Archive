### Post by reiki on 2007-02-02
I have an Epson R200. It will print directly onto CDs. There's a template for The Gimp for creating the CD image to print and then in the print manager (when you say... "Print"...) you select printable CD media as the "paper". Pretty easy. You don't need a special application. Everything you need is already there.
