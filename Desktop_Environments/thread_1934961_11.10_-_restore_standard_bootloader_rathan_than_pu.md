### Post by Tiganjero on 2012-03-03
Open up ***/etc/default/grub*** and change
```
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"
```
to
```
GRUB_CMDLINE_LINUX_DEFAULT="text"
```
then run
```
echo 'manual' | sudo tee /etc/init/lightdm.override
sudo update-grub
```
That should do it.

Cheers,
George
