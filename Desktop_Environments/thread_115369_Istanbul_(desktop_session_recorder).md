### Post by Limulus on 2006-01-10
I found the "istanbul" package and its really neat (I get little videos of what I'm doing on my desktop :)), but I can't seem to access the preferences (see the bottom of [http://live.gnome.org/Istanbul](http://live.gnome.org/Istanbul) for a pic).  When I right-click the red dot in the notification area, all I get is a weird miniature box that either has an up arrow or down arrow in it, but clicking on it just makes it disappear.  Also, I can't seem to close it once its open without ending it via System Monitor. Uninstalling and reinstalling in Synaptic makes no difference.

Does anyone reading this have the same problems?

Update: that "weird miniature box that either has an up arrow or down arrow in it" is apparently a squished menu; I was able to access  the preferences by using the mouse and arrow keys to go up and down in it (I could see a tiny line under it which I think were the selections).  So the question is then, does anyone know how to unsquish the menu? ^_-;

Update 2: My Gnome Panel was 24 pixels high; I temporarily made it bigger and then the menu expanded :)  So for those having the same problem, the menu reads: "About", "Preferences" and "Quit".

Final Update: I filed a bug on this :) [https://launchpad.net/distros/ubuntu/+source/istanbul/+bug/6625](https://launchpad.net/distros/ubuntu/+source/istanbul/+bug/6625)
