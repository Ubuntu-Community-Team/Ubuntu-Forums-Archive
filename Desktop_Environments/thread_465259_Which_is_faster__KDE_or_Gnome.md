### Post by aysiu on 2007-06-05
The answer to the original question: KDE is not definitely faster than Gnome, and Gnome is not definitely faster than KDE. For more details, read these threads.
[ KDE is actually faster than GNOME?](http://ubuntuforums.org/showthread.php?t=464132&highlight=faster+kde+gnome)
[ Kubuntu Faster than Ubuntu? ](http://ubuntuforums.org/showthread.php?t=463335&highlight=faster+kde+gnome)
[Which one is faster? GNOME or KDE?](http://ubuntuforums.org/showthread.php?t=447364&highlight=faster+kde+gnome)

This seems to be your *real* question, though: [quote=Entity]If there is some other Display environment that is stylish yet easy for this configuration to handle, please inform me.[/quote] And the answer to that is Xubuntu (which uses Xfce instead of KDE or Gnome)

Yes. ```
sudo aptitude update && sudo aptitude install xubuntu-desktop
``` If you want to remove it later: ```
sudo aptitude remove xubuntu-desktop
```
