### Post by ijsje on 2005-04-16
You can add the "system monitor" applet to the panel, and select network at the preferences. You can add the system monitor by right clicking on the panel menu, select "add a panel" and add the system monitor. It provides a small network graph in the system tray.

Alternatively, you can use ksysguard, it also works with gnome. But it probably won't show a graph at the panel.
