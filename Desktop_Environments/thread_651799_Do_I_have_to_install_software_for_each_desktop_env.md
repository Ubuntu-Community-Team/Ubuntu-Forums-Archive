### Post by dlegend on 2007-12-27
If you want them to run seamlessly then you probably want to install both of them -- however, many applications run fine in both KDE and GNOME but they may have to run extra crap to get them to work if it was made specifically for one DE. 

Just install one first and see how it runs in that environment -- if it doesn't seem to run well then install the  DE-specific application.

### Post by Half-Left on 2007-12-28
KDE apps will work in GNOME and vise-versa but will take up more memory outside their own desktop environment because the libraries need to be loaded(already loaded in their own)

Firewall just has a frontend, iptables loads up on boot.
