### Post by deepwave on 2006-11-13
My guess you picked the wrong keyboard layout.  Try using the 'us' version, that is assuming you a regular English keyboard.  When you change the layout setting then restart your X.  Logout of Gnome or KDE and do a CRTL+ALT+BACKSPACE to kill the X server.  The X server should restart automatically, and use the new layout.
