### Post by Frogs Hair on 2012-02-20
One thing to try is backup  your bookmarks for import later . Open home hidden folders and delete the .mozilla folder . Firefox will create a new one when opened the next time . This removes the profile and any corruption , which, is likely the problem . You will have to install any extensions again .after removing this folder .
