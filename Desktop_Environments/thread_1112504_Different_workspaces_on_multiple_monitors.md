### Post by tutwabee on 2009-04-21
I also want to find a non-tiling window manager that has the capability to treat each monitor as a separate workspace.  I've been using Xmonad, but I'm looking to switching to a non-tiling window manager like due to tiling problems with a couple odd applications I frequently use.

I think Xmonad might be of interest to you if you are willing to try out a manually-configured tiling window manager.  In Xmonad, instead of extending each workspace to stretch onto two monitors each monitor has one workspace visible on it at a time.  Be warned though, transitioning straight to Xmonad is not easy if you have only ever used desktop environments like KDE, Gnome, or Xfce
