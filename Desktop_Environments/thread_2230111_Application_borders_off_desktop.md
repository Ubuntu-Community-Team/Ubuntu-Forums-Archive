### Post by AnotherKevin on 2014-06-29
Well, the silence is deafening!

Here's the fix I figured out myself.  Once again, I'm running 14.04 Kubuntu (v.4). 

In order for this to work, you have to *reposition each window to the place on your desktop you want it to go when you start the application.*

Next click the icon in the upper left corner of the app's border.  Choose *More Actions* >> *Special Window Settings*.  

In the box that follows, tick the boxes for *Position* and *Size*, then, using the drop-down lists, choose *Remember*.

Don't forget that you _must_ have the application where you want it _before_ you make changes to the settings.
