### Post by RAOF on 2005-12-02
When you type in passwords linux will generally **not** display anything - this is (slightly) more secure than showing one star per character of the password.  Just type in the password you set up, and press enter.

Did you select "server" when you installed it, though?  It should be coming up with a graphical login screen.
