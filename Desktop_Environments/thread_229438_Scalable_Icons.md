### Post by Tomosaur on 2006-08-04
It will probably be a little while coming. SVG stands for 'scalable vector graphics', which is significantly different from other, raster based image formats. What you're asking for is a program which can trace an existing image and convert the lines to vectors. Adobe Illustrator has this function, but it's far from perfect, and is Windows only anyway. 

By the way, you can download a plugin for The GIMP which will allow it to open .svg files.

### Post by Irony on 2006-08-04
SVGs are scalable vector graphics rather than bitmaps (pixels).

You can fire up Inkscape import the .png image and go to path and trace bitmap which may trace a good scalable graphic.

You'll have to go to document properties and set the output size to say 48x48.
