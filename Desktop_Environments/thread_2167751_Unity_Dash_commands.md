### Post by Frogs Hair on 2013-08-17
Alt + F2 in dash replaces the run dialog window that appeared in the Gnome 2 desktop. Any number of commands can be run from there . Example: Alt +F2 gksudo nautilus or killall conky . It can be used to start applications also. Grab the Ubuntu manual from my signature for a full list of dash functions.
