### Post by sudodus on 2013-10-25
I suggest that you try to check/test think about a few things that could make a difference:

- Try with a live session booted from a 13.10 CD/DVD/USB drive, and also some older version, if you have it available.

- What about the hardware: Brand name, model, cpu, graphics chip/card, wifi chip/card

- Is some other peripheral equipment connected (that might disturb the keyboard)?

- What about different USB ports?

- Have you install some special software (ppa, deb file ...)?
