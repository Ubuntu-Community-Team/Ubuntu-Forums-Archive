### Post by atoponce on 2005-12-01
Xfce is a very minimalist window manager with its own DM.  It is great for older hardware, and uses very little overhead.  It is completely mouse driven, which means there are very little keyboard shortcuts.  It has a few oddities that take some getting used to if you are coming from Gnome or KDE.  At first,  I thought I liked it, but I didn't.  However, I have been using it a little more and it is growing on me.  It uses the GTK, so Gnome aps and themes work great with xfce.  But it isn't for everyone.  I guess you'll just have to try it.

```
sudo apt-get install xubuntu-desktop
```
