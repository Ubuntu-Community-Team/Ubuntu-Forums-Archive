### Post by Tiede on 2006-02-03
No you aren't missing any icons. The Debian menu is an extra menu you can install that lists all the many apps on the computer in a nice menu-like form. It's like digging through a sorted out /bin! You are probably gonna find more than just the debian menu empty, it just means it hasn't been installed yet. If you'd like to install it, [follow this **HOWTO**](http://ubuntuforums.org/showthread.php?t=80394)
