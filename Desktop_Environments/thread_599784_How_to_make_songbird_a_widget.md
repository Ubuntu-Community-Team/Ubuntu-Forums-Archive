### Post by CJay554 on 2008-05-07
With Ubuntu 8.04 simply go:
System > Preferences > Advanced Desktop Effect Settings

Go under "Desktop"
Find Widget Layer
(if you havent yet, check that in order to enable the Widget Layer)

Now just click on the Widget Layer (the text itself)

From there go into Behavior tab in "Widget Windows" 

click + button on the very right

Leave it as a Window Class
under "Value" press the "grab" button

A cursor will appear with which you can click on any window or program and automatically set it as widget.
Then press +ADD
And that program will automatically run as a widget.
