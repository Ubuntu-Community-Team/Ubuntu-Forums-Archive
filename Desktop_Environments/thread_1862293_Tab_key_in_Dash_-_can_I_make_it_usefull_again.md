### Post by postmanx on 2011-10-16
I just upgraded to Ubuntu 11.10 and was surprised to find that the Tab key had been hijacked in Dash. Hopefully I'm using the right terminology. Dash is what pops up when I press the Meta key (aka the Windows key).

What was
========
After pressing the Meta key, I could type a partial word and see a list of completions. I would type 'term', for example, and see a few matching items. Selecting 'Terminal' was just a Tab key away. It was never necessary to move my hands from the home row on the keyboard. It was productive.

What is
=======
Same situation, but now when I press the tab key, it cycles through four pages of what appear to be separate search categories. So far I've been unable to find any way to select the 'Terminal' entry after typing just a few keys, at least not without moving my hand off the home row and using the arrow keys. This is annoying since I use this feature often. It reminds me of the hateful location bare in the Chrome browser.

Is there some setting that will restore the more productive behavior that was in 11.04?
