### Post by the yawner on 2007-10-24
I say, there's no need for you to do that. Just install CCSM (compiz-config settings manager), run it (Advanced Desktop Effects Settings), and check the actions for the plugin Rotate Cube. You'll find that some options  there can be set to work with screen edges. By default I can rotate the cube by scroll button when my mouse is on the desktop. I've also set the Expo and Scale plugins to initiate on the upper left and upper right corner.
