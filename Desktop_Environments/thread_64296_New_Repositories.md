Some programs don't install menu entries when you install them.  Try running "kappfinder" from the terminal to find KDE programs.  For other programs, you might have to manually add them to the menu yourself.  Synaptic can show you a list of all the installed files for a package.  Usually, one of the programs in the /usr/bin folder is what you want.
