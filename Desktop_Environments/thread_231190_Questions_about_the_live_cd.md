### Post by anthro398 on 2006-08-07
It isn't difficult to create a custom live CD or DVD.  I do it for each new version that comes out.  I usually create a live DVD with all the multimedia codecs, mplayer, mp3 support, and encryption software.  It's easier to create a DVD, in my opinion, and to not remove any software from the orginal iso, since this is where one is most likely to make a mistake resulting in a non-bootable disc.  It takes some patience since I usually have to go through the process a few times to get the iso exactly as I want it, but by following the steps it's quite easy.  

Start with the directions at [https://help.ubuntu.com/community/LiveCDCustomization/6%2e06]("https://help.ubuntu.com/community/LiveCDCustomization/6%2e06") and also see [http://www.atworkonline.it/~bibe/ubuntu/custom-livecd.htm]("http://www.atworkonline.it/~bibe/ubuntu/custom-livecd.htm").  Like I said, I've done this several times and I think it's well worth the time I invested in it.
