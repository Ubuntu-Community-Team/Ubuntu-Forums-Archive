### Post by SimonJW on 2006-07-23
If it hasn't finished downloading the packages yet, then there should be no problem to quit it. If it's started installing the packages tho... well, it could well break :S If you have to, I guess you have to.

You can always try running apt-get dist-upgrade again from within recovery mode. Unless it's a very vital package that's halfway through installation, in which case you most likely will be a little messed up :S
