### Post by fritz_monroe on 2006-10-05
You could set up some key mappings to allow you to change workspaces.  I use 3ddesktop as my pager, and I set up F7 as my key to change workspaces.

If you go to Preferences >> Keyboard Shortcuts.  There's a Switch to Workspace 1 already configured.  Just assign a key combination to it.
