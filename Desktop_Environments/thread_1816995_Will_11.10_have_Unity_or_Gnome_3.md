### Post by MARP1961 on 2011-08-03
It will have the Unity 'bodywork' running over the new Gnome 3 'engine'. It will look pretty much the same as Unity running over Gnome 2 as it is now.

However, because Unity is just a 'shell' you will not have to put up with it! Ubuntu 11.10 will have Gnome's own shell (Gnome 3 Shell) available in the repositories (Synaptic or Software Manager) and if you choose to install it you will have the choice of Unity 3D, Unity 2D or Gnome Shell as your login. I've tried Gnome Shell, I think it blasts Unity away and it can be customized too!

People have been confusing Gnome 3 with Gnome Shell. They are not the same thing. Gnome Shell cannot run on top of the present Gnome 2, that's all.
