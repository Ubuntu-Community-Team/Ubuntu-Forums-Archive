---

### Post by ravisghosh on 2009-12-10
1. Start thunar.
2. Select View>Side Pane>Shortcut.
3. All the shortcuts in the side pane will be visible in Places.
4. To add shortcuts in side pane  and hence shortcuts in Places, just right click on any folder in thunar and Send To> Side Pane.

Edit /home/user/.gtk-bookmarks files to add/remove default folders (e.g., Music, Videos, etc).
