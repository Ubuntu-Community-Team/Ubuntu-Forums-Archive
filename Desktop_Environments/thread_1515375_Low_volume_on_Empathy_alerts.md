### Post by evran on 2010-07-30
Try install Pulse Audio Volume Control (pavucontrol) and run it. Empathy will plays any sound while login in chat, logout or receive messages, and if we are using default sound theme (Ubuntu), it will appears in pavucontrol about 0,5 second, move its volume slider to the right. Try to use other sound theme if you think it is too short to be able to move the volume slider.
