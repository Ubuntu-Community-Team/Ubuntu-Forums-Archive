### Post by Frogs Hair on 2010-10-30
If you use an image  directly from a website (set as desktop background) that image will  only remain until you change it. If you save image as to pictures you can then use it any time by opening the pictures folder, double clicking the image and right click (set as desktop background). Make sure the image matches screen resolution so it fits.
