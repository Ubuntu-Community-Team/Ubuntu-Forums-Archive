### Post by 700 on 2012-06-28
Most useful terminal commands:
lsb_release -a [shows ubuntu version]
find /home/ -name 'foo.txt' [searches foo.txt file in /home directory]
sudo dpkg -i foo [installing foo]
sudo dpkg -r foo [removing foo]
ls -l [file access settings]
sudo chmod a+r foo.txt [sample access change for foo.txt file]
