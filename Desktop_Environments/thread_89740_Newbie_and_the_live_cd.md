### Post by johannes on 2005-11-13
Hi and welcome to Ubuntu. :) I'm sorry to tell you that none of your problems can be solved as long as you use the Live CD, as far as I know. You can't change driver configuration or install any programs using that. If you decide to try and install Ubuntu however, you might want to look at the following pages.

> The live-CD booted up just fine, but would only run in 640*480*60Hz. My graphics card (Voodoo3 3000) can do better... Half of the dialogue boxes do not fit on the screen, so I do not see the "Close" Button.

> Also, I tried to install the Opera browser (because it has the fit-page-to-screen-width feature), but doubleclicking the downloaded .deb file would only result in an error message. It will help you install a whole lot of basic things, including Opera.
