### Post by John.Michael.Kane on 2006-11-27
GTK 1.x are older versions though some users do make use of them

GTK 2.x This is the current version that most themes in ubuntu are based on

Metacity These change the theming of your window manager ie metacity

Compiz is a window manager and a composite manager using OpenGL for rendering. A "window manager"

Beryl is a fork of Compiz

GDM Themes are themes that change the login screen

Splash Screens is the screen you see for a few seconds when you login

XMMS Themes are skins to change the look of your xmms player



To set up beryl look at these,and find the one that is suited for your hardware ie: ati.intel,nvidia
