### Post by munk on 2005-01-23
Hey,

I know what you mean, and I'm not sure if it's a bug or if the interface is plain confusing. Perhaps it depends on what the theme itself provides. Anyway, since you have already clicked 'Install Theme' and selected the correct tarball, try this:

[list]
[*] Click on Computer -> Desktop Preferences -> Theme
[*] Select an existing theme (any one) and click Theme Details
[*] The theme you previously tried to install may appear in one or more of the Controls, Window Border or Icons lists. Experiment. You can combine different selections in these three tabs to create a customised theme.
[*] Once you are happy with the theme appearance, click Close. You have the option of saving the new custom theme under a new name.
[/list]

Hope that helps :)
