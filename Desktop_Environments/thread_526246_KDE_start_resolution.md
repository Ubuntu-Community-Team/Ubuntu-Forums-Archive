### Post by prox2far on 2007-09-02
One of my friends gave me the same advice as Happy_Man, but deleting the .kde folder was not an option, here is what i did.

I used some console magic and deleted all entries with the text "1280" in my .kde folder and initialised a fresh KDE session using kcontrol.

This fixed my problem, but I deleted to many entries, the cost of using one line in console and not caring about what it deleted, and lost a couple of my personal settings including some easily restored mail settings.

### Post by Jaasi on 2007-09-08
Try this:

mv ~/.kde/share/config/displayconfigrc ~/.kde/share/config/displayconfigrc.orig

then restart kde
