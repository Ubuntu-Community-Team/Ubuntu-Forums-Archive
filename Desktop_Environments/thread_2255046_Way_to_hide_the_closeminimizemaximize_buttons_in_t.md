### Post by vasa1 on 2014-12-01
The documentation for your window manager should tell you whether you can do what you want. I use the Openbox window manager and that allows me to display any or all of the following in the order I desire:
```
      N: window icon
      L: window label (AKA title).
      I: iconify
      M: maximize
      C: close
      S: shade (roll up/down)

```
