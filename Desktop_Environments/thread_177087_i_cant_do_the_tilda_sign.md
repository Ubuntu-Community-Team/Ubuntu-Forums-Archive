### Post by onlysolution on 2006-07-02
I had the same problem, but when I removed Japanese keyboard mapping from the list of availible keymaps my tilde came back.  I think this is because on Japanese keyboards the tilde key is used to switch character modes, however I know on other int'l keyboards there are similar modifiers.  If you have one of those enabled, you might need to disable it to type tildes.
