### Post by tacantara on 2009-11-28
> What does that key combination even do in ubuntu?The ctrl+alt+del function brings up the shut down menu, where you can select a complete shut down, restart, suspend, or hibernate.  By not making a selection, the default is to shut down after 60 seconds.  In 9.10 this is a default feature (I believe it is also a default feature in the other earlier releases too).

### Post by rudy.b on 2009-11-28
Look under System > Preferences > Keyboard Shortcuts.  The current definition of Ctrl+Alt+Del is under Desktop > Log out, you should be able to modify the shortcuts there.
