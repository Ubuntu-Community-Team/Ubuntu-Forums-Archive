### Post by irv on 2006-06-12
I have one package "wine" that I don't want to update because it screws up my Quicken. I have taken the wine repository out of the source list, but it still comes up in the auto-update. Do I need to remove something from somewhere else?

---

### Post by aysiu on 2006-06-12
System > Administration > Synaptic Package Manager > Edit > Lock Version

---

### Post by irv on 2006-06-12
Perfect! That did the trick.
