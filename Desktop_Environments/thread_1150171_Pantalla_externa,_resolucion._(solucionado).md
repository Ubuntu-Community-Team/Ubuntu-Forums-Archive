Bueno, tengo una laptop vieja con ubuntu 8 y anda muy bien. Tengo un LCD 32' philips y lo conecte a la laptop por VGA.
Tenía el problema de no poder extender el excritorio y no podia poner la resolucion que realmente necesitaba el lcd, para solucionarlo utilicé la interfaz gráfica de Xorg (que es el controlador de pantalla)

En la consola de comando se ejecuta:
[COLOR="Red"][SIZE="4"]sudo displayconfig-gtk[/SIZE][/COLOR]
[CENTER][IMG]http://www.cesarius.net/wp-content/uploads/displayconfiggtkdriver.jpg[/IMG]
[http://www.cesarius.net/xorgconf-con-interfaz-grafica-en-ubuntu/](http://www.cesarius.net/xorgconf-con-interfaz-grafica-en-ubuntu/)
[IMG]http://www.cesarius.net/wp-content/uploads/displayconfiggtk.jpg[/IMG]
[http://www.cesarius.net/xorgconf-con-interfaz-grafica-en-ubuntu/](http://www.cesarius.net/xorgconf-con-interfaz-grafica-en-ubuntu/)[/CENTER]


Es sumamente sencilla la configuración desde allí. Si no lo tienen lo agregan desde el Synaptic (Sistema >> Administración >> Synaptic).
Lo escribo en este post pq no encontre algo tan directo que explique algo tan sencillo para gente q recien comienza como yo.
