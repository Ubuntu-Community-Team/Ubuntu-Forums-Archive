### Post by Ramses de Norre on 2006-06-19
In that case you can run ```
sudo grub-install /dev/hdc
``` from live disc.

### Post by newdarkness on 2006-06-19
(I am lys, I just happened to forget my original name when the error occoured so I couldn't post with my main account)

```

mount -t reiserfs -o rw /dev/hdc1 /mnt/hdc1
grub-install --root-directory=/mnt/hdc1 /dev/hdc
```

solved the issue.
