### Post by croak77 on 2006-09-27
Install flashplugin-nonfree it's in multiverse. Then open konqueror, Settings -> Configure Konqueror -> Plugins -> Scan for New Plugins button.

For kaffeine install kaffeine-mozilla.

When konqueror blocks a pop-up, it will show a little icon with a red X in the lower right of the window. Right-click on it and you can view the blocked pop-up.
