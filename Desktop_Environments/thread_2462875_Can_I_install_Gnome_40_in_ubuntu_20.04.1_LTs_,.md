### Post by grahammechanical on 2021-05-29
See here for the reason Gnome 40 was not included in Ubuntu 21.04

[https://www.omgubuntu.co.uk/2021/01/why-ubuntu-21-04-wont-include-gnome-40-or-gtk4](https://www.omgubuntu.co.uk/2021/01/why-ubuntu-21-04-wont-include-gnome-40-or-gtk4)

It is possible that Ubuntu 21.10 will start to include some parts of Gnome 40 so that it can be user tested before the release of Ubuntu 22.04 which will be a Long Term Support release.

There is absolutely no chance that any part of Gnome 40 will be backported to Ubuntu 20.04 because 20.04 is an LTS release. Business users choose LTS releases for several reasons. One of which is that the User Interface does not change and they do not need to give employees additional training.

If you really. really want a version of Ubuntu that will have in the future parts of Gnome 40 then install Ubuntu 21.10 development version and update it regularly and experience the fun.

[https://cdimage.ubuntu.com/daily-live/current/](https://cdimage.ubuntu.com/daily-live/current/)
