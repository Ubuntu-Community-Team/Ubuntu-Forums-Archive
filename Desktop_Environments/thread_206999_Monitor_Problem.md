### Post by falcon15500 on 2006-07-01
I had a similar problem and found that if I made each OS use a different refresh rate, I could set the screen size for each and they worked fine.  I only had a problem when they were trying to use same res & same refresh rate.

Obviously my suggestion is only a workaround.  I don't know why the same res / same rate from the same card would produce a different position because of the OS.  I would have thought the card controlled that more than anything - but perhaps its driver related.
