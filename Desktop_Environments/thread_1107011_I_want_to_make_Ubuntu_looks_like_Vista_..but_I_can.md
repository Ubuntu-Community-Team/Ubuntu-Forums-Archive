### Post by blazemore on 2009-04-03
The theme in question is probably an emerald theme.
You'll need to install stuff to make it work:

```
sudo apt-get install emerald fusion-icon
```

You can then load the emerald theme manager with ```
emerald-theme-manager
```, import the .emerald file that way.
