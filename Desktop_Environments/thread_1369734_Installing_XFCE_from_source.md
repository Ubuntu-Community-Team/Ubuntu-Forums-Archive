### Post by northrup on 2010-01-01
You don't need to install Xfce from source to use it.  Xfce is pre-included in the xubuntu-desktop package.  Just put this into the terminal:

```
sudo apt-get install xubuntu-desktop
```

But, if you insist on building from source, you can follow [this guide]("http://www.xfce.org/documentation/4.2/userguide/xfce4-install"), which will guide you through the install process.
