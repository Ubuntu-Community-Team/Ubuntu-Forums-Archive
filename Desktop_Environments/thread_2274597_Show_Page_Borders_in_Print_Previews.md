### Post by mcduck on 2015-04-23
Could it be that you are using a GTK theme that messes with the preview colors, making the paper & background same color? Each and every program I tested Print Preview with wither displayed a clear border, or the preview was limited to the actual page (so the page border is same as the window border)

Edit: Also, check if there are zoom buttons available in the print preview window. If you have set it to match the window width, you of course won't see the borders since the document preview gets scaled to the window. Set it to a fixed zoom size and zooming out will show you the borders.
