### Post by Sef on 2006-05-04
For Linux there aren't any real virus threats.  Most of the viruses that have been designed have been for proof of concept, not for release.

Though if you run an email client, you should have anti-virus softwear.   That's because a microsoft virus won't infect you, but you can give it to your friends who run Microsoft software.

However, you should run a firewall: either hardware or software.
