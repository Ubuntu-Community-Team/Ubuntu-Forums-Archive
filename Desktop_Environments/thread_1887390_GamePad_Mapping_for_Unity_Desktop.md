### Post by neu5eeCh on 2011-12-21
A solution. Found [here]("http://ubuntuforums.org/showthread.php?t=1885379&highlight=gamepad+unity").

To quote the original author, Unity doesn't like QJoypad, it has to be started via the following command:

**qjoypad --notray**

The qjoypad icon will probably show up in the middle of the screen but it's usable.
