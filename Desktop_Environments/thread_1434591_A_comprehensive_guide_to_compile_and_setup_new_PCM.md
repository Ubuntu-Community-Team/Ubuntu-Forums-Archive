Since the lightweight file manager PCManFM has underwent a total rewrite, much has been changed. To try the new version out, you can installed packages from Lubuntu project. Or, if you want to test the latest source code, here is a complete guide to compile and setup pcmanfm/libfm which can solve most of the FAQs.

[http://wiki.lxde.org/en/index.php?title=PCManFM_build_and_setup_guide](http://wiki.lxde.org/en/index.php?title=PCManFM_build_and_setup_guide)

Please give it a try. If you don't like compiling source code, you may try Lubuntu 10.04 Beta1, which is the LXDE flavor for Ubuntu.
