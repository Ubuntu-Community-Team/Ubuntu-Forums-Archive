### Post by morgengenuss on 2009-09-07
it's quite simple: xfwm4 is the window-decorator/manager. so thats the bar that includes the buttons hide, maximize and close on top of every window. if you install a theme for xfwm4 you can change the looks of that bar.

if you install a gtk-theme, this changes the rest of the window and even the looks of the panel.

since you can use different window-decorators/managers there's no unified place for changing both window-manager theme and gtk theme (at least that's one possible explanation).
