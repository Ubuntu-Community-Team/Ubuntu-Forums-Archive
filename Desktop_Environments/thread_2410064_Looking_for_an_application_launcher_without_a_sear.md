Guide for how to get and install extensions:
[https://linuxconfig.org/how-to-install-gnome-shell-extensions-on-ubuntu-18-04-bionic-beaver-linux](https://linuxconfig.org/how-to-install-gnome-shell-extensions-on-ubuntu-18-04-bionic-beaver-linux)

Note there is a second method, installing through your web browser from the gnome extensions webpage, that many users (like myself) prefer.
