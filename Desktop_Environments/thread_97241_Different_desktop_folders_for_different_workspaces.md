when I'm working with something that makes me produce or save a lot of files, especially images, I like to save various intermediate files on the desktop for easy retrieval and delete or move them when I'm done - so my desktop may end up looking like this:

[[IMG]http://ubuntuforums.org/gallery/files/2/0/0/9/7/Screenshot-med-billeder_thumb.png[/IMG]]("http://ubuntuforums.org/gallery/showimage.php?i=1411&original=1&c=2")

As it is now, if I drop a file on the desktop  in one workspace it will show up in all of them, and I'd prefer them to behave like separate desktops. 

Is this in any way possible, or is this actually something similar to a "requirement for new functionality"?
