### Post by Svictor on 2005-09-23
Well, after a bit of try-and-miss, I finally got it working\\:D/. The install was setting my keyboard to "fr latin9" ; in gnome it appeared as "French alternative". For a somehow obscure reason, this layout does not let itself being switched to another one. Everything went back to normal when I chose plain "French" (simply "fr"). I also unticked the box overrideSettings in configuration editor --> desktop --> gnome --> peropherals --> keyboard --> kbd (don't know if this is important or not).
