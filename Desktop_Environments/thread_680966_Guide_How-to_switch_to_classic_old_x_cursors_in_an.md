now i will tell you how to switch to classic x in any gui (gnome/xfce/kde/other) in ubuntu linux. ok so lets start :):
first ensure that u have your cursor set to default
then open terminal, type sudo nautilus /usr/share/icons/DMZ-White/cursors (in gnome nautilus, in kde konqueror or dolphin, in xfce thunar)
then type ur password and press enter
then mark all icon files (CTRL+A) and delete or move them. WARNING: if u remove these cursor files u will cant get back to ubuntu default cursors, so better backup if u some time would want to back to ubuntu default cursors.
now restart x server. have fun :KS. now u will never see flashing cursor :mad:.
