### Post by bluevoodoo1 on 2006-02-24
Open up System > Preferences > Theme 
Drag the tar file into the open window. (You don't have to extract anything, or drag any folders). Just drag the original tar file you downloaded into the theme dialog window. And that is all.

### Post by kpkeerthi on 2008-02-13
System -> Preference -> Appearance it is.

To change icons, you need to get icon themes from gnome-look usually packaged as tar.gz. And there are GTK2 themes in that site that change the way your controls look. You do not have to unpack the archives.

Anyways open System -> Preference -> Appearance. Click Install and select the archive you just downloaded. Select the GTK theme (if you installed a GTK theme) that appears in the dialog. To apply the newly installed icon theme, click Customize -> Icons tab and select the Icon theme name.

[Step by Step]("http://www.howtoforge.com/mac4lin_make_linux_look_like_a_mac") to apply mac like theme to Ubuntu.
