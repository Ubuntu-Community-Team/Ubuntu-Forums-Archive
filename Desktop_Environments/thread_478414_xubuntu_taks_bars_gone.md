### Post by ugm6hr on 2007-06-19
Is your Desktop still there?

If so:
Double-click home on Desktop)
Left-click File System in left column of File Manager
Go to usr/bin/xfce4-panel (double-click in turn)

Panels should re-appear.  Best to tick the box "Save session for future login" when you click on the Exit icon to turn off / log out.
