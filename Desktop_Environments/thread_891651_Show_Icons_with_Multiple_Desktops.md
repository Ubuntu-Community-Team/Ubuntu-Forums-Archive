### Post by the yawner on 2008-08-16
If you're using Compiz to draw the wallpapers then it's not possible at the moment. As pointed out on the posted link:

> If you simply enable the wallpaper plugin and set up your images, nothing will change. This is because GNOME&#8217;s *Nautilus is already drawing your wallpaper along with your desktop icons*. Unfortunately, you&#8217;ll have to give up desktop icons to use this plugin because Nautilus can&#8217;t draw icons onto a transparent background. This is planned to be fixed in the next version of GNOME and Ubuntu.
