### Post by nickoli_02 on 2007-02-21
I was right, the problem was with fglrx, not beryl. Since with version 7.1 of xorg you need to disable composite in order to use the fglrx driver, this turns off direct rendering, which beryl needs. I gave up on getting beryl set up for awhile and just tried it again after downgrading to dapper. Beryl works great now (so far :) )

So, the problem was with xorg 7.1 and fglrx. If you're having the same problem you should be able to fix it by downgrading xorg to 7.0, though don't take my word for it and don't ask me how :)
