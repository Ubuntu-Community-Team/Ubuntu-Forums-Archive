### Post by 3Miro on 2010-02-16
In my experience it is better if you were to enable and disable effects from the compiz-config-settings menu. Go to Synaptic and install compiz-config-settings, then from System -> Prefs -> compiz-config-settings you will have an enormous amount of effects to set up and customize including shortcuts, number of desktops and so on.
