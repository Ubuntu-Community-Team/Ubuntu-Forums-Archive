Hi to all.

Why DownThemAll! opens the containing folder with Gwenview which is an image viewer?

I had the same problem with the Firefox native download manager and I solved it by following those:

[http://forum.mandriva.com/en/viewtopic.php?t=119723]("http://forum.mandriva.com/en/viewtopic.php?t=119723")

[https://bbs.archlinux.org/viewtopic.php?id=72741]("https://bbs.archlinux.org/viewtopic.php?id=72741")

The problem with DownThemAll! is solved following the second of those, in detail this post:
[https://bbs.archlinux.org/viewtopic.php?pid=561615#p561615]("https://bbs.archlinux.org/viewtopic.php?pid=561615#p561615")
So, after opening the System Settings -> File association ... I opened the "inode" section and removed the Gwenview entry.

Hope it helps someone else.
