### Post by DirtDawg on 2006-12-23
I try to stick with GTK apps on my Xubuntu box to keeps things speedy. For example galculator is a great GTK calculator, Sylpheed-claws-gtk2 is a speedy mail client (the decision to use Thunderbird by default is an odd one, IMO), and XMMS is a super lightweight mp3 player in GTK. All of these are in the repos.

The [Xubuntu Blog]("http://xubuntu.wordpress.com/") is unfortunately dead now, but the author posted lots of neat-o tweaks before he stopped and worth a look.

I personally think Xubuntu is a great lightweight desktop but it's up to you to keep it that way.

### Post by maxamillion on 2007-01-03
When running Xfce, stick with GTK2 (non-gnome lib requiring) applications. The main reason many users say they don't see an extreme advance in performance from Gnome to Xfce is because they continue to use the same bulky applications as they did under Gnome and therefore your ram will be consumed by the same libraries and the only thing actually performing better is just the Desktop Environment.

Office: AbiWord+Gnumeric instead of OpenOffice.org (some people really need open office because it handles heavy customization of microsoft formatted files better than these counter parts, but for those of us not bound by a microsoft world, these alternatives show good speed increase)

Audio: XMMS (I know this was already mentioned, just had to re-iterate) also, Streamtuner if you want net radio

Web: Firefox is generally ok, but swiftfox will provide a little more speed

E-Mail: Sylpheed GTK2 is awesome
