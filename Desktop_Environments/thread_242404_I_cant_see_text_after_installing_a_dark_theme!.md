### Post by PHuN on 2007-08-11

To fix what you are talking about when it comes to gmail


> 
Well, I guess that worked! But I still cant read text when I'm checking my Gmail. Is that refered to as Imbedded?

goto edit > preferences > content; click colors and deselect the option box that says: "Allow pages to choose their own colors, instead of my selection above"

That will then force firefox to use your custom selection.

That worked for me quite well.:)

### Post by nalmeth on 2007-11-10
FYI, I had this problem using a dark KDE theme, fonts on certain websites wouldn't show up.

I went into edit -> preferences -> colors
and unchecked 'use system fonts'. 

Fonts look fine with the ubuntustudio theme in GNOME, but this fixed the color problem using Firefox in dark KDE.
