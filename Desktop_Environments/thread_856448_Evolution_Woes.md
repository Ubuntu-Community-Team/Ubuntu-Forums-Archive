### Post by kubug on 2008-08-18
To answer my own question: By chance (and yes I googled it before) I came by a comment made by someone else that had the other problem - Evolution would scroll DOWN on him all the time. 

I figured it had to be the same problem and sure enough it was.

Select "View" and check if "Caret Mode" is on, if so turn it off.

That would then fix the scrolling issue.
