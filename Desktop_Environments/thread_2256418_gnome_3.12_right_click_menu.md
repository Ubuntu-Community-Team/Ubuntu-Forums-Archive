### Post by Frogs Hair on 2014-12-12
Hello and Welcome

Install the Gnome Tweak Tool if not already and navigate to the desktop tab to enable the desktop icon setting. This allows the file manager to handle the desktop and should give you a working context menu. It is not necessary to  check the boxes for desktop icons unless you want them to appear on the desktop.
