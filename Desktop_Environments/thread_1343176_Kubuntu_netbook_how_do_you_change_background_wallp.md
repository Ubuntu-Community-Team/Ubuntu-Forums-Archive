### Post by riste on 2009-12-04
You can change the wallpaper from the Settings menu for the active widget, which you should be able to get to from the Plasma icon near the top-right next to "Applications" or whichever widget is currently active.  The different widgets can have different backgrounds too.

It's different from normal KDE because it doesn't use the standard desktop and program launcher.

To set up filesharing (Windows SMB file shares), you need to install the samba and kdenetwork-filesharing packages from kpackagekit.  Then you can set up sharing for a folder using the Sharing tab on the Properties page for that folder.
