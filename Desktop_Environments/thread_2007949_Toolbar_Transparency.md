### Post by zombifier25 on 2012-06-21
The setting could be found in ccsm or Ubuntu Tweak. For ccsm, Unity Plugin > Experimental > Panel Opacity. For Ubuntu Tweak, Tweaks > Unity > Panel Opacity.

EDIT: Kinda missed the part that you said about your graphic card. Transparency is only possible for compositing windows manager, such as Compiz.

### Post by Frogs Hair on 2012-06-21
If you are using Unity 2D transparency isn't possible. You need 3D compatible graphics. If you have not run the Unity 3D test use the following command.```
 /usr/lib/nux/unity_support_test -p
```
