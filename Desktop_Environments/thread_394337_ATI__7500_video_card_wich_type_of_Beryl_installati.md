You might want to give Fedora Core 6 a shot.  I got it working on my T30, the ATI 7500 card was enabled from the start and installing beryl is a simple yum command.  I'm using KDE and it's:

yum install beryl-kde

And that's it.  Direct rendering was enabled from the start and beryl works fine.
