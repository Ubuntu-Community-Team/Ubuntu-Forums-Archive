### Post by mcmuffy on 2005-10-20
How do I convert a mail box from Kmail to Evolution?

I am sorry if this is a well asked question but I could only find advice for converting TO not FROM Kmail.

I gather that Kmail uses a format that Evolution can understand but i'll be buggered if I can find the directory it stores its files in.

---

### Post by claydoh on 2005-10-20
Kmail stores the mail in /home/<your-username>/.kde/share/applications/kmail

Note that the directory is "dot"kde, meaning it is a hidden directory, so you will need to enable that in your filebrowser.
