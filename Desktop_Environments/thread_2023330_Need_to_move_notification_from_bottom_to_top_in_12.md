### Post by Krytarik on 2012-07-12
Seems like you are talking about the notification bubbles specifically (not indicators and the like on the panel) - for non-Gnome Shell sessions, please see this guide:[INDENT][http://www.tuxgarage.com/2011/06/tweaking-notify-osd.html](http://www.tuxgarage.com/2011/06/tweaking-notify-osd.html)
[/INDENT]For Gnome Shell, you can use this extension, for the notification bubbles:[INDENT][https://extensions.gnome.org/extension/243/shell-osd/](https://extensions.gnome.org/extension/243/shell-osd/)
[/INDENT]And if you are instead meaning the notification area in the bottom panel of Gnome Shell, you can use this extension to move it to the top panel:[INDENT][https://extensions.gnome.org/extension/204/shell-notification-move/](https://extensions.gnome.org/extension/204/shell-notification-move/)
[/INDENT]Regards.
