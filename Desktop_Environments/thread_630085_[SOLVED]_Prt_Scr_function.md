### Post by ItsManaged on 2007-12-03
Are you expecting print screen to capture the dropped down list in the captured image?
It won't do that because screenshot is another 'button press" action which will null the last action (drop down list).

Use "Take Screenshot" Accessories -> Take Screenshot and set a delay, say a few seconds, then click on the download list and wait for the screen shot to activate.

Another approach is to capture your screen using "gtk-recordMyDesktop", then use a video editing tool to select the shot you want.

Hope this helps.
