### Post by benerivo on 2009-01-25
You can edit the gnome menu to remove the kde stuff you don't want, by right clicking on the menu icon and selecting 'edit'. If you installed kubuntu by installing the kubuntu-desktop package, then i think  all the kde packages should be marked as 'auto removable' in synaptic, once you uninstall it. If that doesn't work, you can remove the qt packages, which kde relies upon, and that should clear out kde from your system.
