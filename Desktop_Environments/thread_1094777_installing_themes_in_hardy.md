### Post by mcduck on 2009-03-13
Emerald only runs .emerald themes. If your theme is in some kind of archive extract it first and then install the .emerald theme inside.

If your theme is not in .emerald format you are using wrong type of theme and either can't use it all all, or need to use some other program to install & configure the theme.
