### Post by ciscosurfer on 2006-07-14
Flash doesn't work "out of the box" with any browser on a Linux system...there are however some workarounds.  Some of these require editing some files, and other produce audio out-of-sync errors.  Follow [this thread]("http://ubuntuforums.org/showthread.php?t=187207&highlight=flash+problems") to determine what's best for you.  The workaround that I prefer is to prepend "aoss" before launching firefox like so:```

aoss firefox
```It may work for you, it may not.  You be the judge. :)

### Post by aysiu on 2006-07-14
It sounds as if you need Flash 8, which they don't make for Linux (only Flash 7 or in the future Flash 9): > In order to enjoy the media-rich nature of the Tenth Dimension website, you will need to upgrade to the latest version of Macromedia Flash. You can download the latest Flash plugin by clicking here. Below is the site content, made available for reference and ease of indexing. Note: OS 9 does not support the most recent flash plugin. As a result, OS 9 users will not be able to view the media-rich elements of this site. The only way I know of to do this is to install with Wine the Windows version of Firefox and then install Flash 8 from there.
