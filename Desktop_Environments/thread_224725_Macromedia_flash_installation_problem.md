### Post by yuv on 2006-07-28
IMHO Automatix or EasyUbuntu are not so good in time (update & dependencies), so you will have to make a new reinstall in 2-3 weeks.
The best (I also used it) is to install it manually (just read and follow the instructions on the screen): download and expand it in /home/your_user and run in a new terminal:

cd inst*
./flashplayer-installer

(the rest is very easy,..)
