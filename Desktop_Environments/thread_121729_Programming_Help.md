### Post by BigMurph on 2006-01-25
Hmm... you don't really need a piece of software to be able to program. All you need to do is write a program in an empty file in whichever language you choose, double check the syntax, and compile using the appropriate compiler (eg. gcc for C++ code). A good way to see this in action is to download a simple program's source code and read through it. Keep in mind I've never actually programmed in anything but Microsoft Visual C++ 6.0 so I may just be talking out of my ass...
