### Post by cap10p on 2022-03-29
I want to use qt5ct and Kvantum, for that I need to set the QT_QPA_PLATFORMTHEME  environment variable to "qt5ct".
But, I am unable to change it. I have tried setting ```
export QT_QPA_PLATFORMTHEME="qt5ct"
``` in ~/.profile and /etc/profile.
I have also tried setting ```
QT_QPA_PLATFORMTHEME="qt5ct"
``` in /etc/environment.
Yet it's still set to "gtk2" for some reason. It's probably being overwritten by something else.
I tried setting the variable in ~/.bashrc, then when I ran qt5ct from Bash, it worked. But it didn't work if I ran qt5ct from the applications menu. So, this is not an ideal solution.
Any help or advice will be appreciated.

---

### Post by &amp;KyT$0P# on 2022-03-29
It's set in [FONT=Courier New]/etc/X11/Xsession.d/56xubuntu-session[/FONT] .  You need to comment out that line in that file to be able to set [FONT=Courier New]QT_QPA_PLATFORMTHEME[/FONT] yourself.
