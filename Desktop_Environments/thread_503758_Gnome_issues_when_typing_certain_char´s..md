### Post by mcduck on 2007-07-19
That is because those symbols are supposed to be combined with a letter. For example pressing "¨"and then "a" gives you "ä". Pressing any letter or key that cannot be combined with the symbol will print the symbol itself.

If you want to turn this on go to your keyboard settings and select a keyboard layout with "No Dead Keys" (or whatever the exact phrase in the dialog is).
