### Post by Xian on 2005-10-29
The following is your problem:

```
Section "Monitor"
Identifier "Generic Monitor"
Option "DPMS"
EndSection
```

You need to include the HorizSync and VertRefresh rates.
Also, if would be helpful if you also included the [monitor size](http://www.advogato.org/person/robertc/diary.html?start=35).

Here is an example.
Do not copy/paste this info. 
Use only your specifications.

```
Section "Monitor"
	Identifier	"COMPAQ 7500"
	Option		"DPMS"
        HorizSync       30-70
        VertRefresh     50-140
        DisplaySize     320 240

EndSection
```
