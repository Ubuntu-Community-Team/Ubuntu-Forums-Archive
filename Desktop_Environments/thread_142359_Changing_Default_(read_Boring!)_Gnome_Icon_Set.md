### Post by Klejs on 2006-03-11
There is an really easy way to change the iconset for the whole system. Download an set of icons from gnome-look.org or some other place and make
sure its in **tar.bz2** format.

Then fire up the thememanager and click to custumize the actual theme, select icons and just drag and drop the tar.bz2 files you've downloaded

### Post by stalefries on 2006-03-11
Some icon sets will be .tar.gz, and some will be .tar.bz2. It's not always bz2. 

To change the icon theme, go to System>>Preferences>>Theme. Then click Theme Details, click the icons tab, and them drag-and-drop the icon theme file to the icons list. Then make sure your theme is selected. It should automatically change all the icons.
