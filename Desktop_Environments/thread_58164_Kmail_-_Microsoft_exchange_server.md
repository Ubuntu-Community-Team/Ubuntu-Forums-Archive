### Post by SanderAnt on 2005-08-20
Basically you need to ask the system administrators if they have IMAP or POP/SMTP support enabled in Exchange.

If they do then get the name of the server and user name you have to use for login and then set up a Kmail account using those protocols.

Neither of the solutions will be as well supported as Outlook, but you'll be able to send email.

Another solution would be the Outlook Web Interface in Exchange, which works under Konqueror.
