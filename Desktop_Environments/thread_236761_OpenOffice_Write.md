### Post by adriantry on 2006-08-15

> Could you tell me how I can intall a language module (for example Dutch) in OppenOffice, please?

Hi Linux-beginner

Sure!

From OpenOffice.org Writer, select
File / Wizards / Install New Dictionaries

You will be shown a list of languages you are able to install. I can't see Dutch there, but I don't know how to say "Dutch" in Dutch!

Hope this helps. There may be another way of installing languages not listed in the Wizard (like the command line options already given).
