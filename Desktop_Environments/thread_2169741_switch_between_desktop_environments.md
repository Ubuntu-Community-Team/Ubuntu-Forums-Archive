### Post by sudodus on 2013-08-23
You will be able to select 'session' alias desktop environment at the log in screen. It looks a bit different depending on which log in screen (greeter) that you are using. In some it is easy to find where to do it. In the Ubuntu (Unity?) log in screen, you click on the round Ubuntu symbol according to the attached file.

### Post by ibjsb4 on 2013-08-23
There are also two methods of installing extra desktops.  Some will suggest installing a complete desktop.  example:

Xubuntu
```
sudo apt-get install xubuntu-desktop
```

For Xubuntu I prefer to:
```
sudo apt-get install xfce4 xfce4-goodies
```

Same for Lubuntu
```
sudo apt-get install lubuntu-desktop
```

vs

```
sudo apt-get install lxde
```
