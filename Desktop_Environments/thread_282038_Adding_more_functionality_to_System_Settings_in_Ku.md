### Post by zellis on 2006-10-22
A while back I was reading some help files saying that it was possible to configure Autostart via  System Settings  (aka the Control Centre) in KDE. I was confused because my kubuntu install didn't have any option listed there.

Turns out it needed to be installed: the package name is kcontrol-autostart.

I've found another package which also adds some functionality to Systems Settings: the kdmtheme package installs a module under System Settings->System Administration that (unsurprisingly given its name) lets you conveniently install and configure themes for the KDM login manager.

Are there others?
