### Post by steinerd on 2011-07-09
Found some information on LXDE.  If this makes sense then please give it a try I am currently loading Lbuntu in a VM and will probably have much better advice once that is done.

"Open pcmanfm the file manager and go to Edit --> Preferences to the tab Desktop and make sure the Manage the desktop and show file icons checkbox is checked this will give you your desktop icons back. And if you would rather have the menu back you had make sure Show menus provided by WM when desktop is clicked is unchecked."

This advice came from here [http://forum.lxde.org/viewtopic.php?f=8&t=280](http://forum.lxde.org/viewtopic.php?f=8&t=280) and is a little old but I think the principle should still apply.  As far as the GUI is concerned the file manager should have the setting for the desktop view.
