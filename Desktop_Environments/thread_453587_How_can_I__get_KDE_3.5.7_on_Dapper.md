### Post by pbw on 2007-05-24
If you're not comfortable compiling kde on your own, just use konstruct. It's user friendly and easy. - [http://kde-apps.org/content/show.php/Konstruct?content=9725](http://kde-apps.org/content/show.php/Konstruct?content=9725)

> 
Stable KDE Version

 This version installs the current stable development releases of KDE and applications. Currently it gives a complete KDE 3.5.7 installation. Optionally you can install additional applications like KOffice 1.6.2 or KDevelop 3.4.1.
