I get the same error regarding the jockey, and also the "no seat-id" warning from GDM, so you don't have to care. And if you don't have autologin enabled or another reason for a custom.conf of GDM, that warning is also fine. Note that those are warnings, not errors. In GDM's case, it searches for that file and if it isn't there, the default settings are applied.
