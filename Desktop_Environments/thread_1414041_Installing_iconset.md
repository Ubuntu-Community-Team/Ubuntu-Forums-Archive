### Post by Tibuda on 2010-02-23
Forget the terminal.

1. Install the [p7zip]("apt:p7zip") required to open 7z files.
2. Open the download file (if you are using standard Ubuntu, it will open with FileRoller).
3. Extract it in the /home/yourusername/.icons folder (this folder is hidden, but you can display it pressing Ctrl+H)
4. ????
5. PROFIT
