### Post by mrmcctt on 2005-11-28
For the location bar, go to MAIN MENU>SYSTEM TOOLS>CONFIGURATION EDITOR>APPS>NAUTILUS>PREFERENCES and check always_use_location_entry.  If I remember correctly, that should add the location bar.
