### Post by rax_m on 2007-06-12
Hi 

I think you're looking for something like Virtualbox ([http://www.virtualbox.org/](http://www.virtualbox.org/))
or vmware.. this will allow you to run virtual operating systems within another. If you're going to run a few VMs at the same time you'll need a decent spec computer tho'. 

Regarding the s/w question, I believe ( and I may be wrong ) that anything you install from the Ubuntu standard repositories you should be pretty safe. I believe that things are looked over. However if you install software from other sources then there is a possibility that a bug could be introduced.. 
Don't think it's happenned to me yet tho ;)
