### Post by helino on 2008-06-03
The easiest way to start is to download a theme from Gnome-Look, and then extract it and look at the content. You will find one xml file which describes where everything is placed on the screen, and one other file which describes  the author of the theme, the name of the screenshot file and other stuff.

Good luck making your own theme!
