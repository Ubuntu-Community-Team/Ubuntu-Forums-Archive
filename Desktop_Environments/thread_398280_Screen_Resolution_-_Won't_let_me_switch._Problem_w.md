### Post by taurus on 2007-03-31
You can try to remove the high resolutions from /etc/X11/xorg.conf.  

For Gnome:
```
gksudo gedit /etc/X11/xorg.conf
```

For KDE:
```
kdesu kate /etc/X11/xorg.conf
```
Then, either reboot or restart X with <Ctrl><Alt>Backspace.
