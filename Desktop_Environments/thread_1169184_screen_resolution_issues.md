### Post by coffeecat on 2009-05-25
System > Preferences > Appearance > Fonts tab. Adjust the font sizes (and rendering while you're about it) to what is comfortable for you. 

For desktop icons, open your home folder (or any Nautilus window) and.. Edit > Preferences > Views tab and change Icon view default zoom level to whichever is comfortable for you.

I'm posting from a laptop with the same resolution as yours. I've set icons to 66% and font sizes to 9. The fonts are still bigger than with Windows, but that's the way I prefer it because then I can actually read things. If you want your fonts to look more like Windows, then set a smaller font size and get yourself a magnifying glass. :wink:
