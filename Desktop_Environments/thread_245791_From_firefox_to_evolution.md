### Post by iTrendsNET on 2006-08-29
This is not that difficult to do and I did it again myself recently.  First, make sure you know the hidden directory where Thunderbird keeps your "mbox" file files.  You' most likely only need your inbox, sent and possibly draft files out of the many files you may find in that hidden directory.  It is OK to go ahead and remove the "." before the directory name to make it unhidden to ease the importing of the mail files into Evolution.

Install Evolution if it is not already installed.  My preference would be to use aptitude, but you can also use apt from the command line or synaptic.  Set up your mail account(s).  Then, from Evolution, just go to File, then Import, then point to the directory where you have your Thunderbird mail.  Depending on how large your mail files are, Evolution may "think" for a while before it moves to the next step.  Don't start thinking your computer locked up on you!  Finally, just make sure you are importing inbox to inbox, sent to sent, etc.  

The above should be a breeze.  If you run into trouble, post back or send me a PM and I will walk you through it again.
