My problem was that file-roller does not show you the file permissions of the files inside the zip file (or file owner for that matter). I needed to see this so I installed about 5 different GUI archive tools (xarchiver, karchiver, ark, and one other k-one that I forget). The only one that does it is karchiver. Unfortunately, karchiver cannot open a file with the standard extension.

My problem is solved, just posting this message so the next person who comes along can find the solution.
