### Post by mlmurray on 2005-10-20
While not as light as fluxbox or blackbox, XFCE is much less resource intensive than either Gnome or KDE, but is almost as featureful as those two.  You can easily install it with ```
sudo apt-get install xubuntu-desktop
```.

Another one to consider is Icewm.  I'd put it between flux/blackbox and XFCE in terms of 'lightness'.

Good luck!
