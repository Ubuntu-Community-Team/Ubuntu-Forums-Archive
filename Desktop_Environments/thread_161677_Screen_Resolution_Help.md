
If you see the generic "nv" driver you should install the "nvidia" one.

### Post by Jason_25 on 2006-04-18
If you have a DVI connection with driver 81.78 or earlier, adding these lines to the monitor section should force the driver to your specified resolution.
```

Option		"NoDDC"		"true"
Option		"IgnoreEDID"	"true"

```
If it doesn't work, post your Xorg.0.log file.
