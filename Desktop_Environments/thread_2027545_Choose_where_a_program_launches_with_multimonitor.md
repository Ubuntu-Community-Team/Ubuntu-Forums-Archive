### Post by beejee on 2012-07-17
Yay, found a way to steer my programs!

For anyone else searching for this, after a lot of digging I found the settings I was looking for in the CompizConfig Settings Manager:

In the section "Window Management" choose the item "Place Windows",
in here you can select the default placement mode for all applications and specify for every application its position, its viewport or what it's placement mode is.
