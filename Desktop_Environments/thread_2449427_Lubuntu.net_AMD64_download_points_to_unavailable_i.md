### Post by guiverc on 2020-09-10
@maicolinux  The site you mention is an *official* FAN site.

If you want to go to official sites affiliated with Ubuntu, search ubuntu.com for *official* sites, you'll end up at [https://ubuntu.com/download/flavours](https://ubuntu.com/download/flavours)  which **will** send you to the Official sites of official Ubuntu *flavors*.  You will find no mention of the *fan* site on any official Ubuntu site, or wiki page.

The .net site you mention is not owned by Canonical/Ubuntu, nor Lubuntu. We cannot control what is on it, or the other *fan* site(s) that exist.  Yes the owner does link to a ubuntu.com web site for download (*to avoid bandwidth costs** for most if not all downloads*), but if you want to ensure you get an official and *untainted* image you should use official websites for everything.
