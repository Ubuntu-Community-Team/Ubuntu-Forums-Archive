### Post by swerner on 2005-10-29
install qtparted, this will allow you to format and partition a disk drive.  You can install it using synaptic, or the commandline:
```

apt-get install qtparted
```

It will be accessable via Applications>System Tools>QTParted

Enjoy ;)
