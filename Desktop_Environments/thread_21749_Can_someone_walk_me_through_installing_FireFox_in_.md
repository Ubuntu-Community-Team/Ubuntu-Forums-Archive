### Post by AhYup on 2005-03-23
Its in the apt-get repository. 

So you open up konsole. If you have not used apt-get yet first type:

sudo apt-get update   and then enter your password when asked. This will go to the online repositories and refresh the list. 

When that's done type: sudo apt-get install mozilla-firefox  and it should download and install.
