About a week ago I started getting a small (at the time time) problem.
Everything would be fine but at some point I would try to launch an app and it would appear on the taskbar and there would be a startup notification but the app never started.  After this, any app I tried to launch, whether from the panel, menu or terminal would do the same thing and the message in the terminal was "GTK Warning **: Unable to open display".
The only thing I could do was to try logging out and when I did this I only got the option to log out, instead of the usual log out, shutdown, reboot options.
So I clicked ok and the screen would go black for some time then I would eventually get a pop up message that the gdm chooser was crashing and it was going to try another one but nothing happened and I had to reset the comp.
Anyway, as I said, at the time it was a small problem since it happened maybe once a day.
Now it has gradually been happening more and more and for the last 2 days it has happened every time I've booted into ubuntu. Sometimes within 5 minutes, sometimes up to half an hour but every time.
It seems my ubuntu is slowly dying.
I can't think of anything that may have caused this.  Up until the last week everything has been running well and I haven't installed anything or changed any config files for a long time.
I have gnome, kde, xfce, e17 all installed and using gdm as the display manager if that is any help but I haven't used any other environment other than gnome for several weeks.

[EDIT] Solved by re-install.
