My sister runs Ubuntu with the Unity DE.  I run Fedora with Xfce.  Now, she wants to switch to Xfce.  Are there any "gotchas" I have to watch out for if I just install Xfce, log out and back in, selecting Xfce?  I don't want to have to install Xubuntu over what she has because there's enough customization and so on that I don't want to mess with.

### Post by 3Miro on 2012-07-22
Just install xfce4 and you will have xfce4. This will come with just default settings and it will not affect the rest of Ubuntu. If you want other plugins or applets, you can install them additionally.
