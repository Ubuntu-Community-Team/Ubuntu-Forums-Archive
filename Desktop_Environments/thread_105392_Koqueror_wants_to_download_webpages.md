### Post by Zotova on 2005-12-18
I installed KDE 3.5 yesterday and everything was working ok.

However today when I decided to try out Konqueror it just will not browse any webpages. It was working fine yesterday when I tried it for a few moments however today whenever I type in a webpage address I get a pop-up dialog asking where I wish to save the html file or the php file etc

Anyone have any idea what I might have done to make konqueror do this? Plus a way to get things working again as I'd really like to try konqueror out and see if it is up to scratch with Firefox (so far it isn't).

---

### Post by aysiu on 2005-12-18
Go to Konqueror > Settings > Configure Konqueror > File Associations > Known Types > Text > HTML > Application Preference Order and bump Konqueror up to the top.

---

### Post by Zotova on 2005-12-18
Thanks a lot, that fixed it.
