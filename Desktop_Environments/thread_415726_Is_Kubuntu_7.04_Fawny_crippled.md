I'm not using Kubuntu (I really like Gnome), but if the MP3 player you're using is based on [GStreamer](http://gstreamer.freedesktop.org/) you can try installing the **gstreamer-plugins-ugly** (and optionally **gstreamer-plugins-bad**).
