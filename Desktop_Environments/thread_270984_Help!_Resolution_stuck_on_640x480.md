### Post by akadruid on 2006-10-04
looking at your xorg.conf I am guessing that Ubuntu wasn't able to detect your monitor specs - you need to find these out and fill them in.

There is a detailed explanation on that page you found
([https://help.ubuntu.com/community/FixVideoResolutionHowto](https://help.ubuntu.com/community/FixVideoResolutionHowto))
but basically if you work out the exact name of your monitor (it
probably says on the back) and use google you should be able to find out the horizontal sync frequency and the vertical refresh rate.

There is a model number 'LCM-17v8' in the xorg.conf which google
thinks is a 'Westinghouse - 17" Flat-Panel TFT-LCD' - does that sound right?  If it does, then you have a Vertical Frequency 56-76Hz and Horizontal Frequency 30-82kHz.

So if you back up your existing config with this command:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```
and then edit it with this:
```
sudo gedit /etc/X11/xorg.conf
```
and find this section
```
Section "Monitor"
Identifier "LCM-17v8"
Option "DPMS"
EndSection
```
and add in two lines for the refresh rate:
```
Section "Monitor"
   Identifier "LCM-17v8"
   Option "DPMS"
   HorizSync 30-96
   VertRefresh 50-160
EndSection
```
