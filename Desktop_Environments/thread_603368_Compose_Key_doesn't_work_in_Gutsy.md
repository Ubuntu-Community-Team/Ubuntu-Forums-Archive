### Post by mac.ryan on 2007-11-05
The compose key (System -> Preferences -> Keyboard -> Layout Options) is the key you can use to use "combos" of keys in order to type characters that are not native to your keyboard layout. For example:

ComposeKey + 1 + 2 gives ½
ComposeKey + o + / gives ø

etc...

Hope this clarify the issue! :)
