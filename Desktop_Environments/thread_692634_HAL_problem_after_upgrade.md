### Post by Bubba64 on 2008-02-10
If you can access synaptic through system-administration click on filters then broken packages and then reload anything broken. If it wont reload with standard repositories you can find legitimate repositories via Google. When I upgraded from Feisty to Gutsy my computer showed some errors in the download and shutdown in the install. The computer opened correctly with a restart and I found Gnome desktop was broken and would',t install with regular repositories so I went to the university I attend Linux mirror and found the desktop and installed without a problem.
