### Post by mcduck on 2010-03-10
No, the server version of Ubuntu only installs a command-line setup. Of course you can add some desktop yourself, but if you plan to use the machine for desktop tasks and not just run a server then you might want to install the desktop version instead and then just install the servers you want. That way you'll have a working desktop straight from the install (and adding the server apps is really easy anyway).
