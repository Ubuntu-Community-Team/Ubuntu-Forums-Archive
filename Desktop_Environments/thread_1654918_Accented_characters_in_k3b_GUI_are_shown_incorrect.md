### Post by claracc on 2010-12-29
I have a problem with GUI frontend of k3b: The accented characters (spanish language) are shown bad, ie Más or acción or menú, see attached image file. Desktop is in spanish obviously.

My desktop is gnome, but i have some kde apps as k3b or ktorrent. The gui frontend of ktorrent shows correctly the accented á, ó,í,ú,é characters but no k3b.

¿Is there any workaround?. Thanks in advance.

---

### Post by claracc on 2010-12-29
I have found the solution. As it is indicated in this thread [http://ubuntu-cosillas.blogspot.com/2010/08/solucionar-problemas-idioma-espanol-en.html](http://ubuntu-cosillas.blogspot.com/2010/08/solucionar-problemas-idioma-espanol-en.html) (in spanish), you have to download software package k3b-i18n_2.0.0-1_all.deb (which is not in the ubuntu repositories,...why?) and to install it. Then, accented spanish characters are shown correctly in k3b.

Argentina LoCo Team has a very interesting thread discussing and solving too this problem: [http://ubuntuforums.org/showthread.php?p=10291006#post10291006](http://ubuntuforums.org/showthread.php?p=10291006#post10291006)
