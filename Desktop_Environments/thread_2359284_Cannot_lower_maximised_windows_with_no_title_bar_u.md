### Post by again? on 2017-05-09
If you really wanted this functionality with the mouse in Gnome you could use easystroke.
Set a keyboard shortcut for lower.
[ATTACH=CONFIG]275029[/ATTACH]

Then in easystroke set a gesture or a long press of middle button to run the shortcut for lower.
[ATTACH=CONFIG]275030[/ATTACH]
You can perform the gesture anywhere in the window, maximized or not.
