### Post by bailout on 2005-03-28
[QUOTE=swatch123]hi,moopere:
     
Filemanager is very important role on kde but I could not find it, why?[/QUOTE]

I have seen several posts on this so it is obviously causing confusion. Konqueror is the kde file manager as well as the browser. The tool tip unfortunately just says Konq-web browser. If you open Konq and go Settings you will see an entry "load view profiles", hover over that and a submenu will appear which allows you to choose preset configurations for the different tasks,, although you can browse files in web view and the internet in file man view.

How I set it up is to go win/show side bar and then settings/save profile. The side bar shows buttons which link to bookmarks/home/root file system etc and having this on all the time allows you to use konq for multiple purposes very easily.

I think the description of Konq out of the box needs to be changed to Web Browser and Filw Manager to avoid this confusion.
