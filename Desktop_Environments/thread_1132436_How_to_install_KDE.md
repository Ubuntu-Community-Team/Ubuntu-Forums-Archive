### Post by kpkeerthi on 2009-04-22
If you want to add everything that comes with kubuntu on top of you current desktop environment, run this command
```
sudo apt-get install kubuntu-desktop
```

If you want to remove GNOME, 
```

sudo apt-get remove ubuntu-desktop
```
