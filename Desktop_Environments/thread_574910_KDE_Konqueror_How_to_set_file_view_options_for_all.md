### Post by Whatawonderfulday on 2007-10-13
Dear (K)ubuntufolk:

On a fresh install of Kubuntu over formatted partitions, when I set the file view option in Konqueror to 'detailed list', it takes until I click on a folder, at which time the view reverts to 'icon view'.  How do I get Konqueror to show detailed list view always?

Thanks.

Whatawonderfulday

---

### Post by lucidapparition on 2007-10-14
Goto Settings > Save View Changes Per Folder
Make sure that it is NOT checked, Then switch to the view option you would like and it should stay in the same view mode regardless of the folder you enter.

### Post by Whatawonderfulday on 2007-10-14
There does seem to be a solution:

If you uncheck Konqueror > Settings > Configure Konqueror > Open Files in Separate Windows, then it opens the folder in the same window but maintains the detailed list (or, I imagine, whatever your preferred setting is).

This seems to solve the immediate problem.
