### Post by hictio on 2008-11-13
Are you running Compiz?
If you do, open CompizConfig Settings Manager, and goto to the Window Management category, and enable the plugin called "Place Windows".
Open that plugin, and you might want to set the 'Placement Mode' to 'Smart' from the pull down menu.

Another thing, you can move a window by pressing the left mouse button anywhere on the window, and also pressing the Alt key.
(Unless you have changed those settings to use other keyboard/ mouse button combo.)
