### Post by astoltz on 2005-10-03
There have been reports with similar symptoms when using the nVidia drivers with the "RenderAccel" option is turned on.  See if ```
Option "RenderAccel" "true"
``` is in your xorg.conf file and if so, set it to false.  
Does anyone know if this is fixed with the latest nVidia driver in Breezy?
