### Post by dqsis on 2006-09-24
Hello there, 

I am using Ubuntu (and not K-ubuntu). I had a similar problem. For me the solution was:

System > Preferences > Keyboard > Layout Options > Group Shift/Lock behavior 

Then I could set the button-combination I wanted in order to change language keyboard. 

Hope that helps! :| :-s
