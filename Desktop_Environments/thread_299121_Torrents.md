### Post by anodizer on 2006-11-13
You probably need to forward a port to your local IP for your web interface to work and no, don't put the same port you're using for Azureus transfers.

PS: Of course this solution applies assumed you have a router and it doesn't matter that the 2 computers are on the same LAN, you still need port forwarding. Then, you'll just type 127.0.0.1:xxxx (xxxx = the port you chose) from a browser running in the other computer and you'll get the web interface.
