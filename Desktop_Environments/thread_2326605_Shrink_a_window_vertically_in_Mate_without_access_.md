### Post by mcduck on 2016-06-03
Try Alt-middle mouse button (or possibly Super+middle mouse button) which are common shortcuts for resizing window from any point inside the window rather than from the edges. I'd be surprised if Mate doesn't support either one.

In case the window can't be resized, Alt(or Super)+left mouse button should allow you to move it, so you could access the rest of the window that way. Pretty helpful shrtcut for dealing with low-resolution displays, for example,a s you often run into settings dialogues and other things that have minimum size that's still too large for your display ;)
