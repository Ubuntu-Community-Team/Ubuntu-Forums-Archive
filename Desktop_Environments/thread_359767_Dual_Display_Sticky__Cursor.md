However, on the secondary display it keeps the shape it had when it enters the display. If it was a hand, it stays as a hand, if it was an arrow, it remains and arrow no matter what I hover over.  Cursor actions still work (clicks, tooltips, highlighting etc) but the indicator wont change.

This was not a problem until I upgraded to edgy last month and it is most annoying. A co-worker has the same machine/graphics card and xorg conf  running Ubuntu with Gnome and does not have this problem. This leads me to believe that it might be a KDE problem. 

My graphics card is an ATI Radeon X300/X550  (RV370 5B60) .  The system is fully updated. I would appreciate any help on this, I can't be the first person to have this problem and it is likely something silly I have done. Here is the xorg.conf

I was able to get it working by removing this line:

then moving my monitors around. At least the cursor is woking, but now I have to get used to this set up. I would say that this is definately a bug.

My co-worker did not have this setup. When he added that line to his xorg.conf, he had the same cursor problem.
