### Post by maitreya on 2006-02-01
[QUOTE=syklitengutt]ok... As far as I can see:
in xfce you can change the look of the windows by selecting different
Styles in window manager. Ok great, it comes with about 60 styles if you install some packages.

And you can change the themes by selecting another theme in user interface.
There I have some themes I installed to ubuntu.
But how do I install themes when Im in xfce?[/QUOTE]

Download a gtk-2.0 theme, for e.g.

[http://oceanic.wsisiz.edu.pl/~slabosz/pliki/Alien-0.3.tar.gz](http://oceanic.wsisiz.edu.pl/~slabosz/pliki/Alien-0.3.tar.gz)

Extract with path to ~/.themes directory or extract and move it there. You can see the theme name (same as the directory name) in the user interface settings in xfce.

You can find more themes at:
[http://www.gnome-look.org/](http://www.gnome-look.org/) or
[http://www.xfce-look.org/](http://www.xfce-look.org/)
