### Post by taygan on 2005-04-03
Yeah, gcursor (cursor selection) only works in gnome.  You have to install the cursor theme and then select it from the terminal (update-alternatives) for it to work in xfce.  You want the Jimmac cursor theme.

This is how I did it a month ago.  I don't know if things have changed since then.
