### Post by An Sanct on 2011-04-22
Welcome to the forums!

Modify the appearance, in the system menu, but since you cannot get to it, here is the command line
```
gnome-appearance-properties %F
```The "%F" is there, because I just copied the exact command, that runs from the menu, maybe it will work without it ... give it a try, if "%F" doesn't work.

PS. Word of advice ... In compiz, find the option to save the current settings, when you are happy with the results ... it will serve as a "restore point" if anything goes bad and will take up only a few kilobytes of space. Making a backup is important (specially with compiz ...).
