### Post by Buffalo Soldier on 2005-03-21
[QUOTE=Taoism]In the Windows world I know there was Stardock that let you make some pretty wild desktops, and widgets, I assume Gnome would have similar levels of customization available?[/QUOTE]

There is a certain level of customization in GNOME, but I hope you won't be too dissapointed because it's not a simple mouse click away. You'll be expected to be comfortable running a few commands in console (terminal or command prompt) and editing text configuration file.

More so GNOME is not much about wild widgets, bling bling and the fancy decoration. It's more about making thing simple, minimal and clean according to guidelines set in GNOME [HIG]("http://developer.gnome.org/projects/gup/hig/") (Human Interface Guideline).

But if you'll keep an open mind I assure it will be a wonderful journey of discovery. And if GNOME is not your thing, I hope you will not totaly turn of Ubuntu or GNU/Linux. There are many other alternative Desktop Environment you can try out such as [KDE]("http://www.kde.org/"), [XFCE]("http://www.xfce.org/") and many others.

Welcome to Ubuntu :)
