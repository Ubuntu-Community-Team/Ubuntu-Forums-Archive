If you are using Compiz, the transparency is "true". If you are not using the compositing manager, then "transparency" will show the desktop, rather than the window underneath the active window. See [this screenshot]("http://ubuntuforums.org/attachment.php?attachmentid=93711&d=1227378448") for an example of "true" transparency.
