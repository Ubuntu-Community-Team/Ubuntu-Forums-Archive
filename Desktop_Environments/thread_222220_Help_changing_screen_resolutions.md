### Post by catlett on 2006-07-24
Did you install the nvidia drivers? They will get you better resolution.
[http://doc.gwos.org/index.php/Latest_Nvidia_Dapper](http://doc.gwos.org/index.php/Latest_Nvidia_Dapper)
To reconfigure your xorg file, you can run this command ```
sudo dpkg-reconfigure xserver-xorg
``` That will run you through your monitor and video card specifications. The more info about your hardware you put in, the better your options will be later.
But install the drivers first if you haven't already.
