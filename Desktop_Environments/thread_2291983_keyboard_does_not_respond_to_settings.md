### Post by HenkHarmsen on 2015-08-24
After unplugging the external keyboard from my laptop it does not want to go back to its original keyboard settings anymore. 

I need to press Fn for most keys at the right side of the laptop keyboard, so I think the keyboard is still mapped as 105 keys.
/Settings/Keyboard/Layout = "Use system defaults".
System = Ubuntu 15.04. XCFE 4, AT Translated Set 2 keyboard.

When I start the laptop with a live USB everything works fine, and the settings are as above [system defaults]. 
Does anybody know in which direction I the solution can be found?

---

### Post by HenkHarmsen on 2015-08-24
found it: go to settings/keyboard/behavior and uncheck : Restore num lock state on startup. nice gotcha.
