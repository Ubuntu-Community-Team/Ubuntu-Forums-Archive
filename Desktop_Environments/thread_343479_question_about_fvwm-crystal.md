### Post by K.Mandla on 2007-01-21
If you install opera and restart FVWM-Crystal, it might add it to the launch bars automatically. It should work; Opera is fairly common and I think I remember seeing that one in /usr/share/fvwm/fvwm-crystal/Applications (is that it?).

If not, you need to set up a custom menu item and FVWM-Crystal will try to add it to the menus as well. Take a look at this:

[http://www.mail-archive.com/fvwm-crystal-users@gna.org/msg00004.html](http://www.mail-archive.com/fvwm-crystal-users@gna.org/msg00004.html)
