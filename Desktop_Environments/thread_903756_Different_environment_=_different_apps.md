### Post by snowpine on 2008-08-28
Hi Frito, this page will show you the main applications that come with Xubuntu (Ubuntu+Xfce): [http://xubuntu.org/tour](http://xubuntu.org/tour)

You can install any of the applications from Ubuntu or Kubuntu if you're using Xubuntu. They might not be as fast as the "native" Xubuntu apps (because they require Gnome or KDE dependencies) but they should work just fine. 

Mainly the difference between the 3 distros is the windows manager. The "core" is the same.

Hope that helps, let me know if you have any specific questions.
