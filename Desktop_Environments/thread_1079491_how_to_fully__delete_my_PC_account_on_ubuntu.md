### Post by marco123 on 2009-02-24
If you want to delete the old users /home directory, "/marco" or whatever you will need root privileges, so press Alt+F2 and run "gksudo nautilus" enter your password and then you can delete the directory.

Be VERY careful what you delete when running Nautilus as root!
