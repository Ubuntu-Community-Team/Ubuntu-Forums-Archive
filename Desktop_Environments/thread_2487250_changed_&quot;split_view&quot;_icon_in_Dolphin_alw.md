### Post by Tom_ZeCat on 2023-05-28
As many of you know, you can use custom icons in Dolphin.  The process is: 
1. Right click on Dolphin's toolbar
2. Choose "configure toolbars."  
3. Under right list of icons (named "current actions"), scroll down and choose the icon you want to change.  Example: the "create folder" icon.  
4. There is then a button named "Change Icon" that you click on.  
5. There is then a browse button you can use to surf to the folder where you've stored your image file.  
This worked great for the "Create Folder" icon.  Instead of the plain, black default icon, I wanted one with color so that it's easier to find.  I replaced the default with a blue icon of a folder with a green plus sign on it.  
That one worked great!  However, I'm not having the same luck with the "split" icon.  I successfully changed it to a colored icon that is easy to find and looks great.  Then, when I click on this Split icon, it of course changes to the "Close" icon, which I would expect.  However, when I click on this "Close" icon, the default split icon returns.  My custom one is gone.  The same thing happens if I have not split dolphin, but I simply close it and re-open it.  My custom Split icon always vanishes and is replaced with the default.  

Does anyone know a way around this?
