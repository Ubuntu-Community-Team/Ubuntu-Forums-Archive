### Post by mgmiller on 2007-03-09
a) pretty much true
b) I don't think so
c) You won't get the drop shadows or transparent window borders without Beryl, but you can get a glassy look.

At the moment, I am getting a nice 3-d glassy blue button effect by using the Glossy P control set from the glossy P theme I got from gnome-look.org.  ([http://gnomelook.org/]("http://gnomelook.org/")) In addition, I use the Alpha cube color borders for my window borders and dropline Neu! for my icon set.  My cursors are the silver x cursors.  There are also some very glassy looking icon sets called Glossy-Glass, but I don't find them as nice as Dropline Neu!  I also ran across some very glassy looking cursor sets, but I don't recall their names. I spent, "wasted" LOL a lot of time messing with my gui the other night and I think for a non-beryl setup, it looks really nice.  For a really pretty GDM logon screen, try Relaxing-water.  All this stuff is on gnome-look, you just have to spend some time looking around there and then playing mix and match in System>Preferences>Theme  till you get something you like..

Have fun.
