### Post by John Bean on 2010-01-10
You could install wallpaper-tray from the repositories, it may do enough for your purposes.

I ended up writing a script to do what I needed, using gconftool to set the new wallpaper on each change. FYI the gconftool usage looks like this:
```
gconftool-2 --type string --set /desktop/gnome/background/picture_filename someimagefile.jpg
```where you should of course replace someimagefile.jpg with the path and name of the actual image you want as the current wallpaper.
