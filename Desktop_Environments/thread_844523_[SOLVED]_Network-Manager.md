### Post by untermensch on 2008-06-29
Is your clock and username gone too? If so right click on the toolbar, then click add item, then scroll down to find "notification area" and put it back on there. you can go to applications -> internet -> pidgin, click and hold on it, then drag it up to the bar. (that will put pidgin on there =p) and if You haven't deleted everything else, just you're network manager, open a terminal and type in sudo nm-applet. and it should appear again.
