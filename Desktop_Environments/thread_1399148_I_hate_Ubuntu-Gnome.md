### Post by jken146 on 2010-02-05
Launch gnome-appearance-properties (System->Preferences->Appearance and use the options in the Themes tab to change the icon set to 'Gnome' and tweak the rest of the settings until you get what you like. IIRC GNOME uses the Clearlooks theme by default.

I *think* that if you started out with a minimal install and then installed the package gnome rather than ubuntu-desktop you'd get the default GNOME desktop, but this would be a lot more faff than just trying to recreate the settings yourself.
