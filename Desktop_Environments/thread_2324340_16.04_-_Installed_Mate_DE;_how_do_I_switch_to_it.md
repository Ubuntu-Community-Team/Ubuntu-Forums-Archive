### Post by grahammechanical on 2016-05-13
When we install an alternate desktop in Ubuntu + Unity we get a Ubuntu icon at the top right of the panel where we put our user password. Clicking that should present a drop down menu with the alternate desktop sessions. It works from me on 16.04 with Gnome Flashback.

The chosen session becomes the default and its icon replaces the Ubuntu icon as a reminder of which session is the default. These icons are not present if Unity is the only session.

Regards.
