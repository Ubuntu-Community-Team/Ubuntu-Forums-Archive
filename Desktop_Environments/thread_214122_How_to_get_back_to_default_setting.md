### Post by AZzKikR on 2006-07-12
Generally, all settings in Gnome are stored in the home directory of each user. So my guess is to delete the directories starting with a period in your ~ home folder. 

**NOTE:** do NOT immediately start deleting the directories until someone here validates this! 

You can create a new user from within Gnome (Administration IIRC). This will create a new directory in the /home/ folder with  no preferences stored yet. You can also create a new user using the _adduser _command (or _useradd_).
