### Post by GoldBuggie on 2005-08-31
this error msg
> in the prefix, you've chosen, are no KDE headers installed. This will fail.

This means that you need the header files for some kde things installed. These are installable thrue 

sudo apt-get install kdebase-dev

Depending on which version of libidn11 you have installed you will get this to work. Since having the latest version is the worst thing. Having version 0.5.13-1.0 installed will not make you get the chance of installing libidn11-dev and doing a force version seems to remove some apps that uses 0.5.13-1.0.
