### Post by garvinrick4 on 2011-06-21
Compiz is installed and configured for Unity by default in Natty.
Have to install CCSM so you can get to settings. Gnome Classic
runs very well also if you choose to run that. Unity gets rather quick
once you learn to use it. Like any other Operating System is a learning curve to go through.
```
sudo apt-get install compizconfig-settings-manager
```
