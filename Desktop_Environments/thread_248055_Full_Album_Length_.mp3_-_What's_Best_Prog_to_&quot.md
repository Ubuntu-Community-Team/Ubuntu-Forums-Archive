### Post by aysiu on 2006-08-31
Try *mp3splt*--it's in the Universe repositories.

If you need to add extra repositories, try this:
[http://www.psychocats.net/ubuntu/sources](http://www.psychocats.net/ubuntu/sources)

Once you have them added, you have a wealth of new software at your fingertips. You can install *mp3splt* with a single command you paste in the terminal: ```
sudo aptitude update && sudo aptitude install mp3splt
```
