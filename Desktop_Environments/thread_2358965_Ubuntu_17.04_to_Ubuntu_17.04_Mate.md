### Post by RobGoss on 2017-04-19
Hello and welcome to the forum, Ubuntu 17.04 and Ubuntu 17.04 Mate, are two different desktop environments. You can install the different destop environments if you just wanted to experience what the difference is between the two

It sounds like you want to do a full installation of Ubuntu mate 17.04, if this is correct your options are to ether do a clean installation meaning wiping your existing hard drive and installing Ubuntu mate 17.04, or creating another partition so you can install Ubuntu mate on that new partition. This means you will have to dual boot between each system to use each OS

As far as Kodi is concern I'm not sure how you can backup that data and transfer it to the new installation, maybe some else can give you some help with that
