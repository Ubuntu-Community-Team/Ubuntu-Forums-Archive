### Post by tseliot on 2006-05-21
In Ubuntu (GNOME) if you right click on a file you have the "Send to..." function.

Something similar exists in KDE.


If your sendto is not installed (in GNOME) try this:
```
sudo apt-get install nautilus-sendto
```
