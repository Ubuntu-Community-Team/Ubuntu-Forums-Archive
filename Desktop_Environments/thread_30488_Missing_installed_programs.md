### Post by jodef on 2005-04-28
Not all programs when thay are installed create menu items you can try running them from the console or terminal. For gnome-chess try typing **gnome-chess** in a console window. For gnuchess I believe you need to install xboard as well though at least if you want graphical interface and then you type **xboard** in console. I am not 100% sure of the commands though haven't used either program in a while.

As far as editing the menus in gnome well you can try this app : [Gnome Menu Editor](http://myosc.org/ubuntuguide/menueditor_0.4.3ubuntu1_all.deb)
