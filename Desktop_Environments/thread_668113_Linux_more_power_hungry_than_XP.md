### Post by mbsullivan on 2008-01-22
The premise of this thread is definitely not bogus... as of Feisty / Gutsy, out-of-the-box Ubuntu does consume more power than XP (and probably Vista).  This is a problem in the implementation that should be addressed.  However, it is not an inherent deficiency of the platform.  As of kernel 2.6.21, Linux is by default tickless (no fixed clock), meaning that the CPU can idle in a low-power mode for long periods of time.  However, it can only do this if not interrupted by some needy program.  As such, there are optimizations (to running programs, etc) you can make to radically improve power usage.  These changes, although useful,  may not readily obvious (especially to desktop users).  

I know that these were mentioned previously, but I'd like to reiterate that the following will make your install much more power efficient:

(1) Enabling speedstep, if it's not already working, will minimize power usage.

(2) Discovering what daemons / programs are consuming the most power, and optimizing your install with Powertop.  It is available in the new repositories.  Some tips on reducing power consumption can be found [here (http://ww.linuxpowertop.org/faq.php)]("http://ww.linuxpowertop.org/faq.php").  It is important to note that any graphically or computationally demanding program will adversely affect power consumption, such that it may be worth your while to use a lightweight window manager (such as Fluxbox) and minimize the number of background services and daemons running on your machine.

(3) Laptop mode will park your hard-drive for most of the time, greatly reducing power consumption.  The disks will be spun up periodically (about once every 15 minutes) to be written to.  Pros: minimizes power usage, energy consumption.  Cons: may reduce hard-disk life (if computer is continually running) and some minor data loss is possible (due to fact that files aren't truly written to secondary memory until the drive is spun up).

If you are concerned about power consumption, I suggest you treat your machine as a laptop.  Most people are much more concerned about power efficiency with their laptops, since greater power consumption leads to reduced battery life.

As for the "greenest" CPU / PC, it's impossible to tell.  However, I'd say that my attempt at an ultra low-power machine might include a [pico-itx motherboard with a fanless VIA CPU]("http://www.mini-itx.com/store/?c=41") and a [solid-state disk drive (SDD)]("http://www.tomshardware.com/2007/08/13/flash_based_hard_drives_cometh/page11.html#energy_requirements").  It definitely would be not support gaming, however.
