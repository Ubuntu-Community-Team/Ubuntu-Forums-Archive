### Post by mcduck on 2014-03-21
PCManFM has a built-in support for managing your desktop (just like Nautilus has), so if you want to keep on using Nautilus for the desktop and PCManFM for separate browser windows, you need to go to PCManFM's preferences and make sure it's set to not manage the desktop.
