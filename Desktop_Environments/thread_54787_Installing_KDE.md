### Post by Mishura on 2005-08-06
To install KDE, do a

```
sudo apt-get update
sudo apt-get install kubuntu-desktop
``` 

To remove Gnome.. thats, a bit more complicated, as I am not a Gnome user.

### Post by Matchless on 2005-08-06
It will do the settings automatically and then give you a choice of logging in to Gnome or KDE session on the bottom icon of the login window.Go here for details [http://mrbass.org/linux/ubuntu](http://mrbass.org/linux/ubuntu) I suggest downloading Kubuntu, burning the CD and doing a clean Kubuntu install. If you ever want to fall back to Gnome, just install the Gnome desktop from your Ubuntu CD and you can switch between the two again without rebooting. I personally prefer KDE.
