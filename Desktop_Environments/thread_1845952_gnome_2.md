### Post by MARP1961 on 2011-09-18
My advice is to install Lucid Lynx 10.04 which is fully supported until April 2013 (an eternity in the computing world). For the next year and a half you can continue to enjoy a fully functional and fully updated Gnome 2.

Perhaps in that time, you will be able to investigate an eventual replacement. There are a couple of possible candidates:

**Xubuntu**: Is based on a simpler desktop experience but quite similar in many ways to Gnome 2.

**Gnome 3 Fallback**: Can be accessed if you install Gnome Shell in the upcoming Oneiric Ocelot 11.10. This will give you the top and bottom panel back and the chance to 'tweak' by installing extensions to change behaviour and appearance.

You have 18 months to plan and cheer yourself up a little!

### Post by cariboo on 2011-09-18
I think what the op is talking about is gnome-session-fallback, which is the fallback mode if your system can't run gnome-shell, the gnome-fallback-session is similar to the old gnome 2 panel interface, although it is all new code.

@jimrew111, what you want to do is impossible with 11.10, Oneiric is based on GTK-3, so the libraries for gnome 2 (GTK-2) are no longer used. I'd suggest you do a search for gnome-session-fallback to see if that is what you want.

[This]("http://www.webupd8.org/2011/08/installing-using-classic-gnome-desktop.html") may be what you're looking for.
