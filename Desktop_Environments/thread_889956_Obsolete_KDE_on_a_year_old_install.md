### Post by carolinason on 2008-08-14
KDE release cycles are dictated by KDE not Kubuntu. Kubuntu releases two versions a year, but has an LTS version every year and a half. KDE releases updates when they see fit. You are only a point release version from the latest 3.5.9 release.

Upgrade your Kubuntu desktop to version 8.04 for KDE 3.5.9
[http://www.ubuntu.com/getubuntu/upgrading](http://www.ubuntu.com/getubuntu/upgrading)

KDE is up to version 4.1, but it is incomplete.
[https://lists.ubuntu.com/archives/kubuntu-devel/2007-December/002066.html](https://lists.ubuntu.com/archives/kubuntu-devel/2007-December/002066.html)

If you're feeling up to it, you can install 4.1.
[http://www.kubuntu.org/news/kde-4.1](http://www.kubuntu.org/news/kde-4.1)
