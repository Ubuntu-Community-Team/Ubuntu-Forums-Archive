### Post by ajgreeny on 2018-06-24
That BIOS killing bug was squashed in the point version 17.10.1 which was rushed out very quickly; the intermediate Ubuntu versions, ie not LTS versions, do not normally get a point update but for 17.10 it was absolutely essential.

There is no similar problem with 18.04, so you can put that worry aside and go ahead.

You will be best to clean install as it will be necessary to double upgrade, ie 17.04 to 17.10 then 17.10 to 18.04; not worth doing in my opinion, but I have never upgraded but always clean installed.
