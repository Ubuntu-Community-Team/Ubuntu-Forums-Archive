### Post by MyR on 2007-04-28
the look of the panel is controlled by ubuntu's theme in system > preferences > theme.
i recommend picking a theme you like that allows you to customize the colors (click customize then the colors tab to see if it lets you) and you can modify it from there.
save the new theme so you can experiment later.
you can right click on the panel then click properties and change the background color or make it transparent.
i also modified my emerald theme somewhat.

with some patience, skill, and luck you can make it look pretty awesome.

### Post by SishGupta on 2007-04-29
The taskbars are themed by GTK2 not Emerald. 

In gnome there are two types of themes. 

There is one type for your window manager, this themes the border around each window, the title bar, and the buttons for the window. 
Emerald is a window manager, so an emerald theme only affects these things.
The default window manager for gnome is Metacity, so were you to not be using beryl/emerald  you would want to download Metacity themes.

The second type of theme is for controls, or GTK controls. GTK controls are buttons like okay/cancel/apply and objects like taskbars. These use GTK2 themes.

So  you can either find a GTK2 theme that looks like your emerald theme (as per the post above), or download emerald themes that come with GTK2 themes.
