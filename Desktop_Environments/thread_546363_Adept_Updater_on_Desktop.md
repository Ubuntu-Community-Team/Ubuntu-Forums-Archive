### Post by Happy_Man on 2007-09-08
Just right-click, say Create New --> Applications Launcher, and enter the command as ```
kdesu adept_updater
```

### Post by Happy_Man on 2007-09-09
Well, if you want the default KDE icon for it, you can just right-click the icon, go to Properties, click the blank icon space, and select the adept_updater icon from there. The advantage of doing it that way is that if you ever change the icon theme, that icon will change along with it. If you want another icon, I would try looking around at [http://kde-look.org](http://kde-look.org).
