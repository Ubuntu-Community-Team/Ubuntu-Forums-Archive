### Post by martrn on 2009-10-19
Can you right click on the panel.  Eg for the top panel right-click on the panel and then select the item to add to the panel.  It should be really intuitive.  You can add the items to the panels at different places, or put them back in the same place you took them from.  Right-click on the panel and click help for help about using the panels.

This presumes you are using gnome.

Hope that helps.
