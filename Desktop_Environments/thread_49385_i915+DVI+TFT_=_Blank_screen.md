### Post by damonw5 on 2005-07-16
Have you tried Feanor's suggestion?

To make it simpler:
1) Type ALT+CTRL+F1
2) Type "sudo nano /etc/X11/xorg.conf" (no quotes, capitalization matters!)
3) Change these lines:
```
Section "Device"
 Identifier "Intel Corporation i915 Integrated Graphics Controller"
 Driver  "vesa"
 BusID  "PCI:0:2:0"
EndSection
```
To this:
```
Section "Device"
 Identifier	"Intel Corporation i915 Integrated Graphics Controller"
 Driver	"i810"
 BusID	"PCI:0:2:0"
 VideoRam 64000
EndSection
```
4) Press CTRL+X to exit. When it asks if you want to save, type "Y".
5) Reboot (you can type "sudo reboot" or hit CTRL+ALT+DEL)
