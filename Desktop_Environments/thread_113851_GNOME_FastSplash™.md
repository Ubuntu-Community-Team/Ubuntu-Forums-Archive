### Post by Haegin on 2006-01-07
Surely the point of the splash screen is to show where the computer is in loading up the various things it needs when it starts. If your pc is fast it will load each program quickly so the icons will appear very quickly then the loading will finish and the splash will disappear. I guess you could find what it is loading and write a bash script which makes the icon appear, loads the program, waits 5 seconds, moves onto the next icon but I don't know how you would do this and I can't see that much point. If you are trying to get on your pc quickly then waiting a self-imposed 25 seconds for something which could be done in 2 seconds would be really annoying.
