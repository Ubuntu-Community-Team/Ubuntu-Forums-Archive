### Post by FIONEX on 2007-11-11
I don't know of any programs that do edit the menu man, but there's a file that contains the XFCE4 menu and it's found here.
/etc/xdg/xfce4/desktop/menu.xml

It's basic XML style and you can edit it as root by typing:
sudo mousepad /etc/xdg/xfce4/desktop/menu.xml
