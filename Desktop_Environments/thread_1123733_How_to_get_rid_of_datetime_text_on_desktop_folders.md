### Post by RuleMaker on 2009-04-13
Go to any directory, in window menu go: Edit -> Preferences -> Display and you should see 3 drop down menus.
Make sure None is selected in the first one.

EDIT:
If you don't want to see anything except the icon name under it no matter of the zoom, select None in all 3 drop-downs.

### Post by TenPlus1 on 2009-04-13
Open a window, click EDIT menu and select PREFERENCES...

Use the DISPLAY and LIST COLUMNS to turn on/off what you want to see...
