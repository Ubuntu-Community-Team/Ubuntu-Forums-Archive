### Post by geeky-1 on 2022-11-03
Occasionally, a notification will open several duplicate tabs in Chromium. I've noticed this problem with both Ubuntu 22.04.1 LTS and 20.04 LTS. The typical suspects are Facebook and CNet notifications that end up opening as many as 8 or more duplicate tabs in Chromium. I Googled this problem and there was a Reddit post of someone having the same issue on PopOS with Facebook as an example, so it seems to be a Debian notification bug.
