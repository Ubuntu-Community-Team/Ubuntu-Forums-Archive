### Post by Merrattic on 2013-04-28
On Xubuntu 12.04 LTS  ;)  :

1. Paste will appear for every folder / blank space you click on in the right hand pane of thunar, as long as you have permissions to paste into that folder. Paste does not appear if you right click on folders in the Left hand pane set to "Shortcuts" but it will appear if you have tree view and again have permissions to paste. To get paste everywhere run thunar as root.

2. As long as there is more than one file in an archive, thunar will do what you want by design, admittedly if you right click and "Extract Here" it will create folder and extract to the current folder, but if you choose "Extract To.." you can select your destination.
