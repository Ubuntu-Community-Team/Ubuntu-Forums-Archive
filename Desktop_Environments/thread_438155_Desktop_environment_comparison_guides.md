You can install Fluxbox from the repositories. It's a little old now, but not *too* old. It works just fine and it's very fast and very stable.

Fluxbox does take a bit of getting used to and you have to learn how to configure it, but in my opinion it repays careful study.

Go ahead and install it. It won't screw anything up. All you'll get is another option at the login screen (under Sessions) called 'Fluxbox' and you can play with it at your own pace. You can hang on to GNOME or whatever environment you're more comfortable with so you can go back to that if you need a break from configuring. Just take baby steps until you get the hang of things.

Check out the links in my sig and feel free to ask questions. There are lots of avid Fluxbox users here on ubuntuforums. :)  (If you search these forums, you'll see that a lot of questions relating to Fluxbox have been asked & answered, which is handy.)


PS - If you do a search for 'fluxbox' in Synaptic, you'll get a list of all the packages related to Fluxbox. There's one called **fluxconf** which is a graphical tool to help you with the configuration. I prefer to edit the config files by hand, but you might want to try that fluxconf. Be sure to read carefully the descriptions of the other packages to see if you want to install those too.
