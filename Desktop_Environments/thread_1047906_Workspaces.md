### Post by rotf101 on 2009-01-22
So, I remember when I first installed Ubuntu, I kept hitting Win (Super) + D in an attempt to get to my desktop. Of course, it would never work, but instead brought me to a nifty little screen that showed all my workspaces. It was really quite convenient (after I got used to it), because I could move windows from from workspace to another simply by dragging and dropping.

After a while, I suddenly realised that Super+D doesn't work any more. I currently have Workspace Switcher 2.24.1. Did they update and remove that function? If not, how can I set it up again?

Thanks (in advance)!
--rotf101

### Post by Yashiro on 2009-01-23
It's part of Compiz, the 3d desktop effects manager.
If Compiz is not running this wont work.

If it used to work and now it doesn't, run Compiz config settings manager and fix it.
