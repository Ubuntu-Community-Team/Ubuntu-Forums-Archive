I don't know about the main menu, but you can always add it to a panel, and if you don't want to see it all the time you can have the panel auto-hide so it shows you the time when you mouse over it.

[edit: You can create a *new* panel off to the side, or wherever you don't already have one, and have it have *only* a clock, have it *not* expand, and auto-hide turned *on*.  This would supply an unobtrusive clock ;-) ]
