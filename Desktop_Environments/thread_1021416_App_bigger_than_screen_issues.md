### Post by Zorael on 2008-12-25
You want to configure Compiz to disable windows sticking to edges (I don't remember the name of the plugin, haven't got Compiz installed. Sticky Windows?), and also disable Constrain Y in the Move Window plugin. Then you just need to practice Alt-mouse dragging your windows outside the desktop area confinement when you need to access hidden parts. :3

And perhaps file launchpad bug reports to get developers to add sliders. Or write patches and submit them to the maintainers, if you have the expertise.
