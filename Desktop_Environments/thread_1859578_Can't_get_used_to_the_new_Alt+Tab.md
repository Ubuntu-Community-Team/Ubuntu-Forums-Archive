### Post by gordderp on 2011-10-14
I don't know what you guys feel, but to me the essential function of multiple workspaces is that I can have less windows to switch from once I separate them into different workspaces. But now Ubuntu 11.10 is forcing me to switch among all windows no matter how I manage them into workspaces.

It is also very weird that now having three windows of one application is quite different from having three distinct application windows. I even have to press some direction keys in order to switch among 3 or more firefox windows!!

I just wonder if there is any way to turn back to the old Alt+Tab? It seems that changing to gnome-shell doesn't solve this problem.

Thanks!

### Post by Copper Bezel on 2011-10-14
I think it's a feature in the Unity plugin in Compiz. Install compizconfig-settings-manager, find the Unity plugin, and disable the keybinding on the Switcher tab. Unless something very silly has happened, you should be able to enable any of the other Compiz switchers.
