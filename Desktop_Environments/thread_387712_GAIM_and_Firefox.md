### Post by CSandman on 2007-03-18
I'm going to go out on a limb and assume that you're using the kde desktop because of the whole konquerer thing.  The GUI method would be to go into the K Menu and open up the System Settings window.  The options you're interested in are under "Default Applications".  By default it should be set to open URLs 'based on the contents of the URL' by switching it to use firefox however you should force it to open firefox instead of Konqueror.
