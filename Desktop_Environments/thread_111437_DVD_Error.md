I have this same problem, I am not sure of an actual "fix", but if you go to: System: Disks: CDROM: DVD Video and press the "Play DVD" button, it *should* work.

This is also assuming that you have the necessary packages to play DVDs in the first place, if not then check out [the wiki page]("https://wiki.ubuntu.com/RestrictedFormats#head-cd84b8e23927ccdb4bb55ffd3074687abec0cf3b") for more help with that.
