### Post by irrdev on 2008-09-19
Kubuntu is only a month away from becoming a 100% KDE4-based distro. I have KDE 4.1 installed currently, and I know that the gtk-qt4-engine package is now a "dummy" package to gtk-qt-engine. I don't see any replacement packages for KDE3 users. Apart from requesting a gtk-qt3-engine package from the Kubuntu developers, you might try to install the gtk-qt-engine package which is in the [gutsy repositories]("http://archive.ubuntu.com/ubuntu/dists/gutsy/") - they haven't been updated for KDE4.
