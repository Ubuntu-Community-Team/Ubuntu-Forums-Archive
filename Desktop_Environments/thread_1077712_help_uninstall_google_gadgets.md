### Post by celticbhoy on 2009-02-23
Goto System --> Administration --> Synaptic Package Manager
Enter your password
Then use the searchbar to find gadget
Select google-gadgets
Select Complete removal
Click on Apply
Once finished open nautilus on your home folder press 'Ctrl & H' to display the hidden folders & files
Look for a folder named .google and double click on it, then select and delete the gadgets folder
