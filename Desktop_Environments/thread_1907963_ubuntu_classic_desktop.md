### Post by BC59 on 2012-01-12
First you have to install the Ubuntu classic.

```
sudo apt-get install gnome-panel
```

Then select logout from the up right wheel and when you see the logon screen, push the wheel which is at the right of the name box and choose Gnome-classic. Then put your password and logon.
