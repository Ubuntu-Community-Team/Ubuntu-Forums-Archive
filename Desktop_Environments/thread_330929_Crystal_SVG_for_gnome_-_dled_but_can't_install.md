### Post by mcduck on 2007-01-04
There are 2 ways to install icon themes (and metacity & gtk2-themes).

1. Open System/Preferences/Theme and drag&drop the tar.gz package to theme window (do not extract it first).

2. Unzip the package yourself to ~/.themes (to install only for current user) or /usr/share/themes (to install for all users)
