### Post by Juergen on 2005-07-01
You have to enable the driver after installing it.
Did you run 'sudo nvidia-glx-config enable'?

Or, if you want to know whats happening, look into '/usr/share/doc/nvidia-glx/README.gz', 'installing the driver'

If the driver is enabled you should see a nvidia logo on X11-startup.
After you got it to work you can suppress this logo: search in the README, Appendix D
