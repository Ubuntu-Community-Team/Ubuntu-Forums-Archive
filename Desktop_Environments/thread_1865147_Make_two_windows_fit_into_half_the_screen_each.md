### Post by Copper Bezel on 2011-10-19
Are you actually using 11.04 as your information says? If so, there's a plugin called the Grid that's enabled by default that works exactly like Windows 7's Snap. Drag a window to the top, it maximizes; drag it to the left, it fills the left half of the screen; drag it to the right, it fills the right half. You can change these settings or add keyboard and other shortcuts in compizconfig-settings-manager (which you may need to install; access it from Settings > Preferences) under the Grid plugin.
