### Post by undecim on 2009-10-21
The gnome-panel should exist on every desktop. I've never heard of this situation before. This is likely a glitch with your window manager.

For now, the key combination used to switch desktop is ctrl+alt+[left or right arrow]

Since you have 4 desktops (unless you manually changed it from 2 to 4) you are using compiz (the window manager with the special effects). Try switching to metacity by pressing ALT+F2 and typing "metacity --replace", and see if that fixes your problem temporarily

Also, don't worry about asking "newbie questions" no one around here gets mad at those kinds of things. Like 90% of my teachers said: The only bad question is the one you don't ask.

And besides: The best part about Ubuntu or any other Linux distro or Free Software is the community. As long as you are being nice (i.e. using less than 20 exclamation marks at a time, not using all caps, and not calling everyone you see a fascist nazi) almost everyone else will be nice in return.
