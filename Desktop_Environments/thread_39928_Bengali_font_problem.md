### Post by omiazad on 2005-06-11
[b]Well, no one of this forum has much knowledge on the Bangla issues I mentioned above. But finally I've solved the problem and here is my little guide, so that people can sort it out easily in future.

The problem happened because Ubuntu (and most other distro) comes with a font called [color=SeaGreen]FreeSans[/color]. That font has Bangla letters in it, but doesn't have any Open Type Layout. Well, I don't want to take you in the technical way, for now I'll just say that [color=SeaGreen]FreeSans[/color] font dosn't work well with Bangla.
[/b]
[color=Red]**Download Unicode Bangla Fonts**[/color]
---------------------------------------------------------------
Firstly you have to install Bangla fonts. The best place for Bangla Unicode 4.1.0 fonts is [Ekushey.org]("http://ekushey.org/projects/otf_bangla_fonts/") website. In this guide I downloaded the fonts into [color=SeaGreen]/home/omi/fonts[/color] folder.

[color=Red]**Installing Unicode Bangla Fonts**[/color]
 ---------------------------------------------------------------
You have to open [color=SeaGreen]Root Terminal [color=Black](Applications -> System Tools -> Root Terminal)[/color][/color] and put the command  ```
cp /home/omi/fonts/*.* /usr/share/fonts/truetype/freefont
```[color=SeaGreen][color=Black]This will copy all the downloaded fonts into your font folder.

[/color][/color][color=Red]**Tweak Unicode Bangla**[/color]
 ---------------------------------------------------------------
[color=SeaGreen][color=Black] Now put following commands...
 ```
cd /usr/share/fonts/truetype/freefont
rm FreeSansBoldOblique.ttf
[/color][/color][color=SeaGreen][color=Black]rm FreeSansBold.ttf
[/color][/color][color=SeaGreen][color=Black]rm FreeSans.ttf
fc-cache
[/color][/color][color=SeaGreen][color=Black]
``` This will remove the stupid FreeSans font from your computer and let your computer know that the font no longer exisit. Also the [/color][/color][color=SeaGreen][color=Black]fc-cache command will tell your system about the newly installed fonts.

Now enter 
[/color][/color][color=SeaGreen][color=Black]```
gedit /etc/fonts/fonts.conf[/color][/color][color=SeaGreen][color=Black]
[/color][/color][color=SeaGreen][color=Black]
```This will open the [/color][/color][color=SeaGreen][color=Black]fonts.conf file in Gedit text editor as root. Find all the  [color=SeaGreen]FreeSans[/color] there and replace with [color=SeaGreen]SolaimanLipi[/color]. As I have installed SolaimanLipi and want to use this font as my User Interface Bangla font. The texts of the file may look quite like this:

### Post by soumyadip on 2005-06-22
Well, you don't have to be so drastic as to removing default fonts and hand-editing the fonts.conf file. There is a comparatively simpler workaround :

Get the ttf-bengali-fonts packages from [here](http://packages.debian.org/cgi-bin/download.pl?arch=all&file=pool%2Fmain%2Ft%2Fttf-indic-fonts%2Fttf-bengali-fonts_0.4.4_all.deb&md5sum=e271e863aa8a70538de2d99b42c79359&arch=all&type=main)

install the package (from a terminal):
> sudo dpkg -i ttf-bengali-fonts_0.4.4_all.deb 

Open up System->Preferences->Font and change the Application font to one of the Bengali fonts.

As regards to Firefox, please select a Bengali font as the Serif font in Edit->Preferences->Fonts and Colors

Hope this is of help. ttf-bengali-fonts will be shipped in Breezy Badger, then you can 
> sudo apt-get install ttf-bengali-fonts 
to install the Bengali fonts
