### Post by jammet on 2005-10-19
How would you tell gnome to use a differnt app for a type of file (like mp3 and mpg's and such) so you dont have to use totem?

---

### Post by Ampersand on 2005-10-19
Browse to a directory containing an mp3, or whichever file type you're interested in. Right click on one of the files and go to Properties. Go to the Open With tab, and you'll be able to set the default application for opening files of that type.
