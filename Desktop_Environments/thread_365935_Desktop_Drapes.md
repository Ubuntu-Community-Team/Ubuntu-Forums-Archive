### Post by gjtoth on 2007-02-20
I know KDE does this very elegantly but, the only way to randomly change wallpapers in Gnome (that I am aware of -- and I'm sure there are other ways) is via Wallpapoz.

While Wallpapoz is good, it is still a PITA to set up (especially if you have a LOT of wallpapers!).

If you want a simply solution, Desktop Drapes is it.  You can check it out at:

[http://freshmeat.net/projects/drapes/](http://freshmeat.net/projects/drapes/)

It was a piece of cake to set up and works flawlessly (for me, anyways :) )
