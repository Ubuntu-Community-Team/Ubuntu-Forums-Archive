### Post by Jucato on 2006-11-09
You can manually "save a session", by going to Settings -> Save View Profile. I set Ctrl+S (usually used for things like Save) to this feature (in Settings -> Configure Shortcuts).

You might also be interested that Konqueror has a Crash monito plugin which will list your opened tabs if/when Konqueror crashes. You can pick the ones you like to open, or open them all. You can get this monitor by installing *konq-plugins* and enabling it in the Settings -> Configure Extensions section.
