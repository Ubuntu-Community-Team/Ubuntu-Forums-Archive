### Post by deadflowr on 2014-12-21
Full screen or simply Maximized Window Mode.
Nothing would show in fullscreen, you don't even need to hide the launcher because full screen hides that as well.
But maximize window mode only covers what the current window space you have, excluding the panel and, if set, the shown unity-launcher.

typically you can toggle fullscreen with F11, fwiw

From my own personal experience, the only time fullscreen did not function correctly was on a machine which really should never have used unity in the first place.
