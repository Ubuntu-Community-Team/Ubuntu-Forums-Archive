### Post by Azion on 2006-01-16
Yes, but getting them to dual boot will be tricky.
When you install XP, it'll take over the boot sequence itself(called the MTF).
So your gonna need a way to boot back into Ubuntu, to setup Grub/Lilo

If you have seperate hard drives, it's simple.
Just disable the XP one.
If not, you'll need a linux boot disk
