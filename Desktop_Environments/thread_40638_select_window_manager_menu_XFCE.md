### Post by derrick1985 on 2005-06-09
First, dont' use XFCE, use xfce4

sudo apt-get install xfce4

If you want the addon's to xfce4, type it in synaptic to get a good list.

Secondly, it should be automatically added to your GDM list. Logout then select session and choose xfce4 as the manager. Enter in your name/password and welcome to xfce4
