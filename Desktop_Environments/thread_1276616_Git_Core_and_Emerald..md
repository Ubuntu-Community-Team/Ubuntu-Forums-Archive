I'm afraid I cannot help you here.

However, I do have one point to make:
Emerald is no longer being supported or developed, and, therefore, it is not advised to be using it anymore. Therefore, I say use another window manager (compiz, GTK, metacity) and get themes for that ([http://gnome-look.org](http://gnome-look.org)). If you're using emerald to change the window button order (maximize, minimize, close etc) and where it is, there's some g-conf edits you can make (in the Terminal or via the gconf-editor, whichever you prefer) to do so.
