### Post by c2olen on 2008-07-28
Have you set the "save automatically on exit" checkbox in the session panel?
Go to system->preferences->sessions. Select tab "Session options" and check the "Automatically remember...." checkbox.
Or try the "Remember Currently...." button first.

I know it says applications, but it will also save the settings to the gnome-panels.

Cheers.
