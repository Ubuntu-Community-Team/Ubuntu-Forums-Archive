### Post by jordanmthomas on 2007-02-11
You need this in your xorg.conf

```
Section "Extensions"
   Option "Composite" "Enable"
EndSection

```

...if all you did was install it from synaptic, you are not done configuring it.
[http://wiki.beryl-project.org/wiki/Install_Beryl_on_Ubuntu](http://wiki.beryl-project.org/wiki/Install_Beryl_on_Ubuntu)

Follow one of these guides and you'll be fine I'm sure
