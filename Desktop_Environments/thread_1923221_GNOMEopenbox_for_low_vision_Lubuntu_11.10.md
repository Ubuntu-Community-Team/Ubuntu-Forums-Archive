### Post by 2F4U on 2012-02-10
Openbox is already installed on Lubuntu since it is the window manager of LXDE. If you want Gnome, it depends a bit on what you are looking for. For example, the package ubuntu-desktop would install the current desktop of Ubuntu with Unity, while gnome-core is just the essential Gnome environment. The package gnome-desktop-environment provides the standard Gnome environment, while gnome-shell installs the newer Gnome3.
