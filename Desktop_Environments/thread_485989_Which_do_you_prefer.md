### Post by Neobuntu on 2007-06-27
I prefer Kubuntu. You are free to prefer what ever you like. Ubuntu is good as well. I am concerned about division of effort but if one project upgrades to crap, the other saves the day.

I think that once you get into it, KDE molds better to your personal wishes. I like the way it starts out and I like the  looks and styles available.

Whether a program is KDE or Gnome based is irrelevant. If you know how to set the look/theme/style for the one you are NOT using.

Two many duplicate programs can be confusing to a light computer user, but removing them is easy.

Sure you may (and probably do) run both (and more) desktop managers since hard drive space is cheap. The (minor) problem is two fold.

First, when you upgrade, it will take longer. 

Second, totally removing one or the other as if only one was installed is difficult compared to a reinstall (of your preference).

BTW. The bottom line is, a distribution release upgrade is NOT as good as a clean install. So do what you want but just start over AFTER a new release has already been out a while. This is not as laborious as installing and building up Windows. 

So after running both to test, clean install the one you like best. Sure, it can be done but don't beat you head against the wall, backup and clean install.
