### Post by aysiu on 2006-08-18
I have a tip, if they're all the same hardware, tweak one installation exactly the way you want it--then use PartImage to copy the partition over: [http://www.psychocats.net/ubuntu/partimage.html](http://www.psychocats.net/ubuntu/partimage.html)

There are also good kiosk tools for both Gnome and KDE.
