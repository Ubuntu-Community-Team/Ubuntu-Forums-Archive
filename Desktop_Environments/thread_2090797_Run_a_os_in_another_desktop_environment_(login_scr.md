### Post by sudodus on 2012-12-03
Yes, you can run a virtual machine, for example VirtualBox, and in that machine you can run another OS (another Linux, Windows, ...). As a matter of fact this is rather common, and many of us at the Ubuntu Forums run virtual machines, for example when we test new operating systems, and when we want some application, that only works in Windows.

But of course there is some overhead processes involved, so it will be slower.
