### Post by happyponcho42 on 2006-01-20
I recommend following the procedures in the thread I linked to.  I remember trying what morphodone suggested and X server wouldn't come back up.  If you're going to attempt his way, be sure to backup your xorg.conf file first; if X won't return, just delete the modified xorg.conf and rename the backup to xorg.conf.

Backup
> sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_working

Restore
> sudo rm /etc/X11/xorg.conf
sudo mv /etc/X11/xorg.conf_working /etc/X11/xorg.conf


If you're having trouble editing, try
> sudo gedit /etc/X11/xorg.conf
modify and hit the save button.  You can also use nano instead of gedit.
