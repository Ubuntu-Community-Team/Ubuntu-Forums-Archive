### Post by Alan James on 2010-04-25
When installing something like openSUSE they will ask you during the installation process which interface you want. Ubuntu does not do this. Canonical makes separate distros for each interface, although they all share the same repos. Ubuntu, Kubuntu, Xubuntu, each has its own interface and many unofficial distros exists as well that take advantage of the great code under the hood.
