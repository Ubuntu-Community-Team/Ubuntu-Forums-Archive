### Post by srunni on 2006-07-21
OK, I was just browsing the forums when I got a good idea for having full Flash/Shockwave support. Some of you might have used the IE Tab extension in Win Firefox. It basically points to an .exe that is launched in Firefox when you want to use IE: thus IE in Firefox. But what if the IE tab extension was slightly modified so that you could launch Win Firefox in native Linux Firefox? This way, everyone could continue to use Linux Firefox, but when needed, you could easily switch right inside Linux Firefox to Win Firefox. This might even be possible by simply installing IE Tab in Linux Firefox, going to the settings for it, and pointing it the Wined Firefox installation instead of IE.

Then, for sites like YouTube, you could tell the IE Tab extension to automatically launch Win Firefox, giving you full Flash support. Same with other sites that use a lot of Flash/Shockwave.
