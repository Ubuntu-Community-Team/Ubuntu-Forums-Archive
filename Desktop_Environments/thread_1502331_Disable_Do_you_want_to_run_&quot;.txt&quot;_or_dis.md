### Post by Tamlynmac on 2010-06-05
I'm not running 10.04. But in every other version if you open your file browser and right click on one of the files, it will display a context menu. In the context window select gedit as the program you wish to open the txt files with. If gedit isn't displayed select "open with other application". Find gedit and select it.

Then double click on the file and it should open in gedit. Once a type of file format is selected to open in a specific application, Ubuntu assumes you wish to open all files of that format with that application.

After opening that file, try another file of the same format. I think you will find that it automatically open in gedit.

Good Luck & I hope this helps.
