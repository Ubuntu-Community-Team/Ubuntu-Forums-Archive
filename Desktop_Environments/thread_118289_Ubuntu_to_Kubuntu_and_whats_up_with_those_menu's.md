### Post by raggamuffin on 2006-01-16
[QUOTE=corefile]If I've already got Ubuntu installed, is there any advantage of wiping it out and installing kubuntu instead of just apt-getting the KDE destop with my current Ubuntu install?[/QUOTE]

not really...you can have both GNOME and KDE installed without any problems...unless you hate it and want GNOME banished from your system, I don't see a reason to do a clean install...

to install the newest KDE is easy...

step 1...update your apt sources.list to include KDE 3.5 packages by following the instructions at [http://kubuntu.org/announcements/kde-35.php](http://kubuntu.org/announcements/kde-35.php) ...

step 2... sudo apt-get update && sudo apt-get install kubuntu-desktop


and then you can choose between GNOME and KDE at the login screen when you start a session...
