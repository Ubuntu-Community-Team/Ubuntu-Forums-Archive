### Post by deadflowr on 2014-08-26
Nvidia-settings will generate an xorg file if one does not exist.

Anyway, do you not see the entry for the existing xorg.conf file in your screenshot?
It's in between xkb and xorg.conf.backup.

In nvidia-settings, unclicking the merge with existing xorg.conf will make the saved file it generates override the existing one, where as merge will add the new changes to whatever existed already.
