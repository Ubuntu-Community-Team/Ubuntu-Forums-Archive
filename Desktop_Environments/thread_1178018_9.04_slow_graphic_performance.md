### Post by Youresorock on 2009-06-05
I have a system with the same GMA950 video card.  With a fresh 9.04 install it was all hosed and super buggy.  I found some really bad advice online on reverting the driver, etc.

In the end just running a reconfigure on xorg fixed it 100%.


```
sudo dpkg-reconfigure xserver-xorg
```
