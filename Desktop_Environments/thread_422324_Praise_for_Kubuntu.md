### Post by ardchoille42 on 2007-04-25
I'm not totally sure this is the correct section for this, so bear with me.

The last time I used KDE was in Mandrake Linux - before Ubuntu was ever released - and KDE was, in my opinion, a piece of junk. Every other app crashed, the default settings were a mess and it was just too time consuming to get it running correctly. I'm not sure if these problems were the fault of KDE devs or Mandrake devs but I didn't like it at all.. so I used gnome. Recently, though, gnome has, in my opinion, just gotten worse and a few friends suggested I try KDE again.

Well, I am now using KDE full time again and all I can say is either the KDE devs or the Kubuntu devs (or both?) have done an outstanding job. Kubuntu is stable, fast and the default settings are excellent... not to mention the fact that a lot of things are integrated better in KDE than in gnome. Kontact is awesome!

Whoever is doing this awesome work, please keep it up. **Kubuntu ROCKS!**

Bye bye gnome :)
