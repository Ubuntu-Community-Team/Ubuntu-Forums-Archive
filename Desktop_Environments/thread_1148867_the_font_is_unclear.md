### Post by Marlonsm on 2009-05-04
Right click the desktop > Change background.
Go to the fonts tab and try changing that setting on the botton part of the window.

Also, you can install proprietary fonts too.
You can either look for a guide on the forums (I forgot how to do it now) or just open the Synaptic Package Manager, find and install ubuntu-restricted-extras (that will not only install the fonts, but many other proprietary stuff, like codecs)
