### Post by oobuntoo on 2006-06-17
The panel already have the original taskbar added to it by default, so you have remove it from the panel first. After that, add the taskbar version 2 to the panel. 

I'm not at home right now, so I don't know the exact steps to tell you, but you can start by right-click on the panel and see what options you have.

### Post by dave73 on 2006-06-17
You're right, that's all! I did: right-click on the panel, remove program, remove standard taskbar, then install new program, taskbar v2! It works... thanks again!!
