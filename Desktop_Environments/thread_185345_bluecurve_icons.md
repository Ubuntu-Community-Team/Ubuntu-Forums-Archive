### Post by BenTyreman on 2006-05-31
Download the artwork package for Fedora:
```
ftp://ftp.linux.ncsu.edu/pub/fedora/linux/core/development/i386/os/Fedora/RPMS/redhat-artwork-0.243-1.i386.rpm
```

Install using alien:
```
sudo alien --scripts redhat-artwork-0.243-1.i386.rpm
```

This will install the entire Bluecurve theme. Never had any problems doing it like this.
