### Post by jerstew on 2009-09-02
Not sure why you'd want that. Except for nostalgia maybe. Looks to be an early KDE distro, but not sure what. Notice the K menu icon has been changed. Could be anywhere between a KDE 1.1 to a 2.X, circa '99 to '03 or so(?).
As for the file man, I think that came with the Corel Office suite, which would be a spin off of the WordPerfect brand of office software. Good luck finding that one. There aren't gonna be any tarballs floating around for you to install, much less source code to compile for compatibility with current runtime librarys.
So if you can figure out what distribution that might be, then get hold of [www.cheapbytes.com](www.cheapbytes.com) and see if they have an old copy on cd in the archive (if they keep one). Most likely for free!
Interestingly though, the date in the systray clock is '07. That makes for one oddball screenshot you found.
