No problem.The .deb file installed the theme.
If you download other themes in the form of a compressed file eg. tar.gz
just drag and drop in the appearance window to install.
Sometimes you will download a file which contains an emerald or metacity theme as well as a gtk theme
and you have to unpack this file first to get to the  compressed gtk theme.
If you are running compiz, install fusion-icon.

window managers.........eg compiz, metacity
window decorators......eg gtk, emerald (different types of themes)
