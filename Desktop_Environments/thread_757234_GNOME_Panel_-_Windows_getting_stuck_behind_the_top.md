It seems to just be a case of swapping the roles of the monitors, but I'm completely stumped as to how to achieve this.

Edit

---

Made some further progress in finding out what it could be.

It appears TwinView just merges the two monitors and reports them as one large monitor to the rest of the operating system, perhaps bar Gnome. Although there is an option within the device options in xorg.conf for nvidia to default the display to a particular monitor... It does not seem to make Gnome fully comprehend which monitor is default, thus the new items on the desktop appears in the closest possible to top left place and the windows are ignorant of the fact that there is a Gnome Panel in the way.

### Post by dje on 2008-07-05
If you are using Compiz, open up the Advanced Desktop Effects Settings and ensure that the 'Place windows' plugin is enabled
To install the effects manager if you don't have it:
```
sudo apt-get install compizconfig-settings-manager
```

hope that helps,
dje
