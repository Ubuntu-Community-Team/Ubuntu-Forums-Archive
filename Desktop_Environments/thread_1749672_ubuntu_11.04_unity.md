### Post by 3Miro on 2011-05-04
Logout, select Classic or Classic No Effects, login.

Unfortunately the Gnome2 interface was killed by the Gnome people (this has nothing to do with Ubuntu or Canonical). Ubuntu 11.04 comes with the new Ubuntu Unity interface and Classic interface as fallback/alternative. Ubuntu 11.10 will come with Unity 3D and Unity 2D as fallback. However, on Ubuntu 11.10, you will be able to install the new Gnome-shell interface (perhaps even more different then classic).

If you like the two panel interface and you want to keep it for longer than 11.04, then you should consider XFCE or Xubuntu.

### Post by fremantle on 2011-05-06
"Clean" installation generally means obtaining an iso of the new release, burning it to a cd or creating a live usb, and using them to install the release in a system. On the other hand, what seems not so clean is doing a network upgrade from one release to the other. 

Clean installation can be done by wiping out the hard drive, or you can just back up the files in your Home folder with something like deja dup, take the backup files outside your computer, install a new release, and restore those files again using the same utility.
