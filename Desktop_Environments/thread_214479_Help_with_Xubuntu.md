### Post by MetalMusicAddict on 2006-07-12
You at some point tried to edit the menu and it starts with a blank one. If you click on the desktop it will be blank also. I would gather your working off a fresh install. If you update then edit the menu it wont start a blank one. For now though you need to delete the blank file you created and the other one will come back. Might be called menu.xml in your home folder. Just delete it if you can get to it.

See [THIS]("http://www.ubuntuforums.org/showthread.php?t=194680&highlight=xubuntu+menu") thread.
