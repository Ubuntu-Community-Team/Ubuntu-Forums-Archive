### Post by Krytarik on 2011-01-26
I've just today tried all popular docks.

Cairo is much too hardware-hungry, and overly effectfull by default. May work good if you have a really fast, pretty decent machine.

Avant Window Manager (AWN) doesn't work for me currently, can't select items in the grouped window picker, looses focus before I can click it, depends on the autofocus setting in Compiz. Otherwise it's imo the best of the docks. Good balance between graphical effects and hardware resources load.

Docky is a too spartanic, blocks the space where it is located, instead of laying above the windows, like the other docks do, unless you activate autohide, and you have to right-click on the items to get the group window list, and window preview isn't even in it's options.

DockbarX is imo the best choice if you just want a panel applet which resembles the main functions of the above mentioned docks, which is: combined launcher and task item, grouping of windows, and possibly windows previews. DockbarX is fast and you can have window previews.

Make your choice, a feedback would be nice!
