### Post by lovinglinux on 2009-06-29
I want to be able to hit the "Show Desktop" app or use a command to minimize all windows at once and still be able to see the screenlets. If I simply use the "Show Desktop" app, the screenlets are also hidden, which doesn't make sense to me. I have already tried to set the screenlets to "below", but they still hide.

I know there is a compiz widget layer, which I use for other screenlets. Basically I want to have a another set of screenlets on the desktop, which I also want to see with another shortcut.

The keybinding in compiz general settings only allow to minimize windows individually.

Any ideas?

---

### Post by 243kof on 2009-06-29
I solved that recently. In CompizConfig Settings Manager, under "General Options" uncheck "Hide skip taskbar windows". I read it in another thread and it worked. I think that's all it needs.
