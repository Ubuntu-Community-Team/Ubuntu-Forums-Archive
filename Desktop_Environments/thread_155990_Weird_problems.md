### Post by localzuk on 2006-04-06
> my RAM memory has only ~300 MB free constantly ?!

Don't worry about this. This is how linux works - it uses all ram and then frees the stuff that it isn't actually using when required. It allows for applications that are used regularly to be loaded into memory and left there instead of having to access the HDD all the time.
