### Post by mcduck on 2012-04-16
Compiz allows you to disable closing/minimizing/etc a window, by defining a rule for it in the Window Rules-plugin (Using CompizConfig Settings Manager).

However that doesn't give you any easy way to enable/disable the lock, so you can't do it from right-click menu or a keyboard shortcut or anything like that. And it also doesn't prevent the user from shutting down the computer if the program is still running, or closing the program from it's own menus.

As far as I know there's nothing that would work like you described, at least not for normal desktop environments. But it sounds like a kiosk mode setup might be something worth trying, of course depending on what exactly you are trying to accomplish...
