### Post by myand on 2006-06-29
Is it possible to change the way text is painted using the keyboard? It seems that in Gnome using shift+ctrl+left/right arrow paints everything upto the next space between words, whereas in Windows the separation is not between space/non-space but between alphabet/non-alphabet characters. My fingers have found the latter very handy when editing html; for example, painting a link text is easy because it's limited by the less than/greater than characters (< and >) of opening and closing tags around it. I'd like to have Gnome text painting behave like that, but is there any way to make it so?
