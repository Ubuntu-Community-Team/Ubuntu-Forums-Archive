### Post by 3Miro on 2011-09-08
Assuming you have desktop effects enabled (you do if you are using Unity), then you can install compizconfig-settings-manager. It is in the software center.

Form CCSM you can adjust a million effects under Ubuntu (getting the cube to work with Unity it tricky, make sure to read first). You can also set several effects for Alt+Tab.
