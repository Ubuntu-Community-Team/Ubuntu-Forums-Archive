### Post by zombifier25 on 2012-05-06
Don't know if you need this anymore, but when upgrading from one version to another the Compiz profile tends to get messed up on the way. If you want to use the 3D Unity, press Ctrl - Alt - T to open a terminal and type this:
```
unity --reset
```
