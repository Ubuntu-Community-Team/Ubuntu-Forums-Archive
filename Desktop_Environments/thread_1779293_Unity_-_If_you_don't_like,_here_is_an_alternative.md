### Post by birkopf on 2011-06-10
I personally hate new Unity bar on the left. After quick search in the net I came out across many posts with same opinion. Even whole groups have been created! ;)

Simple way if you don't like Unity but want to have (better) functionality:

Install docky from repositories. 
Configure to Autohide.
Remove (from docky) icans that you don't want there by dropping them on your desktop 
Add docklets and other functionalities in docky options.

Simple and works great!
For me - fastest way to get directly to my folders.

One more thing to mention. Docky requires compositing so it's good to make sure this element is enabled. 

First, make sure that Compiz Fusion is turn off:
Open “System”->” Preferences”->”Appearance”->”Visual Effects”, and select “None”.

Then press “Alt + F2 ” or open the Terminal, type: gconf-editor to open the Configuration Editor.

Find the key: /apps/metacity/general/compositing_manager, then turn on.

The metacity is a compositing manager now! 

And you can run some application like Avant Window Navigator without Compiz, that’s the power of Compositing!
