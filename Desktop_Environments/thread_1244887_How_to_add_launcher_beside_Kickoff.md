I want to add launcher( widget) beside Kickoff instead of system tray, but can't move this launcher to the left. Please help me! Thank you!

### Post by krazyd on 2009-08-20
1. Right-click the panel and select 'Unlock Widgets'. If there is no 'Unlock Widgets' option, go to step 2.

2. Click the cashew-looking icon on the left edge of the panel (it should highlight when you mouse-over it). The panel options will appear. While in this mode, you can drag elements in the panel around wherever you want them. Drag the icons over next to the kickoff menu.
