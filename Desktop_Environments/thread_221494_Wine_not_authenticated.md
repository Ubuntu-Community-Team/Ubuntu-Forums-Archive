### Post by CatKiller on 2006-11-30
> **Biggus said:**
> Just wondering if anyone had any thoughts at all on this one?

I found this thread when searching for help with a message I recieved from the Update Manager, namely that wine is "NOT AUTHENTICATED"

It's because you're installing from a non-Ubuntu repository. Technically this means that the owner of the repository can do anything to your computer if you're not paying attention.

There were a series of threads about it recently, since the owner of a repository that got included in a big list of repositories for people with little understanding found out he was on this list and wanted to warn people of the danger. So he changed the background of the desktop and made it so that they couldn't change it back. Obviously, much more malicious things are possible.

If you're OK that the wine package is not guaranteed in any way (which you should be, since it's from a wine repository that you've added), go ahead with the upgrade. Upgrading other non-authenticated packages, like **sudo**, say, would be... silly.

EDIT: [This]("http://www.ubuntuforums.org/showthread.php?t=297814") is a thread describing the background-changing fun-and-games.
