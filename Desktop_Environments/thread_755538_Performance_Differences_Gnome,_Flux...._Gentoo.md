### Post by bingoUV on 2008-04-15
It depends on what you want to do on the system. For general desktop activities, you cannot get much faster than xubuntu(or xubuntu-desktop installed on ubuntu). 

If you go for even lighter distros, your RAM, graphics card, and graphics memory would remain inefficiently used. RAM will be used for disk caching but generally we are not likely to read the same files again and again so disk caching does not always give good performance benefit. 

On the other hand, if you want to run heavy applications (number crunching, databases, web servers etc) on the system along with a user interface, using a lighter distro would give more room to the server.
