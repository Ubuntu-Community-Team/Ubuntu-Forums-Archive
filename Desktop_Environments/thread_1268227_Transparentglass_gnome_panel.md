### Post by steveneddy on 2009-09-17
I looked at the pic AFTER posting above. Sorry.

The link to my other post is still relevant and if you are customizing it tells how to get all the cool transparent effects for your desktop.

**Do you mean the transparent window borders?**

That is using Compiz, and [COLOR=Green]**Emerald**[/COLOR] as the window decorator.

Then you just DL themes from [gnome-look.org]("http://gnome-look.org") and put them in [COLOR=Green]**Emerald**[/COLOR].

Also you might try installing **fusion-icon**

```
sudo apt-get install emerald
```

```
sudo apt-get install fusion-icon
```
