### Post by crazyamerican on 2010-04-05
I did it! Instead of finding a new theme, I altered an old one.

Steps:
Download MidnightOSX theme in Controls.      [www.art.gnome.org/themes](www.art.gnome.org/themes)
Apply MidnightOSX.
Open Appearance.
Customize a theme.
Select MidnightOSX as controls.
Select Dark Room as window border.
save theme.
Open Compiz Config Settings Manager.
Go to Trail Focus.
Change number of windows to 15. (may or may not be necessary)
Change opacity of focused & unfocused windows to 90.
Go to Window Decoration.
Change Shadow Radius to 18.0.
Change Shadow Opacity to 6.0.
Change Shadow Color to Red.
Apply cool, dark desktop background.
Sit back and bask in your awesomeness.:popcorn:
