### Post by JRV on 2011-04-16
> **BigJules said:**
> 
+ Having xmacro driven scripts as launcher 


Right click on the desktop, choose "Create Launcher.
Make your launcher and move it where you want it.
(I put mine in ~/USERNAME/.local/share/applications.)
Then drag it to the launcher bar.

> **BigJules said:**
> 
+ can we reduce the size of launcher icons? 

Install ccsm, enable the Ubuntu Unity Plugin.
There is an option to change the icon size there.  The smallest it will go is 2/3 the original size.
