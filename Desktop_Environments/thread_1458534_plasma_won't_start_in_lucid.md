### Post by Untitled_No4 on 2010-04-20
Can you run commands using Alt F2?
If so, try to run Konsole and then type
```
plasma-desktop
```
and see what happens.

---

### Post by fadey on 2010-04-20
Wow. Great idea!!! I don't know why that happened, but it results that after upgrade to lucid plasma-desktop has got uninstalled. After installing it, now I have my lovely panel back :-)
Thanks a lot
