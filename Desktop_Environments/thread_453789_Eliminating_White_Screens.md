### Post by tgoose on 2007-05-24
Large font sizes can be used by going to

System > Preferences > Font

and increasing the size of each one. This should work for most applications; certainly any that are installed by default (for those based on KDE another step may be required, but unless you have installed lots of extra software that shouldn't matter.)

As for dark colours, these can be customised in

System > Preferences > Theme.

There is a theme called Ubuntu Studio which has lots of dark backgrounds, which can be installed using the method described on [http://ubuntuforums.org/showthread.php?t=448494](http://ubuntuforums.org/showthread.php?t=448494)

I personally find the darkness of that colour scheme very wearing on the eyes and have made it lighter, but it might be useful to avoid too much light. 

I hope that's useful to you!
