### Post by smilingfrog on 2009-01-05
Suddenly, I can no longer read pdf files in kde. I open KPDF, which seems to work. I navigate to a folder that has pdf files in it and the folder is blank.
If I double click on a pdf file through the menu, it does not open. File associations seem to be correct.
All these files are visible and readable in gnome without error.

Help?

### Post by smilingfrog on 2009-01-05
Well, god bless google.

I found the solution to this here [http://sgruendel.blogspot.com/2008/06/unable-to-open-pdfs-ind-kpdf.html](http://sgruendel.blogspot.com/2008/06/unable-to-open-pdfs-ind-kpdf.html)

The solution was to delete the kdeuser1 association with pdf files. I have no idea how that association formed, but I'm glad it's fixed.

Hope this helps someone else.
