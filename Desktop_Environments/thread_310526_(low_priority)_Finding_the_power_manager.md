### Post by LLRNR on 2006-12-01
For the monitor stuff, you can go to the K Menu -> System Settings -> Computer Administration -> Monitor & Display -> Hardware tab and Power Saving tab.

Alternately, you could type **kcontrol** in a konsole and see what else you can change from there.
