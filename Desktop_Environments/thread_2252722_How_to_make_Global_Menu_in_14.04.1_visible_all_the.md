### Post by grahammechanical on 2014-11-13
The information in the global menu changes according the particular application that has focus at the time. So, what information is there to be made permanently visible? What we can do in Ubuntu 14.04 is to put the application menus (such as File, Edit, View, etc.) back into the application title bar.

Open System Settings>Appearance>Behaviour and the in panel entitled Show the menus for a window we can tick the radio button labelled In the window's title bar. Then you can ignore the menu bar (left top panel)
