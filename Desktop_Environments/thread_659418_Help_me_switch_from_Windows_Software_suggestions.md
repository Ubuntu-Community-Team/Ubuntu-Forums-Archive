Yep, Ubuntu uses GNOME by default and Kubuntu uses KDE.
KDE and GNOME are called desktop environments. They're basically what make up the whole structure of the graphical interface you use. KDE is extremely tweakable but can be extremely confusing/overwhelming as well, whereas GNOME is aimed at being usable but still customizable. I personally prefer GNOME, but you can try them both. ### Post by drcranium on 2008-01-05
The short answer=
Gnome is based on the GTK toolkit.  KDE is based on the QT toolkit.  Basically, all this means is that a GTK app won't run on a system that only has QT libraries.  However, Ubuntu and most distros(I don't know of one that won't) allows you to use KDE apps under Gnome and vice versa by using both toolkit libraries.

There is often debate about which is better but really a combination of gnome and kde apps works best.  Try everything out.  Find what you like best.
