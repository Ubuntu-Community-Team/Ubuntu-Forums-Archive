### Post by mcduck on 2012-01-13
Dragging anything from the dash to desktop will not move it there. And you defintiely don't have that knid of power over system files... :)

It simply created a new launcher for the same application. The original launcher still exists in your menu just like before. (and the actual program files sit elsewhere in their correct system directories just like before)

What cmes to where you should install a game, that's pretty much your own decision, based on what kind of partitons you have, what you are using them for, and how much space each has (and the game needs).
