### Post by raymacey on 2006-11-03
I'm using Kmail to access my ISP provided IMAP email account, and I'm having some trouble with the filters.  For some reason, the filters run perfectly when I first open KMail, finding spam, moving the odd email based on subject line etc.  However, from this point on, the filters don't work.  KMail is set to check for new mail every 2 minutes, but these just sit in my inbox without having any filters applied at all.  When I check the filter log, they don't appear at all.  Once I select an email and apply my filters to it though, it processes correctly and appears in the filter log.

What am I missing?

### Post by raymacey on 2006-11-03
Well, I found the answer.  I wasn't using disconnected IMAP mode.  Why that let the filters work when I first ran KMail though, I don't know
