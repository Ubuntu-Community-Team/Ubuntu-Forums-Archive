---

### Post by Richard Kut on 2007-06-08
Yes, it is possible. If you select are using Nautilus, which is the default file manager in Ubuntu, then the following will work. Click on the "View" item in the pull-down menus of Nautilus, and then select "Zoom Out" as many times as needed to get to the desired size.
