### Post by grahammechanical on 2013-08-16
I do not have this utility installed on my system. So, none of my activity is forgotten. The purpose of Activity Log Manager is to give the user some control over what information is logged and how long it is kept. I do not know if 12.04 has a System Settings utility called Security and Privacy. I think 13.04 has it. I know 13.10 has it. These utilities are designed to deal with users concerns about privacy.

The operating system logs activity. Otherwise an application would not be able to give us a Recent Files list and the Dash would not be able to show recently used apps and files and neither would a web browser have a working History function. The user now has more control over what is logged and how long the data is kept. The default setting is usually no limits. If we don't change the settings or install a utility that by default deletes the logs after 60 minutes then nothing will be forgotten by the OS.
