### Post by Rick St. George on 2018-04-19
OH ... For general info, from what I can tell, the collaboration between Canonical and Microsoft has to do with Azure.
MS Azure is a PaaS, IaaS operating in conjunction with SaaS to Windows or 3rd party software and systems such as Linux.
It is a "closed source" platform for "open source" client SDKs. From: [https://en.wikipedia.org/wiki/Microsoft_Azure](https://en.wikipedia.org/wiki/Microsoft_Azure)

WSL is a compatibility layer running on Win10 for running Linux binary executables, providing a Linux compatible kernel interface developed by MS, which can run flavors of Ubuntu on top of it. For more info: [https://en.wikipedia.org/wiki/Windows_Subsystem_for_Linux](https://en.wikipedia.org/wiki/Windows_Subsystem_for_Linux)

Hope that clears it up for anyone else on this subject matter, at this current time (April 2018).
