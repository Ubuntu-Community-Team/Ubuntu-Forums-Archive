### Post by matt_symes on 2011-02-09
Hi

Try this.

Open nautilus and from the menu

Edit ->Preferences ->Icon View Defaults

and select the size you want.

For your second question...

System->Preferences->Startup applications

is the most basic way.

Kind regards
