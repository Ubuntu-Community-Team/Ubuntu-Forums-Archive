### Post by Wolki on 2005-10-20
[QUOTE=Murgle]So I just noticed that when I right click on a tab in the Window List there is an option Roll Up.  Being the curious being that I am I tried it out and poof my window seems to minimize, the reverse option for this is Unroll.  Is there any Difference between this and Minimize?[/QUOTE]

Yes. Roll Up means reducing the window to its title bar. This can be handy if you want a window around (ie not minimized), but not take up a lot of space either.

In the metacity preferences, you can set windows to roll up on a double click on the titlebar, too.

---

### Post by Wolki on 2005-10-20
[QUOTE=Murgle]I'm not sure that i understand the difference... with roll up the window is only in it's title bar... isn't that what minimizing does?  When a window in rolled up how is it more around than when minimized?[/QUOTE]

If you minimize it, the window is completely hidden and only visible in the task list. If you shade it ("Roll Up"), the contents of the window are hidden, but the title bar (the one containg the window name and the minimize, maximize,close buttons) is still visible. Try it out and look closely, you'll see the difference.
