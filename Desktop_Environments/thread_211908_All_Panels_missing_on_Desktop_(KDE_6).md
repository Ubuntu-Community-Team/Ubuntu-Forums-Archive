### Post by Jucato on 2006-07-09
Just a few tips for the next time you encounter another problem like this:
1. Alt+Tab will cycle you through opened windows (minimized or maximized).
2. Alt+F2 will launch the Run Command dialog box, where you can type in the command name of the application (lik *systemsetting*s for System Settings)
3. Alt+F1 will launch the K Menu.

These are the default shortcut keys in Kubuntu/KDE. Unless of course you changed them already. :D
