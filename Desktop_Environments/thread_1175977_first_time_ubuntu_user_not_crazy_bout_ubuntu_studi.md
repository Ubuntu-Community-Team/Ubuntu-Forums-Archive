well the desktop that is. I have used fedora for several years with no issues but figured I would give ubuntu studio a spin to get into free home recording. well the good first. all apps I tried seemed to open a bit faster then on fedora. the bad? well apart from ugly desktop background theme, unbelievably minimized apps just "disappeared" and could only be brought back with alt tab. I was really shocked that a variant of the number one linux would do that out of the box. well next I installed the windows list applet on the upper panel and could see the opened windows. ok. well then I decided I should have a bottom panel for the windows so I installed it with the windows list applet. I removed the applet from the top panel. ok so far. hmm, I moved (dragged) the few items I had added on the top panel (firefox and the open office apps) a bit to the right so as to have space between them and the main menue. well unbelievably as I moved the items, like magic in there place appeared the open browser and gnome terminals that were on also on the bottom panel. wow. weird stuff. I guess I am going to try regular ubuntu and see if I can get it into a studio like state. and yes I made absolutely certain I had removed the windows list from the top applet. strange indeed.
