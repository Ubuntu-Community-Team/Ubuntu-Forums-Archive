### Post by ugm6hr on 2010-01-02
There are some specific remotes available for computer-based media centres - just check your local shop for options.  Some are Windows only (I think), while most work as a wireless keyboard, and you can program each remote button to act as a keyboard shortcut.

Otherwise, get an all-in-one wireless keboard / trackball for full control.
