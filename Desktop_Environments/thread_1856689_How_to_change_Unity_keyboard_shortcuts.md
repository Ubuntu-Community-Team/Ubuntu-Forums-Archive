### Post by krapp on 2011-10-08
More specifically, how do I free up the "Super" key (Mod4) and stop it from invoking the dock?!

The GUI available in System Settings doesn't represent all possible shortcuts.

---

### Post by Copper Bezel on 2011-10-08
You can install compizconfig-settings-manager and run it (with the command ccsm) and change or disable the key combination in the Unity plugin ("Key to invoke the Launcher"). You might lose the <super># shortcuts, though.
