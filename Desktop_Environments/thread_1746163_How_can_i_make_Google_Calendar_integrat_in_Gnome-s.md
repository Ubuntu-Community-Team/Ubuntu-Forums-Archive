### Post by mvlaghate on 2011-05-06
Thunderbird has a "Evolution Mirror" plugin. You can find it on the addons site. It syncs all your thunderbird calendars (GCal or otherwise) into evolution, so that gnome can pick them up.

So just set up the Google calendar in thunderbird, and then install this plugin.
