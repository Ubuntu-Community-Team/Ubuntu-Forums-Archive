I had this problem during install.  I went back to the partition section, (I am using multiple partitions on my PC (2 windows, 1 linux, 1 NTFS data drive).  I deleted all old linux partitions, had the installer auto-partition the new blank space, and the resumed from that point.  It worked flawlessly after that.
