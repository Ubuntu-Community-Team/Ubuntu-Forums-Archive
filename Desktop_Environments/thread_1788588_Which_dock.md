### Post by cbowman57 on 2011-06-22
I'd go with Docky if I were trying to emulate Unity, but I always preferred AWN to Cairo-dock, which makes me biased.

I think Docky is lighter/quicker and has lower system requirements.  You could also use xfce4-panel, it's really flexible. In fact, you might be better off using xfce or xubuntu for a base distro.
