### Post by fishexe on 2010-04-21
I saw this in another thread ([http://ubuntuforums.org/showthread.php?t=921425&highlight=xournal+xinput](http://ubuntuforums.org/showthread.php?t=921425&highlight=xournal+xinput)) but it was long and contained a lot of contradictory information about upgrading to the newest version and various other things which don't solve the problem.  The problem is that Xournal has trouble recognizing touchpads by default.  When I plug in an external mouse, it works until I click the touchpad and then stops working.

The key is to go to "Options" in the menu bar and uncheck "Use Xinput".  I don't know the purpose of Xinput but I think it is something you will need to check on if you use a tablet or some sort of stylus.  For a normal laptop, however, Xournal seems to only work if Xinput is off.
