So i searched the forum, and found a lot of topics, where people ask for keyloggers in linux.
So i found one. 
So how to get this working:
:arrow: First get this [Here]("http://prdownloads.sourceforge.net/lkl/lkl-0.1.1.tar.gz?download")

:arrow: Save it on your desktop or wherever.
:arrow: Install * build-essential * package. You can use synaptic.
:arrow: First extract everything from the tar.gz you just downloaded.
I did everything in lkl directory. so now its like home/datelus/Desktop/lkl
-> Now using Terminal cd to your lkl directory
:arrow: Type * ./configure *
:arrow: Now *make *to compile it 
:arrow: Now type* sudo make install*
Everything seems to be done.
now you can remove temporary files * make clean*

TO RUN LKL

:arrow: type simpely *lkl* in terminal


Now everything seems easy- follow the instructions
