### Post by larry on 2006-07-27
Dear All,

My requests could sound a bit strange: I mainly use (X)Ubuntu for research, but I am not a linux Guru at all.
I understand the need for stability which drives Ubuntu and I know that some software I need (e.g. R, Maxima, FFTW, Lapack, Blitz++ and Blas) is useful only for a tiny minority, but in some cases I'd really like to see them updated more often.
E.g. R version 2.3 contains some fixes wrt version 2.2 available on Ubuntu and some packages would not work with 2.2.
Beware: I am never talking about core applications: a bug in R will never break your system, so it would not be risky to have them updated a bit more often.
Downloading and installing them from the net can be a complicated business and could lead (in my experience) to problems when you dist-upgrade the system for a new release.
Can the situation of scientific software be improved somehow?
BTW, I love Ubuntu in particular for the excellent tools it gives me for my research; just I would like to see them a bit more up-to-date, as it happens e.g. with Fedora.
Cheers

Larry
