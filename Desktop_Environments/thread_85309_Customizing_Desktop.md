### Post by phanboy_iv on 2005-11-02
Find some GDM themes from the excellent sites listed above, and then  

->System->Administration->Login Screen Setup.

and select the Themed Greeter tab. There's an "Add New theme" button here. Just navigate to the donloaded tar.gz or whatever and install it. Then you should be able to select it from the Themed Greeter tab.
