### Post by monkeybrain20122 on 2015-11-26
I don't know about Krusader, but the general procedure is probably the same: install the replacement file manager and makes it the default. Here are the instructions for replacing Nautilus with nemo.[http://www.webupd8.org/2013/10/install-nemo-with-unity-patches-and.html](http://www.webupd8.org/2013/10/install-nemo-with-unity-patches-and.html)

 The only part that should concern you is the section Optional: set nemo as the default. Now bear in mind that some functions may not work well if Krusader doesn't play well with the rest of the DE, as you see from the article some work arounds are required to use nemo as an alternative file manager, but those work arounds would obviously specific to what you want to replace Nautilus with. Since nemo is a nautilus fork it is probably simpler than if you know something totally different.

BTW, nemo also has twin  panel if you mean a side panel.
