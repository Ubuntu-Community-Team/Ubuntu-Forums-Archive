### Post by TheBlackNoodle on 2006-07-24
If you're using GNOME, you can use Desktop->Preferences->Screen Resolution. But if you can't find that, you can start a terminal and run:

sudo gnome-display-properties

I don't know anything about KDE or XFCE, though, so you should specify what desktop you have.

Another possibility... Map a shortcut key to "Move Window" and hit that shortcut. Your mouse should end up in the middle of the window, and you'll be able to move it much farther up than if you grab it on top.
