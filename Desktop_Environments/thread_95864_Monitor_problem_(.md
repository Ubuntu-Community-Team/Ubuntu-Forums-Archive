### Post by thespazzz on 2005-12-05
I had this same problum.

Try this... After running sudo dpkg-reconfigure xserver-xorg and setting it to your desired size restart the computer.  This should give you GDM in the correct resolution.  Start a terminal session from that screen and run "gconf-editor"

Open /desktop/gnome/screen/default/0

You should have two options Resolution and Rate, Change these to the desired setting then restart X and login through gnome.
