### Post by mariner09 on 2008-04-09
There isn't a properties option like SLED's implementation of Gnome-Main-Menu.

In /usr/share/gnome-main-menu, there's a collection of xml files which dictate the layout and content.

Also, gconf-editor will allow you to do some minor tinkering with layout.
