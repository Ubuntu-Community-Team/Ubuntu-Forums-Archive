### Post by jackalparty on 2008-05-11
Well, I just hit up the irc and fixed my own problem. I met another guy who encountered the same problem so here is the solution for anyone who hits the problem.

Your keyboard is set with dead keys, probably when you pressed the keys to autoset your keyboard during installation you hit no on the " key.

Go to
System - Preferences - Keyboard

then  go to the layouts tab

press add layout

Layouts = USA

Variants = Default (which should pop up auto when you select USA)
