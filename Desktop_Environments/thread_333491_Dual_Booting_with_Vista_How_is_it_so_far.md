### Post by Cosu on 2007-01-09
I have a quad boot machine with ubuntu vista xp and opensolaris and i'm switching between the first and the last all the time. 
My machine is a few years old ( AMD sempron 2200+ on a Nforce2 motherboard, 1.2GB of RAM and a nvidia 6600 video card and both ubuntu and vista run smoothly (running Beryl and gnome). The Aero interface is cool and it does not hog up on resources. The computer is still usable. The biggest problem so far with vista is the lack of drivers. I saw it running on two laptops( a Toshiba Tecra M3 with centrino and a FSC with core duo) and on both of them something didn't work: wireless card, bluetooth or even sound card. Surprisingly on my old pc everyting ran smoothly. 
Other than it looks quite nice vista has no advantages over windows XP **yet**.  Stick with XP if you're playing games or running windows only applications. Maybe in a few months the hardware manufactureres will offer (more/better) drivers for it. 
I wouldn't worry to see a vista only sticker on some hardware. No sane hardware vendor would do that.

### Post by shrimphead on 2007-01-11
I've been dual booting Vista and Ubuntu at home for a while now and whilst they do play fairly nice together (Grub can boot vista no problem). I would seriously advice people against running Vista. whether you intend to buy it or not

Just for a little bit of background: I am a linux user at home, I have been using ubuntu since 5.04 and slackware before that. 

I am employed, however, as a C# developer in a microsoft based software company. I am being 'upgraded' to vista shortly on my development machine at work so I thought I'd take the time to try it out at home (legally, as work have provided me with an MSDN subscription).

IMO Aero is not as functional as XGL/Compiz and serves more to hinder my work than make my desktop look nice. The transparency effects simply serve to 'muddy' the desktop and the windows behind it instead of providing any reasonable viewing of the lower windows. The new interface on explorer's file manager is confusing and too crowded. 

The file system is also confusing. As an example, there are 2 folders where your documents are stored C:\Users\ (vistas answer to /home) and C:\ Documents and Settings (to maintain backwards compatibility with XP programs.

These are just a couple of the points that I found with Vista, there are many more... Ironically the thing that bothered me the least is the UAC dialogs that prompt you to click OK whenever an application accesses the system, as being used to Ubuntu and sudo I am used to this; however what is disturbing is just *how many* applications need to call the UAC just to be useable.

Over all I felt that Vista is far too slow and resource heavy to be much of an upgrade over XP (let alone Ubuntu!)  and I will definitely be sticking to ubuntu on my home machine :)

just my 2pence :)
