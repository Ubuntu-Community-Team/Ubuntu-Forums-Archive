### Post by jr2 on 2009-06-19
Hey,

I have an HP dv4 series laptop.. I can't customize the keyboard shortcuts to use Super (aka "Windoze") + L .. when I hold down the Super key, it immediately defaults to "SuperL" (IDK why the "L", maybe "left"?, but I only have one on this laptop) and won't let me add the actual "L" key.  Now, the show desktop function *works* fine, it is shown as "Mod4+D" in the list.  If the Super key is "Mod4", why is it showing as "SuperL" and acting like I pressed and released it?  IDK how to fix this, any help appreciated.

> **mcubed said:**
> I believe you have to enable the "Super" (aka Windows) key before you can use it as a modifier.

Menu:  System -> Preferences -> Keyboard

This brings up a "Keyboard Preferences" box with numerous tabs, one of which is "Layouts."  Go to that tab, and click on "Layout Options," and you'll see an item called "Alt/Win key behavior."  Under that, you can set "Super is mapped to Win key."  That will let you use the Super key as a modifier.  Or it should anyway.

### Post by mcubed on 2009-06-19
I believe you have to enable the "Super" (aka Windows) key before you can use it as a modifier.

Menu:  System -> Preferences -> Keyboard

This brings up a "Keyboard Preferences" box with numerous tabs, one of which is "Layouts."  Go to that tab, and click on "Layout Options," and you'll see an item called "Alt/Win key behavior."  Under that, you can set "Super is mapped to Win key."  That will let you use the Super key as a modifier.  Or it should anyway.
