### Post by martinpi on 2006-10-17
Using the memory card via USB shouldn't require any special drivers. 
I have a Nokia 6230i and a DKU-2 cable. 
With Breezy it is a breeze, i.e. I connect the phone, confirm to use the phone as an extenal memory by pressing OK on the phone, and a drive icon appears on the Gnome desktop. The path is /media/mmc. 

When I tried with Dapper, I think that it worked once out of many tries, but I am not sure. When I do as described above, I hear that there is access to the harddisk, but nothing else happens. 

When I connect any other usb storage device (memory stick, mp3-player), it works, both with Breezy and Dapper.  

I have not yet investigated any further.
