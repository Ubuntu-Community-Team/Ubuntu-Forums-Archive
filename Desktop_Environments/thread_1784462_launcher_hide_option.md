### Post by steinerd on 2011-06-17
Try this utility.  It's written in python but I have used it and it really takes all of the configurable features of unity and drops them into one program/area.

[http://deep-sea.sytes.net/download/linux_files/confity_1.4.1.tar.gz](http://deep-sea.sytes.net/download/linux_files/confity_1.4.1.tar.gz)

For more information search for confity.

Or you can try the following tutorial for using the compiz ccsm in order to more directly manage your changes.  Just be careful inside of ccsm.  Many settings will corrupt your unity desktop and cause it not to launch.  This is why I suggested confity. If you stick with just editing the unity plug-in you ought ot be pretty safe though.

[http://deep-sea.sytes.net/ubuntu/unity-launcher/how-to-configure-unity/](http://deep-sea.sytes.net/ubuntu/unity-launcher/how-to-configure-unity/)

Hope I was able to help.
