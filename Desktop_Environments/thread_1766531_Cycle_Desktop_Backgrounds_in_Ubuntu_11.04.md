### Post by hexxd on 2011-05-24
I just updated to 11.04, and noticed in the "Background" tab of the appearance app, that there was what looked like a default option to cycle a set of backgrounds. However, when I selected it, the only background that appeared was the first one in the series, and it never cycled. 

1) Is cycling backgrounds possible?

2) If so, can I create my own series of them?

---

### Post by Larkspur on 2011-05-24
> **hexxd said:**
> 1) Is cycling backgrounds possible?

2) If so, can I create my own series of them?


1) Yep.  The slideshow you selected ("Contest") does cycle, but it changes every half-hour.  It's the same with the other other ("Cosmos")

2) Yep.  If you want to make one yourself, you can write one in gedit.  Use the xml file in the slideshow folder (e.g. /usr/share/backgrounds/Contest).  An easier way would be to download DesktopNova from the Software Centre.  That one allows you to change the order and the time between switches.
