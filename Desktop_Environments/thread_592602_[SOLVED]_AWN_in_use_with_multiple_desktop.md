Hi!

I have one question about Avant Window Navigator (Great product by the way)

When I use window-list on the gnome panel, it only shows what is on the current
desktop. When I use AWN (wich has now replaced the gnome window list), it shows
all window icons for every desktop. 
To me, that rips the whole point of virtual desktops appart.

Is there any workaround for this?

---

### Post by Vansinnesvisan on 2007-10-26
In AWN Manager, under the General tab, there is a setting for "Show windows from all viewports" you can uncheck.
