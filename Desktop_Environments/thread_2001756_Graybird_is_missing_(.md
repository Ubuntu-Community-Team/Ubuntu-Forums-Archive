### Post by LewisTM on 2012-06-11
The easiest for you would be to install Ubuntu Tweak:
```
sudo add-apt-repository ppa:tualatrix/ppa
sudo apt-get update
sudo apt-get install ubuntu-tweak
```
Then run it and go to Tweaks/Theme
Pick Greybird in the Window Theme

Cheers!
