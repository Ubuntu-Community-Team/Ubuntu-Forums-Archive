### Post by warp99 on 2008-04-12
You can already do that with Window Matching and setting the viewport I believe.

[http://wiki.compiz-fusion.org/WindowMatching](http://wiki.compiz-fusion.org/WindowMatching)

Edit:

You can use the 'place' plugin and set an application to a fixed viewport with a window match:

[http://wiki.compiz-fusion.org/Plugins/Place](http://wiki.compiz-fusion.org/Plugins/Place)
