### Post by SoftwareExplorer on 2009-09-19
Or, the GUI way:
Go to Places-> Home folder. Hit View->Show Hidden Files. Delete the following folders .gnome. .gnome2. .gconf, .gconfd, .metacity. This will reset gnome settings.  Hit View->Show Hidden Files. (Unchecking it this time, so you don't have to always see the hidden files.)

(This is assuming the Places menu is still there, if not, tell me and I'll help you around that. )
