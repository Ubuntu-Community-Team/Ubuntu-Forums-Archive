### Post by steveneddy on 2009-09-17
OK - **what theme engine is it asking for?**

Murrine, aurora, nodoka and magic chicken are four of the top engines.

Open Synaptic

System --> Administration --> Synaptic Package Manager

click in the list and start typing

**gtk2**

in the list. You should see the gtk2 engines.

You also might need to install some engines manually.

Let us know if you need any help.
