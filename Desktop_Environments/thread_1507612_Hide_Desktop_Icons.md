### Post by gingivere0 on 2010-06-11
In order to put the desktop icons in the panel, I would suggest doing what llawwehttam suggested to do, with a pop-out tray. It is very easy to hide the desktop icons from showing up on your desktop. Install Ubuntu-tweak by doing the following inside a terminal:

```
sudo apt-add-repository ppa:tualatrix/ppa
sudo apt-get update
sudo apt-get install ubuntu-tweak
```

Then go to Applications and System Tools and click on Ubuntu Tweak. On the left side look for **Desktop** and Desktop Icons below that. Then un-check show Desktop icons.  

Good Luck!
