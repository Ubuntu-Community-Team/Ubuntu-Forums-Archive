### Post by Edward The Bonobo on 2006-03-14
Those replies miss out some other bits that you'll need to know about.

First of all...you're going to have to configure Ubuntu to handle mp3 format.  Its a proprietary format, so it's not done as default.  A more useful answer would have been "See here: [https://wiki.ubuntu.com/CDRipping](https://wiki.ubuntu.com/CDRipping) "  

See also [https://wiki.ubuntu.com/RestrictedFormats](https://wiki.ubuntu.com/RestrictedFormats) for how to handle other formats.

There's a lot you have to learn when starting out in Ubuntu.  Some people like to give you a crumb of information to encourage you to start searching.  A kinder way is to direct you to the User Documentation wiki: [https://wiki.ubuntu.com/UserDocumentation](https://wiki.ubuntu.com/UserDocumentation)
