### Post by debian_n00b on 2005-06-07
New Ubuntu user here. Colleague from work threw a cd at me and told me to try this. I've tried and used Debian off and on for a couple of years now.

First impression is polished. The default desktop is rather pleasant to look at. And snappy too. All my basic hardware(sound, video, network, etc...) appears to work as well. There also appears to be a nice compliment of software here as well. Nicely chosen.

I was never asked for the root user account during install. Strange. Should this happen? I managed to issue passwd using sudo and now have access to root. My fat32 partition was also not reccognized. And not knowing how to address partitions on the dev filesystem, I was left floundering a bit on that one. Doing DF hinted at the naming scheme, so I was able to eventually mount the win partition rather quickly and painlessly. 

Otherwise, very impressed.

I have noticed that Ubuntu has it's own repositories. Uncommenting the suggested "universe" repository did not fill a gap I have for required packages. Do the users here add sources from pure debian? What about non-Ubuntu/Debian repositories? 

I found the unofficial Ubuntu guide by searching the forum. Guess I'll have a browse through there to meet any other questions I have before posting again. Seems like a nice and friendly place here.

### Post by rjs on 2005-06-07
By default ubuntu doesnt use a root account, instead you are meant to sudo everything. apparently this is better in terms of security and what not.

there are more repositories hosted here at ubuntuforums which are for less free stuff (like the ACC codec and the like). Look up backports or restricted formats on the wiki for how to add them.
