games:
1. connect slvr to compy via usb
2. toss .jar files into kjava folder
3. unmount phone
4. go to [on phone] My Stuff >> Games & Apps
5. press top middle button [the one directly above the circular dealio]
6. select Switch Storage Device
7. select card [phone comes with 512mbyte card btw :)]
8. select install
9. browse to the .jar u wanna install and select that
10. install [personally i install to the card..]
enjoy :)
and of course im claiming all my games are homebrew >_>

ringtones:
1. connect slvr to compy via usb
2. open local folder with ringtones [mine are mp3 96kbs]
3. open remote ringtones folder OR audio folder
4. copy paste from local to remote folder 
5. unmount phone
6. go to [on phone] My Stuff >> Sounds
7. press top middle button [the one directly above the circular dealio]
8. select Switch Storage Device
9. select card
10. enjoy :D
