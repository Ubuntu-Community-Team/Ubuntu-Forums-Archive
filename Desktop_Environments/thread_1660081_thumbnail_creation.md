### Post by Krytarik on 2011-01-05
Try this:

in file browser goto "Edit -> Preferences -> Preview" and set "Show thumbnails" for "Other Previewable Files" to "Never".

I don't know if that affects all thumbnails stored in the .thumbnails-directory, in each users profile-dir (/home/USER_NAME) is also one.
