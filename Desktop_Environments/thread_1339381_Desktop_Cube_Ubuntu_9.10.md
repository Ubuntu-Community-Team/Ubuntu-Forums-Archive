### Post by practic on 2009-11-27
It would help if you told whether you are using Ubuntu Netbook Remix, or just regular Ubuntu 9.10...but the instructions are pretty similar. I'll keep them as simple as possible:

1. Open the **Ubuntu Software Center** (usually at the bottom of the Applications menu).

2. Type:

**compiz**

...in the Search box.

3. Make sure three of the programs that show up on the search list are installed:

[B]Compiz
Advanced Desktop Effects Settings (ccsm)
Simple Compiz Config Settings Manager
[/B]
4. Once this is done, close the Ubuntu Software Center, and go to the **System->Preferences** menu and choose **Appearance**. The Appearance dialog will open.

5. Select the last tab on the Appearance Preferences dialog, which is called **Visual Effects**.

6. Select the **Custom** radio button. If you didn't have effects turned on, you might have to wait a bit for the graphics driver to initialize.

7. Now click the **Preferences** button to the right of the Custom radio button. That will load the **Simple CompizConfig Settings Manager**, which has some of the basic and most popular Compiz settings.

8. On the Simple CompizConfig Settings Manager click on the **Desktop** tab, change the Appearance to **Desktop Cube**, and make sure your Desktop columns are set to 4, and the Desktop rows to 1.

9. Now hold down Ctrl and Alt and either click and drag the left mouse, or use the right or left arrow keys. You should see the cube effect.

Set whatever you like in the rest of the Simple CompizConfig Settings Manager, and then close it, and close the Appearances dialog.

You can also get into a huge array of advanced settings. From the System->Preferences menu, choose **CompizConfig Settings Manager**. There you can adjust the speed of the cube, the zoom effect while rotating, 3D windows while rotating, put a background pic behind the cube, adjust transparency of the cube while rotating, etc., etc. It's almost endless!
