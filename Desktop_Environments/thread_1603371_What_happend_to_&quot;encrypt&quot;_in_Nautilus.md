### Post by s_raiguel on 2010-10-22
With earlier versions of Ubuntnu, if Seahorse and GnuPG were installed, the right-click menu of Nautilus would offer to encrypt a file based on a password created in the Seahorse password manager. With 10.10, encryption no longer appears on this menu. Is there any way to get it there?

### Post by s_raiguel on 2010-10-23
Oops! There it is! Apparently, it was necessary to reboot after creating a password before "encrypt" appears on the Nautilus menu. Problem solved.
