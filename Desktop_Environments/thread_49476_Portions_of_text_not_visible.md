### Post by javiadip on 2005-07-16
check if you have java script enabled in your firefox browser and you will also need to install flashplayer-mozilla and jre. 

here's the link on to how to install jre [http://wiki.ubuntu.com/Java](http://wiki.ubuntu.com/Java)

to install mozilla flashplayer plugin, 

add these two lines in to your sources.list, sources.list is located under /etc/apt/

```
deb http://ubuntu-backports.mirrormax.net/ hoary-backports main universe multiverse restricted
deb http://ubuntu-backports.mirrormax.net/ hoary-extras main universe multiverse restricted

```

after adding these lines in sources.list, type sudo apt-get update

and then type sudo apt-get install flashplayer-mozilla
