### Post by krazyd on 2009-09-11
```
sudo apt-get install mplayerthumbs
```

then in dolphin go to Settings > Configure Dolphin > General > Previews tab and enable the "Video files" option.

edit: to configure, run ```
mplayerthumbsconfig
```
