### Post by mcduck on 2007-08-28
Yes, there is, and it's even installed by default. Just select the files/directories you wan to put into the archive, right-click and select 'Create Archive". You can then choose what type of archive you want to make.

To add rar support install packages 'rar' and 'unrar'.

If you want to enable the menu entry for Archive Manager right-click on the applications-menu, select "Edit Menus", browse to "Accessories" and enable "Archive Manager".
