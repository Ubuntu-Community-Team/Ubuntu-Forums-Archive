### Post by benmoreassynt on 2006-03-08
I've tried a few which I did not like much, and am now using mail-notification. It provides a window when new mail arrives, adds a "system tray" in icon, and allows you to open Evolution, or whatever by double-clicking it.

It would make sense for Evolution to include a system tray icon by default, as anybody making the transition from Windows will miss this ubiquitous Windows feature. It's the sort of little thing which makes Linux difficult to get used to.
