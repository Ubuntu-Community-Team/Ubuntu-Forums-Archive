> so how do i find the XXX part
The XXX part is which ever kernel you are using.  For example mine is 2.6.15-27-686.  You can see which version it is from the GRUB menu when you first turn the computer on.  If you have not changed your kernel you could get linux-restricted-modules-386 and that should get the correct one for you.
```
sudo apt-get install linux-restricted-modules-386
```
If you have changed your kernel replace the 386 part with which ever one you are using.
