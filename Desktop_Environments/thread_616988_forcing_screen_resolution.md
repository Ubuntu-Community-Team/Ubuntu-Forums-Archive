### Post by TeaSwigger on 2007-11-20
Well in that xorg.conf it's not specifying your video devices at all, not the nvidia card and neither monitor. 

Easiest check first, I have to ask; have you tried running 'sudo dpkg-reconfigure -phigh xserver-xorg'?

In case you've no idea what I mean, here's the deal. The command regenerates a configuration (the xorg.conf). Make a backup of this apparantly wrong but working configuration file like so: enter this in a terminal:

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```

This makes a safety copy. Mkay. Then enter this into a terminal:

```
sudo dpkg-reconfigure -phigh xserver-xorg
```

When it's done, reboot. If it does the trick, great! If not, we'll get back to it. If you can't boot up, try booting using the "recovery mode." It'll scroll umpteen miles of stuff and stop at a command prompt. You can then put back the previous version so you at least have a picture, like so:

```
mv /etc/X11/xorg.conf /etc/X11/xorg.conf.wrong
```

making a copy of the non-working config if you want to look at it, then you put back the old version:

```
mv /etc/X11/xorg.conf.backup /etc/X11/xorg.conf
```

and enter 'reboot' and pick the regular boot. And come on back.
