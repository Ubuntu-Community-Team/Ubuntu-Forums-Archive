### Post by Toxicity999 on 2006-09-07
Nothing by default I mapped it to a script I use to reset my LCD Display as it's starting to crap out... (inverter) if you run into gconf (Applications>>System tools>>Configuration Editor) it's under Apps>>metacity global_keybindings lists what Key combination runs what command listed on keybinding_commands So I have Super_L listed as the key for Command one, and one keybinding_commands I have command_1 set to lcdreset. This is all just for bin commands and such theres a graphical editor for graphical keybindings such as playing with windows, manus, etc.

Edit: Also, in most Linux community environments we refer to it as the 'Super Key' Personally I think it would be nice and ironic to script the super key to spawn a full screen Blue Error prompt on someones Linux system ;) Faked of course, But fun none the less.
