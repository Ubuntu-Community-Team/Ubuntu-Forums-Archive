Now I have the normal Wine apt repo and i've just installed wine, that's it, is their anything else i should know or install to make this better?Don't use Wine to run UT. There's an install script from the sadly deceased Loki games that will make UT run natively. Much better, and seems to perform better than under Windows, at least once you've got your graphics drivers working. You can either try one of the links from here [HTML]http://www.lokigames.com/products/ut/updates.php3[/HTML] or search for ut-install-436.run or ut-install-436-GOTY.run, depending on which version you have.

As to making games run better under Wine, **winecfg** is the command that you need to set up your preferences. And this [HTML]http://appdb.winehq.org/[/HTML] is the place to go to see if the applications you're trying to run are actually known to work with Wine.
