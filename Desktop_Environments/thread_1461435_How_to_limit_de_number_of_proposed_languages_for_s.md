On my Ubuntu system, I have three dictionaries installed: English, French and Spanish.

### Post by Hagar Delest on 2010-04-24
Not possible AFAIK. You can remove the blue checkmark (with OOo) if you want to remove for example the Belgian, Swiss, ... flavors of the French dictionary but you can't remove all the unused entries of the lists.

Note that the list behaves as you want in the spell checking dialog (F7). So you can ask for a similar behavior in all the other lists also: [[Tutorial] Reporting bugs or suggestions](http://user.services.openoffice.org/en/forum/viewtopic.php?f=6&t=13490).
