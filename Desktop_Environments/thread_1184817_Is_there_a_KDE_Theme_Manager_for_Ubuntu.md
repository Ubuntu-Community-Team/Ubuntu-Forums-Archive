### Post by Chronon on 2009-06-11
From [https://help.ubuntu.com/community/CustomizeKubuntu](https://help.ubuntu.com/community/CustomizeKubuntu)
> You can reach the Theme Manager by running KControl, KDE's Control Center. Click on the K Menu and select Run Command (alternatively, you could also press Alt+F2). In the Run Command dialog box, enter the text kcontrol and click on the Run button or press the Enter key. The Theme Manager is under the Appearance group of the Control Center.
