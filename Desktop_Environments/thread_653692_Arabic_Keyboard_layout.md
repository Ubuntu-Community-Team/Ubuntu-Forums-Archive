Did you try to choose a different Arabic layout for your keyboard from *System* > *Preferences* > *Keyboard*, *Layouts* tab, remove the current Arabic Layouts then add *Arabic* *Default* layout, it should solve your problems.

> **minacross said:**
> 
BTW: how can I upgrade OO 2.3 to 2.3.1?
Currently only 2.3 is available from the official repositories, however you can download the stand-alone openoffice.org from its official website, unpack it and use it.
