### Post by psychicdragon on 2005-10-30
Xfce has no desktop icons by default. If you want icons on your desktop you need to run **nautilus** or **rox --pinboard=desktop**.

### Post by aysiu on 2005-10-30
[QUOTE=eyebrowman92]ok is there a way i can get nautilus to start automatically on xfce startup?[/QUOTE] Yes. Alt-F2 (for the "Run" dialogue), then run this command ```
nautilus
``` When you log out of XFCE, choose to save the session for future logins.
