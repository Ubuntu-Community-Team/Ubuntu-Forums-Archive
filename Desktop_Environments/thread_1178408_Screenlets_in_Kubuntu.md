I don't think so. AFAIK the kde plasmoid API and gnome screenlet API are not compatible. There are heaps of plasmoids on kde-look though - what screenlet from gnome are you missing?

Edit: forgot to mention that you can run OSX widgets and google desktop widgets as plasmoids on KDE if you can't find a native plasmoid.
