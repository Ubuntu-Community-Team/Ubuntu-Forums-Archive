I need to apolagize for giving you some wrong info!

If you are trying to install a GDM theme then you will need to go to System>Administration>Login Window

After opening that up, you then click on 'Add' and point to where you stored your tar.gz

You were apparently trying to install GDM themes where you install GTK themes.  Gtk themes are what you want to get if you're interested in changing the look of your windows. They would be installed in your System>Preferences>Theme

GDM Themes are your Login Window eye candy!  System>Administration>Login Window
