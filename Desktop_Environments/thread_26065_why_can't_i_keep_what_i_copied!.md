### Post by kassetra on 2005-04-11
Ok, here's the key point you're both missing:

The copy-paste clipboard is with the program you're copying it from, so if you CLOSE firefox before pasting, the clipboard disappears.

Also in some instances, the clipboard also has issues, so I suggest using one of the original Linux methods as well when wanting to copy-paste: You hightlight the text, and then use your middle mouse button (or left+right buttons at the same time) to paste the highlighted text.
