### Post by LewisTM on 2012-10-01
Thunar will not remember your previous folder view. I think it's intentional; I for one agree to this choice. When I browse my file details, I hate to see them disappear when I jump folders. When I'm looking for a specific thumbnail, I don't want the file manager to choose not to show them in the next folder. What Thunar lets you do is quickly switch between views. Learn these hotkeys and you will find how efficient you can become:

Ctrl-1 : Icons
Ctrl-2 : Details
Ctrl-3 : Compact

However, there is not hotkey for sort order. I would LOVE to see that function implemented.

Cheers!
