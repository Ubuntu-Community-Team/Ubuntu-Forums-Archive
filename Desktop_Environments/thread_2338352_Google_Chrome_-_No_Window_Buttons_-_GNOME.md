Hello,

Not seeing this on my installation so, first make sure the gtk theme is compatible with your Gnome version. If using 16.04 you would get the best results with gtk 3.18 themes. Keep in mind if using 3rd party themes this may be feature of the theme and in that case right clicking the title bar would be used to control the min , max, and close options.
