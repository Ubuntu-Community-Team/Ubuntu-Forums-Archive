### Post by Nosphky on 2014-06-14
I'm using UbuntuStudio 14.04.

When I open a new window, File Manager -Home for example, the new window appears top and hard left on the screen and this obscures the icons in their standard position (vertically arranged on left side of screen).

I have been trying to find a setting to make them open hard right.  The nearest I can get is in Settings Manager / Window Manager Tweaks/Placement.  If I set the 'Minimum size of windows to trigger smart placement'  well over towards the large side of the settings, the new windows open in the center of the screen.   This has apparently prevented smart placement, whatever that is, from working.

I realize that an easy way to move a window  is with Alt+left click.   But is there a setting to make a new window open hard right ?

---

### Post by Toz on 2014-06-14
If you go to Settings Manager >> Workspaces >> Margins and set the left margin to its highest value, then most windows will open hard right. I'm not sure why some don't, perhaps they have an internal settings that specifies the window start location.

### Post by Nosphky on 2014-06-15
Thanks Toz, that seems to do the trick.
