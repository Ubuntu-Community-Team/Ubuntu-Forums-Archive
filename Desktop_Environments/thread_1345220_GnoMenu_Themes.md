### Post by oakgrove on 2010-01-30
The manual way: You have to sudo cp your theme to /usr/share/gnomenu/Themes.  Go into that directory and look around.  It should be apparent what to do.

Or you can just click install in the preferences menu to load new themes the automatic way.
