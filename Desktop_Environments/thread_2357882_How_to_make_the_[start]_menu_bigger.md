### Post by ajgreeny on 2017-04-07
I did once add the xfce4-whiskermenu-plugin to an installation of Lubuntu so as to get the whisker menu from xfce, which is a great menu GUI in my opinion.
The screenshot is of that OS.  With a bit of fiddly panel configuration it is possible to leave the Lubuntu panel in most of the panel space, ie, about 95% wide and to the right, but also have the xfce4 panel running in the remaining 5% on the left hand end, with only the whisker menu applet added.

I did not keep that OS so neither did I keep that strange way of running two panels, side by side, but it proves it can be done.
Alternatively you cpould just add the xfce4-panel package and use that with the whisker menu apple added, in place of the Lubuntu panel.
