### Post by inso_741 on 2010-11-04
Is there a way to disable gnome panel but retain the launcher(Alt+F2)?....or an alternative for the purpose?

Edit: I'd also like to retain the key bindings

---

### Post by kerry_s on 2010-11-04
you just need to use an alternative run command.
i use grun.

install grun
then disable the current run
add the new one.

i set mine to alt+2, so i don't have to use the fn key, but you can do alt+f2.

then you can go into gconf-editor & change "gnome-panel" to your new dock name, like "avant-window-navigator" or "cairo-dock".
