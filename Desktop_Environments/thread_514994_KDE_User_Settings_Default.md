### Post by Greslore on 2007-08-01
If they are local users, then you can go ahead and use Kiosk Admin Tool.  It allows you to completely customize menus for users, even lock out the ability to change the desktop and such.  Very powerful.  It's all GUI driven, so its intuitive.  

But, if this is only for four users, you may just be able to get away with copying your $HOME/.kde directory into their home folders.  

As for the photo sharing, I'd just create a directory someplace and put the pics there.
