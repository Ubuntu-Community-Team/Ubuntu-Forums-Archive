### Post by blackbelt_jones on 2008-11-04
Dolphin is the default file manager in later versions of KDE3?  Are you sure?  I'm running KDE 3.5.10, and I can't even tell you for sure if Dolphin is installed on my system.

What you guys are experiencing is the heartbreak of KDE4, which is still under construction.  Dolphin is indeed the default file manager of KDE4, and that makes fixing the file manager features in Konqueror not a priority.  I have it on good authority that Konqueror will be back to its old self eventually.

For me, there's just no contest.  Konqueror is the number one reason why I use KDE, and it's now the number one reason why I refuse to migrate from KDE3 to KDE4, but I'm a specialized user, with specialized needs.  It's a long story.

For now, if you want to run Konqueror at full strength, that means using KDE3.  There is no official support for KDE3 in Intrepid, but community repositories are already being established by unhappy KDE3 fans who want to keep their systems current, and want to make sure that KDE3 continues to be viable in the future.  

Here's the repository that I used to install KDE3 on my Intrepid Ibex system.

[http://apt.pearsoncomputing.net/](http://apt.pearsoncomputing.net/)

KDE3 continues to be supported and will be regulary updated, but its in "maintanance mode".  There aren't going to be a lot of new features, but as long as there's a user base, KDE 3 will continue to get security updates and bugfixes, and right now, it's believed to be the choice of a majority of KDE users.  For lots of people, the decision by Ubuntu to discontinue support for KDE3 was a shock.  Personally, I was appalled... and, for just a moment,  livid.  However, in typical free software fashion, the community has moved quickly to take up the slack.  KDE3 repositories for Intrepid began to appear within a couple of days of the release.

If anyone tells you that you can't run KDE3 on *buntu Interpid Ipex, I don't care if it's Linus Torvalds himself.  That person is wrong. And I've got the system to prove it.

And if anyone tells you that there's no expectation that KDE3 will continue to be maintained, that is also not true.  I've seen this particular falsehood posted on the Kubuntu wiki, and it's completely inaccurate.

Here's a good place to find out about the prospects for KDE3 in the future:

[http://aseigo.blogspot.com/2008/01/talking-bluntly.html](http://aseigo.blogspot.com/2008/01/talking-bluntly.html)

So if you want Konqueror as you have known it, you can have it by using KDE3, and you don't have to revert to Hardy for that.  But since you already have KDE4 installed, you probably ought to give Dolphin a try.  Goldenboy thinks it's superior to Konqueror.  I don't.  See what you think.
