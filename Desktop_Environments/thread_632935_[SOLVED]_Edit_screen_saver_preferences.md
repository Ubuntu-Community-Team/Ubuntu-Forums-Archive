### Post by ray bot on 2007-12-06
The default screensaver preferences app doesn't really let you configure your screensavers at all.  If you want to be able to tweak things, you should install xscreensaver.  Just open a terminal and enter ```
sudo apt-get install xscreensaver
``` to install it.  After that you can run it by typing ```
xscreensaver
```
