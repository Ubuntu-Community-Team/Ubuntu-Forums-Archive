### Post by note32 on 2009-11-04
here it is in english


	
 something similar happened when I installed 9.10 to my Assus UNR 901 The default view does not convince me and so I decided to switch to conventional desktop. My surprise was that in version 9,10 was not possible with a "simple click. What I did was go to synaptic package manager in the edit menu and selected the packages Flag option task, hence I chose the Ubuntu desktop option and gave him to apply. Then from within the package manager installed the packages on desktop UNR (Netbook Launcher, ubuntu-netbook-remix and ubuntu-netbook-remix-default). When the session resumed on 901 and went into a classic gnome desktop environment but ... not show the title bar! To resolve this mishap did what I commented in another message System / Preferences / applications at home and removed the check in windows Maximus management.

I currently have the 901 with the effects of Compiz, in classic gnome environment with full capacity UNR 9.10!

Hope you helps.
