### Post by mc4man on 2014-05-31
Don't think that alt~ & scale are that related, I don't use alt~ though a quick look suggests it orders based on the order windows are opened.
As far as changing current scale behavior, not sure,  though there isn't a setting regarding this.

You could look at the scale-addons plugin (not default installed, part of compiz-plugins) which lets you have a window title overlay in spread or the Scale Window Title Filter plugin (not default installed...). It works by  - when in the spread type a letter(s), only windows that start with that letter(s) will be displayed in the spread, ect.
