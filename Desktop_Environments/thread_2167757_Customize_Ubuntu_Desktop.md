### Post by Y^2om&amp;#7sgP on 2013-08-15
It seems that Fences is not available for Linux

But hopefully this link will assist

[https://groups.google.com/forum/#!topic/alt.comp.freeware/TB0inAOeETk](https://groups.google.com/forum/#!topic/alt.comp.freeware/TB0inAOeETk)

You can add alternate desktops from either Synaptic or the Terminal with the command

sudo apt-get install *desktop*

where *desktop* is the one you want to install.  Try Googling for Linux desktops as there are several varieties.

Once installed, logout and back in, but on the login screen there'll be a dropdown box to select session

You can then start customizing to your tastes :)

I personally user Xubuntu, which has XFCE as it's desktop, fast and light, and can easily create desktop shortcuts in the same manner as windblows.

In that to configure multiple monitors, it's from the main menu, settings, display.

Hope this helps answer some of your questions
