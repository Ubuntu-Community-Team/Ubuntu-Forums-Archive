### Post by amd-64 on 2008-02-12
KDE and GNOME are desktop managers. Each comes with a set of default applications for browsing filesystems, editing text files, viewing pdf files, viewing images, writing emails, etc... They can reside happily together on the system. 

You could boot into one or the other and you can run both simultaneously on two xservers if you wish, although there will be a small increase in system load.

It is more than changing a theme, it is a different environment, because you would be using a different set of programs but the OS ( ie kernel, modules, filesystem, memory, swap) will not change.
