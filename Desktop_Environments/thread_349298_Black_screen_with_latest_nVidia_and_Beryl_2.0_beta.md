### Post by keith11 on 2007-01-31
I don't know why and how but I did post a reply this evening describing how I got rid of beryl. I had tried uninstalling beryl through sudo too but it didn't work. I found out that I had to uninstall the packages called "emerald", the theme manager and "beryl-core". When it asked me to autoremove other related packages I did sudo apt-get autoremove and beryl was completely removed after that. Then I changed back the totem_config file and everything seems fine now.
