### Post by silkstone on 2008-07-28
Choose a file you want to open with Totem (aka Movie Player), right click and select Properties. In the 'Open with' tab, select Movie Player. That change will then apply to all files of the same type.

### Post by silkstone on 2008-07-28
Ah, I understand.

Right click on the *folder* and Open with other application. Then select Movie player.

Then right-click on the folder again, but this time select Preferences. The Open With tab should show 'Movie Player' and 'Open Folder'. Change it back to 'Open Folder'.

Now the Movie Player option should appear when you right-click the folder.

I've just tried this and it worked for me - I didn't have the right-click option either!
