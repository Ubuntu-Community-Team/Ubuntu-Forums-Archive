Hello everybody:

I've a computer with a nVidia GPU, running Ubuntu with the latest nVidia proprietary drivers (these have been taken from the Ubuntu-X PPA). To avoid the tearing problem when watching fast scenes, in the "NVIDIA X Server Settings" panel I've marked every option corresponding to the vertical synchronisation (VSync): one for XVideo, and another one for OpenGL.

After having read several tutorials on the Internet, I've learnt that after doing this, one needs to set up the window manager accordingly. I've tested Compiz and its VSync option and the result has been very positive: there were no more tearing when watching films or when moving windows fast.

However, I want to know if there's a way to get something similar with other compositing window managers like Metacity or Openbox. Or does this feature require access to the hardware acceleration necessarily?
