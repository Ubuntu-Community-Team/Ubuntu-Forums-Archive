 [http://www.jankratochvil.net/project/captive/](http://www.jankratochvil.net/project/captive/)

I've never tried it myself but from all accounts that I've heard, it works flawlessly. Since it uses the Windows NTFS driver instead of the open source one, stability isn't really an issue. That said, I can't use it because I'm using a PPC box and even if I ran an x86 box I still wouldn't use it since it isn't really open sourced. 

But if you really want NTFS read/write support, go with captive-ntfs.
