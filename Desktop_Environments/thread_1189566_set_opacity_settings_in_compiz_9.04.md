### Post by mcduck on 2009-06-26
> **Wa1k3rTXRang3r said:**
> well i figured out how to make windows see through with opacity, brightness, and saturation. but now the problem is that i accidently set the opacity of ccsm to 0. so i can not see the window at all to fix it. 
Help please ?

Press Alt-F2 and run "metacity --replace", change the opacity setting to something more usable, and then hit Alt-F2 again & run "compiz --replace".
