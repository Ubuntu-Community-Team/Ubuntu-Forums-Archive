### Post by hw-tph on 2006-04-01
[Gallery](http://gallery.menalto.com/) is widely used and very powerful. It is a complete PHP based site engine that requires imagemagick or netpbm.

I also use [minimalGallery](http://minimalgallery.net/home) which is a very simple PHP script that uses GD to create thumbnails from the images you upload.
