### Post by haqking on 2011-07-06
backtrack is just a collection of tools from many sources and packaged together under backtrack as a pen test distoro (the current BT being version 5) and is based on Ubuntu Lucid Lynx as a distro.

All the tools can be installed and configured on pretty most Linux distros.

However if you are new to it then the chances of messing things up are high ;)

Just run the BT CD/DVD/USB or Virtual machine.

The reason i say that is if you need to ask such a question then you are probably not that experienced with them in which case stick to the easy methods for using the tools ;-)

There will always be some issue with one tool or another and a certain build, most are CLI based for the most powerful options.

### Post by Dangertux on 2011-07-06
Yeah they all can be made to work, like Haqking said it's easier a lot of times to get them in one spot. Especially considering some of the tools packaged with Back Track are rather obscure. 

Which ones are you wanting? 

Metasploit / SET / Fast Track - are all fairly easy to get working under Ubuntu 
Aircrack-ng - is pretty easy to get set up under Ubuntu
HashCat is easy to set up (unless you want GPU mode in which case you may spend days messing with drivers) 

It all depends on what you want it for. If you just want to play with it, I would do a LiveUSB (so you can save your sessions) , then see if it's something you want to continue with learning.
