### Post by chikko on 2007-08-24
hey everybody,
i just spent about two hours searching for a way to synchronize my korganizer with an online calendar.
i've been using google calendar for quite a long time now, but unfortunately it has no ability of synchronizing (can be fetched into korganizer but it is read-only).
about an hour ago i found a nice solution:  memotoo  ([http://www.memotoo.com]("http://www.memotoo.com"))
it enables you too sync it's online calendar with any organizer which supports a remote file feature (such as korganizer ;-)).
it syncs beautifully.  the only problem is that it only lets 40 calendar records in it's free version.. :(

did anyone find another solution..?  i'd be very happy to any suggestion! :p
