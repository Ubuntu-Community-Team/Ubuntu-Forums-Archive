### Post by treacl on 2010-09-10
I recently upgraded to 10.04, and I notice that when viewing the contents of folders in compact mode, file names are truncated with an ellipsis after about 14 characters, which is kind of inconvenient.

If I recall correctly, in 8.10 (which is what I was using before) compact view would show the whole file name by default. Generally the only time it didn't was when you added a file with a long name to a folder that was already open, but even then you could click reload in the menu bar and it would rejigger the width of the columns so that you could see the whole file name. This doesn't seem to work in 10.04.

Is there any way to specify after how many character the file name should be truncated?

Thanks,
treacl

---

### Post by stinkeye on 2010-09-11
In nautilus > edit > preferences > views
unmark **all columns have the same width**
