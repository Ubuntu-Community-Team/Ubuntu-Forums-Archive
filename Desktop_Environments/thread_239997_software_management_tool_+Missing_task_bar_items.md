### Post by mithras86 on 2006-08-20
Supposing you are using Gnome, the task/startbar you mentioned is called a panel. On the panel you can place several applets. The date&time you see is an applet, the volume icon is an applet, etc.
Also the window list is an applet, and you probably deleted it in an accident. So just richt click on your panel -> add to panel. Choose the window list (or something, I know only the Dutch term) and it will be placed on your panel.

Now you see all the applications you opened. Also something like synaptic or update-manager. And for security reasons, you can only have one 'install,remove&update programs' app running, so just kill those, and you'll be able to run the update-manager again ;)
