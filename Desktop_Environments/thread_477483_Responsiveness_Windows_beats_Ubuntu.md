### Post by wyth on 2007-06-21
> **ernstblaauw said:**
> So, one of the 'tweaks' is: use a good graphics driver (as someone mentioned above).
I use the open source 'radeon' driver for my Radeon 9250. As I've already said:[LIST=1]

[*]KDE

I don't really think swapping out a gfx card is a tweak, especially if you're on a laptop. Getting a new computer ain't a tweak.

Here's what I'm talking about:

In /home/USER/.gtkrc-2.0 you can change the Gnome menu speed.  I don't recall what the original was, but mine now reads ```
gtk-menu-popup-delay = 50
```.

You can achieve the same at the terminal by ```
gtk-menu-popup-delay = 0"| tee -a .gtkrc-2.0
```I set mine to 50 because 0 was a little too quick, and it was hard to get mouse focus on an item.  Play around with it and see what you get.

See [here]("http://ubuntuforums.org/showthread.php?t=479716&highlight=menu+speed") and [here]("http://lifehacker.com/software/linux-tip/speed-up-gnome-menu-269934.php").

Another tweak is to use Swiftfox, which is in the repositories. Find out what processor you're using and pick the Swiftfox for your processor.  I forget the command to show your processor, but if you're on Gnome, you can find it in System-Preferences-Device Manager.  Of course [there's more tweaks in Fire/Swiftfox to make things that much snappier]("http://www.ubuntugeek.com/speed-up-firefox-web-browser.html").
