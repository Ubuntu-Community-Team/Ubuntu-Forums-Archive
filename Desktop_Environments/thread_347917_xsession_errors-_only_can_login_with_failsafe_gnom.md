### Post by solo23 on 2008-02-12
it took me 5 hours but I got it working

used envy to install nvidia drivers (i let it do it automatically)

Added these lines to my xorg.conf

Section "Device"
	Identifier	"nVidia Corporation NV15 [GeForce2 GTS/Pro]"
	Boardname	"nv"
	Busid		"PCI:1:0:0"
	Driver		"nvidia"
	Screen	0
	Option          "NoLogo"        "True"
        Option          "AllowGLXWithComposite" "True"
	Option 		"RenderAccel" "true"
EndSection

Section "Extensions"
	Option		"Composite"	"Enable"
EndSection

Then I installed xserver-xgl ; Ctrl + Alt + Delete 
and it works, I must say its slow.

Forgot to say that when you select the effects it still says it can't load them, but if you wait it loads them. Also when you log in it might say some error with Nautilus or it will glitch as if its reloading the desktop effects (Ctrl+Alt+Del will fix that).
