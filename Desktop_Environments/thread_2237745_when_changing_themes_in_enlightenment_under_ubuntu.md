### Post by Frogs Hair on 2014-08-03
As I remember there is an option select system or personal in the themes application and you also have to make sure the theme is compatible with the E version in use. Some themes may install but not work.

Edit: If you are runnig E18 or 19 on top of a full Ubuntu installation you might want install a theme from the Ubuntu desktop that matches the E theme or the GTK apps will look strange.

In the Elightenmet settings under Look there is/was a theme matching setting. It has been my experience that it is best to use E file manager, wallpaper application and image viewer.
