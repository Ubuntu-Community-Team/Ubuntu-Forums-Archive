### Post by WinRiddance on 2010-07-21
Before you report a bug you should try to verify that this indeed related to Ubuntu by rebooting your computer and playing another game. It could also be a weird system glitch, or some fluke with an app that's running in the background (which you can't see), and so on.

In order to qualify as an actual bug I believe you have to be able to duplicate the problem whenever you want by following the same steps. If others have the same issue that's even better ... as far as corroboration for a found bug goes. Last but not least, it might also be strictly related to the game only whereby the game developers should be notified if this is indeed a bug. I don't think the Ubuntu developers will deal with it if it's not directly related to the Ubuntu operating system. Games are generally third party apps ....

Also, see this article here:
[https://help.ubuntu.com/community/ReportingBugs](https://help.ubuntu.com/community/ReportingBugs)
