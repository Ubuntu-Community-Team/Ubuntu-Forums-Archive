### Post by Sef on 2006-01-20
> What's the difficulty of the install?

Install is easy.  The only tricky part is the partitioning if you haven't done it before.

Install Tutorial:  [http://users.bigpond.net.au/hermanzone/index.htm]("http://users.bigpond.net.au/hermanzone/index.htm")

To install fluxbox from the terminal:  sudo apt-get install fluxbox

However, read there is a problem, so read this before installing:

[https://wiki.ubuntu.com/Fluxbox?highlight=%28fluxbox%29]("https://wiki.ubuntu.com/Fluxbox?highlight=%28fluxbox%29")

Fluxbox is a very lightweight manager, so if you have an older computer, it will run better than Gnome.  It is not as fancy as Gnome, but its job well.   You opposite click to get to the menu.  Minimalism to the max.  Which is appealing to me: I don't like desktops cluttered with icons.  It's not as fancy as Gnome, but what do you want?  If you want a full fledged window environment, then you have to use Gnome or KDE.  

Anyway, at the heart of this is choice.  What ever you decide is right for you will be the right choice.

### Post by clemslacker on 2006-01-20
install is as easy as

apt-get install fluxbox

it will then show up at your login under the session option.

as far as the advantages:

fast and lightweight
leans towards utilizing the mouse, such as the scroll wheel switches between desktops, and it does this extremly fast.  you can even scroll in a different application from the one your are highlighted on.
very customizable
cool looking

disadvantages

because of the customization some people dont like modifying text files to change the look of their desktop.  although gui applications have been written if this bothers you that much.

take it for a spin, you just might get hooked.
