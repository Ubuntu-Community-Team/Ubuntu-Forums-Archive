I don't know how to set up rules to limit the size of windows (if this is even possible), but do you know about the alt trick? Hold the alt key down together with the left mouse/pad button, and you can move the window above the top edge of the screen. The mouse cursor can be anywhere over the active window - it's not limited to the title bar for this. (Obviously!)

Useful for those occasions where the OK/cancel/whatever buttons have disappeared off the bottom of the screen.

### Post by falconindy on 2009-11-08
1) Make a new rule
2) Click the +
3) Change Type to "Window Name"
4) Click "Grab" (your cursor should now be a +)
5) Click the window you're setting a rule for.
6) Click OK
7) Set Width and Height
