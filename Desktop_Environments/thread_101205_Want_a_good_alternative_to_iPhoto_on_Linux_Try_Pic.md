### Post by magnusbb on 2005-12-09
I was tired of not having a proper program for organization and management of my various photos on Linux. Yes, I know - there are F-Spot, KView, Image Viewer, gThumb and what not. And in most circumstances these work allright.

But, I want something as smooth as iPhoto. On Windows I've used Picasa 2 made by Google, and I really like it. I am happy to tell that if you use the latest version of WINE, this works application works as good as in Windows.

To get the latest and greatest WINE, add this to /etc/apt/sources.list:

```
deb http://wine.sourceforge.net/apt/ binary/
deb-src http://wine.sourceforge.net/apt/ source/
```

Then sudo apt-get install wine, if you don't allready have it installed. 

Picasa can be found here:
[http://picasa.google.com/](http://picasa.google.com/)

Enjoy!
