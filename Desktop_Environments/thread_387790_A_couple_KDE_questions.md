### Post by Jucato on 2007-03-18
> **bg1256 said:**
> 
First, How do you create a desktop icon for the trash can?  In Gnome, it was in gconf-editor, but I can't figure out how to in KDE.  I have it in the taskbar, but I prefer it on the desktop.

Right-click on the desktop, select Create New -> Link to Location (URL). In the Location (URL) field, type in "trash:/" (without the quotes).

> **bg1256 said:**
> 
Second, how can I disable the feature which makes the taskbar display every window on every desktop?  I can't figure out how to rid myself of that. I can do it on my laptop which is not running Beryl, but on my desktop which is running Beryl, I can't figure out how to disable this feature.

Unfortunately, this problem lies more on the Beryl side than on KDE. I believe there's no fix available for it yet. Beryl just doesn't always play nice with KDE's panel (Kicker).
