### Post by stuporglue on 2005-12-13
I'd first just try rebooting. 

If that doesn't fix it, can you switch to a terminal (ctrl+alt+F2) and log in there. Do you get a prompt there, and can you run basic programs like "ls"? 

(almost) Worst case scenario, boot into recovery mode from the grub menu, and reset the password for your user. 

#password $USERNAME
 
then reboot
