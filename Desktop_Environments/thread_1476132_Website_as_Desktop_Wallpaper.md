### Post by lmarmisa on 2010-05-07
Capture an image of Firefox showing your HTML file and use this image as desktop wallpaper. 

For capturing the image, Click on Applications -> Accesories -> Take Screenshot  and select Grab the current screen.

Click on System -> Appearance -> Background and Add... for the selection of your screenshot as wallpaper.
