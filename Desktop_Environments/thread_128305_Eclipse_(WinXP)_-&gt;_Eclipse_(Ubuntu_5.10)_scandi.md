### Post by kaamos on 2006-02-11
You can set a character encoding for a specific project in Eclipse. Ubuntu by default uses utf-8 and windows something like iso-8859-1.

EDIT: right click on a project -> properties -> text file encoding -> Other -> ISO-8859-1
