### Post by Tenken on 2008-01-20
If you don't use the Network Manager at all you can stop it from running by going to System->Preferences-Session and un-checking the Network Manager. Most applications, Pidgin and Sonata for example, have options in their preferences on where or not they should appear in the tray.

### Post by Anduu on 2008-01-21
Once you network etc is set up and working properly network manager doesn't need to be running.Just go to System>Preferences>Sessions and untick its box in the Startup Programs.

As for the reboot icon you can right click it and remove it from the panel but you can't "hide" it.Remember if you do this you will have no way to reboot
.
