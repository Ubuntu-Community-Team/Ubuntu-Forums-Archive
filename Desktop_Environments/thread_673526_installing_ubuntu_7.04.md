### Post by Casual Fan on 2008-01-20
Whoa,,,this is a trainwreck. First, you want to download the current release, which is 7.10, "Gutsy Gibbon." The next release is due out in April, 8.04, or "Hardy Heron." You do NOT want to download that as it's still in testing and is unstable and suitable only for hair loss and indigestion. :)

6.04, 6.10, and 7.04 are older releases which you don't need to try unless you have problems with 7.10.

An in-depth guide to installation is here: [https://help.ubuntu.com/community/Installation](https://help.ubuntu.com/community/Installation)

You'll need a blank DVD or CD (and a drive that will burn them) and a program that write disk images (ISOs) to discs.

Post if you need more help and WELCOME TO UBUNTU!
