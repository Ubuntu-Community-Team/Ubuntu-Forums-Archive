### Post by Ampersand on 2005-12-23
Are you sure you connected the hard drive on the same IDE cable as before. The boot loader is set to load operating systems from a particular hard drive, so if it changes from primary to secondary, or master to slave, it could cause problems.

Your problem is probably not Linux or ubuntu related.  More likely you connected the hard drive back differently.  You might have changed the IDE connector you put the hard drive on.  Maybe you changed the master/slave jumper setting on the hard drive.  (seems unlikely)
