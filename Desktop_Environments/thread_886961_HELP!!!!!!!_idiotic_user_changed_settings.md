### Post by mali2297 on 2008-08-11
Boot in *recovery mode* and see if you can reach a root terminal. Then create a new user, say *tony*, and add him to the *admin* group with the commands
```

adduser tony
adduser tony admin

```
Type
```

reboot

```
and try to log in as the new user.
