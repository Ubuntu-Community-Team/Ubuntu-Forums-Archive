### Post by grahammechanical on 2013-04-11
> [COLOR=#000000]Hasn't 12.04 more users than 13.04?[/COLOR]

That is true but those users install an LTS release for the stability that comes with having the OS code base stable over the support period. So, although 12.04 has had 2 point releases in the last year and another 2 point releases to come in the next year it will not be converted over time into 13.04. People do not like change in LTS releases.

Further more, Canonical does not have responsibility for third party software. As we are told in the Software Centre, "Canonical does not provide updates."

Mark Shuttleworth recently said: "If you want stability, install an LTS. If you want 'bleeding edge,' install the development branch." Or words to that effect.
