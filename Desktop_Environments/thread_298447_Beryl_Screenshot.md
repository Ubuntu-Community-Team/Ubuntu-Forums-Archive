Another way for screenshot in cube mode which no one mentioned, I think, is:

```
 sudo apt-get install scrot ## install scrot 
```

and then execute the command:

**scrot 5 intrabutt.jpg**

That will time the screenshot to fire in 5 seconds and create an image with the name intrabutt.jpg, in your current directory.
You can obviously adjust the time and the name.
