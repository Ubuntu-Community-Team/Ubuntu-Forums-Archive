### Post by Ubuntwan on 2008-04-23
although this is en extremely old thread, today I ran into the same problem.
I think you are doing the procedure wrong (I think I was). The way to actually do this thing seems to not use the F4 key to show and select your database source and not use the 'Data to Fields' button there.
The way that does work is to leave the labels not filled in so there is only '<first name> <last name>' or whatever fields you wanted in your label, but no actual names. Then go to file-> print and after saying yes to the form letter dialog choose you data source and record there. then print to file (or directly to paper) and the resulting file will have multiple pages with all the labels.
