### Post by Zwack on 2005-12-30
I run VNC on my windows box rather than my Linux box...

Then on Ubuntu I run x2vnc with appropriate options.   The result is that I control my Windows box across the network using the same keyboard and mouse. 

The screens are side by side but connected to different machines.

When I move the cursor to the edge it "jumps" from one to the other.

I can also cut and paste between the two.

The only thing I can't do is press Ctrl-Alt-Delete on the windows box...
