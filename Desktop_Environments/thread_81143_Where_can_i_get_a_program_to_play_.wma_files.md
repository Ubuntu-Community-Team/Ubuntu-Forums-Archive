### Post by bored2k on 2005-10-23
* Install mplayer-386 and totem-xine

* Download the codecs:
[http://antesis.freecontrib.org/mirrors/ubuntu/plf/pool/non-free/w32codecs/w32codecs_20050412-1ubuntu2_i386.deb](http://antesis.freecontrib.org/mirrors/ubuntu/plf/pool/non-free/w32codecs/w32codecs_20050412-1ubuntu2_i386.deb)

* Install
sudo dpkg -i *.deb
