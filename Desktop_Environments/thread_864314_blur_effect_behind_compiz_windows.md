### Post by otternox on 2010-08-09
I'm sure this post is long dead, but hey. ;)

In CompizConfig Settings Manager, go to Effects, Blur (don't activate it yet), then set "Blur Filter" to Gaussian, leaving everything else as it is.  Now activate blur (checkbox).

Hope this helps someone.
