### Post by schdefan on 2005-08-16
edit this lines in xorg.conf to the following:
```

Section "Monitor"
	Identifier	"hp p1230"
	Option		"DPMS"
        HorizSync    30-95 # play around with the numbers, carefully
        VertRefresh  60-85 #
EndSection


Section "Screen"
	Identifier	"Default Screen"
	Device		"NVIDIA Corporation NV10DDR [GeForce 256 DDR]"
	Monitor		"hp p1230"
	DefaultDepth	24
	SubSection "Display"
		Depth		1
		Modes		"1280x960"
	EndSubSection
	SubSection "Display"
		Depth		4
		Modes		"1280x960"
	EndSubSection
	SubSection "Display"
		Depth		8
		Modes		"1280x960"
	EndSubSection
	SubSection "Display"
		Depth		15
		Modes		"1280x960"
	EndSubSection
	SubSection "Display"
		Depth		16
		Modes		"1280x960"
	EndSubSection
	SubSection "Display"
		Depth		24
		Modes		"1280x960"
	EndSubSection
EndSection
```

or you can build a new xorg.conf with 
```
sudo dpkg-reconfigure xorg-xserver
```
but don`t forget to save the current one.
