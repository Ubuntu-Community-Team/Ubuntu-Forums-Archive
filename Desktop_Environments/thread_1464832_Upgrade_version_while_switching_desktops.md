### Post by snowpine on 2010-04-28
Yes, of course. :) Psychocat has good how to's on switching desktops, for example: [http://www.psychocats.net/ubuntu/puregnome](http://www.psychocats.net/ubuntu/puregnome)

Always backup  before you upgrade to a new distro. You can have any number of desktop environments to choose from at the log in gui. You just have to install them 1st, but do this after upgrading not before. Now some will attach new programs inherent to the new desktop to the original used desktop. This is basically because you are generally using a desktop that share many features with another in it's basic setup.
