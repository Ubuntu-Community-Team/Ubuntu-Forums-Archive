### Post by Samuel on 2005-10-26
system>administration>printing   is the printer showed here? if not click add new printer. if it is right click > properties > print a test page, to see if its working properly.
different steps are taken in different apps to print, i use abiword for text documents and use ctrl-P and you can select which printer you want to use there, it should show your printer, pdf or generic post script. 
i think most apps work in pretty much the same way for printing. 
i have used a stylus C64 and a stylus R200 with no problems.
