### Post by cmulax on 2007-04-28
I am having some annoying problems with gnome currently.

Everytime i restart X (ctrl+alt+backspace) or reboot, gnome will always start up with the sessions window (the one you get from going to System -> Preferences -> Sessions), the file manager at my home directory, and a couple of other windows that were open in my last session.  I assumed this was because i had checked "Automatically save changes to session", but unchecking it did not solve this issue.

I will continue experimenting to see if i can solve this, but any help would be appreciated.

### Post by nuttiplutt on 2007-04-29
Just my two cents.. I've been playing around with sessions although I'm a newbie.

You can try to first make sure "automatically save changes to session" is unchecked, then close all other open windows and programs (not your startup programs of course) and then click "Save this session". Maybe a reboot is also in order after that.

This fixed my session at least.
