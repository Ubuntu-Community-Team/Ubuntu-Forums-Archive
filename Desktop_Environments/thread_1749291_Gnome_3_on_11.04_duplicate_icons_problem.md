So to aid others I first installed lxde to get past the ICEauthority error message that would not allow me to start Gnome 3. That solved the problem but duped a bunch of icons and they looked terrible. Removing lxde via synaptic and they doing a cold reboot (logging and then back did not solve the issue) did it!
