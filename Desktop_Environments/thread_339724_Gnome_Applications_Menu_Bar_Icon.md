### Post by skoubidou on 2007-01-17
Hey raven-

You know, you'd think it'd be that easy. I've been using the T-ish-ubuntulooks theme and i couldnt even find that distributor-logo.svg icon. But then i read your directions a bit closer and...

login as root and  change every single distributor-logo.svg icon 'till i found the one that t-ish-ubuntulooks was linking to. 

I think it was the "Tango" or "Tangerine" theme found in "/usr/share/icons/Tangerine (OR Tango)/scalable/places" . And now I finally got that gnome foot I always wanted!

So yeah, for anyone using the t-ish-ubuntulooks theme change the .svg file in /usr/share/icons/Tango/scalable/places and you're good to go.

Thanks a bunch
