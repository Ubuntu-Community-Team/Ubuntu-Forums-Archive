### Post by AndreiCiprian on 2006-03-04
[https://wiki.ubuntu.com/RestrictedFormats](https://wiki.ubuntu.com/RestrictedFormats)
You can find answers to the majority of your issues on the wiki.
Install xmms, and then you can uninstall rythmbox, as it has issues when importing music files from ntfs partitions.
After you install the goodies according to the wiki go:
apt-get remove --purge rhythmbox
