### Post by jimmydean886-2 on 2011-11-21
Try re-installing ubuntu-desktop. You're definately missing a component.
You should also be able to search for shut down in the Dash.

Here's an image demonstrating.


[IMG]https://lh4.googleusercontent.com/-gYPEQH7Tdp8/TsrSpu4nqnI/AAAAAAAAAEs/9QJVnj8HDak/s800/shut%252520down%252520in%252520DASH.png[/IMG]


Also, the shutdown command you mentioned above will only send your computer down to  "recovery mode"

```
 ~$ sudo shutdown -h now 
```

is the only one that really consitantly works.

Also try using

```
 ~$ halt 
```
