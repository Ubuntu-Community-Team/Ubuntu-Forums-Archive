### Post by 3Miro on 2011-08-26
> **video editor said:**
> but i have ubuntu 10.04. i would like to have gnome three but  i dont have a cd or dvd

Ubuntu 10.04, 10.10 and 11.04 come with Gnome 2. Ubuntu 11.10 coming this October would come with Gnome 3.

There are ways to get Gnome 3 running on 11.04, but I am not sure about 10.04. You will have to use an unofficial ppa and things will surely be buggy (there is a good chance that it will break your system). I would recommend, if you want to try Gnome 3, you should either install a Gnome 3 distribution (like Fedora 15) or wait until Ubuntu 11.10.
