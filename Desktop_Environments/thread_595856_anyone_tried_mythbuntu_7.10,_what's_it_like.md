I've never used Mythbuntu specifically, but I've run MythTV on Linux before...

MythTV runs like an application in Linux. So you can open it like any other program if you want. However a common configuration is to set it to automatically launch into MythTV after it finishes booting (which is great if you have it hooked directly to your TV). So, basically, it's up to you how you want it to behave.

You will certainly have access to a normal desktop if you need it.


As to how well it works with hardware, there is some variability. It's definitely a good idea to check hardware compatibility lists before buying anything. In general the Hauppage cards work quite well. With some tweaking I was able to get my Hauppage capture card and its remote control working perfectly.


MythTV is a really great system. It's fun to tweak and revolutionizes TV-watching! But, it does require a bit of research. For instance to get it to control a set-top box (cable box or whatever) may require an "IR blaster," and so forth...
