### Post by mravik on 2006-06-02
We all know that Canonical is going to support Dapper for 5 years. I like this a lot! But, from a security perspective, I think this could become for the average user a security problem. Universe is supported by the community. If the community will stop building packages for Dapper as there will be new Ubuntu versions coming out, the average user will miss security updates for his universe-packages.

So, is anything "planned" of the community to support Dapper for 5 years with security bugfixes? Otherwise, one should reasonably think about installing from universe, if one plans to user Dapper for 5 years.
