### Post by grahammechanical on 2012-01-23
What do you want to do by way of customization? The top panel in Ubuntu is now a reserved space for things called app indicators. Many of the old style applications that would put an icon in the top panel have been modified to work with the new Unity Interface.

Here is a link to some that are available:

[http://askubuntu.com/questions/30334/what-application-indicators-are-available]("http://askubuntu.com/questions/30334/what-application-indicators-are-available")

The Unity Interface can be modified in a limited way by installing CompizConfig Settings Manager through the software center. You will then be able to load CCSM through the Dash. When the utility is open click on Ubuntu Unity Plugin and you will get the options to modify or customize Unity. I have been able to make the top panel transparent this way.

Another alternative is this:

[http://deviceguru.com/ubuntu-11-10-without-shell-shock/]("http://deviceguru.com/ubuntu-11-10-without-shell-shock/")

I have not done this, so I cannot recommend it as a solution but only as available information.

The question remains: What do you want to do in the way of customization?
