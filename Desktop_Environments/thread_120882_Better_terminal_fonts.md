### Post by moephan on 2006-01-23
Go Applications->Accessories->Terminal

In Terminal pick Edit->Current Profile...

In the general tab, unlcick the "use system terminal font" checkbox. This will enable the Font button. Click the Font button and select the font from the font chooser. Click "Close" and the new font will take effect and will be the default font in terminal. The "Profiles..." command will allow you to create different configures that you can easily choose by going Terminal->Change Profile.

Cheers, Rick
