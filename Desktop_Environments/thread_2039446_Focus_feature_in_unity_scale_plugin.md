### Post by deadvirus on 2012-09-10
I don't know about any "focues" feature, but you can make the window title appear over the window.

Just install "compiz-plugins-extra" and then in Compiz Configuration System go to the "Scale Addons" plugin, Appearance tab and configure the Window Title options.

Make sure that the "Text" plugin is also active.

Apart from the scale option above to add text labels (which I used to use but no longer do because I find Compiz too fragile when you set esoteric options like that), don't forget about the ALT+<key above tab> option to see and step through only the windows of the current app. The <key above tab> is normally backtick (`) on US keyboards.
