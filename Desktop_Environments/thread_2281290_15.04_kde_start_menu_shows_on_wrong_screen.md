### Post by sypher2 on 2015-06-06
I don't know if this will help but it fixed some of my issues. Right click the taskbar where the task manager is at, and in one of the settings, you should be able to check a box that says display only current screen tasks and display only current desktops tasks. Do it for both and they should become separate individual screens with each it's own tasks. That fixed the problem you have for me, if I understood your question correctly
