AFAIK there is no way to remove some of the default Places (Computer, Home and the media) from the menu, and having them as possible submenus would violate the Gnome HIG (no sub-sub-menus).

You can add things, though, by bookmarking them in nautilus or the file selector.

### Post by Sepero on 2007-09-11
Adding a location to the 'Places' menu in Ubuntu:
Browse to the location, say, /data, and then select Bookmark -> Add.

Edit/Remove location:
Open a Nautilus/Navigator window, click the Bookmarks menu.
