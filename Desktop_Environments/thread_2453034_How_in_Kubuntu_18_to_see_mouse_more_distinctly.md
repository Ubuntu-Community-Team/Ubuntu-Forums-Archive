### Post by Holger_Gehrke on 2020-11-05
Activating the 'Track Mouse'-option enables a keyboard combination. Hit that combination and you'll get a moving graphical effect around the cursor. The icon in the same panel as the configuration option should allow you to set the keys to use. If you still don't see anything happening, check whether the compositor is set active (settings->system-settings->display and monitor->compositor), it's needed to draw the effect.

Holger
