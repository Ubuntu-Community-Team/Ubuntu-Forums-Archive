### Post by phibuntu on 2006-05-26
I solved my problem. Not the elegant way, but it works now.

I changed the Option "MonitorLayout" under Device (/etc/X11/xorg.conf) the following way:

** original (see my first posting [= second posting in this list])
```
Section "Device"
       ...
        Option          "MonitorLayout" "LVDS, CRT"
        .. 
EndSection
```

** changes (swap CRT LVDS)
```
Section "Device"
       ...
        Option          "MonitorLayout" **"CRT, LVDS"**
        .. 
EndSection
```

Whenever I want to see a dvd (using gxine) on an external Monitor, I switch "CRT, LVDS". Probably it has no other effect than beeing a "wrong configuration" and so disabling the LCD Monitor of my Laptop? 
But after the chanes I have what I want. The Video Output comes on the external Monitor. (Unfortunately the laptop screen remains black, but the main goal is reached.) 
Dont forget to change xorg.conf back, after your DVD afternoon!

### Post by wcbaker on 2006-06-17
I found a fix for my problem.

When I added these three lines it started working
> 
	Option	    "MonitorLayout" "TMDS, LVDS"
	Option	    "CloneMode" "1280x1024"
	Option      "OverlayOnCRTC2" "on"
Yet, I believe the last line is what did it. So if you are keeping it minimal, try that one.
