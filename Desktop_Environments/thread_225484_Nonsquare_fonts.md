### Post by reclusivemonkey on 2006-07-29
Few here for you to try ;-)

[http://www.gnome-look.org/content/show.php?content=9883](http://www.gnome-look.org/content/show.php?content=9883)

To install, simply unpack the archive, then make a font directory and use fc-cache to update your font list. I wouldn't suggest installing them all; have a look in nautilus and you will get a rudimentaty preview. Copy them to a temporary dir, then just intall all of them from there.

```

sudo mkdir /usr/share/fonts/truetype/ttf
sudo cp /path/to/my/fonts/*.ttf /usr/share/fonts/truetype/ttf/
sudo fc-cache

```
