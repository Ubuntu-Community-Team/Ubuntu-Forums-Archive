### Post by Dr_Willis on 2010-06-27
If you put the game in to 'window' mode - you should be able to resize the  window to take up both monitors. But thats not exactly useing the 'fullscreen' option. 

I also have to drag the window where its part way across both monitor  Then manually resize it so it fills up both. (i do this in Lord Of the Rings Online)

You Might be able to simply disable twinview and use just xinerama and  it Might fullscreen across both monitors automatically. Normally this would be considered a bug, because most games are not playable that way. But in your case it would be a 'feature'

I have had cases where games in wine do NOT properly see the res of the screens with using twinview. You may need to alter the games configs by hand to set the proper res.

Good Luck.
