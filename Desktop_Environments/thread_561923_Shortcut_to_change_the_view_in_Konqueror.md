Hello,

one thing that bothers me with Konqueror (compared to Thunar for instance) is that there are no shortcuts for 2 things: (i) changing the View mode, to detailed list for instance, and (ii) to see  hidden files.

Is there a way to set a shortcut for these actions? Maybe one for Icon view, one for detailed view, and one for seeing or not hidden files.

Thanks for the help :)

---

### Post by GeneralZod on 2007-09-28
Settngs -> Configure Shortcuts -> Type "hidden" in the search box (must be in file-manager mode).

Views are pluggable and so, to my knowledge, do not have shortcut keys.
