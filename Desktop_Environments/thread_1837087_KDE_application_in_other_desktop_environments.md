### Post by malspa on 2011-09-01
> **tommpogg said:**
> Since the main feature of xfce is being a lightweight desktop environment. Am I throwing it all away by running a KDE application which consumes more resources?

Well, for example, I have SalineOS installed here, which uses Xfce by default, but I've added a few of my favorite KDE apps, like Dolphin, KWrite, and KSnapshot. I think that's about it. Anyway, those apps brought in a lot of KDE stuff, but the system still seems as snappy as always. The only thing is that the first time I launch Dolphin or another KDE app, there's a bit of a pause while the KDE stuff loads up. After that, everything seems as fast as normal, and there's no pause the next time I launch a KDE app.

Seems about the same when using other desktop environments or window managers. For example, an old notebook that I have is really sluggish when I run KDE on it, so I installed Fluxbox on it, and then it responds really well even when I'm running KDE apps within Fluxbox.
