### Post by rainwalker on 2008-02-20
VLC is definitely the media player you want, and +1 for Amarok, too. To play DVDs and get other media-related things check out [www.medibuntu.org](www.medibuntu.org)
Also, enable the restricted and multiverse repositories (System > Administration > Software Sources) and then install the Ubuntu Restricted Extras through Applications > Add/Remove

For a everything media-related, this is a really nice thread: [http://ubuntuforums.org/showthread.php?t=661833](http://ubuntuforums.org/showthread.php?t=661833)
