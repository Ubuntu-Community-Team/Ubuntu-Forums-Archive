### Post by SeijiSensei on 2014-06-20
First thing I would do is disable [akonadi]("http://userbase.kde.org/Akonadi").  Instructions are [here](http://userbase.kde.org/Akonadi#Disabling_the_Akonadi_subsystem).  Note that you must modify the configuration of the standard clock widget in the panel so it won't use Akonadi.  You also cannot use other Akonadi-enabled apps like KMail or Kontact.  I don't use those so I routinely disable Akonadi on weaker machines like my ASUS 1201n netbook.
