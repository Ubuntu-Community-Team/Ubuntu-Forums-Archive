### Post by 5nak3 on 2009-05-01
when you get to the load in screen, you have to click on options on the bottom left corner (assuming you have the default GDM)

then from there select sessions

and then check the xfce option

press ok

type in user name and password

and then select if you want to use xfce from now on as default or just for one session only.

Hope that helps :D
