### Post by aysiu on 2006-07-01

Don't give me too much credit. What I did was quite simply, actually. I popped in a Xubuntu live CD, typed ```
sudo aptitude update && sudo aptitude install ubuntu-desktop
``` said "no" to installing all the packages and then did ```
sudo aptitude update && sudo aptitude install kubuntu-desktop
``` and also said "no" to that.

Then I copied and pasted the list of packages to be installed.
