Hi,
This is a request for contributors to the Ubuntu Mobile Guide.

The Ubuntu Mobile and Embedded (UME) project aims to derive an operating system for mobile internet devices using Ubuntu as a base. Sponsored by  Canonical and Intel, the project was launched at UDS-Sevilla following a preliminary announcement to the Ubuntu community.

UME documentation is due to be published as Ubuntu 7.10 (Gutsy) is released, but in the meantime you can read and contribute to this "beta" documentation online.

[https://help.ubuntu.com/community/UMEGuide](https://help.ubuntu.com/community/UMEGuide)

As the UME becomes more widely used it would be great to have an area here on the forum for discussing mobile use and development.
