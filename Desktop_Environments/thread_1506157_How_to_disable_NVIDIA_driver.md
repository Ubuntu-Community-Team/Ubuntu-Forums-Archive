### Post by Bobulous on 2010-06-10
If you simply want to stop using the proprietary (non open source) NVIDIA driver, you can do this in Ubuntu by going to the System menu, then Administration, then Hardware Drivers.

You should see one or two NVIDIA drivers, and one will be marked as active. Highlight the active one, and there should be a button marked "Remove" or something else that sounds ominous. I clicked on that and the driver was disabled (but not deleted from the system).
