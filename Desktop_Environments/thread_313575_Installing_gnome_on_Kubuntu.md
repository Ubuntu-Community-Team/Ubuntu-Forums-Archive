### Post by kazuya on 2006-12-06
(1) select synaptic from system menu
in synaptic click update
(3) In the search box, type: ubuntu-desktop
(4) Right click on it and select install.. - this would add all the other necessary stuff you need for gnome.

alternatively, you could simply search for gnome from synaptic and install the things that say gnome. You would see that selecting one of these gnome stuff automatically adds a ready to install notation to other things required.

At the end, you can choose to leave kdm as is or select change to "gdm."

Viola, you now have gnome desktop.

PS: For XFCE desktop, you would search and install xubuntu-desktop. You can have as many window managers as possible. Have fun. Some may argue that with the extra window managers comes bloat and system may be slightly slower. 

I like having them all and Ubuntu makes it easy.
Enjoy.
