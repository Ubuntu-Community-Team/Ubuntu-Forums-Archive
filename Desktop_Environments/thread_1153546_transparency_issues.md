### Post by mcduck on 2009-05-09
Sorry, that's not possible.

Support for transparency is still very new thing, and currently in order to get partial transparency in program you need couple of things:

1. Working compositing manager. This is included by default and if you can use desktop effect you have this.

2. You need to use Murrine-based GTK2-theme. Not a hard thing, but definitely restricts your desktop setup.

3. The program in question has to have support for partial transparency. Only a very few have, Firefox doesn't.

The last step is where things get bad. Hardly any program has any kind of support for partial transparency. Without it, you can only make the whole window transparent.
