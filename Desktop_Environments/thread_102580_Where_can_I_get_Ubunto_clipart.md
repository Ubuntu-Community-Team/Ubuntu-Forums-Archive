### Post by anil_robo on 2005-12-12
OK I'm back from work. Had a tiring day. Now for your question, Runlevel0!

Open a terminal window, type this:
```
sudo apt-get install gnome-art
```
press enter.

It'll install gnome-art manager.

To run that, go to System-->Preferences-->art manager.
Note: you must be connected to internet every time you need to access gnome-art!

ENjoy! :)
