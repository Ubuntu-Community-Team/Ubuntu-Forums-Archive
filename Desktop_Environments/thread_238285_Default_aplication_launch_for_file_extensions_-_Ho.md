### Post by reacocard on 2006-08-17
To change the default application used to open a particular type of file, right-click the file, choose properties, open with, and selct which program you want. that's it.

BTW, ubuntu doesn't use extentions to determine filetype, it uses mime-types, which are a bit more advanced, but it works out to about the same thing, except the file can be named whatever you want and ubuntu will still know what type it is.
