### Post by leech on 2006-06-30
Personally, and maybe I'll get flamed for this, but I never liked using k3b in Gnome.  Because it takes forever to load and has a lot of overhead as far as RAM goes (it has to load all the KDE libraries).  NeroLinux DOES fill a gap that is needed.  It will actually burn .nrg files along with pretty much everything else.  Plus it's getting better and better.  They really are trying to put out a good application.  Plus it's only $20.  Nero for windows is around $70 for the non-OEM.  Not to mention that Nero doesn't use cdrecord, which hasn't really been developed on for a long time.  

Besides, if you've already paid for NeroLinux, then keep it.  But honestly, it's not like Gnomebaker takes all that much space anyhow, just use whatever you feel is best right now.
