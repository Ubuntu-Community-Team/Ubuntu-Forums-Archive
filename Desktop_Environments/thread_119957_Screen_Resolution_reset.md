### Post by cayamara on 2006-01-20
[QUOTE=SpyderCS]What section should I edit?[/QUOTE]
For me, it works when I refresh rates to the "Monitor" Section:
```
Section "Monitor"
        Identifier      "SyncMaster"
        Option          "DPMS"
        **HorizSync       30-65**
        **VertRefresh     50-75**
EndSection

```
To reconfigure the xorg.conf easily in the future, you can use dpkg-reconfigure:
```
$ sudo dpkg-reconfigure xserver-xorg
```
Hope it helped
