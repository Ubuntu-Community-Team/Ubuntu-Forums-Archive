### Post by aysiu on 2006-03-17
You know how to install themes, right?

You just drag the icon .tar.gz file to System > Preferences > Themes and it gets installed automatically to /home/randal/.icons.

You don't need anything on your desktop to link to.

Or are you trying to create shortcuts to those folders? If so, find the folder and middle-click-drag it to the desktop and select **link here**.
