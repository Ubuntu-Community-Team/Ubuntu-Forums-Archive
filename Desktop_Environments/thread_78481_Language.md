### Post by daller on 2005-10-18
Run this from commandline:

sudo apt-get install language-pack-kde-LANGUAGECODE

Let's take Danish as an example:

sudo apt-get install language-pack-kde-da

To see a complete list, do:

sudo apt-cache search language-pack-kde
