Hi,
using Ubuntu Gutsy I have a problem that changing default language from English to my local language, the login still remains in English and after login my local language is working well. So How to make my login in screen my local language?

I have installed and set the following settings:
1. I selected System | Administration | Language support and select my local language and installed my local language support.
2. Reboot.
3. At Username login screen I selected menu Options | Select language. I selected my local language and press Change Language button. 
4. The question appears: "Do you wish to restart the login screen with the chosen language?" I pressed Yes buttoon and login screen has reloaded in my local language.
5. I entered userid/password and Ubuntu and question "Do you want to make xxx the default for future sessions?" (Question is in my local language. XXX is my local language). I selected "Make default".
6. Reboot to see if settings preserved.
7. Login screen is still in English after login my local language is set and working well.

How to change my login screen to my local language?
