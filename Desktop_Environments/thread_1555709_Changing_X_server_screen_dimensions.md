I was thinking about this earlier, it might work to use a smaller common resolution and set the graphics card to not scale the image assuming you're using DVI and the drivers have the feature. For example run a 1920x1080 display at 1280x720 (or another smaller 16:9 ratio). Doing this would be much more straightforward since there are already settings to do this. The caveat is you lose that screen real estate on all side of the screen, not just the one with the dead pixels.
