### Post by ajgreeny on 2020-09-28
I do not see this problem on my Xubuntu 20.04 system whether with an open application menu such as File (that menu closes), or with the whisker menu open (that stays open).
If I open the Date/Time panel applet Alt+Tab still works but shuts the applet's calendar window.

For screenshots with open application menu or whisker menu I can also just use the Print-Screen key.
If that does not work for your system you can open the xfce4-screenshooter, choose to delay the shot for a few seconds and in those seconds open the menu you want to show in the image.
