### Post by ardchoille on 2006-07-22
Gnome is a desktop environement that uses Metacity as its default window manager. I have found that changing the window manager in gnome does make gnome a bit faster and I really like gnome. There are lots of window managers that you can try, I have tried openbox, fluxbox and WindowMaker and they are all fast and quite nice. However, there is a site [http://xwinman.org/](http://xwinman.org/) that lists a lot of window managers that you can try. Try some of those and see which one meets your needs best.

If you're interested in changing the default window manager that is used in gnome, I wrote a tutorial about it: [https://help.ubuntu.com/community/ReplaceMetacityWithOpenbox](https://help.ubuntu.com/community/ReplaceMetacityWithOpenbox)
