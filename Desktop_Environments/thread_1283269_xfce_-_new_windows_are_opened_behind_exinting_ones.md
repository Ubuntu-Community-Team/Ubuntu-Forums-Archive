### Post by guano on 2009-10-05
I've been using Karmic since Alpha 6, and everything works fine, but there is one annoying thing: every new window I create (doesn't matter which app) will open _behind_ all other open windows, and not in front of them as it used to happen. So if I am working on some Latex file and call "view pdf", evince will open behind Kile or Texmaker, and so on.

Anyone has seen this?

TIA

---

### Post by sisco311 on 2009-10-05
Go to: Xfce4 menu -> Settings -> Window Manager -> Focus tab and make sure that the *Automatically give focus to newly created windows *checkbox  is selected

---

### Post by guano on 2009-10-05
Works, many thanks.
