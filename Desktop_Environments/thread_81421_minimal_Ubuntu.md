### Post by TimelessRogue on 2005-11-05
There is a problem with removing "gnome-games" using this method:  

Firstly, you will be advised:  "Cannot remove 'gnome-games':  There are other applications depending on this one. If you really want to remove it, please use the "Advanced" mode from the menu."

Secondly, going to the "Advanced" mode, the process also insists on removing "ubuntudesktop" which does not sound like a good thing to do and in fact effects the system "because it is used to carry out certain upgrade transitions (such as adding new packages to the system)" to quote the notes.

Is there some other means to remove "gnome-games"?  Or even a need to since they really don't take up much room ...
