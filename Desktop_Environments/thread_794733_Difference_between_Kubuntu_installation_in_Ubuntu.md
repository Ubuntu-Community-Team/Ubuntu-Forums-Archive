### Post by Xiong Chiamiov on 2008-05-15
Kubuntu-desktop installs all the packages, just as if you had installed Kubuntu (like me! yay).  Kde-core will only install the necessary parts to get KDE running.  Past that, I'm not sure, but you can mark one for install and take a look at what other packages will be installed as well (in synaptic, I think it's custom filters in bottom-left, then marked changes).
