### Post by dcstar on 2005-11-25
[QUOTE=briguy]I've got a bit of trouble too - I'm running version 2.0 from ubuntu debs found on this forum.  Spellcheck so far only checks footnotes!  I think I'm going to revert to the old version...[/QUOTE]
Double check you OO Language settings.

You should have the little ticked "ABC" next to your default language - that tells you that there is a dictionary loaded for it.

Then go into Writing Aids-Edit and check that you have the various installed options ticked to enable them.

The last thing you have to make sure of is the language set for the part of the document you are working on matches the other language settings you have a dictionary for - otherwise no spell checking!!!     :???:

BTW, on my OO the "Install New Dictionaries" wizard does nothing, I had to use the handy little DicOOo macro file utility to install the dictionary files I required.
