### Post by Brandon Williams on 2009-05-13
There is no UNR ISO image. Just install the standard Jaunty i386 desktop, 'sudo install ubuntu-netbook-remix', and then 'sudo remove ubuntu-desktop'.

Before using the desktop-switcher tool, review the 'Known Jaunty bugs' thread over in the 'General Help' forum. There is a note there about the desktop-switcher.
