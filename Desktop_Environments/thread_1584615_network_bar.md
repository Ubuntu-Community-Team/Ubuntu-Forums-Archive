### Post by roggenschrotbrot on 2010-09-29
run
```
nm-applet
```
in a terminal emulator. if it reports to be allready running your panel is missing a notification area (right-click, add, notification-area). if it appears in the panel after executing the command you have to add nm-applet to your startup programs.
