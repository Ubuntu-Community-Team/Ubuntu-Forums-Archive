I found that, after having installed compiz-config, some (but not all) program starter icons are missing, i.e. are replaced with a blank. If I hover over them, their names appear, though, so I _can_ click on the program or folder. But this wasn't so before I installed compiz-config.

### Post by Jenske on 2012-09-16
Oddly, I found that, clicking "default" in MyUnity > themes, reinstalls the icons of all programs. So it must have something to do with the AgingGorilla theme I set.
So in a way this issue is resolved.
