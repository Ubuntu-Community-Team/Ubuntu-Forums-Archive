### Post by ajgreeny on 2011-01-17
OK, but I've never come across anything like that.

The only way I can think of that might be possible is to make a new folder in your home, and in that put a copy of every application.desktop file that you already have in your menus.  If you set the view of that as compact it is almost the sort of thing you want, and if you choose not to show the sidepanel, toolbar and locationbar which normally show in nautilus, you will get a nice clean look like my screenshot.
