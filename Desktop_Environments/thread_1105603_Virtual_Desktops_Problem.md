### Post by bigbencg on 2009-03-25
The number of desktops can be specified by right clicking the desktop selector in the task bar and selecting "Configure Desktops".  You should be able to set up to 20 desktops, whatever setting you select they should appear in the bar.

I installed compiz window manager to try and replicate your issue as I am also using kubuntu.  Even after the install and loading compiz, I was still able to right click the desktop selector in the task bar and set my number of desktops.
