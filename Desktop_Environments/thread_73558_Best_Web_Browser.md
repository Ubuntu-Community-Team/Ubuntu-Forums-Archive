### Post by aysiu on 2005-10-09
The best one is the one that works for you.
What are your needs?

For example, I like Firefox because of all the extensions I can use and the integrated search engines (you can add as many engines as you want), as well as the "I'm feeling lucky" search from the location bar. I also like the keyboard shortcuts and the order in which tabs cycle.

Someone else, though, may like the speed of Opera, its slightly better W3C-compliance, and its tab cycle behavior.

Still someone else may want a lightweight browser with no images. That person may go for Lynx.

What are your needs/browsing habits?

### Post by rosslaird on 2005-10-09
The "best browser" is indeed the one that works best for you.
Here's my two bits:

The latest Opera really is amazing: fast, stable, loaded with options. Much faster than Firefox on my system.

Galeon is based on the same rendering engine as Firefox, I believe, but Galeon is reliably much faster on many systems.

Konqueror is a wonderful browser if you use KDE. The things I like most about it (aside from stability, speed, etc.) are inline form spell checking (this post is being checked on the fly as I write it), the kio slaves, the preview of forum postings (when you hover over a forum posting Konqueror shows about a paragraph's worth of the text in the post), and overall integration with KDE.

And Kazekahase is pretty cool as well. Very customizable.

If I had to choose one browser, it would be Opera. It has everything: browsing, mail, news feeds, notes, etc.

Happy hunting.

### Post by rosslaird on 2005-10-09
My Firefox doesn't do the pop-up preview thingy.

Iakudi:

1. Go to opera.com
2. Click on the big green button that says "Free Download"
3. Select Ubuntu, then your version of Ubuntu, from the drop down list and selection box.
4. Download and save the .deb file somewhere you can find it.
5. Open a terminal and navigate to the directory where you saved the .deb file.
6. Type in "sudo dpkg -i whatever-the-filename-is.deb" (without the quotes). If the .deb file is the only one in the directory, you can just do: sudo dpkg -i *.deb
7. Wait for the install to complete. If you have errors, post them here.
8. Look for Opera under your Internet Menu, or use alt-f2 and type "opera"

This should work.

Ross
