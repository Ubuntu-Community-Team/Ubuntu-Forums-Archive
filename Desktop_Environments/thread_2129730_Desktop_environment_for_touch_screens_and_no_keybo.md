### Post by ocwo92 on 2013-03-27
I'm looking for a windows manager that is well suited for (small) touch screens and no keyboard. In particular, I find it very difficult to operate a touch screen when a window manager uses slider buttons in lists or other containers whose contents don't fit inside of the container all at the same time. The Android-like scrolling where you simply drag the contents is much easier to use on a touch screen (and might even be easier with a mouse, but that's another discussion).

Which windows managers are available for Linux--preferably as Ubuntu repositories--that provide Android-like windows scrolling?
