### Post by zombifier25 on 2013-04-09
Well, I don't use SETI@home (I'm more of a Folding@home guy), but if it has a full screen graphical application that shows the details, then it is possible to run it as a screensaver (or any app actually)

Follow this tutorial [http://www.webupd8.org/2011/11/how-to-get-electric-sheep-to-work-with.html](http://www.webupd8.org/2011/11/how-to-get-electric-sheep-to-work-with.html) (replacing "electricsheep --root 1" with the executable of the graphical app, if any)
(though I don't see the point of it. Running a screensaver may potentially waste power that could have gone to SETI@home, which is already running in the background)
