whenever I double click on a folder on desktop it tries to launch it with VLC player... ### Post by SeijiSensei on 2011-06-03
Use System Settings > File Associations to control which applications are used to open files and folders.  

Can you not change the default "Open With" for folders by right-clicking on some folder and making Dolphin be the default?  If not, use the File Association application, and select inode > directory from the tree in the left-hand window to change the default for folders.
