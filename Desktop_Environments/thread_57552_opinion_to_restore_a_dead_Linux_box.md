### Post by André on 2005-08-17
Wow,

what a fubar system.

I don´t know if it is the best way but here is what i would do:

Use the Ubuntu Live CD which should have USB support and copy your home directory over. If you haven´t one of the free shipped CDs you can get it here: [http://www.ubuntulinux.org/download](http://www.ubuntulinux.org/download)

Than do a reinstall of the whole system. I wouldn´t just copy all files over tot the new home directory but see which ones you need so you have a nice clean install (but that´s just my personal opinion!).

Hope you have a nice and new install again in a few hours :-/
