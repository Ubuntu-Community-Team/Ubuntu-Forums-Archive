### Post by palsyboy on 2006-07-20
I personally have an Athlon XP 2000+ (1.67 GHz), 512 MB RAM, and a 128 MB NVidia card.  My system flies along just fine for any task I throw at it.

I have installed on many friends' computers, and theirs work just fine using a 1 GHz CPU, 256 MB RAM, and 32 MB integrated graphics (with slight variations from one machine to the next).

The only time I've ever had a problem is running Ubuntu (GNOME) with 128 MB RAM.  It's slower than dirt, but those particular users would be uncomfortable with a lighterweight desktop manager.  If you're willing to use Fluxbox, everything's fine on 128 MB.
