### Post by Xian on 2005-07-11
[QUOTE=ItIsMe]I've accidentely installed some files that I can't delete.
Does anyone know how to use terminal to delete files using the sudo command or anything like that?[/QUOTE]
There are some 'rm' commands that will delete anything - and that's the problem as it is just too easy to make a typo and wipe some important data. Much better to just open Nautilus with admin privileges, navigate to the files you want to delete and remove them individually or in groups with the right-click option.
```
$ sudo nautilus
```
