### Post by JarrettBillingsley on 2007-04-01
This is more of a peeve than anything, but whenever I open a folder in KDE, the viewing Konqueror window is gigantic.  It nearly covers the screen, and I'm running at 1280x1024.  I've looked for options in Konqueror to see if there's some kind of default window size, or remember folder size options etc. but to no avail.  

Is there any way to make the default Konqueror window size a bit smaller?

---

### Post by tbroderick on 2007-04-01
Open konqueror and re-size to desired size. Before closing go to Settings->Save View Profile "Kubuntu File Manager". A Profile Management window will open, just check the "Save window size in profile" box, and it should remember the size.
