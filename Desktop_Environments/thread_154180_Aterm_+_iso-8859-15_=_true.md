### Post by colo on 2006-04-02
You need a font capable of rendering all of iso-8859-15's chars, and an adequately adjusted locale.

`man locale` for more info.

I'd suggest you forget all that iso-crap asap and make the switch to UTF-8 Unicode, if Ubuntu has not already done so.
