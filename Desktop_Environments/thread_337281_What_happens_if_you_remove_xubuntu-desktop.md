### Post by jem7v on 2007-01-13
AFAIK: More or less, xubuntu-desktop is a package that contains nothing except dependencies for all the packages necessary to create a working Xubuntu desktop install, so if you didn't have Xubuntu installed (i.e., perhaps you had a different version of Ubuntu installed), you could select to install it in Synaptic and it would select all the programs you needed (coz they're dependencies).  If you remove the xubuntu-desktop package it isn't really removing anything except what is essentially a list of all the packages Xubuntu needs to run.

But it would be good to reinstall it before you do a distro upgrade, because those depend on the (version)-desktop packages to work, I think.
