### Post by lvleph on 2009-05-01
All these posts and not a single person tries to address the OPs issue?

To change icon sizes:
- Open the File Browser and choose: Edit -> Preferences
- In the Views tab, "Icon View Defaults" section, set the desired (50% for example) "Default Zoom Level"
To change icon previews:
- Open the File Browser and choose: Edit -> Preferences
- In the "Preview" tab, set desired options.
