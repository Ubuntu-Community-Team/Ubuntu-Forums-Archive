### Post by jchau on 2006-08-13
Well, you got the splash screen part working, so at least part of the CD is working.  What I'm curious about is when that error pops up.  

Here are a few possible problems:
-Bad CD ISO.  try downloading it again
-Bad CD burn (though unlikely)
-Ubuntu doesn't know how to use your CD drive (Debian had that problem with my CD drive, but the Ubuntu 5.10 install didn't)

You can also try the Alternate Install CD.

But these suggestions are just a shot in the dark.  Good luck.
