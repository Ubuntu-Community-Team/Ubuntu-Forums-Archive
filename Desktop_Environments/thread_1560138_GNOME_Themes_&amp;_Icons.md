### Post by Frogs Hair on 2010-08-24
The themes you install yourself are in your home/ hidden folders. The themes that come with Ubuntu are located in the file system User> share> theme folder . Some of the icon themes from Look .org are incomplete, I have found three missing the fonts icon for the software center.
