### Post by Bloemetjesgordijn on 2009-08-18
My personal experience is: (just stepped from Gnome to KDE)

[COLOR="Black"]Gnome (standard Ubuntu)[/COLOR]
Looks (eye candy): ok (out of the box)
Eye candy, is easy to install (Concky, docks, etc)
Ease of use: great
Feels smoooth and fast

[COLOR="Black"]KDE (=KUbuntu)[/COLOR]
Looks (eye candy): Great (out of the box)
Ease of use: ok; Tweaking, somewhat more difficult then Gnome.
Feels somewhat sluggish (KDE 4.3)

Like mcduck says you can run Kubuntu (KDE), Ubuntu (GNome) and Xubuntu together on one pc. They can use each others applications and determinem at login which interface you would like to use.
