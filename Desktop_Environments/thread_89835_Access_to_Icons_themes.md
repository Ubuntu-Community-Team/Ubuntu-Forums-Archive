### Post by audax321 on 2005-11-13
When you see the icons you want to see shaded, just click OK and then when you get back to the screen that had /usr/share/pixmaps .... you'll notice that the address in the bar has changed. Click in the bar and push ENTER and the icons in that directory (not sub-directories) should display below.

The icon browser thing is a nice feature for Gnome, but they really should consider auto-updating the list when a user selects a new directory.

### Post by audax321 on 2005-11-14
Well this is the screen I think you see:

[IMG]http://www.dieburnbot.com/Screenshot-Browse_icons.png[/IMG]

you should click the Browse button here and choose a folder that contains the icons you want to use... notice that the icons are greyed:

[IMG]http://www.dieburnbot.com/Screenshot-Browse.png[/IMG]

Click the open button. Now notice the path in the text box has changed:

[IMG]http://www.dieburnbot.com/Screenshot-Browse_icons-dirselect.png[/IMG]

To see the icons in this path, click in the text box where the path is so there is a text cursor there and then push ENTER on your keyboard and the icons will display:

[IMG]http://www.dieburnbot.com/Screenshot-Browse_icons-shown.png[/IMG]

Hope this clarifies things.
