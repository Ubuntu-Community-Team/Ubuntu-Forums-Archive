### Post by FurryNemesis on 2006-11-16
I'm doing this from an XP machine so I can't remember the exact steps, but run beryl-manager from the terminal if you've not got it installed in the taskbar, then go to one of the options on the right, possibly under "windows" or the like. The "mouse" tab in the correct option will have motions ticked to centre active windows - untick them.

Anyone else know exactly where that option is?  

Good luck.

### Post by 23meg on 2006-11-16
Run beryl-settings, go to Scale / Screen Edges & Corners and uncheck the appropriate boxes.
