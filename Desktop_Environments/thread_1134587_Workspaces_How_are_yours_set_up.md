### Post by Anzan on 2009-04-23
As for myself:

In Fluxbox, running useful GNOME services:

Always on:
Workspace 1: Firefox (Gmail), Pidgin
Workspace 2: GNOME terminal running htop, several tabs with man pages or such
Workspace 3: Nautilus for files and network shares
Workspace 4: Gedit (with embedded terminal etc), OOWriter, Evince or Okular tabbed into one window
Workspace 5: Rhythmbox (is ending development so trying different tools)

A few other workspaces for miscellaneous GNU tools run in xterm, or GIMP, Scribus, Stellarium, Totem or Xine or Mplayer, Xchat, ripping CDs, uploading photographs and such.

Tomboy Notes running in notification area
Xclock pinned
Tiny Xeyes pinned at bottom near taskbar so I can find the mouse cursor to click something I need to click
