### Post by Emerzen on 2005-10-21

**sudo nautilus**

browse to the partition and you should be able to change permissions and write to it.  If you want to have a root filebrowser availabe to you in Applications Menu -> check out the directions at [www.ubuntuguide.org](www.ubuntuguide.org) -> makes it easy.
