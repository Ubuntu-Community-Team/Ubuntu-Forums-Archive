### Post by steveneddy on 2008-01-27
System --> Preferences --> Advanced Desktop Effects Settings

Enable "Desktop cube" under the Desktop section

Preferences section select Skydome

and then scroll down to the Skydome Image and browse your PC for the skydome image of choice.

If memory serves, it has to be an image that is in the format of 2:1 (800x400, 1200x600, etc.)
