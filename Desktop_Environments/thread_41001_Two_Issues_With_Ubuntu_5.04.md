### Post by YaAqoB on 2005-06-22
To change the Splash Screen make sure you have the new one saved in a good location.
Then go :
Applications --> System Tools --> Configuration Editor

Then drop down to /apps/gnome-session/options

On the right there is a part about Splash_screens. Right click and you can then enter the new location of the image want to use.
