### Post by aysiu on 2007-12-28
For an older computer, I would use neither Kubuntu *nor* Ubuntu.

You're better off doing a custom install of a minimalist window manager like IceWM or Fluxbox:
[http://www.psychocats.net/ubuntu/minimal#barebones](http://www.psychocats.net/ubuntu/minimal#barebones)

It kind of depends on what you mean by *old* or *low-end*.

If you have a 900 MHz processor with 192 MB of RAM, Xubuntu might be a great choice.

If, however, you have a 766 MHz processor with 64 MB of RAM, you're better off with Fluxbox or IceWM.
