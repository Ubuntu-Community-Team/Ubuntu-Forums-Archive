---

### Post by Leif on 2005-05-01
It's just a meta-package encapsulating dependencies for other programs and settings ubuntu installs by default. It's not a problem if you break it now, it won't do anything bad, but you might want to re-install it when breezy (the next release) comes out, so you'll get all the changes that come with it too.
