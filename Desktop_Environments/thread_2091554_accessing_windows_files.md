### Post by Corelogik on 2012-12-07
> **dharmvir tiwari said:**
> i have installed ubuntu inside windows.
how can i access the file on my windows desktop using ubuntu

Open a file manager window, click on file system look for the host folder. That is the Windows "drive". Browse like normal.

Since you will not have the normal Windows file structure in the Linux file manager, you'll have to get to your individual user folders the old fashioned way.

File System/host/Users/<user name>
