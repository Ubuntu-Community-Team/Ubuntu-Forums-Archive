As it's been pointed out, this is posible. However, for your purposes it may actually make more sense to have multiple user accounts. You'd log in (as you currently do) as "derrick" (or whatever) and any friends could log in as "guest". This way, your friends wouldn't be able to mess up your files or settings, e.g.

You could set KDE as the default desktop for "guest", while you can running GNOME whenever you log in (as "derrick" or whatever.)
