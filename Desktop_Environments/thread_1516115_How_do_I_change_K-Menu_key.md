### Post by Been Told on 2010-06-23
I'm running Ubuntu 10.4 with KDE. I have a keyboard with two Windows keys.
Now, in a perfect world I would map the left Windows key to the K-Menu and the right one to that run-panel (or whatever it's called).
I switched to Linux a few weeks ago, mainly because it seems like a lot of at least near-perfect things are possible on it. :D

So far I've managed to remap my Windows keys to F13 (left) and F14 (right). But how do I bind them to the K-Menu and that little run-panel (the one you open with <Alt>F2) respectively?

---

### Post by Zorael on 2010-06-23
System Settings -> Keyboard & Mouse -> Global Keyboard Shortcuts. Or just pop open a run box and enter 'global'.

Component: Plasma Workspace, *Activate Application Launcher Widge*t: F13.

Component: Run Command Interface, *Run Command*: F14.
