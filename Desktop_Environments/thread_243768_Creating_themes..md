### Post by HeavyAl on 2006-08-25
There isn't a lot of documentation out there on themeing for Gnome but you might want to start here: [http://live.gnome.org/GnomeArt/Tutorials/GtkThemes](http://live.gnome.org/GnomeArt/Tutorials/GtkThemes)

Another tactic would be to simply tear into an existing theme and see how it works. Try modifying it to your own tastes in order to get an idea of how different things are done.
