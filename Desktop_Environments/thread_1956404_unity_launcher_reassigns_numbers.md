So I think my favourite thing in Unity is super+number as a way of switching between applications.  There's one thing about it that trips me up.  I've got seven applications permanently in the launcher.  So it assigns super+1 through super+7 to them by default.  So far so good.  It never changes the numbers assigned for 1 through 6.  But it constantly changes the number assigned to the seventh one whenever I open an application that isn't one of the permanent seven.  I find this annoying because I have to keep checking what applications are attached to 7 through 0 or else I'm hitting the wrong number a lot.

I *assume* there is a config file somewhere which controls what numbers are assigned to what and which of them are allowed to be reassigned.  But in searching through some hidden folders (e.g. ~/.local/share, ~/.compiz-1, ~/.config, ~/.gnome2, etc.) I didn't come up with anything.:-k
