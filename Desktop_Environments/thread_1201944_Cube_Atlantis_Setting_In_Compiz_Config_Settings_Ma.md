Open the Synaptic Package Manager, type in the search field "compiz plugin unsupported" and look for the package "compiz-fusion-plugins-unsupported" and install it. After that in the Compiz Config Settings Manager will appear the "Cube Atlantis" option next to the "Cube Gears".
