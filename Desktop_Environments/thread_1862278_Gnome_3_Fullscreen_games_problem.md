### Post by Vesem on 2011-10-21
Hey,

I was having the same issue with Starcraft 2.  What I did was I turned "Allow the window manager to control the windows" off inside winecfg.  That took care of the problem, however, I couldn't type.  So I also had to turn "Allow the window manager to decorate the windows" off.  This is currently allowing me to scroll down, but it doesn't allow me to use alt tab or change workspaces.  So, not a perfect solution, but works for me.
