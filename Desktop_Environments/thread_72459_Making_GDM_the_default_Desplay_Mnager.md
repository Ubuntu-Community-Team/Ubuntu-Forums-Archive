I apt-getted kubuntu, but now I'm switching back to Gnome. No troubles, except that KDM is still my default display manager. Is there a configuration I can set to make GDM my default display manager?

---

### Post by Ampersand on 2005-10-06
running 'sudo dpkg-reconfigure gdm' in a terminal should give you the option to select the default display manager.
