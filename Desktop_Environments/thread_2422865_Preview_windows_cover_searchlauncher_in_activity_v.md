### Post by again? on 2019-07-14
You can reload gnome-shell by hitting alt+F2, typing the single letter "r" and pressing Enter.
You can also try the application overview shortcut instead ...super+a
Check gnome extensions if installed.

There is also an extension, you may find useful, to open the applications overlay as the default view when you hit the Super key.
[https://extensions.gnome.org/extension/1198/start-overlay-in-application-view/](https://extensions.gnome.org/extension/1198/start-overlay-in-application-view/)
