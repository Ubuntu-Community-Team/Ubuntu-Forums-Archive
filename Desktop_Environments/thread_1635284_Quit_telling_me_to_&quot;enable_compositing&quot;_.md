### Post by Phrawm48 on 2010-12-01
Dear Ubuntu / Gnome / whomever (I really don't care who):

A strong suggestion: Please quit putting messages in the GUI that tell me a feature "needs to enable compositing" but that provides me with no guidance or pointers to HOW, exactly, to satisfy that application's or feature's need for compositing.

I'm really, really tired of having to do research after receiving "you need to enable compositing" messages.  It's stupid!

Better yet, if a feature or tool -- Docky, Gnome Do, what have you -- requires compositing, why not add the necessary modules or whatever they are to the list of that feature's or tool's dependencies?

Yours in continuing annoyance and frustration,
"Needs to Enable Compositing"

### Post by Liverbones on 2010-12-01
> **a-user said:**
> they cannot tell you how, cause they can not know at which point of the "chain" you are missing something.

what graphic card do you have and what driver are you using? proprietary?

It's also worth asking what window manager (or desktop environment) you are using as well. If you're using the default GNOME desktop, you should be able to get compositing by enabling Compiz (otherwise known as Visual Effects in Appearance Preferences). If you're using another window manager, you can always try using xcompmgr to provide some simple compositing, but it's woefully out-of-date, and honestly, a bit clunky at times.
