### Post by Jucato on 2007-08-20
How you save a file depends on what text editor you are using. The keyboard shortcuts you gave are only for nano so, presuming that's what you are using, it should work even in Konsole.

Unless you are using vim, then you should type :w and :q to save and quit, (or :wq to do it at the same time).
