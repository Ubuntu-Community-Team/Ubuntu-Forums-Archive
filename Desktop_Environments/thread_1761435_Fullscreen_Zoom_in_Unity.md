### Post by NosLycn on 2011-05-18
So,I found out that the compiz zoom function is still active.  They keybinding, on the other hand, has been removed.

It doesn't zoom in on the unity launcher/panel, but it does zoom in on everything else.  For those of you out there wondering how to get (nearly) the same level of fullscreen zoom we used to have in Ubuntu, install Compiz Config Settings Manager, click "Enhanced Desktop Zoom", and set the hotkeys for zooming in/out.

To install the compiz config settings manager, run the following in terminal:

```
sudo apt-get install compizconfig-settings-manager
```

To run the compiz config settings manager, run the following in dasher (alt-f2):

```
ccsm
```

Best of luck to you all.
