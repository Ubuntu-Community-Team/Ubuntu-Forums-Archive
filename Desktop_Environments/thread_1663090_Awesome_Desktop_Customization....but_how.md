### Post by Krytarik on 2011-01-09
Compiz is the program that you want (to configure): [http://wiki.compiz.org/](http://wiki.compiz.org/)

It is already installed by default, to easyly configure it's settings, install CCSM (CompizConfig Settings Manager), enter in a terminal:
```
sudo apt-get install compizconfig-settings-manager
```You will it after that in the "Preferences" menu.

You may also want to install the extra-plugins:
```
sudo apt-get install compiz-fusion-plugins-extra
```Have fun with it!

PS: You are probably right with WINE, running a Virtual Machine is another option:
[https://help.ubuntu.com/community/VirtualMachines](https://help.ubuntu.com/community/VirtualMachines)

EDIT: Oh, you may mean the theme also. See [http://gnome-look.org/](http://gnome-look.org/), like Frogs Hair suggests.
I have no hint for the desktop widgets, if you mean that too.
