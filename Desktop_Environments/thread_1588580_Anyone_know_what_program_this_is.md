### Post by 3Miro on 2010-10-05
If you have Ubuntu installed, just go to the Software Center and install ccsm (compiz config settings manager). Then from Sys -> Prefs -> Advanced Desktop Effects, play with the options. Cube/Transparency and so on. Look for tutorials on Compiz.
