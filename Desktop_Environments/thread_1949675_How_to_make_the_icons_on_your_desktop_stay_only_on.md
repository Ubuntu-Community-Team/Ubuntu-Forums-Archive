### Post by 3Miro on 2012-03-30
As far as I know, this feature is available only for KDE's Plasma Workspace (look for activities and folder widget). You can install kde under 10.10, but it is quite a bit different from what you have now. You can install kde by typing in the terminal:

```
sudo apt-get install kubuntu-desktop
```

You can check this tutorial for how to switch between different environments, just make sure to look at the instructions for the right version of Ubuntu.

[http://www.psychocats.net/ubuntu/kde](http://www.psychocats.net/ubuntu/kde)
