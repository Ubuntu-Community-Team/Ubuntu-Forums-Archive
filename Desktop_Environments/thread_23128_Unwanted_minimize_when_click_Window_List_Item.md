### Post by kassetra on 2005-04-03
Ah - here's the scoop:

If you have two windows up, and one is in front of the other one, but the one behind is the one that has the focus, and you use the button on the bottom bar to "give it focus" ... then yes, it will minimize it - because it already has focus.
