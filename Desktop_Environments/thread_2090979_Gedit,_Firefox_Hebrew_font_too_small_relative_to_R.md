### Post by orb9220 on 2012-12-04
Well in firefox I go to the Preferences>Content and click on Advanced in the Fonts & Colors section. In there is the default settings for different fonts. Can change the default fonts used and click or un-click Allow Pages to choose their own fonts.

Not up on all the different settings and just fiddle with firefox open on the page and see the results right away. 

Just did this for this for the ubuntu forum reply box. As all the text was huge. So changed the Monospace from 16 to 12 now repy box text is normal size.

Below the Advanced button is Colors and you can adjust or overide web site defaults there too.
