### Post by 23meg on 2006-11-04
You're looking for Beryl / Compiz in a compositing environment (an XGL server or xorg 7.1). Beryl's Scale plugin is more or less a clone of it.

You can also do it in a non-compositing environment (regular X) with an app called Skippy, but it's slow, and besides, testing Beryl is a Good Thing to do if your hardware is suitable.
