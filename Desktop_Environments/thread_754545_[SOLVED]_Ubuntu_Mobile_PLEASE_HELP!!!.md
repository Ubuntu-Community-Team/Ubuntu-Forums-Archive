### Post by LeoSolaris on 2008-04-13
It's not the same definition of 'mobile' that you're thinking of. Mobile as in mobility rather than mobile as in cellular phone. The hardware for Ubuntu Mobile is not actually out yet. They are called MID's and according to [moblin]("http://moblin.org/") they will not be available till the summer of '08. It seems to be aimed at the new Intel Architecture called the Intel® Centrino® Atom™.

Apparently there are very few UMPC's running the Intel architecture, including the Samsung Q1 Ultra. That comes from the [FAQ]("https://wiki.ubuntu.com/MobileAndEmbedded/FAQ")s and the [wiki]("https://wiki.ubuntu.com/MobileAndEmbedded"). 

This seems to be in a pretty earily alpha state. Even if you find that your cell uses x86 architecture (which it should run on according to the FAQ's) It may not be quite ready yet, but the package is available in the repo and should be able to be flashed to your device.
