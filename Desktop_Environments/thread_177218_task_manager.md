### Post by Dr. Nick on 2006-05-15
Open a terminal and type
> killall easytag
and it will close

You could also add the desktop switcher applet to the gnome panel so you could navigate to that desktop

### Post by nanotube on 2006-05-16
if you want a gui taskmanager-like thing, you can go and open Applications>System tools>system monitor (from the main menu), and you can kill processes from a gui list (among other things you can do). 

but getting comfy with a commandline is at any rate a good thing to do. if you have the desire, try going to the first link in my sig, go to the "other command line resources" section, and you will find some nice tutorials about it.
