### Post by ardchoille on 2006-07-25
There are lots of advantages of Linux over XP, too many to list. But, I'll list some which come to mind. Also, I use my computers for email, web surfing, graphics design, listening to and burning music, programming, watching video and DVD movies, creating and editing office documents, and more. I have found that Linux will let me do much more than Windows is capable of.

1. Linux is free. You can download it for free, install it for free, and use it for free. In fact, I have been using Linux for 5+ years and have never paid a penny for it.

2. There aren't any active viruses for Linux at the moment. This is not because Windows is more widespread, but rather because writing a virus for Linux is pretty much a waste of time. Even if Linux was the primary desktop OS being used in the world, a virus doesn't have much of a chance of destroying the system and spreading.. simply because of the way the system is set up. This explains it better:
[http://www.theregister.co.uk/2003/10/06/linux_vs_windows_viruses/](http://www.theregister.co.uk/2003/10/06/linux_vs_windows_viruses/)

3. The source code for Linux is open, so there are more people auditing it than there are people auditing Windows source code. More eyes in the code equals better chances of malicious code being caught and the community being informed. If a problem is found in Windows, Microsoft will let you know when they're good and ready.. meanwhile your Windows machine is sitting there with a problem/exploit that you don't even know about.

4. In Windows, a crash has a better chance of bringing down the entire system. In Linux, crashes do sometimes occur but rarely do they bring the entire system down with it. I haven't seen a crash on any of my 11 Linux boxes in a few years.

5. With Linux, you end up learning a lot about software and hardware, which is a very good thing. Windows does too much for the user and keeps the user stupid.

There are many more reasons why Linux is better than Windows, but the best way to find out is to install Linux and learn how to use it. Only you can tell which OS is best for you.

> **kornmation said:**
> Ok I got other Linux OS(Mandreke, RedHat, SuSe, Gentoo, YellowDog, etc)

You might have a bit of trouble installing Yellow Dog on your laptop, as it's for PowerPC computers (i.e. Macs).

Xubuntu is for machines with low specs, and as a result it isn't quite as user-friendly as regular Ubuntu, so bear that in mind.

> **kornmation said:**
> Photoshop cs2, All macromedia programs, LightWave o Maya7, Visual Basic anyone, All Sony Progrmas(Vegas, DVD Architech, Sound Forge, Acid Pro) Fruty lopps 6 XXL Produce edition, etc and i need to know if linux can run  or something similer (Im willing to liike i paid those) progrmas with that power of production

Xubuntu comes with The Gimp; if you really need the pure power of Photoshop, then you'll need to run Photoshop within Crossover Office (it's a kind of non-free emulator for Windows programs). Otherwise, The Gimp should serve you well.

Lightwave has a very good open-source equivilant called Blender. Blender is available from the repositories. It also provides audio recording and sequencing capabilities for your rendered footage rather than pass over the duty to a dedicated program.

I'm unfamiliar with the music programs you mention, but you should check out programs like Seq24, Ardour, Audacity, and Rosegarden for music production and sound editing (all from repositories). There are probably a lot of other good Linux programs too that I haven't heard about.

Assuming that your footage is in DV format, Kino or Diva should float your boat as far as video editing is concerned. Kino is in repos, I think. QDVDAuthor is also available for creating the DVD, but I haven't used it yet.
