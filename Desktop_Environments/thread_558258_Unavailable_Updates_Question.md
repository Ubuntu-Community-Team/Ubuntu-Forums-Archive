### Post by yabbadabbadont on 2007-09-23
For version updates, you'll probably have to wait for the next release.  Generally, updates are only made to the release for major bug fixes and security updates.  For security updates, they usually hit the repos somewhere between a few hours and a couple of days after the notice goes out.  In the case of the last Firefox security notice, I believe that it only affected the Windows version of Firefox.  I'm not positive on that though.

Edit: It appears that it does only apply to Windows.

[http://www.mozilla.org/security/announce/2007/mfsa2007-28.html](http://www.mozilla.org/security/announce/2007/mfsa2007-28.html)
