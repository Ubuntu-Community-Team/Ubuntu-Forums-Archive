### Post by markupstart on 2007-04-16
Here are a few useful links.  I have found a lot of the how-to and help on E17 is outdated or not useful anymore. I have been running e17 full time for about a month now.  If you need any help, let me know.  I am in feisty and installed it through this repository:

deb [http://e17.dunnewind.net/ubuntu](http://e17.dunnewind.net/ubuntu) feisty e17

I also installed Exhibit, which is a image viewer for e17. E17 is quickly becoming my favorite, it had been Xfce4, but I like E17 alot, I do use thunar file manager though in e17, as it it one of the best file managers out there.

[E17 user guide]("http://www3.get-e.org/E17_User_Guide/English/")

[Additional themes]("http://www3.get-e.org/Themes/E17/")

[Animated icons]("http://www3.get-e.org/Resources/Animated_Icons/")

[Couple of useful tips here]("http://gentoo-wiki.com/HOWTO_emerge_e17")


You can't tell by the screenshot, but I am using some animated icons, including firefox, thunar, terminal, and audio player found on the above link.  I also have 5 shelves loaded up with different things on each shelf, some shelves, like for the weather, I made transparent, which is just one of the configurations for the shelf. By, the way, I am not on a slow computer, I just like the way e17 runs, once in a while it will give me some kind of error message, but it always recover just nicely from it.
