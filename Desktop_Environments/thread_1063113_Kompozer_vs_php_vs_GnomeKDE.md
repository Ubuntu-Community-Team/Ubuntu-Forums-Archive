### Post by Tim Sharitt on 2009-02-07
For Gnome; right-click on a php file and select properties. Click on the 'Open With' tab and check the button beside kompozer if it is on the list. If kompozer is not on the list, click add at the bottom to add it to the list.

I'm not entirely sure what the procedure is for KDE, but if memory serves, I think it is similar to the Gnome procedure.
