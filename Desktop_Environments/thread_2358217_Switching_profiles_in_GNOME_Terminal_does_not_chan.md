Hello,

I am running GNOME Terminal 3.18.3 on ubuntu 16.04 LTS. During a GNOME Terminal session, I can successfully adjust the window size via the menu Terminal>80x24 or 80x43 etc. I created a profile, called mycustomprofile, with a custom initial terminal size 80x60. The works as expected if I select File>Open Terminal>0. mycustomprofile. However, if switch profiles **during** the GNOME Terminal session via Terminal>Change Profile>mycustomprofile, the window size does not change. However, other properties of the profile, such as background colour are updated instantly.

Is this standard behaviour for GNOME terminal or a bug?
