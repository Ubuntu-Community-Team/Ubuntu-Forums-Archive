### Post by 3Miro on 2012-07-20
DE is a collection of applications that give you the desktop experience.

The update manager is a tool that looks for updates of ALL installed programs. So long as you have KDE installed, it will give you updates for KDE. Update software is not tied to the desktop environment.

In the same way, Synaptic gives a list of all packages that can be installed. Ubuntu (and Linux in general) does not distinguish between KDE apps and Gnome apps, in fact many of us are running one KDE app or another under Gnome.

Why do you want to remove the KDE apps from Synaptic, why is this a problem for you?
