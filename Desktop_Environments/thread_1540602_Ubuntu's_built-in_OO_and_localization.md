### Post by Hagar Delest on 2010-07-28
Ubuntu uses the Novell version of OOo. It's slightly different from the official Sun/Oracle version. The former has more features but also more bugs.

Personally, I only use the Sun/Oracle version. If you want to file a bug report, you can do that on the Ubuntu bug report system.
