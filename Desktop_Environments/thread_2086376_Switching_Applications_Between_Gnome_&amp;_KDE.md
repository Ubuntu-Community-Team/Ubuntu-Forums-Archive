### Post by cat2005 on 2012-11-20
I know I can have both Gnome and KDE in my Ubuntu / Kubuntu. 
 
Must I use only Gnome-related programs while in Gnome and KDE-related programs while in KDE?
 
Or could I mix? 
 
For example, could I use KDE but run synaptic or nautilus? Both are Gnome-related programs.

---

### Post by snowpine on 2012-11-20
You may use any application in any desktop environment.

The one "gotcha!" is that certain applications may have unintended effects when run in a different DE; for example nautilus will redraw your wallpaper unless you launch it with the --no-desktop flag.
