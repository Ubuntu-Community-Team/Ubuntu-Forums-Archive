How do you move close- minimize- and maximize buttons to the left upper windows corner in Lubuntu? In Gnome you can change with with gnome editor in metacity like this:
[https://help.ubuntu.com/10.04/config-desktop/C/window-button-order.html](https://help.ubuntu.com/10.04/config-desktop/C/window-button-order.html)

### Post by Rodney9 on 2012-03-04
Go to Menu / Preferences / Customize Look and Feel 

Click on Window Border and reverse the code ie -

> NLIMC to CMILN
