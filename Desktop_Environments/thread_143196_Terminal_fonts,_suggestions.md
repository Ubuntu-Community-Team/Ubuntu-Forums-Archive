### Post by hw-tph on 2006-03-12
Assuming you use Gnome Terminal or urxvt (or anything else that uses XFT): Grab some of the programming fonts from [the Proggy collection](http://proggyfonts.com/). Get the Truetype (TTF) files and copy them to ~/.fonts (create the directory if it doesn't exist). Most will look good only at one size, usually around 10 or 11. I prefer Coding Font Tobi but Proggy Clean and Proggy Square come in handy too (I use the latter in gvim).
