### Post by Copper Bezel on 2011-10-13
@ ceti - it's not just a UI shell, though. All those updating live tiles depend on having apps written to support them, so the whole DE would have to be tailored to that style of interaction.

With that said, although I haven't played with it extensively, some spins of Android Honeycomb are similar in design.

KDE's [_Plasma Active_]("http://community.kde.org/Plasma/Active") project, a relatively new initiative, is intended to be precisely this, by the way, a UI and a set of apps designed to bring mobile flexibility to a conventional desktop OS, but it's just getting started. Still not as pretty as Windows 8, though. = (
