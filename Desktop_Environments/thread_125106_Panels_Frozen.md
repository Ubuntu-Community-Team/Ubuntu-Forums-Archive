### Post by bikeboy on 2006-08-27
> **MrCheese said:**
> Tip - ALT-F2 opens a "run program" dialog box.

 Try ALT-F2 and type "gterm" to open a terminal.

If that doesn't work the other option is "crtl + alt + F1" or F2-F6.
This takes you to a virtual terminal that where you can login and run commands. Do "ctrl + alt + F7" to get back to the gui desktop.
