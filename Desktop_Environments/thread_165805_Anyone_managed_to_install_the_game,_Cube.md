### Post by Hachi on 2006-05-09
[QUOTE=TrojanSkin]I have the Cube tarball on my desktop tried it. Nothing. Could someone post EXACTLY how to do it? Or is it in a repository anywhere? Thanks.[/QUOTE]

First of all you need to extract the tarball, then go into the cube dir, give cube_unix permission to execute (properties), bring up a terminal, *cd* to the cube dir, then run *./cube_unix*

And that should be it, unless you're missing some libraries. Let me know if you have any problems.
