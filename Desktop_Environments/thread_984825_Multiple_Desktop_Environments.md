### Post by john.nicholls on 2008-11-17
Your straight Ubuntu installation uses Gnome, and as you are aware brings in a lot of extra programs, as does the alternative KDE. However Xfce is a much lighter desktop, and adding it will not add a great number of extra programs. You can load the xfce4 meta-package by the one command ```
xfce4
```
