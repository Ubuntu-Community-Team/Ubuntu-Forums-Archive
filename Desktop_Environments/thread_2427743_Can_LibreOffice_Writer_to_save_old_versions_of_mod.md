### Post by Holger_Gehrke on 2019-09-25
Not by default. But you can set it to create backup files before you make any changes (Menu Tools->Option->Load/Save->General->Always Create backup copy). You can also set it on a per file basis to track changes (Menu Edit->Track Changes->Record). With this you should be able to accept or reject changes as needed. See the Libre Office Help System, "Common Help Topics", "Document Version Tracking" for details, but obviously only for files which had tracking of changes activated before the changes were made.
