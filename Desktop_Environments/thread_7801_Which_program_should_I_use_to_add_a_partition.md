### Post by az on 2004-12-11
Use parted.  It is a command-line program.  I cannot be run from the hard-drive you are using, since the harddrive needs to be unmounted.  Use the install cd to repartition your disk, if this is the case since it rund from a ramdisk.

You can also use qtparted in knoppix.

There is also gparted, but the same goes - you need to unmount your disk...

### Post by Freakwitch on 2004-12-11
Definitely for ease of use I would recommend qtparted. This program is on the Knoppix CD, which is how I generally redo partitions. Boot into the Knoppix livecd and run qtparted. Very straightforward.
 
One suggestion, if you are shrinking a windows partition, make sure you defragment it before you shrink it. You can use the defragger in windows for this.
