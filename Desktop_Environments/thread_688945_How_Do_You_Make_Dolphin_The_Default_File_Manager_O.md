### Post by samir85 on 2008-02-06
I had this issue too. 
Go to Systemsettings / Advanced / File Associations. There you you have to search for a filetype called directory or inode something like that (I dont remember entirely anymore), then for this fyle type put Dolphin at the top of the applications.
