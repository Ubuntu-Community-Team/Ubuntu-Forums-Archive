### Post by Dullstar on 2009-08-15
I was going to ask some questions about some things, but I figured them out on my own!  I'll post them here for everyone's reference.

Q:  How do I clean up and/or customise my menus?
A:  System -> Control Centre.  If it's not there, crud.  If anyone has more information about whether or not this option is there by default, please let me know.  If it's not there by default, tell me how to get it because I wouldn't remember how.  Once there, find Main Menu.  You can tweak things to your liking now.

Q:  How do I change a launcher icon?
A:  Just go to the launcher's properties and click on the icon.  Pick an icon.  I have found them in /usr/share/pixmaps/

Q:  How do I add another DE?
A:  This applies to any DE (desktop environment).  Go into the terminal and use
```
sudo apt-get install [desired environment here]
```
