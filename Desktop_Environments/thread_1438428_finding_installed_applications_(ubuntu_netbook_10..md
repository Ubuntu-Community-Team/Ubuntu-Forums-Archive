### Post by Thwart on 2010-05-14
Had the same problem. Go to System>Main Menu>Applications>Accessories. Check the box for disk usage analyzer and then it comes up in accessories. Even I found these missing application icons bizarre, but it seems that from editing this main menu, its possible to decide which apps are shown/ not shown.
