**Update: Just found the answer on the web...for anyone else looking to add Scroogle. Simply add http://www.scroogle.org/cgi-bin/nbbw.cgi?Gw=\{@}&ie=UTF-8@oe=UTF-8 to Konqueror's search engine configuration file.**
