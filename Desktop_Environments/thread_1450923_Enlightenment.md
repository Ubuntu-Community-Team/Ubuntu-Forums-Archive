### Post by agnes on 2010-04-12
> I want it as the desktop environment, not just the window manager. I have tried MoonOS, but I'm not sure if it uses Enlightenment as the DE or WM.
Enlightenment = WM thus can only be used as a WM.

You could try Macpup Foxy, it's based on Puppy Linux.

openGEU would be the easiest (in the sense it's based on Ubuntu and has some Gnome integration) but openGeu based on 9.10 is not downloadable yet from their homepage.  
In case you might try openGeu based on 8.10, it has an error: the default installed repos-location is gone and you need to download a .deb which fixes that, you can Google for it.
OpenGEU also has packages for ecomorph. I believe after install you have to log off and switch session to ecomorph and then login again, for it to work.
