### Post by lsepulve on 2007-01-22
Hello there,
I've installed "Signature Switch" extension. It works very good.
You have to create simple text files with the signature you want, then, in the configuration, you choose the file and put a name to the signature. 
When you write a mail, it appears an icon who let you choose which signature you want to use .

Very simple and nice.
