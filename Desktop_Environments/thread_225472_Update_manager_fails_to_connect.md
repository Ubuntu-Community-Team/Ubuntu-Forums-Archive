### Post by Ziox on 2006-08-02
seems like there are many people with this problem, try this:
 
open up terminal and make a back up your sources.list file:
 
sudo cp /etc/apt/sources.list /etc/apt/sources.list_backup
 
then open sources.list file:
 
gksu gedit /etc/apt/sources.list
 
after that delete everything. Save and do:
 
sudo aptitude update
 
then copy back the backup:
 
sudo cp /etc/apt/sources.list_backup /etc/apt/sources.list
 
now do:
 
sudo aptitude update
 
see if that works.

### Post by Psiblade on 2006-08-02
I've found a solution that worked for me!

Open a terminal and do "sudo gedit /etc/apt/apt.conf"...

There should be 1 line of text in the file, "Acquire::http::Proxy "false""

Delete that line, save, and it works!

Seems that it's set to that on default installs of Ubuntu 6.06 LTS... because reinstalling produces the same results!
