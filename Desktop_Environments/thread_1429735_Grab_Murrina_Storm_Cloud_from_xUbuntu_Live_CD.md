### Post by jjpcexpert on 2010-11-27
I have a fixed one for any Ubuntu 7.10 or above. In Ubuntu 10.10 (and with all possibility 10.04 and 10.07), the checkboxes appear rounded. So I have fixed the theme to almost have the broken Gutsy look (as far as I can go)) that I and (surely) many others prefer. Included are MurrinaStormCloud and MurrinaStormCloudSilver. You must uninstall prev. versions of the theme before installing this version.
Changes:
 * Square checkboxes (instead of the 10.10 "murdered" look)
 * Rectangular progress bars (instead of the 10.10 "murdered" look)
Otherwise, it is the same as the original. Looks the same as the original in Ubuntu versions prior to 10.04.

BTW, Xubuntu used the original theme, no modifications.
