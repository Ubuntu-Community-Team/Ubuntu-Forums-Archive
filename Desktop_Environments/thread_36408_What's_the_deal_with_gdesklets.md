### Post by darkaudit on 2005-05-23
Psi has been removed from the gdesklets site due to lack of updates. Very little of this series still works with the current gdesklets. I suggest one use SideCandy.

### Post by Xian on 2005-05-23
Some of the desklets in that gdesklets-data .deb file will cause bad freezes and even a re-install of the gdesklet package if you happen to try and run a few of them. Remove it immediately and only use current desklets from the gDesklets website.
