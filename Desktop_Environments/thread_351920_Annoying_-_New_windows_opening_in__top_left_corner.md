### Post by checkeredshawn on 2007-09-12
I had this problem after using Compiz, this is how I fixed it:

Go to System -> Preferences -> CompizConfig Settings Manager

Scroll to the bottom to Window Management 

Enable "Place Windows"



You might also want to enable "Resize Window" and "Move Window" as you may not be able to resize and move windows with those two options unchecked.  

Hope this helped :)
