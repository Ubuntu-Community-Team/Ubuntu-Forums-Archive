### Post by 23meg on 2006-06-14
> 1) During installation, is there anyway you can choose the programs that need to be installed? Cos ubuntu seems to just install everything without leaving us any options. For instances some of us might like to use Opera instead of Firefox, and in those situations installing FF by default wouldn't be helpful.
Ubuntu has a "one app per task" principle and will not let you select which packages to install.

> 2) is there anyway to run kde programs in gnome and vice versa?Yes, whenever you install a KDE app in GNOME the required KDE libraries will be installed as well, letting you run it in GNOME (and vica versa).
> 3) I just installed opera. Now say I want to uninstall it again. How do I go about doing it. Cos it doesn't show up in my Add/Remove application.
Depends on your installation procedure; not sure about Opera.

> 4)I installed XChat through automatix. But now when in Add/Remove application, XChat remains unchecked. How do i get the OS to realize that XChat has been installed?[http://ubuntuforums.org/showthread.php?t=90797](http://ubuntuforums.org/showthread.php?t=90797)

> 5)I love to use the CLI for most applications. For instance I would like to use the XMMS player in cli mode. I want to add songs to playlists, play songs, remove songs from playlists all from the CLI. Is that possible? Cos doing all that thru CLI just looks soo cool...
Not sure about XMMS but there are media players you can use that way; check out MPD for a start.
