### Post by merlyn on 2008-08-19
Thunderbird is available via the Official repositories.

I think the application for installing software in KDE is called Adept, (I run GNOME).

Or in the Command line you can install by typing the following.

```
sudo apt-get install thunderbird
```

and enter your login password when prompted.

Cheers.
