### Post by howefield on 2013-08-22
> **angel mike said:**
> Thanks howefield - how do I put the web link URL into the sidebar list ?
Regards Mike

The sidebar as in the Bookmark bar ?

Go to the page that you want bookmarking, then press the "star" symbol in the address bar, the bookmark will then appear near the bottom of the bookmark bar within the Unsorted Bookmarks link. Drag from there to the Desktop.

Then right click the shortcut, select properties, rename and change the icon to tidy it up a bit.

same problem solved by doing this- go to tools -addons in firefox. then in deskcut preferences under set path type "/home/cmcanulty/Desktop" without quotes and then set radio button to "set path" then click on find. Started recently for me and this fixed it.
