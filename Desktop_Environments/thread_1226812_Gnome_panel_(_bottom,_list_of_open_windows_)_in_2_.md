### Post by mcduck on 2009-07-30
> **credobyte said:**

Yes, just adjust the panel size to large enough and it will automatically split the window list to multiple rows.

(or use smaller font size for the same result without having to make the panel as large. The main point is that the panel height must be at least twice the font size + padding from your GTK theme. In other words to have two rows of buttons there has to be enough space for two rows of buttons. :))

The bad side is that only the window list-applet changes to multiple rows, so increasing the panel size will actually waste quite a bit of panel space as all the other applets become bigger..
