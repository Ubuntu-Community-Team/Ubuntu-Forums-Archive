### Post by lovinglinux on 2011-08-16
Try [deskCut]("https://addons.mozilla.org/en-US/firefox/addon/66/") or [SaveLink]("https://addons.mozilla.org/en-US/firefox/addon/savelink/") extension.

There is also [Drag Tab as Shortcut]("https://addons.mozilla.org/en-US/firefox/addon/drag-tab-as-shortcut/"), which works with tabs instead of links.
