### Post by shat on 2006-10-04
Hey, hope you're enjoying Ubuntu!

First off, a .tar.gz file is a compressed file, and the easiest way to uncompress these is to get into Nautilus (through computer in your menu) and double click on the icon, and click on extract (there is a terminal method for this as well).  From there, you'll have to figure out what you need to compile/build the source code.  You'll probably want build-essential (sudo apt-get install build-essential), and probably some more.  Check the readme and see what it says.  Good luck!:cool:
