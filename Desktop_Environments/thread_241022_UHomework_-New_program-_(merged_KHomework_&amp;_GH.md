[COLOR="DarkGreen"]Description:
UHomework is a merge of GHomework and KHomework. It has a user-friendly interface and loads quickly for you on the go people. UHomework is program with many helpful search engines for homework that include SparkNotes, Wikipedia, Wiktionary, and Google.[/COLOR]

UHomework updates:

[/LIST]

[COLOR="Blue"][SIZE="4"]Any questions, comments, or requests can be posted in this thread and/or sent to [email]jsprojects@inbox.com[/email][/SIZE][/COLOR]

[COLOR="DimGray"]Download here:[/COLOR] [https://sourceforge.net/project/showfiles.php?group_id=175397](https://sourceforge.net/project/showfiles.php?group_id=175397)

To Install:
```
tar -zxvf uhomework-2.0.tar.gz
```
```
cd uhomework-2.0
```
```
./configure && make && make install
```
Now you are done. Type the code below to run the program.```
uhomework
```

Other links: [https://wiki.ubuntu.com/UHomework](https://wiki.ubuntu.com/UHomework)
