I'm sorry, but 6.06 if is newer is because it contains things which are not present or more outdated ina 5.10 installation. This sounds trivial, but it means you can't transform your 5.10 cd in a 6.06. :( What you can do is install 5.10 and then do a dist-upgrade, but I'm not sure this would result in a less heavier download. Ship it is not a solution, also, because you will probably have to wait more than a month for your cd; by that time you will be able to download a new iso. Have you got any friends  which are able to do the download for you? I think this would be the simplest way.
