Ubuntu has a policy of providing *only* bug-fixes and security updates for existing software, so with the default repositories, KDE 3.5.1 will never show up (this is both to ease the burden on the Ubuntu devs/ packagers and to make life easier for businesses, who don't want to receive complaints when the entire UI changes for all their staff overnight! :)).  You'll need to enable the repositories for KDE 3.5.1 - there's quite a few threads on the subject so it's worth searching, but here's the "official" page for it: 

[http://kubuntu.org/announcements/kde-351.php](http://kubuntu.org/announcements/kde-351.php)

There *should* be an entry called "System Settings" on your K-Menu - if you find it, you might want to spend 10 minutes or so just exploring all the options so that you can see what kind of things can be configured under KDE (briefly, the answer is - lots! :)).

Good luck!

Edit:

Be warned - the default Kubuntu install contains several annoying bugs with the System Settings app - you can't enter administrator mode even if you enter the correct password (an update fixed this for me, so make sure you have all available updates from Synaptic!); you can't enable static IPs in the networking section; sometimes the app itself is too big for your screen and can't be easily resized; etc.
