### Post by stalker145 on 2006-08-31
I, personally, use [Acronis True Image Home]("http://www.acronis.com/homecomputing/products/trueimage/") for my needs. It is a bootable disk that creates a proprietary image of your drive, all applications, and settings. It suppports saving the file to a network server or to another partition on your local computer. I am able to create the initial image of a base Ubuntu install in about 20 minutes (saving to a network drive) and restore that image in less than five minutes. 
It also supports incremental backups. So, if you make some system changes and like the changes, you can update the image in just a few minutes. Nice stuff and well worth the cost, in my opinion. I don't know of any FOSS products. Maybe someone else could chime in.
There is also Norton Ghost, but I don't have any experience with that.
