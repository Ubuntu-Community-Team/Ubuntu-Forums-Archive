### Post by steeldriver on 2013-12-21
it would be

```

sudo jockey-text -e xorg:nvidia_304

```

I think - no space between the xorg: and the nvidia_304 part, and nothing after that (the remaining text is just descriptive, **xorg:nvidia_304** is the name of the thing you're trying to enable)

You could also do

```

sudo jockey-text **-d** xorg:nvidia_304**_updates**

```

i.e. disable the current driver (making it fall back to the next available one, i.e. xorg:nvidia_304)
