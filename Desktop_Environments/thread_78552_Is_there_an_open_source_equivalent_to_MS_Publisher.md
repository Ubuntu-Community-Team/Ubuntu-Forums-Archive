### Post by mpettitt on 2005-10-18
Depends what you want to do. If you are just wanting to generate new content, then try Scribus as grj recommended. If you want to import your existing documents, then you are out of luck. The Publisher format is arcane and complex and hasn't been completely decoded by anyone outside MS, so you can't import files that you made in it to anything else. To be fair, the same is true of the Serif PagePlus format as well.

### Post by SickTwist on 2005-10-18
Desktop publishing is still kind of bleak in Free Software land. The only thing I know of besides [Scribus]("http://www.scribus.org.uk/") is [Passepartout]("http://www.stacken.kth.se/project/pptout/") but I don't think that it can import Publisher files either.

EDIT: I just stumbled across an idea. You could install something like [PDFCreator]("http://sourceforge.net/projects/pdfcreator/") in Windows and convert all your .pub files to PDF format. I realize it is not optimal but at least you could view them.
