### Post by shareMenaPeace on 2007-02-08
You can manually backup themes & profiles.

Browse with nautilus to the /home/user folder and hit "CTRL + H" (to see hidden files). Now browse to /home/user/.beryl and  for emerald themes located in /home/user/.emerald.
Copy the content you want to backup or integrate.

To backup, copy the themes and profiles and setting file to another folder eg. in /home/user/backups.
Reload Beryl Window Manager to see the new profiles.
Then you can load them in Beryl manager under General Options -> Profiles or in Emerald Theme Manager.
