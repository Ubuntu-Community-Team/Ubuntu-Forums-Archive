### Post by sokopok on 2008-10-10
Are you sure you didn't make a typo?
You write: ... couldn't read package ap**p**ache2
So it should be: *sudo apt-get install apache2* and not *sudo apt-get install ap**p**ache2*

Another (and maybe easier way is to use a graphical package manager like *Synaptic* or *Adept*. You can find these in the menu under Applications/System.
