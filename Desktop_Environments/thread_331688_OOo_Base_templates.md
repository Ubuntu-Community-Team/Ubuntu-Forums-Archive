### Post by HeresJohnny on 2007-01-09
Hello!

You need to install some additional packages to read Access databases.  If you call up the 'Add/Remove' program and look up OOo Base, the summary gives you some additional packages to install that will enable a bunch of things: read/write Access databases, interface with MySQL, stuff like that.  Copy those package names to a text file, and then search for them and install using Synaptic.  It's not a terribly user-friendly way to have the information, but at least it is possible.  Have fun!!:)
