### Post by nothingspecial on 2010-01-18
You don`t necessarily have to install Xubuntu itself. Try some lightweight Desktop Environments (DEs) within your existing Ubuntu install.

```
sudo apt-get install xfce4
```

will install xubuntu`s DE

another one that might suit you if you have a slow machine is openbox.

It takes a bit of configuring but is well documented.

You can choose between them at login.

You could do the same for other applications such as your file manager. Installing xfce4 will pull thunar in with it, but you could try pcmanfm.
