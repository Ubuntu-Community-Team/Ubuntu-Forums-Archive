### Post by Geekboy on 2007-01-21
I had the missing icon problem too.  I found this on [Ubuntuguide.org]("http://ubuntuguide.org/wiki/Ubuntu:Edgy/AddOnApplications#How_to_install_Diagram_Editor_.28Dia.29") and it took care of my problem.

```
sudo apt-get install dia-gnome
```Then go to : Applications -> Graphics -> Dia

Hopefully that works for you.  If not PM me or send an email!:D
