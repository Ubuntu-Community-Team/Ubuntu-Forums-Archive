### Post by sudo smith on 2012-01-31
Had same problem this is how I fixed it. Go to /usr/share/unity/4 look for the edubuntu icon, it should be somewhere there. If you want the original dash icon go on another pc where ubuntu is installed go to /usr/share/unity/4 look for the icon copy paste it and send it to the pc with the edubuntu icon. You can also get any .png with the right size (for me 54px by 54px). Then open terminal type in " sudo mv (in this spot drag and drop the edubuntu icon from the file) (in this spot drag and drop the new icon you want) " log out and back in and your icon should be changed.
