### Post by PacSci on 2009-04-27
To verify the files, try double clicking them and seeing if File Roller will open them properly. If it doesn't, then the tarballs are corrupted and need to be redownloaded.

If File Roller does open them, though, then it is probably a bug, and you should check Launchpad to see if anyone else has reported this bug, and file it if it hasn't been reported.
