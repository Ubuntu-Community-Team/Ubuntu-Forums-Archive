### Post by mcduck on 2006-01-31
first, you get basic documentation of a command with 'man command', like 'man apt-get'

Then, there are info pages. Same thing, more info. 'info command'

Rest is mostly located in /usr/share/doc/

You can also use the Help (in System/Help), most of man pages and documentation are also there. It's actually useful tool, unlike windows help ;)
