### Post by Harry.k on 2011-10-23
yup. <ctrl><alt><f7> for switching to gui, <ctrl><alt><f1> to switch to CLI.
CLI is quite good once you get used to it.
otherwise, if you continue to get the same error, switch to a more lightweight gui like xfce or lxde
