### Post by grahammechanical on 2021-07-29
The official guide for installing Ubuntu Desktop on a NUC requires a monitor, keyboard and mouse to be connected to the NUC. It may be best to wait until those are available.

[https://ubuntu.com/download/intel-nuc-desktop](https://ubuntu.com/download/intel-nuc-desktop)

A possible alternative would be to network the NUC with the Laptop and run the Ubuntu installer from the laptop and direct it towards the NUC. That may allow you to install Ubuntu but without a monitor, keyboard & mouse I do not see how you can test the installation.
