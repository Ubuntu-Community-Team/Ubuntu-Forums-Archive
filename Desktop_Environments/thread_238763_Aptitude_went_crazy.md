### Post by hugmenot on 2006-08-18
Aptitude keeps track of which packages were installed together with a certain package as dependencies or recommendations. If the one package is removd the other packages are removed along with it to prevent clutter on the system.
When you switched package managers it regarded all those packages as orphans (likely because they were no longer or never depeneded upon; by ubuntu-desktop for instance). 
You just need to reinstall all those packages.
