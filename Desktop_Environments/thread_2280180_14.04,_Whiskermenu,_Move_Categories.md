### Post by Dennis N on 2015-05-28
It's easy to fix:

Right click on Whisker Menu icon, and select Properties. Under Appearance, "Position categories next to panel button" controls which side the categories appear on. Check or uncheck that box depending on how you want it. 

My Whisker Menu is version 1.4.3 and has this option, so yours (being 1.5.0 and newer) definitely should have it also.
