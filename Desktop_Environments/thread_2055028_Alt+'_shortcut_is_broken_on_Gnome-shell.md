### Post by markbl on 2012-09-09
youngunix is right. I hadn't noticed you typed ' instead of `. Actually to be precise, the keyboard shortcut used by gnome (in both gnome-shell and unity) is alt+<key above tab> because the actual char depends on the particular keyboard layout you are using.
