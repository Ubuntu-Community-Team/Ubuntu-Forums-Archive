### Post by localzuk on 2005-03-28
A wireless printer should be just the same as a wired network printer. It has an IP address so should be able to operate just the same.

I have personally (and I know of at least one other person) have been having problems connecting to a shared printer on a windows box, so I would recommend printing directly to it via cups.

The only problems that might occur is the mis-queuing of jobs - in the highly unlikely event that the 2 computers send print jobs to the printer at exactly the same time, it might/will cause an error.

You could try doing it the other way around - sharing it on the linux machine and connecting to the share from the windows box, this would remove that possible problem.
