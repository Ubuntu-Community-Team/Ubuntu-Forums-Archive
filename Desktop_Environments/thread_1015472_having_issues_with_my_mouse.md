### Post by ugm6hr on 2008-12-20
If this is getting annoying, I suggest installing:
[http://www.mousegestures.org/installation/](http://www.mousegestures.org/installation/)

Just turn off all the actual mouse gestures if you don't want them (I did).  It does, however, remove the need to hold down the right-mouse button, since it only brings up the menu when you release the button.
