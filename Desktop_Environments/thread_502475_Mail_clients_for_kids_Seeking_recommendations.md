### Post by iqag1060 on 2007-07-17
Thank you for your answers.

My five year old has gotten good about understanding what's ok to click, and what requires a grown up. My two year old, on the other hand, has a magic knack for finding obscure key combinations which open up destructive menu options and the like.  I really like apps for kids that go full screen and keep them in a little sandbox. 

I would like to set the account settings in a configuration file, fill up the address book with grandparents - maybe with photos attached to keep it absolutely clear, and automatically archive into folders after some time period, so that the kids only have to see the inbox, compose, send/receive, and the address book. While googling I saw someone suggesting that gcompris should have an email app. The same could work in childsplay.  That would be so great. 

These are the more realistic possibilities I'm leaning towards now:
* Desktop apps: Thunderbird, KMail, or Evolution, themed to hide as much of the interface as possible. I'll probably try on KMail - since the 5 year old uses KDE. When that doesn't work, I'll try Thunderbird.

* Web apps: I have SquirrelMail running on my mail server, but I wouldn't want to run a modified version just for the kids. (That might be a great commercial opportunity for someone, though.) I could do something with the user css in firefox. More than this, though, I'm thinking of GMail, for which there are already user scripts like [Super Clean.]("http://gmailsuperclean.wordpress.com") I could hide all of the messy stuff, and just periodically access it from another account to do clean up. With this I get the added benefit of GMail's magical spam filters.

* Command line: I was several years older than my kids are now the first time I saw a GUI. I could set Konsole with big fonts and bright colors and make them use Mutt or worse. Then I could only teach them the commands I want them to know.

I really like the third option, but will probably end up doing the first, but using a gmail account or at least one filtered through gmail.  But if any Python masters jump on that whole adding mail and chat clients to childsplay, I'll do whatever I can to pitch in.
