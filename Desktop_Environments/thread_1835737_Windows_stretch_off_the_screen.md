### Post by mcduck on 2011-08-30
If you hold down Alt you can move a window from any point in the window. And Alt+middle mouse button allows you to resize a window from any point inside the window.

...although windows really shouldn't go outside the desktop area, unless you are running at very low resolution and there simply isn't enough room for the window to fit into.
