The current version is 11.10 (with 12.04 due in April).

The choice of distro (Ubuntu, Kubuntu, etc) will depend on your configuration. Here is a rough analysis:

a) Ubuntu: Highly recommended. Medium-High system configuration required. Challenging interface for Windows users.
b) Kubuntu: Recommended. High/powerful system configuration required. More familiar for Windows users.
c) Xubuntu: Good. Low-medium system configuration. 
d) Mythbuntu: For media center focused usage. I think you can safely ignore this for now. Based on XUbuntu.
e) Lubuntu: Unfamiliar interface for Windows users, takes some getting used to. Runs on relatively low configuration systems as well.

### Post by andrewc on 2012-01-11
To answer your question, Ubuntu uses Unity and Gnome (or just Gnome in earlier versions), Kubuntu use KDE and Xubuntu uses XFCE.

You probably are not interested Mythbuntu, it is basically for using a computer as a dedicated Media Server or Media Centre PC hooked up to a TV.

KDE and Gnome are both full featured, heavyweight Desktop Enviromnments, that provide everything that Microsoft Windows and OSX do and a lot more. Each is very different from the other in look and feel. 

XFCE is lighter weight than the others but still powerful, it is preferred by people who think that KDE and GNOME are bloated, or who dislike certain features of KDE, GNOME or Unity.

Also if you are just trying out Linux it doesn't particularly matter if you are using an old version, but Ubuntu and Kubuntu have changed drastically since 9.04 and both have very different look and feels now, so bear that in mind. Ubuntu now uses Unity and Gnome 3 instead of Gnome 2 and Kubuntu uses KDE 4 not KDE 3.

Whichever version you choose, you should still be able to install the other Desktop Environments in that and try them out as well.
