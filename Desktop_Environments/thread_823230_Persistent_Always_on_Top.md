### Post by Zorael on 2008-06-09
If you're using Compiz this is possible via the Window Rules plugin in ccsm (CompizConfig Settings Manager). You just enable that plugin, go into its preferences, find the Always on Top field, click the + icon, pick an identifier (Title, Class, something), point at your Kaffeine window, accept it and try it out.

As for which identifier to pick, make it whatever turns out with 'kaffeine' (or something *generic* to kaffeine) when clicking your Kaffeine window. Media players like to include the currently played file in the window title, so "title" may be a bad identifier.

If you *don't* use Compiz, I'm not sure I can help.
