### Post by Zimmer on 2008-10-21
file:///home/yourusername/.gnome2/backgrounds.xml

is the controlling file that holds the locations of default wallpaper and files you have added to be wallpaper.

If you added a file to be wallpaper that is in your home/username/pictures folder then if you delete the file it will no longer be available for wallpaper.

Suggestion: create a folder inside the pictures folder called wallpaper and store your favourite personal wallpaper pictures there. Easier to control/delete unwanted items of personal wallpaper.
