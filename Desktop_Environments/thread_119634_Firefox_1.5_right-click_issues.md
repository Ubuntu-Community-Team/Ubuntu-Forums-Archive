I had Firefox "weirdness" after I upgraded to 1.5 that wouldn't display certain external images on web sites, the only cure for it was to rename my profile and let Firefox create a new one (and then I manually moved my Extensions, Bookmarks, Cookies back into the new profile).

The easiest way to test if this will fix your problem is to create a new Ubuntu User Account, log into it and start Firefox, if the problem has gone then it is a profile issue.
