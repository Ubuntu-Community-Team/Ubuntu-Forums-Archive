### Post by TeaSwigger on 2007-10-26
I'd call Xubuntu an alternative to ubuntu and kubuntu, but not a light WM. 

If you want a lighter system but want to retain the advantages of ubuntu / kubuntu, then try using Fluxbox, IceWM or WindowMaker for example, as WM. You can simply install them from the ubuntu repos and try them out. They take configuring by hand to really suit one, but on the other hand that has advantages, and they are definitely snappier.

Or wait for fluxbuntu ( [http://www.fluxbuntu.org](http://www.fluxbuntu.org) ), whenever it's finally released. That's supposed to be configured to be lighter than the existing *buntus.
