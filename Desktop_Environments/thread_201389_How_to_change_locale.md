### Post by vibhu on 2006-06-26
if you are using gnome here are some steps :

- add the applet 'keyboard indicator' to the panel. It will show 'USA' if you have done a default install.
- right click -> Open Keyboard preferences -> Layouts -> Add -> click on India (will give you hindi). 

now when you click on the USA on the applet -> it will change to Ind. Now whenever you type in any window - it will use the hindi charset ( you can get the char map by rt-click applet -> Layout ).
