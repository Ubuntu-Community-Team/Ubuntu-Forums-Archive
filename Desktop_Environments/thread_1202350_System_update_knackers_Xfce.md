I'm using (X)Ubuntu 8.10, and was using Xfce when I allowed it to update. I restarted the machine like it asked, but it froze for ages whilst shutting down.

When the machine eventually restarted and I logged back in to an Xfce session, the task bars at top and bottom of the screen have disappeared.

### Post by Hetor on 2009-07-02
Alt+F2, type "xfce4-panel". Also it's recommended to start a new session.

### Post by michaelc.nash on 2009-07-02
Solved it completely!

It even made it start at log on; I had thought I was going to have to muck around with settings.
