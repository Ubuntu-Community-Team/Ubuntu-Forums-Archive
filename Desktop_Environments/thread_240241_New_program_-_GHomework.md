[SIZE="4"]**[CENTER]This is the main forum thread for K]GHomework[/CENTER]**[/SIZE]

[COLOR="DarkGreen"]Description:
GHomework is an open-source program availible to everyone. It has a user-friendly interface and loads quickly for you "on the go" people. GHomework is basically an HTML file with many helpful search engines for homework that include SparkNotes, Wikipedia, Wiktionary, and Google with Google Calculator.[/COLOR]

[/LIST]

To Install:
```
tar -zxvf ghomework-0.28.tar.gz
```
```
cd ghomework-0.28
```
```
./install.sh
```
Now you are done. Type the code below to run the program.```
ghomework
```
