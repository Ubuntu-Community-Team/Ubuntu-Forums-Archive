### Post by glennric on 2011-11-21
I always create a custom keyboard shortcut to open the gnome terminal and use the command "gnome-terminal --geometry 100x35-0+0" to force both the size and position of the terminal to be what I want.  I also usually modify the terminal's menu entry to make it use this command as well, so that if I open the terminal with that shortcut it will also open that way.
