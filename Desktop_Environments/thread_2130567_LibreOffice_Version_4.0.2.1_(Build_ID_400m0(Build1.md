### Post by cigtoxdoc on 2013-03-30
This problem is SOLVED.  You need to remove all traces od Libreoffice using Synaptic.  Next you need to download tar files directly from Libreoffice download site, tar -zxvf them and follow Libreoffice install direction.  In synaptic, installed version will show as 4.0.1.2-2 instead of 1:4.0.2-rc1-0-ubuntu.
