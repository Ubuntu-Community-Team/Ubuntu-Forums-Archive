### Post by mbeach on 2006-05-03
Perhaps if you edit your /etc/X11/xorg.conf file paying particular attention to the Section "Monitor" area, and use a mode line like:
```

HorizSync 31.5-48.5
VertRefresh 50-70
# 1024x768 @ 60 Hz, 48.4 kHz hsync
Modeline "1024x768" 65 1024 1032 1176 1344 768 771 777 806 -hsync -vsync

```

which a poster at:
[http://www.linux-on-laptops.com/forum/archive/index.php/t-135.html](http://www.linux-on-laptops.com/forum/archive/index.php/t-135.html)
says worked for him on the same laptop I think.

good luck.
