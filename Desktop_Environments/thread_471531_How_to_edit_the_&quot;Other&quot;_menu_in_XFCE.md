Since I have some residual entries in the "Other" menu in XFCE left by already uninstalled WINE programs (but not only that) I would like to know how to get rid of them/move them elsewhere. I've found out that most menu entries can be found in   /usr/share/applications , however this is not the case with the "Other" menu.  Anyone ?

Found the answer, at least as far as WINE applications are concerned:  /home/username/.local/share/applications/wine
