In ubuntu when I downloaded something in firefox I could either double click on the item (in built-in download manager) or click open and it would open it in the default app.

### Post by philipacamaniac on 2005-04-20
When you click on a link to a downloadable file, Firefox asks if you want to open it or save it. I have found that with KDE, if you choose "Open" then you also have to associate a program to open the file with. To open a PDF, you choose "Open" and then select /usr/bin/kpdf. I always say "Remember these settings" so it does it everytime.

When Firefox has an associated program with a MIME type, then even the download manager should open it when you click "open".

Firefox needs to be KDE-ified, or else Konqueror needs to use Gecko.
