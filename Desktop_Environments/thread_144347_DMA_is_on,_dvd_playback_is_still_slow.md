I was having a similar problem and was given this trick to do and it worked like a charm.  Try it and see...

I will say that when dinking around with this fix, my playback problems didn't go entirely away when using the hdparm -d1 command but I went ahead and made the changes permanent and my playback has been flawless since.  Hope it helps.

Ken


Try this:
1. Assuming that /dev/cdrom is the location of CD/DVD-ROM type:
Code:

sudo hdparm -d1 /dev/cdrom

3. Then try to watch your film and see if it solved the problem.
4. To make the settings permanent:
Code:

sudo cp /etc/hdparm.conf /etc/hdparm.conf_backup sudo gedit /etc/hdparm.conf

5. Append the following lines at the end of file
Code:

/dev/cdrom { dma = on }

6. Save the edited file

### Post by rs3 on 2006-05-01
Not sure if this will help...but I was just searching the forums and had the same apparent issue as you.  (DMA on, but still dropping frames/slow)

I wasn't sure what to do--none of the other players seemed to make a difference, so at a loss for other options, I started messing with the Multimedia Systems Selector.  My Default Video Sink was set to XWindows (X11/XShm/Xv)--I changed it to SDL and Totem seems to be playing DVDs just fine now.

I'm quite new myself so I hope I don't lead you astray, but maybe it's worth a shot!  Good luck
