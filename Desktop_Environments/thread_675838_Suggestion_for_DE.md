### Post by nutts4life on 2008-01-23
Since when is a amd 3000 an old PC!

Man... maybe i'm getting old!

One of the most important aspects is RAM. how much RAM do you have?

If it's 512 - 1Gb. Then ubuntu and kubuntu, will do a good job.
If its 512mb or below, then xubuntu is great.

You could go for other linux distro's but that's up to you. Personally if you want support from a very large community then go for a ubuntu based system. 
In fairness xandros does also have a good community as well.
you could consider PCLinuxOS as well.

so let's explain a bit.
If you want easy to use, you will need to modify the users experience. That means you are more interested in the configurablity of the Desktop Envirnoment and the Window Manager.

When i say configurability, you want to set up a easy to use desktop (with icons) maybe a nice menu and good panel of icons, so the user can just click on an icon instead of going through a menu.

KDE, Gnome and XFCE are the most used of these and combine both the DE and the WM in one system.

KDE - is distributed with ubuntu and it's called kubuntu.
Gnome - is the normal distribution with ubuntu.
XFCE - yes you guessed it, it's xubuntu.

People often say that KDE (kubuntu) is a little easier to use if you're coming from windoze. I'm too familar with how set up the user environment (menus etc..) but people have been using it for years and it can't be difficult.

Gnome is similar to KDE in a lot of respects. It's the core environment for ubuntu, so let's not beat around the bush here, it'll be better supported for questions.

XFCE is more light weight than the other two, so it's a little more difficult to setup. But that, said it aint exactly hard.

This is what i would do:
Download the iso for kubuntu, ubuntu and xubuntu (or just get the DVD of all of them) and burn them.
From here you can just try them from the CD, just boot them from the CD.

Have a fiddle with each envrionment, change some menus, panels etc. and see what was the easiest to configure and make it a easy to use system.

Plus, just enjoy it.
