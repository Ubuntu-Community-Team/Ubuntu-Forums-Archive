### Post by dahlek on 2006-11-13
Linux is AN OS in handhelds, but not the MAJOR OS, no. Microsoft doesn't have the PDA market cornered, however, there are many players.

Palm was supposed to be using Linux behind their GUI at some point, I'm not sure where that idea went...

You can install Linux unto many PDA's, including iPAQs which normally come with CE, but that won't help you - that's like isntalling Linux on your PC - CE apps won't install onto your PDA if you've installed Linux.

Don't look for great CE support in linux, however, Microsoft has made this very difficult. SyncCE has some a long way - I can now transfer files to my iPAQ without running Windows98 in a VMWARE session like I used too...

I haven't tried the tools to let you install applications yet from linux which do exist. Most CE apps seem to require being executed in Windows so they can copy the files to the iPAQ where they are installed ](*,) , which is, of course absurd...

I've not heard of any emulators of PDAs that run CE, but who knows, they might exist...
