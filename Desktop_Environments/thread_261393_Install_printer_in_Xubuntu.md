### Post by nwgray on 2006-09-20
Good morning,
   I found this in the archives:

Install gnome-cups-manager! 
Simply run apt-get install gnome-cups-manager, then sudo gnome-cups-manager. It will open a GUI that you can use to add a local or network printer. I added my work printer fine using it. I then created a launcher on the panel so I could open the manager any time I needed.

Please post again if you are having issues or if this doesn't work for you.
