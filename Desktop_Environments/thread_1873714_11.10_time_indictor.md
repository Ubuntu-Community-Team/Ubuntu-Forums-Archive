### Post by Frogs Hair on 2011-11-01
First search in dash and if date and time doesn't appear run the following command and logout and in . ```
sudo apt-get install indicator-datetime
```
The icon should now appear in dash . I think Gnome 3 uses a different indicator , so it doesn't appear for some users after upgrade .
