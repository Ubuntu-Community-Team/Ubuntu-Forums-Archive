### Post by NightwishFan on 2008-05-18
To install themes is mostly intuitive. GDM themes are under System -> Administration -> Login and just install the tar.gz there.

Beryl Emerald themes just import the .emerald file in into the emerald manager.

Gtk 2.x and Metacity archives (this also works for icon themes) can be drag and dropped into System -> Preferences -> Appearances in the first tab, then accessed under the customize option at the bottom.

Some themes are not packaged like so and may be more or less difficult, but those kind generally just involve being extracted to a specific area.

[http://www.gnome-look.org]("http://www.gnome-look.org")
