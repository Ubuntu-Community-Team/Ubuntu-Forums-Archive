### Post by adam.tropics on 2006-07-18
There seems to be a run on Evolution today! (probably why I have moved to Thunderbird!) Anyway, a couple of choices for you to think about. The first, is to check the 'use secure server' setting. For my aim imap account, and this is a first for me, it has to be set to never. So try changing those around to see what works.

Second, and this doesn't fix the problem, more offer a workaround, is to send all your mail by the working pop3 account, and for each account, simply specify the relevant return address. Not a fix, but would work.
