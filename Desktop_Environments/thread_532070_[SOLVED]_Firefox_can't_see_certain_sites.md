### Post by stopsatgreen on 2007-09-02
Can you not see the site at all, or does it just not display properly? If you can't see it at all (ie, it tells you your browser is not supported), you could try the User Agent Switcher:

[http://chrispederick.com/work/user-agent-switcher/](http://chrispederick.com/work/user-agent-switcher/)

This will report to the site that you are using IE6, so you may be able to see it; some sites are just badly written!

Alternatively, you could use Firefox for everything else, and IE6 for the critical website.

BTW, do you use Ubuntu at work? If so, isn't it a bit stupid that you have an IE-only website?
