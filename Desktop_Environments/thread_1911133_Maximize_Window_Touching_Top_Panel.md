### Post by kmand on 2012-01-18
I'm not a fan of the feature that maximizes a window if the cursor drags the window up and touches the top panel. I do it accidentally too much. Is there a way to turn that off (Ubuntu 11.04 Unity)?

---

### Post by Krytarik on 2012-01-18
Install "CompizConfig Settings Manager" if you haven't already, then either tweak the settings under "Grid -> Edges" (the feature that concerns you the most is bound to "Top Edge", obviously), or disable that plugin completely if you wish so.

Regards.
