### Post by guiverc on 2021-08-01
Xscreensaver settings can be found in the manual at [https://manual.lubuntu.me/lts/3/3.2/3.2.20/screensaver.html](https://manual.lubuntu.me/lts/3/3.2/3.2.20/screensaver.html)

The Lubuntu default time for it to *kick in* is a little shorter than I like; so it can be changed, or just ***disabled*** using the "Mode:" selector.

I'd also try changing some of the options, or even using a different screensaver and see if still occurs...  Finally you can of course just disable it.


To get details that may help recognize why; the crash has likely left a .crash file in /var/crash/ that can be submitted (`ubuntu-bug /var/crash/_[filename]_.crash` or examined for clues.. The top of the file is *easier* to understand before it to pure *memory* dump.  If you file more details maybe discernible, but I suspect it relates to your currently *unstated* video card but that's just pure guess.
