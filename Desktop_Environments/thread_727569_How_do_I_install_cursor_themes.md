### Post by BluntBox on 2008-03-18
Assuming you are using Gnome: If you have a theme in an archive, you can drag and drop them onto the Appearance Preferences window. (System -> Preferences -> Appearance). Or click the Install button on the window and find the archive.

You can also extract them and copy the theme folder to either the .icons folder in your home directory, or /usr/share/icons for all users to access.

Then they become available in the Customize -> Pointer tab of Appearance Preferences.
