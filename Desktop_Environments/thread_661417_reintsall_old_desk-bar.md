### Post by chewearn on 2008-01-07
Right-click on the top bar; select New Panel.  Drag the new panel to the bottom.  Right click on the new bottom panel, select Add to Panel.

Select the following one-by-one, click Add button:

Show Desktop
Window List
Workspace Switcher
Trash
