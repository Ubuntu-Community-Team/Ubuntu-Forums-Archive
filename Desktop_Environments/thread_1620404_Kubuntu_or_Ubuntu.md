### Post by adit on 2010-11-13
**Ubuntu:**
The default installation of ubuntu (without any configuration, additional software) with *ubuntu-10.10-desktop-i386.iso* is officially called "Ubuntu".  It should be properly called "Gubuntu".  This contains ubuntu_base_system and gnome
Gubuntu=ubuntu_base_system+gnome

**Kubuntu:**
The default installation of kubuntu with *kubuntu-10.10-desktop-i386.iso* is officially called "kubuntu".  This contains ubuntu_base_system and kde.
Kubuntu=ubuntu_base_system+kde

**Your system:**
Your system contains:
ubuntu_base_system+gnome+kde
ubuntu_base_system is common for all derivatives (kubuntu, xubuntu, edubuntu).  Your system has no official name.  You can call it as *"ubuntu(gnome+kde)"*.

The confusion arises because ubuntu_base_system+gnome is officially named as ubuntu (should be properly called gubuntu).
For kubuntu the official name is OK.
