### Post by Frogs Hair on 2014-11-01
Installing different desktop environments is possible but you'll end up with with some application overlap and more than one file manager depending on the environment . XFCE and the Xubuntu desktop are both in the software center and XFCE is much lighter than the Xubuntu desktop. XFCE is very plain when installed but can be  customized once you learn where things are. As you wrote Xubuntu has much more under the hood. I ran  XFCE  as a separate session regularly until 14.04.
