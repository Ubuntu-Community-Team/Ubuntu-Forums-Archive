### Post by redpants on 2006-09-07
Got a newbie question for you...how do I remove icons from my menu?  

Specifically, I uninstalled Clam Antivirus through the synaptic package manager because it was causing errors when I was trying to install other programs.  My other problems are fixed now, but I still have the Antivirus icon on my System->Adminstration menu and I don't know how to remove it.  The icon doesn't do anything when I click it.

---

### Post by OtubrabNad on 2006-09-07
On the task bar go to Applications -> Accessories -> Alacarte Menu Editor. From there you can navigate to the unwanted icon, uncheck it so it doesn't show up anymore, or right-click and choose delete to get rid of it.
