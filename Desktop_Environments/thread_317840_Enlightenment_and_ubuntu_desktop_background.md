So you are having trouble with gnome keeping control of the background and that can be fixed if you know where the wallpaper file for the e16 background is located and set it while logged in under gnome by right clicking the desktop and adding the picture to the available wallpapers

Because from my impression what you are doing is logging in under e16, setting a background, logging out and logging back in under gnome and the settings aren't carrying over, which is technically what is supposed to happen, an example would be like using firefox to access a website and having it save a password and then attempting to access the same website with dillo and expecting it to also have your password saved  ....  if I am wrong, please correct me and we can diagnose the issue from there.
