### Post by humansuit on 2006-07-12
Hi

I just re-installed mozilla-mplayer plugin for Firefox, and now when I open a link to a mp3-file, mozilla-mplayer opens the file in firefox. I changed the settings for the plugin (right-click in the tab where the plugin is loading the mp3 -> configure) and unchecked the "Enable MP3 Support", but that didn't change a thing. I also looked at the firefox's edit->preferences->download actions but there wasn't anything about mp3-files and no option to add one. I also tried "Mime Type Editor" extension for firefox and added an action for mp3-files to use the system's default action and no luck there.

So, how can I change the default action in firefox for mp3-files so that they won't open with mozilla-mplayer plugin (which works great for other media), but instead let's me choose if i want to download it or perhaps open with rhythmbox?

Thanks!

### Post by Bossieman on 2006-07-12
Try this.
Go to edit/preferences in firefox and choose the tab responding to download files.
Now click "show and edit filetypes"
Find the plugin responsible for MP3 and doubleclick it. There you can choose what program or if you want to download the file.
