### Post by marshall.robert on 2009-12-21
Only if you have something like an Intel graphics card. Cards by Nvidia or ATI require drivers to be installed then a restart, and when you restart on a live cd you lose everything including installed programs.

However, there is probably a way of keeping installed applications and settings with some sort of Live CD/Flash Drive combo.
