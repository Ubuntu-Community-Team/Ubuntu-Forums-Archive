### Post by aysiu on 2006-06-28
Kubuntu comes with Kopete instead of GAIM and Konqueror instead of Firefox. But you can easily install GAIM and Firefox in Kubuntu, too.

There is automounting, but proprietary codecs do **not** come with Ubuntu/Kubuntu. You can install them manually or use a script to install them for you.

Your brother may be better off with Mepis, actually--a KDE environment that's based on Ubuntu repositories. It also comes with Synaptic, Firefox, and a lot of popular proprietary software.
