### Post by HermanAB on 2005-10-12
Hmm, many a savvy administrator keep a Mandriva CD#1 in their toolbox, just for diskdrake...

Basically, you start an install, go to expert mode, partition the disk drives the way you want, format them as required, then power off, instead of continuing the installation process.

The other way is to run qparted, but that assumes that you have a working system to run it on:
[http://qtparted.sourceforge.net/faq.en.html](http://qtparted.sourceforge.net/faq.en.html)
