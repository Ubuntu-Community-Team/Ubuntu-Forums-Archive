> how do i make it look like this [http://youtube.com/watch?v=xC5uEe5OzNQ]("http://youtube.com/watch?v=xC5uEe5OzNQ")

HI!
First, you need to install compizconfig-settings-manager from the repository, you can install it through Synaptic, or just type:
```
sudo apt-get install compizconfig-settings-manager
```
on a terminal.
Then, open it, go to **General Options** and under the **Desktop Size** tab, set the **Horizontal Virtual Size** to 4.
Cheers!
