### Post by powel212 on 2008-11-15
I agree that any Nvidia card should do well but if you want specifics I would recommend Nvidia Gforce 8600 or newer. 

in addition:

I have found that running Ubuntu on my flash drive and sticking it into many different machines with different graphics cards. The default driver that the OS uses is usually good enough. But many times can't enable desktop effects easily. To fix this problem just download $sudo apt-get install compizconfig-settings-manager

Then using this tool should be able to enable desktop effects more easily.

Please note I have not tried this method on any machine older than 2 years old. (most computers of this age won't allow booting from USB).
