By default Ubuntu uses window manager "compiz", which allows for a large variety of visual effects. You can change from "compiz" to a more basic and more resource conservative "metacity" by going to System -> Prefs -> Appearance, then Visual Effects tab and then select "No Visual Effects".

You can also try to adjust "compiz" to use less resources by installing CCSM (compiz config settings manager) from the Software Center and then turn off some of the default compiz effects.
