### Post by dave9191 on 2005-07-16
The icon theme that you set , was set for yourself (ie. the user that you log in with). When you browse as root you are running the program as another user ( ie. root), for which the icon theme was not set. You need change the theme for root as well so that the icons will match up :) 

Dave
