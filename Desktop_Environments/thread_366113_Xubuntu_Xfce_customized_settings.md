### Post by kerry_s on 2007-03-31
The problem is you think xfce4 has a default look, it doesn't, if you want the default look delete both panels, that's it. A stock install of xfce4 is just 1 panel(collapsed) in the upper left hand corner with a single empty launcher on it. You can do a server install with nothing but xfce4 and that is what you will get, 1 single launcher. Xfce4 does not have the mac like setup of the older versions. It is up to the distro and the end user what the look is. That is why if you go to the xfce4 site, you will see many screen shots, but none stock-> [http://www.xfce.org/about/screenshots](http://www.xfce.org/about/screenshots)
