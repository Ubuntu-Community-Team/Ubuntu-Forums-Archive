### Post by Andrew Stone on 2008-02-07
Writing a desktop environment from scratch is not for the faint of heart.  If you've got a cool feature that you'd like to see in one, I'd recommend adding it to one of the environments already out there.  Choose something lightweight like Xfce or IceWM so you aren't deluged with massive amounts of code.  Learning an existing env will also tell you how to do it from scratch.
