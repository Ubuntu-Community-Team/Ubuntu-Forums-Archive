### Post by invalid on 2005-11-29
```
sudo apt-get install kubuntu-desktop xubuntu-desktop
```
This will install both the other desktops you want.

During the installation, it will ask you which login manager you prefer (kdm or gdm). You can use either one you like. When you restart X now, the login manager will have extra options in the "session" area where you can select which desktop you would like to boot into. You can also set a default desktop here if you like.
