### Post by Jucato on 2006-07-25
"Themes" in KDE cover a large area of customizations. As of now, there aren't any complete theme that would include **everything** that can be themed. There are .kth (KDE Theme) files, but this is just a file that basically says "use this window decoration, this icon theme, this (widget) style theme, this color scheme, etc.) and would only work if you have those corresponding "themes" installed, too. You can find most of these .kth files from [http://www.kde-look.org](http://www.kde-look.org) and they would probably mention what other themes you need to install. Remember, .kth files are just like a sort of "configuration" file which tells which styles/themes to use.
