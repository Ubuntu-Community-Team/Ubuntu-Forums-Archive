### Post by warp99 on 2009-08-04
Right mouse click on the screenlets choose Window > Sticky to keep the screenlet always available on every desktop. If that doesn't work you can install the CompizConfig settings manager and use the "Windows Rules" plugin to exclude the screenlets application from minimizing.
