### Post by MeNtuxRoKinU on 2010-02-16
As far as running faster is concerned, if you were running Compiz on a regular machine (not virtual), and now are not, this is most likely the cause. Compiz takes up graphics resources, which in turn can slow the computer down slightly. I could be wrong.

Regarding the visual effects in a virtual machine, forget about it. You need to be running a native installation to utilize the graphics card. Tuff luck, I know because I ripped countless thousands of hairs out of my head trying.
