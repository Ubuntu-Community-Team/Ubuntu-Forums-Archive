### Post by highatrisk on 2007-04-30
You can go into the config file in the style directory, and edit the size of just about everything. It should be fairly easy to read (i found it so). Just change the numbers, decreasing what you want. It takes some practice but it's worth it because then you know how to edit ANY flux style to your personal tastes. For example, almost every style I use has a titlebar that is just way too wide for my taste, so I generally change all of those. Hope this helps.
