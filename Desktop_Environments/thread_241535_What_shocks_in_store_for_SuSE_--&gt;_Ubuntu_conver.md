### Post by mchatel on 2006-08-22
I'm not too familiar with a couple of the items you mentioned, not having used them myself, so I'll let someone else more knowledgeable comment on those.  Iptables is definitely included with Ubuntu though.  That is a standard thing.

Probably the biggest adjustment people have to make with Ubuntu, coming from other Linux distributions, is that Ubuntu does not have a 'root' user account, setup by default.  Instead, Ubuntu tries to train people to use the 'sudo' command, to attempt 'root' user actions.  This seems to initially throw some people off, but one gets used to it very quickly.  I actually like the idea.  You can browse the forums, or the Ubuntu docs themselves, for more info on this particular difference.

Ubuntu is a fantastic Linux distribution.  :)
