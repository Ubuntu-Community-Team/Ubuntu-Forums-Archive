### Post by mvaniersel on 2006-04-05
Is there a keyboard shortcut to open the Gnome menu? I'd like to do as much as possible with the keyboard only, but without being able to open the Gnome menu I can't get very far.

If there is no shortcut by default, how can I assign one?

---

### Post by vloris on 2006-04-05
Alt-F1 is the default shortcut, but you can configure it through System->Preferences->Keyboard Shortcuts, look for "Show the panel menu"
