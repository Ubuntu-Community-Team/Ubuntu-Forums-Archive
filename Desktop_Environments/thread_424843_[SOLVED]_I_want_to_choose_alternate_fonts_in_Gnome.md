### Post by airtonix on 2007-04-27
**choosing font: **
[LIST=1]
[*]*system -> preferences -> font*
[*]click to choose font.[/LIST] 
**to install new fonts : **
[LIST=1]
[*]press alt+F2 then type :
>  gksudo nautilus
[*]type your password
[*]type the following in the location field : 
> fonts:// 
[*]drag and drop new fonts into this folder[/LIST]
