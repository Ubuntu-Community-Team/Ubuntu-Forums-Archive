### Post by Rubi1200 on 2011-08-01
Hi and welcome to the forums :)

If you are trying to install with Wubi and are getting error messages, try the following:

Download the correct Desktop ISO version and matching wubi.exe and place them both in the same folder before running the executable:

[LIST]
[*]For 10.04 use [http://releases.ubuntu.com/lucid/](http://releases.ubuntu.com/lucid/) 
[*]For 10.10 use [http://releases.ubuntu.com/maverick/](http://releases.ubuntu.com/maverick/) 
[*]For 11.04 use [http://releases.ubuntu.com/natty/](http://releases.ubuntu.com/natty/)
[/LIST]
If you have dynamic disks in Windows, then trying to install Ubuntu whether with Wubi or regular is inadvisable.
