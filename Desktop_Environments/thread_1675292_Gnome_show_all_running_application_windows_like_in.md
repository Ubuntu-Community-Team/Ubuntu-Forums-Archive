### Post by Krytarik on 2011-01-26
I'm not sure if I got you right, but maybe the "Scale" plugin of Compiz is what comes near that feature: "System -> Preferences -> CompizConfig Settings Manager -> Window Management -> Scale". You can bind it to a screen edge. If you choose "Initiate Window Picker For All Windows", it will show you all (not minimized) windows from all workspaces.

You may try this custom script to make it show all, even (before and after) minimized windows, though it may be a bit unstable and cause a temporary resources load peak, I haven't tried it:
[http://ubuntuforums.org/showthread.php?t=976002](http://ubuntuforums.org/showthread.php?t=976002)
