### Post by 3Miro on 2012-02-12
By default, Gnome 2 comes with a bunch of libraries and utilities and Metacity windows manager. The default Ubuntu 10.04 comes with all of Gnome 2 except Metacity is replaced by Compiz WM. Metacity is available as "no desktop effects" mode.

By default, Gnome 3 comes with libraries and utilities and Gnome-shell interface. The default Ubuntu 11.10 comes with Gnome 3 except Gnome-shell was replaced by Compiz + Unity interface (Unity is mostly a plugin for Compiz). Gnome-shell is available in the Software Center. Ubuntu 12.04 will come the same way, however, it will come with updated versions of Gnome 3, Unity and Gnome-shell.

On theory you can install Gnome 3 on Ubuntu 10.04, however, this will require way too much change into your system and will probably kill any benefits that an LTS brings (like stability). Just wait for April, or if you don't want to wait, install Ubuntu 11.10. For me 11.10 is buggy, but only until I install the updates, then it runs just fine. Depending on your hardware, 11.10 may be either better or worse, can't tell unless you try it.
