### Post by 2F4U on 2011-07-27
You probably need to install the proprietary ATI drivers for Unity to work correctly. Since this cannot be done during the normal installation process, you have to do it afterwards with the Additional Drivers tool.
If you don't like to install non-open source, you can choose another flavour of Ubuntu, for example Kubuntu, Xubuntu or Lubuntu.
