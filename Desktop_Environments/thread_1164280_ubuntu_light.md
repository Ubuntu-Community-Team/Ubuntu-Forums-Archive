### Post by snowpine on 2009-05-19
The "lightest Ubuntu" is a command line only install using the minimal cd: [https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)

From there, you can install any additional packages you need. Many users who aren't comfortable with the command line 100% of the time like to install a lightweight windows manager like icewm. Here is a good tutorial on that: [http://www.psychocats.net/ubuntu/minimal](http://www.psychocats.net/ubuntu/minimal)
