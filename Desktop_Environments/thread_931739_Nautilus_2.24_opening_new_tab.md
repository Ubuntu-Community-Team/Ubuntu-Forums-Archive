### Post by KOld Iron on 2008-09-29
As already said, key codes are like the ones in Firefox:

New tab: Ctrl + t
New window: Ctrl + n

The other key codes are pretty much all shown in the menu. Here the more important one:

Switching between tabs: Ctrl + PageUp or Ctrl + PageDown

Changing view mode: Ctrl + 1 is Icon View, Ctrl + 2 is List View and Ctrl + 3 is the new Compact View
