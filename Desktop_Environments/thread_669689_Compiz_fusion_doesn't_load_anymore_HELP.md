### Post by Toadinator on 2008-01-16
Hmmm... sometimes in Compiz Fusion I've noticed that it quits unexpectedly. One thing I'd try is to download CompizSwitch from [www.compiz-fusion.org](www.compiz-fusion.org). Just click on Planet at the right of the screen and look for Compiz Switch somewhere in there. Just download the .deb file and install it with the debian package manager (it should be in the main applications menu, but not shown by default). After it's installed, it should appear as an application on the applications menu somewhere. Then, right click the launcher and add it to the top or bottom panel, so you can switch compiz on and off just by clicking!

If that doesn;t work, then install the Advanced Desktop Affects Manager (if it's not in Add/Remove programs, look in synaptic). With that you an customize all of the affects of Compiz Fusion. Launch it from Preferences. The main problem may just be that Compiz Fusion has been switched off, hence the Compiz Switch. You may also want to try uninstalling all of the regular compiz packages; that may help.
