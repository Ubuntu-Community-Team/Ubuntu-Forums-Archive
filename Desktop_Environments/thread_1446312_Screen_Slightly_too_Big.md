### Post by Millamber on 2010-04-03
I have an Asus P5Q-EM with the Intel G45 Chipset.
 X 
I am using the onboard video.

It is connected to a 50" Plama and outputs via HDMI to an Onkyo Reciever which then outputs to the Plasma.

The only resolution available to me is 1920 X 1080 which is ok because that is the correct resolution.

My problem however is that the windows when maximised are slightly to wide and tall for the screen.  So I end up not being able to see the top/bottom menus in programs that are running in full screen mode.  Also the side menus are cut off.

Is there a way of manually adjusting the width and height of the screen without changing the screen resolution ie make it slightly shorter top and bottom.

I had this same issue when running windows, but windows gave me the option to manually decrease the width and height of the screen.

TIA

Millamber

### Post by Millamber on 2010-04-04
Found the solution, it was actually a setting on the tv, which was a bit obscure.
 
The setting on the TV was called PC Overscan.  I switched this off and all is as it should be.
 
Hope this helps someone else.
