Though, mouse input works fine inside of Windows' GUI. So, I don't suppose anyone has any tips on how I can normalise how VMWare translates mouse input to the guest operating system?

Although this was more a little experiment for myself than anything else, I think it goes to show that VMWare Server is already a stable product.
