### Post by bullgr on 2006-08-27
try another brand... it's very strange sometimes how the burner react with some blank cd-dvd brands...

for example in my burner i can burn the sony dvd+r blanks but not the sony dvd-r (exact the same brand only differs in +r and -r)

and ubuntu use different chipset drivers than windows.

i can only burn dvd in 2x speed in windows because the chipset driver (installs only in windows, no flashing bios) don't work well with my burner.

in ubuntu i can burn dvd in maximum speed (4x) without any problem

strange, is it?

so, try to update the firmware of your burner (use only official firmware from the manufactures website) and try some different blank cd-dvd brands
