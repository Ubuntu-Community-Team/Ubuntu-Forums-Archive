### Post by LewRockwellFAN on 2011-01-09
gucharmap will do this. I'm pretty sure you can edit what it contains too. So maybe if you want to strip it down to contain just the characters you use you could.

On the command line type
gucharmap
and press the enter key.

You can of course put this in a script and make a launcher for it if you like.

If you want to use onboard (how on earth do you manage to read the horrible ui with onboard?) maybe you could make a script to change the keyboard layout and launch onboard and another to change back. I bet if you look around you can find some program that will give you a hot key to toggle between two keyboard layouts. If onboard takes it's cue from the keyboard either of those approaches should work. For that matter, if you find a keyboard toggler, maybe you won't even need to use onboard if you can remember which keys mean what in the foreign layout.

### Post by LewRockwellFAN on 2011-01-15
Maybe you could use a key remapper so that, for instance, you could set contrl-e to give you that character that looks like an e with an extra part, alt-e would give you another e-like character, and so on. This wouldn't be QUITE as mnemonic as a French Onboard but shouldn't be too hard to remember and would be faster for most typists since you wouldn't have to use a mouse. I've used this approach in Windows, but not yet in a 'nix, for a similar problem and it works quite well from a writing perspective.

### Post by grahammechanical on 2011-01-15
Have you tried installing a different keyboard layout.

System>Preferences>Keyboard>Layouts.

It should give you a keyboard icon to use to switch layouts. I use it to switch between English (GB) and Greek. I only use the physical keyboard not an on screen keyboard. This might be what you want.

Regards.
