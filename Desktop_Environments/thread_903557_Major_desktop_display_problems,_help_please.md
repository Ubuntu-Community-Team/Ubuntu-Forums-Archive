### Post by jim_mule on 2008-08-28
sounds like a problem with the monitor settings. like it has a bad refresh rate horizontal/vertical sync. you probably set it up using your friends monitor and then it configured itself for that monitors refresh rate horizontal/vertical sync stuff. though i am a little unsure since you say the login screen is readable/working. while your in safe mode try bring up the 'System --> Monitor Settings' program and change the settings to the resolution you want to use, log out, then log in using non-safe mode and see if it works.
