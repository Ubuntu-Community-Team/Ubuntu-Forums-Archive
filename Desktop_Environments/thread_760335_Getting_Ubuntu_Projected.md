### Post by spupy on 2008-04-20
What is your video card? If it is ATi, installing the fglrx driver (that is the proprietary one - the one you download from ati.amd.com -> graphic drivers), gives you a nice configuration tool. With this tool it is really easy to set up a projector. I did it once for a presentation, only needed a restart.

But for other graphic cards - no idea!
