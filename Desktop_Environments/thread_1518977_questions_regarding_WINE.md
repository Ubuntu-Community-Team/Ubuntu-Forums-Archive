### Post by mkelliott21 on 2010-06-27
One of the neat things about Wine is that you can load as many or as few "programs" as you want in "C" drive.  As the above post suggest, some programs may "install" just fine, while others are a nightmare.  Wine maintains a database of programs that work and rate how well they work in Wine.
 
There is a program out there called CodeWeavers that is a really good program.  [www.codeweavers.com]("http://www.codeweavers.com").  Alot of the configuration that you need to do for Windows programs is done for you.  And it doesn't cost much.  Also, CodeWeavers has a "games" flavor to.  
 
You can spend alot of time and hair configuring all this ...  but again, CodeWeavers is the best I have used for Wine
