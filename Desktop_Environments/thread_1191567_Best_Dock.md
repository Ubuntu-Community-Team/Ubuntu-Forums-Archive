well essentially its a 3 horse race.

avant window navigator

cairo dock

gnome do

customizability goes to cairo dock.
versatility goes to gnome-do , however it's superfluous in kde.
avant is a basic good looking dock.

for my recommendation, I recommend installing gnome-do
```
sudo apt-get install gnome-do
```

Learn to use it [http://do.davebsd.com/wiki/index.php?title=Main_Page](http://do.davebsd.com/wiki/index.php?title=Main_Page), and in the settings turn on Docky.

### Post by bronkeydain on 2009-08-11
Awn for panel, and gnome-do to launch applications directly (not in docky mode).
AWN is very sexy looking and very easy to get going.


To install AWN type:
```
sudo apt-get update
sudo apt-get install avant-window-navigator
```

To install gnome-do
```
sudo apt-get update
sudo apt-get install gnome-do
```
