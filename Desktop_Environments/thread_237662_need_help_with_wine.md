### Post by 505 on 2006-08-16
Open Nautilis and go to your home folder. Press Ctrl+H to view hidden files. There should be a directory called '.wine'. In this dir a folder 'drive_c' this is your 'Windows C Drive' You'll see a Windows dir and a Program Files dir. Normally programmes are installed in the Program Files dir. Double click on a .exe files and select wine to open the file.
