### Post by bazuka on 2004-12-08
You can go to Computer > System Settings > Configuration Manager

In the tree, go to Apps > Nautilus > Desktop and check the boxes next to what you want to see on the desktop.

### Post by kamstrup on 2004-12-09
You can also right-click the panel and add a user defined launcher. It should have the command set to "nautilus --browser".

It mixes well with the Ubuntu "Computer" menu which uses spatial Nautilus. Quick acces to browser mode while preserving superfast acces (without icons on desktop) to Documents and Computer places.
