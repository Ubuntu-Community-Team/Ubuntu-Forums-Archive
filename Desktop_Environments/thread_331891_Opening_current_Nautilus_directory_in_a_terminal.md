### Post by mattG on 2007-01-05
While navigating a directory structure in Nautilus I often have the need to open a terminal window to perform some operation in the directory I am currently viewing in Nautilus.

Does anyone know a comfortable way to do this? (An option "Open terminal in this directory" would be perfect). Right now I always navigate to the directory "by hand"...

Thanks!


Matt

---

### Post by ildfroe on 2007-01-05
what you need is:
```
sudo apt-get install nautilus-open-terminal
```
And then restart.

This will give you the option when you right-click in nautilus
