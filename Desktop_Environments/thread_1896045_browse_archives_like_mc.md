### Post by LewisTM on 2011-12-16
If I understand correctly, you want a GUI file manager that will treat archives like normal directories and open them in the file list pane (not in a separate application)?

Two choices:
Krusader (my favorite, needs lots of KDE stuff)
Tuxcmd +tuxcmd-modules (old but light, doesn't work with 7z or RAR)
