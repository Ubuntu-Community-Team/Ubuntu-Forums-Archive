### Post by jwh400 on 2006-05-30
You'll need to create a root password for yourself so you can log-in and make the necessary password  changes to your user account. Here's a good tutorial on how to create the root password;

[https://wiki.ubuntu.com/RootSudo](https://wiki.ubuntu.com/RootSudo)

Scroll down to "**Going back to a traditional root account**" 

Once you reset the user password log out of root and back in as user. Use the root account only when absolutely necessary and for no longer than necessary.
