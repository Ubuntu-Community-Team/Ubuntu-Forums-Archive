### Post by mcduck on 2010-07-08
I doubt that there's any thread that could tell *everything* that can be done, as many cool effects are done by creatively combining features from many Compiz plugins, but anyway you'll find a list and descriptions of each plugin in the Compiz Wiki: [http://wiki.compiz.org/CompizFusionPlugins]("http://wiki.compiz.org/CompizFusionPlugins")

### Post by mcduck on 2010-07-08
Compiz and the main plugins package are installed by default, but if you want more plugins you can install the compiz-fusion-plugins-extra package: [compiz-fusion-plugins-extra]("apt:compiz-fusion-plugins-extra")

Of course you might want to install CompizConfig Settings Manager as well to be able to easily enable/disable the plugins and adjust their settings: [compizconfig-settings-manager]("apt:compizconfig-settings-manager")
