### Post by mcduck on 2009-10-15

> Is there any way to use multiple colors for differnt folders
for example green-music
            Blue-docs
            violet-pictures
or can there be only one standard folder color icon in use at one time.

You can set any individual directory's, item's, device's or program's icon to whatever you want. Just right-click the item and select "Properties", and then drag the picture you want to use as icon to the icon you see in the upper left corner of the "Basic"-tab.

You can also simply lick the icon in that dialog and browse to find the image you want to use.

Personally, I prefer using emblems for this purpose instead. :)
