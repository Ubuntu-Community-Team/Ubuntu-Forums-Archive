There will be unofficial support for Gnome3 on 11.04 Natty, but since [Unity](https://secure.wikimedia.org/wikipedia/en/wiki/Unity_%28desktop_environment%29) took over Natty has been shear chaos between gnome and ubuntu devs...

It was available through 2010 until the fallout with ubuntu and ppa madness. 
(Hence this method no longer works for anything other than Natty)
[http://digitizor.com/2011/04/07/install-gnome3-desktop-ubuntu/](http://digitizor.com/2011/04/07/install-gnome3-desktop-ubuntu/)

If you want to try it, get a LiveCD of another distro ( [http://www.gnome3.org/tryit.html](http://www.gnome3.org/tryit.html) ) or install VirtualBox and break ubuntu inside that first. :)

### Post by Jedcurtis on 2011-04-15
```
sudo add-apt-repository ppa:ubuntu-desktop/gnome3-builds
sudo apt-get update
sudo apt-get install gnome3-session
```The above is what I did to get it installed.  HOWEVER, within 30 minutes I am now using Unity for the first time.

Gnome3 looked OK and booted in fine.  When I ran any app though my mouse and keyboard failed to have any effect on the open app.  No matter what it was.  I ended up having to close the app using the icon on the top panel telling it to quit. (i.e., clicking the icon representing the open app that appeared in the top left panel, then clicking quit in the resulting "balloon menu" that popped up)

I've been using Unity now for the better part of an hour I would say and so far so good.  After installing it through Synaptic, I rebooted, chose the "Ubuntu Netbook Edition" and logged in.  Of course I'm assuming (fingers crossed) that the above really means that your booting into the "Unity Desktop" environment.

I'm anticipating an upgrade to Natty and wanted to "preview" Unity before switching.  Maybe in a couple days I'll post again about how I like it. (Unity, that is)

Currently running Ubuntu 10.10 Maverick Meerkat on a laptop with mostly the latest/greatest for hardware.  Linux RULES!!!
