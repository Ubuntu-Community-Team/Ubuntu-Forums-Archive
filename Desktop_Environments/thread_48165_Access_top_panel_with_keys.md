### Post by varunus on 2005-07-12
ALT-F1 will open the applications menu, and you can then use the keyboard to navigate through all 3 menus.  You can configure a different keyboard shortcut if you like (some people like the Windows key), and also many other keyboard shortcuts in System->Preferences->Keyboard Shortcuts.

Enjoy.
