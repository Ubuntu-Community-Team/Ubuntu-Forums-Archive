### Post by jchau on 2006-08-13
Update Breezy to the newest packages available for Breezy first (or just get the newest update manager for breezy).  The latest update manager should have a button for updating from Breezy to Dapper.  You should probably look through the update forums before updating though.  You can avoid some problems that way.
