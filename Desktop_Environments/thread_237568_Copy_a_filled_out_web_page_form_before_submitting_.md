### Post by amac777 on 2006-08-16
Is there a way to copy or somehow save a filled out web page form to a file before submitting the form?

I have tried selecting the full web page, and then copying and pasting it into an openoffice document or gedit document but the data in the input boxes of the form is not copied. I have routinely used this functionality in windows (using both firefox and IE) and it works there to copy and paste filled out forms into Word. Basically, I see all the form html text including the data in any text boxes of the form all pasted into the document. However, in Ubunutu, using both firefox and opera, I can only get the text of the webpage to copy and paste but can't get the data that was entered in any form input boxes to copy and paste.

To recreate this problem, go to google.com, type in a search query, for example: "firefox linux copy paste form data", and then try copying and pasting the full web page into openoffice. The search query text you typed won't be copied but all other text will be.

As another approach, I tried saving the web page before I submitted the form, but the data I entered in the form was not saved with the page. Same problem. 

I did some google searches and even though it seemed like a long shot I tried installing the AllowClipboard Helper Extension for firefox and allowing the website I want to copy the filled out form from but that didn't do anything for this problem either.

Anyone have any suggestions?
