solved!

Under the beryl settings manager item "rotate cube" and the mouse tab first entry which is "initiate" the default setting is control + alt and button number 1.  this setting does not work or conflicts on my machine.  on another menu which has a "grab key" function I fond that a windows flag key on my keyboard translates to "Super" - I chose that option with mouse button 1 and now it works - press the super key and left click then you can hold the left button down release the super key and move the cube manually.  Attached are screenshots of the menu item and my rotated cube.
