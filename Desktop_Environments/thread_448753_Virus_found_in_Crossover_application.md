### Post by keithk50 on 2007-05-19
I recently installed the demo of Crossover [http://www.codeweavers.com]("http://www.codeweavers.com").   I used the demo to install itunes & Quicktime(windows version 4.9).  I then ran the Aegis Virus Scanner and it found this, quote:  " File is infected!  The file /home/keith/.cxoffice/win98/drive_c/windows/system32/advpack.dll is infected with the W32/Magistr.a@MM virus."  
I deleted the file then ran a google search which came up with this:[http://vil.nal.com/vll/content/v_99040.htm]("http://vil.nal.com/vll/content/v_99040.htm")
To be on the safe side I have removed crossover and the applications from my system.   I should have known better really as I moved over from Windows a couple of months ago because of the amount of time spent just keeping such viruses and such at bay.   Full time job really!    This post is not a plea for help but thought I would warn those who wish to install windows based apps on their systems.   Once bitten twice shy in my case.
It proves another point though, in that Aegis did its job well.   I also have Avast installed which I run from the command line.   This failed to pick up the virus.
Due to my lack of experience with Ubuntu it would be good to hear from those that know of such things.  For instance can a windows virus infect a Ubuntu system once it is on the PC?
