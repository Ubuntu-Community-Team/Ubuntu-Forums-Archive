### Post by nad on 2005-04-23
If you had accepted updates that required a restart, they would obviously not have taken effect till a reboot.
This could explain why you now have a problem.

Is the nVidia video driver the correct one to use? You may wish to save this file and issue the command ' dpkg-reconfigure xserver-xorg ' to see what the automatic configuration file includes.
