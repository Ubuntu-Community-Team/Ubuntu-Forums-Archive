### Post by ktulu1115 on 2006-06-09
VMWare works quite well - obviously as stated earlier, no 3d acceleration support, but if you install it in a virtual disk, you have an option to easily back it up since it's just a file on your drive.

However, it's *possible* to do both - if you install XP on a seperate partition, you could set up a dual boot....  then install & configure VMWare to have direct access to that physical paritition/drive it's possible to have your cake & eat it too.  The obvious advantages are that you can have hardware support when booted natively in XP.  The only caveat is that you need to have dual hardware profiles set up in XP (physical & virtual) and tell it which to want to use when you boot.

I saw a reference to it on VMWare's website somewhere...  this may help - [http://www.vmware.com/support/reference/common/hardwareprofiles.html](http://www.vmware.com/support/reference/common/hardwareprofiles.html)
