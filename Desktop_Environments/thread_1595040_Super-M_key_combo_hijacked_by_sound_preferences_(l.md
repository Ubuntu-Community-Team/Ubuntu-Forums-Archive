### Post by Krytarik on 2011-04-13
Those options are set in "System -> Preferences -> CompizConfig  Settings Manager -> Negative". Sometimes those key combinations  overlap with those of other apps. For me, Super+m sometimes opens the menu of the  messages envelope instead:
[http://askubuntu.com/questions/13751/stop-superm-from-opening-messaging-menu](http://askubuntu.com/questions/13751/stop-superm-from-opening-messaging-menu)

Here is a possible workaround:
[http://ubuntuforums.org/showthread.php?p=9402497](http://ubuntuforums.org/showthread.php?p=9402497)

Note: The package the the post is referring to is in the official repos.

Another workaround would be of course to just change the shortcut for the Negative plugin.
