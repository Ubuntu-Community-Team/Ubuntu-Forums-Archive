### Post by vanadium on 2018-10-27
I assume you are using standard Ubuntu with the Gnome desktop. You cannot open/close the workspace selector using a hotkey. The workspace selector is expanded as you hover the mouse above it, closed when you hover away. In the latest Ubuntu 18.10, the behaviour has improved. As soon as you are using multiple workspaces, the workspace selector will remain open all the time, providing for a more "stable" experience. You can implement that behaviour in previous versions using the Gnome Shell extension "Expand Workspaces Switcher". If you would prefer workspaces to be expanded always, the extension "Always zoom workspaces" is available.

Based on your question, "Workspaces to Dock" might suit you. This makes a Dock from your workspace switcher, that you can display any time from your desktop without having to go to the overview.
