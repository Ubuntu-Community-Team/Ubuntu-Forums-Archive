### Post by racecat on 2005-08-12
To upgrade to 3.4.2, insert the following line into /etc/apt/sources.list:

deb [http://kubuntu.org/hoary-kde342](http://kubuntu.org/hoary-kde342) hoary-updates main

Thanks to Itman for this one.

Then just run Kynaptic - Refresh Package Information From Repositories.

Hope this helps,
