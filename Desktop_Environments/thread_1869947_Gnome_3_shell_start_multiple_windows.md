### Post by crdlb on 2011-10-26
There are none that I know of. There are a number of ways to get a new window though. You can Ctrl+click, right-click or long-click to get the context menu, or drag the launcher and drop it anywhere on the desktop area. Also, middle clicking will open a new window on a new workspace.
