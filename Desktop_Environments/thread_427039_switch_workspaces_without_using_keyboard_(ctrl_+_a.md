### Post by dboot on 2007-04-29
My desktop rotates (with Beryl) when I scroll with my mouse on the desktop. It's useful and doesn't get annoying. 

I also use AWN. When an application is on another desktop, I can select it in AWN and the desktop will rotate to the application. I love it and feel like I can actually use my multiple desktops. If you look around the beryl settings you will find many many ways of customizing it.
