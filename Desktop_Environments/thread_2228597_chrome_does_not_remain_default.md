### Post by jeff68 on 2014-06-08
I am new to xubuntu, but have used an lubuntu derivative (peppermint) for about 9 months. That had chromium as default browser so never I had an issue with my preference for chrome/chromium over firefox.

With Xubuntu, I am having trouble getting chrome to remain default. If, for example, I go to menu and click on "xubunu website" I get a pop-up: choose preferred applications saying "please choose your preferred web browser". I choose chrome and it opens the page in chrome. Then if I close chrome and reopen it, it says "chrome is not your default browser". 

I can click to make it the default and if I then just use the browser and not any outside links, chrome remains the default when i close and reopen it or restart the computer. So that much is working as it should. 

Initially chrome was not showing up as an option at all in the preferred application list so, based on this:  [http://askubuntu.com/questions/307842/why-does-chrome-not-stay-as-the-default-browser-in-xubuntu-13-04](http://askubuntu.com/questions/307842/why-does-chrome-not-stay-as-the-default-browser-in-xubuntu-13-04)
I clicked on "other" for the preferred application and put " /usr/bin/google-chrome-stable %U" there. I found other similar suggestions for that but none have kept chrome as the default, currently it is using:  /usr/bin/google-chrome "%s"
