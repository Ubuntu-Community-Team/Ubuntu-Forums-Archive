### Post by aysiu on 2007-04-08
There's nothing you're missing. The settings Gnome hides are things other people care about. Clearly you don't care about those things. Use Gnome.

If, for some strange reason, you still want to stick with KDE even though you don't like it, you can turn off the bouncy icons by following [these instructions](http://ubuntuforums.org/showthread.php?t=113941).

There's a weather display applet, too: ```
sudo apt-get install kweather
``` I'm sure there's a search applet, too, if you bother to look. A quick repositories search found *kicker-applets* and *strigi-applet*.

It appears to me you'd rather bash KDE than actually get to know how to use it.

P.S. Let's keep this as a support thread. If it turns into a KDE v. Gnome argument, I'm just going to lump it in with [the KDE v. Gnome thread.](http://www.ubuntuforums.org/showthread.php?t=190994)

### Post by lexxonnet on 2007-04-09
I assume you mean things like configurability in KDE. Well, you just need to look in the right places :)

I love things like the shortcut manager. I can assign shortcuts for pretty much eveything. Each application can have global shortcuts(even if its minimised, think play/pause in amarok) and normal shortcuts(when the application is being displayed). And this exists for every application, and can be configured, quite uniformly(always under settings)

Another thing I like is the fact that shortcuts can be made to call DCOP calls. This can be done with Input Actions in the Control Center.

Another thing I like is that there is a unified interface for notifications. The screenshot will explain more :)
