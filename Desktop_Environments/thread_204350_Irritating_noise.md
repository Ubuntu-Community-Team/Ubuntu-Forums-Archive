### Post by kpkeerthi on 2006-06-26
I presume you are talking about system beeps. If you prefer to get rid of it in the Terminal, open terminal, from the menu, Edit -> Current Profile -> General tab -> Uncheck 'Terminal Bell'. 

Sorry this is for GNOME only.There should be a similar option in KDE.

### Post by kpkeerthi on 2006-06-27
Yep. It can be done. Again this is GNOME specific... but should work in KDE as well... I guess.

System -> Preferences -> Sound -> System Beep tab -> Uncheck 'Enable system beep'
