### Post by Sef on 2006-05-01
> Is it possible to use icewm panel with gnome desktop? 

Both Icewm and Gnome are window managers. At log in click on session to choose which window manager you want to run.

To download Icewm:

sudo apt-get update

sudo apt-get install icewm-desktop
