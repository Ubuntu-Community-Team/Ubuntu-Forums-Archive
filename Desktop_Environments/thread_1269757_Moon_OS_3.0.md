### Post by meditatingfrog on 2009-11-19
I ran MoonOS in a Virtual Machine.  I was so impressed with it, including how well it ran in the VM, I decided to download and compile the enlightenment desktop manager.  I'm currently using enlightenment e17 as my default windows manager.  I can see that it has greater potential to rival the likes of MacOS X, or even Windows XP for that matter.  It just seems a lot more responsive to me than Gnome and KDE, and it has plenty of eye candy without the need for additional graphic support (compiz doesn't currently work with Intel GM960 graphics reliably for me).  

This all having been said, I can tell there is still work to be done.  It isn't ready for primetime like Gnome, at least, not in my hands it's not.  I haven't really figured out how to configure it entirely.
