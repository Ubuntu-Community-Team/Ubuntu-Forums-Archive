### Post by MacMan on 2006-09-20
> **Quickdood said:**
> Does anyone know to enable this function in K3B or is it default?

I'm not on my Kubuntu system right now so I can't be very precise, but you can burn a gapless audio CD in K3B by setting the "pause" of every track to 0. If I remember correctly, it's a just a matter of right-clicking on a track, selecting "Properties" from the contextual menu, and then setting "Pause" from the default 2 seconds to 0.

Do this for all the tracks but the first and you're OK.

I hope this helps. If it doesn't, let me know, I'll try to be more precise.
