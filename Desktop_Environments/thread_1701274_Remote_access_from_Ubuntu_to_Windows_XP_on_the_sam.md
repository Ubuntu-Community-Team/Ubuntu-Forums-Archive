### Post by cchhrriiss121212 on 2011-03-06
You need an OS to be running in order to have remote access to it. And you can't run Ubuntu and Windows at the same time on the same machine.
Possible solutions:
If you need access to files, mount the Win partition and browse with Nautilus
If you need program functionality try looking into virtualbox or WINE

If you explain what exactly you want to achieve, people can post more helpful solutions
