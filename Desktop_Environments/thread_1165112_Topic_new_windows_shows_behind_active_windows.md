### Post by jbo5112 on 2009-05-20
The solution will depend on your window manager, whether you are using Compiz, kwin from KDE 3.5 or kwin from KDE 4.2.  I'm guessing you're running kwin from KDE 4.2 (default on jaunty), but I can only check the specifics for KDE 3.5 at the office.

In KDE 3.5, open up the control center.  Click on desktop, then window behavior.  Under the advanced tab there's an option at the bottom for "Focus stealing prevention level" that should adjust it.  KDE 4.2 might have something similar, but it seems to be missing a lot of the extra configuration settings that used to enjoy.
