### Post by doclivingston on 2005-08-05
If you only use a single user account on your machne, you can install fonts by opening "fonts://" in nautilus and dragging then there (it installs them just for your user).

There is probably a nautilus extension that will add "open as root" (or something) to the context menu, although I haven't actually looked for one.
