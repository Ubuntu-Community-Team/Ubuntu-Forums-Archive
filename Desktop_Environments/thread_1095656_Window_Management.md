### Post by mannheim on 2009-05-06
If you are using compiz ("Desktop Effects") then you can do tiling.

You need to first install some compiz plugins by installing the package compiz-fusion-plugins-unsupported and the package compizconfig-settings-manager.

Then launch the CompizConfig Settings Manager from the Preferences menu, and enable the "Tile" plugin.

This applies to Ubunut 8.10, and I don't know about 9.04.
