I like the default 12.04 wallpaper and fancied using it in kubuntu but found I cannot use it.

I copied it over and went to set it as wallpaper but just get a black screen.
Tried opening it in the image view and it said the image cannot be loaded.

So I went into ubuntu and it was there as normal so I tried copying it to a shared partition but again, as soon as I booted kubuntu I cannot do anything.

Of course, I can just google it and download but I would like to know why it isn't viewable in kubuntu.

### Post by Krytarik on 2012-05-14
Weird enough, the "warty-final-ubuntu" wallpaper is used to be a JPG image with the actual file extension being "png" - for whatever reason. #-o Just change its extension to "jpg" and you should be fine. :D

Regards.
