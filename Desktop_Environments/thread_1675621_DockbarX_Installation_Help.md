### Post by kerry_s on 2011-01-25
dockbarx is a gnome panel applet, to use it in xfce4-panel you need to install "xfce4-xfapplet-plugin" then add that to the panel & select the dockbarx applet, just a reminder you can use more than 1 xfapplet, but be aware that xfce4 applet is iffy with some gnome applets.
