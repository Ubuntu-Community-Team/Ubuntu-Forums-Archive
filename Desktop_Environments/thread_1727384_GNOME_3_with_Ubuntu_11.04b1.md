### Post by Deadboy420 on 2011-04-12
I don't know who or what site keeps referring you to git locations but you know you can still use synaptic or ubuntu software center in 11.04b w/ Gnome 3 to add packages.  Gnome tweak tools is already in the repositories. Just search synaptic "gnome tweak" and it's listed as "gnome-tweak-tool". 

As far as git, I haven't used it often enough to explain it's full capabilities but basically you install "git" and use it to copy packages from git repositories to your computer. Here's one helpful link: [https://help.ubuntu.com/community/Git](https://help.ubuntu.com/community/Git)   Use google for more information.
