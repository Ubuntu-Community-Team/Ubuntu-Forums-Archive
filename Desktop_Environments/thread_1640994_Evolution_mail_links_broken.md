### Post by Tamlynmac on 2010-12-08
Just a thought....
I'm wondering if Ubuntu (for some reason) isn't identifying Firefox as your default browser. This can be checked by:

Firefox/edit/preferences/advanced tab/general tab/system defaults/Always check to see if Firefox is the default browser on startup/check now button

If Firefox isn't seen as the default browser, it will not open urls linked by another app. Potentially, this could occur if you tried a different browser and it was set as the default browser.

Good Luck & hope this helps
