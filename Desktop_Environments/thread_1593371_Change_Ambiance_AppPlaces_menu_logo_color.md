### Post by ubunterooster on 2010-10-11
Right click the following picture in firefox. select "save image as..." Save it as Ulogo.png (or *whatever*.png)
[IMG]http://openbox.org/images/distro/ubuntu.png[/IMG]

Download and install Ubuntu-tweak [http://ubuntu-tweak.com/](http://ubuntu-tweak.com/)

go to applications>system>ubuntu-tweak>gnome settings>click this button to change menu image logo 

find the location of the image you downloaded and select that.

 You may have to log out for changes to take effect
