### Post by Xerak on 2006-03-02
Hello,

Since most people post "problems" here I felt like saying something about the greatness of Ubuntu. 

I'm a total Linux noob but I'm a PHP/MySQL programmer and a big client has a Linux server running Debian. I noticed how much time I wasted uploading files after files to test my stuff (how much time am I wasting writing this?). I decided to install Linux on my laptop and work locally until the project was over.

Gosh, what an adventure. I first tried Ubuntu because I liked the philosophy behind it, and it was a descendant of Debian, so closer to the stuff my scripts are going to be running on. I first installed Ubuntu and then everything was working okay (beside me screaming out of anger every 5 minutes because I didnt know anything about Linux). But I could not install MySQL 5 correctly. The "mysql" user wasnt created properly and creating it manually wouldnt work.

So I gave up on Ubuntu and went with Debian. Darn, Debian wouldnt even recognize my wireless network. So I gave up and went to Mandriva. It was okay, but sometimes my laptop would hang when shutting down and at one point all menus (Gnome) disappeared, and updating the menu engine didnt bring it back! So I was left with an empty screen, looking with puzzlement at my "few days" attempt to get into Linux.

Well, it was 3am that night when the menu disappeared and I decided that I was going to try Ubuntu again (my first "love") or I would give up and would go back to developing on Windows XP.

I did have the same problem with MySQL 5 but this time instead of giving up, and having a little bit more knowledge with Linux, I decided to try to compile MySQL instead (even though it is not "recommended"). Well, after compiling MySQL (took like half an hour) it freakin' worked! It was running on my machine. So I installed Apache and PHP 5, a PHP editor, new version of Firefox and I was good to go!

Ubuntu is the only distribution that recognizes I got a laptop (must recognize the chipset) so I got those nice little battery icon, and also Ubuntu is the only distribution who recognized my wireless chipset right away. So, to me, Ubuntu looks very solid, more "plug&play" than most Linux distributions, and is good lookin'! 

Please feel free to flame me for taking up tech support space, but I really needed to share my joy. Damn, now I even know about "init.d" and "apt-install" and "alien", isnt that cool?
