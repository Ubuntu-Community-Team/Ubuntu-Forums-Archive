### Post by ardchoille on 2006-02-10
1) You can drag a window in openbox by holding down the ALT key while dragging with the left mouse button.

2) When I resize a window, there is a small box that appears in the middle of the window that shows the window dimensions.

Also, there is a graphical menu editor for openbox, it's called obmenu and can be found here: [http://obmenu.sourceforge.net/](http://obmenu.sourceforge.net/)
