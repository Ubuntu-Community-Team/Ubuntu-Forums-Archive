### Post by aysiu on 2006-06-13
A Synaptic search for *expose* came up with Skippy: > **full-screen X11 task/window switcher, similar to OSX Expose**
Skippy provides an alternative to taskbars or regular task-switchers, by showing full-screen view of all the open windows on the current desktop. You can navigate among the windows and select using the mouse or keyboard. Skippy will work with any NETWM or GNOME WM Specs compliant window-manager.

If you are using X.org's Xserver with the XDamage, Xcomposite, and XFixes extensions, you probably want skippy-xd instead. (they will be packaged in debian when X.org is available).
