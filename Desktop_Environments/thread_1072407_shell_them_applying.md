Open your gnome-terminal. Go to **Edit --> Profiles...**

Check the button New, give a name and create it. Then choose the profile and edit it as you please!

Put a picture as backgroud, make it transparent,change the font.... whatever you want. Sava the changes, and select the new profile in **Terminal --> Change Profile**


[CENTER]Enjoy![/CENTER]
