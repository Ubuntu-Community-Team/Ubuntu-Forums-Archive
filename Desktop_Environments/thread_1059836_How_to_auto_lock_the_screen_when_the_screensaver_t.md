I have migrated from Windows for about 3 months. In Windows, when the screensaver starts, if the user interrupt and want to use the computer again, he has to enter the password if the screensaver runs for a user-specify time.
I cannot found this feature in xfce and there seems to be no key combination (under Windows is Win+L) to lock the screen. Because the computer might have the chance to be accessed by other people, so I am seeking the key combination similar as Windows and the feature that the authentication is requested when the screensaver has taken effect for a while.

The solution I am using now is
```

$ xflock4

```

I prefer the key combination and auto requesting for authentication feature.
