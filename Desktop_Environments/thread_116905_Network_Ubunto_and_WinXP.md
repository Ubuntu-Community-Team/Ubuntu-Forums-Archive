### Post by dcast on 2006-01-13
you could use a crossover cable which is just like a normal ethernet cable but is designed to go computer to computer with no hub/switch/router/whatever in between. However both computers would need a second ethernet card. If they are already networked together you shouldn't have a problem setting it up.
