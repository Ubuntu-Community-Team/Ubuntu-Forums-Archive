### Post by tom4everitt on 2010-04-12
There is a package called thunderbird-3.0, its probably as you get through the repos (i'm not sure if its 3.0.4, or if it is slightly older). You can install it by running

sudo apt-get install thunderbird-3.0

in a terminal (for some reason it doesn't show up in synaptic/software center). 


Your other option would be to download it from their website. Then you will get a compressed file (a tar.gz) which you extract. In the extracted folder you will find a file called thunderbird, which you double click to launch thunderbird. I would recommend the first option.
