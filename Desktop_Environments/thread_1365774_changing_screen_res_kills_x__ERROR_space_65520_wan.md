### Post by &lt;mike&gt; on 2010-08-08
When you get to the login screen, try switching to a new tty (ctrl+alt+f2), logging in to the text interface, and either editing or deleting your monitor config file:
```
rm ~/.config/monitors.xml
```
nb: I am not sure this file exists in Lucid. 
then log out
```
exit
```
and then switch back to the graphical session (ctrl+alt+f7) and try to log in again.
