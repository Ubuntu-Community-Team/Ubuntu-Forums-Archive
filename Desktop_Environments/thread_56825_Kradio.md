[http://linux.about.com/b/a/253229.htm](http://linux.about.com/b/a/253229.htm)
"KRadio is a piece of software that allows you to add a radio to your Linux system, assuming it has a radio card and satisfies a few other requirements.
KRadio provides a convenient user interface to select radio stations and perform other basic radio functions. Advance features include recording options and "timeshift playing" where you can press the pause button when you get interrupted so that you don't miss anything. The audio stream can be recorded in MP3 or Ogg/Vorbis format.
The software comes with a database of station preset files. Its dynamic plugin library architecture makes it possible to extend the system to new radio types like internet radio. "
