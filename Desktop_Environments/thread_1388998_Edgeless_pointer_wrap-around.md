### Post by zerubbabel on 2010-01-23
Is there any equivalent for Linux to the "Edgeless" software for Windows that lets the mouse pointer wrap around from one screen edge to the opposite edge? That is, if you move the pointer off the screen to the right, it comes back in on the left side.

### Post by wantime on 2010-05-27
Try using Brightside:  
```

sudo apt-get install brightside
```

It works to extend Metacity in Gnome.  I had it working with Compiz as well.  It basically lines your workspaces up and then switches workspace instead of staying in the same workspace.   Ie, mouse goes over left edge of your first workspace and then reappears on the right edge of your last workspace.

Other than this you could use another window manager such as fvwm2 where you can set the EdgeScroll parameters.

Any other solutions?  I'm interested.
