### Post by benguin on 2006-03-29
Hi there,
Try this:

```
 sudo apt-get install libc6-dev 
```

Then reconfigure and try building it again. Should work.

The problem apparently lies here:
```

configure:2376: gcc conftest.c >&5
/usr/bin/ld: crt1.o: No such file: No such file or directory
collect2: ld returned 1 exit status

```

Installing the libc6-dev package should resolve that linker problem.
