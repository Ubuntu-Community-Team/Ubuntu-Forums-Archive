### Post by PowerBarry43 on 2012-10-17
install CCSM (Compiz Config Settings Manager) by running

```
sudo apt-get install compiz compizconfig-settings-manger compiz-plugins-extra
```

then run by typing in a terminal

```
ccsm
```
