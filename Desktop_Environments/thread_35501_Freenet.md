### Post by jobezone on 2005-05-27
I've used 2 years ago, so my experiences are from that time.

First, I wouldn't use ubuntu's or debian's packages, as they probably are outdated. I would recomend you downloading from the site.

The only catch at the time I used was that when first running freenet, I had to leave it running for a few hours alone, I would even recomend about 6 hours before trying to use it, since it took some time for my node to be recognized and be well inserted inside freenet.
I think still today, If you go right away and try to acess freenet pages, or files, you'll get frustated by all the waiting and non availability of things.

After that, I started geting sometimes huge, and I mean huge, download rates. There wasn't as much files to download as there in other networks, and there was also some really questionable content there. There also some interesting "ilegal" documents that could be found at the time, like Scieontology's high level books.
It's main features are anonimity, and not so much file sharing. For these features, it worked pretty good, especially using the proxy feature, which gives a world wide web view of freenet, through which you can acess many webpages for rants, files, documents, etc.
