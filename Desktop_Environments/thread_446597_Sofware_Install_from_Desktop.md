### Post by llamakc on 2007-05-17
You can install it from right there. Before you do so, what is the software? Have you checked the repositories for an Ubuntu version of what you want to install? 

You need to bunzip2 the archive first though.

Open a Terminal.

cd ~/Desktop

bunzip2 NAMEofarchive.bz2
