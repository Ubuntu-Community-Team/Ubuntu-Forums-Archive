### Post by Aztek on 2008-04-03
1. Good

2. To get some of the cool features you only have to enable them since they actually come built-in to 7.10. Go to Settings>Preferences>Appearance and you can choose the extent of the visual effects. I believe that Beryl is now merged with compiz to make Compiz-fusion, which is included with Ubuntu. Search for compiz-fusion in Synaptic for more details and some apps to help you manage the visual effects.

3. For  WoW you will probably want to use Wine [http://www.winehq.org/]("http://www.winehq.org/")

4. I know there are widgets to do this but I'm not sure of the name.

5. Google "Gnome Themes" (Ubuntu uses the Gnome desktop environment by default). gnome-look.org is a good site. Go to GTK2 themes and download one you like. Then go back to Settings>Preferences>Appearance and go to themes. Click on Install and navigate to the downloaded file. You can also install compiz themes but I'm not sure how.

6. I don't know.

Yes gaming is possible but in my opinion not worth the effort.
See cedega [http://www.transgaming.com/products/cedega/]("http://www.transgaming.com/products/cedega/")

 I dual boot with windows for gaming.
