### Post by kosmic on 2005-05-08
To change the login screen do the following :

1- go to [www.gnomelook.org](www.gnomelook.org) and chosse a GDM them you like.
2- Go to System->Administration ->Login screen setup (sory if I'm wrong with the names but my menu is in Portuguese)
3 - choose the 3rd Tab
4 - click install new theme
5 - choose hte theme you have donwloaded and voila, new login screen
