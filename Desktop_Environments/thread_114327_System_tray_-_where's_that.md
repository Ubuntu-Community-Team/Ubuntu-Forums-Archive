### Post by dcraven on 2006-01-08
GNOME has a "Notification Area" applet that can be placed anywhere on any panel. Right click on a panel, chose "Add to Panel", and select the notification area from the resulting dialog.

It is worth noting that the default Ubuntu configuration has a notification area already added to the top panel on the left. The idea is that all programs that add an icon to the notification area appear in that spot. If you have removed this area, you will need to add it again as described above.
