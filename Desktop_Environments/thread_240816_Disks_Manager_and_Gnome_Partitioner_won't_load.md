### Post by dannyboy79 on 2006-08-21
This is has happened to me before also. What I did was shut everything down, un-plug my power cable from my comp. Then I opened her up and double checked power connections, made sure all jumpers and ide cabling was correct, even changed from using cable select, to master and slave jumpers cause I wasn't sure if the end of an ide cable is the master or the slave so if you jumper the hard drive, the cable doesn't matter! Anyway, I also tried to make sure that if you are using a power cable splitter if youhave many hard drives or fans etc that you don't put all hard drives on one power cable, I did a fan and hard drive and a cd drive for example. Since hard drives are always spinning I figure they take up the most amount of power. Then I buttoned her back up and booted up and everything was fine with both disk manager and partition manager. Hope this helps!
