### Post by Untitled_No4 on 2009-11-06
When you install kubuntu-desktop it doesn't affect your booting at all -- it just changes the default usplash theme to the kubuntu theme and that's why you see the Kubuntu splash screen.

It's once you log into your system that you get to choose which desktop to use. Your login screen should have a few buttons there, one of which will give you several options, including choosing whether to log into Gnome or KDE. 

If you have changed the theme of your login screen then the button might not be there since some themes don't include it, so change back to your default GDM theme. Since I haven't used Gnome for a while I don't remember what is how is that button labelled, but it should be there.
