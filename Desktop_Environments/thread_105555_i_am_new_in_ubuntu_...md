### Post by Sutekh on 2005-12-20
Why do you think need a universe cd? 

You won't need to download every package, only those you want.

You can use [Synaptic]("https://wiki.ubuntu.com/SynapticHowto"), the package manager in Ubuntu, to download specific packages when you want them.  There are literally thousands of packages available, so using Synaptic is far more efficient.


I did find a web-page that has created DVDs with the universe/multiverse packages on them.  Due to the thousands of packages in the repositories it would be a very large download of two DVDs, almost 9GB.  

In my opinion this would be a waste of a download, because most people will only use a small fraction of the available packages.  Also it won't take long before the packages on the DVD are outdated, or have newer, better versions available.  This is why the online repositories method makes more sense.

If you really want to go this way, here is the link.  Keep in mind that there is no official support for universe packages, so there is definately no guarantee about these DVDs
[http://cargol.net/~ramon/ubuntu-dvd-en]("http://cargol.net/~ramon/ubuntu-dvd-en")
