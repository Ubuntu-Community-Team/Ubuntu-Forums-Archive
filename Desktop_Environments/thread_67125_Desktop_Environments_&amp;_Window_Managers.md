[QUOTE=dbott67]
1. What's difference between the various WM's and why would  I want to change it? (generally speaking --- are some more customizable, better performing, taste better, fewer calories)

Here is a site with review and full descriptions on almost every different Windows Manager.
[http://www.plig.org/xwinman/](http://www.plig.org/xwinman/)
[QUOTE=dbott67]
2. Can Gnome use a different WM other than Metacity & if so, how do you change it?
[/QUOTE]
With Gnome 2.8 you could switch to the Sawfish manager manager inside of Gnome, but I'm not sure if that is still true with Gnome 2.10+
[QUOTE=dbott67]
3. Some WMs (fluxbox, blackbox, etc.) can run independent of the DE.  Can all WM's do this? For example, can someone run Metacity without Gnome?
[/QUOTE]
I think Metacity is too dependent of the gtk libraries to be used separate of Gnome.
[QUOTE=dbott67]
4. How can one check to see what WMs are installed and which one the system is currently running in the Desktop Environment?
[/QUOTE]
Do a search in Synaptic for the term manager or window manager.

[QUOTE=Mike Basinger]With Gnome 2.8 you could switch to the Sawfish manager manager inside of Gnome, but I'm not sure if that is still true with Gnome 2.10+

I still use the Sawfish window manager in stead of Metacity in Ubuntu 5.04. It's much faster and it has one theme I really like (tastes better).

I don't exactly remember how you could change it, but I think I just killed-all metacity and started sawfish in a terminal and logged out while saving the settings. There's also a howto in the Tips & Tricks forum to run Enlightenment under gnome. So it is definately possible to run other WMs than Metacity under Gnome. Most WMs I know run perfectly without Gnome.

### Post by shakin on 2005-09-20
[quote=dbott67]
1. What's difference between the various WM's and why would  I want to change it? (generally speaking --- are some more customizable, better performing, taste better, fewer calories)
[/quote]

Different needs: speed, light weight, features, look and feel, etc.

[quote=dbott67]
2. Can Gnome use a different WM other than Metacity & if so, how do you change it?
[/quote]

Already answered. There are good how-tos about it. Generally, it's best to keep with the default or you will likely run into a few weird problems. Sawfish replaces Metacity pretty well, though.

[quote=dbott67]
3. Some WMs (fluxbox, blackbox, etc.) can run independent of the DE.  Can all WM's do this? For example, can someone run Metacity without Gnome?
[/quote]

Yes, Metacity can run without Gnome. [Here's some geneal Metacity info](http://xwinman.org/metacity.php). Since Metacity isn't really meant to run by itself, it's probably not going to fill your needs.
