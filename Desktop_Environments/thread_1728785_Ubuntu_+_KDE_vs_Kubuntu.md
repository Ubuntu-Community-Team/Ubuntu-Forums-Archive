### Post by marl30 on 2011-04-14
Well, they are just desktops environments, so if you remove Gnome and install KDE it should work the same. There may be some minor config issues though. For instance, if you had Rhythmbox by default automatically launch when you subscribed to a podcast, when you switch desktop you may have to configure Amarok with Firefox, whereas if you had installed Kubuntu, Amarok would have opened podcasts without needing to configure with Firefox.

As for running multiple DE, the only issue you'll run into is app crowding, and different apps from different DE setting themselves as default. 

So it's always better to just install the version which has the DE you prefer. You won't have to worry about config issues.
