### Post by augur80 on 2007-03-01
> **js_anoop said:**
> HI,
  I am using ubuntu Dapper 6.06. I am trying to install Beryl in my pc. I have never seen how beryl look like. I followed many how-to's and still cannot make it work.
 Now my problem is that when i try to run beryl, the following errors are shown:
****************************************************************
 beryl: GLX_EXT_texture_from_pixmap is missing
beryl: Using non-tfp mode

** (beryl-manager:7067): WARNING **: Beryl caught deadly signal 11

****************************************************************


Try modifying your xorg.conf to look like this in your graphics driver section:

Section "Device"
	Identifier	"Intel Corporation 82845G/GL[Brookdale-G]/GE Chipset Integrated Graphics Device"
	Driver		"i810"
	BusID		"PCI:0:2:0"
	VideoRam	32000
	Option		"UseFBDev"		"true"
        Option "XAANoOffscreenPixmaps" "true"
        Option "AllowGLXWithComposite" "true"
EndSection

See if that helps... could also try adding:
        Option "AddARGBGLXVisuals" "true"
        Option "DisableGLXRootClipping" "true"

These options should appear in the same graphics driver section of the xorg.conf.  Let me know how it goes.
