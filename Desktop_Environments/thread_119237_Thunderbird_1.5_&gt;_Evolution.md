Thanks! I hadn't know WHAT exact file to import before. I found out it was the "Inbox" file. This should work.

To import an Inbox:
/home/$user/.thunderbird/PROFILE/mail/MAILSERVER/inbox

To import a folder:
/home/$user/.thunderbird/PROFILE/mail/MAILSERVER/Inbox.sbd/FOLDERNAME
