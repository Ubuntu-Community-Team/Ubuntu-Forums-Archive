### Post by ajgreeny on 2015-02-20
Do you have the system set to save sessions and use the last one when you start.

I think U Studio uses xfce and if so go to the Sessions and Startup in settings-manager ->Session tab and click on "Clear saved sessions".  Make sure you don't have anything running when you logout next and also make sure the "Save sessions for future logins" is not selected on the logout dialogue box.

Hopefully that will clear everything and give you a clean start next time.  If not come back and we'll dig a bit deeper.
