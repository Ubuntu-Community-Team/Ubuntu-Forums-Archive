### Post by ugm6hr on 2009-08-31
If by toolbar you mean the panels, then this was a recognised bug.  Although I haven't had a problem since upgrading to 9.04.

Anyway, if you press Alt+F2 and type:
xfce4-panel

That should return them.

If a restart wipes them out again, just add that command to the startup sessions list.
