### Post by StonePiano on 2010-04-20
I'm using Ubuntu 9.04 and Evolution 2.26.1. 

When someone sends me an email which they composed with coloured text, it all displays as black text on my screen.  If I look at the email source, I can see tags to specify the desired colours.  But in my evolution program, it all displays with black text.

Does anyone know how to fix this?
(I've tried both 'Human' and 'Clear looks' themes without a resolution.)

Many thanks,
Stone

---

### Post by mcduck on 2010-04-20
Go to Edit/Preferences/Mail Preferences, and on the HTML Messages-tab make sure "HTML Mode" is set to "Show HTML if present".
