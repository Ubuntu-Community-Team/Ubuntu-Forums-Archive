### Post by mojoman on 2007-10-13
I'm not familiar with ATI as I use nVidia and TwinView but the same problem can arise with TwinView as well. There, the login (at least when one uses GDM) always seem to be on the left screen so the TwinView fix is to add the following line in xorg.conf

```
        Option "TwinViewOrientation" "RightOf"
```

Now, I'm pretty sure that TwinView only works for nVidia cards this fix will not work for you but there might be some equivalent of this option for ATI cards. That should fix your problem unless you for some reason need or want the projector screen to the left of your monitor. In that case, you might want to consider using slim as a login manager. It's fairly good and you can edit the configuration to place the login anywhere you want. However, slim has, to my experience serious shortcomings when it comes to choosing sessions (as in it don't work) so if you're using multipe desktop environments or window managers slim is not a good substitute for GDM.

You could have a look at these threads, they might help. 

[http://ubuntuforums.org/showthread.php?t=301951&highlight=twinview+howto]("http://ubuntuforums.org/showthread.php?t=301951&highlight=twinview+howto")
