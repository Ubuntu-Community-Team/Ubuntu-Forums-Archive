### Post by mali2297 on 2008-03-28
**Mod4+Enter** or right-clicking on the root window (desktop) will open xterm. Then type
```

firefox &

```
or whatever. That should get you started.

In addition to keyboard shortcuts, **dmenu** is a nice tool to combine with awesome window manager.

(EDIT: Mod4 is usually recognized as the *Windows* key.)

### Post by mali2297 on 2008-03-29
> **Chris. said:**
> how do I restart my computer while in Awesome?

To reload your configurations, press **Mod4+r**.

To quit awesome, press **Mod4+q**.

To reboot the computer, open a terminal and type
```

sudo reboot

```
To turn off the computer,
```

sudo poweroff

```
