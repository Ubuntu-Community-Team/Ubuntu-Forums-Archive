### Post by wd5gnr on 2009-03-12
I installed the "Magic Folder" widget and got a big red X telling me it couldn't find a python scripting engine (this is 8.10 x64 by the way). 

No problem. Open up the package manager and installed phyton-plasma. That did the trick. Until I restart the system. Then I'm back to the big red X. However, if I remove the magic folder and then put it back, it works until the next restart (well, probably until the next log out/log in but I rarely do that). Of course, I lose all the settings so it is more than just annoying. 

Anyone else run into this?

Update: Odd. The key is that you have to set up some file types in the magic folder. If you don't (which makes it useless) it doesn't work. If you set up file types it works until you reboot and then you get the big red X.
