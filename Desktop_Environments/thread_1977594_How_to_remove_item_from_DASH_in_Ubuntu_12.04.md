### Post by Oldhacker on 2012-05-10
It would be very helpful if I knew how to remove an item from being visible in DASH using Ubuntu 12.04

For example, I keep a private file in TrueCrypt.
It makes it too easy for an attacker to know the name of a recently acessed file instead of pouring through the thousands of files on the computer.

Can this be done, and if so how?

Thanks

---

### Post by Enigmapond on 2012-05-10
In the Privacy part of System Settings you can tell it not to record a certain app or folder. So just add TrueCrypt and the folder(s) with the TC volumes in them to the blacklist. Then erase all activity. Erasing will remove it from Dash and the privacy setting will keep it from being logged. I'm guessing this will work.
