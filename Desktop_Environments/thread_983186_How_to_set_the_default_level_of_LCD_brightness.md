I am not sure what kind of hardware you have. but    	 	 	 	 	 	  I have installed ubuntu 10.04 on emachine E627 laptop successfully.It is a 64 bit machine. It has AMD T20 processor and Ati randon 3200 graphic card. I have successfully set the default value for brightness. Now when ever I turn on my laptop....brightness is 60% default value not 100%
 PROCEDURE:
 SYSTEM>PREFERENCES>ATI Catalyst control center>Color.>adjust the brightness
 

 Reduce the brightness as required and click on Apply button. This value will be set as default value.:P
