### Post by arizonagroovejet on 2006-06-07
Go to SystemSettings > Regional & Acessibility > Keyboard Layout

Tick the box marked 'enable keyboard layouts' then look for you keyboard in the 'Keyboard model' drop down menu. If you find a match select it, click 'apply' and se what the buttons do. You can assign buttons to alunch applications and stuff in the Keyboard Shortcuts section. E.g. I have the WWW button on my keyboard set to launch Firefox and the Email button set to launch Thunderbird.
