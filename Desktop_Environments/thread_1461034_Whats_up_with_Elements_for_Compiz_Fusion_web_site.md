### Post by Alan James on 2010-04-23
Open a terminal and cd to the directory where you have the script. Enter super user mode by entering “sudo su” and typing in your password. Now type “sh elementsinstall.sh” and let it run.  
 

 Think of the “sh” in front of the the script name as a way of telling the computer that you are running a script. Do this all scripts you want to run.


I hope this helps.
