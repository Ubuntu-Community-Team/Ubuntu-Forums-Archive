### Post by manoynmonic on 2009-06-25
In the xfce menu:

settings->Xfce4 settings manager->mouse->theme

There is an adjustment for cursor size.

Also, you might try gnome-look.org for some cursor themes that are a bit higher contrast.

Download there, and copy the extracted download (the whole folder) to /usr/share/icons
