### Post by dr_d12 on 2007-01-30
:::I find that this is the quickest way if you want to actually type the characters (short of learning a spanish keyboard layout):::

Rightclick on the panel and add the Keyboard Indicator applet. Mine says USA.

Right-click it and open up keyboard preferences. 

Click add and select "US English International (with dead keys)".

Toggle between the layouts by clicking once on the applet. USA* is the international layout.

With USA* activated, you type ' then the vowel:
' + a = á
' + e = é
' + i = í
' + o = ó
' + u = ú

~ + n = ñ
" + u = ü

To get a ` or ~ you have to type them twice:
' + ' = '

For the punctuation, hold down the Alt key (only the right one works on my keyboard):
Alt+1 = ¡
Alt+/ = ¿
