### Post by kassetra on 2005-03-28
[QUOTE=mesocool]Is that possible to input different languages in Ubuntu such as Chinese? Thanks...  [-o<[/QUOTE]

Yes, it is possible to input from Chinese and Japanese and some other languages as well.  

You need to do some extra setup work if you've already installed Ubuntu.  Obviously, the easiest way to get Chinese character input would be to install a Chinese Linux, but - you can make most Linux distros accept character input.

Here is the way I know other people have allowed Chinese input in Linux:
[http://www.linux.com/howtos/Chinese-HOWTO-4.shtml]("http://www.linux.com/howtos/Chinese-HOWTO-4.shtml")

For the most part, you'll simply need to get the packages from Synaptic and install them - but I suggest you read through that howto - in order to find exactly what you need.  :)
