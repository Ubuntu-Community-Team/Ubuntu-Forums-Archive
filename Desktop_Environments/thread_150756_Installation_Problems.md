I'm in the process of doing an install of Ubuntu V5.10 on a new PC (just built).  The installation is consistently hanging on the install of the GRUB boot loader.

This is a new PC configured with a P4 -Prescott; 80 G SATA and one DVD RW optical.  The M/B is an Intel.

As a point of reference, the CD used initially to attempt the install was the same one I successfully used on an old P3 that I have been using for evaluation.  I did download and re-burn an iso and it had the same effect.

Any suggestions would be appreciated.

### Post by indytim on 2006-03-27
I'll post the resolution to this problem.  After "zeroing" out the new hd and doing the re-install again I (ugh) think I found the cause of the problem with the install.  I was attempting to enter the same name for the user's full name as well as the userid.  Guess the lesson on this is to not attempt to do the new install when you're tired and not paying particular attention to the prompts.

Everything is up and running well.  Off to customizing Ubuntu for my wife's new PC.

IndyTim
