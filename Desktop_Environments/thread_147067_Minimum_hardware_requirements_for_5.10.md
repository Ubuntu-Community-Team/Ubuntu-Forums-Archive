### Post by tuxcantfly on 2006-03-19
there are no "minimum" hardware requirements
it'll run on just about anything, only it'll be very, very slow
if you plan to use gnome (ubuntu default) or kde (kubuntu default) you should have at least 256 MB ram and a Pentium 3 processor if you don't want it to be too sluggish
if you have less than this, I recommend using xfce, which is more lightweight
to do this, install ubuntu, then

sudo apt-get install xfce4

and on your next login, select xfce as your session type
