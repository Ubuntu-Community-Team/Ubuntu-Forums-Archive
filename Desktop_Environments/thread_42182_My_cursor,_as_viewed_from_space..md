### Post by ow50 on 2005-06-16
Some cursor themes scale, others don't. Try Bluecurve:
[http://gnome-look.org/content/show.php?content=24237](http://gnome-look.org/content/show.php?content=24237) 
[http://gnome-look.org/content/show.php?content=5600](http://gnome-look.org/content/show.php?content=5600)

To install, untar to ~/.icons and use gcursor (apt-gettable) to select the cursor theme and pointer size.
