### Post by jem7v on 2007-01-08
Different themes have different colors for different parts.  From the looks of it, though, it's just the BUTTONS on the taskbar that aren't dark like the rest of theme, which leads me to suspect that perhaps the GTK2 theme you're using requires the gtk2-engines-pixbuf package in order to render pretty buttons, and you might not have it installed.  Make sure it's installed and if it isn't, do so and see if it changes anything.
