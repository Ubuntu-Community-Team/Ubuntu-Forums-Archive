### Post by tom4everitt on 2010-11-18
I'm not sure what you mean exactly. What do you mean with "a view"? Perhaps you could provide a screenshot if it is hard to explain in words. 

Generally, to view files, there is a button "Places" in the top left corner. There you'll find a number of options on where to start browsing your file system. All of them will open "Nautilus" which is the file browser in Ubuntu. From any Nautilus window you can browse through all your files and folders.

Above the file area are some buttons, to the left some shortcuts. The shortcuts are more or less the same as the ones you see under "Places". Adding a shortcut to the left pane (by drag-and-drop) will also add it to the "Places" menu. 

That's the general file viewing in Ubuntu/Linux I think. Does it sort of answer your question?

And, by the way, Welcome to Ubuntu!
