### Post by Favux on 2012-02-24
Hi twgray,

Don't know that there is one.  The Wacom Tablet applet's first appearance is with GNOME 3.2, i.e. Oneiric.  It uses the gnome-settings-daemon, which I doubt is in Xubuntu.  It doesn't do much yet so you aren't missing much.  The one in 3.4 should have more capability.

You don't say what tablet you have but if it has tablet buttons you'll have to manually set up:  [http://sourceforge.net/apps/mediawiki/linuxwacom/index.php?title=Tablet_Configuration](http://sourceforge.net/apps/mediawiki/linuxwacom/index.php?title=Tablet_Configuration)  Don't know if Wacom Control Panel is working now or would work for you:  [http://sourceforge.net/apps/mediawiki/linuxwacom/index.php?title=External_applications](http://sourceforge.net/apps/mediawiki/linuxwacom/index.php?title=External_applications)

Both pages from the Linux Wacom Project's mediawiki HOWTO page:  [http://sourceforge.net/apps/mediawiki/linuxwacom/index.php?title=Category:HOWTO](http://sourceforge.net/apps/mediawiki/linuxwacom/index.php?title=Category:HOWTO)
