### Post by JayD2 on 2009-11-09
If you don't have the cube feature in ccsm active but you do have the desktop wall active in ccsm try ticking off the 'edge flip move' box on there. You shouldn't have to restart to see it take effect.

### Post by JayD2 on 2009-11-09
Go to system --> appearance --> visual effects tab.
Is it set to none?
When you go to set it to normal or extra does it find available drivers and let you?

If you set visual effects to normal or extra and have the 'desktop wall' enabled with 'edge flip move' checked it should work.
