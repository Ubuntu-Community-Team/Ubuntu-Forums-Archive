### Post by rainwalker on 2009-07-09
If you're talking about having a cube inside of firefox that rotates when you change tabs, I know that is done with an extension. I don't think it's available for Linux, though.
If you're talking about the cube that shows when you change desktops, though, that is done with Compiz. If you haven't already, install compizconfig-settings-manager with Synaptic (System > Administration > Synaptic Package Manager). Then go to System > Preferences > Advanced Desktop Effects Settings. I believe the option is in the "General" section, otherwise you can search for "desktop size" in the search box on the left side of that window. Change that number to 4. Once you've done that, enable the "Desktop Cube" plugin.
