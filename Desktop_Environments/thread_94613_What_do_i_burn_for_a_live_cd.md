### Post by mgmiller on 2005-11-24
No.  you need something that recognizes how to deal with .iso files.  What OS are you trying to burn the live CD from?  If it is WinXP, you would need to have Roxio or Nero installed.  They both know what to do with .iso files.  You put a  blank CD in your burner and just doulble click the .iso ico.  This will recreate the file structure on the blank CD and make it bootable.   As I recall there is a free WinXP add-on that also enables .iso burning.  Here is a link to it.  [http://devhood.com/tools/tool_details.aspx?tool_id=143](http://devhood.com/tools/tool_details.aspx?tool_id=143)

If you are using an earlier version of Ubuntu like 5.04, it already knows how to burn .iso's.  Just double click the .iso icon and it will expand it onto a blank CD.

If you are using Mac.  I don't know for sure, but I would almost bet it also knows how to burn from .iso by clicking on the file also.

Hope this helps.
