### Post by jpka on 2008-06-14
In 8.04, 
System / Preferences / Appearance / Customize / Colors
the column 'Text' is what u need.
But, there are dummy applications exists, which not used this setting, and use 'black' font color anyway. I do not know right way to correct this.
If u love dark color shemes (light fonts on dark background), look at 'invert' option in Compiz. Always helpful! I use it. But pictures in browser look terrible.
