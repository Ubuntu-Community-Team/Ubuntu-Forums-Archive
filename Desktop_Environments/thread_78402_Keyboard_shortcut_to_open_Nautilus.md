### Post by TLE on 2005-10-18
The problem is that, at least on my computer, the Win key as not an add key like Ctrl, and so can not be used together with others keys to make shortcuts. You could try to play with the alt-win keys settings.
System -> Settings -> Keyboard
Then choose the "layout choise" tab (or what ever the name is, I'm using a danish version of ubuntu, it is the third tab on my system)
Click the little arrow next to Alt/Win key behavior, and try some other settings. Maybe it'll allow you to use win as a add key. I don't use it myself so I don't know if ot works. But it is a possibility.

### Post by flaming_monkey on 2005-10-18
I know it's not exactly what your after but if you go to...

System > Preferences > Keyboard Shortcuts 

...and select "Launch home directory", you can assign <ctrl>+e to load your home folder. If you've set up the sidebar in nautilus as directory structure, you should be set.

Hope that's of some use.
