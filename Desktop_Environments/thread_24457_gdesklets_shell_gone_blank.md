### Post by bobmitch on 2005-04-07
Close down any apps running -> Ctrl-Alt-Backspace

or simply reboot.

There's always a chance that live updates will mess around with a live running system - I think that the ubuntu system update daemon should have some dialog to this effect. 

eg.  Some Desktop / Application files were updated.  Would you like to restart your system to ensure 100% expected functionality?
