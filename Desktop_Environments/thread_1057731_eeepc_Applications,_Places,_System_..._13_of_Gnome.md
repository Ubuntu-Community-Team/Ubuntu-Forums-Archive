### Post by Rhubarb on 2009-02-02
A better way would be to remove the Applications Places and System Menu (right click it, select remove), and add a little menu button (a bit like the start menu from windows).

To do this, right click on an empty part of the bar, select "Add to Panel...", and select "Main Menu", then press the "Add" button.

You should now have a nice little icon that doesn't take up much space at all, that'll let you access all your Applications, Places, and System menus easily.
