### Post by hal10000 on 2006-11-17
If you see the "Debian" submenu, you can find it in Debian-->Applications-->Networking--KMail.

The easiest way to get a (missing) menu entry is to run kmenuedit (<ALT>+<F2> ,then type kmenuedit) an make an entry in a submenu where you like to have it (you can also create a new submenu). You have to choose a Name for the new entry and the command to run the program, which is in your case kmail.

The reason KMail isn't listed separately in the menu is because it is usually launched together with Kontact, the KDE Personal Information Manager (sort of like Evolution). But KMail can still be launched separately, as you have discovered. To add an entry for it in the K Menu, just right-click on the K Menu button and select "Menu Editor".
