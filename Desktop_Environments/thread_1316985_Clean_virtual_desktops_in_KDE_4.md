### Post by TommyMac501 on 2009-11-06
Is there a way to have each virtual desktop "clean"?, in other words, when I move from desktop 1 to desktop 3, I don't want to see the open apps I have running on desktop 1 on my desktop 3 panel.  I want each destop to appear as though it's an entirly new instance.  Can I do that?

---

### Post by Untitled_No4 on 2009-11-06
Yes you can.
Right click on an empty area in the task manager and choose Task Manager Settings and there tick "Only show tasks from the current desktop" and click ok. 
You're done.
