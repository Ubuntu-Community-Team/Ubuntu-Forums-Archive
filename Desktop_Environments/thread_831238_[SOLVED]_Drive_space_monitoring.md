### Post by sea_monkey987 on 2008-06-16
you can also install screenlets : 
```
sudo apt-get install screenlets
```
Then go to System > Preferences > Screenlets (alternatively, you can type screenlets-manager in the terminal).
Say "yes" to create the startup folder.  use the diskusage screenlet
