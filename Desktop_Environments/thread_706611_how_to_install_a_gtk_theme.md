### Post by 3rdalbum on 2008-02-26
If the GTK ends off looking like a default theme (like Windows 95), then you probably don't have the right theme engine installed. By the looks of it, you're probably missing gtk2-engines-pixmap or gtk2-engines-murrine. Install those in Synaptic and you should be alright.
