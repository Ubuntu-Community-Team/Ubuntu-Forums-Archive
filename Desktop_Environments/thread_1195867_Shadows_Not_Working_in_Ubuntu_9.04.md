### Post by gettinoriginal on 2009-06-24
Compiz (window manager) is installed in 9.04 by default, You need to install Compiz Configuration Settings Manager (in repositories) either from terminal ie:
```
sudo apt-get install compizconfig-settings-manager
```
or if you do not like CL, then it is in Synaptic Package Manager under System > Administration.  Once that is installed, it will be under System > Preferences, open it and change settings to achieve the look you want.  :P
