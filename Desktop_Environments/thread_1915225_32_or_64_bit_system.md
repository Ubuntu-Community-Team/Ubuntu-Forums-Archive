### Post by aarceng on 2012-01-25
I had to find out if my operating system is 32 or 64 bit. This isn't as easy as it sounds but I finally found the following command that gives a clear answer.

[INDENT]getconf LONG_BIT
32 [/INDENT]

returns the number of bits for the operating system
