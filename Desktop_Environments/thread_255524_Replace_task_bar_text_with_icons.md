You might want to take a look at creating a 'drawer' or 'menu' to the panel.  You can either create one based on category (ie Games) or custom create one with the programs of your  choice.  It would really depend on how simple you want to make things for your lady-friend.
To get an idea of what it would look like, select Games -> right-click on any game -> select Entire Menu -> Select either option.

The difference is that as a drawer the selections display as icons, whereas the menu lists the titles of the selections.
