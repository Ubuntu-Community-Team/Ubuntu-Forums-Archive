### Post by Michael Schmidt on 2008-03-14
Although its been a little while since this thread has been active, I wanted to add a comment for the archive readers.  I used Nvu on Windows a couple of years ago.  Liket the design and features, but it was buggy.  It appeared the developers had abandoned the project and that no further work was going to get done on it.  

When I moved to Linux (Ubuntu Gutsy), I started out with Amaya 9.54.  Nice enough features and design, but it would lock up.  This happened when I changed views and even xkill wouldn't work.  I have switched to KompoZer and this seems to work fine.  It has good design and features, is easy to work with, and I haven't seen any bugs or glitches.  For those not familiar with KompoZer, apparently it is based on Nvu.  Some folks updated and did bug fixes on Nvu and released it as KompoZer.  Congratulations to them!  Nvu was a nice project but needed to have some bug fixes, and I'm glad the KompoZer folks took over.  It would have been such a waste to let Nvu just languish.

### Post by MetroPietro on 2008-03-25
As of 25 March 2008 the discussion above is outdated. 
Two descendants of Mozilla Composer are available. One is Iceape, which is available in the Ubuntu repositories.
You may need to enable the full repository set to get Iceape to show up in Synaptic.
1. Start Synaptic Package Manager.
2. Go to Settings -> Repositories...
3. in leftmost tab of new dialog box ("Ubuntu Software"), check all the boxes. This will enlarge the pool of packages that Synaptic will offer you.
4. Search for Iceape. Make sure to install "iceape" itself, the whole suite which includes Composer.

The other Mozilla descendant is the SeaMonkey project, which is in fact the direct continuation of Mozilla Suite development (Iceape seems to be a fork or variant). SeaMonkey is not in the Ubu repos, but you can download it directly or add it into your list of third-party sources. 

DreamWeaver running under WINE is still an option.

Quantum+ seems strong and stable, but an unfamiliar interface for former Mozilla users. However Iceape Composer is the same old interface, running on an updated version of the same old core.

KompoZer does not seem to be ready for prime-time yet.
