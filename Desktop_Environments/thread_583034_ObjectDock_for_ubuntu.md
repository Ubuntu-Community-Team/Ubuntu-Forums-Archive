### Post by Depressed Man on 2007-10-20
You have a choice between Kiba Dock, AWN, and others such as Gnome dock. Depending on if you have some sort of compositioning desktop also will determine what you can use. These would be Beryl, Compiz, Compiz Fusion, etc..

For AWN you don't have to compile it. Just follow the instructions to add the repository of whatever version of Ubuntu you are running. Then install it. For Kiba you'd have to follow the guide to compile it (or use an older .deb which is pretty outdated).
