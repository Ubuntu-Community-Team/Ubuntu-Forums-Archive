### Post by valbaca on 2010-07-20
Ubuntu Tweak [http://ubuntu-tweak.com/](http://ubuntu-tweak.com/) is an easy way to manage advanced settings in Ubuntu (in this case try the power management and security settings to disable Suspend). 
 
Remember, in Ubuntu Suspend=Sleep=Save to RAM (low power). Hibernate=Suspend to HD (no power)
