### Post by kio_http on 2010-03-16
1. Run ccsm (ATL + F2, type ccsm)
2. Enable Desktop Cube and setup your desired configuration.
3. Start compiz (Run compiz --replace)

Note: Controlling (Starting, configuring etc) is easier if you install fusion-icon.
In terminal:
```
sudo apt-get install fusion-icon
```

Then run fusion-icon, it will open in your system tray.
