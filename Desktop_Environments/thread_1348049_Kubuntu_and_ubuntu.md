### Post by aysiu on 2009-12-07
If you downloaded the Desktop CD of Kubuntu, that won't help you.

You need to install Kubuntu using the package manager (Synaptic, apt-get, aptitude), and that means you need to use either a broadband connection on your Ubuntu computer (to install directly) or a broadband connection on another computer (to download and burn the Alternate CD of Kubuntu) to install *kubuntu-desktop*.
