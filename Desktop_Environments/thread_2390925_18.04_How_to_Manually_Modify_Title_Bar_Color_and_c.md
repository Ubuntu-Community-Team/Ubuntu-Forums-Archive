### Post by apollothethird on 2018-05-26
I gratefully appreciate all the time and input from all who helped with suggestions and workarounds to get a resolution to my issue.

The question was answered on [AskUbuntu]("https://askubuntu.com/a/1040536/29012").

Place the configuration in ~/.config/gtk-3.0/gtk.css

This code did the trick:

```
.titlebar {
    background: #0759EB;
}

.titlebar:backdrop  {
    background: #8A0829;
    color:white;
} 

decoration {
    border: 3px solid #0759EB;
    background: #0759EB;
}
decoration:backdrop {
    border: 2px solid #8A0829;
    background: #8A0829;
}

```
