### Post by mister_p_1998 on 2006-08-22
[QUOTE=aresthedog;1407838]Could anyone please tell me whether the motherboard AsRock 775i65GV is supported by Ubuntu? Especially the onboard ethernet card, the onboard graphic card, the onboard ide controller and the onboard sata controller.

Yes it is, I run one, all the onboard stuff works out of the box but I wouldnt recommend it if you want to use an AGP video card, you can only use certain approved cards and the ones I tried hung in dapper, although let me into the BIOS. If you dont want to play 3D games youre probably ok. Im debating whether to get another mobo. I dont really want to play Mega 3D games in Linux, so Ill probably keep it. Its ok with PCI soundcards, but Dapper wont run sound in duplex, so theres no poing replacing the onboard sound really. I suppose you get what you pay for.
