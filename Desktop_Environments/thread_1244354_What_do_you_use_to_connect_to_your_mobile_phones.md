### Post by Arminho on 2009-09-05
Hi!
I'm using Multisync top keep my contacts and caledar in sync on mobile (nokia 6120c), evolution in my laptop (ubuntu 8.10)  and gmail. It works fine with contacts, calendar has some issues.
To send SMS from my PC i use the phonemanager (you can install via synaptic). It works fine for simple messages and is also integrated with evolution contacts, but has some serious drawbacks (send SMS to multiple receipeints).
That's why I'm currently evaluating wammu/gammu. Seems to be rich of functionality but it seems not straight forward to connect via bluetooth...

### Post by Shack2424 on 2009-09-07
I'm using the latest 9.04 Ubuntu on my unlocked Nokia....and had some trouble at first. The default tether wizard works great (for us in the US anyway).
The trouble I had was.....that even though it would connect right away after I setup using the wizard...I still could not get pages to pull up in Firefox.

Found that by disabling my Abit Gigabit Ethernet, Firefox and Ubuntu updates now work perfectly,,,,guess it was a priority override issue of LAN vs. USB modem....
