I recently shoe horned Breezy onto an Acer Ferrari (I'll make another post on my experiences) leaving Windows permanently at last except for gaming.

However, I'm a developer and used to living in my file explorer, and for me frankly Nautilus just doesn't cut it. So I desperately need either some hardcore customisation or a replacement.

What I really want is Windows Explorer :smile:. However what I need is:

Decent file associations - Amongst others Nautilus thinks IDL and JS files are "Executable Files" and prompts me every time even though they have association - very annoying.

Drag Select - no need to CTRL+Click, so you can select files with mouse only.
