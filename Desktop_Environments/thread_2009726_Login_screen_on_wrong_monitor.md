### Post by LocoMan on 2012-06-24
I have an ubuntu computer that serves mostly as a regular internet browsing computer and sometimes as media center trough an hdmi cable plugged to an LCD TV.I use an nvidia card using twinview.

Problem is, no matter what I do the login screen appears on the TV, even if I have the computer monitor as the main screen, which is annoying because sometimes other people (usually my wife or my son) are watching TV or a movie already so it can be a bit of a problem having to switch to the computer input to log in. For regular log in there's no problem because I can just blindly press enter and type the password, but it can be when I want to do something more specific like changing the window manager or log in as a second account.

Googling around I've found that the problem is that when there are more than one monitors, ubuntu puts the login screen on whichever monitor the mouse cursor is. Problem is, twinview creates one single big monitor, and ubuntu centers the mouse on the screen on log in. Because of that, if one of the monitors on twinview mode has larger resolution than the other (in my case 1920x1080 on the LCD TV, 1280x720 on the computer one), then the center of the screen will always fall on the bigger monitor.

Any workaround I could use?
