### Post by Freddy on 2006-01-13
I'm using a Swedish translated KDE, so maybe the names not entirely correct.

For changing the default mail client.
Kcontrol/Systemsettings | KDE-components | componentchooser

To change which application that will open when pressing a .mp3 (for example) go to.
Kcontrol/Systemsettings | KDE-components | Filebindings.
