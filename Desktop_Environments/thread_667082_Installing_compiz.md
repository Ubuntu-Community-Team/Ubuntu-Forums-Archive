### Post by PeterJS on 2008-01-14
The first step is to get the right graphics drivers installed. They can be installed through System > Admin > Restricted Drivers Manager. Compiz itself is installed by default, but not it's configuration tool, so that's the next thing you're going to want to [install](apt:/compizconfig-settings-manager). After that the settings manager is in System > Preferences > Advanced Desktop Settings, configure and enjoy.
