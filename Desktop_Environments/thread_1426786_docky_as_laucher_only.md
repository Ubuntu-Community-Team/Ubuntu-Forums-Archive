### Post by vancouverite on 2010-03-10
Is there a way to use Docky as only a launcher?  I prefer my good 'ol task bar for window management, but Docky is a slick and well made launcher.

As a task manager it does two things I would like to disable:
1) It gets busy and things move around when I am doing lots of stuff.  I have it on auto hid and I would like to move my mouse to the same position to launch a program every time, not have to search for the button based on the number of programs I'm running.

2) I steels window previews from the task bar.  That is, sometimes I get previews mousing over the task bar and not from docky and other times it's the other way around.  I want the previews associated with the task bar only.

Overall a great program and well made.  I hope that I can configure it to just my taste!

### Post by new_to_ubun2 on 2010-03-15
Solution is very simple,
1.Add new panel
2.Right click on it, click add to panel
3.Add window list

And now you can easily use docky as laucher while managing opened window from task bar.
