### Post by Achtung on 2008-05-08
You might have to restart X (ctrl+alt+backspace) the first time you enable desktop effects, but I don't recall having to do so.

That said, if you have checked all options, then you should have window decorations such as shade displaying, you should have wobbly windows when you move them around, you should see the cube rotation when you press ctrl+lat+left or right arrow, or when you middle-click on the desktop and move your mouse around. Do any of these effects show up?

### Post by asrai on 2008-05-08
If you click on each option (not the checkbox, the name of the effect) in the advanced desktop effects settings you should be able to see the key bindings - this will tell you how to activate each effect.  Please let us know what happens when you do ctrl-alt-left arrow or move a window.  

To make the rotate cube work you will also need to add additional workspaces.  If you right-click on the boxes on the lower right panel next to the garbage bin you should be able to select 'preferences' and then change to 4 columns, 1 row.
