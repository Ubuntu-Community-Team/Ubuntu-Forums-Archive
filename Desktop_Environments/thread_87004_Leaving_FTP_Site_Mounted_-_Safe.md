### Post by audax321 on 2005-11-07
I have some FTPs setup using the Connect to Server.. dialog as well and just leave them as is. I don't think it actually mounts or connects to them, it just provides a convenient link. Usually if I double-click it asks me for a password. But if I try to reconnect a short while later and either it still remembers the password or the FTP hasn't logged me out, it will let me connect without asking me for a password. After some time though the connection closes and then it will ask me for a password again. But I don't see why it would be a security risk unless someone can physically access your computer while it still has you logged in.
