### Post by LLRNR on 2006-11-30
> **Cronjob said:**
> How can I configure KDE in 6.10 to switch to other virtual desktops by moving the mouse off the screen? It seems this would be a built-in option, but it doesn't seem to be. I'm not using compiz, beryl or any of that.

It's easy to do this without Beryl, but it involves wheel-scrolling... there might be another way to do it too but this is the only one I know: right-click anywhere on your desktop -> Configure Desktop -> Multiple Desktops tab -> move the slider to get the desired number of desktops -> check the box labelled "Mouse wheel over desktop background switches desktop" -> Apply -> OK

That should do the trick, but with the mouse wheel only...

### Post by der_joachim on 2006-12-01
> **Cronjob said:**
> I've searched high and low without finding the answer to a simple question:

How can I configure KDE in 6.10 to switch to other virtual desktops by moving the mouse off the screen?

- Open KDE control panel
- Select Desktop->Window behaviour
- Click the tab "Advanced"
- Tick "Always Enabled" under "Active desktop borders"
- Click "Apply".
