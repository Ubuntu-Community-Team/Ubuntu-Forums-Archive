### Post by RAOF on 2005-12-16
You probably don't want to mess with them.  Many of the k* processes will actually be **kernel** processes.  I'm not quite sure what killing them will do, but it is unlikely to improve system performance :)

Unless your system is actually running too slowly for your liking, leave those processes.  They will be doing useful things in the background for you.

If your system **is** running too slowly for you, maybe it's time to give up KDE and run a lighter desktop system?  You could install xubuntu-desktop and run XFCE, which is apparently much lighter and faster.
