### Post by hhh on 2010-10-12
If you have Compiz then you have Compiz Fusion. You can add the configuration editor and some extra plugins via System>Administration>Synaptic Package Manager. Search for compizconfig-settings-manager and compiz-fusion-plugins-extra.
