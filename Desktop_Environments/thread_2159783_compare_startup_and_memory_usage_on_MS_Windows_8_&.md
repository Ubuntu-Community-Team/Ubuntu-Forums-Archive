### Post by silv3rm00n on 2013-07-04
Windows 8 does boot much faster than xp or 7. 
However it is nowhere near to ubuntu in terms of resource usage. It eats a lot of ram and cpu, which ubuntu does not.

A normal ubuntu system with lxde/xfce desktop would not eat more than 512 MB in idle condition. However windows 8 needs around 2gb for an empty desktop, that is no application running.
