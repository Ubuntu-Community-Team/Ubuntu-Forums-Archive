### Post by gettinoriginal on 2009-01-01
Install Compiz Fusion Icon from Synaptic, that will allow you to choose your Window Manager ie Compiz/Metacity.  Your post is a little confusing, because you say that metacity is not loading at boot, but then asking compiz questions.  The other way you can change your Window Manager is in Terminal type:
```
metacity --replace
```
```
compiz --replace
```
