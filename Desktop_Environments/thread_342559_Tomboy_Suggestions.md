### Post by mcglnx on 2007-01-20
Hi,

I started to use Tomboy. Really like the Idea.
IMHO, the open/save buttons should not exists anymore! (considering the power of today's computers!)

I got a few suggestion for Tomboy team:
- Have the hability to import/export other Tomboy's notes 
- Having different 'subject' (Work, Linux, News,....) Those subjet would be represented on different colors to make things more clear (Yeallow, blue, green, orange,...) Just like 'real-life' sticky notes.
- having hiearchical Table of contents with folding options.
- Having a little bit more edition options (lists, tables,...)
- Be able to navigate from notes to other without opening new notes.
- Be able to navigate (previous) using keyboards, just like a web browser.

I'd see the default behavior just like now, but having the possibility to activate the others feature if the user wants it.
