### Post by TheFu on 2019-10-18
.jar files are Java archives.  Most require extra environment variables, so I wouldn't assuming that just calling java whatever.jar would actually accomplish what you want.  But it wouldn't hurt to try.

"associations" are a GUI thing, tied to the specific GUI you are running.  The Ubuntu Desktop Guide should explain how to do that for popular DEs.  Lacking that, Google found this: [https://askubuntu.com/questions/90214/how-do-i-set-the-default-program](https://askubuntu.com/questions/90214/how-do-i-set-the-default-program)
