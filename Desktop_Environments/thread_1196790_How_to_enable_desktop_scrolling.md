### Post by mcduck on 2009-06-25
I'm not sure about how to enable the virtual resolutions in latest Ubuntu versions, but here's something that should help you using Ubuntu on small resolutions: if you hold down Alt key, you can move windows around from any point in the window (not just the titlebar like usually). This should allow you to access those parts of windows that wouldn't otherwise fit in your screen.

(Alt+left button moves window, alt+middle button resizes it)
