### Post by DavePlummer on 2010-09-28
There is no GUI specifically for 7-Zip, just the command line tools in package p7zip and related packages. File Roller, the GNOME archviing application uses the command line tools to give you most of the 7-Zip functionality. You will need to use the command line tools directly to use the more esoteric options.
