### Post by ruibernardo on 2007-08-19
Hi Nessa,

you can install **kubuntu-desktop** package, but it will change the splash and login screen of your Ubuntu. Another option (not tested but should work) is to install the package **kde-core**. This one will not modify your splash and login screen, but will install KDE.

To login in a specific desktop, you only have to choose it in the login screen the first time you login (on the left bottom corner in the login screen). After that, the user will always login to the chosen desktop, unless he modify his login options again.
