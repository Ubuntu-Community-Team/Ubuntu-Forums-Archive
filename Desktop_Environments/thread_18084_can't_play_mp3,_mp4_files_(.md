### Post by darrenadams on 2005-03-04
As well as that, you can add MP4 support by installing gstreamer0.8-faad and gstreamer0.8-faac from [http://apt.cerkinfo.be](http://apt.cerkinfo.be), which also contains packages for playing MP4 files from other programs, such as XMMS and Beep Media Player

If you don't know how to add repositories using Synaptic, check this howto at - [http://www.ubuntulinux.org/wiki/AddingRepositoriesHowto](http://www.ubuntulinux.org/wiki/AddingRepositoriesHowto)

You might also want to read up other repositories at httphttp://www.ubuntulinux.org/wiki/RestrictedFormats
