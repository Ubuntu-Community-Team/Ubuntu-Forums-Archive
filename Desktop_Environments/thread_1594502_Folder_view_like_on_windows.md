### Post by 3Miro on 2010-10-12
What is it that you find different between Ubuntu and Windows? Is there something wrong with the previews?

For editing pictures, the closes you can get to Photoshop is Gimp. You can install it from the Software Center.

Some versions of Adobe Photoshop run under wine, however, it can be hard to set up. Read the instructions carefully.

[http://appdb.winehq.org/appview.php?appId=17](http://appdb.winehq.org/appview.php?appId=17)


You can use Photoshop with Virtual Box, however, you will need a copy of windows. 
[https://help.ubuntu.com/community/VirtualBox](https://help.ubuntu.com/community/VirtualBox)

Get the user manual from [http://www.virtualbox.org/wiki/Documentation](http://www.virtualbox.org/wiki/Documentation)
