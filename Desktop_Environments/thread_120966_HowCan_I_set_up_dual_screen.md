### Post by minisori on 2006-01-24
You have to edit the /etc/X11/xorg.conf file, remember to do a copy of it before.

If u want to see both displays at same time..

1. First duplicate the device identifier section and add for each one Screen 0 and Screen 1 respectively, change the identifier for "whatever_identifier0" and "whatever_identifier1"

2. Add another section monitor wich will be your LCD monitor.

3. Edit section screen and change the device for "whatever_identifier0". Add a new section screen where device will be "whatever_identifier1" the identifier "any_name_u_want", the monitor "the LCD one" and add the subsection display with the resolution you want.

4. Edit section ServerLayout and add (for Xinerama):
Screen "the identifier you put in second screen section u added before" LeftOf "Default Screen"
Option "Xinerama"
