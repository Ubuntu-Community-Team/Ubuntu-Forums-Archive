### Post by djxcon on 2012-05-02
I have two installations of Ubuntu 12.04 (One Desktop and one Netbook). They both have Unity and Gnome Shell. On the Desktop, the window buttons (close, maximize) show on windows that are maximized. On the Netbook the buttons only show on windows that are not maximized. I have Gnome Tweak (Advanced Configuration) installed on both and although I know how to change which buttons show, I would like for the buttons to show on windows that are maximized on the netbook. The netbook is a EEEPC. 

Any ideas? Is this a gconf-editor change?

### Post by djxcon on 2012-05-07
Solved. Because the EEEPC is a netbook, there was a program called Maximus running in the Startup. Un-selecting this program causes the buttons to display on maximized windows.
