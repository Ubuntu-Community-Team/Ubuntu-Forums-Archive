What are you running ?

1. If you mean placing an icon in to the launcher bar, once the program has been installed, you can drag the icon from the dash to the launcher.

2. Bookmark gmail to your browser bookmark bar, then drag the shortcut to the desktop. Right clicking the newly created desktop shortcut will let you rename and give it an appropriate icon.

3. If you are running Unity 2D, you won't see the resize icon option in Appearances.
