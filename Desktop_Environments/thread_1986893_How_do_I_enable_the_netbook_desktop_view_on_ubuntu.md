Simple answer:  you don't.

The netbook version has not been available for a while now, not even downloadable for 10.04 now, though it was in the first place when 10.04 first appeared.

In the gnome versions of ubuntu the netbook is available as an add package for some later versions at least, but not for the usual architectures of desktop or laptop machines, only for armel, whatever that is.  There is no separate ISO file to download.
