### Post by aggiechemist on 2005-07-21
Well, I know you can put both on pretty easily. Start with installing plain old Ubuntu, and then go into synaptic. Then just look for the right package, I think it is just named KDE. Then install it. It is fairly large, is hooked to a lot of files, and will take some time, but that is all there is to it. To switch between the two, just log out. When you look at the login screen, you should be able to find a button that gives you a list of options as to what desktop environment you want to log into.

Don't forget there are other options too. You may want to try windowmaker. It is slightly different looking, but usually extremely fast.

As for the beautiful desktop, I don't know much about that one. Look around the forums and such, there is probably already some good advice out there.

### Post by Mr Frosti on 2005-07-21
By beautiful desktop, if you mean a lot of visual effects that are meant to impress, then look into the following utilities and websites for Gnome:

gdesklets is a program that gives the user access to the same tools you already have, but a lot of the graphical work has been redone. There are some new utilities, but most are just re-worked graphics and effects that wrap over the original program. 

You can get gdesklets by typing: ```
sudo apt-get install gdesklets
```, or by searching for gdesklets using Synaptic ("System" -> "Administration" -> "Synaptic Package Manager") 

GTK themes and icons, wallpaper, etc can be customized by going to websites such as [http://gnome-look.org/](http://gnome-look.org/), or [http://art.gnome.org/](http://art.gnome.org/)  

To install these themes, download the *.tar.gz file to a location, and then navigate there using Nautilus. ("Places" -> "Home Folder") To install the theme, click "System" -> "Preferences" -> "Theme" and drag the *.tar.gz file from Nautilus into the Theme Window. It will ask you if you want to install the theme, click "Yes" to continune. After the install, just select the new theme, icons, etc from the list.

### Post by shakin on 2005-07-21
[QUOTE=aggiechemist]go into synaptic. Then just look for the right package, I think it is just named KDE.[/QUOTE]

Install the kubuntu-desktop package to get the real deal. It will run alongside Gnome very well.

As for customization, I recommend bookmarking kde-apps.org and kde-look.org. A couple apps you may want to check out are SuperKaramba and TaskbarV2, both of which will give you extremely cool KDE desktop items. Also check out Kompose for KDE, which will give you a way to switch between apps similar to Expose on OS X, although personally I think TaskbarV2's previews handle the job better.

I think the best way to customize any desktop is to completely tear down the defaults, then build it up as you like. I built mine slowly as I needed things.
