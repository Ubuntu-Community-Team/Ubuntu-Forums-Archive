### Post by mcduck on 2007-12-13
Are you using Desktop Effects (Compiz Fusion)? In nthat case you need to change the cursor in the compizconfig-settings-manager, so install that if you haven't done it already.

Then go to System/Preferences/Advanced Desktop Effects Settings, and in the General Options , under the General-tab there's a entry for Cursor theme. Just type the namew of the theme you want to use here.

You may need to log out and back again before the new cursor is used with every application.
