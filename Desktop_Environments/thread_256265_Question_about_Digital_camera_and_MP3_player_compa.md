### Post by 3rdalbum on 2006-09-13
If it's a flash-based player that does not require any extra drivers or software for Windows (and does not require Windows Media Player), then it will work with Ubuntu.

Apple, Creative, and iRiver are brands to stay away from. Yeah you can load an iPod Shuffle on Linux, but in my experience it's unreliable.

With digital cameras, since they only come with a tiny amount of internal storage, you'll probably only save onto a memory card, right? And memory cards work fine in memory card readers on Ubuntu (as long as they use the standard plug'n'play driver)
