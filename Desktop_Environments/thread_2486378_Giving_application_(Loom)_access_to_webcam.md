### Post by zoomiest2 on 2023-04-27
I am using a Chrome addon to use the Loom.com service to do screen recording - which can record my face, while recording the screen.
PROBLEM: This application can't show my face because the webcam doesn't seem to be allowed.

HINTS: Their support page gives instructions for Mac and Windows Environments:
[HTML]On MacOS, navigate to System Preferences &#8594; Security & Privacy &#8594; Privacy Tab. Check to make sure you've given Camera and Microphone access to your web browser.

On Windows, click Start &#8594; Settings &#8594; Privacy. Check to make sure you've given Camera and Microphone access to your applications.[/HTML]

But, what about Linux? Where do I give this central access to the Loom Chrome Add-on to the Webcam in an xubuntu desktop?

Yes, I have looked at the Chrome Extension Settings > View Web Permissions. Yes, the Camera and Mic are allowed.

No luck yet. Any suggestions would be helpful.

Update: I was able to get the Loom windows application to full work (including integrating camera input into the screen), but not the Chrome Extension on Windows (nor Chrome Extension on Linux).

### Post by zoomiest2 on 2023-04-27
Solved. You just make all of the necessary approvals in the Chrome extension, then uninstall it, and reinstall it. Solved.
