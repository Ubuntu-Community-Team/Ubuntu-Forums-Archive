### Post by matthew on 2006-01-20
The absolute easiest way is to use Synaptic and install the package "gnome-art"

First, make sure the [extra repositories are enabled]("http://www.psychocats.net/linux/sources.php")

After it's done you will have a new menu item in System->Preferences called Art Manager that will help you easily install some new themes and more.
