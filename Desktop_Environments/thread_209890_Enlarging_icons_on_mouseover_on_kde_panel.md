### Post by arizonagroovejet on 2006-07-06
Older versions of KDE had an option in the Panel config for 'Enable icon zooming'. When selected the application launcer icons on the panel get bigger when then have mouse focus. It may have been that you've seen. Or it may have been what Fenyx suggested.
The icon zooming feature got removed in KDE 3.4 I think. It's certainly there in 3.3. (See attached screengrab from my elderly Fedora Core 2 machine where the mouse  is over the Firefox icon)
