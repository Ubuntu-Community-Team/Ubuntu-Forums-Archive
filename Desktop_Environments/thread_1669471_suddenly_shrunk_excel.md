### Post by jimcooncat on 2011-03-06
If the content is only data, not formulae (or they can be easily recreated like a totals row) then you can save the spreadsheet as a .csv file - close - and reopen. This will retain all the values and purge the formatting.

Another thought might be to look at Format Cells -> Alignment -> Shrink to fit cell size. If that's checked, then select all cells in the spreadsheet, get to the dialog, and uncheck it.

There's another option that might cause this, Options -> Openoffice.org Calc -> General -> Use printer metrics for text formatting.

If all fails, try it in Gnumeric.
