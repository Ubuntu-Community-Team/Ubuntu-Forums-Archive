### Post by EvilNemesis on 2009-05-14
In Ubuntu 9.04...

Every time I connect a USB storage device the OS auto mounts the drive... which I like =) 

It also pops up a File Browser window every time... which gets annoying. 

Is there an easy way to stop the File Browser from popping up every time?


Thanks in advance!

---

### Post by logos34 on 2009-05-14
nautilus>edit>prefs>media>UNcheck "Browse media when inserted'

hope that does it
