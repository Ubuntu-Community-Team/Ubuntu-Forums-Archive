### Post by fjgaude on 2011-06-27
Gosh, there are other ways... if a program as a File menu and from there a New Window then you have two or more browsers on the same desktop. The click the Workshop Switcher and click and drag the extra browser to the desktop you wish.

When I start work in the morning I click on six Launcher icons letting all six programs open on one desktop, then I drag each one to where I wish it to be for the work to be done. You can open two or more Terminals using the File menu and Open Terminal.
