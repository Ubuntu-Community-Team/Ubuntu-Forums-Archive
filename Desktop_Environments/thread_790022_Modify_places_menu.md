### Post by Gruximillian on 2008-05-11
There is a way, I know because I've done that, but unfortunately I can't remember how!
You can do this also... open nautilus, browse to the partition/folder which contains folder that you want in the Places menu, then select that folder and drag it to the left panel of the nautilus window (be shure that the panel shows places). By doing that you've made a bookmark to the desired folder which will show up in the Bookmarks folder of the Places dropdown menu.
I dont know, it might appear directly under the Places, but that's not my case, although I'm pretty shure I've done that way before. Why is it showing it in Bookmarks folder now... I dont know!
Hope this helps a little to find out how to do this.
