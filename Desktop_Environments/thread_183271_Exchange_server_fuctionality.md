### Post by CitizenKane on 2006-05-28
Microsoft Exchange has a bunch of features besides email.  For instance, storing contacts, storing calendar events, informing people of meetings, etc.  I would suggest staying away from thunderbird or kontact.  I really think that the route to go in this case is evolution.  It is normally a gnome app, but it runs fine in KDE (minus integration that it does with the gnome calendar).  I don't really know what features it has.  I looked for quite awhile and really didn't find anything.  There are supposedly other connecting applications and plugins to connect to an exchange server but I am not really sure what is out there.  I would say evolution is worth a try though.
