### Post by qrees on 2015-09-15
Hi

I have very strange issue. Mouse selection by word is not working. I can select text letter by letter but not word by word. In detail:

1. Double click word
2. Word gets selected
3. Drag mouse to next word
4. Nothing happens, selection does not change 

Expected result:
4. Whole next word is selected

System details:
Xubuntu 14.04 in VirtualBox

This does not work in any program I've tested: terminal, text editor, web browser. In gnome and xfce. It is always the same. I've tested this on other computers and it is working as expected. 

Anyone know if there is some setting to change this behavior? Or is this a bug?...

---

### Post by qrees on 2015-09-17
Problem solved!

It was Virtualbox issue. After updating to version 5.0.4 mouse selection by work works again.
