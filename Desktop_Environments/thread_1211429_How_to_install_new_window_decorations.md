### Post by izizzle on 2009-07-13
Alot of themes are DeKorator themes, so you want to do a search in kpackagekit for dekorator and install it. Then you select dekorator in the "Window Decorations" window in Appearance under System Settings and imprt the theme you downloaded from kde-llok (if it is a dekorator theme).
