You can remove the Evolution envelope from the indicator applet with: ```
sudo apt-get remove indicator-messages
``` or do it in Synaptic. It's trivially easy to add a gmail launcher to any panel, just put something like: ```
gnome-open https://mail.google.com/mail
``` as the command in a new custom launcher.

### Post by goehle on 2010-05-09
If you like the new indicator system, but want to use gmail without using evolution, you could try gm-notify

[http://bleedingpaper.com/gm-notify/](http://bleedingpaper.com/gm-notify/)

You can download the package there.  If you want to remove _just_ evolution from the indicator applet (since you would be using gm-notify) just remove the evolution-indicator package.  All together its

sudo add-apt-repository ppa:gm-notify-maintainers/ppa
sudo apt-get update
sudo apt-get install gm-notify
sudo apt-get remove evolution-indicator

(You'll have to log out before the evolution indicator dissapears)
