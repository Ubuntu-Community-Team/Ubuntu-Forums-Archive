### Post by mali2297 on 2007-12-11
Go to **Xfce4 Settings Manager -> Window Manager -> Keyboard Shortcuts** and create a new shortcuts theme. Then you can change the default shortcuts as you wish.

[http://www.xfce.org/documentation/4.2/manuals/xfwm4#keyboard_shortcuts](http://www.xfce.org/documentation/4.2/manuals/xfwm4#keyboard_shortcuts)
