### Post by Revolutionary101 on 2010-08-22
Compiz-fusion, Compiz and Beryl are all the same thing now. The used to be separate projects before, but now they are just called Compiz (for more explanation go [here]("http://www.compiz.org/")). To download Compiz go to Applications>Ubuntu Software Center and search for CCSM (the Compiz manager). Once you search for that select the "Advanced Desktop Effect Settings" and click install.

After that it should all install and you will be able to access it from System>Preferences>CompizConfig Settings Manager.

I hope this helps.
