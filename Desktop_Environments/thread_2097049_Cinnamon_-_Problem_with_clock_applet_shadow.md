I just upgraded from this year's LTS to 12.10, and I'm running Cinnamon as my desktop environment.  The upgrade didn't change much on Cinnamon, but there's one odd little annoying quirk.

The 'Cinnamon Calendar' applet is the clock on the main panel.  It works fine, but when the minute changes, only the main text color (light gray) refreshes.  The background shadow is frozen.  So it will show a ghost of whatever the time was when I logged in last.  

The only way to refresh the shadow is to hover over the time so that it brightens up in expectation of being clicked. 

I've tried disabling the applet and re-adding it as well as restarting the computer.  Actually, disabling the applet caused quite a scare for a few minutes, because the time wouldn't refresh at all unless I changed the display format, and even then just up to the current time where it would freeze again.  But logging in and out fixed that.

Anyway, not sure if this is just a bug in Cinnamon, but I thought I'd throw it out there in case anyone's seen anything similar.
