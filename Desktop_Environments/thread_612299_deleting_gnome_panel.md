### Post by Caffeine_Junky on 2007-11-13
Try, right-click on the other panel and select 'New Panel' ..this should put a new one back where the old one was..

It will be empty, so you will have to add the app's back on to it again, ..to do this, ..right-click the New Panel and select "Add to Panel" from the menu, then select what you want to add from the window that opens..
