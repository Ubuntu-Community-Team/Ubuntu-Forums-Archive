### Post by KanadaKid on 2007-05-11
Hey, everyone.

I'm using Ubuntu Feisty at the moment. My question pertains to GNOME 2.18 and its associated panels. I have placed a panel along the bottom of my screen, and another to the right side. I would like the latter to allow application windows to cover it, but I don't see any obvious options to enable such an action. I know there is a possibility to manual hide the panel in question using the show/hide arrows, but that's not exactly the type of behavior I'm looking for. I'm also aware that certain windows allow themselves to be put into fullscreen mode, but that will also cover my bottom panel, which is something I don't want.

Any ideas? Thanks in advance!

---

### Post by CameO73 on 2007-05-11
The only thing I can think of is using the 'Autohide'-option in the panel properties window. That way, when an app maximizes, it takes (almost) all the space of the panel; until you hover over it with your mouse (then it pops up over the window).

It's almost the same as how (dare I say it) Windows does it ...

---

### Post by hikaricore on 2007-05-11
You can also decrease the size of the auto hidden bar to 1 pixel to make it even smaller.  :)

I forget where it's found in the gnome configuration, but search around the forums any you should stumble on it.

### Post by KanadaKid on 2007-05-15
Yep, using the Autohide option did the trick, along with changing the panel's hidden size to 0 pixels. :) Thanks!
