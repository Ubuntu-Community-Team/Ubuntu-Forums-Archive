### Post by Dennis N on 2023-08-03
That's the "toggle shade" option that can be set in the Gnome Desktop with Gnome Tweaks Tool. Look at:

Tweaks > Window Titlebars > Titlebar Actions

Here, by default, "Double Click" is set to "Toggle Maximize". "Toggle Maximize" works as expected. But setting to "Toggle Shade" doesn't seem to be working, at least in Ubuntu 23.04 (which I currently use). 

CentOS may have it working, and it might have worked in some earlier Ubuntu releases, but I'm not sure if it ever did.
