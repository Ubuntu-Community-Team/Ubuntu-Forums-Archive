### Post by Wolki on 2007-09-28
> **DamjanDimitrioski said:**
> If i drag some file or icon from the file or the Desktop, i want for example while dragging the file into the file manager to press backspace to go one folder level back, or in terminal to add or remove some text, while he file icon is on the mouse cursor.


Dragging causes a grab of all input things, so you can't have unrelated keyboard shortcuts while doing it. In the terminal. you can of course edit the text after dropping.

If you're in Nautilus' list view, hovering over a directory will also expand that directory, and hovering over a window list entry or workspace will rais that window or switch to that workspace. I think there's no way to go up a directory though.
