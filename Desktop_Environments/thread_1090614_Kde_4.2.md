I am using Ubuntu 8.10, and have added the kubuntu-desktop pack, but this contains 4.14 of KDE. Is there any way to upgrade to KDS 4.2?

### Post by perlluver on 2009-03-08
For KDE 4.2, visit [http://www.kubuntu.org/news/kde-4.2](http://www.kubuntu.org/news/kde-4.2), and follow the instructions for Intrepid.  As for the Kubuntu-Splash, install startup-manager and change it to the Ubuntu-Usplash.
