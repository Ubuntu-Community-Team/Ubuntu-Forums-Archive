### Post by Dennis N on 2019-05-17
Which version of Lubuntu? Drag and Drop only works with Lubuntu 18.10 and 19.04. The trick is you have to drop it over the quicklaunch bar, not elsewhere on the panel. Then it adds the icon. It initially goes to the right of the existing icons. Then adjust position with 'right-click > move left' if desired. 

If you use 18.04, follow post #2 advice, which doesn't apply in 18.10 or 19.04.

Note: Your 'Distro' entry says Ubuntu 18.04 so we can't tell from that.

### Post by Liamdale on 2019-05-17
I have finally understood how the panel works.  Reviewing and experimenting with the panel configuration I discovered that the task manager takes up most of the panel.  I reduced the maximum button size which seems to reduce the task manager size.  I then succeeded in inserting application Icons on the panel.  In other distros I have used the task manager and panel seem to be integrated.

I will also update my distro information
