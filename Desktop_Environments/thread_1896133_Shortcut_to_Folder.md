### Post by WinRiddance on 2012-01-27
I don't think that you can do that ever since Ubuntu 11.04 with the "Unity Enhanced" Nautilus file manager. I had the same problem since I just wanted home, system, mounted drives, and my bookmarks to show. An alternative is to install the **pcmanfm** file manager instead. It's almost identical to the previous Nautilus version, but without some of the context menu options. Heck, just play with both of them and see which one you like better.

You can try **Thunar** too, plenty of folks like that one as well. Last but not least, create yourself new folders within Pictures, Documents, Movies, etc., and then create bookmarks _to just those_ folders. For example ... create a folder entitled My Documents within the default Documents folder. Dump all of the contents of documents into My Documents, and finally create a shortcut to the My Documents folder. That ought to do the trick, no? :D
Please mark the thread as solved for others, if this answered your question.
