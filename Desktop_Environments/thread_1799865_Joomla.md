### Post by DavidTangye on 2011-07-11
I have installed a few Joomla 1.0, 1.5 and now 1.6 onto various versions of Ubuntu, over the past several years. It has all been reasonably painless, as long as you follow the installation notes from Joomla. You need to install apache, mysql and php5 first, and the installation notes from Joomla cover this.

Joomla 1.6 is currently working on my Ubuntu 10.04, but I currently have a problem with Joomla 1.6 on Ubuntu 11.04, which i am currently looking into. See [this thread on the Joomla forum]("http://forum.joomla.org/viewtopic.php?f=622&t=634531").
