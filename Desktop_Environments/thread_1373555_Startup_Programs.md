### Post by tpjk on 2010-01-06
to make certain programs run at startup go into preferences -> startup applications.

if you want certain applications to start on specific workspaces you could try adding window rules in your compiz settings: in ccsm, go to 'place windows' and activate it; in the tab 'fixed window placements' add some rules where it says 'windows with fixed viewport'. you can check out the following link if you need more information about window rules: [http://wiki.compiz.org/WindowMatching](http://wiki.compiz.org/WindowMatching)
