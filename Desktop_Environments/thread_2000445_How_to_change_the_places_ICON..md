### Post by neu5eeCh on 2012-06-09
Figured it out.

The icon is under /usr/share/icons/Faenza

From there, there are various scaled .png's to be found under apps. The icon is called:

system-file-manager.png

I replaced every occurrence in the Faenza directory with the icon I wanted. I'm happy now. O:)
