### Post by wayfarer_boy on 2007-04-04
Just like to add that you can shift the window a number of pixels at a time with the arrow keys whilst using the 'move' option with the 'on-top' option.  You can get to both options through the context-menu by Alt-right-clicking the window.

I was wanting to position my 1280x800 output window from processing to the centre of the screen (rather than under the title bar, as there isn't a fullscreen option for the java app), and found this was the best way to do it.
