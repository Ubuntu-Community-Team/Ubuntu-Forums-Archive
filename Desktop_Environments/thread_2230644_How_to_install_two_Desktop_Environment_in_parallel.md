### Post by oldrocker99 on 2014-06-20
You could also try installing another lightweight desktop:

```
sudo apt-get install lubuntu-desktop lxde
```

This will install my own preferred lightweight desktop, and the theme problems (the odd colors) **may** go away.

I had wondered if it might be your display manager. Ubuntu, Xubuntu, and Ubuntu all use lightdm as a display manager (which is also what you log in to; the desktop you get after you log in is called a window manager), so that probably isn't the problem. However, just to be safe:

```
sudo dpkg-reconfigure lightdm
```

This will open up an interface in which you can see which display managers are installed, and change from one to another if you wish. Select **lightdm**, even if it's already selected, and finish.

This may be a stab-in-the-dark suggestion, but it occurred to me while reading the thread. Good luck!
