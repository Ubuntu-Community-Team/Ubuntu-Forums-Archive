### Post by Ampersand on 2005-12-09
Firstly, install crystalcursors by selecting it in Synaptic or running
```
sudo apt-get install crystalcursors
```
You can then select the set of cursors you want by going to System - Preferences - Mouse (or Mouse in the gnome-control-center), and going to the Cursors tab.
