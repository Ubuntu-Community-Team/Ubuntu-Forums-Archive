### Post by W.McL on 2009-08-22
Totem is suppressing the start of the screensaver. Somehow this has and effect on some of the notifications.

For audio files this behaviour can be disabled by setting the option "Allow the screensaver to activate even when audio-only is playing" ("Display" tab in the "Settings dialog)"
