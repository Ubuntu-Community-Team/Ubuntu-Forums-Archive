### Post by Qrk on 2006-02-14
Use the link below or automatix. (I recommend it manually, particually if you are in a class)

[https://wiki.ubuntu.com/RestrictedFormats](https://wiki.ubuntu.com/RestrictedFormats)

As for the amarok file, read up on how to enable multiverse/universe (if you havent already) and use synaptic to install it. Life is too short to not use apt. 

```
sudo apt-get install amarok
```

I'm not sure what version is in the repositories.

Aysiu has a good write up on how to install software in Ubuntu in his signiture, You can install from source (your tar) but amarok depends on parts of KDE so its not going to be nice to install without apt.
