### Post by Fass on 2006-08-19
It is, but that's not the kernel a stock Ubuntu Dapper uses. It will be the kernel that a stock Edgy Eft (the next version of Ubuntu) uses, though.

It's like this: as Dapper was being developed, 2.6.15 was the kernel of the day and that's what they used. Since then, Linus and the rest have released newer kernels, but Dapper stuck with the one it was developed with for stability's sake. Ubuntu devs still add security patches from the newer kernels to 2.6.15, but won't use the entire newer one until the next release of Ubuntu.

This of course doesn't preclude you from running the newer one by compiling it yourself. There are several guides on how to do that in the HowTo forum.

### Post by Fass on 2006-08-20
As you can see, the Dapper kernel is not 15.26 but 15-26. A dash, not a point. The -26 indicates it's version 26 of the Dapper 2.6.15 kernel. The point releases on kernel.org indicate revisions of the "vanilla" kernel - revisions probably incorporated into the Dapper kernel, but which has seen additional things added and changed in it, which explains why it's up to "-26."
