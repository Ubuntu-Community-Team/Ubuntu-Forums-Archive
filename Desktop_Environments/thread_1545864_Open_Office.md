### Post by gdonwallace on 2010-08-04
Unfortunately you cannot open a .odt file in Word.  MS has not implemented compatibility for that format of file as of office 2007.  Not sure about office 2010 though.  

However; you can save the file in OpenOffice as a .doc and you should not have any problems.

---

### Post by mcduck on 2010-08-04
There is a third-party plugin for Word that allows it to read .odt files.

However, like gdonwallace mentioned, OpenOffice is also able to save files in Microsoft Office's formats.

On the other hand if you want to be sure that the receiver sees the document exactly as you created it, and the receiver doesn't need to be able to edit the document, you should export it as PDF.

### Post by Dragonbite on 2010-08-04
Yes they can, if they are running Microsoft Office 2010.

Previous versions of Office can, providing they have the plugin. The one I had from Sun (pre-Oracle) wasn't "Open" or "Save As", instead provided "Import" or "Export" functionality.

There is also the option of uploading it into Google Docs, and then Exporting it as a Word document.

### Post by howefield on 2010-08-04
