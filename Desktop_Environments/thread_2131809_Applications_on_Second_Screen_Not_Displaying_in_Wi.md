### Post by forliberty on 2013-04-02
I've been running Ubuntu 10.04 with dual monitors for years, but I just started started getting some strange (and frustrating) behavior. Whenever I move a window from my main screen to the secondary screen, the window disappears from the Window List or taskbar. I did check to make sure that my Window List preferences are set to "Show windows from all workspaces." I have an Nvidia GeForce 210 with the proprietary drivers (version 195.36.24). I am using the TwinView mode (although I tried switching to separate X screens, and that did not help). I also tried swapping the primary and secondary screens, but that just moved the problem from one screen to the other. Any ideas??

---

### Post by forliberty on 2013-04-04
Well, I haven't found a solution to this problem, but I have found a workaround. I created a panel on the secondary screen and added a window list to that panel. Now I have a separate window list for each screen.
