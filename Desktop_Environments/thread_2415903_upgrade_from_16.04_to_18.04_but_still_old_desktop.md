I have a system with a fresh install of Ubuntu 18.04 LTS.  everything is fine.  I also have a system that was upgraded from 16.04 to 18.04.  It has a slightly different desktop, basically the one from 16.04.  How do I get the new desktop on the upgraded system?

---

### Post by deadflowr on 2019-04-02
Switch the desktop at login.
Upgrades should retain the old desktop, but also add the new one.

If the login screen is the new one Ubuntu now useds, in order to switch desktop sessions click on the user and next to the sign-in button is a cog icon, click on it the reveal the available desktops.
Current Ubuntu's desktop session is Ubuntu.
Old desktop was and is now called unity.

hope it make sense
