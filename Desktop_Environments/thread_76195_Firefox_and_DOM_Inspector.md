How can I use/install the DOM Inspector that comes with Firefox? 

I'm using Firefox 1.0.7.

---

### Post by darkmatter on 2005-10-14
```
sudo apt-get install firefox-dom-inspector
```

You'll find it in the Firefox menubar, under 'Tools' after install
