### Post by deeminter on 2010-10-25
Not a one click feature that adds the sender to address book, but there is a Multi-click(3) method. At the top of the screen under Message, there is an option to add the Sender to Address Book. This brings up another option box, where you select which Address Book (Ubuntu One, or a local address book in Evolution.
