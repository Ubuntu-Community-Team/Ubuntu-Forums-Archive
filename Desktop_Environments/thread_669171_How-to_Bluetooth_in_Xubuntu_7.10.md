### Post by gn2 on 2008-01-16
I just received a bluetooth adapter in the post which I purchased for sending files to and from my mobile phone.

I use Xubuntu 7.10 and added these packages using Synaptic:

bluetooth (3.19-0ubuntu3)
bluez-utils (3.19-0ubuntu3)
libbluetooth2 (3.19-0ubuntu1)
gnome-bluetooth (0.9.1-0ubuntu1)
libbtctl4 (0.9.0-0ubuntu1)
libgnomebt0 (0.9.1-0ubuntu1)
libopenobex1 (1.3-3ubuntu1)

There is now a "Bluetooth File Sharing" icon in the Accesories section of my System menu whichactivates my USB Bluetooth.

Then I created a Send To item for Bluetooth in Thunar using the instructions near the bottom of the page of this guide:
[http://thunar.xfce.org/pwiki/documentation/sendto_menu](http://thunar.xfce.org/pwiki/documentation/sendto_menu)

Hopefully this info will help someone.
