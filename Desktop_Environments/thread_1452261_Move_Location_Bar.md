### Post by chaanakya_chiraag on 2010-04-11
That could be one of two things:
1) A mockup (meaning the feature doesn't exist right now)
or
2) A custom version of Nautilus (such as Elementary Nautilus, Simple Nautilus, etc.).  This means the feature exists, but only in some version of Nautilus.  You would have to figure out which custom mod of Nautilus this is and install it.  Then, when you restart Nautilus (by logging out and back in or by killing Nautilus), the new version of Nautilus should be running.
I don't think it's possible to do this with stock Nautilus, unless there's some obscure option I'm missing.
