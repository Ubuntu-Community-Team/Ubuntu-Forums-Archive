### Post by urukrama on 2009-01-21
I haven't used Deskmenu in a long time (not since the Dapper days), but I have used 9menu in Awesome. If I remember correctly, you need another application (such as the window manager) to launch 9menu when you click on the desktop. In other words, 9menu won't come up when you click on the desktop unless you specify it should do so in an application that can control the desktop.

You could launch xfdesktop (from Xfce). You'll get the Xfce desktop menu with it. You can run it with any window manager.
