I have recently installed KDE on Ubuntu. 

The thing is that Kubuntu-Desktop replaced my boot slash screen with the kubuntu one. I would like to have the Ubuntu one right back

Thank you

---

### Post by pmlxuser on 2009-08-13
```

$sudo apt-get install startupmanager

```

start the program after install and you will get a graphical mechanism for changing the startup screen and also for changing it to any other you might want.
