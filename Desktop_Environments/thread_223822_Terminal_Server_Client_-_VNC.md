My girlfriend's PC runs in 1024x768 (due to having a 15" monitor) and my PC runs in 1280x1024.

So, in the options I select "Show in 1024x768" - basically my PC via VNC shows up in a smaller window with scroll bars.

Now, after VNC'ing in, I can only use the scroll bars once. If I need to navigate to the bottom of the screen to bring up Evolution, I click and drag the scroll bars in the VNC window and I can go about my business.

If I wish to drag the scroll bars back up so I can see the top of my screen, it simply won't work. It's like once you have used the bars, you can't use them again until you close the session and start it again from scratch.

### Post by OpsVentus on 2006-07-27
The scrolling screen is how VNC deals with differing resolutions, so no fix there.

But you can scroll up again, I don't know why but if you want to scroll up you have to use the right mousebutton.
And then the left mousebutton to go down again.

I know Terminal server handles the difference in resolutions better, but I'm not sure if there's something like that for Linux.
