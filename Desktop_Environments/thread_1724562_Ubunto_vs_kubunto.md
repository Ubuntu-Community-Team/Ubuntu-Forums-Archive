### Post by 3Miro on 2011-04-08
The difference is KDE vs Gnome. You can try both environments by installing Ubuntu, then from Synaptic Package Manager install kubuntu-desktop. Then logout, select KDE and login. You can then try both Gnome and KDE and see which one you would like better.
