### Post by ankspo71 on 2010-10-18
Hi,
If you haven't done so already, right click on the zuma installer, select properties, then in the permissions tab, mark the file as executable. Now try to open the file with wine again.

Also, make sure you have wine1.2 installed, because there is a wine1.0 too.

This link below is where you can see the ratings and reviews for Windows applications running in wine. Zuma's Revenge is in there too.

[http://appdb.winehq.org/index.php](http://appdb.winehq.org/index.php)

Hope this helps.
