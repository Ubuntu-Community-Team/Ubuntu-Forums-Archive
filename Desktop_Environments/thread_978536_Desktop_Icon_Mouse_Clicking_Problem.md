### Post by L815 on 2008-11-11
It's a safe option to disable compiz temporarily to check if that may be the resulting issue.
If you are unsure how to do so, navigate to Syste -> Preferences -> Appearance.
Click the 'Visual Effects' tab, and click "None".

If doing this allows expected functionality with the desktop icons, you have found the culprit!
