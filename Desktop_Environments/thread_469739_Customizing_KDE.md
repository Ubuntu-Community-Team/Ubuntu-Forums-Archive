### Post by luisromangz on 2007-06-10
Kicker is the KDE panel, where the tasks and the notification area are placed, and KBFX is KMenu replacement applet.

I use the QtCurve widget style, which can be found in Kde-look. I think it's the best style out there because it is very customizable, and there are versions for KDE and GTK, so all your apps will look the same. In KControl, there is a panel in which you can customize the appearance of GTK apps when running under KDE, so you can select QtCurve there.

Bye!
