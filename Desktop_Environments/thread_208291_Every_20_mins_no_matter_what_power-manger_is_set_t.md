### Post by avtolle on 2006-07-03
I had the same problem w/my Mac G4 Cube. What I discovered was that the power manager was not enabled. So, as I recall, after fumbling and stumbling around for a while, I added gnome-power-manager to the programs which are run on start up. This was done through System --->Sessions--->Startup Programs; agaain, from memory, the power manager program was "greyed out", and when I selected Add, it became active, I clicked on it, and to shorten this too long post, haven't had any troubles since. Hope this helps.
