### Post by sgosnell on 2009-11-22
Edit/Preferences/Content/Fonts and Colors/Advanced.  You can set whatever fonts you like, and you can uncheck the box for allowing websites to use their own fonts, keeping your choices all the time.  Some sites use fonts which don't work well on all monitors, so you can change them to what you like.  Some sites use a default font, which is displayed as whatever you have selected as your default.
