### Post by 23meg on 2007-03-02
If you open, say, ten windows, all maximized, none go black? 

The following is a known workaround for the black window bug, albeit with a compromise of performance: start Beryl with

```
beryl --force-aiglx --use-cow
```

I don't know if this is still valid (maybe autodetected?) in recent Beryl versions since I don't use Beryl any more.
