In Ubuntu 10.10, I was using the following theme:

art.gnome.org/download/themes/metacity/1337/MCity-VistaBasic.tar.gz

How do I install this theme in 12.04?

---

### Post by Frogs Hair on 2013-02-11
1. Open the  Home Folder press Ctrl + H and create a .themes folder. 

2. Extract the download package with a right click extract here.    

3. Move the extracted theme folder into the .themes folder and close the home folder

4.Choose the theme from the Gnome Tweak Tool which is in software center if not installed.

I just tested the theme on 12.10 and it seems to work.
