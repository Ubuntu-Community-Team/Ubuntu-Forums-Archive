### Post by glug101 on 2006-10-20
By 'graphic interface' do you mean:

1.  A graphical means of connecting to the computer remotely like VNC.

2.  A graphical windows manager (like Gnome or Enlightenment) to run on the remote computer that will work well with your chosen remote connection utility.

If it's the 1st option, the only thing I can think of is VNC.  VNC is dead slow on a slow network, but it works, and it has a web component so you don't even have to download a viewer.  There is XDMC logins, which I use on my personal network, but I don't think it would work well over they internet (not to mention that I don't think it will work if you are connecting on a windows computer).  

If it's the 2nd option, twn (tiny window manager I think) would work the best.  It's ugly, but there is practically NO eye candy to eat up bandwidth, and I think it will work in 16 color mode.

Did this help?
