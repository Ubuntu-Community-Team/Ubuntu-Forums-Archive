### Post by bobbob94 on 2006-10-09
To install Azureus, just right click on the  file you downloaded (Azureus_2.5.0.0_linux.tar.bz2), then select extract here. This gives you a new folder called azureus, go into this, and read the readme.txt you'll find there for info on what to do. (You just need to click on the file "azureus" and select the option to run in terminal, it'll then install. Azureus also needs Java to work, so if you don't already have that installed you can do that from Synaptic Package Manager)
