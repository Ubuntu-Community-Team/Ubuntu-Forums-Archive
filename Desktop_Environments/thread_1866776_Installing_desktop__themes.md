### Post by 2F4U on 2011-10-22
1. Go to, for example, [xfce-look.org]("http://ubuntuforums.org/xfce-look.org") and download the themes you like

2. Go to the directory where you downloaded the tarball/file and extract it.

3. Move the extracted folder to /usr/share/themes (for all users) or ~/.themes (for just you)..

4. GTK theme is available here: Menu --> Settings --> Appearance

5. You select your xfwm theme in: Menu --> Settings --> Window Manager
