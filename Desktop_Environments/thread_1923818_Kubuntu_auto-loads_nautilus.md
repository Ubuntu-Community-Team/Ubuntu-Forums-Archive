### Post by drmrgd on 2012-02-11
It could be that you have "Restore Previous Session" turned on.  Go to System Settings -> Start Up and Shutdown -> Session Managment, and in the On Login section, click Start with an empty session.  I noticed that sometimes the Restore Previous Session actually sort of loads some things that were not present at logout of the previous session.  Anyway, give that whirl.
