### Post by evets25 on 2008-07-17
I think what he's looking for is in "main menu -> system -> preferences -> sessions". From there you can add a program to automatically be run whenever you log in, though you do need to know the exact command to run. Note that this is for gnome; KDE has it's own way of doing this.

What you put in the box is the name you want to appear in the list, and the command to run the program. So for instance, I have this neat little calendar program that displays a calendar on my desktop, and to have it run at startup, I add an entry to sessions, and I fill in the name as "Rainlendar" and the command I put in is "rainlendar2" (which is what I put in to terminal to run the program).
