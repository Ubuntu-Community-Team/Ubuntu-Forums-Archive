### Post by steveneddy on 2009-12-23
The app on the right of the screen is called conky

```
sudo apt-get install conky
```

Delete Gnome's bottom bar and move the top bar to the bottom.

[gnome-look.org]("http://gnome-look.org") will have the theme for Emerald (transparent window borders used with Compiz)

along with the pointer theme and the icon theme

Did the web site you got the pic from say anything about the set up?

EDIT:

If you find the themes you want to install we will help you if you need it.
