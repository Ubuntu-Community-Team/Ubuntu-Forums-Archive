### Post by gweaver on 2007-08-14
I've used the debian package from [http://dvbcut.sourceforge.net/](http://dvbcut.sourceforge.net/)

The i386 version seems to work fine on Feisty, unfortunately there is no AMD64 version available there.

It's great for editing video recorded from Freeview, to remove the adverts and the junk before and after the program.

I think this should be included in Ubuntu Studio and probably Ubuntu itself. It should be pretty useful for anyone with a DVB card.
