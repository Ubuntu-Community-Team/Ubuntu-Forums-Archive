### Post by Shepherd X on 2011-04-14
I don't want this to grow into a debate, but I just wanted to say that I've been a Windows user for almost 20 years (starting with Windows 3.0). I've been using Ubuntu 10.10 for a number of months now. I tried Linux Mint for some months too, but I think it was because I needed to be weaned off Windows. I've seen numerous screenshots of the Gnome 3 Shell and of Ubuntu Unity, and I think they are very innovative. I've also read articles praising and criticizing the goal of these interfaces. I know that a lot of people have complained and the main cause of their complaining is the deviation from the expected, historical norm. We shouldn't fear change and innovation. It is this very lack of fear (courage) that pushed many of us to try Linux. If we were to have insisted upon what we've been adapted to, none of us would have abandoned Windows except under coercion. I want to congratulate both the Ubuntu and Gnome team for their innovations and hope that if there are any mistakes, that we all can learn from them together as a FOSS community.

Long live Linux! Down with ultra capitalist software.
