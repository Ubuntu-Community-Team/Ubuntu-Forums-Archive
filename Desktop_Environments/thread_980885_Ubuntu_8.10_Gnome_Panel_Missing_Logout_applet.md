### Post by mcduck on 2009-02-16
> **RedMartin said:**
> Sorry to resurrect this thread but I've had this too and semi-solved it using this thread.

This applet was originally on the extreme right of the top panel, next to the time. How do I get it back there and also remove the text from next to it so that it is just the icon?

You can move the panel applets freely around by dragging them with middle mouse button, or by right-clicking them and selecting "Move".

If you are not able to move some applet, or to move one applet past another one, right-click them and make sure "Lock to panel" is disabled.

To remove text from the User Switcher-applet right-click it and select Preferences". You'll be able to select the applet's style from there.
