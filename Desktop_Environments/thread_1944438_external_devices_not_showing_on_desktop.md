### Post by kazztan0325 on 2012-03-21
Hi samopn,

There is another way to deal with this problem.

1. Install **"Advanced Settings"** (its package name is **"gnome-tweak-tool"**) with _'Software Center'_ or with _'Synaptic Package Manager'_.
2. Open up **"Advanced Settings"** window.
3. Click on **"Desktop"** on the side pane.
4. - Have file manager handle the desktop => **ON**,  - Show mounted volumes on the desktop => **ON**

Hope this helps.
