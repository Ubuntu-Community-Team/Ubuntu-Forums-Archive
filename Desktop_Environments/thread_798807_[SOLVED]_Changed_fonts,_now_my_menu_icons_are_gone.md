### Post by Wandering on 2008-05-20
It is hard for me to see the connection, but there's a lot I don't know.

Here are my settings obtained by right clicking on the window background, selecting Change Desktop Background, and then clicking the fonts tab:

Sans, Sans, Sans, Sans Bold, Monospace, Best Shapes checked.

In the Interface Tab"

Check   Show Icons in Menus
Select  Text Below Items
Uncheck  Editable Menu Shortcut Keys

That was the standard default settings, and even with all the whiz bang features running, I haven't changed that.  I hope it helps you.
