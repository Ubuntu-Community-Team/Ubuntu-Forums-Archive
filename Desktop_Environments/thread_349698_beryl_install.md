### Post by Brunellus on 2007-01-30
thread moved to appropriate sub-forum.  Beryl is BETA SOFTWARE and therefore NOT RECOMMENDED FOR ABSOLUTE BEGINNERS.  Install it at your own risk.

General advice:  learn to be comfortable in Ubuntu & Linux first before you go off beta-testing.  

If you DO decide to install Beryl:

1) Keep careful notes, especially noting WHOSE directions you are following.

2) Do not expect it to be worry/bug free.

3) In the event that it does crash, file useful crash data and forward this to the developers.

If you want to walk on the cutting edge, you must understand that you might bleed.  You have been warned.
