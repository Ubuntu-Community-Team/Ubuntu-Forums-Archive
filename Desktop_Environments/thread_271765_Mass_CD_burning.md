### Post by JGuru on 2006-10-05
You can use **Gnomebaker**, CD/DVD burner. It's very good one. In case 
 You haven't installed it. Open the Terminal Window & type:

 $ **sudo apt-get install gnomebaker**

  Wait until the installation is complete. You can also try **K3B**, it's
 based on KDE. You don't need to install KDE!!! It's also a very good application.

 $ **sudo apt-get install k3b**.

  Use whichever CD/DVD burner, 'Gnomebaker' or 'K3B'
