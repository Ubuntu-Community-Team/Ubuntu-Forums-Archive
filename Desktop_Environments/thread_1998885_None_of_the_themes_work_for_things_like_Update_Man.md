Currently none of themes that I change under the "Apearance" seem to effect applications such as Synaptic Package Manager or Update Manager, instead they use a grey windows looking theme. ### Post by brainwash on 2012-06-07
Those themes are indeed "incomplete", they simply don't support GTK3 yet. So you have to find a GTK2/GTK3 compatible theme like the Greybird one.
