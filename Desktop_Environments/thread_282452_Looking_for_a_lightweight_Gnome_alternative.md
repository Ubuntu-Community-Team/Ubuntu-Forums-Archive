### Post by EagleX on 2006-10-22
I recently installed Ubuntu, and it has been a great OS so far. My only pet-peeve is the memory usage from the Gnome desktop. It's using up almost 60 MB of system memory, and I only have 256mb of RAM total. I'm not running the newest computer, and I'm not looking for it to be super fast, because I know older computers are slow. But it seems to me that 60 MB just for a desktop environment is a little steep.

I've looked around at a few other desktop environments. From what I've read, it looks like KDE and Gnome are both RAM hungry programs, so I'd like to stay away from them. I installed Xubuntu and used the Xcfe desktop environment. It was a little better.

So what I'm looking for is a lightweight desktop environment, one that is configurable, looks half-decent (I'm not picky about flashy icons / themes, but I'm not thrilled about square chunky "command-line-text" minimal either), and runs efficiently. I use Firefox allot, as well as OpenOffice, and XMMS, which take up quite a bit of memory. I'd just like something that doesn't stall, and can run well!

Not to say that I don't like Gnome, I actually have grown to like it allot, so if there are some ways that you can cut down on it's memory usage, those would be appreciated as well.

Thanks! :-)

May be worth your while either checking out ["Fluxbox"]("http://www.fluxbox.org") which is a lightweight WM but really functional, and after a little bit of reading, highly configurable..... or having a look at ["FluxBuntu"]("http://www.fluxbuntu.org") which is basically a server install of Ubuntu, with Fluxbox already installed & ready to go. It saves the hassle of doing a server install etc yourself, and is a really great little Distro. In it's infancy at the moment but growing rapidly. :)

[OpenBox3]("http://icculus.org/openbox/") is really good too. Very lightweight (Doesn't even have a panel) but again very configurable. Probably needs a bit more work than FluxBox to get it how you want it, but it certainly does fly. :)

Last but not least, is the amazing ["IceWM"]("http://www.icewm.org") which has pretty much everything you need from the get-go, is incredibly fast & lightweight, and very easy to configure. 

I use all 3 by the way, on and off, plus XFCE when i feel like a change. But my main one is still FluxBox..... It's just really nice, with a good balance of features and still small & fast & (Relatively) easy to configure.

Have fun...... :)
