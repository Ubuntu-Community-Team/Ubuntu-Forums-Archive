### Post by angry_johnnie on 2008-05-26
I don't usually tinker with xfce too much, but, unless I have a really bad memory, xfce doesn't like to be borderless.

I remember being left without window decorations a couple of times, and then I couldn't even type anything into a terminal to get things back to normal.  I had to start with a new, clean, session.

Maybe you'd be better off with a window manager that is meant to behave that way.  Try Xmonad, dwm, wmii, or awesome.  All of them are available in the repositories, and, while they're not exactly easy to use, they're not all that difficult either.

They're tiling window managers, but can be set to floating mode.  Read up on them, they're actually pretty good. :-)
