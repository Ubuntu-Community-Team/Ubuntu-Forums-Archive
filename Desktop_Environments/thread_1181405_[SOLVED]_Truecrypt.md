### Post by sgosnell on 2009-06-07
It will work.  You need to install Truecrypt on each computer, obviously using the correct version for the OS, but the removable drive which is encrypted can be accessed by any computer that has Truecrypt installed on it, regardless of the OS.

### Post by sgosnell on 2009-06-07
You have to have software on the OS to do the encryption/decryption.  Truecrypt will run on Windows or Linux, and either version can deal with the encrypted volumes.  All you have on the removable device is the encrypted data.  Without the software on the OS, it's not accessible, and even if you have the software installed, you have to enter the correct password.  That's encrypted on the removable device.
