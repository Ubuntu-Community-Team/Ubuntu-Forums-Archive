As far as I remember, there is no easy way to completely reset the panels. However, as there are a small number of applets involved in a default configuration, you should be able to recreate it very quickly. Just delete one of the panels, remove everything from the other panel, then re-add stuff in the following order:

Top: menu bar, Firefox launcher, Evolution launcher, Help launcher, <lots of free space witing to be filled>, notification area, volume icon, clock, quit
Bottom: show desktop button, window list, <lots of space for the window list to fill>, workspace switcher, trash icon.
