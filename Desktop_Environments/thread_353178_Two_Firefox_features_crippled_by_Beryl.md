### Post by AstarothCY on 2007-02-04
Yes, this is another Beryl thread and I apologise profusely for that, but this is a sort of obscure issue and I have found nothing on this by searching.

I use a wonderful Firefox extension called [MileWideBack]("https://addons.mozilla.org/firefox/334/"). This adds a 1px bar to the left edge of your FF window and functions as Back/Forward and for using the scroll wheel to change tabs, so whenever you want to change tab you just pull the mouse all the way to the left and scroll, it really is a magical little extension.

This extension does not work if you are using Beryl. What happens is, if the FF window isn't maximized, you can see the MWB bar and if you hover over it, it works fine, but when you maximize the window it disappears. This, however, varies depending on what theme engine you use. Some engines will do this, other engines add a very thick window frame so whereas the MWB bar is there and functional, you have to aim for it because there is window frame to its left, so it really defeats its purpose.


The second thing Beryl screwed up for me is a nifty little feature of forums such as this one which use vBulletin. This feature allows you to submit your post by just pressing Alt+S, so as soon as you're done typing your post you just press Alt+s and don't need to pick up the mouse and click the Submit button. This, for some reason, does not work if you are running Beryl. I have looked through the keyboard shortcuts and made some alterations but it still does not work.
