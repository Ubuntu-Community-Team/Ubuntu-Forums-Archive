### Post by h4rdc0d3 on 2005-08-16
MPlayer can do hard and soft frame dropping for extremly slow computers. Also, it does not require X Window System as it can use the console and show the movie on the framebuffer. See [ubuntuguide.org]("http://ubuntuguide.org/#mplayer") for help on installing.
