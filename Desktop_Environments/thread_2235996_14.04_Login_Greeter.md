### Post by deadflowr on 2014-07-24
Are you running lightdm or gdm?

unity-greeter does exactly what you want.
It's built for lightdm.

look over my response in post#2 in this thread
[http://ubuntuforums.org/showthread.php?t=2235834](http://ubuntuforums.org/showthread.php?t=2235834)
The default for unity-greeter is to show user name.
It defaults to the last user logged in/out.
(ie if you have multiple users, the user to login/out last will be the user to be shown)

If by chance you don't know which display manager is in use
run
```
sudo dpkg-reconfigure lightdm
```
and proceed to follow it and select lightdm.

I don't use gdm, normally, and as such have no idea what needs to be done to change it's settings.
