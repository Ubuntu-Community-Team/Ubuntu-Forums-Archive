### Post by matthew on 2006-05-13
It's not listed here, but similar ones are. 

[https://wiki.ubuntu.com/HardwareSupportComponentsScanners]("https://wiki.ubuntu.com/HardwareSupportComponentsScanners")

However, here's someone who says his works.

[http://www.justlinux.com/forum/showthread.php?t=144753&goto=nextnewest]("http://www.justlinux.com/forum/showthread.php?t=144753&goto=nextnewest")

I would say the best place to start is to plug it in and see what happens. Then start the sane program and see if it finds the scanner Applications->Graphics->XSane Image scanning program

If Xsane isn't installed the package name you want is "xsane" and you can find it in Synaptic. If you need more help doing that, just ask.
