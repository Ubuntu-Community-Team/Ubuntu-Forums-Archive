### Post by efflandt on 2010-01-20
Look though /var/log/Xorg.0.log to see which modes X recognizes.  You can experiment with other modes using **xrandr** in a terminal (see **man xrandr**).  Although run from a terminal that only applies to current session (if you mess up you could log out/into X again).  For more info and to automate a setting you find works see [https://wiki.ubuntu.com/X/Config/Resolution](https://wiki.ubuntu.com/X/Config/Resolution)
