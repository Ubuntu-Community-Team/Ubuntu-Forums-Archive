### Post by dabang on 2005-04-21
Hi Jacobo,

I don't know about the *.deb packages, but if you install the drivers from the nVidia site, it's recommendet to comment out
#Load	"GLcore"
#Load	"dri"
under Section "Module".

By the way, I'd prefer the orignal drivers from the nVidia site, the installation isn't so hard: 
1. Get the drivers from [here](http://www.nvidia.com/object/linux_display_ia32_1.0-7174.html) and make the file executable.

2. You need the coresponding headers for you kernel, too. For exampel, if you use kernel "linux-image-2.6.10-5-386", then you have to install "linux-headeres-2.6.10-5" and "linux-headers-2.6.10-5-386". You can do that via synaptic.

3. If you havn't done so by now, you need to install gcc: ```
sudo apt-get install gcc
``` 

4. Quit X. If you use GDM, then a ```
sudo /etc/init.d/gdm stop
``` will do the job. 

5. cd to the directory where you saved the donwloaded nVidia-file and execute it with ```
sudo ./filename
```. If it comes to the question, if the installer should look for a kernel-module at the nVidia ftp-site, I say NO and let the installer compile a new one. Shoudn't take to long.

6. When done, edit your xorg.conf. I do that with midnight commander (from universe). ```
sudo apt-get install mc
``` (press "F4" for editing)
Comment out
Load	"GLcore"
Load	"dri"
under Section "Module" by putting a "#" in front. 
Make sure ```
Load "glx"
``` is there.
In the device section alter the driver from "nv" to "nvidia", save the file (with "F2" if you use mc).

7. ```
sudo shutdown -r now
``` will restart your system an you're done!

If you have any lock-up problems read appendix F of the nVidia readme or take a look at [this](http://www.ubuntuforums.org/showthread.php?t=26385&highlight=nvidia)  
thread.

Cheers!
