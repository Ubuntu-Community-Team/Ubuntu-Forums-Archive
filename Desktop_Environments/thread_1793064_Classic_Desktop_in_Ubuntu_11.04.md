### Post by wightghost on 2011-06-29
Hi, 

If you mean the gnome desktop (instead of using Unity), enter your name and password on the login screen then, before you login, look down the bottom of the screen and you will see a drop-down box where you can select the desktop.  It most likely says "Ubuntu".  Simply select Ubuntu Classic to get the gnome desktop then login.

Hope this helps

### Post by pjd99 on 2011-06-29
And if you are already in classic and you want things back to the way they were on 10.04 (buttons on the right etc) try installing ubuntu tweak - you could find the values in gconf-editor and change there, but tweak makes it easy.

If you want the old scroll bars back:
```

sudo apt-get remove overlay-scrollbar liboverlay-scrollbar-0.1-0

```
