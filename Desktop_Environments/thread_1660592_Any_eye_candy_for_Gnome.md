### Post by Crazedpsyc on 2011-01-05
Gdesklets is a good screenlets alternative, although it has a few drawbacks, it is what I  have used for a long time.
Avant-window-navigator - the best dock ever! Fully configurable, cool animations, plenty of applets (including one for comics) and the autohiding is great, so I have a big pop-down digital clock as well as using it to almost entirely replace the gnome-panel. 
Compizconfig - turn on some amazing 3D and 2D eyecandy effecs like a cool window switcher and a cube for the workspace switcher
Synapse - a simply amazing run dialog/search tool with tracker search. get it on lauchpad: [https://launchpad.net/synapse-project](https://launchpad.net/synapse-project)
