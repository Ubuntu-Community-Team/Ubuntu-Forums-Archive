### Post by mmcmonster on 2006-02-11
I am running the clearlooks theme on Ubuntu 5.10.

Where are the icons for the theme located?  I want to change the icon for one of my folders to make it look like the home folder, but I don't know what file to point to in "Set Custom Icon" (When I right click on the folder I want to change and select properties).

---

### Post by endersshadow on 2006-02-11
If it's an installed theme, check in ~/.icons/

If you're using the default, check in /usr/share/icons

### Post by mmcmonster on 2006-02-11
Thanks.
  The icon I was looking for was in /usr/share/icons/gnome/scalable/filesystems
