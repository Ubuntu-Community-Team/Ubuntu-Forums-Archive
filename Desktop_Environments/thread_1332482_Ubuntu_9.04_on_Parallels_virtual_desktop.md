### Post by 3Miro on 2009-11-20
During the install of Ubuntu, it asks you for your name. It selects your first name as default user name (unless you change it on the second row). If your name is John Smith, your user name is probably john.

Then it asks for password and then to verify that. I don't know what happens if you don't provide a password, it should just not let you go on, but it may also set the machine with no password (it shouldn't be doing this, it is a bad idea).

Try to remember what you entered on the screen that asks for your name or try your first name + some possible passwords. If it doesn't work, you may have to reinstall the virtual machine.
