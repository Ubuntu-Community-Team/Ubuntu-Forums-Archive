### Post by davosmith on 2008-03-28
Beryl is all the fancy bits of Compiz that got added back into Compiz when it became Compiz Fusion.

If you have version 7.10 of Ubuntu, you already have Compiz Fusion and should not install Beryl as well.

In case you don't know what Compiz Fusion is - it's the bit that gives you all the fancy effects when you open and close windows (wibbly windows, rotating cubes, etc.) which you should already be enjoying.

### Post by bharadwaj on 2008-03-28
Beryl is a combined window manager and composite manager written in C using OpenGL to provide acceleration. It is designed to be highly flexible, extensible, and portable, all the while keeping in mind that the users know how they want their desktops to act better than we do. 

Beryl has merged with compiz a couple of months ago. No point in installing Beryl, it is now part of Compiz. Almost all plugins are already ported.

[https://help.ubuntu.com/community/CompositeManager/CompizFusion](https://help.ubuntu.com/community/CompositeManager/CompizFusion)
