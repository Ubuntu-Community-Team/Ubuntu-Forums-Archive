Ever since I installed compiz, when I double-click a title bar, instead of maximizing the window it collapses into the title bar leaving it by itself on my desktop; double-clicking it again expands it. I tried to find any setting in gset-compiz that controlled this but couldn't find one and my setting for "Double-click title bar to perform this action" in System -> Preferences -> Windows is still set to "Maximize." Anyone know how to get back the default title bar double-click functionality of gnome?

### Post by chavo on 2006-08-05
The setting was changed from the metacity setting and can now be changed with gcompizthemer as long as you have an extremely up to date version. Keep in mind that this is alpha software and in constant development it will break at times.

### Post by _simon_ on 2006-08-05
Open the themer, select your current theme and click EDIT

Select the last tab "CGWD Settings"

Change the top dropdown box from "Shade" to "Maximise/Restore"
