Thanks for replying, Pablo :D 

Proximodo is an open source customizable webpage filtering proxy server... You can find more information about it here... [http://proximodo.sourceforge.net/]("http://proximodo.sourceforge.net/")

The following is a quote from Proximodo's home page:

"When you access the Internet with your web browser, a lot of things happen that are hidden to you. The browser and the website exchange "headers" that define the behavior of the browser's cache system, create or use cookies, tell the website who you are and where you came from, and so on. Moreover, the page's html code contains hidden tags or javascript programs that control the browser's behavior, such as "mouse-over" animation, page layering, URL masking or pop-up messages. And last but not least, the page often contains ads and banners that pollute the screen and slow down your web experience
Proximodo acts as a mediator between the browser and the website. It "sees" everything that is going on and, according to your choices, modifies headers and html page content on-the-fly to make things right. You don't want pop-ups and banners? Just activate the corresponding filters and they will disappear. You want cookies to be deleted when you close your browser? Proximodo will do it for you. You want a fast navigation in the pages you already visited? Page caching will be forced"

Proximodo is actually a clone of a software called Proxomitron.  I have been using it for more than an year.  In fact I don't surf the Internet without it.  I have not come across even a single ad, ad banner, flash ad, flashing text, animated gif images, pop ups, pop unders or any other kind of annoyance since I started using Proxomitron (along with Firefox) in Windows XP (I no longer use Windows).  Browsing the Internet with Proxomitron/Proximodo sitting between your browser and the Internet is fast, clean and enjoyable.  I have tried other similar software but nothing comes even close.   Some of the software I tried doesn't really stop the ads from downloading... The ad is downloaded into the browser's cache... It is just not displayed in the browser which would be ok for some users who don't want to see the ads but it doesn't really speed up the loading of the page.  Proxomitron/Proximodo actually stops the browser from even contacting the ad server and therefore the webpage you are trying to see loads quicker and it doesn't contain any ads.... even the empty space left by the ads is intelligently used (I could post a couple of screen shots if you want to see the difference).  Now the problem is Proxomitron is Windows only.  It does work in Linux using Wine but it is not very stable.  Proximodo on the other hand is multiplatform.  I have been dying to use Proximodo ever since I completely switched to Linux (Ubuntu)

Here is a small list of what Proxomitron/Proximodo can do:
    *  Stop windows that pop-up, pop-under, or pop-over
    * Stop those un-closable endless banner chains
    * Stop pop-up JavaScript message boxes
    * Remove web-branding and other scripts tacked on by "free" web providers.
    * Convert most ads and banner pictures into simple text links
    * Freeze all animated gifs
    * Make blinking text appear as bold instead
    * Remove slow web counters
    * Stop web pages from "auto-refreshing"
    * Prevent pages from changing fonts
    * Get rid of or replace web page background images
    * Protect against getting "trapped" inside someone else's frames!
    * Make all frames resizable
    * Close top or bottom frame banner windows
    * Make background MIDI songs play only when you choose.
    * Remove status bar scroll-texts
    * Remove "dynamic" HTML from pages
    * Disguise your browser's identity and version from JavaScripts
    * Remove style sheets
    * Un-hide URLs when the mouse is over a link
    * Disable frames or tables altogether
    * Change or delete cookies
    * Change your browser's user-agent and other identifying fields
    * Hide where you've been previously from inquisitive web servers
and much, much more.
