### Post by K.Mandla on 2008-03-05
ROM?

IceWM is a natural for a Windows lookalike. That WinClassic2 theme has prepackaged icons for most applications (including the MS Office suite). IceWM has one configuration file for the menu, and it's really easy to decipher.

I didn't install the Tahoma font, but I could have. Raleigh is the default GTK1.2 theme, and it's a dead ringer for the traditional Windows background.

After that it's really just picking software that matches the theme. Xfe has an Explorer theme that will likewise follow the old Windows classic style. Other GTK1.2 applications will just look right.

As far as doing this in Gnome ... I'm afraid I can't help you. I try to avoid Gnome whenever I can. :???:
