### Post by wxnker on 2007-03-11
well how did you install? I'm installing it right now so I do not have the insight on "looking glass" yet. But maybe this info helps.

I used:
```
sudo aptitude install lg3d-core
```

To remove:
```
sudo aptitude remove lg3d-core
```

or maybe you can search for these in synaptic/adept:
lg3d-core 
lg3d-java3d
lg3d-jdk

and uninstall.

By using "sudo aptitude install" revoming packages is always easy. So I normally always use that method. :)
