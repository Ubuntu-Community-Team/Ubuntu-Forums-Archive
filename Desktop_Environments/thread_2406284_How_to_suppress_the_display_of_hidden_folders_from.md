### Post by GwibberFooey on 2018-11-18
When I save document, the Save As window displays **both** regular **and** hidden folders. I want to **suppress** the display of hidden folders in this window. How can I **achieve** this? 

I use Ubuntu MATE 18.04.

---

### Post by GwibberFooey on 2018-11-18
I have found the solution to my own question. In the Save As window, I right click on the header of the list of folders. In the context menu I find an item with a check-box called Show Hidden Files; I uncheck the check-box. The problem is resolved.
