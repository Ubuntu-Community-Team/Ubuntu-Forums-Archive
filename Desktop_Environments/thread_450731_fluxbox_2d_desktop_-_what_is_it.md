### Post by RedSquirrel on 2007-05-21
The link for the perl script is defunct. It looks like a way of switching workspaces. I don't need it. I find the commands that Fluxbox provides to switch workspaces are more than adequate. You can even do workspace warping with a mouse wheel if you have one (I don't).

Here are the entries from my ~/.fluxbox/keys file that I use to switch workspaces:

```
Control Mod1 Left :PrevWorkspace
Control Mod1 Right :NextWorkspace
Mod1 F1 :Workspace 1
Mod1 F2 :Workspace 2
Mod1 F3 :Workspace 3
Mod1 F4 :Workspace 4
```
