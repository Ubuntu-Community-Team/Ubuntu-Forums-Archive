### Post by LastDino on 2017-05-23
This depends on local settings, there is way to change it from your command line by editing local file but usually easier gui way is to go into **system settings>Language and region>let language be US English and select your region** 

That should work to get also correct format of symbols as well. You will have to log out and log in again. If this doesn't work then in your selected region Monday is actually not considered to be first day of week, then we will have to either set for the region which does or go edit your local file.
