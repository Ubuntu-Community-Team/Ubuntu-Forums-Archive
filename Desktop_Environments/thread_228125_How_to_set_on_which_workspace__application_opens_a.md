### Post by orb9220 on 2006-08-02
Now if you have programs open and you have save sessions on exit in your sessions manger.

Then it will launch those apps on reboot or login.

But I don't know if it remembers workspace location or not.

Just goto systems>prefs>sessions  and check that Automatically save changes to session is checked.

Open and place apps on desktop

Then logout and log back in.

Hope this helps
