### Post by Tiganjero on 2012-03-03
Open *System > Administration > Printing*, click *New* and see if your printer is listed there. **_If it is_**, enter the wireless network information using the panel on the printer and go into the aforementioned dialog and have Ubuntu look for a *Network Printer*. Once it finds the printer, it'll set it up automagically. **_If it's not_** you'll have to install *HPLIP* [[SIZE="1"](link)[/SIZE]]("http://hplipopensource.com/hplip-web/install/install/index.html"), unplug the USB cable and repeat the process from above. Hope this helps!
