### Post by Jethro_uk on 2012-09-16
I let Webmin upgrade my x2go packages today, went to login via x2go and had lost my panels.

After a LOT of dicking about, I downgraded the x2go server package to 3.0.1-5 and x2gognomebindings to 2.0.1-1, and recovered my panels.

Trying to manually start gnome-panel was giving an error about xrrgetscreen resources.

I hope this helps anyone else who has the same issues.
