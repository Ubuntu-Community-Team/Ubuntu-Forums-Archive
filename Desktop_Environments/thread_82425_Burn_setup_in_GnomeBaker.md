### Post by Dr. Nick on 2005-10-26
Not sure I fully understand your question so I may be wrong here

Are you tying to burn a iso file to a cd?

The name of the disc is built into the iso file, also iso must be burned a certain way. In gnomebaker go to actions, burn cd image and pick the iso. As far as I kow you cant change the name when burning iso files.

If you are trying to burn a data disc try a program called graveman, It can easily rename disc before burning. It can also burn ISO files to a cd the correct way. If you want to try it go to "Duplicate CD" and in the from field choose the iso and put the burner in the write to box.

EDIT
I think a blank disc will show up as cdrom untill you burn it with a name or burn an iso with the name built-in then it should change to the correct name
