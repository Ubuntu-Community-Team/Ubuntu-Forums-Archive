### Post by babbling on 2007-10-22
Gutsy got VGA-out on my laptop working, which I'm thrilled about. Now I have a 1680x1050 screen connected to my laptop and the laptop screen seems to mirror this, but only showing a 1024x768 (the maximum resolution that my laptop can do) portion of the screen.

I'm okay with the laptop screen just being a mirror (trying to configure the screens in the new "Screens and Graphics" program doesn't work) but whenever I maximise a window, it will maximise to the 1024x768 screen rather than the 1680x1050 screen. The only exception to this is that if I drag the window into the portion of the screen not visible on the 1024x768 screen before maximising, the window will maximise to the 1680x1050 screen.

I imagine this is a GNOME setting/quirk rather than a hardware thing, so I was wondering if anyone might know how I can set the screen/resolution that I would like my windows to maximise to?

Thank you to anyone who is able to assist me with this!

---

### Post by babbling on 2007-10-22
Turns out there is a workaround for this. After changing the resolution using System->Preferences->Screen Resolution, the windows will maximise to whatever resolution was set.

The initial resolution in there was 1680x1050, though. So it seems this is actually a bug.
