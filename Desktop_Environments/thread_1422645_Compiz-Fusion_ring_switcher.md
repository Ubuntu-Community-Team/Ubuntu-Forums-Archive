### Post by thunk77 on 2010-03-05
There's a very nice little program called Simple CompizConfig Settings Manager which is a simplified frontend to the illogically complicated CompizConfing Settings Manager (duh) :p

You can install it by terminal:

```
sudo apt-get install simple-ccsm
```

Or by the Software Center. (search for simple compiz or sth like that)

Then open the app by System -> Preferences -> Simple CompizConfig Settings Manager.

Select "Effects" tab and from a dropdown menu in "Switcher" select "Ring Swithcer"

There are some other options you might like ;)
