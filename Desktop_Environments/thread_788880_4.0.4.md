### Post by Zorael on 2008-05-10
Likely they just haven't made significant changes to konqueror-kde4, which the about box reflects. If you're concerned the upgrade might've failed, you could tag it for reinstallation and see if it updates.

Terminal way would be:
```
$ sudo aptitude reinstall konqueror-kde4
```
