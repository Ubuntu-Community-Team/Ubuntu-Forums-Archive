### Post by lowracer on 2007-06-19
I'm not sure who on the developer team should be notified of this, but when clicking the mouse to wake up the screen saver, the screen saver should "eat" that mouse click so it doesn't get passed on to the programs running "behind" the screen saver.   Every time I wake up my screen saver with a mouse click, the wake-up click gets passed to the random browser or application or icon that the mouse pointer happened to be hovering over.  This morning I nearly booked an unwanted flight to Houston with a random wake-up click.  Other mornings I've closed applications I've wanted to keep running, opened folders that should have best stayed closed, played audio files I didn't want to hear, and launched programs that should have remained dormant.  Other screensaver apps I've used on other systems all eat that mouse click.  Ubuntu should as well.  For the record, I'm running Kubuntu 7.04.
