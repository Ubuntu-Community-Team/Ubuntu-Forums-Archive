### Post by Koybe on 2007-01-07
You can reconfigure your Xorg configuration by doing this :

sudo dpkg-reconfigure xserver-xorg

Or you can simply add this resolution to your /etc/X11/xorg.conf in the screen section :
```

Section "Screen"
        Identifier      "Default Screen"
        Device          "NVIDIA Corporation NV35 [GeForce FX 5900]"
        Monitor         "CM766"
        DefaultDepth    24
        SubSection "Display"
                Depth           1
                Modes           **"1280x1024"** "1152x864" "1024x768" "832x624" "800x600" "720x400" "640x480"
        EndSubSection
        SubSection "Display"
                Depth           4
                Modes           **"1280x1024"** "1152x864" "1024x768" "832x624" "800x600" "720x400" "640x480"
        EndSubSection
        SubSection "Display"
                Depth           8
                Modes           **"1280x1024"** "1152x864" "1024x768" "832x624" "800x600" "720x400" "640x480"
        EndSubSection
        SubSection "Display"
                Depth           15
                Modes           **"1280x1024" **"1152x864" "1024x768" "832x624" "800x600" "720x400" "640x480"
        EndSubSection
        SubSection "Display"
                Depth           16
                Modes           **"1280x1024" **"1152x864" "1024x768" "832x624" "800x600" "720x400" "640x480"
        EndSubSection
        SubSection "Display"
                Depth           24
                Modes           **"1280x1024" **"1152x864" "1024x768" "832x624" "800x600" "720x400" "640x480"
        EndSubSection
EndSection

```Then restart xorg using CTRL+ALT+BACKSPACE

### Post by Koybe on 2007-01-07
If you installed nvidia drivers you can change 
```
Driver		"nv"
```
by 
```
 	Driver		"nvidia"
```

```
Depth		1
		Modes		"1280[COLOR=Red]*[/COLOR]1024" "1024x768" "800x600" "640x480"
```

You got an error there.

This is the only things I see. How did you installed your nvidia drivers?

Don't forget to restart X after doing changes.

If you make an error you'll still be able to use dpkg-reconfigure to go back. You can also make a copy of your xorg.conf file. Just in case...
