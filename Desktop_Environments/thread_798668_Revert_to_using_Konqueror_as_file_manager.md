### Post by Rul on 2008-05-18
I think is enough if you select a folder, right click it, then go to properties. In the general tab you click the icon at the right of where it says 

type:     folder            (icon here)

And then set konqueror as the preferred application to open it.

### Post by dresnu on 2008-05-19
> Run kcontrol (e.g. with K-menu -> Run command) 
Open KDE components -> File Associations -> inode 
Choose directory. In the right part of the window, change the application preference order so that Konqueror would be the first app in the list. 
Do the same with system_directory.

Found here: [Kubuntu Guide]("http://kubuntuguide.org/Hardy")
