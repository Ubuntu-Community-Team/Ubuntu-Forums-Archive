### Post by odrop on 2005-03-30
You'll probably be fine, but I'd consider upgrading the memory as soon as possible.  It'll help immensely with if you have several apps open, or considering the programs size, plus just improve performance in general. (It will also help with graphics, since you didnt mention if you were getting a video card also, and if you're running it on integrated graphics off the mobo, that'll eat away even more memory).

One nice thing about KDE is they keep improving performance each upgrade, so it keeps getting faster and faster.

What I'm running with now is just a P3 1.8ghz and 512 megs of ram. And it all runs pretty snappy and great.
