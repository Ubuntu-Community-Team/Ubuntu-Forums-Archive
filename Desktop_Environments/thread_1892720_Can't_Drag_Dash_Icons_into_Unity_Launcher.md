### Post by BC59 on 2011-12-08
No you cannot do that. The way to add something to the launcher is first open the app and while it's open and has icon on the launcher, right click-->keep in launcher.

For the applications you cannot do that I have a trick, I don't know if it's official. Create a hidden folder on ~/home with the name e.g. /.links. Then drag and drop inside the application icon you like to add to the launcher. After that, drag and drop again the icon to the launcher. It stays there and to the .links folder. Don't delete it from this folder because you lose it from the launcher.
