### Post by mat1t on 2006-06-01
It's a feature to draw attention to the dialog box. It's also designed so that nothing else will cover it, and that it stands out so you can type in your password and get on with it.

The screen doesn't update in the background anyway, so not having it dimmed out could cause people to think that their applications had stopped responding.

I think we're stuck with it like this as far as I know
