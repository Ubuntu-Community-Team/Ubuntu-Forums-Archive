### Post by louis_nichols on 2006-05-04
Depending on the exact model, there is at least a solution.

I have a quick cam chat working great, with a driver called [spca5xx]("http://mxhaard.free.fr/spca5xx.html"). Although the list there might actually be incomplete. There is an spca module in ubuntu but doesn't work correctly sometimes. it can cause hardware freezes. You can try it. Just start, say, xsane or gnome-meeting (now called [ekiga]("http://www.gnomemeeting.org/")). If it doesn't freeze, lucky you. If it does, follow [this thread]("http://ubuntuforums.org/showthread.php?t=75284")

NOTE: If it does freeze, the only way out will be a hardware reboot, just so you know in advance.
