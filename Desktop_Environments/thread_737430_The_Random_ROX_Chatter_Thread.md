### Post by Mark76 on 2008-03-31
Here's a list of must have ROX apps (ymmv)

1. Filer (obviously). Not only does it manage your files, but you can also use it to set your desktop background and up to 4 panels.

2. Session: Manages your ROX desktop session. Right click on the icon to find the Autostart option. Drag, drop and symlink any apps you want to autostart in to the folder that pops up

3. OroboROX: The native ROX window manager. Has Unix/Windows/Mac/RISC OS focus and raise options). Alternatively, Openbox is also available through 0install.

4. Appearance: The ROX GTK themes and icons manager.

5. Fonts: The ROX font manager . As well as the usual smoothing, hinting and subpixel rendering options there's also a resolution control.

6. Screensaver: Pretty basic (no fancy animations, just a blank screen), but it does the job.

7. Wallpaper: Not only does it let you set the pinboard image, but you can also have it change at regular intervals (and on start up) . There's also an advanced mode that I've not been brave  enough to try yet.

8. Keyboard: Lets you choose your keyboard layout and other options

9: Mouse: Mouse settings

10. App Factory: Lets you make ROX wrappers for non ROX applications.

11. Gamma: On screen gamma correction

12. Edit: A simple, but surprisingly feature packed text editor

13. Archive: Create and unpack compressed archive files

14. Execute: Run applications .

15. Picky: A simple image viewer

16. Weather: Let's you see if it's still raining outside :D

17. and I nearly forgot... Rox Term: A lightweight terminal emulator.

That's all for now.
