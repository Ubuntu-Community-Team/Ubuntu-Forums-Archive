### Post by jgordon510 on 2007-07-16
For audio preview, in ubuntu, the two packages are mpg321 and vorbis-tools and to install them you would type:
```

sudo apt-get install mpg321 vorbis-tools
```

in a terminal.

For PClinuxOS, I don't know, but look for similar packages in your package manager.
