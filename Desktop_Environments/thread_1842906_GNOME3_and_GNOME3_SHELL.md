### Post by 3Miro on 2011-09-13
Wait a second here. The video talks about Gnome-shell and Unity, not Gnome 3 and Gnome-shell.

Gnome 3 is collection of programs and libraries designed to create a desktop environment. Gnome is a project independent from Ubuntu.

Gnome-shell is one of the programs in Gnome 3. Gnome-shell is the default Windows Manager. What makes Gnome-shell different form other Windows Managers is that Gnome-shell creates its own interface.

Compiz is the classic Windows Manager from Ubuntu 8.10 and newer. Canonical decided to add some functionality to Compiz and make it into its own interface called Unity. In 11.04, you have Compiz + Unity interface running on top of Gnome 2. 

In 11.10, Ubuntu will come with Gnome 3, but they will replace Gnome-shell with Compiz+Unity as the default Windows Manager and interface. You will be able to install Gnome-shell from the Software Center and logout/login between Gnome-shell and Unity.

However, as of couple of weeks ago, this wasn't working. Gnome-shell wasn't working at all. Things are still in a testing stage, so if you want to use 11.10 with Gnome-shell, I suggest you wait some time.

I could install LXDE, XFCE & E17 too but I think 5 choices is enough for now.:)

Once Gnome Shell's extensions library grows I think it will be a nice choice for those that just can't leave things alone :cool:.
