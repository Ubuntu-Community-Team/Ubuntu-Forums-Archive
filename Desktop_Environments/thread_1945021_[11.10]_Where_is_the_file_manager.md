### Post by littlebigman on 2012-03-22
Found it: Move the mouse to the Desktop bit in the upper left-hand corner, and in the menu that is displayed, choose Goto > Workspace (My own translation: using the localized French edition).

Is there a way to add a link in the bar on the left (Unity) so users can simply click on it instead of going through the menu?

### Post by drakemata on 2012-03-22
Once you have Nautilus open, you should see the icon for it in the bar on the left.  Right click it and choose "Keep in Launcher"
