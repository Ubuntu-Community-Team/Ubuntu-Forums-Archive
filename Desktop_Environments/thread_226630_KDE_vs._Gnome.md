### Post by RenZO on 2006-07-31
Whole graphic interface is different !
Eyecandy depends on theme you are using... kde icons are available for gnome...
Usual applications change, Konqueror is the file manager, Kaffeine the media player, k3b the burning app.
But the real difference is the philosophy : Gnome is one soft for one task, Kde is one soft for all tasks. 
That's why konqueror is too a web browser, image viewer etc...
My point of vue is kde have confused menus for someone who don't want specific needs.
