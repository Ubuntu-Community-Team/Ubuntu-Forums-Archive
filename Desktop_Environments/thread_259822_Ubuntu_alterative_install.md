The only difference is the installer.  It installs without booting up the liveCD.  The 'desktop install' can have issues installing with little RAM as it boots the liveCD and *then* does the install.  The end installs are essentially the same.

### Post by lagagnon on 2006-09-17
Quite simply you will not be able to run ANY Gnome or KDE based Linux distro effectively with only 128MB of RAM. It will work - but as you say, like a dog.

If you/he didn't like xfce then I'd say it might be pretty hard to satisfy you. Other even more minimalist window managers than xfce (such as fluxbox, fvwm, windowmaker, IceWM, etc) are wonderful but  require a knowledge of editing text files to get the desktop as you like it.

My recommendation to you is to try VectorLinux v5.1 Standard and see if you like it. It runs really well on older machines with low  RAM. You get a choice between the xfce, fluxbox or IceWM window managers.
