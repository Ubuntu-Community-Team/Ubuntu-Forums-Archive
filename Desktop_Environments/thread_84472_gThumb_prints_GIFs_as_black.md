### Post by justmehere on 2005-11-01
I seem to have fixed it. I changed the ICM color profile to No ICM color correction and the color Mode from color - Photos and Text to just Color and now applications like gthumb and gphoto print in colour. I probably need to fiddle with the settings a bit more to get the best results but this is the first time I've managed to produce colour output from Breezy.
