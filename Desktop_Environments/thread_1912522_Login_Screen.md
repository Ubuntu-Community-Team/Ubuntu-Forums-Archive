### Post by grahammechanical on 2012-01-20
What version of Ubuntu are you using? It does make a difference. From 11.10 onwards login is done by LightDM. But before that it was/is done by GDM.

If I assume it is 11.10, then there are two ways.

1) Wait until the Month of May and upgrade to 12.04. In this release a users static background image becomes the background image to the LightDM log in screen.

2) Install Simple LightDM Manager. This will let you select a background image and also a logo image. This little utility is useful if you use the background slide show option in both 11.10 and 12.04.

[http://www.omgubuntu.co.uk/2011/10/simple-lightdm-manager-lets-easily-tweak-ubuntu-11-10-login-screen/]("http://www.omgubuntu.co.uk/2011/10/simple-lightdm-manager-lets-easily-tweak-ubuntu-11-10-login-screen/")

Regards.
