### Post by Jucato on 2007-05-30
1. Dolphin is a window manager. It's a file manager.
2. When KDE 4 is releases, it will probably be possible KDE 4 apps, like Dolphin, on GNOME, like running KDE 3 apps on GNOME.
3. Although you'll be able to run Dolphin on GNOME, you won't be able to "integrate" into GNOME. It will still be a KDE app, with its own KDE way of doing things.
