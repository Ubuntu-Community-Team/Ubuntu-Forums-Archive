### Post by wwastro on 2008-07-04

I have been using Kile for LaTeX work in this desktop, and it generally works fine, and can produce ps and pdf files as expected. However any attempt to use KDVI for previewing the dvi file produces nothing but the error message:

[COLOR="Red"][ViewDVI] finished with exit status 127[/COLOR]

All Kile settings appear to be as expected, and the system tells me that kdvi is properly installed, but I am not convinced!

Any help in resolving this irritating little problem would be much appreciated.
