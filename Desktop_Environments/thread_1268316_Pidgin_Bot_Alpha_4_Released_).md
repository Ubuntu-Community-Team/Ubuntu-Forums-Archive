### Post by cnr437 on 2010-04-03
Firstly, you need to sign in one account at pidgin then open purplebot,
it will list persons in box, you can choose there and when you click "Add" button, this person will be added to list that bot talks with persons in, they you only need to click "Start" button.

In AI Editor, there is "!random" key which replies ramdom answers to person if there is no key in database, also question keys and answers can be several alternatives.

Example:
-- how are you?,how r you?,how r u?
.|
..-- fine thanks.
..-- fine you?
..-- fine
..-- ))

You can separate questions by comma, and you can list down answers under keys.

* There is some problem at last version for "Gtalk", i will be fix as soon as possible.
