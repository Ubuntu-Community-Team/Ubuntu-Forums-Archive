### Post by buzzingrobot on 2014-06-30
Font color in the elements displayed on screen are set by the theme.  While it's possible to edit those themes, the simpler path is just to try some different themes.  If the font color does not change, something else is going on.

Themes should be compatible with the version of Ubuntu (and GTK) that you're using.  If not, they might work well, or they might not.
