### Post by 23meg on 2005-11-02
What's meant by this usually is that to accomplish most system tasks in Linux you don't need to reboot the computer. Restarting X is enough at most, which takes a few seconds. Plus, on a stable installation performance does not degrade after a long time of usage due to better memory management.

### Post by Jonne on 2005-11-02
It just means that Linux will be able to run continuously without needing a reboot a lot longer than Windows. Windows needs a reboot at least once a week (security updates), if you're lucky enough that it hasn't crashed in the meantime. In Linux the only thing that needs a reboot is updating the kernel.

But to be honest, you can get decent uptimes in XP too, if you know how to administer it. (but windows update will force you to reboot it anyway, once in a while).
