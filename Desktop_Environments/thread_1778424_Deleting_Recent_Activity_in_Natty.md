### Post by mc4man on 2011-06-09
Thru this ppa you can get some control with the app 'Activity Log Manager'
[https://launchpad.net/~zeitgeist/+archive/ppa?field.series_filter=natty](https://launchpad.net/~zeitgeist/+archive/ppa?field.series_filter=natty)

```
sudo add-apt-repository ppa:zeitgeist/ppa 
sudo apt-get update
```

Then just update/install as seen in synaptic and open activity-log-manager
