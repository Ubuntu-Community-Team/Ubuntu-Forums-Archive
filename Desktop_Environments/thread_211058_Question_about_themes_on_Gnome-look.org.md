### Post by Jucato on 2006-07-07
Just to "summarize"
GTK1 - Older version of the GIMP Tool Kit, which is what GNOME uses to draw it's controls (buttons, tabs, bars, etc). Some apps like XMMS and Audacity still use this, AFAIK.
GTK2 - Recent version of the GIMP Tool Kit
*Basically GTK1/2 Themes modify how your controls look.
(BTW, "controls" are what GNOME calls its "widgets")
*GTK themes are installed through System > Preferences > Theme > Theme details > Controls tab

GDM - GNOME Display Manager, but more commonly understood as a Login Screen Manager. GDM Themes change the appearance/layout of your Login Screen. Can be installed through System > Administration > Login Window

Metacity - GNOME's default Window Manager, which controls how windows, window borders and window controls (close, maximize, minimize, etc) are displayed. Metacity Themes changes these. Can be installed through System > Preferences > Theme > Theme Details > Window Border tab.

Hope that helps.
