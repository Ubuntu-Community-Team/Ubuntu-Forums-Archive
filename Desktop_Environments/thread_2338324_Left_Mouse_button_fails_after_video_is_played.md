When I bring up a video, the left button appears to no longer be associated with the current location of the mouse arrow.  I click in a new location, and it behaves as if the mouse arrow was where the video started, and you can't move the focus for the mouse buttons, even though the pointer is in a new location somewhere else on the screen.  Even if you close the video, things don't start working correctly again, and mouse motions like selection are done in the wrong window and clicking on a window will not change the focus.  You have to do ALT-TAB to get to a new window.

3.13.0-93-generic #140-Ubuntu SMP

In time the mouse seems to work again, but if you play a video again, it loses the mouse location/focus again.

I have run top while this is going on, and the CPU will be 95-97% idle while this is going on.

And if you start a video again, things go to pot again.
