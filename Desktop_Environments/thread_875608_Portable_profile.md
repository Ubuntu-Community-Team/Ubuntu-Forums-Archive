### Post by AlexTALL on 2008-07-31
This is sort of an abstract idea in my head, but I am going to try to be as specific as I can. I am able to run Hardy both at home and work (most of the time). On my way home tonight I was wondering if there was a way to transport a user's profile (easily) on a USB drive.

Minimally, what I am looking for is the ability to seemlessly transport a user profile from one machine to another. In my limited Linux experience, I would think that it should be relatively simple for Ubuntu to automatically recognize my USB drive (or a folder therein) and mount it at /home/*user*.

Ultimately, I think it would be nice if gdm could recognize the drive and either begin the login process (insert username and prompt for password) or even authenticate completely (though this would greatly increase the need for physical security of the USB drive).
