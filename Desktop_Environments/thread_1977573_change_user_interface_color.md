### Post by 3Miro on 2012-05-10
Install a different GTK theme. Look for GTK3 themes from gnome-look.org. The windows decorations are controlled via the Unity or Matacity themes. You also have the icon themes.

You might want to install Ubuntu Tweak to easily switch themes.
