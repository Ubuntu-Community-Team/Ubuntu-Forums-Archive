### Post by Rory on 2005-10-10
I've loaded 64bit and 32bit systems (Ubuntu, Fedora, SuSE) on my AMD XP Athlon 3000+ box.  

For general computer use (I'm not a developer who compiles, etc.), I can't see a difference in speed.  I'm sure there is, but I just  don't see it.

But, I do see that it's difficult to install things like win32codecs, flash, java, etc. in 64bit.  

So, once in a while I jump to 64, see if it's improved, find that it hasn't and jump back to 32bit, which works perfectly fine for me.  

Hope that helps.
