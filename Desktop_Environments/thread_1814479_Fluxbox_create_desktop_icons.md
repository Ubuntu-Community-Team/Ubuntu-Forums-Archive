### Post by kerry_s on 2011-07-31
so, it looks like for pcmanfm you just add "pcmanfm -desktop" to the startup & it will manage the desktop.
i use lubuntu which uses pcmanfm.

i used the --help command cause the man page was useless.
with pcmanfm you just copy the apps from /usr/share/applications to the ~/Desktop, it's simple. for custom launchers, you would use lxshortcut to create the launcher.
