### Post by asimon on 2006-06-28
You could make a menu entry yourself with the menu editor.

From the panel menu start Applications->Accessories->Allacarte Menu Editor

In the left choose where in the menu you want your gens entry to appear, for example Games, then in the File menu choose "New Entry" and enter a name and the command to start it. You can also give it a icon and a comment. Afterwards it should apper in the menu.

BTW, if you installed it from a debian package I would see it as bug if it doesn't appear in the menu automatically.
