### Post by 23meg on 2006-11-06
[http://en.wikipedia.org/wiki/XGL](http://en.wikipedia.org/wiki/XGL)
[http://en.wikipedia.org/wiki/Compiz](http://en.wikipedia.org/wiki/Compiz)
[http://en.wikipedia.org/wiki/Beryl_%28window_manager%29](http://en.wikipedia.org/wiki/Beryl_%28window_manager%29)

XGL is an independent X server meant to draw everything on your screen via OpenGL instructions. It will most likely be obsoleted by AIGLX technology which is integrated into X.org 7.1 (which you're running right now if you're using Edgy).

Compiz is a compositing and window manager that takes advantage of the OpenGL paradigm. 

Beryl is a fork of Compiz.
