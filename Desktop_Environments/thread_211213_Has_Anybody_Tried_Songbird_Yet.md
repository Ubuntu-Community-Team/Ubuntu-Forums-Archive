### Post by 3rdalbum on 2006-07-08
I've tried it in Windows, and for a 0.1 release it's very good. Looks promising - actually, the interface reminds me of Rhythmbox!

EDIT: In my original post, I said that I hadn't tried the experimental Linux version and that the Windows version doesn't run on WINE. Well, I just got it running on WINE, but you'll need the version from the WINE repositories, and you'll also need to use a Windows DLL file. It's a little slow, it keeps an empty window open, there's a Metacity border around the main window, but otherwise it works fine! I'm playing my WMAs in it right now :-)
