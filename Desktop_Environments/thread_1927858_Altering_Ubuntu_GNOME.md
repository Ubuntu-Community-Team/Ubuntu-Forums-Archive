### Post by Copper Bezel on 2012-02-19
For the window controls, install Gnome Tweak Tool from the Software Center (it'll come up in the Dash search as Advanced Settings.) Navigate to Windows, and you'll see controls for the other buttons there.

For other tweaks, try [extensions.gnome.org]("https://extensions.gnome.org/") (it'll only work in Firefox for now.) It can be hard to find things, but there's a lot of fun and useful stuff in there. (Why they can't implement a simple search is beyond me.)
