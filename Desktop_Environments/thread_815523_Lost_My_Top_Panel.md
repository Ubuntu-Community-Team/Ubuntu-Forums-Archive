### Post by mcduck on 2008-06-02
The icon in the Menu BAr applet (the one with Applications/PLaces/System) depends on the icon theme youa re using. So, to change the icon you'll either need to use some other theme, or find the picture from the theme's files and change it manually to something else.

If you use the Main Menu applet instead (which combines all 3 enus into one button, more like classic Windows menu) it's possible to select the icon used through gconf-editor.
