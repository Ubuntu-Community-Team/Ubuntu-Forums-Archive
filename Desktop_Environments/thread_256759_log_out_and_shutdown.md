### Post by ciscosurfer on 2006-09-13
Check out these new programs (I haven't had a chance to check them out myself, but I believe they will solve your issue from what I've read):

Pessulus >> [http://www.gnome.org/~vuntz/pessulus]("http://www.gnome.org/%7Evuntz/pessulus")
or
Sabayon >> [http://www.gnome.org/projects/sabayon](http://www.gnome.org/projects/sabayon)
Apparently, a later versions are actually in the Ubuntu repos for both Pessulus and Sabayon (or course, if you want the bleeding edge, you'll want to build from source or use the cvs entries), so you can also try installing via your favorite package manager...i.e., sudo aptitude install sabayon (you may need to enable your universe/multiverse repos to get this package)

Let me know how these work out for you as I'm quite interested!
