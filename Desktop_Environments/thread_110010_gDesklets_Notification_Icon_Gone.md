### Post by thespazzz on 2005-12-29
Ok I was fiddleing around with my GDesklets program.  If you right click the tray in the notification area you can get a configure screen to mess with some options.  One of the options is to HIDE the notification tray Icon.  However there is no way to get to this screen from the main program that I can see.  Now since i went and messed something up and I have to get back to the configuration screen to fix it how do I get this icon back?

---

### Post by thespazzz on 2005-12-29
Figured it out.

For those who have this same problum in order to restore the notification icon, open up a terminal window and type

gdesklets configure

This will bring the configure screen back up and you can uncheck the HIDE ICON button.
