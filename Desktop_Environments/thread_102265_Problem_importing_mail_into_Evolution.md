### Post by chefmac on 2005-12-11
Hello all,

I recently installed Ubuntu and I am trying to transfer my mailboxes from Outlook from another computer into Evolution.  As suggested on these forums, I installed Thunderbird on the Outlook machine and it auto-imported all of my mail and contacts.  I have transferred all of the mbox files from the Thunderbird directories (the ones with no extensions, right?) onto my Ubuntu desktop.

I click Import from the File menu in Evolution and then select Import Single File.  Here is the problem... there is no "Automatic" option in the drop-down box and when I browse to the mbox file the drop-down box does not let me select "mbox".  As a result the Forward button is greyed out and I can't proceed.

I have also tried copying over the original Inbox file and restart Evolution; however, none of the emails show up.  I am at a loss.  Any help is appreciated.

Cheers.

### Post by chefmac on 2005-12-14
In case anyone comes across the same problem, I figured out what was happening.  I did not have my USB pen or floppy disks on hand when transferring the mbox files from my Windows machine to my Linux laptop so I mailed the files to myself.  For some reason the mbox files were altered when emailed as attachments and neither Evolution nor Thunderbird would recognise them.  A couple of days later I transferred them using a USB pen and they imported fine.

Crazy.
