### Post by Ramses de Norre on 2006-08-26
In firefox you need to enable auto scrolling and smooth scrolling in the advanced tab of the edit > properties dialog, you can then use that scroll option and you can still use the paste option in fields were you need to type things.
In OOo you can somewhere in the preferences choose between the paste or the scroll option.
I don't know about other apps..
