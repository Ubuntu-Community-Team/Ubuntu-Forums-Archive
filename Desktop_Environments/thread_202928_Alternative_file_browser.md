### Post by Winblowz on 2006-06-24
I'm using KDE also, and I find KFTPGrabber to be a great file manager app. Of course, the original purpose of it was for FTP, but you can also transfer and manager  files locally. You can make it the default location for opening up files by right-clicking a file, then go to "Open With...", find the app you want to use in the menu, and then check the box which says "remember application association for this type of file."

### Post by Jucato on 2006-06-24
You can make Konqueror "lightweight" by moving/removing some toolbars you don't need. It might take some time, but the rewards of having a file manager fully customized to your needs and preferences makes it worth it.

Still, if you want to use an even lighter file manager, you can try installing Thunar, although it might look a little bit out of place. Just take note that by changing file managers in KDE, you will be giving up some integrated functionalities that Konqueror has. 

I'm not really sure how to make another file manager the default one.
