### Post by suRoot on 2005-11-23
So far as I know, Evolution is the only client that provides "native" exchange support.  You can use pretty much any other mail client you like if you turn on IMAP or POP on the Exchange server (I recommend IMAP & it may already be enabled on your server - it is by default).
