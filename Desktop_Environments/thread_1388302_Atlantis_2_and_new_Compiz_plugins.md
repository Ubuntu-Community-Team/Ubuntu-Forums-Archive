### Post by lidex on 2010-01-23
Don't know about Atlantis but there are compiz plugins available in the ubuntu repositories. Do you have the CompizConfig Settings Manager installed? That's what you want to manage your plugins. Opening Synaptic ("Applications>System>Administration>Synaptic") and searching "compiz" shows 3 plugin packages:compiz-plugins, compiz-fusion-plugins-main, and compiz-fusion-plugins-extra. To quickly install all four simply run this command in a terminal ("Applications>Accessories>Terminal"):
```
sudo apt-get install compizconfig-settings-manager compiz-plugins compiz-fusion-plugins-main  compiz-fusion-plugins-extra
```
