### Post by Woei on 2006-10-16
> **cd-r80 said:**
> I Can't get rid of windows because of notepad++. Is there any very clear simple editor using color higlights & lang support. Using Xubuntu. No G 0r K libs! Only one window, no fat toolbars... Mousepad is the nearest now. Im not gonna use keybindings, I have forgot the vi...

Well, Emacs doesn't use the GTK or QT libraries, only plain old motif or even Xaw3D. Emacs isn't really what you asked for though, as it's an extremely malleable editor that, together with the right lisp extension libraries can be transformed in a coffee-brewing kitchen-sink with extreme productivity enhancements. The downside is that's incredibly hard to master and writing your own extensions can be quite bothersome if you can't wrap your head around the functional-programming paradigm of lisp.

BTW, in case you didn't know yet, XFCE is based on GTK2, so any GTK2 based editor will do. gVim for instance is VIM with a menubar and a toolbar, written in GTK2, making it extremely easy to use the basic features of Vim. And VIM has syntax-highlighting modes for lots and lots of programming languages and configuration files. Anjuta is an entire IDE, so a bit heavy for your purposes I guess.

I'm using a mixture of jEdit, NetBeans and Emacs myself, but the first two are more geared towards Java programmers, and the latter takes some getting used to.
