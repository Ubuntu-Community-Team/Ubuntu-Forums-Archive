### Post by MARP1961 on 2011-08-17
This seems to be a common Gnome 2 problem and I've not seen a permanent fix for it. ```
killall gnome-panel
``` will sort it out but it will always rear it's head again.

In the next version of Ubuntu in October, the Unity 'shell' will run on top of Gnome 3. Perhaps it will not have this problem. Although you will not have a 'Classic' option, you will be able to install Gnome 3's own 'Gnome Shell' which will give you a login choice. To me Gnome Shell looks so much better than Unity and it's search facility is excellent.
