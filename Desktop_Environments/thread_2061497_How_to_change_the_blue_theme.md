### Post by kiwihandsome on 2013-06-09
I think what you want is light-themes. Firstly install it by typing into terminal:
sudo apt-get install light-themes
then to to appearance settings > icons and select either "Ubuntu-Mono-Dark" or "Ubuntu-Mono-Light". By doing so ur icons will have orange color
