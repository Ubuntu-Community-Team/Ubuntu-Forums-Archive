### Post by ciscosurfer on 2006-09-02
Go to [COLOR=DarkRed]System > Preferences > Mouse | Motion[/COLOR] for general use.

For Firefox/Swiftfox, you can install an extension called [SmoothWheel]("https://addons.mozilla.org/firefox/357/").
