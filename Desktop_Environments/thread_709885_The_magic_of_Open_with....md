### Post by prensing on 2008-02-27
The easiest way is:
Find a file of the type you want to handle in a Nautilus window. Right click on it and open up properties. Go to the Open With tab. Click on the radio button next to the app you want to use. This should set the default app for this file type. 

I am not sure if this propagates to all of the Gnome system, but it works for Nautilus.
