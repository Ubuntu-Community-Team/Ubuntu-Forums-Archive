### Post by ireneshusband on 2009-11-11
When I was using KDE4.2 in Jaunty my four virtual desktops were laid out in a single row. After upgrading to Karmic and KDE4.3 (possibly because of something I did), the virtual desktops are laid out in two rows of two. Is it possible to change back to having one row of four virtual desktops?

Thanks in advance.hh

---

### Post by benerivo on 2009-11-11
I think it's in System Settings > Desktop Effects > All Effects, then go to Desktop Grid configuration button and you can change the layout.

---

### Post by ireneshusband on 2009-11-17
Thanks for the suggestion. It turns out that Desktop Grid is not the same thing, but actually you can change the number of rows by right-clicking on the pager in the KDE panel and selecting Pager Settings. It's actually quite easy to find and I'm not sure how I managed to miss it.
