### Post by DavidW2 on 2006-09-16
In Gnome, just right-click on the Firefox icon in the menu bar and select Properties.  Then click on the icon in the basic tab.  It will open up a browser to change your icon.  If you want to change the icon in the Applications menu go to Applications > Accessories > Alacarte Menu Editor and open the Internet group.  Right-click on Firefox and select Properties as you did for the menu bar.  Icons for Gnome are available at [www.gnome-look.org](www.gnome-look.org).
