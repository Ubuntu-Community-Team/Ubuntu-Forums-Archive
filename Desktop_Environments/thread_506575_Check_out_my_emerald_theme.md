### Post by crimesaucer on 2007-07-23
> **VictorR said:**
> I think the common problem of all the themes in this thread is that some text, especially in captions, is just un-readable. I tried to play with transparency myself and found that in most cases it makes the affected window useless. So my opinion is the foreground window should be fully opaque.

My text is "unreadable" because I have it set in the Emerald Theme Manager to be "lightly" readable when the windows are behind the active, open window....some themes turn the rest of the unfocused window decorations a gray color, or a different color, while keeping the titlebar text black or white.


I have my windows stay the exact same, except for the titlebar text color, which changes to about 10% transparency for the non-open window's titlebar text. Then for the "open" window it has the text at the normal 100% transparency (readable). That way I can lightly read the titlebar text of any "unfocused" window, without it distracting me away from the window that is actually open. I find it less cluttered that way.


I always know what apps I have open, so I don't need all of the text, but if I sort of forget, then I can still read the titlebar text if I have to.


I also have a unique text that matches my desktop...see, the thing with Emerald windows is that you can download any theme and then "Edit" the theme and change the titlebar font, color, outline, and transparency...you can even add a background glow behind the titlebar font.
