### Post by pete224 on 2008-03-28
I'm sorry to awake this from the dead but this is a top search from Google. I found this forum when i had this problem. Now that I've fixed it i feel like sharing it :) . My solution was wine doors. Here is how to install it (don't uninstall wine):

1) Go to [http://www.getdeb.net/release.php?id=1997](http://www.getdeb.net/release.php?id=1997) and click on Download: wine-doors  (269.2 KB).

2) Install the package

3) And your mic should be working. (I don't exactly remember how i did it. If it doesn't work now, do the next step)

4) Install cs:s through wine doors. This step is different depending on weather you downloaded it from steam or bought the CD. IF YOU DOWNLOADED IT: scroll down through the list of applications and find steam, install it, and install cs:s through steam normally. IF YOU BOUGHT THE CD: go on on file/Install CD (when you have got the CD inside silly :P )

You might not need step 4 i forgot :P . Oh well either way i hope I've helped.
