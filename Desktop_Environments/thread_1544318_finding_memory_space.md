### Post by ajgreeny on 2010-08-02
What OS are you running at the moment?  If it is windows I have no idea how you do it any more as I got rid of windows a while ago, and it's amazing how quickly you forget how you did things.

To find out you can run a  Ubuntu live CD, double click on the partitions shown in  Places to mount them, and then using the terminal type ```
df -h
``` and a readout will appear with the total and used space available on all the disks present.  Much easier than palimpsest.
