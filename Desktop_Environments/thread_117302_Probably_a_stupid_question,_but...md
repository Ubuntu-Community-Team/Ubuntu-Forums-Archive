### Post by poiuytr on 2006-01-14
Be careful, though.  In a previous installation of Ubuntu, I just wanted to remove alsa and reinstall it.  Synaptic told me it was going to remove all kinds of packages, including the KDE ones.  I had installed but hadn't yet started using the Kubuntu packages, so I thought, what the heck, it can't be as bad as I think this means, right?  Soon thereafter, I had  completely borked Ubuntu installation I could no longer boot into.

I am loving Ubuntu, but I would caution to be careful when REMOVING packages.
