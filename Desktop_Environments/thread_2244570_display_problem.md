### Post by grahammechanical on 2014-09-17
In Ubuntu we have System Settings>Software and Updates>Additional Drivers tab. We must be connected to the internet and allow time for the utility to find drivers. You say the browser display does not fit the screen. Does Ubuntu fit the screen? Is the image too large or too small?

If you are already using the proprietary video driver then there will be a settings management utility for that driver and that utility will have a detect displays option. If we are using the open source video driver then we go to System Settings>Screen Display and click detect displays. Just to make sure the system knows what monitor we have. It usually does know. The system reads the Extended Display Identification Data (EDID) in the monitor when it loads and that is how the system knows the correct display settings.
