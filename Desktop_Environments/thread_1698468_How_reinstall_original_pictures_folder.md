### Post by Copper Bezel on 2011-03-02
The simple, GUI way to do this is by downloading Ubuntu Tweak, which will give you a button to click to restore it.

You can also do it with a quick text configuration file edit as described [here]("http://silverwav.wordpress.com/2010/03/21/note-ubuntu-default-folders/"). To do this, you'll have to create the folder manually, add it to your Default Folders list as outlined in the link, and finally drag the folder into your Shortcuts pane in Nautilus to get the full effect.
