### Post by gerkin on 2007-04-09
Don't worry about the errors to much, it could be as simple as a sever temporarily down at the other end.  Often this will be self-correcting at the other end when you run "apt-get update" the next  day

Occasionally you have a apt-source added to your "/etc/apt/sources.list" that is no longer available etc.

Try posting your "/etc/apt/sources.list" file and I'll have a look at it if you want, or

just run "apt-get update" again, (you can ignore the errors if they are still there today) and then run "apt-get upgrade" anyway

cheers ...............gerkin
