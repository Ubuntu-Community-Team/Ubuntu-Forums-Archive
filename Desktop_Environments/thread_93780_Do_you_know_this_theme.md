### Post by SickTwist on 2005-11-22
That screenshot is [XFCE]("http://www.xfce.org/") and it is most likely running the XFCE window manager (I can't remember what it is called, perhaps xfwm).

You *could* use the XFCE window manager in GNOME if you really wanted to use that theme. Otherwise, look for themes for Metacity (GNOME's window manager) on [gnome-look]("http://www.gnome-look.org/").
