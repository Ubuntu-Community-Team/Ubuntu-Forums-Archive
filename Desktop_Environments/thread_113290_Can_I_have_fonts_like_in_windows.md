### Post by Nikusan on 2006-01-06
Hi cadumg,
I have the msttcorefonts package installed for compatibility with *.doc files that I have to deal with from various sources... But I found that once it was installed text started to look aweful in firefox.
So in firefox I went to Edit -> Preferences -> General -> Fonts & Colors and ticked the "Always use my fonts" check box. Now everything looks great. When I use gmail the menu text looks a lot like your IE screenshot. Hope this helps.
