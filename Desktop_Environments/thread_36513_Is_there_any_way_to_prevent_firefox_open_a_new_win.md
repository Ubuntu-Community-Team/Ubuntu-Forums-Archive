### Post by jkndrkn on 2005-05-23
Install the extension tabbrowserpreferences.

You can find it here:

[https://addons.mozilla.org/extensions/moreinfo.php?application=firefox&id=158](https://addons.mozilla.org/extensions/moreinfo.php?application=firefox&id=158)

After installing: Tools > Options > Tabbed Browsing > Tab Focus > Load windows diverted into tabs in the background
