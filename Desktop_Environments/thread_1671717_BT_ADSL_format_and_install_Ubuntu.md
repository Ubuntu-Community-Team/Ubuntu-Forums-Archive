### Post by bouncingwilf on 2011-01-23
If you install Ubuntu, just hook it up to the router (Homehub) via ethernet or wifi and it should work "out of the box" - you do not need ( or probably even want) BT's crap Yahoo software. If you want to configure the router for games etc. you can access the router @ 192.168.1.254, user : admin Password is initially the router S/N (on the router).  Any problems - get back to us.
