### Post by Dennis N on 2012-01-22
To install a GTK 2 theme:

The theme comes in an archive file. I extract the entire archive into:

~/.themes

To do that, right click on the archive and select open. Then Extract, and browse to the target directory. If necessary, use CTRL+H to show hidden directories (.themes is a hidden directory)

When you next open the Appearance dialog, the new theme should show up. If not, log out and back in.

Cursor (mouse pointer) themes are extracted into:

~/.icons

You need to use the customize button to change the cursor used on a theme.

---

### Post by Dennis N on 2012-01-22
I will add this second method for a gtk-2.x theme:

In the appearance dialog, there is an install button on the themes tab. You can use that, and browse to the themes package you downloaded.


Note: when you customize a theme and save the customization with "save as" button, it is then saved into:
~/.themes
