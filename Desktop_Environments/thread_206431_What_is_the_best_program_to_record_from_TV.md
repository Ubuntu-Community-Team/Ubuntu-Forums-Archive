That is a question I have asked many times but I never had a clear answer from someone. Anyway the only program I have found so far and does this job pretty well even if it has some annoying bugs is xdtv.
It supports recording in almost any format and it has many configuration options (Actually you need few of them. Do not panic :) ).
You can also schedule recordings.

I have installed it from marillat repositories on a kubuntu amd64 machine.

Good Luck!
