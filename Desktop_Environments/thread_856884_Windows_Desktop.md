### Post by MattBD on 2008-07-12
I'd recommend Icewm. It's light and fast, and has loads of themes, but be warned, it's not terribly pretty and you need to learn to configure it using text files. If you install the icewm and icewm-themes packages, you'll get it together with a load of extra themes such as SilverXP (which is used on the Asus Eee PC).
If that's scared you off, KDE is probably a better bet.
