### Post by Wobblybob on 2010-11-27
Right click the top panel, select [new panel] and it should put a panel at the bottom, if it puts it at the side right click it select properties and select bottom from the orientation option. You just need to right click it again and [add to panel] then add the windows list, workspace switcher and show desktop options.
