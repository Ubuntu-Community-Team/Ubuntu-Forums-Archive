### Post by stuporglue on 2006-01-02
Plugging in the ethernet cord should work fine, unless you've played with the settings. I'm pretty sure the default settins are for DHCP on both the airport and Ubuntu. Static addresses would be fine too, you just have to do a little more setup. 

Ubuntu (and Linux in general) does support wireless internet. For example, the original airport cards (not Airport Extreme) work better and have more features under Linux than under OSX. Some wireless cards don't have Linux drivers though. This is the case with the newer Airport cards. 

Airport, the base station, shold work with Linux regardless of the Airport version.
