### Post by MikeMLP on 2006-08-19
I have a 64 MB mobility Radeon 9600, and this also happens to me.  I think that it must be a bug with the latest (8.27.10) version of the ATI proprietary drivers.  I remember that with the previous version of the ATI driver (I was using XGL / Compiz with that version as well) The ATI control panel worked as expected, providing dual monitor support, and a very small collection of other tools.  I would either wait for the next version if you aren't missing vital features (like me) or downgrade to the previous version.  Of course, I don't know that this is a bug with ATI for sure, this is just speculation on my part... but I have a strong suspicion that I am right, perhaps someone on here could confirm my suspicion, or correct it if it is wrong.
