### Post by mssever on 2006-09-07
Getting Windows programs to run on Linux is often difficult. If you're having trouble with wine, you can try cedega or crossover office, but there's no guarantee that those will work, either. The reason is that Microsoft keeps the Windows API (how programs talk to Windows) secret to a large extent. So wine (or other smilar programs) has to do a lot of guesswork. And it often isn't able to guess correctly.
