### Post by SeijiSensei on 2013-01-29
The first time you run Thunderbird it launches a "wizard" to set up your primary email account.  It asks you things like your name and email address.  Is that what you mean?  If so, just fill in the form and let it try and do its magic.  If your using it with your own server and domain, it may not work, but if you're using a popular free email provider like GMail it should set up everything for you automatically.

If you have an existing Thunderbird installation that is giving your problems, here's a quick way to test.  After making sure Thunderbird is closed, open a terminal and type the command "mv .thunderbird .thunderbird.bad".  Now start Thunderbird again and walk through the wizard to set up your account.  There are ways to move things from the old installation to the new, but before heading down that road, we need some more details from you.
