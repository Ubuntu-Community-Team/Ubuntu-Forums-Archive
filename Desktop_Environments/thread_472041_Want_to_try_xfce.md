### Post by MadMarv on 2007-06-13
Actually it can be quite a bit more than that. It is highly configurable, and has its own compositing engine built in. So there is no real need to install Beryl or Compiz, etc. to get things like a truly transparent terminal winow, drop shadows, or other eye candy. I started using it because I do have an older machine, but have come to like it so well I plan on keeping it even when I build my dream machine.

By the way, if anyone wants to install just the xfce desktop without the additional programs in the Xubuntu meta-package, you should be able to install only the xfce4 metapackage from the repositories. Gnome apps run fine on xfce-

### Post by DonnieP on 2007-06-14
Pitster:  If you installed xubuntu-desktop then after a reboot your login screen should say xubuntu rather than ubuntu.  However, this doesn't really matter.  What determines which desktop environment you're in the session you choose on the login.  From either the ubuntu splash or the xubuntu splash you can select either GNOME (ubuntu) or xfce (xubuntu).  From the ubuntu splash you have to select options to choose a session.  You can tell which you're in once you've logged on by the way the desktop menus look, assuming you've not radically altered the defaults.  The default GNOME menu has Applications, Places and System at the top.
