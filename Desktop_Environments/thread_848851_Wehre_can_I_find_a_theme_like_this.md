### Post by balaknair on 2008-07-03
From what I can make out, it's more than just a system theme.
The desktop background is a variant of Vista Aurora, the dock at the bottom is Avant Window Manager, the widgets are part of the Screenlets pack- Meter, Clock, Clear Calendar(I think), ClearWeather.
The window decorations are probably part of the metacity theme, or an Emerald theme. 
Not sure about the icon theme,  I'd like to get that one too(p'bly won't use it, but it does look pretty nice)

You'll be able to find most of this on Gnome-look.org, and Avant Window Manager is available from the Ubuntu repositories(an older version with fewer plug-ins), or you can get the newer version by following instructions here
 [http://ubuntuforums.org/showthread.php?t=762363](http://ubuntuforums.org/showthread.php?t=762363)
