### Post by mysticrider92 on 2007-02-21
You probably need a proper driver for the graphics card. That is set to use the i810 driver which is a lot older than the i965. I saw on the forums where someone else had a problem getting wide screens to work with this card. According to one person, setting the driver to "i965" worked for them, but I would look for the package before trying this so you don't get any xorg errors (one of the most annoying things in Linux).

The good news for you is that Intel just released source code for the card drivers, so packages should be out soon if they aren't already.

### Post by pekvain on 2007-03-03
I updated Edgy to Feisty beta and installed xserver-xorg-video-i810 and 915resolution from repos. Then I modified /etc/default/915resolution and set MODE=30 and XRESO=1680 and YRESO=1050. I restarted 915resolution and X. It worked. Even 3D works fine.
