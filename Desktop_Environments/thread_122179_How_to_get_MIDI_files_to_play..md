### Post by FarEast on 2006-01-27
Unless you have a soundblaster, you need a software MIDI synthesizer
to play MIDI files.
Install 'timidity' and 'freepats'.  Then you will able to play one like this.
$ timidity foo.mid
Please read below for more information.
[http://ubuntuforums.org/showpost.php?p=634915&postcount=8](http://ubuntuforums.org/showpost.php?p=634915&postcount=8)
