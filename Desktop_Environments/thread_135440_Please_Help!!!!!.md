### Post by cstudent on 2006-02-23
You didn't say, but I'm assuming you do have a CD drive. You can use a program called Smart Bootmanager to boot your computer from a floppy and then install an OS from a CD. You can find SBM here:

[http://sourceforge.net/projects/btmgr](http://sourceforge.net/projects/btmgr)

I used it to install Gentoo on an old computer that couldn't boot from the CD. If you don't have a CD drive, you might look into installing one. I don't know if any floppy install disks are available for Ubuntu. I would highly doubt it.
