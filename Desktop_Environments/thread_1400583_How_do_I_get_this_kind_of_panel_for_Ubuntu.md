### Post by mcduck on 2010-02-07
Tint2 doesn't have any configuratio tool, so any configuration is done by editing it's config file (~/.config/tint2/tint2rc) by hand. You'll find some instructions frok Tin2's web site: [http://code.google.com/p/tint2/wiki/Configure]("http://code.google.com/p/tint2/wiki/Configure").

Also note that Tint2 is not a fll-blown panel, so you can't use any of the panel applets you'd run in Gnome-Panel with it. And it doesn't have any support for menus either. If all you want is a tasklist with only icons, there are some Gnome-Panel applets available for that purpose. DocBarX, for example.
