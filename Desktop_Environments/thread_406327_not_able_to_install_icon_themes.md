### Post by volksolwagen57 on 2007-04-11
firstly: go to sytem-->Preferences-->Theme

secondly: leave this window open and out of the way, don't minimize it as you will have to drag and drop your downloaded theme package into there

thirdly: go to where your download is, it doesn't matter if it's a .tar.gz file or a .bin file; you don't have to use terminal, simply drag your packaged file and drop it into the list of themes available

you can even customize your themes: for example, you can download an icon theme like nuovoXT and a panel theme like Brushed and drag them both into the themes list and furthermore (this is so cool) you can apply either the icon or the panel them, whichever, it doesn't matter, and you can click on customize to combine both of them so you can have a stylish, sexy, cutting edge, ubuntu use provoking desktop that will leave all of your windows using buddies in the dust ;-)
hope this helps, good luck!
