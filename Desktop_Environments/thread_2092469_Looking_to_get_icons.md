### Post by Toz on 2012-12-08
Hello and welcome to the forums.

You have a few choices in getting this icon theme into xubuntu.

[LIST=1]
[*]Copy the /usr/share/icons/Fedora directory from the fedora spin and paste it into xubuntu.

[*]Grab the files from the rpm file ([clicky]("ftp://fr2.rpmfind.net/linux/fedora/linux/releases/17/Everything/i386/os/Packages/f/fedora-icon-theme-1.0.0-12.fc17.noarch.rpm")) and extract them into xubuntu.

[*]The theme looks alot like the cheser theme ([clicky]("http://xfce-look.org/content/show.php/Cheser+Icons?content=154787")). You can install that theme instead.
[/LIST]

As to where to copy the files in xubuntu, you have 2 choices:
[LIST=1]
[*]**/usr/share/icons** - for full system use (requires root privileges)
[*]**~/.themes** - for user-specific use (it is a hidden directory and if it doesn't exist, you need to create it first)
[/LIST]
