### Post by undead-monkey on 2006-09-22
> **momo66 said:**
> I've been running Ubuntu on my macbook since I've had it.  It installs very easily.  I am using Parallels, so I run Ubuntu on top of OSX.  But you can also run dual boot.

In fact I have OSX, Windows XP, Ubuntu, and SUSE all running on the macbook.

Dont tell people Ubuntu installs very nicely on the Macbook if all you are using is Parallels. Ubuntu installs great with Parallels but trying to dual-boot or triple-boot is a whole different set of problems.

Anyone who is new to Linux who is thinking about doing a Dual-boot setup with thier Intel Mac, i suggest you find a friend who has done it them selves to help you install, because its not for the faint of heart.

I have had quite a few problems with my Ubuntu install, and finally gave up for the meantime and use Parallels in which i have XP, Ubuntu (Kubuntu,Xubuntu), Fedora Core 6, and SUSE all running fine.
