### Post by buzzingrobot on 2014-05-28
There are good Mate installation instructions at that link.  Current Mate release is 1.8 but there are a few nitpicky issues with it on Ubuntu.  The previous release, 1.6, installs and run fine.  When the Gnome project ended Gnome 2 work three years ago, that code base was picked up by a few people who wanted to keep the Gnome 2 interface alive.  It morphed into the Mate project.  Mate is technically a fork of the Gnome 2 code base, but in use it is almost entirely identical. In addition to bug fixes, GTK3 compatibilty is being added to allow contemporary applications to run smoothly.
