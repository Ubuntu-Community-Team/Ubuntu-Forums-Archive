### Post by lukem on 2005-02-06
From the menu click Applications.  Move the mouse down to highlight Multimedia.  Move the mouse over into the list of multimedia applications and RIGHT click anywhere.  In the window that pops up, move the mouse down to "Entire menu".  In the window that pops up select "Add new item to this menu".

Fill in the name, command and select an icon.  This should do it.

Good Luck

Luke
