How do I install all of the flavours of Ubuntu on one machine? Do I just have to install ubuntu then:
```
sudo aptitude install kubuntu-desktop xubuntu-desktop
```

---

### Post by ButteBlues on 2006-09-01
That command is correct.

And yes, the three environments use the same files for everything except for their apps and for the GUI. So the same /home /bin et al is used in all three environments.

After installing them, if you restart X by hitting ctrl+alt+backspace, you can choose an Xfce, KDE, or Gnome session to start into.
