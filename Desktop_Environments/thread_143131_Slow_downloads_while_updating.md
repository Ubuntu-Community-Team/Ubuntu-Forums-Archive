### Post by John.Michael.Kane on 2006-03-11
Synaptic Package Manager is downloading the files from the ubuntu server, and this is what governs your speed. you can always build a source list yourself using [http://www.ubuntulinux.nl/source-o-matic](http://www.ubuntulinux.nl/source-o-matic)  enter your 2-letter code of your country code where it is asked for, this may get you a server closer to you...
