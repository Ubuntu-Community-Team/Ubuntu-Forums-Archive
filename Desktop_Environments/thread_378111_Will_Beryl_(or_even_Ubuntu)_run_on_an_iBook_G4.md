### Post by tgalati4 on 2007-03-09
If the machine has some form of OS 10, you can go to Apple-->About this Mac-->More Info and it will give the hardware specifications.  The type of video card and video memory will determine how well Beryl will run.

You can also spend some money and put Tiger 10.4 on it.  It's a pretty solid operating system and it will recognize all of mac hardware.  You can install XDarwin and then run Gnome Deskop Manager and many Linux applications that have been ported to the PPC platform.  This allows all the goodness of Tiger (hardware that works properly) with the expansiveness of Open Source software.  Best of both worlds in my opinion.
