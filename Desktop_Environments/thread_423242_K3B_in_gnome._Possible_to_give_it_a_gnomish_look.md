### Post by jfinkels on 2007-04-25
The reason it looks KDE-ish is because it is written using the Qt toolkit (which is what KDE apps use). If you want it to look GNOME-ish, find yourself another program. 

You can try GnomeBaker, that's a GNOME application.

### Post by SeanHodges on 2007-04-25
If you use the "Clearlooks" theme in Gnome... try this howto, it will change the theme of your KDE apps to resemble your Gnome desktop:

[http://ubuntuforums.org/showthread.php?t=56630](http://ubuntuforums.org/showthread.php?t=56630)

Even if you don't use Clearlooks it might be an improvement.


Also, subscribe yourself to this blueprint, which requests the same thing:

[https://blueprints.launchpad.net/ubuntu/+spec/human-theme-for-qt-and-kde](https://blueprints.launchpad.net/ubuntu/+spec/human-theme-for-qt-and-kde)
