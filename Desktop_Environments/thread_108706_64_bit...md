### Post by WebLOCH on 2005-12-26
All you SHOULD need to do is install the 64bit kernel, 'linux-image-x' in synaptic/apt  however I tried this once and found that quite a few things broke.  Also if you rely on w32codecs or similar, there is no 64-bit compatible version yet, and may never be.  Some of the software you break you will need to compile yourself.

Overall, good luck, this is just my experience.
