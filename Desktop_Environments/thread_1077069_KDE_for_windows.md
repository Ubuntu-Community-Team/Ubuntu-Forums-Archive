### Post by jacksaff on 2009-02-22
The underlying libraries kde uses are now available under an open source license for windows which means a lot of kde software will be very easy to port to windows. The whole of kde will probably not get ported. The kde for windows guys actually seem to be doing better than even they expected at getting kde to run on windows though, so who knows, maybe a full kde session will be possible soon. It's not ready yet however and you'll need to really know your way around both kde and windows to be able to see where it's currently at.
