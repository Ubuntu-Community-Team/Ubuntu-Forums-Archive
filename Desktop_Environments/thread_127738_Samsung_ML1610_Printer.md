### Post by spandon on 2006-02-10
Have now found the answer, thanks to a post by Manis.
When you use the printer setup via System/Admin/Printer, the driver for this machine defaults to ML 1650, but when you try to use it - nothing happens.
If you change the driver from ML1650 to ML1510, everything works straight away, If only this info had been available easily!!!!!!
