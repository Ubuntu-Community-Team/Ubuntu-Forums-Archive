### Post by Wolki on 2005-11-25
[QUOTE=Biased turkey]I want to add the game Lgeneral in the Applications-> games submenu

The help says that I have to enter "applications:///" URI  in Nautilus Go->location menu ,but that doesn't work.[/QUOTE]

"applications:///" was used till gnome 2.8, then dropped. Unless you're running warty, it won't work that way. to edit the menu in Breezy, right-click it and choose "Edit Menus". It should open Smeg, a menu editor for gnome.

If you're on Hoary, there's no editor included by default. you should be able to install smeg/alacarte manually, look in the subforum here about how to do it.
