### Post by jasbur on 2007-10-21
One way is to install "Advanced Desktop Effects Settings" from the Applications > Add/Remove... function.  Once installed it will be under System > Preferences.

Click on the plugin you want, then on Actions. All of the bindings will be listed there. Even better, you can change them to anything you want.

### Post by burgundy on 2007-10-21
I was able to find the "Advanced Desktop Effects Settings" in the Add/Remove Programs menu, but only after enabling more than what Ubuntu considers supported software packages.  I think I had to enable community supported software to download/install what I needed.

I'll also post here that the command I was looking for was SHIFT + ALT + UP ARROW as the default keyboard command to execute the Scale Effect.
