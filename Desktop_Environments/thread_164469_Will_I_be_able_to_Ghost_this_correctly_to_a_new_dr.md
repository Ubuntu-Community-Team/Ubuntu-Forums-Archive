### Post by fdoving on 2006-04-23
Yes you will be able to.

I use the system rescue cd from:
[http://www.sysresccd.org/](http://www.sysresccd.org/)

They provide partimage:
[http://partimage.org](http://partimage.org)

With partimage it's easy to backup and restore partitions for example over the network.
