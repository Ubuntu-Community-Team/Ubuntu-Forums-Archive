### Post by stchman on 2007-04-17
> **hammadr89 said:**
> I was running Ubuntu, then I decided to install KDE... and I kind of like it a lot :S. So is it possible that when 7.04 is actually released, that I can change my make from Ubuntu to Kubuntu? Even remove GNOME altogether?I tried doing that right now, but it didn't work (tried removing ubuntu-desktop).

I would not call Kubuntu an upgrade from Ubuntu.  It is just a different version.  

You can install the Kubuntu GUI desktops by issuing the following commands:

sudo aptitude update

sudo aptitude install kubuntu-desktop

Follow any onscreen prompts.  This takes about 20 mins to do the upgrade.

Aptitude is a better method of installing desktops than apt-get.

Thanks be to the folks over at [Psychocats]("http://www.psychocats.net/ubuntu/kde") for this information.
