### Post by psychicdragon on 2005-12-01
Sorry, for throwing accronyms around like that. Here's an overview.

WM = WIndow Manager. 
This is the program that draws the window borders around your applications and allows you to move them around and stuff. There are a _lot_ of different WM's and they all have their own little quirks and features.

DE = Desktop Environment.
A Desktop Environment is basically a WIndow Manager, file manager, panel and a bunch of other programs that make up the whole "desktop." Gnome is a Desktop Environment, and you could say that WIndows is too.


If you're completely new, I would recommend you try out all, or as many of the different Desktop Environments as you can (primarily Gnome, KDE, E, Xfce). Fairly quickly you'll be able to discover what you like and dislike about each of them, and what features are most important to you. You may even find that one of them is the perfect fit for you.

### Post by Iandefor on 2005-12-03
Ok, so I've been playing around a little, and for anyone who's interested, I've found a pretty primitive way of making enlightenment start up with xfce4.

Make a new file, call it "buddha" and put the following in.

```

#!/bin/sh

xfdesktop &
xfce4-panel &
xfce4-iconbox &
enlightenment
``` Save the file and type in ```
chmod 0775 buddha
``` at the terminal. Copy/link/move your script to "/usr/bin/", make a new file in /usr/share/xsessions and call it exfce.desktop. Put the following in it:

```

[Desktop Entry]
Encoding=UTF-8
Name=Enlightened Xfce4
Comment=
Exec=buddha
Icon=
``` Save it, log out, and when you log back in, under sessions, look for an option entitled "Enlightened Xfce4"- Go for it!

Obviously, replace the commands in the script with whatever options you want. Just follow every command save the last with an ampersand. The only drawback to this is that you can't save sessions. I'm going to fiddle around some more and see if I can't get enlightenment to work with the session manager with xfce4.

NOTE: I borrowed some elements from poofyhairguy's "Enlightened Gnome" guide (Mostly, setting up a new session option in GDM). Thanks, poofyhairguy!

### Post by mhancoc7 on 2005-12-03
I am using Xubuntu (xfce4) with Enlightenment. It is AWESOME!

I used a pretty noobish way to accomplish it as well, but I think it is nice because I did not have to edit any configuration files.

Here is basically what I did.

1. Log in to your Xfce4 session

2. Create a directory (folder) in the Desktop directory (folder). Name it Autostart.

3. Open the new Autostart directory and create a blank file named start.sh.

4. Open the new start.sh in a text editor and add the following:

#!/bin/bash
enlightenment &

5. Now open a Terminal window and run the following commands. You will notice that the windows will look funny. Once done use File > Quit to close any open windows.

killall nautilus
killall xftaskbar4
killall xfdesktop
killall xfwm4

6. Now log out. Be sure to save your session on log out. Now when you log in you will have a nice Enlightened Xubuntu. 

Here is a link to my other post about this where you can see some screenshots.
[http://www.ubuntuforums.org/showthread.php?t=98067&highlight=enlightened+xubuntu]("http://www.ubuntuforums.org/showthread.php?t=98067&highlight=enlightened+xubuntu")

Note: Be careful when opening Nautilus. It will take over your desktop. There is a way to disable that feature, but the best way to keep that for happening for me was just to use Rox filer instead. It is much faster anyway.

God Bless, Jereme

### Post by mhancoc7 on 2005-12-06
wushumofo:
> have you guys been able to save your settings?
mine never saves my settings...
The method that I am using lets me save my session just fine.

Here is the basics to my method. It makes startup a little slower, but once setup it runs great. I have not run into any problems yet.

1. Log in to your Xfce4 session

2. Create a directory (folder) in the Desktop directory (folder). Name it Autostart.

3. Open the new Autostart directory and create a blank file named start.sh.

4. Open the new start.sh in a text editor and add the following:

#!/bin/bash
enlightenment &

5. Now open a Terminal window and run the following commands. You will notice that the windows will look funny. Once done use File > Quit to close any open windows.

killall nautilus
killall xftaskbar4
killall xfdesktop
killall xfwm4

6. Now log out. Be sure to save your session on log out. Now when you log in you will have a nice Enlightened Xubuntu.

God Bless, Jereme
