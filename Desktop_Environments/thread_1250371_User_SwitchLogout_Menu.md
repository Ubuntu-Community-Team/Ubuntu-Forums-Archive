### Post by wuhaa on 2009-08-26
Hi,

I am using Ubuntu 9.04 desktop on which I have installed a bunch of services for my web development stuff. Now with all these services, I have a bunch of users that are used by these services. They all show up on the logout menu. Is their a way to get rid of these service users from this menu.

Thanks,

---

### Post by VCoolio on 2009-08-26
Open gconf-editor, navigate to apps > fast-user-switch-applet and try the show_active_users_only box.
