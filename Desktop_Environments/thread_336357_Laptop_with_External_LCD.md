### Post by tweedledee on 2007-01-13
> **3fingersalute said:**
> I'm guessing I need to edit the xorg.conf file, but I want to be able to do so in a way that will allow me to boot up with either the LCD or the laptop's display and use the highest resolution of either without needing to reconfigure each time.

Yes, you do.  You need to edit the "screen" section (make sure you back up this file before making any changes!).  Not having 2 monitors, I can't say whether you'll have two screen sections or not.  If you do, then it's easy - find the one for the external display and add the extra resolution(s) you want to each "display" section.  If you only have a single screen section, it may be that you still just need to add extra resolutions to get it to work, but I'm less optimistic that will help.
