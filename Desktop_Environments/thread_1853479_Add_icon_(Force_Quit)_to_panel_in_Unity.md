### Post by Copper Bezel on 2011-10-02
Well, Unity doesn't put launchers in the panel - only the Dock. 

Alternately, just set a Compiz keybinding in compizconfig-settings-manager's Commands plugin for xkill. I personally use Super+X. Doing it that way, you don't take up any space in the UI at all.
