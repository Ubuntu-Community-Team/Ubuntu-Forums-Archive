### Post by Zorael on 2010-05-17
I'm unsure of what "widget plotter" you refer to, but there is a Widget Background plasma theme item you can change.

Head into System Settings -> Advanced tab -> Desktop Theme Details, pick a base theme you would like to modify (Air being default) and then change the Widget Background entry to that of another theme. Install **plasma-desktopthemes-artwork** for more themes, or use the Get Hot New Stuff feature in System Settings -> Appearance -> Style, Workspace tab.

When you apply your changes in Desktop Theme Details, it will be saved as a new theme called (Customized). It may not apply automatically, so go to the Workspace tab (in Appearance -> Style) and select it. See if that also changes the background for your widget.
