### Post by xy77 on 2005-09-06
Congratulations to your ubuntu installation!

Ubuntu provides synaptic for easy package management. you don't have to download the .deb (debian packages) by yourself.

* Hit Alt+F2 to execute a command.
* enter "synaptic" (no quotes)
* hit "Enter"
* provide your users password

The programm is pretty self explainatory, let us know, if you have any problems.
You could now install kde or install kubuntu-desktop or whatever you like.
Hope this helps, if you have any more questions, feel free to ask.

### Post by shakin on 2005-09-06
To make it a bit clearer, you need to install the kubuntu-desktop package, which will setup KDE and all the associated packages that come with a normal Kubuntu install.

Synaptic is one way to do this, or you can go to the console and run 'sudo apt-get install kubuntu-desktop'.

Good luck!
