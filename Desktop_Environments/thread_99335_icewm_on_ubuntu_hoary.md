### Post by az on 2005-12-05
Yes, it is easy.  By default, you do not have all the package repositories enabled.  It is in universe.

You enable the universe repository, update and and install it with synaptic.

Install the menu package, too.  Icewm uses the debian menu system and so that way you have access to all your stuff.
