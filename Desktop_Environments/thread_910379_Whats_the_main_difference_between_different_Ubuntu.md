### Post by Kevbert on 2008-09-04
[COLOR="Red"][Xubuntu]("http://www.xubuntu.org/tour")[/COLOR] is used for older PCs with low resources and uses the Xfce. It's fast compared to Ubuntu and Kubuntu as some of the applications are cut down versions of those used in the other two *buntu flavours. [[COLOR="Red"]Kubuntu[/COLOR]]("http://www.kubuntu.org/tour") uses the K Desktop Environment (KDE as opposed to Gnome in Ubuntu).

### Post by Kevbert on 2008-09-05
The KDE user interface is different from Gnome. Some things are better with KDE and some are better with Gnome.  The only thing I've noticed is there seem to be a few more bugs in Kubuntu when compared with Ubuntu of the same release (at least with Intrepid at present).  The best thing is to try a  Kubuntu Live CD and see (try Hardy or Gutsy).  All the applications should work in Kubuntu that work in Ubuntu.
