### Post by bkline on 2010-06-18
Has anyone else noticed that Gnome doesn't always use the chosen theme for all applications you open?  For example, I've got Thunderbird and Chrome open with the theme set to "Ambience" (the default for lucid), and while Thunderbird has the system buttons in the upper left corner of the frame window (where they're supposed to be for this theme), the browser window gets them in the upper right corner.  I don't need to tell you how annoying and confusing it can be to have to look in different places for the same functionality.

### Post by bkline on 2010-06-21
It looks as if this is a Chrome-specific issue.  Chrome is deliberately ignoring the system title bar layout and coming up with its own.  Fortunately, this behavior can be overridden.
