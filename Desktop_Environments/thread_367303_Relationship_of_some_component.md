### Post by Stephen Howard on 2007-02-22
I'm not sure what you mean, but maybe the following will help.  All of the software you listed form layers of programs that build up to produce a modern graphical computer user interface.

Fglrx: is a video card driver that allows software to use ATI brand video hardware.

Xorg  is a windowing system.  It manages input from programs, mice & keyboard and sends the results to the video driver.

XGL:  is a layer of software that uses modern video hardware to greatly increase the speed at which graphics can be manipulated and displayed.

Compiz/Beryl:  are competing programs that allow fancy graphic effects on the desktop.

KDE / Gnome:  These are competing desktop systems that the user actually uses.
