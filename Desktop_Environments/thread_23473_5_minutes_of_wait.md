### Post by blueplazma on 2005-04-06
If you want to have a firewall up and have gnome load in a decent amount of time, just run this command:

```
sudo iptables -A INPUT --src 127.0.0.1 --dest 127.0.0.1 -j ACCEPT
``` 

This command will allow any traffic coming from the loopback interface to come back into the loopback interface.  If you are using firestarter, you probably do not need this, but I am not sure.
