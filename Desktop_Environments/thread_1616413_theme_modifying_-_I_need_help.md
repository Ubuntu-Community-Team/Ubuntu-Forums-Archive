### Post by 3Miro on 2010-11-08
You can go to System -> Prefs -> Appearance and then select "customize" on the theme tab. There will be a new windows with a color tab. See what you can edit there. If this doesn't work ...

Depending on how you installed the theme, it is either in Places -> Home then .themes (hit Ctr + H to see the hidden folders that start with .) or it is in /usr/share/themes. Find it as a subfolder by the name. Then you will have to edit some text files, I am not sure about the right ones.
