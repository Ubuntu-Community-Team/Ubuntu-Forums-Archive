### Post by maxzorin73 on 2011-11-20
I am having a problem with the menu that pops up when right clicking on the Lubuntu desktop. The default menu came with options to create folders and manage desktop preferences such as wallpaper. I was experimenting and changed the option on the Advanced Tab of the Desktop Preferences. I checked "Show menus provided by window managers when desktop is clicked." Now I get a menu with OpenBox Options such as ObConf. I want to change this menu back to the menu with Desktop Preferences. However, the new menu does not have an option for me to change it back.

How do I change the right click menu back to the default one? Thank you.

---

### Post by ajgreeny on 2011-11-20
Alt+F2 and in the box type ```
pcmanfm --desktop-pref
``` hit enter and the box will pop up again, allowing you to change back.  It was possible in 11.04 to make the change back from the right click menu, but it is no longer in that context menu and has caught out several users users, including me.
