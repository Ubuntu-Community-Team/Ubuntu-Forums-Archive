### Post by mcduck on 2005-12-22
Open Applications/System Tools/Configuration Editor and go to apps/nautilus/desktop and select 'trash_icon_visible'.

You can also enable Computer, Home and Documents icons there. To use Documents-icon you also need to create a directory called 'Documents' in your home dir. :)

### Post by kalin on 2005-12-23
Or if you don't want to mess around with gconf (the configuration tool) you can install gTweakUI.
It adds additional configuration options to the System->Configuration menu that deal with several settings, which are only accessible using gconf without it.
