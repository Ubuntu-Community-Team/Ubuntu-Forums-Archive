### Post by pataphysician on 2006-05-01
i have an older ([m35-s359 satellite](http://www.amazon.com/gp/product/B0000DYFLF/104-2921244-1759969?v=glance&n=541966)). pretty much everything worked out of the box. i have an nvidia chip so i had to install support for that, and it took a few minor tweaks to get hibernation working. only thing that doesn't work here are the "action buttons" that are supposed to control cd playback or something. the volume control is hardware driven and does work. function keys to control brightness, hibernate, turn on/off the wireless card, etc., all work as well. 

i'm not sure about the intel display in the model you're looking at, but you're probably ok there. i'd check the forum. i'd also wonder about the remote control.

### Post by cyangecko on 2006-05-01
Now I am far from an expert on Ubuntu, only been using it for about 6 months.  I recently bought a customized Toshiba M100.  The main difference that might be of consideration regarding Ubuntu I noticed between the specs of the M105 and the M100 is that mine is a Dual Core (2.0 ghz).   I did encounter some problems when trying to install Breezy on this computer (old computer Toshiba Satellite 2435-s255 has Breezy running nicely).  I had problems with xorg and could not get into anything other than console.  Breezy also did not recognize my LAN connection.  After several frustrating attempts at getting Breezy to work, I downloaded Dapper Beta and installed.  I had no problems with Dapper (except for the usual install/configure of my wireless internet).
