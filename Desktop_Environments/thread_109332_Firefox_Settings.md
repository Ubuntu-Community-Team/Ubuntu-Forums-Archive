### Post by Submerged on 2005-12-28
i would try to manually export these settings within the programs themselves. For example:

Export Firefox BookMarks From Xandros:

Bookmarks>Manage Bookmarks>File>Export

THEN

Import Firefox BookMarks to Ubuntu:

Bookmarks>Manage Bookmarks>File>Import

Hope this helps
