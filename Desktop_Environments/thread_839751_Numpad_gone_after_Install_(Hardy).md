### Post by onineko on 2008-06-29
ok, i found it, it was because of a gnomebug.
workaround, in case anyone has the same prob: system - preferrences - keyboard, the mouse-tab, deactivate the setting, that allows you to use the keyboard to move the mouse-cursor..
