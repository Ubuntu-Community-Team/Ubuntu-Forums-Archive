Well, you could do what I've been doing and learn GNOMEs default keyboard shortcuts- My mouse is currently not working, so I've had to do everything so far like this ;). You can do everything- they are quite well thought out and very functionally thourough. For a list, see: [http://gnomesupport.org/wiki/index.php/KeyboardShortcuts](http://gnomesupport.org/wiki/index.php/KeyboardShortcuts)
