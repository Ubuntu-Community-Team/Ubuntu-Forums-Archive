### Post by miggols99 on 2007-08-04
Xfce is much faster than Gnome. I personally use KDE, but it runs sluggishly on my 1GHz 512MB RAM comp. So I changed it to Xfce. It's really fast now. Their slogan really is true "Xfce Desktop Environment... and everything goes faster!" On here I've got 1GB RAM, 3.2GHz processor and an ATI X300 graphics card. It's pretty fast on here, but I'm using Arch, which is much faster than Ubuntu.

I'd recommend Xfce if you want fast. You can still use Gnome apps on Xfce and still be pretty fast, but I personally think that KDE apps are much better (Amarok, K3B, KTorrent etc.) than Gnome apps.

### Post by pooslinger on 2007-08-04
You can also run just a window manager.  I use fluxbox due to the bloat of KDE, GNOME, XFCE, etc.  You will have to edit some text files but they will all be in ~/.fluxbox  .

> sudo apt-get install fluxbox

When GDM boots up click Session-> Fluxbox and log in.  It's pretty basic and bland but you can add styles

[http://fluxbox.sourceforge.net/themes.php](http://fluxbox.sourceforge.net/themes.php)

Add this to .fluxbox/menu to change backgrounds easy:
> [submenu] (Backgrounds)
		[wallpapers] (~/.fluxbox/backgrounds)
[end]
