### Post by bmac on 2008-04-17
You can turn the desktop mount icon off in the Configuration Editor. Open Configuration Editor &quot;Apps&quot; &quot;Nautilus&quot; &quot;Desktop&quot; Uncheck &quot;Volumes_Visible&quot;

### Post by bmac on 2008-04-17
