its not the same, but:

the latest greybird theme for xubuntu is brighter blue with pale context menu
```
sudo add-apt-repository ppa:shimmerproject/ppa
``` ```
sudo apt-get update
``` ```
sudo apt-get install shimmer-themes-greybird
```window decoration
preview [http://xfce-look.org/CONTENT/content-pre1/72290-1.png](http://xfce-look.org/CONTENT/content-pre1/72290-1.png)

gksudo thunar

sudo apt-get install gnome-icon-theme-full
