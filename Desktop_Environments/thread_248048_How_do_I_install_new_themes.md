### Post by aysiu on 2006-08-31
System > Preferences > Themes is for icon themes, Metacity themes, and GTK themes.

If you have GDM (login screen) themes, they go to System > Administration > Login Window

For all themes, you have to install the .tar.gz exactly as you downloaded it. Do not unzip it or do anything to it.
