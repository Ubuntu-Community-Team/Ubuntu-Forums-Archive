### Post by Forzan on 2009-09-18
I have a minimal installation of Ubuntu and a minimal installation of Gnome, and I would like to be able to share folders via Samba by going through the folder's properties menu in Nautilus. This isn't a part of the minimal Gnome installation, but I don't want to install the full Gnome package and get all the cruft that comes with it.

What packages do I need to install to enable network file sharing through Nautilus?

---

### Post by madverb on 2009-09-18
sudo apt-get install nautilus-share

### Post by Forzan on 2009-09-19
