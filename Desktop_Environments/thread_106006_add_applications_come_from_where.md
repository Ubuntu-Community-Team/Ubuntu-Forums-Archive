### Post by earobinson on 2005-12-19
they are coming from your repos there is a file named /etc/apt/sources.list that lists all the places your computer will dl programs from.

you can in the terminal type ```
sudo synaptic
<password>
``` to get more install options or I think its under System->Administration->Synaptic

You can edit your sources.list also to include more places that your computer will dl programs from.
