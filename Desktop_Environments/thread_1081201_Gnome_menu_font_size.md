I wanted to know how can I change gnome menu font size? I want to increase font size. There is no Preference -> Fonts in Ubuntu 8.10. Where can I change font size then?

---

### Post by Anxious Nut on 2009-02-26
press right click and choose "change desktop background" move to "fonts" tab and change the one that is entitled "Application font"

---

### Post by Anxious Nut on 2009-02-26
but note that once you change the font and the size it'll change the font and the size for the windows and its text too. ;)
