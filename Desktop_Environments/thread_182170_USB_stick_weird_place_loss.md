### Post by Sutekh on 2006-05-25
The hidden trash folder perhaps?

In Nautilus (the file browser) try using Ctrl + H or View -> Show Hidden Files and check to see if there is a hidden trash folder - .trash

If so, clear it out to reclaim that extra space.

In future if you use Shift + Del, I think that Nautilus will permanently delete files, rather than sending them to the trash.
