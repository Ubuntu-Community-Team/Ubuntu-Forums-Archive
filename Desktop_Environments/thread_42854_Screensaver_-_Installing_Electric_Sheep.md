### Post by ametade on 2005-06-19
In synaptic search for "electricsheep", or open a console and type:

```
sudo apt-get install electricsheep
```

You will have to have [Universe enabled](http://ubuntuguide.org/#extrarepositories).

By the way: great screen saver! ;)
