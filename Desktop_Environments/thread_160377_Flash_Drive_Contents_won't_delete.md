### Post by RS3York on 2006-04-14
When deleting from a flash drive, highlight the file(s) that you want to delete and press Shift+Delete.  This will permanently remove the file(s) instead of sending them to the "trash".

### Post by RS3York on 2006-04-14
No problem!  These forums have helped me out a lot, so it feels good to help someone else.  Be sure to help who you can.

BTW, if you go to Settings-> Configure Konqueror ...there's a check box for "Show Delete context menu entries which bypass the trashcan".  Then you can also right-click a file and select "Delete" which will appear right under "Move to Trash".  I'm pretty sure there's also an option to bypass the trashcan by default somewhere too, but that's KDE for you...LOTS of options :D.
