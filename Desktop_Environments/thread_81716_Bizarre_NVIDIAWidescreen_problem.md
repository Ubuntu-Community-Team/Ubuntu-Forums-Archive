### Post by RAOF on 2005-10-25
Do you mean the nvidia drivers from the nvidia website, or the nvidia-glx drivers through synaptic?  Apparently the latest nvidia drivers are somewhat buggy, this may be part of it.

If you haven't tried the nvidia-glx package through synaptic/apt-get, I suggest trying that.  From a console run:
```
sudo apt-get install nvidia-glx
sudo nvidia-glx-config enable
```
Then restart X (log off, and press Ctrl-Alt-backspace).  If that works, you're done :)
