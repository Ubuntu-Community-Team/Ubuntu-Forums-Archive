### Post by ssodhi on 2007-04-25
Hi,
I'm going to suggest that you boot to standard Ubuntu 7.04 Feisty Fawn. Ubuntu Server does not ship with a Graphics system (The X Server) installed. It physically does not know how to get graphics going.

The Server edition is for experience server adminstrators who know are adept at using the command line for anything and everything. You can still run a webserver, mailserver etc on Desktop Ubuntu, though it is not optimized. Download a new disk image and work your way to specific goals with [Ubuntu Guide]("http://www.ubuntuguide.org"). I'm sure it will be of much use to you, it walks through installing lots of common programs, and does have a good section on servers.

You will slowly gain proficiency in using the command line as you use Desktop Linux. The Terminal **is** a part of everyday Linux life, no matter what.
I don't mean to dispel your apparent enthusiasm, but you have to start small. Like they say, you don't learn to swim by diving in the deep end.

I hope this helps

Sanjay Sodhi
