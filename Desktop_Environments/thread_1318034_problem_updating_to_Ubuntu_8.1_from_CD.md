### Post by ajgreeny on 2009-11-07
It is another verrsion of the install CD that has been available for all ubuntu versions since the live CD versions appeared, because some machines could not use a live CD for a variety of reasons.  If you use the Alternate Install CD it will give you a text version of the installer, as most distros used to do.

You can download a CD for your required version from the usual download site; just look deeper than the first option.  Burn it to CD as usual, and then use it as your source for the upgrade.
