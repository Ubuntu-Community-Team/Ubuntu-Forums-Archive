### Post by asipi on 2007-06-14
use Konqueror profiles. as default in kubuntu there are 2 profiles: webbrowser and file manager. I made anotherone with two panels as commander style and also enabled the embeeded terminal (I named this profile as Kommander ;) ) so it is perfect for me.

The bookmarks u could not separate for the profiles, but it is easy to organize them, make a folder for the filemanager bookmarks and anotherone for browsing so u can find them easily
Konqueror is very good for browsing, but I am an opera fan so I rarely use it... (but this new 9.2 opera make me crazy with its CPU eating so I will soon move back to Konqy)
