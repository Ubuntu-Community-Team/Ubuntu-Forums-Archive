### Post by inCursorated on 2007-01-30
Hi VoodooSteve! On my system I have a "System Menu" next to my "K Menu". This guy has a "Remote Places" link that opens a window where I can click on "Samba Shares" to browse Windows shares. If authentication is required it will prompt for it. 

You can add this by right-clicking on the panel and go to "Add Applet to Panel..." (unlock panel first if necessary) and locate the System Menu in the list... hit Add to Panel.

edit: I see you're using Kubuntu dapper just like me, so the applet should already be there...
