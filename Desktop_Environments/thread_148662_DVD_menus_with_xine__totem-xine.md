### Post by NoNo_231 on 2006-03-22
I use totem-xine and I listen to music, watch DVDs, .avi, for almost all multemedia. I really like it. The only problem was that I couldn't have DVD menus (only the one given by the movie starting). The greatest problem were subtitles. However, there is a plugin called xine-dvdnav (available at prdownloads.sourceforge.net/dvd) that adds this functionality to Xine. (There is a libdvdnav4 in the synaptic, but it didn't seem to work with xine). So I 
Downloaded the latest rpm
sudo alien -k (name-of-package).rpm
sudo dpkg -i (name-of-package).deb

and now totem-xine has dvd-menu. 

Hope I helped someone.
