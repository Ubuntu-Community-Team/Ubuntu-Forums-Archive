### Post by Minsky on 2009-12-15
On my system, the Shade setting is an Emerald feature. the Shade icon can be deleted by editing the **Title Bar Object Layout key** which is within the **Titlebar** tab which in turn is within the **Edit Themes** Tab in Emerald. You can still use the Shade feature after deleting the icon by double clicking on the title bar. BTW I like the wallpaper image, can you tell me where you got it from?
