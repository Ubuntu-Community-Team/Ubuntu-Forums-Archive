### Post by Terrycymru on 2008-05-19
I want to add some of my own created folders to the Places menu. How can I do this?

---

### Post by sdennie on 2008-05-19
If you start nautilus and simply drag a folder to where all the other "places" folders are in the lefthand pane, it will add that folder to the places menu.

---

### Post by Linja on 2008-05-19
Bookmarks > Add bookmark from the nautilus menu bar. Use Edit Bookmarks to modify existing bookmarks.

### Post by atliax on 2010-07-18
In order to remove an item from the Places menu:
Open nautilus and right-click the item in question (from the Places list on the left). Select "Remove" from the menu that pops up.
