### Post by Kubunteando on 2007-12-09
It seems that the installation of the new drivers did not go well.

Did you build the drivers for your Ubuntu version?

Try this on the console:
grep EE /var/log/Xorg.0.log

It will look for error messages on the Xorg log file.
Probably it will tell that it was not able to load the fglrx module, because of something...

That something will tell you what is the issue.
But likely is that you have to find a proper way to install the new drivers.
Check they are compatible with your Ubuntu version.
