### Post by vaul on 2010-06-27
Hello.

About first thing &#8212; no, this is not a bug, rather aesthetically unpleasant thing. You see, these are different application, and for some reason they draw symbols masking password entered in different ways.

I think this is to be fixed one day.

About second thing &#8212; this is really a bug and it already have been submitted. 

Until this is fixed it is recommended that you enter password manually, since other workaround is to store password in plain text, which is obviously *very insecure*.

Other workaround for that are available on the forums, see [this]("http://ubuntuforums.org/showthread.php?t=1110931&page=2") for example. 

Or you just could have tried searching the forums. That's not that hard, seriously.
