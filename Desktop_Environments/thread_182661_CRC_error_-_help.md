### Post by Requiem on 2006-05-26
CRC is "cyclic redundancy check" a verification system to see if two files are the same. A crc error means that you are trying to use a corrupted file, either because the disc was scratched or badly downloaded OR because some one handled you a tampered version.

but I've seen this happen only on installation, this could happen with a livecd. Did this happen on a hard drive installation?
