### Post by 23meg on 2005-10-27
Which format are the images in? Ubuntu's default image viewer should open most common formats. Instead of double clicking, right click the file, select "Properties", hit the "Open With" tab and select "Image Viewer" there, and try double clicking again. Files of the same type should open with the default image viewer from now on.

Also, I'd recommend GQView (which you can find in the repositories) over the default viewer.
