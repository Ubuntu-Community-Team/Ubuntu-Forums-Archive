Hello. I'm using Hardy 8.04. 
Recently I installed the famous Latex IDE, Kile.
But all preview tools are set defaultly as those in KDE, such as kdvi,kghostview,kpdf,konqueror (for viewing dvi,ps,pdf,html repectively). Currently I'm using evince to view ps and pdf, firefox html. Could anyone tell me how to modify Kile options so that evince becomes the default previewer?
I really appreciate your help. Thanks a lot.

### Post by dvase on 2008-06-21
> **cf3372338 said:**
> Could anyone tell me how to modify Kile options so that evince becomes the default previewer?

This can be done under "Settings > Configure Kile > Tools > Build", there the settings for ViewPS, ViewPDF, etc. can be changed to call evince instead of its KDE counterpart.
