### Post by nelfer on 2007-11-17
Hi,
This is an annoyance more than a problem. It seems like they were trying to do a "nice feature" to remember your printer settings (whatever was the last thing you did) when going to print again a document.
That'll be nice if I'm only printing the same document and always to the same printer.

The problem is that I work with many documents, some go to the Monochrome printer, some to the color, etc. Some I have to print a sub-set, some I have to print all pages, some I have to print two by page.

The problem is that after I tweak the document the way I want and print it, I close evince and move to the my next document. Maybe now I just want to print (the print should just 'work'), but no, I have to go and clear whatever settings are there from my prior print of ANOTHER document.

Say for example I was printing page 64 to 80 of a document. Close evince and now I open another document that only has two pages. Lo and behold: when I go to print it, the selected option is "range" and it is for pages 64 to 80, that don't even exist here!

And instead of selecting my "default" printer, it always selects the last printer I printed too (there is a reason why the printer is set to DEFAULT!).

It's there a way to reverse this logic or change it? it is very annoying and no other program does such thing. Maybe it will be ok if it would remember the settings for the SAME document. That will be 'smart memory' of what I was doing with that document. But to keep the settings for all documents is just non-sense.
