### Post by markbl on 2012-07-08
Adobe has discontinued development of flash player for linux so google chrome are developing their own version which currently is quite buggy (at least I find it so and have read a few reports like yours etc). A temporary solution for your problem is probably to disable chrome's new flash player and fall back to the original version. Go to chrome://plugins and disable the pepperflash (PPAPI) version. This is not a long term solution of course. Just check that plugins page occassionally and try the pepper version when it updates.
