### Post by Dryvlyne on 2007-06-03
Hello, I messed around and chose to save my "Current Session" settings in Gnome on Feisty Fawn. Well ever since I did this not all of my apps (like Beryl and wireless) startup immediately upon logging into Gnome. They eventually come up, but there is now a long delay. I would like to return to my default session settings - how can I do this?

Thanks

**EDIT**
Nevermind, I figured it out. I deleted my Gnome user session file per the instructions on this site - [http://www.gnome.org/learn/admin-guide/latest/sessions-3.html](http://www.gnome.org/learn/admin-guide/latest/sessions-3.html) and the default session file was automatically restored.
