Hi!

I'm running Xubuntu 16.04. If I drag windows to the edge of the screen they end up being resized and sometimes moved. I don't want them to do anything! How can I stop this?

Thanks!

---

### Post by ajgreeny on 2016-08-17
Go to Settings Manager ->Window Manager ->Advanced tab and deselect the Window snapping options there.

I also dislike that action when dragging windows and always turn it off.

---

### Post by xadder on 2016-08-17
Ah, I thought I had tried that, but with one more tweak I think all is okay now. I also had to do Settings manager->Window Manager Tweaks -> Accessibility -> uncheck 'Automatically tile windows when moving towards the screen edge'.

Thanks.
