### Post by p_quarles on 2008-04-28
For KDE 3.5.x, there is Superkaramba, which displays widgets of the type you are looking for. Many widgets, including RSS tickers, are available at kde-apps.org. For KDE 4, there are "plasmoids" that act much like Superkaramba widgets, but I am less familiar with these.

There is also Conky, a system monitor application that works in any X window manager, including KDE. It is relatively easy to add an RSS feed to this monitor.
