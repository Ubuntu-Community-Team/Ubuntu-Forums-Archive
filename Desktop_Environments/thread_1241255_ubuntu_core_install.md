### Post by urukrama on 2009-08-19
You would use the alternate install CD, and choose a minimal/command line install. Once your Ubuntu install is completed, you'll have a fully working, but very bare command line system running, to which you can then add whatever you want. Have a look at aysiu's guide for the details: [http://www.psychocats.net/ubuntu/minimal](http://www.psychocats.net/ubuntu/minimal)

Aysiu's guide uses Icewm; if you prefer Openbox, replace icewm with openbox in the commands he gives. You might also want to install openbox-themes, obconf and obmenu. The first is a collection of themes, the second an application to configure Openbox, the third an application that makes editing the Openbox menu very easy.

If you need help configuring Openbox, have a look at the guide I link to in my signature. It should help you get started.
