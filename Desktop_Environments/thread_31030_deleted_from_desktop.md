[QUOTE=n-babe]I have a problem-
permanently deleted folder from desktop. which command I must use for to get it back?  :shock:[/QUOTE]
 If you deleted it - not just moved to trash - then it can not be easily recovered. This is why it is called "permanently delete". 

There are some very complicated methods which allow partially recovering it, but it is not easy, and you get back parts and pieces at best. So unless this file was the masterpiece you worked on for 20 years, you should probably forget it.  

But if you do not mind spending several hours on this, take a look here: 
[http://recover.sourceforge.net/linux/](http://recover.sourceforge.net/linux/)
