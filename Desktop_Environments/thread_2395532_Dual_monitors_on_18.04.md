### Post by PaulW2U on 2018-07-03
I've been using 18.04 with two monitors with various flavours of Ubuntu for sometime now and very successfully. My set up is probably a little different than yours in that I use a laptop with its monitor configured as the secondary display. The external monitor is configured as the primary display. I had few problems setting it up but I did have to experiment with the settings as to get things as I liked as the monitors are of different sizes.

I'm attaching a screenshot of the relevant settings page on Ubuntu 18.04.

You need to start by selecting "Join displays". Once set up you will be able to drag windows from one display to the other.
