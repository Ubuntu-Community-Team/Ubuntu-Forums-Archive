### Post by Nerotriple6 on 2011-05-05
Hello.

You could be more clear but anyway..
First of all. Gnome-look has a lot of themes, some of them appear glassy.
[http://gnome-look.org/](http://gnome-look.org/)

Second, if you install Compiz Config you can make the windows, menus and everything transparent. As I have done and added some blur for a "glassy" effect. See attachment for how it looks. :D
To install Compiz in Debian (Ubuntu):

Open a terminal and enter
```
sudo apt-get install compizconfig-settings-manager
```
Enter your password and when installation is complete you will find it in System>Preferences in the Gnome menu.

As for messaging, I use Pidgin. It supports Yahoo, MSN, Gtalk etc etc..
```
sudo apt-get install pidgin
```
