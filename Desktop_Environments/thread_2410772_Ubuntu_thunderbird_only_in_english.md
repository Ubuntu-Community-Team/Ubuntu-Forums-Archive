### Post by vanadium on 2019-01-22
Why did you uninstall in the first place? You now probably installed the snap version, which is a beta version and probably comes with very limited language support. Remove it using Software, the search Software again. This time, take the second entry that appears down the list, the one with five stars under it. Install that one and it will automatically adopt to the language of your desktop.
