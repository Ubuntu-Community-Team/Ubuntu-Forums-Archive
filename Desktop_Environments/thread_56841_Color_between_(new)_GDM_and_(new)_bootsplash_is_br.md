### Post by AndyAWS on 2005-08-14
Try this:

System->Administration->Login Screen Setup->Standard Greeter Tab

At the bottom left you can set a background color.

Also:

System->Preferences->Desktop Background

There is Desktop Colors at the bottom.

I believe that between these 2 settings you can set what colors are displayed while graphics are being loaded.
