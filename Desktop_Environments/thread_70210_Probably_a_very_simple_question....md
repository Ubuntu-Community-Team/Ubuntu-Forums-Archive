### Post by heimo on 2005-09-29
And to follow up on this interesting thread... ;)

This is what I do when I'm lost and don't know which command renames
```
apropos rename

``` 
I get:
> dpkg-name (1)        - rename Debian packages to full package names
ifrename (8)         - rename network interfaces based on various static criteria
lvrename (8)         - rename a logical volume
lvrename (8lvm-10)   - rename a logical volume
mv (1)               - move (rename) files
rename (1)           - renames multiple files
vgrename (8)         - rename a volume group
vgrename (8lvm-10)   - rename a volume group
