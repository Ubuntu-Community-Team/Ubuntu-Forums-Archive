### Post by urukrama on 2008-07-02
To edit the menu, use [obmenu]("http://obmenu.sourceforge.net/") (it is in Hardy's repositories).

The bar at the bottom of the screenshot was probably a panel. There are plenty of panels you can choose from. Have a look at my Openbox guide (link in signature) for help with this.

For icons on the desktop, either use idesk, or a file manager that can manage the desktop, such as PCManFM or Rox-filer. See the above mentioned guide for more details.
