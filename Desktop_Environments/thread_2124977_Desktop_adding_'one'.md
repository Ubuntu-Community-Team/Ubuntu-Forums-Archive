### Post by Frogs Hair on 2013-03-12
Install the dconf Editor and try the following ```
killall gnome-panel
``` See the link for additional command options. [http://askubuntu.com/questions/125662/how-to-reset-gnome-panel](http://askubuntu.com/questions/125662/how-to-reset-gnome-panel)
