### Post by UbuntuNerd on 2009-06-19
to browse to that location as root type this command in a terminal:
```
gksudo nautilus
```
or you can install it by right clicking on an empty space of your desktop scroll to the bottom and click on (Change Desktop Background)
when the menu opens make sure you have (Theme) selected, then drag the theme(the whole folder) to the appearance Preference box to install it. 

some themes may need to be unpacked before you can install it, the right format should be tar.gz
