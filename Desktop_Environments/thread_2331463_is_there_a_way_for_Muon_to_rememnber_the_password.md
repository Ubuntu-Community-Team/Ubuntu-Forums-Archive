### Post by PaulW2U on 2016-07-22
> **allaf said:**
> Is there a way in kde to make Muon not ask the password every time ?
allaf, that's not a muon prompt but a system prompt asking you to give muon administrative privileges. It gives you time to think "Do I really want muon to do this?". If muon remembered your administrator password then that would be about as good as not having a password at all. There are probably ways to accomplish what you want but your system could then be just one mouse click away (or less) from being damaged or corrupted if you made the wrong decision. Being a system prompt, would you also want other applications to be able to take possible destructive action so easily?

How often do you use muon anyway?

Once an installation is set up to your liking the use of the administrator password will decrease as updates become fewer as time passes.
