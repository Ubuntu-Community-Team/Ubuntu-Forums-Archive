### Post by aslamdoctor on 2009-10-03
Use this command :

sudo tasksel

& select Kubuntu-desktop from the list of packages & select ok.
This way we can install KDE desktop safely, so if u don't like it, u can uninstall it from here & u will be back to ur main Ubuntu safely without any bug.
