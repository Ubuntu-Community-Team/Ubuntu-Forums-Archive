### Post by molo on 2006-11-18
> **KnevetS said:**
> In my situation, I have matching LCD's (just picked them up).

...

I did find that the setup does not work at all if the display size is set to above 1024x768.

I think this is because our ATI video cards don't support a texture size of above 2048x2048 (if you have 1280x1024, the size for the screen would be 2560x1024).  This is the same reason that you can't use an image larger than 2048x2048 as the Skydome background under XGL.

You can test for maximum texture size by doing:
```
glxinfo -l | grep GL_MAX_TEXTURE_SIZE
```

See what the output is.
