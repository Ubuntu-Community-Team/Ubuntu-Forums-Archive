### Post by Toet on 2008-11-06
I really like Virtual Box running Windows on it. Fullscreen, not able to see the difference, PI running faster than original Windows (using paravirtualization). Seemless mouse and keyboard integration with add-on software. 

Cons: 
- Host netwerking setup is a pain. VMware is much easier on that. 
- No standard USB with Open Source version
