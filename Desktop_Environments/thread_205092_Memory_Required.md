### Post by kzutter on 2006-06-28
I think you are confusing disk space with memory.

You probably have a 9 gb hard drive.

IF it is an older Windows box, I would be surprised if you have more than 128 mb of memory.

IF the computer boots Windows you can right-click on My Computer and select 'Properties' to see how much memory you have.

Yes, you can get rid of Windows, but you better get linux working first.

Good luck and have fun!
