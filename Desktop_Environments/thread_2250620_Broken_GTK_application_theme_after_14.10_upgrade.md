### Post by Frogs Hair on 2014-10-30
If you are using the linked ppa to get the gnome 3.12 packages not included in the Ubuntu repository there are applications that will appear differently as I found out on a beta 2 installation. The applications that were affected included gedit, nautilus, and the gnome terminal. I have since reinstalled and excluded the ppa . If you aren't using the ppa I don't know the cause. The window decoration in the picture looks as it should for the Gnome Shell but not Unity unless the ppa is installed .
