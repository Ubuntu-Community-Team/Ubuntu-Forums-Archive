### Post by wojox on 2009-08-15
```
gksudo gedit /etc/X11/xorg.conf
```

Will open the file as root. Scroll down and look for:
Section "DRI"
Mode 0666
EndSection
Section "Extensions"
Option "Composite" "0"
EndSection

Change Option "Composite" "0" to "1"
