### Post by Saibot on 2006-08-30
Your computer should be able to handle Compiz fine.  You could try opening gconf-editor and navigate to apps/compiz/general/screen0/options and check slowness_fix. I read that it helps sometimes.  Also if you're running the blur plugin try disabling it.
