### Post by grahammechanical on 2015-06-18
> because mostly people telling me desktop not good servers

You have not installed Ubuntu Desktop edition or even Xubuntu Desktop Edition. You have installed Ubuntu server edition and then added a desktop environment. That is not the same as installing Ubuntu Desktop Edition. By adding a desktop environment you have not taken anything away from Ubuntu server edition. You have made it a little easier for yourself. 

I will accept that Ubuntu Desktop edition does not make a good server machine because it does not come with server software but we can add that kind of software if we want to. In the same way Ubuntu Server edition does not make a good desktop machine because it does not come with the desktop software, But we can add that kind of software if we need to. 

It is also a fact that whether we install a Desktop or Server edition we will also get a lot of software that we will never use because we do not need it. That does not stop us from making good use of Ubuntu as a desktop machine or a server machine.

One of the reasons why Server editions do not come with desktop environments is because the machines do not usually have a graphic adapter installed. So, everything is done using the command line. Your machine has a graphic adapter does it not? So, where is the problem?
