### Post by Klarsin on 2007-02-05
Many of the Distros don't read certain graphics cards very well. Ubuntu is one that does. Don't know what version of Kubuntu you have, but this should work for dapper 6.06 as well as  edgy 6.10:  After you insert the live CD and restart, look to the bottom of the boot screen and hit 'F4' and selcet the maximum screen resolution for your monitor ( if you know it) or any higher resolution that might  allow you to see the boot window entirely. If this doesnt work, you can hold down the alt key and drag the window around with the mouse to see the areas  that are off the screen. This at least should allow you to get through the install process. After you have sucessfully installed Kbuntu, navigate to 'system configurations' (control panel) and find  the screen resolution settings, and make changes there if the resolution still is not set properly.
