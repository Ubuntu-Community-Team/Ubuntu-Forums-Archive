### Post by urukrama on 2008-02-06
If you want a Windows look, and a light working environment, you could also try Icewm (it is in the repositories). It is a light window manager with a Windows 98 look. There are quite a number of Icewm users on these forums, so you'll get support if you encounter some difficulties.

The default Icewm theme is rather ugly, but there are some decent themes available from [box-look]("http://box-look.org/index.php?xcontentmode=7311") and [freshmeat.net]("http://themes.freshmeat.net/browse/925/").

Aysiu has a [good guide]("http://psychocats.net/ubuntu/icewm") to set up Icewm.

### Post by siepo on 2008-02-06
