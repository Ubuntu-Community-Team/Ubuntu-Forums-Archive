### Post by HarshReality on 2011-05-01
I just installed it yest. on a system I just upgraded here at home. Have been using Meerkat since its release and I would simply comment that Unity is totally disagreeable to me. Yes I know "You can choose when logging in" but then you get into the grey area of bloatware. One thing I have always loved about linux is if I didnt use it it was not there when I installed but with this release its not the case.

  Also using gnome I grew used to the menu and finding what I was after despite me not knowing the name of the program. With Unity all I find is a search menu so if you dont know the name of the program what in the hell are you going to use a search for?!

  So, while I was an avid end user (not into programing or the like) I find this release just a bit alienating and ironic. Maybe if they offered a desktop version at the time of install... but its a bit funny how I get a distaste for it (as a typical lin user) and at the same time all the windows users are chiming in all over the net going yippee!

Just my 2 cents.. of course with exposure if the opinion changes I will update this posting but I honestly dont see it happening
