### Post by Dennis N on 2020-04-25
Application themes and Gnome Shell themes have always been separate - neither affects the other. The window colors settings only affect the default Yaru application theme, not any Gnome Shell theme. And if you use a different applicatiion theme than Yaru, these settings are of no use to you, as it switches your application theme to Yaru.

If you want the menus in the panel to be dark, you must use a dark Gnome Shell theme. There are lots of them available.

Dark Window Color is only for Default Yaru Theme and automatically sets the applications theme to Yaru Dark. There is a matching gnome shell theme, Yaru Dark, that gives you the dark menus. Select that as well (from Tweaks). Everything matches perfectly.
