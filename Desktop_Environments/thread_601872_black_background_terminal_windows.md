### Post by seachnasaigh on 2007-11-03
Heyo ... dcas, that's one option, man. Works, too.

Try this though: click the EDIT menu and choose PROFILES.
Default is the one there by, well, default. You can add others. 
Then you can edit the default profile, one of the others, or choose to make some other one the default by choosing EDIT>CURRENT PROFILE. This gives you a lot of options. One of them is making the background black; another is changing the font (I like a fixed-width system font like Courier at 14 point or so, makes it easier to see) and the transparency, which is very cool and allows some or all of your desktop background to show through. I like the Gnome terminal for its config options ... try a few things, create a few profiles. You can also have the terminal auto open to a root terminal if you've set root on your system; I usually create this as a profile on servers and only use it when I'm doing sysadmin stuff.
Have fun!
