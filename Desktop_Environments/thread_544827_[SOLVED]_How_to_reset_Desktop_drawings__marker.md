### Post by mcduck on 2007-09-07
That's the Annotate-plugin in Compiz. You can set the keyboard shrtcut for clearing the drawings with CompizConfig Settings Manager. It's in Extras/Annotate.

The default shortcut seems to be Alt-Super-k.
