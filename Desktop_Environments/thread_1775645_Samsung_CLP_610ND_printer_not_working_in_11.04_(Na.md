In 10.04 and 10.10, the printer worked fine. Unfortunately, because of this, I didn't take much notice of how it was connected. I remember doing "find network printer" and following the bouncing ball and everything was OK.

There are 2 windows machines on this network which are printing with no problems, so I am guessing that something is wrong with the way 11.04 is treating this printer.

In case it was something in the upgrade process, I rebuilt my system from the 11.04 CD, but the problem persists.

Has anyone else experienced this sort of printer hassle with 11.04? Any input would be appreciated.
