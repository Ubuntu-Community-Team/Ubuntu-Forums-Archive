### Post by 3Miro on 2011-01-19
On the wallpaper: How are you changing the wallpaper? You should right-click on the desktop, add the image to the list and then select the image as a background.

Themes: Check Gnome-looks for many different themes.

[http://gnome-look.org/](http://gnome-look.org/)

On Upgrading: you should be able to upgrade from the update manager (or Synaptic Package Manager). the latest version is 10.10 (11.04 is still in alpha and it is not recommended for regular use yet). 

IMPORTANT: make sure you have backups on all important data, just in case something doesn't work right. You should probably also make a liveCD so that you can boot your machine even if things break down.

### Post by LinUxaliVe on 2011-01-19
You could use a very simple, and small program like 'feh' to handle your wallpapers. 

It can use scripts to randomize your wallpaper each time your desktop is loaded, or even rotate your wallpapers every few minutes.

Personally I wouldn't do an upgrade, if you are definetely running v10.04, it's called "Lucid"; Jaunty is v9.04. 10.04 is a Long Term Support (LTS) version and receives updates for 3 years, whereas normal versions get updates for 18 month: [https://wiki.ubuntu.com/LTS](https://wiki.ubuntu.com/LTS) .

If you have indeed Jaunty 9.04, dist-upgrade with the "Update Manager", like 3Miro said.
