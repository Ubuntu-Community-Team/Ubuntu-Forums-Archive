### Post by grahammechanical on 2013-11-09
For Workplace Switcher in 13.04 and onward go to System Settings>Appearance>Behavior tab and tick the box called Enable Workplaces. With 13.04 we do not need to install Compiz Configuration Settings Manager in order to reduce the size of the Launcher icons. That function is also in the Appearance utility.Regards.
