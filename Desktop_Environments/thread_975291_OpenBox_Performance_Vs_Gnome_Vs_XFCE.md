### Post by snowpine on 2008-11-08
Hi Sniffer, Linux uses ram a little differently than maybe you are used to. Linux likes to cache things into the ram to boost performance. The more ram you have, the more ram Linux uses for cache.

You say you have 2gb of ram available. There would be no performance boost using 60mb of ram vs. 160mb of ram. It might even be slower, since Linux is using that extra 100mb for cache. RAM is a unit of memory, not speed. :)

Rather than measure how much ram is used, you should measure things like: does Openbox load faster? Do windows open and close quickly? Do applications start quicker? That sort of thing. If you love Openbox, isn't that what matters? :)
