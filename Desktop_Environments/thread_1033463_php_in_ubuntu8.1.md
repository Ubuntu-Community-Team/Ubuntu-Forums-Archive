### Post by mcduck on 2009-01-07
Two easy ways: Open Synaptic, go to Edit-> Mark packages by task and then enable "LAMP server" & click the Apply-button in Synaptic.

To do the same from command line run "sudo tasksel install lamp-server"

This will install & configure Apache2, PHP and MySQL for you. They will be started automatically on boot.
