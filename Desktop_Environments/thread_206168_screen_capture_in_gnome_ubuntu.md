### Post by ardchoille on 2006-07-01
[QUOTE=lofty00]How do you do this? I used to use kscreenshot in kde on debian, but I can't find a screen capture utility in ubuntu, apart from a command line one called scrot, which isn't what I want.

thanks.[/QUOTE]
There are a few options with gnome-screenshot. Open a term and type:

```

gnome-screenshot --help-gnome-screenshot

```
You can use those options (eg. --window --delay --border etc) and add new commands to the gnome menus.
