### Post by skrribble on 2007-12-13
you can get icon themes from [http://www.gnome-look.org](http://www.gnome-look.org)
if i were you, i would download one from there, and then modify it with the icons you want.  After you install an icon theme, its contents will be placed in /home/yourname/.icons

you can name your icons the corresponding program to be used with and put them in the right folder

for example, if you have a new icon for firefox, name it firefox.png and put it in the /home/yourname/.icons/yourtheme/scalable/apps folder.  when it asks you to replace hit yes.  then you will have to restart your icon theme by going to System>Preferences>Appearance and switching it to something else and then back
