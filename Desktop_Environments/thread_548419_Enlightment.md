I think that the good move/bad move argument can pretty much be left with figuring out what you want to get out of Enlightenment.

I have used e17 some in the past and have always gone back to Gnome for my own reasons - none of which can be attributed to the stability or interoperability of Enlightenment. I found E to work very well with all Gnome apps that I had installed and it was an extremely attractive and customizable WM. 

So, to answer your questions directly:

Stable - It was very stable on my Micron Transport GX2 laptop.
Updating - Yes, you can update the same as with any other WM/DE
Gnome apps - It worked with the standard Ubuntu load and the miscellaneous apps I had installed.

I hope you enjoy E if you decide to try it out. Remember that you can always install it and easily choose to go back to Gnome (or KDE, or BlackBox, or, or, or) by simply choosing from the login screen.

Check back if you have any other question.

### Post by the yawner on 2007-09-12
I'm currently using E17 installed over my Ubuntu (GNOME) setup on my Thinkpad and on my Xubuntu (Xfce) setup on my desktop. Both are working fine but I find my Xfce/E17 combo better. For the programs, there's little issue. For wifi, I've read some workarounds are needed.

Some issues I've encountered:
E17 inherits the application menu layout of your DE of choice and there seems to be no easy way of editing that. Thus, on a default install you may not get immediate access to some integral programs (i.e. Synaptics). But there are some workarounds (launchers, favorite menus)

E17's config panel are for E17 related configurations only. The window themes work on all programs (as expected), but there's no immediate way to change the GTK themes to match (as close as possible) the window theme. I have to create a .gtkrc-2.0 file for that (with the theme already installed on the system).

E17's default file manager is still too basic so you would want to use another that'd suit your needs. Thunar (Xfce's file manager) works well. Running Nautilus... would create an interesting output.

That's what I can come up so far. Heh.

=====
Edit:
Now that I've checked my desktop PC, I now remember why I preferred this setup over E17 installed over GNOME. The application menu retains the administration sub-menu, which holds most of the admin programs (Synaptics, Update Manager, Network Manager). Less work for me. :D
