### Post by Tenken on 2007-12-15
The nVidia setting tool gives you an option to create two separate X entries, under X Server Display Configuration there is a drop down to choose TwinView, Xinerama, or Separate X files, choose the separate X option and click the button to save the configuration file, making sure to save a back-up of your original xorg.conf first.
