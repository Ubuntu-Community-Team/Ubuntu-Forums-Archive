### Post by Perfect Storm on 2005-01-18
Hello there, Sham.

GDM is your login screen. If you're looking for themes at gnome-look.org go for the GTK2 and download the one you wanted. 

Click the 'Computer' tab at your desktop, go 'Desktop preferences' and choose 'Theme'. Now press the 'install theme' button and browse to where you saved the theme file. No need to unpackage the file the theme installer do it for you.

To install Icons is a diffrent matter, because the theme installer is bugged. Instead of the 'install Theme' press 'Theme Details'. Choose the 'Icon' tab and select 'Go to theme folder'.
Now unpackage your icon theme file to this folder. Now you can choose that icon theme in the list.


Edit: Also check: [http://ubuntuguide.org/index.html](http://ubuntuguide.org/index.html) there is alot of good stuff if you are new.
