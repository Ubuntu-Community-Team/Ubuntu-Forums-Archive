### Post by Dutch70 on 2010-12-17
"There is no option" to move it? Or the option to "move" is greyed out so you can't select it? 

If it's the latter, then when you right click it, select "lock to panel" which will actually unlock it. If you're system looks like mine, it's difficult to tell if it's selected or not. 

Also, If w/e is next to it is locked, you've probably got it blocked in & will have to unlock both.

Deleting it & putting it back can be a pain sometimes.
