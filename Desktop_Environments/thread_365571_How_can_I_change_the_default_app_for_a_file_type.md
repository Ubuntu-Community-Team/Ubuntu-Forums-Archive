### Post by ubix on 2007-02-19
In **nautilus** righ-click a file for which you are setting an application to open with. Select **Properties** and open **Open with tab**. There click **+ Add** button and select an Application the usual way.```
 [ nautilus; R-click (a file) ] -> [ Properties ] -> [ Open with tab ] -> [ + Add ]
```
