### Post by SuSUntu on 2007-03-30
In some cases, launching an application via AllTray will enlarge the notification area icon.

The only reason I'm aware of this "side-effect" is because NoteCase's traying action does not work properly while using Xfwm as the window manager, so I have to launch NoteCase with AllTray. As you can see from the screenshot, the NoteCase icon launched with AllTray (third icon from the left in the notification area of the bottom panel ) is about twice as large as the NoteCase icon launched without AllTray (first icon in the notification area of the bottom panel).

Again, this is a side-effect of using AllTray and is not optimal as it doesn't affect all icons in the same way. However, since I don't know of any better way to do what you are looking for (aside from compiling your applications with larger icons replacing the originals), this might work for you if it's that important.
