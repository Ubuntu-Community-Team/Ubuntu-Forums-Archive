### Post by trevorj on 2008-12-30
I've had the same problem.  Maximize would only use the upper 3/4 of the window space.  I searched for 'gnome-window-settings' update and didn't find one.  So I selected and upgraded the 'gnome-settings-daemon' and 'libgnome-window-settings1' packages.  (I don't know if both were required to fix the problem or just one of them.)

It did solve the problem!!
