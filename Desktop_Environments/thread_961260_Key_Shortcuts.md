### Post by mali2297 on 2008-10-28
Go to *Settings Manager -> Keyboard Preferences -> Shortcuts* and create a new theme. Then add a new shortcut to the command *xfterm4*. This launches the default terminal emulator as set in *Preferred Applications*.
