### Post by Double.J on 2012-02-01
I usually add lubuntu+kubuntu to my ubuntu/mint install, Just bare in mind that there's a lot of doubling up on programs, it doesn't bother me at all, but for some it's a real bug that you end up mixing QT and GTK programs.

As for the order, no it doesn't really mater, it's all the same system under the hood anyway. I suppose by default the last one you install controlls the splash images... but these are easy changes! You do sometimes need to change your start up applications so you don't double up on things :)

Personally I find going between gnome and KDE pretty simple  - chrome even opens a session in one that I exited in another :)
