### Post by winh8r on 2012-03-09
Right click on bottom panel and select "new panel"

New panel should appear at top of screen, if it appears at side of screen drag it up to top and it will stay there.

Then right click on top panel

select ```
add to panel
```

add the following:

```
indicator applet

menu bar

indicator applet session
```


these are the basics

add any others you like to have.

Hope this helps you.
