### Post by krazyd on 2009-07-25
Note: I'm using the KDE4.3 RC, so not sure if this works in KDE 4.2.

you can edit the device notifier settings in:
System Settings > Advanced tab > Device Actions

Also, just click the Device Notifier icon in the taskbar to make the popup disappear.

### Post by Zorael on 2009-07-25
Yeah, it's the device notifier *widget* that's spawning the popups, so giving it a click should close them. By extension, removing the widget should stop them from appearing altogether.
