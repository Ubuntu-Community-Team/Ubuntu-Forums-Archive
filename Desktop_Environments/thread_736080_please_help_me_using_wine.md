### Post by warp99 on 2008-03-26
You have to install the program first using Wine on your Linux machine, then you can run the program. Also I would obtain the latest release of wine direct from winehq.com:

[http://www.winehq.org/site/download](http://www.winehq.org/site/download)

Their is a FAQ and manual at winehq.com on how to use Wine:

[http://wiki.winehq.org/FAQ](http://wiki.winehq.org/FAQ)
[http://www.winehq.org/site/documentation](http://www.winehq.org/site/documentation)

Instructions for enabling the Wine repositories are on the website. As far as compatibility Borland Turbo C is an old program and should run fine with Wine. It's not the DOS version, is it? ;)
