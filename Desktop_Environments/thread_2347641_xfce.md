### Post by Bucky Ball on 2016-12-27
You get it going by selecting it from the session menu at the login screen so if it's not showing there, something wrong. What is showing there? How did you install xfce4? You need to install just that:

```
sudo apt-get install xfce4
```

... or use a package manager to install. That's it. Log out, choose from 'Sessions' menu, login. Tell us how you go. Better still, just install Xubuntu 16.04 LTS and get rid of the dross and bloat you don't need in Ubuntu. :)

PS: Tried [Xubuntu-core]("http://xubuntu.org/news/introducing-xubuntu-core/")? Community ISOs linked at the bottom of that page. Even lighter, but you need to add whatever apps you are going to want/need once installed. Good luck.
