### Post by poptones on 2005-12-14
Gthumb is the worse image viewer I have ever seen. Even if you select "full screen" if you click the "next" button you have to click "resize" AGAIN to make it full screen. I don't know what that thing is designed for but it ain't designed for editing and it sure ain't designed for browsing.

I have been using gqview forever. the older version is in the repository; Download the latest version and compile it (you need build-essential and gtk+ and that's it). It opens instantly, it's very configurable and you can set "edit as" commands yourself so if you really must have a resize command you can add one via imagemagick. it comes with "rotate" and "edit in gimp" and that's always been enough for me. the best part is the navigation - if you "hide menus" and set a couple of preferences then it opens to nothing but the image you clicked and turning the scroll wheel will navigate the directory (or, if you open it with one, the images listed in the "gallery" text file).
