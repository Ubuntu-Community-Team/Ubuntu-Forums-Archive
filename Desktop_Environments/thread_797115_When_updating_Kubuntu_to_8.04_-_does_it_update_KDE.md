### Post by Xiong Chiamiov on 2008-05-17
If you already have the hardy repos enabled, then everything will be up-to-date.  KDE4 isn't automatically installed with the kubuntu-desktop package (at least, afaik), but if you have the kde4 packages installed, then they will be updated to the latest version in the hardy repos whenever you enable them (you could right now).
