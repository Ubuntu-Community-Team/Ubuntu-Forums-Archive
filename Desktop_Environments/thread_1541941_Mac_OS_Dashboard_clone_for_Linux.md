### Post by mcduck on 2010-07-31
There's no single program that would behave like Dashboard, but you can easily get the same functionality.

Compiz has the Widget Layer-plugin which allows you to set any applications to be hidden by default and only show when you press a certain key, just like the Dashboard. Use that with some Screenlets and you'll have your dashboard (but keep in mind that you aren't limited to the Screenlets, you can add any programs you want to the "dashboard"). :)
