### Post by heylinus on 2006-11-04
I installed Maya 8.0 on Edgy(Gnome Desktop) yesterday. 

The problem is:

[COLOR="Red"]I can not use the shortcuts of Tumbling(Alt+LMB), Tracking(Alt+MMB), Dollying(Alt+RMB) in Maya GUI.[/COLOR]

When I click "Alt+LMB" and move the mouse, it moves the whole Maya Window; 
when I click "Alt+RMB", it pops up the Window Menu.

I think the "Alt" key may be hijacked by gnome, buy I can not find an option to release it in "System -> Preferences -> Keyboard / Keyboard Shortcuts".

Could anybody tell what's going on?

PS:
I remember that in Dapper I can use the "Win" key to replace the "Alt" key.

### Post by jujoje on 2006-11-05
The option to change the behaviour of the "alt" key is in "system--> preferences--> window preferences". Just change it to "super" as you did before and it should all work again.

---

### Post by heylinus on 2006-11-05
How dumb I forgot to check out the "Window Preference". Thanks for help, jujoje!

I also figure out another way to use "Alt" key correctly in maya without change the default behavior in other windows.

I just go into "System -> Preferences -> Keyboard -> Layout Options -> Third Level Choosers", and select "Press any of Alt keys to choose 3rd level" and then comes back my sexy maya :)

Hope this will help others with familiar problems.
