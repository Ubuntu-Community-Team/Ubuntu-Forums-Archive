### Post by x64Jimbo on 2006-09-17
The notion that there are no 64 bit apps is simply untrue. Every time you use a 64 bit OS, all of the apps there are 64 bit and will run better than their 32 bit counterparts because the AMD64 instruction set is more efficient than x86. There are some apps that you can install separately, but Ubuntu's 64 bit versions use 64 bit code for everything in the repositories, which leads to increased performance on a systemwide level.
