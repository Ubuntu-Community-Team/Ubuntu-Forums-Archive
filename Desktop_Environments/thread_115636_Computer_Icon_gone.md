### Post by leech on 2006-01-11
Which computer Icon do you mean?  The one on the desktop?  If that's the case, then it's an easy fix.  Applications -> System Tools -> Configuration Editor.

Then inside of that it's under ```
apps -> nautilus -> desktop and just put a check box in "computer_icon_visible"
```

If you're talking about inside the Computer area for Nautilus... yeah, it's not supposed to be there, you're already inside that part.  Computer is the area that encompases your main Filesystem, plus all removable devices.
