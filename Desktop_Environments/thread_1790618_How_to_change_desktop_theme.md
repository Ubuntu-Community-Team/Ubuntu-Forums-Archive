### Post by Frogs Hair on 2011-06-25
The installation tab takes you to the home directory because if had downloaded a theme for installation and it would have been saved there in a a file of your choice . You could first try restarting the computer and if that doesn't work try the command below followed by _restarting the computer_ ```
killall gnome-settings-daemon
```
