### Post by Krytarik on 2011-09-10
So, Jaunty 9.04, eh!? Not supported anymore since Oktober last year. ;-)

[http://en.wikipedia.org/wiki/List_of_Ubuntu_releases#Ubuntu_9.04_.28Jaunty_Jackalope.29](http://en.wikipedia.org/wiki/List_of_Ubuntu_releases#Ubuntu_9.04_.28Jaunty_Jackalope.29)

So, either consider upgrading or, if needed for some reason, try modifying your sources in "/etc/apt/sources.list" following this pattern:
```
deb http://old-releases.ubuntu.com/ubuntu/ jaunty main restricted universe multiverse
deb http://old-releases.ubuntu.com/ubuntu/ jaunty-updates main restricted universe multiverse
deb http://old-releases.ubuntu.com/ubuntu/ jaunty-security main restricted universe multiverse
```
Greetings.
