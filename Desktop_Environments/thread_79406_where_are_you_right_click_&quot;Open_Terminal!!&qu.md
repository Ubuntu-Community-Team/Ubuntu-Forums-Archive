### Post by Miguel on 2005-10-20
If you open terminals so frecuently, would you mind if I make a suggestion?

Under System->Preferences, the keyboard (IIRC) entry allowed you to map shortcuts. If you assign, let's say Ctrl+Alt+t to "Run in terminal", pressing that combination will run your default terminal (it is set in System-> Preferences-> Favourite Applications).

It is soo comfortable... Just remember that if you just changed from a virtual desktop, you will need to release and press Ctrl and Alt again.
