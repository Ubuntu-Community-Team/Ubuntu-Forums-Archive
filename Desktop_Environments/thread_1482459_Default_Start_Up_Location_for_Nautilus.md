### Post by cforput on 2010-05-13
Is there a way to automatically have Nautilus start up in a specific folder? I run it out of the desktop not out of terminal.

---

### Post by mikewhatever on 2010-05-13
Yes, you can create a launcher on the desktop with 'nautilus /path-to-folder' in the 'command' field, for example, 'nautilus Desktop'.

---
