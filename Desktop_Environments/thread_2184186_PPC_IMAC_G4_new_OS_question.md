### Post by Lars Noodén on 2013-10-28
Lubuntu supports PPC still.  You can get a disc image from the Lubuntu.net site.  Watch out for the size, though.  13.10 is oversized and won't fit on a CD RW, you'll need DVD RW or boot from USB stick.  

A nice thing about it is that the Desktop Environment, LXDE, is a bit lighter that others and the default applications are also chosen for being lighter weight.  And, of course, still being Ubuntu, you can add in whichever applications you want to complete your system.  There are a few missing from PPC, like Chromium, but for the most part everything is available.

### Post by 3rdalbum on 2013-10-28
Flash and Java are a no-go on PPC machines unfortunately. They are available for Ubuntu/x86 and MacOS/PPC but not Ubuntu/PPC.

When you get to the login screen, click the little Ubuntu logo near where you type your password, and then select the Unity 2D option. Log in and see if that helps with the CPU use. If it doesn't, then you might have a program that's causing the high CPU use; you'll need to open System Monitor or type "top" in a terminal to find out which one it is, and then kill it.
