### Post by jetpeach on 2008-06-03
CTRL-F2/F1 just switch to the "second desktop" and "first desktop". if you open something in one of those desktops and go to the other it will look like you've hidden the windows, but then once you've opened something in both desktops you won't have the ability to see the desktop again...

the true replacement is supposed to be crtl-F12, which works ok for me. i just wish they would add an alternate hotkey by default that was "windowskey-D", since that is the default for the equivalent function in windows (although obviously that isn't justification, it is also much more convenient than crtl-F12, which takes two hands...)
