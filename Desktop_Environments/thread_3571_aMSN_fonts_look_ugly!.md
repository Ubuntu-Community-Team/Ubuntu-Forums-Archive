### Post by wallijonn on 2004-11-07
If it is the file menu fonts:

Computer -> Desktop Prefences -> Fonts -> Application Font -> Sans (Bold) -> 12

If it is within the application, then there is probably a 'Use my fonts" checkbox / section in its preferences.
