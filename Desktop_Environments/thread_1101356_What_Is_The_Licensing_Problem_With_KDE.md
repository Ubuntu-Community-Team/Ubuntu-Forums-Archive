### Post by jacksaff on 2009-03-20
KDE has always been open source. The underlying qt libraries were not but are now.
It's  been a dead issue for linux users for several years now as qt made a commitment to open up their code if no new releases were made within a certain time period. Qt remained commercial on other platforms though.
In the last few months all of the qt code has been released under open source licenses for all platforms, so it's no longer an issue for anyone.
