recently I have noticed that root apps like Synaptic or GParted started to use larger font size than regular ones. At the same time, they do respect my GTK theme. How do I permanently fix the fonts?

Thanks in advance!

### Post by omshivaprakash on 2008-08-29
* Run the command 'gnome-appearance-properties'
    * Go to the 'Fonts' Tab
    * Click on the 'Details' Button at the bottom
    * Change the Resolution to 96 Dots per Inch(DPI)

That should solve your problem.
