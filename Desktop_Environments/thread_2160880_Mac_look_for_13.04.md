### Post by BreezyBrooke on 2013-07-08
Sure! That is easy!

First install gnome-session-fallback, then install indicator-applet-appmenu, and finally cairo-dock.
Like this: sudo apt-get install gnome-session fallback indicator-applet-appmenu cairo-dock deskbar-applet gnome-themes-standard

logout and log back into the gnome session with effects by clicking the ubuntu icon near the login box. After log in, ctr or alt rightclick the bottom pannel and remove it, then ctr or alt rightclick the top panel and add the global menu applet to the top panel. If you want, you can remove the the standard gnome menus applet and replace them with the unified gnome menu (the one that just gives you an icon the open everything). Finally start cairo dock. There you go.
