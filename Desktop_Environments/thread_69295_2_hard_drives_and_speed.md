The only thing you would want to check for, using your drives cabled like this...

Do you have an IDE CD-ROM drive (I assume you do).  If so, it makes more sense for both hard drives to run off the same IDE cable... and then put your CD-ROM drive(s) on the other separate IDE cable.  Running a hard drive (which runs faster than a CD ROM drive), on the same IDE cable as a CD-ROM drive, limits performance somewhat.
