### Post by coffeecat on 2008-07-12
Right-click on the panel and choose 'Custom Application Launcher' and click Add. In Launcher Properties, type:

Name: Computer (or, whatever you want)
Command: nautilus computer:///
Comment: (whatever you want)

As soon as you have typed 'nautilus' into the command line, the icon automatically changes to the Nautilus one. Now click on OK and you have a panel launcher which does exactly what Places > Computer does.
