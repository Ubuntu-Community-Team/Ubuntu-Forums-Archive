### Post by BQAggie2006 on 2009-10-09
All LTS really means is that updates will be available for the release longer than the regular releases. An LTS release will receive updates for 3 years for the desktop edition and 5 years for the server edition. A regular release will receive updates for 18 months.

You can try running Xubuntu, it's pretty much the same except it runs Xfce in place of Gnome, so it's slightly less of a resource hog.
