### Post by vanadium on 2021-04-18
This appears to be an issue with the specific application you use, or your application taking over control of the wallpaper. desktopfolders is not installed in Ubuntu by default. I recommend to try the default build-in Gnome Shell extension "Desktop Icons", or alternatively, install and use the improved "Desktop Icons NG" extension that Ubuntu will adopt by default in the next release 21.04.
