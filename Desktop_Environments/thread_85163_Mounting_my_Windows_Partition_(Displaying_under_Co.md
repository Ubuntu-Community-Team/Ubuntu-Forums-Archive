### Post by mrmcctt on 2005-11-01
Well mine does that now.  Not sure why yours does not. I haven't done anything special and my CD-ROM, Windows Mount and File System are displayed.

If you want to get icons off the desktop go to the Main Menu>System Tools>Configuration Editor>Apps>Nautilus>Desktop and look at the right side of the screen.  Untick any boxes for any Icons you don't want on the desktop.

As for doing things to the main menu, you might try SMEG.  Someone with more smarts on what SMEG does and how it does it will hopefully jump in and explain that if you need it.

Hope this helps you out some.
