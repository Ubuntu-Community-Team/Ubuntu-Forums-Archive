### Post by m_duck on 2011-05-28
There are equivalent menu programs that you can set as keybinds/mousebinds using CCSM to get effectively the same thing. The one I have used before is called compiz-deskmenu, and there seems to be another (fork?) called compiz-boxmenu. The former I have only used in Arch using the AUR so I've no idea how one would go about installing it in Ubuntu, but I'm sure there would be a way.

Arch AUR Pages:
[compiz-deskmenu]("http://aur.archlinux.org/packages.php?ID=15953")
[compiz-boxmenu]("http://aur.archlinux.org/packages.php?ID=46428")
