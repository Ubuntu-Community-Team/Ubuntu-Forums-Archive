Hi. Is there a way to get a 'favorite', ie frequently used, directory to appear on the left when I am uploading files etc?

### Post by mikewhatever on 2021-09-07
You can add any folder to the left pannel as a bookmark. Either use CTRL+d or "Bookmarks" from the top menu. 
After the first addition, a new bookmark section will be added to the left pannel, and then it is possible to drag and drop folders.

It appears, you can even add bokmarks from that popup window, just rightclick any folder, and select "+Add to bookmarks".
