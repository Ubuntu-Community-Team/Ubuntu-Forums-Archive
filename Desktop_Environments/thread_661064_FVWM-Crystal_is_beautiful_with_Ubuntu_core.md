I have setup a FVWM-Crystal system using the ubuntu core download at [https://help.ubuntu.com/community/Installation/MinimalCD]("https://help.ubuntu.com/community/Installation/MinimalCD")

I Downloaded 

32-bit PC (x86)
Ubuntu 7.10 "Gutsy Gibbon" Minimal CD 9.1MB

Installed text mode, i didnt install any additional software

NOTE: when its booting up it might not seem to be working or show a login, hit enter when you think its done and it will ask for you to login, and enter the password you setup, the password will not show but it is being typed

rebooted into the system logged in, and used this tutorial [https://help.ubuntu.com/community/FVWM-Crystal]("https://help.ubuntu.com/community/FVWM-Crystal") to get it to work perfectly



I additionally installed synaptic and once it was installed i used synaptic to install other programs in a list some codecs and firefox, it was beautiful, and easy to setup using that tutorial if you fallow it and read all of it you shouldnt have a problem with it enjoy!

EDIT:

### Post by ThomasAdam on 2008-05-01
> **x1a4 said:**
> Not FVWM, FVWM-Crystal.  While its window manager is originally based on FVWM, FVWM-Crystal is a whole new environment.

No.  Grr.  Why do people have such a hard time understanding such a simple concept?

FVWM-Crystal *is* FVWM.  I'll say it again for effect this time in capitals:

FVWM-CRYSTAL IS FVWM.

All FVWM-Crystal does is provide some *configuration* files *FOR* FVWM.  It's FVWM in the same way my bare-bones ~/.fvwm2rc config *is* FVWM.  In the same way that fvwm-themes is also FVWM.

It's all FVWM.  FVWM-Crystal is not "based" on anything.

-- Thomas Adam
