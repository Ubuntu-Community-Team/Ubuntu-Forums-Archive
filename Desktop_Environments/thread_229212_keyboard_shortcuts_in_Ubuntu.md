### Post by OpsVentus on 2006-08-04
You could use gconf-editor, but you should be able to give multi-key shortcuts in at "keyboard shortcuts".
But if you want to set 'win+e', you cannot have only 'win' in an other entry. Becaus it will launch that command as soon as you press 'win'.
