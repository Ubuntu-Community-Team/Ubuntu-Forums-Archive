### Post by SpotHT on 2010-06-13
You need to install Emerald. Press Ctrl+Alt+T to run the terminal, then execute this command:

          **sudo apt-get install emerald libemeraldengine0**


To download themes for Emerald, click [here]("http://compiz-themes.org/index.php?xcontentmode=103").

 To open Emerald go to** System >  Preferences > Emerald Theme Manager **and import themes you download[I].
[/I]
