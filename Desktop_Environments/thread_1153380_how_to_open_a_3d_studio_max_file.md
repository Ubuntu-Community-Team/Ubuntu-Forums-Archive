### Post by hoagie on 2009-05-08
I would recommend you using [blender]("http://www.blender.org/"), it is an open source 3D content creation suite. You can install it by going to Applications>Add/Remove... and then make sure that on the drop-down menu at the top of the window the option show all open source applications or show all available applications is selected. Then in the search field, in the upper right side of the window, type blender. Afterwards click on the Blender (windowed) check box (it should be the first option) and then select apply. After the installation is complete launch blender by going to Applicatons>Graphics>Blender and select File and then Open and finally select the file you want to open.

This should do the trick.
