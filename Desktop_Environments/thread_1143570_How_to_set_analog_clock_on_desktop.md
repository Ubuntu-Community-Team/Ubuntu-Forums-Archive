However any naive ubuntu user may install analog clock,calculator,calender etc in the following way:
(1) Download [SIZE="5"]screenlets_0.1.2-0~getdeb1_all.deb 
[/SIZE] from [here]("http://cesium.di.uminho.pt/pub/getdeb/ubuntu/hardy/sc/").
(2) install it by double clicking in it.
(3) Go to Application>Accessories>Screenlets
(4) select the analog clock(or calender or calculator)
(5) click "Launch/Add"
Thats it...
