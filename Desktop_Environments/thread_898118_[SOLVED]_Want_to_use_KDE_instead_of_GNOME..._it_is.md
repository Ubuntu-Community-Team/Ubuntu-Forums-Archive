### Post by maniheer on 2008-08-23
Which one do you want

KDE 3.5 or KDE 4

for KDE 3.5

Three Options (biggest first, smallest last)

```

Three Options (biggest first, smallest last)

sudo apt-get install kde

or

sudo apt-get install kubuntu-desktop

or

sudo apt-get install kde-core

```

for KDE 4 (biggest first, smallest last)

```

sudo apt-get install kde4

or

sudo apt-get install kubuntu-kde4-desktop

or

sudo apt-get install kde4-core

```

KDE 3.5 = Less features but more stable

KDE 4 = More features less stable (will be soon though)

Hope I helped
