### Post by Madpilot on 2005-12-11
1. [https://wiki.ubuntu.com//FontInstallHowto](https://wiki.ubuntu.com//FontInstallHowto) - short version is create a directory called .fonts and put your font files there. (note that that period is part of the name, and it'll be hidden by default - Ctrl+H shows hidden files by in Nautilus)

2. I don't think Rythymbox includes tag editing, but there are several apps just for retagging music available. I like EasyTag myself - search Synaptic for it.
