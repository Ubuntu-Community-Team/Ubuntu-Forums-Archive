### Post by grahammechanical on 2014-08-17
I think that there are two parts to what you want.

1) the automatic loading of applications at start up. I have not tried this at all but 

[https://help.ubuntu.com/community/AddingProgramToSessionStartup](https://help.ubuntu.com/community/AddingProgramToSessionStartup)

2) the loading of applications into specific workspaces. From the above wiki page

> [COLOR=#333333][FONT=Ubuntu]Using the way described above, all the applications start in one workspace, maximized and so on. If you wished for example to start an application in workspace 2 minimized, you would have to do it manually - unless you install devilspie - detailed howto can be found on the forums:[/FONT][/COLOR][http://ubuntuforums.org/showthread.php?t=75749t](http://ubuntuforums.org/showthread.php?t=75749t)

[http://ubuntuforums.org/showthread.php?t=75749t](http://ubuntuforums.org/showthread.php?t=75749t)

The 14.04 software centre has devilspie and gDevilspie, a GUI front end that helps you configure devilspie.

[http://www.burtonini.com/wordpress/tag/devilspie/](http://www.burtonini.com/wordpress/tag/devilspie/)

Do not forget that we can reduce the size of the icons in the launcher and that when there are many icons in the launcher those at the bottom fold up on one another and the launcher scrolls up to revel the icons. We can have a lot of icons in the launcher. I have 31 icons in my launcher. Mostly icons for partitions.
