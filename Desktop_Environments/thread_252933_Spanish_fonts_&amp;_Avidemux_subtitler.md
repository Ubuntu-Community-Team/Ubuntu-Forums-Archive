### Post by maol62 on 2006-10-29
Make sure that you have the correct path to the spanish font, that you want to use, in the subtitler view. I use arial as font and I have to change the default location of fonts in subtitler to /usr/share/fonts/truetype/msttcorefonts/arial.ttf to get it to work. When the path is changed to the correct one, you should be able to se the subtitles when using the preview function i Avidemux. Good luck!
