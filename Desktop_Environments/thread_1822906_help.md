### Post by 3Miro on 2011-08-11
Splash screen doesn't work with proprietary drivers. The issue is that the screen has to show before the driver can be loaded and without a driver you cannot show a screen.

I don't think there is a solution to this one. You can technically get a screen with a weak default driver, but it will be very ugly and not worth the trouble.
