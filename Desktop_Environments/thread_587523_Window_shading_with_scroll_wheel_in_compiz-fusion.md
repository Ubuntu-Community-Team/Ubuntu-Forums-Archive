### Post by holomorph on 2007-11-24
I found the solution:

Set /apps/gwd/mouse_wheel_action to "shade" in gconf-editor

and it works.  There is an equivalent setting for kde in the kde control center(or so I read).

Really unfortunate that this is so hidden (I would think it would be a good default, since it's not likely to bother people that would not use it, and be very handy for those who do)
