### Post by starcraft.man on 2009-07-10
Open Nautilus Edit Menu > Preferences > Show Hidden and Backup files (check).

I don't advise this by default. Your home folder is filled with many folders with config files that will clutter your view needlessly. When you need to edit hidden folders, I'd just chit ctrl + H, the key combo to show hidden files. Hit again to hide em.
