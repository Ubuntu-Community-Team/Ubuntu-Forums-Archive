### Post by darrenadams on 2005-03-24
I haven't used it myself, but installing the printing plugin seems to be a relatively painless task:

[list]
[*]Right-click on the Tomboy Icon and select the "Install Plugins..." item
[*]In the window that appears, double-click on the "Uninstalled Plugins" folder. You should see a file in there called PrintNotes.dll
[*]Copy this file over from the "Uninstalled Plugins" to the "Plugins" folder
[/list] 

Once that's done, open a Tomboy note, and you should see a Print button on the toolbar which you can use to print your notes.
