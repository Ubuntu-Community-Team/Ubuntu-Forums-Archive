### Post by mike998 on 2009-11-30
If you can log into CLI, try using the aptitude command to access the package manager...  That might make it easier to remove the GTK you installed.

Aptitude is a ncurses based program that is usable from command line.

I've had a google around, and the only other thing that's been suggested is possibly you're running out of disk space.  The command below will give you a good indication.```
du -h
```

The other thing that this may be to is a full /tmp directory.  you COULD try the command below.  
NOTE: This is a superuser command and CAN be VERY dangerous if used incorrectly.```
sudo rm -rf /tmp
```

For future reference, it might be a good idea to dedicate a partition on your drive to your /home directory (make it big!).  That way, if this DOES happen again, you can re-install the whole system without feeling (too) bad about it.
