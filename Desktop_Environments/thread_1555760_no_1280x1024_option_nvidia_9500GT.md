### Post by realzippy on 2010-08-18
If you know that your display can run the "1280x1024" (pretty likely),
you *can "save to X configuration file"* with "1024x768" resolution,then
edit xorg.conf:


from 

*Option         "metamodes" "1024x768 +0+0; nvidia-auto-select +0+0"*

to

[I]Option         "metamodes" "1280x1024 +0+0; nvidia-auto-select +0+0"
[/I]

restart X required.
Feel free to ask if not sure what to do..

### Post by sylosis on 2010-08-19
i downloaded phoenix edid designer which looked alot better than softMCCS, exported the file which is .raw instead of .bin (edid.raw) to my X11 directory

i then used this guide [http://ubuntuforums.org/archive/index.php/t-316985.html]("http://ubuntuforums.org/archive/index.php/t-316985.html")

and changed this:

```
#Option "AddARGBGLXVisuals" "true" #NVidia specific, uncomment if needed
#Option "UseDisplayDevice" "DFP-0" #Uncomment and modify if needed
Option "CustomEDID" "DFP-0:/etc/X11/nvnew.raw"
```

to this:

```
Option "AddARGBGLXVisuals" "true"
Option "UseDisplayDevice" "CRT-1"
Option "CustomEDID" "CRT-1:/etc/X11/edid.raw"
```

and it worked :D im now running 1280x1024 and have a LOT more options for resolution.

thanks for all the help man, couldn't have done it without you.
