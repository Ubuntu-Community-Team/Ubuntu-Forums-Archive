### Post by Jucato on 2006-02-22
The only reason you don't see tabs in those screenshots are because he has only one tab opened. This behavior can be set in Settings > Configure Konqueror > Web Behavior > Check "Hide the Tab bar when only one tab is opened"

Though I have to say that I find it strange that you don't want tabs when everybody else just hates no tabs. And tabbed browsing is more effecient (system-resources-wise) than having multiple windows opened.
