### Post by Ramses de Norre on 2006-04-12
I don't think you will be able to get two working ip adresses from your isp.
A router has only one outgoing ip adress and creates an internal network behind himself so that every machine behind the router has his own ip, though the isp sees only one ip.
With 2 modems the isp will see two ip's which probably isn't allowed.
