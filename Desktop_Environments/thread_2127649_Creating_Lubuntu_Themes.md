### Post by cortman on 2013-03-20
Lubuntu uses LXDE (basically just a panel, as far as themes go), Openbox, and GTK. Each of these have their own theme. Info on the LXPanel is available [here]("http://wiki.lxde.org/en/LXPanel"). It seems pretty simple and straightforward.
For theming Openbox and GTK I usually find it easiest to tweak an existing theme. You can do this by making a copy of a theme before you install it and editing it, or by just changing the config file of the current theme. I usually tweak the colors to match between GTK and the window theme.
Hope this helps.
