### Post by jgcamp99 on 2006-07-08
Are you sure you got all the gstreamer files. I didn't follow an instruction process. But I was able to use the Application add/Remove method to get Rhythmbox and had no issue with playing mp3's except that the ones I had on my ntfs couldn't be opened on the ntfs partition. I had to copy them to the ext3 Linux partition and they played fine as I recall, which is strange, but something I accepted. The gstreamer files and related dependencies, I used Synaptics Package Manager to get. Those then made the default movie and mp3 players that Ubuntu installs with work. I also used Aplications Add/Remove to get Xine and XMMS. All of them work properly for what I've tried playback for thus far now.
