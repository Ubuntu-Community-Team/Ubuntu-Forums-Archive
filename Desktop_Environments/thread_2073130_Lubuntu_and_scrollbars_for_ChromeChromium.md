### Post by vasa1 on 2012-10-19
You can get the extension from here:
[https://chrome.google.com/webstore/detail/lubuntu-scrollbars/hemkkjpjknkkndhconammdhlhdkjclim](https://chrome.google.com/webstore/detail/lubuntu-scrollbars/hemkkjpjknkkndhconammdhlhdkjclim)

Looks pretty and should work for other desktops as well, IMO.

If one doesn't like the idea of using an extension, it's got nothing evil in it: three icons, a tiny text file (manifest.json) and the meat which is a .css file. So people concerned about their "privacy" needn't worry at all. The disclaimer that appears while installing the extension is a standard one.
