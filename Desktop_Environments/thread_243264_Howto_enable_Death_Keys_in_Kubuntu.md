### Post by gunksta on 2006-08-31
OK, for starters, I don't know what the heck death keys are, but changing the Keyboard Language is easy.

Menu -> System Settings -> Regional & Accessibility (Top Row) -> Keyboard Layouts.

They will all be greyed out. You need to activate KDE's regional kb layouts by checking "Enable Keyboard Layouts".

If this isn't what you are looking for, explain what a Death Key is.

--andy
