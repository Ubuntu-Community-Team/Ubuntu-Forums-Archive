### Post by wpshooter on 2006-10-01
> **Turin Turambar said:**
> While playing one game in Dosbox, screensaver activates and blanks my screen. It does not detect mouse or keyboard keys. Is it possible to switch off or do something with the SS?

Are you sure this is screen saver.  

Sounds more like power management (display) is kicking in.

Go to **System** - **Preferences** - **Power Managagement** and see what the **display slider** is set on.
