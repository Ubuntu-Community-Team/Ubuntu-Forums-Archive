### Post by clappboard on 2011-10-22
I don't know if this is what you want, but if you install CompizConfig Settings Manager, you can set it so that the icons only light up coloured when you have them opened (comme dans Windows 7).

Install it like this:
```
sudo apt-get install compizconfig-settings-manager
```

Then hit Alt+F2 and type 'ccsm' without the quotes.  Go to 'Unity Plugin' and it's there somewhere...  I'm sorry I don't use CCSM :P
