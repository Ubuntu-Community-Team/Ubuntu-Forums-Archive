### Post by gmargo on 2011-02-16
System -> Preferences -> Keyboard Shortcuts.
Scroll down to "Window Management"
"Close Window" entry is set to alt-F4.
Click on it and hit backspace to clear.

---

### Post by mcduck on 2011-02-16
If you have Visual Effects enabled, you need to configure this in Compiz settings instead of Gnome's settings. Install CompizConfig Settings Manager (if you haven't already got it), and then under General options, on the Key bindings-tab, disable the "Close Window"-shortcut.
