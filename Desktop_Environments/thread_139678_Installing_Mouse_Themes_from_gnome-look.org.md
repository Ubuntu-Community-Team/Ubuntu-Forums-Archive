### Post by alsac on 2006-03-05
Simplest way is to download the file that contains the mouse theme you like - probably a tarball (filename.tar.bz2). Remember where you downloaded it (I created a "dwnl" dir in my /home/alsac).  
 - open "Theme Preferences" from the gnome menu, 
 - open "File Browser" in gnome menu, 
 - find the file you downloaded in the File Browser
 - drag the file to the open "theme preferences" window and click install when prompted.

then ...
 - open "Mouse" from gnome menu
 - select "cursors" tab
 - select the cursor theme that you just installed.
 - changes take effect immediately.

That's how I'd do it. Fool-proof. This way you don't worry about any commands to extract the tarball, etc..
