### Post by moopere on 2007-12-07
These days Xubuntu provides a different experience but is not a lot lighter than Ubuntu proper (and uses many gnome apps and libs/etc).  For truly RAM challenged machines you would be a lot better off with fluxbuntu or start with server ubuntu (no desktop) and install icewm.

I've got a legacy test machine here, K6-300 with 128MB and its really slow with Ubuntu, Kubuntu or Xubuntu (interestingly, runs ok with Win XP).  Works great with Icewm.

Pretty sure that E17 has a very light footprint too.
