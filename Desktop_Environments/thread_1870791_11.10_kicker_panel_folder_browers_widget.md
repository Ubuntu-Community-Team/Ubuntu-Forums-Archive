### Post by toasterboy1 on 2011-10-27
So I did a clean install and realized the little icon next to the application launcher that pulls up your home directory and you can navigate around the folders is not there. Anyone know what this widget was called? Or even better how to get it in 11.10? I tried folder view but it is not the same.

---

### Post by toasterboy1 on 2011-12-28
Finally found my answer. The widget is called QuickAccess Browser and it was not installed.
```
sudo apt-get install plasma-widget-quickaccess
```
Then just added it to the task bar. Thanks to kubicle on kubuntuforums.net
