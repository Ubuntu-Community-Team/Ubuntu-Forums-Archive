 Hi globalspace

I'm afraid this is the new behaviour, as you can see in /usr/share/doc/gnome-applets/changelog.Debian.gz:

```
    - stickynotes always float on top (required since they no longer appear in
      the Alt-Tab list with Metacity).
```

However, don't despair. There is another project called xpad which does what you want. You can find it in the universe repository. Homepage: [http://xpad.sf.net](http://xpad.sf.net) ; I have been toying with it for a short while and seems OK. Notes don't stay on top, and there is a tray icon and all :P

Hope this helps, andvaranaut
