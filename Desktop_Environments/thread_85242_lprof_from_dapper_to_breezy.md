### Post by malex on 2005-11-23
I have brand new lprof 1.11.0 packages for both hoary and breezy in my repository. If you want details please read [http://wiki.scribus.net/index.php/Scribus_on_Debian_GNU/Linux](http://wiki.scribus.net/index.php/Scribus_on_Debian_GNU/Linux). Note that lprof is only in the debian.tagancha.org repository and _not_ in the scribus.net one.

If someone wants the source package then use my "unstable" deb-src line. This package hasn't been uploaded into Debian, yet, I just finished packaging this new version of lprof, so it won't be in dapper either for a while I guess. A note of caution: 1.11.0 is a developmental snapshot and could have bugs. Please report them at [http://sourceforge.net/tracker/?group_id=146038&atid=764370](http://sourceforge.net/tracker/?group_id=146038&atid=764370)
