I'm running Breezy with KDE 3.4.3. I've noticed that Konqueror has no Window menu any more, and there doesn't seem to be any way to split the window into 2 or more panes. I used to use this feature a lot for FTP transfers etc, and now I have to use 2 Konqueror windows which is rather annoying.

James

### Post by Jucato on 2006-02-20
A couple of ways:

1. Ctrl+Shift+L (split left-right) or Ctrl+Shift+T (split top-bottom). You can also configure your shortcuts to your liking (Settings > Configure Shortcuts)

2. Right-click on the status bar at the bottom of the window. it will pop-up a menu for splitting/closing windows.

3. Put a button for these actions on the toolbar (Settings > Configure Toolbar).

Have a nice day.
