### Post by Matakoo on 2007-10-17
It's easy to install, so if KDE is what you want, that's what you should get. And it's equally easy to remove afterwards should you want to. As far as KDE 4 goes, I would be most surprised if it will not become available as an upgrade sooner or later - and presumably, the installer would remove compiz at that time if it's installed as well.

And if Ubuntu, for whatever reason, should decide not to a third-party repository in all likelihood will so either way it will be easy to install. Compiling something from scratch is VERY rarely needed. I've never needed to since starting to use Ubuntu (although I've compiled a few things myself anyway...mostly to see if I could, and to get updated versions of some programs a bit quicker).

With that said, do like this and you should get Compiz up and running.

1. Install Kubuntu, obviously. If you have no install at this time, wait a couple of days to get the release version instead of the release-candidate. I will here assume you know how to do that, otherwise there are good guides available all over the net.
2. If you have a ATI or nVidia card, you will get a small icon in the lower-right corner at first login. Since you want to use Compiz, you need to click on that icon to bring up the restricted manager. In that window, you need to activate the proprietary graphics-driver. After that is done, reboot the computer.
3. Once back, check if there are any updates that need to be installed. That's also indicated by an icon in the lower-right corner. Click the icon, fetch and install the updates. Rather like Windows update.
3. Finally, time to install compiz. In the K menu, you will see a submenu called System. In there, you will see a program called Adept Manager. Choose that one. Once the app has loaded, you will see a search field on the left hand side. Type compiz in it and you will get a list of the packages that are related to Compiz. There's quite a few of them but you don't need all. The ones you need are called:
Compiz
Compiz-kde
Compiz-settings-manager
Emerald

The rest are extra plugins or things the packages above selects automatically.

Right click on each and select Install. Once you've done that, click on preview changes to see if extra libs/or packages have been selected for you. And finally on Apply changes. Now the selected packages will be downloaded and installed for you. After that has completed, quit adept manager.

4. Open up a terminal. Can also be found in the same menu as the Adept Manager. Run this command:

```

/usr/bin/compiz.real
```Close the terminal. Now you should have a new menu in the start menu, settings. In it you will find a program called Advanced desktop effects settings. Config it to your heart's content, and from now on it should work.
