### Post by Bagboy23 on 2007-02-11
1. Make sure you are using the ATI Drivers. - Do a fglrxinfo.

2. Make sure you have the correct Beryl apps and XGL installed. Just check synaptic and check em all if you're unsure.

3. Make sure you have XGL enabled for that session.Check your session file (startxgl.sh) and make sure you have all permissions.

If any of the above doesn't work, do two things;

1. Post any error messages you get.

2. Post the guide you used to get XGL to and Beryl to run.
