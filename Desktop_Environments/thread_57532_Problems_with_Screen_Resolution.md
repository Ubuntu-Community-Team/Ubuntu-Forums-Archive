### Post by AnythingBut on 2005-08-16
It looks like your backup is valid.  To put it back use:

sudo mv /etc/X11/xorg.conf_backup /etc/X11/xorg.conf

Then restart X, as you have before.  Before you lost xorg.conf, were you stuck at 640x480?  If so, that's a whole other problem to fix.  Using the xorg.conf_backup you should get  1024x768.  If not, post /var/log/Xorg.0.log which may contain some useful debugging information.
