### Post by aged hippy on 2009-06-21
I'm not too sure about installing a down-loaded theme (i'm on Hardy 8.04 (KDE 3.5) at the moment), but i think that you need to navigate to *->System Settings ->Appearance* and hit the *Install theme* button, then navigate to where you downloaded it.

An easier way is to right-click on the (not)desktop, select *->Appearance ->Get new themes,* it takes you to KDE-look.org, and you simply hit the *Install* button to get one you like. :)

### Post by Bachstelze on 2009-06-21
Open the System Settings panel. In the Advanced tab, select the Login Manager item. You will be prompted for your password and a window will open. Under the Theme tab, click Install new theme and browse for the archive you downloaded from kde-look.org.
