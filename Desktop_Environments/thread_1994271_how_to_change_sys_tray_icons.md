### Post by Dennis N on 2012-06-02
The easy way is to select or install a new icon theme. If you are looking for colorful icons (not the default monochrome ones) install gnome-colors, which includes several sets of different colored icons. gnome-colors is not in the Lubuntu Software center, but can be installed from Synaptic Package manager (or use apt-get install). Select the new icons from Preferences > Customized Look and Feel dialog.

You probably have to log out and back in for the new icons to appear.
