### Post by RAdams on 2006-06-30
Hi. I was wondering if anyone had found a way to keep 3ddesk up to date, so that when I activate it, it has captured the latest images of my desktops. Otherwise, it becomes rather useless.

I did see this in the FAQ:

3. The images sometimes do not match my real desktop.

Don't use any pagers other then 3ddesktop.

3ddesktop has to periodically do a screen capture of your current desktop to maintain updated images. If you always use 3ddesktop to switch desktops no problem, but if you use the default screen switcher then you may switch before 3ddesktop gets the most recent update of you previous desktop so it could become out of sync.

### Post by Jucato on 2006-06-30
I don't think it's possible yet, because when 3D Desktop acquires screenshots of the virtual desktops/workspaces, it goes through them 1 by 1, and the process happens visibly, not in the background. So scheduling it to take snapshots of the desktops every few minutes will actually be annoying and intrusive.

I'm not sure if 3D Desktop is currently being maintained. It might be a feature that could be added in the future.
