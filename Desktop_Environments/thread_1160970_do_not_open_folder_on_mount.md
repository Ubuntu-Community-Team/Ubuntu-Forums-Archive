### Post by MyS Tyle on 2009-05-16
Hi,

When I mount an USB hard drive in ubuntu, ubuntu automatically opens the root folder of that drive by default. It really annoys me, so I want to disable that function. Does anyone knows how to do that?

grtz

---

### Post by kpkeerthi on 2009-05-16
Click on Places -> Home -> Edit -> Preferences -> Media Preferences and you change that behaviour there.

---

### Post by Keith Hedger on 2009-05-16
From nautilus goto Edit->Preferences-Media and uncheck "Browse media when inserted"
