How do I get Gnome to run .xinitrc upon startup? I've got some screensaver settings on there (xset -s 0 0 -dpms) that I need to run upon startup, but I have to run it manually for some reason. I installed the Gnome environment through aptitude.

### Post by yabbadabbadont on 2007-02-19
To the best of my knowledge (and past experience ;)), Gnome does not read the .xinitrc file.  It might read .xsession, but I can't test it as I no longer use Gnome (nor have it installed).  You could always add those commands to your autostart program list using the gnome session manager found in System->Settings->Sessions.  I believe it is on the last tab.

### Post by ardchoille42 on 2007-02-19
Is you use startx to get X running, then ~/.xinitrc is read. If you use login via gdm, then ~/.xsession is read. However, there is a note in the openbox docs that said gnome was switching to some other type of setup.. which may be why ~/.xinitrc doesn't work for some folks.
