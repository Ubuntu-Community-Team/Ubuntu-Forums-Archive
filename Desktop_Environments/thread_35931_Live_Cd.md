### Post by britbrian on 2005-05-22
tikal26

Archie 0.4.1 is a  livecd that's based on Arch 0.7.  Its a one man project and the Xfce ver is about 325MB. See   [http://www.archlinux.org/](http://www.archlinux.org/)

Also there is Xfld live-demo at  [http://www.xfld.org/Xfld/en/index.html](http://www.xfld.org/Xfld/en/index.html) which IIRC is about 650MB.

BeatrIX is an Ubuntu based Gnome livecd thats only 182MB and it might suit your sisters older PC. It looks very clean, has OpenOffice 1.1.4, Firefox, Evolution, Gaim and runs well on my slowest 600MHz PC. See  [http://www.watsky.net/](http://www.watsky.net/)


Beware if you set the Xfce panel to autohide, it sometimes does'nt unhide and needs restarting. Plugins are available to add a tray  into the main panel but the original tray remains somewhat visible even if minimised & underneath.

If you want to remaster your own XFce live/install cd with a simple script, you could use PCLinux which is KDE based & uses apt/Synaptic. Adding Xfce adds less than 20MB to install (6MB downloads). The mklivecd script compresses about 1.8GB down to about 630MB. I also removed about 100MB of stuff so I could add Gnome and more office apps to make a great showcase multi desktop live/install CD.

hope this helps
