### Post by lovinglinux on 2009-10-21
Boot from a LiveCD and delete /etc/X11/xorg.conf then reboot. This should allow you to boot normally. Then use jockey ( System >> Administration >> Hardware Drivers)  to install nvidia drivers. If it doesn't complete the driver installation and do not ask you for a reboot, then open Synaptic and verify if nvidia-185-kernel-source, nvidia-185-libvdpau, nvidia-185-modaliases, nvidia-kernel-common and  xserver-xorg-video-nv are installed. If not, install them from Synaptic and reboot.
