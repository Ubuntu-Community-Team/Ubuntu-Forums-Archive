### Post by mgmiller on 2006-09-19
You could try running System > Administration > Synaptic Package Manager and in the settings > repositories area, try unchecking everything except for the security repos.  (My system has 4 of them.) 
This might still have kernel updates enabled, because, sometimes the kernel updates fix/add security features.

Also, on the internet updates tab in the same area, there is an option to have it automatically download and install security updates without asking.  I personally don't like to do that, because I prefer to read about all the updates before I commit to them.


Good Luck
