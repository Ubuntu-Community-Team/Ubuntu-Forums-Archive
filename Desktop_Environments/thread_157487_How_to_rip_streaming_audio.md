### Post by pranav on 2006-04-09
This has worked out for the music from [www.magnatune.com](www.magnatune.com) (which is a lot of music btw)

1. I right clicked on the file in xmms

2. Selected File info

3. Copied the url

4. Pasted it in a download manager (KGet in my case) to start a new dwld

This might however not work on all the streams.
