### Post by ow50 on 2005-04-28
You'll just have to let it finish hashing your files. It'll take maybe 10-30 minutes depending on how much you're sharing. After that your cpu usage will return to normal. Hashing only needs to be done once for each file. A list of files already hashed seems to be at ~/.dc++/HashIndex.xml, if you're interested in how far it got.
