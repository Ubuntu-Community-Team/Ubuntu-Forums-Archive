### Post by DavidTangye on 2008-08-27
> Download the Icon theme tarball.A tarball is a file normally ending in either tar.gz or .tgz, containing a set of files packed inside it. 'tar' is short for 'tape archiver', but the program is used to archive and compress to a file on disk rather than to tape these days. The archive file it produces is commonly known as a tarball, as it is a set of files 'rolled up into one', for archiving or copying as a unit, between machines etc.

Then he was showing which commands to run from a Terminal. Open a Terminal from menu 'Applications', 'Accessories', 'Terminal'. The commands with my description of what each command does, added after each '#' below are:


[LIST]
[*] cp tarball.tar.gz /home/username/.icons # Copies the downloaded tar file to /home/username/.icons directory. However you must change 'username' to your actual username.
[*]    cd /home/username/.icons #Change into that directory
[*]    tar xfz tarball.tar.gz # This command extracts the icon files from out of the tarfile. Now you can ...
[*] 'Goto **Computer > Desktop Preferences > Themes** to change to the new Icon theme.'
[/LIST]
