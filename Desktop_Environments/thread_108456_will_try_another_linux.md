I'd probably try a clean Ubuntu install but depending what you want to do in Linux there is lots of choice.  
If you like to experiment yourself and are in no rush to have a fully configured system then Gentoo is good if you don't mind editing text files for configuration.  .  
If you want everything to just work with minimal graphical tools to configure but the conveniebnce of lots of packages but don't care about the latest packages Debian stable is unbeatable.  If you want an up and running Debian unstable you hardly need to add anything to then try kanotix.  

If you don't care about learning linux but want a functioning system then suse and mandriva are good.  The downside is they have weird config tools that stop you actually understanding what is happening in the background.. but they usually work ..(like Win2K)
