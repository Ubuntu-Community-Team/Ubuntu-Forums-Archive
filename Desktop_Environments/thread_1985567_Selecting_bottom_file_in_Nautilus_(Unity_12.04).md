### Post by dreamquartz on 2012-05-23
It is a challenge to select the last file in a folder in Nautilus under Unity in 12.04.
The reason is the horizontal slider that will show if you move the cursor to the bottom of the screen.
If you want to place the cursor anywhere else, the slider will quickly follow.

I do not know if it is a bug or if there is a way to avoid this issue.

### Post by Col-1023 on 2012-09-25
I was having the same problem with folders in list view.

I solved it by selecting view and selecting "statusbar".

This creates a 'Statusbar" at teh bottom of the folder view, and the last file in the list appears just above it, so you can click on that file without touching the statusbar.

The sideways arrows appear in the statusbar, so you only click on them if you NEED them.

Hope this helps people with this problem.
