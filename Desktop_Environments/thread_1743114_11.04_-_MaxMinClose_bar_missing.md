### Post by 3Miro on 2011-04-29
Install Compiz-Config-Settings-Manager and setup Windows Decorations -> compiz-decorator (I am not sure if this is the right name).

If compiz-decorator is there already, then press Alt+F2 and start it manually (compiz-decorator --replace).
