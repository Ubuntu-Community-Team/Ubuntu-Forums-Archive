Hello and Welcome 

It is possible to add items to the Nautilus context menus but I find nothing so far in regards the title bar context menu . It appears that you can add  key binding actions via system settings > keyboard > shortcuts > windows.Some functions are disabled by default and this may be because of key binding conflicts 

Advanced settings includes some window action options and the CCSM may also offer some possibilities.
