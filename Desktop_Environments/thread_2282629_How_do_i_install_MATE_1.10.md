### Post by buzzingrobot on 2015-06-16
> **timmy8 said:**
> I used Terminal in Ubuntu 15.04 to install MATE. but its 1.8.2


What do u mean its source packege?? u mean ISO. 

The Mate project ([http://mate-desktop.org/](http://mate-desktop.org/)) has released the source code for 1.10. The Mate project, like most other similar projects, does not compile and package its source code into packages for installation on distributions. Each distribution does that individually, according to its own wishes and schedules.

Distributions, like Ubuntu, that release on fixed schedules, usually do not add major upgrades like this to previous and current releases. 

As mentioned, 1.10 will likely be available in Ubuntu's 15.10 release, specifically the Ubuntu Mate spin.  

It's very unlikely that 1.10 will be built and packaged for Ubuntu 15.04. Still, you might keep on eye on the Mate site and the Ubuntu Mate site ([https://ubuntu-mate.org/](https://ubuntu-mate.org/)).
