### Post by soce_32 on 2005-07-20
Nautilus can hook up to an ftp server.  You can go to the File menu and choose Open Location.  Put in the address of your ftp server and it will ask you if you want to connect anonymously or put in a username and password.  gftp is a graphical ftp/http/sftp client that you may want to try as well.
