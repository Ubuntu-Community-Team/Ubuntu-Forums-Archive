### Post by BlueSkyNIS on 2008-06-12
If you have compiz enabled and nvidia 8600 card then it might be hardware problem. I experienced this a while ago with my new gigabyte 8600gt card in windows XP while watching DVDs (using hardware acceleration) or playing 3D games (note: 2D mode was running just fine). I returned the card to the seller and he gave me other gigabyte 8600gt card which works without problem.

Do you have multicoloured mesh in windows when you are using 3D? Try to go to nvidia control panel where you can see the animated 3D nvidia logo.
