### Post by astoltz on 2007-12-24
So once you get nautilus started and then save the session (press the "Remember my currently running applications" in system -> preferences -> sessions), will everything startup OK after a log out / log in?

If that doesn't work, the only other suggestion I have would be to wipe out your gnome configuration and start fresh.  Move the ~/.config and all the ~/.gnome* directories to a backup location.  Then log in and keep your fingers crossed.  ***WARNING*** that will remove all the customizing you've done as well as any saved settings in some applications (totem, rhythmbox, etc).
