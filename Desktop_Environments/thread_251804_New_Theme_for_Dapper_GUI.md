### Post by aysiu on 2006-09-05
[http://www.gnome-look.org](http://www.gnome-look.org) is one place to get themes.

When you first go to the page, you might be overwhelmed with all the different types of themes on the left side. For the most part, all you really need to concern yourself with are *GTK* or *Metacity* themes, *icon* themes, and *GDM* themes. The *GTK/Metacity* themes change the look and feel of your windows--window decorations and borders and controls and buttons. The *icon* themes replace the icons for programs with... other icons. *GDM* themes are for the login screen.

To install the *GTK*, *Metacity*, and *icon* themes, go to System > Preferences > Themes. A Theme Manager window will appear. Drag and drop into that window the .tar.gz file you downloaded from Gnome-Look. .tar.gz is a zipped file extension, much like .zip in Windows or .dmg in Mac OS X. **Do not unzip the file**. Just drag and drop it just as you downloaded it.

To install GDM themes, go to System > Administration > Login Window and then add a theme. Same deal--do not unzip the .tar.gz. Just install it exactly as you downloaded it.
