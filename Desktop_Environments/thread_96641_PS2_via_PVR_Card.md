I run my Playstation 2 with an ATI TV Wonder capture card.
tvtime works very nicely for that, it is based on the Windows program Dscaler
Get  tvtime with Synaptic ( enable the universe and multiverse repositories ).

1) connect the pS2 to the Svideo input of the capture card
2) run Tvtime
3) start ypur PS2
4) right-click on the tvtime windows and select the input ( cable, svideo etc... )

And voilà.
