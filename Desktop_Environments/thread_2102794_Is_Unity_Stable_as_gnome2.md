### Post by tartalo on 2013-01-08
If you like Unity as implemented in 12.04 you can stay there for up to 5 years, if not don't expect any stability.

Unity is changing and will change a lot still, on one side because it needs to change to suit better users' needs, and on the other because the elements it uses change and Ubuntu can't do anything but adapt, Gnome 3 is changing fast and Unity uses many Gnome 3 elements: recently Gnome simplified Nautilus, Ubuntu stayed in a previous version but that's a short term workaround. Gnome dropped the fallback mode too, this meant the death of Unity 2D, and if hardware acceleration is not supported for your graphic card the Unity experience won't be very nice because software acceleration is slower. A few days ago the main Compiz developer announced that he will maintain it but not further develop it, so I'd expect Ubuntu to replace it. Will they stay on Compiz until Wayland's Weston is ready? Will they go back to Mutter? Jump to Kwin?...

Gnome 2 is a dead end, but there's a fork called Mate that intends to keep the Gnome 2 experience alive, I haven't used it myself though. The pressing question about Mate is what will happen when distros start dropping GTK2 support.

If I was you, given the specs of your PC, I'd give XFCE a chance, or maybe LXDE. Most say that XFCE is the best replacement for Gnome 2, but I have a very good impression about both.
