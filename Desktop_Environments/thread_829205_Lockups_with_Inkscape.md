I've been experiencing system lockups with Inkscape. I think it might be because I'm running out of memory and it starts going into swap, and since the memory is shared with the graphics card, **the graphics card starts using the swap instead of the physical memory!** That is when I experience the lockups. So my options are:

[LIST=1]
[*]Buy more memory (upgrade to 2GB)
[*]Find a way to reduce memory usage of other applications
[*]Find another way to fix it
[/LIST]
I've also noticed that **my system only locks up when I'm running an Internet browser (in this case Opera) as well as Inkscape**. But I like to have the browser open to help me check my website works with the images (the images I make are mostly for my website)...
