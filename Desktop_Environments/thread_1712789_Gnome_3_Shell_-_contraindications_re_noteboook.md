I suspect Gnome-shell will be in the Software Center, so you should be able to just install 11.04, then get Gnome-shell and use it (logout and then get a different session).

11.04 will also support Gnome Classic mode, which is pretty much exactly like 10.10 (Gnome Classic will not require anything new to be installed).

BTW: it isn't really important, but there is no Gnome Shell 3, there is Gnome 3 and Gnome-shell with is a component of Gnome 3. Unity replaces Gnome-shell with Ubuntu customized version of Compiz. Thus Unity is Gnome.
