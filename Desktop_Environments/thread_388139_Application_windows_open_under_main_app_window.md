I have a strange problem where all the 'sub' windows of applications now open underneath the main application's window.  e.g. I'm in firefox now and I go to 'File > Open file' and the dialog box doesn't appear unless I click on the main firefox window again.
I'm running Beryl and updated it to the most recent version last week which seemed to coincide with this problem but I could be mistaken.

---

### Post by adam.tropics on 2007-03-19
On your first Beryl Settings Manager screen is a setting 'Level of Focus Stealing Prevention'. Set it at normal or below, and maybe reload window manager for good measure!
