### Post by coffeecat on 2006-07-22
I've found a way but it's pretty ugly. Assuming you are using Ubuntu/Gnome, go to System > Preferences > Font. In the Font Preferences window click on 'Details'. In the next window click on 'Go to font folder'. You get a Nautilus window with - I believe - a pseudo-location with all the fonts installed in your system. In the Nautilus window drop down the View menu and change to 'View as list'. Make sure the window is active and do ctrl-A to select all and then ctrl-C to copy all. Now open the text editor and do a ctrl-V in the editing area. Now save and/or print from the text editor.

I can't guarantee that every font is listed - I've only just discovered this by fiddling around. And as I said it's ugly, and it doesn't print a sample of each font which would be useful. If anyone knows a better way I for one would be glad to hear of it.
