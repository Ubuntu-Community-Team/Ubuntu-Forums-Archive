### Post by slickymaster on 2015-01-14
**Ubuntu 13.10 (Saucy Salamander) is end of life since July 17, 2014**, so my advice is to [install Ubuntu 14.04]("http://www.ubuntu.com/download") and afterwards to get the classic Gnome desktop press Ctrl + Alt + T to open a terminal window and run the following:```
sudo apt-get update
sudo apt-get install gnome-session-fallback
```When the installation is finished, close the terminal and log out of your session and log in again. A list of available desktop environments will be displayed for selection. Select the GNOME Flashback option for either Compiz or Metacity.

### Post by grahammechanical on 2015-01-14
Ubuntu 13.10 has reached end of life. It only had nine months support. Upgrade to 14.04 and search the software centre for Flashback and install gnome-session-flashback. Restart and at the login screen click the Ubuntu icon just above the panel for the password and select either Gnome Flashback (Compiz) or Gnome Flashback (Metacity).

Regards.
