[B]#	Horizsync	31-80
#	Vertrefresh	60-75[/B]

Firstly, faster refresh rates are usually of use for CRT monitors, LCD monitors are usually limited by the pixel response rate so a faster refresh rate is essentially pointless.

Secondly, commenting out the lines that define the rates allows the system to auto-detect them, and this can allow more options.

The next option is to do:
```
sudo dpkg-reconfigure xserver-xorg
```
and let it choose the right ones itself.
