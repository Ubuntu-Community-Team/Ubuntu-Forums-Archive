### Post by hajk on 2006-08-31
I am using a Parallels VM to run Ubuntu on my MacBook laptop. Installation of Ubuntu is real easy. 

Advantage: ease of use, easy switching between Ubuntu and underlying OS (that also keeps running), ideal for combing the strengths of both OS'es while avoiding their weak points (like Ubuntu's multimedia and wireless support). 

Disadvantage: Ubuntu will run slower in the VM, both because of the extra software layer between it and the hardware, and also because Parallels uses only a single core in a multi-core processor setup.
