### Post by calande on 2011-05-05
Hello,

I'm now using natty, and Compiz, but if Compiz is enables, I can't move any window. If I try to drag a window with the title bar and my mouse, it has no effect. The cursor won't grab the window. How can I solve this problem please?
Thanks,

---

### Post by Krytarik on 2011-05-05
Make sure that "Move Window" is enabled in "CompizConfig Settings Manager -> Window Management", btw. also check "Resize Window".

Greetings.

### Post by calande on 2011-05-06
