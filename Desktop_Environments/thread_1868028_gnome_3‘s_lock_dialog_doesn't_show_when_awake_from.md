I think it's this bug: [https://bugs.launchpad.net/ubuntu/+source/gnome-screensaver/+bug/848562](https://bugs.launchpad.net/ubuntu/+source/gnome-screensaver/+bug/848562)

GNOME is planning on putting screen locking in the compositor at some point, which should make it more robust.
