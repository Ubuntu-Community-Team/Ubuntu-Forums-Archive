### Post by aysiu on 2006-05-03
I don't know about other programs, but for Kate, there's a simple fix.

Kate's default behavior (boxed in red) is to save the backup file with the suffix **~**

If you change that to (boxed in green) saving the backup file with the prefix **.** then those backup files will be hidden.
