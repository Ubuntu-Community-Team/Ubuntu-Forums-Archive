### Post by dcroal on 2005-12-01
For a Sony Clie which connects by USB you want to use /dev/ttyUSB0 instead of /dev/pilot. Don't worry if you don't see /dev/ttyUSB0 in your system, it only appears when you are doing a HotSync.
