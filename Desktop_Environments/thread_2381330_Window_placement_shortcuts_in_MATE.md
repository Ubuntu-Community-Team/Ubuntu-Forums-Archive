### Post by hojjat on 2017-12-29
In Ubuntu with Unity, I often use keyboard shortcuts (such as [FONT=Courier New]CTRL[/FONT] + [FONT=Courier New]ALT[/FONT] + [FONT=Courier New]NUMPAD 6[/FONT]) to place windows to the right or left side of the screen.

I am now using Ubuntu with MATE, and using it on a MacBook laptop through VirtualBox. So I am eager to know if (a) is there a way to in MATE to do window management, i.e. shortcuts for placing windows in different sides of the desktop, and (b) if so, can the shortcuts be configured (as I will most likely need to change them, since a Mac's keyboard does not easily translate to Linux/Windows keyboards).

---

### Post by hojjat on 2017-12-29
Found the answer myself: System > Preferences > Hardware > Keyboard Shortcuts then under Window Management fine Tile window to east (or west) side of screen
