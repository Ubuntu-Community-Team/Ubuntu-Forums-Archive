Don't quote me but I think this is how it is intended now. Empathy is "integrated" into gnome-shell. Once you configure chat in "online accounts" then your status is set via the me menu (top right), new chats will pop up in the notification window, and you can see your contact list at any time by starting empathy from the launcher (or via search).

So it all "just works". What is your actual problem? Empathy is not intended to be displayed with a sys tray icon anymore.
