### Post by e2k on 2005-12-16
You can remove it by going to: Applications>System Tools>Configuration editor>apps>metacity>general and checking the box for reduced-resources..
This gives an ugly "wireframe" while moving windows though, which you can work around by going to: System>Preferences>Assistive Technology Support and enabling it.. This should do the trick :)
