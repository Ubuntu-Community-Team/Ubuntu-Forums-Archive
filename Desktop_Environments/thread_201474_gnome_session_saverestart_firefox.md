Why does gnome fail to save/restore firefox session? I don't expect url level support, but kde at least restarts firefox after a restart.

### Post by aysiu on 2006-06-21
Go to System > Preferences > Sessions and add Firefox to the startup programs.

Otherwise, install the SessionSaver extension in Firefox.
