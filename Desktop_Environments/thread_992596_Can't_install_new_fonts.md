### Post by kostkon on 2008-11-25
In general to install fonts you just put them in the hidden *.fonts* folder that exists in your home folder. If you open your home folder with the file browser and select *View &#8594; Show Hidden Files* you should be able to find it. If it does not exist, just create it yourself and put your fonts there.

If you want the MS fonts, then do what *keithld* suggested.
