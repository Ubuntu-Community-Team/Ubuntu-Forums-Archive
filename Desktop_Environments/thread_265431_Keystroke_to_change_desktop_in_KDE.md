### Post by bforbes on 2006-09-25
In Breezy I was able to press Ctrl-Tab and Ctrl-Shift-Tab to change between desktops (in KDE). In Dapper this does not seem to be enabled by default. Is there some way I can enable it? Additionally it would be great if I could customize the keystroke, since I would prefer to use Ctrl-Tab in Firefox. Is that possible?

---

### Post by Mr Frosti on 2006-09-25
If you are still using KDE in Dapper, right click on the desktop and go to the display settings. There will be a section dealing with workspaces. Make sure that you have more than one workspace to toggle between.

Also, under personal settings in KDE, do a search for keyboard shortcuts. You should see a section dealing with what keybinding will switch between desktops.

If you are now using Gnome, the keystroke "Ctl+Alt+Left/Right" will alternate workspaces (if there is more than one enabled)

### Post by ltmon on 2006-09-25
System Settings -> Regional & Accessibility -> Keyboard Shortcuts

You can set it up here.  Although, in my install of Dapper, ctrl-tab is the default.
