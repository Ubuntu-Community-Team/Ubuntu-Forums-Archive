### Post by ModelM on 2008-07-02
If you're intending to completely wipe Linux & install Windows, the Windows CD will do that for you. And, if you're saying that the machine won't boot from the Windows CD, the problem is not with Linux. It should boot from the Windows CD even with no harddrive installed at all.

You might check your BIOS settings & make sure that "boot from cd/dvd" is the first choice on the boot list, otherwise the machine will skip the CD & go right to the harddrive & boot from there.
