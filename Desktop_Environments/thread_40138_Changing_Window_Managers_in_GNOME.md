### Post by erik-the-red on 2005-06-07
How can I change from metacity, the default window manager, to something like fluxbox or openbox, given that I have already installed those two window managers on my system?

GNOME is, after all, a desktop environment.

---

### Post by Arthemys on 2005-06-08
Go to a shell window and type in 'openbox --replace'
It will then replace the WM, to make sure it sticks that way. Logout of your gnome session, and make sure the session is saved with no other applications running if you wish. Log back in, and it'll be there.
