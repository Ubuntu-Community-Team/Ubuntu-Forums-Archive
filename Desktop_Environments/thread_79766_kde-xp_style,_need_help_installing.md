### Post by aysiu on 2006-01-25
Click on the .tar.gz
Click on the purple "up" arrow to extract it.
Save the extracted folder to the desktop.
Open a terminal (Alt-F2, ```
konsole
```)
Type ```
cd Desktop
cd kde_xp
./install.sh
``` Enter your password.
