### Post by sanderd17 on 2011-07-18
The problem is that the sound menu is an Ubuntu option. Gnome Shell is not supported by Ubuntu (some users only hacked their way to it) so the Ubuntu only features aren't working yet.

You should also tell your installation method. Did you add the gnome PPA, did you do a jhbuild or did you install it with the gNatty live CD?

btw, the icons on the top are called the "status icons", the icons that appear on the bottom are "notification icons" so they have a different purpose.

This extension gives you a similar functionality: [http://www.webupd8.org/2011/06/soundmenu-like-gnome-shell-extension.html](http://www.webupd8.org/2011/06/soundmenu-like-gnome-shell-extension.html)

And everything should work out of the box in 11.10.
