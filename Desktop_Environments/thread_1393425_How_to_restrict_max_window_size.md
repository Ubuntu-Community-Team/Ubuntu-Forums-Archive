### Post by Zorael on 2010-01-29
(I can't be more than vague as I use KWin.)

I believe this would have to be a Compiz setting, for which you'd need to install the settings manager (**ccsm**) if you haven't already done so.

Is there a Window Rules plugin, or perhaps a Workarounds one? If so, you should be able to make rules in there for your apps by [matching](http://wiki.compiz.org/WindowMatching) their window properties. Look for an option to limit maximum size. If there isn't one, perhaps there's an option to clamp it to a specific size and disallow resizing.
