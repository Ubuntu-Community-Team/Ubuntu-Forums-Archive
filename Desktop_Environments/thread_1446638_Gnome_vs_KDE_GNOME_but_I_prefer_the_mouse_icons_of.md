### Post by jfabrizio on 2010-04-05
I have resolved my problem.

I have installed the oxy cursor theme:
```
apt-get install oxygen-cursor-theme
```

After..
System->Preferences->Apparance->Customize->Pointer and choose oxy-white

;-)
