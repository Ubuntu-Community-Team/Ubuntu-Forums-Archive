### Post by shakin on 2005-06-13
I used Thunderbird for a long time, but switched to Evolution when I started using Ubuntu (because it was there). Evolution's features are first-rate.

However, Evo is a RAM hog and the interface is clumsy. After I switched to KDE I tried KMail, but after only a day I decided I didn't like it. I finally found Opera and so far it has been the perfect email client. It takes up *much* less RAM than Evolution, the interface is very simple and the tabs are nearly as good for email as for browsing the web. There are lots of organization features (filters, labels, etc) and Opera is very fast.

Opera even remembers its last startup mode, so it starts up in email mode because I don't use it as a web browser.

I highly recommend it to anyone who hasn't tried it yet.
