I am new to KDE and tried to find how to set up shortcuts keys for terminal, firefox, thunderbird etc.

### Post by aysiu on 2006-04-14
Right-click on the KMenu
Select "Edit KMenu"
Go to the Firefox entry and define the keyboard shortcut in the bottom right-hand corner.
