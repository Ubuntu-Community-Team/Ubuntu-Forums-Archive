### Post by Impavidus on 2014-10-11
You can edit your user settings for keyboards. I don't think you have to download any additional packages. You can set a key to switch keyboard layout. If it's just to get the additional characters without turning your qwerty keyboard into an azerty one (which would be very confusing), you may be able to use some sort of international layout, with third and fourth level choosers. Dead keys can be useful too, for typing accents and other diacritical marks.
