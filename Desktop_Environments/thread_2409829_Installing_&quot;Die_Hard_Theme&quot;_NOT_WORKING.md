### Post by Frogs Hair on 2019-01-13
The GTK version for gnome changes yearly and it's up to the theme developer to maintain the theme. This is why you have to check the GTK version if listed when browsing on gnome look. This theme also uses the emerald theme manager which requires compiz and is not compatible with the current gnome DE though it does work with the Unity desktop and gnome flash back session. Emerald would have to be built because it is no longer maintained.
