### Post by Emerzen on 2005-09-09
Hi Wayne,

Direct Response: 
if it's a full install CD or reinstall CD it will not be a problem.  However, a recovery CD or Upgrade CD is more problematic and I don't know the answer.  If I were you, I wouldn't try it unless I was sure I had one of the first 2 in hand.

Unsolicited Advice:
Don't install Ubuntu until you've got it working w/ a Live CD.  I would redownload the LiveCD and reburn the ISO w/ your burning software (and turn overburning off if you have it).  Then recheck the Live CD and see if it works.  You should also check to see if the Live CD you have has any smudges or scratches that can easily be removed.    There are ways to check the integrity of your CD directly, the MD5sum check, a forum search will give you more details.  If that works, I would redownload the install CD and reburn it w/ the same settings on your burner software before trying the install.  I also recommend Bittorrent for the downloads for, theoretically, smaller possibility of data corruption.  

Other option would be to set your system up for dual-boot.
