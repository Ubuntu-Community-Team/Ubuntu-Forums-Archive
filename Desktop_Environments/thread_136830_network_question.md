### Post by ltmon on 2006-02-26
Hi there,

I know in Dapper (the next version of Kubuntu) there is a great little tray applet called "KNemo" which works for wired/wireless networks.

I'm not sure if this package is in Breezy though (I'm living on the edge and using Dapper myself).  It might be, so check under Adept (or 'sudo apt-get install knemo') and see.  It will appear in the tray next time you log in to KDE.

L.

EDIT: If it's not in the repositories (make sure you have Universe enabled) you could use the unofficial package here: [http://www.mpe.mpg.de/~ach/kubuntu/breezy/Pkgs.php](http://www.mpe.mpg.de/~ach/kubuntu/breezy/Pkgs.php)
