My Ubuntu home machines check for updates every morning when I turn them on.  This is how it should be and it works well.  However, my Ubuntu work machine is left on 24x7 and it doesn't seem to do any checking for updates.  I noticed this a few days ago when my home machines reminded me of updates but my work machine didn't.  Even after 3 days.  I finally told it to check for updates and it found many. The update procedure went fine.  But how come my 24x7 machine isn't checking for updates automaticallyu at least once/day?  Thanks!

### Post by darkmatter on 2005-10-12
Hmmm...first make sure that updating is enabled.

Open either the Update Manager or Synaptic on your work machine. Under Preferences, click the Settings button and make sure that 'Automatically check for software updates' is checked and that the update interval is set to you preferences (the interval is in days, so if you want to update daily, have it set for 1)
