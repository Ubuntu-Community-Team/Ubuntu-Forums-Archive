### Post by jasutton on 2006-07-31
That's an interesting config file (the one made by the ATI driver).  It looks like you have two 'Device' sections: one that's using the binary ATI driver ('fglrx') and one that's using the open source ATI driver ('ati').  For Xgl/Compiz, you'll want the binary one.  However, your 'Screen' section seems to be using the 'Device' section that has the 'ati' driver associated with it.  Here's a modified file that I might use:

```

# (Type "man /etc/X11/xorg.conf" at the shell prompt.)
#
# This file is automatically updated on xserver-xorg package upgrades *only*
# if it has not been modified since the last upgrade of the xserver-xorg
# package.
#
# If you have edited this file but would like it to be automatically updated
# again, run the following command:
# sudo dpkg-reconfigure -phigh xserver-xorg

Section "Files"
  FontPath "/usr/share/X11/fonts/misc"
  FontPath "/usr/share/X11/fonts/cyrillic"
  FontPath "/usr/share/X11/fonts/100dpi/:unscaled"
  FontPath "/usr/share/X11/fonts/75dpi/:unscaled"
  FontPath "/usr/share/X11/fonts/Type1"
  FontPath "/usr/share/X11/fonts/100dpi"
```

### Post by jackkerouac on 2006-08-01
I had to run

```

sudo gedit /etc/modprobe.d/blacklist

```

and then

```

blacklist agpgart

```

In order to get my Radeon X700 to work properly. Turns out the AGPGART was failing. You can take a look in your /var/log/Xorg.0.log to find out where your problem is.

### Post by endfx on 2006-08-01
Hmmm, I wouldn't use/trust ATI's install script that you get from their site. Does the install file come with a uninstall script?

I would first try to uninstall it.
Then here is what you do:

1)
sudo apt-get update
sudo apt-get install xorg-driver-fglrx

2)
In your original xorg.conf file (NOT the one made by ati's installer) simply change the following from:
#-----------------------------
Section "Device"
Identifier "ATI Technologies, Inc. Radeon Mobility M7 LW [Radeon Mobility 9000]"
Driver "ati"
BusID "PCI:1:0:0"
EndSection
#------------------------------------
to

#------------------------------------
Section "Device"
Identifier "ATI Technologies, Inc. Radeon Mobility M7 LW [Radeon Mobility 9000]"
Driver "fglrx"
BusID "PCI:1:0:0"
EndSection
#--------------------

Notice I only changed the Driver name from ati to fglrx

That's it. Just restart X or better yet, reboot.
Oh and the above packages that you installed are described here:
[http://packages.ubuntu.com/dapper/misc/xorg-driver-fglrx](http://packages.ubuntu.com/dapper/misc/xorg-driver-fglrx)

Its ubuntus package of the ATI binary, so you know it will be installed correctly.

Oh, and one last thing, are you using dapper or breezy? I have installed a lot of ATI cards on ubuntu over the past couple of months and I noticed that with breezy some ATI cards just won't work, so I upgraded to dapper and they just started working.

Hope this helps you out.
