### Post by maxamillion on 2007-01-30
1. The reason is because under Xubuntu OpenOffice.org looks like OpenOffice.org actually looks by default and under Ubuntu it uses the package openoffice.org-gnome (if I remember correctly) in order to make it look more uniform and keep the look of the gnome theme, but that is something that will cause gnome-libs to take up system resources which is something the Xubuntu devs work very hard to keep from happening.

2. That is actually an issue with the way Xfce-Beta (I believe the issue is fixed in the current stable release that will be part of Xubuntu Feisty) because it only indexes the icons in 3 different sizes and just uses which one will fit in the scale of the panel

3. Editing the Applications menu can be done by going to Applications->Settings->Menu Editor

4. I think it is just default for the check box to be checked when you log out, but it could also just be a bug that is being worked on.
