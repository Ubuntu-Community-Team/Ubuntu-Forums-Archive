### Post by mgmiller on 2006-08-27
I use Picasa for linux to print and it works great.  Recently I started printing to 4x6.  I used to print only to 8.5x11, which worked perfectly.  When I tried printing to 4x6, I got the same result as you.  What you need to do is in your system > adminsitration > printing area, bring up the printer you are using and make sure its paper size is set to 4x6.  This would get very tiresome, switching the paper sizes back and forth, so what I did was create a few more printers, with name like photo_4x6 and photo_5x7 with their drivers set to the appropriate paper sizes, now when I want to print to 4x6 or 5x7 or etc.  I just select that printer from the list in whatever program I am using to print from.  I left the default printer as using full size paper. Make sure you are also setting the paper type to the kind of photo paper you use, this makes a huge difference.  My default printer uses full size plain paper, my photo_printers, use canon photo paper.  Doing it this way, I can produce pictures that look better from my linux machine than they do from my windows machine.  My printer is a 6 color Canon i960.

So,in short,the solution is to define several printers with different paper sizes and types specified for each.

Good Luck
