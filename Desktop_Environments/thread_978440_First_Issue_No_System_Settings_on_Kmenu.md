### Post by gsmlinux on 2008-11-11
If you wish to add System Settings to your favourites list, run the program from the menu (I used the search bar to find it). Then go back into kmenu and look at the "Recently Used" tab. You will find System Settings there. Now right click on System Settings in this list and select the option "Add to Favourites". 

Now you have added it to the Favourites tab. If you wish to remove it from Favourites just right click on it in that list and select "Remove from favourites"
