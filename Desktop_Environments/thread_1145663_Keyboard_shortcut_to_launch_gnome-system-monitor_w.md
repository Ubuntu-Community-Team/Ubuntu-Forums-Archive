### Post by msriram on 2009-06-01
1. go to keyboard shortcuts
System->Preferences->Keyboard Shortcuts

2. click Add
Name : Start Gnome System Monitor
Command: /usr/bin/gnome-system-monitor

3. click Apply

4. Click Disabled under the Shortcut tab

5. Use Ctrl+Shift+Esc.. Dont type. Press these buttons...
