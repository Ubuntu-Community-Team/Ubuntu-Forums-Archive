### Post by ankspo71 on 2011-04-02
Hi,

If you create an application launcher (*.desktop file) to the cartoon folder, you can add that launcher to the unity dock. 

1 Right click on your desktop and select "Create Launcher".
```

Name = Cartoons
Command = nautilus /path/to/cartoon-folder
```
2 Choose a icon for the launcher.
3 When you are done click on OK to save the file.
4 Move the Cartoons launcher that you created on the desktop to your home folder.
5 Open your home folder, then drag the Cartoons launcher to the Unity dock.
6 The Cartoons launcher should now be inside the dock. 

This works for me in Ubuntu 11.04 beta1

You can use this way to create a 2 firefox launchers too.

Hope this helps.
