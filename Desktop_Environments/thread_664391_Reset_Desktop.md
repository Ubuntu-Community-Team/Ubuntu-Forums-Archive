### Post by mali2297 on 2008-01-11
There is an option in the file manager to view hidden files, enable it. Your personal configurations are stored in the hidden directory **.config**, remove the things in there that are related to xfce. Also remove the directory **.cache**. Then restart without saving the session.

### Post by az on 2008-01-11
Creating a new user works in all window managers and desktop environments.  The new user gets a default configuration.  This applies to desktop effects, too.
