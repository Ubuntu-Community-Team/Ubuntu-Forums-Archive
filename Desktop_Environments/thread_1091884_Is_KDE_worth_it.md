### Post by taurus on 2009-03-09
Why not install KDE and check it our yourself.

Applications -> Accessories -> Terminal
```
sudo apt-get update
sudo apt-get install kubuntu-desktop
```
Once it's done, log out and at the GUI login screen, pick what session (KDE or Gnome) you want to use and log back in.

### Post by Aries-Belgium on 2009-03-10
I've used several desktop environments over the past years and I've settled with KDE for now. The only thing that I don't like about KDE is their update philosophy. They don't distribute updates for separate applications (like Konquerer, Kopete, ...) but update the whole system at once. That's quite frustrating when you there's a specific bug or feature that needs to be fixed that you have to wait for the next release of the complete KDE system. I know you could use the development version of KDE via CVS/SVN but you have to compile those packages from source and that can be a pain in the *** :P

I think you better try all the desktop environments to really know which one fits your needs. Try every desktop environment for a month or so and then pick the one you liked the best ;)
