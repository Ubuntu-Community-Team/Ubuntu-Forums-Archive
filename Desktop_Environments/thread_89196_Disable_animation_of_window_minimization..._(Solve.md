### Post by aysiu on 2005-11-12
I tried poking around in System > Preferences and Applications > System Tools > Configuration Editor, but I can't find anything that gets rid of the annoying animation of the windows minimizing (it looks like a stilted shrinking of the window with black box outlines).

Anyone know what I'm talking about? Anyone know how to fix it?

---

### Post by 23meg on 2005-11-12
You should enable /apps/metacity/general/reduced_resources in the configuration editor. With this you also lose the ability to see window contents while dragging windows though.

### Post by Buffalo Soldier on 2005-11-12
