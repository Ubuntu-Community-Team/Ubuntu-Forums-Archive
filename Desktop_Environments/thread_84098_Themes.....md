### Post by Neobuntu on 2005-10-30
I suggest you try the MacOS-X theme which included "controls", "Window boarders" and "icons" but I find only "Sandy" or "flat-blue" icons (use Theme details) works to my likeing.

Some "themes" are only controls for example. Some total themes change the look of Synaptic and some don't (for some reason.)

Over all I use the MacOS-X theme for it's "controls" and then use Theme details to set the Window boarder (only) to "Clear looks" or the downloaded "ale-panther_gtk2".

"Crystal dlb" seems to be a simpler and nice choice for all details.

Note that you can simply download the compressed theme and use the theme preferences "Install Theme" to select it without uncompressing it first.

Also, I set KDE theme to "Baghira" (and tweaked a bit) so the when I use KDE/QT apps in Gnome they match. It's all good!

Basically if you work hard enough at it you can make it look like anything you want.

There's also a post about getting rid of the brown in ubuntu that you should be able to search easily. In the GDM login settings the start uo background color is simply set in the regular greater (not graphic) setup screen.

Then downloaded Gnome splashes can be put into ".gnome" as well (renamed "splash.png")

Hope that helps.

Except for the Icons.

Also, do not set (In KDE Kubuntu) the gtk theme (for when running gtk apps in KDE) becuase if you do in will impeed the function back in Gnome (ubuntu) for Theme preferences. If they don't change (and you use Gnome) log back into KDE and unselect a GTK theme. This is with gtk2-engines-gtk-qt (also known as gtk-qt-engine) installed. "Please note that this package is targeted at KDE users and therefore provides a way to configure it from within KControl."

I had to uninstall gtk2-engines-gtk-qt so that the Gnome theme would change correctly. If you notice your taskbar isn't changing (when you click a different theme) then this might be what you need to do to fix it.

Note too the "gnome-art" package (Beta I think) give a nice program to get AND install themes (and stuff.)

Install gnome-art (Use Synaptic perhaps)
