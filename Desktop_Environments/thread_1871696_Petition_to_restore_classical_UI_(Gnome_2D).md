### Post by kurt18947 on 2011-10-29
For anyone wanting to restore the "classic" panels etc. interface I recommend installing Xfce-desktop.  It's in the repositories, just install like any other package then log out and log back in while clicking the "cog" and selecting xfce.  This will keep the standard gnome apps and users can log onto Ubuntu (unity) or Gnome (shell,my preference) and experiment with them in small doses while still having a security blanket (xfce).  Xfce will not support ALL gnome 2 functions but navigation should feel familiar.
