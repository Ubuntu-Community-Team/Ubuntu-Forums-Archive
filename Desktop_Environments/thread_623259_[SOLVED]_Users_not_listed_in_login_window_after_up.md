### Post by C-A on 2007-11-25
After upgrading to Gutsy, users are no longer listed in the login window. I have checked the user settings for the login window and have tried to "include all users from /etc/passwd" and have tried adding my three users in the include list.  Either way, the users do not show up in the user list for the login window even though I can login fine with each user account.

:confused:

---

### Post by C-A on 2007-11-28
In Administration > Login Window > Local > Style:

I changed it from "Themed" to "Themed with face browser" and my user list was back!
