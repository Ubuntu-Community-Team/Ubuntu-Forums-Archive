### Post by shakin on 2005-06-22
KDE user here.

Unfortunately, Konq is missing Nautilus' 'highlight the name only' rename feature and I miss that. But it *does* offer the 'home goes to beginning of current line' feature.

What I absolutely love about Konq as a file manager is the fact that it opens so many different file types without needing to launch another app. It can open zips, tars, gzs and bz2s as if they were normal directories. If you open an archive with a file that depends on another one (like an html page with image tags pointing to images in the archive), Konq will treat it like a normal directory and those files will work. It also opens PDFs and images in the file manager window with almost no delay waiting for a new app to start.

A feature that I can't live without is Network Folders (connect to server in Gnome, but a bit better imo). Whatever did I do without them? Using a system without an equivalent feature is incredibly painful for me since I use them many times every day.

I'll also include apt-get and Synaptic here when comparing against any non-Debian distro or Windows. Yesterday I was asked to install Foxpro 8 on a coworker's Windows 2000 box and it took me 45 minutes to track down and install the dependencies. Yes, Windows has the dependency hell problem too. It was like Red Hat all over again, searching the web for installers.
