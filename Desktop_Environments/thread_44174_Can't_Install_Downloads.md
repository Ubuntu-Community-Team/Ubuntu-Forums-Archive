### Post by Leif on 2005-06-24
You don't need to download tarballs when using ubuntu (at least not for most programs). There are two ways you can install things. One is to open a terminal and type "sudo apt-get install thunderbird", and enter your normal password when prompted. The other (and probably easier) way is thru synaptic. Go to System->Administration->Synaptic. Search for thunderbird. Right click and choose install. Click apply. Voila !

You should also probably read [http://ubuntuguide.org/](http://ubuntuguide.org/), it's a goldmine.
