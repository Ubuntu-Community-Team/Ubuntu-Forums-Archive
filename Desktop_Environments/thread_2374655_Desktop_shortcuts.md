### Post by TheFu on 2017-10-18
Welcome to the forums.

The DE used controls whether this is possible or not.  For most DEs, the "Desktop" is just a directory in your HOME, so you can create a symbolic link into there from anywhere and it will work.  I haven't used Unity since 12.04 ... so I don't know if that DE pays any attention to this method.  **Every** other DE or WM I've used does work this way. 

[https://en.wikipedia.org/wiki/Symbolic_link](https://en.wikipedia.org/wiki/Symbolic_link) explains more about symlinks, symbolic links, soft-links ... those are all the same.
