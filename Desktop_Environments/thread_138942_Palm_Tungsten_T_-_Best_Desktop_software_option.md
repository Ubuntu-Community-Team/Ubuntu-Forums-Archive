### Post by bergersau on 2006-12-13
I'm now using a combination of:
* Jpilot for the PDA sync basic stuff. - Good App once you get the sync setup sorted for your system.

[http://www.jpilot.org/](http://www.jpilot.org/)

* Bibelot script for converting text files to palm DOC files. Command line apps are still the best tool sometimes...

[http://sourceforge.net/projects/bibelot/](http://sourceforge.net/projects/bibelot/)

* A USB hardware SD card reader. - I thought it better value and faster to transfer files than to register Card Export - Cool app though it is.

I also love the palm app TCPMP for audio AND video playback on the palm platform. FREE (Open Source) software and works very well - Though a very basic interface and playlist control.

[http://picard.exceed.hu/tcpmp/](http://picard.exceed.hu/tcpmp/)

Covers most of my palm usage.
