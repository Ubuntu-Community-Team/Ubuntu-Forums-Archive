### Post by PaganHippie on 2007-11-21
```
sudo apt-get install g77
```should do it.  apt-get will take care of the dependencies automatically.

If you want to know what those dependencies are, you can look up g77 in Synaptic, turn on "Show Package Properties in the Main Window" in Preferences, and then click on the "Dependencies" tab and it's all right there.
