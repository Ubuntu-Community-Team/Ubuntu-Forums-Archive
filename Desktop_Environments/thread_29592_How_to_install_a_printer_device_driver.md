### Post by NeoChaosX on 2005-04-25
Make sure you have hpijs and foomatic-db-hpijs installed. If they're not already installed, just enter *sudo apt-get install hpijs foomatic-db-hpijs* in a console.

Then, the driver will be avalible when you try to add a printer (System -> Administration -> Printing).
