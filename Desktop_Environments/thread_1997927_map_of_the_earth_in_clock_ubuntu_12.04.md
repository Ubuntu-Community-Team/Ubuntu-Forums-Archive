### Post by uncledonn on 2013-02-21
Getting the original panel back using &#8220;gnome-session-fallback&#8221; desktop environment in 12.04:
 

 *(Windows Key + Alt + right click on menu bar* activates the sub-menu*)*
 

 
[LIST]
[*]Install *sudo apt-get install     indicator-applet-appmenu*
[LIST]
[*]This is similar             to the old style applications menu found on the left side.
[/LIST]
 
[/LIST]
 

 
[LIST]
[*]Activate     the sub-menu and choose &#8220;Add To Tray&#8221;
[/LIST]
         Add:
 
[LIST]
[*]Clock
[*]Indicator     Applet Session
[*]Notification     Area
[*]Indicator     Applet Appmenu
[/LIST]
 

 &#8220;Remove From Panel&#8221; &#8220;Indicator Applet Complete&#8221; and move the other ones where you want them.
 

 You will want to leave the &#8220;Indicator Applet&#8221; with the envelope, network and sound on the panel.
