I am  running Ubuntu 10.04 GNOME.

I am using the Faenza icon theme.

Faenza has its own custom icon for Firefox.

I want to change the Firefox icon back to the normal Firefox icon but still keep th rest of the Faenza con theme.

How can I do this?

---

### Post by 73ckn797 on 2010-10-31
You can right click over the icon in the top panel and select properties. Then click over the icon which will allow you to change the icon. You can search for the icon files from there.

You will find icons by going to /File System/usr/share/icons from Places/Computer/File System/usr/share/icons/hicolor/scalable. Look around in there. Typically the "scalable" folder will have what you are looking for.
