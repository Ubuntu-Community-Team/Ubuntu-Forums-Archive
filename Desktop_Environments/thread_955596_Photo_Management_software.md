### Post by Coreigh on 2008-10-22
I admit I do not know the exact answer to "Does Ubuntu have an ACDSee equivalent?"

BUT

F-Spot is a good place to start. I doubt you will find an 'exact' replacement for ACDSee, one that works the same way, but F-spot is similar.

I think it is part of the default installation. It would be in the Applications menu under Graphics. If you don't find it there, use the Add/Remove program and look under the Graphics section for F-Spot.

### Post by C. Wizard on 2008-11-02
Gwenview is very similar to ACDSee in form and function. 

I like Gwenview. It is a very good graphic program and the best I've seen for Linux to date, but I wish I could get ACDSee to run in WINE or CrossOver. No luck so far.

I use Gwenview to view photos and have it linked to The GIMP for any editing. Gwenview and The GIMP are a dynamite combination. :)

Adobe Photo Shop will run in CrossOver, but it is more than I would ever use (or want to know :) ).
