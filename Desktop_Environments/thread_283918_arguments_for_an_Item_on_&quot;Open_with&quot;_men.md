### Post by CatKiller on 2006-10-24
If you right-click on the file and select Properties, and then go to the Open With tab, you can add in a new way to open that type of file. There is a "Use a custom command" option, although you'd need to read the documentation of the application you want to use to find out exactly what the arguments would be for what you wanted to do.
