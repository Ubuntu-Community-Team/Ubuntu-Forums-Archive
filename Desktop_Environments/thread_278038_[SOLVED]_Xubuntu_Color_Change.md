### Post by aidanr on 2008-01-06
You can change the colours by editing the themes gtkrc file (eg. /usr/share/themes/Xfce/gtk-2.0/gtkrc). Although I'd recommend first copying the themes folder to ~/.themes and renaming the folder something else like "mymodifiedtheme" or whatever. You can use the gimp to find the colour values and the widget factory (sudo aptitude install thewidgetfactory) to quickly see the changes you made.
