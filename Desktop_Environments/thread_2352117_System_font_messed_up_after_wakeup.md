I tried to apply temporary fix by opening Tweak Tool and changing the interface font (Liberation Sans Regular was the default one, I changed to Liberation Sans Narrow Regular), it helps somewhat, but the problem resurfaces after some time again. Putting PC to sleep and waking up does not change anything, restart is the ultimate fix.
One thing that might be to blame is that I recently upgraded from 14.04 LTS to 16.04 LTS. Another non-ordinary thing in my setup is that I use a secondary display.
