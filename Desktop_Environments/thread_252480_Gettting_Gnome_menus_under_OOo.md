A friend on the OOoForum posted this in answer to a question I had:

> Another approach: if you're using Gnome, try enabling the "system" native dialogs (Tools > Options > General > Use OpenOffice.org dialogs (UNcheck this). This will cause OOo to use the gnome file dialogs. At the left side of those dialogs are "bookmarks" pointing to folders. The ones at the top left are automatically determined (e.g. your home directory). At the bottom left is a user-configurable set of folders. You browse and select a folder in the list on the right, then click "Add" on the left and you have a new bookmark. The bookmarks are persistent across sessions and apps. Clicking on a bookmark takes you to that folder and it should work for any location that you can browse to.
