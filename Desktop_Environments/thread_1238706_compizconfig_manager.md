### Post by AlexKpow on 2009-08-12
if you have the package compizconfig-settings-manager installed, then it should be under System > Preferences > CompizConfig Settings Manager. Alternately in the terminal just type in "ccsm".

To install it:
```
sudo apt-get install compizconfig-settings-manager
```
