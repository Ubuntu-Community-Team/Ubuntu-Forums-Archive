### Post by Jucato on 2007-05-06
"Can't find X libraries" means you need the X headers and development files, which is provided by the **xorg-dev** package.

If you encounter something like "Can't find Qt libraries", then this means you need the Qt headers and development files, provided by the **libqt3-mt-dev** package.

If you encounter something like "Can't find KDE libraries", you need the **kdelibs4-dev** (yes, that is for KDE 3.x) package.
