### Post by blackened on 2009-04-28
That's because those are GDM themes that effect the look of the login window only.

To install, go to System -> Administration -> Login Window. Go to the "Local" tab, add press the "Add" button. Navigate to the .tar.gz file you downloaded and press OK. You should then be able to select the Login Window theme you want to use from the list in the "Local" tab.

Hope this helps.

**Edit:** Sorry, the second link you gave does contain metacity themes after all, and you should be able to drag and drop them on to the Appearances dialog as long as the creator/packager arranged the archive correctly. Otherwise you can likely install them manually by extracting them to /usr/share/themes or ~/.themes.
