I want to know how I can change the gtk-themes in Openbox3.0.
I am using a pure openbox enviroment and want as less gnome/kde/etc dependencies as possible.

EDIT:
I found the solution :)
Just install gtk-theme-switcher. Now you'll be able to extract gtk1/2 themes in your ~/.themes folder and select them by running switch (gtk1.x) switch2 (gtk2) :-D
You have also the possibility to change your standard font ;) I'm using augie at the moment *SWEET*
