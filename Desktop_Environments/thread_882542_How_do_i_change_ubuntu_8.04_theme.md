### Post by axmasta on 2008-08-07
You could try saving the archive to your desktop, and then changing its file extention to .emerald.
Then use Emerald Manager to import the .emerald file.

If you click on the theme in the Emerald Themes window and you don't see a change, press alt+F2 and type > emerald --replace then press enter.

### Post by rainwalker on 2008-08-07
1. Download the .tar.gz file from the Gnome-look page
2. Save it somewhere simple, such as your Desktop
3. Right-click on the .tar.gz and click "Rename", and replace the ".tar.gz." part of the name with ".emerald"
4. Open the Emerald theme manager
5. Click the Import button and select the theme
