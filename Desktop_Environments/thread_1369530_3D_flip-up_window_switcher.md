### Post by jmonk86 on 2010-08-16
To get the 3D flip effect like aero in Windows.

First make sure you have all the compiz packages from the ubuntu software center including the extra plugin packs. In /system/preferences/ CCSM or Compiz Config Settings Manager... Click window management on the left side and check then click "Shift Switcher" then go to the appearance tab. Click on the drop down menu for switcher mode and select Flip. Then hit Super or the windows key ---put an ubuntu sticker on it--- and tab to check your effect. if you would like to make it alt tab like in windows go into key bindings and click the paper and pencil to type in the new combinations of key you would like to use. BUT IT IS EASIER TO JUST USE THE DEFAULT AND USE SUPER+TAB..
