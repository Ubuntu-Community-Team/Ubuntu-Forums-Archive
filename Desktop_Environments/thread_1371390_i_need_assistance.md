underquark's option is the first thing to try. If you are not satisfied, then, consider using KDE. You can go to System -> Admin -> Synaptic Package Manager and install kubuntu-desktop (this will take up 700+MB of HD space). Then you log out and from the log in menu you can select Gnome (the default for Ubuntu) or KDE (the more Vista-ish Desktop Environment).

Also, please don't write with all caps and please don't make generic thread names such as "Help" or "I need assistance", try something like "How to make Ubuntu look like Vista", it will be much easier for people to help you.
