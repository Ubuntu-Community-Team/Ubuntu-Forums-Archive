I don't know if this might be useful to you, but, since you mention that you removed Unity, I'll go ahead :D
I don't like Unity either, and I like Gnome Shell a bit, but I like more the traditional and now dead GNOME, so I first installed 12.04, then GNOME Shell and used that and Classic GNOME, but I was a little concerned to have to pay the Unity toll (and having all that crud underneath that I was sure I'll never use) just to get my system going (BTW, no problem at all Alt Tabbing)...

Currently I'm using that on a Thinkpad T60 (32 bits) and Linux Mint 13 on a T60p (32 bits).
I have no problem with Alt Tab on any of those setups.

Hope this helps.
