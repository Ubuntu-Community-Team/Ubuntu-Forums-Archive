### Post by weakhead on 2010-11-28
Breadcrumbs are available only with Nautilus-elementary, not vanilla Nautilus found in official repository.
If you're using Nautilus-elementary, then make sure you have breadcrumbs enabled (Preferences -> Tweaks -> Show like breadcrumbs).
