Okay I am AMD64 so I downloaded the source and managed to get it working. Once inside the kbfx folder type: ./configure --prefix=/usr ; sudo make ; make install. It will then be an applet that you can add to you task bar by right clicking your taskbar and going Add To Panel, Appelet, KBFX.

Enjoy!!!
