From what you are describing, it sounds like you want to just add a Firefox (or other) launcher to the application launch bar that already exists on the panel?

1 - Right click on panel, and select "Panel Settings".
2 - Choose "Panel Applets" Tab.
3 - Select the Application Launch Bar, then click the "Edit" button on the right.
4 - Under "Available Applications" in the right box, locate Firefox under Internet, and select it.
5 - Click the "Add" button.
6 - Use the "Up" and "Down" buttons to position the Firefox launcher where you want it on the launch bar (use "Applications" box on left as a guide).
7 - Close the dialog windows.

As to the trash icon on the desktop, I don't use one (relying on the file manager's trash disposal system instead) but looks like google can offer some possible ways.
