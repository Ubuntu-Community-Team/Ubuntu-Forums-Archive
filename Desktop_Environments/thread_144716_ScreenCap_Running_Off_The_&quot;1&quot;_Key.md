### Post by skymt on 2006-03-14
Look under System > Preferences > Keyboard Shortcuts > Desktop. I'm guessing you have the 1 key set to either "Take a screenshot" or "Take a screenshot of a window" (the latter would explain why the print screen key still works, as GNOME doesn't allow multiple keys for the same command).
