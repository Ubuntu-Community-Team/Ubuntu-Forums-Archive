### Post by kawabago on 2012-11-06
After the latest update to kernel -18-generic  #29  the window manager crashed and I couldn't access Unity at all.  I used Ctrl+Alt+F2 to get a prompt. Once logged in I tried

sudo apt-get remove nvidia-current
sudo apt-get install linux-headers-generic
sudo apt-get install nvidia-current
sudo shutdown -r now

to reboot and that fixed it for me.
