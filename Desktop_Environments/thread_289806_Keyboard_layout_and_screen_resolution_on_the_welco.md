### Post by mwob on 2006-10-31
Hi,

I figured out the keyboard thing, changed my xorg setting as follows:

Section "InputDevice"
	Identifier  "Generic Keyboard"
	Driver      "kbd"
	Option	    "CoreKeyboard"
	Option	    "XkbRules" "xorg"
	Option	    "XkbModel" "pc105"
	Option	    "XkbLayout" "gb"
EndSection

Still stumped about the welcome display/login screen though. I heard that you can hold Ctrl-Alt and +/- to change the resolution, but that doesn't change anything for me. This could be because I only specify one resolution - 1280x1024.

Anyone have any ideas?
