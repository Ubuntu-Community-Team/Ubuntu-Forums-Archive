### Post by logan2004 on 2005-05-19
yes

for example, if you'd like to replace the empty trash can icon for the "human"(default) theme, overwrite the following file with your own image

/usr/share/icons/Human/scalable/filesystems/gnome-fs-trash-empty.svg

it may be easier to find the theme you would like to use and copy it so you have two seperate themes.
