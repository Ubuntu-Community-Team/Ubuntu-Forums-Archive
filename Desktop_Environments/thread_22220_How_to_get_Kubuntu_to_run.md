### Post by digby on 2005-03-26
If you've already got KDE installed then on the login screen, select "session" at the bottom and choose KDE.  If you haven't installed KDE yet, from the terminal, run:

```
$sudo apt-get install kubuntu-desktop
```

Then follow step one.
