### Post by AppleCrow on 2006-08-22
Transmission is a GTK2 BitTorrent client, easy to configure, well integrated with GNOME Desktop, and can resume downloaded torrents.

It can be downloaded here: [http://acorbeaux.free.fr/packages]("http://acorbeaux.free.fr/packages")

These packages are for Ubuntu 6.06.

Latest build: 0.7 SVN 806 (2006.08.22)

Build based on snapshot from: [http://transmission.m0k.org]("http://transmission.m0k.org")

Included translations: ES, IT, FR, PL, RU
