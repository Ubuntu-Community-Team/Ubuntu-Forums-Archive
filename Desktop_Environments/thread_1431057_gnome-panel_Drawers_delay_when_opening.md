The drawers used to work really fast some Gnome versions ago, but in the recent versions it seems as if the icons aren't cached but instead read again and again each time you open a drawer. Because of that there is no setting that would make the drawers open more smoothly.

You could try searching if there's already a bug filed about it (most likely there is) and add yourself to the bug so the developers know that this really is annoying many people even though the drawers are one of the less-used features of Gnome. Then hopefully they'll fix the issue some day...

(and no, sadly it doesn't seem like there is any other drawer applet around. Amazingly few people seem to actually realize how useful the drawers are..)
