### Post by OpenTangent on 2010-07-18
I was experiencing a strange bug with mouse capturing where only the focussed window seemed to react to mouse clicks. Not even the close, minimise and maximise buttons on the border of the focussed window worked.

Solution: The problem was caused by the fact that i had changed the behavior of ALT+TAB using CCSM (CompizConfig Settings Manager). As soon as i set ALT+TAB back to application switching, the mouse bug disappeared. I have no idea why this would affect the mouse but i hope this thread will save someone the frustration i went through.
