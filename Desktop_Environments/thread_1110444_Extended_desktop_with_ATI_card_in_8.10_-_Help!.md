### Post by Pasdar on 2009-04-05
I don't have an ATI card in this PC, so I'm not really sure. But I'd like to know the answer to this because I have an ATI card in my laptop, which I want to install Ubuntu on too.

Maybe this will help you a bit > My NVIDIA card has a configuration app that can be run from System > Administration > NVIDIA X Server Settings. After that everything else is obvious, I select TwinView in configuration and as simple as that I have my desktop extended on the TV. The screen was black & white, I knew that had to do with the standard settings of the TV. I fixed that by manually adding "Option         "TVStandard" "PAL-B" " to /etc/X11/xorg.conf.
