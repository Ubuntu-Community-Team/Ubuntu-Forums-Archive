### Post by ykpaiha on 2008-12-15
To Play all kind of media on your playstation3 I found this great tool
PS3 Media Server
[http://ps3mediaserver.blogspot.com/](http://ps3mediaserver.blogspot.com/)

Just download the pms-linux here:
[http://ps3mediaserver.googlecode.com/files/pms-linux-1.00.zip](http://ps3mediaserver.googlecode.com/files/pms-linux-1.00.zip)

It's a mediacenter in Java scripts

Unzip on your desktop (right clic on it -extract)
make the "pms.sh" as exectutable (right clic > properties > permissions)
Copy under sudo in a folder (for me in opt)
create a link on your desktop pointing to pms.sh

install mplayer (see the readme for the codecs but usualy they are already installed with mplayer), mencoder, ffmpeg.

You can as well have a lot of possibilities in the options 

It worth to try it  after some mkv containers played I forget now the twonkymedia and others media player previously used.
