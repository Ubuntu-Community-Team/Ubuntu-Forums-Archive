### Post by rmrf on 2005-08-31
you can edit the user agent by going to: 

tools -> user agent switcher -> options -> options

then select 'user agents', and you can either create a new agent, or modify an existing one to say whatever you want. now just go to a windows box, and see what shows up when you visit whatismybrowser.com and paste that in the user agent config.
