Go to your Virtual Box settings.  What do you have set for 'network'?  I have not actually installed VB on Ubuntu host, but have managed to install Ubuntu guest on OS X host.  I just used "NAT" for networking in the settings and it worked just fine.

I'd say make sure your network is up and going in Ubuntu and that "NAT" is checked in VB network settings.
