### Post by fng on 2005-01-04
Disable it for your user only (default ubuntu-setup):

```
fng@butters:~ $ cd
fng@butters:~ $ touch .hushlogin
fng@butters:~ $
``` 

That should do the trick :)

### Post by fng on 2005-01-04
Disable it for all users :
```
fng@butters:~ $ sudo vi /etc/login.defs
``` 
Look for lines containing this text :
```
# If defined, file which inhibits all the usual chatter during the login
# sequence.  If a full pathname, then hushed mode will be enabled if the
# user's name or shell are found in the file.  If not a full pathname, then
# hushed mode will be enabled if the file exists in the user's home directory.
#
HUSHLOGIN_FILE  .hushlogin
#HUSHLOGIN_FILE /etc/hushlogins
``` 
Comment the first HUSHLOGIN_FILE line and uncomment the second.  Then it looks like this:
```
#HUSHLOGIN_FILE  .hushlogin
HUSHLOGIN_FILE /etc/hushlogins
``` 
Now add the empty file to /etc :
```
fng@butters:~ $ cd /etc/
fng@butters:/etc $ sudo touch hushlogins
fng@butters:/etc $
``` 
Restart sshd
```
fng@butters:~ $ sudo /etc/init.d/ssh restart
fng@butters:~ $
```
