### Post by wagb278 on 2010-05-30
Not sure if it will help, but --

I had the same problem as you. I resolved my problem by switching my Nvidia driver from the "Recommended" one to the older version 173.  That doesn't make sense to me, but it worked for me. 

After installing Ubuntu 10.04 (and before setting up the Nvidia driver) my display was fine, but the Nvidia card's fan was running at full speed and noisy. So I installed the Recommended Nvidia driver (which slowed the fan) but the frame was not showing right for my monitor, and the Notification area was gone - and I couldn't restore it. So I removed the "Recommended" driver and installed the version 173 (which is what I was running on Ubuntu 8.04 just fine. 

Now my screen looks as I want with the Navigation area showing.

If you are not using a Nvidia card and driver I have no idea how to help. 

I presume you have tried to adjust the vertical on the monitor to see if the Navigatin area is there - just off the bottom.

Good Luck! That is frustrating.
