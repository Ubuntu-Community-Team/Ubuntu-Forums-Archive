### Post by SGC on 2005-05-12
I'am not sure if I understand your question corectly , but ckeck the following option in Konqueror:
Settings > Configure Konqueror > Behavior 
then uncheck "open folder in separate windows"

If you want to change the tabs behaviour in Konqueror go to:
Settings > Configure Konqueror > Web Behavior
Uncheck "Open links in new tab instead of in new window"
