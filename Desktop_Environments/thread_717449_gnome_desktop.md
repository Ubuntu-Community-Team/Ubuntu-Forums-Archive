Hi! that's exactly why I have created the Clearlooks Compact theme. Some parts are still not as compact as I would like them, but it is much better than any other theme I have tried. 

You can also put the window in full screen mode (Windowskey + F) so that you use really the whole screen, this way you gain another 1-2 centimeters.
