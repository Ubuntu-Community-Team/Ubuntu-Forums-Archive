Can I set the Opacify function in CompizFusion to not work for Wine Programs (or any program, for that matter)? I figured if there was a way to indicate in Compiz to not Opacify a program, then I could either get it to not Opacify Wine programs in general, or that specific program.

Any suggestions?

### Post by days_of_ruin on 2009-05-16
The opacify plugin has a "Window Match" entry that specifies windows
that can be opacified. Click on the add button select Window Title for type
and check the invert box. Then click on grab and click on the wine window when its running the program that you don't want to opacify.Click ok and that should do the trick;)
