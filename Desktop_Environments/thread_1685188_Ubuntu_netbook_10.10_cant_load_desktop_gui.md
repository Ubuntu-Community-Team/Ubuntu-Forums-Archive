### Post by Krytarik on 2011-02-11
Try this:

Log into "Ubuntu Desktop Edition", then check if there are panels now. Whereas the "Netbook Edition" doesn't have panels, right?

If that's the case, then check in "System -> Preferences -> Startup Applications" if "Maximus Window Management" and "Netbook Launcher" are in there and enabled. If so, disable those and re-login.
