### Post by anthw27 on 2009-08-21
Easiest way to install murrine is to use the synaptic package manager.

Open Syanptic and then in the search bar type ..Murrine

Mark the the following.

murrine-themes
gt2k-engines-murrine

Click on Apply in the synaptic manager and watch it install

hope that helps
