### Post by chilker on 2007-08-27
I just figured it out! This was so random! I hope this can help anyone forever! Go into a saving window (for firefox, there's "save page" in the file menu and in open office, act like you're about to save a document, even with nothing on it) then in the little field that is empty, where you would type your document's name, right-click and go down to Input Methods and Choose Japanese. with that, you should be able to type &#26085;&#26412;&#35486; *anywhere* on that program, as long as it's open. You may have to do this every time you open the program (hopefully there's a workaround i haven't yet figured out) but it works! I can now type in any other field. I'm so excited!
