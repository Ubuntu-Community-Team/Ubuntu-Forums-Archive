### Post by Dennis N on 2023-09-26
There are no tray icons in Gnome. There is of course the power button on the panel which requires several more clicks to get powered off. Here is an alternate way which uses Gnome's search capability to give you a safe power off.

Press Windows Key to open Activities
Type Power in the search box
The Power Off Icon shows. 
Press Enter, or Click the icon.
See attached image

(If more than one icon appears as a search result, select the Power Off icon first)
