### Post by Emerzen on 2005-10-19
I'm not sure how well known the following is, but I think it's a good resource for the community.  A group known as "PLF - Penguin Liberation Front" has setup a repository in France for Ubuntu.  This repository contains packages that are available for free (jre for example) but unable to be in official repo's for legal reasons.  Here's the web page w/ the apt lines to add to sources.list for this repo.

[http://wiki.ubuntu-fr.org/doc/plf](http://wiki.ubuntu-fr.org/doc/plf)

**Warning** they have moved the repository several times in as many days.  So, keep an eye on the website for updated location of repository.
