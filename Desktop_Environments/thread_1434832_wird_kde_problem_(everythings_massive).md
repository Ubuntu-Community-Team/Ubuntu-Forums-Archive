### Post by my_wan on 2010-03-20
To get your window sizes reduced so you can close them:
Right click the menu bar of the program on the desktop panel and select 'Maximize'.
Close the window normally.
Now when the window reopens it'll be in the same maximized state and can be repositioned normally.

Edit: Just noticed in Control Center-->Windows that you can hold the Alt key and grab to move a window.
