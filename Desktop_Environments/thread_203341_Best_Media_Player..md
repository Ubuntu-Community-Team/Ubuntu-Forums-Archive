### Post by gingermark on 2006-06-25
[QUOTE=Hairy_Palms]try amarok then if u want an itunes clone, add this line to your sources in synaptic
instead of the amarok in the default repos[/QUOTE]

amraroK probably is the best Linux media player, because it can either be used as an "all bells & whistles" player, with lyrics, context-specific info etc, or it can just be used as a music player similar to xmms (after turning some options off) with the internal file browser then being very useful (I use it the second way).

But it isn't a Gtk app, it isn't for GNOME. [Exaile](http://www.exaile.org/) is the closest thing to amaroK for GNOME, but it is in early development.
