I assume you know that ~/.gdesklets/displays is not the same as ~/.gdesklets/Displays!!!  ~/.gdesklets/displays is a file that holds the information about the applets running on your desktop while ~/.gdesklets/Displays is a directory that contains applets regarding Displays...
