### Post by xyphur on 2009-11-19
MDI is the Micorosoft Document Imaging format, a proprietary file format based on TIFF compression technology (but not compatible with it, obviously). Unfortunately, this format can only be opened by Office 2003 (and Office 2007 with the use of a trick), or by using the MDI2PDF converter ([http://www.bugysoft.com](http://www.bugysoft.com)).

There's one small problem with MDI2PDF though, from what I've read someone else tried using it under Wine and had problems with it crashing, so you're likely going to need to run the conversion on a real Windows box.

Alternatively, you could request that your insurance company send you a PDF instead, as there is an add-in for Microsoft Office for saving to PDF (which is what they'd likely need since they're using Office to save the MDIs).

Best of luck.
