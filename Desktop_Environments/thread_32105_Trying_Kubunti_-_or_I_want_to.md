### Post by Terracotta on 2005-05-06
if /home is a mounted partition/hard drive, you can say not to reformat the drive to keep de data :-), but you definitaly need a spare hard drive that can be formatted for the base system. There's no such thing as qt-parted in the install, so you won't be able to resize the partitions you already have without destroying data. To get qt-parted you can download the mepis live-cd and run qt-parted, resize your partition on which you have your /home directory installed, or resize another partition, to get free space for Kubuntu.

PS, maybe you can install qt-parted in mandriva, I don't know,  it would be quite silly to download a complete cd just to get qt-parted, though not the smartest one, but it's a solution, that should definitaly work.
