### Post by yager on 2006-05-04
Another alternative for printing anything to a PDF would be to install the cups-pdf printer.  Of course aysiu's answer is perfect for exactly what you are asking for.

System>Administration>Synaptic Package Manager, search for cups-pdf and install.  Check the web page listed in the description for instruction on how to configure it in cups and then you can print anything to a pdf file from the print menu.
