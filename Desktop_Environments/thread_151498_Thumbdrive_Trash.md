### Post by DirtDawg on 2006-03-28
Whenever I use my thumbdrive with Hoary, when I delete anything off the thumbdrive and into the trash, a file remains, usually called something like ".trashes-dirtdawg", that's the total size of anything I tossed. This ghost file does not appear in Hoary.

For example, if I remove 113MB of information, ".trashes-dirtdawg" will be 113MB. Once I plug the drive into Windows XP, I can throw the file away normally at which time the space is recovered.

I'm using Hoary on a G3 iMac, the thumbdrive is a memorex 256MB. 

Any ideas? Thank you.

### Post by WakkiTabakki on 2006-03-28
Yes, that is a little annoying (since throwing something away from the thumb-drive generally means you want to free up space...)
The folder is the trash-can, just as you have one in your home-folder. 

Any file/folder in *nix starting with a dot is hidden. To remove the trash folder, hit ctrl-h in nautilus to show hidden files (or go into the menu View(?) / Show Hidden files), which'll show the folder just as in windows and then simply delete it...
