### Post by aysiu on 2005-12-06
[QUOTE=Burgresso]Hello everyone:) !

I'm curious about installing xfce - but I was wondering - if I do, it is true that I have the ability to decide which desktop I want at startup? (gnome/xfce?)[/quote] Not at startup. At the login screen. Click on "Session" and select XFCE or Gnome.

> 
Will installign xfce mess with other programs in/for gnome? No, but it may add a couple of programs to the Gnome menu.

> 
Lastly, if the answer to the first question is yes, how do I install and set it up? 

	\\:D/   :-({|=  \\:D/ See the first link in my sig, then ```
sudo apt-get install xubuntu-desktop
``` Alternatively you can System > Administration > Synaptic Package Manager and find the XFCE4 packages you're interested in.
