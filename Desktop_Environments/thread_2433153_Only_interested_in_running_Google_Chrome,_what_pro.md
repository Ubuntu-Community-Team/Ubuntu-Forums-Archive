### Post by Frogs Hair on 2019-12-14
The system related options in Chrome browser are in advanced settings and include disabling back ground applications and hardware acceleration. Being signed into Google services all the time may have an effect battery life also. As for Ubuntu you might consider reinstalling and choosing the minimal option which removes many applications included in the normal installation. I found there was less memory use and any system monitoring could be done through the terminal anyway. If using Google services you won't need an office suite.
