### Post by grahammechanical on 2012-12-07
I read what you are saying but I do not know what you are talking about.

Select a document. Right click and select Copy. Open the partition that you want to copy the document to. Find the folder that you want to put the document into. Select the folder and right click and select Paste into folder. Use File Manager.
