### Post by Irihapeti on 2019-02-07
You would be better off exporting the calendar as .ics (ical), as that's a standard calendar format. According to the official Gnome help page ([https://help.gnome.org/users/evolution/stable/index.html.en](https://help.gnome.org/users/evolution/stable/index.html.en)), you can import ical files into Evolution.
