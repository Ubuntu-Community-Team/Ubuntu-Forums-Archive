### Post by FrostDust on 2009-03-22
> **abhilashm86 said:**
> oh k fine, my distro is ubuntu 8.04 hardy heron, so if i completely remove compiz and install latest version, will it work?
is there any possibility to update my compiz?

Simply put, it won't install the latest version in 8.04; it's marked as incompatible. You could either upgrade to 8.10, and it will auto-update to the correct version, or install a newer version from here:

[http://archive.ubuntu.com/ubuntu/pool/main/c/compiz-fusion-plugins-extra/](http://archive.ubuntu.com/ubuntu/pool/main/c/compiz-fusion-plugins-extra/)

You'd want to install either "[compiz-fusion-plugins-extra_0.7.8-0ubuntu2_i386.deb]("http://archive.ubuntu.com/ubuntu/pool/main/c/compiz-fusion-plugins-extra/compiz-fusion-plugins-extra_0.7.8-0ubuntu2_i386.deb")"or "[compiz-fusion-plugins-extra_0.8.2-0ubuntu1_i386.deb]("http://archive.ubuntu.com/ubuntu/pool/main/c/compiz-fusion-plugins-extra/compiz-fusion-plugins-extra_0.8.2-0ubuntu1_i386.deb")". Replace the "i386" with "amd64" if you're using a 64-bit AMD processor.

Also, keep in mind there's a reason the newer versions aren't compatible with 8.04: it's more likely to crash on you, or otherwise not work properly.
