### Post by 3Miro on 2010-01-02
1. Not all windows programs run under wine and many programs need careful setup.

2. You can try to install the latest version of wine, I think the one in 8.04 repository is wine 1.0, right now most people use 1.35 or later.

3. Check out the winehq forum ([www.winehq.com](www.winehq.com) and/or Google winehq appdb). There is information for many programs that need special care to run. The Ubuntu forum is not the best forum for wine questions.

4. If the program still doesn't run, you may consider VirtualBox. That will let you install a copy of windows XP and run it form within Ubuntu.
