### Post by Mordac85 on 2008-03-14
Whichever one you build your system with, Ubuntu or Kubuntu, just install the other Desktop Environment/Window Manager.  At your login screen you can select which one you want.  Unlike Windows, you're not restricted to the single window manager that came with the OS.  In linux you can change that as well.

imho, I'm moving from Windows and I find the transition much easier in Kubuntu.  My wife and kids like KDE more as well for about the same reasons.  However, since I'm a die hard geek I like Gnome from time to time, depending on what I'm doing.  It's just that on my home box I'm more recreational and gravitate to KDE more often than not so I use Kubuntu.  It just really depends what you're going to do and which one you feel more comfortable with.
