### Post by mgmiller on 2007-03-25
Go to System>Administration>Log in Window.  Click on the Security tab and put a check in the Enable Automatic Login Box.  Lastly, just add your users in the drop down list next to users and give it the passwords for each if it asks.

That being said, I'm not sure if it will do exactly what you want.  It may only just log in one person automatically.  This really is one of the security features that helps make Linux more secure than WindowsXP and you really shouldn't try to get around passwords so much.  Doing as you suggest will give all 4 of your users access to each others desktops and settings.  Do you really want that?
