Hi all

This post will only be of interest if:

- you are using XScreenSaver instead of the regular screen saver
- you have the XScreenSaver plugin "GLPlanet" installed
(although perhaps the regular screen saver has a similar plugin)

I currently run "precise", however this probably applies to other versions of Ubuntu.

So.  I was thinking I would like the GLPlanet screen saver to display a planet other than Earth.  This means finding texture files for other planets.  I was annoyed when reading the description of GLPlanet when it stated:

"Draws a planet bouncing around in space. The built-in image is a map of the earth (extracted from `xearth'), but you can wrap any texture around the sphere, e.g., the planetary textures that come with `ssystem'."

"ssystem"? What is that?

After some googling, I realized that ssystem had been superseded by "Celestia" (an excellent astronomy simulation, btw).  I already had Celestia on my system.  You can find the textures for the planets (and other objects) in "/usr/share/celestia/textures", in three separate sub-directories: "lores" "medres" and "hires".  Pick a texture from the GLPlanet settings dialog by hitting the "browse: button then navigating to the aforementioned "/usr/share/celestia/textures".  Almost everything you want is probably in "medres".

Even better textures are available at [http://www.celestiamotherlode.net/](http://www.celestiamotherlode.net/).

If anyone has the chops to make GLPlanet use a random texture (perhaps stating on the screen which it is) you would impress me immensely (hint hint)
