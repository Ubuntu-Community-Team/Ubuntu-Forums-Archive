Quick question: when upgrading kernel from default i386 to K7 - via the package manager after completing a fresh installation of Breezy - is it necessary, or better, to remove the i386 kernel components listed in the package manager... or leave the i386 kernel items be, just select the K7 components for installation so they're both installed, and leave it go at that?

---

### Post by RAOF on 2005-10-26
You can happily leave the i386 kernel and everything associated with it.  In fact, I'd recommend leaving it until you've run the new kernel a couple of times & made sure it actually works (I'm just paranoid :))  Unless you're running out of disc space, having a backup kernel is not a bad idea :)
