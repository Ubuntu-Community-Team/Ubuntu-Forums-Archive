### Post by crdlb on 2011-10-20
You can learn more about the 'fallback' session here: [https://help.ubuntu.com/community/GnomeClassic](https://help.ubuntu.com/community/GnomeClassic). It is possible to theme this session, but GTK3 has a new theme system.

P.S. Are you using the binary ATI driver (fglrx/catalyst)? If so, that is why GNOME Shell isn't working. The builtin radeon driver will run the Shell if it supports your GPU, but that's a tradeoff.
