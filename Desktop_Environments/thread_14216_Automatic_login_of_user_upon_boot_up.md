### Post by Perfect Storm on 2005-02-05
Yep.

'Computer' tab ----> 'System Configuration' ----> 'Login Screens Setup'

Under the 'General' tab there's something called 'Automatc login'

### Post by Sye d'Burns on 2005-02-05
Computer -> System Configuration -> Login Screen Setup

Automatic Login is right there, just enter the name of the user and click the login automatically option and you're all set
