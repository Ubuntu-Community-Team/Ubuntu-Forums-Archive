### Post by trippinnik on 2008-04-09
1. .kde4 there are a lot of directories there, probably the settings are all in conf, but don't delete the whole folder unless you want to reset the settings for all the kde4 programs.
2. If you really want to use compiz in KDE4 you can use compiz --replace at the command line or put it in a startup script.  Kwin the WM for KDE also has it's own effects although your not going to get all the crazy stuff yet and it still seems a bit slower.
3. right now nothing like avant window manager, but you can look around for plasmoids, i'm sure there will be something like that in the future, but it's pretty early yet.
