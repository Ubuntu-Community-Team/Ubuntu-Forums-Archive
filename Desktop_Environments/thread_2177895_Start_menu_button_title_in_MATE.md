### Post by Kirk Schnable on 2013-09-30
I'm running Mint with MATE on my laptop.  When I read your question I thought it was fairly simple and that surely you had to be overlooking something.  But then I realized I'd never tried to rename my applications menu, and I was surprised to see that there really was not an option to do so in the GUI.

I did a bit of Googling as well.  It looks like you may need to edit the name in this file: [B]/etc/xdg/menus/debian-menu.menu

[/B]See this page which I found in my Googling attempt: [http://www.linuxine.com/story/how-edit-f18-mates-menu](http://www.linuxine.com/story/how-edit-f18-mates-menu)
