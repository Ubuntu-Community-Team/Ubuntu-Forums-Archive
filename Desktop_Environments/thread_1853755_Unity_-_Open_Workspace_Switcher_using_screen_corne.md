### Post by Huskehn on 2011-10-02
Hey all, I'd like to be able to open the Workspace Switcher by moving the mouse to a corner of the screen. I've installed brightside, but I don't know the terminal command for opening the workspace switcher. I've tried searching and couldn't find anything. Any help would be greatly appreciated.

---

### Post by Copper Bezel on 2011-10-02
You don't need any extra software or any terminal commands to do this. The workspace switcher is a Compiz plugin, called Expo. You can set it to an edge by just opening up compizconfig-settings-manager, clicking the Expo plugin, and setting the "Expo Edge" value.
