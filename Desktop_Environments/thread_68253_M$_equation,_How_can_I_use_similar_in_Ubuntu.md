### Post by parktownprawn on 2005-09-23
if you want to insert equations into OOwriter in a similar way to MSword try:

Insert->Object->Formula

(If you want to produce a document with lots of formulae it might be worthwhile to take the time to learn how to use latex - 
try [http://www-h.eng.cam.ac.uk/help/tpl/textprocessing/](http://www-h.eng.cam.ac.uk/help/tpl/textprocessing/)
also [https://wiki.ubuntu.com/LaTeX](https://wiki.ubuntu.com/LaTeX) - has some 
links which may be a good starting place)
