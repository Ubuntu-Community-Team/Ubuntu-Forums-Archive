### Post by liamm1993 on 2008-01-21
I have had this problem for a while now. The only way I can see of getting round it is to use the Jabber, but to be able to do this you will need to do the following...

In the Kopete window, go;
Settings -> Configure
Click the Accounts tab on the left hand side.
Click 'New' on the right hand side.
Select 'Jabber' followed by 'Next'.

_Basic Setup_
Jabber ID: Put your MSN email address (addy)
[INDENT]If you wish for Kopete to remember you password then click 'Remember Password' and type your password in the field bellow you.[/INDENT]

_Connection_
Leave 'Use protocal encryption (SSL)' unchecked.
Cheak 'Override Default Server Infomation' and fill in the follow...

Server: messenger.hotmail.com          Port: 1863

_File Transfer_
I have not yet worked out the Server and Port for File Transfer but you can try the same as above. As soon as I find out I will let you know.

_Privicy_
You can keep this tab as it is.
