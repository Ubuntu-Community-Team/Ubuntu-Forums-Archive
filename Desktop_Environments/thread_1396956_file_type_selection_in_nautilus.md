I already have nautilus action config to allow the "enqueue in audacious" operation for all my music files. However I have some folders that contain a mix of music files, album covers, and various other files. Ctrl-A lets me select all, but how can I add a keyboard shortcut to select all of a particular file extension? ### Post by ajgreeny on 2010-02-02
I don't think you can do that with a keyboard shortcut, but I may be wrong.

What you can do, however, is click on the Type column in nautilus to sort by file type, then simply use the mouse and shift to select all of one type, which will now be sorted into a group, all together, rather than alphabetically from a to z.
