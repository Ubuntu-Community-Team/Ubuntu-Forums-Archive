### Post by aysiu on 2006-03-07
[QUOTE=CarbineReloaded]
-Get a system status on my desktop/background... I've seen it done, but I'd like to monitor my NIC/Processor % and Ram Free[/quote] Applications > Accessories > Terminal ```
sudo apt-get update
sudo apt-get install gdesklets
```

> 
-How do I get my home folder and "computer" folder to show up on the desktop rather then just "places" -- that was one thing I loved about FC4. Applications > System Tools > Configuration Editor > Apps > Nautilus > Desktop > Home_Icon_Visible
