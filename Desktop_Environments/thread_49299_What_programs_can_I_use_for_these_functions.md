### Post by badrinarayan on 2005-07-15
In addition to tikal26's suggestion (which uses KIO-slaves), see this tutorial to fine tune how you want to rip: [http://www.antrix.net/stuff/kde_audiocd/](http://www.antrix.net/stuff/kde_audiocd/)

Similarly ftp can be done (just like in windows explorer) by directly copying the path on Konqueror's address bar. Konqueror goes further and supports fish (over ssh), sftp and many more.

But if you insist on a separate ftp client, I would recommend KBear (Ignore Grip and Gftp if you want to stick with KDE apps)
