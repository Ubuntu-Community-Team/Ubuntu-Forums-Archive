### Post by pdpi on 2005-07-02
poptones: Try using Ctrl+Alt+D. Quite Handy. Plus, you have a panel icon for show desktop. If you're anything like me, and a bit fixated on usability issues, you'll stick that (and other oft-used things) to a corner on the screen. I have mine on the bottom right. I just throw my mouse there and click, and voila, desktop is shown. Unlike windows, this is a full toggle, not just for the last open window, and it also works visibly like a toggle button, so you visually can tell its status.

In terms of what you put there, I'm not yet sure. But the idea would be not really having anything on the desktop except drive mounts (possibly not even those), and make a navigateable desktop, similar to symphonyOS. I was also thinking of keeping some statistics there, and useful info. If I can carefully customize it so that I can find all the system info I want with a couple of clicks, all the better. For example, du, df, ifconfig output would be nice to get from a convenient location like the desktop
