### Post by mjm115 on 2007-01-17
Install kubuntu-desktop.  This will install the kde desktop and all packages that come with kubuntu.  Installing kdebase only installs the base kde libraries, but does not install the kde desktop.

Hope this helps.

### Post by x1a4 on 2007-01-17
Try to stay away form the meta packages if  you can.  I've recently installed the kubuntu-desktop meta package on XUbuntu and things just didn't work right.  Install KDE one package at a time, this way you'll be able to catch any problems that arise quickly and know what caused them.  Also, your system won't be cluttered with apps you won't use.
