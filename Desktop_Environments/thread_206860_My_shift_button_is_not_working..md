For instance, if I do not have caps lock on and I try to type a capital letter by pressing shift + the letter nothing is typed, not even a lowercase letter. Also, with caps lock off I can not type any characters like $ because when I type shift + 4, just like with capital letters, nothing happens. However with caps lock on if I type shift + 4 I do indeed get a $ and if I press shift + a letter it types a lowercase letter ( which is normal because I have caps lock on ).

Another interesting thing to note is that if I press ctrl + alt + F1 to go to a terminal shift works fine in all situations ( with or without caps lock on ).
