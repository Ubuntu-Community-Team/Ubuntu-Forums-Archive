### Post by Krytarik on 2011-02-23
What do you exactly mean by "multi keyboard"?

You can set up the mentioned layout in "System -> Preferences -> Keyboard -> Layouts -> Add... -> By language".

If you have set up more than one layout, a layout selector/indicator automatically appears in the notification area of your panel.
