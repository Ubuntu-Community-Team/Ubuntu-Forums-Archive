### Post by 2F4U on 2012-03-11
Of course, the decision is up to you, but I would install from the regular Xubuntu desktop cd. That way you most likely get a working desktop environment at the least effort. The easiest way to play around with the alternate install cd is probably in a virtual environment, since then you don't have to struggle with repartitioning your hard drive. It is even possible to explore different installation options by using different virtual machines.
What I would certainly not recommend is to install from the default Ubuntu desktop install medium and then add the xubuntu-desktop package.
