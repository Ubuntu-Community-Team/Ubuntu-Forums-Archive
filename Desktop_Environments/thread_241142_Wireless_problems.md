### Post by philbridges on 2006-08-22
I'm still a newbie and had a similar problem but resolved it by installing NETWORK MANAGER in Add/remove programs.  This puts a new icon near the date/time/exit button on the toolbar and lets you pick the correct WiFi network, enter a WPA key etc. not unlike Windows XP zero config routine.  

So far Ubunto has run every WiFi card I've thrown at it just relying on auto-detrct, ditto webcams, this OS just plain rocks!
