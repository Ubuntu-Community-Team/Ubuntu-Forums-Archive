### Post by Copper Bezel on 2011-09-28
I find Compiz quite usable, but this is as I said a weakness of Compiz's "viewports" in contrast to the "workspaces" in Gnome 2's Metacity and Gnome 3's Mutter.

If you find this style of workspace management unusable, and particularly if you don't care about eyecandy, switch to Metacity or Gnome Shell instead (or Unity 2D + Mutter, which is actually quite nice.) Depending on your Ubuntu version and which shell you want, the process will be different, so I can't give you a blanket answer without some more specifics. (If you're using the classic Gnome 2 interface, you can run [font="Courier New"]metacity --replace[/font] and see if Metacity will work for you.)
