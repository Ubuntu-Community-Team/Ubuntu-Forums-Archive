1) No, it's not possible to run Compiz-Fusion on enlightenment17, they are both window managers. You can use Enlightenment17 instead of Metacity and Compiz-Fusion on Gnome though but you can't have 2 window managers running at the same time.

2) Here are some links [get-e]("http://www1.get-e.org/"), [e17-stuff ]("http://www.e17-stuff.org/").

3) Yes it is, you can configure the mouse and other stuff inside the configure module. Don't know why you should do that though, that menu is kind of importent, I use it instead of a systray amongst other things.

4) My experience with e17 is good, in fact I love it but it has some bumps and corners, older themes sometimes breaks if you update you e, the 'favorite menu' is hard to configure like I want with subcatagories for applications.
