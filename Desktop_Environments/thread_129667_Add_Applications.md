### Post by taurus on 2006-02-14
If you want to install something, go for the binary because with source, you need to compile it before you can run it...  However, synaptic is what you want to use if you want to install something besides apps that are already on your system.

sudo apt-get install packagename
``` in terminal and it will be downloaded and installed, there are functions to search for packages either (apt-cache search).
But it's just what you like the most;)
