### Post by void1 on 2006-11-17
Hello and thanks for the help.

How do I swap the keyboard locations of the Caps Lock key and the Ctrl key in Kubuntu 6.10 Edgy?

In Ubuntu 6.10 Edgy, I do the following:
System->Preferences->Keyboard
Layout Options->Ctrl key position->Swap Ctrl and CapsLock.

Unfortunatley, I can't find this option in Kubuntu 6.10 Edgy.

I've found instructions on how to do this for Ubuntu ([http://ubuntuguide.org/wiki/Ubuntu:E...r_Control_key)](http://ubuntuguide.org/wiki/Ubuntu:E...r_Control_key)), and Xubuntu ([http://ubuntuforums.org/showthread.php?t=247470)](http://ubuntuforums.org/showthread.php?t=247470)), but not for Kubuntu; and the Ubuntu/Xubuntu instructions don't work for Kubuntu 6.10.

So how do I swap the locations of the caps lock and ctrl keys in Kubuntu 6.10 Edgy?

Thanks.

### Post by TheWizzard on 2006-11-17
1.Menu -> System Settings
2.Personal -> Regional & Accessibility 
3.choose from the left panel: Keyboard Layout and select tab: Xkb Options 
4.check &#8216;Enable xkb options&#8217;  
5.scroll down to &#8216;Ctrl key position&#8217; and select "Swap Ctrl and Capslock' or &#8216;Make CapsLock an additional Ctrl&#8217;. 
6.apply

source: [http://anticapslock.com/](http://anticapslock.com/)
