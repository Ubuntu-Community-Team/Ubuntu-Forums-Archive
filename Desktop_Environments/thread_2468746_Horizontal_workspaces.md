### Post by Dennis N on 2021-11-09
Ubuntu 21.10 has Gnome 40 and it's horizontal workspaces. No extension is needed. Using Gnome 40 is the best way to experience the horizontal workspaces.   

The current upgrade path is:
20.04 &#8594; 21.04 &#8594; 21.10 

When 21.04 goes out of support in about 2 months, this path becomes available:
20.04 &#8594; 21.10 in one upgrade step.

On release of 22.04 next April, this path becomes available.
20.04 &#8594; 22.04
