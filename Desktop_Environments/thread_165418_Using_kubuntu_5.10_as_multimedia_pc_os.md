### Post by briguy on 2006-04-26
There is of course MythTV which you can install from the repositories.  However, if you are going to be using the PC exclusively for media, I would recommend installing KnoppMyth which is a linux distribution solely designed for making MythTV easy to set up.  When I first tried to set up MythTV, I tried using breezy because I thought my experience with Ubuntu would help.  I had lots of trouble (you may not).  Knoppmyth was painless.
