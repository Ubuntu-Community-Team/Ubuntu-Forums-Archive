### Post by Deathmoon on 2006-11-29
At the login screen you should see an options menu (or press F5 --- or is it F10) and this will bring up a menu).  Once you click KDE (or Gnome) it will ask if you want to make this your DEFAULT session or use it just for this time.

Of course, you have to have the KDE, etc. installed...

### Post by aysiu on 2006-11-30

GDM will prompt you for making a desktop environment the default when you try to log into it.
