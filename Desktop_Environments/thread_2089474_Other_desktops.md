### Post by raja.genupula on 2012-11-29
yeah I understand . you dont want Unity and want Gnome style .

ok you can have a package like gnome-fallback session to get that 
open your terminal and type this 

```
sudo apt-get install gnome-session-fallback
```

and from login screen you can choose the desired DE . for more information follow this 

 [http://complete-concrete-concise.com/ubuntu-2/ubuntu-12-10/ubuntu-12-10-installing-gnome-session-fallback](http://complete-concrete-concise.com/ubuntu-2/ubuntu-12-10/ubuntu-12-10-installing-gnome-session-fallback)
