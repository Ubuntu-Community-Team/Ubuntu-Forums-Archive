### Post by Krytarik on 2011-09-24
Presuming that you

[LIST]
[*]installed Natty 11.04,
[*]have seen the classic Gnome interface when running the LiveCD,
[*]and are now presented with the Unity interface (possibly after installing proprietary or experimental drivers),
[/LIST]
choose "Ubuntu Classic" as the "Session" option at the bottom of the login screen, after clicking on your username, but before entering your password.

If you have auto-login enabled, you obviously need to log out to do so, after being auto-logged in. If you have passwordless login enabled, you need to disable it before via "Users and Groups".
