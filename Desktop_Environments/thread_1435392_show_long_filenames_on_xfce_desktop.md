### Post by David Andersson on 2010-08-03
In Settings>Desktop>Icons you can 1) increase the "Icon size" and 2) enable and decrease "Custom font size". The more you change the numbers, the longer file names will be shown. But there will still be a limit where names are truncated. (Think of this as a workaround until you find the real answer.)
