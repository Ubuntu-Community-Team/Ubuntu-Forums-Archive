### Post by hurtstotalktoyou on 2011-12-11
I used the older version of compiz-fusion-plugins-unsupported found [here](http://xtrabuntu.blogspot.com/2011/08/enable-snow-on-ubuntu-desktop-using.html).  Note, however, that I am running 10.04 64-bit.

So I recommend the following steps:

(1)  If you have not already done so, go to Synaptic and install compizconfig-settings-manager.

(2)  Uninstall compiz-fusion-plugins-unsupported using Synaptic.  (Skip this step if you have not previously installed it.)

**(3)  Download and manually install one of these [32-bit](http://ubuntuguide.net/wp-content/uploads/2009/11/compiz-fusion-plugins-unsupported_083git20090911-1ubuntu1_i386.deb) or [64-bit](http://ubuntuguide.net/wp-content/uploads/2009/11/compiz-fusion-plugins-unsupported_083git20090911-1ubuntu1_amd64.deb) old versions of compiz-fusion-plugins-unsupported.**

(4)  Check the snow box in CompizConfig Settings and press SUPER-F3 !

(recall that the "super" key is the one with the microsoft windows symbol on it)
