### Post by grahammechanical on 2014-08-08
You don't. Not on Ubuntu with the Unity user interface. All that menu stuff is now taken care of in the Dash with its scopes and filter results function. Now it all depends upon the version of Ubuntu that you are using. If it is Ubuntu 14.04 then you will find in the software centre something called gnome session flashback. Install that and at the login screen there will be two additional options, Gnome Flashback (Compiz) and Gnome Flashback (Metacity). Both options will give you that user interface look that I think you are referring to. But those user interfaces do not have the look of Unity. You have to log in to Ubuntu to get that again.

Regards.
