### Post by grahammechanical on 2014-09-05
In Ubuntu 14.04 + Unity we do not add or remove launchers from the top panel. Certain applications will provide an indicator that will be put in the top panel.

System Settings>Keyboard lets us customize short cut keys. And this wiki page shows how to create a desktop launcher in Ubuntu 14.04.

[https://help.ubuntu.com/community/UnityLaunchersAndDesktopFiles](https://help.ubuntu.com/community/UnityLaunchersAndDesktopFiles)

If you are using Gnome Flashback (Compiz) then you can install Compiz Configuration Settings Manager and also Compiz Plugins to make modifications.

If you are using Gnome Flashback (Metacity) then I can offer no advice or instruction. I have Gnome flashback installed but I do not use it exept as a fall back UI if Compiz gets broken.

Regards.
