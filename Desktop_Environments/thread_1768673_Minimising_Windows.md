### Post by Steve! on 2011-05-27
Hello,

I have just upgraded to 11.04 and I'm using the Ubuntu classic view. I have a panel at the top of my screen which just contains the window list widget to flick between open applications.

When I swipe my mouse on the top right hand corner of the screen it automatically minimises the current window I am working in, or if I am just looking at my desktop it restores whatever the last active window was. This is getting a bit annoying - is there anywhere that it can be turned off? I have checked compiz and ubuntu tweak but I cannot find the relevant setting.

Any thoughts please?

Many thanks.

---

### Post by Krytarik on 2011-05-27
Apparently, "Show Desktop" is set to the edge "TopRight" in "CompizConfig Settings Manager -> General Options -> Key bindings".
Disable the binding or set it to another edge.

Greetings.
