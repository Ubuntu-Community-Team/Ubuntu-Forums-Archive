Eventually, I found that the driver was the culprit. Wiki's automatic script had installed a driver *too* current for my setup. So I did this:

```
sudo aptitude install nvidia-glx-legacy
sudo aptitude install nvidia-settings
sudo aptitude install-xconfig
sudo nvidia-glx-config enable
```
