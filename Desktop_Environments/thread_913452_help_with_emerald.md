### Post by TheUbGuy on 2008-09-08
If you installed emerald, then under System->Preferences you should have the Emerald Theme Manager. If you haven't already done so, install CCSM (Compiz Config Settings Manager). Just go to Synaptic Package Manager and search for compiz. You'll see the package compizconfig-settings-manager. Once it is installed, go to your Systems->Preferences and Advanced Desktop Effects Settings (which is the Compiz Confi Settings Manager). Go to Effects, then Window Decorations. Open it, and under Commands enter emerald --replace. This will make sure that emerald is run every time you log on.
