### Post by Gillingham on 2007-03-20
xorg.conf is one place it can get the resolutions from.  But, from some of the problems I have had another source is the monitor itself.

Have you tried replacing the 24bit depth mode with 1152x864_85? e.g.

```
SubSection "Display"
		Depth		24
		Modes		"1152x864_85" 
	EndSubSection
```

nv is the open source drivers whereas nvidia uses the propriety drivers.

Have you looked into the correct log file (I think it is /var/log/Xorg.0.log) for any errors?
