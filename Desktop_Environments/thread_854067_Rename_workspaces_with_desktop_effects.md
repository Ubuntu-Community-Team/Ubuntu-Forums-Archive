### Post by SecretCode on 2010-05-03
The reason for this is something to do with Compiz not actually using multiple workspaces - instead it uses a single one 4 times as wide (or however many you've set it to) and then rotates a **viewport** across this large workspace to give the same effect - and more effects - just not naming.

That's as far as I understand it at least.
