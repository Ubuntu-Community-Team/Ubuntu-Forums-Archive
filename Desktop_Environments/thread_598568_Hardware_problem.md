### Post by ofb on 2007-10-31
Your RAM is the limit here, not CPU or harddrive. I've got Edubuntu 7.10 running fine on a P3 450, but it has 528MB. I believe the current minimum for installing Edubuntu is 256MB, and it takes up about 3GB of drive space. (Edubuntu is largely identical to Ubuntu, but is a slightly smaller version for school use.) 

If you can't find some more RAM, then you'll want to look at Puppy, DSL, and DSL-N.

[http://en.wikipedia.org/wiki/Puppy_Linux](http://en.wikipedia.org/wiki/Puppy_Linux)
[http://en.wikipedia.org/wiki/Damn_Small_Linux](http://en.wikipedia.org/wiki/Damn_Small_Linux)
[http://damnsmalllinux.org/dsl-n/](http://damnsmalllinux.org/dsl-n/)

### Post by TeaSwigger on 2007-11-01
Completely agree with ofb. 256mb may be too little for ubuntu and its variations, especially if your PC doesn't have a dedicated graphics card. May I suggest that if you have a second empty memory slot in your PC, and the PC will accept it, you install an additional 256mb memory card. That should be easy to find and will combine with the 128 to a total of 384mb RAM; a good PIII with 384mb RAM will pleasantly run all but the most demanding linux distros. I have Kubuntu on such a PC and it runs very crisply and reliably, so long as I keep to light applications and don't run multiple apps if using a "heavy" app. 

At the current 128mb I'd suggest you try Puppy Linux, Damn Small Linux, DeLi or DSL-N as ofb suggested. The only ubuntu variation which might work, if you are careful of what you install in addition, is Fluxbuntu:
[http://www.fluxbuntu.org](http://www.fluxbuntu.org)

Good luck :)

### Post by ofb on 2007-11-01
I should say that 256MB is the stated minimum for Edubuntu. So while it won't be sparky, it should still work okay with 256MB.

Some people may suggest Xubuntu for running a little faster. I honestly haven't noticed that with the 7.10 version, and it still requires a minimum of 192MB RAM for installation. The minimum to install Ubuntu 7.10 is now 384MB. Edubuntu requirements may have increased for 7.10 too -- I can't tell for sure because Canonical's website is still full of old information about the 7.04 versions of everything.

I haven't recommended Fluxbuntu because I haven't tried their resurrected 7.10 version yet. Right now it still looks like they're still getting their act sorted, but maybe in a couple of weeks they'll be past the RC stage (Release Candidate) and they will have something worth trying out.

All that said, once any type of RAM is no longer current, the price for sticks tends to climb. So it could be rather expensive to get another stick for an old P3. It's a supply vs demand problem.

If you don't have a computer savvy friend who may have some old memory s/he'll be happy to donate, then you could try asking politely at your local independent computer store about more RAM and *maybe* get an okay deal.

But at a regular retailers, it'll probably be expensive if they have it at all. It'd be better to go with Puppy Linux until you can do move to a newer machine.
