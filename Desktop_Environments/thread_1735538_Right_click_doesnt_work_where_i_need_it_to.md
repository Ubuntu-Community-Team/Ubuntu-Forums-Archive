### Post by 3rdalbum on 2011-04-21
You're trying to follow instructions for Ubuntu Desktop, on your computer which is using Ubuntu Netbook.

Ubuntu Netbook in 10.10 uses a totally different desktop environment than Ubuntu Desktop in 10.10.

You might want to try Ubuntu Desktop 10.10 instead (you can install and use it on your netbook, no problems) or be prepared to work out some things yourself, as most instructions on the Internet assume you're using the Desktop interface. Either that, or wait for 11.04 to be released - then the interfaces are exactly the same on Desktop as on Netbook, and any new instructions that get written will work on both.
