Why would my conky sometimes go white like this? I'm running 12.10, conky lua.

---

### Post by kpkeerthi on 2012-11-04
Changed
```

own_window_type override

```
to
```

own_window_type desktop

```
and added
```

own_window_argb_visual yes

```
