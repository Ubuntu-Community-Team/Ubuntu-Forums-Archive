### Post by Larkspur on 2011-05-28
This should work:

Through Main Menu or right-clicking on the desktop create a launcher.  The command to give it should be "gnome-terminal -e htop."  Change the icon to the system monitor one to save confusion with the terminal.  Launch it and you should be able to stick it to the launcher bar.
