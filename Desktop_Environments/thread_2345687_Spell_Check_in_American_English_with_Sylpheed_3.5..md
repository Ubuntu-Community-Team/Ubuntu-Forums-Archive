### Post by Charlotte18 on 2016-12-06
I am using Sylpheed 3.5.0 in Lubuntu 16.04.

Does anyone know how to set the spell check in Sylpheed 3.5.0 to American English. Mine is set by default to "en" which appears to rely on British spellings.  I can right-click on a message that is being composed and change the language to "en_US," but "en" remains the default for the next message. In Sylpheed, how do I change the default spell-check language to en_US?

N.B.: My other applications already use en_US, or American English. It seems as if only Sylpheed defaults to "en."

---

### Post by howefield on 2016-12-07
Have you tried altering the option in Sylpheed > Configuration > Common Preferences > Compose > Spell Check and make it say en_US ?

---

### Post by Charlotte18 on 2016-12-13
That worked. Thank you.
