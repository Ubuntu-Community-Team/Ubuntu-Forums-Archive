### Post by Rodneyck on 2006-11-27
> **aysiu said:**
> 
I don't think you'll run into stability issues, either, but you may have cluttered applications menus.

Yes, this was a big drawback.  Also, if you decide to remove KDE later, it does leave behind a few applications which you have to remove yourself. 

I have tried the dual system twice, starting with Ubuntu as my base, then downloading KDE. I did not really find any applications in KDE that I could not find an alternative for in Gnome, so it was really just a cosmetic thing in the end. For me, I do not like large task panels and KDE has the largest. If you shrink them down, like I do in Gnome, then the KDE icons are way to small, almost pinpoint size, very odd, so I had to stick with a medium sized task bar. What a hog that is. I do like that you can completely customize the menus and everything else eye candy related. Gnome is lacking in that aspect.

In the end, I found Gnome to be more stable (more apps crashing in KDE/edgy) and faster on my system, so I remain with Ubuntu without a side of KDE.
