### Post by awi on 2006-01-25
VMWare is an excellent tool. VMPlayer is the free one, but you need VMWare to create the virtual machine especifications, and manipulate the hardware configuration.  You can download the demo version, and try it. I did not use it for games, but other applications, (Google Earth, for instance) runs almost as fast as if it were running on the machine itself.

### Post by AusIV4 on 2006-01-25
There are tools out there that let you create virtual machines for VMWare Player without having to pay the price for VMWare Workstation. I've not tried them yet, as I'm at school and don't have my XP install disks. I should be getting them friday, so then I'll try it and post an update.

As far as running games on VMWare player, I'd be skeptical. My experience with VMWare Player (using the premade VMs) is that sound is extremely choppy, and that's on my P4 3.0 ghz with 512 MB of ram. Now, I wasn't running a windows guest on an ubuntu host, it was the other way around, I'm not sure if the same problems will apply the other way, but my guess is they will.

You should be able to easily move files between the host and guest operating systems, and you should be able to get it to recognize jump drives and CDs with little to no tweaking. It looks to me like it should be good for most purposes aside from games, but I wouldn't even swear it won't do games.
