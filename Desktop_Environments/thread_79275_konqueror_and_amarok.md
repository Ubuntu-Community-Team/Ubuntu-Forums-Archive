### Post by claydoh on 2005-10-19
If you don't have your sidebar enabled, hit F9 in a Konqueror window to display it, Amarok should be in there. If it isn't, then right-click in a blank area underneath the sidebar items and add it back in. To save this view (sidebar enabled) you need to go to the menubar and go to "Settings", then "Save view Profile "Kubuntu File Manager"", then click on "OK" in the dialog that pops open.
