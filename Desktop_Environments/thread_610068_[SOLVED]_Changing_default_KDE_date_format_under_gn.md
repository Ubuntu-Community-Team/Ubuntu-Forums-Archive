### Post by garyecobb on 2007-11-11
Is it possible to change the KDE default date format if I am running gnome?  I use kmymoney2 and am used to mm/dd/yy rather than yyyy-mm-dd.  Does anyone know if there is there a way to do this?

Thanks!

---

### Post by cookies on 2007-11-11
Install KControl, and the you can go to

KControl > Regional & Accessibility >  Country/Region & Language > Time & Dates > Short Date Format

And set it to MM/DD/YY
