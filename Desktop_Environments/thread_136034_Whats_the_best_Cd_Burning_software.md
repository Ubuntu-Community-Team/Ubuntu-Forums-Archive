```

sudo apt-get install k3b

```

### Post by BeatBoxRocker on 2006-02-25
To burn a cd audio from mp3 files you have to install the k3b-mp3 package with Apt. Open Konsole and write: 

sudo apt-get install k3b-mp3

Saludos!
