### Post by TheSqueak on 2010-04-07
For KDE applications, there is the [qdbus](http://usrlocalbin.blogspot.com/2008/04/qdbus-tutorial-part-one.html) command, but that's not much help for firefox and openoffice.


There are a limited number of command line options you can use with Firefox, you can find them [here](http://kb.mozillazine.org/Command_line_arguments) (or Google's cached version [here](http://webcache.googleusercontent.com/search?q=cache:8RUm60euW7AJ:kb.mozillazine.org/Command_line_arguments+firefox+command+line+options&cd=1&hl=en&ct=clnk&gl=uk) as the Mozillazine KB appears to be down)


There doesn't seem to be much available for Open Office though ...
