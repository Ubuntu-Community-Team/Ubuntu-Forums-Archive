### Post by Vivaldi Gloria on 2008-06-13
> **underworld288 said:**
> I was just wondering how to change my desktop icon for the trash. Can anyone give me some advice?

In gnome-appearance-properties you can click themes tab and then the customize button to change the icon set. That of course changes the icons of everything, not only the icon of the trash.

To change the icon only for the trash you can try putting a different trash icon in an icon set and then choosing that icon set as above. Icon sets live in

```
/usr/share/icons
```

I don't know if there is a more direct way.
