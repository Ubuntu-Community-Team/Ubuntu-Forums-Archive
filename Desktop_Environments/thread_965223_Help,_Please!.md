### Post by jonnyrockets on 2008-10-31
hi sherri, 

what you have downloaded is an iso , or image file. think of it as one big file that contains all of the ubuntu information for an install. 

when you install ubuntu, you are changing your OS (operating system). you would be using ubuntu instead of windows. (there are options to install within windows, but this can get complex). 

to try out your linux distribution, you need to burn that iso image file to a cd. this must be done in a specific way - you must use an option within your burning program such as "burn image to disk" or "burn disk image". 

once this is completed, you put the cd in your cdrom drive, and reboot your computer. 

hopefully, your computer will automatically boot off of the live-cd. if it doesn't, you need to change your boot priority to boot off of the cdrom drive before it boots the hard drive.
