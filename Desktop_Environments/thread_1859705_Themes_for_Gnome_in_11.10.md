### Post by crdlb on 2011-10-14
Gnome Shell themes control the shell's UI: the panel, overview, etc. There is no official mechanism for switching shell themes yet, so you need the user theme extension, which will give you a selector in gnome-tweak-tool. You could also replace the default theme in /usr/share/gnome-shell/theme/, but I don't recommend that.

You can get the user theme extension here: [http://www.webupd8.org/2011/10/official-gnome-shell-extensions.html](http://www.webupd8.org/2011/10/official-gnome-shell-extensions.html)
