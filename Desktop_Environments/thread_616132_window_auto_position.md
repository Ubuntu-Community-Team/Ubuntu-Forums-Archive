### Post by lian1238 on 2007-11-19
I assume you are referring to Compiz's plugin called "Place Windows" with Placement Mode set to Smart.

Hit Alt+F2, and run
```
ccsm
```

OR

Go to System->Preferences->Advanced Desktop Effects Settings

and configure the Place Windows plugin from there.
