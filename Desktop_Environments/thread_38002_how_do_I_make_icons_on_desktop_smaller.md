### Post by timczer on 2005-05-30
if you are using various icons downloaded from different places, they may be of different sizes or the one you want may not come in the size you need.    I resize mine in Gimp.  Load the image, go to the image tab, and use scale image.  This way you can ensure they are all the same size (48X48 for desktop for example).  With stretching you kind of have to eyeball the size changes.  Save the changes.  I create a file for all the resized icons for my desktop so I can redo them if needed.

### Post by jeremy on 2005-05-30
I copied this from [http://gnomesupport.org/forums/viewtopic.php?t=1975](http://gnomesupport.org/forums/viewtopic.php?t=1975)

- Open up Nautilus.
- Select the edit menu
- Select the Perferences item
- Choose the view catagory
- Under Icon View Defaults modify the Default Zoom Level

All of icons in Nautilus window and on the desktop change.

### Post by woz on 2008-01-13
I resized my desktop icons on Ubuntu GNOME by opening up gconf-editor via a terminal window. Then selecting apps>nautilus>icon_view  Then changing the thumbnail_size to what ever size you want. I used 40 which seems a good size....

Hope that helps.
