### Post by Pandora on 2005-11-26
Hi everyone, 

I have a question that pertains to the nautilus right click menu. In Ubuntu 5.04 you used to be able to right click anywhere on the desktop and invoke a terminal from that menu. In 5.10 that option is now gone, and I would like to restore it on my laptop, which I just did a fresh 5.10 install on. 

Does anyone here know of a way to do this? I'm not real familiar with gnome or nautilus and I would really like to invoke the terminal the same way again.

Thank you in advance for any help
~Pandora

---

### Post by suRoot on 2005-11-26
```
sudo apt-get install nautilus-open-terminal
```

I had to log back in after installing to see the context menu.
