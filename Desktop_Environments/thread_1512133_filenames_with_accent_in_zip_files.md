I am having troubles with filenames with accent in zip files.
the traditional error message: 
"
An error occurred while extracting files.
not matched:  Cr\?mailli\?re Jo & Dj\? Photos Tony 036.jpg
"
Cr\?mailli\?re Jo & Dj\? Photos Tony 036.jpg is the name of the file, and my friend Tony uses Windows...

There are many forums telling the bug has been solved by an update. I am running Ubuntu 10.04, and obviously this update is missing ?

I could get around this bug though, at least for pictures:
1. install comix in your favorite package manager
2. then use the "save as" feature of the "File > Edit Archive..." menu.
3. the saved *.cbz file is a accent-friendly archive (weird letters are replaced by "?") that you can open with your favorite archive manager

If somebody had a less time consuming, all-file flexible solution, I would be glad to have it !
