### Post by stoic jed on 2006-02-05
I was having problems with the Weather Applet not updating. After checking some logs i realized the National Oceanic and Atmospheric Association is on PeerGuardian's blocklist. So any time i was running PeerGuardian the Weather Applet couldn't update. If you're using any programs to block IP ranges, that might be what's causing it.
