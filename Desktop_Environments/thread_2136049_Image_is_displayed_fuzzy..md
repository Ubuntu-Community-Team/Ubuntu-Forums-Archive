Edit: I have typed "Nvidia" in the dash and clicked on "Nvidia X Server Settings". The resolution 1280x1024 works good.

Edit2: Follow these steps to fix this problem:
[LIST=1]
[*]Turn off computer. Turn off the PSU. Unplug the power cord. 
[*]Open the computer case side panel so that you can acess the harddrive. 
[*]Unplug the primary harddrive (both sata and power) and wait for 5 seconds. 
[*]Plug both cables back in to the harddrive. 
[*]Close the computer case side panel. 
[*]Plug in the power cord and turn on the PSU. 
[*]Now you can start the computer and it should work normally. 
[/LIST]

This was the way I fixed my computer. I can now set the native high definition resolution. Read my other thread to understand what other problems you can fix by this guide. Link: [http://ubuntuforums.org/showthread.php?t=2136603](http://ubuntuforums.org/showthread.php?t=2136603)
