### Post by Copper Bezel on 2011-05-01
Are you having trouble using the Launcher to navigate between windows? If you've used Ubuntu in the past, it's not really any different from the "taskbar."

Now, to clarify, you mean *Exposé*, not *Expo*, correct? Expo is the Workspace Switcher. Exposé is a Mac term sometimes used to refer to Compiz's Scale plugin, which is a window selector.

Clicking the Home Button definitely doesn't activate Scale. By virtue of the fact that you're using Unity, you have Scale enabled (as it's used for distinguishing multiple windows of the same application in the Launcher) but you might want to set it to a keybinding or window corner. It's also possible to activate it from a Launcher icon, but that would create clutter in your Launcher. Check out its settings in CompizConfig. ([font="Courier New"]compizconfig-settings-manager[/font] to install, [font="Courier New"]ccsm[/font] to run.)
