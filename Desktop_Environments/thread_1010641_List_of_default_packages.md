### Post by urukrama on 2008-12-14
To install the default Ubuntu applications, just install ubuntu-desktop:

> sudo aptitude install ubuntu-desktop

That pulls in all the default applications. For Intrepid, this is the [list]("http://packages.ubuntu.com/intrepid/ubuntu-desktop").

Have a look [here]("http://www.psychocats.net/ubuntu/puregnome") for how to remove all the default Kubuntu applications.
