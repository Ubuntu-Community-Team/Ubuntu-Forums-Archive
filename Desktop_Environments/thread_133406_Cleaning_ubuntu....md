### Post by Krigl on 2006-02-20
There is GParted for partitioning, it could be added from menu, somewhere under Add applications. Or just type in your terminal:
```
sudo apt-get install gparted
``` 

There's also a lot of basic help here: [http://easylinux.info/wiki/Ubuntu](http://easylinux.info/wiki/Ubuntu)
