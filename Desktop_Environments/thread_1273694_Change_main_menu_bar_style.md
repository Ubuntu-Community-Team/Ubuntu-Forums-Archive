### Post by gadolinio on 2009-09-24
I  know a way, but it's not very simple. I downloaded a theme called "GNU-Linux-Lemnos-UI2". It can be totally modified. If you navigate the folders it consists of, you'll find everything there: scroll bars, panels, etc. All in PNG format, so you can modify it easily. The thing is that you need to make all the other things look the way you want too (unless you like the default style).
The exact package i downloaded some months ago seems to have been updated to this one:
[http://www.ubuntu-art.org/content/show.php/N-M-LEMNOS?content=105854](http://www.ubuntu-art.org/content/show.php/N-M-LEMNOS?content=105854)
[http://gnome-look.org/content/show.php/N-M-LEMNOS?content=105854](http://gnome-look.org/content/show.php/N-M-LEMNOS?content=105854)
Hope you find this useful...
