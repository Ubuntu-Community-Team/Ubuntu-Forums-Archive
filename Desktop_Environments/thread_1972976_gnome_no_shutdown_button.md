BTW, I agree this is a shocking design choice but please don't let this one silly issue put you off gnome shell. Everything else about gnome shell is pretty much beautifully designed. They will correct this issue (somewhat!?) in v3.6. Until then, just install the [alternative status menu extension](https://extensions.gnome.org/extension/5/alternative-status-menu/) and forget about it.

PS: Best way to install gnome shell on 12.04 is:
```

sudo add-apt-repository ppa:gnome3-team/gnome3
sudo apt-get update
sudo apt-get install gnome-shell gnome-tweak-tool

```
