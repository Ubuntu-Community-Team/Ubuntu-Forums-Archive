### Post by mcduck on 2010-01-27
That's just normal. Ctrl+Alt+function keys are used to switch between different running sessions. By default you have  6 terminal sessions running in Ctrl+Alt+F1 to Ctrl+Alt+F6, and your graphical desktop is at Ctrl+Alt+F7.

If you log in another user (without logging out the first one), that user's session will run in Ctrl+Alt+F8. Add one more user, and that will get the next session at Ctrl+Alt+F9 and so on.

The reason Ctrl+Alt+F10 gives you black screen is that it's empty session, there's nothing running there at the moment.
