### Post by yoshii on 2015-10-15
[http://ubuntu-tweak.com/](http://ubuntu-tweak.com/)

Here's the current address of Ubuntu-Tweak.  It's not available via syntaptic nor ubuntu software center.  Instead you download the .deb and double-click that (it should open with ubuntu software center to complete the install).  It then shows up in the System menu and is labeled Ubuntu Tweak.  It also shows up as a green icon in the Settings Manager.  

Be careful with what you change/edit so you don't disturb your distro.  But I have found it to be very useful on Ubuntu Studio (which uses Xfce) to get file associations going a bit better without having to manually edit any files in a text editor.
