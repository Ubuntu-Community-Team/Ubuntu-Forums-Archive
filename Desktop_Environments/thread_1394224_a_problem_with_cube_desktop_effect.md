### Post by 3Miro on 2010-01-30
Install compiz-config-setting manager, go to System -> Admin -> Synaptic search for compiz-config and install it.

Then from the menus for the cube (System -> Prefs -> Compiz Config/Visual Effects), look for the zoom option inside the 3D cube. That one will make the screen zoom out and you will be able to better see the cube. Also, use Ctr+Alt + L Mouse on the desktop and drag left/right.
