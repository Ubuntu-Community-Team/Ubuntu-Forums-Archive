### Post by cajunlibra on 2009-05-03
I had this issue and I know others have also.  You enable a Desktop Cube and you only get 2 workspaces which is essentially a desktop "card" that you flip over.

In the CompizConfig Settings Manager, select General Options.  Under the Desktop Size tab, I chose 

Horizontal Virtual Size = 4(can be increased for even more workspaces)
Vertical Virtual Size = 1
Number of Desktops = 4

Vertical Virtual size doesn't seem to make a difference from what I can see. Even the number of Desktops almost looks irrelevent. To be safe, make sure Horizontal Virt Size and # of Desktops are the same.
