You link doesn't work.

If you want to customize the desktop effects, you need to install CompizConfig-Settings-Manager (available in the Software Center). You must have compiz working too (desktop effects to normal or above in Ubuntu before 10.10, or you must have Unity 3D in 11.04 and 11.10).

From CCSM you have a lot of options for customization.
