### Post by Tibuda on 2009-05-28
> **Entropy_Sam said:**
> Better still, can I just hide the desktop icons altogether?
Yes. Open the gnome configuration editor (Alt+F2, gconf-editor), and disable the /apps/nautilus/desktop/preferences/show_desktop checkbox.

> Ideally, I'd like to specify a given area of the desktop within which they can be placed,.
You can try the Folder View Screenlet: [http://www.gnome-look.org/content/show.php/Folderview+Screenlet?content=102890](http://www.gnome-look.org/content/show.php/Folderview+Screenlet?content=102890)
