### Post by GraeW on 2012-01-11
unfortunately, WINE (Wine is Not an Emulator) is not Windows. There are performance differences between the two which have slowly been improved upon over the last decade. Many games can work on WINE now which wouldn't have done so well just a few years ago. Others still won't run, and newer games many times won't run under WINE. All that being said, you can get more information from winehq.org and look through their database list of apps which work, the work-arounds required to get them to work, and other progress information.
 
Gaming is one area which still lags in Linux-based systems, but it is slowly improving.
