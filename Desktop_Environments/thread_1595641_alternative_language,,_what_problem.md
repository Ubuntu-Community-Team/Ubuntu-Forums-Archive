### Post by ankspo71 on 2010-10-13
Hi,
Do you have an Arabic only or English only keybord? If you do, you can make your keyboard Arabic and English.

First you will need to go to System > Preferences > Keyboard, then click on the layouts tab. Choose a second language, so you will have both Arabic and English keyboard layouts.

Now after you add the second language, an icon of a keyboard with country names will appear by the gnome clock. You can use this to quickly change keyboard layouts. You can also choose to see a picture of the new keyboard layout so it is easier to see which key to press on the keyboard. This works in Ubuntu 10.10 (maybe in older versions of Ubuntu too, but I am not sure). Hint: For letters and numbers on the right side of each key, hold down the right Alt key then press the letter or number.

Also, you can buy stickers for your keyboard to make typing easier like these: [_Link to Amazon.com search_]("http://www.amazon.com/s/ref=nb_sb_noss?url=search-alias%3Doffice-products&field-keywords=arabic+keyboard+stickers&x=0&y=0") You can probably find them at ebay.com too.

Hope these ideas help. :)

### Post by ankspo71 on 2010-10-13
Hi,
Also, Arabic and English keyboard layouts will work in Ubuntu 10.10's virtual keyboard called "onboard". You can press Alt+F2 and type onboard to start it. Or you can right click on your menu, choose Edit Menus, then enable: Universal Access > Onboard.

Hope this helps.
