### Post by Najand on 2006-08-04
In General, the root account and similarly the "su" command is locked in Ubuntu by default, as it might be very dangerous for your system and you valuable data. Instead, you can use "sudo" command to run commands as the administrator of the computer. For more information, see Ubuntu Wiki files: 
[https://help.ubuntu.com/community/RootSudo]("https://help.ubuntu.com/community/RootSudo")
You can also find the way to unlock the root account there too.
