### Post by aysiu on 2006-10-31
Create ones you will remember, then. The whole point of keyboard shortcuts is to be used in the GUI. In the terminal, you don't need keyboard *shortcuts*; you just use the keyboard... *period.*

For example, let's say you frequently use Firefox, Evolution, OpenOffice, and Synaptic Package Manager. You can have the keyboard shortcuts be:

Control-Shift-F (Firefox)
Control-Shift-E (Evolution)
Control-Shift-O (OpenOffice)
Control-Shift-S (Synaptic Package Manager)
