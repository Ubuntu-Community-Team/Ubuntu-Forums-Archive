### Post by MartinG on 2006-03-26
Not sure if I've understood you correctly, but .. While you're installing Dapper into VMWare Workstation your mouse won't work until you've got X running (i.e. after the installation). The VMWare window is like a completely different computer.

The (K)Ubuntu installer only works with the keyboard -- use the TAB key to move around, SPACE bar to toggle selections,  ENTER for OK and ESC for Cancel.  When we get a graphical installer, the mouse should work, but that's not yet.
