### Post by There Was A Time on 2009-06-07
Generally, icons, window borders and the gnome (top & bottom) panels are handled separately.

Emerald themes should only affect the borders, normal Gnome themes affect buttons, borders (without Emerald running) and basically everything inside windows.

You can download icon packages, although some themes come with them.

The Panels are changed manually, by right-click> properties. You can set a background image or a colour with opacity options.

I'm new to this as well, so don't take my word as scripture or anything.

### Post by 3rdalbum on 2009-06-07
Some theme packages only contain GTK themes, or Metacity (window manager) themes.

If you press the Customise button in Appearance, you can choose what theme to use for each element. A GTK theme should generally change the colour and appearance of the panels, but not all do.

An Emerald theme (.emerald) will only change your window border when running Emerald on Compiz.

Don't decompress themes that are distributed as .tar.gz archives. Just drag them onto the Appearance window, and if necessary use the "Customise" button to select the theme.
