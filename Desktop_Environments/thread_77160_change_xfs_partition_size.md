### Post by HermanAB on 2005-10-16
You should try the Mandriva Linux diskdrake, found on CD#1.  

What you do is start a Mandriva install, go to expert mode, which will run diskdrake, then modify your partitions and after that - switch off - don't continue the install, unless you actually want to install it.  Many sysadmins keep a copy of CD1 in their toolbox, just for using diskdrake.
