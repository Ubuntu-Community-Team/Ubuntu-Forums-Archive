### Post by bonzodog on 2007-12-28
eep..um
this is an obvious syntax error.
first off, look at all the other options and see how they are formatted. They have:
```
option "option name" "integer value"
```

now notice that all values are in double quotes?

so yours **should** read:
```
Option "VertScrollDelta" "6"
```

try that, and tell me if it works.
