### Post by ankspo71 on 2010-10-24
Hi,
Open up your package manager...  Software Center or Synaptic Package Manager, etc. Search for a package called ubuntu-netbook. Install that package.

Now log out, then first click on your username, then look at the bottom of the screen to make sure Ubuntu-Netbook is selected (not Ubuntu-Desktop), now enter your password and log back in. You will then have the complete Ubuntu Netbook Edition running. You also still have the possibility to use the Ubuntu-Desktop too.
Hope this helps.
