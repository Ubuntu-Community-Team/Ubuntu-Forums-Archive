### Post by Prisoner_97p904 on 2006-11-17
Hi!

I wanted to change the same thing and I did :) 

Go to "Emerald Theme Manager" -> "Emerald Settings" and there is a drop-down called "Titlebar Double-Click Action", which should be set to "Maximize/Restore".
