Where is the icon directory in Ubuntu? I know there is ~/.icons for custom icons, and /usr/share/icons for system icons. However, I want to change icons for specific things. For example, I have an icon I want to use for all folders. I assume there is a directory that holds the icon set one is currently using, and you can replace the icons separately. Is this true?

### Post by hakermania on 2010-12-30
I think you'll find them at /usr/share/icons/THEME(e.g. Humanity)/places/48/
