### Post by dbott67 on 2005-10-02
[QUOTE=sdnewlinux]How do I get a internet connection if i cant get a driver for my modem[/QUOTE]

A few questions for you:

1. Are your Windows and Ubuntu computer fairly close to each other (or can you set them up in the same general vicinity (i.e. same room)?

2. Does each computer have a network card?

3. How do you connect your XP computer to the internet?

For example, if you have dialup, you can set your XP machine to use ICS (internet connection sharing) and if each computer has a NIC, you could connect them together via crossover cable and get internet via the XP machine.

If you have high-speed (cable or DSL) you could buy an inexpensive router (D-Link or Linksys) and setup a small LAN to share the connection.

I can elaborate more if you can provide me with more details.

-Dave
