### Post by jon.reeve on 2009-04-20
Hey everyone, 

I have a netbook and recently tried out the Netbook Remix. Generally I found it pretty annoying but what I did like about it was that it got rid of the window decorations for maximized windows, thereby saving some screen space. This works in metacity but I can't get it to work in compiz, 

So my question is, is there a way to disable the window decoration only for maximized windows in compiz? In compizconfig-settings-manager I can declare what types of windows to have decorations for, but I don't know the syntax for these or where to find this. Any ideas?

---

### Post by jon.reeve on 2009-04-20
Just figured it out, thanks to [http://listento.jaketolbert.com/linux/remove-titlebar-on-maximized-windows-with-compiz/]("http://listento.jaketolbert.com/linux/remove-titlebar-on-maximized-windows-with-compiz/"). Apparently just add !state=maxvert to the Window Decorations box.
