### Post by NoobieDoobieDo on 2007-01-24
I got XFCE 4.4 installed fine on my Ubuntu 6.06 but the XFCE Goodies failed because it said I lacked libusb 0.1.12

So I tried using synaptic / apt-get to install all the libusb packaged I could find - however that didn't work either.  The graphical installer still told me I was missing libusb.

Then I found this : [libusb_0.1.12]("http://rpm.rutgers.edu/repository/ubuntu/pool/main/libu/libusb/")

Downloaded, went to the console - did the usual configure, make, make install commands.

Now I can install XFCE goodies just fine.

Just thought I'd make this post because I wasn't able to find any info on the subject :(
