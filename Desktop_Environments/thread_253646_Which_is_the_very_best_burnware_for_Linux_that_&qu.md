### Post by Junx on 2006-09-09
I like K3b, but for burning a DVD ISO, I just set up a little script to do it much faster:

```
burn_dvd_iso()
{
    growisofs -dvd-compat -Z /dev/dvd="$1" -speed=8
}
```

That would go in your .bashrc or something of course.  You could save it as a script, too (just the growisofs line with a shebang before it).
