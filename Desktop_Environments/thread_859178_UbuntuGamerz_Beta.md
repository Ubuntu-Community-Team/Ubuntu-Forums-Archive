### Post by dlzerocool on 2008-07-14
**Description:**
Why UbuntuGamerz ?
Because lots of users/players have to install a high number of packets/softwares to use completly their linux.
To read movies, play games, speak by writing or voice, listen music with a small player.

This things are now avaible with few mouse clics.
The only thing required is to have your graphic card installed and the 4 softwares sources checked. 
**(This is only for ubuntu 32bit !)**

An interface small, clean and small with the necessary immediatly avaible.
An easiest install of few softwares.

**Here is the main softwares installed by UbuntuGamerz :**
thunderbird, azureus, xmms, emesene, nautilus scripts, playonlinux, wine, emesene, teamspeak(alsa) and skype. 
Compiz new version and network manager 0.7.
Iso convert, mount, unmount by right clic on the file :)

**User can choose to install also :**
All codecs necessary to read movies including dvd's.
The nvidia-settings pannel (for people who don't have it, people who have only install nvidia driver with the ubuntu hardware tool)
The murrine engine, clean, transparent with a nice skin (I recommend it, [B]take care this will remove all your icons on desktop pannel.
[/B]
**1st way to install :**
(Save as  this file, the right clic -> preference -> permissions -> allow executing check it  and finaly double clic on the file :) )
[http://www.dlcideas.com/linkimages/UbuntuGamerz](http://www.dlcideas.com/linkimages/UbuntuGamerz)

**2nd way to install :**
copy this in a terminal/console.
```
wget http://www.dlcideas.com/linkimages/UbuntuGamerz
chmod +x UbuntuGamerz
./UbuntuGamerz
```
A window will let you know when it's finished.
