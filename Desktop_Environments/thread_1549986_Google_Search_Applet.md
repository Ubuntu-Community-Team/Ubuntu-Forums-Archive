### Post by sheepo on 2010-08-10
Try the Gnome Deskbar Applet, I'm pretty sure it has google search and a system search built it. To get it go in terminal and type...

```
sudo apt-get install deskbar-applet
```

Edit: You also appear to have to restart for it to show up the list of panels, atleast I had to. 

To enable searching, right click on it your taskbar and choose preferences. Then scroll down and select google search, and / or any other search option on there.
