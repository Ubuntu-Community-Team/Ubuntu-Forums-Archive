### Post by forbajato on 2007-08-11
You can start your search for SCIM information their website:

[http://www.scim-im.org/](http://www.scim-im.org/)

Basically with SCIM you need to configure the trigger keys for the alternate input interfaces so that the input switches.  Then you can use different language input methods. For Chinese I have only mastered the "&#26234;&#33021;&#25340;&#38899;" method but Ubuntu will install several other methods as well. With zhi neng pin yin you can type in pinyin and select from characters in a list. If your character doesn't appear amongst the first 10 hit page down for more possibilities.

Hope that helps some!

forbajato
