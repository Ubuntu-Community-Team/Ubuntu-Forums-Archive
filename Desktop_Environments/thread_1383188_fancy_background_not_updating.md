### Post by lidex on 2010-01-17
From the website:
> Usage

For the script to work best (and not to mention stay up-to-date) it's best to run it at startup. Add it to your startup applications

    * System > Preferences > Startup Applications > New Item

Select ~/earthfolder/changer.sh

Click okay.

If you don't add it to start-up automatically you will need to manually launch the script when wanting to update the real-time view.
