### Post by ajgreeny on 2006-07-30
Decide what you want to use for your mp3 files, (or any other files you need to do this for), click on *open with>other program>chosen program*, making sure you have put a tick in the tick box for *"remember this for this file type"* or whatever the exact phrase is, I can't quite remember.
