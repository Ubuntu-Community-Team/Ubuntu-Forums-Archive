I am a total noob when it comes to Linux, so i wil explain what i found out in the "windows" way, 10hi^^

2. Go to compiz config settings manager and edit the actions in "Rotate cube".

3. Same as nr2, only u activate and configure the settings u want in compiz config settings manager.
