### Post by foxmajik on 2010-04-13
When Ubuntu thinks a program is not responding the window turns greyscale.

I want to disable this because when I'm using Nomachine NX, Ubuntu turns all the windows grey.

How do I disable the desaturation effect?

**Edit:**  Answering my own question, you go into CompizConfig Settings Manager, go to the settings for Fading Windows and uncheck Dim Unresponsive Windows.
