### Post by twotonz on 2006-08-11
Hi,
As a default installation, the server version comes without x. After you have the base system installed you can then choose which desktop you want using aptitude. Personally I think that kde & gnome are a bit too much for a server system, try openbox, much lighter. 
As for running apache, mysql & php, doesn't really matter if you use a console or some gui on a desktop, anyway, installing a desktop certainly will not interfere with your server stuff.
