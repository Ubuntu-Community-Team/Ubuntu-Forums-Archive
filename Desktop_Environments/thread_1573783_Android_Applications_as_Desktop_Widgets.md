### Post by Alan James on 2010-09-13
I think it would be very cool to be able to run Android/WebOS/Maemo applications as widgets on KDE or Gnome. There would then be another reason for uses of those Linux based platforms to use a Linux desktop. They could buy an application and not be limited to using it on just one device.

Is this possible to do? If not then what are the technical/legal limitations.

Also, I can imagine a future where Apple's OS X operating system will be able to run iOS applications as widgets on it's Dashboard layer. It will mean more money for Apple as they could then sell apps to people that don't have iOS devices.
