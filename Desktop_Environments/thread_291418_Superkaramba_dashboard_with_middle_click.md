### Post by Jucato on 2006-11-02
Superkaramba doesn't have a "bring forward" feature like OS X's Dashboard. There is, however, a Show Desktop feature in KDE. By default, the keyboard shortcut is Ctrl+Alt+D. You can add a button to the panel by right-clicking on the panel, selecting Add Applet to Panel, and adding the "Show Desktop" applet.
