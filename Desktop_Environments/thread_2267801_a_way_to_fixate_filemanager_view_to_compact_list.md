### Post by ajgreeny on 2015-03-04
I presume you are using the default, which is pcmanfm in Lubuntu.

Go to pcmanfm -> Edit  -> Preferences -> General tab.  At the bottom is a **Default View: View mode:** drop-down box. Make your changes there.
