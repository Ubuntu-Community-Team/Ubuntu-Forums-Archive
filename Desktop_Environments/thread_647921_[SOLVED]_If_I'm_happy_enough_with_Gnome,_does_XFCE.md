### Post by p_quarles on 2007-12-23
Xfce's main advantage over Gnome is that it uses slightly fewer system resources. As far as functionality, you don't gain much, and you don't lose much, it's just a different environment, with a different look and feel. Compiz/Beryl work fine with it, if your GPU supports them, and you can run any application you like.

If you have the spare time for experimenting, go for it. It's always fun to play around with different environments, imo.
