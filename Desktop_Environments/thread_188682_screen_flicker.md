### Post by dbw on 2006-08-21
Have you tried [using a custom modeline?]("https://wiki.caosity.org/tiki-index.php?page=X%20Server%20Configuration")  I had the same issue.  Also, it could be an issue w/ the specific window-manager.  Do you get the static on GDM?  If not, you might try using another window manager ( fluxbox, kde, gnome, xfce ...).

you could also try commenting out the ```
load "glx"
``` line in your xorg. You do not strictly need this function, though it will allow fancier rendering capabilities.
