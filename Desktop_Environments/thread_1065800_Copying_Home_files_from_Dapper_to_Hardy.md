### Post by jbrown96 on 2009-02-10
they could possibly. It depends on what programs have configuration files and whether the programs have changed the format of the config files. Since dapper is fairly old, I would say that there is a distinct possibility of this happening. 

You will know very quickly what works and what doesn't, but if you're doing a clean install anyway, then it can be fixed painlessly. I would certainly make a backup of any hidden files. They don't take a lot of space and it just might save you.
