### Post by jjpcexpert on 2009-05-04
Q1: I need to get Xfdesktop running in GNOME. I hate Nautilus in ways.

A1: (Answering several peoples question incl. my own)
Using GConf Editor, /desktop/gnome/session/required_components/filemanager.key needs to be changed. So change it to xfdesktop (or thunar).

Q2: I don't have xfce installed so this didn't work.

A2: Install Xfce (4.6 reccomended, 4.0 minimum) or single Xfdesktop thunar and xfce4-utils. Now you are ready for real action. Now do Q1 + A1.
