### Post by susanoodin on 2012-01-16
According to Googles help pages on browser language settings:

"On a Linux, check your system language support."

It seems like you can not change Chromes Language setting independently.
So you could go to System Settings > Language Support and drag the language you need to the top of that list. That'll change the language for the whole system.
If the language you want is not listed you can probably install it by clicking on install/remove  and selecting the appropriate one (or get an admin to do it, if you don't have proper permissions). 
Hope this helps

Susa
