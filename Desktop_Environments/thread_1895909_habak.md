### Post by thaelim on 2011-12-16
Not sure about Habak itself, but you could try using Ubuntu Tweak to disable Nautilus from drawing the desktop (it draws the wallpaper as well, right over the root window as you've noticed). The option is in Ubuntu Tweak under Desktop Icon Settings - uncheck "Show Desktop Icons"
