### Post by mcduck on 2009-06-21
It should work without problems as long as you keep both applications running (so don't close the first program after copying, or you'll loose what you just copied. There is no actual clipboard unless you install one, things are copied directly from source to destination.)

But if you can't get copy/paste to work, use primary selection (the Unix/Linux-style copypaste) which is also available. It's easy, just highlight what you want to copy and then middle-click to paste it where you want.
