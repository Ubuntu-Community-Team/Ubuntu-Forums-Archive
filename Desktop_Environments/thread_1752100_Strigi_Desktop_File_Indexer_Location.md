### Post by 55tptag on 2011-05-09
from bcooksley at [http://forum.kde.org/viewtopic.php?f=154&t=94984&p=195536#p195536:](http://forum.kde.org/viewtopic.php?f=154&t=94984&p=195536#p195536:)
"Strigi is simply the indexer, it stores the information it extracts in Nepomuk. Therefore Strigi's index is also in ~/.kde4/share/apps/nepomuk/repository/main/data. 

Note: Other Nepomuk data like ratings, tags, and groups for information will also be stored in Nepomuk."
