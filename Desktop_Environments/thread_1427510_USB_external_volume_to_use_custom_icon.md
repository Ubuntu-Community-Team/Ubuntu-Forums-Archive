### Post by asmoore82 on 2010-03-11
It's so simple that it's complicated...

Just right-click the volume's existing desktop icon and select "Properties"
Then, click the icon in the properties window to change it.

No fiddling around with *.inf's, or *.ico's ... but I think Gnome will honor that too.
