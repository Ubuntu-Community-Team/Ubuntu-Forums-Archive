### Post by fbnaia on 2008-04-30
Xubuntu seems to always been suffering from missing menu entries, lets keep in mind that the PolicyKit integration is a new feature.

But... its also displayed in the [xubuntu site]("http://www.xubuntu.com/news/hardy/release") as being located under "Applications / System / Authorizations interface" so it can be frustrating for new (and old :)) xubuntu users.

One way to fix this and other menu issues is trying out with [Menu Maker]("http://menumaker.sourceforge.net/") , but again, it would be great to have all the main features included in the menus by default.

Fix released: [https://bugs.launchpad.net/ubuntu/+source/policykit-gnome/+bug/232234](https://bugs.launchpad.net/ubuntu/+source/policykit-gnome/+bug/232234)
