### Post by brentoboy on 2005-09-27
[QUOTE=erik-the-red]
What exactly does installing kubuntu-desktop do if you're on GNOME?
[/QUOTE]

Kubuntu / ubuntu - same thing
if you use kdm for a display manager, you get kubuntu backgrounds, and the KDE based utilities.

you can use apps from both worlds if you so choose, just select them from synaptic.

in general, I have found that KDE apps have lots of user interface options, and gnome apps have more avaialble settings.  pull down kynapic and synaptic compare them.  Kynaptic is beautiful and simple - easy. synaptic has menuitems for settings that kynaptic conveniently leaves out.

if you want the in depth options, go gnome. If you like eye candy, go kde.

either way, you can swap wihtout reinstalling, just use the synaptic and remove the stuff in the gnome area, and add the stuff you want in the KDE area.
