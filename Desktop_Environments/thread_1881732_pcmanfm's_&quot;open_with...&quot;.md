### Post by ac_d600 on 2011-11-16
Right-click the .jar file and choose Open With.. a dialog box will open letting you choose the application
you want to use to open the .jar file with.

You are given two ways to do this, either **Installed Applications** or **Custom Command Line**, I don't have
java installed on my machine so I'm not sure which one you will have to use.

Make sure you select **Set selected application as default of this file type** (bottom of the dialog box)
so you don't have to do this again next time you want to run a .jar file.
