### Post by burnit on 2008-08-21
On Xubuntu with Thunar you can open xTerminal on all folder, but when you are on simply user and you want to open this folder with root, you cant.

This trick it's to easy on 30sec you can do it and after you can open on Thunar explorer all time root xterminal on all folder automatically.

1: Go on Thunar explorer and open Edit menu then clic on "Configure action..."(sorry ive got french version, maybe you understand)

2: Clic tow time on "Open Terminal here.." and change command you find normaly like this one "exe-open..." from this one :

gksudo /usr/bin/x-terminal-emulator

One time he ask pass root and after ist going ok.
