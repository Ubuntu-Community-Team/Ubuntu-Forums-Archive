### Post by bandreasen on 2006-12-15
I recently installed ubuntu edgy eft (6.1.  I'm new to ubuntu and have what seems like a stupid question.  I just installed vmware and the executable file turns out to be a script file.  I want to put it on my desktop with an appropriate icon.  However whether I copy it or link it to the desktop the icon remains a script icon and I can't change it. How can I change it to the vmware icon that's been installed in the /usr/share/pixmaps folder?

---

### Post by mcduck on 2006-12-15
Right-click on your desktop, select 'Create Launcher', type path to your script to the 'Command'-field (or 'Browse' to find the script), name for your launcher to the 'Name'-field, click on the 'no icon'-button and select your icon.
