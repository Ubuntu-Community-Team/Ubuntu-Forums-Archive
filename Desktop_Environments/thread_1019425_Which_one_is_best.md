### Post by gjoellee on 2008-12-23
None OS' are best or bad, it is just what YOUR needs are, and what YOU think is best for YOURSELF.

Here are a few tips:

[COLOR=Red]GNOME:[/COLOR]
-If you are a developer and extract a lot of files, and need meny functions at your right click, using GNOME will spare you fora lot of time
-If you don't want to much eye candy
-You don't want so many options, but still have full functionality

[COLOR=Red]KDE:[/COLOR]
-If you like having a lot of eye candy
-Don't care so much about performance (KDE is still pretty fast though)
-If you don't need so many GTK applications (applications made for GNOME), this is because KDE has some kind f bad support for GTK apps. If you want to make them look good see this link: [http://www.kshoster.net/content/howto-make-gtk-apps-look-nice-kde-4](http://www.kshoster.net/content/howto-make-gtk-apps-look-nice-kde-4)

[COLOR=Red]Xfce:[/COLOR] (I am taking this with as well)
-If you like something simple, fat and still has about the same functionality as GNOME Xfce is what you would like (NOTE: Xubuntu is actually quite heavy, and I would not recommend it for an older computer)

I rcommend you try using al of those desktops, I started using Ubuntu when I was new to Linux, but went over to Linux Mint because if found it easier. However I started using Kubuntu, but I did not get the functionality i needed so I was switching between Ubuntu and Kubuntu for a long time (experiencing KDE and GNOME). Howrever I ended up noe using Arch Linux, and installed Xfce which i just fell in Love with and have been using since...
Why Arch Linux? Well you do in some kind of way build your own system whith what you need, and nothing more, thats makes a fest and flexible system...

### Post by ajgreeny on 2008-12-23
I started with Kubuntu about three and a half years ago, partly because it looked more like windows but also because it was, or at least I though it was, more adaptable and had more options.  When you get to know gnome properly, however, you can do almost anything with it that you can do in kde, and you can, if you want, make it look a bit like kde, with only one task bar at the bottom, etc etc.

Now I am so used to gnome that I dislike the look of kde, and find gnome so much more intuitive, that I use it all the time.  I do, however, still use one or two kde applications like k3b, kmymoney2 and most often digikam, a bit like f-spot, but better to my eyes.

My overall point is that there is no "better" desktop; both do largely the same thing and it is simply down to personal preferences.  Also remember that using one does not stop you using applications from the other.
