There are two separate things here. The first one is the IM service you use. Pidgin cannot transfer files in GTalk as far as my experience goes (but file transfer works fine in any Jabber server, so it is a Google propietary thing).

The second one has to do with the router you may be using. Some support IM file transfers automagically, others do not. In the latter case you have to forward the ports to your local machine in the router's NAT configuration.
