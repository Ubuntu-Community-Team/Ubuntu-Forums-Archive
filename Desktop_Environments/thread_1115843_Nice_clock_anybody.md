### Post by mcduck on 2009-04-04
> **longtom said:**
> For the cairo clocks - do I need to have cairo installed for this?

You already have Cairo if you are using Gnome desktop (as Gnome uses Cairo for font rendering, PDF printing and quite many other graphics tasks).

However you'll need to have Compiz (the "desktop effects") or some other compositing manager running for cairo-clock to work properly.

By the way, If you don't want to install Screenlets but only the clock, Cairo-clock is also available as it's own program (the screenlet was later made based on the original Macslow's Cairo Clock). As far as I know both the standalone Cairo-clock and the screenlet should use same themes.
