### Post by AlanR8 on 2008-11-11
Two things you could try, but first BACK UP any files you want to keep as a precaution!

First, open your Home folder in Dolphin and select "Show Hidden Files" from the View menu. Find the folder called .kde and delete it. Reboot and the folder will be re-created to default values.

If that doesn't work you could try a fresh install

### Post by Nesnej Trick on 2008-11-11
KDE4 suffers of occasional rendering errors. Usually this is restricted to scroll bars only, though.

Have you done a clean install or a dist-upgrade? My installation of Kubuntu 8.10 works fine now that I've re-installed it, but it was completely insufferable because of how the distribution upgrade tool botched the configuration.
