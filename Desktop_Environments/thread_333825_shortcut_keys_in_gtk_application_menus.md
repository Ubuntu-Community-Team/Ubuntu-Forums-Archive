according to the Liferea blog, you should be able to assign shortcut keys to menu items simply by moving the (mouse?) cursor over the item and pressing the key:

> As every good GTK program Liferea (since v1.1) allows you to dynamically assign key bindings. To do so open any menu from the menu bar, move the cursor above the relevant menu option and press the shortcut you want to assign. This shortcut should appear right next to the menu option label.

are they saying that shortcut keys can be assigned in this specific way in any "good" GTK application? or does it only apply to Liferea? and if it does apply to Liferea, why doesn't it work for me? :(

### Post by mcduck on 2007-01-08
It should work for every GTK app, but you need to enable it first. Go to System/Preferences/Menus & Toolbars and enable 'Editable menu accelerators'
