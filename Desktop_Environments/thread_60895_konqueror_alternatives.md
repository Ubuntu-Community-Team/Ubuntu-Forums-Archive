### Post by bizzaroMike on 2005-08-29
I also use konqueror as both a file browser and a webbrowser.  I really like its powerful features and its speed.  Anything that runs well on an EPIA is a good pick.  Firefox renders too slowly and has some ugly font errors.  Konqueror may not handle every website, but it does do 99% of what I need it to do on the web.  

Plus, it has cool features like the man page browser, the fish protocol, locked konsole sessions, midnight commander mode.

I guess the worst thing about it is that the default config is bulky and unfriendly.  I turn off the sidebar (F9), then massage the menus until there's only one toolbar with (up) (back) (forward) (home) (reload) (stop) (location bar) and a throbber.  It's also worth binding "Open a terminal" and "Show terminal emulator" to function keys so they're easy to get to.  Just my two cents.

Hey Drisek, I tried the kubuntu kaffine, but it likes to crash.  It seems like an okay interface to xine, but it is power hungry and I don't like the way it renders compared to mplayer.  What's changed with the latest version?
