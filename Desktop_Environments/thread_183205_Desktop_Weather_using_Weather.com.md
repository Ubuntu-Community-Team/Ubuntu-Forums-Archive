### Post by Trespasser on 2006-05-27
If all else fails install deskcut (Firefox extension), go to weather dot com> right click>down to deskcut. It will create a desktop link directly to weather dot com via Firefox. Or you could right click on an open space in Desktop, choose create launcher, under type choose link, paste in this address [http://www.weather.com/weather/local/(your](http://www.weather.com/weather/local/(your) zip code), name it and click OK.
