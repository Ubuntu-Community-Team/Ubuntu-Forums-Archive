### Post by Dennis N on 2018-07-23
Something weird going on as to how that stuff got in there. Not Normal. Anyway, try using PCManFM (the Lubuntu file manager). You can delete menu items: Sidebar > Applications > Choose Category > right-click on application > delete. Looks like you can also hide them, in case you later want them to show again.
