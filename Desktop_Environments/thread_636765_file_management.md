### Post by sirdilznik on 2007-12-11
Also try rox.

Once installed you can find them in your menu.  Or you can run them from a command line.  Generally the command will be the same as the name of the program and since they are installed globally then you do not need the full path so for example to run rox simply type ```
rox
```.

Again they should pop up in your menu/start button whatever (I'm using Fvwm a different Window manager...  Window Manager is very different from File Manager) so my menus and stuff will be different (I set them up myself) and I can't tell you exactly where they will appear on your GNOME desktop, but I'm pretty positive they will be in there.
