### Post by aysiu on 2006-07-14
Ubuntu and Xubuntu (Gnome and XFCE) have the same login screen--it's called GDM.

To select a different session, you have to go to **Options** > **Select Session**.

Mishal, Gnome is the default session since you installed it first, you will need to select xfce if you want to use it. So if you do Ctrl-Alt-Backspace to restart X, you will see a session option below where you enter your userid, click it and select the xfce session, you should now be able to start the xfce desktop. Also you can set this session as you default there as well.
