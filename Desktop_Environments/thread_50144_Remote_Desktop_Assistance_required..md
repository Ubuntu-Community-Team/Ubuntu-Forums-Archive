### Post by az on 2005-07-19
You can enable Remote Desktop in your prefences.  Just allow the remote user to control your desktop, do not ask for premission and set a good password.

Install a firewall, too...

You can then connect to you machine using any VNC client.   All you need to know is your IP address.  you can use DynamicDNS to find your box.  A good client for dyndns is ddclient.  It is in universe.
