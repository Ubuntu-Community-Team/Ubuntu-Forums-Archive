In Breezy, you can turn off the minimize animation if you follow code-junkies advice. If you want to get rid of that ugly animation and get rid of the wireframes at the same time go to > System > Preferences > Assitive Technology Support & Enable Assistive Technologies...

It's not really an answer but it is a workaround I found to work nicely... I would settle with the wireframe *IF* the wireframe mode was a bit more cusomizable... It is currently not to taste... Good luck!
