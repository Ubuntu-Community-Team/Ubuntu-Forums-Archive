I just installed the OS, and during the installation was prompted to create a limited account, which I did, but now I can't access anything.  It won't let me run updates, or even look at the user accounts on the system. ### Post by suRoot on 2006-02-24
When prompted for a password, enter the password you entered for your first user account (most likely the account you're using).  Likewise, if you run a 'sudo' command from the console, enter the same password.
