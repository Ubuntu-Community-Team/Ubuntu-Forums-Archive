### Post by Goatvirus on 2012-02-21
Well, here it is a few years later... same problem exists in a standard install of Ubuntu 11.10 followed by Konqueror 4.7.4.  What you now need to do is hit F4, or else Tools->Open Terminal in the menus.

If you get a message something like "KDEInit could not launch 'konsole': Could not find 'konsole' executable." then it simply means that the konsole (KDE version of terminal) isn't installed.  After a bunch of false leads, I finally just looked in the Ubuntu Software Centre and yes "konsole" is listed.  Install it and you are good to go.
