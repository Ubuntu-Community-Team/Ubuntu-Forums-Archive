### Post by shimi on 2008-08-31
You need to share this drive with your virtual guest OS (XP) using the VirtualBox tool. This drive needs to be mounted in your Ububtu. If it is not mounted you will need to mount it first.

The VirtualBox User Manual is great. You will find everything you need about VirtualBox in it. You can access it from the Help menu. Search for 'share' and you will get all the information that you need.
