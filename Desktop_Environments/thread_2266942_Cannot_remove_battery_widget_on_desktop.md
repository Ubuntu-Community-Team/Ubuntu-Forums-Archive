### Post by PaulW2U on 2015-03-21
I don't like anything on my desktop either. ;)

If the original poster comes back the he should try right-mouse clicking on the desktop and **Unlock Widgets**.

Mouse-clicking on the offending item should bring up several icons, one of which, an **X**, can be used to delete the offending item.

Finish with **Lock Widgets** to protect desktop settings.

To add items to the System Tray, right mouse-click between the icons, then **System Tray Settings**, **Unlock Widgets** if you need to, then check any items you wish to be displayed, un-check any items that you wish to remove.
