### Post by vimsical on 2017-03-14
I am using Ubuntu 16.10 Desktop.  When I hold down the <Super> button, the launcher sidebar appears with icons labeled 1-9 etc so that I can quickly launch them.  But I am used to using keyboard shotcut <Super>+# to switch to Workspace # (I very much dislike the default Ctrl-Alt-#, since I have to press three keys with one hand).  Can I disable the launcher's shortcut feature to get back to using <Super> to switch workspace?

Thanks in advanced.

Solved: Using ccsm, unbind <Super> to "Key to show the Dash, Launcher, and Help Overlay" in "Ubuntu Unity Plugin".
