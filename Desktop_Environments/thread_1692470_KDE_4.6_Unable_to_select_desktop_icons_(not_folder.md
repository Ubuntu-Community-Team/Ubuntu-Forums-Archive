For some reason I can't select the desktop icons.  I can right click on them, I can left or right click the folder icons, but I can't left click on the file icons on the desktop.

### Post by jimbo99 on 2011-02-22
Found the cause of this issue.

The cause happens when you change the mouse option from single click to double click.  If you switch it back to single click you can grab file and folder icons again and move them around.

It is 100% reproducible by me.
