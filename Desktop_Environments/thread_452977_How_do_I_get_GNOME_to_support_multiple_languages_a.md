### Post by ButteBlues on 2007-05-23

However, I still talk to all my American friends in English on Pidgin and whatnot. As a result, whenever I'm using French as the language, I get all the annoying red lines indicating misspelled words.

Is there any way to get GNOME to use both the French *and* English dictionaries for spell checking at the same time?

### Post by Kobalt on 2007-05-24
Yes you can. You need to install the package *language-pack-gnome-fr*. After this you can also install the french dictionnary for Firefox, and you're all set.

My 0.02 euros worth... I work mostly in English but also in Spanish and French.
