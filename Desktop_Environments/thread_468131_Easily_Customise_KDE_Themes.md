### Post by bunker on 2007-06-08
Look for KDE theme manager packages.  You can import them using kcontrol.  Appearance and Themes -> Theme Manager -> Install New Theme (Button).  You'll find theme manager packages [here](http://kde-look.org/index.php?xcontentmode=8), but some qt styles require extra configuration, and various applications have extra options on top of those in kcontrol.  That said, kcontrol does 90% of what you need.
