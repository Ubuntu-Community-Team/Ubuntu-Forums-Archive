### Post by erginemr on 2008-03-25
Assuming that you are using a single monitor only, I can safely confirm that your xorg.conf file is corrupted with erroneous / redundant settings. You should stay away froom the Gnome tool "Screens and Graphics" at all costs as its implementation in Gutsy is buggy and may produce such impredictable config files. I will suggest you to configure your graphics by using text-mode tools, but please don't be intimidated as it is really easy:

1. First things first, open a virtual terminal with Ctrl+Alt+F1-F6 (your X server lies under Ctrl+Alt+F7 by the way), login with your user name and password and back up your current config file:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.working
```

2. Run the following config tool, which will configure your system automatically:
```
sudo dpkg-reconfigure -phigh xserver-xorg
```

3. Restart your X server by using the following commands:
```
sudo /etc/init.d/gdm stop
```
```
sudo /etc/init.d/gdm start
```

4. Check if the resolution 1280x800 has already been set. If not, you can edit xorg.conf file this time using Gnome terminal window (Main Menu -> Apps -> Terminal) with:
```
gksudo gedit /etc/X11/xorg.conf
```
finding the line with resolutions:
```
SubSection "Display"
	.....
	Modes		[COLOR="Red"]**"1280x800@60"**[/COLOR]	"1280x720@60"	"1280x768@60"	"800x600@60"	"800x600@56"
	.....
EndSubSection
```
and adding the resolution 1280x800 with a refresh rate of 60 Hz (which is standard refresh rate for LCD monitors) manually. The first item in this list will be the one to be selected as default by X server. Otherwise, please re-post your new config file and we shall work it out together. 

5. Alternatively, you may opt to configure your X server manually with:
```
sudo dpkg-reconfigure xserver-xorg
```
Then, you will be asked a series of simple questions, most of which you can pass with default selections. When asked, you should fill in these values:
```
Busid -> "PCI:1:0:0"
Driver -> "nvidia"
```
as based on your current xorg.conf file. Finally, you will be asked to select a set of screen resolutions, where you should select 1280x800 on top of the list and a few lesser values as fallback resolutions.

6. Once you have a working desktop with a reasonable resolution, there is a graphical tool "nvidia-settings" as an alternative to the buggy "Screens and Resolutions" tool. You may nvidia-settings use to fine-tune screen resolutions and refresh rates.
```
Alt+F2 -> gksudo nvidia-settings
```

7. If everything goes well, it is time to do a second back up:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.mybackup
```

So, that's all I guess. Good luck...
