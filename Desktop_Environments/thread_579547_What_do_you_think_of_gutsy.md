### Post by kevmacmills on 2007-10-18
I like it - very smooth.  

As always, I approach each new release with the "how would my mother use this?" angle.  I install the software, install a printer, do some word processing (checking compatibility with MS formats) and do other "normal user" stuff trying to put myself into a beginner's shoes.

From what I can see, a beginning user would enjoy the Gutsy experience.  I have one gripe however:  Why, why, why, **WHY** is there not a "run as administrator" option somewhere in the file manager?  If I insert a CD that requires me (or my mother) to execute an installation script it is almost guaranteed to fail if I just double-click on the script.

For instance, one CD I use for testing has a setup script written in perl (setup.pl).  I double-click on this and I am asked if I want to run the script, run the script in a terminal, view the script or cancel.  Where is the "run as administrator" or "run as root" option??  The beginner **should not** have to know about opening a terminal window and executing "sudo ./setup.pl".  IMHO, this is a big hole and I don't see why Ubuntu and/or gnome hasn't addressed this feature.

Perhaps someone knows some history here?

Other than that - congrats to the Ubuntu team, I really love the ease of getting codecs, etc.  Good work!
