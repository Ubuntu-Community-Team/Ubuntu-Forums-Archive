### Post by loseyou2him on 2007-06-16
I also have the same problem, in addition when I start maximized windows, the frame shows, but the content is black/blank.

Addition: 
It seems that there is a certain point where items black out.  For example, I enlarge a window a certain amount and it goes black.  Certain menus drop down black, while others don't.  I fussed with some settings and it gave me my desktop back.  I only have 32 mb of video memory.  Is this an issue?

A (Sort of) Solution:
I installed the compiz manager and put "compiz-tray-icon" into my startup programs (Preferences->Sessions->Startup Tab).  To install compiz manager you can just do this.
> sudo apt-get install gnome-compiz-manager
No beryl, but at least it's something.
