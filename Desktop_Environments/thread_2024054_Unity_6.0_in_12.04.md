### Post by jbicha on 2012-09-26
Unity 6 involves a massive change from gconf to gsettings. We're still cleaning up some bugs from that transition. If you tried backporting it to Ubuntu 12.04, you would lose the stability advantage of 12.04 so you might as well use Ubuntu 12.10 if you really want the new features.
