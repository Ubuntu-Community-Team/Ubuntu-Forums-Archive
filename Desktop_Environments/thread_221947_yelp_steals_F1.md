### Post by mvaniersel on 2006-07-24
I noticed that by default, whenever you press F1 yelp starts, no matter which application you are using. I think this is rather counter-intuitive, as it hides the application-specific help.

I've tried firefox, Scite text editor, gedit (!) and openoffice, and all three programs have their own help subsystem that starts when pressing F1, but only if you disable that keyboard shortcut by going to system->preferences->keyboard shortcut and then disabling "launch help browser"

Yelp sucks. It always starts with the same stupid start screen, so you have to find your way through all of that to give info on the app you're using at that moment. And more likely than not, it doesn't even have the help you're looking for.

Does anybody else agree this is a bug?
