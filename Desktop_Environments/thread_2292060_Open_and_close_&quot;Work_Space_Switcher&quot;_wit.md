You set the title as "[ubuntu]" so I am assuming you're using unity/compiz.
Use the expo plugin in ccsm.
```
sudo apt-get install compizconfig-settings-manager
```
[ATTACH=CONFIG]264053[/ATTACH]


Another solution is to install and use **easystroke** to assign keystrokes or commands 
to mouse buttons or gestures.
