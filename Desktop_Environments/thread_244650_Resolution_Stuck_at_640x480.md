### Post by aysiu on 2006-09-03
**This is the only fix I've ever had to use**
Log out of Ubuntu.

Press Control-Alt-F1. Log in again. ```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
sudo nano /etc/X11/xorg.conf
``` Replace this section: ```
Section "Monitor"
	Identifier	"Generic Monitor"
	Option		"DPMS"
	HorizSync	28-64
	VertRefresh	43-60
EndSection
``` with this ```
Section "Monitor"
	Identifier	"Generic Monitor"
	Option		"DPMS"
	HorizSync	31-83
	VertRefresh	56-75
EndSection
``` Then save (Control-X, Y, Enter) and press Control-Alt-F7 and then Control-Alt-Backspace

That should get you the correct screen resolution.

**What if it doesn't work?**
If, for some reason, it ends up worse than before, then press Control-Alt-F1 and type ```
sudo cp /etc/X11/xorg.conf.backup /etc/X11/xorg.conf
``` and then do Control-Alt-F7 and Control-Alt-Backspace again. That'll get you back to the 640x480 you had before. Then, you might want to look into [these fixes](http://ubuntuforums.org/showpost.php?p=129379&postcount=21).
