### Post by taurus on 2006-06-04
That's pretty much it.  Ubuntu uses Gnome as default window manager while Kubuntu uses KDE and Xubuntu uses XFce.  However, you can install and run KDE in Ubuntu or Gnome in Kubuntu.  The underlayer is the same no matter what window manager you are using...
