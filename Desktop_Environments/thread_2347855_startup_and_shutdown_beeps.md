### Post by Keith_Helms on 2016-12-29
Are you using Xubuntu?  Your post referenced xfce.  

The first thing I'd try is to click on volume control on the top bar, select sound settings, select playback, and then set system sounds to silence.

If that doesn't work, I'd select Settings Manager, then Appearance, then the Settings tab and uncheck event sounds.

My Xubuntu systems don't make any sounds.  It's always possible the sounds you're hearing are coming from the PC and not the OS.  In which case you'd need to go into the BIOS settings and see if there're any options to mute them.
