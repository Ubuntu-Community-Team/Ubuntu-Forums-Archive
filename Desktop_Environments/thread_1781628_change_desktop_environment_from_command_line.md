### Post by Frogs Hair on 2011-06-13
You may need to install a proprietary video driver in order to use the Unity desktop . This can be done from classic gnome . System > Administration > Additional drivers . If a driver is available them you can use Unity by selecting Ubuntu from the session list at login . Using Unity 2D is another option.
