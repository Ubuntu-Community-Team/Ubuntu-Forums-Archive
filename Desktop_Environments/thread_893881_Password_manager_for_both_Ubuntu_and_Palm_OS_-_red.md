### Post by nlvivar on 2009-02-12
I have used Keyring for many years now.
[http://gnukeyring.sourceforge.net/](http://gnukeyring.sourceforge.net/)
This allows you to save passwords on your Palm, and sync it to your computer.
In Linux, I use jPilot, which has it own plugin made for Keyring. The plugin is in Ubuntu's repository.
In MS Windows, there's a separate program that decodes the password database after you sync the Palm to the Palm Desktop.

I hope this helps.
