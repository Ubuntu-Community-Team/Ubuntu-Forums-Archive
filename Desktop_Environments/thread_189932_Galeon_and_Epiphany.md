### Post by fordfan753 on 2006-06-05
Why should people have to install Firefox, just to get Galeon and/or Epiphany running? I think it's a stupid idea, most people use only one web browser, so I'd like to see a way to install the **one** you want.

The way I see it (yes, in a primitive "I'm not a developer" way) is that the whole reason these packages depend on Firefox is because they depend on some libraries that Firefox has included.

I have no idea if this is possible with the current Firefox licencing (I'm not a laywer either, and I haven't done much research), but I would like to see an extra package, one with the libraries from Firefox that both Epiphany, Galeon, and possibly even others need. This would make the Firefox package smaller, since it no longer has all the libs in it. Then we could have all browsers that need those libs to depend on the package with those libs in it. Want Firefox? Install Firefox and the libs. Want Epiphany or Galeon, just install Epiphany or Galeon and the libs.
