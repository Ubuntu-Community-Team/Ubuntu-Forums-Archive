### Post by Hooya on 2009-02-04
Yes.  I had never tried this before, but sure enough, exactly the function you are describing works like a charm in Thunar with that other package I described installed.

Important Edit:  No, wait, it doesn't.  The option seems to appear, but it still just extracts to the directory your original zip file is in.  But what you can do, is right click, drag the file to the destination folder and the "Link Here" option does put a link in that destination folder.  You can right click the newly formed link and say "extract here" and it extracts to that directory.  An extra step, but much faster than copying a large zip file across partitions or something.
