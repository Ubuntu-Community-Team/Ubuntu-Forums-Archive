### Post by 2F4U on 2011-10-15
That is correct, kubuntu-desktop is the same as kubuntu. Having seperate Linux distributions for different desktop environments seems to be specific to Canonical. This is because Ubuntu is still the core product, while Kubuntu, Xubuntu, etc. are derivatives, often developed by people outside of Canonical.

### Post by tartalo on 2011-10-15
The default software selection is also different, if you install Kubuntu on top of Ubuntu you will have lots of programs that duplicate the purpose.

In my case installing Kubuntu on top of Ubuntu also meant that the computer went very slowly.

I'd recommend a fresh install instead.
