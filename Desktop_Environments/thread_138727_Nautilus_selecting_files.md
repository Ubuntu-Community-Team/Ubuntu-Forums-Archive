### Post by dpm on 2006-03-02
Hi everyone,

in Nautilus, in 'View as List' mode, is there a way to select files only with the mouse, without having to use the keyboard?

On most file browsers you can draw a rectangle with the mouse and select the files you want, and in fact you can do that with Nautilus in 'View as Icons' mode as well, but it seems that you cannot do that in the mentioned 'View as List' mode.

That's getting a bit annoying now, because I prefer the List view instead of Icon view, but I simply don't use it because of its crippled features. Anyway, I think what I'm trying to do is not that exotic, so I guess there should be a workaround ...

So, does anyone know whether that is possible, or is it just another one of those "Welcome to GNOME" features?

Thanks.

### Post by varunus on 2006-03-02
The list view works the same way it does in windows.  Click on a file to select it.  If you hold shift and click on a another file, it will select those two files and all files between them.  Hold down control while clicking to select files that are not next to each other.

### Post by wylfing on 2006-03-02
Eh, not exactly like Windows, because in Windows you can still band-box around a bunch of files in List View (actually the equivalent is Details View). The answer to the OP is No, you cannot band-box in List View in Nautilus. I hardly call this "crippled," however. It's just different. Try this in Windows: switch to Details View, sort by date, and then try to click *on the dates* to select the ones you want. Can't do it!
