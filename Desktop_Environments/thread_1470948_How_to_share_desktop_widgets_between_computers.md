Hi,

One of the new features for the KDE 4.4 series is that you can share any widget from the Plasma desktop with other computers in the network but I never made it working and I never saw any information of anybody talking about it.

You can enable it to the selected Widget preferences -> Share -> "Share this widget to the network" (I traslated this in english so is possible that the checkbox has a different name).
Then other computers in your network using KDE 4.4 will receive a notification message that if you click to add it will add a new widget to those computers... the problem is that is a local widget and not a shared remote widget as is supposed to be so you can't share data between computers (for example the text content of a Note).

Any one used this feature successfully or it is bugged in Kubuntu Lucid? :(
