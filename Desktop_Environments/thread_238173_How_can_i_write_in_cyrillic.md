### Post by murataht on 2006-08-18
i can't say i will solve your problem, but i can give you some tips.

in gnome, i can add keyboard layout using, system->preferences->keyboard.
there are lots of pre defined layouts. you can surely find the one you are looking for. ( i don't know which country+language you are looking for, sorry for my lack of knowladge about cyrillic). 

but the above method is for gnome. i don't know if it works for xubuntu, i mean xfce. if does not work for that, try to configure the xorg.conf file. there is a keyboard section inside. you can specify the layout you want to use there. check /etc/X11/xkb/symbols, there are the symbols you can use. 

you may post back with more informaiton about which layout you want to use, and etc.

good luck.
