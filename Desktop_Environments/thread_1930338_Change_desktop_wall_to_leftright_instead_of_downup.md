### Post by Copper Bezel on 2012-02-25
Yeah, cbanakis is talking about Unity (or Gnome 2?). Gnome 3 under Shell doesn't really work that way. The desktop switcher filmstrip is vertical, so it wouldn't really make sense for the switching animation to be horizontal. However, you can try the [Frippery extension pack]("http://intgat.tigress.co.uk/rmy/extensions/index.html"), which includes options for static workspaces that work left-to-right. It depends on a taskbar extension, though, to house the workspace switcher.
