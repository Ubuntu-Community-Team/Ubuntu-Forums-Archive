I have Xubuntu 13.04 installed and just noticed something. I have compositing enabled but have all shadows disabled. The window manager theme has shadows set to false. Yet shadows for windows and menus etc. still appear. Kind of annoying if you don't want shadows.

edit: duh me. I saw shadows were off in the theme but that was for text. turned off the shadows for frames and popups and all is well.
