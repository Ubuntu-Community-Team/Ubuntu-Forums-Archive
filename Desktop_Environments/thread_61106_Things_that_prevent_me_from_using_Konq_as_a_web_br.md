### Post by Freddy on 2005-08-30
> 1. Home page is shared with Konq file manager. I want the browser's home page to be Google and the file manager's home page to be my home directory. 
As far as I know that can't be done but if you open your Konq in browser mode, direct it to the page you wan't as your default opening page and choose "Settings | Save view Profile "Web-Browser" | Save URL in profile", at least when you open your Konq in browser mode, the default will be your choosen page.

> 2. When I'm browsing the web, if I click on my panel 'home' icon I get a new browser tab with my home directory's files. These need to be clearly separate... if I click on the browser icon I should get a browser window and if I click on the file manager icon I should get a file manager window. It works this way if I open the file manager first, but not if I open the browser first.
I haven't found any option to do this but at least you don't need a multiple windows, "Settings | Configure Conq | Web Behavior and choose open links in new tab".

This will of course still use browser mode but at least you don't need to open a new window.  /// Freddan
