### Post by brentoboy on 2006-07-18
on windows, photoshop is an MDI application, meaning, child documents live inside of a main window that houses the entire application.

Linux doesnt have MDI apps (which I think is a major improvment)

gimpshop, as aysiu mentioned, rearranges menus and changes the names of menu items so that they will be easy to navigate for those photoshop users out there who find gimp a bit of a shock.

but gimpshop is more like the macintosh version of photoshop, which , as I understand it, doesnt have the grey background parent windows to house the rest of them.

I use gimpshop, when I first used it, that was when I decided the GIMP really does do pretty much all that I actually need it to, and that the only reason I doubted that it did was because I was unfamiliar with  it.
