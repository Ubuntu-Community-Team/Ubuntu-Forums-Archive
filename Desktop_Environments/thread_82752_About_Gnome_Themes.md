### Post by 23meg on 2005-10-27
I dislike all lookalike themes, but I don't agree about unifying the look of Linux desktops. On the contrary, we need more diversity, more choices; original, unobstrusive diversity that is. After all, Linux isn't even an OS :)

I like minimalistic themes that don't depend heavily on bitmaps, and immediately rule out themes that oversize GTK widgets or use XP-like thick title bars (more than 75% of the themes on gnome-look.org fail these criteria). I use small application fonts (8px, 96dpi) with a high resolution for the same reason: I hate my screen real estate being wasted. Since I've found the perfect combination of Gperfection2 (GTK) + Blended (Metacity) I've stopped looking for anything else. Before that I was using Xfce4-winter (GTK) + Sloth / h20 (Metacity), modifying both to use smaller title bars.
