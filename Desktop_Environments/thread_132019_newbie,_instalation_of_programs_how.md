### Post by aggiechemist on 2006-02-17
First off, I would encourage you to check the Wiki for Ubuntu. This is such a common question that it is most likely covered in great detail. This Wiki has a link in the upper right of the forum screen, or can be found here:
[http://wiki.ubuntu.com](http://wiki.ubuntu.com)

But in general, the best place to start is a program called "Synaptic" for Ubuntu, but I think Ubuntu actually comes with Adept. It is under the system folder on the main menu. It calls it a "package manager" since applications in linux are often like a package of items. Play around with that, you should be able to search for programs and then just ask it to install it for you. Have fun, the thing is full of more programs than you can ever possibly need. Common ones like firefox should definitely be there.
