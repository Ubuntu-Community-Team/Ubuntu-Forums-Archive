### Post by joey-elijah on 2009-05-14
So you're trying to show how awesome Ubuntu is by dressing it up like OS X? That's, er.. a bit of a contradiction. 

To answer your question - Screenlets are the Linux equivalent of 'dashboard' widgets and if you're using compiz they can be 'hidden' on a widget layer and only be shown when you press a key or navigate to a hot corner... just like dashboard! 

As for grouping - yes there are many ways of grouping windows... 

And yes there is expose, docks, flurry screensaver, OS X icons, replacement gnome panel that acts like the mac taskbar... etc...

Surely the best way to show Ubuntu is better is to let it be Ubuntu?
