This is the best Xorg card can do. Moving windows between seperate Xscreens is not supported under Xinerama. You can drag windows between monitors connected a Nivida card that has two video outs under a mode Nvidia card can do called "Twinview". Twinview can treat the monitors like one big desktop because it is using video memory on one card and not doing video memory addressing across the bus.
