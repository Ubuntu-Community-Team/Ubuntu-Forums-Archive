### Post by stray on 2005-07-01
Real quick question here.

I remember that in Windows you could do things like Alt+0225 and get an accented character. I'm getting tired of having to use the character map in Ubuntu, so is there a quick way to enter accented characters like áéíöû?

---

### Post by Stemp on 2005-07-01
Choose the compose key (in Menu System, Pref., keyboard)
Then by using it, you'll have the accentued characters....
eg : 
[compose] ' e --> é 
[compose] , c --> ç
