Currently, KDE shows programs in the task manager panel that are open from all workspaces. I would prefer that the task manager only display applications that are open in the current workspace. How may I go about doing this exactly?

### Post by SeijiSensei on 2015-01-18
Right click on the panel and choose "Task Manager Settings."  Check the boxes for "Only show tasks from the current desktop" and "from the current activity."
