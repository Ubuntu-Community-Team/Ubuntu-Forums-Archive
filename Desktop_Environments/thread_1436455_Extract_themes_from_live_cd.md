### Post by gadolinio on 2010-03-22
Boot with the 9.10 liveCD. Go to /usr/share/themes. There you have a folder for each theme. Copy the one you want to a pendrive. Then back in your other system paste those themes in /home/YOUR_USER_NAME/.themes.
