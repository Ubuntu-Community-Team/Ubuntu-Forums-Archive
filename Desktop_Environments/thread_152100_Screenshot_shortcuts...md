### Post by adam.tropics on 2006-03-29
Ok, so <PrtSc> will take a screenshot, <Alt><PrtSc> will take a screenshot of the active window, but how in god's name do you take a screenshot of the active window with a menu, for example the tools menu open??? It can't be this hard!!!

---

### Post by Sutekh on 2006-03-29
Use
```
gnome-screenshot --delay #
```
replace # with the number of seconds you need to open a menu up.
[ATTACH]7753[/ATTACH]
The GIMP can also do it from the File -> Aquire - > Screen shot menu
