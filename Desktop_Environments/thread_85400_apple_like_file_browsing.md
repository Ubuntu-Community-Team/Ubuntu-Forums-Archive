### Post by Ampersand on 2005-11-02
Try opening nautilus, navigating to the folder containing your movies/music/whatever  and selecting Add Bookmark from the Bookmarks menu. This will add the folder to the places menu (in the Gnome menu bar, if you didn't remove it) as well as the bookmarks menu in nautilus.
