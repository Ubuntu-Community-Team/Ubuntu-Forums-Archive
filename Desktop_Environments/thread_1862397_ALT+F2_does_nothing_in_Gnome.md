### Post by Pithikos on 2011-10-16
I installed Ubuntu 11.10 and then gnome-shell. Gnome-shell was horribly slow on my system although I have Nvidia so I just boot into classic gnome.

Now the problem is that neither in gnome-shell or in gnome classic does ALT+F2 do anything. How can I fix this?

---

### Post by Kexolino on 2011-10-16
By opening System Settings > Keyboard > Shortcuts > System, selecting "Show the run command prompt", and configure a keyboard shortcut (Alt+F2).
