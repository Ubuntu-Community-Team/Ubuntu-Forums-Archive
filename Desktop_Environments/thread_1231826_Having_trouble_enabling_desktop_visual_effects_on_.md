### Post by gettinoriginal on 2009-08-05
Check System > Admin > Hardware Drivers, make sure that your driver is activated.  If not, activate it.  Close that window, now System > Preferences > Appearance > Visual Effects, either Normal, Extra, or Custom should be checked.
