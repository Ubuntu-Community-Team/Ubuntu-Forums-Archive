### Post by aysiu on 2006-07-18
They're all just different metapackages. Here's a rundown from small to large.

**kdebase** is the absolute bare minimum to run the K Desktop Environment.

**kde-core** is a typical but also fairly minimalist KDE

**kubuntu-desktop** is the default Ubuntu version of KDE with the default Ubuntu packages associated with it

**kde** is all of the KDE applications--all the bells and whistles.
