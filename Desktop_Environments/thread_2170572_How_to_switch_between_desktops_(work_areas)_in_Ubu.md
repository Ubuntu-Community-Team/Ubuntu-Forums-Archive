In previous versions of Ubuntu, there was an option on the quick launch toolbar to alternate between the four desktop work areas. I don't find this option any more in Ubuntu 13.04 and I realy need it because I have a very small screen (netbook). Any idea, please?

### Post by Kevin_Arnold on 2013-08-26
