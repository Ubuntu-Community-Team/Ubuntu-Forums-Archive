Is there a way to gain access to the third mouse button on an Acer Aspire One? I have been playing around with Fluxbox, but with no third mouse button I can't tab windows (unless there is another way). Is there some way to assign this button to a keyboard button?

### Post by RedSingularity on 2009-12-20
Sure you can assign a keyboard button.  Do you have the Compiz settings manager installed?  If so go to it and click "Commands" in the "general" section.  There you can configure your key shortcuts.
