### Post by RAOF on 2005-11-30
You can get the Sun JVM - the 32bit version contains a firefox java plugin, too.  The 64bit version doesn't, but I think the 64bit blackdown java package does.

You can find out how to install the Sun java [here,]("http://www.ubuntuforums.org/showthread.php?t=76735&highlight=java+1.5+package") and you can install the blackdown java from synaptic/apt-get (it's in the multiverse repository, so you'll need to enable the extra repositories).
