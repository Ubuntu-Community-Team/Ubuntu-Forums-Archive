### Post by varunus on 2005-11-17
That's called "gdesklets."  You can get it from synaptic package manager, the packages gdesklets (the program to run the desktop apps) and gdesklets-data (the package containing some sample desktop apps).  You can get more on the gdesklets website.  There's a good howto on how to easily set up gdesklets; search the forum and you'll find it, or install it from synaptic and give it a shot.

They work a lot like Apple's "Dashboard;" in fact, they've been around for longer...so I suspect an Apple rip of a Linux idea.  :) Though I'm sure someone can prove me wrong.
