### Post by Mr. Picklesworth on 2006-12-30
I decided to play with Xubuntu today, and it is a really great improvement in speed over GNOME (Ubuntu) on my slow computer. After discovering the applet that lets me use my Gnome panel applets (such as Tomboy), I felt right at home. Even Samba seems to work better here, and programs built for Gnome (such as Epiphany) fit in fine.
I am missing a few features like configurable menu accelerators and detaching toolbars, but I'm sure I will get used to their absence. (And some day, when the faster computer is running Ubuntu, I can use them!).

However, there is one problem: When I install xubuntu-desktop, it comes with an xfce alternative to practically all of the programs I've gotten used to in Gnome! I like a lot of these Gnome programs, and the way that XFCE programs all have names following some weird XFCE alliteration motif (one of the things which drove me off Kubuntu) kind of drives me crazy.
So is there a way to install just the base Xubuntu desktop with none of these extra applications?
I would rather add stuff later, selectively, as I see fit.

One more thing: Using Xubuntu, I really miss Gnome's Places menu. Can I get the Gnome menu in there, or (even better) is there an XFCE alternative to the Places menu? (Or maybe an XFCE panel applet that is as nicely organized as the Gnome menu?)

Thanks in advance!
