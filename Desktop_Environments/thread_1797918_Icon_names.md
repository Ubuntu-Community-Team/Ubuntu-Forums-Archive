### Post by exploding5heep on 2011-07-05
I'm working on some custom unity quick lists and want to set icons for them. I looked at a few existing quick lists and they have lines like: ```
Icon=utilities-terminal
``` Is there a list of these icon names somewhere? I've been trying to find one and can't.

---

### Post by Krytarik on 2011-07-05
Those names correspond to the image file names of the chosen icon theme, and it's set fallback themes. You will find them in "/usr/share/icons".
