### Post by Garyu on 2006-09-17
Ubuntu version numbers mean year.month, so 6.06 is June 2006 and 6.10 is October 2006. So no, Ubuntu 6.10 Edgy Eft is not out yet.

There is however the possibility to run 6.10 as a tester, where you will be able to try out the new things, but since there is still some development going on you will experience a few bugs. So I would recommend that you wait until the actual release date in October.
