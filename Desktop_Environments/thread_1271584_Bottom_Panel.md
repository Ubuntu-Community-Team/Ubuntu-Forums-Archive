### Post by mcduck on 2009-09-21
Right-click the panel, select "Add to Panel" and then drag& drop all the applets you want to the panel.

(Gnome doesn't really have any "top panel" or "bottom panel" with a defined set of features. You can run as many panels you want, in any place in the screen you want, and add any applets and launchers you want to those panels. So creating a new panel will always give you empty panel and you need to add the applets you want yourself.)
