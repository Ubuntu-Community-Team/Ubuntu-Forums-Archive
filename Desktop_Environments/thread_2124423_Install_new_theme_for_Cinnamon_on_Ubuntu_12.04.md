### Post by ManamiVixen on 2013-03-10
That theme is too old for Cinnamon 1.6. You have to pick from the 1.6 Certified themes.

Personally, Baldr for Cinnamon is gorgeous when paired with the Orion GTK theme. Make sure to download the 3.4 version of Orion if you want to check it out. Icon theme as a suggestion would be Faenza.
