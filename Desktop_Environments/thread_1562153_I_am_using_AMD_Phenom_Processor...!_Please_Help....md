### Post by 3Miro on 2010-08-27
Both will work fine. 64-bit a bit faster (depending on the app that you are running), while some 64-bit apps (like flash, some specific windows codecs and some very specific drivers) may require more tweaking to run.

I recommend you go for 64-bit, that is what I have been using for some time now.
