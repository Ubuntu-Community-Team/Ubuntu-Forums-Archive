### Post by Yashiro on 2009-03-02
It's called a Visual Bell.It's an Accessibility feature. 
Disable any visual bell options you can find. Perhaps you have been playing with the Accessibility panel? 

Try disabling audio bell too. Occasionally they are linked.

---

### Post by send4m3 on 2009-03-02
Thanks Yashiro...

After researching I think I know what make it happened. 
I accidentally set the Visual Bell to flash screen.
For all Ubuntu users that get the same things, this is how to disabled it.

Click System-Preference-Sound, then click the Sounds tab -  make Visual alert disabled.
