### Post by frodon on 2005-07-29
You can create a symbolic link of a directory to your desktop folder :```
cd /home/user_name/Desktop
ln -sf the_wanted_directory .
```

### Post by tvoss on 2005-07-29
Open up nautilus and drag'n'drop the directory, you want to link to, on the desktop while holding down ALT. You will be asked if you want to copy, move or link the directory.
