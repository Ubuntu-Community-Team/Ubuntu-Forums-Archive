### Post by manish671 on 2012-10-14
Thanks oldos2e to make it proper.  
Found the answer as follows-  Select Keyboard Layout Setting -then Select Hindi(Wx) - Choose Options  Then click 10th option "Key to choose 3rd level".  Check any option suitable to you. I selected Left Alt.
  So now   &#2309;=Alt-a,   &#2310;= Alt-Shift-a,   &#2311;= Alt-i,  &#2312; = Alt-Shift-i,  &#2313; =Alt-u,  &#2314;=Alt-shift-u,  &#2323; =Alt-o,  &#2324;=Alt-Shift-o
  &#2360;&#2306;&#2351;&#2369;&#2325;&#2381;&#2340; &#2357;&#2381;&#2351;&#2306;&#2332;&#2344; - &#2325;&#2381;&#2359;, &#2340;&#2381;&#2352;, &#2332;&#2381;&#2334;, &#2358;&#2381;&#2352;
  &#2325;&#2381;&#2359; = k-a-R,  &#2340;&#2381;&#2352;=w-a-r,   &#2332;&#2381;&#2334; =j-a-F,   &#2358;&#2381;&#2352;= S-a-r  
for creating half letter use "a"  e.g. &#2340;&#2340;&#2381;&#2325;&#2366;&#2354; =w-w-a-k-shift-a-l
  &#8377; (Indian Rupee Symbol) ctrl-shift -u20b9 ( In English Keyboard)
