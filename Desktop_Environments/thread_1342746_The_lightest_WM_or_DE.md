### Post by m_duck on 2009-12-01
The tiling WM, [dwm](http://dwm.suckless.org/) is very light. If you are only using CLI apps you could forego X altogether and just use the TTYs. You could always use something like [tmux](http://tmux.sourceforge.net/) or [screen](http://www.gnu.org/software/screen/) to make this more versatile too.

### Post by rabidbadger on 2009-12-01
[Musca]("http://aerosuidae.net/musca/Musca_Window_Manager") isn't very well known, but a pretty straightforward tiling WM. If you can live with some odd behaviour (odd, unless you were an Amiga fan), a GUI WM that's worth a look is [WindowLab]("http://nickgravgaard.com/windowlab/"). This isn't EWMH compliant, though, but if you want something simple -- no docks etc. -- it's pretty nippy.
