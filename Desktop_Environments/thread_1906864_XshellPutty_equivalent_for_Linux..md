### Post by przemnet on 2012-01-10
Hello,

I'm looking for a programe which is an equivalent for Xshell/Putty under Linux. I know that Putty is avalible in repos, but it does not support tabs and in my case it has problems with logging session in the file- it stops in some point and the only way to make it work again is to restart Putty- quite annoying stuff...

I know that I can connect via ssh with **ssh** and telnet with **telnet** via Gnome Terminal/Konsole, save session output with **script**, keep connection configuration in text files but this solution is not suitable for me.

I'm looking for a ssh/telnet client with nice and user frienly GUI that will allow to:
- save session output in a text file
- predefine connections (define IP address, port or protocol, log file etc.) like in Putty
- use tabs for opening new connections and easy navigate between them (keyboard shortcut)
- configure fonts (type, size colour)

Virtual Terminal Emulators like Gnome Terminal, Konsole, Guake, Terminator, Yakuake, Yakuake, Eterm, Rxvt, Aterm, Wterm, Xterm, Tilda, Stjerm do not support all above features.

If someone was uisng Xshell, than knows what am I looking for ;-)

Thanks in advance for answers!

Looking forward,
Przemek.


***SOLUTION:***

**PAC Manager**
WWW: [http://sites.google.com/site/davidtv/](http://sites.google.com/site/davidtv/)
Download: [http://sourceforge.net/projects/pacmanager/](http://sourceforge.net/projects/pacmanager/)

**GNOME Connection Manager**
WWW: [http://kuthulu.com/gcm/](http://kuthulu.com/gcm/)
Download: [http://kuthulu.com/gcm/?module=download&lang=](http://kuthulu.com/gcm/?module=download&lang=)
