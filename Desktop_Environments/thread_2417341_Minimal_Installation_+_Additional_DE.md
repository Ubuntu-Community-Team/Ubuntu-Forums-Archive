Is it possible to install the Budgie DE without installing the entire Ubuntu-Budgie distribution?

---

### Post by Frogs Hair on 2019-04-22
Yes, I used the following along side gnome on 18.04/10  and it does not install default applications or boot screen. To get all the panel applets install the backports ppa.   [https://launchpad.net/~ubuntubudgie/+archive/ubuntu/backports](https://launchpad.net/~ubuntubudgie/+archive/ubuntu/backports)

```
sudo apt install budgie-desktop 
```

Budgie is still based on gnome.
