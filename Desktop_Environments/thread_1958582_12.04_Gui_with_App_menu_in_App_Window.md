### Post by Frogs Hair on 2012-04-14
The Gnome Shell and Gnome Classic have the application menu in the application window . Unity will not give you that option by default except in Firefox and the requires disabling global menu integration. I remember reading about disabling App Menu in Unity. If you are planning to disable the Unity features you might as well use Gnome.
