### Post by mdwuznik on 2006-08-24
Actually install 5.10 and upgrading afterwards to 6.06 may be a better idea for the people who use more than en_whatever locales. 6.06 installed directly seems partially translated (like 2/3 non-English, 1/3 English commands in the menu) in quite numerous places, whereas 5.10 upgraded after language environment configuration to 6.06 doesn't suffer from that language schisophrenia(hmm, I'm not sure I spelled it OK). So I would recommend installing plain breezy 5.10, configuring all the locales/languages one wants to use, and than a very simple dist-upgrade to dapper (and installing the non-installation-cd-included software after upgrade).
