I have a Network Administration Question for any Ubuntu gurus out there.

How do we set up the network so that users can log into the network from any computer they sit down at in the lab, without creating an account for each user on each machine?  Also, their home directory should be on a central file server so they can get to their work from any machine.

Say we have six computers in the lab.
Say we have 10 people who use the computers.

On Monday Joe sits at computer#1 and does some work and saves it in his home directory, which is on the local hard disk.

On Tuesday Joe comes back and computer#1 is being used by Sally, so he has to go to computer#2.

First, he needs an account on computer#2.
Second, the work he did Monday is not on computer#2 anyway.

So we need a way to administer the user accounts centrally for users on all machines.
