### Post by aysiu on 2006-07-02
Alt-F2 ```
gnome-search-tool
``` Search for *.* within the folder you want.

Highlight all the files found and then copy and paste them where you want them to be.

A similar process can be done with Alt-F2 ```
kfind
``` if you're using KDE.

Both Gnome and KDE's search tools will search recursively.
