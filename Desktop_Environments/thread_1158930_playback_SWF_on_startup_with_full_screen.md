### Post by tehkane on 2009-05-14
Well, firstly, you're going to need to find an SWF player that'll accept the SWF file as a parameter from the terminal and also support full screen playback.

No such player comes, by default, on Ubuntu. So you'll have to search around. If it's just an SWF video, perhaps you could convert it to an actual video file then use VLC? If it's an interactive file, you'll have to take a different route.

As for getting it to run on startup, I'm a Linux noob, but, I think you add it to /etc/init.d/. Either try that and see what happens or wait for someone who knows what they're talking about to come and help :P

Hope that helps.
