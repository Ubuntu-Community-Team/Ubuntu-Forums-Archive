### Post by Jucato on 2006-03-23
1. Go to Settings > Configure Konqueror > File Associations
2. Type in the search bar the file type you want changed (mp3 for example, it's under the audio group as x-mp3)
3. Go to the Embedding tab (it's at the right) and check on "Ask whether to save on disk instead".

Hope that works.
