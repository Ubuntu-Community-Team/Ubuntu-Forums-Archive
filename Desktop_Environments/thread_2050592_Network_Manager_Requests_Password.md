### Post by MoebusNet on 2012-08-31
This is a minor issue, but in Lubuntu 12.04 whenever I need to connect to a WiFi hotspot, Network Manager (I assume) asks for my password before it will connect. Then everything is fine.

The other *buntu variants I've tried don't do this; is this a choice the Lubuntu team has made? If I want to eliminate the need for a password to connect to WiFi (like other *buntus) what do I need to do?

TIA

### Post by Dennis N on 2012-08-31
I've had that problem too with Lubuntu. For me, I did the following and it solved it:

Go to edit connections (right click on signal strength bars), select your wireless connection, edit, and check the box at the bottom which says something like "make available to all users".

After closing the window, you may need to log out and back in, or it may simply reconnect (don't recall which). After that the problem did not recur.
