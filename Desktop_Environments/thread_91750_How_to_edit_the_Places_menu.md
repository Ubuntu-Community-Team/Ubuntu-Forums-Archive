### Post by casper_2095 on 2005-12-01
Sufficiently non-intutive for me to have to come looking...

Using Nautilus...

First navigate to the directory you want as a Place/Bookmark
Then select "Bookmarks->Add Bookmark"    (or hit control-D).

You need to 'edit bookmark' then 'delete' to get rid of it.
