### Post by urukrama on 2010-07-05
Pypanel has not been in the Ubuntu repositories since Intrepid. 

Pypanel has not been updated in a long time. You could try installing a .deb from an [earlier Ubuntu version]("http://packages.ubuntu.com/intrepid/pypanel") (which I suspect will work fine), or compile pypanel from source using the [source code from its website.]("http://pypanel.sourceforge.net/")

If you'd like another panel that is somewhat similar to pypanel, try tint2. If you configure it right, you can make it look like pypanel quite easily.
