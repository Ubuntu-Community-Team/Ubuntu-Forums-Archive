### Post by Sutekh on 2006-04-16
Hmm, you should fine somewhere/someone who has broadband and use the LiveCD there to download the KDE packages.

Of course this isn't always possible so the other option, if you want to save space and time, is to download a minimal KDE desktop.

Having a look through [Ubuntu Packages](packages.ubuntu.com),  it seems that [kubuntu-desktop](http://packages.ubuntu.com/breezy/misc/kubuntu-desktop) is the full Kubuntu desktop,  [kde](http://packages.ubuntu.com/breezy/kde/kde) is the full KDE desktop and [kde-core](http://packages.ubuntu.com/breezy/kde/kde-core) is a minimal KDE desktop.

So maybe downloading [kde-core](http://packages.ubuntu.com/breezy/kde/kde-core) would be the best option to try.
