Don't try to uninstall konqueror. it is a central part of KDE. I doubt KDE would work without konqueror.
Anyhow, it is A LOT faster than firefox; give it a try... if you don't like it, you can still use firefox...

You can use Synaptic. Just install & run it.

Kaffeine is not primarily meant to play music. It DOES play music, but it's interface is geared towards a video player.
Use amarok instead. It has been repeatedly voted as the best linux music player. If you did a normal Kubuntu install, it should already be installed.
And it does playback *.m3u playlists.

As for the theming-issue, if you visit kde-look.org, and find something you like, there is always a "How to install..." link. Just click it, everything will be explained.
