### Post by pelle.k on 2006-11-18
Hello, and welcome to the community!

The driver you should use is ati. You could choose it when running "xorgconfig" in a terminal, or just edit the file with "nano". If you chose to edit /etc/X11/xorg.conf with "nano", then remember this shortcut (or key-combination if you will), ctrl + x means "save and quit".
```
Driver "ati"
```
This is where it should be```
Section "Device"
	Identifier	"ATI Technologies, Inc. ATI Default Card"
	Driver		"vesa"
	BusID		"PCI:6:0:0"
EndSection
```

Also, you can try to remove the @60 in the last mode-line (24 bit)
This is where i found it```
	SubSection "Display"
		Depth		24
		Modes		"1440x900@60" "1280x1024" "1280x960" "1152x864" "1024x768" "832x624" "800x600" "720x400" "640x480"
	EndSubSection
``` and make it just 1440x900. Because
```
Modeline    "1440x900" 106.5   1440 1520 1672 1904   900 903 909 934 -hsync +vsync
``` is only valid for any "1440x900" mode-line in the screen section.

Also, where did you find that modeline? The reason i ask is because i've got a 1440x900 screen, and while i have no need for such a "modeline" in ubuntu, i do use it in other distros, and it doesn't look like that. (yes i also use 60 hz)

Do you understand me?
Tell us how it works out! :)
