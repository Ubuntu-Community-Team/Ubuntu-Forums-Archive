Hi!
For IM, I use Pidgin, and sometimes finch.
For audio, depends. I like Exaile, which has lots of options, like Amarok. But most of the time I use MOC (Music on console). It's run from a terminal and can play all my files, including mp3 and m4a.
To install exaile, the command is:
```
sudo apt-get install exaile
```
Remember you need to have installed the gstreamer codecs to get mp3 and m4a.
To install MOC (give it a try, it's SO COOL! :) ):
```
sudo apt-get install moc
```
and to get m4a support:
```
sudo apt-get install moc-ffmpeg-plugin
```
The command to run MOC is:
```
mocp
```
Cheers!
