### Post by CautionaryX on 2007-06-12
The free version of AVG Anti-virus does have a linux port. It's simple to use to me b/c I'm familiar with the interface, the only thing is to get updates one has to  ```
sudo avgupdate -o
``` in the terminal to recieve definition updates.
```
sudo avggui
``` allows you to scan the entire partition for viruses (not that you need to, but still...).
