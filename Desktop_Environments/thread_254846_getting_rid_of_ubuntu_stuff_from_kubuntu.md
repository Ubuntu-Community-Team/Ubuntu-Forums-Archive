### Post by paddy1978 on 2006-09-10
There's some info here ([http://www.psychocats.net/ubuntu/index.php]("http://www.psychocats.net/ubuntu/index.php")) on adding/removing KDE and Gnome.

If you've only 256Mb, my guess is that it will already be using the swap if you have many things running at once! You don't want to use more swap because disk access is A LOT slower than memory access, and it will only be using the swap once all the memory is taken anyway. Have you considered getting more memory - it's a really cheap way to really noticably improve your PC. I usually get mine second hand and cheap on eBay! Go to, say, 512Mb and  you'd notice a huge difference.

Have you tried Xubuntu? I used to use it on a very low-spec laptop (128Mb) and it still ran pretty well.

Don't know anything about startup files I'm afraid - sorry.
