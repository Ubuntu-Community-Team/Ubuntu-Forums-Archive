### Post by jonesy on 2005-10-17
first, comment out the "dri" module line by putting a "#" in front, you don't need to load DRI cause it's built into the driver.  Second, make sure that the driver is loaded by typing:

sudo modprobe nvidia

Lastly, make sure you exit out of X-windows by rebooting or pressing ctrl-alt-backspace while in X.
