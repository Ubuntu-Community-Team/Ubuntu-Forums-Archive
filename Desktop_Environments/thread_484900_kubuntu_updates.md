### Post by bailout on 2007-06-26
Ubuntu and kubuntu packages are all in the same repos so the kde stuff will be updated through synaptic just like the gnome stuff. However, the kubuntu devs sometimes release new packages for kubuntu that they put in seperate repos which is what you may have seen before. Normally ubuntu only updates security fixes to release packages. If a new version of a program is released it doesn't get added to the repos until the next release of ubuntu. The kubuntu devs sometimes package these new app versions for the current release and put them in the seperate repo to get around this policy.
