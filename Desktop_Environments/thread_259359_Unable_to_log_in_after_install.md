### Post by lagagnon on 2006-09-17
Asking for a username is definetely part of the install process. You probably hit your enter key too quickly and by-passed that page.

The easiest way is just re-install. You could enter repair mode from the LiveCD and use command line commands to create a new user but I would not recommend that process for a newbie.
