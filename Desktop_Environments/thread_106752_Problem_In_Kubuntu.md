### Post by localzuk on 2005-12-21
If there is no error message then it sounds to me like your system is suffering from hardware issues. It could be any of: graphics card, processor (overheating), memory or motherboard, or even your power supply.

For hardware issues the best way to try and identify the specific problem is to replace each bit with a temporary component and see if that fixes it. If you don't have spares, try a computer shop - they will have a look for you (most only charge ~£40 - 50 to check a computer over and identify/repair issues + parts).
