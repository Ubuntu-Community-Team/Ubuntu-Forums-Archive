### Post by calrogman on 2009-03-14
Presumably you don't want to go and try to compile Gentoo or install Arch just yet, so I recommend (as the next logical step from Ubuntu) that you install Debian.

The latest release, "Lenny", has a very nice graphical installer, as well as a text based installer for people on *really* weak hardware, not only is it easy to install whatever desktop environment you want (out of a selection between GNOME, KDE, Xfce and LXDE (atleast with the net install disc)) but if you want to build a system from *nearly* ground up, you can install a very basic system by deselecting everything at the "install extra software" screen.

And hey, you might even get some geek cred. ;)
