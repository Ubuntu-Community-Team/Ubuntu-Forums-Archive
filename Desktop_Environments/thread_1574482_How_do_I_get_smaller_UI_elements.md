### Post by 3Miro on 2010-09-14
What I can think of is to go to System -> Prefs -> Appearance and change the fonts. If you decrease the font size (or maybe even font type), this will decrease the size of the windows and tabs. You can also try to change that in the program itself (if it has that option).

There might be something about the size of the toolbars as well.

I don't know if you have already done this or not, but you can also play around with the positioning of the panels as well as all of the applets. This may increase some vertical space.

### Post by 3Miro on 2010-09-19
I got it. You have to install another theme. The size of the tabs and other such elements is controlled by the theme that you have enabled. You can go to Synaptic and/or Ubuntu Software Center, search for "theme" and install everything that has to do with Gnome and/or GTK. Then from System -> Prefs -> Appearance you can select different themes. You can also select "customize" for even more choices.

You can also go to:
[http://gnome-look.org/content/show.php/Aero-clone?content=57352](http://gnome-look.org/content/show.php/Aero-clone?content=57352)

and download a theme that supposedly looks a lot like windows 7.
