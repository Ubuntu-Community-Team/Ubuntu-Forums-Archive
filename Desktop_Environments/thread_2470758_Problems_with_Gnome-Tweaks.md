### Post by Dennis N on 2022-01-10
You now have an app called** Extensions** that handles enabling/disabling gnome-shell-extensions. Search for that in the Activities Overview. I think it is installed by default in 21.10, but if not, install the package gnome-shell-extension-prefs (you can search for "Extensions" in Ubuntu Software)
