### Post by Bartender on 2011-05-16
The modem doesn't get a number assigned to it.  You tell the modem which number to call.  In other words, you have to use a PC that's able to go online.  DSL, wi-fi, dial-up, even go to the library if you have to.  

Then you find a dial-up internet service provider (ISP) who provides one or more numbers that are local (not long-distance!) to your house.  Then you set up an account with them.

The ISP has to be compatible with Linux.  toast.net, Copper, Fry's; there are hundreds of them.  NOT Juno, PeoplePC, etc.

Then be prepared for the joys of connecting Ubuntu to a dial-up connection. :roll:
