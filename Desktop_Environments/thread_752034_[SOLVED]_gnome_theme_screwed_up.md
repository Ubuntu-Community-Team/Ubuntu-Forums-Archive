### Post by daranz on 2008-04-11
Go to System > Preferences > Appearance and enable one of the themes there (Human is the default Ubuntu theme).

If that doesn't do the trick, you might've accidentally enabled compiz and for some reason it isn't working right for you. You can disable it by selecting None in the Visual Effects tab of the same window.
