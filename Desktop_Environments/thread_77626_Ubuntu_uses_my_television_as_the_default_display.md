### Post by TimoP on 2005-10-17
I have Geforce 6600gt and I use tv-out. When I install the nvidia-glx driver it automatically sets the tv as the display it uses. There's nothing on the monitor. It installs correctly if I unplug the s-video cable from the tv. I have got tv-out working using [this]("https://wiki.ubuntu.com/NvidiaTVOut?highlight=%28tvout%29") guide in the wiki. The problem is that it still uses the tv as the default display. When Ubuntu boots the login screen is displayed on tv and not on my monitor. Also when I start some programs, they start at the tv's desktop and not in the monitor, which is annoying.
