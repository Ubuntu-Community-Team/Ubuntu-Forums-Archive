### Post by mrkona on 2013-02-18
I'm running Ubuntu 12.04 using the Gnome Classic Desktop.  In the Nautilus file manager when a file is selected a status text area with yellowish background is displayed at the bottom of the scroll frame that shows the filename and size.  Similarly for when a directory is selected.  I think this is called a "floating status bar."  How do I disable this?  Or create a permanent status bar that does not cover the last item in the scroll frame.  This floating status bar covers the last item which is a terrible design as the whole point of a file manager is to see the files.  Also when attempting to rename the last file the floating status bar covers up the editable text.  Again this is a serious flaw.   If this so called feature cannot be changed then what do users do to get around this?  Change to a different file manager?
