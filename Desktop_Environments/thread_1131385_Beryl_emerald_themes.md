### Post by collinp on 2009-04-20
I cannot answer the first question, but I can answer the last one: installing Beryl. Yes, it can be done, but I strongly recommend against it. Compiz Fusion offers the same features as Beryl (as it was/is a merge between Beryl and Compiz), plus the features of Compiz, and it is maintained. Beryl has many bugs, may not work right under newer software, does not have as many features, etc. The easiest way is to just use Compiz Fusion.

### Post by Jhongy on 2009-11-09
the first replies here are BS -- you do not need to "install Beryl" -- Beryl is dead, and is now Compiz Fusion, which you already have.

You just need to install emerald, which can be done via synaptic.

Then, in your compiz settings, put emerald as your window decorator. You can then install the themes using the emerald theme manager.
