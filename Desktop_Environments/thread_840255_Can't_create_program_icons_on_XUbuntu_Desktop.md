### Post by Revhead on 2008-06-25
Here I am as fresh as a daisy and already giving advice.
If you navigate in the file system to the folders "usr" then "bin" using your file manager you'll find that's where it hides the programs.
Pick one with a colored icon (the executables I'm guessing), right click it and then select Desktop (create link).
Bingo you should have a shortcut on your desktop.
