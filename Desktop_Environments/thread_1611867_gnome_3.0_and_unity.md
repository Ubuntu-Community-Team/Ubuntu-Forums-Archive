### Post by 3Miro on 2010-11-02
Unity is Gnome. The difference between Unity and Gnome-shell is like between Compiz and Metacity. Right now you can chose to run Ubuntu with Gnome and Compiz for effects or Metacity without effects. In the case of Unity, you will have the choice of Gnome + Unity for effects or Gnome + Gnome-shell for effects. The default will be Unity and worst case scenario you will have to get Gnome-shell from the Software Center.
