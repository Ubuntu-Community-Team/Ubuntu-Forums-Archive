### Post by RCC2k7 on 2007-05-04
Other than the sudo command on a terminal, the only other thing I know (and I use a lot) is gksudo from the Run Command (ALT+F2) dialog.

To open a folder with sudo privileges, type...

**gksudo nautilus /path**

... replacing **/path** with the actual path you need, such as /boot/grub or /etc.
