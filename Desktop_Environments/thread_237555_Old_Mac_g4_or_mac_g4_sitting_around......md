### Post by 3rdalbum on 2006-08-16
The question is less about whether the card can handle Compiz, but more like "Will the open-source drivers for the card run Compiz properly?"

The answer is "not terribly likely", and of course the proprietry ATI drivers aren't available for PowerPC. I think the G4s came with the ATI Rage 128, which means you'd have to use the "ati" driver - I don't know what kind of performance you'd get on Compiz.

To answer your later question: Yes. On OS 9, if you go to the Apple menu and then come down to "Apple System Profiler", you should be able to find out the model number of the graphics card.
