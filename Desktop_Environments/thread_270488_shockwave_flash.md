### Post by ske1fr on 2006-10-03
> **moon_dog said:**
> how do you save an embedded shockwave flash clip?  is there some special software you need?
No, no special software required.  For you to view the Flash clip it must download into your browser cache...so find that, open a file browser window into it, sort the files into descending date order, and the one at the top should be your Flash file.  Make sure you wait for it to finish downloading, then copy it to a new location of your choosing, renaming it to something more meaningful than, for example, ZL4HjBsSLc. 

I tried VideoDownloader but it still meant renaming, so I ditched it.
