### Post by Anquietas on 2012-04-09
Hello,

I'm using Kubuntu, so I use KDE and "ark" as an archiving tool.

I know that the RAR file format is proprietary and I don't want to use it!

When I right-click a file, in the "Compress" menu, I have "Compress as RAR"... If I click it, it says that it cannot find the "rar" program. OK.
Don't worry, I do NOT want to use it!

Now, how can I remove the RAR context menu and RAR support from ark ? Because I will not use rar. Only ZIP or TAR/GZ or other open formats.

Thanks.

---

### Post by Zorael on 2012-04-09
In Dolphin; Settings -> Configure Dolphin -> Services, untick Compress: As RAR Archive.
