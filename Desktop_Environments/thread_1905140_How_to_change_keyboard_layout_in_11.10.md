I've installed from bootable DVD the Oneiric (11.10) (It's Kubuntu). My problem is that I can't switch keyboard layout to my native language (Russian). ### Post by drmrgd on 2012-01-06
You can find keyboard layout settings in the Input Devices of System Settings.  Once in there, select the Layouts tab, and then put a check mark in the "Configure layouts" box.  That will allow you to click the "Add" button and add the Russian keyboard.  

Hope that helps!
