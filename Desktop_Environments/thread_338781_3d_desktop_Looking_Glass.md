### Post by Hendrixski on 2007-01-14
I use GNOME with Beryl, I've tried KDE briefly, can't wait for Enlightenment to get stable, but I wanted to try out something new.  So I checked out Looking Glass as yet another desktop manager.

There's a very easy install package on lg3d.dev.java.net and I had no problem getting it up and running.  It's got some interesting ideas on what to do with a 3D desktop, and a few not-so-brilliant ideas.

I haven't gotten to show it to any of my friends yet, and I will be working out of town at a client site for the next 2 weeks, so I was hoping to talk about it with some of the people at Ubuntu Forums.

There is a forum from a few months ago with a few posts about how slow it ran on people's computers, but it runs fine on mine.  I'm more interested in what people have to say about the features that it offers.  Or any cool tricks that they can share (like how to add applications to the start menu rather than open them from the command prompt),

If you haven't tried it yet, you should, it's different.  And of course write about what you like and don't like about it HERE.

### Post by floke on 2007-01-15
I really, seriously, absolutely definitely would not bother with this. It's rubbish!
Might be worth checking out in the future, but right now it's a load of old pants.
Basically you get three desktops which you can pan between (a la XFCE or E17), but set against a panoramic backdrop. Its very buggy (couldn't get OOo to run on mine) and largely pointless. Why anyone would want to have this when we now have Beryl and Compiz is beyond me (apart from the curiosity value of course!)

For anyone interested I would check out some of the other threads on this issue, they all report the same thing.

Give it another whirl in six months maybe, but for now its Beryl all the way (until Compiz gives me a transparent cube!)
