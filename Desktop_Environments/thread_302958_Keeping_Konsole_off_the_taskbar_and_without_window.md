Except, of course, for the obligatory one thing.

I like having a inconspicuous Konsole (terminal for non-KDE folk) window open all the time, integrated with my desktop.  A long time ago, a learned forumite somewhere taught me this command:```
kstart --window Konsole --alldesktops --skiptaskbar --skippager konsole --nomenubar --notoolbar --noframe --noscrollbar
```

This launches a konsole with no window decorations and with no taskbar or pager entry, so I can really integrate it into my background.

I'm not sure how to do this in Beryl.  I had some luck on the Beryl forums: [http://forum.beryl-project.org/topic-6007-selectively-hiding-window-decorations](http://forum.beryl-project.org/topic-6007-selectively-hiding-window-decorations)

Ideally, I'd like to have it skip the taskbar and the expose window pager.  Is that possible?
