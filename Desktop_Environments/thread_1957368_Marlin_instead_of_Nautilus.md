### Post by kiirokurisu on 2012-04-13
The core principles in Marlin are excellent: simple, extremely fast. It's really just a case now of squashing bugs and adding features. Example: network browsing didn't work very well at all last time I tried it. Definitely not ready to be used as the default file manager, but once it gets there I think it will be superior to Nautilus.
