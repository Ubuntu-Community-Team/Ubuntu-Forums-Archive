### Post by Arndt on 2006-06-29
[QUOTE=Lster]An application I need requires ncurses. How can I get it?[/QUOTE]

There are a number of packages with "ncurses" in their name. I think the ones you may need are:

```
ncurses-base
ncurses-bin
libncurses5
libncurses5-dbg
libncurses5-dev
```

Perhaps libncurses5-dev alone will suffice.
