### Post by deathbyswiftwind on 2006-06-21
Yes Ubuntu does with the right kernel installed. If you do decide to go with Ubuntu after you install your system you will want to enable all repositories and then open synaptic package manager and install the kernel ending in smp depending on what brand your processors are will determine which kernel you install
the options i know are as follows
i386 - For pentium and pentium 2
i686 - for pentium 3 and pentium 4
k7 - for amd athlon and up
