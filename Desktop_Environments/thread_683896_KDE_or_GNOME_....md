### Post by ajgreeny on 2008-01-31
If you install Ubuntu, use synaptic or apt-get to add kubuntu-desktop, and if you install Kubuntu do the same but install ubuntu-desktop.  Now at boot time you can select either gnome or kde by clicking on the Options button on the login window.

That's the way I always used to do things in previous ubuntu versions but have now decided that gnome is the one for me, with just a few kde applications for certain activities, eg k3b, because it's the best CD/DVD burner software on any platform, as far as I'm concerned, and digikam for photos, which I like better than f-spot.

Give them both a try and you may find you use one a great deal more than the other.  Then you can just get rid of the bits you don't want or need any more.  You won't even have the same problem that windows has of leaving lots of rubbish in registry, slowing the machine to a crawl.
