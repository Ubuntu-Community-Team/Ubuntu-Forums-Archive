### Post by Soarer on 2006-09-10
Hi and welcome to Ubuntu.

Sadly, as far as I know, you cannot install new programs when using the Live CD witout a hard disk. The idea of it is to see if you like it, and if it dectects all your hardware correctly. After that, if you are happy, the normal procedure is to install it on a hard disk, which sadly in your case you don't have.

In order to listen to mp3s etc. you need to install some codecs. This is really easy to do - if you have Ubuntu installed on a hard disk. CR-ROMs are, as you probably know, read-only so there is nowhere to load these coedcs to.
