### Post by ajgreeny on 2014-06-01
In XFCE Settings manager go to Window Manager ->Keyboard and there you can set what you want to Tile Window to top, bottom, left or right.

I am still on Xubuntu 12.04 at present, but I have updated the XFCE4 version to 4.10 instead of the default 4.8, however, I am not sure what happens now in Xubuntu 14.04 nor if things have changed.
