### Post by deadflowr on 2019-04-24
alt + tab + down, or alt + ~
then use right or left and then enter to select.
There's also overview (press the super key or click on Activities up top) which will show all open windows on the current desktop workspace.
(alt+tab, though, shows all windows across all workspaces, so that's better if you scatter programs across multiple workspaces)

And if that's not what you want you can always dig around for a gnome shell extension, either in Software (the software store) or browse gnome shell extension web site here:
[https://extensions.gnome.org/]("https://extensions.gnome.org/")
