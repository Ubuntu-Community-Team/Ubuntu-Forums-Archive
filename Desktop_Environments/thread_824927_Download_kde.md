### Post by benerivo on 2008-06-10
You could do any of this through Synaptic Package Manager rather than the command line. Which kde do you want -- kde3 or kde4?

sudo apt-get install kdebase

or...

sudo apt-get install kde4-core

These packages are kde3 and kde4 respectively, and will give you the basic desktop environment. If you want the full Kubuntu desktop, then it's...

sudo apt-get install kubuntu-desktop

or...

sudo apt-get install kubuntu-kde4-desktop
