### Post by Nytram on 2006-09-06
The default "system new mail sound" is the beep you hear from the speaker.

To jazz it up a little:

Download a new mail sound file, they have some at this site:

[http://www.eventsounds.com/mail.htm]("http://www.eventsounds.com/mail.htm")

Save the file into a convenient place, such as your home folder.

Then, go into Thunderbird options, choose a custom .wav file for your email notification, and browse to the file you downloaded.
