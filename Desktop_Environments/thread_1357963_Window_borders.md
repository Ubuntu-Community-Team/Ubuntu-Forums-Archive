### Post by jcoles on 2010-09-26
I just experienced the same problem with Ubuntu 10.04.

I was able to get the window frames back using the Visual Effects tab of Appearance Preferences. When I opened the tab, the None option was selected. I selected Normal instead. When the visual effects were enabled, my window frames returned.
