### Post by bikeman on 2006-01-10
Yngve, there's a couple of things.  First, you need to re-burn a new CD (if you really copied the .iso).  Rather than "copy" the .iso to the CD, you make the CD from the image (the .iso file is an image).  That should be an option in your CD-burning application (create CD from image, or similar).  I recommend burning at the lowest speed that your burner supports, it seems to burn with fewer errors.  If this is how you created the CD, then that is fine and you don't need to do it again.

Once you have the CD correctly burned, you install the OS by booting your PC from the CD.  The easiest way to do that is to put the CD in the drive and then re-boot (of course, your PC BIOS must be set to boot off the CD before booting from the hard drive).

But, since you are not very experienced in computer handling, please, **please** start with the Ubuntu Live version which runs off the CD rather than installing Ubuntu to your hard drive.  Linux is very different than Windows, and it is easy to run into problems if you don't know what you are doing.  Installing Ubuntu could erase your Windows OS and all of your files depending on the options you select.  The Ubuntu Live version (downloadable and free just like the installable Ubuntu) will allow you to run Ubuntu, learn the commands and applications, and allow you to make mistakes without losing data.  The Wiki that Leif gave a link to is a good start, as are these forums.

Good luck!
