### Post by teachop on 2011-05-07
Firefox will allow web pages to set their own fonts.  You can override that.  In Firefox preferences, go to the Content tab, pick Advanced and start changing things gently, starting with unchecking "Allow pages to choose their own fonts".  Some pages don't like being forced, but I think you have to do this in the applications.
