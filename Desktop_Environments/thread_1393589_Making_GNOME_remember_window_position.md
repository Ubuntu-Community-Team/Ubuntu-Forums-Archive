### Post by ThinkBuntu on 2010-01-29
Is there any way to do make Metacity (or whatever else may be responsible) remember window position of closed applications, in Ubuntu 8.04, GNOME 2.22.3? It is very frustrating having everything open up in the top left corner.

---

### Post by howefield on 2010-01-29
If you are using compiz, you could go to System > Preferences > CompizConfig Settings Manager > Window Management > Place Windows > Placement Mode > Centered.

Not sure about a non compiz way, but it sounds like there should be ;)

---

### Post by chewearn on 2010-01-29
Unfortunately, most applications don't remember their position before quiting, so they don't open again in the same location.

For non-compiz, you might want to try: [https://help.ubuntu.com/community/Devilspie](https://help.ubuntu.com/community/Devilspie)
