### Post by practic on 2011-06-10
You can do it with Search Folders. It was discussed here:

[http://ubuntuforums.org/showthread.php?t=1568699]("http://ubuntuforums.org/showthread.php?t=1568699http://ubuntuforums.org/showthread.php?t=1568699")

...which references a blog article here:

[http://www.oak-tree.us/blog/index.php/2009/10/27/unified-inbox-evolution]("http://www.oak-tree.us/blog/index.php/2009/10/27/unified-inbox-evolution")

I just set this up today on my computer, and it works well. Only problem is that Search folders end up at the bottom of the list, but Evolution remembers which folder you had selected, so when you start it up, it will go straight there.

If you include the Sent Items as well as the Inbox folders in your Search folder, then you can view message threads (emails and responses) in one place...you might find that useful also.
