I'm sorry if I am repeating an old query but there are 250 posts listed as relevant to the above title and the ones I have read do not answer my question.
I am running Xubuntu and on the desktop I have an icon, a picture of a folder, which takes me to a file when I click it. To launch the program contained therein I have to open the file, click on the .exe file and select Wine Windows Program Loader. I would like to replace the folder icon for one that is associated with the specific program but have not found a way to do this. The folder icon is not a program launcher as such and therefore when I 'right click' it I do not get the option to edit it and change the icon.
Is there a 'work around' for this minor problem please? I realize that changing the icon will not alter the procedure for launching the program.

---

### Post by kerry_s on 2015-02-14
just right click on the desktop-> create launcher, point the command to the *.exe file.
right click on the *.exe file-> properties, set "open with" to wine.
