### Post by avik on 2007-09-21
What does your current theme look like?  Is it what you had before Compiz-Fusion?  In that case, you're using the default window decorator, meaning you get the same themes you would have always gotten.

Otherwise, you're probably using Emerald already.  Type in Alt-F2 and type in emerald-theme-manager.  If it works, then you're good to go.  If it doesn't, install the emerald package.

Hope that helps.
