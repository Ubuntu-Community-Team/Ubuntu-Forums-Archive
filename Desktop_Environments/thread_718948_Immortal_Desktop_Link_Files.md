### Post by mrund on 2008-03-08
I've got a Windows NTFS partition on my machine apart from Ubuntu Gutsy. Recently I created a few links to files on that partition and dragged them to my Gutsy desktop, Now I can't delete the links. "Error: not on same file system", it says. How can I get rid of them?

---

### Post by mrund on 2008-03-11
Found a solution. I created links to files on a linux drive, gave them the same names as the immortal files on my desktop, and dragged them onto the desktop. Gnome asked me if I wanted to replace the old files, I said "yes please", and away they went.
