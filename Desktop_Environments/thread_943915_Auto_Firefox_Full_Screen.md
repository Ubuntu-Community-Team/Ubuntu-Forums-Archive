### Post by alberto ferreira on 2008-10-13
That's because firefox stores the last session's window size and placement, however if you have used F11 once to restore it:

if it's still in full window mode press F11

Now, resize the window and quit firefox.

Everytime you use F11 to put firefox in full-screen mode you must do these steps to put it back to normal.
