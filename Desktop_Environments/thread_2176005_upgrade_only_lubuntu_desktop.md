### Post by sudodus on 2013-09-22
It depends. If you have multiple boot yes. But if you have one system with several desktop environments, no.

If you are thinking about end of life, Lubuntu 12.04 has eol next month while Xubuntu lasts until April 2015, so there is no hurry to switch Xubuntu to a new version. In order to keep a secure system, you could remove the Lubuntu specific packages.
