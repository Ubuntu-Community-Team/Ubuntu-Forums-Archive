### Post by turbojugend_gr on 2006-09-14
Check iff you have anything of the following running, in a gui-window or in console or in sys-tray or if you don't find them anywhere check for them in gnome-system-monitor. the apps you can't run at the same time are SYNAPTIC (or apt in a terminal) UPDATE MANAGER or add/remove programs (that's the last selection on the applications menu-my system is in greek so the name might differ) or Gdebi package installer. If you are trying to use anyone of them while using the other you get that message, because they all call the same "routines" to do their job.
