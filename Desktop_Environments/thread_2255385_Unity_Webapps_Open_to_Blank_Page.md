### Post by milliman on 2014-12-04
Maybe no one cares about this issue because my search through this forum turned up one closed thread on the topic. Unity Webapps have not worked in 14.10 since the release came out. No matter which app you select, it opens to a blank page. It disturbed me a bit that the release would come out with a significant feature broken, but I quickly moved on because I used Chrome to create my webapps. Still the anal retentive tendencies in me left me wanting a solution. I found one of a few bugs open on the topic in Launchpad. The problem is in the package unity-webapps-qml, and two days ago a release was fixed and submitted to [Vivid]("https://launchpad.net/ubuntu/+source/unity-webapps-qml/0.1+15.04.20141202-0ubuntu1"). I downloaded the package and installed it. Now my webapps work just fine. Eventually it may make it to pre-released updates but I couldn't wait. If anyone else is looking for a fix just download the package for Vivid from Launchpad and install it.
