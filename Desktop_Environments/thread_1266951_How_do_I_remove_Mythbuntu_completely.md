Just wondering how I can remove Mythbuntu completely and get back to my normal ubuntu updates? Mythbuntu stops at start up and I have to restart in recovery mode then download packages from there. I want the normal setup again.

---

### Post by Tux+ on 2009-09-15
```
$ sudo apt-get remove mythbuntu*
```
