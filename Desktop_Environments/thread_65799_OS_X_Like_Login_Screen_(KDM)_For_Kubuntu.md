### Post by jbus on 2005-09-15
Hi, guys I'm new here...

I've been using Kubuntu for a few months and I'm very pleased.

I have been trying to make it look as nice as possible, but I have been unhappy with the Kubuntu KDM login screen, so I made one for us Kubuntu users that is based on OS X's login. It's a great, classy alternative to the default Kubuntu KDM Theme.

I posted it over at [www.kde-look.org](www.kde-look.org). I hope that you guys enjoy it. Here is the [link.](http://www.kde-look.org/content/show.php?content=29072) 

I also have not been able to install KDM Theme Manager on my Kubuntu box due to dependency issues, so I posted instructions on how to manually install this KDM theme there as well. This method should work for other KDM themes as well.

Sorry about the poor screenshot, it's actually a picture of my monitor.
