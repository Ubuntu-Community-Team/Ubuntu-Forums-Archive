### Post by dusu on 2005-03-31
Hi,

if the goal is only syntax highlighting, I would recommend to simply use emacs with its associated tex-mode ! 
If you furthermore combine it with auctex, you can just do everything from emacs, there's nothing to be done in a shell.
This is no GUI, of course, but for such a markup language, I would anyway not recommend any GUI.
I've been using LaTeX for years now, and I really see no need for a GUI.
If in real need for a GUI, I've heard Lyx is a pretty nice tool.
