### Post by jnet3000 on 2007-03-29
With Feisty, Gnome Drawer Add to Drawer doesn't add the selected file.

Add Drawer in Panel:
* Right click panel
* Click Add to panel
* Click Drawer
* Click Add
* Click Close

Add file in Drawer:
* Click Drawer
* Right click in the drawer
* Click Add to drawer
* Click custom application launcher
* Select File for Type
* Enter Todo for Name
* Enter file path in Location
* Enter To do list for Comment
* Click OK

Result: It doesn't add the file.
Note: On Edgy, it works.
