### Post by tea for one on 2020-11-11
[https://github.com/thiggy01/gdm-background](https://github.com/thiggy01/gdm-background) requires you to have an image available to set as a background.

Have you created a solid colour image yet? (via gimp possibly)

Then run the utility similar to:-
```
sudo ./ubuntu-20.04-change-gdm-background /destination-to-image.png 

```

Secondly, there is a possibility that Vanilla Gnome itself is preventing you from using this utility?

The author of the software states > It was made specifically to work with Ubuntu 20.04 as it now bundles all configuration files inside a .gresource file.
