If I'm in a KDE4 session, how do I bring up KDE 3.5.x's kcontrol settings so that I can match kde3 apps UI with KDE4?

### Post by der_joachim on 2008-04-07
Go to desktop, press Alt+F2 and type 'kcontrol' at the prompt. Note that KDE3 and kcontrol have to be installed. ;)
