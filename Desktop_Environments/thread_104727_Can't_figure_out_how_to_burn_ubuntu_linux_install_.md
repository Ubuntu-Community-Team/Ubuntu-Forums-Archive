### Post by sjh on 2005-12-16
Go to this site:

     [http://www.md5summer.org/](http://www.md5summer.org/)

and download MD5Summer.  It is for windows.  Run it against the ISO and it will generate a code string.  Compare it against the MD5 sum posted on the download site.  This site:

     [http://ftp.lug.ro/ubuntu-releases/5.10/MD5SUMS](http://ftp.lug.ro/ubuntu-releases/5.10/MD5SUMS)

has the MD5's for 5.10

Hope this helps
SJH
