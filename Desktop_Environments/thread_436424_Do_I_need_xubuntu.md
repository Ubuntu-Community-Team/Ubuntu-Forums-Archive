### Post by aanderse on 2007-05-08
yeah sure i'll run through the steps to get my setup going.  i'll make mention that i always install debian based systems from a server install so i go from scratch ... i'll modify what i did so that you can do this to your existing ubuntu desktop which i will assume already has gnome loaded.

_[SIZE="4"]stage 1: installing the software[/SIZE]_
- log into you ubuntu machine and fire up synaptic package manager (or use apt-get/aptitude) to install the following packages: xfwm4, xfdesktop4, xfce4-terminal, thunar, mousepad (optional)
** note that synaptic/apt-get/aptitude should also install the following packages as dependencies (and maybe a few more): xfce4-utils, xfce4-mcsmanager, xfce4-panel.  if for some freak reason these are not installed make sure you install them as well


_[SIZE="4"]stage 2: switching to xfdesktop4, the xfce desktop[/SIZE]_
- once these are all installed you can open xfce4-terminal and type "xfdesktop4 &" (without the quotation marks around it).  this will replace you gnome desktop background with an xfce desktop background, and change what happens when you right/middle click on your desktop.
- now you need to change some gnome application menu entries so fire up alacarte (system->preferences->Main Menu or just type "alacarte").
- once in the menu editor go under "Prefences" and UNcheck "Desktop Background" so it is no longer visible in your gnome menu (or delete it).  gnome no longer manages the background image so there is no point to having this menu item as it can't do anything anymore.
- now we need to add an entry to the menu ... the xfce equivalent program for gnome's "Desktop Background" which xfce calls "Desktop Settings".  so under "preferences" still create a new item (click the button "+New Item") and enter in the following information... Type: Application, Name: Desktop Settings, Command: xfce-setting-show backdrop, Comment: Settings for the Xfce 4 Desktop Manager, Icon: i use tango icons so i selected /usr/share/icons/Tango/scalable/apps/background.svg
- now you can configure your desktop by going under the gnome menu to system->preferences->Desktop Settings

_[SIZE="4"]stage 3: switching to xfwm4, the xfce window manager[/SIZE]_
- in order to switch to xfwm4 instead of metacity we have to kill metacity and be without a window manager for a minute ... make sure you know your way around a computer without the mouse before doing this (keyboard shortcuts) in case anything goes wrong here and you need to re-enable metacity.
- open up a couple terminals and make sure that you close all other windows.  make sure the terminals are on the screen (not minimized) and not covered by anything (including each other)
- click on system-preferences->Sessions to open up the gnome session manager, then click on the "current session" tab.  click on "metacity" in the list of currently running programs and then click the "-Remove" button.  this should kill metacity and your window manager will be gone.  get to one of the open terminals and type "killall metacity" just to make sure it's really gone.
- now it is time to start the xfce window manager, so type "xfwm4 &" in a terminal and then you should have xfce's window manager running.  under the gnome session manager (which should still be open) go to the "session options" tab and click "save the current session" which should make it so next time you log in xfwm4 will run instead of metacity.
- go back to the gnome menu editor alacarte and under preferences UNcheck "Windows" because it is a gnome setting manager that is no longer applicable.
- we also need to add 2 programs to the gnome menu under preferences so click "+New Item" and add the following values Type: Application, Name: Window Manager Settings, Command: xfce-setting-show xfwm4, Comment: Window Manager Settings, Icon: /usr/share/icons/Tango/scalable/apps/gnome-window-manager.svg and then one more menu entry to add with Type: Application, Name: Window Manager Tweaks, Command: xfce-setting-show wmtweaks, Comment: Advanced Configuration, Icon: /usr/share/icons/hicolor/scalable/apps/wmtweaks.svg


and that covers the basic setup.  you should log out, then log back in and make sure it all works.  i tried to make this as clear as possible so anyone can try it out, good luck!
