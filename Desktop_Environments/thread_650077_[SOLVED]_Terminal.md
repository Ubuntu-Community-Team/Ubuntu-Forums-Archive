### Post by mali2297 on 2008-01-05
> **cartisdm said:**
> In the screen shot of the magazine it doesn't look like it's just a terminal because you can see all the processes running.

The program that shows the processes is called [htop]("http://htop.sourceforge.net/index.php"). 

To install htop, just type (or copy/paste) into the terminal:
```

sudo apt-get install htop

```
and to start the program:
```

htop

```
