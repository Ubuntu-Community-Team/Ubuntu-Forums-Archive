### Post by Zeroedout on 2005-06-17
what i did for using enlightenment is log into gnome normally, then in a shell type " killall metacity && enlightenment & " and metacity is killed and enlightenment replaces it, then hit system log out, and check the box marked save current setup. what the command doess is kill all the metacity processes and fires up englghtenment and the & makes it do this in the background. then when you save the current setup, it'll start enlightenment by default. so just repleace enlightenment with waimea and you got it. the other thing some window managers let you do is go --replace and that replaces the current one, and then do the save current setup. Good luck, hope this helped.
