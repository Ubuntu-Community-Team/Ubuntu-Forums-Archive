### Post by Jonne on 2007-06-25
I use beagle all the time and it seems to suit my needs perfectly. I definately prefer it over Spotlight and Google Desktop Search.

Spotlight is  *slow*, and if you press <return> after your query it opens the results window instead of opening the first relevant result.

Google desktop search displays in the browser and returns crap results. Plugins are a pain to install/configure.

On windows I use Copernic Desktop Search, which is bearable, but I still prefer Beagle

You can configure Beagle's advanced options by editing its xml conf file, btw, a gui will probably be written for it in the future.
