### Post by mtbsoft on 2008-12-05
Could be your Nautilus preferences.  If the partition has photos on it that could trigger the Media handling settings.

Open a File Browser window, select Edit | Preferences | Media tab...

Check to see if any of the settings, particularly Photos, is set to open The Gimp.  When you open the partition, it is similar to inserting and external drive or memory card or similar - it scans it and launches the appropriate application.

If you don't want this behaviour you can set the appropriate setting to Do Nothing.

Hope that helps.
