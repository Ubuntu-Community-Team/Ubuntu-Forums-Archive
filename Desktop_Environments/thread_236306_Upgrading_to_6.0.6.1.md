### Post by lamego on 2006-08-14
If you are connected to the network and doing the usual system updates you already have all the updates from the .1 .
If you don't have those updates and want to perform an update from the CD, go to:
System -> Administration -> Software Properties
and use the "Add CD-ROM" button.
This will make the updates available.
