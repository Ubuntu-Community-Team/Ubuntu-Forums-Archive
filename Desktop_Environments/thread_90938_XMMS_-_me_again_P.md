### Post by matthewv on 2005-11-16
What you need to do is add the extra repositories. There's plenty of info around on how to do it, but here's how I would do it.
Go to synaptic (System --> Administration --> Synaptic Package Manager)
Go to Settings --> Repositories
Press the add... button. Select Ubuntu Breezy 5.10 Binary from the drop down list.
Select all the check boxes.
Press OK, then OK again. 
Press Reload in synaptic. Wait...
Then you're all set. Go to add applications or use synaptic to install xmms.
