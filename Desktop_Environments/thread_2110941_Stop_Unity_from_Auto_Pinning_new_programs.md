### Post by Hishighness on 2013-01-31
Hey all, I'm running Ubuntu 12.10 and I'm just wondering if there's a way to stop Unity from auto pinning new programs. Like I'll install a program and run it and then close it. Then I'll look in the dock and see the icon and think "I thought I closed that" then I click on it again to bring it to focus and only then realize "Oh yeah, I did it's just pinned there."

Is there any way to turn this off?

Thanks,
H2

---

### Post by Krytarik on 2013-01-31
In Software Center, disable the option "View -> New Applications in Launcher".

Regards.
