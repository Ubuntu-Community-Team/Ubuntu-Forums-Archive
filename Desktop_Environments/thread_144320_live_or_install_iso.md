### Post by Sutekh on 2006-03-14
A LiveCD allows you to install a working system in your RAM so no changes are made to your hard disc.  Currently the LiveCD cannot install the system for real.  The Install CD is needed to actually install Ubuntu.

I think in future releases there will only be the need for one CD that can do both.  

(The current releases distribute DVD that contain the LiveCD and Install CD, but you can't install while using a LiveCD.)
