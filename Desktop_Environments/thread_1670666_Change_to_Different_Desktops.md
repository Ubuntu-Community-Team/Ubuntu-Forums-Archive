### Post by 3Miro on 2011-01-19
You can use either the Software Center or Synaptic Package Manager to install xubuntu-desktop (XFCE) or kubuntu-desktop (KDE). Then you can logout and from the login screen you should be able to chose between Gnome, XFCE and KDE.

Note that once an environment has been installed, it is very very hard to remove (although if you are not using it, it only takes space on the harddrive and adds programs to the menu).
