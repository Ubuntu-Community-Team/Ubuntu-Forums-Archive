### Post by ankspo71 on 2011-04-30
Hi,
See if you can right click on the desktop and create an application launcher like this: 

Name= Unity Reset
Command= unity --reset

When you are done you should see the launcher's icon appear on your desktop. Now you can double click on the icon to fix the Unity desktop.

PS. I notice this also removes the changes I make in compizconfig-settings-manager too. 

Hope this helps.

Another method that may work is to create a launcher for the command "ccsm" (CompizConfig Settings Manager), run it, and re-enable its "Ubuntu Unity Plugin", and other plugins that may have been disabled.
