### Post by hoodwink on 2006-06-17
How can I set the font size to my desired setting when gnome-term starts.

I've looked at everything I can think of under the profies, but no joy.

I've set the default profile to my desired color setting (black on light yellow), but each time I have to do View -> zoom in a couple of times to get the desired font size.

---

### Post by Ramses de Norre on 2006-06-17
Right click in terminal > edit current profile > uncheck "use the system terminal font" and specify your own font and size.
