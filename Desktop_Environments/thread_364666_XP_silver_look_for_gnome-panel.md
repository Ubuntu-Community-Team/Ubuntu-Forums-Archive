### Post by wxnker on 2007-02-18
It's not that I really miss Windows XP but still I wanted to make the gnome-panel look a little better and I don't really like the transparent option much.

So I thought I'd share this with anyone who would like a taskbar similar to the one in Windows XP in silver.

I have attached a gif-picture that I altered in Gimp. I use it as background on my panel to get the silver effect. I have also attached a few screenshots of what the result looks like. The gif height is 30 px as this is the height I use on my panel. If necessary this can be altered in Gimp.
[U]
Like in Windows XP I use the same panel for:[/U]
Windows XP like start button (added "Main Menu" to the panel).
Launchers/icons (the 3-4 I use the most).
A drawer (with an empty transparent icon) for extra launchers (the ones used regularly).
Windows list (open tasks).
Notification area. 

I don't really like to have a lot of panels if a single one can do it. I also chose "always group windows" in the Windows list preferences to group simular windows like in Windows XP. Since there's less room for open tasks this seemed to be the best option for me.

Perhaps there are other ways to do this, but this is how I did it. The themes I tried didn't really do the trick, so I did this instead and then chose a theme that works well with this particular panel-background.

It's all pretty simple so I'm not going to mention the details on how to do this (all you really need is to create a panel, add the features and use the gif-picture) but if anyone needs more info then just reply.

Regards Wxnker
