### Post by onero on 2009-05-07
> **medicalystoned said:**
> I am kinda still learning, can some tell me if you can install Flash Player 10, from terminal? If so, do you know how? Thankx..... Peace

Sure, you just need to install the adobe-flashplugin package, that has Flash 10. If you don't have it, just search for how to add the medibuntu repositories (although I think Flash should be in the main repositories, I'm not sure).

EDIT: Just to note, this can all be done via the GUI through Synaptic Package Manager (under Systm > Administration) as well, but in the terminal it's

sudo apt-get install adobe-flashplugin
