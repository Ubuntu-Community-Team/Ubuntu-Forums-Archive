### Post by yetimon_64 on 2016-09-19
Open your home folder, or any nautilus window, then go to the Edit menu > Preferences > Display tab and under the Icon Captions section reset the drop down menus to "none". It appears the upgrade may have set the options in that section to show "Type", it can easily be reset to "None" if you just want the folder name and no other details.

Edit: you may need to refresh the nautilus window with the F5 key to have the changes set in the currently open window. Or just open a new window and those details should disappear.

Cheers, yeti.
