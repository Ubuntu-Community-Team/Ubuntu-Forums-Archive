### Post by aysiu on 2005-12-18
Unfortunately, it's up to the package developer to include an icon with the package. If there's no icon, it doesn't appear. Even if there *is* an icon included, you may have to refresh the panel (*killall gnome-panel*) to get the icon to appear in the menus.

Other than that, you can add the icon manually by going to Applications > System Tools > Applications Menu Editor.
