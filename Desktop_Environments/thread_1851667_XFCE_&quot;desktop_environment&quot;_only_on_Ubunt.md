### Post by ewz on 2011-09-28
Hello what you can do I suppose is install Ubuntu and then install Xfce on top that way you will get your Gnome Apps eg, "Nautilus" but using the Xfce Panel, with a bit of moving around you can make the Xfce Panel look more like Gnome
To install Xubuntu Desktop, ina terminal type:
**sudo apt-get install xubuntu-desktop**

### Post by 3Miro on 2011-09-28
If you want to experiment with different DE, check here:

[http://www.psychocats.net/ubuntu/](http://www.psychocats.net/ubuntu/)

LXDE is lighter than XFCE and this is especially noticeable on low graphics machines.

If you want to stick Gnome 2 for at least 2-3 more years and especially on older hardware, I would recommend using Debian 6. Installing it and setting it up is a bit harder than Ubuntu, but in the end, you get Ubuntu that is a bit lighter on resources and with Gnome 2.
