### Post by paddy1978 on 2006-08-31
You can get some windows apps to run on linux. Two broad ways of doing it:

1) Wine ([here]("https://help.ubuntu.com/community/Wine")). There's also [Crossover Offic]("http://www.codeweavers.com/products/")e that I think is built on Wine. It allows you to install MS Office and various other software.

2) Virtualization - i.e. running a complete Windows install within a 'virtual machine'. Vmware (or the free 'player' version available - see [here]("https://help.ubuntu.com/community/VmwarePlayer")) or [Parallels]("http://www.parallels.com/") can this.

I use a mix of both. I've got Internet Explorer running under Wine ([http://www.tatanka.com.br/ies4linux/index-en.html]("http://www.tatanka.com.br/ies4linux/index-en.html")), but also am forced to maintain a full install of windows and MS office under Vmware Player for the sole purpose of syncing my pocket pc, as unfortunately ActiveSync doesn't seem to work with Wine!

Hope this helps :-)
