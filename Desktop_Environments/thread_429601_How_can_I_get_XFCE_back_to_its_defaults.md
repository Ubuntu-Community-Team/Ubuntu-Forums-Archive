### Post by gazj on 2007-05-01
Hopefully you can still open a file browser.

If so go to you home directory press CTRL+H this should show you your hidden files.  I am not so sure where the XFCE settings are, but probably in a hidden folder called .xfce or something similar,  don't delete this folder but rename it to something else like .xfceold.  Try logging out and logging in again.

With any luck xfce will create a new folder and restore its original settings.
