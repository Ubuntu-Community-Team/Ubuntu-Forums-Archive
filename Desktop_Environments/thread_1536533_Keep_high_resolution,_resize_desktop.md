### Post by WinRiddance on 2010-07-22
START ----> SYSTEM ----> PREFERENCES ----> Look for Display or Monitors.

If you have a laptop and the TV is your second display it might be a little tricky since Ubuntu always lists the laptop display first which is what you'll then get to see on the second screen as well ... with the limited options of the smaller laptop display. If you're using a laptop, this will explain how to set things up better:

[http://www.ubuntu.steinfein.org/ztip01_laptopdualdisplay.html](http://www.ubuntu.steinfein.org/ztip01_laptopdualdisplay.html)

If it's not a laptop and you've connected directly to the TV then the first line of this reply should work for you. ;)
