### Post by sharm on 2011-03-21
[http://live.gnome.org/Tomboy/Directories](http://live.gnome.org/Tomboy/Directories)

Notebooks are just special tags in the note XML.

The note file names are in the form [guid].note, and are not really meant to be human-readable.
