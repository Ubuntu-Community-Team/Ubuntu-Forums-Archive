### Post by dtmilano on 2007-10-10
I've been working on a tool named **autoglade** that could be of help to
simplify application development using glade as the interface designer.

You can download **autoglade** from [http://sourceforge.net/projects/autoglade]("http://sourceforge.net/projects/autoglade").
A basic tutorial can be found in the wiki: **[autoglade tutorial - first steps]("http://autoglade.wiki.sourceforge.net/autoglade+tutorial+-+first+steps")**.
As the tutorial demonstrates, [nautilus actions]("http://www.grumz.net/index.php?q=taxonomy/term/2/9") can be greatly enhanced using [b]autoglade[/a] generated GUIs. You are not limited to zenity anymore.
More to come.

Comments, suggestions, critics, bugs, fixes, etc are gladly welcome.
Please use **autoglade**'s mailing lists.
