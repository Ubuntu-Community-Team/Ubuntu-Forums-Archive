### Post by Sutekh on 2005-12-18
Do you want to add Nautilus to the Applications menu?

It should already be there, go to Applications -> Accessories -> File Browser

If it isn't, go to Applications -> System Tools -> Applications Menu Editor.

Add a New Entry and put "nautilus" in the command box.
