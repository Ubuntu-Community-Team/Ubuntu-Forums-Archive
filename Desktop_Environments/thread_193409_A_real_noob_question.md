### Post by bluevoodoo1 on 2006-06-10
Ubuntu uses the GNOME desktop environment. [http://www.gnome.org](http://www.gnome.org)

Kubuntu uses the K desktop environment. [http://www.kde.org](http://www.kde.org)

They each have their own look, applications and [loyal?] following of users. There are tons of threads here arguing which is better. You can use K apps in GNOME and vice versa, so use whatever you like.
