### Post by nkkromhof on 2007-01-30
> Well I finally found a solution some time ago. In Section "Device" I added the following line:

VideoRam 131072

which is the memory in kB of my graphics card. Then compiz and beryl worked fine (with aiglx). And I'm running the i810 driver I hope this helps you now

Found [here]("http://www.ubuntuforums.org/showthread.php?t=333746&highlight=i810+beryl")

Ofcourse if you've got 64Mb of RAM it should be...ummm... 65536? Better do some googling around if you have a different amount of RAM than 128Mb ;-)
