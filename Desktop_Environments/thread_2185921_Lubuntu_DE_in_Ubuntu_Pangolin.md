### Post by sudodus on 2013-11-05
I think the safest way to try Lubuntu is to ***try it without installing***. You can do that ***from a boot CD/DVD/USB driv***e made from a Lubuntu desktop iso file.

Lubuntu 12.04 Precise Pangolin had no long time support, so I would not recommend trying to install that desktop environment into Ubuntu 12.04 Precise Pangolin, which has long time support, LTS. (Lubuntu 12.04 passed end of life last month.) ***Try the current version instead, Lubuntu 13.10!***
