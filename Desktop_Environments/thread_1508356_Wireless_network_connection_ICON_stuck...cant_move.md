### Post by rtimai on 2010-06-12
The icon you show represents the Network Manager which actually appears within the Notification Area applet. Next to the "signal" icon, you will see six little stacked dots which represents the Notification Area. If you can't see it, right-click on any blank part of the toolbar, select Properties, Background tab, select Solid Color, white, opaque. Then the Notification Area will be visible as a textured slivver. Right click on it to Unlock and Move.
