### Post by foximaxi on 2006-08-12
Hello,

there actually is a "recycle bin" in Ubuntu, it is just not easily accessible. Open your home directory in Natulius, click "View" -> "Show Hidden Files". After that, you'll see a ".trash" directory. This is where deleted files and directories are stored. Now you can restore the accidentaly deleted dir.

Have a nice day. ;)

### Post by oghanchi on 2006-08-12
If you used Nautilus to delete them, then the files are in the trash:/ folder. However, my understanding is that if you used Thunar, which is the default file manager for Xubuntu, the files are deleted and gone. In this case you will have to use programs like executive undelete (commercial software) or some other application to see if you can recover them. I am sure there are open source applications that you can google for details.
