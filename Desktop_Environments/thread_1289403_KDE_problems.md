### Post by Lars Noodén on 2009-10-13
Go into the package manager and select the group(s) starting with the name 'GNOME Desktop', then uncheck specific packages.  There might be an occasional package to keep, if you have some specific Gnome tools around, so you'll have to do a bit of trial and error.

A big space saver would be to [remove mono](http://www.theopensourcerer.com/2009/04/23/how-to-remove-mono-from-ubuntu-904-jaunty-jackalope/) and all its dependencies.
