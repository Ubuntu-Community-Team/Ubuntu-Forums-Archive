### Post by Mr Frosti on 2006-09-26
To setup your network connections, you can use the Gnome network monitor. This will be under "System". You can setup the criteria for your wireless connection in this location.

I would personally recommend using NetworkManager, as it is much easier to work with. To install this, search for this using Synaptic and install it. If it is not already running, type in "nm-applet" into a terminal to begin the application.
