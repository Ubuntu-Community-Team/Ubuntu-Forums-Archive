### Post by o_fortuna on 2006-02-04
[QUOTE=XIII]How can i make application auto-start with system turning on, instead of typing /opt/lampp/lampp start 
every time i turn on my system i want it to start automatically.[/QUOTE]
You can set it up in the Gnome-Sessions menu. Go System-->Preferences-->Sessions, then click the startup programs tab. Just click add, and enter /opt/lampp/lampp start in the line. You should leave the order at 50.
