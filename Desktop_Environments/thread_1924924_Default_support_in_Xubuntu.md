### Post by LewisTM on 2012-02-13
In Thunar, go to a PDF file, right-click and choose Properties.
The first tab should have a drop-down list of applications to use.
Choose something like Document Viewer, which is evince.

This selection will be used by all applications that are compliant with Freedesktop/XDG standards.

Cheers!
