### Post by jerenept on 2010-05-28
You have desktop cube transparency enabled. Use CompizConfig Settings Manager to set it back to how it was.
when you have ccsm open, select the button 'desktop cube' - not the check box-
select tab 'transparent cube' and set all sliders to 100%
