### Post by Zorael on 2008-08-30
Gnome programs, which use the GTK window framework, need to load those libraries into memory to be able to display properly, so if you're on a *very* tight memory budget you may want to consider keeping to one window framework, but then you'd be using a lighter environment than Gnome/KDE (which are fairly equal in that department). KDE is based on the Qt framework.

You needn't worry. KDE themes all GTK programs properly; you'll only notice some disrepancies in the UI. 

To be nit-picky, KDE programs' looks are generally very consistent, which your Gnome programs will differ from. Again, a minor detail. File dialogs (open/save/etc) will also look different and have different bookmarks.
