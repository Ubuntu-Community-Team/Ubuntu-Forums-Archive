### Post by barnabas on 2006-08-17
I had the same problem when I installed XGL as well. Mine was fixed by changing the keyboard layout: System>>Preferences>>Keyboard under the layout tab instead of "Generic 101-key" select "Generic 104-key" or 105. This got the zoom function and the F12 functionality working. 

I am still having problems creating a keyboard shortcut to my terminal, though. For some reason all of my shortcuts work except the terminal. 

Hope this helps, and if y'all figure out the terminal problem let me know. :)
