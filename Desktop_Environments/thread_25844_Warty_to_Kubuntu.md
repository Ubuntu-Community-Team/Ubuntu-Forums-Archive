### Post by weblars on 2005-04-11
Kubuntu is a regular Hoary install including the KDE packages. You can just follow the instructions for an upgrade from Warty to Hoary (change warty to hoary in sources.list, sudo apt-get update, sudo apt-get dist-upgrade...). Then install the package kubuntu-desktop via apt-get or synaptic.
Now you can choose between Gnome and KDE at the login screen.
