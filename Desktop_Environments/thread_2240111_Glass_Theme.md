### Post by CantankRus on 2014-08-18
No specific theme needed.
You can set the opacity saturation and brightness of inactive windows using compizconfig-settings-manager (ccsm).
Install ccsm and the unsupported plugins.
Terminal...
```
sudo apt-get install compizconfig-settings-manager compiz-plugins
```

Log out and back in.

Search and run ccsm in the dash.
Under accessibility enable the **Dim Inactive** plugin and configure options.
