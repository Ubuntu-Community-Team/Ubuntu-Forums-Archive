### Post by aysiu on 2006-06-12
Press Control-L.

By the way, the location bar in Firefox is also an "I'm feeling lucky" search in Google.

For example, instead of typing ```
www.ubuntuforums.org
``` you can just type ```
ubuntu forums
``` in the address bar.
