### Post by amunimanghi on 2006-06-27
Yes you can. Do this in the terminal. ```
sudo apt-get install gaim-guifications
```. Then open gaim, go to the plugins section. Check the box. It will appear under the plugins tab. Then choose your options for it. Hope this helps.
