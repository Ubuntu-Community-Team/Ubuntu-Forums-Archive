### Post by ptonino on 2011-06-01
Hi gnagnibu,

Your notes must be stored in the file

~/kde.bak/share/config/plasma-desktop-appletsrc

So or you can copy part of the code that is present in there in your current ~/kde/.../plasma-desktop-appletsrc file, or you can manually copy them to new sticky notes.

Hope you get them back :)
