As far as I know, googleanalytics is a tracker to help websites measure statistics; it doesn't add any content to the user, and if your page loads and displays (except for googleanalytics) it should be fine.

Using the NoScript plug-in, you can choose which site is allowed to run scripts (java, flash, javascript), and which don't.  For sites with multiple urls, you can selectively allow or forbid scripts for each url.  It can remember your choices  temporarily (this session only) or permanently (every time you run firefox).

You can also globally enable all (for your wife).:)

But yeah, it take some work to set-up initially, what to allow and what to block.
