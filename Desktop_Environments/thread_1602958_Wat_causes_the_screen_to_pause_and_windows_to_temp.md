### Post by sikander3786 on 2010-10-22
Most probably a hardware fault if you seemed to have it in almost all the distros.

First of all, run memtest from Grub menu and see if all the RAM modules are just fine.

Secondly, check your HDD as it might be failing and causing the problems. Schedule a scan overnight as it is going to take some time.
