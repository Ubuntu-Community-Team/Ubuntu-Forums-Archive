### Post by Alexander Kirillov on 2005-04-28
[QUOTE=23meg]ok, sorry, i should have been clearer..

anyone?[/QUOTE]
 I'm afraid the answer is "no". That is, you can add a note if a file opens inside Nautilus (using Nautilus "viewer component") -- IIRC, by default this is what happens with image files and plain text files. I can be wrong, though: I have long changed default apps for opening most file types. 

But if you configured files to be opened by some app other than Nautilus, then I do not think there is a way to add notes other than going to Properties. The only way of speeding it up is to use Alt+Return keyboard shortcut for Properties.   

FIle a bug report with priority "wishlist" - maybe the developers will implement something better. However, given the number of open bugs in  Nautilus, this does not seem likely...
