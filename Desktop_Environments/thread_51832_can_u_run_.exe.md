### Post by grim42 on 2005-07-25
I assume that by .exe files you mean Windows executables. The short answer is no. Programs compiled for Windows can't run on a Linux system.

BUT, there is a "Windows compatibility layer" called WINE that will allow you to run Windows programs, but depending on the program you may not have any success.

See [http://www.winehq.com/](http://www.winehq.com/).

To install *sudo apt-get install wine*, I think.
