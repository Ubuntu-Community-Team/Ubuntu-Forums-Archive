### Post by pastalavista on 2009-05-19
If you have A USB flash drive, 1GB or larger, you can install the iso file to it with [unetbootin]("http://unetbootin.sourceforge.net"). Then you'd need to make the USB drive boot before the hardrive in the BIOS to load your distro. You can try it before you install and it's really a lot faster to work with than a live CD.
