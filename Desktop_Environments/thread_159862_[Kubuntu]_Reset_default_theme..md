### Post by Neo Ex on 2006-04-13
The default Kubuntu theme is Lipstik. Anyway, you can restore all settings to the Kubuntu default ones by deleting your ~/.kde folder (maybe you can make a backup of it before deleting!). Then log out and re-log in again.
