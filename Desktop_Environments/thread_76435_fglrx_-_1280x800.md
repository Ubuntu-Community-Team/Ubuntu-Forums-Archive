### Post by pecos7 on 2005-10-15
It's a known bug of the 8.16.20 version of ati drivers: they don't support 1280X800. You need the next version that was released two days ago (8.18.6).
See [HOW-TO: ATI fglrx driver 8.18.6]("http://www.ubuntuforums.org/showthread.php?t=76116")

### Post by zi99y on 2006-04-15
I got this working by commenting out the modeline in xorg.conf like so:

```
Section "Monitor"
	Identifier	"Generic Monitor"
	Option		"DPMS"
#	Modeline	"1280x800@60" 83.91 1280 1312 1624 1656 800 816 824 841
EndSection
```

No need to change anything else, just put the # at the start of the line, and restart Xorg, voila!
