### Post by mikewhatever on 2017-07-15
You can select a theme in the Look and Feel window, and click the Customize button to select different icons. The orange ones should be there as well.

The highlight color was a tricky one, but I've found that gtk-theme-config from the repositories is an easy workaround. It pulls in some xfce dependencies, but nothing major really.

### Post by oldfred on 2017-07-16
Still another option is fallback or gnome-panel which I have used since they converted to Unity.
It gives the same look as the old versions of Ubuntu. And has updated to gnome3, the similar flashback was based on gnome2.
There are some changes, but every version of software has some changes & you learn to live with them.

       [https://wiki.archlinux.org/index.php/GNOME/Flashback](https://wiki.archlinux.org/index.php/GNOME/Flashback)
The differences to the MATE project is that GNOME Flashback uses GTK+ 3 and tries to follow the current GNOME development by integrating recent changes of the GNOME libraries.
