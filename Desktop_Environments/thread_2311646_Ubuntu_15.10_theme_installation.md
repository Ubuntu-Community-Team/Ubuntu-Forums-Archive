### Post by Frogs Hair on 2016-01-31
Third party themes will only appear in the unity tweak or the gnome tweak tool and also make sure you are using compatible themes . Themes for gnome 3.14 or 3.16 should work in 15.10. Be sure the themes are fully extracted, some theme packages include more than one color variation and if not removed from the master folder into .themes won't appear the the tweak tool. GTK 2 or old GTK 3 themes will not work in 15.10.
