### Post by amd-64 on 2008-02-18
Try booting from a live CD. The best and surest way to know. 

Booting from external HDD may or may not work. Not sure if the bios would accept booting from USB. The bios is recent enough, so my guess is that it should work. 

In any case, I tried this one time and found that when I used a very short usb cable, 6 inches, the bios recognized the external boot drive but when I used a standard 3 ft cable it did not. 

By the way, do you really need a 2Gb Virtual Memory. I suggest changing this to whatever is recommended by Windows XP.
