I've googled and searched on and off for weeks, and can't find a way to set the defaults in nautilus 3.8.1. Specifically, I want nautilus to list files 'by type' (directories first, files later) for every directory.  I can set them in the GUI, but they only apply to the current directory.

---

### Post by jbicha on 2013-06-08
Click **Files** in the top left corner of your screen and select **Preferences** in the menu that opens. You can set the defaults on that first tab.
