### Post by Peripheral Visionary on 2013-07-11
Wallpapers are stored in **usr ->share -> xfce4 -> backdrops**.  You can move favorite downloaded wallpaper images there (you need root access), and then use the regular **right-click -> Desktop Settings** procedure to change your wallpaper.  Some file managers insist on managing your desktop for you, lol.
