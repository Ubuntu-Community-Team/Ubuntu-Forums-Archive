Hi,

I'm a big fan of Gnome's alt/middle-click to resize a window.  However, it doesn't work anymore for me, and I believe it stopped working (or got toggled off) when I installed the Unity desktop package (just wanted to check it out).

### Post by sdhoigt on 2010-11-04
Ok, I found it.

Since I'm running Compiz, I loaded the compizconfig-settings-manager package, ran 'ccsm' from the command line (to load the app), and clicked on Resize Window under Window Management.

'Initiate Window Resize' was the option I needed to (re)set.

SD
