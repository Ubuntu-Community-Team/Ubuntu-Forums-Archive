===

SOLUTION:

On my Toshiba Regza 37Z3030DB, the exact scan option is hidden inside the WIDE / SUPER WIDE / CINEMA etc options menu and is only available when HDMI input is detected. Took me soooo long to work this out. Proper blonde moment I guess, but I honestly expected it to be in the PICTURE SETTINGS part of the menu system. Oh well, so pleased to have found a solution now! :)
