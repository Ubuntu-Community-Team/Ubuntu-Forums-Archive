### Post by shinkaide on 2010-10-21
Since both Gnome-Do and Docky are now separate programs, you must launch *and* use each program separately. It sounds like you're trying to configure Docky from within the Gnome-Do config panel. Gnome-Do has its own configuration panel, as does Docky; they have nothing to do with each other now.

Pet peeve of mine: I like being able to launch programs using my Keyboard, but I want to be able to do it from Docky and not have the Gnome-Do thing appear in the middle of my Screen. Oh well.
