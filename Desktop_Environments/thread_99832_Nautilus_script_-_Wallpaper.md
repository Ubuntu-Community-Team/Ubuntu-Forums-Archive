### Post by DMon on 2005-12-06
One of the feature I always missed in Nautils is to be able to set the wallpaper through the Nautilus menu. Recently I discovered nautilus scripts which means u can make your own scripts to do various tasks. Dunno if such script already exists on the forums, anyway I post my and maybe someone could make use of it. It is based on some script I found written in perl gtk which I ported to python gtk.

To install just put it in ~/.gnome2/nautilus-scripts.
