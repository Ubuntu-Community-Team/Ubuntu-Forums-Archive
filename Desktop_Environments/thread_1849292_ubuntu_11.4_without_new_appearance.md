### Post by howefield on 2011-09-24
Do you mean it has the gnome panels instead of the Unity interface ?

Try looking in System > Administration > Additional Drivers and check for proprietary drivers which may enable Unity to work.

If there are proprietary drivers available install them and reboot, selecting Ubuntu from the session menu at the login screen.

### Post by ajgreeny on 2011-09-24
However, there are many machines that simply can not run the standard unity and will then default back to the classic gnome interface.  If this is your uncle's problem install **unity-2d** fro the software-center or with synaptic, and then choose unity-2d at the login screen session menu
