### Post by NeoLithium on 2007-06-22
You can go to System -> Preferences -> Menus and uncheck some of the applications that you don't use, that would normally shrink the menu quite a bit if it's become clogged with apps that you don't normally use. Or simply create a new menu and split them up so they're not as large, should you not want to hide any programs from the menu.
