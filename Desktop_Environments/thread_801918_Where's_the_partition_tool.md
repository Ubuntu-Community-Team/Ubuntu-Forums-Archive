### Post by BobCFC on 2008-05-21
The app you want is called gparted


Either type  

sudo apt-get install gparted 



or look in add/remove programs



EDIT:  you have to unmount a drive to change it, so you can't change your system drive while it is running.  If you need to do that you can do it from the live CD
