### Post by qpieus on 2007-06-12
I found this on the opensuse konqueror tips and trick page:

"Popup Blocking 	

Konqueror incorporates popup blocking, although it does not make as much noise about is as Firefox. In fact, it is enabled by default on SUSE Linux. To change or view the settings, go to Settings > Configure Konqueror > Java & JavaScript > JavaScript (tab). Under Global JavaScript Properties set Open New Windows to configure the popup blocking setting."

My konqueror is set to "Smart" for this setting. I see that one of the choices is "Allow". Maybe that's the setting you want.

[http://en.opensuse.org/Konqueror_Tips_and_Tricks](http://en.opensuse.org/Konqueror_Tips_and_Tricks)
