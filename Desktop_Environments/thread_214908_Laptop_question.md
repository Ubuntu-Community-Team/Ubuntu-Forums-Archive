### Post by taurus on 2006-07-13
If you want to use Gnome, you probably need at least 256MB of RAM to get it run reasonable well.  Anything less 256MB of RAM, you will feel sorry you ever run Gnome (but of course, the more RAM you have, the faster Gnome will be).  So, I would say that anything from 256MB of RAM or less, try Xfce4 or fluxbox...  ;)

### Post by snek on 2006-07-13
I **couldn't** install Ubuntu 6.06 on a CPx 500Mhz with 128MB ram and NO CD-ROM & not able to boot from USB or NET. Mostly because the wiki had not been updated on how to do this for 6.06.. So I followed the wiki for 5.10 (of which I have a few original cd's) and managed to install it just fine. The NIC which came with the laptop was detected automatically and I proceeded to update, and eventually updated to 6.06..

The speed of Ubuntu itself was pretty :-k on the 500Mhz so I decided to install Xubuntu-desktop instead and uninstall Ubuntu-desktop. (I followed a guide to uninstall ubuntu-desktop, but this uninstalled waaaaay too much). This worked just fine and I have the laptop running with XFCE and even a Broadcom Wifi card (Asus WL-100g Deluxe). The speed is acceptable, it is definitely faster than Windows 98/ME/2000!

All the hardware was found automatically btw, haven't configged anything except the wifi card.
