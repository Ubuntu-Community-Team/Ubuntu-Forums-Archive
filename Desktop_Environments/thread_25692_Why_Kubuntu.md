### Post by judyz on 2005-04-12
[QUOTE=void_false]Can somebody tell me how to remove Gnome? TIA[/QUOTE]

Synaptic will allow you to remove things as well as install them.  If you really want to get rid of gnome this is probably the safest way.

Personally I much prefer KDE.  Not suprising since I came from the Windows world and KDE is designed to act as much as possible like Windows where Gnome is more Mac'ish.

I would recommend installing Ubuntu hoary and then "upgrading" to Kubuntu rather than using the Kubuntu CD's based on what I have read here.  Since you are not losing out on anything and it seems to be more stable.  I have not had any of the crashes discussed in these forums on my system and this is how it was done.  If you wanted to free up space you could always remove the Gnome components later with Synaptic.

In my humble opinion what Ubuntu/Kbuntu need to do now is merge the two products together.  Put the basics for both on one disc and ask the user whether they want Gnome, KDE or both during the install.  Maybe in the process they could ditch the Debian installer and create a more appealing and easy to use installer similiar to Xandros?
