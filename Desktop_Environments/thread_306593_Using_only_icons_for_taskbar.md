### Post by CatKiller on 2006-11-25
You can use the Window Selector panel applet instead - that just gives you one icon of the uppermost application, and then you can click on it to get a list of the others.

Alternatively, with the Window List panel applet, if you right-click on it (its handle, at least) and select Preferences, you have the option of "Always group windows".
