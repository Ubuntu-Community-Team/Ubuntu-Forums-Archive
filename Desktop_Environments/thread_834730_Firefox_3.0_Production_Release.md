### Post by MeanStreak on 2008-07-07
Successfully upgraded to Firefox 3.0 Final. 

To ensure apps like Firefox are automatically upgraded, 
[LIST=1]
[*]Go to System > Administration > Software Sources. 
[*]On the Updates tab, ensure the following are checked:
[LIST]
[*]Important security updates
[*]Recommended updates
[*]Unsupported updates.
[/LIST]
[*]Under Automatic updates, select an option that suits. 
[*]Click Close.
[/LIST]
