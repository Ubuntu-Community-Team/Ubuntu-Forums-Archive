### Post by catlett on 2006-08-20
Put the cd in windows and browse the contents. there should be a folder "isolinux" with a file "isol;inux.bin" inside.
Did you do a md5sum check to make sure the download wasn't corrupted?
To get a md5sum in windows and then check it against the iso's original md5sum, folow this guide [http://www.psychocats.net/ubuntu/iso.html](http://www.psychocats.net/ubuntu/iso.html) It has a section on md5sum checks. This ll assumes you have a wroking install of windows.
