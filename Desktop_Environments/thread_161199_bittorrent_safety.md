### Post by dr.drake on 2006-04-17
azureus has a plugin called safePeer, you can get it from [their site]("http://azureus.sourceforge.net/plugin_details.php?plugin=safepeer").> 
**Author's Description **	
This plugin allows you to use SafePeer to import a list of IPs to block to the IpFilter from some database (by default the one of PeerGuardian) Forums & Bug Report.
See history.txt for a more detailed list of changes.
but with this plugin azureus will take aaaaaaaaaaages to load and be an even more rescource  hog than it already is.
a friend did a test once and told me it blocked several million (!!!) IP addresses, which came down to 60% of the internet...
but if you don't download anything illegal, you shouldn't have to worry about "being safe" anyway.
