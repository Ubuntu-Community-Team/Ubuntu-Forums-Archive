### Post by rapha on 2006-09-09
Hi,

if you just want to run XP and nothing else, VMWare Player (just look for vmware in Synaptic) could be for you. It's unable to create any virtual machines but with a bit of fiddling around you can get XP installed. Take a look at this website, has good explanations: [http://www.ffnn.nl/pages/articles/linux/vmware-player-image-creation.php](http://www.ffnn.nl/pages/articles/linux/vmware-player-image-creation.php)

Good luck!
