I have noticed that many times Numeric lock is not enabled in Ubuntu 9.10.

Usually, you can turn on numlock by setting a BIOS option. If for some reason, it does not stay on when Ubuntu starts, consider using numlockx by installing it with ```
sudo apt-get install numlockx
```
