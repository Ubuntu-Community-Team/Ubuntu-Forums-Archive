### Post by d3v1ant_0n3 on 2006-12-30
Assuming you've followed correct instructions for installing Beryl onto your system (the correct method for different graphics cards varies massively), you can start Beryl with ```
beryl-manager
```
 in the terminal.

That should start beryl, it's notification area icon (which contains links to BSM and Emerald theme manager) and emerald itself.

You might want to check that emerald is set as the window decorator (right click on the notification icon, select 'Select Window Decorator' and check Emerald).
