### Post by apjjr on 2010-11-24
Nobody has a clue?  I thought surely I could not be alone with this problem.
Try right clicking a file in Nautilus, then select send to, then set Send AS to Email and click ok.  That should open your preferred email program compose dialog with the file as an attachment.  This will work with System->Preferences->Preferred Applications set to Thunderbird, but not when it is set to custom, which it gets set to every time I update the system.
Give it a try and let me know what happens on your system.
