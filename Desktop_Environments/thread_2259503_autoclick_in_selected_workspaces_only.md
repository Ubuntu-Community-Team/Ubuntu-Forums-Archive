### Post by deadflowr on 2015-01-04
What's autoclick?

But yes you can set certain applications to open and/or run in specific workspaces.
Install ccsm (compizconfig-settings-manager) and then open it,
Then go to the Window Management section.
Then open the Place Windows section.
Then go to Fixed Window Placement tab.
Then go to the Section for Windows with fixed viewports and click the plus(+) button to add the new application.
Then add it to which ever workspace you want.

This works on Ubuntu, which uses compiz.
But most likely will not work for any other desktop, such as gnome-shell, lxde, xfce, or kde; among others.
