### Post by Fred the Penguin on 2010-04-02
You can install a better version of AWN than is in the repositories by adding 
             [http://ppa.launchpad.net/awn-testing/ppa/ubuntu](http://ppa.launchpad.net/awn-testing/ppa/ubuntu)
to your sources list under system-administration-software sources under other sources tab. 

Install the packages **avant-window-navigator-trunk, awn-settings-trunk** and for extras:[B]awn-applets-c-core-trunk, awn-applets-c-extras-trunk,  awn-applets-python-core-trunk, awn-applets-python-extras-trunk
  [/B]You can start AWN under applications-accesories.
