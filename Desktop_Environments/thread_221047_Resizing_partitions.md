### Post by reacocard on 2006-07-22
Unfortunatly, it is impossible to expand a partition backwards. you should create a new partition between your windows and ubuntu partitions, install ubuntu to that, and then use the livecd to copy the contents of your old /home folder into the new install. this will keep all your settings, so all you have to do is reinstall any software you added. after you have moved /home, you can delete the old ubuntu partition.
