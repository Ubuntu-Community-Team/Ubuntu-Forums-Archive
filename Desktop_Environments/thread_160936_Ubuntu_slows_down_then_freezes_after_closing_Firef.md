### Post by bscbrit on 2006-04-16
The first thing to try is a little patience :D Give it 15 minutes and see if it recovers. Mine does after 6 minutes - which is a not a long time (unacceptable yes, but not long) but gives me enough time to make a cup of tea.  Secondly, have the System Monitor open (Applications -> System Tools -> System Monitor) on the display that shows the various processes operating. You will then see which process is causing the problem.

This might be another bug in Firefox - but make sure you have carried out the numerous fixes in other threads on this forum before reporting it. Hope this Helps (HTH)
