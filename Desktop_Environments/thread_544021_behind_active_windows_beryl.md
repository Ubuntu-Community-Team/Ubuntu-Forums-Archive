### Post by gomez_in_the_south on 2007-09-06
I had everything working fine in Beryl. Then I played around with some settings and got the same symptoms that you described.

I found a setting in the Beryl Settings Manager -> General Options -> Main Tab called "Level of focus stealing prevention". It was set to Extreme. Setting that to anything other than Extreme  fixed the problem for me. Funny thing is I'm sure I didn't change it before.
