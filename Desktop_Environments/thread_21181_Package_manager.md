### Post by lorap on 2005-03-21
hi friend!
all you've to do is just:

i.open Synaptic Package Manager;

ii.go to the Settings-->Repositories;

iii.mark all options possible ignoring warnings;

iv.press Ok button;

v.after what this window closes and you press Reload button in main window.

vi.now press the Search button and in the popup window write in the program   you want to install,then press Ok.

vii.after the program's been succesfully found out mark in for install and press Apply button.

that's all!
