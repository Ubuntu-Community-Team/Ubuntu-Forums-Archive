### Post by andrea000 on 2009-09-11
Click appearance>icons go to the bottom were it
says icon size,separators,and icon order from there
you can set the size keep in mind that the dock's
size depends on the icons size,separators,and 
icon order's size.I have used cairo dock for 2 years
and have not found a way to just set the dock size
it depends of the size of the stuff that you put on
the dock.
Hope this helps.
