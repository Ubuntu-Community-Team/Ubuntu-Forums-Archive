### Post by jcs296 on 2005-06-27
Did you install dvd playback as outlined at ubuntuguide.org?  If not, see [http://ubuntuguide.org/#dvdplayback](http://ubuntuguide.org/#dvdplayback).  You might also want to check out [http://ubuntuguide.org/#codecs](http://ubuntuguide.org/#codecs) for how to install codecs for some of the other media formats you listed; also the instructions for xine, realplayer, etc on the same site.  If you haven't heard of ubuntuguide.org yet, it's a thoroughly tested and fantastic resource for ubuntu howtos and tips.
