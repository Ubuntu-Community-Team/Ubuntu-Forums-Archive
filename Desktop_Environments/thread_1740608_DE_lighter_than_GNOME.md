### Post by Copper Bezel on 2011-04-27
LXDE uses Openbox; Openbox is a window manager, not a complete desktop environment, and can be used with LXDE, Gnome, etc.

But yes, they're worth trying out, and there's really no risk, so why not? It really is as simple as installing the packages, logging out, selecting the DE you want, and logging back in. Just keep track of what you install with the full DEs so that you can remove anything you don't end up wanting to keep. Openbox with LXDE particularly is *insanely* fast.

### Post by 3Miro on 2011-04-28
You can use OpenBox + Gnome, just install Fusion icon and you can switch the WM. IT will be about the same as Gnome without effects (i.e. with Metacity).

Standalone OpenBox would be the fastest followed by LXDE, but those are missing some features. You can try XFCE, it is considerably lighter than Gnome and it is hardly missing anything.
