### Post by leech on 2005-03-25
There is one (minor to the GUI, major to the actual program) difference between NeroLinux and GnomeToaster.....  the Nero API.  GnomeToaster just uses 'cdrdao' but NeroLinux uses the actual NeroAPI that the Windows version of Nero uses.  So it can burn DVD's, Nrg, and many other file formats that cdrdao/GnomeToaster cannont.

The only coasters I've ever made using the Memorex ones are in linux....  So I'm pretty sure it's not just the disks.  I could try burning some things to my Sony DVD+RW's to see if they have the same issue.....
