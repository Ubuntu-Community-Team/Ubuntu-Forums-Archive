Yes, it's the same thing. The trick is to try not to use gnome or kde applications. Don't use evolution, don't use konqueror, kate, or anything with a K in it. try to use only Gtk applications (i don't have a bad machine, and i do this because i like a responsive OS). If you use Gnome's or Kde's applications, you'll end up loading gnome and kde's libraries and that is a little heavy
