### Post by rykel on 2006-08-12
Hi friend,

After years of using Ubuntu since Warty, I still find that the only way to let other users of my system install programs without me giving away my root password, is to direct them to install autopackages. ([http://www.autopackage.org](http://www.autopackage.org))

Somehow, I just cannot seem to find an easy way to install Debs etc. without root. Furthermore, it is a conflict of principle within me when I advocate "safety first" (ie. Do NOT run as root), yet when it comes to the most elementary thing to do before being productive on a computer, that is, installing software that you need to use, I have to go against the Linux Golden Rule and instruct my users to install as root. (via Synaptic or gDebi)

Fortunately, autopackages of some of my daily and favourite Linux programs do exist, such as Firefox, Inkscape, Gaim, Xara Extreme, AbiWord and SuperTux, but the list is still small.

I hope that if you are a developer reading this, you can make it a point to release your programs primarily as autopackages (like what the Xara and AbiWord fellows have done), or if not, then please, educate me on how I can install software in Ubuntu with GUI but without root password.

Thanks for reading, just a word off my chest for today.

**Ubuntu ROCKS!**
