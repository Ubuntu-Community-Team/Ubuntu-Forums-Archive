### Post by Kevbert on 2008-06-28
You need 4 workspaces. Right click on the boxes in the bottom RH corner, select preferences and set to 4 columns.
Next, go to System - Preferences - Advanced Desktop Effects - Double click on Rotate Cube, check its enabled and set Flip Time to 200, Speed to 1.7, Zoom to 0.8.
Now go to Desktop Cube, Behaviour, check its enabled, set Acceleration to 4, Speed to 1.7 and Timestep to 1.2. If you want a background to your cube select the Appearance tab, select Skydome and select an image to display.
You should now have a cube when you select a new Workspace when clicking the workspace in bottom RH corner or using Ctrl-Alt-Left or Right arrow.
Next select Cube Reflection and set the intensity to something around 1.
