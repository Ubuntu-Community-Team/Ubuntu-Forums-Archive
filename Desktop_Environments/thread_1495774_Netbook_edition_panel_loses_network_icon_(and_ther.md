### Post by misGnomer on 2010-05-28
Scenario: User #1 (admin) powers up netbook, logs in and out of wired and wireless networks using the network icon (applet) in the awfully locked-down netbook top panel.

System changes hands and User #2 logs in, but the said network icon has disappeared from the panel! Despite having user rights to operate wired and wireless networking, User #2 is unable to connect (or disconnect) networks since the panel applet icon for networks has gone missing.

Same thing happens if User #2 logs in first and gets the network icon goodness; after switching to User #1 the network (control) icon has disappeared.

After switching the initial user back, the network icon returns to the top panel.


Off topic: Considering that Folder Sharing also failed under this Netbook Edition Lucid (due to its wizard failing to install SMB properly), it appears that at this point the netbook version hasn't seen much multi-user love...
