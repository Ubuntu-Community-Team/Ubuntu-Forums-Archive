### Post by zombifier25 on 2012-12-18
If that's all you want, then the default Radiance theme will help. Right click the desktop, choose "Change Desktop Background", change Theme to Radiance.

If you want to install other custom themes (e.g. themes downloaded from gnome-look.org) then you need to install an external tool like UnSettings, Ubuntu Tweak, ..., download the theme to ~/.themes or /usr/share/themes, then use said tools to switch.
