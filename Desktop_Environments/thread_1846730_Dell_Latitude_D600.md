### Post by Karlchen on 2011-09-19
Hello, mysticalone.
> Details: 1.3 GHz, 256MB, 20GBTelling from my personal experience, anything below 512 MB of RAM is guaranteed to give you a sluggish Ubuntu system in the end.
So the best option, if it is feasible, is to double the available RAM to 512 MB. Of course, 1024 MB would be even better.
In case the machine simply cannot handle more than 256 MB of RAM you may have to look for a Ubuntu version which is really lean by design. I assume the current Lucid Lynx based [Puppy version v5.2.8]("http://puppylinux.org/main/Download%20Latest%20Release.htm") might do the trick.
In case Puppy is too far away from the genuine Ubuntu versions, you might give [Lubuntu]("https://wiki.ubuntu.com/Lubuntu") a try. The Lubuntu makers try to reduce the resources needed to run Ubuntu by using a lightweight windows management system named LXDE.

Kind regards,
Karl
