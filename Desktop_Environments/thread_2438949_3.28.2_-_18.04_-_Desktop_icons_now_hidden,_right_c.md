### Post by KenUBF on 2020-03-24
Hi dusf,

I'm sorry you've been having issues. To fully purge the random wallpaper program you can run in the command line (also called a Terminal) by typing [PHP]sudo apt purge [program name][/PHP] By doing the purge command you will not only uninstall the program but the configuration files that are left behind if you only use the uninstall command. 

As far as your desktop icons, did your settings for Gnome get changed by the programs? By default Gnome doesn't allow desktop icons and you have to change that in settings.
