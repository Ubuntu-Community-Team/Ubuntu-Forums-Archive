### Post by firedrow on 2008-04-27
I don't know of any screenlet or program that will allow you to do this, but if you want to keep Gnome, then also install Openbox (or a *box variant).  You can change your session to be Openbox+Gnome on the login screen, now you will have Gnome desktop manager, with Openbox environment.  You will have to configure the menu for Openbox before hand, but there are lots of tutorials for that across the forum.
