### Post by jromer on 2005-07-07
It was the fonts that attracted me to Kubuntu. 


I'm using Bitstream Vera Sans for all my fonts. The important thing is to go into control center and click the "Use anti-aliasing" box and then click the "Configure" button and unclick "Exclude range" box.

### Post by shakin on 2005-07-07
A while ago someone posted about how good a commercial font called Frutiger is and he showed some nice screenshots. Trouble is, the font is $185. I did some searching and found a knock-off font called Segoe UI that is very nice and I've been using it ever since.

The [topic is here](http://www.ubuntuforums.org/showthread.php?t=30357&highlight=segoe%20UI) and I attached the Segoe UI font to one of my posts along with instructions on where to copy it so it shows up as a system font.

I've been using Segoe UI ever since and have always found it to be a very nice font. Much better than any of the default Ubuntu fonts or anything in the msttcorefonts package.

### Post by oobuntoo on 2005-07-07

You can rerun this command and choose other options, but "Autohinter" seems to work best for me on CRT monitor. I'm using Bitstream Vera San for KDE.
