I am running a fresh install of Lucid 64bit with Desktop effects enabled and the latest driver from Nvidia 195.36.24 (but had the problem also with the ubuntu repo driver.

The issue can be reproduced as follows.  Set AWN to group common windows (I believe this is the default) Open several gedit windows (or any other app) Then open firefox and maximize the window (or simply move the window so it's directly above your gedit dock icon.)  In the AWN dock, select the gedit icon and try and select an open gedit window.  When you click the gedit icon in the AWN dock, you'll see a popup with all of your gedit windows.  If you move the mouse to attempt and select one of the gedit's, the menu disappears and you can't select anything.
