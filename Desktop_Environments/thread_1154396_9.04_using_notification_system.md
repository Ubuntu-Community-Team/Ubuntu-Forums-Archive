### Post by evrkusd on 2009-05-09
this is a sort of odd question but i've been trying to be able to show the time on my desktop without having to have it in the panel which i don't really use, and instead having it come up via a hotkey or something.

what i want to do is use the new notification system to display the output of the date command for the duration that i hold down a hotkey. is this possible at all? i've been reading up on the system some and would appreciate some hints about what direction to go in to accomplish this.
thanks

---

### Post by Brandon Williams on 2009-05-09
You could assign this command to a hot key: 'notify-send -t 10000 "Current Time" "$(date)"' It will stay up for 10 seconds instead of staying up for as long as you hold down the hot key, but it's close to what you're looking for.
