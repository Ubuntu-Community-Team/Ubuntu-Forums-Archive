### Post by shawnhcorey on 2010-09-07
**FYI**

To shutdown from a terminal:
```
sudo shutdown now
```

To reboot:
```
sudo shutdown -r now
```

You can switch to terminal mode: Ctrl-Alt-F1

There are 6 terminals, number 1 to 6: use F1 to F6.  Ctrl-Alt-F7 brings you back to the GUI.
