### Post by kpkeerthi on 2008-01-29
Basically, xfce comes packaged with different set of stock tools & utilities. For instance, you use a different tool to change your desktop background or window border. 

As for the softwares/packages, xubuntu comes prepacked with all the basic packages (like editors & terminal emulators) and you can always install the missing ones from the repositories. A package that works in GNOME also work in XFCE. There will not be any compatibilty issues as both are GTK based. IDEs like netbeans run equally good (perhaps faster) in XFCE and GNOME.

I find XFCE very stable.

### Post by x1a4 on 2008-01-29
Hi,

Xfce doesn't have its own file search tool, but you can use Gnome's, although Gutsy comes with the Tracker Search Tool installed.  Also, Xfce's menu could be better, but it's not too bad.  

Overall there is nothing you can do with Gnome that you can't do with Xfce.  As a matter of fact, Xfce has some decent compatibility with Gnome and KDE.  For example, you can have many of Gnome's panel applets in an Xfce panel. You can also run Gnome and KDE services in Xfce.
