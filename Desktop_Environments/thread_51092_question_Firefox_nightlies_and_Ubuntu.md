### Post by Morcas on 2005-07-22
As I have been using nightly builds of both FF and TB on Windows for the last year or so I was wondering what the process of using these builds was like on Ubuntu.

On Windows its a simple matter of downloading the zip, extracting it to where ever you want and starting either FF or TB with the profile manager to create a new profile, job done.

If it works, great if it doesn't just go back a night, its really that simple.

Can I expect the same ease of use with Ubuntu and can I keep the curent installed version intact, i.e. place the nightly build binaries and profile in a different directory?

Thanks.

### Post by mad_scientist03 on 2005-07-22
Yes, you should be able to do this. You can download the .tar.gz file from the nightly build section of the Firefox website, and then unzip and untar them into your home directory. A directory like "firefox-install" will be created in your home directory, and you can just ". ~/firefox-install/firefox" to run the nightly build. If you don't like it, wipe out the firefox-install directory, download a different build, and do it again. It takes no time at all.
