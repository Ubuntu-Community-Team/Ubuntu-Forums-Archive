### Post by DMurray on 2008-08-01
If you are using 8.04, this is a LTS version, which means it has special attention from developers, so it will be frequently updated in the first months.
There should be no problem updating everything. Some updates may depend on other packges being in the latest version, so, if you don't update 1 or 2 packages, and they are prerequisites for other packages, you won't be able to update these while those are not updated. Chain effect.
But there were updates that trashed some of my systems, specially back in the 6.10 days. But the last LTS, 6.06, never had this problem. 6.10 was doomed.
