### Post by Blutack on 2006-10-02
If you know the make of the card (you can usually find it on the manufacturer's website for you laptop) that would make things easier, especially if there is a native linux driver for it.  Or try ndiswrapper, which can take windows drivers for most cards and make them operate under linux.
Sorry this is marginally out of date but its better than I could write.

[http://ndiswrapper.sourceforge.net/mediawiki/index.php/Ubuntu](http://ndiswrapper.sourceforge.net/mediawiki/index.php/Ubuntu)

I would also highly recommend 'networkmanager' which you can install using the program synaptic.
Good Luck!
