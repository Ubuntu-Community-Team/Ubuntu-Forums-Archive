### Post by Zotova on 2005-10-10
When on XP I always used to have a software firewall, whilst my routers firewall is very good as protecting the incoming traffic it has no protection, like many router firewalls, for outgoing traffic. Thus in XP I used a software firewall as well so I could define what I wanted to access the net. The main reason for this was just on the off chance I ever got spyware (which was very rare) and so I could stop any programs which I felt didn't need to be accessing the net.

Anyhow, I've now being using Linux for about 6 months and about 1 month with just ubuntu on my computer.

So to the question really, now that spyware and junk programs calling home aren't an issue, do I need/would you recommend a software firewall for out going traffic or just another defence for incoming traffic?

### Post by frodon on 2005-10-10
I agree, there's no need of a firewall for basic use.
I run a software firewall (firestarter) because i run services on my linux box : ftp and samba server. But it's more for control reasons (block unwanted IP for exemple) than for security reasons.

So if you don't run services on your computer there's no need of a firewall.
