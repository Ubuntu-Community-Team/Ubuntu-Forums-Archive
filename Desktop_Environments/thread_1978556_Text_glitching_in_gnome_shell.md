### Post by stevedude on 2012-05-12
I can confirm the same issues. What I did was change my video driver. I use Gnome Shell too on a 64-bit Ubuntu 12.04 system. 

My video driver is for an NVidia chipset. 

For simplicity's sake, I clicked on my Name from the panel > System Settings > Hardware:Additional Drivers. Mine was actually set to "Post-release updates, Current Version updates" and I changed that to the "Recommended" driver version and restarted my system.

It's been about a week and so far I have not experienced that issue again.


PS - Some themes/windows decorations can cause issues too so try to restart the gnome shell with the default version, if that applies to your situation.
