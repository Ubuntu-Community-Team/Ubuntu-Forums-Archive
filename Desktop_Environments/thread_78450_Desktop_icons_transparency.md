### Post by Stormy Eyes on 2005-10-18
**beforefirstlight**, I think the problem is with the Modules section of your xorg.conf. Make it look like this, and you should be OK:
```
Section "Module"
	Load	"ddc"
	Load	"dbe"
	Load	"dri"
	Load	"extmod"
	Load	"glx"
	Load	"bitmap"
	Load	"freetype"
	Load	"type1"
	Load	"vbe"
	Load	"int10"
EndSection
```
You shouldn't have the GLcore module loaded if you're going to use the nvidia driver and not nv or vesa, by the way.
