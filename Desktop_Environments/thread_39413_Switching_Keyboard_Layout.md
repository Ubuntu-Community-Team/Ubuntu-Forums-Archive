### Post by Alexander Kirillov on 2005-06-06
[QUOTE=CospeFogo]Yes, but I'd like to switch between them without doing this. That's not practical to do it everytime I give focus to the other app.[/QUOTE]
 First of all, you can have more than one layout and switch between them by pressing, e.g., left_shift+right_shift (combination can be configured in references - keyboard - layout options-group shift/lock). This is what I do.

But it is also possbile to have layout switched automatically when you change window focus.  Add keyboard indicator  applet to your panel. In preferences for it, there is "separate group for each window". Check it.
