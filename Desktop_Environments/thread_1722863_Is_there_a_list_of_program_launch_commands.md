### Post by xdominex on 2011-04-06
Hello, everyone!
Is there somewhere that I can get a list of the actual commands to launch all of my programs? There are lots of nice gui launchers in my menus, but I would like to know all of the actual commands for my programs for those cases in which I want to add conditions to a program launch (i.e. "gksudo" before the command). ### Post by Copper Bezel on 2011-04-06
Got to Preferences -> Main Menu. You can edit the menu there, but double-clicking any item will show you the actual command it drops into bash. = )

---

### Post by xdominex on 2011-04-06
Ah! Excellent idea! Thank you very much. It's not the slickest way ever, but the complexity of the issue probably makes this the best way currently available.

### Post by Krytarik on 2011-04-06
You can also lookup their commands via the file "/usr/share/applications/desktop.${LANG}.cache".
