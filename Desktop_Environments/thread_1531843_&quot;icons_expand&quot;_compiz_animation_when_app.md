### Post by mcduck on 2010-07-15
Actually no, it's not a Compiz effect. It's done by the gnome-panel itself when there's a compositing manager running (which is why you only see it when you have Compiz enabled), and it's not customizeable (apart from the ability to disable it together with all other panel animations).


The only way to get a similar effect but with more customization options would be using some other panel or dock that has such options. With a bit of configuration you could probably make some of the popular docks to more or less like the panels do, so you'd only loose the option of running panel applets in that dock.
