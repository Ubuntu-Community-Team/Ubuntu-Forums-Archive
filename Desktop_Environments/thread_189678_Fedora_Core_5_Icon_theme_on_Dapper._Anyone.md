### Post by mejy on 2006-06-05
The package you're looking for is 	download.fedora.redhat.com/pub/fedora/linux/core/5/i386/os/Fedora/RPMS/redhat-artwork-0.241-1.i386.rpm.
Open it with file roller and go to /./usr/share/icons/Bluecurve/ (or any other directory in \icons\ you want) and copy it into either /usr/share/icons/ or ~/.icons.
