### Post by d_yat on 2008-03-20
lol!

I've just been trying to solve this problem myself (different users, different resolutions in xubuntu)!

And I accidentally stumbled on the answer. So if you are still having the problem, here's the answer.

I presume before, you were changing it by going through,
Menu -> settings -> Screens and graphics

Yea, that does seems to change the default setting for everyone, to change it for just a single user (just their session)
instead goto,
Menu -> settings -> Display Settings

Changing this will change only that user's resolution!

hope you still needed the help, or at least find the info useful anyway.
