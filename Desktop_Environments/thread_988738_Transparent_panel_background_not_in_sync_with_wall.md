Are you sure you've set Transparency? Looks to me like you've set a Pattern.. here's how to confirm:

Right click an empty area in the panel, then choose "Panel properties", then "Background", then make sure "Solid Color" is checked, and not "Background Image".

You can then set the transparency as you want with the "Solid Color" slider bar.
