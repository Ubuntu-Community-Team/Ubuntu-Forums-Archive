I am having a problem with this hardware. It contains an Intel GMA X4500HD Video Adapter, but it is only allowing me to use VGA modes and resolutions. Could someone tell me how to get the higher resolutions enabled. It may involve changes to the configuration for the xorg, in which case I will need specific instructions. ### Post by fballem on 2010-08-16
Solved - I was able to use Systems > Preferences > Monitors to select the LCD monitor. Once that was selected, I was able to choose the right resolution. Now if only I could get this as the system default so that the screen would always come up with that resolution.

As it is, it changes when the user logs in, and that works well enough for now.
