### Post by luddezxc on 2012-08-02
I remember in old versions of ubuntu, I can hold a window with my mouse click, then use keyboard shortcut to switch workspace, the window will be moved into the new workspace. It seems that this convenient feature no longer exist in 12.04. Can someone tell me how to get it back? Thanks a lot.

---

### Post by OM55 on 2012-08-02
Hello luddezxc,

You can create a keyboard shortcut.

1. Launch 'System Settings' and select Keyboard
2. Select the 'Shortcuts' tab
3. Select the 'Navigation' option on the left pane
4. Click on any of the first 4 (in the default installation there are only 4 workspaces) to program a keyboard shortcut to your liking

Hope that helps!
