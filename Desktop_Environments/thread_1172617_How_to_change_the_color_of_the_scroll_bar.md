### Post by Yvan300 on 2009-05-28
The scroll bar colour is part of GTK themes. Head over to gnome-customize.org and install a theme that has your scroll bar's colour. But i'm not sure if you can individually change the colour of the scrollbar when using metacity. Try using emerald, it is more customizable.

### Post by Viva on 2009-05-29
Apologies, just noticed that it doesn't work with all gtk engines. Go to Engines tab in gnome-color-chooser and choose clearlooks engine for scrollbars. You can notice the changes now.
