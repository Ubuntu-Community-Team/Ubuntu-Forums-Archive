### Post by Feanor on 2005-07-20
do this 

```
sudo aptitude install kde-i18n-es
```

this will set your kde to spanish language

Also install language packs

```
sudo aptitude install language-pack-es      
sudo aptitude install language-pack-es-base
sudo aptitude install language-pack-es-update  

sudo aptitude install language-support-es
```

Now all your system should be in spanish  ;-)
