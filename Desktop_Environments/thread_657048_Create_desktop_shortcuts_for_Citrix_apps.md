### Post by najames on 2008-01-16
NOTE: I am using Mint Daryna 32bit (gutsy?)

1) I used sudo ./setupwfc and installed the Citrix client from their tar file, let it install to the default directory /usr/lib/ICAClient
2) I installed libmotif3 with synaptic.
3) I opened a Firefox browser and went to my company's Citrix logon page, dropped the URL icon on the tool bar and just click on it to logon, entered my ID, 4digit code, keyfob code and it worked. I guess one could drop it on the desktop too as a shortcut, tried it and it works too.

I can click on the link icon and access all the company web pages, log on to the mainframe (gag), check email in Outlook on the company home page, but I can't remote desktop because I recently got a new PC and the computer ID has changed, they must not have it in their system it says "the specified remote computer can't be found".
