### Post by Seq on 2011-06-30
Looks like they are both Intel chipset-based graphics, from what Google turns up. As far as I know, that should run Unity. They are different chipsets though (Q35 and Q45 iirc, I closed the tab already), so it is possible there is some compat issue with one that would not affect the other.

You don't need to install and migrate to test. Plug a monitor into the other computer and boot a live cd.

Also, if you can swap the hard drives between the two, there is no need to reinstall and move your data in if you decide to go with the 780. It should work fine.
