### Post by Jucato on 2006-02-06
@MichaelVD: oooh I remember that thread. ehehe!

Well, first of all, let me say that I've grown to love Konqueror because of it's abilities both as a browser and file manager. Sure there are some sites which still don't display well, but it doesn't mean that the situation will be like that forever. let's wait for the next update of Konqueror. 

As for the features you wanted:
1. Flashblock and Adblock - **integrated** in Konqueror. That's right. Built-in. No extensions needed.
2. Ctrl+F search: true, this is something I'm still looking for. But there's an alternative. "/" - Find text while you type, and "." Find link while you type. It get's a bit of getting used to, but almost equally as powerful.
3. Shortcuts? I don't know actually what you mean.
4. Passwords: KDE has it's own built-in Password Manager, KWallet. It takes care not only Konqueror's passwords, but everything else in your system that needs passwords EXCEPT for your user passwords.

Things to love about Konqueror:
1. Awesome integration with KDE (sure beats the crap out of IE)
2. built in blockings
3. lightweight, not so much of a resource hog unlike Firefox
4. extremely customizable, without the need to use extensions.

Things I'm looking for in Konqueror:
1. extension support (probably firefox's or develop its own)
2. better web page rendering
3. probably the option to have its own theme/style
