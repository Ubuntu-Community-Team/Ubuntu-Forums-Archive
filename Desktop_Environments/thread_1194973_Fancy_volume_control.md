### Post by david.konsumer on 2009-06-23
I just made a really fancy volume control for xfce/xubuntu.

File is [here]("http://code.google.com/p/dkbits/source/browse/trunk/volume_control.py").

Download and set executable:
```
chmod 755 volume_control.py
```

use it like this: assign multimedia keys (mute/volume up/volume down) to these actions:
```

LOCATION/volume_control.py mute
LOCATION/volume_control.py up
LOCATION/volume_control.py down

```

you can set keys in Settings/Keyboard/Application Shortcuts
