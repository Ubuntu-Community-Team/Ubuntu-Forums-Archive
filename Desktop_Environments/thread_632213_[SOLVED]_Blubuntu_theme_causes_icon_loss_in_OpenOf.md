### Post by aridus on 2007-12-05
I have installed (on 7.10) the beautiful Blubuntu theme from Synaptic and have found that it causes the toolbar icons to disappear in OpenOffice (leaving only text). Reverting to the Clearlooks theme causes the icons to reappear. I have tested this on two computers and have also searched the Forums for a solution to no avail. Does anybody know how I could have both Blubuntu theme and the OpenOffice toolbar icons? 

With thanks!

---

### Post by metalheart on 2007-12-05
I guess the Bluebuntu theme does not include all icons for OpenOffice.

You can use the Bluebuntu theme and set different theme for OpenOffice. (I have non-English OO, so look for similarly named menu items) In OpenOffice select Tools -> Options -> OpenOffice.org -> View. For Icon size and style change Automatic to Tango, Human or something that pleases you.
