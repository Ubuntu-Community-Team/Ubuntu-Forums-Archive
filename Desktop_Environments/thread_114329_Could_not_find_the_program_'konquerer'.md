### Post by GeneralZod on 2006-01-08
Try System Settings -> User Account -> Default Applications -> Web Browser.  If there's nothing untoward there, try opening up a konqueror instance and going to Settings -> Configure Konqueror -> File Associations and seeing what "html" is associated with.

Hope this helps,
Simon

Edit:

This is for KDE 3.4.3 - I'm not sure if things are any different in 3.5 :/
