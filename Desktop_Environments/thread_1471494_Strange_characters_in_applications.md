### Post by tom4everitt on 2010-05-04
Does the problem still persist if you create a new user, and open/install the programs as that user? 

It sounds indeed as if something must have gone wrong with your character sets. If nothing else, a fresh install (where maybe you do not install any cyrillic characters) might help. Note that you don't have to delete your current system to do that, you can just shrink the partition(s) of the current system, and add one for the new system. Then you haven't lost anything if it turns out it didn't work.
