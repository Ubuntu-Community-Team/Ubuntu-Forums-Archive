### Post by drs305 on 2011-01-31
bigfootjonesy,

Welcome to the Ubuntu forums.

The best way for us to help is to see the status of the Ubuntu and Windows boot files. If you can boot to the Ubuntu installation/LiveCD Desktop, go to the following site and download/run the boot info script. Then post the contents of the generated RESULTS.txt file.

When you paste the contents, click the # icon in the post's menu, which will generate 'code' tags, and paste between them.

[http://bootinfoscript.sourceforge.net]("http://bootinfoscript.sourceforge.net")
