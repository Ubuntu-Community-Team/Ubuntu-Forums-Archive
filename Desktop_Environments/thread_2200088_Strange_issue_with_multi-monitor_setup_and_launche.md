### Post by tozer83 on 2014-01-17
So to begin I am using a laptop with an external monitor plugged into it which is to the left of my laptop. This issue never happened before until today and since I last had the multi-monitor set up it was working as it should.

What happens is when I have the external monitor plugged in and the launcher is set to only show on my laptop the launcher will not show up when I mouse over to the left side of my laptop screen(whereas a couple of weeks ago it was doing so). If I press alt+f1 or superkey then the launcher does show up so I know it's there but it won't show up when I mouse over. If I go into the display settings and change the external monitor to be on the right side then the launcher will show up on the laptop when I mouse over to the left side of the screen. So for whatever reason, if the external monitor is set up on the left side the launcher will not show with the mouse but if it's on the right side it works as it should.

In displays I have sticky edges off, the reveal sensitivity under Appearance>Behaviour does not change and I've even installed CompBiz Settings Manager to see if the Launcher Reveal Pressure(under Ubuntu Unity Plugin>Experimental) has changed for some reason and it has not.

There have been some updates done on my laptop as well so I'm almost wondering if there was something changed that I'm not aware of that doesn't allow what I want to do anymore. If anyone has any suggestions or anything please let me know because it's rather annoying.

I am running ubuntu 12.04.

Thanks!
