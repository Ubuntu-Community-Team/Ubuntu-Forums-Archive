### Post by larrygingras on 2013-06-05
After careful research, I found out that it was "mouse mark" desktop effect.
Also arrows feature seems to be undocumented. It is activated by pressing Ctrl+Shift+Meta twice (first for end and then for beginning of the arrow). It seems that I accidentally pressed this shortcut.
But this effect itself seems to be handy for doing screenshots in which you need to highlight something. No gimp editing anymore.
