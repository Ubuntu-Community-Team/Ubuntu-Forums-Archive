### Post by TheFu on 2016-04-06
The installation of programs shouldn't cause any issues - gnome3, unity, Mate, whatever.

The issue is with config files that get re-used.  To avoid that issue, create new userids for each DE and login under the userid you want to configure for use with the new DE until you are certain it is the one you'd like to make permanent.  If you find that you don't like it, simply delete the userid and the HOME for that userid - no harm done.  Make sense?
