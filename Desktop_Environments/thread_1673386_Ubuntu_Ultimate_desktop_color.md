### Post by Krytarik on 2011-01-22
> **Tigerlinux said:**
> desktop background can be changed, but it has nothing to do with other applications color.
You didn't mention that you want to change the application background/colors. Those are indeed included in the theme, sometime one has the option to change some colors via the "Customize" dialog, like with the default themes. That requires that the author of the theme has build in those options.

If you want to change the colors of your specific theme manually, open the file "gtkrc" in the "gtk-2.0" subdirectory of that theme. In that you will find -usually at the beginning- a section specifying the default colors of the theme. It may be that in the sections further down the file some colors are specified again or different.

Good luck!

**EDIT:** If UU doesn't apply all the colors included in a theme, I don't have a solution at hand currently.
