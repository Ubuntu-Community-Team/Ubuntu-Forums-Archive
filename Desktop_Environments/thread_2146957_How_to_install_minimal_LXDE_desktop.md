### Post by 2F4U on 2013-05-20
The LXDE desktop environment is in the package lxde, while the desktop from Lubuntu is in the package lubuntu-desktop. The package lubuntu-desktop contains, besides LXDE, additional applications. So, for a minimal installation, the package lxde would probably what you are looking for.

---

### Post by kalehrl on 2013-05-20
For minimal Lubuntu installation, choose lubuntu-core.
For minimal LXDE installation, choose lxde-core.
