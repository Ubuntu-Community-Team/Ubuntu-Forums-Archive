### Post by ubunterooster on 2010-11-02
I use Ubuntu Tweak for this.

[**Ubuntu Tweak **Version 0.5.7]("http://launchpad.net/ubuntu-tweak/0.5.x/0.5.7/+download/ubuntu-tweak_0.5.7-1_all.deb")

After installing go to system tools>Ubuntu tweak>Gnome settings>"click this button to change menu logo"
Use something like this: [http://albertux.ayalasoft.net/wp-content/plugins/useragent-spy/img/24/os/ubuntu.png](http://albertux.ayalasoft.net/wp-content/plugins/useragent-spy/img/24/os/ubuntu.png)
Or any 24x24 png
