### Post by Mahinva on 2008-12-14
Is there any way to remove applications from the "Open With >" menu that appears when you right-click on files? 

I had downloaded a program called Notepad++ and installed it via Wine. Currently, it has been completely removed from my OS as far as I can tell. For some reason, it still shows up if I right-click on a file and select "Open With>" and it shows up 3 times in the"Open with other Application" menu. Furthermore, Notepad++ still shows up as the default application for handling certain files (such as .lua used in World of Warcraft).

---

### Post by ajgreeny on 2008-12-14
Right click on that type of file in nautilus, choose properties and go to the "Open with" tab.  There you can highlight the application and remove it.
