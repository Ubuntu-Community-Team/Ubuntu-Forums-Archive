### Post by MarseHole on 2007-03-03
To make a panel smaller all you have to do is right click on it, choose **Properties->Size** and change the size.

The minimum size of the panel (and in fact the size of window title bars) is however determined by your default font size, so if you've got a large font, it won't let you set the panel heights very low. To change the font size simply go to **System->Preferences->Font**. 

Hope it helps
