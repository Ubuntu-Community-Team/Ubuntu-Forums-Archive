I'm running ubuntu 10 which has been packaged as Moon OS in a VM on a Mac host. It uses the Gnome desktop. It's a very nice environment to work in and all the software that I want to run seems to be quick and responsive. 

The only nuisance is that sub-menus can be very slow to draw. Whenever a submenu is not drawn properly it is not cleared from the screen. It stays there, drawn above everything else until I go back to the submenu.

It often happens because I move too slowly across a menu item that has a submenu. The submenu starts to appear but my mouse has moved on. 

My work-around is to locate the correct menu, then the correct sub-menu, then hover the mouse over it so that the sub-menu has time to draw. I may or may not need to use the menu item to clear it from the screen.

Is there a system setting that I can alter? This is a VM so I am able to bump up the RAM if that is a solution.
