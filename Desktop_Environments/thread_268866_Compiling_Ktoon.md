### Post by xtingray on 2006-11-26
When you try to work with the Qt packages of *any* distro, it is a little messy... cause it is very hard to differentiate between 4.x binaries and 3.x binaries. 
i.e. /usr/bin/moc usually is pointing to the 3.x version.
If you can to compile Qt 4.1.4 and to install it in a different path, it makes things very easy to you.
It takes time, but if you want to try it, check this link:
[http://ktoon.toonka.com/documentation/index.php?title=Compiling_KToon_using_QT4](http://ktoon.toonka.com/documentation/index.php?title=Compiling_KToon_using_QT4)
