### Post by smartboyathome on 2007-07-02
To make GNOME start up Fluxbox instead of Metacity, just open system > preferences > sessions, click new, and create an item with "Fluxbox --replace" as the command (without quotes). Now GNOME will start up with Fluxbox instead of metacity. If you would like to switch back to metacity, push alt+f2 and type (without quotes) "metacity --replace". Hope this helps!
