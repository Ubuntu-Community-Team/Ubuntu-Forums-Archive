### Post by Musicalmidget on 2009-04-25
You can install the ordinary version of Ubuntu (with Gnome) add then KDE to your current installation by installing the kubuntu-desktop package.  Similarly, you can add XFCE by installing the xubuntu-desktop.  I know that when you install the kubuntu-desktop package you're asked which login manager to use, Gnome or KDE.  I suspect the same is the case when installing XFCE but I haven't yet done this myself.

If after installing KDE or XFCE you decide to remove Gnome, you've essentially converted your system from Ubuntu to Kubuntu or Xubuntu, without the need of separate images for each distribution.
