### Post by 3o5 on 2008-04-08
I have Windows XP Pro dual booted with Ubuntu 7.10

After i did the major updates after installing ubuntu 7.10 i got a pop up with Restricted Drivers Manager, you can check there to see if you are able to download and install drivers for the 8800, im using the 8600GT so im not sure if atm 7.10 will support these video cards but i would think so, as the other person said the new beta works great with the 8800, so best bet would be wait till it releases then you should be good to go. Most of the programs you are looking for like to burn CDs are already installed not sure about DVD burning, you can always check Applications > Add/Remove Programs - there you will find a list of software that you can install straight from ubuntu instead of looking through google for stuff, some of the programs that have are great. Would probably be personal perference. I usually dont burn DVDs cuz i will either watch them on my computer or burn them as a DATA and watch them on my Dvix DVD player. 

I remember seeing a couple programs for ipods, i havent tried it with mine, i have a 2nd computer which i use for my ipod Windows installed on 2nd comp. Image collection, you can either partition your HD into parts or just have a folder for ALL images, i used folders but then again i have 4 different HDs.

Playing music, i have used the default music player and didnt like it much was to used to Winamp, downloaded 2 different programs and i enjoy both Audacious & SongBird both very good programs, songbird reminds me of winamp. I use VLC for movies. Firefox is installed as default web browser, you can also install Opera and ive seen some people using Safari. Bit Torrent is installed with ubuntu, not sure if utorrent is supported havent downloaded torrents yet with ubuntu. As far as codecs for your movies/vids VLC has always been one of the best players out ive read that MPlayer has a lot of codec support as well MPlayer can be installed thru the Add/Remove Programs feature.

Also Ubuntu has Open Office installed which is just like Microsoft Office

Hope some of my rambling helped any :lolflag:
