### Post by schnelle on 2012-02-14
I am using Kmess and it works very well.

Note that Kmess version in official repositories doesn't work well because Microsoft changed some server configurations and it broke all free messengers.

You have to compile Kmess from git or find ppa with updated Kmess. I found this one: [https://launchpad.net/~bill-zt/+archive/kmess](https://launchpad.net/~bill-zt/+archive/kmess) .
I hope it works for you.

Cheers.

### Post by schnelle on 2012-02-14
I should add that amsn and kopete are no longer maintained (they are dead projects).

There is new project which will replace kopete. It is currently in beta faze. Telepathy-kde: [http://martys.typepad.com/blog/2012/01/kde-telepathy-03-ninja-released-first-beta.html](http://martys.typepad.com/blog/2012/01/kde-telepathy-03-ninja-released-first-beta.html)
ppa: [https://launchpad.net/~telepathy-kde/+archive/ppa](https://launchpad.net/~telepathy-kde/+archive/ppa)
