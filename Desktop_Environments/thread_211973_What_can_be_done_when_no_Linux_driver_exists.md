### Post by GarlicFish on 2006-07-09
A lot of printer models can use the same driver.  Sometimes you can get away with using a the driver for a different model.
I use the HP Photosmart 8400 driver entry for my Photosmart 8450.  Actually a lot of the HP models use a common hpijs driver.

You could try configuring your printer using the cups imagerunner 300 driver or try any of the other cannon drivers listed in the cups printer list.  or try the HP laserjet or deskjet drivers.

try simple text print jobs from gedit before trying formatted text or graphics.  If the printer does not like the driver it will churn out page after page of garbage, so best only load a couple of sheets.

Trial and error may get you a good result.
