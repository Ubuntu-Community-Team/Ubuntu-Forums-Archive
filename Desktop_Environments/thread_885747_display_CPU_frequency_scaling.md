### Post by Marrkk on 2009-11-01
Yes.. 
 its a widget. here's how.
right-click the panel, panel options, add widget,
click the ' install new widget' button
then
'download new plasma widgets'
order by ' most downloads '
search for cpu
select  'cpufreqdisplay' then  after they've ticked and d/loaded, close that window

you have now installed the new widget. now to use it.
back in the All Widgets window, scroll down to cpufreqdisplay
(the icon doesn't usually show)
select cpufreqdisplay then click the 'add widget' button ..
Hey Presto !!  - it should be working, sat just right of yout clock
