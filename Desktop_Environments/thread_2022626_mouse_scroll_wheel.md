### Post by sudo smith on 2012-07-12

> I have a problem with my ubuntu. When i push muy mouse scroll wheel up, the browser o konkeror, etc go back. Is like when y press the return button on W7. How can disable this option?

Hmm seems like you might have the mousewheel set up as a shortcut to go back. To check if this is true go under setting -> configure shortcuts [might be in the older versions under configure konqueror] and make sure that the shortcut for "back" isn't set to scroll down. If you never messed around with shortcuts I would recommend recommend just clicking "reset to defaults" in case anything was accidentally changed. I'm not very familiar with konqueror, but I know that in firefox if you press shift scroll down it will go back. Hope it helps.
