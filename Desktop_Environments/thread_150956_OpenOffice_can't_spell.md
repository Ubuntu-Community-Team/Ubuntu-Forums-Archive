### Post by Neeko on 2006-03-27
I have finally got this sorted. Why it didn't work on installation is beyond me, but if anyone finds themselves in the same position go here: [http://lingucomponent.openoffice.org/dictpack.html](http://lingucomponent.openoffice.org/dictpack.html)
download the file called DicOOo.sxw and open in OOorg Writer. It contains a macro, which will download and install the dictionaries for you.
