### Post by MJN on 2006-03-02
You'll need to elaborate on exactly what you want to do.
 
As per the above, if you want statistics over time then I would recommend '**AWStats**' ([http://www.awstats.org/]("http://www.awstats.org/")). You can see the typical (default) output of this running on my site at [http://www.newtonnet.co.uk/cgi-bin/awstats.pl]("http://www.newtonnet.co.uk/cgi-bin/awstats.pl").
 
If however you want to see connections in real time, then something as simple as the console app '**IPTraf**' ([http://iptraf.seul.org/]("http://iptraf.seul.org/")) works very well. Check out the screenshots at [http://iptraf.seul.org/shots.html]("http://iptraf.seul.org/shots.html"). Incredibly useful for other network-related tasks too.
