### Post by Krytarik on 2011-06-20
In Natty 11.04, there is no "Visual Effects" tab in "Appearance" anymore, because whether Compiz is run or not is now determined by the session settings, and respective support checks.

Please see this guide on (1) seeing the results of those checks, and (2) skipping them if needed:
[http://ubuntu4beginners.blogspot.com/2011/05/force-unity-compiz-to-run-natty-narwhal.html](http://ubuntu4beginners.blogspot.com/2011/05/force-unity-compiz-to-run-natty-narwhal.html)
