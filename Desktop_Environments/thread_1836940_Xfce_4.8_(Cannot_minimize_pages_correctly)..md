### Post by Toz on 2011-08-31
Try middle-clicking on the desktop to get a list of all running programs (including minimized ones). I believe you are talking about the window menu plugin (to see the names of the running programs). To get it back, right-click the panel, select Panel->Add New Items, and add the "Window Menu" plugin.
