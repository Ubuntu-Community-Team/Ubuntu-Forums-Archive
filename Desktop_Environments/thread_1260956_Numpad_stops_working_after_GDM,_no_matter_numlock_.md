### Post by kurtkraut on 2009-09-08
Hi,

I'm using Jaunty and out of the blue, my numpad stops working after I log in in GDM, no matter if numlock is on or off. I have no acessibility config set, but the numpad keys when pressed moves the mouse cursor.

I'm sure my keyboard is working properly since in gdm all keys work (while filling up username and password) and because this keys are currently equivalent to moving the mouse.

Any clue how can I fix this?

---

### Post by Linteg on 2009-09-08
Start System->Preferences->Keyboard, select the "Mouse Keys" tab and uncheck "Pointer can be controlled using the keypad"
