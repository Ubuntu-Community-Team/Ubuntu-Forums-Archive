### Post by willie_wang on 2007-12-17
ok. it seems as though the problem might have gone away. :)

it's been a fair while and the black bars haven't reappeared.

i edited my xorg.conf and added the following line to the device section:

```
Option		"XAANoOffscreenPixmaps"	"true"
```

So the whole section looks like:

```
Section "Device"
	Identifier	"aticonfig-Device[0]"
	Boardname	"ati"
	Option		"XAANoOffscreenPixmaps"	"true"
	Busid		"PCI:1:0:0"
	Driver		"fglrx"
	Screen	0
EndSection
```

fingers crossed! :)
