### Post by taurus on 2008-03-15
If you want to install it by hand, make sure you are in the directory where what file is and run

```
sudo ./vmware_install.pl
```
However, I believe vmware is in the repos so why not Search for it in System -> Administration -> Synaptic Package Manager -> Search and install it from there.  Much easier.
