### Post by Teamgeist on 2006-08-10
As an alternative to gnomebaker you could try Bonfire! [http://perso.orange.fr/bonfire/](http://perso.orange.fr/bonfire/)
It uses gstreamer0.10.
Simply add the following line in your **/etc/apt/sources.list** :

```
deb http://ketsugi.com/ubuntu dapper main
```

then type

```
sudo apt-get update
```

and then

```
sudo apt-get install bonfire
```

It really is a cool programm and for me it works better than gnomebaker 0.5.1!
