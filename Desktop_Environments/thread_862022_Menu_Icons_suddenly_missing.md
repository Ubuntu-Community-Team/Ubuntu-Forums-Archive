### Post by ripps818 on 2008-07-17
All icons in menus (Gnome Menu, firefox menu, context menu) are missing. Any other icons aren't affect, including panel icons and desktop icons. I've tried gtk-update-icon-cache several times, but nothing happened.

### Post by ripps818 on 2008-07-17
Nevermind, somehow the "Show icons in menus" option in System->Preferences->Appearence->Interface had somehow been ticked to off. Strange considering that this occured while I was watching a movie with smplayer.
