### Post by rsambuca on 2007-05-07
I think it totally depends on what type of document you are creating.  If it is not too complex and can be created easily in OpenOffice, I would just use that as the pdf export functions very well.

If you are creating a more elaborate document like brochures, then you would probably be better of using a Desktop Publisher like [[COLOR="Red"]Scribus[/COLOR]]("http://www.scribus.net/").  It is in the repos so you can install it simply using the Synaptic Package Manager.
