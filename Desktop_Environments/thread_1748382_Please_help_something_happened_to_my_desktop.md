### Post by Copper Bezel on 2011-05-04
Open a terminal by pressing Ctrl+Alt+T. It sounds as if Unity is having a problem - do you remember what you clicked that might have caused the problem? If it was a setting in CompizConfig, where you can set Unity preferences, you can run CompizConfig Settings Manager from the terminal by typing [font="Courier New"]ccsm[/font] .

If you don't remember or weren't editing anything in CompizConfig when the problem started, try [_this guide_]("http://www.webupd8.org/2011/04/how-to-reset-unity-launcher-icons-or.html") for resetting Unity to default settings. If none of those options work, we'll have to start installing some things. = /
