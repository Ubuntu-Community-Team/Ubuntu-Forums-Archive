### Post by magisterms on 2010-02-25
Hey, I had a similar problem, no right click are working in Firefox **at all**...

This is definitely not a mouse sensitivity issue, because when I am changing over from "right hand" mouse to "left hand" mouse, suddenly the left click stopped working.

This disables the user to do certain things in Firefox, most stupidly to open something in a new tab for instance.

_Workarounds:_

[LIST]
[*]Ctrl + click - open the link in a new tab (unfocused)
[*]Shift + click - open the link in a new windows
[/LIST]
If you are using mouse gestures, make sure that that you turn off the trigger for scroll wheel navigation (which is assigned to the right click in the *All in one Gestures* addon in Firefox)
