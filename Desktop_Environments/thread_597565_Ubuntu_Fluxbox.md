### Post by taurus on 2007-10-30
You can install .deb with

```
sudo dpkg -i filename.deb
```
You can unpack .tar.gz and .tar.bz2 with

```
tar xvzf filename.tar.gz
tar xvjf filename.tar.bz2
```
