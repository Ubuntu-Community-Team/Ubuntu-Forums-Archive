sudo smbpasswd -a yourusername

### Post by Ramses de Norre on 2006-04-11
You'll need to add a line ```
ramses = "network username"
``` in /etc/samba/smbusers (ramses is my username) and then execute ```
smbpasswd -a yourusername
```
To set a user name and passwd, those are asked when browsing the shares through windows.
