### Post by john.nicholls on 2007-04-30
Make sure you have ticked "Automatically save session on logout" under Sessions and Startup.
Then when you have xfwm4 running and exit, the setting should be saved.

By the way, what you described under Beryl ("the four sticks") is normal.
