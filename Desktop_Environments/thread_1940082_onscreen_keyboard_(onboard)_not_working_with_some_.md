### Post by Favux on 2012-07-05
Hi xianbei,

Consider using Magick Rotation:  [https://launchpad.net/magick-rotation](https://launchpad.net/magick-rotation)  to get automatic rotation when swivelling to tablet mode.  It also allows you to toggle touch on and off.  Its default onscreen keyboard is CellWriter which may avoid the problem you are seeing with Onboard.  Plus you may consider CellWriter a better choice for a tablet PC.  So first install CellWriter and see if you like it.  You can change the default onscreen keyboard in Magick if you want.
