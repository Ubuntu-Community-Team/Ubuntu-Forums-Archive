### Post by lgoss007 on 2005-09-08
I was running E17 as stand alone as well. But I wanted nautilus, so all I did was create an eap file for gnome-settings-daemon, and started that on E17 startup.

It works pretty well except the first time you open nautilus it opens the gnome desktop (basically the background image fills the screen). All I do is put the mouse over the desktop though and press "ctrl-alt i", and viola, no more desktop (it's actually just iconified). It's not perfect, but I like nautilus better than any other file manager (and it's looking even better in breezy).

### Post by Wolki on 2005-09-10

You can disable that in gconf. Uncheck apps/nautilus/preferences/show_desktop.
