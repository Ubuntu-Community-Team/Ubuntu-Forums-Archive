After some particular update in Gutsy about 2 weeks ago I had the following problems.


1.Menu icons disappeared for all users

SOLVED by going to System|Appearance|Interface. Deselect and reselect 'Show icons in menus'.  Repeat for each affected user (all users in my case)


2. window title bars centered and skewed (see attached photo)

TRIED different theme, rebooting, reinstalling gnome, reinstalling compiz



Any help or suggestions would be greatly appreciated!
