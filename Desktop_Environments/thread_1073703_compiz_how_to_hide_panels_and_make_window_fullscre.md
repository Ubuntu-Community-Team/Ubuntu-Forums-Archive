### Post by mcduck on 2009-02-18
Go to System/Preferences/Keyboard Shortcuts, browse down to Window Management and set "Toggle Fullscreen mode" to whatever you want (Alt+Return is pretty common combination for this).

This works with and without Compiz, and with any program that allows resizing it's window.
