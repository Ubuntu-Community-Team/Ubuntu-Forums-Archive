### Post by Tomosaur on 2006-08-05
Ubuntu doesn't just update your operating system, it automatically updates all your installed software aswell. It's entirely up to you what you wish to update. If you see something you don't wish to update, just uncheck the box, I personally just allow everything to update.

### Post by az on 2006-08-05
After it's released, the only new package you will get are from the updates and the security-updates repositories.  There usually is a really good reason to upload a package to the repositories (think potential security vulnerability or bug which can cause data loss).  It is reccommended to install the updates.

Just don't go around adding thrid-party repositories to your list (non-ubuntu ones).  If you blindly install software from a non-ubuntu source, that is a security risk.
