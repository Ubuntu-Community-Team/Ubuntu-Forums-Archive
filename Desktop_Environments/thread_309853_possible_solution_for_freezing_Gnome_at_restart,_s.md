### Post by thejonnybgood on 2006-11-30
I am a complete noob to Linux & installed Ubuntu dapper last week. Sometime over the last 2 days (whilst adding stuff thru synaptic) I managed to find myself in the apparently common situation of my system freezing everytime I quit the Gnome desktop - whether it by reboot in terminal, any of the shutdown options or via ctrl + alt + backspace.

I dont know if this is pure luck or if more use to someone with more knowledge, but I seem to have got rid of the problem by doing the following:
-restoring my system to a 'pure Gnome environment' ... altho the reboot problem was still present after this step
- uninstalling aMSN (which was installed from the aMSN website using their instructions .. and uninstalled using their instructions too).
- lastly, but maybe most significantly, reinstalling my ATI driver using the instructions from [http://wiki.cchtml.com/index.php/Ubuntu_Dapper_Installation_Guide#Method_1:_Installing_Dapper.27s_Included_Driver_.288.25.18.29](http://wiki.cchtml.com/index.php/Ubuntu_Dapper_Installation_Guide#Method_1:_Installing_Dapper.27s_Included_Driver_.288.25.18.29)
the last two of these (aMSN & ATI) were done in same session and the system froze when I tried to restart the system from the terminal window, but after I shut down via the power button the problem has gone away!!
My guess is that reinstalling the ATI driver was the key, but I have no idea why or how?!?!?

Anyway - just thought I'd post this to see if it helps anyone!!
