Hi, I'm on a Thinkpad R500 and I'm using middle mouse button for scrolling (hold it down and move the trackpoint). The problem is that when I middle-click and don't move the trackpoint the clipboard text gets pasted in Eclipse or any other text editor. 

Is there a way to disable this system-wide (Ubuntu 10.04, Gnome)?

Edit: I found an archived thread with the same question and the answer was to disable the middle mouse button via xorg.conf. That would also disable scrolling and I like my middle-click-scrolling very much since I'm using a trackpoint.
