### Post by zombifier25 on 2012-06-24
If you're using Unity (with the big Launcher containing icons on the left) then no you can't.

If you're using GNOME Classic (which looks like old GNOME 2 with a top and a bottom panel) then it's Alt + Right Click (or Alt + Super (Windows button) + Right Click)
