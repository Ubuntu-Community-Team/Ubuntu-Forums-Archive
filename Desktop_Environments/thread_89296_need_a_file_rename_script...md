### Post by bartc on 2005-11-12
Mp3rename renames mp3 files based on their id3tags. It has an option to limit the name to 32 chars. Ofcourse it requires the id3tags to be set correctly. If that is not the case, I'm sure you can find a tool that allows you to set the tags based on the current names.
