### Post by Krytarik on 2011-01-06
If you set desktop effects to other than "None", Compiz gets used as window manager, whereas "None" choose Metacity. Thus different plugins get used.

In Compiz it is called "Static Application Switcher", you find it's settings via "System -> Preferences -> CompizConfig Settings Manager", if you have it installed.

Unfortunately there is apparently no option to disable the thumbnails.
