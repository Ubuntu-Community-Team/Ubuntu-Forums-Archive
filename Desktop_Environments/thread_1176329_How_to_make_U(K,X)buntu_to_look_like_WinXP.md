I would strongly suggest you don't use Ubuntu at all but try Puppy Linux to start. the more computer savy ones will want to try more things soon and can move to a bigger Linux, while the more simple users will not be confused by Puppy's simple interface. the big thing with Puppy is it runs as root and the user has access to everything and the file structure is as familiar as Linux can be to a Windows user.
