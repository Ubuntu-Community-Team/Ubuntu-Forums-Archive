### Post by DirtDawg on 2006-06-28
Do you mean the stroke width? 

Select the line in question with the select tool (F1), Open "Fill and Stroke Dialogue" (Shift+Ctrl+F), select the "Stroke Style" tab, adjust the "width" property to desired effect.

I hope this helps.
