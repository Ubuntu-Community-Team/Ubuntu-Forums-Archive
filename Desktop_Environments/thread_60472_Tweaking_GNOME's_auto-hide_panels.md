Just a quick question:

I have my bottom panel set to "auto-hide" in Gnome.  Do you guys know you to make it appear any quicker than it normally does?  It normally takes about a half-second, but I'd rather it be nearly instantaneous.

### Post by arnieboy on 2005-08-28
[QUOTE=GreyFox503]Just a quick question:

I have my bottom panel set to "auto-hide" in Gnome.  Do you guys know you to make it appear any quicker than it normally does?  It normally takes about a half-second, but I'd rather it be nearly instantaneous.[/QUOTE]
yeah thats easy. Go to System Tools--> Configuration editor. When it opens go to apps->panel-->global and change the panel_show_delay (right click on it and edit  key) to anything of your choice. By Default its set at 300 milliseconds. U can set it as close to zero as u wish.
Hope this helps.

Thanks for the reply, arnie.  I tried what you said, but that exact location did not change my panel behavior at all.  I think that might be the default behavior for new panels or something, but because of what you told me, I just poked around in the config editor.
