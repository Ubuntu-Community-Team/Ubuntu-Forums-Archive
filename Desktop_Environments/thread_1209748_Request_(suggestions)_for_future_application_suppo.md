### Post by QIII on 2009-07-10
I doubt you will find much support for such an idea.

The Synaptic Package Manager and APT are designed with the idea that they download and install packages specifically for Debian-based distributions (like Ubuntu) in a Linux environment.

Wine supports Windows applications in its "sandbox" on a case by case basis.  Not all applications work well with it and cannot, therefore, be guaranteed to work.  Wine is a concession to the popularity of Windows applications, not a core Linux goal.

You may make a post in [http://brainstorm.ubuntu.com/](http://brainstorm.ubuntu.com/), but I doubt it will be received well.
