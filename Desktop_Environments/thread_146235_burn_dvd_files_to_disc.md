### Post by taurus on 2006-03-17
You can use k3b to burn it.  Fire up k3b.  Then, File -> New Project -> New Video DVD Project.  Now, navigate to where both AUDIO_TS and VIDEO_TS directories are.  Just drag them to lower window and you are all set.  Then, click Project -> Burn...
