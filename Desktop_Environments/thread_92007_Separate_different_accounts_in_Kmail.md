### Post by asimon on 2005-11-19
Kmail is rather flexible. You can create multiple accounts under kmail for receiving mails and for sending mails. Kmail differencieates these two types of accounts for more flexibility. See Settings -> Configure Kmail... -> Accounts -> Receiving/Sending.

And you can create multiple identities (Settings -> Configure Kmail... -> Identities). You can also specify for each identity which sending account it should use and into which 'sent' folder it should put sent mail.

With those three things (receiving accounts, sending accounts, and identities) you should be able to get the same (or something similar) like with thunderbird.
