### Post by george.brindeiro on 2009-11-10
After a lot of googling for answers, found out the icon is called the MessagingMenu, which works with the indicator applet. The way I got it back was quite idiotic -- I right clicked the panel, selected Add to Panel and tried adding the indicator applet options. Along with some wrong icons, which I deleted, my mail/chat icon came back!

George
