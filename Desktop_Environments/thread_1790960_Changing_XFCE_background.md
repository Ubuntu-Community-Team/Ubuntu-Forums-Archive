### Post by TFD on 2011-06-25
If I right click on the Xbuntu XFCE desktop I see something about the background. But it only seems to let me select a finite set of backgrounds.. I don't see how I can add additional backgrounds, either from a web repository or from local image files.

---

### Post by nzjethro on 2011-06-25
To add additional image files to the list of selectable wallpapers click the **+** button below the list of images. This will open a file manager, in which you can navigate to any saved images you would like to set as your background. Just find the image you want, and click "+ Add".

---

### Post by Toz on 2011-06-25
Or create the directory **~/.local/share/xfce4/backdrops** and put your images there and they will show up in the dialog box with the others.
