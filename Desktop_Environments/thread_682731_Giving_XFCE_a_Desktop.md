### Post by x1a4 on 2008-01-30
Hi,

What exactly is it about Gnome's desktop that's better than Xfce's?  

Anyway, you can replace Xfce's desktop by killing **xfdesktop** and running Gnome's or KDE's desktop (gnome-desktop, kdesktop).

One way to kill Xfce's desktop is from Settings -> Desktop Settings and uncheck 'Allow Xfce to manage the desktop'.  Then open the Run dialog (Alt+F2) and run **gnome-desktop** or **kdesktop** or any other desktop you wish (matchbox-desktop, lsb-desktop etc), provided you installed it of course.
