You can easily switch icon sizes by doing the following things:

- Open Konqoeror and make sure that you are in Icon View. If you are not, main menu->View->View Mode->Icon View
- Main menu -> Icon size -> Choose preferred size

AFAIK konqueror remembers these settings automagically.

Don't know about your first question though. :|
