### Post by ThinkBuntu on 2007-06-25
I believe that controls icons are compatible, but not all window decorations. In Debian, Clearlooks wouldn't work in Xfce. However, Xfce includes some really great themes (esp. if you install the xfce-goodies package. Some good OSX emulation's in there), I think you'll have no problem setting up a lovely theme.
