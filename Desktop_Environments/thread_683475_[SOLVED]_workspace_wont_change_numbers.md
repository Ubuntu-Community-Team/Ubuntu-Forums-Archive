for some reason, no matter what i set the workspace numbers to be, its always on two columns, one row. i want to make it four columns and one row, any ideas on how to fix it?

---

### Post by mcduck on 2008-01-31
Are you using desktop efects (Compiz)? In that case install "compizconfig-settings-manager", then go to System/Preferences/Advanced Desktop Effects Settings. Next go to General Options, and in the Desktop Size-tab set the "Horizontal Virtual Size" to 4.
