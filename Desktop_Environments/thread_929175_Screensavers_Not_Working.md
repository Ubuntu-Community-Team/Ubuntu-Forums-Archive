### Post by Sub101 on 2008-09-24
There may be a simpler solution however:
If you go to "System"->"Administration"->"Synaptic Package Manager" and search for "gnome-screensaver" you can remove the package. Better still you can install xscreensaver (which is another type of screensaver system) instead.
Good Luck

EDIT: Obviously this will get rid of your screensaver if you don't install xscreensaver.
