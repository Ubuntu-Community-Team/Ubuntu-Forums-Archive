### Post by ajgreeny on 2011-10-06
If you mean the icons in the left side launcher-bar of the unity interface in 11.04, natty, you need to install compizconfig-settings-manager (ccsm) and use the "Unity" plugin in that, which allows you to change several things about the launchers.

Be careful not to change settings in some of the other plugins if you are not sure what you are doing, or you may stop unity working completely.
