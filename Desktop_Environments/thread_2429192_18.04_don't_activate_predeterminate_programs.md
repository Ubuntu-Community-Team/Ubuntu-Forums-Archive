### Post by Xian on 2019-10-14
This solution is based on the premise that you've already tried the system settings menu to choose your default applications and yet for some reason it does not cause your choice to persist when clicking the media file. There is a file in your home directory which can be edited that will result in a user-only override. 

With a text editor open the file at ~/.config/mimeapps.list

Scroll down to the [Default Applications] section.

Here you can set your preferred application to open the desired file type(s).
