### Post by ashdezign on 2008-02-11
It took me a while to get kubuntu set up the way I liked, which is mainly just using compiz fusion with emerald. The scale, cube and shift switcher features are the only way I switch between applications now. And the effects, especially the window wobble just mnakes my computer seem more "alive" or "organic" somehow. Sitting in front of this machine for 10 hours a day, it needs to have some life in it ;) !

Compiz kept crashing until I turned off the "disable icons on desktop" feature in desktop appearance settings. Once I enabled desktop icons compiz ran smoothly and with emerald running I did not have to look at kde's default window decorator. Good thing because compiz would have been a deal breaker for me.

For music, amarok is a great program. IMO superior to anything available for Gnome (though exaile is nice, it is still not as well developed as Amarok). Amarok also works with my Creative Zen Stone mp3 player and  as a native KDE application runs much faster, smoother and with better integration in KDE than in Gnome.

Ktorrent I find to be much more robust than bitTorrent and in fact while using Gnome I avoided bittorrent altogether and instead opted for Deluge.

Dolphin (file manager) is still a bit cludgy, but Konqueror is also available as a file manager.

Strigi desktop search is buggy in KDE, but I found tracker in Gnome to be wanting as well.

Open Office runs on both platforms

For video in Gnome and KDE I opt for VLC. In my experience the best video player available with native support for just about any video type and a nice small footprint.

Kopete seems on par with Pidgin, still too early to say which is better though.

I do miss revelation password manager though, its the best I have used but runs buggy on Kubuntu. All the KDE ones I have looked at so far don't import revelation files though so I am stuck with it for now.

Kmoney is to my limited use a better accounting program than is available for Gnome. very user friendly, by comparison.

I find the package managers available on Gnome to be easier to use than the ones on KDE but that could be as much the learning curve as any inherent flaws in the KDE ones.

One application I miss is tomboy. For note taking it was just about perfect. Knote is too light and Basket is to heavy on KDE, but Basket does integrate nicely into Kontact which makes up for it as Tomboy would only integrate with Evolution in Gnome and not Thunderbird.

With Gnome I was very happy when compared with Windows, with KDE I think the potential is there to make me even happier, but at the end of the day it comes down to personal preferences.

What I have done is installed a new drive in my PC and taken the old one with the Gnome install and mounted it as an external. That way If i need to switch back its as easy as taking out the kubuntu drive and installing the gnome one. 2 minutes work.

If you are curious about KDE I would say install it and give it a whirl. Linux is as much about exploring your options as having them in the first place.  You can install it as another DE on your Gnome install and just choose it at login or do like I have done and do a clean install.
