### Post by 3Miro on 2012-07-19
I am sure someone has a link to Unity guide, but if you are not happy with the way it works, you can install CompizConfig-Settings-Manager, disable the Unity Alt-Tab behavior and replace it with a more classic windows switcher. That's what I did, I didn't like the default behavior at all.
