### Post by snowpine on 2011-05-17
Welcome to the forums! All you need to do is simply:

```
sudo apt-get install fluxbox
```

Fluxbuntu was never an official branch of Ubuntu; it was an unofficial one-man project that never got past the "release candidate" phase.

If you are looking for a good "out of the box" Fluxbox experience then I highly recommend AntiX or Linux Mint Fluxbox Edition.
