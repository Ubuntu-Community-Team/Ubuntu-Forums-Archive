If you would like to try out Enlightenment E17 on your Edgy system it can easily be installed by following the guide I created.  You can find it here.  

[http://www.ubuntuforums.org/showthread.php?t=319336]("http://www.ubuntuforums.org/showthread.php?t=319336")

I have also been using Elive since it was in the 0.4 release.  I have 0.5 installed on an extra partition on my machine.  Elive is using a much older development version of E17 at this stage, and I think you will find E17 much easier to configure and use at its current development release you will get if you install it following my guide. 

Also, if you install Enlightenment from Ubuntu's standard repositories you will only get E16, which is a much older version of Enlightenment and a little outdated visually.  E16 is still being maintained though.
