You mean the window decorations(title bar and window borders)? Can't do it with standard gnome, but you can use the window automation tool devilspie.

I wrote a guide for it that is by now rather outdated (devilspie changed its configuration format, and I only go very briefly over how the new one works), but might still introduce you to how it works: [http://ubuntuforums.org/showthread.php?t=75749](http://ubuntuforums.org/showthread.php?t=75749)
