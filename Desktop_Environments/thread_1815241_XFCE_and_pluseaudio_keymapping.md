### Post by azertyh on 2011-07-31
hello,

see [this thread]("http://ubuntuforums.org/showthread.php?t=987149") for pulseaudio commands for volume up, down and mute.

and in xfce, settings manager > keyboard > application shortcuts > add, type the command in 8th or 9th post (for example volume up) in the thread above. on the second window, press the key (for example f3) that you want to allow to volume up. 

repeat these steps for volume down and volume mute.
