### Post by zerohero on 2009-05-12
fair and well, but was looking for keyboard shortcut (OK, could do alt-space, w, X, but..). But found the solution myself: Enable Put, Under put to arbitrary viewport set up shortcuts. This make it behave as move-and-follow. For only move, check "unfocus window" under misc. options.
