### Post by LIB53 on 2009-04-11
Are you talking about the default selected item color? Because if that's what it is you can change that quite simply. I kept that brown so it fits in with the whole Ubuntu palette. You can change that by

1) going to the appearance window (right click desktop> "Change Desktop _B_ackground". 
2)Click the theme tab.
3)Click the customize button
4)And change the color under the colors tab
