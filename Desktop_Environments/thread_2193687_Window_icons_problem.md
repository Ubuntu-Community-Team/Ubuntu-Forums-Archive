### Post by Copper Bezel on 2013-12-14
I'm only seeing one image, but what theme are you using? Controls for maximized windows in Unity are defined in a separate part of the theme, for reasons. Also, some themes won't provide theming for them at all, if they're not intended for Unity. Last, this'll happen if your GTK and Metacity themes aren't the same,  because the Unity theming follows the GTK, not the window theme.
