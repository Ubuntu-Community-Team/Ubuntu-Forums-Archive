### Post by pndragon on 2008-10-08
This really irritated me also until I figured it out.

Right-click on your plasma-desktop and select: 

*Add Widget>Folder View*

This will add a folder view of your /home folder with a transparent background. If you then click the wrench (Folder View Settings) you will get the option to create a desktop view. This desktop view behaves just like the old desktop with the ability to create folders and application launchers.

--- Jim
