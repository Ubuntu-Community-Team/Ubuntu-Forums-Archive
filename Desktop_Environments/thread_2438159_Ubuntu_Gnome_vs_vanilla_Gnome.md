### Post by deadflowr on 2020-03-06
Different set of dependencies allow you to control things like whether or not the dock is installed.
An example, in ubuntu-desktop you must have ubuntu dock installed.
In vanilla it doesn't matter.
So it can be better if you want to build a gnome desktop as you see fit.
For the most part.


(I should mention that even though I say ubuntu-desktop and you posted about ubuntu-gnome, the ubuntu-desktop meta package is a dependency of ubuntu-gnome-desktop,
so same applies to both)
