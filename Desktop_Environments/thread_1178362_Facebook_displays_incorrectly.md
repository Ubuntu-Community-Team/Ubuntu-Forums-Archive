### Post by krazyd on 2009-06-10
If Ctrl + '+' works temporarily, then it's just the browser zoom that's the issue. Firefox should remember the setting on a per-site basis. This sometimes doesn't stick if you have a couple of different tabs open with different zoom settings.

Try this:
[LIST]
[*]Open facebook.com, making sure it's the only page open (no other tabs)
[*]press Ctrl + 0 (that's a zero) to set zoom to default level. Does facebook look ok?
[*]Close firefox completely, then open it up again and go to facebook.com. Does it still look ok?
[/LIST]
