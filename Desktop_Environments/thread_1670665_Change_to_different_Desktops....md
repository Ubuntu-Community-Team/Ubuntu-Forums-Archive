### Post by cascade9 on 2011-01-19
You can either install xubuntu-desktop or xfce (or kubuntu-desktop or kde-full).

I'd install kde-full or xfce4 over kubuntu-desktop or xubuntu-desktop. Less programs to install, no changes to the splash/login screen (if you change to xxx-desktop you will get the splash screen and login screen for that system). 

easiest way is from the command line- 

sudo apt-get install xfce4 

or 

sudo apt-get install kde-full 

Then you can choose the desktop enviroment you want to use from the login screen. ;)
