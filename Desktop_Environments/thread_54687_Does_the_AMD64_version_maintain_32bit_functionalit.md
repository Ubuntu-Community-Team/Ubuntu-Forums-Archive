### Post by charlieg on 2005-08-05
A lot of apps don't run on 64bit because the developers (usually inadvertantly) made assumptions about the sizes are various values that change on a 64bit system (thing likes the size of longs and floats).  However, that's more bad coding than 64bit backwards compatability.  Assuming robust code, your 32bit apps should work fine an a 64bit system.
