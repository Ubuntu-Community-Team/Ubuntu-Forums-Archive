### Post by PaulW2U on 2018-05-06
> **martago said:**
> Two weeks after the official release of version 18.04, should not this  release upgrade be available through the described ways?
In "Software & Updates" are you set to be notified of any new version or of an LTS. If set for LTS notifications then I don't think you'll be notified until 18.04.1 is released in July. Notifications for any new version were enabled a couple of days ago, something that I have seen for myself.

**Edit**: As an afterthought what version are you running now? If it is 16.04 then you new version may well be 17.10 and not 18.04.
