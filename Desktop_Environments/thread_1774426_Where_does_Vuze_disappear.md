### Post by Shay Guy on 2011-06-03
Unity doesn't appear to have an analog to the taskbar button Vuze uses. And once you close the window, the program disappears from the dock. But this always works for me when I need to reopen the window:

[FONT="Courier New"]alt-f2[/FONT]
[FONT="Courier New"]vuze/azureus[/FONT]

I don't know if the path is different for different versions, but on mine, the [FONT="Courier New"]azureus[/FONT] executable is in the [FONT="Courier New"]vuze[/FONT] folder in my home directory ([FONT="Courier New"]~[/FONT]). If yours is the same, the above command should work. If not, adjust accordingly.
