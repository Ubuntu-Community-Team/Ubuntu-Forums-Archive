### Post by bukwirm on 2006-06-25
Linux is based on Unix. For a console in GNOME, Applications > Accessories > Terminal, or press Ctrl-Alt-F1 though F6, and Ctrl-Alt-F7 to get back to desktop. If you're new to Linux, I would suggest that you find a book on linux and read though it - it will help a lot!

For Java, seach for 'Sun Java' in Synaptic (System > Administration > Synaptic), scoll down to find the sun java pacjages, and install them. There is a thread about IDEs in the Programming section of these forums.
