### Post by n3had on 2009-05-11
HI all i just made a tutorial for adding some cool themes to your system

check it out

```
sudo gedit /etc/apt/sources.list

deb http://ppa.launchpad.net/bisigi/ppa/ubuntu jaunty main
deb-src http://ppa.launchpad.net/bisigi/ppa/ubuntu jaunty main

sudo apt-key adv --recv-keys --keyserver keyserver.ubuntu.com 0x1781bd45c4c3275a34bb6aec6e871c4a881574de

sudo aptitude update

sudo aptitude install zgegblog-themes
```

Video tutorial here: [http://www.youtube.com/watch?v=GCOBhUpV9XA](http://www.youtube.com/watch?v=GCOBhUpV9XA)
