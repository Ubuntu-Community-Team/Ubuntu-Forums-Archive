### Post by ibutho on 2008-06-19
The file you downloaded is already prebuilt, so if you go into the directory you extracted and click on the icon labelled Firefox, Firefox 3 will run. If there are other people using your computer, you maybe better of extracting the package into /opt. You will also need to edit your launchers (short cuts) to point to /opt/firefox/firefox or wherever the firefox executable is located.
