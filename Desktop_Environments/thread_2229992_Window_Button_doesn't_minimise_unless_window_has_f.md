### Post by dave0109 on 2014-06-16
I'm sure this is new behaviour but on Xubuntu 14.04, if I click on a Window Button (i.e. the icon & text within the top panel that represents the program), then the program's window will only minimise if it has focus.  This is compounded by the issue that when the window is unminimised it doesn't have the focus, so I can't quickly unminimise something, then click again to minimise it once more.

I'm not sure which particular bit of behaviour has changed, but certainly it used to be possible to click to unminimise and click again to minimise.

In my Window Manager settings, I have 'Focus follows mouse', 'Automatically give focus to newly created windows" and "Raise window when clicking inside application window" all checked.  These are the same settings that I had for (what feels like) forever.
