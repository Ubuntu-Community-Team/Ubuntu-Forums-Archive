### Post by jpeddicord on 2007-11-06
Check your programs in System > Preferences > Sessions and see if you have any "bad" programs there. Try unchecking them one by one and restarting and see if you can narrow down the culprit. Also, note that if you are running Compiz, there used to be some issues with it hanging the GNOME session upon startup, though I am not sure if it still is a problem. As a failsafe, disable Compiz also to see if that helps.
