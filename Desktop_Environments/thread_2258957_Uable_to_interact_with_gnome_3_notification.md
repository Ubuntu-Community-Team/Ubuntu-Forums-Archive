### Post by buzzingrobot on 2015-01-01
"Super-M" toggles the display of the message area in Gnome, if that's what you're looking for. Not all applications seem to have adapted to working with this message tray. (Also, try a right-click. too.)

The 'TopIcons" extension (visit extensions.gnome.org) will relocate most of these icons to the top panel.
