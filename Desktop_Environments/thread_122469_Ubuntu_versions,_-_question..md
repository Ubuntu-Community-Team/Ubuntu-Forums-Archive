### Post by jasmuz on 2006-01-27
What makes them different is exactly the version number and release date (wich are the same), at that point the best software is handpicked and squeezed into the distribution.

So a Warty (4.10), is older than a Hoary (5.04), Breezy is the latest (5.10) and will be replaced by Dapper (6.04), if you havent figured it out, the release cycle is every 6 months.

### Post by byen on 2006-01-27
yes. Jasmuz pretty much nailed it! Many applications in linux usually have short life-cycles before they come out with newer versions. In the very same fashion... Ubuntu too has a life cycle of 6 months where the necessary stable software are updated (/upgraded) with every release...

Dapper (6.04) - 2006, April
