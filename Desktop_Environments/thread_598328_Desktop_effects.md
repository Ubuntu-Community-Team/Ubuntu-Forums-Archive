### Post by alaskazimm on 2007-10-31
It's not under System > Preferences.

Right click on the desktop and choose Change Desktop Background. The far right tab of the window that comes up is Visual Effects.

There are only three choices there, so to do any tweaking to the effects you still need to install Compiz from Synaptic.
