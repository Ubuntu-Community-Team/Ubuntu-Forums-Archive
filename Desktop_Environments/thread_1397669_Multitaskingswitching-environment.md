### Post by MichaelRX8 on 2010-02-03
Are you talking about the windows panning out and scaling down? If so that's probably my favorite compiz plugin. Its called scale and scale add ons. You can enable them in the compiz settings manager. I don't remember what the default key binding is but I just set mine to scale all windows on a given desktop by moving the mouse pointer to the top right corner, I have it scale all windows from all desktops by moving the cursor to the bottom right corner.

 Hope that helps.
