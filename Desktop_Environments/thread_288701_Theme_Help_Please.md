### Post by CatKiller on 2006-10-30
> **_simon_ said:**
> I think you maybe misunderstood me.

I'm talking about that block of grey.

I'm definitely misunderstanding you still, then. The block of grey **is** the Notification Area. The other block of grey **is** the Window List. If they didn't have a visible handle, how would you move them? How would you modify them? It makes no sense to have them completely invisible by default. The style of them is changed per "Controls" theme, so you may want to look at how different themes change the appearance of them if no one turns up that already knows.

EDIT: From poking around in my ~/.theme folder, it appears you **may** be able to change these handles by changing the handle PNG files for the theme you're using in the ~/.theme/<themename>/gtk-2.0 folder. Might be worth still searching for a proper tutorial though.

EDIT 2: Once you've established how to do it, [this guy]("http://ubuntuforums.org/showthread.php?t=285142") will probably be interested in your results.
