### Post by AtrejuT on 2007-03-28
You'll need to add the Window List applet to the top panel (right click panel, select add, select 'window list' click add.
then you have it in both. Now you can remove it from the bottom panel by right-clickint the left edge ot the window list in the bottom panel and select remove.
