### Post by dlenmn on 2009-10-10
I believe this will work:

System Settigns -> Advanced tab -> Session Manager -> "Start with an empty session" radio button under "On Login"

I think it's nice to be able to start up just where I left off, but to each his own. Under "On Login" you can see that you have some options, like excluding certain applications from starting up on login, but allowing others.
