### Post by benerivo on 2007-08-28
You could just make a new user account and log in to it, to see if it loads the default desktop. If it does, then you can delete your original one. The downside to this is that you would lose all your individual program settings (such as Firefox bookmarks, etc.)  and personal files in your /home/YOURNAME directory once you've deleted it.
