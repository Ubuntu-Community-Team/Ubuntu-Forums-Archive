### Post by siphiuel on 2011-05-20
Hi all!

This is the first time i'm posting to Ubuntu forums, so greetings to all :)

I've been using Ubuntu on my laptop for some time now, starting from 9.10 version i think. Currently i'm using 11.04 version, and it is all good. And in terms of screen fonts, it's getting better really, and default antialiasing look much better than ClearType in Windows 7 - it's sharper and does not hurt eyes that much.

Still, i prefer clear non-antialiased fonts. So i've made some changes in setup - turned of antialiasing in Appearance window, set the hinting to 'Slight', and also created ~/.fonts.conf file, where antialiasing is also off, and hinting is 'Slight', so that Firefox understands those settings too.

But the issue is, what font set is the best to use for small clear non-antialiased symbols? Most of the time i do coding/Latex editing/web browsing, and smaller fonts mean more text on screen, and the more readable, the better.

In Appearance settings i've left the default Ubuntu font, because it looks ok even without antialiasing, and for monospace Liberation Mono is nice (though Terminus perhaps is even better).

And as for ~/.fonts.conf and Firefox default fonts, i've set all to Liberation family fonts, partly because Ubuntu fonts don't have serif version, while Liberation fonts do.

So finally the question is - what fonts do you prefer/recommend? I know that MS core fonts (Tahoma, Verdana, Times New Roman) are nice, and i did install those on previous Ubuntu versions, but i'm sure there are better free alternatives.

Looking forward to your suggestions/experiences.

Many thanks,
Vitaliy.
