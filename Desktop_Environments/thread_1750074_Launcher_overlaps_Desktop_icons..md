### Post by babylonian13 on 2011-05-05
Never mind, I fixed it.
It seems that it was the fault of nautilus-elementary.
I removed the nautilus-elementary and it's PPA repository and installed the full nautilus.
I had some problems with "Could not update .ICEauthority file" but were fixed simply with "sudo apt-get install gnome-session".
