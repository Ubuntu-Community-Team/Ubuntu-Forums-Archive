You may concidering compiling the application instead due to the fact that installing .rpm with help from alien is not a guarentee that it works or worser 'may mess up something'.[/QUOTE]

Agreed. The best bet would be to utilize checkinstall. Download the source. Extract these to your home dir. Then enter the folder, in the terminal.

```

sudo apt-get update
sudo apt-get install checkinstall
./configure --prefix=/usr
make
sudo checkinstall

```
most all of the options can be accepted as default when checkinstall runs. This will then create and install a deb for you. If this reports any errors, it is usually due to missing libs.
