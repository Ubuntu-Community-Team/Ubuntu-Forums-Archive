### Post by Jucato on 2006-02-17
No need to extract. If you installed it through a tarball in the "Themes" tab, just click on "Set Theme Path's" and it will automatically set the path. Actually, it automatically extracts the theme to ~/.kde/share/apss/dekorator directory (if you want to check).

I'm not sure if all deKorator themes can be installed this way. I think themes for earlier than deKorator 0.2 must be extracted and set manually. Not entirely sure.
