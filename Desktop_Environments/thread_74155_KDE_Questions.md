### Post by Manny C on 2005-10-11
Right click the panel. Select Configure Panel. Go to Menus Tab. Ensure "Menu Item Format" contains Description (ie. select either Name (Description) or Description (Name)). Click on KMenu and read through Menu items Description.

I believe that most descriptions would be self-explanatory.
