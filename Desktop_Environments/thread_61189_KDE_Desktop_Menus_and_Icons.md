### Post by shakin on 2005-08-31
If you right-click on the K button you can select 'Menu Editor' to edit the menu and add what shortcuts you want. You can also right-click on a panel and add any app to the panel.

If you want something run in superuser mode you would prefix 'kdesu' in front of the application executable. For instance, if you have created a panel shortut to konqueror, edit that shortcut to be 'kdesu konqueror'. Same thing with 'kdesu konsole' for a superuser terminal.

### Post by Lord Illidan on 2005-08-31
CDs should be mounted automatically in KDE, and even in Gnome, but in Kubuntu, they might not show under the Desktop..

Going to /media with Konqueror will locate any mounted CDs.. to unmount, just rightclick on the icons and press unmount, or eject.
