```
sudo apt-add-repository ppa:gurqn/systray-trusty
sudo apt-get update
sudo apt-get upgrade



```

That worked great and my programs are showing up in the tray.  Now my question is, how to disable the new tray and go back to the way it was before?  I can't find that answer anywhere.

### Post by ajgreeny on 2014-07-07
The easy way is to install ppa-purge and run that.

Or simply go to software-sources --> Other software tab and disable the ppa, refresh the repos and then re-install whichever package(s) installed by the ppa.

However, as the ppa seems to have done what you want, why do you now want to go back to the way things were?
