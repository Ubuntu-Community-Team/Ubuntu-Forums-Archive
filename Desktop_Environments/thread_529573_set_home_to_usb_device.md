Perhaps.... but I would test this on a computer you don't need or don't care about first, as this could really screw things up. Basically, you would have to use this guide: [http://www.psychocats.net/ubuntu/separatehome](http://www.psychocats.net/ubuntu/separatehome) . 

You don't need the stuff about partitioning, all you need is the part at the bottom that details how to move it all over. It could work, it could not work. Testing is the only way to find out. 

Alternatively, you could use this guide: [http://xubuntublog.wordpress.com/2007/06/17/ubuntu-feisty-on-your-usb-drive-finally/](http://xubuntublog.wordpress.com/2007/06/17/ubuntu-feisty-on-your-usb-drive-finally/)

Hope this helps!
