### Post by HTML-insane on 2009-11-21
In many cases, it's as simple as typing the URL into the address bar - typing an FTP URL into Dolphin's address bar will open it like any other folder, and shared network folders can be found in the Network sidebar section. Konqueror can also connect to FTP URLs as normal, using the same UI as Dolphin for file-management.
