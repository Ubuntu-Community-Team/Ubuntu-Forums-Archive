### Post by irielion on 2008-06-26
Hi,
I would like to announce intipunku photo manager 0.3.
The new release i feature rich. Here is some of the things you can do:
- photo adjustments (brightness, contrast)
- auto-tools (auto-contrast, sepia, etc)
- crop
- blue sky
- red eyes removal
- printing
- picasaweb uploading/importing
- camera import
- more

translations: dutch, catalan, spanish, swedish, italian, german, french and russian

I attached some screenshots.
