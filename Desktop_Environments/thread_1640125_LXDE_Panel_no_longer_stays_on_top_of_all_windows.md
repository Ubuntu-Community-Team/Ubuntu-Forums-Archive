### Post by cavalier911 on 2010-12-07
Under panel preferences properties I have treat panel as a dock checked and reserve space,not covered by maximized windows checked. If this is ok I would logon on another regular user account to see if it just affects one account or global problem.If it only affects that one account move that users configuration files for lxpanel to another location.Logout/in and the configuration files will be regenerated by lxpanel hopefully fixing the problem. You will end up with a default panel,customizations will have to be reconfigured.

```
mv ~/.config/lxpanel/Lubuntu/panels/panel ~/
```
