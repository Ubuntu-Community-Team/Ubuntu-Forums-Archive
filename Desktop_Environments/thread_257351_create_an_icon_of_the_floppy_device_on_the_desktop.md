### Post by theturtlemoves on 2006-09-14
I don't have a floppy drive so I'm not sure if this will work, but try this: Add the Disk Applet to the panel, and every time you have a floppy in there, click the floppy icon and click "Mount disk". Then an icon for the disk should appear on the desktop.

Otherwise, right click the desktop and select "create launcher". The change the launcher type to Link and make the link to /dev/fda1. I don't think this will work, but it's worth a shot.
