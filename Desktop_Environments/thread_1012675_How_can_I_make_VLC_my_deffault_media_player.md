### Post by charlescarroll1 on 2008-12-16
I would like to make VLC my default media player instead of Totem.  I assumed I could do this in System > Preferences > Preferred Applications, but there is not option given.  Does anyone know how to do this?

---

### Post by pholden on 2008-12-16
One way is to right-click on a media file, click *Properties > Open With* and select the application you want to use. This will set the application as the default for the file type you selected.
