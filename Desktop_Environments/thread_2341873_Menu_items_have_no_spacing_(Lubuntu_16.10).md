### Post by Dennis N on 2016-11-01
I see this spacing problem with Lubuntu's pdf viewer. I changed the theme from Lubuntu Default to Clearlooks, and that fixed the spacing of the menu titles. So it is a theme problem.

Correction: the problem is in Lubuntu Dark Panel, not Lubuntu default. So you just can use Lubuntu default theme and avoid the problem.

---

### Post by fgg2 on 2016-11-01
That did it! Changing Lubuntu-dark-panel to Lubuntu-default worked! Thanks!
