### Post by twrock on 2006-11-04
I had the same initial "problem" when installing compiz according to that guide. However, simply going into the emerald theme manager and choosing a new theme solved it (in System-Preferences-Emerald Theme Manager or the Beryl Manager depending on what you installed).
