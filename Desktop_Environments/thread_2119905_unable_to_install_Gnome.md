### Post by grahammechanical on 2013-02-25
Itb is Ubuntu's way of telling us that we take responsibility for installing that software.

> You download and install PPA packages at your own risk. Ubuntu, Launchpad and Canonical do not endorse these packages. You must be certain that you trust the PPA owner before you install their software.

[https://help.ubuntu.com/10.04/add-applications/C/adding-repos.html]("https://help.ubuntu.com/10.04/add-applications/C/adding-repos.html")

If you want that software you tick the Details link to see what is going to be downloaded. You then choose to click OK or Repair. Why are you getting an option to 'Repair?' Do you have some other problem with downloading or updating? You may have broken packages. Are you trying to re-install something that you removed? Did its removal remove more than you wanted it to. That can sometimes happen.

Oh, one more thing. Why do you want to install Gnome Desktop Environment? If you are using standard Ubuntu with Unity, then Gnome DE should already be installed. Ubuntu Unity sits on top of Gnome DE. The other flavours of Ubuntu have different desktop environments. But Ubuntu has Gnome DE 3. May be you only want to install Gnome shell 3.
