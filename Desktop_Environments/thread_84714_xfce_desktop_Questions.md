### Post by psychicdragon on 2005-10-31
The desktop menu is an xml file located at **~/.config/xfce4/desktop/menu.xml**. You can use the **xfce4-menueditor** app to modify it.

Nautilus stores deleted files in **~/.Trash** I believe.
