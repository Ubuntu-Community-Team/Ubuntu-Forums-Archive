### Post by JAwuku on 2006-02-26
Also is the port of the Really Slick Screensavers, called **rss-glx**.

To install it, enter ```
sudo apt-get install rss-glx
```

To make the newly installed screensavers show up in xscreensaver, type the command ```
/usr/bin/rss-glx_install.pl
```
