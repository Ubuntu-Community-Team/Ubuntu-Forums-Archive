### Post by hariks0 on 2010-12-21
My latest ccsm profile uses both **Add Helper** and **Opacify** Plugin of Compiz. The former will make all unfocused windows transparent until clicked again whereas the latter temporarily makes the blocking [foremost] windows transparent if mouse is hovered over it. If clicked on some part of desktop, the desktop will be made visible through the stack of the transparent windows using the **Add Helper** addon.

Just try the compiz profile attached herewith. Hope the screenshots help.
