### Post by BoardStupid on 2009-01-19
[http://www.kubuntu.org/](http://www.kubuntu.org/) has announced that KDE4.14 is compiling in the proposed updates section. To install it enable Pre-released Updates in Adept, then under Changes click on Upgrade and Apply.


You can also add ```
deb http://ppa.launchpad.net/kubuntu-experimental/ubuntu intrepid main
``` to your repositories list and use KDE4.2 RC1 right now. I did last week and I am enjoying the updated version. Be aware that it is still classed as beta software though.

[http://www.kubuntu.org/news/kde-4.2-rc1](http://www.kubuntu.org/news/kde-4.2-rc1). Scroll down to instructions for full details
