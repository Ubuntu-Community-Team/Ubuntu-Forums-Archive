### Post by raziel09 on 2013-02-21
Which version of ubuntu are you currently running? 

If running 12.04 you can install the gnome session fallback which is similar to the 10.04/10.10 by running the following command.

```

sudo apt-get install gnome-session-fallback

```Then log out and choose either gnome classic or gnome classic (no effects).
