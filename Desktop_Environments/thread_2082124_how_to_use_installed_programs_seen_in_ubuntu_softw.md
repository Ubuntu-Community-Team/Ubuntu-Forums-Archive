### Post by grahammechanical on 2012-11-08
Open the Dash and use the applications lens. Super key (windows) + A. Click Installed see xx more results. From the icons click the application you want to use. It will load and put an icon in the Launcher. Right click the icon and select Lock to Launcher.

You should also read the Ubuntu Desktop User guide. It is called Help. You can search for it in the Dash. By the way, those icons you see in the Dash can be dragged on to the Launcher and they will stay there until you right click and select Unlock from Launcher.

Regards.

### Post by jerome1232 on 2012-11-08
Also as opposed to poking through that filter, you can start typing, for example I want to edit a photo so I start typing photo and both gimp and shotwell show up quickly.

Anything you access frequently drag out to your launcher, other stuff should be a very quick search away.
Both the HUD and the Dash learn as you use it, for example the HUD remembers that last time I was in Tomboy notes and i typed "new" I wanted to create a new note, as soon as I hit "n" it's already showing that option I want.

Press and hold the super key (windows key) to get a list of keyboard shortcuts to make life easy.
