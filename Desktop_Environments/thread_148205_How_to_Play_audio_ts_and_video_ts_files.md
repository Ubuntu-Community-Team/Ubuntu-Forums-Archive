### Post by danrom on 2006-03-21
xine /media/cdrom1/video_ts/*file*.ts

You might get an error asking you if you want to continue. If this comes up hit yes. I recommend you install xvid and possibly dvd/vcd player.

### Post by nalmeth on 2006-03-21
if you have libdvdcss installed, you will be able to play the files (video_ts) with most any media player (totem/mplayer/xine) just open up the mounted drive and nautilus should show you previews for those files.
