### Post by Hanj on 2006-04-14
When I use Gnome I always set my shortcuts so that alt+1 switches to desktop 1, alt+2 to desktop 2 and so on. I've gotten quite used to this, and would like to have the same shortcuts in KDE. However, I can only find shortcuts to switch to next/previous desktop. Is there any way to set things up in KDE so that alt-number switches to that particular desktop?

---

### Post by gingermark on 2006-04-14
This is set by default to ctrl+ the F key (so ctrl+F1 for Desktop one).

Do K-Menu --> System Settings --> Regional & Accessibility --> Keyboard Shortcuts.
Click on the 'Shortcut Sequences' tab under 'Shortcut Schemes'.
Scroll down to 'Desktop Switching". Here you should be able to configure the behaviour as you'd like (by clicking "custom" for each shortcut and entering what you'd like it to be).

Good luck.

### Post by Ensnared on 2006-04-14
What you're looking for can be found in the Control Center under Regional and Accessibility -> Keyboard Shortcuts. In that configuration-screen you have three tabs under the "Shortcut Schemes" tab, named "Global Shortcuts", "Shortcut Sequences" and "Application Shortcuts". Go to the "Shortcut Sequences" tab and scroll down, and you'll find the entries for "Switch to Desktop 1", "Switch to Desktop 2", etc...

Just select one of them, click "Custom", and enter the key-sequences you want to use.
