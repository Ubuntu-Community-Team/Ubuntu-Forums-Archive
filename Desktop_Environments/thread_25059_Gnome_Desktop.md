### Post by astoltz on 2005-04-09
If you want to make the trash / computer / home icons show up on the desktop, click on Applications -> System Tools -> Configuration Editor.  Then, in the Config Editor go to apps -> nautilus -> desktop.  There are check boxes to show the different icons which are hidden by default in Ubuntu.
