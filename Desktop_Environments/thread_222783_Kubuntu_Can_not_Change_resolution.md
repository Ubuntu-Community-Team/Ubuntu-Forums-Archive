### Post by thegnark on 2006-07-25
you can try forcing xorg.conf to only use your resolution by removing referances to existing resolutions

MAKE A BACKUP FIRST!
then, remove everything except the specified resolution and refresh rate that you want

you probably want to cange the "modes" line to look something like this:
```
 modes "1152x864@75"
``` 
if that doesn't work, try removing references to the unneccessary modelines. also, i see a reference to a larger virtual desktop that you (probably) don't want
