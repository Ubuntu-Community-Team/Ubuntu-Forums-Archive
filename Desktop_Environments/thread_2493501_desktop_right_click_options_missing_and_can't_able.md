### Post by MAFoElffen on 2023-12-17
First: Only those 3 options "are" the default options if your right click on the desktop. That is normal.

That behavior changes if you are in an application, based on what is added within that application. What your described, in what you expected is more like what you would see inside of Nautilus...

The default behavior of the desktop is not to show desktop shortcuts, unless you override that with what 'Tea For One' posted with adding that Gnome Extension. There is also Dash-To-Panel, where it would change a lot more things, as well as being able to customize right-click actions.
