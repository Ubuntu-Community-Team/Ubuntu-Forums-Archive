What do you want to use it for?

Do you want to use it to bring up the root menu in Openbox? Or do you have more ambitious ideas? Knowing more about what you want to achieve might help finding a solution.

Keep in mind, though, that the Windows key is a 'modifier' key in Openbox, which means you can't use it on its own, but only in combination with another key.
