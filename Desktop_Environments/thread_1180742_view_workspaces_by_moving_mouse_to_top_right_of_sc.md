### Post by sigixv on 2009-06-07
enable desktop effects first. In compiz settings you can set "hot-corners" and actions to be executed when hitting a corner or screen border. I think the effect you are looking for is exposé (can be set in compiz)

Perhaps you have to install the simple compizconfig settings manager (SCCSM)
```
sudo aptitude install simple-ccsm
```

It will then be located in System>Preferences> Simple CompizConfig Settings Manager
Setting the borders is in the last tab
