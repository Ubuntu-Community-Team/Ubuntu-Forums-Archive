### Post by Ramses on 2005-08-28
If you have kmymoney2-0.8-1.kubuntu5.04.i386.deb file, you can install it in command line (Konsole).  

```
sudo dpkg -i kmymoney2-0.8-1.kubuntu5.04.i386.deb
```
