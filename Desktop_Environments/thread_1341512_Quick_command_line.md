It was to bring your desktop back. Actually, if you are in a TTY, ALT+F7 is enough.

The GNOME terminal is in the Applications menu. If you want to open it via a keybind, you need to configure it yourself (I don't know how, not a GNOME user).
Or ALT+F2 and execute ```
gnome-terminal
```
