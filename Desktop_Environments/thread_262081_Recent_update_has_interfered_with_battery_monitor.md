A recent update seems to have interfered with the battery monitor in the system tray (Xfce and Gnome).  In Xfce it does not display at all when my laptop is on battery, in Gnome it says the battery is 0% charged (even when plugged in to A/C).

When I unplug the laptop, the screen dims, so it recognizes that it's not on A/C power.

UPDATE: this was fixed (in Dapper) fixed by installing the latest version of Xfce plugins.  Since upgrading to Edgy the issue appears to be moot.
