### Post by Copper Bezel on 2011-06-08
Yeah, here, check out [this guide]("http://www.webupd8.org/2011/04/things-to-tweak-fix-after-installing.html"):

```
sudo apt-get remove overlay-scrollbar liboverlay-scrollbar-0.1-0
```
