### Post by ltmon on 2005-12-31
Most (if not all) DVD drives have the region code set in firmware.  This shouldn't be a problem on Linux DVD playing software (Xine, Gstreamer, mplayer etc.) can pretty much ignore the region code.

But if you want to play around, try using the "regionset" program: [http://linvdr.org/projects/regionset/](http://linvdr.org/projects/regionset/)
