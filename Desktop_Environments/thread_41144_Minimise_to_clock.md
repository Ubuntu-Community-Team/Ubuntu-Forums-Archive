### Post by anatole on 2005-06-12
there is an app called alltray [http://alltray.sourceforge.net/](http://alltray.sourceforge.net/) - install it with typing "sudo aptitude install alltray" in the terminal 
you can start applications trayed by starting with the command "alltray <application>".
for example, i launch "alltray -s thunderbird" which launchs tb trayed, but the -s switch does not hide it instantly. you can set these parameters also to the system>preferences>preferred applications. 
what i don't know how to do, is to remove the "(AllTray)" text from the titles...
