### Post by Ramses de Norre on 2006-10-04
As far as I know AIGLX is a more stable, less hacky and more reliable method but it doesn't work with most graphic cards' drivers yet.
So most people are forced to use XGL untill new drivers are developped.

XGL works quite good too though ;) But if you're able to use AIGLX I'd go for that.
