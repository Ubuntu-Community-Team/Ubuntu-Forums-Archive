### Post by philipacamaniac on 2005-05-25
The "missing" features you refer to can be accessed via the Control Center.

1. Go to KDE Control Center --> Desktop --> Panels
2. Select the Menus tab
3. Select the K-Menu features you want


You might be able to set the K-Menu to KDE defaults rather than Kubuntu defaults by removing the kubuntu-default-settings, but since that is just a meta package, it may not make a difference.
