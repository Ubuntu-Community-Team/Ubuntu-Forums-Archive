### Post by merlinus on 2007-09-08
You only need one version of each app, no matter what de or wm you choose.

The terminal commands are mostly the same.  ntfs-3g will work on all.

The main differences lie in the desktop appearance and menus, and of course they can be highly customized.

I run gdm, xfce (xubuntu) and fluxbox, all using mostly the same apps, if I so desire, with only minor differences (e.g. gedit or mousepad, nautilus or thunar).

Lots of choices, unlike life in Gatesville.

Have fun!
