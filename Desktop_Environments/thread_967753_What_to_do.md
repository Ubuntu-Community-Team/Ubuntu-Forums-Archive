### Post by ibutho on 2008-11-02
Hi. 

I think Photoshop CS2 is well supported by WINE (Google sponsored developers to get it working well), so you can use that if its an option. Photoshop CS3 does not have the same type of support as Photoshop CS2, so you can either run it natively in Windows via dual boot or if your system can handle it, you can run Windows in a virtual machine.
