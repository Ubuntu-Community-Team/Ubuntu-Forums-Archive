### Post by el000 on 2005-05-05
You can refresh gnome-panel with:

killall gnome-panel

in order to see if the app will be automatically added to the menu. If not you can install the menu editor. [http://ubuntuguide.org/#menu-editor](http://ubuntuguide.org/#menu-editor)
