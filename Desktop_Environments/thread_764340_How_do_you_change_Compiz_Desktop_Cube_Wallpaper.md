### Post by LeoSolaris on 2008-04-23
You can set it through the Compiz manager, it's under Advanced Desktop->Desktop Cube->Skydome

Skydome is what that background cube image is called. (You can even animate it!)

Leo

### Post by wannjanjic on 2008-04-23
Install compizconfig-settings-manager. Open up the terminal, and type this into it:
```
sudo apt-get install compizconfig-settings-manager
```
After the installation, go to System => Preferences => CompizConfig Setttings Manager => Desktop Cube => Appearance => Skydome, and browse for the image you want.
Hope this helps. Cheerz!
