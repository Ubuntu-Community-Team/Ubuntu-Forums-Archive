### Post by bschelst on 2010-08-05
Silince is session means that a konsole most probably sends a 'bell' to the system is nothing happens  in the shell .
This is very helpful if you are running a script, and you want to be alerted when it's finished.

If you go to the konsole menu, you can enable/disable that for a specific session
