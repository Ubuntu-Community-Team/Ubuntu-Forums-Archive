### Post by timbosa on 2006-02-16
> 1.Are these two sessions totally separate from each other or is the second one actually running “on top of” the first one? Any way that one could cause problems with the other?
Yes they are independant. I don't think it would cause any major problems.

> 2.If you have multiple sessions open, is there any way to switch between them?
CTRL+ALT+(F7, F8, F9...etc.) will switch between sessions.
