My Environment: Lucid Lynx on HP DV7T 
Occasionally, the desktop stops accepting keyboard input. The mouse still works.

Circumvention Incantation:
Press <alt><ctrl><F6> to get to a character mode console.
Press <alt><ctrl><F7> to get back to the graphical desktop.

There is nothing special about F6. F1..F5 work just as well.
