### Post by sikander3786 on 2010-10-19
Install the compiz-config-settings-manager (if already not done that).

```
sudo apt-get install compizconfig-settings-manager
```

Install the extra plugins (that too if not already installed).

```
sudo apt-get install compiz-fusion-plugins-extra
```

Navigate to System > Preferences > Compizconfig Settings Manager. Enable 'Animations' and 'Animations Add-on'.

Now go to 'Animations' and then, Minimize Tab. Double click the effect listed there and select 'Magic Lamp' instead.
