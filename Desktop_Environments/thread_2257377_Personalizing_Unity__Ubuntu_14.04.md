### Post by deadflowr on 2014-12-19
It's quite literal.
It's stating exactly where you are.
The reason it says Ubuntu Desktop is because that is where it is in the Home Folder.
Ubuntu has the File manager handle the desktop, so the Folder in Home for Desktop is what you see.

Try this for fun, right click and create a folder.
Now open your file manager(12.04 called home folder, 14.04/14.10 called Files) and navigate to your Desktop folder.
You should now see the newly created folder in that directory.
If you move that to trash, when you go back to the Main Window area, that newly created folder will be gone.

Hope this gives you some direction to look.
Don't really know if it will.
