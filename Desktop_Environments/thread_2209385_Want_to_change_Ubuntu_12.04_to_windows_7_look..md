### Post by buzzingrobot on 2014-03-05
The default interface you see on Ubuntu is called Unity. And, the short answer is that you cannot change Unity to look or behave like Windows 7.

But... 

A number of other interfaces are available that can be used on Ubuntu instead of Unity and that use a panel and menu design that is much like that in Win7.  Whether or not any of them are close enough to Win7 in appearance to meet your needs is something you'd need to decide.

XFCE is one of these interfaces and it is the default interface on the Xubuntu variant, ([http://xubuntu.org/](http://xubuntu.org/)) . KDE is a very popular interface available on the Kubuntu variant ([http://www.kubuntu.org/](http://www.kubuntu.org/)).  Lubuntu targets older and less powerful hardware ([http://lubuntu.net/](http://lubuntu.net/)).

You might visit those sites, check out the screenshots and, perhaps, burn and boot a LiveCD/DVD to test the waters.

(XFCE, KDE and the Lubuntu interface can also be installed and used on an existing Ubuntu Unity system.  But, that's probably something for later.)
