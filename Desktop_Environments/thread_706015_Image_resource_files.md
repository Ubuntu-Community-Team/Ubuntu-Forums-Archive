### Post by Bubba64 on 2008-02-24
If you right click on an icon like the Firefox or almost any besides the menus and open up properties then click on the enlarged icon it will open up the access to icon directories I am not sure if all, but you can use this to add an icon to a png that you have added to the  bars or desktop.  in gutsy if you put  /usr/share/pixmaps/ in the search bar after clicking on the icon in properties this will bring up a large choice of icons, this has a drop down also. /usr/share/icons/gnome/scalable/categories/redhat-system_settings.svg also another icon file. Also 
/usr/share/icons/gnome/scalable/actions/gnome-run.svg so you might see the pattern of /user/share/icons  etc
