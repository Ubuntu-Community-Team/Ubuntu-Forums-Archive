### Post by zombifier25 on 2012-03-29
Workspaces are basically different desktops for one user (multiple user accounts logging in ANY OS using the same desktop is by definition impossible. Yes, I do know about the fact that you can run an app as a different user, but that's another story. Just ignore this.) You can put a window on a workspace and another window on another workspace so that you get less distractions, classifying windows, and whatever you want to use workspaces for.
you can put your daughter's websites open with cartoons on one space and your stuff on another space, if you want to.
