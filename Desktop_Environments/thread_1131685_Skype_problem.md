### Post by stumbleUpon on 2009-04-21
How did you install skype....? Using the .deb filed from skype download site...?

If so, to remove skype from your machine, open a terminal (Applications > Accesories > Terminal) and enter the code below followed by your password

```
sudo aptitude remove skype
```

OR using the Synaptic package manager.


Try the package in the medibuntu ([http://www.medibuntu.org/](http://www.medibuntu.org/)) repository. See here for more info
