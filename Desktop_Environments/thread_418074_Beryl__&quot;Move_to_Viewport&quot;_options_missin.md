### Post by CameO73 on 2007-04-22
I noticed it too. I guessed it was done to simplify the menus. But there's more to it, as you can see in [this thread]("http://forum.beryl-project.org/viewtopic.php?t=186"). One solution would be to set the Number of Desktops (in Beryl Manager's General Options) to 4 (the same as the Horizontal Virtual Size). This gives some functionality back (it's now called Move to Desk1, Desk2 etc.), but it messes up the cube view. 

It is not really the same as what Edgy provided (since that was 1 desktop spread over 4 viewports; the above solution gives 4 desktops over 4 viewports), but it might help you out.
