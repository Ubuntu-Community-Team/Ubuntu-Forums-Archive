### Post by Krytarik on 2012-04-12
If you are using regular Unity, thus with Compiz as the window manager, you can use its "Commands" plugin for that. Therefore, you need CompizConfig Settings Manager, as always. And the command for the default terminal emulator is "gnome-terminal", of course.

Regards.
