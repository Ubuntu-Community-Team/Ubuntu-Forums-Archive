### Post by calvinpriest on 2005-05-08
"Courier New" is one of the fonts in the msttcorefonts package.  You can install it from Synaptic (System->Administration->Synaptica Package Manager).  But first you will need to enable the "multiverse" repository.  In Synaptic you can enable extra repositories by going into Settings->Repositories and selecting multiverse.  After that you will want to click on the Reload button to refresh the package database.

A lot of this information is in [www.ubuntuguide.org](www.ubuntuguide.org), although the solutions are nearly all given in commandline form.
