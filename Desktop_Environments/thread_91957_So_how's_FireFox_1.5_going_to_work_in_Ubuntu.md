### Post by Zotova on 2005-11-18
[QUOTE=aysiu]Breezy Backports?
It probably won't happen on the exact day of the release, but...[/QUOTE]

Firefox 1.5 has an update system though, so Firefox will have to be modified to connect to a ubuntu server to look for the Firefox updates. Either that or the ubuntu version of Firefox is going to have to cut out the update feature. If ubuntu users have to download a full firefox installer each time I can see people getting annoyed when the real Firefox update might only be 200kb - not an issue for broadband users but for dialup users I'm sure it would be.

[quote=linkunderscore]I just want to know when the totem or mplayer will work with 1.5[/quote]

They might if you use the Firefox 1.5 in the backports but I've downloaded 1.5 as I was fed up of waiting (have they even sorted out oo2 final yet? :confused:). I have found a great work around though. An extension called media player connectivity - it basically takes over the totem/mplayer plug-in and launches the video in a new totem window (or whatever media player you choose). I've personally found it to be a lot more reliable than the plug-ins as they often did nothing when trying to play a video from my experience.

[Media Player Connectivity](https://addons.mozilla.org/extensions/moreinfo.php?id=446)
