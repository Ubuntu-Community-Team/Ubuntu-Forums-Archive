### Post by skullmunky on 2009-01-16
Try adding some widgets to the panel.  You can either Right Click the panel and choose "Add Widgets", or click the little yellow plasma icon on the right edge of the panel.  The default ones are:

Application Launcher
Quick Access
Task Manager
Pager
Show Dashboard
System Tray
New Device Notifier
Digital Clock
Trashcan

The reason windows aren't showing up in the panel is because you're missing a Task Manager widget.
