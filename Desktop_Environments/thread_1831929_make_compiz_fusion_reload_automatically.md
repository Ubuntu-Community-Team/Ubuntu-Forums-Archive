### Post by Frogs Hair on 2011-08-24
You could try to add the Compiz or the CCSM to your startup applications . To find the command needed , select the Compiz/CCSM properties in the main menu.

I have only used the Compiz Fusion Icon with Emerald on 9.10 and the only time it started automatically was when I installed Cairo/GLX dock . The dock was set to auto-start , so the Fusion Icon and Emerald started at boot . Before that I had click the icon to get Emerald to start .
