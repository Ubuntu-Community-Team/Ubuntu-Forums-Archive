### Post by Frogs Hair on 2013-08-09
I have installed and used  E17 on 13.04 both from the repository and PPA.  Try the following. Open the home folder - hidden folders and delete the .e folder . Log out of Ubuntu and attempt login to E17. A new .e folder and profile will be created at login.  If there is a problem in the .e folder re-installing won't help if the folder remains.
