### Post by llamakc on 2006-09-01
Themes are located at gnome-look.org and art.gnome.org (plus other sites). Don't worry about the "Ubuntu"-ish quality of themes. You'll be installing them to the ~/.themes directory (/home/yourusername/.themes) and if they don't work, you can delete them.

What you do is download a file from one of these sites with a name like nameofthemeorstyle.tar.gz and you literally drag-n-drop THAT theme from you desktop (or wherever you saved it) onto the open Theme program. So, open the theme program by going to SYSTEM | PREFERENCES | THEME and drag the downloaded package onto that window. It will prompt you to install it.

Next, go into Theme Details and make your choices.
