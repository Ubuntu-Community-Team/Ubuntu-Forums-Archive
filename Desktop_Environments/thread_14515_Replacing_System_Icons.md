### Post by Yukonjack on 2005-02-09
Click or double click on your new icon tar ball, extract the content to the .icon folder you will have to show hidden files to see .icons folder.
If you don't see an .icon folder make one make sure you have the .in front.
Now choose the icons you want to change make a back up of the originals and make sure the new icons have the same name as the original. A lot of these icons theme will do this automatic for you. 
Click on Computer--->Desktop Preferences--->Theme choose theme detail tab and choose your new icon set and it will change your icon theme.
