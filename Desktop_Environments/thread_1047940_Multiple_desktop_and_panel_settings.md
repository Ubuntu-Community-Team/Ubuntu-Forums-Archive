### Post by jacksaff on 2009-01-23
Kde 4 is still new and not all of the features of kde3 have been replaced yet. 4.2 makes some more progress and is out next week (can try RC now). Still don't seem to be able to set separate wallpapers for different workspaces though. 

It will happen soon enough - the way desktops are organised in plasma allows far more customisation than was the case in kde3. It's just that no-one has gotten round to writing good interfaces for it yet, or has even really pushed plasma to see what is now possible.

Give it a couple more weeks of use. There is certainly a lot kde4 can already do that kde3 couldn't, and the features you miss will return (or might already be included in the latest versions).
