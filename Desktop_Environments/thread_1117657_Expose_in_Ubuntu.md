### Post by mcduck on 2009-04-06
Install the CompizConfig Settings Manager (from repositories). All the effects and stuff are already there by default, you only need CCSM to configure which effects and features you want to use. :)

After you have CCSM installed use it to enable the "Scale"-plugin (and define what shortcut you wish to use for it).
