### Post by ProNoblem on 2005-07-15
The easiest way to install a gtk theme, metacity themes and icons is to open 'Theme' through the Gnome menu System -> Preferences -> Theme.

First download the Milky theme to your desktop (or somewhere you'll remember)
In 'Theme' you press 'Theme details' and just drag the file in the window (just like the hint suggests)
