### Post by Nis on 2005-02-22
Web developers are becoming smarter at getting around pop-up blocking.  Firefox gets a majority but every so often one will slip through.  I sugest getting the [AdBlock extension](https://addons.update.mozilla.org/extensions/moreinfo.php?application=firefox&version=1.0&os=Windows&id=10) and [Filterset.G](http://www.geocities.com/pierceive/adblock/); this has helped elimate even more ads for me.

As for spyware in Linux: most spyware is written to work only on Windows machines (where it will affect the most users).  Due to the nature of how programs are installed in Linux this makes it unlikely very effective spyware will become prevalent on Linux.  The best I've seen is a Java applet that popped up ads every so often (and being Java was cross platform).  The thing about this is it stops as soon as you kill the JVM (or reset the machine).  I wouldn't worry too much about spyware if you use AdBlock and Filterset.G.
