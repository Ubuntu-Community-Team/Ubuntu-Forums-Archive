### Post by prshah on 2008-09-15
> **igb said:**
> However, I can't find anywhere to add/configure printers. 

You can start a browser and open the address ```
http://127.0.0.1:631
``` which will launch the CUPS (Common Unix Printing System) administration page, a very easy way to install / manage printers.
