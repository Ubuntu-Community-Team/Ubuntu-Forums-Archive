### Post by mcduck on 2010-02-11
No, you will never get it in 9.04 release. 9.10 has it.

All program versions are frozed during Ubuntu's development, and after release you'll only get security updates and fixes for serious bugs. That allows the developers to test all the program versions to make sure they work fine together and don't cause any issues.

While some smaller programs may later be updated through the Backports repository, backporting somethign as large and complicated as Gnome is not an option. Such update would have far too large effect on other packages installed on your system.

If you really, really want to, you can try compiling and insalling it on your on, although that's quite likely to just break things. I recommend just upgrading to 9.10 instead.
