### Post by Bucky Ball on 2011-07-08
Or just install Unity from Synaptics and leave the desktop bloat. If you do it with ubuntu-desktop you will get *all* the ubuntu-desktop packages as well as Unity (much of which you won't want) and effectively be running kubuntu *and* Ubuntu; you can choose which at login.

Installing just Unity should make it available from 'Session' tab at login.

Cool. Just like you would install xfce4 for the xfce desktop environment (rather than xubuntu-desktop which installs pretty much Xubuntu) or gnome (or Unity for that matter) DE rather than all of ubuntu-desktop.

I love Xfce incidentally. Trying that is as easy as installing xfce4 from Synaptics and logging out, choose 'xfce session' from the sessions menu and log back in again. This method *only* installs the DE, your apps and settings remain. This would leave you in a situation where you are running Kubuntu but with Xfce as your DE rather than KDE.

If you install just Unity, you will be running Kubuntu with a Unity DE. Everything is the same except for how it looks. ;)
