### Post by Krytarik on 2011-09-18
You can change the "Placement Mode" in Compiz' "Place Windows" plugin. Therefore, install "CompizConfig Settings Manager" if not already done so (package "compizconfig-settings-manager"). You will find those plugin in the section "Window Management".

Greetings.
