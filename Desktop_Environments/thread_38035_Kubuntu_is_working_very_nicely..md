### Post by rosslaird on 2005-05-29
I've been switching back and forth between gnome, xfce, and kde over the past few months. I did like gnome for a while, then I was devoted to xfce, but then I started having problems with the "nvidia lockup bug" that so many ubuntu/nvidia users have been struggling with. The sort-of fix, which is to disable RenderAccel, did not work for me in gnome or xfce, but it does work in kde. So I've been forced to switch to kde, after a long absence from it. And I'm pleased to report that kubuntu works great. All the little bells and whistles are very nice to have (spell checking in Konqueror, the Amarok player, Kontact, a snappy interface, etc.).

Kudos to the kubuntu team.

Ross
