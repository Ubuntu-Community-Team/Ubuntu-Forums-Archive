### Post by shirilover on 2006-09-14
I have done this for my music album folder.

Open two nautilus windows
Use one to open the folder you wish to change
In the left pane, where it normally says Places, choose Information
Use the other to browse for the icon,photo,whatever you wish to use
drag the icon to the Information pane
