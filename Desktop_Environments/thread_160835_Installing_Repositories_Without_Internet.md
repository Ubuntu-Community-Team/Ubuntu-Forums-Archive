What I would advise, if you have another computer and a CD burner, is to burn [this .ISO](http://www.tikal26.net/ubuntu/addon.iso) as a disk image and then pop it into Ubuntu, fire up System > Administration > Synaptic Package Manager > Edit > Add CD-ROM. Then Reload.
