### Post by palsyboy on 2005-09-09
You can also add a Delete option to Konqueror's context menus.  Open a Konqueror window:

Konqueror -> Settings -> Configure Konqueror -> Behavior -> Check "Show 'Delete' context menu entries which bypass the trashcan"
