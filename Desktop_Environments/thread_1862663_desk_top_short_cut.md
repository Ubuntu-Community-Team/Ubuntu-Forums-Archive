### Post by pavi_elex on 2011-11-04
Right click on "example_of_webpage.htm" and click on make link. Now cut that link and put anywhere in system. when you will click on it, it will open the web page. If there is folder beside .htm file, It is used automatically by the link.
If you want to make link through terminal.
ln -s /opt/lampp/htdocs/mycameramyeye.com/index.htm /root/Desktop/pavi.htm
In this command, it is creating shortcut of "/opt/lampp/htdocs/mycameramyeye.com/index.htm" on desktop as a file "pavi.htm".
