### Post by ankspo71 on 2011-07-30
Hi,
I don't think KDE has the ability to change the default folder icon color, but I know there are ways to change individual folder colors, but of course that would take forever to change each and every folder icon displayed on your system

You might like using "oxygencolors" iconset found at kde-look.org. it includes a script to change the default folder style and colors. There are many other icon sets there to choose from too.

It might be possible for a script to change colors of your *.svg based icon sets too (usually Gnome icons). I haven't tried these on any KDE icons, but I have used them with some gnome icons before. (update: the default KDE oxygen icons are *.png icons)

[http://gnome-look.org/content/show.php/IconColors?content=127968](http://gnome-look.org/content/show.php/IconColors?content=127968)
[http://gnome-look.org/content/show.php/MagIcons?content=120508](http://gnome-look.org/content/show.php/MagIcons?content=120508)
[http://gnome-look.org/content/show.php/icoGEN?content=124870](http://gnome-look.org/content/show.php/icoGEN?content=124870)

Most Gnome icon sets should work well for KDE4 too, but some of the sets might not have every kde application icon needed, in which case those missing icons will (or should?) default to the oxygen icons. At the moment I am using Ubuntu's Humanity icons in KDE4.

At KDE-look.org you will also find some of the popular Gnome icon sets ported over to KDE such as elementary, faenza, nuove?, human & humanity, etc. 

Hope this helps.
