### Post by GrumpyBob on 2006-01-25
I'm using Bibus with OpenOffice.org 2.  It's quite well integrated, and the installation seems much easier than with previous versions (but read the documentation carefully).  One attraction is that I can use it with WinXP at work as well as Ubuntu at home.  It can use MySQL or SQLite as the database (I use SQLite), and it can import from Endnote (via one of the EndNote export options).

[http://bibus-biblio.sourceforge.net/](http://bibus-biblio.sourceforge.net/)

I think there has been some discussion about Bibus on this forum - it might be worth searching for it.  Also check out the OO.o support forums.

Robert
