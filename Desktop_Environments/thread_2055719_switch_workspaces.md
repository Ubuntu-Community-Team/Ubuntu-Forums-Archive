### Post by PaulW2U on 2012-09-10
Ctrl+F1, Ctrl+F2 etc can be used to specifically select the workspaces but I don't think anything by default will switch to the workspace on the left or right.

But remember this is KDE where everything is configurable. Go to "Global Keyboard Shortcuts" in System Settings and select "Kwin" to see what has already been set up and configure whatever you want.
