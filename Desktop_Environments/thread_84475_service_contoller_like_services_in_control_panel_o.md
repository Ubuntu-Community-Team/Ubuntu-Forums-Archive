### Post by 23meg on 2005-10-31
[QUOTE=frodon]In System > Administration > Services[/QUOTE]

It doesn't let you administer all running services though. You can monitor all running processes with Applications / System Tools / System Monitor, and there's a GUI tool called Boot-Up Manager that lets you tweak your startup scripts to remove certain services that you don't need.
