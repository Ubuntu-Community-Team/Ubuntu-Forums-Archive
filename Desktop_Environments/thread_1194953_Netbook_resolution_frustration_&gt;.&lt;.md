### Post by Closed_Port on 2009-06-23
Not a perfect solution but hopefully it works.
Try pressing Alt (and keep it pressed down) and then left click on the window. You should now be able to move the window up to see the missing options.

Also, on my netbook I use smaller font settings, which helps a lot with this problem.

### Post by billgoldberg on 2009-06-23
Alt+click should allow you to move the windows around.

As the user above said, on my netbook I use some tweaks to get more screen space for the apps.

Smaller fonts. Smaller panel or autohide panels.

Only use one panel.

A "more compact" gtk theme. There are themes where things are just closer to each other.

A smaller metacity theme is also useful.
