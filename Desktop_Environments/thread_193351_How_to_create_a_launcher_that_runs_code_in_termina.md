How can I create a launcher that runs code in terminal upon clicking on it?

---

### Post by bignickel on 2006-06-09
Assuming your code is already in an executable file (i.e. /home/cody/foo), right-click in the title bar, add to panel, custom application launcher.  In the command box either type in /home/cody/foo or browse to the proper file.  Click OK, and you're golden
