### Post by km4hr on 2005-12-02
Is there a way for non-privileged users to stop/start a modem connection?
I'm using  System -> Administration -> Networking to start the modem. I then have to provide a password. Is there a way that just allows any user to start/stop the modem?

---

### Post by amohanty on 2005-12-02
The easiest way is to activate your modem and setup/register an ISP there. Then you can use pon and poff to turn it on or off.

To allow a user to be able to do that just check 'Use modem' in the User Privileges tab in the User Properties.
