### Post by nocloud on 2006-06-04
i have this really annoying problem...

when i browse through my files with konqueror, it always opens up at a certain size and then i have to resize it, is there a way to get it to remember the size i would like it to open at so i don't have to resize the window every single time?

---

### Post by skymt on 2006-06-04
Yep!

Settings > Save View Profile *profile name*, check "Save window size in profile", and hit Save. There, wasn't that easy?
