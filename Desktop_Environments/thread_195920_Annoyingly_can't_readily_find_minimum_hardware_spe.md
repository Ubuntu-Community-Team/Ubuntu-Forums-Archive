### Post by IYY on 2006-06-13
On that link you'll get the reccomended specs. It's very difficult to state a minimum, since you could technically use a 200 MHz machine with 64 MB of RAM  if you used a lighter window manager, and lighter programs (and I know it's possible, since these are the specs of my laptop). But if they published  specs like that, people would actually try installing Gnome and KDE on these systems and become very frustrated. 

I'd say you need 128 MB of RAM to run Gnome at all, but it would be very slow. The suggested 256 is a good number, and if you can't match that, try XFCE, Fluxbox or IceWM.

If your machine is even slower (486 with 32 MB of RAM, for example) try Damn Small Linux instead.
