### Post by mcduck on 2010-08-12
You need to be running some compositing manager (like Compiz) for the Gnome-terminal (or any other program, really) to be able to use real transparency.

Go to System/Preferences/Appearance and make sure you have visual effects enabled (that turns on Compiz). If you can't enable them, then you probably need to re-install your graphics card's drivers after dong the upgrade to 10.04 or something... :)
