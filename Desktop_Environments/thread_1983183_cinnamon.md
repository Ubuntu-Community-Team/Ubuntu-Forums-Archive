### Post by cortman on 2012-05-19
You'd have to install Gnome 3 first. You cannot install Cinnamon on Gnome 2. Cinnamon's whole point is to emulate Gnome 2, so if you're using it already, Cinnamon isn't really going to give you any more options than you already have.
