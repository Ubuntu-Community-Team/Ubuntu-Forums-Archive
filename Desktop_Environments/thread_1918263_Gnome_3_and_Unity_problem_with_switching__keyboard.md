### Post by sunfromhere on 2012-02-01
Try enabling the log-in window: relog and there you should be able to choose which keyboard layout to use for the session, try to choose the non-default one and it should put the keyboard indicator back to the panel. This way worked for me.

I have installed 6 different distros with different desktop environments, and each one has given me problems with keyboard layout switchings. :D
