### Post by goobsoft on 2008-01-08
One of my favorite things about working in Gnome is the ability to use ALT to move the window with the mouse.  This keeps me from having to move the mouse up to the top bar which saves me some arm-stress and time.

When I open a program, typically the window needs to be moved and resized.  I know of a simple way to move the window now, but not a simple way to resize it.

There are two methods of resizing windows that I know of right now.  One is moving the mouse right on the edge of the window and click dragging (which I currently use), and the other is using the resize short cut and then using the arrow keys to select which side I want to change and then adjusting each side individually.  I say this because typically I adjust two edges to get the right width and height.

After realizing the benefit of the ALT and Mouse to move a window, I'm seeing the opportunity for a similar function to resize the window.  If I could design this, it would be simple and would work one of two ways.

1) The user would click the hotkey (maybe the windows key) and then left mouse-down on the window they wanted to resize -- anywhere in the window.  The user could then drag the mouse and the window would resize as the mouse is dragged just as it does when you click and drag the bottom right corner of a window.  For example if I clicked in the middle of the window and drug the mouse to the right 10 pixels and let go, the window would be 10 pixels wider that it was before I started.

2)  This option is the same as the one above except, the window would be logically divided into four quadrants.  If the operation was started in the top right quadrant, moving the mouse would work similarly to the way it works now when you move the mouse all the way to the top right edge to click and drag.

Does anyone know if this functionality already exists?  Does anyone else think it's a good idea?

### Post by mcduck on 2008-01-09
You can use Alt & middle mouse button to resize windows just the same way Alt & left button can be used for moving windows. :)
