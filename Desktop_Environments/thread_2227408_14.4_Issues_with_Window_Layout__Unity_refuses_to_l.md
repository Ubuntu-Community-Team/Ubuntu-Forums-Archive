### Post by 7-webmaster on 2014-06-01
I am copying information from an image in one window to a database application in another window.  So I have split the display between two windows with each using approximately half the space.  However with 14.4 Unity I am encountering a frustrating behavior when I click on the window resize icon, the third icon which looks like a rectangle, in the top bar of the window.  In setting up the dimensions of the two windows I have dragged the vertical size to the top, which causes the Unity window manager to extend the window in both directions to the maximum.  However the window manager does not seem to remember that it has done this if I click to expand the window to use the whole screen, and then click to restore it to half a screen, because it does not restore it to the maximum height, but rather to whatever height the window had BEFORE I stretched it vertically.  This means I have to manually stretch the window vertically every single time I restore from full screen.  I see a number of threads relating to tiling which all converge on the conclusion that Unity has no support whatsoever for tiling and indeed seems to go out of its way to frustrate anyone who WANTS to tile the screen.
