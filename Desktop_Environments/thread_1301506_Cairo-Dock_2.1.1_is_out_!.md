### Post by fabounet on 2009-10-26
A new version of Cairo-Dock is available.
This is mainly a bug-fix version, which improves the overall usability. Here is a short description of the news :

 - This version corrects many bugs appeared with the v2.1.0 (sorry for that !)
 - There is no more visual artifact under Metacity, OpenBox, KDE, etc.
 - Applis can now be placed amongst launchers and applets.
 - A new and convenient config panel for launchers.
 - Switcher provides a list of all opened windows on middle-click, and allows you to name your desktops.
 - Logout can be programmed to shutdown at a given time.
 - The dock fades out smoothly when it hides below other windows.
 - Clipper has an option to copy all items at once.
 - The DBus plug-in provides a new and powerful API that lets you control your dock from an extern application.
   You can even write an applet in any language with it !

About the DBus API, you can find a very comprehensive HOWTO here (with many exemples) :
[http://www.cairo-dock.org/ww_page.php?p=Control_your_dock_with_DBus&lang=en]("http://www.cairo-dock.org/ww_page.php?p=Control_your_dock_with_DBus&lang=en")
