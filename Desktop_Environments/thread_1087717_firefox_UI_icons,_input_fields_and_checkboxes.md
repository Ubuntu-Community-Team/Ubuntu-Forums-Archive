### Post by C-- on 2009-03-05
You could try installing a firefox add-on theme to change the UI icons for that program.

Also, you need to need to change your desktop theme controls. Controls are different from window borders and icons like folder icons; they are things like radio buttons, checkboxes and so forth. I don't know how to do this in KDE but in GNOME you can find installed controls under System -> Preferences -> Theme -> Theme Details. This will change the input fields and checkboxes for every program on your system.
