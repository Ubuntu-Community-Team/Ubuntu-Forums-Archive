All Ubuntu versions (Kubuntu comes with kwin, and Xubuntu comes with its own compositing manager) since 8.04 come with compiz, just not enabled if the driver doesn't support it.

So yes, you have compiz, and yes, it can be used as you have demonstrated.

Does having desktop effects set to none help your issue? Because compiz has Expo, which relies on <Super>+E (Windows Key + E)
