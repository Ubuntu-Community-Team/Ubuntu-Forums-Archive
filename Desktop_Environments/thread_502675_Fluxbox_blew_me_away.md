### Post by RedSquirrel on 2007-07-17
The best way to answer that is to try it yourself:

```
sudo apt-get install fluxbox && sudo update-menus
```

A few random thoughts:

Xfce:

- a desktop environment
- middle-weight in terms of resource consumption
- easy to use graphical interface for most things
- a lot of functionality built-in (panels etc.)
- fairly customizable
- generally not hard to learn its use


Fluxbox:

- a window manager (looks bare when you first install it, which is a plus for me) [you can add as much eye candy as you like later on with separate applications]
- a number of things require the installation of a separate (and small) program (setting the desktop background, desktop icons, fancy panels, etc.)
- is lightweight in terms of resource consumption
- configuration is done through text files
- very customizable
- takes time to learn (especially to set everything up the way you want, but it's not that hard)
