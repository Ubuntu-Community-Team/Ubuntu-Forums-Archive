### Post by bojzi on 2006-06-10
Did you try to go to System -> Preferences -> Keyboard and check what you have under the "Layout" tab? If Norwegian isn't there just go to Add... and then select your keyboard layout under Norway. 
It's also good to check if your keyboard model is listed in the keyboard model list because I had problems with some keys until I selected the right type of keyboard model.
