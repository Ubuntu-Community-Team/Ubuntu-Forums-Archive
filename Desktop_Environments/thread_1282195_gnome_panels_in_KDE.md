### Post by VCoolio on 2009-10-04
Just try to run "gnome-panel", it works with several other window managers, so why not in kde? Remember to add it to startup apps if it works and also "killall gnome-panel" will not respawn the panel like it does in gnome. It is a bit heavy on resources in comparison with others panels, but if you like it, use it.
