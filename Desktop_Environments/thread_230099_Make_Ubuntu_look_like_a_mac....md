### Post by sitedesign on 2006-08-07
If you like the MAC look for just having one bar for tasks launchers etc similar to windows uses then you could do away with the bar at the bottom and just have one bar for tasks and menus.

If you look for my posts there is one on there to do this making it similar to windows but I would suggest having the bar at the top like MAC does, it just seems more sensible since that is where the menus are in the apps and close buttons etc, so that is the region you normally have your mouse pointer when you want to start a new app.

The other icons are also in that directory where the distributor logo was so you could hunt down the MAC icon in there too. If not then try looking into the gnome config (small app for configuring all the icons etc) if you search this forum there is plenty of info on how to customise the icons etc.
