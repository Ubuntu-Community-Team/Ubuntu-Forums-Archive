I have installed pinta, but it did not add itself to the "open with" menu in gnome so I cant right click and image and select pinta to open it. I have been searching a while and cannot find how to manually add anything to this menu. Is there any way to do this?

### Post by tom4everitt on 2010-05-25
Right click on a file of the type you want to add pinta to, choose 'Properties'. There you will find a tab 'Open with', where you can simply add the program.
