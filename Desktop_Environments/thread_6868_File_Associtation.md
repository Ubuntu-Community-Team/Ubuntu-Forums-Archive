### Post by gborbolla on 2004-12-02
For having xmms play CDs when inserted go to Computer->Desktop Preferences->Removable Storage, In "Audio CDs" section change command to the xmms, check the syntax in the man page of xmms, don't have it installed so can't help with that

For adding a new program to the "Applications" menu open nautilus. select File->Open Location (Ctrl + L) and type "applications:///" or "applications-all-users:///" for adding it to every user, add your program where you want it, sometimes I have some problems when adding new folders.

Hope this helps
