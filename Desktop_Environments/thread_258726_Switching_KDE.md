### Post by aysiu on 2006-09-26
*kdebase* is the absolute bare minimum KDE.

*kde-core* has the basic set of applications (more than the bare minimum.

*kubuntu-desktop* has Ubuntu's default set of applications for KDE.

*kde* has *everything* you could want related to KDE.

KDE's equivalent for Synaptic is Adept, and it's terrible. Use Synaptic instead.
