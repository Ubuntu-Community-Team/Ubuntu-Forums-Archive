### Post by Frogs Hair on 2012-05-16
The Gnome Tweak Tool is an official package. Official packages are recommended. over third party applications and if you want to use the Gnome Shell at some point you will want to keep the Gnome Tweak Tool . I use Ubuntu Tweak but, it simply localizes settings that already exist and offers a few extra options. 

I don't know if Myunity will work as a replacement for the Gnome Tweak Tool or Just localizes settings.
