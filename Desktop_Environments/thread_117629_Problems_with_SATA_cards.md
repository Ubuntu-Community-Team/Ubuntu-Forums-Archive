### Post by obibok on 2006-01-15
Don't know about the particular card you're looking into, but I can tell you that my system is using the NVIDIA SATA controller (*nForce4 Ultra* chip mobo built-in, using *nv_sata* driver) and everything got detected automagically. I went with 2x WDC WD2500KS-00M HDD (soft RAID) and no problems or hassle so far. *hdparm* is a no go AFAIK, though.
