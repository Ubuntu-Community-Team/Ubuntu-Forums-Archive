### Post by Zelut on 2006-01-03
there is an option in File for 'create iPod directories' (or something similiar).  You'll need to run that so that it creates the directories it needs in your /media/ipod/ area.  Give that a try & see if that solves your problem

note:  a lot of people have issues with their ipods.  It seems the common error here is ejecting before properly, well, ejecting.  I suggest 'sudo eject ipod' ALWAYS before disconnecting your iPod.
