### Post by varunus on 2005-07-10
System sounds can be changed through System->Preferences->Sound.

Cursors are a bit more complicated, as the cursor is an X thing and not a GNOME thing.  There is a good howto over at [www.kde-look.org](www.kde-look.org) in the cursor section (these cursors will work in GNOME too, as will the howto).
