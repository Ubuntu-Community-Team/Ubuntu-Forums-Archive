### Post by ljubomir on 2006-10-31
No, you can't. But you can choose similar icons, theme and colours in kde control center (apt-get install kcontrol), so kde apps look similar to gnome ones. Take a look at [kde-look.org]("http://kde-look.org")

### Post by Jussi Kukkonen on 2006-10-31
qt3-qtconfig package is what you are looking for, if I remember correctly:

>  The Qt Configuration program allows endusers to configure the look
 and behavior of any Qt3 application. It is mostly only necessary
 on systems which don't run KDE because the KDE control center already
 covers this configuration automatically for the users Qt3 applications
 according to his desktop settings in KDE.
