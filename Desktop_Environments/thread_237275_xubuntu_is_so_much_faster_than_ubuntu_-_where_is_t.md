### Post by reacocard on 2006-08-15
basically, just the gnome desktop. gnome applications will run fine under xfce. xfce's desktop just has a slightly different set of features, and is more resource efficient. you won't be able to use gnome-panel applets, but that's not a big deal, since xfce has some of its own.
