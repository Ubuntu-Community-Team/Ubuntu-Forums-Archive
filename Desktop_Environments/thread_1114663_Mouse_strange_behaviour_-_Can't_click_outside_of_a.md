### Post by CopaceticOpus on 2009-04-03
This is a strange issue I'm seeing on the Live CD for Ubuntu 8.10. (And the same thing happens with the 8.04 CD.)

Whenever I start an application and click inside of it with the mouse, the application window seems to steal the mouse focus. I can still move the mouse all over the screen, but any clicks outside of the inner application window are ignored. I can't click on the Ubuntu menu bar, nor can I click on the current application menu bar or move the current application window.

I can break the mouse free by using the keyboard to select an item from the application menu, or by pressing ALT+TAB. At this point the mouse is free again, until I click inside an application window.

Does anyone have any ideas?

### Post by CopaceticOpus on 2009-04-19
I found the cause of this odd little problem. It was the mouse, which was old and which had its scroll wheel locked in the down position.
