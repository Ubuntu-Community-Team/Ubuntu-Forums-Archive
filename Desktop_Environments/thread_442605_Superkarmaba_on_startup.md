### Post by MZaza on 2007-05-13
You don't have to switch to KDE to install superkaramba. You can install screenlets, it's gnome close.
About the start-up problem, as far as I know that if you leave an application open in KDE and shutdown then startup it will be auto-loaded. So remove it from .kde/Autoload and just open the karamba then restart. If it didn't work try removing beryl.
