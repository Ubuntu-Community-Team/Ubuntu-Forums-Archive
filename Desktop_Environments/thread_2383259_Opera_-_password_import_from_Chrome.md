### Post by makem2 on 2018-01-22
Having searched around it appears that Opera cannot import passwords which have been exported from Chrome as a csv file.

Opera seems to rely on syncing but only across devices which are already using Opera. A 'LastPass' program for Linux has been suggested but this stores the passwords outside of my network and effectively beyond my control, relying on a master password.

I suppose the reason Opera does not import passwords, (there is in fact an option to include passwords whilst bookmark importing) is probably one of security. However, if one has say 100 passworded sites it is quite a chore to visit each one and manually set-up the password for Opera.

There must be another way, is there? :P
