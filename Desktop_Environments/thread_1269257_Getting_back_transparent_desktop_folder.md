### Post by Clove7 on 2009-09-18
When you install KDE or kubuntu by default, a transparent desktop folder appears in the top left corner. I have had it taken off, how do I get it back on now?

---

### Post by krazyd on 2009-09-18
it's called the Folder View plasma widget. Just right click the desktop, select add widget (unlock widgets first if that option is not available), then drag the Folder View widget wherever you want it.
