### Post by salva on 2005-11-12
As far as i recall Coh/CoV are Windows only games. You could try to get them running under wine/cedega, and depending on the support for these games you will have varied success.
You can check the [cedega games database]("http://transgaming.org/gamesdb/") for games that are supported and some reports on how they work. If cedega (which you have to pay for) supports it, theres a chance you can get it working on wine (which is free and in the ubuntu repositories).

In general though NCSoft the makers of CoH/CoV have a bad history of running on Linux due to gameguard and the likes.

Hope it works though, very nice games :D
