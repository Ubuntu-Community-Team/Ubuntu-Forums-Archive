### Post by alank on 2005-12-04
Hi,

The following is from this page for help installing Firefox 1.5 on Linux :

 [http://www.mozilla.com/firefox/releases/1.5.html#install](http://www.mozilla.com/firefox/releases/1.5.html#install)

====================
Linux/GTK2

Extract the tarball in the directory where you want to install Firefox:

tar -xzvf firefox-1.5.tar.gz

This will create a firefox subdirectory of that directory.
