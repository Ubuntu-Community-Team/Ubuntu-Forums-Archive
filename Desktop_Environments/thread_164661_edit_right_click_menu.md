### Post by ncappel1 on 2006-04-23
if you mean adding options to the "create file" entry in the right click, then you simply have to create a "Templates" folder in your home directory, then add files there. For example, I added a gedit text document, named it text, an empty abiword document, and called it abiword. get the idea?

you can also create shell scripts and add it to the scripts folder (not sure exactly where that's located) they will show up immediately in the right click "scripts" entry.

### Post by Wolki on 2006-04-23
There's also nautilus-actions, which allows you to add pretty much anything to your right-clcik menu, and to specify that some actions should only appear under certain circumstances (like set as desktop background only for images, enqueue in $Mediaplayer only for media files and so on)

You can find out more here: [http://www.grumz.net/?q=taxonomy/term/2/9](http://www.grumz.net/?q=taxonomy/term/2/9)
You'll find it in the repositories for dapper, installation help for breezy is here:[http://ubuntuforums.org/showthread.php?t=91377&highlight=nautilus+actions](http://ubuntuforums.org/showthread.php?t=91377&highlight=nautilus+actions)
