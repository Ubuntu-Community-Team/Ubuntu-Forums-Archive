### Post by kellemes on 2011-11-01
Personally I'm a sucker for twinpane filemanagement so I use [Worker]("http://www.boomerangsworld.de/worker/").
It's ugly but it has more power then any other filemanager.

Also you can use Nautilus.. start it like so..
```
nautilus --no-desktop
```
So it won't take over your desktop.

### Post by ankspo71 on 2011-11-01
Hi,
Pcmanfm is nice in Xubuntu because it feels lightweight, and it has an "Open Current Folder As Root" and "Open Current Folder In Terminal" features, and folders can be opened in tabs, and has "copy to" and "move to" features. There is no dualpane feature in pcmanfm though.

I have not removed thunar because it makes a good bulk file renamer (when I select mutiple files) and it has the ability for me to create custom actions (similar to nautilus scripts in a way).

Hope this helps.
