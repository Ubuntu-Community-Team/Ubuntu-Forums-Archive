### Post by mcduck on 2006-08-11
try this instead:

```
#!/bin/sh 
killall cgwd 
wait

cgwd & 
compiz --replace gconf &
```
the first line is just to tell that this is a shell script. Second line stops cgwd if it's running. Third line waits for a while, fourth line starts cgwd and the last one starts compiz with the gconf plugin.
