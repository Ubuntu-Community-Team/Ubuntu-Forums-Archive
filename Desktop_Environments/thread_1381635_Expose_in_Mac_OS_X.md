### Post by nerdtron on 2010-01-15
I usually have a lot of open windows when I'm on Linux. I saw this feature when my classmate showed me his Mac Book running Snow Leopard. Whenever he hover his mouse to the top-right most part of the screen, all of the open applications will show up. And then he can click any window he like to bring it in front. Something like this: [http://en.wikipedia.org/wiki/File:Exposeleopardall.png](http://en.wikipedia.org/wiki/File:Exposeleopardall.png)
It's called Expose in Mac OSX. Can I achieve the same effect in Ubuntu? How?:popcorn: 
Thanks in advance!

### Post by Linuxforall on 2010-01-15
You can achieve something closer with emerald window decorator. You can use the scroll button to roll up your windows or use the shade window button on the top left to do same.

### Post by bedhead75 on 2010-01-15
It's equivalent to the "scale" plugin in Compiz.  I use it all the time.  Install compizconfig-settings-manager and run it configure your settings.  I think scale is listed in the windows management section and you can customize your triggers.

### Post by hariks0 on 2010-01-15
You can use both ccsm-->Window Management--> Scale--> Corner Binding [With a monitor icon] 

or

Ubuntu tweak to set the corner binding.

I use Right bottom for this feature [Called 'Show windows' in Ubuntu Tweak]

You can do a lot more with compiz. Follow the link at my signature.

### Post by hariks0 on 2010-01-15
One more thing. Right click on any window toggles zoom. That is, you can use right click to bring a window to front without giving it focus. If you need another window right click again the window goes back.:popcorn:
