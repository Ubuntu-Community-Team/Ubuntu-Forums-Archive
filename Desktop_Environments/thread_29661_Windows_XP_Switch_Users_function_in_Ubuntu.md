### Post by Firetech on 2005-04-25
[QUOTE=derrick1985]Is there a way to switch users in Ubuntu like Windows XP without logging off (this way, all my programs continue running)[/QUOTE]
 Absolutely, but it works a bit different. There is a menu entry somewhere in the "system" menu, named something like "new login". (If you run kubuntu it's K Menu > Change user > new session.)
Clicking on that will start a new session, where another user (or yourself again) can login. I don't know how many sessions you can start, though... (Just try, it won't break anything. You might have to login to make the session stay "on".)
To change between sessions, you press Ctrl+Alt+F[number]. The first session (the one you're probably running in now) is F7, the next will be F8, etc. (F1-F6 are text mode terminals, when you're booting you're (kind of) in "tty1" (F1).)
