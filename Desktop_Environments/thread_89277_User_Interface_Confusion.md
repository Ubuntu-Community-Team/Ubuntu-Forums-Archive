### Post by Xoctor on 2005-11-12
I am really impressed with Ubuntu. The design goals are excellent and the implementation is very good.

One area I'd love to see improved is Synaptic. It is really cool that you can browse the whole library of available software, with version numbers and good descriptions. Its fantastic - and its far superior to anything the big corporates have come up with. Its a nice and simple process to select your app for installation. At this point you are probably thinking 'This is great - my grandmother could do this' - which is how it should be - but then you realise that while synaptic may have installed your software, you have no idea where, or how to configure/run/access it. Chances are it wont show up on the Applications menu, or have a man entry, or be available on the desktop. Not being able to see how to run the application once it is installed really detracts from the otherwise seamless installation procedure.

Each synaptic package should either create a menu entry or pop up instructions for completing the install process and/or accessing the app. If users are using a GUI to install an app, they don't want to have to go rooting around the command line looking for the right document and the right files to run, or googling to find the developer's homepage and instructions (especially since they may not really apply to Ubuntu). Some examples are phpmyadmin, zoph, and jsboard. Ubuntu does the best job of any distro at keeping it simple, but that's no reason not to make it even better!

### Post by Ampersand on 2005-11-12
It's probably worth suggesting features in the development forum.

For future reference, you can see what files have been installed by a package using Synaptic by viewing the package's properties. The executables tend to be created in one of the bin folders (usually /usr/bin). You can usually run programs by typing the program name in a terminal, or by entering the first few letters, then pressing tab to complete the command.

I've found that gnome and ubuntu have become better at adding programs to the menus recently, hopefully it'll continue to get easier. (:
