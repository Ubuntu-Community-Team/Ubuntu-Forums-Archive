### Post by Simanek on 2007-01-08
GRAMPS - Genealogy

Amarok is very cool and Exaile is apparently a GTK implementation of Amarok (Amarok is built on Qt, FYI). I use Amarok just fine in Ubuntu aside from Rhythmbox auto-starting everytime I plug in my iPod.

VMWare Server is free. It's a little tricky to install. Virtual machines let you keep a Windows install handy or try out other Linux distros without putting your primary system at stake.

Kontact is a collection of KDE Personal Information apps that communicate with eachother: Email, Calendar, Palm-synching, Journal, RSS Agregator, Notes, To-do list and Address Book. The interaction is priceless. I only wish there was a Gnome program that did so much.

GEdit: Look around through the preferences on this program before you disregard it as a simple text editor. It has a lot of great features.
