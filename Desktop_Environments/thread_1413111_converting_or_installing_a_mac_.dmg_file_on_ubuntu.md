### Post by Hero of Time on 2010-02-22
How do you expect software made for a different OS to run on Linux? You can't run .exe on Mac OS, BSD, Linux, etc. Not if you don't use wine. You can't run a Linux program on Windows, nor on BSD or Mac OS.
So a Mac program, in .dmg format, has no chance to be able to run on Linux. The systems may look similar, but they aren't. Mac OS is based on BSD.
