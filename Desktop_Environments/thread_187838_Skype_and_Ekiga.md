### Post by angkor on 2006-06-03
[QUOTE=ZombiekE]How do I configure ekiga to work with my Skype account?[/QUOTE]

Skype will not work on any other softphone software I know of. Why? Because skype doesn't use open protocols. Ekiga supports the sip protocol, skype sadly doesn't.

If you can convince you buddies to give up skype and use sip, you'll be happy. Because skype for linux has been crap for the last year or so. No development whatsoever. The sound quality I get with ekiga (or any other sip client like openwengo for that matter) is waaay better than I've ever had with skype.
