### Post by orphean on 2008-06-26
If you are willing to use (abuse?) your notification area to hold the icon for the application you could use alltray ([http://alltray.sourceforge.net/](http://alltray.sourceforge.net/)) to put anything you want in there.

It is in the repos so a simple apt-get should work to install it.
