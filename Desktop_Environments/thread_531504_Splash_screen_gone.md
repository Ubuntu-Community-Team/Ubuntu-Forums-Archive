I changed the splash screen from the Unbuntu standard one to another by using System->Preferences->Splash Screen.

Now I want to get the default one back, but it's not available in the Splash Screen preference application.

---

### Post by Milk &amp; Toast &amp; Honey on 2007-08-23
The default splash screen located in:
/usr/share/pixmaps/splash/

Try choosing or installing from there.
