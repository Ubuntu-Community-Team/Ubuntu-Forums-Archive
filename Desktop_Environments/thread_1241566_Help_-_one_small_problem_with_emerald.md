### Post by coldReactive on 2009-08-16
ALT+F2

and make sure NOT to check run in terminal.

Also... if you want it to run on startup:

System > Preferences > Startup Applications (or Sessions)

Add a new entry, name is **Emerald Theme Manager** and put in the command box **emerald --replace**.
