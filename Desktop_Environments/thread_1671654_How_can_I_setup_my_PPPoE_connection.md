### Post by inobe on 2011-01-20
the nameserver may need changing, for example:

in opensuse with at&t dsl, i needed to use an alternate dns, you must find yours because **this is just an example**

```
ATT DNS Name Servers

68.94.156.1
68.94.157.1
```

then in terminal

```
kdesudo kate /etc/resolv.conf
```

when kate opens i needed to remove the old and add the alternate dns nameservers to the bottom of resolv.conf similar to this 

```
# Generated by NetworkManager
nameserver 68.94.156.1
nameserver 68.94.157.1 
```

after a quick reboot i had a connection.

do you have at&t dsl ?
