I'll answer the second question first:
The "Utilities" tile that collects several applications is called a app folder. You can create one by dragging one application tile onto another. This immediately groups them together as "Unnamed Folder". To give it a permanent name, click on the tile and use the pencil in upper right of tile to edit it. To add more applications, just drag them onto the app folder. Screenshot attached shows an app folder that collects 4 games, and named "Games".

As to the first question, the Dash (or Ubuntu Dock if you use that) is provided to hold often-used applications (favorites).
