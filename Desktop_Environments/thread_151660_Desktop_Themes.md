### Post by mustang on 2006-03-28
[QUOTE=k9dxr]As a newbee, how do I change or add a desktop theme similar to changing Windows themes[/QUOTE]

Systems->Preferences->Themes

There are some themes there installed by default you can try out. If you want more themes go here:
[http://www.gnome-look.org/index.php?xcontentmode=100&PHPSESSID=c542f8beb380b31d44da6cdb5630a2cf](http://www.gnome-look.org/index.php?xcontentmode=100&PHPSESSID=c542f8beb380b31d44da6cdb5630a2cf)

Find a theme you like. Drag the downloading link into the Themes window.
