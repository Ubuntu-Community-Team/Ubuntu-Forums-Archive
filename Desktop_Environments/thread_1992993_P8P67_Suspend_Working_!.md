### Post by drumour on 2012-06-01
My i7 systems using ASUS P8P67 LE motherboards which require ACPI 5.0, now suspend and resume following the latest proposed kernel update 3.2.0-25. Prior to this it was almost working but quickly hung after resuming.
 
Note 1
This is with AMD/ATI catalyst drivers from the ubuntu repositories. 

Note 2
One personal caveat is that on two near identical systems the mouse cursor is stuck and requires replugging - but this is most likley to do with previous attempts to get this working.

Many thanks to all involved.
