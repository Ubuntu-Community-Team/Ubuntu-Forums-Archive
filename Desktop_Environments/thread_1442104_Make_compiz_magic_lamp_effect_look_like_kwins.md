Kwin's “Magic Lamp” effect is very similar to OS X's “Genie Effect,” nearly identical. I think Compiz's effect is not as appealing because it stretches the image more then it should. I think the OP wants to remove the horizontal variation and simply make it a stretch vertically into the task bar.

The Magic Lamp effect originally was able to only do one horizontal curve during the animation, but it was changed since Apple has some patents on that effect. There used to be a modified version of the plugin that allowed setting wave amount to 1, perhaps it's still around and might even work with the current Compiz version.. Other than that, the exact effect is not possible with Compiz.
