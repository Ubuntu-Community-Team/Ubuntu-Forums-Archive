### Post by TheWizzard on 2006-10-06
the KDE Edutainment Suit seems to be available from the standard repos. Just open adept and do a search for "kdeedu". you can also pick the applications one by one. GCompris is also available on the repos. I couldn't find tux4kids  as a complete suit in the repos, but individual applications like TuxTyping and TuxPaint can be installed from the repos. 
So you can make your "Kedubuntu" pretty easily. 
If you have plenty of disk-space you can install edubuntu-desktop. This installs all the applications you want. You can continue using KDE and acces the applications from KDE. Linux allows you to have both KDE and GNOME installed and to use both. 
Hope this helps.
