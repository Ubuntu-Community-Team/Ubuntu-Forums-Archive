### Post by jward3010 on 2009-11-05
Why do you just want to make it two, there is every chance it will go 20MB/s. Formatting the stick can surprisingly bring speed back. I highly recommend using a FAT32 filesystem. The easiest way is using Gparted, or if you're using 9.10 Karmic just right click on the stick and choose Format.

**PLEASE BE WARNED:** This deletes all data off the stick!!!

### Post by jward3010 on 2009-11-05
Most modern sticks will get speeds on average as follows:

Read: 10MB/s to 30MB/s
Write: 5MB/s to 12MB/s

Thats my experience. And it completely depends on file size. Putting a 700MB AVI file directly on (one continuous piece of data) will go quite quickly. Writing 1000 30KB files will be a different matter.
