### Post by mcduck on 2006-08-01
Theme engine is just a program that draws the widgets, it's not a theme so it's not supposed to show anywhere.

But if you install themes yourself they will show in theme selector.

Themes only show in theme selector's main window if they are complete themes with GTK theme, Metacity theme and icons.. So if you only have a GTK theme it will only appear in theme details. (unless you select icons & metacity theme for it and then save that complete theme)

### Post by patrickfromspain on 2006-08-01
EDIT: I just have realized what you have to do to use the engine :)

After installing the murrine engine, you must download the murrine theme pack from the gnome-look site. Extract the files in the archive and you'll get 4 folders. Move them to ~/.themes and you'll get 4 new control themes. Enjoy :)
