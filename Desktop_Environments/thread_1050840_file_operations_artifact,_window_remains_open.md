When emptying the trash occasionally a window pops up for a bit titled file operations. Once it popped up and didn't go away, so I closed it and redid the empty trash. 

Now whenever I do a file copy and the window opens with the progress bar there is a second empty progress bar above it titled file operations. When the file finishes copying, instead of the window closing the second bar disappears and the other empty bar remains and I have to manually close it.

This may be a bug, not sure though. Any ideas how to fix it? Thanks.

### Post by WeDanceStupid on 2009-03-26
I got rid of this problem by adding the "Force Quit" item to my panel. Leave the File Operations window open after a file transfer, click the Force Quit icon, and then click the window. It will close and the artifact will be gone the next time you move a file.
