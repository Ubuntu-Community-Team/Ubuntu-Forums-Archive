### Post by kripkenstein on 2007-10-23
Hi,

I'm not sure, but here is an idea. Nautilus has a 'preview' for sound files. So perhaps when you mouse-over an mp3 file it tries to play it and there is some problem with that.

You can check if this is related by disabling the sound preview. Do Edit->Preferences in Nautilus, then go to the "Preview" tab. Select "Never" for "Preview Sound Files".

Then try to mouse-over mp3 files and see how it goes.
