### Post by 3Miro on 2011-04-05
Gnome uses Metacity as the default Windows Manager. Metacity is fast, but very limited in functionality, that is why many people prefer compiz. So in short, the answer is: you need compiz.

As for other environments, KDE has the feature, but KDE is somewhat demanding itself, so it may or may not run on your machines.

I don't think any other Window Manager has this functionality.
