I designed new photo manager for linux. It originally is based on picasa. It has many features including tagging. Here is a short description:
IntiPunku? is a photo manager which aims at feature richness and ease of use. Originally the idea was to create a clone of the popular photo manager Picasa on windows. The name means Sun Gate and originates from historical site Tiwanaku in Bolivia. Tiwanaku is till now know as the legacy of the oldest civilization of South America.

For gutsy i recommend to use the following apt repository:
deb [http://intipunku.googlecode.com/svn/apt/](http://intipunku.googlecode.com/svn/apt/) gutsy main

the website is: [http://intipunku.googlecode.com](http://intipunku.googlecode.com)

I will soon upload my gpg key.

I m also looking for more translators and developers. If you are interested please contact me. 

Screenshots can be fount on the website, however they are outdate, but give a global impression.
Features include:
* folder and tag view
* picasaweb uploading
* export to html page
* Slideshow
* advanced photo enhancements (brightness,contrast,temperature,color,sharpness, autocontrast, equalize)
* rotations
* crop
* removing red eyes
* support languages: english, spanish, catalan, italian, dutch and german.
