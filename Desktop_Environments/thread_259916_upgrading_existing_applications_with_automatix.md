### Post by ekravche on 2006-10-27
Seems that with every release of ubuntu there is an automatix release and you can use the latest release of automatix to reinstall ("upgrade"?!?!) the application, which you have installed with the previous automatix release. A new feature in automatix2(edgy eft) has the ability to uninstall the apps that you have installed. So before you move to a newer release of ubuntu, you may want to use the uninstall feature of automatix and then get the automatix for the next ubuntu release and just reinstall the apps that you've uninstalled from the previous release. It seems thought that reinstalling the apps on top of the existing ones also works, so either way is good.
