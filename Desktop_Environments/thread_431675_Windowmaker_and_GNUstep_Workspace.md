They are actually different animals. Windowmaker is a desktop environment, while GNUStep is a whole development environment:

> GNUstep is a free, object-oriented, cross-platform development environment that strives for simplicity and elegance. GNUstep is based on and completely compatible with the OpenStep specification developed by NeXT (now Apple Computer Inc.) as well as implementing many extensions including Mac OS X/Cocoa.

There is no advantage or disadvantage that I can see to using one or the other. If you want to learn/use objective C, or want a complete integrated desktop environment, then GNUStep is the way to go. If you just want Windowmaker instead of GNOME or what have you, then use Windowmaker.
