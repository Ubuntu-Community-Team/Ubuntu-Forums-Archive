### Post by georgetom on 2009-11-17
Install Screenlets from Synaptic. Lyrics Screenlet did not come as default for me.
Download it separately from [http://www.gnome-look.org/content/show.php/Lyrics+screenlet?content=98762](http://www.gnome-look.org/content/show.php/Lyrics+screenlet?content=98762)
Open Screenlets from Accessories menu, use the Install/Add option in it and locate your Lyrics Screenlet download.
