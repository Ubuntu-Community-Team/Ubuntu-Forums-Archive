### Post by chalex on 2006-08-16
There's a freeware program for Windows called DVDShrink which can rip, re-encode and create an ISO from a DVD.  People have reported that it runs well under Wine. Then just burn the ISO as you would normally, using Nautilus or GnomeBaker or whatever.

### Post by azmrb on 2006-08-16
I have DVDshrink (latest version) running under the latest version of wine.  It is VERY fast and I recommend it.  

IMPORTANT:  When you create the app entry for it in winecfg make sure you check the "Show dot files" box on the drives page or the Browse folders window will hang.:D
