### Post by denham2010 on 2009-01-04
Hi Kato,

As Xfce uses right click on a desktop as a special menu function, this won't work for what you need.

The "Desktop" context menu is accessed by right clicking on an existing icon on the desktop (eg. Trash or Home).

The last item on this menu is called "Desktop" where you will find the create launcher option.
