The quote reads
> Ubuntu includes more than 16,000 pieces of software, but the core installation fits on a single CD. 

Basically, a bunch of core apps are on the install CD.  Once you install the base system, you can use programs called "synaptic" or "apt-get" to download any of the other packages ny automatically install the software for you.

The beginners guide explains this an a lot of other frequently asked questions:
[http://www.ubuntuguide.org/](http://www.ubuntuguide.org/)
