### Post by Frogs Hair on 2014-01-20
Ubuntu Studio applications are available for all versions of Ubuntu , but you would have to install them yourself . U. Studio used a real-time/ [COLOR=#333333][FONT=Ubuntu]low-latency kernel[/FONT][/COLOR]  in the past and I don't know for certain if that is still the case. Starting with another Ubuntu flavor as a base would exclude that kernel from being installed by default if that's important.
 
I have given most of the well known window managers an honest try and KDE is one I won't install again unless it's with Kubuntu. The bottom line is use what suites you functionally and appearance wise. All  of the Ubuntu desktop environments can be customized once you know your way around. I use the same themes and icons on XFCE as I do on Unity . XFCE has a few nice composting settings under window manager tweaks if you like transparency .
