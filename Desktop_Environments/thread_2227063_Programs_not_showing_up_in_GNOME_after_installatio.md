### Post by digerati-RC1 on 2014-06-01
I have this strange error on mine as well, but mine were actually just hidden from even the "All" option in the menu.  To find it, just hit the windows key on your keyboard or slide the mouse in the upper left corner to open you menu and then start typing the name of the app if you know it.  If you type the name all the way out and you get no search results from the menu, from the gnome desktop (not the applications menu) hit alt+f2 and it will open a launcher.  Start typing the name of the program and hit "Tab" to autocomplete the program name.  If it finds it, it will fill the rest of the word in.  If it does not, go back and check the package name and see if the launch name in Ubuntu differs.  Just don't let the fact that they don't show up in the applications menu after install fool you into thinking it wasn't installed like I did.  Hope this helps!
