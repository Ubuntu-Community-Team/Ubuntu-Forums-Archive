Short answer. Yes, you can!

Longer answer: You have to make sure to download all the dependecies also. You can find packages at [http://packages.ubuntu.com/](http://packages.ubuntu.com/)

Example:
Using , Intrepid, want to install FileZilla. Go to 
[http://packages.ubuntu.com/intrepid/net/filezilla](http://packages.ubuntu.com/intrepid/net/filezilla)
You will need to install all packages with a "red dot", those packages can also have dependencies and you will need those also... Some of these you will have installed by default, so try to install and se what error you get.

Print Screen is just a press of a button! (The print screen button :))
