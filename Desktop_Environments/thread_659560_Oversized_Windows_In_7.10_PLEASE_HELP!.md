### Post by Fallom on 2008-03-12
> **ingvildr said:**
> Go into System > Preferences > Appearance and then the font tab and change the font size of the window title bar to something sane like 10.

This doesn't fix the problem. On a fresh install the Window title font is already set to 10 and making it smaller only slightly reduces the oversized font.

Setting it to 1, however, does seem to fix the problem. Anything above that will just make the font huge again, and the problem persists with fonts other than Sans (The default).
