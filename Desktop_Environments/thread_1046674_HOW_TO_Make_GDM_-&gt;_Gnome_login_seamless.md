### Post by blazemore on 2009-01-21
It's annoying that the beautiful GDM theme drops out its background image to a solid, ugly brown after you input your details.

This will patch the relevant files to make sure the background remains the same until Nautilus draws its own.

For a really seamless login, you can change the GDM background to the same as your Nautilus background.

To run: Just run the install command from the terminal
```
sh install
```

to uninstall,
```
sh revert
```
