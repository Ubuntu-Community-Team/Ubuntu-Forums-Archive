### Post by SEMW on 2008-08-19
How about:
[IMG]http://www.justuber.com/_media/gnomenu.png?w=400[/IMG]

If you like the look of that, open a terminal and run:
```
sudo apt-get install gnome-main-menu
```
Then right click on the gnome panel, select &#8216;Add to Panel&#8217;, find the section called &#8216;Utilities&#8217; and select the &#8216;Main Menu&#8217; app indicated by the picture of the computer.
