### Post by ZeldaFan on 2009-04-28
However, I need the cube (I know that sounds stupid), but I'd rather have the cube working on an Intrepid setup than no cube and Jaunty. However, disabling fusion-icon, and using simply CCSM along with the Ubuntu toggles for desktop effects in their Appearance menu, I was able to get a desktop cube set up with both screens fully functional (with the problem of apps still only loading on the first screen persisting). 

I think it is interesting to note that when I enabled the Grouping Windows plugin, the second screen froze, so that may be a reason for problems. Additionally, I can no longer set two sets of settings in CCSM for each screen, whatever I change in one screen affects the other screen as well. However, certain things like the number of workspaces and wallpaper plugin can only affect the first screen.
