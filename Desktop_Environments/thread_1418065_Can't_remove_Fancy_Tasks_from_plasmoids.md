### Post by Zorael on 2010-02-28
Very curious.
```
trying to overwrite '/usr/lib/kde4/plasma_applet_fancytasks.so', which is also in package **fancytasks** 0:0.9-8
```
There *should* be a **fancytasks** package. See if either of these do the trick.
```
$ sudo dpkg --purge fancytasks

$ sudo aptitude purge **~n**fancytasks
```
There is a way to force the second package to install, but leaving the first like this would be bad in the long run.
