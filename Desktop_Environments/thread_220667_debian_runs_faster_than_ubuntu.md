### Post by VirtuAlex on 2006-07-23
If you are experienced enough to use debian - then use debian. Ubuntu is debian configured to fit average user on average machine. For that reason running ubuntu means you running bunch of stuff you do not need.

Basically if you start with barebone debian unstable you can add and add apps until you'll get ubuntu, but... without twice-a-year release schedule and security patches. Or you can start with ubuntu and tweak it removing unneeded stuff untill you get sought performance enjoing all the support. Whatever fits you.
