### Post by flyingbrass on 2005-10-16
It is annoying.  I wish the Ubuntu folks had added it back as part of the standard installation.

Enable the universe respository.

sudo apt-get update
sudo apt-get install nautilus-open-terminal

Might need to restart gnome-panel or log out and back in again for it to show.
