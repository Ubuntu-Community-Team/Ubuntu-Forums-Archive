### Post by 3Miro on 2011-11-02
If you have Unity 3D, then compiz is already working. You can install CCSM to adjust the way you want it to work.

From CCSM -> Desktop Wall -> Edge Binding, you can select to drag windows to adjacent workspaces.

You can see all workspaces and all windows by clicking on the applet on the Unity bar.

Activating the cube is tricky:

[http://ubuntuguide.net/enable-compiz-desktop-cube-in-ubuntu-11-10-oneiric-unity](http://ubuntuguide.net/enable-compiz-desktop-cube-in-ubuntu-11-10-oneiric-unity)

I had success with this video:
[http://www.youtube.com/watch?v=yjTP8BGjn_A](http://www.youtube.com/watch?v=yjTP8BGjn_A)
