### Post by lorap on 2005-04-27
hi friend!
to get all those things you've mentioned in your request install VLC player.
it'd give you the possibility watch movies and listen to the music, including mp3 files.
open Synaptic Package Manager, then go to the Settings-->Repositories.
mark in that window all checkboxes possible regardless any warnings and press Ok.
after what press Reload button. once the process completes press the Search button and write in VLC then press Ok. the moment synaptics picks it up for you mark it for install and press Apply button.
you're done now, enjoy linux world.
