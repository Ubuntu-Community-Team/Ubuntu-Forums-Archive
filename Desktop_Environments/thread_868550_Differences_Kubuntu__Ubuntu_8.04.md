1. Kubuntu doesn't have the Games section as Ubuntu. How to install these games?

```

```

> **UranUtan said:**
> 
2. How to install Firefox 3.01 in Kubuntu? I have heard that KDE's browser is Konqueror, but I want Firefox.

```
sudo apt-get install firefox
```

Or you can download it from the Firefox homepage and compile it yourself.
