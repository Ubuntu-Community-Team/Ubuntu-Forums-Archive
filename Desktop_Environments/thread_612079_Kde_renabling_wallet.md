```
kwallet
```

And then re-enable the wallet.

### Post by oobuntoo on 2007-11-13
alt+f2
type: kcontrol
Under "Security & Privacy", select "KDE Wallet", on right-hand side, tick "Enable the KDE wallet subsystem"
