Hello and Welcome

Have you attempted to move the dock to the bottom from settings ? There is also an applications menu shell extension that allows for a more traditional [application menu]("https://extensions.gnome.org/extension/6/applications-menu/"). The [dash to panel ]("https://extensions.gnome.org/extension/1160/dash-to-panel/")extension is another option. Extension can be installed and enabled when gnome tweaks is installed.
