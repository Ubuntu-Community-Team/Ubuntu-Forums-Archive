### Post by mcduck on 2009-08-18
You need to change the resolution used in TTY's. Changing the font size alone wouldn't really help anything, it would be pretty much unreadable.

This is a fairly good guide for configuring framebuffer resolution:[https://wiki.edubuntu.org/FrameBuffer]("https://wiki.edubuntu.org/FrameBuffer")
