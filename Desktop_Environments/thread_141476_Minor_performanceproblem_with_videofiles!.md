### Post by Pragmatist on 2006-03-09
> DMA is on, but it shouldn't affect playing fiiles from the HD.
That is exactly the purpose of DMA.  It allows the hard drive to directly access memory rather than having to go thru the CPU.  This saves time and often is the difference between slow, choppy, playback and smooth playback.

Install these:
**libxine1c2 xine-ui**  I also have **libxine-dev**
