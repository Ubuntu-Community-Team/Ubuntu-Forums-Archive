### Post by 23meg on 2005-12-30
It's metacity theme dependant, which means its position is declared in the metacity theme you're using. You'll need to use a theme that specifies a right aligned title bar text, or you'll have to manually override the behavior with your custom .gtkrc file. I'm not sure how you can do the latter but if you do a search you'll probably find how to.
