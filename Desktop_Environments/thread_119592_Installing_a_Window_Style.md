### Post by aysiu on 2006-01-19
[QUOTE=kubuntu32]
How do you install Window Styles?  I downloaded crystal style and want to install it but unfortunately I can't drag and drop / extract it into the appropriate folder without getting a read only error. (I dont like the fact that i dont get full access) so do I need to login as root everytime I want to install a window style or is there a terminal command to do so?[/QUOTE] In Gnome, you just go to System > Preferences > Themes and drag and drop the .tar.gz and it gets untarred and installed for you.

In KDE, it's a bit more complicated, and it usually involves some command-line commands. If you want those command-line commands to have "full access," just put a *sudo* in front of them. Any .tar.gz theme usually comes with a README file inside of it explaining how to install it.

I just use Gnome. It's easier to install themes for.
