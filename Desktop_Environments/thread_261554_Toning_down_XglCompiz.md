### Post by Qrk on 2006-09-20
You can disable the plugin "wobbly" or set the spring constants to be far higher than they are now. To disable wobbly you can use the Compiz manager (Gset-Compiz, simply uncheck it) or remove it from "thefuture" if you use that method.
