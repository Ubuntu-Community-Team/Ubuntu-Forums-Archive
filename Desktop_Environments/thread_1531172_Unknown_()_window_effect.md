### Post by bigsmitty64 on 2010-07-14
That looks like the magic lamp effect in compiz. Go to system>preferences>compizconfig settings manager. Then in there go to effects>animation, enable animations then go through the tabs and have a blast!
If you don't have compizconfig settings manager yet, install it through synaptic package manager, make sure to get the compiz-fusion-plugins-extra too!
