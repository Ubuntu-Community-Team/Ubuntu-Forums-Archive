I would like to know how I can install windows software using Wine.  Once I've got the .exe file I run wine xyz.exe on the console and that launches the installer.  But on what path should I install it? It asks me for a windows path.

### Post by meng on 2006-09-03
There is a virtual C:, complete with Program Files directory, and it is located under /home/yours/.wine
Therefore you can generally accept the default installation directory, with a windows-like path.
