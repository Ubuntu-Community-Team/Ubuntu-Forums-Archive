### Post by spartas on 2006-01-27
You could always install ubuntu using the server option.  To do this type 'server' at the installation prompt.  This will get you a fully functional ubuntu, but without a desktop environment.  From there you can install xfce with 


```

sudo apt-get update && sudo apt-get install xfce4

```
