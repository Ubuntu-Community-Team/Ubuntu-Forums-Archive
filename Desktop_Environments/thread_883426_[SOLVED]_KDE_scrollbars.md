### Post by Sknig on 2008-08-07
Hello, I just started using Kubuntu yesterday, and am starting to be driven crazy by scrollbars. When I move the mouse to the extreme right, it is no longer on the scrollbar; one pixel to the left and it's fine. I don't have this problem with the upper-right close button. Is this a setting that can be changed, or is KDE just meant to work this way?

The only thing Google turns up related to this is someone who had this problem only after installing Compiz (which I haven't done), and was able to fix it by changing some setting in Compiz. The only other thing I could think of was that it somehow had to do with the mouse trying to go to another desktop, but the problem was still there with only one desktop. Has no one else had this problem, or do people just not mind?

Edit: System -> Window Behavior -> Advanced -> Active Desktop Borders
Setting this to always enabled fixes the problem. It seems like it actually thinks you're resizing the window, but it still works.
