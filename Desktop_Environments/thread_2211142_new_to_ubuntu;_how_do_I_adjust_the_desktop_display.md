### Post by grahammechanical on 2014-03-14
Go to System Settings>Appearance>Behaviour tab and If you are using Ubuntu 13.10 you will find a slider that will reduce the size of the Launcher somewhat. I am not sure if this function is available in Ubuntu 12.04 but I know that in 14.04 (to be released at the end of April) we can reduce the Launcher size to a much smaller size than in 13.10.

In the behaviour tab we can also set the Launcher to autohide. It disappears into the left screen edge only to reappear when we move the mouse pointer to the left screen edge. So, we can have a Launcher size appropriate to our vision capabilities but also get more screen real estate.
