### Post by abasel on 2019-02-05
I have installed a [Xenial]("http://wiki.pine64.org/index.php/Pine_A64_Software_Release#Xenial_Base_Image") base image (no desktop) on my Pine64. I then ran
```
sudo apt-get install ubuntu-gnome-desktop
``` as per [https://wiki.ubuntu.com/UbuntuGNOME/Installation]("https://wiki.ubuntu.com/UbuntuGNOME/Installation") to install what I thought would be the gnome desktop. While lots was installed, nothing appears to have changed. I still don't appear to have an X environment.
