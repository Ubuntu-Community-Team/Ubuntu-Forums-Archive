### Post by Dennis N on 2017-06-17
An alternate method to resize:

In XFCE, if the window is the active window,  Alt + F8 should put the window in "resize mode" and a resizing cursor will appear in lower-right corner of the window. Moving the mouse will then resize the window. Once resized, a mouse click ( or another Alt + F8) will return cursor to normal mode.
