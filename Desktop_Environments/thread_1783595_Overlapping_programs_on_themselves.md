When i first played around on Ubuntu 11.04, one of the first things i noticed was the desktop wall kind of wrapped around so that there were no edge restrictions. well curious i expanded i file till it overlapped itself. it took me a while to find out how to fix it but i found that using key board shortcuts was the only i was able to fix it. i used
**Alt+F7**  Moves the current window (can be moved with mouse or keyboard). 
**Alt+F8** Resizes current window (again,can be moved with mouse or keyboard).
minimizing didn't help and the screen went all glitchy the application went invisible and then i couldn't see the buttons. but even when i grabbed the sides to rezise i couldn't thats why i had to use the keyboard shortcuts.
Hope this helps someone this is not a question it is a solution thanks.   :D
