### Post by Hooya on 2009-01-11
You should be able to install it pretty simply through Synaptic.  I have both running, they maintain separate config folders as far as I know.

sudo aptitude install kde-core

Should give you the basic KDE package.  I have KDE3.5, KDE4, Gnome, and Fluxbox and can run all four in separate sessions simultaneously!
