### Post by Jucato on 2006-10-01
Kuickstart has been dropped because it uses an obsolete library that Kubuntu cannot use (starting Dapper).

Here's the bug report on that: [https://launchpad.net/distros/ubuntu/+source/kdebase/+bug/42081](https://launchpad.net/distros/ubuntu/+source/kdebase/+bug/42081)
