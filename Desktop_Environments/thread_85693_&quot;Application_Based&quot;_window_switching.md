### Post by Ptero-4 on 2006-04-29
quietglow. The "Application-based" focus was planned to be implemented on metacity, but when metacity became part of gnome the gno(M$)e devs decided that "Application-based" focus was a forbidden feature (a feature that makes gnome work like OS9 in one or more ways) and thus had to be dropped and left unimplemented. So to shorten, you're in a though luck, that feature will never make it into metacity.
