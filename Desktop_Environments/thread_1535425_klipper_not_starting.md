### Post by SurlyLHT on 2010-09-24
This worked for me:

Start Klipper via Krunner (Press ALT+F2, enter Klipper, press Enter)

The Klipper icon will appear in the System Tray.

Right-click the Klipper icon and select Quit.

You'll get a pop-up that says "Should Klipper start automatically when you login?"

Click Start.

Next time you start KDE Klipper should be in the System Tray.
