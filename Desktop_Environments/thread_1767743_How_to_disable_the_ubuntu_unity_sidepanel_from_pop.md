### Post by robert shearer on 2011-05-26
Yeah, that is what it did for me.
Maybe unrelated but for another reason (always popped out when I moused left in anything) I changed the reveal position to top left and disabled left.

Now it only reveals if I mouse very top left,  ie: hard into the corner,  or use the keyboard.

For some reason all the other random reveal behavior ceased as well.
