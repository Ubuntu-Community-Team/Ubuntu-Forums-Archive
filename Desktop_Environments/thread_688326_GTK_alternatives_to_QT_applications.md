### Post by kahlil88 on 2008-02-05
I would like to find good GTK alternatives to some of the Qt apps that I use:
* K3B - Tried GnomeBaker and Brasero, but I didn't like the layout, and GnomeBaker didn't let me move project files into different folders if I remember correctly.
* K9Copy - Tried DVD95, but it seems to have some weird issues.
* Konqueror - I just want a tabbed file-manager.
* KPPP - Tried Gnome PPP and love the interface, but it doesn't seem to work, and I think development has been stopped.

To be fair, I'd like to suggest a good alternative that I've found:
* [Exaile]("http://www.exaile.org") - Almost identical to Amarok but MUCH smaller and lighter (takes one second to open!). Currently it's only at version 0.2.11, so it doesn't have all the premium features found in Amarok, but I'm sure it will dominate once development picks up.
