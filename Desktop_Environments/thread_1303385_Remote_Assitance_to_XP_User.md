### Post by Zalbor on 2010-08-21
Try teamviewer: [http://www.teamviewer.com/index.aspx](http://www.teamviewer.com/index.aspx)

It's free if you're not using it "professionally", and it has a linux version of the main application (sort of; it uses wine).
You can tell the windows user to download the "small version" (there will be an extra link called "join a session" under the big download button if you go to the site using windows) which is easier to use for them. All they have to do is run it and it will display a code that they can tell you to use.
