Restricted drivers are not open source and tend to have problems with some software. However, they are usually faster than the open source ati drivers. Can't hurt to try them, you can now turn them on and off very easily under "restricted drivers" in the settings of Gnome. But the restricted drivers make you use XGL instead of the nicer AIGLX, essentially making you install a separate custom version of X.

Beryl and Desktop Effects (also known as Compiz) started as the same project. Compiz was a little too conservative for some of the developers, so they forked and started Beryl. The code bases are now back together, but Compiz still goes for simplicity while Beryl goes for a excessive amount of options and customization. I've since changed to the official Beryl project packages [here]("http://wiki.beryl-project.org/wiki/Install_Beryl_on_Ubuntu_Feisty_with_XGL").
