### Post by benerivo on 2008-07-20
You can install gtk-qt-engine-kde4 from the ubuntu repositories. This is description given in synaptic...

"The GTK-Qt Theme Engine (also known as gtk-qt-engine) is a GTK+ 2 theme engine that calls Qt 4 to do the actual drawing. This makes your GTK+ 2 applications look almost like real Qt 4 applications and gives you a more unified desktop experience.

Please note that this package is targeted at KDE 4 users and therefore provides a way to configure it from within Systemsettings."
