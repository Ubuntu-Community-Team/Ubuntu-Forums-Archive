### Post by wyohermit on 2006-07-31
If you go to Applications-> System Tools -> Configuration Editor

Then under apps/panel/global there is a box to check for tool tips enabled.  Un check it.

Configuration editor may not be visible in you applications menu.  If not you can make it visible by using the alacarte menu editor.

### Post by aysiu on 2006-07-31
By default, in Dapper, Configuration Editor is unchecked from the A la Carte Menu Editor.

If you know how to use A la Carte, just check the box again. If not, press Alt-F2 and type ```
gconf-editor
```
