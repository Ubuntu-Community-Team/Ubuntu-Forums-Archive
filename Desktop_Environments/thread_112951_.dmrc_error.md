### Post by AndyCooll on 2006-01-05
Had this myself before.

Try changing the permissions of the home folder concerned allowing "Group" and "Other" read and execute permissions. I believe it is to do with the fact that all your settings are stored in the Home folder (and usually hidden) and on start-up many programs need access to that folder during the start-up process, which is why the default is usually set to 755.
