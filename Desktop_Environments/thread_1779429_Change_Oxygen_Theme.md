### Post by Zorael on 2011-06-10
System Settings -> Workspace Appearance -> Window Decorations.

Install the [**kdeartwork**](apt://kdeartwork) package to get a bunch more themes, color styles and emoticons. Also try the **Get New Decorations** button there to download stuff from [kde-look.org](http://kde-look.org).
