### Post by Frogs Hair on 2011-05-12
Another forum member helped me with this and it should work with other icon sets.

Navigate to the folder where your icons are stored , either .icons in hidden/home folder or usr/share in the file system .

Open the 128x128 Apps folder and locate the icon for the switcher and copy to home.

Rename the icon workspace-switcher.png and move back to the original folder , logout and login .

This worked with the AwOken icon set.
