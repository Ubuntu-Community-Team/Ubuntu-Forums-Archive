### Post by renzokuken on 2008-10-20
use a live CD and gparted to wipe and partition the drive first, then install Ubuntu onto that clean partition.....

.....although the install process should give the option to wipe the drive and automatically set it up for you

### Post by neilengineer on 2008-10-20
Format the partition you want to install first.

When install using Ubuntu LiveCD, at the step of partition select, there is something like a checkbox. Check it to format the partition.
