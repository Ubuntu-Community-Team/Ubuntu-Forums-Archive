### Post by mcduck on 2009-11-03
Just put any files you want to use as templates into ~/Templates, and they'll show in the right-click menu's Create Documet-menu.

For example if you save an empty .odt document as"New OpenOffice Document.odt" in the Templates directory you'll get a new option "New OpenOffice Document" in the Create Document-menu (the file name extnesion will not show in the menu entry).
