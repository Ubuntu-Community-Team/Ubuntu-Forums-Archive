### Post by stuporglue on 2005-10-28
> When i try to change it via the dialog box, I can't access the Azureus folder (presumably because it hidden and doesn't show me hidden files)

Do you know the location of the actual Azureus program? It's probably not in a hidden file. You can navigate to any folder, hidden or not, by opening a "select folder" type dialog box then typing /path/to/folder/you/want/to/open. 

For example, if I wanted to edit a file that was in a hidden folder named .folder (notice the dot!) in my home directory, I would type /home/username/.folder then hit enter. I would still be in the open file dialog box, but inside the hidden folder now.
