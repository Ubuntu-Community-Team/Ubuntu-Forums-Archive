### Post by chewearn on 2008-04-03
I'm behind a firewall, which blocks youtube, so I can't see your link.

But, you don't need beryl because it's obsolete.  Since you have gutsy installed, you already have compiz installed by default.

First, make sure you have the video GPU that support 3D graphics.  Then:
Go to System > Preferences > Appearance > Visual Effects > Select an option that is not "None"

### Post by Wyrmboy on 2008-04-04
For Added Effects run this command in the Gnome Terminal

sudo apt-get install compizconfig-settings-manager

It will install the Compiz Fusion Manager will allow you to set up this options.

After it has been installed it can be found at System>Preferences>Advanced Desktop Effects Settings
