### Post by Sef on 2005-12-30
:oops: Nevermind.  I figured out how to do it.  For those who don't like to log in manually, this is how to do it automatically:   System ------> Administration -------> Log in Screen Set Up -----> Automatic Log in -------> Click on 'Log in a user on the first boot up.' -------> Scroll down and highlight name to automatically log in -----> Click on Close.
