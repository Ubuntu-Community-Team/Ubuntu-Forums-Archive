### Post by teet on 2006-07-28
As far as I know, you can get rid of the ubuntu firefox...it will probably also uninstall ubuntu-desktop but that's just a metapackage.  You'll also want to point to your install of Firefox in:

System --> Preferences --> Preferred Applications

so that, when you click on a link in Gaim (or whatever) it will open up the correct firefox.

May I ask why you're switching from the ubuntu version?  1.5.0.5 is already in the ubuntu repositories.  It makes things a lot easier to stick with the ubuntu version in my opinion.

-teet
