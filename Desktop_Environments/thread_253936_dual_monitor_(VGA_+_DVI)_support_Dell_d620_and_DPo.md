### Post by phansen on 2006-09-09
Correctemundo :-)

Same Bus ID and Change Monitor layout to pipe signal out on VGA and DVI port - now Xinerama works.. 

Section "Device"
Identifier "Card0"
Driver "i810"
Screen 0
Option "DevicePresence" "true"
Option "MonitorLayout" "CRT, DFP+CRT"
BusID "PCI:0:2:0"
EndSection

Section "Device"
Identifier "Card1"
Driver "i810"
Screen 1
Option "DevicePresence" "true"
Option "MonitorLayout" "DFP, DFP+CRT"
BusID "PCI:0:2:0"
EndSection
