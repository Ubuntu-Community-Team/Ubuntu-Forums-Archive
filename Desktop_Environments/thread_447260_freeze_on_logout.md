### Post by kr33z on 2007-05-28
This seems to be an fglrx driver issue.

Try adding the following line to etc/X11/gdm/gdm.conf :

AlwaysRestartServer=true

Or you can use the login screen configuration tool (gdmsetup) and check the 'restart x server with each login' option.

This worked fine for me (ati x200m / fglrx)
