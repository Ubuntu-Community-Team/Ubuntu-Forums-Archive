### Post by Dryer Lint on 2006-05-01
If your CF card is FAT-formatted (which is very likely) there are a number of recovery tools available. I had success in recovering deleted files with the tools from this site:

[http://www.pcinspector.de/file_recovery/uk/welcome.htm](http://www.pcinspector.de/file_recovery/uk/welcome.htm)

They're free, but Windows programs...

AFAIK there are no undelete-programs for files on EXT3 (= linux formatted) drives, so be very careful what you delete...
