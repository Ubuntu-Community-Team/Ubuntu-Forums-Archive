### Post by cuhor on 2005-02-10
I tried to add new lauchers and folders to the "Applications" menu with the right-click method, but nothing showed up in the menu after I restart Gnome. 

When I open applications:/// in Nautilus, all the new lauchers and folders are already there. They can start the corresponding programs when I click them. 

Does anyone know why the menu is not able to load these new items?

Thanks!

---

### Post by jeremy on 2005-02-10
Came across this, it works for me:-
When creating or editing a launcher, check the "Advanced" tab. If "Try this before using" is left empty, the launcher will not appear in the menu. If you type in "true" (no quotes), it is shown.
