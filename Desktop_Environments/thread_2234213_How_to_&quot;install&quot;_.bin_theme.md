### Post by deadflowr on 2014-07-13
The theme says emerald.
Unfortunately I believe emerald does not work on the newest versions of Ubuntu w/unity.

I does work with gnome classic though.

For the record, emerald is a window decorator for compiz.
To use you would install emerald, and then change the window decorator setting from whatever it is to emerald.

To that end, unity on 14.04 does not use the built in window decoration plugin for compiz, but instead uses its own.
So changing the window decoration in the traditional manner, will have no effect on unity.

for what it's worth though, [cavsfan's howto install emerald]("http://ubuntuforums.org/showthread.php?t=2192044") is here.
And it does work, at least for gnome-fallback with compiz.
