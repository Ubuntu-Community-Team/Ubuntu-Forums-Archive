### Post by BobW on 2006-08-27
I found that using gnome-volume-manager-gthumb for photo import was messing up my photos.

This software it trying to rotate vertical photos, but it doesn't do it correctly. It rotates the image, but doesn't update the exif data. The result is that any exif aware image viewer will display the photo incorrectly.

I wasn't able to find any way to shut thsis off and Ifinally hade to switch to using a different program for import.
