### Post by heimo on 2005-08-16
To install software on Ubuntu, use Synaptic package manager (In administration menu). Search mozilla-thunderbird, select for installation and hit Apply. Or alternatively, type this on terminal:
 ```

sudo apt-get update
sudo apt-get install mozilla-thunderbird
```
