### Post by JThundley on 2012-08-10
I'd like a nice gui tool also. My workaround so far is to use mtpfs. To use, install it with sudo apt-get install mtpfs then run it as a regular user like this: mtpfs /mnt or wherever you want to mount it. You should then be able to browse your mtp device on your filesystem. This isn't perfect though. I've noticed it's slow and can screw up copying directories sometimes.
