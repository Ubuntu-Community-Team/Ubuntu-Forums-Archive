### Post by 3Miro on 2010-06-23
This depends somewhat on the specific settings/environment, but suppose you want to have Gnome and KDE together for userG and userK.

1. Install Ubuntu.
2. From synaptic, install kubuntu-desktop (now on login every user can select KDE or Gnome).
3. Create users (userG and userK)
4. Login as userG selecting Gnome, then logout.
5. Login as userK selecting KDE, then logout.

Now you have the two users with the corresponding environments setup as default.
