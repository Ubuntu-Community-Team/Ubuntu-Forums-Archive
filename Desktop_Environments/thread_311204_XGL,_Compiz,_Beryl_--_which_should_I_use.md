### Post by kragen on 2006-12-05
My opinion is that beryl crashes too much and feels tacky. Compiz is as stable as a rock, and the default settings on the plugins gives a desktop that feels nice to use. Compiz also has an easy way of turning off the GL desktop (for example when you want to run a GL game - compiz and beryl both seem to halve the framerates somehow, I hope thats not permenant...)
