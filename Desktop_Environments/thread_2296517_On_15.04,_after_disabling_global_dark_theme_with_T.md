I'm using the default Adwaita theme in Gnome. 

After I enabled the global dark theme in Ubuntu Gnome 15.04 and then disabled it again, some window title bars are still staying dark gray, while others have been restored to Adwaita's default white.

Any idea why this is? How can I fix it?
