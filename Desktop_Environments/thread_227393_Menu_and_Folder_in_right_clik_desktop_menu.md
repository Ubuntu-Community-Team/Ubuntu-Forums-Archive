### Post by ardchoille on 2006-08-01
You can use nautilus scripts, but this is limiting. You can also switch window managers in gnome, which is so much better.

I didn't like the desktop right-click menu either and then I learned how to switch window managers in gnome. Metacity is the default wm in gnome and is, IMO, the worst pile of junk I have ever seen. 

I wrote a tutorial about how to switch window managers in gnome, you can find it here: [https://help.ubuntu.com/community/ReplaceMetacityWithOpenbox](https://help.ubuntu.com/community/ReplaceMetacityWithOpenbox)

Openbox has a right-click desktop menu that is extremely customisable. Below is a screenshot of mine. As you can see, I have set up my desktop right-click menu (which is controlled by openbox) to mimic the gnome menus.
