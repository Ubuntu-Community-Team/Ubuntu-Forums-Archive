### Post by Jawbreaker4Fs on 2006-04-14
in case anyone has this problem in the future, it can be solved by disabling the framerate buffer (I think that's what it was?). When you get the "boot:" prompt with the install cd, instead of just pressing enter or typing "install" you should hit I think either F1 or F7 and it will tell you what to do.
