### Post by thenovelist on 2012-03-16
Okay, I think I have found my way round this.  As it was installed within Windows I copied all of the Ubuntu file to another USB drive.  I turned it off and uninstalled Ubuntu, then re-installed it.  I then restarted Ubuntu which then re-instated the boot sequence on C drive.  It worked fine, then I rebooted back to Windows.  I copied over the new install from my old Ubuntu and restarted it.  All hopefully is working okay and back to normal.  I have lost no updates, software or folders!!
I had tried for hours copying boot sequences and doing all that was written on the internet, but NOTHING worked.  So out of shear frustration I came up with what I did.  Hope it helps anyone else that has a problem with the boot problems.
