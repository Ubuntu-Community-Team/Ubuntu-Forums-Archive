### Post by grizzlefist on 2007-02-08
On my default Xubuntu install, there is an option in the Xfce menu, under Accessories, called Appfinder which lists a fair number of applications, some of them not found under other categories of the menu.

If you can't find something that way, you can most likely find it in [FONT="Courier New"]/usr/bin[/FONT].

As for creating desktop shortcuts, if you are using Xfce Desktop, right-click and select "Create Launcher...".  You can type in anything you want for name, comment, etc., but for command, type in the path to the application you want to run.

For instance, my media player, Quod Libet, is at [FONT="Courier New"]/usr/bin/quodlibet[/FONT], so that's what I'd type in the command text box if I wanted to create a shortcut to it.
