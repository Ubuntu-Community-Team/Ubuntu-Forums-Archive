### Post by davidgypsy on 2005-11-14
[quote=ConnyBK]And where is this main menu configured? If I want to customize it, how can I do that?[/quote]

Right click on the taskbar where you want to put the menu, then choose the "add to panel" button, then choose "main menu" under "launchers and menus". Voila! :D

### Post by jasay on 2005-11-14
If you want to edit the menu use Applications>System Tools>Application Menu Editor (or right-click on the applet > Edit Menus).  This editor only works on the applications menu, not the places or system menus.
