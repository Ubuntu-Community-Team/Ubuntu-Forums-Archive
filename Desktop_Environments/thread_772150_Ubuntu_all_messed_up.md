### Post by bingoUV on 2008-04-28
To determine whether it is a user configuration issue, or a system wide issue, create a new user and login through that new user. See whether it behaves normally. Of course it will not remember settings from earlier user but problem 3 and 4 might get fixed. 

In case they are fixed something is messed up only in your home directory and you can try copying over your configuration files one by one and hopefully get an almost completely configured system.


Otherwise it is a system wide issue and we will tackle it later.
