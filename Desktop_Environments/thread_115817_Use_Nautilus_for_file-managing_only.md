### Post by Vinze on 2006-01-11
Hey, I rund Xfce on Ubuntu, and I really like it. As file-manager I use xfe, but it really isn't perfect (nor does it look good). Therefore I like to use nautilus, but whenever I start Nautilus, it will also take over windowmanagement. Is there any way to start nautilus as filemanager only? Thanks in advance.

---

### Post by Thirsteh on 2006-01-11
Yes, launch it like this:

nautilus --no-desktop --browser


--browser is optional. --no-desktop is what keeps it from "taking over" your desktop :)
