### Post by [h2o] on 2006-09-02
What program were you trying to install/run?
You can't run "Windows programs" under Linux unless you run them through an emulator (Wine). But, luckily, there are probably a lot of Linux programs that does the job just as well.

If you really want to run your .exe-file then install Wine by going to Programs->Add/Remove  and then type "Wine" in the searchbar at the top and click Wine and then "install".

Stuff might be named a bit differently depending on the language of your installation.
