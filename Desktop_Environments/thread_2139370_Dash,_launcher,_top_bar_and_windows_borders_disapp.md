### Post by NBPX on 2013-04-26
I tried to install the ATI proprietary drivers on my laptop, but the edges of the windows, the launcher, the dash and the top bar disappeared in Ubuntu 13.04.

I tried to install the driver again but only x.org window borders are back to normal.

It's the same problem we had in Ubuntu 12.04 and 12.10, but worse. It was enough to to reinstall the driver x.org everything returned to normal on Ubuntu 12.x. Now it no longer works.

This is a bug very well known and very common. Why Canonical keeps those drivers without worrying about to protect users?

### Post by NBPX on 2013-04-27
First I had to install the Compiz Config Settings Manager. Then through Compiz Config Settings Manager, enabled the Ubuntu Unity Plugin.
