### Post by ajgreeny on 2008-04-25
Install compiz-config-settings-manager if you don't already have it and then look for the plugin "Move window" right down the bottom, nearly, in Window management.  Make sure that is enabled and you may be OK. Perhaps also make sure "Snapping Windows" is **not** enabled as this holds windows to the sides of the screen, and is, I think, a flippin' nuisance.
