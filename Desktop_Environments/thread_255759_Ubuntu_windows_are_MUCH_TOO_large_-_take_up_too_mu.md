### Post by mssever on 2006-09-12
As far as the windows go, try switching your screen resolution to at least 1024x768. Some windows don't play nice with lower resolutions. The devs should make sure that all their windows will play nice, but they don't. Also, you can try Alt+dragging a window to see all of it's contents. I don't remember if this feature is on by default in Ubuntu or if I turned it on myself.

You also might want to try different themes, which might give you more screen space. Check out [http://art.gnome.org](http://art.gnome.org) and [http://gnome-look.org](http://gnome-look.org).

As far as the Open dialogs go, they're dependent on the toolkit used by the program. So if you want KDE dialogs, use KDE apps (in that case, you might as well switch to Kubuntu). Gnome's open/save dialog has grown on me over time, and while it's not my favorite, I no longer think that it's terrible. So giving it time might help.
