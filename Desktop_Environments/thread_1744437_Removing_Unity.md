### Post by 3Miro on 2011-04-30
If your hardware cannot run Unity, then there is a good chance that you will not be able to run Gnome 3 either.

Check System -> Admin -> Additional Drivers to see if this is a simple issue of installing a driver. Unity may work if you just enable some proprietary driver (depending on your video card).

If there are no drivers available, then Ubuntu Classic No Effects and Unity 2D are your safest bet.
