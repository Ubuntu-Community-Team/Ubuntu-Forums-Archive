That's not a noob question! That's actually a very legitimate question! That called Unity, and Unity is a shell that runs on top of Gnome. (unity is the new default desktop in 11.04, as long as you have hardware acceleration)
If you want to switch from the Unity desktop to the Classic desktop, log out, on the bottom of the log in screen there is a drop down box, in that box choose "Classic" and you will be back in familiar territory.  :D
