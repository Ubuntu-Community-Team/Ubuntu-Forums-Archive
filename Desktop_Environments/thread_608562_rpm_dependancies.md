### Post by ardchoille42 on 2007-11-10
You don't install .rpm packages in Ubuntu.. Ubuntu is .deb based.. and using alien to convert a .rpm to a .deb is quite dangerous.

Follow these rules for a safe and stable system:
1) Install packages only from official repositories
If you can't do that, then:
2) find a package that was made for your distro
if you can't do that, then:
3) Compile the app yourself
if you can't do that, then:
4) Find another app to perform the task
if you can't do that, then:
5) switch distros
This is the ONLY way to maintain a safe, stable, and supportable system and I've never had to venture beyond step 4 (above) in any of the 17 distros I have used.

I've been using Linux since 2001 and Ubuntu since 2004. I've seen lots of people hose their system by installing foreign packages.
