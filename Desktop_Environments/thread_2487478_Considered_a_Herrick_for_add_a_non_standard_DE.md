### Post by Holger_Gehrke on 2023-06-06
To each their own. There's several official 'flavours' of Ubuntu which differ mostly by the DE they use, so not using Gnome is quite normal (actually, there is a vocal part of the community which dislikes Gnome because - in their view - Gnome is dumbing the desktop down by not giving the user enough options). If you want to use something else as your desktop, feel free to do so. Free software always has to also mean freedom of choice which software to use - or not.

One warning, though: the more exotic your choices of DE or WM are, the fewer people will be able to help you if you find yourself in trouble ...
