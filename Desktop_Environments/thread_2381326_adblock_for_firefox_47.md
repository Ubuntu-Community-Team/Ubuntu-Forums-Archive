### Post by Holger_Gehrke on 2017-12-30
And if it is a typo and you are on 57, you can just select "Add-ons" from the menu or type "about:addons" into the address bar or hit ctrl-shift-a. A page will open in the browser window. The second item of the menu on the left of the page should be "Extensions" or "Add-ons" or something to this effect (it's "Erweiterungen" on my machine, since it's set to German). There should be a search field in the top left of the page. Enter "adblock" and click on the looking glass. You should have some choices at that point ...

### Post by Holger_Gehrke on 2017-12-30
According to the Firefox version history at Wikipedia ff 47 is about two years old. I don't think that a machine that can handle a 2 year old ff will have trouble with a more current release. In your place, I'd update at least to version 52, which is an extended support release and will receive security fixes by Mozilla to August 2018. 

To give you a point of reference: I've run ff 57 on a single core Pentium4 notebook from 2002 with 3 GHz Clock and 1 GB of RAM with Lubuntu 16.04, and while it is notable slower than my main machine it does work and is actually a bit faster in displaying pages than the older ff (55 I think; I only use that machine every now and then when I need a serial or parallel interface or -- God forbid - a floppy drive) I had on that machine before. The limiting factor seems to be memory more than processing speed; both big browsers are absolute pigs when it comes to memory consumption.

Holger
