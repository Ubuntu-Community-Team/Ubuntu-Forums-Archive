### Post by Bob D. on 2005-04-27
You need to install the SpellBound extension for Firefox.


 It is a 2-step installation. You need to install the libraries and dictionary(s) with Firefox launched as root. The SpellBound extension itself should be installed with Firefox launched from your non-root user account.

Complete info can be found here: [http://spellbound.sourceforge.net/]("http://spellbound.sourceforge.net/")


 Bob
