### Post by Keith_Helms on 2015-08-09
In order to receive updates, you'd have to have gotten the newer kernel from a repository of some kind and not just manually downloaded it.  The Ubuntu folks maintain a PPA that has newer kernels in it.  You can read about it here:

[https://wiki.ubuntu.com/Kernel/MainlineBuilds](https://wiki.ubuntu.com/Kernel/MainlineBuilds)
