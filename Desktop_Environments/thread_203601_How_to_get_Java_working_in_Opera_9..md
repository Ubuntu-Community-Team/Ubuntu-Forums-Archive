### Post by RobTN on 2006-06-25
I found a way to get Java working in Opera 9.

Run Opera by using this command:

```
opera --debugplugin --debugjava
```

I don't know if it's smart or even recommended to run Opera in this manner, but Java does work.  (Assuming you have Java installed, of course.)
