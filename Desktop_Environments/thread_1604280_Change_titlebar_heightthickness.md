### Post by 3Miro on 2010-10-23
How are you trying to edit the theme. Your best bet is to copy the Ambiance theme folder from /usr/share/theme to your home folder .themes (that is dot-themes) you can press Ctrl+H to see the "hidden" files that start with dot-something. You can create the folder if it is not already there.

Then you should edit the theme as Ambiance-mine or something like that (that would be the folder name as well as the index.theme file and maybe other files as well).
