### Post by jespdj on 2007-10-22
Install the Compiz configuration settings manager:
```
sudo apt-get install compizconfig-settings-manager
```
After installing, you'll have a menu item System / Preferences / Advanced Desktop Effects Settings, where you can enable and disable and configure all the Compiz plug-ins.
