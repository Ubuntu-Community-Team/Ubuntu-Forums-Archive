### Post by Pogeymanz on 2008-11-13
Nope.

A DE is just like any other program. If you aren't running it, it isn't using any resources.

---

### Post by snova on 2008-11-13
No. It'll take up some disk space, though.

Actually, if you run programs from each desktop environment at the same time, you might use a bit more memory than usual, since Gnome and KDE depend on different (and both are large) libraries as their widget toolkits.
