 Open up Package Manager Synaptic.
Pref > Repositories > Settings > Show disabled
and check everyone.

Then add this repository 
deb [ftp://ftp.nerim.net/debian-marillat/](ftp://ftp.nerim.net/debian-marillat/) unstable main
[This is a repository line]

After the update, 
install w32codecs and vlc.

Vlc wont appear on the menus so you will have to run it through alt f2 or create a shortcut.
If you wnt video streaming install mozilla-mplayer.
