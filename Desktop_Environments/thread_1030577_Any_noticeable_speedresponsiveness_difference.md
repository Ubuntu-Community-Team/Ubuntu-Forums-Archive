### Post by mcduck on 2009-01-04
Yes, there is small but noticeable difference in all disk reads/writes, caused by running inside a virtual file system.

The difference isn't very big but since waiting for data from disks is usually the biggest reason for delays experienced in normal desktop use it's something that's easily noticeable.

Of course if you feel that programs are loading quickly enough now you shouldn't worry about it, as apart from the disk access everything else runs just as fine as it would run in normal Ubuntu install.
