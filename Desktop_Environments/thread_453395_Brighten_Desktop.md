### Post by aktiwers on 2007-05-24
Open Terminal and type:
```
 xgamma -gamma 2
```

Just change the number to something between -9 and 9 (i think thats the limits). 
1 is how your screen is now.

When you find whatever you like, you can set it to run on startup.
[http://ubuntuforums.org/showthread.php?t=414180](http://ubuntuforums.org/showthread.php?t=414180)
Explains how, pretty easy.
