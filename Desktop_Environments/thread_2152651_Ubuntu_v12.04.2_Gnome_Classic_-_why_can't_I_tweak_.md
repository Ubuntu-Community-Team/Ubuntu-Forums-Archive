### Post by kansasnoob on 2013-06-10
> **zombifier25 said:**
> Try Super + Right click, or Alt + Super + Right click.

+1!

If you read down just a bit here:

[https://help.ubuntu.com/community/PreciseGnomeClassicTweaks#Changes_to_Panels_and_Menus](https://help.ubuntu.com/community/PreciseGnomeClassicTweaks#Changes_to_Panels_and_Menus)

You'll see;

> Note: While this isn't written with Compiz in mind you should know if you're using 'gnome-panel' in a standard Gnome classic session you'll need to press both the Alt & Super keys at the same time while right-clicking on the panel/applet you wish to edit, move, or remove. (The Super key is typically the one with the Windows logo). 

The Precise log-in options include both Gnome Classic and Gnome Classic (no effects). The prior uses the Compiz window manager while the latter uses the Metacity window manager. That's why [my original Forum's thread]("http://ubuntuforums.org/showthread.php?t=1966370") was titled **12.04 LTS / Precise Classic [COLOR="#FF0000"](No effects)[/COLOR] Tweaks and tricks **.
