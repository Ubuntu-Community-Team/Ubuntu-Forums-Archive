### Post by zombifier25 on 2013-02-13
There is Ubuntu Minimal, which contains only the bare Ubuntu system and nothing else, From there you can work your way up, installing a very lightweight WM and stuffs.
Instructions [here](http://www.psychocats.net/ubuntu/minimal).

If you don't strictly need a Ubuntu-based distro, there are some Linux distros even more lightweight, such as [Puppy Linux](http://www.puppylinux.org/) or Damn Small Linux. Setting them up is a little more difficult, but they're ridiculously small and light.
