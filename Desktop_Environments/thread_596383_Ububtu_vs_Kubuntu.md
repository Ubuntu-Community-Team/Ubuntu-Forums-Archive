### Post by Zorael on 2007-10-29
> **stlcoptony said:**
> As I understand it, Kubuntu 7.10 is missing things that are in the default download of Ubuntu 7.10, like Compiz and so on.

Of course it's missing things! Else it'd be Ubuntu instead of Kubuntu. :>

Ubuntu doesn't come with Ark, Kopete, Konversation, and other K-applications, not to mention KDE. While Kubuntu doesn't come with Gnome, gThumb, Gaim, etc.

The repositories are still the same, so install whichever you want, then add whatever's not included in the default installation. Such as Compiz. Or Gnome, for that matter.
