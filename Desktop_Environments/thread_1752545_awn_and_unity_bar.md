### Post by deltacomp on 2011-05-07
Hello, yes the unity bar can be disabled by going to system->administration->login screen, hit unlock, type password, then select Ubuntu classic. 

Yes the cube be enabled in compiz settings manager the same way you set it up the first time will work.
