### Post by geoffjay on 2009-04-06
Everytime I load an instance of conky on my desktop Emerald picks it up as a window and applies the drop shadows. I know it's possible to eliminate the shadows all together but I would prefer to just exclude conky from Emerald because the rest of my setup looks stupid without the nice shadows. It's probably not possible but if anyone knows of a way I would love to hear it. Thanks.

---

### Post by mcduck on 2009-04-07
Yes, it's possible. If you have the CompizConfig Settings Manager installed you can define which windows get decorations (and/or shadows) and which don't in the Window Decoration-plugin's settings.
