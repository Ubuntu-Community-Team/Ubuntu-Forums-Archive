### Post by Copper Bezel on 2011-10-23
Well, it's not a zoom button, but a maximize button, so that can't be changed. You're not running OSX. = )

The launcher autohide can be changed in compizconfig-settings-manager. Look it up in the Software Center, and then launch it from the Dash once you have it installed. You can probably just search for "ccsm" both places. Once you open it up, find the Ubuntu Unity plugin and change the Hide Launcher behavior to Never.
