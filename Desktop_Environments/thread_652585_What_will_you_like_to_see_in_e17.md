### Post by smartboyathome on 2007-12-31
Go to accounts > manage, and click add in the window that pops up. on the next window that pops up, click the drop down menu to add IRC. Now, set the server to irc.freenode.net, and choose a screen name (you will know later if the screen name is taken). Now, click save (no need for a password unless you have registered on freenode.net before and are using that account). It should log you in and a window should pop up saying that your name isn't taken. Follow the steps on Freenode.net to register with them. Once registered, use your password in the manage accounts portion to get in. Now just add a chat!
