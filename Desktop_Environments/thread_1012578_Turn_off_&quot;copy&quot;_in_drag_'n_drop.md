### Post by insub2 on 2009-06-29
I found this thread while looking for answers.  I also found this

[QUOTE=held]
modifiers:
Shift+Drag: Move
Ctrl+Drag: Copy
Ctr+Shift+Drag: Link
(Note how the mouse pointer changes)

This is true for both desktop and thunar, however each seems to default to its own liking.[/QUOTE]
Found at [Xfce Community]("http://forum.xfce.org/index.php?topic=4660.0")
