### Post by NunoLava1998 on 2015-02-08
Corruption or overheating, Check the ubuntu website and the file that you downloaded's md5 and see if they match.
If they match the md5, then its the computer's fault.
If it is not matching the md5, then ubuntu got corrupted.

### Post by mörgæs on 2015-02-08
A failed download is unlikely. Then the install wouldn't have succeeded in the first place.

More likely is that Ubuntu is too heavy for the graphics processor, and I suggest a fresh install of X/Lubuntu.
