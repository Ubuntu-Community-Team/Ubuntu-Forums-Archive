### Post by psychicdragon on 2005-10-30
Are you using rox to actually draw the desktop? Because xffm doesn't support desktop icons.

If you just want to change the launcher in the desktop menu, run **xfce4-menueditor** and you can make it point to xffm instead of rox. You can change the launcher on the panel by right clicking on it > Properties.

Have you ever tried xffm? It is honestly one of the ugliest, strangest applications you'll ever see. Rox, while a little odd, is a pleasure once you get used to it.
