### Post by schilcha on 2007-03-13
I hope I understood your question correctly. In short, you ask whether it is ok to run e.g KDE-Applications under gnome?

This is definitely all right. I'm running ubuntu using gnome and do use some KDE-applications (e.g. k3b). The first time I installed a KDE-app, I also had to install an armada of KDE-libraries (as you suspected in your post). The same is true for the opposite direction or also for running applications under xfce, icewm, windowmaker, whatever your favourite window-manager is.

In fact one of the power of X11 is the separation of what is happening inside the window from what is happening around it.

You might loose some functionality of applications that are developed towards a single desktop environment, such as icons in some panel, maybe some special copy/paste or drag/drop stuff. But I've never noticed that an application does only run in a single environment.

Hope that helped.
