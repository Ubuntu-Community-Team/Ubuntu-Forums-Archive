### Post by Jucato on 2006-05-26
You can either:
1. Use a shortcut key to make the embedded konsole pop-up when you need it (Settings > Configure shortcuts); or

2. Add the icon/command to a toolbar (Settings > Configure Toolbars); or

3. Using either of the methods above, make the emulated/embedded terminal visible and save your own profile. This will make that terminal a permanent part of the profile until you save the profile again, without the terminal. (Settings > Save Profile)
