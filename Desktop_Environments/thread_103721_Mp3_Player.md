### Post by Perfect Storm on 2005-12-14
xmms

Open the terminal and write:

```

sudo apt-get install xmms

```

To make it to play music CDs when inserted. Go to **System** ---> **Preferences** ---> **Removable drives and Media**.
Select Multimedia tab. In **Audio CD disc** write **xmms -p /media/cdrom**
