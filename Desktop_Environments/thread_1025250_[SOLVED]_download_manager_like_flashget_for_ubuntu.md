### Post by LiL T on 2009-01-06
If you want to use flashget with FF install wine, then download the latest version of flashget and install it with wine. Now download the flashgot plugin for FF, once you've restarted Firefox right click on somewhere in the browser window select FashGot Options > More Options and a window will popup. Select the General Tab and it will have Wget listed in the download manager box but it won't let you change to flashget, so click Add and enter Flashget as the name click ok. Now navigate to the directory where flashget is installed it will be something like '/home/User/.wine/drive_c/Program Files/FlashGet/' Select the flashget.exe file and click open and you're done.

Now you should be able to use it just how you would in windows firefox.
