If the XP is in a virtual machine (eg in VirtualBox or Qemu), it will still need antivirus. It won't be nearly as easy for hackers to break into it (take it from one who's tried) but virii can still get into it via infected .exe and .dll files. I recommend AVG Free. I use that on XP myself.
