### Post by ajgreeny on 2010-12-03
Why is it a lot of work to change it manually?

Just right click on a .doc file and go to properties, then click on the icon and change it to what you want.  In future all .doc files will show the M$ doc icon and all .odt files will show the OOo icon.  You do not have to do that for every file, just one of the type.

Interestingly, my system shows the two types of file with the different icons, and I did not make any changes to the defaults.  I wonder why yours is different?
