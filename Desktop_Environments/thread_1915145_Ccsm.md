### Post by Frogs Hair on 2012-01-25
If you can open the terminal with Crtl + Alt + T run the following command . ```
unity --reset
``` If you can't get into the terminal restart the computer with the power or restart button . Choose the recovery console from the session list below the greeter box on the bottom panel . A terminal will open and you can run the same command .  You may have to try the command more than once . See the link if that fails . [http://www.tuxgarage.com/2011/04/missing-top-and-side-panels-in-unity.html](http://www.tuxgarage.com/2011/04/missing-top-and-side-panels-in-unity.html)
