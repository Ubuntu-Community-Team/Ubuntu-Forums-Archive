### Post by KaiJordanDay on 2010-02-20
For Desktop Effects on my dual-monitors I have to forcefully start it using compiz --replace &

This enables desktop effects for my laptop, and dual screens.

Any possible way of doing this, on startup without manually typing it in every time through the terminal?

---

### Post by howefield on 2010-02-20
Add the command to your startup sessions ?

System > Preferences > Startup Applications.

Type whatever you want into the Name and Comment fields, and put compiz --replace in the Command field.
