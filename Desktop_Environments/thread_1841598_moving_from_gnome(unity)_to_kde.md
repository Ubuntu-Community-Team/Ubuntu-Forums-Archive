### Post by graphius on 2011-09-09
I have problems/issues with Unity, so I have decided to move to KDE. I thought I would share some of my experiences for others.

I installed Kubuntu ```
sudo apt-get install kubuntu-desktop
```
and proceeded to customize my desktop. My opinions:

[LIST]
[*]I prefer evolution to kmail or thunderbird
[*]KDE is much more customizeable than Unity, or even gnome classic/gnome2
[*]GTK apps are a pain to get looking good. It seems font style definitions don't carry over from gnome to kde
[*]It is better to use kde native apps, but transferring is a pain. for example moving bookmarks from gftp to ??
[*]I seem to find many "third party" apps are more at home in gnome than kde. For example, I have used gallery remote to upload images to my website. It looks ugly in my kde desktop.
[*]I miss scripts for nautilus. In fact I think overall, nautilus is a better file manager than dolphin.
[/LIST]

Maybe I am just more used to the old gnome way of doing things, Maybe as I learn the desktop, things will get easier. Don't get me wrong though, I really like the overall kde experience, and I think it is worth a few bumps to learn.
What do others think?

Have you tried System Settings > Application Appearance > GTK+ Appearance?

> It is better to use kde native apps, but transferring is a pain. for example moving bookmarks from gftp to ??

Why not just use gftp it you like it better?  You can also simply enter an ftp:// URL in dolphin and connect that way.  I especially like the support in KDE for the fish:// protocol which can browse remote file systems on any machine with an SSH server. 

> I seem to find many "third party" apps are more at home in gnome than kde. For example, I have used gallery remote to upload images to my website. It looks ugly in my kde desktop.

See above re QTcurve.

> I miss scripts for nautilus. In fact I think overall, nautilus is a better file manager than dolphin.

I use bash for scripting so I can't comment on this.  I can't recall a time when I'd want to script a file manager, but YMMV.  I just started using Krusader which is a full-featured two-pane file manager.   You might give that a look.  

I'm not someone who is obsessive about the appearance of my desktop, so I may be more willing to accept visual inconsistencies that make others wince.  I find the stock KDE environment fairly decent, though I get rid of the standard widgets that appear on the desktop, switch to Oxygen for the theme, and change my wallpaper.  That's about it for customization for me.
