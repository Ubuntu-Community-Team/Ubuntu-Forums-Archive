### Post by Jucato on 2006-08-05
Actually you should use:

```
sudo aptitude update
sudo aptitude install kubuntu-desktop
```

However, you should note that this will install Kubuntu over Ubuntu, which means you'll basically have both GNOME and KDE on the same system, which could be good, depending on your needs/wants.

However, if you want a completely clean Kubuntu/KDE system, I suggest you download the Kubuntu installer (Desktop CD or Alternate Install CD) instead.
