### Post by HermanAB on 2005-12-01
There are different profiles for file and web browsing.  

First you have to load the relevant profile, either by starting Konq in the correct mode to begin with, or by loading it using Settings, Load View Profile.  Only then, should you navigate to the page that you want to use as the default and save the view profile using Settings, Save View profile.
