### Post by Darrin on 2005-11-20
I have Gdesklet installed. I would like to have it auto start when Ubuntu loads. How do I do that?

---

### Post by aysiu on 2005-11-20
Start it. Go to System > Preferences > Sessions > Automatically Save Sessions. Then log out.

---

### Post by kperkins on 2005-11-20
Go to System > Preferences > Sessions > Startup Programs Click Add, and add gdesklets. Close.  Next time you start up gdesklets will start automatically. 
You can do this with any program, by the way.
