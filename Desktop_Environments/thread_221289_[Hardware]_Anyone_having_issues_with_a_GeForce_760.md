### Post by mkw87 on 2006-07-25
I have an Opty 165 with the latest dapper and 64 bit kernel, as well as an eVGA 7600GT.  I have had no problems running it at all (xgl/compiz is a different story, but thats eye candy).  I have been able to play many games fine as well.  How did the machine hang?  Any errors?  Did you install the nvidia-glx drivers and change "nv" to "nvidia" in your xorg.conf file?  

Its also possible you got a bad card, the 7900GT's had quite a few problems, the 7600GT's seemed to be OK but every once in a while you see a bad one.
