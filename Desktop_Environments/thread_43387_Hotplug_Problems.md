### Post by cwaldbieser on 2005-06-21
Here are some things you can try:

1) Do you have a USB drive plugged into the computer?  Try unplugging before rebooting.  If It is not plugged in, try plugging it in before rebooting.

2) Try hitting CTRL-C when it seems to freeze-- it might skip this part.

3) Can you boot into failsafe mode?  If you can, maybe then you can check the output of dmesg to see if there was some kind of error.
