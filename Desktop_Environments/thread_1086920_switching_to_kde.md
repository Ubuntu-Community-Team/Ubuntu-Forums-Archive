### Post by taurus on 2009-03-04
Applications -> Accessories -> Terminal
```
sudo apt-get update
sudo apt-get install kubuntu-desktop
```
Once it's done installing, log out and at the GUI login screen, pick KDE as your window manager and log back in.  You can set it as your default window manager if you wish.

### Post by Saunter on 2009-03-04
After installing Kubuntu, you can select which desktop you want each time you login.  At the login screen, go to the lower left corner and click on "Select Session", then choose which desktop you want.

You might also use Synaptic to get the KDE4 desktop.  The next version of Kubuntu (9.04, due in April) will include the KDE4.2 desktop, which is worth waiting for.
