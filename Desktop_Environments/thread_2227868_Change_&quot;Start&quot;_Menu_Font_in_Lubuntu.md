### Post by Dennis N on 2014-06-04
I was able to change the menu font through:

Menu > Preferences > Customized Look and Feel > Widget Tab

In the lower right, you see Default Font. Click on the font name, and a dialog pops up. There you can change the size or the entire font. However, this also changes the fonts in other places as well. Click OK and Apply to finish.
