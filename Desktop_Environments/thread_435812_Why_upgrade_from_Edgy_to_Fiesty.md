### Post by metalheart on 2007-05-07
Updating from packages is easier but usually new version of a program comes with Ubuntu's new release. Updates usually include only bug fixes. You can find community created binary packages (from Ubuntu forums, for example) for latest version.
If you install from source code you can also use the latest, but it's sometimes more complicated and you risk to break the whole system.

### Post by Tanker Bob on 2007-05-07
The big reason to upgrade is that you get a newer kernel version and programs compiled for that kernel.  You also get newer versions of major apps in the repositories.  Plus, Feisty has more proprietary drivers in the restricted respositories.  Feisty also has a nice QuickLaunch applet for the task bar and other small stuff that adds to its ease of use.  

I was using Edgy until Feisty's release.  I install Feisty on its release day and find it more stable than Edgy.  Edgy wasn't bad, but Feisty seems better in that regard.
