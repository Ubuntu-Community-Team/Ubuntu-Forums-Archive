### Post by Jucato on 2006-07-26
You could also check the main Kubuntu page ([http://www.kubuntu.org](http://www.kubuntu.org)) since they will announce when a new Amarok version is already available for Kubuntu. Take note that the amarok.kde.org web page is intended for all KDE distros, and that Kubuntu will have it's own Amarok release, usually on the same day a new Amarok version is released.

Here's the release announcement for the latest Amarok version in Kubuntu (1.4.1) [http://kubuntu.org/announcements/amarok-1.4.1.php](http://kubuntu.org/announcements/amarok-1.4.1.php)

However, if you have some audio in FLAC format, I would suggest not upgrading to 1.4.1, probably just staying with 1.4.0, as there seems to be a slight problem with FLAC-xine interaction in 1.4.1.

Here's the 1.4.0 release announcement: [http://kubuntu.org/announcements/amarok-1.4.php](http://kubuntu.org/announcements/amarok-1.4.php)
