### Post by ajgreeny on 2008-12-24
Most of the configs you will need are in the hidden folder (use Ctrl+H to see those if you can't at the moment) /home/username/.gconf

Copy that or back it up and you should be able to get the same setup on another computer.  Just remember that if your first setup contains references to files, eg desktop background, that are not on the new computer, there may be some cosmetic differences.
