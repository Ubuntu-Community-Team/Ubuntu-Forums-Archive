### Post by Frogs Hair on 2013-03-03
The ambiance colors pack is not Gnome 3.6/12.10 compatible and that is stated at Gnome Look.  If you are using a PPA to install themes make sure they are for Gnome 3.6 which 12.10 uses.   While some 3.4 themes work in 3.6 I would avoid them for best results.

Also,  are you using a proprietary graphics drivers ? I ask because the fonts look funny in the video and if there is a driver available in software sources > additional drivers I would consider installing it.
