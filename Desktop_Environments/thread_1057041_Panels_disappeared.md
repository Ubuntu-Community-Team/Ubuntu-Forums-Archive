### Post by gjoellee on 2009-02-01
> **Gamecubic said:**

The panels don't show and I don't know how to get them to show. Any help ?

Use ALT+F2 or terminal and launch this command: ```
xfce4-panel
```

If they are gone next time you log in, to the same as above, but when you exit, make sure the "Save sessions" button on the logout icon window is closed.
