### Post by skymt on 2006-03-12
I'm assuming gnome-light is just a convenience package, pulling in as few packages as possible to make a usable GNOME desktop. If that's the case, then no, there isn't. Ubuntu installs, by default, a rather full-featured, not light by any means, GNOME desktop. If you want a minimalist GNOME, you have to use the server (read command-line only) install and install the bits by hand.
