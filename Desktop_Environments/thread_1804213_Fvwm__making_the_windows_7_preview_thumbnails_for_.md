### Post by sikander3786 on 2011-07-15
Compiz can easily do that.

Install CCSM:

```
sudo apt-get install compizconfig-settings-manager
```

Go to System > Preferences > CompizConfig Settings Manager or search the Dash if using Unity. In the CCSM search box, type 'Preview' and enable the plugin. You can click it for changing some settings also.

You obviously need 3D accelerated graphics card/drivers for Compiz to work.
