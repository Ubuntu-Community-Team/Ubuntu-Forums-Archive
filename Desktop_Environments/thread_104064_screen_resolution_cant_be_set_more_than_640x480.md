### Post by Joonas on 2005-12-15
frodon: laptop?

Anyway, on all my desktop computers runnig Ubuntu I have found it easiest to use

```
sudo dpkg-reconfigure xserver-xorg 
```

to set up my xservers. The essential things for you there are the correct graphic card drivers and the monitor refresh rate settings, as well as selecting the resolutions you want to use.

You should find out the correct refresh rate settings for your monitor. Setting the rates too low will result in only low resolutions and refresh rates, too high might damage your display.

### Post by Gurgeh on 2005-12-16
For me this one was pretty simple and really easy to overcomplicate. Basically I would replace:

> Section "Screen"
Identifier "Default Screen"
Device "Intel Corporation 82845G/GL[Brookdale-G]/GE Chipset Integrated Graphics Device"
Monitor "500E"
DefaultDepth 16
SubSection "Display"
Depth 1
Modes "1024x768" "800x600"
EndSubSection
SubSection "Display"
Depth 4
Modes "1024x768" "800x600"
EndSubSection
SubSection "Display"
Depth 8
Modes "1024x768" "800x600"
EndSubSection
SubSection "Display"
Depth 15
Modes "1024x768" "800x600"
EndSubSection
SubSection "Display"
Depth 16
Modes "1024x768" "800x600"
EndSubSection
SubSection "Display"
Depth 24
Modes "1024x768" "800x600"
EndSubSection
EndSection


With this:

> Section "Screen"
Identifier "Default Screen"
Device "Intel Corporation 82845G/GL[Brookdale-G]/GE Chipset Integrated Graphics Device"
Monitor "500E"
DefaultDepth 16
SubSection "Display"
       Depth 24
       Modes "1024x768"
EndSubSection


for some reason you can still select the resolutions under this but it forces X to start up in the only specified resolution. Worked for me anyways....
