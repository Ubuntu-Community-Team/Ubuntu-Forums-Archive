### Post by hictio on 2008-10-29
Actually, if this is just for the fonts you see on Firefox, open it, and goto:
Edit -> Preferences -> Content -> on the line of Fonts & Colors -> Advanced

And set a "Minimum font size"

Another option that I use a lot is unticking the option "Allow pages to choose their own fonts, instead of my selections above"; this after setting the fonts I like.

This might not be convenient to you if you are a web designer or something like that, but I like it, and don't bother with my job.
