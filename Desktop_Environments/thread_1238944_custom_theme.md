### Post by mcduck on 2009-08-13
The application theme and window border themes are two separate things, so it should not be a problem changing your window border theme while still keeping the "Glossy" theme for applications.

Just go to System/Preferences/Appearance, and on the Theme-tab click the Customize-button. Now you can select control, window border and icon themes separately from each other.

(If you like your current window borders apart from the buttons, and can't find a new border theme with buttons you like the only option would be to edit the theme yourself. Emerald themes can be edited with the Emerald Theme Manager, all other window border themes are mainly text files that you'd need to edit by hand)
