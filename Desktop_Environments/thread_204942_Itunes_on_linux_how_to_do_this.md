### Post by digimars on 2006-06-27
However (if you wish to pay $39, which helps support Wine development) there is Crossover Office, which is a project built on Wine with some extra refinements.  It can run iTunes, but I believe only up to 4.5.  As far as I know, this is the only way to have iTunes under Linux.
