### Post by jISh on 2006-09-18
Easy. Open a console and write the following:
```

sudo aptitude update && sudo aptitude install kubuntu-desktop
```

Then log out, and in the login manager choose Session. Pick KDE, and select to make it your default when you log in.

### Post by Beernut on 2006-09-18
If you have already installed KDE just log out and click on "sessions" in the bottom left hand corner > select KDE and login.  Once you try to login you should be asked if you want this to be your default select yes.
