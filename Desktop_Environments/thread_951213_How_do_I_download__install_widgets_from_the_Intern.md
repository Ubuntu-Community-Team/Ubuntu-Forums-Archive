### Post by benerivo on 2008-10-18
I'm using debian rather than kubuntu but i only get one plasmoid (widget) listed. I know they come from [www.kde-look.org](www.kde-look.org) where you can download them manually.

However, the plasmoids/window decorations/styles, are not available for all distros. They are only downloadable as source packages, deb files, rpm files etc.. This is in contrast to gnome themes/window decorations/emerald themes which are distro agnostic - that is to say you download the theme and it can run on any gnome desktop. For me kde4 is inferior in this way as it seems that you can only have a window decoration/plasmoid/style if it has been packaged for your particular distro and even then you need to install it as root. This may explain the lack of options to download new plasmoids/themes/window decorations directly from the desktop. Colour themes, icons and plasma themes come in files that work for any distro (which explains why they are all downloadable from the desktop).
