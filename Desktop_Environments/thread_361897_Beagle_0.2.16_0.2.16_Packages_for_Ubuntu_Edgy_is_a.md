The latest Beagle 0.2.16 Ubuntu Edgy package is available.
Add one of these site into your source list

```
deb http://beagle-project.org/files/ubuntu/edgy/ ./
  or
deb http://kubasik.net/ubuntu/edgy ./
 
```Then, update the list by
```
sudo apt-get update
```and install Beagle using Synaptic or in terminal with apt-get install 
whatever method you like. :lolflag:
