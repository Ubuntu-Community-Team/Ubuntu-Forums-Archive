### Post by arnieboy on 2005-07-13
file browser and nautilus are the same.. do this: run nautilus and go to edit-->preferences-->behaviour and check "always open in browser windows" this will let natilus open as a file browser everytime.
The terminal u like can be run by the command "gnome-terminal"
