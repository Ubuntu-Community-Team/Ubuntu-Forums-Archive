### Post by mcduck on 2012-02-08
Firefox has separate shortcuts for zooming text only, or for zooming text and images. THe noSquint extension mentioned by stinkeye also allows you to configure per-page defaults for both these settings.

What comes to the zooming done by the desktop environment, if you run Unity session (or any other session using Compiz as your window manager, you can use the "Zoom Desktop" or "Enhanced Zoom Desktop" plugins, the later one is enabled by default but you need to use [CompizConfig Settings Manager]("apt:compizconfig-settings-manager") to set a shortcut for it.

....and of course the accessibility tools on pretty much every desktop environment provide some kind of "magnifying glass" if that's what you are looking for.
