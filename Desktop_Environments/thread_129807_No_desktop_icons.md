### Post by Sutekh on 2006-02-14
Open up the **Configuration Editor**.

Its in the **Applications -> System Tools ** menu

Go to **apps -> nautilus -> dekstop**

Make sure the **home_icon_visible**, **computer_icon_visible**, etc are checked.
