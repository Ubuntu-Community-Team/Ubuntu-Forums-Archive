### Post by braete on 2009-06-07
yes.
  usingthe place windows thingie in compiz settings
in the second tab, under windows with fixed view ports

new/ window matches (the + sign), in the drop down list -> window title (the others for some reason didnt work for me every time ...), grab the window of the program you like to set, accept, then on the 1st dialog chose the viewport you like.
