### Post by enimeizoo on 2011-07-14
Hello,
Did you check systemsettings > input devices > keyboard. In the layout tab, you can configure layouts.
You can add a layout, you choose the language, the actual layout, and you have a variant option. Here you can choose if you want dead keys or not. It is actually pretty well hidden :P
I actually still prefer xmodmap rather than layouts features since I need a very versatile keyboard.
