### Post by OutOfReach on 2008-09-28
Ok, it appears as if that is a GDM theme. Installing a GDM theme is a little different than installing a normal theme. BTW GDM is the login screen.

To install GDM themes, goto System>Administration>Login Window, enter your password and goto the 'Local' tab. Now, just drag and drop the theme there and Select it and you should be OK. Just log out to see the new login screen theme.
