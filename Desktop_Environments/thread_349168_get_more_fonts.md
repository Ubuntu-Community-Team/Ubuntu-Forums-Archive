### Post by yabbadabbadont on 2007-01-29
You can search the repositories for more fonts using Synaptic.  (In Gnome, System->Administration->Synaptic Package Manager)  You can also copy your own fonts into ~/.fonts directory.  (make the directory if needed, note that it starts with a period)  I'm not sure if the second method will work with all fonts, or just truetype fonts.  Try it out and let us know what you find.
