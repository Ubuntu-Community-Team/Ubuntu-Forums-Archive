### Post by drlock on 2006-08-05
Did you try the Alternate install CD?  This is supposed to allow you to install on machines with less memory.  According to the Release Notes ([http://www.ubuntu.com/download/releasenotes/606](http://www.ubuntu.com/download/releasenotes/606)) 256mb is the bare minimum for Ubuntu Desktop.

However, on a laptop with limited memory, I would be installing Xunbuntu ([http://www.xubuntu.org/](http://www.xubuntu.org/)).  My laptop has 500MB of memory and runs very nicely on Xubuntu.  A few quick tests have shown that XUbuntu uses about half the total memory resources that my Ubuntu Gnome install uses at work.
