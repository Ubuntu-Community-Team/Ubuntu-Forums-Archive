### Post by eggdeng on 2007-12-25
I'd be very surprised if you could get Dapper to install in 128MB SDRAM. The release notes recommend a minimum of 256MB for the desktop edition. The server edition, however, will install in as little as 64MB. It comes with no GUI so you could try adding a lightweight GUI once you got it installed.
