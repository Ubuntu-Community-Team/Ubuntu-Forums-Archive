You should be able to follow the guide in the wiki or at ubuntuguide.org to get apache and php setup.  If following ubuntuguide, note that hte latest versions are apache2 and php5.  So long as when you create any php files you stick them in /var/www (or a symlink of said) then you should be able to view them in firefox by typing 'localhost' in the address bar.
