### Post by Tenken on 2008-01-31
Are you using the default Ubuntu theme of did you install a new one? Some themes change your Nautilus(file browser) background. 

If not, to change it back to white in a Nautilus window go to Edit->Backgrounds and Emblems->click the colors button and select white.
