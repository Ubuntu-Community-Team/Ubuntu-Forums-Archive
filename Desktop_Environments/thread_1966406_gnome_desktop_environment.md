### Post by kansasnoob on 2012-04-27
Ubuntu 12.04 uses mostly gnome 3.2 with some gnome 3,4 components. Installing the package 'gnome' from the repos brings in some older gnome 3.0 packages from Debian which may really mess things up.

On a fresh Ubuntu 12.04 I can get a fairly sweet classic DE following the appropriate steps here:

[http://ubuntuforums.org/showthread.php?t=1966370](http://ubuntuforums.org/showthread.php?t=1966370)

Since you now have a garbled mess of packages I'm unsure of the outcome, I'm guessing it won't be pretty :(
