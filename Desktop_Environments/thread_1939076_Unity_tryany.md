### Post by Bucky Ball on 2012-03-11
> **Marzata said:**
> xubuntu-desktop package is the answer or just clean Xubuntu install.

NO! Installing xubuntu-desktop is no answer and will make no difference to the OP's original issue (defaults to Unity 2D regardless of DE selected from 'Sessions'). Xubuntu-desktop will install ALL Xubuntu packages, dependencies and apps resulting in duplication, an overstuffed machine, possibly slowing down of the machine, over-use of hardware resources and unexpected conflicts. You *don't* want that! If you do, install Xubuntu directly as Marzata suggest secondly. (Download ISO, burn to CD/USB, install Xubuntu.)

As I suggested, if you just want to change desktop environments, install xfce4 ONLY!!! But I'm assuming this is what you have already done and you just can't get to it. ;)
