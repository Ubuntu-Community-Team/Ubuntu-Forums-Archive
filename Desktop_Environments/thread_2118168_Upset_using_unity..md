### Post by sudodus on 2013-02-20
I recommend that you upgrade to (or make a fresh install of) Ubuntu, Xubuntu or Lubuntu 12.04.
See this link for Gnome Classic

[https://help.ubuntu.com/community/PreciseGnomeClassicTweaks](https://help.ubuntu.com/community/PreciseGnomeClassicTweaks)

You can also install Xubuntu-desktop (or only the light-weight desktop environment XFCE) with

```
sudo apt-get install xubuntu-desktop
```or
```
sudo apt-get install xfce4
```and/or Lubuntu-desktop (or only the ultra light-weight desktop environment LXDE) with
```
sudo apt-get install lubuntu-desktop
```or
```
sudo apt-get install lxde
```and select desktop environment at the log in screen.

Feel free to test these alternatives, and if you want to focus on one of them you can use this link to remove the others

[http://www.psychocats.net/ubuntu/index](http://www.psychocats.net/ubuntu/index)

Good luck :-)
