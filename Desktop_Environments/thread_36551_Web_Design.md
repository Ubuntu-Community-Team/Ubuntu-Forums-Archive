### Post by thorN on 2005-05-24
I'd use Gedit to edit web code, it has good syntax highlighting and tabs are useful too.

Screem isn't bad either.

--

It's a great idea to consider learning HTML, it's very simple - in fact it's intentionally designed that way.

[http://www.w3schools.com/](http://www.w3schools.com/)

This site will teach you how to code proper XHTML or HTML.
HTML adds meaning to plain text, such as signifying which parts are headers.

(Please not that most of the guides on HTML on the web are completely wrong. Stuff like <FONT> and <TABLE> tags shouldn't be used in design.)

If you want to do any web design, you'll need to know CSS as well, which is slightly more difficult.
CSS is how you choose fonts, colours and layouts.
