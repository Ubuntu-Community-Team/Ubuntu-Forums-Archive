### Post by GoodHumorJO on 2010-01-01
I played with Kubuntu for a bit recently, and am looking to switch another computer of mine to KDE, but I don't want to install Kubuntu over Ubuntu and do it that way. What would be the best way to swap from GNOME to KDE?

--Edit

Another question, would it be possible to have both, and switch between which one or the other similar to dual booting operating systems? Might be a nice way to show off the different enviroments to friends without having to toss in the Kubuntu or Ubuntu disks.

---

### Post by northrup on 2010-01-01
You don't need to dual boot.  Just install KDE.  The easiest way to do it is through the terminal:

```
sudo apt-get install kubuntu-desktop
```

When you login, you can select between GNOME and KDE.
