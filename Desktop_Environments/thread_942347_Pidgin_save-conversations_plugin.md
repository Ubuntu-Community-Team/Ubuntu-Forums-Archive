Hi to all!

I wrote Pidgin plugin named "save-conversations", that saves your opened conversations at shutdown and restores its at startup. I'm searching for such plugin for a long time and cant't find it, so I decide to create my own one.

You can download it at [http://sourceforge.net/projects/save-convs/](http://sourceforge.net/projects/save-convs/)
To install just put included save-conversations.pl file in your Pidgin plugins directory (for example ~/.purple/plugins under Linux, create it if doesn't exists) and restart Pidgin.
