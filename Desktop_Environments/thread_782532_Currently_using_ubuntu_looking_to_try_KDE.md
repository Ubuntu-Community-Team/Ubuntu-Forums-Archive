### Post by angry_johnnie on 2008-05-05
Ok, you have a resolution problem, for which I have no idea (never had it myself :-)).

But, a quick and dirty fix would be to install a different gdm theme --one which will show the **Sessions** option some place else, where you can actually see it.

Go to System > Preferences > Login (or, maybe, login window), and choose a different one.

There's more themes available from Synaptic, and some more at places like [www.gnome-look.org](www.gnome-look.org)




You could also try Alt + S (S as in session), or Alt + O (O as in options --although sometimes that's Alt + T as in opTions), or some other combination.
