### Post by lapubell on 2008-03-19
I use a really old 700mHz Celeron machine at an office and it uses both XFCE and gnome.  Here is what I can tell you:

Boot into XFCE after GDM is about 4 sec faster than gnome,
OpenOffice.org boots about 2 sec faster,
Thunar is WAY faster than nautalus

What i do is install xubuntu-desktop on top of a standard ubuntu install and set Thunar to my default file manager.  This makes using gnome quite a bit faster for my everyday tasks when browsing files.  True, you might not see too much of a performance increase on a really new comp, but XFCE is a dream come true on old machines.  If you want to see some real speed increases, load xfce panel on top of fluxbox or openbox.  Now that is a fast desktop on an old machine.

As for nothing happening when you plug in a flash drive, that might not be the case.  it probably mounted just fine, but you didn't see anything happen.  Open Thunar and it should be on the side panel waiting for you to access it.
