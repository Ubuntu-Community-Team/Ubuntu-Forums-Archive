### Post by partypants on 2011-05-06
I was playing around with compiz in Ubuntu Classic (11.04) when I enabled desktop cube, after which all my window edges disappeared.  I was able to get the windows to return by enabling Window Decorations, but I can't move my windows.

How do I solve this problem?

---

### Post by Copper Bezel on 2011-05-06
It's possible that whatever disabled your Window Decorations plugin also disabled Move Windows under Window Management. Check that, first.

---

### Post by wilee-nilee on 2011-05-06
compizcofig settings manager-window management-move windows
