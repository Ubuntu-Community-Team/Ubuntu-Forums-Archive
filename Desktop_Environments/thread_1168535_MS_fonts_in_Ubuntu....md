### Post by coffeecat on 2009-05-24
Go to System > Administration > Synaptic and install the package msttcorefonts. Even better, install the package ubuntu-restricted-extras which will install msttcorefonts and a whole load of useful multimedia codecs and other things like the flash plugin for Firefox.
