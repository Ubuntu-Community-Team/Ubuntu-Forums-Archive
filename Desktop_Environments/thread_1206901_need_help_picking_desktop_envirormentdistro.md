### Post by XubuRoxMySox on 2009-07-07
I find that on my computer at least, the dewsktop environment makes all the difference in the world when it comes to speed. 

Try installing [LXDE]("http://lxde.org") from the Ubuntu repositories, then log out. When you log back in, choose LXDE for your Session.

It's graphical like KDE, but much simpler and so very lightweight that it's almost like having no desktop environment at all.

It makes my Ubuntu much lighter and faster.

An Ubuntu-based distro called [U-Lite]("http://u-lite.org") is a fine precursor to - and maybe a preview of - the upcoming official LXDE-based Ubuntu release from Canonical called [Lubuntu](https://wiki.ubuntu.com/Lubuntu).

### Post by snowpine on 2009-07-07
My advice is find a distro that you like (runs fast, easy package management, friendly community, etc.) and then add the desktop environment of your choice. Almost every distro gives you the choice to is install Gnome, Xfce, KDE, LXDE, etc. If you love Ubuntu, there are ways to make Ubuntu faster. Check out Xubuntu, CrunchBang, U-Lite, etc. or simply keep your existing install and try LXDE as your desktop environment. Also experiment with little things like using the proper driver for your video card, browsing with Opera instead of Firefox, disabling effects, and so forth. Your computer should be capable of running Ubuntu comfortably.

### Post by Pogeymanz on 2009-07-07
Did you turn off compiz effects first? Just hit ALT+F2 and run ```
metacity --replace
```

To replace Compiz with Metacity (Gnome's default window manager). Which might make you happier.

But I agree that Ubuntu's Gnome is a little fat.

Try installing Debian Lenny. It's much more basic than Ubuntu (it's also Ubuntu's daddy, so Synaptic is there, just like you're used to). And it comes with Gnome by default.

I love XFCE, just not the way Ubuntu does it. Try installing Debian+XFCE and be pleasantly surprised at how much sleeker your machine will feel.
