### Post by Jussi Kukkonen on 2007-02-22
Did you want to know something about that?It means that the bug is fixed in Gnome 2.16.2 -- so Ubuntu 7.04 will have the fix. 

Older version do not get most bug fixes -- only security bugs and ones that fix major usability issues. If you think your bug fits that description, file a bug in launchpad.net if you haven't already and link it to the gnome bug.
