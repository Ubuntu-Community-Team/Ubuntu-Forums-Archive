### Post by grunge09 on 2006-07-13
To create a new toolbar...

Right click on the taskbar, left click toolbars and then choose NEW TOOLBAR, create a directory for it and then it should appear on the original XP taskbar and you can drag it to the top or sides of the screen so it is a stand alone toolbar or quickbar.  Then create or drag shortcuts on that.  It will even auto hide if you make it.
