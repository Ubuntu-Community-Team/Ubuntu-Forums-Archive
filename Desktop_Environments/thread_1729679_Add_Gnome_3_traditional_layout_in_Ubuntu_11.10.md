Gnome itself has also change it's own desktop too, but Ubuntu decided not to use that new desktop and create their own called Unity and Unity-2D.

[http://www.gnome3.org/](http://www.gnome3.org/)

### Post by Mercury_Alpha on 2011-04-28
> **ingeva said:**
> 
If I want to install with access to the old panel, do I just install ubuntu-desktop like before  (asp-get -y install ubuntu-desktop) - and then choose classic desktop before logging in?

Ubuntu Classic comes pre-installed alongside Unity. When you start up the computer and reach the login screen, click on your account name, then open the menu that appears at the bottom of the screen. (For some reason, the menu doesn't appear automatically. Maybe aesthetic reasons.) Then log in.

Now that you've logged in to Classic, Ubuntu will default to that environment, until you change your selection in the menu. So you won't have to choose an environment every time you log in.

(You may know some of this stuff already -- I just like to make my answers as complete as possible.)

At least, that's how it works with the desktop version. I've never used the server version, so my information may be inaccurate.
