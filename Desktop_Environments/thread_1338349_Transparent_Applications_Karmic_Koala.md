### Post by dentaku65 on 2009-11-27
Yes it is possible; open your application (the one that you want transparent) then:
[LIST]
[*]open CompizConfig Setting Manager
[*]enable Opacity, Brighness and Saturation option
[*]in the configuration of this option choose Opacity tab and under Window specific settings choose new
[*]in the Edit window choose **+** button, will open another window Edith Match
[*]in Edit Match choose Grab, and the mouse will change the cursor, point and click your open application, the name of you application will appear into the Edit match window, click Add button
[*]in the Edit window, put the value of Windows value to 90% (at your taste), click close
[/LIST]
...and you're done! :D
