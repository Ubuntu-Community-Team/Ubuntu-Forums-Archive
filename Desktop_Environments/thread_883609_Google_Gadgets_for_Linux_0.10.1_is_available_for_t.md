### Post by suzhe on 2008-08-08
It's a major bug fix and feature enhanced version, to celebrate the opening of the Beijing 2008 Olympic Games. Lots of bugs have been fixed in this release. Two new built-in gadgets have been added, you would love to use them:
  * Gadget Designer gadget
    It's a Gadget Designer similar than the one provided in Google Desktop SDK for Windows, but written in Google Desktop Gadgets API. Yes, itself is a gadget. You can just add it from add gadget dialog, then drag it out of the sidebar, because it's too large. Currently, it only supports editing existing gadgets, creating new gadget is not supported yet. In Gadget Designer, you can edit the gadget with your mouse very easily. If you have any question about it, feel free to ask on our user group.
  * Photos gadget
    It's a gadget to display your favorite photos (either from web or local directory) as a slideshow. It's very simple to use, just add it from add gadget dialog and add your favorite sources in options dialog.

  Besides these two new gadgets, many serious bugs have been fixed. Many popular gadgets are supported now, such as facebook and Google Calendar gadgets. If you have some favorite gadgets which are not support in previous version, please try them in this new version.

Have fun.

[http://code.google.com/p/google-gadgets-for-linux/]("http://code.google.com/p/google-gadgets-for-linux/")
