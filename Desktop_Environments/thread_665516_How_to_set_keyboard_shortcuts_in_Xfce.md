### Post by aysiu on 2008-01-12
> **Lomz said:**
> How do I in Xfce set a keyboard shortcut for the xfce4-terminal to be fn+super?
In the Keyboard preferences, add a new theme (you pick the name of it to be whatever you want) and have the command be ```
xfce4-terminal
``` and then press whatever key combination you want.
