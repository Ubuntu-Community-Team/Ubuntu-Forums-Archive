### Post by Enigmapond on 2011-09-11
JDownloader...hands down the best I've worked with. RS and the like work really well. You need to add their PPA first:

```
sudo add-apt-repository ppa:jd-team/jdownloader
sudo apt-get update && sudo apt-get install jdownloader
```
