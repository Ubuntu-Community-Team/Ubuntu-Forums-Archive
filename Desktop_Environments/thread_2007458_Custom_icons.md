### Post by Frogs Hair on 2012-06-20
It would be time consuming , but if you have alacarte installed it is possible to change icons for applications from the main menu. Download a set like Faenza and look in the folder after extracting the package and you will learn something about how to create your own set.

Gimp is a great tool to create icons with . I have a few icons that I created and use but an entire set is a lot of work. Open advanced settings > fonts and there are some options to change fonts. I have found some nice and unusual fonts at the link .          [http://www.urbanfonts.com/](http://www.urbanfonts.com/)
