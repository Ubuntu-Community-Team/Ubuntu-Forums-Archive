### Post by Anchovie on 2005-10-17
Bascially I want to use a lower resolution so it's faster, but when I do that everything gets all big, is there anyway to scale down the panels and the icons, the buttons, etc.?

---

### Post by bdash on 2005-10-17
To do that, you should reduce the font in the preference. That will shrink everything.

Then you can choose a small Metacity theme, and maybe a small gtk theme (not sure if it change a lot for Gtk).

Finally you can reduce the default icon size in the Nautilus preferences.
