### Post by javierfh on 2006-07-26
Hello,

i have used gimpshop and it is very nice and resembles a lot to photoshop. 
For little things, its just great, but to be honest there are things that are missing or at least couldnt find. For example the rotate option, you can only rotate 90,180,270 degres, but you cant specify the number of degres you want to rotate (i needed that to correct some tilted horizons in pictures). As i said, might be that its possible do it, but I couldnt find how to do it.
Then, another complain i have been reading a lot, is that you can only use 8 bits (dont know if for the colours or for what, but 8 bits anyway :) ) so i tried to search if there is something else and it seems that there is a project called cinepaint ([http://www.cinepaint.org/)](http://www.cinepaint.org/)), that started as a derivative from gimp and supports bigger number of bits (and seems big movie companies as Sony or Pixar are using it). The ui though, its very primitive in my oppinion.

Hope this helps,


Javi
