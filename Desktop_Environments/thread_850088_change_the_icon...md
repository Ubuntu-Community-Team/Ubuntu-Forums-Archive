### Post by sayakb on 2008-07-05
Open a terminal and type:
```
gksudo nautilus
```Now navigate to /usr/share/icons/icon-theme-name/22x22/places
Replace the start-here.png with the one you want to. It should have exactly the same filename and 22x22 dimension. Also change the start-here.png within the 24x24 folder. If you have the panel at it's default thickness, changing these two will do.
Also, in case you are using a custom installed theme, simply open your Home folder, press Ctrl+H, goto the .icons folder and find the icon theme folder there.
