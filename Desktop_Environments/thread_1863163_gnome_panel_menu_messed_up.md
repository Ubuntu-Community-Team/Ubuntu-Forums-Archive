### Post by crdlb on 2011-10-17
You can edit the panel by holding the Alt key when you right click. The System menu has been replaced by the user menu on the right side. Any configuration applets that have not been merged into System Settings are currently in Applications->Other.

For the record, these changes come from upstream GNOME, not from Ubuntu.
