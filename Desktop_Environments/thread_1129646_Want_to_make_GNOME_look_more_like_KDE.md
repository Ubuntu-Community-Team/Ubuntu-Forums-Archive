### Post by neogarv on 2009-08-06
Sorry for bumping a two week old post, but...

Recently, I've discovered that you can run KDE's plasma desktop on gnome. I'm still experimenting, but I'm making pretty good progress.

Here are a few simplified steps:
    - Install the kubuntu-desktop package.
    - Disable gnome-panel from appearing. You can do this like so:
        -- Open up gconf-editor.
        -- Go to desktop > gnome > session.
        -- Edit required_components_list and remove panel.
    - Add plasma-desktop to startup applications.
    - Logout and log back in.
    -Enjoy!
