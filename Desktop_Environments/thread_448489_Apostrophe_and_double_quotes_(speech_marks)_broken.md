### Post by ryanthemadone on 2007-05-19
OK I solved it and it was really simple.  Hopefully this will help anyone else who ends up in the same position, I saw some other people had done the same and never found a solution.

Choosing a 102 key (intl) generic keyboard will allow you to select a plain United Kingdom keyboard layout rather than United Kingdom international or Dvorak.
