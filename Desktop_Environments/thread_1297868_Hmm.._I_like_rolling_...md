### Post by mcduck on 2009-10-22
Not with Metacity (or Compiz+GWD). Emerald does support that option (although most Emerald themes lack themed button images for anything else than the default buttons).

If you use Compiz you could do what I always do, and set mouse wheel to toggle window shading. To do that open gconf-editor, and set "apps/gwd/mouse_wheel_action" to "shade".

Or if you are not using Compiz, you could replace Metacity with some other window manager, like Openbox.
