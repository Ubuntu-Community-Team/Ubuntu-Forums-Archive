### Post by derekeverett on 2010-06-03
> **wojox said:**
> When you install 10.04 it will install Compiz. You can then download CompizConfig Settings Manager.

Yup that's the way to do it.

You can get it through the Ubuntu Software Center. They call it "Advanced Desktop Effects Settings (ccsm)"

When it's installed you will have to play around with the settings in System->Preferences->Appearance (visual effects tab)
and
System->Preferences->CompizConfig Settings Manager.

A lot of the success with it depends on your graphics card.

### Post by blchinezu on 2010-06-03
```
sudo apt-get install compiz-fusion-plugins-extra compizconfig-settings-manager
```
*compiz-fusion-plugins-extra* - extra plugins which are not installed by default
*compizconfig-settings-manager* - configurator
this should do it :)
then just play with the settings

### Post by smithshn on 2010-06-03
To change any desktop effects you have select the 'System' menu from desktop and go to Preferences->CompizConfig Settings Manager. If you do not show the option of CompizConfig Settings Manager then you have required to download it.
