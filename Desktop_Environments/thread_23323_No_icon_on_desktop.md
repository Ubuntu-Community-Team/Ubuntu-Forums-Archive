### Post by Xian on 2005-04-01
[QUOTE=ykpaiha]I don't understand as trash.desktop is inside the folder? An it appear when I open the file browser.[/QUOTE]
Do you also run Gnome? KDE and Gnome share the same desktop path by default and so you may be seeing Gnome's trash.desktop file in the folder, but that will not of course show up on your KDE desktop as a visible icon.

You can always change your desktop path in the KDE control center.
