### Post by ugm6hr on 2007-07-10
You can manually add keboard shortcuts with XFCE keyboard settings (from Menu).  I use Banshee, and have my media keys set to the following:
Play/Pause:* banshee --toggle-playing* (starts Banshee, or toggles between play/pause track)
Stop: *banshee --hide* (sends Banshee to system tray)
Prev: *banshee --previous* (previous track)
Next: *banshee --next* (next track)

If xmms has terminal commands to control playback, you should be able to do the same.  Just google for xmms terminal.
