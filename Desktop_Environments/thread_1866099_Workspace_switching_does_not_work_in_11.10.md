### Post by mcduck on 2011-10-21
In the view where you can see all the workspaces, you can simply drag&drop the windows to whichever workspace you want.

The keyboard shortcuts work from the normal desktop view, not from that view. (ctrl+Alt+arrow keys to switch desktops, Ctrl+Alt+Shift+arrow keys to switch desktop with the current window, and as mentioned, right-click on the titlebar for the option to send the window to a certain desktop.)
 
And of course you can use [CompizConfig Settings Manager]("apt:compizconfig-settings-manager") to define what ever keyboard shortcuts you want, and to check what are used at the moment. There is no shortcut for sending a window to a certain desktop by default, but you may of course configure one if you want to)
