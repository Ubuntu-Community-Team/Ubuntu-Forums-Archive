### Post by nalmeth on 2006-12-18
Beryl is very cool, but do you have a graphics card that supports 3D?

If you don't, you can still have some cool things that don't require 3D

Gdesklets is pretty nice (or superkaramba on KDE), alternative Desktop environments like E17, or even something so small as new themes (desktop, icons, etc) can make a big difference for a users comfort.

Checkout themes at gnome-look.org, KDE themes at kde-look.org
