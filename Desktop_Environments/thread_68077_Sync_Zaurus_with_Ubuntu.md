### Post by RikoW on 2005-09-22
Hi bam,

I used to use multisync (it's in Ubuntu universe repository) to sync my zaurus (calendar, to-do, adressbook) with evolution. That worked ok. But it only syncs (as far as I know) with the local files but not with calendars on an Exchange server for example.

I don't know, how it works now with the evo 2.x. When I used it, it was still under evo1.4.6 ... it was quite some time ago :)

in addition to multisync, you need the proper plugins, e.g. evolution-plugin and the opie-plugin for the Zaurus.

Connection can be made via usbnetwork or WLAN, if you have a wlan card for the Zaurus. I used to have some problems with usb while WLAN worked well for me.

I think, also kitchensync can sync with evo, but I never used it, so I'm not sure.

Good luck,

            Riko
