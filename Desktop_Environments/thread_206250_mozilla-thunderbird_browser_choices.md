### Post by fragos on 2006-06-29
Thunderbird uses the default browser as established by the widow manager of choice.  If you and your wife are sharing a login there can only be one choice.  Separate logins allow more personal control.  KDE supports a system default but some applications have their own parameter or predetermined choice.  I've seen this work correctly in both gnome and KDE but have little experience with xubuntu.
