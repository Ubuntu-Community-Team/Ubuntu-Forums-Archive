### Post by Buffalo Soldier on 2006-07-02
[QUOTE=lnxquestions]So can i call telephons via internet with this program ? I know that i can with skype .[/QUOTE]
Yes you can, but it is not free.

Ekiga (or any other SIP-based software phone) to 
Ekiga (or any other SIP-based software phone) is free.

Ekiga to conventional phone or mobile phone is not free. You need to register with [Diomand Card](https://www.diamondcard.us/exec/voip-login?act=sgn&spo=ekiga).

### Post by YannickDefais on 2006-10-15
Hello,

In theory you can use *any* sip provider to call land line or cell phone. In practice the problem is most of companies providing SIP want you to use their software (using windows...) by only allowing you to register their service using their software.

I've started a documentation about Ekiga wich gives some hints about this problem. It is in french, but you'll maybe figure the important thing : how to configure an Ekiga Account once you'll get the nick and password :

[http://doc.ubuntu-fr.org/applications/ekiga#communications_avec_les_telephones_fixesmobiles](http://doc.ubuntu-fr.org/applications/ekiga#communications_avec_les_telephones_fixesmobiles)
