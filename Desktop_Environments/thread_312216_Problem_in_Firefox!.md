### Post by d3v1ant_0n3 on 2007-01-10
Try adding the UserContent.css file from [ THIS](http://www.gnome-look.org/content/show.php?content=48088) theme (you'll have to unpack it first) to the following location:

```
/home/YOURNAME/.mozilla/firefox/NUMBER-LETTER MIX.default/chrome
```

It will make the text dark again in Firefox:D
