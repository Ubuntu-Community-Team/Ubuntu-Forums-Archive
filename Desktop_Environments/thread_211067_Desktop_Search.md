### Post by FredB on 2006-07-07
If you want desktop search using beagle, this is simple. Applications menu / Add - remove.

Then, click on both checkboxes on the right.

Look for beagle. Install it.

Then, right-click on your menu bar,  select "add to..."

When you're in the new panel, select deskbar. And right click on the new icon freshly added. Beagle should be listed in it.

Then, type something. All actions available will be shown.

Hope it helps ;)
