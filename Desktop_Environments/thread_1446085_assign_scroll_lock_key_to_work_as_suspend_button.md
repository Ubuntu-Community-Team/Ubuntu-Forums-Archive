### Post by kblft on 2010-04-03
See this post for suspending your system from command line [http://ubuntuforums.org/showthread.php?t=410570](http://ubuntuforums.org/showthread.php?t=410570) 

Once you figure out which command you need and check that it does indeed suspends your system you need to add that command as a keyboard shortcuts menu.

To add a keyboard shortcut you have several options :

1. and the simplest imho if you're using gnome is to go to System -> Preferences -> Keyboard Shortcuts - scroll down and add new shortcut. The only thing is that I think you'll need it to be Ctrl+Alt+Scroll lock - or something like that - but I'm not sure.

2. try this tutorial : [https://help.ubuntu.com/community/KeyboardShortcuts](https://help.ubuntu.com/community/KeyboardShortcuts) using xbindkeys
