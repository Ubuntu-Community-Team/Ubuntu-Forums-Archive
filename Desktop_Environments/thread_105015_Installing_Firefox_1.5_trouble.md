### Post by aysiu on 2005-12-17
The Wiki actually includes instructions for how to remove Firefox. Go ahead and follow those.

Then, do a ```
sudo apt-get update
sudo apt-get install firefox
```

Finally, follow the Wiki instructions again from the beginning.
