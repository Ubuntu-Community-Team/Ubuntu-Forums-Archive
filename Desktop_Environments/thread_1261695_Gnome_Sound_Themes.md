### Post by knepig91 on 2009-09-10
If you are talking about login sound, you should search [http://www.gnome-look.org/](http://www.gnome-look.org/), there is some good login sounds there. If you want to change the login sound go to System --> settings --> Startup programs, then look up loginsound and navigate to the sound file you downloaded.
