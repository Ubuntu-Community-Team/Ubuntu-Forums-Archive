### Post by Rancoras on 2005-01-28
Open a terminal and use this command:
```
sudo gedit /name/of/the/file
``` 
When it asks for a password, use the password of the account you created during installation.

Check out [www.ubuntuguide.org](www.ubuntuguide.org) for more great tips and tweaks.  :)
