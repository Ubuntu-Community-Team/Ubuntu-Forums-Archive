### Post by mmcmonster on 2006-07-05

### Post by mannheim on 2006-07-07
fsck will check the filesystem for errors. The basic usage is:

```
sudo fsck /dev/hda1
```

but you **don't** want to do that while the filesystem is mounted. Boot from a live CD, make sure that /dev/hda1 is not mounted, and **then** run it.
