### Post by Iandefor on 2005-11-30
Most people get the drivers so they can play games; the cards themselves provide the ability to process the graphics so the main CPU doesn't have to (Essentially). So: if you like to game, you'll probably want the drivers; otherwise, don't worry about it.

 I've heard complaints that the ATI drivers are slow; I've not had firsthand experience with them, so I don't actually know. Just a tip.
