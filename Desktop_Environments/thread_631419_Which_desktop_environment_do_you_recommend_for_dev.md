### Post by swerner on 2007-12-04
That really depends on what you want to program, it looks like you aren't into graphics apps, maybe more web related apps.  IMHO I would use the desktop that you prefer to use for normal use, like web surfing, etc.  Switching between desktops can be a pain.  You are able to run Gnome application on KDE and vice versa, so it's not too much of a problem.  

It terms of an IDE, for Gnome there is Ajunta, for KDE there is Kdevelop.  While the Eclipse IDE is desktop independant, but I don't know if can handle all the languages you wish to develop in.
