### Post by raziel on 2006-04-21
Hi,

I've just bought this same cammera, the Sony Cybershot DSC-S600 and decided to import the pictures to Ubuntu ( I'm with Gnome) - selecting **"usb storage"** didn't work but selecting **PTP** worked right up!

Ubuntu recognized the cammera, same wrong model as it did with yours but I was able to transfer all the pictures to the Desktop! I got a pop-up notice on the desktop asking me where I would like to save the pictures to, with small thumbnails even. Very nice!

I'm quite pleased with this cammera so far, even more now that it's working with Linux, I'm using Dapper Drake on this laptop, hope it helps!
