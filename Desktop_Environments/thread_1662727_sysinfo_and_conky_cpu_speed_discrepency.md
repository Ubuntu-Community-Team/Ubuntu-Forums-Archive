### Post by gordintoronto on 2011-01-08
Ubuntu supports "CPU scaling." Basically, this means that when you aren't doing much, your CPU slows down -- and uses less power, and generates less heat. 

For example, my CPU has a rated speed of 3.2 GHz, but it only runs that fast when I am rendering a video. Almost all the time, the CPU runs at 800 MHz. Even playing flash videos seldom requires running the CPU faster.
