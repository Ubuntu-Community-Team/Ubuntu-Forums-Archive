### Post by Perfect Storm on 2005-10-18
Try:

```
sudo apt-get install libgl1-mesa
sudo apt-get install libgl1-mesa-dev

```

If that doesn't work see if you have libGL.so in the /usr/lib directory. If you have  libGL.so.1 then
```

cd /usr/lib
sudo ln -s libGL.so.1 /usr/lib/libGL.so

```
