### Post by rmemm on 2007-04-12
I don't have a specific answer -- but I have used VMs before.

First -- I think it depends on the VM.  If it's native like VMWare (and I think Xen?) -- and especially if you have a PC that supports the builtin virtualization instruction set that some PCs do today -- I think execution speed should be OK.  You might want to make certain the VM is configured for multi-core if you have a PC with multiple cores too.

Where I would be concerned is video card support.  I know for example in VMWare -- at least the version I'm running you get a virtual video card -- not a high end one usually used in CAD and gamming.  You may not get the graphics support you need.

You might consider running Linux in VMWare on top of Windows if you must (I hate to say that).  That's what I do at work because I have a Windows workstation but use Linux some.
