I installed IE6 and it made a shortcut on my desktop with a nice icon.
I would like to have this shortcut in *Applications->Internet, *so I created a new one there but the shortcut does not have the pretty icon.
How can I make my shortcut have the same icon?

---

### Post by aashay on 2007-09-25
Right click the menu and click on 'Edit Menus'
Click on the Internet icon in the left pane
Search for IE6 in the right pane.
Right click that n select properties.
Right click the desktop launcher n select properties. Here, click on the icon to get its path
Click on the Menu launchers icon n enter the same path here
Should work
