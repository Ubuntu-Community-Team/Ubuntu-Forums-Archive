### Post by frodon on 2006-02-03
to convert your wma fils to mp3, ogg i use this nice script : [http://www.ubuntuforums.org/showthread.php?t=82806&highlight=audioconvert](http://www.ubuntuforums.org/showthread.php?t=82806&highlight=audioconvert)
to read wmw videos you will need to install w32codecs : [http://www.ubuntuforums.org/showthread.php?t=81577](http://www.ubuntuforums.org/showthread.php?t=81577)
xmms and beep media players have some plugins to play wma files, you can find it in synaptic.

If you are completely lost run automatix ;) : [http://ubuntuforums.org/showthread.php?t=66563](http://ubuntuforums.org/showthread.php?t=66563)
