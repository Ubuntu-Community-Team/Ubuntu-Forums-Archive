### Post by f1dave on 2005-09-15
You shouldn't have to create the menu yourself. There is a package, kde-games or something similar, that should have been installed by default. It includes games like kolf, kbounce, kbattleship, kpoker, etc. If you cant run these from a console, it probably hasnt been installed- the only reason i can think that it's not there. By default, if such menus are empty (edutainment, development, etc) then they will not appear in the kmenu.
