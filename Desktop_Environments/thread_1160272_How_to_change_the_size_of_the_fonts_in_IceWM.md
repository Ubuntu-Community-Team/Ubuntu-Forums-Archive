### Post by Frem on 2009-05-15
Are you asking about changing the fonts within the applications themselves, or on the window titles, taskbar, and start menu?

If the former, LXappearance should do it for GTK applications. (unless you've got gnome-settings-manager running or something crazy like that). Note that QT applications like Amarok will not be affected.

If the ladder, yeah, what kerry_s posted. Yes, it's a pain; you have to edit themes. The best way to do it is to set up all your fonts in the IceWM config file. Unfortunately, most theme authors are jerks who think they know what fonts you want better than you do; you'll have to manually delete the font settings in each individual theme, IIRC.
