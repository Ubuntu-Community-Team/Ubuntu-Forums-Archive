### Post by sub2007 on 2007-12-21
I had the same problem, how I solved it was: 

Settings > Preferences > Advanced Desktop Settings.
Click General Options > Desktop Size tab.

Increase the horizontal virtual size to 4.

Hope that helps :)

PS: there is in the Desktop Size panel a setting for "Number of Desktops" but I don't know what that does, mine is set to 1 and the cube works fine - it's the horizontal virtual size that's important
