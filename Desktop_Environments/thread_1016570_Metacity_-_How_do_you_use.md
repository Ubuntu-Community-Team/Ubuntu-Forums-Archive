### Post by gettinoriginal on 2008-12-20
I believe you will get better resolution and effects using emerald.  But that having been said, Install Compiz Fusion Icon from synaptics, and using that will enable you to set your window manager.  To use metacity without the Icon, just type the following in terminal:   :P

gtk-window-decorator --replace &

Themes can be found here:

[http://gnome-look.org/](http://gnome-look.org/)
