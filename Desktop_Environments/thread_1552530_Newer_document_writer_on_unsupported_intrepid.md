### Post by snowpine on 2010-08-13
Hi igsen, Intrepid has reached its "end of life" and is no longer supported. You will not get any more updates or security patches through the Ubuntu package manager. My advice is to use a supported release.

You can always install the latest OpenOffice on any release by following the download instructions from [http://openoffice.org](http://openoffice.org) (obviously this is not officially supported by Canonical).
