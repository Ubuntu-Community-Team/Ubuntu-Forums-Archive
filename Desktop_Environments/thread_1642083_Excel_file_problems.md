### Post by owiknowi on 2010-12-10
With which version of ms office was it created?
I experience some problems with the newest version from ms (yes, it should be compatible and in ODF format...)

Probably obvious: is the .xls is a read only file?
Right click on the file and look for properties.

A workaround (helped me):
Copy the entire content - click on the grey button in the left top of the sheet, next to where you select rows and records, to select all - to a new spreadsheet created with OpenOffice.

Also see: [http://support.openoffice.org/](http://support.openoffice.org/)
