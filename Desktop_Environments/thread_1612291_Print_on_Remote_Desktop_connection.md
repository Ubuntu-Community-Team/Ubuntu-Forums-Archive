### Post by mutex023 on 2010-11-03
For remote desktop connection from ubuntu to windows server 2003, use VNC viewer on ubuntu (install from synaptic package manager) and a VNC server on windows server 2003. The remaining part is just sharing the printer on your win xp machine and accessing it via windows 2003 server, which is pure windows configuration :)
