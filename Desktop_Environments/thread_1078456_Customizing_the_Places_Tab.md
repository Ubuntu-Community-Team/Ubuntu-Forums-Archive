### Post by mcduck on 2009-02-24
Yes. Just open the file manager, then select Bookmarks/Manage Bookmarks from the menu. You can remove and arrange you Nautilus bookmarks there. (Nautilus bookmarks are what you'll see in the file manager's side panel and in the Places-menu.

To add your own bookmarks just browse to that directory and select Bookmarks/Add Bookmark.
