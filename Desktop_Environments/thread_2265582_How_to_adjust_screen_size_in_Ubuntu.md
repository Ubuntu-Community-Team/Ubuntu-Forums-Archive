### Post by grahammechanical on 2015-02-16
What version of Ubuntu are you using? When you installed Ubuntu did you tick the box "Install third party software?" If you did then an Nvidia video driver would have been installed by default. I am assuming that you have an Nvidia graphic card. If you have an AMD/ATI graphic card then an appropriate proprietary video driver would have been installed. In either case a settings utility would also have been installed at the same time. You might find what you are looking for there. Serach for the setting utility in the Dash.
