### Post by DaveKong on 2008-06-19
I don't know all the technical details but by default xclient script will run xfce if you installed xubuntu, gnome if you installed ubuntu, and kde if you installed kubuntu. More advanced users can change the xclient script to perform customized startup tasks. See this thread [http://ubuntuforums.org/archive/index.php/t-452231.html]("http://ubuntuforums.org/archive/index.php/t-452231.html") the links in it or just try googling xclient scripts.
