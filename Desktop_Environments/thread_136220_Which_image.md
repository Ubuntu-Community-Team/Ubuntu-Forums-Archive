### Post by localzuk on 2006-02-25
Not necessarily.

64 Bit application availability is not currently as good as 32bit. Some 32bit apps do not work correctly in the 64 bit environment and running a 64bit environment can create a nightmare of dependencies for supporting older apps.

Personally,  I would say use the 32bit unless you require support for massive amounts of memory. The 64 bit amd processors support 32 OS's at native speeds.
