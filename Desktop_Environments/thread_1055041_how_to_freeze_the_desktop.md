### Post by Yashiro on 2009-02-07
It's either Ctrl+Alt+Right or Mousewheel Up on a blank desktop.
Mousewheel only does this when you're using 'Desktop Effects'.

So either disable these key shorcuts or reduce your number of virtual desktops to 1.
