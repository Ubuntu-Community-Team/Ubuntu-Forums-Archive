### Post by ltmon on 2005-07-16
Firstly, if you press ctrl-alt-backspace does your desktop restart itself?  Have you used Ubuntu before without errors?

Anyhow, if it is a xorg problem you could try installing the official nvidia drivers, given you have a nvidia card:

> sudo apt-get install nvidia-glx
> sudo apt-get install nvidia-settings
> sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup
> sudo nvidia-glx-config enable

Cheers,

L.
