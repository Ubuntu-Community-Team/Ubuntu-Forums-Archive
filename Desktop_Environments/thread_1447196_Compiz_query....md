You don't need both, just one or the other. Uninstall simple-ccsm.

> If I'm not mistaken, the cube effect is where if you have 2 or more apps open in the same desktop window, you can rotate between them, creating a cube-type transition.
You are mistaken. The cube is where you have at least 4 virtual desktops running and then you can rotate between them. I haven't used Compiz in quite a while but you'll need to (at least) set "Horizontal Desktops" to "4" in the General options of the Compiz Configuration Settings Manager.

I'm sure someone who runs Compiz will give you more info shortly. Until then...

### Post by sikander3786 on 2010-04-05
As said by hhh, you only need Advanced Desktop Settings Manger(ccsm) installed on your system. You can easily search for it in the Software Centre.

Secondly you need at least 4 workspaces for the cube effect.

After installing ccsm go to

System --> Preferences --> CompizConfig Settings Manager

Tick the Dekstop Cube and Rotate Cube boxes.

And you should have a rotating cube.

Post back if problems persist.
