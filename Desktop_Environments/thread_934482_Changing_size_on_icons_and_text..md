### Post by slinkey1981 on 2008-09-30
You can change the default size for all of the icons on the Gnome desktop and the file manager, Nautilus.
type "nautilus" into a terminal. The nautilus GUI will appear. In the edit menu select preferences. Under "Icon View Defaults" and "List View Defaults" change the zoom level to 75% or whatever. The results are both immediate and lasting.

The way your system fonts are displayed can be a bit more tricky. Do you use Compiz and/or Emerald?

If using the default Live CD install, you can change all the fonts and sizes by;

Menu > System > Preferences > Appearance 

Click on the tab labeled "Fonts" and you should have a five different fonts that you must change to make them all of the fonts on the screen larger.

*********************

If you use a Window Decorator, you might have to use it's options to change Title Bar Fonts. I did.

For Emerald;

Menu > System > Preferences > Emerald Theme Manager

Make sure the top tab is on "Theme Settings" and then click on the "Edit Themes" tab.
yet another set of tabs will appear. Click the one that says "Title Bar"

On the right hand side it will say "Title Bar" and directly under that are the options for changing the font style and size.

When you are done, click save, overwrite if you want, make a new theme if you don't.


Hope this helps!
