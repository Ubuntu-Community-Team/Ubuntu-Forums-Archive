### Post by Larissa on 2008-03-22
Typically everything you want should reside in your home-directory:
```
~/.kde/share/apps/kmail
```
Please check to make sure before feeling save ;)

---

### Post by firephoto on 2008-03-22
If there was a ~/Mail directory in use the first time you used kmail it might of used that, otherwise the above is what you're looking for.

~/.kde/share/apps/kmail also holds the account information too but there's a specific subfolder for the actual mail messages.
