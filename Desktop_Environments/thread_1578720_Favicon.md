### Post by mcduck on 2010-09-21
IE is rather picky about the use of favicons, and even with supported icon types has a tendency of not showing them.

Anyway, I've never been able to actually get IE to show a .png icon, it only seems to support .ico files for favicons.

edit: I checked this and indeed IE only supports .ico files: [http://en.wikipedia.org/wiki/Favicon]("http://en.wikipedia.org/wiki/Favicon")
