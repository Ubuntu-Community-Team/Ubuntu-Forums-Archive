### Post by ardchoille on 2006-08-13
> **lonelypauly said:**
> thanks guys...lol...i could have figured that out if i wasnt so lazy...i looked around in there too hahaha

so, what about docklets? anything like that available for linux?

You mean dockapps? You can run them, but they won't be in a dock because Metacity (gnome's default window manager) doesn't have a dock. There is an app in the repos called "docker", but be warned, it doesn't play well with the gnome panel. You can, however, switch Metacity out for a different window manager. I wrote a tutorial about how to do that, it's quite simple:

[https://help.ubuntu.com/community/ReplaceMetacityWithOpenbox](https://help.ubuntu.com/community/ReplaceMetacityWithOpenbox)

I got rid of Metacity a long time ago because I think it is the worst pile of junk I have ever seen. There are much better window managers with support for dockapps, customizable desktop menus and such things. You can find a nice list of window managers at:

[http://xwinman.org](http://xwinman.org)

I'm not sure which ones will run well in gnome and which ones won't. I do know that openbox does run quite well in gnome with very little tweaking.

**EDIT**: a friend just told me that she got enlightenment (e16) running as her window manager in gnome and didn't require too much work. I'll have to try that.. I like enlightenment.
