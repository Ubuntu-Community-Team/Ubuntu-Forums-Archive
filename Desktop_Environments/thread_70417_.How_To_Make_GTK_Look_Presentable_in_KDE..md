### Post by MartinG on 2005-09-30
Have you tried gtk2-engines-gtk-qt, available from therepositories?
To quote from the description:
"The GTK-Qt Theme Engine (also known as gtk-qt-engine) is a GTK 2 theme engine
that calls Qt to do the actual drawing. This makes your GTK 2 applications
look almost like real Qt applications and gives you a more unified desktop
experience.
"Please note that this package is targeted at KDE users and therefore provides
a way to configure it from within KControl."
--
MartinG

### Post by Floker on 2005-10-01
Hi there
Like i was angry about that too, so i decided to go crazy and do something about it.
At first, you have to know that im kinda a linux noob, i use linux for half a year now. 
So, here we go.
Start synaptic, enter your password, look at it - looks poor.
Try to find every GTK / GTK2+ related package and install 'em.
Well, i dont knew which packs are needed, so i installed everything that sounds like if it would help :D 
Reboot (dunno if an x-restart is enough, better reboot)
Go to your Contol-center (or whats it called in the english kde)
Select Appearence (or similar) and then "GTK Styles and Fonts"
Set both RadioButtons to "use my kde whatever"
Apply the changes
Reboot again (and again im not sure about that)
Start Synaptic, Gimp, nvidia junk, whatever, look at it and be happy :)
