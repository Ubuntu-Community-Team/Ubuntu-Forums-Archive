### Post by meng on 2006-09-04
I don't think win+D can be set as a shortcut under GNOME, at least not using the standard keyboard shortcuts interface. Win (or superL) is only recognized as a standalone shortcut, not in combination with other keys (except maybe ctrl, shift or alt). In KDE it is different, Win is not recognized on its own but only in combination with another key.
