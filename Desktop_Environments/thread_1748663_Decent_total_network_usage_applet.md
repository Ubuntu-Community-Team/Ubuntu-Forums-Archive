### Post by hamstap85 on 2011-05-03
I'm trying to find a decent applet that shows, over time, how much network I'm using. I'm not talking about the network graph in Gnome System Monitor, I'm talking about the equivalent to the integral curve of that, if you understand calculus.

It would show usage over a user defined period of time, and for the sake of examples, I will assume one day. It would show the total usage so far for the last day, and as time goes on, it would reduce the usage accordingly so that the usage at one day ago is always 0. Those of you with satellite internet (namely WildBlue or HughesNet) know what I'm talking about. It's essentially 'rolling usage'.

It would be awesome if it could not monitor usage if some condition can't be met, or you can manually switch it on and off.

It's okay if it runs in a KDE environment, I'm planning on moving there anyway.
