### Post by smartboyathome on 2007-12-21
Also, use WICD with the network module. Enable it in Configuration > Modules, then add it by right clicking a shelf > configure shelf contents > then add the net module. Now right click the net module and click configure. In the text box below Launch Application on Double-Click, put /opt/wicd/gui.py in. This is way better than Network Manager in GNOME in my opinion.
