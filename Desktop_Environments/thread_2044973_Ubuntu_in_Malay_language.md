### Post by duanedesign on 2012-09-20
Open a Terminal and run the command:

```
locale -a

```

To see if the Malaysian locale is installed. If you do not see it run this command to see what locales ae available:

```
less /usr/share/i18n/SUPPORTED

```

If you see the Malaysian local try installing it by running the command:

```
locale-gen ms_MY

```


You can also get some help from others in your area here:

[http://www.ubuntu.com.my/](http://www.ubuntu.com.my/)
