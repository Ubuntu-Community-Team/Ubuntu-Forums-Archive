### Post by pipodnmy on 2006-06-08
after updating to 6.06 (without updating azureus OR java in any way), i have this small but very annoying problem with azureus - the little popups in the low right corner are not clickable so i can not hide them. very annoying as they stay on top of every thing.
restarting azureus does not work as the first pop up is almost instantenous after restart.
i thought it has to do with gtk theme, so change them around a lot as well but nothing worked.
the only thing (and it is pretty weird) that worked for me is: if u go to Help->About Azureus the problem actually goes away, the popups become clickable.
I hope this helps people as this has been annoying me for some time now.
