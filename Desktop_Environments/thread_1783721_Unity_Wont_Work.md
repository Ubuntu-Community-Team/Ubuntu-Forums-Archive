### Post by 3Miro on 2011-06-17
Virtual Box can run Unity 3D, but only with the latest version of Virtual Box, Guest Addons installed AND 3D acceleration enabled. Even after that, it would be buggy. 3D acceleration under Virtual Box is always buggy.

You can install Unity 2D and that should run OK, look for Unity 2D in synaptic package manager.
