### Post by Rex Bouwense on 2013-03-01
It is relatively easy to turn off the screen saver in Lubuntu.  Go to Menu->Preferences->Screensaver.  Under Display Modes, change the Mode to Disable Screen Saver.
You can check for updates manually using Apt-get, Synaptic Package Manager, or Software Updater.  Conversely you can allow the system to update itself.  Launch the Software Updater and then navigate to settings.  Under Updates you can choose when you want to update.
