### Post by Frogs Hair on 2014-08-27
This function does't seem work since the ability to display the application menu in title bar was added in 14.04. It has been reported and Ubuntu Tweak has removed the setting in the latest version. It might still be in UT because 13.10 was still supported when 14.04 came out.  [https://bugs.launchpad.net/ubuntu/+source/unity-tweak-tool/+bug/1310056](https://bugs.launchpad.net/ubuntu/+source/unity-tweak-tool/+bug/1310056)
