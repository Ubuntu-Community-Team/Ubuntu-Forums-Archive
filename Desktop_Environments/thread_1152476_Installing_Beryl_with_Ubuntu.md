Beryl is no longer updated nor supported. It merged with compiz to create compiz fusion.

[http://www.compiz-fusion.org/](http://www.compiz-fusion.org/)

Compiz Fusion is already installed in Ubuntu, you can install "compizconfig-settings-manager" through synaptic to change compiz's settings and such.
