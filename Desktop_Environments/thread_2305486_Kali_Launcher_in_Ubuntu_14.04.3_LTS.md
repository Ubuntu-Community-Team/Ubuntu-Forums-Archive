### Post by grahammechanical on 2015-12-06
As Kali is not an official member of the Ubuntu family then Kail specific software will not be available in the Ubuntu software repositories. So, it would not be possible to install Kali specific software without adding Kali repositories. And doing that might cause version conflicts with packages in the Ubuntu repositories.

On Ubuntu we can install gnome-session-flashback. That will give us 2 extras login options. Gnome Flashback (Metacity) & Gnome Flashback (Compiz) and with it comes the old Gnome 2 panel look. But I do not remember there being any launcher type thing with that look.
