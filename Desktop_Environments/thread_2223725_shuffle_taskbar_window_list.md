If by "shuffling the window list on the taskbar" you mean dragging and dropping the individual buttons to re-arrange their order, then yes, provided you set the Sorting Order to "None, allow drag and drop" in the Window Buttons properties dialog.

### Post by rybnik on 2014-05-13
The feature exists.  I just hadn't realized how to use it.  Here's how to enable it:

0. Right-click on a panel that contains the window list.
1. Click Panel --> Panel Preferences
2. Select the "items" tab.
3. Select "window buttons".
4. Click the wrench.  (It has alt-text "edit the currently selected item")
5. Under "sorting order", select "none, allow drag and drop"

The actual dragging is noticably less sensitive than that of KDE or Gnome 2.  (not a good thing, in my opinion)

Thanks to Toz and Luke for leading me on to the solution.

I'm glad I discovered that xfce has this feature too.  Now I'm comfortable recommending xubuntu as a beginner distro.
