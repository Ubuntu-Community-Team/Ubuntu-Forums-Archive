### Post by benerivo on 2008-02-24
I think it's done through the [kde control centre]("http://docs.kde.org/kde3/en/kdebase/userguide/kcontrol-kde-components.html"). One of the sections (kde components>file associations) allows you to enter a filter. Type flv in the filter box and if anything comes up then you can change the default to vlc. If nothing comes up, then i think you can make a new file association for a .flv file type. Sorry this is all a bit vague, but i no longer use kde - although i know this is the way to do it.
