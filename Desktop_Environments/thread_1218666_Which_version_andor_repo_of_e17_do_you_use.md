### Post by VCoolio on 2009-07-21
You could use the howto with repos [here]("http://cafelinux.org/OzOs/content/how-install-ozos-desktop-existing-os"). There have been made big changes in e17-svn lately so read [this]("http://cafelinux.org/forum/index.php?topic=3.msg270#msg270") first.

Stability may also depend on svn version. I have version 41055 which is a stable one; the enlightenment [site]("http://www.enlightenment.org/") recommends 40284 (though I'm not sure how often that site is updated). The repo you mentioned seems to contain version 40546. Maybe an update will fix some stability issues. Or you could use a [snapshot]("http://download.enlightenment.org/snapshots/LATEST/") release.
