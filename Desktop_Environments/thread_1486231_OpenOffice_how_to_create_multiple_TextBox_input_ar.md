### Post by ajgreeny on 2010-05-17
What is your difficulty?  I've just tried it and it works for me.

Make the first frame/textbox and then copy/paste it.  You will now need to move the top one away from the first as they will, of course be exactly the same size and in exactly the same place on the screen, ie. the second will completely cover the first.

The simplest way depending on the placement of your text boxes is to make the first and copy it, then move it to the page bottom.  Now paste and move that pasted box, and keep repeating till you have them where you want.

It does work; as I said I have just tried it out, as I seemed to remember doing it once before a long time ago, and wanted to double check.

Incidentally have you looked at glabels, which could do what you are trying to achieve more easily.  Depends what you are using OOo for with this template.

### Post by ajgreeny on 2010-05-21
glabels is a package in the repos that allows quick production of full A4 or Letter pages of the standard label sizes.  You choose label size, enter the text and it will add the same text, or mail merged text different in each box, with just the one type insert.  Type one box, all the rest are filled automatically.
