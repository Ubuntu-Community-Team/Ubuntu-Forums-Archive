### Post by KdotJ on 2010-07-17
Hello and welcome to Ubuntu,

I'm guessing you set up the cube via CompizConfig settings manager?
To get more sides,you need to go back to CompizConfig manager, and select "General Options". Then at the top, the last tab is called "Desktop Size". To make it a cube, you would need to set the horizontal virtual size to 4

Hope this helps!
