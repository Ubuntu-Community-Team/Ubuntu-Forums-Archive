### Post by mcduck on 2009-03-06
I don't think Metacity has any options for window placement, at least there isn't any gconf keys for that.

If you want your terminals tiled neatly I recommend using Terminator (which runs as many gnome-terminals as you want tiled inside one window): [http://www.tenshu.net/terminator/]("http://www.tenshu.net/terminator/"). It's available in Ubuntu's repositories.
