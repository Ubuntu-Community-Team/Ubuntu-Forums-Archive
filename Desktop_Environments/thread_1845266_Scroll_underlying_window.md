### Post by alenis on 2011-09-16
In Gnome 2 I was used to move the mouse over a window, and, without clicking on it and without giving it focus, use the mouse wheel to scroll it. This does not work in Xubuntu, because if I use the mouse wheel when the mouse pointer is over a window which currently has not focus, it immediately gets focus. How can I configure XFCE to behave as Gnome 2 in this respect?

---

### Post by Toz on 2011-09-16
Settings->Window Manager Tweaks->Accessiblity tab and unselecting "Raise windows when any mouse button is pressed", should do it.
