### Post by Note360 on 2006-08-07
ok, go through the menus and turn on Expert Mode it will warn you and then hit ESC to go to normal(vim) mode (hit ESC again to go into typing mode you can stay in typing mode but in order to start the shell you need to be able to issue commands). Then type
```

:!bash

```
That should boot up the bash shell.
