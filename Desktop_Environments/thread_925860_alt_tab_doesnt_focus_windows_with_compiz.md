### Post by billgoldberg on 2008-09-21
Try opening up the "application switcher" plugin in the compiz fusion manager and see if something got unmarked or something.

Or use the "shift switcher" plugin instead. That one is activated by pressing "super + tab".

Super is also known as the meta key or the key with the windows logo on it.

You could set this plugin to open when using "alt+tab" in the bindings dialog of the plugin if you don't want to change your habit.
