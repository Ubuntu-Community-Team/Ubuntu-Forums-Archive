### Post by claracc on 2015-09-26
You have to install the package gnome-shell-extensions. There you can select applications menu extensions in order to have
a menu for your applications.

Please see: [https://extensions.gnome.org/](https://extensions.gnome.org/) and [https://extensions.gnome.org/about/](https://extensions.gnome.org/about/)

Installing gnome shell you can access gnome-desktop, please see:[http://www.howtogeek.com/112620/how-to-install-use-gnome-shell-on-ubuntu/](http://www.howtogeek.com/112620/how-to-install-use-gnome-shell-on-ubuntu/)
