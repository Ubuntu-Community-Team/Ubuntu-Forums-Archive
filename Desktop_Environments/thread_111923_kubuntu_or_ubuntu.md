### Post by Brynster on 2006-01-03
Ubuntu uses the Gnome Desktop
Kubuntu uses KDE Desktop
Xubuntu uses XFCE desktop

Both are the same high quality and both perform extremely well. Gnome interface is very simple intuitive and streamlined. KDE is alot more (don't hate me for saying this) Windows like. Some people do claim that Gnome runs quicker than KDE version however i cannot verify this is i run Ubuntu as my preference is for gnome over KDE.

Xubuntu uses the very light weight XCFE desktop. This is designed for users of older equipment to enable them to experience the joy of Ubuntu, however i have not played with this either and cannot possibly comment on its pro's or cons as i simply don't know them


Hope this helps
