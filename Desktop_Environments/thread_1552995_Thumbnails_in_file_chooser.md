### Post by lovinglinux on 2010-08-15
I have thumbnails here.

Make sure you are using the system file picker and not the one built-in Firefox by setting the preference **ui.allow_platform_file_picker** to true.

You can find instructions on how to do that at [http://firefox-tutorials.blogspot.com/2010/05/preferences-tweaks.html](http://firefox-tutorials.blogspot.com/2010/05/preferences-tweaks.html)

 If that doesn't help, then check Nautilus preview settings.
