### Post by mayagrafix on 2012-08-20
Was having some streaming video playback problems, so I entered CMOS and and upped the allotted memory for the Dynamic video memory technology (DVMT) to MAX (it was set at 8 megs). 

Not only did the streaming improve but the Launcher doest not "stick" no more and works much better IMO.

Just a heads up for those who might be having problems with their Unity experience. :KS
