Pretty much a standard feature of every Linux window manager since the first days of graphical desktops.. :D

Just go to System/Preferences/Windows and set the "Titlebar Action" to "Roll Up".

edit: it's also possible to do this with mouse wheel, which I think is even smoother than the doubleclick. To do that press Alt-F2 and run "gconf-editor", browse to apps/gwd and set the "mouse_wheel_action" to "shade". If you use Emerald then you'll find the setting in Emerald Theme Manager.
