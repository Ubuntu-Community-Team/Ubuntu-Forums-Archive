```

**Backup your xorg.conf file:**

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak

**Open it up with an editor:**

sudo kate /etc/X11/xorg.conf 

**Then comment out (Add a # in front of) the red areas as shown in my example:**
```
