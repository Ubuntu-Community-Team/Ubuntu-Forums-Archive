### Post by bdb112 on 2012-05-18
in Ubuntu 12.04, using a metacity VNC desktop, the control-D Key functions as  show-desktop, which makes it hard for emacs and other use.
By referring to 
[http://ubuntuforums.org/showthread.php?t=1595871](http://ubuntuforums.org/showthread.php?t=1595871)
in which similar problems with the D key  (not control-D) were solved by using
gconf-editor, and disabling the show-desktop function, I worked around by changing
the setting of show-Desktop to <Super>-d, and logging out/in.
