### Post by Arathorn on 2008-07-14
If you like to keep things light and install only the programs you want to use, install kde-core. Full KDE installs a lot of programs you're not likely going to use. Off course you can remove those programs later, but I personally feel the first way works better. Look [here](http://psychocats.net/ubuntu/kde-core) for more information.
