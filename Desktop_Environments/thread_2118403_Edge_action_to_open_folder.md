Is there a way to set an edge action (e.g. when I hover mouse over bottom right corner) to open a folder, such as my "Documents" folder?  I am running on Ubuntu 12.10 with Unity.

Wayne

---

### Post by BigCat4 on 2013-02-21
Yes, use CCSM / General / Commands.  Assigned the action you would like under the Commands tab, then assign the corresponding edge action under the Edge Bindings tab.  If you use Ubuntu Tweak, be careful not to duplicate edge actions from Tweaks / Workspace.
Have Fun!
