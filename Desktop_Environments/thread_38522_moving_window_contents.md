### Post by shakin on 2005-05-31
I don't know how to *just* turn off window content dragging, but you can turn it off along with some other animations from gConf (install with synaptic if you don't already have it).

Run it from Applications -> System Tools -> Configuration Editor

Now expand the tree from Apps -> Metacity -> General

Now check 'reduced resources'.

Close gConf and the change should take effect immediately.
