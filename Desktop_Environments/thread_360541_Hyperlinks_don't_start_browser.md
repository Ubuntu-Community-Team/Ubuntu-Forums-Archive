### Post by elreteipos on 2007-02-25
I figured it out and it was a rather stupid problem. I had KDE set to execute the command "mozilla firefox" *(which is of course invalid)* whenever I clicked a URL. Here's how I fixed it.

Go to System Settings > Default Applications and choose Web Browser. Select the second radio. Use the "..." button to locate the web browser of your choice. Don't forget to click the Apply button before closing the window. It should work now.
