### Post by frodon on 2006-02-09
Skype is ugly because it use QT libraries (KDE) and you're using gnome which based on GTK libraries, if you want skype to be less ugly look here : [http://doc.gwos.org/index.php/Gnomeish_QT_apps](http://doc.gwos.org/index.php/Gnomeish_QT_apps)

For the webcam support look at the version you use .... it don't offer web cam support, we have to wait the next release for that but it will come really soon.
