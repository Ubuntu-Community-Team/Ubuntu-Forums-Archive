### Post by GTvulse on 2005-08-09
The X graphics system uses a client-sever architecture based on TCP/IP and Unix sockets. The X process (which displays graphics on the screen) which many call the "X server" is really a **client** that listens for application servers that provide graphics information, such as your web browser. You usually use it locally, but you can use it across the network, by design.

Orbit is part of the Gnome desktop and it is an interprocess communication (IPC) server that uses the CORBA standards to broker client and server requests between the different components of the Gnome desktop and the applications that use the desktop component services such as, again, your web browser.

P.S. The fact that your network interface shows two IP addresses means that you are doing NAT behind a router.
