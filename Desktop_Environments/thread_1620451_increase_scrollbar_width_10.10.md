### Post by gauravm on 2011-06-24
install gnome-color-chooser from synaptic or ```
sudo apt-get install gnome-color-chooser
```

Then launch: System->Preferences->GNOME Color Chooser
Under Tab: Specific, you can change the scroll bar width.  Then Click Apply.

It should affect almost all software.  I have noticed that Firefox doesn't get affected.
