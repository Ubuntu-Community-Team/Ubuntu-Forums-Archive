### Post by FyreBrand on 2006-09-21
Have you tried just letting the partition manager partition and install the system itself?  If you are new to Linux or any OS for that matter you should let the installer do what it wants until you learn what you're doing.  That can take a little time and research.

If you are dual booting then you should read this: [**[COLOR="Teal"]Windows Dual Boot[/COLOR]**]("https://help.ubuntu.com/community/WindowsDualBoot").  There is a detailed guide here: [**Detailed Guide**]("http://users.bigpond.net.au/hermanzone/p3.htm").

There is also some good reading and advice on the aysiu's psychocasts site here: [[COLOR="DarkRed"]**psychocats.net -- Ubuntu Linux Resources**[/COLOR]]("http://psychocats.net/ubuntu/index.php").  I love this site and it's very helpful.

Also a final recommendation would be to use the search function.  You can find many helpful answers and even step by step HowTo's on the forums here.  I love Ubuntu because it's a great OS, but the community and technical support of these forums has kept me because I've been able to find solutions to common problems from friendly people.
