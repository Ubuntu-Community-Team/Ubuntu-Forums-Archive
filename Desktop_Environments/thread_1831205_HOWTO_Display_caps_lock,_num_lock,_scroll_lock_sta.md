### Post by skief on 2011-08-23
If you have a laptop that lacks LEDs to indicate the state
of the lock keys, then this how-to is for you.

This information is already out there, but I found it only in
scattered places. Here I gather it together.

There is an applet that you can get with

```

sudo apt-get install lock-keys-applet

```

Next you need to add the applet to your panel.
Right click somewhere in the blank space on
your panel, and select Add to Panel...

Scroll down and find "Lock Keys". Select it, click Add, close the
window.

Now right-click on the Lock Keys applet that has just been added to
your panel, go to Preferences..., and select those lock indicators
that you want.

That's all.
