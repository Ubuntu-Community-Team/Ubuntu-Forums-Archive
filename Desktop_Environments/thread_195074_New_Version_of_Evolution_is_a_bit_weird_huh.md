### Post by ayoli on 2006-06-12
for the first question, you have defined shortcuts to swap views (calendar, mail, contacts,...) they are Ctrl+1, Ctrl+2, etc
you can modify them if you have enabled editable keyboard shortcuts (in system > preferences > toolbars and menus)
sorry cant help for the2nd question.
