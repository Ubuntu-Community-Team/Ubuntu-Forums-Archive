### Post by 888gavin on 2008-06-07
I'm trying to set up Nautilus so that every single folder has the same view options as the others. I want every single folder to display files in Icon View at 100% zoom. Even if I set a default in Nautilus, some of my folders still have files display in List View at 33% zoom. Is there a way to make all the folders have the same view options?

---

### Post by Panna-cakkhu on 2009-02-08
Hi Gavin,

I assumed you already set your Defaults preferences via "Edit" --> "Preferences" (in any Nautilus window).

Once you have change the defaults settings, the command "View" --> "Reset View to Defaults" should reset the folders for which you still have files display in List View at 33% zoom.

### Post by palantir6er on 2011-01-13
In nautilus, go to Edit -> Preferences and under the view tab the first option is to set the default view for new folders. 

You've probably already set the folders you normally visit to list view and this will change any others. I hope that helps anyone who maybe stumble across this thread.
