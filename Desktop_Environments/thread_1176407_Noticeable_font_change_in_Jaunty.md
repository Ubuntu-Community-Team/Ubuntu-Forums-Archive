### Post by Blittzd on 2009-06-03
In Appearance Preferences, go to the "Fonts" tab. Check the box for "Subpixel smoothing" (this is assuming you are using an LCD). Click "Details," Then the check the box "Full."

This made my fonts look much crisper. Also, you can change the font type & size in the same "Fonts" tab in "Appearance Preferences" if the fonts or font sizes aren't to your liking.

Hope this helped :)
