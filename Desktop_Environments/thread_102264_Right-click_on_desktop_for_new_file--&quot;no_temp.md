### Post by Zelut on 2005-12-11
What I did to solve this (not sure if this is the *correct* way, but it works) is create a folder Templates/ in ~/ and create empty files for each 'template' I would like.  For example I have an empty gedit file, gedit & an empty OpenOffice template, Template.odt.

After creating those files in Templates/ that right-click > Templates now works.
