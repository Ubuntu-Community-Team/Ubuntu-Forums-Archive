### Post by canadianwriterman on 2005-12-30
I assume you're talking about the default Ubuntu wallpaper. Just create a folder in your Home directory and put any downloaded wallpapers into it. Then, rightclick on the existing wallpaper. Choose install new background and navigate to the wallpaper you've stored in your folder. Select it and you're done!
