### Post by ebash on 2006-09-21
You can install xubuntu over an ubuntu/kubuntu distribution. It's quite easy, everything will be left unchanged on your computer. To install xubuntu do the following:

```
sudo apt-get install xubuntu-desktop
```

Then logout, no need to reboot. At the login manager choose the xubuntu session. That's it!
