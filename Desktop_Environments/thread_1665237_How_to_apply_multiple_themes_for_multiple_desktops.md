Do you have several different machines, several accounts on the same machine or several "workspaces" on the same machine.

You can easily do different themes on different machines/accouts by downloading them from Gnome-looks.

[http://gnome-look.org/](http://gnome-look.org/)

You cannot have different themes per-workspace, since only one theme can be used by a user at a time. (would be cool though)
