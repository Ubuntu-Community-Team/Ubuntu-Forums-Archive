A good user interface should be one that feels as though it does not exist (as though it does not stand between you and your computer).  Even though some people have trouble learning it, I think the best user interface model is that which Blender ([www.blender.org](www.blender.org)) employs.  It is organized and intuitive, and does not waste any space on the monitor, or overuse any space by overlapping panels.  On this note, I have the idea to create a DE that will extend this model from Blender to the desktop.  I have received some positive feedback so far, and I would like to know if anyone else is interested in helping in some way.  I an proficient in Python, but rather rusty at C,C++ and need help in those areas.
