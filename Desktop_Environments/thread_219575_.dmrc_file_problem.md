### Post by simonn on 2006-08-09
I dunno, you could always try doing what the error messages says? Maybe? Perhaps?

*"File should be owned by user and have 644 permissions."*

```

$ sudo chown [user]:[group] [home]/.dmrc
$ chmod 644 [home]/.dmrc

```

*"Users $HOME directory must be owned by user and not writeable by other uses."*

```

$ sudo chown [user]:[group] [home]
$ chmod 750 [home]

```

NOTE: 750 is better than 755 as only you have read/write access and members of your group (which is probably only you, but opens up options if you want) have read access.

EDIT: I suspect that you accidentally gave your home dir write permissions to everyone and/or gave .dmrc 777 permissions.

EDIT2: If you are going to be using any form of *nix, learn how permissions work. They are more fundamental than being able to copy files and IME most mysterious problems end up being permissions related. Have a read of the rute tutorial in my sig.
