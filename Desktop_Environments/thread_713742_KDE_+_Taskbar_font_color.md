Hello,

i want to use a dark/black colored background for the taskbar, thus i need to set a brighter color for the taskbar fonts. I achieved this by setting the "Window Text"(= inactive tasks) and "Button Text" (=active tasks) colors in kcontrol/appearence. However this brightens up my normal window/button colors as well, so it seems like buttons are disabled, while they are just brighter than normal.
is there any way to change the font color of taskbar items without changing the normal kde button/windowtext colors?

### Post by .nedberg on 2008-03-03
Not sure if this is what you want, but it is what I do (and I also use a dark taskbar).

Right click taskbar -> Configure Panel. Go to Taskbar and select Appearance: "For Transparency". It will give you dark text with a light border.

I don't think there is any other way of doing this...

### Post by bibstha on 2008-04-30
Nice Vista Theme there
you can independently set the color of the fonts in the taskbar/.
1) Click on the handle of the taskbar
2) Configure Taskbar
3) The taskbar option is selected in the left, on right click on use custom colors and select the color that you want.
