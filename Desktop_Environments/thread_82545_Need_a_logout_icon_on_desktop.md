### Post by jr.gotti on 2005-10-26
Hmm....the terminal command for shutting down it

```
shutdown -h now
```

btw...if you want to restart from the terminal switch h (halt) with r (restart)

### Post by gillion on 2005-10-26
you can put the logout icon directly on the panel where she does not have to navigate anything. same way default KDE does it.

right click on the panel, select add to panel, choose the logout option, drag it to where you want it to be.
