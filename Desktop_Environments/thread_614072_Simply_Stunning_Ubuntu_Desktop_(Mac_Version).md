### Post by yuvlevental on 2007-11-15
Note: this tutorial is designed for a gutsy desktop. It requires the ability for your computer to run composite effects.

Now many of you have seen the simply stunning ubuntu desktop:
[[IMG]http://img441.imageshack.us/img441/7875/mattsdesktopuz6.th.png[/IMG]](http://img441.imageshack.us/my.php?image=mattsdesktopuz6.png)

I think it is ok, but I wanted something mac-esque, so I took the default gutsy desktop and modified it to look like this:

[[IMG]http://img441.imageshack.us/img441/8950/screenshot3at8.th.png[/IMG]](http://img441.imageshack.us/my.php?image=screenshot3at8.png)

[[IMG]http://img441.imageshack.us/img441/2029/screenshot1xp3.th.png[/IMG]](http://img441.imageshack.us/my.php?image=screenshot1xp3.png)

Steps:
1. Type the following into a terminal:
```

sudo apt-get install compizconfig-settings-manager emerald

```
2.Go[ here]("http://packages.ubuntu.com/feisty/x11/emerald-themes"), download, and install the "emerald-themes" package for feisty, but it works
3.Starting with the black background and reflection in the cube background and cube caps: Right-click desktop, change background, visual effects, custom. select the plugins "rotate cube" and "cube reflection" and "cube caps". answer "change settings" to any popups. go to "desktop cube" settings. Go to appearence -> skydome -> make both gradients 100% black.
4. Emerald themes: system -> preferences -> emerald theme manager. Select the "human-ubuntulooks" theme to keep the human theme.
5. right-click bottom panel, select delete.
6. The dock used is the awn dock. Go [here]("http://wiki.awn-project.org/index.php?title=DistributionGuides#Reacocard.27s_Ubuntu_Gutsy_Repository") to install
7.Launch awn, and install the theme [Milky]("http://www.queervisions.com/img/awn/QV_Milky.tgz"). Install by right-clicking on bar, preferences, themes. install themes, select the .tgz package and press install, then apply. Make the corners square in general -> appearence -> uncheck enable round corners.
8. Transparencies: for the top panel, right-click, properties, background, drag transparency slider so that the right edge is right after the "o" in opaque. For the terminal, Open the terminal, click Edit then Current Profile, click the effects tab and choose transparent background. make the transparency about 7/10 ths full opacity.
9.wallpaper can be found at [http://art.gnome.org/download/backgrounds/nature/1326/NATURE-LeafPerspective_1024x768.png](http://art.gnome.org/download/backgrounds/nature/1326/NATURE-LeafPerspective_1024x768.png)
9. That's all! Enjoy your Simply Stunning Ubuntu Desktop (Mac Version)!
