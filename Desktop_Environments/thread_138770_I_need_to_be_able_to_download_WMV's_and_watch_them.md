### Post by jrib on 2006-03-02
Install the [w32codecs.]("https://wiki.ubuntu.com/RestrictedFormats#head-fda9cc5147253891fe3047263b82d787ab025bba")

Enable [universe]("http://wiki.ubuntu.com/AddingRepositoriesHowto") if you haven't already.  Then install totem-xine using system > administration > synaptic.
