### Post by kd_jm on 2009-04-29
a) Preferences -> Switch Desktop Mode
b) Change Workspace Switcher Preferences (increase number of workspaces)
c) Preferences -> Switch Desktop Mode again

I don't know how to add the minidesktops to the panel, but you can use the shortcuts.
