### Post by efflandt on 2010-02-28
If this is just a live/install iso made with USB Startup Disk Creator, you shouldn't have to do anything.  It should auto login as default user "ubuntu".  If you have persistent data, you can create other users.

If it is a regular installation on USB, you should have remembered who you set up as user.  And in that case I hope you put grub on mbr of USB (instead of default to mbr of main drive).
