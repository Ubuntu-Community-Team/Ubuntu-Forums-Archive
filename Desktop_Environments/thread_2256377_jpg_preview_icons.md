### Post by papibe on 2014-12-11
Hi alek5a.

You can either change the icon's size momentarily by going to
```
View -> Zoom out
```
or set the default bigger in
```
Preferences -> View (tab) -> Icon View Defaults (default zoom level:)
```
and then close and open again Nautilus (the file manager).

Hope it helps. Let us know how it goes.
Regards.
