### Post by Frogs Hair on 2012-07-26
You can reset the launcher icons to default with the following command, but you will have to add your favorites again. I don't know the cause of the problem or if the command will solve the it . ```
unity --reset-icons
``` 

An application has to include a panel icon before it will appear in Unity . You can't add items by dragging and dropping and there is no add to panel menu. [http://askubuntu.com/questions/30334/what-application-indicators-are-available](http://askubuntu.com/questions/30334/what-application-indicators-are-available)
