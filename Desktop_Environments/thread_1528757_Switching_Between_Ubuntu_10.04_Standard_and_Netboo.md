### Post by KdotJ on 2010-07-11
If you have the desktop version if ubuntu currently installed, you do get the netbook interface by doing the following:

type this into the terminal:

```
sudo apt-get install ubuntu-netbook
```

then you can reboot (or logout) and on the log in screen, you can select whether to boot log into GNOME (the normal one) or Ubuntu Netbook Edition by selecting either one from the Sessions drop-up box
