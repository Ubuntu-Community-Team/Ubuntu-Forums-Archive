### Post by MatthiasM on 2007-04-04
Hit the orange function-change-key switcher one time and the keys become normal F1-F12 keys. Hit the orange function-change-key switcher again, now MS Word, Excel, PowerPoint, Undo, Redo are active which are not yet supported. See *[Kernel Bug Tracker: Logitech Cordless Desktop Remote S510 - some keys do not work]("http://bugme.osdl.org/show_bug.cgi?id=7352")*
