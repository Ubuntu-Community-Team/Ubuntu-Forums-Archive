### Post by xycris on 2012-08-08
hi izzygalvez,

it is easy to install gnome-screensaver:
```
sudo apt-get install gnome-screensaver
```

and uninstalling xscreensaver:
```
sudo apt-get purge xscreensaver
```

however, upgrading to 12.04 from 11.04 just completely does this for you. 

gnome-screensaver has a good looking log-in window while the only screensaver it does is just a black screen. they say, it is more energy saving that way.
