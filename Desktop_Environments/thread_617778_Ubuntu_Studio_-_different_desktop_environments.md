Desktop Environments are basically a more complete version of a window manager (they handle programs, panels, toolkits, window decorations, and pretty much everything else you see in GNOME/KDE). Window Managers rely on external apps to be more fuction via GUI (they operate more on text files and the command line). Hope this helps explain it to you!
