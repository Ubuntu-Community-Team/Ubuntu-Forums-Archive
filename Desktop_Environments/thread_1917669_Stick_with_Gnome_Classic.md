### Post by ColdnitroX on 2012-01-30
If you still want to use Ubuntu Classic (or GNOME2), consider using MATE, a GNOME 2 fork. It's pretty stable now, and mostly compatible with GNOME 3 and Unity. You can use all three on the same system and it still works pretty well. Get MATE at [http://mate-desktop.org]("http://mate-desktop.org"), and you will get the Ubuntu Classic experience, mostly... :D.
