### Post by realzippy on 2009-11-04
Install ccsm to configure/enable installed plugins.
More plugins to compile (screensaver,cubemodel,atlantis and many more):

[http://forum.compiz.org/showthread.php?p=68054](http://forum.compiz.org/showthread.php?p=68054)

(works in karmic also)

open synaptic and search for "compiz" to check if all available plugins
are installed.Here you also find ccsm (compizconfig-settings-manager)
