There's no download involved. You can just follow the instructions given right below the picture to get this theme, and it will install directly from the repositories. Here's what to do, from the link you referenced:
 
```

For those on Ubuntu, you can install with:
 
sudo add-apt-repository ppa:elementaryart/ppa
sudo apt-get update
sudo apt-get install elementary-theme
 
## IMPORTANT ##
Murrine engine is required for this package.
Nautilus elementary is HIGHLY recommended. There WILL be visual glitches without it.
## IMPORTANT ##

```
