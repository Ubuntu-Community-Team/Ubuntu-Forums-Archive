### Post by eggdeng on 2007-01-23
Think cinelerra might be a tall order. I use kino which is lightweight, has a nice user friendly interface & is only slightly buggy. Once you get the hang of it, you can get first class results quickly.
```
 sudo apt-get install kino
```
& you're good to go.
