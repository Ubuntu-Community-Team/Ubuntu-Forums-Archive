### Post by ugm6hr on 2009-11-29
If you want to start from scratch, try something like adesklets with yab Bar for launchers: [http://adesklets.sourceforge.net/verbatim/yab.html](http://adesklets.sourceforge.net/verbatim/yab.html)

You can use whatever WM you want, although XFCE is perhaps the least "integrated" (excluding LXDE), which means picking and choosing components is perhaps easier.

Out of interest, I recently came across Eldy, which might give you some inspiration: [http://www.eldy.eu/](http://www.eldy.eu/)
