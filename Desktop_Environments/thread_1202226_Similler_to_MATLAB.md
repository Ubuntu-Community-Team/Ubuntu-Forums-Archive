### Post by Hetepeperfan on 2009-07-11
You could also try Octave.
It is not so fancy as matlab, but it sure works just like matlab. In contrast to matlab it's open source. Try: "sudo apt-get install octave"
It also needs gnuplot thus "sudo apt-get install gnuplot" as well.

For more info see [http://www.gnu.org/software/octave/](http://www.gnu.org/software/octave/)
