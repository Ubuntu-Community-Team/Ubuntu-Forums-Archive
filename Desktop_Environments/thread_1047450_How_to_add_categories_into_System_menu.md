I don't know about going as far as underneath the "About Ubuntu" one but you can go to 

System -> Preferences -> Main Menu

From inside this application, on the left hand side click "System".  Then, click "New Item" on the right.  Point it to the command you would like to run.  If you would like to make a new menu then click "New Menu" on the right. This will place it right underneath "Administration" in the System menu.  I do not know how you could move it though...
