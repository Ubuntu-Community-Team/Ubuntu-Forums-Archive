### Post by mcduck on 2006-01-18
The Widget Factory is just a tool for theme makers to show all widgets (buttons etc.) at the same time so you know what you're doing (as the theme making/editing itself is mostly (if not completely) done by editing a text file)

I think that you could edit the icon theme to make icons smaller. It's not exactly a GTK thing, and none of GTK themes I've edited have had anything about icons. (that's not to say that GTK theme couldn't do anything about icons, just that most themes don't)
