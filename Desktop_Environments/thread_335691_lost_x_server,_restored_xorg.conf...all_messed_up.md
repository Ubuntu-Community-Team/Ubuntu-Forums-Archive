### Post by Stephen Howard on 2007-01-10
Loss of title/border/frame is a classic problem associated with messing around with Beryl.  Maybe its not as 'uninstalled' as you hope?

That said, did you edit the xorg.conf file just before the problems began?  If so, do you have a backup and/or can you point to what you edited?

Have a look at /var/logs/Xorg.0.log and see is there are any error messages "(EE)" that might give us a clue.  Look especially for a section that lists 'validated modes', and see if there are any errors nearby.

Also, according to your xorg.conf file, you aren't using the nvidia proprietary driver.  Is that right?  It seems a bit odd given that you're messing around with Beryl.
