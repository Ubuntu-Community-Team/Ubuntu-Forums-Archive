### Post by dolby on 2006-08-04
1. ```
sudo aptitude install update-notifier
```
& applications-system-software properties-internet updates

2.i think theres a how to create desktop items somewhere in the forums. try searching. if u want u can change the right click action in settings-desktop preferencies-behaviour

3.make a new launcher on the panel

4.u can switch between workspaces by scrolling with mouse on the desktop

btw u can have all the gnome u want by installing selected packages, u just gotta know which one ;)
