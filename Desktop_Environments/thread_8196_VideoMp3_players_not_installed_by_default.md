> How do I list all the programs already installed on my computer?
There's two ways:

1) Fire up Synaptic (Computer->Sys. Conf.->Synaptic) and click the header of the row "Installed Version". This puts all installed software packages in the top of the list.

2) Hit alt-F2, and expand the list of known programs :)

### Post by adbak on 2004-12-15
For XMMS open up Synaptic and install the package XMMS.  To play MP3s, which I assume you want XMMS to do, you also have to install the package "libmikmod2", which I believe is in Universe.  (how to enable universe [http://www.ubuntulinux.org/wiki/AddingRepositoriesHowto#uncommenting-universe](http://www.ubuntulinux.org/wiki/AddingRepositoriesHowto#uncommenting-universe) )

For playing movies, I'd use gxine, which is also in Universe, but if you want you can follow the Ubuntu Multimedia Howto which demon666_nl has kindly linked you to.  Also make sure to install mozplugger (which you can find in either Universe or Multiverse repo or on the second page of the Ubu Multimedia Howto).  Mozplugger lets you watch video and listen to audio embedded in websites.

As far as listing all the programs you have installed, open up Synaptic.  On the very left of the window you'll see the Reload button and below that, a collapsible menu.  I forget if that's what it's called, but that's what I'm calling it.

Click the arrows to bring up your choices and click Status.  Below that menu, there will be a list with the programs that you have installed.
