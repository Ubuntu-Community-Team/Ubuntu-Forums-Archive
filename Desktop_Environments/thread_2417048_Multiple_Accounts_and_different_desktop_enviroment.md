### Post by Frogs Hair on 2019-04-19
Hello and Welcome !

Additional or different desktop environments would have to be selected in the login screen regardless of user name . If user Joe was setup on gnome then Joe needs to select gnome during login and that would be his default. User Joe could also have an account in cinnamon, but the login screen still comes into play to select the cinnamon DE.
