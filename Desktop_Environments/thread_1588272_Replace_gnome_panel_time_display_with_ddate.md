### Post by sanderd17 on 2010-10-04
If you open the gconf-editor (alt+F2 and type "gconf-editor"), you can go to apps->panel->applets->clock_...->prefs.

There you can set the field "format" to the value "custom" and fill the custom format in the field "custom". I didn't try with ddate though. Don't know if that will work.
