### Post by jviscosi on 2007-09-23
Probably the closest you can get to this is to install an icon theme that works in Gnome and KDE and use it in both, and pick or create similar theme colors.  

In Openbox, I use gtk-chtheme to pick my GTK theme and kcontrol to pick my KDE theme.  If I can't find a good fit between the two, I make a custom theme by matching the colors in kcontrol using the eyedropper tool to grab them from gtk-chtheme.  For icons, I've had good luck with Noia, Crystal, and Aaron Spuler's icon sets such as Apollo.

It might not be exact but it's enough to keep the differences from being jarring.
