### Post by NEUR0M4NCER on 2007-12-21
Hiya - i'm pretty new to this all myself, but I got Compiz and Emerald figured out -

Emerald handles the window borders (The title bar, sides and bottom)

Gnome (GTK or the Appearance settings) handles everything unless you've got Emerald running - which you have. Whilst Emerald is running, Gnome only changes the window contents. That's everything inside the window border (buttons, colours, top and bottom panel, etc).

Compiz deals with how the windows as a whole are drawn by the computer, so the settings there will change how windows maximise, minimize, open, close, etc.

Have a look at [gnome-look]("http://www.gnome-look.org"), that might make things a little clearer.

Hope this helps.

### Post by Tenken on 2007-12-21
To customize GTK (panels, scroll bars, menus, etc.) go to System->Appearance, click the Theme tab and there is a button near the bottom of the window that says Customize. This is where you can change fonts, icons, mouse pointers, and the GTK theme. Ignore the Metacity tab, that is the default window manager that Gnome uses, it is replaced by emerald if you are running it.
