### Post by mcduck on 2010-04-08
Make sure that your display actually supports such resolution.

Other possibly relevant things are if you are using visual effects or not, and what kind of graphics card you have. If you *are* suing visual effects, and your graphics card has little memory, it might be that the memory it has isn't enough to run the visual effects on that high resolution. If you think this might be the case then try turning off visual effects and *then* setting that resolution.
