### Post by The Cog on 2012-10-25
There's GTK2 and GTK3. Some progs use one, some use the other. Clearlooks only works in one of them. 

This similar theme works in both - it's the one I've begun using:
[http://gnome-look.org/content/show.php/Clearwaita?content=145210l](http://gnome-look.org/content/show.php/Clearwaita?content=145210l)
Unzip it and copy the resulting folder to /usr/share/themes.
