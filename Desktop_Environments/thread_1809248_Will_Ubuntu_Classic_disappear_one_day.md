### Post by snowpine on 2011-07-21
Gnome Foundation has dropped support for Gnome 2.x. All of the fast-moving distros like Ubuntu and Fedora are leaving it behind. 

There is a strong possibility somebody will "fork" a desktop environment based on Gnome 2 (just like what happened with Trinity and KDE 3.x) in which case you'll probably be able to install it through a PPA.

Gnome 3 and Gnome shell are not the same thing. Ubuntu will be using Unity 3D by default in 11.10 (like 11.04) running over Gnome 3 (gnome's software stack). The fallback **will** be Unity 2D. Gnome-shell (gnome's UI for Gnome 3) will be available to download from the repositories. So yes, Unity 2D will be the fallback. But if you like, you can install Gnome-shell and force fallback mode.

### Post by MARP1961 on 2011-07-23
I have been trying Gnome 3 (with Gnome Shell) installed via a ppa in Natty 11.04. The whole family is using it with ease after hating Unity.

However, if you still want a 'Gnome 2 type experience' in the future, you can use Gnome 3 Shell with forced Fallback Mode. This will give you a top and bottom panel and I believe it is increasingly likely to be customizable as Gnome 3 gets further developed.

Another option is Linux Mint, which seems bent on avoiding both Unity and Gnome Shell at present. Since it is closely associated with Ubuntu, it will be using Gnome 3 from version 12 sometime in November. Presumably they will use a more traditional approach as is their habit.

XFCE is just a poor man's Gnome 2. Plain and with a silly mouse logo/branding everywhere.
