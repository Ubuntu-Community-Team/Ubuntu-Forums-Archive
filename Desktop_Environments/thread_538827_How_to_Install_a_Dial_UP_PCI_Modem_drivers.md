### Post by ray bot on 2007-09-04
PCI modem support is pretty limited in Linux. You can check [http://linmodems.org/](http://linmodems.org/) to see if yours is supported, and to download drivers if it is. If you don't know which chipset your modem uses, typing lspci in a terminal should tell you.
