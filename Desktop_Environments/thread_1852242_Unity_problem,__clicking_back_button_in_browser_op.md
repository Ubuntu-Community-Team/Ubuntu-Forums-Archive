### Post by Copper Bezel on 2011-09-29
Really a thread for the Desktop Environments forum.

If you're finding the Launcher fiddly, install compizconfig-settings-manager (if you don't have it installed already) to reconfigure it a bit. Run ccsm and find the Ubuntu Unity plugin, then change the "Reveal Mode" from "Left" to "TopLeft" (click the button at the end of the line from "Reveal Mode," which currently says Left, which will bring up a little window with a graphic of the display; unselect the left side and select the upper left corner instead.)

That way, the Launcher will appear only when you swing the mouse to the upper left corner, past the Ubuntu logo ("Home Button.")
