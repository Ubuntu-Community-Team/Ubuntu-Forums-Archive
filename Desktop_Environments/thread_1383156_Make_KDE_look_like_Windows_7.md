### Post by benerivo on 2010-01-17
In kde the 'icons instead of titlebars' functionality is a plasmoid called smooth-tasks which can be found in kde-look...

[http://www.kde-look.org/content/show.php/Smooth+Tasks?content=101586](http://www.kde-look.org/content/show.php/Smooth+Tasks?content=101586)

To make it look more like Windows 7, you are best off searching kde-look for "windows 7", to get themes, icons, etc..

It is also worthwhile checking all the theme options in system settings, to see if there is anything you can tweak.
