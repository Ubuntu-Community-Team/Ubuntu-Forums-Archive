### Post by ChadMMc on 2006-10-07
There should be an entry in your 'Preferences' menu titled 'Login Photo'. If you do not see it there, run Accessories->Alacarte Menu Editor, go down the list to the preferences menu and you should be able to activate the menu entry.
Hope this helps.

### Post by ChadMMc on 2006-10-07
Oh, I also came across what seems to be something new (well, at least new to me) under the Admin menu. The 'Ubuntu Control Center' entry. That has an entry under the Personal tab that allows the change of the login photo too.
