### Post by netcyrax on 2009-10-29
just use the CompizConfig Settings Manager (you will have to install it via the Ubuntu Software Centre)..

Then run it (System -> Preferences), go to ViewPort Switcher (under Desktop) and in the "Desktop-Based Viewport Switching" tab disable everything you don't need.
