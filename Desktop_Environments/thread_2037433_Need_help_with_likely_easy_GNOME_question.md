### Post by markbl on 2012-08-04
FYI, note that Gnome shell is the Gnome 3 UI available when you log in with "GNOME" from the login session menu. You are logging in with Gnome Classic which has nothing to do with Gnome shell. It is a UI which emulates the older Gnome 2.

You complaint is about [startup notification](http://developer.gnome.org/integration-guide/stable/startup-notification.html.en). That's how it is supposed to work. I've noticed that Nautilus starts slowly the very first time you launch it in a gnome session on ubuntu. Probably an interaction between ubuntu/gnome but really it's a very minor problem.
