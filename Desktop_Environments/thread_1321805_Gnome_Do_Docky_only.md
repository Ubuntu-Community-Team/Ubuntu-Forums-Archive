### Post by RochJer on 2009-11-10
It sounds like you are looking for specific dock navigator software that emulates the MAC OS X docking station. 

To get this feature enabled in Ubuntu, they have multiple docking station as you mentioned "widgets" - it is AKA AWN Navigator. You can enable those in AWN Navigator if you wanted those widgets enabled. Look for it in Synaptic Package Manager.
