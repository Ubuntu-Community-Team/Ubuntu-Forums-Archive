### Post by DimitrisC on 2006-08-28
I had the same issue when a theme i installed changed the colour of the location bar in firefox.  I  got help from a linux forum so here it is. I don't know if the whole theme can be customized but some theme settings can be changed by editing a file called gtkrc found in the themes directory.  E.g. for the Mist theme the settings can be found in /home/username/.themes/Mist/gtk-2.0/gtkrc  Just play around with the file and see what happens.  I hope this helps!
