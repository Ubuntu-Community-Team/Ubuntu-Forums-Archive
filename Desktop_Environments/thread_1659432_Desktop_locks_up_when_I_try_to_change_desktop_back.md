### Post by XxPrismismxX on 2011-04-20
I figured out how to fix the desktop background issue from this thread: [http://ubuntuforums.org/showthread.php?t=814329](http://ubuntuforums.org/showthread.php?t=814329) 

Changing the download location to main in software sources then going to update manager and installing the new packages did the trick.
