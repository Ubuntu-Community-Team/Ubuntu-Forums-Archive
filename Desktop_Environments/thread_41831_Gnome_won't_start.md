### Post by adrian440 on 2005-06-15
I had problems similar to this once. It turned out that when I installed a version of 'smeg' (Simple Menu Editor for Gnome), it installed updated some things. Certain packages were too new, and things became broken. The add/remove programs thing didn't work anymore, and sometimes logging in would not work.

And so when things start going pear-shaped I usually question myself on what I've installed manually recently.

Hope this helps.
