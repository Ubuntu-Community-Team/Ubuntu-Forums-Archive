### Post by kkruecke on 2012-06-28
You can use **gsettings**. See: [http://askubuntu.com/questions/85786/how-do-i-change-the-colors-of-the-ambiance-radiance-themes]("http://askubuntu.com/questions/85786/how-do-i-change-the-colors-of-the-ambiance-radiance-themes").

After setting the Clearlooks-Phenix theme using gnome-tweakt00l,
I overrode the default window border color and the background (of the interior of a window) like this:

```
gsettings set org.gnome.desktop.interface gtk-color-scheme selected_bg_color:#5598D7;selected_fg_color:#EDE9E3"
```
