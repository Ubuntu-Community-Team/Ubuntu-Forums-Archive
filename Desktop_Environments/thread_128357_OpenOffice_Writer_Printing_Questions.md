Now it is the default action (after quitting and restarting)

### Post by rufousfelix on 2006-02-11
Thanks again.  Back home and able to check out Tools>Options>Reversed Pages. That does work.  Changing this once in the OOWriter program means new, future files will print in correct order.  Existing OOWriter files, however, need to be changed on an individual basis--the program doesn't retroactively modify their settings.  That might have confused the issue for me and others. 

I have an inkjet & I understand because ink doesn't dry instantly it has to exit face-up.  Not a limitation for laserjet printers, I understand.  Hence, the need for face-up and face-down options :-).
