### Post by ajgreeny on 2010-12-25
What spec computer?  Flash on linux is not as good as it is for windows (that's Adobe's fault, not Ubuntu's) so if you have a low spec machine, I'm afraid this is quite normal.

You could try the 10.2 beta version from Adobe's website, which is better than the current 10.1 in Ubuntu.  Download the tar.gz for linux, unpack it and then replace the libflashplayer.so file from wherever it is at the moment (/usr/lib/adobe-flashplugin/libflashplayer.so?), after backing up what you have at the moment as libflashplayer.so101, so you can go back if needed.
