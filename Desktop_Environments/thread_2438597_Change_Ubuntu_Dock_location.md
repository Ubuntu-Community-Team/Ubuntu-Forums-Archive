### Post by Dennis N on 2020-03-14
> I was wondering is it possible to center the icons in the ubuntu dock on the left?
You can center the dock and icons if you install the dash-to-dock gnome-shell extension. See screenshot attached. As you open more applications, the dock and icons stay centered. The 'show applications' button can only be moved from the bottom to the top of the dock (use the configuration dialog for the extension).
