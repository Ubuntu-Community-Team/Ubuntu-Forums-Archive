### Post by mgmiller on 2006-09-03
to use unrar, navigate to the directory you want the final file to be in and give it the command unrar e /full path to the first file in the rar archive giving its entire file name and .rar suffix.

for example, if I want the episode of House to unrar to the TV_Shows directory, I open a terminal in TV_Shows and enter the unrar e command.  In this case, the rar file is in TV_Shows/House.S2.E3.PSPVIDZ 

```
marty@tux:~/TV_Shows$ unrar e /home/marty/TV_Shows/House.S2.E3.PSPVIDZ/House.S2.E3.PSPVIDZ.part01.rar
```
