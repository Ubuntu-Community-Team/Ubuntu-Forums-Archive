Hi and welcome to Ubuntu forums.

### Post by QIII on 2014-10-01
I don't recall a kernel 3.08.  3.0.8 may have been in the series for Oneiric Ocelot (11.10) which ran 3.0

3.8 was in Raring Ringtail.  Some of the later point releases of Precise Pangolin (12.04) run it (it's running on my Precise machine right now.)

As stated, 3.13 is associated with Trusty Tahr (14.04)  and 13.16 with Utopic Unicorn (14.10).

Of those, Precise and Trusty are the only currently supported versions on the desktop (the server version of Lucid is supported until April, 2015.)
