### Post by paulisdead on 2009-10-15
I've not looked into trying this myself, but the default behavior in gnome is that they share the same virtual desktop, so this makes for some really wide desktops.  So it'll change to the second desktop on both screens if you switch.  If you click the top left button in a window, there's an option to always show the app on the visible workspace, so the window will always show up in the same spot on all desktops.  I know that's not quite what you're looking for, but it might work for you.
