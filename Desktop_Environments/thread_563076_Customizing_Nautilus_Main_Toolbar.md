### Post by mgmiller on 2007-09-30
I'm not sure why you would need those buttons to appear.  I looked around a bit to see if there was a way to add them, but could not find anything.

But as a time saver, they really don't help at all.  
Let's say you wanted to move a file to trash using the buttons as you want them.  
You would first have to click on the the file to select it and then move the mouse up to the trash button and click that.  

Doing the same trash action with Nautilus the way it is, you would right click the file and this will select it and open the choices menu.  You now just slide the focus down to "Move to Trash" and you're done.  All the other choices are there as well, like copy, paste, etc. so they're all done with a single right click on the file.

Actually, by default, you can't delete a file, only move it to trash.  You would permanently delete it from there.  If you want to add a delete option, it's easy.

Open Nautilus by opening any folder,  Your home folder works fine.  
At the top, click on Edit > Preferences and go to the Behavior tab.  
You can now add check marks in the "Include delete command that bypasses trash" box and it will now always be a right click option along with move to trash.

Be careful using that option, because once a file is deleted, it's gone.

Fun, Easy, Ubuntu.  :popcorn:

### Post by mgmiller on 2007-11-14
amorangi, I have to disagree with you.

They both take 2 clicks and one mouse movement.

In windows with the delete button:
1) left click the file
2) move the mouse to the delete button
3) left click the delete button

In Gnome or with Windows default behavior:

1) right click the file
2) move the mouse to the Move to Trash item
3) left click Move to trash


Depending on where in your window and how large the window is, the Windows way with the delete button, requires moving the mouse further than the gnome way does, which always presents the Move to Trash option in exactly the same spot relative to the file you just right clicked.

In fact, the default procedure in Windows is identical to Gnome.  I never bothered to to add the delete button in any of my Windows machines.

If this option is very important to you, try running a kubuntu live CD.  kde is way more configurable than gnome is, so maybe there is a way to add the button you seek in that environment.
