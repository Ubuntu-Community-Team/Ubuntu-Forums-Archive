I run several computers with compiz. I can crash compiz on my older machines by running lots of applications at the same time. Disable compiz and try and crash the machine by doing heavy multitasking. 

Also Some of the plugins in compiz can cause problems.
I suggest you start with a minimal configuration and add features one by one until the crash occurs. That way you can isolate the problem. Then stay clear of that plug-in.
