### Post by Wolki on 2006-03-24
[QUOTE=Rackerz]Is there any way to get it on Breezy, or can it just not happen?[/QUOTE]

Not easily. You can probably install it from source using the garnome utility, see here: [http://www.gnome.org/projects/garnome/](http://www.gnome.org/projects/garnome/) , It will install into your home directory and thus not break any existing gnome installation, or at least it shouldn't. 

Not sure whether it's worth the effort, though. 2.14 is a great release, but dapper isn't that far away anymore, and the preview (supposed to be quite stable) will be here in less than a month.
