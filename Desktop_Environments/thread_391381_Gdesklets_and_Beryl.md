### Post by louis_nichols on 2007-03-23
That's been happening to mee to after the last update from the svn repos. We have to keep in mind that these versions are development versions and are quite likely to contain bugs.

If you are using these packages, there isn't much you can do, really. Filing a bug report with the beryl devs would be a good thing to do. Other than that, you can either wait for an update to the packages or you can revert to using the last stable version of beryl (1.999 I think it was), which doesn't have such issues.
