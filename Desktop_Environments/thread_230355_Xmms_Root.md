### Post by Jagot on 2006-08-05
To open Nautilus, the file manager, as root hit alt+f2 and type gksudo nautilus.  You'll then be able to drag files wherever you so desire.

### Post by SilentGreg on 2006-08-05
In the terminal:

```
sudo su
```

and then enter your password that you use to login at boot-up/login. 


That's it!
