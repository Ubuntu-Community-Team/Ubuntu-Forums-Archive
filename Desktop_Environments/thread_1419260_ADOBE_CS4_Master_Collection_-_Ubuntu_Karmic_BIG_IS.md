### Post by Vostrocity on 2010-03-01
I'm not sure if you realize this, but WINE does not enable *all* Windows apps to run under Linux. You will usually save yourself some time if you check WINE's AppDB for compatibility before you try installing yourself. For example, [here's]("http://appdb.winehq.org/objectManager.php?bIsQueue=false&bIsRejected=false&sClass=vendor&iId=12&sAction=view&sTitle=View+Developer") the AppDB page for Adobe software. As you can see, only some components of CS4 will work. In fact, Adobe software is one of the biggest reasons why people dual-boot Windows. If you have a strong computer, you can also [setup a VM]("https://help.ubuntu.com/community/VirtualBox") with VirtualBox. 

About the Wacom Bamboo, I actually have one too, but I never bothered getting it to work 100% on Linux (I don't use it much anyway). I'm not sure if you're still concerned in getting it to work if you can't get CS4 to work first.
