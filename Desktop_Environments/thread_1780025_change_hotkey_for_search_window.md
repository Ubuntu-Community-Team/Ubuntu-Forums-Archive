### Post by CryptKeeper777 on 2011-06-11
When I click the key on the left of the spacebar (cmd on my macbook, win or something on most keyboards), that search window appears which can be used to start applications. It also appears when I click on the button on the left end of the menu bar on top of the screen. 

How can I change the hotkey for this search window from cmd/win to cmd/win+space?

The problem is that the window often accidentally appears when I use other hotkeys including the cmd/win key and press the key too long, that's annoying.

I already looked though the hotkey list in the settings, but I didn't find a setting for this window...

I use Ubuntu 11.04 on a MacBook 4, BTW [SIZE=1](but didn't post in the Mac-subforum 'cause it's not a Mac-specific problem)[/SIZE].

### Post by CryptKeeper777 on 2011-06-13
Alright, problem solved, I coincidentally stumbled over it. I wanted the launcher never to disappear and the solution was to install the compizconfig-settings-manager which provides this setting. And right next to it, I found the setting for the hotkey for the search window (the real name of which I've already forgotten again...).
