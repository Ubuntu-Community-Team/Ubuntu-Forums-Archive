Yes, the only difference between Netbook and Desktop is the GUI; I don't believe they ended off changing any of the default applications either.

Performance will be the same, but bear in mind that Unity uses GL so it will naturally take away some resources from other GL-based programs such as Google Earth, 3D games etc.
