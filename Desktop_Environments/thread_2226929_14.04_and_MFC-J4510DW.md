### Post by pdc on 2014-05-29
so Robert: you downloaded the Brother linux installer tool: if you downloaded and saved it; it should be in your [COLOR="#0000FF"]Downloads[/COLOR] directory:

so if you open a terminal and copy and paste the commands 

> [COLOR="#FF0000"]cd Downloads[/COLOR]         (and hit ENTER key..)

you should be inside our Downloads directory; and if you type > [COLOR="#FF0000"]ls[/COLOR] ..............that asks the system to list what is there and you should see [COLOR="#008000"]linux-brprinter-installer-2.0.0-1.gz[/COLOR]

What Brother's aim it that you then do 

> [COLOR="#FF0000"]gunzip linux-brprinter-installer-2.0.0-1.gz[/COLOR]           and that creates a package called [COLOR="#0000FF"]linux-brprinter-installer-2.0.0-1[/COLOR]

the next command would be 

> [COLOR="#FF0000"]sudo bash linux-brprinter-installer-2.0.0-1 MFC-J4510DW[/COLOR]

and then the system should go off and get both the printer and the scanner drivers I believe
