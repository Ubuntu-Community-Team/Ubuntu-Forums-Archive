### Post by dolphin_oracle on 2007-05-04
I actually did this myself last night as well :)  To get menus, you need from the command line of your gnome session type:

$ sudo update-menus

This will set up debian standard menus in fluxbox.  NOT all your programs will show up, but enough will to get started.
