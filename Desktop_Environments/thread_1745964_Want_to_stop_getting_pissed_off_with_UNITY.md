### Post by truck87bp on 2011-05-01
TURN OFF the AUTO HIDE function for the Launcher Bar.

Go to **System Settings** (Where you normally shut down), type in **COMPIZ**, then type in **UNITY**, under _Behaviour_, Hide Launcher, set to **NEVER**, then go to _Expermental Tab_ and change **Launcher Icon Size to like 38**.

Now you will have a set-up where you won't get lost so easily. **[COLOR="Red"]This simple change fixes the whole aspect of UNITY[/COLOR]**.=D>
