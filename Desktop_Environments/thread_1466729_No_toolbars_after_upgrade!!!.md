### Post by timay91 on 2010-04-30
I fixed it!!! It takes a little doing but it works, well at least for me. here's what I did:

1: Download the latest ubunutu-tweak

2: create a desktop launcher for ubuntu tweak.
   a: right click on desktop and select create laucher
   b: title it "ubuntu tweak" or whatever you want and in the command line type "ubuntu-tweak" as the command.

3: In Ubutnu tweak under applications unistall and then reinstall avant windows navigator. and close Ubuntu tweak

4: Create another desktop launcher for AWN
   a: right click the desktop and select create launcher 
   b: title it "AWN" or anything you want and in the command line type "avant-window-navigator &"

5: Launch AWN and change customize your settings!!!

It's awesome when you figure something on your own, well kind of!!!

Hope this helps. I never got the original toolbars back but I never cared for them anyway
