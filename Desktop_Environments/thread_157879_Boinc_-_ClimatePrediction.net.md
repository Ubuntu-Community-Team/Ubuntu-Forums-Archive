### Post by tribaal on 2006-04-10
To access the boinc manager you should open a terminal window then run ```
boincmgr
```.

More information on how to use BOINC under linux can be found [here](http://wiki.debian.org/BOINC#Installation).
As you can see, this site maintains an Ubuntu version of the client, and you should really download it (you'll get a little more credit per WU since it's optimised for Ubuntu, and you don't get that weird looking font problem).

I'm running it right now and it works like a breeze.
