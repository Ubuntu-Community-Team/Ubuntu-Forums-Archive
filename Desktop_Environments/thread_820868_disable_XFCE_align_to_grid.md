### Post by denham2010 on 2008-06-08
Hi,

I have had a good look through the config settings for XFCE in ~/.config and there doesn't seen to be an option to disable the align to grid settings for the desktop items, but you can reduce their size, if that may help.

Open the XFCE settings manager and click on Desktop. In the behaviour tab, you can change the font size and icon size for the desktop icons.

Hope this helps.

Cheers.
