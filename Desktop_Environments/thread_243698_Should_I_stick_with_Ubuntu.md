I switched from Fedora's FC5 a week ago.  Ubuntu 6.06 is running on a Dell Inspiron 8100.  It made the machine sluggish to an extreme, with a few kernel related problems (or so it seems). From previous Ubuntu history, would these problems get fixed as the release matures?

:( Gripes:

.  PS/2 mouse loses synchronization often, going into erratic movements and mouse clicks.

.  Once every hour, at least, the Gnome interface freezes for 30sec before coming back to life.

.  All audio application stutter often.

.  When booted with an audio CD loaded, system crashes at gnome start-up.
