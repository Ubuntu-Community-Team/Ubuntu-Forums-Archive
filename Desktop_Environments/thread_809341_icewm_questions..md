### Post by urukrama on 2008-05-27
For Synaptic, add a menu entry with the following command:

```
gksudo synaptic
```

As for the extra monitors on the panel, that depends on what you are currently using. You can have a network and CPU monitor and an email checker. These are controlled in the [preferences file]("http://icewm.org/manual/icewm-10.html#ss10.2"), and the theme file (colours).
