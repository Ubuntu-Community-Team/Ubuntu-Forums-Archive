 I use kde, so I can't help you out on the actual icon creation.   But if you can create a shortcut to a program, modify the file to execute your favorite browsing program with the argument being the media you want to shortcut to (for example):


konqueror /mnt/win_c
konsole /mnt/win_c
xterm /mnt/win_c

or whatever.  Like I said, I don't use gnome.   Right click should let you create shortcuts, if not, check the control panel.
