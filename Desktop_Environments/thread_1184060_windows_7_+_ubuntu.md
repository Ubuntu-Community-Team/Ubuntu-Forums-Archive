### Post by PureLoneWolf on 2009-06-10
Hi

I would recommend Virtualbox ([www.virtualbox.org](www.virtualbox.org)).  Make sure you read the instructions and get the PUEL edition if you want to pass USB devices through to the guest OS (in your case Windows 7)

I haven't tried Windows 7 myself, but I use Vbox PUEL and XP to allow my Canon Scanner to function (not yet Linux supported) and it works perfectly.  I also have dual monitors and just drag the Vbox window to monitor 2 and maximise.

On the virtual machine config, you can specify an ISO image to be mounted for the machine or indeed a physical drive, so this would do what you need.

Hope that helps
