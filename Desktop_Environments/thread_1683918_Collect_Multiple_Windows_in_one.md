### Post by deconstrained on 2011-02-21
Install the package compiz-fusion-plugins-extra.

Then open up the Compiz Config settings manager and scroll to the bottom, to the section "Window Management".

The plugin to use for this functionality you require is "Group and Tab Windows".

That can do exactly what you described (I think), and more.
