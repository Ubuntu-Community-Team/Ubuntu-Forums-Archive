---

### Post by Boondoklife on 2009-06-29
Well assuming it was installed into you local themes folder and icons folder and not into the main system folder(/usr/share/themes) Then the delete button should light up when you click on the theme name.

If you saved it in the main system folder then you will have to browse there and delete the folder that contains the theme.
