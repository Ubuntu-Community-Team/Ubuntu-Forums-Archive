### Post by JF382 on 2011-11-28
Get the themes / icons you want from Gnome-look.org or another site that has themes. Another good one in my opinion would be deviantart.com

Then when you have what you need to install. Extract it so it is in folder form.

Then go to terminal and type, gksu nautilus. it will ask for a password because you need root permissions to drag icons / themes in that folder. so type in your password.

then when that folder comes up go to where you need to be

Themes go here: [FONT=Verdana]/[/FONT]usr/share/themes

Icons go here /usr/share/icons[FONT=monospace]

at the top of the root folder window. click go, then location and just put the themes / icon target place there.

then when you get to where you need to be, you just drag and drop the folder in the correct place

then you need to get Advanced Settings from Ubuntu Software Center so you can change the icons / themes from there. Enjoy.
