### Post by stozball on 2006-04-11
I'm not sure exactly how you'd go about converting it (although I'm sure it can be done). You should know however there is no "standard" avi format.

AVI is simply a container which can have videos encoded with any codec, ranging from efficient ones like DivX right up to uncompressed footage from a DV camera which weighs in at a whoping 178MB a minute.
