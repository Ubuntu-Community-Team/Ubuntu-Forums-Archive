### Post by grahammechanical on 2012-03-08
The answer is no.

In 11.10 there is a limited set of modifications that we can make to Unity but we need to install Compiz Configuration Settings Manager (CCSM) and use its Unity Plug-in to make those adjustments.

Even so, the option to move the Launcher reveal location to screen bottom or right side never worked at all.

In 12.04 there are a few adjustments to Unity that can be made through the System Settings Appearance utility but moving the reveal location is not one of them.

In fact although there is an increase in the number of modifications we can do to Unity through CCSM the option to change the reveal location is no longer available.

We also get a warning when we run CCSM that some of its options may break the desktop.

12.04 is now at feature freeze. So, it will not change from what I am at this moment using. This means that even when 12.04 is officially released at the end of April we will still need to install CCSM. And changing the reveal location will not be an option either.
