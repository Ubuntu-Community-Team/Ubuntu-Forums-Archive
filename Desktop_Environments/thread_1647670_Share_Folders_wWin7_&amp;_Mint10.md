### Post by linux18 on 2010-12-17
the 20+ year old netcat (or nc) command can do that just cat ANY_FILE | nc -vvnl 8080 if you have a static ip, anyone with your ip address can download the file by using nc YOUR_IP 8080 > FILE. A few tweaks (like password entry) and you could have a nice file sharing shell script.
