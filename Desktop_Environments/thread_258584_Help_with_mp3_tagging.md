### Post by kidders on 2006-09-16
Hi there,

Enabling write support for NTFS is not altogether safe :-( A far smarter idea is to put your music on a FAT32 partition, so you can continue sharing it with Windoze, but mess with it in Linux too.

If you don't like FAT32, there are Windows utilities available that will teach it to read HFS+ partitions :-)
