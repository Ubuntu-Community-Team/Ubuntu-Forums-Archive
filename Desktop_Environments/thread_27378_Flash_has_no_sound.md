### Post by honeywelljr on 2005-04-15
Type in a terminal 

```

sudo ln -s /usr/lib/libesd.so.0 /usr/lib/libesd.so.1

```

go [HERE](http://www.ubuntulinux.org/wiki/SoundProblemsHoary) for other sound issues that you might also have.
