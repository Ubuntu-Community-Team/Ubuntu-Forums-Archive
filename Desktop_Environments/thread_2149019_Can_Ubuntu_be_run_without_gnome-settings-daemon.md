### Post by zt14427 on 2013-05-28
gnome-settings-daemon seems like a pretty mission-critical thing.  If I were you, I would switch to XFCE (install the xubuntu-desktop package).  XFCE is a Desktop Environment, where fluxbox is just a Window Manager.  XFCE comes with some awesome features such as optional window transparency etc.  It is fast and it looks nice.
