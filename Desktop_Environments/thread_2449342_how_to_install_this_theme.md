### Post by webaake on 2020-08-25
You CAN install it on Xubuntu. Works well. Here's how you do it:

create a directory in your users home dir named exactly: .themes   
Yes (dot)themes

Unpack the theme and copy it into the dir .themes

Select the theme from within settings, might be named "Looks" or whatever. You'll find it in the menus. The theme will only be available for your user. On the other hand, in the dir /home/your_user/.themes you don't need admin rights.
