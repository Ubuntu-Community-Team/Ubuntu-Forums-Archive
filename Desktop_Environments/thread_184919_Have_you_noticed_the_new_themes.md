### Post by Ahriman on 2006-06-07
[QUOTE=tha_rainman]Yeah thats what I thought as well but it didn't fix the problem which is the white letters on the green/gray background at the top. Compare the normal nautilus window to the firefox window in the screenshots.[/QUOTE]

If you navigate to your ~/.mozilla/firefox/ directory, then into your profile directory (it will be some random letters and numbers), and from THERE to your chrome directory, you should see two files: userChrome-example.css and userContent-example.css

If you open userChrome-example.css with gedit, you will see some commented out CSS rules.

I added this section to mine, then saved the file as userChrome.css in that directory.

```
menu {
	color: white !important
}

/* Add this section for nested menu titles to show up black instead of white */

menupopup > * {
	color: black !important
}

```

Once I saved the file, I re-opened up Firefox's windows and voila! White text on the dark bar. You can change alot with this file ifyou know CSS at all.
