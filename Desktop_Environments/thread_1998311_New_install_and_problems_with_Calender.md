### Post by Rex Bouwense on 2012-06-09
The installed version may not be compatible with the installed version of Thunderbird.  Export your calandar file to a file in Documents of wherever you want to put it.  Uninstall Thunderbird and all of its config files.  Then reinstall it with the Lightning add on.  The current version of Thunderbird is 12.0.1 and Lightning is 1.04 I believe.
