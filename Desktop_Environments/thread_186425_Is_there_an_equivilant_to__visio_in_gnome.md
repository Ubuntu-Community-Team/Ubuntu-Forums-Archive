### Post by dapps on 2006-06-01
It depends what types of diagrams you are looking to create.

There is an application called &#8220;Dia&#8221; which will allow you to make various UML diagrams such as class diagrams, entity-relationship diagrams, and network diagrams.

Another app that you may want to take a look at is &#8220;Inkscape&#8221; which isn't necessarily like Visio, but allows you to a lot of things with SVG &#8211; Scalable Vector Graphics.

The last one that I remember is the most similar to Visio called &#8220;Kivio&#8221; which is an all-purpose flow-chart diagram application.

At the end it comes down to which diagrams you are looking to create as some applications exist for specific types of diagrams, whereas Visio is an all-purpose diagram application.
