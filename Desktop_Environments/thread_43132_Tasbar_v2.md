### Post by dejitarob on 2005-06-20
[Taskbar v2](http://kde-look.org/content/show.php?content=16261)  is a replacement for the original taskbar in KDE's kicker. It supports thumbnail images of application windows, transparency on (semi) transparent panels and a few other features that improve the kicker panel without losing functionality. I created a [deb package](http://kde-look.org/content/show.php?content=25615) for convenience. If you install, be sure to right click on the panel, remove the old taskbar applet, and add taskbar v2. Advanced configuration options can be access through clicking on the applet handle on the right side.
