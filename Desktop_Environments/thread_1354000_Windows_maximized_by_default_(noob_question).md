### Post by PepeLapiu on 2009-12-13
Hi all!!!
Whenever I open a window, (any window) it always opens fully maximized as default. Is there a way to change it so that when I first open a window, the default is set to unmaximized?

Or can Ubuntu remember the size of a window the last time I used it and open it at the same size?

Cheers,
PepeLapiu (Linux noob and loving it) :D

---

### Post by tars_ac on 2009-12-13
In the terminal, run gconf-editor.  If you don't have gconf-editor, you can install it in Synaptic.

Go to apps/maximus, and check the no_maximize option.  That should stop automatically maximizing all the windows.
