### Post by gettinoriginal on 2009-07-12
Alt + scroll wheel (up or down) changes the opacity of windows via compiz.  If you want to change it permanently, System > Preferences > CCSM > Accessibility, untick or change settings in opacity, brightness and saturation.  :p
