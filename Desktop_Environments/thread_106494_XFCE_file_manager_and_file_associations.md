### Post by aysiu on 2005-12-21
You can use Nautilus in XFCE. Just Alt-F2 and ```
nautilus
```

Note: it will change your wallpaper to Gnome's wallpaper.

Also, you'll notice a speed difference between XFCE and Gnome if you use under 256 MB of RAM. If you're 256 MB or above, the difference is probably negligible.
