When I change the resolution with CTRL+ALT+Plus or Minus, it works but I get like a virtual desktop which I can scroll with the mouse.

### Post by zlogic on 2005-08-25
If you need to change the resolution, open system>preferences>screen resolution.
What Ctrl+Alt+Plus does is changing the *desktop* and not the *screen* resolution.
