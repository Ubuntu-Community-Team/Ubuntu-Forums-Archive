### Post by elettronicha on 2006-09-09
From the Control Center, go to "system administration->login manager", click the "administrator mode" button and enter your admin password, then choose the "background" tab and tune kdm as you like.

### Post by TheCelloFellow on 2006-09-09
To change the actual theme of the login screen, called KDE Display Manager, KDM, you need to install a package called kdmtheme. Once you've installed it, a new item in System Settings appears called KDM Theme Manager. You can now use it to install KDM themes that you can download from KDE-Look.org.
