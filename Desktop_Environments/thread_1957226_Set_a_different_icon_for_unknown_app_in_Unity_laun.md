### Post by Frogs Hair on 2012-04-12
Open the main menu from dash and select the category where the application is placed on the left pane.  Click on the line where the application is displayed and enter properties. Select the icon picture on the left side of the box. Migrate to a folder containing an icon you want to use. Logout - in are restart may be required before the new icon is displayed.
