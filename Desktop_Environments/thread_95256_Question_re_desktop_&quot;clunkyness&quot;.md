### Post by suRoot on 2005-11-26
[QUOTE=StueyB]
Reduce the gap between the desktop icons, atm they are huge.
[/quote]

Not exactly sure what you mean here, but you may need to tweak the font settings to your liking (System -> Preferences -> Font -> Desktop font).  Perhaps you can reduce the size to allow you to move them closer.

Aside from that, there's really no more to it than clicking the icon & dragging it wherever you want.  You may want to right click the desktop & make sure 'Keep Aligned' is unchecked, as that should give you a little more control over placement.

[QUOTE=StueyB]
Also is there any way to reduce the size of the actual Icons ?
[/quote]

Right-click & select 'Stretch Icon'.  Drag the box that appears to scale the icon - larger or smaller.

[QUOTE=StueyB]
Id also like to install the OS X theme from Gnome-look.org, but every time i try to install it as a theme after i untar it etc it gives me a an error " The file format is invalid" Is there any way to fix this ?
[/QUOTE]

Download the archive & extract it to ~/.themes.  You can then choose it using System -> Preferences -> Theme -> Theme Details

### Post by sizzam on 2006-02-11
To make the desktop icons smaller by default,  open up Nautilus, then:

Edit -> Preferences -> View Tab

"Icon View Defaults" - you can pick a smaller zoom level here.  75% looks pretty good on my desktop.  

After you change this, you may have to right-click on your desktop and choose 'cleanup by name' to have it fix the spacing for the icons that were already on your desktop.
