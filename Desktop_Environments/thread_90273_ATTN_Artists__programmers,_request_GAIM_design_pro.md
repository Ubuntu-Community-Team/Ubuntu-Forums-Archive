### Post by muted on 2005-12-06
I have just "apt-got" devilspie... pretty cool program...!

As far as i can see... an decorate command to gaims buddylist would in the .devilspie.xml look like


```
<?xml version="1.0"?>
<!DOCTYPE devilspie SYSTEM "devilspie.dtd">
<devilspie>

<flurb>

<matchers>
<matcher name="DevilsPieMatcherWindowName">
<property name="application_name" value="gaim"/>
<property name="window_role" value="buddy"/>
</matcher>
</matchers>

<actions>
<action name="DevilsPieActionDecorate">
<property name="decorated" value="FALSE"/>
</action>
</actions>

</flurb>
</devilspie>
```

Got it working, thaks, its so close ... :) yet, not perfect...
