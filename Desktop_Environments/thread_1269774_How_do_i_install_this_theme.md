### Post by Soley on 2009-09-19
This is just a copy & paste from translate.google.com from the readme file in the archive.

> 	
1 - Emerald:
Open the folder Emerald. On the inside is a folder named "UbuntuSavana. Place this file in the directory / home / [yourusername] / .emerald / themes.
Then open the configuration manager and select the Emerald theme "UbuntuSavana.

2 - walltime:
Open the folder walltime. On the inside is a folder named "Savana". Place this file in the directory / usr / share / backgrounds. At the moment did not change the background of your desktop, this will happen only in point No. 4.

3 - Icon:
Make Menu ---> system ---> preferences ---> appearance to open the Appearance Manager.
Make a Drag & Drop (drag) from the archive "Meliae-Dust.tar.gz" your window to the preference of appearance.

4 - Gtk Theme:
Make Menu ---> system ---> preferences ---> appearance to open the Appearance Manager.
Make a Drag & Drop (drag) from the archive "gtk-UbuntuSavana.tar.gz" your window to the preference of appearance.
Following message -> "the theme gnome has been successfully installed"
Select the theme Ubuntu Savana, everything should fall into place, and the screen icons and dynamic if you have placed.

5 - Theme Savana Cubemodel:
Open Cubethemer and download the theme Savana. Activate. That's all.

6 - Usplash:
WARNING, it only works for Intrepid Ibex!
Double-click the archive savanna-intrepid.deb, installation is automatic.

7 - Gdm Theme:
Open menu ---> System ---> Administration ---> Login Window
Enter your password.
Tab "Add" ----> point to the archive GdmUbuntu Savana.tar.gz. The installation is automatic. Check the appropriate button to activate it.

For more information or to manually install each part (including the walltime): [http://doc.ubuntu-fr.org/personnalisation_/cubemodel_de_compiz_et_wallpaper_dynamique](http://doc.ubuntu-fr.org/personnalisation_/cubemodel_de_compiz_et_wallpaper_dynamique)

If you have problems post here: [http://forum.ubuntu-fr.org/viewtopic.php?id=255827&p=1](http://forum.ubuntu-fr.org/viewtopic.php?id=255827&p=1)
