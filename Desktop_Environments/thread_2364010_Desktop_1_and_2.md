I am also using Lubuntu 16.04 and regularly have different programs on each desktop, each with a different wallpaper image. The easiest way to switch between them is to have the mouse pointer on the wallpaper and roll the mouse scroll wheel. I only use the taskbar icons if either desktop has a full-screen program on it.
