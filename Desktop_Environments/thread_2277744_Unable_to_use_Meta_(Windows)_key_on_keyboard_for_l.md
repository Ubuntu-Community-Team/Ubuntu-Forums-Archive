I do not think this is possible.  If you did that then it would override all other key combinations.  If you have Meta-T open Thunderbird then it would no longer work as Met would open the launcher.  What we do is make Meta-Z or Meta-X be the one we want the most.  You can mash them pretty quick.

### Post by anj-tuesday on 2015-05-15
Using the MetaWindowsSuper key/keys by itself/themselves is possible with a tiny additional tool called ksuperkey.

Github: [https://github.com/hanschen/ksuperkey](https://github.com/hanschen/ksuperkey)
PPA: [https://launchpad.net/~mehanik/+archive/ubuntu/ksuperkey](https://launchpad.net/~mehanik/+archive/ubuntu/ksuperkey)
