### Post by FuturePilot on 2007-05-22
They're all pretty much the same except with updated software and some new features here and there. The exception being Dapper. Dapper is a Long Term Support (LTS) edition and is supported for 3 years on the desktop and 5 on a server. Whereas the other ones, Edgy, Feisty, Gutsy, are only supported for 18 months. Since those releases are on a 6 month release cycle they tend to be a bit more buggy than Dapper. If you're after stability I'd say go with Dapper.

Right now Gutsy is in very early testing, so it's not recommended to use it as a main OS.

### Post by dbott67 on 2007-05-22
If memory serves me, Ubuntu's release schedule coincides with the release of Gnome (which is also every 6 months).  This allows developers to integrate the latest stable packages into each release.

Confirmation of above from [linux.com]("http://tips.linux.com/article.pl?sid=07/04/12/1250208"):
[quote=linux.com]Scott James Remnant, the Ubuntu development manager,  also [posted]("https://lists.ubuntu.com/archives/ubuntu-devel-announce/2007-April/000277.html") a tentative development schedule for Gutsy, with six milestone releases, a beta release after the next GNOME release, and a final release three weeks after the beta.[/quote]

As an end-user, this gives you access to the most up-to-date packages (although there may be a stability sacrifice if you always want to run the latest versions).

More general info here:
[http://www.ubuntu.com/products/whatIsubuntu/releases](http://www.ubuntu.com/products/whatIsubuntu/releases)
