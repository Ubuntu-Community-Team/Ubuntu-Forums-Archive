With Hoary Hedgehog it was possible to circle through all running progs, even running on a different virtual desktop, using key combination alt-tab (also known as program task switch shortcut). This seemes to be gone since Breezy. It allows only switching between the programs running on only one virtual desktop.

Is it possible to revert this behaviour to work as in the past?

### Post by lptr on 2006-11-15
Answering myself - because I found the answer:

In Systemsettings, Personal, Workplace, Windows settings, tab activation there is a control area named Navigation. Activate all checkmarks, there. This will bring back old behaviour. I don't know why they (Kubuntu team) disabled especially that point no. 2.
