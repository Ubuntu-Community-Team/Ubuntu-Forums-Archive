### Post by Jasonx86 on 2009-08-25
Yes, Conky works brilliantly. Thanks.

I installed it using **sudo apt-get install conky**, replaced the **/etc/conky/conky.conf** with a configuration found [here]("http://wiki.conky.be/index.php/ExampleConfigurations"), and then made conky startup automatically (using System > Preferences > Startup Applications from the menu).
