### Post by kevinlyfellow on 2007-07-13
> **ruscoe said:**
> I tried it in vmware last year and found it really awkward to use.

One of the main criticisms of the OLPC project is how little the Sugar interface prepares users for the desktop interfaces used by everyone else.

That is my only real complaint about the project.  I don't see a good reason to use something that far from a standard interface.  I accept the idea that these laptops are meant for a different purpose than most users (actually, I demand it!), but there is no reason not to teach kids using a modern style interface.  Kids are not dumb, and will be able to figure out any interface you throw at them.  I wouldn't even mind the weird interface so much except that it doesn't allow multi-tasking (please tell me I'm wrong), which is a good computer skill to have.  I don't believe that an interface matters when teaching, just as long as the same skill can be applied to other interfaces.  I'm not convinced that sugar does that.
