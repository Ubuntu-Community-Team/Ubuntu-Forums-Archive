### Post by mtron on 2005-04-05
an easy method is:
open Synaptic Package manager, go to File - Save Markings As
give it a name, like programs04052005 and check save full state.

This will generate a text file with all the progs & libareis currently installed on your system and it is very handy when you want to do a re-install
