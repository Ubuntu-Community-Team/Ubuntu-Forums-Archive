### Post by PalapaGuy on 2009-09-25
I can't find a font selection that displays Web pages on Firefox / Ubuntu 9.04 exactly the same way as Forefox does running under Windows.

In particular, in my Yahoo Webmail home page the fill-in box entries aren't centered vertically, so the bottom halves of the characters I enter in the boxes are cut off and hard to read.

How can I fix this?

---

### Post by mcduck on 2009-09-25
First, make sure you have installed the same fonts (msttcorefonts package). Then just set the font sizes to same as you have in Windows (Content->Fonts & Colors in Firefox preferences) click the Advanced-button for detailed font settings).

Of course you might also want to send feedback to the site's admins that their layout doesn't scale properly to different font sizes.. ;)
