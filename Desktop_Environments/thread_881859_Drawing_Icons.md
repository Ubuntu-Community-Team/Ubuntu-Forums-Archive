### Post by swisscow on 2008-08-07
openbox is a window manager which doesn't by itself have the ability to put icons on the desktop. From the community minimal install pages, if you install idesk (sudo aptitude install idesk) this is what many distros do to draw icons on the desktop.

I haven't used it or tried to configure it but there is a very limited wiki which may help: [http://idesk.sourceforge.net/](http://idesk.sourceforge.net/)
