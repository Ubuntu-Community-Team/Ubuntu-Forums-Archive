Hi, I've been using ubuntu from May of this year; I've been fixing errors around using google, this is my first post in this forum ;D. Anyway:

I've noticed that font antialias for small text doesn't work quite well, and since gnome "Font" settings aren't the best (can't exclude size ranges), I downloaded Kcontrol and used the following config:

-Full Anti Alias, No Subpixel hinting (using a LCD).
-Exclude 6.0-11.0

[IMG]http://img.photobucket.com/albums/v517/CanXp/uglybold.jpg[/IMG]

This works great, but there is a minor problem: Bold and italic fonts within the 6.0-11.0 range look like crap (just look at the "Home Folder" in bold!). Is there a workaround for this? A config file to edit?

Also, anyone having problems with fonts, I recommend this configuration; small text looks sharp, and big texts smooth.
