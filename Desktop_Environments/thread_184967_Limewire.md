### Post by Perfect Storm on 2006-05-30
Grab the .rpm version of Limewire and save it to your Desktop.

Open the terminal:
```
cd Desktop
sudo apt-get update && sudo apt-get upgrade
sudo apt-get install alien
sudo alien -i XXXXXXXXXX.rpm
```


XXXXX = the name of the package.
