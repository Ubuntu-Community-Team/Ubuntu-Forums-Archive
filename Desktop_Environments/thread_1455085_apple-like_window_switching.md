you can move the mouse to a specific corner of the screen where then the windows cascade and you simply click on the window you want focused.

### Post by mcduck on 2010-04-15
Answer to both Windows, and workspaces, is yes.

For windows, use the "Scale" plugin in Compiz.

..and for desktops, the "Expo" plugin is what you are looking for.

Both allow you to trigger the action using a screen corner. Install the CompizConfig Settings Manager (if you haven't got it already) to enable these plugins and adjust their settings.
