You actually don't even have to compile it.  Download the [latest binary](http://www.xiaprojects.com/www/downloads/files/kxdocker/kxdocker-0.32-bin.tar.bz2).  Extract the tarball and cd to the directory that you extracted to.  Then simply run

```

sh kxdocker.sh
```

to start the program.  If you get an error about not having the libpng3 library, just open up Synaptic and install it.  Then try again and it should work fine.
