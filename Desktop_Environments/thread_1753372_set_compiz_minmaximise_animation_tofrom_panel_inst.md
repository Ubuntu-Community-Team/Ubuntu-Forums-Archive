### Post by Copper Bezel on 2011-05-08
Do you have a task list on the panel itself, then, as well? If you have two task switchers on the same desktop (which is, in itself, a little unusual,) there's really no way to define which one you want the windows to minimize to. 

You *can* turn off the "Zoom to taskbar on minimize" behavior for most of the animations that support it, within the Effect Settings tab of the CompizConfig Settings Manager, but I only know of one animation that can support an arbitrary direction (Skewer, which comes in the Animations Add-On plugin of the Extras package.) Personally, I use it for minimizing - normally it breaks the window into little parts, but it can be reconfigured to move the window in a smooth glide in the direction of the taskbar.
