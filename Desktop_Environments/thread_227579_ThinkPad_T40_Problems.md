### Post by flof on 2006-09-06
Hi cdsboy,
hope my answer doesn't come too late...

This sounds very much like a very fine crack in the motherboard which leads to an interrupted circuit when the TP is bent. This is a common problem with the T40 and often, but not necessarily comes with a crack in front of the trackpad buttons. If you still have warranty on the TP, IBM will fix it by replacing the motherboard.

Only other option would be a damaged screen cable, you can rule this out by pressing the cable between the base and the screen: If the screen goues out, it's the cable, else I'd go for the motherboard...

HTH,
flo.
