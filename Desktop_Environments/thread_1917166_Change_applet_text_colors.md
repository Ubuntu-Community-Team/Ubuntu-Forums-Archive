### Post by Frogs Hair on 2012-01-29
Yes you can , but the gtkrc layout may differ from theme to theme . You would really just have top open the file and display the gtkrc files in the text editor . When you open the files the option to run or display will be offered . It's possible to make and save changes to the files from the text editor .
