### Post by teaker1s on 2005-11-12
well put it this way I had a hitachi deskstar drive with xp and it died 'search deathstar ' and ibm and you'll see it was game over.
My reason for not reinstalling xp was I don't like the upcoming ms vista and DRM-for the first week I couldn't decide if I'd done the right thing. Then I realised
1)the support for linux/ubuntu is better and more helpful than MS
2) the system can be customised as you like virtually no limits
3)if you kill part of the system it can be repaired because it's more modular than xp and if you damage say gnome you can still boot the computer and fix it with internet access and a full os
most software is free and if there is a bug then developers will accept and try to fix-when you pay for software there is profit as the bottom line. Linux developer seem to accept suggestions and want to make their programs as reliable as possible.
