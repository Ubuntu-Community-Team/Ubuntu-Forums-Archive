### Post by andrea000 on 2009-11-27
> **BALJEET OBEROI said:**
> Themes, icon looks, window transition effects, menu bar

Ok on themes you will have to go download one i go [here]("http://www.gnome-look.org/?PHPSESSID=5194a8da2c6ee479d0a4d3dbdb5055f1")
after that right click on the desktop and click change
desktop background then click the theme tab and just drag
the theme you downloaded over in there and let it go.Then
make sure that theme is selected.

you can install cairo dock or awn for a mac like dock.

effects you need to go and install compizconfig setting
manager system->administration->synaptic package manager
then check compizconfig setting manager and click apply
to install it and then go and set up the effects.There is
a lot to choose from cube desktop,burning windows etc.just
do a google search to see what effects you would like

You should be able to find a icon pack the same place you
downloaded your theme from
