**I'm new to linux myself and I'm surprised how much I've learned and I want to share that with others, this is my first tutorial so go easy on me.**

Okay, I'm gonna show you how to make Amarok and other all other KDE apps on your system  look way better and fit in with your GNOME theme.

Here's the before:
[[img]http://xs321.xs.to/xs321/07482/6oavkaq.png.xs.jpg[/img]](http://xs.to/xs.php?h=xs321&d=07482&f=6oavkaq.png)
And here's the after:
[[img]http://xs321.xs.to/xs321/07482/81fnls9.png.xs.jpg[/img]](http://xs.to/xs.php?h=xs321&d=07482&f=81fnls9.png)

On my system I have Buuf icons and a dark GNOME theme, so I downloaded a Buuf KDE Icon theme and a dark color theme from [KDE-Look.org]("http://kde-look.org").

Fist we need to install kcontrol.

```
sudo apt-get install kcontrol
```

Now lets's start kcontrol.

```
kcontrol
```

Now you can go into the appearance section and change settings and install new themes. It's pretty self explanatory, just go to [KDE-Look.org]("http://kde-look.org") for a large selection of themes and icons.

If anyone needs help with something, just reply and I'll try and help you out.
