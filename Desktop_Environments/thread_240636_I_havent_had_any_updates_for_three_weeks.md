### Post by crackseller on 2006-08-21
Well, first of, the font update only happened if you have the Compiz/Xgl Repo (beerorkid.org) activated. To see if you actually missed an update, and to get a clue if your update manager is working or not you could just try a dist-upgrade via console:

```
sudo apt-get update
sudo apt-get dist-upgrade
```
If the last command leads to an (or several) updates, then your auto update does not seem to be doing its job. However, I can't tell you how to resolve that as I don't use that GUI update thingy, sorry.
