### Post by robert shearer on 2010-06-02
I seem to recall that ipv6 is disabled by default.

Try this, right click on the network manager applet and select 'edit connections'.

In the window that opens select your active connection and 'edit'

This should open another window that has a tab for ipv6.

You could try changing the default 'ignore' to 'automatic'

I do not know if this will work or what other effects may arise but it seems easy enough to reverse. :)
