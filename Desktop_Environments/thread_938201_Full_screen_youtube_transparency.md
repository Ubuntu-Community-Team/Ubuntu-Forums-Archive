### Post by Shunt on 2009-08-16
Late answer, but ...

This works for me... start CompizConfig, General Options and the Opacity Settings tab. In Window Opacities, edit the value for Dropdownmenu and set opacity to 100.

For some reason fullscreen flash & the dropdown menu seems to share opacity. No idea why... but I can live with solid menus.
