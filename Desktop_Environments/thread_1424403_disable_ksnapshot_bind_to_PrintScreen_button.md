### Post by jynyl on 2010-03-07
Kubuntu Jaunty is having trouble with the ksnapshot infinite loop bug ([http://bugs.kde.org/show_bug.cgi?id=166190]("http://bugs.kde.org/show_bug.cgi?id=166190")).  This is where one press of the Printscreen key causes ksnapshot to repeatedly start instances of itself, soon using up all system resources.  This trouble appears to have started since a recent system update.

How can I remove the binding that links the PrintScreen key to ksnapshot?
I'd prefer to not have to remove ksnapshot, as it is still useful from kmenu.

TIA

---

### Post by inobe on 2010-03-08
system settings > input actions > preset actions > un-check printscreen.
