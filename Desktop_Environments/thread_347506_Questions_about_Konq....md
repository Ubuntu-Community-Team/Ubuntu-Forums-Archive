I can't answer your first question about scrolling, maybe you can adjust the preferences to smooth/auto-scrolling?

Well, Konqueror isn't just a web-browser. It's the Emacs of web-browsers. Konqueror is also a file manager, FTP client, and probably more.

Kubuntu, well KDE, uses the Qt graphics library, rather than the GTK graphics library, which is what GNOME, and Ubuntu, use. Firefox is an application built with the GTK library, and while running any GTK app in a Qt environment, it will probably behave slow or maybe uglier. Or vice versa.
