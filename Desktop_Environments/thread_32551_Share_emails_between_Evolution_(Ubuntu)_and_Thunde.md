I read all my emails in Windows XP only.

What I would like to do is to be able to read emails in Ubuntu while having the same emails found in Windows XP. I guess this barrier still makes me rely on Windows.

### Post by enquiry on 2005-05-08
The easiest way to accomplish this is by setting Evolution and Thunderbird not to delete the email from the server once downloaded. This way your email will be downloaded to both mail clients.

You have two options:
[list][*]With POP protocol: Set your clients to not delete mails from server after downloading messages.
[*]With IMAP protocol: Best for your needs, but not every servers support it. The mail management is done by the server, not the client. This means you can create folders and put mails into, mark mail read/unread and it's set on the server: when you open an another client you'll have the same environement.[/list]
