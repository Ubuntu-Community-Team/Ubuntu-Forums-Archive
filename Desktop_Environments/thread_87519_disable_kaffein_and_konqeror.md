### Post by shinygerbil on 2005-11-08
You can either right-click on the file and use the Open With menu, or go to Settings-> Configure Konqueror, click on File Associations, click on the type you want (eg. image) and click the "separate viewer" radio box. You can also configure individual file types/programs from there :)

Steve
