### Post by Frogs Hair on 2012-02-04
Hello and Welcome 

First , enable the command prompt  . Open keyboard > shortcuts > system and choose Alt + F2 for the command next to " Show the run command prompt." Run the following in the command prompt  to restart the shell in case of a possible glitch . ```
r
```

Next , I would open additional drivers to see if there is a driver for your graphics card and install the recommended one .
