### Post by urukrama on 2007-11-12
> **elect said:**
> With a ubuntu base system on a k6 350, which DE &WM do you suggest?

Remember I need the bast DE&Wm not only from the point of view of lightweight, but of assistance of community too...

I thought to Enlightenment with iceWM o JWM.....any ideas?

These things are largely a matter of preference and personal taste. If you want plenty of support from the community here, I'd suggest Openbox or Fluxbox. I prefer Openbox, and use it daily. With its latest release (3.4) it has really matured and become a very pleasant, easy, customizable window manager. There are quite a number of Openbox users here, who are all very willing to offer help and assistence should you need it. 

I've never been able to get used to Fluxbox, though it has a large fan base too. IceWM is nice too, and I use it occasionally. It has less Ubuntu users, though (or so it seems). Enlightenment is a bit fancier -- which may or may not be a good thing in your dictionary. JWM is not as popular as any of the others, though KMandla and fuscia have been playing with it.

You could try all of them out and see which one you like best. If you already have an existing Ubuntu installation, install them and play around with them. You generally only realize after using the WM whether you like it or not (I initially did not see the point of Openbox, but now love it).
