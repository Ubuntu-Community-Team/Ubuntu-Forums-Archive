### Post by MountainX on 2009-10-31
SOLVED:
Here are the steps:
1. click the "cashew" in the upper right corner of the screen.
2. click "zoom out"
3. click "Configure Plasma"
4. check "different activity per desktop"
5. click the zoom icon on one desktop to return to normal mode.
6. set your wallpaper on each desktop. They can be different.

Kubuntu needs a better way to access the "Configure Plasma" dialog. If there is a more direct way, I have not found it. 

Caveat: do not check the option for "use a separate dashboard". It introduces a bug and the different activities per desktop will not work well (there will be a 5th orphan desktop if you have 4 now).

Also, on my laptop Kubuntu shows a bug that makes it run super slow while doing this configuration. Once the options are set and you zoom back to one desktop the performance goes back to normal.
