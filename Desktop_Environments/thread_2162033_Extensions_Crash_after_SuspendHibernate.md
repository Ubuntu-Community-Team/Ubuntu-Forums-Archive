Hi,

I have recently installed Ubuntu Gnome 13.04 with gnome 3.6 on my laptop. After the computer returns from suspend or hibernate, some (but not all) of my extensions stop working. In the tweak-tool some are grayed out and have the little orange error triangle next to them.  Others have no error message and are set to ON, but don't work.  Others are reset to OFF. If I hit alt+f2-r, when gnome resets, everything returns to normal.  Any ideas why this might be happening? Let me know any further information you may need.  

**These extensions show up greyed out with an orange triangle next to them:**
Dash to Dock
Removable drive menu
hide top bar
alternative status menu
setting center
media player indicator

**These extensions have no error message and are set to ON but aren't working**
User Themes
Force Quit

**And this extension is reset to OFF**
Windowoverlay icons

Thanks

**Edit: **I firgured out the issue was Icon Hider causing the crash. Without installed everything works fine.
