I have just received a Dell laptop with Ubuntu 12.04 preinstalled, and I'd rather not use Unity. So I installed gnome-session-fallback, but if I select "Gnome classic", I get a Unity desktop. But at least I know the desktop effects work.

OTTH, "Gnome classic - no effects" does start the good old gnome desktop, but without any desktop effect.

How could I enable desktop effects under the Gnome classic desktop? TIA for ideas.

### Post by grumblebum2 on 2014-04-10
When logged in to the Gnome classic session use **c**ompiz**c**onfig-**s**ettings-**m**anager
to disable the unity plugin.
May need to install ccsm.

---

### Post by daniel-clement on 2014-04-10
Yes, that was it. I couldn't find this setting because it was in CCSM, which was not installed. Thanks a lot Grumblebum2!
