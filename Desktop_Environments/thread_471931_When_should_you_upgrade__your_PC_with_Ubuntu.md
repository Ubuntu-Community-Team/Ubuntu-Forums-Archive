### Post by RedSquirrel on 2007-06-12
There's no need to upgrade unless you find the applications you need to use are sluggish. Even then, rather than upgrading the hardware, you could make some software changes such as using lighter applications and/or a lighter environment / window manager (Xfce/Fluxbox/IceWM etc.).

Beryl is one example I can think of that requires a *somewhat* newer machine.
