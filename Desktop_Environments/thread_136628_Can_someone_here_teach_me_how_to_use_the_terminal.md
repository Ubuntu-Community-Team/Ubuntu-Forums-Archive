### Post by aggiechemist on 2006-02-26
I have mostly learned by doing, but I have two big hints. 

1. Apropos. This is a command to help you find commands. For instance, a while back I was trying to figure out how to do something with novell. So part of my process was going to the termnial and typing 

apropos novell 

and

apropos netware

and anything else I could think of. It gives me a list and brief description of every command that is related. It is a huge help, and a good starting place.

2. Man. Every command, and many files, have a man page (man stands for manual). For example, I eventually decided that the command mountncp was what I needed to get my novell thing to work. But I had no idea how to use the command. So I typed in

man mountncp

and got a huge text file telling me everything I could want to know about it. Everntaully, with a lot of work, I learned what I needed and got the thing working. Want to know more about manuals? Try this:

man man

A manual for manuals!!

The last hint, of course, is the forums. I wouldn't be anyhting without lots of help from the forums. 

Good luck, and have fun with it. If you ever get stuck, check the forums. There is almost nothing I haven't been able to solve with this system.
