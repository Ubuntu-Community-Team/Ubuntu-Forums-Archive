### Post by khughitt on 2007-10-28
Heya,

Freemono is the main fixed width font for Ubuntu I believe. I like to use windows fonts myself since they look a little sharper. You can get some of the standard win fonts (included the fixed width fonts courier and courier new) by doing:

**sudo aptitude install cabextract msttcorefonts**

I would also try playing around with the way the fonts are displayed in the "system" -> "preferences" -> "appearance" settings.

Goodluck!
