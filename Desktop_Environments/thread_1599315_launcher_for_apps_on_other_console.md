### Post by kbryk on 2010-10-17
Not 100% sure of your question, but you can use the same console to launch multiple applications by using the ampersand character.

```

user1@user1-computer:~$ firefox&
[1] 3122

user1@user1-computer:~$ virtualbox&
[2] 3139
[1]   Done                    firefox

```
