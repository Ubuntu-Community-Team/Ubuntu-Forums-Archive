### Post by bingoUV on 2008-04-22
What do you want the software to do? Is it to run wireless LAN ? Wireless bluetooth? I will assume that it is to use some hardware.

When you insert CD into optical drive, what happens? Does the file manager open with contents of CD? If yes, check what are the contents of the CD. Maybe there is a readme text file. Read it and see if linux is supported.

There could be system requirements mentioned on the CD, which may or may not include linux. If it lists different versions of microsoft windows, (maybe apple) and not linux, throw the CD away in the smelliest garbage can in the town and search this forum for how to make your hardware work.


Post here what kind of hardware is it? PCI card? USB device? PCMCIA? Integrated with motherboard/laptop?
