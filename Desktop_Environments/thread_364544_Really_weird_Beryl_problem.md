### Post by Waappu on 2007-02-20
Hi

Try if this kind startup script helps:

type in terminal:```
kdesu kwrite /usr/local/bin/start-beryl.sh
```
type these lines to file and save
```
#!/bin/bash
killall beryl
killall emerald
beryl-manager
```
then add execute rights to file```
sudo chmod a+x /usr/local/bin/start-beryl.sh
```
and try to start beryl with that script```
start-beryl.sh
```if it works ok then you can add it to startup programs.
if not just remove script```
sudo rm /usr/local/bin/start-beryl.sh
```
