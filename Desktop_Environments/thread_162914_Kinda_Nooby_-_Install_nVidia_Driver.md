### Post by wylfing on 2006-04-19
You should use Ubuntu's pre-packaged nVidia driver instead of the one from the nVidia web site. It'll save you a lot of headaches. Do this:

Step 1: enable the universe and multiverse respositories. Instructions here:
[https://wiki.ubuntu.com/AddingRepositoriesHowto](https://wiki.ubuntu.com/AddingRepositoriesHowto)

Step 2: still in Synaptic, click the Find button and search for "nvidia" (no quotes).

Step 3: right-click the package called nvidia-glx and choose Mark for Installation.

Step 4: click Apply.

Step 5: reboot and enjoy.
