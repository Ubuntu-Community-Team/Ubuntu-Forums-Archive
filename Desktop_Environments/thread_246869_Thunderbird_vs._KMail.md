### Post by editrix on 2006-08-30
IMO, you're talking about the 2 best email programs around, and there's not much difference between them, feature-wise. KMail is lighter than TBird, which makes a difference on my 256MB system. But it doesn't integrate well with FireFox. For example, if I click on a URL in an email, the bouncing icon shows up even though FF is already open, and it takes so long to load it's actually quicker to copy/paste the URL. 

One thing I like about KMail is that HTML mail comes in as text, but a mouse click turns it into HTML if I think it's safe. I can't recall if TBird does this.

It's really, really easy to convert mail folders from MBox to MailDir in KMail. Again, can't remember about TBird.
