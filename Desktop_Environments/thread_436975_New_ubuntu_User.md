### Post by LaRoza on 2007-05-08
What are you going to be using this compute for?

I run 7.04 and 6.06 on a computer with 512 MB of RAM and they work fine, but I do not do anything that would require more.

If you intend on using fancy graphics, like Beryl, and you have the money, get 1 GB of RAM and a good video card, not ATI though.

 I do not know what you intend, but getting the best processor you can is always good.

### Post by snoop on 2007-05-08
It would depend on the desktop environment and also how much you want to spend.

if you are using gnome, 512 will do fine, but as mentioned by LaRoza, if using beryl, 1 gb + nvdia card.

### Post by JawsThemeSwimming428 on 2007-05-08
You can run Ubuntu on almost any setup. I did the same thing back when Dapper first came out. My machine was running on a Pentium 4 2.0GHz 40GB HDD Intel D101GGCL Motherboard and 512MB of RAM. I recently upgraded my system to a nVidia graphics card as well as added another 512MB of RAM (mainly because I play a lot of video and games). I would start out somewhat barebones like I did and then add what you need. Ubuntu will run just fine on a minimal hardware setup.
