### Post by vanadium on 2021-06-29
It is your choice. Gnome Shell extensions are external software, not checked by the Ubuntu developers. The risks that may exist, are yours. These risks slightly decrease if you limit yourself to Gnome Shell extensions that are distrubuted in the Ubuntu software repositories (main and universe). See the available ones in Synaptic Package manager or with the command [FONT=Courier New]apt search gnome-shell-extension-[/FONT].

### Post by monkeybrain20122 on 2021-06-29
I don't know what you mean by 'essential'. Others may disagree but IMO gnome is not usable without extensions (and no extension can fix the sluggishness), but many extensions are poorly maintained because gnome doesn't care about api consistency and may in fact see these extension as a rude interference with its "design vision", so every new gnome release breaks a ton of extensions that many people use (as of now dash to dock does not work with gnome shell 4 but chances are it will be updated). The internet is littered with unmaintained and broken gnome extensions.


So if I have to use GS I would keep a few core extensions with a good chance to be updated (e.g dash to dock) and that's it. Meanwhile I am a happy unity user in 20.04.
