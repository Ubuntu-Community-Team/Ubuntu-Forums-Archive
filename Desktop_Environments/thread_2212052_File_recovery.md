### Post by TheFu on 2014-03-19
Welcome to the forums.

Linux doesn't have a trash bin, though some very few GUI programs might use a recycle-bin. NEVER count on this.

Restore those files from a backup will be the easiest way to get them back.

If you have lots of time, try the **testdisk** package. It can scrub a HDD for old files - you'll have to name every one of them. there are probably hundreds of thousands of files there 99.9999999% are worthless to you. I hope it will find the 1 file you want.  Oh --- and stop using the HDD completely. Every time the drive is written, you risk losing more of the file you want.

Backups are much easier.
