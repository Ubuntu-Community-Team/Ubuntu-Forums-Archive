### Post by alec on 2006-10-29
I know that you can open a file in your chosen application by right clicking on it (in Nautilus) and choosing the application, but this seems long-winded to me.

I use Notecase quite a lot and want to just left-click on one of its files and have Notecase open. However Nautilus sets Firefox as the default application, so I have to go through the right click path to get what I want.

Is there a way of setting my own default applications?

Cheers Alec

---

### Post by sam81 on 2006-10-29
Try right clicking on the file, then go to Properies > Open With and set Notecase as the default app to open the file, or add it if it's not listed
