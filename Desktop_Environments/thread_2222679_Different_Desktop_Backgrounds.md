### Post by deadflowr on 2014-05-07
I don't know if it still works, but in ccsm, if you have the compiz-plugins package installed, there is an option called, Wallpaper( it should be in the utility section)
Enable it and open. Simply click on new to add the wallpaper.
You do this until all the workspaces are done. 
(So simply keep clicking add.)
It lists them workspace 1 and up.
So the first will be the first workspace, and the second will be the second workspace, and so on.

At least that is how it is suppose to work.

I haven't do it since quantal, or raring.
So I don't know whether it's broke or not.
And I have no need to try.
