### Post by bistros on 2007-12-13
your xorg.conf file is confusing what you want for screens.  The screens are defined as numbers in one area and by name in another - the desktop layout section is referring to both.  Eliminate the screen 0, screen 1 in the device sections and make sure the desktop layout section refers to the screens by name.

That should fix things.

Hey there,

Try the following GUI.
```
gksudo nvidia-settings
```

Then you can go into XServer Display Configuration and play with the specific Color Depths etc to see if any of these speed up your display.  Also, you may try the detect displays feature.  In the end, this should actually "Hide" your displays from X and allow the nVidia driver to do the dirty work for you.  X will see it as one wide screen.
