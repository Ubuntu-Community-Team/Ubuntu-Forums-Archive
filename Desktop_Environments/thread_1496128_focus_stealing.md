I run netbook remix, and need to know if there is any way to make focus stealing more effective. I end up typing on things and hitting enter on buttons that I dont mean to. It becomes a security concern when I accidentally type my password in cleartext into some random window that pops up. It is especially bad with update manager, which pops up without being asked for and then does it about 3 times while updating. If anyone can fix this, me and many other people would be grateful.

THANKS.

### Post by kerry_s on 2010-05-28
the update notifier window can be turned off in gconf-editor, you'll still get the icon to tell you, but it won't open automatically.

for focus try changing it to "strict" instead of "smart" in gconf-editor
