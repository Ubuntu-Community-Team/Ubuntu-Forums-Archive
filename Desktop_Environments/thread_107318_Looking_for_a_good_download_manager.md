### Post by JAwuku on 2005-12-22
Hi there,

There are two download managers in Ubuntu I can recall.

One is Downloader4X (d4x), which works in Gnome. To install it, type

sudo apt-get install d4x

The other is part of KDE, called KGet. Type

sudo apt-get install kget

Hope this helps.

### Post by anil_robo on 2005-12-23
Free download managers for Ubuntu:
1. Downloader for X (*sudo apt-get install d4x*)
2. Gwget download manager (*sudo apt-get install gwget*)
3. Kget (*sudo apt-get install kget*)

Firefox and Opera for Ubuntu both include their own downloaders as well - Opera is better in my opinion.
