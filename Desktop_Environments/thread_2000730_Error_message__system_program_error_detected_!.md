### Post by sanderj on 2012-06-10
Well, as your screendump says: a "system program error detected" in oneconf-service version 0.2.8.

Click "Report Problem". There's probably an option "Ignore Future Problems"; check that option.

Make sure your system is fully updated & upgraded. There seems to be a Latest version:	0.2.8.1, dated 6-6-2012.

### Post by VanillaMozilla on 2012-06-14
If you look in Synaptic, you'll see that that is the Ubuntu One configuration service.  If you're not using Ubuntu One, you can disable it.  You can either remove it from your startup applications, or I would think you can simply uninstall it.
