### Post by jimbob on 2006-06-11
It also should appear under Applications -> System Tools.  The Dapper distro appears to come without System Tools under the applications menu for some reason.

If you right click on the empty space between Applications and Places on the top menu bar and then select Edit Menus you will be able to scroll down and click on system tools.  Just select those items you wish to appear under System Tools and close the window.  Now those should appear as selections.
