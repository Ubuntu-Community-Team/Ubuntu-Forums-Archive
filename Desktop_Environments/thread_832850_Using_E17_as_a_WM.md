### Post by squaregoldfish on 2008-06-18
It should be OK. I've been using E17 as my sole window manager for a couple of years now, and it's pretty solid. The only time it crashes for me is if I try to use any kind of composite manager on top of it. But since I don't need see-through windows, it's not really an issue :)

Also remember that you won't remove any of your other window managers, so you can always switch back if you don't like it.

Steve.

### Post by atomkarinca on 2008-06-18
For a better experience, I would suggest going with [OpenGEU]("http://opengeu.intilinux.com/"). It's got a built-in system tray (which sole E17 lacks) and gnome configuration tools which makes the transition easier. And you can just install it as a meta package, too.
