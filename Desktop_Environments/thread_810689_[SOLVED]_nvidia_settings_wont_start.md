### Post by Zorael on 2008-05-28
Do you have the [FONT="Courier New"]xserver-xgl[/FONT] package installed? If so, nuke it, save your work, log out, restart X (Ctrl+Alt+Backspace), log back in, see if things improved.

---

### Post by Forlong on 2008-05-28
Please post the output of
```
nvidia-settings -q VideoRam
```
Then remove Xgl:
```
sudo apt-get remove xserver-xgl
```
Afterwards, restart X and then post the output of Compiz-Check as well as the nvidia-settings command again.
