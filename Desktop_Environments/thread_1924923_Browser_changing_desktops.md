### Post by jpaulb on 2012-02-13
In Xubuntu 11.10 the browser of choice, in my case Firefox, has the nasty habit of moving from one desktop to another. If I am using app that has a link some where, say my mail program Claws, When I right click on the link to open in a Browser, Firefox which is open on another desktop jumps to the desktop where claws is open. This did not happen in Ubuntu 10.10 or in Debian testing using XFCE.

Is there anyway of stopping this annoying behaviour?

Thanks

Paul

### Post by LewisTM on 2012-02-13
I assume you are running the Xfwm4 window manager, i.e. unmodified Xubuntu desktop.

Go to the Settings Manager/Window Manager Tweaks
In the Focus tab, choose to Switch to the window's workspace when the window raises itself. Or to do nothing, as you prefer.

That's it!
