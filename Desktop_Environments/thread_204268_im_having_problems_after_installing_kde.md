[QUOTE=shiroma]the only reason i have a problem with this is under applications in gnome, it shows only kde applications. nautilus doesnt show up, only if i type nautilus in terminal.[/QUOTE]You could add a launcher to the toolbar. Right-click the panel, then "Add to Panel", then "Custom Application Launcher". In the "Command" field, put "nautilus". You can choose an icon if you wish, then click okay.
