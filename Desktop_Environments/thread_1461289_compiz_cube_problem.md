### Post by vamc on 2010-04-24
The solution is quite simple. Actually you opted to view the cube from inside. To set it back follow these steps:

Compiz config settings manager > Desktop cube > Behaviour (tab)


there you will find the first option "inside cube". Unselect the option. It will do.
