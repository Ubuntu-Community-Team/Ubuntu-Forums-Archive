### Post by meng on 2007-02-09
No ubuntu-desktop is just a meta-package (empty wrapper) for a whole bunch of applications. You will NOT remove GNOME if you remove ubuntu-desktop.

### Post by mcduck on 2007-02-10
> **paparucino said:**
> You are right, but, IMHO, in same way the system is "damaged". I don't know why, but I think that a lot of tools depends on ubuntu-desktop, so removing it it's no a nice idea.

No, nothing depends on ubuntu-desktop. It's safe to remove. No other packages will be removed with it.

Only thing is if you later on wish to upgrade to new Ubuntu version (instead of doing clean install) you should then reinstall the ubuntu-desktop package first to make sure that all new packages get installed.
