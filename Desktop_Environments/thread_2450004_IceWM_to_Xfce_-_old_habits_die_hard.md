### Post by steven43 on 2020-09-05
I'm really used to IceWM as my Windows Manager.

But I'm working on a project where I'm sort of, kind of trying to switch some users over from Windows to Ubuntu (although I'll primarily be using it).  I know that IceWM itself is really barebones - that's what I like about it.  But there's some aspects of Xfce that are also nice and it's mostly barebones too - mainly being able to control WiFi connections and power management (don't suspend, shut off screen on lid closed, etc).

But a few things that really bug the crap out of me with Xfce is window placement and taskbar placement.

For example, I generally use a lot of xterms.  I may have 5 to 10 xterm windows opened at any time.  With IceWM, starting a new xterm creates a kind of tile experience.  The first one is started in the upper left corner, then the next one that is started is exactly adjacent to the right of that first one.  With Xfce these seem to always open in the middle of the screen.  Is there a setting I'm missing in Xfce that accomplishes this?

The other thing that bugs me is the taskbar placement.  When I start a new application, xterm, firefox, chrome, whatever there doesn't appear to be any rhyme or reason as to where it's placed on the taskbar.  With IceWM these are sequential.  The first application you start is going to be the left most on the taskbar, followed by the next application started, etc.  And when you ALT+TAB in IceWM you get a full list of applications opened on that desktop and it highlights the selected application as you hold ALT and cycle through TAB to get to the right application.  Is there a way to do this in Xfce?

Like I said, I'm just really used to IceWM.  Been using it for, probably close to 20 years now - old habits die hard.

The other idea is to try and bring some of the Xfce usability into IceWM.  I can use idesk to give a desktop icon list.  I can use nm-applet to manage WiFi connections.  Haven't figured out power management yet.  I'm not sure how Xfce manages this behind the scenes.  And I haven't gotten mouse pointer control down yet in IceWM.  This is really my first foray into using a laptop or at least one that I really cared about management control like this.

Any pointers?

### Post by &amp;KyT$0P# on 2020-09-05
> **steven43 said:**
> The other thing that bugs me is the taskbar placement.  When I start a new application, xterm, firefox, chrome, whatever there doesn't appear to be any rhyme or reason as to where it's placed on the taskbar.  With IceWM these are sequential.  The first application you start is going to be the left most on the taskbar, followed by the next application started, etc. 

Settings > Panel > Items, choose the panel in question and select the "Window Buttons".  Click the gear to configure it.  Set "Sorting order" to your liking.
