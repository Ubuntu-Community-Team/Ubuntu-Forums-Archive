### Post by andrewski on 2006-05-26
On my desktop, the Windows key does nothing (OK, fine) so I'd like to use it for my compose key, which I tend to use pretty frequently.  I can do that just fine in the GNOME keyboard preferences, but then how can I make the right Alt key work like the left?  Especially for shortcuts such as Alt+Left, using the right Alt can be very handy, but it seems to do nothing at all by default. :confused:

---

### Post by andrewski on 2006-05-31
I figured this out.  In GNOME Keyboard preferences, on the Layout Options tab, I changed "Compose key position" to something other than Alt (Menu works for my purposes, the most useless key ever), and changed "Alt/Win key behavior" to (IIRC) "Alt/Meta are on the Alt keys (default)."  This made both Alt keys work as a modifier, rather than the default AltGr that it had otherwise.

By the way, I did figure out what it does by default: [http://en.wikipedia.org/wiki/AltGr](http://en.wikipedia.org/wiki/AltGr)
