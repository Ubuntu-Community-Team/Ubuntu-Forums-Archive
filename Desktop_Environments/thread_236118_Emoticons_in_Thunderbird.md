### Post by mssm on 2006-08-14
Hi,

For a long time I was using thunderbird 0.9 and there while composing a reply I had a nice bar just below the subject bar, containing emoticons and tools for inserting/enlarging fonts. Now I am using thunderbird 1.5 in Dapper Drake but that bar is no longer there. Even if I want some fonts in my letter to be bold-faced I can not do that easily. This bar is present in windows and mac osx version of thunderbird 1.5. Is their any remedy for it? I found no discussion about it in the forums here. I think I have installed thunderbird using automatix.Any help will be greatly appreciated.

### Post by mssm on 2006-10-16
Two months had passed since I first posted this problem but I didn't get any help from any reader. However, I was able to solve this problem myself. In the Account Settings dialog box of Thunderbird, under "Composition and Addressing" section, one has to check the box "Compose message in html format". This will bring back the bar just below the subject line with emoticons, while composing a reply to any msg.
