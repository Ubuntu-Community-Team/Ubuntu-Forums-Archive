### Post by lantzanders on 2010-11-25
> **Dave_L said:**
> What is a "queue function"?

Basically it lets you do stuff like: copy this will to here, then after that move this folder over there, and then delete these files.

Instead of having the file commander do everything at once, it places the task in a queue and executes them one after the other. 

I mainly use total commander to move stuff over networks in bulk, so I dont wanna bring down the entire network because I want to move around a couple of folders and files.
