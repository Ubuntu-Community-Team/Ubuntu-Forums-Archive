### Post by lkjoel on 2010-06-18
LinuxEssentials is a project to restart the Desktop Environment, without closing any windows.
It has been tested on GNOME and KDE, but not XFCE.
It is just a bunch of shell scripts, but it can save you from restarting the computer.

The Home Page is at [http://code.google.com/p/linuxessentials/](http://code.google.com/p/linuxessentials/).

There are many shell scripts, but usually you just need
```
chmod +x FASTGNOME.sh
./FASTGNOME.sh
``` or ```
chmod +x FASTKDE.sh
./FASTKDE.sh
``` or ```
chmod +x FASTXFCE.sh
./FASTXFCE.sh
```There are 4 bugs, noted on [http://code.google.com/p/linuxessentials/issues/list](http://code.google.com/p/linuxessentials/issues/list).
