Gnome is easy to configure for the vision impaired. You might want to give Epiphany - Gnome's browser - a try also. It is very easy to adjust the text size from the toolbar and uses the same rendering engine - Gecko - as Firefox. 

You can install the Kubuntu desktop if you want to give it a whirl. 
```
sudo aptitude update
sudo aptitude install kubuntu-desktop


```
