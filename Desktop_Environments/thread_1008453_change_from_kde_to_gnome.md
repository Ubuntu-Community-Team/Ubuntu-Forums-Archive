### Post by gjoellee on 2008-12-11
use this command:
```
sudo apt-get install ubuntu-desktop
```

when the installing is starting (after download) it will ask you which display manager you would like to have as the default one. If you are playing games as Urban Terror I would recommend keep using KDM (KDE Display Manager) otherwise change to GDM (GNOME Display Manager).

When the installation in finished simply log out, click on "Sessions" in the login window and choose GNOME. Now login.
