### Post by emperor on 2005-10-23
The "Boot Up Manager" (bum) lists all the services. 
Install via synaptic or apt-get.
sudo apt-get install bum
 see:  [https://wiki.ubuntu.com/BootServices?highlight=%28services%29](https://wiki.ubuntu.com/BootServices?highlight=%28services%29)
