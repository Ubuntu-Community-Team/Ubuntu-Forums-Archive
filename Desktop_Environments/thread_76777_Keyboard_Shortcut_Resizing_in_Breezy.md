### Post by pizzach on 2005-10-15
Anyone have any clue how I can get the window resizing back to old style hoary hedghog?  

Background:
I set up a Ctrl+Alt+s to let me resize windows in System>Preferences>Keyboard Shortcuts.  If I hit left, It would move the pointer to the left border and change to the resizing icon.  I can now resize the left border with the keyboard or mouse.  Then hit enter or click to set it.

The Problem:
1.  Press Ctrl+Alt+s
2.  Press left.
3.  Press down.
4.  Move the mouse around.

Expected:
The resize curser goes to the corner so you can resize width and height at the same time.

Actual:
The arrow move from horizontal resizing to vertical resizing.

With how I use a computer, I am very anti-mouse.  I use a mouse when I need to aim, nothing else.  I don't like to do fine point aiming.  For example: clicking the border, clicking window close buttons, menus, etc. I use a mouse for "Photoshopping," resizing once I do the resize shortcut, moving windows after hitting the keyboard shortcut (If I wasn't on a laptop I would probably use alt+click), changing window focus (I have the computer set to change focus by hovering not clicking), etc.

Thus, being able to use the keyboard shortcut to select a corner is insanely useful when you want to use a mouse to complete the action.  Plus, I can no longer resize mplayer because the windows has to be in proportion which isn't just annoying, it's a pain in the but.

### Post by pizzach on 2005-10-20
Found a temorary fix that is good enough.  I have a horrible feeling that I'm one of a very select few who uses their computer this way though.

Do your resizing key combination to initiate it.  In my case it would be ctrl-alt-s.  Then move your mouse to select the side/corner.  It's a bit of an acrobat act to get the exact part of the window select that you want and is no where near as precise, but it is still way better than aiming for a freaking small border to resize.

*Sigh* People give way too little love to the keyboard.
