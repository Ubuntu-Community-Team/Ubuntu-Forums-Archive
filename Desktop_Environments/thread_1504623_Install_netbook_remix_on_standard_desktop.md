### Post by sunk8 on 2010-06-08
Yup,
Very much possible.
Firstly, it's Netbook 'Edition' now, not Netbook Remix.
To install UNE, simply run the commands in the terminal...

```
sudo apt-get install ubuntu-netbook
```

For Kubuntu lovers,

```
sudo apt-get install kubuntu-netbook
```

If you don't like the transition remove the newly installed packages. If you do, remove the Ubuntu Desktop

```
sudo apt-get remove ubuntu-desktop
```

Solved your issue?
