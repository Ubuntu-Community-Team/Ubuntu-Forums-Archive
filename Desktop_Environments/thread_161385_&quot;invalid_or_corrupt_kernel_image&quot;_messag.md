### Post by davescafe on 2006-07-07
I just had this same error ("Invalid or corrupt kernel image") when trying to install 6.06 (Dapper Drake) on my desktop computer.  

My solution:
1.  I re-downloaded the ISO
2.  I downloaded the MD5SUMS file
3.  I checked the MD5SUMS against the ISO - they were good, so I...
4.  Burned a new CD-ROM

When the new CD-ROM was burned, I was putting it in the drive and I noticed that my old CD-ROM had a lot of fingerprints all over it.  I think this may have been the problem with the old CD.  In any case, the newly downloaded CD worked for me.
