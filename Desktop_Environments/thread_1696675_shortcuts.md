### Post by howefield on 2011-02-27
Make a launcher on the desktop ?

Right click on the desktop and select create launcher. For the Name field and Comment field, put whatever you want but in the command field type firefox followed by the URL of the page(s) that you want to open.

Click on the "springboard" icon to select an appropriate icon for your launcher if required.
