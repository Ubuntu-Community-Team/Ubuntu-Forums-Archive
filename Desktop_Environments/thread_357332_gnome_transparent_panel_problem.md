### Post by kevinf311 on 2007-02-09
For adding transparency to the terminal you would:

Open up a terminal >> Click Edit >> Click Profiles >> Click Edit >> Effects Tab >> Transparency/Opacity is a slider in one of the options.

If it goes all wonky like your panels did, then there is some corruption of data between the wall paper and the panel. I think all the transparency does is copy in wallpaper pixels for the corresponding position. It isn't true transparency as windows between the terminal (and panel for that matter) are cut straight through and the desktop is still displayed.

True transparency is actually one of the main reasons I installed Beryl ;)

I don't see any glaring bad from your xorg.conf file, and since this seems to be affecting all themes including the default Human, I'm a little bit at loss. If this were windows, I'd say re-install your video drivers. Perhaps this would be a fix here too? I know that I have had some problems with my video card drivers in the past and I'm a little gun shy when messing with them, so proceed with caution if you choose to do so.
