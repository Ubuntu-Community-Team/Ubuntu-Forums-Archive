### Post by ikt on 2010-10-12
Hey everyone!

I'm looking to test a screensaver to see if anyone else is having problems with compiz and electricsheep.

Specifically: (Electricsheep) Sits in upper left corner of screen when Compiz is enabled"

The screensaver is [SIZE="5"]electricsheep[/SIZE], you can get it via the ubuntu software centre (just search for electricsheep) or the old school way:

```
sudo apt-get install electricsheep
```

[SIZE="4"]Head into System > Preferences > Screensaver and set it to electricsheep and you should be g2g![/SIZE] 

note: it may take a few minutes for it to download a sheep, if you are of the instant reward type(like me) simply run electricsheep from the terminal, and it will give you a status that it is downloading a sheep.

Once all that's done: if you're happy reply in this thread :)

If you're unhappy please report a bug here:

[https://bugs.launchpad.net/ubuntu/+source/electricsheep](https://bugs.launchpad.net/ubuntu/+source/electricsheep)

The issue I'm specifically looking for comments on is here: 

[https://bugs.launchpad.net/ubuntu/+source/electricsheep/+bug/151380](https://bugs.launchpad.net/ubuntu/+source/electricsheep/+bug/151380)

Thank you :)
