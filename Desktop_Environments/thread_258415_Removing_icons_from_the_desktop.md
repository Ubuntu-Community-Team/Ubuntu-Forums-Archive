### Post by Omnios on 2006-09-15
Go:

 Menu / Applications / System tools / "open" configuration Editor

 In editor go:

 apps / nautilus / desktop!

 In desktop there will varios entrys. unclicking volumes_visable will remove all volumes and there should be entries for you partitions.
