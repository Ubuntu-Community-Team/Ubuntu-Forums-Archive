Go to "System -> Preferences -> Keyboard" and check
- "Layout" (above)
- "Keyboard model" and
- "Options -> Numeric keypad layout selection"

### Post by Krytarik on 2011-04-20
Obviously, you inadvertently enabled "Mouse Keys". Check in "System -> Preferences -> Keyboard -> Mouse Keys (tab). The keyboard shortcut to toggle those feature is Shift+NumLock.
