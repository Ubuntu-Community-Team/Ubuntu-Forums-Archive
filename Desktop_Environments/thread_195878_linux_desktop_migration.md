### Post by sgla1 on 2006-06-13
Kubuntu provides an (apparently) easier migration path from windows...

My wife, an experienced windows user, recently migrated to Ubuntu.  Initially she tried the gnome desktop but found it's layout and conventions counter-intuitive.  She was particularly frustrated by the gnome file dialog boxes.

When I installed Kubuntu-desktop for her she achieved a much greater level of comfort.  For her, the layout of the KDE desktop, and especially konqueror and the kde file dialogs were much more logical.  Now, her only complaint is that wine won't run wordperfect 12.

I think there are two related reasons for her preference for KDE.  First, it is similar to windows in many respects.  That's not a bad thing as Microsoft spent a boatload of money on UI research.  Second, KDE, like windows, is very consistent internally.  The file dialogs are like mini-konquerors--if you understand one, you can understand the other very easily.  Programs also are internally consistent, easing the learning curve for new users.

OK, fla^H^H^H^H away.  Just remember, we who have voluntarily adopted linux are not the target audience for linux desktop adoption.
