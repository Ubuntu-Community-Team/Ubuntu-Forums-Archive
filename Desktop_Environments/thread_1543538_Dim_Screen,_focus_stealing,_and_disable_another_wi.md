### Post by mcduck on 2010-08-01
You can't get exactly the same effect, since that's related to the type of the window itself (modal dialog window versus a normal window).

But simply enabling the "ADD helper" plugin in Compiz gets you pretty close. It fades everything else to background leaving only the currently active window fully visible. (It won't prevent you from interacting with other windows, though).
