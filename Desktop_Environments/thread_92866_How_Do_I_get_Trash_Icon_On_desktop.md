### Post by SavageBeginner on 2005-11-20
Select:  Applications -> System Tools -> Configuration Editor

Then:  /apps/nautilus/desktop

Check:  trash_icon_visible

EDIT:  ^^Sorry these instructions are for Gnome.

For KDE:  Right click on the destop and select:  Create New -> Link to Location (URL)

Then:  For 'File name' type 'Trash' , and for 'Enter link to location (URL)' type 'trash:/'

Click OK
