### Post by rax_m on 2007-07-21
In Nautilus, where you see the directory "buttons" that appear as you navigate, there is an icon with a text document and pencil. If you click that it shows you exactly where you are in a directory form.

### Post by mcduck on 2007-07-23
Other way is to hit Ctrl-L to show the location bar in Nautilus. And of course you can just configure Nautilus to always use the location bar, I think the setting is in Nautilus settings but if it's not there you can change it with gconf-editor.
