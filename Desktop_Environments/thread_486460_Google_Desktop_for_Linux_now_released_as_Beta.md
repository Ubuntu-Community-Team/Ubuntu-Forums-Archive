### Post by tagra123 on 2007-06-28
CONDENSED VERSION IF YOU ARE IN A HURRY

The quickest way to install Google Desktop is to download it from here:

[http://desktop.google.com/linux/](http://desktop.google.com/linux/)

Download  google-desktop-linux_1.0.1.0060_i386.deb  (debian/Ubuntu)

Once downloaded right-click on google-desktop-linux_1.0.1.0060_i386.deb and install with gdebi, 

OR

from a terminal after downloading

sudo dpkg -i google-desktop-linux_1.0.1.0060_i386.deb

Once installed to make everything show up on the panel type

killall gnome-panel



Once installed it defaults to folder that most would want to commonly search, add the other folders with the preference menu.
