### Post by Sutekh on 2005-11-18
I have both GNOME and KDE installed on my PC, is it possible to make GNOME the default DE for a particular user and KDE the default for another??

---

### Post by miscz on 2005-11-18
Yes, install them both, then in GDM/KDM (login screen) choose session -> desired DE (Gnome/KDE), it will ask if you want to make it default for selected user.
