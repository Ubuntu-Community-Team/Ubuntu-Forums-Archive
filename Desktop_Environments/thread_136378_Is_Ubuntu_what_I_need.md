### Post by Rumor on 2006-02-25
I have a Linksys router as well and was also running BPFTP Server on ym WinXP box. The XP Box was my home PC and I would access it from the office via TightVNC.

Now my home computer is running ubuntu 5.10, Pure-FTPd and remote desktop. I am able to access it from the office using TightVNC just like I could when I was running WinXP.

So, yes, I belive you should also be able to do the same things if you go ubuntu on the computer you are contemplating.

Set up Pure-FTPd using this guide after you've installed it from synaptic:
[http://www.ubuntuforums.org/showthread.php?t=91052](http://www.ubuntuforums.org/showthread.php?t=91052)

Remote access via VNC is set up from System | Preferences | Remote Desktop
