### Post by habtool on 2008-02-20
As above, Xubuntu, should be faster as a standalone install.

however once you install a few gnome applications and a few kde ones like say Amarok/k3b/katapult/kontact/Akregator/ktorrent, then the libraries of gnome and kde are also installed and running in memory when the apps are running..

At that point there will be little speed improvement over a Ubuntu/Kubuntu.

If your are to run Xubuntu with only GTK apps, then you may have a small speed improvement.

It all depends on what apps you install, if you use a mixed bag as mentioned above, then you will only really use XFCE because you like XFCE as a desktop environment.
