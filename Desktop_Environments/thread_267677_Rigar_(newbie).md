### Post by orb9220 on 2006-09-29
Not needed in linux or linux partitions since the ext3 filesystem is not prone to fragmentation.

Be sure to empty .trash folders in / root and /home directories.

Note to see those in nautilus you have to select show hidden folders in the view menu ot ctrl+h to turn on.

Also nautilus prefs you can add a right-click delete file command which will show up under send to trash.
