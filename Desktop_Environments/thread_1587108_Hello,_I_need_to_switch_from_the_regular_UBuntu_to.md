### Post by ankspo71 on 2010-10-03
Hi,
Look in Synaptic Package Manager for ubuntu-netbook-remix-default-settings and ubuntu-netbook-remix and install those packages. Once it is done installing, you can then log out and change your session type to the netbook desktop, then log back in. I think that is all there is to it.

To upgrade from 9.10 to 10.04, I think all you would have to do is this: 
Press ALT + F2 and type:
update-manager -d
then follow the instructions.
(it might take approximately 2 hours)

I have tried out the Ubuntu Netbook Edition, but I didn't use it long enough to notice any differences in packages. It does have the different desktop design for smaller screens and easier application access though. 

In 7 days the final release of Ubuntu 10.10 will be available, and the Ubuntu Netbook version will have a completely new netbook desktop. 

Hope this helps.
