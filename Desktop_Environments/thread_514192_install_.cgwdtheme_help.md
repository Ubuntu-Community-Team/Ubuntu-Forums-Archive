### Post by mcduck on 2007-12-01
Just do what was suggested, rename the package to .emerald. Then all you need to do is doubleclick it and the Emerald Theme Manager will install it for you.

Emerald is used in Compiz Fusion just he same way it was used in Beryl. It's one of the possible window decorators that Compiz/Beryl/Compiz Fusion are able to uset o draw your window borders and titlebars.

(edit: of course you also need to have both Emerald and Emerald Theme Manager installed to be able to use that theme ;))
