### Post by omkrasu on 2010-09-24
Hi!

Reviving this old-old thread, but i give it a try...

So there's this applet called Window Picker from the Software Center.

It replaces really neatly the titles with large icons.

I didn't know about it until i started to use Dockbarx and Talika.

On Xfce, if you like their panel more, there's Iconbox, it's really nice too.

The Window Picker cannot work as a launch bar like Dockbarx, but on some specially titled open windows, it expands the name. The only thing that might be a problem, that it don't work vertically. Gives me too big icons.

A simple solution for bigger screens, where it's easier to get info from the icons, than trying to read all the windows titles.
