### Post by oldos2er on 2009-10-20
Run ```
sudo apt-get install rar unrar
``` in a terminal, which will give you rar compression in Archive Manager.

### Post by mcduck on 2009-10-20
..and of course if it doesn't have to be a .rar package, then you don't need to install anything. Just right-click whatever you wish to compress and select "Create Archive", and you'll get options for selecting between several different archive types.
