### Post by Krytarik on 2011-04-01
I'm afraid I can't help you with the wifi issue, since I have not much experience with wifi, for my machine I use only a wired connection so far, and the wifi at my mom's machine worked out of the box, both in 7.10 and 10.04. But I may lookup this later.

As for the video driver, try upgrading the driver through Ubuntu-X-Swat PPA, it offers more recent drivers than the offical repos:
[https://launchpad.net/~ubuntu-x-swat/+archive/x-updates]("https://launchpad.net/%7Eubuntu-x-swat/+archive/x-updates")
```
sudo add-apt-repository ppa:ubuntu-x-swat/x-updates
sudo apt-get update
sudo apt-get upgrade
```If you removed the driver after the last failed trial, run those commands instead:
```
sudo add-apt-repository ppa:ubuntu-x-swat/x-updates
sudo apt-get update
sudo apt-get install fglrx
```You shouldn't install the driver manually with those installer, because
- it is not a safe install, you can't remove the driver easily and safely
- you have to re-install the driver each time when the kernel gets upgraded
- the driver will not get updated automatically
- the driver is not necessarily considerably more recent
- the driver doesn't necessarily provide a considerable higher performance

As for learning commands, and using the terminal, please see here:
[https://help.ubuntu.com/community/UsingTheTerminal](https://help.ubuntu.com/community/UsingTheTerminal)
[http://linuxcommand.org/](http://linuxcommand.org/)
