Now I would like to go to the normal Gnome desktop, but I don't want to do a complete re-install since I have many things set up the way I like them.

---

### Post by Endwin on 2006-08-25
All you need to do is install ubuntu-desktop. You can search for it in synaptic or in a terminal type ```
sudo aptitude install ubuntu-desktop
```
