### Post by mfox on 2019-11-21
I only experienced this problem since upgrading to Ubuntu 19.10. I'm running gnome. The problem is that when I cut or copy text by right-clicking with a usb mouse and then go to paste it, the text is being picked up every other time. In between, pasting enters no text. I am experiencing this on a desktop computer (2011 iMac), as well as a laptop (Dell xps 13 2 in 1) with a mouse plugged in. I also tested this with a live usb of 19.10 on the desktop, and it happened there as well. I tried with several different mice; same result. I then tried it on the desktop running the Cinnamon DE instead of Gnome and the mouse worked properly in this case. Oddly enough, it doesn't occur in all applications (it does in LibreOffice Writer but not in Gnome Terminal), and it doesn't occur when I use the keyboard to cut or copy instead of a mouse right-click. Very strange!
