### Post by Frogs Hair on 2011-11-02
If the application has an icon  open the main menu .Locate the icon for the application , right select on the line , enter properties and add the command in the in the command dialog box .

If the application has no icon you may be able to add on from the main menu by selecting add new item . you can add an icon .png   by selecting the box on the left and moving to the directory with the chosen icon .
