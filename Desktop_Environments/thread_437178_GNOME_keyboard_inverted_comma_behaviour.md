### Post by mistermax on 2007-05-15
I didn't realise that the top node in the Keyboard tool was selectable - I had been using the UK with dead-keys type, but found that I could change it to the normal UK type and now everything is fine.

If anyone needs to know where this is look in

System
Preferences
Keyboard
Layouts tab
click add, then search for the layout, testing it in the typing box on the first tab of the Keyboard tool.

Hope this helps!
