### Post by ninotob on 2006-01-01
Multiple desktops is one of the uber-features you just don't get elsewhere.

When I'm at work, I usually use 3 of my 4 desktops.  One has nothing but my procrastination stuff in it.  This place, slashdot, stock info -- that kind of stuff.

The second has only my mail and calendar so that both are easily accesible.

The third has open-office and my DB app running so I can work on a clean desktop.

The last, that gets special things.  If I need to do some CLI stuff or whatever, I'll do it there and close it out when done.  Basically, any misc. tasks.

My 2nd biggest beef with OSX is it's gaping omission of multiple desktops.  There are 3d party apps to create that functionality, but none are as good as what we have right here.

### Post by anil_robo on 2006-01-01
If you have used windows XP, and if you tried opening more than 6 internet explorer or windows explorer windows open at the same time, you'd fall in love with workplace switcher.

I normally use four workplaces. The fourth always has XMMS playing music for me. The first one almost always has firefox open. I use the other two for doing some document editing in openoffice, as they also open a lot of windows.

In windows, it's almost impossible to have more than a dozen windows opened without killing your system resources. That's why they never implemented the workplace switcher (besides the fact that they may not believe in making things simple yet powerful for the user). Vista has copied several features from Linux (like the root/user system, password entering for administrative tasks etc) but they still aren't able to accept workplace switcher.

I find myself helpless without workplace switcher because when I work (on the internet or edit my documents) I **like being given a lot of space to work with.**
