### Post by realistik on 2009-11-08
After experimenting around with the common methods of automagically switching wallpapers in Xfce, my coding muscle itched, so I created a simple app to sit in the tray and switch wallpaper for me. I wrote the app in C#/Gtk# with mono because I wanted it done quickly. After I had it working to my satisfaction, I rewrote it in C/Gtk+ because I figured if I'm sharing it with everybody, there are lots of people using Xubuntu (or Xfce) who'd want to use it that for one reason or another hate mono too much.

Anyway, info and such are on a blog I created the other day, as well as links to where to get the files. The blog is [http://xubuntuexperience.blogspot.com/](http://xubuntuexperience.blogspot.com/)
