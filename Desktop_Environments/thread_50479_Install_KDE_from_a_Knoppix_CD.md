### Post by Joeb on 2005-07-20
Most likely, installing the KDE from Knoppix will not work as it will be looking for dependencies that are specific to Knoppix and you won't have them on your Ubuntu installation.

Your best bet, if you are on dial-up, might be to find someone with a broadband connection and download the kubuntu iso image.  It is my understanding that the kubuntu project is basically an Ubuntu install that has KDE instead of Gnome as the desktop.

You do realize, however, that you do not need to install all of KDE to run KDE apps.  You could just install the app and any dependencies will install for you automatically (ie you can install k3b without installing KDE).  It will still need to download a lot of stuff, but not nearly as much as a full KDE install.  Also, if you have several KDE apps, the shared kdelibs only needs to be downloaded once.  So your first kde app installed will take the longest time to download.

Joeb
