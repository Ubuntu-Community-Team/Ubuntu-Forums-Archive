I started studying in Sweden this year. I am enrolled in a Swedish language course and I sometimes have to type an assignment. The problem is that my laptop has the US layout (and therefore misses the letters åäö). Right now i copy/paste each letter, but this becomes really annoying. In windows i could get the å by using the key combo cntrl + alt + w, but that's not working in ubuntu. Could someone please give me a tip to work around this problem ?

### Post by sbrown1992 on 2009-11-23
System > Preferences > Keyboard
Then go to the Layouts tab
Click on your layout, then click the Layout Options button

Click on the arrow next to "Compose Key Position"

Choose a Compose Key. (I use the right windows key)

Then, if you press Compose > ' > e you would get é

Wikipedia has a list of key combinations you can press
[http://en.wikipedia.org/wiki/Compose_key](http://en.wikipedia.org/wiki/Compose_key)
