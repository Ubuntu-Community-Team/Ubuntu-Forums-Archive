### Post by PaulFr on 2008-02-03
Is this what you want? 
1. Right click on KMenu button in left corner, select Configure Panel in Panel Menu (or right click just right of this KMenu button and select Configure Panel).
2, There is a size in pixels in the dialog that pops up. The value I had was 48 pixels and I changed it to 24 pixels without any problems.

** {UPDATE} No this is not what you want - this is for 3.x and not for KDE 4. Please ignore.**
