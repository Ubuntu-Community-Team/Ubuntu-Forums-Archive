### Post by ubunterooster on 2010-07-23
pres alt+F2 type ```
alacarte
``` now hide everything but firefox and games

Oh, and see privoxy a web filter [http://www.privoxy.org/ ]("http://www.privoxy.org/")

or  Foxfilter [https://addons.mozilla.org/en-US/firefox/addon/4351/](https://addons.mozilla.org/en-US/firefox/addon/4351/)
