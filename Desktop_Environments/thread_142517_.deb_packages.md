### Post by taurus on 2006-03-10
What do you mean you try to open it?  If it's a .deb, then you can install it with dpkg...

```

sudo dpkg -i linux-wlan-ng_0.2.0+0.2.1pre21-1.1ubuntul_i386.deb

```
