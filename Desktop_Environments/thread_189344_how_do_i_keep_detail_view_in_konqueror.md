### Post by ajgreeny on 2006-06-05
I agree with you completely, it is so much easier to use the list of details (tree view in KDE, but I,m not sure in gnome.)

In Konqueror, after setting the view as you want, just go to the settings menu, Save view profile, chose "file manager" as the profile name, and this wil save the view as you have it, even down to the window size and folder opened if you tick the right boxes.

Brilliant, isn't it?
