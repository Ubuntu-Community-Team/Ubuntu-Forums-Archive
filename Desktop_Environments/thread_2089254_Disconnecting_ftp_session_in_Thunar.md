### Post by LewisTM on 2012-11-29
There is no point in putting an FTP site in the shortcuts menu because it will disappear as soon as you disconnect/logout/reboot. Thunar up to 1.4 does not support remote bookmarks, that's what Gigolo is for.

If you want Thunar to manage your FTP connections, you can try out the development version which now is at 1.5.3. It has a few bugs but is safe to use. It support tabs and remote bookmarks, among other improvements.
[http://www.webupd8.org/2012/11/install-thunar-with-tabs-support-in.html](http://www.webupd8.org/2012/11/install-thunar-with-tabs-support-in.html)
