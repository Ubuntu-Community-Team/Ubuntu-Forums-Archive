I have found [Mnemosyne]("http://mnemosyne-proj.sourceforge.net/index.php") quite helpful. I added flashcards from Wheelock's Latin 6th Edition ([source]("http://www.warmenhoven.org/latin/vocab/")) to the sourceforge page of the project some time ago if that helps. The program's in the repos, but it's a little [out of date]("https://bugs.launchpad.net/ubuntu/+source/mnemosyne/+bug/180625"). The [ubuntuguide.org]("http://ubuntuguide.org/wiki/Ubuntu:Gutsy#How_to_install_Mnemosyne") has some info on it too.
