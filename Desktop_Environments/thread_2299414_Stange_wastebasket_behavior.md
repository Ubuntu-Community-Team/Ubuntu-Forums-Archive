### Post by ajgreeny on 2015-10-18
There is a bug for this problem at [https://bugs.launchpad.net/ubuntu/+source/thunar/+bug/880533](https://bugs.launchpad.net/ubuntu/+source/thunar/+bug/880533) which shows that thunar 1.6.10 has overcome this.

I am still using Xubuntu 14.04 but have added the ppa for XFCE 4.12 from [https://launchpad.net/~xubuntu-dev/+archive/ubuntu/xfce-4.12?field.series_filter=trusty](https://launchpad.net/~xubuntu-dev/+archive/ubuntu/xfce-4.12?field.series_filter=trusty) and it all seems to be fine in this which does use thunar 1.6.10.
