### Post by z-vet on 2005-07-09
[QUOTE=fig_jam_uk]Hi im actually at work at the moment so i will post the xorg.conf file later and yes i have the nvidia drivers installed. thanx[/QUOTE]
 Well, i'll wait to. For now i can say that if you try to use Xorg's composite extension with NVidia, it will not work. I had X crashes when installed Kubuntu and found a message in Xorg.0.log saying that i can't use them (NVidia GLX and composite) together. So i just disabled composite and have no crashes since then. 
Imho, composite is nice, but too buggy to be used right now. Yes, i love eyecandy on my desktop, but love a stable system even more.
