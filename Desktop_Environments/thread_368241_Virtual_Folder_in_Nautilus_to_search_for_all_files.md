Anyone know how to setup a Virtual Folder in Nautilus to search for all files of a given type?  So I could, for example, click on the sidebar to see all my .pdf's.

---

### Post by mcduck on 2007-02-23
Open Nautilus, click on the search button, do a search for whatever you want, and when you got the results go to Bookmarks/Add Bookmark. That should add your search on the sidebar :)

Edit: if you want to save the search as icon on your desktop (or anywhere else), instead off adding a bookmark go to File/Save Search As..
