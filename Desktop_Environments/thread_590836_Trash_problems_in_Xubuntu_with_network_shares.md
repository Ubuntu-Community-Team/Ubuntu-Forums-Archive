### Post by Tvanover on 2007-10-25
I have mounted a network share that gives me access to household DVR.  As I watch shows I need to be able to delete them, freeing up space for new shoes.  Currently when I attempt to delete a show it tries to first, copy the show to my computer, and then move it to th trash.  While this is a functional solution it does make management of the DVR a bit slow.  Is there anyway to make exceptions when a file is deleted that they won't be sent to the trash, but just deleted?

---

### Post by Jouke74 on 2007-10-25
Select the file you want to delete, and press shift-delete. It will skip the trash, and delete the file.
