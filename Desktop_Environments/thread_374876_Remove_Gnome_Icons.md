### Post by ComplexNumber on 2007-03-03
**navilon**
bapoumba is correct - its determined by the theme.
the only way that i know that is workable that will enable you to get rid of the icons there altogether is to replace the icons in the theme that should be there with transparent icons. therefore, it will look as if there is no icon there at all.
the OK icon that you wnt to be looking for is called gtk-ok or gtk_ok or gtk_accept (it can be under a different name, depending upon which theme).
