### Post by davmor2 on 2006-06-14
This is for several reasons:
1/ the size of the picture.  If the picture is smaller than the window it tends to make it as big as the picture rather than the window.
2/ You need to go full screen then select fit to window.
3/ It's gthumb it general use and basic if you want something that does the job better use something else.  (f-spot,GTKAM,etc)
