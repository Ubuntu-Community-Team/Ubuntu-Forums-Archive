### Post by TheFu on 2022-05-07
The CPU being supported really isn't the issue. It is all the other components which will be new as well where the concern is.  The NIC, audio, touchpad, any fingerprint reader or webcam?  Are those supported by linux out of the box - no hassled?

A Core i9 really is a premium purchase.  Only a Java developer or Android developer would likely need that. Be certain to get 64G of RAM too, if you are 1 of those.  For everyone else who isn't full of money, a current Core i5 is typically faster than a Core i7 from 2-3 generations ago.  Core i3s are faster than Core i5s from 2-3 generations ago.  Just something to consider.

Plus, if you are a gamer, the GPU is _much_ more important than the CPU.  I'd say to get a desktop for gaming, not a laptop.

If you are doing office work, a Core i3 with 16G of RAM would be a pretty great laptop and provide 10+ hrs of battery.
