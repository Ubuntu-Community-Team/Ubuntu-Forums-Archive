I have only used Kubuntu and Ubuntu but I hope this will help.

I use Ubuntu mainly but can also boot in Kubuntu ([http://www.psychocats.net/ubuntu/kde](http://www.psychocats.net/ubuntu/kde)) via session selector.

VirtualBox([http://www.virtualbox.org/](http://www.virtualbox.org/)) which works fine on KDE, Gnome, and should work well on XFCE

or

Wine([http://www.winehq.org/](http://www.winehq.org/)) which should also work on all distros.

Thank you for using Ubuntu!

### Post by 3Miro on 2010-10-01
Ubuntu is the official one an is the most stable of the three. For a fist time user, it is probably the best.

Kubuntu comes with KDE, which is a different Desktop Environment (i.e. it looks different, has a different set of default applications and some differences in underlying technology). Some people like KDE some people like Gnome (the Ubuntu DE). It is a matter of personal choice really, however, in my experience Kubuntu works better on a Desktop. I have had some glitches with Kubuntu + Laptops.

Xubuntu comes with XFCE, which is yet another DE. XFCE aims to be extremely light-weight and has considerably faster than the others. Overall it looks a lot like Gnome, however, speed comes at the expense of a few features.

The good news in all of that is that you don't have to choose. You can install all three DE (actually 4, you can add Lubuntu + LXDE to the mix) and upon boot, you can choose which one you want. You can try them all and pick the one you like (or maybe different people in your family would like different ones).

### Post by 3Miro on 2010-10-13
I have spend the last month or so playing with different distros. I have 5 currently installed don my desktop and I am loving it. Here is what I learned:

Ubuntu = Gnome and while the other variants like X/L/KUbuntu are nice (you still get this forum as well as the hardware center as well as the huge repository), the *ubuntus are not he best representatives of their respective environments.

Kubuntu often runs into sound problems due to the different schemes used by KDE and Gnome. Xubuntu has too much of Gnome in it (for example I couldn't "safely remove" drivers with Thunar, I had to use nautilus). All of my experience with LXDE is that it is very fast and snappy, but also somewhat unstable and it lacks some crucial features. I don't recommend Lubuntu for someone completely new to Linux.

The best bet is to install Ubuntu. If you want to try the other DE, then you can install the corresponding desktop packages and check out what others have to offer (just keep in mind that it is sometimes hard to uninstall everything that you have installed).
