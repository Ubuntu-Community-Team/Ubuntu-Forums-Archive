> **Perpetual said:**

Many Gnome programs have individual config files which will allow you to specify the window coordinates (size and placement), so you can do this without Compiz, but it would be time-consuming.

KDE, btw, does allow you to set window-opening behavior. In the "desktop" section of of Kcontrol, there are several options for default window placement: center, smart, tiled, etc.
