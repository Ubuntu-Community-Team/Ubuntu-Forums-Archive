### Post by peterhoeg on 2010-10-12

> is there a way to disable the behavior where the desktop will switch if I use the scroll mouse on it? I find this very annoying.

thanks!

Yes, in KDE 4.5 do the following:

1) Right click on the desktop
2) Desktop Settings
3) Mouse Actions
4) Delete the "Vertical-Scroll  Switch Desktop" line
