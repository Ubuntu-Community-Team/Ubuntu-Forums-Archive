### Post by GoldBuggie on 2005-12-03
Ok I don't have a complete solution but maybe something you can fiddle with.

I have a Toshiba Laptop and some keys when set to basic aren't working so change it to nodeadkeys and it worked. Not saying that this is the solution for ya but what you could do is try and find the closest model to your keyboard. Both Toshiba Satellite and Generic works for greatly for me but I have to set it to nodeadkeys as mentioned before.

You can do this at kcontrol->Regions & Acc... ->Keyboard Layout->Then Enable keyboard layout
There you can fiddle with Keyboard Model and Layout Variant.

The effect is instantanously so when you apply so you can try many different models and layout var. in no time.
