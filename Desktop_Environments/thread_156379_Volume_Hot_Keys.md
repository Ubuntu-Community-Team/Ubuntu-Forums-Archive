### Post by rejser on 2006-04-07
install aumix, then in khotkeys you add function (I think it's called function).
name it, chooses that it will be a keyboard shortcut. and command to run is aumix -v -5 (volume down), aumix -v +5 (volume up), mute (for mute)
you kan run khotkeys from command-line if you don't find it in the settings-manager
