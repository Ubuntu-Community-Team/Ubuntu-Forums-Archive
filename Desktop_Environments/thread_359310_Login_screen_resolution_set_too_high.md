### Post by stinkball on 2007-02-13
I seem to have fixed mine by just playing around a little.

I am using fglrx drivers 8.33.6

try this:
```

sudo aticonfig --resolution=0,[put your wanted resolutions here separated by commas]

```
for me, it was:
```

sudo aticonfig --resolution=0,1280x1024,1024x768,800x600

```
from aticonfig:

```

--resolution=Screen#,W1xH1,W2xH2,W3xH3,...
        Set the modes for the specified screen.  You may specify several
        resolutions separated by commas.
        Screens start at 0.  You can use 1 for dual-head

```
hope this helps :)
