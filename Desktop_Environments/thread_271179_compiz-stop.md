### Post by megamads on 2006-10-04
You can use the compiz-manager for that. It's a tray icon, where you can restart compiz, or switch to metacity. I think the package has dissapeared now though since compiz isn't called compiz anymore. 

Its now called beryl, with new packages and a new manager (Beryl Managager). The window manager is called Emerald. I don't know if i should reccomend you to upgrade yet, but it works just dandy on my lowend laptop.

Check [beryl-project.org]("http://forum.beryl-project.org/") for more info.
