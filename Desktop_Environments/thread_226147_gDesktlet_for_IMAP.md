### Post by adamonduty on 2006-07-30
I haven't found a gDesklet for secure IMAP connections, but I did find one that ran in the system tray. gnubiff is in the repositories and supports secure IMAP.

If you're using thunderbird you might try out the new mail notification extension: [http://moztraybiff.mozdev.org/](http://moztraybiff.mozdev.org/) . I use this and find it very reliable and well integrated into thunderbird. gnubiff does have the advantage that thunderbird doesn't need to be running in order for you to see that you have new mail.
