### Post by BionicGear83 on 2010-02-13
Is there a way to edit KDE applications' (K3b, Ktorrent, etc) font size, general looks, etc in Ubuntu(GNOME environment) without using [sudo apt-get kde-desktop]?

---

### Post by Lord Blackadder on 2010-02-13
You would want to install "systemsettings" from the package manager.  It should then install as "System Settings" in your menu (or just run "systemsettings" from a run box or terminal).

That should allow you to adjust the look and feel of all KDE apps.
