As far as I remember, it does if you let the installer use the entire disk.
Btw, the usual approach to dual booting is installing Windows first, then install Linux.  In your case you might want to use the product recovery discs for ThinkPad to restore Windows.  Read some dual booting guides before installing Ubuntu again...
