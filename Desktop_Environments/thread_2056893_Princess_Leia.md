I came across [http://dagobah.princessleia.com/Xubuntu/12.10/](http://dagobah.princessleia.com/Xubuntu/12.10/)

Since 12.10 will have Xfce 4.10, it may be worth mentioning the ways in which 4.10 differs from 4.8. One difference is the use of expanding separators in the top panel if one wants to tweak some things there:
> The window buttons plugin no longer expands, which makes the plugin positioning more flexible. In order to restore the previous behavior please add a transparent separator plugin with the Expand option enabled just behind the window buttons plugin.
 from [http://www.xfce.org/about/tour?lang=en](http://www.xfce.org/about/tour?lang=en) under the section **Windows Buttons**. The link has more on how 4.8 and 4.10 differ.
