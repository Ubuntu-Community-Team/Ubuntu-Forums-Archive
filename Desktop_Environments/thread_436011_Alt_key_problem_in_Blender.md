### Post by snoop on 2007-05-08
I am not sure how to stop moving it around, but to work around it for now, you can use the fullscreen blender mode and this will not happen.

Edit: Found out how to stop it. Go to system > Preferences > Windows and change the movement key from alt to ctrl or super.
