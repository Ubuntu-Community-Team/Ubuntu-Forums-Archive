### Post by dresnu on 2005-11-30
Make sure you have superkaramba installed. Otherwise just type "sudo apt-get install superkaramba" in a console. Then you should have it in Kmenu->Utilities->Desktop.
For karamba themes check out [www.kde-look.org](www.kde-look.org)
