When I rename a file into dolphin, it open a new little window where to insert new name files; is it possible to deny dolphin to open this new little window and rename file into the same dolphin window???

---

### Post by Zorael on 2011-06-03
Settings -> Configure Dolphin, General tab, check **Rename inline**.
