### Post by TeaSwigger on 2007-10-17
Oh it's possible all right. Fluxbox has a similar capability in that you can drop pics into a specific folder for desktop pics and the pics in that folder will be listed in the selection menu for choosing a desktop. The list self-refreshes if contents change. Further, if you select one of the files listed, it Fluxbox calls an app to set the selected image on the desktop.

How to do it in XFCE or anywhere else for that matter, I've no idea. But keep looking and asking, hopefully someone more knowledgeable about scripts etc can help.
