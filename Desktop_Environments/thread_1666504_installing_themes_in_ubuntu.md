### Post by Frogs Hair on 2011-01-13
1. Download theme and save .
2. Open downloads folder.
3. Open appearance preferences .
4. Drag the theme package to appearance preferences and drop
5. select new theme(if the theme doesn't appear use customization tab to locate the control and
   window border and select them.)    
5. Download wallpaper by right clicking the image and save to  pictures.
6. Right click image and set as desktop background.

There are many ways to install themes , this will work for theme you have chosen.

### Post by Frogs Hair on 2011-01-13
That means the theme is installed . Open Appearance Preferences to the themes tab and select customize.

When the control tab opens , locate and select the control for your theme from the list . Next go to window border tab and locate and select the window border. 

Sometimes themes don't appear in the main window of Appearance Preferences and you locate them using the customize tab.
