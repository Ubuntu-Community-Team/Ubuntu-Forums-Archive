how do i alias the ls command so that i could type di instead?

---

### Post by intangible on 2005-07-01
**gedit ~/.bashrc**

add this:
```
alias di='ls'
```

It will be applied in every new terminal window you open from now on.
Type it into the command line to enable it in the current terminal.
