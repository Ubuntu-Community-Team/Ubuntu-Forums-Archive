### Post by grahammechanical on 2013-05-17
I really do not know what you mean. There has been a massive improvement in font design in 13.04. It is also possible to install community designed themes and icons. In fact, I read somewhere that Canonical had employed one of the most respected theme/icon designers to give a work over to the Ubuntu icons.

The default font is a Ubuntu created font. Canonical is not going to change that other than to improve it. There must be community designed fonts available for installation.

You seem to be forgetting two things.

1) Ubuntu is a community project as much as it is a Canonical project. If Canonical provided everything then there would be no place for community.
2) Working with the Ubuntu community gains a person influence. Ubuntu is not a commercial product. We do not buy it. A user's wish list has little influence with developers because developers are free to write code for projects that interest them. We, the user, accept what we get. Or, we go and pay for a other product. Only then do we have a right to complain about the product.
