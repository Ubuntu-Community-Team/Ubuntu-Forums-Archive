### Post by doublewitt on 2011-10-16
You can try this simple way:

1) go to system: preferences: - MAIN MENU
2) click on OTHER
3) in the list, right-click on OPEN FOLDER
4) select PROPERTIES
5) for the command, enter **dolphin %i -caption "%c" %u**

after that, dolphin should open instead of nautilus...
