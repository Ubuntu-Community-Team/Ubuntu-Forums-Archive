### Post by CatKiller on 2019-02-09
Forms are rather different. There are a bunch of tools that have implemented some parts of the form-filling stuff, including Evince, Okular, and Libre Office, as well as dedicated pdf editors. Thankfully it's not something I've ever had to do. Only someone with access to the particular file you're interested in could say whether all that you need is implemented in a given application. The seemingly-crude version checking you've shown from that website may well be another hurdle. 

If you do have to use Acrobat Reader I would pick a new version running in Wine over the abandoned Linux version. All versions of Acrobat Reader are notorious as malware vectors: the most recent version will give you the most band-aids.
