### Post by mcduck on 2010-09-30
Sure, the option is in Firefox itself (I assume you *are* talking about Firefox, since it's the default browser and you mentioned "address bar". Anyway, mentioning what program youa re talking about when asking for help would be a good idea... :))

Just type "about:config" as the address, and then use "urlbar" as filter and you'll find options for changing how clicks on the address bar work.

In this case you'd want to change "browser.urlbar.doubleClickSelectsAll" to "true".
