### Post by HDave on 2011-11-10
OK -- here are my fixes/hacks/solutions

5) button to see/switch workspaces - the launcher workspaces button is badly broken and underpowered

The primary solution to this was to install a cool workspaces indicator that some awesome Ubuntu user developed.  Been using it for a week and it is VERY reliable.  See the instructions for the PPA in the answer here:

[http://askubuntu.com/questions/77160/how-can-i-tell-what-workspace-i-am-in-with-unity](http://askubuntu.com/questions/77160/how-can-i-tell-what-workspace-i-am-in-with-unity)

Also note the following two keyboard shortcuts:

Ctrl-Alt-Left / Right  -- this will change workspace instantly
Ctrl-Shift-Alt-Left / Right -- this will move the current app to a new workspace




6) show desktop - lack of a show desktop button was a real problem, though I eventually found a great hack to do that.

Again, the solution can be found on askubuntu.com  here:

[http://askubuntu.com/questions/77249/where-is-showdesktop-button-on-11-10](http://askubuntu.com/questions/77249/where-is-showdesktop-button-on-11-10)

I went with the launcher icon solution that is based on the wmctrl utility.  It is fast and reliable.

Pathetic that this doesn't come out of the box....
