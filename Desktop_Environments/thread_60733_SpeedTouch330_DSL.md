### Post by Keffin on 2005-08-29
No, the modem itself is not changed. This modem has firmware loaded onto it by the OS every time it is used. Following the guide will load up the same firmware on it that windows does. And if it didn't, it does not matter as it is lost as soon as you stop using it anyway. I also use this modem between windows and linux.
