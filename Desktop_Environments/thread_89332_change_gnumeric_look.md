### Post by SickTwist on 2005-11-12
I believe Gnumeric uses the default GNOME icon set which is also used by most of the other programs (Firefox and OpenOffice.org are two execeptions that I can think of off the top of my head). I'm not sure if changing the default icon theme (System --> Preferences --> Theme, click on "Theme Details" then "Icons" tab) will affect these icons or not.

I think that one of the issues is that there are a *lot* of different icons in GNOME/GTK+ and few icon themes have enough icons included to replace them all.
