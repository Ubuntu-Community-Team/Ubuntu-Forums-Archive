### Post by lifestream on 2008-10-23
1) gksu vs gksudo.
I believe gksu asks for the root password, but gksudo asks for your user password.
Since ubuntu doesn't have a root password, use gksudo. :)

2) mystery donut?
It makes  the window stay in *ALL* workspaces. 
A workspace is like having several desktops. Not sure if you have them? Try ctrl+al+left/right/up/down arrow keys.
The donut could be useful, say for a terminal, if you want to work on it from all workspaces.

3) blank run line?
Hmmm I'm not sure! Mine doesn't do that!
Then again a simple backspace will clear the entire word.
