### Post by mcduck on 2013-09-01
What kind of display are you using? And what cable are you connecting with? It sounds like overscan, which some televisions might enable unless you specifically tell them that the connected device is a computer (usually either by using a computer-specific connector, or by enabling pixel-per-pixel mode).

So if it's a TV instead of a computer monitor, check the television's manual first. (and no, Compiz won't help you there, and while there are some ways to force the image to fit on Ubuntu's side, it's usually the display where this should be fixed)
