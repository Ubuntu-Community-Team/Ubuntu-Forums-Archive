### Post by lovinglinux on 2010-02-08
To make it like the Mandriva menu, right-click on the menu launcher and select the option "Switch to Classic Menu Style".

To edit the launcher icon, right-click on it and select the option "Application Launcher Settings", then click the Icon option in the editor dialog.

### Post by Zorael on 2010-02-08
Depending on what version of Plasma this is, it could have been broken out into its own widget. This is at least the case in 4.4.

So to add it, unlock widgets, click the panel cashew, hit Add Widgets, then find Application Launcher Menu and double-click it. Drag it to where you want it, and le done™.
