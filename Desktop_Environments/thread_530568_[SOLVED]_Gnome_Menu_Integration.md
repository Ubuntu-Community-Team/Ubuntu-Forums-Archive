### Post by scruffybeard on 2007-08-20
Ok, I'm fairly new to Ubuntu having previously been using Debian. What I'm wanting to know is if there is a way to make programs I have installed show up in the Applications menu.

For example, I am a big fan of Dopewars, and have compiled it from source. In Debian (using the same source file) this program automatically appeared in the Games menu, but in Ubuntu it does not. I am quite capable of firing it up via command line, but I'd rather cut out the extra step.

I'm looking for more of a general response rather than a specific response based on that package because I would love to do this to several other programs that I use frequently as well.

### Post by Edward Hou on 2007-08-21
You can add the Debian menu as an option to the Ubuntu main menu. Just install package menu-xdg and restart gnome or relog or something to that effect.

You may need to enable the submenu using alacarte.

### Post by rainwalker on 2007-08-21
As people have said before, you can right-click on the menus and choose "Edit menus" or go to System -> Preferences -> Main Menu
That's where you can check the boxes for what you want to show up in your menus, and if things aren't there that you want in them, you can add them in there yourself.
