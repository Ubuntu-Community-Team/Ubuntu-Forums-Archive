### Post by Jakin on 2012-12-16
I was using a widget called "py-cashew" for the longest time- however if you are using KDE 4.9.4. it may cause stability issues- in which case you should then remove it.

[http://kde-apps.org/content/show.php/Py-Cashew?content=147892](http://kde-apps.org/content/show.php/Py-Cashew?content=147892)

The cashew at the end of the topbar there, just indicated that the desktop widgets aren't locked, and to make it go away- simply Right Click "lock desktop widgets"
