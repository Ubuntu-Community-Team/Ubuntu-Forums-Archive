### Post by Jucato on 2006-02-21
The current release of Ubuntu (Breezy or 5.10) only has 2.6.12-10 as its latest kernel version. You would have to wait for the next release (Dapper or 6.04) to have a more updated kernel (2.6.14 I think) installed. It's possible to compile the kernel yourself, but I wouldn't recommend it, since these latest kernels have not been customized for the Breezy release yet.
