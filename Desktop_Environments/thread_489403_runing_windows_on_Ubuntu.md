### Post by mig5 on 2007-07-01
I doubt that you can use a recovery CD to install on a virtual machine.  It depends on whether the recovery disk lets you install a fresh copy of Windows.
Also, if it is Windows XP then you might have problems with activating the virtual machine as the hardware in the virtual machine is different, and therefore microsoft might not acitvate it if you have already installed windows on your computer.  By the way it's actually illegal to use the same windows disk for your windows install and your virtual install, you are only supposed to use one disk for one thing.
