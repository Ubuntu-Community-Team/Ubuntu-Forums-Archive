### Post by SnickerSnack on 2010-05-03
I don't know a whole lot about gnome, but I can list what I would try, so you can try that in the meantime until someone with more knowledge comes along.  (After each step, either log in and log out, if that was getting the error before, or you may need to restart each time.)

1. Disable all special desktop effects: System > Preferences > Appearance, then click the "Visual Effects" tab, then click the "none" option.
1.1. Possibly unnecessary, but try it anyway if step 1 has no effect: uninstall Compiz if you have it installed.
2. Go into Synaptic and reinstall gnome.
2.1. Reinstall everything that seems gnome related.
3. Install xfce and/or kde.  Log out and look at the panel at the bottom of the screen.  There should be a "sessions" option; set it to another desktop environment, and log in.  Same problem?  Try this with the other DE.

Edit: was this a clean install or an upgrade from 9.10?
