### Post by mafitzpatrick on 2006-06-13
Ok. Answered my own question:

To remove these items you need to go Places->Network Servers, then right-click on the item & choose "unmount".

Not the most intuitive interface I've seen!  Hopefully this function will be added to the right-click remove that appears on nautilus places panel.
