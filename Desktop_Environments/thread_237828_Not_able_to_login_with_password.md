### Post by taurus on 2006-08-16
Maybe you use a wrong password for that account.  You can change it to something you can rememeber by booting your Ubuntu into recovery mode from GRUB menu.  Then at the prompt, type

```

passwd john
(assuming john is the login name...)

```
Remember, lower case letter is NOT the same as upper case in Linux/Unix so be real careful about that...
