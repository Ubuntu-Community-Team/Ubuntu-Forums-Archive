### Post by ajgreeny on 2013-04-21
No, I don't think you are missing anything.

Some users (many?) seem to prefer Mint to plain Ubuntu, on which Mint is based, because it saves a few minutes getting codecs and a few other things.  Personally I have used both distros and always come back to Ubuntu which seems to run faster on my machines (actually, Xubuntu since unity came along).

Some have described Mint as "what Ubuntu should have been", but I think such things are purely down to personal taste and preferences, and this was not my opinion at all.

If Ubuntu plus cinnamon works for you, go with it, and enjoy the choice.
