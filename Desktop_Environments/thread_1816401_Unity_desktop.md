### Post by mcduck on 2011-08-02
> **b1lancer said:**
> Ok. At the risk of sounding like a total noob (I am, even after a few years), can you explain what a shell is and how it works?

A shell is software that provides an user interface (be it graphical or text-based).

In case of Unity and Gnome Shell, they are graphical shells that provide you with a desktop interface. Both run on top of Gnome (meaning that all the underlying components, frameworks, tools, programs etc. come from Gnome desktop environment).

I suppose a fairly good way to put it would be that the shell is what you see, while the desktop environment is what makes it work. :)
