### Post by sgosnell on 2009-02-21
You shouldn't need to edit grub or anything else.  Run the Ubuntu install, and when it asks for partition options, have it keep the Fedora partition and add another partition for Ubuntu.  That should be the default choice.  After the install you should get a menu giving choice for booting, into either Fedora or Ubuntu, and get it on every boot.
