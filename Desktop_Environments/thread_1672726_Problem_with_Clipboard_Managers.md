### Post by ajgreeny on 2011-01-21
You can get rid of parcellite history totally by deleting the file ~/.local/share/parcellite/history but you need to shutdown parcellite first (right click in panel) to be sure everything goes.

I have tried editing that file with gedit (won't open) and with nano in terminal (does open) to get rid of one entry, but it does not seem to work, though I suspect it could be made to, if you knew exactly how to delete just the parts of that file you want rid of.
