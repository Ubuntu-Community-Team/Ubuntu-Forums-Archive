### Post by RedSquirrel on 2007-08-04
**xubuntu-desktop** will give you the full Xubuntu with all of the applications (abiword etc.). If you just want the core Xfce, then there is the **xfce4** metapackage which will install only the basic stuff that Xfce requires.
