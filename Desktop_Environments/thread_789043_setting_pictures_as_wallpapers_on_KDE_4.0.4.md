Does anybody know how can I set a picture I just downloaded as wallpaper? because I don't the option anywhere at all...I'm using KDE 4 as my default desktop enviroment but I can also use Gnome where I see the option all the time...

---

### Post by I3aH on 2008-05-10
Right click the desktop.
Select Configure Desktop.
Click browse (It's a little icon at the right side of Picture: )
Browse and select picture.
Click Apply :)
