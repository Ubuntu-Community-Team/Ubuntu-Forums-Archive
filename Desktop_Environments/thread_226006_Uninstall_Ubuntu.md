### Post by Fass on 2006-07-30
No, it will not. When you reinstall Windows XP (I'm assuming that's what you want) it'll replace grub with its own bootloader. If you aren't reinstalling windows, then you'll have to boot off the windows install CD and choose to run a recovery console and then use the fixmbr programme found on the CD.
