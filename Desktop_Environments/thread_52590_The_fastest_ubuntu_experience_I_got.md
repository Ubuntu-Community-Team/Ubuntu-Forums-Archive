### Post by suoko on 2005-07-28
Hi, 
I'd like to report the fastest ubuntu esperience I've reached up to now.
Here is the list of programs I use to get a fast responding sense:

1) Xfce with rox-filer as file manager and desktop manager (you can only put simlinks on desktop but its fine once you get used to it)
2) Softmaker office suite. I took the free licence edition out of the SAM live distro by just copying the softmaker directory into my home in ubuntu. [I know its proprietary software but its bloody fast and very good with MS documents]
3) Galeon as webbrowser (faster than firefox in loading)
4) Mainactor as movie editor. The latest version is fast and really stable. [again proprietary software but it works very well]

hope it can help people who thinks ubuntu is sluggish.

### Post by shakin on 2005-07-29
If you want a really fast browser, try links2. It can run with the -g option to open in an X window and browse in graphical mode. It's quite good and makes Opera look like it's on a 2400 baud modem :)

You will need to get used to keyboard shortcuts since the menu is weird to use with a mouse. Ctrl-g - opens the address bar dialog. It's certainly not good enough as your only browser, but does offer very fast browsing for most sites.
