### Post by paulvandenberg on 2005-06-10
I don't seem to have any stability issues with Kubuntu. However, I have upgraded KDE to 3.4.1. The Kubuntu home page has an entry in the news section that tells you how to get 3.4.1 packages. I know that this release of KDE fixes the problem gaining access to admin mode in KControl.

I agree that Kynaptic needs more work and I think that it is coming. However, I think it is usable now. The function I miss is the option that Synpatic has to clear out the archive where downloaded packages are stored. However, a 'sudo apt-get clean' command from a terminal does that job well.

Another thing to remember that Hoary is the second release for Ubuntu, but only the first for Kubuntu. No doubt Kubuntu will improve next release. I am sufficiently impressed with it already, that I don't think I will download Fedora Core 4 on Monday as I had planned.
