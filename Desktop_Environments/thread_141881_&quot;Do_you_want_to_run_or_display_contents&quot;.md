### Post by stig on 2006-03-09
I have set up Ubuntu and started to copy my folders of files from a Windows PC and transfer them to the Ubuntu PC. Now I find a problem opening some of the files, even though they are simple documents saved as ASCII text files and RTF files.

When I double click on these files in Nautilus I get a box saying the files are executable (which they are not) and asking me "Do you want to run [file name] or display its contents". I then have to click display for the file to open in Gedit. If I open Gedit first, browse to the file and click it then opens OK.

How can I stop this warning and have the files open immediately from a double click?

Thanks!

Many of these files

---

### Post by kaamos on 2006-03-09
You can remove the execute permissions for the files (right click -> properties) or choose the "display always" option in nautilus preferences.
