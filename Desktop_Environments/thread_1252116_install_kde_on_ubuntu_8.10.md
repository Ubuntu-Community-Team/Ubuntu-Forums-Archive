### Post by Tibuda on 2009-08-28
Add your kubuntu cd to your software sources in system > admin > software sources. I think if you insert the cd while running gnome, it will popup a message to add it.

After the cd is in your software sources, you can install the [kubuntu-desktop]("apt:kubuntu-desktop") in system > admin > synaptic.

If you have problems adding the kubuntu cd to your ubuntu sources, you can download the kubuntu package from the internet using synaptic.
