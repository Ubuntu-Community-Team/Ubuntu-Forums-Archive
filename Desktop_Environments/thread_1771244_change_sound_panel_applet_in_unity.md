### Post by stinkeye on 2011-05-31
According to dconf-editor players are added to the sound menu if they use the MPRIS2 spec.
I dont think vlc uses this spec and can't use the sound menu.
Rhythmbox and Clementine are added automatically.
I uninstalled Banshee (mono aversion) and removed it from the list using dconf-editor.
