### Post by bswilson on 2006-06-04
Friends, I upgraded via apt-get to Dapper the first day it came out.  No major issues, but one thing I noticed is that there was no more Screensaver Preferences menu item on my Gnome System->Preferences menu!

You can run this command to open the Screensaver Preferences:

```
/usr/bin/xscreensaver-command -prefs
```

I used the Alacarte Menu Editor to add a menu item on my Gnome menu.
