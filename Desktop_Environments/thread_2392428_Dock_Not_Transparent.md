### Post by monkeybrain20122 on 2018-05-23
You can make the dock transparent with the dock to dash extension (not the crippled one bundled by Ubuntu, but the one you get from the website) So install gnome-tweak-tools (tool?) ,  remove the Ubuntu dash to dock (whatever they call it) Install dash to dock from the gnome extension website, then configure transparency from tweak tool. Even though I use unity in  18.04 I find the dynamic transparency feature in dock to dash pretty cool.
