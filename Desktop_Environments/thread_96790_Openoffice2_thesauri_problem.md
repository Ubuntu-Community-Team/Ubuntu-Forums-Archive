In OO, go into Tools-Options-Language Setting-Writing Aids-Edit (Available Language Modules) and see if there is a Thesaurus option **for the language you are using**. Then make sure it is ticked.

In Languages you will have to see if your default language has the "ABC" tick icon set, if not then you may not have the right dictionary files installed.

The DicOOo tool is very handy for installing missing OO dictionary files (do a Google search for it).
