### Post by Zorael on 2009-11-20
Right click your desktop and pick Add Panel (unlock widgets first if locked). Once it's up, right click it and pick Add Widgets, then find Application Menu in the list and add it back.

If it's not responding when you're right-clicking the desktop, Plasma might have crashed. Hit Alt+F2 and run 'plasma-desktop', then retry the above if needed.
