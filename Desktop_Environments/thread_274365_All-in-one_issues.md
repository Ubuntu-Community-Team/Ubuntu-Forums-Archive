Linux or Ubuntu are not know to cause any hardware problems of any kind.

If you cannot even get to the bios screen, I would suspect hardware failure.  If you can get to the grub menu and boot into recovery mode, perhaps your graphics card has been misconfigured (or you installed the latest kernel update without the corresponding restricted modules package and despite that are trying to use the proprietary drivers....  Something like that...)

But I would not think that installing Ubuntu borked your box.
