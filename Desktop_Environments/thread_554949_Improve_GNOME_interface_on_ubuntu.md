### Post by Wolki on 2007-09-19
You should be a bit more explicit in what you want, the SuSE style GNOME may be an improvement to some, personally I quite dislike it.

For the suse menu, it's (confusingly) called gnome-main-menu, you should be able to install it from synaptic.

> **iangregson said:**
> Can anyone tell me how i can improve the gnome that is installed with ubuntu 7.04 .... For example the bar is at the top that has application, places and system..

IMHO there are countless ways to improve it (most of them involving filling the empty space with lots of more or less useful things (music applet, various system monitors, Tomboy, Deskbar, Wanda etc.). If you tell us what you'd consider an improvement, we might help you.

Right-click the panel on an empty spot to add stuff ("add to panel"), right-click an applet to move or remove it.

> I presume ubuntu supports themes of some kind also?? (gnome) ....


System -> Preferences -> Theme
(in gutsy/7.10, it will be System -> Preferences -> Appearance)

> Any advice would be really appreciated... i have installed compiz fusion and the nvidia driver in restricted driver manager... but should i really download Nvidia driver direct from nvdia website...


In general, you'll have a lot less trouble if you use the nvidia driver from the restricted module. I'd only recommend installing it manually from nvidia's site if you really need a newer driver.
