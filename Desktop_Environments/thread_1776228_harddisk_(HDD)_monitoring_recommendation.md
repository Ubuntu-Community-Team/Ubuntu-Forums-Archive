### Post by kjuergen on 2011-06-05
After playing around a bit with the new to me Xubuntu 11.04 I couldn't find anything but mdadm to monitor my RAID1 array. mdadm is a great tool but if you don't use it often you most likely will forget the CLI commands.
Google was again my friend and I came accross a great tool which is installed in regular ubuntu and I believe it should be a default in Xubuntu too: Disk Utility.

I would strongly recommend to install it as it makes monitoring HDDs a snap. The only thing I am missing is that you can't start a demon to report critical changes to email (or root@localhost) but this can easily be done with mdadm.
