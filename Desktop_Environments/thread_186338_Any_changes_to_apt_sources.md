### Post by ash211 on 2006-06-01
Nope, no changes are required.  The big hullaballoo about Dapper coming out was just that the final CD image was made and put up for download.  For those of us who have been running Dapper for a while now, the release isn't anything new.  Just run ```
sudo apt-get update; sudo apt-get upgrade
``` and you'll be fully up to date!
