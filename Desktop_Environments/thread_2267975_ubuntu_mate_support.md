### Post by grahammechanical on 2015-03-05
8 years ago when I first started examining Linux as a suitable operating system for the computer that I was thinking of building there were a few concepts that I found very hard to understand. 

1) Free and Open Source Software (FOSS) development which meant that Linux distributions were available for download and installation without charge.
2) Programmers writing code in their free time and not demanding payment.

I have learned to accept these concepts as facts of life in the FOSS software development universe. Some software projects are managed by foundations that are supported by voluntary donations which in turn are used to pay some developers for their time. But individuals giving of their time and skills freely for the benefit of others is a foundation on which FOSS projects rest and that includes Ubuntu itself as well as those distributions that are built on Ubuntu.

Ubuntu is supported by a commercial organisation called Canonical. It employs software engineers and directs the area in which these engineers work but there is still a very large volunteer community that is part of the Ubuntu developer community. The help provided by members of this forum is an example.

If you are really concerned about Ubuntu Mate needing full time, paid developers then perhaps you would like to donate to the Ubuntu Mate project.

[https://ubuntu-mate.org/blog/](https://ubuntu-mate.org/blog/)

If you truly only want a computer operating system that is developed and maintained only by paid, full time employees, then forget about Linux. Stick with what you pay for when you buy a computer from a shop. But you will miss out on the quality, professional work done by the FOSS community.

Regards

One difference between Xubuntu and Ubuntu MATE is that MATE uses the full LibreOffice suite, while Xubuntu uses the Abiword/Gnumeric pair as word processor and spreadsheet.  I tend to avoid Abiword, as it is not fully compatible with LibreOffice and many other forms frequently used on web sites.

On the other hand, if you're as determined as I am, you can do as I sometimes do: install Xubuntu, remove Abi, and install LibO (provided you don't mind a lot more downloading).
