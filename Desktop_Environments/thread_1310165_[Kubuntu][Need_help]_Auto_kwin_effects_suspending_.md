### Post by mrboojive on 2009-11-02
Under System Settings > Desktop > Desktop Effects > Advanced, there is a check box that says "Disable functionality checks." You might try messing with that. I had to check it to get desktop effects to work, but maybe you need to uncheck it to get the functionality you want.
