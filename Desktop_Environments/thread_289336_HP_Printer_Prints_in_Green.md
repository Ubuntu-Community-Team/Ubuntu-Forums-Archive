### Post by Lord Illidan on 2006-11-02
When you print, go to Properties, then click on the Device Tab and adjust Resolution, Quality, etc, to 300 dpi, Color, [B]Black + Colour Cartr
[/B]You can change the dpi, but make sure that it will use Black + Colour Cartr.

EDIT : If you want to make it permanent, then go on File : Printer Settings, Properties, and follow the same steps as above. Here is a screenie:
