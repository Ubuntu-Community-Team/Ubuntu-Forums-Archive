### Post by 3rdalbum on 2013-11-13
[www.gnome-look.org](www.gnome-look.org) has themes. You'll generally want to look for Metacity and GTK-3 themes; Metacity is the window borders and GTK governs the look of buttons, popup menus, etc.

You can install them through Gnome Tweak Tool which is available in Ubuntu Software Center. Certain themes might have their own install script or install instructions.

There are no "animated" themes, GTK and Metacity don't support it. You can change the amount of time it takes to fade in/out windows and menus by tweaking around with Compizconfig Settings Manager, but I highly recommend leaving it alone until you have several years Linux experience. If you touch the wrong thing you can easily break your desktop and cause it to stop working.

### Post by Frogs Hair on 2013-11-13
The Compiz Configuration Settings manager from software- center includes various desktop effects. The CCSM  may break the desktop and unless you take the time to learn how to fix it you may want to hold off before experimenting with it. The CCSM also has some practical useful effects combined with resource hogging eye candy.
