### Post by tahuya_rat on 2007-05-31
There is no way to truly dual-boot between OSes without restarting the machine.  The closest thing you will find would be something like [VMWare Server]("http://www.vmware.com/download/server/"), which would allow you to run a virtual XP machine under Ubuntu, or vice versa.

It really works quite well, but don't know if it would meet your specific needs.

-tahuya_rat

-edit:  VMWare Server is available in the repositories (Synaptic), btw, and is also part of [Automatix]("http://www.getautomatix.com/").
