### Post by squirebug on 2014-10-15
I have a Cambridge Audio DACmagic USB dac I wanted to use with Xubuntu 14.04 and Banshee.  Initially I got only laptop speaker output.  Searching for pcm and volume control yielded no answers to how to get output through the DACmagic from Banshee.

By accident I found that if you go to multimedia>pulse audio volume control>playback when Banshee is playing you can select "2.0 root hub Analogue Stereo" on the button on the top right of the Banshee section to get output throug the DACmagic.

This is confusing because the DACmagic documentation says you need to select 2 channel pcm output, which is not listed.
