### Post by guilloip on 2010-01-11
A change in the default button from CANCEL to OK have been recently done in the "Change Resolutuion" dialog box of Ububtu (with gnome).

When you change your screen resolution and the system tries it, the default option during the trying process used to be "keep the last resolution", but now it is "use the new resolution".

This is very dangerous, because if something goes wrong and you are not able to see your desktop with the new resolution, you could have a lot of head aches trying to go back to you original reslotion, specially if you are a not experienced linux user.
