### Post by charlieg on 2005-07-27
It is possible if you run bleeding edge (and I mean eye, nose, and ear bleeding edge) X stuff which provides the composite extensions which allow things like [Luminosity](http://live.gnome.org/Luminocity) and [in the future] tricks like transparency for icons when dragging them.  However you'd be in a bit of a bind if you wanted it to happen without lots of hackery at this point in time.  So, er, the answer (despite my blurb) is still, "no."

Still, you can chat about it:
