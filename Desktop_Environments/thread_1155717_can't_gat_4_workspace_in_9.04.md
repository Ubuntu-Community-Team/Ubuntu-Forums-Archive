### Post by Psicolonia on 2009-05-11
go to sinaptics package manager (system -> administration -> synaptics...)

click the "update button"

find "compiz-settings-manager", right-lick and select "install"

Say ok to whatever may come up.

Then go to System-Preferences->compiz settings (or something alike)

Select the desktop cube option, that should be somewhere.

I can't remeber if before you have to go to "system->preferences->apeerance->effects" and select the middle button. (I don't remember...)
