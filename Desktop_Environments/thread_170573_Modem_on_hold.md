### Post by granny4linux on 2006-05-05
ZOOM V.92external modem Model=3048C works great, plugnplay; it's a serial modem though. I've used it with Knoppix, Mandrake, Fedora Core 1-2-3, Kubutu and Ubuntu.

Re receiving phone calls, when my land line is busy it is set up to forward to my cell phone; with caller ID I can choose to take a call or let the cell take a message. It costs $1.00 per month.

I'm not sure if this program would help; found it in synaptic; it's called mgetty. Info on it can be found here:
