### Post by guildofghostwriters on 2010-10-19
Install a package called Art Manager from the software centre. Launch that from system>preferences. Then in Art Manager, art>desktop themes>icon. Select an icon set, click install & Appearance Preferences launches once the icons are downloaded. Customise>icons and your new set is listed there for you to select. If there's no icon theme there that catches your eye, there are websites with others - [http://gnome-look.org/index.php?xcontentmode=121](http://gnome-look.org/index.php?xcontentmode=121) for example. Usually you click to download an archive & select to save it. Then from appearance preferences select install and the archive should be in your downloads directory.
