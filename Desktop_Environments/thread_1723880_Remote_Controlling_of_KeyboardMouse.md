Some times I want to do some maintenance work through the GUI so the obvious thing to do is have some kind of remote administration software like VNC. ---

### Post by zitch on 2011-04-07
You can look into Synergy.  Looks like it's available in the repos (universe, I think?), and is cross-os capable (I've used it to control a Windows XP client with an Ubuntu laptop's keyboard and mouse; it should work fine the other way too).

More info might be available here: [http://synergy-foss.org/](http://synergy-foss.org/)
