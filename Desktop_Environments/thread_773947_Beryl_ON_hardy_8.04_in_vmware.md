### Post by mozetti on 2008-04-29
Beryl ended back in Gutsy. Compiz Fusion is the combination of Compiz & Beryl, so it's the successor to Beryl. It is installed by default in Ubuntu 8.04 (and Gutsy, i think).

Are you running Ubuntu 8.04 in VMware, or running another OS in VMware that is installed on your Ubuntu 8.04 installation?

If you have Ubuntu 8.04 installed in VMware, I don't think you'll be able to use Compiz for the effects. Virtualization software (VMware, Virtualbox) doesn't have 3D graphical capabilities.

If you are running another OS in VMware that is installed on Ubuntu 8.04, I'm not sure what might happen. It could work, but I don't know.
