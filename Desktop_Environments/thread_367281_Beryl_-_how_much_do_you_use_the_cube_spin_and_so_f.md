### Post by vexation on 2007-02-22
A lot of it is gratuitous.  I hardly ever use the Cube.  But there are a lot of features that help in my workflow, too.  Scale, window transparency, window thumbnails, live previews in the switcher, to name a few.  And it all runs quickly without much penalty to performance, so there's no harm in a few transition animations to make it flow better.  Beryl can be tampered down easily by simply disabling the plugins you don't want to use.  I don't think they should be completely axed, since who knows when somebody will find that stuff useful.
