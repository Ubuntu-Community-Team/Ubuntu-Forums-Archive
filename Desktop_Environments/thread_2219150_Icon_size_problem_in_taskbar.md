Try this out and see if it works:


> [LIST]
[*]Increase font dpi (System Settings &#8594; Application Appearance &#8594; Fonts &#8594; Force font dpi, enter a number such as 125 or 150)

[*]Increase icon sizes (System Settings &#8594; Application Appearance &#8594; Icons &#8594; Advanced, here choose a higher icon size for all icons, eg. just increase each icon type by one step) 
[/LIST]


If not, there are a few tips here that might be helpful: [https://community.kde.org/KDE/High-dpi_issues](https://community.kde.org/KDE/High-dpi_issues)
