### Post by standa12345 on 2011-07-16
I managed to find the database files in home/.config/xpad. I couldn't find the folder at first because it is a hidden folder and you have to change the view settings in nautilus to show hidden files. Just copy the text files to make a backup. It seems like the server file located in that folder prevents you from copying the whole folder.
