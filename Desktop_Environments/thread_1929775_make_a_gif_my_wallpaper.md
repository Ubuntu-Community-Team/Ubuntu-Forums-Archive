### Post by SteveDee on 2012-02-22
> **cold37 said:**
> how can i gif my wallpaper like this...

You can use Gimp to create an animated gif.

To get started, just create a new image and then add (say) 4 layers. Add different coloured shapes or letters to each layer in different places (i.e so they appear side by side). Then save as an animated gif and set the delay value for each frame/layer to (say) 300ms.

You could also open your example gif in Gimp and see how it has been made.
