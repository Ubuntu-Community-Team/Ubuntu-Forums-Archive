### Post by DarthNull on 2010-05-11
If anyone is interested in using Adobe's BlazeDS which has libraries for Java <-> AMF, there is a new tutorial posted on Adobe's website written by Dwayne Nickull.
[http://www.adobe.com/devnet/livecycle/articles/blazeds_ubuntu_linux.html](http://www.adobe.com/devnet/livecycle/articles/blazeds_ubuntu_linux.html)

AMF is the native binary format used by the Flash Player. It offers really fast communications between your server and any Flex, Flash or AIR application (about 5 X faster than JASON and 30X faster than DOJO for RIA's).
