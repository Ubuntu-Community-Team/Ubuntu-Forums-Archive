### Post by Danny Dubya on 2011-06-01
I don't know if this will help at all, but what I've found after messing around with this a little, seems to indicate that this is an issue caused by the NVIDIA binary driver, and is not Kubuntu-specific like the black resize handle. (I can't reproduce it on a laptop using Kubuntu 11.04 with Intel graphics, but can on the original desktop running openSUSE + NVIDIA-binary).
