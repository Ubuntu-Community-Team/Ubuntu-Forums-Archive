### Post by mcduck on 2011-04-14
You can select if from the login screen. There's a "Session" option at the bottom of the screen after you've selected a user (but before typing in the password).

### Post by Copper Bezel on 2011-04-14
Notice, though, that you can't run Unity and Gnome Shell installed on the same system, because they depend on different versions of Gnome. Still, there are a wide variety of desktop environments to choose from, and most won't conflict with one another.
