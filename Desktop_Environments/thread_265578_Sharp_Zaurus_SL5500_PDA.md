### Post by RikoW on 2006-09-26
I used to use a Zaurus some time ago. Then, I was using multisync to synchronize PIM information between evolution and the Zaurus. Worked pretty reliably. But, as far as I know, it wonly works with the local PIM information of evolution, so you can't sync with your exchange calendar if you access one with evolution.

As far as I remember from back then, there is also kitchensync, which is a KDE application. Never tried it, though.

Hope that gets you started :)
