### Post by bcbc on 2010-12-30
If you install it via CD from windows (using wubi) you will lose everything. If you want to upgrade from CD you need the alternate CD.

You probably shouldn't upgrade the wubi install anyway as it's known to cause boot problems. (I think you already had this problem with 10.04 - so it will most likely do exactly the same when you upgrade).
