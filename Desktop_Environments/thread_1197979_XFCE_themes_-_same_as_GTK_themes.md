### Post by 4Orbs on 2009-06-26
Since last year Xubuntu can use gtk2 and xfce themes, but the window mgr is still xfce. So some gtk2 themes might appear incomplete when the window border doesn't match the theme's screenshot. Those gtk2 themes might also require a different gtk2 engine be installed to fully appreciate their features. Some of them depend heavily on Emerald theme mgr also.
