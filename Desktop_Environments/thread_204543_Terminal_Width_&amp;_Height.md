### Post by Ramses de Norre on 2006-06-27
Launch it with ```
-g=700x600+70+2

``` and set the values to your likings ;) the ???x??? are the dimensions and the +x+y are the x and y offset.
You can add this option to your menu entries and/or launchers or create an alias for it.
