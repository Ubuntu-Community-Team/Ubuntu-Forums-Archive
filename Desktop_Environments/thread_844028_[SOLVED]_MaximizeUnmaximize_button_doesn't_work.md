### Post by cpetercarter on 2008-06-29
When a window is "unmaximised" it is possible to drag the edges using the mouse, or to drag the bottom right hand corner diagonally upwards to alter the size of the window. When you have adjusted the window size for a particular application, Ubuntu will normally remember this size for the next time you open the application.
