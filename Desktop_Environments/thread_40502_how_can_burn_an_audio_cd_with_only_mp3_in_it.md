### Post by littlepr on 2005-06-09
Select the data option instead of the create audio cd option. Then load the MP3's and burn away. That should result  as an MP3 format cd.  

Just in case you intend to play these on an mp3 enabled player, make sure the player reads from root. If it doesn't then chances are you will have to create folders with names preceded by numerical digits starting from 01,02,03 ( Ex.  01Rave, 02Rap,03 Reggae .... as so on). This is how I have to do it for the cd player in my car.
Yours might be different so read the manual that came with your cd-cdrw-mp3 player.
