### Post by santiagoward2000 on 2008-02-24
Hi!
For what I can see on the image you are running Screenlets 0.0.10, which need a composite manager (I know E17 has one, but I haven't tried E17, so I can't help you with that).
Perhaps you could upgrade to the new 0.0.12 version, which doesn't need compositing. You can get it here: [http://gnome-look.org/content/show.php/Screenlets+0.0.12+final+rev+174?content=73346]("http://gnome-look.org/content/show.php/Screenlets+0.0.12+final+rev+174?content=73346")
The repositories have changed because the project is now developed by a different person, and the original developer can't be contacted.
Cheers!
