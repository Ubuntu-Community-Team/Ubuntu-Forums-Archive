Yes, you will loose metacity and you will loose compiz, enlightenment is a window manager so will replace other window managers.
It is very cool and worth the try.
You won't loose you standard software, a window- or desktopmanager will only add programs, not delete them.

### Post by kellemes on 2009-11-11
> **collinge said:**
> Can it be used without replacing gnome .. like login in to enlightenment at the gdm...is the ubuntu software center still there??

You cannot use enlightenment together with another windowmanager (including metacity and compiz), this because it's a window-manager itself.
Obviously you can choose to run your own loginmanager (like GDM or one of the many others).

You can also have enlightenment morph with Gnome like [OpenGEU]("http://opengeu.intilinux.com/Home.html") implements it, very very pretty.
Surely there are a hole lot of howto's to be found.

### Post by UltimatePisman on 2009-11-11
Install it, log out and choose for the Enlightment Session and there you have it.
Wanting back to Gnome?
Just choose for the Gnome Session while logging in.
Nothing will been lost, only the configuration you've made in Gnome will not work in Enlightment.
In Enlightment there is no Ubuntu Softwarecentre, but something else.
