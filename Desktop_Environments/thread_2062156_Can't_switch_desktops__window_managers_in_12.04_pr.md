### Post by grahammechanical on 2012-09-26
The last User Interface chosen becomes the default. Just click on the Ubuntu icon on the password panel at the log in screen. By the way, there was information about this months and months ago when 12.04 first came out.

I have fvwm-crystal installed on the 12.10 versions that I am using. Clicking that Ubuntu icon at the password panel brings the menu up for me. It worked when I tested out other desktops because Unity 2D is removed in 12.10 and I wanted a fall-back mode. I have settled on fvwm-crystal.

for those using pre-2012 versions of Ubuntu and thinking about upgrading to 12.04, then expect massive design changes all the way to 14.04.
