### Post by gatorbrit on 2006-06-19
You can use parallels or VMWARE, which is basically the same thing, but free.  Speed depends entirely on your box - how much memory and the processor speed.  
I have managed to get VMWare up and running with WIN 2K, and it works fast and fine, but I have a pretty fast machine. 

Getting accross OS access for files is a bit more of a challenge and requires Samba.
