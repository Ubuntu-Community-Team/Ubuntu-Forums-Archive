### Post by rrwo on 2010-02-25
Since upgrading to Karmic, the behavior of copy & paste has changed.  Now, whenever I select text, it is automatically copied.

I don't want this behavior. I am unable to overwrite text by selecting it and pasting in place.

Instead, I have to go to select the text that I want to delete, and then paste over the text.  If the text is selected in the clipboard already (for multiple pastes), then I have to use the mouse to select it in a clipboard application rather than use the keyboard to simply paste over it.

How do I change this behavior?

---

### Post by Hero of Time on 2010-02-25
Check if clipman is installed. If it is, run it. It should ask if you want to run it on logon, accept that. A new icon will appear in the tray. Right click, select Properties and clear the checkmark for 'sync selections'. That should now prevent your clipboard to be filled with selections.
