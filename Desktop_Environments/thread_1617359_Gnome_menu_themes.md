### Post by thomaswei on 2010-11-09
Try the Appearence button in the settings menu, where you can choose your themes.

### Post by Frogs Hair on 2010-11-09
If the Gnomenu is installed , right click on the start button go into properties on the right side you will see a drop down menu with the installed menu selections . select the install button and navigate to the folder where the packages are stored , select and open  them . On the left side there is a section for button installation. Select the new menu and button from lists and select ok.
You may have to restart the menu a couple of times.
