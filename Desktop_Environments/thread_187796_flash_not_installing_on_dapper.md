### Post by Zerocool10482 on 2006-06-03
Try downloading the flash from Adobe.

1. download the file to the desktop from Adobe.

[http://www.adobe.com/shockwave/download/download.cgi?P1_Prod_Version=ShockwaveFlash&promoid=BIOW](http://www.adobe.com/shockwave/download/download.cgi?P1_Prod_Version=ShockwaveFlash&promoid=BIOW).

2. Extract the file to on the desktop.

3. Right click on the flashplayer-installer file and go to properties.

4. Click on the permissions tab and check the owner: write box.

5. Close the properties box then double click the flashplayer-installer file.

6. It will say ''Do you want to run "flashplayer-installer", or display its contents?''

7. Click on the Run in Terminal button.

8. Read what is says in the terminal. (remeber to close firefox. You can copy this in the text editor.

9. After it's installed. Test it. Good luck.
