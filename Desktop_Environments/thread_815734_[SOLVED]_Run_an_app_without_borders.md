### Post by mcduck on 2008-06-02
Yes, at least to some level..

If you use Compiz (the Desktop Effecs-thing) install compizconfig-settings-manager, and it should be possible to use that to disable window borders from your program.

Otherwise, if you wan tthe app to run full-screen without borders you can configure a keyboard shortcut for fullscreen in either Gnme's own keyboard shortcut settings, or in Compiz settings, depending on if you use desktop effects or not. After that pressing the shortcut shoud make (almost) any program to run full screen without any title bar or window borders.

### Post by mali2297 on 2008-06-02
I don't know to what extent Xfwm4 (Xfce's window manager) supports this. 

If you permanently want to live without borders, you could change to a minimal theme like [this one]("http://xfce-look.org/content/show.php/Minimalistic?content=26432"). 

However, if you want to be able toggle the borders on/off, then you might have to switch from Xfwm4 to another window manager. For example, I think Openbox has this feature. (You can run Openbox inside Xfce.)
