### Post by gjoellee on 2008-08-10
lets do it easy with no commands: First install startup manager by clicking on the link below:
[apt://startupmanager](apt://startupmanager)
(you may need to wait up to 10seconds before the download starts)


when installed go to System->Administration->StartUp Manager go to the Appearance tab and find "Usplash theme" change it to "usplash-theme-ubuntu" and click on close.

NOTE: The little windows that appears after and before you load the program must not be closed, it will close automatically!
