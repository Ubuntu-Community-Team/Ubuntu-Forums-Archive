I installed some IBM development tools on my Ubuntu 8.10 machine a few weeks ago and I just noticed that my Application\System Tools menu contains following launch programs:

- IBM Installation Manager
- Uninstall IBM Installation Manager

Everything else that used to be there under Application\System Tools is gone :( 


Is there some sort of way I can rebuild the System Tools menu to contain whatever was there (defaults) ?   I can delete the IBM options currently contained there because they are also contained in another area on the Application menu bar. 

I currently have: GNOME: Version: 2.24.1

---

### Post by COMTIBoy on 2009-02-05
I sent this out to quickly.

I found the problem by going to :

System\Preferences\Main Menu

then navigated to the System Tools option.  Under that screen I saw the items (that were previously listed) were unchecked.  I simply rechecked them and things are back to normal.  Not sure what the IBM tools installation did...but no harm.  Everything is all good.
