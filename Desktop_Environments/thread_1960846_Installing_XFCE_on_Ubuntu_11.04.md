### Post by 3Miro on 2012-04-18
If you just install the xfce4 package, it will only install the file manager (as well as panels and WM that you need anyway). xfce4 comes with the bare minimum.

Use Synaptic Package Manager to see which packages will be installed if you select xfce4.
