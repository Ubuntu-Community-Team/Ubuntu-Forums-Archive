### Post by timetunnel on 2006-10-07
Under Linux, file browsers belong to a *desktop environment* like GNOME or KDE, and not to a *distribution* like SUSE, Mandriva or ubuntu. This means that if you have SUSE or ubuntu or Mandriva is not relevant. Using any of these distros with KDE will give you the same file manager: Konqueror. Or, like already said before, Nautilus for GNOME.
