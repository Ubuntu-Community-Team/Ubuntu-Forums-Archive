### Post by 3Miro on 2012-07-27
In Unity 3D, you either install CompizConfig-Settings-Manager (available in the Software Center) or you can install Ubuntu Tweak ([http://ubuntu-tweak.com/](http://ubuntu-tweak.com/)). Both of those are powerful tools that will let you all kinds of adjustments to your system, including set the number of workspaces.

### Post by deadflowr on 2012-07-27
Install Myunity, go to desktop and change v desktop and/or h desktop which will expand or contract the workspaces you have.

Easy way to install
In a terminal:
```
sudo apt-get install myunity
```
