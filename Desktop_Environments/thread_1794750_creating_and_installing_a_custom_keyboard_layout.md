### Post by robertmf on 2011-07-02
> **pswin said:**
> How to create and install a custom keyboard layout?

[PREFERENCES -> KEYBOARD -> LAYOUTS]

If you want to modify one of these kbd templates, the files are in
/usr/share/X11/xkb/symbols

Then, you can chose a hotkey from [KEYBOARD SHORTCUTS] 

For example, I use Russian Phonetic layout as an alternate kbd with several key swaps made in the symbols/ru file
