### Post by Waappu on 2007-02-01
Hi

Try this:

- Open Beryl setting manager
- Go to Window management tab
- uncheck plugin Grup and tab windows.

If you like this plugin is enabled then:
- select Grup and tab windows plugin "tabbing" tab
- go to misc. options
- uncheck Autotab windows on creation

That fix it for me

### Post by jsully1 on 2007-02-01
Under Window Management > Move Window > Misc Options, make sure you have Constrain Y to top of screen checked.

Also keep in mind that you can hold Alt and click the window anywhere to drag it out from behind the menu.  Not a permanent fix, but should help you!
