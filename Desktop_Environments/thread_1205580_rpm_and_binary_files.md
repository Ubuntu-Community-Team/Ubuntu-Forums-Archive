### Post by ibutho on 2009-07-06
RPMs are packages created for distros that use the RPM Package Manager (formerly Red Hat Package Manager). These distros include Fedora (and therefore Red Hat and derivatives), Mandriva, PCLinuxOS, openSUSE and others.
