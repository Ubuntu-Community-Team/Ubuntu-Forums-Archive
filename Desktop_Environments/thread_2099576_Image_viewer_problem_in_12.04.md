### Post by grahammechanical on 2012-12-30
You may, if you wish, report this as a bug but is it a bug in Image Viewer or Unity.

A way to work around this is to spread the four workplaces and then drag the application window partway into the lower workspace. Then expand that lower workspace and now you can adjust the application window size. And then push it back into the first workspace.

Regards.
