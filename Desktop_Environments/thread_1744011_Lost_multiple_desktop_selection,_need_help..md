### Post by Krytarik on 2011-04-30
If you indeed removed the entire bottom panel, you need first to right-click at the remaining panel (if any) and choose "New Panel" if you haven't done so already, then
- right-click at the desired panel
- choose "Add to Panel"
- select "Workspace Switcher" in the upcoming list
- click at "Add"
- if you haven't done so already, do the same for "Window List" and "Trash"

Greetings.
