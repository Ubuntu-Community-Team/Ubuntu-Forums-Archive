Yes if you have xfce, xubuntu, or are using themes with gtk2-engines-xfce

I haven't tried to customize any other theme engine because I am new to Linux and xubuntu was my first install and I was so happy with it I still haven't tried anything else.

I also just theme with the xfce-engines because I thought it would work better with xubuntu, and also because it was written so easy that I was able to customize it myself without any tutorials.

So I went ahead and made a cheat sheet and a simple tutorial for anyone, and if you would like to try and change your over all theme and panel colors that way, I'll link you.

The first link is to someone else's post where I explained how to start, by creating a modified version of a theme you already have installed: [http://ubuntuforums.org/showthread.php?t=371110](http://ubuntuforums.org/showthread.php?t=371110)

The second link is to my "How To" for gtkrc files for the xfce-engine: [http://ubuntuforums.org/showthread.php?t=377397](http://ubuntuforums.org/showthread.php?t=377397)

There is a cheat sheet of where the colors go for browser color, panel color, and font color for you whole theme. And also places to change the buttons, and scrollbars.

This all from a beginner so it's not totally complete, but it does the trick for my themes.
