### Post by tbresson on 2005-11-17
First of all - I'm a big fan of Ubuntu!

I decided to totally remove my 5.04 and install the new Breezy, the new interface is cool and everything seemed a little more tweaked. I was totally excited!

I saw some new things, e.g. the disk information tool, the Open Office 2.0 and so forth it seemed you guys took into account some of the criticism the commuity have talked about, you improved a lot of stuff we can't even see (I espect, hehe - probably some WiFi I would think) and you removed some stuff.

I couldn't find the root terminal - I guess that's ok, no biggie - it isn't that hard to write sudo infront of a command :) but then I ran into the Add/remove programs, which is only called "add programs" now. A bit confusing since a lot of stuff come pre-installed.

Anyways, as always I decided to clean out the things I don't use, but hey! that's not possible anymore with just unchecking the stuff you don't want (due to dependencies). You have to go into advanved mode and search for all the components and dependencies (as far as I could tell) - I was afraid to touch anything, I started thinking about if this distro just made it worse for the crowd they are already producing this great OS for.

I decided to let it go and leave the stuff there and got on the net to find my beloved Azureus - who doesn't use this lovely program today - getting distro's of the net with bittorrent this way is simply too smart. But there where problems. I did install the newest Sun java like on 5.04 but something was wrong and only a few people have actually posted about it on the net. But it seems there's some issues with the Open Office settings and Azureus - both "programs" seems to need some Java path/link/thingy.

Of course some linux users who's a bit more into the Linux world have already posted a few command lines that would solve the problem but I'm a simple user and want things to work without too much hasle.

I mean I'm new to linux, I've tried a lot of distros and I managed to get my samba running, installing Sun java, Azureus, partioned my harddrives from NTFS to ReiserFS, made auto-mounting of my harddrives, burned some CD's and other cool stuff from my Ubuntu only with the help of the forums here and trying things out.

But it all seemed a bit to complicated with the new version... so sadly I must admit I fdisk'ed my harddrive and went back to 5.04 and said to myself, I guess 6.04 will be better.... but will it?

In all fairness - there's a lot of work behind every release, I have no doubt about it, and there's not hard feelings, and no issues from my point of view to would speak badly about the work being done here, AND I barely touched the surface of the OS and already am I complaining because it doesn't fit MY needs. So I'm sorry if this post is taken the wrong way - I'm only saying I'm a newbie in this world, I like what you did, but I have problems with your new release and it seems you made it better some places, but a lot worse other places which was fine before.
