### Post by lfridkis on 2008-01-28
Is there a way to move between windows and between desktops without using the mouse? This would be similar to using ctrl pg up to move between tabs.

---

### Post by jpeddicord on 2008-01-28
Alt+Tab will switch between windows on the current desktop. Use Ctrl+Alt+[Direction] to switch between desktops. For example, to move to the desktop to the right, hit Ctrl+Alt+Right. To bring the current with you to the next desktop, hold down shift in addition to the other keys.
