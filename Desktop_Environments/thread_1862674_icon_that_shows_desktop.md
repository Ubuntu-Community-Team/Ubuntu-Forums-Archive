As it's definitely in the [official repos]("http://packages.ubuntu.com/search?suite=default&section=all&arch=any&searchon=names&keywords=wmctrl") of all current Ubuntu versions, just make sure that the second option, "universe", is enabled in "Software Sources -> Ubuntu Sources". If you happen to have to enable that option there, make sure to choose "Reload" upon closing the window, then try again to install it.
