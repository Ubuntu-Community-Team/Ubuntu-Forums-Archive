### Post by slux on 2005-11-13
Because of freedesktop's standardization efforts they share the same desktop dir, have the same structure for desktop icon/menu item files and a single standard for representing the menu structure etc. Eventually GNOME/KDE programs should be able to unteract much better than they currently do. It used not to be this way. I'm surprised someone wants to return to the bad old days... :P

That said, I'm pretty sure there's a way to change the Desktop directory from the default for either one but I've never done it so I don't really know. Maybe someone else or google does. :P
