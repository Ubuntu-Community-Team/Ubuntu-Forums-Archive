### Post by igsen on 2009-11-11
I'm a Xubuntu user and trying Ubuntu for the first time. I have installed ubuntu-desktop, so far no problem except for the splashscreen (not sure if it's the right term). Why is it that during the boot up process it keeps on flashing "Xubuntu"? How do I turn it into "Ubuntu"?

### Post by jeb800e on 2009-11-11
Because you are trying to run Ubuntu on top of Xubuntu. Think of Ubuntu, Xubuntu, and Kubuntu as parallel movements in the Linux world. They all are now organized by the same company, have the same updates, etc., but when you try to copy one of the parallel movements to it and remove the original, you will likely run into problems, as some things aren't set up EXACTLY the same as the other.

### Post by alienclone on 2009-11-12
if you want to keep xubuntu but still change the splash back to the default ubuntu, then go to System>Administration>StartUp-Manager  and in the appearance tab change the Usplash theme to usplash-theme-ubuntu.

if you dont have the startup manager you can get it from the repos. there are some other nice tools in it.
