### Post by randomnote1 on 2008-02-04
Unfortunately Gnome doesn't manage the state of windows at all.  It has to do with the back end being metacity.   However, there is hope.  There's a program called devils pie that is supposed to allow you to accomplish the kind of window management that you're looking for in Gnome.  Just search for devils pie on Google and it comes up at the first result.
