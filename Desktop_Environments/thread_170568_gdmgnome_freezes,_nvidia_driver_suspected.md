### Post by Rasitiln on 2006-05-04
Yes the NV driver is bad. If you installed the nvidia glx drivers change the driver line to "nvidia" not "nv". If you haven't installed the nvidia glx drivers then change it to "vesa" then you can boot into the GUI and install the nvidia Glx drivers. I've described the solution to this in further detail [Here]("http://www.ubuntuforums.org/showthread.php?t=170475") and so did a few other people.

Hope this helps :D
