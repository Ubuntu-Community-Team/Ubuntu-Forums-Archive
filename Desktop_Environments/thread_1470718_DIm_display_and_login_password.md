I just upgraded to Ubuntu 10.04 and I see the display dims after a while and finally it asks for password to show my desktop.

### Post by Hambombj on 2010-05-03
Go to, system,preferences,screensaver.
Uncheck "Lock screen when screen saver is active, and you can alter the time that it comes in using the slider :D
