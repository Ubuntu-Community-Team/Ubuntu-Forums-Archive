### Post by dratfect on 2008-08-16
I am a linux newbie. I want to change the icon of '.tex' files in Ubuntu 8.04. They, by default, seem like a generic text file. I searched the Internet and found the similar question about the 'pdf' file icons. The solution for 'pdf' was to change the file 'gnome-mime-application-pdf.svg' in '/usr/share/icons/gnome/mimetypes/scalable/'.

I replaced that file with my own svg file and it worked fine. However, for 'tex' files, I did the same with the file 'gnome-mime-application-x-tex.svg' in the same location. Surprisingly, as a result, the icons for '.dvi' and '.ps' files changed, while 'text' file icons remained unchanged!

I even deleted that icon file, but the problem is persisting.

Any help is appreciated! Thanks a lot in advance.
