### Post by 23meg on 2007-01-04
> By this I mean the borders of windows seem to be anti-aliased whereas mine are rather pixellated, and I can't seem to have basic window shadows.Window borders and title bars can be styled with Metacity themes if you're using Metacity, the default window manager of Ubuntu. For shadows you need a compositing manager such as Compiz, Beryl or xcompmgr. 

Try xcompmgr if Beryl and Compiz are too much for you. Keep in mind that it's not very stable.
