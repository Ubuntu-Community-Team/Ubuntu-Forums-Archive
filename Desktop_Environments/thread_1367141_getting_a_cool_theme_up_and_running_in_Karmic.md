hello there.... 
1. update your repository
2. From the terminal sudo apt-get install compiz
3. again sudo apt-get install emerald
4. also download conky its cool the same way from the terminal or from package manger

Now go to gnome-look.org and download some cool themes u would like.....

appl those themes by simply dragging the file into the themes window u get from the following action System>Preferences>Appearance. this goes the same for icons too..
Note that the file should have the format .tar.gz 

while u r there in gnome-looks download some emerald themes too...

5. choose System>Preferences>Emerald theme manager and import the theme you hav downloaded ....

6. System>Preferences>Compiz Config Settings manger ..i bet you'll have a hunch what to do next.. go wild... but enable Windows decorations...

7. Now System>Preferences>Startup Applications in that add conky as name and in command box type conky

8. System>Preferences>Startup Applications in that add emerald as name and in command add emerald --replace

9. thats it folks conky is like a sidebar which gives u a pro look... now there are docks out there ...

Also just to let u guys know if u highlihjt something with ur leftmouse bttn it means u copied that text ..wanna see that well bring up text editor and click the wheel or both mouse buttons at the same time (this emulates the middle mouse button)..

Hav fun and happy holidays
