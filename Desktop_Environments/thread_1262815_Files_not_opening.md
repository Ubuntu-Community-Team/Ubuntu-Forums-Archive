### Post by hellblazer on 2009-10-14
solved my problem, no thnaks to you lot ;)

I changed my menu entries by adding ```
bash -c
``` to the front of the command so that the program opens on the correct X-Screen. This however caused files not to open when double clicked as it appears that Ubuntu uses the menu commands to do this.
