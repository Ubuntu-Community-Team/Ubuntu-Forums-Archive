### Post by 5-HT on 2005-12-05
Hi, from what I understand this seems to be a question on mounting a floppy disk, but correct me if I'm wrong...

Have you ever been able to mount a floppy in breezy?

I know that for some reason or another, there is an issue with breezy and pmount when trying to mount a floppy from the GUI.

I find the easiest way to get around this is simply to mount the floppy from a terminal with: 
```
 mount /media/floppy0 
``` 
and to unmount...

```
umount /media/floppy0 
```

HTH
