### Post by renzokuken on 2007-03-09
they were always there, but werent shown by default before, but for some reason, nautilus is showing up these hidden folders now. (i'm assuming you are using gnome here)

a folder that begins with a '.' is a hidden folder, they contain config settings and other things for various programs.

to toggle "Show hidden files/folders" on or off use [color=red]Ctrl+H[/color] when in nautilus (nautilus is the files browser that opens when you click on Home)
