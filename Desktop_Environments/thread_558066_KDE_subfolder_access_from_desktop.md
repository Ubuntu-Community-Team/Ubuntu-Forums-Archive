On my desktop, I have a symbolic link to a folder I use a lot. I use the main folder but often want to go directly several sublevels down. 

Is there a way to set up KDE so I can access the subfolders the same way I get to Firefox bookmarks that are in a folder on the toolbar? That is, a sustained left click while mousing down to the item I want?

I know I could do symbolic links to all the subfolders, but that's less elegant than this would be.

TIA

### Post by benerivo on 2007-09-23
Could you add an entry for your main folder in to the main menu, using the KDE menu editor. Then add sub folders to this menu item for each location that you are interested in, so that the menu sub folders match the structure of your own folders.

### Post by benerivo on 2007-09-24
> Can I make items from the menu into icons on the desktop?

Well you can, but they only go on as individual icons (ie. Firefox), and not as menus (ie. the Internet submenu)

The closest I can get, is to use the menu idea, but using a separate panel that is configured so that it ends up as a small square in the top left of the desktop. In my screenshot, this panel has had just the utilities section of the main menu added to it, but I think you should be able to add any of the submenus, such as your newly created folder directory section.

Someone who knows kde better than I do may have a better way.

### Post by esekyavuz on 2007-09-25
Found another possible solution.

Under Configure Desktop/Behavior you can set a mouse button to give the bookmarks menu. So I might set up my folders as bookmarks and get to them with a click on the disktop.
