### Post by CatKiller on 2019-03-11
Ctrl-F12 isn't Minimise All, it's Show Desktop, which is different. I believe you can add a Minimise All button, though, somewhere in the Settings.

Coming from Compiz, I've never used any of these kinds of window management. Having everything in one desktop feels really cramped. I just flip round to an empty virtual desktop if I want something from the desktop, and Present Windows (which was Scale Windows in Compiz, and is Exposé in OS X) for switching between open applications.

### Post by SeijiSensei on 2019-03-12
You can add the "Minimize Windows" widget.  I have one in my panel.  Click on the settings button at the right-hand end of the panel, then choose Add Widgets.  You should see a Minimize Windows choice.

If you prefer you can install the widget on the desktop.
