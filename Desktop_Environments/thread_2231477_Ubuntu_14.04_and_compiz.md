### Post by grumblebum2 on 2014-06-25
Compiz runs fine here. Compiz is the default window manager so you would expect it to run ok in ubuntu/unity.
If you install the **compiz-plugins** package for the cube, don't enable the 3D Windows plugin.
Plugins in this package are not officially supported.
Most still work ok though.
