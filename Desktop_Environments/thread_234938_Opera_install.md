### Post by smeager on 2006-08-12
You can either download it directly from Opera's website at
[http://www.opera.com/download/]("http://www.opera.com/download/") (select ubuntu from the drop-down menu) or you can add a repository to your list and install it through apt-get/synaptic, etc...

```
##Canonical Commerical Repos (Hosted on Canonical servers, not on Ubuntu servers. RealPlayer10, Opera, and more.)
deb http://archive.canonical.com/ubuntu dapper-commercial main
```
