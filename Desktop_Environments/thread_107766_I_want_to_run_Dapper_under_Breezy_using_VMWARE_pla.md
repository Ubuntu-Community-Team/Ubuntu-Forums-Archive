### Post by zoiks on 2005-12-24
vmplayer only opens "virtual machines" which have already been created; it won't create a virtual machine to run.

This thread
[http://www.ubuntuforums.org/showthread.php?t=39513](http://www.ubuntuforums.org/showthread.php?t=39513)

can show you how to create a virtual machine using qemu (a machine emulator).

You could also try installing regular vmware for a 30-day free trial and creating your virtual machine that way.  Hopefully nobody gets on my case for saying that since vmware is very good software (costs $199) and some might think that's unethical if you don't intend to purchase the software.
