### Post by grikdog on 2008-07-02
Maybe a Nautilus issue?  If you drag a file across "File System" (or some other large potential destination) in the vertical left-side panel, then Nautilus immediately grabs cpu time and begins calculating file sizes or something.  The result is, the window freezes and is unusable until the progress dialog pops up in a few seconds (or minutes), allowing you to cancel the unwanted operation.  Even so, you get a Retry? dialog.

In practice, you learn to drag AROUND the side panel (but you always forget).  <RANT>In userworld, such behavior is BONED.  Not even Windoze does this, although I think Mac HFS 8 or 9 used to ... but OSX, never.</RANT>
