### Post by bio_amateur on 2011-01-25
I have a system preinstalled with Ubuntu 10.04 server. Now I want to enable GUI, so I installed the following packages.
[I]sudo apt-get install xorg gnome-core gdm gnome-applets gnome-system-tools gnome-utils ubuntu-artwork compiz-gnome firefox sysv-rc-conf
[/I]
The system is equipped with a Matrox graphics card and a Fermi card. However, after installing those packages, it always detect the Matrox, even when I only plug the monitor into Fermi. 

When I open nvidia-settings, nothing show up. When I open Administration/Hardware settings, the green light is enabled with a nvidia driver (latest driver 270.x). 

[http://pkgs.org/ubuntu-10.04/ubuntu-x-i386/nvidia-settings_270.18-0ubuntu1~lucid~xup1_i386.deb.html](http://pkgs.org/ubuntu-10.04/ubuntu-x-i386/nvidia-settings_270.18-0ubuntu1~lucid~xup1_i386.deb.html)
