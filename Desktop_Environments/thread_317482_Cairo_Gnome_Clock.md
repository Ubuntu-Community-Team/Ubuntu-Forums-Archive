### Post by mcduck on 2006-12-13
It's not an application, just a theme for MacSlow's Cairo Clock.

Get the real app here: [http://macslow.thepimp.net/?page_id=23]("http://macslow.thepimp.net/?page_id=23")
(Dapper's packages work just fine in Edgy too, but to get the clock to run without ugly box around it you need to run some compositing manager such as Compiz or Beryl)
