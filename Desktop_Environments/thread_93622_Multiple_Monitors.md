### Post by LittleReinhart on 2005-11-22
Hey,

Multiple monitors works just great. But I don't know if I understand you question completely.... Do you want to make dual-screen work? Or do you want virtual desktops?

For multiple monitors, you have to activate the Xinerama option in your xorg.conf (found in /etc/X11/xorg.conf).
You also have to define two grafical devices (even if you have onely one grafical card with two heads), two monitors and two screen definitions.

**EDIT: Problem solved!**

Answer: add the following line to the "Server Layout" section:
```
Option		"Xinerama" "On"
```
So your "Server Layout" will look something like:
```
Section "ServerLayout"
	Identifier	"Default Layout"
	Screen		"LeftScreen"
	Screen		"RightScreen" RightOf "LeftScreen"
	Option		"Xinerama" "On"
	InputDevice	"Generic Keyboard"
	InputDevice	"Configured Mouse"
EndSection
```
