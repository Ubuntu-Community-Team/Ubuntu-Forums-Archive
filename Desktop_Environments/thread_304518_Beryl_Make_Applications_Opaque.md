### Post by Peepsalot on 2006-11-21
The settings that affect this are in the "Trail Focus" plugin in beryl settings manager.  One option is disabling the entire plugin by deselecting the checkbox for it in the left menu.

There are also some options in the plugin to configure what types of windows it works on, which you could try playing with.

Edit:
Oh, and there is a "String Lists" tab under the settings here that is supposed to allow you to exclude certain windows from this effect.  But, I can't seem to get it to work right.:-?
