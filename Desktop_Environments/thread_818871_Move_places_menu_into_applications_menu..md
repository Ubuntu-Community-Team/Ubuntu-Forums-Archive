### Post by helino on 2008-06-05
Instead of using the standard Ubuntu menu, you can rightclick on a panel and choose "Add to panel". Then choose "Main menu". This will give you a menu where the Applications, Places and System are submenus in one menu.

This is the only way I know to solve this, perhaps there's another solution

### Post by Totalchaos02 on 2008-06-05
The best way I have found to do it is to go to menu editor and manually create a Places subfolder. Within that sub folder you create a launcher with the command 'thunar /home/*user name*', and then a launcher for each other folder you want quick access to. Takes a bit of time but worked great for me.
