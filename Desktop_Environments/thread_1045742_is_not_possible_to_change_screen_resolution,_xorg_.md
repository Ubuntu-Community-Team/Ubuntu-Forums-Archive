### Post by oaqster on 2009-01-20
per my understanding the list of modes in the xorg for the various sections is usually an ordered list, i.e. the one you want picked up first needs to be placed in the list first as well, making the first valid on in the list to be the one that becomes the default one.

what ubuntu & kernal version are you on.  i believe that the latest version relies on auto detecting your hardware, so make sure that you can probe your hardware and get the correct results.

e.g. run the following commands and check what you get

xrandr -q
sudo ddcprobe
xdpyinfo

also try looking into /var/log/Xorg.0.log to see whats going on at startup to get a better idea of what X is doing with your displays.

- oaqster
