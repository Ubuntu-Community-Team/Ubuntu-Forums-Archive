### Post by fuscia on 2006-06-03
openbox starts out blank (i had the same reaction). right click anywhere on the blank page and a you will get a menu. you can edit the menu using a tool called obmenu - [http://obmenu.sourceforge.net/index.html](http://obmenu.sourceforge.net/index.html)
you will need some kind of program to put up a wallpaper for you. i use feh. open up the terminal and do...

*sudo apt-get install feh*

once it's installed, you can use any pic from your collection as a wallpaper, just do...

*feh --bg-scale picofmomfallingonherbutt.jpg*

...and you'll have a nice background (you can do *man feh* for other options and features).

openbox is awesome, especially for old equipment that can't really handle the starship gnome.
