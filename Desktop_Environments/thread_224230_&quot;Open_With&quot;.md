### Post by Footer on 2006-07-27
When I'm in Konqueror and I right-click on a file and then select "Open With", I get a submenu with the programs I can open this file with (DUH!) along with an "Other" option to find programs that aren't in the list.

So far, so good.  However, I've gone and removed some programs but they still appear in the "Open With" list.  My question is this:  How do I remove these removed programs from the list?

Thanks!

---

### Post by croak77 on 2006-07-27
Alt-F2 type kcontrol. Navigate to KDE Components -> File Associations. All the file types and associated apps are listed there.
