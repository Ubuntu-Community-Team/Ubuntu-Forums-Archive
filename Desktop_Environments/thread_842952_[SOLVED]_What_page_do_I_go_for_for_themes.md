### Post by x1a4 on 2008-06-27
GTK 2.x - themes for applications which use version 2 of the GIMP Toolkit (GTK) to draw widgets (buttons, menus, scroll bars, check boxes, etc.)

GTK 1.x - themes for GTK version 1

Metacity - themes for GNOME's default window manager (window decorations)

Compiz - themes for the Compiz window manager

Beryl - themes for the Beryl window manager

GDM - GNOME Display Manager (GDM) themes--login screen s.

For more themes (GTK and Compiz for instance) point your browser to [http://xfce-look.org/](http://xfce-look.org/),
