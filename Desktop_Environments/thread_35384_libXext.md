[QUOTE=sandman545]what is Kynaptic[/QUOTE]

Actually, even better would be Synaptic :)

Synaptic is the Package Installer, and using it will make installing stuff far, far easier.

Open up a command prompt, and type:

```

sudo apt-get install synaptic

```

When that's finished, click on the "K" menu, go to System, and click Synaptic package manager.  Enter your password when asked, click search, and search for kdegames.  I'm sure you can figure out the rest :) Have fun!
