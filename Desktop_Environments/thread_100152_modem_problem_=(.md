### Post by Bartender on 2005-12-07
muzz - 
Dude, when you say you've got the install CD, you're talking about the CD that came with the modem, right?  Won't do you any good in the Linux world.  The data on that CD is all written for use with Windows.  Maybe someday they'll see the light and start including Linux drivers. 

I've got an Intel 536 also.  I'd love to tell you exactly how I got it to work with Linux but I haven't crossed that bridge.  

Take a look here
[https://wiki.ubuntu.com/IntelFiveThreeSixEPModemHowto?highlight=%28modem%29](https://wiki.ubuntu.com/IntelFiveThreeSixEPModemHowto?highlight=%28modem%29)

This one looked interesting but I just glanced at it cause this guy wrote a LOT of info on his webpage...
[http://www.tldp.org/HOWTO/Modem-HOWTO.html](http://www.tldp.org/HOWTO/Modem-HOWTO.html)

Google "linmodems" for more discussion on the subject.  The bottom line is that most modems rely on software, written for Windows, to make them functional.  As you research this a bit you'll see that hardware-based modems  offer a relatively painless path, but "winmodems" can be either a tough nut or impossible.
