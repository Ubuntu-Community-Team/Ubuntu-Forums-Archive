### Post by Sutekh on 2006-05-25
When you re-configure the X server you should be able to choose which resolutions you wish to make available; by selecting them and activating them with the space bar. 

If at some time in the past the NVIDIA installer changed your maximum resolution, it would have done so through the X server configuration file in the same way in which you would manually do it.

So you should try the reconfigure command again```
sudo dpkg-reconfigure xserver-xorg
```and see if you selected the higher resolutions.

If you are sure that you selected higher resolutions other than 640x480, perhaps you could post your xorg.conf file, to see if there are any problems in it.  

This command prints the contents of the file to screen (note the capital X)
```
cat /etc/X11/xorg.conf
```
