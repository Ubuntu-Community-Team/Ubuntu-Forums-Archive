### Post by cocorocara on 2011-03-17
Ubuntu 10.10 Screenshot of full screen works. Screenshot of "selected  area" works. But Screenshot of current (active) window doesn't. Nothing  happens. The save screenshot window doesn't open.
  

Tried using Alt+Print as well as from the Panel->Accessories->Screenshot. Same result. No screenshot.
  

GIMP window screenshot works. But thats not the point. Why doesn't the normal way work? 
  

Any ideas?

### Post by cocorocara on 2011-03-17
Aha! Got it. There are two reasons for this.

(1) From the Panel menu, after selecting the grab current window option, one needs to increase the delay above 0 seconds (say 3-4) and then CLICK on the window. See: [http://askubuntu.com/questions/30767/ubuntu-10-10-screenshot-of-active-window-doesnt-work/30774#30774](http://askubuntu.com/questions/30767/ubuntu-10-10-screenshot-of-active-window-doesnt-work/30774#30774)

(2) The Alt+Print shortcut conflicts with the kernel Magic SysRq in Ubuntu 10.10. Changing the keyboard shortcut to something else like Ctrl+Shift+Print fixes the problem. See: [http://superuser.com/questions/258735/ubuntu-10-10-screenshot-of-active-window-doesnt-work](http://superuser.com/questions/258735/ubuntu-10-10-screenshot-of-active-window-doesnt-work)

Hope this helps somebody else.
