### Post by Innigo on 2010-09-19
Hi all, 

I'd can't seem to find a preference to turn off gedit's default behaviour of opening a new file in a new tab. I'd like it to open in a new window, so I can view two pages of code side by side. How can I convince gedit to open a new instance?

---

### Post by qamelian on 2010-09-19
I don't see any obvious way to set this behaviour, but you can get what you're looking for by letting Gedit open the second document in a new tab, then click on the second tab and drag it out of Gedit to create a second instance of the editor UI.
