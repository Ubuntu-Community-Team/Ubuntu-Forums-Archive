### Post by Metheos on 2007-07-30
I'm not sure, but in my conf, metamodes are separated by commas for the two monitors, then semicolons for alternate modes. Here's my device section:
```
Section "Device"

	Identifier  "AGP1"

	Driver      "nvidia"

	BoardName   "NVIDIA GeForce 7800 GS"

	BusID       "PCI:3:0:0"

	Screen 0

	Option		"ConnectedMonitor" "Center"

	Option		"EnablePageFlip" "on"

	Option		"RenderAccel" "true"

	Option		"AGPGART" "3" #try AGPGART and then NVAGP
	Option		"AddARGBVisuals"	"True"
	Option		"AddARGBGLXVisuals"	"True"
	Option		"NoLogo"	"True"
	Option		"TwinView"	"yes"
	Option		"MetaModes"	"1280x1024,1152x864;1280x1024,1280x1024"
	Option		"TwinViewOrientation"	"CRT-0 RightOf CRT-1"
EndSection
```
So, yours should be A: In the device section? and B: "1600x1200,1600x1200"
