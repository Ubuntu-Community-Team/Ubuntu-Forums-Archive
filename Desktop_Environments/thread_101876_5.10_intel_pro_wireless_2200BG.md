### Post by emperor on 2005-12-12
Looks like it's loading OK

The firmware should be in "/lib/hotplug/firmware" directory.
Look for files starting with "ipw-2.3 ". These are the firmware files for the 1.06 ipw2200 driver.

It appears  that the firmware may indeed be automaticly installed with the kernel now. If not, you can get the firmware here: [http://ipw2200.sourceforge.net/firmware.php](http://ipw2200.sourceforge.net/firmware.php)

### Post by emperor on 2005-12-13
Is the gateway address set?

You can turn off ipv6, by editing /etc/modprobe.d/aliases and setting net-pf-10 to "off".

see:
[http://www.ubuntuforums.org/showthread.php?t=97221&highlight=net-pf-10]("http://www.ubuntuforums.org/showthread.php?t=97221&highlight=net-pf-10")

You may need to add additional  wireless settings to: /etc/network/interfaces
The gui network setup screen does not allow setting the channel number and other options.

I would fiddle with the setting using "iwconfig" manually and get it working here first.

There is some info in here that will help:
/wiki.ubuntu.com/WirelessTroubleshootingGuide?highlight=%28ipw2200%29
