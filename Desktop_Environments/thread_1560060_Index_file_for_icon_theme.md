### Post by Frogs Hair on 2010-08-24
If it is tar.gz package you can open appearance preferences and just drag and drop the package and it will install . If it is a zip file you will need to extract  the tar.gz first. 

Some icon sets  come with multiple color options and the scripts and instructions can be found by extracting them from the tar.gz  To extract right click on  the package an select extract here. I usually do this in downloads , but you can drag the package to another folder if you wish.

If you got the theme at Gnome  look.org  or Ubuntu art, sometimes the maker will supply instructions on the page you found the icons in comments.
