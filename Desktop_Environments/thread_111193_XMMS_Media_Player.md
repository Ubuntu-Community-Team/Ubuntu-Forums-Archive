### Post by CarlosinFL on 2006-01-01
I have XMMS installed for a audio player but when I listen to MP3's, they are opened in MPlayer via Firefox plugin. Can someone please tell me how I can tell Ubuntu I want to play all MP3's via XMMS?

---

### Post by Ampersand on 2006-01-01
Open Nautilus, and go to the folder containing your music collection. Right click on an mp3, and go to 'Properties'. Under the 'Open With' tab, you can select the default program for that particular file type.
