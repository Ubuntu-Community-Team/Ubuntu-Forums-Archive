### Post by wallijonn on 2006-05-15
It could be anything: bad memory, video card not supported, SATA chipset not supported, internet keyboard (with all the extra buttons), sound, USB, NIC, modem, wireless, 1394 not supported, etc. 

Remove any USB devices before trying to install, along with removing any WinTel modems.
