### Post by Xian on 2005-10-22
[QUOTE=floyd27]I assume i have to log on as root in the GUi but i dont know how?[/QUOTE]
Just open a terminal and use the sudo command w/Nautilus.
That will give you admin privileges in that file manager.

```
$ sudo nautilus
```
