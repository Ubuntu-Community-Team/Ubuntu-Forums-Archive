### Post by deserthowler on 2007-11-16
I've only used this on my Debian box.  It's TWM.  I use FVWM on my Ubuntu laptop.

```
sudo apt-get install menu menu-xdg twm
sudo update-menus
```

Now open twm and click on the screen.  There will be a Debian menu.  Just cruise it and click on what you want to open.  You can't get much lighter than this.  Easy too.
