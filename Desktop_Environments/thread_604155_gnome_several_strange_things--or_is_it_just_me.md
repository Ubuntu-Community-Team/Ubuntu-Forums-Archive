### Post by dgermann on 2007-11-05
Hi--

These quirky things all seem to point to a problem with gnome, perhaps even a bug. Has anybody else seen them or solved them?

1. (This is the one that bothers me most.) In OOo, when I go to save a file, about a third of the time I will get a message box popping up that says "Could not create backup." I hit escape and the same screen comes up a second time and hit escape. (Always 2 times.) Then I hit save again and the save works--no error messages.

If I shut off always create backup (such as I have to do if I am saving in Word format), then the error message box only appears once. I get this on all my Linux boxes (Ubuntu 6.06), but not on my one WinXP box.

What it seems to me is going on is that the computer is tripping over its own feet: it cannot open the backup file fast enough; but when it has once found it, it knows where to look the next time I call for a save and it is ready!

2. Opening OOo files from Nautilus locks the computer.

3. Opening a file from Nautilus (does not seem to matter what kind of file, jpg, pdf, odt, doc) about one in 10 times it will take 3 to 7 minutes to open the file (it pops up a screen that says it is opening the file, but if you want to cancel, click here). If instead I go to "search for files" and find the file with that, I can click on the file I want and it opens instantly.

4. (This has become a major aggravation the last few days:) Inadvertent double clicks, drags, or file selections with the mouse. I can click on the panel at the top of my screen and then move to another part of the screen and find that I have dragged the panel with me--it ends up at the side or bottom. Or, I can be in open file in OOo and click to get inside the scroll down list of files, and the file I clicked on opens (it should take a double click to open it). Or I am in evolution and click on delete for a message and two get deleted.

In short, it just seems to read single clicks as double clicks or as if I am holding the mouse button down.

I have tried system > preferences > mouse, but the only thing I see that seems close is the double click timeout. I have tried setting it at both extremes and that does not seem to make much if any difference.

My system:
Ubuntu Dapper 6.06
Dual screen using Twinview
Production environment
Samba
Recently installed LAMP server
Mostly use OOo and Evolution

Thanks!
