### Post by banjo man on 2009-02-22
depends on what os they are running...

if they are running windows, there should be built-in remote desktop features...so they would need to configure their desktops for you to connect to them...(google:"windows remote desktop")


if they are running ubuntu, (or other *nix distros) you could try "NX" (google it)...it's a nice setup and I use it for multiple headless servers of my own...


to know how to connect to them, you'll need to know their IP addresses, and if they have routers/firewall (like wifi switches, etc) they will need to forward certain ports for you to be able to connect)



as for the weather I don't know what the deal is, but I use another little built in weather thing...

on the top bar, right-click and add, then find the weather report...
