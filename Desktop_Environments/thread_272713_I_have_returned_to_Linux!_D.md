### Post by strabes on 2006-10-07
welcome back! 

there are some great OS X XGL/Compiz themes, although you don't really need XGL/Compiz to make your desktop look like OS X. Mine looks pretty much exactly like OS X except for the menu bar on the top panel - I don't like that feature of OS X. Here are a couple guides that can guide you:

[http://www.taimila.com/ubuntuosx.php](http://www.taimila.com/ubuntuosx.php)
[http://www.supriyadisw.net/2006/09/ultimate-ubuntu-dapper-look-like-osx](http://www.supriyadisw.net/2006/09/ultimate-ubuntu-dapper-look-like-osx)

One thing that these guides don't seem to work for dapper is changing the MENU BAR icon. If you use the main menu, then ignore this. Anyway once you install the OS X iconset, you have to do this: make an icon called distributor-logo.svg, and put it into /usr/share/icons/Tango/scalable/places. You can use OOO Draw to make svg's from png's btw.

If you figure out how to change the network-manager icons, tell me how!

Hope this helps.
