### Post by Icarosaurus on 2007-04-04
try using modelines:
[http://www.bohne-lang.de/spec/linux/modeline/]("http://www.bohne-lang.de/spec/linux/modeline/")
or some other modeline generator (you can easily find them by google).
I had problems using modelines with my ati fglrx...I hope nvidia has no problem...

you should put the code generated in your monitor section, just like this:
```

Section "Monitor"
	Identifier   "F700P"
        HorizSync    86.0 - 98.0
        VertRefresh  50.0 - 160.0	
	Option	    "VendorName" "LG"
	Option	    "ModelName" "LG F700P"
	Option	    "DPMS" "true"
       [B]ModeLine     "1024x768@100" 124.8 1024 1088 1240 1528 768 768 771 817
	ModeLine     "1152x864@100" 167.7 1152 1240 1448 1824 864 864 868 919[/B]
EndSection
```

Hope this helps you!

### Post by zero-Q on 2007-04-05
thx for the replies and i solved the problem with your hands. i changed xorg.conf:
[PHP]Section "Monitor"
    Identifier     "SyncMaster"
    Option         "DPMS"
    HorizSync      30-96
    VertRefresh    50-160
    Modeline       "1024x768"    113.31   1024 1096 1208 1392   768  769  772  814  -HSync +Vsync
    Modeline       "1152x864"    143.47   1152 1232 1360 1568   864  865  868  915  -HSync +Vsync
EndSection[/PHP]

and now i can use 100 mhz
