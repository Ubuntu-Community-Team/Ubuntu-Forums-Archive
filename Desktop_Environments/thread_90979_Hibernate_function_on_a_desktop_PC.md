### Post by chele on 2005-11-16
All modern PC's can use hibernate and/or suspend, whether they are desktops, towers, or laptops.

What you described happening sounds like hibernate (suspend to disk). the same thing happens on my laptop, though I did not need to re-install anything, it did not break anything. It just seemed a bit slow and klunky, but everything worked normally on resume.

I much prefer Suspend, (suspend to ram) as it is much quicker and smoother, when it works. Right now it doesn't on this laptop. It did when I had an iBook running Debian.
