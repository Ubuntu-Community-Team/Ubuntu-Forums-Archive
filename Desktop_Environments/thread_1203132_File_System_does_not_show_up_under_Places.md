### Post by Mayfairy on 2009-07-03
The link 'File System' points to your root directory '/'. 
You can add it there again by opening nautilus or any other file manager of your choice and navigate to '/' directory by hitting CTRL-L and typing '/' on the address bar. Then from the bookmarks menu choose Add Bookmark.

---

### Post by coffeecat on 2009-07-03
But having added the bookmark '/', it will show as '/' under Places and in your Nautilus Bookmarks menu. If you want to call it 'Filesystem', open a Nautilus window, go to Bookmarks > Edit Bookmarks and change the name.
