### Post by elsewhere on 2005-07-25
kubuntu-desktop provides the core kde pacakge with kubuntu-specific customizations, as well as key k/ubuntu components.  If you're already running ubuntu, most of those dependencies are already fulfilled and you'll probably just wind up installing the kubuntu-customizaed kde installation.

kde alone is a base "vanilla" installation of kde.

My recommendation would be for kubuntu-desktop, if only for the customizations and to make sure you have all the core components.  It will probably be required as well for an eventual upgrade to Breezy when it's released.  But if you're planning on customizing your desktop anyways, kde should be fine for now.  The core kde libraries and applications are the same either way.

Don't forget to download the dev packages if you're planning on compiling kde apps, they're not included with either kde or the kubuntu-desktop.

Hope this helps...
