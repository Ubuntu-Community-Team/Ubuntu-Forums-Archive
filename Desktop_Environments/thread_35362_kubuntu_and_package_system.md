### Post by dave9191 on 2005-05-18
Hey, 

I know exacly what you mean about compiling Gentoo. Its the only real reason I dont run it. Kubuntu is great tho, well at least I think so. 

Its pretty stable on the whole part, Ive had some trouble with hibernate and suspend stability tho. But thats still a beta, so I cant complain. 

apt-get is a great package manager system that will reslove dependacies for you on install and removal. You will want to add extra repositories to your list after install. You can find how to do that here [http://ubuntuguide.org/](http://ubuntuguide.org/) as well as some other useful stuff to help you get started. And if you miss the compiling, you can get apt-get to compile for you like emerge does with the -b option :) Just so you dont feel homesick.

The install should take you about 30 min or so, so there is no harm in trying it out. 

As for a list of all the packages, you can get that using apt commands or opening a graphical tool like Kynaptic or KPackage (if you install it). That will list all packages on the repositories. I dont think that there is an online one that you can browse tho ( I could be wrong ).
