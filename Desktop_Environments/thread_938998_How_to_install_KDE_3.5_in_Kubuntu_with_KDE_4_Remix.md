### Post by SuperSonic4 on 2008-10-05
```
sudo aptitude install kde-core
``` or ```
sudo aptitude install kubuntu-desktop
```

The latter will install kde3 and all the apps that come with it whereas the former installs only the core programs (useful if you don't want to instances of the same apps in the menu)

It is safe, you select KDE or KDE4 at login under session.
