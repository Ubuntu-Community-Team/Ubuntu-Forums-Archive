### Post by D-- on 2007-04-02
I installed xubuntu from an original ubuntu install using the Synaptic Package Manager. In Xfce4, on my xfdesktop, I have icons for whichever removable drivers are inserted into my machine.

I don't want them.

Is there a way to keep my xfdesktop icons and still get rid of the removable disk icons?


A second question: is there a way to keep an icon size, like 48, but just adjust the horizontal and vertical spacing between desktop icons?


Edit: may have found it. "Show icons for: removable devices" was unchecked. I checked it, enabled and disabled xfdesktop, then unchecked it and once again enabled/disabled. The SEEM to have vanished now. Will see if it holds true after reboot or if something from GNOME services sticks them back in.
