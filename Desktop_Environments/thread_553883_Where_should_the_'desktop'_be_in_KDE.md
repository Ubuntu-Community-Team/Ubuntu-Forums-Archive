### Post by GeneralZod on 2007-09-18
Files saved to /home/simon/Desktop should indeed appear on your actual Desktop (I've just checked).  Have you tried right-clicking on the Desktop and clicking "Refresh Desktop"? It shouldn't be necessary, but some people find that they have to do this.

To minimise all windows - either add the Show Desktop button to the panel, or set up a hotkey for it (System Settings -> Regional and Accessibility -> Keyboard Shortcuts -> Toggle Showing Desktop).  Win+M may already be assigned to this; I use Win+D instead.

In Konqueror, if you're using Icon View, do View->Preview and check off the filetypes you want to have thumbnails generated for.
