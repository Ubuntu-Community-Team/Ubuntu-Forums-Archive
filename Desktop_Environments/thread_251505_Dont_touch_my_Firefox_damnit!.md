### Post by Tomosaur on 2006-09-06
Solution: Don't use a black theme.

The format of drop down menus is built into firefox, not the web page. You can download a different firefox theme if you so wish, by clicking Tools > Themes. You should be able to find one which suits you better, but editing a system-wide theme will override your firefox theme, because that's just how it works.
