### Post by mcduck on 2009-04-01
Yes, that's only new feature for when you change your wallpaper.

The ability to use wallpapers that crossfade between different pictures over time has already been in Gnome for quite a long time. (but there still isn't any official tools to create that kind of wallpapers so it's not much advertised feature. You currently need to write the required XML file by hand.)
