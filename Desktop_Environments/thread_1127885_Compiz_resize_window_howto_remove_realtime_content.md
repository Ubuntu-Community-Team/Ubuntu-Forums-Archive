Hi,

I have installed the simple-ccsm package and have played around with some compiz effects. I have done something that changed the way the windows is resized and I would like to undo this effects.

BEFORE: when a windows is resized, the windows surface is represented by a semi transparent rectangle that move with the mouse.

NOW: when a windows is resized, the windows content is visible, the window itself is resized dynamically and the content of the windows is zoomed in/out real time proportionally to the window size.

How do I change the resize behavior back as it was before, while still preserving other compiz effects?

Thanks very much in advance for any help.

### Post by lnxguit on 2009-11-26
I found your question while trying to solve this problem myself. If you install compizconfig-settings-manager, go to the "Window Management" section and use the "Resize Window" plugin. On the "General" tab you'll see "Default Resize Mode". Set it as you like.
