### Post by jcooper on 2005-05-04
To modify my places menu (and this is probably the long way round) I do the following:

[list]
[*]Open Text Editor (Gedit)
[*]Click File > Open (this shows the open dialog, with the places on the left)
[*]Select the unwanted place from the list on the left
[*]Click Remove
[*]Click cancel in the open dialog
[*]Close Text Editor (Gedit)
[/list]
