### Post by grahammechanical on 2013-04-27
Do you have Compiz Configuration Settings Manager installed? If so, open the Ubuntu Unity plug in and make sure that the Dash background is set to #000000. Or set the Dash blur to no blur. See if that has any effect.

Regards.
