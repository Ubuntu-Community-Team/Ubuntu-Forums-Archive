### Post by Lawand on 2010-10-13
I recently installed Xubuntu 10.10 and I really liked the new default theme "Bluebird" but I noticed a very small issue which is that you can't close a maximized window by click exactly at screen corner, rather you have to point at the [X] button which is near the corner.

So, I made a modified version of the theme that doesn't have this issue and I decided to post it here FWIW.

If you're interested, here is how you can download and install the modified version:
[LIST]
[*]Download [this package]("http://www.box.net/shared/fsxk1htpya")
[*]Unpack and copy the folder "Bluebird-Edited" into "/usr/share/themes" (you'll need root privilege to do so)
[*]Select the theme "Bluebird-Edited" in [Settings > Appearance] and [Settings > Window Manager]
[/LIST]

I have only tested it with Xubuntu 10.10, but I assume that it works with other versions of Xubuntu as well as other distros or OSes that contain Xfce4

Note: This is the first time I edit/create Xfce4 themes, so the way I fixed the issue might not be perfect, but it works fine with me.

Comment and testers are welcome!
