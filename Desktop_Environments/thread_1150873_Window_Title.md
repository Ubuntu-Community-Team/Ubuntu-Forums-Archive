### Post by mcduck on 2009-05-06
This depends on the window border theme you are using. There is no (easy) option for configuring it apart from using a theme that puts the title where you like it to be.

Of course you can just edit the theme to change this, but Metacity themes aren't the easiest ones to learn to work with..

(If you use Emerald as your window decorator for Compiz you can use Emerald Theme Manager to change the title location.)
