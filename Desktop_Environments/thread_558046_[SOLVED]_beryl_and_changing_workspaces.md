### Post by PanopticChaos on 2007-09-23
Go to Beryl Settings Manager -> Desktop -> Desktop Cube
There are a number of options and reasons why it isn't doing the cube effect.  Could have a different animation set (Slide when changing viewpoints instead of rotating) and there are a number of time base settings which could just be making the effect so quick it's functionally removed (for this case check Beryl Settings Manager->Desktop->Rotate Cube->Misc Options, Rotation Speed, if that's too high, it'll look instant).

I'm sure there are a few other time settings which could make it occur over-fast as well, but those two would be the first spots I'd look.
