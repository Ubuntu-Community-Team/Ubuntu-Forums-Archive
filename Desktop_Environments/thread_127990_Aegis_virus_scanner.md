### Post by pauljwells on 2006-02-10
Look in /usr/bin. You should find the executable in there. It doesn't show up in the apps menu - I put a launcher in my Gnome top panel.

The first time you run it you need to be root, as it updates some system-wide config files with the virus definitions. To be honest though, there's hardly any chance of getting a virus in Linux, so it's not such a big deal as in Windows whether you have AV or not.
