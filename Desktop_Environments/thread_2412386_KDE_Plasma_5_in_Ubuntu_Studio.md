### Post by deadflowr on 2019-02-12
Plasma can be installed from the plasma-desktop package.
```
sudo apt install plasma-desktop
```
logout and select plasma from the login screen session selection menu.

Personally, though, I would run Ubuntu Studio as is (with xfce) and run Ubuntu Studio with plasma when they make a version with plasma.
