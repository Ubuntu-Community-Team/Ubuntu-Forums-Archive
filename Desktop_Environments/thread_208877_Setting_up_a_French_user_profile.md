### Post by JoWilly on 2006-07-04
At the login prompt click on options and select the language you want. The profile you then use to login will use that language.

(first make sure you have installed the complete french language support in menu/system/admin/language support).

Edit: alacarte will only modify the menu for the currently logged in user, the changes are saved in the user's home dir.
