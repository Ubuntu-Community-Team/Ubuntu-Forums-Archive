### Post by 3Miro on 2011-02-18
Gnome now comes in three flavors:

Gnome 2: currently the default Ubuntu desktop, soon to be deprecated in all versions of Linux.

Gnome-shell: the next generation of Gnome environment causing some controversy, currently in a beta stage of development (currently not included by default anywhere)

Unity: this is Gnome with different interface, currently in alpha stage, will be included by default in Ubuntu 11.04

For 11.04 Gnome 2 and Unity will come by default and one will be able to install Gnome-shell via the Software Center.
