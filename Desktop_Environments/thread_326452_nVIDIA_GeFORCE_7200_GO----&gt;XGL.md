### Post by l.tambiah on 2006-12-27
As you have a NVIDIA Card you are better off with XGL as the current NVIDIA Drivers support it. AIGLX is built into edgys xorg I believe but you can only use it with the beta NVIDIA Driver. I would use XGL with compiz or beryl and you should be able to use AIGLX in feisty.

I always use beryl, have not tried compiz manager yet.
