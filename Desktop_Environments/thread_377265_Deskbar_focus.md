### Post by diddymao on 2007-03-05
I'm having a problem with the Gnome Deskbar losing keyboard focus when I use.

When I use the hotkey to open the deskbar (mine is set to <Super>Space), the dropdown appears and I immediately get focus.  When I use the hotkey to close the window and then again to reopen it, the dropdown appears but does not get keyboard focus.  In fact, it appears over my window and I can't close it until I mouse click it and press escape (annoying).

At this point, the deskbar enters some state where when I press the hotkey, the deskbar appears underneath my maximized window where I can't see it and can't close it without repeated mouse clicks.

Anyone else have this problem?  Any workaround/fixes?  I'm currently using Edgy with beryl installed, all updates installed.  Thanks!

### Post by sixstringbeliever on 2007-03-08
I was having the same problem with the Deskbar using Edgy+Beryl.

To fix it:
Open Beryl Settings Manager
Under General Options, on the Main tab, change **Level of Focus Stealing Prevention** to None.

That worked for me.
