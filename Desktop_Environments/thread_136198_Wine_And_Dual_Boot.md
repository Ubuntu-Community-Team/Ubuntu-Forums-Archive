### Post by o_fortuna on 2006-02-25
[QUOTE=jaiaucuneide]During dual boot exactly how big is breezy and also where do you get wine and how would i get it to use warcraft III on linux...[/QUOTE]
Breezy takes up a few Gigabytes minimum, though you should leave space (mine currently takes up 3.53 GB) You can get wine by using Automatix ([here]("http://ubuntuforums.org/showthread.php?t=66563")) and typing "winecfg" into the terminal (Applications-->Accessories-->Terminal). Then, just doubleclick on the setup icon when you insert the CD. It should work. If it doesn't, there's always [cedega]("http://www.transgaming.com"), which is $5 a month (but it works).
