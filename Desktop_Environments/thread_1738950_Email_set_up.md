### Post by SeijiSensei on 2011-04-26
If IMAP is enabled on the Exchange server, most any client like Thunderbird or Evolution will work to some degree.

If the Exchange server supports web access, you can use that.

Have you tried running Outlook in WINE?  Otherwise you might be stuck having to install Windows in a virtual machine.

Remember that Exchange, along with Active Directory, are Microsoft's two biggest "lock-in" products for business.  It's not in Microsoft's interest to make it easy to use Exchange with anything other than Windows and Outlook.
