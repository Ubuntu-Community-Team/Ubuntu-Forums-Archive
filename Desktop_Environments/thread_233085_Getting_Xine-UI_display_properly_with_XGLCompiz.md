I've just found this on a spanish forum.
If your xine-ui looks like [this]("http://ubuntuforums.org/attachment.php?attachmentid=6174&d=1140104016"), enter this command to correct it (in your home directory) :```
echo "SharedPixmaps off" >>  ~/.imrc
```
