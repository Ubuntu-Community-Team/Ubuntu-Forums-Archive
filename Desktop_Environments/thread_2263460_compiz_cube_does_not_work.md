### Post by grahammechanical on 2015-02-01
Compiz is the compositor/window manager used in Ubuntu. So, if we are using Ubuntu we are also using Compiz.

We can also install Compizconfig Settings Manager (CCSM) that will give us the ability to configure Compiz and produce special effects. But, some of the special effect plugins are no longer installed with CCSM by default. Thhis is because they are no longer officially supported. We need to install compiz plugins to get these extra plugins. But keep in mind that development of these plugins may not be as advanced as we may wish.
