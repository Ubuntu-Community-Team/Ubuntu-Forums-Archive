### Post by britjm on 2005-12-17
I've actually had a problem with this question not too long ago. I've used everything mentioned in this thread and they all lacked the features that I needed. See, I can't be trusted to go and consciously check for new articles, so I leave my reader in the notification area. 
Liferea has this feature but the popup thing is horrendous in the fact that it tries to show all the new articles. Also I had a problem with articles being placed as being new everytime someone left a comment.
RSSOwl has the feature also, but I couldn't configure the Mozilla handler so the majority of my feeds were roughly useless.
Straw sucked.
Blam! sucked more.
Bloglines is pointless to me. It's only useful when you aren't at home. Otherwise, you may as well just go to the originating site. Same with Google's new feedreader beta-thinger.

So any other suggestions, guys?

### Post by DaMasta on 2005-12-17
What I was looking for was something that incorporated well with any mail account. I wanted to be able to set a variable to signify my mail client or something. I also wanted to be able to delete old articles and not just let them sit there. I wanted to be able to label good ones. I wanted to be able to be gone for several days and come to my reader and all the news has accumulated. I was only able to find all this with opera.
