### Post by mcduck on 2008-03-07
No, but if you want, you can replace the Menu Bar applet (the one with Applications/Places/System) with the Main Menu applet, which merges all 3 menus into one, quite like the Windows Start menu..

Just right-click on the Menu Bar applet, select "Remove from Panel", and then right-click on the empty space in the panel and select "Add to Panel", find the Main Menu applet and drag&drop it into your panel.
