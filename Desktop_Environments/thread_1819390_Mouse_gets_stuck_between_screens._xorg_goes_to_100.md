I can reproduce the issue by clicking on a window and moving it quickly between all 4 screens.  This will get the mouse stuck.

When i SSH'd into the machine and ran 'top' it showed the xorg at 100%.

I thought it might be a kde issue, so i installed and ran the gnome desktop with gdm as default, but the issue still existed, so switched back to kdm.

I've tried a few different 'recommendations' from various webpages but none seam to fix the problem.  I dont need any 3d capabilities because all i use the system for is programming.  

I do need a stable system.  Love some help as to how to not have to hard reboot at least once every day. :)
