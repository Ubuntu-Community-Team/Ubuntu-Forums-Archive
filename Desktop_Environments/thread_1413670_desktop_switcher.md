### Post by koleoptero on 2010-02-22
You'll have to install compizconfig-settings-manager and configure by the Vieport Switcher plugin the way you want to move from one desktop to the other.

You can also right-click on the panel and select "add item" and then add the workspace selector applet, if you don't already have it, and use that to move from one desktop to the other.

Hope this helps.
