### Post by denham2010 on 2008-11-01
From Synaptic, install CompizConfig Settings Manager

In Settings Manager - Autostarted Apps, add a new starter and name it Compiz with the command line compiz --replace

Restart your system and compiz will now start on boot. CompizConfig Settings Manager will now be available in the Settings menu for you to configure it to your liking.

Cheers.
