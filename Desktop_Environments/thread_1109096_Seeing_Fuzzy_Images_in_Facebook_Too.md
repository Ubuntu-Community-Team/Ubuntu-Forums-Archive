### Post by mcduck on 2009-03-28
Are you sure you haven't just accidentally zoomed in with Firefox? Because that sounds exactly what you'd get if you'd do that.

Press Ctrl-0 to reset the zoom. Or select View/Zoom/Reset from the menu. (Ctrl-+ zooms in and Ctrl-- zooms out).

You can also configure Firefox to only zoom text, not graphics (which I think is the way to go as long as web pages you are using are coded properly and don't break if the text size changes..). To do that enable View/Zoom/Zoom Text Only in the menu.

### Post by HyperHacker on 2009-03-28
Maybe you're just having connection issues and the images aren't loading fully. JPEG images are often stored in a progressive format (I think that's the right term) so that if the image loads slowly, instead of seeing one line appear at a time, you see a very fuzzy image appear and gradually get clearer as the more detailed bits arrive.
