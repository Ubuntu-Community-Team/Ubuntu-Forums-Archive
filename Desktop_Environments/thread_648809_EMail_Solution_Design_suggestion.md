### Post by jmaryinchina on 2007-12-24
I have been running the trio Postfix+IMAP Server+Fetchmail for years on Debian and Ubuntu.

I strongly disagree with the idea of evolution as default email client and the reason is simple : it's really too buggy.

Running this trio, with a config solution «system wide» (needs an API layer to make one XML config  for mail accounts converted to each mail client config layout) on Ubuntu would allow to switch from one mail client to another, effortless because the mail storage is done in the IMAP server.

No need anymore to import or export emails from one client to another.
