### Post by Flying Pikachu on 2009-06-27
Yes, This is possible just go to "Add/Remove Programs" and click the education category and you can choose separate programs or bundles of programs for your education level, i.e Pre-School, Primary, Secondary etc, or in the terminal type in these commands

PreSchool Bundle:
 ```
sudo apt-get install ubuntu-edu-preschool
```

Primary School Bundle:
```
sudo apt-get install ubuntu-edu-primary
```

Secondary School Bundle
```
sudo apt-get install ubuntu-edu-secondary
```

Tertiary Bundle:
```
sudo apt-get install ubuntu-edu-tertiary
```
