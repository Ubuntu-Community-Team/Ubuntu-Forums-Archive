### Post by mcduck on 2007-12-09
You can get new GTK themes from [http://www.gnome-look.org/]("http://www.gnome-look.org/").

After downloading just go to System/Preferences/Appearance and drag&drop the theme package to that window to install the theme. Then click the Customize-button and you shoud be able to change the theme from there.
