### Post by sirjoebob on 2008-11-17
Hey b3rt. Backing up Compiz is pretty straightforward. Fireup CCSM and go to preferences (lower left hand corner) and you can backup a configuration file from there. Just export, save it somewhare, put it on the new machine and import it.
