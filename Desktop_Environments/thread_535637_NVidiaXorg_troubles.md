### Post by mlind on 2007-08-28
make sure that the HorizSync VertRefresh are exactly correct for your monitor. You can lookup these values from your monitor's manual or google.

Then disable EDID probing if you're using nvidia (proprietary) driver
```

Option "IgnoreEDID" "true"

```
[http://http.download.nvidia.com/XFree86/Linux-x86/1.0-8178/README/appendix-d.html](http://http.download.nvidia.com/XFree86/Linux-x86/1.0-8178/README/appendix-d.html)

(if this doesn't work, try *Option "UseEdidFreqs" "false"*)
