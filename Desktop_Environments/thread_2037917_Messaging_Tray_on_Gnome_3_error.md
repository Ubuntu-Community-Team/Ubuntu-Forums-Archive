### Post by grege on 2012-11-27
You have to be careful with themes. 3.4 themes will wreck 3.6, particularly notifications.

Go to extensions.gnome.org and install the theme changer extension.

Get a real 3.6 theme and put in ~/.themes Log out and in again.

Then use Tweak Tool to select the theme.

Might I suggest XGnome Enhanced from 

[http://browse.deviantart.com/customization/skins/linuxutil/desktopenv/gnome/gnome/?order=5#/d5iweo6](http://browse.deviantart.com/customization/skins/linuxutil/desktopenv/gnome/gnome/?order=5#/d5iweo6)

as a starting point that I know works properly

I also use the Gnome PPA to complete all the Gnome bits.
