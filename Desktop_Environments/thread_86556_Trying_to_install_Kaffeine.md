### Post by Ampersand on 2005-11-05
libdvdcss isn't un the official Ubuntu repositories, but you can get it by following the instructions on this page:
[http://wiki.ubuntu-fr.org/doc/plf](http://wiki.ubuntu-fr.org/doc/plf). (ignore the bits about java unless you happen to need it for something)

You'll only need this if you want to use Kaffeine to watch DVDs, it should work for other video files as it is.
