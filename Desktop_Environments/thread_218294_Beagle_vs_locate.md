### Post by mazirian on 2006-07-18
These are very different things.  locate allows you to search for filenames matching patterns on the system against a table of filename information generated by updatedb.  Beagle indexes information about the contents of the files on your system, and so is much more thorough for searching.  The analagous unix command would be more like find . -exec grep "pattern" {} \; But that command will take a long time.  Beagle should have already done that work for you.
