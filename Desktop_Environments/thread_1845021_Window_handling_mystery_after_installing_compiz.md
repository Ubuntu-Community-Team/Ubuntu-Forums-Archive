### Post by ManualSparrow on 2011-09-17
I think compiz creates its own set of viewports on top of the xfce ones. Try middle-clicking the desktop. It will probably show however many desktops you were using when XFCE was managing your windows, but switching between them won't work as expected. You can use Compiz features like "scale" or "shift switcher" to switch between windows in a variety of different ways - one of them might suit you.

Note: Search compiz-config-settings-manager in Software Center to install the GUI config program.
