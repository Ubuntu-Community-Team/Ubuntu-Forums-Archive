### Post by crdlb on 2011-10-18
Those buttons were in a very old version of the shell. In the current shell, workspaces are created on demand. In other words, you will always have one empty workspace. If you'd rather static workspaces, there's [an extension]("http://intgat.tigress.co.uk/rmy/extensions/index.html") for that.
