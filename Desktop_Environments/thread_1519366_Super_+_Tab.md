### Post by howefield on 2010-06-28
Go to System > Preferences and select Keyboard Shortcuts.

Scroll down to the option "move between windows, using a popup window and click on the shortcut to highlight it, the current key combination will change to read New Shortcut, press the keys you want, presumably the Super + Tab, the shortcut will now read Mod4 + Tab.

Close your way back out, all done.,
