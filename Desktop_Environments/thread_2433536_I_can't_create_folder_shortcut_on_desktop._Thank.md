### Post by Dennis N on 2019-12-20
> **billyivy1228 said:**
> Hello. I am a beginner of ubuntu 19.10. I want to creat folder shortcut to go wine folder and I download Tweaker but I can't do it. Would you help me to solve thia problem? Thank for your helping.
You can create the link from Files:
First, you need to enable "Link Creation" in Files. In Files, go to Preferences > Behavior Tab. Check the box shown under "Link Creation".  See screenshot.
Then in Files navigate to the parent folder of the folder you want to link. Then Right-click on the folder and Select "Create Link"
The link appears. Move it to your Desktop folder. It will also appear on the Desktop.
Done.
