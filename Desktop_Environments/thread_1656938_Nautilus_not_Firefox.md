### Post by Tom_AUT on 2011-01-01
Right-click somewhere on the desktop and select *Create Folder* to ... well, create a folder. Now right-click that folder and select *Open With Other Application...* and select either File Manager or Nautilus from the list of applications.

If neither of them are displayed in that list, click the arrow next to "Use a custom command" and type *nautilus* in the box.

Before clicking the OK button, make sure that the box next to *Remember this application for "folder" files* is checked.
