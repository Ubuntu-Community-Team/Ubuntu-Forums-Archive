Okay, apparently after searching through every directory mentioned in various googled LXDE main menu editing guides for the menu entry, one directory seems to never get a mention till I found it now. Also it never seemed to show up when I did a file search. No idea what's going on there, but problem solved anyway. The menu item was in...

```
/usr/local/share/applications/
```

Deleted it, now it's gone from the menu.
