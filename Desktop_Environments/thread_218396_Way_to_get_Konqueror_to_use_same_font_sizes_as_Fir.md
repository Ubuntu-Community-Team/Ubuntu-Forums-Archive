### Post by Arnaud_B on 2006-07-18
Hi,
The font size are treated differently in konqueror than in firefox... what you might be interested in can be found there:
[http://kudos.berlios.de/kf/kisimlar/fixes.html#setxdpi](http://kudos.berlios.de/kf/kisimlar/fixes.html#setxdpi)
I kept 8 and 10 as font size for konqueror but changing the dpi and the fonts (if you wish...) make a big difference... I had the same problem at the beginning and I solved it like that... :-) 
I know that it seems a little overwelming but it's not that hard, set a dpi of 96 if you want it more "windows-like" ;-) you might want to have a look at this threat also: [http://ubuntuforums.org/showthread.php?t=20976&page=1](http://ubuntuforums.org/showthread.php?t=20976&page=1)
