### Post by flummoxed on 2006-10-05

### Post by dwasifar on 2006-10-05
If you plan to install Windows on this drive and use it as your primary system drive, follow flummoxed's advice, above.

If you are installing this as a second drive in your system that is already running Windows, then once it is physically installed, start Windows, go to Control Panel>Administrative Tools>Computer Management, click the Disk Management option on the left side, and find the new drive in the list.  Right click on each partition you see in it and choose Delete Partition.  Then right click on it one more time to create a new partition.  Format your new partition and you'll be all set.
