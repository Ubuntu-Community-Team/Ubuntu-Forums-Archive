### Post by ddtmsa on 2006-01-27
You need to find out what chipset is in your modem, not just the make.

eg just like if you have a Dell3100 computer, we need to know which processor is in it, either Intel or Athlon and which particular type.

Scanmodem will give you this information, although it should also be available from the windows control panel if you dig deep enough.
