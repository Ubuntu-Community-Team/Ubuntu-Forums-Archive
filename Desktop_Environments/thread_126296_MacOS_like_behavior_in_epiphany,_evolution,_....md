### Post by GreenCountry on 2006-02-10
To my (limited) knowledge, you won't be able to do that in Linux.  The "Dock" (what Mac OS calls the feature you noticed) is a combined launcher and taskbar.  Rather than list windows that are open, it lists applications that are open, and you have the opption to leave an application's icon in the Dock even when the app is closed - thus making the icon a launcher as well.

In KDE and GNOME, which behave more like Windows (or rather, Windows behaves more like them :) ), you can't make a taskbar entry "stay" there after the window is closed.  I think some of the other windows or desktop environments might do what you want, but that's a long way to go for what, inb my opinion, is a pretty small feature.  Just click on the entry for the window in the taskbar to pull it up, and don't go for the icon.
