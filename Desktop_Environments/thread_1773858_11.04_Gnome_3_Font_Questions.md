### Post by LarsKongo on 2011-06-03
Font colors are now changed in the CSS files.
The default theme Adwaita is located in /usr/share/themes/
It requires some CSS knowledge though.
Some global settings can be found in the gtk.css file.
Change theme_text_color and theme_fg_color to #000;

For the best font rendering in Ubuntu you should enable this in gnome-tweak-tool:
Hinting = Slight
Antialiasing = Rgba
If you are deciding to use any other distribution than Ubuntu or Mint you may need to install font rendering patches to get the absolute smoothest fonts.

The default font Cantarell is however pretty bad in my opinion. So change it to ttf-ubuntu-font-family, Segoe UI or Trebuchet MS.
