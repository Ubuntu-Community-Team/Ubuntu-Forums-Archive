### Post by Bou on 2006-01-15
Hi, I recently discovered a nice feature of the way the gnome clipboard works: just by selecting text, it will be copied anywhere by pressing the middle button.

It's so fast to work with, I love it. But there's a problem, you can't replace text this way because if you select the text to be replaced, it's copied to the clipboard. Bummer.

I think you should be able to select text with the middle button (dragging the mouse while pressing it), thus automatically replacing that text with the previous selection. So:

Select text with left button -> copy it to the clipboard
Select text with middle button -> replace it with the clipboard contents

What do you think about it? Do you think that would be useful? Should I file a bug to the Gnome people?
