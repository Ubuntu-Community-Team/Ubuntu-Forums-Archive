 they go to /usr/bin.  to run them, just click applications -> run  and type the name in(or from the terminal, either or).  if you download gcrontab, it does not appear on menus but is gui based.  simply open a terminal and type gcrontab.  Think of it as windows, having every EXE together in one big directory so that you could find them easily.
