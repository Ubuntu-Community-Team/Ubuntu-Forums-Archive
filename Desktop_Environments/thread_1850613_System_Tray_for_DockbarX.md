### Post by Copper Bezel on 2011-09-27
Not by itself, no. It's very often used as an applet for Avant Window Navigator, which does have a systray. It's only quite recently that DockBarX gained its stand-alone mode - it's always been used in AWN or a Gnome Panel, which of course makes it possible to have a system tray in the same panel.
