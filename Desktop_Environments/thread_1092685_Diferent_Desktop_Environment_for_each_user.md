I've searched, but haven't found any information on this, Is it possible for each user to load a different desktop environment, preferably from a GDM login window?

---

### Post by taurus on 2009-03-10
Each user can pick whichever window manager or desktop environment she/he wants to use at the GUI login screen.  She/He can set it as a default so she/he doesn't have to answer the question every time.

---

### Post by blazemore on 2009-03-10
So what, you type the username, then click Sessions and choose? I thought that was systemwide?

---

### Post by taurus on 2009-03-10
Or you choose which session to run.  Then type in the username and password.  If that is not the default session, the system will ask you if you want to make it as your default or you just want to run it for this session.  It's not a system wide; it's an individual's choice/session.
