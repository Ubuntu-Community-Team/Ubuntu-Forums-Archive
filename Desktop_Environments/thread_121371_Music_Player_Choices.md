I haven't found anything quite as powerful as foobar (or at least, as powerful in the same way).  I'd recommend either Amarok, which is slick, fairly fast, and powerful, but a KDE app (if that makes any difference to you), or Banshee which is **really** slick, Gnome, and I can post a patch which enables something like the columns-ui album view. :)  Banshee's in heavy development at the moment, but the Ubuntu package should work :)
Banshee is more of a complete music manager, actually - it handles CD ripping/burning and iPods (and some other portable music players) - both reading & writing.

### Post by doclivingston on 2006-01-25
Which is best depend on exactly what features you want, and which ones wou don't need. Some options include:

* Rhythmbox (installed by default)
* Banshee
* Quod Libet
* amaroK (requires some KDE libraries)


Exactly what kind of audio player is the Nomad Zen Xtra? If it is mass-storage (you can get to it via Nautilus), then I think the latest Banshee will work with it, and the next version of Rhythmbox will (which hasn't been released yet). I don't think any of the standard versions of the music players in Breezy will, although versions in Backports might.
