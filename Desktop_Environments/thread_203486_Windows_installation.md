### Post by evil_elman on 2006-06-25
If you didn't get a CD with Windows or not even a rescue CD then you probably had a rescue partition on the HDD of the laptop.

Many laptop manufacturers (incl. my Acer laptop) has got a special way of getting to this rescue partition during boot and in that way restore the laptop.

However, if you formatted the HDD there's a good chance that you wiped the rescue partition in the process as well. The only way to go is to call the manufacturer to get Windows back one there... Or maybe install Ubuntu instead... ;-)
