### Post by tehwa on 2005-09-20
I use Audacity [[http://audacity.sourceforge.net]](http://audacity.sourceforge.net]) which is a good audio editor/recorder for Linux, which is a little more friendly than the default GNOME sound recorder when it comes to selecting the audio source.

It's available (after adding repositories from [http://www.ubuntuguide.org](http://www.ubuntuguide.org)) via apt-get install audacity.

It's common to see a wee error when you start it up, something about audio resources not available. Just open System > Preferences > Sound and uncheck "enable sound server startup" then restart audacity.

With the proper plugins, Audacity can export captured sound as WAV, mp3 and ogg.
