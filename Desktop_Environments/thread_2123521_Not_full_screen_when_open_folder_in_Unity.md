### Post by grahammechanical on 2013-03-08
If a utility like the File Manager is in full screen mode when it is closed, then it will open in full screen mode when it is next opened. It remembers that last state it was in. So, you need to reduce the utility from full screen mode. Move the mouse the the top right of the screen and then Close, Minimize and Maximize buttons will appear. Click on the right hand button (with the square printed on it) and that will un-maximise the window application. Now close the application.

Oh, by the way, did you know that if you hold down the super key (Windows key) you get an overlay showing you keyboard short cuts. Some of these have been changed since 12.04 came out. I am on 13.04 and Ctrl+Super+Down arrow = Restore or minimize window.
