### Post by Doovoo on 2006-06-27
The most refined and used (even by big production studios) is [Ardour]("http://ardour.org"). Ardour is really well made, but I've never gotten it totally working on a Linux system. This is because last I checked, it required you to manually install it in the Kernel itself (as well as requiring alot of libs). There was even a Fedora distro that was basically "The Ardour Distro" that came with Ardour all set up. From what I understand, this has changed and you can apt-get it (maybe I should try that). 

I would try [Rosegarden]("http://www.rosegardenmusic.com/") (Which is fairly far in development, 1.2.3 i think) for a lighter solution, and the [companion]("http://rosegarden.sourceforge.net/tutorial/en/chapter-0.html"). It takes a little bit of work setting up (you can install in synaptic, but you need to set up JACK and all of your audio stuff) but it's a pretty solid program.

You should also check out [Wired]("http://wired.is.free.fr/index.php"). It's way early in development, and pretty buggy right now, but I firmley believe it will become **the** audio production program for Linux.

Audacity is great too.

Sorry, can't help you with Visual Studio.
