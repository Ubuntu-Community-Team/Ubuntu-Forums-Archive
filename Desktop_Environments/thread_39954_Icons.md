### Post by jcooper on 2005-06-07
Try [Gnome Look](http://www.gnome-look.org) or [Gnome Art](http://art.gnome.org).

Then simply drag the .tgz file you download onto the theme window to install the icon theme. You can also extract this file into the .icons directory in your home folder (i.e. ~/.icons). I personally like [gPerfection](http://gnome-look.org/content/show.php?content=22989).
