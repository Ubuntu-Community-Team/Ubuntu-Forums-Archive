### Post by Dennis N on 2020-05-06
There are fundamental differences that you can't overcome stemming from the fact that Ubuntu Mate 16.04 was based on gtk2 while 20.04 is based on gtk3, so the panel and many MATE packages have been redone. The 16.04 wallpapers are [here]("https://oswallpapers.com/ubuntu-mate-16-04-default-wallpapers/"). These can be used in 20.04.
