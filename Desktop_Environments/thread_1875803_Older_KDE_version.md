Sure.  You're best off installing entirely from scratch, though.  I'd also write /home to a backup device before doing this unless you already put /home on a separate disk partition during installation.  (If you're installing from scratch, I'd recommend taking that route this time around. During installation you can choose "manual" partitioning and create a partition for / and one for /home.)

Even though I'm stuck back on KDE 4.5, I've been quite happy with Kubuntu 10.10.  It's quite stable and works well with my hardware, particularly my wifi adapater.
