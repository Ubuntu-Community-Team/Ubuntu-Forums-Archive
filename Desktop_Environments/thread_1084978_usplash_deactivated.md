go to system > administration > startup-manager


when it opens, check "show boot splash"

if that doesnt work then you will have to resort to the default ubuntu usplash. to do that, open the startup manager again and click the "appearence tab". where it says "usplash theme" select "usplash-theme-ubuntu" and close it.


reboot and you should now have a usplash screen.
