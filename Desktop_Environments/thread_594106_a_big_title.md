### Post by Victorq10 on 2007-10-29
I install ubuntu 7.10 on my laptop HP Compaq nx7400 two days ago.
When ubuntu was loaded from liveCD I had big title. Only after change in 
Appearance -> Font -> Window title -> size 
on 1 px
the window title became as it 10 px height.

After installing on hard drive the same problem is present.

When system load 
Wellcome theme (Human) I have next problem
- in Input field typed text is too big. (Yes, that no prevent me to pass login page) 
- Font is too big in menu after pressing options button. Only two rows are placed on a screen.

One method to resolving problem change "Visual Effects" to "None" and than back to "Normal"
and another change font for window title on 1 px (don't perform first point).
