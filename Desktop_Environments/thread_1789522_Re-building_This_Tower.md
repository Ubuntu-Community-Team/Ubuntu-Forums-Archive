### Post by ankspo71 on 2011-06-24
Hi,
First you need to make sure of the type of PCI slot you have available on your motherboard, because there are different types of PCI slots (PCI , PCI Express x16, etc) Some motherboards will have the type of PCI slot printed right next to the PCI slot. If yours doesn't, then you can compare your available PCI slot with the images on this page: [http://technologyinthefuture.com/the-new-fastest-of-bus-pci-express/](http://technologyinthefuture.com/the-new-fastest-of-bus-pci-express/)
Or you can look up the model of the computer you have and see if you can find the computer's hardware specifications.

Now see if you can identify the old graphics card's model name, memory size, and memory type. This information will be helpful when selecting a new card. If it is an older computer, you should probably stick closer to the same specs. If the computer isn't that old then maybe you can get something a little more powerful.

Now go to newegg.com and search for **ATI PCI** or **ATI PCI 256MB** if you want to include searching for a certain amount of memory. 

You are going to see alot of listings for 'PCI Express' video cards because it is getting harder to find the older PCI video cards. You don't necessarily have to buy the "ATI" branded ATI cards, you can go with the cheaper brand names too.

Some more video card advice from the internet:
[http://www.pcmag.com/article2/0,2817,2371083,00.asp](http://www.pcmag.com/article2/0,2817,2371083,00.asp)
[http://www.wikihow.com/Buy-a-Video-Card-For-Your-PC](http://www.wikihow.com/Buy-a-Video-Card-For-Your-PC)
[http://www.computerhope.com/btips/video.htm](http://www.computerhope.com/btips/video.htm)

Hope this helps.

### Post by ankspo71 on 2011-06-24
Hi,
I found a Gateway E4600 System Manual online [support.gateway.com/s/manuals/servers/eseries/8506984.pdf]("support.gateway.com/s/manuals/servers/eseries/8506984.pdf") (with 423 socket Pentium4) but it doesn't give the specifications to the original graphics card. It does say that it has 5 PCI (not PCI Express) slots though, so at least now you know which type of slot you will be buying a graphics card for. 

I would imagine the original graphics card has about 128mb to 256mb memory.

If you find out the motherboard itself is burned up too, ebay.com shows listings for used Gateway E4600 (several 478 socket and one 423 socket) motherboards with a "buy it now" price for as little as around $20 (some of these include a used Pentium4 (478 socket) processor too). One example is: Gateway E4600 Motherboard Socket 478 + P4 2.66Ghz CPU = $25.95 
Hope this helps.
