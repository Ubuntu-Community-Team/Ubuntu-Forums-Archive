### Post by escobar on 2005-10-20
Yea I'm pretty sure *everyone* hs this issue. I did as well. 2 ways you can go about it. 

1) With the 'Network Settings' window, or any other window that requires admin rights to change, press 'alt+m', That should bring up the prompt for you to type you admin password.

2)You could always run 'kdesu kcontrol'. This is far easier, and allows you to navigate the various menus without having to type in your admin password to make changes.

Hope that helps. Take care.
