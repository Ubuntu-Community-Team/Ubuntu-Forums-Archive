### Post by ardchoille42 on 2007-02-23
I have a truly stable desktop, and have had zero crashes and zero problems. I'll tell you how I did it.

First of all, I use Ubuntu 6.06.1 LTS (Dapper) because Edgy was rushed and has way too many problems.

Secondly, I following these rules religiously:
1) Only install from the official repos and then only use the package manager that was shipped with Ubuntu
2) Never compile anything, never needed to, the Ubuntu repos are huge
3) Never install packages that were made for other distros (eg .rpm's, debian .deb's, and alien **is** dangerous)
4) Never "--force" anything, there should never be a need to do this
5) Never enable the root account, use sudo/gksudo.. this model is in place for a reason

I follow those rules and have never had any crashes or other problems since I have been using Ubuntu. I take that back, I had lots of problems with Edgy, but that wasn't my fault.

Problems begin when you start using 3rd party scripts to install things that you can do yourself.. or when you start using alpha and beta software (beryl?).. or when you don't pay attention to what the system is trying to tell you.
