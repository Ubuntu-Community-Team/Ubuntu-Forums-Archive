### Post by mojoman on 2008-01-04
If you have Ubuntu you have Gnome so install KDE-desktop as well (through CLI, synaptic whatever you use). KDE should then show up as an option in the GDM or KDM login screen (whichever you decide to use). I'd recommend that you do a minimal KDE install.

### Post by Tyke91 on 2008-01-04
if you have gnome, type this into a terminal
```
sudo install kubuntu-desktop
```

if you have kde

```
sudo install ubuntu-desktop
```

then to switch, just click the *sessions* button on the login screen, if you have the default human login screen, it should be under the options menu.
