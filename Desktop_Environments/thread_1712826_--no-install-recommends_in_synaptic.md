How can I tell synaptic not to install recommended packages. Using command line "apt-get", there is an option "--no-install-recommends". How can I do that in synaptic?

---

### Post by mcduck on 2011-03-23
> **mahmoodn said:**
> How can I tell synaptic not to install recommended packages. Using command line "apt-get", there is an option "--no-install-recommends". How can I do that in synaptic?

Go to Settings/Preferences, and on the General-tab unmark "Consider recommended packages as dependencies".
