### Post by varunus on 2004-12-22
One handy little tool is gnome-clipboard-daemon, found at [http://members.chello.nl/~h.lai/gnome-clipboard-daemon/](http://members.chello.nl/~h.lai/gnome-clipboard-daemon/) 

It makes the clipboard behave like the windows one, so closing an application won't erase the clipboard.  Some people like this, some people don't.  I personally do.

not programs, but here are a couple of nautilus scripts that has made my newbie life much easier ..

1."open a terminal here"... (drill down to any directory, right-click, open a terminal in that directory, no more cd'ing or typing the path)

2. "gedit as root"... (right click any file and edit that file as root, safer than removing sudo password altogether which is also an option on the unofficial start guide)

find these scripts and others on this wikipage

[http://www.ubuntulinux.org/wiki/NautilusScriptsHowto](http://www.ubuntulinux.org/wiki/NautilusScriptsHowto)
