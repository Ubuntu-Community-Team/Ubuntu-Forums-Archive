### Post by cwaldbieser on 2005-03-11
I think alastair meant you should open up the terminal and launch your graphical mail client by typing in the command there.  For example:

```
$ kmail
```
or
```
$ thunderbird
```
or
```
$ evolution
```

The terminal will not return you to a prompt while the app runs.  Then, you can try to upload your attachment, and when it fails, maybe something will print out to the terminal console telling you what the problem is.
