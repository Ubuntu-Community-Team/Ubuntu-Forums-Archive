### Post by Jucato on 2007-04-02
To get rid of that Mac OS X-like menu bar, Right-click on the desktop, select Configure Desktop, go to the Behavior options page, and select "None" for the "Menu bar at top of screen option.

The first run wizard that was run is called **kpersonalizer**, which you can run by pressing Alt+F2 and typing in the name (all lowercase). If I remember correctly, the last "page" of that wizard tells you how to run the personalizer app again.
