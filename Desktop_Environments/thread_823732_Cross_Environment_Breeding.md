### Post by beanie0610 on 2008-06-09
Is there any way to access my partition with ubuntu on it if I have kubuntu installed as a separate partition on my computer?

---

### Post by benerivo on 2008-06-09
Yes. This can be set up when you install kubuntu. If you already have kubuntu installed, you can add a line to your /etc/fstab file in your kubuntu partition so that it automatically mounts your ubuntu partition when you run kubuntu.
