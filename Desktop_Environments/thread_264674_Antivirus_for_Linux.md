### Post by sarhento_lobo on 2006-09-25
When I first started using Ubuntu, I tried to duplicate the environment that I had in Windows, including installing an antivirus (clamav) and setting up a firewall (using iptables); needless to say, I found that it was just something extra that, in my over a year of using Ubuntu Linux, I found that I never needed  (imagine using Window$ for over a year without an antivirus and firewall and let's see how long your system will last).

This isn't to say that a Linux box is a perfect ivory tower sealed off from all kinds of malicious stuff, but if you're only doing normal things like office, programming, media, and all that, then you'll be fine.

Well... I have a different opinion here. If you deploy a Linux server and have only Linux clients on your network, I would say it's generally safe without a Virus Scanner.
