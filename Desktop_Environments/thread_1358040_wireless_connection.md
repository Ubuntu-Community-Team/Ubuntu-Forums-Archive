### Post by Zorael on 2009-12-18
When you click the networking manager tray icon, it will only list networks that you have set as trusted. When connecting to a new network, you have to click the "Connect to other network" link and select it there.

You only have to do this once to make it trusted, then it should show up in the normal right-click list. You can remove trusted networks in System Settings -> Network Settings -> Wireless.
