### Post by subjectdenied on 2004-12-23
hi,

i did the following for getting it work

get the deb-file from here: [http://prdownloads.sourceforge.net/guifications/gaim-guifications_2.6-2_i386.deb?download](http://prdownloads.sourceforge.net/guifications/gaim-guifications_2.6-2_i386.deb?download)

become root and use:

<code>dpkg -i gaim-guifications_2.6-2_i386.deb</code>

this should be enough

however you have to restart gnome, so you can enable the plugin in the gaim-options
