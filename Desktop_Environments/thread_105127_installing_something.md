### Post by psychicdragon on 2005-12-17
Ubuntu uses a program called apt to install software. You can use a program called Synaptic to browse for software to install and apt will download and set it up for you. Synaptic is in the System > Administration menu at the top. It's not advisable to install software from source unless it's not available through apt.

Here's how you can install freeciv from the terminal:
```
sudo apt-get install freeciv
```
