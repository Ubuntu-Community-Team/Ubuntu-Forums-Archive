### Post by jualin on 2009-09-14
Do you have compiz enabled? Check under System, Preferences, Appeareance and Visual effects. If you do then you need to install compiz settings manager using this code on the terminal > sudo apt-get install compizconfig-settings-manager or by installing the package "compizconfig-settings-manager" from synaptic package manager (under system, administration). Hope this helps!

### Post by staf0048 on 2009-09-14
Well, my advice may not be what you're looking for, but IMHO it's not worth it.  It's cool, very cool indeed, but since it relys on your graphics card, which probably does not have an open source driver, there can be some unintended consiquences which turn into major annoyances.

But, if you really, really, really want the desktop cube, make sure you have your Hardware Driver's enabled under System/Administration and you should be set.  The computer should let you know if it wont work.
