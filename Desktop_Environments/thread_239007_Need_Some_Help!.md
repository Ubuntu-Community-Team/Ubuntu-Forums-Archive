### Post by waldschm on 2006-08-18
As to your first question, I just use the print screen button when I need a screen shot but if you need more features I'm afraid I don't know any programs.  To see your processes, you can do several things. The easiest is to go through the menus System -> Administration -> System Monitor and view the processes there. Alternatively, if you want to do this from the command line you could either do:
ps auxww (for a complete listing from the time issued)
top (for an updating view of processes)

As for your wireless problem, I don't know about your specific device but you could check the wireless forums:
[http://www.ubuntuforums.org/forumdisplay.php?f=139](http://www.ubuntuforums.org/forumdisplay.php?f=139)

### Post by Malakia on 2006-08-18
A suggestion for your wireless problem is to check out madwifi or ndiswrapper depending on which one might support your card. I had to use ndiswrapper after upgrading my kernel and it made my internet run a lot faster without all of those dips in speed.
