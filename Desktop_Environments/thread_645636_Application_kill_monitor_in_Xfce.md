### Post by mali2297 on 2007-12-21
If you want something more convenient than kill from the terminal, then you can try **xkill**.
When the command is run (for example via a keyboard shortcut), you get a special cursor which you place on the window you want to kill and just *click*.

xkill is available in the repositories.
