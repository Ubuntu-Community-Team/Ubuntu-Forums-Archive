### Post by happysmileman on 2007-11-20
As well as the top file being bigger and containing all the main derivatives, it's also the previous version (7.04 instead of 7.10) so just get the regular one (if you want a different desktop environment you can go get Kubuntu/Xubuntu etc. but if you're new it's probably best not to trouble yourself with that choice at least for another few months).

Also, something I always say to new people installing Ubuntu, if you're sharing the harddrive with Windows or anything like that, be sure to defragment the drive first, doesn't seem to cause problems for most people but it lost all my data the first time
