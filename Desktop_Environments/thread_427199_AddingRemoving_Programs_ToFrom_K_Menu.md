### Post by miggols99 on 2007-04-29
Add/Remove programs should be in the K menu. If you're looking for something like synpatic (or however you spell it) your use adept, found in the System submenu. If add/remove programs isn't there, you can type ```
kdesu adept_installer
``` into the terminal.
