### Post by mcduck on 2007-09-03
Not all themes are complete Gnome themes, most only have either GTK2- , window- or icon theme. And only full themes will show right in the theme manager, the rest you have to select by clicking the customize-button. When you have selected the GTK2-, window- and icon themes you want to use you can save them together as a  Gnome theme, which will then appear in the main Theme Manager window.

Ad just to make things clear:

Gtk2/Applications themes change how your applications look like, what color your windows, menus and buttons are and what they look like.
Metacity themes change the window frames and titlebars. Metacity is the default window manager in Gnome.
Icon themes should need no explaining ;)
Combine one of each together and you'll get a full Gnome theme that you can save in the Theme Manager.

Emerald themes are used instead of Metacity themes if you are running with Compiz and Beryl .
GDM themes are for the login screen, these themes are changed in System/Administration/Login Window.
