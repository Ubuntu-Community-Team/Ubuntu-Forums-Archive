### Post by lkjoel on 2010-07-05
Yes!
Try this in a Terminal(Applications->Accessories->Terminal):
```
sudo apt-get update
sudo apt-get upgrade
sudo apt-get install compiz-fusion-plugins-main compiz-fusion-plugins-extra
```
And you don't need to bump every 15m. Usually volunteer helpers check up to 1 day.
