### Post by miggl on 2009-06-26
When opening the Styles and Formatting window (F11) in OpenOffice Calc, I expect to be able to modify existing Cell Styles. Unfortunately, when I right-click on a style and choose Modify..., nothing happens. (I can, however, modify page styles; but that's not my goal.)

Does anyone know what's going on, and how I can access the cell style modification window?

Thanks!

Running OpenOffice 3 on Ubuntu 9.04.

---

### Post by miggl on 2009-06-26
Bah! Not 5 minutes after posting, I realized what the problem was: cell styles cannot be modified if some of the pages are protected. After unprotecting all worksheets I was able to modify the cell styles.
