### Post by Sokraates on 2005-12-11
I thinks it's definitely worth upgrading to FF 1.5.

Not only is it faster than FF 1.0.x (though not in a way that will make you go WOW, AWSOME!!!), it is also more comfortable. The new option-tabs are a lot more sensible and the new update-feature is great.

For installation try this [HOW-TO](https://wiki.ubuntu.com/FirefoxNewVersion).

If your old extensions and themes do not (yet) support FF 1.5, try the [Nightly Tester Tools](http://users.blueprintit.co.uk/~dave/web/firefox/buildid/nightly.html). With them you can make them compatible (easier then manually editing .rdf-files). Unfortunately some themes will mess up FF 1.5's GUI, so better be careful.

For updating i recommend running FF 1.5 as root in safemode (as described in the HOW-TO).
