### Post by fritz_monroe on 2006-11-25
I've only used a switch box to connect a single monitor to multiple computers, but I would think they would work fine just the opposite.  You may run into a problem with the connectors.  If not, you would connect the computer to the connector that would normally go to the monitor and the monitors would be connected to the connectors that would go to the computers.

I'm sure you can get a gender changer if there is a problem with the connectors.

### Post by orb9220 on 2006-11-25
> It is just a straight "marchanical" connection?

Yes but that is only if the monitors have the same resolutions and monitor vert frequencies.

Example if I had a 15" and a 17" they would both have to be let's say have vert sync at 60hz and be set for same screen resolutions.

It would not work to run the 15" at 800x600 and the 17" at 1024x768 since this would be a function of the graphics driver and not the monitor.

Hope this cleared up a few things.
