### Post by canadianwriterman on 2005-10-28
gdesklet is available through the repositories, I believe. Just install it though Synaptic. Then go to the gdesklets Web site, [http://gdesklets.gnomedesktop.org/](http://gdesklets.gnomedesktop.org/) , downloads the gdesklets you'd like. Open gdesklets and then drop the downloaded file into the gdesklets window for an automatic install.
