> **andysharpie said:**
> I have the regular 4 workspaces that the system settings provide in Ubuntu 16, but I was wondering if I could create more of them.

Yeah install ccsm (compizconfig settings manager) go to "General Options" and pick "Desktop Size"

Alternatively, install Unity Tweak Tool, go to Work Space Settings and choose horizontal and vertical workspaces.

### Post by kjobermeyer on 2018-02-14
I'm running Ubuntu 16.04 with Gnome3 and could not get the number of workspaces to change via ccsm, the gnome tweak tool, nor some command line methods I found elsewhere. The numbers in the settings forms would change, but they did not actually change for the system. But, I did get it to work when I changed the numbers in the unity tweak tool. Install with `sudo apt-get install unity-tweak-tool`. Search for "Unity Tweek Tool" with "Search your computer" app in launcher. Inside Unity Tweak Tool, got to "Workspace Settings" and make desired changes.
