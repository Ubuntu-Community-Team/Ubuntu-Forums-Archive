### Post by Siamang on 2007-06-18
I use Dapper and just changed 'Internet' to 'WWW' by doing this:

Right click on Applications, select Edit Menus(with Alacarte Menu Editor).

Find 'Internet', right click that and select Properties.

Type in a new name, click Close.

There ya go. Oh, and:
