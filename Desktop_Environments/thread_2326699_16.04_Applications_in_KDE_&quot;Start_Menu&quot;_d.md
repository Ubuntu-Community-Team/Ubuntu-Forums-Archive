### Post by stephan15 on 2016-06-04
After lots of search on the web I found a solution: call
*sudo  glib-compile-schemas /usr/share/glib-2.0/schemas*
It would have that the installers of the respective software perform this call.
Next challenge for me: I will try to file a bug report to the kubuntu team.
