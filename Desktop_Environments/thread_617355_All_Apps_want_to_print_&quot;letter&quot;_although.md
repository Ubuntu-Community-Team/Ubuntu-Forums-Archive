I have a HP Laserjet 4000 N which I installed into Ubuntu via *System -> Administration -> Printing*. There I chose "A4" as Media Size under *Printer Options* because I am in Germany where A4 is the default letter format. Additionally I created the file /etc/papersize and wrote "a4" to it because I read in the forums that this file could matter.

My Problem is: All applications I am using for printing (OpenOffice, evince, Firefox) still use the "letter" format when printing! When I change the format to A4 for example in the printer settings of OpenOffice, everything works fine, but when I close OpenOffice and start it again, the format is set to "letter" again. So, everytime I want to print something, I have to go to the printer settings of the application I am using and switch to A4 format. That's pretty annoying. I want all applications to default to A4. Any suggestions?
