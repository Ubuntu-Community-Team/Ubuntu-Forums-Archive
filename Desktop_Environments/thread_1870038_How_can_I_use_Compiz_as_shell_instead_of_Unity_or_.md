### Post by stinkeye on 2011-10-26
Unity is just a plugin for compiz.
Just disable it with ccsm.
You should still have a terminal with ctrl+alt+t

You may want to install **grun** first as an alt+F2 replacement and set up
a keyboard shortcut for it. 
Cairo-dock is good and has most the stuff you need.
