I don't know about Windows 2000, but you can make it look just like XP with [XPLuna](https://gnome-look.org/content/show.php/XPLuna?content=83797) and GnomeXP icons.  I've never installed the theme before but the Tails OS comes with it as an option, although Tails is Debain but the theme should work on Ubuntu.
