### Post by NICU on 2007-03-20
I have manually entered incorrect resolutions into my x conf file.  It did not physically 'hurt' my system, but x had a problem starting and I had to manually edit the file again through the command line using mono.  I would not suggest using resolutions that your hardware doesn't support, but if you know how to use the command line to edit your x configuration then you should be able to recover.
