### Post by _simon_ on 2007-01-31
Do you have the beryl-manager icon in your task tray?

If so, right click on it, choose "Select Window Manager" and select "Beryl"

It should remember that setting now.

If you don't have the icon, open terminal and type

```

beryl-manager

```

Remember, to have beryl-manager run on login you need to add it to your startup session.
