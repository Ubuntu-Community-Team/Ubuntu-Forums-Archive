### Post by thomasmayer on 2007-02-01
the only way i found to change the size of the thumbnail generated by the gnome-video-thumbnailer is to edit the entry for each desired filetype under:
"gconf-editor->desktop->gnome->thumbnailers-> video@..." (only tested with edgy)

you have to change the "%s" to the desired width in pixels.
e.g.:

"/usr/bin/gnome-video-thumbnailer -s **%s** %u %o" to
"/usr/bin/gnome-video-thumbnailer -s **64** %u %o"

i have no idea where the %s parameter comes from but it would be better to change it there.
remember that already cached thumbnails won't be resized.
