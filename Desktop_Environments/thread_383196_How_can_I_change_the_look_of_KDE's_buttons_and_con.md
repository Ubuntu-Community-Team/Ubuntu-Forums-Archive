kde isn't exactly the easiest to write themes for. i know where there is documentation for gnome theming, but not kde theming. or at least, i never found any when i had kde installed. 
to change the look of the scrollbars, buttons, menus, etc, these are dictated by the styles(they are the equivalent of theme engines on gnome). you can find more style [here]("http://www.kde-look.org/index.php?xcontentmode=14") which you have to compile(although some are available in synaptic). i think you may be able to  alter them further - you will have to have a look at  the xml file in the theme manager themes [here]("http://www.kde-look.org/index.php?xcontentmode=8").
the shape etc of the widgets are determined by the style, whereas the theme manager themes (possibly) alter the widget's dimensions and possibly texture etc.

and no, you can't install gnome themes in kde.
