So even if you like the modal way of working, choosing a manager seems hard.

I thought we could discuss it a bit? I have some spare time during the holidays and I'm looking to try out some window manager using part of this time.

Which ones do you like? Have you been using modal window manager A and praising it, only to find out after a while that you really like B more? Am my observation above wrong? If you use a modal window manager, do you still use a normal one? Have you tried modal window managers but still quit using them and went back to Gnome/KDE/Whatever?
