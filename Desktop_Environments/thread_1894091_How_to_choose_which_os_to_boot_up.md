### Post by westie457 on 2011-12-11
Boot your computer using a LiveCD/Usb in 'Try' mode. When you have a Desktop to use open a terminal (Ctrl+Alt+T) and type in ```
sudo update-grub
``` when that has finished reboot and you should now have a Grub menu to choose from.
