### Post by Airjoe on 2005-12-23
My friend uses Ubuntu, and has XMMS installed. But whenever he opens an MP3, it opens in "Totem" instead of XMMS. Is there anyway to change the default program?

---

### Post by Madpilot on 2005-12-23
Right-click on a file of the type you want to change - in this case, mp3, ogg,  or whatever - and click Properties.

One tab in Properties is "Open With..." - pick the app you want to use there with the radio buttons.

Doing that with .mp3, for example, will get all .mp3 files opening in the selected app; you'll have to change all file types individually.
