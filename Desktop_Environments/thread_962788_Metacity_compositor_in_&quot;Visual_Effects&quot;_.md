### Post by megabytemonster on 2008-10-29
I just found out recently that GNOME has its own compositor, which you can enable by "gconf-editor", then going to apps -> metacity -> general -> compositing manager (set to true).

It's a shame this is so hidden. It offers virtually all the features of the "normal" compiz settings (such as drop-shadows and thumbnails in "alt-tab"), only without needing to have a proprietary graphics driver installed, and uses a lot less CPU/memory.

I suggest that this should be provided as an option in "visual effects", perhaps as "Light", or even just to replace "Normal" since it offers virtually the same with less overhead.

Thoughts/suggestions? I'm not sure how one suggests this change officially.
