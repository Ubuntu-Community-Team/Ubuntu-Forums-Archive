### Post by Jimmy The Clown on 2007-09-09
If you create a much smaller image you can run it thru [www.whatthefont.com](www.whatthefont.com) to try to figure out the likely fonts. Installing fonts may be more difficult, but you can probably find a package that includes it.

To set the font go to System-> Preferences -> Font and set the monospace font to whatever font you desire.
