### Post by tflanders on 2006-12-18
Right click the link to download the unifont.ttf.gz file to your desktop.
Then right click and select extract here.
You should now have a file called unifont.ttf.
Copy this file to ~/.fonts folder inside your home directory
If the folder doesn't exist just create it
```
mkdir ~/.fonts
```

Now open on the gnome fonts manager and use your new font.
