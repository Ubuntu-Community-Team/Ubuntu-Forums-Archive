### Post by Copper Bezel on 2011-10-09
Anyone else experiencing an issue with certain windows - particularly new Chrom(e/ium) windows - not gaining focus when created? If I had a Chrome browser session running anywhere and launched a new window, the new window appeared at the top as expected, but didn't receive the focus, so mouse actions were unaffected but keyboard actions went to the window under Chrome, since it retained the focus until the new window was clicked. If I was using only the keyboard, that meant having to hit Alt+Tab and Alt+Shift+Tab to focus back to the Chrome window before typing an address or search string. 

Deactivating the "Smart" "Focus Prevention Level" in General Options fixed the problem.

Has anyone else seen this? 

Thanks.
