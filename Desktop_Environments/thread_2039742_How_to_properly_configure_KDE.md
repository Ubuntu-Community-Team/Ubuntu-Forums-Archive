### Post by Kalantir on 2012-08-09
So I tried Kubuntu for a little while and then decided to install openSuSE. After messing with openSuSE for awhile, I have determined that I prefer Kubuntu because of the huge amount of community support and the software manager (apt-get vs zypper).  Anyways, since I have started using openSuSE, I have noticed a couple differences in the way KDE behaves. Before I get rid of openSuSE, I would just like to make sure I can reproduce this behavior in Kubuntu.

1. Grid Desktop and Grouping Desktop
In the desktop settings on Kubuntu 12.04 there are several layouts to choose from. They are: Newspaper Layout, Folder View, Desktop, and Search and Launch. In openSuSE I additionally have the option to select Grid Desktop and Grouping Desktop. I really like Grid Desktop and would like to know how to enable that in Kubuntu.

2. System Tray
I have also noticed that the system tray widget on the panel behaves slightly differently. For example, in Kubuntu, when I have Deluge running and it is minimized to the system tray, I have to right-click and select &quot;Show Deluge&quot;(and even then it shows up minimized). I must also follow the same procedure for other programs which are minimized to the system tray. In openSuSE I only need to left click once on the icon and it maximizes the program.
