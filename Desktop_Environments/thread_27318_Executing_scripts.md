### Post by JmSchanck on 2005-04-15
If you have nautilus set up to prompt you for what to do when you open an executable file like a .sh one of the options is "Run in Terminal". If you're running a script that you need to see the output for you can set up ur shell to hold itself open when a command finishes (in gnome-terminal: edit > profiles > title and command ). Another option is applications > run application which will let you specify the command you want to run and you can check the little "run in terminal" checkbox.
