### Post by BIGtrouble77 on 2005-10-07
I need to change the behavior of [ALT] so that the OS essentially disables it's [ALT] + mouse click behavior.  

If I hold [ALT] + left click + move the mouse cursor around it grabs the current window and moves it.  I normally wouldn't have any issues with this except it overrides programs that utilise the [ALT] key + mouse commands.  

For instance, Maya is completely unusable as you need to use [ALT] to work in the viewports.

I went to system/preferences/keyboard/layout options and tried to adjust the alt/win behavior to no avail.  I just want the os to disregard [ALT] as an OS function.

This worked fine in hoary in gnome, but I did have these issues with Elightenment.  I've upgraded to breezy and they've come up again.  I'm pretty sure this is an xwindows issue, but I haven't found the setting yet. Any suggestions?

Go to "System -> Preferences -> Windows"
and select a different Movement key.
