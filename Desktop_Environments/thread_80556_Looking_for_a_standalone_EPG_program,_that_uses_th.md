### Post by Harleen on 2005-11-06
You could try TVbrowser ([www.tvbrowser.org](www.tvbrowser.org)), a nice Java programme, that works really well for me. Because it's java, it requires no installation and looks nice. But it needs SUN-java - at least it won't work with the Java shipped by default with Ubuntu (gcj).
Unfortunally it provides data for german TV only. But because your name sounds somewhat German, I dare to suggest this application.
