### Post by leech on 2006-06-19
Generally you would just run 'sh install-crossover.sh' or whatever the filename is called from the terminal.  Unless of course you want to install it system wide, then you'd need to run it with sudo.

My prefered apps in these categories.

Email: Evolution, yeah, I know a lot of people don't like it for whatever reason, but it is the only one I've seen that actually supports Exchange well, and I had to have a client that worked with exchange at my last job, so I just got used to it.

Web Browser: Can't beat Firefox.  Granted hopefully all the GTK stuff will be built in soon, I did read that all future builds will have it so the buttons look right on pages.

Music Manager: Rhythmbox, again people don't like it for whatever reason.  It does have Ipod support though, at least last I checked.  I think people were complaining because it didn't have id3tag editing.  I also use Listen, I'll give that Exaile! a try though.

IRC: Is there anything worth using besides X-Chat?  Maybe BitchX if you're a console guy...

IM: Gaim, though I do wish they'd hurry and get voice/webcam support for Yahoo.

Image Manager: Nautilus.  But then I don't use photos all that much.  Otherwise I'd just set up a gallery2 on my webserver.

Bittorrent.  Usually I just use the built in one for Dapper (Gnome-btdownload) but sometimes that doesn't work, so I use Azureus.

Media Player: Xine or Mplayer or VLC.  They pretty much play everything.  Hopefully some day Gstreamer will have a legal dvd player for it and everyone can use that, 'cause it'd just make life better.

FTP: gFTP.  Yes, it supports SFTP, SSH, etc.  I use it all the time, drag and drop copies are a snap.

Leech

P.S. I generally try to just stick with as much of the standard Gnome as I can, in case you hadn't noticed.  It does make it easier to support other people who install Ubuntu if they're using the standard install, rather than a ton of different apps.  My biggest gripe now is probably gstreamer, though at one point I did get DVD playback to work.  It's just never been as fast as running either Xine or Mplayer or VLC is.  (I use VLC in Windows over the crappy WMP.)
