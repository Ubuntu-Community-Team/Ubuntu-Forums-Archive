### Post by mchatel on 2006-01-24
In GAIM, look under the Tools menu...
Then go to the Accounts menu option.  GAIM sets a buddy icon for each account, so you have to go in there to set it.

Just choose the account that you want to set a buddy icon for, and modify it.  You should be able to set it now. 

Hope that helps!
