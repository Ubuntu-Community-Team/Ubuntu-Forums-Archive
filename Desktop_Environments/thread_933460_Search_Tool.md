### Post by airtonix on 2008-09-30
tracker will index and allow you to search your documents, pdf's openoffice docs, pidgin logs, etc etc.

give it time to index first.

you will need to install the WV packages to be able to index ms word docs.

```
sudo apt-get install wv
```

I hear abiword has this as part of its package also...not sure i dont use it.

The other solution is that if you have only a few word docs, then open them in openoffice and save them as *.odt files (which are just xml files, which in turn are just ascii files and hence easily indexed by tracker)

### Post by ckonestroh on 2009-05-24
Tracker sports a multilingual word stemmer that might be an advantage if you have documents in many human languages. A word stemmer converts many forms of a word into a common form that can be used to find all variants. For example, removing the "ing" ending in English words might form part of a stemming algorithm. Tracker includes support for tagging files so that you can find them again using the tags. The tracker-tag tool makes it simple to add, remove, and list the tags associated with a file from the command line.
