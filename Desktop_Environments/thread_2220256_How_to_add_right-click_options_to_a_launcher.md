### Post by Frogs Hair on 2014-04-27
The Gnome Shell uses the Muter window manager and not Compiz and Quick lists as those found in Unity have never been one of it's default features. The Fallback and Flashback Gnome sessions don't have this capability either. This is really Gnome issue rather than a Ubuntu issue and there may be an extension for the Gnome Shell only. If using the other Gnome sessions installing a dock could add this functionality. If you choose to file bug report make sure it is for Gnome and not Ubuntu. 

[https://extensions.gnome.org/](https://extensions.gnome.org/)
