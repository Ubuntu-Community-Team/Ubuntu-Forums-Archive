### Post by tom4everitt on 2010-12-07
Hi, welcome!

I've seen users having problem with their Places menu after (presumably accidentally) having switched preferred application for folders. Try 
right clicking on a folder, 
choose 'Open with other application', 
pick 'File Browser' while making sure the 'Remember this application for "folder" files' checkbox is checked. 
Then test whether your Places menu work as expected. 


If this doesn't help, the problem must be something else. One good thing to try is to create a new user (from the System->Admin->Users and Groups menu), and see if the new user has the same problem or not.
