### Post by lagagnon on 2006-08-27
If you found Flurry wasn't in the list after you apparently "fixed" screensavers then it seems to me like you buggered up screensavers, rather than fixed it!!
 
If you run the console command "locate flurry" you will see where flurry is located in your filesystem. Hope it is still there for you: for me it is:

/usr/lib/xscreensaver/flurry
/usr/share/man/man6/flurry.6x.gz
/usr/share/xscreensaver/config/flurry.xml
/usr/share/gnome-screensaver/themes/flurry.desktop
