### Post by marsdenf on 2020-04-29
I am trying to add launcher items to Xubuntu 20.04 panel.  Adding an item from "add one or more existing items to the launcher" works fine, but then trying to edit the item fails.  Also, trying to "add a new empty item" fails, as does "add a new hyperlink".
The failure looks like this:  the window button appears for the new edit, or new blank item, or new hyperlink appears, but the window itself doesn't appear.  Clicking on the window button doesn't show the window.

   I have tried reinstalling xfce4-panel and xfce4-panel-profiles in synaptic, didn't work.  Is there a way to do these operations in a terminal to see what error messages there may be?  Thanks in advance for any help you may have.

UPDATE:  I forgot about the tv which is a secondary monitor for this computer.  The two displays were not mirrored so the window button was in the main monitor, and the window itself was in the tv, which was turned off so I didn't see it.  Now I have mirrored the displays, so everything is back to normal.  Sorry to waste your time!
