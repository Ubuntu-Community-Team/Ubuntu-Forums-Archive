### Post by aysiu on 2005-11-13
It's not writing files to your hard drive. It writes files to RAM (your temporary memory). It sounds, though, as if it might have written something to your palm--that's far more likely what messed it up. I don't see why Ubuntu and Windows would give your palm the same user ID **from the same PC**, though.
