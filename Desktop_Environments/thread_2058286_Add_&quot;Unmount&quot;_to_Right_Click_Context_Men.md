### Post by LewisTM on 2012-09-15
I'm not sure why Thunar only offers to Eject and I don't think you can add a new Unmount item.

What you can do is install and add the Places plugin to your panel.
Set it to show removable media. You can then open the Places menu, right-click on you drive and select Unmount.

Cheers!
