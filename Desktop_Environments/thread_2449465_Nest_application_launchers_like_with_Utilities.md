### Post by Dennis N on 2020-08-27
> Is there any extra step I must do so that these new menu entries are grouped together under the folder in "Show Applications" - just like with Utilities?

You can group items in the "Show Applications" display by creating AppFolders. These appear as single tiles that, when clicked, show all the applications you placed there. See the screenshot, where I have an AppFolder called "Games" and then added my games into it. 

There is a gnome-shell extension to create and manage AppFolders, called "Appfolders management extension".
And there is also a utility called "Gnome Appfolders Manager" that you can install with a .deb file that does the same thing.
