### Post by MasterJS on 2008-01-26
Enlightenment is a window manager like compiz, fluxbox, openbox, and xfce(the difference with xfce is that xfce also has programs for desktop functions like files, etc.). I haven't used enlightenment personally, but it should have some GUI where you can install themes you have download for IT. You won't lose file data because all it is is a window manager, not the desktop environment itself (that would be either GNOME, KDE, or XFCE).
