### Post by stepheny on 2012-05-21
The auto-hide of the Unity launcher is switched off by default in 12.04. To switch it on go to &#8216;Control Gear &#8211;> System Settings&#8217; then 'Appearance &#8211;> Behaviour' and set the auto-hide to on.
