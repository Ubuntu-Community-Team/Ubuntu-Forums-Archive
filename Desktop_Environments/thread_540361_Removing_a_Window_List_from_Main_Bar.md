### Post by pdavit on 2007-09-01
I've added a Window List to the main (top) bar of Ubuntu by mistake. Unlike other items like for example Keyboard Indicator where I can right-click and select Remove From Panel this option is not feasible since by right-clicking you get the options for closing, maximizing and the like of a running application.

How is it possible to remove the Window List?

---

### Post by orb9220 on 2007-09-01
> How is it possible to remove the Window List?

It is there go all the way to left side of window list applet and right click until you have the menu popup that says prefs,about,remove from panel.

Do not right click inside where apps are.
