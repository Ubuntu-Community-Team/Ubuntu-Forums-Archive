### Post by stu on 2006-08-27
Firestarter is in one of the community maintained repositories, so you'll need to enable them first: [http://monkeyblog.org/ubuntu/installing/#enabling_extra_repositories](http://monkeyblog.org/ubuntu/installing/#enabling_extra_repositories)

After you have done that, the command you used above should work just fine, or you can use some of the other instructions on the linked page to install firestarter using the GUI.
