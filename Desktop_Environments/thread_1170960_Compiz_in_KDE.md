---

### Post by krazyd on 2009-05-27
No need for compiz, because the KDE window manager kwin has compositing (wobbly windows, cube etc.) built-in. Go to System Settings > Desktop > Enable Desktop Effects.

For doubleclick: System Settings > Keyboard and Mouse > Mouse > Double click to open files and folders.
