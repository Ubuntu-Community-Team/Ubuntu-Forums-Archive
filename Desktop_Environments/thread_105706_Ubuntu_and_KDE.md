### Post by One Quick Question on 2005-12-19
Hehe, you're trying to run the Gnome and KDE desktops at the same time!
Do this: logout or, if you want, start a new session (sorry, I don't know exactly what Gnome calls that function).
Here it asks you to put in your name and password, somewhere on that screen (depending if you're using kdm or gdm) is a "Sessions" button.  There you can switch between KDE and Gnome (and any other DE/wm/session manager thing, such as XFCE).

That ought to do it.
