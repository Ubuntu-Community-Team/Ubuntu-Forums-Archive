### Post by p_quarles on 2008-02-20
Since Fluxbox and Openbox are, like Compiz, window managers, you can only have one of them at a time. Fluxbox has a built-in dock-like function called the slit, but this certainly isn't at the same eye-candy level as the ones that require compositing. 

An alternative is to use Kooldock, which does not require compositing but acts very much like the OS X dock bar.
