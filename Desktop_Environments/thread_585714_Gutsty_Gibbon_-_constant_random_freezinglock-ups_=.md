### Post by Ruazinn on 2007-10-26
Got an idea!  This can be a dangerous thing, because I might be totally wrong, but here's something you might try if you're having problems with freeze ups.

First a little background info.  I just replaced my Nvidia GeForce 7500LE with an Nvidia GeForce 8400 GS.  It booted right up fine without changing a thing.  But I decided to play it safe -- so I restarted into recovery mode and ran:

dpkg-reconfigure xserver-xorg

like all good video card updaters are supposed to do.  One of the prompts asked about having the card use the kernel for frame buffering.  (I forget the exact wording.)  I chose NO, because I thought this new card probably could do direct rendering just fine.

Well, here's my idea...  I wonder if the source of a lot of the lockups is because of kernel buffering.   It could be a classic case of deadlock that only happens on dual-core systems.

Try running dpkg-reconfigure and answering "NO" to the kernel buffering question and see if your freezes disappear.

BTW, after running the reconfiguration, just type "reboot" to get your system started again.

### Post by ions on 2007-12-30
Just in case that blog post disappears:

# Download the NVIDIA version 100.14.09 video driver from NVIDIA.
# [ctrl] + [alt] + [F1] to goto command line console.
# "sudo /etc/init.d/gdm stop" to stop the xserver.
# Install the driver by
"sh NVIDIA-Linux-x86-100.14.09-pkg1.run". Follow the instructions and decide accordingly.
NVIDIA now provides a utility to assist you with configuration of your X config file.
# 'sudo /etc/init.d/gdm start" to start the xserver.
# Cheat the Ubuntu's restricted driver manager by editing /usr/lib/python2.5/site-packages/RestrictedManager/handlers/xorg_driver.py file.
Insert "return True" line just under
------------------------------------------------------
def is_enabled(self):
"""Returns True if the module is enabled."""
------------------------------------------------------
So that it will read
------------------------------------------------------
def is_enabled(self):
"""Returns True if the module is enabled."""
return True
------------------------------------------------------
Note: This step needs to be redo each time the Restricted Driver Manager is updated.
