Not sure if this is the correct forum - or what the real bug is, but try this on a kubuntu 7.10 machine and see what you think.

Save your work FIRST.

I simply copied over 35 or so pictures from a camera (about 600kb each), selected them all in konqueror or dolphin, and then used the mouse right-clik to open-with Gwenview.

Bug#1: 35 seperate instances of Gwenview started up. Or so it seems.
Bug#2: The machine became very sluggish, the disk light stayed lit, and when it went out, the whole machine was locked.

Firstly I think an out of memory should have chimed in - secondly Linux really should not be able to be killed in this way, it wasn't really a user error either.

So I think there is a possible GUI bug here, possible Gwenview bug, and a definite Linux bug (I could not even open a text console via the keyboard).
