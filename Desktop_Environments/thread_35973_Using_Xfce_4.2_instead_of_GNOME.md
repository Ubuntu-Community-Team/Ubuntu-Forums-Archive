### Post by kori.mendocino on 2005-05-21
xfce is not included in the installation cd, and by default, you will run gnome when you first start ubuntu. after the basic configuration, tho [explained in number of howtos], which includes setting up repositiories, etc, you can easily install xfce via apt-get [or synaptic, if you prefer gtk]. after downloading and installing the necessary packages, you can restart x and in gdm [the login screen], under sessions, you can pick xfce.

although i prefer gnome over most of the visual environments out there, i have to say that using xfce under ubuntu is easy, smooth and sweet, as usual:)


have fun.
