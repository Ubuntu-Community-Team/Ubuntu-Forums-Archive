### Post by phanboy_iv on 2005-11-21
Linux stuff doesn't usually have an "install.exe" program like Windows. Normally you install stuff in Ubuntu by using Synaptics in the System menu, which does everything automatically once you select the application you want. If you want to install something that's too new to be in synaptics, like Firefox 1.5RC, you'll have to do a little command line stuff. Here's the wiki tutorial page I used: [https://wiki.ubuntu.com/FirefoxNewVersion]("https://wiki.ubuntu.com/FirefoxNewVersion")

Remember, RC3 is beta, so do this at your own risk. But you should be okay
