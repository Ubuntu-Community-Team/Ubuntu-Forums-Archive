> I want to use a emerald theme and i have emerald runnin, but the issue is, I used the emerald --replace, and it only applys the theme to my window borders

Emerald is a window decorator. It only governs window borders. This is perfectly normal behaviour. If the website where you got the theme shows any screenshots of other things that you want, you should ask the theme's creator where those other elements come from.

### Post by howefield on 2010-03-12
Emerald only works on the window decoration, not on the panels, (taskbars to use your terminology).

You can add emerald --replace to your startup sessions, this will save you having to use the terminal and losing the decoration when you close it. (Alternatively, run the command in a run box, Alt+F2)
