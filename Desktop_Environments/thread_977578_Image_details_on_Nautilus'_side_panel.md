Hello.

One Nautilus feature I have wished for some time (and have hoped for with each new Gnome/Ubuntu release) is the ability to have basic image details (e.g. width, height) displayed on the application's side panel when an image file is selected. I know one might also rely on an image viewer application and browse images from there to get the desired info, but I'm guessing I'm not alone in thinking it would be quite handy if the file manager would handle it.

Does anybody know if a feature such as this is planned or has at least been proposed for Nautilus?
