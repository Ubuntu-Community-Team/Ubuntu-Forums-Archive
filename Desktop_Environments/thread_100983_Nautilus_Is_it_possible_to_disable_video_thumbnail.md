### Post by everdred on 2005-12-08
I looked high and low for this before someone here answered a thread I had posted a few days ago.

From the main menu, go to System Tools -> Configuration Editor. From there, choose desktop > gnome > thumbnailers. From there, you can enable/disable on a type-by-type basis. (Though some types don't show up there, like HTML.)

You'll need to clear out your icon cache afterwards (~/.thumbnails), to prevent old thumbs from showing up.
