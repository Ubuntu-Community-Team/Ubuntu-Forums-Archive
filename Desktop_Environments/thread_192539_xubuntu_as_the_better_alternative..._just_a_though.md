### Post by bluemuffin on 2006-06-08
i don't have anything against ubuntu or kubuntu. i'm pretty satisfied with my ubuntu desktop, but when i found out about xubuntu i think it is more appropriate to my needs and those of my kind. 

i don't need pretty interfaces, what i want are function and efficiency. these were promised by Xfce so i proceeded to download a xubuntu iso. i want to test it on a retired workhorse of ours, a P3 with a much older motherboard, 128 of RAM, and a 5GB HDD. this machine was not worth a damn with an XP. if i succeed, i hope to do the same to our P2 dust collectors. 

most of our users are using only the basic office productivity suit, a good browsers, and yahoo messenger. i was thinking, if xubuntu will allow these old machines to run the latest software at par with XP, then linux will look more attractive to them since they are on antiquated Win98. 

problem is, i've been working on the P3 for 3 days now and i still have to see xubuntu installed on it. i've finally used my old copy of breezy and is currently upgrading it to dapper. hopefully, there is a way from ubuntu to get into xubuntu. quite a run around but i'm not throwing in the towel yet.

i know, there are ways to get xubuntu on but it becomes complicated. ubuntu is about Linux for people like me and xubuntu is about simplicity and functionality, that's what ubuntu developers said. 

i believe the problem with my P3 is the graphical installation interface. It seems that it just don't have the juice to run xubuntu and install it at the same time. so i was thinking if the dear good xubuntu developers can make a simple installation disc for xubuntu like what we used with breezy and older versions, as an alternative to the live CD installer.

Such a simple, and light weight, installation procedure would be more appropriate for the legacy machines that Xfce and xubuntu were designed for. it would also be true to the calling of ubuntu. 

just a thought...

peace!

:-D
