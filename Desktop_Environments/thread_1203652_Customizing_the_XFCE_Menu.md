### Post by Brandon Williams on 2009-07-03
If you're using Jaunty, then editing the menu is a manual process that requires creating or editing various files using a text editor. [This post](http://ubuntuforums.org/showthread.php?p=7240572#post7240572) describes how it's done.

If you're using a previous release, then look for the 'Main Menu' app in the System menu (I think that's where it is).
