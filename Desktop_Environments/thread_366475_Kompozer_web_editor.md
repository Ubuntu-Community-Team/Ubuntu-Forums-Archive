### Post by chggr on 2007-02-21
If you are using Gnome:
1) Right click on "Applications"
2) Edit Menus
3) New Item
4) Fill in the name, choose an icon, write a comment and the command to be used in order to open the program (most of the times it is simply the name of the program)

That's it.
