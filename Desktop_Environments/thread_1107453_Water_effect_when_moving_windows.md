### Post by Lunx on 2009-03-26
If you have compiz settings manager installed, you can select it there and take note of the key bindings to run it. It says Initiate and the default is Ctrl + Super. Shift + F9 gives a really cool rain effect. Have fun :)

### Post by ooburns on 2009-03-26
The ripple effect for moving windows, sadly, is not part of the official Compiz packages. There exist patches that enable it if you compile Compiz from the source, but there's no way that I know of to enable it without compiling it yourself. :(
