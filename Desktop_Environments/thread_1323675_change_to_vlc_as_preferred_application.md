### Post by mrboojive on 2009-11-12
In Ubuntu, you change which application opens a type of file by using the file manager. Go to the file manager, find a movie file, right click on it and select properties, then click on the "Open With" tab and choose VLC. In the future, all movies of that type will open with VLC.
