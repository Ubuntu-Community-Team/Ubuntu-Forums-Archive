That walkthrough is amazing--screenshots and everything. I'm a bit concerned about this statement, though > I have 5 years worth of stuff saved on my C: drive and can't back it all up. Even though the partitioning tool is good, your statement is a bit like saying *I've never ridden a motorcycle before, but I can't afford a helmet*. Even if I teach you everything there is to know about motorbike riding, if you get in an accident, you're screwed.

Please, please, please--this is five years of work you're talking about. Five years. Think about it. Even if the chance of that five years of work being damaged is .001%, do you really want to take that chance? Find a way to back it up! Burn a gazillion CDs. Borrow a friend's external hard drive. Hell, *buy* an external hard drive. US$100 or whatever may seem like a lot at first, but what is that compared to the five years' worth of data you could possibly use.

Wear a seatbelt. Wear a helmet. Be safe. Back up your work.

### Post by anil_robo on 2005-10-19
**1. Backup! Backup! Backup!**

I know one can accumulate a lot of data in 5 yrs, and it can be a pain to back it all up. I had over 40GB of data lying in my 80GB hard disk, and I was wondering how would I back it up prior to partitioning. But then good sense prevailed, and I soon found out that most of that data was music (which I had ripped through my own audio CDs), and some downloaded programs (which I could download again from the internet anyway). So I made a list of downloaded programs (take screenshots of your windows explorer by pressing PRINT SCREEN if you're lazy :rolleyes: ). Then Print the screenshots.

If that data is really important (e.g. if there is no way you can get it back) then burn it all to CDs or DVDs whatever you can. If you have no way to burn it into CDs, get yourself a new hard drive, and install Ubuntu on it. Ubuntu would not require more than 2GB space to begin with, and you can get yourself any cheap hard drive from ebay etc.

2. If you are not willing to burn CDs or DVDs, and are also unwilling to buy a new hard drive, then use **Partition Magic.** Mind you, even this program is not 100% safe (the company itself says that), so I still would recommend that you backup your most essential data when you're using partition magic.

3. If you do not have partition magic, or are unwilling to use / buy it, there may be a workaround (but I haven't tried that yet - so use at your own risk). Boot into Ubuntu from Live CD, install **GParted** (I dont' know if you can install that from a Live CD - try the command "sudo apt-get install gparted"). If the command doesn't run, well, sorry! If it does, you get a program similar to partition magic, and use it to repartition your hard drive.

4. When you're done, install ubuntu.

5. Write back to me, to this forum, and **let us know your experiences.**

**6. Have a nice day.**
