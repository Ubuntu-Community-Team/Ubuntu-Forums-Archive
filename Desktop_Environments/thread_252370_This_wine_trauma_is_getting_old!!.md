### Post by funkydan2 on 2006-09-06
After installing wine did you run *winecfg*?  (Open a terminal to run it).

Also, did you delete your .wine directory before installing a new copy of wine?  You need to do this to purge your old settings.

Also, you might like to try running a newer version of wine - see [http://winehq.org/site/download-deb](http://winehq.org/site/download-deb) for details of getting the newest version of wine in Ubuntu.

### Post by meng on 2006-09-06
I am doing my best to make suggestions and explanations based on a very brief description of your problem. Perhaps if you specified the name and location of the file you downloaded, I could help you better.
Terminal is the program that allows you to run a command-line interface in a window within your GNOME/KDE. Change directory to where the file is downloaded with the cd command, then type "wine [name of exe file]"
Nautilus is the file browser. One way of opening Nautilus is by clicking on the button at the bottom of the Firefox download window, to the left of the "Clean Up" button.
