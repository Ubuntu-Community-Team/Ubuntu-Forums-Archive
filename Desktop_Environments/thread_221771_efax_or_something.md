### Post by Sef on 2006-07-24
I don't know how well this works or not, but in the universe repository there is 'asterisk-app-fax'.  If you have enabled the repositories ([enabling the repositories.]("https://help.ubuntu.com/community/Repositories/Ubuntu"), then you can either get by doing a search in Synaptic or by using the terminal.

Terminal:  Applications > Accessories > Terminal

then type

sudo apt-get update (enter)

then

password

then

sudo aptitude install asterisk-app-fax (enter)
