So far as #2, here are some customization instructions:
[http://askubuntu.com/questions/29553/how-can-i-configure-unity](http://askubuntu.com/questions/29553/how-can-i-configure-unity)

> Open up the Configuration Editor by Pressing Alt + F2 and typing gconf-editor

Navigate to apps -> compiz-1 -> plugins -> unityshell -> screen0 -> options

Set "Hide Launcher" to "Never".


I do have to say though, the default launcher behavior is one of my favorite things about Unity.  Give it a chance.
