### Post by aysiu on 2007-05-22
Go to KMenu > System > Adept Package Manager, search for *firefox* in the **Quick Filter**, mark it for installation with a right-click, and then **Commit Changes**.

Read more here:
[http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)
