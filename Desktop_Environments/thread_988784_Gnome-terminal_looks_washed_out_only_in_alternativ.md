### Post by nivix on 2009-05-26
Old thread, I know, but Google led me here and I would have found out earlier if the answer was to be found here. Just set the default background color in the Tango palette in gnome-terminal to plain black.

Edit -> Profile Preferences -> Colors -> Palette entry 1
