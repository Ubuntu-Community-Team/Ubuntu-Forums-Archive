### Post by Dark_X on 2007-11-21
Go into synaptic package manager, search for xgl, install xserver-xgl, search for compiz, install compizconfig-settings-manager and emerald, and click apply.
The settings manger will allow you to enable the cube and many other plugins.  Emerald is a decoration manager.
