### Post by kingsidy on 2006-03-04
if you want to add a new starter item, just drag the icon from the menu for example and it will add it to the starter bar location. for example if you would like to add gaim to the starter bar, just click it from the menu and drag it and drop on the starter bar. Do not forget to go in the system menu> session, click on the start up tab, then click on add program and add gdesklet to it. that would allow it to start automatically whenever you boot.
