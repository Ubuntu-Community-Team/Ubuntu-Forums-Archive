### Post by jacksaff on 2008-09-05
To run kde 3 programs some parts of kde3 will need to stay on your machine. Use a package manager like synaptic to remove programs you no longer want. It will make sure that the bits of kde3 that your remaining kde3 apps need won't be removed. There is no harm in having both at the same time though, except that it will use up a bit of harddrive space.

### Post by jacksaff on 2008-09-05

When kde3 goes kde4 apps will lose their kde4 tags.
