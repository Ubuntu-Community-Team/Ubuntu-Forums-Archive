### Post by 3Miro on 2010-12-15
Are you using compiz or metacity? If you are using compiz (desktop effects enabled), you can install CCSM (compiz-config-settings-manager) from the Software Center. From System -> Prefs -> CCSM you can got to windows management and I think you can setup windows rules or another plugin like that. It is somewhat complicated, but everything that you say can be done.

If you are using Metacity, you can try this:
