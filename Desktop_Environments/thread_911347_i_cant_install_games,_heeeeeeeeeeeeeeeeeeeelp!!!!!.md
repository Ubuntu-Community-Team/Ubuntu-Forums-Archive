### Post by yjwong on 2008-09-05
Unfortunately, it seems that VDrift is only available to Ubuntu Feisty Fawn's repositories, since VDrift cannot be re-distributed due to licensing issues, as detailed [here]("https://bugs.launchpad.net/ubuntu/+source/vdrift-data/+bug/139843").

You can however, download the VDrift sources from [Sourceforge.net]("http://sourceforge.net/project/showfiles.php?group_id=137283&package_id=150901"). These require you to compile the program before you'll get a working game. If you want to avoid the complexities, you can download the "vdrift-2006-10-06" package, which does contain a Debian (.deb) package.
