### Post by ajgreeny on 2018-11-08
You are using the Applications menu which does not, and never has had a search option showing, as far as I remember.

To get the search facility you need to use the whisker menu which is normally the default in the panel, though can easily be added back if for some reason it is gone, or you removed it without realising what you'd done.
