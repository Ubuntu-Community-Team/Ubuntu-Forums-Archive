### Post by RAOF on 2005-11-09
IPv6 is the next generation ip protocol.  Basically, it replaces the 4*8 bit IP addresses (234.23.15.2, for example), with much longer addresses.  This is a good thing(tm), but is unnecesary at the moment - for example, Windows (AFAIK) doesn't support IPv6.  At the moment (and for the next couple of years, probably) you don't need it.
