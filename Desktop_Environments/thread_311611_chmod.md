### Post by hraposo on 2006-12-03
How can use chmod to change permissions of all contents in a folder?

---

### Post by derek on 2006-12-03
Use -R i.e. chmod -R 777 /home/derek would give read/write privileges to everyone for all the files and folders in  /home/derek

---

### Post by bapoumba on 2006-12-03
Please **do not** change permission of any file in the root (/) file system.
