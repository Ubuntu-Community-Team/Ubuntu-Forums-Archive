If you want to remove games, it will only uninstall "meta" packages, like ubuntu-desktop. The point of ubuntu-desktop is to make sure that the whole system is installed (you will want it if you are going to upgrade your system). Plus, if the games or anything else are ever needed as a dependency, Ubuntu will automatically install them.

Also, if you want Firefox 1.5, as well as a host of other things, get [Automatix.]("http://ubuntuforums.org/showthread.php?t=66563") It's an awesome program that can configure your system automatically, as well as install common softwares. It also makes it easier to upgrade packages by adding repositories like the backports.
