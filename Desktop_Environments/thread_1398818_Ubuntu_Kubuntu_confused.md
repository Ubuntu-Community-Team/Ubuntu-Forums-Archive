### Post by adam814 on 2010-02-05
1. When you update to a new version of Ubuntu it will upgrade ALL the packages on your system.  It will update both your KDE and GNOME desktops.  You won't have to choose unless you do a clean install (and then if you want you can just install the other desktop anyway).

2. Shell commands will be the same for the same shell.  Your shell should be the same for both environments as they're usually configured on a per-user basis with a default to /bin/bash

3. I believe KDE uses KWin compositing by default rather than using compiz.  Do you have it working in GNOME, but not KDE?
