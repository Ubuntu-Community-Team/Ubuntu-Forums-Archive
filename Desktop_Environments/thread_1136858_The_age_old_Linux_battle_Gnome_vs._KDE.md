> **davbren said:**
> I ask the question, why have we not seen a merger yet? a Gnome platform with KDE eye-candy?

They are entirely different codebases, not to mention they use different GUI toolkits, and languages.

Also note that there is a *lot* more to the differences between KDE and Gnome than how they look. One could compare it to the difference between programming for Windows and for Linux (though not that extreme).

> Is it really *that* difficult to compile a program for two different DE's? I am currently using a KDE theme for Gnome, in an attempt to get that so needed KDE feel to my friendly Gnome. In doing so, I sorta feel like I'm using Mac4Lin, 'Close but no cigar'

Difficult, no. Compiling is simple. It's getting the same code to work on two different platforms that is hard. I can only think of one theme that does that, QtCurve. And I think it is two separate themes that just look alike, though I don't know.
