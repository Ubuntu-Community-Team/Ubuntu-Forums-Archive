### Post by russellc on 2007-02-24
If you didn't install it there is no need to uninstall it. What you're seeing is just its availability in the repositories, meaning you can choose to install it. There is no loss in terms of disk usage besides a few bytes/kilobytes of text to display that entry in your package manager (Synaptic).
