What is the actual console command for the screen lock that comes with gnome?

I am running FVWM but I really would like to use the screen lock that comes with gnome. I would like to know the *actual* console command that launches screenlock, so I can implement it into my FVWM config.

### Post by psychicdragon on 2005-12-06
Try this:
```
xscreensaver-command -lock
```
