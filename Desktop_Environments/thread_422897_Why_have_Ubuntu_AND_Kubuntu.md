### Post by AusIV4 on 2007-04-28
Ubuntu, Kubuntu, Xubuntu, etc. all use the Ubuntu repositories, so all they're all capable of running the same software. The difference is the default software. Kubuntu comes with primarily QT apps, Ubuntu comes with primarily GTK apps. While Kubuntu is capable of running GTK apps, it's more memory efficient to run QT apps.

I don't think it would be possible to fit two entirely different sets of applications on one CD without:
A) Reducing the number of applications.
B) Requiring an internet connection for the installation.
OR
C) Using applications with an inappropriate desktop environment by default.

A and C would both reduce the experience of the distro for everyone, and B would make it difficult for people who don't have the internet available.

It would be nice if there were a DVD or something that would allow you to choose from a variety of desktop environments, but it just wouldn't work on a CD.
