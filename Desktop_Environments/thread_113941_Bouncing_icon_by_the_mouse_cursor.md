### Post by ajgreeny on 2006-01-07
If you use KDE do the following.  Sorry, but I'm not sure how to do it in Gnome.
Open Control Centre. or do alt+F2 and type in the box kcontrol, which will open it if you can't find a menu item.
Look in Appearance & Themes/Launch Feedback, where you can set the time for the cursor to blink or bounce or whatever you want, and also the taskbar notification time for the item to show in the taskbar.
Set these to what you want and click OK.  All done.
