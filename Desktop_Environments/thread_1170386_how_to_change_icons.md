### Post by pro003 on 2009-05-26
download icon pack from gnome-look.org

open terminal, type:

```
gksu nautilus
```

enter passwd

navigate to /usr/share/icons

paste the package there and extract it among other default icon packs, go to desktop right click appearance settings themes, customize upper tabs theres icons tab, go there and suite yourself.
