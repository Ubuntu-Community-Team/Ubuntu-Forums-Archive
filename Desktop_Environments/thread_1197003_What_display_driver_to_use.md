### Post by gradinaruvasile on 2009-06-25
Ubuntu already has the xfree radeon driver. It will be set up by default for your card. There are proprietary ati drivers available but there are not available for the latest ubuntu (9.04) for your card because Ati dropped support for the older cards in the newer versions of linux kernels.
