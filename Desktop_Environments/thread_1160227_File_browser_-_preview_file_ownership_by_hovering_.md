### Post by kanikilu on 2009-05-15
I don't think there is a way to add a tool-tip in nautilus. Check out this bug report, which also has a couple links to ideas on brainstorm:

[https://bugs.launchpad.net/ubuntu/+source/nautilus/+bug/277604](https://bugs.launchpad.net/ubuntu/+source/nautilus/+bug/277604)

For what it's worth, an alternative file manager called **xfe** has tooltips that include ownership in all the views. Panel > Small icons, is similar to nautilus' "compact view".
