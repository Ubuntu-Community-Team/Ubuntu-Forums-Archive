### Post by firephoto on 2007-04-26
I've always used KDE with my dual display setups. It handles panel, window, and notification placement all on a per display basis. The desktop wallpaper is also able to be set per display if you'd like. Newly launched apps can launch on the display with the mouse cursor if you want or go where they were last time. There's probably some other things too but the default behavior is what I use except that I set it to have new apps launch where the mouse cursor is.
