---

### Post by SGC on 2005-07-31
- Go to the web page that you want to make it a home page
- From **Settings** menu choose **Save view profile "web browser..."**.
- Click on the save button

---

### Post by hod139 on 2005-07-31
Go the page you want as your home page, then Settings->Save View Profile "Web Browser" ...

Select Web-Browser and make sure the Save URLs in profile is checked, then click save.
