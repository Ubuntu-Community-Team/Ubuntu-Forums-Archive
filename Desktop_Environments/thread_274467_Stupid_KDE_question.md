### Post by jalefkowit on 2006-10-09
Hey everybody...

This is probably a dumb question, but it's stumped me, so I figured I'd see if I'm the only one :-)

I'm running Kubuntu Edgy, and I've been playing around with configuring KDE.  One thing that annoys me about the default settings is that the Taskbar panel applet shows **all** running applications, no matter what virtual desktop they're running on.  I'd prefer it to only show me the ones on the current desktop.

[According to the KDE docs]("http://docs.kde.org/stable/en/kdebase/kicker/taskbar-settings.html"), you can supposedly set things up to act that way by choosing "Configure Taskbar...", then going to "Taskbar" in the menu that pops up, then un-checking "Show windows from all desktops".  Sounds easy enough...

... except that I don't have that checkbox in my Taskbar options.  It's just not there.

What's the story?  Is there somewhere else I should be looking for this option?  Any help would be appreciated...

thanks!

---

### Post by kerry_s on 2006-10-09
kcontrol> desktop> taskbar> uncheck the box next to "show windows from all screens"

### Post by jalefkowit on 2006-10-10
