### Post by JakeOfOz on 2010-03-31
try pressing alt+f2 and then typing 'knetwork' you should be able to pick the knetwork manager.

but i advise you, when you have internet ofcourse, to install wicd. that will give you some better support, especially with the hidden wireless networks. has a better interface too...
(type sudo apt-get install wicd in the konsole)
