### Post by dyoung66 on 2006-11-02
You could probably run a scheduled task to run every n minutes/hours to change desktops. In kde, KAlarm lets you schedule a command(among other things) so you could try this way(dunno about Gnome equivalents, I imagine it has something similar). If you know the hotkey to change desktops, you could setup the scheduler(KAlarm or XP Scheduled Tasks) to "press" it at each time interval. Im not an XP person, so I dont know if you can parse keypresses straight to Task Scheduler.
Settting up ubuntu with 3 plasma monitors might take some work.. but by all means go for it if you're feeling keen. :)

Try googling a way to parse hotkeys to Task scheduler in XP. (Provided you can use a hotkey to change virtual desktops).
Hope this helps.
