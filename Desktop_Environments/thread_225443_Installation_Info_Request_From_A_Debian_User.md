### Post by Ziox on 2006-07-29
i suggest doing a clean format and install, because even though devian and ubuntu are similar, they are not the same. Some debian apps/.deb files won't work on ubuntu, and vice versa.  For you Dell, download the alternate CD, and i would assume that you have a Intel or AMD none 64 processor, which means that you would get the alternate_i386.iso:

[http://mirror.cs.umn.edu/ubuntu-releases/6.06/](http://mirror.cs.umn.edu/ubuntu-releases/6.06/)

many people find ubuntu detect hardware very decently, except for wireless cards, but since there are so many people in this forum, chances are you'll find the solution to any hardware problems you'll encounter.

Hopefully you'll have a nice Ubuntu experience :)
