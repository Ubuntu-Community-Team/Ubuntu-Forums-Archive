### Post by Shazaam on 2009-11-20
tint2 is basically a "window-list" bar with a clock. Great for those dock apps that don't have a decent one included (or one that is very small). When I was running Cairo-dock and a few others it came in very handy. Either autohide the panels or remove them and run your favorite dock and tint2 together.
For configuration go to /home/yourname/.config/tint2/tint2rc The "." in the folder name means it's hidden so CTRL+H to unhide.
