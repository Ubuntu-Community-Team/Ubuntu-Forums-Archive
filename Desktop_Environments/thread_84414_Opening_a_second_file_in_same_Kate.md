### Post by shinygerbil on 2005-10-31
There's a check box in the General Options menu which says "Allow Kate to use more than one UNIX process". If you uncheck this, you should find they'll all open in the same instance of Kate.

Funnily enough, I had the opposite problem - I wanted them all to open in separate windows :P
