### Post by jmanning on 2008-11-09
I had the same problem. In my case, Compiz was controlling the number of desktops, so you have to turn of visual effects, set the number of desktops and keyboard shortcuts to your liking, then turn back on visual effects.Try this: 

Go to System>Preferences>Appearance
     Select Visual Effects tab
       Set to None

Right click on the workspaces applet, set your desktops how you want them. Set your keyboard shortcuts as normal, System > Preferences > Keyboard Shortcuts.

If you want, turn Visual Effects back on: System > Preferences > Appearance
Visual Effects
