### Post by Frogs Hair on 2010-12-29
The theme failed to install after extracting the folder from the zip . It is possible to use the theme icons by using the customize icon option found when right clicking the launchers on the dock.

1. Extract and save the apple awn folder in home somewhere .
2. Right click the (task manager icons only) that you want to change.
3. Select customize icon and navigate to the apple awn folder and open.
4. choose icon and select open.

There is a pending question as to how to install this theme at the link you provided. The awn how to provided with the icons doesn't appear to be compatible with the version of awn I am using.
