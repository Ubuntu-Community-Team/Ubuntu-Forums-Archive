### Post by ajgreeny on 2013-09-06
I suspect the SIS graphics and suggest you try using the various boot options available by hitting F6 after the first screen from the CD bootup. You may need to use any key at the screen with the keyboard and man icons at the bottom, which should get you to a language selection page annd then you'll see all the F# options along the bottom of the screen.
