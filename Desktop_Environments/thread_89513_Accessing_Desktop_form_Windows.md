### Post by HermanAB on 2005-11-12
Essentially, there are two ways: VNC or SSH.

To use VNC, follow this guide:
[http://www.aerospacesoftware.com/vnc-howto.html](http://www.aerospacesoftware.com/vnc-howto.html)

For SSH, you need to install Cygwin using this guide:
[http://www.aerospacesoftware.com/cygwin-howto.html](http://www.aerospacesoftware.com/cygwin-howto.html)

then establish a ssh connection with X forwarding and launch any program, simply by typing its name on the command line.
