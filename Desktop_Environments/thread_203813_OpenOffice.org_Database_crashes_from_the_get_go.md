### Post by doubled on 2006-10-28
Another issue:

I have tried -- and failed -- at setting a drop cap with some of the fonts in OpenOffice.org such as Bitstream Charter and Century Schoolbook. OOo stalls and closes with these and other (but not all!) selected fonts. It is a problem common to the Linux kernel 2.6.17 and every NEW Linux release version I have tried (Mandriva 2007, PCLinuxOS 0.93a, (K)Ubuntu 6.10, Fedora Core 6). The common thread is the Linux kernel version, not the OOo version. It is not an issue in Window$ or Mac.

I'm disheartened to read of all these other problems. I had no idea there were so many. It would appear that the quick, temporary solution is to revert to the 2.6.15 kernel, as the problems appear with OOo 2.0.2 - 2.0.4.

I hope these non-technical observations help get to a solution. I have reverted to Ubuntu 6.06 until these issues are resolved. OOo is too important to my use of Linux to have to struggle with this stuff.](*,)
