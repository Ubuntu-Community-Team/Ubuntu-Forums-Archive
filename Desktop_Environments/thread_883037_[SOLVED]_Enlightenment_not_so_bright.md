### Post by ilrudie on 2008-08-07
Unless you uninstalled gnome it should be under session under the options button on the login screen.

If its not there you probably uninstalled gnome and to get it back the command will be 
```
sudo apt-get install ubuntu-desktop
```.  To run it hit <ctrl>+<alt>+<f2> once you are logged in to get to the console.  Run the command and next time you login you should have the option to choose gnome as your session.
