### Post by 5-HT on 2006-07-31
Yup, it's possible -- all you need to do is configure the Update Manager to automatically download/install security updates without confirmation.

 Right-click the Update Manager -> Preferences -> Internet Updates

or

Using Synaptic: Settings -> Repositories -> Internet Updates.
