### Post by Haegin on 2006-01-07
open synaptic and search for "gaim". Install the program called gaim that appears in the list. When it is finished installing close synaptic and open gaim from the internet menu. Choose to add an account and set the account type to msn. The Screen name field is your msn address (whatever@hotmail.com) and Alias is what you want to see yourself as. This is different to your nickname as nobody else sees your alias. Set your friendly name (nickname) from the accounts menu after you have logged in.
