### Post by corsairgt on 2009-01-08
When you start up Ubuntu v8.10, it prompts you to select a language interface. Once you select a language, the OpenOffice interface will also follow. In my case, when I select Traditional Chinese, OpenOffice's interface will also switch to T Chinese automactically.
