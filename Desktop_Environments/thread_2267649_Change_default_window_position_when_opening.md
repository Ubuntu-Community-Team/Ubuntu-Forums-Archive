### Post by buzzingrobot on 2015-03-03
If you are using Unity, you can install Compiz Config Settings Manager. Run it and you will find an option to center new windows in the "Place Windows" panel. Be sure to close it correctly using the "Back" and "Close" icons.

Unity is, in fact, a Compiz plugin, so be very careful about altering or randomly experimenting with the many other options that tool exposes lest you break Unity.
