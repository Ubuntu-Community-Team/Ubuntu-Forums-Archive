### Post by jacksaff on 2009-02-24
If you have the kubuntu-experimental repository enabled then "sudo apt-get install kubuntu-desktop" should install kde4.2 (plus all the other stuff kubuntu has by default). Otherwise the same command will install kde4.1 which is the version of kde that comes with 8.10.
