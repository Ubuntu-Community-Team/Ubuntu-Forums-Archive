### Post by sikander3786 on 2010-10-19
AWN and Cairo are just the docks that sit anywhere on your screen and act like a panel/application launcher.

Compiz is enabled by default as it was in 8.04.

Compiz + Emerald + AWN/Cairo-Dock is a good eye candy combination.

Metacity is a lighter window manager if you prefer not using compiz but then you'll also not be able to use any docks at all.
