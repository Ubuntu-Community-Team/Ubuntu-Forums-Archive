### Post by mcduck on 2006-02-01
Right-click on the top panel and select 'New Panel'. A New panle will appear, if it's not on bottom you can drag it there. Then right-click it, select 'Add to Panel' and add 'Window List'. I think in the default setup you also have 'Workspace Switcher' and 'Show Desktop' in the bottom panel, so you can add them too. (or whatever you like ;)). You can drag things around in panels with middle mouse button or right-click and select 'Move'. 'Lock To Panel' will lock the item's place in Panel.

If you right-click the panel and select 'Properties', you can set panel to autohide, fit size to items in it or change it's size. You can also set it's coulour and transparency or choose your own background image.
