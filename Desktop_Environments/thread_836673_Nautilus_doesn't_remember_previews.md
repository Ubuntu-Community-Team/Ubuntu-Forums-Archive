I upgraded from 7.10 to 8.04 when it came out.. I had this problem back then and filed it as a bug report, but the conclusion was that this was something to do with my own configuration =(

Basically what's happening is this: Nautilus is not remembering the higher-resolution previews of video / picture files, just the normal-resolution ones. This happens when the zoom level is at or above 150%; under that, Nautilus uses the lower-resolution previews and is fine. When the folder first loads, it loads with the normal resolution previews, then scans each file and replaces the previews with the higher resolution ones as it goes.

This is getting really, really, really annoying.. does anyone have any ideas on how I can stop it from doing this?
