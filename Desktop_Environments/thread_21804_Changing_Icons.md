### Post by arrizaba on 2005-03-24
Hi. I think you just have to go to the icons directory of the theme you are using and there you should replace the pictures of the icons you want to change. 
go to Desktop Preferences ->  Themes , then choose Theme details and you can know which icon theme you are using (I think by default it is the Human one at [COLOR=Sienna]/usr/share/icons/Human/[/COLOR]). In the [COLOR=Sienna]scalable[/COLOR] directory you can find the default icons used for several applications/mimetypes/devices and so on. Just replace the ones you don't like by others you prefer more. 
You can also try to download some cool icon packages from: [COLOR=DeepSkyBlue]www.gnome-look.org/[/COLOR]. 

Enjoy!  \\:D/
