### Post by Perfect Storm on 2005-01-22
Okay

GDM is your login screen (if you prefer graphical login)
GTK2 for theming your ubuntu windows etc.
Icon is Icon theme(s)

Here's how you do to install GTK2:

Download the theme you like (and save it)

Click 'Computer' tab. Go to ' Desktop Preferences' almost in the bottum of the menu is something called 'Theme', click it.

Now you see a window with some themes in it and some options. Click 'Install Theme'. Now browse to the saved theme file (you don't need to extract it, the program do it for you) and click install. Now if the theme doesn't appear in the window (some does other don't) Click 'Theme Details' . Here you can choose the 'Controls' theme and 'Window Border' theme. 
Go in both tabs and select your newly install theme or mix it as you pleased.


Icons is a diffrent matter. As you might see there's a tab in the 'Theme Details' menu called Icons. It's a bit broken when it comes to install Icons, so we have to do it manually.
Click the 'Icon' tab, choose 'Go To Theme Folder', a window appear. Now extract the Icon theme you downloaded to this window and bingo! It's now avaible for use in the Icon theme selection :)

Sound:
'Computer' tab, 'Desktop Preferences' and click 'Sound'. Now choose the 'Sound Event' tab. Here's a list of Events and which sound files its using.
So the Sound theme you are downloading needs to be extracted before you can set which sound to where.
Click on the event you want to change and click the 'Sound file' browser to set the path to the specific wav file.

Done :)
