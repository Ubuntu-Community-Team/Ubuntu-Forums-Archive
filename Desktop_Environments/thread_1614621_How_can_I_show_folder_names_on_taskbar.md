### Post by 3Miro on 2010-11-05
What part of this have you already done. The things on the top and bottom of your screen are usually called "panels", "taskbar" is an applet that lists all the open windows (like the one that you have by default on the bottom panel). You can right-click on any panel and select "new panel", when the new panel shows up, you can right-click on it to go to Properties and move it to the left/right/top/bottom. 

If you want a shortcut that open a specific folder, you can right-click on the panel -> add to panel -> Custom Application Launcher and you can give it the command:
```
 nautilus /path/to/whatever/folder 
```

The bad news is that I don't know how to show text next to the icon. The only way I could think of is to add "character map" applet and then play with its pattern properties. This is ugly and should be a last resort.
