Twinview isn't an option across multiple devices; only across multiple screens running off of one device (e.g, a dual-head graphics card).

What you need is Xinerama (which won't work with Compiz).

If you're really interested in using more than 2 monitors, my suggestion would be to go back to Gutsy, which carries an older version of X that actually works with multiple x sessions, and do that.

I'm in that situation with jaunty, which is very, very buggy with this kind of setup, generates a persistent cursor for each display (I have two extra cursors that don't do anything at any point in time), and will only start most of the GNOME apps on screen 0.

### Post by Lartsa on 2009-05-09
You need to have similar GPUs for xinerama (see 1st requirement on [NVidias docs]("http://us.download.nvidia.com/XFree86/Linux-x86_64/180.51/README/chapter-14.html")). I had Nvidia 9400GT and 7600GS, on which xinerama didnt work. After buying second 9400GT (replacing 7600GS), xinerama started to work.

As far as I know, xinerama doesn't support compiz (or compiz doesnt support xinerama), so choices are either 2 screens with twinview&compiz or 3 without compiz.
