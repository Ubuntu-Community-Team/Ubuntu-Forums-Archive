### Post by CelticWarrior on 2020-01-31
Not a downgrade, just different desktop environments and a few of different apps/tools. You can install Ubuntu desktop if you want and then you'll have the option to use one or another at login.

Now, if you really want the vanilla Ubuntu only, the best way is to install the vanilla Ubuntu. That shouldn't be a problem if you have backups (and you should have them, obviously).
