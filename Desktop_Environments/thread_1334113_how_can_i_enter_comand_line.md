### Post by MelDJ on 2009-11-22
if you are using grub 1.97 beta or if you are using karmic, just pick the recovery console/mode and boot into it. 
if you are using an earlier version of ubuntu, press esc before grub boots into ubuntu and the select recovery console/mode
see: [https://wiki.ubuntu.com/RecoveryMode](https://wiki.ubuntu.com/RecoveryMode)

### Post by onyxwolf on 2009-11-25
Worse case scenario grab the liveCD you installed from (valuable tool, so if you don't have it make it) and boot from it. Select try ubuntu without changes to computer. There (unless you chose to encrypt your home folder) all you have to do is pull up terminal, mount your hdd disk drive and a backup device (even if its a crud load of cd-rs) and copy over your files! Long live liveCD! I've never had to use it for backup but I've had to repair my grub with it a time or two.
