### Post by thrice_loved on 2009-07-20
Ok- in each screenlet's settings there is an option called "treat as widget" - click that and switch it on.

This moves the screenlets to the widget layer - meaning it won't be visible all the time, but will be when you have enabled the widget layer.

Doing this is easy. First, install the compiz settings manager, just search for "compiz" in 'add and remove programs'. Open the program and you will see that there are a whole lot of options that you can play with (desktop cube, raindrops, writing fire, transparent windows etc.). Find the one called "widget layer" - click on it and enable it.

By default the shortcut key is F9 - try that out and it should be good to go. You can also try googling "widget layer for linux" or "widgets compiz ubuntu" - etc. to get more in depth answers.

Have fun! :-)
