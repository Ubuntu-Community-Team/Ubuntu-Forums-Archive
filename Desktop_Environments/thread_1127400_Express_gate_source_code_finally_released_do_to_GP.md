### Post by avl555 on 2010-05-10
It has been updated: 2009/09/01

For who doesn't know where to find the source.
To download the source, go to support.asus.com
Select "Download" at the top
In the middle, select "motherboard" as product, select "Utillity" as serie, and select "Open Source Code" as model.
Search, and I recommend to use a download manager (because the connection is very slow and unstable). I hope there are not too many people downloading at that moment.
