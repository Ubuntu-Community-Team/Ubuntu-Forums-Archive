### Post by gingermark on 2006-03-04
I'm pretty certain that you'll have horrific dependency issues if you do this, in a similar way to if you added a pure debian repository. Sure, the odd application might work, but the majority won't, and you are likely to break your system.

You could always compile new apps you want, and if you use checkinstall they will be easy to remove in the future. Kopete is part of kdenetwork, so I don't know how that particular example would work though.
