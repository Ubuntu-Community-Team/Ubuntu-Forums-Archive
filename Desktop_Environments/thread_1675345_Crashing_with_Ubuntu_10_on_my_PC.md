### Post by grahammechanical on 2011-01-25
I cannot be certain but I understand that when Ubuntu is installed to the hard disc a certain amount of disc space is set aside as swap space. This happens under Windows as well. When processes use up too much system memory then data is page in and out of the swap space and the system does not slow down too much.

I do not think that this happens when running an OS from a USB flash drive. I may be wrong. I think that the OS runs in memory. I would suggest that you keep everything to a minimum. Keep an eye on the resource usage in system monitor. You may see that all the system memory is being used up.
