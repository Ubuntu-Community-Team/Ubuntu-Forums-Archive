### Post by grahammechanical on 2013-10-13
It is always better to install Xubuntu rather than install the Xubuntu desktop on Ubuntu. The same applies to the other flavours of Ubuntu. The developers of these flavours do a lot more than simply provide an install script that installs a Ubuntu base with a different desktop added on top. Over the six months of the development cycle the developers work on Xubuntu as a complete distribution, which of course includes the Xubuntu desktop. It would be a mistake to think of them as only working on the Xubuntu desktop.

Alternative desktops are an offshoot of the flavours and they can conflict with each other. I tried installing Kubuntu, Xubuntu, Lubuntu and Gnome desktop on top of Ubuntu and I found that the OS became unusable. I think that things will get worse regarding alternative desktops and the flavours become more and more distributions in their own right and Ubuntu develops in its own direction.

Regards.
