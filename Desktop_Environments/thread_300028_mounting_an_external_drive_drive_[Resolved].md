How can I keep gnome from automatically showing the contents of a filesystem when I plug in an external disk drive?

My disk drive has 5 partitions so when I plug it everytime 5 windows pop up and I have to close them one by one...

### Post by catlett on 2006-11-15
Go to the top panel and enter into System>Preferences>Removable Drives and Media when the window opens, go to the "Storage" tab. In the Removable Storage section, un-check the box next to "browse removable media when inserted". That should solve the problem. What should happen now is, the drive will still be recognised and mounted but the file manager won't open. At least that is what should happen.
