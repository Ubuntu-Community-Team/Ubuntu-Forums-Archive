### Post by sybille on 2008-02-11
Hi, This is an old thread, but I found it in a search and then I found the answer, so...

To open excel files in Gnumeric, you need to enable a plugin. The plugin is installed by default on Ubuntu, but it's not enabled.

To enable the plugin, open Gnumeric and go to Tools -> Plug-ins.... In the window that pops up, you will see a list of available plugins. Scroll down to "MS Excel (tm)," and click on the check box to enable.

Hope that's useful for someone else. :)
