> Can the left-hand menu bar be moved to the bottom of the screen? When opening Firefox and other apps, the menu bar is always on top thus obscuring part of the app.Hi, no it can not be moved if this is 11.04 you are using if not disregard the post.

You can install compizconfig settings manager and click the unity plugin and set it to auto hide.

Please do not deactivate the unity plugin or change any other settings in there because you could break unity.
