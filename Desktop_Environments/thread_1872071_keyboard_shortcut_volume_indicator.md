### Post by moon-dog on 2011-10-30
Does anyone know how to change the icons for the volume and screen brightness indicators when using the keyboard shortcuts? I'm on Ubuntu 11.10 and Gnome3, NOT Unity.

---

### Post by crdlb on 2011-10-30
Those icons are pulled from the "status" directory in your icon theme by name. For volume, it uses: "audio-volume-muted-symbolic", "audio-volume-low-symbolic", "audio-volume-medium-symbolic", and "audio-volume-high-symbolic". For screen brightness, it uses "display-brightness-symbolic".
