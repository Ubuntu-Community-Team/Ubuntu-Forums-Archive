This is in Natty with Gnome. 

Hi, I have several file association style questions. I've looked in the choices under system | preferences, but haven't seen anything appropriate. 

1. When I pull down Places | Bookmarks and select an ftp site, Gnome opens Chrome with the ftp:// URL.  When I open Gedit and select the same site from the bookmark list, Gedit shows the directory tree, and the desktop contains an icon for a mountpoint (ftp as user on domain). I'd like that behavior when I use the Places | Bookmarks.  Can I change this somewhere?  Where?

2. When I launch a .ods (Libre Office) spreadsheet, I get the archive mounter as the default, and I don't launch Libre Office.   This behavior is inconsistant with the other Libre Office file extensions, which all launch Libre Office.  Where can I control the file associations?
