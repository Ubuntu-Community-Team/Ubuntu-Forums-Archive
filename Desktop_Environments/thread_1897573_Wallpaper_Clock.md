### Post by Nir Friedman on 2011-12-19
Haha, how do you get it working or how do you get it weird and buggy? I think the first time I tried to install the KDE wallpaper clock package off the website, and when I tried to install it, it failed. I didn't notice at the time but it might have actually gotten partway into the installation and caused the buggy version. To get it working for real, just go to plasma and add widgets, if you set it to most popular one of the first will be wallpaper clock. You install it, and then there will be new option in desktop settings. 

If you don't use them, I'd recommend, as Vladstudios has beautiful wallpaper clocks, and of course it's functional as well. I have my task bar on the right side of the screen (because my laptop is 16:9 I can better afford horizontal than vertical real estate), so the time doesn't really fit in well on a vertical task bar. So this was a good solution for me.
