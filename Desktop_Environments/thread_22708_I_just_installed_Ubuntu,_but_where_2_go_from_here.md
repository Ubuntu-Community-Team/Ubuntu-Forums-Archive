### Post by az on 2005-03-29
"Now that I have installed it, where to start from here?"

What do you want to do?  Everything should already work.


"Can I install  RPMs on Ubuntu, what about sources, and how to compile them?"

Rpms?  Sure, but debs are better.  RPMs are for RPM based distros.  UBuntu is based on debian and uses debs.  Use synaptic.  Compiling from source is the same as any other unix.


"Ubuntu does not come with** gcc** installed, does it?"

install build-essential.  It is on your cd.
