The easiest place to find GNOME icon themes is at GNOME-Look.org

Download an icon set to your desktop.  It should be in "tar.gz" format.  

Then go to System > Preferences > Appearance.  When the Appearance Settings window opens, simply drag the tar.gz file into the window.  Most of the time, it will ask you immediately if you want to apply the theme.  If you want to use that icon theme now, click yes.

If that does not work, download the icon set to your desktop, extract it, and then move the folder to ~/.icons
