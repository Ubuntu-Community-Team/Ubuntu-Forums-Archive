### Post by Goombie on 2007-07-22
Try IrfanView: [www.irfanview.com](www.irfanview.com). It's Windows-only, so you will need to use it with Wine, but it's a simple image cropper, among other things. To use it, you'll first need to install Wine, then install IrfanView through Wine. I've never had any problems using it, on Windows or on Ubuntu through Wine.

### Post by mannheim on 2007-07-22
gthumb is a very simple image viewer which can also crop images. If you set gthumb to be the default application for opening image files, you can have a feel very similar to Windows Image and Fax Viewer, with the added plus of things like lossless rotation, cropping etc.

gthumb also has some cataloging ability; but it doesn't get in your way if you don't want to use it.
