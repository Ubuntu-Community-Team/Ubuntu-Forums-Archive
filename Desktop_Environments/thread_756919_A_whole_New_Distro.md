### Post by FakeOutdoorsman on 2008-04-16
> **diasje said:**
> I wold like to have one distro that wont install all the apps that the normal ubuntu install.

A clean one, something like ubuntu without any apps, like we have slax base...
You should try a command-line install from the Ubuntu alternate disc.  It is easy to do and I use it to create a base system and then I build up from there.  It is very effective and usually installs just want I want.  You may also be interested in the [Ubuntu Minimal CD Image]("https://help.ubuntu.com/community/Installation/MinimalCD").  If you want to go a step further than that you can try [Arch Linux]("http://archlinux.org").  It uses the "keep it simple" approach and by default installs a minimal base and you decide what to do from there.

> **diasje said:**
> ...The other question is, where can i get a list of software installed by ubuntu-desktop or the normal ubuntu distro?
[Details of package ubuntu-desktop in gutsy]("http://packages.ubuntu.com/gutsy/ubuntu-desktop")

Also see:
[Installation / Low Memory Systems]("https://help.ubuntu.com/community/Installation/LowMemorySystems")
