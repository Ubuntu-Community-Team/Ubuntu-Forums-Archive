### Post by Azriphale on 2006-08-03
katalog integrates into konqueror.

Right click on the mount-point of a DVD, or whatever, and click create catalog. Enter a name, and a *.katalog file to save the data to. If the file exists, the data will be appended to the file.

To see what is inside a catalog, enter "katalog:/path/to/my.katalog" into the konqueror address bar.

see [http://salvaste.altervista.org/](http://salvaste.altervista.org/) for more info.
