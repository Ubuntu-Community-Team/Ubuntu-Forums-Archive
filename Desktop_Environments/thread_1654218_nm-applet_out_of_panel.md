### Post by germanix on 2010-12-28
Try the following:
1. Right click on the nw-applet on your desktop and see if you are given the choice to add it to panel or try to drag and drop it to panel.
I f this does not work try a right click somewhere on the panel, choose add to panel and from the drop down window see if you find the nm-applet listed. It may also be the "notification area" which contains the nm-applet but also contains the volume applet. Choose add to panel, move it where you want it and lock to panel.
If this is not possible the best would be to go to the software center and un-install the applet then make a new install and re-boot. (this is probably the best idea and you could start with this if you like)
If all works un-install the wicd applet.
