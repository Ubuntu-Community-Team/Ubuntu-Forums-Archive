### Post by Wandering on 2008-06-15
I don't know that you did anything wrong, nor is it likely that the installation or removal of Ubuntu had any effect on Windows of any version. Ubuntu respects the existence of other operating systems, and will not touch their files.

Now, if you install Ubuntu and then XP, Windows will raise heck with Ubuntu.

So,  Clean it all up. Create a partition for Windows and leave an empty space on the disk for Ubuntu. Do a clean install of XP and configure it to suit.Then install Ubuntu in the empty space. It won't touch XP. 

It's been done thousands of times, so there is much confidence in the approach. Good luck.
