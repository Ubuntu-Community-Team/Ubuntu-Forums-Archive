### Post by husnos on 2012-05-09
If you have performance issues with flash in chromium, you can try the built-in flash within chromium. I think it is disabled by default in ubuntu and uses the installed flash instead.

To enable, type "about:plugins" in the address bar and enable from there.

On my laptop, the built-in flash works better in chromium.
