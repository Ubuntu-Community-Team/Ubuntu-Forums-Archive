It's a link to a package for the theme that was made to be installed on Ubuntu 12.10 (the version of Ubuntu that will be released in October).

What this means is that when Ubuntu 12.10 is released, you will be able to install this theme by typing "sudo apt-get install clearlooks-phenix-theme" into a terminal like you would an ordinary program.  This gets rid of the need to go back to GNOME-Looks every time the theme updates and manually download it and install it.
