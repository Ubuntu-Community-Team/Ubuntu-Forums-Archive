### Post by Dennis N on 2014-07-10
There are two color versions for whisker menu icon on the system - blue and aqua (a lighter blue)

My Xubuntu 14.04 uses the blue as the panel button. The button icon is specified in the whisker menu's defaults.rc as **xubuntu-logo-menu.png** and you find it in [B]/usr/share/pixmaps
[/B]
The aqua icon is named **xfce4-whisker-menu.png** and you find it in **/usr/share/icons/hicolor/**
