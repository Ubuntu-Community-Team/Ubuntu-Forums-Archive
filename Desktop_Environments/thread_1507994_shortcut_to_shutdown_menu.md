### Post by vince2678 on 2010-06-12
You could always use Control-Alt-Delete .....

but if you really wish to change it , then go to system menu >> preferences >> keyboard shortcuts

then:

go to Desktop then click logout and press the new key combination. If you use alt-f4, this will clash wih the "close window" shortcut , but you can still change it ...
