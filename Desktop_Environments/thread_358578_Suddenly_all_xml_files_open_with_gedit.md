### Post by ardchoille42 on 2007-02-11
OK, I learned a few things today in #ubuntu on irc.freenode.net.

One of the things I learned was you can right-click a file, choose Properties, go to the Open With tab and choose a list of applications that this file will open in. The top-most application in that list will be the one that file opens in upon double-clicking the file. There are Add and Remove buttons to allow you to add/remove items from that list. When you have an item you want that file to open in upon double-clicking the file, simply tick the radio box next to that item.

I'm not sure if this helps you or not, but it may give you some good information about how to proceed from here.

I hope this helps.
