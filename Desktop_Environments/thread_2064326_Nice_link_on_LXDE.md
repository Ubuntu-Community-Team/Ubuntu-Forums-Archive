### Post by vasa1 on 2012-10-02
One nice tip (Section 3: Desktop) for thicker window borders: open themerc and edit this line (near the end of the file):
```
border.width: 1
``` to a suitable value.
My themerc is here: ~/.themes/Theme_name/openbox-3/themerc
