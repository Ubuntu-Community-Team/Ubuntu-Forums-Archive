### Post by SeijiSensei on 2015-05-05
If you are using Firefox, check View > Page Style and make sure "Basic Page Style" is selected.  If "No Style" is chosen, Firefox will not use any CSS stylesheets, and the page will appear largely as text.

Also are you blocking anything with AdBlockPlus or its brethen, or filtering scripts with something like Ghostery or NoScript?  Disable all of the those and see if the page displays correctly.
