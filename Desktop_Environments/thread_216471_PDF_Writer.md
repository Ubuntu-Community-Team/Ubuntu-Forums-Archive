### Post by jonnymccullagh on 2006-07-15
vasimakhtar, i have been looking for some time but there is a serious shortfall of PDF support on linux. KPDF has some nice features but is a viewer rather than an editor. KDE has a PDF Printer option in its print dialogs. KDE also has some nice Konqueror service menus for joining, splitting PDFs etc (using PDFjam and PDFtk) - see KDE-apps.org
Also Scribus has some support for creating PDF forms but I don't think it will allow you to save forms data as you have requested.
I have also tried using Acrobat through Wine etc but I have not been successful because of the postscript drivers issues.

We really need a PDF Editor for Linux.
