### Post by tgeer43 on 2009-08-30
Once you have downloaded a new theme and want to install it simply go to System->Preferences->Appearance and either drag-and-drop the theme into the dialog box or click 'Install' and select the theme that you downloaded.

### Post by tgeer43 on 2009-08-31

I've got a much better option for you.
Just install 'gnome-art' in Synaptic Package Manager and you'll have a nice little GUI app for browsing, downloading, and installing all sorts of backgrounds, themes, icons, and other artwork. Just point and click.

After installation it appears under System->Preferences->Art Manager.
