### Post by GeneralZod on 2005-11-22
Dillo is designed to be a *very* stripped down and light HTML (no Javascript or anything) browser; it's entirely possible that that is precisely how it is supposed to look ;) (Mine looks the same as yours).

As for options, an annotated sample file is stored in /etc/dillorc; you can make per-user changes by modifying your ~/.dllo/dillorc file.
