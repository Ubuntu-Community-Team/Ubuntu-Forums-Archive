### Post by Jucato on 2006-08-05
I'm not on Kubuntu right now,so I'm not sure if the menus are the same, but you can try this:

1. Set Kate up the way you want to
2. In the Sessions menu (it should be right beside Settings), click on save. This will save the settings for the Default session.
3. In Settings > Configure Kate > Sessions, choose Start New Session and Do not Save Session. Click on Apply.

Hope that helps.
