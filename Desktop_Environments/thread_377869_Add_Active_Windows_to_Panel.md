This one's easy. I'm surprised no one else posted to help you out. It's been a couple of hours since you posted, so maybe you figured it out by now. If not:

- right-click on the panel
- select 'Add To Panel ...' from the context menu
- from the 'Add To Panel' window that pops up, select 'Window List' in the 'Desktop & Windows' section, and then click the 'Add' button at the bottom of the window **

** Alternatively, you can add the 'Window Selector' to the panel using the same method to see if you like it instead.
