### Post by linbetwin on 2005-11-16
If you are the only user, the root password is your user password, the one with which you log into your system. By default in Ubuntu root password is the password of the first user created after the installation.

When you type your password in the terminal you won't see any *s. Just type your password blindly and hit ENTER.
