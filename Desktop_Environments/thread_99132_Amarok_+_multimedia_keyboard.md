### Post by jamboarder on 2005-12-06
For anyone else that's interested, you can get your multimedia keyboard working by enabling and selecting a keyboard layout.  Go to System Settings>Regional and Accessibility>Keyboard Layouts.  Enable keyboard layouts and choose a keyboard layout from the list matches your keyboard.  

I have wireless Logitech multimedia keyboard setup this way (setup as Logitech Cordless Desktop iTouch).  You may sometimes have to pick a keyboard layout for a model that close to yours if its not listed.  You can now setup you keyboard shortcuts wherever you want to (whether globally or in your favorite kde app).  All my multimedia keys work perfectly.
