### Post by Jucato on 2006-08-07
well, basically "one app does one thing, and does that well." So Evolution is a PIM app, and does that well. PIMs don't usually include RSS readers, so the function should be on a different app.

I'm not 100% sure that this is really the GNOME philosophy, but it seems to be what they've been doing.
