### Post by Cronjob on 2007-11-12
I'm a big fan of The Unix Way, so I'm not a big fan of Konqueror. I hold to creating a very well crafted utility for a purpose and chaining multiple utilities together as necessary. That means I don't like having one giant kitchen sink that performs everything for me, including viewing PDFs directly, browsing the internet, file management . . . why not be my email client, contact manager, development IDE and everything else while you're at it?!

I might feel differently if I used Konqueror as my browser, but there are too many Firefox features that prevent me from being able to switch. So all Konqueror does for me is add a bunch of useless bloat. Cookies, bookmarks, etc.. blech. I'm not saying it isn't a great system. It just isn't my style.

While Dolphin clearly mimics (intentionally or not) the Krusader presentation (which of course mimics Midnight Commander and others), it doesn't mimic enough of the functionality. There are no tree views. It doesn't seem easy or obvious as to how to add new actions and features into it so that I can have scripted or pluggable events (using krename, exploding certain files based on rules, etc). There doesn't even seem to be a way to copy or move a file without using the mouse. In Krusader, I use F5/F6 to move/copy a file. In Dolphin, my *only* option appears to be dragging and dropping files.

Oh, not to mention - Dolphin doesn't have tabs, either. So you're limited to a two-pane window at best and nothing more.

I'm a big fan of separating the file management from the whole "Konqueror package", but that doesn't mean the file manager should be weak in functionality or dumbed down in any way whatsoever.

And while I'm mostly a user of Krusader, it can be very problematic. It often slows to a crawl. Crashes too frequently. The performance is just generally rather poor. But the functionality is there. And I can make it a rather compact interface, so I can leave it on the screen all the time. That's kind of hard to do with a lot of other file managers. Especially if you want split views.
