### Post by gzarkadas on 2010-09-24
If you select "List View" from the drop-down to the right of the location bar, you will get a table-like view where you can click the column headers to sort per this column. If you want more properties to appear, click the menu "View|Visible Columns..." and check the desired properties in the dialog box that appears.
