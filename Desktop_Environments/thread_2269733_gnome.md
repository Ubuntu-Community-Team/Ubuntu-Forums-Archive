### Post by buzzingrobot on 2015-03-17
The preferred way to use Gnome without Unity is to install Ubuntu Gnome.

That said,installed correctly, Gnome works fine on an Ubuntu Unity system. Unity does not execute when you use another interface, like Gnome. So, unless it really bothers you that Unity packages are on your system, that's another alternative.
