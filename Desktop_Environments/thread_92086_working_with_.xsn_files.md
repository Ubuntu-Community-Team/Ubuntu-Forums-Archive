### Post by asimon on 2005-11-19
I know of no Linux application cabable of handling xsn InfoPath files. You have to use MS Office for that.

AFAIK xsn files are just CAB files which contain xslt, xml, xsd, .js, and dlls. You could unpack this file (I am sure there is some program which can unpack CAB files under Linux) and then edit the xslt and xml with some editor and after that repack them again. But this is not very convenient.
