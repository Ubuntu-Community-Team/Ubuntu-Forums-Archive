### Post by Zemblan on 2010-05-26
These are features of compiz. If you install CompizConfig Settings Manager(ccsm) and run it you will see on the left side a "window management" button. Click it and you will see the various styles of application switcher that are available, click on them to investigate their keyboard shortcuts, or change them as you like.
