### Post by grahammechanical on 2020-03-27
I think that what you are asking for is application/program icons to be on the desktop. Am I correct?

Some years ago Ubuntu used its own User Interface called Unity and not Gnome Shell. When the Dash/Dock had too many application icons on it the icons folded up on top of each other and the Dash could be slid up and down with the mouse. Maybe the same thing will happen to the dock in Gnome shell which Ubuntu is not using. I do not know.

The Desktop folder stores application icons that are on the desktop. It does not store applications/programs. There may be an extension to Gnome Shell  that does what you want. This page lists Gnome Shell extensions.

[URL="https://extensions.gnome.org/#page=1"]https://extensions.gnome.org/#page=1
