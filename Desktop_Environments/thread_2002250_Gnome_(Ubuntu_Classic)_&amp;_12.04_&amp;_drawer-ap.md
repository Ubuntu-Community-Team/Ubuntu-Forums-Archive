### Post by Frogs Hair on 2012-06-12
The applet is no longer synaptic . Gnome 2 is no longer being developed by Gnome(EOL). 10.04 is the last supported release that uses it.

The closest you can get to Gnome 2 is the fork project "Mate." Some Ubuntu functions won't work with Mate because it is based on pure gnome, so some modifications of  made by Canonical are absent. Mate comes as an option on Mint which is made to work with it. 

If a program is missing it is probably because it's developers have not made it Gnome 3 compatible yet.
