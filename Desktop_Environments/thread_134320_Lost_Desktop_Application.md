### Post by o_fortuna on 2006-02-21
[QUOTE=brn]A recent 'glitch' caused the application icons to disappear from my desktop.  I was able to restore all but  *"Image Viewer"* (eog).   Although eog is present in /usr/bin/ and runs fine from a command line, it has also disappeared from the **Graphics **section under **Applications** on the menu bar.  How can this be restored?[/QUOTE]
Under Applications-->System Tools is the menu editor. Open it, go under Graphics, and select the New Entry button. Then, type Image Viewer into the name space, a description (if you want) and eog in the command box. That should bring it back. You can also create a launcher on your desktop. Just right-click and select Create Launcher and fill in the same information as above.
