### Post by 5-HT on 2007-06-17
> **jayk121121 said:**
> Thank you for replying.  Is more software the only difference?
Yup :D. The xfce4 metapackage will get you the core xfce environment. Xubuntu-desktop provides everything in xfce4 plus other recommended xfce packages, and additional software (e.g., abiword, firefox, acpi, etc...).
If you already have another desktop environment installed, you could install xfce4 + some of the recommended xfce packages as a way to avoid the bloat of having multiple programs for the same job that might happen if you install xubuntu-desktop (e.g., abiword and openoffice)
