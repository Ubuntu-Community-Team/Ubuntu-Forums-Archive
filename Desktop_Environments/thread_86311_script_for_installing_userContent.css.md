### Post by bored2k on 2005-11-04
The "correct" location depends on the browser you are using.

Using firefox, that would be $HOME/.mozilla/firefox/*profile_name*/chrome/userContent.css

Using epiphany-browser, that would be
$HOME/.gnome2/epiphany/mozilla/chrome/userContent.css (I think)

A script? to do what exactly? What I just showed you? Sure.. anything goes.
