### Post by MiKuS on 2006-09-02
-----[the intro]
OK to start off, i have fixed this error so there is no rush for a fix but i just thought i should let someone know, as when i went googling to find a fix for this problem i could find nothing of relevance. 

-----[the error]
the issue was whenever i quit firefox or changed my bookmarks around i would get these errors  [http://img62.imageshack.us/img62/5586/screenshotgu5.png](http://img62.imageshack.us/img62/5586/screenshotgu5.png)
it didn't really effect my usage it was just annoying. 

-----[the fix]
after much googleing and playing around in synaptic re-installing and removing and installing packages and not having any sucess a friend suggested to try moving my ~/.mozilla folder to a backup position whih fixed the problem completely. cause i didn't know what was causing the problem in the first place i had backed up my bookmarks and all i needed to restore was that file, but if you have more settings in your profile etc perhaps you should restore the back up.

-----[the suspect]
the suspect for this whole issue in my opinion was a firefox extension called "Yoono" ([https://addons.mozilla.org/firefox/1833/)](https://addons.mozilla.org/firefox/1833/)).

i'm reporting this issue because i couldent find any information about it, i hope anyone who has the same problem will find this information useful.
