### Post by daller on 2005-08-31
CREATE HEADER:

Insert -> Header -> Standard

CREATE FOOTER:

Insert -> Footer -> Standard

----

Pagenumber:

Now enter one of them, and click "Insert -> Fields -> Pagenumber"

Number of pages:

Now enter one of them, and click "Insert -> Fields -> Pagecount"

----

Did that work?
