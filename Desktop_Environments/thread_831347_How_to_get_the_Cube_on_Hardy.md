### Post by rainwalker on 2008-06-17
If you're using Feisty or Hardy (the last two releases of Ubuntu), then you just need to install the package "compizconfig-settings-manager". It will add the Advanced Desktop Effects Settings entry to your Preferences menu, and with it you can configure anything you want (including the desktop cube).
