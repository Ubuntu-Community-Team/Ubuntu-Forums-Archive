### Post by Upkeep on 2010-08-07
I've been looking at that too.  I've got a couple of answers.  If you happen to use Ubuntu Tweak to help with your configuration, you can launch it and then...

Desktop>Gnome Settings>History and then uncheck the Enable system-wide "Recent Documents" list

That wipes the recent documents and prevents them from being re-populated.

That's a bit of a sledgehammer approach I suppose.  Alternatively, if you choose an Ubuntu Netbook Edition 2D session at log-in, rather than the default Ubuntu Netbook Edition, you get top and bottom panels with "Applications Places System" available so that you can clear recent docs as usual.

The other thing you can do is to customise the GNOME session to mimic the UNR interface as described here:

[http://maketecheasier.com/unlock-gnome-panel-in-ubuntu-netbook-edition-une/2010/04/25](http://maketecheasier.com/unlock-gnome-panel-in-ubuntu-netbook-edition-une/2010/04/25)

Good luck
