### Post by geek.de.nz on 2006-07-11
If you've been wondering (like me) if there is a tool like partition magic for Linux. There is! Just install parted (gparted, qtparted).

```

sudo apt-get install gparted ntfsprogs

```

and you'll be able to partition away like in Partition Magic (even with support for NTFS (resize etc).

If I would have known this earlier ... :-D
