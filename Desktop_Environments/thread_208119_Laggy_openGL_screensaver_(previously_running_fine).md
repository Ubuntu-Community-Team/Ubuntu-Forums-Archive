### Post by chingazo on 2006-07-18
i don't know if you got this resolved, yet, but i was having the same problem, and searched the forums and got a fix.  seems like gnome-screensaver is the culprit.

the link to the thread is:
[http://ubuntuforums.org/showthread.php?t=189807&highlight=slow+screensaver](http://ubuntuforums.org/showthread.php?t=189807&highlight=slow+screensaver)

basically, you have to remove gnome-screensaver through synaptic/terminal etc.  then install xscreensaver and xscreensaver-data.

you can test it out by running the command xscreensaver-demo.  the screensavers are fast again, but i can't seem to get all the kinks worked out yet like having the screensaver actually turn on.  from the preview button, i can see that it is full screen and much, much faster than gnome-screensaver.
