### Post by RAOF on 2006-03-05
It looks very much like you're trying to install OpenSSH.  You don't need to compile this from source - running **sudo aptitude install ssh-server** will download, install, and configure the OpenSSH server for you.

I believe that Ubuntu installs the OpenSSH client by default.  If not, **sudo aptitude install ssh-client** should do the trick.

In general, you really shouldn't need to compile from source to install anything, unless you want to run CVS versions of stuff.  Check out the "installing software in Ubuntu" link in my sig.

EDIT: Oh, and that warning (it's not an error) is saying that the -mcpu= switch is deprecated - that means that it's not guaranteed to be supported in the future, so you (or the person writing the configure script) should stop using it now.
