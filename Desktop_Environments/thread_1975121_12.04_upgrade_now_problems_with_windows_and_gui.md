### Post by dorseye on 2012-09-30
Had this same issue (new application windows crammed all the way in the upper left, appearing UNDER Applications / Places) with Gnome classic under 12.04 -- I had luck addressing this with compizconfig-settings-manager

```

sudo apt-get install compizconfig-settings-manager

```

Start compizconfig-settings-manager
```

ccsm

```

Under Category on the left, click on Window Management. Click on Place Windows (not the checkbox, the actual "Place Windows" text) Under Placement Mode, choose "Centered". New windows now should show up in the middle of the screen instead?
