Something I loved in Gnome was the ability to show local weather next to the clock. I got a bit annoyed with Gnome and switched to Kubuntu. I'm currently using Kubuntu 8.04 with KDE 3.5.  How can I view the weather in this?

---

### Post by diaa on 2008-06-27
install kweather panel applet
```
sudo apt-get install kweather
```

then right-click on the panel, Add applet to panel, then choose weather applet for kicker.
