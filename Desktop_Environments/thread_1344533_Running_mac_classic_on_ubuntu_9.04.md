### Post by the_waka on 2009-12-03
[SIZE=2]You could install virtualbox (sudo apt-get install virtualbox-ose) and torrent an old copy of OSX from torrentreactor or piratebay.
install the OSX OS to a virtual disk in virtualbox, enable network leeching (so the virtualbox can use host interface), share the folder containing the game on your local machine, and it will appear in the network shares section on your virtual machine...
[/SIZE]
