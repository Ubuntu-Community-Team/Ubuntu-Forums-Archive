### Post by wheeze on 2012-08-26
Yes, that's how I build all my systems. Very nice because you can install only the things you want/need. You need to decide which desktop environment you want then just issue the command to install it. I currently have a very nice 12.04.1 setup running MATE.

In addition I install a display manager such as lightdm to get a graphical login, then install the synaptic package. Anything else can be installed from synaptic.
