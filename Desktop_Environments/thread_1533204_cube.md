### Post by mgmiller on 2010-07-17
Much much easier to install the simple ccsm, which will install both a simple gui and the full ccsm at the same time.

```
sudo apt-get install simple-ccsm
```

Run it from System > Preferences > Simple CompizConfig Settings manager

Just walk through the tabs to turn on the cube, select its shape (cube, cylinder or sphere) and how many faces it has. 

Lots of other nice things to tweak also, like edge & corner effects.

The full ccsm is overwhelming for someone just starting with effects.

Once you play with the simple-ccsm for a while, you can explore the full ccsm application at your leisure.
