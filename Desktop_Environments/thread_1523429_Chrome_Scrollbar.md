### Post by sanderd17 on 2010-07-03
The chrome layout is inconsistent with the normal OS. The scroll bars are different, the titlebar and window buttons are different.

This is a part of their minimalistic design. So if you want to change it, you'll have to adapt the source code and I would advise that to anyone not involved in the dev process.
