### Post by AppleCrow on 2006-05-30
Because included Bittorrent client is a little bit outdated,
This package is based on official Bittorrent 4.4.0 sources.

The only difference between this package and the one from bittorrent.com is that this package replaces the client of the distro cleanly.

For features see on official website.

Added a gnome-btdownload dummy package.

[Bittorrent 4.4.0]("http://acorbeaux.free.fr/packages/bittorrent_4.4.0_python2.4.deb")
[Gnome-btdownload]("http://acorbeaux.free.fr/packages/gnome-btdownload_4.4.0_all.deb")

These packages can be installed on Breezy or Dapper Drake, and you can keep ubuntu-desktop package.
