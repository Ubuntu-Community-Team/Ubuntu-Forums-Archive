Hello,

I've created a plugin for monitoring stocks in Conky.
It's written in Python and uses the Google Finance API as data source.

Reason for creating is because I find the Yahoo-based scripts poor performing and they're sporadically outputting garbage data.

See the following URL for downloads and howto;
[http://code.google.com/p/conky-stocks/](http://code.google.com/p/conky-stocks/)

Usage example:
conkyStocks.py --symbols goog,amzn,msft


It's been tested with python 2.4 and 2.7.

Let me know if you need help or have any questions!
