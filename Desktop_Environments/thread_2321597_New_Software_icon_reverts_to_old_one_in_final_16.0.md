### Post by grahammechanical on 2016-04-23
You are not the only person to see this. It is not a bug. It is intentional. The Ubuntu Software Centre has been replaced by Gnome Software & a few days before 16.04 was released Gnome Software was re-branded as Ubuntu Software and so the original Ubuntu Software Centre icon is being used. As this was intentional by the Ubuntu developers there will be no fix.

If you had remained on the beta version & ran update/upgrade to bring it up to the final release code then the icon change would have happened there as well. That is what I saw.

By using Free & Open Source Software from other software projects the Ubuntu developers are free to modify the code to their designs provided they comply with the licensing requirements. And they have done that many times.
