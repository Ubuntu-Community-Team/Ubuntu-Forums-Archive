Dear Ubuntu lovers,

i have installed the new ubuntu version 10.04 on many machines and find out login failed with unauthorized login.. try to find a fix for it.

to solve this login issue with ubuntu 10.04

reboot and choose in the linux Grub recovery mode ( second line ) 
chose from the recovery  list command promt using root 

type ---> passwd username 
enter the new password 

type reboot now  

and login using the new password made by the root
