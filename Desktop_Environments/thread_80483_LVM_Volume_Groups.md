I'll answer that for you philt80, I had the same problem on my own machine and I fixed it myself. 
You see philt80, I seemed to have a flash drive in my usb. Somehow, this caused ubuntu to crash at boot. After a lot of trouble of trying to figure it out, simply unplugging my pen drive fixed the problem and ubuntu booted fine. Funny that windows would still boot but ubuntu wouldn't.....
