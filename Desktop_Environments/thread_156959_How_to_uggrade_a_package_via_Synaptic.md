### Post by ec2 on 2006-04-08
Maybe the new version of firefox just isn't listed in ubuntu packages. Might be, because 1.0.7 is working fine for everybody else...
If you look at [http://packages.ubuntu.com/cgi-bin/search_packages.pl?keywords=firefox&searchon=names&subword=1&version=breezy&release=all](http://packages.ubuntu.com/cgi-bin/search_packages.pl?keywords=firefox&searchon=names&subword=1&version=breezy&release=all)
then you can see, that the latest version IS 1.0.7, so you can't upgrade... nobody can, except if you can install packages without apt-get.
