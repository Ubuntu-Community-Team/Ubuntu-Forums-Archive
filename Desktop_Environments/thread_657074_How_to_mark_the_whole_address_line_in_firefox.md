### Post by mcduck on 2008-01-03
If you want to make Firefox select the whole address when you click on the address bar, like it does in Windows, here's how to do it:

1. type "about:config" in the address bar (+ press enter)

2. type "urlbar" into the Filter field to make it easier to find the right setting.

3. Find the key "browser.urlbar.clickSelectsAll", right-click it and select "toggle" from the popup menu to change the value to "true".

Now a single click in the address bar will select the whole address. :)

### Post by randcoop on 2008-01-07
Notwithstanding the apparent problem you have with Ctrl-L, it is, in fact, a keyboard shortcut to select the address bar.  If you take a look at the Help menu in Firefox, and then look through the help table of contents for Keyboard Shortcuts, you will find that
Ctrl-L, the F6 key and and the Alt-D key combination all work for this purpose.

If it doesn't work for you, there's either something wrong with your Firefox or you've altered shortcut keys somehow.
