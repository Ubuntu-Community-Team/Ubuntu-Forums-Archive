### Post by robbiemacg on 2011-07-08
Hi, [volvo14]("http://ubuntuforums.org/member.php?u=1144696")!

I think there are lots of places you can go to find advice on how to set up a pretty bangin' desktop... surf the forums a bit, check out some of the desktop set ups your fellow members share via the monthly screenshot thread ([http://ubuntuforums.org/showthread.php?t=1794152](http://ubuntuforums.org/showthread.php?t=1794152)), ask questions. Most forum members are decent folks and will treat beginners with respect, so long as they appear to be making an effort.

Conky has a little bit of a learning curve, but you can start making cool things happen by borrowing bits of other people's scripts, and reading up. Rather than just installing Conky via Software Centre, try takign a look at some of the documentation the devs have made available ([http://conky.sourceforge.net/](http://conky.sourceforge.net/)). Once you have a bit of a working knowledge visit the very popular Conky screenshot/rc thread andpoke around or ask questions ([http://ubuntuforums.org/showthread.php?t=281865&highlight=conky](http://ubuntuforums.org/showthread.php?t=281865&highlight=conky)). Finally, if you wan to cheat a little, give Conky-colors a shot ([http://gnome-look.org/content/show.php/CONKY-colors?content=92328](http://gnome-look.org/content/show.php/CONKY-colors?content=92328)). If you're runnign 10.10 you might check out this little blog post before installing ([http://www.webupd8.org/2009/05/how-to-install-and-configure-conky.html](http://www.webupd8.org/2009/05/how-to-install-and-configure-conky.html)).

Good luck!
R.
