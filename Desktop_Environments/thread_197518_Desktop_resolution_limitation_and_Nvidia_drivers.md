### Post by mlind on 2006-06-17
Identifier doesn't matter on that section, it must be unique though.
Yes, use "nvidia" instead of "nv"


```

SubSection "Display"
	Depth		24
	Modes		**"1280x1024_60" "1280x1024"** "1024x768" "800x600" "640x480"
EndSubSection

```

first option is for requesting 60Hz mode, don't use it if you don't need it.
