### Post by ajgreeny on 2009-10-06
I'm afraid that Workington is not a possibility in this applet.  You will need to accept whichever town/city listed when you try to add one that is nearest to Workington.  You can't just type a town name and expect it to work that way; only places that have weather stations that can be web searched are listed, I think, such as RAF Stations etc etc.  If you have any of those nearby, search for their name in the Location ->Add tab of clock preferences
