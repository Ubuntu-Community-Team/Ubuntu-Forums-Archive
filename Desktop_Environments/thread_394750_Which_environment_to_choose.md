It does take some getting used to. It's certainly more "hands on" than some of the other WM. I have Fluxbox and Xfce installed. Xfce is perhaps easier to configure when you're first learning it, but I still prefer Fluxbox.

- very lightweight
- minimal desktop clutter by default
- configuration is very easy once you get used to it (love those text files!)
- lots of documentation and help on these forums
- loads of third party tools and apps if you want a more "complete" desktop experience (I don't use these myself, but they are out there)

I used GNOME under another distro for quite a while and I liked it, but it was time for something new. I have never cared for KDE. Xfce is nice, but I am so used to Fluxbox now it's just not as much fun using other WM. (I also tried WindowMaker, and while it was fast, I didn't like all the right-clicking and the default settings looked unattractive to me. I'm sure it is possible to change all of that, but once I saw Fluxbox, I was hooked.)

A new version of Fluxbox came out last week and I think I'll give it a try (maybe later today if I have time... have to compile it from source, I think).

In terms of getting comfortable with Fluxbox, some of the links in my sig might help if you haven't seen them already. ;)
