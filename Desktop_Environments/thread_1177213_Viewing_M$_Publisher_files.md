### Post by aaronchall on 2010-02-18
If you're just interested in viewing the files, there are online services that can process the .pub file and email you the .pdf (I found a free one here [http://www.en.conv2pdf.com/](http://www.en.conv2pdf.com/).) I wouldn't use these services for sensitive documents though, since you don't know who else is going to read it.  

You might be able to then edit the doc with Scribus according to this site: [http://wiki.scribus.net/index.php/Import_Publisher_to_Scribus](http://wiki.scribus.net/index.php/Import_Publisher_to_Scribus) " 
[LIST]
[*] Install a pdf printer driver (there are several opensource programs available)
[*] print your page to a pdf file.
[*] Open the pdf in Inkscape and save as SVG
[*] Import into Scribus"
[/LIST]
this description had its own caveats too.

Good luck.
