### Post by XubuRoxMySox on 2010-01-16
Try installing it using Synaptic instead. First **open Synaptic** (password required), then **Reload** (this makes sure you have the up-to-date stuff), then **select** xubuntu-restricted-extras, SunJava, etc (listed in post #2 of this thread) and check **Mark for Installation**. Then click **Apply** and let Synaptic do the work for you.

Let us know if that works... Xubuntu on an XBOX huh? Kewl.
