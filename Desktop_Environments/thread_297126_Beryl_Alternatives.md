### Post by igknighted on 2006-11-10
There is a window manager by Sun (its java) that I tried once.  I think it was called Looking Glass.  If you go to [www.distrowatch.com](www.distrowatch.com) there is a live cd called LG3D.  It boots the Looking Glass window manager on top of SLAX.  I think there are packages available (.deb), but you might need to compile the source.  It is an interesting project, different approach than compiz/beryl tho.  Here's an article w/ some pics: [http://java.sun.com/developer/technicalArticles/javaopensource/plg.html](http://java.sun.com/developer/technicalArticles/javaopensource/plg.html)

### Post by skymt on 2006-11-10
There's xcompmgr, but it's outdated, has very few effects, and (in my experience) is more crash-happy than Compiz.

Another possibility is enabling composite support in Metacity. You'll have to compile it yourself, along with a library it needs, but it's doable. It's also probably less stable than Compiz.

### Post by skymt on 2006-11-11
