### Post by RAOF on 2005-10-13
You should be able to install transcode through apt-get/synaptic.  It's in the "multiverse" repository, so you will need to enable that.

Additionally, any program you install through synaptic/apt-get should pull in all its dependencies automatically.  Plus, anything you install through this way gets automatically updated when a new version is put in the repository.
