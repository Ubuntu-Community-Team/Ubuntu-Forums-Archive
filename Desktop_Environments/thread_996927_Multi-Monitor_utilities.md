### Post by Hydrohs on 2008-11-29
In Windows I have two utilities that I find very useful for having multiple monitors, one is MultiMon, which simply extends the task bar to the second monitor, so windows are displayed on the taskbar based on which monitor they are on. 

The second is Display Fusion, which ads a lot of different wallpaper features. I mainly use it to allow me to have two different desktop backgrounds, one for each monitor.

Is there any such utilities I can use for Ubuntu to do the same things?

### Post by jaygo on 2008-12-29
Haven't heard of anything isolating windows to the taskbar of one monitor or another. 

As for wallpaper ... compiz fusion lets you pick a wallpaper for each monitor and the compiz fusion plugin "wallpapoz" is supposed to give some advanced options but I haven't tried it yet.

If that's not good enough, you might see if KDE can do it for you -- it's generally much more tweakable.
