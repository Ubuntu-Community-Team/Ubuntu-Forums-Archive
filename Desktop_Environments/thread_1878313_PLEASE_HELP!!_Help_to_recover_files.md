### Post by Mark Phelps on 2011-11-10
You said you lost files "copying" them.  If you did really "copy" them, then the original files would still be there.  Unless you manually deleted the original files.

If you actually "moved" them, in general, that's a bad idea -- for the reasons you discovered.  If the move fails, you've lost them.

Also, good luck on the file retrieval. I've found, in general, that file retrieval from USB sticks to be very unreliable.
