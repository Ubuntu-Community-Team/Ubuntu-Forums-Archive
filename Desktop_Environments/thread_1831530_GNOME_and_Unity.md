### Post by sikander3786 on 2011-08-23
With Ubuntu 11.04, you already have the choice to login to either Unity or classic GNOME. All you need to do is to logout, click your username and 'Session' option will appear in the bottom of your screen. Choose 'Ubuntu Classic' and login and you'll get classic GNOME. For Unity, choose 'Ubuntu'.

But this option isn't going to last longer, they have already removed classic Gnome from Ubuntu 11.10, the next upcoming release. So try to get used to either Unity or GNOME-Shell, that's the future as we see it.

Not to mention other DEs like KDE, Xfce, LXDE etc. They all exist, I know ;)

### Post by coffeecat on 2011-08-23
> **sikander3786 said:**
> But this option isn't going to last longer, they have already removed classic Gnome from Ubuntu 11.10, the next upcoming release.

It's true that the default desktops in 11.10 will be only Unity 3d and Unity 2d, but the user will be able to install the package gnome-session-fallback which will provide a desktop GUI that looks and works very much like classic gnome, but which will be built on gnome3 libraries. It's the fallback mode for gnome shell. There will be some loss of the functionality that some people have become used to through 3rd-party utilities such as some panel applets and nautilus plugins, where the original developers have lost interest and have not rewritten the apps for gnome 3. The one that I will miss will be nautilus-image-converter which only works with gnome 2 so it won't work with Unity in 11.10.
