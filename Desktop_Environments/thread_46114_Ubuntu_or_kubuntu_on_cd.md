### Post by Morimando on 2005-07-03
[QUOTE=mohaham]issued CDs contain Ubuntu only

A Complete Kubuntu CD can be downloaded from:
[http://releases.ubuntu.com/kubuntu/hoary/](http://releases.ubuntu.com/kubuntu/hoary/)

or you can install KDE in UBUNTU this way:
sudo apt-get kubuntu-desktop[/QUOTE]
By doing the latter you'll have to insert the shutdown commands into your main menu. Elseways you'll just be able to log off KDE and then shutdown from gdm. 
A minus when you use Kubuntu is (IMHO) that you'll have kdm set up, which isn't quite as powerful as gdm.
