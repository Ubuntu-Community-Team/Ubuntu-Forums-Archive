### Post by grahammechanical on 2013-09-27
You should also be able to open the Dash by tapping the Super (windows) key. The Dash usually opens minimised, taking up about 25% of the screen but anchored to the top left corner. If you maximise the Dash it will cover the whole screen. It should be semi opaque (like looking through frosted glass). There should be some icons along its bottom edge that will bring up applications, document files, music files, etc.

If you hold down the Super key it will bring up an overlay showing keyboard shortcuts. The Dash is something more that rollout menus and wallpaper full of icons. It learns from you. It will show recently used applications and recently used files. But first we have to start using stuff for it to remember what we used. We do not have a Dash screen full of application icons unless we click the Application lens and open "Installed see more".

There should be a search panel where we can load applications or files by searching for them. If we open any of the icons (lens) along the bottom we will get a filter option at the top right of the Dash. Click on that and we get a kind of menu structure that we can use to filter our search categories.

So, basically I am saying, is there something missing or is this a lack of understanding of what to expect?
