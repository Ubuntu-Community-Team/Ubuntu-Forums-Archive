### Post by Arktis on 2005-10-14
Ah, you mean the window titlebars.  Those are controlled by metacity, the window manager.

There are two things you can do.
1.) System > Preferences > Theme, then click 'Theme Details' , then the window border tab, and select a new style in the list.

2.) Applications > System Tools > Configuration editor, then in the list on the left select apps > metacity > general. Then in the pane on the right, find and edit 'button_layout' to re-arange or remove buttons from your titlebar layout. 

Note: Personally, I like to remove ALL the buttons. It looks nicer IMHO, and you can still get all the same functionality of the buttons by simply right-clicking on a titlebar and using the menu it gives you. It might also be helpful to go to System > Preferences > Windows and set the double-click action to perform a roll-up. :smile:

### Post by jasay on 2005-10-14
If you don't like any of the themes installed by default with breezy you can try a nifty program called gnome art.  It allows you to easily browse and download the icons, window borders, backgrounds, splash screens, etc located at art.gnome.org.  Check System > Preferences for Art Manager and if it's not there use synaptic or apt to get gnome-art.
