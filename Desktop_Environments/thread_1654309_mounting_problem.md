### Post by vanadium on 2010-12-29
This is an issue that frequently pops up. Something must have changed in Ubuntu that allows to break the association between folders and nautilus easily.

* Open a file manager window by opening a terminal and typing "nautilus &" (or any other way that works).
* Right-click a folder, select "Open with other application", then select "File Browser". Make sure the checkbox "Remember this application ...) is checked. Press Open.

This should restore the association of directories with nautilus.
