### Post by Hoppiesbox on 2007-09-04
I believe the answer is to install the 'aspell' package, and the languages of your choice using the synaptic package manager. Next time you open evolution, you should start seeing the red underlined words, and you will find your languages have been added to the context (right-click) menu's for misspelled words. Other programs use this too, such as Firefox.
