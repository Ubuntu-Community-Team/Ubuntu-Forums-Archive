### Post by BinaryDigit on 2006-03-21
Hi there,
You can install this [gmail manager]("https://addons.mozilla.org/extensions/moreinfo.php?id=1320&application=firefox") exention into firefox (if you use FF), it allows you to have a "send with Gmail" link on any webpage. Also when I click on a mailto: link, it opens my gmail in a new window so I can send mail through my gmail account...make sure you select "use gmail for internet mailto: links" option.
