### Post by GrumpyBob on 2006-02-15
I have taken two approaches to this situation:

1.  On one Ubuntu box I have Win4Lin 5.1 (this is the predecessor of Win4Lin9X).  It's good for running Win98/Me.  It does have some limitations, notably the requirement for a patched kernel.  It has no USB support.

2.  On my Ubuntu laptop I have Win98SE running under QEMU.  QEMU is an emulator which is really quite easy to install.  Carefully reading the user documentation for how to create disk images and install the OS is quite important, though the procedure itself is not difficult.  It does have limited USB support.  This works for all Windows version I think.  I have this set up with samba which allows the Windows file manager to see the Linux partition.

3. I have also used Crossover Office, which is a commercial tweaked Wine.  If you decide to go the Wine route, and you are strapped for time, I'd recommend trying a demo of this to see if your apps will run.

Robert
