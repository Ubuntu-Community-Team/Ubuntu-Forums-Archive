### Post by adam.tropics on 2007-03-08
Not entirely sure I understand, but for me, if I use the 'recent documents' from the menu bar, what I get. is a list of the last documents I viewed in the default editor, for me, gedit. However, if I open _any_ OpenOffice.org application, the 'recent documents' menu (which is located in the file menu) shows all the recent OpenOffice.org documents, whether they be Microsoft Office or OpenOffice.

I don't know if you can change this, and I certainly haven't come across it. However, if you are going to be dealing with OpenOffice, and Microsoft Office documents, then you should be aware that you can choose how OpenOffice deals with Microsoft documents. ie. open as is, or open and convert to OpenOffice format etc etc. (tools-->options-->load/save)
