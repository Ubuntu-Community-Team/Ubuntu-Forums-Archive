I am using the MATE desktop environment as it is easier to cutomize than gnome fallback and I prefer Caja to Nautilus.

One of my favorite programs is Kolourpaint, but when I run it it has a gray "Windows 95" type of look. Is it possible to make K type programs follow the MATE theme please? If so, how?

---

### Post by afixane on 2012-07-30
```
sudo apt-get install qt4-qtconfig

```

and then run "QT 4 setting", select "GTK+" in the "GUI style" option
