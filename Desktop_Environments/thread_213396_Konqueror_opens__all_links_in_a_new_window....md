### Post by Jucato on 2006-07-11
Try this:
- In Konqueror, go to Settings > Configure Konqueror > Web Behavior.
- enable/check the option "Open links in new tab instead of in new window"
- click on "Advanced Options"
- enable/check the option "Open as tab in existing Konqueror when URL is called externally"

Hope that works.
