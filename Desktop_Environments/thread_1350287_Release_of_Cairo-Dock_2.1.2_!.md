### Post by fabounet on 2009-12-09
Hi,
the latest version of Cairo-Dock has jut hit the repository.
To install it, see the wiki : [http://www.cairo-dock.org/ww_page.php?p=From%20the%20repository&lang=en]("http://www.cairo-dock.org/ww_page.php?p=From%20the%20repository&lang=en")
Here is a brief list of the news :

- The config panel has been improved again (new icons, more clear, better option naming)
 - Desklets and launchers can be fixed on a given desktop, not only on the first one.
 - Auto-hide has been rewritten
 - A new killer applet : "RSS-reader"
 - A new animation : "firework"
 - ShowDesktop can now show the desklets, the Widget Layer, and the Expose on middle-click.
 - The Mail applet can mark all messages as read.
 - A KDE-integration plug-in has been started, any help is greatly welcome to achieve it !

Enjoy ! :-)
