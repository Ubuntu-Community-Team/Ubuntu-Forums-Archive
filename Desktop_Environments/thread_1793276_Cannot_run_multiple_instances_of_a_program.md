### Post by ajgreeny on 2011-06-29
Is this in 11.04 Natty?

If so you can open a second instance of the same program by using the **dash**, in upper left of the top panel, and going to the application launcher in question.  That should open a second version.

Or in some applications you can also use the **File->New window** from the window (or global) menu.

Silly way to do it, though, in my opinion, instead of the old way.
