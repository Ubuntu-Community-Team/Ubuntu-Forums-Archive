### Post by p!=f on 2005-04-11
[QUOTE=geo926]Is there any way to add a kde session option to ubuntu 5.04?[/QUOTE]
```

sudo apt-get install kubuntu-desktop kde

```
Or select those packages in synaptic.
Note that kde is optional since it will install complete KDE.
