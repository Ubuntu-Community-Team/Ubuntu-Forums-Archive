### Post by der_joachim on 2006-08-15
Although I have used LyX for years now, LaTeX files are best edited by hand when you have to use non-standard libraries. Any good text editor will do. Vim is my personal favourite. 

My running logs and RPG stuff are all written in LyX though. LyX is a WYSIWYM LaTeX editor. WYSIWYM stands for What You See Is What You Mean.

See [the official LyX site]("http://www.lyx.org/") for more info.

### Post by SteFrankie on 2006-08-15
I tend to use TeXlipse for writing LaTeX documents. You still have to write the LaTeX source yourself, but it does give you some hints with the autocomplete.

[TeXlipse website]("http://texlipse.sourceforge.net")

TeXlipse is a plugin for eclipse so you need to install eclipse first.
