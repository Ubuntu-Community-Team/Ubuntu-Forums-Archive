### Post by craty on 2005-12-19
simpler solution could be is to use VLC media player. it will not have any problems. Just go to Synaptic Package Manager. 
Make sure that you have updated the repository. you can do this as follows:
> in synaptic go to setting>repositories. Be default you must see CD Ubuntu 5.10....
Now click add button and in the popup you get select the first three checks in all the three options available in the combo box. select yes when asked for reload 

After updating repositories simply search for VLC in the synaptic list and install it.thats it
