### Post by raja on 2007-03-30
Right click on Beryl-manager and open emerald theme-manager. It will allow you to select from locally available themes and also download themes. You can also look at the Beryl site and at gnome-look.org for emerald themes. You can download a .tar.gz file from these locations. Dont extract it. From the emerald theme manager, choose the downloaded file and it will import it.

### Post by mcduck on 2007-04-10
If you installed Emerald with Beryl, you should already have the themer app, as it's included in the same package with the Emerald itself.

If you don't have them, 'sudo apt-get install emerald emerald-themes' should fix your problem.
