Are you using Gnome or the Kde window manager (keep in mind this is the Kde part of the forum)? In any case, I never liked the Gnome package manager/add applicaton suites - in their place I like to use Adept. 

I like Adept because, like Synaptic, you can see descriptions of the packages you want to install but unlike Synaptic it makes it easy to view the description of more than one package at a time. As far as I know Adept uses the same libs as Synaptic and your failing Add Application suite so you shouldn;t be missing anything there - but don't quote me on that, I'm only like 90% sure.
