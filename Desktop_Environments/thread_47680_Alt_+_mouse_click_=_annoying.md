### Post by Caboto on 2005-07-09
I'm not quite sure (new to Ubuntu and Linux in general), but in Applications -> System Tools -> Configuration Editor under /apps/metacity/general is a "mouse_button_modifier" option normally set to "<Alt>". Try changing that to <Control> or something different.

Hope I could help you.
