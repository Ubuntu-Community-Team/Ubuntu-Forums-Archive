### Post by urukrama on 2007-11-16
Fluxbox and Openbox are both window managers, and don't deal with the desktop. If you want shortcuts/launchers on your desktop, you can use [idesk]("http://idesk.sourceforge.net/wiki/index.php/Main_Page") (in the repositories). If you want to be able to see the files that are located in the /home/USERNAME/Desktop directory, you'll need a file manager that can manage the desktop (Rox-filer, PCMan-File-manager, or Nautilus, for example).

It doesn't really make any difference whether you use Fluxbox or Openbox. Neither manage the desktop, and the applications that do will work with both of them.
