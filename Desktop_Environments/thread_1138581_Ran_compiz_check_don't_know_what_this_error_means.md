### Post by djbushido on 2009-04-26
Can you run compiz? If not, then you are pretty much screwed. The software rasterizer means that your CPU is rendering all the effects. This means that it will go _extremely_ slow unless you are either running a server or overclocking your cpu so far that if you didn't have a water cooler your entire house might go up in flames.

A bit drastic, but point being that this pretty much means that you don't have a graphics card, and that if you do force compiz, you run the HIGH risk of your computer hanging.
