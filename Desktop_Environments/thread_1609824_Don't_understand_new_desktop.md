### Post by CraigPaleo on 2010-10-31
If the desktop activity is set to "Folder View," you can drag and drop to the desktop but it ***should*** ask you if you want to move, copy, or create a link. 

To make sure you're in Folder View, right click on the desktop and select "desktop settings". Click on "Activy" and chose "Folder View."

### Post by ankspo71 on 2010-10-31
+1,
Hi, 
By default, the KDE 4x desktop has an "Activity" called "Desktop" view enabled. This prevents icons from being displayed or added to the desktop to keep the desktop cleared from clutter, even though your Desktop directory may have files in it. The only way around that is to use the "Folder View" widget, or change the activity to "Folder View" as described by the above poster. ...and I didn't understand it at first either.
Hope this helps.
