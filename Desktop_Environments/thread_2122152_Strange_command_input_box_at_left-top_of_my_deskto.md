### Post by Frogs Hair on 2013-03-05
Alt + F2 invokes the command prompt which works like typing the application name into the terminal. If using an application like conky you can start the application by using Alt + F2 and typing conky which has no laucher.  You can also kill/shutdown an application with a kill command. 
Example: ```
killall conky
```

To close click the dash home icon.

### Post by stinkeye on 2013-03-05
What your seeing is the hud invoked by the alt key.
It gives you quick access to menu items of the currently focused application.

eg Your in firefox and you want to bookmark the current page 
you can hit alt and start typing in bookmark and you will see
**Bookmarks>bookmark this page** come up.

My not seem very useful but with programs with a lot of menu items like gimp,
can speed things up a fair bit.

You can disable it in keyboard shortcuts.
Click on the key value then hit backspace.

Hit the windows+a key and type in "help" for a Desktop Guide.
