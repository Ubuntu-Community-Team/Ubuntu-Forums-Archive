### Post by coffeecat on 2011-11-13
> **Harris6310 said:**
> Hello, I'm now downloading Ubuntu 11.10 to dual boot with Windows 7 and I was wondering, does it have Gnome 3?

Yes, it does have gnome 3 because it is built on gnome 3. But I guess you mean gnome-shell. Once you've installed it, install the package gnome-shell and you will be able to choose from Unity 3d or 2d, or gnome-shell from the login screen. If you install the package gnome-session-fallback you will also get the gnome 3 classic desktop near-lookalike.
