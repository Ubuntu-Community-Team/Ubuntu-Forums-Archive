### Post by Ramses de Norre on 2006-09-14
Or to do it quickly: copy the address (it has to be an explicit copy, not jut selecting), then select for instance the word "here" from your post and click the link button. You'll get a dialog in which you can paste the address.
Press enter and you'll get the same thing as aysiu showed you.
