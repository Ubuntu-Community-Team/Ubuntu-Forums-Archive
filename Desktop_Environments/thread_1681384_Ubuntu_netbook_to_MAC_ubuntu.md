### Post by Copper Bezel on 2011-02-04
You'll need to switch to the classic Gnome environment first. Log out, select your user name, and change the bar at the bottom that says "Ubuntu Netbook Edition" to "Ubuntu Desktop Edition." What you're changing here is the desktop environment you're logging into (although both use Gnome, which gets a bit confusing.)
