### Post by bogan on 2013-01-02
Hi!, **Mia1990**,

That is the correct driver for a FX5500 and ubuntu 12.10, though you may need to install 'gnome-session-fallback' from Ubuntu Software Center, and login to 'Gnome Classic [no-effects]'.

How it runs depends a lot on how much Ram it has, if it is only 512Mb it will probably need to be a slim version, Lubuntu or Xubuntu.
