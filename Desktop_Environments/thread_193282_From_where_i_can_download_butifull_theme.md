### Post by mgmiller on 2006-06-10
I use gnome, to add a new theme, just open the theme application from system>preferences>theme and drop the tar.gz file that you downloaded onto the theme manager window.  It installs and is ready for immediate use.  I assume it is handled in a similar fashion in KDE.

### Post by gogators on 2006-06-10
Kmenu->Run->type "kcontrol"->Appearance and Themes->Theme Manager

There click Install New Theme.  Select the file that you downloaded.

Also this is where you install other things, such as icon sets in the same fashon.  For example.  From kcontrol you would click on Icons then there is the button "Install new theme"  This is for icons.

The link posted above for kde-look.org has so many great themes for you to try. Have fun.
