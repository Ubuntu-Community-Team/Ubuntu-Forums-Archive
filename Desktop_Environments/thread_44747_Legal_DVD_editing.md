### Post by apollo2011 on 2005-06-27
The best Linux program for video editing, is MainConcept's MainActor.  It runs on Windows and Linux.  There is a fully-functional demo available, but you need to purchase it in order to not end up with a watermark on any video you process in it.  It is really the only video editing program I know of that runs on Linux, and it is pricey at $199  :-?
