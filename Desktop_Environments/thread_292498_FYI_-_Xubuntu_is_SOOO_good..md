### Post by Velotix on 2006-11-11
I'll chime in on this one - I'm in the middle of trying to test out all the different Ubuntu variants to see which one is best (and ironically I haven't properly tested GNOME yet). I started with Xubuntu and have migrated to Kubuntu for now. Whether this is permanent remains to be seen (first I need to fix a minor problem with Xubuntu's session logout function - it hates KDM and shuts down instead of simply logging out of the session, so back to GDM for the login... if I can find it ](*,) ).

First, take Xubuntu's memory usage: even with Firefox running, which is infamous for chewing up memory at insane rates, I never saw my RAM usage go up beyond 250MB. Superb.

Now take Kubuntu, also running only Firefox: 650MB earlier on today, 950MB now and it's been running for less than twelve hours... I have no idea what's using all that RAM, though it's probably the fact that I have all the graphical effects activated by default. Fortunately this PC has 2GB RAM, but I feel sorry for the poor soul with 256 megs.

On that alone Xubuntu comes up trumps. KDE reeeeeeeeally needs a slimmed-down variant, but I have to give it props for just how thorough it is.
