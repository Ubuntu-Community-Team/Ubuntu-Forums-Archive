I have been looking at some software options for Linux, such as

MT-Daapd
[http://www.mt-daapd.org/](http://www.mt-daapd.org/)
GiantDisc
[http://www.giantdisc.org/index.php](http://www.giantdisc.org/index.php)

I'd be most interested to hear from anyone else out there who has tried or is trying to do something similar, with suggestions regarding particular software used etc.
