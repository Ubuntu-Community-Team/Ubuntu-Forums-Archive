### Post by lagagnon on 2006-08-18
When you choose to live in a capitalist society that has very restrictive software patent issues, DRM, and a host of other ways of making money from software users then you have to live with the consequences. The encoding and other multimedia software stuff pre-included in Windows is there because Microsoft has paid the license fee to the vendors and then passes on that fee, plus profit, to the consumer of the MS Windows license.

Ubuntu (and many other Linux distros, but not all) legally cannot distribute those codecs with the distro - at least not legally in the USA (many of those laws are valid only in the USA). But yes, if you are a true newbie and DO NOT READ the documentation that comes with the distro (many obviously do not) then it can be a bit confusing to have to download the stuff to get full multimedia functionality.

So in part I agree a quick initial boot message might be useful but really - how much bloody handholding do we have to do???
