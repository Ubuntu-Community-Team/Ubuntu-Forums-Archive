### Post by johnc4510 on 2006-05-20
Download the theme to your desktop
Make a folder in your home file, I call mine Gnome Themes
Drop and Drag the download to this file
System>Preferences>Theme
Click on Install Theme and point the browse to the folder you created
This should install them
Wallpapers just download to desktop, and drop and drag them to Change Desktop Window(Left click on desktop. Then store in a home folder(Wallpaper)

### Post by Nonno Bassotto on 2006-05-20
Actually GDM themes are installed in a different way (this is because they are common to all users). You find the dialog under System->Administration->Login screen settings (I'm not sure about the translation, my Ubuntu is localised in italian). Then it's under the tab "Themed greeter".
