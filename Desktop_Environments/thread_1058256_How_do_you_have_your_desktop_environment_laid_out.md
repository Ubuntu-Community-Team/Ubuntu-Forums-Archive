### Post by ajgreeny on 2009-02-02
I use gnome, simply because I like it, but I customise it to only have a single taskbar (I know; windows like, but that's what I prefer) and I use the gnome main menu instead of the menu bar.  I also have lots of screen icons because I find it quicker than using the main menu for most applications and also have the system setup for single click activity, instead of the double click.  It surprises some people who always expect to need to double click, but I used to use KDE, and I seem to remember that was the default in that DE, and I got so used to it it was too difficult to lose.
