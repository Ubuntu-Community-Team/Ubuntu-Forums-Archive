### Post by ajgreeny on 2010-11-23
You would not need to download and install the gnome desktop to UNE; it is already there, you just choose it from the session menu on the login page.

I don't think the kernel is any different on UNE to the normal desktop edition of ubuntu, it is just the unity desktop layout (or the old UNR desktop on 10.04) which is supposed to make it more effective on a small netbook screen.  Try it by all means, but you may find, like I think many have including me, that the unity desktop is still almost unusable in practice, and I think most users have moved to gnome instead.

I don't use compiz on my netbook so can not comment on that particular situation, though I know some people do use it on netbooks with no problems.
