### Post by varunus on 2004-12-20
You can upgrade to XFCE 4.2RC2.  This functionality has been added in it, along with a bunch more...for example, a panel applet that has the desktop menu, and a taskbar applet for the main panel if you don't like the external taskbar.  There are some .debs that work with hoary (i'm not sure about warty, you may need to steal a couple of .debs from sid or sarge, but this works...I installed XFCE before switching to hoary).  You can get it from the XFCE website, there's a link to the debian packages and how to add them to your apt sources.
