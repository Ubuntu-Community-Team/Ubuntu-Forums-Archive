### Post by chris.zeman on 2007-10-31
Hi oerllikon,

I think you'll find that Ubuntu Server will run pretty well on what you have available there. I run it on one server with a PIII 667MHz w/512MB RAM, and another server with a P4 1.6GHz w/2GB RAM.

There are plenty of tutorials to be found online. A couple of the greatest resources are this forum and [http://howtoforge.com](http://howtoforge.com). There is a software package I found yesterday that I want to use on my servers call eBox. [http://ebox-platform.com/](http://ebox-platform.com/)

I'm looking for a definitive guide on how to install eBox on Ubuntu. From what I've been told, it can be installed via apt-get (my favorite way to install), but some of the modules are missing. The good news, however, is that eBox will soon be the official web-based server interface for Ubuntu very soon. eBox makes many aspects of running a server much easier. It always pays to know command-line, but it's always great to have a simple interface. :)

Good luck!
Chris
