### Post by benerivo on 2007-08-19
I think this system tray problem also occurs with other applications, whose icons can alter the rows in the system tray. Try adding -notrayicon to the program launcher (ie. the menu entry / desktop icon / quicklauncher). For example...
  
opera %u -notrayicon

Of course this workaround is not a proper system fix, and you will lose your opera icon but i doubt that's much hardship for users as most programs don't have one.
