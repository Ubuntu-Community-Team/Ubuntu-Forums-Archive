Appreciate your response to try help solve my issue. I got it solved on Cairo-Dock forums. 

It was an issue with its sensitivity. When set to 'Default' it doesn't work (at least for me that's what happened) and somehow it was set to Default value. But now I changed it to more sensitive(0.044) and it started working.
