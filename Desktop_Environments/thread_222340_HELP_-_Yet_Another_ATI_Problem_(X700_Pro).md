### Post by BungaMan on 2006-07-26
check out this post first
[http://www.ubuntuforums.org/archive/index.php/t-187833.html](http://www.ubuntuforums.org/archive/index.php/t-187833.html)

btw, you are loading 2 drivers for your agp (amd64_agp and sis_agp).  I think that is wrong.  Check it out first but you are supposed to load only one and it needs to be loaded before fglrx.
So... unload the wrong agp driver (leave agpgart!), unload fglrx (just to be sure)
reload fglrx and restart X
There is also a file in /etc/ where you define the load order of the modules.  Make sure you remove that wrong agp driver there and make sure the right one loads right after agpgart.

good luck


UPDATE:
I didn't see the 2 previous posts when the above was posted...

when you put the following in comment
	# Option 	    "UseInternalAGPGART" "no"
it takes yes as default.  You also blacklisted agpgart so this combination makes it that fglrx uses the internal agpgart and successfully as you noticed.

I don't know if you feel like experimenting but you could unblacklist agpgart and uncomment the option useintagpgart and leave it to no.  I think it's in the link that it says this is preferrable.  Maybe it gives you better framerates.
