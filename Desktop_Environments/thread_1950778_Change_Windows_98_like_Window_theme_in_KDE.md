### Post by PaulW2U on 2012-04-01
To change or install new themes, go to System Settings and then Desktop Themes. You'll see the pre-installed themes listed and a "Get New Themes" button which will download new themes from the internet. Do have a good look at what you can change in System Settings, there is a lot that you can customise.

To check your KDE version it's 'Help | About KDE' from any KDE application menu. If you're using a fully updated KDE on Ubuntu/Kubuntu 11.10 then your KDE version should be 4.7.4.

Have fun with KDE.
