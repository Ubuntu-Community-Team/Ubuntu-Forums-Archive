### Post by Spyderkid on 2011-02-27
This is a HOW TO: to tell you how to make your Ubuntu system look sleek amazing and turn your system into eye-candy.

[COLOR="red"]Step 1:[/COLOR] Install Gnomenu by running
```

sudo add-apt-repository ppa:gnomenu-team/ppa
sudo apt-get update

```

[COLOR="red"]Step 2:[/COLOR] Install screenlets from the software centre
        add loads of widgets to your desktop using this

[COLOR="Red"]Step 3:[/COLOR] [code]gksu gedit /usr/share/themes/Ambiance/gtk-2.0/gtkrc
[/code
Gedit will open
Press ALT+F to open the ‘find’ bar
Enter apps/gnome-panel.rc into the find bar
Enter a hash (#) at the beginning of the line to ‘comment it out’.
Hit Save.

[COLOR="red"]Step 4:[/COLOR] add a background to your panel (the step above solved any                       issues with the panel looking weird when you add a background).

[COLOR="Red"]step 5:[/COLOR] add a gnomenu...
right click on panel then click 'add to panel' then add a Gnomenu
edit the gnomenu by right clicking the icon and click on 'properties'.

[COLOR="Red"]step6:[/COLOR]
install compiz from the software centre and add the effects desktop cube wobbly windows and any others you want :)


[COLOR="Orange"]**_*FINISHED*_**[/COLOR]
