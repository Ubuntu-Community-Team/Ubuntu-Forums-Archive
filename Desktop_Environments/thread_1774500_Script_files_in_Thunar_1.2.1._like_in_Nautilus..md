I just made the switch from Gnome 2.32 to Xfce and am running Thunar 1.2.1 file manager. Under Gnome I had a directory .gnome2/nautilus-scripts and under there I had all my script file for Nautilus. I was wondering if there is a way to do this under Thunar file manager.

Edit: by the way there is a directory named .conf/Thunar under my home directory with only two files: accels.scm and thunarrc. I didn't know what these were for? but from the looks of them there were config files that stored shortcuts and settings.

### Post by Toz on 2011-06-03
Look in the Edit menu for "Configure custom actions". Scripts are defined there.

EDIT: When you create your first script, a uca.xml file will show up in that directory in which the script info is stored.
