### Post by pytheas22 on 2008-09-01
This is probably because of a bug with the flash plugin that Hardy uses by default.  It's especially problematic on 64-bit.

There's a fix [here]("http://mikesubuntu.blogspot.com/2008/08/how-to-fix-flash-crash-in-hardy.html") that you might try.  I don't know if it works or not.  Another option is to try installing a different flash plugin, but at this point they all have their issues (thanks to Adobe's refusal to release documentation for flash until recently and the poor quality of its support for the binary Linux flash plugin that it provides, which is only available for 32-bit).
