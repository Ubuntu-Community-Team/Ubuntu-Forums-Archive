[QUOTE=macapp]How do u set up a internet connection on Warty Warthog 4.10.[/QUOTE]


On your Apps panel [top of ur screen by default] , select COMPUTER > System Configuration > Networking and enter ur user passwd ; u can also go here by typing gksudo network-admin in ur terminal.

here u should see ur network card / modem , if not clic add and follow the instructions as easy as the "other" operating system .

if u see ur net. card, u can click properties on it , and select dhcp or whatever u use, in case u hav static address, go manual and edit . 

not as hard as it may look like, im just bad at this  :???: 

btw, if u learn how to do it here u can do it on any gnome based distro u use, not only on ubie...
