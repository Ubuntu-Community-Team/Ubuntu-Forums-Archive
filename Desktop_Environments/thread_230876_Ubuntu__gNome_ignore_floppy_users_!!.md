### Post by mssever on 2006-08-06
I don't think that auto-mounting floppies is possible due to the way that floppy drives work (unless you're using Mac hardware from before Apple dropped support for floppies). The only way to auto-mount floppies would be to continuously check the drive for a floppy, which would doubtless be hard on the drive.

The best option would be to use a USB flash drive instead. Flash drives work better and have greater capacity than floppies, and they aren't so outdated.
