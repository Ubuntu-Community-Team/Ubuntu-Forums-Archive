### Post by IanWood on 2011-03-06
I have "fixed" this by setting the Web Browser to "Open link in new window"

System --> Preferences --> Preferred Applications. 

I needed to do this as after Installing Chrome it decided it was to be my default browser. I don't like Chrome due to the ctrl-tab behaviour. However it seems to handle Flash better than FireFox sometimes. 

Anyway - when I open links now a new Firefox will open in my current desktop, which I prefer.
