### Post by buckeyered80 on 2012-08-25
Well I see that you switched already, but your issue was easy to fix. You just need to go to open up nautilus, select "edit", select "preferences", select the Desktop tab, and then select "None" for Icon captions. You must have accidentally set it to show that. So, if you ever come back to Gnome 3, there's your fix. If not, have fun with lxde.
