### Post by AlaCarcuss on 2011-04-30
Yeah, the gnome 3 shell is pretty clunky in VirtualBox.  You're lucky you can even run it.  However it's smooth as silk for me on my real hardware.

As long as you've got a machine built in the last 5-6 years with an ok video card (preferably nvidia, but any supported 3D capable card should do) you should be fine.

Once you get used to gnome-shell, I think you'll find it a very pleasant experience from a ergonomic/usability standpoint.  If you're a mad tweaker though, you might be dissapointed.  It's way, WAY better than Unity IMHO.  Can't understand why canonical decided to go that way.  Unity is like a cheap nasty version of the Gnome 3 shell.  (In fact, untill I knew better, I though Unity was just a really bad implementation of the gnome-shell).

Also, in my experience it's easier to upgrade Xubuntu from xfce to gnome 3, than it is Ubuntu/Unity.  Unity and the gnome-shell have some confilicts.  Also, once upgraded in Xubuntu, you still have xfce available from the login prompt so you have some choice.
