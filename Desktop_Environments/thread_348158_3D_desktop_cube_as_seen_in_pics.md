### Post by mrdeeno on 2007-01-29
At first, i figured i'd go with Compiz since it was supposed to be more "stable".  At first on Dapper, and then on Edgy (with ATI mobility 9000 r250 card). 

I never got Compiz working and since I heard Beryl was less stable, I was afraid it would give me more problems.  But it was actually easier to install and use than Compiz was.  I would not try it on Dapper and would upgrade to Edgy first considering AIGLX is included on Edgy.  If it installs everything correctly like it did on mine (direct rendering enabled and such), then all one has to do is install Beryl and set it up with a few minor xorg.conf tweaks to get things working optimally.
