### Post by robertsaron on 2010-04-12
Next to the kicker icon in Kubuntu there is a folder that brings up shortcuts to the following folders: Documents, Downloads, Pictures etc. . . These are currently pointed to the Default home folder on the system drive.

I have a secondary drive where all my data is stored. What I want to do is change where the Downloads folder is pointed from /usr/home/downloads to the downloads folder on the second drive: /storage/media/downloads.

Where in the system can this be done? I know how to do this in Win 7, So there must be a way to do it in the latest KDE4.x.x Kubuntu release.

Just to be clear, I am not talking about changing where files are stored when downloading them from the internet in Konqorer or firefox, I know how to do that. I want to change where the system folders are being pointed to.

---

### Post by lovinglinux on 2010-04-12
Open System Settings, then go to "Personal >> About Me >> Paths".
