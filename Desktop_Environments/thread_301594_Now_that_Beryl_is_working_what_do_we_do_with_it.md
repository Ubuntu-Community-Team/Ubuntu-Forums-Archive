The tasks:

1. Create a bottom menu that acts like OS X's menu.

2. Slap some clouds in the background of the cube.

3. Have fully 3D-positionable windows (meaning being able to set the depth at which the window is located from off of the cube).

4. Use Beryl's ripple as an e-mail reception notifier.

5. Make windows burn down when closed.

### Post by KnevetS on 2006-11-18
> **z00s said:**
> 
1. Create a bottom menu that acts like OS X's menu.

2. Slap some clouds in the background of the cube.

3. Have fully 3D-positionable windows (meaning being able to set the depth at which the window is located from off of the cube).

4. Use Beryl's ripple as an e-mail reception notifier.

5. Make windows burn down when closed.

The bottom menu can be done with Kiba-Dock.  Search here in the forums.

The clouds are configurable under the filenames tab on the desktop cube thing in the beryl-manager.  They have to be png, and they have to be sized as an exponent of 2 (i.e. 1024x512 or 1024x1024)

The positionable windows and ripple thing are in development, as is the burn effect.  You can query around the beryl forums for how to get them installed.

### Post by d3v1ant_0n3 on 2006-11-18
For the burn effect, check in 'Animations' plugin under the 'Choices' tab- It's in the drop down menus for events (minimize/close/create/etc). I love the random colored burn. Beam Up is neat too (is beam in 0.1.2?)

Your 3rd question is answered by A 3d World plugin- it only effects windows when the cube is rotated, but when they do it looks great. I think you need to be running the svn version of beryl, and you need to compile the plugin manually. Have a look at [ THIS ](http://forum.beryl-project.org/topic-6199-world) thread for more info.

Kiba is quite cool as a dock (search either on these forums or the beryl ones for more info). It's still somewhat buggy, and, IMO, gets a bit annoying after awhile if you have the effects turned up too high. There's also KXDocker and a few others worth checking out. I think kiba is the only one that lets you play 'keepie-uppie' with the icons tho:p

Have fun with Beryl!
