### Post by Version Dependency on 2012-02-25
The Software Center should have an icon on the launcher by default in a fresh install of 11.10.  If it's not there, then find the Software Center using the dash.  Hit SUPER to open the dash and type "software" and one of the choices should be the Software Center.  If you want to add it to the launcher, right click on the icon while the application is running and "pin it."

Also, it seems you may have been confused by it asking for your password (for authentication).  This is normal in Linux. When you install or remove anything on your computer, it requires your permission in Linux.
