### Post by drs305 on 2012-05-09
I have no use (until now) for the Pictures folder. But I like to tinker, so when I read your post I tried something. 

Renamed the Pictures folder to something else. Made a link to Desktop and renamed it Pictures. When I used gnome-screenshot, the image shows up on my Desktop.

If you use the Pictures folder for other things this probably won't be a good solution, but if it otherwise remains empty it is a pretty easy solution if you intend to keep using gnome-screenshot or PrtSc
