### Post by Warbo on 2006-12-23
K/X/Ed/Ubuntu are all the same system with the same packages available, just that each one comes with slightly different ones already installed. Installing the package xubuntu-desktop will get all of the packages that Xubuntu comes with. Yes, in the GDM login screen the "Session" list will get XFCE added to it. As for "the same as Xubuntu", I would say that they are all just Ubuntu anyway, and even making the distinction between Xubuntu, Kubuntu, Edubuntu and Ubuntu is pretty meaningless when people are installing whatever packages they want in whichever flavour they want (I personally use E16 inside GNOME with Kdesktop, what the hell would that be called :) )
