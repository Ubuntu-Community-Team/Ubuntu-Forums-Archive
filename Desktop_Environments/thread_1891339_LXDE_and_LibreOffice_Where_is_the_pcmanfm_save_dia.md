### Post by marinara on 2011-12-05

** System File Dialogs or LibreOffice Dialogs **

 On most operating systems, you can choose to use the system file dialogs or LibreOffice dialogs. 
 
[LIST]
[*]Choose **LibreOffice - PreferencesTools - Options - LibreOffice - General** to switch the type of open/save dialogs.
[/LIST]

quoting the libreoffice help file.

### Post by Dennis N on 2011-12-20
> **jcmhunting said:**
> I am having the same problem.  I really like Lubuntu, but I need to save docs to a server.  Is there some way to see bookmarks in the save box?

One thing you can do is change the default path for saving your documents at: 

```
Tools > Options > Libre Office > Paths
```

You may be able to change this to point to a server rather than a local folder.
