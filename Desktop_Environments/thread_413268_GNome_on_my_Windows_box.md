### Post by graeme_p on 2007-04-19
X-Win32, Xmanager and Hummingbird Exceed and Xming can all do this. The first three are proprietary (and cost), the last is free (in both senses).

You should not need to install anything on the Ubuntu machine.

If you search a bit you might find other solutions.
