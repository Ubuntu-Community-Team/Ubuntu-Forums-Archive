### Post by Confused Computer User on 2016-01-04
My sugestion would be to try the following:
1) go to System **Settings -> Workspace Behaviour -> Virtual desktops**
2) write on a scrap paper what the settings are
3) modify them so that N**umber of Desktops** is set to 1 and the box next to **Different widgets for each desktop** is not ticked.
4) Click Apply and see if you can see the widgets on the desktop.

This should work for KDE 4.13. Not sure about 5.
