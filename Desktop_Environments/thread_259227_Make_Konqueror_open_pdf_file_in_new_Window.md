### Post by linuxa on 2006-09-17
Hi there

Konqueror opens up pdf files using the kpdf file plugin on my system in the same window that the file is at.

Can I change it so it opens it in a new kpdf window?

As if I opened up kpdf separated and loaded the file.

Thanks

---

### Post by nereid on 2006-09-17
Right click on any pdf file and select Properties (or something like that at the bottom of the menu). In the type section press on the little tool button (in the right of "Type: PDF-Document"). Select the second tab (embedding or something like this) and choose from the options to view the file in an external viewer.
