### Post by mssever on 2006-09-12
Could you provide a couple of examples of specific apps that aren't working properly? I'm not quite sure what you mean by "administrative services."

Also, could you post the exact message(s) you get?

Often, it's helpful to try to start non-working programs from the command line so that you can see any error messages they generate. If you open Alacarte (Applications > Accessories) and browse to the offending program, you can get the properties on it and see the command that is being used to start the program. Paste that in a terminal, and see what happens.

While you have Alacarte open, you'll notice checkboxes next to the programs. Only the checked programs are shown in the menu. If, after checking that, there are still programs that aren't showing up, please list them.
