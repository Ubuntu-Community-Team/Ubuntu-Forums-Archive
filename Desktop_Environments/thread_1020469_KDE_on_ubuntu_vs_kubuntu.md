### Post by gjoellee on 2009-01-02
Installing KDE on Ubuntu is done like this: ```
sudo apt-get isntall kde
```

If you want to install Kubuntu with Ubuntu: ```
sudo apt-get install kubuntu-desktop
```

If you want to speed things up in you Kubuntu install going to the upcoming KDE 4.2 is good. It should be a little faster and brings the features of KDE 3 which has been missed. 

Read this article: [http://www.kubuntu.org/news/kde-4.2-beta-2](http://www.kubuntu.org/news/kde-4.2-beta-2) for more information about how to install KDE 4.2 beta 2
