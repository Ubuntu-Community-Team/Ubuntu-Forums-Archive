### Post by nrwilk on 2005-11-18
If you're using the command ```
su
``` this puts you into root mode until you specify that you want to get out of root mode.  To execute single commands as root, use ```
sudo
```.  Could this be your problem?  ubuntu likes sudo more than su, as the root account is by default disabled.
