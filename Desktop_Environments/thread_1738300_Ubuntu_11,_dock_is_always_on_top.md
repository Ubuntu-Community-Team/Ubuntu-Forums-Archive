### Post by coffeecat on 2011-04-24
If you haven't already, install compizconfig-setting-manager. Then, from ccsm, Desktop > Ubuntu Unity Plugin > Hide Launcher. Choose Autohide or Dodge Windows. It sounds as though yours will be on "Never".
