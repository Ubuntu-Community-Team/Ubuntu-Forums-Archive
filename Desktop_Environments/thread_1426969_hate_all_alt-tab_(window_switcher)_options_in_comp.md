### Post by hexamon on 2010-03-10
I'm looking for a plugin or mode in compiz that will allow me to switch windows without any kind of overlay or effects.  I want a mode that when I alt-tab it simply moves the focus, and raises if necessary, the next window in the desktop.  The scale plugin kind of does this, but even if you map the window chooser initiator to alt-tab, you can't repeat to cycle to the next window.

  I hate the XP style default overlay boxed list.  I hate the OSX style cover flipper thing.  I hate the ring thingy.  I don't want any extra animation or overlay, I just want the next window to be focused.      
 
 I could swear that I had it this way back in the fvwm and enlightenment days, but I really like compiz aside from this one thing.  Any options out there?

### Post by asmoore82 on 2010-03-11
I think the closest you can get to this with compiz is to use the "Static Application Switcher"
and user compizconfig to turn its "Popup Window Delay" way up to like 1 second.

It will still do the switching but it will only bring up the interface if you hold
"Alt" down for that long...

see attached compizconfig screenshot.
