 Well firefox isn't an ftp "Client", which is what you're looking for, it's just displaying the site as a web page, basically. What you need is something like FireFtp, an FTP client extension for firefox: [http://fireftp.mozdev.org/](http://fireftp.mozdev.org/) or you can use a standalone one such as gFTP
