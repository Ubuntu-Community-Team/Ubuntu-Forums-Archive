### Post by akudewan on 2009-11-25
I installed KDE today, and I'm having trouble figuring out some keyboard shortcuts.

Can someone tell me what the shortcuts for the following are?
1) Maximize/restore window
2) Minimize window
3) change tabs in applications like kopete and Dolphin (I tried Ctrl+pagedown and Ctrl+tab, but those don't seem to work)

Also, is there any way to change the Window Options shortcut from Alt+F3 to Ctrl+Space?

---

### Post by Zorael on 2009-11-25
I don't think Maximize/Restore has a keybind by default, but you can set one in System Settings -> Keyboard -> Global Keyboard Shortcuts: KWin. You will find both Maximize and Minimize Window there. *(edit)* Window Operations Menu (alt+f3) likewise.

In the case of tabs in Dolphin and Kopete, those are not global shortcuts (i.e. the application needs focus to listen to them), so you have to set them in the applications themselves. For what it's worth, to change tabs, the default in both seems to be ctrl+period and ctrl+comma. But as in basically all KDE apps, you can change their shorcuts under Settings -> Configure Shortcuts.
