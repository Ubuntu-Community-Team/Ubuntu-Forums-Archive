### Post by Imerio on 2005-07-05
I had same problem - solved it by reconfiguring Xorg using the cmd:

  sudo dpkg-reconfigure xserver-xorg

in a terminal window and answering about a dozen or so questions on my graphics card, keyboard, mouse and monitor. I did this since the install had completely failed to recognise my monitor type (its on a KVM). Once done, restart X and jobs a gooden.

Warning: you must have to hand details on all your peripherals, e.g. monitor horizontal and vertical scan rates etc.
