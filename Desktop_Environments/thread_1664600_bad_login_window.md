### Post by mrkazoodle on 2011-01-11
Isn't it because you use 'switch user' instead of 'log off'? That would explain why you're not logged off.
[LIST]
[*]Switch user interrupts your session, but it keeps it safe by still asking for your password if you want to return to that session.
[*]Log off ends your session.
[/LIST]

Both of these commands take you to that login window, where another user can log in. You don't actually need to use the switch user command if you want to switch users.
