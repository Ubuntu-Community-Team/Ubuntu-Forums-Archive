### Post by Jucato on 2006-02-10
The icons themselves are either PNG or SVG and can be opened by any image editor/viewer (GIMP, Krita, GwenView). The theme files however are XML and can be edited by any word/text processor (Kate, Kwrite, KWord, OO Write).
