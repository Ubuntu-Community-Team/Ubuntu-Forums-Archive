### Post by Toz on 2011-05-06
Or, perhaps you can create a customized menu ([http://wiki.xfce.org/howto/customize-menu](http://wiki.xfce.org/howto/customize-menu) -> don't customize the main menu but use the instructions to create a second links menu) and add this menu to the panel? Still, you have to create the .desktop files and no, they don't have to be on the desktop. The default user location for menu .desktop files is ~/.local/share/applications.
