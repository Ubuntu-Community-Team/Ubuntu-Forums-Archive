### Post by gamblor01 on 2012-05-21
Ok so for anyone interested, I did find a sort-of workaround for this.  Instead of using a .desktop file from the launcher (in my case AWN dock now), I enabled the following extension:

[https://extensions.gnome.org/extension/37/quicklaunch/](https://extensions.gnome.org/extension/37/quicklaunch/)


Then I just created a new launcher from there to my start and stop scripts.  So I have to click on a different area of the screen now but it is functional.

I would still like to figure this out though.  Who knows what I (or something else) might use it for in the future...
