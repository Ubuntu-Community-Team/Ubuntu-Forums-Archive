### Post by 2cute4u on 2007-12-02
It appears that compiz is set to use the emerald window-decorator rather than the GTK window-decorator.  You need to use the Emerald theme manager to change the window border.  If you would rather Compiz to use the regular GTK theme windoe borders you need to set window-decorator in Compiz to GTK (installing ["compiz fusion icon"]("http://ubuntuforums.org/showpost.php?p=3163821&postcount=8") it makes it easy to toggle back ad forth).
