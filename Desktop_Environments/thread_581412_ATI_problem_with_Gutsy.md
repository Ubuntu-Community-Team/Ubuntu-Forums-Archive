### Post by btt on 2007-10-19
OK, so:

```
(WW) fglrx(0): Kernel Module version does *not* match driver.
(EE) fglrx(0): incompatible kernel module detected - HW accelerated OpenGL will not work
```

indicates that your kernel module does not match your running kernel (or X driver - this always confuses me...).  Either way, this is what you need to be looking at.  I have no experience of envy, but I can only guess it's incorrectly installed the fglrx kernel module.

If it has the option, I'd suggest you uninstall via Envy, then either try to re-enable the driver through the restricted device manager, or use Synaptic to install fglrx.  If neither of those options work, you could try downloading and installing the driver directly from [www.ati.com](www.ati.com).  This will compile a kernel module for your specific kernel version.

See [https://help.ubuntu.com/community/BinaryDriverHowto/ATI](https://help.ubuntu.com/community/BinaryDriverHowto/ATI) for walkthroughs, etc.
