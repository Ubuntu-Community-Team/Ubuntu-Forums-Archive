Hi you can use this command to install the manager
```
sudo apt-get install compizconfig-settings-manager
```
It has been awhile since I have used XFCE so you can use the alt, F2 keys at the same time and type ccsm. This will open the advance desktop settings
Then you can use to change. Choose the general options , desktop size set the horizontal virtual size to 4.
