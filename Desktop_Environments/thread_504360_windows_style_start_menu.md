### Post by benanzo on 2007-07-19
You're looking for Novell's SLAB-style menu.

```
sudo apt-get install gnome-main-menu
```

This will create another **Main Menu** applet for the panel.  Right click on a panel and select **Add to Panel**.  Then scroll to the bottom and add the *new* **Main Menu** item.
