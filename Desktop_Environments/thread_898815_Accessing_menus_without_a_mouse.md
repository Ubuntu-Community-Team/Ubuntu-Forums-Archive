I have a box running Mythbuntu 8.04.1, on which I occasionally turn off MythTV and work under the default Xfce desktop. On this desktop I have a horizontal panel at the top, with a menu labeled "Applications", which when clicked expands to display other menus.

  My question is, is there a way to get this menu to expand, without using the mouse to click on it? That is, by means of the keyboard alone? The reason I am asking is because I can't really use a mouse in this box, which is physically far from my TV set. I can use my keyboard because it is an RF one, but I have no wireless mouse.

### Post by OneHundredAndTen on 2008-08-24
I found it. The Shift+Num Lock key combination allows one to use the keyboard's keypad in lieu of a mouse.
