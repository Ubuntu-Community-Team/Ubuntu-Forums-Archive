### Post by chewearn on 2007-03-04
I have seen some suggestions in this forum on using different video output module in VLC:
VLC-> Settings -> Preferences -> Video -> Output modules
In the dropdown list, select another, e.g. X11, XVideo or OpenGL.

Personally, I am able to use the default in my machine (I didn't get the black screen like some people do).  However, the video quality is crappy (blocking and stuttering).

in the end, I have no choice but to disable Beryl anytime I need to run a video.  Luckily, it is possible to switch quickly between Beryl and Metacity via the Beryl Manager icon.
Right-click on Beryl Manager icon -> Select Window Manager -> Metacity.
