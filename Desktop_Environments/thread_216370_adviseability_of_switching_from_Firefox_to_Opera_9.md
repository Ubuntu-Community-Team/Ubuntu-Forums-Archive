### Post by blitzd on 2006-07-15
I have often had both Firefox and Opera installed on many different configurations and not had any problems. On Windows I mainly use Opera 9, because Firefox seems to amplify some clipboard glitches in my setup. On Linux I use Firefox (though I have Opera installed too).

Both browsers can be made to have most of the same features, but I find Firefox a bit easier to configure through adding extensions (NoScript/AdBlock/Mouse Gestures are the ones I most commonly use).

I don't think you'll see any major distros replacing Firefox with Opera any time soon - they're both free to use, but Opera is not open source and there would most likely be issues with bundling it in any default install.
