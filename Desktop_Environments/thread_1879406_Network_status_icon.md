### Post by efflandt on 2011-11-11
If using a laptop or other portable computer, make sure that your wireless is actually turned on and not accidentally turned off.

You could try **Edit Connections...** and try adding a **Wireless** connection.

Connection name can be whatever you want, but if you enter a specific SSID that is likely case sensitive (proper upper or lower case letters to match exactly).

Note that if you have an ethernet cable connected and enabled, it will default to that and wireless would not be used, even if **Enable Wireless** is checked.
