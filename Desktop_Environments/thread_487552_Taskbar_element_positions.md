### Post by jsimmons on 2007-06-29
When Ubuntu is installed, the task bar elements are locked to a specific position.

If you want these elements to move according to the currently selected resolution, UNLOCK them before changing resolution.  If you're installing nVidia or ATI drivers before changing resolution, be sure to unlock these elements before installing the drivers (because the drivers will allow you to change the resolution).
