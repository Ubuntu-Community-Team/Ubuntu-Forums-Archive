### Post by alh on 2010-12-09
Just some help for those using cairo clock for their desktop and have trouble with it not remembering screen position.
Only works for those using compiz.
Open up compiz and choose 'enable place windows; select the 'fixed placement windows' tab; in the 'windows with fixed positions' section select new. In the box enter in the 'positioned windows' box enter (without the quotes) 'name=cairo-clock. 
The X position slider should be all the way to the right and the Y position slider will need to be all the way to the left. Select 'keep in work area.
Done.
Al.
