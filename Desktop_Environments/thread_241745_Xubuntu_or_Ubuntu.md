### Post by thetictacaddict on 2006-08-22
I have been running xubuntu for a couple weeks and I just installed gnome today.  Gnome seems to actually run *faster* and I have it layed out visually in a pretty similar way.  If you want to try it out, I believe you can just "sudo apt-get install xubuntu-desktop" to install xfce, then pick an xfce session in GDM.

XUbuntu appealed to me because of the minimalistic window manager and small number of preinstalled packages (so I can decide what I want later).  After trying it out, it lacks a few things that are found in gnome, but it also has a few features (such as some panel items and built-in compositor) that I haven't found in gnome yet.

I think you should give it a try, you might like it!

### Post by bodycoach2 on 2006-08-22
What is your P3 speed. I'm guessing about 1.1 Ghz. 

I install Linux on different older systems for people, and I use processor speed and memory amount as a determining factor.

Anything under 800 Mhz, I use Xubuntu. Anything over 500 Mhz, I use what I call 'Xubuntu Loaded' - OpenOffice added, amule, Frostwire, and a few other apps. Anything under 500 Mhz, I use Xubuntu Straight-up.

So far, I've gotten a Compaq 233 Mhz AMD laptop with a 4 Gb HD to work fast enough for the owner to say, "And I thought this thing was toast! This is excellent!".

Amount of memory is important too. Anything under 256 -that can't really be upgraded- use Xubuntu.

### Post by K.Mandla on 2006-08-22
I'll second that. Your system specs sound like more than enough, and that's what will determine how well it works. I can run Xubuntu on a 1Ghz machine with lousy onboard graphics and it's acceptable, but a 1Ghz with a decent video card gives me Edgy Xubuntu + XGL. 

bodycoach2's suggestions are close to what I'd offer. If you want OpenOffice you can add it without too much stress, but Abiword and Gnumeric do all the same stuff for me. 

Java (I'm thinking of Bookworm) works fine in Xubuntu on a 1Ghz machine. Azureus is a dog, but that's because Azureus is a dog anyway. Go with Frostwire, or even rtorrent (my favorite).

It's a matter of what your machine can handle, hardwarewise. Try it and see.
