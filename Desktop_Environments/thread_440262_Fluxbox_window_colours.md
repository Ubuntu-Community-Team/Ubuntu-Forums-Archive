### Post by RedSquirrel on 2007-05-11
One easy way is to install **gtk-theme-switch**. It's in the repositories.

You can run it from the command line with **switch2** to change GTK+ 2.0 themes, and **switch** to change GTK+ 1.2 themes. It might also show up in your menu depending on how you have that configured.
