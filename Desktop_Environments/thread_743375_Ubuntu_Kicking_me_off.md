### Post by 3rdalbum on 2008-04-03
If you are running Compiz, try disabling it. Press Alt-F2 and type the following:

```
metacity --replace
```

And that switches Compiz off. Sometimes when you run Compiz with the Nvidia driver, it crashes the X server.

If turning Compiz off fixes the problem, then you should upgrade to the new Nvidia driver.
