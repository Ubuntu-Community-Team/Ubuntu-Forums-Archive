### Post by swamytk on 2010-05-04
I connect to my workstation at works through Citrix Presentation Server (also called as XenApp Server). This can also deliver the Windows applications such as Outlook, Internet Explorer through internet to a client system. In my case the client system is a linux running laptop. So across the OS and architecture, windows applications can be delivered online. 

The installation of Citrix client for linux will enable you to run your windows applications in works server system on your linux laptop. Unfortunately it is not straightforward installation, it needs some tweaks. I have tried my best to document it. This HOW-TO is applicable for Firefox only. It is NOT applicable for Google Chrome browser, which is not working as of this moment in my linux workstation.
