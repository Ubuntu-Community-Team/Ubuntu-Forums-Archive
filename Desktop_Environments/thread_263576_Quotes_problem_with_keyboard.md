### Post by kesomir on 2006-09-23
I had a problem with quotes in gnome on the keyboard and saw a post in the development section which was closed without a fix. I couldn't post there so I'm posting the problem and solution here to help others.

This isn't a bug, just a configuration issue.

Issue:

Typing quotes " using shift+key requires a space after typing it to produce the " character and typing different keys such as o produce foreign characters.

Solution:

In system -> preferences -> keyboard

Change the keyboard to select the country you want eg: UK English. Click OK after selecting the menu option (don't open to select from the sub-menu) it isn't just an organisational unit, it's also selectable itself.
