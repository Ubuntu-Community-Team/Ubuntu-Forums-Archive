### Post by onero on 2009-05-07
You need a composited WM. If you don't want to (or can't) use compiz, you can just enable metacity compositing using this command:

```
gconftool-2 -s '/apps/metacity/general/compositing_manager' --type bool true
```

### Post by Earl_Maroon on 2009-05-07
I, too, recommend Gnome-do's Docky mode. As a dock it does have fewer features/plugins than AWN, Kiba and Cairo, but it is simpler, faster and far more stable than those in my experience. The auto-hide feature works well and consistently, which is the main pro for me, but it also automatically selects the icons it displays depending on which apps you use most.

Also, if you use Gnome-do already there's no reason to have an extra app running for your dock.
