### Post by Copper Bezel on 2011-04-01
Not in Ubuntu 10.10, but the option is available in 11.04, where the new interface is much more complete. You'll want to install the package ccsm:

```
sudo apt-get install ccsm
```

Then go to Preferences -> CompizConfig Settings Manager and find the Unity plugin. There are a couple of different "hide" behaviors you can set up for the "Launcher" bar.
