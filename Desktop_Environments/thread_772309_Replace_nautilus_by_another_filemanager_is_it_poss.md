### Post by The Warlock on 2008-04-28
KDE icon sets aren't compatible with GNOME and vice versa. A unified icon set standard is coming but isn't here yet. 

You can use konq/dolphin if you want, although it'll look funny compared to other programs since it uses QT and everything on Gnome uses GTK+. You might also want to look into Thunar (the xfce file manager); I hear it's pretty good.
