### Post by lariosa42 on 2009-08-16
Hello,

I have been annoyed by this problem for a year, and I'm finally asking about it.  It's probably just a compiz settings thing, but I can't find it.

I usually work with several windows open on every workspace.  Oftentimes a window is just *slightly* off the edge of a workspace and as a result it shows up in the taskbar on both workspaces.  However, I do like being able to drag a window *way* off to the side, so it is say, half on one work space, half on the other.

What I want is some kind of tolerance, where the windows won't move past an edge unless I really force them over.  Does anyone know how to do that?

Thanks!

### Post by benerivo on 2009-08-16
You could use the snapping plugin. It conflicts with the wobbly windows plugin, but there is an option in wobbly windows called 'snap inverted' that will snap when you have a key combination pressed.

---

### Post by lariosa42 on 2009-08-16
Brilliant!  That was it!  I knew about the snapping pluggin, but I guess I wanted to keep my wobbly windows.  I even had used the "shift" key to "bump" the edges, but it was annoying, and I had forgotten about it.  "Snap Inverted" worked like a charm!  Thank you!
