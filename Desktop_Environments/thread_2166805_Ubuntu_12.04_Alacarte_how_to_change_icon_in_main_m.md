### Post by ibjsb4 on 2013-08-11
You need to go the the theme your running and change it there.  An example:

Im running the gnome theme and using 24x24 pixel size.  So I go there, find the icon and right click on it and go to properties and then click on the icon to change it.  You have to be root to do this, in terminal enter:

```
gksudo nautilus
```
