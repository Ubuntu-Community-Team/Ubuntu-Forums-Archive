### Post by sgtGarcia on 2012-02-16
Applications Menu -> Settings -> Session & startup -> Session ( tab )

There You will have list of programs & conditions when they are started on startup of system.

Change "Restart style" to "Never" for the applications that You don't want to start automatically.
