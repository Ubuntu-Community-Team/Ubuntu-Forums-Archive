### Post by evilgold on 2008-07-18
Hey, thanks for the great post. I never knew about win+space and i've been using IceWM on and off for probably 3 years now, I always used bbrun or grun. 

I have some suggestions for you too:

roxterm is a good gnome-terminal/konsole replacement. Rox-filer Also makes a good desktop replacement, for icons and wallpaper. 

If you use "cp -R /usr/share/icewm ~/.icewm" after installing the icewm-themes package, you end up getting 2 sets of every theme listed on your menu, its best to run the command before installing themes, or run "cp /usr/share/icewm/* ~/.icewm" instead.
