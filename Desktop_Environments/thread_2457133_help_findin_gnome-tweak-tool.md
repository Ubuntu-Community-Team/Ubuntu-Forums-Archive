### Post by deadflowr on 2021-01-26
It's called gnome-tweaks now.
Though gnome tweak-tool should still be there. But it's just a transitional package that will install the gnome-tweaks package.
Enable or check that the Community (universe) repository is enabled in Software and Updates.
