### Post by drmrgd on 2012-11-16
I think that gnome-terminal and Konsole (the KDE version, which is what I use) are very similar (I sort of remember this being the case when I used gnome-terminal in the past).  If so, then you need to edit your session profile (probably under a setting menu somewhere), where I think there should be a box called "Start in the same directory as the current tab"  Checking that should allow you to launch a new tab in the same directory as the current tab as you'd like.
