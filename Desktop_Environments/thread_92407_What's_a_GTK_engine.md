### Post by splot on 2005-11-19
IF you're running Gnome, you should already have GTK, since that's GimpToolKit and I think that's what Gnome is built with.  (I'm having a hard time coming up with a good description, but I think the best way to describe it is maybe to say that GTK gives programs the parts to build their GUIs.)

A GTK theme will change everything that uses GTK, all the software you have that uses the gtkrc file to define its look.  If you're frustrated with trying to get everything to match, that maybe your key. :)
