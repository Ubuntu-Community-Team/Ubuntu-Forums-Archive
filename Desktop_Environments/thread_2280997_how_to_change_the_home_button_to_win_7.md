### Post by Mark Phelps on 2015-06-04
>  I am not able to change the home button to win 7,

If your goal is to use the Home button to mount your Win7 "home" folder read/write -- that is a BAD idea!  Mounting the Win7 OS partition (where your Win7 "home" folder is kept), risks corrupting the file system and rendering it unbootable. If you want to read and then change files from your Win7 "home" folder but inside Ubuntu, the better approach is to copy those files to Ubuntu and change them there.
