Hi. I'm running Ubuntu 8.04 and I'd like to know if there's a way to customize the desktop by creating icons like Computer, Network places, to kinda make it look like M$ Windows desktop. Thank you.

---

### Post by ajgreeny on 2008-06-23
Open the configuration editor with ```
gconf-editor
``` in terminal or the run dialog, and go to apps>>nautilus>>desktop where you can chose all those you mention.
