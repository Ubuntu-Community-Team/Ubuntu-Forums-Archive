### Post by fabounet on 2013-10-15
A new version of Cairo-Dock has been released.
Cairo-Dock is a tool that can greatly enhance your desktop environment.

The official announce [here]("http://glx-dock.org/mr_article.php?b=5&a=70").

To install it, you can simply copy/paste these 3 lines into a terminal:
```

sudo add-apt-repository ppa:cairo-dock-team/ppa
sudo apt-get update
sudo apt-get install cairo-dock cairo-dock-plug-ins
```

if the dock is already installed, replace the last line with: *sudo apt-get dist-upgrade*
