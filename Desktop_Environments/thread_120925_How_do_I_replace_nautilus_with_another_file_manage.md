### Post by John Jason Jordan on 2006-02-18
[QUOTE=fuscia]try rox-filer. i liked nautilus and gnome, but both were a drag on my vintage equipment.[/QUOTE]
I first heard about rox-filer in this thread. I installed it, fiddled with it for a day or so, and then unistalled it.
Pros: It was nice and fast. Fastest file manager I have tried. Also simple. 
Cons: 1) I couldn't create a window with a split pane, like with the tree on the left. 2) Having the window resize according to the number of files in a folder drove me nuts. 3) Documentation sucks. (OK, most Linux documentation sucks. But that's not an excuse.) 4) Couldn't just select a file -- clicking on anything made rox-filer want to try to launch it or open it.

Some of my kvetches may be due to bad documentation. Some may be due to the fact that I have 64-bit Ubuntu and I may have had only an older version. Synaptic-64 is famous for sticking you with last year's leftovers.

If my problems can be resolved, please advise. I'd be happy to give rox-filer a second chance if my problems were just due to bad documentation. 

In the meantime, I'm still shopping for a good file manager. Currently I'm using Konqueror, which I like, except it takes for-freaking-ever to launch. I suspect that is because it is looking for a net connection. Which reminds me, can I please have a file manager that si JUST a file manager? Why is it that everyone thinks a file manager should also be a browser? Or a program launcher? Can't I have something that just moves files around on my computer?

I've also tried Gnome-Commander, and recently Endeavour2. Endeavour2 wasn't bad, except for the user interface. It uses GTK 1.2 and I could never configure its appearance to be better than butt-ugly.

### Post by Kimm on 2006-02-19
If you have room for it, I suggest you install kubuntu-desktop then remove nautilus from Preferences -> Sessions -> Current Session

I think KDE handles the desktop using a program separate from Nautilus, so if you run this program it should take command of the desktop (something like 'kdesktop' perhaps) Then change all buttons that call for nautilus, to konqueror (or the more unclean:

sudo mv /usr/bin/nautilus /usr/bin/nautilus_backup
sudo link /usr/bin/konqueror /usr/bin/nautilus
)
