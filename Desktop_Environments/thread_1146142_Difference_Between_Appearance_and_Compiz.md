### Post by tragicglee on 2009-05-02
Visual Effects enables _some_ of the compiz effects, but you can't select which ones run, or change the way they do. If you want greater control, compizconfig-settings-manager (CCSM) provides it. It's in the repos, if you don't have it installed (it will show up under the Preferences menu as 'Advanced Desktop Effects'). CCSM will allow you to enable the desktop cube (you may have to go into "general options" and increase the virtual horizontal size under "desktop size").
