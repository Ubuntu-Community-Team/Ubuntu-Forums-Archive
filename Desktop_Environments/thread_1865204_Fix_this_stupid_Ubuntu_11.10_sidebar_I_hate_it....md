### Post by Alwimo on 2011-10-19
Install a different shell or desktop environment and then log in as that next time by clicking on the cog icon when logging in.
 
Some other shells and DEs (type these into the terminal):
sudo apt-get install gnome-shell
OR
sudo apt-get install kubuntu-desktop
OR
sudo apt-get install xubuntu-desktop
 
Alternatively, try another distribution. 
Linux Mint is based on Ubuntu but does not have the Unity launcher. [http://linuxmint.com/](http://linuxmint.com/)

### Post by yelvington on 2011-10-23
sudo apt-get install gnome-session-fallback

This installs a fake Gnome2-like UI that, while it still has a lot of annoyances, gets rid of Dash and the Dock, puts menus back on application windows, and runs a lot faster. When you log in you'll need to use the gear icon to choose "Gnome Classic" as your desktop experience.
