### Post by Davepet on 2005-03-22
Installing the icon into the menu is up to the installer routine for the application. It's not something the OS does for you.

It's the same in windoze, the only difference is that most windoze coders know that they need to make their SW easy to use, or it won't sell.

Most Linux programs are written by someone who wants to *use* it. If it works, they release it The're not worried about selling it, so user friendly features are only included if the hacker feels they are useful.

The operating system can only do what it is told to do....

Dave
