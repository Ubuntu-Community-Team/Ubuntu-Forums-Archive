### Post by mbailey on 2006-03-09
Make life easy for yourself and install Windows first. Most guides you'll find will assume that you're a Windows user who wants to add Ubuntu. I had an Ubuntu install and wanted to add XP on a 2nd drive, but had difficulty figuring out how to do it. I'm sure it can be done, but I couldn't find anything describing how, so in the end, I decided to install Windows and then reinstall Ubuntu. Also, this way, the ubuntu install takes care of setting GRUB to see your existing Windows install. If details of any of my config would help you, let me know.
