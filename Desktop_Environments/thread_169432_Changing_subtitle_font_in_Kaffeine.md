### Post by Neo Ex on 2006-05-04
Subtitles: Settings -> xine Engine parameters -> subtitles (I haven't tested it, so I don't know if it will do what you want!). Of course you need the xine engine installed (kaffeine-xine package).
Menus: well, I don't know how you can show the menus again. If you delete ~/.kde/share/config/kaffeinerc you'll lost the Kaffeine settings, but you'll get back the menus the next time you open Kaffeine.
