### Post by Copper Bezel on 2011-11-09
They're one and the same. The Grid plugin handled by mouse can do maximize and half-screen, but it can also be configured to do quarter-screen configurations. Through the keyboard, you can set a shortcut to throw any window into any quadrant or half. It's, of course, not auto-tiling, which isn't available under Unity. There is a nifty Gnome Shell extension for it, however.
