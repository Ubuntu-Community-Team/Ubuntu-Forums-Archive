### Post by taurus on 2006-03-02
The reason there is no icon on his desktop is because you installed it in your home directory.  So, make sure you have set the right permission so he can run it from your directory.  But if you want everybody to access it, then perhaps you should have install it in /opt!!!  

So, just do the same, creating an icon on your desktop, in his desktop.
