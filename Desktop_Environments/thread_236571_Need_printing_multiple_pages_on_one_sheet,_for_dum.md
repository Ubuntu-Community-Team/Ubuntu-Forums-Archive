### Post by metalheart on 2006-08-15
From official OpenOffice.org2 User Guide, page 59. Available from [http://documentation.openoffice.org/manuals/index.html](http://documentation.openoffice.org/manuals/index.html)

> To print two pages beside one another on one sheet of paper
1. Choose File > Page Preview.
2. Locate the Page Preview icon. If there are more or fewer pages than desired, long click the icon to the right of the grayed out Page Preview: Two Pages on the object bar and select the number of pages needed for previewing.
3. Click on the Print page view icon. Clicking the Print options page view icon opens a dialogue where one can enter settings for reduced printing.
4. Click OK to confirm
