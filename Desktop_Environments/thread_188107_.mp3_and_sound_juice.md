i just want to inform other users about a bug that i discovered. When I upgraded to dapper, my sound juicer didnt allowed me to rip cds using the .mp3 format, actually it did but it ripped huge unplayable files (over 40mb per song). Searching on this forum somebody suggested to install gstreamer0.10-plugins-ugly-multiverse in synaptic.
when i did everything worked fine. So I guess something happend when i upgraded to dapper. Maybe was that dapper installed the new gstreamer 0.10? Hope this can help someone.
