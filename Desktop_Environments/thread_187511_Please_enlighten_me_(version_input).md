### Post by pandisv on 2006-06-03
The LiveCD now features Ubiquity, which allows you to install from it. In other words, you can install from the LiveCD while using your computer. Cool, or what? The one with the classical installer is the 'alternate' ISO.

In the 5.10 installation, install all the updates (not the backports, the updates). This will bring it up to 5.10 stable. If you want then, you can update it to 6.06 using the procedure described in the release notes. 

Hope this helps...
