### Post by mcduck on 2008-06-02
Sorry, no.

The image preview in file selection dialog is an optional feature, and application developoers have to specifically enable it if they think it's usefull in their program.

There is no way to force all applications to use it.

You should send feedback to developers of those programs you think should enable the preview.
