### Post by stinkeye on 2013-02-20
In my shortcut overlay the keys are alt+` (the key above tab)
to switch windows of the currently focused application.
This enables you to alt+tab to an application with multiple windows
and after a short delay all windows are shown in the switcher.
Continue holding alt and press the ` key (above tab) to cycle through the multiple windows.

If the application with multiple windows is already in focus you can go straight to the alt+` key.
