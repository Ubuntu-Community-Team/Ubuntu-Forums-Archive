### Post by Denestria on 2008-12-06
Open your package manager from the menu, type unrar in the search box and unrar-free is the second on the list, check it and apply to install it.

### Post by scragar on 2008-12-06

if you want rar support for rar3.0 files you'll need to use the non-free unrar as well:
```

sudo apt-get install unrar unrar-free

```
