---

### Post by stylintile on 2013-05-19
Hi lazylew,
     To have conky display a dollar sign you need to put in two of them.  For example;
```
${font Ubuntu Mono:size=12}${color 75b3d1}$$
```
displays
```
$
```

I hope this helps
