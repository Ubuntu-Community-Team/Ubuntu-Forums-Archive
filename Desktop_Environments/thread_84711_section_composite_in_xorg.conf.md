### Post by oobuntoo on 2005-10-31
Since you are using "nvidia" in Section "Device", you should comment out Load   "GLcore"  and Load   "dri"  in Section "Module" by putting # in front of them. Also add to Section "Device" these two options:
        Option          "RenderAccel"           "true"
        Option          "AllowGLXWithComposite" "true"

I have GeForce 6600GT and these options work OK for me. Some people report system lockup when using  "RenderAccel".
