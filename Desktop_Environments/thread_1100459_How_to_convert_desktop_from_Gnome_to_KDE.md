When I started using Ubuntu 8.10, I setup a GNOME desktop.

---

### Post by mcduck on 2009-03-19
Just install the "kubuntu-desktop"-package (with Synaptic, or by running "sudo apt-get install kubuntu-desktop").

When it's installed you are able to select which DE to use if you click the "Session"-button in the login screen.

If you want, you can then remove Gnome and it's programs.
