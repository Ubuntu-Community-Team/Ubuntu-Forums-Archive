### Post by weirdtalk on 2010-06-04
most likely you have either removed the task manager, or squished it with another panel icon (so it's really small). right click on a empty part of the panel and add a new widget. Add the task manager.

Alternatively to save some time, alt+tab can switch between running apps (visible & minimized).

---

### Post by sa2u on 2010-06-04
Thanks, Weirdtalk, that fixed it! I mean, adding Task Manager to the panel fixed it.
