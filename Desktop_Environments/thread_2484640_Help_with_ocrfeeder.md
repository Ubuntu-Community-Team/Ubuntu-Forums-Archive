### Post by coffeecat on 2023-03-05
I was intrigued enough by your post to install OCRFeeder to have a go for myself. When I export ODT format, I get a complete mess, with some giant text, but not as giant as yours. I didn't find a way to choose a font size, but I do have a suggested workaround. After OCRFeeder has recognised a document from the Document menu, choose Export from the File menu. In the little popup window, choose "Texto simples" from the format drop-down and export to a plain text file. That way you get no formatting at all, and you can then edit the plain text in a word processor to make your own choice of  font properties.

Thanks for posting this. This is a useful application that I had not come across before.
