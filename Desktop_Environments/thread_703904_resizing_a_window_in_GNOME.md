### Post by iThinkergoiMac on 2008-02-21
It's a Compiz thing. You need to install the compiz settings manager. Do a search for Compiz in synaptics. Once you install it you'll be able to access all the different extra settings in Compiz. (System -> Preferences -> Appearance -> Visual Effects -> Custom). Once you're there, go to the bottom and click on Resize Window to edit the settings. Let me know if that doesn't work!
