### Post by camper365 on 2009-05-27
try logging into a tty (Ctrl+Alt+F1) and see if it tells you that firefox can't find a display. After that you can hit Ctrl+Alt+F7 to go back to X.

If Firefox does tell you it can't find a display, then it is probably in the startup programs file (like .bashrc). If not then it's a problem in X.
