### Post by bgturk on 2007-09-28
In principle this should be possible by using the accessibility magnifier:
magnifier -fiz 1

However, as noted here
[http://www.linuxquestions.org/questions/showthread.php?t=572393](http://www.linuxquestions.org/questions/showthread.php?t=572393)

The gnome magnifier is still buggy and does not work very well.

### Post by bgturk on 2007-09-28
I am doing gutsy testing now, and the magnifier package has been fixed in gutsy. The above command does indeed result in screen inversion :-)
