### Post by CatKiller on 2017-07-15
Pulling everything useful out of the application window and smearing it over the panel is called the Global Menu. It was a Unity "feature."

There is a community extension in progress (called Gnome Global AppMenu) to have something similar for Gnome. I haven't tried it, for obvious reasons.
