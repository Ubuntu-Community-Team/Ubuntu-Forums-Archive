### Post by Anduu on 2006-08-07
Well I did a little searching and it seems this is a bug we have to live with for now...there are a couple of different 'hacks' which can be used to fix it but these could quite possibly mess other things up.

If you need to shutdown you could always ctrl+alt+backspace to kill the xserver and when the login screen pops up select shutdown from there.

Or you can create a shortcut on your desktop to shutdown.Right click on the desktop and select create launcher.Name it Shutdown or whatever and in the command field enter ```
sudo shutdown -h now
```Also check the run in terminal box.

When you double click it a terminal window will pop up asking for your password then everything will go bye-bye ;)
