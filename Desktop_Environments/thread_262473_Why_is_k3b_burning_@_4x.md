### Post by g8m on 2007-05-24
k3b is a graphical frontend for  burning tools like growisofs. So if burning is slow, it's probably caused by those programs. If it's really slow like 1x, its probably caused by dma settings of the drive or connected to a non-dma ide controller. But then again I have yet to see a fast burn. K3b burns dvd's with my 18x burner (media 16x) between 1x an 7x, it varies and I don't know why. I suspect it is also influenced by system load. If I'm doing a lot of stuff I see burning speeds drop. I dont have windows, so I cant compair. I accept 20 minutes to burn a dvd, so I don't really mind it being slow.
