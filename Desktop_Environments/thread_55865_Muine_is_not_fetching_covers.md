### Post by Stormy Eyes on 2005-08-11
Automatic cover fetching didn't work for me the last time I tried it. But if you go to amazon.com, search for the album whose cover you want, bring up the larger version in a separate window, click on the image, and then drag it onto the bass clef icon Muine shows in place of an album cover, Muine will download the image and associate it with that album. 

Try it; I personally like it better than Muine trying to guess which image I want. I've had Muine use the cover to Iron Maiden's *Powerslave* as the cover image for *Fear of the Dark*.

### Post by mättä on 2005-08-17
Muine is fetching the covers from Amazon. Since recently amazon.com stopped serving requests without a valid amazon developer ID. Therefore Muine can't gets its covers from there anymore.

See the following bug report for further information: [http://bugs.gnome.org/show_bug.cgi?id=310795](http://bugs.gnome.org/show_bug.cgi?id=310795)

I guess we have to wait for a next release to get it working again. Until then you have to manually drag-and-drop images from your web browser to Muine.
