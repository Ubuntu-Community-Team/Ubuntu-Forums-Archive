### Post by StarscreamD on 2007-03-01
From one beginner to another, I found that you must restart your system in order to get the themes to apply properly to my experiences, which are few. This will add buttons like close/maximize/minimize etc. depending on the theme. Also, if you right click the beryl, and execute Beryl Settings Manager, Move Window is under Window Management. I had to enable this to move the windows around personally. Heop this helps you out a little! Good Luck!
