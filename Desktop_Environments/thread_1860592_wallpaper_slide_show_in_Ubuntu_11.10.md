### Post by stinkeye on 2011-10-14
Wallch works for me installing from the Wallch ppa for GNOME 3.

```
sudo add-apt-repository ppa:wallch/wallch-ppa
```
```
sudo apt-get update
```
```
sudo apt-get install wallch
```
