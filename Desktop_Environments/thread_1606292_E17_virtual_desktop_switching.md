### Post by squaregoldfish on 2010-10-26
To disable edge flipping, open the Settings Panel, go to the Input tab, and select Edge Bindings. You can disable all the edge bindings there.

There aren't really any guides out there yet, as e17 is still pre-alpha software. Various options are listed on [http://www.enlightenment.org/p.php?p=support&l=en](http://www.enlightenment.org/p.php?p=support&l=en) - but the #e IRC channel and the mailing lists are pretty good.

Steve.

### Post by squaregoldfish on 2010-10-27
The developers are currently pushing hard into getting the underlying libraries (evas, ecore and all their friends) to a release state. They went into beta a couple of weeks ago.

Once they're released, there will be a similar push to get e17 fully released. People have muttered something about a full release by the end of the year, but I wouldn't hold your breath. Rest assured that development is happening though!

By the way, there hasn't been an official stable release. The developers occasionally produce snapshots of the source code to simplify installation, so that's probably what they meant.

Steve.
