### Post by lapubell on 2008-03-19
I would install kubuntu-desktop.  That is what is called a meta package.

It will include all the files you need to get KDE working on your machine.  one thing I should point out, it will install a program called KDM which is a different log in method.  When you are installing, it will ask which you want as your default, KDM or GDM.  I usually use KDM as it has both the user name and the password on the same screen but you can choose which ever you want.  When you go to log in, there will be an option for "Session".  This is what will let you choose between KDE and Gnome.  Good luck!
