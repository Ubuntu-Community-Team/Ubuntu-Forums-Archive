### Post by svivian on 2009-05-05
I'm not sure if you can explicitly pick certain icons to use for the buttons, but you can change the icon "theme" by going to:
System > Preferences > Appearance > Customise [button] > Icons [tab]
and choosing something else e.g. GNOME. This will change icons for the whole system.

### Post by tdietz87 on 2009-05-05
Places > Home

Then ctrl + h (display hidden files)

Scroll to /.icons in there you will find all the icons you can change.  Either by the method mentioned above with themes or sort through til you find the icon you wish to change.  I am fairly new to ubuntu and I use 9.04 but I've managed to change specific icons to my liking this way
