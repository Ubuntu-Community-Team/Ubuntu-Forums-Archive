### Post by oobuntoo on 2005-09-05
This is a KDE bug. The taskbar does not refresh correctly. In addition to system tray border problem, you'd also get a block of different color in place of an icon after you quit a program that put an icon in system tray, like Azureus or amaroK.  The taskbar would refresh if you move it, but the problem will come back again later. We'll see if this bug get fixed in KDE 3.5.
