### Post by 3Miro on 2010-01-01
Make sure you have installed all the proper drivers from System -> Admin -> Hardware Drivers.

Then install compiz Config Settings Mamager from Synaptic. Then from System -> Prefs -> Compiz Config you can play with the many different effects to see if one in particular is giving you hard time.

Alternatively, you can disable compiz altogether by going into a terminal and typing:

```
metacity --replace
```
