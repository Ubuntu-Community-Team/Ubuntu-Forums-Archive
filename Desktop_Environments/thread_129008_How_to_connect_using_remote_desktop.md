[QUOTE=shultzjr]Once I set the parameters in remote desktop, how do I know what address to use if the Ubuntu machine is on another network?[/QUOTE]
It would be the same address if you were remote desktoping from another Windows PC.  There must be a route from one network to the other (e.g. you can ping the remote computer or do a traceroute to it).
