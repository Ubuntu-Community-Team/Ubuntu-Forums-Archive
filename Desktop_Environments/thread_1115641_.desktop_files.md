### Post by VCoolio on 2009-04-04
.desktop files are sort of lauchers for your apps / commands. You can create them like this in a text editor and save as .desktop. For an app / command:```

[Desktop Entry]
Version=1.0
Encoding=UTF-8
Name=choose-a-name
Type=Application
Exec=choose-your-command
Icon=/set/path/to/your/icon.pngjpgwhatever
```
If you want the command to be executed in a terminal, add a line 'Terminal=true'.
For a location, change Type and Exec to:```

Type=Link
URL=/path/to/folder
```
I'm not sure what the 'version' parameter does. In all cases you can add a comment with a line 'Comment=yourtext'. Well, this should get you started.
