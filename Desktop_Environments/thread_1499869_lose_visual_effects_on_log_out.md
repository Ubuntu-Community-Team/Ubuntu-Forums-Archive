### Post by faust99 on 2010-06-02
I installed 10.4 today on Macbook Pro 5,5 and there is one problem in that I lose visual effects every time I log out or restart. I have to set it back to extra or normal every time I log in otherwise it goes to none. This would not be so much of a problem normally, but as well as this, there is the problem that I can't see the top of windows (where the maximise/minimise buttons are), and cannot move windows around or close them. This is strange behaviour. Any ideas/suggestions/similar stories out there???:confused:

### Post by faust99 on 2010-06-02
This [post]("http://ubuntuforums.org/showpost.php?p=8654196&postcount=6") seems to have helped:

> For some reason, re-saving my Gnome session seemed to fix the problem. In System->Preferences->Startup Applications, on the Options tab, you can click the Remember Currently Running Applications button (I did this with no other applications running, as I did not want to run any extra startup programs).
