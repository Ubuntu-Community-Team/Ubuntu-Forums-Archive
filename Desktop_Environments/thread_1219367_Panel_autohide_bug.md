### Post by ugm6hr on 2009-07-21
It appears this has beena long-standing bug in xfce4-panel.

Using autohide, it sometimes decides to remain visible, on top of open windows.

This was an interesting "solution"
[http://forums.fedoraforum.org/showthread.php?t=219496](http://forums.fedoraforum.org/showthread.php?t=219496)

Essentially, create a panel launcher with command to restart the panel!
```
xfce4-panel -r
```
