### Post by foxy123 on 2012-03-11
In Unity Ctrl+S annoyingly assigned to showing the desktop switcher (same as super+s). As a result this shortcut does not work in LibreOffice as Save. Is there any way to disable it in Unity so I could use it for saving in LibreOffice?

---

### Post by stinkeye on 2012-03-11
It's not a default binding.
Open up ccsm and click on **Advanced search**
Mark the **Settings value** box and enter
```
<Control>s
```into the filter box.
