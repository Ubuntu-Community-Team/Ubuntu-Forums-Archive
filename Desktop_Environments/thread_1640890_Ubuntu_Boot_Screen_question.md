### Post by sikander3786 on 2010-12-08
That is the newest Kernel on top of the list and below it is its Recovery mode entry.

I hope you are boot fine using the new kernel.

And remember, it is advised that you at least keep one older kernel at least in that menu, so that if the newer one fails for any reason, you can boot using the older one and diagnose/solve the problem.

I would recommend to leave it just as it is.

You can remove the Recovery mode options (not recommended), or the memtest options if you feel your Grub menu is over-crowded.

How to remove older Kernels courtesy of forum member **drs305**
