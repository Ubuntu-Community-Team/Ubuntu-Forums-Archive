### Post by az on 2005-02-27
1- The gpl requires that the sources be available.
2-  If you only want to use binaries, you do not need the source repository listings.  If you are on a modem connection, for example, they are a pain in the ass.
3-  Yes you can mix.  In fact, you will tend to run into less trouble when using "foreign" repositories if you install them from the deb-src.  There is more poking around, though.
4-  Dunno about synaptic compiling them...  Good question.
