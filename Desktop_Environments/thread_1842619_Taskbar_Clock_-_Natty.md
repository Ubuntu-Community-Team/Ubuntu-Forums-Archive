### Post by Krytarik on 2011-09-11
So, you went a bit too far, eh, in removing all packages associated with Evolution!? ;-)

First, install "indicator-datetime", for the clock, obviously. Then make sure that "indicator-session", the power button, is installed. You may need to add one or both of them to your panel if using classic Gnome.

Greetings.
