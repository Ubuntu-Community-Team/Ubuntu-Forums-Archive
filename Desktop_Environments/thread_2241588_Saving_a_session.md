### Post by Frogs Hair on 2014-08-27
The Compiz Configuration Settings Manager has a plugin named " Place Windows" which would only be of value in the gnome session-flashback with effects. There is a tab in the plugin named fixed window placement although more information is needed because the plugin is still available though much of the information I find is outdated and might not apply  . I know of no way to do what you want in the gnome-shell unless there is an extension that makes application workspace placement possible.
