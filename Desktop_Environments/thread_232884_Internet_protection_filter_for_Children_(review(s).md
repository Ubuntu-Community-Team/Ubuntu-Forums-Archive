### Post by LaserJock on 2006-08-30
I know this thread is a bit old but I thought you guys might want to know that Edubuntu has been working on a new content filter for  Edgy as a Google Summer of Code project. The same guy who wrote the Alacarte menu editor for Gnome (what we now use) is writing it. It is called willowng and is in the Edgy repo right now. It isn't fully finished but will be quite cool. It is based on the Willow project but was rewritten to make it cleaner. It uses a Bayesian learning algorithm to learn what is "bad" but also includes domain filtering. This has been a long standing goal of Edubuntu I believe.

-LaserJock
