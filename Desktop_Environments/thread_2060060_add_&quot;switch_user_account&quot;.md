### Post by Buntu Bunny on 2012-09-19
Is there any way to add a "switch user account" to the Xfce log out dialog? 

We have 2 users on this computer, and it would be so nice to not have everything in my DE closed out every time my husband needs to use the computer.

---

### Post by Morbius1 on 2012-09-19
```
sudo apt-get install xfswitch-plugin
```Then Right Click a panel > Add New Items > User Switching
