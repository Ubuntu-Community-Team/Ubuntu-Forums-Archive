### Post by Quirky on 2005-11-09
Thunderbird automatically moves junk email to a seperate folder and it's possible to hide junk in this folder so it's never seen. This is great. Unfortunately, the folder shows **Junk(1)** when a junk mail arrives, which is a bit annoying. 

It would be even better if junk could be automatically marked read. According to this bug report [https://bugzilla.mozilla.org/show_bug.cgi?id=193625](https://bugzilla.mozilla.org/show_bug.cgi?id=193625) it has been possible to automatically mark junk mail as read since around May 2004. However, Breezy's Thunderbird 1.0.7 doesn't seem to have this feature. Open Tools->Junk Mail Controls... and there is no 'auto mark read' option as expected. Is this feature really available in the non-Ubuntu Thunderbird?
