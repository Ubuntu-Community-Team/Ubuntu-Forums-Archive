I had the same problem but I fixed it on my ati card. If you have a ati card try this in the terminal:  
sudo apt-get update
sudo apt-get install linux-restricted-modules-$(uname -r)
sudo apt-get install xorg-driver-fglrx
sudo depmod -a
sudo aticonfig --initial
sudo aticonfig --overlay-type=Xx
--Reboot The Computer--
