### Post by RedSingularity on 2009-08-16
To see the gears go to Compiz>Desktop>Desktop Cube>Transparent cube tab.  There set the transparency to what you want.  Now you should be able to see through the cube when rotating.  And if you have gears enabled you should see them inside.  Spin speed in under the behavior tab.  And as for the backgrounds....i have never seen a cube with different backgrounds on each side.  I am not sure thats possible.
