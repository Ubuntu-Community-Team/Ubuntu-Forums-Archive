### Post by jmoisan on 2006-07-13
I switched from XP to Ubuntu a few months ago, and I must say that my experience so far has been great !

Here's a little-known trick that I wanted to share with you, fellow Ubuntians :cool: :

If you want to burn IMG files under Ubuntu (any version), fire up [**NeroLinux**]("http://www.nero.com/enu/NeroLINUX.html") v. 2.x, go under the **Recorder** menu, and select **Burn Image...**. Browse your folders to locate the IMG file (CD or DVD) that you want to burn. Click OK.

Nero will tell you that the IMAGE SEEMS TO BE INVALID. [COLOR="Red"]**Ignore this warning**[/COLOR]. Choose the correct medium type (CD-R(W) or DVD) in drop-down list, and start burning using the **Burn** button.

Trust me, you won't waste your CD or DVD. I burned 20+ IMG files using this method, and everything worked flawlessly. Call that an undocumented feature of NeroLinux :)

Happy Ubunting \\:D/
