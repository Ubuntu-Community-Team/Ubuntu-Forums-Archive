### Post by dave9191 on 2005-08-05
Your ram will always be packed. Thats because linus caches files in ram so that it doesnt have to keep accessing the disk. I usually use about 150mb ram for programs, the other 350mb or so is being used up by the cache. If the swap isnt getting filled, it means that programs are not using all of your memory. And the problem probably lies somewhere else.
