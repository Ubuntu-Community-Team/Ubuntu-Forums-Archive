### Post by rai4shu2 on 2012-11-11
I haven't tried it myself, but you could go to Settings Manager / Keyboard / Application Shortcuts, and then add the commands you want those keys to have.

```
gmusicbrowser -cmd PlayPause
gmusicbrowser -cmd Stop
gmusicbrowser -cmd NextSong
gmusicbrowser -cmd PrevSong
gmusicbrowser -cmd DecVolume
gmusicbrowser -cmd IncVolume
gmusicbrowser -cmd Forward 1
gmusicbrowser -cmd Rewind 1
```
