### Post by kerry_s on 2009-08-07

icons is a little tricky, the desktop is under desktop> icons
the filemanager is under view in the file manager, there's zoom in/out it will remember the settings. also under edit> preferences for the side panel icons.
the menus i don't see a way.

### Post by kerry_s on 2009-08-07
right click panel> customize panel.

firefox: edit> preferences> content> fonts&color> advanced
set the size you want and uncheck " allow pages to choose ..."

firefox should also remember zoom level on a per page setting.
