KWin has slower effects than Compiz. Hopefully this is something the KDE team is working on for the next KDE4 point release.

I can run Compiz very quickly on my 8 MB integrated i915 video chipset at work, but I also see slowdown with KWin, although it's still usable. I disable transparency. At home I have an Nvidia 7950 GT and it's still slower in KWin than Compiz is on the i915!

You can run Compiz in KDE 4. Just load up fusion-icon and set compiz to your window manager exactly like you do in KDE 3.5.
