### Post by huck416 on 2009-03-14
I'm having the same issue. Really liked being able to set different wallpapers for each desktop in kde 3.x. It can be done in kde 4.x but only in the "activities" pages. Right click the desktop, add widget and select "activity bar". It puts up a little bar with 'Desktop' named 20 times. Not sure why anyone would want 20 different activities but I haven't found yet how to reduce that to a more manageable number. But when you select a different activity from the bar, you can then right-click on the desktop and select 'Appearance Settings.' Here you can change the wallpaper and the name of that Activity. The new name will then show in the Activity bar on your desktop. 

I can't get the shortcuts to work however. I think they are supposed to give a method of switching between different Activities but it doesn't work. 

Big difference between Activities and Desktops is all windows are on all Activities but not on all desktops.

Hope this makes sense.
