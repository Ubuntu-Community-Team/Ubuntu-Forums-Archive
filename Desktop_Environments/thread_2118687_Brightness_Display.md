### Post by slickymaster on 2013-02-22
If you're using XFCE desktop, there is a plugin that lets you adjust the brightness of the monitor easily.

To install it just run at a terminal window:
```
sudo apt-get install xfce4-power-manager-plugins
```

After install it, just right click on the top panel and choose **panel>add new items** and search for "Brightness Plugin".
