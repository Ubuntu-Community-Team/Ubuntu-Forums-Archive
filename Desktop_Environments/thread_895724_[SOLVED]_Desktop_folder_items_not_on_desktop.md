### Post by TheUnabeefer on 2008-08-20
In KDE4, the Desktop directory can be shown in a plasmoid (widget) called Folder View.  This also allows you to view any other selected folder instead of just the Desktop folder...

This subject ALSO causes FAR too many arguments, as a lot of people don't like this way of doing things.
