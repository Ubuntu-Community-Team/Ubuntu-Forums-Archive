### Post by 67GTA on 2009-05-23
This is the closest I've seen to the actual Vista icons: [http://www.gnome-look.org/content/show.php/LiNsta-icons?content=62759](http://www.gnome-look.org/content/show.php/LiNsta-icons?content=62759)

Download the file, open Menu>System>Preferences>Appearance, and drag and drop the file into the window. This will install the icon theme. You can then select the "customize" button on the appearances window, select the icons tab, and choose the icons you want.
