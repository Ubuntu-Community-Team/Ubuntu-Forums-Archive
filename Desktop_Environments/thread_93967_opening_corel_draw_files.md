### Post by Orval on 2005-11-23
No. What you can do is this.

Open CorelDraw (in Windows of course) and the file you want. Open OpenOffice Draw (also in Windows of course). Copy the drawing from Coreldraw and paste into Draw and save the drawing in the file format you want. In Ubutu you can copy the file from your Windows partition to your home folder.

This works, however not all features from CorelDraw will be preserved. You will lose a lot of your fillings, groups and some other stuff, but it is better than create your drawings all over again.
