### Post by ad_267 on 2009-05-23
Install the Advanced Desktop Effects Settings from Applications -> Add/Remove. Then go to System - Preferences - CompizConfig Settings Manager. Select the Viewport Switcher plugin and go to the Desktop-based viewport switching tab then disable the mouse shortcuts.
