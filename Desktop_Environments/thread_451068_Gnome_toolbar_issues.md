Problem though is that on my left monitor there is a lot of blank space between the window selection and my icons.  How do I get rid of this?  Running dapper btw.

### Post by Jhongy on 2007-05-22
The window chooser is just the same as any other widget or icon on the panel.

Just need to right-click on the handle for the window chooser, make sure "lock to panel" is deselected, the select "move", and move it left or right to eliminate the space.

By right-clicking, you can also select the window chooser properties, which includes the width of tabs, and the ma & min space it takes up.

J
