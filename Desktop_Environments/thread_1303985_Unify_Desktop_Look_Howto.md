### Post by xerman on 2011-03-26
Hello there,

After some time, I came back to this thread.

What is a must in any Linux desktop for an end user?
- all apps show same interface

So, what happens if we want to use KDE apps in Gnome or viceversa? Most apps will not match the desktop. I've seen it easier to solve in KDE, much more difficult to solve in Gnome.

First thing is to find a good Icon Set, for both. Oxygen and OxyGnome, whatever that has same icons for toolbars, desktop, dialogs,... it is not that easy. KDE art developers just care about KDE, Gnome do the same about Gnome.

Qt3 Settings and Qt4 Settings are a must, though I never found them useful.

Font rendering differs a lot in both. So need to install KDE control center in Gnome to use same font and preferences for KDE apps as we use in Gnome. The problem is with some apps that rely only on Qt3, like QCad.

I tried Qt4 Settings with KDE 4.x stock apps and nothing seemed to change, so better have KDE control center and set there appearance settings for KDE apps. This needs same Window and window Control engine in both, so controls will look like identical. I have not found this yet. I thought that freedesktop.org will have something to say about this, but as for now I still have not found that I've been looking for.
