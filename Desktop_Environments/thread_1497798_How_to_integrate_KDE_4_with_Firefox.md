### Post by lovinglinux on 2010-05-31
Dolphin is located at /usr/bin/dolphin

If you setup files to be opened with dolphin in the "Choose Application" window it will work just fine.

Not related to your problem, but you might like [KDE Wallet password integration](https://addons.mozilla.org/en-US/firefox/addon/49357/) extension for Firefox. I don't know if the author fixed some recent issues, but I'm using version 0.4 of that extension and it works great. You can find that version [here](https://addons.mozilla.org/en-US/firefox/addons/versions/49357).
