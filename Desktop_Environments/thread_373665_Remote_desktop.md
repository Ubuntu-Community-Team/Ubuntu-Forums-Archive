### Post by dannyboy79 on 2007-03-01
i would suggest using nxserver. i know it works in xubuntu and ubuntu in dapper. not sure about edgy. there are .debs for the nxserver, nxnode, and nxclient. there is even a nxclient for windows. what this does is allow remote desktop over ssh. there are already plenty of guides for this. good luck. or you could just use ssh and use x-forwarding. or you could just use ssh if all you need is a terminal? I don't suggest using rdp (remote desktop protocol) because it's not very secure.
