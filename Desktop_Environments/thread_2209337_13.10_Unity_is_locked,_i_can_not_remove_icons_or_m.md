### Post by Portaro on 2014-03-05
You probably delete some themes on the compiz.

Try to change the theme decorator on compiz.[URL="http://wiki.compiz.org/Decorators/GTKWindowDecorator"]http://wiki.compiz.org/Decorators/GTKWindowDecorator.


[/URL]  To do things easy try to login in a safe mode or, in a session without effects, like classic session.

After this proceed to edit the config of compiz decorate, and then log on effects session.
