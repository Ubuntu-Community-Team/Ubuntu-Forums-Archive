### Post by FuturePilot on 2008-02-29
VMWare has an option where you can run Windows in a VM in seamless mode which might be kind of like what you want to do. It's very cool. :) This can also be done with VirtualBox as well. You'll need to install the PUEL licensed version of VirtualBox if you want USB support in the VM (for your iPhone) as the open source version doesn't have USB support.  I've had better luck with VirtualBox but you can use whatever you want ;)
