### Post by bruno321 on 2009-11-26
Hi. I have Kubuntu 9.10, and two desktops as set by default. But I want them to act like in KDE 3, that is, I don't want programs in the tray to be shared between them. How can I change that?

Thanks

---

### Post by Zorael on 2009-11-26
It's in the task manager settings. Unlock your widgets, then hit the cashew on the panel, then right-click the task manager (anywhere that's not occupied by a currently running app), then Task Manager Settings. I think the option is called *Only show tasks from the current desktop*.
