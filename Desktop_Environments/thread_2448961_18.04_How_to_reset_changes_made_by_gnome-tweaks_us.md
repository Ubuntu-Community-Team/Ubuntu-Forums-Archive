### Post by soulflight on 2020-08-17
I tried to install arc-darkest-suru from gnome-look.org and something went wrong. Now gnome-tweaks wont open anymore to change it back. I also dont have access to my file browser. My System is more or less unusable that way.
 When I try to open gnome-tweaks in terminal, it returns an error message like:

```
"Gtk-Warning: Could not load pixbuf from icon theme - Aborted.  Arc-Darkest-suru/actions/16/image-missing.svg: unrecognized file format  (gdk-pixbuf-error-quark, 3)"
```

I need a way to revert to the default ubuntu gnome theme, or change the icon theme, without needing to open the GUI of gnome-tweaks. Any way of doing that?

---

### Post by soulflight on 2020-08-17
"dconf reset -f /" done the trick
