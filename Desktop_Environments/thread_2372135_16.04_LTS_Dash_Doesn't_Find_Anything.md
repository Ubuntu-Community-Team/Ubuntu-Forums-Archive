### Post by untrustytahr on 2017-09-22
I had a 12.04 or 14.04 (not sure which) unity install that did this.  The solution I found was to rename the zeitgeist file to something like zeitgiest.old.  Log out and back in.  I think the file was in ./local/share/zeitgiest.

If it doesn't work you can just rename it back to its original name.
