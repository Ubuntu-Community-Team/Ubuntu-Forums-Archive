### Post by daller on 2006-11-22
Just like creating a link on the desktop.

Right click -> Create -> Link to application

* Now name the link something appropriate

* Goto the tab: "Application"

* Enter the command you would like to be executed. (stardict)

* Press the button "Advanced options"

* Unmark the "Enable launch feedback" checkbox, and click "OK"

* Click "OK"

- That should be it!
