### Post by CantankRus on 2015-03-18
Any downloaded cursor theme must also be installed to alternatives and set as the x-cursor-theme in alternatives to be consistent.
See [**_[COLOR="#B22222"]HERE[/COLOR]_**]("http://ubuntuforums.org/showthread.php?t=2221942&p=13028225#post13028225").
