### Post by 3Miro on 2012-05-15
Under Linux you should use Firefox or Chrome/Chromium.

Wine has an implementation of IE for Linux, but it my experience it doesn't work very well. Wine is available for the Software Center and here is the info for their iexplore

[http://wiki.winehq.org/iexplore](http://wiki.winehq.org/iexplore)

If you absolutely must use IE, then you should look into Virtual Box and installing a full copy of windows to work under Linux.
