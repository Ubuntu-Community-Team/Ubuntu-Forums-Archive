### Post by Frogs Hair on 2011-07-31
Unity uses the global menu , so the the menus as they are displayed in Classic Gnome don't appear. The application menu is displayed on the top panel for the currently active application .

The launcher panel has replaced the traditional menus . If you would like to use the traditional menus you can login to Classic Gnome see the link . 
The Classic Gnome option will be removed in the next release .

I should add that the application menu in Unity , File , Edit , and the rest will remain hidden until the pointer is moved over the panel .

[http://ubuntugenius.wordpress.com/2011/05/27/ubuntu-classic-desktop-how-to-log-into-the-old-gnome-environment-instead-of-unity/](http://ubuntugenius.wordpress.com/2011/05/27/ubuntu-classic-desktop-how-to-log-into-the-old-gnome-environment-instead-of-unity/)
