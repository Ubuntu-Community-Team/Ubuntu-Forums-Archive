### Post by Warbo on 2006-09-12
Definetly check out Nvu (it is in Universe repository). It is WYSIWYG (graphical), but also lets you see the HTML if you want to. There is also Mozilla Composer (if this doesn't have it's own package, then it will probably be in a more generic mozilla package), although Nvu is basically Mozilla Composer rewritten in GTK (GNOME's GUI).

Other than those I will just say to NOT use Abiword, because it creates such crazy HTML it is stupid!
