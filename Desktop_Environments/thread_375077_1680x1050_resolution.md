### Post by scxtt on 2007-03-03
-- you shouldn't need any modelines (the cleaner xorg.conf is, the better) ...
-- make sure you're using the nvidia driver (or something better than 'vesa', i use 1600x1200 and only 1280x1024 works w/ 'vesa' [until i get fglrx going])
-- use these (specific) H/V values:
[indent]Horizontal sync.: 30-82kHz
Vertical sync.: 56-76Hz[/indent]

i'd start by doing a "sudo dpkg-reconfigure xserver-xorg" and using nvidia or nv and select a res. you know works, then edit xorg.conf specifing the H+V resolutions and 1680x1050 ...

and as a side note - CTRL+ALT+BACKSPACE will restart X, no (real) need to restart - even restarting gdm reloads changes for me ...

edit: iamhugeinjapan's xorg.conf is NICE :) ... i'd use it, then change the refresh rates to above specs (make sure nvidia is installed) ...
