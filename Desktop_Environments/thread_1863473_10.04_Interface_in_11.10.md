### Post by BuddMan on 2011-10-18
I figured it out.  You have to manually download and install the Netbook Launcher EFL.  This was available in the Ubuntu repository up to 11.04, but I guess just recently removed for 11.10.  Regardless, it is possible to manually install and use it as long as you have all the dependencies, most of which are available in the repository.

[Here]("https://launchpad.net/launch-lite-proj") is the link for those interested.
