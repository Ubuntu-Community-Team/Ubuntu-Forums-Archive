### Post by 2uQ6KKAD on 2015-08-19
Hey!

The standard procedure should be:

1. Set the number of Virtual Desktops to more than 1 (I suggest 4 for a cube, duh!)
       System settings -> Workspace Behaviour -> Virtual Desktops.
2. Activate the cube effect.
       System settings -> Desktop Effects -> All Effects     (check "Desktop Cube" and "Desktop Cube animation")

The cube should work now if you switch virtual Desktops using Ctrl. + F11 or by any other means, for example if you use the scroll wheel on the empty desktop. You can also add the "Pager" widget to your task bar or set one of the screen edges to activate the cube:

System settings -> Workspace Behaviour -> Screen edges  (click on one of the fields on the Monitor image and check "Desktop Cube - Cube".
