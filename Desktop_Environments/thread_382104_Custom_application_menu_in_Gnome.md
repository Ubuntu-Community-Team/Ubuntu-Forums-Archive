Does anyone have any suggestions on how a person could go about having a custom application menu in Gnome? ### Post by 23meg on 2007-03-11
Go to System / Preferences / Menu Layout to edit the menu.

### Post by steve0921 on 2007-03-11
I've just done a bit of playing around with drawers, and you can (at least slightly) get around your two weaknesses:

submenus - just add a drawer to a drawer
no names - while it won't display immediately, you can add a name and description to custom launchers (I assume these are what you're using) that will appear as a tooltip.

I don't really know about the bugginess. Nor can I add and edit a second menu without changing the main menu in exactly the same way (which eliminates the point)--this really should be possible.
