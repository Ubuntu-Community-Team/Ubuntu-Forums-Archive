### Post by mothergoose729 on 2009-12-26
Well I ended up solving my own problem. I found out that twinview with the monitor that wasn't working set as the main display works really well and I know have excellent multimonitor functionality. To apply the changes I configured everything in my nvidia control panel, clicked "save changes" then "view prefiew". Opening up a terminal I typed 

```
sudo nano /ect/X11/xorg.conf
```

I backed up my conf then copied and pasted the new one from Nvidia. Rebooted and everything works now.
