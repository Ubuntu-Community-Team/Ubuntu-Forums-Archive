### Post by taurus on 2009-02-02
It's mounted.

```
df -h
```
If you want to fix the "unclean shutdown" message, you can use ntfsfix to fix it.

```
sudo umount /dev/sdc1
sudo apt-get update
sudo apt-get install ntfsprogs
sudo ntfsfix /dev/sdc1
sudo mount -t ntfs-3g /dev/sdc1 /media/hardas
df -h
```
