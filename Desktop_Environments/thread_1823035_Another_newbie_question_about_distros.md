### Post by XubuRoxMySox on 2011-08-11
Hi!

For newbies I always urgently recommend the LTS editions. Long Term Support versions starting with 10.04 (Lucid) are built from Debian *Testing* rather than Debian *Unstable* like the "regular" releases. The "Debian stuff" that 10.04 was built from has since been released as "Squeeze," which is the current version of Debian *Stable*. And in Linux, it jjust doesn't get any more stable and rock-solid than Debian Stable.

It's alot less likely to be b0rked by an update, and it's supported for another two years! Awesomeness.

I'm running Xubuntu 10.04 on my old hand-me-down 'puter and it totally rocks the universe! LOL, I'm a fanboy, I admit.

But yeah, use the LTS edition until you learn Linux a bit more, then when you're brave and can afford to take some risks, go for the latest version and play!

-Robin
