### Post by Frogs Hair on 2012-07-15
If I remember correctly  that menu was removed in 11.04 . The Gnome Shell and Classic are provided by Gnome and  that is why there is no Ubuntu branding on the panel.

The main menu application still has the add new menu option ,but I have not tried it on 12.04 because I use the Gnome Shell and Unity . "Mate" is the Gnome 2 fork and can be installed on 12.04.
