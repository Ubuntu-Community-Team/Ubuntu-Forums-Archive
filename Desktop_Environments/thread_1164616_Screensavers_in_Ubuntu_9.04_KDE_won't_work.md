### Post by AMV_Ph34r on 2009-05-19
I installed KDE on Ubuntu 9.04 today. While using it, I realized that the screensaver was set to blank screen. I tried to change it, but the only available screensaver is the blank screen. When I look in the KDE menu, under Applications>System>ScreenSavers, I see a bunch of screensavers there, but they don't do anything when I click them. How can I set any of them as my screensaver?

---

### Post by it_fixxer on 2009-08-11
Go into your package manager and install the following packages...

kscreensaver-xsavers

kscreensaver-xsavers-extra

kscreensaver-xsavers-webcollage

These will allow you to use the open GL savers that come with x.

Hope it helps.
