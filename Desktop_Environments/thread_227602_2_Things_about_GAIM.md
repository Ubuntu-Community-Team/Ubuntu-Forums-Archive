### Post by fourchannel on 2006-08-02
I tried to get MSN profiles for a long time, but could never get it to work. Things might have changed since then.

But I can answer your question about emoticons.

It's a drag and drop setup. You have to go to gaim's themes page [https://sourceforge.net/tracker/?group_id=235&atid=746976](https://sourceforge.net/tracker/?group_id=235&atid=746976)

download the packages, then launch file browser and unmaximize the window(so you can drag and drop the files from the file browser directly into the smiley window.) In gaim, go to preferences - conversations - smiley themes.

and that should work.

### Post by roman5x3 on 2006-08-02
I am not sure if this is what you were looking for but try searching synaptic for gaim.  In there you should find "gaim-themes" this will give you roughly a dozen different sets of emoticons. If you are not running synaptic pull up a terminal and type
```
sudo apt-get install gaim-themes
```
This should install the themes for you.

You can also have a look at this website [http://sourceforge.net/tracker/?atid=746976&group_id=235&func=browse]("http://sourceforge.net/tracker/?atid=746976&group_id=235&func=browse")

Hope this helps.

1: Do you mean the "personal message", which also shows the current playing song sometimes? There's no support for it in GAIM and I couldn't find any plugins.

2: GAIM doesn't support custom emoticons, it only allows you to change the appearance of the default ones.

You should have a look at aMSN, it supports MSN functions better, signing in appear offline is supported, as well as drawn messages, nudges, and personal messages from contacts are shown, but I haven't figured out how to set them yet. It also supports custom emoticons according to their front page, but I haven't dug around the options much yet.

Oh yeah, here's a link: [http://amsn.sourceforge.net/](http://amsn.sourceforge.net/) Just grab the latest dapper .deb file.
