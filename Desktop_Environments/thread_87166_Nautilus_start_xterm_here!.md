### Post by spizkapa on 2005-11-07
Hi there,

does anyone know if there is a way to tell nautilus to start me an xterm in a folder that I'm browsing? For example, you've gone down to something like:

/home/fred/myfiles/somefolder/anotherone/evendeeper/getting_silly/even_more/

in nautilus and then you realise that you need to do a ./configure or something. I'd love to be able to right-mouse click and say start me an xterm at this directory.

I understand that perhaps it's a stupid directory structure but you'd be surprised how often it turns up. I know that you could symlink the deeply nested tree somewhere else but that's not the question.

Anyone?
