### Post by Zeroangel on 2006-02-11
GTK2 is a set of widgets and other interface components that determine how your applications will look even as the themes are changed. GTK2 is one of the more modern interface improvements introduced in gnome. I would recommend you get the GTK2 version of any program if there should be a choice.
