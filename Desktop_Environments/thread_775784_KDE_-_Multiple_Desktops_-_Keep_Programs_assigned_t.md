If your using Compiz-Fusion then you can install the Compiz Config Settings Manager (CCSM) and use the place plugin to put the application on any desktop of your choosing using windows matching. More about this at the Compiz-Fusion wiki:

[http://wiki.compiz-fusion.org/Plugins/Place](http://wiki.compiz-fusion.org/Plugins/Place)

In order to install the CCSM manager you need the universe repositories enabled then:
```
sudo apt-get install compizconfig-settings-manager
```
the Compiz-Fusion WM is **very** powerful and using CCSM allows you to customize the controls to an infinite degree.
