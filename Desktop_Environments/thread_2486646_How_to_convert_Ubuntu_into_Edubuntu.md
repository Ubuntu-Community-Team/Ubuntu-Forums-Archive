### Post by deadflowr on 2023-05-07
Installing the edubuntu-desktop meta-package will install all edubuntu packages.
it uses the same desktop as Ubuntu (gnome) so that won't change much.
if you want a more focused package base for specific ages then there are several including the following

[list=1]
[*]ubuntu-edu-preschool
[*]ubuntu-edu-primary
[*]ubuntu-edu-secondary
[*]ubuntu-edu-tertiary[/list]

The edubuntu-desktop packages all of the above.

edit: i believe they have only been packaged for Ubuntu release 23.04, so far.
