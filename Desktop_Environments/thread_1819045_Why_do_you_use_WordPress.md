### Post by forrestcupp on 2011-09-12
It's for dynamic sites, like blogs. It's pretty good for creating dynamic sites that look like static sites, too. 

The advantage is that once you get the site designed, WordPress makes it very easy for a less experienced person to maintain it and update it.

Sorry, I use Joomla for my clients and I do not agree with that.
Joomla is secure. 
However, like with all software, you'll need to keep the software up to date.
And because 3rd party extensions for Joomla are so easy to install, people just install those and forget to upgrade them.

I am not sure how WordPress or Drupal work with extensions of the core functionality, 
but I'll guess that if it's easy to install 3rd party extensions in those CMS systems, 
that they will have the same problem with not updated 3rd party extensions.

Drupal, Joomla and WordPress are all open source software, supported by large communities.
That would be enough reason for me to choose any of them... ;)
