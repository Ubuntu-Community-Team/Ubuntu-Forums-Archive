### Post by jem7v on 2007-01-09
Try installing the edubuntu-artwork package in Synaptic (System->Administration->Synaptic Package Manager).  I think that probably contains the theme files and icons you want, and then you would change to them from System->Preferences->Theme.

Additionally, you can find more themes in a lot of different places!  Here are a few I can think of off the top of my head:
[http://www.gnome-look.org/](http://www.gnome-look.org/)
[http://art.gnome.org/](http://art.gnome.org/)

GTK2 themes (aka Application Themes) change the way scrollbars and buttons and windows and colors look; Metacities themes (aka Window Borders) change the way the borders around your windows look.  To install them, download the file and then drag and drop it onto the theme selector (the one mentioned above).
