### Post by Frogs Hair on 2012-03-21
Check the software center anyway to see if it is listed. If you used a PPA to install it would have required 3 commands . You can install and use the synaptic package manager to remove E17/ Enlightenment.

Use the search filter in synaptic to locate E17 or Enlightenment and all the related packages will be listed . Right click on the line where the package is displayed and mark for complete removal . Select apply after marking the packages. If any auto removable or residual packages appear remove them the same way .
