---

### Post by zombifier25 on 2013-02-18
You can't, because Fluxbox is a window manager, and Unity already has a WM which it will not work without, Compiz.

---

### Post by zaa526 on 2013-02-19
DO you know if there is another way to activate the window tabs feature? I tried doing it in Compiz but that plugin doesn't come with the 12.10 repositories. :/

---

### Post by zombifier25 on 2013-02-19
The group plugin (which includes the tab feature), along with some other extra Compiz plugins, currently does not work with the latest Compiz and therefore disabled. Some of them have been fixed, but you'll have to keep your finger crossed for the rest.

You can keep track of the status of the plugins here: [https://bugs.launchpad.net/compiz/+bugs?field.tag=gles](https://bugs.launchpad.net/compiz/+bugs?field.tag=gles)

I don't know what other WM has this feature, but you'll have to give up on Unity to use them.
