### Post by shiroyoshida on 2009-08-03
I'm using Ubuntu 9.04 and I have installed CompizConfig Settings Manager from the Synaptic Package Manager. 

Once you install it, go to System -> Preferences -> CompizConfig Settings Manager and from the sidebar on the left select Window Management. Apart from an option to Group and Tab Windows, you'll also find loads of other ways to tweak your desktop to perfection! 

Hope this helps. :)
