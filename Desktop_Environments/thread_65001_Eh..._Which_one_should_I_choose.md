### Post by Jackster on 2005-09-12
Intel x86 is for Pentium, Celeron, Athlon XP and Duron machines

PowerPC is for an Apple Mac

AMD64 is for Athlon64 and Opteron


Basically, if your running Windows Home/Proffesional, you have x86
If you run Mac OS X, you have a PowerPC
And if you run Windows XP Proffesional x64, you have an AMD64
