### Post by birkopf on 2010-03-18
I actually solved the problem by using method from other problem. 

I turned off all applications and went to System -> Preferences -> Startup Application -> Options -> Remeber Currently-Running Application and unclicked Automatically remember running apps before log out. 

That had soled problem.
