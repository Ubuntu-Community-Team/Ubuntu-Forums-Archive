### Post by arm-c on 2011-01-29
Displex 0.4.1 RELEASED

This little indicator has added some new features since my last post.  You can now get from the [displex]("https://sourceforge.net/p/displex") page on sourceforge:



Change log:

version 0.4.1 -- 29 JAN 2010 
* Bug Fix: Emerald was not showing correctly when installed. Emerald Theme Manager and Decorator Menus now show correctly. (Thanks for the report Andrew) 

version 0.4 -- 28 JAN 2010 
* Fixed inconsistancy in menu. Now, if emerald is not installed, it will not show. Same with decorator menu. This will help de-clutter the menu. 

* Modified menus again. 

* Indicator Icon now updates according to theme. Verified with the following icon sets: 
- ubuntu-mono-dark 
- ubuntu-mono-light 
- hicolor 
- elementary-monochrome 
- Faenza 
- Faenza-Dark 

* Added BASIC support for Disper (Feedback needed) - Menu doesn't show unless Disper is installed. - Note -- Disper only works PRESENTLY for NVIDIA cards and I do not have an NVIDIA based system. 

* Added BASIC support for the ACERHK driver (activate / deactivate some hardware used in several older computers.) - Menu doesn't show unless acerhk.ko is installed and active. - If you know you need that driver, download and install "AcerHK GUI" from sourceforge [https://sourceforge.net/projects/acerhkgui](https://sourceforge.net/projects/acerhkgui) 

version 0.3 -- 17 JAN 2010 
* Added BASIC support for Disper ([http://willem.engen.nl/projects/disper](http://willem.engen.nl/projects/disper)) 

* Added support for Metacity Compositing 

* Modified Menus
