### Post by kansasnoob on 2013-01-27
Actually 13.04 is still in development, but regardless there is no need for any PPA.

If you want the *buntu branded version of LXDE you can just install "lubuntu-desktop", or if you want to go lighter weight w/o all of the apps you can just install "lubuntu-core".

But the "unbranded" version is in the repos too, package name is just "lxde", or the lighter "lxde-core" :D
