### Post by zombifier25 on 2012-04-30
Install a tool like Ubuntu Tweak: [http://ubuntu-tweak.com/](http://ubuntu-tweak.com/)
```
sudo add-apt-repository ppa:tualatrix/ppa
sudo apt-get update
sudo apt-get upgrade
```Then open it, go to Theme and you can configure the windows decoration and themes separately.
