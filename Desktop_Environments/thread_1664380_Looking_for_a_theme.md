That dock you're talking about is probably AWN (Avant Window Navigator), Cairo Dock or maybe Docky. I'd say there's no harm in trying them all :).

For your last question: I'm not familiar with yahoo, but I know hotmail only supports the pop3 protocol (so you'd have to make sure the message isn't deleted on the server if you want to access it through webmail or mail client on another computer/OS) and Gmail supports imap (besides pop3).
If you're fine with Evolution I'd recommend to check that program out since it makes use of the messaging menu. ### Post by OldSmoky2 on 2011-01-11
If you haven't already, check out the Bisigi themes at bisigi-project.org. They are varied and really, really nice. And as a newbie, sort-of, (less than a year from Mac), I've learned it's very easy, once you install a theme, to play with it in the System>Preferences>Appearance menus. Install a theme, then click "Customize" there and you can change the icon set, borders, colors and controls to really personalize it. Right now, for example, I'm using Bisigi's Eco theme, but tweaked a few colors, changed to a Faenza icon set from another theme and installed a wallpaper from some other theme.
As for the email thing, I haven't used either one, but if you install Screenlets from the Software Center (or Synaptic), I see that it has a GMail notifier screenlet and a more general email notifier screenlet that might give you what you're looking for.
