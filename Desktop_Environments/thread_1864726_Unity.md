### Post by Bao2 on 2011-10-19
You must install the gnome-panel:

sudo apt-get install gnome-panel

and then log out and click the gear in the login window and choose Gnome Classic.

### Post by 3Miro on 2011-10-19
If you mean classic as in 10.10 or 10.04, then the answer is: you can't.

If you don't like Unity, you can try Gnome-shell (some people think it is worse, some people think it is better), Gnome-session-fallback (similar to Gnome 2, but not quite the same), XFCE (very similar to Gnome 2, but usually faster and perhaps a bit more involved to setup), KDE (powerful DE). All of those are available in the Software Center.
