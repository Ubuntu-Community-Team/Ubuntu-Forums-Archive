### Post by chucky chuckaluck on 2008-08-01
you could try this solution from the arch forums - [http://bbs.archlinux.org/viewtopic.php?id=48563](http://bbs.archlinux.org/viewtopic.php?id=48563)
if you don't want nautilus to control the desktop, make sure you include *--no-desktop* whenever the *nautilus* command is needed, so that the whole thing looks like *nautilus --no-desktop*.
