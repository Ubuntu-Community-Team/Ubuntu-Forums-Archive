### Post by TransitMan on 2007-04-12
I was a MSMoney user for a longtime.
When I decided to switch to a Linux OS, I looked at all the options available, finally installing both GnuCash and KMyMoney2. Ran both to do a side by side comparison to see which one would handle the transactions I needed and used in MSMoney.
Finally decided to use KMyMoney2.

If you are going to get away from Windows, you have to decide on not hanging onto those programs that you are familiar with in Windows. 
You must open your mind to the endless possibilities of Linux and go forth. Otherwise, you'll end up in going back to Windows because because you can't get comfortable with or you don't want to learn a new program due to the "Comfort" factor of MSMoney.

Try the programs for a month, without relying on Windows. You'll eventually see that using one of the Linux programs can be just as useful as what MSMoney was.
