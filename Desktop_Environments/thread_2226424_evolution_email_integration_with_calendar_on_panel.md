### Post by Toz on 2014-05-30
I don't have/use evolution so I can't comment on whether this works, but...

Xfce has two plugins that provide a calendar on the panel. One is called the "DateTime" plugin" and the other "Orage Panel Clock". The Orage panel clock calendar has an "Exchange Data" function that allows you to integrate with other calendars. As far as I can tell, it will integrate with Evolution. More information is available [here]("http://www.ghacks.net/2010/07/09/orage-xfce-calendar-application/").

I'll see if I can find some time to test this, this weekend.
