### Post by An Sanct on 2012-03-04
in the Wine Configuration (shown on picture) check the "Graphics" tab and check, if the "Emulate Virtual Desktop" checkbox is checked, if so, uncheck it and try again 

This setting makes your windows applications believe, you are running a x*y limited display, most commonly set to 800*600.
