### Post by ehamberg on 2005-10-09
Hi.

In a default install of Kubuntu you get a CD-icon on the desktop if you insert a CD, a Memory Stick icon if you plug in a memory stick and so on.

I installed Kubuntu but kept my homedir with my KDE settings, because it would take forever to reconfigure everything, but then I don't get desktop icons for media I plug in to the computer. How can I get those icons to appear?

The device gets mounted, and the "What do you want to do" window pops up, but I still want the icon on my desktop.

Can anyone point me in the right direction?

---

### Post by xtacocorex on 2005-10-09
Right click on the desktop, click on Configure Desktop, go to the Behavior section, then the Device Icon's tab

Then you can set what you want shown [if it's mounted or not]

### Post by xtacocorex on 2005-10-09
Glad I could help. If it doesn't work, check the General tab under Behavior to see if the option for allowing desktop icons is checked.
