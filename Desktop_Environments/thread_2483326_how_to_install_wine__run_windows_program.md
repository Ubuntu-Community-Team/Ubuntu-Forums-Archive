### Post by TheFu on 2023-01-27
> **cmacc77 said:**
> It wasn’t in my software list.   I followed some other instructions (everywhere I looked provided a different solution) and my pc won’t turn on now and the yellow light is blinking.  I like this OS but it’s kicking my ass

a) If you come from some other OS, then the learning curve for Unix-like OSes is steep.  That's to be expected. Linux is a new OS. It works different.
b) Be careful randomly searching for instructions on the internet.  Until you have more experience, might I suggest that you only follow instructions from websites with "ubuntu" in the domain (.com) name?  
c) When looking for how-to guides, search for ubuntu + the release + the topic.  e.g.:  "ubuntu 22.04 wine install"
d) When looking at how-to guides, if there aren't lots of screen images, then it is assumed you are in a terminal. I usually have 2 open ... 1 for reading the README file and the other to enter commands.

WINE isn't magic and there are many, many, many, Windows programs that don't work with it.  IME, about 80% of programs don't work, usually because they use some MSFT technology that WINE doesn't have implemented - like OCX controls (last time I looked OCX didn't work). Almost none of the programs that quote" work "unquote, only work about 80% of the functions.  

WINEHQ is the website where there is a compatibility application DB with lists of thousands of applications, versions, and tricks to get them working along with how well they work when done.  We all want "platinum" level support (everything in the program works fantastic!), but usually end up with "bronze" or none.
[https://appdb.winehq.org/objectManager.php?sClass=version&iId=40213](https://appdb.winehq.org/objectManager.php?sClass=version&iId=40213) says that Garmin Express is "bronze" level support. That may be sufficient for your needs. IDK.  There are install instructions for Linux Mint, based on Ubuntu 18.04 (tested in 2021) which claim "gold" support for Forerunner 920XT watch.  Newer tests with newer Linux in 2022 reported "garbage" ... that's the "it doesn't work" answer, but they weren't on Debian based Linux. Perhaps it works perfect now on Ubuntu? There's only 1 way to know.

I'm just try to set expectations.  Sadly for me, it seems that games are where the most effort is spent to get things working under WINE.  I don't really game, so that isn't very interesting to me.  I want business and financial software to work. Sadly, that usually isn't what the masses working hard to make these things "just work" care about.

WINE is a program launcher for Windows programs. Typically, you need to pre-configure the fake Windows environment specifically to your program (and the installer) will work at all.  The appdb for the specific program usually has the steps needed.  Most require the use of "winetricks", which is a script to add different environments and DLLs to your WINE environment. ### Post by monkeybrain20122 on 2023-01-28
I think the problem you cannot just click an .exe file to launch it is that the wine launcher .desktop file is hidden maybe for some security reason.  To make your .exe files clickable you have to first copy the wine.desktop file to the right place

In my system it is in /usr/share/doc/wine/examples/wine.desktop

so you should copy it to /usr/share/applications

```
sudo cp /usr/share/doc/wine/examples/wine.desktop /usr/share/applications
```

now (you may have to logout and login again) if you right click on an .exe file and check properties > open with and go down the list you should see the wine launcher icon, make it the default (if it is not the default already) Now clicking .exe file should launch it.

To install Wine and run Windows programs on Ubuntu, you need to follow these steps:


1. Open the terminal: You can open the terminal by pressing Ctrl + Alt + T.


2. Change to the directory where the Wine source is located: Use the "cd" command to navigate to the directory where the Wine source is located. For example:


javascript


cd ~/Downloads/wine-5.21

3. Run the configure script: Type the following command and press Enter:

bash

./configure

4. Build Wine: Type the following command and press Enter:
go


make

5. Install Wine: Type the following command and press Enter (you may need to enter your password):
go

sudo make install

6. Run a Windows program: Type the following command and press Enter, replacing "program_name.exe" with the name of the Windows program you want to run:


wine program_name.exe

After completing these steps, Wine should be installed and you should be able to run Windows programs on your Ubuntu system.


Note: If you encounter any errors during the installation process, you may need to consult the Wine documentation or seek assistance from the Ubuntu community.
