### Post by illu45 on 2007-04-02
Its quite easy to switch between KDE and Gnome in Ubuntu, you just need to log out and log back in. If you haven't yet installed KDE,  then look at Majorix's post on this thread: [http://www.ubuntuforums.org/showthread.php?t=399332&highlight=install+kde](http://www.ubuntuforums.org/showthread.php?t=399332&highlight=install+kde) 

If you already have it installed, then go into System -> Prefernces -> Session and make sure there is a KDE session, if one doesn't exist, then add one. If there is a KDE session, then to switch into KDE all you have to do is log out, hit the Sessions button on the bottom of the screen and select KDE. Then log in, and you'll have KDE :).

EDIT: If you've yet to install it, aysiu's guide is the way to go.
