### Post by raja on 2007-05-25
I assume you are using Ubuntu with the gnome desktop. Go to System -> Preferences -> Themes in the menu and you can choose among the available themes. If you want to try other themes, you can look at [http://www.gnome-look.org/](http://www.gnome-look.org/) where you can select a metacity theme that you like. The file to download will be in the form of *.tar.gz. Download it and **do not uncompress it.** Now go to the Themes from the menu again and choose 'install theme'. Select the file you downloaded and it should become available to use.

### Post by Happy_Man on 2007-05-26
Take a deep breath.... calm down..... good. 

Now, let's assume you have a file named theme.tar.gz on your desktop. It is for a theme, that you like. Go to System --> Preferences --> Theme. A window will come up. Drag theme.tar.gz into the theme manager window. It will automagically install and give you an option to either keep the current theme or change to the new one. Did you at least get that?

NB: If you are trying to install icons, open up the theme manager, and select customize. Highlight the the "icons" tab, click install, and navigate and select theme.tar.gz with the icons in it. Hope this helps!
