It's very helpful to be able to zoom thumbnail pictures without zooming.  Nautilus (or whatever it's called now) does this fairly well, but because Gnome stripped several features from Nautilus, I would like to replace it.

Does anyone know of a workaround?  Is there a hidden configuration option for Nemo and Caja?  Or an imminent bug fix?  Or is there an alternative file browser that does not have these problems?
