### Post by Copper Bezel on 2011-04-04
You can, but which version you get depends on the version of Ubuntu you're running. Before 10.10, you get the old NBR interface, which was phased out and won't run on later versions of Ubuntu; in 10.10, you get a sluggish, limited version of the new interface (basically a test) with minimal desktop effects; and in 11.04, you get the full Unity interface, but that's shipped by default.

It would probably be best to wait until the end of this month and upgrade to Natty with the full Unity interface.
