### Post by spjackson on 2021-10-04
Thunar, the default File Manager on Xfce, supports this. If you drag the current url from url box in Firefox and drop it on a Thunar window, you get a dialog for creating a desktop configuration file (.desktop). I don't know about other File Managers.
