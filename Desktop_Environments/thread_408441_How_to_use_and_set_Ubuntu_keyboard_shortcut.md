### Post by aysiu on 2007-04-16
You don't have to know what 0xe9 is. Just press the key combination or key you want. The system will interpret it as it sees fit.

If you want keyboard shortcuts for custom commands (more than you can find in System > Preferences > Keyboard Shortcuts), then try this:
[http://doc.gwos.org/index.php/GnomeKeyboardShortcut](http://doc.gwos.org/index.php/GnomeKeyboardShortcut)
