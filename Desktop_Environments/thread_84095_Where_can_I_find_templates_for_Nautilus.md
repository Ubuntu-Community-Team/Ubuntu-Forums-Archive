### Post by magnusbb on 2005-10-30
Funny, I did also wonder about that this morning, and found an easy solution for it.

First, create a directory called **Templates** in your home folder, if doesn't exist. 

Then open OpenOffice Writer and save an empty document as a file you ie. call "OpenOffice Document". Then put this file in the Templates folder. 

Do this with all kinds of applications. Make a prototype, and put it in the folder. It will be available at once.
