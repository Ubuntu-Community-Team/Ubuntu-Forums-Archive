### Post by frodon on 2006-04-14
The fastest desktop is XFCE, this desktop always respond faster than KDE or GNOME.
However, you have to know that the dapper drake release will add some speed improvements to ubuntu, also by default ubuntu install the i386 kernel (if you use the X86 version) and you should want install the kernel which correspond to your processor (i have an AMD therefore i use the k7 kernel), if you have a pentium 4 with hyperthreading you have to know that it's not enabled by default and therefore you will gain a lot of speed if you enable it.
In my case i use GNOME which use metacity as default window manager, metacity in gnome 2.12 (breezy version) is quite slow and RAM eater therefore i changed my window manager to xfwm4 which is the XFCE window manager, thus i gain a bit of speed and free RAM.

Tell me if you want to know more about one of those way to improve speed, there are a guide for each in this forum.
