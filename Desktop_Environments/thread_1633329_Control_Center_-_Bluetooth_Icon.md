### Post by Frogs Hair on 2010-11-29
Icons are located in two places , one is the home > hidden folders > .icons. This is where icon sets that you download are stored. The the other location is filesystem / user /share > icons , these are icons that come pre installed .
 
When you have found the folder with the theme you are using look in the apps folder for the Blue tooth icon and the name is under the icon. in the icon set that I am currently using it is called bluetooth.jpg but this is a set I downloaded from look.org .
 
Some icon sets downloaded from the internet my be incomplete and missing icons , replacing the icon with a paper document symbol or an X .
