### Post by Miata-MS on 2009-04-27
```

sudo apt-get install awn-applets-c-extras awn-applets-python-extras

```

PyNot - System Tray
Awn Notification Daemon - Nice 'glass' look notifications
AWN Main Menu - NOT the same as Cairo Main Menu. I agree... the Cairo one isn't' very good
Places - Replaces your gnome-panel 'Places' menu
Quit/Log-out Applet - Quit/Logoff Dialog


Hope that helps!!
