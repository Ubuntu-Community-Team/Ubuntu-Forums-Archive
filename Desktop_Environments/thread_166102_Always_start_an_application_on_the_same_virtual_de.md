### Post by Ptero-4 on 2006-04-25
does it support the --desktop (or similar) parameter. If it does you can pass it with the number of the virtual desktop you want it in. This parameter works in most modern WM's (kwin and sawfish works, metacity may work if the gnome team haven't removed the required functionality yet).
