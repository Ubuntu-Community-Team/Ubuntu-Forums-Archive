### Post by Dennis N on 2012-03-04
The color is coded into the theme, so you have two choices:

1) Install a different mouse cursor theme that provides cursors of different colors. Many can be found at gnome-look.org. See X11 Mouse Themes category.

2) Create your own cursor, or possibly edit an existing mouse theme with the cursor editor plugin for GIMP. Google search should locate this plugin. I have never used it, but I know it exists.
