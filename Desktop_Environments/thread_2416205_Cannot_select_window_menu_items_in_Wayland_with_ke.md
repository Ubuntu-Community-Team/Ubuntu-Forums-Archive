[*]Clicking on menu item with the mouse works.

[*]The move shortcut <Alt> + <F7> also works from keyboard alone.
[/LIST]
The only thing that doesn't work is the first step: selecting the menu item with <Enter>.

Has anyone else been affected by this and if so, did you find a solution?
