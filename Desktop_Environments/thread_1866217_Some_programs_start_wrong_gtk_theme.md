### Post by mcduck on 2011-10-21
What theme you are using currently, and is it compatible with both GTK2 and GTK3?

All the new applications from Gnome are using GTK3, so if your theme only has GTK2 support they would end using the default theme instead.

### Post by mcduck on 2011-10-22
No, there definitely isn't any GTK3 version of Clearlooks available, and as both Rhythmbox and Aisleriot are Gnome's programs you really need a up-to-date theme for them, regardless of what desktop environment you are using.

There are some GTK3 themes available here, you should look for one that says it includes both GTK2 and GTK3 if you want all your programs to use the same theme: [http://gnome-look.org/index.php?xcontentmode=167]("http://gnome-look.org/index.php?xcontentmode=167")
