### Post by 23meg on 2007-02-26
Breezy's library versions are too old for you to let you use a recent version of Compiz the way you do with the current releases. If you search the forums for "breezy xgl", you'll find a guide that you can use to get an ancient version of Compiz running, but I don't recommend it for anything other than demonstration purposes; you'll have all the serious bugs that have been fixed by now.

### Post by 23meg on 2007-02-26
> ok cool so are you suggesting i upgrade to dapper? because i seen few compiz install guides for dapper...If you must have Compiz, you'll have to upgrade to Dapper or Edgy. Edgy's Xorg has AIGLX built in, which is preferable to XGL in most cases, so chances are you'll have better performance and easier setup there. It depends on your hardware though, and there *may be* other drawbacks of switching to Edgy for your particular case.

I don't have particular experience with a Breezy to Dapper upgrade; all I can recommend is that you follow the official procedure. Check these out for a start:

[https://wiki.ubuntu.com/DapperUpgrades](https://wiki.ubuntu.com/DapperUpgrades)
