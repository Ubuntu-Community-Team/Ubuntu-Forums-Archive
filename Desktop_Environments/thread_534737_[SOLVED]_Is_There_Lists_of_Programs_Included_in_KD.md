### Post by miggols99 on 2007-08-25
kdebase has programs like Konqueror, Kate, Kwrite, Konsole, Kcontrol, kdelibs, qt and other "must have" programs. kdeutils has things like ark, kcalc etc. I would install things one by one, except for kdebase. The kdemultimedia package installs Noatun, which I think isn't very good. But it installs kmix, which is a very important thing to have. Hope you can work this out!

EDIT: I personally just install the base packages, then when I find out that for example k3b isn't installed, I just install it. :)
