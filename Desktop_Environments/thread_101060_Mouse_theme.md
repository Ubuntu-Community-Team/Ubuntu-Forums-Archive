### Post by Ampersand on 2005-12-09
You just need to copy the folders in the tar.gz file to /usr/X11R6/lib/X11/icons/ as root, so either run ```
sudo nautilus
``` or use the ```
sudo cp [source] [destination]
``` command. They should then appear in the cursor list in the mouse preferences.
