### Post by 3rdalbum on 2013-04-12
> **jamespoo said:**
> that sucks i only have a 4gb flash stick so i cant back up 121gb

If Linux Mint still uses the same installer as Ubuntu, you don't need to format your hard disk. Use the "Something Else" option, tell it what mount points each partition is (for instance, if you currently use /dev/sda1 for / and /dev/sda2 for /home, just set those as the mountpoints for those partitions), DO NOT CLICK THE 'FORMAT' CHECKBOX, and click Install. It'll preserve your data. Even if your /home is inside the / partition (i.e. one partition for Ubuntu), it will still preserve your data as long as you haven't ticked the 'Format' checkbox.

You might even get an option in the Linux Mint installer called "Upgrade", which will basically do the above without any input.

Just because your data will be preserved, does NOT mean that you can happily live without a backup of your data. Disks go bad, filesystems crash, it's a fact of life.

Linux Mint is based heavily on Ubuntu, so most of your skills will transfer over.

If you mainly just wanted a different desktop, it's a lot quicker and easier to install a different desktop on Ubuntu than it is to install a whole new distro.

If you're willing to keep an open mind, you should try Unity for a week; your first impression was probably "WTH, where's Gnome 2? How do I work this thing?". Unity is definitely winning a lot of converts these days, but of course it's very different to Gnome 2.
