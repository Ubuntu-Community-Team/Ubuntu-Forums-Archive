I needed this for lububntu but it seems to be useful for all.
You will create an entry in /usr/share/applications
the idea is to find simple enough entry (galculator for example)
and use it as a start 

```
cd /usr/share/applications
```

copy one simple example and change it using some text editor

icon:
* can be full path to icon image file or system-icon
