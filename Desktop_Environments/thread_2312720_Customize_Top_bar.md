### Post by grahammechanical on 2016-02-06
If it is Ubuntu + Unity then the answer is no. But if it is Xubuntu or Lubuntu or Kubuntu or Ubuntu Mate then the answer from me is may be but I cannot tell you how to do it.

Also, it depends on what kind of customisation. If we install Compiz Configuration Settings Manager then we can alter the transparency of the top panel. And then we have System Settings>Screen Displays>Scale for menus & title bars. Which will change the size of the text in the top panel and also the text in the top panel of an application window.
