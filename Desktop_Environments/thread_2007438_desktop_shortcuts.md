### Post by black veils on 2012-06-21
> 
can i move the launcher to the top rather than on the left side?
i dont think so, but if it is possible, try **MyUnity** from the software centre.


> where is the show desktop thingy in ubuntu?using **MyUnity**, go to the **Desktop** tab and activate **Show desktop icon**.

*or*  use what seems to be the default keyboard shortcut keys **Ctrl + Super (Windows key) + D** to minimise all windows.


> what is a sudo password?you have a user account, but when you need higher privileges (root) for a brief task, like installing a kernel update, you are required to enter your user account password, it is like a portal into the root account. this method provides good security, because it means avoiding using the computer with root privileges all the time.
