### Post by konqueror7 on 2009-03-05
download emerald from the repos by,
```
sudo apt-get install emerald
```
download the first link, explore it with emerald and then in the terminal,
```
emerald --replace
```

as for the other links, it depends on what desktop environment you use now, just download those links accorfing to you current desktop environment...

also you might want to install *fusion-icon*, just a try icon used to manage your basic desktop settings like choosing emerald themes or metacity(gtk) themes be used...

hope it helps...
