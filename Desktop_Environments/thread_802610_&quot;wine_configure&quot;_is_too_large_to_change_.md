well, i couldnt use wine or pokerstars at all because of the screen size.. so, i removed wine using "add/remove" (couldnt remove pokerstars via wine), then deleted .wine from my home folder, then reinstalled wine via "add/remove" and reinstalled pokerstars to be safe.

everything is working and fine now.. save the resize issue that i was attempting to fix in the first place..
