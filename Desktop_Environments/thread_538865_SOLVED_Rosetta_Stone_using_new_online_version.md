### Post by ddumanis on 2007-08-30
You can't use the current version of Rosetta Stone CD (it won't install using WINE, since there's an incompatibility with the Java-based installer they use).

But, you can use the new online version.

Here's how:

- Install WINE and cabextract, using Synaptic
- Install Internet Explorer (unfortunately necessary) using ies4linux (ies4linux.org)
- Log into your Rosetta Stone online account--you'll get a system auto-testing page. By clicking on the links it presents, install Java Virtual Machine and (if you want it) the Rosetta Stone speech software.
- Rock on

I'm able to use Rosetta Stone online perfectly this way--sound, animation, and all. (I haven't tried the speech recognition part--not really interested.)

Best part: because the new online version doesn't use shockwave, there's *no need for Windows.* (The old version required dual booting or running Windows some other way.)

Good luck!
