### Post by mtbsoft on 2009-02-16
Do you really need VMWare or would any VM product do?  For VMWare, workstation is the more powerful solution but, once you have the VM created, Player would probably suffice - Workstation is a paid product but Player is free.  Alternatively, there are a number of non-proprietory solutions available in Linux - VirtualBox being one, can't think of others.

I use VMWare Workstation myself but that is  because my images need to be runable under both Linux and Windows.  For VMWare, if you don't want to pay for Workstation you won't be able to create any VMs using Player.  However, if you still have access to a Windows box, you can download MS VirtualPC, create a vanilla image and then use VM Player to use that, even under Linux - crazy but true!
