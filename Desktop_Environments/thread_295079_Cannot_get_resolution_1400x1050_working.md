### Post by matthew on 2006-11-07
You should be able to get this working by reconfiguring your xserver. This is easier than it sounds. There is a program with menus to help you do this.

Open a terminal (Applications->Accessories->Terminal).

Type```
sudo dpkg-reconfigure xserver-xorg
```and follow the menus. 

One of the first things you can choose is the video driver you want to use. If there's not one that looks exactly the same as the GMA950 (it might say i950) integrated video card you have you can try again with the i810 you are currently using. I don't have the same hardware so I can't comment specifically on that but my gut feeling is that either should work--obviously one designed specifically for your hardware would be best, though.

If you don't know what the options mean just hit "ok" to choose the default. 

When you get to the option where you can choose from several screen resolutions, scroll down and use the spacebar to highlight (like this: 
[*]) the options you would like available.

Once you are done with the process, reboot and you should have 1400x1050 available...it may even boot up in that resolution for you if it is the largest one you choose.

If it doesn't work you can run the same thing again so write the command down before you reboot. If you only boot to a command line this will help you recover as well.
