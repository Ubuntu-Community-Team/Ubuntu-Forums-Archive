### Post by kellemes on 2011-03-31
[Cairo Composite Manager]("http://cairo-compmgr.tuxfamily.org/") has comparable features..
I use it sometimes with Fluxbox for some eyecandy with opacity etc..
Cannot tell you how to get it for Ubuntu since I'm using Arch myself.

By the way, Compiz is a wm and can only replace another wm, so it won't work with Fluxbox or enlightenment.

### Post by Seidojohn on 2011-03-31
Actually, you can use Compiz with Enlightenment 17. Check out [eLive]("http://www.elivecd.org/Download/Stable") ([http://www.elivecd.org/Download/Stable](http://www.elivecd.org/Download/Stable)). I've been using it for a couple of months, and am really liking it (after a bit of customization). It's Debian-based, so will be somewhat familiar, and you may have to tinker with the Ecomorph settings a little to get it to be stable on your machine, but it's worth a try. 

Happy hacking! :)
