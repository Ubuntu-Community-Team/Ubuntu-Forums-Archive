### Post by Krytarik on 2011-03-28
If you want to install just KDE alongside Gnome to regular Ubuntu, you have the choice between the metapackages "kde" and "kde-base":
[https://help.ubuntu.com/community/MetaPackages](https://help.ubuntu.com/community/MetaPackages)

If you want to remove the Gnome parts of regular Ubuntu and install the Kubuntu parts instead, run the first stated set of commands:
[http://www.psychocats.net/ubuntu/purekde](http://www.psychocats.net/ubuntu/purekde)
