### Post by Jucato on 2006-06-18
A display manager is basically a login or GUI session manager: it's responsible for starting a KDE/GNOME session in GUI (you can start a session in the command line, too.) So basically what you see when you login to Kubuntu is KDM. 

A window manager is in charge of how windows are displayed and how windows behave. It also sometimes handles virtual desktops/workspaces. So anything related to the physical appearance/behavior of windows is handled by the window manager.
