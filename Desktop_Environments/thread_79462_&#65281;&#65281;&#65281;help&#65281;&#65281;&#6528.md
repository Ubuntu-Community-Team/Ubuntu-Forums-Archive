Doing a wild guess here...
What if you try removing your xorg.conf file and setting the old one instead?
```
sudo mv /etc/X11/xorg.conf /etc/X11/xorg.conf-older
```
And then
```

```
This will load your old config file for Xorg... Maybe it will work?
