### Post by Dozy Van on 2013-07-03
you can test them out with out re-installing. You can install differnt envirments and just select them at the log in screen. I have Unity, Cinnamon and Gnome2 on this PC.

some you may like:

Lubuntu
```

sudo apt-get install lubuntu-desktop
```

Cinnamon
```

sudo add-apt-repository ppa:gwendal-lebihan-dev/cinnamon-stable
sudo apt-get update
sudo apt-get install cinnamon

```


Gnome2
```


sudo apt-get install gnome-panel

```


KDE
```

sudo apt-get install kubuntu-desktop

```

### Post by Peripheral Visionary on 2013-07-03
Xubuntu and Lubuntu are both simple and fast.  Unless you have a really old computer (say less than 512 RAM), I recommend [Xubuntu]("http://xubuntu.org").  It has more features and is much more easily configurable in my opinion.  I like simplicity and the Xfce desktop offers all it's beauty and versatility with a sweet, simple interface.  And on my old computer (a 9-year-old Dell rescued from the landfill, has a 2gHz processor and 512 of RAM), I found *no noticeable difference in speed or performance* between the ultralight Lubuntu and the more full-featured Xubuntu.
