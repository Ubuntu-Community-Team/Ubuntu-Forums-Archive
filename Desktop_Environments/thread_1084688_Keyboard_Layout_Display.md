### Post by InspiredIndividual on 2009-03-04
You can just check what character the keys you hit produce in any text field. To display the complete keyboard layout at once, right-click the panel, choose 'Add to panel' and add the 'Keyboard Indicator'. This will show what layout you're currently using on the panel. Right click the indicator and choose 'Show Current Layout' for an overview of the complete layout at once. It will also show you what key you're currently tapping.
