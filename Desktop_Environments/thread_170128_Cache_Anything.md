### Post by tsrjzq on 2006-05-04
I think, if you just want to speed up your firefox, you can ban the ipv6. then, type "about:config" in your address bar, and set "network.http.pipelining" to "true", set "network.http.proxy.pipelining" to "true", set "network.http.pipelining.maxrequests" to "8". then, add a new string named "nglayout.initialpaint.delay" and set it to "0". that's ok.
besides, you can install the extension named FASTERFOX, it can accelerate the firefox, too.
