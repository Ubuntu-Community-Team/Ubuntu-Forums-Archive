### Post by psychicdragon on 2006-03-19
The session manager remembers what applications you had running, and runs them the next time you start a new session. 

If you don't want any programs to run when you start a session, close any running programs and logout (selecting the "Save current session" option).

### Post by deavik on 2006-03-19
Actually I think I've got a handle on it now. When you log in, you can choose whether to run the "Last" session or the default session (+ a bunch of other options). The "Last" session == the last session you saved.
