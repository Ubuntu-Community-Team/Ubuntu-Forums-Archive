### Post by pinguy on 2008-01-20
I use [ted]("http://www.ted.nu/") its a great little program, you do need java installed for it to work.

There isn't a .deb or install file you get a zip file with the program in it, all you need to do is double click on ted.jar to run it.

What I did was to place the unzipped folder in the root and added it to the menu list.

System > Preferences > Main Menu

Then I added the link to "Internet" New Item.

 Name **TED: Torrent Episode Downloader**
 Command  **/tedv091/ted.jar**

You can use this as the icon

 [IMG]http://www.ted.nu/img/header_logo.png[/IMG]

I also have it so it starts with the PC, all you need to do is add it to your sessions
System > Preferences > Sessions
