### Post by gnuskool on 2008-05-01
- Open thunar (or any file manager)
-click on your home folder
-click view , show hidden files
-navigate to the .themes folder
-download your theme (if not already done)
-extract or unpack it 
-drag n drop it to the .themes folder in thunar

-xD

(Tip of the day - in linux the . before a folder name means it is hidden)
