### Post by Krytarik on 2011-01-27
Yeah, you can adjust such things by using the "Window Rules" plugin of Compiz, "System -> Preferences -> CompizConfig Settings Manager -> Window Management", and the focus settings in its "General Options".

Those window matching guide may be handy for the first mentioned one:
[http://wiki.compiz.org/WindowMatching](http://wiki.compiz.org/WindowMatching)
