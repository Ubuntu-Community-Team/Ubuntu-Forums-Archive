### Post by istoyanov on 2005-03-24
Xfce is just a very clean and lightweight desktop/window manager with some additional apps specially designed for it. I'm enjoying its beautyfully simple environment for a month now, and every Gnome application I use works flawlessly with it.

I can strongly recommend it to everyone that is looking for a sleek alternative to the Gnome desktop!

Greets!

P.S. The *easyest* way to get it going, is to use the installers from os-cillation [http://www.os-cillation.com/article.php?sid=42](http://www.os-cillation.com/article.php?sid=42) and to do
```
apt-get install libgtk2.0-dev libice-dev libsm-dev libdbh1.0-dev a2ps libxml2-dev libxpm-dev libvte-dev libncurses5-dev dbus-1-dev dbus-glib-1-dev libxml-parser-perl
``` before launching the installers.
