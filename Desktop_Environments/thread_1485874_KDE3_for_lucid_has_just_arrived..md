### Post by kio_http on 2010-05-17
KDE3 for lucid has just been released!

To install add this to your repository sources
```
http://ppa.launchpad.net/kde3-maintainers/ppa/ubuntu
```

Then
In terminal
```
sudo apt-get update
```
then
```
sudo apt-get install kubuntu-desktop-kde3
```

Enjoy! 
Note: Other desktop environments currently installed will be unaffected by this, including KDE4 series.

### Post by kio_http on 2010-05-18
Known issue: It may try to remove sudo, kdesu, gksu etc will stay.

Just install sudo again after installation and everything will be fine

```
sudo apt-get install sudo
```
