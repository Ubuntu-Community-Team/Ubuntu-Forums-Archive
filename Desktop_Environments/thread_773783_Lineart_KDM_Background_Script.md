### Post by FrozenFire on 2008-04-29
**[EDIT] Added Hardy Fix**
I tried to change my kdm background in the control panel, did not work.
Ended up hacking myself through all the kubuntu theme files.
So i made a script that does it automagically, to save others from this.
Download: [http://www.nathan7.eu/downloads/kdmlineart](http://www.nathan7.eu/downloads/kdmlineart)
Automatic Install ```
sudo true && wget http://www.nathan7.eu/downloads/kdmlineart -O - | sudo sh && sudo reboot
```
(PS: Requires KDM restart to show up.(Ctrl+Alt+Backspace (reboot recommended)) Set your desktop background to Lineart and your splash screen to MoodinKDE after using.)
