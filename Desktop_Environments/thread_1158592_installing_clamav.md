### Post by hansdown on 2009-05-13
Hi Pedroparamo.

Click system> administration> synaptic package manager.

Search for clamav, click on the box, click mark for installation,

click mark all upgrades, click apply, click apply again.

This will install it.

To get a gui, search for clamtk, and install it also.

UbuntuNerd has a great tutorial.
