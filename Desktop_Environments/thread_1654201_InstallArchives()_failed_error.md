### Post by germanix on 2010-12-28
Go to "administration" - "Software sources" then go to the tab "other Software" and see if you have "archives" selected. If so un select it and try the update again.

### Post by germanix on 2010-12-28
Should normally be there. Then go to the main menu, you should find this under "system" - "preferences", select "main menu", once there go to "administration" (right side of the window right at the bottom. You may have to scroll down).  Here you will find "software sources" under the item list to the right.  Activate it so it shows up under the normal administration menu. Then go back to "administration" and do what I suggested above.
