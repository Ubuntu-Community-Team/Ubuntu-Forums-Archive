### Post by Xian on 2006-03-06
[QUOTE=fjcouto]Does anyone else have some kind of tip ?[/QUOTE]

If you are talking about the entries in the System > Prefs and System > Admin menus then you can modify those in the /usr/share/applications folder. Just remove or add any desktop files that you want to revise. You will need to be certain that if you add them you get the proper menu path issued in the 'Categories' line.
