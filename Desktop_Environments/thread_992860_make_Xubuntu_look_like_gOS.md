### Post by Izek on 2008-11-25
First, [get some desklets]("http://adesklets.sourceforge.net/desklets.html"). I've never ran adesklets before in my life, but [according to their site](http://adesklets.sourceforge.net/doc/en/Using-adesklets.html#Using-adesklets), you'll have to use the command line for all this.

There is an alternative to adesklets, as I'm more of a gnome man, gdesklets has a menu item in the menus. But you'll want to uninstall adesklets to install gdesklets.
