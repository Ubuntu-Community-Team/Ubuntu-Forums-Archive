### Post by rmalta on 2006-09-20
Probably not actual any more, but I needed it also today ;-)

To post into a newsgroup you have to

1. Reply to an existing message.
   This way you will see a "Post to" field with the
   newsgroup you are using.

2. To create a new thread:
   - Create a new "Mail-message" (just like usual)
   - In the compose window goto view and select "Post-To Field"

Now you should see a "Post to" additional field in the compose window. There you can place the name of the newsgroup you want to add the new thread.
