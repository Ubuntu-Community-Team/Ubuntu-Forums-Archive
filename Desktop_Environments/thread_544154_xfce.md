I'm with you on this: XFCE rocks. Without a doubt it is the best windows manager out there, bar none. I got hooked on it when I did UNIX system admin work for a company a while back. I've been using it ever since.

Long story short: XFCE keep up the good work!

### Post by Warren Watts on 2007-09-08
My  first exposure to Xfce was with an install of Xubuntu, and initially I wasn't particularly impressed.  In fact, I switched back to Gnome.  But I recently started playing around with Wolvix (which comes set up with Xfce and Fluxbox), and my opinion of Xfce has changed.  

The Wolvix folks have done an excellent job of implementing Xfce on the desktop.  It's much more intuitive and easy to navigate than Xubuntu's default implementation in my opinion.  In fact I have really grown to like it.  It's much faster than Gnome, and a lot more flexible.  There's even a panel application that will allow you to run Gnome applets in an Xfce panel! ([xfapplet]("http://goodies.xfce.org/projects/panel-plugins/xfce4-xfapplet-plugin"))
