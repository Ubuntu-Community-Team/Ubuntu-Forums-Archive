### Post by papibe on 2012-07-16
Hi userqq.

That functionality is called HUD (head up display).

The easiest way is to disable it is by blanking its shortcut:
```
Open System Settings > Keyboard > Shortcuts > Launchers > Key to show HUD 
```

Regards.

### Post by beejee on 2012-07-17
You have a switch for the hud in the configuration program 'unsettings'

You can install it with this line:
```
sudo add-apt-repository ppa:diesch/testing && sudo apt-get update && sudo apt-get install unsettings
```

You can start Unsettings from System Settings or from the dash, choose 'windows' in the list on the left and flip the switch for 'Show HUD'.
Now the Hud won't pop up when you press alt. :popcorn: Happy gaming!
