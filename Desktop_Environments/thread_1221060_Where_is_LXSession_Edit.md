### Post by XubuRoxMySox on 2009-07-29
If you mean the Login Window (where you change change login sessions, make one a default, enable remote login, etc), then:

Click on the bird (lower left corner where the Windblows "Start" button would be), then choose Preferences and select Login Window.

If you installed LXDE from the Ubuntu repositories, you should have LXsession Lite rather than LXsession. The Lite is much preferred. 


More help for LXDE is available [here]("http://forum.lxde.org/index.php").
