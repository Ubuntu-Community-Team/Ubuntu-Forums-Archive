### Post by kerry_s on 2006-09-02
Try hitting the reload button in nautilus,sometimes it doesn't pick up new items. If that don't work try dragging the image into the background window. Also try right clicking the image and set as wallpaper or open it with the image viewer(click on it) and choose set as background in the menu.

### Post by Crooksey on 2006-09-02

Note. Remove old images with the same file name first.
