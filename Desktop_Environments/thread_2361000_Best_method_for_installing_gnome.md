### Post by mcoyle on 2017-05-18
After running straight Ubuntu for years, I did a fresh install of Ubuntu-Gnome. 

First thoughts: It's all Gnome and no Unity. It feels like a much more cohesive desktop environment than Unity. 

Previously, I had installed Gnome on top of Ubuntu and there were situations where some apps had Unity windows and some had Gnome.  

Be sure to go to [https://extensions.gnome.org/](https://extensions.gnome.org/) and enjoy all the goodies.
