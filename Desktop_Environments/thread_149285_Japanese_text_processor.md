### Post by dmizer on 2006-03-23
i'm not sure exactly what you mean by word processor.  you can select system > administration > language selector to add full support for japanese.

if you want to be able to type in japanese, use this wiki guide:
[https://wiki.ubuntu.com/JapaneseInputHowto?highlight=%28japanese%29%7C%28support%29](https://wiki.ubuntu.com/JapaneseInputHowto?highlight=%28japanese%29%7C%28support%29) <-- this will allow you to type in japanese to any word processor on your computer.
