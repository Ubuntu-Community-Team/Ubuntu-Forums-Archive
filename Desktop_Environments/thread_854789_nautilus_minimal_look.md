### Post by thegreatbitbucket on 2008-07-09
The way Ubuntu implements Nautilus is different than Fedora, etc. There is really no difference. By default, Ubuntu uses the browser mode instead of the spacial mode. The spacial mode is the default Nautilus behavior. Just do what the other guys told you, and you will be fine. Do the opposite for Fedora if you want the browser mode instead.
