Just download the current version (8.04).  Use the desktop cd to install.

Start the cd and when you are at the desktop, click the install button.  Answer the questions.  If at the partitioning stage it does not offer you the option of resizing your windows partition and using the freed space, stop and reboot into windows.  Defragment your drive and start the whole procedure again.

By default, if windows is already on your computer, Ubuntu should install itself as a dual boot.  You don't need to do anythig special, other than make sure you don't pick the "use entire disk" option.
