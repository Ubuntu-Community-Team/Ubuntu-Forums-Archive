### Post by dante.regis on 2007-04-30
on the System / Preferences / Theme dialog, there's an option to Customize the current Theme. (It could be Edit too. I don't use the english version of ubuntu). There, you can find a tab saying ICONS. You can change there your icon theme. I don't know if you are already doing this... if not, give it a try.
