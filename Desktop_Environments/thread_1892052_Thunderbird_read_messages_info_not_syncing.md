### Post by Krytarik on 2011-12-07

Nope, not really; it works here (with Thunderbird 8.0 on Lucid 10.04, and Firefox 8.0 for running the web mailer).

Only one thing: Make sure the GMail account is set up as IMAP in Thunderbird; although if you had set it up as POP3, the concerning mails would still be "unread" after re-opening Thunderbird.

Regards, and good luck!

### Post by thaelim on 2011-12-07
As Krytarik said, ensure you have configured Thunderbird to connect to Gmail using IMAP, and also that you have set Thunderbird to periodically check for messages (that will update the read status as well).

Having said that, getting Thunderbird working 100% with Gmail takes a bit of tweaking, as a lot of the 'helpful' things Gmail does break the rules of IMAP.
