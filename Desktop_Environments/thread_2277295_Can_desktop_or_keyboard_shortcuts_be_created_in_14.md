### Post by Copper Bezel on 2015-05-07
Click the gear at the upper right of the screen and select System Settings in the menu. Once System settings comes up with the grid of icons, click the Keyboard icon in the Hardware section. Then click the Shortcuts tab. 

Notice that anything that's already assigned to a Compiz / Unity shortcut won't work in here - like, say, flag key + D, which spreads the windows on the current desktop. You can change those shortcuts, too, elsewhere, but it's more complicated.
