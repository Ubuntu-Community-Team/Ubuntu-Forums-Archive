### Post by guitard00d on 2005-08-16
Or, just type "sudo -s" and enter your password, and pretend you're running any other *nix system logged in as root.  :-)

Mac OS X is the same way. The lack of having a formal root password makes no difference when you can use "sudo -s" to basically log in as root. You can limit who has access to sudo, so it's all the same having a root password and selectively giving it to certain users.
