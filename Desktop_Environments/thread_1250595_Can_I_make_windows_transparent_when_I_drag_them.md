### Post by tuxxy on 2009-08-26
Indeed you can, once you have Compiz (Desktop Effects) running then launch Compiz config settings manager located in **system > prefs > compiz**

If its not there just install it, either through synaptic or run this command in terminal;

```
sudo apt-get install compizconfig-settings-manager
```

Once you have Compiz config settings manager loaded navigate to the Move Window plug-in which is located in the Window Management section.  Here you can set the transparency level of windows while being dragged.
