### Post by RobCampbell on 2009-10-06
Ok! I have got rid of it! Here's how:

System Settings -> Window Behavior -> (Window Behavior -> Focus) -> uncheck "Popup desktop name on desktop switch".

Mine was already unchecked so I checked and unchecked it and now it's gone:
[http://forum.kde.org/viewtopic.php?f=111&t=82724&p=132294#p132294](http://forum.kde.org/viewtopic.php?f=111&t=82724&p=132294#p132294)
