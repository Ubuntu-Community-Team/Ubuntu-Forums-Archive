### Post by taurus on 2009-05-19
The problem is with the graphic driver that you've installed.  The command above would reset your /etc/X11/xorg.conf back to the default, using a generic driver.

```
sudo dpkg-reconfigure xserver-xorg
startx
```

Or you can boot your machine with Ubuntu LiveCD.  Mount the partition where / is and copy /etc/X11/xorg.conf from the LiveCD to your harddrive.
