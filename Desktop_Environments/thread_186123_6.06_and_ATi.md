### Post by johnniecarcinogen on 2006-06-01
you have to 'sudo nano /etc/apt/sources.list' and uncomment the 'restricted' repo and:
```
sudo apt-get update
sudo apt-get install linux-restricted-modules-$(uname -r)
sudo apt-get install xorg-driver-fglrx
sudo aticonfig --initial
sudo aticonfig --overlay-type=Xv
sudo reboot
```
