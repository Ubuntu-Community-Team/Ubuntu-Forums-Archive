### Post by reahjw6 on 2009-02-27
Spin the cube by pressing mouse scroll wheel and drag.
To set the gears open ccsm>effects>mark>cubegears
Under desktop cube setting go to last tab transparent cube and change the opacity values as desired.
This will allow you to see the gears.
Under desktop make sure you have marked rotate cube as well.
close out as above Spin the cube by pressing mouse scroll wheel and drag.
