### Post by mcduck on 2015-10-26
Assuming you are using default Unity desktop, the window manager (Compiz) already has support for that built-in. Just use Compizconfig Settings Manager, and the "Place Windows" and "Window Rules" -plugins  allow you to define all kinds of default positions, sizes, and other settings for various applications and window types.
