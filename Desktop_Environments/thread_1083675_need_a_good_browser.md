### Post by thtrgremlin on 2009-03-01
lol  +1 glotz

firefox shouldn't, and doesn't typically, just crash on a regular basis. Sadly, my recent experience with exactly that issue was on a computer with bad RAM. I'd recommend running memtest at startup. let it run 3 cycles or so, as it doesn't always find things the first run. 

Further, going to assume based on the hardware specs, you have had the computer for some time. When is the last time you cleaned it out? heat sinks, fans and such.

I have seen a lot of computers start to exhibit strange behavior / random crashes of applications where simply cleaning it fixed everything. firefox + flash is pretty intense, so if there is a problem, they are good at "finding" them. Did some tech for a render farm quite some time ago, and you wouldn't believe how often problems were solved with some dust-off.
