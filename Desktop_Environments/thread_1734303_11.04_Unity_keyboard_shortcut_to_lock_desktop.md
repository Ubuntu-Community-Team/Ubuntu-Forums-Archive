### Post by Krytarik on 2011-05-11
> **kah00na said:**
> 
I'm interested in how to lock the screen... either with a keyboard shortcut or shortcut button.  Anyone know how to do either of those?
Ctrl+Alt+L should do it, like mentioned in the OP:
[http://askubuntu.com/questions/28086/unity-keyboard-mouse-shortcuts](http://askubuntu.com/questions/28086/unity-keyboard-mouse-shortcuts)

You can change the key combination in "Keyboard Shortcuts", except for combos that involve a Super key, apparently.

You can also create a launcher for the actual command:
```
gnome-screensaver-command --lock
```Greetings.
