### Post by T3chno on 2009-10-31
Recently installed Ubuntu 9.10 and I love it so far. My only problem with it so far is that the function of the mouse wheel from previous versions, the one where it allows you to switch left and right between workspaces if the mouse is focused on the desktop, has been missing on 9.10.

I don't think my mouse is the problem, since the mouse wheel works with scrolling up and down on web pages. It's also never been a problem in past versions.

Any help would be nice.

---

### Post by shafin on 2009-10-31
install compiz config settings manager from synaptic.
system>preferences>compiz config settings manager>viewport switcher> Desktop based viewport switching> set move next to button 5 and move prev to button 4.
