### Post by linuxaddix on 2011-11-07
to get the new msge desktop for gnome shell enter this in terminal:
you can use this if you have gnome shell installed to get the new mgse

sudo add-apt-repository ppa:webupd8team/gnome3
sudo apt-get update
sudo apt-get install mgse-bottompanel mgse-menu mgse-windowlist

(be advised that advanced settings must be installed also.get it from ubuntu software.this is for gnome shell)
after installation logout then log back in.go to advanced settings/http://www.google.com/shell extensions and turn on menu extension and bottom panel extension.now log out and log in for the adjustments to work.now with gnome shell you will have activities at top left and menu at bottom left.everything works perfectly.they said it might have bugs but i never found anything wrong yet.now you have msge(mint shell gnome extension)
log out is no longer needed
hold alt-f2  
type in      r
press enter  
this restarts gnome shell
