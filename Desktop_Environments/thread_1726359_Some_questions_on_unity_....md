### Post by mcduck on 2011-04-11
a) Unity is basically a Compiz plugin, so the window manager is Compiz and desktop environment is Gnome, even when you are using Unity.

b) Every single Compiz plugin that allows you to switch between application windows has the option to include windows from all workspaces. You might want to check your Compiz settings again... ;)

c) That's a Gnome thing, and not related to Unity in any way. So no, it's probably not going to change and if it does, that's because Gnome changes it and not because of using Unity.
