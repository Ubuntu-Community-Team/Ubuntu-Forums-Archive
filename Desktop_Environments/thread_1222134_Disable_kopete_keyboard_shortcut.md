### Post by grkuntzmd on 2009-07-24
I normally use gnome as my desktop manager, but I am using Kopete for IM. I was trying to save all files from Eclipse using the standard Shift-Ctrl-S, but the Kopete buddy list keeps popping up. Obviously, it has remapped this as a shortcut.

Is there any way to disable this shortcut? Is there any place in gnome/Ubuntu where all shortcuts are kept? I've already tried System->Preferences->Keyboard Shortcuts, but this one is not listed.

Thanks, Ralph

---

### Post by Zorael on 2009-07-24
How about Kopete's options? Settings -> Configure Global Shortcuts -> Show/Hide Contact List.

edit: To access the list of all global KDE shortcuts, you'd have to install systemsettings, I believe.
