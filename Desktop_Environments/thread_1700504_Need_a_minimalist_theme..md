the top panel bar can be made transparent as follows:

right click on the panel and go to properties, select background, choose the color to suit ur wall paper shade, adjust opacity / transparancy. enjoy...

try changing window border to elementary. before doing that, edit the metacity file in the elementary folder created in ~/.themes folder.

edit the metacity-1.xml file using gedit or ur fav editor.
top part read carefully. distance-name part, make all values to 0. save and restart ur theme. now u should have a minimalistic elementary window border with grey, matching with the controls of gray revenge gtk theme controls. select icons to elegant awoken. u should enjoy this.
