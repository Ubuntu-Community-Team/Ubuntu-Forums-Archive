### Post by LeoSolaris on 2008-03-29
It's in the System Preferences. Called Sessions. You will have to know the command for awn to add it to the list.

It's avant-window-navigator

Put what ever you want in the name and the comment sections, so long as you have ```
avant-window-navigator
``` in the command section it will work.
