### Post by landotter on 2005-02-24
[QUOTE=touronster]hello all, 
I a new user of Ubuntu linux, I have installed it and now, I want to install KDE over Gnome. [/QUOTE]

If you have universe enabled in Synaptic (open up "repositories" and check the box, then refresh listings), a simple search for KDE will give you your results. There should be a metapackage "kde" that will install all you need.

I've got it installed for fun, and run it occasionally for the hell of it...but honestly, I think that if you're both French speaking and a KDE user--you might be happier with a tweaked version of KDE such as Mandrake Linux's--I think KDE on top of Ubuntu w/o any changes is lousy compared to Gnome. Not that KDE is lousy--but it doesn't do the nice stuff out of the box like Gnome does--automount cameras and burn data CDs, those things would probably already be set up with a KDE centric distro.

BTW, you can install and use KDE apps under Gnome with no problems and run both KDE and Gnome on the same box--I highly recommend this if you've got the HDD room. :D
