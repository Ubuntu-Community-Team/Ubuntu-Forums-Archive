### Post by false truths on 2011-04-19
No, the Gnome 3 repositories do not work for Ubuntu 10.10. The only way to install the final version in Ubuntu 10.10 is to compile it from source using git. There is a guide for that here: [http://www.webupd8.org/2010/10/install-gnome-shell-from-git-in-ubuntu.html](http://www.webupd8.org/2010/10/install-gnome-shell-from-git-in-ubuntu.html)

This takes a VERY long time, and doesn't show up as a proper desktop environment, but it works very well.
