### Post by infol on 2007-02-14
You currently have 

```
Section "Device"
Identifier "ATI Technologies, Inc. RV370 5B62 [Radeon X600 (PCIE)]"
Driver "vesa"
BusID "PCI:1:0:0"
EndSection

Section "Device"
Identifier "aticonfig-Device[0]"
Driver "fglrx"
Option "VideoOverlay" "on"
Option "OpenGLOverlay" "off"
EndSection
```
in your xorg.conf

try changing it to 

```
Section "Device"
Identifier "aticonfig-Device[0]"
Driver "fglrx"
BusID "PCI:1:0:0"
Option "VideoOverlay" "on"
Option "OpenGLOverlay" "off"
EndSection

```

hope this helps..  ;)
