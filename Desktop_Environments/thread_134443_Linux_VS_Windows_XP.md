### Post by stuporglue on 2006-02-21
If your graphics card has decent Linux support, then YES, Linux should be faster. 
[B]
Edit:[/B] In light of aysiu's comment, I'll ammend mine. A typical Ubuntu install, in my experience, is usually faster than a typical Windows install, especially if Windows has been installed for a while. I think this is probably due to registry bloat in Windows and lots of unused running processes. If you're meticulous in keeping your Windows install clean, then the default Ubuntu install may not be faster than your current Windows install. 

The Ubuntu Breezy (current stable) installer has a Windows resizer built into the installer, and will set up a dual boot system under normal conditions.

In Windows, backup your files (resizing is ocasionally dangerous), defrag the disk, then pop in the Ubuntu Installer CD and reboot. Follow the prompts, and when you're done you should have a dual boot system. 

Good luck!
