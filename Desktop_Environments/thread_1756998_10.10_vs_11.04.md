### Post by sikander3786 on 2011-05-13
Go to the Terminal and run this command.

```
sudo apt-get update
```

Again, fire up update manager. Hopefully, the error should be gone.

For finding out if your system is compatible with Unity or not, you need to post your specs or the output of this command.

```
/usr/lib/nux/unity_support_test -p
```
