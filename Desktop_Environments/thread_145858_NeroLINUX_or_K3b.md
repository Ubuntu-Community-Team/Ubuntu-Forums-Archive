### Post by Lux Perpetua on 2006-03-17
I've been quite impressed with K3b. Yes, it is easy to install, like pretty much everything in Ubuntu. I use K3b, which is a KDE application, on my Gnome desktop, because it simply destroys every Gnome CD writing program I have used in terms of functionality and usability. I've only encountered one bug (verifying written data yields false negatives), which is supposed to be fixed in the development version. 

The great thing is that you can install it, and if you don't like it for whatever reason, uninstall it, and you'll be none the worse for it.
