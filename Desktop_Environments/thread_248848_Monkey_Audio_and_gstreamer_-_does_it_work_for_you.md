Monkey Audio is a lossless very efficient audio codec with extension file.ape. It is not open source so support in linux is not very good. There, however exists some alterntives. One being codec and plugin for xmms - they can be installed from this repository [http://http://morgoth.free.fr/ubports/]("http://http://morgoth.free.fr/ubports/"), packages libmac2 and xmms-mac. Other options seems to be gstreamer plugin gstreamer0.8-monkeysaudio.deb that can be downloaded here [http://http://rarewares.soniccompression.com/debian/packages/unstable/]("http://http://rarewares.soniccompression.com/debian/packages/unstable/"). However, it seems not to be working, at least for me. I do not know if Ubuntu using mainly gstreamer 0.10 now is the problem, but it just does not work. I am curious if you got .ape playback working.
(converting .apes works great with the codec and perl audio converter)
