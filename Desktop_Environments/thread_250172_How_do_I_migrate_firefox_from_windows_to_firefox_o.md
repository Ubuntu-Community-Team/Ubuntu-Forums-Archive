### Post by Leonivek on 2006-09-03
The best way is to get the FEBE Firefox extension and the CLEO extension.  You can find both here:  [http://www.customsoftwareconsult.com/extensions/](http://www.customsoftwareconsult.com/extensions/)

They allow you to backup all of your extensions and themes and makes one extension that contains all of your extensions.  So then you load Ubuntu, open Firefox, drag the one extension that contains all of your extensions to Firefox to install them all.  I haven't tried with themes cause I don't use them, but it should be the same principle.

I use these extensions all the time when using different computers.
