### Post by ijsje on 2005-04-16
I don't know of a global setting in Gnome for the number of lines to scroll using the mousewheel. However, you can set this in some of the applications that use the mousewheel. In firefox for example, you can type about:config in the location bar, and change the value of "mousewheel.withnokey.numlines".
