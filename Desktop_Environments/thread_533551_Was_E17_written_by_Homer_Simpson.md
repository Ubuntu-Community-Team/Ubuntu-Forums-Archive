### Post by kvonb on 2007-08-24
I thought I'd try E17 on my laptop seeing as it ran quite fast on my desktop.

I was hoping it had improved a little since the last time I tried it a few months ago, but no such luck!

The first glaring omission is the lack of a tray app, this cripples the whole thing, it's pointless if you can't run things in the background and easily switch to them.

The panel clock looks trendy, but is completely bloody useless.  You can't tell what time it is because it is way too small and obscure, and there is no date display, not even on mouse over.

You can't configure most of the things that sit on the panel.

The window list thing doesn't seem to do anything, sometimes things show up on it, but mainly it remains blank!

They have now got a basic file manager working, but again you can't configure the damned thing, and each folder is opened in a new window and placed at amusingly random locations around the desktop.  This was acceptable behavior back when Windows 95 came out, it was fascinating to see all those windows all over the desktop, but the fascination wore off after ten seconds!!!!

And the icing on the cake is the damned key editor.  That must have been written by retarded aliens for the exclusive use of people with an IQ rating somewhere beneath that of the fictional cartoon character Homer Simpson!

I mean try it and see.  It makes absolutely no sense to me, you can't simply assign a key to a function, oh no, you have to first delete the key combination (if it exists already), then quit the editor so that no key/app is selected, go back in and select the function that you want to modify, then click on "add key", press the key combination, save and exit again!

Then repeat the whole sequence if you want to change another key combination, otherwise  the last key you had selected will be assigned to a seemingly random app ](*,).

Insane!
