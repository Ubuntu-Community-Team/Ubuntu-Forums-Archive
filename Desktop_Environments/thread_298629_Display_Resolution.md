### Post by abbeychase on 2006-11-13
ok, well first, unless you have a wacom tablet?  I would delete all those entries regarding it.  It's safe to do so.  Second, you need meta modes to set custom resolutions.  here is my xorg.conf:

```

Section "ServerLayout"
    Identifier     "Layout0"
    Screen      0  "Screen0" 0 0
    InputDevice    "Keyboard0" "CoreKeyboard"
    InputDevice    "Mouse0" "CorePointer"
    Option         "AIGLX" "true"
EndSection

Section "Files"
    ModulePath      "/usr/lib/xorg/modules/extensions/nvidia"
    ModulePath      "/usr/lib/xorg/modules"
EndSection

Section "Module"
    Load           "freetype"
        # Load "xtt"
    Load           "extmod"
    Load           "glx"
    Load           "dbe"
    Load           "record"
    Load           "xtrap"
    Load           "type1"
#       Load  "speedo"
EndSection

Section "InputDevice"
    Identifier     "Mouse0"
 Driver         "mouse"
    Option         "Protocol" "ImPS/2"
    Option         "Device" "/dev/psaux"
    Option         "ZAxisMapping" "4 5"
EndSection

Section "InputDevice"
    Identifier     "Keyboard0"
    Driver         "kbd"
    Option         "XkbModel" "pc101"
    Option         "XkbLayout" "us"
EndSection

Section "Monitor"

 ### Comment all HorizSync and VertSync values to use DDC:
    Identifier     "Monitor0"
    HorizSync       31.5 - 92.0
    VertRefresh     50.0 - 100.0
EndSection

Section "Device"
    Identifier     "Card0"
    Driver         "nvidia"
EndSection

Section "Screen"
    Identifier     "Screen0"
    Device         "Card0"
    Monitor        "Monitor0"
    DefaultDepth    24
    Option         "RenderAccel" "True"
    Option         "AllowGLXWithcomposite" "true"
    Option         "AddARGBGLXVisuals" "True"


Option         "MetaModes" "1680x1050_60"


    Option         "XAANoOffscreenPixmaps" "true"
    SubSection     "Display"
        Viewport    0 0
        Depth       24
        Modes      "1680x1050_60" "1280x1024" "1280x960" "1152x864"
    EndSubSection
EndSection

Section "Extensions"
    Option         "Composite" "Enable"
EndSection


```
I tried to isolate my meta mode setting up there ^^^
the *_60 indicates my refresh rate.  hope this helps.

### Post by camilluk on 2006-11-13
I'm no expert, but I'll just say what worked for me...

I also have an LCD TV that I wanted to use as a monitor for my ubuntu machine. I read and tried a lot of configuration changes to the xorg.conf and other files in order to get the TV's native resolution. More than once I had to reinstall the whole OS to recover the damage...

The only thing that worked was to install the graphics card's manufacturer's proprietary driver (ATI in my case) which, based on various threads, was not recommendable (I'm not sure why, other than because it's not open source, I guess). Also, once I had done that, it did not require any additional editing of any files. It just detected and selected the necessary resolution, refresh rate etc. It's now working like a charm.

Have you looked for a specific driver for your card? :-k
