### Post by usa on 2005-04-02
Dear all,

You also can add your program shortcut on Applications Menu bar(On the panel) by follow this.
1. go to Applications menubar 
2. select a submenu in which you want to add your program shortcut. For example, you want to add your program shortcut in Office Menu, you just move your mouse to Office.
3. Then, move the mouse to the program list and then right-click
4. select --> Entire Menu --> Add New Item To This Menu
5. Just fill a program name, command (it is a program path, eg. /usr/bin/R)
6. add icon of the program if you want, and finally u will see your program shortcut on the Applications Menu.

Hopefully, it will be useful for you.
Bye.
