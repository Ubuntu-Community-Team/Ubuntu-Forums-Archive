### Post by Super King on 2006-02-28
No problem.

Yeah, Update Manager and Synaptic are linked; if you open up Update Manager and then Synaptic (or vice versa) you'll get this message:

```
Unable to get exclusive lock

This usually means that another package management application (like apt-get or aptitude) already running. Please close that application first.
```

In Synaptic you can hit the "Mark All Upgrades" button at the top and as far as I know this does the same thing as Update Manager.
