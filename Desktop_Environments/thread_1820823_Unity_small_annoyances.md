### Post by ptsant on 2011-08-08
I just registered to write a few simple remarks on Unity. For the record, my first linux install was Slackware on floppies downloaded from a BBS so don't assume that I've never seen linux before.

Anyway, I appreciate the effort you make in creating a new user interface. In order of annoyance, however:
1. Why do the icons for applications and files&folders have to be so similar??? They both are grey with a circle in the center. I mean these are large icons and most systems have 2^24 colors so I bet you could make them different. To make things even worse, these two icons are next to each other, so that every time I need to pause 1 sec and think until I remember which one is for applications. No, it's not that long and it won't diminish my productivity, but it is very annoying.
2. Placement of said icons is not consistent. I know you get the point, because you put the menu bar on the top (consistency for all programs), yet every time I launch some stuff the icons change place. Why?? So that I lose 0.1 sec looking around? Can't at least some icons occupy a fixed position, like the trash bin (which I never use...)?
3. I suggest a very simple metric for UI efficiency: count the number of clicks. Why do almost every time I have to click to "see more results" when I try to launch an application? Surely you can fit some more icons in the window that launches applications! Or at least use a scroll bar, or ask for the "default" number of applications shown. 
4. Why do I need to see, everytime I launch a program, a list of applications I could download? Launching a program and searching for applications to download are two very very different mental tasks. When I'm looking for the program I want to launch I am not looking for the new kewl appz. I am just looking for my favorite program. And, on the other hand, when I want to search for software I could get I prefer something like the software center and I'll go on from there. I don't need to mix the two functions in the same window.

Anyway, I rarely contribute in user forums but I feel that there is the potential for a good user experience in Ubuntu... I don't know whether this is the appropriate forum, will gladly repost elsewhere if I have to.

Thanks,
P.
