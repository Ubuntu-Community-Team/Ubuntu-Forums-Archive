### Post by 2F4U on 2012-08-25
It is no secret that desktop environments such as Gnome, KDE, Unity, etc. are requiring better hardware with every software release. Ubuntu in particular also starts a lot of applications and daemons in the background in oder to provide a certain user experience and to make it more simple for new users to convert to Ubuntu. However, this is not different from, for example, the Windows world.
If you don't need all the bling bling or are not willing to continuously invest in better hardware, you should go for a desktop environment or maybe even for a different Ubuntu derivative. Xubuntu and Lubuntu, for example, are much lighter on resources and many applications present on Ubuntu are missing (a prominent example is the zeitgeist daemon, which can consume a lot of resources).
