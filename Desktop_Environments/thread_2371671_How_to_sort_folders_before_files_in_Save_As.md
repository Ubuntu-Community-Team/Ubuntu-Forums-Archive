### Post by Dennis N on 2017-09-17
The wiget toolkit provides the file chooser dialog, not the file manager. The settings for sorting and bookmarks have to be made in each. The file chooser will have a different appearance depending on the tookit (gtk2, gtk3, qt) used by the application.
