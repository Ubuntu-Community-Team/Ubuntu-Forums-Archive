Why should my global keybindings work only when I use one window engine, Metacity, and not Compiz?

---

### Post by lswb on 2009-09-20
If you configure the shortcuts from Main Menu/System/Preferences/Keyboard Shortcuts, they will work regardless of which window manager you are using (provided you do not reuse the same key combo in the WM keyboard preferences)
