### Post by Dennis N on 2013-04-03
If you are using Ubuntu with the Unity Desktop, you can't make a traditonal taskbar, but you can substitute a window list indicator for a taskbar by using a PPA. Instructions here:

[http://www.noobslab.com/2012/12/install-window-list-indicator-in-ubuntu.html](http://www.noobslab.com/2012/12/install-window-list-indicator-in-ubuntu.html)

OR see item #8 in this list:

[http://www.maketecheasier.com/10-must-have-indicator-applets-for-ubuntu-12-04/2012/06/15](http://www.maketecheasier.com/10-must-have-indicator-applets-for-ubuntu-12-04/2012/06/15)

This indicator creates a drop-down list on the panel of all windows on all desktops.
