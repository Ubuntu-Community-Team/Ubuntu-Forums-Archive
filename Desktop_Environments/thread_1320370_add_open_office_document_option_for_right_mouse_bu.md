Do you have any idea, how can I create Open Office document by pressing right mouse button? I recently used Vista and there was that kind of option available.

I think it is a good use case to create document directly to folder by this way. I have Ubuntu 9.10 and under right mouse button menu there is an option 'create document' but so far only empty file can be created.

Please give me a tip if you are familiar with this issue.

---

### Post by JohnGalt131 on 2009-11-09
Open Open Office and save an empty file, whichever kind you want, to the folder 
/home/{your user name here}/Templates

When you restart the shell, or maybe immediately, it will be in the right click menu.
