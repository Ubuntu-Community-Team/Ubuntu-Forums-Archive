### Post by jpkotta on 2008-11-10
There is no default config file, it's built in.  fvwm is a blank slate when you install it.  You get a config by starting from someone else's and tweaking it.  "Someone else's" might mean something from [Fvwm-Crystal]("http://www.fvwm-crystal.org/"), or on the [Fvwm forums]("http://fvwm.lair.be/"), or using the ancient config generator in the default menu.
