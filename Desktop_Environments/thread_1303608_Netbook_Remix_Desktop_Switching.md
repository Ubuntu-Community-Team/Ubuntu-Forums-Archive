### Post by TommyMac501 on 2009-10-28
It's in synaptic as 'desktop-switcher'.  Be careful however, one big problem I've noticed is that in Desktop mode, there are no "open", "close" or "minimize" buttons on windows.  This is by design in Launcher mode as all windows are opened and closed in the top of the page using Launcher.  In Desktop mode however, this SHOULD be reconfigured to display these buttons and it doesn't.  I just don't know where to post this problem.
