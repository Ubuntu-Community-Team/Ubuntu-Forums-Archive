### Post by hellmitre on 2011-11-23
I wrote a desktop-environment-agnostic tinygrab clone in bash. It's called zt, and works best if you bind it to a hotkey. In order to be useful, you'll need ssh keys set up to the sftp server you upload to.

It takes a screenshot with scrot, copies it to the desktop clipboard with xclip, and uploads it to an sftp server of your choosing with a randomized filename.

The script can be downloaded [here.]("http://hlmtre.zero9f9.com/hosted/zt")

My [blog posting]("http://blog.zero9f9.com/2011/11/tinygrab-for-linux/") with an explanation and screenshot.

Hopefully this is useful to someone.
