### Post by TransitMan on 2007-04-22
*Buntu 6.06.1 LTS was the first offering from the *Buntu team to test the feasabiltiy of Long Term Support for the desk-top user.

The next several releases, 6.10, 7.04, 7.10 are on the 18 month support cycle. 

The proposed next LTS release is being bantered about is 8.04, next year [2008]
The LTS versions are supposed to be supported for 3 years desk-top and 5 years server.

I have three computers using the LTS version, and updates come through as the OS needs them. I have not had any problems with them.

On my main machine I update (so far) with every release to see what is new so I can tell folks about it and try and convert them.
