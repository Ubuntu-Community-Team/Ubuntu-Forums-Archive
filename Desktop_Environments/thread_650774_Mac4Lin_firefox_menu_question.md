### Post by mmcmonster on 2008-02-22
From the [Sourceforge.net site]("http://sourceforge.net/project/platformdownload.php?group_id=204373"):

> Download all the .tar.gz files and Mac4Lin documentation PDF to a temporary folder, Mac4Lin. Untar (extract) all these .tar.gz one by one. There will be folders created for individual components. For the complete pack you will need to download all the files. Read the Mac4Lin documentatin for installation instructions. Some folders may contain release specific README files.

Mac4Lin_Part1_v0.4.tar.gz contains GTK Metacity theme, GDM theme, Mozilla Firefox addons, Pidgin theme, Media Player skins (XMMS, Audacious, BMP etc.), Sounds, Cursors, GTK Splash theme, Font enhancing guide, USplash theme, Grub Splash theme, Emerald theme, Exaile and Pidgin AWN plugins, AWN theme, VLC theme.

Mac4Lin_Icons_Part2_v0.4.tar.gz contains GTK Icon theme.

Mac4Lin_Wallpapers_Part3_v0.4.tar.gz contains a collection of Mac Wallpapers.

Mac4Lin_Documentation.PDF contains illustrated installation instructions which supercedes individual README files. README files are now included only if there is some release specific information.
