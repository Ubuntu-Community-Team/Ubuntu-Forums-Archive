### Post by entropy1 on 2013-06-27
Interestingly, in 13.04, if you use CompizConfig Settings Manager > Desktop > Ubuntu Unity Plugin > Launcher, you can set the Launcher Icon Size as small as 8.  With System Settings > Appearance, the minimum you can set is 32.

### Post by rrnbtter on 2013-07-10
Greetings,
Install CompizConfigManager.  Open CompizConfigManager. Open the Ubuntu Unity Plugin. Choose Launcher. Scroll down to Launcher Size and set it to whatever size you want. The Launcher should respond to the setting immediately as you change the value. Exit CompizConfigManager
