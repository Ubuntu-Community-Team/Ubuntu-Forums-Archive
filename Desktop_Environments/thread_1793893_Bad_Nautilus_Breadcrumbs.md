### Post by megabytemonster on 2011-06-30
Nautilus breadcrumbs appears to draw custom GTK widgets, while your default GTK theme is adding spacing as they are separate buttons. I suggest trying with a different theme like Clearlooks and see if the problem still occurs. If not, report the issue to the breadcrumbs developer, maybe they can override the default spacing.
