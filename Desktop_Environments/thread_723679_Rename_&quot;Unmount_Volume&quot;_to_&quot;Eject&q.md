### Post by fdimmling on 2008-03-14
AFAIK unmount and eject are different commands. Unmount keeps the device but removes the insertion into the file tree. Ejects removes the device node. In Breezy if I remember correctly you had the problem that eject could only be done as root for USB harddrives while USB sticks worked as expected. Created a lot of confusion.
