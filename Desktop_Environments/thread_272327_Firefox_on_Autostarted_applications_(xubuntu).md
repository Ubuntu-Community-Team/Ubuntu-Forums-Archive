How can I set firefox to be automatically open when user logs on to their account? where do I need to find the firefox executable file?

---

### Post by aysiu on 2006-10-06
Just click on the XFCE menu and go to Settings > Autostarted Applications

You don't need to find the Firefox executable. Just type ```
firefox
``` for the command.
