### Post by AliTabuger7 on 2008-06-10
Well, hopefully you've figured out that alt + f2 will allow you to run programs.

So, try hitting alt + f2 and then type "gnome-terminal".

In the terminal that pops up, type "gnome-panel" and tell us what it says in the terminal.

You can also use "alt + f2" then typing "firefox" to get the browser up. This way you don't have to restart your computer to post the output.
