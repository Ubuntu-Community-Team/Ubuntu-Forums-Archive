### Post by aze555666 on 2009-04-26
Try "x-window-manager --replace" , it will replace compiz with metacity as window manager (but no more effets since you don't have compiz as window manager ... you can then get back to compiz by editing the file  "/home/YOU/.gconf/.desktop/gnome/session/required_components/%gconf.xml " and replace the word "metacity" by "compiz" in it.


Another idea : I had black screen problems with compiz since I upgraded to Jaunty. It came from compiz cube making the window manager crash. I desactivated  "desktop cube" and "rotate cube " using compiz config settings manager (was able to do it, cause have a dual screen ,and only 1 of them had a crashed compiz making it black). Now it works ... I don't know if your white screen has the sale cause, but it is worth trying.
