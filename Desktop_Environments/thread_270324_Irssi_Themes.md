### Post by Cronjob on 2006-10-03
How are you installing and activating the new themes? It's a straightforward process, so I'm not sure why it wouldn't work.

Download the .theme file to your ~/.irssi directory and then /SET theme theme_name within the program and you're done.

I've only recently started using irssi, but there is a lot of scriptability in it. Do you perhaps have it configured to automatically keep selecting the same theme, so it reverts to that instead? Maybe you can back your irssi user directory somewhere and then delete the existing contents and start irssi up and see if it works?
