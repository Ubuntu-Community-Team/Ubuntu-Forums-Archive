### Post by itsjustarumour on 2010-05-02
I had this problem on one machine which I upgraded from 9.10 to 10.04 - thus upgrading AWN from v0.32 to v0.4.0.  Removing AWN from System>Preferences>Startup Applications, and then going into the AWN settings and putting a tick in the "Automatically start AWN on login" checkbox, fixed the problem.
