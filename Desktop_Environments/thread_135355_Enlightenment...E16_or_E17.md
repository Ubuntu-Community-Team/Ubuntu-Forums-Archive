You had all the advantages of gnome with enlightenment all at once....does anybody know how to have a setup like that under ubuntu? I think it would be way cool.[/QUOTE]
I use openbox as my window manager in gnome and here is how I did it:
1. Put the following into ~/.gnomerc (create the file if you don't have it)

  export WINDOW_MANAGER=/usr/bin/openbox

2. Open gconf-editor, go to apps/nautilus/preferences and uncheck "show_desktop" to make the openbox desktop menu appear. Log out of gnome and back in.

That is for openbox but you should be able to use that info to make enlightenment your default window manager.
