### Post by varunus on 2006-03-20
If you want GNOME 2.14, you need to wait for Ubuntu Dapper Drake to be released.  Then, upgrade your computer using Synaptic Package Manager to Dapper Drake.  If you want it now, along with potential instabilities, you can upgrade to Dapper (the development version) now....but I wouldn't recommend it unless you know what you're doing.

Ubuntu times its releases close to GNOME's so that each new version of Ubuntu has the newest GNOME in it.
