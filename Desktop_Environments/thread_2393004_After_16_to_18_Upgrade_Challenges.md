Well....this has been an interesting run.  So far *most* things worked fine after the upgrade, but there are several small annoyances, cataloged below:

Synergy tray icon doesn't show up when running 
After every reboot I have to change to the correct sensor in system-monitor
Almost after every reboot I have to change the auth type for a gmail account from OAuth2, to Password (I don't have to input the password though, just make the change)
Certain applications and their .desktop files refuse to get added as favorites (ida pro and steam).  They work fine double-clicking the desktop files, but no way to save (even tried the method here:  [https://askubuntu.com/questions/370592/how-can-i-add-a-custom-launcher-to-favourites-in-gnome-shell]("https://askubuntu.com/questions/370592/how-can-i-add-a-custom-launcher-to-favourites-in-gnome-shell")

Files app has gotten some better with tabs, still not having a dual pane mode is a drag so I'll stick with nemo for now.  All in all.....the Ubuntu 18/Gnome setup just feels....flaky....not solid.  I'd love to see if anyone else has the same experiences that I've had, or solutions to the above...thank you.
