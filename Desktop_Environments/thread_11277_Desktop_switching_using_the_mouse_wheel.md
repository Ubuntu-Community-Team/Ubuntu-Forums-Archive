### Post by grenier on 2007-04-26
Short answer: No!

More precisely, it's impossible with Metacity (as someone said in a previous post).

If you want to be able to do it under gnome, you have to either:
- switch from Metaciti to Openbox (but a couple of releases back, there issues doing this, most notably with the auto-updates daemon. No idea how it goes now).
- use either Beryl or Compiz. Both provide that functionality by default.

As for me, after giving the later a try, I guess I'll go back to using Windowmaker that I find so much more to my taste.
