### Post by mali2297 on 2009-06-14
If you switch to workspace 2, can't you see the app there?

I interpret your problem as that you previously had set the task list to show windows from all workspaces. If so, you should be able to reenable this setting by right-clicking on the task list and choosing *Properties*. There you will have the option *Show tasks from all workspaces*.
