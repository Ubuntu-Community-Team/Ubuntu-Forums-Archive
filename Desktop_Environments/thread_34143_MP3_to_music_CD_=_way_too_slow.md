### Post by cowbud on 2005-05-13
Currently Gnome does not have a good solution for on-the-fly Audio cd burning. I suggest installing k3b. It will install a lot of KDE stuff but it is the best solution currently available.

### Post by mattack on 2005-05-14
K3B seems to do the trick. I didn't actually burn a CD but rather an image. The conversion went by in 5 minutes, which is about right... Plus K3B integrates with Madman, which I've been using for a few months now. 
Thanks for your suggestion!
