### Post by Rinzwind on 2005-12-07
To delete gnome:> 
sudo apt-get remove ubuntu-desktop

To delete KDE:> 
sudo apt-get remove kubuntu-desktop

This does NOT delete all the lib files I think.
But if you have deborphan installed that can be used to delete all the libs you do not use anymore.
Here is a topic about deleting files that remain after you did the removal of gnome
[http://www.ubuntuforums.org/showthread.php?t=96046&highlight=deborphan](http://www.ubuntuforums.org/showthread.php?t=96046&highlight=deborphan)
