### Post by jamespetts on 2007-05-07
Sometimes, this sort of problem can be solved by turning off all the analogue inputs into the soundcard, such as line-in, microphone in and CD audio (most CDs these days play via the IDE bus so that the computer can process their sound, not via the analogue CD audio cable).
