### Post by jfed on 2011-08-31
Of course there is. Most things are possible in Linux, due to the whole open source deal. ;)

This sort of post was extremely common with the release of 10.04, in which for some reason the Ubuntheads (what I refer to the devs as xD) decided it would be a good idear to puts teh buttons on the otherside with the default theme. This can be easily changed by changing the theme. To do that, go to system>preferences>appearance and you should be able to change the theme from there.

If you like the current theme and wish to have the buttons moved, just follow this guide.

[http://www.howtogeek.com/howto/13535/move-window-buttons-back-to-the-right-in-ubuntu-10.04/](http://www.howtogeek.com/howto/13535/move-window-buttons-back-to-the-right-in-ubuntu-10.04/)

Or you could use a program like Ubuntu Tweak to change it, but honestly I just think that changing the theme is easiest. :lolflag:

Hope I helped.
