### Post by grahammechanical on 2013-02-10
I am concerned that we may be mixing things up.

There is a difference between a Desktop Environment and a User Interface. Ubuntu is based upon the Gnome 3 DE. Kubuntu is based upon KDE (version - unknown). KDE = K Desktop Environment.

Ubuntu uses the Unity UI. There are other User Interfaces that we can install. We can log in to them at the login screen. The Gnome organization has it own UI. It is called Gnome 3 Shell.

If someone wants Ubuntu with a different Desktop Environment than Gnome DE, then the choice is Kubuntu (KDE), Xubuntu (Xfce), Lubuntu (LXDE).

As for Fluxbox there is this

[https://help.ubuntu.com/community/Fluxbox]("https://help.ubuntu.com/community/Fluxbox")

The information is 2 years old. You would be best served by checking out Fluxbox on their web site.

[http://fluxbox.org/]("http://fluxbox.org/")
