### Post by Qrk on 2006-03-30
Those fun things are gdesklets

```
sudo apt-get install gdesklets
```

In universe, I believe, you'll have to have enabled your extra repositories.

Themes are found in [http://art.gnome.org](http://art.gnome.org) and [www.gnome-look.org](www.gnome-look.org) install them with System->Preferences->Theme

If you want them to show up on root apps, extract the theme to /usr/share/themes/
