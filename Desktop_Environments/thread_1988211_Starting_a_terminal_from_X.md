### Post by bogan on 2012-05-27
Hi!. **piriton**,

You Posted:>  There doesnt appear to be a menu which says start a shell In Ubuntu a 'Shell' is called a 'Terminal'.  [ Incidently, Gnome enthusiasts call Gnome itself the Gnome-Shell.]

In addition to the Short-cut: 'Ctrl+Alt+t', there are several other ways:

Click on the 'Dash' button top left in Ubuntu,  or on 'Activities' in Gnome, Enter "term" in the search string of Ubuntu or Gnome and they will offer you four different ones; in Gnome Classic Noeffects, the Menu: 'Applications'>'Acessories' will list two terminals, the other two are in System Tools. [Edit: No-Effects Menu added.]

You can choose one and drag it to the Launcher bar on the left side.> nor an option  where you can execute any command,The option is: 'Alt+F2' which opens a command string box in both Ubuntu and Gnome.

Another way to open a 'shell' is to use that option - 'Alt+F2' -  enter "gnome-terminal" in the search box and press enter. The Icon for the command will stay in the display as History so the next time youi enter 'Alt+F2' you can just click on it without having to type anything. The same applies to any command run from a command string box like this.

To get a list of some other useful Keyboard Short-cuts, hold down the 'Super key' - That is either of the MS Logo keys, in Windoze talk.
Chao!, **bogan.**
