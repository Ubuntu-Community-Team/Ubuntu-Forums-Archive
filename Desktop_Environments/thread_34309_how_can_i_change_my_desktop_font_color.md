### Post by Xian on 2005-05-14
AFAIK you need to make a new gtk theme to change this, or you can edit one of the existing themes. For a reference you can review the [gtkrc file](http://developer.gnome.org/doc/API/gtk/gtk-resource-files.html) file for the High Contrast Inverse theme. It uses white text on a black background for all components.

Or you can install the [Gnome Configurator](http://www.krakoa.dk/linux-software.html#COG). I believe it can alter and manage theme colors:
