### Post by aysiu on 2005-12-13
[QUOTE=earobinson]True but its all about preference, more options cant hurt.[/QUOTE] But I feel a lot of people who ask "How do I log in as root?" assume that that *is* the only option.

Far more users in Ubuntu who use sudo have used *both* sudo and root (either in this distro or another), but few people who ask "How do I log in as root?" have even tried not logging as root and just embracing the sudo model.

How many Mac users demand to be able to log in as root? I would guess it's not as many as Ubuntu users who make that same demand. Still, Mac users seem to get along just fine using sudo.

People have the expectation that if they're using a Linux distro, they must have a root account that can log in separately. I'm just telling them there is another option. You can only "decide" on an option once you know the others exist. It's like a Windows user complaining about Linux before giving it a real shot... same with Linux users complaining about sudo before giving it a real shot.
