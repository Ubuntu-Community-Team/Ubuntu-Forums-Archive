### Post by manimal347 on 2008-02-04
Fluxbox isn't a Desktop Environment; it is a window manager. Fluxbox is a minimalist window manager that is up to date, takes a few megabytes of disk space, has almost no dependencies, and can work with both Gnome and Kde applications assuming you have the right backends installed. If you have a standard Ubuntu install, you'll already have everything you need.

When I say minimal, I do mean it. There's very few config menus; most things are changed by tweaking config files in a text editor. It uses a command line utility to set wallpaper, and doesn't support a desktop folder or related icons without certain add-ons like Rox. It works nicely on old hardware running light programs such as XFCE's Thunar, terminal software, or Abiword. It's certainly not for everyone, but it's often what is used to make those hacked out Gentoo Desktops with the system monitor on the desktop and all. Basically, it hews closely to the hacker spirit of Unix without becoming unusably obtuse, like Ratpoison or Ion3.
