### Post by kerry_s on 2006-09-04
I use eterm, it's easier to change and save the settings an you can also go transparent with no frame ( Eterm --borderless --buttonbar 0 --trans --shade 0 --scrollbar false ). Just a reminder you will need xterm no matter which one you choose, the x window is set up to use xterm if there's a problem with gui, if it's not there you'll just be stuck. plus most things are set up to use xterm as default. I found out the hard way. ;)
