### Post by Jucato on 2006-07-17
It's called filter search, and it shows up only when you're browsing directories/files. Unfortunately, it's not as "instant" as Nautilus' or Thunar's because you have to click on that search field. However, I personally think it's worth it because it does something that neither Nautilus or Thunar could do. In Nautilus/Thunar, you type "ab", it will highlight (only highlight) the first file that starts with "ab". In Konqueror, if you type "ab",it will show files/directories that have "ab" in the name, including "abcd", "cabd", and "cdab", hiding everything else that doesn't match the filter search.

I'm still looking for a way to activate this search without having to manually click on the filter search input field...

### Post by aysiu on 2006-07-17
Try hitting *a*, then *a*. The second time you hit *a*, it will go to the second folder/file beginning with *a*.

In other words, let's say you have two folders: *aardvark* and *abacus*.

In Nautilus, you would type *ab* to get to *abacus*. In Konqueror, you would type *aa* to get there.
