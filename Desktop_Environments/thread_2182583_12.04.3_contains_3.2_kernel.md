We're looking at a wiki entry, which apparently has been superceded by some subsequent decisions, or whoever wrote it had incorrect information.

The manifest for Xubuntu 12.04.3 64 bit indicates:

[FONT=Courier New]**linux-image-3.2.0-52-generic	3.2.0-52.78**[/FONT] so that is what you get.

see:
[http://cdimages.ubuntu.com/xubuntu/releases/precise/release/xubuntu-12.04.3-desktop-amd64.manifest](http://cdimages.ubuntu.com/xubuntu/releases/precise/release/xubuntu-12.04.3-desktop-amd64.manifest)

_Added information:_

The wiki statement would be correct if it were about Ubuntu 12.04.3. The manifest for the Ubuntu 12.04.3 release shows: 

[FONT=Courier New]**linux-image-3.8.0-29-generic	3.8.0-29.42~precise1**[/FONT]

see:
[http://cdimages.ubuntu.com/releases/12.04.3/release/ubuntu-12.04.3-dvd-amd64.manifest](http://cdimages.ubuntu.com/releases/12.04.3/release/ubuntu-12.04.3-dvd-amd64.manifest)
