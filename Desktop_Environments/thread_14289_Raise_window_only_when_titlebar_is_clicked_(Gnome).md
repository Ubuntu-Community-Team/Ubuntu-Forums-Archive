### Post by Fubdub on 2008-07-10
Better late than never.

I prefer this way to handle my windows as well. I found a way to do it, and it seems to work in Hardy. (There might be siteeffects, we'll see I suppose.)

1) click alt+f2 and type in gconf-editor
2) go to apps>metacity>general
3) remove the tick in raise_on_click

Done, it should take affect straight away.

If anybody find any problems with this method, please let me know.
