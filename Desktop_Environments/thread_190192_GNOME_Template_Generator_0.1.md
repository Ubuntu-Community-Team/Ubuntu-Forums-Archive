Hi, it's 06/06/06 and I like to post my first forum topic.

I keep wondering why does Templates are non existent by default. So I made a simple app to help new user to create those.

see, [http://www.gnomefiles.org/app.php?soft_id=1408](http://www.gnomefiles.org/app.php?soft_id=1408), it's the GNOME Template Generator, an easy tool to create $HOME/Templates documents based on your locale and application installed. 

Templates can be accessed via right-click > Create Document. Currently only have OpenOffice.org, and GIMP templates. I hope it will have a larger template database. Help me by uploading one, and also translating the template.
