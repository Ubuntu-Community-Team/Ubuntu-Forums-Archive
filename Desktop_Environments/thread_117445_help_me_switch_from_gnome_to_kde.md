### Post by aysiu on 2006-01-14
[QUOTE=pedwards]
1 how easy it this switch to do.  
         is it as easy as apt-get install kde   ???
2 how permanent is that switch, 
         meaning if i dont like it how do i get back to gnome[/QUOTE] It is as easy as that. I actually prefer *sudo apt-get install kde* to *sudo apt-get install kubuntu-desktop*, but both are meta-packages and, thus,  not as easy to *uninstall* as to *install*.

I did write [HowTo on removing kubuntu-desktop](http://www.ubuntuforums.org/showthread.php?t=96048), but since you already know you're installing it and may want to uninstall it later, the best thing to do is this:

1. Type *sudo apt-get install kde*
2. Before saying "yes," highlight all the packages that are to be installed and copy them to a text file you save on your desktop (or wherever, actually).
3. Say "yes."
4. If you later decide to get rid of KDE, you can type *sudo apt-get remove* and then paste in the package list you copied to your text file earlier.
