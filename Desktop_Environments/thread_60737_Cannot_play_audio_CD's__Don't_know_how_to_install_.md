### Post by uperjer on 2005-08-29
for installing programs, i'd really recommend going into kynaptic (i'm assuming you're using kde since  you're posting in this forum) and doing a search for 'synaptic' and installing that.  when i first started i only used kynpatic and it sucks in comparison to synaptic.  synaptic is a nice graphical user interface package downloading system.  if, for example, you're wanting to install firefox, just go into synaptic, do a search for firefox, and click install.  the end.  it was VERY helpful when i was first starting out (not too long ago) and someone suggested this VERY POWERFUL program.
