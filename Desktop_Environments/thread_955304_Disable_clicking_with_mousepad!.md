### Post by woodenbrick on 2008-10-22
You will need to enable SHMConfig in your xorg.conf and install qsynaptics.
There is a guide here:
[https://help.ubuntu.com/community/SynapticsTouchpad]("https://help.ubuntu.com/community/SynapticsTouchpad")

Gnome has System -> Preferences -> Mouse where it can be changed too, I don't know about XFCE
