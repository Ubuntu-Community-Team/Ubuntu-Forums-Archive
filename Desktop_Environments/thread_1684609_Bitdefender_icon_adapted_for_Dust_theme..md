### Post by Dondermans on 2011-02-09
I recently installed Bitdefender and noticed that the icon which is added to the notification area is displayed against a greyish background:

[IMG]http://i133.photobucket.com/albums/q41/Don82/Publiek/Ubuntuforums/Originalicon.png[/IMG]

I adapted the image used for the icon to make it blend in with the Dust theme:
[IMG]http://i133.photobucket.com/albums/q41/Don82/Publiek/Ubuntuforums/Adaptedicon.png[/IMG]

1. Download the attached icon_small.png
1. Press Alt + F2 to bring up the 'Run application' window 
2. Enter 'gksu nautilus' to navigate in nautilus with root permission
3. Navigate to /opt/BitDefender-scanner/var/skins/Default/
4. Locate the file icon_small.png
5. Backup the existing icon by renaming it. I appended '_backup' to the filename
6. Add the icon_small.png you downloaded 
7. Finished
