### Post by Krytarik on 2011-05-01
Create a launcher at your Unity desktop for the command "ccsm" (CompizConfig Settings Manger), launch it and enable the "Ubuntu Unity Plugin", enable "Desktop Cube" before if you want to use it, the same for "Rotate Cube", and check if there are other plugins that you want/need to enable.

After doing that, relogin.
