### Post by g14 on 2007-09-22
Beryl has been officially discontinued. Do not use it.

The beryl team merged back with the compiz team and the resulting project is named "compiz-fusion". Compiz-fusion will be on by default (if your card supports it) for Ubuntu 7.10 codenamed Gutsy Gibbon. It will be out in about 3 weeks.

There are really good compiz-fusion tutorials for Feisty if you are using that. If you just wait until gutsy is released and use it, things will likely be much more simple.

[Place to see that Beryl has been discontinued]("http://www.opencompositing.org/")
