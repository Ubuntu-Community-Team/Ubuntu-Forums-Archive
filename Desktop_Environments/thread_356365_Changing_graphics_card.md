### Post by taurus on 2007-02-08
Press <Ctrl><Alt>F2 and log in with your name and password.  Then at the prompt, type

```
sudo dpkg-reconfigure xserver-xorg
startx
```

### Post by igknighted on 2007-02-08
general rule of thumb is don't bump until 24hrs has passed...

please post your xorg.conf.  It sounds like you have a missmatch somewhere between your screen section and your serverlayout section.  I would double check that the identifiers that serverlayout points to are the proper ones.  Failing that, try running aticonfig --initial and let the ATI tool do the setup.  Last but not least, you may need to recompile the kernel module for your new card if you compiled it for the old one (if it was an ATI).

### Post by taurus on 2007-02-08
Edit /etc/X11/xorg.conf and change the Driver from "vga" to "vesa".

```
sudo nano -w /etc/X11/xorg.conf
```
Then, see if you can start X again with

```
startx
```
