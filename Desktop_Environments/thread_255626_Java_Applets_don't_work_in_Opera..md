### Post by Dinerty on 2006-09-12
I installed Java manually and then just told Opera where to look for it.

Tools > Preferences > Advanced > Content > Java options

I installed my Java to /usr/lib/j2re1.5-sun/lib/i386

Copied that into the box and validated it, worked fine from there on :)
