### Post by shadarack on 2007-02-14
Okay, I have an interesting question here...

I am using XCFE and Thunar, but when I delete a file that is on a partition other then my home partition, Thunar insists on moving it into my trash file which is on my home partition. This can be quite time consuming when the file is very large, and I enjoy torrenting anime. I tried using Nautilus, but it takes over my desktop display settings and I really like what XFCE does with my display settings(I have scripts set up to change my desktop picture every half hour through XFCE). So my question is - is there a file manager that I can use that won't take over my desktop settings, but will still allow me to delete a file from where it is instead of making me move it first? Perhaps is there a way to make Nautilus leave my desktop alone? A way to make Thunar not move files? Please, any advice on the subject would be nice. This is a chance to tout your favorite file manager I guess...

### Post by erkki70 on 2007-02-14
Hi,
Try pressing the Shift key while deleting file (shift+del or delete from menu).
This shouldn't move anything to your trash...
