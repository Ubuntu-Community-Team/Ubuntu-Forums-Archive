### Post by adelahunty on 2008-03-30
In xfce, you can do one of three things to get to an application without needing to type the command at a shell:

1.  Add a button to a taskbar: right-click on the taskbar, and add a new item.  Choose a Launcher item.  In the dialog which appears, give the command you would type at the shell, a description, and an icon.  When you clock it, it will start the app you want.

2.  Go to Settings in the xfce menu, and choose Main Menu.  From there you can add an item (with icon if you want) which can run your program.

3.  Right-click on the xfce menu, and choose the Properties option.  Choose to add a new item in the new dialog which appears, and it will be put wherever you choose it.
