### Post by wakediktion on 2007-03-13
mrmonday,

I actually was in the same boat as you. I used that same page last night to install the KDE environment. It seemed a little too easy and the install was as easy as it looks. Took about 10 - 15 minutes to download and install all of the packages but it went flawless for me.

1.Switching between Gnome and KDE is very easy. KDE becomes available at login through the sessions button.
2.Beryl - I also had Beryl in Gnome and I had it set to start up with login but I noticed that when logging into KDE, it didn't start right away (haven't gotten that far to change it but I am sure KDE has that option). For some reason it consistently takes two attempts to get Beryl started but it does start and runs the exact same for me. 
3. I did not have to reinstall Beryl. Hopefully you don't either.

GDM = Gnome & KDM = KDE

I set mine for GDM just because I wasn't sure if I would like KDE but this is easily changeable through the same sessions button you use to run KDE. I really enjoyed my first experience with KDE and I am strongly considered using Kubuntu 7.04.
