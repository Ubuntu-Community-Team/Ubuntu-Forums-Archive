### Post by philippe_carlo on 2006-07-17
I had similar problems before. The problem IS the memory, or in fact, it is the motherboard that probably does not deal well with the 2GB of memory. I once bought an MSI MoBo that theoretically could handle 1GB Memory Units, but practice proved otherwise. The only solution in my case, was to buy smaller memory units. If the MoBo were new, you could complain about it and demand a better one, but I'm afraid you're stuck now.
