### Post by Erunno on 2007-06-30
I can't really say at the moment. The svn version runs fine on my 2 GHz, 1 GB RAM desktop computer but that was to be expected I guess. Qt 4 is supposedly more resource-efficient than Qt3 but at the same time a lot of new features like Strigi are introduced into KDE and the complete switch to the Modell/View pattern and some additional abstraction layers might also cost some performance.

I guess it's kinda inevitable that in the long run desktop environments will use more resources as people are expecting more and more features from their desktop. After all, KDE1 is probably faster than KDE4 but who'd want to use that today?
