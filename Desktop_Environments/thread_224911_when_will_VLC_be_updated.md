### Post by stevea1210 on 2006-07-28
I don't know when that will happen.  However you can add

for Dapper:
```
deb http://nightlies.videolan.org/build/dapper-i386 /
```
or for breezy:
```
deb http://nightlies.videolan.org/build/breezy-i386 /
```

to your repos, and get the nightly builds.  There is also direct download links for the nightlies on this page:
[http://nightlies.videolan.org/]("http://nightlies.videolan.org/")

The ubuntu info is at the bottom of that page.
