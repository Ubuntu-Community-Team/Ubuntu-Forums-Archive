### Post by Copper Bezel on 2012-02-23
If the machine has a Blu Ray player, then you can already play Blu Ray disks in Ubuntu. Same with the TV tuner - it's nothing special if you have the hardware and the configuration. The idea with Ubuntu TV is that you buy an all-in-one smart TV that has all of this set up and ready to go. They're not available yet - I don't know if any vendors have even actually  said they're going to be working with Canonical. But an HTPC and an ordinary TV, with a Blu Ray drive and a TV card and so on, can do this with an ordinary Ubuntu install and a little bit of setup.

> is Ubuntu TV strictly for Smart TVs later or for desktops/laptops/tablets as well? 
Since it's a modification to the OS, it wouldn't make much sense on a laptop or something, but I think they intend to make it available when it launches for HTPCs and the like.

> Where do recordings get stored @ the "cloud" or an option to store to a dedicated/networked hard drive
Locally, and shared to your private cloud through Ubuntu One.

### Post by grahammechanical on 2012-02-25
In my mind the title of the link to Ubuntu.com says it all.

> Everything you want in a TV. **In a TV**.

It is an interface for Internet capable televisions. I do not think that Ubuntu TV is intended to be a Home Theatre Personal Computer (HTPC) operating system.

When Microsoft brought out Windows Media Centre Edition it could only be purchased pre-installed on specifically designed media centre PCs. In this way everything worked as it should and there could be no complaints about Win MCE not having the drivers for someone's hardware or not giving a satisfactory viewing experience.


If Canonical do intend to release a HTPC Ubuntu then they should only make it available pre-installed.
