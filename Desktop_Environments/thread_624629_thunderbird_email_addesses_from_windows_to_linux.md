### Post by adam_kimber on 2007-11-27
If you copy the profile the addresses should come with it. However if that fails just export your address in book in Windows  to say a comma separated file and then import in Linux (Tools --> Import) making sure you select the type of file you selected in Windows for export (in this case comma separated).
