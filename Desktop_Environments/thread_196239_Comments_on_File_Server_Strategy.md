### Post by treris on 2006-06-14
you could just install the ubuntu-desktop and then use that to set everything up and then jccj7 said boot up without the gui to save resources

installing the ubuntu-desktop should be as simple as typing

sudo aptitude install ubuntu-desktop 

in the cli and the installation 'll start, using aptitude makes it easier to install the entire gnome environment should you want to at any point in time

for more information you could also check [http://psychocats.net/ubuntu/gnome.html](http://psychocats.net/ubuntu/gnome.html)
