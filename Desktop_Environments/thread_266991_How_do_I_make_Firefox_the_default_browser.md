### Post by abhiomkar on 2006-09-28
Open firefox, goto Edit > Preferences . In General tab, check "*Firefox should check to see if it is default browser when start*" ,  and click on **Check Now** button then it will ask you to set the firefox as default browser , click **Yes**.
