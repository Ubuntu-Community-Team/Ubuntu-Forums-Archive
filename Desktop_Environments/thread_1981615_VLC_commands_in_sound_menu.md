### Post by ewz on 2012-05-17
Have you already enabled dbus in VLC?

The way I did it was in VLC select Tools - Prefenences, tick All in Show Settings, Navigate to Interface - Control Interface and tick dbus and save settings.

Restart VLC and they should show up in the Sound Menu. 
The Controls will only show up when VLC is open though.
On another Computer I had to log out and back in for it to work 

I use VLC From the Ubuntu Software Center it's Version 2.0.1
