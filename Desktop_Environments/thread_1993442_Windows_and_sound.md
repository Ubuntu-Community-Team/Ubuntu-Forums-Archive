### Post by bug67 on 2012-06-02
> **lFuidy said:**
>  1. I prefer my apps to be in Window mode but everytime I close them and start them up again, it reverts the windows to the far left side of the screen instead of remembering where they were positioned last. Is there anyway to fix this?
The only way I've found to anything somewhat similar is to:

Settings Manager > Windows Manager Tweaks > Placement.  Then, make sure "By default, place windows:  At center of the screen" is checked.  And finally, move the "Minimum size of windows to trigger smart placement" is all the way to the right at Large.

The only problem with this is, it causes *all* the windows to open in the center of the screen.  Personally, I find that preferable to one side/corner or another.  [You could try installing Compiz]("http://ubuntuforums.org/showthread.php?t=1754637").  That is supposed to allow for custom window placement but, I never could get it working right in Xubuntu.
