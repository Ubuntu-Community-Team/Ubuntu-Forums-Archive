### Post by SeijiSensei on 2013-06-18
You need to be running version 13.04 to have full support for the MTP protocol that the most recent Android versions use.

Otherwise you can install a file transfer app on your phone like WiFi File Transfer Pro and manage your phone's storage with a web browser on the PC.
