### Post by rooijan on 2010-05-07
Actually I think I can see what they did.  

Set up alternate "Locations".  You can see the "Locations" option when left clicking the clock applet.

Now if you hover over a location you configured you will see a "Set" option. As easy as that!  The existing selected timezone will have a little house icon next to it.
