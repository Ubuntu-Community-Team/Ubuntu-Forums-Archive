### Post by taurus on 2006-02-25
If you want to copy CDs in Ubuntu, k3b is a nice app that you can use for that task.  Of course, there are others apps but personally, I go with k3b.  To install it, run

```

sudo apt-get update
sudo apt-get install k3b

```

Then, Click Applications -> Sound & Video -> K3b
