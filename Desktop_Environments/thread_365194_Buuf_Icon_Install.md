### Post by Zeek00 on 2007-02-19
Now upon unpacking them to the .icons folder how do I use them. They don't show up under system->themes

I don't know if you have ever worked with these icons but they are not in the typical format that most icon themes you download off gnome-look or other sites are. They are packaged with a .bz2 file extension which the theme installer doesn't recognize and if you open up the archive the contents are of an unknown file extension.

Goto gnome-look-org and search buuf and download one and you'll see what I mean.
