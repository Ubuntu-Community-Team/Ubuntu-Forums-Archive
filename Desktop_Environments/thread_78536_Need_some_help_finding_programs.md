### Post by billycub on 2005-10-18
Nero does a lot of different things; each is accomplished a separate way with a standard install of Ubuntu.

- To burn audio CD's, use serpentine (Applications - Sound/Video)
- To burn data CD's, use the file manager (Places - CD/DVD Burner)
- To burn ISO image files, use the file manager (right click the ISO file, choose "Burn to CD")

I don't know anything about dvdshrink or dvdencrypt; if you're looking to rip DVD's you should install dvd::rip using synaptic (System - Administration - Synaptic Package Manager).  You'll have to enable the universe/multiverse repositories first before installing dvd::rip.

Note that there are lots of other apps which perform similar functions in Ubuntu.  These are the defaults.
