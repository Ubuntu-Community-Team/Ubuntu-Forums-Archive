### Post by ardchoille on 2006-10-02
> **lawina said:**
> Are these themes Ubuntu compatible?
I am afraid they might break Ubuntu!!:(

Themes should work fine from these sources. I have been using the sources the folks above mentioned for years and I haven't had any problems.

The only thing I will mention is that the gnome-theme-manager installer doesn't do a very good job of installing multi-theme tarballs, such as T-ish_Brushed, so you need to untar GTK and Metacity multi-theme tarballs into /usr/share/themes. Icon themes can be untarred into /usr/share/icons.
The neat thing is that you can either use the theme installer or untar them into their appropriate home :)

If you find a theme that doesn't work at all, most likely it's a problem with the themerc or other rc file associated with the theme and it can be fixed easily. I've been messing with themes for a long time and have never seen one break a distro. If a theme actually did break a distro, then I would highly suspect that the distro itself was hosed to begin with.
