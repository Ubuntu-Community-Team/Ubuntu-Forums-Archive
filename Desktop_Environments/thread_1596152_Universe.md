### Post by ankspo71 on 2010-10-14
Hi,
Some local computer shops might download and burn a copy of Ubuntu for you, for a small fee of $10 or so. Otherwise, here are some places to order some Ubuntu cds online:
[https://shipit.ubuntu.com/](https://shipit.ubuntu.com/) 
[http://www.ubuntu.com/desktop/get-ubuntu/cds](http://www.ubuntu.com/desktop/get-ubuntu/cds)
[http://www.osdisc.com/](http://www.osdisc.com/)
[http://on-disk.com/](http://on-disk.com/)

The Ubuntu DVDs are basically the same except they include language packs and I think they still offer a non graphical installation method, so a CD is fine for most people. 

You will be able to download and install as much additional software as you like right from within the Ubuntu's Software Center or Synaptic Package Manager, so there isn't much of a need for additional CDs or DVDs.

Btw, if you are planning on having wireless, you might first need to have a wired connection, because some of the wireless drivers might only be available online.

Welcome to Linux.

Oh yeah, I forgot to mention that the latest version of Ubuntu is 10.10. It came out 4 days ago.
