### Post by de0xyrib0se on 2009-04-28
Open up your home folder or some other folder using the file browser in gnome, then go to Edit -> Preferences, under views change the 'Icon View Defaults' to a percentage lower than 100, that should do it. Enjoy!

I know it doesn't make a lot of sense but thats how its changed...
