### Post by ankspo71 on 2010-08-05
Hi,
First, since you said that the windows open behind the panel, do this first:

Right click on your panel and unlock the widgets if they are locked.
Then right click on the panel again and select "panel settings"
When that panel option bar opens up, select "more settings".

Where it says "visibility" make sure "Windows Can Cover" and "Windows Go Below" are not selected. In other words, choose either "Always Visible" or "Auto-hide" to avoid having the windows and panel overlap each other.


Part 2 
Go to Applications, Settings, System Settings. Then look for a configuration called "Window Behavior" Inside the window behavior configuration is category called "Window Rules". With these rules you can create a generic rule to have KDE open all windows under your mouse pointer, for example.

Hope this helps.
