### Post by benerivo on 2008-01-06
You can check what you have done in synaptic by looking at the [history]("https://help.ubuntu.com/community/SynapticHowto#head-b9237e6d2038333a1986f10cc001111c3f89a3b8"). If this does not help, you could try making a new user account and logging in to that one to see if the desktop has the same problems (if it doesn't then it's not a package problem).
