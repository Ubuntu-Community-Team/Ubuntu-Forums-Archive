### Post by Hero of Time on 2009-12-01
If you have CUPS installed, make sure that you also have the right packages with drivers installed. After that, open a browser and point it to [http://localhost:631](http://localhost:631) and add the printer through the web interface of Cups (under Administration). If the printer is already connected and turned on, you can easily add the printer, give it a name and the right driver should be pre-selected.
