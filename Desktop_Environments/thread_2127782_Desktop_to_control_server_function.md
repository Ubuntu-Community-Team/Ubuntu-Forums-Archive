### Post by tgalati4 on 2013-03-21
You can do either.  Webmin is through a webpage on another machine.  If you install the desktop, then you can remotely log in (*ssh -2 -Y*) and run the desktop on the remote machine.  Most servers don't need a lot of maintanence, so the command line is really all you need.  If you are learning how to run a server, then command line is like getting an Instrument Rating to fly--you need to learn how to do it with your eyes closed.
