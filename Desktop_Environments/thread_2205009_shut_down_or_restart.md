### Post by grahammechanical on 2014-02-11
What operating system are you using? With the latest Ubuntu it does not matter whether we click Restart or Shutdown we get the same dialog. We need to move the mouse to click one of the two options so how hard is it to move the mouse over to the restart box? We can also use an arrow key.

About a year ago there was an attempt to do away with the restart option from the menu. People complained. So, it came back. The Restart menu option seems to be a simple link to the Shutdown menu option script. Saves having two scripts.
 
In my opinion two options on the power/cog menu are not needed. I would not be surprised if some time in the future the Restart option will be removed. I would not complain.

As far the style of the Shutdown dialog goes, well, it is your opinion. The Design team have a different opinion. And it does fit in with the design style that is in Ubuntu phone/tablet and which will be converged in the Ubuntu desktop so that there is one ubuntu code base for phones, tablets and PCs with a consistent design style. Imagin using Ubuntu on a touch screen device or monitor. Then it will make sense.

The fact is, this forum is not the place to complain. No one with authority to make changes will hear you.
