### Post by Copper Bezel on 2011-06-08
The desktop background and the login screen theme are set separately, and the login screen is a little more difficult. To change the backdrop, just switch to the Background tab on the Appearance screen and drop an image in from the file manager. For the login screen, check out [this]("http://www.webupd8.org/2011/05/change-gdm-theme-background-in-ubuntu.html").
