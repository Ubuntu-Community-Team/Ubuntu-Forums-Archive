### Post by kansasnoob on 2012-04-27
Only if you're using 'gnome-panel' in a Gnome classic (no effects) session try pressing the Alt key and then right-clicking on the applet you wish to remove.

But if you're using 'gnome-panel' in a standard Gnome classic session try pressing both the Alt & Super keys at the same time then right-clicking on the applet you wish to remove.

The difference between "Gnome classic" and "Gnome classic (no effects)" is that the prior uses Compiz and the latter uses Metacity. I've had a lot of trouble with Compiz so I've focused on Metacity:

[http://ubuntuforums.org/showthread.php?t=1966370](http://ubuntuforums.org/showthread.php?t=1966370)

Note: The Super key is typically the one with the Windows logo.
