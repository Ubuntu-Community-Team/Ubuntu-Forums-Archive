### Post by astoltz on 2005-12-03
Under the System Menu, go to Preferences, then Sessions.  In the sessions dialog, there is a tab to add start-up programs.

This will only start FAH when you log in though.  You might have to add a script to the appropriate /etc/rc* directory to have it start at boot time.
