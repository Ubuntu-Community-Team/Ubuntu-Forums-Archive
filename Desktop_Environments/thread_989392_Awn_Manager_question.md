How can I set up Awn as a default? I'm running 8.10 on an Acer Aspire One, so I really don't have the room to have the menu bars on the top and bottom. If I cant set up awn to run as default, what can I use that has a similar setup? Thanks in advance.:KS

### Post by Therion on 2008-11-21
From the Desktop, Go to System > Preferences > Session.

Select Startup Programs, click "New", and type in the command of your favourite application, AWN in this care. Close the tab.

The application selected in the Startup Program dialog should automatically start the next time you logged in your computer.
