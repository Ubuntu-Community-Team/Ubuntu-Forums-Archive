### Post by ClearBlueSky on 2005-08-02
Not sure if this will make a difference or not, but I noticed that your xorg.conf has one important bit incorrect...

```

Section "Device"
Identifier "ATI Technologies, Inc. Radeon Mobility M300 (M22)"
Driver "ati"
BusID "PCI:1:0:0"
Option "VideoOverlay" "on"
Option "OpenGLOverlay" "off"
Option "UseInternalAGPGART" "no"
EndSection

```

I'm pretty sure Driver should be set to "fglrx" if you were using the HowTo thread you mentioned.

```

Section "Device"
Identifier "ATI Technologies, Inc. Radeon Mobility M300 (M22)"
**Driver "fglrx"**
BusID "PCI:1:0:0"
Option "VideoOverlay" "on"
Option "OpenGLOverlay" "off"
Option "UseInternalAGPGART" "no"
EndSection

```

Hope this helps. Good luck.
