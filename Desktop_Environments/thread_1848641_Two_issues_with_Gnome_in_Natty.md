First issue is that windows no longer raise when I use the move-window command (Super+Button1) without moving the window. I do not want windows to raise when I move them, nor do I want click-to-focus, but I want them to raise when I click on them this way, rather than having to find and click the title bar. Second issue is that when using Alt+Tab to switch windows, sometimes focus is not given to the new window. I fiddled with some Compiz settings on this and thought I fixed it, but just now trying it, failed to remove focus from the window under the pointer. Third: also when using Alt+Tab, no window name is displayed near the icon of any window I want to switch to.
