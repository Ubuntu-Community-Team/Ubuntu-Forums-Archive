### Post by Ms_Angel_D on 2009-05-07
I too was having this issue and came across this thread, so I thought I would post my fix to help others having this issue.

My Problem was caused by the extended preferences plugin I fixed it by entering the plugin options and checking the box for "Allow buddy list to shrink below normal size constraints". Then I restarted pidgin and now my contact list can be resized.

I hope this helps anyone else who might be experiencing this problem.

Angel
