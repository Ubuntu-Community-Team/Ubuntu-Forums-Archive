### Post by gordintoronto on 2012-04-26
It's Ubuntu 10.10, which is no longer supported. You could install Ubuntu 12.04, then install gnome-panel, and select "Gnome Classic" at logon. Then install all the things listed in the text below the youtube video, and configure them. Plan on spending some time reading about how to configure things.

Or just install 12.04, then figure out what you really want from the video.
