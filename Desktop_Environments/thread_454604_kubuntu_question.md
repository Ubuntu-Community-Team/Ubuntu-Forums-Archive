### Post by qpieus on 2007-05-25
Yes you can make a separate kubuntu installation. Create an empty partition on your hard drive and install kubuntu on it. This would be a dual boot setup, ubuntu and kubuntu. You'll need to use a kubuntu installation cd of course for the separate install. Be careful when installing kubuntu that you don't accidentally overwrite the partition that ubuntu is installed on. You can use the same swap file partition as the ubuntu installation uses though.
