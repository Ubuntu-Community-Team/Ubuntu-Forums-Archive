Hey everyone,

I've been trying to figure out how to do this in Ubuntu/Gnome for a couple of days, but it has me stumped.   On my Mac, I used a little utility to add an option to the finder's contextual menu that is called "Go here in terminal..." and will open a terminal window, and go to the folder I've right clicked on.   If I've selected a file, it will go to the folder that contains that file.

Is there a way to do this in Nautilus/Gnome/Ubuntu?

Thnanks!

---

### Post by ninkendo on 2006-08-25
apt-get install nautilus-open-terminal

Then just restart nautilus (or logout and back in)
