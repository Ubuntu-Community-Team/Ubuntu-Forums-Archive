### Post by kinematic on 2006-07-02
right click on the bin file and make it executable and execute it.
it will install to your /home directory and you only have to make a starter fo it with a link to url..../home/sdamron/googleearth/googleearth.
when you click on that google earth will start up.
no need to make a .deb file.
