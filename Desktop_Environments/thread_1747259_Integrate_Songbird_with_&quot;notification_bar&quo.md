### Post by idechix on 2011-10-19
A guy made an addon for Songbird which is really close to what we need to integrate Songbird in the sound menu like Banshee.

You can find it there: [http://logansmyth.com/project/mpris-songbird-extension]("http://logansmyth.com/project/mpris-songbird-extension")

For now it won't show anything in the sound menu if you install it, but it adds MPRIS support in songbird.
The sound menu indicator uses MPRIS, so it should be easy to modify this addon to do what we want.
