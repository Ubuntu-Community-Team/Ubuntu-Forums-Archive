### Post by jaykleg on 2005-08-10
You need to go [here](http://www.memtest86.com/) to get memtest. You create a bootable floppy or CD, then boot the machine from that bootable medium to run memtest. You want to do rigorous testing, maybe overnight.

There are other hardware problems that might cause this behavior, too. But it really does sound like it could be hardware rather than a software issue. Of course bad hardware drivers can cause behavior that is very much like what you see with bad hardware, too.
