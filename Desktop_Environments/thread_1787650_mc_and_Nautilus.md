### Post by 3xp10r3r|X13 on 2011-06-21
well, the only advantage you gain by using mc is that you are able to use the terminal as well, but unfortunately not the auto completion-tab function, because tab is used to change from one side to the other using mc, which is in my opinion a huge disadvantage.
In nautilus it is very difficult to do changes to your OS as a standard user, unless you run it as root.

Conclusion:

Just use the terminal itself - well that's just what I prefer over both.
You gain  complete control and it is quite easy to handle.

I never got the point of mc, because it is just a weaker graphical interface of nautilus. (it runs in the terminal, that might be an advantage)

Just my opinion
