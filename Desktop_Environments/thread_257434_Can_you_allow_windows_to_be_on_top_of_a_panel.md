### Post by ChadMMc on 2006-09-14
Some of the applications are able to go full screen. hitting the F11 key or the 'F' key in some. (depends on the app you are running) will cover the panels. 
Hope this is what you were looking for.

Note: you can also set the panel to 'autohide' mode (panel properties)

### Post by danmeetswood on 2008-06-16
Hey, don't know if anyone still cares. But I went into gconf-editor, searched for "panel," enabled autohide, and set "auto_hide_size" to 0. Worked for me. Or at least that's the effect I was looking for. Hope that helps anyone else.
