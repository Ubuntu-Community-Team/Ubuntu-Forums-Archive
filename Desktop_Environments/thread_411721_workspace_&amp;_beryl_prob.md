### Post by Gallardo Spyder on 2007-04-17
On problem #1.

It sounds like you have the check box under Beryl manager checked - for supports multiple view ports...  Uncheck this box.  I don't have my system in front of me - but under Beryl Manager, and under Cube I believe.

For the question number 2.  Check under extras and see if you have Blur set as active.  Uncheck if you do.

If it is not that - go to desktop options uncheck Vblank on (set it to off) and set your refresh rate to 200.  That should clear it up.

Gspyder
