### Post by milstead on 2005-12-15
I have installed MythTV and am using it on my old CRT TV. I am using an NVidia card so am able to use overscan.

I am looking for a Window Manager to use with this setup. It needs the following features:

1. Ability to fullscreen every window, or even better, nearly full screen every window. This would leave a space round the window so it does not go into the overscan region unless I want it to - like Myth.

2. Keyboard window switching. Using Alt-TAB or ideally a userdefined keystroke.

3. I know it is not really a window manager issue but I would also like a menu that appears with the (shudder) windows key. This menu includes a list of applications and window switching.

I was wondering if anyone had any experience and could recommend the right window manager?

I have tried GNOME (Metacity), KDE (kwin) and IceWM so far with no satisfaction.

Thanks in advance.
