### Post by lunxer on 2009-10-07
I actually solved this myself, but wanted to share the solution if someone else has the same issue.

I run a computer that i vnc into from a windows computer, but i didn't get any feedback in my vncviewer, everything i did showed up om the servers screen. 
Just the initial screen from the state present at the time for login showed up. 

The solution was to check "None" in the visual effects settings (right click anywhere on the desktop and choose "change desktop background > visual effects tab)

I understand that some would like to have these effects enabled, you could try to vnc into an account that has them disabled, and use the other account when them enabled.
