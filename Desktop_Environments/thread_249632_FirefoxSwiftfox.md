### Post by aysiu on 2006-09-02
If you want Firefox, just use Adept to install it or paste this command in the terminal: ```
sudo aptitude update && sudo aptitude install firefox
``` If you want Swiftfox, [this script](http://pykeylogger.sourceforge.net/wiki/index.php/Ubuntu:Chronicles#Install_Swiftfox) can help you install it.
