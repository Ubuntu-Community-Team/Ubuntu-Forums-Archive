### Post by mali2297 on 2008-07-08
To change login sound, go to *Applications -> Settings -> Login Window*.

You can also set a logout sound; see [http://ubuntuforums.org/showthread.php?p=3907604#post3907604](http://ubuntuforums.org/showthread.php?p=3907604#post3907604).

Besides that, there is no support for event sounds in Xubuntu.
