### Post by phico on 2006-03-08
Hello,

I tried XmlSpy with Wine but it is a hack and fails when you change either
Wine or XmlSpy versions ...

I finally found a great XML product for Linux : Eclipse !

Just install Eclipse and the WebTools Eclipse project 
[http://www.eclipse.org/webtools/](http://www.eclipse.org/webtools/)
That gives you :
- a great XML editor (with completion based on the XmlSchema!!)
- a great XmlSchema editor (with a graphical view like in XmlSpy!!)
- a great WSDL editor with a nice design view ...
many other features 

For XLST I suggest you to install the excellent eclipse plugin
Orangevolt EclipseXSLT ([http://sourceforge.net/projects/eclipsexslt](http://sourceforge.net/projects/eclipsexslt))
That includes XSLT processing (with Xalan or Saxon), XPath navigator ...

As that you have a good XML support under Linux and
have no reason to be stuck in Windows because of 
XML editor missing !
