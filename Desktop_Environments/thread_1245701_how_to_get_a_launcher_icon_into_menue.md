### Post by drs305 on 2009-08-20
To add a new menu, right click the main menu icon (the Ubuntu symbol, if you have it) or on Applications in the menu bar:
Edit Menu
New Menu
Enter a name for a new menu. Click on the icon and select a new one if desired.

Now highlight the new menu in the left pane, and in the right select "New Item".
Create a launcher: Add a name, command and select the icon.

You will now have a new menu and new item under Applications.
