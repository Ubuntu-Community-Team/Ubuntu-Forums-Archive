### Post by unutbu on 2008-03-13
What window manager are you using?
Using fvwm2, you can customize a 'run menu' which would pop up when you press a key (say F6). Then you can use arrow keys, a key press, or the mouse to run an arbitrary program.

For example, you could have F6-f run firefox, F6-e run a text editor, etc.

### Post by BluntBox on 2008-03-13

You will find "Show the panel run application dialog" in the Desktop section. Click it and press the key combination you want.
