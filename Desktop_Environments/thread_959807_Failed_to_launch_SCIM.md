### Post by simosx on 2010-10-08
Ubuntu now uses IBus instead of SCIM. So, do not install SCIM but use IBUS, which is already installed in Ubuntu.

Depending on the Indian language you want to use, you can either use the basic Devanagari keyboard layout or Indian with IBus. 
The basic Devanagari layout is found at System/Preferences/Keyboard/Layouts.
