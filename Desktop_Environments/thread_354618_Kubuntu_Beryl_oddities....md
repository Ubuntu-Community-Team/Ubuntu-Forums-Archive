### Post by AusIV4 on 2007-02-06
The problem your having that I noticed is the adept_notifier icon, which is supposed to be in the system tray, shows up as a little window. It almost always shows up as a window for me, and occasionally kmix and the power manager show up the same way. I just close them and, if I really need them, I restart them from command line.

The other problems seem unusual. I might note that I installed Kubuntu from the Kubuntu CD, and don't have gnome installed at all. I am curious as to how you run beryl - do you use XGL or AIGLX? I had a lot of odd quirks when I tried XGL, and found that not only do I not have the quirks with AIGLX, it uses a lot less CPU. I don't know if that's available for your card, but you might look into it.
