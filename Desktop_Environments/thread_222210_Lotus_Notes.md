> Yep, where is it ?

You can download it from the IBM Passport Advantage site.  [http://www-306.ibm.com/software/howtobuy/passportadvantage/](http://www-306.ibm.com/software/howtobuy/passportadvantage/)

You will however need a valid Passport Advantage account, i.e. be an administrator of a licensed Lotus Notes Customer Site.  

If you work for a company that uses Domino/Notes your Sysadmin should be able to get you a copy of the software, either from Passport Adv, or from the local IBM partner/reseller.

Alternatively, keep a lookout in the Trials section.  [http://www-128.ibm.com/developerworks/downloads/ls/lsndad/?S_TACT=105AGX28&S_CMP=TRIALS](http://www-128.ibm.com/developerworks/downloads/ls/lsndad/?S_TACT=105AGX28&S_CMP=TRIALS) 

Usually when they release a new version (current version is 7.0.1) every 6 months or so, they make a trial of the previous version available for free download.  At the moment is trial is for version 7.0.0, which is only Windows  client - perhaps in a few months they'll make the 7.0.1 client available, which should then have the Linux version as well?
