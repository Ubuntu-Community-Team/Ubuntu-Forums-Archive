 [Macbuntu ]("http://sourceforge.net/projects/macbuntu/")-   Mac OS X Transformation Pack. Similar to Mac4Lin but dedicated to   Ubuntu Linux. It can also be used in other OS based on Debian/GTK.   Package includes all necessary system components to mimic Mac OS X   appearance on Ubuntu Linux.

- Not compatible with Ubuntu Netbook Edition
