### Post by Peter Great on 2008-06-14
Mercury Messenger ([http://www.mercury.im/](http://www.mercury.im/)) has perfect offline message support. You can send messages when both parties are offline. It even experimentally supports cross-talking between MSN and Yahoo Messenger. The only thing you might not like about it is that it's written in JAVA.

However, currently I can't find any MSN client for Linux that supports voice chat.
