### Post by Frogs Hair on 2011-08-13
I'm not clear if you were able to get Unity running or not .

Unity Reset

Restart with the button on the computer , when the login screen comes up select your user name and enter the password . 

Before login , select the recovery console from the session list below the greeter box on the bottom panel. When the terminal appears.

Enter 
```
unity --reset
```

Restart the computer . When login screen comes up again select Ubuntu from the session list .

See the link for enabling the cube , I don't know about wobbly windows .

[http://ubuntu4beginners.blogspot.com/2011/05/enable-desktop-cube-in-unity-ubuntu.html](http://ubuntu4beginners.blogspot.com/2011/05/enable-desktop-cube-in-unity-ubuntu.html)
