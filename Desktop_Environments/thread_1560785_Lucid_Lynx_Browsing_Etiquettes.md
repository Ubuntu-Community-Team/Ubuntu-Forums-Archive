Just install the updates when available to make sure your browser is up-to-date.

Use AdBlock if you get annoyed by ads on web pages. Disable cookies from 3rd party sites and/or use NoScript if you are on the paranoid side. You'll be fine with the default install anyway.

Seriously, I even stopped even using a firewall years ago. :D
