### Post by thelinux_guy on 2007-08-02
Oh! well this should be easy :)

I recommend Avant Window Navigator. It can be used as an application launcher and a taskbar-replacement ( It is not included in the default Ubuntu repositories meaning you would have to enable some unsupported ones, and  requires Beryl or Compiz to function properly )

Also there is Kiba-dock, which has silly bouncing icons that jump all over the screen and then return to the dock at  the bottom ( where its default location is ) It can be used as an application launcher and a taskbar-replacement. This one also requires Beryl or Compiz

There is Kooldock with is just like the Mac OSX dock when it comes to looks and function, it is in the default Ubuntu repositories and is easy to configure ( however, if you have Beryl or Compiz installed, Its animations can appear rough when It tries to animate the zooming icons )

There is KXdocker which I hear is nice and cool. But It threw an error in my face when I tried to launch it ( after I installed it, of course ) so I don't mess with it. It also is included in the default Ubuntu repositories

Good luck with finding the right one for your needs! :)
