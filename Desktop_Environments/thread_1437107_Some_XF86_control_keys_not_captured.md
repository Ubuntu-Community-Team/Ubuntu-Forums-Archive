### Post by Arancaytar on 2010-03-23
I am using a Microsoft Digital Media Keyboard 1.0A, which includes several XF86 control keys: XF86HomePage, XF86Mail, XF86Favorites, XF86Messenger, XF86Calculator, XF86Sleep and XF86LogOff.

Most of these work, but about four just don't respond: XF86Messenger, XF86LogOff, XF86ZoomIn and XF86ZoomOut. When I attempt to assign these keys to functions in gnome-keybinding-properties, the application does not appear to notice the key press. The rest appear to work fine.

Is this something that can be fixed with a different keyboard driver, a different keymap, or not at all?

---

### Post by ninja togo on 2010-03-23
This has also happened to me, my fix was to assign the commands to other keys instead. In my case the Fn+F9 (Mute) Combo did not respond so I set it to Fn+M and problem solved.
