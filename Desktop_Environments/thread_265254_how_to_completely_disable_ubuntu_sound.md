### Post by Sarisar on 2006-09-25
Prefs -> Sound
'Enable software sound mixing' - make sure this is not 'checked'.  This will disable all the ubuntu sounds.  But nothing else which confuses me.  I have that disabled and can get sound in flash games, amarok, rhytmbox, xmms, totem player and everything else basically!
