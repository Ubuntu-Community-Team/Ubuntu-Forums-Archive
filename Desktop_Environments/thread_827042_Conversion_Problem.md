It would be better to present using OpenOffice.org rather than Powerpoint.  But, if you have no choice, here are some tips to ease conversion:
[LIST]
[*]Use a font that you are sure is the same on both systems, e.g. Verdana.
[*]Use a very, very simple slide layout with no fancy crap.
[*]Place the boxes in such a way that small shifts in their position will not mess up the presentation.
[/LIST]
