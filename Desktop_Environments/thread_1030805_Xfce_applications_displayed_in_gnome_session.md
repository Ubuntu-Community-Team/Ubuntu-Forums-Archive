### Post by mali2297 on 2009-01-05
The available applications are automatically detected and displayed. The system makes no difference between 'xfce' and 'gnome' applications. If you want to hide certain apps, I think you can do that in gnome through Applications -> Accessories -> Alacarte Menu Editor. Unfortunately, there is no easy way to do the corresponding thing in xfce.
