### Post by Shwefty on 2008-12-08
First off, make sure after you downloaded the .iso image you [check the md5sum]("https://help.ubuntu.com/community/HowToMD5SUM").

Second off, make sure you burn the .iso image at the slowest speed possible.

Third off, make sure you check your cd for defects after booting to it.

If it's a problem past that, somebody else will have to help you! :)

### Post by Shwefty on 2008-12-08
