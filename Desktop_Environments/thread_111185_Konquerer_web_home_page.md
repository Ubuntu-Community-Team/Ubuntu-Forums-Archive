### Post by vayu on 2006-01-01
I can't find how to set a home page in Konquerer.  How do I get rid of the intro screen?   In configure I can only change the file manager home.  I would like to create a link on the panel that has one icon that goes to the file manager home and another to go to the web browser home.

edit:

I found how to go to a web home page when I click the icon in the taskbar instead of the intro screen.  Go to the web page you want as your homepage, go to settings/save view profile.  There will be a button to "Save urls in your profile".  After that Clicking the Web browser link in the taskbar will go to that page.
