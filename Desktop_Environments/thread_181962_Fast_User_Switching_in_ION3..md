### Post by lukanova on 2008-03-29
you can exit ion3 by using session/exit command in main menu.
so:
12 (for main menu, appears at the bottom of current frame/window)
type "session/exit" and press enter.
that's it.
if you want it to save your windows layout, session/save

that's not really fast user switching, but your gdm should now ask you for login afaik.
