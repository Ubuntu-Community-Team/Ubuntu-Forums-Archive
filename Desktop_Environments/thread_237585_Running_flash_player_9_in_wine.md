### Post by patrickfromspain on 2006-08-16
The easiest way to get flash 9 in linux is the one I've done. From my windows install, I've copied over the mozilla firefox install (it already had flash and so on) and then run firefox.exe with wine... it works. So, I've renamed the folder to firefox_windows, moved to my home and createad a shortcut called Windows Firefox, and the app is wine /path/to/windows_firefox/firefox.exe

and it works just fine!!

EDIT: you also, of course, download the windows installer and run it with wine. After it exits, you'll firefox installed and a shortcut in the desktop "Mozilla firefox". Of course, with no plugins or themes. For flash, go to a site that requires flash and it will do the same as in windows or standard linux firefox: install missing plugins. Themes and extensions install the same way as you already know.
