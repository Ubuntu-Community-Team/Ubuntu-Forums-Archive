### Post by jimbob on 2006-08-31
If you have a stand alone CD you can boot (like Knoppix) you can mount your drive you installed Ubuntu on and look at the /etc/passwd file to find the name you put in.  

If you haven't entered any other users it will be the last entry in the list with a UID=1000.

Actually I think there is a way to do this from the Ubuntu install CD also.
