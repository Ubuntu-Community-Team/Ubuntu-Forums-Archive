### Post by benerivo on 2007-09-10
You could try opening the Control Centre, and going to System Administration > Login Manager. On the Convenience tab there is a 'preselect user' option. There are various other login options that may be of some interest to you as well.

---

### Post by Z13 on 2007-09-10
Thanks a lot!

It worked like this: I set "the select none user" option from Login Manager (System Settings). I'm new in KDE :P you're tip really helped me. Thanks once again.
