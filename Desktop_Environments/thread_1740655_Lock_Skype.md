### Post by 3Miro on 2011-04-28
Try locking your entire computer. The screensaver has an option to ask for password to deactivate.

On 10.10 and 11.04 Classic mode, you can install the "lock screen" applet to the Gnome-panel (right-click on the panel and add a lock screen applet). This locks the entire machine, you will still receive messages, but others will not be able to read them.
