### Post by HanZo on 2006-08-27
why not just try it? Kororaa offers a free live cd to try XGL/compiz... it's not the latest version but still shows all it can do: [http://kororaa.org/](http://kororaa.org/)
anyway... some nice things are: you can set inactive windows to become darker so you always know which one is active, alt+tab shows thumbs of the windows you are cycling through, you can have some useless stuff like rain on the desktop or a watery mousetrail... anyway you cannot fully understand how cool it all is until you try it out yourself!

### Post by HanZo on 2006-08-27
> 
well It's just eyecandy as far as I know... There is no real addtition to basic functionality.

Not for the basic one maybe... but most things make working with the dektop easier... just for an example: the desktop cube gives you a spacial feeling about the whole desktop metaphor that changes how you percieve it... it makes it a lot less "abstract". the minimize function (basically expose from mac os) is really practical when you have lots of windows.
of course transparent window borders and wobbling stuff is not really a productivity enhancement... but it can be cool, and we also want to have fun with them buzzing boxes, don't we?
