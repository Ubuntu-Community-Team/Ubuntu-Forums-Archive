I'll stand corrected on this, but I believe that you can change the default icon size in[COLOR="Blue"] configuration-editor[/COLOR] .

When the window opens, navigate to[COLOR="Blue"] apps>nautilus>icon view[/COLOR]

Right click on [COLOR="Blue"]default zoom level[/COLOR]

Choose [COLOR="Blue"]edit key[/COLOR]

This is the crunch. I'm not sure of the correct list of options that you can now type in. In your case, maybe small?
