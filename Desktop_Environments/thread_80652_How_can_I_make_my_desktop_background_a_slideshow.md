### Post by poptones on 2005-10-22
Copy the script [on this page](http://www.ubuntuforums.org/showthread.php?t=49336&page=6&highlight=wallpaper) into a file called .randomwallpaper and put it into  a folder called "wallpapers"  Put your wallpapers in tha folder (or you can just put links to files in other folders to save space).

You can start it with your startup options or you can comment out the last two lines and make a toolbar launcher (which is how I use it)
