### Post by rosslaird on 2005-11-19
Yes, both Epiphany and Opera are faster than FF. Firefox has the extensions and so on, so it is more versatile and modular (especially with things like the web developer extension), but Epiphany fits beautifully into the gnome interface (as does Konqueror, for kde, which is also much faster than FF and renders pages more compliantly), and Opera is virtually a whole software suite (email, news, chat, browsing, blogging, etc.).

Firefox has the buzz, but E and O have the edge, in my opinion.
