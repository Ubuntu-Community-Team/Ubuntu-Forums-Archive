### Post by dbott67 on 2005-10-14
[quote=Kirky_D]I don't know if it's configurable, but you can simply press ctrl+l while browsing files to get it to appear[/quote] 
Yes, you can change it in the Configuration Editor:

1. Open Configuration Editor (**Applications --> System Tools --> Configuration Editor**)
2. Browse to **APPS --> NAUTILUS --> PREFERENCES** and place a checkmark in **ALWAYS_USE_LOCATION_ENTRY**.

-Dave
