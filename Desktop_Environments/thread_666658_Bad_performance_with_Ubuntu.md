### Post by TeaSwigger on 2008-01-13
Weird. Anything could help narrow it down. It's good you posted the xorg.conf. I don't have a graphics card like that and I don't know enough about xorg.conf to advise if that's correct or not, but hopefully someone who does will chime in.

It looks like you'll have to look at HTop and see what's consuming the cpu. If it were me, I suppose I'd be suspecting a graphics or sound card driver issue at the root of it. Unfortunately I know near zilch about sound card drivers and all that. But if you're playing audio and that's when it flares up... Another thing; have you tried using only one monitor and seeing if it performs right then? 

There's a little utility to help with the xorg.conf / graphics config, called xorg-edit. [http://ubuntuforums.org/showthread.php?t=156243&highlight=xorgedit](http://ubuntuforums.org/showthread.php?t=156243&highlight=xorgedit)

 I suppose you could back that file up and try, as it says:

and see how it goes. 

Another thing you can try... in a terminal enter 

```
glxgears
```

and see if the little graphic that pops up of two spinning gears moves in a steady, smooth manner. That little test can uncover some graphics issues when it seems that everything's setup ok under normal use. 

Good luck to ya. It's a great system, I hope you can get it working right in your setup.

[https://a248.e.akamai.net/f/674/9206/0/www2.ati.com/drivers/linux/ati-driver](https://a248.e.akamai.net/f/674/9206/0/www2.ati.com/drivers/linux/ati-driver)
-in
staller-8.42.3-x86.x86_64.run

After you download it,
Execute it by:

sudo chmod +x ati-driver-installer-8.42.3-x86.x86_64.run
sudo ./ati-driver-installer-8.42.3-x86.x86_64.run

Go through the install options, don't worry about generating a package
whenever it asks you, just do a default install.

Before you reboot, and after you install the driver,

sudo aticonfig --initial=dual-head --dtop=horizontal \
--screen-layout=right

This will set your primary screen to the right. change screen layout to
left if you want you want the secondary monitor to the left. Also, if
you have your monitors stacked, change the horizontal option to
vertical. After you install the driver, and issue the command, restart
your box and it should be enabled.
