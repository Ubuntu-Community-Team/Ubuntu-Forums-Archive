### Post by em3raldxiii on 2006-08-21
Most Windows-oriented software will require "installation" under WINE into it's virtual environment. However, many W98 programs are simply .exe files (without any .dlls or other crap) so they will run happily from the Windows partition with no other input from you.
 
The more recent the program, the more likely it will require "installation" under the virtual environment. :D
