### Post by crazy bird on 2011-11-22
Go to this webpage: [http://www.webgapps.org/add-ons/flash-aid](http://www.webgapps.org/add-ons/flash-aid) and download the flash-aid add=on for Firefox.
 
From the website:
*Flash-Aid is an extension designed to remove conflicting flash plugins from Ubuntu Linux systems and install the appropriate version according to system architecture.*
 
You can try this.
