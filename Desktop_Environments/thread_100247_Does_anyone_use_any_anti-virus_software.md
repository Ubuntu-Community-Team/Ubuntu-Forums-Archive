### Post by A-star on 2005-12-07
Actually you don't really need virus protection on linux since there are no real viruses for linux.
But you can install clamav and scan your pc once in a while.
Another good tool is rkhunter, this scans for rootkits which can be installed on your system if you don't make it secure enough (ubuntu is very secure by default unless you enable the root account).

I'm sure other people here will have some other suggestions.
