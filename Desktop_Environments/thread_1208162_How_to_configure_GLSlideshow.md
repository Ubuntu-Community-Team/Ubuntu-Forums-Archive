### Post by tomreid on 2009-07-08
Hi

Does anyone have any easier ways to configure GLS Slideshow to show a random selection of your own images?

I understand the "Advanced" option was taken away in 9.04, so you have to try this manual work around, but it did not work for me, but I am not very good at this more advanced stuff in Linux.

I probably messed up the code.

I made a file in the text editor, called it .xscreensaver and put it in my home folder. The code I put in the file was: 
[FONT=monospace]
imageDirectory:/home/tom/Pictures/Photo Library chooseRandomImages: True
imageDirectory:/path/to/your/pictures
chooseRandomImages:   True

The GLSlideshow only shows me some basic default images now.

cheers
[/FONT]
