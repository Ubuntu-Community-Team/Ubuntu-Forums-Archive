### Post by FoXoF on 2009-12-12
Every time I try to use gmail firefox has zoomed out completely. I have to use ctrl+0 to reset the zoom and be able to read any of the text. Each time the page updates everything is zoomed out again. This is really frustrating for me.

I've tried uninstalling and reinstalling firefox but with no success. Any other ideas on how I can solve this problem?


Edit: SOLVED

To fix put 
> about:config
in the address bar and go there.

Then set browser.zoom.siteSpecific to false and the problem is fixed.
