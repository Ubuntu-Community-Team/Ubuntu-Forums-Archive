### Post by fahimdadream on 2008-05-02
Hey, 

There is a couple of things you can try. First off under system there should be an appearance button (not on Ubuntu ATM).
On the last tab in that menu there should be a way to start the desktop effects, set them to extra. If that works then install the advanced desktop configuration (check forums or google). If it says, "cannot enable". Then you may have to install some restricted drivers. 
Note: If the latter is the scenario, check the forums for your video card before you do, some people are getting black screens after installing restricted drivers. 

Good Luck.
