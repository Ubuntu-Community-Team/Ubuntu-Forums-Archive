### Post by grumpymole on 2005-12-07
Some applications add an entry to the "Applications" menu - these are usually programs from the Main repository.

Programs from other repositories need to be started from the commandline and you can add a launcher to launch them from the desktop.  To find the name of the program to start, have a look at the applications homepage, or it is usually the application name in lower case.  Fro example, mozilla-thunderbird.

### Post by aysiu on 2005-12-07
> **j&#248 said:**
> The problem is that after i install software, i cant find it. My point is that you don't have to find it. Let's say, for example, that you installed *juk*. If you hit Alt-F2 and type in ```
juk
``` Juk will launch. You don't need to know where it is.

If you want a launcher for it, right-click on the panel and select **Add to Panel** and **Application Launcher**. If you want it in the menus, go to Applications > System Tools > Applications Menu Editor and add it with the command being the name of the program (see example above with Juk).
