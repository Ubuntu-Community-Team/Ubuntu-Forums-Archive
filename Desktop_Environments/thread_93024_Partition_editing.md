[QUOTE=billpoo90]
Just wondered if it was possible to alter the sizes of my HD partitions to give more to ubuntu without reformatting or losing any data.[/QUOTE] Theoretically, yes, if you follow [these instructions](http://users.bigpond.net.au/hermanzone/). Practically, you're an idiot if you don't back up all your work before installing a new operating system. 

Theoretically, I can drive a motorcycle for an hour and not get into an accident that splits my head open. Practically, I'd be an idiot if I didn't wear a helmet.

**Edit**: Sorry. I re-read your post. You already have Ubuntu installed. What you would do is defragment Windows and use a live CD (something like Knoppix) and run a program called QTParted to resize the partitions (make sure they're unmounted first!). The part about backing up your work still holds, though.
