### Post by Thund3rstruck on 2006-07-07
absolutly. The XP install is the exact same thing as running a seperate standalone XP machine. However, you should only use XP as a crutch of last resort so if you're using it rarely you should be fine. If you're worried about the malware then ensure that you have not configured networking between the host and guest and do all your important work in your host

### Post by slider on 2006-07-07
Depends - 

If you are sitting behind a router and aren't planning on doing much websurfing from XP, then you can probably afford to be a bit lax. If you are direct to a dsl modem and will be doing a lot of surfing then keep it up to date religiously. I'm behind a router and don't surf from XP. I have the vmware image setup exactly the way I want and keep all my data on samba shares on the host. I took a snapshot, keep it suspended when not in use and just roll back from time to time.
