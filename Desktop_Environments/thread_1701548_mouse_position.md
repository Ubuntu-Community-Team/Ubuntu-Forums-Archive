i would like to find some application that will launch a program or script when the cursor is placed in a certain position (i.e. the corners of the screen). is there anything out there that will do this?

---

### Post by Krytarik on 2011-03-06
You could use Compiz' "Edge Bindings" or "Button Bindings" for that:
- go to "System -> Preferences -> CompizConfig Settings Manager -> Commands"
- add the desired command at the first tab, "Commands"
- assign the desired binding to it at the respective "...Bindings" tab
