### Post by mcduck on 2006-03-23
MPD can be controlled with many different interfaces, including some web based ones. Take a look at this: [http://mpd.wikicities.com/wiki/Main_Page]("http://mpd.wikicities.com/wiki/Main_Page")

PlayGUI is java-based MPD client, and it seems to run on windows too: [http://sccs.swarthmore.edu/~cbr/PlayGUI/]("http://sccs.swarthmore.edu/~cbr/PlayGUI/")
