### Post by bgs100 on 2008-11-11
Try hitting alt+F2 (bringing up the run app box) and type in ```
gnome-panel
``` then click "Run"

---

### Post by Warner White on 2008-11-12
Thank you! That worked. Turned out I had somehow uninstalled the gnome-panel and trying to run it revealed that fact and told me how to re-install it.:)
