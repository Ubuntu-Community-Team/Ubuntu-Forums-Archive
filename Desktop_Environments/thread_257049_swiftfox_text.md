The problem is that Swiftfox disables pango to make itself start faster. Pango is a library for rendering text. After it starts there is no speed difference between it and Firefox that a human can tell.
Another thing you may not know it Swiftfox is distributed under a non free (as in freedom) license. I dont recommend anyone who uses free software use or support it.
