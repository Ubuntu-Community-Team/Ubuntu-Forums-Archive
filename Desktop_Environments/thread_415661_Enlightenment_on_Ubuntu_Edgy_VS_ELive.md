### Post by worldwithoutgurus on 2007-04-21
Elive is a very particular and capable distro. I use it and I love It.

But I run Ubuntu + E17 (cvs) on another partition. Elive and Ubuntu don't mix... and it's OK.
They are different distros, serving different purposes.

My advice: Don't mess with Elive's recommended repositories, or you'll get into trouble. 

Elive use an older __ and heavily hacked (Elpanel, etc)__  version of E17. The way Engage is conceived is no longer relevant (it doesn't
conform to Freedesktop standards...). Technically speaking, Engage is dead, or it's a dead-end if you prefer.

Sure, it takes some time to become familiar with the shelf, but it's worh it. You can do pretty cool effects with two or three shelves.
Try it for yourself.

### Post by worldwithoutgurus on 2007-04-25
1) You need Exalt Network Manager:

cvs  z3  -d:pserver:anonymous@cvs.exalt.berlios.de:/cvsroot/exalt co exalt

Then:

  cd ~/exalt/libexalt
  ./configure
  make
  sudo make install
  sudo ldconfig     
      
  cd ~/exalt/exalt
  ./configure
  make
  sudo make install

Exalt details: [http://forum.ubuntu-fr.org/viewtopic.php?id=108951](http://forum.ubuntu-fr.org/viewtopic.php?id=108951)

2) This is a known issue. Don't know if it is already fixed in cvs...

Hack:

Edit the file /home/<your_user>/.e/e/modules/bling/module.desktop
And make it look like this:

[Desktop Entry]
Type=Link
Name=Bling
#Exec=enlightenment
#StartupNotify=false
X-Enlightenment-WaitExit=false
X-Enlightenment-IconClass=bling,gadget
#Icon=./module.edj
Icon=module

3) No 'follower' in E17 cvs...

### Post by darkmaster on 2007-05-07
> **loell said:**

I can help you on this, I had a similar problem. Open a console and:

```
enlightenment_remote -theme-list
```

To see all of the themes available. Then:

```
enlightenment_remote -theme-set OPT1 OPT2 
```

Wich sets theme category (OPT1) and edje file (OPT2).
There's to say that a lot of wonderfull themes that once worked for E17 now doesn't work anymore with the cvs version, it's too advanced and something must have changed drastically. Sadly, these themes, such as the wonderful nightly blind, are discontinued, so, there won't be an update to the latest E17 version :(

So I had the very same problem and had to go back to a working theme. To find a comprehensive list of things you have to know about E17, read the manual:

[http://www4.get-e.org/E17_User_Guide/English/](http://www4.get-e.org/E17_User_Guide/English/)

Believe me, do it. You'll learn a lot and will enjoy E17 sooooo much more :)
To have a lot of new themes, visit this site (that is the same of the user guide)

[http://www4.get-e.org/Themes/E17/](http://www4.get-e.org/Themes/E17/)

And remember: there's a note indicating that the last themes listed DOES NOT work with E17 CVS, so, if you'll try to install those themes, you'll get stuck with a not working theme exactly like now :) Try to avoid it.

Byeeee!
