### Post by sakisds on 2010-05-21
You can have both Gnome and KDE installed at the same time, choosing which one you want before you log-in.

You can install gnome with the following command:
```
sudo apt-get install ubuntu-desktop
```

If you install Gnome, you might as well want to install [Gnome Menu extended]("http://gtk-apps.org/content/show.php/Gnome+Menu+Extended+(Debian+Package)?content=73515"), it will separate Gnome applications with the KDE ones in the gnome menu.

Also, Gnome won't mess with your KDE settings, it will only change its own ones.
