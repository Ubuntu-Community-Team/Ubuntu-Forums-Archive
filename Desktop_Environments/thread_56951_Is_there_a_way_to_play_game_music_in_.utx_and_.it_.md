### Post by GameGod on 2005-08-14
Try installing the "xmms-modplug" package trough Kynaptic or Synaptic. :)
(Not only does it install a plugin that'll play at least .it files (probably .utx if modplug plays them because it's based on modplug's code), but also a stand-alone player... try running "modplugplay" after you install it)

Good luck. :)
