### Post by badrinarayan on 2005-07-14
Ubuntu uses repositories which have about 10000 applications which you can install through a single application called **Synaptic Package Manager** (find it **System > Administration** menu). Read [http://www.ubuntuguide.org](http://www.ubuntuguide.org) for more

No, windows applications, don't normally run. You may install wine to run some of them. Also check out the commercial application sections in the above site. You may also run a virtual machine software like [vmware](http://www.tuxme.com/node/536)
