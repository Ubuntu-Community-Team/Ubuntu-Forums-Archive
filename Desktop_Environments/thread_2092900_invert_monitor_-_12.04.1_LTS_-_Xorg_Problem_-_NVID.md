### Post by para-banana on 2012-12-09
Hi all,

I have a workstation connected to four monitors via the NVIDIA Quadro NVS 450 running  "NVIDIA accelerated graphics driver (post-release updates ) (version current-updates)".
One of the monitors should be rotated by 180° (inverted). In  Ubuntu 11.10 Oneric Ocelot I used the NVIDIA X Server settings to generate a Xorg.conf template, which I edited to invert one of the monitors by adding the option "Rotate" "inverted" to the screen section of the corresponding monitor. This worked quite well, but after upgrading to 12.04.1 LTS this does not work anymore.
All four monitors work but I cannot invert this one monitor.
I also tried to reinstall the system (12.04 LTS) without upgrading, but I am still not able to invert the monitor.
(By the way, the randr extension is not compatible with the NVIDIA driver and has to be removed in order to run it).

Does someone have a solution?

Thanks a lot!
Best!
