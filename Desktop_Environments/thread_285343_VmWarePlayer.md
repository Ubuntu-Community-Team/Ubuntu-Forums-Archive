### Post by WoodyMahan on 2006-10-26
What Ive done is install Windows and then download vmware server from their website.  This allowed me to partition a portion of my hard drive to run the virtual server and I was able to install ubuntu to the virtual server in Windows.  You can also do this vice versa and load the vmware server onto the linux machine and then install windows on that virtual server.  This is probably the better option but I haven't setup a virtual environment in Ubuntu so I can't really help that way, but my virtual install of Ubuntu runs great, there are some limitations (there is some eye candy I can't use because there is no 3D graphics acceleration) but it is minimal.
