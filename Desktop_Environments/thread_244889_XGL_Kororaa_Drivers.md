### Post by 3rdalbum on 2006-08-27
Nup, sorry. It's not possible because the drivers are built against the precise kernel version, and the kernel in Kororaa is not the same as the one Ubuntu uses.

It's not too difficult to get 3D acceleration going on Ubuntu as long as you have a HOWTO guide. Apart from anything else, it'll be good command-line practice :-)

By the way, make sure you don't give any Kororaa CDs to anyone else - the ones with XGL included were actually illegal to distribute in the first place
