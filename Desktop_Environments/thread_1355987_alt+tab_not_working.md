### Post by qrees on 2009-12-19
Ok, since no one want's to help, I 've found the solution my self:

In gnome-control-center choose:
```
Skróty klawiszowe -> Zarz&#261;dzanie oknem -> Przechodzenie pomi&#281;dzy oknami z u&#380;yciem okna podr&#281;cznego (you want english version, translate this from Polish).
```
And reboot. This is quite strange, because changing other binding didn't require to reboot, but this one does.
