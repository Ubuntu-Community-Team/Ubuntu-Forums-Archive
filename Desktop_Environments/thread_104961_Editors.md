### Post by shakin on 2005-12-17
I use Quanta+ for PHP. It doesn't have the same code completion features as Zend, but it's a nicer overall IDE. I love the file picker that gets out of my way when I'm editing. The current version of Quanta has a bug where when you add a new file to your project it thinks it's been modified outside of Quanta and asks you what to do every time you save it. Restarting Quanta fixes it, or you can turn off the feature that detects if the file has been changed.

KDevelop is my second choice. It's primarily a C++ editor and it's very good at that. I also used to use it for PHP. It's got more feaures than Quanta, but is still not quite as comfortable as for me.

Eclipse is ok, but it's still large and slow when compared to Quanta or KDevelop. I also have used Zend Studio and find it's just too bulky, like Eclipse. Zend's code completion is very good and its debugging is outstanding, but I never really debugged that way enough to justify staying with a clumsy editor. With any luck, Zend's partnership with Eclipse will help a lot.
