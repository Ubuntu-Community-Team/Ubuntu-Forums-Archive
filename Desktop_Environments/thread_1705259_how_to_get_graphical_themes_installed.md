### Post by Copper Bezel on 2011-03-11
It's not the theme, it's the Visual Effects settings. Run

```
sudo apt-get install ccsm compiz-fusion-plugins-extra
```

Now, go to Preferences -> Compiz Config Settings Manager. Scroll down the tiles to "Animations" and "Animations Add-On". Check the boxes next to them, then click the Animations tile to jump to its configuration screen. You can set any of the available plugins to actions like open, close, and minimize according to window classes, like normal windows, menus, and dialog boxes. You can also tweak the settings of each of the animations to get the visual effects you want.
