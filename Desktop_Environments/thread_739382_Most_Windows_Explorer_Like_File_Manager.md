### Post by MattBD on 2008-03-29
Dolphin (the new file manager in KDE) might be ideal for you. It's a lot simpler than Konqueror, so you may find it better than that.

Several others worth considering are Thunar (similar to Nautilus but lighter), Krusader (a twin-panel file manager) and PCManFM (it's got tabs similar to Konqueror but is otherwise fairly standard).

Believe me, there's plenty of choice over file managers - just try the following:
```
apt-cache search "file manager"
```
You'll see plenty of applications! Just try a load of them till you find one you like.
