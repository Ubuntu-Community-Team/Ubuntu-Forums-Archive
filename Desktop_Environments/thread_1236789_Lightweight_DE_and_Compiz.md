### Post by earthpigg on 2009-09-03
compiz works *just fine* with normal default unmodified LXDE. :D

install LXDE, if this is not done already.

install the hardware drivers for acceleration if this has not been done already. (easiest way is to use the jockey-gtk package - this program is what is normally found in system -> administration -> hardware drivers. in lxde, it will be in menu -> system tools -> hardware drivers.)

install the compiz packages, if not already done. 

then either do alt+f2 and "compiz --replace" or have that command run at startup.

to disable compiz, it is "openbox --replace"

ill go ahead and post a screenshot to put doubts to rest.
