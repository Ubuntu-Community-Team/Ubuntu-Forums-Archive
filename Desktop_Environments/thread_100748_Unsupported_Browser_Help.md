### Post by flaming_monkey on 2005-12-08
Some misguided web developers use user agent scripts to detect which browser you're using and then decide what to do with you, more often than not that means blocking anything that isn't IE 5+. But you can fight back.

Firefox has an extension (plug-in) that allows you to masquerade as different browsers, it's appropriately named [User agent Switcher](https://addons.mozilla.org/extensions/moreinfo.php?id=59&application=firefox)
