### Post by sylentdogg on 2009-12-04
Moblock is available through Launchpad.net [https://launchpad.net/~jre-phoenix/+archive/ppa]("https://launchpad.net/~jre-phoenix/+archive/ppa").
Just add the ppa repository in software sources under the other software tab and it will be available in synaptic. If you want more info on Moblock check out [http://moblock-deb.sourceforge.net/]("http://moblock-deb.sourceforge.net/").
