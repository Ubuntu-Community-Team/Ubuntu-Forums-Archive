### Post by Copper Bezel on 2011-10-27
No, it's Gnome's own interface shell for Gnome 3. Install it, log out, and choose it in the cog menu after selecting your name at the login screen. It'll actually install two interface shells, Shell and Fallback, either one of which might turn out to be more to your liking.
