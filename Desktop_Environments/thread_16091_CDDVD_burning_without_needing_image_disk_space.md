### Post by cwaldbieser on 2005-02-19
xcdroast is a great GUI front end for CD burning.  It lets you burn CDs on the fly without having to creat an ISO image on your hard disk first, and I have always been impressed that it has been able to use the max speed of my burner without ruining the CD (unlike when I used to use Nero for Windows many years ago).

xcdroast also had some options that let you select hidden files a little easier than k3b, but I havent' been running k3b in a while, so maybe they got that wrinkle ironed out by now.

The only thing negative that I have to say about xcdroast is that the interface can be a little confusing at first, especially if you have come from Windows tools like Roxy or Nero (k3b seems to have its interface organized more like those than xcdroast).

Hope that helps!
