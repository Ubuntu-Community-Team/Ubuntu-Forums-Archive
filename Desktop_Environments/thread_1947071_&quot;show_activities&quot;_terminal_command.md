### Post by markbl on 2012-03-26
According to [http://live.gnome.org/GnomeShell/CheatSheet](http://live.gnome.org/GnomeShell/CheatSheet) ALT+F1 should also open the activities overview but I have noticed that this shortcut (and the ALT+F2 run dialog) get disabled whenever you start Unity. So if ALT+F1 does not currently work for you then just restore that shortcut in the System Settings/Keyboard/Shortcuts/System settings.
