### Post by jespdj on 2007-04-26
There are several options to make Windows applications work under Linux.

There is WINE, a Windows emulator. This doesn't work for all Windows applications. Adobe CS will probably not work under WINE.

You could also use virtualization, which is a way to create a "virtual machine" on your computer in which you can boot up Windows, and then you can run all your Windows software in there.

I haven't tried those things, so I can't give you more info about it, but I hope these are good leads for you to search for more info.
