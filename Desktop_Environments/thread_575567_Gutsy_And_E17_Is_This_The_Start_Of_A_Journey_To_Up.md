### Post by kelvin spratt on 2007-10-14
Over the weekend I installed Gutsy it was a total disaster and kept freezing, The screen resolution could not be altered. Being a person that does not take defeat as a option I decided to fix it the simple way. I used a Base Feisty system with E17  this is minimal every thing is command line + Synaptic. So I locked the xorg drivers. Then did a full upgrade to Gutsy over 1,000 packages were upgraded and 25 new packages were also installed the result was somewhat amazing. I now have a setup that rivals my Arch setup for speed and much faster than my Elive setup. The bonus is my Cannon Scanner, now works a first in Ubuntu, My external nfts USB hardrives now read + write by default another first in Ubuntu. No more Automatix needed for that. Cinelerra installed like a dream, As did Devede with no sound problems, The sound system even sounds better. E17 has some faults but it is stable the weather module does not display all the others do. The active wallpaper works fine as do all the themes, Font  themes work as do system themes with GTK theme switcher. Its not Utopia but this is one big step in the right direction if the Xorg driver issue is sorted this could be the best Ubuntu on my standard desktop, As Rav Tux said the journey continues.

### Post by kelvin spratt on 2007-10-31
Its 2 weeks since I set up E17 on Gutsy its not all been good after 2 days mencoder upgraded, Then Devede 3.2 started playing up so I downgraded mencoder that was sorted, Avidemux kept crashing at the same time with mov files, thats a to do?. The next major problem was swap or lack of this took a bit to sort out, As the set up script showed it as activated on boot  fstab also showed the entries as ok. I finally tried some script from Arch Linux pasted in  etc/rc local and that was sorted, swap now working again  thank you  Arch Linux, the fastest distro on the planet. Still using Feisty video drivers have not played with that yet. The positive side is very rosy gone is the usb blues, External hardrives mount every time at boot up. Gutsy is so much faster than Feisty. Boot up time is now 35secs before 1min. It also feels nicer to use slightly more stable than the last version. Mplayer in Gutsy is really cool and plays everything i need always had problems before. I hear there are problems with the latest version of Mplayer from [http://www.getdeb.net/](http://www.getdeb.net/) so I'll give that a miss for now. keep changing themes gone off green, Now using Dark Blue and Dark Transparent Purple with the same animated background,I have very few problems with E17 for me its very stable, But that maybe because I've used it for a long time and know what can and can't be used with it.
