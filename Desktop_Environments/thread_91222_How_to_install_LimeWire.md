### Post by n00blar on 2005-12-06
Easy:
1. Make sure you have Sun Java on your Ubuntu box
2. Go to LimeWire's download page, since you paid for the pro version, and download the Other version (zipped file)
3. Unzip this file to your Home directory, or whichever you like
4. Run it from that directory.

This has worked nicely for me; I do also own the professional version and I don't have any issues.

Hope this helps.
