### Post by endfx on 2006-07-25
You were explicit enough, I don't think they read closely enough.

You want to goto (NOT in firefox) System -> Preferences -> Font
and change the "application" font size.

This changes all applications so it may not be exactly what you want. As far as I know, you can't change *only* firefox application fonts.


Did this help you?
