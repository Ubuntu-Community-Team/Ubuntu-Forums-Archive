### Post by noneofthem on 2007-06-12
Hey man!

Here you go! I have got some answers for you:

1. System requirements:

Go to the bottom of this page:

[http://www.ubuntu.com/products/whatisubuntu/desktopedition](http://www.ubuntu.com/products/whatisubuntu/desktopedition)

While you are there, read some of the details on that page while scrolling down.


2. Difference between Ubuntu/Kubuntu/Xubuntu:

Basically it is all Ubuntu. The main difference is the default desktop environment. Ubuntu comes with GNOME, Kubuntu with KDE and Xubuntu with XFCE.

Which one you wanna use is entirely up to you and you can even change it afterwards if you want to.

Here are some futher details on each of these desktop environments:

GNOME: very accessible and "clean" desktop. There are three main menus (Applications, Places and System). All your apps go to Applications, all places can be found under Places and all system tools and configuration tools are listed under System. Easy enough, isn´t it?

KDE: I hope I don´t insult any KDE user now, but KDE is the desktop environment that looks the most similar to MS Windows. There is kind of a "Start"-button on the same spot as it is under Windows and there is only one system bar on the bottom of the screen. However, KDE is much more flexible than the Windows environment. Almost everything can be customized as you like and there is a huge amount of very very good programs and tools that come with KDE (e.g. Amarok, K3B etc.).

Xubuntu: very fast and clean desktop environment that looks a bit like GNOME. However it is not as memory hungry as GNOME and KDE. This is why Xubuntu is the better choice on very old computers.

Apart from those three there are many other desktop environments available:

Blackbox, Fluxbox, Enlightenment, ROX etc...

These DEs are even better for old computers as they require only a very small amount of memory. However I find the configuration kind of too difficult for beginners.

From experience I would say your best choice to start with would be Ubuntu using GNOME. It is very clean, looks great and I have the feeling that it runs a bit smoother than KDE (even though some KDE users will tell you different). It all depends on your system configuration of course.

If you have any further questions, come back and just ask. We are all friendly and we all had to start as beginners.

greetz

none of them

### Post by RedSquirrel on 2007-06-12
Here are the requirements for Xubuntu, taken from
[http://www.xubuntu.org/get#requirements](http://www.xubuntu.org/get#requirements)

> Minimum system requirements

To run the Desktop CD (LiveCD + Install CD), you need 128 MB RAM to run or 192 MB RAM to install. The Alternate Install CD only required you to have 64 MB RAM.

To install Xubuntu, you need 1.5 GB of free space on your hard disk.

Once installed, Xubuntu can run with 64 MB RAM, but it is strongly recommended to use at least 128 MB RAM.
Depending on how old it is and how much refurbishing you plan to do, there is also the barebones install. Here is one guide that might help:

[http://www.psychocats.net/ubuntu/minimal#barebones](http://www.psychocats.net/ubuntu/minimal#barebones)
