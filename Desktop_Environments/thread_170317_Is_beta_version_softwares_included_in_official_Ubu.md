### Post by ruaraidh on 2006-05-06
You'll find that a lot of software in the free software community is in an almost perpetual state of beta. This doesn't necessarily mean that the software is bad or needs a lot of testing before casual use - it just shows a greater commitment to quality than in the proprietary software world, with its habit of slapping a 1.0 label on any trash it can throw together (see Windows 1.0... well, any windows version actually). 

So yes, beta software (including synaptic, the main GUI package manager) is installed in a default Ubuntu release, but it's nothing to worry about. You'll be hard pressed to find a distribution that includes no beta software, but if you so desire you can remove basically all of it with a little work. Again though it's worth stressing that the beta software in ubuntu is very stable and usable, often outcompeting its counterpart on Windows at a 1.0+ release.

Hope that helps!
