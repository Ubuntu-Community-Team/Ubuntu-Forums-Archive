### Post by tea for one on 2016-12-21
The media player controls are probably one of the Gnome Shell extension packages.

Are you familiar with Gnome Shell extensions?

[https://extensions.gnome.org/](https://extensions.gnome.org/)

You do not have to upgrade the whole Gnome desktop to use extensions - you simply pick and choose which extension suits your personality/workflow.

Even if an extension is installed, it can be disabled and/or removed very easily.

You use Gnome Tweak Tool to manipulate the extensions.
