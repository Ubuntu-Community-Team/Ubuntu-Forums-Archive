### Post by benerivo on 2009-04-12
Where is the shortcut? Is it on the desktop, in a folder in nautilus, or in the gnome menu? If it's on the desktop, then you can use the gnome configuration editor (gconf-editor), and navigate to apps>nautilus>preferences and change the desktop font to 0. The only drawback is that it will remove the text from all your desktop icons.
