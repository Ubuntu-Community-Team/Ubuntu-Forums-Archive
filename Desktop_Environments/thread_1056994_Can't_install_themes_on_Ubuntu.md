### Post by mcduck on 2009-02-23
That's not a GTK or Metacity theme, but a collection of Emerald themes. (for Emerald window decorator which can be used to decorate your window borders).

To use those themes you need to have Compiz ("Desktop effects") enabled and both Emerald and Emerald Theme Manager installed (and Compiz set to use emerald as window decorator). Then can extract the archive and install the themes inside it using the Emerald Theme Manager.
