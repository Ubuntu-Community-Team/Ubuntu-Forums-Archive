### Post by Tamlynmac on 2011-09-16
[Calibre]("http://calibre-ebook.com/about#features") is a format conversion app. It can convert pdf, etc. to mobi - heres's a [list]("http://manual.calibre-ebook.com/faq.html#what-formats-does-app-support-conversion-to-from") of formats. I would probably export the document (LibreOffice) as a pdf/odt and then convert it to mobi. It's available for install in the Ubuntu Software Center.

As for LibeOffice help, you might wish to add this to your bookmarks:
[LibreOffice Forum]("http://libreofficeforum.org/forum").

Good Luck
