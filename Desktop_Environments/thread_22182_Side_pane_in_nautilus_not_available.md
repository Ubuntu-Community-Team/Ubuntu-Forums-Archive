### Post by Buffalo Soldier on 2005-03-26
Open Nautilus as you usually do. Then go to:
**Edit -> Preferences -> Behaviour -> *(check)* Always open in browser windows -> Close**

Close all Nautilus window.

Open Nautilus once again. Hit **F9**.

### Post by Buffalo Soldier on 2005-10-16
That checkbox enable/disable nautilus browser mode. The side pane is only available when nautilus is in the browser mode.

Please do not confuse browser mode with side pane. They are related but both word does not mean the exact same thing
