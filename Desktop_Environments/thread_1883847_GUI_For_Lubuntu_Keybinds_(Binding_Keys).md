### Post by marinara on 2011-11-20
[obkey - Openbox Key Editor - Google Project Hosting]("http://code.google.com/p/obkey/")

Lubuntu's window manager is openbox, which lacks a GUI for binding keys to stuff;
click on above link to download a tarball. (With Oneric, Dont worry about PyGTK bindings)

after unpacking the tarball you will need to run it from the terminal.  
Go to the directory where you unpacked, type
```
./obkey ~/.config/openbox/lubuntu-rc.xml 
```Enjoy

P.S.  obkey has no help file, no nothing
