For those of you who had trouble aswell with the .jar files I'll give you detailed instructions on how to fix it in Ubuntu 13.04. You can thank me later.

Step 1: Download Ubuntu Tweak
Step 2: Open the Admin tab
Step 3: Click File Type Manager
Step 4: Click All
Step 5: Find the Java Archive
Step 6: Click the Edit button
Step 7: Click Oracle Java 7 Runtime
Step 8: Enjoy playing (or running) whatever app you wanted to run
