### Post by _Purple_ on 2009-04-18
Go to System > Administrator > Lockdown Editor and select Panel. Then uncheck the check box that says Lock down the panels.


Gnome Lockdown Editor is called pessules. If it is not there, you can install it using the following command:
```
sudo apt-get install pessulus

```
