### Post by grahammechanical on 2013-02-22
> Accessibility is central to the Ubuntu philosophy. We believe that computing is for everyone regardless of nationality, race, gender or disability. Fully translated into over 40 languages, Ubuntu also includes essential assistive technologies, which are, of course, completely free.

Source?

[http://www.ubuntu.com/ubuntu/features]("http://www.ubuntu.com/ubuntu/features")

We get persistence in two ways. 1) Install to the hard disk. 2) Create the USB memory stick with persistence.

a) From Ubuntu

[http://www.ubuntu.com/download/help/create-a-usb-stick-on-ubuntu]("http://www.ubuntu.com/download/help/create-a-usb-stick-on-ubuntu")

b) From Windows

[http://www.ubuntu.com/download/help/create-a-usb-stick-on-windows]("http://www.ubuntu.com/download/help/create-a-usb-stick-on-windows")

c) From os x

[http://www.ubuntu.com/download/help/create-a-usb-stick-on-mac-osx]("http://www.ubuntu.com/download/help/create-a-usb-stick-on-mac-osx")

We localize Ubuntu during the installation to hard disk by selecting a location and a keyboard layout and language. The initial language selection is for using a Live Session or for installing to hard disk. At the moment you are not using Ubuntu to the full extent of its capabilities.

The language packs are either present in the ISO image or are downloaded during the installation. Then when we activate another language and keyboard layout the necessary packages are installed. We will also be invited to install writing aid (dictionaries, etc.) for that lanuage.

It is possible to have the OS in one language and switch to writing in another language without changing the language of the OS. We do not need to have different language versions of Ubuntu installed. Just the one version of Ubuntu for all 40 languages.
