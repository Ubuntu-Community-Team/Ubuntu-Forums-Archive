### Post by vanadium on 2018-03-25
Most certainly, Ubuntu Budgie will follow along with the latest Budgie, thus also switch to a desktop environment based on QT. How this will change the whole experience depends on how Budgie takes the change. They might move all  desktop applications to QT based ones, or stick with the current GTK based applications, including control center.

> Ubuntu Budgie is targeting a minor point release of the current
desktop v10.4 --> v10.4.1 with a whole lot of nice project developed
