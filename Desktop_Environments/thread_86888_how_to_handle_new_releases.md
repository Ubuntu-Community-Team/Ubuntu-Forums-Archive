### Post by Xian on 2005-11-06
[QUOTE=maxwellcom]but with a new release every six months, how do you migrate all your customization?  do you do a fresh install and then reconfigure everything?  or is there an "update/upgrade" option for new-release installs (and is the result stable)?  is there a way to do a clean install of a new release and somehow preserve custom configurations?[/QUOTE]

Most people will just keep their /home directory on a separate partition. This will enable them to do a clean install if necessary and still keep the majority of their user configs and prefs. There is an official release upgrade method that should work if you don't have a lot of non-Ubuntu packages installed on your machine (this can cause numerous upgrade problems).

[How-To Upgrade To Breezy](https://wiki.ubuntu.com/BreezyUpgrade?highlight=%28upgrade%29%7C%28breezy%29)
