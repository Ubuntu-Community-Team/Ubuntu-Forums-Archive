### Post by aysiu on 2006-06-19
Are you hard-up for hard drive space? If you have a few MB to spare, you may want to try ```
sudo aptitude update
sudo aptitude install gnome-utils
``` That has *gnome-search-tool*, which will integrate in well with XFCE.
