### Post by yetimon_64 on 2015-09-24
Assuming you mean the Unity launcher, No. 

Once upon a time in about 2011 you used to be able to with a PPA, sadly no longer possible.
It is however possible to autohide it [s]and disable any hot corner settings that show it [/s] then install a dock application like docky or even the gnome-panel package as a substitute to run on the bottom of the screen.** Edit:** just checked ccsm, can't see nowadays where the edges for the autohide function are turned off. May not be possible by the looks of it, though the autohide is available still.
 
Another option is to install another DE, which sets out the desktop differently, beside the standard Unity interface and log into it from the lightdm greeter screen (use the "power cog" icon at the side of the password input box). If you switch to another DE the lightdm greeter remembers it and will only change back to Unity if you do the same procedure with the lightdm greeter.
