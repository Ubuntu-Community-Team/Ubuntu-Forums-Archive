### Post by redrumrogue on 2014-09-29
Ok - automount options for devices can be set using gnome-disks, or by editing your /etc/fstab file.  In gnome-disks you select a device and you'll find a buttom labelled "more actions" where the device partitions are displayed.  Click the "more actions" button and select "Edit mount options..."
