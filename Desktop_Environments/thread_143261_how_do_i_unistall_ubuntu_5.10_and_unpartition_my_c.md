### Post by alamba on 2006-03-12
If possible put in a Windows 98 CD or one of the boot disks available online and run fdisk /mbr. This will clean up grub from the MBR. 

Then boot into windows and go to Computer Management >> Disk Management. Right click on the "unknown" partition and click on delete.

You can now repartition that space to use it with windows.
