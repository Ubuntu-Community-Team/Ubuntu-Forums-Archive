### Post by PaulFr on 2007-06-17
Take a look at KDE Control Center (command line way to start it is to run **kconfig** in a terminal). In the dialog that comes up, select **KDE Components**, then **File Associations**, expand the **video** node on the right hand side.

If you select the video node, it tells you whether the player starts in embedded or separate mode. If you select any one video type, it will tell you the description of that type and the list of applications that can handle it. Move your preferred application to the top to make it the default. You have to do it for each type you are interested in, since the list of players for each type are different. *This assumes that you already have mplayer installed :-)*
