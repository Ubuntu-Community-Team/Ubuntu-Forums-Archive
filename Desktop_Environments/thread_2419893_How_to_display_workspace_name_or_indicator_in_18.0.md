### Post by greg2lapa on 2019-05-26
Anybody know a way to visually display some kind of indicator or name in the TOP PANEL for what workspace one is in?

Vanilla gnome has an extension called 'Workspace Indicators' which displays a '1' in workspace-1 and a '2' in workspace-2 and so on.  I would be happy with this.

I found a gsetting called 'org.gnome.desktop.wm.preferences workspace-names' and I named my two workspaces but the names don't show anywhere that I can find.

Is there anyway to display something in the top panel to indicate which workspace I am in?

EDIT: I found the Workspace Indicator app in the 'Ubuntu Software' app. I did an 'apt search workspace-indicator' before posting here and nothing came up. So I guess it goes by a different name for CLI installs.

### Post by Dennis N on 2019-05-26
**Workspace Indicator** gnome shell extension can be installed from **Ubuntu Software**. Just search for "Workspace Indicator", install it and it will appear on your panel.

You can also install and manage all your gnome shell extensions using Firefox from the website at:

[https://extensions.gnome.org/](https://extensions.gnome.org/)
