### Post by JKyleOKC on 2012-08-29
I can't answer question 1, but question 2 is fairly simple. Right-click the icon, select "Panel" from the context menu, there select the Items tab, and locate the appropriate launcher from the list that results. You can count from the "Separator" item to locate the one you want. Select it, then click the pencil-like "edit" button beside the list to bring up the editing dialog for the selected launcher. Next, click the icon shown in the editing dialog; it will bring up a window of application icons. You can change the category to "All" to see everything, then pick the one you want to use, highlight it, and click OK/Save (I'm not at my 12.04 system right now). That does it.

You can edit them all with only one trip to the panel items list.

Hope this helps!

### Post by LewisTM on 2012-08-29
You can change the icon for a **launcher**, hell you can even put your portrait as the icon. You will never see that portrait on the **taskbar** i.e. the Window Buttons panel plugin.

A dock application like Cairo-Dock or Unity can do that because the launchers capture the launched tasks and become task buttons, keeping their custom, themed icon.
