### Post by the yawner on 2008-10-18
It's impossible to get a virus (specifically written to run on Windows) in Ubuntu or any other Linux variant. That said, it doesnt hurt to have an anti-virus on your Ubuntu installation just so you could prevent sharing files w/ virus with Windows PCs.

### Post by Mazin on 2008-10-18
ClamAV is the popular open-source virus scanner that's most often used on Linux.  Usually, it's used when the Linux server acts as a file server or mail server, and viruses need to be detected before they reach Windows computers.

If you're just using your computer for your personal stuff, then you probably don't need antivirus.  The most you need to do is keep your system up-to-date.

### Post by TransitMan on 2008-10-18
> **bagle said:**
> remember to allways have an antivirus nomatter what os your runing. you shhould allwas be on the  safe side.i recomend clam av

That type of response would be typical if one used Windows. However, in Linux there is NO need for Anti-virus scanners/programs, unless you share files with a Windows computer, and then only if the Linux user wants top install one. 

Linux is inheritly safer to use than Windows OSes due to the fact that in Linux it takes /root to make anything launch to interact and install on the drive.
In Windows, simply clicking on a link is enough to get hammered.

I have been using Linux since the end of 2005, and have taken note that I do NOT need all the little AV/AT/Spyware prevention programs that I had to have in Windows.

Now if the OP wants to install an anti-virus program, there are several out there that are Linux compatible, as well as ClamAV. 
It is a matter of research and decideing what one wants.

Lastly, with anything you download and open, always make sure it is from a trusted source. When in doubt, ask or search for the original download site.

(As for launching and installing - it can be done in the /home file or partition, but the rest of the system is safe unless one invokes "sudo" prior to using the program. Then all bets are off.)
