### Post by Patb on 2008-03-21
Ubuntoexpert, if you can run the live CD, you should see an icon "Install" when it is up.  If you cannot run the live CD on your system, you could try installing from the "alternate desktop CD" available from [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download).  

If this doesn't work, please give more details about how it "will not let me install to the hard drive".  

Hope this helps. Cheers, Pat.
