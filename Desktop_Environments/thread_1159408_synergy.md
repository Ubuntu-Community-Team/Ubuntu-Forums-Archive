### Post by smokyink on 2009-09-09
I'm just about to set mine up so I found your post...
So I'll tell you how i do it and how it works for me:
 
To download synergy on Ubuntu I type 
in the terminal(Which can be found at Main menu> Accessories> Terminal)

```
sudo apt-get synergy
```

Then it asks you for your pass...and you type it...and it installs it:)

For configuring synergy see:
[https://help.ubuntu.com/community/SynergyHowto](https://help.ubuntu.com/community/SynergyHowto)
