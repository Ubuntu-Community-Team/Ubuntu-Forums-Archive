Hi,

I've found a neat little text editor, called leafpad that I would like to use as my default editor. It is fast unlike kate that takes an eternity to open up. But I don't know how to make it default. I've already downloaded and installed it but it doesn't show up in the System settings>user account>default applications drop-down list. The only way i can use it-rightklicking on the file and choosing it from the option menu; but this is clumsy. My question is how can I disable kate and make leafpad my default texteditor. 

Thanks,
J. Mak

---

### Post by 23meg on 2005-11-23
Right click a text file, hit "properties", and choose leafpad in the "open with" tab. If it doesn't appear there, add it by hitting the "add" button.
