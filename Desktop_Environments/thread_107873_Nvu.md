### Post by anil_robo on 2005-12-25
Good features in Nvu:
 1. Free
 2. Excellent HTML editor
 3. Supports drag and drop for most actions
 4. Simple, user-friendly interface
 5. Images can be resized by simply streching them with mouse.
 6. Supports templates
 7. Supports CTL and right-to-left languages (Arabic etc.)
 8. Includes positionin grid for easy layout structuring
 9. Includes spellchecker.

I have used Dreamweaver for almost four years now. I have downloaded Nvu, but there are several features where it differs from Dreamweaver:

1. Does not automatically refresh directory list when you upload a file. You have to do it manually.
2. Can't create files on the server directly. You have to create them offline, then upload them manually. (This is a big drawback for me)
3. No synchronization option between offline and online files.
4. Right click (context)  menu not available for files/folders listed in directory view
5. No support for frames (but they're rarely used these days anyway)
