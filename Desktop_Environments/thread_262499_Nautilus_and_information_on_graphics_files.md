Hello.

It would be quite useful if one could get basic information about a graphics file (e.g. its dimensions) by selecting them (e.g. info could be displayed on the status bar) or hovering on them (e.g. info could be displayed on a tooltip). As far as I can tell, the only way to get to that info in Nautilus is to right-click the file, then open the 'image' tab on the properties window that pops up.

As it stands, the only info one can get about a file on the status bar of the Nautilus windows is its size. I assume that is the case because a 'size' is a common denominator among file types. However, because graphics files are so common, I believe adding some basic info about them would be appreciated by many.

Is there anyway that can be done? Does anybody know if there are any plans in that regard?
