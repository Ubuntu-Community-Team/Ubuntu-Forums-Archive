### Post by Dennis N on 2014-01-05
Are you using the Wallpaper Selection Box in the Desktop Preferences dialog?

As you say there is no save option - you use the Open button. Use the browse button on the right end of the wallpaper selection box, browse to and select the new wallpaper, and Open. The wallpaper should change immediately, and you close the Desktop Preferences dialog. That is how it should work.

Update Manager:

There is no new LTS version available so update manager won't offer a newer release if update manager is set to "notify for long term support versions", which means only those. The next would be 14.04, in April 2014, when I think Lubuntu is planning to have an LTS.
