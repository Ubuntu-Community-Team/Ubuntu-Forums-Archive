### Post by kyrealtor on 2011-12-24
With Ubuntu 11.0 to 11.04, the default desktop is Unity.  Make sure you have set your user name to enter your password every time you log in.  Log out.  Click your User Name.  Before entering your password, select "Ubuntu Classic" as your desktop.  This should give you the same Gnome 2 desktop as Ubuntu 10.10.   If your 11.01 does not have the "Ubuntu Classic" option, then try upgrading to 11.04, but NOT 11.10.  If you upgrade to 11.10, you will be stuck with Unity.  When Ubuntu 11.04 is no longer supported (it will still work but with no updates), I will either learn Unity, use KDE or XFCE, or install Linux Mint.
