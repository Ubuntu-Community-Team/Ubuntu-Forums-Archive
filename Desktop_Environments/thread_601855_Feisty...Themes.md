### Post by erginemr on 2007-11-03
Sorry, I can tell little to none on ATI cards. I am using an NVIDIA card (GeForce MX400) unfortunately. All I know of ATI cards is that they are not supported well under Linux, becuase the ATI company refused to share its card driver code with Linux developers. (Luckily lately, they agreed to do it.)

Actually, I was trying to emphasize another aspect of the problem. I also had a similar experience, I used Edgy with no issue whatsoever. For 6 months it was the epitome of perfection to me. Then came Feisty along with the problems. I have first tried upgrading to it, and then making a clean install, both to no avail, because in each time I tried, there was a show-stopper issue (more or less like the ones you have described above), and I had to go back to Edgy in despair.

But as time passed by, my good-old system became obsolete with each new package being issued of Feisty only. The Edgy repositories would only hold the older version a program, whereas its new version packed a handful of new features which I had craved for badly. 

You see, Linux progresses lightning fast and the programs that run on it are no exception. Each new version of a Linux program hosts brand new features and many bug fixes.

Sooner or later, whether you like it or not, you are forced to upgrade. I was using Edgy, avoided Feisty at all costs, but am now using Gutsy. My impression on Gutsy is that it is the best Ubuntu release ever, even better than Edgy. Upgrading Ubuntu is definitely not the best idea, The odds are that, upgrading will break something irrevocably, especially if you customize your previous system to your liking by installing extra software from outside the repositories. 

So, maybe, you may want to try once last time with a clean install before giving up on Gutsy.
