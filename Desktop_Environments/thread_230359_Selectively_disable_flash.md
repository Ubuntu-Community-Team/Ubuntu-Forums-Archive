### Post by teasum on 2006-08-05
Is it possible to disable flash for certain websites only?  Or, perhaps more accurately, I would like to use flash only for particular websites--homestarrunner.com, for example.  On the other hand, the only thing flash does on 99% of other sites I visit is display adds.  On top of that, the flash adds capture the keystrokes that constitute scrolling, which is really annoying (see this thread: [http://www.theonion.com/content/node/51159)](http://www.theonion.com/content/node/51159)).

So, can I disable flash for most sites, but keep it for some?  My guess is probably not, but it never hurts to ask.

### Post by rogblake on 2006-08-05
If you are using Mozilla Firefox, install the "Flashblock" extension (from the Firefox menus, see Tools/Extensions/Get More Extensions and search for Flashblock).

By default, Flashblock puts a "play" button in place of the animation. You can also whitelist particular sites that you still want flash to play on.
