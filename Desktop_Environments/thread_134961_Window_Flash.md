### Post by ardchoille42 on 2007-03-20
Have you installed some other window manager? Metacity is the default window manager in gnome but if you install a different window manager (openbox, fluxbox, beryl, etc) and use it in gnome, then the System->Preferences->Sound->System Bell is not going to work for you. The new window manager would have to have that functionality in-built  for it to work.
