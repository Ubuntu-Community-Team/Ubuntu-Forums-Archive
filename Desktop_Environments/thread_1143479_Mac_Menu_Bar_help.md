### Post by CaptainEvilStomper on 2009-04-30
Do you mean the Dock (where all the applications are) or the Menu Bar (where the File, Edit, View, etc. menus are)? Usually the menu bar is at the top of the screen, and if that's what you're talking about then you can get that with the GNOME Global Menu Bar ( [http://code.google.com/p/gnome2-globalmenu/](http://code.google.com/p/gnome2-globalmenu/) ).Just follow the instructions here to install it: [http://code.google.com/p/gnome2-globalmenu/wiki/InstallingonUbuntu](http://code.google.com/p/gnome2-globalmenu/wiki/InstallingonUbuntu) I'm using Jaunty 64-bit and it's working great for me (though it's not quite as stable as the 32-bit version).

If you mean the Dock, there are three different apps that I know of - Cario Dock, AWN Dock, and Docky (which comes as part of Gnome-Do). I'm currently using AWN Dock but I've been thinking about switching to Docky. There are plenty of guides floating around the internet that will tell you how to install any one of those, though - just do a Google search and you'll find a ton of them.

By the way, if you need a good OS X theme, try Mac4Lin: [http://sourceforge.net/projects/mac4lin](http://sourceforge.net/projects/mac4lin)
