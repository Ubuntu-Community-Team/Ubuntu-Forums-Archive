### Post by GeneralZod on 2005-05-20
I'm not at my Kubuntu box right now, but here's an idea which might work:

Load Konqueror; go to Settings->Configure Konqueror->File Associations(?)

Find the file association for tar.gz

On the right hand side, I *think* there is a tabbed pane which lists the applications that you want to open this file type with.  If the option to use embedded viewer(?) is not there, try clicking the tab and see if it appears.  If it does, uncheck the Embedded Viewer option.  Ensure that Ark is in your list of applications to open with.

Note that I'm working purely from memory here, so a lot of these names (e.g. "File Associations", "Embedded Viewer" etc) might not be correct.  Let me know how you get on :)
