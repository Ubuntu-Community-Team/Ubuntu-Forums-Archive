### Post by orb9220 on 2007-06-28
Close everything you don't want. Then open sesstions manager.
On the Current sessions tab see if there are any nautilus there if there is remove them.

When you have the default desktop they way you want it to start up. Then on Session Options tab uncheck Automatically save changes to session. And click on Save the current session.

By unchecking Auto I can shutdown desktop with many apps running knowing when I boot up I will start with a clean desktop.
