EDIT:

Its fixed yAY!!! I did the following.

Installing Ati "fglrx" drives:

1. Open the "synaptic" (System > Administration > Synaptic)
2. Search for fglrx
3. Mark xorg-driver-fglrx and fglrx-control
4. Apply changes (it will download and install the drive)
5. Open a terminal window
6. sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup (Creating a backup)
7. sudo gedit /etc/X11/xorg.conf
8. Find "device" change driver = "vesa" or "ATI" to driver = "fglrx"
9. Save
10. ctrl+alt+backspace (restart X)


After this step you may run sudo dpkg-reconfigure xserver-xorg to add your desired out of standart resolution or refresh rate value

*Backuping xorg.conf is essential, specially if you are running a non english language, because if you use the comand sudo dpkg-reconfigure xserver-xorg your keyboard setting will get missconfigured. Use your backup xorg.conf to restore your regional keybord setting

**After all this you will be able to get a real 85Hz or 100Hz refresh rate under 1024X768

*** Now that you have your Ubuntu Linux running fine you can install Automatix - It'll install everything you need easier than count one, two, three

The original article is here [http://www.ubuntuforums.org/showthread.php?t=190133&highlight=screen+resolution](http://www.ubuntuforums.org/showthread.php?t=190133&highlight=screen+resolution)

### Post by JoWilly on 2006-07-11
You say you have installed the ati drivers. Then you should change      Driver        "ati" to      Driver        "fglrx".

If you put "ati", it will use the default xorg driver and not the ATI one.

PS: Also, before messing with ati.com driver (because it can be easier to make a mistake in setting it up) it is best to get it working with the driver incuded in the repo (install linux-restricted-modules and fglrx...), because this works out of the box as soon as you replace ati with fglrx.
