### Post by ajgreeny on 2012-07-30
In the Files and Folder Browser, (nautilus) make sure that **View->Sidebar** is set to show places not tree, and all your nautilus bookmarks should then show in the left-hand pane.  You can then add a bookmark by opening the folder and going to **Bookmarks ->Add Bookmark**
