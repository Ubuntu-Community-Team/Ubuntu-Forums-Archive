### Post by mcduck on 2010-05-14
That thread is talking about rounded window corners (which quite many themes, like the default themes in 10.04, already have).

And being able to do anything in Linux doesn't mean that there would be a configuration option or tool for whatever one can imagine, it means that you have the freedom to do anything, there is no legal (or other) restrictions apart from your own skills and the time and effort you are willing to put into it.

(so you are *free* to do anything, but you might not be *able* to. :))

So yes, you can have round windows, you'll just have to write the programs to do that yourself (I doubt anybody would have done that already, as round windows would get pretty hard when you start to think about fitting the actual programs into those windows (not to mention that it would be pretty bad way to use screen space...).
