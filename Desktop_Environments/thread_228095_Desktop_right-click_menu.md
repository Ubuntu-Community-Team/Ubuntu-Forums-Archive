The desktop right-click menu is managed by nautilus, and is not very customisable. If Suse does it, either they did a good bit of hacking or they are using nautilus scripts. You can, however, switch window managers in gnome to a window manager that has a more customisable right-click menu.

I wrote a tutorial about switching the default window manager:

[https://wiki.ubuntu.com/Replace_Metacity_with_Openbox_in_gnome](https://wiki.ubuntu.com/Replace_Metacity_with_Openbox_in_gnome)

Switching to another window manager in gnome is quite easy and will allow you to customise the desktop right-click menu to suit your needs. Take a look at the screenshot below, it shows my desktop, using the openbox window manager in gnome, and you can see that I have set up the desktop right-click menu to mimic the gnome menus.
