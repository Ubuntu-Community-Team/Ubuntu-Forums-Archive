### Post by coffeecat on 2011-06-04
> **Emanuele_Z said:**
> How to open multiple instances of the same program from the sidebar (i.e. multiple instances of LibreOffice or the terminal)?Middle-click on the application launcher icon for a new instance of the same application. If you have a scroll-wheel, depressing that give you a middle-click. You can also use the application's File menu or ctrl-N for most apps (shift-ctrl-N for terminal). Interestingly, Libre Office is one of the few (the only one?) in which  you can open a new window with a right-click to the launcher icon. For Libre Office, right-click > New Document.
