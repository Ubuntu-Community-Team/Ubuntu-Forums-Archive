### Post by Alwimo on 2011-11-09
There are lots of little visual differences. One is the ubuntu button at the top of the launcher (side dock thing) is translucent in 3D mode of 11.10 and opaque in 2D.
 
You can try looking at the Additional Drivers (jus type that in to the dash search bar) and see if there is something that lets you run Ubuntu 3D if you don't have it.
 
Or you can switch between 2D and 3D in the login screen by clicking on the cog icon next to your name.

### Post by Frogs Hair on 2011-11-09
To run 3D you will need to install a proprietary driver from additional drivers and install the CCSM from the software center . Unity 3d has features that 2D does not , such as panel transparency , icon size , and back lighting adjustments to name a few . see the link .[http://en.wikipedia.org/wiki/Unity_%28user_interface%29](http://en.wikipedia.org/wiki/Unity_%28user_interface%29)
