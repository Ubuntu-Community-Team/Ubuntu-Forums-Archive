Actually I've noticed the same thing.  I suspected that it has to do with memory usage, as I notice other programs being similarly flaky when I've got a few other apps open at the same time, for instance playing movies bombs out after a second or so.  Also, I don't have a heck of a lot of RAM on my system, only 128MB.

So try increasing RAM if it's below 256MB as a start.
