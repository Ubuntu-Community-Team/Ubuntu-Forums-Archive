### Post by smtanner on 2005-04-10
It is working properly for me.  I have a fresh install of kubuntu.  Try right clicking on the desktop and selecting "Configure Desktop...".  Then select Behavior->Device Icons and make sure it is checked.  If it already is then I don't know.  Reinstall maybe.
