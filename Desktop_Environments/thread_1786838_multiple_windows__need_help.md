### Post by ajgreeny on 2011-06-20
I nautilus you can show two panes on the right of the window, key F3, each with a different folder open.  If your screen is big enough you could open several nautilus windows, sized to fit on screen (maybe the compiz plugin **grid** will do that) and all visible, each with two panes, showing a different folder in each one.

It is a bit clunky, I agree, and there may be other file managers or other utilities that can do it more sensibly and easily, but I will leave you to search more about that.
