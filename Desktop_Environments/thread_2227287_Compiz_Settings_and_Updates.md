### Post by deadflowr on 2014-06-01
For the first one, when you first open ccsm, whatever settings you have set will still be set.
(One exception that has been noticed is that when you open General Options > Desktop Size, if you have set workspaces, it reverts it back to just one. Other than that anomaly, running ccsm doesn't change anything, normally)
