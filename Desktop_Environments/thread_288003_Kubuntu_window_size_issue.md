### Post by DeVelaine on 2006-10-29
Having recently deciding to give Kubuntu a try, I've come across some window size issues. Most notable is if you resize the panel at the bottom, Firefox doesn't default to loading maximized. I found the solution to that problem by adjusting the Special Application Settings, but this has created another issue. All of my other programs are now starting maximized, completely throwing off my vision of the windows. So I have two questions:

1. If I remove all of the special window settings, will all of these windows go back to their default sizes?

2. Is there an option anywhere within KDE/kdm/Kubuntu to tell it to remember last window position and size for each different program without having to manually set every window position and size individually?

---

### Post by Coral Sea on 2006-12-10
Right click on the Firefox icon on the task bar.  Click on "Configure..."  Insert "kstart --maximize" in front of the command on the Application tab (without quotes).  Now, Firefox will launch maximized whenever you click on the Firefox icon.
