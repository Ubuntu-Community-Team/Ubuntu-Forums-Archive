### Post by chriscrutch on 2006-06-20
[QUOTE=aysiu]I've never heard of this Directory Opus thing. What kinds of features do you miss from it?[/QUOTE]

There are simply too many features to list here, but basically it boils down to dual-panes and customization of everything.

Almost everything in DOpus can be customized to your liking.  Everything from what commands are in the menus to what happens when you Ctrl-Alt-Right-Click on a JPEG file as opposed to a PSD.  Right click menus can be defined for individual file types, groups of file types, files only, folders only, or everything.  There are view settings that change automatically based on pre-defined paths, or on whether a certain (definable) percentage of the files in the directory are of a certain type (e.g. jpegs, mp3s, or movies), or cycled through on the fly.  There is tabbed file management like Opera or Firefox.  Toolbars are defineable, as are the buttons on those toolbars and the commands that they run.  Drag and drop events can be customized, backgrounds, fonts, everything.  I can even define what's in the InfoTip boxes.  I can set filters, either on the fly or permanently, for individual folders or groups, that will either show or hide files based on different patterns such as file type, "hidden" or "system" attribute flags, file name patterns, even things such as shutter speed if working with digital camera files or album titles if working with tagged music files.

That list just scratches the surface.  The flexibility and customization possibilities are near endless.  Anything that can be done in any file manager can be done in DOpus, and, most likely, more quickly and efficiently.

I have considered trying out KDE, but I'd never seriously considered XCFE.  I remember trying out Konqueror using a Knoppix live cd before making the switch to a dual-boot system, and remember it being pretty lacking, but better than GNOME's version.  I'll try the other suggestions and still be open to new ones.  Hopefully I'll be able to find a file management system under Linux that has half the functionality of Directory Opus.  

Thanks for your thoughts, everyone!

--chriscrutch

P.S.  I'd actually seen the web site for Thunar and thought it looked promising, but haven't gotten around to trying to install it.  I also thought that it was for XCFE, will it run under GNOME?
