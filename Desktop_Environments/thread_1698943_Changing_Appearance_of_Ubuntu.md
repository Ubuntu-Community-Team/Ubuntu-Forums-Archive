### Post by nickleboyblue on 2011-03-03
There is a great post about that [here]("http://lifehacker.com/#!5665765").  The script is still in alpha for ubuntu 10.10, so unless you are willing to try it without knowing 100% if it works, only use that script under ubuntu 10.04.

Otherwise, you can [configure the look manually]("http://www.howtoforge.com/how-to-make-an-ubuntu-10.10-desktop-resemble-a-mac-with-elementary-docky-and-gloobus-preview").  This requires the installation of several additional software packages.  Once finished, it does look very much like a mac, and the instructions for this link come with screenshots of every step.  THIS would be the rout I would recommend, as the script rout may be difficult to reverse for a beginner if it turns out that it doesn't work.

Also, just because the how-to has more steps available doesn't mean you have to do them.  If your system reaches the look you want, by all means, stop where you are!

### Post by Ghost|BTFH on 2011-03-03
Changing Ubuntu's look to anything you want just takes time and patience and a few tutorials like the one mentioned above.

However, if you want to have a little fun and strike out on your own to find your own personal style (and who doesn't, right?) just go to SYSTEM - PREFERENCES - APPEARANCE and have at it.  Make sure to hit the CUSTOMIZE button at the bottom and really go to town.

The changes occur as you make them, so you can see right away if it's exactly what you're wanting. :)

Cheers,
Ghost|BTFH
