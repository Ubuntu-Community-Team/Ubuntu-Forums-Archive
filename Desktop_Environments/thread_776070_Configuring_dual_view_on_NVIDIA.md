### Post by EKa on 2008-05-08
Hello everyone,

I just want to report that I reached my goal. Maybe in an unorthodox way, but still. What I did was that I unplugged my second monitor and then reinstalled nvidia-settings, just for sure.

> sudo apt-get install nvidia-settings

After that I got my standard resolution for my Flatron Wide rihgt. 
Then I could run properly 

> sudo nvidia-settings

and it showed that my system was trying to use third option, beamer on s-video, as a tv-monitor! Now it was possible to disable that and introduce my true extra tube, Compac S710. After that I reconnected the monitor.  

Then, during the following  

> sudo reboot

my system adjusted automatically the screen drivers and after that the result looks just fine.

Lesson to learn was: Restarting with a single monitor and using standard NVIDIA X Server Settings-application, allows you to see the real situation. Then you can gradually increment your system to full scale.

If you recognize your NVIDIA-problem to be similar kind like mine in this thread, I hope this encourages you to try the same on your desktop site.

Cheers!
