### Post by aysiu on 2005-12-04
Maybe go to System > Administration > Users and Groups > Add User and uncheck all the boxes except "Connect to internet using modem"? Incidentally, why would you want a guest to have access to Firestarter? Isn't that the firewall configuration program?

I don't know about the screensaver stuff, though. You might be able to find some extra preferences about that in Applications > System Tools > Configuration Editor if you poke around a bit.
