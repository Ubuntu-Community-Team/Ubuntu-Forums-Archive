### Post by alyson on 2006-06-29
I have the same problem.  I don't know if this will work for anyone else, but if I leave a program window open (not minimized) before I suspend, then my laptop will resume from suspend without a problem.  I've also found that if I forget to leave a program window open and my system freezes, CTRL+ALT+BACKSPACE will restart the display without having to do a hard restart.

It's not a fix, but it's certainly better than forcing a shutdown and booting back up.
