### Post by Dennis N on 2018-11-30
> **Cushie said:**
> My 'Dock' are smaller than the desktop shortcut icons.  
How do you edit **all** the desktop icons?
The size of any desktop icons depends on the icon size in the file manager when using icon view mode, so adjust the size there.
When in icon view mode:
Ctrl + Mouse Wheel; or
Change the zoom level from Options (button) on the window header.
