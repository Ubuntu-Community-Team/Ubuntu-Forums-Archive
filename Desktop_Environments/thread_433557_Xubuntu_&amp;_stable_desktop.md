### Post by voided3 on 2007-05-05
I had a problem with an install of Xubuntu Feisty where it would boot with the icons on the desktop, but no panel. To fix the problem, I just added "xfce4-panel" to the startup items, but I wonder why it did that in the first place. It might have something to do with the fact that it was installed in VMware Fusion on my Macbook Pro, but that still seems like something strange to not come on a startup. I've grown to really like Xfce though so I don't plan on changing; Fluxbox is incredibly fast and I use it on a few comps with Damn Small Linux, but with my experiences with installing it over Ubuntu, it seems a little more difficult to configure (though the pay off is quite worth it). Don't forget about ICEwm too, that's another good lightweight window manager.
