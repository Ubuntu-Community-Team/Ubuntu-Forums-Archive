### Post by asdir on 2008-06-16
The deskbar usually needs some time to go through all folders ("index") in order to have them ready at the time you are starting a query.
If your install is fresh the deskbar might be done with that.

Moreover you can include and exclude certain folders. Maybe your particular folder was not included by default or you excluded it some time ago.
