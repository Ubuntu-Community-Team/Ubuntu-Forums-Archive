### Post by Jucato on 2006-07-09
You do know that you can also post Kubuntu-related questions here, too. The reason why Kubuntuforums.net is a "shadow" compared to this one is because there are more members here and that the majority of Ubuntu users are Ubuntu (GNOME) users. 

But anyway, you can easily add programs that you want to be run upon logging in to KDE in the ~/.kde/Autostart folder. Just open to Konqueror windows (or just one, and split the view to make two views), set one to ~/.kde/Autostart and the other one to /usr/bin (or to where the program you want to run is located), drag, drop, and choose Link Here. 

I'm not sure how gDesklets will behave in KDE. Also, you might be interested in taking a look at [http://www.kde-look.org](http://www.kde-look.org) for more Superkaramba themes.
