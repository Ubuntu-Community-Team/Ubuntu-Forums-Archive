### Post by ACMiller on 2009-04-24
Hey, I'm not sure I completely understand your problem, but from what it sounds like it seems emerald is doind exactly what its meant to. Emerald is a window decorator (i.e. it changes to title bars and frames of windows). Using emerald replaces metacity's window decorations. To change anything inside the windows (e.g. colour schemes, buttons, and the panels) you need to select a new GTK theme to go with your emerald theme. This is found under system > preferences > appearance. 

If you haven't found it, [Gnome-look]("http://www.gnome-look.org") is a great sources of themes. I hope that helps. Cheers
