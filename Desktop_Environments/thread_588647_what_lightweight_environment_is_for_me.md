### Post by kerry_s on 2007-10-23
try the debian xfce4 it's not as bloated as xubuntu.
[http://cdimage.debian.org/debian-cd/4.0_r1/i386/iso-cd/debian-40r1-i386-xfce-CD-1.iso](http://cdimage.debian.org/debian-cd/4.0_r1/i386/iso-cd/debian-40r1-i386-xfce-CD-1.iso)

when you get familiar enough with using debian i recommend you do a custom install with just what you need. xfce4 is a great to start to learn, but for speed custom is the best. while your using xfce4 you can try other wm's to see which 1 you like and works the fastest for you, learn how to set them up, so if you do a custom install you won't get stuck. good luck
