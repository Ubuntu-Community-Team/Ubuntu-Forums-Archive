### Post by tacantara on 2009-05-14
I've found that once you install the KDE package, the Kubuntu startup splash is pretty much standard.  When you get to the login screen, you have the choice there (through the options menu) to continue booting the KDE desktop or the GNOME desktop. 

There's a place in the Preferences where you can select which desktop is your default (i.e. I have GNOME as my default, which means I will get that desktop after I log in if I don't change it before entering my password).  I haven't yet found anything in the preferences that will let you change the startup splash.

Hopefully this will shed some light on the issue :KS
