How can I NOT use the cdrom?[/quote] open a terminal and enter ```
sudo gedit /etc/apt/sources.list
``` and comment out the line with your cd in it like this
```
#deb cdrom:[Ubuntu 5.10 _Breezy Badger_ - Release i386 (20051012)]/ breezy main restricted
```save and exit. make sure to run 
```
sudo apt-get update
```after you made those changes or it will still
ask for your cdrom.
