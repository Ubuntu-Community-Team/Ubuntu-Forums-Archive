### Post by codejunkie on 2006-01-26
[QUOTE=iostream]i just tried to compile a few progs but there always occurs the same problem: after typing ./configure the message appears: checking for C compiler default output file name... configure: error: C compiler cannot create executables can anyone give me a clue please? thx in advance!

p.s. i also tried sudo, and via an activated root account, but it didn't help[/QUOTE]
open a terminal and copy and paste this in there
```

sudo apt-get install build-essential linux-headers-`uname -r` gcc-3.4 libstdc++5

```
this will install the basic compilers kernel headers and things you need to install software however this is just the basic set with some packages you'll have to install other development packages to get them to work hope this helps.
