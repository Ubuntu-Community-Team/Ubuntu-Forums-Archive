### Post by canadianwriterman on 2006-09-15
NVU is available by using **Applications > Add/Remove**. However, you must check the "Show unsupported applications" checkbox before you can see it.

Vic (if you mean the video conferencing software, is available in Synaptic, however you probably don't have the repository that it resides in selected. In Synaptic, open **Settings > Repositories**. Check all the checkboxes, except the backports ones.

That should get you both applications. Hope that helps.
