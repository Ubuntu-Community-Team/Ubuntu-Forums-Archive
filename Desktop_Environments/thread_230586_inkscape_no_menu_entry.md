### Post by jISh on 2006-08-06
Not everything adds itself to the GNOME menu.
However, you should be able to just push alt+f2, type inkscape then hit enter.

If that works, just create a launcher on your desktop for yourself with inkscape as the command line. You can drag that into the menu afterwards.
