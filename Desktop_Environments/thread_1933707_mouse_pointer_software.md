### Post by Krytarik on 2012-02-29
Have a look at Compiz' "Show Mouse" plugin; it may be a bit too excessive for you :P , but it can be configured very much. Therefore, you obviously need to be running Compiz as the window manager (e.g. in a regular Unity session), "CompizConfig Settings Manager", and, depending on what Ubuntu version you are running, one of these packages:

[LIST]
[*]for Natty 11.04 and later: "compiz-plugins-extra" (the real package)
[/LIST]

[LIST]
[*]for earlier Ubuntu versions: "compiz-fusion-plugins-extra"
[/LIST]
Regards.
