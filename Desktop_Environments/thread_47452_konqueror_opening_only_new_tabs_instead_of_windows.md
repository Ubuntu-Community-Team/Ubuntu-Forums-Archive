Hello everybody, I've got a little issue that I'm not able to solve, konqueror refuses to open new windows!
I'll explain, if I open my home dir I've got my classical konqueror file management window, but with this window open, if I try to open another folder from the desktop it get opened in the same konqueror window in a new tab, instead of a new window, I've tried several configuration options, but everything seems to get ignored by konqueror that ccontinues with this odd behaviour, it's a bug? or there is a way to solve it?

---

### Post by SGC on 2005-07-08
In konqueror go to:
"Settings" menu -> Configure Konqueror -> Web Behavior -> "Advanced options" button

And uncheck the last box that says "Open as tab in exicting Konqueror when URL is called externally"

This will works for both URLs and the icons on your desktop.
