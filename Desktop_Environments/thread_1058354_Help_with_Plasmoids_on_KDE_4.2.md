### Post by Darkshade on 2009-02-13
You need to install superkaramba first either typing "sudo apt-get install superkaramba" in a terminal or trough the package manager. After that right click on your desktop -> Add Widgets -> Install New Widgets -> Install Widget From Local File -> SuperKaramba -> *select your file* -> Finish.
Then just select the widget you installed and place it on your desktop or panel.
