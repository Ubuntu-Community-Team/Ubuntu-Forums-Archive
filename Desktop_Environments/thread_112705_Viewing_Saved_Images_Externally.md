### Post by bulldogzerofive on 2006-01-05
if you want this to happen automatically, 

1.  Open konqueror, 
2.  In the adress bar type in the address bar: settings:/
3.  Click on the components icon (i think i don't have it in front of me right now but it is something like that)
4.  Click on the File Associations icon to open the configuration GUI
5.  Click on the "images" group, and to the right there will be a check box to allow you either open items of the group embedded or in a new window.  Check the one you want.
6.  Click on the particular file type you want to open and make sure that it is set to open the same as the group settings.
