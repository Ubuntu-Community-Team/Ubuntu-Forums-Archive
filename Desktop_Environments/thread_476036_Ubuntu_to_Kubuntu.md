### Post by Happy_Man on 2007-06-17
> **scrime said:**
> I use Ubutu Feisty 7.04 which is better, Ubuntu or Kubuntu?
Some might say Ubuntu, some might say Kubuntu. The best way to find out is to try them both. If you're on Ubuntu:
```
sudo aptitude install kubuntu-desktop
```

If you're on Kubuntu:
```
sudo aptitude install ubuntu-desktop
```
