### Post by stinkeye on 2012-01-01
It's the Compiz Grab Handles plugin.
I don't think by default there is a shortcut set, so I don't know how you triggered it.
Open ccsm and set a shortcut.
To install ccsm...
```
sudo apt-get install compizconfig-settings-manager
```

Then to run it open the dash (Super or Windows key) and type in ccsm.

I set a spare mouse button to initiate the resize handles using **easystroke** mouse gestures.

See [**_[COLOR="DarkRed"]HERE[/COLOR]_**]("http://ubuntuforums.org/showpost.php?p=10495426&postcount=11") for how to assign keystrokes to a mouse button using easystroke.
