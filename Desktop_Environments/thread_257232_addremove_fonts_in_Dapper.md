### Post by moore.bryan on 2006-09-14
[I]then you're looking to add fonts to xserver.  xserver saves fonts in a couple places...
[/I] The default font path is /usr/X11R6/lib/X11/fonts/misc/, then /usr/X11R6/lib/X11/fonts/TTF/, /usr/X11R6/lib/X11/fonts/Speedo/, /usr/X11R6/lib/X11/fonts/Type1/, /usr/X11R6/lib/X11/fonts/CID/, /usr/X11R6/lib/X11/fonts/75dpi/, and/or /usr/X11R6/lib/X11/fonts/100dpi/ .
