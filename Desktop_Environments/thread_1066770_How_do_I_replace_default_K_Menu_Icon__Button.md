### Post by redseventyseven on 2009-11-12
(updating this thread with new advice because it came up near the top of a google search)

This can now be done (without resorting to Lancelot) by right-clicking the KMenu icon, and selecting Application Launcher Settings. The option to change the icon appears in the "General" settings tab.

FYI, there is a Kubuntu-ized version of the KMenu icon in the default icon folder - start typing "kubuntu" into the search box and you'll see it.
