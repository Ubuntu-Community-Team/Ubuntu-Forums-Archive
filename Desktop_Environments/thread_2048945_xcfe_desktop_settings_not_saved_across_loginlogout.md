### Post by Toz on 2012-08-29
Maybe try clearing your sessions cache. 

- Logout and go to the first tty (Ctrl+Alt+F1)
- login to the console
- _carefully_ issue this command:
```
rm -rf ~/.cache/sessions
```
- go back to the gui (Ctrl+Alt+F7) and log back in.

You'll need to make some changes and do one more cycle of log-out and log-in to see if anything was saved.

If still not, lets have a look at your ~/.xsession-errors file to see what is happening.
- log out 
- go to tty1 (Ctrl+Alt+F1)
- login to the console
- make a backup copy of the log file (it will get overwritten on a fresh login and we want the old file):
```
cp ~/.xsession-errors ~/.xsession-errors.BAK
```
- go back to the gui (Ctrl+Alt+F7) and log back in
- post back the link to the log file that is generated by running:
```
pastebinit ~/.xsession-errors.BAK
```
