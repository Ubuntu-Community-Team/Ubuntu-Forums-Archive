### Post by j.hill on 2005-08-15
When you install xfce, it will be added as an option on the "session" menu on the login screen.  So you choose to run an xfce session, then log in, and there you go.  You can also set xfce as the default window manager if you like it.  It's my default WM, and I like it, though I think you'll find it's not as user-friendly as GNOME.
