### Post by jimmydean886-2 on 2011-11-21
If you decide to upgrade to 11.10, uninstall gnome-screensaver and install xscreensaver. Then, add xscreensaver to your list of startup apps. It worked for me, and was REALLY fast. The only downside is ctrl+alt+L no longer works unless you manually set it up, and the screen doesn't lock on suspend. Not sure about hibernate.

There is a package of extra xscreensavers in the Ubuntu official repos which also work with the old Gnome-Screensaver. Just do a search in USS or Synaptic.

Enjoy!
