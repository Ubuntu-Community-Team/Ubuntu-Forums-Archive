### Post by ankspo71 on 2012-03-01
If you have Java installed you can try NRuler: [http://noutash.com/freebies/nruler/](http://noutash.com/freebies/nruler/). I'm using the JAR (java) version download with OpenJDK in Lubuntu 11.10.

To start NRuler, right click on Nruler.jar file and select "open with", then choose "custom command line", then type **java -jar** into the box. Then click on OK. 

Kruler is also nice but it requires the KDE runtime which will make it a very large install, which probably makes it less suitable for Lubuntu.
