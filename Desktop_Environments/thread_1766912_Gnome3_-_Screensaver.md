There seems to be a way to get the screensaver to work, but of course it is not perfect.

See [http://tech.shantanugoel.com/projects/linux/shantz-xwinwrap](http://tech.shantanugoel.com/projects/linux/shantz-xwinwrap)

The 'full screen' mode runs in the background, keeping all active windows as is, so it's not exactly a screensaver. But I think this problem is fixable by minimizing all windows and hiding the desktop through a perl/bash script.
