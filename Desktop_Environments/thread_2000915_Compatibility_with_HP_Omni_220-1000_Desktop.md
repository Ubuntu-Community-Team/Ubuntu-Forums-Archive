### Post by zilog6502 on 2012-06-10
I set 8 of these up in a classroom a couple of months back with Kubuntu 11.10. Installation went fine on the G620 processor versions but some of the apps were as slow as hell (LibreOffice, Kivio) but others ran fine (KStars). I ended up going back and installing Xubuntu 12.04 instead which works very nicely and is still closer to an original desktop style than Ubuntu.

One thing that we did encounter was a problem with webcams and Skype. We only have access to Genius Webcams (and horrendously expensive Logitech) in my part of Ecuador and trying to get either to work was a nightmare though the Logitech eventually went ok with some colour problems every now and again.

I installed both off the standard live-cd.
