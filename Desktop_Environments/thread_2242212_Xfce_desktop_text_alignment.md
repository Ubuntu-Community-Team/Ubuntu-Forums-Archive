### Post by ajgreeny on 2014-08-31
Try renaming the ~/.config/xfce4 as ~/.config/xfce4-bak then logout and in again.  That will reset all the desktop configuration to the default and should get you back to what I think is normal.

I have my desktop icons set as 34px and font is ubuntu, size 11, and that gives me icons as in the screenshot.

### Post by Dennis N on 2014-09-01
All images shown so far show only one line of text under the icon. If the icon size is too small, the text will overflow to a second line and that will left-justify with the first line. See the attachment. This is probably what you are seeing. Still, the entire 'block of text' is centered in that it could possibly be further to the left - compare the top and bottom icons.
