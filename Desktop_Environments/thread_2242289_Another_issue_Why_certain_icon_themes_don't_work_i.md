### Post by Toz on 2014-09-01

I would like a simple way to make Ubudao-style works, though.
[This]("http://www.noobslab.com/2014/01/install-ubudao-style-icons-in.html") worked for me on 14.04. Specifically:
```
sudo add-apt-repository ppa:noobslab/icons
sudo apt-get update
sudo apt-get install ubudao-style
```
