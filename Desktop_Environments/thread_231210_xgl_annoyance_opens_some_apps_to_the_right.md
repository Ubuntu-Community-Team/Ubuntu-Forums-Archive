I installed XGL on my Ubuntu box. Everything is working well on my nVidia card (which is a little old so some things can be a little slow ;) ). I like it but I am experiencing one major inconvenience.

I have 4 workspaces. Whenever I start OpenOffice it appears on the workspace to the *right* of the one I am working on. That is, it I start it from workspace 1, it appears in workspace 2. If I start it in workspace 2, it appears in workspace 3. You can *just* see the very edge of the app using 5 or so pixels on the right edge of the screen.

If I switch workspaces OpenOffice itself is working just fine. It does not matter what OpenOffice application I start, and it does not matter it I start it from a menu icon or a document icon. And so far, it only seems to happen for OpenOffice.

Previous to XGL it always opened on the workspace that I was in.

Any suggestions?

### Post by h00s on 2006-08-07
Actually, it does not run on workspace next to it. It runs on current workspace, but it's positioned too much to the right :rolleyes:

Try this: open application, grab it on that few visible pixels and move it whole (with alt+click) to current workspace. Maximize and exit. Next time it *should* be on current workspace.
