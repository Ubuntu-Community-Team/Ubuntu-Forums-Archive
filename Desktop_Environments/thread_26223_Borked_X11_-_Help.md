### Post by erkki70 on 2005-04-12
Hi,
If you use the rescue mode you'll have just a console or you can go to another one from current session by typing CTRL + ALT + F2 or F3, F4 ... 
CTRL + ALT + F7 returns to current.

Hope this helps.

Cheers,
Erkki
