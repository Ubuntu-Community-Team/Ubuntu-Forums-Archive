### Post by vanagon on 2012-06-13
Running Xubuntu 12.04 with compiz.  I'm using just the top panel, with cairo-dock on the bottom.  The problem is that many applications when started start at the top of the screen, rather than lining up at the bottom of the top panel.  For example, terminals open in the very top left, then I have to drag them to stick them against the panel.

Thanks for any advice!

---

### Post by LewisTM on 2012-06-13
Activate the Place plugin in CompizConfig Settings Manager and set it to Smart Placement.

Cheers!
