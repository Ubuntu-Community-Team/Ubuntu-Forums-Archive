### Post by RainCT on 2008-05-15
To get a terminal, try pressing Alt + F2 and writting «gnome-panel» in the window that should appear. If no window appears after pressing Alt + F2, you can go to a tty pressing Ctrl + Alt + F1 (or any other Fx key up to F6) and Ctrl + Alt + F7 (or F8/F9 if that one doesn't work) to get back to your desktop session.
