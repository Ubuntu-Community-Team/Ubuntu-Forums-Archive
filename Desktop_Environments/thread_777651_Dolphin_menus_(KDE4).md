### Post by Xiong Chiamiov on 2008-05-02
There are several menus over at [KDE-Apps]("http://www.kde-apps.org/index.php?xsortmode=high&page=0&xcontentmode=287").  However, Dolphin does some funky things sometimes, so don't expect full compatability.  You can put any service menus in ~/.kde4/share/apps/d3lphin/servicemenus (assuming it's the same location as in KDE3).  If the servicemenus folder doesn't exist, you can create it.
