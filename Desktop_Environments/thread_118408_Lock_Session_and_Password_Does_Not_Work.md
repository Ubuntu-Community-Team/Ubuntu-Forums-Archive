### Post by obibok on 2006-01-17
A temporary fix: CTRL+ALT+F1, log in, type: **passwd**, change/update password, CTRL+ALT+F7 and it should be OK now.

For a permanent solution a simple [search]("http://www.ubuntuforums.org/showpost.php?p=587494&postcount=3") would've done the trick... :rolleyes:
