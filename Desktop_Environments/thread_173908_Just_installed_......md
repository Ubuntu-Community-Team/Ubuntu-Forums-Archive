### Post by az on 2006-05-11
64 megs of ram is not enough.  You need at least 128 for a more "normal' experience and more thaqn that is better.

To install different desktops, open synaptic, enable the universe repo and install the icem and menu packages.  Then, exit your session (log off) and tell your system to log you into the icewm session.

If you don't like that, install the xubuntu-desktop package.  Do the same for that.
