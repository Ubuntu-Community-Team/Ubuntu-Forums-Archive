### Post by BlackJack on 2006-02-18
I am new to Linux in general so take this with a grain of salt, an awfully big one, but this is what I tried and found.

I installed Ubuntu and then installed the KDE desktop (I am still undecided which one I will use) with the following command:

sudo apt-get install kubuntu-desktop

The end result of this was that I could select either Gnome or KDE as a session when I started up, but even more importantly, I was able to access most, if not all, of the KDE oriented utilities while I was running under Gnome and visa versa. If what you are looking for is all those little utilities and applets that come with KDE but still want to use Gnome, This “might” be a possible solution.
