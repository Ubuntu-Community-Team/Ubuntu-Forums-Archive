### Post by mscman on 2006-06-17
Ubuntu uses the "sudo" command for root access.  More information can be found [HERE]("https://wiki.ubuntu.com/RootSudo").

"Apt-get remove" works on some programs and not others because some programs are installed via synaptic (or apt) while others are compiled from source.

You may want to check out more information about Ubuntu by visiting the following links:

[Ubuntuguide.org]("http://ubuntuguide.org/wiki/Dapper") -- a great starting point for beginners.
[Ubuntu Documentation Storage Facility]("http://doc.gwos.org/index.php/Main_Page") -- good general information and HowTo's
[Ubuntu Wiki]("https://wiki.ubuntu.com/UserDocumentation#head-53abb59509da14eef4bfa451d74af2c8f7eaa02b") -- Ubuntu's official Wiki documentation site

Finally, any questions you have can (usually) be answered by posting on the forums, but it seems you already know that one... :p
