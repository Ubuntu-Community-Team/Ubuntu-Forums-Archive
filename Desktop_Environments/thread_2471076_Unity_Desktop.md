The Ubuntu Unity Remix project is seeking for a official flavor status for a while now if i'm not mistaken. Rudra, the project lead, knows more about this. And iirc Popey even suggested to use the name Ubuntu Unity when they first started out.
Rudra is also working on a successor to Unity7 called UnityX.

I'm also involved in the project as administrator and moderator in the various support media (Telegram, Discord and Matrix).

On a side note: Users who prefer to run Unity on top of Ubuntu simply can install it with: ```
sudo apt install ubuntu-unity-desktop
```
