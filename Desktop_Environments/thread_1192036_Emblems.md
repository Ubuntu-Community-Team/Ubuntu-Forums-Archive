### Post by asdfhjkla on 2009-06-19
To find new images to use as emblems, check out [ICONlook.com]("http://www.iconlook.com/popular.icon")

If I recall correctly (someone please correct me if I'm wrong) - emblems are set by the icon theme you are using. This can be changed through the Appearance settings window. You can get more icon themes [here]("http://www.gnome-look.org/index.php?xsortmode=high&page=0&xcontentmode=120x121&PHPSESSID=1ced0243bab3b01f8e976756fda82d85") if you'd like.

So to add more emblems to your theme, first find the directory of the icon theme (usually in /home/USERNAME/.icons/ or /usr/share/icons), then move the emblems into to the appropriate folder, which is usually named "emblems", for each icon size. You'll have to do this as super-user if it is in /usr/share/icons.
