### Post by AndyCooll on 2007-12-28
I had this problem too, i.e, no English Bible installed.

I eventually figured out how to fix this. Open up Gnomesword and go to Edit > Module Manager > SWORD > Configure. Then change the install source to remote (crosswire). In the same box click on Modules > Install/Update. You can then select Bible texts, dictionaries etc.to install. And of course I chose some of the English ones!
