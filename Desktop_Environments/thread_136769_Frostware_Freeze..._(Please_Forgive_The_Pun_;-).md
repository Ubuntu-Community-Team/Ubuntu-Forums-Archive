### Post by Zeroangel on 2006-03-12
I've had similar problems with Frostwire crashing.

Swiss, The kill command only does a 'soft' kill of the process, it tells the process that its time to shut down, but a failing process will not listen. So you are better off doing a 'hard' kill on both frostwire and java by using kill -s kill (processid) command.
