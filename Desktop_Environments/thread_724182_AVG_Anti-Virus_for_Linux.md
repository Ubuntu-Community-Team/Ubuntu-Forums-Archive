### Post by sonatso on 2008-04-15
The issue isn't whether there are viruses. There are two points regarding viruses on Linux:

1) Linux makes it inherently difficult for viruses to spread, because of how every user only has certain privileges. To make a virus infect the whole system, somebody would have to open a normal file as root. This makes virus spread difficult.

2) *Who cares?!* Unfortunately, Linux only has at most 2% of the desktop PC market. Any sane hacker, if he were so inclined, would create viruses for Windows of Mac.

So, I think most people can consider themselves fairly safe if they are running Linux. But that said, you should always be careful.
