### Post by strabes on 2007-07-18
You can install KDE and GNOME at the same time and just choose between the two when you log in. That way you can test out KDE without erasing gnome. I did this and switched to KDE fairly quickly after. I find it to be far more powerful and attractive than gnome. Just do this:

```
sudo aptitude install kubuntu-desktop
```or[code]sudo aptitude install kde-core[code]
