### Post by howefield on 2009-02-16
As said above, once you have the compizconfig-settings-manager installed, go to System > Preferences > Compizconfig Settings Manager option and set the following to get the 3d cube.

1. General options and select the number of horizontal desktops you require.
2. Activate Desktop Cube and Rotate Cube.
3. Go to "cube reflection and deformation". Set as required.
4. Spin the cube by pressing mouse scroll wheel and drag.
