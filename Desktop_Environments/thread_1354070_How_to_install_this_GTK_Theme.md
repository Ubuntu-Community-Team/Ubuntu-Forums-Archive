### Post by Raffles10 on 2009-12-13
Install pixbuf first:

```
sudo apt-get install gtk2-engines-pixbuf
```

Then right click on your desktop select Change Desktop Background, Theme, Install.. Then select your theme package. :D

### Post by akashiraffee on 2009-12-13

You can also change the color and make the taskbar transparent, etc.
