### Post by Jucato on 2006-02-26
Umm... In KDE, there are no "default" desktop icons (unlike Windows). You can make links on your desktop by right-clicking on the desktop and choosing Create New > Link to URL or Link to Application. 

However, mountable devices (drives, cd-roms, etc) can be enabled on the desktop by right-click on desktop  > Configure Desktop > Behavior > Device Icons tab, enable Show Device Icons, and choose which ones you like. Also, CD drive icons only show up on the desktop when  they are loaded w/ a CD.
