I would like Gaim to start only in system tray without its window popping up every time I log in into the computer. Anybody has idea?

---

### Post by flaming_monkey on 2005-10-18
The gaim-extendedprefs plugin allows you to control this.

```
sudo apt-get install gaim-extendedprefs
```

Load Gaim, choose Preferences, enable the Extended Prefs plug-in and then choose the option to hide the buddy-list on log-in.
