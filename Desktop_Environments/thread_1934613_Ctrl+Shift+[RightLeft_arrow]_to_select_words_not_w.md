### Post by Paddy Landau on 2012-03-03
It should work correctly.

I suspect that one of the Compiz settings has been set to note that combination.

Unfortunately, I know of no quick way to find it. Just go through all the enabled settings to search for that combination.

Before that, however, open Keyboard Settings and check if anything there is set to take Ctrl-Shift-Right or Left.
