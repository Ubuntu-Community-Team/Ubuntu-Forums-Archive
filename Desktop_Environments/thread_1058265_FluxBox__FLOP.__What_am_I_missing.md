### Post by lykwydchykyn on 2009-02-02
LXDE?  

Most lightweight WM's don't support native icons.  You can use rox-pinboard for the desktop, which supports some kind of drag'n'drop icon creation.  But nothing that I've tried is as straightforward from a user standpoint as the big DE's for creating icons.  It usually comes down to config files.

Arch is not a windowing environment, it's a distro.
