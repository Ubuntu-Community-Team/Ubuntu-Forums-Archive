### Post by jdong on 2005-03-24
I don't recommend you to install Firefox manually... A lot of Debian work goes into making sure that the Firefox install with integrate peacefully with the rest of the system. This includes making sure development headers end up in the right place, chrome registration via APT occurs properly, etc etc etc.

If you want newer Firefox, try the Ubuntu Backports Project ([http://backports.ubuntuforums.org/)](http://backports.ubuntuforums.org/)). Currently, I have the Hoary version of Firefox 1.0.1, and am building Firefox 1.0.2.
