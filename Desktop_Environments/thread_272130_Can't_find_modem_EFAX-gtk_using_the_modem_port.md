### Post by clarknova on 2006-10-05
> **Dutch said:**
> can I use the analog modem next to the ADSL network card?

This shouldn't be a problem, however many modems have no linux driver. See [http://64.126.95.102:8080/gromitkc/winmodem.html](http://64.126.95.102:8080/gromitkc/winmodem.html)

A good start would be to go into **System > Administration > Device Manager** and find out what kind of modem you have. If it's a connexant, you can test a driver from [www.linuxant.com](www.linuxant.com), then purchase it if it works for you.:KS
