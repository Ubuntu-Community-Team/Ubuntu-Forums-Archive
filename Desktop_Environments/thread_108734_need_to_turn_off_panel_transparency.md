### Post by NeoChaosX on 2005-12-27
Yeah, but the panel's missing, so he can't get to the K menu. 

taniwhaiti: You have to right-click on the desktop, select "Run Program" and run the command "kcontrol". That'll get you the KDE Control Panel, where you can disable the transparency on the panel. Once you've disabled that, just run kicker again.
