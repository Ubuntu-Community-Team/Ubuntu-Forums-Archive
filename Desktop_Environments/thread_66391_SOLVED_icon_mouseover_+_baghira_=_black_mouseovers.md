### Post by woahdae on 2005-09-17
When I switched to the baghira style, it made my kicker mouseover effects black blobs. I couldn't find the solution anywhere, and it was quite annoying, but after playing with it, it turns out you must uncheck "stipple on aqua mode" to "fix" (work around) it. Note that this option is set by default, as is aqua mode. Hope this helps someone.
