### Post by Giradman on 2007-11-14
I've noticed that page sizes & placements vary considerably w/ these various PDF readers, whether in Linux or Windows.  Check out the 'View' menu item - mine is set to page width, so the entire height of the page will not be included (of course this will also depend on the zoom or % level selected) - you can change the layout quite easily, e.g. to 'best fit' or 'dual', the latter for 2 pages; then, use the zoom in or out functions or the box in the tool bar that allows various % choices - guess I'm just use to changing these parameters, so not a big deal - but please report back if you find 'something' more interesting - thanks.  :)
