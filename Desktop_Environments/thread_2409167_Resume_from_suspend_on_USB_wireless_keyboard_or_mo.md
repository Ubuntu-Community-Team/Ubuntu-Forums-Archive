MoBo Bios set up to wake from sleep by Kb or mouse.

### Post by 7-123 on 2019-02-04
Replying to myself, it may help others.

The MoBo supports suspend so the system can wake by input from PS2 kb / mouse. No reference to USB ones.

Connected the usb adapters for wireless keyboard and mouse to the USB ports on the monitor (HDMI to the system)

Now it works fine, can wake from suspend to memory (sleep mode) by wireless keyboard or mouse.
