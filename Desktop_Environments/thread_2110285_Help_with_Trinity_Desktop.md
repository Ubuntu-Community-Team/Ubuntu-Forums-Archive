### Post by orb9220 on 2013-01-30
Read closer at the link as for adding the different repositories to your source lists 
Which in your case none for 12.10 only supported for 12.04.

As trying to backport and downgrading libraries and older packages will end causing breakage and an unstable system.

If you really want it then you will have to install 12.04 Ubuntu KDE.

At the bottom they do have liveCD Ubuntu with Trinity pre-installed.
