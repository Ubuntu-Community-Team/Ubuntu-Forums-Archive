### Post by fredbird67 on 2010-05-12
Well, never mind, I've figured it out on my own.  I discovered a little bit of good news and bad news.  First, the bad news:  I discovered that Enlightenment themes are compiled into machine-readable (and definitely NOT human-readable!) code -- no wonder Enlightenment is so dadgum fast!

But the good news is, I found a program in the Ubuntu repositories that can de-compile Enlightenment themes! :-)  I entered the command to do that, and VOILA!  I now have a directory in my downloads folder with all the parts of the Enlightenment theme I wanted to take apart and port to GTK.  That, of course, is something I'll get cracking on whenever I get the chance.
