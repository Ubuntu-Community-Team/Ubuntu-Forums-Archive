### Post by ohzopants on 2007-10-30
Beryl and Compiz are two very similar programs.  You only need one of the two.  From my understanding the "plugins" (or effects, or whatever you want to call them) are supported by both.  In 7.10 compiz is installed by default, so you probably do not need beryl.

You can enable desktop effects from the Appearance Settings dialog (System -> Preferences -> Appearance) on the visual effects tab.  If you want to customize which effects are turned on and their particular settings you should get the compiz config settings manager
```
 sudo apt-get install compizconfig-settings-manager 
```
