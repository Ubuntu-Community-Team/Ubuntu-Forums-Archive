### Post by Joe_CoT on 2006-11-06
I recommend [Beryl-Project](http://wiki.beryl-project.org/index.php/Install/Ubuntu) for their how-to guides (you'll notice that XGL guide is rather old). If you're using a not-ATI video card, using AIGLX is better in every way (it comes standard in Edgy Eft). I've had no issues with effects using their guides.

As for your F12 question, you can use beryl-manager (or compiz-manager, if you're using compiz for some reason) to change the key bindings.
