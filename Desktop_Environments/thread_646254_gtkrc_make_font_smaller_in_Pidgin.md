### Post by Tenken on 2007-12-21
You can write a config file named .gtkrc and put it in your pidgin config folder (called .purple in your home directory), I don't know much about gtkrc files, maybe [this link]("http://developer.pidgin.im/wiki/Using%20Pidgin") can help you out. To change Ubuntu's default fonts go to System->Preferences->Appearance under the "Themes" tab click the "Customize" button, and settings can be changes under Fonts.

Use the Extended Prefs plugin!

[https://sourceforge.net/projects/gaim-extprefs/](https://sourceforge.net/projects/gaim-extprefs/)
