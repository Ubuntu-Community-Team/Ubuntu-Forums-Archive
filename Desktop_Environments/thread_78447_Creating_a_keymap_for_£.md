### Post by wylfing on 2005-10-18
There is a thing called the "compose" key that does kind of what you're after. For example, you can hit <compose L -> and it will make £. From the main toolbar, choose System > Preferences > Keyboard and click the Layout Options tab to set the compose key. I can't tell you what all the combinations are, but they are pretty intuitive, and some Googling should turn up a reasonable list.
