### Post by mcduck on 2005-09-06
From what program ar you trying to import the bookamarks (favourites would point to IE)..

Instead of trying to copy some folder, you should export bookamarks from program itself, so you'd get a nice text file that you can import to Firefox. 

If you used FF in XP, boot to Xp, start firefox and go to Bookmarks/manage bookmarks and in File-menu you have both Import and Export functions..

I think there is also a similiar function in IE, and Outlook also, so you really shouldn't need any extra programs or special tricks to move your bookmarks and mails to new programs..

### Post by madjo on 2005-09-06
it is also possible to just grab the bookmarks.html file from your firefox profile on the Windows XP partition, and copy that to your firefox profile in Linux, overwriting the default bookmarks file. :) (I did this procedure, and it works for me perfectly) 
or import that file through the method mcduck explained.

the first method will overwrite any bookmarks you have already made in Linux' firefox, the second method keeps those bookmarks :)
