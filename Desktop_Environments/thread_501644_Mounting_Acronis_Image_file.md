I doubt that you can.  I use (what used to be) PowerQuest Drive Image, which is a very similar program, and those pqi / v2i files are not mountable, because Ubuntu does not recognize the file type.   

You might wish to try running your Image Explorer program - whatever you use to open, browse and mount your images - under Wine.   If you can export your image as an ISO file, then mount the ISO, that might be another option ..
