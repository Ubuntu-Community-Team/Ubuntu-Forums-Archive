### Post by lodp on 2007-01-27
that's something that's been bugging me also -- found out how to do it now:

set the view mode the way you want it, then hit Settings>Save View Profile

there definitely should be some reference to this in the Preferences tab ... many people seem to overlook that in the settings menu.
