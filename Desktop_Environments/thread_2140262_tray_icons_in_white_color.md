### Post by Frogs Hair on 2013-04-29
I read on a another forum that Dropbox uses its own icon unless the set you are using includes a Dropbox icon . As for the others the mono dark icons tend to be lighter for dark panels and mono light are darker for light panels if they are displaying properly. If using is own icon the icon may be stored in a folder other than usr/share/icons.
