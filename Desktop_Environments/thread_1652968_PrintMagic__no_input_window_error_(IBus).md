Seasons greetings! I was looking for some software that would let me make Japanese new year cards and found the seemingly brilliant PrintMagic. 

[http://www.print-magic.net/](http://www.print-magic.net/)  (Japanese language warning)

Anyway, all seems perfect, with cross-platform compatibility, etc, apart from Ibus (Anthy) not allowing me to input Japanese characters. The software required the installation of AdobeAir, which I suspect probably has something to do with the error it gives: No Input Window 

Of course, IBus works fine with all my other applications.

Any ideas/thoughts on solving this would be appreciated.


EDIT: Solution is here: [http://kazuspara.blog134.fc2.com/blog-entry-56.html](http://kazuspara.blog134.fc2.com/blog-entry-56.html) 

It allows for IBus input using AdobeAir using 64 bit. (Yay! Now onto making my New Year's cards)
