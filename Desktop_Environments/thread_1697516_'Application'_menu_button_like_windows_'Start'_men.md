### Post by Tamlynmac on 2011-02-28
I believe your requesting a menu button similar to the start menu in Windows.

If so, try the main menu in panel applets.

1. Right click on the panel you want the menu in.
2. Select "Add To Panel"
3. Select "Main Menu" from the popup window and left click on the "Add" button on the bottom of the window. The main menu icon should appear in that panel.

One nice thing about the main menu is you only need one menu button for all apps and admin/system.

Good Luck & hope this helps.

If you want something a little more versatile like the Windows Vista and 7 menu, look into [GnoMenu]("http://gtk-apps.org/content/show.php/GnoMenu+-+consolidated+menu+for+gnome?content=93056") or [Cardapio]("http://www.omgubuntu.co.uk/2010/06/cardapio-alternative-gnome-panel-menu-launcher-so-much-more/").
