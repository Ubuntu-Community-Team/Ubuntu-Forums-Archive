### Post by keypox on 2009-04-05
yes you have to use compiz.  Which is installed by default.  

Download compiz config settings manager via synaptic.   Then use it system>preferences>compizconfig settings manager.  Then pick the scale settings and set it to F9...

though i like ot mouse to button right corner for mine. 

Ubuntu does everything osx does plus much more.  IE: Scale=expose, expo=spaces, avant=dock and so on.
