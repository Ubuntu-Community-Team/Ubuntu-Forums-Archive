---

### Post by 23meg on 2005-11-07
You can't. If you must do it with Nautilus, launch it as root with ```
sudo nautilus
```, delete the file and close the window.

### Post by 23meg on 2005-11-07
No problem. Be careful when using Nautilus as root though; you can break things if you wander around system folders carelessly. I'd suggest decorating the root nautilus window with a different background to keep yourself aware, and closing the window whenever you're done with it.
