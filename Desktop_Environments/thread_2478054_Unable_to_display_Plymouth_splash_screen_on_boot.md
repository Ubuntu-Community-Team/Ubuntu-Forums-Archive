### Post by #&amp;thj^% on 2022-08-19
Just a heads-up there was a upgrade to grub on my Debian Rolling.
But for now:
Assuming that this is Ubuntu 22.04...

From the GRUB menu, choose Additional Options, then Recovery Mode, Root Access.

At the # prompt:
```

mount -o remount,rw /

sudo update-alternatives --config default.plymouth
```
from there 1st Choose the right "logo" screen, 2nd and very important to then repeat the command with the** bgrt choice**. (it will show in that window) see screenshot....
Now run:
```
sudo update-initramfs -c -k $(uname -r)
```
Reboot, if this don't work, I'm just stumped.
