### Post by Xian on 2005-06-16
Generally the best thing to do is untar those files and place the theme folder contained therein into either your /home/username/.themes directory or the /usr/share/themes path if you want global access. They may not appear on the main theme window of the gnome-theme-manager, but you should be able to access them when clicking on the 'Theme Details' option and scrolling in either the 'Controls' or 'Window Border' modules, Select what you like (or just mix and match) and then save your theme from the main window using whatever name and description that you prefer.
