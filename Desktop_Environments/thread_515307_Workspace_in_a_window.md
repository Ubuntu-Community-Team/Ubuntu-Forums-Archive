### Post by obrie on 2007-08-01
Is there a way to create a window which acts as a workspace?  For example, rather than switching the entire screen when switching workspaces, I'd like to just have a window, which I can minimize and maximize, that acts as a workspace (as if I were logged in remotely) nested in my current workspace.

Thanks for any insight.

### Post by weblordpepe on 2007-08-02
You can group windows together and maximize/minimize them at the same time. Thats the closest thing that I'm aware of.

Otherwise you're 'stuck' with having multiple workspaces that you can switch between. I dont think many people thought of multiple workspaces as being nestable.

### Post by Jhongy on 2007-08-02
Compiz Fusion can group windows from any number of apps like that, and you can tab between them. You can use the mouse for thumbnail previews of each grouped window,  or use a keyboard shortcut to switch between each group member. I've found it to be very stable. (Plus, of course, it looks neat as hell)

(another kludgy alternative: A virtual machine pulling app data from a shared directory. Although you can't drag n' drop applications to/from the VM)
(even more off the wall... a "remote desktop" (FreeNX) connection to your own computer! Again, can't drag/drop though).
