### Post by RikoW on 2006-08-25
Sorry, maybe that was not clear.

I have a dual boot set-up on my laptop: Windows XP and Ubuntu. In 98% of the cases I boot into Ubuntu. With VMware workstation you are able to boot the XP which is physically installed on the machine (so one of the dual boot OSes). That's what I call the physical install. THere are instructions on the VMware website how to do that. It's not entirely trivial. If you are interested I can dig out the link.

The actual idea behind VMware of course is not the above approach, but to install the guest OS inside and with VMware. This installation you will only be able to run within VMware but not stand alone.

Hope that clarified it :)

                 Riko
