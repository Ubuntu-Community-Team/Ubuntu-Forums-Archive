Hi,

I've just been trying to install the Humanity theme from 9.10 into 10.4, and since I found the solution I thought I'd post here. I didn't find the answer via google, so hopefully this thread will help some others.

Just install the human-theme package:
```
sudo apt-get install human-theme
```Now you'll see Human and Human-Clearlooks in System -> Preferences -> Appearance -> Theme.

Hope this is useful!

Cheers
