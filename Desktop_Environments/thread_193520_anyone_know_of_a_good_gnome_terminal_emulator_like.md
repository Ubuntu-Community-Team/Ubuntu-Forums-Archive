### Post by em3raldxiii on 2007-10-14
I tend to agree that Yakuake is substantially more professional looking than Tilda.  In addition, I have found it to be generally more stable; when I activate Tilda, I often get a useless yellowish blank window, and it is possible to accidentally start more than one instance of Tilda.

Yakuake looks clean, cool, and professional.  It comes with a number of handy customizations, and it runs very stably, even on the Gutsy Gibbon Beta that I am currently running.

The net result:  I don't know why more people aren't using it!

Here are the only gripes I have with Yakuake.  #1 it is a KDE native app, and therefore some of the color and customization options don't apply if you are using it with an alternative desktop environment.  #2 I would like to be able to turn off the bottom tabs so that I have a cleaner looking terminal.  #3 I would like to be able to associate a system sound (like an unobtrusive swish or something) to activation of the terminal window.

For those new to using a Terminal Emulator:  You have to try it!!  I believe it should come standard with an Ubuntu install.  No more clicking on menus to run the Terminal!!  Also, depending on your current flavor of Ubuntu and/or whether you've experimented with some of the new "bling" features (such as compiz), you can *very* easily make Yakuake transparent (even in gnome) by holding ALT and moving your scroll wheel up or down.

To install Yakuake, you can use Synaptic Package Manager (or other package manager) and search for Yakuake, or you can simply run this command line:

[COLOR="Red"]sudo aptitude install yakuake[/COLOR]

Once installed, have it automatically start with your login by (in Gnome anyway) going to [COLOR="Red"]System > Preferences > Sessions[/COLOR] and adding [COLOR="#ff0000"]yakuake[/COLOR] to the list of Startup Programs.

I recommend using ALT-` as your activation keys.
