### Post by openbox on 2007-10-08
you're using fglrx driver instead open source radeon or ati. Try to remove fglrx:
```
$ sudo dpkg -P xorg-driver-fglrx
```

Then in the end of your xorg.conf file, add this:
```

....
Section "Extensions"
        Option "Composite" "Enable"
EndSection

```

This make your desktop using AIGLX then XGL
