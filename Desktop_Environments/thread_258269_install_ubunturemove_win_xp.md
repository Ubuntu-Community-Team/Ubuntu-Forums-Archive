### Post by bribri124 on 2006-09-15
It might be best to remove your files first before installing Ubuntu.  That way, you can remove the Windows partition, and just mount the media and move the files without any worries.  If you want to, or have to, move the files after the Ubuntu installation, you should get a mounted folder of the Windows partition on your desktop.  Once you are completed moving the files, go to Disks under System -> Administration.  You can also install qtparted.  Take a look at this article.

[http://www.ubuntuforums.org/archive/index.php/t-222025.html](http://www.ubuntuforums.org/archive/index.php/t-222025.html)

### Post by bribri124 on 2006-09-15
If you are able to load Windows, you can remove your files then.  If not, then install Ubuntu on a separate partition.  I guess it all depends on your experience working with Linux/Ubuntu.  Removing the files first would be the safer way, while installing Ubuntu first and removing the files on the mounted Windows partition would be more fun and give you the chance to play around with your new system, if this is something you never have done before.
