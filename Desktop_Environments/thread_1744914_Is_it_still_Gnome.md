### Post by 3Miro on 2011-04-30
No, you cannot uninstall Gnome.

Ubuntu 11.04 runs Gnome with custom interface called Unity. Many people wrongly believe that it is a new environment or that Ubuntu is moving away from Gnome, but this is not the case. Unity is Gnome, just a different shell/UI. Furthermore, it is still Gnome 2.32, Ubuntu 11.10 will come with Gnome 3.
