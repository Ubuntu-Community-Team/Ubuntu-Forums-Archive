### Post by izizzle on 2009-06-07
Hi. I currently use the Aya theme for Plasma, and I really like it except the fact that it does not provide transparency for my widgets. I was wondering if I could use The [Seamless]("http://www.kde-look.org/content/show.php/seamless?content=102237") theme with the widgets and Aya for everything else?

Thanks.

---

### Post by txcrackers on 2009-06-07
System Settings -> Advanced -> Desktop Theme Details

Once it's set to your liking, right-click on the desktop, select Appearance Settings. Set your Desktop Theme to (Custom) and you're good to go.
