> **sentinelace said:**
> every time I do something I have to type in the password.  Can I remote this so I never have to type in a password?

whenever you carry out an Admin task it should ask you for a password where you provide your own password to gain SUDO privelege. (if you are an admin/sudo user)

This is for your own benefit, if you disable this requirement then you open yourself widely to risk as you are effectively doing everything as root.

Imagine browsing the internet and running a script in your browser without your knowledge and it now runs with root privelege ?

not good.

So it is not recommended or condoned.

read all you need here [ROOTSUDO]("https://help.ubuntu.com/community/RootSudo")
