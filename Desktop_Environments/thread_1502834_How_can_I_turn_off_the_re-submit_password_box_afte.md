### Post by Linye on 2010-06-06
I guess you refer when the screensaver appears. If so, go to Preferences > Screensaver and check "lock screen when screensaver is active"

---

### Post by Kerie on 2010-06-06
Thanks for the steer, Linye.  What I found was  that option to "lock screen when screensaver is active" was already ticked. What has worked however, is de-activating the screensaver altogether. So now as long as I don't use screensaver -  problem solved.
