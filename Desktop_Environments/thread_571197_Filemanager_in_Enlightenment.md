### Post by blackened on 2007-10-09
Been a while since I've used E, so I'm not sure about how to set the default browser, but as far as the file manager goes, if you have Gnome installed then you can create a launcher or menu entry for Nautilus with ```
nautilus --no-desktop
```. You could also try rox, thunar, or xfm. All of which are in the repos.
