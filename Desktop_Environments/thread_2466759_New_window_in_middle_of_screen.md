Hi. Is there a way to get the desktop to place a new window in the middle of the screen, not top left?

Running Ubuntu MATE 20.04.

---

### Post by mikewhatever on 2021-09-05
There is a setting for that in MateControlCenter, Look and Feel Section, Windows icon, Placement tab, Center new windows checkbox.

That is a lot of clicks, so it is possible to open directly from a terminal with "mate-window-properties".
