### Post by the most naive user on 2011-10-15
> **CraigGB said:**
> I fixed it!, I needed to install ccsm, run it and re enable the unity plug in :), now its working fine again

The new value for the key binding for the action Key to start the switcher in reverse in plugin Ubuntu Unity Plugin conflicts with the action prev_key of the Static Application Switcher plugin.
Do you wish to disable prev_key in the Static Application Switcher plugin?

because of this conflict, unity was disabled.
