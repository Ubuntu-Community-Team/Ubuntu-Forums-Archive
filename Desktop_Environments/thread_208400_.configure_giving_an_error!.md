### Post by krye on 2006-07-03
1) Yes you need to install kde to be able to log into it. Unless you already have it installed. You can do it by searching for kubuntu-desktop on synaptic, or using

```
sudo apt-get install kubuntu-desktop
```

on a terminal.

2) That Xdbe looks like some X11 development package, you can try making a search for it using 'sudo apt-cache search xdbe' on a terminal or synaptic.

You can also try and follow some of the how-to's here in the forums to get conky work for you.

Good luck!

### Post by aysiu on 2006-07-03
Please use *aptitude*, not *apt-get* to install KDE: ```
sudo aptitude update
sudo aptitude install kubuntu-desktop
```

Full instructions here:
[http://www.psychocats.net/ubuntu/kde.html](http://www.psychocats.net/ubuntu/kde.html)
