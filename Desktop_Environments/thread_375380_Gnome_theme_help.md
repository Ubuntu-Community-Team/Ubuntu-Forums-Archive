It's /usr/share/themes for all users of the system, and ~/.themes for only current user :D

Anyway, the important ones are GTK2-themes (for applications), Metacity themes (for window frames) and icon themes (do I need to explain this ;)).

GDM themes are for GDM, the login screen where you input your user name & password. These are installed through System/Administration/Login Screen.
