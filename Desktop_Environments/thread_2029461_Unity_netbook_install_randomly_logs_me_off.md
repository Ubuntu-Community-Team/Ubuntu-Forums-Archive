### Post by jspike397 on 2012-11-21
A while later, I fixed this thread.  I am running on a slow netbook, which I really push to the limits.  Now, what I do is I have a CPU monitor at the top of the screen and before I sleep my computer I make sure that it is not using 100% of the CPU when sleeping so that when I bring it back up, I do not overload it and cause it to log out.

Hope this helps anyone.
