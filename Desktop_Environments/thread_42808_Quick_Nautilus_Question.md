### Post by hw-tph on 2005-06-19
[QUOTE=NotSoSuperUser]Hello

Cheers

 Run gconf-editor (Applications --> System --> Configuration Editor), open /apps/nautilus/preferences and check the always_use_browse check box. That's it.
