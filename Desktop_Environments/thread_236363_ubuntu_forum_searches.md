### Post by Cable on 2006-08-14
If you look at what aysiu posted, the site that Google should search through is specified before the search criteria.  using the *site: *"tag", you can give Google a specific domain to search so as to limit the results you get.  So, to search this forum (ubuntuforums.org), you would type
```

site:ubuntuforums.org

```

Then after that, put a space, then (in quotes) the phrase you are searching for.  That will limit your results to only instances of the **whole** phrase.
