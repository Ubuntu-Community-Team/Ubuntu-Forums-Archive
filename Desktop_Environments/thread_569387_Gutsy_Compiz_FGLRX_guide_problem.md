### Post by hebetude on 2007-10-07
fixed by doing:

sudo apt-get remove linux-restricted-modules-generic restricted-manager
sudo apt-get install linux-restricted-modules-generic restricted-manager
sudo apt-get remove xorg-driver-fglrx
sudo depmod -a

I then restarted X and compiz worked
