### Post by Copper Bezel on 2011-11-10
You shouldn't have to locate the file again from Gnome Tweak Tool. It should just be in the drop-down menus for Window Theme, GTK Theme, etc. The "Install" button is only needed if you haven't already installed it manually (by moving it to ~/.themes); all you need to do after that is select it.

If the theme doesn't have both a GTK2 and a GTK3 theme in the folder, though, it won't show up under the GTK Theme menu, so be mindful of that.
