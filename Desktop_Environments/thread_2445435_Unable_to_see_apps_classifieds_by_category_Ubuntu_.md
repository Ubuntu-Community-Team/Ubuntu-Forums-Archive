### Post by Dennis N on 2020-06-14
Gnome does have menus that show the applicatons in categories. These are installed as Gnome shell extensions which you install from your web browser. Look here:
[About Extensions]("https://extensions.gnome.org/about/")
[Find and Install Extensions]("https://extensions.gnome.org/")

Two Suggestions:
**Applications Menu** by fmuellner
**Arc Menu** by linxGem33

It's also possible to make folders of categories within the "Show extensions" display with AppFolders, but that is another topic entirely that you would need to read about.
