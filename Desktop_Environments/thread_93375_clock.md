### Post by not_yet on 2005-11-22
on the toolbar right click the clock and choose adjust date/time

select your time zone

click synchronize now

if no clock is shown in the toolbar right click the toolbar and add it from the list

for mp3's I use xmms

install it using synaptic

system/administration/synaptic package manager
