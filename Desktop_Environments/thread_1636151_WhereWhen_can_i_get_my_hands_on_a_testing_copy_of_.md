### Post by Liverbones on 2010-12-02
There is a Launchpad PPA for the GNOME 3 desktop, but of course, GNOME 3 is still heavily under construction. I certainly wouldn't recommend installing it on a production computer, or on a computer that you can't live without.

Now, I haven't tested GNOME 3 myself. I have no idea how well it will work for you, but you can get it by typing the following:

```

sudo add-apt-repository ppa:ubuntu-desktop/gnome3-builds
sudo apt-get update && sudo apt-get install gnome3-session

```

But really, you should probably look into this more before actually doing it. Be careful. :)
