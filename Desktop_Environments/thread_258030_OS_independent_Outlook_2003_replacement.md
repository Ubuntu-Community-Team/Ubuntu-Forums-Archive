### Post by sawjew on 2006-11-28
Did you ever get a reasonable solution to this?  If not have you tried Thunderbird with the Lightning calendar extension?

Thunderbird can be set up to share between Windows and Linux (or Mac I would think) as described in this Howto [http://www.ubuntuforums.org/showthread.php?t=203524&highlight=thunderbird+share]("http://www.ubuntuforums.org/showthread.php?t=203524&highlight=thunderbird+share")

The Lightning extension is available from here [http://www.mozilla.org/projects/calendar/lightning/]("http://www.mozilla.org/projects/calendar/lightning/")

This provides a pretty full featured email, calendar, task list, etc. and Lightning is adding more features all the time.  I have been successfully sharing between a dual booted Windows 2000/Xubuntu system on my laptop for a while now.
