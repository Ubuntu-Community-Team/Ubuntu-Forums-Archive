### Post by fdoving on 2006-04-14
Go to System Settings -> Sharing -> Remote Desktop to enable and manage the server.

If I remember correctly this server is fully VNC compatible so you can use the VNC client you prefer. The KDE client is named 'krdc'. ALT+F2 'krdc' <enter>. If you can't find a menu entry for it.


Hope this post is useful.

- Frode
