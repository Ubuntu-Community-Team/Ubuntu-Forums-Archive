### Post by Krytarik on 2011-01-22
It seems that the option to add/switch to another keyboard layout isn't included in the XFCE desktop by default. Try this app: xfce4-xkb-plugin, to install it via command line:
```
sudo apt-get install xfce4-xkb-plugin
```
