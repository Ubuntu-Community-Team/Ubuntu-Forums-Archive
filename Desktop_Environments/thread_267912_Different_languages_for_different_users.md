### Post by FISHERMAN on 2006-09-29
on the log-in screen you can choose between all the installed languages and if choose a differant language it will ask if you want to make it standard for that account.
So just set up Norwegian Bokmal-system, but when the user of the English account logs in he/she should first chose English in the language menu on the log-in screen.
