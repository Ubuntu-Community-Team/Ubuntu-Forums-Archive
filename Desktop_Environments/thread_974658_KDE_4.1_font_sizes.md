### Post by kernelhaxor on 2008-11-10
To fix the font sizes in GTK apps like Firefox and Thunderbird, go to:
K-menu -> System Settings -> Appearance -> GTK Styles and Fonts (in the left panel) -> you will see a section called 'GTK Fonts' on the right .. where you can change the font and font size ..

Let me know if you experience any problems while doing this.  Hope that helps
