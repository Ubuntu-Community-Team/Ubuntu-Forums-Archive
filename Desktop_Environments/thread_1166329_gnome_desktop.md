### Post by coffeecat on 2009-05-21
Open Synaptic and install 'ubuntu-desktop', or...

```
sudo apt-get install ubuntu-desktop
```It's a meta-package which will pull everything in that you need for Ubuntu gnome. You will then be able to select gnome at the login screen.
