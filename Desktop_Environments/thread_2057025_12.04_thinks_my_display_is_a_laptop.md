### Post by RHAnthony on 2012-09-13
Got it!  Found the answer online in an old blog post but it still holds up as a valid fix for this (or updating nvidia correctly in general.  Downloading the driver and following nvidia's instructions on their own site, did not do it correctly.

```

sudo apt-add-repository ppa:ubuntu-x-swat/x-updates
sudo apt-get update
sudo apt-get install nvidia-current
reboot

```
