### Post by zjagannatha on 2010-02-15
I'd like to add support for Roll Left and Roll right. (Similar to the [Opera Gestures]("http://www.opera.com/browser/tutorials/gestures/"))

It seems that whenever that key combination appears, it always does something unintended. We could add a feature in the Mouse settings menu.

I can see many applications for such a binding such as shifting left or right in a workspace interface (such as the Cube with [CompizFusion]("http://www.compiz.org/")). It could also be used as undo/redo within office environments or Back/Forward within browsers.
