### Post by bluenova on 2006-05-08
If you have no data to lose on the drive, just reinstall and select to auto partition. The setup will do it all for you. If you don't want to format the drive, and just want to resize partitions then you would need to run the live cd version and use gparted to resize (delete the others and resize the main one).
