### Post by ash211 on 2006-08-25
If X doesn't start at all, you could try using the generic vesa driver by changing the driver in this section
```
Section "Device"
Identifier "Generic Video Card"
Driver "nv"
EndSection
```
to vesa:
```
Section "Device"
Identifier "Generic Video Card"
Driver "[COLOR="Red"]vesa[/COLOR]"
EndSection
```

That would allow you to get back in the system with a GUI and have a better chance at getting the nvidia drivers working again.
