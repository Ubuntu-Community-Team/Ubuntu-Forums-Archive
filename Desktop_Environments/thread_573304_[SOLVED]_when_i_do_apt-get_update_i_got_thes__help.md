### Post by safaricity on 2007-10-11
terminal:

sudo apt-get install gui-apt-key -y


this will install a GUI to be used to add gpg-keys to your keyring, which is the authentification method used in ubuntu package management.

after you have this application installed through synaptic you can copy the errors synaptic spits after reloading package information, and simply use gui-apt-key to add the missing keys.
Then entry to be chosen is the number in the gpg-error. Hope that helps you to get rid of the errors/warnings.
