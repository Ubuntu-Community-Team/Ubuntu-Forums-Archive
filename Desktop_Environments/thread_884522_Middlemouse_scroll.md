### Post by mali2297 on 2008-08-09
You can change the behavior in Firefox; go to *about:config* and set *general.autoScroll* to **true**.

However, you might want to get accustomed the Linux way of using the middle mouse button to paste from the clipboard; you copy text by highlighting it and then paste it with a middle-click. This works in and between all applications.

In Firefox, if the text in the clipboard is a http address, then you can middle-click anywhere on a page and it will go to the address. (That is, if you haven't changed the default setting.)
