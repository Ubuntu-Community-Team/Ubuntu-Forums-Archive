### Post by christoph411 on 2011-06-11
If you are still interested in using unity, but don't have any 3D acceleration available, you should try unity 2D! Unity 2D is almost exactly the same as unity 3D, but with a few less "bells and whistles" (so to speak :p )
Just search "unity 2D" in the software center, install it, and once it's done installing, log out, and on the bottom bar in the login screen choose "Ubuntu 2D" , log back in and your enjoying unity 2D! :D  
(please note, that unity 2D may not be as complete as unity 3D in some aspects, at least for now!)  :popcorn:
