### Post by grumblebum2 on 2014-06-22
I found flash works best in google-chrome as it's the most up to date flash.
Chrome uses Pepper Flash Player plugin.
Adobe has only supported Pepper plugin APIs on Linux for about a year now.

non-pepper flash only gets security updates.
