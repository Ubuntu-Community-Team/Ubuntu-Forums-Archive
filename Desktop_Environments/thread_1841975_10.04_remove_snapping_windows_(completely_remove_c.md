### Post by Krytarik on 2011-09-10
LOL:D By removing Compiz you ended up using Metacity's snapping windows feature! And - as opposed to Compiz - it doesn't seem to be possible to disable this in Metacity at all.

So, install Compiz again, also install CompizConfig Settings Manager if it isn't already:
```
sudo apt-get install compiz compizconfig-settings-manager
```And then just disable "Snapping Windows" in "System -> Preferences -> CompizConfig Settings Manager -> Window Management".

Greetings.
