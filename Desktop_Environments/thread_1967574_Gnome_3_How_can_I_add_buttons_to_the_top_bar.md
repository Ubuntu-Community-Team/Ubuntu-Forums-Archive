The best way to customize Gnome Shell is by the use of extensions.  Just go to extensions.gnome.org for an easy way to install extensions.  There are extensions to add a menu and many other things that you may want to do. 

It should be possible to launch the application then go down to docky and right click on the application to pin it to the dock, although I could be wrong.

I'm not familiar with docky but there are a couple extensions that might be useful.  For application launching I use Favorites menu by cvine and Frippery applications menu.  Frippery has a favorites extensions which will put icons from favorites on the top bar.  If you have a lot of favorites the top bar can get pretty crowded.  Another must-have for me is Alternative Status Menu.  That gets me separate suspend and power off entries on the upper right corner menu.  I'm not sure why Gnome thought it was a good idea to not have a power off entry, requiring instead the alt key combination.  Not all extensions work with gnome 3.4 which is default on 12.04.  There are about 17 pages of extensions total and about 8 pages work with 3.4.  Presumably others will be rewritten to work with 3.4 in time.

Edit:  These extensions can be found at extensions.gnome.org
