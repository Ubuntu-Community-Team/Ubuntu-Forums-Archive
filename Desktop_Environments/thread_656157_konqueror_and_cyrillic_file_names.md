### Post by jamaisvu on 2008-01-02
[FONT=Franklin Gothic Medium]Press ALT-F2. Type in [FONT=Courier New]kcontrol[/FONT] (and press enter). Select from the tree in the left pane, Appearance & Themes&#8594;Fonts, then change the fonts in the right pane to ones that include Cyrillic. If you don't know which ones do, look at table 4 of various fonts in KCharSelect (ALT-F2, [FONT=Courier New]kcharselect[/FONT]).

Hope this works![/FONT]
