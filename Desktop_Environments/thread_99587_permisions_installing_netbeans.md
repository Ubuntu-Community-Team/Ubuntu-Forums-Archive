### Post by professor_chaos on 2005-12-05
IF a file is non-executable on a cd, then there is no way to execute it by changing permissions, because it a non writable medium. If its a shell script, you can execute it from the command line by typing "source name_of_shell_script". This makes the shell execute the file line by line. Otherwise, you will probably need to copy the contents to your harddrive and then modify the permissions from there.
