### Post by sayakb on 2008-08-03
> **yo2lux said:**
> I hear kubuntu is bad, not a quality linux.
KDE is an excellent desktop environment. Kubuntu is just Ubuntu + KDE. It is as good as Ubuntu with Gnome. KDE infact has an excellent group of apps (like Amarok, K3B, Konqueror)

Though, KDE4 is a bit buggy (though 4.1 fixes quite many of them), while kde 3.5.9 is very stable. 
To install KDE4 on ubuntu, at a terminal:
```
sudo apt-get install kubuntu-kde4-desktop
```To install KDE3.5:
```
sudo apt-get install kubuntu-desktop
```

Now to log into KDE, at the login window, press F10, click **Select session** and select KDE.
