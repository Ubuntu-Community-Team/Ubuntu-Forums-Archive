### Post by FIONEX on 2009-01-16
No no, you can run both KDE and gnome on one installation of ubuntu.  You were on the right track!  You just select "session->KDE" on the login screen, which tells GDM which desktop manager to start when you login.  Your user account doesn't default to gnome.  When it asked you what you want the default session to be and you said gnome, then every time GDM starts up and you login, it'll go to gnome.  So you have to select Session->KDE before every login now.  You can change this if you want (let me know if you want to).
