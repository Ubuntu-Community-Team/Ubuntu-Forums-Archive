### Post by mlind on 2006-06-27
/etc/mailcap is automatically generated by update-mime script.
You should read update-mime manual page

The place you should look for package which is adding unwanted mime type is
/usr/lib/mime/packages and remove it


You should be able to override this behaviour using local
~/.mailcap file too
