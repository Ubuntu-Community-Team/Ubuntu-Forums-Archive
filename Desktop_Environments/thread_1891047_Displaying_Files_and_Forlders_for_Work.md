### Post by ManualSparrow on 2011-12-04
In generic Ubuntu the file manager is called Nautilus. To make sure that's what you're talking about, hit Alt-F2 to bring up the run dialog and type ```
nautilus
``` and hit enter. Is that what you're looking for, or did you already see it and just find it lacking?
