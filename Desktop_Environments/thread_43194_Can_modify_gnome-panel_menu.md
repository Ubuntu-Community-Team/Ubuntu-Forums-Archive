Q: How to install Menu Editor for GNOME?

   1. Read [General Notes](http://ubuntuguide.org/#generalnotes)

   2. How to [add extra repositories?](http://ubuntuguide.org/#extrarepositories)

   3. [SIZE=3]***sudo apt-get install smeg***[/SIZE]

   4. Read [How to refresh GNOME panel?](http://ubuntuguide.org/#refreshgnomepanel) 

   5. Applications -> System Tools -> Smeg Menu Editor
