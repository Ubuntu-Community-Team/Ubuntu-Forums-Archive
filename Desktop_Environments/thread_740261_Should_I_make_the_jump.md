Now, I am not sure if I should make the jump, so I will point the pros and cons of each one and you help me take a decision....

KDE

    Pro:
       
        Applications integrate nicely
        Has tons of apps
        Nice GUI
        I have all my stuff configured: IM, Music, mail, etc
        Has amarok

    Con:

        Takes too much resources
        The kubuntu variant IMO is not very well supported by the community
        Gutsy will be the last ubuntu using KDE 3.5.x
        Does not integrate well with non KDE apps
        Messes up firefox appearance
        Konqueror does not work with flash based sites anymore


GNOME

    Pro

        Nice GUI
        Clean, uncluttered interface
        Runs fast, resource light
        Main variant support from the community
        Integrates well with firefox
        
    Con

        I should configure everything: mail, torrents, IM
        Has no amarok

Now, if there was a way for me to import all my configurations from KDE to gnome, it would rock... so if you know any way to migrate my configurations, it would be ideal
