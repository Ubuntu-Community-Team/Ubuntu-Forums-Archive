### Post by VCoolio on 2009-09-30
Emerald is only a window decorator, which rules your window borders and shadows. The rest inside the windows (in appearance > customize it's called: controls) is still gtk, so you'll need to configure your gtk-theme for window background and text colors (unless you mean the window titles).

### Post by thomasboleyn on 2009-09-30
Emerald is extremely easy to customize, though it requires a little more memory to run. It works out great if you're using xfce or something else lightweight, but with gnome I find it makes the compiz animations too sloppy for me to use it all the time in conjunction with gnome-do docky or AWN. A good rule of thumb for me is that if firefox can minimize/maximize while on a page with flash without stuttering or the animation looking horrid, then the setup is a keeper.

Also a lot of the emerald themes on gnome-look look absolutely awful. :P
