### Post by Bob D. on 2005-05-16
Would filters do what you want? Since Evolution uses a global inbox, I don't know a way to set them up completely separate as Thunderbird does (Though I understand they added a global inbox option).

I have my 3 POP accounts all reeding into the global inbox and then I've set up filters to separate items. Several Yahoo Groups I subscribe to each have their own sub-folder under the main inbox that mail is filtered into. Same for UbuntuForums.

I think this might let you do what you want.
