### Post by lagagnon on 2006-09-25
How much RAM do you have? If less than 512MB you will find KDE and/or Gnome very slow on almost any machine. That is a fact of life. If you don't like xfce then none of the other even lighterweight window managers are unlikely to satisfy you.

I believe only a limited amount can be done to speed-up Gnome. The best thing to do is work on decreasing RAM useage. That means using lighterweight applications, removing unused services, and using a lighter weight window manager.
