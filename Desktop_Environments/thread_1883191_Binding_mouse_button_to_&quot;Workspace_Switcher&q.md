I have a Logitech Performance Mouse MX.  This mouse has 9 buttons, which currently only 3 of them work.  How can I bind a specific one to a specific task?  Specifically, I want to bind the thumb button to the Workspace Switcher so when I click it, I see all 4 desktop.  Though, knowing how to apply other bindings wouldn't hurt either....

Thanks!

### Post by nilarimogard on 2011-11-18
For this you need to use CompizConfig Settings Manager - install it using the following command:
```
sudo apt-get install compizconfig-settings-manager
```

Then launch CompizConfig Settings Manager, search for the "Expo" plugin and under the "Bindings" tab, set the mouse button you want to use next to the "Expo button".
