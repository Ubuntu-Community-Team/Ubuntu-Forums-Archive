Ok, here's the scoop - you can't remove it yet, but you will be able to soon.  The next version of gdesklets should have the ability to remove the systray icon.

If it really bugs you, you can uninstall your current gdesklets and install the unstable gdesklets version that's in cvs... but there is no guarantee that it will compile and run on your computer - nor is there any guarantee that the current gdesklets you're using will work on the new version either.
