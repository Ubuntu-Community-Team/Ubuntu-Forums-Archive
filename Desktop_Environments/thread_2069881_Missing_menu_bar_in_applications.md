### Post by JRV on 2012-10-11
All the menu selections that used to be in a window are now in the top panel.  Move the cursor to the top panel and it will show the menu selections for the active window.

If you wish to change this and have the menu selections in the window run this command:
```

sudo apt-get remove indicator-appmenu

```
