### Post by Xzallion on 2006-02-19
Heres how I installed Flash, Hope it helps.

Go [here]("http://www.macromedia.com/shockwave/download/index.cgi?P1_Prod_Version=ShockwaveFlash&P2_Platform=Linux&P3_Browser_Version=Netscape4") and download this to your desktop 

now open a terminal (Applications>Accesories>terminal)

type or copy & paste these commands, hitting enter after each command

```
cd Desktop
tar xzf install_flash_player_7_linux.tar.gz
cd install_flash_player_7_linux
./flashplayer-installer
```

The installer will begin, and prompt you a couple of times. just answer "y" and hit enter.

That should do it.  Let me know if it worked for you.
