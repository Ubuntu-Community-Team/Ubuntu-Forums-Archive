### Post by mcduck on 2009-08-12
By default dragging a file inside the same partition will move it, but if you drag it to another device, or another partition, it copy the file instead.

So even when you are drag&dropping the file inside the same disk, make sure that it's also inside the same partition.

(I don't remebber how Thunar does this, but at least with Nautilus dragging a file with middle mouse button gives you a menu where you can select if you want to copy, move or link the file)
