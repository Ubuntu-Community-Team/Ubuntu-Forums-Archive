### Post by sethmahoney on 2005-11-03
If a lot of the apps you need are for KDE, you might consider switching to Kubuntu (which is just Ubuntu with KDE instead of Gnome).  If you'd rather use Gnome, though, you can go ahead and install KDE apps.  They might look a little weird, and they'll be flashier than you're used to, and your system might run a little more slowly with all the extra libs loaded (though if you have even a fairly high-end system you probably wont notice the difference), but you shouldn't have any showstopping problems running KDE apps in Gnome.
