### Post by Psychomechanix on 2007-04-25
Hi Julie. 

I went through this exact same thing. I'm still very new to Linux so I can't tell you exactly WHY this happens but I did find out that the problem is not with Beryl or the Emerald Themer. It's a problem with the way the person packages the theme. Many of the themes out there for download are broken in this way. But if you use the Emerald Theme Manager (mine is version 0.2.1) and select "Fetch GPL'd Themes" on the Repositories Tab it will download only themes that are not broken. As for all other themes (ones you find on the net and in other repositories) it's just hit and miss. Some are broken and some are not. You just have to try them all out. 

Like I said, it is reported that MANY themes are broken so it's possible you might go through an entire list of them and none work. I have the same problem. But some do.

Hope this helped.

Take care!
