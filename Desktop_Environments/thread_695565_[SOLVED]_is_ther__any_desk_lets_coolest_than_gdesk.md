### Post by santiagoward2000 on 2008-02-24
Hi!
First, I wanted to say Screenlets 0.0.10 would have worked on Beryl too, and the lastest version (0.0.12)  doesn't need any of those.
To install the lastest one, go to **System/Software Sources**, and under the **Third-Party Software** tab click on **Add...** There, type:
```
deb http://ppa.launchpad.net/gilir/ubuntu gutsy main universe
```
Click **OK** and update.
Then install the **screenlets** package. You can either look for it on Synaptic or open a terminal and type:
```
sudo apt-get install screenlets
```
That's it!
If you want to get more screenlets, look under the **Screenlets** section in [http://www.gnome-look.org/]("http://www.gnome-look.org/")
Cheers!
