### Post by SGC on 2005-10-01
Yes its safe, removing kdenetwork will not remove any program since kdenetwork is a meta package, its only used for installation of other packages, but not removing them.

### Post by darkmatter on 2005-10-01
SGC is correct. Removing kdenetwork will not affect your system.

### Post by Takis on 2005-10-01
There are a lot of dummy packages that you get warned about (for example, kde-desktop I think is one). Next time you get warned about something, have a look at the package it complains about in Synaptic/Kynaptic. The description says pretty explicitly whether a package is safe to remove.
