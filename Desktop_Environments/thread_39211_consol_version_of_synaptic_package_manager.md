### Post by benplaut on 2005-06-03
[QUOTE=nilus]What do you mean? apt-get?[/QUOTE]i'll expand on that...


synaptic is not the actual package manager for Ubuntu or Debian, it is only a graphical "front end" (pretty GUI  :grin: ) for the package manager- apt-get. Apt-get is run from the command line, and can do everything that Synaptic does.

here's a manual for apt-get (Note: you have to run apt-get with sudo/root, and it can't be run at the same time as synaptic, another terminal of apt-get, or ubuntu udate manager)

[http://www.debian.org/doc/manuals/apt-howto/index.en.html](http://www.debian.org/doc/manuals/apt-howto/index.en.html)

actually, apt-get is a front-end (command line based) for dpkg  ;-)
