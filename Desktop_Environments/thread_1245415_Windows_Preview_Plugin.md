### Post by VCoolio on 2009-08-20
As far as I know that's not a feature (yet?). The idea is that minimized windows really get minimized, to use as little resources as possible. If the preview should also work on minimized windows (especially live preview) that would take a lot of memory. Of course the big company behind the worlds most used os doesn't care about resources, so it enabled it. Linux does care. If you minimize windows, you're not supposed to be interested in what they're doing, I think is the reasoning behind it.
