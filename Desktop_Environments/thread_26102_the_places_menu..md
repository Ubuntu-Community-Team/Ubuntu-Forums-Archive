[QUOTE=Spark*]You need to open a file selector dialog and then use the Add/Remove buttons (yes, it is ridiculous).[/QUOTE]

It's not ridiculous if you think of it this way:

Places = Nautilus Bookmarks.
You can edit your Bookmarks in Nautilus.
They show up in the Panel for Convenience.

If your Firefox Bookmarks showed up in your panel, but you still had to edit them in Firefox, that wouldn't be ridiculous, would it?
