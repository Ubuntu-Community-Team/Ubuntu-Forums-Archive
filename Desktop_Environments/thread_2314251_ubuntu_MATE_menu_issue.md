### Post by T.J. on 2016-02-25
Sounds like a bad icon theme.  In rare instances, gnome/mate code has been known to act strange when you get a bad set of theme files.  I would try downloading an updated version of the theme you want to use, or try reinstalling mate-icon-theme in Software Centre.
