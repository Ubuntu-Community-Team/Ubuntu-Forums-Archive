### Post by sayakb on 2008-08-11
To change the media player that starts when you insert a movie CD/DVD:
Open your home folder. Goto *Edit->Preferences->Media tab* and change the applications for CD and DVD video.

To change the default player for specific media formats: Right click on the file, select Properties, click on Open With tab and select the application.

---

### Post by sayakb on 2008-08-11
You may also goto System->Preferences->Preferred Applications->Multimedia tab and select the player from the list or add a custom command to it.
