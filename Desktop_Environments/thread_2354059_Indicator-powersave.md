### Post by quequotion on 2017-02-27
[s]I've finally launched my own PPA: [QQ Desk]("https://launchpad.net/~quequotion/+archive/ubuntu/qq-desk").

The first package is [indicator-powersave]("https://launchpad.net/~quequotion/+archive/ubuntu/qq-desk/+sourcepub/7515759/+listing-archive-extra"), a fork of [indicator-cpufreq]("https://launchpad.net/indicator-cpufreq") with additional toggles for system-wide powersaving. It is neither a detailed reporting tool like [powertop]("https://01.org/powertop"), nor a complex manager like [tlp]("http://linrunner.de/en/tlp/tlp.html").

It is a user-discrection power management utility.

The frontend is an indicator:
[[IMG]http://s1.postimg.org/wqgqfd4sb/indicator_powersave.jpg[/IMG]]("http://postimg.org/image/wqgqfd4sb/")

You can read more about it [in the Archlinux forum]("https://bbs.archlinux.org/viewtopic.php?id=180762").[/s]

*Never-#&#%&?#-mind*. Trying to package this and distribute it on launchpad has been a very good reminder of why I don't use Ubuntu anymore. Maybe someday I'll try this again; don't anybody hold their breath.
