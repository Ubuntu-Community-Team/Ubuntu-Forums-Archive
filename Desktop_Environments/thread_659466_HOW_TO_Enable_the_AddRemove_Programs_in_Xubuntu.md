### Post by ziploc67 on 2008-01-05
So your add/remove programs are not working? I have been there, and getting the same 'refresh' suggestion by it. This happens because add/remove programs thinks that you dont have an internet connection. this is very easy to fix.

1. Go into the add/remove programs

2. To the bottom left, there is a button called preferences, please click on that

3. There should be many check boxes, that are unchecked. this is why the message shows.

4. Go to the terminal, and type in 'sudo apt-get update'

5. Go back into the add/remove programs

Your Done!!!

If this fix does not work, then there is a networking error, please consult with other how-to's, threads, or make your own thread to consult the problem.
