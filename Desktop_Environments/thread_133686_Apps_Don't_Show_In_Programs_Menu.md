### Post by Ramses de Norre on 2006-02-20
Type the name of the program in a terminal, that should launce it. (ex. firefox &  , the & makes it a background process so that you can continue using that terminal).
In the same way you can add new entry's for them to your menu's, just add the name as command and don't make them run in terminal.
(some programs have extra options in menu's like firefox (firefox %u) I don't know what they exactely do but I've added apps with just their name to my menu and they work perfect.)
