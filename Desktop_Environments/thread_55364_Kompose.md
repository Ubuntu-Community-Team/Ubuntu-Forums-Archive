### Post by ltmon on 2005-08-08
Find kompose in your menu and start it.  It will put a new system tray icon in the tray.  If you right click on this and select the configuration menu item, you can set keyboard shortcuts etc.

If it's not in your menu for some reason, hit alt-f2, type "kompose" and hit enter.
