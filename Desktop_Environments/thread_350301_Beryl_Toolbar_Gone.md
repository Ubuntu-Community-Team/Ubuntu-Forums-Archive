### Post by xopher on 2007-01-31
That's called the [B]titlebar[/B (because it contains the title of the window).

(Easiest way to run these is pressing ALT+F2, which will bring up a window..)

To get the gnome (non-3d) window borders back, run: ```
metacity --replace
```

To get beryl running, run: ```
beryl-manager
```

Now , if that doesn't do it, right click on the tray icon, -> select window manager -> Beryl.

To get beryl to startup automatically when you boot / login, add beryl-manager to 'System -> Sessions -> Startup Programs'

Hope this helps
