### Post by macewan on 2005-02-08
Computer > Home

Now go to the folder that contains a file with the mp3 extension. **Right** click on the file with the mp3 extension and select **Properties**. In this new window you will see tabs at the top, select the **Open With** tab and select the program you want to use. If it isn't listed then click the **Add** button and type its name.
