### Post by aysiu on 2006-05-06
You can install applications with the live CD just as you can with an installed Ubuntu. The only difference is that the "installed" applications from the live CD are "installed" only to RAM, so they won't be there the next time you boot up.

Also, you can't install more programs than you have enough RAM for. So if you have 2 GB of RAM, you can probably install a lot. If you have 128 MB of RAM, you probably won't be able to install much at all.
