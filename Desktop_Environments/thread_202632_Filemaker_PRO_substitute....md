### Post by jgcamp99 on 2006-06-23
Open Office has Base, it is the database office application that installs with Ubuntu by default. 

Look in Applications, Office and you'll see OpenOffice.org Database.

Just from playing around with it briefly, it's everything File Maker Pro is, you just have to readjust to the interface when it first opens up, you need to create an empty/new database first or have an existing .odb file or connect to one of the other formats listed. Also, like any database application that isn't MS Access, it has it's own little nuances that there is a learning curve that will have to take place. I'm relatively impressed with it.

I believe you are using File Maker Pro in OS X, maybe even Windows XP ?

EDIT: BTW, the reason why I steered you to Base, it is part of the entire office suite that Open Office provides, integration with Calc, Writer and so on.

[http://www.openoffice.org/index.html](http://www.openoffice.org/index.html)
