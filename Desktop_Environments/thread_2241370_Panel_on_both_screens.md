### Post by Jordan_Berlyn on 2014-08-25
I have noticed that you can extend the panel across multiple monitors, but I recently moved away from Windows 8.1 and one of my favourite features of this was having the start menu (panel) cloned across both screens. Is this something that is possible Xfce? I am using the Whisker Menu, which is default with a Xubuntu 14.04 install. I'd like to have the menu button shown on both panels as well as the currently active windows on both panels.

---

### Post by Toz on 2014-08-29
If you just need the Whisker Menu and Window Buttons plugin on the second monitor, then just create a new panel and move/set it to display on the second monitor. Add the Whisker Menu and Window Buttons plugins to it and configure the Window Buttons plugin to _not_ "show windows from all monitors".
