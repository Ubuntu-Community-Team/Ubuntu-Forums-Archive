I tried unity8-desktop-session-mir (it is in the software centre) on 14.10 the other day, and if I remember correctly I got a black screen after logging in. You might get better results.

### Post by buzzingrobot on 2014-10-28
Per this ([http://mhall119.com/2014/10/unity-8-desktop/](http://mhall119.com/2014/10/unity-8-desktop/)) currently you need to manually create a password to log in to the live session of the Unity 8 desktop.

However, I tried yesterday's 15.04 Unity 8 daily and the live session, via the 'try' option, worked without a password, as usual.  What I got was the phone interface. That was my first look at Unity 8, so maybe that's all that's there at the moment.
