### Post by Rocket2DMn on 2008-06-27
Since FF is integrated into the system a bit, they will probably not give the upgrade to FF3 from FF2 in older versions of Ubuntu (except maybe in the backports repository).  This is because it can create dependency issues and possibly make things unstable.  Because of this, programs usually aren't updated except for security fixes during the life of an Ubuntu release - this helps keep stability in the system.
