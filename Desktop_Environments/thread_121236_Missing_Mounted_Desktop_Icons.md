### Post by Sutekh on 2006-01-24
First make sure that the media is actually mounted.  If it is make sure the folowing setting is checked.

Open the configuration editor; Applications > System Tools > Configuration Editor.  

Go to apps > nautilus > preferences and make sure that "show_desktop" is checked.

Go to apps > nautilus > desktop and make sure that "volumes_visible" is checked.
