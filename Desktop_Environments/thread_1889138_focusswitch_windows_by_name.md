### Post by jchien on 2011-11-30
I have many terminals open with window names prefixed with terminal profile. All tabs in each terminal window use the same profile. I'm bad at scanning the taskbar panel with my eyes for the prefix I'm looking for, and alt-tabbing through many windows is also slow.

What I really want is a program sort of like gnome-do that I can just start typing in the name of an open window and switch to it. Does something like that exist?

---

### Post by Larkspur on 2011-12-01
I think Gnome-Do is still in the repositories; if it's not, then Synapse or Kupfer have similar functions.  Otherwise, download ccsm and use it to switch on Scale Window Title Filter.  After compiz restarts, you should be able to find windows by going into Scale (Super+w) and typing the window name.

### Post by jchien on 2011-12-02
