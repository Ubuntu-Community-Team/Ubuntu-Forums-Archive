### Post by Random_Dude on 2012-08-16
I don't know if you are going to see this reply. But there are plenty of tiling window managers.

The best resource for this kind of stuff is arch wiki: [https://wiki.archlinux.org/index.php/Window_Manager#Tiling_window_managers](https://wiki.archlinux.org/index.php/Window_Manager#Tiling_window_managers)

As for my favorites I would say:
DWM - if you don't mind compiling C code
awesomeWM - which you should know a bit of lua. It has an icon tray, which is nice.
WMFS - this is the one that I would recommend to start with. It's more user-friendly than the others and it supports tray icons out-of-the-box.

Cheers
