### Post by K.Mandla on 2006-07-05
I know ndiswrapper is on the Dapper CD; if you use the terminal window and enter ...

```
sudo apt-get install ndiswrapper-utils
```
it should install ndiswrapper for you. 

The other packages you mention might be there too, and should install with the same command.
