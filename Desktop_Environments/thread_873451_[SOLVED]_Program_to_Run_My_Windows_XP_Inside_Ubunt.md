... or if you don't have the money and wish to be compliant you can get VMware Server free, as in beer. VMWare Server is pretty much the same thing as workstation, but with less features. You can also create your own images using a vmx generator like [easyvmx.com]("http://www.easyvmx.com/") and VMware Player which is also free as in beer.

Edit:

I have XP SP2 running inside a VMware session on Kubuntu 8.04 and everything works well except accelerated graphics is a little slow, so I don't enable them. The graphics issue has more to do with my hardware then with VMware.

### Post by timjohn7 on 2008-07-29
I have 1 word to answer your main question... Sun Virtualbox (OK, 2 words).  I have a couple of esoteric apps for which I can't find a Linux equivalent and run XP in Virtualbox with USB support, wireless networking and access to shared drives over my local machine and home network.  And I'm a 6 month old noob with no Linux experience AT ALL before loading Gutsy in Feb 08.
It does take a bit of setup sweat, but there is plenty of advice in this forum and I certainly learnt a lot.
Try it! but make sure you get the right version from Sun for USB support.
