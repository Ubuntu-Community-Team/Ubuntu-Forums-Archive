### Post by canadianwriterman on 2006-08-17
Just a guess here, but it took me a while to figure out how to change an icon through Alacarte. When browsing to the directory where your icon is stored, the icon will not show in the browser window. The window is blank, leading you to think there are no icons in it. 

You have to browse to the directory containing the icon and then hit OK. Any icons that are in the directory, whll then show up in the Alacarte window. Select the icon you want and you're set.

If that's not your problem, perhaps it's file format of your icon. I think a few formats work, but I think it's SVG that works best.
