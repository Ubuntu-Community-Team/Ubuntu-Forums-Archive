### Post by screaminj3sus on 2010-06-06
**there is no way to get smooth corners in metacity** because it does not support rgba (And I dont know WHY in gods name ubuntu's graphic designers and others keep friggin making rounded metacity themes while it doesnt support this...)

The one solution is to use emerald as your window decorator (sudo apt-get install emerald) and find an ambiance theme for it. (you can set emerald as your default window decorator by installing compizconfig settings manager > go to window decorator and in the command box type emerald --replace. Or you can just add emerald --replace as a startup item)

I personally use this one that goes well with ambiance:
[http://kde-look.org/content/show.php/Ambiance+Emerald+BW?content=122117](http://kde-look.org/content/show.php/Ambiance+Emerald+BW?content=122117)
