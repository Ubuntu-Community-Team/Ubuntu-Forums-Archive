2) Screen Colors - The "dropdown menu" is now black.  I had set a different color, been in place for years.  How do I change this color back?

3) Screen Colors - The "slider color", that moves the displayed screen, is now orange.  I had set a different color, been in place for years.  How do I change this color back?
