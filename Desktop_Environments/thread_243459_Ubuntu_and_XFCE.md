### Post by tigerdyr on 2006-08-25
That depends on what you want...

If you install the xubuntu-desktop you  get a full-fledged desktop enviroment including text editors, terminals and so on. So this is a one-shot covers all, but you get a "generic" package

If you only install the xfce4 package you only get the desktop - then you will have to also get the text editor(s), browser(s) and do on yourself. So here you have to tweak from the beginning yourself - but you get a desktop that is completely yours (with all the joy and aggrevation that may bring)
