### Post by sajro on 2008-03-21
I'm trying to set up the perfect fluxbox config for me and I'm well on my way. However, I can't figure out how to make an option that will log me out and drop me to the log-in manager (SLiM). How can I do this?

This is, BTW, in Arch, no Ubuntu. However, I think this is not distro-specific.

---

### Post by p_quarles on 2008-03-21
Add a line that says this:
```
[exit] (Exit) <>
```
