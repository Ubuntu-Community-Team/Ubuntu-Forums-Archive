### Post by coffeecat on 2006-09-11
Most people are able to install Ubuntu from the live CD with no problems, so two things to consider: possibly a faulty/corrupted install disk, or issues with your particular hardware.

Did you check the md5sum when you downloaded the iso, or are you using the shipit CD? If you downloaded the iso, what speed did you burn the CD at? It's generally advised to burn at 4x; faster speeds can lead to errors.

Also, suggest you post details of your hardware. Others with same/similar hardware may then be able to comment. And have you been able to run any other Linux distro on that machine?
