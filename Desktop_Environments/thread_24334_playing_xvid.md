### Post by lorap on 2005-04-06
hi friend!
there's a solution as soon as i'd the same troubles at beginning :-)
all you've to do install VLC player.it renders almost all kinds of video codecs.
go to Computer-->Computer Management-->Synaptic Package Manager.
once you get the synaptic's window open go to the Settings-->Repositories.
once you've the repositories' window open mark there all checkboxes ignoring the warnings then press Ok button.the repositories' window closes so you get back to synaptic's window back.now press Reload button and wait unril the process completes.once the process's completed press the Search button and in the search window write in VLC then press Ok button.after that synaptic will find it out for you.you'd mark the VLC'S checkbox for installation after what press Apply button. :-)
that's all :-)
now you're able to watch movies.
well,to be fear,after you get it installed close the synaptic's window and press:
ctrl+alt+backspace to reload grafic shell.
have a nice day!
pavel
