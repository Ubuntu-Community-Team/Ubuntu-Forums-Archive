### Post by ltmon on 2006-06-19
Unfortunately the developers of kvim gave up.  It was too hard, and the Vim lead developer wasn't about to modify his custom event loop (or something) in order to make it any easier.

To move forward the kvim developers created "yzis" - a Vim-like editor specifically tailored at embedding in different environments.  It's almost usable but not quite.  They are targetting KDE 4 as a release for a stable version.

 [www.yzis.org](www.yzis.org)

The upshot of this long story is that kvim has been removed from just about everywhere because it is so buggy, and caused so many problems.  I don't know if there's any easy way to patch it back in yourself.
