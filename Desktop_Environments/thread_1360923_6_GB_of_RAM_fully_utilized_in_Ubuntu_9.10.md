### Post by foldingstock on 2009-12-21
If you install the 64-bit version, yes, it will be able to see and use all of your ram. 

The 32-bit version will not be able to use all 6GB (will only see up to 4GB) unless [PAE]("http://en.wikipedia.org/wiki/Physical_Address_Extension") is enabled, and even with that not all applications support PAE. 

You would be best off installing the 64-bit version with that much ram. 64-bit Linux has come a long way. I use the 64-bit version of Ubuntu 9.10 on my primary laptop and everything runs quite smoothly.
