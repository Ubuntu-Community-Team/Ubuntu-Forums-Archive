### Post by guiverc on 2020-08-10
Welcome to Ubuntu Forums.

It's always useful if you're specific with your release, I'm betting though it's Lubuntu 20.04 LTS.

The icon itself is part of the LXQt theme, where the theme change is covered in the Appearance section of the Lubuntu manaul

[https://manual.lubuntu.me/stable/3/3.2/3.2.2/appearance.html](https://manual.lubuntu.me/stable/3/3.2/3.2.2/appearance.html)

[IMG]https://manual.lubuntu.me/stable/_images/lxqt-theme-tab.png[/IMG]

If you scroll down to the 3rd screen pictorial, you'll see what LXQt themes and how the logos (icon) appears for each default theme.

The icon itself is stored within that theme (/usr/share/lxqt/themes/).  Rather than change it directly, I'd suggest using a theme more to your liking (*more can be downloaded too*), or cloning the directory and making changes to your own version (*it's easier to revert should a mistake be made*).
