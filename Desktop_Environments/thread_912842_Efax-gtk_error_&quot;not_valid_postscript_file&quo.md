### Post by basheeer on 2008-09-07
In text editor:

1- File > Print

2- select:
Print to file
Postscript

3- Name it

4- Save in folder (select folder destination)

5- Print


In Open office Word:

1- File > Print

2- Select Print to file, click OK

3- In save dialog:

Name your file
Choose a folder to save in
Click Save

Now you have a postscript page to be sent by Efax...
