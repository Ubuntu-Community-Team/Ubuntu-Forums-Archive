### Post by bionnaki on 2006-06-21
yes, the soulseek admins are using a different server for 157 and many people are on there. the network is currently spilt.

but with nicotine, you could easily use either network.

simply open up nicotine (please use the latest, the old nicotine sucks) and in the server settings, change server.slsknet.org:2240 to server.slsknet.org:**2242**

restart nicotine.

nicotine is not obselete - the latest versions are just as good as the official client. when the 2240 server has been shutdown, simply switch over to 2242 using nicotine.
