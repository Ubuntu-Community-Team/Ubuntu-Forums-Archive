### Post by Tsuki on 2005-08-24
What happens when you try to open synaptic?  Does it produce an error, or just do nothing?  If you open up a terminal window and run "sudo synaptic" (entering your password when prompted), what happens?

The easiest way to install KDE is probably to get the "kubuntu-desktop" package. (Don't worry, it won't mess up your existing Ubuntu install - it should mean that when you get to the login screen you can click "Sessions" and choose between Gnome and KDE.)  You can do this through Synaptic once it starts working, or by running "sudo apt-get install kubuntu-desktop" in a terminal.
