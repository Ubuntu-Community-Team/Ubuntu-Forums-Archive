### Post by dude051 on 2006-08-25
I've searched quite a bit for this, and it's actually not that hard! Check it out, at the Compiz forum [http://www.compiz.net/topic-3103-howto-change-the-background]("http://www.compiz.net/topic-3103-howto-change-the-background")

To get the same 'far space' feel, I changed my zoom plugin settings.
1. Open gconf-editor
2. Goto apps/compiz/plugins/rotate/screen0/options  -  Zoom - set 6-10, try different ones out.

Its called skydome, and if you want some pics to try go here [http://www.compiz.net/topic-10-skydome-images]("http://www.compiz.net/topic-10-skydome-images")
