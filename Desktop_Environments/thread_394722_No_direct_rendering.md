### Post by igknighted on 2007-03-28
You appear to have been running an XGL session.  When XGL is running, the glxinfo command and Cedega will report no direct rendering.  XGL is using that rendering and won't share.  Try logging in to a plain gnome session, no XGL or beryl, and running the test that way.  I have that same card and Cedega runs fine on it.  My FPS for games isn't great, but its playable.  I would use envy to start fresh, then install the latest driver (not legacy or anything, 6600gt is a newish card) and run Cedega without beryl or XGL at all.
