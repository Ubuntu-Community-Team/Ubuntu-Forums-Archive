Moderator feel free to remove this, I'm not sure what good it'll do anyone but...

I found away to get some updates and packages on to my Ubuntu box that doesn't have a working NIC yet.

From a different computer (or other OS on dual-boot), download the lastest Ubuntu ISO. Burn said ISO and take it to the no interent PC. It'll ask you if you want to run Synaptic after the CD is mounted. Select you packages and enjoy.
