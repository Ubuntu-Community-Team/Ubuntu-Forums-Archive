### Post by feveryear on 2007-12-18
There was a solution I found to this somewhere on this forum, but I can't find it right now. It's a problem with Compiz or something of that effect. Whatever way that Compiz initiates the screensaver screws it up. I play Metal Blob Solid and AssaultCube in fullscreen, and have had the same issues. The fix is to either disable the screensaver (while Compiz is enabled) or move the slider to a very long time. (I personally don't use a screensaver because I have a laptop, so if I'm not using the laptop, the screen is closed.) ANOTHER fix, if you won't miss Compiz that much, disable the Desktop Effects. That helps as well. I still disable, or elongate the time, for the screensaver. Another issue with Compiz was when I tried to view fullscreen from Youtube, it wouldn't work. It would only open a smaller new window. That was also a compiz problem, so I turned off my Desktop Effects all together. Those fixes should work. I know they worked for me.

Hope this helps you all.
