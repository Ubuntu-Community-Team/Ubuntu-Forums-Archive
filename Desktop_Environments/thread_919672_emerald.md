### Post by benerivo on 2008-09-14
Add a startup entry in System > Preferences > Sessions. The command should be emerald --replace

---

### Post by TheUbGuy on 2008-09-14
Or, if you have the Compiz Config Settings Manager installed (will be under System->Prefrerences->Advanced Desktop Effects Settings), open that up, go to Effects->Window Decoration, and under Command put emerald --replace there.
