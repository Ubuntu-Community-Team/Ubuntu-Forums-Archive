If you want the gnome menu while using unity, you can do the following...

Log into unity

Press ALT+F2 to open the run command

Type gnome-panel

ENTER

Delete bottom panel with right click

Change properties of top panel to be at bottom of screen and uncheck Expand so it doesn't interfere. If the panels at the top it stops working after using unity.

You can set your new panel to auto-hide ( I did )

You can also add the work space switcher to the panel and create up to 16x16 unity workspaces.

Make sure to add gnome-panel to your startup applications so it loads at login.
