### Post by Nythain on 2007-07-05
most of those are extra kde metapackages that will install a lot of stuff... they arent included in teh base kubuntu install, so they didnt make it into kubuntu desktop... everything those metapackages install can be found individually in synaptic if you dont want to install every package in teh meta package... others like kde and kde-core are more simple metapackage alternatives to the whole kubuntu desktop package... like kde-core will give you just the core of kde, the libs and teh desktop, but no extra apps... for those who like to install thier own preferred apps
