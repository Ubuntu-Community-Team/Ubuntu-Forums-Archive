### Post by Shay Stephens on 2006-04-12
Almost anything is better than windows ME.  That was the worst release ever, very buggy, lots of crashes.  Windows 98 is more stable, and windows 2000 is better, XP is the best of the bunch if it were not for the having to ask permission to run it "activation" "we are prepping you to rent your OS" scheme.

Ubuntu breezy was the only linux OS that ran on my computer without problems.  I first ran across it late last year, and I am just now getting familiar enough and experienced enough with it to be productive and solve problems like I could with windows.

If you have any windows only applications that you can't live without, do give dual booting a go, it is your best bet in the short run.  Looking a little further out, learn to use wine to run windows apps in linux.  I am just now getting results from that, and I only really have one app that still needs windows to run.  

Learn to look for alternatives to windows only apps.  It takes new learning, but usually the learning curve is not too bad if taken is managable chunks.  Don't try to learn it all overnight, you will just get frustrated and give up.
