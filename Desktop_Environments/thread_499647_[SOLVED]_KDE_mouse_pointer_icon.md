Nevermind.  I found that this is called the "Busy Cursor".  I'm using Kubuntu Feisty Fawn.  There is no option to change this behavior from the System Settings.  You need to launch kcontrol from the command line and then you can change it under the Appearance and Themes section.  There's a bug for this in Launchpad:

[https://bugs.launchpad.net/ubuntu/+source/kde-systemsettings/+bug/109395](https://bugs.launchpad.net/ubuntu/+source/kde-systemsettings/+bug/109395)
