### Post by ArcaVex on 2009-10-15
I've tried this on Ubuntu and Kubuntu:

sudo apt-get install kubuntu-desktopor for gnome

sudo apt-get install gnome-desktop 
Be careful however, changing from one to the other will potentially remove certain packages. Things like networkmanager,  (if you are using wireless i would only change desktop if you really need).

### Post by lovinglinux on 2009-10-17
You can install **kde-desktop**, which will also install all default kde applications, or you can install **kde-core**, which will install only the basic necessary stuff.

Take a look at [http://www.psychocats.net/ubuntu/kde](http://www.psychocats.net/ubuntu/kde) and [http://www.psychocats.net/ubuntu/kde-core](http://www.psychocats.net/ubuntu/kde-core)
