### Post by 23meg on 2006-12-05
> Gnome would not let me use the Windows button, however. Does anybody know what I can do?See if selecting a different option in System / Preferences / Keyboard / Layout Options / Alt/Win Key behavior helps.
> Again in KDE you can assign applications to particular workspaces, so that application A always opens in workspace 1, application B always in workspace 3, etc. Is the same possible in Gnome?Yes, with [Devil's Pie]("http://www.burtonini.com/blog/computers/devilspie").
