### Post by ankspo71 on 2011-04-29
Hi,
This feature you want turned off is part of the Oxygen Theme style. Press Alt + F2 and run "oxygen-settings" in the run dialog. The when that application opens, go to widget style (category), general (tab), then where it says "Window's Drag Mode:" choose either:"Drag windows from titlebar only" or "Drag windows from titlebar, menubar and toolbars", instead of "Drag windows from all empty areas". There are several other useful settings in there too.

alt + left click 'move' is used to move a window above the screen display for accessibilty reasons.  If you have a large enough monitor then you will probably never need to use this feature. Normally moving a window upwards is stopped when the title bar reaches the top of the screen, but it keeps moving if you are holding down the alt button. 

Hope this helps.
