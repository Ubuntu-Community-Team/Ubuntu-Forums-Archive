### Post by Rhapsody on 2007-01-02
I'm planning to upgrade from Kubuntu Dapper to Kubuntu Edgy via a full wipe and reinstall soon, and I'd like to keep my current custom colour schemes. This would sound easy as I can import old colour schemes, but the problem is that KDE won't tell me where my colour schemes are stored so I can make backup copies! Does anyone know how I'd go about doing this?

---

### Post by Jucato on 2007-01-02
In System Settings -> Appearance -> Colors, when you click on Save Scheme, that particular color scheme you just saved gets saved at /home/<username>/.kde/share/apps/kdisplay/color-schemes.
