### Post by gnuvistawouldbecool on 2009-07-17
Any KDE application will run in GNOME. If you choose to install one, there will be a few dependencies to satisfy, but once those are installed the application will run.

GNOME seems to be more popular here, but I'm not sure why.

I would personally say KDE is better, but I use GNOME most of the time since it tends to need less resources on my computer.
