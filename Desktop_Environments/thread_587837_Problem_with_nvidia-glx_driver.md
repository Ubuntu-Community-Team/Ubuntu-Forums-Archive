Hi,

So, my problem is that when I install the nvidia driver and boot my laptop it would load and everything, but then it would just stop in a black screen and stay there, as if X wasn't loading.  If I do ctr-alt-bcksp it does go to the login screen and I'm able to log in, it shows the nvidia logo and X loads correctly.  Seems as if it is hibernating right from the start.

I've tried using nvidia-glx-legacy and -new and those don't work.  Right now I'm just using the nv driver and it works just fine, but that means no acceleration and slower performance, which was the reason I moved to xubuntu in the first place.  I've tried a lot of different tricks I've found around to no success,

I just updated to Gutsy, but Feitsy had the same problem before.

Does anyone have any idea how to solve this?

Thanks a lot in advance.

EDIT:  Well, nobody offered advice or direction here, I don't seem to get the codes of the 'community' right, since nobody ever answers to any of my posts.  Anyhow... It took me a clean install of Ubuntu 7.10 (as opposed to the xubuntu 7.10 update that I had), and a little change in my Dell's (Inspiron 2650) BIOS (in the display section to LCD as opposed to the default option), and now the nVidia driver is working fine.
