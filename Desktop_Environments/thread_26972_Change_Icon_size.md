### Post by Alexander Kirillov on 2005-04-14
I do not think it is possible to set globally. For panel, the icon size is determined by panel height. For desktop, I actually do not know how Nauitilus sets the size - but you can right-click on the icon and select "stretc icon". You'll have to do this for every icon, though.

### Post by dmitry_baburkin on 2008-02-24
You can set it globally.
Start Nautilus file browser, Menu *Edit* -> *Preferences* . And try to set icon zoom level in the  "Icon View Defaults" section. This affects icons on the desktop as well.
