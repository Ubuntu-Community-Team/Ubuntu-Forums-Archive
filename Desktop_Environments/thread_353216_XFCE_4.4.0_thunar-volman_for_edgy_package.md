### Post by Rommidze on 2007-02-04
As you many asked, I've builded a *thunar-volman* package for ubuntu edgy. Binary for i386, and debian source packages avaliable. This package is to be used with the [XFCE 4.4.0 backport for ubuntu edgy]("http://tech.tolero.org/blog/en/linux/xfce-440-packages-for-ubuntu-edgy").

Pay attention, that this package can be not installable if you're already installed XFCE from the *xfce.org binary installer* or any other way **not** from the packages. :(

PS: If you're already using my repository, but after *update* in package manager *thunar-volman* is not visible, just move to the new repository path as explained on my [blog page]("http://tech.tolero.org/blog/en/linux/xfce-440-packages-for-ubuntu-edgy"). If you're already on the new repository - all should pass well.
