### Post by kvonb on 2006-07-05
you might have to make it executable first:

Open a terminal:

MENU -> Applications -> Accessories -> Terminal

Change to the folder that contains the downloaded file:

cd ~/downloads/ (change downloads to the folder that you d/l'd to)

chmod +x whatever-it-is-called.sh

then:

./whatever-it-is-called.sh
