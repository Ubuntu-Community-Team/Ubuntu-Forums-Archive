### Post by sanderd17 on 2011-07-18
Gnome 3 is nothing like Unity. Gnome shell is more minimalistic than Gnome2, but you can install extensions to make it just as bloated as Gnome2.

You can install a menu extension, so that you get the old menu back, a taskbar/dock, a lot of applet-like extensions ...

This is also the main difference between Gnome3 and Unity. Gnome3 is made minimal, so that others can write extensions and themes for it without problems. Unity is made to be complete and not extendable.

Every theming/extension making in Gnome3 is done with a combination of CSS and JavaScript to attract the huge amount of web developers.

Please don't judge Gnome3 if you have not seen it. If you want to try it, take a look at gNatty: [http://ubuntuforums.org/showthread.php?t=1754198](http://ubuntuforums.org/showthread.php?t=1754198)

gNatty is not vanilla Gnome Shell, but it comes quite close. If you want to look for extra extensions, check WebUpd8: [http://www.webupd8.org/search/label/gnome%20shell?max-results=10](http://www.webupd8.org/search/label/gnome%20shell?max-results=10)

Also note that Gnome Shell (certainly in combination with Ubuntu) is not very stable yet. For most it works well once it's installed. But if you get annoyed by bugs, please give it some time. The combination Ubuntu-Gnome Shell should be stable when Oneiric comes out.
