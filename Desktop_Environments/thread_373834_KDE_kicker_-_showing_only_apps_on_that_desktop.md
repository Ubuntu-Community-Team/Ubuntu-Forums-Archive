### Post by AndrewNeo on 2007-03-01
I believe I've found it before on previous installs of KDE, but I can't for the life of me figure out where the option is to only show applications in the kicker that are open on that desktop.

For example:
Desktop 1: Firefox, Konsole
Desktop 2: Thunderbird
Desktop 3: Nothing
Desktop 4: Kwrite

When I'm on Desktop 1, the kicker only shows Firefox and Konsole, when I'm on Desktop 3, nothing's in the Kicker, etc.
I'm fairly certain it's in there somewhere, I'd really appreciate someone who could point out where it is, or something that can do this if it's out there.

---

### Post by Jucato on 2007-03-01
Right-click on the panel -> Configure Panel -> Taskbar -> uncheck "Show windows from all desktops"
