### Post by mrboojive on 2009-11-11
One problem is that you are not actually using Emerald. Based on your screenshot, it appears that your window manager is KWin, KDE's window manager. Consequently, changing your Emerald theme is not going to do anything. Is there some reason that you want to use Emerald + Compiz? KWin has a lot of Compiz-like effects. You can customize it pretty extensively through System Settings > Desktop > Desktop Effects and System Settings > Appearance > Windows.

That said, if you are set on using Compiz + Emerald, there is probably a way to get them working with KDE. This thread might be helpful: [http://ubuntuforums.org/showthread.php?t=1233945](http://ubuntuforums.org/showthread.php?t=1233945).
