### Post by Dennis N on 2017-05-07
> Because I honestly couldn't figure it out by watchings someone use the desktop. Are they using keyboard shortcuts? Can you actually interact with the DE efficiently without using them? I'm absolutely scared of what newbies will think when they suddenly have no minimise and maximise buttons when updating their Ubuntu distro.

It is easier than you might expect once you try it. Since the original gnome 3 concept, gnome extensions have allowed the user to customize the way things are done. There is a configuration and settings application called Gnome Tweak Tool that is included with Ubuntu Gnome by default. It has the option to enable with one click the minimize and maximize/un-maximize buttons. Menu list enabling is also there. 

As for switching windows, I find that Alt-Tab is often the quickest for me - it returns to the previously use window in one Alt-Tab press; also it can show windows on all workspaces. Menu list cannot do that. Dash to Dock to switch between windows of same application can take repeated clicks on it's icon, as it cycles through the windows of that application.

### Post by malspa on 2017-05-07
I first tried GNOME 3 almost 6 years ago, in a live session of Fedora 15; didn't care for it all that much. Then I took another look, spending some time with GNOME Shell in a Fedora 16 installation, and became comfortable using it rather quickly.

Pages like this one have helped me: [https://wiki.gnome.org/Projects/GnomeShell/CheatSheet](https://wiki.gnome.org/Projects/GnomeShell/CheatSheet)

I'm using GNOME Shell in Ubuntu 16.04, but with ***no extensions***. Also, I use the mouse to move between windows, most of the time -- hot corner to open the overview, click on the desired open window. No big deal, but perhaps it's a matter of getting used to it. Old habits die hard.

Rarely, I use Alt+Tab for window switching. I simply don't need to use the keystrokes very often. I do think it's useful to know how to restart GNOME Shell by typing the letter "r" at the Alt+F2 prompt.

Seems to me that so much depends on the individual user's mindset. Others feel that they cannot efficiently multi-task and interact with the GNOME 3 desktop without extensions. Piece o' cake, for me.
