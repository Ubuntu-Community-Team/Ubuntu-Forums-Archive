Prior to install ubuntu i didn't have a seperate partition for /home now i've created a new ext3 partition to store home in how do i go about changing it without having to reinstall ubuntu?

---

### Post by Tomosaur on 2006-08-04
System > Administration > Users and Groups

Find your user, click 'properties', then 'advanced', and type in the path to your new home folder.
