### Post by rmccabe3701 on 2007-09-16
Update:
  I got a nVidia card, installed the nVidia driver via apt-get, and to set up the dual screens I did ```
gksudo nvidia-settings
```
 ... rediculously easy with a nVidia card -- I totally recommend going that route.
