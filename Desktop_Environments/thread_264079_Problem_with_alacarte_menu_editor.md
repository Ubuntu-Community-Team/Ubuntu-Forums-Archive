### Post by CatKiller on 2006-09-24
> **TheVelho said:**
> In fact my problem is that I don't know how to reduce the ``Application Places System'' menu bar to the single ubuntu logo in the taskbar.

Can anyone help?

Yes. Right-click on the larger menu and select Remove From Panel. Then right-click on the panel and select Add To Panel. Scroll down to Utilities and select Main Menu.

You can move it where you want with a middle-click&drag, and you can lock it in place by right-clicking on it and selecting Lock To Panel.
