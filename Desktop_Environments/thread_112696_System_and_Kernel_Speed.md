### Post by FLeiXiuS on 2006-01-05
As I told the last person, don't be greedy and try to get your self on the top of the list.  It's only been 15 hours.  Give users time.  There's plenty of help to go around, don't demand it to your self.  

The biggest difference between the two is they are designed for different architectures which your processor supports.  The biggest difference that I can see is how the program defines the instructions for your processor.  686 generally allocates more memory to the program allowing it to open faster and run at a faster speed.

386 on the other hand will send just many instructions, but neglecting to allocate so much memory to the typical program.  If the program requires quite a sum of memory then this may not be the best choice!
