An update a few days ago included KMail and KDE updates.

Since then, when I use KMail, it has no title bar, and it displays as if it were in full screen mode. I loose the top menus and also the bottom task bar (where windows are minimised).

I cannot move or resize the KMail window either, because these rely on the top title bar. I loose access to all other applications, because the bottom task bar, where apps are minimised, disappears.

All other applications I use are okay, and their windows are behaving as normal.

### Post by oygle on 2009-02-17
Thanks, I also found this post - [http://ubuntuforums.org/showpost.php?p=6746569&postcount=3](http://ubuntuforums.org/showpost.php?p=6746569&postcount=3) , was an immediate workaround.

> On the top tool bar select "System".
Then select "Preferences".
Then select "Appearance"
Then select "Visual Effects"
Then select "NONE"
Then "CLOSE"
