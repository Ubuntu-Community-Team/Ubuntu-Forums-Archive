### Post by dodo3773 on 2012-09-14
It depends. Usually just extract the theme and move the folder to "~/.themes" or "/usr/share/themes" and they should show up in your theme program. Or some theme programs have an install button or similar that you can point directly to the archive. This should be universally true unless the theme has some kind of installer in it. In that case I just avoid them altogether.
