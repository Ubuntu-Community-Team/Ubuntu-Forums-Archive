### Post by keithld on 2008-12-22
In synaptics package manager tick the following to install:
compiz-config-settings manager
simple ccsm
fusion-icon
Then under system you'll find compiz config settings manager.

Check out this link and see if that's what your looking for... [Compiz Fusion Desktop Effects in Ubuntu 8.10 Intrepid](http://reformedmusings.wordpress.com/2008/12/09/compiz-fusion-in-ubuntu-810-intrepid/)

### Post by xjcannonx on 2008-12-22
I think your talking about compiz. Compiz is what is used for the effects...like the spinning cube

It is installed by default but you will need to get the compizconfig settings manager```
sudo apt-get install compizconfig-settings-manager
```

And go in System-->Preferences-->Appearances to enable custom effects i think...Im not at my ubuntu now but i think thats it...
