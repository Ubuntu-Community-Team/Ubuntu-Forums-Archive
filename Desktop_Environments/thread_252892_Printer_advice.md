### Post by whizbaby on 2006-09-07
This printer is supported by the *gutenprint* driver (see home page [http://gimp-print.sourceforge.net/](http://gimp-print.sourceforge.net/) for info).
The packages (cupsys-driver-gutenprint) are in the repositories and installed by default.(so no downloading should be necessary, I'm only verbose for the case that the package did not install for some mystic reasons)
