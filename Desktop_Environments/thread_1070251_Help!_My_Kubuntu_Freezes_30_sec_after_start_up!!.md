### Post by Lord Landis on 2009-02-15
Did you change anything other than the login screen?  Were you editing any config files?  From what I can tell, Kwava.kss is a screen-saver file (most of what Google returns when searching for 'KDE .kss' refers to screen-savers, so this is my best guess), so the odds are very good that you either have a corrupt file specified for the screen-saver, or your install doesn't like that particular one.

I've seen the same sort of breakage when changing mouse themes (yes, I know it sounds weird, but it happened to me - I changed to a different mouse theme, and X broke).

Honestly though, since you've just performed the install, it might be easier to do a clean re-install than to try to repair the system.
