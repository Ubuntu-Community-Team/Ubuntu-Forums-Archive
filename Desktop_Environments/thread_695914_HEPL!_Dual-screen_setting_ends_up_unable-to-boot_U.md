### Post by ad_267 on 2008-02-15
You should still be able to boot into recovery mode to get a command line, here you can try running:
```
sudo dpkg-reconfigure -phigh xserver-xorg
```

This will reset your xorg.conf file back to default and with the nv driver.

Then you should be able to boot into ubuntu as normal and re-enable the nvidia driver. Instead of using the gnome screens and graphics tool type this into the console to set up both the screens:
```
sudo nvidia-settings
```

I found that the normal screens and graphics tool wouldn't detect my monitors correctly and when I set the resolution to 1280x1024 at 60Hz the monitor would display a message saying incorrect resolution, please set to 1280x1024 at 60 Hz :confused:
But the nvidia-settings tool worked great for me :)

Hopefully this will help you too
