### Post by redactech on 2005-11-07
Using another window manager can help, true but I've been using a P3 and 512mb ubuntu box with gnome It was usable. That said gnome is not easy on memory. There's way to speed it up :

[http://ubuntuforums.org/showthread.php?t=61483&highlight=speeding+gnome](http://ubuntuforums.org/showthread.php?t=61483&highlight=speeding+gnome)

Your choice isn't limited to iceWM, you have others (enlightenment, blackbox etc) but they will make your life harder because if you are not used to the terminal they'll make you use it more than KDE and Gnome...

[https://wiki.ubuntu.com/Installation/LowMemorySystems?highlight=%28install%29](https://wiki.ubuntu.com/Installation/LowMemorySystems?highlight=%28install%29)

You probably have a better choice with XFCE, it is not as feature-packed as kde and gnome but will give you a lot of non-terminal tool. THere' s several desktop installed in my 12 yrs old daughter P3 laptop and she tried them all and xfce and enlightenment were the finalists. XFCE won because it include more tools. Altough enlightenment is more beautiful and run faster

Check project page at [https://wiki.ubuntu.com/Xubuntu](https://wiki.ubuntu.com/Xubuntu)
