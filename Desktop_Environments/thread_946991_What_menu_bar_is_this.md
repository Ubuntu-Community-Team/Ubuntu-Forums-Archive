### Post by pytheas22 on 2008-10-15
Yes, I agree that it can be hard to get used to the way that AWN works.  I used it for a while but gave it up because I wasn't pleased with it.  Also it's still in heavy development and is not always very stable.

There are other docks that you can check out, though, that do similar things.  Google something like 'ubuntu docks'.  KDE in particular has a nice dock called kdock that works the way you'd expect it to and has some fancy eye-candy effects; unfortunately I've had some issues trying to run it in Gnome (it works but with effects turned on it has some redraw issues), but in KDE it works great.
