### Post by CantankRus on 2015-04-04
If your using unity-tweak-tool, it doesn't allow you to choose separate themes for decorations and windows.
As you don't appear to be using unity, install **gnome-tweak-tool** to choose separate themes.
Be aware installing  **gnome-tweak-tool** will install gnome-shell-common and a couple of other dependencies.
