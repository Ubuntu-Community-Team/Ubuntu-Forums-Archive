### Post by qamelian on 2007-04-07
> **xavier_r said:**
> When I fetch mail through Evolution it would'nt fetch any images...
So there would be blank boxes...

How do i repair this?

This is the normal behaviour and it is done for security. You can download images for a nindividual message by selecting it and pressing Ctrl-I. Or you can go to Edit > Preferences > Mail Preferences and select the HTML Mail tab and change the behaviour for all messages here.

And like I said, it's not something to repair. For the best security practice, this default setting is what would be desired.
