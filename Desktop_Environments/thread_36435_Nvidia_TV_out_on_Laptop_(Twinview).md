
I get the image on the tv, so that works fine. The issue is when there is no tv connected to my laptop my desktop gets a virtual resolution of 2624x1200 with a display of 1600x1200 so that I can pan left and right.

According to the nvidia docs the metamodes line should take care of what I want to do. A mode for displaying only on my laptop (for when the tv is absent),  a mode for displaying only on the TV for movies and then one for both screens.

You can set the size of the display by specifying something like "1024x768 @1600x1200" in the metamode, which would give you a 1600x1200 virtual desktop in 1024x768. Even specifying a size that would put my 1600x1200 desktop on a 1600x1200 screen doesn't work, I still get 2624x1200 on a 1600x1200 screen.

The second issue is that ctrl-alt +/- doesn't work to change my X modes.

Can anyone help me with this? I don't want to change my xorg.conf everytime I want to watch a movie on the TV.
