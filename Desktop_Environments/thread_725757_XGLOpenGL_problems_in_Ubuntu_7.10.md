### Post by coskierken on 2008-03-16
You did not say what version of driver you are using.  If it is 8.3 and newer, you should not have XGL loaded.  It is built into the latest drivers.  If you card is supported by the latest drivers, I would use Envy to set up the drivers. It is painless and works well.  My X1700 ATI Mobility Radeon is much smoother after I used it.  I hope this helps!
