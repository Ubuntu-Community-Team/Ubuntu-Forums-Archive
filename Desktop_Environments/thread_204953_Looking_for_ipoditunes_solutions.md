### Post by mobilehavoc on 2006-09-07
GTKPod works the best from everything I've tried. It reads the ItunesDB and writes to it directly so it pulls out ratings, playcounts, etc. The interface is meh..

What I do is use Rythmbox as my media player and GTKPod to manage the iPod (transfer stuff back and forth). GTKPod does have a great sync feature where it will sync directories to the iPod and keep them sycned on both sides.

Give it a shot..you can install using Synaptic.

### Post by brendon on 2006-09-22
Amarok does have iPod support now.  I've used it for a few months now and am quite pleased with it.  Amarok the synchronization of playlists, podcasts, song play counts and song ratings with your iPod.  The only problem I have had so far is that you have to select each playlist individually for synchronization with your iPod as Amarok is not yet smart enough to remember which playlists are to be synchronized.  Even with this limitation I would highly recommend Amarok.
