### Post by darkstaar on 2011-05-21
If you're running Ubuntu 11.04, that would certainly install Kubuntu 11.04 to your system.

Then to upgrade KDE to 4.6.3 you would add the Kubuntu PPA to your Software Sources:

```
ppa:kubuntu-ppa
```
