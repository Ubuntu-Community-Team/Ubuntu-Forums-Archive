### Post by Copper Bezel on 2011-09-18
I don't know of any way to make this work, no. The way it is now, if the same command invoked both and Scale happened first, then on the first strike, it would switch to Scale and ignore the command for the Ring, and on the second strike, it would switch out of Scale and bring up the Ring, but you'd have no way to tab through the Ring Switcher without invoking Scale again. 

On the other hand, you *can* navigate Scale from the keyboard with the arrow keys, which would get you a similar effect.
