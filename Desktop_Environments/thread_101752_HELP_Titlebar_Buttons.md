Does anyone know how to rearrange the buttons on the titlebar in Gnome?  Any help would be appreciated.

---

### Post by m0biu5 on 2005-12-10
Open the configuration editor, go  to apps > metacity > general, find button_layout and rearrange the key. For example, I have buttons on the left and my key is 
```
close,minimize,maximize:menu
```
