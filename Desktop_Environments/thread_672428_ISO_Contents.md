### Post by smartboyathome on 2008-01-19
You can mount an ISO by installing gmount-iso (from synaptic or add/remove programs) and using Applications > System Tools > Gmount-iso.

A tutorial on this can be found [here]("http://www.ubuntugeek.com/easy-way-of-mountunmount-iso-images-in-ubuntu.html").

### Post by Thund3rstruck on 2008-01-19
mount it on loopback

```

sudo mount -o loop file.iso /media/cdrom

```
