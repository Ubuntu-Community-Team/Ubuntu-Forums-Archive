1. Multimedia Codecs: [http://ubuntuguide.org/#codecs](http://ubuntuguide.org/#codecs)

2. Although you are the main user, you are _not_ root. Ubuntu does not use root, it uses sudo (yes, just like Mac OS). By default you only get permissions to write only on your $HOME directory. To learn about sudo and how to take advantage of it, check out this website: [http://www.ubuntulinux.org/support/documentation/faq/root](http://www.ubuntulinux.org/support/documentation/faq/root)
