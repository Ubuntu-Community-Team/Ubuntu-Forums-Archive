### Post by coffeecat on 2011-11-30
This will be a network manager issue. I very much doubt that Unity has anything to do with it. Click on the network manager icon -> Edit Connections -> Wireless tab. Is your network listed and saved? If yes, highlight it and click on Edit. Is the "Connect automatically" box ticked? Open the Wireless security tab. Is your password saved there?
