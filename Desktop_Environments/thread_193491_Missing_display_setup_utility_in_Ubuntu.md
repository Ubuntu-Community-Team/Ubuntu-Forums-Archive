### Post by Rondonjin on 2006-06-10
In the Administration menu of another distro running on my old laptop there is one more (at least) utility: system-config-display. If it's a Gnome utility I'm surprised that it's missing from Ubuntu. It is for setting up your monitor and video card, changing the colour depth and turning on dual display. Ubuntu sets up the display to run at 24bits. On a Thinkpad with only 2.5Mb of video memory do they realise how painful that is! I had to edit xorg.conf by hand to set 16bit as the default. That is not very noob friendly, and I thought that was one of their aims. This machine's onboard intel graphics uses a measley 4Mb of system memory. Luckily I had an AGP ATI 2X video card with a more generous 8Mb laying about!
