### Post by fabounet on 2009-10-26
if they have been packed with the dock, just open the Theme Manager, drag and drop the tarball from nautilus inside, check the 2 little boxes if you want, and apply.
otherwise, you can always unpack them in ~/.config/cairo-dock/themes, then select the theme in the Theme Manager as usual.

As far as I know, only the themes proposed by the Theme Manager (around 30 themes !) are correctly packed. The themes on gnome-look.org are almost all broken (missing files or very old config)
