### Post by lammert-nijhof on 2017-09-03
For Ubuntu 16.04 and 17.10 (Ubuntu/Gnome Beta) You can select the launcher placement display in the settings program in the display tab. There is also an on/off button for each display. I only have one display now so it is grayed out for me. For Gnome 17.04 it might be different.
