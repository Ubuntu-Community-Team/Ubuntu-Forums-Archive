### Post by 235235 on 2007-02-18
Yes, it is simple to change the default application for different file types. Open the file manager (Konqueror), find a file of the kind you want to use, right-click, and choose "Open with" and then "Other" (even if the program you want is in the list of options). Choose the application you want to use as default from the menu, check the box that says "Remember application association for this type of file", then finish by clicking on "OK". You will see a short progress indicator that says "Updating System Configuration". 

You can accomplish the same task from the Settings/ Configure Konqueror/ File Associations within Konqueror.
