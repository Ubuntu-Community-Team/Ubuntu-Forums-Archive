### Post by zerwas on 2009-04-28
> So if you are running into this error, all you have to do, is:

   1. exit all qt applications,
   2. go into &#8220;System>Preferences>Appearance>Fonts&#8221;,
   3. change VRGB/BGR to RGB/BGR.
from [here]("http://www.screenage.de/blog/2009/04/21/possibly-scrambled-qt4-apps-in-ubuntu-jaunty-jackalope-while-using-vrgb-and-lcd/")

Fix should be in progress: [https://bugs.launchpad.net/ubuntu/+source/qt4-x11/+bug/334657](https://bugs.launchpad.net/ubuntu/+source/qt4-x11/+bug/334657)

-- zerwas
