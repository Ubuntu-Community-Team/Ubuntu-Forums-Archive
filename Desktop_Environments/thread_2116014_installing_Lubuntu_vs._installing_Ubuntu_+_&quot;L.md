### Post by iMac71 on 2013-02-14
the difference is that if you choose to install lubuntu you'll only have the applications of that flavor, while if you choose to install ubuntu and then lubuntu-desktop you'll have the applications of both of these flavors and you'll be able to choose what desktop environment you wish to use each time  you do login.

### Post by offgridguy on 2013-02-14
To add to the previous reply, there is less default software installed with lubuntu than 
there is with ubuntu + the LXDE desktop.  If you desire the extra features, the latter is the way to go, if you want a simpler but faster system then lubuntu is the better choice.

I use lubuntu most of the time,ubuntu when i need features not available by default in 
lubuntu.  :)

### Post by Gaygerbil on 2013-05-12
Unless you want Unity and all it's features there's no point in installing Ubuntu with the Lubuntu Desktop. Lubuntu can run everything Ubuntu can run outside of the Unity/Gnome environment.
