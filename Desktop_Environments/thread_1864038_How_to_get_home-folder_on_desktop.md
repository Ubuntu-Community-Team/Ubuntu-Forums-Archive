### Post by kriegoamigo on 2011-10-18
Hello Azyx

Brother you can make a launcher but you cannot bring the complete home folder to your desktop as the desktop folder itself resides in the home folder....

You can follow the following steps to create a launcher...
Its really simple and I think you must also be aware of it but still :)

Go to your desktop -> right click -> create launcher

then you will get a window asking for type, name , command and comment

type -> location
name -> Home-Folder
command -> /home
comment -> your comment


I hope this helped you :)

### Post by Frogs Hair on 2011-10-18
You can move the folder with right click > move to desktop , but the folder still needs an application to open with and will use the file browser by default . The desktop is not an application that will open folders and doesn't appear on the open with another application list  .
