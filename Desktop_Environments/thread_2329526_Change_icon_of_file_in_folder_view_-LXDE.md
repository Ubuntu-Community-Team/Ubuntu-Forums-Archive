A thread where this subject was discussed in May 2015:

[http://ubuntuforums.org/showthread.php?t=2278819](http://ubuntuforums.org/showthread.php?t=2278819)

The reference article for the method:

[http://unix.stackexchange.com/questions/23776/how-to-change-a-file-type-icon-in-xfce-thunar](http://unix.stackexchange.com/questions/23776/how-to-change-a-file-type-icon-in-xfce-thunar)

It would think it will be the same with other file managers, not just Thunar.
