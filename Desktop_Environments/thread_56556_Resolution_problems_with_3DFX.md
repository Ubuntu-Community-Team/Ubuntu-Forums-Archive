### Post by xmastree on 2005-08-13
Why does it seem unlikely? **It worked!**   :grin: 

I found this:
```

Section "Monitor"
        Identifier      "Daewoo CMC-1707B"
        HorizSync       30-70
        VertRefresh     50-160
        Option          "DPMS"
EndSection
```on a website somewhere, so I pasted it in.

Didn't work first time though, as the Identifier wasn't the same and I forgot to change it lower down...  :roll:

Thanks!
