### Post by Zelut on 2006-01-14
You could check that it is displaying mounted media on the desktop.  If you're using gnome you can look here (KDE, you're on your own...)

Applications > System Tools > Configuration Editor > Apps > Nautilus > Desktop & check: Volumes_visible (this'll display mounted CDs or other partitions you might have).  Also available there are display 'computer' icon, 'trash can', etc..
