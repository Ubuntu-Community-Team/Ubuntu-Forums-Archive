### Post by JKyleOKC on 2012-09-14
My solution to a requirement that I use MS Publisher (our print shop requires it) was to install VirtualBox to create a virtual machine, then install XP Pro in that VM, and finally install Publisher in that copy of XP Pro. It's been working flawlessly for some five years now, and survived a move from one Xubuntu box to another along the way with no need to reinstall anything other than VBox itself on the new machine. Simply copying the VM files across to the new machine transferred everything else.

I highly recommend this approach, but only if you have at least a dual-core (or hyperthreaded) processor and at least 2 GB of RAM. It's much less complicated to use than is WINE!
