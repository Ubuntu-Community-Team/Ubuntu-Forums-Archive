### Post by zevs on 2007-03-28
I know this post is pretty old, but i just managed to solve this problem on my own laptop. Hopefully someone needs this.

So, just like you i have looked through tons of posts and guides on how to install the drivers correctly and set the resolutions properly. Of course, none of those things mentioned worked. My screen resolution is 1024, so what i need is really hdtv playback.

I found that there is an offscreen buffer in the i810 driver that needs to be set up properly. As default it is set to 0kb. According to the i810 driver manpage you have to set it to 6144 (6MB) for playback up to 1080. Though this gave me only playback of 720 video, i assume this may be because most 1080 really is 1088. Anyways, i put in 32MB (i'm sure it's too much, but just to be sure) and i got flawless playback of hdtv video, both 1080 and 720.

All you have to do is this:

Edit to your xorg.conf file.
Find the device section for your display driver (it says something like 'Driver "i810"' or similar)
Add the line 'Option "LinearAlloc" "32768"
Restart the x server with ctrl+alt+backspace
..and you are all set. i have testet with all my players, and none of them crash anymore :=)
