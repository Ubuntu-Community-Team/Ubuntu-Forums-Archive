### Post by Little Bones on 2010-01-02
If you go to your Synaptic Package Manager, and search 'Compiz', you should come across the Compiz Compositing Manager. Install the package named 'compizconfig-settings-manager' and its dependencies. If you then go to System->Preferences->Compiz you'll have the ability to edit all sorts of effects.

For the one you want, go to Effects->Animations. You'll see the tab there for 'Minimize'. There's where you can edit the minimize effect to magic lamp (basically vacuum). I have mine set to a time of 400 and I think that looks quite nice.

So for nerdtron that might be to your liking.

To get it without the wavy bit you'll have to ask someone else, or google it. I'm quite happy with the wavy bit personally.
