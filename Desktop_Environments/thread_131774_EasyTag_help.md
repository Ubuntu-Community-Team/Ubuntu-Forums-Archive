### Post by waynemcl on 2008-03-17
Hi 

Docs at [http://easytag.sourceforge.net/EasyTAG_Documentation.html](http://easytag.sourceforge.net/EasyTAG_Documentation.html)

In summary:

If you need to fill tags, check that the option “Overwrite fields when scanning tag” in the “Scanner” tab of the “Preferences” window is set.

Press the green "Scan files" button. This brings up the scanner window. 

Select the options you need, then press the "Open scanner window / Scan selected files" button (This has the same green icon as the other button! That confused me for a while...) from the new window.

This will now perform the operations on the selected files (only).
