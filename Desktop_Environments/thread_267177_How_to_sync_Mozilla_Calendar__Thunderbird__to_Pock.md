### Post by k_smolka on 2006-09-28
Over the past year I have been trying to get Ubuntu to talk to my windows mobile device without luck, after following tutorial after tutorial I eventually gave up trying.

I recently stumbled across this neat application which allows Thuderbird and Mozilla calendar to sync with my mobile device over the http protocol. The server part of the application is written in Java so will pretty much run anywhere and the client side application is written in c# so there is no need to install a JVM on your PDA.

This application allows 2 way syncing with local calendars but also allows me to sync my Google calendar to my PDA as Mozilla calendar supports remote calendars.  There is also a version available for smart phone.

Please note that this application can also work with sunbird and lightning but there are problems with version above 0.3 but there is a work around available, see the site for details.

Enjoy

Application can be download from:  [http://www.finchsync.com/download.html](http://www.finchsync.com/download.html)

Support forums are available at: [http://www.finchsync.com/forums/](http://www.finchsync.com/forums/)
