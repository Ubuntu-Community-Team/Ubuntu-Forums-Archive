@OP
if you have had your ubuntu pc connected to the internet and had package lists downloaded. 
go to synaptic, select files to install, 
go to file-->generate package download script, 
then you can use the web addresses in that to down load packages, put thoses into /var/cache/apt/archives then go back to synaptic and install what was selected.

if you havent got the package lists, or you want to update it this [tutorial is pretty straight forward]("http://www.planetoss.com/detail.php?id=13")
