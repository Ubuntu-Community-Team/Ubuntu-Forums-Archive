### Post by Sendervictorius on 2006-02-23
gnucash does not automatically add to the menu. You need to add it yourself using Application Menu Editor (Under Applications > System Tools > Application Menu Editor.

Edit: Choose "New Entry"

Just type "gnucash" in each text box, and find the icon by clicking on the icon, and choosing from usr/share/pixmaps/gnucash.
