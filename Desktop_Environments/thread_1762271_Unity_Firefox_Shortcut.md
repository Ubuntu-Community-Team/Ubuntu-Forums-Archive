### Post by mcduck on 2011-05-19
> **Ujaco said:**
> When I press the super button the launcher pops up . Each Icon shows a number or an alphabet to press to open that application. But Firefoxshows no such letter or number. Can someone tell me what the default shortcut is or how I can configure my keyboard shortcut for firefox?

How many launchers you have in your panel? You only get the quick shortcut for 10 first launchers, plus the three built-in lenses (Workspace Switcher, Applications and Files & Folders). There are only that many number keys... ;)

If you want to define custom shortcuts, that can be done in Control Center->Keyboard Shortcuts. Or in the CompizConfig Settings manager with the Commands-plugin.
