I installed Inkscape 0.45 using the Autopackage build. While the application works fine, the installation seemed to create two duplicate menu items in the Applications->Graphics submenu. Now I've uninstalled it (going to install 0.45.1) but only one of the two menu entries is removed. Upon using System->Preferences->Menu Layout the remaining Inkscape launcher does NOT appear in the checklist, although it's there in the menu. 

While it's a minor issue, how can I get that remaining Inkscape launcher out of the menu. Is there a way to do it manually?

### Post by floke on 2007-03-22
If you right click on the menu (rather than left click to initiate) you will have the option to edit menus - simply uncheck the box for the thing you don't want and it won't appear.
