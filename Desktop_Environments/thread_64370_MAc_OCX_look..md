### Post by Freddy on 2005-09-11
> im mostly interested in the dock but a link to anything on info on how to do/get it would be great
 thanx guys..
In that screen I used Kxdocker, which is a highly configurable Maclike docker for KDE and since the 0.37 even for Gnome. Grab a copy of the resource package to (alot of themes and plugins).
You can find Kxdocker @ [www.xiaprojects.com/www/prodotti/kxdocker/main.php](www.xiaprojects.com/www/prodotti/kxdocker/main.php)

The icons I used was NuvoeXT 1.5 which is easy to find @ [www.kde-look.org](www.kde-look.org)
If you like the Maclook of things, maybe you should grab the OS-L icons, you can find those @ [www.kde-look.org](www.kde-look.org) to, a new screen of mine showing those @ [http://ubuntuforums.org/gallery/showimage.php?i=787&c=3](http://ubuntuforums.org/gallery/showimage.php?i=787&c=3) 

Both the windowdec and style is Baghira 0.6e (I couldn't compile the much better CVS version, but hopefully it's not long until 0.7 is out) You can find the Baghira theme and a little more stuff @ [http://baghira.sourceforge.net/index.shtml](http://baghira.sourceforge.net/index.shtml)
