Hi,

It is a minor thing but it's been bugging me: I prefer the menu layout that has the "Application", "Places" and "System" across the top vs. the single menu with just the Ubuntu logo showing.  

When I built my Ubuntu system, I installed regular Ubuntu 9.10 and later upgraded to Ubuntu Studio.  Now, all of the original users have the 'normal' menu layout and all subsequently created users have the 'studio' layout.  I've searched around trying to find out where the menu files are stored so that I could just copy the layout from one user to the other but I've come up dry on the subject.  What do y'all suggest?

Mark

### Post by mister_playboy on 2010-02-10
If you right click on the panel and pick "Add to Panel" you can search for the "Menu Bar"... this is the Apps/Places/System menu.  Just delete the "Main Menu" Studio uses in the panel and replace it with Menu Bar.
