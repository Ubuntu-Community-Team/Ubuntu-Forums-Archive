It will result in slightly larger memory footprint, but if you are already using any GTK applications the difference isn't really that big unless you are seriously low on RAM.

And if you are low on RAM, the most important thing is to stick to apps from one desktop environment, so when you need apps that have lots of DE dependencies stick to apps either from Gnome or KDE but not both and things should stay tolerable.
