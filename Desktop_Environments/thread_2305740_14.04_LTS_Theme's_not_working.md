### Post by buzzingrobot on 2015-12-08
Where and how are you installing the themes?  What's their source?  Have you installed a PPA and then installed the themes from that PPA? Or, have you downloaded zipped archives from GitHub or elsewhere. If the latter, you should check the README file or the site to see if the themes have dependencies. Those should be installed automatically by the package manager if you're pulling the theme packages from a PPA.  They won't be if you manually installing from an archive. (gtk2-engines-murrine and gtk2-engines-pixbuf are two common dependencies.)

You also need to use themes built for the GTK version used in 14.04.  (3.12, I believe).  Themes requiring later GTK versions will probably not work well on 14.04. (GTK is a large software toolkit used to make the widgets, etc., themes manipulate.)
