### Post by komencanto on 2006-06-30
Take a look at the VLC player website and compare:

[http://www.videolan.org/vlc/download-windows.html](http://www.videolan.org/vlc/download-windows.html)
[http://www.videolan.org/vlc/download-ubuntu.html](http://www.videolan.org/vlc/download-ubuntu.html)

For Windows it is possible to download an .exe file on a computer with internet access and pass it around as much as you want. With Ubuntu and Debian in general you are only given instructions for how to apt-get the program - assuming both that you have access to an Ubuntu machine, and that it has a high-speed connection to the internet.

I am aware that you can get .deb files that are similar to .exe files, but they do seem to be rarely used and hard to get.

I think this is an important thing to consider - not everyone who wants to use Ubuntu has access to the net (think people in South Africa or poorer countries in general), and even if they do it may well be a slow dial-up one. For them, getting exes off of a disk is a much easier way of installing things than apt-get.

Is there an easy way to get around this problem?

### Post by JoshHendo on 2006-06-30
From [http://packages.ubuntu.com/](http://packages.ubuntu.com/) it is possible to download any program that you can get in apt as a deb file. This way would be a good way to get the files for people without a internet connection.

Another way would be to order dvds with every program in apt for ubuntu, but that costs about 30$.
