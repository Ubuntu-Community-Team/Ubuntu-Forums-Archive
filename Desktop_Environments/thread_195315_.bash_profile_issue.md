### Post by Thund3rstruck on 2006-06-12
Ok, so I noticed in the .bash_profile file the following line

# set PATH so it includes user's private bin if it exists
if [ -d ~/bin ] ; then
    PATH=~/bin:"${PATH}"
fi

So I created a /home/linuxuser/bin folder and moved some scripts into it. However, when I attempt to execute any of the scripts I get a not found error. Running echo $PATH does not produce a ~/bin in the output. 

So I'm wondering if this segment is suppossed to add ~/bin into my path, why isn't it working?

---

### Post by taurus on 2006-06-12
Okay, add these two lines to the end of your ~/.bashrc then.
```

PATH=$PATH:~/bin
export PATH

```
Now, you need to read the new changes in ~/.bashrc again by running this command from a terminal.
```

source ~/.bashrc

```
