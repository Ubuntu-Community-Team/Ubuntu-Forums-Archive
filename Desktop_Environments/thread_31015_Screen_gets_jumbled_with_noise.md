### Post by fdimeglio on 2005-05-08
Last but not least, it seems that there are two drivers for the NVidia GPU:

- "nv": seems to come from Xorg
- "nvidia": comes from NVidia

Switching to NVidia driver solved this issue but be aware that there is a lot of mess around it. If you do a search into the Ubuntu forum with "nvidia" you will see what I mean  :???: 

BTW, [http://ubuntuguide.org/#installnvidiadriver](http://ubuntuguide.org/#installnvidiadriver) gives you all the infos you need to install the NVidia driver.

After install, you need to CTRL+ALT+Backup to kill your X and then relaunch it with a new session.

If the NVidia driver is correctly install you should have the NVidia splash showing up before you have the Ubuntu login; then after login, open a Terminal and try "glxinfo" or "glxgears".

Let me know if it solves your problem.

Fabrice
