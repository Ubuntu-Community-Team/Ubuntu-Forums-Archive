Hi,
I'm looking for an equivalent to the Windows "run" dialog. I'm aware of the application launcher that comes up with Alt+F2, but I often want to open a folder using the run command, and with Alt+F2, I have to type the full folder name. there is no autocomplete (as exists, for instance, in the terminal).

Do any alternatives exist that have autocomplete for folder navigation?

tia....

### Post by Rocket2DMn on 2008-03-19
Under Nautilus, where you see the path at the top, if you click the pencil/paper icon on the left, you can type out a folder's location on the filesystem in that address bar.  It accepts TAB autocomplete (and should show a visual autocomplete).
