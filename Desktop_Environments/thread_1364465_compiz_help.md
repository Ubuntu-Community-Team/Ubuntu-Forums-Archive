### Post by gradinaruvasile on 2009-12-25
After the settings manager install fusion icon too. Makes life soo much easier.
The burn effect is part of the animations category. You must assign it for actions you want: close window, open window. You cannot assign it for other (shade, focus) animations. 
In the configuration editor you have 3 window categories: 
 - normal - the big normal windows, 
 - menu - dropdown menu lists + misc smaller windows, 
 - tooltip - the tooltips (yellow background small text boxes) that appear when hovering with the cursor over an item (that has associated tooltips of course)

You can select different animations for opening/closing separately for the above 3 classes. In the list there is the burn animation also. Be aware that the burn animation is quite resource intensive and used with big windows can be slow.
