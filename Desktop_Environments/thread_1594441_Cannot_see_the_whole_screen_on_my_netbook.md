### Post by roggenschrotbrot on 2010-10-12
most applications don't play well with the tiny vertical resolutions netbooks offer, and its not a linux only problem. there are a few ways to handle this:
[LIST=1]
[*]you can move a window by pressing alt and left mouse button
[*]you can move/autohide your panels (if using gnome-panel instead of unity) by right-clicking them and select autohide/orientation in its preferences
[*]you can use gnome-color-chooser and set the widgets spacing to 0. i highly recomend this as it results in much more compact windows
[/LIST]
