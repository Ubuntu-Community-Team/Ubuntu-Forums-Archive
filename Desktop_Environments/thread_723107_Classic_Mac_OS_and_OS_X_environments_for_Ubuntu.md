### Post by kapello on 2008-03-13
I customised my Ubuntu to resemble Mac OS X in several ways - don't know of a true 'one-step solution' though.

Try searching gnomelook.org for Mac themes, also the Mac4Linux project at Sourceforge: [http://sourceforge.net/projects/mac4lin](http://sourceforge.net/projects/mac4lin)

Other things are just twiddling the standard Gnome interace (assuming you are using GNome - I think it's already more mac-like than KDE).

For example if you do the Preferences for the toolbar at the bottom of your screen, you can make the bar much fatter, uncheck 'expand', set the background to transparent, and it already starts looking more like the Mac Dock. 

Icons and window controls can be got from one of the gnomelook.org themes, and if you google 'mac fonts linux' you should be able to find versions of Lucida Grande and other Apple fonts, which make a huge difference to the look. Find some mac-like wallpaper off google, etc....
