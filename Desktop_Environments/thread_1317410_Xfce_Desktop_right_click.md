### Post by Brandon Williams on 2009-11-06
Settings -> Desktop

On the Menus tab, make sure that "Show applications menu on desktop right click" is checked.

On the Icons tab, make sure that "Icon type" is "None" or "Minimized applicaiton icons". If it's "File/launcher icons", you'll get all that extra stuff in the menu.
