Actually, 10.10 is no LongTermSupport version, you may mean 10.04:
[https://wiki.ubuntu.com/LTS](https://wiki.ubuntu.com/LTS)
> **Gajendra1987 said:**
> 
Problem is when i am upgrading i am getting very low speed(4000 B/s on an average always). i was fed up with that, Though my internet download speed is far better then this.
I am from India, is there any way to locate a nearest server so that i can get faster speed.
I know that there are regularily issues with the servers in India, try this:
- go to "System -> Administration -> Software Sources"
- click at the dropdown menu "Download from"
- choose "Other"
- click at "Select Best Server"
> **Gajendra1987 said:**
> 
Can i upgrade my operating system to NetBook Edition ?

If you plan to run the Netbook Edition, I recommend installing Natty 11.04 right-away, because it includes the new full-featured Unity, whereas to 10.04 you can only install the old Netbook Remix (there is no way to install Unity, really), and to 10.10 you can only install the slightly buggy half-featured early version of Unity.

If you don't want to wait for the final release of Natty 11.04, you can install its current Beta, after upgrading all its packages it should run fairly stable, since it  is set for final release at the end of the month.
[]("http://www.ubuntu.com/testing/natty/alpha3")[http://www.ubuntu.com/testing/natty/beta](http://www.ubuntu.com/testing/natty/beta)

Greetings.
