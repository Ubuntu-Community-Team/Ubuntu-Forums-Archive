### Post by ankspo71 on 2010-09-19
Hi,
Go to [http://gnome-look.org/](http://gnome-look.org/) and browse through Compiz and Beryl themes. There are many to choose from over there.

Or you could get the theme packages from Ultimate Edition by adding their repository: [http://ultimateedition.info/ultimate-edition-repository/](http://ultimateedition.info/ultimate-edition-repository/) 
Instructions on how to add the repository are included on that page.
Individual packages can be found here:
[http://downloadue.info/repo/](http://downloadue.info/repo/)

Once you have a theme imported and selected in Emerald Theme Manager it is necessary to use this command:
```
emerald --replace
```
to activate the emerald theme. To make emerald automatically run at startup, you will need to make an entry in your startup applications list using the same command.
Hope this helps.
