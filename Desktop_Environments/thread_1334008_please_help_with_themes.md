gtk2 themes from [_[COLOR="DarkRed"]here[/COLOR]_]("http://www.gnome-look.org/index.php?xcontentmode=100&PHPSESSID=47d5871570c184da8d6464f5363a065f") will change your window and panel appearance.
Just drag and drop in the system > preferences > appearance    window.
Sometimes you have to extract the downloaded file first to get to the gtk2 archive before dropping in the window.
Another window decorator is emerald.
If you want to install emerald themes , install emerald through synaptic package manager.

You can also download different icon sets from gnome-look.
Extract the icon set from your downloaded archive.
To install icons, open your home folder and press ctrl + h to show hidden files.
Place your new icons in the .icons folder in your home directory.
If there is no .icons folder, create one. Don't forget the "." prefix.
Then in system > preferences > appearance
click on the customize button to change your icons.

Here's a nice icon set you might want to try
[_[COLOR="DarkRed"]Gartoon Redux[/COLOR]_]("http://www.gnome-look.org/content/show.php/Gartoon+Redux?content=74841")
This one is packaged in a deb file.Download from the *Debian	(Debian installer, 7.6 MB)* link.
Just double click the file to install.
