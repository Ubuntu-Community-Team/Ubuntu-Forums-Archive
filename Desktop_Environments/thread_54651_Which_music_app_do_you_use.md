Rhythmbox: clean and simple UI, but lacks some of the features of the others players
XMMS/BMP: good if you want a traditional playlist-based, but not if you want a library-based player.

### Post by doclivingston on 2005-09-07
[QUOTE=sandmanfvr]XMMS, only thing I need.  I don't need some big ugly piece of crap program like iTunes taking gobs of memory.  XMMS is like Winamp and it is small, does what it is designed to do without being a resource hog, but it still has alot of features.  iTunes is fine when I sync my iPod up, but other  than that, iTunes or programs like it are just resource hogs.[/QUOTE]

I don't know why everyone say they use xmms, et al, because they use less memory - because (for me at least) it doesn't really.

As a quick comparison I started up some of the music players I have installed which have my ~3000 song library loaded, started playing, and then immediately checked the memory usage. The memory size listed is the heap size (RSS - shared) because the most of the shared libraries are either going to be in memory anyway (gnome, etc) or will be loaded by all the players (codecs).


Rhythmbox cvs + a few additional patches (self-compiled): DAAP support (2 iTunes' on the lan), audio cd support (audio cd inserted, musicbrainz lookup done), audioscrobbler support (logged in) and several complex automatic playlists (top 50 songs, 3+ star, 4+ star, 5 star, not played in last 3 days, unrated spngs, australian artists, new tracks in last 2 weeks and 3 normal playlists). Total heap size = 12.9Mb

XMMS 1.2.8.cvs.20050209 (breezy): entire library loaded into the playlist. Total heap size=10.3Mb

Quod Libet 0.1.1 (breezy): cover art for 50% of my library, no playlists or anything special. Total heap size=22.5Mb

Muine 0.8.3 (breezy): covert art for 50% of my library, other mono apps running. Total heap size=19.2Mb

Banshee 0.9.4 (self-compiled): audio cd inserted, musicbrainz lookup done, other mono apps running. Total heap size=48.6Mb



Taking into account the memory used by loading cover art, all the players except Banshee use about the same amount of memory.
