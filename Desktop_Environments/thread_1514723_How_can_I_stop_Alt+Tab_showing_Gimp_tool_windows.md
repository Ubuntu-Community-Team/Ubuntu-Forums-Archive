### Post by KonfuseKitty on 2010-06-21
When pressing Alt+Tab to switch between applications, I notice that it's actually all open windows that are presented, including tool windows in Gimp. Is there a way I could configure this so only document windows are presented, or better still, only applications like on Mac OS X?  Or a third party solution?

---

### Post by KonfuseKitty on 2010-06-29
In case this helps someone else... I've finally found the solution.  I downloaded the CompizConfig Settings Manager, and enabled the Static Application Switcher.  In Behaviour settings I set Switcher windows to Normal only.  That's it, Gimp's utility windows are no longer showing up.
