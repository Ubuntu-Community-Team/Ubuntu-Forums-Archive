### Post by Copper Bezel on 2011-04-18
The setting is in the Viewport Switching tab of the Desktop Wall plugin. By default, Compiz treats desktops as non-sliding windows because the desktop is present on every workspace.

### Post by mc4man on 2011-04-29
You could try adding Desktop to the Desktop Wall > Viewport Switching > non sliding windows box
Note that your systray icons will continue to slide, the systray is not actually in the unity panel itself even though you see them there
(not an issue here, I use rotate instead of wall

Edit: see this post, uses type=Desktop instead of just Desktop, don't see any diff though I'd defer to those more knowledgeable
