> Install Ubuntu Tweak, it has an option to change the thumbnail sizes... [www.getdeb.net](www.getdeb.net)
Actually the best way to install it is from Launchpad. The UTweak Downloads page[http://ubuntu-tweak.com/downloads]("http://ubuntu-tweak.com/downloads") has all the information you need to grab it from there, along with making sure it's updated (although UTweak is able to update itself).

Click on System on the left side then Nautilus. It's the first changable setting.
