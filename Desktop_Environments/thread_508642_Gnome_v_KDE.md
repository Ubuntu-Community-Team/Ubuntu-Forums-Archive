### Post by ikkefc3 on 2007-07-25
KDE is better, because it takes less memory than gnome, its faster and it looks nicer.
KDE also detect's changes automatically, where gnome doesn't (e.g. in your music collection in Amarok compared with Banshee)
But if you use KDE, you should go to sessions in kcontrol and set that it always starts with a blank session and not saves de previous session (or KDE will boot up really slow after a while). 
And KDE writes directly something to for example your USB flash driver, but in Ubuntu you first have to unmount it. This isn't very useful if you want to delete something and after that write something else at your USB flash drive. In Ubuntu:
- Delete
- Unmount
- Mount
- Paste your new files
- Unmount
- Mount
Kubuntu:
- Delete
- Paste your new files
Also: try to install a font in ubuntu...

And KDE isn't windows like...
