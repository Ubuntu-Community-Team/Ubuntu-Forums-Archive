### Post by orb9220 on 2006-08-15
The firmware is not specific to the OS you are running. The firmware is binary code burned to a chip inside the burner. It is not a driver. It's firmware. So if you burn the firmware to your burner in windows it dosn't change because you booted into linux. All firmware updates will stay no matter what OS you are running.

Hope this helped.
