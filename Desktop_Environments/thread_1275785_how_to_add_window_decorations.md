### Post by benerivo on 2009-09-26
Unlike gnome, some theming of kde has to be installed with root privileges. So there is no drag and drop, or one click install for window decorations in kde4. 

Taking the example of [Crystal]("http://www.kde-look.org/content/show.php/Crystal?content=75140"). You download a .tar file that includes instructions for building the package, where you install it in the same way as most other most other linux software is installed. In the crystal example there is a file called INSTALL which provides the instructions.
