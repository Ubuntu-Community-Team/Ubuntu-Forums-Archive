### Post by peakpc on 2009-03-28
There are many menus that can be used however the most basic that is built into gnome is "main menu". Just right mouse on the panel and "add to panel" then drag main menu into the panel.

It works but I found that on first click (after bootup) it takes a long time to populate, after that it seems to work at normal speed. I found this to be annoying, so I found and use this solution [http://ubuntuforums.org/showthread.php?t=89136&page=2]("http://ubuntuforums.org/showthread.php?t=89136&page=2") it does not change the menu layout but you can make it take up much less panel space. I use "Apps GoTo Sys".:)
