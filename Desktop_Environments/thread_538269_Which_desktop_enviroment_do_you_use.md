``` and see the list of what it will install. The -s option means that the install will be simulated, and if your satisfied with what will be installed, repeat the command without the -s option.

Another option is to do the same but with ```
 sudo apt-get install -s xfce4

### Post by Rupertronco on 2007-08-30
If you're comfortable messing around, I think it's a great idea to have multiple DEs installed. KDE and Gnome don't share any of the same settings so right now i have KDE with it's dock and desktop setup for web site developing and photo editing/manipulation. Gnome's setup for every day use. Xfce/Fluxbox are on my laptop in addition to KDE+Gnome for battery consumption. 

If you've got the HD space there's really no disadvantage to having multiple environments installed, you just have to make sure you weed out the packages you don't want. In addition: if you don't keep your menus clean, they get a bit jumbled with the kubuntu desktop package in addition to a gnome install.

### Post by tbrminsanity on 2007-09-08
I tried Project Looking Glass yesterday (via LiveCD).  I really like it.  If there was an easier way to change background colours and window themes I would so switch over.  It fully utilizes multiple screens and the GUI effects are on par with the best other desktop GUIs can present.  If you haven't tried it already I strongly recommend it.  It's optimized for Solaris 10 but it is available for most unix OSs (including a .deb file for Ubuntu).

Good Points:
+ True 3D desktop with full utilization of depth
+ Great multiple screen/desktop support
+ Mac style doc bar with more functional abilities (like stacking apps, and creating application "start" buttons)
+ Easy to learn and use
+ Access to terminal for advanced users
+ Works for BSD, Solaris, Linux, Mac, and even Windows
Bad Points:
- While it is easy to switch between exisiting backgrounds, adding new ones is a PITA
- Can't easily apply desktop themes
- Can't move the doc bar from the bottom of the screen
- No pop-up menus with right click (it actually closes the app)
- Still in beta and full of bugs (though over 1000 have been fixed already)
