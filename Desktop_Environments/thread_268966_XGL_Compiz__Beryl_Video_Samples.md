### Post by JayTee on 2006-10-26
To activate Skydome and set a background go to Beryl Settings Manager. Click on the Desktop Cube in the left pane of the window.
On the Choices tab make sure Skydome is selected.
Click on the Filenames tab. 
There are three filename options to set here. One is for the top of the cube
One is for the bottom and one is for the background. That should do it!
One word. The images have to be in png format and I've found that scaling them to 1024x768 or 1024x512 works best but you can try other options. Use the Gimp tool in the Graphics menu to manipulate the image and Save As a different file type. Enjoy!

JayTee
