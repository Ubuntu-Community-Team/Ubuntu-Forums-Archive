### Post by 3Miro on 2010-01-11
Might be a KDE 4.4 issue, but on all alder versions of KDE, Gnome and XFCE you get 2 clipboards. One is activated with select + middle click, the other one use the standard Ctr + C, Ctr + V. Also, right click copy + right click paste also works just as it does in windows.

Basically, the select + middle click is a perk, not the one you would use to copy over a text.

### Post by benerivo on 2010-01-11
I think it's in the klipper options (right click the klipper icon in the system tray). You need to tick 'Ignore Selection' so that it doesn't copy selected text.
