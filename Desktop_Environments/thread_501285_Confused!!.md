### Post by Koybe on 2007-07-15
I think if you want to get started, don't mess too much with those things. Just take Ubuntu and go for a ride ;)

Anyway : 

- Ubuntu as Gnome as a Desktop Environment. Which i easy to use and offer standard use.
- Kubuntu comes with KDE as a Desktop Environment. Which as more options and customizations. It feels a little more like Windows.
- Xubuntu comes with XFCE as a Desktop Environment. Which is a lightweight Desktop.

This is a new think you'll got to leran with open-source software. You always got a choice. And then... you'll find many flaming discussion about the better use of one or another but this is just a question of taste ;)
