### Post by mustang on 2006-02-17
I'll try to answer the questions I can.

> currently I make copies of my movies using dvd decrypter and dvdshrink, are there  any compatable utilities that function remotly the same ? 

I think you'll find this guide useful:
[http://ubuntuforums.org/showthread.php?t=78611](http://ubuntuforums.org/showthread.php?t=78611)

However, I'd first suggest searching for "dvd decrypter equivalent" or "dvdshrink equivalent" or something along those lines before trying to the guide above. I'm saying this because you might find a program that does the job rather than wasting time emulating it. 

> are there any utilities that work remotely like Nero or any versions of Nero for this distro, if so what are they ? 

gnomeburner and k3b are good cd burning programs like nero.

>  I also have a kodak digital camera that isusb, what software would be good to import the pictures to and from it ? 

Try connecting it to your computer and see if it autodetects. A program will automatically start up that lets you import/delete photos.
