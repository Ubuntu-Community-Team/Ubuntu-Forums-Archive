[http://www.linuxcanada.com/](http://www.linuxcanada.com/)
Much like quickbooks pro.

Add this line to /etc/apt/sources.list
deb [http://adfinity.entercall.co.za/~iburger/quasar/hoary-amd64](http://adfinity.entercall.co.za/~iburger/quasar/hoary-amd64) ./
for  Ubuntu amd64 version or
deb [http://adfinity.entercall.co.za/~iburger/quasar/sarge-i386](http://adfinity.entercall.co.za/~iburger/quasar/sarge-i386) ./
for i386 version
