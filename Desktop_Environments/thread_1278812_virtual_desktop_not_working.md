### Post by ooburns on 2009-09-30
Did you have any open windows when you tried? If desktop effects aren't enabled, the virtual desktops will switch instantly with no visual feedback, so if there aren't any open windows, you won't be able to tell it worked.

If that's not the problem, add the pager applet to a panel and see if you can switch using that.

### Post by coldReactive on 2009-10-01
> **infernox said:**
> Yes, thanks for the response. I can switch if I use the applet but my keyboard shortcut doesn't work. I tried the Windows key+E causing it to zoom out.  I tried CTRL+LEFT or RIGHT and nothing. What function should I look for in the keyboard shortcut setting?

Your desktop effects are enabled then.

Install ccsm, and find compizconfig settings manager after you install. In CCSM, there should be an option called **Desktop Wall** enabled. If it's not enabled, enable it or rotate cube.

Keep in mind that if you have desktops that are on more than one row, CRTL+ALT+UP and CRTL+ALT+DOWN will NOT work with rotate cube.

Also, the keyboard shortcuts are CRTL+ALT+Arrow Button and NOT CRTL+Arrow Button.
