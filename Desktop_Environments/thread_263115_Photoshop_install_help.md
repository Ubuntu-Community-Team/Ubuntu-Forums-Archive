### Post by petersjm on 2006-09-22
I'm not sure, but I think the last Photoshop version to install in Linux was v6. I could be wrong though. To try it out, open the terminal (Applications > Accessories > Terminal) and cd to CD-ROM drive directory with the exe file in it and type

```
wine photoshop.exe
```

Replace "photoshop.exe" with whatever the exe file is called.
