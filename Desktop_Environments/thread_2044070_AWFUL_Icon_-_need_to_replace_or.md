### Post by RogerDavis on 2012-08-18
OK, I figured it out.

1) Open Properties for that desktop icon
2) Search in Google for a Nautilus icon
3) Copy it and drop it into Gimp
4) Save it as a .png
5) Select the existing trashed icon on the properties window, and shout "Geminio" in your best Harry Potter voice.
6) Go to where you saved the icon, select it, and click "Open"
7) Admire the replaced icon, as it should have been in the first place.
