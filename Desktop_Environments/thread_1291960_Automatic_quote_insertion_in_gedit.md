Hi there,

i just saw a nice feature in a mac os text editor called wrangler. 
in wrangler you can select a text block an in / decrease quote levels.

so if you have 

-- 
Hi this is 
a test
--

you can mark the two lines, Choos edit -> incease quote level from the text menu. and voila, you have 
 
--
> Hi this is 
> a test
--

Of course you can increase many times.

I've searched for a similar feature for gedit, but haven't found one. Maybe you guys know a plugin or a hint? :)
