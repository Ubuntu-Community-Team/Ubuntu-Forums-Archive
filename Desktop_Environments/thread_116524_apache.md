### Post by Ampersand on 2006-01-12
what error did you get when you ran it?

You might need to run 
```
sudo gedit /etc/apt/sources.list
```
and remove the # from the beginning of the line for the universe repository (deb [http://archive.ubuntu.com/](http://archive.ubuntu.com/) universe or something similar).

Then save, and run
```
sudo apt-get update
sudo apt-get install apache2

```

Edit - looks like you've got the universe, what happens if you run 
```

```
