### Post by ankspo71 on 2010-10-06
Hi,
Try going to the theme's configuration settings to see if the opacity settings are still correct. Go to System Settings > Application Appearance > Style > Widget Style > QtCurve > Configure: Then look at the General category to see if Window Background Opacity and Dialog Background Opacity are set to about 85% or lower. I am mentioning this because the opacity settings will revert back to 100% if we flip back and fourth between qtcurve themes. I think importing a theme will revert the opacity settings back to 100% too... but I'm not sure. It doesn't hurt to double check though. :)
