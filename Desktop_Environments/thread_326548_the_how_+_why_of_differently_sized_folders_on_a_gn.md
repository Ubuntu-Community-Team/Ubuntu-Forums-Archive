### Post by Thenotsowyzewun on 2006-12-27
The feature is built into Ubuntu 6.10, not sure whether it's native to Gnome but yes I would presume so.
The resizing may be done manually (however some icons are automatically a bit bigger, e.g. PDF documents display as a thumbnail of the actual document, as do photographs and various other common file types, and so are slightly larger than program/drive icons to make them identifiable.

Based on frequency of access, that is a cool idea, I wouldn't be so surprised if someone had designed that (it's not built-in though), haven't searched Google but I don't think that'd be too hard to code.
