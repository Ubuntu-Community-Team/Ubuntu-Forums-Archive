### Post by ariel on 2006-06-19
I had this problem with certain wmv files (wmv9). The problem is weird, it happens only when I boot with XGL/Compiz and use XV for video output in mplayer. It does not happen when I load the "URL" in a standalone gmplayer session.

So I found a workaround that helped me: using the MediaPlayerConnectivity firefox extension. 

It works great for me, and also helped me get rid of other firefox annoyances. Now I can easily control and switch between media players for the different media types, and launch them in new windows when I want it. It also allows you to use GXine or VLC which are great too but don't have fancy mozilla plugins. I works great in Linux, it will autodetect most installed media players.
