### Post by Xian on 2005-06-16
That How-To is dealing with sensors:

*"You can easily create new desklets without any programming knowledge by using existing sensors in a new .display file. The first part of this tutorial teaches you how. Unlike creating displays, creating sensors requires some progamming skill. It is not difficult for programmers, and the second part of this tutorial talks about sensor programming."*

But as of gDesklets v.30 sensors are no longer supported:

*"If you intend writing new desklets please don't use any Sensors! They are deprecated since v0.30. You should also think about rewriting your old desklets using inline-scripting."*
