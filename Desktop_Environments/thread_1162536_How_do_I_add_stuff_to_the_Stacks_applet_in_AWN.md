### Post by balboost on 2009-05-22
well, there are many ways.

first you have to check whether your stack is on link mode or if it opens a folder. you can assign a folder to browse with stacks, it is more stable than the link option - to my experience.

basically, you can left-click on the folder, document, etc, and drag and drop it on the stack you want.

but my advice : create a folder which will be assigned to a stack. you can hide it by placing a . in front of it. i made one called 'links'. then, everytime i want a folder in my 'links' stack, i create a symbolic link to my '.links' folder. you can create a nautilus action to do it, it is very simple. 

the same with an application stack, in which you put launhers to apps you want.

it is very stable, it works fine, and very eye-candy. 

also, if you want, you can create launchers in your links folder and assign them a custom folder icon. it is very nice.

just ask me if you want more details :)
