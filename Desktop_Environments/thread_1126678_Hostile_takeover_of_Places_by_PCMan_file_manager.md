Hello,

I installed LXDE a couple of weeks ago to check it out, and now I have their file manager in my menu as well. It's kind of nice with the tabs, although I dislike that it doesn't use the icon theme I have set for the system and instead falls back to the default icons. Anyway. today for some reason it took over the Places menu, meaning all links in Places open in PCMan, except for Computer, which still opens in Nautilus.
My questions is, what do I have to do to get Nautilus back as the default application to open the Places?

Thank you.

---

### Post by Niniel on 2009-04-20
Turns out this was very easy to solve once I looked in the right place - all I had to do was go to any folder, click on Properties, go to the Open With tab, add "nautilus" as a custom command - since it wasn't on the list of apps anymore, only PCMan and Thunar were left, and then check Nautilus. 
I don't know why I didn't figure this out a lot sooner. :)
