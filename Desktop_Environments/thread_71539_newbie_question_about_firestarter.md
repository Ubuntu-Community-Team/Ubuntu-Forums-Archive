You only need to run the firestarter GUI after it is first installed. It will give you a basic configuration dialog to set up the firewall (including starting with the system).

After that firestarter will run everytime you boot without the need for further input on your end. You only need to run the GUI if you need to configure the firewall further (such as opening ports), or if you want to keep track of incoming hits to your ports.
