### Post by Frogs Hair on 2021-07-13
If you have installed the Mate DE as a second desktop environment It can't be set as default , but you can choose to login to it for daily use. You would be better off backing up and installing [U-Mate]("https://ubuntu-mate.org/download/amd64/") from the ISO. Having more than one desktop makes for a bloated system and can become problematic especially if you intend to upgrade the system. The upgrade will be applied to the primary/originally installed DE.
