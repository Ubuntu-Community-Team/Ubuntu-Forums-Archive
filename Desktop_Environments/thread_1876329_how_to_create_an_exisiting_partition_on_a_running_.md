### Post by Sigma1 on 2011-11-06
Hello,
To partition a disk, you can use gparted, for example. Gparted is not part of the default ubuntu installation, but it is generally on the LiveCD. (There is dedicated, gparted only, LiveCD, too.)
You'll need the LiveCD to partition the disk holding your ubuntu system, since, in order to do this sort of thing, the disk cannot be mounted. In other words, as far as I know, it's impossible to partition the disk you're running the system from. Good luck!
