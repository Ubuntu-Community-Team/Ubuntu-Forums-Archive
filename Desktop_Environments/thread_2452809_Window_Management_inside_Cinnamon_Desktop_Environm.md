### Post by DuckHook on 2020-10-28
I'm not sure that I understand your question. The Window Manager is foundational to the Desktop Environment. Therefore, properly speaking, it runs "outside" of the DE; not "inside".

If you want to change Cinnamon's WM to another, this is possible in theory, though I've never tried it and don't know to what extent you would have to customize and fiddle to get them to play nicely with each other.

If you want to run a Window Manager within a Virtual Machine within a Cinnamon&#8209;based OS, then this is conceptually simple to do. See the link in my sig: The Best 'buntu Flavour

Although that sticky deals with the various DEs available to 'buntu users, it does clarify the relationship that WMs have to DEs, and lays out a pathway to running different DEs inside a VM.

Ultimately, though, I'm stumped by the multiple ways of which your question is to be interpreted.
