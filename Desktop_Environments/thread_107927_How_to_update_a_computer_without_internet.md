### Post by az on 2005-12-24
You can install with the installer, or upgrade from one release ot he other with the new install cd.  Just boot into your old linux setup, pop in the new (breezy, for example) cd and follow the instructions for upgrading.

You an install packages on one machine and transfer them to another machine using apt-move:

[https://wiki.ubuntu.com/AptMoveHowto](https://wiki.ubuntu.com/AptMoveHowto)
