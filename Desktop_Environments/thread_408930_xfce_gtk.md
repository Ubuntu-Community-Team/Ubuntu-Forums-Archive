### Post by Heliix on 2007-04-14
hello,
i think it's perfectly compatible, at least, i didn't ran into any problems in this issue so far. though, some gnome apps may require gnome libraries (which are in ubuntu but have to be added to xubuntu).

to avoid possible problems, i recommend you install classic gnome-based ubuntu and then add xfce desktop environment (via synaptic package manager or using aptitude).  each time you log in, you will be able to choose your session environment (gnome or xfce) which is ideal if you want to try it. 
you may also install xubuntu and then add your favorite gnome apps if you like. the result is +- the same (i have tried both ways)

you'll like xfce :D it's really faster than gnome (tested on 256MB memory laptop)
