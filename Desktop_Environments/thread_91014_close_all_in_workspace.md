Hi, a couple things you could look at:

Right click on panel hide button, and add the "Window Selector" tool.

or

Run "System Tools > System Monitor" for a "Taskmanager-like" process list.
I installed Automatix and had it bind the keystroke CTRL-ALT-DEL to run system monitor.
