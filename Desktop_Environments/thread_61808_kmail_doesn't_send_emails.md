### Post by shakin on 2005-09-02
I'm using Kontact so it might be slightly different for you. To get the system tray icon to appear (I assume this is what you meant when you said "docklet") you go into the settings, under Appearance there is a System Tray tab. Check Enable system tray icon and either to show it always or only when there is an unread message.
