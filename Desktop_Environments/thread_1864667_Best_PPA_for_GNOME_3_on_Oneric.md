### Post by satanselbow on 2011-10-19
webupd8 keep up to date with the extensions - add the ppa, install the extensions you want/need then disable the ppa. Job done!

If you want a stable box to work on avoid a wholesale update of experimental updates ;)

Don't forget that the extensions have to be enabled in the tweak-tool / advanced settings now... should prevent you from coming back saying you've installed but they don't work ;)
