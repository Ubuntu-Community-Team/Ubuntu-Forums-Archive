### Post by Jucato on 2006-08-24
Press Alt+F2 to bring up the Run command dialog box (remember that keyboard shortcut, as it will be very useful). Type in "systemsettings" and press Enter. Then in System Settings, go to the Panel options and reconfigure kicker (the KDE panel).

Hope that helps.
