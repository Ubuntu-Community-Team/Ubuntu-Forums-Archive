Oops, can't read - ignore whats below its for the exact thing you say you don't want :s...one actually useful thought is to look at using the scale plugin, which is much like Mac's epose feature. It will show you all your open winddows in a little scaled down view, and then you just click the one you want. It uses the whole screen to do that though rather than putting them in a nice little bar.
<IGNORE>
There is a compiz plugin to give you exactly what you want, go into ccsm compiz config settings manager - install from synaptic if you don't have it already)

then make sure that the shift switcher plugin is enabled and set it up to work with whatever key binding you want.

Finally, make sure that on the appearance tab of that plugin's configuration you select switcher mode to be "flip"

That will give you a vista like windows switcher.
