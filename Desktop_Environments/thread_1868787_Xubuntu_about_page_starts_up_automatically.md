Go to 'Settings' 'Session and Start Up' check to see if the Environment Page is in 'Application Auto Start'
if it is un-check it.

Another way is to get your desktop how you like and then  go to 'Settings' 'Session and Start Up' again and check the ' Automatically save session  on logout'
