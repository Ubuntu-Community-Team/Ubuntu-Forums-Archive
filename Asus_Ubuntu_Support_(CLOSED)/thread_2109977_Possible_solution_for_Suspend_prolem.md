Hi Folks,

My Asus eeePC 1000HG 'developed' a problem where it would not resume from suspend or would not actually turn on after s shut down.

I applied the following advice..
[http://forums.linuxmint.com/viewtopic.php?f=90&t=121953](http://forums.linuxmint.com/viewtopic.php?f=90&t=121953)

and this worked for a while until an update broke everything again. So after much net trawling I re-applied the above script BUT have also changed the following system settings....

Go to System Settings > Brightness and Lock
- Turn off the 'Lock'
- Turn off 'Require my password when waking from Suspend'

So far this seems to work. I can leave my net book Suspended for varying lengths of time and it does resume. Until the next update anyway.

Hope this Helps,
nictu.
