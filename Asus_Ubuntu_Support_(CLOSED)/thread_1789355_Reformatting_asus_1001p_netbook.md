### Post by toor58 on 2011-06-24
I don't know why you would want to reformat. If you want to resize the partitions then reset the windows boot ( [http://ubuntuforums.org/showthread.php?t=508927]("http://ubuntuforums.org/showthread.php?t=508927") ) and then remove ubuntu. Use the disk management tool to resize your partitions leaving the space empty where you want ubuntu to go. Doing this preserves your boot booster partition. If you wipe everything then you must find out how to replace the boot booster partition manually.
