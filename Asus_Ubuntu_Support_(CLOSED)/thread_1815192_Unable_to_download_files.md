### Post by lovinglinux on 2011-07-31
You are trying to install a Windows application, which won't run on Linux. There is an application, called Wine, that allows to install some Windows apps on Linux. However, you should avoid using this method if a native application is available. 

Most of the time, you don't need to download applications from the web. In fact, is not recommended because of security reasons. All you need is to launch Ubuntu Software Center, search for the applications you want and install. They are installed from the Ubuntu repository, which are secure, since all applications there are tested by Ubuntu developers.

All applications you mentioned are available in the Software Center.

### Post by Rasa1111 on 2011-07-31
You don't need to do any of that to download anything you mentioned. 

Want Chrome and GIMP?

~Open Ubuntu software center
Type "chromium" into the search bar at the top.

When it shows in the list, click "Install", and let it install. 

Now type GIMP into the same search bar
When it shows in the list, again.. click "Install"

Want Flash?

Type 'ubuntu restricted extras' into the search bar. 
When it shows in the list.. Once again, just click "Install".

Now Chrome(ium), Gimp, and Flash (and all other codecs you'll need) are downloaded and installed. 

Piece of cake! ;) 
Then just find GIMP and Chrome(ium) in your app 
Menus. 
Remember, this is not windows..
things are done a bit differently. 
(for the better).

Once you get used to it, I think you'll dig it. 

Good luck.
