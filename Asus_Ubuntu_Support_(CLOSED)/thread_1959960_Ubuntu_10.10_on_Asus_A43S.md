It will be a security risk to continue to use it. If you upgrade to 11.04 now you will need to upgrade again in October, when that version reaches end-of-life. However, if you install 12.04, your system won't need a distribution upgrade until 2017, since an LTS release is now supported for 5 years.

Assuming you have a new system, this is the best time to make the switch.
