### Post by SeijiSensei on 2011-07-20
> **toor58 said:**
> is your laptop an x64 bit machine? Maybe you need to use the 32 bit install.

In general, attempting to install the 64-bit version on a 32-bit machine will throw an error and refuse to install.

"amd" refers to Intel's competitor, Advanced Micro Devices.  They came out with 64-bit processors before Intel, so the kernels are often designated as "amd64."  Personally I'd rather see them designated as "x86-64" to indicate that they work with Intel processors as well.
