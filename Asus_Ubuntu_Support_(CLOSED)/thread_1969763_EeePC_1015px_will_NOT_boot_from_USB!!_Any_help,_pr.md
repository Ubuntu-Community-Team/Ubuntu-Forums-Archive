### Post by Chrisj303 on 2012-05-10
Fixed myself.

Found that I needed to allocate at least 512mb of pen drive space as "persistant" memory, i order to boot Ubuntu. Not very well documented TBH, I only noticed it when I used Live Linux USB Creator.
