### Post by oldfred on 2012-12-23
Yes.

It is a setting in UEFI and Microsoft requires the vendors to turn that setting on in UEFI. 
But users are also supposed to be able to turn it off and then it is like any non-secure boot system. You could then install in UEFI or BIOS mode.
