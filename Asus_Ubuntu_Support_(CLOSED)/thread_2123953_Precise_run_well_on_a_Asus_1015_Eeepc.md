### Post by SeijiSensei on 2013-03-09
I run Kubuntu 12.04 on an Eeepc 1201n, the one with the dual-core Atom processor and an NVIDIA card.  It works fine except for, you guessed it, suspend/hibernate.  That seems to work better if you choose one of those explicitly during shutdown rather than just closing the lid, but I often find I have to reboot from scratch.  That adds only a minute or so, though.

I suggest using the Startup Disk Creator tool to put a copy of the 12.04 installer on a USB thumb drive and boot from that, then choose "Try Ubuntu" when it appears.  It will be slower running off the stick than the hard drive, but you'll be able to give it a test drive.
