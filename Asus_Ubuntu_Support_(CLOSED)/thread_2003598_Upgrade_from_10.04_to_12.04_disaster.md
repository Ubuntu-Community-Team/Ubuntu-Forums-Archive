### Post by fbicknel on 2012-06-14
I've had some bad experiences with upgrades, as well.  Especially moving ahead more than one revision would make me think twice about upgrading.  Previous post is good advice: download 12.04 and make a Live USB drive, boot it... install it... have fun.

hi lesliem,

ubuntu has implemented its unity interface as the standard interface for its current releases since 11.04. so there are no more netbook editions since the interfaces are just the same.

about your lagging update, we don't know technically if it is with your unit or with the internet connection. since there are no post as to the last message of the update process.

however, since you are using an acer aspire one and if you want to reinstall a new os, you can try picking one distro to install on your usb drive from a list here: [http://www.pendrivelinux.com/](http://www.pendrivelinux.com/)

pendrivelinux.com offers a variety of linux distros from simple ones that can be less than 1000kb up to full featured distros with known big application packages starting at 1gb. if one works or suits your taste, an install feature is also supported just follow the instructions.

you might find lubuntu and xubuntu to be big but there are other lighter distros so i further suggest these:
