### Post by john stiles on 2012-10-19
installing 12.04 is straightforward.
There are proprietorial drivers for the non intel graphics chip set. These are awkward to install with the need to negotiate a blank screen and finally only allowing unity 2D.
A nice machine marred by Intel/Asus using a closed graphics chip without full linux support.
Give this model a miss and look instead at the Asus F201E, if you can find it!
