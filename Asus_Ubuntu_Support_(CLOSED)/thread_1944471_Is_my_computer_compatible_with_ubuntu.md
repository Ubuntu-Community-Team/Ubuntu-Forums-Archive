### Post by Bucky Ball on 2012-03-21
Check the BIOS and make sure you are booting from the hard drive you have Ubuntu installed on first (and not a USB or anything else). Make sure you have no USB plugged into the machine at boot.

10.04 LTS should work fine on that system and perhaps you should have researched the issue you were having as there's every chance you would get the same with 12.04 LTS. If you boot from the 10.04 LTS LiveCD and 'Try Ubuntu' does everything work fine? If so, the problem is no doubt resolvable. 

Incidentally, 10.04 LTS is supported for another thirteen months and is a one click upgrade to 12.04 LTS. I would wait a couple of months after release to upgrade, though. ;)
