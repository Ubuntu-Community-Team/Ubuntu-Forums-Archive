### Post by Linuxisfast on 2012-10-16
Picked this laptop up recently and installed Ubuntu LTS 12.04.1 and it works brilliant right out of the box. No issues, sound, wireless, webcam, bluetooth all works and works well. Laptop runs the same temp as it did in Windows but far more snappier. Added x-swat ppa for latest Intel driver. 

If you are looking for metal body high performance i5 laptop, this is one great and affordable option that works well with Ubuntu. HDMI with intel VAAPI and VLC is quite good and tear free, looks like 2nd gen Sandy Bridge has weeded out all the initial issues with graphics and now works well. Unless you are a gamer, this is the platform for you. Comes with 4GB RAM and 750GB HDD, the RAM can be upgraded to 8.

### Post by LemonLime on 2012-10-29
Saw this mentioned in the laptop compatibility thread, figured I'd add a few comments here. K53e definitely works with Ubuntu out of the box, and I think the audio experience is better than Win7, which is included with the machine. I use it on Ubuntu for live DJ'ing at a local college radio station and it's been faster and more comfortable than running Windows Media Player, Winamp or any other application. 

I am still a beginner Linux user, but the major difference between running Ubuntu and Windows on the Asus K53e would be **power settings**. The included "Power2go" Windows software seemed to work quite well and I could squeeze about five hours out of the machine if I was just doing some low-intensity browsing and word processing. However, I can only get about 3 and a half hours in Ubuntu doing the same, even if I turn down the brightness. Perhaps there are utilities built in to Ubuntu or downloadable from the community for power management, but so far I've only used Redshift.

Nevertheless, K53E runs relatively ice cold even while it's at maximum performance, even more so in Ubuntu.

EDIT: Has anyone tried using the USB 3.0 with Ubuntu yet? I'd love to hear how it performs.
