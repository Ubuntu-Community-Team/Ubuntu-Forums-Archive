### Post by haqking on 2011-11-06
> **UncleMonty said:**
> Thank you. Are they basically an external hdd for a netbook or can an operating system be put on them?

They are solid state memory devices.  They can be used in cameras, mp3 players, phones etc for storing photos, music, data etc.

Then you can take it out and stick it in your machine to make transfer of the data easy.

Or depending on the size of the card, in things like Windows you can use readyboost to use the card as extra memory (sort of)

if large enough you can install a OS on them yes.
