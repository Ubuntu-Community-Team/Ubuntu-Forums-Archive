### Post by magnet18 on 2013-02-06
been working for days on this, just can't get the brightness to adjust on my G75!

I'm running ubuntu 12.04, 64bit, nvidia660M graphics with nvidia drivers, and everything works but the brightness

I tried the brightness applet, that just gave me a slider that slides, and changes nothing

the only other thing i could find was this -http://www.linlap.com/asus_g75vw?

which said to-
1) Add the line Option &#8220;registryDwords&#8221; &#8220;EnableBrightnessControl=1&#8221; in the appropriate Device section of the /etc/X11/xorg.conf file.
2) Do sudo pico /etc/default/grub and add acpi_osi=Linux and acpi_backlight=vendor to the GRUB_CMDLINE_LINUX_DEFAULT variable. Press Ctrl+O to save.
3) Run sudo update-grub to rebuild grub.cfg.
4) Reboot.

i went to try this and xorg.conf was blank!
apparently this is ok in ubuntu, and is blank by default, but if configuration is placed there it will be used

so i ran nvidia-xconfig as root, and that generated the xorg.conf

after that i followed the above instructions, and the result-
the fn keys now adjust the brightness slider... which is progress... but the slider still does nothing!
