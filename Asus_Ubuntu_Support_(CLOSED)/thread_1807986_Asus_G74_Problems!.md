I bought the same PC mid-August and it's been killing me not having backlit keys. I assumed that the innards of the G74 would operate the same as the G73, so I installed the fix there and it works great.

Most of the Fn keys work too, Sleep just seems to lock the screen, then crash Ubuntu. Not sure about Mutli-displays ( F8 ) cause I haven't tested that. Also not sure about Trackpad Toggle (F9) because I disabled mine in the BIOS. My trackpad would cause the infamous Asus keyboard lag issue where it would miss a lot of keys if you typed to fast, so I had to disable it. Other than that, they all work great. Fix takes about 10 minutes to apply.
