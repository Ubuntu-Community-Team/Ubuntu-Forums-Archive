### Post by JRV on 2011-11-13
You can use unetbootin to create a bootable USB stick from the .iso

If you have an Ubuntu installation it's in the repositories. Unetbootin is also available for Windows, Google it.

On newer versions of Ubuntu there is a catch that stops the installer from working on the small SSD.  I don't know if this applies to 10.10.

If it does follow the instructions in this thread to install from the netboot CD:

[http://ubuntuforums.org/showthread.php?t=1876241](http://ubuntuforums.org/showthread.php?t=1876241)
