### Post by mikewhatever on 2012-01-03
Here you go: [http://releases.ubuntu.com/lucid/ubuntu-10.04-netbook-i386.iso](http://releases.ubuntu.com/lucid/ubuntu-10.04-netbook-i386.iso)

The Netbook edition was discontinued starting with Ubuntu 11.04, as it's deemed that the new Unity shell is suitable for large and small screens alike.

Unfortunately, the Netbook ISO has not been updated since the release date, which will result in a large amount of initial updates. If that's a problem, get the updated [Desktop ISO]("http://releases.ubuntu.com/lucid/ubuntu-10.04.3-desktop-i386.iso"), and when it is installed, install the 'ubuntu-netbook-remix' package.

### Post by IWantFroyo on 2012-01-03
If you want to stick with Ubuntu, Lubuntu or Xubuntu will work fine.

If you're willing to venture out a bit and try new things, CrunchBang (crunchbanglinux.org) runs really quickly while still looking really good. That's my #1 recommendation, although there may be more of a learning curve because you edit the menus manually.
It's based on Debian 6, however, so it shouldn't be too different from Ubuntu under the skin.

Generally, most distros that use XFCE, LXDE, Openbox, Fluxbox, or Enlightenment are "light."
