### Post by ddumanis on 2011-12-23
I have an ASUS UL30 laptop. At random intervals, both the mouse and touchpad will slow down at the same time, rendering them unusuable. Then they go back to normal after 5-15 seconds. Doesn't seem to be related to any event although it does seem to happen more often when the system is busy.

Running 11.10 in regular Unity mode. Thanks for any help.

---

### Post by ddumanis on 2012-01-01
This problem seems to have been due to overheating.

I installed Jupiter, set it to "High Performance," and things are much better.

Apparently 11.10 has a problem with overheating laptops in general. Would have been nice if that had been posted as a clear release note. (It seems to be mentioned in the kernel section, but in technical language that is easy for non-developers to overlook.)
