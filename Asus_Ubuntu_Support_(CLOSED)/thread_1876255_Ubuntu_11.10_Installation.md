### Post by psykotic25b on 2011-11-06
Since you're using a netbook, I'm gonna go ahead and assume you're using a thumbdrive/hard drive to boot from.  My advice to you would be to reformat the device, and try to load your 11.10 image back onto it and repeat the installation process.  If you have an external cd-rom available I would advise you to use that and boot from a cd/dvd.  That's how I loaded my 11.10 on top of my W7 Asus u52f and it worked like a charm.  Try these suggestions and let me know where you get.  I'm always eager to help fellow ASUS users!
