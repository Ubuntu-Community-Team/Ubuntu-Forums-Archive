### Post by BC59 on 2011-12-26
Try reinstalling the ATI driver from Synaptic (you have to install it through Ubuntu Software Center, if you don't have it already).
Search for FGLRX. There are 2 options for FGLRX, with updates or not. Choose complete uninstall and then reinstall it. After all this, reboot to take effect.

---

### Post by mattasus on 2011-12-26
I went into synaptic package manager after I did the first step and I noticed that FGLRX wasn't installed. So installing it and rebooting fixed the problem.
Thanks:D
