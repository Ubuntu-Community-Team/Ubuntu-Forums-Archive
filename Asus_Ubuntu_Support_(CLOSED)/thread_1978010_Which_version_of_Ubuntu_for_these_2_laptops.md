These specifications are OK (Just OK, not good) to run Ubuntu 12.04 with Unity 2D.

Neither of these processors are 64bit capable, so you'll have to stick with 32 bit. Fortunately, both seem PAE capable, so you shouldn't have any trouble with Ubuntu 12.04.

I don't think you can get Unity's (especially Unity 3D/regular) performance as you are expecting. It seems presumptious to me to compare performance of a 2102 OS with an OS released in 2002,  but then that's just me.

if you are unhappy with Unity performance, you can consider using Xubuntu or Lubuntu (non-Unity interface).

For reference, I'm using Ubuntu 12.04 with Gnome Shell on an Intel Atom N450 (64 bit) with 2GB RAM and Intel integreted graphics. It's no speed demon, but it handles day-to-day usage smoothly.
