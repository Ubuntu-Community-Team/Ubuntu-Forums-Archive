### Post by littlecodemonkey on 2011-04-05
I think I agree with structure. Some things like moving rooms is unforeseeable; however, we didn't get a lot of debugging in. We also did not submit any bugs to fix. 

I think if we all had the system installed beforehand that might have sped things up, also the topic tended to stray a bit.

I think if we all sat facing each other in a round table and focused on bug submission we would have gotten more done. If there was a 20min social section at the beginning and end with definite time limits that would improve work throughput. 

If everyone knew where and how to submit bugs, that would help also. I hadn't submitted a bug before, so I didn't know where to go to submit one.
