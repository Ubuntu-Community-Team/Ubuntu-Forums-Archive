### Post by vsteel on 2011-05-29
If you are a Kubuntu user like myself there are a couple of things I would recommend you do after you do an install.  

1.  Install the VLC backend.  (Seems to sound better and be a little more stable than the gstreamer backend)
2.  install the upgrade to Amarok, (need to add the PPA backport)  This changes your Amarok from 2.4.0 to 2.4.1 which fixes several bugs and also increases stability.
3.  Install the VLC mozilla pluggin.
