### Post by tghe-retford on 2010-10-02
> **Pablo Pablovski said:**
> I haven't reported a bug - tbh, I wouldn't know how to...
You'll need a launchpad account, but all you have to do is type in at a terminal:

```
ubuntu-bug <package>
```

Replacing <package> with the name of the package which is causing the bug, in our case, it was libdvbpsi6, and then following the instructions on how to file your bug in the web browser which is launched. I have created a bug report for this bug which you can also add to or say that it affects you too. It may be worth you noting in that bug report that you get your DVB-T service from another transmitter yet you have the same problem.

Bug report: [https://bugs.launchpad.net/ubuntu/+source/libdvbpsi/+bug/653616](https://bugs.launchpad.net/ubuntu/+source/libdvbpsi/+bug/653616)
