### Post by inportb on 2010-10-05
Well, Kubuntu chooses automatically which interface to show, with the option to manually override the setting. Both interfaces are part of KDE proper, so there is only one desktop environment that does both netbook and the usual.

The difference between UNE and Ubuntu is bigger in that they can be treated as different (similar) environments, with UNE running Unity on a subset of the Gnome stack. UNE is a bit more finger-friendly, which is why I use it with my netbook, but I'm leaning towards just installing Kubuntu and using E17 (in the repos) when I need the extra battery life or other resources. As of now, KDE is a lot more finger-friendly and functional than UNE (albeit a bit heavier, the last time I checked).
