### Post by ranch hand on 2010-09-28
The option to not install  grub is to not install grub to the MBR or any partitions.

I would run;
```

sudo grub-mkconfig

```
and post those results here to make sure that the MS entry is right.  It probably is.

To boot to Ubuntu you will need to install grub on the MBR.  It really does nto need to be anyplace else.  To do that you need to run;
```

sudo grub-install /dev/sda

```
If you Ubuntu is on a different drive than sda you could, and probably should run that command again and install it on that MBR too.
