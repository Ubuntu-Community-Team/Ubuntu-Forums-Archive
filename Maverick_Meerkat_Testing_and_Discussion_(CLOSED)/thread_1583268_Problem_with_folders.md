### Post by mc4man on 2010-09-28
As was just noticed by [dkjMusic]("http://ubuntuforums.org/member.php?u=25218"), the culprit seem to be the 'Remember this application for "folder"' checkbox.
uncheck it and the app will still show up in the 'open with' menu but will not be set as the default folder handler.
