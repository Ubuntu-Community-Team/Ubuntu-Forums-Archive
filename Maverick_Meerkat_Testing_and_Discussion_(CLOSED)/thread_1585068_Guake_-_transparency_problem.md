### Post by sonicb00m on 2010-09-30
I really love Guake but there's always a new problem with it between Ubuntu versions.

This time it's the transparency again. If I load quake the first window is properly transparent. If I create a new tab the 2nd windows is solid black while the first window remains transparent. Each subsequent new tab is also solid black.

Any idea how to fix this? I tried going to the site but I can't even figure out how to sign up and create a bug report. Plus the fact they seem to take forever to release new versions.

Since it's python source maybe there's a hot fix I can implement.

Normally I am not so fussy about minor things like this but I work with servers and perl/php coding all day and being able to drop the terminal and still see through the window is amazingly helpful.

--Actually for now I just found a good work around. Disable the transparency in Guake and use Alt-Wheel to make the entire application see-through. Better than nothing!
