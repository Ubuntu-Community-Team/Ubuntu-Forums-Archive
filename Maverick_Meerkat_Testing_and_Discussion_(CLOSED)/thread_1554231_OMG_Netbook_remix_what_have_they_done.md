### Post by cariboo on 2010-08-16
There were some database problems, so the pages weren't updated.

Did you try clicking on the Ubuntu icon in the top left corner, or click the applications icon in the panel. The terminal is located under Applications->Accessories, like normal. To add an application to the panel, just open the application you want, then right click, and select keep in panel.

Unity is still in heavy development, and there is no going back to the Lucid interface, with out re-installing Lucid UNE.

It is probably a good idea to check here before upgrading, as there have been a lot of changes to the UNE.

### Post by 23meg on 2010-08-16
The goal of this forum is to help people test and improve Ubuntu, not venting frustration or hatred. There are [other]("http://ubuntuforums.org/forumdisplay.php?f=11") [forums]("http://ubuntuforums.org/forumdisplay.php?f=103") where that's acceptable within limits.

If you'd like to turn your frustration and hatred into something productive, [reporting]("https://help.ubuntu.com/community/ReportingBugs") the issues you've found will be a good start. If you need help with that, please start an appropriately titled thread, preferably without the derogatory bits.

If you just want the old interface and don't care about making Unity better, look for a project that aims to maintain it in the future, or consider starting one if it doesn't exist.
