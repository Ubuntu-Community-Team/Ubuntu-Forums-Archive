### Post by MacUntu on 2010-09-08
You can always get older packages from Launchpad. In this case, the package you want is ubuntu-wallpapers, so go here:

[https://launchpad.net/ubuntu/+source/ubuntu-wallpapers](https://launchpad.net/ubuntu/+source/ubuntu-wallpapers)

Now just click the version number under the release name you want (eg. Karmic -> 0.30) and then download the source file (*.tar.gz):

[https://launchpad.net/ubuntu/+archive/primary/+files/ubuntu-wallpapers_0.30.tar.gz](https://launchpad.net/ubuntu/+archive/primary/+files/ubuntu-wallpapers_0.30.tar.gz)

Tadaa - Butterfly.jpg! :)

**Edit:**
If you wanted info on the author, its EXIF data states "by Kenneth Wimer".

**Edit2:**
Here it is in his Flickr photostream: [http://www.flickr.com/photos/kwwii/3865853724/](http://www.flickr.com/photos/kwwii/3865853724/)
