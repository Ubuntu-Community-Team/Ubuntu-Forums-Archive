This is confusing alright.

Launchpad tracks bugs across upstreams and distributions; a bug can have multiple so-called tasks. Each line you see in the "Affects / Status / Importance / Assigned to / Milestone" table is a task. In this bug, the "language-pack-gnome-en-base" task has a priority of "High"; that why when you look at the list of bugs for that package, you see that priority.
