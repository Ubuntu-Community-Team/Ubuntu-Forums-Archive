### Post by chrismine on 2010-07-19
Every time I boot Maverick Firefox also starts automatically in full screen. I have checked the Preferences-> Start up Applications and also deleted Firefox from AWN Task manager thinking it may start automatically with AWN. I also see no options in Firefox itself.

### Post by x-shaney-x on 2010-07-19
In the Startup Applications window, did you have a look at the second "Options" tab?

There is an option to automatically remember running applications when logging out.
Check that is disabled.

---

### Post by chrismine on 2010-07-19
Thanks it is disabled.

Clicking on Remember currently running application with Firefox not open seems to cure the problem, reboot two times and Firefox not opening automatically.

Thanks for that.
