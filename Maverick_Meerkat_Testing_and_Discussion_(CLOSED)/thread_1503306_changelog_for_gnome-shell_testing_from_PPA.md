If you want to see changelog entries pertaining to the PPA, go to [https://launchpad.net/~ricotz/+archive/testing/+packages](https://launchpad.net/~ricotz/+archive/testing/+packages) and click the expanders next to each package.

If you want to see the upstream git changes, go to [http://git.gnome.org/browse/gnome-shell/](http://git.gnome.org/browse/gnome-shell/).

I don't know about this particular PPA, but they might end up being the same thing, if the PPA ships automated builds.
