Hello Ubuntu installer

go to google type in 10.10 download the latest batch which as of June 16th is slightly oversized to fit on a CD.  Instead burn it to a DVD and run the install.  

I used 10.04LTS and found way too many bugs, first off not being able to install without doing some testing and then finding that my ATI R5670 wasn't supported properly by ATI until you can install Ubuntu to get the Updated driver 10.5 from ATI website.  

This version 10.10 installed without any problems.  I'm writing from within 10.10 from Meerkat.

Then get the 10.6 Yes as of 6/16 new driver update from ATI and run the install for the Graphics card.  

here's the link: [http://cdimage.ubuntu.com/daily-live/current/](http://cdimage.ubuntu.com/daily-live/current/)
