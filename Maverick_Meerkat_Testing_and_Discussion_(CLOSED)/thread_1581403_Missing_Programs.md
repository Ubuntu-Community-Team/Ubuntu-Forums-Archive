### Post by scradock on 2010-09-24
You can certainly add programs individually, and it's no more time-consuming than the daily updates. Make a list of what you want to add, mark them for installation in synaptic and click Apply. If the install is running sweetly I'd suggest you stick with it and beef it up with the programs you want.

You can even remove programs you don't want at the same time......

### Post by ranch hand on 2010-09-24
One thing to do, if you are interested in the default programs, is check to see if you have the ubuntu-desktop installed.  You may have removed it sometime with an update that wasn't quite ripe.

Synaptic would be one place to check or just;
```

apt-get install ubuntu-desktop

```
iif it is installed Apt will let you know.
