> Can someone tell me what package to install to get olive-gtk?  I found a development email that said olive-gtk was split out from bzr-gtk but I can't seem to figure out which package to install to get it.

I think it's still bzr-gtk. Not sure where the vanilla "olive" went off to, but all of Olive's basic commands are accessible via bzr itself.

For example, run bzr gcommit to open a GTK commit dialog, or bzr viz to show branch history. There's a lot more: see `bzr help gtk` for all of them.
