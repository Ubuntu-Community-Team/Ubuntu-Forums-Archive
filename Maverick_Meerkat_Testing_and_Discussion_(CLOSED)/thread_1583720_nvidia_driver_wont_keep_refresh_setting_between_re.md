### Post by VinDSL on 2010-09-28
> **sdowney717 said:**
> I need to set it to 85 or auto every time. I tried saving the configuration but makes no difference.

Where are these setting stored?

Could it be generating an invalid xorg.conf?*nvidia-settings* doesn't save anything, unless you are running it as an admin.

Put another way, if you aren't running *nvidia-settings* as an admin, it won't permanently save anything.

You can start *nvidia-settings* in admin mode via terminal...

```
gksudo nvidia-settings
```

Or, better yet, edit your Menu Properties and change the *NVIDIA X Server Settings* launcher command to...

```
gksudo /usr/bin/nvidia-settings
```
