### Post by mohegan on 2010-09-26
It is better if you install this package : firmware-b43-lpphy-installer (with synaptic or ubuntu software center). But, before, you have to connect to the Internet via an ethenet cable. After the installation, the wireless works immediately.

Don't use the Additional Drivers tool (jockey). In fact, the Broadcom STA drive don't work perfectly (disconnection, low flow, ...) with maverick and the other driver don't want install properly.


firmware-b43-lpphy-installer is an installer package for firmware for the b43 driver BCM4312 (with Low-Power).
