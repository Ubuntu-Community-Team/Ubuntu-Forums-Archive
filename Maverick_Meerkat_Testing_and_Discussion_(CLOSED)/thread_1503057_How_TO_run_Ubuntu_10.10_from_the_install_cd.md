### Post by ranch hand on 2010-06-06
You need to have your bios set to boot from the CDrom first.  Put your CD in and reboot.  You will get a menu that give you 2 choices;
1>Try Ubuntu
2>Install Ubuntu

If you have trouble booting the CD reboot and when the first purple screen comes up hit any key and this will take you to a more complete menu.
