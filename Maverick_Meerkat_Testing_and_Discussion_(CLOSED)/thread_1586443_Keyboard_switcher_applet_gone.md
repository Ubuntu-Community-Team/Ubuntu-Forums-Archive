### Post by kumaryu on 2010-10-02
I have two keyboard "switcher" icons in the system tray.

1. Ibus; for switching (between) input methods. (IBus 1.3.7 by default). This system tray item may be turned on/off from Ibus Preferences:
 System>>Preferences>>Keyboard Input Methods

2. Keyboard Preferences; for switching the actual hardware keyboard. This appears automatically when you have more than one keyboard layout selected in Keyboard Preferences:
 System>>Preferences>>Keyboard

I usually only use item (1) above. This allows me to turn Ibus input method (Anthy) on and off. My H/W keyboard layout always remains English (US International) since I don't normally swap keyboard layouts.

As of the latest package updates (2 Oct '10), both icons are still in the system tray.
