### Post by ranch hand on 2010-07-28
If there is a "theme" folder for your theme put it in /usr/share/theme.  If there is a separate "icon" folder put it in /user/share/icon.

Usually works for me under Gnome but I do not fool with KDE (just don't like it).  I should think that the source directory for the themes would be the same, could be wrong.

Another place you could put the theme is in /home/<your user>/.theme.  There can be a ~/.icons too so if either are not there you could just create the buggers.
