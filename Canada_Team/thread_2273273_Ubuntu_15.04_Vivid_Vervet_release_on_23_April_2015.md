Hellooo Everybodeee!  It's been a long time since my last message to the mailing lists and forums.  In fact, the last time was to announce an Ubuntu Release Party, and now this message is to announce at least two Ubuntu Release Parties for the new release coming out on 23 April 2015, Ubuntu 15.04 codenamed Vivid Vervet.

Registration at the LoCo Event site for both events is appreciated, but feel free to just drop in.

If anyone is holding an Ubuntu Release Party in other locations, stick it on the LoCo Event page and let us all know.  Holding an URP is easy: Pick a spot, make an announcement, and show up. Doesn't have to be in a restaurant or pub; could be at the picnic table in your local park, or in a corner of the library, or in a classroom after school.

 For those of us near Kitchener, Charles McColm (@chaslinux on Twitter) is reviving the Kitchener Ubuntu Hours. There hasn't been one in quite a while, and Charles is looking for a location to host: [http://charlesmccolm.com/content/ubuntuxubuntu-hourlinux-user-group-meetings](http://charlesmccolm.com/content/ubuntuxubuntu-hourlinux-user-group-meetings) 

 Can you believe it's already been two years since we renewed the Ubuntu-CA team's Local Community standing? Last time we created a ReApproval page to demonstrate to the LoCo Council that Ubuntu Canada was a vibrant user community: [https://wiki.ubuntu.com/CanadianTeam/ReApprovalApplication2012](https://wiki.ubuntu.com/CanadianTeam/ReApprovalApplication2012)

This year, the process is called "ReVerification". I suggest that people who would like to participate in creating a new Ubuntu Canadian Team ReVerification Page start discussions on the Ubuntu Canada mailing list, but there's also an IRC channel for real-time discussions, and also the Ubuntu Forums:


**Mailing List**: [https://lists.ubuntu.com/mailman/listinfo/ubuntu-ca](https://lists.ubuntu.com/mailman/listinfo/ubuntu-ca) 
**IRC Channel**: [http://webchat.freenode.net/?channels=ubuntu-ca](http://webchat.freenode.net/?channels=ubuntu-ca) 
**Ubuntu Forum**: [http://ubuntuforums.org/forumdisplay.php?f=395](http://ubuntuforums.org/forumdisplay.php?f=395)

 Hope to see you all at a Release Party! If not there, then online!
