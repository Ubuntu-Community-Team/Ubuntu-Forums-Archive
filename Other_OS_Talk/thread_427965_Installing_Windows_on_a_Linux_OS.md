### Post by Calash on 2007-05-02
Gaming on Ubuntu is not bad.  I ended up installing Cedega to play some of my games.  The rest I have on a second partition with Windows XP.  It is all I used that partition for so it is nice and clean, very optimized.  This is all on my XPS Gen 5 system

If you really are set on reloading XP, the best method is to use the Dell system restore disk.  Boot to that and when it asks you tell it to delete all partitions and create a new NTFS partition.  This will DELETE all of your data, files, and everything on that system, so be sure it is what you want to do before you do it.
