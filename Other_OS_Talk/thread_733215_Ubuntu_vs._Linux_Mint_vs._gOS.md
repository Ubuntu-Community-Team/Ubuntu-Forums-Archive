### Post by Ub1476 on 2008-03-23
Ubuntu: The two others are built upon Ubuntu. Most documentation is referred to Ubuntu and not "Mint" nor "gOS". Is also released first, then the others come a little later.

Mint: Codecs, java, flash etc is preinstalled, unlike Ubuntu. Has a more refreshing theme. Probably more, but I never used it so..

gOS: Based upon Ubuntu, bus uses the Enlightenment DE. Lots of google apps "installed" (maybe using prism or something to launch them, or just the default web browser). Enlightenment also has some nice eye-candy effects which you can't find in Gnome/KDE/Compiz. 

That's the basics I know..

### Post by jdgiotta on 2008-03-23
I've used all 3 in the last week or so. This is my take on them.
Ubuntu:
Solid environment with a very large user base. Plus the original branch for both gOS 2beta and Mint. However, I've experienced some annoying issues with Ubuntu at its current version.

Linux Mint:
Solid environment with a very new and sleeker design and GNOME layout. The differences between it Ubuntu and gOS, is that in my experience, is its less buggy. I'm actually starting to prefer it over Ubuntu Gusty.

gOS:
Potential, but extremely buggy. The File Manager is home grown and is very feature-less and crashes often; definitely not big time ready. Even avid gOS users push replacing it with Thunar, but once done it does integrate well.

### Post by seanc7 on 2008-03-23
I've never tried gOS but I've tried Mint and am running Ubuntu now.

Ubuntu is good and very stable - it's directly based off of Debian. For installing/removing programs it has everything from DPKG to APT-GET to APTITUDE to SYNAPTIC (for Gnome) and the KDE equivalent of Synaptic.

It comes with a decent media player, office apps, utilities, etc. You just need to install the MP3, MPG, AVI, etc codecs manually.

Mint is good but I found it slower then Ubuntu on the exact same system. But that could just be me because I just find KDE slower in general. It has a good media player, office apps, utilities, etc installed as well. Plus the MP3, MPG, etc codecs are installed automatically.
