Slackware is a binary distro, unlike gentoo. That said, if you're looking to learn about linux slack is the way to go. You could almost say slack is 'vanilla linux'.

Gentoo is sourcebased. Although Sabayon is based on Gentoo, it's actually a binary distro (I only tried the mini edition and its the most bloated distro I've ever tried). But gentoo should also be worthwhile to play with if you have a LOT of spare time for compiling in addition to the learning curve.

Then there's arch, a bit similar to slackware, but perhaps a bit more bsd like.Perhaps a bit easier to learn than the former two, whilst still being a distro that will teach you a fair bit about how linux works as you will still have to set everything up yourself.
