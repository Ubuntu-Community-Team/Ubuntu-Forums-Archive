### Post by agim on 2008-02-27
I am using kde right now and I am loving it. But there is no reason to dual boot, if you have a fast connection, just use the command

sudo apt-get install kubuntu-desktop

It will install kubuntu and you can choose between kubuntu and ubuntu (gnome and kde) on the login screen. Just hit the icon that says sessions to choose.

To be specific. After using kubuntu and ubuntu side by side (along with xubuntu and various window managers), I now have an ubuntu install with kde-core downloaded, I have added the few apps that I needed and save myself a lot of apps that I don't. This gives me even better ram usage than xubuntu.

I would suggest the kubuntu-desktop though, to get the full range of what it has to offer.
