### Post by ezsit on 2008-04-22
Sorry for the lame attempt at a John Lenon rip-off subject line, but seriously, if you haven't given PC/OS Linux a try, then I highly recommend a spin. It is available at [http://pc-os.org/](http://pc-os.org/).

The developer used remastersys to customize Xubuntu and create a very nice spinoff. Version 7.10.1 takes a few cues from the old BeOS interface design and packages flash, java, and a whole slew of Xine plus codecs into a very sweet distro.

There is a client and server version for dvd-r and a minimal cd-r sized version. I have tried the cd-r version and can say it is quite full featured. The dvd-r version is another 300mb above the 685mb for the cd-r version.

The developer intends to release an updated version 7.10.2 the first week of May, but I like the design of 7.10.1 and some of the changes in store for the next release will diverge from the BeOS look and feel.

Enjoy!
