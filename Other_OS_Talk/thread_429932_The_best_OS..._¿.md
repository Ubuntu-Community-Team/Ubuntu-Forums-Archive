that none of the options were clearly better. My main work is all done on a Mac and for security reasons 
it is never connected to the internet and has nothing but my work applications so my PC is where I have fun.
And I must admit that If i had another Mac I would't consider another OS.

you hear over and over again in forums is that windows isn't secure, windows isn't stable... windows suxors... :)

cause any problems and for me it seems that it does the job better and faster than linux. Wich means

viruses/trojans/spyware to keep it nice and clean...
(but all this is is really not a big bother since I can do it with my eyes shut).

So currently I'm trying to get a permanent solution to this constant juggling. I hate the reinstalls Windows
requires and not being able to keep long time archives or whatever in my system without backing up
(wich ofcourse I always forget come every 2 months).
Most of the software I use on windows is open source, I absolutely love VLC, GAIM, Firefox, OpenOffice.org
so big thanks for the opensource community for creating those AND supporting windows.
What ties me down to windows is World of Warcraft, Newsleecher and being able to calibrate my monitor
using my Eye-One Match hardware. I'm sure someday all this will be supported and more...

It was a fun time in linux but it's not quite there yet atleast for me. Anyone that can find better alternative 
applications on the linux side change to it immediatly!

Anyways, big thanks for the opensource community and the developers that will eventually make linux the 
clearly best OS there is.

### Post by ThinkBuntu on 2007-05-03
I have a few suggestions, in no particular order:

1) Ubuntu Edgy, Feisty if you're willing to deal with some extra bugs for the sake of being cutting-edge. If you really don't care about being cutting-edge, you could use 6.06, the long-term stable version

2) PCLOS if you want a replica of Windows (in look and general usability, not in software), only with Linux stability and impressive speed.

3) Debian if you want a *permanent* solution. I've always had a good time with Debian.

4) MEPIS or Mint. MEPIS is an ease-of-use distribution based on KDE. Its speed is decent, and its hardware recognition is superb. Mint is similar, only it's directly based on Ubuntu. There will be a new release of Mint soon (3.0) and it will be based on Feisty. In my experience, Mint is just like Ubuntu with a different look/feel, a few extra utilities, more useful configurations from the install, and overall snappier performance. However, it's very dependent on Ubuntu which is not so nice.

If I were just playing around, it would be very tough to choose. I'd probably go with Ubuntu or Mint. If I were in a store, and these all costed the same, I'd buy Debian and stick with it for a few years. Since they're all free and easy to get, it's always tough to pick one and stay with it.
