### Post by Sorivenul on 2009-01-25
> **Giant Speck said:**
> I don't have any problem *using* the command line.  I use it all the time.  I just don't want the operating system to be nothing *but* the command line.  If that makes sense at all.
If you install plain Free/Net/OpenBSD, your default will be the command line (even if you install an X server it won't start automatically). This isn't too much hassle though. Make sure your network is working, then (for FreeBSD):
```
pkg_add -v -r gnome2
```

Then:
```
ee /etc/rc.conf.local
```

Add:
```
gnome_enable="YES"
```

Then:
```
echo "exec gnome-session" >> ~/.xinitrc
```

If you want a login screen so you don't have to type "startx" to get to GNOME, enable GDM with in /etc/rc.conf.local with:
```
ee /etc/rc.conf.local
```

Add:
```
gdm_enable="YES"
```

Done. Add what you wish from there. Cheers!
