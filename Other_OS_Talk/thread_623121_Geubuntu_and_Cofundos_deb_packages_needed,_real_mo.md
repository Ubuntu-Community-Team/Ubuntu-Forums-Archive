### Post by darkmaster on 2007-11-25
Geubuntu is now a Cofundos project. You can visit [THIS page]("http://www.cofundos.org/project.php?id=27") if you want to help the project. Geubuntu needs deb packages for its development. If you think you can create very simple deb packages, then you can be paid for it. Offer yourself to complete this task and if you create really working packages, then you'll receive the money bidded :)

Find more informations [HERE ]("http://thedarkmaster.wordpress.com/2007/11/25/geubuntu-in-cofundos-join-the-work-and-get-paid-for-it/")and remember to visit the [Geubuntu Home Page]("http://www.geubuntu.intilinux.com/Home.html") and the[ Geubuntu Cofundos Page ]("http://www.cofundos.org/project.php?id=27")to see the bids, to offer bids yourself or to offer your help to the project.
