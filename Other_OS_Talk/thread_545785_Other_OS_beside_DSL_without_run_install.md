### Post by avik on 2007-09-08
If you're going to use virtualization (the only way to run another operating system *in* Windows), you can use any distro.  However, due to speed concerns, you'll want a lighter distro.

Try Puppy Linux, or Wolvix.  I've heard good things about them.

There's always Xubuntu as well.
