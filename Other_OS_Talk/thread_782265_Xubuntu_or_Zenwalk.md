### Post by timzak on 2008-05-05
I think Zenwalk is lighter on RAM than Xubuntu, at least out of the box.  You could tweak Xubuntu (disable unneeded services and auto-started apps) to reduce memory usage.  But 256MB of ram is going to be tough.  You won't be able to do much multi-tasking.  

Another option is to do an Ubuntu or Debian base install, then just install the apps you want.  So for example, after the base install you have no GUI.  You'd then install xfce4, xorg, gdm, and snaptic, then reboot.  Now that you have a GUI, just run synaptic and install everything else through that.  Doing this, I got RAM usage down to about 60MB on system startup with an xfce4.4.2 window manager.

### Post by digger95 on 2008-05-06
> **sw1995 said:**
> I too am curious about Zenwalk--Does anyone have any experience running KDE with Zenwalk or is it optimized to run best with Xfce?
Zenwalk is optimized out of the box to make the most efficient use of Xfce, which gives it a really nice 'zip' that you just won't find on a Kde system.  You can however use Kde or Gnome (or whatever you desire) with Zenwalk since it's based on the highly stable and configurable Slackware.  I would however strongly recommend to give Xfce a fair shot first.  I was a diehard Kde man prior to Zenwalk but I've found I can do anything I want (and more) with Xfce.  I just plain love it's speed and straightforwardness.  And don't forget that as long as the Kde libraries are installed, you can use the same software in Xfce that you use in Kde.
