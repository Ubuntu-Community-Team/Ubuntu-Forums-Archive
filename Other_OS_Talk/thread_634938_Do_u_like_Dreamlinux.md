### Post by sub2007 on 2007-12-22
When I first tried it I loved Dreamlinux! I briefly trialled it a couple of months ago but ended up having to move back to Ubuntu because it couldn't recognise my graphics card (no matter what I tried). If I could have tried it via the LiveCD then it would have saved a lot of pain (I'm sure the developers have a reason for disabling OpenGL on the Live CD) but from using the LiveCD I was very excited to try the actual distro - even on Live CD the speed was amazing. It also had the fastest install of any distro I've tried (took only about 15 minutes). I could definitely be be tempted to try a future version, it was gorgeous all round. OpenGL is a necessity for me though, but I'm sure that if my graphics card was recognised then I'd certainly have used it for longer. Don't know what I'd be using now though, I can't imagine myself using long term anything other than xubuntu at the moment - it's perfection for me in (almost) every way and is as perfect as I think a Linux distro can get for my needs.

My experience with Dream did make me realise how much I prefer XFCE to GNOME or KDE and so I'm glad I did try Dream just to realise that. And it is always fun to try new stuff.
