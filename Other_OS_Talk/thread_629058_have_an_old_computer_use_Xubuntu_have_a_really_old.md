### Post by drascus on 2007-12-01
I have an old ('98) IBM Aptiva e series. its got a clunky pentium II 400MHz processor and about 128mb of ram. I have considered once or twice getting rid of it though I was able to run it managably with Xubuntu. Then I found out about fluxbuntu. I had tried damnsmall linux and knew that fluxbox was great on this old computer. I gave it a shot. My old computer now runs about as fast as my year old laptop running ubuntu. I am glad this program has come along and taught this old dog some new tricks and gave it a second lease on life. 

pros: ultra fast desktop environment especially good for old, low resource computers. a nice Gui with intuitive controls once you get use to them. pretty well optimized out of box with no non essential packages. 

cons: slow on boot. this can be resolved by removing the boot splash but of course you have to be confortable with command line for this. Also no automatic usb mounting if you know pmount this is no problem. As far as I can tell there is no flash support which doesn't matter on my machine because it can't really handle flash very well anyway. 

overall: if your ready to junk that old computer and you are an intermediate ubuntu user than fluxbuntu won't pose a challange to you at all. you should try it out and give it a second thought as to wether or not you want to junk that computer.  
~Drascus~
