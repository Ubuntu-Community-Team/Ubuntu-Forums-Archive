### Post by Darkhack on 2008-01-24
Damn Small Linux 4.1 which was released on December 2, 2007 uses Linux kernel 2.4.31.  DSL does not use Linux 2.6 for a[ variety of reasons]("http://www.damnsmalllinux.org/wiki/index.php/FAQ#Will_DSL_ever_use_the_2.6_kernel.3F_Has_it_even_been_considered.3F").  However if you want the 2.6 kernel you can try [DSL-N]("http://www.damnsmalllinux.org/wiki/index.php/FAQ#What_is_DSL-N_and__what_are_Minimum_Requirements_for_it_with_X-window").
