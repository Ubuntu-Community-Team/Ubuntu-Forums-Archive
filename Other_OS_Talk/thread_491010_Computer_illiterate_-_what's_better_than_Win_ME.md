### Post by raul_ on 2007-07-03
I have pretty crappy specs too, and you can see in my signature. In this moment I run Arch Linux with E17, but it also runs Xfce just fine, I just use E17 because it's way more pretty. The only thing I can't do with it is view Flash animations. It does everything else just fine. Browsing, music, reading .pdfs :)

Setting up Arch isn't for beginners though, but after it's set up there's nothing much to do.

You could also try a Ubuntu Server install, but Ubuntu just loads everything, even if you don't need it, whereas Arch is more controllable. Not to mention that Pacman is superior to Apt (IMO). 

Also, check out Elive and Fluxbuntu. Elive is Debian based, and Fluxbuntu is a stripped down Ubuntu that uses Fluxbox as the WM, although you can change that easily.

### Post by exploder on 2007-07-03
I have been looking at PCLinux TinyMe it is very light. TinyMe is still beta but shows real promise for older machines. It can be installed to the hard drive but you would have to look on the PCLinux forum to find out how. I read it has the same installer as PCLinux but the installer is tucked away some place not obvious. 

AntiX a MEPIS community project is also very light. AntiX uses Fluxbox and IceVM as window managers. AntiX has the same excellent hardware support that MEPIS has.

Vector Linux seems to succeed on older hardware where other distributions fail. The Standard version is the lightest. Vector Linux Standard provides the fastest boot time I have ever seen on an old PC! 

These distributions have shown the best results for me on an old Gateway P3 with 128 mb of RAM. 
I have tried Xubuntu 7.04 on this Gatewat but it keeps freezing up. Perhaps 6.06 would work but I have not tried it so I can not confirm this.
