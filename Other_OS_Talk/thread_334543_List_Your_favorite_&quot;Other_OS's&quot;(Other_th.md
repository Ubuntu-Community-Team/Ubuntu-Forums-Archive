### Post by mushroom on 2007-01-09
1. Mac OS X:
Other than the dock, the interface rules. Quartz Extreme and everything it allows (Expose and such) is great, especially considering it outclasses in performance Beryl running on a 128 MB NVidia card while running on a 32 MB integrated card. Global menubar is really nice. The fact that iLife is included is cool, too. I like Ubuntu better because of apt-get, the ability to install it on whatever I want, and of course, freeness. I attribute the QE performance to the fact that the video is hardware accelerated; without that point, Beryl runs very nicely.

2. Linux Mint
Cheating a bit, but Mint does include documentation on how to get wireless working without having to connect to the internet. It also includes all the patented codecs out of the box along with a script to get NVidia drivers installed. I prefer Ubuntu, simply because all of those things I can do manually fairly easily, and I like knowing exactly what I'm doing when I install those.

3. Knoppix
It's a monster out of the box. It includes everything, and everything works with absolutely no fuss whatsoever. Unfortunately, when I say everything, I mean everything. Way too many applications for me to be comfortable using it as my main OS.
