### Post by cptnapalm on 2006-09-16
If interested in source code for ancient Unix, [www.tuhs.org](www.tuhs.org) has it.  It was released under a BSD license several years ago.  Might need to run it on a PDP-11 or VAX emulator if you don't feel like porting it to modern architecture (although if you do, send me a copy).  There are old BSD releases as well as Bell Labs research versions of Unix.
