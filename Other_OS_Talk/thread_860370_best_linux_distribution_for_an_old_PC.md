Hi there! First of all, Xubuntu should work just fine with those specs, so if you are already familiar with it, that would be my suggestion. 

If you want something in the *Buntu family, but want a really light system, you can start with a minimal installation ([https://help.ubuntu.com/community/Installation/LowMemorySystems](https://help.ubuntu.com/community/Installation/LowMemorySystems) and [http://www.psychocats.net/ubuntu/minimal#barebones](http://www.psychocats.net/ubuntu/minimal#barebones) for tips) and then build the system up with the applications of your choice. 

Good luck!

### Post by tamoneya on 2008-07-15
I recommend you look at these three:
1. Damn Small Linux
2. Puppy
3. gOS

If you insist on staying with Ubuntu however you could try using fluxbuntu as well.

As usual with linux distros they all have liveCD so you should be able to test out what features you like on each.  Just remember that they will all run a little faster and boot a little faster when you actually get the installed.  CDs are much slower than harddrives.
