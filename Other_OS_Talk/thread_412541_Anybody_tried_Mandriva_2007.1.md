### Post by AdamWill on 2007-04-19
Hi, everyone. This is rumour control, here are the facts ;). I'm Adam Williamson, I work as the official forum rep for the Mandriva forums (among other things).

wxnker is correct to say you do not need to pay anything to be a Mandriva user. Two editions of Mandriva are available for download free of charge: Free and One. Free is a full traditional installer-based edition containing nothing but free / open source software. We've been releasing Free since 1998. One is a hybrid live / install CD (much like Ubuntu, only we've been doing it longer ;) not as long as Mepis though). It contains some non-free drivers and firmware, particularly the NVIDIA and ATI proprietary graphics drivers and the Intel wireless firmware.

*All* the free / open source software that makes up Mandriva is hosted on our public mirrors and available for anyone to use. There is also, from 2007.1 onwards, a non-free repository on the public mirrors that includes the non-free drivers and firmware like NVIDIA, ATI etc. It's easy to set up Internet repositories for Mandriva from the software management tools: [http://wiki.mandriva.com/en/Docs/Basic_tasks/Installing_and_removing_software#Making_more_applications_available](http://wiki.mandriva.com/en/Docs/Basic_tasks/Installing_and_removing_software#Making_more_applications_available)

*All* official updates (security fixes, bugfixes etc) for Mandriva are released through the public mirrors.

The commercial editions of Mandriva differ in the following ways: they include printed manuals, trial membership of the Mandriva Club and the Mandriva Expert support service, they come with a month's installation support, they include a slightly wider range of the free software packages on the disc (but since *all* these packages are available through the public repositories anyway, this is no big deal) and they include a few commercial applications that are not in the free editions and are not in the non-free repository. This is stuff like Acrobat Reader. These packages are also available to Club members via the 'Club Commercial' repository: they're the *only* packages exclusive to Club members and purchasers.

For a while, Club members got new releases a month or so before the public, but we stopped that with 2007.0. Club and public releases now happen simultaneously.

Now to answer some specific points :)

Amorphous_snake, that was a bit silly in 2007.0, yes. 2007.1 has a much improved update interface that doesn't have this problem. (There was also an update for rpmdrake in 2007.0 that fixed it - if you install the updated rpmdrake, then in future when you do updates, all the available update packages will be pre-selected).

wxnker, using rpmdrake to add repositories does *not* add PLF repos. It does add non-free in 2007.1, though.

Hope this is useful to everyone. I'll be installing Feisty in a VM later to check it out. :)

### Post by AdamWill on 2007-04-19
Generally we don't include patent-encumbered stuff in the official repositories. There is a third party repository, PLF, which provides many of these. Of course, we would point out that it's illegal for anyone living in a jurisdiction where these software patents are recognized to use such libraries.

MP3 playback is actually included by default in Mandriva. Encoding is not, encoders are in PLF.

Ndiswrapper is included by default and you can set it up with our network configuration tools (when setting up a wireless card you can choose to use 'a windows driver', which means ndiswrapper: point it at the .inf file and it'll set everything up).

LinDVD was in 2007.0 but it's not in 2007.1, so please don't buy 2007.1 and be disappointed that it's not there :(. We're looking at making it available in the Club Commercial repository for 2007.1. Same goes for Cedega.
