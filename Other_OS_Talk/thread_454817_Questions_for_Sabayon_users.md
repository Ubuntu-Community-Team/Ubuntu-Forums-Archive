This question belongs under the Gentoo section here where Sabayon is discussed.

Kuroo/portage is no more easier/harder to use than synaptic/apt-get, never had dependancy issue. Only thing you get are the odd package that is masked.

The sabayon install itself does not rquire compiling like gentoo. Installing new packages does require compiling with the exception of a few ones like OpenOffice, Java, Firefox etc which have binaries available in addition to source.

Printers work just fine in Sabayon, just install what you require from portage.

I don't think it is any/much harder than debian based distros. Things are just slighly "different". If you install the DVD verion (which has to much bloat for my liking) you pretty much dont have to install anything after that.

Head over to the Gentoo subforum and go and read a few of the sabayon post there to get a better impression as to peoples impressions & opinions.
