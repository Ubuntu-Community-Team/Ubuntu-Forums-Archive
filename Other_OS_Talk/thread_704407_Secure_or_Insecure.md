### Post by xeth_delta on 2008-02-23
> **PrivateVoid said:**

Sorry, but I will have to slightly disagree. AFAIK Mac OSX actually uses a similar method to Ubuntu, not Vista. The user you normally use is not root, it is a regular user. When you are prompted for an administrative action, you are actually performing a sudo. Go to the command prompt in OSX and type "sudo foo". Remeber OSX is BSD derived and hence *NIX like. You are welcome to correct me if I am wrong :)

When you run synaptic, or any other GUI administrative tool in Ubuntu, you are prompted for the password.

That said, I guess that which method you find more useful is a matter of preference. I would really find Windows' methode to be intrusive, and to run as administrator all programs insecure. I prefer the Linux method in which you can become root when you need to modify something in the system or simply call sudo. You know what you are doing when you become root. Just my opinion, of course.
