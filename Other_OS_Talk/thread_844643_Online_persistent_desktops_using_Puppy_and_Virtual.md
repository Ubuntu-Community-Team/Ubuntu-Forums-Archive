I had an idea today that seems interesting, at least to me. I thought it would be cool if people could have persistent online desktops from which they could do anything. I went searching on Google, and apparently there are a few services like this.

My idea though, would be to have a server running systems of Puppy on VirtualBox, and use something akin to VNC to display a users own system. Each user could be limited to say 100MB and 64MB of ram (just guessing).

One default system would be set up. Making a new account on the system would consist of a person signing up, and then the vdi of the system would be copied over for a new system, with an identifier for that person's account.

How possible do you guys think this is?
