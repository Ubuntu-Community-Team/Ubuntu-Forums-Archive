[CENTER]**Welcome Distro Hoppers**[/CENTER]
[SIZE="4"]*This guide will give you some basic information on distro hopping*[/SIZE]


**Tools/Programs needed**
1.A laptop or desktop
2.Basic computer knowledge
3.Virtual Box
4.The live cd or installation cd of the distro you want to try.
5.(optional)A ban hammer to ban me with

Ok now are you set,good.

_Now lets install virtual box first ok?_
Go to This website and follow the guide to install on your Linux,Windows,Mac,BSD,and/or other OS.
[https://cds.sun.com/is-bin/INTERSHOP.enfinity/WFS/CDS-CDS_SMI-Site/en_US/-/USD/ViewProductDetail-Start?ProductRef=innotek-1.6-G-F@CDS-CDS_SMI](https://cds.sun.com/is-bin/INTERSHOP.enfinity/WFS/CDS-CDS_SMI-Site/en_US/-/USD/ViewProductDetail-Start?ProductRef=innotek-1.6-G-F@CDS-CDS_SMI)

_Once you have installed that lets set up your virtual OS ok?_
[http://youtube.com/watch?v=19vqH9oasAQ](http://youtube.com/watch?v=19vqH9oasAQ)
That video is a bit old but hey it shows you how it works =].
If you run into bugs just tell me I can probably help you out.

Now you have your Virtual Machine.

*Troubleshooting*
_Network not working?_
Check in your VM settings if you set your network device.
Also in the VM machine if you installed the software for your network card.

_Not enough ram?_
Ah yeah this problem I came across once.
I had to simply set my Vram a bit higher and it worked just fine.
Also make sure your specs virtual specs are meeting the system reqs.

_Extra info_
People should be aware that VM testing is a bit different in terms of hardware because VM uses Virtual hardware, as opposed to your actual hardware specs
