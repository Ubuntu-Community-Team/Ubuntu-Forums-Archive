### Post by RumorsOfWar on 2008-05-21
Gentoo. 
Everything compiles from source (usually). Everything can be configures exactly for your specific hardware. Ultimate power your hardware can deliver.
If you want to ease into it, start with Sabayon. Its basically Gentoo pre-built with a nice set of apps, but fits most hardware. ( but starting with Gentoo directly will teach you more !)
When I used Sabayon, I found myself tinkering with the OS more that I was actually *using it.* (Hence my return to Ubuntu)
 It supports a separate home partition, so I'd suggest sharing your home with another OS like Ubuntu, that way when your experiments break it, you can boot back and learn the fix. :)

About the same.

Gentoo: install stage 3 (base) system, get the portage tree, then start downloading and compiling.

Arch: install base system, then update pacman, then start downloading and compiling.
