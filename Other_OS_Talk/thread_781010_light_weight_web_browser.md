### Post by stream303 on 2008-05-08
Epiphany-browser is the lightest I go (don't download epiphany, download epiphany-browser)

Lot of lynx fans in here.  Don't forget elinks.

Aside from using lynx or elinks as web-browsers, they make nice file browsers, and lightweight file managers

```
 lynx /usr/share/doc
```
or just your home directory:
```
lynx .
```

Subsitute elinks if you like.  In lynx, hit the "f" key for full file management menus.  Not mc for sure, but getting off topic...
