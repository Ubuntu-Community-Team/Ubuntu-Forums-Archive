### Post by tommcd on 2007-12-05
See this page from the Feather linux site:
[http://featherlinux.berlios.de/booting.html](http://featherlinux.berlios.de/booting.html)
It explains that you can create a /KNOPPIX directory on your /root partition to place feather linux. You then download the boot.img and place that on a usb stick or a CD (or possibly even a floppy) to boot from. The feather site says the whole think can run off a usb stick, or be installed on a hard drive.
