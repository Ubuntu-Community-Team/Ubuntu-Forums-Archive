### Post by BoyOfDestiny on 2007-06-21
"The prohibition against running Vista Home Basic or Home Premium in a virtual machine is easily one of the most disappointing features of Microsoft's end-user license for Windows Vista. It's a move that has frustrated PC enthusiasts and particularly Mac OS X users, many of whom (believe it or not) would like to be able to legally run Home Basic or Premium in a VM. Users were outraged that Microsoft expected them to pony up for more expensive copies of Vista Ultimate or Business, just to get the legal okay to run in a VM."

[http://arstechnica.com/news.ars/post/20070620-microsofts-ditches-about-face-on-virtualization-in-11th-hour.html](http://arstechnica.com/news.ars/post/20070620-microsofts-ditches-about-face-on-virtualization-in-11th-hour.html) 

I'd just like to bring some attention to this. I know there are Ubuntu users here who may want to use virtualization to run Windows Vista, and would rather not have to buy a more expensive version when there is no technical reason to do so.

Let MS know how you feel about this.
