### Post by exploder on 2007-07-08
LinuxMint 3.1 will come with a new configuration tool called mintassistant. 

The tool gives you the ability to:
                                                           Select the default wallpaper
                                              
                                                           Select the menu style (Gnome or Mint style).
                             
                                                           Lock in the kernel

                                                           Turn Fortunes off or on

                                                           Gives you the choice of having a graphical root log on or not.


I personally like having a real root account to log into. Now I can use Mondo Recovery in LinuxMint!

Check out the Romeo section on the LinuxMint forum if you want to use mintassistant in LinuxMint 3.0.
