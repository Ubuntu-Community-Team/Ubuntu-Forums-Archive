### Post by notwen on 2008-03-19
> **RJ Hythloday said:**
> I've heard good things about arch, but I've never compiled anything, would it be user friendly enough for a m$ family?

Read up here on Arch's official [Beginners Guide]("http://wiki.archlinux.org/index.php/Beginners_Guide") or try [Archux's more user-friendly guide]("http://archux.com/page/installation-guide").  If either of those seem easy enough for you, give it a shot.  I've installed Arch in virtual PCs and all went smoothly enough.  Just know going into the install that it is a CLI install and you will need to use CLI to install your preferred GUI(probably XFCE in your case) and other things.  It's not that hard and of course there are plenty of Arch users here on ubuntuforums(oddly enough, lol) should you run into any issues.  =]

PS
---
You may also want to consider doing a base [XFCE Debian install]("http://mirrors.kernel.org/debian-cd/current/i386/iso-cd/debian-40r0-i386-xfce-CD-1.iso").  Imagine a stripped down version of Xubuntu.
