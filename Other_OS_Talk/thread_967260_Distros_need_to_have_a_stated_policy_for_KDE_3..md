### Post by blackbelt_jones on 2008-11-01
I was totally blindsided by Ubuntu going from using KDE3 as the default to dropping it entirely in one release.  I know all about Hardy being LTS, but KDE 3 is current, is still maintained, and has a huge user base.  KDE 3 is no longer the focus of development, but the KDE team is commited to maintain KDE 3 as long as it has the users, and I need to support that by using a distro that's not going to try to impose an artificial timetable on that.  Two years from now, if KDE is going to continue to maintain 3, and if I want to use 3, I dont want my distro getting in the way of that.

So, I guess I'm going back to Debian... but I'm also going to start communicating with the Debian organization about the intentions vis a vis KDE 3.  It's not unreasonable for KDE 3 users to press their distro of choice for an idea of what to expect.  *Distros should have a started policy about KDE support, and users should be able to find out what that policy is. *  I don't intend to be blindsided again.
