back around.. 5.0-ish?(whenever they came back into play) redhat got out of the desktop game completely. it's used primarily as a workstation and server OS.

I have very limited knowledge of the two, but from what I know fedora is designed to be an absolutely free and open source distro aimed at the desktop, while redhat is aimed at servers and workstations utilizing proprietary software and drivers.(although they DO support opensource and incorporate it as much as possible) both come loaded with very different packages.

### Post by Simian Man on 2008-11-22
Fedora and Red Hat use the same administration tools, so they are similar that way.  However, Fedora includes the latest versions of many packages, some of which are development versions.  Red Hat, on the other hand, only uses perfectly stable stuff.

If you want to learn Red Hat by using a free OS, go with [CentOS]("http://www.centos.org/").  It is basically a recompiled version of Red Hat without the Copyrighted branding.

Hope that clears it up.

### Post by Simian Man on 2008-11-22
Yes Red Hat has always been committed to open source.  That is why CentOS can exist - because Red Hat distributes the full source code.  They just do not offer or endorse the system for general purpose desktop use any more.

In fact the Red Hat company [contributes time and code ]("http://fedoraproject.org/wiki/RedHatContributions")to many OSS projects that other distros rely on.
