### Post by kalwisti on 2008-02-23
Hi, m-k,

I happen to be a Xandros user (although I don't own an Eee) and I can verify what others have said; the OS running on the Eee is a modified version of Xandros which Xandros, Inc. and Asus developed together. Xandros does not officially support the OS, but instructs users to contact Asus for assistance. 

My understanding is that the Eee actually has two OS "modes": a simple, minimalistic GUI (the one you're speaking about); and a more advanced GUI which is similar to the "regular" versions of Xandros. 

I don't know if you can somehow separately download and install the "simple" version, but I recommend that you visit the Eee User Forum -- where it sounds like adventurous geeks are tweaking the daylights out of the Eee. Hopefully someone there can give you a more definite answer. 

[http://forum.eeeuser.com/]("http://forum.eeeuser.com/")

If you're interested in trying a free version of the "regular" version of Xandros (Xandros 4.1 OCE [Open Circulation Edition]), you can download it from here:

[http://wwbizz.com/Download/Xandros/xandros4oce.iso]("http://wwbizz.com/Download/Xandros/xandros4oce.iso")
(This .iso is large enough that is has to be burned to a DVD rather than a CD).

The full discussion thread from the Xandros User Forums is here:

[http://forums.xandros.com/viewtopic.php?t=30692&postdays=0&postorder=asc&start=30]("http://forums.xandros.com/viewtopic.php?t=30692&postdays=0&postorder=asc&start=30")

Following their agreement with Microsoft, it seems that Xandros, Inc. has made it increasingly difficult to find versions of the OCE.

HTH.
