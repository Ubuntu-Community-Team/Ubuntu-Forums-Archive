### Post by LaRoza on 2008-02-17
OpenSuSE would be very easy for a Windows user to use and setup.

The newest version is very complete. It comes with all the apps you need, and has what people typically want: Compiz, Flash, and Java.

I don't know about Codecs, it may come with them as well.

It has a familiar interface for Windows users, and OpenSuSE has a menu that reminds me of a start menu.

The DVD has GNOME and KDE on it, and you can get live cd's with either.

If you are going to introduce them to a new OS, make sure you explain it isn't Windows, and you provide support.

To do this, you would have to use a distro you are familiar with. (OpenSuSE is easy to learn, if you go that route)

### Post by tdrusk on 2008-02-17
I've been using PCLOS lately. It runs great on my computer (specs in my sig) and works great after I upgraded the kernel (which is in the repositories). The sound didn't work. That's why I upgraded the kernel. 

They have a gui for almost everything. It is the easiest distro I have ever used and it runs faster than ubuntu and mint for me. 

It also comes with needed codecs. 

If I were you I would set it up and then if they need help then use a virtual desktop to fix it. 

Other than that, you have experience with it, so it shouldn't be to hard for you to setup.

That and it's pretty :P

I'm not a fanboy. I don't go around telling people it's better, because better has different meanings for everyone. I do think that it's great for people coming from Windows and people who want to keep it simple.
