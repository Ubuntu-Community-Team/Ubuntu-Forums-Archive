### Post by darkmaster on 2007-11-30
Good new today, finally I learned how to create properly setted and configured .deb packages for any debian based distro, so I've been as well able to develop and complete the new Geubuntu Prima Luna deb packages :)

Install them now, even if you already run a Geubuntu desktop installed from Live CD! These packages will not only enhance your Geubuntu experience but they will also give you the possibility to upgrade easily to the new upcoming Geubuntu Luna Nuova, based on Ubuntu 7.10! So jump to the guide now and install the new Geubuntu packages.

If you are a normal Ubuntu user, on the other hand, why not trying out Geubuntu installing its metapackages? It is very easy now to try Geubuntu on you desktop and remember that Geubuntu is also the easiest way you have to try out the power and flexibility of E17 on your machine ;)

Stay tuned, very soon Geubuntu Luna Nuova will be released... and it will be a completely new Geubuntu experience :D

Sorry, I should have ecplained that better.
These packages are for Prima Luna and that means you can install them on UBUNTU FEISTY, infact, if you read the guide, the apt line specifies it is for Feisty. My fault, sorry.

Well, as for the new OS; is Kubuntu a new OS? For as I see it it is not. Ubuntu is the new OS. Kubuntu is Ubuntu with KDE. The same way, Geubuntu is Ubuntu with a little of Gnome and a lot of E17. So, you have a working Feisty? Well, you install Geubuntu packages for Prima Luna and voilà! A new Enlightenment Session appears and you can try Geubuntu at reboot or logging out and in again. 
That's it. 

The distribution of course exhists, as Kubuntu and Xubuntu exhists, but they are still Ubuntu. For Geubuntu it is still a little different because E17 is not in the official repos so Geubuntu is not technically just a remix like Kubuntu and company, but on March E17 should be released, so for the next Ubuntu maybe Geubuntu will really be a remix :)
But that was only a technical problem.

The distribution is only to show people the goals and level of usability reached. Of course if someone prefers E17 than plain gnome, it chooses directly Geubuntu and not Ubuntu. But in Geubuntu: apt-get install ubuntu-desktop and here you are... it's always the same wheel :)

Also, Luna Nuova (Gutsy) will be out soon, so you'll be able to test Geubuntu on gutsy too very soon.
