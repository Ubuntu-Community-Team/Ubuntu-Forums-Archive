### Post by mthei on 2007-09-30
I really like Sidux, which is based on Debian Sid and seems easier to use than some other "rolling" distros (although I know once Arch is set up, it's easy to maintain than Gentoo, for example, although I like Gentoo). It comes with KDE by default, and are discouraged from using Gnome, which kind of bothers me. However, it's got a very fast booting time, and installation takes less than 10 minutes, despite how many programs it comes with (in fact, you may spend your first day using it removing whatever you don't need (I still don't understand why it needs to come with two variations of Firefox, both of which are indistinguishable from each other). It's hardware detection is fantastic there are scripts that really help out setting things up (it sticks heavilly to the Debian Free Software Guidelines, so you may need to enable a lot of different repositories for multimedia codecs and so forth).
