### Post by riromero on 2008-12-02
Yes, I tried it some months ago. It was great for awhile. Once you get into it though, it's really just a frontend for regular Ubuntu and various multimedia apps. If you can get each of the applications to work then the frontend is a nice convenience. If you can't, well... Problem is, I think earOS is dead now. Nothing has happened for months. In the meantime, I installed XBMC on Ubuntu. It's way, way better and comes self contained with a player, codecs, database backend, everything.  And it worked perfectly the first time. Plus it has a very active development team and user community to offer helpful advice.I would recommend XBMC over earOS.

### Post by riromero on 2008-12-02
Here is the XBMC link...

[http://xbmc.org/](http://xbmc.org/)

Note that XBMC isn't a distribution but a self-contained application you can install and maintain through apt-get in regular Ubuntu. In fact, I have done the minimal command line Ubuntu install with no window manager and run XBMC in fullscreen mode automatically after booting. My hardware is old but the performance is fine with or without the gnome window manager. I have a 2.0 GHz Pentium 4 with 500 MB Ram and 6 year old nvidia AGP video card. All my music and videos are on a separate machine and stream over my home network perfectly. And it's free!
