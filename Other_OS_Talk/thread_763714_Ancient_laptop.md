### Post by LinuxGuy1234 on 2008-04-25
I'd suggest a minimal install of Ubuntu and then adding IceWM, Dillo, sendmail-bin and maybe adding xedit.

Add xdm for a log in manager.

Do this:
1. Do a minimal (command-line) system install from the alternate CD.
2. Uncomment the universe and multiverse repos.
3. sudo apt-get update
4. sudo apt-get install xserver-xorg dillo sendmail-bin icewm x11-apps xdm
5. Tada!
