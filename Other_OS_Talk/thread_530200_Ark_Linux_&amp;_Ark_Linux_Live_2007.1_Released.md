[IMG]http://forums.erodov.com/showthread.php?t=1820[/IMG] Ark Linux is a GNU/Linux distribution designed especially for desktop use, easy to handle even by people without a Linux background experience while still powerful enough for experienced users.

 
**The main goals of Ark Linux are:**[LIST]
[*] Be easy to learn and use
[*] Include all the tools and applications the typical desktop user will need
[*] Avoid becoming "bloated"
[*] Make the installation of additional software as easy and fast as possible
[*] Be a technically sane development environment[/LIST]
**About Ark Linux Live:**

Ark Linux Live is a Live CD version of Ark Linux, which means it can be booted directly from the CD, without requiring installation.

***The primary uses of the Live CD are:***
[list]
[*] Recovering a normal system after a big hardware or software failure (Ark Linux will launch a special live CD for this purpose later, with more recovery tools and fewer other tools)
[*] Checking the system out without any risk
[*] Showing others what Linux is like, on their own hardware
[*] Taking your favorite operating system with you; e.g. surfing the net with Linux even if you're on vacation and the only Internet cafe in town is a Microsoft shop;
[/list]
