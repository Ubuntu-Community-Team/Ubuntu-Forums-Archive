### Post by Presto123 on 2007-12-20
First off, check to see if your Vista partition is locked. If it is, follow this tutorial: [http://unlockforus.blogspot.com/2007/10/unlock-me-partitioning-your-drive-in.html](http://unlockforus.blogspot.com/2007/10/unlock-me-partitioning-your-drive-in.html)

That little fun thing will unlock a portion of your NTFS system and resize it all at the same time. My advice for dual-boot is to free up about 20-30 gigs unless you have more space. This is because, if you really like Linux, it's nice to have that space for games and such. FYI: If you like first person shooter games...there's several nice games that can run natively in Linux or are CREATED for Linux.

Great Linux FPS Games:
Cube (Several different versions)
Tremulous
Open Arena (It's okay...anyway)
Padman

Games with Linux Installers:
UT2004 (YUP!)
Quake Wars: Enemy Territory (New Game on the market)
and Possibly the latest Unreal Tournament (Information's a little flaky right now, though.)
