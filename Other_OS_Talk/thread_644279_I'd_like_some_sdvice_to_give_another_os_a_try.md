### Post by chippy99 on 2007-12-18
I used to run Gentoo and it was great - fast and excellent online documentation. The problem is that you have to compile everything so setting up your system takes about 3 days and whenever you install a new app it starts recompiling loads of other apps. I was spending more time looking after my system than I was using it.

So unless you have lots of time and are fairly experienced with Linux I'd recommend trying one of the others or you could stay with Ubuntu :)
