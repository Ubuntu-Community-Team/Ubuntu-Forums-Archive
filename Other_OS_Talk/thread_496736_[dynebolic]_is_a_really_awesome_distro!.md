### Post by ThinkBuntu on 2007-07-09
> **Eggnog said:**
> Be warned.  Mint is extremely addictive.Not nearly as addictive as dyne:bolic. I'm positively hooked on that distro, not only because it's an artistic machine (means it's great for all arts), but because of the attitude of the distro and the lead developer. For one, the whole thing's built from scratch, and as such is optimized for audio production and making the most of limited resources. It has built in farm-like tools to that, if my library weren't so damn locked down, I could drop the disc in, boot, and use the processor power over the local network to do some very fancy 3D-rendering and the like.

Another neat feature is that it is "installed" by just dragging the CD's contents into your current system, which makes it non-destructive, enabling me for example to keep my Arch install while fiddling with the best multimedia distro whenever I please. The distro automatically detects the dyne/ directory on boot, and you're in!

In addition, you can save your dyne:bolic Home directory to a USB and essentially work anywhere securely: it uses 256-bit encryption if you please. So in theory, I could just carry around my USB thumbdrive, the LiveCD, and a Serial to USB connector, and I could produce amazing media (or whatever else I need to do) in any computer produced in the past twelve years, no problem.

Call it Linux for nomads, Linux for Rastas, Linux for Artists, but whatever it is, it's a really unique project that I'm hoping to make good use of.
