I like your idea of adding the additional info. Maybe make the flip side larger to contain everything.

The idea behind Lampost is to make Linux visual and user friendly. I also wanted to differ from what has been done in the past. The icons are an example of that. Instead of a picture and a name below it, I wanted to make them user friendly and full of info. I also wanted to take a page from Apple's widget book and use the back of them.

The way the icons are presented for the past 25 years hasn't changed either. They're all aligned the same. At the bottom right are the resize and rotate options. They allow a user to change an icon's size and rotation. This allows for greater customization of the look and feel.

On the side of each icon is an "info" tab. By clicking the tab, the icon flips over and expands to show more info about the program and also allows the user to start it instead of just double clicking it.

I also would like to see Lampost incorporate Virtual Box and Wine into it's makeup. I would like to see it scan for Mac and Windows programs and then create a desktop icon for them. When a user clicks on the icon, it starts up the program without the user knowing that it's in either program.
