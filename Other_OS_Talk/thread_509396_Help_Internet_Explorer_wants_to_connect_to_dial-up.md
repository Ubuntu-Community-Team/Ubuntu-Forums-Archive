### Post by qamelian on 2007-07-25
Right-click on Internet Explorer and select Internet Options. Alternately, launch IE and go to Tools > Internet Options. Click on the Connections tab. Near the top of the box is a section labelled Dial-up and Virtual Private Network settings. Immediately under that section is a list of three options:

1) Never dial a  connection
2) Dial whenever a network connection is not present
3) Always dial my default connection

You should select 1). Below this is a section labelled Local Area Network (LAN) Settings. You may need to click on the LAN Settings button to ensure that IE is configured correctly to connect to the net via your local connection.
