### Post by snowpine on 2008-09-26
> **Therion said:**
> Isn't that what LiveCD's are for?

Some advantages of Virtualbox over Live CD:

1. You don't have to burn a CD
2. Live CDs are slower
3. If you have enough ram, you can have lots of virtual machines running at the same time 
4. You can't save changes or update a Live CD
5. You can save "snapshots" of your virtual machine
6. You can control the specs of your virtual machine, for example if you want to test whether a distro will work with 256mb of ram.

### Post by sub2007 on 2008-09-28
Just so you know, from my experience, getting Arch set up in Virtualbox can be quite difficult - it was more difficult than setting it up for real. There's a Virtualbox graphics driver for Xorg in the Community repo so make sure you download that before trying to install Xorg. To install it then you'll need to enter this:

```
pacman -S virtualbox-ose-additions
```

If you do that it shouldn't be too much trouble, follow the much loved Beginners Guide at the Wiki and you'll be fine.

Enjoy!
