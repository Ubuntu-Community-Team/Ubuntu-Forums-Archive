### Post by RAV TUX on 2007-07-07
The CafeLinux Development Team announces the new release of Oz Gnome 1.0, after the popular Live version release of Oz Live(GNOME). CafeLinux Labs is proud to announce the release of [Oz GNOME 1.0]("http://www.rpath.org/rbuilder/project/oz/") install DVD and dual CD's.

Oz GNOME is custom built for GNOME users who enjoy a GNOME environment. This differs from the Oz Enterprise release in that it does not include KDE. Like all Oz releases it does include Opera installed by default along with Firefox, I have also included GnomeBaker. 

Download, install and enjoy [Oz GNOME 1.0]("http://www.rpath.org/rbuilder/project/oz/") ;)

### Post by RAV TUX on 2007-07-07
> **corney91 said:**
> In what ways is it different to normal Gnome?

GNOME is simply GNOME, no one is reinventing GNOME, The OS is complently different then other Linux OS's, Oz Gnome utilizes Conary, much like Oz Enterprise(Conary is standard on all rPath based systems).

This makes for a very stable and fast OS overall.

The main difference with this release from previous releases of Oz Enterprise is that Oz GNOME 1.0 is only GNOME based for it's DE, while Oz Enterprise includes both GNOME & KDE(which during install you can choose one or the other or both).

It is a release for GNOME users who only want GNOME.

Like all Oz releases it includes Opera by default, as well as Firefox, this release also includes GnomeBaker.

This for the most part is a install version of Oz Live(GNOME).

Enjoy. ;)
