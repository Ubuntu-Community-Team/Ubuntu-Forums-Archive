### Post by LeoSolaris on 2008-03-19
Sabayon isn't too bad if you have a higher powered system. I popped it in a a Live and it is kinda fun, though personally KDE and I do not get along at all. While I know it's as bloated as Gnome, it never really feels right. Maybe it's a Window's aversion.

Sabayon had a rather nice black/red theme right out of the box, so if you're a dark theme type, that by itself would be appealing, less to have to tinker with to get looking 'right.'

Maybe PCLinuxOS, or if your not looking for KDE distro's specifically, OpenGUE uses enlightenment, and it is rather pretty, even Ubuntu based. Granular has a preview out that is neat, though it needs more time to be really all there.

One of my favorites would be Puppy, you can get one of Puppy's derivatives to be a little more fleshed out if you do not like the minimalist thing that Puppy does. One of these days I intend to build some tiny machine with no hard drive, just a GB of RAM, and probably a Epia pico mobo, just for smallness sake. Add some 7" touch screen, and I would have a nice little system for Puppy.

### Post by jayson.rowe on 2008-03-20
Try out Mandriva (with GNOME if you wanna change, or KDE if you wanna stick w/ it)
Also set up Slackware w/ XFCE for a lightning fast system.

If there are parts of Ubuntu you like but want to try something different, go to Ubuntu's roots and install Debian.

Etch (stable) is pretty boring, but you can update to Sid and live on the edge :D

Check out Fedora as well while you are at it - it's the "consumer/testing" version of Red Hat Enterprise and is always an up-to-date leading (bleeding) edge distro that updates at a very fast pace.

I've also heard great things about Foresight Linux, and I'm planning to play around w/ it some in a VM to see what it's all about.
