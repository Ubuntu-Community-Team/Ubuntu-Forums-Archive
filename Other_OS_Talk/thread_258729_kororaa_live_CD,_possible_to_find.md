### Post by 3rdalbum on 2006-09-17
The Kororaa Live CD is actually illegal (for the developers to have made), because it combined proprietry code with the Linux kernel. So unfortunately, a Live CD test of XGL is impossible.

If your graphics card is old, it might be supported by the open-source drivers, rather than the proprietry ones. If this is the case, you should download the Edgy Knot 2 Live CD which has AIGLX, and install Compiz on top of that.
