### Post by Minus0 on 2007-05-19
[http://doc.gwos.org/index.php/VirtualBox](http://doc.gwos.org/index.php/VirtualBox)
That link will help you with the running XP on Ubuntu aspect.  It's pretty easy and runs really well.

[http://www.vmware.com/download/player/](http://www.vmware.com/download/player/)
That link will allow you to download the VMware Player for Windows XP.  Once you install that, you can go to [http://www.vmware.com/vmtn/appliances/](http://www.vmware.com/vmtn/appliances/) and find premade images, which include Ubuntu 7.04.

Don't expect much help with the Windows aspect, but there are plenty of threads here about using VirtualBox on Ubuntu if you do a search for "VirtualBox".
