### Post by handy on 2008-11-09
Arch is the easiest most enjoyable learning experience I have had out of the distro's & BSD's.

The Arch [***_Beginners Guide_***]("http://wiki.archlinux.org/index.php/Beginners_Guide") in combination with [***_Official Installation Guide_***]("http://wiki.archlinux.org/index.php/Official_Arch_Linux_Install_Guide"), the [***_Post Installation Tips_***]("http://wiki.archlinux.org/index.php/Post_Installation_Tips"), & these [***_How-to_***]("http://wiki.archlinux.org/index.php/Category:HOWTOs_(English)") pages, make Arch so easy to learn with.

Gentoo took so long, (waiting for all that compilation) & still failed due to an incompatibility with my BIOS apparently, on the same machine that Arch had no problems with.  In its favor Gentoo also has great documentation.

I'd recommend Arch with Openbox, as setting up Openbox gives you the opportunity to have the additional fun of configuring your new super light & fast window manager using configuration files also.  [***_Urukrama's Openbox Guide_***]("http://urukrama.wordpress.com/openbox-guide/") is wonderful in this regard.

Personally I don't use Openbox anymore, (though I'm glad I did) I use XFCE, which must be the easiest front end to install & setup of the lot. I find that XFCE suits me best out of all of the options I have tried thus far.

It is the going, not the getting there, where the fun is though. :-)
