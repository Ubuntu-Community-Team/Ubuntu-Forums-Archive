### Post by Adamant1988 on 2007-03-28
Foresight is a great distribution, and actually does only download the changed parts of the files, resulting in much smaller download times.  The commands aren't as friendly as apt-get or anything, but the distribution is still very nice.  What else is really nice is that the distro's package manager acts as a revision control system, "Update breaks your computer, just roll back to an old version, no problem".
