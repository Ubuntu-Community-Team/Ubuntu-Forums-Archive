### Post by igknighted on 2007-12-06
Don't think we can discuss OSX on windows-PCs here.  Forum rules forbid discussions on things like pirating (or breaking license agreements).

> Mandriva - As I recall , it wanted users to sign up for software

Absolutely false.  Mandriva offers a "powerpack" version which comes with commercial, pay-for software as well as all the free stuff in Ubuntu, but both the free and one versions have about 16,000+ packages (comparable to Ubuntu) of all the free software you could need.  You would want to add the PLF repo's (rough equivalent of multiverse) to get some of the patent encubered software that they cannot legally host at their own repo's.

EDIT: On your fedora system: any reason you are not using icedtea? It was installed with F8 by default and is about 99% compatible with jre1.6.  As far as installing the sun version, I just followed the directions on java.com and it worked perfectly.  Not to mention, if you really prefer synaptic, you can use apt/synaptic with fedora or other rpm distro's.  Just surch for it in Yumex.  Also, try out yum-fastestmirror and yum-presto (again, look in yumex).  These should speed up the package management quite a bit.
