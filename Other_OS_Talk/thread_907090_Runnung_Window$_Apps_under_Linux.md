### Post by Bakon Jarser on 2008-09-01
If you have a windows install disk you could use virtualbox.  It's basically like running windows on top of linux.  Here's the guide I used to set it up.

[http://ubuntuforums.org/showthread.php?t=770745&highlight=virtualbox](http://ubuntuforums.org/showthread.php?t=770745&highlight=virtualbox)

I would try and use wine first and if you can't get the program working with wine then try virtualbox.  By the way, look up the program you are trying to usee in the appdb at [www.winehq.com](www.winehq.com)  sometimes there will be special instructions for getting that particular program to work with wine.
