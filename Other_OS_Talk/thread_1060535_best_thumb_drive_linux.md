### Post by scottuss on 2009-02-07
I would recommend using an Ubuntu computer to run the Create a USB Startup disk. That way you can select an Ubuntu ISO (Ubuntu is good for USB boots, even if it is a little slow at times) and you can change how much persistence you want.

Provided that the computer you are sat on supports it, the eye candy you desire should be available in the form of Compiz.

The other distros are OK, but for me Ubuntu is the best. The smaller "light" distros tend to be either A) Very ugly B) lacking functionality or (most commonly:) C) Both. ;)
