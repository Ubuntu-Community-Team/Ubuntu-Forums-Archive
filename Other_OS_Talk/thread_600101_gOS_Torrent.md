### Post by kensan2 on 2007-11-13
For those waiting for gOS to exit Alpha stage, *for me personally* I haven't found any crashes or reboots.  Even hibernate works out of the box, which doesn't even work in my windowsxp partition on this same dual boot laptop!

I will say though that the gOS installer stomped my existing grub setup (as well it should as I blew away my previous gentoo install for this) and left my laptop unbootable after the install.  I had to use the livecd, mount the root partition, edit the default file and create a menu.lst file by hand to get xp and gOS booting again.

Luckily my experience with gentoo made me completely confident in doing these things by hand;-)

I would imagine on a dedicated linux box with 1 hard drive the install would have just worked.

Just giving a heads up to anyone, and a thumbs up to gOS for being slick and fast and smooth!  (This is on a Sony VAIO laptop PCG-K27.)
