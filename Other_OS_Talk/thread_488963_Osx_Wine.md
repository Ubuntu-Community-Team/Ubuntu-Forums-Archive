### Post by benanzo on 2007-07-02
If you're looking for something that emulates the PPC architecture have a look at [PearPC]("http://www.pearpc.net/").  Though I'm not sure that's what you want.

I'm not aware of an emulator that will run OS X software (ie iLife) the way WINE runs Win32 software.  The most progress has been made toward installing OS X itself in an emulated environment, in which you could then run your apps.  If you find something post back and let us know!
