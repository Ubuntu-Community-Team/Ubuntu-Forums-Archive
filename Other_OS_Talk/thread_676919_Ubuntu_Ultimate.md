### Post by Zero Prime on 2008-01-24
UE has a lot of preinstalled software.  Some of the software installed happens to be software that newer users have a hard time installing.  There is also an upgrade script in the OS that allows the user to install more software that can not be installed by default.  

TheeMahn has also written a couple of nice scripts to go along with UE.  I like the Usplash maker script.  You just right click on an image and click on make usplash.  Nice little script.

By signing up to the Ultimate Edition forums you can download the UE Christmas Edition.  Consider this a 2.0 version of the original Ubuntu Christmas Edition.  It comes with Gnome, KDE, Sawfish, and more as WMs.  Then there is also the 1.6 Gamers Edition that of course includes a lot of games.
