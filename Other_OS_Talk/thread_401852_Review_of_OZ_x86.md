### Post by charlie85254 on 2007-04-05
OZ DVD x86 Install version
|It-5| Install
Easy install no problems.  If you can foul up this install you don't need a computer:KS 

|H-5| Hardware support
Great! Everything just worked.

|C-3| Codecs
Everything that you expect to find missing in a American distro was in fact missing...No surprise there.


|U-5| USB
Great USB support.  Thumb drive, Camera, MP3 player.


|G-4| Graphics
As expected I just needed to set everything up


|S-5| Speed
Impresive for Gnome!  Seriously this is a example of what can be done with Gnome speed wise...Time for other distros to follow suit.

|F-NA| Forum Support
I am not giving a grade based upon forum support as I did not need to seek any.  The forum is small so it may take awhile to recieve a response HOWEVER you can talk directly to the developer of OZ so I would suspect you would get good support.

Overall I am really interested to see where OZ goes.  Right now it is in a fledgling stage and I would add trying to find a personality of it's own...But it is clear that a lot of love has been poured into it.  If your a distro junkie you need to check out this distro... If on the other hand you are a user searching for a new flavor of Linux for what ever reason you would do well to check this one out.  

Overall---OZ rocks :guitar: I can't wait to see the next release.
