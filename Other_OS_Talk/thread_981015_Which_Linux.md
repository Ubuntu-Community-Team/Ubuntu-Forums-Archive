### Post by candtalan on 2008-11-13
> **gaurav200x said:**

Kindly advise which linux would be suitable for me? *I tried installing from Ubuntu 8.04 LTS but the screen would get stuck at the orange-coloured page with a peacock. The processor and the dvd rom showed that it was still working but despite an hour, nothing happened.*

Welcome!
As has been said in other responses, the 256 MB ram your machine has is too small to use the conventional Live CD installation of 8.04

Ubuntu 8.04 will work reasonably well in your machine once it is installed on the hard drive, but you need to re consider how you install it, avoiding use of the Live cd facility which by definition needs to run in RAM.

You can certainly install 8.04 directly from the booting startup menu of the live Cd. Partitions will be changed, at least, and you will have some risk of loosing existing data, if any resides on the machine, so backups would be important.

An alternative method of installation, accommodating the amount of RAM in your machine, would normally be from a CD called the Alternate CD. This is a bootable CD which is available for download also. You might find that this 'text only' installation with the Alternate CD is a bit faster than from the boot menu of the Live CD.
