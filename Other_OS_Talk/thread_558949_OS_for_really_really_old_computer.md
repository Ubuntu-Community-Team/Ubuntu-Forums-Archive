### Post by Bungo Pony on 2007-09-25
I've got a Pentium 133 in my garage running DSL. It has 36M RAM in it, a 1.2G hard drive, and it runs quite well. You only need 16M of ram to run DSL well. 

Do yourself a favor and put in a PCI Ethernet card. DSL works pretty good with those. It won't auto detect any ISA cards, but they can be manually configured. I got an old ISA Soundblaster card to work with it.

Also, you might want to look into picking up a PCI USB 2.0 card. I stuck one into the Pentium 133 and DSL works extremely well with it. You can use USB flash drives with it, although I find you have to boot up with it installed for it to be detected.

Oh yeah, and don't be afraid to stick a DVD ROM drive into it. I've got two in mine and I'm searching for a third (for $5 or less)

DSL really does bring old PCs back to life. Win95 and Win98 wouldn't be able to make my old Pentium 133 nearly as useful as DSL does.

### Post by dimeotane on 2007-10-15
For a pentium with 64mb ram with a couple of gigs for a HD:

Get the latest Ubuntu "alternate install CD" and choose a command line  system install option.
That'll take up 400mb of your hard drive.
  From there you should:
```
sudo apt-get install xorg xterm

```
That'll take up another 111mb

If you can get your system online with networking then you can install your favourite  lite window manager, like openbox or icewm
