### Post by mattismyname on 2008-01-18
Wrong forum.

But, it's pretty simple:

1) Make sure you're at firmware v1.1.1 (1.1.2 procedure is more difficult.  1.1.1 works fine)
2) Make sure you're connected via Wifi (and not edge)
3) Open the web browser and go to [www.jailbreakme.com](www.jailbreakme.com)
4) Scroll to the bottom of the page and click "Install app snapp"

Your browser will crash and a couple minutes later you'll see an installer progress bar pop up.  Let the install finish and then you'll have a new app on your home screen called "Installer".  Use that to install new programs.  Must-haves are ssh, BSD subsystem, term-vt100, vnsea, apollo, and probably summerboard.
