### Post by molom on 2008-07-18
The Maryan Linux 1 RC1 has been released after long delay due to basing the distribution on Ubuntu 8.04 opposed to beta 1, which was based on 7.10.

For the people who aren't familiar with Maryan Linux. Maryan Linux is an Ubuntu based distribution, using E17 as its default desktop environment providing two other WM's included with the full installation (Fluxbox & PekWM). The next release of Maryan Linux will also provide LXDE as another secondary DE.

The release notes for Maryan Linux are found here - [http://en.maryanlinux.com/content/view/7/1/](http://en.maryanlinux.com/content/view/7/1/)

The Maryan Linux Iso can be found here providing a mirror or a torrent (Make sure to download the RC, not the beta) - [http://en.maryanlinux.com/content/blogsection/3/13/](http://en.maryanlinux.com/content/blogsection/3/13/)

Please test the distro and submit bugs if you have any time at - [https://bugs.launchpad.net/maryan-linux/release-candidate-maryan-linux-1-0](https://bugs.launchpad.net/maryan-linux/release-candidate-maryan-linux-1-0)

Note: The Maryan Linux website is currently under development, the forums will be available soon.

Minimal requirements are:

Estimated Minimal Requirements for LiveCD

    * 100 MHz x86 processor
    * 64 MB of system memory (RAM)
    * Graphics card capable of 1024x768 resolution
    * Sound card
    * A network or Internet connection 

Estimated Minimal Requirements for Full Installation

    * 100 MHz x86 processor
    * 64 MB of system memory (RAM)
    * 6.5 GB of disk space
    * Graphics card capable of 1024x768 resolution
    * Sound card
    * A network or Internet connection 

Recommended Requirements for LiveCD and Full Installation

    * 300 MHz x86 processor
    * 128 MB of system memory (RAM)
    * 6.5 GB of disk space
    * Graphics card capable of 1024x768 resolution
    * Sound card
    * A network or Internet connection

### Post by molom on 2008-08-11
Wow, you guys pick up announcements pretty quick ;)
Yes, we have moved pretty quickly, I guess we learn fast that Debian can pack in the same if not better experience giving better stability and speed ;)

So. Yeah, we are moving to Debian Testing. Why? Check the forum update - [http://en.maryanlinux.com/forum/viewtopic.php?f=7&t=19](http://en.maryanlinux.com/forum/viewtopic.php?f=7&t=19)

We will not try to make an E17 control panel yet like what Elive has done, we find it unnescesary after the E17 devs integrating the gnome utilities into the E17 setting manager. We will be planning to improve modules and make them. We will be working closely with the OpenGEU developers, sharing modules and so on.

We have many things to finish, first we have to finish the 1.0 Ubuntu based release. We then have to create a top quality web experience, fix wiki, web site and so on, meanwhile starting our debian based distro.

Any questions, advice or comments are appreciated.
