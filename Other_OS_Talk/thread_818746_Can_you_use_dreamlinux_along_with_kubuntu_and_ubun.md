### Post by RiceMonster on 2008-06-04
> **N3um0rin said:**
> I know you can switch back in forth between ubuntu,kubuntu or any other ubuntu..But can you put dreamlinux in that list and switch back and forth to it?

Ubuntu and Kubuntu are the same system, just with different desktop environments, so essentially what you're doing is switching between those environments when switching between Ubuntu and Kubuntu. 

So no, you can't do that with Dream Linux. You'll have to dual boot or run it in a virtual machine.
