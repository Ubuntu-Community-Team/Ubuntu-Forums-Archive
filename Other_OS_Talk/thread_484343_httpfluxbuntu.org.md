### Post by zeifertstc on 2007-07-06
So far, Fluxbuntu is a bit less than I had expected, sure. But, if you read the fine print, it says it is based on Dapper currently. It has some dated software but, the project does hope to be using Feisty repos in the near future. However, the near future is not as easily defined at present. 

The main focus of Fluxbuntu is a collection of lightweight, yet powerful, apps to run on minimal hardware. I do hope that they get things working as well as Ubuntu has been as of late. I'm going to continue using this distribution of Fluxbuntu until it breaks. I like it aside from the fact that it uses XDM by default. If XDM were to be modified so as there was a shutdown script, easily accessible, from that screen, I'd like it a bit more.

Aside from that, I agree with the increased use of CLI as it is extremely powerful if you know what you are doing. If you like linux but don't care for CLI, stay away from Fluxbuntu and keep going with (K/X)Ubuntu as they are a bit more user friendly.
