### Post by HermanAB on 2007-07-01
Note that there is almost no difference between various flavours of Linux.  There are some differences between Linux, Solaris and BSD, but again, from a system administrator's point of view, the differences are just large enough to be somewhat annoying.

So, whether you use Redhat, Debian, Ubuntu, Mandriva, Suse or whatnot, doesn't matter from a performance point of view.  The differences are mainly cosmetic and restricted to a handful of system management wizards.  The main difference between distributions are the user forums like this one.  The best user forums are run by Ubuntu and Mandriva, but Fedora is OKish too.

From an ease of wizard use point of view, Ubuntu is on par with Mandriva.  Slightly behind, you'll find Suse and about 1 year behind that you'll find RedHat.

So, as to which Linux you should use?  Pick the one with the friendliest user forums.

Cheers,

Herman
