### Post by RumorsOfWar on 2008-03-28
I have Puppy linux on a pendrive. It works very well, but some computers just don't want to boot off a pendrive. 
If you can't, try booting off a Puppy CD, Install any programs you need, remaster it and burn a new Puppy CD. 
Alternately, you can boot from cd, and store data on the pendrive. 

Yes, Its always good to keep puppy linux on a cd or pendrive for emergencies.
