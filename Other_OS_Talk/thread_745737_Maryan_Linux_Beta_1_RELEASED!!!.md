### Post by molom on 2008-04-04
Maryan Linux is a E17 Ubuntu based distro, which has codecs pre-installed, but will also have a complete propriety free edition as well. Its aim is to be the most user friendly e17 distro there is without the clutter.

For more info - [www.maryanlinux.wordpress.com](www.maryanlinux.wordpress.com)

You can download the Beta 1 using a torrent or a Mirror.

Cheers,
