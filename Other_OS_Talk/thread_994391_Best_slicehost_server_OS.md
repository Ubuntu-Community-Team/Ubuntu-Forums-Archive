### Post by Zack McCool on 2008-11-27
Depends on what you need it for, and what kind of support you want...

I usually use CentOS for my personal servers, as I learned linux with Redhat Enterprise Linux.  It is highly stable, fairly easy to configure, and doesn't install a bunch of extra things you don't want or need.

Of course, for workstations, I stick with Ubuntu...   And I don't have a problem with any of the other distros for a server.  

Best bet, download them and try them out.  See which you like best.  If you need professional support, go with Redhat or Ubuntu.  If you don't, the choice will all be based on what you end up liking best.
