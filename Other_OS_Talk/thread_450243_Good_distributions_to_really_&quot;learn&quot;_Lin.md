### Post by yukonho on 2007-05-21
I thoroughly enjoy Ubuntu, but lately I've been interested in trying to learn more about what's going on "under the hood", if you will.  I know some distros like to tout themselves as being good "for those who really want to learn Linux" (or something to that extent).  I guess I'd like to be forced to use the command line a bit more and learn about editing configuration files, but I'd like the distribution to intend for that approach and have them be accessible/documented if possible.  Case in point: I've been tooling around with a bunch of Live CDs and could never get my wireless to work, but once I learned how to use iwconfig/ifconfig/dhcpcd it became completely trivial no matter what distro I was in.  Having knowledge like that at my disposal is great!

Any recommendations?  Something like Arch comes to mind but I'd like to try out something with a Live CD first if possible before taking the plunge.  I'd still like to have a decent package manager solution.  I've played around with MEPIS, SLAX & Wolvix, Zenwalk, Vector, DSL, Puppy, PCLOS & SAM, DreamLinux, and KateOS.  Some have been more enjoyable than others, but none were really all that bad (making it hard to choose!). Which ones have you folks found to be complex enough for someone who really wants to learn (i.e., not holding your hand all the time), but still usable?

I'm using a T40 Thinkpad, and would favor something on the lighter side.
Thanks! I look forward to what you have to say.

### Post by ThinkBuntu on 2007-05-21
Some distributions **make** you learn, and others don't. In my experience, I've learned just as much using Ubuntu and Sabayon as I have Arch and Zenwalk. Your hands will get as dirty as you want them to be.  :^)

To learn without wasting time doing non-essential tasks, just think of stuff you want to do over a Debian or Ubuntu install (or other distro of your choice). For example, you'll learn a lot by trying to reduce boot times.

Another simple way is to resort to the Terminal whenever possible. Especially file manipulation.
