2003 - A friend installed Mandrake on my old 350 mhz Dell computer running Windows 98.
2004 - I built a PC and installed Mandrake on it. To me, Mandrake was an infuriating OS. I was in RPM hell, trying to deal with Mozilla plugins, NVIDIA drivers, and DVD playback.
2005 - I gave up on Mandrake and moved to PCLinuxOS. This was the first Linux distro that really impressed me. Nvidia drivers, plugins, and codecs worked right out of the box. You could install RPM's through Synaptic, but the selection was limited and I still found myself searching Google for RPM's and dependencies. I then found Ubuntu, and tried installing it along with my new Nvidia 6200. Unfortunately, the card didn't like the open-source nvidia drivers included with Ubuntu. I ended up installing MEPIS, because it supported my new card right out of the box, unlike Ubuntu and PCLinuxOS. I still wanted to install Ubuntu because I loved how GNOME was set up in Ubuntu and I liked how you could get programs like Automatix that set up all of the codecs/plugins for you. So I figured out how to install the Nvidia drivers from the console and made the switch to Ubuntu. I then reached Linux Nirvana, with no more annoying problems to deal with!!!!
2006 -  Upgraded to Ubuntu Dapper Drake.

My timeline:

Returned back to Windows, upgrading to XP, and managing a small network with Win2000 and WinXP computers.

2001 - 2006 Generally get increasingly frustrated with MS software / proprietary software in general. Find that most software is 'crippled' on purpose. Despise the proliferation of proprietary user interfaces for things like printer / peripheral drivers. Do like and use MS Media Center Edition. Novelty of games is wearing off as credibility of console games increases.

2006: Bored with computer. Want to upgrade. Nowing doing quite a bit of web development and PHP programming. Was looking forward to Vista, but it's still not out yet, and reviews of Beta do not make it look attractive at all. Check out Mandrake site again... now Mandriva. Seems 'off'. Check out distrowatch and everyone is talking about Ubuntu. download Dapper to try out... error during install. Re-burn slowly. Try again... works! 

Decide I like Dapper... Windows is due for its bi-annual re-install due to various glitches. So buy a new HDD and re-install both from scrath using a carefully-planned partition layout. Realise that Windows is *a lot* harder to install than Ubuntu. 

Several things still not working -- slow printer; multiple monitors; evolution and OOo too ugly. So still dual boot into Windows. Rarely spend time in Dapper, but its intuitive layout brings me further along the road to 'getting it'. Keep meaning to move Windows to top of boot list but never get round to it.

Fast forward to early this month -- have some time free so decide to make a 'go' of fixing the monitor issue. Upgrade to Feisty. Beginning to feel at home, and decide to dump Windows for good. Precipitated by my general avoidance of Vista, and my anger at all the DRM creeping in and stymying my efforts to create a truly useful general purpose PC. I also found myself mostly using FOSS software on Windows anyway. Windows partition duly erased.

There are many things - user interface and otherwise - that still need improving. But the speed of development on this side of the fence is astonishing. The future timeline is more interesting. Linux has a bright future. Windows doesn't. I actually want to upgrade my computer again to take even more advantage of the OS. I haven't wanted to do that in Windows for years.
