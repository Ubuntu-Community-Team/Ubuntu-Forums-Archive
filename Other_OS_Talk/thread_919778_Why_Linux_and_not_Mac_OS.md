> **silvestros said:**

But we all know that Mac OS is a great operating system

It's an okay operating system, but I think it's inferior to Vista.

> and it's also unix-based, which means no viruses.

Being given a Unix-certification does not mean "no viruses". Certain versions of Windows are certified Unix too, but the reason why those still get viruses is because the Unix-like security system is not mandatory. Mac OS X's Unix-like security system is constantly worked-around by Apple, to the point where it's not mandatory either. 

It's a masterstroke by Apple - by making it so easy to get root access on a Mac, it kills the joy for crackers, so they don't bother targetting the OS again. If you had the keys to the jail, why would you bother digging a tunnel?

> you don't have to spend hours trying to get something working.

Correct, but you don't have to spend hours trying to get something on Linux working. If you buy compatible hardware for Linux, just as you do for the Mac, then everything works out-of-the-box.

At least with Linux, if you make a mistake and buy something with poor support, you can do some hacking around and sometimes get it working. If you make a mistake buying hardware for the Mac OS... you'll have to beg the shopkeeper for store credit.

I love Linux because I can use it on the hardware I want, it provides me with a very good desktop experience in terms of me being able to choose a way to organise my work, and unless I want to tinker it stays out of my way. It's also very fast and easy to set up.
