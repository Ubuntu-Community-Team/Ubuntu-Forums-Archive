### Post by Bungo Pony on 2008-11-20
> how do i keep security updates?

I don't think that DSL offers regular downloadable updates like Ubuntu does. I'm pretty sure they just give you new versions.

> how can i upgrade firefox & install a flash player?

Check the [MyDSL repos]("http://ftp.osuosl.org/pub/damnsmall/mydsl/"). I'm pretty sure there isn't much available that gobbles up a lot of resources, such as Flash. DSL is optimized for older computers, and Flash isn't very friendly when you have less than 128M RAM, no matter what OS you use.

### Post by Bungo Pony on 2008-11-22
You don't need to download the .dsl files. 

If you're using Damn Small Linux v4.x right click anywhere on the desktop. Go to Apps -> Net -> MyDSL Browser. Click on the button that best describes the app you want. If you're connected to the net, you should be able to scroll through a list of software packages. Click on the one you want to install, and follow on-screen instructions. It will automatically download and install.

Easy!
