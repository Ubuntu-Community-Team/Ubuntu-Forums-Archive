### Post by hizaguchi on 2006-09-11
Suse 10.1 was totally hassle-free as far as 64-bit goes, but the package management was severely broken and the system as a whole was sluggish.

Arch 64 is insanely fast and works well, but you have to do everything by hand and you need to compile alot of stuff with pkgbuilds because the 64-bit repos are maintained by like 1 guy.  I stopped using it because some power management stuff failed to compile and I use a laptop.

Edgy 64 is my new favorite.  Everything is working well for me, and it is very fast (not quite like Arch though).  I don't install non-64-bit stuff though, so I can't give you any input on that.  Same deal with proprietary stuff like Flash and Adobe.  But I have had great luck using VLC to watch just about every kind of media you can think of, including WMV, without any 32-bit black magic.
