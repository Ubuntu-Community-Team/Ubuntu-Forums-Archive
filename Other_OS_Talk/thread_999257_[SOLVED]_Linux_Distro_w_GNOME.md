### Post by jayson.rowe on 2008-12-07
Check out Foresight Linux - they have an up-to-date distro (it's a rolling release), they ship basically an upstream GNOME (in fact the GNOME Developers Kit is built on Foresight), the community is AWESOME, and very helpful. 

It has it's own package manager called Conary that is different from apt, but very easy to get used too - you can even roll back updates if something breaks - it's awesome.

It uses the Anaconda installer from Red Hat, so if you've ever installed CentOS or Fedora you'll get through the installer w/ no problems. Also they use a tar based back-end to the installer, and it's probably the fastest installing distro I've ever seen.

Here's the link: [http://www.foresightlinux.org/](http://www.foresightlinux.org/)
