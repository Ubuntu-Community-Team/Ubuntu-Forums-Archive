### Post by elctb on 2007-11-26
I've never used xfce so I can't really comment on it. However, I believe it's a complete desktop environment, not just a window manager. A desktop environment is slower than a pure window manager, but provides a lot of things that make life easier for most people.

A window manager just allows windows to be created and deleted. It doesn't do anything automatically like mounting cds, usb flash drives, etc.

I say try as many as you have time for.

### Post by b9anders on 2007-11-27
some of the suggestions aren't very useful for a beginner. Most will find it more useful to go with a light distro already pre-configured rather than 'stay with ubuntu and spend time setting up this window manager you know nothing about'.

For speed, the short answer is - move away from ubuntu. Imo, xubuntu is one of the worst implementations of xfce I have tried.

Your pc sounds ok powerful to me - I'd say you could run a fullfledged kde or gnome if you pick the right one. For KDE you might want to try sidux which is very fast. I have heard others report some gnome distros (foresight?) being as fast as xfce, but I don't know how usable they are.

That said, it sounds like you want the pc to be truly flying, so maybe you'd be better off looking at even lighter wms, ie XFCE or the like.

My main recommendations would be Zenwalk or Wolvix - two very solid, easy to use and blazing fast xfce distros, owing to their slackware roots.

For a bit more eyecandy (but also a bit slower), there's SAM which is a nice middleground between the need for speed provided by xfce and having the bling of beryl/compiz.

A different kind of eyecandy which is also very fast and easy to use is (unless you're a tinkerer - then you will probably not like it at all) is elive - definitely one of the most beautiful distros out there whilst still being among the fastest, thanks to the delightful e17.

good luck with it. :)

### Post by Rumor on 2007-11-27
If you are looking for lightweight, fast and stable, I'd recommend Antix. [http://antix.mepis.com/index.php/Main_Page](http://antix.mepis.com/index.php/Main_Page) 

Antix is a lightweight version of Mepis, so it is Debian based which makes it quite stable. It is lightweight and uses IceWM / Fluxbox for the window manager. The menus are already configured so you don't really have to do anything, and the network setup is very user friendly (IMO). it is a live CD with an installer in the menus, so you can try it first and see if it is something you'd like. 

I use it on a old laptop (P2/266) and it is quite snappy on that. With your specs, it ought to absolutely fly.
