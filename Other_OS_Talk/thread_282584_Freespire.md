Distribution choice is a matter of preference. Ubuntu uses software that is a bit more up to date than Linspire. The KDE version is 3.3 I think?? However they do use a recent kernel. It does come with all the proprietary stuff already included. That is pretty nice. Their Click N Run is really slick and impressive. I do enjoy it. Click N Run is what makes me think it would be good for my nearly computer illiterate mother. However I believe the first user created has too many admin privleges. There is a lot about Freespire that I like but it seemed to be a little heavier than Ubuntu/Kubuntu. It is also branded a lot. IF you like to customize things and make it your own it is possible but it takes more hacking at it. If you have an extra partition not doing much then install it on there and see if you like it. 

Good Luck!
