### Post by init1 on 2007-07-26
Selling Linux is OK. I don't have a problem with that, but I do have a problem with commercial distros that lie, saying that there distros are better or easier than the free distros. Take a look at this:
[http://www.xandros.com/products/faqs.html](http://www.xandros.com/products/faqs.html)
Click on "I thought Linux is free. Why does Xandros charge for its software?"
> 
It is true that Linux is free. You can download the core Linux operating system through the Internet at no cost. But installing and using Linux can be very difficult, especially for a non-technical user and free Linux distributions are not packaged with the additional software that you will most likely need.

Xandros has built upon the Linux base to develop an operating system that is an open-source alternative to Windows. Anyone familiar with Windows will be able to start using Xandros right away, unlike the free version of Linux. While Xandros does charge for these additions, its Linux base enables it to offer you an alternative to Windows at a significantly lower cost.

And this:
[http://www.linspire.com/linspire_faq.php?faq=linspire_diff_linux](http://www.linspire.com/linspire_faq.php?faq=linspire_diff_linux)
>  	
There are several good Linux operating systems, but these products are much more difficult to install and use than Linspire. Highly technical users looking for an operating system with many installation options and configuration choices enjoy those types of operating systems.


And also this:
[http://www.linspire.com/products_linspire_whatis.php](http://www.linspire.com/products_linspire_whatis.php)

So go ahead, sell you distro, but don't lie to everyone. True, there are some distros that are hard to install, but Ubuntu was not.

### Post by justin whitaker on 2007-07-26
I don't know if it is a lie, per se....it's Marketing. That may be splitting hairs, but there it is. :)

They are competing against other Enterprise level linii, some of which can be a bit arcane in their administration, like Red Hat. 

For Xandros, they are correct in asserting that they include all the stuff a typical user needs: you can play Mp3s out of the box, you have Flash, Java, etc...and the install is dead easy. 

For Linspire, the install is almost unattended, like Vista. It's that simple. So from that sense, they are correct...most Linux installs ask you for far more information or thought, including Ubuntu. 

That doesn't mean either of them are any good...it just means that there is a grain of truth in their marketing. 

And it's an easy sell, because Microsoft is over there portraying Linux as "strictly for the geeks". Their marketing basically writes itself.
