### Post by Sorivenul on 2009-02-08
This thread was inspired by [this post]("http://ubuntuforums.org/showpost.php?p=6699386&postcount=25") from cmay in the thread "[Rename this forum suggestion? BSD, Solaris, Unix?]("http://ubuntuforums.org/showthread.php?t=1051934")". 

Which Unix/Unix-like system to you use? For a brief definition of "Unix-like", see the corresponding Wikipedia entry here:
[http://en.wikipedia.org/wiki/Unix-like](http://en.wikipedia.org/wiki/Unix-like).

For the purpose of this thread and poll, *I have not included Linux*, which sometimes gets lumped into this category. If you are on these forums you probably have used or currently use Linux. I have also not included the Apple OS X; it has its own sub-forum and its users are active and vocal enough on the forums to not warrant the inclusion here. 

The purpose of this thread is to gather a rough distribution of how many people use other Unix-like systems. The purpose of this thread is *not* to start a war regarding which system is more "Unix"; being certified under the Single Unix Specification (SUS) alone does not define a "Unix" system for many users. 

I have grouped many similar systems together for the purpose of brevity and simplicity. If you use more than one, please choose the one you use more frequently.

I use FreeBSD, but spend a large amount of time testing the other BSD systems as well as Solaris and OpenSolaris.
