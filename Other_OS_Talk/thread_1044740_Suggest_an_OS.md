### Post by GammaRay256 on 2009-01-19
**LinuxMint** - [http://linuxmint.com/](http://linuxmint.com/)
It's closely based on Ubuntu, fully compatible; but has some of its own management tools and features.  It's my favorite desktop distribution.  It has multimedia codecs, DVD playback, and flash preinstalled for convenience.  Has a nice clean look too.  It's a smaller community, and features can be added or updated quickly.  (See their "[About]("http://www.linuxmint.com/about.php")" page)
_Difficulty:_ Easy; _Open-source:_ Good (except for media codecs); _Features:_ Very Good

----------

I have tried various distros in Virtual Machines over the past few weeks.  Here are my first impressions on some of them:

**Fedora** - [http://fedoraproject.org/](http://fedoraproject.org/) - Looks very nice, has an impressive default look.  File manager looks extremely simple though.  Otherwise, it has many of the same apps and features as Ubuntu.
_Difficulty:_ Easy; _Open-Source_: Very Good; _Features:_ Very Good; _Performance:_ Good 

**Slackware** - [http://www.slackware.com/](http://www.slackware.com/) - Has a great traditional Linux feel to it, not too many distro specific changes.  Somewhat difficult to set up (a little more than the alternate cd for Ubuntu).  Would be perfect for servers.  Very flexible, but a bit too difficult for the average user.
_Difficulty:_ Medium-Hard; _Open-Source:_ Very Good; _Features:_ Good; _Performance:_ Increadible

**OpenSUSE** - [http://www.opensuse.org/en/](http://www.opensuse.org/en/) - Looks polished, has some bugs though.  The installer was nice because it gave a lot of different choices.  It was also nice to have the most common apps all on the install DVD.
_Difficult:_ Easy-Medium; _Open-Source:_ Good; _Features:_ Very Good; _Performance:_ decent, a bit bloated with software

**Mandriva** - [http://www.mandriva.com/](http://www.mandriva.com/) - Looked prety nice; but I felt like it was pushing its commercial version a bit too much.  *But we should be happy that companies like Novell, RedHat, and Mandriva are sponsoring open-source distributions even though they sell commercial versions.*
_Difficulty:_ Easy; _Open-Source:_ OK (has some advertising about the commercial version); _Features:_ not too sure, I didn't use it long enough; _Performance:_ seemed about the same as Fedora

Just for comparison, here is my impression of Ubuntu vs the others
**Ubuntu **- Great general purpose OS, packaged software interacts well and is very consitant.  Might be a little difficult to enable DVD playback.  Other than the fact that it takes a little longer to set up for general desktop use, it is a great OS.
_Diffculty:_ Easy; _Open-Source:_ Great; _Features:_ Good; _Performance:_ Good

----------

If you're going to experiment with different distros, instead of installing them on your hard drive like a normal OS, I **highly reccomend** that you install them inside VirtualBox ([http://www.virtualbox.org/](http://www.virtualbox.org/)).

### Post by cardinals_fan on 2009-01-20
> **DeadRobot said:**
> Yeah i had trouble setting up my wireless with SliTaz today. What do I download to get it working and where do I get it from?
Manually download linux-crypto, linux-wireless, and wireless_tools from here.  You may need additional drivers; post your wireless card and I can try to help.  Copy the packages via usb stick (or just mount your partition from withing SliTaz) and install them with "tazpkg install".

Once the packages are installed, you must load the proper kernel module with "modprobe".  If you don't know what the module is, post your hardware info and I'll help.  You can then configure the card with CLI tools such as ifconfig and iwconfig or with the Netbox GUI.

Is this as automated as Ubuntu?  No.  But you should learn something :)
