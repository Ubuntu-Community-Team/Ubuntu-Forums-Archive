### Post by MethodOne on 2008-10-23
DSL's goals are to support old hardware and keep the distro within 50MB.  Don't expect the latest versions of the included apps or for it to work on newer hardware because of the old 2.4 series kernel.  They don't plan on going to a 2.6 series kernel anytime soon.  I recommend you keep on using Puppy if you have newer hardware.

### Post by Bungo Pony on 2008-10-23
I love DSL (v3.x). It makes my old crap computers run beautifully.

I really didn't care for Puppy. I really do enjoy having an OS that has a password for root access, and Puppy doesn't do that. I also had one heck of a time trying to figure out how to mount drives the first time.

Puppy also doesn't run on as old hardware as DSL will. DSL requires only 16M to run while Puppy requires somewhere around 128M.

Oh yeah, and just to top it all off, I think Puppy Linux is ugly. The only thing I liked about Puppy Linux is it barks when it boots. But I'll be damned if I use a Linux distro solely because it barks.
