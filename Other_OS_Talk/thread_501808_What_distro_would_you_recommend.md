### Post by bonzodog on 2007-07-16
The Gnome desktop and KDE desktops are memory hungry. Ubuntu in it's default form also contains a lot of unnecessary Stuff.

I would go for a lightweight desktop or window manager, such as Xfce, Fluxbox, or Openbox (I run openbox myself. It's sweet, *very* fast, and highly customisable).

Maybe think about a distro that doesn't install everything but the kitchen sink out of the box, but does a functional install.

I am going to point at my own choice here, Zenwalk Linux, which uses Xfce out of the box, and is used as the 'poster child' for the Xfce Desktop. The ISO is only 400MB, and we follow a one app per purpose philosophy.

It's is the Slackware worlds answer to ubuntu, more or less, and it has a sizeable fan club. Pidgin is included with the current stable release 4.6.1, as is Firefox 2.0.0.4,, xorg 7.2, kernel 2.6.21, and Xfce 4.4.1. 

We have a package manager called netpkg that does dependency checking, but the repos are a lot smaller than ubuntus, and contain the most popular packages.

[http://www.zenwalk.org](http://www.zenwalk.org)
