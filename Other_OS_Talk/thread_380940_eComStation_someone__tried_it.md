### Post by ADH on 2007-06-08
I use eComStation (currently version 1.2R - 2.0 should be available soon.)  eCS is an OEM version of IBM's OS/2 Warp 4.52, with an improved installation and some new drivers and utilities.  The website is [www.ecomstation.com](www.ecomstation.com).  I've used OS/2 since version 2.11, and chose eCS when IBM support and driver and kernel updates changed to require subscription to something called Passport Advantage - eComStation came with that support, plus the improvements Serenity Systems added/adds, and it cost less than Passport Advantage.  IBM no longer supports OS/2 at all, after years of mishandling it, but eCS is still being worked on.  It will always be based on Warp 4.52, but it remains more secure than Windows, and more stable, and still handles all the Dos and Windows 3.11 apps OS/2 handled (I still use a bunch of those.).
