### Post by luisromangz on 2007-07-15
Well, FreeDos is compatible with MSDOS programs, not Windows ones. And using virtual machines for gaming doesn't do the trick, as virtualized video cards doesn't offer real 3d acceleration (yet) to the guest OS, so games would work so slow.

If you want to play Windows games in Linux, and they don't work using Wine, you should go for the dual boot option.
