### Post by coolaj86 on 2008-07-05
It depends on which version of the card you have.

[http://www.linuxquestions.org/hcl/showcat.php/cat/144](http://www.linuxquestions.org/hcl/showcat.php/cat/144)
[http://forums.fedoraforum.org/showthread.php?t=142665](http://forums.fedoraforum.org/showthread.php?t=142665)
[http://ubuntuforums.org/printthread.php?t=225206](http://ubuntuforums.org/printthread.php?t=225206)

ndiswrapper (it uses the windows driver and fakes it into being a linux driver) is probably going to be your best solution from my brief google. See what others have said and post back if you still have problems.

BTW, whenever you have driver questions the first and best place to look is probably LinuxQuestions.org/hcl (hardware compatibility list). Back before Ubuntu and Fedora Forums LQ was the best forum on the net for Linux and it still has some perks.
