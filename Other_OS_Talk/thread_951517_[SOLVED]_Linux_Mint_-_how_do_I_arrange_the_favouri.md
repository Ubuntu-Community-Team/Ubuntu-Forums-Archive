### Post by Rytron on 2008-10-19
I found that if you right click on the mintMenu button(the one with the name Elyssa) and choose Preferences, you can arrange the favourites in a single column(easier) and then put them in a 2 or even 3 column layout.

There is also an option to use original names, so Image Editor will then show up as GIMP.
