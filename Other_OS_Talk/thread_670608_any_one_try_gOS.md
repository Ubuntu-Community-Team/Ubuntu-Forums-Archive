### Post by p_quarles on 2008-01-17
I tried gOS in a VM, and wasn't that impressed. Don't get me wrong: there's nothing wrong with it, and it uses an E17 desktop, so it's very customizable. But the basic setup really is all about making things easy for novice users who don't want to install an OS on their own. If you're looking for something that doesn't require much configuration, but are still willing to install it yourself, go with Mint.
