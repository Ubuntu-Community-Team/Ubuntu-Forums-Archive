### Post by voided3 on 2007-08-09
I use puppy or DSL on my "vintage" computers. Puppy is stupidly fast on anything more than 500mhz and runs totally in RAM is you have 128mb or more (ran it on a 633mhz celeron w/192mb of ram and it was VERY fast). Puppy also is very easy to get up and running; it has flash player pre installed now and wifi/LAN setup is incredibly streamlined. I currently use it on a 800mhz PIII computer w/512mb of RAM and an MX420 that I built into a briefcase and dual boot with Win98 for those older games (and they never have ran faster!). 

DSL lives up to its name; only really needs 24-32mb of ram to be smooth for IMing and web browsing (I used it to power a 200mhz Pentium one Compaq w/ 72mb). The included apps are all very small and most OS config apps run in a terminal shell instead of being GUI, but it still is reasonably easy to use, especially considering what you can do with it.
