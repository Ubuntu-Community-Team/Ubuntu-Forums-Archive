### Post by tgalati4 on 2008-12-02
I recently installed both Fedora10 and opensuse11.1beta.  I have to say that the Fedora10 graphical boot (AKA Plymouth) is seriously cool, although it only lasts 30 seconds on my machine before it goes to X and then the login screen.  It's supposed to reduce annoying graphical mode switching, but it still switches to a different resolution for the graphical login manager.  I'm still trying to figure that one out.

It took about 2 hours to get Fedora10 set up the way I wanted--with similar functionality to Linux Mint.  Only locked up once--when logging out (not shutting down) which was strange.

Opensuse11.1beta uses KDE 4.1.3 which has some problems.  It's usable, but a few KDE apps don't work the way they are supposed to.  Kmix doesn't come up to adjust volume.  Again it took about 2 hours to add all the useful desktop stuff that comes working out-of-the-box with Linux Mint.  There are useful tutorials for both Fedora and Suse, but you have to search for them.  

For instance, it does not help for a media player to suggest a missing codec, look for it, not find it.  You have to add 3rd party repositories, security keys, update the repos, install the codecs, then restart the media player.  Hardly plug-n-play.

But if you want something different:  Fedora10 and OpenSuse11.1 are certainly worth hacking on a spare partition.

If you are using an older GRUB boot loader (Gutsy and below), you may have better luck installing Grub only to the installation partition (not the Master Boot Record) then use in your grub.conf or menu.lst file:

root (hd1,0)  (or wherever you put your new distro)
chainloader +1
boot
