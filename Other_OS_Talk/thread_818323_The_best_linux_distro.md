I guess this subject will come up many times on these forums.

### Post by handy on 2008-06-04
I haven't read the thread, so I'm just answering the OP title:

There is no such thing as a best linux distro' (for everybody): 

We all have different desires/needs so what is one persons gold is another's poison.  

The only way to truly find out what suits you is to go to [***_DistroWatch_***]("http://distrowatch.com/"), download & try; many are on liveCD's which makes it easy, others like Gentoo & Arch (my personal favorite) can not work on liveCD by design, so you don't get a complete picture by using liveCD's, but it will help you understand what you do like in an OS & what you don't.
