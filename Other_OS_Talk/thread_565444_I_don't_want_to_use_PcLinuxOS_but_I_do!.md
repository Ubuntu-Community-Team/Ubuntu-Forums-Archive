### Post by igknighted on 2007-10-02
Actually, commercialization is probably the last of your concerns with PCLOS.  I might look for a better supported distribution, however.  You might look into Mandriva (the same control center, but newer packages and better support...). The base distro is always free, and you can buy the "PowerPack" if you want commercial software as well.  Another good choice would be OpenSUSE.  They only distribute 100% F/OSS software, so anything non-free will need to be added on.  There is a one-click package install feature, as well as a wonderful control center as well (yast).  Both of these distro's ship new versions this week (suse due thursday, mandriva due friday), so you might want to check them out.

You might also look at Mepis.  It is based on Debian so it will be similar to Ubuntu (tho it uses KDE like PCLOS), it has most non-free stuff included by default (codecs, java, etc.), and should be different enough that whatever Ubuntu bugs you are running into are likely not an issue.
