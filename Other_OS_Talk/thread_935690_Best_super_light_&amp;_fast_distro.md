You're never going to get close to the 5-second boot time of a rom-based OS. Hi Bart, it is funny you ask that question. :) I am running a dual boot of Crunchbang and SliTaz on both of my computers. I have not timed the boot on my faster desktop yet. On my old slow laptop, Crunchbang takes 2:45 to boot and SliTaz takes 0:55. So maybe 50Words will be able to break that elusive 30-second mark, who knows?

I would like to use Slitaz a while longer (it's only been a couple of weeks) before I make a full comparison, but here are some initial thoughts. First of all, SliTaz has a stable version 1.0 and an unstable "cooking" version, which will become SliTaz 2.0 sometime in spring 2009. I am using the cooking version, so yes, I have encountered various bugs and issues, as one would expect. Crunchbang has been very, very stable for me, no complaints.

Installing Flash in SliTaz is quite easy. Open a terminal and type 'get-flash-plugin'. Works fine for Youtube but I haven't tested it for much else.

The #1 reason I have kept Crunchbang as a double boot, instead of switching to SliTaz full time, is that SliTaz simply doesn't have the vast Ubuntu repositories. Its package manager (tazpkg) is the best I've ever used, but you won't find openoffice, vlc, wicd, audacity, etc. in its repositories yet. I don't have the Linux-fu yet to install these things from scratch (would like to learn though) so I use SliTaz for daily internet use and Linux self-education, and use Crunchbang for intensive office and multimedia tasks. I also carry around a SliTaz USB stick for when I'm using someone else's computer. It's great for that purpose because it fits on a 2gb stick with 1.9gb free for documents.

That's a long-winded way of saying, keep using Crunchbang because it's awesome! SliTaz is awesome in a different way. I think the common element that makes them both awesome is Openbox. It's my new favorite windows manager.

ps for those who care about what is essentially a meaningless indicator of performance, when I boot up Crunchbang on my 256mb laptop, it uses about 64mb of ram, while SliTaz uses about 30mb. I have never, ever seen SliTaz use any swap whatsoever.
