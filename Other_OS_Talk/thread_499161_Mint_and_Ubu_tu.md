After reading the posts yesterday about LinuxMint, I decided to try it out. (The posts are still on this page as I type, and I haven't read them since yesterday, so I do not know what people are saying)

I tried it live, as I usually do.

My overall opinion of it is that I prefer Ubuntu.

For:

+
* Debian based, and easy for Ubuntu user to use
* Seems just as stable as Ubuntu

-
* Vista-like start menu
* a bunch of apps that if I want, I'll install myself! (nothing wrong with having them on the disk, just don't install it, especially Beryl)

Neutral: It is similar to Ubuntu that if it has what you want, it will be good.
