### Post by mips on 2007-07-23
> **LaRoza said:**
> What is the difference between the DVD and CD versions? Is the CD suitable for an installed system?

The standard version is the DVD version which has everything and I mean everyhting and the kitchen sink on the DVD. The new installer will allow you to select package groups if you don't want to install everything. The CD version is the mini edition which comes with Sabayon and all the essentials to get you into a working system, it has all the stuff like compiz, koffice etc but lacks all the apps of the dvd version. Then there is also a business edition that is a smaller version of the dvd with lots of stuff stripped out like games etc and it only uses STABLE packages and has no compiz etc.

You are really spoiled for choice here. My personal preference is usually the mini edition CD. After the install I just add a few applications I want. With 3.4 I think I'm going to download both the CD and the DVD. I already have the business edition sho I might as well try them all :)

You also have the choice of 32 or 64 bit versions. I use the 64bit versions as they are hasslefree unlike some other distros trying to do 64bit.

But wait for the official 3.4 to be released.
