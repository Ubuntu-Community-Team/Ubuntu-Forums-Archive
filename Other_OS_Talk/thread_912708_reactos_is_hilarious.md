### Post by smoker on 2008-09-07
ReactOS isn't a clone or an emulator, it is a wholely new operating system, but based on the design of windows and aims to run applications that run on windows:
"The ultimate goal of ReactOS is to allow you to remove Windows® and install ReactOS without the end user noticing the change."
[http://www.reactos.org/en/index.html](http://www.reactos.org/en/index.html)

ReactOS has been in development for 10 years, and is still in the alpha stage and not recommended for everyday use, and this is stated clearly on their website. i'm sure the devs would welcome any feedback about bugs or such: [http://www.reactos.org/bugzilla/](http://www.reactos.org/bugzilla/)

As to ReactOS being pointless, i'm sure as time goes by and improvements are made, more and more people will think otherwise, and i'm sure a lot of the devs learn through the process of development that makes the project more meaningful to them. i also believe that some of the devs on the project also work on Wine, so indirectly, improvements in Wine could be due to the knowledge and dedication of the devs that work on ReactOS.

imo, it is an admirable project, and i look forward to them reaching their goal.
