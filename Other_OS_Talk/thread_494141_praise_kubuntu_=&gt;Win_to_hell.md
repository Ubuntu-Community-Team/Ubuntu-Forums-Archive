### Post by Jense on 2007-07-06
This is just to vent some Win-Frustration. I switched to kubuntu about 3 month back and must admit, there where times I was frustrated with it as well. But after a while I got the hang of it. Not that I am anything like an expert. Still some things (skype) don't work as I would like them to :( . But the essentials where there, right from the start. Nice browser, Office, ktorrent, etc..
Now, back to WinXp for a friend of mine. I have some years of painfull Win experience, but a pain like this, never. This mashine uses an sata controller. It took me a while to figure out i needed to feed him some extra drivers during installation, just so win would recognize the harddrive. I was lucky both mashines still have floppys . :o Must have been the first time in ages using it. So I made some partitions and thought it would go smoothly from here on. But then he wouldn't find loads of files on the winCd. So after a new Bios and loads of trails&errors, I got the idea of using an older driver for the sata-controller, which .... worked. So I thought maby from here on... but know! First reboot, an error saying there was some great grand problem with the system date not being consistent. So google, and after that going into bios and resetting systemdate to march 2002, can you believe it? It's like the milks gone bad years before I bought it... Sometime during install I thought maybe the cd is kaputt, so I tried another one... some Manifest file didn't like that much. pfuuhhh.Then it was finally there. ahhhhhhhh. But no network, no generic driver for this mainboard, arrghhhh. I do wonder how you are supposed to make any winxp install without another computer right next to you with a working internetconnection. Quiet impossible.
So after about 10 h straight on this install I am really looking forward to the next linux installation. 
Many thx to all the competent advisors I found in this forum and to the people who make such a great os free. I am loving it the more I am using it.
