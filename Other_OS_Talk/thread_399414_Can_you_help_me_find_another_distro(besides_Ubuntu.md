### Post by Majorix on 2007-04-02
If you don't have wireless then go for Debian Etch. I have been using that for quite a while and I must say it is really stable and serves multi-purpose. Like you can use it as a gaming machine or in contrast as a server.

Slackware is good too, but if you have Logitech mouse you have to update the kernel and it is usually not that easy. But if you say you are intermediate, then go for it.

Also I used to use Pardus and I liked it somehow. But it has its own package installation system (PiSi) and it's quite hard to find those packages. But other than that it is the best KDE-using system after Slackware.

Finally, Fedora Core is OK too. Not a big fan of it though.

My ratings to summarize are:
Debian Etch (4.0 Testing): 9/10
Slackware 11.0 8.5/10
Pardus 2007.1 7.5/10
Fedora Core 6 5.5/10
