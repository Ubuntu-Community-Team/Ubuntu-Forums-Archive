### Post by igknighted on 2007-07-15
I'm just trying Pardus right now... It's really nice.

For someone new to linux who wants to cover all the bases, I would strongly recommend you try a non-debian based distro next (you are already using one).  Try Fedora, Mandriva, or OpenSuse next.  Slackware/Zenwalk might also be good choices if you want a little challenge.  This is not to say that any of those other distros are better than Debian, but you should get a broader experience.  There is much more to linux than what Debian offers, so try it out.  Debian is really nice, so theres a good chance you might come back to it, but play the field a bit first.

### Post by ezsit on 2007-07-19
How about Berry Linux - [http://berry.sourceforge.jp/](http://berry.sourceforge.jp/)

From Distrowatch - 
Berry Linux is a bootable CD Linux with automatic hardware detection and support for many graphics cards, sound cards, SCSI and USB devices and other peripherals. Berry Linux can be used as a Linux demo, educational CD or as a rescue system. It is not necessary to install anything on a hard disk, although this option is also available (it needs 1.2GB of hard disk space). Berry Linux is based on Red Hat Linux and Knoppix.

At the moment, Berry Linux version 0.82 is based on Fedora 7 with a great deal of customization.
