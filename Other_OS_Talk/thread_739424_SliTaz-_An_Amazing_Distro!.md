### Post by cardinals_fan on 2008-04-01
I've tried it, and here are some thoughts:

Hardware Detection: Awesome!  It configured my screen resolution perfectly, which no other distro has done.  No RT73 support, but I didn't really expect it.

Installed Apps: Pretty good for 25 MB.  I like that they included Geany.  The file manager annoyed me and included options for programs that weren't installed.  Just needs a little polishing.

Stability: Acceptable, though the not-installed app links in the file manager was bad and the top panel (something-with-a-z panel?) crashed once.

Package Management: Didn't try.  Just running as Live CD.

Speed: Superb.  SliTaz flew.  True, this is with 1 gig of RAM, but it loaded into memory perfectly and didn't lag at all.

Boot Speed: Not bad.  It didn't fly (extraction?) but wasn't that much slower than Puppy.

Look: Generally, I don't care about a distro's 'look and feel'.  After all, it can be customized.  With a live CD, however, the default will boot up each time you use it (unless you remaster a custom version), so it should be nice.  And with SliTaz, it is.  A graceful and clean wallpaper greets the eye, and things look nice.  It doesn't have any fancy stuff (come on, this is 25 megs) but it includes drop-shadows and fading animations (on open/close) that worked **out of the box**.  Who knew that JWM could actually be pretty?

With a little more polishing, SliTaz could be one of the best Live CDs on the market!
