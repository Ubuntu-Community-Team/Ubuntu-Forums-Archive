### Post by chang47 on 2007-10-30
Just like to share my experience with scim on Fluxbuntu 7.10.  I installed scim using Synaptic, everything looked fine until you wanted to use it.  Scim showed on sys tray, but the hot keys wouldn't work.  This was an issue for some users on other Buntu 7.10. Googling got me to [http://www.mrbass.org/linux/ubuntu/scim/](http://www.mrbass.org/linux/ubuntu/scim/).  By reinstalling scim following the instructions there, I was able to get scim working.

Note that when you remove scim language support will be removed at the same time, so before you install following instruction at the site, re-install the language-support code first.  I assume you know how to do that in the first place.

Hope this helps someone.
