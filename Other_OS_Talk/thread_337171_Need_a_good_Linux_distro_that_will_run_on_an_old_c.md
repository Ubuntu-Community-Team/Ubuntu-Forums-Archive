If you use the Dapper 6.06 LTS version then I would say 'yes'. The underlying OS is the same as any dapper version.

[http://wiki.fluxbuntu.org/index.php?title=Main_Page#Is_Fluxbuntu_stable](http://wiki.fluxbuntu.org/index.php?title=Main_Page#Is_Fluxbuntu_stable)
> **Is Fluxbuntu stable**

 Yes, fluxbuntu is built on Dapper 6.06 LTS and is thus a stable operating system. It is ***nbuild1-rev2*** because the final packages have not yet been determined and is not intended to infer instability of the OS itself.[LIST]
[*]The Fluxbuntu look-and-feel and default package selection are under development not the OS.[/LIST]I however cannot comment on the development release of Feisty 7.04.

Just make sure you have the version that suites your needs. Alternatively do the server install thing and add it manually.
