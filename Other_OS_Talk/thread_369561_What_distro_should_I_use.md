### Post by Jessehk on 2007-02-25
If you somewhat know what you're doing, I'd quickly recommend Arch Linux ( [http://archlinux.org](http://archlinux.org) ). It's a fantastic distribution that has frequent updates and a rolling release system that means you never have to reinstall to the newest version (like Gentoo).

Its package manager, pacman, works like apt-get and it has the ABS build system ***IF*** you want to compile everything.

### Post by tommcd on 2007-02-27
If you like to be up to date you might try zenwalk.
[http://zenwalk.org/](http://zenwalk.org/) It flies on the cutting edge (2.6.20 kernel, xorg 7.2, all apps up to date). It is very light and fast too. The only potential downside for you might be that their netpkg repository has a limited amount of software. Their philosophy is "one app per task". But you can use slackware packages from these places also:
[http://www.linuxpackages.net/](http://www.linuxpackages.net/)
[http://www.slacky.it/](http://www.slacky.it/)
Zenwalk has a small, but friendly and helpful community, and the developers hang out in the forums with the rest of us.
