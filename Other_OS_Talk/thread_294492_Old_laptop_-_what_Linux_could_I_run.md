I figured this would be the best place for my question.  I am in the process of using Ubuntu on my desktop and love it so far...even though it's taking a while to get used to.

### Post by K.Mandla on 2006-11-06
DSL and Puppy Linux will run fine. [You can put Ubuntu on it]("http://www.ubuntuforums.org/showthread.php?t=294292"), but it's almost more work than it's worth, and probably won't perform very well. It is possible though. 

[DSL]("http://www.damnsmalllinux.org") is perfect for something that old, in my opinion. The entire distro is only 50Mb, which means you won't need to read a whole CD, and it comes with a pdf reader, Firefox, word-processor-type-program ... and so forth.
