### Post by nerdman978 on 2008-04-27
Hello,

My Windows XP desktop refuses to connect to my freshly set up wireless network. I have WEP enabled (apparently my adapter doesn't support WPA or WPA2) but whenever I click "connect" to my network from the the view available wireless networks window, a window will pop up say "detecting network type" and less than a minute later will either say the network is out of range (it isn't, my laptop is sitting right next to it and is connected) or merely does nothing and returns me to the original window. Needless to say I am moderately pissed off :confused:. 

Any help is greatly appreciated, and thanks in advance.

### Post by smoker on 2008-04-27
just a suggestion, go into your modem settings and ensure that it will accept wep enabled devices, sometimes this is an option, also, you may be able to change the channel, usually from 1 to 12, try your modem on another channel, see if that helps, also, maybe check that the windows zero wireless config service is running, if you are using windows to connect, rather than some other software.

best of luck

---

### Post by nerdman978 on 2008-04-27
oh. duh! I made a newbie mistake. I realized my router was only broadcasting 802.11g, and my adapter only supports b! So I set the router to broadcast both signals, and now it works perfectly. I thought WEP was the problem when it was actually the router!
