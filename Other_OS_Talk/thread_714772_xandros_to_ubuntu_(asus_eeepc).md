### Post by Tomatz on 2008-03-04
Im running xubuntu with compiz fusion on my eeepc, runs a treat! Xubuntu is a version of ubuntu but instead of running the gnome desktop it uses xfce (which has a much smaller memory footprint)

get xubuntu:

[http://www.xubuntu.org/](http://www.xubuntu.org/)

ubuntu drivers for eeepc with great howto (you will need these wifi, graphics and more):

[http://ubuntu-eee.tuxfamily.org/index.php5?title=How_to_use_the_ubuntu-eee_script](http://ubuntu-eee.tuxfamily.org/index.php5?title=How_to_use_the_ubuntu-eee_script)

### Post by Tomatz on 2008-03-06

True but i didn't like it.  The wifi drivers were buggy and it was a bit of a large install for the 4GB ssd on my eee.  Personally I did a network install (booting from a flash drive).
