I can answer this one; Because sometimes you want to be able to choose or work in a different DE{desktop environment} Also, xfce uses some gtk apps or themes present in gnome, but it operates faster than gnome. Some folks can customize their xfce desktop to resemble gnome desktop while being faster as it is lighter. As an addition, try adding kdemod on your arch install and you would see kde in its full power and beauty. Arch's kdemod was the biggest attractant of me to arch at first. It is super polished and nice with its domino implementation.

I am in agreement with other folks here.
Arch is my first choice for this. Then also you have Zenwalk, Vectorlinux which are easier to install and use. They are equally functional as Arch and almost as fast {this may be one of mere perception}

Arch unlike zenwalk or vectorlinux does not come with a GUI package manager.They may already have one, but most folks there tend to prefer the CLI like I do as well. I used to be more about synaptic, but now I love pacman.


Other non-debian-based, light, fast, good distros besides these include kateos, frugalware, slax.

### Post by crimesaucer on 2008-01-12
> **b9anders said:**
> Why install gnome if you intend to use xfce?

Well, I linked a simple guide for beginner installing that use gnome... and adding xfce4 afterward from the Arch wiki takes about 1 minute.

Yes, I assume that you could just ignore gnome all together and just install ```
pacman -Sy xfce4 xfce4-goodies gtk2-themes-collection esd
``` instead of ```
pacman  -Sy  gnome  gnome-extra gnome-system-tools
```

...There are pages in the wiki that have detailed instructions and I know there a lot of people in the Arch forums that run openbox/fluxbox and don't want gnome at all... so I'm sure if you searched around for a few minutes that you would find the correct way of installing a gnome-less Archlinux using only xfce4.

I just like to have gnome as a backup, I also like some of the panel apps and use them on my xfce4 panel, and I like messing with the gnome-svn for metacity to see how it's improving.

I'm still a beginner at Archlinux also so I just follow the easiest guides (the one I linked above) for installing, and then customize afterward.
