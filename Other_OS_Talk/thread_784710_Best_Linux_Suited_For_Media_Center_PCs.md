### Post by duckville on 2008-05-30
So that you know exactly what's going on why not use either:
* Debian net install [http://www.debian.org/CD/netinst/](http://www.debian.org/CD/netinst/)
* Ubuntu minimal install [https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)

Use one of these then build on it, installing only what's needed.
use a light window manager like openbox: [http://icculus.org/openbox/index.php/Main_Page](http://icculus.org/openbox/index.php/Main_Page)

Then have mythtv load at boot, controlled by your USB MCE remote, job done.
