### Post by ajgreeny on 2007-12-03
In the options dialogue you can also set the whole suite to save documents as .doc and spreadsheets as .xls by default.  If you are always needing to work and exchange things with others who use MS Office, this can be worth doing.  If you only need to send things occasionally, it's probably easier to keep to the ODF formats.

### Post by Luggy on 2007-12-07
One of the problems you may notice if you save OpenOffice documents as .doc is formatting issues.

You might create a document but all your teammates may have to spend time formatting the document after.

You could get your teammates to use the Windows version of OpenOffice but that might cause other issues if they were to pass documents around to other people in the office.


What you could also do is install [CrossOver Office]("http://www.codeweavers.com/"). It will allow you to install and run Microsoft Office applications on your Linux machine. This way you can use the office program the rest of your team uses.
