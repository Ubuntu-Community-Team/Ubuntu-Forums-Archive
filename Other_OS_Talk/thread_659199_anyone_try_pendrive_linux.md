### Post by wolfen69 on 2008-01-05
i decided to go to [www.pendrivelinux.com](www.pendrivelinux.com) and give it go, since i heard how cool it is to have a personal desktop to go.

freaking awesome!

i put the "Pendrivelinux Distro" (debian etch tweaked nicely/comes with Envy) on a 4gb pendrive. this is too cool for words. i can now bring my "computer" with me in my pocket. it saves all settings and files too.(persistance) i now have it tweaked just like my regular desktop computer. 

i also put linux mint on a 1gb drive. but it doesnt support persistance mode. (that i know of) but nice anyway. it acts like a live cd.

think about giving it a try, it's awesome.

one thing though, it took me 3 hours to figure out that maybe fdisk isnt the best method of setting up the drives.(maybe it was just me) i wound up using gparted instead. just remember to umount /dev/sdxx(your pendrive) before using gparted.

check it out. 

if anyone has any troubles, (not the easiest thing to do) post here and me or someone else will help. have fun.

btw, i did the majority of the work from a windows machine. much easier this way.  to get persistance mode for debian, you configure it after you boot it for the first time.
