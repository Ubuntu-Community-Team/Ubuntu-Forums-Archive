### Post by 67GTA on 2008-11-26
Mepis supports most multimedia out of the box. You still have to add the Debian multimedia repo to get everything(libdvdcss). Mandriva, Arch and Opensuse do not. You have to add repos after install to get multimedia support. IMHO, Opensuse is the best KDE, and the best desktop experience around. Starting with version 10.3, you can install most of what you need with the "one-click" install feature without chasing repos. When you click on the button, Yast will add the repo, and install whatever it is for you. There is a slight learning curve if your used to Ubuntu/Debian, but not much of one. Most of what you have to do in a terminal in Ubuntu/Debian can be done via GUI in Yast.
