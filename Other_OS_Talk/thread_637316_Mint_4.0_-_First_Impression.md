### Post by diaz028 on 2007-12-10
Awesome. Mint is what I would call it too! 

Dont get me wrong, Gutsy is awesome. The only thing that frustrated me were a few issues that I couldn't seem to resolve, no matter how hard I tried. I had problems with flash and compiz. Not to mention GDM, Emerald and all themed things. Envy wouldnt install video drivers, neither did automatix. I had to manually install video drivers, I'm sure my vid drivers had something to do with my issues (5 fresh installs + updates)... Updates would also cause things to not function anymore. These problems are all gone with Mint.

Anyways, I find mint to be up to par with my everyday train, although I did like Gutsy version very much, I find Mint just picks up where Gutsy left off, with a well oriented (still not perfect) change in the panels, and some cool Minty software that is more intuitive IMO: Being a windows brat since 3.1, I find this desktop environment to be a tad more familiar.

I find this distro to be very friendly (if not more than Gutsy) for windows users migrating to Linux. I would definitely suggest it to my friends.

Now, on to testing Hardy 8.04a ... :popcorn:

-D
