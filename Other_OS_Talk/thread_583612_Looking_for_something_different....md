### Post by miggols99 on 2007-10-20
Arch is very good if you want to 'learn' Linux. It's installer is text based and starts very minimal. It's not as hard as Gentoo and the config files are very easy. It is also much faster than Ubuntu, Suse or Mandriva. Pacman, it's package manager is very fast, and there are lots of packages in the repos and if not probably in the AUR (Arch User Repositories). There is also a special KDE called KDEmod which is a modular KDE with patches and bugfixes. If you like KDE you should try it :)
