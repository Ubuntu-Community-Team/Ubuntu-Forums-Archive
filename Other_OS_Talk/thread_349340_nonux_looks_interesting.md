### Post by RAV TUX on 2007-01-30
[> ]("http://distrowatch.com/nonux")[[IMG]http://distrowatch.com/images/icon-large/nonux.png[/IMG]]("http://distrowatch.com/nonux")         [Nonux]("http://distrowatch.com/nonux"), a Dutch Linux distribution based on Slackware with Dropline GNOME and optimised for desktop use in Dutch business environments, has reached version 4.1. What's new? Upgraded the Linux kernel to version 2.6.18.6; upgraded Mozilla Firefox to 2.0.0.1; upgraded OpenOffice.org to 2.1.0; replaced WiFi Radar with NetworkManager, which simplifies network detection and configuration considerably; several small upgrades of Slackware and GNOME packages. Users with Nonux 4.0 installed on their hard disk can upgrade to version 4.1 directly from the live CD. Please visit the project's [news page]("http://www.nnlinux.com/nieuws.html") (in Dutch) to read the full release announcement. [Download (MD5)]("http://www.nnlinux.com/download.html") the Nonux 4.1 CD image from here: [nonux-4.1.iso]("ftp://nonux.net/nonux_iso/nonux-4.1.iso") (698MB).[http://distrowatch.com/?newsid=03999](http://distrowatch.com/?newsid=03999)
