### Post by jariku on 2007-03-31
I installed Foresight 1.1 on my desktop computer today and I have to admit, I'm pleased.

This machine had Feisty until a few days ago and then Gentoo until this afternoon. Now it has only Foresight and I'll probably keep it that way until Haiku is released. :) 

I had some issues during install and initial configuration, but nothing that would've stopped the show:
- the installer didn't ask for my keyboard layout to be used during installation
- the "Running Post-Install Scripts" window (or something like that) didn't indicate whether it had crashed or not and I had to look at my computer's IDE light to see if anything is happening
- my Radeon 9000 Pro wasn't automatically identified and I had to edit xorg.conf
- Gnome hardlocked on me a couple of times when I tried to Ctrl-Alt-Backspace
- gutenprint is an old version so my printer isn't supported
- Thunderbird wasn't installed by default
- Foresight gave my user account a different user/group numbers than Ubuntu/Gentoo so I had to chown/chgrp my home dir (which resides on a different partition than the root filesystem) to log in
- I'm unable to install tilda due to unmet dependencies (I'll work that out later)
- the initial install was sluggish until I updated the kernel and other software to current versions
- default icons are ugly (I changed them to Tango right away)
- the webgui to the package manager doesn't allow me to add several programs to be installed but I have to install one program at a time
- gDesklets don't work (they don't really work in my Feisty either, but at least clicking the gDesklets icon in Applications does something)
- it took me a few minutes to figure out why volume control was muted even though I had unmuted everything with alsamixer (I had to adjust the master volume control to unmute it "properly")
- the System Manager doesn't really tell me what I have installed
- update after installation gave me some unhelpful error messages about something related to python but I think everything got updated anyway
- opening "About Myself" and clicking apply without actually editing anything gives me a password error without actually asking for a password first
- no SCUMMVM in the repositories? Come on!

Foresight had some very good sides also:
+ Compiz works almost out of the box (which it didn't in Feisty)
+ Banshee instead of Rhythmbox
+ the GL Desktop icon is visible automatically
+ Glipper is started automatically
+ gstreamer-plugins-ugly are installed by default
+ Gnome is more recent version than in Gentoo (even with ~arch)

I'm pretty sure that in six months or a year Foresight will be as good as Ubuntu is. Right now I'd give it 7.5 out of 10 when Feisty gets 9 out of 10.

EDIT: I changed the rating from 8 to 7.5 due to lack of SCUMMVM.
