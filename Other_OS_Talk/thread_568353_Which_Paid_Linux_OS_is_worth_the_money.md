### Post by American_Outcast on 2007-10-05
I created this poll for personal reasons, which is why I only have three listed in the poll. I really don't have a problem with paying for Linux, as long as it is not an insane amount.

The things I am interested in is how stable the Distro is, how reasonable the price is, how the support is, how reliable it is, what software packages do they offer  that are not Open Source, how customizable is it, if you stray from the "normal" setup/settings will it start acting buggy, are the consistent security updates..... and so on.

I really don't want this to turn into a thread about how a distro in question shouldn't charge, or it costs money when it should be free, etc. I think 40-60 USD for a distro is more fair then, for ex., MS which costs around 150 USD to start and then you need to buy other software, services, etc or Apple which has an OS that only runs on their hardware.

One thing I am not at all happy with concerning any OS is a monthly or yearly fee of any sort, ex. Clubs, etc. But that is a personal opinion of mine.



If anyone has had any issues or positive experiences with the three linux OS' please leave a comment.
