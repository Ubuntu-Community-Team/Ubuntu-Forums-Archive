### Post by justin whitaker on 2007-07-27
> **fistfullofroses said:**
> I was wondering if anyone here has used Gobo Linux. If so what did you like/dislike? Was the community friendly and helpful? During install did you have the choice to custom-select packages or package groups/meta packages? Any input on the distro would be truly helpful.

I'm waiting for .14 final before trying it again. .13 and .14 pre1 do not like my hardware much when I tried to install it. In particular, it really hates my audio card, and didn't partition my drives correctly last time I tried to install it. 

It has a very active group of core developers, a very small community, and you have to want to get your hands dirty in the code, because things have a tendency not to work. 

When they do work, however, as it did on my old PC, it has a lot of nice features: Compile and Manager are really nice, the fact you can load more than one version of an application because the file system is a pseudo-file manager is very slick....

In about 5 years, it will be awesome. :(:)
