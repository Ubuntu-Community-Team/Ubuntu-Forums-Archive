1. Zenwalk uses Xfce and is reasonably light for a decent machine, so I give it the edge here.  TinyMe (a PCLinuxOS remaster) is fast as well.

2. Both include lots of multimedia stuff by default.  Zenwalk has everything except libdvdcss, which can be installed from the repos.  I think PCLinuxOS may have everything.

3. PCLinuxOS has more packages, but I prefer Zenwalk's netpkg package manager.

4. I think you'd learn more with Zenwalk.  It is a reasonably pure Slackware system at heart, while PCLinuxOS... is Mandriva at heart.  Nothing wrong with that, but Slack is more vanilla and more manually oriented.

5. I don't know.  I think Zenwalk can have some issues on that machine, but my memory could be off.
