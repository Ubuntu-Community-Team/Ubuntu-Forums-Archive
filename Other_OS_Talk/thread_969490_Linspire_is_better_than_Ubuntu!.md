### Post by aysiu on 2008-11-03
Linspire sounds great in theory and has sounded great in theory for as long as I've been using Linux (more than 3.5 years now), but it isn't as easy to use as its advertisements would have you believe.

I found CNR to be quite a nuisance, and I didn't like any of the defaults, the "clever" renaming of normal open source applications, or the "security" implementations.

Ubuntu I've found to be a little buggy at times (random breakage can occur, especially with kernel updates) but basically feels very comfortable. If you install it yourself it doesn't come with all the popular codecs preinstalled (neither does Windows, by the way), but if you buy it from Dell, it does. Linux Mint is a version of Ubuntu that comes with all the popular proprietary codecs already installed.

And I actually prefer Add/Remove, Synaptic, and *apt-get* to CNR. I like the *idea* of CNR, but its implementation isn't simple enough for me.

When it comes down to it, Ubuntu has the combined power of [list][*]Good word-of-mouth advertising[*]A proper balance between a push for open source and appropriate compromises for proprietary elements[*]A grounding philosophy of humanity toward others[*]A solid financial backing from a multimillionaire[*]Improved usability with every release[*]A strong community of new-user-friendly but still knowledgeable users[*]Sensible defaults[/list] And I don't necessarily consider the inclusion of KDE a positive thing, but Ubuntu does include KDE in Kubuntu, so if KDE is your thing, use Kubuntu.
