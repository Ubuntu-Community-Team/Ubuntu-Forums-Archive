### Post by COLiNx86 on 2008-12-19
I recently installed Foresight linux on my laptop and I have to say it's pretty awesome!  It has everything I need/like. Two big things for me are Rolling release and Very easy installation.  Conary is also pretty nice, it's like rpm/yum but faster.  The theme isn't that great but I have yet to find a distro that has a nice theme pre-installed anyways.  The Repo's aren't that big either but I haven't had any problems compiling from source.  Also a lot of guides written for fedora work on Foresight, you just have to slightly modify the commands.

Overall I give it a 4/5.
[Foresight Linux]("http://www.foresightlinux.org/")

### Post by tgalati4 on 2008-12-20
Foresight preserves the gnome desktop environment.  It's interesting to compare Ubuntu's changes from true gnome.  Foresight is built with rPath technology so a new distro can be built in half an hour after a new gnome or kernel release.  Ubuntu's tweaks require some time to incorporate and test, so it releases with an older version of gnome.

Foresight is a solid distro.
