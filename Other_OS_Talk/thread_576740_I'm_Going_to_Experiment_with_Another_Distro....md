### Post by KCPokes on 2007-10-15
Really depends on what you want to get from it.  I've run more distros that I can even remember at this point, but each has some pros and cons, thus you need to decide what you want to get out of it.  If its just to try something else, I'd try some of the "hot" distros just to compare what everyone is saying/raving about.  If that is the case then you might try OpenSuSe, PCLinuxOS, etc...  If you want to just dive into something completely new, which doesn't sound like thats what you are looking for, you might try something you work at more to set up.  My personal choice right now is Arch Linux, but its definitely not for everyone.  The choice was made based on the requirement to have/build a lightweight distro on an old (and I stress OLD) laptop.  Arch  accomplished that where others didn't.  Which brings me back to the original question of what you are looking for.

### Post by TheThinker on 2007-10-18
If you don't like to install new systems on your harddrive yet you want to be able to save settings on a temporary O.S., you can always try out tiny linux distros like Damn Small Linux and Knoppix by installing them onto a flashdrive. Among the many benefits to this are:

1) It is ususually faster and more reliable to run than through a live-cd.
2) You can save settings onto the flashdrive if you desire.
3) It's more portable than a cd and is more damage-resistant.
4) You can reuse the flashdrive by installing a new O.S. on it or by reformatting it.
5) These O.S.s are typically small, which means that old or cheap flashdrives can be used (DSL uses just 50mb).

I highly recommend this option if you don't want to spend a lot of time burning multiple CDs. It's also surprising to see how much functionality some of these linux-versions have for such a small amount of memory!
