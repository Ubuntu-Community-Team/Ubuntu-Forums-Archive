### Post by rrwo on 2006-12-07
A few years ago I worked for an organization that used DOS machines to run some simple software that controlled gas pumps, parking garages, truck scales, and download news stories from satellite feeds from various news services like the AP. (Different machines did different functions, not one machine doing all of these.)

There were upgraded versions of the software that ran on Windows 98/NT/etc, but they had no need of the newer features, so it wasn't worth the money to upgrade the software, or the machines.  In particular, the machines were often blown up by lightning strikes, so rather than spend a lot of money on a machine powerful enough to run the latest Windows, and then spend a few hours installing/configuring Windows, they would get an old 286 or 386 being used as a doorstop, install DOS and the extra software and have the system back up within an hour.  The machine would last a year or two until the next lightning strike.

There's a lot of legacy applications that run on DOS, and they do their job well.  There's no benefit to upgrading.
