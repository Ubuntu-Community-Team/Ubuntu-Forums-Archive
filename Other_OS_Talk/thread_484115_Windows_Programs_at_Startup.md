### Post by dca on 2007-06-25
All users have a startup directory in each of their profiles...
 
C:\ docs and settings \ *user profile* \ start menu \ programs \ startup
 
Also, you can use 'MSCONFIG' utility from the 'run-line' to see what start's up within the registry on the last tab...
