### Post by Bungo Pony on 2008-02-26
This is a fantastic little distro! Just about everything you need for working on partitions, hard drives, broken linux installs, and retrieving files! Here's some of the stuff you get in here:

Visparted - Seems to work much like Gparted (which is no longer being developed)
Terminal - Very necessary :)
3 File Managers - Emelfm2, PCman, Thunar. You can mount hard drives with these things and pull files off broken Linux installs. It also supports NTFS!
lftp - transfer files via FTP
Firefox - Browsing the net
Conky - Tells you what's happening.
Screenshot- Take screenshots!
XFburn - Burn CDs - useful for pulling files off hard drives
ISOmaster - Create .iso files
Xarchiver - Crunch those files into an archive!
Hardware Lister- Tells you what's in the PC
And multiple desktops!

There's a few other things in there as well, but those are the most useful ones from what I can tell. You can also create a bootable USB flash drive too! All of this is currently running in just under 30M of RAM - with the entire distro running in RAM! Did I mention that it looks great too?

I'm incredibly happy with this live CD, and it's going to come in VERY useful!

For the question on how it compares to Gparted, I think this distro is much nicer and has a lot more useful features. It also seems to be less resource-hungry, and less problematic for booting (I had problems booting off Gparted on older hardware)
