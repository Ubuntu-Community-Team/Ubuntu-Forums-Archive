### Post by donkyhotay on 2009-02-12
It won't run linux programs as it's not linux. Basically it's meant to be an open-source reverse engineered version of windows. Linux was originally made because linus wanted to play with unix and couldn't afford it and didn't like the license behind it. ReactOS is the same thing for windows. I've played with it and unfortunately it's definitely alpha software. I've had better luck running windows programs in wine then I have running windows programs with reactOS. I wouldn't recommend running it as a regular desktop at this time (even the reactOS website says this is a bad idea) but you may want to play with it in a VM environment for experimentation, debugging, development reasons.
