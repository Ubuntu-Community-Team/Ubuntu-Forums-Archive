### Post by Antman on 2007-12-29
My top 4 linux distros are (in any order):

1. **OpenSuse 10.3** - this distro has worked the best on my laptop so far (besides XP/Vista). The only real issue I had was that Knetwork manager wouldn't let me connect to **open** wireless spots; this apparently is a known issue and also happened in Fedora 8, and debian lenny and sid. My fix in those other distros was to remove network manager and install WICD.
Once I get WICD is work in Opensuse, I will use that as my primary laptop OS.

2. Sidux - If you like cutting edge distros that you only have to install once (then just dist-upgrade via a cool script) then this is for you.

3. Mandriva 2008 - Very nice distro.

4. LinuxMint - if you like Ubuntu but wish it came with all the multimedia extras, this is for you.
