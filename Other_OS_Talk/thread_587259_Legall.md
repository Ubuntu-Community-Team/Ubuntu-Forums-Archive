### Post by 3rdalbum on 2007-10-26
If you are distributing mini-distros like Puppy Linux or DSL, you can use mini CDs. They hold 180 megabytes and fit into a standard envelope (so in most parts of the world, you can post them for the cost of a letter!). Be aware that these discs themselves are a bit more expensive than standard CDs or DVDs, but if you are sending over long distances it might just be worthwhile.

EDIT: It is legal to charge for GPL'ed software, as long as you provide source code on request (preferably, provide source code with the binary). Your obligations as distributor of GPL software do mean that you would have to actually provide the source on request - not just tell someone where to get it.
