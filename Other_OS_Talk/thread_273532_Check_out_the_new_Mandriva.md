### Post by wondering_jew on 2006-10-08
I have to say I'm quite impressed so far with Mandriva 2007... I downloaded the live/install cd. XGL/compiz worked right from the get go off the live cd with almost no hassle. This prompted me to go ahead and install it on an empty partition. The first boot wizard then proceded to effortlessly configure my wireless network through a USB adaptor that in ubuntu took some effort in finding drivers that would work (even then the signal strenth icon in the task bar didnt work, but works perfectly in mandriva). A fresh install of ubuntu doesn't even have the ability to compile the drivers until it can get on the internet first (which means dragging my pc upstairs so I can get a wired connection. Overall Mandriva makes a great first impression.
