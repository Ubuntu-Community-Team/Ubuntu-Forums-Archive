### Post by timzak on 2008-07-22
Hi,

I installed Thunderbird in Puppy 4.0, which I can launch via menu, but I'd like to have the email icon on the desktop launch it. I tried right-clicking the email icon, selecting File 'email', then Open As Text, and found the default email configuration file, which lists "sylpheed" as it should.  I tried changing this to "thunderbird", "tbird" and every other option I could think of.  What is the correct launcher name for Thunderbird?  How do I go about finding this out on my own in the future?

Thanks!

### Post by timzak on 2008-07-23
Any help would be greatly appreciated.  I tried going through the Puppy help file but couldn't find anything.  I think I know what to change (see my post above) I just don't know what to change it to.  How do you find out a program's launcher name?  Not just Thunderbird's, but any program?  I imagine that if I want to install any other packages in Puppy and want to have a desktop launcher, I will encounter this issue again.

Thanks,

### Post by timzak on 2008-07-24
Well, I was finally able to find a solution.  I'll post it here in case someone else can benefit from it.

Go into your Puppy home folder, hit CTRL-H to show hidden files, then open .jwmrc with a text editor.  This file has every program installed in Puppy along with its launcher name and directory.  I was able to find Thunderbird, copy the launch command into the email shortcut icon, and it did what I needed it to.
