### Post by izalac on 2006-11-20
My first Linux was an ancient Slackware I installed on a 486 back in '96 :D
Don't delude yourself. Although ancient, you can still do wonders on it. You *can* do graphics on it, and you can even run X. 16 MB of RAM help a lot considering its configuration, but I wouldn't mess with X if it's below 50 MHz. If you got 486DX, you might as well go for it. I remember running FVWM on it back in the days :)
Your graphic card should support VESA standard, most cards of that time did.

On the other hand, command line can be very powerful. If you can get it, I'd wholeheartedly reccomend some of the old distros. Most of them came on CDs, but contained instructions on how to make floppies.
100 MB of system space can be quite a lot, if you use it properly.

Suggested uses for your 486 box:
1) Make a powerful commandline Linux system
2) Make an aged, but nevertheless impressive (for its hardware) desktop system
3) Make a router/firewall
4) Make a web server
5) Put FreeDOS on it and play legacy DOS games
