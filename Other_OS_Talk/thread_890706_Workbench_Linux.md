### Post by cyberpython on 2008-08-31
Hi everyone!

I am the "developer" of Workbench Linux Distribution, which is in fact a remix of Xubuntu (I used remastersys) that comes bundled with a selection of software mainly targeted at developers and some eye-candy.

It has nothing to do with the Amiga workbench (in fact I had no idea there was such a thing when I picked the name).

The distribution basically is Xubuntu and as such everything an Ubuntu/Xubuntu user would expect are there : Ubuntu repositories, Synaptic etc.

Thanks for downloading!

### Post by drbongo on 2008-10-19
Cyberpython,

Congratulations on the release of Workbench Linux (Now: Excelixis) it is a superb remaster!

I have been working on similar lines over the last few months, trying to create a developer based remaster so I don't have to download, install and reconfigure my computer every time I switch, reinstall or upgrade a distro.

I have created remasters based on PCLinuxOS and Puppy (DingoPlus) as well as Ubuntu/Kubuntu remasters using Reconstructor but found it fiddly and inconsistant. I discovered Remastersys when I first tried Workbench Linux and was surprised at how simple it was. I have made a few remasters of Ubuntu using this since, but it takes a lot of time and experimentation to get everything right - and why reinvent the wheel?

Your remaster contains nearly all of the apps/features I want anyway and it looks great! I have only had to install tsclient (Remote Desktop Client for Windows) and update Gambas to 2.8.2 (from [http://azores.linex.org/gambas-other](http://azores.linex.org/gambas-other) hardy main) and it now it ticks every box!

I have used all of the major linux distributions and many of the lesser known ones, and I have always come back to ubuntu as it has the best hardware detection and all the basics tend to just work. However it has always been aimed at desktop users and required a lot of work to create a satisfactory development system.

I believe you have identified a very important niche in the linux user-base: Developers who want an uptodate distro that has the necessary tools as well as all of the desktop glitz that cutting edge distros like Ubuntu offer.

Your distro is not well known yet - but wait till it hits distrowatch! I suspect you are suddenly going to get a lot of interest. If it is possible it would be nice if you had a user forum, or at least a thread in the ubuntu forums for Excelixis as I am sure there will be plenty of information to share and questions to ask. There is currently no way of contacting you or other users on the Excelixis website.

If I can contribute or help in anyway please let me know. I have pimped my desktop using the screenlets and the cairo toolbar which I will post as a screenshot later on. However the only way to retain this is to make a full backup including the home directory - is it possible to save the way the desktop looks using the distro option?

drbongo
