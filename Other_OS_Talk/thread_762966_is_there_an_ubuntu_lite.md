### Post by Ozor Mox on 2008-04-22
I also recommend burning a copy of the Ubuntu alternate CD and using it to install a command line system. This way you can literally build the operating system any way you like, with any window manager or desktop environment using apt-get. If you install packages which are on the CD, apt-get will even use those instead of downloading from the repositories if the versions are the same.

I have been trying this recently, and it works very nicely, so long as you are comfortable with the command line :)
