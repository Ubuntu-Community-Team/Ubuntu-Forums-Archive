### Post by Rui Pais on 2008-07-02
Tikkun olam

Well in fact we has been debating this for some time now...

The question is essentially that it's Cafelinux purpose to support and help new Open Sources projects (on past it host and seed Geubuntu, PokerTH, too).

Now gOS has grown up, have a nice reception among linux users, it's a commercial (made it's own money) project.
So seems to be no point in an altruistic, non-lucrative, project like Cafelinux to support (in only one way direction) the gOS.


gOS forum was made read-only while we wait for some answers from gOS devs, to either stop it or move them to they servers :)


Of course, if they move definitly to Gnome, they turn to be an Ubuntu with Google apps and a theme. Ubuntu forum seems to be a much natural place for support...

Enlightenment version support (don't know if they'll continue to implement it) can be done on main Cafelinux forum (very enlightenment centric). 
Enlightenment (e17), either CVS/OzOS way or precompiled binaries, will be always accessible for gOS users, anyway.
   

Hope that helps to clarify things a bit.
