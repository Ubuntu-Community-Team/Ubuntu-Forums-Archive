### Post by ThinkBuntu on 2007-06-25
I'm posting from Pardus 2007.2 (it has a different name. The LiveCD and the Install each have their own codename...) and it's great! Pakage management is a bit slow, and you'll want to add the community repos, but besides that, it's a dream to use. Fairly nimble, and best of all, its wireless performance is second to none. Install it; I promise you won't regret it! Besides a stint with Debian, I've been on Pardus for a couple weeks so far, and everything's a snap.
