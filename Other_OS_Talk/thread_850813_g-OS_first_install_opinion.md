### Post by squidmaster on 2008-07-06
Meh.
It gets a giant and well deserved "meh".
Installed "space" to see what this internet based system was all about.
Sure it's decently easy for the first time linux user (I think it would be at least but I had to work the command prompt a bit).
Set up was a breeze to be honest. It said it would import my settings from firefox from 8.04 into space but didn't happen, after tinkering I slapped myself on the forehead for realizing this is using FF 2.x and 8.04 I was using 3b.
The dock they use is Avant Window Manager and the only complaint I have about this is the ease of use for someone using this OS that knows VERY little about linux. You can't just drag and drop and application into the dock.
I would have personally chosen Cairo-dock. It's nicer looking has better effects and has a "drag and drop" feature that greatly simplifies opening the preferences and going into launchers and finding out the exact run string for your application (which isn't too hard for someone that's been around linux for a while, but again this is based on someone that is pretty new to linux)......

It boasts that you can access the entire internet just from the dock without any mod but there is no direct link to pidgin, the myspace app docklets should really be placed somewhere else, and there is no direct link to myspace home (doesn't matter much to me but seems like they would have placed something like that, but instead there is just a ton of other links to things that don't seem very necessary, or maybe I am just not that myspace "savvy").


Another complain, I have been trying to get the Last.FM docklet working for hours with no success, works fine on every other linux install I've had, but not this one for (what ever) reason.


Others opinions?
