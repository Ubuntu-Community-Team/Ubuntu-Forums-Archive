### Post by seanc7 on 2008-05-12
The office is 99.9% Windows. AD, Exchange and XP desktops. The admin has a Fedora VSFTP server and a Fedora Nagios monitoring server, but he has to hide them from the corporate office. Corporate is 100% MS, FTP, SMTP, IIS, Sharepoint, Exchange, AD, SQL Server, etc, etc, etc...

In our lab, which is separate from the office (so corporate can't remotely see anything in it) we have 8 Fedora Linux servers running VMWare for software testing and network admin - Win2003 for pretty all the VMs. We have a Linux SMTP, FTP and NTP server, a Linux network monitoring system, a local YUM repository server and soon we'll have a Linux Nagios monitoring of the entire lab. Our in-house software products only run on Windows. That's not changing any time soon unless banks switch to Linux.
