Nobody I know. :)
> Is it of any value to learn it (except for fun of course)?
Not really. VMS was a pretty straightforward (i.e., dull) operating system. Its main drawing point was excellent documentation.
> Are there any other _stable_ OS that run on a MicroVAX?
There is a SF project for [a Linux port]("http://linux-vax.sourceforge.net/download/index.html"). Also, there appears to be [a BSD port]("http://www.openbsd.org/vax.html")
