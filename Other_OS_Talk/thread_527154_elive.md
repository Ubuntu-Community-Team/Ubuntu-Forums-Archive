### Post by Freddy on 2007-08-16
You don't change docks in enlightenment, you can change the appearance slightly though. 

I don't know if you can do this in elive, since the old version of e that elive uses doesn't use the shelf system but if you should be able to change the appearance (if the theme has any other appearances of them) by left clicking on the desktop for the e menu, then Configuration | Shelves, but since you don't have any shelves, you might need to go into Modules instead and mark you iBar module and configure it from there.

If the theme you use doesn't have any appearances compiled into the edje file, you are out of luck.
