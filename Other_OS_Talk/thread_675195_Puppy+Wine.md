### Post by Antman on 2008-01-22
Download wine for Slackware from [www.winhq.com](www.winhq.com)

When you get it, open a prompt and type

```
tgz2pet *filename*
```

Then click the resulting .pet.

Edit: This will work if you are using Puppy 3.01
