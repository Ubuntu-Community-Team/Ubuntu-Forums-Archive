### Post by SomeGuyDude on 2008-12-19
Funny thing is, as much as the more "hardcore" Linux users talk about how Compiz sucks and work at trimming down system resources (guilty here, although now I'm just more used to OpenBox) and all those other technical aspects, if you want to show it off or get some converts, flashy bloated is the way to go.

Your average computer user doesn't give a damn if their system uses 450 or 1500 MB of memory during usage as long as it's not dragging the experience down, and thus is a lot more impressed with a desktop cube than having the entire system running on a 512MB flash drive.

Anyway, back to my point, if your system WORKS, don't mess with distro-hopping just yet. I went that route and it was pretty disastrous when things didn't go well. I'd lose time to work (at best) or I'd lose work itself (at worst). It got terribly stressful when I was trying to get something done for class and I'd lose a day because I thought I could use distro X only to discover this or that wasn't working right and I had to go back.
