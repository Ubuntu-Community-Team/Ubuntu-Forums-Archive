> **Oz Linux**

                      **From Wikipedia, the free encyclopedia**


**Oz Linux** is a [user-friendly]("http://en.wikipedia.org/wiki/Usability") [Linux distribution]("http://en.wikipedia.org/wiki/Linux_distribution"). It allows the user to choose between a [GNOME]("http://en.wikipedia.org/wiki/GNOME") or [KDE]("http://en.wikipedia.org/wiki/KDE") [desktop environment]("http://en.wikipedia.org/wiki/Desktop_environment") install (or enable both and switch between the two by restarting the [X Window System]("http://en.wikipedia.org/wiki/X_Window_System")).

 Oz Linux is named in honour of two [Dutch]("http://en.wikipedia.org/wiki/Netherlands") [physicist]("http://en.wikipedia.org/wiki/Physicist"): [Leonard Salomon Ornstein]("http://en.wikipedia.org/wiki/Leonard_Salomon_Ornstein") & [Frederik Zernike]("http://en.wikipedia.org/wiki/Frits_Zernike"), who developed the [statistical mechanics]("http://en.wikipedia.org/wiki/Statistical_mechanics") [Ornstein-Zernike equation]("http://en.wikipedia.org/wiki/Ornstein-Zernike_equation"), or OZ-equation. The OZ-equation is an [integral equation]("http://en.wikipedia.org/wiki/Integral_equation") for defining the [direct correlation function]("http://en.wikipedia.org/wiki/Correlation_function_%28statistical_mechanics%29"). It basically describes how the [correlation]("http://en.wikipedia.org/wiki/Correlation") between two [molecules]("http://en.wikipedia.org/wiki/Molecules") can be calculated. Its applications can mainly be found in fluid theory.

 Oz Linux utilizes the [Conary]("http://en.wikipedia.org/wiki/Conary_%28package_manager%29") software system which combines [package management]("http://en.wikipedia.org/wiki/Package_management_system") and [Software Configuration Management]("http://en.wikipedia.org/wiki/Software_configuration_management"), which replaces the need for programs like [apt-get]("http://en.wikipedia.org/wiki/Apt-get") and [yum]("http://en.wikipedia.org/wiki/Yellow_dog_Updater%2C_Modified"). [Conary]("http://en.wikipedia.org/wiki/Conary_%28package_manager%29") accomplishes this with such features such as rollbacks, changeset (incremental) updates, and a distributed development model.
 [http://en.wikipedia.org/wiki/Oz_Linux](http://en.wikipedia.org/wiki/Oz_Linux)
