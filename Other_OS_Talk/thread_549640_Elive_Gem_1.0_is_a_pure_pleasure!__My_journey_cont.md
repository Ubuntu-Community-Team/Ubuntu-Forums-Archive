I was going to publish a quick How-to on it with screenshots, but I'll explain it a little here:

Open up Entangle via your Elive Configuration Panel

Elive Configuration Panel>Look N Feel Configurations>Enlightenment 17>Entangle, Icons and menu manager

once Entangle is open, click on "iBar" then simply grab and drop your icon that you want to add to the iBar(right-click on the icon you want to drag and drop it on the iBar list in the position you want), 

To remove icons off the iBar simply middle click on the icon you would like to remove.

Last step and most important: hit "save".

I have set up a icon for Entangle directly on my iBar so I can easily access it quicker if needed
