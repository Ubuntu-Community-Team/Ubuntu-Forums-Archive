> how long does it take?

depends on your specs and how long it takes you to read.

> How difficult is it?

the instructions makes it easy, similar to gentoo

> did you find it a good learning experience?

for me not really, it was like when i did gentoo. to me it's not really worth it. i think it's better to start with a base install and build up from there, using a well supported base distro.
arch and debian are both nice starting points, has lots of docs and how to's. i wanted to get the most out of this old rig so i tried everything i could find. one of the things i really hated was compiling, some was fast, others took forever, but given my specs(see signature) it was reasonable.

i went debian etch/lenny base and built up from there, it's simple and the speeds are about the same as doing it from scratch. installation/removal of software is a snap, speed and resource use depends on what you add.

i'll like to add: you got to have patience, if your short tempered, it will piss you off at times. also the best thing to do is get familiar with various programs, before you even try, know your vi, know how to move around, learn to love command line, learn the short cuts. for example moving around auto complete can save tons of typing. example:
cd /e press tab type X press tab
editing is the same thing:
vi /e tab en tab

you can try it on your installed system. learn cp & mv, untarring, etc...
