### Post by danielph on 2007-05-01
I would second Arch as a good distribution to try. It has the advantage over Gentoo that you will not suffer the long compile times on a slower machine. Arch works to KISS (Keep It Simple Stupid) philosophy and allows you to set up everything by hand how you want it. It is very up to date and it is x686 optimized (good for a PIII). You can install base in minutes and it is easy to add a wm or desktop from here. The community is knowledgeable, but will not hold your hand. The documentation is good, but not as good as say Gentoo. The advantage of Gentoo is nuts and bolts learning. I loved it and would have stayed if it were not for the compile times. I tend to change things a lot!

If you want server stability then Debian stable is a good choice, but you have already got the t-shirt.
