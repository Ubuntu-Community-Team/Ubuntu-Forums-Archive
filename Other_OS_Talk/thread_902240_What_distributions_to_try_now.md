### Post by RedSquirrel on 2008-08-29
> **kagashe said:**
> ... My preference is for small and fast distributions.

I had compiled LFS and BLFS in 2005 which I am going to do again.

I understand that Arch is easier than Gentoo.

Suggestions please.

kagashe

If you haven't tried Gentoo, I would recommend giving it a try. It's my favourite. :)

I would say Arch is *faster* to setup because it is a binary distro. You can be up and running fairly quickly.

With Gentoo, it doesn't take that long to setup the base system, but it may take a while to do the rest depending on the applications you plan to install (and the speed of your computer ;)). Gentoo offers binary versions of some of the larger programs, for example, Firefox (mozilla-firefox-bin), Thunderbird (mozilla-thunderbird-bin), OpenOffice (openoffice-bin).

Installing Gentoo: I would recommend avoiding the Gentoo "installers". The manual method in the [Handbook]("http://www.gentoo.org/doc/en/handbook/index.xml") is more reliable. I used Gentoo's minimal CD as the environment to install Gentoo, but you may want to consider using a different CD, for example, [sysrescuecd]("http://www.sysresccd.org/Main_Page"). It is a good idea to use the stage3 files from [funtoo.org]("http://funtoo.org") since they are very up-to-date and will likely save you a lot of headaches.


**Edit:** If you get stuck on the installation, drop by the [Gentoo forums]("http://forums.gentoo.org"). There's a subforum there devoted entirely to Installing Gentoo. ;)

### Post by stephenj2008 on 2008-09-23
My main distro is PCLinuxOS 2007(KDE), which is great, but I wouldn't suggest trying this one as it is a 700mb (approx.) download followed by about 600+mb of updates.  (PCLinuxOS 2008 apparently is "coming soon".)

There are however, others based on PCLinuxOS that are more up-to-date, but which I have not tried. PCLinuxOS MiniMe (KDE) came out at the beginning of this year.  It is a minimalist version of PCLinuxOS; so too, is TinyMe 2008 (LXDE desktop) which was released in May, this year.  These might suit you as you expressed a preference for smaller 'light weight' distros.

There is also a Gnome version of PCLinuxOS (approx. 700mb) which also came out at the very end of last year.  I can't see that this has been updated (it's probably a rolling distro, like PCLinuxOS (KDE)) so perhaps this is getting a 'bit long-in-the-tooth', too.

Also closely related to (based on) PCLinuxOS is the Indian distro named 'Granular' (KDE).  Release Candidate of Granular Linux 1.0 came out at the beginning of July, this year.  It is a 'full strength' distro measuring almost 700mb.
