### Post by Pobega on 2007-05-18
If I were you I would try Debian. Gentoo would work well too, but I'm not sure if all of the sources needed would make you go to +1GB (As far as I know the Portage tree is 700MB itself).

Debian is nice, and if you need added speed you could use apt-build to build all of your programs optimized for your processor. Just make sure to clean out /var/cache/apt/archives/ every once in a while and you should be fine.
