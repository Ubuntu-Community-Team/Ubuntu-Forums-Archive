### Post by igknighted on 2007-05-06
> **izizzle said:**
> hey igknighted, can I have you SAM linux config file, PLeeeeZ? I have the wbar on the bottom but the programs keep overing it. Can i pleez use your config file?
//izizzle

```

sleep 5
wbar -j 1 -p bottom -idist 5 -nanim 3 -z 1.8 -above-desk -bpress -isize 54 -balfa 38

```

I assume you meant the wbar config file.  As far as apps covering it, that will happen.  If you want you can install Kiba-dock, I don't think that happens with Kiba.  The nice thing about Xfce is that when you turn on compositing, you don't need beryl or compiz to run Kiba.  Just add kiba-dock to the autostarted apps and remove wbar.  Kiba acts really funky tho, so beware :)
