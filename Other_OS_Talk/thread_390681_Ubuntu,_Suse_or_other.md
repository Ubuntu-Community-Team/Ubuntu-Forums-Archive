### Post by igknighted on 2007-03-22
I might recommend trying Linux Mint.  It has lots of the multimedia codecs preinstalled, plus I like the way it is put together a lot.  PCLOS is a decent choice, 2007 should be out very soon (SAM, a project based on PCLOS released this morning... but no PCLOS? Odd.).  If you like PCLOS, try Mandriva.  PCLOS is basically re-branded Mandriva.  With Mandriva you can buy a version that legally plays mp3's, DVDs, etc., or use a free edition that requires setup like ubuntu.  Another distro that is nice is Sabayon.  It is based on gentoo, but the install is fast and as easy as Ubuntu.  It will set up EVERYTHING for you (including 3d video drivers and often wireless).  If you use the DVD it comes with several linux native game demos, as well as almost any app you could want.  The mini edition is a single CD, is a little snappier, but the game demos are gone and you don't get nearly as many programs installed initially.

Give Mint a look.  They are doing some great work over there tweaking the already great work of Ubuntu.
