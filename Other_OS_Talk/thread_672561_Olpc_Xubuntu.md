### Post by francisco_athens on 2008-01-19
You can install an Ubuntu CLI system on the XO and add the parts of XFCE that you want to get a Xubuntu like desktop. I've posted instructions on my [Free Like GNU]("http://www.freelikegnu.org/?p=21") blog for installing to USB or SD flash to try it out. You can get accelerated video using a Debian xorg driver modified for the XO with [these instructions]("http://www.freelikegnu.org/?p=30").

This is based on how Debian is installed and on usb/[sd]("http://lists.laptop.org/pipermail/devel/2007-April/004902.html") installation instructions

A full Xubuntu install would take up too much space on the 1GB flash (even with jffs2 IIRC)
