### Post by Sorivenul on 2009-01-29
> **Mason Whitaker said:**
> Anyone know of a OpenSolaris compatible boot loader? <_<
Unetbootin isn't working...and I don't have any CDs
I honestly don't know outside of a USB dump and booting from the USB key.

This can be done with BeleniX (an OpenSolaris derivative), but requires a LiveCD (of course, this could be run in a VM). Once the LiveCD is booted running "pfexec usbdump" should wipe your USB key and give you a bootable BeleniX USB key. 

Outside of that, I know MilaX (a small OpenSolaris LiveCD) provides a LiveUSB image [here]("http://www.genunix.org/distributions/dss/milax032.usb").

Of course, both of these options require you be able to boot directly from USB.
