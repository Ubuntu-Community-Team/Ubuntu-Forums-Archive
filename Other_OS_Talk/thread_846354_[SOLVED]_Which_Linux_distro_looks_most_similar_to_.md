### Post by darrelljon on 2008-07-03
When I was a beginner I wanted the same thing, but truth is I wanted a distro with a similar Control Panel, Directory Structure and Installation method to Windows. ReactOS (not a Linux distro) comes closest to these things, but really the best advice is to choose a popular stable distro (like Ubuntu) learn how to adjust settings in Linux, particularly appearance if thats important. Learn the UNIX/Linux directory structure (put personal data in /home find your drives in /media or /mnt and forget about everything else). And don't install stuff except through the add/remove programs (adept, synaptic, apt-get). If you want to install Windows stuff use WINE. There's other stuff like repositories and making files executable too but you might be able to manage without.

### Post by darrelljon on 2008-07-16
The actual aesthetics of Windows XP are fairly meaningless and can be replicated on any distro. Set the taskbar to 32 pixels with a blue background, set title bar colours to blue, and change the wallpaper to some fields and sky. If you're using KDE there are even styles and windows decorations specifically named "Redmond" and "Redmond XP" aswell. You could download some icon sets too, but what is it you are trying to achieve?
