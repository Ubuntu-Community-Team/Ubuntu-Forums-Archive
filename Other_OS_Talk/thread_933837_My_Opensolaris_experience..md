### Post by kaldor on 2008-09-29
I want to learn more about Unix. So, I found out that the openSolaris OS is a Unix based system. I did not want to install a Unix system either, so opensolaris LiveCD was a good choice.


Overall, I like it a lot. The GUI is Gnome. I always liked Gnome, but this has a different yet very familiar feeling. For those of you who have used Mac 9.1 (which is the only Mac I used much of, I am unsure about newer ones) it should feel slightly familiar. In the top right corner it has that little space that lists the running applications, which I like :)


Even though I am running on a live CD, it is running almost as fast as my Linux install. 

If you want a fast, stable system, give opensolaris a try. It does not have a lot of support so far, but it is getting there it seems. This is the first time in months that I have actually found an OS besides Ubuntu that I love :)

I must also say that this is nowhere near finished. It is fairly new, so do not judge it by what it is right now. In my opinion, this OS could overtake Linux if it keeps up. It is Unix, it is free, and it is relatively simple. 

[http://www.opensolaris.org/os/](http://www.opensolaris.org/os/)

I would write more but I am currently lacking sleep and wrote this for something to do. =p

Enjoy!
