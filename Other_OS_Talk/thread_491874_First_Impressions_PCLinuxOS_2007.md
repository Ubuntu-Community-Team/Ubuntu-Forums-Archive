### Post by ramjet_1953 on 2007-07-04
Out of curiosity, I downloaded and installed PCLinuxOS 2007 a few days ago.

I must confess, this is one slick distro!

I have an Acer TravelMate 4101 and it recognised all of my hardware and there were no hiccups on a dual install with Ubuntu.

Most impressively, it recognsed that I have an Intel graphics chipset and asked me if I wanted to install 915resolution, when I went into the menu to change screen resolution. It automatically chose the correct aspect ratio (1280 X 800) and there was no need to do a manual patch. Very impressive. It would be interesting to hear how this distro handles NVidia and ATI cards.

Another good thing is that it uses Synaptic for its' package management, even though it uses rpm packages.

There is an extensive number of packages in the repositories and they all downloaded at a good speed.

Visually, this is a stunning distro and the Ubuntu people need to take a good look at it and spruce-up Ubuntu, especially with the boot and login screens.

Audio codecs are installed by default and all I had to do to play DVD's was to install libdvdcss from the repositories.

By default, it is KDE based, but it is easy to install GNOME, or XFCE from the repositories.

A newbie shouldn't have too much trouble installing this distro, but if there are problems, the forums are only a fraction of the size of Ubuntu's. 

However, as I didn't need any help, I can't fairly comment whether these forums are effective and responsive, or not.

Another nice feature is that most of the software installed is a later version than Ubuntu offers in Feisty. Of course, bleeding-edge packages can have their problems, but I have not noticed any, as of this time.

For distro junkies, and those just wanting to check-out a "pretty" distro, this is well worth a look.

Regards,
Roger :cool:

### Post by ThinkBuntu on 2007-07-05
Personally, I find Ubuntu to be far more polished than PCLOS. The Ubuntu default login theme is far more attractive (no ugly bold text all over, and a nice interface in Ubuntu), and PCLOS' default KDE theme is essentially a Windows ripoff, except for the background which seems to be a lightly tweaked KDE-3.5 default desktop (not "Default Blue", the other one). Beyond that, visually, PCLOS doesn't have much to offer. There are a few odd popups with a wizard moving about, and the Control Center has a nice look.

I don't know how a distro can consider itself to be "radically simple" though, when the first message a new ueser sees as they log in is "Setting up interprocess communications" (of course!), and the only clear way to do a system upgrade is via the command line.

All that being said, I enjoyed using PCLOS. To me, the main advantage is not user-friendliness, but rather the speed of the distro. It handled Compiz effects as well as Debian, speed-wise, which is impressive.
