[http://distro.ibiblio.org/pub/linux/distributions/](http://distro.ibiblio.org/pub/linux/distributions/)


it is indeed hard to find mirrors for some distros. i have had good luck find these distros on their own website: zenwalk, kateos, puppy, dsl, wolvix and knoppix.

the ibiblio site can be somewhat hard to navigate if you're not sure which version of the distro you want, so be sure to look for the most updated version on the distro's main website.
