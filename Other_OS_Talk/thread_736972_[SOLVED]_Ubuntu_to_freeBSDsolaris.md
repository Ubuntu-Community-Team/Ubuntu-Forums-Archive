### Post by SunnyRabbiera on 2008-03-27
I would say there will not be much of an advantage of using BSD or Solaris over ubuntu for a home computer.
Solaris is far from ready for desktops, it is mainly a server OS and has to catch up.
BSD might be a little better as BSD has desktop versions, but if you like flash and multimedia it wont be as easy to come by.
Flash is still an issue on BSD so if you visit a lot of flash pages you will run into trouble, you can use wine but I dont feel wine is as good on BSD right now.
Perhaps the best thing going for BSD is its stability, by far it is much more stable then Linux as it doesnt experiment with the kernel too much, but this looses support for many devices in the process.
