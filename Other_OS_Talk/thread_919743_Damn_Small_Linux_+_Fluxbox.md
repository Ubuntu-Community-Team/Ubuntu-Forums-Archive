there's 2 versions the first is the older 3.* series, fluxbox is the default desktop and jwm is optional from the menu. the 4.* series is the newest it uses jwm as the default. jwm is lower resource than fluxbox. fluxbox is still included with the newest, you can easily switch to it from the menu.

so your getting bad downloads or just bad burns?

try the net installer, you can install the base, then add xfce4.

[http://cdimage.debian.org/debian-cd/4.0_r4a/i386/iso-cd/debian-40r4a-i386-businesscard.iso](http://cdimage.debian.org/debian-cd/4.0_r4a/i386/iso-cd/debian-40r4a-i386-businesscard.iso)

after that's done installing, press ctrl+alt+delete
that should reboot and bring you to the log in screen.log in as your user.
from there you can open synaptic and add what ever else you want.

PS: a straight install from the net installer will give you gnome, which should run fine on your 700mhz 192mb ram, you could then just add xfce4 to that.

if you haven't got the ram upgrade yet, do the base install, but:
log in as root:
type> apt-get install xorg gdm fluxbox menu rox-filer leafpad dillo
