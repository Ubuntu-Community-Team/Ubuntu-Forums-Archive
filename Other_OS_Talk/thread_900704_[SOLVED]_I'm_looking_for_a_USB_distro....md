### Post by snowpine on 2008-08-26
4gb is plenty of space... you could even run Ubuntu if you wanted to! As far as the ram issue, that's really determined by the computer--it doesn't matter whether you are booting from a "normal" hard drive or a USB pen drive. You can install Ubuntu, Xubuntu, whatever to a pen drive just as you would a normal hard drive. Personally I am experimenting with Crunchbang Lite on a 2 gb drive, it seems to work fine. 

"Lightweight" distros like Slitaz, Puppy, DSL, etc. are a good alternative.

As much as I like the idea of usb linux, the reality is that a lot of old computers can't boot from USB. I find that booting from a live cd and storing my data on the USB drive is a more flexible and versatile setup. 

Good luck!

### Post by snowpine on 2008-09-16
Lately I've been running SliTaz as a Live USB. I really like most things about it, however, the speed is quite a bit slower than from Live CD (since the SliTaz LiveCD runs entirely from RAM). The compromise I've discovered is to boot from CD with the boot prompt 'home=usb'. It's the best of both worlds, and it has the advantage of working on old computers that can't boot from USB. But if I don't have a Live CD on my person, I can boot from the USB as well, with slower performance.
