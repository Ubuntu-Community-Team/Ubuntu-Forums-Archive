### Post by darkmaster on 2007-12-06

Hallo Silva, Geubuntu is based on E17 exactly like GOS, so, be aware that it will mess with your configurations. The Geubuntu repositoryes are different from those of GOS. You can find Geubuntu repos reading the WIKI in the main site. I have to update it but repos for Gutsy are the same as for Feisty, just replace feisty with gutsy and that's all but...

then again, I don't recommend you to add Geubuntu's repos on a GOS system because the GOS repos also provide e17 but a differently customized version! I suggest you to try and load Geubuntu on a Live CD instead. If it doesn't boot correctly (it should, it is based on E17 like GOS), then wait for the alternate CD to be released but... then again, there's no way to add a different session because both systems use E17 :(
Sorry.

If you just want to install the themes, icons and other media, you can add to your system only one line of the geubuntu repos, that is:

deb [http://geubuntu.intilinux.com/geubuntu-repository](http://geubuntu.intilinux.com/geubuntu-repository) gutsy/

and NOT the one with dunnewind inside it. At this point you can install geubuntu-desktop normally, without fear to break your system. Just remember NEVER to run the geubuntu-configuration utility found on Applications--> System, NEVER run it or the Geubuntu E17 configurations will be overwritten on your GOS settings and you'll have a normal Geubuntu desktop, loosing any setting you may have added to your system.

So this is it. Once you install geubuntu-desktop adding ONLY the deb line I wrote about here, you'll also be able to make some extra tuning. For example, you may want to run the E17 configuration panel and add to the startup applications:
- xfce4-panel
- network-manager
- gnome-power-manager
- update-notifier
- keyring-daemon
- gnome-settings-daemon
You'll find all of these apps in the system category :)

Good luck then! Please follow my instructions if you want to try Geubuntu on a GOS system, if you have problems about it, please post me back for help :)
let me know how the situation evolves then! Bye bye! :D
