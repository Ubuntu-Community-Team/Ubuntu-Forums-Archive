### Post by joninkrakow on 2008-06-23
IMO, the only, truly unique Linux distro is PuppyLinux. It is completely different from everything else I've looked at--no package management, no repos. Uses the squashfilesystem, can boot from pretty much any medium with four files, utilizes a union file system to keep all those squash files working together. If you want unique, I recommend Puppy. Oh, and it runs like lightening with a full gui on my PII 366mhz 160meg RAM (and I have it running on two other identical laptops with only 128 megs of RAM).

Oh, and it runs s root by default--beat that with a stick. ;-)

Ok, I exaggerated a bit on the no repos and package management. There is something, but not like in the other distros I've looked at and used. Try it, and you'll see what I mean. Learn to love the "pets."

-Jon

### Post by Erik Trybom on 2008-06-23
Let's see here. Ubuntu has the following features:

- An advanced package tool
- Gnome
- Updated every 6 months
- Highly graphical
- Lots of automatic tools

A different distro would have:

- Simple or no package management
- KDE, XFCE or perhaps no DE at all
- Updated infrequently or rolling release
- Depends a lot on text scripts etc
- Lots of manual configuration

Slackware might be a good choice, or one of the BSD's (not actually Linux, but fun anyway). If you really want something odd, check out GoboLinux.
