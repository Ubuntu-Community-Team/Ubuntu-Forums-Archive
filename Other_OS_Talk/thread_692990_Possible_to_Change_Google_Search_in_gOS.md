### Post by smartboyathome on 2008-02-11
You probably can if you find where the module is located (I can't tell you because it varies from each different package), and then copy it to your home folder and run this command in a terminal:
```
edje_decc file.edj
```
If you get an edje_decc: command not found message, I think you are up a creek without a paddle then, as you don't have the necessary tools to edit it.
