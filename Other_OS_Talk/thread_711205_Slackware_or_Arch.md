### Post by cb951303 on 2008-03-04
I used arch for 6 months and slack for 2 years.

1- Slackware is more stable than arch (it doesn't mean arch is unstable)
2- Arch's package manager and aud build system is probably the best there is

Although I used slack for 2 years I recommend arch because of the package manager ;)
