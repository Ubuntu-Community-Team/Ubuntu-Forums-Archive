### Post by kerry_s on 2008-04-29
you should be able to run any xfce4 distro comfortably.
for the best performance, a custom install with only what you want and an even lighter wm can make that machine fly.

i use a debian custom build up on 450mhz 256mb ram, nothing on mine is slow. i barely use swap.

yours is what 1.6mhz? you can get alot of speed.

xfce4->
[http://cdimage.debian.org/debian-cd/4.0_r3/i386/iso-cd/debian-40r3-i386-xfce-CD-1.iso](http://cdimage.debian.org/debian-cd/4.0_r3/i386/iso-cd/debian-40r3-i386-xfce-CD-1.iso)

net installer, if you want to install a base and build up.
