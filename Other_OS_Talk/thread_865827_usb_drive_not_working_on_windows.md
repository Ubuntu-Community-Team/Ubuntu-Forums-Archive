### Post by fiddledd on 2008-07-21
Have you tried it in a different USB slots. Not just to check for a faulty interface, but also because XP treats each slot differently. For instance, if you install, say, a USB DVB stick in one slot, then put it in another slot, XP will want to install it again for that slot. I believe this is because XP maps each external USB device to the ID of the interface it's inserted into. Sorry if that doesn't make sense, I'm not awake properly yet. :)
