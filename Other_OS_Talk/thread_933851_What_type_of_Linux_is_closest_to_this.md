### Post by Todd Gack on 2008-10-17
If you just want those apps and nothing else, here's what I would do. 

1. Download Debian netinstall iso: [http://www.debian.org/distrib/netinst](http://www.debian.org/distrib/netinst)
2. During the install, it will prompt you with what selection of packages you want. Deselect all of them.
3. Boot into your new bare-bones Debian Linux system and use apt to install the desktop environment of your choice, along with essentials like Xorg and so on. 
4. Start your desktop environment and install synaptic package manager. Build your system from there. 

I think this would be the quickest and easiest method to obtaining the system that you desire. Hope this was helpful.
