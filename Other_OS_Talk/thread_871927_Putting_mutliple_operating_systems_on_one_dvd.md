### Post by maybeway36 on 2008-07-27
It is most likely possible, but requires some tinkering with ISOLINUX and gensioimage. If you want, you can look at [my script]("http://ubuntuforums.org/showthread.php?t=718016") and see how it works. It combines distros onto one ISO image, but it doesn't have support for any big 700MB distros like Ubuntu. (It should be possible, but I don't really use DVDs so I'm no that interested in writing code for it.)
