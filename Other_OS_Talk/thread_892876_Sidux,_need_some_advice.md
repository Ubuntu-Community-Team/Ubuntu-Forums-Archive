### Post by p_quarles on 2008-08-17
> **p_quarles said:**
> [http://packages.debian.org/sid/firmware-iwlwifi](http://packages.debian.org/sid/firmware-iwlwifi)
That's the firmware driver for the card, but I forgot to mention the other part: Sidux uses Ceni, a network connection script that is quite handy. You may prefer one of the more traditional graphical tools (Knetworkmanager, etc.), but Ceni will certainly do the trick for the initial connection, so there is no need to put any of these tools on a separate disk.
