### Post by I-75 on 2008-09-25
Need more information, is it a Pentium II? What model, make? What is the upper ram limit that you can put in the laptop? In most cases, usually Pentium II have upper ram limits of 256 MB. If that is the case with your laptop. 

You might have the common 144 pin PC 100 laptop ram and maybe two spots for ram. Now on some laptops, part of the ram is on the board and the other part is one slot on the bottom, others have two slots for the ram stick.

 You can do a surrogate install or otherwise known as transplanting the hard drive. If you have another laptop, swap out the hard drives and install your favorite Linux distro that way. 

 DSL while great, it is limiting on what it can do. If you can put more ram into the laptop and if you can get it to 192 MB Ram or better then go with Xubuntu using the alternate install disc. 

I hope you don't have the Chembook 6133 or 5033 laptops. I have them and could never install any form of Ubuntu on them. 

On a positive note I have the Gateway 2550 laptop, it has 280 MB and I run Ubuntu on it with wireless, it too has a 20 GB Hard drive.

Good Luck
