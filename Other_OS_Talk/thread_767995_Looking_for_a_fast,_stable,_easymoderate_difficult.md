### Post by chronographer on 2008-04-26
If you want something fast and stable, look at debian (etch), which is the rock upon which ubuntu is built.

The downside is that debian tends to go out of date, so you can try debian testing (lenny), which is less stable, or debian unstable (sid).

I used debian etch for a while, then changed to some testing AND unstable repositories and it broke... So I now use Ubuntu which is using many bleeding edge packages but put together to some form of stability...

You either get the latest packages OR stability. Another distro I heard about is arch linux, but I don't like it as it is differently designed than debian based ones.

so... I say debian for your stable home server!
