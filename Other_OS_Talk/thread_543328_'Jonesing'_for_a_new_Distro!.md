### Post by yabbadabbadont on 2007-09-06
> **RAV TUX said:**
> 4. Gentoo liveDVD amd64 installer 2007.0

Just be aware that your first world update from the base 2007.0 install will include, glibc, gcc, and expat updates.  The glibc update will require that you rebuild world.  The gcc update takes a while, but is not a big deal (I would update gcc first).  The expat update broke the ABI compatibility with the previous stable version, so you will (again) need to rebuild most of world.  If you are serious about using Gentoo, be sure to install gentoolkit as it contains revdep-rebuild.  Which makes fixing the expat problem much easier.
