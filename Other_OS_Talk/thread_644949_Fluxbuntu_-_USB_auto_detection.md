### Post by meborc on 2007-12-19
fluxbuntu automounts usb sticks with no problem... BUT there is a known bug in the RC, you have to add .ivman directory to your home folder for ivman to function properly and automount USB devices

using fluxbuntu now, have had no problems i couldn't fix

i wouldn't suggest fluxbuntu to linux-newcomers though, it lacks gui conf utilities... it is perfect for me, as i try to run a really crappy lappy, but for some it might be too little :)
