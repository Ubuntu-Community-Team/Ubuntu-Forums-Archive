### Post by RAV TUX on 2007-07-10
My review on Linux Mint Enlightenment edition(beta):

I choose this edition of Linux Mint because I enjoy e17 so much, I really believe e17 is the best DE environment. e17 is truly a unique Linux experience(a desktop not influenced by Windows or Apple).

While I am impressed with any Debian based system that can even work on my computer(past editions of Debian and Ubuntu have failed).

So this is accomplishment #1 with Linux Mint(outstanding).

I am also impressed with any distro that utilizes e17 as their desktop, so far aside from elive, Yellow Dog, Uberyl I know of no other distros using e17. Now Linux Mint is using e17. I plan to use e17 in [Linux Nirvana]("http://LinuxNirvana.com/") eventually.

So this a honorable accomplishment #2 with Linux Mint(Awesome).

I am writing this from Linux Mint enlightment edition, while it is not as polished and set up as well as the stable version of [elive.]("http://elivecd.org")
Admittedly, Linux Mint Enlightenment edition is still in Beta, so I suspect the polish will come with a Stable release.

I did hit a bump in the road when trying to boot into the live cd, it asked for a username and password.....which I figured out the username is either, mint or root.

I could not figure out the password and on the Linux Mint forum they simple said there is no password just hit enter(like there is no spoon, I could not bend the spoon). Just hitting enter does not work. Fortunately, aysiu came up with the solution, Thank You again aysiu.

The solution is:

hit 

```
control+alt+F1
```then type:

```
sudo passwd mint
```reset the password

then hit

```
control+alt+F7
```and sign into Linux Mint Enlightenment edition

Reference these threads:

[http://ubuntuforums.org/showthread.php?t=491784](http://ubuntuforums.org/showthread.php?t=491784)
[http://www.linuxmint.com/forum/viewtopic.php?p=24902#24902](http://www.linuxmint.com/forum/viewtopic.php?p=24902#24902)

In conclusion, besides the small bump in the road, Linux Mint Enlightenment Edition(beta) is awesome, I look forward to the polish and shine. I look forward to the first stable release. I also will be studying the progress of all Linux distros that include e17 as a default desktop. Making mental notes on what to include and set up in [Linux Nirvana]("http://LinuxNirvana.com/") when I set it up with e17.

Awesome job to the Linux Mint devs....

(Note I did not test an installation of the Linux Mint (beta)....just tested the Live CD)
