### Post by Bungo Pony on 2007-10-07
So I was modding my Garage computer (again). For those who don't know, here's the specs:

- Pentium 133
- 40M RAM
- 1.2G Hard Drive
- 2 DVD ROM Drives
- USB 2.0 card
- OS: Damn Small Linux

I modded the case so there was a USB port on the front. Now, this PC doesn't have the option of booting from USB nor CD. So, for testing my new USB port, I plugged in a flash drive and then booted. I forgot that the flash drive was bootable with DSL on it. What happened was it did a system restore of my DSL hard drive install. That damn Dillo started popping up again and my background was set back to the default.

Although it was a bit of a pain setting everything back up, I thought it was a really neat feature. If I ever botch my DSL install, all I need to do is boot it with the flash drive in!
