>  Enlisy is a new linux-distribution optimized for i686 processors (p2 or better), with InitNG as init system and Apport as our package manager. We are a currently mostly aimed at people that knows or are willing to learn open source more in depth.[http://enlisy.com/en/index.php?title=Main_Page](http://enlisy.com/en/index.php?title=Main_Page)

> ** About Enlisy **

 Enlisy is an i686 optimized Linux distribution for computers with a Pentium 2 processor or better. It uses our own package manager, Apport, which is based on Libpysrc and Libpypac and is inspired by [Archlinux's]("http://www.archlinux.org/") package manager [Pacman]("http://www.archlinux.org/pacman/"). It's easy to update your programs with Apport. 
We are always trying to have the latest stable version of a package (program) available. 
Apport, and all other programs and tools we have written, is written in the programming language [Python]("http://www.python.org/"). 
Enlisy uses [InitNG]("http://www.initng.org/") as its initsystem, which is a new initsystem that is faster than the more usual system "SysV Init". 
Enlisy is a distribution that takes ideas that we like from other distributions to make the best possible distribution. For example it uses [Gentoo's]("http://www.gentoo.org/") "baselayout". That means that the base packages that is installed with Enlisy is the same as Gentoo's base packages. 
Ofcource we're not only taking ideas from other distributions, but we have ideas ourself too. 
Enlisy is NOT a beginners distribution! 
Instead, our intended audience is people who know alot about Linux and people who want to learn more about it. For beginners, we recommend using [Ubuntu Linux]("http://www.ubuntu.com/") or [Fedora]("http://www.fedoraproject.org/"). 
Enlisy is a perfect choice if you're using for example Ubuntu, but want something more difficult to play with. 
Remember that Enlisy is still being developed, so it may still contain bugs.
[http://enlisy.com/en/index.php?title=About](http://enlisy.com/en/index.php?title=About)
