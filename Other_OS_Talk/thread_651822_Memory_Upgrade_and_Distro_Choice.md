### Post by Telecaster72 on 2007-12-28
I second Zenwalk, it should run just fine on your system, on my setup it uses 65 mb ram on idle.
It has the necessary media codecs installed from start.
If you are used to ubuntu, you will miss synaptic and the huge repositorys.
It has a packetmanager that has the basic stuff though, and most slackware packets works with zenwalk.
There is more compiling from source involved, but it is easier on Zenwalk than Ubuntu in my opinion since it seems to have more libraries and dependencies installed from start.

Xubuntu should work fine also, maybe a little slower than Zen, but then you still have the Ubuntu repos (and Ubuntuforums) I have had some weird font issues with Xubuntu, but they are easy to fix if they arise by searching this forum.
