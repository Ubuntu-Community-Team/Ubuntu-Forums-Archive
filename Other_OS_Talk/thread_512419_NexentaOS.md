### Post by izanbardprince on 2007-07-29
I'm sure a lot of you already know about Nexenta, which is basically the GNU operating system using the OpenSolaris kernel, my question is why hasn't there been a lawsuit over this, since OpenSolaris is released under Sun's CDDL, and the GNU utilities, GNOME, and many other core components used in Nexenta are under the GNU GPL license.

The FSF has said that even though CDDL is a free software license, it's not compatible with GPL and GPL licensed programs can't legally be linked with CDDL ones, is there a gray area here or something?

---

### Post by smiggs on 2007-07-29
As I understand it, you can use two different licenses together just so long as long you don't rerelease the original code under a different license. What Nexenta are doing is no different from releasing closed software on Linux or GPL software on Windows. So if they need to make changes to the Gnome code they release that as GPL2 and if Open Solaris needs tweaking they'll release that under CDDL.
