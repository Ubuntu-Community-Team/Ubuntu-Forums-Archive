### Post by beyboo on 2009-02-04
Been using Ubuntu vesions since Hardy on my acer 5920 without much heartburn. Check out what wireless network card comes with the laptop. Mine has Intel graphics and wireless and things work out of the box. Broadcom Wifi and ATi GFX do cause some issues.

TO answer your question, Ubuntu, Linux Mint will all be easy to load. And I would advise since you already have Vista, try Ubuntu compatibility by loading it via Wubi. 
 
Wubi installs Ubuntu just like a Windows program, so you dont need to worry about partitions etc. If you dont like it - just uninstall from Vista !!
