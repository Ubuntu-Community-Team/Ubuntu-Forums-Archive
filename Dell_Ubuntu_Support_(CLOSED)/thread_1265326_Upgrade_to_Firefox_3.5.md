### Post by snowpine on 2009-09-13
Hi Bofak, sounds like you replaced the factory preinstalled 8.04 with 9.04 Jaunty, is that correct? If so, you can install it with:

```
sudo apt-get install firefox-3.5
```

It will co-exist side by side with 3.0 under the code name Shiretoko.

Or, you can wait a month until Karmic, when it will be the default browser.

Or, you can use one of the other methods described in the many existing threads on this topic...

ps Canonical is committed to fully support FF 3.0.x through at least April 2011; you don't NEED to upgrade if 3.0 fits your needs.
