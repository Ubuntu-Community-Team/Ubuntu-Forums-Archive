### Post by 2F4U on 2012-08-23
Frequent and unexpected crashes are often due to bad RAM. So the first thing I would do is to run memtest, for example from a liveCD or liveUSB. Let it run through several iterations to get dependable results. Another thing that you should check is if the machine gets overheated. Since it seems to be an older machine it may be worth opening it and doing some cleaning, especially dust inside the case and on the fans.
