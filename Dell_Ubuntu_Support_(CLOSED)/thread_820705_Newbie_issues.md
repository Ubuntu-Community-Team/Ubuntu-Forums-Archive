### Post by mojoman on 2008-06-08
64-bit isn't really that much of a hassle these days. Java plugin for browsers and flash is about the one thing that's lacking but there are lots of solutions to this. Java is available for konquerer (as it doesn't strictly speaking, use a plugin). Flash takes about 30 seconds extra to set up. OF course, there always the option of using a 32-bit browser, as browsing isn't really something where you'll get the upside of 64-bit anyway. 

So, is 64-bit worth it? Well, yes, if you do a lot of stuff that let you use the upside of 64-bit, such as decoding or archiving. You'll see a huge increase in speed for these thing but for most things there isn't really any difference. Of course, if you want to use 4GB+ RAM 64-bit is, I think, the only way to go.

Always use the 64-bit forum for your questions, that way you'll get your answers from people who actually use 64-bit.

/mojoman
