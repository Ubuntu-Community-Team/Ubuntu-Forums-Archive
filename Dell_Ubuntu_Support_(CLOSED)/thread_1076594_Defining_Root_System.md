### Post by Rallg on 2009-02-21
What are you trying to do? Are you trying to dual-boot Ubuntu with an existing Windows sytem? Are you trying to permanently remove Windows, leaving only Ubuntu? Does your drive contain any valuable information (files), anywhere? If you are trying to remove Windows (or dual boot), do you have Windows installation CD, in case you change your mind (the alternative is a "restore" partition without CD)?

The partition editor is a powerful tool that can obliterate whatever you have now, and let you start from virtually nothing - if that is what you want to do. But if used incorrectly, it can destroy what you have.
