grub does not modify your computer...

grub will be your bootloader & give you choices of the OS that you wish to load at boot time...

while installing ubuntu, it will give you a choice of yes / no to install grub at the master boot record (MBR). select yes (which is recommended for dual boot) and it will install grub at the MBR.

also, always install XP before you install ubuntu...
