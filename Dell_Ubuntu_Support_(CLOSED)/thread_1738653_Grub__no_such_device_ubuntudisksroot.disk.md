### Post by Rubi1200 on 2011-04-25
Hi and welcome to the forums :-)

Enable the option to see hidden files and folders in Windows and then look for a file called C:\found.000 or something similar.

If the root.disk is there copy it back to the default location in the Ubuntu folder at the root of the C drive (change to D or wherever you installed obviously).
