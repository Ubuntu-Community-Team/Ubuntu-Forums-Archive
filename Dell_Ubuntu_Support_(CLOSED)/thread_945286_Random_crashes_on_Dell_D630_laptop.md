### Post by Toribor on 2008-10-13
That looks a lot like a RAM issue to me. If you still have the Dell partitions intact you may be able to run a memory test by booting to the Dell utility partition, or you may want to look for a bootable image that will let you check your RAM for possible errors. If it happens after using the computer for a certain length of time it could definitely be heat related.
