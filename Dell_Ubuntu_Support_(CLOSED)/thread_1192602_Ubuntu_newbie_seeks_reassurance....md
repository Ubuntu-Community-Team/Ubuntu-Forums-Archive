### Post by snowpine on 2009-06-20
Don't worry, Dell and Canonical put a lot of thought into the "Dellbuntu" pre-install. It is a fine choice for the average user, with access to the most popular software (Openoffice, gimp, etc). The biggest limitation is that it uses 'lpia' (low power intel architecture) instead of the more common 'i386' architecture. So at some point, you might run into the issue where a less popular application is not packaged for lpia, but you can cross that bridge when you get to it. You might also choose to install the "regular" or netbook remix version of the current Ubuntu (9.04) as many Dell users (including myself) have done. I would double check about hardware support for your specific model (I have a a Mini 9 so my positive experience might not match yours).

As a first time Linux user, you will likely find that the pre-installed OS is a good introduction, as it is specifically designed for your hardware and all features should "just work." Check out the Add/Remove Programs feature to get a sense of all the great, free applications available. Good luck!

(edit) ps Virtualbox runs fine on a netbook, though if you have a small drive (mine is only 8gb) it might be tight.
