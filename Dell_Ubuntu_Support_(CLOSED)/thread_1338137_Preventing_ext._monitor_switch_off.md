Hi all,

I have a "problem" with my Inspiron 1520 with 8600m GT graphics card and Ubuntu 9.10: when I use an external monitor as primary device, and then close the notebook's lid, then the external monitor switches off too. How can I prevent the external monitor from switching off?

### Post by efflandt on 2009-11-27
In **Power Management** settings, there should be a setting to tell it what to do when the lid is closed.  But you might need to check CMOS settings too (that you can get into during Dell boot splash screen) to see if any settings there affect that.
