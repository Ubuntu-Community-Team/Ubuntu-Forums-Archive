The reason Dell does not include a swap partition by default is that many people feel a swap partition reduces the lifespan of a solid state flash drive.

### Post by anjilslaire on 2009-03-25
I installed 8.10 via a usb flash drive, created with the "Create a usb startup disk" utility available in 8.10 on my desktop.

For sound:
```

sudo gedit /etc/modprobe.d/alsa-base

```
At the end of alsa-base file add:
*options snd-hda-intel model=dell*

Sound will be available after reboot. Go into the volume properties and turn the Intel HDA (Sound Mixer) volume up.
