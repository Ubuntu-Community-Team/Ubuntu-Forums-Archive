### Post by Dmitko on 2012-01-01
Hi, try the ironhide project :)
In a few words: It enables the cpu embedded GPU to proxy control to the powerful GPU .
I've got an N5110 and it works great for me(UT2004 & open arena run great) :)

github page : [https://github.com/MrMEEE/ironhide](https://github.com/MrMEEE/ironhide) 
installation (search for ironhide) : [https://launchpad.net/~hybrid-graphics-linux](https://launchpad.net/~hybrid-graphics-linux)

some history : ironhide is a successor of the bumblebee project. The reason for all of this is because Nvidia seem to refuse to develop an 'optimus' driver for linux.
