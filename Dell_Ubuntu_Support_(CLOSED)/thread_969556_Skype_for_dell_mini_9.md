### Post by redDEADresolve on 2008-11-04
Ther is an easier way to install skype on your mini 9

use the medibuntu version, it works like a charm and is the newest (Version 2.0.0.72)

You can go to [www.medibuntu.org](www.medibuntu.org) to download the deb or you can install the medibuntu repository.

to install the medibuntu repo
[http://www.ubuntumini.com/2008/10/medibuntu-multimedia-entertainment.html](http://www.ubuntumini.com/2008/10/medibuntu-multimedia-entertainment.html)

then to install skype
sudo apt-get install skype
