### Post by migs73 on 2010-12-12
Try this

[http://linux.about.com/cs/linux101/g/i8kutils.htm](http://linux.about.com/cs/linux101/g/i8kutils.htm)

It is available in the Ubuntu Software Centre, just type the name (i8kutils) into the search box, click on the listing and then click install. It should allow you to use the Fn keys to allow screen brightness changes.

Good luck
