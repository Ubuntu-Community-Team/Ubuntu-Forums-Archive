### Post by Janeleaper on 2008-09-17
I think you'll find this printer is not supported. I also had a Dell 920.  I could not get the 920 to work with Ubuntu either. I'm still using it with a computer using the Windows XP OS.  I have a home-network but my Ubuntu machine will not recognise the printer even though it connects to the XP computer.

I'm not technically very knowledgeable, but I've picked up that Ubuntu prefers 'postscript' printers. My research suggests that they tend to be the pricier ones to buy new.  However, the good news is that old printers no longer supported by Windows may be supported by Ubuntu, so there are second hand bargains to be had.  I've just bought a HP Deskjet 845c for a song from a garage sale and got it working from my Ubuntu machine in minutes.
