### Post by mikewhatever on 2010-07-07
One possible reason for those issues could be errors in the downloaded isos. Try checking the isos for errors.
[https://help.ubuntu.com/community/VerifyIsoHowto](https://help.ubuntu.com/community/VerifyIsoHowto)
That said, if everything works after the installation, I wouldn't worry too much.

As for the wireless, get an ethernet cable connected for a minute, then head to System->Admin->Hardware Drivers.
