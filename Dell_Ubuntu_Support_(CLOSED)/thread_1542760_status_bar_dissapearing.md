### Post by mikewhatever on 2010-08-01
Ubuntu Netbook Remix maximizes open windows and hides the status bar by partly integrating it into the top panel. All that is intended by design to maximize the vertical screen space on netbooks' small screens. If you want the regular desktop behavior, select 'Gnome Desktop' at the login screen under Session.

### Post by jonthysell on 2010-09-02
I keep getting the same thing too, was not having it for months on 10.4. Then I changed a bunch of things at once: fresh installed with a new SSD, started using chrome, moved /var/log, /tmp to ramdisk, and probably more I can't remember.

To get it back, just press Alt-F2 and then:

```
killall gnome-panel
```

Annoying but quick. Hope this helps!

### Post by jonthysell on 2010-09-03
I found a more permanent solution, seems Metacity in 10.4 has compositing turned on by default for fresh installs, which is why I never saw it before because I had upgraded to 10.4.

Disabling compositing fixed the problem (it's a known bug):

```
gconftool-2 -s '/apps/metacity/general/compositing_manager' --type bool false
```
