### Post by MyR on 2008-06-10
greetings!

make sure you have i8kutils installed

```
i8kmon
```

for future reference, the command line supports tab-completion, meaning if you type i8k and hit tab twice it will list all of the commands starting with i8k.  if there is only one option it will be completed after hitting tab only once.
