### Post by Schpammor on 2012-09-05
So on my Dell Vostro 1000 i wanted to install xubuntu (or ubuntu).
But when i try to start the installation (tried: burned CD and USB installation medium) it hangs up in a loop printing all over this error message:

udevd[152]: timeout: killing '/sbin/modprobe/ -bv pci:v000014E4d00004315ssv00001028sd0000000Bbc02sc80i00' [253]

i  already tried all boot options i could like noacpi and all the others  offered in the startup menu and what i could find on the internet,  nothing worked. also, it was the same error with CD and USB, and with  the old xubuntu iso (12.04) and the new (12.04.1).
also, i tried to  simply upgrade to 12.04 from 10.04, which runs through fine, but at  rebooting the first time also hangs up itself in booting, this time  without error messages.

so because i found this in one post i  check what hardware the modprobe is failing on searching the pcidatabase  vor the vendor id (hex number after v: 000014E4) and device id (hex  number after d: 00004315) to get this: [http://www.pcidatabase.com/search.php?device_search_str=4315&device_search=Search](http://www.pcidatabase.com/search.php?device_search_str=4315&device_search=Search)

It  states the modprobe fails for a Broadcom Wireless Chip. So, see, i can  understand that this may be problematic, because my Laptop does not have  any wireless chip at all^^

The Vostro 1000 series can be equipped with wireless, but mine isnt.

Now my question is: How can i tell it to not try to load this driver / module / whatever this is?

---

### Post by Schpammor on 2012-09-05
got it.
digged myself through bunch of boot parameters at [http://www.kernel.org/doc/Documentation/kernel-parameters.txt](http://www.kernel.org/doc/Documentation/kernel-parameters.txt) and found this one:

pci=nommconf

from what i read elsewhere this is useful when one has a broken pci device on his bus with incorrect MCFG information (whatever mcfg is).
i used this as boot option and it fixed my problems, happily installed xubuntu 12.04 and successfully started it.
