### Post by sisco311 on 2010-12-26

> I am using** Dell Studio1558 laptop with ubuntu10.04lucid and windows7.**........the eject key is **not working in ubuntu**.........**how to enable** that??............can anyone help???

Create a new keyboard shortcut (System -> Preferences -> Keyboard Shortcuts). In the command field type:
```
eject -T
```

Please use the default font size. Thanks.
