### Post by jasil on 2010-11-18
I had the same kind of problem when installing Ubuntu 10.10 to my Dell Optiplex 980. Nothing appeared on screen when booting from the install CD.

As a workaround I used DVI cable while installing. After the install was ready, I installed nVIDIA proprietary display driver and all the Ubuntu updates. Then I switched the DVI cable back to DisplayPort cable and after a reboot all worked.
