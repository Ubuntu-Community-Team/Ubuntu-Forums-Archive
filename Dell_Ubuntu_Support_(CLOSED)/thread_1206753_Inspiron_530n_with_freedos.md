### Post by gbrainin on 2009-07-07
I suggest taking a look at the Dell Linux Wiki for possible compatibility issues.  Of course, to the extent your hardware is different from that sold with Ubuntu, you will want to check those individually.

Also, if the price difference isn't huge (or even if it is), you might want to consider "voting with your dollars" for Ubuntu.  Dell does spend some money on people to keep their systems compatible, modify software, etc., and it's good to reward them for that so they'll keep doing it.

Finally, if you want to legally view DVDs, the preinstalled version comes with a viewer, while "stock" Ubuntu doesn't.

### Post by RedRat on 2009-07-09
The Dell 530n is Dells designation for their machine with Ubuntu installed on it. You should have no problems.

Well I bought and run a Dell 530n since last year, been running it for a year and half. This machine came with Dell's version 7.10 installed. I screwed it up by trying to install Studio so I had to go ahead and do a complete clean install. Dell sent an 7.10 disk which I found out was just plain vanilla Ubuntu. I installed that as a clean install including reformatting the hard drive. Worked like a charm and later upgraded to 8.04 with few problems, mostly with the nVidia graphics driver. But since that has been worked out, the machine runs like a charm. 

I would suggest that you might want to download the Ubuntu from Dell itself, it is free. I think the Dell people do some tweaks on the drivers, etc. and a few other things. However, my vanilla Ubuntu 8.04 LTS seems to run ok.
