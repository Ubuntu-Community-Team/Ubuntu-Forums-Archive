Hello. Welcome to the Forums.

A clean install of Windows will completely over-write the hard drive unless some ground work is done first. If you want a Dual-boot system go to this page for excellent advice about partitioning. [https://help.ubuntu.com/community/HowtoPartition](https://help.ubuntu.com/community/HowtoPartition)

Next you will need either a USB flash-drive with at least 4GB capacity or a USB CD\DVD drive.
    If using a USB stick then go to Microsoft.com and download the version of Windows you want (money to be paid) and search that site for their USB Startup Creator (this works only on a Windows computer). Use a friends computer or any pc running Windows. Follow the instruction provided by Microsoft to create the Install Media. Boot your pc with the stick plugged in remembering to set the BIOS to boot from USB.

   If using a USB DVD drive it is easier.

   If you have decided to have a dual-boot system come back here for further advice after you have installed Windows
