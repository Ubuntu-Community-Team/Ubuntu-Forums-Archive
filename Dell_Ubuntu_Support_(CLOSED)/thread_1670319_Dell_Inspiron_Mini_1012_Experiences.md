### Post by Gordon D on 2011-01-18
To be honest, trying to get Ubuntu 10.10 working on this has been a frustrating and time consuming experience and I have finally had to throw in the towel and revert back to Windows 7.

I am a big fan of Ubuntu and have multiple desktop and server editions running fine, but this one is just way too much effort.

The main problem has been the wireless card. It is a BCM4312 802.11 b/g LP-PHY (14e4:4315) (rev 01). I have gone through all the driver options, B43, wl, backports, ndiswrapper, etc etc to no avail. I can't find an option that works reliably.

The main issue is the Internet connection hangs intermittently, usually when running on battery. I have a number of wireless devices running simultaneously. It is a dual boot and the wireless connection is rock solid under Windows 7.

There are also hundreds of posts out there that would seem to be solutions to this or similar problems, but sorting the wheat from the chaff is almost impossible. PC now in such a mess with hacked solutions I am probably looking at a clean reinstall anyway.

I hope people out there have have better luck than me! ;)

### Post by jnguyen on 2011-02-08
Hi Gordon!
I bought the same mini 1012 two weeks ago. It came with Win 7 Starter. I installed Ubuntu Netbook version 10.10 and every thing works right out of the box including wireless (b43xx installed & activated automatically), webcam (Cheese, the resolution is not great, but doable), the screen is default to highest resolution ( 1366x768 ), touch pad... If you don't like Unity than you can log into the Ubuntu desktop (it installed along side when you install Netbook version nicely). I really like this because I'm more familiar with GNOME look and feel ) or you can download an iso image from Dell, a crafted version for this mini 1012 called ubuntu moblin 2.1. Every thing will work out of the box with a really quick install. The interface is really flashy, really nice, but need to do some learning curve to memorise where things are. That's why I preferred GNOME desktop. I hope this info would help you with your mini 1012.
