### Post by lswb on 2009-08-22
You can try Main Menu/System/Preferences/Keyboard Shortcuts, select an action, then press the Fn key combination you would like to reset. On many notebooks, some of the Fn combos are read as actual key codes that can be assigned to shrotcuts. Others, though, may bypass the keyboard and act directly on other hardware; these will not easily be useable for other purposes.

### Post by superm1 on 2009-08-22
