### Post by anjilslaire on 2009-04-30
> **WindPower said:**
> But I have a little preference towards KDE. I know you can install most of Kubuntu's functionality by installing the kde desktop package, but it's not as sweet as a plain vanilla Kubuntu, is it?


You can get the full Kubuntu environment with all the bells & whistles from regular Ubuntu by opening a terminal and doing
```

sudo apt-get install kubuntu-desktop

```

Personally, I've has good success with Dell laptops, specifically the 1420.
