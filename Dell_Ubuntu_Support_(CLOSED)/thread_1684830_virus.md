### Post by gbrainin on 2011-02-09
Viruses are pretty rare in Linux, and are usually addressed not so much with anti-virus programs but with updates to the affected programs themselves.  So step one would be to make sure that your computer has all the latest updates.

Step two would be to try to isolate what is causing the problem.  Have you tried a different: web page, computer, web browser, or internet connection?
