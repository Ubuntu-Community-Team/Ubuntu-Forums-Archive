### Post by gbrainin on 2010-04-11
It would help to know what happened that led to this message.  Did you just install Ubuntu?  Was the display working before, and you changed something?

The message indicates that your video card has been instructed to use settings that are not supported by the monitor.  This is usually pretty easy to fix, except for the fact that you can't see what you're doing.  But you can get around that, don't worry.

First thing to do is figure out how your card got set to the wrong settings.  So let us know a little more background, and it shouldn't take long to get things fixed.

### Post by gbrainin on 2010-04-12
The first thing I would try is getting the Live CD to start up, possibly in a different graphics mode.  If you hit F4 when you get the CD menu, it should pop up a menu that lets you choose "safe graphics mode."  That mode is compatible with most hardware, and should (hopefully) let you boot.

If messing about with the settings doesn't work, you may have a peculiar piece of hardware that requires you use the alternate install CD.  That's not likely with newer hardware.
