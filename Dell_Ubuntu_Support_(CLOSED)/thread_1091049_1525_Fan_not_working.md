### Post by modena on 2009-03-10
I had the same problem.

If you want simple, try any or all of these:
- Get ubuntu 8.10. The newer kernel in 8.10 seemed to be cooler on my 1525.
- Use the 64 bit. The 64-bit ubuntu also seemed to be cooler than the 32 bit.
- Get a USB-fan cooling pad. This was the most effective solution for my laptop.

If you don't want to do any of those, then I doubt you'll find a 'simple' solution. AFAIK there is no fan control software for the 1525 for any of linux/xp/vista.  And even with the three changes above, XP keeps my 1525 cooler than linux.
