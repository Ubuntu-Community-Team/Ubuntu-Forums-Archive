### Post by elise17 on 2012-10-22
Hey, I own a Dell L502x as well, I managed to create custom shortcuts  for Key 2 and Key 3 using the pre-installed "Keyboard" settings in  System Settings without installing Ubuntu Tweaks.

I went to the  shortcuts tab in the Keyboard Settings > Custom Shortcuts, then  created a new shortcut. For me, I use Key 3 for launching gedit. So I  filled in the command area /usr/bin/gedit 

Next, you will see that it's still "Disabled", just click on the "Disabled" word and press Key 3 or Key 2. 
There you go! Hope this helps.
