Let's see if I understand, you want a driver for an undefined old Intel integrated graphics card, right? Well, there might be, but generally speaking, Intel's graphics drivers are open sourced and integrated into the kernel, so that if you have GUI desktop, the driver is up and running. It's probably worth noting that older Intel graphics were not designed with gaming in mind, and you might be better off investing in a more powerful card.
