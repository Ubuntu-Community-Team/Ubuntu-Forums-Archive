### Post by coffeeaddict22 on 2010-03-20
You can install most Linux distros.  

Best if you've got access to another Linux box.  Get an empty USB key (if you don't delete the info on it, the setup process does ;) ), and a CD image (I guess you could do it with a DVD image if the USB is big enough) to put on the key.  Then in Ubuntu go to System/Administration/USB Startup Disk Creator; that will walk you through the process of creating a USB startup disc.  You can then use it to either install or for Live CD type use, where you just run off the USB instead of installing.

And go for 9.10 if you need to download.  8.04 works, and is OK if you like stable, but it's starting to look like it came with a set of flared trousers...
