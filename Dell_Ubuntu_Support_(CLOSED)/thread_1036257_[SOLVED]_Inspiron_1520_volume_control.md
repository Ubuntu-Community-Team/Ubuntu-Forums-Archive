### Post by Radiofloyd on 2009-01-10
Fixed the issue.

I went to System > Prefrences > Keyboard Shortcuts

Selected the volume up, down, etc... and just pressed the control keys on the front of the laptop for each one.  Pause would not assign the same key as play but I'm sure it will do the same function if no... no biggie.

Hope this helps some of you guys out there!
