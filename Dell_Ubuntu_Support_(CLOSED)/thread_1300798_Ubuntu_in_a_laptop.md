### Post by jml on 2009-10-27
bear345, I have been using Linux since 1998 and Ubuntu off and on ever since the first version shipped.  I have found Ubuntu to be the most laptop friendly distro out there.  To specifically answer your question, I have successfully installed Ubuntu on the following laptops:

1.  IBM ThinkPad R40
2.  HP Compaq nx6110
3.  A no-name grey box laptop that I believe was built by ASUS
4.  ASUS 900HA netbook
5.  I also run Ubuntu 9.04 on three laptops I bought from a company called System 76.  They host a sub-forum here as well.  The laptops in question are a first generation Darter, a third generation Darter, and the Starling netbook.  They only sell computers with Ubuntu preinstalled.  You can see their products here:

[http://system76.com/index.php](http://system76.com/index.php)

In general, ThinkPads, HP and Dell business class computers work well with Ubuntu especially if you avoid models using the Broadcom ethernet and wireless cards.  Hope that this helps.

Joe
