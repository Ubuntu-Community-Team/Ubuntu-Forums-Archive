### Post by t.rei on 2010-06-12
In a terminal type:

sudo apt-get install firefox firefox-gnome-support ubufox

or:
start the synaptic packagemanager (from menu, or alt+f2 "gksudo synaptic") 
enter 'firefox' in the search field
rightclick 'select for installation' on the package named 'firefox'
hit apply in menubar
