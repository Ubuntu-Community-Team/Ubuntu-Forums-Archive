### Post by ugm6hr on 2009-11-28
You might want to check out this site: [http://www.ubuntumini.com/](http://www.ubuntumini.com/)

Essentially, 9.10 should work fine on the 10v.

As far as I know, any external USB DVD drive should work fine.  It is equally possible to install from a 1GB USB flash drive.

For printer details: [http://www.openprinting.org/show_printer.cgi?recnum=Okidata-B4100](http://www.openprinting.org/show_printer.cgi?recnum=Okidata-B4100)

If one Linux distro runs OK, most should.  How much work is required will depend on drivers etc, but the 10v is mainly Intel components, so should be OK.

PS: I have a Mini 9, which is very similar, but not identical.

### Post by undfined on 2009-12-03
My mini 10v arrived yesterday so I have only a limited amount of experience with it.  I used a new 4 GB USB stick to install UNR, then eventually the regular i386 Desktop version.  I got the 1366x768 HD graphics card which was a pain to get working correctly (search my username for the fix link). Once the graphics were proper I started to like the machine.

Ubuntu Netbook Remix was a train wreck - slow & choppy.  That may be a result of the video driver.  Either way, the regular i386 install works as it should.  Battery life is better than my Inspiron 1420 and my USB external CD drive seems to read CDs fine.

Memory upgrades are difficult.  Not an issue for me since I know how to disassemble a laptop.  Wireless worked after installing the Broadcom STA driver.

I'm liking it.
