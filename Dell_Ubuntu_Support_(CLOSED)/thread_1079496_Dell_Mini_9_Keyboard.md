### Post by dustigroove on 2009-02-25
> **musicologyman said:**
> I'm curious: Does Dell include the Windows Start menu key on the Mini 9s that come with Ubuntu preinstalled?

Yes, the Super L key (left "Windows" key) is present on the 910n.

> **musicologyman said:**
> If so, what does it map to?

Nothing by default, however within Linux you can really map this to whatever you wish it to do. Mine brings up the main menu as I have removed it from the top panel to reduce clutter/save space.
