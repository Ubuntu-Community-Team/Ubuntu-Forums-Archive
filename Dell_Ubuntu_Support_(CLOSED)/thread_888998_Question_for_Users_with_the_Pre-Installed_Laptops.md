### Post by bobbob1016 on 2008-08-13
Long answer:  By internet, I'm guessing you mean wifi?  I have a non-preinstalled Ubuntu computer and it still works.  It isn't that upgrading erases your wifi outright, it is that older methods of getting wifi working might be outdated, or not needed anymore, since the wifi support gets better and better.  If it works now, without hacking around, or using ndiswrapper or something complicated, it should continue to work through an upgrade.  Since the Ubuntu pre-installed computers come with working wifi, it should continue to work, not 100% sure, but 90-95%, and they should have good support to get it working if it doesn't work.

Short answer:  It SHOULD, since they usually use well supported cards, and it is unusual for them to take away support for a card via an upgrade.
