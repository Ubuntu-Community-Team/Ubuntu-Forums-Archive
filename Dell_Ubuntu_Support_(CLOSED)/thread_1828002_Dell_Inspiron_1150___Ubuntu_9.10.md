### Post by MARP1961 on 2011-08-22
Is your RAM less than 526MB? Ubuntu 10.04 should run just fine at 1GB of RAM but anything less than half a GB of RAM would make it struggle. It might be possible to quite cheaply upgrade your RAM to 1GB. This can make quite a difference to an older computer and can give it a useful life extension.

Ubuntu 9.10 (Karmic Koala) is now unsupported by updates and should no longer be used. Lucid Lynx (10.04) is a good, stable release and if it will not install then there must be some other problem that installing 9.10 will not solve.

If you post your computer's specifications then someone on these forums could set you in the right direction. A full Ubuntu install may not be possible if you are running it on a very old computer. I hear Puppy Linux will work on quite old hardware.
