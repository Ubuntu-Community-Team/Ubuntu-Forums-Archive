### Post by ugm6hr on 2009-04-03
> **Derekuda said:**
> By the way, I wanted to ask...I ordered with 8gb ssd and 512mb of ram.  Was just wondering what is the min req for ram and how much space does the OS take of the hard drive.  Just wondering if I see an upgrade for either in the future...

The Dell OS takes about 3.5GB.

It will run OK on 512MB (I have 1GB though, which makes it a nicer experience).  Minimum recommended is 384MB.

If you find the default (Gnome) interface a bit too slow, you can slim it down with alternatives (e.g. XFCE, LXDE).  The main upside is customization options for Ubuntu.
