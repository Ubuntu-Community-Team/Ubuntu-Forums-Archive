### Post by wgarider on 2008-08-27
Jonabyte,
I'm **REALLY** liking this! The machine I bought has the following specs - Dell Inspiron 1525/ Intel Core2Duo 2.0Ghz/ 3Gb RAM/ 250Gb SATA HD/ Intel Media Accel X3100, DVD-RW, Vista Home Premium ed. For $599 USD, I thought it was a great deal.

It is very fast - I am used to using Windows and this thing screams! I love this. Some other thoughts....I chose to start out by downloading the Dell OEM install of Ubuntu 8.04.1 - it installed easily and by using the Dell support area here, I was able to get most things working.

I need to get a few things working, then I'll call the project a complete success. Sound sucks through speakers but is quite acceptable with earbuds.., WiFi and the movie player are both not functioning....there seems to be a bit of work required to get those going.

All in all, 4.5 out of 5 stars !
