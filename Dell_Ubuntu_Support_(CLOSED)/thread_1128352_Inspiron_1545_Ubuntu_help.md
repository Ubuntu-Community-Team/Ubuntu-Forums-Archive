### Post by danwood76 on 2009-04-17
To install XP over the top simply stick in an XP install CD and overwrite all the partitions with an XP one.

Word of warning:
My friend recently bought a new dell laptop and he wanted to install several OS on it.
It came with Vista which worked fine, he isntalled Ubuntu on it which worked fine.
The problem came with XP, some of the hardware drivers had no XP versions, I would check that the hardware (like net cards and so on) have drivers for XP before you start.
