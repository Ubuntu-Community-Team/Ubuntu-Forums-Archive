### Post by Secrest on 2009-06-26
Double click the speaker icon on the top menu bar.  Make sure all the different devices have the volume turned up and make sure they are not muted.  Under the volume control for each device there is a small speaker you can click on to mute/unmute.  If you see an X then it is muted.  Try that and see if it works.
