After making that last reply, I did a lot more trolling of the internet to figure out how to speed up my install. Basically, the video card the Mini 10 uses isn't compatible with Linux at all (the GMA 500). So, I'm stuck running at whatever resolution I'm currently at.

Which brings me to the latestp roblem I'm having. Last night I managed to get UNR sped up by changing the default mode to traditional Ubuntu so that instead of the very graphically intensive menu system I just have the two bars. Everything was going really great - I bought this netbook for writing and surfing the web, and nothing more extravagant than that, so I was very pleased. I set up a few wallpapers, turned the Mini off and went to bed.

I woke up today, sat down to write on the Mini, and now I don't have any panels. I've tried rebooting a few times but it doesn't do anything. Clearly, things are working - sort of - but it's very strange. I'll post again when I've got this sorted out.
