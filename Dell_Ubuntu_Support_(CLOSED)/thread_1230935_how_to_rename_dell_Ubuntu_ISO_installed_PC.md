### Post by gbrainin on 2009-08-04
This is not a problem with the Dell ISO.  All versions of Ubuntu from 8.10 on do not have the Network tool by default.  However, you can install it.  It is called gnome-network-admin.

Also, when you install Ubuntu on the second machine, there should be an option for entering the machine's name during the installation.
