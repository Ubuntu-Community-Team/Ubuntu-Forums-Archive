### Post by ghingataff on 2010-05-05
Hi there.
My GF just got an old Latitude D600 from her work place and we decided to install Ubuntu (10.04). The installation was fine and the only problem was the 'wandering cursor', which I know has been covered before. 
I found a thread today where someone suggested removing the keyboard and disconnecting the cable to the pointer (unfortunately I now can't find that thread). 

I did this (although the end of the cable looked a mess so I trimmed the end off and taped it up) but for some reason the computer no longer connects to the internet.

FWIW our main computer is an iMac and until now just plugging the laptop into the same router works fine without any kind of setting up.

Is it possible that the disconnected cable has caused a hardware problem or could it be something that I can resolve with the software?

---

### Post by ghingataff on 2010-05-05
Ok, problem solved!
It seems that something caused the automatic ipv6 setting to be changed to 'ignore' so changing it back to automatic fixed everything.
It was just too easy ](*,)
