### Post by lukjad on 2009-10-19
First thing you should to is back up your /home folder so that if you have any data in there it is backed up somewhere. If you already have backed it up, or you really don't have anything worth saving, do the following.

First, reboot, and get into the BIOS. Check to make sure the CD/DVD is the first boot drive. 

If it is, then simply reboot again with your XP disk. Delete the partitions, then format it to NTFS (slow). Install XP there.

If you are installing Vista, use the Ubuntu disk you have to delete and resize your partition(s) to create a primary and an extended partition. Install Vista in the Primary Partition, and then install Ubuntu in the extended partition. Any more questions, just ask.
