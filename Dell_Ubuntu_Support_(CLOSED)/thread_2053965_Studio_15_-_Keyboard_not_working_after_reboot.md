### Post by mombius on 2012-09-06
Hi, I was playing to Wolfenstein: Enemy Territory with my Dell Studio 1537, and the laptop just shutted down unexpectedly. It happens sometimes when it gets very hot, so I just turned it on again. The problem is that now the keyboard does not work. I've logged in with the on-screen keyboard (USB mouse and trackpad do work), but it doesn't work after that either. (By the way, it's Ubuntu 12.04) Not only that, but it behaves as if there was some button pressed, so that I cannot navigate the menus correctly with the mouse. I have no idea what to do now. Any help would be appreciated. Thanks!

### Post by mombius on 2012-09-06
Well, fortunately I've found it's a common problem and has a very simple solution!

Shut down computer, unplug everything, remove battery and press power button for 5 seconds or so. Voilà!!

(Got it from: [http://en.community.dell.com/support-forums/laptop/f/3518/t/19246719.aspx](http://en.community.dell.com/support-forums/laptop/f/3518/t/19246719.aspx))
