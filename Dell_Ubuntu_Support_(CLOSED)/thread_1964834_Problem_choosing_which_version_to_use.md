### Post by Lars Noodén on 2012-04-24
Use the newer version, for sure.  The next version will be out on Thursday, but if you want you can download the beta and update from that:
[http://releases.ubuntu.com/precise/](http://releases.ubuntu.com/precise/)

What will make a difference is the Desktop Environment or Window Manager you use.  Several versions of Ubuntu have different defaults.  Ubuntu defaults to Unity, Kubuntu defaults to KDE, Xubuntu defaults to XFCE and Lubuntu defaults to LXDE.   Of those, Lubuntu is the lightest.  A cautious way to try them would be to use the Live CDs before deciding which to install.

As with any of them, you can go the the software center or Synaptic and pull in any packages from the other versions that you might like to have.  Any distro can be made into any other by adding or removing packages and changing the configurations.  However, you can try the pre-configured ones provided by Ubuntu, Kubuntu, Xubuntu, or Lubuntu.

### Post by jerrrys on 2012-04-24
If you want to try other desktops you can add them to ubuntu.

For Lubuntu [open a terminal]("https://help.ubuntu.com/community/UsingTheTerminal") and enter:

sudo apt-get install lxde

For Xubuntu:

sudo apt-get install xfce

You will be able to choose a desktop at log in and welcome to the forum :)
