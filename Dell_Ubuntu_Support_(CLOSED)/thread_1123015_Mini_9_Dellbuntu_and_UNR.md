### Post by joey-elijah on 2009-04-14
> **Cowchip7 said:**
> I am getting my mini 9 Tuesaday and I guess I am still on the fence as to whether or not I am going to install the vanilla version. I guess I have a week to play around with Dell's version.

There is just some small things that I know will turn me off... like the Yahoo junk (which I know I can delete) and the generic icon for firefox. Is there an easy way to change the internet icon to the default firefox icon?

Yes, just go to to system > preferences > main menu > find the web browser and click on the icon, this will open a window to select a new icon to replace it. Save the [firefox logo]("http://en.onsoftware.com/wp-content/uploads/2008/06/firefox-logo.png") somewhere and then navigate to it, select it and click okay. You may have to log out/back in for it to take effect.

### Post by Brandon Williams on 2009-04-18
If you like to tinker with the computer, install Jaunty. If some specific piece of software isn't available in the Dell repos, consider switching to Jaunty ... just check to make sure the software is available there to. If you don't want to tinker and the Dell repo has all the software you need, then I suggest sticking with the Dell version.

I like to tinker, so I run Jaunty (the lpia version). I stuck with Dell's version for about 6 weeks, and found it to be perfectly adequate functionally.
