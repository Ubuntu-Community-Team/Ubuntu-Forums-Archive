OK first, I have one and it works great with Ubunto 9.04 and 9.10. I cant comment on 10.04 except that the liveCD takes 9 minutes to boot up and 9.10 works fine.

What you see isnt a problem. I've gotten past that many a time. So lets go on.

Have you been able to boot from the liveCD? I mean the regular ordinary Ubuntu i386 CD, not a 64 bit version.

Later when you want to install you can make sure your had drive is seen. It will be called auto I think. But for now, press F1 to go on and see what happens... if it fails then... Go into Bios and make sure the CD is first in the boot order. Then try again hitting f1 to bypass the "seeing the HD problem."
