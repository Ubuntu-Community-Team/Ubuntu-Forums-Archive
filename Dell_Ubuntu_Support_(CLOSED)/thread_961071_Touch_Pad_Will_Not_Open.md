### Post by ytsejam1138 on 2008-11-23
Ubuntu 8.10 or 8.04?

If Ubuntu 8.04

Just add
Option &#8220;SHMConfig&#8221; &#8220;on&#8221; in /etc/X11/xorg.conf

Section &#8220;InputDevice&#8221;
Identifier &#8220;Synaptics&#8221;
Driver &#8220;synaptics&#8221;
Option &#8220;Device&#8221; &#8220;/dev/input/mice&#8221;
Option &#8220;Protocol&#8221; &#8220;auto-dev&#8221;
Option &#8220;Emulate3Buttons&#8221; &#8220;yes&#8221;
Option &#8220;SHMConfig&#8221; &#8220;on&#8221;
EndSection

Now Reboot and you should be able to start Touchpad.

If Ubuntu 8.10:

In terminal type:

gksudo gedit /etc/hal/fdi/policy/shmconfig.fdi

Then add the following to the file:

    <?xml version="1.0" encoding="ISO-8859-1"?>
    <deviceinfo version="0.2">
    <device>
      <match key="input.x11_driver" string="synaptics">
       <merge key="input.x11_options.SHMConfig" type="string">True</merge>
       </match>
    </device>
    </deviceinfo>

Now Reboot and you should be able to start Touchpad.  Also, if you would like to add tap areas to your Touchpad in Ubuntu 8.10, see my post here:   [http://ubuntuforums.org/showthread.php?t=987766](http://ubuntuforums.org/showthread.php?t=987766)
