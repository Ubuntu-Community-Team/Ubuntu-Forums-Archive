### Post by teledyn on 2009-06-10
This thread is for all of us who bought Dells based on the AMD/ATI *fglrx* graphics chipsets such as the Radeon X1200 found in the Dell Inspiron 1521 laptops; thanks to ATI, we are forever bound to the Ubuntu 8.04 release where hardware accelleration for our chipset was still included.

And it is not just ATI graphics, the real-time kernel essential to Ubuntu Studio is also incompatible with the Dell hardware in the 8.10 and 9.04 releases.

And that means we'll have to work together if we want to stay abreast with future free software developmen. I thought it might be useful if we had a forum to trade tips on backporting packages, trade backport binaries, or report on any success strategies with the free versions of the Radeon drivers and the RT kernel with the Dell hardware.

### Post by teledyn on 2009-06-10
For those who don't already know, there is a page where you can request specific packages for back-porting to 8.04.  First, you will need to read [Ubuntu Backports]("https://help.ubuntu.com/community/UbuntuBackports") and set your /etc/apt/sources.d to include a repository line like

```
deb http://ca.archive.ubuntu.com/ubuntu/ hardy-backports main restricted universe multiverse
```

then you can enter a report in the [Hardy Backports Request]("https://launchpad.net/products/hardy-backports/+filebug") tracker.
