> is it fast on 1 gig ?

No, the Dell Mini is not a "fast" computer. :) It uses an Intel Atom processor that's comparable to a Pentium 3. It is great for low-power activities like checking email and surfing Ubuntu Forums. 

You can't upgrade "Dell Ubuntu" to 8.10. You will have to do a fresh install of "regular" Ubuntu if you want 8.10 or newer. For a new user, I recommend sticking with 8.04 for the time being. 8.04 is a "long term support" release, so it will be supported through April 2011 (support for 8.10 ends in April 2010).
