### Post by HotShotDJ on 2009-07-16
I checked OpenPrinting.org and did not find a listing for this printer.  Generally speaking, Dell printers do not work outside of the windows world.  While there are some exceptions, the AIO 948 does not appear to be one of them.  Dell is aware that most of their printers are incompatible with Linux, and there is no sign that they have any intention of fixing that problem.  Unfortunately, there is little that Linux developers can do about it.

See also: [https://bugs.launchpad.net/ubuntu/+bug/91270](https://bugs.launchpad.net/ubuntu/+bug/91270)
> These printers use a properietary protocol and Dell does not provide Linux printer drivers for these printers.
So there is nothing Ubuntu can do about this.
You should report this to the manufacturer of the product!
Closing this bug as won't fix but in reality it is "can't fix".
