### Post by rehilliard on 2009-06-12
It's really up to you. Do you want the "convenience" of a netbook or the performance of a laptop. Personally, I'd get an iphone or blackberry or pre and "pretend" that's a netbook and go for a real laptop. 

Anyway, as for buying a new router, I'd say no. If it's working right now, and the connection speeds are good for you, then stay with it. As for connecting the two pcs...well, ubuntu "out of the box" can connect to a shared folder on windows by going to Places-->Connect to server menu. Just share a folder on the windows PC and you can use this to copy files to and from. Windows to Ubuntu...you'd have to run Samba on Ubuntu to "present" a shared folder so that Windows can see it. Although, you could also install FTP server software on Ubuntu and use that way to transfer files. That kind of thing. It really depends on what you mean by "connect the two PCs".

It appears that your computing needs are fairly straightforward, so (unless Powerpoint is involved!) throw out that Windows PC.

### Post by ugm6hr on 2009-06-13
> **Janeleaper said:**
> My partner does use Powerpoint. He's had a look at Open Office Presentation and thinks it will be ok.  What do you think is wrong with it? This is important, because he is a college lecturer and uses PP a lot.

Depends.

I use OO.org Impress to write presentations, and it works fine.  I don't use animated slides though.

Additionally, I do still have to give presentations at work, and I don't always want to take my laptop into work with me; hence the actual presentation is often done on Windows (MS PowerPoint).  I have never had a problem with basic formatting with this transition.

Nevertheless, I always take the presentation as a PDF too (easy to do from Impress), which ensures compatibility and maintains formatting; it does remove all animations though.

However, I do still have to use Windows from time to time, since I am still unable to adequately loop short (e.g. 1sec) avi files flawlessly on Linux, which PowerPoint manages with no problem.

So best to check what kind of things he requires out of PowerPoint.  Dual video output support is much better now than it used to be, so plugging into a projector should not pose problems.
