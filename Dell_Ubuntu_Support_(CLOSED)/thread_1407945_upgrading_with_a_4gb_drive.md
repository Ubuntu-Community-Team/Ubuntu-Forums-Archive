### Post by Cabs21 on 2010-02-16
USB sticks are non-volatile memory
[http://en.wikipedia.org/wiki/Non-volatile_memory](http://en.wikipedia.org/wiki/Non-volatile_memory)
meaning when power is turned off the data that is there will still be there when power is returned.  Hard drives are a good example of this.  Also since USB sticks are flash memory they are rewritable.  Yes there is a limit to the number of times flash memory can be written to because each time the memory is rewritten it wears down but it usually takes close to 10 years of use before the Flash memory breaks down and stops working.  In 10 years Flash memory could be a dead technology.
