### Post by jarviser on 2010-06-15
I assume you have version 10.4 Ubuntu. Or version 9 at least. 

Right-click on the connection icon , then Edit Connections. Then Wireless Tab. Ensure the settings on the connection for that router's SSID are correct, especially the security is WPA + WPA2 Personal.

If that is no good, delete all the wireless connections, reboot, then try to connect by left click on the wireless icon and select the  SSID.

If still no good check if the wireless card has Broadcom chipset and if so search on here for Broadcom driver fix.
