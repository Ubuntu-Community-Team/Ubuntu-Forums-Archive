### Post by hypetech on 2010-04-06
I just installed Ubuntu 9.10 on my Dell Inspirion 1545 laptop.  I'm having a strange problem where just hitting the function keys (F1-F9) toggles the alternate function (brightness, volume, wireless, etc.) instead of acting as a function key.  Anybody know of a fix for this?

Edit: Erm, actually after closer inspection I suspect this is intended behavior as the F1-F12 button functions are in blue, so I guess standard operation is actually the alternate function on this keyboard....is there any way to flip flop this?  Seems rather silly.

---

### Post by hypetech on 2010-04-06
Found the answer for anyone having this same issue.  There is an option in the BIOS to switch this.  Hit F2 on boot to get to Setup.  Go to System Configuration -> Function Key Behavior, and select "Function Key First".
