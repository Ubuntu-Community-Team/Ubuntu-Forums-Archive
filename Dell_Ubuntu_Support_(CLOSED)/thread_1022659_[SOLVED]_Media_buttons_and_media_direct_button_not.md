### Post by suxenexus on 2008-12-27
When ever I wanted to configure the dell media direct button, kubuntu detects it as the 'back' button, the stop button as the 'volume button' even if it is detecting the real volume down button.  Also, when I try configuring the amarok 2 player's previous and next track using the 'media previous' and 'media next' buttons, respectively, it doesn't work.  Any suggestions?


I'm using dell inspiron 1420 laptop right now.

Since no one helped me, I tried looking for solution.  And here it is!

Go to the Kmenu>System Settings>Regional & Language>Keyboard Layout

you should change the model to Dell Latitude Series Laptop.  Now my media direct button works!
