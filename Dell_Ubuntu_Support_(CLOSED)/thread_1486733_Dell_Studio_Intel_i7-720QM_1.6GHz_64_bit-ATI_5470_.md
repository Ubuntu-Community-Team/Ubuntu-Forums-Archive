### Post by mellowtothemax on 2010-05-19
If you have 4GB or RAM or more then you should install the 64-bit version otherwise you will only see 3 or 3.5 GB used.  32-bit OS does not support more RAM. 

Most 32-bit Linux applications have a 64-bit installers.  And some of the ones that don't work anyway with 32-bit.

If you need applications that only have 32-bit installers (look them up before you decide) and/or you don't mind about the RAM then stick to 32-bit.

Apart from that performance difference is minimum for most tasks.
