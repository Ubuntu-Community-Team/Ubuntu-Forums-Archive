### Post by madallig on 2011-03-20
Just purchased Dell Vostro 3500 -380S-W7PRO  i3 CPU in Hong Kong
Here's the input so far:
1. Ubuntu 10.10 installs with no problem
2. The propriety broadcom driver installs with no problem
3.  The propriety  nvidia driver is NOT installable for this unit - DON'T  in stall - otherwise you will not be able to boot into Ubuntu correctly
4. When turning off BT, the BT light at top of keyboard does not turn off 
5.  Dell will fully support Dell Vostro 3500 if you believe the comments on this link:
[http://www1.ap.dell.com/in/en/business/notebooks/vostronb/cp.aspx?refid=vostronb&cs=inbsd1&s=bsd](http://www1.ap.dell.com/in/en/business/notebooks/vostronb/cp.aspx?refid=vostronb&cs=inbsd1&s=bsd)
6. Battery life is not bad - got two hours at least when out and about with basic web browsing using WiFi
7. If you buy same unit - make sure you get screen with non gloss finish - much better on eyes!
8.I have found no driver support for multi-touch gesture track pad for its special features 
9. Webcam works with no problem with no configuration required - mic also no problem
10. When using audio function on skype on imo.im - cannot get mic to work via flash or similar
11. Big problem to install Brother DCP-J125 on 64 bit system - any help welcome
