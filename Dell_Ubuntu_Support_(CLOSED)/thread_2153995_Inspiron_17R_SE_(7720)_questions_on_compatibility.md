### Post by GralsPlayer on 2013-07-13
I've got an Inspiron 17r se running with Ubuntu, during the installation there were no problems, you just have to disable the UEFI secure Boot and use the Legacy Bootmode. If you want to use Windows and Ubuntu you will have to do the partitions manually. To boot into Windows 8, just press while the Dell Logo is on the screen F12 and choose the Windowsbootmanager. The function-keys are working well, to use the trackpad with gestures or the nvidiagraphics, you will have to install the drivers.
