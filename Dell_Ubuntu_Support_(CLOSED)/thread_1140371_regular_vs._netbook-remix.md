### Post by snowpine on 2009-04-27
> **2roombas said:**
> What is the difference between installing the regular 9.04 vs. the netbook-remix on my Dell Mini 9?

A different interface. "Regular" uses a classic Gnome desktop; NBR has an iphone type of interface with big icons (optimized for the small screen). NBR also is missing a few applications (to save disk space) but they are very easy to install after the fact if you need 'em.
