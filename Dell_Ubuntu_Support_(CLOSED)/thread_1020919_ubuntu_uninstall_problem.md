### Post by balaknair on 2008-12-25
Errr. You just deleted the Ubuntu partition? I'm assuming you used the Vista DVD or EasyBCD to fix the MBR so that it could boot again. If not, improperly rebuilt partition tables could cause system errors(though not shutdowns as you are experiencing).

Well, the only changes that a properly done Ubuntu installation makes that could affect Vista performance are restricted to the MBR, which apparently is fixed now, since you can boot into Vista. So I don't think it's the Ubuntu installation that is causing these problems, and agree with llamakc.

Try using a Vista DVD- boot into the DVD and repair the installation. If that doesn't work, you'll probably be better off with a reinstall.

Before you reinstall Vista, you might want to check your hardware, especially the RAM, graphics card(unlikely, since most laptops have onboard cards), and the hard drive. Is the laptop making any odd noises or loud fan noises or heating up?
