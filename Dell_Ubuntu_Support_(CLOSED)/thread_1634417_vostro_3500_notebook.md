### Post by mauricio.dev on 2011-01-24
I've recently found problems getting the wireless to work. After many (MANY) builds and how-tos I got it working with ndiswrapper as some happy user suggested.
So i sugest you to try the ubuntu's restricted driver and if it gets the same problem I had (terrible latency with about 0.2s delay) just jump to ndiswrapper.
I've read in a few other bugtracks that broadcom drivers are a problem in linux for a long time. 
I've even tryed to compile the opensource alternative to the broadcom driver but it also got (much softer) latency problems. And my FPS games were unplayable.
At least I got it working with the windows driver.
Good luck!
