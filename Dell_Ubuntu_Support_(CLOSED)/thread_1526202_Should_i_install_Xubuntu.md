### Post by Yarui on 2010-07-07
If you like it I don't see any reason not to try it out.  The only difference is that it uses xfce instead of gnome.  Pretty much everything like java and flash should work the same way as Ubuntu and other Ubuntu derivatives.

### Post by bobcollard on 2010-07-07
Xubuntu is light or as heavy as you want it to be.  It will take any application you want to run on it including those from the Gnome and KDE desktops.  Try live CDs of the different desktops before making a choice.  I like Xfce because it has none of the problems associated with changes in KDE separate from the system and I don't care for the menus in Gnome. to each his own.  Make an informed decision then stick with it awhile until you learn all the ins and outs of the desktop.
