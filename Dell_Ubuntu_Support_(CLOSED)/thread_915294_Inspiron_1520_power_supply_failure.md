### Post by toaste on 2008-09-09
Yes, it's generally safe to use any Dell adaptor with any other Dell system.

The BIOS will detect 65W adaptors on systems that can require more power, warn you, and run the system in low power mode (your CPU is placed in the lowest power state).  You can also charge the battery with a 65w adaptor with the system powered off with no problems.
