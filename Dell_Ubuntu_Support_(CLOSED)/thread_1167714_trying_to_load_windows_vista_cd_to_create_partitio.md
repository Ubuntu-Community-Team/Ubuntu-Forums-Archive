### Post by ugm6hr on 2009-05-23
Couple of things:

Intrepid is not the latest Ubuntu (Jaunty 9.04 is).

Have you tried booting the computer with the Vista DVD in the drive and boot priority set to CD first?  If yes, how far does it get?

If it still does not boot, try asking for a new Vista DVD from wherever you bought the computer from.

Installing Windows after Ubuntu can be troublesome; you need to use a Linux LiveCD with GParted (e.g. Ubuntu) to shrink the Ubuntu partition and create a new primary partition for Vista before you boot the Vista DVD.
