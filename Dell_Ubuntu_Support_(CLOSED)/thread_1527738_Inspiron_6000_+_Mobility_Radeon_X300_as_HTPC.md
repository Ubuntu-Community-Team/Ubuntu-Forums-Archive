Last night I installed Mythbuntu 10.04 on a Dell Inspiron 6000 with (I think) Mobility Radeon X300 (I definitely know it's an ATI chip and not an Intel one, but I need to boot back to XP to double-check the model). For one thing, the X300 is apparently no longer supported by the proprietary ATI drivers. When I use the atitvout package with the open source Radeon driver that is installed by default, it gives VBE call failed messages for almost all functions.

I'm also trying to use it with a composite video + coax audio breakout box from my Dell M1730, and it outputs a picture on my TV in both Windows XP and Mythbuntu - but the picture looks bluish, so I'm afraid that the X300 may not support that breakout box. I haven't tested the coax sound - if that works then maybe I could use the VGA out for video instead of component (would rather avoid going down to S-Video or composite). Digital sound output is a must because I have a 5.1 sound system and would like to do SPDIF passthrough playback.

It's looking like this Inspiron 6000 is not going to work out for me as an HTPC :\ Anyone have any thoughts/advice?
