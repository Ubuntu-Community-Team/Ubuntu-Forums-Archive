### Post by coffeeaddict22 on 2009-06-22
Hi,
I think the first bit of advice would be:
Calm down
take a big deep breath
Start again.

Firstly, Linux is a constantly changing OS.  You'll have been using an OS where once every year or two (or three, or four...) the system changes the way it works.  Linux is on a constant upgrade cycle, so no-one can give you advice on how to fix something if you haven't decided what to install.

As a rule, if you're not on a production machine, use the latest version.  Plain Ubuntu has the biggest installed userbase, so if you ask a question you're more likely to find knowledgeable help.  However, K, X or whatever 'buntu you're into is fine; just pick one.  For fewest problems on a laptop, most people would recommend 9.04 as the wireless has improved a lot lately; however, if you're running an Intel graphics card, you may be better off on 8.10 for now.  The cooling fans were a problem with older kernels; it's fixed in the latest ones.  If you have cooked your PC, well, Linux is hot but it's not oven-proofed :)

Once you've done that, run a Live CD.  Download the .iso file of your choice, cut a CD, and boot up from it.  It gives you a chance to look at what Linux will be like without putting anything on your hard drive; if you're going to have major issues, this is a painless way of finding out what the hassles will be.   Once you find a distribution that works for you, install it.  

If it works, good; if it doesn't, identify as well as you can what the problem is, then post back.  "Doesn't work" is a little more difficult to troubleshoot than "the display looks garbled" or "the sound and video keys on the keyboard aren't working", so you're much more likely to attract help if you can specify the problem (and, as always, knowing the question is about 50% of finding the answer :-) )
