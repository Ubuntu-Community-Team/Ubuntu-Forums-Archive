### Post by mikewhatever on 2012-03-22
The server edition might work for you, it only requires the following specs:

300 MHz x86 processor
128 MiB of system memory (RAM)
1 GB of disk space
Graphics card and monitor capable of 640x480
CD drive

[https://help.ubuntu.com/community/Installation/SystemRequirements#Ubuntu_Server_.28CLI.29_Installation](https://help.ubuntu.com/community/Installation/SystemRequirements#Ubuntu_Server_.28CLI.29_Installation)

For Minecraft, you probably want Oracle's Sun Java.

[https://help.ubuntu.com/community/Java#Oracle_.28Sun.29_Java_6](https://help.ubuntu.com/community/Java#Oracle_.28Sun.29_Java_6)
