### Post by pmsakkas on 2008-05-20
About a couple of weeks ago I installed Hardy in my Dell Optiplex 745. After that I started having problems with my Internet connection. Obviously there was a problem with my Ethernet interface, because I get Internet service in my office directly from the University's sockets I study.

The solution:

After upgrading from the old BIOS (2.1.0 or so) to the new one (2.6.1, [http://support.us.dell.com/support/downloads/download.aspx?c=us&l=en&s=gen&releaseid=R174121&formatcnt=1&libid=0&fileid=236439](http://support.us.dell.com/support/downloads/download.aspx?c=us&l=en&s=gen&releaseid=R174121&formatcnt=1&libid=0&fileid=236439)) using the Windows XP installation I still keep just for safety (having used the wubi.exe option for installing Ubuntu). I only had to run the .exe program and the system restarted, updated the BIOS and booted with the Windows/Ubuntu selection page. I chose Ubuntu and started using the Ethernet without any problem.

I hope this helps.
