I'd heard blu Ray was not working, but it seems that I've been misinformed.

[https://help.ubuntu.com/community/RestrictedFormats/BluRayAndHDDVD](https://help.ubuntu.com/community/RestrictedFormats/BluRayAndHDDVD)

[http://apcmag.com/how-to-play-blu-ray-in-linux.htm](http://apcmag.com/how-to-play-blu-ray-in-linux.htm)

[https://www.ebower.com/docs/ubuntu-bluray/](https://www.ebower.com/docs/ubuntu-bluray/)

Hardware wise Nvidia is better supported than ATI at the moment. Gnome 3 won't even run on some ATI cards at this moment in time as I've found out myself.

I've run the fusion chips from AMD that don't work straight away on Windows yet works OK on Linux.

Hope it all works out for you. If all else fails run a Win 7 VM as long as you don't mind only have W7 classic interface. Aero does not seem to be supported yet in virtualbox.
