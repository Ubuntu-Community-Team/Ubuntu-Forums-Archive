### Post by RTrev on 2011-01-30
I don't recall ever seeing a CD/DVD drive which didn't have a manual eject feature.

Look very carefully all around the door to the drive for a tiny hole, about the size a straightened paperclip would fit into.  Pushing the paperclip into that hole should release the mechanism and the door should pop out enough for you to open it the rest of the way and get your disk out.

HTH,
Bob

ETA: Sorry, missed the part where you said there were no little holes.  All I can say is look again.  If all else fails, perhaps boot a USB linux distro and get to a terminal so you issue an eject command.  Or go into the BIOS and change the boot order so it doesn't try to boot from the CD.
