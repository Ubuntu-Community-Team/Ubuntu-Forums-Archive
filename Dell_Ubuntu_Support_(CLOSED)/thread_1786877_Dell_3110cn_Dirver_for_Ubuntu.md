### Post by lviggiani on 2011-06-20
Hi Everybody,
after digging around for a while I discovered that the Dell 3110cn CD Driver from Dell comes with a .rpm package for linux.
I've used alien and converted into a .deb and installed on my system and it works perfectly.

```
sudo alien -k Dell-Color-Laser-3110cn-1.0-1.noarch.rpm
```

I'm posting the deb for everybody's convenience.
Cheers, Luca.
