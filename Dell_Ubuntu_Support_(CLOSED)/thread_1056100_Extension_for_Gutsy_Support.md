### Post by snowpine on 2009-02-10
You can certainly keep using Gutsy after April, you just won't receive any more updates. You could install newer versions of mission-critical applications (for example Firefox) directly from source if you wish. 

Also, the Gutsy repositories won't go away--they'll just move to old-releases.ubuntu.com. If you update your /etc/apt/sources.list file to reflect the changes, you will still have access to the Gutsy repos.
