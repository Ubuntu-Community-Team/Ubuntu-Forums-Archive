### Post by JoshStrobl on 2009-08-19
download Wcid Manager. In order to do so go to System> Administration> Synaptic Package Manger.
Type in Wcid Manger, click on it then go to Network manager and right click> Click For Removal. Apply Changes and it'll work 
Do Not Remove Network manager before trying to install WCID manager or you won't be able to downlaod anything. Do it at the same time by doing what I said above.
