### Post by ActionpackedPIVII on 2012-08-13
I would say a Lubuntu or Xubuntu. 

They both offers fully up to date software and is quite a lot more light-weight than Ubuntu with the unity interface. 

Although standard Ubuntu might run decently well on your system, it is without question that you will be using less ram and probably less CPU power by running L/Xubuntu.

Lubuntu is from my experiences the most light-weight quickly followed by Xubuntu.

However Kubuntu (ubuntu with the KDE environment) can actually run very well, if you deactivate desktop effects and/or install the kubuntu-low-fat package
([http://www.omgubuntu.co.uk/2011/09/kubuntu-low-fat-settings-speed-boost-older-hardware](http://www.omgubuntu.co.uk/2011/09/kubuntu-low-fat-settings-speed-boost-older-hardware)) (works on Kubuntu 12.04 aswell)

Hope this helps you a bit :)
