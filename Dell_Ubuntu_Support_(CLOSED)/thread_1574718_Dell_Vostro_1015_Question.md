### Post by ajgreeny on 2010-09-14
Not in one jump, you can't, even if it's possible with the Dell disk, which I doubt.  You also need the Alternate Install Disk to do a CD distro upgrade.  The live CD will not work that way, as far as I'm aware.

It is only possible to update one version at a time, except for the LTS versions, so you could go from 8.04 to 10.04 as both are LTS versions, but you can only go from 8.10 to 9.04 (for another month, then 9.04 is no longer supported), 9.04 to 9.10, and finally 9.10 to 10.04.

I shouldn't bother if I were you; it will be much simpler and quicker to backup all your data and do a clean install.
