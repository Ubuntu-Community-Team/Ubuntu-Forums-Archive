### Post by mjitkop on 2010-01-05
Hello, renjumace.

The Live CD does not use the hard drive so even if the hard drive is damaged, you will still be able to use the Live CD. Were you able to see the hard drive using the Live CD?

I'm not sure a BIOS update would make the hard drive unusable.

I am not aware of any Linux application that you can use to change the version of a BIOS. I would be interested to see if someone finds one.

If your laptop is still under warranty, I would strongly suggest that you let Dell handle the problem. I have the feeling that it is not under warranty anymore.
