### Post by notwen on 2008-05-15
You are very capable of running 64 bit, but unless you specifically know you want it and would use it, I wouldn't bother.  There are some issues w/ 64 bit and lack of certain packages available to the 64 bit versions of Ubuntu.  They are getting better and fewer and fewer issues are coming up last I checked, but you may want to ask someone who uses 64 bit and can confirm this for you.

As far as which install you should download, I'd try the LiveCD first, if that doesn't work then move on to the alternate install CD.  Wish I could give more insight, but I haven't dabbled too much w/ the 64bit versions of Ubuntu.  =]
