### Post by superm1 on 2009-04-28
Hi everyone:

Dell now publishes the open source components of the framework used for loading Ubuntu onto PCs in the factory.  If you would like to create a Dell/Ubuntu ISO that fully emulates this process, the instructions for creating such an image are available here:

[http://en.community.dell.com/blogs/direct2dell/archive/2009/11/09/dell-recovery-tool-enhancements.aspx](http://en.community.dell.com/blogs/direct2dell/archive/2009/11/09/dell-recovery-tool-enhancements.aspx)

The tool requires that you use an Ubuntu DVD to generate the ISO, so that it can pre-install the closed source graphics drivers when necessary.

You can burn this ISO to a DVD using your favorite burning software or to a USB stick using USB Creator.

The Automatic Reinstallation option will remove all partitions on your primary hard disk and create Linux Utility & Recovery partitions.


If you want to install Ubuntu side-by-side with another OS (Such as Win7 or Win Vista), you can choose interactive install to choose how to partition the disk.  You will still get the other Dell addons such as flash installed, but without losing data.

### Post by seaoffecundity on 2011-04-11
> **VicFerrari said:**
> I want a netbook with Ubuntu on it <snip>

  you are probably aware that dell offer some systems with ubuntu preinstalled.  these come with the hardware specific proprietary drivers, which is what the dell specific images and the 'recovery' media creation tool seek to provide.  basically dell only support ubuntu, so these methods are the only (simple) way to have a gnu linux operating system distribution support all dell hardware. 

  as you seem to be open to other hardware vendors bear in mind that the proprietary hardware drivers for any netbook may not be readily available for any given distribution.  it can be painfully hit or miss.  for example my asus eeepc netbook came with a special distribution of xandros preinstalled but it wasn't long before asus gave up on it.  it was a while before some other distributions' generic drivers covered the audio and network hardware.  i was forced to wait or find fixes to use new features like flash 10 

  it is rare for there to be such a dedicated pairing of netbook and operating system.  i would suppose this is why this thread is so busy (commiserations to dear mario) so please consider this in your decision.
