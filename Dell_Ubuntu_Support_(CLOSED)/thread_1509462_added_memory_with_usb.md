### Post by jarviser on 2010-06-16
If the RAM never filled, then there would be no benefit at all in increasing it. If the memory fills up in use, and Swap disk space is used frequently, then the speed plummets. You would need to find some way to see if the 2GB ever got used up with any of your apps. 

The processor does not per se use memory to think, but to store what is being thought about. The human brain analogy would only apply if you had dementia and you needed post-its to remember what you were thinking about a minute ago. That would be swap disk space. Apologies for the crude analogy!
