### Post by projectwarbird on 2011-07-29
As the title suggests I've run into a few issues with installing Ubuntu 11.04 on my inspiron 8600 in the form of graphical corruption, specifically the mouse, icons, and for that matter anything that is 3D in the slightest way. I think that it's a problem with the 3D accelerators (or lack thereof.) I installed 10.04 and had similar issues but was able to fix them by going into System>Preferences>Appearance>Visual Effects and turning the effects on. This forced the system to find, download and subsequently install the 3D accelerators. The problem with 11.04 is three-fold. First) I cannot run Unity (no shock there due to lack of 3D acceleration) Second) When I run classic ubuntu I have to run the one without effects otherwise the whole computer will lock up and Thirdly) in the only classic ubuntu that I can run I cant access visual effects. So I essentially have a circular problem in which I can't fix problem A until I fix problem B but I can't fix problem B without fixing problem A. 
My laptop is running a Nvidia Geforce4 Ti 4200 card by the way. If you know of any way to fix the problem the assistance would be appreciated.

### Post by mikewhatever on 2011-08-02
In the menu, top left, go to System->Administration->Additional-Drivers. Install the recommended driver for your card.
