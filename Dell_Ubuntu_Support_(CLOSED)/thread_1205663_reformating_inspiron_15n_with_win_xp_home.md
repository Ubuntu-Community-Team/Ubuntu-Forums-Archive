### Post by ugm6hr on 2009-07-06
> **wining said:**
> i am planning to buy inspiron 15n 
after buying it can i reformat it to win xp using my original winxp cd which came with the last dell laptop?

Short answer: No.

Long answer:

Dell supplies OEM Microsoft licenses (as do every other computer supplier) with newly purchased computers.  The CD is designed to install to a specific computer (i.e. the old computer); hence, it will not work on the new 15n.

Additionally, I believe MS OEM licenses do not permit you to install on a different computer.

Drivers are available from the Dell support website.

If you want to do this, you will have to have a Full XP CD and license, which are generally quite expensive.

### Post by Bubbatech on 2009-07-09
Wifi works great, sound is OK but volume is the only negative thing about my 15n. 

I have the 6 cell battery and it seems to never run out.

I dont like touch pads I use a USB mouse.

CD burner is not too loud and works fine.

I received a standard Ubuntu CD and a multimedia player add on cd. I never opened either. I downloaded the Jaunty .iso (after backing up the system with the builtin backup program) and did a full re-install of Jaunty 9.04. For multimedia you can't do much better than VLC.

If you have never used any Linux program I would suggest trying a few distros live cds before dropping any real cash into a new system. I have been using Ubuntu on my desktop a while now. I still dual boot for gaming and the like on Windows but Linux is just plain faster and more enjoyable for most of everything else I do.
