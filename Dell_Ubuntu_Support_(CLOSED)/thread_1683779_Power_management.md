### Post by TBABill on 2011-02-08
Did you happen to install from an old iso? When 10.04 first released my machine (Dell 1564) did the exact same thing. It would race up to 80C+ and I shut it off myself to prevent damage. But the updates to 10.04.1, whatever they were, corrected the issue and it has run cool in 10.04 and 10.10 ever since. I moved away from Ubuntu for a while because of it and was happy a month or two later that the issue was resolved. 
 
If you did this from an old iso, will it run cool enough to get all updates installed? If that's not the issue, cleaning it and running top to find out what's pushing your system so hard would be the best start. And does it run cooler in Windows (if you have it installed)?
