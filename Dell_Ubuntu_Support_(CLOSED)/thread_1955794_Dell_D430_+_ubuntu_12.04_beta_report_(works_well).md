Hi all - Just thought I'd let you know how I got on with the 12.04 beta on the old d430. An excellent laptop, but I have had a few problems with previous distros.

Only one major problem - the wireless card isn't recognised out of the box. 

However, top marks on the ability to plug an ethernet cable from my other ubuntu laptop into this one and get straight on the web! Ubuntu downloaded the restricted wifi (b43?) drivers. However, still no luck (wifi light not even coming on). Eventually found the solution after many many attempts. If i remember rightly, you need to install extra bcm drivers from the software centre, then turn off the restricted drivers. Its all lost in a long session of trying many things one after the other - but after only about 40 mins - all works!

So - still getting used to the new menu system (its growing on me). Libre office is great. Everything else worked straight away. I was especially impressed with:
* Chrome found all my old bookmarks by syncing (not sure how it did this as I hadn't told it my email!)
* Ubuntu printers - found my wireless HP printer straight off! (just went to add printers and there it was in the list!)
* Flash and audio worked straight off (not always guaranteed on this laptop)
* I do like the alt+tab, which will expand multiple windows of the same application if you wait on that icon.
* Multiple windowspaces works well (and can be configured to 4x1 rather than 2x2 - i've seem some people saying this can't be changed.
