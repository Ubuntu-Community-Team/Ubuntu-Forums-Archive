### Post by njpatel on 2008-10-22
Hi,

There have been a few issues with the Dell Inspiron Mini with the custom Ubuntu 8.04 installed, so I'm happy to announce that there is now a dedicated Launchpad project page for Mini hardware/custom UI bugs:

[https://edge.launchpad.net/dell-mini](https://edge.launchpad.net/dell-mini)

The idea is that we can see the bugs that people experiencing easier, and (hopefully) get the fixes out there faster!

Please remember that this is for Dell Inspiron Mini specific bugs only, general bugs should be filed in Launchpad (or upstream) to the correct projects as per usual.

Thanks!

Neil
