### Post by Dennis N on 2009-10-14
To change the homepage of Firefox:

Edit > Preferences > Main

There is a box to type in URL for the home page you want, or if you have navigated to the desired home page, just hit the button 'use current page'.

Caps lock: Don't have your particular model, but look for a lighted indicator above the key area. Press the caps lock a few times and you should see it come on.
