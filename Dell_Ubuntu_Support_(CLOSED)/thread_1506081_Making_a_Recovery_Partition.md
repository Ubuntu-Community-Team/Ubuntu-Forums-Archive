### Post by dv3500ea on 2010-06-10
I'm not sure why you would want to make a partition to do this when the live CD can make a new installation anyway.

What you might want to do is create a separate partition for your home folder. That way if you mess up your installation you can reinstall Ubuntu from the live CD while retaining your personal files and settings.
