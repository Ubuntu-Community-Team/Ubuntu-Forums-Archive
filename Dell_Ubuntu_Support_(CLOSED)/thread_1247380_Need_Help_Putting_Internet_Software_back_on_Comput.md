### Post by staf0048 on 2009-08-23
If it's your router it typcially comes with it's own admin settings you can access through a browser.  

For example, when I type 192.168.100.1 in Firefox's address bar, I get promted to login to my Motorola Router.  From there I can change passwords, set firewalls, etc, etc.

Most home networks/routers use the 192.168.x.x IP address set up.  Check your owner's manual for the specific address then save it as a favorite so you don't have to remember the specific numbers.

Hopefully that helps.
