### Post by gbrainin on 2011-04-07
For some time now, people have been told by phone/chat reps that there are no Ubuntu offerings.  However, no one that I have seen has reported being unable to order a computer with Ubuntu installed via the website (once the proper area is located).

Honestly, one has to wonder about Dell's remaining commitment to the Ubuntu option.  From the beginning, it was clearly a niche market and had the kinds of problems you would expect with that (like phone techs not knowing about it).  There have always been only a handful of options available, and they have been difficult to find on the website.  However, it seems to me that the gradual trend has been toward making it more and more difficult to find and/or get a computer with Ubuntu pre-installed.

So far, it hasn't gone away, but I wouldn't be surprised if it did at some point.
