### Post by jcolyn on 2010-09-12
At startup hit either f2, f10, or delete to go to the bios and first thing set defaults then set your CD drive as first boot. Save and exit then see if it will boot on the CD.

If the CD is good but you cannot install you may have a bad drive. The killdisk software mentioned can do a diagnostic test to see if smart has been tripped which would mean the drive has many bad sectors.

The drive maker would also have a downloadable diagnostic tool to determine the same..
