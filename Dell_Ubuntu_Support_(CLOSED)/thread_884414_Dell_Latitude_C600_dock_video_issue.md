### Post by Johnkozz on 2008-12-15
i8kfan, is the program your looking for, for your fan control. I used this before i converted my c600 over to ubuntu and it worked just fine (xp). After surfin around the internet I am sure that you noticed that there are a few other questions out there about the dell c600 fan speed, both involving ubuntu and windows; the best answer so far is that Dell set the temperature in which the fan is to turn on rather high, but as you will notice there are many users still using there c600 even thought they are more that 7 years old. 

if you do decide to use the i8kfan gui, your going to have to get some advice because in order to install you need a daemon because on of the files will not install since that drive does not exist.

in the end i say just stick with what you got, i wouldn't worry too much about the fan speed.
