### Post by empthollow on 2008-05-18
sadly, it looks like your printer is not yet supported under linux.  Despite the fact that dell distributes ubuntu computers they have failed to write a printer driver for that model for linux.  That is not to say that it will never work but someone has to reverse engineer a driver that will work for it.  [Here]("http://www.openprinting.org/show_printer.cgi?recnum=Dell-Photo_AIO_Printer_926") is a link to your printer on the linux foundation website which provides a info on linux compatibilities and printers.
