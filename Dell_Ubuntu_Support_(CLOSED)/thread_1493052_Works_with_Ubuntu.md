### Post by dFlyer on 2010-05-25
Dell does provide some laptops with Ubuntu pre installed. I currently have a Dell Studio 1735. It came with Windows Vista. Dumped Vista and installed Ubuntu and everything just worked out of the box. Dell has a site that you can take an Ubuntu iso and add the Dell overlay to it. It's really easy to do. The only thing the overlay does is add a few drivers and other applications along with the Dell recovery partition. The web site is:

[http://en.community.dell.com/support-forums/software-os/w/linux/building-base-ubuntu-factory-iso.aspx](http://en.community.dell.com/support-forums/software-os/w/linux/building-base-ubuntu-factory-iso.aspx)
