### Post by Brandon Williams on 2009-04-18
> **sirebral said:**

Disadvantages

2. Maximus. Killed the love for me.  Always resizing windows is a pain.
3. Not well supported.  Being that it is LPIA is probably why.  Every new update needs to be reconfigured for each distro and then the LPIA version of that distro. LPI Suck.
4. No cool effects available.


The fact that the packages are compiled for lpia is _not_ a disadvantage. It is a good thing that the packages are specifically compiled for the platform. You don't gain much from it here, since the lpia is essentially a small variation in i386, but you certainly don't lose anything, either. If you run software that is not in the standard Ubuntu repos, then you will have to re-package it to install it on the Dell, but it can be done, and only needs to be done for software that is not directly packaged and supported by Ubuntu.

Maximus can easily be turned off, whether you're running Dell's Ubuntu or a standard Ubuntu release. Turning it off isn't well integrated into the system, so you have to do some command line magic, but it's not hard.

All of the software that is available in the standard Hardy repository is available in the Dell repository. So all cool effects that are available in a standard Hardy install are available in the Dell distro.

The one problem with the Dell repositories is that the updates are not very timely. It can take a while for even security updates to turn up in the Dell repository. If this is a concern to you, then it's another reason to go with a standard Ubuntu release.
