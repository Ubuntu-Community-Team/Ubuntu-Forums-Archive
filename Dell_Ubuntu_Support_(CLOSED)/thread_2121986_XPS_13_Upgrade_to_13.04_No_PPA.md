### Post by hkb41 on 2013-03-12
It is true that you need no special kernels to run the XPS 13 with 13.04.  The kernels contain all the needed fixes without having to lean on the PPA.  Unless you want to run the latest and greatest bits (or you have some other reason), then I don't see any need to upgrade past 12.04.  With the arrival of 12.04.2 you now have the option of updating to the lts-quantal enablement stack which would essentially give you a 12.10 kernel and x-stack for the LTS release.
