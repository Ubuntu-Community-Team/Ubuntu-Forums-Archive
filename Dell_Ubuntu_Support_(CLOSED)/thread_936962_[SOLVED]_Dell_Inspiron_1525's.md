### Post by Idefix82 on 2008-10-03
Yes, with 2GB of RAM you will hardly notice any benefits of 64 bit (although there some but they will probably be outweighed by the disadvantages). If you think that you might add RAM at some point, then you might as well install 64 bit now, since you can't upgrade from 32 bit to 64 bit, you have to do a clean installation.
