### Post by amauk on 2010-11-15
The Ubuntu bug tracker is at [https://launchpad.net](https://launchpad.net)
You'll need an account on Launchpad to file bugs

To file a bug against a package, open up a terminal and do```
ubuntu-bug <package>
```
Eg.```
ubuntu-bug compiz
```
