### Post by ndefontenay on 2009-11-04
+1 to above with a tweak.

When you put your ubuntu CD, you have a test ubuntu without installing and an install ubuntu button.

When you test without installing you get an install button on your desktop. This will bring you to the same install wizard as the install option.

From there, you will be prompted for a few questions, then you will have a choice to install ubuntu without erasing windows or install ubuntu using the whole disk. You should pick that option.

This should work very easily.

The problem you describe here: "got a bus error" however seem to be a hardware issue. If you manage to reproduce this problem, can you post it in detail here? It could help identifying what your problem is and tell us if you can do something about it or not.
