### Post by Teamgeist on 2008-11-01
I say ABSOLUTELY get the Intel Graphics AND the Intel Wifi.

The Intel drivers are all open source and work just lovely.

You will have to use Ubuntu 8.10 for the 5300 Wifi since only the new Kernel of 8.10 supports it.

For the graphics you can safely get the ATI also. ATI released the information needed to write open source drivers to the community, so this card also works.

I still highly recommend to get the Intel Wifi. Mine has never let me down (Intel 4965).

The Card Reader works out-of-the box. No hassle.

This all is based on m personal experience with an XPS m1330 with Intel graphics and Wifi.
