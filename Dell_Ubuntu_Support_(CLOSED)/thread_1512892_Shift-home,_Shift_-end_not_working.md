### Post by joe606 on 2011-08-16
Digging up an old post, but i had this problem with a fresh install of 11.04. The above didnt solve my problem as it was un-ticked by default.

My solution was to go into Keyboard Layout Options, and under "Miscellaneous compatibility options" to tick the box called 

"Shift with numeric keypad works as in MS Windows"

Fixed and working fine now :)
