### Post by __Ryan__ on 2008-11-30
The remote shares the same keyboard commands as the media buttons across the top above the keyboard. Ubuntu doesn't know if the remote or the media buttons are signaling the specific command since it is done in the hardware. I mention all of this because most of the media buttons are supported right out of the box with Ubuntu. The volume buttons work anytime, the others such as play, stop, forward, will work depending which media player you have open at the time.

### Post by __Ryan__ on 2008-12-01
If the media buttons work above the keyboard then the remote should work as well since the hardware maps both to the same commands which it passes to the OS.

To put it another way if your volume works on your media buttons but not the remote then you have a problem with the remote.  If neither work then its a configuration problem in Ubuntu.
