### Post by lindsay7 on 2009-07-21
You can use Gparted or as it is called in the repos (synaptic) partition editor.  You have to unmount the drives to work on them. So it is best to use the live cd to do this or download Gparted and make a start up disk and run it from there.  Before you do this defrag the windows partition using windows first to make sure that you do not have any problems with shrinking or resizing the windows partition.  You will not have to do a reinstall.
