Hey babloo75,
The new 'Jaunty Jackalope' actually is very compatible with various hardware. Just recently had a lot of problems with installing ubuntu on an old Pentium II computer with the 8.10 distro. Now working great. 
Also, if you are still having trouble without windows i recommend using Virtual Box. VirtualBox allows you to simulate another OS by actually installing it as if it was on the same computer. This way, reset is not required and you can keep your files using the Guest Additions and view them from your windows system to the ext3 system.
apt-get for VBox:
```
sudo aptitude install virtualbox-ose virtualbox-ose-modules-generic
```
Good Luck :)
