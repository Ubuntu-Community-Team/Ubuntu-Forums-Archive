### Post by ericmc783 on 2010-05-30
UPDATE:

On this same PC, I have now upgraded to Ubuntu 10.04 LTS.

Ever since I got the PC and had 9.04 on it, there were two graphical related issues. First off, when watching a movie file, the system would occasionally crash, and kick me to the user login screen (like I had selected to switch to a different user). This started getting more and more frequent as time went on, and would happen regardless of the media player I was using (VLC, Totem, and Banshee where the 3 I would use).

Secondly, Google Earth never worked properly. The area where the earth should appear would just flicker like crazy. I never did take the time to troubleshoot this.

Since installing 10.04, neither of these issues now occur. And I have noticed no other issues so far. Every release of Ubuntu seems to have some short-term turbulence at first, but I think the core of the OS is getting stronger and stronger with each release. And I should mention that I am using the "new wave" theme, so I don't have to deal with the annoyance of the window buttons on the left. Sorry, but I have never been a mac user, and I think them being on the left, in mac format, would be way too hard to me to get used to. :)

My thanks to the Ubuntu devs for another fine release.
