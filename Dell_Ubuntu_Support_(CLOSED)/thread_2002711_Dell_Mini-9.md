### Post by piperbarb on 2012-07-05
I just installed Ubuntu 12.04 (32-bit)on my Dell Mini 9 without any problems.  So far, everything works.  It has a 1.6 GHz atom processor, 2 GB RAM, and the 16 GB SSD drive.  I also stuck an 8 GB SD card in it that I had sitting around.  I wasn't sure if it would work well, considering some of the comments that I read.  I was able to install it from a USB flash drive.  

This computer has been sitting around for the past 6 months or so doing nothing.  I had installed OS X (10.6.3) on it last year just to do it.  When I went to install Ubuntu, it asked me if I wanted to replace OS X, which I did.  So far, this little, 3-1/2 year-old computer is purring along.  It does everything it needs to do (mainly Web browsing and e-mail) so I am not complaining. 

If you are afraid of upgrading it, don't be.  Give it a try.  I think you will be pleasantly surprised.  I know I was.  Doing a clean install is definitely recommended.  Don't forget to back up any important files before upgrading.):P
