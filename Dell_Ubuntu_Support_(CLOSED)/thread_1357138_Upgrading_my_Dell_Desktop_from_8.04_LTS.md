### Post by Janeleaper on 2009-12-17
Go to System - Administration Update Manager. Then go to Settings - Updates and choose Normal Releases.

Then run Update Manager.  It will show 8.10 as available.  After you have updated to 8.10, Update Manager will show 9.04 as available, and then 9.10.  

I've just updated one of my Dell pcs from 8.04 to 9.10 and the three stage process took several hours.  

Be sure you want to update.  Personally I'd recommend updating to 9.04, but some people are experiencing difficulties with 9.10.  I can't get BBC iplayer to run properly on 9.10, but the reason I updated is that it is easier to install webcams on 9.10 than the earlier versions and I've just started using Skype.
