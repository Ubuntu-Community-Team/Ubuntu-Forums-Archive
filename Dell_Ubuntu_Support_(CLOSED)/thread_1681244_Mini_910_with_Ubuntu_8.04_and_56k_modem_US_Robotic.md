### Post by davepoth on 2011-02-03
USR5637?

There's quite a bit about this on the forums.  What there's not a lot about is whether the crippled version of ubuntu that ships on the Mini 9 has all the necessary bits to make it work. I never managed to get a dial-up modem to work when I tried it.

I'd recommend doing a clean install of the normal desktop 10.04 or 10.10, which do work on the mini 9 (I'm typing on one now!), as that will ensure you've got the best chance of having the right bits of Ubuntu. It's a really easy job providing you have broadband (to download it) and a USB key to install it to.

### Post by davepoth on 2011-02-04
They both work (I assume the netbook one does!) but it's a matter of choice really. If you use the netbook interface you'll want the Netbook version, if you use the "normal" interface the desktop version will be better. If there's anything you need to save off the netbook do it before installing 10.10, as 8.04 was compiled specifically for the Atom processors so will need a complete wipe and reinstall.
