### Post by CoolJohnB on 2010-08-12
I have a Dell Vostro 3700 I downloaded the latest BIOS from the Dell website.

Then downloaded a XP bootdisk from [www.bootdisk.com](www.bootdisk.com) made a Boot CD and put the BIOS file onto a flash drive, when I booted with the XP CD the flash drive showed up as C: I then just ran the BIOS update which worked fine.

I found it a very easy way to update the BIOS, pretty trouble free, hope someone else will fine this useful.
