My upgrade from 9.10 netbook remix to 10.04 failed with a blank screen. After some searching I found another thread that suggested using the 2/6/31 kernel if you have Intel graphics. Luckily Grub still had that and it solved my problem also. Even the old problem of the mouse cursor in the 9.10 remix window not working properly on the 700m is now fixed.
