### Post by perroazul on 2009-10-23
I recently bought a studio xps laptop with ubuntu. I was aware that the price was the same for a machine with vista, but I wanted a machine with ubuntu anyway, and so far I am very happy and satisfied with my new laptop. 
I didn't mind that there is no price difference because I have been thinking that instead of paying the cost for a software licence I was paying for all the special customizations, adjustments and fine tuning that has been done to the operating system in my machine so that everything works flawlessly. 

One thing I don't like, however, is that couple of days ago the price of the same studio xps has increased by fifty dollars and is now matching the price of the laptops which now come with windows 7. In other words the price of the laptop has been artificially inflated because of the launching of windows 7, software that has nothing to do, whatsoever with the ubuntu laptops being sold!!
