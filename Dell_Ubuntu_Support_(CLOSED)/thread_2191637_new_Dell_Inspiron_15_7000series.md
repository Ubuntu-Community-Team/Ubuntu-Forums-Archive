### Post by papibe on 2013-12-03
Hi

Dell machines are usually well supported (Dell offers some Ubuntu preinstalled machines). Here's a list of [certified hardware]("http://www.ubuntu.com/certification/desktop/") so you can review your particular model.

If you model if not there, it does not necessarily mean it is not supported.

One definitive test is to use the installation media as a live DVD/USB. Boot into it, but instead of installing Ubuntu choose 'Try Ubuntu without installing it'. You'll get into the desktop and there you make your tests (wireless networking, sound, etc).

I hope that helps. Let us know how it goes.
Regards.
