1.  Type to test settings: SHIFT+1=! but SHIFT+2=@ SHIFT+3=# SHIFT+@=" 

2. Change Keyboard model> = Dell 101-key PC

3. Type to test settings: SHIFT+1=! then SHIFT+2=" SHIFT+3=£ SHIFT+@=@

4. Click CLOSE.  Open Office doc type SHIFT+1=! then SHIFT+2=" SHIFT+3=£ SHIFT+@=@.  Hurrah!

6. Restart Mini 9 Open Office doc type SHIFT+1=! but SHIFT+2=@ SHIFT+3=# SHIFT+@=".  Arrgh!

7.  Keyboard Pref Layouts settings still: Keyboard model=Dell 101-key PC; Layout=United Kingdom. ???

8. Goto 2 above and change Keyboard model> = Dell then you progress to 3 all is well until you reboot again.

Any suggestions to get "£@ to stay as they should be?
