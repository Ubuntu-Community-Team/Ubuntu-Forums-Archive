### Post by msvf on 2009-11-07
Im an ubuntu newb who has spent the last week messing around with different terminal commands and packages with no success, so i was gob smacked by this simple method which finaly fixed my dell mini 10v 

1. Install "Windows wireless drivers" from software centre.
2. Download the (windows) drivers for the wireless card from dell support 
3. Extract the drivers
4. Open "Windows wireless drivers" 
5. Navigate to the small .inf file in the driver folder ( in my case bcmwl5.inf which was in the DRIVER folder)
6. Press install

It worked for me  :D and i didnt even need to reboot!
