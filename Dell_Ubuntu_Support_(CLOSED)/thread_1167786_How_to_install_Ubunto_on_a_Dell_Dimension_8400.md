### Post by jsonder on 2009-05-23
The file needs to be burned as an iso file, this is not a data burn.

Windows is not equipped to burn an iso file; typically you have to use Nero or some other windows burner program.  The best bet is to download and burn on another linux computer.  K3B has always burned cd and dvd isos OK for me.
