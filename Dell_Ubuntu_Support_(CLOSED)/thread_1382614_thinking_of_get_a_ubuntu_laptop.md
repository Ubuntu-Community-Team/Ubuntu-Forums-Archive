### Post by Knowle on 2010-01-18
Hey python

I have a Inspiron 15n I ordered 6 months ago and love it. When I got it, it originally shipped with 8.04, the only ISO they have is for 8.04 but you can make a backup on your own with a Dell branded tool that's on the desktop. You can also of course always just download whatever version you want online.

The Intel Core 2 Duo has 2 CPUs. All the hardware works perfectly out of the box, as it should. Even if you do a fresh install with Ubuntu, Kubuntu, Xubuntu it all works perfectly as I've done it myself. Upgrades have worked perfectly for me as well. You would have to be connected with ethernet first to install the drivers for the Broadcom wireless chip through Hardware Drivers.

I'm not sure how it will run with 3GB of RAM as when I got mine the default was 2GB and I left it as is. The only concern I would have for you is battery life. I got the 6-cell with mine and I can only get 3 hours max. That's mainly just web browsing, watching videos on hulu or YouYube.

edit: Seems I was wrong about the latest ISO, sorry about that. There's one for 9.04 here. [http://en.community.dell.com/wikis/linux/ubuntu-9-04.aspx](http://en.community.dell.com/wikis/linux/ubuntu-9-04.aspx)
You can also always grab it from here as well. [http://releases.ubuntu.com/9.04/](http://releases.ubuntu.com/9.04/)

### Post by jml on 2010-01-19
One suggestion for the OP.  When you order your laptop, spend the extra money and get an Intel wireless card rather than useing the Dell wireless card.  The Dell card is usually manufactured by Broadcom.  A very Linux unfriendly piece of hardware.  These forums are replete with many posts describing the problems getting a Broadcom wireless card running under Linux.  The Intel wireless card just seems to have better support under Linux. And if I am reading the Dell website correctly, it only adds $40.00 US to the cost of the laptop. Just my two cents worth.

Joe

### Post by Knowle on 2010-01-20
That's very true jml. The 15n's do come with a Broadcom wifi chip(BCM4312), in my first post I mentioned that as well. Dell already has the drivers installed. If a re-install is done it's as easy as having ethernet plugged in, then going to System->Hardware Drivers-> select Broadcom STA wireless driver then Reboot. Works in any *buntu distro.

I'm pretty sure the Intel wifi would be better but you can only select that one if you are getting it installed by default with Windows 7. If you go into Dell's Open Source PCs section and select the 15n(ubuntu pre-installed) there are no such configuration option for the wifi card.
