You can browse through the Dell Ubuntu repository here: [http://dell-mini.archive.canonical.com/ubuntu/pool/](http://dell-mini.archive.canonical.com/ubuntu/pool/)

(while you are waiting for your Dell to arrive)

I always advise new Linux users to install from the repository whenever possible (using Add/Remove Programs, the Synaptic Package Manager, or terminal commands like apt-get). The repository is the best source for safe and trusted software.

That being said, sometimes an application you need isn't in the repository. When that happens, yes, you can install applications from other sources (at your own risk, of course). The "Windows way" of doing this would be to find the website for the application and follow their download & install instructions.

You might find these pages helpful: 
[https://help.ubuntu.com/community/InstallingSoftware](https://help.ubuntu.com/community/InstallingSoftware)
[http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)
