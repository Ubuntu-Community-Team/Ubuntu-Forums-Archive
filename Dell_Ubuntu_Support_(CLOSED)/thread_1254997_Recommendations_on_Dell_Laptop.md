### Post by edwardLS on 2009-09-01
I purchased a Dell Studio 1555 loaded with Vista.  I am now running Ubuntu 9.04 on that laptop.  I have installed several times and have use generic Ubuntu and the Dell ISO.

I have also installed Windows XP on this machine.  To do that I had to 'Slip Stream' the Chipset and SATA driver onto a Windows XP Pro install disk.  Both XP and Ubuntu are running very well.

Good luck.
