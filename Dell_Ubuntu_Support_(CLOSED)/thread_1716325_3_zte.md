### Post by Ichido on 2011-04-10
Check Synaptic Package Manager and install "Bluetooth" and "libgnome-bluetooth7".

Keep adding the various "Bluetooth" Programs/Files listed till it works.

This worked for me, so now I can Bluetooth Link to my Android Smart Phone with my laptop.

I'm using:
"Blueman applet 1.21", Blueman is a GTK based Bluetooth manager.
