### Post by brianfactors on 2009-07-06
To use the webcam, you should download some software for it. So, I guess my question is: what exactly do you want to do with it?

Try Cheese:
```
sudo apt-get install cheese
```Or you can just hunt it down in Synaptic, if you'd prefer.

Cheese will allow you to take photos and video.

I have the same computer and my webcam works great with everything but flash. I don't know if there's much you can do about that, though.
