### Post by spikoley on 2013-03-25
When I type the cursor jumps on the screen making it almost impossible to type.  It acts like the mouse is being clicked which moves the cursor or selects something else.  Under the mouse settings I have *Disable touchpad while typing* checked and it doesn't help.

12.10 on an Inspiron 1505

Any ideas?

---

### Post by spikoley on 2013-03-26
I think I found the solution.  

I disabled scrolling on the Touchpad tab and the issue appears to be gone!  

```
System Settings>  Mouse and Touchpad> Disable Scrolling
```
