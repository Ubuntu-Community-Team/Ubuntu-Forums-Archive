### Post by employeeno5 on 2008-10-24

It's a great machine and I splurged for a 24" HD screen. Sweet.

My only problem is that it doesn't wake from suspend. The fans will fire back up but it will only give a black screen. The only way out of this blank screen is to be force the power off and reboot.  

This seems a little silly to me given that it's Dell's custom install on a desktop and even my personal installations of  Ubuntu on laptops manage to do this fine. Looks like Dell dropped the ball on this one.  

This problem is consistent whether I'm using the Intel integrated graphics it came with or the ATI Radeon 2600 card I added separately so I don't think it's a video issue. 

Has anyone else had this problem with this model and if so do you have any solutions handy? Any suggestions on where to start trouble shooting this at least?

Thanks for thoughts and idea!
