### Post by Euphorion on 2008-05-29
This is a problem that I am confronted with. In my case I have a Fujitsu Laptop with ATI Radeon HD-2700. I installed Ubuntu Hardy 8.04 2 days ago on this PC. The LCD Panel came up with the correct resolution but the attached screen just mirrored the Desktop. After a while Ubuntu offered me the option to install a propriatry driver (not open source) via notification in the launcher panel. I did this and received additional features for my Video Card. I then found the ATI Catalyst Control Center in Add Applications (do not forget to activate all sources). In the Catalyst there is a section dealing with the use and setup of 2 monitors a la windows.
