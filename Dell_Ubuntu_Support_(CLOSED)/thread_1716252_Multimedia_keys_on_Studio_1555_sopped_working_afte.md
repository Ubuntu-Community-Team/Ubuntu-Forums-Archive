### Post by nandinga on 2011-03-28
SOLVED!

It was the most dumb thing it could be... Salve Occam's razor!

Trying to use the F9 key the effect I've got was rising the volume! ...somehow my setting in the BIOS was changed, I had "Function Keys First" and it spontaneously changed to "Multimedia Keys First". It has happened to me before, don't know why.

If it starts changing again by itself I'll let you know.
