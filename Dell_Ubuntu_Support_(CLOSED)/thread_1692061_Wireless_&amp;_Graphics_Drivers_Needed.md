### Post by sviola on 2011-02-20
Welcome to Ubuntu!
I'm not sure if you've tried this yet, but if you haven't it might be worth a try.

Try going to System->Administration->Hardware Drivers. The Broadcom driver (and hopefully the other one, too) should be there. (Note that you'll need to be connected to the internet via ethernet until your wireless is working.) If they are there, just activate them.

If you can't connect via ethernet, try following the directions in[ this other forum post]("http://ubuntuforums.org/showthread.php?t=1368699") to install your Broadcom driver first.

I hope that helps!
