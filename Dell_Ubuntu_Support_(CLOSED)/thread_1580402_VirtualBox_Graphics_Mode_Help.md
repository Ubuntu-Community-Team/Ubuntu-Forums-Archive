### Post by bamim2 on 2010-09-23
I appears that I did not correctly run the VBox Guest Additions. Now that I DID run it correctly, that fixed my graphics mode problem. 

Solution: 

1) Right click on the VBoxGuestAdditions CD image on the desktop.
2) Choose the Autorun option.
3) Let it run to completion & watch the output on the screen.
4) Reboot when complete.
5) Rock ON!!!
