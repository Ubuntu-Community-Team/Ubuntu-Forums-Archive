### Post by TBABill on 2011-04-14
If the machine is capable and if you have cpufreq-utilities installed (I think it is by default but unable to confirm), then you can check what your frequency scaling is set to by doing ```
cpufreq-info
``` and where you see a word in quotation marks is your setting. Ondemand is the setting by default if memory serves correctly.
 
To change the settings you can ```
sudo cpufreq-set --governor ondemand
``` In place of ondemand you can use performance or another setting, but you want scaling and I have personally found ondemand to be best for my machines.
 
You may also need to go into power settings to see what your machine is setup to do with power usage/conservation in different conditions, such as on AC power, on battery, low battery, etc.
