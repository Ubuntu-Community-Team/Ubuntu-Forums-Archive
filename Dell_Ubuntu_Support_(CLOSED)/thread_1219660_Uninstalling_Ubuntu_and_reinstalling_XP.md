### Post by pastalavista on 2009-07-22
Check your BIOS (before you boot anything) and make sure it will boot from CD before the hard disk and then try booting with the Dell CD in the drive, the way you booted and installed Ubuntu. If it won't boot from the CD, you'll need to download or borrow a WindowsXP boot disk. If you'd been careful and studied up a little before you installed Ubuntu, you could have saved the Windows install you had and dual booted.

### Post by fballem on 2009-07-22
By the way, before you re-install Windows XP, please make sure that you backup your data (documents, pictures, videos, etc.) to an external drive and disconnect that drive from your machine. In this way, you will prevent the loss of your data during the installation.

Once XP is re-installed and working the way that you want it, then reconnect the external drive and transfer the data.

### Post by Wally_dog on 2009-07-22
Just a tip and some parts recommendations for a gaming machine:

Intel Core 2 Duo (Dual cores perform better than quad cores, even if a quad core is running at the same clock speed as a dual core. The Intel Core 2 Duo E8400 can even be overclocked to 4.0GHz+)

4GB of DDR2 RAM. If possible, get the lowest latency and highest speed RAM (For DDR2, the highest speed would be 133MHz. This helps if you are overclocking. If you're not overclocking, DDR2 800MHz will be more than fine).

For a graphics card, obviously the more $$ you spend, the better performance. I am currently using a nVidia GeForce 9800GT and can play every game (except Crysis, of course :P) on max settings. 

My rig:
Intel Core 2 Quad overclocked to 3.4GHz
Corsair XMS2 DDR2 800 overclocked to 928MHz
MSI nVidia GeForce 9800GT OC (factory overclocked)
ASUS P5QL-Pro
