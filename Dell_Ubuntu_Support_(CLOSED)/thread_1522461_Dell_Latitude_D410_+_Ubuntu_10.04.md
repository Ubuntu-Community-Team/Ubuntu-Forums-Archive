For the beginning, Thank you for all your effort with support. Ubuntu creators and Dell support as well.

After long years of frustration with windows I decide to change my OS to linux and my decision was Ubuntu.

So my present configuration is Dell Latitude D410 + Ubuntu 10.04 

Operating system works very well and I have no doubt, that this is a correct way how to use your secondary laptop or PC. Main computer otherwise still use Windows 7 and I think it will be for longer time, because of work and gaming as well.

No general problems with my Latitude D410. To have OS ready for use (running wifi and internet browser) under 1 minute was always my dream. 

With my Dell I have 2 possible things to tweak.

1. During instalation I have a problem to split partition. I was planning to use dual boot with Windows XP for beginning. It always end with some error message, but it was easy to roll back and dont destroy partition. So I install Ubuntu as primary system (and erase Win). Maybe in future I try to install windows XP alongside.

2. Driver for wifi was not present, so I need to put lan cable and install it this way. System found hardware correctly so no big deal, but will be nice to have system ready without some other delay.

If you have some comment or suggestion, you probably need to know, how I install it.
So, I use instalation of Ubuntu 10.04 netbook edition which I put to USB as is describe on Ubuntu download page.

I´m new with Ubuntu and linux distribution generaly, but it looks, you found a good way how to make linux user friendly and accessible to ordinary people.

Thank you all again.
