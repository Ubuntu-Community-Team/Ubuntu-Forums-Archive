### Post by coffeeaddict22 on 2009-07-07
Hi Ian,
Welcome to Ubuntu!
There are a few tools for networking.  However, one of the features of Ubuntu is there's always at least 2 options, so we need a bit more information to help.  
If its the simple problem, there's an icon in the top left of your screen with 4 bars.  Click on that, and connect up.  If that doesn't work, then try the next.
Are you running plain Ubuntu?  
The terminal is easiest to explain things in, not least because you can cut and paste stuff in it.  Open one, put in the following commands, and post the results back.
```
uname -a
nm-tool
```
That should give us enough to figure out the problem; if you want more help, post back here.
