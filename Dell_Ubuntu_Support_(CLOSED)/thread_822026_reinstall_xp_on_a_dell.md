### Post by xandraius on 2008-06-09
Is the restore partition still there? Check using Gparted in Ubuntu and see if there is a small partition on the drive that is NOT an NFS file system. Here is a good article that while a little dated, is fairly accurate as to how that restore partition works: [http://www.goodells.net/dellrestore/](http://www.goodells.net/dellrestore/)

Personally, do a clean format and reload. As long as your data is backed up, you'll spend maybe an hour. Make sure you hit [http://support.dell.com](http://support.dell.com) and get the most current drivers. A flashdrive rocks for this.
