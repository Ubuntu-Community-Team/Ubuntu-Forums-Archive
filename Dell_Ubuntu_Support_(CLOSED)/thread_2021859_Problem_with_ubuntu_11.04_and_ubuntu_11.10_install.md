### Post by piperbarb on 2012-07-10
I installed Ubuntu 12.04 (32-bit) on my Dell mini 9 a few days ago.  It did not have Win XP on it.  I was able to install it from the bootable USB flash that I made by following the instructions on the Ubuntu download page.

I did boot Ubuntu from the USB first to be sure it would work.  I then rebooted it and installed Ubuntu 12.04.  It isn't the fastest install, so just have patience.  Also, be sure you are installing the 32-bit version.  I doubt that any Dell Mini can handle the 64-bit version.

Good luck and keep trying, it should work.
