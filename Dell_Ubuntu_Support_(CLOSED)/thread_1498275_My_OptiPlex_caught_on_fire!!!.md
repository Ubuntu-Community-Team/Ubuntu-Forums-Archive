### Post by Boondoklife on 2010-05-31
Really I only see three options:

1) try to see if you can boot from the pci usb card, which I doubt will work if the keyboard doesn't. If you can, then get a usb stick and install grub on it and put it at the top of your boot list in the BIOS. This will allow it to boot Ubuntu with the stick in and windoes with out it.
2) Replace the motherboard.
3) Only run Ubuntu or Windoes

I'm sure there are others, but these are the most down to earth solutions I could think of.
