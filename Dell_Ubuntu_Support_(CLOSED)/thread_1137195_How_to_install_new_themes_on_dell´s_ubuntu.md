### Post by blackened on 2009-04-25
Some themes will be available via Synaptic. Search for "-theme", using the name field only, to find them.

If none of those suit you, then download ones you like from [http://gnome-look.org]("http://gnome-look.org") and install them by dragging and dropping onto the Appearance dialog (System->Preferences->Appearance) or by extracting them to /usr/themes or ~/.themes (subdirectory of your home folder).
