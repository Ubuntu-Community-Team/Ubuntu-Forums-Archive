### Post by pastalavista on 2009-05-05
Are you using Compiz? If so disable it and in System > Preferrences > Appearance > Visual Effects set to 'None' and see if it helps. If not using Compiz, check System > Administration > Hardware Drivers to see if there are any available that you haven't installed
