### Post by snowpine on 2010-12-06
I've been very happy with my Dell Mini, which I bought almost 2 years ago (wow!) with Ubuntu 8.04 preinstalled. I've also heard good things about System 76 (not a customer myself).

The thing about buying a computer with Ubuntu preinstalled is this: you only get 0-5 months of benefit. When the next Ubuntu release rolls around (next one is in April), you'll probably want to do a fresh reinstall. At that point, it makes no difference whether you bought the computer with Windows or Ubuntu preinstalled. So my advice is buy the computer you want and don't worry about the OS; installing Ubuntu is a 20-minute, free process.. ;)

### Post by fjgaude on 2010-12-06
I too bought a dell mini 10n with Ubuntu 8.04 and a 1366 x 768 screen about a year ago. I just upgraded it to 10.10 using a USB flash drive. Wow!

Everything worked out of the box! Sound, wireless, etc., except the screen was fixed at 1024 x 768. Bummer! But... but, simply running this in a command line fixed that with everything else still simply working as it should:

```
sudo add-apt-repository ppa:gma500/ppa && sudo apt-get update && sudo apt-get install poulsbo-driver-2d poulsbo-driver-3d poulsbo-config
```

I'm a happy camper!
