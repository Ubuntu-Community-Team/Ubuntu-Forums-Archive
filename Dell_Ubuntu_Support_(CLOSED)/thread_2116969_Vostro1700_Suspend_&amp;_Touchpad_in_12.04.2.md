### Post by PeggySue on 2013-02-17
I had a 12.04.1 install on my Dell Vostro 1700 and all worked well.
 
I did a fresh install using 12.04.2 with 3.5 Kernel and the scroll feature of the touchpad doesn't work and the vostro won't suspend.  It makes all the right noises, appears to go down but then powers up again immediately.  I also think the power consumption is higher but am not sure.

I have the 2 disks configured as a raid array and was using the 64 bit alternative distribution.  To fix the problems I did a fresh install of 12.04.1 using the same configuration and did a full update (all 500 odd updates!).  Everything is now fine.

I will stick with this configuration but wondered if anyone else had this problem.  I am concerned about using 12.04.2 for installs on friends' computers.
