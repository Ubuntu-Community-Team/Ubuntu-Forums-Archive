### Post by ugm6hr on 2010-07-20
There is a built-in simple firewall called "ufw" in Ubuntu 10.04.

If you want to control it with the mouse, just install "gufw" with:
```
sudo apt-get install gufw
```

Otherwise, there are plenty of instructions on how to use ufw from Terminal:
```
sudo ufw enable
```

That is all that is required for most people - type once and forget.
