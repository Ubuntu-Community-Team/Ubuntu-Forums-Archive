### Post by swordphsh on 2008-07-12
I have a dell latitude d830 and i have the bios configured to activate the "number pad" when numlock is on. It worked fine in gutsy, but when i upgraded to hardy it stopped working. When i turn on numlock the keys with "number pad" numbers become deactivated and do not type anything. If i use the fn key while numlock is off the number pad acts as a number pad without numlock on, such as using the keys as arrows and moving the mouse pointer with it.

I'm not even sure where to look for the cause of this problem besides some sort of software bug.

If anyone else is having this issue let me know if you have any ideas about it, otherwise any help is appreciated.

Thanks.

EDIT: default setting for hardy i guess, it was set to use keyboard for controlling the mouse.

---

### Post by helpdeskdan on 2008-08-03
System -> preferences -> Keyboard -> Mouse Keys -> uncheck the box.  

Thanks, that was exactly what I was looking for.
