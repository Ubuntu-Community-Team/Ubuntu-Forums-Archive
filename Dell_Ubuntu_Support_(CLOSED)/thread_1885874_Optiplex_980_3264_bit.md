### Post by dinusuresh on 2011-11-25
If you are using ubuntu for production then i best suggest you go in for the earlier versions which are more stable then the latest release.

you can get yourself a copy of the dell ubuntu 9.04 iso here [http://linux.dell.com/files/ubuntu/jaunty/iso-images/](http://linux.dell.com/files/ubuntu/jaunty/iso-images/)
you can download the *A01 version*.

you can install a 64-bit version but make sure all the drivers you install are 64-bit version as well. The application softwares you use can be of 32-bit, it would not be a problem.
