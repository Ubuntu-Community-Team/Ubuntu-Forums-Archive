### Post by thon0925 on 2008-11-01
You could download an Ubuntu 8.10 live cd and pull off all of your files to an external drive/network location. This would also show if there is a problem with 8.10 on the machine, or if the install is corrupted. If the cd did boot and you backed up all of the files, then you could do a fresh install and see if it was just an update failure, or something more serious.
