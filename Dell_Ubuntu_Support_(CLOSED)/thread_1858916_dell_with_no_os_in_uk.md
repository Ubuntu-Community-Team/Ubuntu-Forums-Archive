### Post by Lars Noodén on 2011-10-17
There are sometimes different options available if you order by phone.  If you order by phone it *might* be possible to order a whitebox or even a machine with Ubuntu pre-installed even though it does not show that choice on the web site.  

The hard way to do it is to order a machine and then apply for a [Windows Refund](http://www.google.com/search?q=windows+refund).  That can be done but can be a chore to do successfully.
