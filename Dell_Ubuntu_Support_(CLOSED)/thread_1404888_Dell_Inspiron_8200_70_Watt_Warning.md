It has been a while since I used an Inspiron 8200. If I remember correctly, there should be something in the BIOS about "adaptor warnings" which you can set to "disable". This will eliminate the "Press F1 to continue" message during boot, but is unlikely to help with the performance issue, as the system will still believe that it has to stay within the 65 Watt power limit.
