### Post by Knowle on 2010-11-19
I looked up the Studio 1737, it appears to have a Broadcom card for wireless. So make sure your connected with an ethernet cord and go to System->Administration->Hardware Drivers and install the Broadcom STA wireless driver. Then reboot. Everything should work after that.

Make sure you DO NOT upgrade to 10.10 unless someone else knows otherwise, last time I was using it there were still issues with the B43(Broadcom) drivers.
