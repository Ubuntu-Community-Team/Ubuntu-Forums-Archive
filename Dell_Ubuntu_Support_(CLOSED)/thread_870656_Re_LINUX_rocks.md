### Post by gbrainin on 2008-07-26
According to the user manual (available at support.dell.com), this model can have anywhere from 128MB to 2GB of memory.  It depends on what you ordered when you bought it.

You could go to some trouble to find out how much memory you've got, or even to buy more if necessary, but given the cost of Ubuntu (basically some time and a blank CD), why not just try it?  Running the live CD will not make any permanent changes to your computer unless you choose to install, and if it doesn't work, you're only out a CD-R.

Note, however, that on any computer running from the live CD will be much slower than running a proper installation from the hard drive, so don't let that discourage you.
