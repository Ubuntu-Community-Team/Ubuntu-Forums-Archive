### Post by Island_Jon on 2010-12-09
You could try improving the fonts.
1. turn on subpixel smoothing if you have an LCD screen:
[I]System - Preferences - Appearance - Fonts - subpixel smoothing (lcd's).
[/I]
then see if you like these choices.
Or
2. install extra fonts: 
*Applications - Ubuntu Software Center.*
*View > All Software.*
*search term: restricted*
*Ubuntu restricted extras* (True Type fonts are included here along with other media stuff)
