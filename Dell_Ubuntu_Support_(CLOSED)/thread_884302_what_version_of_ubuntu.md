### Post by tamoneya on 2008-08-08
you should first of all be using the desktop.  As for ubuntu, kubuntu, xubuntu it doesnt really matter.  It is more a matter of preference.  They have different Desktop Environments.  If you have no preferences just start with ubuntu.

As for 32 bit or 64 bit you can use either.  Personally I recommend you use 32 bit unless you have 4GB or more of ram in which case you should use 64 bit.  Read more here: [http://ubuntuforums.org/showthread.php?t=502754](http://ubuntuforums.org/showthread.php?t=502754)

### Post by vandorjw on 2008-08-08
**Version 8.04** (Hardy Heron)

Desktop Edition

Ubuntu or Xubuntu or Kubuntu
(My preference goes to Ubuntu)

Hope that answers your question.

PS: if you have vista, wait till October some time. Ubuntu 8.10 comes out in October, and it uses a newer Kernel. So you will likely have less problems with heat, hibernating and or other things.

(I am not saying that 8.04 has those problems, only that the chance of having those problems is less) - hope that makes sense.
