### Post by ajgreeny on 2011-02-07
Logout from the session on the Dimension desktop, and when you get to the login screen enter or click on username, but before you type the password, go to the session menu bottom right and see what is selected.  You need Gnome to be showing, none of the others that are available.

Is this a machine straight from Dell, or has it been customised by a previous owner?  It does sound a bit unusual to have all those icons from the start; by default ubuntu only normally has Computer and Home showing, if I remember correctly.

If it is Gnome running, the answer is to customise it as you want, but one step at a time; let's see what you've got first.
