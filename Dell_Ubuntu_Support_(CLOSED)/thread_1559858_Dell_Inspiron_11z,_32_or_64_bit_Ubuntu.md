### Post by mikewhatever on 2010-08-24
Since you haven't mentioned what 'everyday use' is for you, nor any particular reason for using the 64bit OS, I'd recommend going with the 32bit version. That said, there isn't much of a difference.
As for the screen size, it's very subjective. Some like the netbook interface, others don't, some like running compiz even on a 9 inch screen, which you can't do with the UNR.
Do you like full screen windows, or do you prefer the standard desktop look?
