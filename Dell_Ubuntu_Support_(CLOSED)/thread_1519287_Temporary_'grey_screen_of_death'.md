### Post by Revolutionary101 on 2010-06-27
This is just a sign that the program has become unresponsive for a few seconds. It's kinda like when you have a program on Windows lock up. Although, with Ubuntu, it will hopefully fix it self.

This occurs to me too on rare occasions when a program doesn't have enough memory available. This is most common with a web browser because they require a lot of memory.

Bottom line it isn't a big problem, unless it is happening frequently.
