### Post by mikewhatever on 2010-08-03
Boot from usb, install Ubuntu while resizing the Windows partition, you'll be presented with the appropriate choice at the partitioning stage.
The wireless not working is not exactly a bug. When done installing, boot into Ubuntu, hook up an ethernet cable and install the suggested wireless driver.
