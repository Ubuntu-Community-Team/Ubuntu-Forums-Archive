### Post by cascade9 on 2010-10-31
I'd guess that the time isnt really random. If its a set number of hours out (like ugm6hr sugggested) then the timezone isnt set right. 

More likely, you've got a flat BIOS battery. When the BIOS battery goes flat, the BIOS will lose the settings after a short while, and then when powered back up will go back to whatever defualt date and time the BIOS is set for.
