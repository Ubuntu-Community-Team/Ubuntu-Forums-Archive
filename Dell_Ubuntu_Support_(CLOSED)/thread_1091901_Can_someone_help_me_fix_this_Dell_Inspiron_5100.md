### Post by armandh on 2009-03-10
may be a bad bios battery. if not that simple

pull all the cards all the memory, pull the bios battery, drive connections and the powersupply

then  put in the new bios battery known good working.... memory, power supply, and if not on board known good video.
test with known good KB/mouse if not beeping add stuff one thing at a time

still beeping?
bad board or bad processor or jumpers/ case connection set wrong
down load board's instruction sheet

if condition when pulled from service is unknown assume the worst.
