Ok, pal, you need the Ubuntu ISO recorded to CD-r and boot from the CD/DVD drive, start installation and wipe the "windows" drive with the partitioner included in the installer. Format it as Ext3 "/" mount point and add "swap" drive, for example 1 gig (if you have enough hard drive space). "Secondly, how do I install Ubunto to my Dell desktop if I no longer have Windows on there?" - you don't need "Windows" to install another OS. Burn the image to Cd and reboot... that's all. I have long experience with Ubuntu and i can tell you that it's very user friendly and easy to use, especially if you have been using "Windows" only so far. the "migration" is very easy, just "one click away".
