### Post by harryhig on 2011-05-13
I've installed 11.04 & it doesn't seem to recognize 1366x768 resolution. I get erratic behavior- blank screens, defaulting to 1024x768, etc. Apparently the graphics card is a ATI Mobility Radeon HD 4570. I'm a relative newbie to Ubuntu, so I need help given without a lot of technical jargon, please. Thanks!!

### Post by mikewhatever on 2011-05-14
You may need to install the proprietary ATI driver. Click the Ubuntu icon (top left) and start typing 'Additional Drivers'. The Additional Drivers utility should offer an easy way to activate the driver for your card.
Note, you should be connected to the internet while performing the above steps.

### Post by fuddledumpy on 2011-05-26
Same thing happened to me with my studio 15. Push the windows key and type "monitors" in the dash search bar. Click the monitors icon and select the 1366x768 (16:9) resolution manually. Hope that helps!
