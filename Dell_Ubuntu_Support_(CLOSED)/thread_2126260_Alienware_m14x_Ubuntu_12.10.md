### Post by prodigy_ on 2013-03-16
> can not get a definite answer of whether or not Ubuntu 12.10 can be installed using Wubi or as the only OS
Yeah, because nobody does that. :) As far as I understand, it might be possible to achieve what you want as long you as you use a separate partition for each of the following:
- Windows boot files
- Windows installation
- Ubuntu installation

Then you could delete the Windows partition. But that's pure theory. Why would you go through all the trouble if you don't need Windows in the first place? Not to mention that Wubi isn't recommended for permanent installations anyway.
