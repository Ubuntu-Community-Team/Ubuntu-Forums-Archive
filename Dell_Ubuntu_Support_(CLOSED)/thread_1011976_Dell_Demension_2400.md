### Post by snowpine on 2008-12-15
Hi there, I think the latest version of Ubuntu 8.10 should work fine, provided you have at least 512mb of ram. I am running 8.10 on a much older Dell than yours. :)

Wireless is hit or miss, so I'd suggest finding out exactly what kind of wireless card you have and using the search function to find a how-to.

You can test things out from the Live CD to see if you like it, and that all hardware is supported, before actually installing to your hard drive.

Good luck!

### Post by snowpine on 2008-12-15
You can just burn a Live CD and test your wireless from the live mode (without installing). It might just work right out of the box. :) If not, there are several different methods to troubleshoot, but it depends completely on what type of card it is. Worst case scenario is you find the Windows driver for your card and use a program called 'ndiswrapper' to use the Windows driver. We were all new to Linux at one point or another, so don't worry about asking lots of questions. But I am optimistic it will just work for you with no tweaking...
