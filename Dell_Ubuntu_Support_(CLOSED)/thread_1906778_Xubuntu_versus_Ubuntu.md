### Post by Toz on 2012-01-09

> I have a Dell Vostro 1720, and I have enabled all of the Ubuntu (Gnome, KDE and Unity) desktops, and I just installed the Xubuntu desktop as an option. When I logged in to Xubuntu, I was pleasantly surprised. Even though Xubuntu uses fewer resources, I really couldn't tell much of a difference between Xubuntu and Ubuntu as far as dependability and software.  Is there any real difference?
The base is the same. The only real difference is the desktop environment (xfce), the default applications (e.g. leafpad instead of gedit), and some minor tweaks and configurations to support the environment. 
> If I had a machine with less hard drive space and I installed Xubuntu, would I still be able to install all of the Ubuntu software that I wanted (such as Libre Office) onto Xubuntu?
Assuming you had adequate free space, absolutely!

Enjoy xubuntu.
