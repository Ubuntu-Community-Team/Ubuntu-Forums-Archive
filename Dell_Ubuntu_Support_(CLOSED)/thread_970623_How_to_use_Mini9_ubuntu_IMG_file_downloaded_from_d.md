### Post by davidi on 2008-11-06
Ok this is what I did:
1. I downloaded the .img file from the dell web site
2. I plugged in a 2GB USB drive into my linux machine 
3. I downloaded [ImageWriter]("http://ppa.launchpad.net/ogra/ubuntu/pool/main/u/usb-imagewriter/usb-imagewriter_0.1-1~ppa1_all.deb")
4. I ran Imagewriter to create a bootable USB drive
5. I booted the Mini9 from the USB key and it installed the dell image
