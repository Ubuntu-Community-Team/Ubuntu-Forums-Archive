### Post by alienprdkt on 2009-12-01
You will want to go to System > Administration > Software Sources 

Click on the Updates Tab 
select Pre-released updates
also make sure that under the same Tab 'Updates' where it says Release upgrade is set to normal releases.

Hope this help you out.
