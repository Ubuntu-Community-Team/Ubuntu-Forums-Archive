### Post by cgbolton1 on 2008-07-11

### Post by Roger Steiner on 2008-09-13
I have two Dell Dimension 2400 that I have converted to Ubuntu 8.04. The first install was flawless and I added the Edubuntu package and all the education programs. Update manager loaded and installed all the updates.

On the second Dell, when Update Manager started to run, things ground to a halt. It would not download the updates. After three tries, I noticed that the on-board network card light was dim. I installed a PCI NIC and Update Manager took off and downloaded all 279 updates.

Both units only have 256 megs of ram. I am going to pick up a couple of 1 gig's and see it things run any faster.

No other problems so far.
