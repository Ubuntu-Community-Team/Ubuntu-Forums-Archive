### Post by TobyReynolds on 2009-02-22
If you don't need it urgently, you could try and wait for a few weeks as Dell seem to be changing round their line-up at the moment. Or you could just buy with Vista and wipe it clean. Here is my experience:

I wanted to order a XPS M1330 with Ubuntu a few weeks ago but they weren't available then, only Inspiron Mini 9, Mini 12 and 530 desktop, yet I was sure when I looked the week before that they had a much larger range. So I ended up buying one with Vista, then doing a format and clean install of XP and Intrepid - works perfectly. I was a bit grumpy when I looked back the other day and they now have 1330s again, but when I customised one on the website to the same spec as mine it came out at only £1 cheaper, which is ridiculous considering OS is free!! And it doesn't even have the fingerprint reader, presumably because that doesn't work with Linux, not that I use it anyway, but still...!

So if I were you I'd just buy one with Vista and then do wipe-clean and install Ubuntu, or you have the option to dual-boot it if you like. It probably won't cost you much, if any, more money and I haven't read the best things about Dell's Ubuntu support either, for example not keeping the repositories up to date or slacking on updates, but maybe that's all sorted now.

Hope this helps
Toby
