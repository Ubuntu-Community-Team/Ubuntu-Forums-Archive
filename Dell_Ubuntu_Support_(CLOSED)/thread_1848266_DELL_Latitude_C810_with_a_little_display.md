### Post by gepponline on 2011-09-22
I solved the problem!

I've downloaded the file
 NVIDIA-Linux-x86-96.43.20-pkg1.run
 from nvidia official site, installed it and now all work!
I've only set session to Ubuntu Classic No Effects 'cause it was too slow with normal session.
