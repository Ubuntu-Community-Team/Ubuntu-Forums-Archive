### Post by falcon61102 on 2008-11-24
The majority of the time it's much easier to start with Windows and then add Ubuntu later.  The reason for this is because windows was designed to occupy the first partition of the first drive and you may run into problems installing it after Ubuntu or any other OS is already installed.  In general, to save yourself some hassle, it's just recommended to start with Windows.

As far as your question with the buttons, I have a M1530, so it's slightly different and it only required a slight tweak to get all the buttons working properly but I'm not sure about the M1330.
