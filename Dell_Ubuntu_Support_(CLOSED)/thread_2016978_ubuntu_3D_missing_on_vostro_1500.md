### Post by mikewhatever on 2012-07-05
That's not an issue. Not all hardware can run Unity, even if you manage to install and run lots of Nvidia drivers. For more info, check out the system requirements:
[https://wiki.ubuntu.com/DesktopExperienceTeam/UnityHardwareRequirements](https://wiki.ubuntu.com/DesktopExperienceTeam/UnityHardwareRequirements)
