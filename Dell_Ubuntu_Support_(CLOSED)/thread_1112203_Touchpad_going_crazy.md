### Post by superm1 on 2009-03-31
There are 3 solutions:

1) Downgrade the BIOS to an earlier version
2) Upgrade to 9.04 (there is a workaround in it's kernel)
3) There is a kernel command line option that you can append to work around this.  You'll have to google for it though, I don't remember it offhand.
