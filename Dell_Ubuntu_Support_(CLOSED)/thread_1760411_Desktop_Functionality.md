### Post by gbrainin on 2011-05-17
There should be a menu bar, by default at the top.  It's possible that your graphic settings are off sufficiently that it is off the edge of the physical screen; you may be able to get it to scroll by running the mouse cursor off the top edge of the screen.

Once you get the menu bar, you can make adjustments to system settings (such as graphics).  If that fails, you'll need to make the adjustments via the text interface, but someone else more experienced will need to tell you how to do that.
