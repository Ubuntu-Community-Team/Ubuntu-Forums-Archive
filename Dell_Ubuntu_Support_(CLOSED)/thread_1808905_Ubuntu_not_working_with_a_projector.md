### Post by Mekik on 2011-07-21
Dear friends,

mine is ATI Radeon HD4330. Steps taken as follows:-
1. install aditional drivers.
bash icon -> system settings -> additional drivers
reboot may required.
2. Connect external monitor / HDTV connection.
3. Open ATI radeon setting.
system settings -> ATI Catalyst Control Center (Administrative) -> display manager.
reboot may required.

cheers...
