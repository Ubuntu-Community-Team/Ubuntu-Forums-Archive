### Post by mikewhatever on 2012-08-02
There is no hardware info, but since it lacks pae support, and isn't a netbook, it's save to assume that the computer in question is quite old. If that's the case, I'd recommend Xubuntu, which has a non-pae generic kernel.

If you must have Ubuntu, the only way to get 12.04 would be to install an older release (10.04, 11.10), and then upgrade, or use the minimal ISO or netinstall.

PS: Read more on PAE here -> [https://en.wikipedia.org/wiki/Physical_Address_Extension](https://en.wikipedia.org/wiki/Physical_Address_Extension)
