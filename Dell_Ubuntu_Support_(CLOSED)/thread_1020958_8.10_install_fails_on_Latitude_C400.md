### Post by delrieu on 2008-12-25
This issue is solved.

Here is what appended:

1) auto log-in prevented me from accessing the log-in panel and the emergency command-line window

2) I installed 8.10 alternate and ended-up with the same issue: after login, the computer freezes

3) Hardware rebooted, and logged-in with the console: found an .xsession-errors file with "WARNING: Application 'gnome-wm.desktop' failed to register before timeout"

4) googled and found that this is due to compiz 
[https://bugs.launchpad.net/ubuntu/+source/xorg/+bug/292376](https://bugs.launchpad.net/ubuntu/+source/xorg/+bug/292376)

5) command line FIX:
sudo apt-get update
sudo apt-get dist-upgrade
metacity --replace &
emerald --replace &
sudo reboot
