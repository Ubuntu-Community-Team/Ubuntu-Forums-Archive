Hi, Fresh XPS 13 owner here too, most stuff works amazingly, so let's iron out those few kinks!

First: Graphics works fantastically, except as told elsewhere, that hybrid SLI ( different cards ) is not yet supported. I did just this:

After selecting nvidia 177 driver from "hardware drivers" and getting the command prompt after restart:
sudo nano /etc/X11/xorg.conf
Added BusID line into Device section:
```
Section "Device"
..
        BusID   "PCI:03:00:0"
EndSection
```

you get this BusID from 
```
lspci |grep VGA
02:00.0 VGA compatible controller: nVidia Corporation GeForce 9200M GS (rev a1)
03:00.0 VGA compatible controller: nVidia Corporation GeForce 9400M G (rev b1)
```
the 9200M chip is obviously the builtin and will be idle in this case, use ID from 9400M line.

the nvidia-fix.py script in dell wiki is weird because it searches for line
```
Option  "IgnoreDisplayDevices"  "TV"
```
in xorg.conf and wants to insert the BusID line after that, but there is no such line in xorg.conf in case of clean ubuntu 8.10 install. So the script just fails and frustrates people.

until now everything I have tried works, except hibernate from shutdown menu. Amazingly running "sudo s2disk" WORKS, so it has to be some cosmetic thing to figure out.. now I'll go and create another thread for that, because 3 hours of searching have not given satisfying results.
