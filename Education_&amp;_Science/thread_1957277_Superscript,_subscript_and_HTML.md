### Post by souravc83 on 2012-04-12
LaTeX is supposed to be the best for that sort of a thing.
Its not too difficult to start with. Just open a regular text editor (like Gedit), and start with a basic template page. Or use TexMaker or one of the other LaTeX editors.

As for other ways, depends on how you want to write the formula. By typing it out with your hand (the LaTeX way) or by using the mouse.
In the latter case, your best bet is mathtype.You can use Word in wine and use mathtype along with it. I think this works, but I have never tried doing it. Also, you can use one of the LaTeX based equation editors, which lets you insert individual letters from the mouse. I have used EqualX, which is in the Ubuntu software center.
You can save the formula as a png image and put it in HTML.
You can also save the LaTeX source code and save it, but if you want.
