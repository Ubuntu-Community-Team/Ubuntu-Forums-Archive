### Post by cwarner7_11 on 2011-04-23
[CAELinux]("http://www.caelinux.com/CMS/") is an Open Source distro built on Ubuntu 10.04 64-bit that focuses on numerical analysis (simulation, FEA, CFD, etc.), with a plethora of applications of interest to the scientist or engineer.  A very versatile package, with everything already set up and ready to run.  I have published a recently completed "[case study"]("http://ubuntuone.com/p/oKn/") on UbuntuOne that illustrates the power of some of the packages such as Salome (3D modeling, meshing and post processing), Elmer (CFD, heat transfer), Code_Saturne (CFD, heat transfer), Paraview (data visualization,post processing), plus a couple of small apps that I have added to my own setup (g3data for extracting data from printed graphs), and Eureqa (an extremely powerful Open Source curve fitting application for creating expressions from raw data).

### Post by disabled_biscarri on 2011-09-23
auto-answer to the question about CAElinux running on Mac: it was necessary to specify in "preferences/boot disk" the desired drive from which to restart the computer; once this has benn done, the mac starts with ubuntu and caelinux apps are available

attention, be careful if you do that because I've had a problem > when shutting down from ubuntu, the mac has kept the order of booting from DVD drive and it was'nt possible to restart it normaly; it was asking for a valid DVD to boot from

since I did'nt have a restart disc prepared (not such caution done), I've tried to restart the Mac from the installation DVD (which is not a valid restart DVD, I've learned); so that the situation was getting worse since I could'nt even extract the installation DVD

finally I've found an emergency way to extract the disc: restart the mac while mantaining the trackpad pushed; then the disc has been extracted anb the Mac initialized normaly
