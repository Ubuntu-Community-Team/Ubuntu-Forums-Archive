Description
===========
This is a VMware virtual machine built by Happy Penguin
Taiwan ([http://happypenguin.tw/](http://happypenguin.tw/)) for people interested in
Zorin OS 3 Educational.

Zorin OS is an Ubuntu-based Linux distribution designed
especially for newcomers to Linux. It has a Windows-like
graphical user interface and many programs similar to
those found in Windows. Zorin OS also comes with an
application that lets users run many Windows programs.
The distribution's ultimate goal is to provide a Linux
alternative to Windows and let Windows users enjoy all
the features of Linux without complications.

Host OS System Requirements
===========================
* Standard x86-compatible or x86-64-compatible PC
* Processor speed – 1.0 GHz or faster
* Memory – 1 GB minimum (2 GB recommended).
  You must have enough memory to run the host operating system,
  the virtual machine, and applications on the host and guest
  operating systems.
* Linux, Windows or Mac OS X as host operating systems

Just download the virtual machine archive, unpack it, open with
