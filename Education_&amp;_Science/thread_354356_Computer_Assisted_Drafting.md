### Post by baslin6 on 2007-02-12
> **Sniper99 said:**
> i have heard about some free linux programs, for computer assisted drafting.....and i need one, which is the easiest program, i am looking for a more PRO ENGINEER type program....if yoiu know one, how do i install it

qcad is a 2D only package with a free "community" version, sans scripting.

If you simply want to learn to use CAD for free, their community version should fill the bill sufficiently.

If you are familiar with Pro/E, I suspect that you desire a package that does 3D.  I am unaware of any freely available linux package that is comparable to Pro/E.  VariCAD is a nice 3D package that, to me, seems reasonably priced at US$500.

As for installation, prior to installing VariCAD, you must have KDE and/or kubuntu-desktop installed; it is required. Qcad has no such requirement.
