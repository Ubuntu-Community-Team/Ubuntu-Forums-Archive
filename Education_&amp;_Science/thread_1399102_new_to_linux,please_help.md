### Post by amsterdamharu on 2010-02-09
Ubuntu uses a generic linux kernel, what makes ubuntu is the dynamics of several programs like apt to manage your installed software and the installer to make installing the kernel, it's software and it's drivers simpler.

I am not too sure what you want to do, reprogram the kernel or see how Ubuntu ties everything together.

Kernel sources should be online I think here:
[http://www.kernel.org/](http://www.kernel.org/)

To see how Ubuntu ties stuff together try to mount an initrd and see what the scripts do.
[https://lists.ubuntu.com/archives/ubuntu-users/2006-August/091194.html](https://lists.ubuntu.com/archives/ubuntu-users/2006-August/091194.html)
