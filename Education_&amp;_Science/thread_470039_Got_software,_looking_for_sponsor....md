### Post by mortenkjeldgaard on 2007-06-10
I am a crystallographer engaged in moving my lab's software base from RPMs to DEB format. I can contribute quite a lot of packages for bioinformatics/crystallography; this is all software that is being used in our lab.

Tonight I have uploaded a few science packages to REVU ([http://revu.tauware.de/](http://revu.tauware.de/)). If you have interest, and/or if you are a MOTU, I'd appreciate if you'd take a look at them. They are:

chooch - Anomalous scattering factors from X-ray fluorescence data
cgraph -  A minimal library for scientific PostScript plots (needed by chooch)
kaksi  - Protein secondary structure assignment program
theseus -  Maximum likelihood superpositioning of macromolecular structures
wulfware -  "wulfware" for monitoring cluster nodes, via curses and web interfaces
kssh - - KDE front end to ssh (makes a secure shell session in KDE's konsole)

If you are able to sponsor these packages, I would be grateful.
