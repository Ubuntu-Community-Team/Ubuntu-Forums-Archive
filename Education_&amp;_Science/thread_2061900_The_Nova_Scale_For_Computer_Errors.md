### Post by Jesse Tustin on 2012-09-23
Code   |         Problem                  |      Level
--------------------------------------------------------------
N-1:  General software error.  |  Unimportant.
N-2:  Severe software error.    | May render program unusable.
N-3:  Registry error.                    | Minor.
N-4:  Registry missing/corrupt. | Bad.
N-5:  Windows program error. (example. svchst.exe)  | Will leave a mark.
N-6:  Missing or corrupt WINDOWS files. | Terrible.
N-7:  Hard disk error                  | Dangerous.
N-8:  Hard disk failure or other hardware failure. | Deadly.
N-9:  BIOS error.                          |  Corrupt.
N-10:  BIOS failure.                     |  Extraditely dangerous :  Computer bamboozeled.
