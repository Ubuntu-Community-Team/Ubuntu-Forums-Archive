### Post by t155mod on 2010-12-03
> **blackplague1347 said:**
> Intriguing. How much does this cost? I'm from the US, if that makes a difference. 

The course currently costs £190.00 and is due to run again in May 2011.  Living in the US should make no difference as the Open University provides all support online.  However you could contact the Open University to clarify this.  Alternatively, read the details at [http://www3.open.ac.uk/study/explained/where-can-i-study.shtml](http://www3.open.ac.uk/study/explained/where-can-i-study.shtml) 

How does it work? 

"The course is designed to be studied over a ten-week period, with approximately ten hours of study a week."  The work is broken down into tasks, with each task taking various amounts of time.  You will learn to install Virtual Box, then different flavours (more aptly distributions) of Linux.  By the end of the ten week period you should be have been to:

[LIST]
[*]install your own version of Linux
[*]update and manage the operating system
[*]learn the basics of the command line environment
[*]work with the file systems in Linux
[*]discover the structure of Linux and similar operating systems
[*]appreciate some of the history behind Linux and other operating systems
[*]set up a basic web server
[*]manage users and user groups
[*]manipulate Linux to suit your own personal preferences
[*]work with some of the lower level-functions of Linux
[*]appreciate the first principles of Linux systems security
[*]appreciate the purpose and quality of the open source community.
[/LIST]

Is it just like any other online class in college/University?

This Open University course is supported by a number of very experience Associate Lecturers (ALs).  The ALs support the forums and answer student questions and of course students help each other too. The ALs generally undertake the marking at the end of the course too.  The final (and only) coursework in T155 is an EMA (End of Module Assignment) is a practical activity that shows you can install Linux and manipulate files, directories and users.   

I assume this doesn't get put toward any sort of degree or anything, so what's the grading policy like? 

The T155 course is designed to be reasonably 'stand-alone', however, the credit for this course can be used against an OU bachelors course called BSc (Honours) Information Technology and Computing, the details of this course can be found at: [http://www3.open.ac.uk/study/undergraduate/qualification/b13.htm](http://www3.open.ac.uk/study/undergraduate/qualification/b13.htm)

I apologize if all of this information is on the website and I've simply missed it >_>

I hope this helps.

Brian (t155mod)

### Post by not_a_phd on 2010-12-03
On a related note... 

I recently completed the Linux from Scratch 'course', located at: [http://www.linuxfromscratch.org/]("http://www.linuxfromscratch.org/")

I can highly recommend this to the intermediate Linux enthusiast who learns best by doing. Looking at the laundry list of course elements given by T155mod, I can attest that Linux from Scratch and Beyond Linux from Scratch cover these as well. 

The student that takes the time to research the fundamentals behind some of the more esoteric commands and packages that are used and installed will gain a foundational knowledge of a Linux installation that will serve them very well.

Oh, and it is free.
