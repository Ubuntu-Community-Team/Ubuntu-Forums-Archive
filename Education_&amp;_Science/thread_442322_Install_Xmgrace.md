### Post by WW on 2007-05-16
Install the package **grace**, either with Synaptic, or with the command
```

$ sudo apt-get install grace

```
The package is in the universe repository, so be sure to enable that first, if you haven't already.
(Here's how: [https://help.ubuntu.com/community/Repositories/Ubuntu](https://help.ubuntu.com/community/Repositories/Ubuntu) )
