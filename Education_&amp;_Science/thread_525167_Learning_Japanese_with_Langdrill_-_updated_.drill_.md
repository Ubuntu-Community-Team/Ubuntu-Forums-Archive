### Post by DNAspark99 on 2007-08-14
I'm studying Japanese, and since I found the 'langdrill' package it's been a great help with memorizing Hiragana and Katakana - but I noticed it doesn't have them all! So I updated the .drill file and also fixed a few minor mistakes I came across with the other stuff, tho I havn't examined it all yet... I looked into submitting this as an 'official' patch to the langdrill package maintainers, but really have no idea how to go about doing that? Perhaps someone can enlighten me on the process.... 

in the meantime, here's my updated /usr/share/langdrill/japanese.drill file for anyone who's interested:

sense=0
directSenseName=Japanese
reverseSenseName=English
quizzAnswerNr=8
hQuizz=40
wQuizz=300
# the timer value is given in minutes.
# it represents the time between the automatical 
# activation of the langdrill.
timer=10
# use this to set how many good answers the langdrill
# waits till hides by itself. limitedAnswer must be 1
# for this to take effect
