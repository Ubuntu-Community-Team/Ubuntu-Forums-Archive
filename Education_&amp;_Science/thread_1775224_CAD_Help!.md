### Post by Gemnoc on 2011-07-31
> **amauk said:**
> holy crap
nice list

It appears like an impressive list, but when you start checking out the links you realize there is a lot of abandonware listed.

And this list does not include one of the most promising 3D CAD app on Linux IMHO (with HeeksCAD), FreeCAD. Well there's one listed, freeCAD, but the fact is there are two different projects named FreeCAD, the one listed is not developed anymore and is freeware, while [FreeCAD registered on Sourceforge]("https://sourceforge.net/apps/mediawiki/free-cad/index.php?title=Main_Page") is open source and multi-platform. It will eventually have similar features to history-based parametric CAD modelers like SolidWorks, but it will take time. The development team is small and they work part-time, but they don't let up. You can actually do some work with it, even if it's still considered alpha software.

FreeCAD is in Ubuntu's repository, but it's an old version. There is a daily build PPA for anyone interested in checking a CAD package in development, details on adding the PPA are on the project's [Downloads page]("https://sourceforge.net/apps/mediawiki/free-cad/index.php?title=Download").

*Disclaimer: I've helped set up the FreeCAD project's PPA, but I'm not one of its developers.*
