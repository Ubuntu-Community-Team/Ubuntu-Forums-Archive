### Post by spinoza666 on 2010-04-03
I don't know of any audio/video sources for learning R, but there's of course CRAN, which has a few manuals and everything R related:

[http://cran.r-project.org/manuals.html]("http://cran.r-project.org/manuals.html")

Might wanna start with 'An Introduction to R', and do the tutorial included. Then I would just search for tutorials online, and grind through a few of them.

The best way to learn R, I presume, is by doing exercises, or when analysing real data of your own. I doubt that you will learn very much from audio/video sources, so you should start with tutorials and such, the hard way like everyone else.
