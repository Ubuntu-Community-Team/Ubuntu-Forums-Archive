### Post by dvase on 2008-02-21
It looks like Christian Schenk would [like to port the MikTeX gui to Linux via Qt]("http://miktex.org/features/Default.aspx"), for a [price]("http://micropledge.com/projects/miktex-mpm-qt?do=pledge") ($3000).

As a user of MikTex/LaTeX under both Windows and Ubuntu, I applaud the thought of an easy to use linux based LaTeX package manager.  However, some of the finicky issues I've had with MikTex and the package repositories in the past make me wonder if more than simple port is needed.  Perhaps we need to rethink what a LaTeX package manager should be.

What do you think about this proposed port?  More importantly, will you add to the pledge?
