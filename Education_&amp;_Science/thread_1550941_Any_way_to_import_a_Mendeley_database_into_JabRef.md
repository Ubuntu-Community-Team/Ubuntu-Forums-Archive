### Post by sibyphilips on 2010-08-11
I think you should select the references by CTL+A or Edit>Select all....

and File>Export as Bibtex format or RIS or Endnote XML format....(remember the citations marked as "need review" will not be exported)

I have not used JabRef so I do not know more about that however Jabref site says that it can import any of these formats. If you have tried all these and I am wrong please ignore this...
