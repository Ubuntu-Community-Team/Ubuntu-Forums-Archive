### Post by italophil on 2009-06-25
I'd suggest zotero, a firefox add-on. For books, you just need to type the ISBN in google books (or similar sevices) and then click on a icon to import the information into zotero.

### Post by slaanco on 2009-06-25
Alexandra Book Collection Manager is probably what you are looking for:

```
sudo apt-get install alexandria
```

allows you to import book information from amazon etc.
