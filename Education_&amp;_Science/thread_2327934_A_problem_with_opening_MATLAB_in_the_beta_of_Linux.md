### Post by rewyllys on 2016-06-15
For the possible interest of users of MATLAB, I have just posted a message, in the Linux Mint Forums, reporting the solution to a problem I recently encountered. 

 The problem was that MATLAB, which had been running satisfactorily in Linux Mint 17.3, crashed when I first tried to open MATLAB after installing the beta of Linux Mint 18 Cinnamon. The problem was first reported to MATLAB as having arisen from a file in Kubuntu 15.04, which file was presumably in Ubuntu 15.04, as well as in Ubuntu 16.04, which is the basis of LM 18.

The solution is reported [in this thread]("https://forums.linuxmint.com/viewtopic.php?f=47&t=223619").
