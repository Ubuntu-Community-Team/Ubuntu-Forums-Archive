### Post by greenstar on 2007-10-03
I think it's a great idea and I'll be watching. Maybe I can  also contribute a little bit. There are lots of people who could benefit from a project like this (IMHO).

With a quick search in synaptic, I found 3 screencapture packages (all in universe):
byzanz, recordmydesktop & istanbul. To install do the following at terminal:
```
sudo aptitude install byzanz
``````
sudo aptitude install istanbul
``````
sudo aptitude install recordmydesktop gtk-recordmydesktop

```Istanbul actually mentioned audio recording also. 

Also see VideoLink in synaptic (it's also in universe). 
VideoLink's description sounds like it might be useful for this project. 

Description from Synaptic: 

VideoLink is intended to provide a simple way of producing DVDs with attractive and usable menus.  It converts HTML pages into DVD menus by rendering them in Mozilla and reproducing their link structure. This allows you to design DVDs using familiar HTML editing tools or your favorite text editor.

To install videolink do:
```
sudo aptitude install videolink
```Greenstar

### Post by bluenova on 2007-10-04
Have a look at reconstructor for a custom Ubuntu:
[http://reconstructor.aperantis.com](http://reconstructor.aperantis.com)

What is Reconstructor?
Image 	Reconstructor is an Ubuntu GNU/Linux CD Creator.

It uses the Desktop(Live), Alternate(Install), or Server disc as a base, and then allows for user customization.

For the Ubuntu Desktop base, you can customize the entire environment.  For instance, you can add/remove software, change the default look (splash, themes, fonts, wallpaper, etc.), add desktop links, etc. 

For the Alternate and Server bases, you can add any additional software to the disc that you would like installed.

Reconstructor is written in python and is licensed under the GNU General Public License (GPL).
