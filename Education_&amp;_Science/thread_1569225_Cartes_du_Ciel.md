### Post by gunksta on 2010-09-06
Try Celestia instead. It's the same sort of program, and really easy to install.

If you are running Ubuntu or Xubuntu:
```
 sudo apt-get install celestia-gnome 
```

If you are running Kubuntu:
```
 sudo apt-get install celestia-kde 
```

It will take a few minutes to install and it works best on a reasonably recent computer, but I've used it with Intel's embedded crap-ware video system on several computers just fine.
