### Post by stumbleUpon on 2008-09-04
Upgrade to 6.0.3 -- see this

[https://bugs.launchpad.net/ubuntu/+source/xorg/+bug/197163](https://bugs.launchpad.net/ubuntu/+source/xorg/+bug/197163)

Or else

Disable compiz (Right click on desktop --> Change desktop background --> Visual effects --> Choose none)

And then start Mathematica from the command line

```
mathematica -defaultvisual
```
