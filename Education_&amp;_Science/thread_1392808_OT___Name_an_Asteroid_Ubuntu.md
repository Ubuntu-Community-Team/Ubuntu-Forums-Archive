### Post by fritzm on 2010-01-28
Sorry for the off topic post, but I think there is some connection to this forum. 

The South African Astronomical Observatory is running a competition to name a recently discovered asteroid; the name is being selected by public vote from a short list of names, one of which is Ubuntu. I think it would be cool to have an asteroid named after a Linux distro, especially appropriate when its founder is an astronaut, so go vote online. The website is [http://www.nameourasteroid.org.za/](http://www.nameourasteroid.org.za/), and voting is open until 31 January 2010.
