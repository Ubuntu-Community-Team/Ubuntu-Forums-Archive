### Post by sonowilson on 2011-04-11
I'd go vendor neutral with the Linux Professional Institute or Comptia Linux+. These are essentially the same cert now and if you get Linux+ you can register with LPI. Kinda like two certs in one in a way. Comptia is well known in the IT world so I'd probably take the comptia test then register my score or results or with LPI.

Id probably only go red hat if I was working for or had a solid job lead with an organization that uses Red Hat.
