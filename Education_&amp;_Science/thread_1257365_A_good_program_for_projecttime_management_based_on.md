### Post by gunksta on 2010-08-01
If you ever thought you wanted to learn EMACS - this is your opportunity. If you don't want to learn how to use EMACS, stop reading.

[org-mode]("http://orgmode.org/")

This is the most amazing text processing software ever written. You can write reports with it. You can manage your calendar / tasks. You can do project management. You can draw pictures with it ala Visio.

Odds are good that whatever you want to do, org-mode can already do. If it can't, it's a few lines of LISP away.

These other solutions - they're just toys.

And for all the VIM fans out there - it is possible to run VIM within EMACS and viper-mode is pretty awesome too. (I use vim all the time because it is a good text editor. It just lacks the universal awesome that is org-mode.)
