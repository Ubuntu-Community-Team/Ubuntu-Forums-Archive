### Post by sawjew on 2008-10-08
I use QTOctave which is in the Hardy and Intrepid repositories.  This is a front end to Octave and provides a MATLAB like environment and functionality.  The version in the Hardy repositories is a bit dated but the later version (0.7.4) is available from the Hardy backports repo.  You can go to the QTOCtave [homepage]("https://forja.rediris.es/projects/csl-qtoctave/") and install Easyplot which gives a slightly improved plotting function over the default Gnuplot.
