### Post by klepto on 2008-10-04
I've looked for a good pdf editor that I could do all of the above with on Linux as I have used Foxit Reader on Windows for years. Thanks to the latest version of Wine it runs on Linux excellently. 

[http://appdb.winehq.org/objectManager.php?sClass=version&iId=12019]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=12019")

Only thing you need to do is grab mfc42.dll from somewhere or your windows box and cp mfc42.dll ~/.wine/drive_c/windows/system/
and you are there.. Enjoy..

Oh.. btw.. be sure to install wine fonts so the formatting is great:

sudo apt-get install msttcorefonts
