### Post by benner on 2007-05-10
you can use the synaptic package manager, click 'search' and type 'gcompris' into the search window.  you will get a long list of stuff.  select everything that looks appropriate and install them.  a lot of the files are language files so you can run it in different languages.  there are lots of good games in there.  if you are running edubuntu 'feisty fawn' there is a classroom add-on cd that you can also burn and install from.  it has a lot more good games and activities.  some will automatically install to the 'games' start menu and some to the 'education' one.  i use gcompris in my grade 2 classroom and the kids are glued to it.

### Post by Bartender on 2009-07-19
I've been maintaining a Linux PC at a friend's house for about a year now.  They have two young boys.
There are several different ways to go about this.  You can install Ubuntu, then download the entire Edubuntu install to a separate LiveCD and install it.  Edubuntu becomes the interface - the splash screen is different, etc.
Thats' the way I did it the first time.  Second time, I  installed Ubuntu, then went to the repos and downloaded the GCompris package.  That's all the kids were using anyway.  It showed up under Education, and we made an icon on the desktop.

I just installed Qimo to a test PC this morning.
[http://ubuntuforums.org/showthread.php?t=1217299](http://ubuntuforums.org/showthread.php?t=1217299)
I made a mistake during the installation but explained the error in the second post.  Qimo has GCompris, Childsplay, TuxPaint, Tuxmath, and a few other bits all in a dock on the desktop.  Very straightforward.  The default desktop is a limited user account, so the kids can just turn on the PC and have fun without screwing things up too badly.  When they're done you can log in under the username/password that was entered during installation to do whatever you need to do.

Qimo uses XFCE (Xubuntu), not Ubuntu.

There are also games available in the repos, like Potato Guy.  The kids like that.  Since I just installed Qimo I'm not sure if it's reasonably easy to add games to the dock, but they'll be available in the Menu (left corner of upper panel),
