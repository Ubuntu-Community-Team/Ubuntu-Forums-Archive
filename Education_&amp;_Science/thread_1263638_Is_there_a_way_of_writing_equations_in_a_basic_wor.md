### Post by hubie on 2009-09-11
There is, of course, the old [symbol font]("http://www.asppdf.com/manual_B.html") for all your greek alphabet and other needs.  However, consider this:  even if you could find such a font (assuming symbol isn't what you are looking for), I don't think you would be happy with the outcome.  Not only would you be doing a heck of a lot of mouse highlighting and clicking to change your characters, I would think it would look pretty bad in the end because you'd be trying to substitute single characters for, say, an integral sign or a fraction.

You might want to check out [LyX]("http://www.lyx.org/").  It is a poor man's LaTeX designed for quickly writing documents leveraging the power of LaTeX for, among other thing, making beautiful equations.  The user interface is very familiar for those who are accustomed to using Word or other word processors.  Check out the [screenshots]("http://www.lyx.org/Screenshots").

I'd wager that even if you found a font set that might do what you intend, you'd still write a much better document in much less time and hassle using LyX.

### Post by jacksaff on 2009-09-13
There is a latex plug-in for open office which allows you to insert chunks of rendered latex into your document. 

[http://ooolatex.sourceforge.net/]("http://ooolatex.sourceforge.net/")

Ooo already has an equation application: insert>>object>>formula, but ooolatex is much more powerful. If you need to typeset a lot of equations eventually you'll find latex easier to use.

### Post by samden on 2009-09-13
Raminaghrobis, I don't know anything about basket, but even if no-one has made a tool yet that directly interfaces LaTeX to it, once you learn the LaTeX markup language for equations you could easily note down your equations in LaTeX code and convert them later using a dedicated LaTeX equation editor. 

Or you could just switch to an equation editor, type the equation there, then drag the picture into basket, that would probably take little time at all.

Check out [Ekee]("http://rlehy.free.fr/") and [Laeqed]("http://www.thrysoee.dk/laeqed/") for a start, there are probably others as well.
