### Post by todoporron on 2007-05-29
It is hard (if not impossible) to find complete english documentation for code aster, however, on the caelinux website there are some great tutorials in english for using code aster and salome. On the code aster site, there's a forum for non french discussions. It does not work as OpenFOAM. It is mainly focused (as far as I know) to finite element analysis such as stress analysis, heat transfer and modal analysis, while OpenFOAM is focussed on CFD.

I'm still trying to compile/ install it on Ubuntu, however a good option is to try the caelinux live dvd.
