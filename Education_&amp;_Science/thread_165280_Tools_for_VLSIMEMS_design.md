### Post by tapaswimakarand on 2009-02-03
Hey
Am not very sure about MEMS, but for VLSI there are quite a few tools.

VLSI Design : Magic (as mentioned above, a cygwin distribution available in case you are using Windows)

VLSI CAD : Alliance (its too good.. has a lot of the optimization, building up the circuitry, cell design, etc parts)

VLSI Digital Simulations : IRSIM (decent simulator, can be used for all your simulations in digital. Use spice (ngspice) for the currents, and exact voltages if required).

Alliance should help you with all the tools required.

For converting your C code to VHDL you could use Spark, again has other feautures like common subexpression elimination, and other reduction techniques.

That should do the job!
