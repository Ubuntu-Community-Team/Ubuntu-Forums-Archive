### Post by samden on 2009-08-05
Two examples of officially recommended citations to give you some ideas.

R:
>   R Development Core Team (2005). R: A language and environment for
  statistical computing. R Foundation for Statistical Computing,
  Vienna, Austria. ISBN 3-900051-07-0, URL [http://www.R-project.org](http://www.R-project.org).

GRASS GIS:
> GRASS Development Team (<year>). Geographic Resources Analysis Support System (GRASS), GNU General Public License. [http://grass.osgeo.org](http://grass.osgeo.org) 

In this case I presume <year> means the current year.

For citing documentation, it is probably best to find an official website containing that documentation and cite the website following the standard format your university recommends for citing web resources.
