### Post by Claus7 on 2015-01-14
Hello,

the other day I wanted to create a graph in matlab and I observed that even if I changed the fonts of the axes and labels, the graph preview of the plot and the file that I was printing the results did not respond to my changes. The ubuntu box was 14.04.

In order to solve the problem I installed the following packages from synaptic:
gsfonts-x11 (0.22)

xfonts-100dpi (1:1.0.3)
xfonts-75dpi (1:1.0.3)

I restarted my system in order for the changes to take effect (logging out and back in might do the work as well).

Regards!
