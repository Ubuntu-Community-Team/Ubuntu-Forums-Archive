### Post by bubblehead74 on 2011-07-28
I, Librarian is a free and open source software for building web-based annotated library of scientific PDF articles. I, Librarian 2.2 beta is now available for download. I will appreciate if you help to test this new release.

This version is a major rewrite mostly of the user interface and the front-end. Some of the new features include:

[LIST]
[*]faster front-end that utilizes AJAX
[*]improved user interface
[*]improved internet database search, including Quick Save and Flagged-item list
[*]improved advanced search with the ability to save search parameters
[*]added a batch upload that works from any computer, not just localhost
[*]improved support for graphical abstracts
[*]built-in PDF viewer with page previews, ability to search in text, and copy images
[*]added discussions for each item
[*]Bibtex key can now be previewed for each item
[/LIST]

A new DEB package is now available for Ubuntu, Debian, and its derivatives. It installs I, Librarian into /usr/share/i-librarian/www to conform with Debian policies about web apps. 

Download from here:
[http://www.bioinformatics.org/librarian/downloads.php](http://www.bioinformatics.org/librarian/downloads.php)
