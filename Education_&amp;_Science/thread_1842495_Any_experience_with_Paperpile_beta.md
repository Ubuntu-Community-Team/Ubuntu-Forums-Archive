### Post by civetta on 2011-10-12
Maybe you'd like [Zotero]("http://www.zotero.org/"). It started out as a firefox extension, but now it's got a stand-alone testing version. I've found it great for organization, and I can use it to cite directly in latex documents I'm creating with lyx.

Also for pdf annotations, I've been using something called xournal which creates a layer above your pdf that you can take notes on. I usually use it in conjunction with a tablet to highlight and underline as well as type in the margins.
