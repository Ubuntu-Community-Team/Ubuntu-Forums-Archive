### Post by nextstep on 2016-02-10
MathMod-4.0 is available for download from :[SourceForge]( https://sourceforge.net/projects/mathmod/files/MathMod-4.0/)
Changelog for MathMod-4.0 (08/02/2016)
1) Add user defined parameters (up to 20) with sliders linked to them for real time mathematical models 
exploration (see "fhelix1" script video [https://youtu.be/1L2VTaLPtrA](https://youtu.be/1L2VTaLPtrA)  )
The sliders properties (min, max, step and initial position) are defined in the script but can be edited 
with a new MathMod module (see "Sliders" tabulation).
2) Add 152 new advanced script examples (for a total of 332).
3) Implementation of some PovRay functions as MathMod scripts/internal functions (fhelix1,fhelix2,f_hex_y,fmesh)

### Post by nextstep on 2016-02-17
> MathMod (or K3DSurf ) is a standalone multiplatform mathematical software that is intended to explore/export mathematical models
 described by **implicit and parametric equations**.  By Mathematical models exploration , I mean active visualization and manipulation in the 3D viewer, real time animation and morphing by using a time parameter, real time math models intersection, 4D parametric surfaces exploration in the 4D space...
Mathematical models can be exported as scripts in JSON file format or as OBJ file for further mesh  manipulation by 3D printers/programs (Blender, meshlab...)&#65279;
For more infos: [https://sourceforge.net/projects/mathmod/](https://sourceforge.net/projects/mathmod/)

### Post by nextstep on 2016-03-16
MathMod-4.1(Linux64) is now available for download from [https://sourceforge.net/projects/mathmod/](https://sourceforge.net/projects/mathmod/)
Changelog for MathMod-4.1 (14/03/2016)
1) Add tools to create/edit/delete sliders without having to edit the script manually. Video Tutorial: [https://youtu.be/FWazJIuC0zo](https://youtu.be/FWazJIuC0zo)
2) Localization Support for decimal separators as Comma (,) or Point (.) . The decimal separator is set by the field "DotSymbol" in the "mathmoconfig.js" file.
3) By default MathMod will use the Fusion/Dark theme but you can set your own style/theme with MathMod's configuration file.
4) Four new MathMod scripts, for a total of 341 integrated scripts.
5) Five Video Tutorials to quickly learn how to use some MathMod features.
Videos availables from: [https://www.facebook.com/MathMod-529510253833102/](https://www.facebook.com/MathMod-529510253833102/) 
and : [https://www.youtube.com/channel/UC-NDhd94LLS0KFUdKnloj4w](https://www.youtube.com/channel/UC-NDhd94LLS0KFUdKnloj4w)
6) Bug fix and code optimization.&#65279;
