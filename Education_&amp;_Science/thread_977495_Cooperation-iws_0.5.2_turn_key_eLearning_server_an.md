Hi,

I am pleased to announce the availability of Cooperation-iws 0.5.2.

Cooperation-iws is a turn key intranet server that fit on USB key / Hard drive. You could therefore carry it with you and boot it from several different PC.

For this release Cooperation-iws comes in sixteen flavors divided into two groups:

[LIST]
[*]The standalone releases with several core activities (CMS, eLearning, Web 2.0, data management, ...) bundled with mail server and intrusion detection system
[*]The system of servers which dedicate a server to a specific activity: eLearning, video conferencing, web 2.0, administration, gateway, groupware, ...).
[/LIST]

With the system of server you need to download several iso images of servers to have a fully functional intranet. But with the standalone server you just need to download one iso image.

The 0.5.2 releases are based on Xubuntu Hardy 8.04.1 and are available here:
[http://www.cooperation-iws.eu/CMS/?page_id=31](http://www.cooperation-iws.eu/CMS/?page_id=31).

The front page of the project is here: [http://www.cooperation-iws.eu/](http://www.cooperation-iws.eu/)

And the blog of the project is here: [http://www.cooperation-iws.eu/blog/](http://www.cooperation-iws.eu/blog/)
