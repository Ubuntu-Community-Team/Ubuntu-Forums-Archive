### Post by cantillon on 2012-07-06
I just found a firefox addon that can be used for qualitative text analysis. It's called Wired-Marker. It can be used to mark text and create a database with the text you marked. I think it was originally designed more as a bookmarking tool, but it can be easily customized to be used for qualitative analysis. You can find more info here:
[https://addons.mozilla.org/en-US/firefox/addon/wired-marker/]("https://addons.mozilla.org/en-US/firefox/addon/wired-marker/")
[http://www.wired-marker.org/en/]("http://www.wired-marker.org/en/")
It seems that it's been around for some time now. I wish I had come accross this earlier. I'll take a closer look and I'll try to write a tutorial one of these days.
