### Post by wirawan0 on 2007-10-31
You may want to check out AXIOM, available in Ubuntu distros---now a free Computer Algebra System (CAS) software. It is by default a command-line program, which may be clunky to use. A nicer way to use is to install TeXmacs, which is like WYSIWYG LaTeX-like word processor in Emacs-like editor environment. Then you can Insert -> Session -> Axiom and start Axiom that way. It will yield nicer-looking equations than its command-line counterpart. And also you can save the nice printout or make PDF out of it.
