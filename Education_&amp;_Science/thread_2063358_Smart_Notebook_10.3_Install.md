You could try Smart Notebook 11 which is now available and supports 12.04 although looking at the small print it seems it does not support the unity desktop!

### Post by ruby644 on 2013-01-16
Update

Smart Notebook 11 has an issue with pen "lag" running in Ubuntu 10.04.
When using the interactive pen on a Smart LightRaise 40wi interactive projector there is a short delay before the line is started on the board. This can be frustrating if you write quickly.
This "lag" does not appear when using 12.04.
