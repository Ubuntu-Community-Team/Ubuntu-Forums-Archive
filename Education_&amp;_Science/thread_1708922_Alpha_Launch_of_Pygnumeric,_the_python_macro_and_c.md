### Post by Cloudform on 2011-03-17
I am currently working hard on making a solid python macro language that allows you to easily write python modules for the spreadsheet program Gnumeric.

I encourage you to check out both Gnumeric and the extension I wrote Pygnumeric.  I need bug testers and users.  Everything is entirely open source

The link for my project is here:

[http://cloudform511.posterous.com/alpha-launch-of-pygnumeric-the-python-macro-a](http://cloudform511.posterous.com/alpha-launch-of-pygnumeric-the-python-macro-a)


I would also like to note that I wrote the module to be very extendable.  The base class DataHandler only requires that you be able to get a single point of data from ANY data-base.  So it should be possible to extend this for Excel or open office fairly easily.  If you do so, please tell me and I will link you to my site.
