### Post by flibble on 2007-03-31
Hiya, I'm a psychology postgrad, so plenty of APA 5th love needed here. Bibus can do most of APA 5th but it isn't possible to do all of it. Sorry I can't remember which bits it can't do, but it can do the basic journal and book/book section citations. Importing from online databases is almost non-existant...

I stopped using it because I stopped using Openoffice due to the appalling font quality I was getting (particularly with the obligatory APA 5th Times New Roman). This is fixed in 2.2! Hoorah! So I'm keen to try it again. If I get any further with APA 5th I'll post here and let you know.

As for SPSS... The best solution I found was to run XP in a vmware window... not such a good solution because you can't cut and paste into Openoffice...

Ubuntu rules and Linux is great for general web/computing/development, but for the academic writing that I do it frustrates me because the great tools available for windows just aren't here quite yet.

Good luck. Let me know how you go :)
