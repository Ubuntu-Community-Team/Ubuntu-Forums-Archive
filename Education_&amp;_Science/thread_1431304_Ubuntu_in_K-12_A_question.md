### Post by gunksta on 2010-03-16
I would start by suggesting he look at the Netbook Editions of Ubuntu. They will be "easier" to use, especially on a limited system like a . . . netbook.

I would also suggest they experiment specifically with Kubuntu/Kubuntu Netbook edition. KDE includes a KDE-Edu applications. Some of these tools are definitely geared towards younger users (khangman), while others (Cantor) are geared towards high school students (in fact, I am evaluating this tool for myself to use at work). Lots of options. On modern laptops/desktops adding the KDE-Edu apps to an existing Ubuntu install would be trivial, but the hard-drive requirements of installing the KDE libraries in addition to the GNOME libraries could be problematic on a netbook.

I think the concern that Linux is "harder" is simply not true. I use WinXP on a nearly daily basis for work (virtualized) and there are some things that are easier in Windows (opening Access databases) but there are many things that are easier in Linux (nearly everything else). I assume the average 1st Grader will have limited experience with computers -- after-all, how many computing habits can a first grader really have? I doubt they will find Linux any harder to use than Windows, unless the school installs a distro like Gentoo or Slackware on the systems.

The best thing about Linux is that the school system can create a set of USB keys with default installation images, making it easy to create a large number of identical clone systems. They can then develop a set of scripts to run for each grade to install appropriate software on the systems. Using Ubuntu gives teachers and students access to an absurd number of relevant programs and tools. There are math tools and science tools. The Internet works great, just make sure the USB image includes flash. You can set up the Web-broswer to include links to things like Project Gutenberg and other free-content sites. 

I'm sure there's a way to create these sort of custom installs on Windows -- but I promise it's going to be more complicated than it is on the Linux platform.

I predict the adults will have more trouble than the children do -- which is typical when comparing the adaptability of children to adults with new technology.
