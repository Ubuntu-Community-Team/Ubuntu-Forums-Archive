### Post by BryanLunduke on 2010-05-27
Just wanted to drop a note here that Illumination Software Creator 1.0 for Linux was released earlier this week.

It is a 100% visual software development environment aimed at making creating software accessible to anyone (including students and non programmers).

The idea is that you utilize colored building blocks of functionality to create your full application (no "coding").

The resulting software that is created is 100% pure Python with PyGTK (so it can run on Linux, Windows, Nokia Tablets, etc.).  In general a great way to learn basic programming and logic concepts as well as a great way to learn Python.

(It runs on Windows too... but the Linux version was first.)

Take a look here: [http://radicalbreeze.com/?page_id=102](http://radicalbreeze.com/?page_id=102)
And see a quick tutorial here: [http://radicalbreeze.com/?page_id=168](http://radicalbreeze.com/?page_id=168)
The support forum (including the beta releases) are here: [http://radicalbreeze.com/forum/](http://radicalbreeze.com/forum/)
