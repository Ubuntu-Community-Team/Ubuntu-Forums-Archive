### Post by samden on 2009-05-16
I've had some problems with moderncv too. In the end I downloaded the latest version I could find and put it directly in the directory where I was writing my cv.

Note that the older version wanted you to define a \moderncvstyle, the new one wants \moderncvtheme. There are a few differences between versions. Check the template files you are using were actually obtained from the same place as the moderncv style files you downloaded, to ensure they are actually compatible.

The version in Texlive seems rather old, even in the Jaunty repos. I presume this will be updated at some point.
