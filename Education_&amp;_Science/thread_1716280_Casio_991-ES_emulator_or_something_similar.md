### Post by gunksta on 2011-03-30
I dunno. The OP is looking for a calculator. Octave is all that AND a BIG bag of chips. (Note, I don't use Octave, but I do know it's considered something of a Matlab replacement.) I don't think the OP is looking for a Matlab replacement.

Open up Synaptic (System -> Administration -> Synaptic) and search for calculators. You could also use the Ubuntu Software Center. If you'd like to do this via the command line:
```
 apt-cache search calculator | less 
```
works well.

If you're up for some fun - bc is the GNU calculator. It's command-line driven but it is considerably less complex than Octave. But, I don't think it can handle any graphing.

I have used [SpeedCrunch]("http://speedcrunch.org/en_US/index.html"). Development looks a little stalled ATM, but it is almost certainly a good option.

I have never used it, but [Genius]("http://www.jirka.org/genius.html") looks quite interesting. And it looks like it is actively developed.
