### Post by Hagar Delest on 2010-02-13
Works fine for me. I just installed it and it works. But the medical terms are checked only if the language is English (USA). However, it can be changed to work also with English (UK).

I've had issues also first with the same problem (common words being misspelled) and it seems to come from my user profile that contained deprecated references to other dictionaries extensions. Try to [reset your OOo user profile](http://user.services.openoffice.org/en/forum/viewtopic.php?p=58403#p58403) but don't transfer your personal data during the welcome process (if you had 2.x before), configuration files from the former version might corrupt the new profile.

Hey Hagar,

The above suggestions didn't fix my problem, but changing my default language to Canadian English (as you suggested in a different thread) did. Thanks

In case anyone else has this problem, open OOo and select 
Tools-->Languages--->All Text and (assuming you use US English) select Canadian English as your default language. OpenMedSpel should work after that. The unfortunate thing is that Candians do spell some words differently, but hopefully you should be able to catch those differences. For me, having spell check for 40000+ medical terms is more important than having to change "defence" to "defense".
