### Post by LaserJock on 2008-03-05
Since there seemed to be some interested I've created a new team in Launchpad specifically for Octave related packages: 

[https://launchpad.net/~ubuntu-octave](https://launchpad.net/~ubuntu-octave) 

If you would be interested in help with the octave packaging in Ubuntu feel free to join. I've activated the Personal Package Archive (PPA) for the team: 

[https://launchpad.net/~ubuntu-octave/+archive](https://launchpad.net/~ubuntu-octave/+archive)

This means we can upload new packages (source packages) for pretty much any active Ubuntu release (dapper - hardy) and they will get built into .debs and placed in an ubuntu-octave repo for people to use.

Interested people should read over:

[https://wiki.ubuntu.com/PackagingGuide](https://wiki.ubuntu.com/PackagingGuide)
[https://wiki.ubuntu.com/UbuntuDevelopment](https://wiki.ubuntu.com/UbuntuDevelopment)
[https://help.launchpad.net/PPAQuickStart](https://help.launchpad.net/PPAQuickStart)

Any questions?

My guess would be that the first step would be to backport Hardy's octave3.0 packages to Gutsy?

-LaserJock
