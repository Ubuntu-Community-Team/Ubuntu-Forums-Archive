### Post by rybu on 2008-07-05
IMO it's not a good idea to ask for clones to Microsoft software -- many of us don't use, nor have ever heard of "Microsoft Math".  So we wouldn't know what you're looking for without looking up the specs for the software you're referring to. 

Microsoft Math seems to be pretty basic education-level software.  There's plenty of packages that'll do what you're asking but maybe not in the package you're looking for.  If you really want the Microsoft software, might as well just load it up in WINE or some such emulator.  But if you want a calculator, there's things like kcalc.  If you want to deal with linear algebra problems and solving systems of equations, graphing, etc, there's software like Pari.  There's plenty of open-source stats software like R and ESS. 

There's also some more integrated packages like Maple, though it's not open-source, nor do they have a completely 64-bit version of their software... the list goes on.
