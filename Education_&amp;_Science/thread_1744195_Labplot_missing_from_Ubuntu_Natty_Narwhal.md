### Post by Lateralis on 2011-04-30
If you really want to use that package then you can download it and install yourself from sourceforge: [http://labplot.sourceforge.net/download/](http://labplot.sourceforge.net/download/).

As for alternatives, you may want to investigate Octave (and Qt Octave), gnuplot and xmgrace.  One, some or all of these programs may provide some or all of the functionality Labplot was able to do but they are all in the repos.

Edit: Another alternative is to stick with Lucid, as being an LTS you'll have support up until April 2013.
