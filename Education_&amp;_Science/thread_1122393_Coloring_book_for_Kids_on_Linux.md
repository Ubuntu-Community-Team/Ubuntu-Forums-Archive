### Post by hubie on 2009-04-13
It doesn't have a whole bunch of pictures, though you can add your own, but I came across the kavlon coloring book.  I couldn't find a direct link to it, but I was able to get at the source code at a [download site]("http://linux.softpedia.com/get/Multimedia/Graphics/Kavlon-Coloring-Book-18508.shtml").  You'll need to install python-pygame to get it to work.  You run it like:
```
$python CB.py
```
I tried it out and it worked on my system (8.10).
