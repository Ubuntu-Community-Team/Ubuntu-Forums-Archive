Zorin OS 4 Educational VirtualBox virtual machine is released.

Zorin OS is an Ubuntu-based Linux distribution designed
especially for newcomers to Linux. It has a Windows-like
graphical user interface and many programs similar to
those found in Windows. Zorin OS also comes with an
application that lets users run many Windows programs.
The distribution's ultimate goal is to provide a Linux
alternative to Windows and let Windows users enjoy all
the features of Linux without complications.

You can get them from sourceforge.net download area:
[http://sourceforge.net/projects/happypenguin/files/zorin/virtualbox/](http://sourceforge.net/projects/happypenguin/files/zorin/virtualbox/)

Please note that, the Additions installed is for VirtualBox 4.0.
I haven't checked if it works with VirtualBox 3.x.
