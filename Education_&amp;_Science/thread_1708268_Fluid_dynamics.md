### Post by cwarner7_11 on 2011-04-23
There may be some packages a bit easier to use than the OpenFOAM package for fluid dydnamics.  The [CAELinux]("http://www.caelinux.com/CMS/") package, built on Ubuntu 10.04 (and totally free) includes Code_Saturne, Elmer, Gerris, and most importantly, the Salome package which provides geometry, meshing and post processing in a very sophisticated package.  Any of these packages can be installed from links at the CAELinux sight, but, by going with the CAELinux distro, everything is already set up and ready to run from the LiveDVD.  Requires 64 bit processor, however- individual packages are available from their home sites for different operating systems and 32-bit machines...
