### Post by whelanh on 2008-02-16
My recommendation is that you install the "R" program.  R is programming environment excellent for numerous mathmatical/statistical problems.  R can be installed from the Synaptic Package Manger.  Install r-base-core and r-base-dev.  See also [http://r-project.org](http://r-project.org)

I'm not sure what the problem you wish to apply genetic algorithims to, but see [http://rh-mirror.linux.iastate.edu/CRAN/](http://rh-mirror.linux.iastate.edu/CRAN/) for a list of available packages people have contributed to R.  There are a number that incorporate genetic algorithims (e.g., gafit, DEoptim, genalg, rgenoud etc.).
