### Post by samden on 2009-10-07
One difficulty with computers is that they are generally designed to make it easier for people - automatic spell checkers, calculators etc - which can potentially reduce a child's ability to do these things if you are not careful how you set it up.

Make sure you disable the automatic spell checker (red underlines) in OpenOffice if you want him to have good spelling - show him how to check the spelling as a once-off through the menus instead. And change the default dictionary to real UK English rather than this terrible US stuff. :)

I'd agree to check out Edubuntu.

### Post by OlsenCNC on 2009-10-09
Mnemosyne and Anki spaced repetition systems Memory Reminder is a great to use for Memorization of different answers and questions.

It knows which ones you get to know and which ones you need more help with.

You create your own questions and answers.

To Install:

*Go into Add/Remove Applications, switch "Show" to "All Available Applications", Type *Mnemosyne * in the Search Box and Seclect *Mnemosyne *to install *Mnemosyne*. *
