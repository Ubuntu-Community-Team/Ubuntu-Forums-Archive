### Post by radioboy on 2010-09-17
I have had that problem about twice, but a simple restart of Kile worked for me. Maybe you should try also removing any related hidden folder it might have left in your home folder, if you haven't done so.
However, consider this: once I screwed it up by setting a default font that would crash Kile every time I started it. I discovered, after some very frustrating minutes trying to find the hidden settings file to modify it, that Kile also installed a Kate configuration file (even if I didn't have Kate installed!), and the font preferences were stored there. Obviously you should be careful with any changes if you also use Kate.
