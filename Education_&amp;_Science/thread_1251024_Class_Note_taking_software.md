### Post by ugm6hr on 2009-08-29
Has anyone suggested using a wiki?

If you get used to wiki annotation styles, they are perfect for note-taking, since links are automatically created.  This gives it a sensible structure with little effort.

Your options for this approach are:

1.  Install wiki on a server / home computer and access it from internet (requires mobile broadband etc and relies on connection when taking notes).

2.  Use TiddlyWiki - which is designed to run from a file (i.e. no web server required).

The latter is an excellent piece of software, and I would recommend it.  No installation required - just grab the file from their site.

EDIT: Just seen the zim website; it essentially does the same as TiddlyWiki but requires specific software to be installed (and hence cannot be portable on a USB etc).  It appears to have a few useful features.  Never used it, but looks good.

### Post by xdaedalus on 2010-01-28
Zim is definitely my favorite. I'm in medical school and everything the professors give us is in a .ppt, .pdf, or .doc. Usually I have their notes open on half the screen while I'm in class. I have Zim open on the other half. 

If they're referring to a diagram, I can take a quick screenshot and put it into my notes. If they're using an equation, I've gotten fast enough at LaTeX that I can insert that pretty well also. 

Zim is also great for organizing my notes. I have them in a series of attached files to the main "notebook" that I use. I also use the calendar to link my notes, plus it's searchable, which makes the whole thing ideal for handling lots of information.

One caution though: make sure all your files are actually attached to your wiki note, otherwise if you move your notes you'll break all the links. But this is easy to avoid. I love Zim.

Along other lines, Anki is a great flashcard program with image, sound, and LaTeX support. These are the two programs I use most. 

Tomboy is great for writing down random ideas because the notes are automatically linked (without using wiki syntax, which I haven't learned yet). I didn't like using BasKet notes because it's an orphaned project, and if for some reason I want to access my notes from a different program I don't know how I'd get at them--whereas Zim stores notes as .text files.
