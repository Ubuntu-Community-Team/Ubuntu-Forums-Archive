### Post by Chronon on 2010-01-21
> **kleeman said:**
> Whatever you do watch out for the string theory groupies. As far as I know this effort has yet to make a uniquely testable prediction (The LHC may be able to test aspects of theories that posit supersymmetry which include string theories).

Supersymmetry is an entirely exterior postulate to string theory, though.  Demonstrating the existence of symmetric pairs of particles would lend credence to supersymmetry but doesn't necessarily bolster the validity of "superstring" theories. 

The problem with string theories, which I have never seen addressed satisfactorily, is the landscape problem.  The physics predicted by string theories depends on how you fold up the extra (compact) dimensions.  The set of ways to do this is infinite and so if a particular string theory fails to describe the results of experiments you can just make the appeal that you weren't looking at the correct theory.  There will be _some_ string theory (i.e. some point on the landscape) that accounts for the results of the experiment.  In a sense, string theories have a large number of free parameters and I can fit any data set, given enough free parameters.  I am confident that many theories containing an equivalent number of free parameters could be formulated that perform just as well as string theories.  

I could roughly claim that there exists some string theory that will be able to describe the outcomes of all conceivable experiments, but the probability of finding that string theory is infinitesimal.

The holographic principle is just a wild supposition deriving from Hawkings speculations about the quantum effects on black holes. The common problems running through all this is the lack of constraining experimental evidence on theories. There is the real problem for modern physics. Theory without experiment is empty.

Without falsification there is no way to distinguish between multiple candidate theories.  We collect more data and pare away the models that fail to describe what we actually see.  Verification is also important, but it only tells us that a given model is good enough.  We actually gain more when we find that a model fails. The density of states for string theories seems fairly uniform (at least without a lot more information this seems like a useful prior).  So, when we discard one as not useful for describing our universe we don't really gain any useful information about which direction to go next.  Basically you just end up doing a kind of random walk around this landscape hoping that you end up with a theory that corresponds to our experiments. I'm not sure about how it was originally derived, but it can be derived without any regard for quantum gravity.  One of the high-energy folks came in and did a guest talk during the GR class I took.  He did a back-of-the-envelope calculation indicating why information should scale with surface area rather than volume.  His explanation, however, took the perspective that black holes 'have no hair' and didn't really directly involve any quantum gravitational arguments.  

First you assert that information is physical, that it must correspond to some configuration of energy.  Then you may observe that out of all possible configurations of energy within a given volume of space, many of them correspond to compact structures with horizons.  If you count all configurations leading to the same amount of energy being hidden by a horizon as the same information state, you will find that the number of accessible information states scales like the characteristic length squared instead of like a volume.  This actually ignores any quantum gravity arguments because it makes a "no-hair" argument about the internal states of black holes (i.e. that we can fully describe a black hole in terms of its mass, its angular momentum and its charge).

So, while quantum gravity/string theory arguments may be used to derive a holographic principle, the existence of this derivation shows that observation of a holographic principle does not equate to a confirmation of those string theory arguments (i.e. the existence of an inverse requires a one-to-one correspondence.)

I think that quantum gravity is certainly the next, most obvious hurdle suggested by failures in our contemporary theories.  I am just troubled by string theories' failure to submit to the normal program of offering falsifiable predictions that will allow us to move on.  It could represent a quagmire that will trap us without offering significant advances in our understanding for a long time.  If someone can really give a good description of the landscape that would be one thing, but at this point we don't have any idea about its topology or even if what a useful metric might be.  How do we navigate this landscape?  If we can't navigate it then how long do we wander around lost in it?

Anyway, this is not my field and this is not a problem I have to work with directly.  It's only of philosophical interest to me.  String theory will probably not revolutionize other fields of physics and people in other fields still have plenty of interesting problems to tackle.
