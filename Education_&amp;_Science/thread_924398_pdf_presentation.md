### Post by DrOlaf on 2008-09-21
If you use latex-beamer, you can prepare pdfs that work this way, and then you should be able to use keyjnote, or its current incarnation, [accentuate]("http://www.cs.rit.edu/~jrm8005/accentuate.html"), to project them. 

[http://latex-beamer.sourceforge.net/]("http://latex-beamer.sourceforge.net/")

Latex-beamer is in the repos, but you need to know LaTeX to be able to use it. I haven't tried to use accentuate to do this yet myself, though.
