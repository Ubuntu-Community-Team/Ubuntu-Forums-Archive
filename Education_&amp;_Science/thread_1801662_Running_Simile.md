### Post by earlycj5 on 2011-07-11
Got it!

Simile's tech support answered my query rather quickly.

> To start the program, execute Simile/System/bin/simile -- or if you have downloaded the 64-bit version, Simile/System64/bin/simile

Apparently the README file was out of date.
