### Post by LaserJock on 2006-11-08
Just make sure you have the Universe repository enabled (check out [https://help.ubuntu.com/community/Repositories/Ubuntu](https://help.ubuntu.com/community/Repositories/Ubuntu) if you don't know how) and install grace. xmgrace is now a part of grace and you can start it either  way.
