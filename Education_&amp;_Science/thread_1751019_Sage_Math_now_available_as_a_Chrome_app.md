### Post by PC_load_letter on 2011-05-06
In case you don't know this already, Sage ([www.sagemath.org](www.sagemath.org)), the open source equivalent of software like Maple, Mathematica, can now be run from within Chrome. 

So, start your Chrome browser and head to: 
[https://chrome.google.com/webstore/detail/dckejpjjnjaagdahkdgighlejbckajak](https://chrome.google.com/webstore/detail/dckejpjjnjaagdahkdgighlejbckajak)

You can then install Sage and run it from within Chrome. I can not tell if there are any limitations, but it doesn't mean there aren't any. It might be slower than the stand-alone app but I use it as a teaching tool by publishing my worksheets and share it with my students. Students can view the published worksheets in any Java supporting browser, but can modify it only from Sage (Chrome plugin or stand-alone).

Have fun.
