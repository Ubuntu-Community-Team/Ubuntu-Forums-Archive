### Post by Favux on 2009-01-21
Hi akimatsu123,

I like Xournal and I use CellWriter for "handwriting" recognition.  There's a Java program similar to Xournal, I think its called Gournal that some like.  Also there is a KDE program called Basket that is even more elaborate and a lot of folks like.

Edit:  I don't want to mislead you.  There isn't, as far as I know, a linux program that has all the features of One-Note.  For example none of them, again as far as I know, have handwriting recognition.
