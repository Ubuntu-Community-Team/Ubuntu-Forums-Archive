### Post by ExileAmongYou on 2011-03-08
I'm running SPSS version 18, so this is going to be a bit confusing, because version 18 came out during the brief period when they were trying to change the name of SPSS to PASW. 
In my case, the command I need to run is "paswstat". If they've changed back to spss, you're likely looking for something like "spssstat". Open up a terminal, type spss, then double tap TAB to check what commands there are that start with that.
