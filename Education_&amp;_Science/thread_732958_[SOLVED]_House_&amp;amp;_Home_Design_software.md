I would give Sweet Home 3D a try. Download it from sourceforge [**here**]("http://prdownloads.sourceforge.net/sweethome3d/SweetHome3D-1.2.1-linux-x86.tgz"), extract it where you want, then run it. If you want something more advanced, try qcad.
