### Post by 3rdalbum on 2009-04-19
Okay, well, Microsoft Publisher is not a web development package; it's a desktop publishing program. An especially entry-level one too. Scribus is a more-than-capable replacement for Publisher in terms of desktop publishing and typesetting.

Openoffice.org can generate HTML from your documents, but like Microsoft Publisher it will do a poor job of the HTML.

You should give Kompozer a try - that's a dedicated webpage development program; or if that doesn't work use Seamonkey (it's a web browser with a composer as well).
