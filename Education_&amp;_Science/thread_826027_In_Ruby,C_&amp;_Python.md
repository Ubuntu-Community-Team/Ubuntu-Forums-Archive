### Post by iQuizzle on 2008-12-16
python is a high-level language, so you don't have to compile anything. it will do it for you on-the-fly. i believe ruby is the same way. try this:

open a text editor and type in:

```
print 'hello world'
```

save the file as hello.py and then in your terminal type:

```
python hello.py
```

it will print "hello world" to the terminal.
