### Post by argo82 on 2009-06-15
Hi!
For economics, I often need to draw graphs such as [http://upload.wikimedia.org/wikipedia/commons/b/b9/Islm.svg](http://upload.wikimedia.org/wikipedia/commons/b/b9/Islm.svg) (no, this one isn't mine :) ). So the focus on this kind of drawing is not on the exact representation of a function (more often than not, I just need a straight line...). Basically I need to
- draw a coordinate system
- draw simple curves by hand
- add some labels
In the past, I used the OpenOffice drawing tool for that purpose, it can draw lines and simple curves and it also includes a formula editor that I often need for quickly inserting Greek characters and indices of variables. When converting it to ps or pdf, the output is acceptable, but doesn't really impress me.
Is there some other kind of software that I can use for that which produces more professional looking output? I know that probably it's somehow possible with gnuplot or some latex package, but I think a gui solution is much more convenient here. And while normal image editing software such as gimp produce better looking output, it doesn't have that neat formula editor and feels a bit cumbersome to use for drawing simple graphs.

Thanks for any suggestions!
argo

### Post by hubie on 2009-06-15
One of the reasons, I think, that the graph you show as an example looks good, is that it is SVG, i.e., it is based on vector graphics.  You might want to look at one of the vector graphics capable programs such as [Inkscape]("http://www.inkscape.org/index.php?lang=en") or [Dia]("http://projects.gnome.org/dia/").

I would probably go with something like [R]("http://addictedtor.free.fr/graphiques/") or [gnuplot]("http://gnuplot.sourceforge.net/demo/") to make the graphs.  If it is something you do often, it would be easy to set those up so you had a generic plot template and go from there.

### Post by tommers on 2009-06-15
If these plots will be used in a latex document, I highly recommend TikZ, PGF and/or PGFPlots.

For Examples:
[http://www.texample.net/tikz/examples/]("http://www.texample.net/tikz/examples/")

For Downloads:
[http://sourceforge.net/projects/pgf/]("http://sourceforge.net/projects/pgf/")
[http://sourceforge.net/projects/pgfplots]("http://sourceforge.net/projects/pgfplots")

If these aren't for a latex doc, you could still produce high quality pdf images for inclusion in other documents.
