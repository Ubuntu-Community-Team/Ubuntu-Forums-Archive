### Post by ekarni on 2006-05-30
Here's a few ways to make dashed lines:

1)  By layer (probably best, lets you turn everything on the layer on and off together)
1A) Assign it when creating a new layer.  Layer -> Add layer, then change the linetype pulldown in the dialog box from "continuous" to something else.

1B) Move existing lines to an existing dashed layer.  Click the edit toolbar button, then select the lines, hit the bottom "continue" arrow button, and re-assign the layer (top pulldown in the dialog box).

2)  Change existing lines individually.  Do it as in 1B, but instead of changing layer, just change the linetype.

The same process also applies to changing line color and lineweight as well.
