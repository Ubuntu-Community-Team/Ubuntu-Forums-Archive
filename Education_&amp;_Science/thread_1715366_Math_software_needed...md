### Post by bryncoles on 2011-03-27
Hi there.  I think it is obligatory in these threads to mention [R]("http://cran.r-project.org/").  Ok, it's also rather complicated and fussy and has a steep learning curve, but it's worth it I promise!

Install R-base from the repos, install R Commander too, then you'll have an acceptable GUI for it.  Fire up a terminal and type 

```
R
``` To start the program, and then ```
library(Rcmdr)
``` To activate the GUI.  

You can also get some good help on using R from: [this wikibook]("http://en.wikibooks.org/wiki/R_Programming") and [this wikibook]("http://wiki.math.yorku.ca/index.php/R:_Getting_started").  

It's hard to get started with, but worth it as it pays off in the end!

Here's more you can check out if they don't meet your need:

> SAGE

SAGE is the new star on the open-source math sky. Essentially it is a CAS written in Python, but it provides interfaces to many math programs available. This includes commercial programs like Mathematica and Maple as well as free programs like Maxima, Octave or Singular.

From the homepage:

Each interface requires that the corresponding software is installed on your computer. SAGE includes GAP, PARI, Singular, and Maxima, but does not include Octave (very easy to install), MAGMA (non-free), Maple (non-free), or Mathematica (non-free).
