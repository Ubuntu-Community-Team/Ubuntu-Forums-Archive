### Post by cariboo on 2011-06-04
When I was using legacy gnome, I removed the bottom panel, and added the windows list applet to the top panel. Using the windows list applet, it will show a list of open windows when you click the icon, otherwise they don't show anywhere. To enable it, just right click on the panel, and select the applet from the list.
