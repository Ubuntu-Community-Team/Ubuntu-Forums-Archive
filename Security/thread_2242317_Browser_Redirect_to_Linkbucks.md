### Post by fugu2 on 2014-09-12
FYI, is you ever want to check a goo.gl redirect without using a browser:
Example for [http://goo.gl/626U3](http://goo.gl/626U3)
```
$ echo -ne "GET /626U3 HTTP/1.1\r\nHost: goo.gl\r\n\r\n" | nc goo.gl 80
```
this will just contact goo.gl without the automatic redirect, it can give you a bit more control to see whats actually going on. So much happens in a browser that we never see. Anything we can do to pull back that curtain can help.
