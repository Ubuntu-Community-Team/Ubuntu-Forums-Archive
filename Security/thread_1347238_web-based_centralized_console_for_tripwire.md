Hi,

If you use tripwire, you may have to review and accept changes on a daily basis. Logging into multiple servers to accept the changes is troublesome. Enter CentralWire: a web-based centralized console to review and accept changes detected by tripwire. You can download it from [http://centralwire.sourceforge.net](http://centralwire.sourceforge.net) and it's free software.

I wrote this software to make my own job easier. Considering that many people face the same problem, I decided to open source it.

Note that Tripwire is a registered trademark of Tripwire, Inc. and that the CentralWire project is NOT in association with the Tripwire, Inc.
