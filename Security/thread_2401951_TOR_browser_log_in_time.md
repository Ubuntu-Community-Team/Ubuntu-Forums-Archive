### Post by TheFu on 2018-09-24
TOR is under constant attack.  Since all software has bugs, I would use it with an expectation of the networking being continuously attacked and that the longer I'm using the same session, the more likely that unwanted fingerprinting data would become known.

If you are using TOR all the time as general privacy against a snooping ISP, that is one thing. Your encrypted packets help everyone.

If you are using TOR inside a country with a repressive govt and lives can be taken based on the activities uncovered, that is completely different.

If you've ever used any accounts without TOR and use those accounts inside TOR, you've already broken the vail of privacy.
If you post online using TOR and have ever written anything without using it, then your writing style can be used to figure out who you are IRL.

OPSec is hard. Very hard.
