It really depends on how paranoid you are, or rather whether you need the additional security from PHYSICAL access to your data. Encrypting the disk prevents folks from using a live cd to access your data. It may be overkill for the average desktop user in a home environment. I definitely suggest doing it if you are a laptop user or use your desktop in a business or open access environment.
