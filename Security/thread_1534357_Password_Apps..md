### Post by sgosnell on 2010-07-20
Tried it long ago, didn't like it.  I use Pyring, which is a Linux port of Keyring for Palm.  I had hundreds of passwords in there on my Palm, and I required a password safe that would use them.  None of the password safes I tried would import much of anything, or export much.  Import and export are essential for me, because I don't want to have to enter hundreds of passwords, usernames, and other information by hand again.  

From what I've read, Password Gorilla works ok, and I've seen many good reviews of it.  It just didn't work for me, because it didn't import anything.
