### Post by yota on 2009-05-19
Hi Bat, welcome to Ubuntu and the community.

you can get documentation about the OS and many howtos oriented to performig specific tasks at:
[https://help.ubuntu.com/9.04/index.html]("https://help.ubuntu.com/9.04/index.html")
and
[https://help.ubuntu.com/community]("https://help.ubuntu.com/community")

In my opinion probably the best advice I can give you is not to rush: changing OS from windows to linux is like changing from cars to motorbikes. It takes time to be proficient riding the latter and directly jumping on seat to do a long ride without experience probably would only hurt.
This doesn't mean I'm trying to descourage you, but probably it should be better to play a bit with linux and learn how ubuntu works before actually using it for "production" everyday purposes.
Maybe a read at [http://linux.oneandoneis2.org/LNW.htm]("http://linux.oneandoneis2.org/LNW.htm") can help to summarize potential pifalls in expectations that a tipical ex-windows user usually has approaching linux.

Also please keep in mind that Ubuntu runs its own software (like OSX software doesn't run on windows and windows software doesn't run on macs) so be prepared that you can get stuck with some piece of software missing (expecially frequent for games).

This said I can assure that if you take your time and you are willing to learn you'll be satisfied and will not be looking back.

Trying to answer to your questions is can see that EVE has a native linux client (also for ubuntu) available at:
[http://www.eveonline.com/download/linux.asp]("http://www.eveonline.com/download/linux.asp")
Fighter ace supports linux [http://linux.ketsujin.com/]("http://linux.ketsujin.com/") but there's not a package for ubuntu

Security wise we can say that ubuntu is already secured out-of-the-box so you don't need to do anything. If you wanto to go more in depth take a look at [http://www.psychocats.net/ubuntu/security]("http://www.psychocats.net/ubuntu/security")

Hope this helps!

### Post by rookcifer on 2009-05-23
You are not going to be able to get MagicJack working as the Magic Jack people don't seem to want to support Linux (even though they keep saying they will, it still hasn't happened and probably wont).  You could try running Vista in Virtualbox and use magic jack that way.

Other than that, everything else you do should be doable in Ubuntu.  You can probably even run some of your "Windows only" apps in WINE.

As for security, you don't need to do much of anything other than always be sure you install your software through the official channels (Ubuntu package manager).  It also wouldn't hurt to set up a firewall (with a router) to protect your entire home network.  If you don't have a router, a very good and inexpensive router is the Linksys WRT54GL.  You can pick one up brand new for about $50.  It is a Linux friendly router and can be flashed with third party firmware.  I recommend Tomato.

Ubuntu is not Windows and does not suffer from Windows' long time single user paradigm (which is awful for security).  Thus, there is no need to worry about AV software or any of that other garbage that does nothing but make Symantec and McAfee billions of dollars while actually doing very little to protect you from viruses.  The AV people depend on M$ being insecure!  Without an insecure Windows, they are out of business.

There are a few things you could do to harden your system even more, but unless you run an Internet facing server it isn't all that necessary.  You can look into installing and running Bastille if you are interested in an automated way to harden (but Bastille does not seem to work with 9.04 Jaunty).  And you can look at this forum's sticky on AppArmor which is a MAC.  You can also run "netstat -autpvnl" in the terminal to see what services are listening.  If you don't need these services you can disable them.  However, if you are behind a good SPI firewall that blocks all incoming connections you shouldn't even need to do this.

Good luck.
