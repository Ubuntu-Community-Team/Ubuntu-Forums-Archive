### Post by ajgreeny on 2017-03-19
If you are running a standard desktop without any servers open to the outside world you probably do not even need to do anything related to  setup a firewall as it is very likely that if you're using a router that already has firewall functions included.

You can get some idea of the security of your machine by going to [https://www.grc.com/x/ne.dll?bh0bkyd2](https://www.grc.com/x/ne.dll?bh0bkyd2) and seeing how well it manages to hide itself there from the web.

If it's any comfort to you, I have not used antivirus, nor bothered to set up a separate firewall since I began using Ubuntu over 11 years ago, having always been behind the hardware firewall of my router.

See the various links on security subjects at [https://help.ubuntu.com/community/Security](https://help.ubuntu.com/community/Security) for lots more details and help.
