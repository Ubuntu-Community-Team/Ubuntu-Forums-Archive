### Post by rg4w on 2011-05-21
I'm not a security specialist, and just about anything is possible, but I use ConnectBot and I've seen no evidence that my keys are being sent anywhere than to the machine I'm connecting to.

Because I'm not a security specialist, I tend to be a little paranoid like yourself.  I prefer to think of it as prudence.  So before I install anything I'll visit the developer's site and search around to see what others are saying.

ConnectBot seems to have a good reputation, FWIW, and folks generally speak highly of it.  If it's a Trojan Horse of any sort it's quite the clever ruse, as it's duped a great many people into believing it's just a really good SSH client.

If anyone has other thoughts in ConnectBot I'd be grateful to hear them.  But for now I'm using it almost daily and enjoying it.

EDIT: It seems the source is available as well:
[http://code.google.com/p/connectbot/](http://code.google.com/p/connectbot/)
I doubt anyone posting a ruse would create fake source.  Possibly, but unlikely.
