### Post by slickymaster on 2013-01-24
IMHO that's not a virus but rather a type of malware called ransomware, which basically  restricts access to the computer system that it infects, and demands a ransom paid to the creator of the malware in order for the restriction to be removed. Some forms of ransomware encrypt files on the system's hard drive, while some may simply lock the system and display messages intended to coax the user into paying.
