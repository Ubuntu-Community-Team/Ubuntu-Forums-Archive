### Post by CandidMan on 2011-02-13
Hi,
I'd recommend reading this thread first: [http://ubuntuforums.org/showthread.php?t=510812](http://ubuntuforums.org/showthread.php?t=510812)

I'm no security expert, but I think a default Ubuntu install is 'safe'.

What it boils down to is 

1) Open ports/running services

2)Where you get executable content, javascript, ELF, Wine etc.

3)How your network aware programs give away your privacy

1 and 2 are pretty much covered, but 3 takes careful consideration. I'd recommend noscript for firefox and append this [http://someonewhocares.org/hosts/](http://someonewhocares.org/hosts/) to your /etc/hosts file
