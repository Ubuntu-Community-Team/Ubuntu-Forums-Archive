### Post by Cheesemill on 2013-07-16
The fact that TrueCrypt is open-source makes it more secure. This is because developers and cryptography experts around the world can look at the code to make sure that it is properly written and doesn't contain any weaknesses. Compare this to closed-source code where you have to trust that there are no back doors programmed in and that the developer knows what they are doing (writing good secure cryptographic code is incredibly difficult).

The ciphers used in modern cryptography are mostly all open standards that are very well documented. Knowing how an encryption scheme works doesn't make it any easier to guess the password.
