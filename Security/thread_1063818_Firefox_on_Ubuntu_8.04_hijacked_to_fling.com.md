### Post by brian_p on 2009-02-08
> **xp_newbie said:**
> One of the reasons I switched to Ubuntu was to avoid all the malware that plague the various Windows platforms.

A good decision.

> I was shocked when, while being on a certain web page (using latest Firefox on Ubuntu 8.04), the page automatically redirects to fling.com. I never visited fling.com before then, how could this happen?

That's the way redirects, in conjunction with JavaScript, work.

> Are you familiar with this problem?

What's causing it? Is it a virus that managed to infect Ubuntu? Is is it something else?

A virus!! Don't panic because your mouse pointer twitches.

> And most importantly - how do I get rid of this problem?

1. Don't go to that 'certain web page' again.
2. Switch off JavaScript
3. Install a redirect blocker plugin. I believe there is one for firefox.
