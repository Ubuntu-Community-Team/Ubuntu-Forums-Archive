### Post by psusi on 2011-02-03
None.  Antivirus is for people who:

1)  Don't know much about computers
2)  Run a broken OS

People in group 1 foolishly run programs from untrusted sources.  People in group 2 have an OS that foolishly allows untrusted programs free reign to trounce on the OS, and will even automatically run untrusted programs without your knowledge or consent.  There is a large degree of overlap between the two.  If you are running Ubuntu, then you at least have eliminated number 2, which mostly protects you from number 1.

### Post by psusi on 2011-02-04
> **awebuser said:**
> I know about computers and I am very security concious just considering using ubuntu is it really ok to run without an antivirus software I cannot believe this especially as I know how sneaky hackers can be.

Viruses have nothing to do with the incorrect movie definition of hackers.  Viruses are a problem on Windows because:

1) Windows ( until recent versions ) allows the user and all programs that they run to completely trounce the system.

2) Automatically runs programs from untrusted sources without users knowledge or consent, such as on web pages or in email attachments.

3) Windows users tend to download programs from untrusted sources and run them.

Take away these 3 things and you remove the attack vectors that a virus can use to infect your system.
