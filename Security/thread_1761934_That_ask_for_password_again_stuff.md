### Post by Gastra on 2011-05-18
How do I stop Ubuntu (11.04) to ask for my password every time I leave the computer for what feels like no more than three and a half seconds? It adds no security; I am the only one ever to be in physical range of the computer.

---

### Post by sisco311 on 2011-05-18
Hi and welcome to the forums!

Run:
```
gnome-screensaver-preferences
```
and make sure that the "Lock screen when screensaver is activ" box is not selected.

---

### Post by collisionystm on 2011-05-18

> How do I stop Ubuntu (11.04) to ask for my password every time I leave the computer for what feels like no more than three and a half seconds? It adds no security; I am the only one ever to be in physical range of the computer.

Search for Screen Saver

uncheck the box that says lock computer
