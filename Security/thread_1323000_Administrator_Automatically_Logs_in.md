### Post by Arsemyth on 2009-11-11
I have 4 users set up on my PC,  1 administrator and 3 users.

When the PC is switched on and Ubuntu boots up, I am not given the option to choose a user profile, but instead the PC automatically loads up the Admin user profile (me) without asking for my password.

If I subsequently log off I then get the window which allows me to choose a user profile and which asks for passwords etc.

The problem is of course that I do not want people signing in as me.

What do I need to do to resolve this?  i am sure there is a simple solution 

Thanks in advance

Roger

---

### Post by cariboo on 2009-11-11
You probably have automatic login enabled, go to System-->Administration-->Login Screen and disable automatic login.
