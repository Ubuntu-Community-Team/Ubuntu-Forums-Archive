### Post by lemming465 on 2008-05-02
> Is there a way to lock folders with a password? 

No, but you can use group memberships and permission modes or even access control lists to restrict which users (other than root) can get into a directory or a file.  If you really need password protection, you will have to resort to encryption, yes.
