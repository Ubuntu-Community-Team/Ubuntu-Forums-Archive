### Post by CharlesA on 2010-04-13
Security on the internet isn't specific to one OS, especially since we have smart phones and whatnot that can browse the net.

Common sense is the most important thing to have. I would think that using strong passwords would be included in that. :)

If you are connected to an unknown wifi hotspot, use an SSH tunnel to secure your web traffic, so that no one on that network can read your packets.
