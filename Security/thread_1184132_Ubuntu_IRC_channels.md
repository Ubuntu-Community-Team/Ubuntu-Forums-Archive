### Post by andrew.46 on 2009-06-11
Hi Goddard,

> **Goddard said:**
> appreciate that but that page doesn't really explain enough.  Atleast in that section it says talk with a network staffer.  I don't even know what that means.

Network staffers can be found by joining #freenode where you simply ask for an unaffiliated cloak. But they will expect you to have a registered nick and an associated email address setup before they will do this for you. I am not sure what irc program you are using but to do this simply connect to Freenode and then type in:

```
 /msg nickserv REGISTER **[COLOR="Red"]<password> <email>[/COLOR]** 
```

putting your own password and email address in place and then:

```
/msg NickServ SET HIDEMAIL ON 
```

You should receive an email from Freenode with some confirmation instructions.

> Can I get one from ubuntu?

Yes you can but this is only issued if you become and Ubuntu Member. Details can be found here:

Membership
[https://wiki.ubuntu.com/Membership](https://wiki.ubuntu.com/Membership)

Most Ubuntu irc channels should be able to help with registering and setting up a cloak if you get stuck.

All the best,

Andrew
