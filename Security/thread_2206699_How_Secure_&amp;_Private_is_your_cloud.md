### Post by TheFu on 2014-02-20
Any cloud computing is non-secure by definition. If there isn't physical control over the network, storage and hardware, a company is just kidding itself into believing any security is involved.

"[Cloud computing" is "Careless Computing]("http://www.networkworld.com/community/blog/cloud-computing-careless-computing-and-chrome")"
- RMS
Of course, [cloud-centric companies]("http://www.eweek.com/c/a/Cloud-Computing/Google-Says-Chrome-OS-Cloud-Arent-Careless-Computing-127833/") will disagree.

There is a place for cloud computing - for things you want to share with the entire world.  I don't know any business that has 100% public data.  Just sayin'.
