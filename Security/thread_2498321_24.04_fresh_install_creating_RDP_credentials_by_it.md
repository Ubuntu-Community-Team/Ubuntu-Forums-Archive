### Post by vbgf3 on 2024-06-15
Hi,

You can uninstall several things:

sudo apt purge libfreerdp*
sudo apt purge libvncserver1
sudo apt purge remmina

RDP is what it says it is. VNC is a remote access protocol. Remina is the bult-in Ubuntu remote desktop app.

Hope this helps.
