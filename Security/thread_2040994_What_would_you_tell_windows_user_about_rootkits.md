Rootkits exist for Linux, but they do not seem to be popular and I too have never seen one in the wild.

I suggest all users new to Ubuntu/Linux read the security sticky as it covers the most common cracks.

The most common cracks we see on the Ubuntu Forums, by far, are :

VNC
SSH

Both with weak passwords.

Crackers now use much more sophisticated tools such as metasploit and you really need to look at Backtrack Linux to get a feel for modern techniques. Such techniques, of course, are beyond what we support on these forums. You will need to read the Backtrack information and/or use the Backtrack forums.
