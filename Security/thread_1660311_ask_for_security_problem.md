### Post by restorator on 2011-01-05
If they can have physical access to the machine you are pretty much never going to be able to 100% secure it with anything, period. In that case the best thing you may be able to do is boot via a live cd or usb and use an encrypted drive or partition for storing your files. But if they can intercept your transmissions or keystrokes your security will still be compromised. Physical access means can pretty much own you. If you can lock the machine in a case with the cables unplugged when you are not using it it may help, and make sure any wifi is encrypted with AT LEAST WPA. The weakest spot would then be the connection out. I don't know if you can do much about that in your situation.
