### Post by fugu2 on 2014-10-17
So ufw is a front-end for iptables, which is the real firewall. You should check yo see if there firewall is really there and ufw is mis-reporting the status, or if the firewall is really down
```
  $ sudo iptables-save 
```  if it yields something like
```
 # Generated by iptables-save v1.3.6 on Fri Oct 17 00:00:00 2014
*filter
:INPUT ACCEPT [0:0]
:FORWARD ACCEPT [0:0]
:OUTPUT ACCEPT [0:0]
COMMIT
# Completed on Fri Oct 17 00:00:00 2014

```  then your firewall in down. if it lists a bunch of other stuff, then your firewall is probably up and ufw is not indicating a proper status
