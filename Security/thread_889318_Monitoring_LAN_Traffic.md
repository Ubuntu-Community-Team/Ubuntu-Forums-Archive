### Post by y@w on 2008-08-14
[Webmin]("http://webmin.com") actually has a built-in bandwidth monitoring tool. If you're trying to do true monitoring, I'd definitely go with either [Hyperic]("http://www.hyperic.com/") or [Nagios]("http://www.nagios.org/"). There's also a project called [Groundwork Open Source]("http://www.groundworkopensource.com/community/community-edition.html") that combines a lot of these monitoring applications.
