### Post by Hobgoblin on 2009-05-10
You shouldn't need to install anything, seahorse is already installed it is known as "Passwords and Encryption Keys", under the accessories menu.

Open that, select Edit > Preferences

Highlight "login" on the Password Keyrings tab.

Press Change Unlock Password.

### Post by withjigs on 2009-05-10
i finally figured out how to change the password..

Applications -> Accessories -> Password and encryption keys.

in "Password and encryption keys" select "Password" Tab.

in the List below... Right click on "Passwords: Login" and select "change password"...
 
There you go!!!

Thanks
