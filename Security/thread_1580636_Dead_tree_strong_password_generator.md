### Post by euler_fan on 2010-09-25
That someone wrote a website to do this with pretty colors and labels is rather cool because it takes the work out of it. 

I've been rolling my own using [http://www.random.org/strings/]("http://www.random.org/strings/") and set the number of strings as the number of rows and length as number of columns. Then I generate ten and use a random number generator locally to decide which one to use.

Then I keep them in KeePassX so if I forget how I generated them or loose the card I still have my passwords.
