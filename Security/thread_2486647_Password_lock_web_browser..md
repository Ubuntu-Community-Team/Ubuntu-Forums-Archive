### Post by makitso on 2023-05-08
Perhaps this would work.  Install the bitwarden plugin in your browser.  When you're using your system, set the vault timeout to "never."  Before your  friends come over, change the vault timeout to "on app restart" and kill your browser.  
Bitwarden takes a little time to get use to, remember to set up autofill.
