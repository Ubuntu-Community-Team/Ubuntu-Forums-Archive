### Post by JK3mp on 2010-09-14
You don't NEED one. But you can configure iptables if you require services open. I believe a gui for it can be installed through the software center (search for "firestarter"). Also just go there and search for "Antivirus" "firewall" etc. Not really needed as with windows. ClamAV(will catch windows binary viruses) is a good one. Like i said just search software center if you feel devoted to needing one.
