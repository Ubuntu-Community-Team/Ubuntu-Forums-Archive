### Post by cariboo on 2011-03-03
It depends on how you set it up during the installation process. If you use the gui installer (Live CD) you can only encrypt your home partition during the installation process. If you use the Alternate install iso (text based installer) you can encrypt the whole disk.

In most cases Ubuntu only does what it is told to do, the rest is up to you. :)
