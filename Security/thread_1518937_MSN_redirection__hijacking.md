### Post by cdenley on 2010-06-29
As I already said, they are probably using your IP to determine your location. They would be using a database such as [geoip]("http://www.maxmind.com/app/ip-location"). It is possible, as I think you are suggesting, that someone is performing a MITM attack, routing your connection with MSN through Thailand for some reason. It seems much more likely that they are failing to identify your location from IP correctly.

There would be no definitive way that I know of to verify that a MITM attack is not being performed. The internet is a cloud, and you can't trust that your data can cross it safely, which is why any sensitive data must be encrypted.

Well, I've tried using the tor network before. When using tor, as far as the web server can tell, your traffic originates from the tor exit node, which would be a different part of the world every time you use it. Google will redirect you to their country-specific search page every time you browse to google.com, which would match the country of the tor exit node while using tor.
