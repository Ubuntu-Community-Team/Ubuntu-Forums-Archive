### Post by Gamma746 on 2008-11-14
You probably don't need to worry about it.

Directories in /proc change all the time, it's normal.  Try running ```
ls /proc
```  Then run it again, you'll get different results.

Furthermore, proc is a pseudo-filesystem.  Try running ```
sudo touch /proc/foo
```  It won't work because you can't create files in /proc.
