### Post by kostkon on 2011-02-01
A more user friendly solution would be gnome Nanny. Search for "nanny" in the software centre. From the package's description:
> An easy way to control what your kids are doing in the computer. You can limit how much time a day each one of them is browsing the web, chatting or doing email. You can decide at which times of the day the can do this things as well. Nanny filters what web pages are seen by each user, so you can block all undesirable webs and have your kids enjoy the internet with ease of mind, no more worries! 
Its website is [here]("http://projects.gnome.org/nanny/").

EDIT: you're using 10.4, thus to install it you'll have to use its PPA. For instructions and a review of the app, check [here]("http://www.webupd8.org/2010/03/gnome-nanny-parental-control-takes-care.html").
