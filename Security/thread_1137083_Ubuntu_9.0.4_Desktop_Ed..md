### Post by Tanker Bob on 2009-04-25
Ubuntu is far more secure than Windows, but I'm not sure what you mean by securing the OS. If you mean anti-malware software, no. Phishing attacks and spam are OS-independent, but if the spam carries a Windows malware payload, it will not run on Linux. Phishing uses social engineering, so your best defense is between the ears.

Ubuntu comes with a firewall called iptables, and there are good graphics front ends available for it in the repositories like Firestarter. The best home firewall solution is a hardware one built into your router.

I hope that this helps.
