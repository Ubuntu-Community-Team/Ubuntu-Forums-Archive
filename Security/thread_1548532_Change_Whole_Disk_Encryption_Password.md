### Post by bodhi.zazen on 2010-08-08
LUKS allows up to 8 passwords. To change your password , you first add a new password, then make sure it works, then finally delete the old password.

See :

[http://www.saout.de/tikiwiki/tiki-index.php?page=EncryptedDeviceUsingLUKS](http://www.saout.de/tikiwiki/tiki-index.php?page=EncryptedDeviceUsingLUKS)

See step 6 near the bottom of the page.
