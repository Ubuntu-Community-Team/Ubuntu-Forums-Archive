### Post by Dangertux on 2011-11-15
Those 3 warnings are typical false positives thrown up when rkhunter is run under a modern distro like Ubuntu. 

The first warning consists of shared memory for pulse audio and mono.

The others are warning of hidden directories in /etc which is normal for .java, .udev and .initramfs.

Keep in mind rkhunter was designed for a more traditional Linux/Unix directory structure, Ubuntu deviates from this significantly. If you would like to see what a Ubuntu system looks like with an actual rootkit detected by rkhunter I put together a demo here : [http://dangertux.wordpress.com/2011/11/08/umm-yeah-so-i-think-i-got-owned/](http://dangertux.wordpress.com/2011/11/08/umm-yeah-so-i-think-i-got-owned/)

It's nothing to worry about in this case
