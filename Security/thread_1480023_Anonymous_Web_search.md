### Post by OpSecShellshock on 2010-05-12
Oh, no, that was my point. Allowing some types of scripts on some sites will undermine the anonymity achieved by using the ixquick proxy (since they're executed on the client side and might be designed to gather identifying info). My suggestion would be to not allow them to run. Doesn't make much sense to do anonymous searching if you're just going to send a bunch of data to the target server anyway once you get there. I deny scripts by default, but a lot of users don't because it's tedious.
