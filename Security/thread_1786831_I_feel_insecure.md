The only way to really secure the machine in this environment is to do full-disk encryption with a strong password and put the unencrypted boot onto a USB thumb drive. Then even if someone has access to your machine, you are still pretty well protected. Maybe not totally safe, but the attacker would have to be pretty sophisticated. This is what I would recommend.
