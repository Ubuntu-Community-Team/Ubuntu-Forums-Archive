### Post by Dennis N on 2013-08-16
FF has to request the master password once during each user session for security to allow use of the saved passwords. Otherwise, someone else who is able to access your computer and user acct. also can log onto any site you have a password saved for.

You can tell if you have one by going to Preferences > Security and see if the "saved passwords" button asks for the master. Also of course the little box there should be checked "use a master password".
