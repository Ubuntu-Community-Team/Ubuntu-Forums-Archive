In firefox:

or type "about:config" instead of a URL

> Also, would there be any way of making the Linux or the browser send copies of posted data and web emails to another IP address?
If the browser were replaced with a modified one or configured to use a proxy server. For the latter, encrypted data would remain encrypted.
