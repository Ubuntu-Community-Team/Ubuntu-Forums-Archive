### Post by cariboo on 2009-08-28
Normally, you would donwnload the source fille, extract it, cd into the directory that you just created, then type:

```
./configure
make
sudo make install
```

There is usually a readme file that will tell you what dependencies you need to install. 

This assume you have build-essential installed. It is in the repositories.
