### Post by TheFu on 2017-08-17
I don't patch daily. Sometimes the first release of a patch has issues, not often, but sometimes.  I prefer to have problems happen when patching is in my mind, so it is something I quickly consider a patch as an issue.  I don't want to worry about an issue on a Tuesday morning when work needs to get done.

I'm fairly religious about weekly patching. The only reason I'd skip it is if I'm about to leave on an overseas trip and stability is more important for the next week or 2.
