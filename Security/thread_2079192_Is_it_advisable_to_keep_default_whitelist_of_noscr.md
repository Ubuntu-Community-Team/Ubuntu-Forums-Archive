### Post by Ms. Daisy on 2012-11-02
Well, the things that NoScripts is whitelisting are JavaScripts.  The particular one you pointed out is a script called "yahoo.com."  The McAfee site you linked is testing the website [www.yahoo.com](www.yahoo.com). They're not the same thing.  You're comparing apples and oranges.

You have to choose to trust something. If you keep Firefox and NoScripts updated, then I see no reason not to trust the whitelists.
