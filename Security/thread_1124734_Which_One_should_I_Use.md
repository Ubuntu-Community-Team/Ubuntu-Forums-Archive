### Post by slowth5 on 2009-04-13
Hi Trafficflow, In descending security order:  WPA2, WPA AES (CCMP), WPA TKIP, WEP.

Don't use WEP if you care about security, because it's horribly broken.  If your Netgear router and adapter support WPA2, then use it with a strong, random password.  I use a 63 character WPA2 password, since WPA is more secure than WEP, but it is still subject to brute force attacks.

### Post by slowth5 on 2009-04-13
Are you sure the wireless adapter supports WPA2?  I've seen someone have trouble connecting with WEP, but when he switched to WPA the connection problems disappeared.  I've also noticed the opposite.  Try WPA TKIP for maximum compatibility, then escalate from there to WPA AES and WPA2.  There are also 64 bit and 128 bit WEP, but as I said earlier, they are not secure.

If you can't get any encryption to work, then try an open access point to make sure you can connect at all.  Make it quick, you don't want any rogues doing nefarious deeds with your connection. ;)
