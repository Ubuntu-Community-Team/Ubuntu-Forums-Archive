### Post by alejandro.mc on 2010-10-30
Hi. I'll be traveling for a while and I'm taking with me an external hard disk. I will move content from my cam (pictures, videos, etc.) to the external HD using computers in hostels, private homes, campings, etc. So I have a couple of questions regarding possibilities and security risks:

1. If I install Ubuntu on the external HD it means that every computer I use along the way will require a motherboard allowing to boot from an external HD. Security will be maximized because I would be running Ubuntu every time, but I don't think every computer out there will allow this. Also, modifying boot order in BIOS in computers that are not yours, may upset the owners of those computers =), moreover if they don't know much about computers they may think you are harming or may harm them somehow.

2. If I simply take the external HD with the pre-installed FAT file system, then I will be able to simply connect the USB in any computer (mac, linux or windows). But security will be extremely poor, connecting anywhere will most likely infect my files many times (public computers are a mess of infections), and I don't want to loose content that is precious and irreplaceable.

3. If I change the file system (not NTFS of course, I mean EXT or other common Ubuntu file system) then security improves, but most computers out there use windows, so they will not recognize the external HD.

So all of this things are in my mind. Maybe something in between this options, that I'm not seeing, or hasn't popped up in my head. I'm a couple of months away from my trip but I wanted to start asking what ideas can you give me on this. Any comment will be useful and greatly appreciated!

I know it's basic and I'm sorry if I offend someone posting this in Security Discussions.

Thanks a lot in advance!

Alejandro
Buenos Aires
Argentina

### Post by peculiar penguin on 2010-10-30
I can't think of a "good" solution with only your existing equipment, depending on untrusted computers for handling the only copy of your data sucks. If you can afford to buy/borrow/rent/... a tiny netbook or a dedicated "image tank" (some sort of gizmo with a card reader and its own hard disk, stick your memory cards into it and their content gets copied to the internal hard disk) you should probably do so.

If you decide to go with a Live CD, which is what I would do, might I recommend you use[ Ubuntu Privacy Remix]("https://www.privacy-cd.org/")? It works the same way as a Ubuntu LiveCD, but it is meant to be used in insecure computers, such as you will be while traveling.

### Post by pizza-is-good on 2010-11-02
> **dogdo said:**
> Without getting off topic is there much difference between a ironkey usb and usb key encrypted with truecrypt?

Yes. Both are really secure, but Ironkey keeps the encryption key in a hardware chip inside the drive, and it is never copied to RAM, so it is 'not hackable'... which is a lot safer.

I have never used an ironkey, but I'd like to, and all the reviews I've read are positive. In paper, it is about as secure as it can get, that's why its one of the only portable storage devices allowed by the US Department of Defense...
