### Post by oldfred on 2011-03-05
Swap is just hard drive which is at least 10 times slower than RAM. If you only have 640K minus some video then 2 times RAM is a good size. System will not be faster with more swap. If you want more speed add more RAM or an SSD if it will work with your system.
