### Post by trikster_x on 2009-02-12
First off, aircrack is a terminal program.  No gui, sorry.  Second, [http://aircrack-ng.org](http://aircrack-ng.org) is the place to go for instructions on how to use it. And finally, you may run into some problems with your wireless card after you are done using aircrack.  Typing airmon-ng stop wifi0 followed by a reboot should fix it. Oh, and everything has to be run with the sudo command, so be careful.
