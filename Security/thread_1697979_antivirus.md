### Post by uRock on 2011-03-01
You really only need AV software if you are sharing files with a Windows system. ClamAV can be installed via Ubuntu Software Center in your Applications menu. 

I hear the Avast and AVG have versions of their software for Linux, but I have never used them. Currently I do not run an AV in Ubuntu. I have MS Security Essentials on my Windows installs and I have confidence in that software to keep Windows safe.

I used to recommend BitDefender for Linux, but when I last visited their site they no longer had a free version easily available.

### Post by tea for one on 2011-03-01
Good evening

I have been using Ubuntu for approx. 3 years and I have not bothered to install an anti virus package.

Allow me to explain further:-

My Ubuntu OS is on a separate partition to my data.
I back up my OS with Remastersys - available via Software Centre
I back up my Data with Grsync - also available via Software Centre

In the event of a virus/trojan etc., I would simply reinstall OS and Data.

However, fingers crossed, I have never received a virus.

I reckon it is quicker to reinstall (say 1 hour maximum) rather than slow down your system by constantly scanning the files with an anti virus package.

I must admit that I am reasonably selective in my website visits and downloads.

Also, I do not have a MS Windows partition.

Not at all. This question is virus pundits' pons asinorum: If they can't think past this fallacy, don't even try to reason with them, as they're hopelessly mired in rationalisation.
