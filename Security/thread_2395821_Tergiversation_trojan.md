### Post by TheFu on 2018-07-28
Any computer can be infected by a number of different vectors and behaviors. Email, websites, or any other internet download.  

Even on reputable websites, there is 3rd party advertising.  Those included ads have been used previously to attack desktop computers.  MSN and other reputable sites were used.  Almost always, only Windows is targeted, but purely from a security standpoint, many people choose to run ad blockers and disable javascript. Also, being careful of which addons/plugins you run inside any browser is important.

At the top of this subforum, there are multiple sticky threads about securing an Ubuntu system.
