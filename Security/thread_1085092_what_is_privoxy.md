### Post by bodhi.zazen on 2009-03-02
Privoxy : [http://www.privoxy.org/](http://www.privoxy.org/)

Tor : [http://www.torproject.org/](http://www.torproject.org/)

Tor documentation : [http://www.torproject.org/overview.html.en](http://www.torproject.org/overview.html.en)

See the "Staying anonymous" section 

You can run them separate (tor without privoxy and/or privoxy without tor).

For a discussion of the issues see also: [http://www.debianhelp.org/node/1237](http://www.debianhelp.org/node/1237)

Be sure to read the first few comments at the bottom.
