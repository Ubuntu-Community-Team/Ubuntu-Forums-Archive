### Post by SuperFreak on 2013-07-14
Formating alone is not the same a securely deleting (DBAN,Boot and Nuke etc) but it will make data recovery difficult. I think TestDisk & PhotoRec can be used to recover data that has not been overwrtitten from a formated disk.
