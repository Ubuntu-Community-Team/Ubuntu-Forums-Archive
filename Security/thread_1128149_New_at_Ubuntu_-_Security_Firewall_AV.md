> **Schrader said:**
> So i'm fairly new to Ubuntu and i've been wondering if i should consider using some sort of security to protect my system - Firewall? AV?

After setting up a large number of Linux based servers, here are some pointers:

 * No Anti-Virus recommended unless you are installing a mail server
 * Built in firewall (iptables or UFW frontend) to close unused ports
 * Decent update strategie (follow ubuntu security mailing lists, update regularly)
 * Strong passwords, use PKI for SSH logins
 * Disable root SSH login
 * Common sense
