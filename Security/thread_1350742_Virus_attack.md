### Post by Jive Turkey on 2009-12-09
This is what is referred to as a "scareware" attack.  The website is designed to mimic a windows security message and scare the user into buying a program to protect their computer.  The software they buy actually has more malicious software in it.  I find these sites hilarious.  I think you can safely ignore this.

I use openDNS and a firefox plugin called Web of Trust.  When I encounter such sites I like to report/review them with those services to help protect other users (and myself) from future attacks.
