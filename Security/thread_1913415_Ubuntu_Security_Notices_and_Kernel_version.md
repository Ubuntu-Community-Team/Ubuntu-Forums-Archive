### Post by 2F4U on 2012-01-22
As far as I understand security notices, they provide information about vulnerabilities. This doesn't mean that they are already fixed. So, if you got a kernel update yesterday, it would be based on a security notices some time ago.
Besides that, kernel updates happen not just because of security vulnerabilities.
