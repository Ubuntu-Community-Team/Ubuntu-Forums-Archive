It kinda depends on what kinda of malware you came across. If it was a Win32 PE/COFF drive-by kinda thing then it probably would be only effective aginst a windows pc or wine environment. If it was some bad JavaScript embedded in a webpage then it can be cross-platform and effect Linux machines. If you have a java plugin installed in your browser or a flash plugin installed in your browser, those can also be cross-platform forms of malware.  Its hard to say what it was without more information about what it was.
