### Post by bodhi.zazen on 2010-03-30
> **Nisal said:**
> i was wondering how many ppl use AV in ubuntu >

if u use one what is it ?

There are currently no active viruses in Linux.

The typical user cases for antivirus in Linux are:

1. Servers - run antivirus on mail servers or samba, etc, to scan uploaded files for viruses.

2. "Netiquette" - Some people feel they should scan files before passing them to windows.

See : [Antivirus - Community Ubuntu Documentation]("https://help.ubuntu.com/community/Antivirus")

Personally I do not use antivirus on Linux at all.
