### Post by QLee on 2009-10-22
[QUOTE=slydevil22]What are the best services to allow/disallow on my machine and SSH server to help better secure my system?[/QUOTE]

Well, the simple answer is to not run any services that you don't need.

One of the big problems with using the word "best" in a question is that people's opinions differ on what is best for a given situation. You'd get better (meaning more helpful) answers if you'd explain what your goal is for the server and tell how you have it set now, then people could more easily suggest what you could do to make it more secure.
