### Post by Bachstelze on 2010-09-16
> **yeleek said:**
> 
Can i still use this without causing myself problems?  If so what line to edit or is it a new one?

New line, put it at the top of the file (before the comment).
> 
  Could someone actually explain please what the success=2 default=ignore means too pls?

It means that if the PAM module returns success, PAM will jump over the next two lines (so it will continue at the pam_permit line). All other return values are ignored, and PAM will proceed to the next line (pam_winbind).
