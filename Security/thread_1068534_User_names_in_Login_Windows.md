### Post by randyklein99 on 2009-04-20
I just ran into this problem myself. If you go into the Login Window Preferences and Local Tab be sure that the style is "Themed with Face Browser".  Without the Face Browser part, my user names don't show either.

### Post by kpictjl on 2009-05-07
I'm having the same problem.  I've tried the "Happy GNOME with browser" and "Human List" GDM themes with no joy.  This problem started when I did a clean install of 9.04 a few days ago.

I worked around this by problem by selecting "Theme faced browser" from the "Style" pull down list of the "Local" tab on the "Login Window Preferences" tool.
