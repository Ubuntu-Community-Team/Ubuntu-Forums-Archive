### Post by ajgreeny on 2010-08-09
A new version of GDM appeared in 9.10 which had, and still has in 10.04, almost no configuration options available.  I think you will need to learn to live with it for now.

As for logging in as root, that is not possible with a default ubuntu install, and if you really want to do it you will need to ask elsewhere than here, as it is against forum policy to tell you how to do that.

See [https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo) for info about root privileges in ubuntu
