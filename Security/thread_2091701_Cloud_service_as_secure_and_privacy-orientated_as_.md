### Post by Ms. Daisy on 2012-12-07
See this related thread
[http://askubuntu.com/questions/75377/encrypting-data-in-ubuntu-one](http://askubuntu.com/questions/75377/encrypting-data-in-ubuntu-one)

Per this FAQ
[https://one.ubuntu.com/help/faq/what-security-and-privacy-policies-does-ubuntu-one-have/](https://one.ubuntu.com/help/faq/what-security-and-privacy-policies-does-ubuntu-one-have/)
> "If you are concerned about storing your files unencrypted in the Ubuntu  One cloud, you could always store the files already encrypted so Ubuntu  One never sees the plain text files. *Doing so may prevent the proper  functionality of some Ubuntu One features such as multiple computer  synchronization, web browser access, and sharing with others."*  So yes, you can encrypt it locally & upload encrypted files to Ubuntuone.
