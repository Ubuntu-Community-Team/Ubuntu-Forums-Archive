Security is a really broad topic: network, operating system, software (apache, mysql, php) you name it... it would be nice to reduce or limit your learning spectrum and start from the very beginning.

And a good start is just knowing your Ubuntu System, the basics, like how the security at Os level works, default Ubuntu firewall, etc. Ubuntu is really safe out of the box but try to learn about basic Linux security 1st and then jump into network.
