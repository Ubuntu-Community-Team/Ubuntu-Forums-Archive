### Post by Irihapeti on 2013-12-24
There are several password managers for Ubuntu, which you can find by searching in either Software Center or Synaptic (if you have it installed). My personal preference is KeePassX, because I can use the same password file on different OSs.

From what I can understand, Data Guardian passwords can't easily be imported directly into most other programs. You might be able to export them first from Data Guardian and then import the resulting file to another program, but you'd need to test that out.
