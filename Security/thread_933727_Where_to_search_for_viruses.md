### Post by cariboo on 2008-09-30
Thunderbird always list included files in a pane below the message window. See attached screenshot. If you see something in that pane that you suspect, don't forward the email. It is not up to you to protect other users from viruses. If the don't have proper protection set up on their computers, they have far more to worry about besides getting a virus from you.

If you find a joke or a file that you think your friends would like send them the url so that they can check it themselves.
