### Post by MCVenom on 2010-07-06
It looks like a shifty site, I'll say that... but I don't think it gave you a virus. Firefox probably froze over a popup or other ad it was trying to display (it's chock full of ads, apparently). I encountered no trouble loading or navigating the site; except for the fact that it does appear to be somewhat suspect.

Luckily, any malware that would be on that site would probably be targeting Internet Explorer on Windows. So I think it's safe to say that you're fine ;)

You might want to look into the NoScript addon for Firefox though, it would stop any malicious scripts from running while you browse. Takes a while to get used to though. :P
