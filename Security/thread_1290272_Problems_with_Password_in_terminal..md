### Post by DaithiF on 2009-10-13
the terminal does not display the characters of your password as you type it (as a security measure to stop others from seeing what you type), but rest assured it is accepting whatever you type.  so type your password carefully and hit return/enter.

### Post by aos101 on 2009-10-13
Do you mean it just doesn't give any indication in the terminal when you type in letters on the keyboard?  If so, that's normal.  When "sudo" asks for your password, it doesn't show on screen your password or even "*" characters when you are typing it in.  Just type in your password when it asks, press enter and it should work.

Incidentally, I think installing the 3ddesktop package will only work with Ubuntu 6.06 (Dapper) as it isn't in the other currently supported Ubuntu releases.
