### Post by lisati on 2009-04-28
If you can, make the key WPA instead of WEP, because it's harder to break.
If you have control of the router, you might also want to restrict access to it by MAC address as well.

There's an interesting discussion about security in Ubuntu at [http://ubuntuforums.org/showthread.php?t=510812](http://ubuntuforums.org/showthread.php?t=510812)
