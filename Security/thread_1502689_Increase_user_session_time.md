### Post by Rhubarb on 2010-06-05
I'm guessing that it's the default screensaver time that's causing the trouble for you.

System --> Preferences --> Screensaver
From there you can either increase the time at which the screensaver is activated, or just uncheck the "Lock screen when screensaver is active" checkbox.

This has been annoying for me as well, it's easy to fix though :)
