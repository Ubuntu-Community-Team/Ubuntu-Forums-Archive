### Post by Kinstonian on 2010-12-13
There is an entire field of work around reverse engineering malware.  The process is broken into three parts: static analysis, dynamic analysis and the recent addition is memory analysis.  I suggest you find a book like "Malware Forensics" or "The Malware Analyst's Cookbook and DVD".  Malware Forensics is less technical and might be best if you're new to the field, but the Cookbook is one of the best books I've ever read.

BTW if you're only interested in figuring out how LOIC works, the source code is available on [sourceforge](http://sourceforge.net/projects/loic/files/loic/loic-1.0.4/).
