### Post by TheFu on 2021-09-27
Standard support for all versions of Trusty ended in April. Many desktop flavors ended support in 2019 with no ESM available. Please use supported releases only.
If you have ESM support, please use those contacts for questions.
Ref: [https://wiki.ubuntu.com/Releases](https://wiki.ubuntu.com/Releases)

On 16.04-18.04 systems, use the **ubuntu-support-status** program to see the support status.
For 20.04 and later, use **ubuntu-security-status**.
