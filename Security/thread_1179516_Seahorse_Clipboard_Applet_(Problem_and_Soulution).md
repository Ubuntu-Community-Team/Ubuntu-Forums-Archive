### Post by urielgm on 2009-06-05
I have been trying to use the encryption applet with little success. Every time that I logged in, the applet forgot the passwords and I had to make a new one in Seahorse in order to make it work. I also tried starting the seahorse daemon without any luck.

In order to fix this problem go to 

System > Preferences > Encryption and Keyrings

Go to the PGP Passphrases tab and select Remember Keys (or something that sounds similar).

You can then set your default passphrase in the Encryption tab.
