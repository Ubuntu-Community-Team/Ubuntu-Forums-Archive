### Post by kshaff03 on 2008-10-26
I've installed SVN and I'm running svnserve to make a repository available. I've created a new user called "svn" that I've setup that has limited privileges on the system. My intent is to run svnserve as this restricted user thinking that this would be more secure.

I've gotten this working, but I have to log in as "svn" and manually start svnserve, using the following command to get it running:

svnserve -d -r /home/svn/svnroot

What I'd like to do is start svnserve automatically once the machine boots.  There are plenty of tutorials that explain how to do that.  What I am unclear about is whether or not I can run svnserve automatically as the machine boots as this restricted user and if so, how do I accomplish that?

Any thoughts on this matter would be greatly appreciated.  Thanks in advance.

-Kevin

---

### Post by KeyserSoze93 on 2008-10-27
Add to /etc/rc.local (BEFORE exit 0, and remembering the ampersand) the line:

su svn -c svnserve -d -r /home/svn/svnroot&
