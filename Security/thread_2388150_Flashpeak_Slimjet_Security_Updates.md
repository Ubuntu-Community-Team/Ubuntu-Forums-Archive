### Post by Irihapeti on 2018-03-29
Thanks for the detail. That means that you need to update the package manually when a new version is available.

Check that you have Update set to something other than "Never". It's in the advanced settings, close to the bottom of the page. You'll then get notified when there's an update and can download it.

It's important to keep any browser up to date for security reasons. If you do that, I think that Slimjet should be as secure as any other browser. (I've not heard anything to suggest that it's not.)
