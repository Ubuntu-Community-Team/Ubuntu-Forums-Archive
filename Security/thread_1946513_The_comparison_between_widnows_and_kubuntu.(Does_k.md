Only if you download Windows files SHOULD you have Antivirus, otherwise I wouldn't worry about it. Most viruses and spyware affect only Windows or Mac OS X; there are very few if any for Linux.

If you do download Windows files from unsafe locations, then get Clam Antivirus or Avira or that. I personally don't use Antivirus on my Kubuntu machine cause I'm not worried. Also I don't torrent or download from unsafe locations.

As for system optimization software, no, Linux keeps itself optimized.
