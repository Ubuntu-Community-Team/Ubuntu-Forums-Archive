### Post by uRock on 2010-11-06
Linux firewalls are not the same as Windows firewalls. GUFW allows all outgoing connections and blocks all incoming by default.

Please read the [Ubuntu Security]("http://ubuntuforums.org/showthread.php?t=510812") sticky to understand how firewalls and security work in Ubuntu.
