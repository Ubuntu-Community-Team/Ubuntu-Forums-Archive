### Post by DarkDancer on 2008-10-08
This should work. Hold down alt and hit f2. A program run box should appear on the screen. Type in "gksudo nautilus" (no quotes), enter your login password when prompted. Nautilus will open in Superuser mode. Navigate to the folder where you need to drop the file, and complete the action (put the file in there). This will work in Gnome, if you are in KDE, some other advice will be needed I suspect.
