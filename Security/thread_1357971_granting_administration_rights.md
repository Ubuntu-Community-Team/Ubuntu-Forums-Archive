### Post by BkkBonanza on 2009-12-18
By default as long as they know their password they get admin rights. If they stay in the GUI then the system will prompt them for the password when they need to use admin rights. The intent is to prevent others from drastically messing up the system. (They can do it themselves!)

So I would say the idea would be to explain to them that when they are doing something like updates or security related they will need to enter their password as a safety measure.

Also, if you select menu item System, Administration, Update Manager and the Settings button you are given a choice that allows doing auotmatic security updates and also a choice for other recommended updates. I usually run with both of those checked without problems. That way I get new updates on my apps even if not just for security reasons.
