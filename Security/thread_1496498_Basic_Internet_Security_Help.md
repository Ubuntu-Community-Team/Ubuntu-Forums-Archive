### Post by mituw16 on 2010-05-29
hello and welcome to ubuntu! 

For a firewall, you can open the ubuntu software center and then install a program called GUFW, this is essentially just a GUI frontend for UFW, which is a simplified iptables, anyways, GUFW is a very simple interface...

There is little need for anti-malware  and anti-virus on linux, unless you want to scan for windows virus... :P 

The best thing you can do to protect your linux desktop is turn off file sharing unless you want it, a good idea is to block all incoming traffic on your firewall unless you specifically need a port for a service, choose GOOD passwords, change your ssh port off the default 22 port (if you even have openssh installed that is...?), and set permissions for files and such. 


Sorry for not going very into depth in this post, but without a more specific question from you, I could write 50 pages on security  :D .

Cheers!
