### Post by ottosykora on 2012-05-14
for all who are at the moment missing some gui for pgp encryption, there is small half way work around.

The java based portable pgp. It rund under win and linux provided java is installed. Alternatively there is also a package available with own java on board.

[http://ppgp.sourceforge.net/](http://ppgp.sourceforge.net/)


note , it does not use the gpg or keyrings installed on the system, you have to import them into the program, it has separate keyrings and is independent.
