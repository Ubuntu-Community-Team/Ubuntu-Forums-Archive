### Post by TheFu on 2015-02-08
Yes. You need AV, IPS, IDS, prayer and luck when running Windows on a network.  IMHO.  If you don't have any network connected, then the first three aren't needed. ;)  

Hoping to run high-end games inside a VM will probably be disappointing without major effort to get graphics passthru working to a dedicated monitor, video card. Don't think virtualbox can do that, BTW.  Xen and lVM can, with huge amounts of skill, extremely specific hardware, and effort.  There's another thread here about it.  It is more effort than I'm willing to do.

OTOH, for chess games, virtualbox video performance will be fine. Somewhere between chess and whatever the latest FPS is where the **gaming happiness** cutoff happens.  If a game works well with a $40 GPU, it will probably be fine under a VM without all the video passthru stuff.  If a game demands a $60 GPU ... well, be prepared for disappointment with the framerates and resolution.
