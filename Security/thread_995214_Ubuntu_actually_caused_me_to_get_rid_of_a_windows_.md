### Post by nintendude7cubed on 2008-11-27
I did a virus scan on my windows pc a long time ago with avg free edition. It said there were 2 trojans on my external harddrive. I tried to get rid of them because i knew they were compromising my computer's performance and it said the file was too big for the vault. So i choose go to file and it says the directory doesn't exist... I knew the directory was called RECYCLER and contained both an executable file and a .dll that contained the trojans, but the directory just didn't show up among the folders in the drive so i couldn't delete it manually. Anyway time goes by and i eventually get my ubuntu 8.10 disk in the mail and installed it within windows with a 30gb (which is plenty) reserve. I am obviously trying to figure stuff out still, but when i went into my external harddrive from there, it contained this RECYCLER folder that I still remembered i couldn't find before. Immediately i just deleted the whole folder. When i returned to windows and ran another virus scan, my computer went from 2 to 0 infections after i deleted this folder. ubuntu not only got rid of these trojans, but also caused the performance of the windows side to improve greatly! I think I'm gonna like this OS once i get used to it.
