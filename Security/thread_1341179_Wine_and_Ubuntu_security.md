### Post by cariboo on 2009-11-29
> 1-If I install Wine,Can I get infected by Windows virusses?

Yes you can, but they won't be able to do much.

> 2-Ubuntu is 100% safe? I mean,I don't have to install anti-virusses,anti-spyware,firewalls,etc? And can insert,for example,credit card numbers in the net?


No operating system is 100% secure, unless the computer is shut off, disconnected from the network and in a locked box. Windows malware has no affect on Ubuntu, and a default installation with no services running doesn't need a firewall.

As far as credit card numbers go, just take the same precautions you use when running Windows.

I would suggest you have a look at this [Ubuntu Security]("http://ubuntuforums.org/showthread.php?t=510812") stickie
