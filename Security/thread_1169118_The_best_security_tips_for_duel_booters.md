### Post by cariboo on 2009-05-25
Windows viruses don't work on Linux, so no worries there. The only Linux viruses are only proofs of concept and have never been released to the wild.

The only other things you should be concerned with are use a good strong password, and stay away from web sites you normally wouldn't go to running Windows.
