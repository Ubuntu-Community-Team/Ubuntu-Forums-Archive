### Post by Bucky Ball on 2016-01-29
Check the hash, ask around, do some research, but the bottom line is when you download and install apps from a third-party and not the official repos you do so at your own risk, exactly as you would if you did the same in Windows. 

If you're worried, install Kubuntu as a virtual machine and install it in that. If it's not what you wanted, no harm done. When posting about these things it helps a lot if you include a link to the app you're talking about. Pretty much guess work for anyone trying to help otherwise and you certainly won't get any opinions on the app that way. Don't know what it is.
