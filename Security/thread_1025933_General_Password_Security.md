### Post by arvevans on 2008-12-30
Many login systems only allow 3 attempts at guessing a password.  After the 3rd failed attempt at logging in within some short  period of time the login system continues to accept guesses, but will not allow any of them to succeed until some longer period of inactivity has transpired.  This usually foils most of the vaunted password cracking programs.

Other login systems detect multiple failed login attempts and lock out any use of that ID.  They then require that the verified owner of the ID re-register a new ID and/or password in order to continue using the system.

Some login security systems detect false login attempts and take action to trace and trap the originator of those attempts.  These systems usually report directly to corporate security teams or to law enforcement.
