### Post by tubbygweilo on 2009-11-10
wasi_deer,
you can indeed hide things from other users but this is security by obscurity and as such not real security but it may well meet your needs.
[LIST]
[*]The file or folder name can be made invisible by having a "." as the first character of a name.
[*]The permissions of the file or folder can be amended to exclude all but you.
[*]You can hide the file or folder within a Truecrypt container.
[*]You can encrypt a file by [nautilus]("http://ubuntuforums.org/showthread.php?t=1312672").
[/LIST]
These are a few ideas that come to mind but I'm sure others can contribute a few more.
