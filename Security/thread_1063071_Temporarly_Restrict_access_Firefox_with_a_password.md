Wouldn't be easier to create another user Linux is designed as a multiuser system and a normal user doesn't have any access to other users data. Another thing you could do to limit what somebody else can do, is before you leave the  computer, start up a guest session, then users have limited privileges and any changes they make aren't permanent, and they can't log out and into your session.
