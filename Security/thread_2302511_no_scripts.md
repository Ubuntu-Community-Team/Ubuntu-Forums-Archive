### Post by Bucky Ball on 2015-11-11
Welcome. Presuming you are talking about Firefox, I use NoScript and AdBlockPlus, both of which are available in Firefox by going to Tools> Addons> search for and install them. They will create icons in the toolbar at the top right of Firefox. 

NoScript may create undesirable outcomes until you get used to it. It needs a little tweaking and training and is great. Once you allow the pages you want to allow everytime you go there, it gets easier. If some function is missing from a page, though, check noscript and see what it's blocking. 

I always allow the page itself, but add anything else if required for what I want to do (just about every page is linked to facebook and google in some way, for example, which slows things down and is rarely required).  

Good luck. :)

### Post by night_sky2 on 2015-11-11
NoScript for Firefox is definitly the most powerful extension available. It allows you to micro-manage every single script on every page, blocks them all by default and provides you with security features such as Application Boundaries Enforcer (ABE) and anti-Clickjacking. BUT it's really a pain in the *** to set up for every site and you may waste a lot of time. Some people feel it's not worth it and prefer to use a multi-purpose bocker like uBlock, Adblock Plus, bluHell Firewall or Ghostery which works based on black lists updating automatically, so it's pretty much ''set and forget''.

### Post by night_sky2 on 2015-11-11
> **Bucky Ball said:**
> I run adblock plus as well, as I stated in my post. It doesn't do quite the same thing as noscript as far as I know. :)
No but you can add filters in ABP and block not only ads but trackers, malware domains and other annoyances.

Fanboy ultimate List: [https://www.fanboy.co.nz/filters.html](https://www.fanboy.co.nz/filters.html)

The thing with NoScript is that it block ALL javascripts by default, including those that are needed for websites to work properly. It is up to the user to figure out what is needed and what is not, hence it's all about micro-management. Some people like to have that level of fine-grained control, others prefer to sit back and enjoy browsing the web without having to bother.
