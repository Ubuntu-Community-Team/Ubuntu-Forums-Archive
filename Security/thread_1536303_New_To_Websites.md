### Post by jpmelos on 2010-07-22
If you are using Apache Web Server, look for how to configure **.htaccess**, that's a file you put in a folder, and it performs a lot of things, including password protecting.

There's something about it [here (http://helpdesk.bluehost.com/index.php/kb/article/115)]("http://helpdesk.bluehost.com/index.php/kb/article/115"), to get you started. But I think some more research will be necessary. I don't know if it's enabled by default in Apache.
