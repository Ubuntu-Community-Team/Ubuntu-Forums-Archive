### Post by TheFu on 2021-06-16
Keepassxc.

I do not integrate into any browser. I want browser bugs and browser addons not having direct access to my passwords.  For me, that is common sense - sorta like not putting any passwords on the internet. A common sense "smell" test.

Keepass has many password managers that are compatible due to the DB format being widely shared.  That means we can find 20+ programs across Linux, Windows, Android platforms that will work with KeePass-V2 DB.  The trick is to only make changes/updates on 1 system, then have that replicated to all the others.  I do it daily, automatically, a little after midnight.  I'm a little afraid to have write access setup on storage that automatically syncs multiple platforms if any are my "write" system.  For read-only (in my mind only) versions/devices, it doesn't matter.
