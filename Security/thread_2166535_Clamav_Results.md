The scan summary is telling you that there are 2557166 known viruses in clamav's database. There are no known viruses in the wild that affect Linux distributions so running clamav looking for a virus is more an academic process than anything else. This is not to say that there aren't Trojans out there that can infect your system but most of these rely on user intervention in order to work.
