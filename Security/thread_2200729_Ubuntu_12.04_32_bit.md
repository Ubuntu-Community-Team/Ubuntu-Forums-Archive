### Post by Bashing-om on 2014-01-20
gordie692; Hi !

pros:
12.04 ! 
As it is a Long Tern Support release - supported 'till 2017. All software in the LTS releases undergo rigorous testing and are under the closest inspection for any difficulties. The more effort is put forth to insure the integrity of the LTS version.

cons:
Any other release other than LTS is short lived (currently 9 months) and many times fixes are pushed "upstream" for the next release. If you want the latest greatest bleeding edge stuff, then these are for you ! It is bleeding edge and some breakage can be expected. It may be no big deal, but every few months one has to upgrade to the next supported version.

For security and stability, hands down stick with the LTS versions - 14.04 is due out in April ! -
