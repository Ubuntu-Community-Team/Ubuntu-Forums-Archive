### Post by TheFu on 2013-11-04
I have bad news, unless you did either of these things:
* made the key recovery file and stored that somewhere else - [http://askubuntu.com/questions/38336/how-do-i-recover-my-data-from-an-encrypted-home-directory](http://askubuntu.com/questions/38336/how-do-i-recover-my-data-from-an-encrypted-home-directory)
* made great backups of the data you need to recover.

Without either of these things, that data is gone, gone, gone, gone.

People using encryption need to have **[backup religion]("http://blog.jdpfu.com/ALE/ALE-NW/a09_backups.html").**
