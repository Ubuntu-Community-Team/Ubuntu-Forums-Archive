### Post by Dr Small on 2008-10-28
Where is UbuntuCodeofConduct-1.0.1.txt located? If, for example, it's on your desktop, then you would run:
```
gpg --clearsign ~/Desktop/UbuntuCodeofConduct-1.0.1.txt
```
