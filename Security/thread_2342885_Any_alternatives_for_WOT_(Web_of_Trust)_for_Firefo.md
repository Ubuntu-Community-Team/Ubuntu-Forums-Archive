### Post by bcschmerker on 2016-11-11
**Not as of 10 November 2016.**  A port of the Chromium extension ScriptSafe, which has an ESM function concerning advertising and similar tracking Web services, to Mozilla® Firefox® is still not ready for public beta.  The closest comparable service I know of, Norton® Safe Web™ (with its client-side app Safe Search™), is only available for Microsoft® Windows® 6-up and Apple® Mac OS® X™ 10.5-up.  (Symantec does not currently offer an Internet security suite for LinUX).
