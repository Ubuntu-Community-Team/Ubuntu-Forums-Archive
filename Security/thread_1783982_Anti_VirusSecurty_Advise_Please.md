### Post by nomko on 2011-06-17
> **sanderd17 said:**
> Hi again :D
 
This page lists all viruses known in the Linux world and gives some tips about who should or shouldn't install an AV.
 
[https://help.ubuntu.com/community/Linuxvirus](https://help.ubuntu.com/community/Linuxvirus)
 
Looking at that list, the youngest virus is from 2007... In my opinion, due to the better security level and rights management of Linux there's no real motivation for creating a virus specific for the Linux operating system. To get a virus installed without the knowledge of the user, the virus creator has to find a way to bypass the security/rights of not 1 Linux system, but every Linux system there is since not 1 Linux system is equal to another system.
 
All other virusses are older. Why the change of getting infected by a Linux specific virus is due to the better level of security of Linux compared to Windows. To install programs you need root priviliges. In some folders a regular user doesn't even have read/wright rigths. As for Windows, a regular user still can install programs, remove files, can access critical files, etc. It's even possible to install stuff in the background without the knowledge of the Windows user! The disadvantage of the Windows operating system is well used by virus creators. And that makes Windows vulneribale to virusses and trojans. The security system of Linux however was designed to avoid all this. All actions has to be approved by a root or an user with root priviliges.
 
If you for some reason do get a Windows virus on your Linux system, it won't harm your system. Just be aware that for Linux the virus is nothing more than a file with inexplicable gibberish, but the data of that virus (or infected file) is still saved in a correct way as it should be. So, if you pass that file on to a Windows user, the Windows user will get the virus in a way it suppose to be, as a virus which contaminates his/her Windows system.
 
My best advice is:
When you exchange a lot of files with Windows users, yes, do use an anti-virus program. And if you get a lot of stuff from Windows users, save those files on a Linux formatted disc (ext3/ext4). This will avoid getting other files infected when you're using a Windows formatted disc (NTFS/FAT32) or that the virus can write itself dwon on a Windows formatted disc. This is also a secure way to scan files for virusses before sending it to Windows users.
