### Post by user201304 on 2013-04-08
In order to SSH in my Ubuntu box I need to use the Windows app PuTTy, which uses a file format called ppk for private key files. This file format appears to be different from the private key generated by the ssh-keygen command. If I've generated a private key using ssh-keygen, can I convert it to a format that PuTTy can understand?  Edit: to answer the question, yes: use puttygen -> Conversions -> import key
