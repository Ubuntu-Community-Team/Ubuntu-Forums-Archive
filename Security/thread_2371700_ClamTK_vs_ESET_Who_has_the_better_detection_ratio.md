### Post by mastablasta on 2017-09-18
they scan for windows viruses which don't work in linux anyway. they are more or less useless unless you also have windows PCs on network, share files with them and would like to save those from infection. but in that case widnows PCs should have antivirus and again there is no point in scanning for virus 2 times (onnce on linux and then again on windows). another example when you might need it is if you are running an email server.

other options are Avira and Bitdefender. there are a couple more all CLI based. but you set them up once and then there is no need for GUI anyway. another option is to do a scan every now and then using for example Avira rescue disk. 

i don't know about ESET, but clamav has relatively bad detection rates.
