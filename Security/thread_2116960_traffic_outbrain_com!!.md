### Post by Ms. Daisy on 2013-02-18
You can block scripts, too. If you're using Chrome then install [NotScripts]("https://chrome.google.com/webstore/detail/notscripts/odjhifogjcknibkahlpidmdajjpkkcfn"). The counterpart in Firefox is [NoScripts]("https://addons.mozilla.org/en-us/firefox/addon/noscript/").  These extensions block all the javascripts, including some 3rd party ads, from executing.  It can also break websites that can't function properly without javascript, so it takes some getting used to.
