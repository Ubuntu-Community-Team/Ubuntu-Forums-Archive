### Post by CharlesA on 2010-12-23
If you are concerned that your machine was compromised, backup yer data and reinstall.

If someone knows your password, they can change it the "normal way" without the need to go into recovery mode.

The reason the recovery option doesn't prompt for a password, is because the root account is locked by default in Ubuntu.
