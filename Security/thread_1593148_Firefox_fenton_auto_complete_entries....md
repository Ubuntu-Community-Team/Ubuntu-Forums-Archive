### Post by lovinglinux on 2010-10-11
Interesting question. Most likely that someone had access to your computer and used it to check e-mail.

Go to Firefox "Preferences >> Security" and click "Saved Passwords". See if there is an entry with such user login and delete it. Set a "Master Password" so others can save/read passwords in your account. If the login shows up again, then you might have been hacked.
