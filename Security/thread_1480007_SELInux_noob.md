Honestly I would start by running Fedora.

The selinux user guide is not easy to comprehend, but Fedora includes (graphical) tools to help you manage selinux.

See also :

[http://fedoraproject.org/wiki/SELinux/Understanding](http://fedoraproject.org/wiki/SELinux/Understanding)
[http://fedoraproject.org/wiki/SELinux/FAQ](http://fedoraproject.org/wiki/SELinux/FAQ)
[http://fedoraproject.org/wiki/SELinux](http://fedoraproject.org/wiki/SELinux)

SELinux is difficult or impossible to run on Ubuntu (meaning last time I tried it failed) and honestly not many Ubuntu users are familiar with SELinux, so support will be much better on the Fedora forums then the Ubuntu forums.
