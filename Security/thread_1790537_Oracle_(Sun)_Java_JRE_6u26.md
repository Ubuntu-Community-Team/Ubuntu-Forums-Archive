### Post by Pjotr123 on 2011-06-25
Oracle has issued a new version of Oracle (Sun) Java JRE: 6u26. This contains security fixes: [http://www.oracle.com/technetwork/java/javase/6u26releasenotes-401875.html](http://www.oracle.com/technetwork/java/javase/6u26releasenotes-401875.html)

Because it's a security update, I expect that it'll be updated in the Ubuntu repo's as well. I advise to wait for that.

For experienced Linux users who (for whatever reason) do not wish to wait for the regular updates: I've updated my how-to for manual installation of Java: [http://sites.google.com/site/easylinuxtipsproject/java](http://sites.google.com/site/easylinuxtipsproject/java)

Have fun! :)
