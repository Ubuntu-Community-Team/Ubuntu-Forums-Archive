### Post by cariboo on 2012-08-24
If you are really worried about family members/others reading your personal documents/email, set up separate accounts for them. Linux was designed as a multi-user system from the start, it is possible to set things up so that others don't have access to each others home directory. Simply set user home directories permissions to 700, and the only user that can access all home directories is the system administrator.

Of course you have to remember to log out, when you leave the computer.
