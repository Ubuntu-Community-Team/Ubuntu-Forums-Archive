I've seen some different hosts all trying to connect to the same UDP port.  One host (probably more) repeatedly trying to connect to the same port.  There are relatively few ports the hosts are trying to connect to.  And there is a payload in the packets, which means you can run Wireshark to view it.

It doesn't look like a port scan to me.
