### Post by maglinu on 2012-10-23
The comodo antivirus for linux (CAVL) doesn't have a a web or network scanner/shield - ie it doesn't respond to or block access to bad sites - unlike a lot of windows AVs.

Once windows platform  malware from such a site appears on your machine (CAVL only detects windows platform malware) it should respond I think - it is a real-time AV (the only free one I know of for linux). I ditched it myself, because it was very heavy on system resources. A weekly scan does me.

(Bitdefender Trafficlight says there's nothing wrong with that site at present, so does virustotal - doesn't mean it wasn't a source of malware recently though).
