### Post by Pjotr123 on 2010-04-15
Oracle (Sun) has released Java JRE 6 update 20. This contains several security fixes: [http://java.sun.com/javase/6/webnotes/6u20.html](http://java.sun.com/javase/6/webnotes/6u20.html)

I've adapted my how-to to this new update. You can update your JRE as follows: [http://sites.google.com/site/easylinuxtipsproject/java](http://sites.google.com/site/easylinuxtipsproject/java)

I don't know whether these security fixes are as necessary for Linux as for other operating systems. Nevertheless I advise to update quickly, to be on the safe side.
