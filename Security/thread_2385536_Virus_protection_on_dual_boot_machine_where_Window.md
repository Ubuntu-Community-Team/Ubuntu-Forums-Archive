### Post by QIII on 2018-02-21
Yes, you should use anti-virus protect with Windows.  Always.

If you cannot afford a third-party anti-virus package (I would bite the bullet and do it, or see if your ISP provides a subscription to an anti-virus package as part of your account), you may activate Windows Defender Security Center.  Bear in mind that it is not as effective as a third-party solution, but it is better than going commando.

Microsoft's instructions for Windows Defender Security Center are [here]("https://answers.microsoft.com/en-us/windows/forum/windows_10-security-winpc/how-to-enable-and-use-the-built-in-windows/0682bf04-b8d2-4646-9dfb-e5823a5e2117").

Cheers!

### Post by ajgreeny on 2018-02-21
There are plenty of free antivirus applications available which will do a surprisingly good job, particularly if you are using Windows only occasionally.

Look at Avast, Avira, AVG, and also have a good read through [https://www.techradar.com/news/the-best-free-antivirus](https://www.techradar.com/news/the-best-free-antivirus) where several others are also mentioned.

Don't forget that the best antivirus protection is from the user keeping their wits about them and not clicking on spurious links in emails or visiting dubious websites. In my Windows days I always used free antivirus and was never hit by anything.

I now run an old Windows XP as a VM in VBox simply to update my satnav device the same as you need, and have had Avast on that for about three years now with no problems.
I know XP is out of support and risky, but I have a clean snapshot of the OS in VBox, so if ever it did get a virus I would simply dump the current version and boot to the clean snapshot.
