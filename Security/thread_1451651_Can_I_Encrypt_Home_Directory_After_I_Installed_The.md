### Post by FuturePilot on 2010-04-10
You can encrypt your home directory post install by following this guide [http://blog.dustinkirkland.com/2009/06/migrating-to-encrypted-home-directory.html]("http://blog.dustinkirkland.com/2009/06/migrating-to-encrypted-home-directory.html")

Encrypting the whole drive would require a reinstall using the alternate install CD.
