> Hi,

There is no universal standard as it will vary depending on the environment and the value of the box you are protecting.

The average home desktop user - no such policy is needed.

Secure servers with sensitive data - such a policy makes good sense.

See : [https://help.ubuntu.com/9.10/serverguide/C/user-management.html](https://help.ubuntu.com/9.10/serverguide/C/user-management.html)
