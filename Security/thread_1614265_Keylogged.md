### Post by CharlesA on 2010-11-05
You would need to have root/admin access to install any sort of keylogger onto a linux machine.

It's likely that there was a weak password used and all you can really do is change the password to something 8-12 characters long with numbers and letters.

I would have to agree.

For a start, keyloggers are uncommon on Windows, let alone Linux.  Most of these breaches tend to be down to bad passwords, or the same password being used on a compromised/untrustworthy site.

I have a strong password for my primary e-mail account, and that password is used nowhere else.  If you have anything else associated with that password (paypal/ebay?) - change the passwords.
