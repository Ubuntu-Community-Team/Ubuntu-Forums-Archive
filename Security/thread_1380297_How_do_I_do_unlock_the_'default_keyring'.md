### Post by ld.4lpha on 2010-02-01
Try this:

1) Open Applications > Accessories > Passwords and Encryption Keys
2) Click the Passwords tab
3) Right-click on the Passwords folder and select "Change Password"

Set this password equal to the one that you use when you log into your machine.
