### Post by rookcifer on 2009-09-04
> **loosie said:**
> Hi,
Brand new to Ubuntu - sort of using by default. I've been told it's virtually virus free, but....

What are the basics that I should do to ensure safety, for web browsing, downloading, etc? Are there virus/spyware programs that I should have??

The first thing to realize is that you download all your software from the Ubuntu repositories.  It doesn't work like Windows (where one must go around the Internet at random places installing potentially dangerous software).  With Linux, the distribution takes care of "hosting" the software for you.  Just open your package manager, search for what you want, and install it.

Second thing is do not install or use any servers you don't need (like ssh or vnc).  If you do use these, then learn how to secure them (use a strong password, for instance).

Don't worry about firewalls or AV.  You will only potentially need a firewall if you plan on running servers.  And you will not need AV at all.

You can use the typical extensions for Firefox to help secure it (Adblock, NoScript, etc.).

When you get a bit more advanced, you can look into using a MAC system like AppArmor.  See the sticky at the top of this forum for more info.
