### Post by Andrew.Z on 2010-11-23
Samy is clear where evercookie writes to now (both in narrative and source code form) and also gives his plans for the future.  If you want to see the filenames, use [BleachBit to delete evercookies](http://bleachbit.sourceforge.net) including browser cache, browser URL history, browser cookies, and Flash cookies.  Make sure to clean all the browsers at once if you don't want the cookie to resurrect.
