### Post by cariboo on 2009-08-10
This seeming problem has come up so often, I'm surprised you haven't seen any other threads concerning the same thing. Windows .exe files do not run on Ubuntu, so you have nothing to worry about. Anything running in a vm can not effect Linux, as it is contained on it's own virtual hard drive.
