### Post by CharlesA on 2012-01-22
Make sure you don't have any ports forwarded that you didn't do yourself. Turn off uPNP (don't need it).

If you want to see if there are any open ports on the internet side, you can use shields up to make sure all are either closed or "stealth" which they should be.

BTW, "stealth" doesn't really exist, it is the same as closed and something that GRC uses.
