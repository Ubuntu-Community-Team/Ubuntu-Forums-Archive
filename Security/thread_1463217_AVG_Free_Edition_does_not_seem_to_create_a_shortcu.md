### Post by cariboo on 2010-04-26
You should really check with the support forums for third party apps. 

That being said AVG and all the other virus scanners only scan for Windows viruses. There aren't any Linux viruses in the wild. 

AVG is a command line app, that's why it doesn't create a launcher in the menu. You have to start and run it from a terminal.
