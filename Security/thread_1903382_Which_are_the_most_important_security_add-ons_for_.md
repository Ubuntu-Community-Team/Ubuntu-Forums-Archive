### Post by OpSecShellshock on 2012-01-02
Absolutely would suggest NoScript most of all, then AdBlock Plus (and it's changed in the latest version, something about "unobtrusive" ads being allowed now, but there's a check box in the preferences to make those go away as well). I use BetterPrivacy just in case I have Flash objects that I allow. Happens from time to time. Those are the only ones that are probably completely necessary in my opinion. NoScript blocks Flash, Java, Javascript and other things already, so there's no need for anything separate to address those scripts or objects.

I don't personally get anything from WOT, but if you have a search-oriented browsing style it might be beneficial. Standard caveats about reputation services based on user input apply of course.
