### Post by TheFu on 2018-01-29
Welcome to the forums.

17.10 is NOT an LTS release.  16.04 is the current LTS.  Servers should run LTS, unless there is a known hardware issue that requires a non-LTS (newer) release.  17.10 support has about 5 months remaining.

The firewall on Linux is built into the kernel.  iptables is just an interface into the kernel firewall.

Any ports hit can be seen by the firewall, if it is configured for logging.  Look up "IDS" if that is your intention.  The Security subforum here has a number of sticky threads which contain all sorts of good information.
