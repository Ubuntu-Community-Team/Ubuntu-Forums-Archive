### Post by lovinglinux on 2009-06-17
As far as I know, it won't scan your e-mails because none of these anti-virus "for Linux" have real-time protection. They only perform manual scans and they only detect Windows viruses.

If you are worried about contaminating your Linux machine then don't bother using an anti-virus. 

I have BItDefender and it works great. I installed it when I was dual-booting with Windows 7 to scan the Windows partition. Now I just use it when I'm bored, to see if something will ever pop up :)

### Post by Sir Jasper on 2009-06-18
Hi,

Antivir is actually an Avira program and it is consistently rated top or second in independent "windows" tests. It can be made to provide a realtime guard running under Ubuntu with additional installations. However, unless you are using the windows version of Thunderbird there will be a lot less point in using an av program though you may help protect your outgoing emails and thus other windows users.

Avast is consistently rated amongst the Top five in the independent tests and I sometimes use Thunderbird via Wine with Debian Avast on demand scans.
