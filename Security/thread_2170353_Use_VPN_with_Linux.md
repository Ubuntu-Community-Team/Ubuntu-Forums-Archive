### Post by Kevin_Arnold on 2013-08-26
You're already going to have strong encryption on a banking website, so you'd have to see what type of vpn you'd use to see if it will even been any more secure.  

As far as key loggers go, don't download or install stuff you don't trust and it won't be a problem. Malware doesn't generally just magically install itself, someone usually has to click yes at some point in something they shouldn't.  

*back in the day when IE was really not secure, crap could just automatically install with no user intervention at all.

### Post by SeijiSensei on 2013-08-26
SSL provides end-to-end encryption between your browser and the web server.  Adding a VPN won't provide much in the way of added security.  I wouldn't worry about keyloggers unless you have installed software from dicey places instead of relying solely on the Ubuntu repositories or trustworthy sites.
