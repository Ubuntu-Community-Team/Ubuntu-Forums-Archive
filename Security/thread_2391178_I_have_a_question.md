I would imagine whatever was said in 2008 about antivirus is still true today. Very few here have an Antivirus application installed. If it was necessity them I'm sure a suitable application would be in the default Ubuntu installation.

Are you really using Ubuntu 11.10? it's extremely out of date and hasn't been supported since May 2013. You won't be able to install any new software of use an up-to-date version of Firefox, Chromium or Thunderbird so you won't benefit from all the security fixes added to those applications and many others over the years..

Please take a look at [https://wiki.ubuntu.com/Releases](https://wiki.ubuntu.com/Releases). Today, you should only be using Ubuntu 14.04, 16.04, 17.10 or 18.04. No other version is supported. If you computer can't run Ubuntu and you need a lighter distribution then try Ubuntu MATE, Xubuntu or Lubuntu. I would recommend that you start with the 18.04 release of Ubuntu or whatever [flavour]("https://wiki.ubuntu.com/UbuntuFlavors") works best for you.
