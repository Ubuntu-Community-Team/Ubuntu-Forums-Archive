### Post by sudodus on 2014-01-31
The software vulnerabilities are fixed rapidly in linux (and security updates are distributed), but there is no protection for virus in the kernel. The situation has been and is that typical virus are made to attack Windows. But in principle linux is not immune to virus, and it can be attacked. Browser malware is often independent of the operating system.

Usually we say that antivirus programs are not necessary for linux users. The most important way to protect oneself against virus and other malware is to use common sense and be cautious when browsing the internet and downloading files.

### Post by sudodus on 2014-01-31
To be even more secure, you can install a dual boot system with one install reserved for internet banking etc, and another install for more risky tasks.
