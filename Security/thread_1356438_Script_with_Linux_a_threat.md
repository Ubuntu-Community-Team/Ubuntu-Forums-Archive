### Post by cdenley on 2009-12-16
Firefox for linux occasionally has scripting vulnerabilities as well. Your entire system shouldn't be compromised by them, but your home directory, firefox profile, or other web applications you may be using can be. It is still a good idea to use noscript. Also, enabling the apparmor profile may help limit what can be done through a scripting vulnerability.
