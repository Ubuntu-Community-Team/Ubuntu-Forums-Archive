### Post by arkhitekton on 2009-04-10
This is not so much a problem but a fix before you have the problem!

I regularly install the Mcafee SiteAdvisor add-on to Firefox in my Ubuntu installations.  I have just gone to install Siteadvisor from [http://www.siteadvisor.com](http://www.siteadvisor.com) and it offered me the Windows add-on via an .exe file.  I looked around for the Linux download with no luck.  I can only assume that Mcafee has recently changed its policy on Linux add-ons as I was able to download it a week or so ago!

HOWEVER if you are in the same situation do not despair.  Go to:
[http://www.siteadvisor.com/download/mac.html](http://www.siteadvisor.com/download/mac.html)
and accept the Mac OS X download.  This is a Firefox add-on (rather than an executable) and Firefox takes care of installing it.

I hope this helps.
