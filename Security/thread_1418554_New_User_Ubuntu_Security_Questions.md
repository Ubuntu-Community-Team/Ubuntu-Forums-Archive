### Post by davec64 on 2010-02-28
Hi,

1. Any currently supported version, i.e. still receiving security updates.
2. Generally no, for a threat (virus etc) to work in Linux it must have access to the system with root privilges. When you're using linux you are logged in as a specific user without root privilges therfore the potential damage is limited.
3. A firewall if you haven't got a hardware one (i.e. router). Only bother with anti virus if you forward a lot of email to friends with windows. You won't be infected but you might save them some bother!
