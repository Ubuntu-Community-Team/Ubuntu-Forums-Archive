> **KenMac said:**

When you updated, did you use an "alternate" cd, or did you update directly from the 'net?

If you used an "alternate" cd, especially one downloaded from a torrent, you should check the md5sum with the check file from the Ubuntu site to ensure that no one has messed around with the files.

If you've downloaded from the 'net, please post your /etc/apt/sources.list file so that we can check if for any suspicious custom repo's or ppa's.

It is a fallacy that linux will not fall prey to viruses, trojans, spyware, malware etc. For example, if you install a suspicious package using sudo, that package, during the installation, can give itself sudo privileges and hence do whatever it wants.

There doesn't seem to be a recorded or reported case of such a happening, but it's wiser to be alert.

Also, I would suggest a commercial anti-virus suggestion such as Kaspersky, or the free (for personal use) AVG linux edition (or is it avast?, can't recall now).
