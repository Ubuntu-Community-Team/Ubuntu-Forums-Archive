Thanks to the [EFF ]("https://www.eff.org/")for this great website:

[Panopticlick]("https://panopticlick.eff.org/")

Thank goodness for the EFF! What a great organization!

It is surprising how much information can be gleaned from a browser.  You can sometimes uniquely identify a user based on their browser configuration.

Apart from security concerns, [already ]("http://online.wsj.com/article/SB10001424052702304458604577488822667325882.html")people are being guided to more expensive offerings based on their computer.

And [insurance companies]("http://online.wsj.com/article/SB10001424052748704648604575620750998072986.html") are using data gathered from web trackers like a kind of blood test! (Thanks to DuckDuckGo for this info.)

TL;DR: Browser Plugins and System Fonts are big give aways
