### Post by AndyCee on 2009-07-20
Hello fellow Aussie.

You may have passed by, but most of the info you're looking for is [here]("http://ubuntuforums.org/showthread.php?t=510812").

The main thing to understand is that most viruses are written to run on Windows, and these won't run on Ubuntu. I've yet to run into an Ubuntu virus.

Most of the anti-malware security software available to run on linux is designed to scan mail and windows shares.

I won't say it isn't possible to get an Ubuntu virus or trojan, but I've yet to encounter one or meet someone who has got one. Have a look through the security forum to see some of the threats to Ubuntu security.

If you really want, you can get avg 7.5 or clamav for linux. I'd be very surprised if you got one.
