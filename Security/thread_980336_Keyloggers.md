### Post by Dr Small on 2008-11-12
> **Bofur said:**
> Hi, this is more of a question rather than a problem. I was wondering if it were possible for a keylogger to work in linux? Thanks!
Sure it's possible, and I am sure there are some out there, but most likely it would need to be installed by root in order to accomplish anything. If you are worried about someone installing a keylogger on your system, then don't, unless you have a weak password.

A hardware keylogger (before it reaches the system) is always a security risk.
