Haha yes I am mostly worried about my server(perhaps a bit too paranoid).

But a desktop user does need to think about malware.  On Linux you may need to be as worried as on windows(but still cautious).  For the most part try not to visit shady sites or download any files you think are suspicious.  Make sure to get most of your software through the Ubuntu Software Center.  It is a great place.  For other things make sure that you trust the source that it is coming from and you should be fine.

I have run ubuntu for some 3 or so years now and have been malware free.  As long as you practice responsible computing you should be fine :P 

If you notice anything behaving oddly on your system feel free to post here and we will help you out!!
