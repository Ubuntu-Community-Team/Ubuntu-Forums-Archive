### Post by cariboo on 2013-01-12
This sub-forum (Ubuntu Application Development) is for people that are developing programs for Ubuntu, not feature request.

If you want to create a feature request, either create a bug on [https://bugs.launchpad.net](https://bugs.launchpad.net), or create a suggestion on [Brainstorm]("http://brainstorm.ubuntu.com/"), you are going to have to create an account for both sites, before you can enter any info.

Check Brainstorm carefully, as this idea comes up almost weekly from users that have recently switched from windows.

I've moved this thread back to Security Discussions, where you originally began this thread, so that you can take part in a discussion about Linux security, which is done differently from what you know about Windows security.
