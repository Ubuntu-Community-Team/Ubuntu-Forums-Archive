### Post by TheFu on 2021-02-27
You can sign up for extended support. I think 3 devices can be covered for free.
[https://ubuntu.com/blog/ubuntu-16-04-lts-upgrade-vs-esm](https://ubuntu.com/blog/ubuntu-16-04-lts-upgrade-vs-esm)

It is only the default server (non-GUI) and the default desktop (Gnome3-GUI) flavors which get this.  Other flavors lost support after 3 yrs.
