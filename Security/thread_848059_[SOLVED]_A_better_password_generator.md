### Post by Vivaldi Gloria on 2008-07-08
KeePassX is a standalone password manager for linux which is a port of KeePass for windows. It's in the repos.

Unfortunately it doesn't fill the forms in a webpage for you. You can double click the password field to copy the password an then paste on the form. Other than this I don't have any problems with it. It is simple, convenient and reliable.

I think it's much better to use a standalone pwd application rather than a browser extension. You can keep real life pwds in it. It survives even if your browser crashes etc.
