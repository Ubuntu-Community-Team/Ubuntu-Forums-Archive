### Post by Therion on 2009-03-05
If you are using Ubuntu exclusively, you will no longer need Norton Security Suite or anything else like it.

To enable automatic logon in Ubuntu open the Login Window under System/Administration.

Select the "Security" tab and check the box next to "Enable Automatic Login". 

If necessary select the user account to enable automatic login. 

Click "Close" to save the settings.

### Post by bodhi.zazen on 2009-03-05
Ubuntu is not invulnerable to threats, but security is much different.

In Windows I get the impression people install tools like firewalls and antivirus without really understanding why they need those tool or really much under the hood of how security works in windows.

The question I would ask is why do you want to use Norton on Ubuntu ? Once you can answer that question you can then configure the appropriate Linux Tools.

Al too often these threads derail into opinions without understanding what and why we are using security tools.

By default, for a variety of reasons, you simply do not need tools such as Norton on Ubuntu.
