### Post by kevdog on 2009-11-03
You could always default to the command line if need be.  I believe gnupg is installed by default.  

The proper syntax would be:

gpg -d <filename>

and if you needed this saved to another file:

gpg -d <filename> > <new filename>

### Post by quadrantids on 2009-11-03
For some reason it looks like they didn't include Decrypt File in 9.10. Go Applications/Ubuntu Software Centre and install Decrypt File. You should then be able to right-click any document and select Encrypt as in 9.04...
