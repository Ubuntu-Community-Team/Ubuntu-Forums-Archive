### Post by perrabyte on 2009-12-12
The idea is for a dedicated distribution aimed at online banking and shopping. Since there is no competition in this potential market Ubuntu will get free advertising from reporters and experts every time they are asked how to stay safe on the Internet while doing online banking.

This can also be seen as an entry point for windows users when they realize the many ways Linux/Ubuntu can be utilized.

The distribution should kept to a minimal and hardened by default. Minimizing the effect of the number of unknown bugs in the system. It should only contain the required and the security enhancement components. Users should not be able to add programs either which could compromise the system. A reduced size will also make for rapid update particularly for those with a slow Internet connection.

Banks using special software can adapt their security solutions to a single rather than to many alternatives using a simpler and cheaper approach when they know that the risk of a compromised operating system is greatly reduced.
