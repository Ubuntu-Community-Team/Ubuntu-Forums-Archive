Hi, 

Well ubuntu is good option for web surfing (assuming you tweak it a little bit like it's fonts and flash support for video sites). What other applications would you run on this system ?

As far as hackers go, make a common user id with no administrator privileges, That should do it. 

Now hacking a system where you 've physical access to the system needs a couple of things. I'm no hacker but i used to mess around with systems in my free time. Always keep in mind every security system can be broken. Nothing is absolutely safe. Here is what you can do. 

1) Keep BIOS, password protected.
2) Remove the CD/DVD drive. 
3) Live CDs can be booted from USB drive but since it's a old computer i 'm assuming BIOS doesn't support that.
4) If you need even more protection. Encrypt the drive itself.

There is a lot of other things you can do. It all depends on what level of security you need.

One suggestion though since this system will be used as public PC, you might wanna install XP or Vista theme. It helps windows users to ease into Linux environment.
