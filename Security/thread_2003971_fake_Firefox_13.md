### Post by na5h on 2012-06-16
> **Macn said:**
> That does not explain how my 11.10 Ubuntu updated to firefox 13 without any update queries? Firefox 13 is a version 12 Ubuntu update?

You'll find any Firefox updates amongst the other software updates in the update manager. The updates that you've received have probably been in the update manager amongst many other software updates at some point...you've probably just failed to notice them. 

As zombifier25 stated: "The Firefox packaged in Ubuntu is specially modified so that it cannot update itself, and instead let Ubuntu's Update Manager do the job." Hence, there is nothing wrong with the screenshot you posted, Firefox is supposed to look like that in Ubuntu. 

If you don't want to update Firefox in the future, you can simply just uncheck any updates for Firefox that appear in the update manager...but as zombifier25 already stated, the updates are often meant to fix some security related issue and it might, therefore, be quite wise to keep your browser up to date. Is there a particular reason why you don't wish to receive updates for Firefox?

edit: as far as I can tell, your browser is acting and looking exactly the way it is supposed to.
