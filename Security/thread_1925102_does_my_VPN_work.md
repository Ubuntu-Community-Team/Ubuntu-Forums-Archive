Assuming you are using a recent version of standard Ubuntu, you need to click on teh networking icon on the top right and then go down to vpn connections and then click on the vpn you set up.  it should then connect and then show a padlock symbol on the network icon.

### Post by ztux on 2012-02-15
Disable the VPN, click here: [http://www.whatismyip.com/](http://www.whatismyip.com/)

Enable it and click again. If the IP address did not change, the you are not configured correctly or the VPN is not working.
