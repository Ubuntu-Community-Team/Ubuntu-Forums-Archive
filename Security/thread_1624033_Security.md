### Post by Rubi1200 on 2010-11-17
Just to add to what CharlesA has already said:

1. use strong passwords

2. only download and use software from the Ubuntu repositories

3. keep the system updated

4. use common sense

If you do all that, then I think it is fair to say that Ubuntu (and Linux in general) is about as safe as it will get.
