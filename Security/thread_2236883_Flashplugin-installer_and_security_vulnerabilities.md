### Post by tgalati4 on 2014-07-29
Install *Ad Block Edge* and *noscript* as add-ons for Firefox and you should be fine.  The flashplayer is needed to watch some youtube videos and some webpages that use flash banners.  Otherwise, I am not aware of any issues with flash being used as a virus vector (carrier) under linux.
