### Post by laeg_ on 2010-08-18
I had some help via email from someone drafting my CV into the correct table format with open office. It's a .pdf file but now unfortunately lists the author in the document tab of properties as that person.

Is there anyway to change it to my own name, and also how do I 'secure' the document so that it's not easy for people viewing it to copy and paste, I've heard this is why many people now use .pdf for their CVs/résumés?

### Post by labinnsw on 2010-08-19
> **laeg_ said:**
> Is there anyway to change it to my own name,Not sure. If you are using Open Office, try using the pdf button in the tool bar to create a new pdf with a different name.  

 > **laeg_ said:**
> and also how do I 'secure' the document so that it's not easy for people viewing it to copy and paste, I've heard this is why many people now use .pdf for their CVs/résumés?Again in Open Office,
1. File >> Export as PDF.
2. Under the security tab Set Permissions Password
3. Click Ok then Export
4. Under "Changes," select "Not Permitted"
5. Uncheck the "Enable Copying of Content"
6, Export and name the file.

Another tool for the job is PDFTK builder.
