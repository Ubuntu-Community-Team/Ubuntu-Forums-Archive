### Post by mfarshada on 2014-12-31
I have the same question. I could not find any source package through [http://repository.spotify.com](http://repository.spotify.com) and I don't think it is published anywhere else, although having an open source does not guarantee it is safe.
If you are concerned about security, you can either use the web-based client or run it under wine. But that way you would not be encouraging spotify to officially support linux. So it's a compromise.

I also stumbled upon an open source client [http://despotify.sourceforge.net/](http://despotify.sourceforge.net/)
I am not sure if it is maintained or working, or even safe!
