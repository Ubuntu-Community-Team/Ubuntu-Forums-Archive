### Post by OpenSourceFuture on 2008-10-28
> **DracoNyon said:**
> Okay so I just got Ubuntu and stuff and no absolutely nothing about security on it. One, would any of these virus's affect Linux. [http://www.switched.com/2008/10/28/14-sneakiest-computer-viruses/?icid=100214839x1211991148x1200713721](http://www.switched.com/2008/10/28/14-sneakiest-computer-viruses/?icid=100214839x1211991148x1200713721) 

The only things Linux is vulnerable of is exploits, which should be few are far between, and code you run yourself. Since it is more difficult to run things as a privileged user in Linux you should be perfectly safe unless you install something malicious which would take quite a bit on your part but isn't impossible.

> **DracoNyon said:**
> 
What would happen if one got on it? Nothing? Is there a way to get rid of any bad virus/trojan/worm, etc. Can a virus physically make a computer not work? like could I just put another OS on it(since it reformats everything,right?) and will nothing be wrong?

The odds of you getting a virus that physically harms your computer are extremely extremely extremely low, in theory they could write to your hard drive until it failed, things like that, but its nothing you need to worry about, also there is very low probability your going to be getting a virus in Linux, you may get a root kit, but thats a different story, and also very unlikely. You could always reformat, use live CDs, etc, but if you only install things from the ubuntu repository I wouldn't worry at all if I were you.
