### Post by sudodus on 2014-07-05
I'd say you need not even use encrypted home, because Windows cannot read the ext4 file system of Ubuntu. But if you want that extra layer of security, yes, use encrypted home.

But ***you have to remember your password***. We can not help you read your home directory, if you forget the password. And it is more important to ***take regular backups***, when using encrypted home.

See also this link

[https://wiki.ubuntu.com/BasicSecurity](https://wiki.ubuntu.com/BasicSecurity)
