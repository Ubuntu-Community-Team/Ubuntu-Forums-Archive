### Post by Tatty on 2009-02-11
The anti-virus software in the repos only really scans for windows viruses.  Its meant for if you want to check a file before sending it to a windows machine.

It is still possible to hack into a linux machine, so firewalls exist to protect you from the outside world.  However, you only really need them in you are intentionaly installing server apps, as in a default ubuntu install, nothing is listening for incoming traffic so it can do no harm.

I have heard lots of different figures for the number of linux viruses.  The long and short of it is that it is possible to make a linux virus, but extremely hard.  And viruses which do appear get patched out of the kernel very quickly.  So no need to panic.

### Post by bodhi.zazen on 2009-02-11
Most people who use Av software either share files with windows and wish (for some reason) to scan them from linux or run mail or other file servers (samba) with windows clients.

The firewall in Linux is iptables and is packages you are looking at are to assist in configuration.

The question should be, what do you want to use a firewall for ? they have lots of uses.
