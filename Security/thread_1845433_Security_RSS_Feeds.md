### Post by josephmills on 2011-09-17
ok go to [http://www.getmantra.com/download/index.html](http://www.getmantra.com/download/index.html) 
and download the one that pertains to you then

open terminal 

change dir to the dir that you downloaded Mantra. In my case I DL it into my Downloads dir 

```
cd ~/Downloads 
```Then extract Mantra 
```
 sudo tar -xvf Mantra\ Security\ Toolkit\ -\ Gandiva\ xXX>XX.tar.bz2 

```Note that XX-XX is going to pertain to the one that you downloaded 


[IMG]http://img217.imageshack.us/img217/6500/mantratut.png[/IMG]



Next change dir into the Mantra file 
```
cd Mantra\ Security\ Toolkit\ -\ Gandiva\ x86_64/
```then start Mantra 

```
./Mantra\ Security\ Toolkit\ -\ Gandiva    
```
[img]http://img545.imageshack.us/img545/9388/mantratut1.png  [/img]



now go to the **bookmarks-->hackery-->exploits and advisory **
