### Post by tgalati4 on 2015-04-06
Welcome to the forums.  It's possible to modify any hardware--with enough time and money.  Your cellular network dongle will either work or not with your network, so that is the first test.  If it does not work, then it might be a fake.

If it does work, then there is little you can do to determine if it is a safe device to use because of the nature of cellular networks.  You can use *etherape* or *wireshark* to inspect network connections and packets, but there is little you can do to inspect what else the modem is doing--like capturing keystrokes and forwarding them to a server.

Before using it with sensitive data, you can bring up the configuration web page and check for software updates.  At least get the latest firmware direct from the manufacturer.
