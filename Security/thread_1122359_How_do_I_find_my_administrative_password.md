### Post by cariboo on 2009-04-11
If you don't remember the password you setup when you installed Ubuntu, you can start up in recovery mode and once you get to the menu select drop to root prompt. Once at the root prompt type:

```
passwd <username>
```

where <username> is your user name without the brackets.
