### Post by lone_wolfII on 2009-02-18
_Currently_ downadup doesn't actually do much other than update itself and check to see if there's any new instructions on some remote server. 
For now you should be safe as long as Windows is kept up to date. 

XP is the most vulnerable to this virus. When you plug in a USB drive, make sure to open it normally through My Computer, rather than in the dialog box that comes up.
