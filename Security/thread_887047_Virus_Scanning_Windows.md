### Post by lisati on 2008-08-11
There are a number of options, including avast, clamav, and AVG free.

My preference is for [AVG free]("http://free.grisoft.com") - if you choose this one, remember to add your normal user name to the avg user group (as mentioned in the AVG documentation), otherwise you'll have to mess around with (gk)sudo to do the updates.
