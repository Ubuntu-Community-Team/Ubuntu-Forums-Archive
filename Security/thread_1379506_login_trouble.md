### Post by garryknight on 2010-01-12
In Kubuntu, passwords are usually stored ni KWallet. She could try running KWalletManager (in the System menu) and clicking the wallet in the window. Then she goes down the list looking for a stored password for the widget (whatever that's called). If she finds it, she should be able to type a replacement password then click Save.
