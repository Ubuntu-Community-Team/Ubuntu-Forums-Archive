### Post by cdenley on 2008-06-05
PFSense is basically FreeBSD preconfigured to be a firewall/router with a web interface. It isn't a piece of software you add to an existing software, it is an operating system itself. If there is a linux version, it would be a completely different product using iptables instead of OpenBSD's packet filter. If I remember correctly, the installation was pretty straightforward, but if you need a tutorial, you can check their website.
[http://doc.pfsense.org/index.php/Tutorials](http://doc.pfsense.org/index.php/Tutorials)
