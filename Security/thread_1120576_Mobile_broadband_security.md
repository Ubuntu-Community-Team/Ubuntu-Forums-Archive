### Post by cdenley on 2009-04-09
You should ALWAYS assume someone may be intercepting your traffic. This is especially true with unencrypted wireless connections, though. If you are using SSL (https), firefox shows a padlock, you're sure you are using the correct site, you know you can trust the site, and you aren't given any warnings about the SSL certificate, your traffic should be safe from eavesdroppers.
