### Post by JohnReimer on 2011-05-25
Hi,

I am running Ubuntu 11.04, and I have installed some games I found on the Ubuntu Software Center.  Out of the hundreds of installed things, almost all are from the category "Provided by Ubuntu".  The only ones not in that category are: Skype, Google Chrome, Teamviewer, and a Skype recorder I downloaded from [http://atdot.ch/scr/](http://atdot.ch/scr/)

Question: If the program is in the "Provided by Ubuntu" category does that mean it is already verified safe?  If true, then the only thing I would need to worry about is the Skype call recorder, which has a website, but I do not know how to tell if it is safe.

Any suggestions or comments greatly appreciated

Thanks!

### Post by EmporerD on 2011-05-26
If it's in the provided by ubuntu category you can safely assume that they're ok. As for other programs, as long as they are reasonably well established in the community they're probably fine as long as you get it from a trusted source.

 Just don't go around loading random programs that no one has ever heard of and you should be fine. If you're not sure about something then google it. Most malware programs have some sort of "stay away" post in a forum somewhere. If you can't find anything at all about it, you should still be cautious and ask on a forum before installing it.
