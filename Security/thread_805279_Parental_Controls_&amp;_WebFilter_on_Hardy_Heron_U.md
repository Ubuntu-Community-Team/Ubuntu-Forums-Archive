### Post by coolaj86 on 2008-05-23
I just snagged an old (Feisty) copy of [Ubuntu CE]("www.whatwouldjesusdownload.com/christianubuntu/")'s [Parental Controls]("http://www.whatwouldjesusdownload.com/christianubuntu/2007/05/features.html") (uses dansguardian, tinyproxy, and firehol) from [softpedia]("http://linux.softpedia.com/get/Security/Parental-control-24501.shtml") (see their nice screenshot) and made some changes to the script so that it will install in Hardy.

I just tested it in a VM on a nearly fresh install of Hardy and it worked without a hiccup. The default configuration is pretty decent for blocking dirty words on google searches at least. I don't generally do the type of browsing that lends itself to putting "STOP" messages in my face.

Once installed: 
System > Administration > Configure Parental Controls

Enjoy.

[http://rapidshare.com/files/117145054/parental_control.tar.gz.html](http://rapidshare.com/files/117145054/parental_control.tar.gz.html)

RapidShare only allows one download at a time so if you have a webserver please put this in your /pub and post a link.
