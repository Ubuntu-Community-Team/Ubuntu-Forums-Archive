### Post by YyYo on 2009-05-08
Hi All...

I have started developing a GUI tool for wrapping some of the common scripts/programs of SELinux.
The tool is called **SEAdmin**, and it should be general to all Linux distributions which SELinux is installed (currently working under Ubuntu 8.04 and Fedora 10)

You can view some screenshots in the site [https://sourceforge.net/projects/seadmin](https://sourceforge.net/projects/seadmin) 
or to search "seadmin" in sourceforge.net site

Hi,

A new graphical management application for SELinux called SEAdmin has been [announced]("https://lists.ubuntu.com/archives/ubuntu-hardened/2009-May/000482.html") on the [ubuntu-hardened]("https://lists.ubuntu.com/mailman/listinfo/ubuntu-hardened") mailing list.
