### Post by cariboo on 2008-10-10
The chances of Vista being infected are far higher then ever getting a virus in Linux. There are no Linux viruses in the wild so far. Most of the Linux viruses are proof of concept type programs and never make it very far in the wild.

As it stands right now the only way a virus would work on your computer is if you install it yourself. You may have noticed that any time you install a program have to enter your password before it will be installed. A virus would need your password before it could be installed too.

The best way to stay away from malware is to only install software from trusted sources, and always question any commands you see, even in the forums.
