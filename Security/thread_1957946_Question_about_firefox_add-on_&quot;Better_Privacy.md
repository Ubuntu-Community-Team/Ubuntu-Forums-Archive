### Post by OpSecShellshock on 2012-04-13
I think the default settings have it run any time the browser closes (which is good) so that's why it ran. The reason it didn't find a default folder is probably because you hadn't actually run any Flash objects during any browser sessions since the time you installed it. The folder it's looking for won't exist until you do. That's why the message came up.
