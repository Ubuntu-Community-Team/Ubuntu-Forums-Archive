### Post by technotitclan on 2008-11-06
Ubuntu is very secure. there are only a handful of infections for linux and i believe a fix for them is already built into the kernel. Ubuntu by defauld doesn't run as an admin like windows does, so if you do get the rare infection that targets linux then it will never be able to run because it has to open the terminal and install itself w/ the sudo command and then enter your personal password to get installed. as far as being hacked. it doesn't happen as much as everyone likes to think but is alwas a posibility regaurdless of os. Ubuntu has a firewall you can install in the terminal called Firestarter. i think the cmd is   sudo apt get-install firestarter   but i'm prolly wrong, google it. you really don't NEED a antivirus but if you plan on media sharing w/ a windows system then try avast. its one of the few that work on linux. thats all i got for you. if you have any questions please post. hope that helps
