### Post by TheFu on 2013-09-25
Well, with the limited information you've provided, it seems like you've been hacked. Time to format.  

Does someone else have physical access to the machine?  If they do, it is time to start increasing boot security and encrypting as much of the HDD as possible.

If this is just network attacks, there are simple things to prevent that sort of stuff - like never connecting any computer directly to the internet, always stay behind a router with every port filtered until you've hardened the system. Then only open the required 1 or 2 ports for the specific service needed.  See my signature for other ideas.
