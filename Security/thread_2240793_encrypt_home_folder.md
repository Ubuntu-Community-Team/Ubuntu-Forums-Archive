### Post by uRock on 2014-08-22
Encrypting the /home is mostly for protecting your system from people who have physical access. If it isn't a laptop, then I wouldn't bother with encrypting the whole system or the /home.

Personally, I prefer using an encrypted private folder for things I don't want to be found. You can do that by following the directions here [https://help.ubuntu.com/community/EncryptedHome](https://help.ubuntu.com/community/EncryptedHome)
