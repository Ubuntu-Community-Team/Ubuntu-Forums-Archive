> **nouar.garcia@sun.com said:**

Ubuntu Gutsy has reached end of life. You can use [http://old-releases.ubuntu.com/](http://old-releases.ubuntu.com/) to get software, but you won't get any more security updates for it.

You can upgrade to Ubuntu Hardy, which is a Long Term Support release : [https://help.ubuntu.com/community/HardyUpgrades](https://help.ubuntu.com/community/HardyUpgrades)
