### Post by cariboo on 2011-09-25
By default all Ubuntu versions require a password to log on, and from Natty forward, the screen lock is automatically activated after 5 minutes of inactivity. If you have enabled autologin, disable it, and create a new password in Users & Groups, and re-enable auto screen locking.
