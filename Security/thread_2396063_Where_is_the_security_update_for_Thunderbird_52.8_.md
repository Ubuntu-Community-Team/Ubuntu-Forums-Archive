### Post by PaulW2U on 2018-07-10
Wadcutter, I don't work for Canonical but I do know that Thunderbird 52.9 is being worked on. 

After any Ubuntu specific patches have been applied Thunderbird, like Firefox and Chromium, has to successfully build on all architectures for all of the currently supported releases. There are are often build failures which hold up the process. After a short period of testing it will be released but the entire process could take a couple of weeks. 

Like we say every Ubuntu release day: "It'll be released when it's ready".  :)
