### Post by agillator on 2012-02-22
Muramasa, let me add my welcome to Linux and Ubuntu to the list. One note of explanation about firewalls. You already have an excellent firewall available - it is called iptables. Most any linux firewall you hear mentioned will be a frontend for iptables. Some are simpler than others, some more complex. Many if not most use them because they are simpler than using iptables directly. So if you, like most, use one of the front ends and then run into a situation it can't handle, there is always the possibility of loading a rule directly yourself that will handle the situation. I know it removes some of the luster and 'magic' from firewalls, but it is nice to know you are free to do it your own way if you want or need to.
