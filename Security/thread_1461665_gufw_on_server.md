### Post by uberlinuxnerd on 2010-04-25
go here: [http://www.whatsmyip.org/ports/](http://www.whatsmyip.org/ports/)

Run a basic port scan. If everything is closed your probably pretty good. The only port I have open on my gateway is SSH (Which I use to mount sshfs folders at work).

With computer security there is no completely safe 'spot' short of pulling out your Ethernet connection and locking your computer in a room with no *windows *(take that what ever way you want) 

What is more important is practicing safe computing. E.g. FTP with anonymous upload access is generally speaking asking for trouble. And if you would like SSH access use keys and turn passwords off.

I hope this helps you understand IT security. :)
