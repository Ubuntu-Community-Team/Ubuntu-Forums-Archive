### Post by cariboo on 2015-06-06
Part of getting the hang of things, is realizing that Ubuntu is not Windows, Generally the only time you need an anti-virus is if you are sharing files with Windows users. There are virtually no Linux viruses in the wild, so the majority of virus scanners only check for Windows viruses. 

If you are only running Ubuntu, don't worry about needing any anti-malware programs, if you are running a dual boot system use the anti-malware programs you have installed on your Windows partition as Windows can not access your Ubuntu partition without the help of third party programs.

As an aside, I do have a few Windows viruses in my email archive, most of them have been there for over 10 years, and they still haven't had any affect on any of my Ubuntu installs.
