### Post by Rubi1200 on 2010-10-31
You almost certainly do not have a virus or anything else!!!

There are some nasty things going around but these do not, generally speaking, affect Ubuntu because of the security model in Linux systems.

Clear the cache of cookies and history and restart Firefox.

Install the Firefox addon, NoScript:
[https://addons.mozilla.org/en-US/firefox/addon/722/](https://addons.mozilla.org/en-US/firefox/addon/722/)

If the problem persists, you may have to consider changing your DNS servers.

See also here for more security information:
[http://ubuntuforums.org/showthread.php?t=671604](http://ubuntuforums.org/showthread.php?t=671604)
[http://ubuntuforums.org/showthread.php?t=510812](http://ubuntuforums.org/showthread.php?t=510812)
