### Post by cdenley on 2010-01-28
It might help if you explain what kind of anonymity you want to accomplish, or what the tools you're looking to replace actually did. I doubt you're going to find a free proxy service that works faster than TOR. Bandwidth costs money. I know I've heard of paid SSH access which lets you tunnel your traffic through an SSH connection.

Google Chrome does support a proxy configuration, so you can use it with TOR or any other proxy service.
