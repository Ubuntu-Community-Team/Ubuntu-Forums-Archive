### Post by kurt18947 on 2012-06-16
In addition to what Ms. Daisy says, I just created a full install - not a live USB - on a 16 GB. USB drive.  My intention is to use that ONLY for sites that are sensitive such as banking, tax, and investment sites.  I think a full install on a USB drive would be too slow for everday use but for relatively infrequent and simple transactions I think it'll be usable.  I think the advantage of a 'real' install is that I can do security updates on a USB full install, I don't think I can install security updates on a Live USB;  I tried it once and seemed to damage the Live USB install.   I can also store "I've been here before and I am who I claim to be" cookies from the institutions.  Not sure if that's good or bad but having to authenticate myself each time before a normal log-on gets tiresome.    I'm certain this is not a be-all end-all solution but it's another step in the security process.
