### Post by Ms. Daisy on 2012-01-15
You should note: [QUOTE=https://help.ubuntu.com/community/Firestarter] Although Firstarter is fully functional, active development ended in  2005 with version 1.0.3. For an overview of alternative applications,  see [Firewall]("https://help.ubuntu.com/community/Firewall").  [/QUOTE]

If you use a program that many have abandoned, then the pool of people that can help you may be limited.  Something to consider.
