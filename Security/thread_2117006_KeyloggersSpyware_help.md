### Post by sudodus on 2013-02-17
It is one thing to wipe the HDD so that no trace will be left for the secret service of superpower countries (when they get physical access to the computer).

It is another thing to wipe the HDD, so that all the files will be gone, and not available or active. This is much simpler, and I would say that it is enough to reformat the drive. If you want to be even more sure about it, zeroise the drive too, but I would say it is not necessary in the case you describe. When all the files are gone, no spyware will remain.
