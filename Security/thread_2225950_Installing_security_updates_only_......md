### Post by Chayak on 2014-05-24
You could turn off update for a month and be OK as long as you're not running any servers.  I've been in bandwidth restrictive environments before with BGAN satellite and operated without doing updates.  If you have the firewall turned and don't have any server processes the risk will be minimal.  You can still check for any critical updates and do them manually if you wish.
