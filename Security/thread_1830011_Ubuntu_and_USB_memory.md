### Post by wacky_sung on 2011-08-21
It depend on the type of malwares and how the code get execute. Most malwares are window based but there is an increase of malwares which work on Macs, Window and Linux OS as well. Most malwares execute through a vulnerability and mostly targeted at the browser. Thus overcome all those browser based malwares are to clear browser cookies, enforce with apparmor, clear system temp files, etc. Running a system with a proxy server and privoxy help to filter out those issues much faster and effective.
