### Post by cogset on 2014-09-06
> **linuxyogi said:**
> Hi,

I have seen that installing Adblock plus slows down both page loading time and the browser.(...)


That may depend from which adblock lists you've subscribed and whether the "allow some non intrusive advertsing" checkbox is enabled (it lets in some very obnoxious stuff):you can pick a different list here [https://adblockplus.org/en/subscriptions](https://adblockplus.org/en/subscriptions) and/or make your custom rules with Adblock-speaking of which,this extension  [https://addons.mozilla.org/en-US/firefox/addon/elemhidehelper/](https://addons.mozilla.org/en-US/firefox/addon/elemhidehelper/) can be very helpful to add custom filters.
I don't think you can get your profile messed up if ads just show up (to run some possibly malicious code you still will have to click on them,hopefully..) but ads and other page elements such as social widgets,web bugs and some 1x1 gif images will track you,so it's better to block as much of this stuff as possible (IMHO).
