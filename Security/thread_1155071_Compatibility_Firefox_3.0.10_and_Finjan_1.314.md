### Post by lovinglinux on 2009-05-10
I personally don't like extensions that monitors my web activity, even if their intentions are good. Besides Firefox is already integrated with a similar solution provided by Google. I don't know if it is effective tho.

I would recommend installing [NoScript](https://addons.mozilla.org/en-US/firefox/addon/722) addon instead. Never had any problems with it. I think its a better security tool,  because it blocks malicious content, regardless if the site is on a database of malicious web sites or not. So if you visit a malicious web site that is not listed in the database yet, you are still protected by NoScript. You can whitelist only trusted web sites, so it doesn't block their content. Additionally, you could also install [AdBlock Plus](https://addons.mozilla.org/en-US/firefox/addon/1865), to block ads and other stuff.
