### Post by bodhi.zazen on 2010-09-26
In the "Ubuntu Software Center "...


Go to Edit -> Software sources

In the Updates tab you can select how often to check for updates and you can elect to install Security updates without confirmation.

See also :

[https://help.ubuntu.com/community/AutomaticSecurityUpdates](https://help.ubuntu.com/community/AutomaticSecurityUpdates)
