### Post by fenugreek on 2008-09-10
I'v got 2 Hardy Heron Ubuntu installations on my disk n Windows XP. On the beginning if i wanted to mount another partition (Win or Ubuntu) in Ubuntu using standard file manager, i'v been asked for authorization. Then i'v changed that option to auto authorization. I didn't know that  i wouldn't be able to set authorization on mounting other partitions again. In this way my other partitions r not safe (w/out authorization access).
  My goal is to hv one Ubuntu 100% virus n other infections free, so I want hv one Ubuntu partition completely unaccesable by still possible malicious software on the other Ubuntu partition.

### Post by fenugreek on 2008-09-10
Sorry for bothering but I'v found simple answer:
System->Administration->Authorization->Storage->Mount..internal drives
Explicite authorization
n [revoke] listed authorization.
Then restart.
Maybe it will be helpful for some newbies like me. :)
