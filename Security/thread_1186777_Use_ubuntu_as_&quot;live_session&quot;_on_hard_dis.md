### Post by Sarai the Geek on 2009-06-13
I'm pretty sure this is possible exactly as you specified, but in case it isn't...

Create a USB boot drive- then you can just plug it in when you want to use your secure live session. Make sure that when you are making it non-persistant, so that it only lives in the RAM which is then erased when you reboot.
