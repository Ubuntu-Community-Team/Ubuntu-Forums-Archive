### Post by doas777 on 2008-10-25
well, as close as i can tell, there really isn;t a good way to do that, no matter what OS you are on, unless you use a proxy network like TOR. keep in mind though, it's extremely rude to run large downloads through TOR. 

[I2P]("http://en.wikipedia.org/wiki/I2P") looks really promising for BT anonominity, but there isn't sufficient acceptance of the protocol yet to make it useful. 

you prolly want to check out [moblock]("https://help.ubuntu.com/community/MoBlock") and [mobloquer]("http://ubuntuforums.org/showthread.php?t=703381") for an peerguardian-like blacklist blocker. 

If you learn anything good in terms of BT anonymity, I'd take it as a favor if you post back here, to let us in on it.

have fun,
Franklin
