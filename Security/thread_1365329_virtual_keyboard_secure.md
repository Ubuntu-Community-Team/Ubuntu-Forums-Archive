### Post by BkkBonanza on 2009-12-27
I think it likely falls into the category of security by obscurity. Maybe right now malware doesn't trap the mouse like it does the keyboard but it's certainly possible to do. Trap the mouse clicks and analyze position. The screen keyboard would have to always randomize it's keys to combat that. And then how does it enter the info into the application - it may be filterable there. There are many possible variations and I haven't read any study done on this so who really knows.

### Post by Jason.TJ.Johnson on 2011-07-08
I would imagine that the software is probably more at risk than the actual hardware. Assuming you didn't have any keyloggers installed, I'd imagine all you would have to do it make certain your physical keyboard wasn't switched with another one which included a hardware-based keylogger. It would require someone to have physical access to your keyboard without your presence while on the internet all it would take is some creative attacks and a bad mouse-click or two. Theoretically.
