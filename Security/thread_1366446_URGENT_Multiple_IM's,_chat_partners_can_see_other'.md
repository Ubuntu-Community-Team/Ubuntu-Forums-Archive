### Post by cariboo on 2009-12-30
There is a remote desktop client and server install by default, If you are talking to your boss again before you get back from holidays, Get him to go to System-->Preferences-->Remote Desktop and under sharing make sure **Allow other users to view your desktop** is unchecked.
