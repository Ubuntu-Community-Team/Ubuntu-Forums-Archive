### Post by Pjotr123 on 2011-10-24
A couple of days ago, Oracle has issued a new version of Oracle (Sun) Java JRE: 6u29. Version 29 contains important security updates: [http://www.oracle.com/technetwork/topics/security/javacpuoct2011-443431.html](http://www.oracle.com/technetwork/topics/security/javacpuoct2011-443431.html)

If you have a manually installed Oracle Java, you should update as soon as possible. In Ubuntu 11.10, Oracle Java is no longer in the repositories, so that means installing the new version by hand. I've updated my how-to for that: [http://sites.google.com/site/easylinuxtipsproject/java](http://sites.google.com/site/easylinuxtipsproject/java)

**Note:** this only applies to those who have (for one reason or another) switched from the default OpenJDK to Oracle Java JRE.
