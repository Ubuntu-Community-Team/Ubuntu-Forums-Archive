### Post by Duncan Williams on 2011-05-16
I have had serious malware issues over the last year using windows which is one of the reasons I switched to ubuntu.

I am not talking about `scams to buy software' or being diverted to a different website. 
Iv'e had bandwidth stolen (ie/ someone using my connection to access internet while I am online.) system failure, loss of data, software malfunctioning, unable to use modem/printer, etc.
And more. All it takes is one good rootkit to take hold and it's like watching dominoes go down.

OK, that was windows, but the same things can happen in linux. the biggest 2 safeguards are 
1. root user approval required (stops most malware from executing)
2. Linux community and developers (who plug the gaps as soon as they appear)

There are still issues with privacy and tracking, etc.
I don't use anti-virus or firewall as there is no real point once it is looked into.
I do however secure nearly all my data in the cloud and use password managers and other security features (as in browser extensions) I also keep confidential information secured and not accessible by others.

My experiences bring me to having a system that can be stolen or fail and I can still bring it all back up from my spare hard drive, a few cd's/usb sticks and cloud based servers.
Whereas the system that was stolen is useless to anyone else as it's all secured.
