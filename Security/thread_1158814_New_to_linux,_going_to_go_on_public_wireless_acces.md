### Post by The Cog on 2009-05-16
> **utnubuuser said:**
> Ubuntu has a built-in firewall. 
True but misleading. In a default installtation, the firewall is not blocking anything. 
>  If you've got a default installation, you're (reasonably) safe.
True. Not because of the firewall, but because a default install has no listening ports that can be a target for attacks. This is why there's no need to configure the firewall to block ports that shouldn't be listening for incoming calls by default in the first place.
