### Post by jbrice on 2011-02-27
Possibly the simplest thing to do is to print your PDF file using "Print to PDF". That should retain the content but create a new PDF file with a mainly blank set of metadata.

If you haven't yet set up a PDF printer on Ubuntu, there are lots of how-tos on the web - e.g. [http://hubpages.com/hub/Print-to-PDF-in-Ubuntu-904]("http://hubpages.com/hub/Print-to-PDF-in-Ubuntu-904")
