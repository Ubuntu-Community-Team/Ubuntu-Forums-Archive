### Post by jerome1232 on 2009-10-19
The packages in synaptic are signed by the repository owners, if your system ever finds a package that is not signed by the owners key, it will alert you that the package isn't authenticated and may not be from the actual repository owner.

All software in the repositories is tested and such to ensure it works with the current release and that it is malware and etc... free.
