### Post by flaymond on 2015-04-05
I got a few tips -

1. Don't install/use untrusted and free proxy while browsing. Use paid VPN instead or don't use them at all.

2. Don't install suspicious add-ons.

3. Always check the links/address and always check if the website is secured. (using https)

4. Don't click any suspicious links or pictures and even ads! (I find some of them are phishing)

5. Delete history, cookies and cache (if you need to)

6. If you paranoid to some search on Google, use other search engine that like DuckDuckGo or Startpage instead.

7. Disable JavaScript, Java and Flash (Would ***not  *** recommend this because some of the websites cannot function properly without them enabled)

As well, probably use Linux will securing (slightly) in security, but still, you not safe. Phishing is still dengerous and effective in stealing critical informations.

### Post by Bucky Ball on 2015-04-05
*Thread moved to **Security**.*

Welcome. You are asking two different support questions here so we'll deal with the first one for now. As for changing how Firefox looks, that is for another thread, so feel free to edit your first post and post a new thread regarding that. ;) As for internet banking ...

If you want to be very cautious, keep the install as 'vanilla' as possible, i.e. don't install third-party PPAs. If you want to be over-the-top about it, best is to boot Ubuntu from a Live USB/disk and do your banking. When finished, remove the Live media.

I take precautions, but nothing severe, and have been using the internet for banking for years without issue. I run NoScript, AdBlock Plus, kill cookies and cache after each session and don't let the system save important passwords (and also have Java and Flash set to 'Ask Me' rather than 'Automatic').

### Post by buzzingrobot on 2015-04-05
Just in case, though I suspect it's not needed:

Reputable banks, etc., don't send email with links for customers to click.  They tell customers to log in at their site and check for messages. Mail with links is either a phishing attempt or an indication it is time to change banks.

Two-factor authentication to log on or to actually move funds out of an account ought to be there, as well.

Bookmark the site you use. Don't rely on your fingers getting the URL right.  Plenty of folks exploit typoed domain names.
