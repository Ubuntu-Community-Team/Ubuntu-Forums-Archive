For Kopete users, see [http://kopeteantispam.sourceforge.net/](http://kopeteantispam.sourceforge.net/) - a nice plugin that helps a bit.  I have written a patch for this that prevents anyone not on your contacts list from messaging you - it's available [here]("https://sourceforge.net/tracker/index.php?func=detail&aid=2333222&group_id=229210&atid=1076376") but this of course does require you to recompile the package, which is not that easy. 

I also have a .deb that was created for Ubuntu 7.10 in a quick and dirty fashion using Checkinstall, but does mean you can easily install this with "dpkg -i".  PM me if you'd like a copy of this.
