### Post by OpSecShellshock on 2010-10-18
If you're not noticing any further problems after rebooting, then you're probably fine and you can just delete it. Looks like the plan with the fake video is to get unsuspecting users to download and install the trojan (which I assume is what "coreplayer" is) from the linked site. Of course, the best thing to do when a file is obviously fake is not to run it. This kind of thing is fairly common with media content files, and is most likely made for Windows users.
