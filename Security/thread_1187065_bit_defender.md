### Post by rookcifer on 2009-06-17
Ah, here we go with the "Linux will get viruses the more popular it gets" argument.  Those who say Linux is just as vulnerable to malware as, say, a default Windows XP install are, to put it nicely, simply misinformed.  In short, the reason this is true has to do with the Unix DAC.  Basically, the best that the malware authors could hope for is that a small percentage of new Linux users fall for social engineering attacks.  

Bottom line: To avoid any trouble:

A) Stick to the Ubuntu repos for your software

B) Don't execute random scripts that you download (if you're new, you probably wont know how to make them executable in the first place.  Moreover, you should ask yourself why you are downloading random scripts in the first place). 

Follow those two easy steps and you wont ever get a "virus."  Besides, even if none of the above applied, these so-called "Linux virus" scanning tools only scan for Windows malware.  They exist primarily as a courtesy for Linux servers that serve Windows boxes.  And Avast and all the others are only jumping into the fray hoping gullible Windows converts will buy their products for Linux since that's what they did on Windows.
