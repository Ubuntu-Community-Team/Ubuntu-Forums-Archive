### Post by mcduck on 2009-08-10
It's not a security risk, and the only danger is that Nautilus running as root allows you to delete any system files so you have to be very careful to avoid accidentally removing wrong files etc.

As long as you only run Nautilus as root when you really need to, and close it afterwards, you are safe. I also prefer changing root's Nautilus to have a red background to make sure I'll never confuse it with Nautilus running as my normal user.
