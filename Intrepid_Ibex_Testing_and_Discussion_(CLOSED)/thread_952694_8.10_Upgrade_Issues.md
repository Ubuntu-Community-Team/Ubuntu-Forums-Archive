### Post by cariboo on 2008-10-20
To cd into a directory with a space in the name, just put quote marks around the name eg:

```
cd "space in name"
```

or if you're lazy like me:

```
cd spac*
```

This assumes you don't have more than one directory named "space in name"

For hidden directories:

```
cd .dirname
```
