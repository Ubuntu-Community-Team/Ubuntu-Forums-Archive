### Post by wolfie2x on 2008-10-28
I don't think there's a way to get all those options in a single window like in Hardy. Some usability expert had figured it's better not to have it all together I guess.. (I liked it all together though)..

You'll have to add a separate shutdown button to the panel via Right Click (on panel) > Add To panel, serach for "shutdown" and add it.
Or if on a laptop, the physical shutdown button brings it up.
Or you'll just have to get used to using System > Shutdown.
