### Post by mejy on 2008-10-16

That we've been promised a new theme for every release since Dapper is starting to get a bit tiresome - I can appreciate that a theme can be a lot of work, but in that case I'd have thought not announcing or pledging anything until there's something substantial to be included would appear to be the logical policy.

The dust theme and foxtrot icons, IMO, look infinitely better than what we have as default for the moment - if Canonical don't have the resources to create something innovative for the release themselves, I don't see why they don't put some weight behind this sort of community effort and give it the final burst of spit and polish to reach release quality.
