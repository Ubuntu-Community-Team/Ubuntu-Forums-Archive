### Post by cariboo on 2008-09-24
Firefox allows you to set a minimum font size, go to Edit-->Preferences-->Content, then clck the fonts & colours advanced button. You will see a minimum font size button in the resulting window.

I was using the Glazoom Firefox add on until I found this button. I set the minimum font size to 16, and now rarely need the zoom button.
