### Post by sethvath on 2008-10-19
I doubt the ubuntu installer has an option to select just the core files and install what you need since most of the programs are bundled under gnome-desktop etc. You get a dependency nightmare without any one of them. 

Another option is [http://ubuntu-mini-remix.crealabs.it/](http://ubuntu-mini-remix.crealabs.it/)
Its only the core files and you can use reconstructor to add the modules you need to create your own custom ubuntu variant.
