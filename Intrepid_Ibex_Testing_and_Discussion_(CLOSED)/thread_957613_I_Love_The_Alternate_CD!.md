### Post by ViRMiN on 2008-10-24
Well, I thought I'd rebuild my box using the RC Alternate CD as it's been an evolution from Hardy with a manual install following the FakeRaidHowto, and I'm very pleased and impressed!

I managed to resize the extended partition on my RAID and created another logical partition within it with ease (I wasn't feeling confident doing it via parted and the gparted live CD doesn't include dmraid so that was out!).  Set-up all the mount points, and a few other doings (private directory, etc), and let it do the rest.

Everything worked perfectly!  I created a new account and copied over my mailbox and my Firefox bookmarks, (and my docs of course!), so I've got a clean account to play with too with none of the crud that I've been carrying around since Feisty!

I did think that my previous install was a bit quirky as it always thought a CD or DVD inserted in one of my drives was blank, but, I could never burn on a fresh disc... turns out the drives a bit sick (as I found when I attempted to boot the RC CD!)

So +1 to the Alternate and those who worked on it, and -1 to my sick DVDRW :D
