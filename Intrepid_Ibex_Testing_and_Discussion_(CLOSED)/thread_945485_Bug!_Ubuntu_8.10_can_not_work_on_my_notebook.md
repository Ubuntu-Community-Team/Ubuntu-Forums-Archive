### Post by nhandler on 2008-10-12
broove, most of the Developers do not look at the forums. If you suspect this is a bug, I would suggest that you file a bug report on [Launchpad]("https://launchpad.net/"). There is a [guide]("https://help.ubuntu.com/community/ReportingBugs") on help.ubuntu.com about how to report bugs. Once you report the bug, I would suggest adding a link to the bug report in this thread. That way, people who have the same problem as you, who find this thread, will be able to easily check the progress of the bug report on Launchpad. If you need any more help with reporting your bug, simply post a comment here saying so.
