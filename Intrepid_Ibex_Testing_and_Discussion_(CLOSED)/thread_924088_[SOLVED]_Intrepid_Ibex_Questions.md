1. If this is your first linux machine then you are almost certain to break it a few times in the first few weeks.  I would recommend keeping backups of your valuable data so you can get stuck in and try a few things out.

There will always be bugs in any computer software, especially if it has just been released, so dont presume things will work flawlessly on your machine just because they worked in the last release.

I keep a separate /home partition so i can install new releases of ubuntu side by side with my current version.  That way I can keep swapping between the two until i know that the new version works for me.


2. The interface will probably look pretty much the same (although ibex is supposed to have a graphical re-design, but i have not seen this yet) and generally it will work much the same as the last release.

As you start to use it you will start to spot things that have changed, hopefully for the better.  Although in Ibex the vast majority of the changes will be "under the hood"
