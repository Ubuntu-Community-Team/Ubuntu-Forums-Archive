### Post by gooseisloose on 2008-10-27
I had a similar problem. After the first screen where you choose to install or run it live the resolution changes as well as the refresh rate. Are you using an LCD monitor? If so that's the cause. You can select different resolutions by pressing F4 but I don't know how to change the refresh rate.

EDIT> Oh yeah and I had that problem with the 64-Bit version of 8.10.

---

### Post by mistywindow on 2008-10-27
You're onto it GooseIsLoose.
I'm using 64-bit as well. Selecting Safe graphics mode from the F4 menu did the trick.
