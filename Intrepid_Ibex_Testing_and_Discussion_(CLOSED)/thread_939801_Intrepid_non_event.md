### Post by nanog on 2008-10-06
Well...thats not a positive attitude. 

I upgraded when alpha4 came out specifically because Calc has openoffice 3 packages that integrate better than the default openoffice debs:

```
#Calcs_open_office_ppa
deb http://ppa.launchpad.net/openoffice-pkgs/ubuntu intrepid main
deb-src http://ppa.launchpad.net/openoffice-pkgs/ubuntu intrepid main

```

Last I heard openoffice 3 would be available in the repos.

FWIW (still no 16 bit) GIMP 2.6 is available here:

```
# Christoph Korn - various apps (VLC, GIMP and others)
deb http://ppa.launchpad.net/c-korn/ubuntu intrepid main
deb-src http://ppa.launchpad.net/c-korn/ubuntu intrepid main

```

gimp 2.6 will probably be backported since a dev already has a working package in a ppa.
