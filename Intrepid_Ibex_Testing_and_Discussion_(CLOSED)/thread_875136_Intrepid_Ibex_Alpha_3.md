### Post by Exsecrabilus on 2008-07-31
Sbooder, if you want to upgrade form 8.04 instead of fresh-installing, you need to download the alpha 3 alternate CD. After burning a CD, just insert it into your computer and a window will pop-up asking you if you want to upgrade. Live CDs will not work for an upgrade.

Alpha 3 was the first development release to have a Live CD.  You can see the various ways of installation here: [http://cdimage.ubuntu.com/releases/intrepid/alpha-3/](http://cdimage.ubuntu.com/releases/intrepid/alpha-3/)

The other way to install would be to enter
```
update-manager -d
```
and install without the need of a CD.
