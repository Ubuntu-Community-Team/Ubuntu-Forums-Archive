I have a Dell Dimension 4600, testing the Intrepid beta Live CD.  Everything works fine during the live session, but when I go to shutdown, the CD does not get automatically ejected.

I get the prompt ```
Please remove the disc, close the tray (if any)
and press ENTER to continue.

``` but the tray does not eject.

I have used the same live CD on three other machines, and it ejects fine on each of them.  It's only this one machine that has the problem.

I also tried the Hoary 8.04.1 Live CD in this machine, and it properly ejects the CD when I choose to shutdown the session.  So something appears to be wrong with this functionality in Intrepid beta on this machine.
