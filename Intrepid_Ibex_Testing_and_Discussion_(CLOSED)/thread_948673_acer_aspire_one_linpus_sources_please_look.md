### Post by FrankVdb on 2008-10-27
Linpus is strange. Although you can download it from [www.linpus.com](www.linpus.com) (= Acer), all the rest seems to be closed source. If you want to run codecs on it, I read somewhere that you have to pay for them (!!!). 

The strangest thing is that although it is Fedora-based, it uses the apt-get Debian-based packaging system for some obscure reason. The result is that a lot of Debian stuff simply won't work on Linpus. 

You're not a Linux enthousiast if you call your distro "Linpus" anyway.
