### Post by BC7333 on 2008-09-10
Have been using Intrepid quite a while now and recently was experiencing a lot of blank screens and freezes, mostly when using wireless or plugging/unplugging wired. We're talking 10 or 15 times a day.. usually within minutes of logging on.

I have an intel pro 3495 abg wireless in my centrino duo based laptop.

2.6.27-1 kernel

I updated today, which included some network manager stuff, but still no improvement, was considering to try ndiswrapper with an old windows driver, but first went to linuxwireless.org/en/users/Download and compiled the latest iwlwifi drivers for my 3495.

Since then, no blank screens and no frezes for hours now, even hot switching between wired and wireless.  My mouse cursor that also used to go crazy from time to time is very quiet - but keeping fingers crossed.

Might be worth a try if you have similar, massive problems with freezes and such.

Thought it also might be worth mentioning that network problems might cause such problems.

The same problems were experienced in both gnome and xubuntu sessions, but xubuntu did seem a wee bit more stable.  Now both work fine.
