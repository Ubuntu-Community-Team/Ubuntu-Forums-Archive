My first reaction would be to suggest some type of phase cancellation might be the culprit.  Sometimes this can be caused by loose or incorrectly wired audio connections to your speakers/headphones etc.  Make sure you're using the correct output jack.

I wonder if there's not an audio mixer setting that could contribute to this problem on the software level.
