### Post by dead_end on 2008-10-23
On Monday I got a hair and decided to beat the rush and go ahead and upgrade my systems now so I would not have such a bad wait on release day. One system in a Dell Demension 3100 and the other is a Dell XPS gen 1 laptop. The Demension was a install on a clean root with a seperate home partition. The laptop was a clean wipe across the board. 

The first thing I noticed was that during boot I got a couple new error messages that simply seemed to me to be the kernel saying "hey this is not here." Boot time however did seem to be slightly better. GDM came up and I logged in. I liked the new look. At this point I was pluged into ethernet as I always need a hard line to get the firmware for my wireless cards. I was a little suprised when Ubuntu could not find the firmware on its own. No problem though, a quick search got my wireless card up and running. Total time to fix fifteen min. 

Once I was able to get wireless up and running I enjoyed the new os. Network manager was much improved. everything was very impressive. It seemed as if I was using a version one release away from LTS, rather then one release after. Once I got every thing set up the way I liked it I decided to watch a vid. The videos are kept on the desktop which has all of the hard drive space and I stream them to my laptop to watch whereever. I set up samba and openssh-server same as always. and then tried to log in to my desktop. "no route to host" ok I've messed something up, fix it try again. check IP. check config. All green. Lastly I checked my router. No dice. "no route to host" I had network. could access any website I wanted. Could ssh into any server, could ping anything so long as it was not on my local net. Spent the better part of a day trying to fix it. but no go. I had to reinstall hardy and everything worked again (mostly - suspend and hibernate always seem to be on the fritz) 

Overall I like this new version but I use my local network to much. If I only had one computer I would have no problem with upgrading at this time however unless my personal sit changes and I am down to only one computer I think I will stay with hardy. At least for now. One of the things that I love/hate about open source is that thing are changed so quickly. The problem that keeps me from staying with intrepid may already be fixed. Or new problems created. (I liked having suspend on my laptop although it works now my guess is that with the next up date it will stop working again. And then will work again on the mass update after that. :)
