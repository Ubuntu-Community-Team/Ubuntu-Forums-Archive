### Post by David Wedelich on 2008-07-18
Does anybody know what the current status of the prefetch project is?  There's a [wiki entry]("http://wiki.ubuntu.com/DesktopTeam/Specs/Prefetch") and a blueprint on the [launchpad]("http://blueprints.launchpad.net/ubuntu/+spec/prefetch"), but it seems like it has been forgotten.

The [PDF]("http://prefetch.googlecode.com/files/gsoc-prefetching-presentation.pdf") on the google code page shows the test machine's boot time went from 61.21 seconds to 47.75, and loading OpenOffice.org went from 12.74 seconds to 11.01.

This is a substantial improvement, and while it may take some time and a lot of testing to get the file reordering done, it seems like it would not be difficult to at least get the prefetching into the kernel for Intrepid.  Is anyone working on this?
