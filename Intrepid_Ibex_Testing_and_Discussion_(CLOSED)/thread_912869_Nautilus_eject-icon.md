### Post by mejy on 2008-09-24
> **SEMW said:**
> But Gnome has the ability to scale larger icons down to the required size (as you can see if you resize the panel, the icons smoothly resize -- the ones that have larger sizes, anyway).  Shouldn't it be doing this?

It does, but artistically in situations such as this it will cause the icon to appear very blurred (since 22x22 are typically designed almost pixel by pixel).  While affecting the layout isn't ideal either, it's at least a more obvious issue that forces icon designers to correct their themes.

You could certainly argue that Gnome should scale the icon from other sizes as a fallback as it does in other places, but frankly I think this is irrelevant since it's vastly preferable for theme designers to include icons tailored to all small sized commonly used in modern desktop environments (there's not too many - 16x16, 22x22, 32x32 and an SVG for 48x48 and above).
