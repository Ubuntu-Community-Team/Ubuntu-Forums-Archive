On Ubuntu community documentation there is a document [InstallationFromLinux]("https://help.ubuntu.com/community/Installation/FromLinux") which is very useful because you could install by copying the iso to a spare partition and not burn a CD.

The Live CD method could not be used for Intrepid since Ubiquity fails to detect the hard disc partitions because one of the partition (where you have copied the files) is mounted at /cdrom.
