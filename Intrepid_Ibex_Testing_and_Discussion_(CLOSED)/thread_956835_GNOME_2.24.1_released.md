### Post by int on 2008-10-23
> This is the first update to GNOME 2.24. Come and see all the bug fixing, all the new translations and all the updated documentation brought to you by the wonderful team of GNOME contributors! A lot of work has been done in the stable branch to make it even more solid than it was.

The next stable version of GNOME will be GNOME 2.24.2, which is due on November 26th. Meanwhile, the GNOME community is actively working on the unstable branch of GNOME that will become GNOME 2.26 in March 2009.

The GNOME 2.24 release notes are available at:

[http://library.gnome.org/misc/release-notes/2.24/](http://library.gnome.org/misc/release-notes/2.24/)
