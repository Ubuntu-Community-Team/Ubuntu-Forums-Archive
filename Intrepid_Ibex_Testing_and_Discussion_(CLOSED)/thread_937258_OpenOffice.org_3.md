### Post by nanog on 2008-10-03
Add calc's PPA to your /etc/apt/sources.list

```

#Calc's_open_office_ppa
deb http://ppa.launchpad.net/openoffice-pkgs/ubuntu intrepid main
deb-src http://ppa.launchpad.net/openoffice-pkgs/ubuntu intrepid main

```

Update

Install all open office files that are designated 1:3.0.0~rc2-1ubuntu2.

Open office 3 is an enormous improvement over 2.4. I've been using it exclusively for about 10 months now.
