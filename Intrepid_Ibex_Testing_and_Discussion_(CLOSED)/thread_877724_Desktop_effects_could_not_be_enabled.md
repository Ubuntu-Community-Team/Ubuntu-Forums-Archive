### Post by Daeluin on 2008-10-27
Well eventually I figured out that nvidia-glx-177 was trying to install to a kernel-2.6.24, which should have been long gone... only I discovered that due to some of my own playing, I had been loading an older kernel from a separate boot partition. After I got all that straightened out everything worked. 

make sure when you install the nvidia stuff that it doesn't throw any weird errors. though your glxinfo was completely different from mine, so it's probably a separate problem
