Not sure why, but I figured the unsupported-plugins of Compiz was still at 0.7.6 so I removed it so all the package versions would match.

Well afterward my notification area was not grabbing the icons, so I did a little searching and some others had an error where a certain compiz plugin would break the notification area so you'd click through it.

Judging by that, I decided it was a Compiz error, re-installed the unsupported plugins package and it worked.

Just thought I'd put that out there.
