### Post by inxygnuu on 2008-10-12
This is a simple tutorial on how to get your Netgear usb adapter working. This worked for me and I have an HP laptop, Ubuntu Intrepid (8.10), and a wnda3100 usb adapter.

1) First, if you have internet to begin with, go to the add/remove programs, and select "windows wireless drivers". install it.

2) after you have installed it, plug in your adapter, get and get the cd.

3) go to windows wireless drivers. it should be in: system -> administation -> windows wireless drivers.

4) click on "install driver(s)" go to your cd and look for an ".inf" file load it in, and you should be ready to go.

For me, the connection automatically loaded, but it might require a restart. 

Hope this helps!:):guitar::popcorn::KS
