### Post by Iehova on 2008-10-20
Hi guys, I know the devs don't often check here, but I wanted somewhere to collate my thoughts on Intrepid and receive feedback before I file bug reports and all that jazz.

So I'll start with the bad stuff. Firstly, it's a known issue but I get a black screen when resuming from suspend due to my using the nVidia driver. Hibernate, however, does work, which is good, but for some reason is really, *really* slow to resume (it takes the same time as a cold boot), so my first question is whether this is in itself a known bug, or is it to be expected for some reason?

The second problem for me, one that is/was an issue with Hardy as well, is that my laptop's microphone (a conexant hermosa sound card) doesn't work at all. On this forum alone it's been discussed many times, but thus far no-one's come up with a solution. I'm pretty sure bugs have been filed in this for launchpad already, but they're all marked as unconfirmed, incomplete etc.

Apart from those issues, which are pretty major to me but minor, I suppose, in the grand scheme of things, I'm really impressed by Intrepid. My SD card reader now works perfectly, which it didn't in Hardy, having installed ubuntu-restricted-extras java finally works for me, whereas I could never ever get it to work in Hardy. Basically, everything else works wonderfully.

Anyway, if anyone can point me in the right direction with the suspend, hibernate and microphone issues that would be lovely. Oh, my final question is am I too late to file bugs and get them acted upon before release?

Thanks,
