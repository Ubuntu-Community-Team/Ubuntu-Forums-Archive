### Post by 23meg on 2008-05-23
1) Design / artwork people work on themes, not developers

2) The current Human theme has adjustable colors (System / Preferenes / Appearance / Theme / Customize / Colors) and the next default theme would certainly be too

3) Separate theme teams are working on different themes, some of which are probably going to be theme variations. The idea is (actually, has been) that anyone who can come up with a complete theme of reasonable quality can get it into at least Universe. You may want to check the ubuntu-artwork mailing list to see the progress and participate.
