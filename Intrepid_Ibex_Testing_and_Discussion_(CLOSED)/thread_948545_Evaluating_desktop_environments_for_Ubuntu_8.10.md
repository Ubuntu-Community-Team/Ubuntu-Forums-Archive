Interesting article with useful metrics --
> In preparation for the release of Ubuntu 8.10 (Intrepid Ibex) I’ve been running several virtual machines with the beta OS. I’ve created 1 virtual server each for Ubuntu Server, Ubuntu, Kubuntu, and Xubuntu. The latter 3 were created so that I could evaluate the different desktop environments in preparation for the upgrades of my own system as well as the systems of others....

I think Ubuntu is looking like the clear winner for Intrepid Ibex....

As it is the only recommendations I can make for Ubuntu 8.10 is to use the Ubuntu Gnome desktop or to strip the OS down and use an alternative desktop environment such as XFCE but without all the cruft of Xubuntu.
