It because of the keyboard layout that you are using.
You probably selected International English with Deadkeys...
... or some international keyboard that behaves the same way.

Example: Pressing **~** then **a** produces **ã**.

To change this, go to:
**System > Preferences > Keyboard > Layout tab.**

And set your keyboard accordingly!
