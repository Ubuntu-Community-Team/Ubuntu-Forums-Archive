### Post by marmuta on 2008-10-17
Well there is one way to use dark themes with firefox (from memory):
In Preferences->Content->Colors
enable  "Use System Colors" and
disable "Allow pages to use their own colors".

It then ignores hard coded colors of all web pages and replaces them with the ones of the current theme.

Of course this removes all color information from web sites and makes them look rather bland, but it's nonetheless useful for pitch black night mode themes. No more white page surprises this way.
