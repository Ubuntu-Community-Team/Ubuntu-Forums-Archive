Hi !
I have a eeepc 900A (intel i945GME) under hardy, and i used to set INTEL_BATCH=1 to improve performance of 3D software (compiz, 3d demos...)
for example, with hardy i get by default 600-700fps with glxgears
with INTEL_BATCH=1 set in the command line, i get 1300-1400fps.
I know that glxgears is not a good benchmark, but it's clear that the performance is also better in openarena, etc... (+50% fps)

Now with intepid, it seems that the tweak doesn't work any more, do you confirm ?
