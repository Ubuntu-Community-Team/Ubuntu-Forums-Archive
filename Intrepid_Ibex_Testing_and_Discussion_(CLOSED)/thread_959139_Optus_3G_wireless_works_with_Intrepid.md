### Post by AndyCee on 2008-10-26
Hi fellow Aussies.

The current release candidate for Ubuntu 8.04 Intrepid Ibex can use the Optus 3G wireless USB modem. In fact I'm using it right now.

The only configuration that needs to be made after plugging it in, is the APN must be set to "internet" (lowercase). Go figure.

EDIT: After looking around, most of the connection problems people seem to have for other carriers seem to be solved by changing the APN.

So here are some I've found online. Might be worth trying them if the default is different and you can't get connection:

CARRIER          APN

3/HUTCH -- 3netaccess
OPTUS ---- internet
TELSTRA -- telstra.internet
VODAFONE - vfinternet.au
VIRGIN --- VirginBroadband
EXETEL --- exetel1 or Connect

etc.

I give no guarantee that they work, but google doesn't lie, does it?


Umm. That's it. Hopefully someone will find this helpful.
