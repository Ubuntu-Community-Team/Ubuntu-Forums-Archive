### Post by the real omni on 2008-10-04
I find a great colour scheme is to just select the 'Crux' gnome theme and use Emerald with the 'Buttonless' theme for the window decorator.

If you need the min/max/close buttons you just have to change the value for "Minimum titlebar height" to 19 in the "Edit theme" tab.

The 'Crux' gnome theme should come packaged with Ubuntu.

Emerald can be installed through Synaptic Package Manager - you'll also want to install the CompizFusion Icon to make it easy to switch from Metacity to Emerald.

Buttonless emerald theme: [http://themes.beryl-project.org/theme_details.php?id=20](http://themes.beryl-project.org/theme_details.php?id=20)

And for desktop background, just load firefox and do a search on google images for your screen resolution, then right-click on the full-sized image and choose "Set as desktop background."

Voila - no more ugly brown "human" theme and a custom desktop background.

If you want to be really sneaky, install xwinwrap and have it loop an .AVI or a screensaver as your desktop background. ;)
