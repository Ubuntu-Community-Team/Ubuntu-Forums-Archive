### Post by tact on 2008-10-07
open up a terminal
type "nautilus"
right click on any folder and select Properties
click on the "open with" tab
set it to file browser

fixed

Intrepid is beta code.  Not really for "absolute beginners" if you are one (you posted in that area).  

One must expect odd behaviour and even breakages before it goes gold.  Example - this morning's updates have broken compiz on my machine.. might be a few days before its fixed...
