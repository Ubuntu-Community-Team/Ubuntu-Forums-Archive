### Post by devoncatt on 2008-10-21
I had an interesting problem with the latest xorg update. My keyboard plugged into my front of the computer USB ports stopped working altogether in KDE but it worked fine in gnome. When i moved it to the rear usb ports it started working in kde.

To get it working, the first time after the initial fresh install (neither usb mouse nor keyboard worked)  I had to add the following to my xorg.conf file. Is xorg not finding usb peripherals?

Section "InputDevice"
        Identifier      "Generic Keyboard"
        Driver          "kbd"
        Option          "CoreKeyboard"
        Option          "XkbRules"      "xorg"
        Option          "XkbModel"      "pc105"
        Option          "XkbLayout"     "us"
EndSection

Section "InputDevice"
        Identifier      "Configured Mouse"
        Driver          "mouse"
        Option          "CorePointer"
        Option          "Device"        "/dev/input/mice"
        Option          "Protocol"      "ImPS/2"
        Option          "ZAxisMapping"  "4 5"
        Option          "Emulate3Buttons"       "true"
EndSection

and relogin.
