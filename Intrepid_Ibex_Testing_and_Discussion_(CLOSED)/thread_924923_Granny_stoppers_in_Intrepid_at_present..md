### Post by Peter09 on 2008-09-20
Overall I feel that Intrepid has been ok but if I compare it to my experiences testing Hardy I find that I have a few major issues that have not been resolved and which impact me more than the Hardy issues. All these issues would be ones that impact a first time, or simple user.

These are:

1. On boot I need to edit the grub menu to remove quiet and splash and insert noapic or the machine hangs.
2. Wireless Networking will not start from suspend, I need a reboot.
3. I cannot 'restart' the machine (from the shutdown menu), even after editing the grub menu it still results in it hanging. I have to power cycle it.
4. Still not able to use the ATI drivers. (most probably could if I put some effort into it, but as a 'simple' user they are not available).

So, if I was going to put Intrepid on my Grannies machine today (browsing and email) these would be the show stoppers.(may be not the ATI drivers).

I'm not sure if these are all identified bugs or not, I know there are a number of bugs in these areas but whether they are a direct match I'm not sure.

Machine - Toshiba Satellite Pro
