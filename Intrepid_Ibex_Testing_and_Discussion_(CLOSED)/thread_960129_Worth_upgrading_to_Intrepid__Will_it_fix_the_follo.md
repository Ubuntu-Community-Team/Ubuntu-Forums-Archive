I'm running Hardy and have it running reasonably well on my Dell dimension 4600.  The only things that really bother me are:

-  I lose wireless randomly and can sometimes be a real PITA to get back (RT2500 based pci card). Usually during a large downloads or streaming.  Laptop in same room has no issues. Really damn annoying. 

- Xbox 360 controller support, takes a bit of fooling around.  I've heard the intrepid kernel fixes this.  True? 

- Nvidia video card.  Was only able to get drivers working via envy.  Still loses my nvidia config settings whenever I leave the system alone for a while.   

I did upgrade my laptop (portege 2000) from Hardy to Intrepid, all went well and it seems to work fine.  But that laptop is a bit of a slug anyway and requires toned down features so I'm not sure if I'm seeing any real benefits on it yet (and I barely use the laptop).  Just not sure if I want to potentially screw up a good thing (minus the above hiccups) with my current Hardy desktop system.
