### Post by God Dammit on 2008-10-26
I am using a computer I built myself. It uses an Intel DP35DP motherboard that fully supports UEFI booting. I recently tried to do a UEFI installation of the release candidate of Ubuntu 8.10. Strangely enough, even though the developers claim that support exists for doing a native UEFI install on x64 platforms there don't seem to be any disk partitioning tools that support creating the necessary EFI system partition. I had to boot from my 64-bit Windows Vista RTM DVD to create a GPT partition table and the necessary EFI system partition.

When will updated disk partitioning tools that are UEFI compatible be available in Ubuntu 8.10?
