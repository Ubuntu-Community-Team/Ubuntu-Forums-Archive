### Post by Lieter on 2008-09-18
Hi there,

I have always hated the default config for compiz that is provided by ubuntu. It is, in my opinion, a real turn-off when you enable desktop effects. So i hereby present a prosposal for a lightweight(works flawlessly on my NVidia 5200 FX om 2880x900(thats two 1440x900 in dual screen setup)) default configuration.

It is mostly aimed at first time user that want functionality and easy usability(i haven't taken accessibility in consideration for this one). The used and configured plugins are(among others):

[LIST]
[*]Desktop wall in a 2x2 configuration
[LIST]
[*]Expo, because it extends the use of desktop wall(No hotkey, left lower corner to activate). 
[/LIST]
[*]Animations are visually pleasing(i.e. not 'heavy' or 'wild')
[LIST]
[*]Open and Close are Glide 2 for Application windows
[*]Dream is used for dialogs
[*]Fade is used for the less intrusive window types(Notification, Tooltip etc)
[/LIST]
[*]Nautilus has been bound to hotkey <Super>e(for former windows users)
[*]Scale has been enabled on corner bottomright
[LIST]
[*]Has large text for window names
[*]Regex Matching has been turned on for searching
[/LIST]
[*]Wobbly windows are less wild than default
[/LIST]

I would really like if some of you would test this configuration and give your feedback in this thread, so i can improve this configuration and submit it for Intrepid+1(JJ). Later on I'll make a configuration for the cube(should be wilder than this one).

Config is attached, import into compiz by untarring, running CCSM(Compiz Config Settings Manager) => Preferences => Import as.. => follow instructions
