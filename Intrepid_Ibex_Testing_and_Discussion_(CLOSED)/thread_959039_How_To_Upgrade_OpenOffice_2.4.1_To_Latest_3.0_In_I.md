### Post by shirshasin on 2008-10-26
Here are the instructions to upgrade default OpenOffice 2.4.1 to latest release 3.0 safely in Interpid Ibex( Ubuntu 8.10 )

1)Click System, then Adminstration, and in it Software Sources.
2)Provide your password to open Software Sources dialogue box, in it click on Third-Party Software tab.

2)Click on Add button below.

3)Copy & pate the following line 
deb [http://ppa.launchpad.net/openoffice-pkgs/ubuntu](http://ppa.launchpad.net/openoffice-pkgs/ubuntu) intrepid main
to APT line: box and click on Add Source button.

4) Reload the repos.

5)Now open up Synaptic Package Manager and click to Origin button below.

6) Click on ppa.launchpad.net/main line above.

7) Click of Mark All Upgrades button above, if you don't have openoffice installed you can make fresh install from here too.

8) Confirm the download and upgrade, sit back, relax and enjoy.
