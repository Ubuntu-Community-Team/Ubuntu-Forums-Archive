### Post by klepto on 2008-10-25
I have been a loyal user of Ubuntu since Edgy, which wasn't too long ago might I add but Ubuntu has come a long ways. I installed Intrepid Ibex and right off the bad I connected to my WPA2 wireless network which always took me like 2-3 steps extra on older versions of Ubuntu. My atheros wireless card had a few problems with Hardy as it would just disconnect and say that it wasn't ready. I haven't seen that problem here so far. 

Using pulse audio server so that I can play Rick Astley's 'Never Gonna Give You Up' :guitar: j/k and play some japanese movies in [smplayer]("http://smplayer.sourceforge.net/") and no one hogs up the audio is wonderful. 

I'm running Ibex on a 1.33ghz with like 600+ megs of ram but it runs just great. I have yet to see one real problem on this installation well maybe the problem is that I am only an intermediate level user of Ubuntu and I need to get my hands dirty. 

Thanks for such a great product. 
You get the gold star :KS and a cookie for your outstanding work.
