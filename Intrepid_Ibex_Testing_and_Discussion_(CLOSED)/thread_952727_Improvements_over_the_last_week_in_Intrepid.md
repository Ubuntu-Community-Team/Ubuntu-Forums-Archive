### Post by WiFi Ed on 2008-10-19
It's starting to look pretty good for me.:)

No compiz.real crashes in the last few days.

Fast, smooth boot, no weird black screens before the wallpaper appears.

Even faster shutdowns.

The Network Manager failure to load message during bootup has gone away (although the network has always worked once I got to the desktop).

After loading some of the "bad" gstreamer stuff to play mp3's and m4a's, etc., most streaming internet media that used to work only in Windows for me now runs fine in Firefox on Intrepid without any fiddling.

Rhythmbox now automatically fills in the "genre" column when I add an internet radio station.

Minor gripes:

Can't find the option to set my clock using a NTP server anymore. It used to be there in Hardy.:confused:
