### Post by phenest on 2008-10-01
Totem-gstreamer will play midi files. Totem-xine will not. If you use:
```
sudo update-alternatives --config totem
```
and select gstreamer, midi files will also preview in Nautilus.

I installed Totem-xine because it supports DVD menus which Totem-gstreamer doesn't have.

Still, this is pretty neat.
