### Post by vgrisham on 2008-10-19
This pertains to 64-bit Intrepid (gnome).

I have my screen saver set to display photos from my Pictures folder. When it is time for the screen saver to kick on, the screen goes dim as if it's working but then goes right back to a normal bright screen, never displaying any picture. Oddly, if I preview from the screensaver setup dialogue, it works just fine. 

Anyone else seeing this?

### Post by ktp on 2008-10-19
are you using compiz...if you are try going to compiz config settings manager, go to general options, and select the general tab and untick "undirect fullscreen windows". If this doesn't work I would suggest unticking "force independent output painting" also, if it wasn't already. This as worked for others.
