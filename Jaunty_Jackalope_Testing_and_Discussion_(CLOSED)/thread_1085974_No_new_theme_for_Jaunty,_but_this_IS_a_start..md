### Post by dentaku65 on 2009-03-05
I found this interesting "solution" in the meantime the new theme will be released (probably in Karmic)

Keep Human Theme

Install Nodoka
```
sudo apt-get install gtk2-engines-nodoka
```

Download Icon set Human-O2 (deb file)
[http://www.gnome-look.org/content/show.php/Human-O2?content=95983](http://www.gnome-look.org/content/show.php/Human-O2?content=95983)

Go in Appearance and choose:
Customize -> Controls -> Nodoka-gilouche
Customize -> Icons -> Human-O2

Done!
I think this is pretty elegant and not much different than the official one.
