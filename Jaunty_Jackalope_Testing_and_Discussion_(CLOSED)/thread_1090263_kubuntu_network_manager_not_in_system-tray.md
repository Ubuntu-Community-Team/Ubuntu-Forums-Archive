### Post by RIchard James13 on 2009-03-08
Click on the cashew on the far right of the panel. Then click on "Add Widgets..." Select the network management widget and click "Add Widget"

BTW the network widget resides on the panel not in the system tray. The system tray has a black background and contains things like the mixer and klipper. 

You can move any object around the panel by clicking on the cashew. Then the mouse icon will change into a move cross. This way you can get your widgets where you want them. The only problem with this is some things in the system tray like klipper bring up a menu even when the move cross is visible. One solution to this is to wiggle the cursor between the system tray icons and move it from there. Another solution is to not move the system tray at all but instead push it by placing other widgets before or after it.
