### Post by panaut0lordv on 2009-02-15
The best way to fix it is:
1. sudo rm /etc/X11/xorg.conf
2x. If you don't have nvidia-settings installed. sudo apt-get install nvidia-settings
2. sudo nvidia-settings
3. Save config after changing settings to whatever you want to have.
4. Reboot. Now it should work perfectly.
