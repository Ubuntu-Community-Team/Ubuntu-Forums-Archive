### Post by sgosnell on 2009-03-28
I've been running Jaunty for the past few alphas, and it works great on my EEE.  Heat is about the same as it has been on many different distros - I don't see much if any difference between any OS I've tried.  If you want cooler, get a different laptop.  Some just run hot.  My HP, which I have since given away, ran very hot, but my EEE is much cooler.

Wireless and pretty much everything else works out of the box, much better than in Intrepid.  The 2.6.29 kernel, which won't be in Jaunty, is much improved, and you can install it easily enough.
