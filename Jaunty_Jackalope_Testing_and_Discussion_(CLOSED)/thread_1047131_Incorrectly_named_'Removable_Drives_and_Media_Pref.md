### Post by forumache on 2009-01-22
This is a leftover from previous versions of Ubuntu.
I think it is gnome-media something package. All the plugged in stuff is now managed by nautilus.

Start synaptic, select Installed on the left then sort them by the column with the Ubuntu logo (meaning they are included on the CD). Remove that gnome-media thing (you can see that it was left to an old version).

Please check the exact name of the package, I removed it long time ago (months) so I don't remember exactly its name, but it is easy to spot.
