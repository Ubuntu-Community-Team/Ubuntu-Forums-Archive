### Post by gotsanity on 2009-04-22
On a side note... i found a workaround that will do until we can figure out how to fix it properly...

Create a quick script named setrez:

```

$ touch setrez
$ gedit setrez

```

Copy and paste the following into the script:

```

#!/bin/bash
randr -s <resolution>_<refresh rate>

```

Example:

```

#!/bin/bash
randr -s 1600x1200_75

```

Make the script executable:

```

$ chmod +x setrez

```

Add the script to your startup applications and you should be good to go.

Its not the prettiest solution but it works for the mean time. Side note: This will only work for the user account that created the script. If you want to use it on all your accounts than you should dump it in /bin and chown it to root
