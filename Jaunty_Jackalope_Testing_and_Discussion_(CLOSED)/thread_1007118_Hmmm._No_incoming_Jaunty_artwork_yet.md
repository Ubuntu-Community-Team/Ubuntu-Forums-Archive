### Post by scaine on 2008-12-14
The thing that gets me about the "just change it" replies is that they've really missed the point completely.  Why change anything in the distro if it's so easy to change?  It comes up in every thread to do with the defaults - Rhythmbox for music, Firefox for web, Evolution for mail, etc.

Over and over - "if you don't like it, change it".  Well - yeah, it *is* easy.  For you.  You know about the alternatives and possibilities.

But the point is, when you're first trying out Ubuntu, as a potential convert from whatever you were using before, I'm pretty sure that nearly every one of those potential converts says "Holy mother of ugly - that's bad".

And if you're a convert from a completely different O/S (ie not linux with gnome), you might have no idea how to change it and make it better.  So wouldn't it be better if the initial experience is pleasing, instead of jarring?

Perhaps it was important for Ubuntu to "stand out" with the Warthog when it was first hitting the scene.  Bizarrely, I'm pretty sure (in my opinion) that Ubuntu became a success, wholly *despite* the theme.

Oh, and [its resize borders are too small]("https://bugs.launchpad.net/ubuntu/+source/metacity/+bug/160311").  :p
