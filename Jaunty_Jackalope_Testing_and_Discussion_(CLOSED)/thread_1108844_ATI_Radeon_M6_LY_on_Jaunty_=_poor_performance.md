The "XaaNoOffscreenPixmaps" disables hardware acceleration, which (perhaps ironically) happens to speed up compiz rendering. It makes things better, but it's not ideal.

I suggest you try EXA acceleration with some tweaks.

Try the following (comment/remove all other entries in this section):
```
Section "Device"
	Identifier	"Configured Video Device"
	Option		"AccelMethod"			"exa"
	Option		"EXAOptimizeMigration"		"true"
	Option		"AccelDFS"		        "true"
EndSection
```
