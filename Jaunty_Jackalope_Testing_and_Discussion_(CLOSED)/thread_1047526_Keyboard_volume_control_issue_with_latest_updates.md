### Post by gspat on 2009-01-22
I just noticed this, so I thought I'd share it in case it helps someone...

It had to have happened with the latest updates, because it was working fine yesterday.

My volume control knob on the keyboard quit working, it turned out the default was unset during the latest upgrades.

To fix it, I just had to re-define the default by right clicking the speaker icon and selecting what I wanted from in the properties menu.

Hope this is useful to someone!
