### Post by Artemis3 on 2009-01-14
In my opinion, anything under 512mb should use xfce instead of gnome. Try removing non-essential services like tracker, pulseaudio, etc. Under 192mb consider using just fluxbox or equivalent light windowmanager (not a full desktop environment). It is also useful to try not mix programs using different widget libraries (ie: with XFCE, prefer gtk over qt). With Jaunty you should test the now stable ext4 filesystem, i think you need the alternate installer for that. For the port issue, get a cheap usb mouse and keyboard or adapter. I don't recommend anything wireless, as your battery might run out at the worst moment...

### Post by MadsRH on 2009-01-15
Perhaps Ubuntu Lite (or U-lite as it has been renamed) would be a solution for you? 

U-lite is an independent project to create a distribution based on Ubuntu that is capable of running on limited hardware, such as a Pentium II with 96 MB of RAM memory. In contrast to other variants of Ubuntu, U-lite is focused on software packages with low resource consumption.

U-lite currently ships with the LXDE desktop environment which uses Openbox as the default window manager.
