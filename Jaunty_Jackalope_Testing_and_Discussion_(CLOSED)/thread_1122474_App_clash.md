Hi
I have found a clash between two apps, I suspect this will effect all the 
CD/DVD burning software and CD/DVD players.
ie anything where a cd or DVD is put in a drive with audio/video files on it.

If FREEVO is installed it appears to watch the CD/DVD drives for audio/video files.
When its found them it takes control of mounting the drive.
So if you have just burnt a CD full of mp3s or similar using K3b, it will not eject at the end of the burn.
If you right click on the desktop icon showing the CD/DVD, you then get the message that only FREEVO can unmount the drive.
Luckily it lies and root can still unmount the dive.
