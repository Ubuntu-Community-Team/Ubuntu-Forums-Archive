### Post by tynen on 2009-04-01
I found the [solution.]("https://bugs.launchpad.net/ubuntu/+source/nvidia-graphics-drivers-96/+bug/226547") Apparently this is a known issue for the version 96 driver. It is the XORG configuration that is messed up. For anyone else that is having this problem change your configuration to

```
Section "Device"
Identifier "Configured Video Device"
Driver "nvidia"
Option "NoLogo" "True"
Option "UseDisplayDevice" "DFP"
Option "AddARGBGLXVisuals" "true"
EndSection
```
