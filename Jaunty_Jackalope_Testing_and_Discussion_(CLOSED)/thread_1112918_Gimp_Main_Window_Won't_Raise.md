### Post by Lunx on 2009-04-01
I installed Jaunty a couple of days ago and everything seemed to be working fine, but last night I discovered a bug when running Gimp. The main window (canvas) will not raise above the toolbox window and the docks window when in focus. I thought it may have been something to do with compiz as I'd been fiddling around with settings and thought perhaps it was something I'd caused, but turning compiz off did nothing. I tried going into System --> Preferences --> Windows and setting it to Select windows on mouse over but it did nothing either (would work on the main Gimp window and windows from other apps, but not Gimp canvas with other Gimp windows) Real PITA as Gimp is one of my most used apps, and so I've now dumped Jaunty and gone back to Intrepid. Works fine in 8.10 and I've even upgraded to latest version 2.6.6 by downloading the deb packages from Getdeb. I may give Jaunty a try again once the final version is released.

As an aside, I will post a bug report on Launchpad once I can actually get to it, website won't load (24 hours now since I've been able to get there)
