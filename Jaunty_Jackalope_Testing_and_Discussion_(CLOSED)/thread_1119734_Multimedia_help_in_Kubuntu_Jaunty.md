### Post by MTGap on 2009-04-08
Do you mean in Konqueror? For playing flash in Konqueror you have to go to:

Settings>Configure Konqueror>Plugins(Under Web Browsing)>Plugins(Tab)>Scan for Plugins 

This should detect the netscape plugins for flash and java just hit apply and flash should work correctly now. It still is a little buggy, install firefox or another browser if you have more problems. Konqueror is still the best though ;)
