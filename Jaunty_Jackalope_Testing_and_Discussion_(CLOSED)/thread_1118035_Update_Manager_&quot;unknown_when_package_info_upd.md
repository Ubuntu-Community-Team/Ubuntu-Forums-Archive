### Post by emendelson on 2009-04-06
I've been building a minimal Gnome-based Jaunty setup, starting with the alternate install CD of beta 1, and installing a command-line system followed by Gnome-core and a few other minimal items. 

Everything works well except for one detail. Whenever I run the Update Manager, no matter how often I check and recheck, the main dialog shows the message: "It is unknown when the package information was updated last. Please try clicking on the 'Check' button to update the information."

This does NOT occur in a separate full-scale installation where I've upgraded to Jaunty from Intrepid. 

Does anyone know what might be missing in the setup that can't tell when the last update was performed? Thanks for any advice.

EDIT: I fixed this problem by installing update-notifier. Maybe this information will be useful to someone else...
