### Post by danf_1979 on 2008-11-20
Hi guys, just wanted to let all jaunty inkscape fans out there, that I began mantaining a PPA for inkscape svn builds. I'm a regular user of inkscape myself, so I'll be updating the PPA very often.

To add the PPA, edit /etc/apt/sources.list, and add:

```

# Dan's PPA: https://launchpad.net/~danf-1979/+archive
deb http://ppa.launchpad.net/danf-1979/ubuntu jaunty main

```

Enjoy!
