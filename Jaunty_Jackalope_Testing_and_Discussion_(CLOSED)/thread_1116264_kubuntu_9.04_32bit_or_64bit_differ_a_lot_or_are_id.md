### Post by CyberMando on 2009-04-04
The Kubuntu installer will create partitions and filesystems in the process. If you have free disk space it it all you will need.

32 bit and 64 bit are very similar. They are identical except for the few packages that are not 64 bit yet, and those are coomercial things like Flash and java. 64 bit support is getting better all the time. Install 64bit.

Kubuntu and Fedora are similar in that they are linux and GNU so they use the same linux kernel and run the same software. The installers are different and the packaging systems are different. This like device names, kernel module loading, supported hardware and so forth are the same.

Jump into installing Kubuntu Jaunty 64 bit and use your book along with the web documantation for Kubuntu. It will go quite well.
