### Post by Anduu on 2009-04-16
Just a little warning....

Be very careful cleaning any Firefox stuff with Bleachbit if you use Foxmarks.

It hosed something for me and Foxmarks stopped syncing properly.Every restart of Firefox Foxmarks insisted it needed syncing and any new bookmarks I had added would get lost in the process.

I don't know what got messed up but I ended up having to delete my ~/.mozilla/firefox folder and start from scratch.Luckily I was able to re-upload my synced bookmarks from my Windows machine to the Foxmarks server.
