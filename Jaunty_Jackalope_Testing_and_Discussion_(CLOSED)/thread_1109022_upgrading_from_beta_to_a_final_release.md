### Post by droazen on 2009-03-28
The only possible issues you might encounter by doing an upgrade rather than a clean install would be related to changes in the default configuration files for your programs, since an upgrade updates the software but doesn't overwrite the config files in your home directory left over from your previous install. For example, there were some known issues related to the pulseaudio configuration files when upgrading from intrepid to jaunty -- the intrepid versions of those files caused problems with the jaunty versions of pulseaudio.

However, the beta is so close to the final release that it's unlikely that there will be any major changes to the default configuration files that would cause those kinds of problems :)
