1&2. I've had them there for the last 2 years that I've been using Ubuntu although they may not be as easy to find (unless you search the forums, where you can find it pretty quickly).

### Post by mb_webguy on 2009-02-09
Some of these are really personal preference, and not things that are necessarily or appropriate.

#1 and #2 I don't think are really that necessary.  New users rarely have need to install software from source or from rpm packages, and should be encouraged to use the repositories.  Compiling software from source isn't something that newbies should be doing, and no matter what you do, installing software from rpm packages is going to be problematic for the basic reason that they are designed for a different family of distributions.  Besides, it's becoming increasingly rare to find a software project that provides rpms but not debs.  However, one thing I *would* like to see is some integration with Launchpad, or at least the ability to easily add the gpg key for a Launchpad PPA.  Since Launchpad is maintained by Canonical, the ability to search and easily add Launchpad PPAs seems feasible.

I agree somewhat with #7.  We already have this functionality to a certain extent with Hardware Drivers, but I'd like to see some added functionality here.  Since wireless cards are a continual problem, perhaps it could detect an unsupported wireless card, automatically install ndisgtk, and inform you that you need to use the Windows wireless driver.  Beyond that, though, providing means to download proprietary software from third-party sources not partnered with Canonical is problematic in several ways.  Legal issues aside, Ubuntu has no control over where and how such software is distributed, so such functionality could easily become broken through no fault of Ubuntu. 

As for #14, I think selecting Extra from the Visual Effects tab of the Appearance dialog should automatically install Compiz Config Settings Manager, or at least prompt you to do so.

### Post by Faolan84 on 2009-02-09
Well I would like to see a system wide equalizer for PulseAudio as an option in the volume controls. plus equalizer options in Rhythmbox and Totem. it shouldn't be that hard to implement and there was one in the works around late 2006 or so for Rhythmbox but it all got pushed under the boat for some reason and never released.

I'm not saying that these should be enabled by default but they should be there if you want them. You could even add the system wide equalizer in another tab in the volume control window and put a little check box in the tab that says "Enable Equalizer" and another one that says "Enable Preamp"

Also, I'd like to see Thunderbird replace Evolution too. That would be awesome. As for Firefox 3, for as much of a bad reception as it has gotten you can always set it back to work they way you liked it before--however i like the improvements. Finally add Sunbird and make them able to integrate all three to work with OpenOffice if needed like Evolution can. Integration is key here for the office environment. Lastly, provide an nice Roget's-based thesaurus program and a grammar checker in OpenOffice for those who need it.

I think Compiz should be dropped from the default install. Yes, I said it. It's a waste of space and doesn't really enhance the user experience. Instead, I think compositing basic could be added to Metacity since Compiz uses Metacity by default in Ubuntu to draw it's themes.

I like the idea of changing the theme to Aurora. Nodoku is also nice, the default engine in Fedora, by the way.

Usplash should also be replaced with splashy AND the splash screen should provide more information about what is occurring during the boot process like OpenSuSE does if you give it the "verbose" option. This way if something fails or if the system hangs at boot you know what is wrong or that the hard disks are being checked for errors. You would not believe how many complaints I get about this. Also on a similar note, the splash screen will not be displayed during boot--only a blank screen--this is annoying and can become problematic.

Qalcutate should replace the default gnome calculator. A program that looks exactly like a calculator has no business being on a computer. If you like that kind of thing then you may was well keep a real one at your desk for the full experience. It would also be a lot faster.

Replace the default iamge viewer with GQView / Geeqie (GQView's eventual successor). When I install Ubuntu on a customer's computer this and evolution are the first things to go in most cases.

on a side note: this would also make mono irrelevant if Tomboy got replaced somehow, not unless Ubuntu starts carrying Gnome-Do and Banshee by default. Removing mono would speed things up a bit, but then again Banshee 1.4 looks like it's going to be a great release that will provide real competition for Rhythmbox. However Gnome-Do needs a lot of work.

Work on replacing X-Sane with something more--shall I dare say it--sane. Let's face it, X-Sane's UI sucks. We really need a feature complete HIG-compliant scanner software.

Providing Inkscape by default would be sweet too.

Start-up Manager should be provided by default. Also the ability for the the Administrator to administer the system without being actively logged into his account. For example if a user clicked "System > Administration > Synaptic Package Manager" then he could call over the system Administrator to provide his username and password. the idea of just having a password box without a user selection box here is dumb. I see no point in having to log out user just to install something which is what MOST users will do since the average Joe doesn't know about pressing "CTRL+ALT+Fx" and using apt-get to search and install. Of course this should be something that can be disabled if needed.

Gnome-Screensaver should have an "Advanced" button on it so that you can tweak the settings of each screensaver to your liking.

Finally, WILL SOMEBODY PLEASE UPDATE GNOME MAHJONGG! Although I'm half joking here it would be nice to be able to import layouts and tilesets from Kyodai Mahjongg which I have installed (version 10.17 with the layouts and tiles from version 2006 added). Improvements to Gnome Mahjongg would be greatly appreciated. Also, network games in the default chess game don't work either and they too need to be fixed.

That's my wishlist pretty much. Other than that, Ubuntu is still rockin'.
