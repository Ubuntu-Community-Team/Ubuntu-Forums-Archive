### Post by lovinglinux on 2009-03-07
I would install Hardy Heron until a new LTS version becomes available. But this is my personal opinion. One of the reasons I switched from Windows to Ubuntu was the huge amount of required maintenance. Once every 6 months I had to format the HD, re-install Windows and configure everything again. That was painful.

Ubuntu has several options to keep your configurations, like having a separate home partition and scripts to install all software again after upgrade, but not everything is the same between versions and some stuff can stop working properly with old configurations.

So, unless your computer doesn't work properly with the current LTS version, I would stick with it and wait for the next LTS to do a fresh install.

I have intrepid on a laptop because other versions doesn't work. But I still have Hardy Heron on my main computer and I feel there is no need to upgrade yet.
