Last suggestion and I'll shut up with the feature requests.

Nautilus has a lot of cool features built into it. One that is missing however is the ability to access a mobile device using Obex via USB. This would be nice to have as well.

Use Case
1) User connects USB cord to phone
2) User connects USB cord to PC
3) Nautilus displays device similar to how it displays USB memory sticks
4) User copies/pastes/deletes files on mobile device
5) User unmounts device via popup context sensitive menu item

I know these features are there in Ubuntu currently but the the user has to play around with fstab, umount and other stuff that, if done incorrectly, can cause some bad things to happen.
