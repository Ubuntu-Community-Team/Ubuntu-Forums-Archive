### Post by biasquez on 2009-01-26
i found this solution:
- edit file /usr/share/nautilus/ui/nautilus-navigation-window-ui.xml
- add <toolitem name="New Tab" action="New Tab"/> in section <toolbar name="Toolbar">

new tab, cut, copy and paste ;)
