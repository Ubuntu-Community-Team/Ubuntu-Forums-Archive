### Post by cariboo on 2009-02-11
Jaunty is quite stable if you have the right hardware. The Nvidia 8 and up series of graphics adapters work really well with the latest driver. Sound works but there has been a regression to Intrepid's sound system. 

Boot times are better, on average my boot time is about 5 seconds faster.

Ext4 works really well, although some people have reported data loss.

If you are able to repair most problems without panicking, you should be able to run Jaunty, but remember it is only at alpha 4.

I would suggesst you go to [Jaunty Jackalope Testing and Discussion]("http://ubuntuforums.org/forumdisplay.php?f=352") and read some of the threads, if that doesn't scare you off, then go for it. :)
