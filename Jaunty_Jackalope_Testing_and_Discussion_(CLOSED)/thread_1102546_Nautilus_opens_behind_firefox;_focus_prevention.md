### Post by jslinux on 2009-03-21
Hi,

Forgive me if this has already been reported. I have noticed that on my computer (when compiz is enabled, everything default) with the alphas, clicking places>home folder, etc, the file browser window will often open *behind* firefox, whether firefox is maximized or not. I figured the next compiz update would fix the problem, but it hasn't. My solution was to install CompizConfig Settings Manager and change focus prevention level from low to off, under general options>focus & raise behavior.    

Has anyone else noticed this behavior?
