### Post by spiderbatdad on 2009-04-05
the easiest way to upgrade without having to reconfigure you drivers, etc, and recommended method is to run:
```
sudo update-manager -d
```9.04 is running great on my acer. Very happy with the upgrade. If you run the above command look at the top of the window, should see new version available...click to install. Of course back up any media you don't want to take any chances with...that is, burn to disk or put on usb.
