### Post by khelben1979 on 2009-04-09
As an alternative to shutting it down or restarting the system you can go to a text console and do it:

```
ctrl + alt + f1
```
to get to the first text console

```
sudo shutdown -r now
``` or ```
 press ctrl + alt + delete
```
to reboot the computer

```
sudo shutdown -h now
```
to shutdown the system and turn off the computer
