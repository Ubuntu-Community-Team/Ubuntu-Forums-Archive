### Post by kansasnoob on 2009-04-06
Well, I figured it out .......... sort of!

I'd been using "private browsing" before I shut Shiretoko down and I guess it stayed there even after closing it last night, so it was still restarting in "private browsing" mode every time I'd restart it.

Not sure if that's normal behavior or not. I guess for now it would be a good idea to check before trying to save anything. Kind of odd though that what I saved showed up in Firefox 3.0.8.

I should add that I'm using the actual Mozilla build of 3.0.8 rather than the one from the repos. Shiretoko is from the repos.
