### Post by knarf on 2009-02-17
That's just Firefox getting stuck which makes it stop updating its window(s). As to why it gets stuck is another question. If a) this happens with the official Ubuntu Firefox package and b) you remember the URL this happened on and c) the effect is repeatable you may want to file a bug report @ launchpad:

```

$ pidof firefox
12345
$ ubuntu-bug -P 12345

```
