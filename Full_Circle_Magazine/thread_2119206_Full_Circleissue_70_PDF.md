### Post by howefield on 2013-02-23
Sounds like a Firefox issue in the way it is handling the link. I use Chromium, but using Firefox I can see that it opens the page rather than giving you the download option. 

Once Firefox has opened the page, you can use the download icon in the top right corner of the page to save as pdf.

PS. You could also open a terminal and use wget, eg

```
wget http://dl.fullcirclemagazine.org/issue70_en.pdf
```

### Post by howefield on 2013-02-23
Go into Firefox Edit Menu and select Preferences > Applications tab > search for pdf and change behaviour as required.

The default preference is to Preview in Firefox, change to Always ask or Save FIle.
