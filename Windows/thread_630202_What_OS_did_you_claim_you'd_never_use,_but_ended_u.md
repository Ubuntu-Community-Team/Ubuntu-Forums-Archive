### Post by xiota on 2007-12-03
I'm reviewing notes I wrote when I first tried Windows XP.

I didn't like the login screen, the window decorations, the new icons, the new color scheme.  I thought MS let color-blind people pick the colors.  (There are limits to non-discrimination policies.)  I didn't like the pointless changes to the user-interface's behavior.  Features I used didn't work the way they used to.  They could not be reconfigured.

I didn't like how difficult XP was to reconfigure.  Every version of Windows had something about it I didn't like, but before XP, they were fairly easy to reconfigure.

I didn't like product activation.  The profile migration tool didn't work.

I didn't like the reduced performance compared with Windows 2000.  I didn't like that some programs I used a lot no longer worked.  I didn't like that XP crashed more than 2000 (blue screens).  Some games also crashed in XP, but not in 2000.

MS eventually fixed many of the stability and compatibility problems and added new features (most of which I didn't like).  Computers got faster.

I finally switched to XP because I couldn't find Windows 2000 drivers for some hardware and some programs wouldn't work in Windows 2000.  I figured out how to disable features I didn't like and to reconfigure XP to be more like 2000.  The rest I learned to live with, except that every year, like clockwork, Windows deteriorated to the point where I needed to reinstall it.  I was sick of reinstalling.  I kept trying different Linux distributions, but none seemed to work for me.  Then I found Ubuntu.  Six months later, no more Windows.  Now, still no Windows.

I'm living happily ever after, except without a computer that crashes all the time, I've got all this time to kill and don't know what to do with it.  Any suggestions?
