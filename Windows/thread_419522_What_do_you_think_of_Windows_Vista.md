Overall if you have to use Windows it is better than XP if you have beefy enough hardware and don't rely on legacy apps. As a power user of Windows at work I'm looking forward to getting it up and running on my work desktop but at home I'll stick with Ubuntu.

-The gadgets set-up is uninspiring.  They should have copied the Yahoo widget engine, which has the best layout of them all in my opinion.

-Still missing a multi-desktop interface.

-The UI could be much more customizable.

-There is nothing ground breaking or revolutionary about Vista.  If you are satisfied with XP then there isn't much to compel you to shell out the $$$ for Vista.  

The Good:

-The native windows apps don't suck anymore (mostly).  Windows Media Player 11 has probably the best interface I've seen on a media program in a while.  Windows movie maker not only works now but is also intuitive.  IE 7, unlike IE 6, is a usable web browser (I still recommend firefox or opera).  Media center is an upgraded version of what was MS's only nice user interface from XP.  Defender is a decent anti-spyware program, the only problem is that it doesn't treat tracking cookies as spyware (which is why a lot of people think it doesn't work).

-Vista is fairly stable and bug-free.  So far I've had only one OS crash (happened while transferring files to a USB flash drive).  I've had a number of application crashes but they haven't taken the OS down with them.  The only bug I've noticed is that my gadgets slowly migrate to the left side of the screen a little more each time I turn on the computer.

-Vista runs very smooth on newer hardware.  Feed it 2gb of RAM and it flies.  The intelligent pre-caching works very well with enough memory.  All your most commonly used programs launch almost instantly.  I had Vista and XP in a dual boot on the same hardware and XP felt very sluggish by comparison.  Vista even boots in less time then Ubuntu on my system (granted when you hit the desktop in Ubuntu everything is already loaded while in vista some things are still loading).

-Instant search is well implemented.  Although when you first get your computer the whole OS will feel sluggish for the first few days while it indexes your files.

-Voice recognition works great. It's fun to try even if you don't need to use it.

-A lot of little things that were done wrong in XP have been done right in Vista.  The automated driver finder actually finds drivers.  Help topics and explanations are posted all over the settings menus and utilities to explain things to the novice user.  The event log is now usable by humans (this is a godsend when I have to do on-the-counter diagnostics).  The performance and stability monitor does a good job of allowing the average user to diagnose their own problems.  The whole UI is more logically organized (goodbye never-ending All Programs cascade, and good riddance).  Aero, for all of its resource related flaws, looks pretty good.


Conclusion:

Would I advise someone to switch to Vista?  That depends on their situation.  If you are trying to install it on anything less then an Athlon 3500 with 1gb of RAM then I would say absolutely not.  If you've got a modern system with a dual-core processor (low end pentium Ds don't count) and 1gb+ of RAM then go for it.  Of course make sure that all your mission-critical software and hardware is compatible (that applies to anyone upgrading to *any* operating system).  If you are buying a new computer then get one with Vista, it is a better operating system then XP (unless your budget limits you to a low end computer).  If you are upgrading then do a clean install.  Vista will clean install in 20-30 minutes, while it will take 2-3 hours to upgrade XP.  You're not saving any time going the upgrade route and you would be an idiot to not back up your data either way.  

Vista's biggest pitfalls are hardware requirements and compatibility.  Both of these problems will go away with time.  All new software and hardware will be Vista compatible.  Most old hardware will get Vista drivers (if they don't have it already) and some older software will be patched.  As for hardware requirements, well, we now sell computers with 3gb of memory for under $1,000 (CAD).  By Christmas 3gb will be mid-range.  Most people will be getting Vista when they get a new computer, and that is the best way to upgrade.
