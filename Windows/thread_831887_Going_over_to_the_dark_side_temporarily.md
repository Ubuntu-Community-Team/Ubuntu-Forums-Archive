### Post by ralyon on 2008-06-19
You don't have to install windows first, but it is a pain to install it afterwards and you always run the risk of windows overwriting something vital, but if your careful it shouldn't. The biggest problem is that windows will overwrite your boot loader with its own and there is no way to stop it. You have to go back and manually install grub or lilo after windows is installed to be able to boot into linux again. 2 other issues is that windows wants to install on the first partition and it can be difficult to install it elsewhere. Also windows won't resize or move any of your partitions no matter what the formatting so you have to set up a partition for it before you insert the windows install disk.

There is not really an advantage to using a different drive. If you know you want to install both you just have to set up the partitions either ahead of time or while your installing. Set up the first partition between 40 to 60GB depending on what software you want installed with NTFS, second one 10 to 20GB for Linux, a swap if you want one about the same size as your ram and then a forth the rest of the drive either fat32 or ntfs to store your files on so you can access them from both OSs.

ralyon
