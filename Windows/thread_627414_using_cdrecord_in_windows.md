### Post by snowboard975 on 2007-11-30
I didn't know I could run cdrecord in windows until today.

There is a win32 compiled version of cdrecord in the ftp server linked in [http://cdrecord.berlios.de/old/private/cdrecord.html](http://cdrecord.berlios.de/old/private/cdrecord.html)
I unzipped it into any folder that is in the path.
And I downloaded the wnaspi32.dll file from [ftp://ftp6.nero.com/wnaspi32.dll](ftp://ftp6.nero.com/wnaspi32.dll) and put it in c:\windows\system32 
then cdrecord just worked!

Thanks to the beautiful open source developers!
