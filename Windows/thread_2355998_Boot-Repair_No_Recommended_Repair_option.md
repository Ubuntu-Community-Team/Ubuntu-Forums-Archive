### Post by grahammechanical on 2017-03-18
You have 2 hard drives. sda = 250.1 GB. sdb = 1000.2 GB but Boot Repair cannot see any partitions on them. I have no experience with Windows 10 and the one time I installed Windows 8 preview it needed at least 1 partition on the drive to be formatted to a Windows format before the Installation would proceed. The Windows 10 install disk should have a utility that will let you format and partition the drives.
