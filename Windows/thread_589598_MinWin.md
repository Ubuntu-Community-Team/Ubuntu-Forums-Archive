### Post by NoSmokingBandit on 2007-10-24
Windows Vienna (windows 7) is under way. They are rebuilding the kernel as to get rid of a bunch of useless crap. Eric Traut has the core of the kernel done and demo'd it at the University of Illinois in a virtual machine. The kernel took up 40mb ram and only 25mb hdd space. Traut says he going to work to make it smaller yet.They have no gui, but there was a basic html support, so he could go online and show off he os. Vienna sounds like it might be a good step for microsoft if they dont screw it up.
