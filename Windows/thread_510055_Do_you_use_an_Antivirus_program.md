### Post by Sunforge on 2007-07-27
If it's a windows box anti-virus goes on by default if it's a coroporate machine. We use Trend Micro which has been a pretty good enterprise platform, although it's detection rate lags behind some products in the same market space. The advantage it has is that it's small, effective and keeps logs (I like logs and so do our auditors).

We have also tried NOD32 (good), Kaspersky (for control freaks), Computer Associates (Yak horrible), Symantec (a little nutty sometimes, brand recognition), Norton (good for home use, big brand recognition). I've heard good things about  AVG as well and know a few people that use this nifty piece of AV software. 

In a large corporate environment you cannot rely on common sense, contracts, policies or even a large baseball bat (only kidding) to keep the peace. You have to enforce common sense in a way that cannot be got around.

At home if it's a windows box (I have one and it's very shiny) it's got AV because I don't want to get caught having a blonde moment and live to regret it.
