### Post by fiddledd on 2008-09-10
Rather than answering you directly, do the following:

Create a shortcut to a folder

Double Click the shortcut

Bring up the context menu by clicking the right mouse button

Select New, and create a new text document

Close the Window

Now navigate to the actual Folder you created the shortcut to

There's the answer to your question.:)
