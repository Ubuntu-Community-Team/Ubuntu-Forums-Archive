### Post by Giant Speck on 2009-01-15
> **NoSmokingBandit said:**
> This is a good thing though. Win7 **caches **a lot to ram, like any OS with good memory management would. Its quicker to read explorer.exe from ram than hdd. If your ram isnt being used it is being wasted.

And that's the key word when it comes to RAM usage in Windows: cache.

In Windows, memory is used more like a cache rather than a resource.  And one of the most important rules of caching is that empty cache memory is wasted cache memory.  The empty cache memory isn't doing anything useful for your operating system.  It's just sitting there, wasting power by doing absolutely nothing constructive.  A cache will try to populate itself to maximum capacity with what it thinks is needed at the moment.
