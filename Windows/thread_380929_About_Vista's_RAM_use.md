### Post by izanbardprince on 2007-03-10
Not having much free RAM on Vista is not a bad thing, Windows tries to make use of all available memory for something, even if it's just a cache, it can run program data right out of the cache even, and dynamically give up cache RAM if it needs to, so "running out of RAM" is not a bad thing.

In task manager, add the "Cached" and "Free" megabytes, and thats how much RAM is actually available. :popcorn:
