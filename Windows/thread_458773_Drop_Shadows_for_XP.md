Hehe, so I was looking for something to spice up Windows for those few times when I do use it. I know there's stuff out there like Window Blinds that will skin everything for you, but I noticed that it didn't do drop shadows and I don't want to spend money. So I went looking and found this little program. [Y'z Shadow]("http://www.softpedia.com/get/System/OS-Enhancements/Yz-Shadow.shtml")
It makes menus semi-transparent and it gives you drop shadows under windows. Pretty nifty.:p
