### Post by DocHoliday52090 on 2008-07-05
Windows 2003 is compatible with 95% of programs that work on XP. The 5% is usually things like **_some _**antivirus and defragger programs (You would need a corporate edition). 

A handful of games don't work either. Rest assured, however, that the vast majority of software you can throw on 2003 will run perfectly. All the programs you've described work well for me (with the exception of NBA Live - I don't have it, but I can't imagine why it wouldn't work). 

There is a slight speed and stability improvement in 2003 inherent to its newer kernel as well. Though, if you really want to experience speed, grab yourself a copy of the free program nlite and rip out the server portions of 2003 (and install the workstation mod). 

The modification utility: [http://www.overclock.net/attachments/windows/30614-upgrade-windows-server-2003-xp-pro-win2003v145.rar?d=1156915193](http://www.overclock.net/attachments/windows/30614-upgrade-windows-server-2003-xp-pro-win2003v145.rar?d=1156915193)

A tutorial to tweak the OS to a workstation (some may have already been done by the utility): [http://win2k3.msfn.org/](http://win2k3.msfn.org/)

Remember, however, that you need a license to use this OS. Yes, there are cracks and pre-made workstation installs of 2003 (like the infamous eXperience release named Tiny2003 floating around torrent sites), but the fact remains that it is illegal to use this OS, or any proprietary non-free software without paying for it. 

Use at your own risk (unless you've bought the license!).
