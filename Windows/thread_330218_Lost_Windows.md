### Post by smoker on 2007-01-03
can you start the computer and enter safe mode (tapping f5 or f8 key during boot). if so, you may be able to log in as 'admistrator' this is a different account from a user with administrator rights, so if you haven't specifically set a password for this account, try just pressing enter. if you get in, then you can either go to control panel and sort out the user accounts, or probably easiest, try using system restore,

if the above doesn't work, you will maybe have to do a windows repair with your windows installation disk, this won't overwrite any data, but will require you to reinstall grub to get the dual boot function back.

it is always wise though, as barfbag suggested, to back up any important data,

best of luck
