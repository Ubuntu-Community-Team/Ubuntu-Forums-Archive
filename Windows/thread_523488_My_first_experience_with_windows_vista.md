### Post by Motoxrdude on 2007-08-12
Ok so with the release of halo 2 on the pc i decided it was time to upgrade from xp to vista. I headed over to newegg and ordered windows vista ultimate. I forked over about $380 for vista ultimate. I decided to get ultimate since it included all the features of every other versions of vista.

I tried to keep an open-mind when using vista, i did pay $380 for it after all. I popped in the dvd and a menu popped up and i selected install from within xp. I tried to do an upgrade but it came up with a huge list of programs that where incompatible with vista and needed to be removed. I decided it was best to do a clean install. I rebooted and selected and fresh install. The install went smoothly and looked beautiful. I walked away from the computer and watched some tv. I came back to a login screen. It looked very pleasing to the eye. I logged in and was bombarded with windows and applications. It was trying to give me a tour of vista while asking me to configure the widgets and windows security saying i have no firewall and antivirus. Not a very pleasent welcoming. After sorting through all of that i proceeded with installing programs.

The aero effects looked nice and i didn't seem to be having speed problems. So i downloaded firefox and installed it. My screen went black, then it came back and asked if i wanted to run the exe. This at the time didnt seem like a problem but after having to execute a few exe's it was getting extremely annoying. My monitor would have to Auto-config every time i would execute an exe. 

After getting settled i proceeded to whatch a few videos on youtube. In the middle of a video firefox closed along with all my other applications. Then vista logged me out and i was greeted with a screen saying "Apllying updates". What the hell? I never told it to run updates let alone log me out! What if i was in the middle of something important? O well.

I installed an anti-virus and i started to notice my computer lagging a lot. I started task manager and i saw that at desktop with avg running it was using 726mb of ram! This is rediculous, i dont want to be using virtual ram with just a few applications running!

After a few days of running vista i could not stand it anymore. It was getting exponentially slower with every application running.

I reinstalled ubuntu and left vista on there. Maybe when microsoft releases a service pack i will try it again but at this point i don't plan on using it.

For $380 you would expect an OS to at least work without much tweaking! I am currently happier with a free OS then one that costs $380. O well, looks like i wasted $380.

O yeah in case you are wondering my computer specs are as listed:
Amd X2 3800+
DDR2 667 1gb
Sata 3gb/s 320gb hdd
ati x800gto
