For that computer, use XP.

Windows 98:
[list]
[*] Is actually a version of MS-DOS with a GUI
[*] Uses the FAT32 file system (no security, and is more easily corrupted)
[*] Will run on lower spec computers, if you have less than 256 MB of RAM, Windows 98 will work.
[*] Because it is older and meant for older hardware, new hardware may not work well on it or at all (especially USB). 
[/list]

Windows XP:
[list]
[*] Is based on NT, not DOS.
[*] Uses the NTFS file system, more secure and reliable
[*] Will run with 256 MB of RAM, but will benefit from more.
[*] It is the most used Windows now, and has been in use since 2001. A lot of hardware and software works on XP.
[*] It is an old OS, but is well supported.
[/list]

### Post by darrelljon on 2008-06-10
I have made some rankings here sympathetic to Win98 because I think it gets too much bad press these days and raise awareness about what can be achieved with a tweaked Win98 system;

**Speed**
1. Win98
2. Damn Small Linux/Puppy Linux
3. WinXP

**Software compatibility**
1. Win98 + [KernelEx]("http://www.msfn.org/board/KernelEx-v034-released-t71476.html") (for NT/2K/XP software like NBALive2008)
2. WinXP
3. Linux+WINE

**Maximum RAM supported**
1. Linux (64Gb)
2. WinXP ([4Gb]("http://msdn2.microsoft.com/en-us/library/aa366778.aspx"))
3. Win98 ([512Mb - 1.5Gb]("http://support.microsoft.com/kb/304943")). [But have a look here]("http://www.msfn.org/board/Help-I-need-to-Get-2GB-installed-RAM-wo-t109574.html").

**Stability**
1. Linux
2. WinXP
3. Win98SE + [Unofficial MSFN.org SP3]("http://www.msfn.org/board/98-SE-SP-30-BETA-2-t61749.html")

**Security**
1. Linux
2. Win98[+Proxomitron+the free version of System Safety Monitor+Kerio]("http://www.msfn.org/board/Last-Versions-of-Software-for-Windows-98-t105936.html&view=findpost&p=767808#entry767808")
3. WinXP

You probably also want to think about hardware compatibility like WiFi and USB which is probably where Win98 falls down (there may be a maximum hard disk space too). Have a look at [this comparison I made]("http://puppylinux.org/wiki/archives/old-wikka-wikki/categorydocumentation/win98") and [this one too]("http://puppylinux.org/wiki/archives/old-wikka-wikki/categorydocumentation/windowsxpcomparison").

### Post by jviscosi on 2008-06-10
> **mac143_01 said:**
> how much memory can XP handle or recognize? if I want upgrade the memory to 4gig or into 8gig will XP recognize and detect all the memory the pc have?

32-Bit XP won't be able to address 8GB of memory, so you'd be wasting your money getting that much.  Even if you go to 4GB, my understanding is that the 512MB for the video card also comes out of that total 4GB addressable space, so you'll effectively only have 3.5GB of RAM.

If you're going 64-Bit XP you can address the 8GB, but ... good luck getting much software or hardware to work.
