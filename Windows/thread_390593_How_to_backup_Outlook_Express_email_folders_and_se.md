### Post by j_g on 2007-03-22
Select Outlook Express' "File -> Export -> messages..." menu item and choose where you want to save all of your messages. (Don't save the resulting file on your hard drive if you intend to wipe it clean when you reinstall. Save to something like a USB drive). This backs up your emails.

Select the "Tools -> Accounts" menu item. Click on the account you want to back up, and then click the "Export" button. This backs up that account.

Later after you reinstall Outlook Express, select the "File -> Import -> messages..." menu item, and locate that file you saved (on your USB drive, for example) to reload your messages. Select "File -> Mail Account settings" to reload your account info.
