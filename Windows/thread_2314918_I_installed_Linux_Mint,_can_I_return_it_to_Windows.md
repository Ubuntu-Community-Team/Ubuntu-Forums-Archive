### Post by QIII on 2016-02-24
If you have a legal key, you can download the installation ISO from Microsoft and install it using that key.  You may get some message during the installation saying that key has been used.  You might have to call Microsoft to clear that up, but they are actually pretty good about that.

The download pages:

[Windows 10]("https://www.microsoft.com/en-us/software-download/windows10")

[Windows 8.1]("https://www.microsoft.com/en-us/software-download/windows8ISO")

[Windows 7]("https://www.microsoft.com/en-us/software-download/windows7")
