### Post by jrib on 2008-02-06
One way is to use ies4linux.  Here's a wiki page, but the home page of the project has useful info as well: [https://help.ubuntu.com/community/InstallingInternetExplorer](https://help.ubuntu.com/community/InstallingInternetExplorer)

Another way is to install Windows in Virtual Box (or vmware or qemu) and run IE from there.  There are ways to set it up so that you can even run IE as a window on your ubuntu Desktop without it having to show up in the virtualized windows screen.  Here are some useful links:

[https://help.ubuntu.com/community/VirtualBox](https://help.ubuntu.com/community/VirtualBox)
[https://help.ubuntu.com/community/VMware](https://help.ubuntu.com/community/VMware)
[https://help.ubuntu.com/community/SeamlessVirtualization](https://help.ubuntu.com/community/SeamlessVirtualization)

Feel free to ask if you are not sure of how to do something
