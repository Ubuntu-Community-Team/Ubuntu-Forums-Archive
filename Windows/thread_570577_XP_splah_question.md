### Post by cogadh on 2007-10-08
It can look different depending on how the laptop's screen handles that resolution. For example, if the screen cannot run at that low of a resolution, the boot screen doesn't actually run at full screen, but will instead produce a 320X400 "window" to display the animated boot image, while the screen itself is at a different resolution (usually whatever the screen's default is). In that situation, the boot image will definitely appear "crisper", though it is also displayed much smaller.
