### Post by varunendra on 2014-06-30
If you mean booting Windows _Installation_, there are plenty of tools freely available for that, I personally use YUMI.

But if you mean running windows from an external drive, I doubt that is possible.

I believe MS intentionally prohibits that capability because that would potentially void their licence agreement (EULA) which allows running the OS only a fixed number (or range) of computers. Installing on a removable device means it is most probably going to be run on any number of computers (even though not simultaneously) anywhere, licensed or not (it is the 'copy' that is licensed, not the machines). Besides there would be the problem of hardware compatibility when the drive is connected to a radically different system, and MS avoids such troubles (better disable a feature instead of wasting time on explaining the problem to users - is their strategy).
