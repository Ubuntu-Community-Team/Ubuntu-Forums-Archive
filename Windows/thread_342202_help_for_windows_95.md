### Post by Frak on 2007-01-19
95 computers use a sort of Pre-Parallel ports, so It isn't accepted by many distributions (Ubuntu is one) because of its lack of respect, but if you can find the drivers go ahead.

And as JDHore stated above, there wasn't much need for personal computers to run on a network back then, so there is no networking support.
