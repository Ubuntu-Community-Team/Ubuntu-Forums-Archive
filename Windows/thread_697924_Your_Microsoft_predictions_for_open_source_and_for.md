### Post by Midwest-Linux on 2008-02-15
I am predicting Internet Explorer will go toward the way of open source and/or that a Linux version will be available. Already MS has dropped passing the WGA for the latest update. Linux would be the next logical course for IE. 

 I can see at some point a Windows Media Linux version being a possibility though there are fine Linux programs that can do audio and video already.

 MS has to take the stance of getting on the train or being left at the station. I believe Linux is MS biggest competition as Linux is free and how can MS compete against that?

 The biggest advance MS can ever make is to make a truly Wine type program that will run any Windows Program (like Wine but better) in Linux. MS could sell it for less than $30 and make money off Linux users. MS could make a decent profit from that, but run the risk of cutting into their Windows market share by doing this.

Lastly, if MS released a Linux Operating System on their own that could run any Windows program on it, sell it for less than $50. It would keep MS in the game. Yes surely a $50 Operating system would hardly be called "open source", yet it could be a step in a new possibly good direction for everyone in the long run.

 On the other hand if MS continues to release bloated DRM infested expensive operating systems that require high hardware requirements then they are only hurting themselves in the long run.

 What are your predictions of Microsoft in regard with Open Source or Linux?
