### Post by kulturloseramerikaner on 2007-10-19
Ugh, you're gonna make me boot to Windoze so I can remember exactly how to run through this. :) The first thing I'd check for is to see if Nero has messed with your file associations; every Windows app I've ever installed seems to want to make itself the primary app for everything and quite often it changes the default program that runs a particular file.
Open up the folder containing your MP3's and right-click on one of them.  Then hit open with > select program.  Underneath the box containing your choices is a little checkbox that says "Always use the selected program for this kind of file," make sure that is checked, then choose your favorite player.
Give that a try then let us know what that does for you.
