### Post by rockface on 2008-08-04
The built in Windows firewall did not prevent outgoing connections that used Microsoft programmes by default (Internet Explorer, Windows Media Player, Outlook etc). So the virus and script kiddies targeted these programme users with a little social engineering (did not help when Microsoft, in their infinate wisdom, hid the file extension).

With the advent of service pack two for XP they introduced a slightly better firewall with a little more flexibility. But this was still worse than even the free versions of Zone Alarm and others.

Though service pack three is available for Windows XP my advice would be to get to know your hardware firewall and not rely on the Microsoft version.

Would you trust a prison policed by it's inmates?
