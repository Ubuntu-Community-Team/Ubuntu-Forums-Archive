### Post by edoviak on 2007-08-30
Just for fun ...

After reading that [COLOR="Blue"][Microsoft's Windows Genuine Advantage (WGA) server crashed last weekend]("http://arstechnica.com/news.ars/post/20070825-windows-genuine-advantage-suffers-worldwide-outage-problems-galore.html") [/COLOR]I decided to try to validate Kubuntu using [ the WGA site.](http://www.microsoft.com/genuine/downloads/Validate.aspx?displaylang=en&sGuid=8cd6cfb6-0df4-43f7-a9ff-b0db99a3e0a1) 

Yes. It's stupid. That's the point! ... Or maybe it isn't stupid. What would happen if GNU/Linux users starting bombarding the WGA servers with requests for validation? Would Microsoft suddenly discover widespread piracy of the 235 patents that GNU/Linux infringes upon?

Of course, I've run into a few problems while validating my copy of Kubuntu on the Microsoft WGA server. I don't use Wine (it isn't even installed), so I have no way of making the EXE plugin work. I also tried  [this alternate validation method.](http://www.microsoft.com/genuine/downloads/RunHTA.aspx?displaylang=en&sGuid=8cd6cfb6-0df4-43f7-a9ff-b0db99a3e0a1&Page=Validate) It asks you to open a [ legitcheck.hta](http://www.microsoft.com/genuine/downloads/GenerateHTA.aspx?displaylang=en&sGuid=8cd6cfb6-0df4-43f7-a9ff-b0db99a3e0a1&Page=Validate) page, but whenever I try to open that page with Firefox or Konqueror it just opens another browser tab and repeats the process.

If anyone has more luck, please let me know. This is a very amusing way of wasting time!
