### Post by buzzingrobot on 2016-04-16
It's only available now via that preview release build which is only available via the Insiders Program.

The Insiders Program, as Microsoft makes very clear when you sign up to participate, provides pre-release Windows software to developers and users who volunteer as testers.

Microsoft has said it hopes to release the Windows Linux Subsystem this summer, apparently to coincide with the first anniversary of Windows 10 in July.

If you do not want these Insider preview builds pushed to your system, you should consider disabling that or terminating your participation.
