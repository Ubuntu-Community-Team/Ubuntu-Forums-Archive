### Post by Paqman on 2008-10-20
I would **strongly** advise against installing an OS you've downloaded from a torrent unless you know the MD5 hash so you can check it's integrity. Installing an OS from an untrusted source is a huge security risk if you can't verify that it hasn't been tampered with. 

You need to get a valid key anyway, and the easiest way to do that is buy a valid copy. There are volume license keys around, but using one isn't likely to be legal.

### Post by Hexagoon on 2008-10-20
assuming you have a valid xp key, i will answer the question :)

There is no way to successfully downgrade xp sp3 to a "faster" version of windows. You will simply have to backup all files and do a clean install of windows 2000, or 98 if the computer is an awful lot of old (absolutely not recommended without installing a good antivirus and firewall before connecting to the internet, and after that install every possible update available).

Anyway, good luck!
