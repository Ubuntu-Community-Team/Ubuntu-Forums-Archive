### Post by darksong on 2007-03-22
If you can find out where the files are stored you could either:

1- Back it up to another pc  (I recommend a non XP machine as you run the risk of getting that virus on the other machine) set you infected hard drive as slave and the other computers one as master - then transfer the file to the working machines hard drive.

2- Dual boot it - put a linux on there and mount the hard drive, copy the files across and burn them to disk, make sure you DO NOT overwrite your windows XP install, many linuxes will give you the option of changing the partition size, do this. 

Hope this helps you in someway.

### Post by DreadPirateRoby on 2007-03-29
It's likely that some system files you may need got removed along with the virus/es. This could happen if a virus has imbeded it's self in such a way that it can't be removed without taking those files with it or it could be due to your anti virus removing extra files it didn't need to. 

See if your anti virus can restore removed files and if it can run an online virus check after doing so.
[http://housecall.trendmicro.com/]("http://housecall.trendmicro.com/")
[Pandasoft Activescan/]("http://www.pandasoftware.com/products/activescan/")

If that doesn't work or give you better results I've found this in microsofts support pages.
[http://support.microsoft.com/kb/270670]("http://support.microsoft.com/kb/270670")

Once you got everything working again make sure you:
-Install a Firewall or use a router if you're not already
-**Run windows update if you can**
