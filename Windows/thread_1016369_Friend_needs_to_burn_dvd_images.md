### Post by pytheas22 on 2008-12-19
DVDstyler is the most intuitive application for creating video DVDs, in my opinion.  You need to [add the repositories for it]("http://dvdstyler.wiki.sourceforge.net/InstallationDebian+") in order to install it on Ubuntu, but that's not too hard.

[tovid]("http://tovid.wikia.com/wiki/Tovid_Wiki") is also nice and pretty powerful, but it's command-line only ([here]("http://www.arsgeek.com/2006/08/08/how-to-burn-a-dvd-in-ubuntu-linux-and-other-distros/") are decent instructions for it though).  There's a GUI but I don't find it very easy to use and it seems to have a lot of bugs.

EDIT: oh, you're probably asking for Windows applications, I see (just noticed which sub-forum this is in).  In that case, DVDstyler is available for Windows as well.
