### Post by p_quarles on 2008-03-18
Moved to Windows Discussions.

With OEM copies of Windows XP, the EULA forbids installing it another machine, so isn't supported here. With "off-the-shelf" copies, you usually have the right to keep the OS installed on one computer at a time. The bottom line is that you should read the license agreement. You can boot up the disk to do so, as you will be presented with a page which requires you to read and accept the license before you can proceed.
