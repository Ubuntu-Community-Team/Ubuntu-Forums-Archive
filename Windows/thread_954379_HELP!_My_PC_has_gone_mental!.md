### Post by mips on 2008-10-21
I have asked the mods to move your thread to the Windows Discussions section here [http://ubuntuforums.org/forumdisplay.php?f=170](http://ubuntuforums.org/forumdisplay.php?f=170)

If you have a cd or .iso of your system restore I recommend you run it through [nLite]("http://www.nliteos.com/") and remove all that extra crap/spamware/norton/yahoo toolbars etc. Also slipstream SP3 into the system restore cd. This way you have a nice clean install without all the extra nonsense they sell PCs with these days. I managed to cut an XP install cd to under 250MB as I even removed windows components like IE, notebad, languages, printer drivers, some network drivers etc. You don't have to go that far though.

You could also try a linux distro like ubuntu. Pro Tools will not run natively in linux or in WINE. You could however install Virtualbox in Ubuntu and then create a XP install in Virtualbox where you could install Pro Tools, I'm not saying it will work but it is worth a try.
