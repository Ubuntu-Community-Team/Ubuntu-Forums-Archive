### Post by jan quark on 2008-02-19
to register a new nickname write into the freenod channel, or the equivalent server channel:

```
/msg nickserv register username
```

to set a new password write into the freenod channel, or the equivalent server channel:

```
/msg nickserv set password yourpassword
```

to identify your username write into the freenod channel, or the equivalent server channel:

```
/msg nickserv identify password
```

to join a new chanel write into the freenod channel, or the equivalent server channel:
```

/join #name of channel
```
