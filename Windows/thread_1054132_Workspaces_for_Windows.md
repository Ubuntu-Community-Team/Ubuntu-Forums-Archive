### Post by fsamuels on 2009-01-29
I run 2 machines on a daily basis (Win XP and Ubuntu 8.10) and I have begun to get very frustrated with Windows not having the concept of workspaces like GNOME does.

I have tried the Windows XP Power Toys "Virtual Desktop Manager" but it doesn't work the same way. All the running applications still show up in the task bar and if I remember correctly clicking on one brings it up in the current workspace regardless of where you are. I also tried their newer "Desktops" but my system apparently can't handle it (dual monitors maybe?). I also tried VirtuaWin which is much closer to what I need but I am hoping there is something better.

I found a list on Wikipedia ([http://en.wikipedia.org/wiki/Virtual_desktop#Windows](http://en.wikipedia.org/wiki/Virtual_desktop#Windows)) and tried a few but they were still not what I was looking for. I really want something as close to the way GNOME works as possible.

Does anyone have any experience running a workspaces add on for Windows that they like? I'd prefer to get suggestions from people who have actually used them over spending all day installing, rebooting and uninstalling to try them.
