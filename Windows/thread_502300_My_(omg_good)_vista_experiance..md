### Post by screaminj3sus on 2007-07-16
Well I've been seeing many threads saying how horrible Windows Vista is. I really like ubuntu but unfortunately I have some unfixable problems at the moment (Gutsy is looking amazing though). I got myself a copy of Vista Ultimate to install on my PC, which is aging (Socket 478 Pentium 4 3.0E HT, 1 GB RAM, 160 Gb HDD, GeForce 7600GS).

**Install:**

I popped in the DVD and it started pretty quickly, the install went flawlessly, and was very easy (Much better installer than XP, although I still like ubuntu's installer better) It took a while,but worked great. 

**First Impression:**

After the install finished and I booted into my desktop I was very pleased, It had automatically configured pretty much all of my hardware, I logged in and apparently it installed Nvidia drivers, it was automatically set to my widescreen resolution, and Aero was running great, It detected my Dell A920 printer and automatically got it working, along with my sound card. I have never had an OS have everything working like that at first boot I was very impressed.

**Problems:**

 I did have a couple problems, first I installed steam and started up Counterstrike Source, and My frame rates did not even exceed 20. I solved this easily by installing the latest nvidia drivers. My other problem was my sound would skip using windows media player at the beginning and end of songs, and it would use a crapload of CPU, I fixed this by turning off enhancements on my soundcard driver, but I still had the high cpu usage problem. In windows XP I could not use hyperthreading because many of my applications would randomly freeze (Paint.NET, pidgin, photoshop, Opera) So I had it disabled. I enabled it and in vista it worked perfectly, and for some reason fixed my cpu usage problem.

**Speed:**

I have seen many people saying " OMG YOU NEED QUAD CORE AND 4 GB OF RAM TO RUN VISTA!!!" and I really don't find that true. My system runs vista great with 1 GB of RAM, I have not had any problem when multitasking or gaming, vista actually seems much more responsive than XP. My framerates in games are the same as XP using the latest nvidia drivers. I have also seen many complaints about boot time, but for me after using it for months it still only takes 35-40 seconds to boot (Same as XP)

**Features/UI:**

Windows Vista has many new features and an improved UI. Aero is nice and runs very smooth compared to Beryl/Compiz in my experience. Vista's search is a great feature, I can always find what I'm looking for instantly. Some newer/updated elements of the UI I like are The redesigned start menu, and Windows Explorer. The New start menu feels much more natural with it's expandable tree view, and the search is very convenient. I also like some updated things in explorer like the breadcrumb view (Which gnome does already have), the better folder thumbnails, and the integrated search bar.The windows sidebar is also Nice and some useful gadgets can be found. The bundled apps are also improved such as the new photo gallery.

**Security:**

UAC can be incredibly annoying, and I much prefer Linux's implementation, but I have not had any security problems... so far.

Networking:

Networking worked great, it automatically detected all My XP machines and I could navigate them. My XP machines did have problem networking with Vista though, When I used the wizard to add a network place, it simply would not add vista, I solved this by right clicking the vista machine and hitting explore before trying to add it, it prompted me for my username and pass. After doing that I could add it.

**The Bad:**

UAC can be very obtrusive and annoying, and it seems Microsoft did not bother to give it a nice animation or anything your screen just abruptly turns black and about scares the **** out of you. It often prompts for unnecessary things.

Vista takes a lot of hard drive space, the default install took about 15 gigs of my drive. Also Vista's System restore and shadow Copies, Take up WAY more space than XP's. They will accumulate gigs and gigs of space.

The defragmenter, which now has a nice scheduling feature has one huge downside, there is no progessbar or anything, you don't know what the hell it is doing and it takes hours to defragment, compared to the XP one which only took about 10 minutes.

Nvidia drivers are still not up to par with XP's. Some software incompatibilities.

Overall I think it is a pretty good OS, It has been rock solid and stable for me, and has many improvements, it may not be good for older machines but it does not need the ridiculous hardware I've been seeing people say. It definitely has it's strengths and weaknesses but I think it is a good upgrade over XP.

### Post by adrianelliott on 2007-07-22
I used Vista for 3 months and can only describe the experience as appalling. 

I don't consider myself to be a true Microsoft hater, becase I rate both XP pro and 2003 server as very good operating systems. 

Unfortunately Vista is a step backwards.  

**Performance**
It is clunky and slow  - on my PC which is a E6600 dual core with 4gb of ram and an 8800GTS gfx card I expected a smooth ride.  How wrong I was! Applications continuously freezing for 5 seconds whenever I tried to multi task, network browsing even slower than on XP.  To top it off, the stupid OS actually pulled a blue screen of death on me! You would have thought that M$ 
would have at least worked out a way of giving more meaningful error messages to the user. 

**GUI**
The core system utilities were either identical to XP's or lacked polish. Option settings were burried under even more icons and layers of functionality than in XP, and overall the UI was a mess.  Take for example the control panel.  In standard mode, there is basically nothing useful on it. In 'classic' mode it is so full of icons you cannot find anything without putting your finger to the screen and counting through them one by one.

Exploring the file system is also rediculous, after installing a few programs I had multiple documents directories.  Finding the root of your hard disk is a nightmare. Overall it seems like the GUI was designed by people who just didn't get along, there is no consistency whatsoever. 

Last but not least, the desktop looks awful, Icons are over sized and childish even at high resolutions. They have an awful lit up square effect when you mouse-over them which looks amateur and cheap.

**New Features**

Even worse is the new "search" innovation.  Every window has a built in search bar which takes up valuable desktop space. When you type something in, you don't know whether it will search text on the page, files on the disk, options in the program etc.  This makes it totally useless.

The Aero theme is disappointing.  It just doesn't look professional at all, its the sort of setting that you would quickly move on from in Compiz.  Part of the problem is that the window borders are so big, its a waste of screen real estate as well as CPU cycles. Also, I find that even the built in programs like internet explorer do not fit in with Aero, there is a hard edge where the porgram meets the Aero panel. Some programs like MSN Live Messenger don't even fit in with the theme and stick out on the desktop like a sore thumb.  

What's worse is that you can't customize it - your stuck with the cheesy rounded borders and the awful semi-translucent smear patterns.  Mac OS looks far better, as well as several of the themes I have tried in Compiz. 

**Power User Features**

The command prompt is still next to useless. I downloaded the optional powershell, this has potential but is still rough round the edges.  

IIS is included in Ultimate. It is a relatively new version and as such is next to useless, you would be crazy to host anything on an unproven platform.

RDP sessions are locked to 1, despite the fact that I paid for the "Ultimate" edition. Come on M$, this is pathetic - it wouldn't have hurt to let me RDP in while I was logged in on the keyboard!

Putting the system into "classic" mode is not as great a move as it was in XP.  The start menu and icons look funny without the curvy windows around them which is a shame - it will be interesting to see how this situation is improved for Windows Server 2008.

**Ultimate Extras**

Vista Ultimate Extras are a real surprise.  The surprise being that there aren't any!  All I got was a buggy 'preview' desktop animated wallpaper.  

This is compounded by the total lack of bundled software.  There is nothing new to see here, just the same old crap that came with XP pretty much. 

**Good Points**
UAC is a good idea, as long as the user doesn't figure out how to disable it.  File copying operations seemed more reliable than in XP.  

Including Windows Defender is a decent move and may decrease the number of Zombie MS systems on the internet sending spam.

The start menu is a work of art and a vast improvement over previous versions of Windows.  This alone was the best feature and possible only truly exciting thing about vista.  Unfortunately it has a cheesy "Web 2.0" style light effect on it, which means it looks terrible in any other location than the bottom of the screen.

All in all Vista is a joke.  The best MS operating system is still 2003 server even for home use due to its stability and speed.  XP pro is also very usable providing you work to keep the system clean of spyware and buggy programs.  Maybe when it hits service pack 2, vista will improve dramatically, until then stay well clear.
