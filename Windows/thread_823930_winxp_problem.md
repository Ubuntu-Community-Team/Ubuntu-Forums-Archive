### Post by smoker on 2008-06-09
you don't say if it is a ps2 or usb mouse, or a wireless mouse, but first i would check the driver. if you go to device manager, and click the hardware button, and highlight the mouse with a right click and go to properties. a box should appear with an option to update driver for this device. i would try that first, then restart (don't unplug and replug a ps2 mouse while windows is running!)

if it is a usb mouse, try in a different usb port, and if a wireless mouse, change the battery if possible, and try again.

best of luck
