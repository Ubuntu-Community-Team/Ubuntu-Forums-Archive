### Post by smoker on 2007-08-25
hi, to get the stop error, it must be booting past the bios. try starting the pc in safe mode, in the safe mode boot options, first try 'last known good configuration'. if that doesn't work, try the 'safe boot' option, and then try a 'system restore' to a date before the first stop error. if the stop error occurred after installing a new app, or a new piece of hardware, then uninstall or disconnect that.
info about safe mode here: [http://support.microsoft.com/kb/315222/](http://support.microsoft.com/kb/315222/)

if you have data to back up, if you have another pc at hand, remove the drive and slave it to the other computer and copy the data that way. remember to set the jumper to the right position if you do this.

best of luck
