### Post by I-75 on 2009-01-12
AFAIK (as far as I know). Windows XP will be supported for a long time. This also means help with keys, reactivation, fresh install and new installs. 

 You should have the Serial sticker on your computer on the bottom of the laptop. If you have a legit install disc, simply use the serial key on the sticker with the legit install disc for a reinstall. 

 If the sticker is scratched or not all that clear, download BELARC advisor. Run the program and it will tell you all your serial keys you computer uses. 

Write down those numbers or print them out and use the Windows key it found to reinstall your OS. 

 Other advice, if your partition is going, you might want to install a new hard drive while you do a new installation. Chances are if its a older laptop it has a PATA HD. 

 In most recent laptops, the hard drive is located on the bottom of the laptop. Usually three or four screws holds the cover on. On my DV 4000 laptop, after removing the cover I merely had to slide the hard drive out.

Other advice, also try to note what drivers your computer uses. You might have to install them separately as chances are Windows XP may not have drivers for your computer.

 If you need a XP computer and don't want to go through the hassle of reinstalling or buying a new hard drive. You can find a low end refurbished XP computer from E-Bay, Geeks.com or Micro Center...which was selling refurbs for as low as $99.

Keep us posted on how it all worked out.
