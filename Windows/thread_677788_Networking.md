### Post by pelle.k on 2008-01-25
first of all, enter "device manager" and make sure the ethernet card is installed properly. otherwise download the correct drivers (through ubuntu) and install them. Also make sure you've got all the networking components installed through "add/remove programs" (there should be a feature there from which you can add remove core windows components, such as networking, tcp/ip, qos packet scheudler etc). You'll need to have the windows install cd to complete this step if i remeber correctly.
You can also complete this step if you right click "network" and select "properties", and use one of the menus to get to the list of protocols/services etc, and from there install tcp/ip and qos...

Good luck.
