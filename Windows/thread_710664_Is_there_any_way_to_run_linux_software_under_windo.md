### Post by bilijoe on 2008-03-29
A product called Crossover appears to be an excellent solution for many of the most common Windows programs.  It doesn't "support" all Windows software, but even many unsupported programs seem to run just fine.  The supported programs cover everything I'm really interested in--everything else has a Linux counterpart.  Also the makers of Crossover have pledged to support any Windows program for which they get a reasonable ammount of requests.  To find out more, go to [www.codeweavers.com](www.codeweavers.com).  It's not a free product, but the cost is very reasonable ($40).  So that makes the cost of a version of Linux that will run Windows software $40.  Pretty cheap, compared to the cost of windows.  I'm not sure of their methodology, but my Windows programs (I've used WORD and EXCEL for years and years, and so wanted to stick with them) seem to run faster (or at least as fast) under Linux as they do under Windows.  And they never crash!  Anyway, if you just can't bring yourself to give up some of your favorite Windows apps, I suggest you check it out.  There's a 30 day free trial period, so you have nothing to loose.  I am very happy with it.
