### Post by oldrocker99 on 2016-09-24
Welcome to the forums!

The DVD has its own boot sector, and Windows' apparent fouling up of your BIOS is a Windows problem. If you have installed Ubuntu and can't boot into it. you have a problem with the boot sector on the hard drive.

Here's the official help for boot sector problems:[https://help.ubuntu.com/community/Boot-Repair](https://help.ubuntu.com/community/Boot-Repair)
