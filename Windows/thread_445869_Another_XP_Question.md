Check to make sure all the XP system files are okay. Heres how:

Start => Programs => Accessories => System Tools => System Information => Tools => File Signature Verification Utility => Start

Let that run (it can take awhile), then restart and see if the problem is fixed.  Windows, as with all software, deteriorates over time (it's called bitrot).  If this doesn't help, backup all your inmportant data and reinstall.

hi, not sure if the vaio has the fan(s) underneath, or vents to the side, but a common problem with overheating is when people, eg, put the laptop on their knees, or a cushion, of something that blocks the passage of cooling air. if a hard drive has been running too hot, for too long, then it can cause trouble such as you have described. sometimes this is fixable, sometimes not, the important thing is to find out if this is indeed the cause. ideally, if you  have a spare drive around, you could swap it, install an os, and if all runs ok over a period of time, then you know the original drive was the problem. if you haven't such a spare working drive around then you can use some diagnostic tools to examine the drive in the laptop. hdtune below is a free download and can be installed, several checks can be done, and you can have the temp of the drive monitored from your taskbar. available from here:
[http://www.hdtune.com/](http://www.hdtune.com/)

another great tool is 'drive fitness test' which you download and burn to a bootable floppy or cd, it is available here:
[http://www.hitachigst.com/hdd/support/download.htm#DFT](http://www.hitachigst.com/hdd/support/download.htm#DFT)

best of luck
