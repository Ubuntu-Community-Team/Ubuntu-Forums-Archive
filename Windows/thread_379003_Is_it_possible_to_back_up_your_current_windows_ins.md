### Post by coffeecat on 2007-03-08
If you don't mind paying for commercial software, I can recommend Acronis True Image.  It's up to version 10 now, but I've got either 6 or 7 and it works very well. It can do either clones to another hard disk or compressed images to usb hd or cd/dvd. If you need to restore a compressed image, you boot up with a special CD you make with Acronis (I believe it's Linux based) and that does the restoring. When I needed to install a larger-capacity hard drive in my laptop, I made a Windows image with Acronis and tar.gz files of the Linux partitions, and soon had my multiboot transferred onto the new HD.

So long as you are using Acronis to clone or backup/restore onto the same machine I can't see that you are falling foul of the Windows EULA.

If you can find a special offer (as I was able to), it's worth getting Acronis Disk Director as well. It's a good partitioning tool which can also deal with Linux filesystems.
