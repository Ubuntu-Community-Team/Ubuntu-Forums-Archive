### Post by twin_57103 on 2008-05-31
It took me several weeks after Vista SP1 came out before I was prompted to install it through Windows update.  When I did, I had no problems.  I never had any errors with Windows Explorer - installed SP1 simply as an update.  

I'm using 8.04 (originally beta, now main release) with the default install of GRUB (specs in sig).  I have also used a downloaded version (i.e. not Windows update) to install SP1 on a friend's computer, but that was a Windows-only system.

Backups are always a good idea.  If GRUB does get corrupted, you can use [Super Grub Disk]("http://www.supergrubdisk.org/") or [Puppy Linux]("http://www.puppylinux.org/") to reinstall/fix it (rarely used them on my main system, but I've played with them on a test box).

Hope that helps!
