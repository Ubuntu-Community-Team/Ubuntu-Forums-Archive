### Post by chrismine on 2007-09-08
It is stored as .dbx files under Documents and Settings/name of user/local settings/application data/identities/number - best is to do a search for .dbx 
You must do a import from Outlook Express from the OE store directory option and navigate to the identities folder. Your address book is stored as a .wab file Just search for that too and do import address book option. Do not copy the data over to the new user.

Hope this helps
