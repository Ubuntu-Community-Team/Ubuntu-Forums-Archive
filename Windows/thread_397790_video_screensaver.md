### Post by Co.Sinecure on 2007-04-04
Dunno if this would help, but if you can find a program that will convert the mov to an exe, you can rename the *.exe to *.scr. 
Screensavers are apparently just an executable (usually with no user interaction), and i've done this a while ago with a little driving demo video that was an exe.

Good luck! :-D
