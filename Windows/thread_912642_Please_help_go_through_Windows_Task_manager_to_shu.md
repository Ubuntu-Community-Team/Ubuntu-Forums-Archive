### Post by Erdaron on 2008-09-07
If you google names of the executables, you can probably find descriptions of what they do.

It's kind of a judgment call, since they all carry *some* functionality, even if it's trite. For example, I don't use all those google apps (picasa, desktop search, etc.), so I got rid of them, but some people rely heavily on them.

If you go to Task Manager (ctr-alt-del), and go to Processes tab, how many processes are running?

If you run msconfig, what processes are listed in the startup tab?
