### Post by yancek on 2014-06-29
> Any idea how to make ubuntu boot a usb drive with windows on it?         

Ubuntu being on the machine in your case is irrelevant.  Ubuntu is on the drive and what you need to verify is whether this machine is capable of booting from a usb.  Enter the BIOS by watching the screen on boot.  You should see a message, usually at the very bottom of the screen, telling you which key to tap to enter "Setup".  Common keys are F2, F10, Del but you will have to check yourself as it varies a lot by hardware.  Then see if you can find an option to boot from usb, probably under the Boot tab.

What happens when you try to boot windows from the usb?  and what version of windows are you using?  Do you see any on-screen messages?

Unsure why you want to do that. You can't install Windows that way. You need to boot the machine, get into the BIOS or the boot device menu and choose to boot from the USB. Nothing to do with Ubuntu.
