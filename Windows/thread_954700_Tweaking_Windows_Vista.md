### Post by fiddledd on 2008-10-21
Look [here](http://www.speedyvista.com/) to speed up Vista. There's quite a lot of services you can disable.

Security is quite good as it is, but I use the following:

[ClamWin](http://www.clamwin.com/) it's FOSS. It's not a resident scanner, so no resources used. I scan new files I download manually, and do a full scan once a week.

I don't use Windows Firewall, I use Zonealarm Free.

I also use Windows Defender, which is part of Vista. It does it's job well, monitors any attempted system wide changes, checks for Spyware and Malware. It also scans once each day, doesn't take long.

I've used Vista for 8 months, and never found any Malware, and I used other scanners to check nothing sneaked in.

EDIT: If you need some reassurance and feel like reading, [here](http://technet.microsoft.com/en-gb/magazine/cc162480.aspx) is some technical info on Vista's kernel.

### Post by fiddledd on 2008-10-22
A few things I thought of that I didn't mention yesterday:

As @smoker said, more memory is always better, but with Vista Basic (that doesn't have Aero) 1 gb will be OK unless it's shared gfx. You could end up losing 384mb to the integrated gfx.

I also forgot to mention I use Firefox with the Noscript extension. However, if your mother needs to use IE for banking sites, it's sandboxed, so it's not as insecure as previous versions.

As well as obviously setting up a LUA you might want to think about using [Parental Controls](http://www.microsoft.com/windows/windows-vista/features/parental-controls.aspx) to limit even more what your mother can access. You have a fair bit of control, so you can protect her from herself. As a by-product you'll also know exactly what actions she has taken, and what websites were visted.:)
