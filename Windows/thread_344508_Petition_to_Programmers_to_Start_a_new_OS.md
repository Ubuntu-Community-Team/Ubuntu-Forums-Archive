### Post by mvaniersel on 2007-01-23
XAML, .net framework, artificial intelligence, gene therapy, That sounds like an awful lot of buzzwords piled together. 

Of course microsoft is putting out these visions, that is expected from the industry leader. But at the same time they couldn't pull off a decent desktop search while they have been talking about that for a decade. I'm a bit sceptical about the ability of either microsoft or the open source world to pull your visions magically together and make it a reality.

If some group could pull it off eventually, more power to them. But what I don't think you understand very well is that the open source world is a meritocracy: those who write the code get to decide on what to implement. So petitioning programmers to follow a certain vision (especially if that vision is rather vague) is an exercise in futility.

In other words: less talking, more coding.

Also (and this is speaking as a PhD student in bioinformatics), I don't think lack of computing power is what is holding back genetic research. Rather, integration and standardization are important areas. But that is a different matter.
