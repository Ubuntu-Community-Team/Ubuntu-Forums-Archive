### Post by grahammechanical on 2016-05-13
> I read that it's important to update bios if you want to install Ubuntu. 

Where did you read that? My machine dates from 2007 and I have installed every version of Ubuntu on it since then and I have never updated the BIOS. I have no need to update the BIOS because all the hardware works fine.

Whether you update the BIOS is up to you. You should follow the motherboard manufacturer's instructions. People have been installing Ubuntu on Windows 8 & Windows 10 machines since the day the machine came on the market and I have never seen any recommendation to update the BIOS before trying to install Ubuntu.

I have seen many posts from people who had problems. Some may have had problems that were specific to Acer machines. But updating the BIOS was never offered as a solution. These recommendations still apply when installing Ubuntu on a Windows 8/10 machine.

[https://help.ubuntu.com/community/UEFI](https://help.ubuntu.com/community/UEFI)

Regards.
