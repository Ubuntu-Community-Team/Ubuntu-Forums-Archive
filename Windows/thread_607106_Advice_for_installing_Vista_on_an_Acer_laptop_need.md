### Post by igknighted on 2007-11-13
> **inversekinetix said:**
> Just install XP for her, it will be less headaches than vista for sure.

Not necessarily true.  XP, over its life, will require frequent reinstalls as it fragments and its trillions of security holes let all sorts of malware come in.  If her hardware is fairly Vista compatible, Vista is much less of a pain due to its greatly improved security.

As for the OP's questions:

1) Don't bother.  You could set this up by creating a separate partition at install, and then just putting things in d:/ instead of f:/users/blah blah blah, but it really isn't worth it.  If you need to do a reinstall then create a partition to save data and drag/drop what you need onto it at a later time.

2) Nope, just tell it what drive to use and it will do its thing.  I have no "recovery partitions" or anything like that on any of my vista computers, and there have been no issues.  I suppose if you find driver issues in the default (as in you had to track them manually) it might be easier to take the install image so should you have to reinstall you wouldn't have to do that again... or just create a CD with the drivers you needed and keep it with the fresh install disk.  #2 sounds easier, and I'm lazy so thats what I'd do :)

Despite what you hear on this forum (from totally unbiased users, no doubt), Vista is actually a great step up for Microsoft.  Listen to the list of complaints and see if it sounds familiar to some other OS we know... It does have issues with not supporting all hardware fully (manufacturer's fault, not providing decent drivers), it has bugs here and there, the security access can get annoying... I mean, it is by no means perfect, and I still prefer the linux desktop experience... but as far as windows OS's go, Vista is far and away the best (assuming your hardware is supported).
