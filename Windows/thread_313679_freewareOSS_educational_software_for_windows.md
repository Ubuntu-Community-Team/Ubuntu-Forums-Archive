### Post by Sef on 2006-12-07
> I would love to install edubuntu but their is too much FUD at my organization.

See if you can run a test on a few machines.  But first draw up a business plan for this project.  Have advantages like no licensing costs; ability to expand without worrying about having license or not, security: no viruses; able to replace lost or stolen disks at no cost.  Disadvantages would include retraining, but that would be a one time cost, not an ongoing cost as a license requires.

### Post by steve.horsley on 2006-12-07
I have to agree with Sef that there can't be any harm in at least trialling Ubuntu on one machine. 

Bear in mind that Edubuntu is intended to be a server installation to which diskless clients connect. You would need at least 2 PCs to trial this - client and server. 

Once you have something working, you can show people it. Remember that trialling it is a no cost option, and nobody can steal your software. If it doesn't work out, just buy some proprietary OS and software and replace Ubuntu. No money lost, no harm done.
