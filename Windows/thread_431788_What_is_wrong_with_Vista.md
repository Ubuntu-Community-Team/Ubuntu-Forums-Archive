### Post by thinkbox on 2007-05-03
As somebody who has to work with Vista, XP, Server 2003, Win2000, and even occaisionally Windows ME/98(SE), I can say that Vista really is a great operating system, though I don't personally appreciate what they've done with DRM either. If you ever install vista, you should try turning off UAC and installing Acrobat reader. That's right, It won't install because there's no write acess to the default temp directory. There are other things that may or may not turn off more in-depth users of the OS, but nothing specifically that I care to vent about.

The Vista OS actually does have some great new features for deployment in large environments, integration into enterprise domains, etc.

I, myself, run Vista on a few of my machines. Why? Because I am working to familiarize myself with it as a part of my full-time job. If you want the eye candy, the improved interface, and you are willing to fiddle with drivers if the 'plug n' play' concept isn't working out for you, I'd say go for it. However, I wouldn't recommend it, per se, over XP.
