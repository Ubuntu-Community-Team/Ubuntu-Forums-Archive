### Post by fballem on 2008-09-01
Actually, your friend is probably better to contact either the manufacturer of his computer, or Microsoft, to get a replacement CD for his PC. If his computer came pre-installed, then the manufacturer should be able to supply him/her with a replacement set of disks.

I believe that the EULA for Retail/OEM Windows Vista is machine specific - allows installation on a single machine. For example, if you buy a retail Vista package, there are usually two CDs, one for 32-bit, a second for 64-bit. You can only install one on a single machine - you can't install on a machine using the 32-bit and a second machine using the 64-bit.

Not sure what the copyright laws are in your country, but your friend should get a replacement from his manufacturer.
