### Post by igknighted on 2008-12-06
A very interesting read (forget that it is windows and just think computing in general).  This line jumps out at me as the major disconnect:

[QUOTE=From the article]But the system is complex because it's so abstracted from the way people really do think about their content.[/QUOTE]

The entire reason you use abstraction in any system (computers, automobiles... anything) is so that the system works like a user's brain expects it too.  Why would you add abstraction that complicates a system?  That makes _no_ sense.

I agree with many of his conclusions.  What windows 7 has done with the system tray is just foolish.  Fix the problem.  End of story.

The libraries seem dumb.  I don't understand how they are useful at all.  So I can sort my filesystem in multiple ways and have something in more than one place?  WHY?  Just use a decent desktop search...

Words cannot express how dumb the new taskbar is.  I can figure it out, but it is not well distinguished as to what everything does.  No excuse for that.

On the bright side... the new quasi-semantic context menus (err... jump lists) do seem like a great idea.  I'm ok with the right click, thats a windows standard.  Keep using it.  The menus seem useful.  Bravo microsoft.

### Post by Skripka on 2008-12-06
> **Paqman said:**
> Doesn't he? From the article:


I personally like the idea of a simpler Windows. Since I only use Windows to run the odd game, i'd like to have something simple (and hopefully lightweight) to replace XP when it expires in 2014.

The problem comes in trying to balance "simple" with easily accessed functionality.  Myself, as a power user, like being able to have lots of info at my finger tips--displayed when I want, otherwise out of the way.  Some of the concepts of Win7 I like.  


On a "Start Bar" type setup, with lots of windows open--it gets impossible to differentiate one button from another on the Bar---KDE4.2 has what I think is a nice solution to this....but Win7 hiding things from the user in an *_attempt_* to make Win7 more "demo-able" and more marketable is obnoxious.

The Start Bar, like the Win95 Start Menu-were good ideas at the time....but as users have lots of applications, as well as lots of apps and windows being opened--both of those devices become useless.  Win Vista addressed the later with the indexed Start Menu--which quite frankly, should have come along much sooner, to my mind.


Screen clutter is bad and annoying.  Functionality impossible to access, is obnoxious and drives away users.
