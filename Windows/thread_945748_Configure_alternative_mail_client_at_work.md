### Post by seanc7 on 2008-10-14
Does your work allow you to change your email client?

If they don't allow it, you shouldn't do it.

If they don't care if you change it, then check if they're using an Exchange server or a regular POP3 or IMAP servers.

If they're using Exchange, then you'll need a program that supports connecting to Exchange.
