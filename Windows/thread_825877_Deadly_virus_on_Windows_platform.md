### Post by DrMega on 2008-06-11
> **cardinals_fan said:**
> How about browsing safely? :)

Browse safely by switching to Linux, or if Windows is a must, then use FireFox instead of IE. Regardless of browser, if it is a Windows box, you need:

1. A good firewall, preferably a hardware one (usually built into your router). It doesn't hurt to have a software one such as ZoneAlarm there as well.
2. An up-to-date antivirus app
3. A popup blocker
4. A good security policy, at least warn before running ActiveX stuff.
5. Some common sense, don't download dodgy stuff or if you must, virus scan it before opening it.
