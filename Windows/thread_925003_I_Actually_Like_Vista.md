### Post by Vince4Amy on 2008-09-20
Around these forums Vista seems to get a lot of stick, I don't know why but it could be because of people using bad drivers, unsupported software or simply just not giving it a chance.

My Vista installation is still running strong after 8 months and I'm very pleased with the way it is still running and I've not had any software or hardware problems and this is the 64 bit version so I was expecting such things, but I never had any trouble at all.

I actually like Aero, I know it doesn't have as many features as Compiz but it's simple and to the point, and although I don't think compositing is productive it just adds a little bit of shininess to the Windows Desktop. 

Vista runs fast for me, I have quite an up to date system I know but I believe that comparing with XP is not ideal because XP was built in 2001 for 2001 hardware. I also remember that when XP was released I had more compatibility problems with it then I do with Vista, in fact I went back to Running 2k until XP SP1 Came out.

I can understand why some people may experience slowness and bloat on Vista even on brand new hardware for one simple reason, OEM BLOAT. OEMS like to add software which does slow the system down, for example Acer has their Wireless Helper which takes up to 50MB of Extra RAM!, what is the point! the Windows Wireless settings are quite sufficient. Then you have these other silly registration services running. 

I have not had any trouble with Viruses/Spyware but I don't just go onto any Website and download .exe's, I always read reviews before trying 'Freeware' Not OpenSource but Closed Source 'Freeware'. I use Opera/Firefox as a browser like I do on OpenSUSE/Slackware. I have to use IE7 for testing websites that I build but even then it's sand boxed in pretty well.

I'm not comparing Vista and Linux here in any way so please do not derail this thread with them sort of comments if you can, I use Linux as well but I'm merely stating my experience with the Windows Only Side of things here, I'm not making this thread to compare two different operating systems. I'd just like to sort of give a more positive view on things here.

I actually tried Vista during the public beta and I was appalled by the bloat of the operating system over XP.  Yes, there had been problems with XP early on, but it had evolved into a stable system that, personally, there was no true reason to replace except for the profit motive.  It was at that point, about 2 years ago, that I started using Linux.

I took on the task of setting up her laptop and making sure that the security was appropriate.  I am comfortable with W2k and XP, but Vista is too different for me.
