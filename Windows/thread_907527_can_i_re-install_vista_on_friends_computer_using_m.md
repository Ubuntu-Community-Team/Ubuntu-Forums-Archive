### Post by Tindytim on 2008-09-02
When you buy a copy of windows, you're buying a licensing key (rights to use a single copy) not a Disc.

That said, if your friends bought his computer from a larger OEM (HP, Dell, Gateway, etc) the factory install, and a fresh install of the operating systems are 2 different things.

If you use that system restore partition it will most likely put it back into the state the PC was coming out of the box. All the software that was installed on the computer will be restored (Drivers, Support Software, Trial Programs).

A fresh install is just the OS, and some generic drivers. There aren't any programs installed (other the basic windows ones). The Benefit here is there isn't a bunch of useless software that may be difficult to uninstall. The downside (for some, me not included) is it doesn't always have the right drivers, and it won't have any support software that the majority of OEMs put on their computers to help with diagnosing issues.

***[COLOR="Red"]Last thing, In order for the install to work with the install Disc, he has to have a license key for the same version of Vista (Home, Home Premium, Ultimate, it doesn't matter if it's 32-bit or 64-bit)[/COLOR]***

> **exploder said:**
> I would try LaRoza's solution first. No matter what it has to be a legal solution because of the activation process.

There are illegal methods that bypass the activation process. So I don't see how that has anything to do with this. Not to mention the License has nothing to do with the Build version of the OS, just which version of the OS it is.
