### Post by the8thstar on 2007-12-16
Vista takes quite a steep learning curve, but overall it's a decent OS. On my machine (Intel 1.9Ghz w/ 1Gb RAM and Intel 945M 128Mb) it runs normally, except for some strange lockdowns sometimes. I'm not the usertype that opens a zillion programs at the same time anyway...

I modded it so that I can bypass the UAC by taking privileges as the Administrator user (same as root in Linux) and I tweaked a few bugs to obtain a better performance. To be specific I did a special defrag and changed the registry to correct a bug related to cut/copy/paste operations.

It's obviously not as open as XP because a lot of software won't run - for no reason BTW - but it's more technically advanced and definitely more stable. 

But Ubuntu takes the prize by far in terms of stability and reliability. It's much harder to learn that Vista, but worth the sweat.
