### Post by Shazaam on 2008-03-12
You should be able to find them in Documents and Settings (XP) under her user name. Or do a search for Peachtree on the drive.
Save the files yes but don't wipe the drive yet if it is still good. When you replace the mobo and start the pc for the first time start it in "Safe Mode" Then you can uninstall any chipset drivers. When you reboot it will load a generic set for you then you can install the correct ones.
