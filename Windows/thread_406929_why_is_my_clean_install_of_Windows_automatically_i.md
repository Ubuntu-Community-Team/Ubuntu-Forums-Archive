You can set Windows update to download and notify instead of install automatically. Then when you get the list of updates select custom, deselect WGA and then select do not remind me again  on the WGA update. In about 3 months Microsoft will try again with a new WGA update, so you will have to deselect it and hide it again. 

Windows 2000 is a good alternative to avoid the DRM in XP. The one to avoid here is Vista.

### Post by jon_h on 2007-04-11
Set the automatic updates to notyify but don't install. Then, when windows detects updates are available it will pop up a bubble that will allow you to do a custom install of updates, so you can hopefully bypass the WGA stuff.
Alternativelly, you can use windows update to manually update and the updates you don't install can be set to never appear again, so if you don't want to install one particular feature, you don't get bugged about it. Not sure whether automatic updates don't install stuff you 'blacklisted'.
