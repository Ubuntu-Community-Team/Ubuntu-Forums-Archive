### Post by Mr. Picklesworth on 2008-06-15
I think there's some posts that need purging, too, even if just for readability; what's with the thought duplication? This thread reads like a bunch of parrots.

XTR0RDiNAiRE, sympathies for losing your work there. It's never fun when that happens.
Another program you may want to try, if you feel up to a bit of transition, is called Inkscape. You can find it if you open Applications -> Add / Remove Applications. That menu lets you install all manner of programs; Inkscape is in the list. (You will have to search for it). Inkscape is a very powerful vector graphics editor. It generates SVG files, which are completely scalable. Kind of tough to explain why I think it is so awesome, but it really is swell to work with...
I think it, possibly combined with a raster editor like Photoshop or GIMP, could work well for your kind of work.

As for Photoshop, the least destructive way to get CS3 running would indeed be a virtual machine, which lets you run another operating system like Windows inside a window in the existing one, without needing to restart your computer. For example, VirtualBox is available in that same Add / Remove apps menu and could do well for you. It's slower that way, but could do the trick. Another possibility is dual booting, in which case you will need to install Windows to a smaller partition and then reinstall Ubuntu to another part of your hard drive (Windows installer does not cope well with dual boot setups, as it assumes to be the only operating system ever. Ubuntu's, on the other hand, detects Windows being installed and sets that up).
