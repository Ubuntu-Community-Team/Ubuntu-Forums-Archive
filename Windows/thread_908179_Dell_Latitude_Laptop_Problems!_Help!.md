### Post by fiddledd on 2008-09-02
A simple way to check if it's a hardware fault is to boot up a Ubuntu, or other, Linux Live CD. If you still have the problem, it's a hardware fault and should be returned if it's still under warranty. If you can't reproduce the symptoms with a Live CD then it's either a faulty driver, or the malware you thought you killed is alive and well.
