### Post by Bungo Pony on 2008-01-28
I asked a question similar to this on the weekend.

The closest thing I got suggested is to use Wine to fix the Windows registry. However, the problem is that Wine will load up the Wine registry instead of the Windows registry, even if you try running regedit from the Windows directory.

In short, there doesn't seem to be anything (yet) to fix the registry in Linux.

Try booting in safe mode (does Vista even have safe mode?)
