### Post by rickyjones on 2008-02-25
You're going to have to provide a lot more information for anything to get done here.

1) What type of internet connection? Dial up? DSL? Cable? 
2) When did this stop working?
3) Error messages?

Having an AOL DNS Suffix just means that at some point that computer was assigned the DNS suffix. This has no bearing on actual connectivity. It just assigns the ending of the FQDN.

Hope this helps!
