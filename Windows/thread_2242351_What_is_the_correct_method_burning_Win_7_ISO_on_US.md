### Post by TheFu on 2014-09-01
You must use the Microsoft tool to do this. Google will find it.

However, many WinVista/7/8 DVDs are tied to the hardware they came with (validated by checking the system BIOS), so only a retail or enterprise copy can be used with an ISO inside a VM.

BTW - I learned this the hard way too - after wasting hours and hours trying to create a bootable flash drive using normal methods - all of which failed.
