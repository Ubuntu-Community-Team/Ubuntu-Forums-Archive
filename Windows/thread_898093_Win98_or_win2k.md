### Post by timzak on 2008-08-25
I have Win2k SP4 on a Pentium 3 650 Mhz 384 MB ram laptop.  If you don't install DotNet Framework from Windows Update, and make sure you disable all of the "automatic updaters" for apps like Java, and don't run antivirus, the OS uses about 80-90 MB on a fresh boot up.  Actually, I have an antivirus installed, but disabled the realtime scanning component.  That by itself uses nearly 100 MB.  Then I occasionally run a full system scan.  This way my RAM isn't constantly tied up.  I trust my browsing and email awareness enough to leave myself open in this way.  If the computer will be used by others, I don't recommend this approach.

I believe a fresh install of Win2k uses 30-50 MB, but once you install all the service packs and critical updates, I could never get ram usage below 80 MB.  So antiX might be better for you given the amount of ram on your machine.
