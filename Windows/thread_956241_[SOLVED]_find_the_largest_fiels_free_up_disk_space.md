### Post by fiddledd on 2008-10-23
The thread is marked as solved, but it's in Windows Discussions. There is no Disk Usage tool in Windows.

There is however a third party app that I use, and it's Open Source. It's also similar, but better than, Disk Usage Analyser.

You can read about it and get it [here](http://windirstat.info/). Oh, it's called WinDirStat.
