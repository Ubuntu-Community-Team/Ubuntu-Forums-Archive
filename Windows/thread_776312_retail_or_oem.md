### Post by sub2007 on 2008-04-30
The OEM is designed to be sold to manufacturers rather than desktop users for installation onto PCs that have been sold preinstalled with Windows. Therefore the following applies:

1. Tech support by Microsoft is NOT included. Part of the OEM license is that if you as a manufacturer are going to buy an OEM copy to install onto a PC then you MUST provide tech support. Microsoft will not provide any tech support.
2. You can ONLY do a clean install with it, there is no upgrade option.

I'm not sure of the others but these are the two that I know of.

### Post by smoker on 2008-04-30
an oem version is tied to the machine it is first activated on. you are not allowed to install it on another pc, even if you wipe the install from the original machine.

a retail version can be reinstalled in another pc, as long as the previous install has been wiped.

also, as oem version are technically for system builders, most retailers will state that you have to buy a piece of hardware if you want to buy an oem (a mouse, or cable, or something!), at least this is the state of things in the uk
