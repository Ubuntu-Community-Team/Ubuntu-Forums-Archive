### Post by aysiu on 2007-04-11
Because it is, especially if you have lost the driver CDs for the hardware you bought a while ago, and you don't know exactly what model video card you have or ethernet card you have.

Hell, even this past Christmas, I was trying to reinstall Windows 2000 for my mom, and the Windows installer kept freezing up during the reformatting of the drive. If that freeze had happened in Ubuntu, people would be screaming left and right, "Linux is too hard!" "Linux is still for geeks!" But since it's Windows... well, no one says anything.

I've done two Windows installations from scratch, and both were extremely painful (not just time-consuming) because I didn't know where the drivers were or what drivers to get or a reliable website to get the drivers from.

But even if you have all the drivers, it can be difficult. My wife did a reinstallation of Windows for a friend a few weeks ago, and she was so glad she uses a Mac. First of all, we didn't know where the activation key was. We thought it was supposed to be on the CD cover. Turns out it was hidden physically on the side of the computer. The Windows installer was text-based and asked questions my wife didn't know the answer to (full reformat? quick reformat? do you want to delete this partition?). The Dell driver CD didn't specify which drivers were for the ethernet card, and when you clicked on the "wizard," it appeared to extract the drivers to some directory, but it didn't actually install the drivers for you. You had to figure out some way to install the drivers (there was a lot of clicking around and guessing to finally find out how to do that). And even after that, Dell's driver "wizard" extracted the drivers to directories that had numbers for names instead of real names. So my wife had to drill down to each directory to try to find out which driver to install.

I can't go on describing it. Reliving that last Windows installation is too painful. In fact, I selfishly had my wife do it, because I just can't stand it. If I could have just popped in a Ubuntu CD, that computer would have been up and running within in an hour (including codecs).
