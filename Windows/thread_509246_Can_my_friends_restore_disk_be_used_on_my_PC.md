Chances are slim - but not necessarilly impossible. First thing you need to do is make sure the restore CD is for the same version of Windows that you have installed. Pop the CD in your drive and open it in Windows Explorer - is there an **i386** folder? If so, you *should* be in business. Next, make sure you have your Windows Product Key written down & handy. If you don't know what your product key is, search Google for a utility called _RockXP_ - this utility will enable you to retrieve it - _write it down_. Now, open up that **i386** folder and look for a file called **winnt32.exe** and run it. The install should proceed from there.

Hope this helps....
