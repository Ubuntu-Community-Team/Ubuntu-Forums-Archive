### Post by coffeecat on 2014-11-26
If you have a Microsoft Windows 7 installation DVD, you simply set the BIOS of the notebook to boot from the optical drive, boot from the Windows DVD and follow the installation wizard, from where you can create your preferred partition layout or let the installer use the whole hard drive.

If you created a set of OEM manufacturer's recovery/restore discs before you formatted the whole drive, then you boot from the first of these and follow the wizard. The details vary according to the manufacturer, but this usually entails restoring the machine back to its factory condition. Some manufacturers have a utility to create a bootable Windows recovery USB flash drive where the machine does not have an optical drive. The same principle applies - you need to have created this before you formatted the hard drive.

If you have none of these, you are basically stuck, since you would have deleted the recovery partition when you formatted the disk. You will probably have to contact the manufacturer to obtain a set of recovery DVDs, for which they will make a charge. What make and model of notebook is it?
