### Post by sandman55 on 2007-04-21
Safe mode starts your PC with minimal drivers so not many files are used. To start in safe mode see this [LINK](http://www.microsoft.com/resources/documentation/windows/xp/all/proddocs/en-us/boot_failsafe.mspx?mfr=true) also if your computer has spy ware download and run [CCleaner](http://www.ccleaner.com/download/) its free you can make a donation if you wish and [Adaware](http://www.lavasoftusa.com/download_and_buy/product_comparison_chart.php) the free version.
Here is a trial version of registry cleaner after the trial when it asks for the product key click cancel it still works I use it often when you install it be sure to select backup registry I've never had to use this option [Amust registry cleaner](http://amust-registry-cleaner.amust-software.qarchive.org/) 
also if you have corrupt dll files click start then run type "SFC /scannow" without the quotes note a space between SFC and / hit enter and have your winXP disk handy for when it asks for it 
I suggest you leave the defragging to last but run the others first CCleaner will clean out most of your crap follow this by adaware which might not find much as CCleaner has done it it would be a good idea to follow with Amust  and SFC /scannow (System file checker) then hopefully you will have a clean machine providing its not too damaged
