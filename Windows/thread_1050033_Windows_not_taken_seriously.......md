### Post by Giant Speck on 2009-01-25
1.  **Reduced price.**  They should reduce the price of Windows, but not make it free.  Microsoft is a business after all.

2.  **True personalization capabilities.**  No more of this crap where the only way you can truly personalize Windows is by changing the colors and wallpaper, installing a shell replacement, or costly third-party theme programs.

3.  **New kernel.** I think it's about time Microsoft made a kernel from scratch rather than upgrading the existing kernel.

4.  **Text file configuration.**  This doesn't have to be included.  I just like it in Linux and though it'd be nice in Windows, too.

5.  **Reduce number of versions.**  Instead of Basic, Home Premium, Business, Ultimate, etc, offer three versions:  Home, Professional, and Lite.

Though, I already take Windows seriously.  If Microsoft implemented the above, I'd take it more seriously.
