### Post by Thelasko on 2008-10-21
or a virus.  You can navigate through Windows by keyboard alone!
alt+tab switches windows
The windows key opens the start menu
alt+F4 closes windows
Tab can be used for selecting various buttons
The arrow keys can be used for selecting everything else
Enter to open
Right click operations can be accomplished by the menu button between the right windows key and right control key.

I went through all of the steps in that WikiHow article and you should be able to do it with the keystrokes I list above. (they are out of order)
