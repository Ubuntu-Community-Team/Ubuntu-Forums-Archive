### Post by Dr. C on 2007-11-11
When comparing Windows XP to Windows Vista one has to ask what has Vista to offer that XP does not. For most users both at home and work it is very little if anything at all. 

On the other hand Windows Vista has given up what is Windows XP biggest advantage: Extensive compatibility with propriety software and drivers. On top of this we have an unfamiliar interface, huge hardware requirements and to top it all off Vista is riddled with DRM. There are even many cases where Ubuntu with Wine can run ***Windows*** applications that Vista cannot run! So it hardly surprising than when given a choice many Windows users will keep XP. After all the main reason to use Windows is to run software or use hardware that needs Windows. 

I have a use for XP at home to run legacy Windows applications some dating to early 1990's originally written for Windows 3.1. Vista would be of no use here

At work we run XP, Vista is out of the question since we have a mission critical application that does not work on Vista. This in addition to a sizable cost in hardware for a decrease in productivity. 

By the way Windows XP both Home and Pro will be supported at least until April 2014 plenty of time for the Wine project to get the bugs out.
