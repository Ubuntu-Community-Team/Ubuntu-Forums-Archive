### Post by 3rdalbum on 2008-08-27
> **yaztromo said:**
> Other than that Vista has some things I can't stand.
- The new control panel. Now options are often 5 to 10 clicks away instead of just 3 or 4.

I noticed this too when I set up someone else's Vista-preinstalled computer.

My favourite feature of Vista is UAC, and how difficult it is to turn off. Unfortunately, I guess it's not difficult enough since most people seem to dispense with it.

UAC was designed partly to keep the user safe from their own folly (turning it off is essentially the same as "running as root"), and partly to force program developers to write to files inside user directories instead of system-wide.

The idea was that users would complain to program developers if a program caused too many UAC prompts. This would mean that developers would update their programs to use only user-specific locations, making the whole system very secure compared to earlier versions of Windows. Unfortunately, program developers seem to be stubborn, either treating UAC prompts as "Microsoft's fault", or assuming correctly that their users have already turned off UAC.

So UAC hasn't fulfilled its purpose really, but I think it's a good feature. I'm also quite impressed with the HURD-like way it can temporarily elevate the privileges of a program that is running, rather than requiring a new instance of the program like sudo does.
