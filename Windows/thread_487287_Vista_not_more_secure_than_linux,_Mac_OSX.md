### Post by lingnoi on 2007-06-28
There are a few problems that this "research" doesn't take into account.

1. Vista is just a base OS. Ubuntu has FAR more stuff in its basic install.
2. Vista is closed source so the amount of bugs people can find is limited to those who can look at the source or figure out what vista is doing.
3. The Vista vulnerabilities we know of are those found by independents. I doubt if Microsoft found a flaw they would tell anyone about it.
4. Vista has when it was release only been available to a select group whereas Ubuntu is available to everyone at any time during the development process.

So I am not surprised that Ubuntu has more vulnerabilities then Vista. 

What it means is Ubuntu has found and fixed more vulnerabilities then Microsoft has in its product therefore it is going to be more secure then the numerous flaws that have not yet been discovered in Vista that will be found over the years to come.

If you look at the chart Microsoft's Vulnerability count is lower then any other operating system but the only closed source operating systems which are included are theirs. You could summarise that this is the norm for closed source OSs that don't accurately report its flaws.
