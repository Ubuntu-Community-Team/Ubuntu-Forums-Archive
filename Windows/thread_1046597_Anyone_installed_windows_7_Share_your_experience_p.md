### Post by stmiller on 2009-01-21
I've tried it out in Virtualbox. The big changes I have noticed are:

You have a 'slider' bar in Control Panel to specify how much or little you want UAC to bug you. (Or set down to none!)

The screen does not flip out and go dark on UAC prompts. That was incredibly annoying in Vista.

The built-in Windows 7 wordpad can open and save .odt files! It also can open and save .docx files.

It has IE8, which is still under development at this time so some sites do not work so well with IE8 as you would probably suspect.

### Post by oasmar1 on 2009-01-21
The good:
The Best Windows Operating system I have ever used in my opinion, infact I even like it more than Leopard (not ubuntu though). It does seem like the taskbar has copied Mac's dock, but in my opinion it works even better - I don't miss expose the dock seems to work just as well, the taskbar is far more functional than the dock, the start menu has become just "Winkey + Type" just like quicksilver. After using OSX I never liked old taskbar so to me the new taskbar is more than welcome. I really like the speed too - Vista was too slow. Control Panel has also been tidied up and a non-dimming UAC works far nicer than before. I never used Vista much so I don't know about which features were added with 7 and which with Vista, but Calibrating Font Rendering and Screen Colour has become just as good as Mac.

The bad:
Aero Peek - what more can I add, its the new "Winkey+Tab" - only there for salesmen to "wow" customers. Photoshop/Illustrator seem to work better on Mac, seems to be designed around the Mac User Interface more than the Windows UI. Lack of multiple desktops which means that the screen can get quite full of windows. Multiscreen support still not as good as it should be. Windows 7 is still nowehere near as customisable as Gnome or KDE with hundreds of themes available.

Other points which are commonly brought up:
It does not look or even feel like KDE4. The effects used in Windows 7 are tasteful, unlike the effects shown in the millions of youtube videos whenever anyone mentions how they like the eyecandy. Final issue, why are developers not utilising Aero into their programs? Firefox needs to incorpate Glasser into the default Vista/7 theme (Hide menu also makes it very nice).

I have Windows 7 installed in Virtual Box and it seems to work without any problems. I did, however, have to install the drivers from the Virtual Box tools CD (VBoxGuestAdditions_2.0.4.iso). You can't just use the 'Install Guest Additions' item from the Devices menu. You need to grab that CD image and mount it. Then open up Device Manager and find the virtual network card (with the question mark). Go to update drivers and select Browse and then the CD drive. This is off the top of my head, so hopefully it is correct enough :p

I'm glad they added the Snipping Tool. It saves a few steps from print screen, pasting in Paint, selecting the relevant area, and then saving the image.

### Post by OutOfReach on 2009-01-21
Note that I have not tried the beta (build 7000) but I have tried an earlier build.


Overall, Windows 7 is shaping up to be really good. I do not usually anticipate Windows releases but 7 might just change that.

It has a very good bootup time, very speedy. I have not measured the boot but I can assure you that it's somewhere in the 20's-30's range.
Shutdown is a little faster too, but not as fast as Linux TBH. (I care about shutdown times because, well, remember XP?)
I love the new taskbar, maybe because I am a fan of KDE. But TBH the only real resemblance between the KDE taskbar and the 7 taskbar are the height and the background style.

I have not tested 7 while on laptop battery because my battery is close to dead. According to Fedora GNOME, it is at about 35% capacity. And only holds for about 20 minutes.

It is fast and responsive. Windows Explorer opens up in less than a second, IE opens up in about a second.

That is pretty much it. So in short, it is fast and good. I might just be using it when it released. It depends if Microsoft screws up and turns it into a resource hog.
