> **hellion0 said:**

There's no reason why it wouldn't work.  An x86 chip is an x86 chip.  To an OS, there's no difference* between a CPU of the same architecture running at 8GHz and one at 8MHz; they both execute the instructions they're given and return a result in a comparable number of clock cycles.  The fact that the clock cycles themselves are a thousand times longer, and so a human outside the computer would see the result happen vastly slower, is irrelevent to the computer and operating system.

*Not quite true, but close enough for our purposes.

(This can be taken to extremes. [This pretty triangle](http://blog.wolfram.com/2007/10/the_prize_is_won_the_simplest.html?lid=title) is Turing-equivalent to an Intel Core 2 Duo -- can it be said to "run Vista"?)
