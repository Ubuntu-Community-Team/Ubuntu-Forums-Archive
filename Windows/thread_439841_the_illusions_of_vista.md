### Post by pjkoczan on 2007-05-11
I've always disliked when a computer tries to be smarter than me. I like when a computer is a guide, perhaps saying something like "Are you really sure you want to rm -rf /? If you really want to do it, OK, but it would be bad." I really dislike how MS has essentially taken even more control away under the guise of a data tracking service.

Not only is this a waste of disk space, it's also a waste of computing power. There's a reason why most file systems only journal metadata, because journaling everything would be slow. Seriously, people create backups/snapshots and have things like CVS and subversion to track changes to files and recover from accidental deletions. And they can access them anytime, from anywhere. However, I will freely admit that CVS can be a pain for newbies. Perhaps what is needed is some sort of easy CVS and/or backup frontends for newbies.
