### Post by I-75 on 2008-12-11
You cannot swap a hard drive containing Windows from one machine and install it on another machine. Older versions  of windows did not have this limitation...MSFT changed this to prevent "cloning".

When Windows first gets installed, it "registers" all the components into a "snapshot" of sorts some say a specific number range...specific to that computer. 

When the motherboard is different, the components are different from the original computer the swapped hard drive with Windows won't work in the other machine due to the security "features" of Windows. This prevents piracy.

So a new install is the only option, if you have the "Retail" version, you can legally transfer a Windows OS from one computer to another. 

You must call up Microsoft and explain what you are doing and see if they will let you do it. This means wiping Windows off the older computer. If the OS is a OEM, I doubt you can do this.

Or if you have separate "license" stickers on both computers, you can legally reinstall Windows using the key on those stickers. One for each computer.

Your other options are, buy a new legit XP disc, Install Linux and use Wine.

Unlike Windows, ... hard drives containing Linux can be easily and legally swapped out between computers.
