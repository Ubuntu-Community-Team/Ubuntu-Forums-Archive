### Post by ubuntu27 on 2007-11-25
Run From my COmputer - means that you wil want the selected feature to be installed in the hard drive.

Run ALL from my computer - Same as above, only that ALL the "feature tree" under the selected feature will be selected to be installed.

installed on First Use. - Now this must be a new one for Office 2007 (I never used it). Never saw it before. 

Not Available - Means: "Do NOT install this feature"


In the previous version of Office (e.g. Office 2003) there was an option of "Run from CD" something like that, in which it tells that you don't want to install the selected feature to the hard drive but you still will like to use it, so, when I want to use it, I will provide the CD.   [It will for the CD when you try to use the non-installed feature]
