### Post by ComputerGeek31618 on 2008-06-24
explorer.exe restarts are what you get when something (usually a complex application, like AOL software,) makes an error with the system, or more specifically, the shell.  It's not uncommon, but it certainly should not be happening often.  If your memory is having trouble too, it would almost surely be the cause (anything you access is in and through the RAM).  You would want to find out if a RAM module is faulty, it could potentially cause more serious troubles.
