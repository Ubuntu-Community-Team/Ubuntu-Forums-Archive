### Post by mpsii on 2009-01-12
Aero requires a compatible GPU to be in effect. Remember, though, that compatible and capable are 2 different concepts. Intel 950 GPUs are compatible but not capable. A nVidia 9-series is both compatible and capable. So, if you have integrated video, then there will be higher CPU demand and possibly some studdering. However, if you have a dedicated DirectX 10 compliant GPU, your system should be fine.

[http://www.microsoft.com/windows/products/windowsvista/features/experiences/aero.mspx](http://www.microsoft.com/windows/products/windowsvista/features/experiences/aero.mspx)

Bottom line ... if you have a Pentium4 system with a Voodoo3 video card, turn off Aero or just install Vista Basic.
