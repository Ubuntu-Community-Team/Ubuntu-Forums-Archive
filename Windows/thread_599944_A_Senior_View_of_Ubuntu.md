### Post by bayvista on 2007-11-01
Just some comments from an older member who has survived 45 years in the Industry:

I tried Ubuntu 2 years ago just for fun. Installation was easy but trying to conquer it was a nightmare. I had almost decided to give it away when my XP laptop refused to boot, even in 'safe' mode. Nothing worked, not even the fanous 'Repair' mode. It just kept crashing with a BSOD (blue screen of death for the uninitiated).

I tried to reload XP from the CD, but it still crashed. I think the Master Boot Record must be stuffed.

Now I loaded Ubuntu from a CD without any problem. I have since upgraded from Dapper to Edgy and can run everything that I used to run on XP.

Two minor hitches:

1. I run a home network with my Desktop and connect to the Internet via a Dlink ADSL Modem which does not support IPV6. Unfortunately Ubuntu does not handle this properly so I had to 'kill' IPV6 in order to get the Synaptic Package Manager to update.

2. The Driver for my Notebook screen did not scroll properly, so a quick search of the Forums provided the fix.

I just posted this note to encourage other Seniors to give it a try. It's a really good package.  :):):):)
