### Post by oldfred on 2013-12-04
Really a Windows question. I might suggest:
       [http://www.eightforums.com/](http://www.eightforums.com/)

You can just use gparted to totally erase drive.
Windows only installs to gpt partitioned drives with UEFI. Standard Windows 7 DVDs have to be converted to USB flash drives and some updates run to install with UEFI.
New UEFI systems have serial number inside the UEFI systems somewhere. So I think you need the vendor recovery version to install using your Windows 8. Your vendor may offer an image (not full installer) of you system for a nominal charge. (Just do no tell them you used Linux, just say recovery does not work).

Always make a image of drive before deleting everything. 
      
 The vendor recovery DVDs are just an image of your drive as purchased. If you have housecleaned a lot of cruft normally included, run many updates with many reboots, and added software you may want a full back up.
