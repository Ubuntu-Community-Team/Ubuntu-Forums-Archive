### Post by deadflowr on 2016-08-07
To run a dos program in Ubuntu you would need to install a dos emulator.
(or possible try to run it with wine(which is windows emulator-like program that allows you to run windows programs near natively in Ubuntu), or inside a windows virtual machine)

But two choices for a dos emulator are
dosbox and dosemu


Also, don't post the .exe file.
As that might be in violation of the programs terms of use, et cetra.
(and we do not allow violations of terms of use)

You can post a link to it's web page, if it has one.
