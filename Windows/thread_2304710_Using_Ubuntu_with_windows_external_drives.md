### Post by brian-mccumber on 2015-11-29
I have a couple USB drives, a micro SD card, and a 1tb external hdd and they were all formatted NTFS in Windows when I was running Windows and I haven't had a problem with them at all. I just plugged them in and was able to access them after I completely switched to Ubuntu without having to convert the file system.

If you are really worried about it, run Ubuntu LIVE before installing it and plug them in just to make sure.
