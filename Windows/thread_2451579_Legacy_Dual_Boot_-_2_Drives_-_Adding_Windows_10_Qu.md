### Post by CelticWarrior on 2020-10-07
If Windows 10 was installed in UEFI mode - it probably was - then no, you won't be able to even boot it directly let alone in a dual-boot in a BIOS machine.
And Windows almost always has problems booting in a different hardware - Linux is a lot more forgiving - often requiring a repair. In most cases there's really no point in doing that because a fresh installation is always preferable. Furthermore there's a license issue. If what you have is a retail license then you can reinstall and reuse that license in yours, as long as at any given time it's installed in only one computer. If, however, it's an OEM license then no, you can't reuse it. OEM licenses are tied to the original hardware.
