### Post by yancek on 2015-06-16
Use the Ubuntu Live CD and the GParted partition manager on it to create a windows partition on which to install.  Format it as ntfs.  Keep this disk handy as windows will overwrite the boot code of Ubuntu without asking/informing you so you will need to reinstall the Ubuntu bootloader after.
