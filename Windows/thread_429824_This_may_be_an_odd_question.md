### Post by carl0ski on 2007-05-02
i dont dual boot anymore.

If you used automatix to install multimedia in Ubuntu 
within Automatix there are a couple options to install

1. Wine (windows capatability layer) usage wine windows.exe

2. Crossover Office Standard Demo far easier to use than Wine however officailly supported by crossover with Macromedia Software. (but costs after 30 days)

or 

3. Virtualbox (virtual machine) use this to install a fully functional Windows Machine inside linux
and then install Macromedia Within Windows Within a Virtualbox Within Ubuntu 
(no need to continually reboot to to swap over.)
