### Post by seanc7 on 2008-04-18
Hit (I think) F8 when you get to the "Booting Windows" screen. it should give you a few boot menu options such as using 

"Last known good configuration".

Select that one and you should boot into XP without the updated video driver.

Then you can reinstall your original XP ATI driver.

If that doesn't do it, there's booting into Safe Mode then reinstalling the XP driver.
