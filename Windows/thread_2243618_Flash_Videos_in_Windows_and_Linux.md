### Post by Sir_Sword on 2014-09-10
I haven't booted into Windows (Windows 7) in a long time, so I booted into it, updated it, made sure Firefox was updated, then updated to the latest nVidia drivers and version of Flash.  Then, I tried to watch some Flash video.

When played in a window (not full-screen), the video was somewhat jerky.  It buffered fine, but it seemed as though it was playing at a lower frame rate than normal.  Playing the video full-screen helped smooth it out, but it still didn't seem optimal.  HTML5 video was beautifully smooth.  As a test, I then installed Chrome and tried it both with the version of Flash I installed manually, and the version of Flash built into it.  I had the same results.

I noticed this issue when I was using the previous versions of the nVidia driver and Flash so I was hoping the new versions would improve things, but they didn't.

Then back into Linux, using the proprietary nVidia driver.

In a window, Flash video is nice and smooth.  Full-screen, Flash video is nice and smooth.  I didn't think I would ever say this, but right now, Flash (built into Chrome) works better and more smoothly for me in Linux than in Windows.

I should add, the card I'm using is an nVidia GTX 750 Ti.  Really nice little card.
