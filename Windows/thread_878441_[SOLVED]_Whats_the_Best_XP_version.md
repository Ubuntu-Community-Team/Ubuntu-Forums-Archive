### Post by crhylove on 2008-08-03
There are many different versions of Windows (and many versions of XP).  It really matters what you want to do.

First and most importantly, you must install a fresh copy of windows whatever you are trying to do.  Installing an old one and then updating to the latest SP (like sp3) is ALWAYS a very bad idea.

I prefer the slip-streamed XP versions for ease of installation, speed, and bloat removal.

My current favorite is TinyXP sp3, which is illegal and a pirate version, but is VERY fast, stable, does 99% of what you might need to do in Windows and seems to run very happily as a VM in Virtualbox, also, meaning you can just stay in Ubuntu and use Windows on one side of your compiz cube.

If your interest is primarily gaming though, you will want to have a dedicated TinyXP sp3 partition to boot into for directx and opengl, which sadly are lacking in most VM programs and setups.

If your interest IS gaming, and you have a good enough rig, check this out:

[http://www.dolphin-emu.com/](http://www.dolphin-emu.com/)

Runs under wine, too, and will likely play some Wii games soon!
