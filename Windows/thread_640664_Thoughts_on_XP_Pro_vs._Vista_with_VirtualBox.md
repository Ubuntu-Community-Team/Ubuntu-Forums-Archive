Possibly because of the ATI card, I cannot get dual screens to work properly, so I have to use Windows for that. Also, I have a paperless office, and Linux does not play well with my otherwise-excellent Fujitsu document scanner (I can use it, but without the scan utility, it loses half the functionality).

Now, however, I am looking to get a desktop for the office.

I think I will plan to go with a System 76 "Wild Dog" box. However, I will need to run Windows in VirtualBox to keep the functionality of my scanner. I also need to be able to use Office 2007 occasionally.

So here is my question: Would you go with XP Pro or Vista for this? On the one hand, I only need a small amount of functionality, which would make the logical choice XP Pro (which is also pretty cheap compared with Vista). However, in the long term, Vista seems like it might be the better buy, since software will be updated for Vista going forward.

Then again, maybe Fujitsu will support Linux sometime in the next year or two, which will remove my main need for Windows at all, and I would be better off with XP Pro.

What do you guys think? If Vista, which version should I get?
