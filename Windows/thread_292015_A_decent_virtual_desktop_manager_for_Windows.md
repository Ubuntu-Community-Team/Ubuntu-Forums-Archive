### Post by foxy123 on 2006-11-03
Well after using Linux for sometime I find it difficult to live without virtual desktops on my work Windows laptop. I tried Microsoft Virtual Desktop Manager but it is painfully slow. I also gave a try to Dexpot and VituaWin. Both are sort of OK, but still they do not provide the same experience as on Linux. 

One of the annoying problems is that I like to have a web browser and email client on different desktops. In Ubuntu is I click on a link in email, it opens in the browser on another desktop and stays there. If I have a number of links I just click all of them then switch to the desktop where my browser is and look at them. In both Dexpot and VirtuaWin clicking on the link brings browsers to the same desktop, which is annoying for me. Also VirtuWin is an old piece of software and I do not think it's been updated in recent years.

I wonder if anyone could recommend something better? The only thing is that I am not going to pay for a virtual desktop manager, so it should be free (at least like beer). Until then I will use VirtuWin.

### Post by derjames on 2006-12-05
I had the same problem with IE7.0 in which I had a copy in all desktops. However there is an option to disable that doing the following:

Right click on IE7 Titlebar -> Virtual Dimension -> select 'Desk N' (or whatever your desired deskname is), rather than 'All dektops'

This put IE7 in the desired desk. So far so good

hope this helps.

### Post by islisis on 2007-06-05
To foxy's problem with virtuawin and firefox, if you go into virtuawin's settings under the expert tab select "ignore the event" for hidden window activations and this desktop assigning behaviour stops... this solved a problem with another app of mine, though you lose the ability to call it from another desktop. the "copy window" option is another alternative, which makes the window appear temporarily in the current desktop until you switch when it will revert back
virtuawin definitely draws desktop switches a lot faster than virtual dimension on my notebook
