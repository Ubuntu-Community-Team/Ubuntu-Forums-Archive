### Post by voided3 on 2007-08-18
For the short time i've had it, i've tried it out on the following hardware setups with the Ultimate edition:

1) AMD Athlon t-bird @ 1.4ghz, 1.25GB RAM, x1300 pro 256mb, 160GB

This was more of an experiment being such an old mainboard, but surprisingly it was usable and was only a hair less responsive than XP. The huge difference was in games though; Flatout 2 and UT2k4 had run quite well on that machine w/ XP SP2 but UT was slower and Flatout 2 basically didn't work (the screen would go on and off between the game and solid black).

2) AMD Athlon 64 3200+, 1 GB RAM, 6800GS 256mb, 160GB

Vista ran actually quite well on this system with the exception of the hard drive going batty on boot up from all of the windows update checks and auto defragging... Games that were ran on XP worked as well on Vista unless they used more than 512mb of system ram, then it would swap like crazy and slow down significantly, but  if the system had 2GB of RAM, there would not have been an issue really.

3) Inside Virtualbox and VMware on a Macbook Pro 2.16ghz Core Duo running OSX, 2GB of RAM, x1600 256mb, 100GB

The extra processing power helps a lot w/ the Vista VMs I made for this setup, but even though I setup the VMs to run off of eSATA II external drive, the boot time was still less than impressive; an XP VM I made the same way booted in about ten seconds. 



From those experiences, I would say that it definitely is meant to take advantage of more powerful setups, though kind of puts them in their place at the same time making them feel less powerful, but that's how it goes. I suppose running Vista on a 1.5ghz system w/ 1GB is like running XP on a 500mhz system w/ 256mb of RAM, it's just meant for the faster hardware. I'm still using XP since my main rig is an Athlon XP 2500+ w/ a 256mb x1300 pro and 2 GB of RAM (while it can run Vista, XP frees up a lot of brainpower in comparison). Hopefully newer apps will remain compatible....

### Post by voided3 on 2007-08-26
Just an update to my prior post, I installed Vista Ultimate on this today:

AMD Athlon XP 2500+ 1.83ghz
2GB DDR PC2700 (333mhz)
ATI Radeon x1300 pro 256mb AGP 8x
ECS Elitegroup SiS based mobo (L7S7A2)
2x 160 GB IDE drives


First off, the install was quite swift and all of my device drivers were automatically found except for my VIA Envy24 based surround card... so I thought. My display driver was giving me an error message that it could not start the device, and installing Catalyst did not help. After running Windows Updates, I noticed there was an SiS AGP driver needed that I was not aware of as it did not give me any yellow exclamation marls in the device manager, yet caused my graphics interface to falter. Fortunetly, both my Envy24 and AGP drivers were installed through windows updates. I noticed that my boot times were quite long (about one to two minutes) so I ran good 'ol "msconfig" and much to my horror was greeted with a mile long list of services put in place by default. I disabled things such as readyboost, fax services, and other things I don't use to help (there also is a feature in the control panel to do some more of this that I also used). Aero ran smoothly and all, but even small apps seemed to drag a bit more; surprising considering the 2GB of RAM. Sometimes when exiting a game it would give me a memory dump bluescreen and reboot and Firefox would sometimes randomly crash while browsing. I also hate all of the admin privellidges popups that flash the screen and the reletively slow windows update process, not too mention the millions of submenus in control panel. 

My goal is to hotrod Vista for speed as much as possible; I even am using the windows classic (looks like windows 2000) skin except I made it into a dark theme with some tweaking and I do notice an improvement in speed, but not in ram usage. On a fresh boot, 1250mb out of 2GB is free, and that quickly disappears after some use to become cache. Games such as Flatout 2, UT2k4, and Quake III Arena all run at the same speed they did in XP, sans maybe a few FPS here and there in UT2k4. I might end up going back to XP considering the older apps I use and lack of DX10 hardware, but this kind of future proofs things for me a bit I suppose. I just find it ironic that I had a nice flashly Royale Noir theme on XP and now i'm using the classic skin on Vista to scrounge some speed. This is proof that vista can function just fine on older hardware, but at a slight cost in speed you once had with XP (EX: I once ran the whole Office 2007 suite, Photoshop CS2, Adobe Audition 2.0, Sibelius 3, Google Earth, and Firefox at the same time on XP with this very setup as it is now for an experiment and still had 768mb or so free). I'll see what the upcoming service pack brings my way.
