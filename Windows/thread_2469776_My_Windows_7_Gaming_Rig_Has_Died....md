### Post by CharlesA on 2021-12-09
As far as I am aware, the older updates are still available, but you need to rely up windows update to pull them down and that gets touchy when dealing with operating systems that are end of life.

The last time I had to install Windows 7, I had to apply a bunch of patches manually before I was able to use Windows update to apply anything.

If you go that route, I hope you leave this machine disconnected from the internet as running an end of life OS as a daily driver is just asking for trouble.

### Post by DeadlyOats on 2021-12-23
I installed the new hardware.  Then I installed Win7, and after installing the hardware drivers, including the network card drivers, I went to install Windows Updates.  There were several error codes and Windows Update wouldn't update.  I dug around and found a fix.

I had to download and install KB3138612.  That fixed the error with Windows Update, and I was able to begin the updates.  A long and tedious process, since I install updates in groups, and in a certain order.  Also, there are a few updates that I don't install.

It'll take me a couple of days to finish installing the updates, but I'm glad that I'll have my gaming rig up and running soon.

Thanks for all of you guys' input and advice.
