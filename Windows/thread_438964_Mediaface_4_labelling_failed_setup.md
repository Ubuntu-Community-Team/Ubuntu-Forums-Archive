### Post by mjp216 on 2008-03-26
You need WINE, its a windows emulating program.... but it is not an emulator. the name is an acronym that means "Wine Is Not an Emulator". Now that you know that little fact, ill tell u how to download it.

 If you run in Gutsy Gibbon, like i do, u can do this:

     STEP#1:
1. Open Terminal
2. Type:
> sudo apt-get install wine
3. Press Enter or Return
4. It will ask if u want to run the installation, so type:
> y
5. Press Enter or Return
6. Now it will say to insert your Ubuntu Live/Install CD, so insert your Ubuntu Live/Install CD
7. Press Enter or Return
8. You are done! now you can run the MediaFace 4 Installer, i installed it and i am having i tiny problem but nothing too big, other than that, it works fine.
