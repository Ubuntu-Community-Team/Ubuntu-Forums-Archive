### Post by izanbardprince on 2007-11-07
Well, my bf, who isn't very computer inclined, decided to buy my laptop off me as I wasn't using it much, I couldnt find my XP discs and so I put the Express Upgrade to Vista Basic on it, I figured that with XP, he'd have it so ogged down with spyware and viruses it wouldn't even be funny.

I have to admit that even though I was cursing at all the stuff Microsoft moved around, that it was fairly easy to set it up.

I hid Internet Explorer and replaced it with Opera, and put Pidgin on it for IM, and Frostwire, configured to remove search results for any kind of file that could be virus infected, then set Windows Defender to automatically remove medium and high threats, and slapped in Avast Antivirus Home Edition.

Setting Opera up to use Adblock and Flashblock was the hardest part.

Oh, and I hid Windows Media Player in favor of Winamp, so far so good, but these virus pushers are getting more clever these days, any guesses on how long the system stays virus free?

### Post by p_quarles on 2007-11-07
It's not a bad idea to install Spybot S&D on the computer as well. I had an incident on a computer at work where it found a keylogger that the AV scans didn't pick up. It's free, and quite frankly I think it's one of the best Windows security apps out there. Set it up to run automatically on a regular basis, or convince him to run it often. 

Another thing is that one of the (few) nice things about Vista is that the non-admin mode is actually functional. If you leave UAC on, and make sure he has the admin password, you'll be setting up a situation where he'll get an extra warning before doing anything possibly dangerous. And it will do this *without* locking him out of his own computer. In that way, UAC is actually kind of like the privilege separation built into *nix systems.
