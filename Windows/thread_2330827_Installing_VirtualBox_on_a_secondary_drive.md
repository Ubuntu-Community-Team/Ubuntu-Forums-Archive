### Post by QIII on 2016-07-15
You might be better asking for help on a Win10 forum, but here are some general observations:

Just as with any other program in Windows, it is best to accept the default location for installation.  When difficulties arise, seek support.

The virtual disk for the VM can be anywhere -- even on a network drive if accessible -- but its location my affect performance.

If you install Ubuntu in a VM when you get this going, then it is reasonable to post questions about Ubuntu in the general Ubuntu support areas here.
