### Post by Curtisc on 2007-12-13
> **cyclefiend2000 said:**

+1 for avast.  It hangs out in the background quite nicely, and only when it's updating the virus definition database does it seem to use any system resources.  I've used Norton and McCaffey before and avast is much more lightweight, and equally effective.

As to other programs, there's a few that I install right away:
- firefox
- 7zip
- pidgin (or msn if I need the extra features)
- gmail notifier
- notepad++
- foxit reader

As to programs that you can uninstall, I'm not sure what Dell adds on, but there are windows services and startup programs that you can disable. [ Here's]("http://www.prestwood.com/aspsuite/kb/document_view.asp?qid=100274") a guide (for XP) on how to do it, and which ones are usually unnecessary.
