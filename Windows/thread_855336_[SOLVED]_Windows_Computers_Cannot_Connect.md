### Post by P235 on 2008-07-10
I understand that this is a Ubuntu linux forum so if I have posted this thread in the wrong forum please forgive me.  

As of this morning the browsers of my Windows PCs cannot connect to the internet.  They receive a strong signal from a Linksys WRT54G router, but their browsers cannot access the internet save for this old Ubuntu machine :KS.  I knew learning linux would be handy, huh?  When I came onto this machine and realized it could still connect others told me they think a Windows automatic update recently rebooted the computers.  

[SOLVED:  A recent Windows update conflicts with Zone Alarm firewalls.  Edit the Zone Alarm firewall strength setting to medium to restore internet access.  [Link]("http://www.broadbandreports.com/forum/r20762077-Connectivity-FYI-ZoneAlarm-Recent-MS-update-No-internet") ]

Wow...just wow.  Without Ubuntu I would probably be in the proverbial ditch.  I am contemplating shutting off automatic Windows updates.

It seems someone posted on the same topic, but I wouldn't have guessed from the [title]("http://ubuntuforums.org/showthread.php?t=853679").
