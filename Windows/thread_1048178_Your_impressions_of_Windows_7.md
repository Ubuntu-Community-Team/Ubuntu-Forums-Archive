### Post by Giant Speck on 2009-02-01
> **the8thstar said:**
> What's the memory requirement? I have 2Gb of RAM and 10Gb of disk to allow for this install.

Both of those should be fine.  I have it working perfectly with only one gigabyte of RAM.  I installed it on a 35 GB partition, so I don't remember how much space it took by itself.  I think the minimum is 8 GB, so you should be fine.

> Also, can I configure Grub to launch it with dual boot?

Yes.  Once Windows 7 is installed, you have to use a Live CD to go back in and install GRUB.  When you start up, it will give you the option of Windows or Ubuntu.

Now, if you already have Vista or XP installed on your computer and you install Windows 7 on another partition, it will give you the option to boot Windows 7 or the other Windows when you select Windows from GRUB, but not from GRUB itself.  Selecting Windows on the GRUB menu simply brings up the Windows boot loader.
