### Post by Trebuchet on 2007-03-18
Most OEMs will provide a System Restore disk with all necessary drivers free or for a nominal fee. While I hate Restore Disks, they are often the only way to get an OEM system working properly again - especially laptops, which often have proprietary hardware.

If you want to do a clean install from a Windows retail CD, you'll probably have to install from the Restore Disk, use a driver copying utility such as DriverMax to burn the necessary drivers onto a CD,  install Windows again from the CD, and then reinstall the drivers. It's a pain in the butt, but that's the only way to do what I think you're trying to do.
