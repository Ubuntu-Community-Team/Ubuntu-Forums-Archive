### Post by yancek on 2015-09-01
> Is there a way to revert to Windows 7 through my Ubuntu partition?

No.  I think you would be better off going to the microsoft site or some windows forum.  In fact, I don't know if you can do that in windows, revert to windows 7.  Probably something you should have looked into before the upgrade.  I imagine there will be quite a few people in the future wishing to do the same.  The product key is in a file somewhere on your windows partition so you might be able to find that from Ubuntu if you can mount your windows partition but since you have upgraded windows 7 to windows 10, I'm not sure what impact that will have.   You might try an online search to see if you can find the location of the product key for windows 7.

> Even if there is a way to update Windows 10 to the published version through Ubuntu

I can't imagine that happening.  windows is a proprietary operating system and cannot be upgraded or updated from a totally different operating system such as Linux.

The general recommended methods of doing something like this are to use the Recovery CD you created when you first used windows 7 or the installation DVD you bought with the computer.  Good luck.
