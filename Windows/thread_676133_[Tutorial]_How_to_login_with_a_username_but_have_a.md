### Post by lenswipe on 2008-01-23
Hi there

This is a tutoiral on how to log in with a username say smithj4 but have another name show in the windows start menu such as John Smith.

Ok firstly

1. Click start.

2. Click Run

3. In run type "control userpasswords2" without the quote marks (copy paste may be a good idea as the spacing needs to be exact.

4. In the window that comes up with a list of usernames, double click on the one you wish to change. A box should come up with the following feilds in it:

Username

Real Name

Description

5. In username type the username you want the person to type in the logon prompt.

6. Type ther person's real name in the real name feild (yeah i know DUH!) 

You dont need to put anything in the description box.

Hope this little tutorial helped anyone who didnt know about this.

NOTE: This also works for windows vista!! (probably the only thing that does :))
