How many computers are you trying to connect?  Is your network right now just the XP box and the Vista box?  If so, the first thing I'd do since you said you recently moved the connection from the USB to ethernet is try bypassing the router altogether and plug the the XP box directly into the modem using the ethernet cable.  If you can connect (after jumping through the "setup network connection" hoops), you know it's a router issue; if still no, you might have a bad or improperly installed NIC.  My next question - did the router come from the ISP?  I know I had to get my wireless router separately, and it had an install disk of its own that I had to run to get the encrypted wireless going and the like.  It was a bit of a pain at the time, but I've only had to run the configuration tool once since I moved in to the house I'm at three years ago, and we now have my 2 'nix boxes, a Vista, and three XP (I know, I'm trying...) connecting to it.
