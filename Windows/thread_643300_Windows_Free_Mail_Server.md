### Post by wthanna on 2007-12-17
[www.citadel.org](www.citadel.org) 

Citadel.org has a virtual machine that is a complete messaging system. It can do email, instant messaging, shared calendars, bulletin boards and more, all integrated into one product. It actually runs on CentOS linux, but you can install vmware server and run the virtual machine on a windows machine.  I highly recommend it.  If you prefer Ubuntu (as a guest operating system in vmware) you can create your own virtual machine running Ubuntu (that's what i did), and run it in Windows. I, however use Ubuntu as both the host and guest operating systems, and it works great. Great stuff.

### Post by rickyjones on 2007-12-17
Check out the free version of MailEnable.

[http://www.mailenable.com/standard_edition.asp](http://www.mailenable.com/standard_edition.asp)

It is pretty easy to install and configure and will run just fine on Windows XP, Windows 2000, and Windows Server 2003.

The free version gives you unlimited users and domains. The only caveat is that you only have POP3 access to get the mail, there is no webmail client.

I used to use this in a production environment for my business and it worked great!

-Richard
