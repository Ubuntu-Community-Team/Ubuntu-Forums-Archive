### Post by kevinatkins on 2007-11-22
Hi,

Er this isn't really a Windows forum, but to point you in the right direction.. I'm assuming you've got your XP Pro set up as a standalone machine, not part of a domain. Using an account with administrative privileges (your own?), go into Control Panel -> Users and set your brother's account type to 'User' (I think). What you don't want is his account set as a 'Power User' or 'Administrator'. Doing this should prevent him from installing software and making any changes to the system configuration.... ;-)

On a side note, on the occasions when I have to use / administer Windows, I always make a point of password protecting the Administrator account and limiting the privileges of all other accounts - it's crazy that, by default, users have full privileges, allowing viruses, malware, etc to run amok....

### Post by rickyjones on 2007-11-23
Create a new account per the second post in the thread. Make him the standard "User" which is also called the Limited User account.

He will not be able to install software, except to his own user directory (similar to *Nix). He'll be able to download files but can only save them to his home directory (just like *Nix).

Works great!

### Post by rickyjones on 2007-12-11
As far as just escalating yourself without a password - No. To my knowledge there is no way of doing this. Although you might be able to boot with a recovery CD and wipe the password for the local Administrator account. If you can do that then you can do anything, as the computer is in your possession. In regards to the proxy from what I know the only real way around it is to know exactly how the network is laid out and whether or not the gateway will route your traffic without authorization from the proxy server. If it will then just remove the proxy server from the web browser and make sure you are pointing your NIC to the gateway correctly.
