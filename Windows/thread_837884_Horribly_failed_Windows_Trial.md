### Post by Methuselah on 2008-06-23
I had a similar experience.
I have a pretty powerful machine but DRI is not working with my card in ubuntu so there's no 3D acceleration. I had another reason to test out windows 2000 SP4 on the machine and while I was doing that I decided to install one of my favorite games. 

The installation that takes a few minutes in wine was taking forever in windows 2000. I just got fed up and never did install it.

Other annoyances:

Entering a nonsensical long product key during installation.

Ubuntu recognizes my RAID hardware. To install windows without disabling RAID I'd have to install a driver off a floppy during setup.

Installation process took forever and you can't use your machine while it's happening. Ubuntu installs in probably ten minutes while the CD environment is available.

Nothing to open zip files, read pdfs after install. To microsoft's credit, I think later windows can at least open zip files.

Drivers for most hardware has to be downloaded after install.
Linux recognizes and configures most things.

The driver installation interface is stupid. Keeps asking you to locate the drivers. They're in the installation package I just downloaded dummy!

XP's install isn't much better than this with the added annoyance of activation. While I have no personal experience with a vista install, I've heard it's not much of an improvement.

I think if more people had to install windows more often, they'd stop complaining about the difficulties of installing ubuntu/linux. 

After all the trouble, I felt bored and restricted in windows whereas in linux I feel ready to have fun with my machine (the sky's the limit). It just underscored for me that there's probably no way in hell I'm using windows again as my main OS.
