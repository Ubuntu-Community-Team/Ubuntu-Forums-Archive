### Post by VChief on 2008-05-04
I'm trying to fix an XP for a co-worker. He said the issue just happened one day, so I'm thinking virus/spyware, but thought I'd check here and see if anyone knows.

His wallpaper/background will only stay a solid color. I was messing with it today. I changed the theme to the default XP, theme/wallpaper. Then, I can go to the background and select the one he wants. I hit Apply and it works. However, if I try to change the theme, it just turns the background into the background color (the wallpaper was a picture set on center with the background color around it). It's not just the theme. If I change the color to what he wants (from the XP theme of blue), the picture disappears. Looks like if I open a program, the picture opens (after running OpenOffice install, it disappeared again).

I found some forums on the net talking about regedit and none of that worked. I also check gpedit.msc. He can change it in that he can click what he wants, it just doesn't actually take effect. It does show up on startup/shutdown, but that's it. It happens with any background selected. The color is the only change that takes effect. At first, I thought it was a driver issue because he said that it happened about the time that he stopped using dual display so I uninstalled and reinstalled his drivers as well as hooking it up to another monitor. Anyway, I'm at the point where I can't figure out what could be doing this. My gut says that there's a program interfering, but I don't know. Anybody have any ideas?

if he has any of the above apps installed at the moment, then try uninstalling and see if the problem is rectified.
