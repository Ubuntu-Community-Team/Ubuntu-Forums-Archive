 There is, its called dial a fix. Something in the windows updates gets corrupted and needs fixing. This little tool fixes it. I had a similiar problem after reinstalling Windows XP.

 It was awhile ago that I had the problem. I read some MSFT forums and some recommended using Dial a Fix. My issues were that I was able to get the updates, but I always got the "Installation failed" message. 

 You did not say if you had XP or Vista. I had XP Pro and it worked. Just follow the prompts and I think you have to reboot once or twice.

[http://wiki.djlizard.net/Dial-a-fix](http://wiki.djlizard.net/Dial-a-fix)

There is a forum that can help

[http://lunarsoft.net/forum/index.php?showforum=33](http://lunarsoft.net/forum/index.php?showforum=33)




P.S. Just remember that for each installation of Windows XP the serial key  is tied to the motherboard and its components. If there is a major change of components, there will be problems trying to reinstall Windows and Windows will recognize that new motherboard as a new installation and may not install or you might have to call MSFT to get a change in key. 

Lastly on a separate note, but kind of related. There is a utility elsewhere on the net that will find your Windows serial key code on your hard drive. That utility will run and bring up the key on the screen. This greatly helps those who wants to reinstall their OS but who lost the sticker on the computer. 

If you have a Windows disc of the same type on the computer, you can reinstall your system with the key from that system. It is 100% legal be this enables a clean install on the same computer that the original serial key was assigned to. It does NOT generate keys except what was originally and legally assigned to that particular computer.

 Somebody I know picked up a computer from a flea market with XP Home on it, the HD was way too small and had a lot of junk on it. He already had a XP home disc, all he did was run the utility to bring out the code that was already legally registered to that computer. Took that number and the disc and a new hard drive and reinstalled XP Home on that same computer resulting with a fresh install completely legal as he was using the same number already assigned to that computer. 

 


Good Luck and keep us posted
