### Post by yancek on 2018-09-05
Not sure what you mean by "windows is on the "outside" of my drive".  You currently have both Ubuntu and windows installed, correct?  Is the problem that you have the Ubuntu bootloader booting both?  You should be able to easily use a windows DVD (if you have one) or the windows Recovery CD you created when you first booted windows to Repair the bootloader and have the windows bootloader take over the system.  If you don't have either that could be a problem but you should be able to download a windows iso file to repair your specific version of windows.

If you want only windows and not Ubuntu, I think you would get more useful instructions at a windows forum.
