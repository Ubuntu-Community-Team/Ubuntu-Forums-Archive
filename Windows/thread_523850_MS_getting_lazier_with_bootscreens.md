### Post by jamieh on 2007-08-12
Take a look at the Windows bootscreens:

95/98/98se: Background with clouds, Windows logo looks good
2000: Removed cloud background, windows logo less 3D-looking
Me: Removed "Starting up..." progress bar, as seen in win2000
XP: Background is plain black
XP sp2: Removed Pro/Home text
Vista, removed OS name

Do you see a pattern? With every Windows release, the bootscreen gets less and less detail, Vienna's bootscreen will probably look like [this]("http://www.astro.princeton.edu/~frei/Gcat_htm/Uguide/black.gif").
