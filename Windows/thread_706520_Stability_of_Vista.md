This is not completely true. While Windows has gradually been migrating toward a more Unix-like security model, it certainly isn't there yet. Administrator does not equal root, although that does seem to be what it's modelled after. The token system isn't modular in the same way as the privilege system in most *nix operating systems are, although it is a good first step toward modularity. 

However, that first step may be the only step, I fear. Look at the rest of the Windows operating system. Look at Internet Explorer. Do you call that modular? I think not. 

Without a modular operating system, there will probably always be the threat of viruses, and a greatly increased threat from black hat hackers, because users and various parts of the operating system will not truly be isolated from each other. Ubuntu Linux, while not completely perfect, has done a very good job of staying modular. Windows is the very incarnation of a monolith. 

In addition to all that, Windows source code is not open source, and that presents a whole slew of other security vulnerabilities.
