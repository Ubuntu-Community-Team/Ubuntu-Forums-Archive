### Post by jviscosi on 2007-07-14
I've run both bblean and LiteStep on a Windows XP box and I have to say that although bblean was easier to get up and running, in the long term I liked LiteStep better.  It seemed to be more customizable and easier on the eyes, as well as a bit more stable.  (If anything on Windows can be called stable ...)  I'm not sure which one is lighter.  Both were easy to get rid of.  In fact LiteStep can run on top of Explorer, if I remember correctly, although they don't recommend doing so.  Anyway you shouldn't have any trouble trying both.

The most recent version of LiteStep that I used had a built-in theme deployment utility that would retrieve and install whatever modules the theme needed, so that was kind of neat.  (*COUGH* synaptic? *COUGH*)  It made it a lot easier to install skins and utilities than tracking all the bits down manually.

After running Fluxbox for so long I find that I really can't stand the Windows model of the "start" button and desktop icons and such rubbish.  There was one LiteStep theme in particular that I liked because it had something more or less equivalent to the slit where you could stick applications.  Unfortunately I don't remember the name of the theme and I can't rdesktop to my machine at work and look because I turned it off for the weekend.  Oh, I remember now -- it's called  [Blister Silence]("http://phar0e.wincustomize.com/skins.aspx?skinid=366&libid=6&c=1").  This is my favorite of the LiteStep themes I've used.

The reason  I stopped using the alternative shells is that in my current position I do a lot of coding of Windows apps, and I found that on my machine, under the alternative shells, sometimes the apps didn't look the same as they did on machines running boring old regular Windows Explorer.  If you're not doing this type of work then it shouldn't be an issue.  I do so miss my LiteStep ... :(
