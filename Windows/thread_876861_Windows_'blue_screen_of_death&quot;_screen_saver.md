From SysInternals, (a wholly owned subsidiary of Microsoft Corporation) you can get a Windows screen saver that shows the infamous "Blue Screen of Death" then does a fake reboot which returns to the BSoD to repeat the cycle.  Someone in Redmond has a sense of humor.

Google for "blue screen of death screensaver" and follow the first listed link.  

On my XP machine, I extracted the .scr file directly into the \windows\system32 directory where other screensavers are kept.  It then shows in the list of available screen savers as "SysInternalsBluescreen"


I'll probably share this with a few friends (and only explain it after they panic)
