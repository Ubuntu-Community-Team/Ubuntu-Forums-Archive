### Post by izanbardprince on 2007-08-19
After installing Service Pack 3 and clicking on the Windows Media Player icon, it ran the setup program and prompted me to select file associations, 4 new options, FLAC, Ogg Vorbis, Theora, and Speex.

Microsoft has also upgraded the audio stack with some of the improvements in Vista, example being it no longer downsamples everything to 44.1 khz from 48 khz, no per-app volume control though, but my music definitely sounds "crisper".
