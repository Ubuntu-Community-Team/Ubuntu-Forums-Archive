### Post by oldsoundguy on 2008-10-24
I install Media Player Classic on Windows machines;

[http://www.google.com/search?hl=en&client=firefox-a&rls=org.mozilla:en-US:official&hs=kYt&sa=X&oi=spell&resnum=0&ct=result&cd=1&q=remove+paperport&spell=1](http://www.google.com/search?hl=en&client=firefox-a&rls=org.mozilla:en-US:official&hs=kYt&sa=X&oi=spell&resnum=0&ct=result&cd=1&q=remove+paperport&spell=1)

Along with the mega codec pack.

It will play almost every video/computer video format known to man or beast including Quick Time & Real Player (with the alternate codec installed for each)(and, since it replaces WMP, no more phoning home to MS to let them know what you are viewing!)
Some tweaking is required but, it can be set to open full screen.  The audio is decent.  And it works.

BTW, it is FREE
