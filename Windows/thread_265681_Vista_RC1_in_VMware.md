### Post by Dr. C on 2006-09-27
Yes I have. I used VMware server, with Ubuntu Dapper as the host OS. 

Some applications such as firefox work just fine. The built in games eg freecell are very slow since I have not tried to enable any 3D support in the VM. 

Samba shares worked for the most part, except that I have not been able to connect to files on the Vista VM from Ubuntu. I can see the Vista files from another VM on the same system running Win NT4 and an XP machine on the local lan. 

As for the system I am using a P4 1.8 Ghz with 1.5 gigs of ram and an Nvidia 6200 graphics card.
