### Post by qamelian on 2007-04-04
There is a setting in the Windows Registry that tells XP to display the build number on the desktop. The easiest way to get rid of it (since I can't remember the registry key!) is to download and install the TweakUI Powertoy from Microsoft. It allows you to adjust many obscure settings including this one. There are other configuration tools, such as X-Setup, that also give you access to many hidden configuration options.
