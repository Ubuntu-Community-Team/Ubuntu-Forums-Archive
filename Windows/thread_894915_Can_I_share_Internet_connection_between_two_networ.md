### Post by Baelus on 2008-08-19
If PC1 is using XP then you need to enable ICS (Internet Connection Sharing) on that machine. That effectively makes PC1 the internet Gateway for the other PCs. A google search will give a bunch of sites that will go into a lot of detail.
