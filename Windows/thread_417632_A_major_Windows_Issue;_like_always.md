### Post by samadams on 2007-04-22
Is there something preventing you from clean installing XP and then downloading Linux?  When you said you don't have a restore disk, do you mean you don't have an actual copy of XP at all?   Do you have copies of older an older OS like WinMe, or Win98?  Did your PC come with a restore/shadow partition? (there should be a way to access it from either within windows or pressing F8 at boot up.You might even have the option(a one shot deal) of making a CD copy of the system in an OEM state.

If you don't mind waiting a few days - since you have a net connection, go to Ubuntu.com and order a free Fiesty CD.  You can also go to any bookstore and buy a copy which is in the back of just about any linux or ubuntu book. (your really buying the book - the CD is free)  Of course it will probably be Dapper on the CD but use the LIVE CD to download and make you a new Fiesty disk and then fresh install from there.

As for your Windows issues, have you tried Adaware or Spybot or even Windows Defender?  Do a Google for "Black Viper."  This guy has a very good page on essential XP services and how to pare down your system.  Doing so might turn off a service that a virus is exploiting to restore itself.  At the very least it will significantly speed up your system.  30 seconds per megabyte is not terribly slow.  It may be compared to what you should be getting, but it could be worse - you could have 14.4Kbps dialup.  In fact- I'd bet your slow connection is not directly related to any infestation issue but is a software incompatibility or corruption that is chewing up CPU time or a seriously full and fragmented hard drive.  (have you tried using all the features in the Task Manager to monitor your system performance?- most of the stats available are off by default)  As a good rule of thumb on an average system with a fair amount of bloat - you should have at most 25-40 process running.  (Hint: google their names to find if any of them are spyware/viruses/trojans, etc.)  A slimmed down bare bones system should be about 13-15.  Delete everything in Start>Programs>StartUp.  Check eveything in the System Tray and tell everything NOT to start when windows starts.  Running Hi-Jack this won't hurt either in cleaing things up a bit.  And it's a really small download.  The forums for that program are VERY helpful in cleaning all sorts of infestations.

If you really want to save XP - here's what I'd do after turning all non-essential services off (note - there are NO essential programs [as opposed to services] that I am aware of - your system should function fine without them autostarting):

**Stop using IE - download FireFox. (save IE for updating windows only or if you absolutely cannot view a page properly in FireFox.)**

1)In Safe Mode with Networking (Don't use normal mode - till all is clean - period) download and install all of the above mentioned software if you don't already have it.  Run them and get them up to date (definition files, etc.)

2)restart in SAFE MODE (NO NETWORK - that way there is no possible chance of re-infestation in the background via the net) and run a scan in each program in this order Anti-Virus, Spyware Removers, Hi-Jack This.  Do not go to the next type of program until you get a clean scan with the one you're on.  Restart after EACH Scan and Clean. (if AVG never gives a clean scan after several tries go ahead and move on, but go back to it after each subsequent spyware scan)

3)When you get to Hi-Jack This, save a log when you scan and then get back on with Networking Support and post it to the forums for that program.  The gurus there can walk you through cleaning whatever is left.

4)Once all is clean, go to update.microsoft.com (using IE) and choose "Custom" then select and install all relevant updates. (skip the Genuine Validation tool if it will let you until you have all the others done - just in case)

5)Restart in Normal mode (even if you had to as part of the update process) and download Fiesty.

Hope that helps - and if you already did all that - see my advice on getting a copy of Linux above.
