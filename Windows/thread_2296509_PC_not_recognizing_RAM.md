a few days ago I decided to upgrade my parent's old desktop by putting it an SSD and RAM to the maximum supported.
In order to know how much RAM the motherboard supports, I installed an application called Speccy that tells me the model of the motherboard, and then I googled for the motherboard specs page, here: [https://www.asus.com/Motherboards/P5G41TM_LX/specifications/](https://www.asus.com/Motherboards/P5G41TM_LX/specifications/)
Speccy also told me that the motherboard has 2 RAM slots, with only one being used with a 2 GB memory. 
In the motherboard specs page, it says it supports up to 8 GB, so bought 2x 4 GB memories, here: [http://www.amazon.co.uk/dp/B00J8E91K4/](http://www.amazon.co.uk/dp/B00J8E91K4/)
