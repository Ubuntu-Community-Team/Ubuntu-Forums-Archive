### Post by farmer ck on 2008-11-05
Once installed I set mine to automatically free ram every 30 minutes. That setting will be under the memory tab under settings and configure settings... I also set mine to automatically free ram when I double left-click on the icon on the taskbar. That setting is under the tab labeled other in the settings menu.
