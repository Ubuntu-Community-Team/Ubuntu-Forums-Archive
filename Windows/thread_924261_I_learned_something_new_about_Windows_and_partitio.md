### Post by jaqie on 2008-09-19
You don't even have to do that in windows.

You can change the directory which is known as "program files" by default to any drive and directory, guides available all over with a simple google.

Myself, I merely change the directory to programs and then install all games to d:\games (I keep my windows partition to 20GB and make a seperate partition for all data storage and games installs). Putting resource intensive apps on another actual hard drive as well increases performance in a heavy multitasking system quite radically.
