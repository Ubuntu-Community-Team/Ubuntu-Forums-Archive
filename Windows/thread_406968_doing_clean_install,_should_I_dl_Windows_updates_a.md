### Post by samadams on 2007-04-24
I've always been able to pick and choose the updates I wanted.  When you chose custom, it should have given you a list (by category or priority - you can select all in the left side bar) with check boxes.  That way if you want you can update in the order you choose.  Also, if you just install Xp, then set autoupdates to download but ask you to install there is a custom or advanced option that lets you tick off the one's you want.  It will even ask you if you want to hide the one's you did not select form future update notices so you don't have to unselect them every time.

Personally, I'd get XP 100% up to date FIRST, do a back up then add my drivers, do another back up overwriting the first (if all goes well) then add in my essentail software INCLUDING "fix-it" software like virus/malware cleaners, firefox, etc. as the first things I installed and do a final back up.  I wouldn't add office or other programs until these are in the back up first.
