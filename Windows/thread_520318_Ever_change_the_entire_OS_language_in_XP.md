I was in Ukraine in the Peace Corps for a few years, and I looked into setting up my laptop to dual boot between the Russian and English versions of Windows XP. I researched it a little bit and based on what I found, it seems that you would need the Windows install media for the language you want (Japanese in your case). If you wanted to dual boot between to different language versions of Windows, you would also need something called the MUI (multiple user interface) from Microsoft.
