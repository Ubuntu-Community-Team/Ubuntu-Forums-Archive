### Post by qamelian on 2007-07-21
> **billdotson said:**
> Yes but if it is the upgrade version wouldnt it wipe the XP partition and demand to be on the first partition?

No. By upgrade, all Microsoft means is that existing Windows users get a special price on the new version. There is a minor addition to the install script that confirms that you are eligible for the upgrade by confirming that you either have an existing Windows installation on the  PC or are in possession of a valid Windows installation CD for a previous qualifying version of Windows. Other than this difference in the install CD, the contents of the installation media are identical to the non-upgrade version. They've done this since way back in the the days of DOS. It has no effect on you installation strategy and will still allow you to set up a multi-boot between different versions of Windows.
