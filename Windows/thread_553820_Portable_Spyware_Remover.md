### Post by RevThwack on 2007-09-18
I don't believe there is an anti-spyware package out there that will run off of a thumb drive, but you shouldn't have a problem installing either Spybot S&D or Ad-aware off of a thumb drive. The one thing you need to make sure of though is that you have the latest definition files on your thumb drive as well. The issue you were running into with Spybot is when it went searching for definitions, something that isn't truly required for an install, but is needed if you want it to have any effect on the system. I know for a fact that with both Spybot and Ad-aware, you can download the latest definitions and install them off of a thumb drive.
