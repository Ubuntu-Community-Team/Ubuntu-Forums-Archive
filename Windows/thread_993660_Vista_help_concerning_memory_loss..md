### Post by karellen on 2008-11-27
run a disk cleanup as admin, choose "Files from all users on this computer", select drive C (where your Vista installations is), select the second tab "more options", then below System Restore and Shadows Copies click cleanup. it will delete all but your last restore point, thus freeing a lot of hard disk space
