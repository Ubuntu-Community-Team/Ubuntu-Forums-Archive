### Post by FoolsGold_MKII on 2007-07-10
You can, sorta, but unless you have a religious hatred towards IE, I strongly advise **against** removing IE from the system.

The reason I have is one of compatability with other programs. You can remove stuff like Messenger, OE and WMP fairly safely, but there are programs which assume Internet Explorer is installed on a Windows sytem and use it as a backend to their help systems or navigation (Valve's Steam app is one example). If it's not there, those programs will complain.

Keep in mind that without IE, you will also be unable to access Windows Update, although I imagine the automatic updates will still work and there's that web site (forget the name) which packages all the updates for download.

### Post by izanbardprince on 2007-07-29
Well, a better question is why would you want to remove IE?

I mean it made some sense when Windows 98 was out, with it's resource sapping Webby Desktop shell, that one might want to use 98Lite or something to remove it and put the Windows 95 shell back in, because PC's back then were REALLY REALLY slow (A 233 Mhz Pentium 2 was considered smoking fast, and most systems shipped with 32-64 megabytes of RAM, and had maybe 4-8 gigabytes of hard disk space)) and removing all the fluff helped speed it up enough so that you might get another 150 megs of disk space or an extra frame or two per second on Quake, but now with dual and quad core processors in the Ghz range, with 30-40 times as much RAM and hard disk space, the extra burden of IE is rather insignificant.

I assume that removing IE could increase the security of the operating system, but you would break so much functionality in removing it, especially since third party apps pretty much expect it to be there now.
