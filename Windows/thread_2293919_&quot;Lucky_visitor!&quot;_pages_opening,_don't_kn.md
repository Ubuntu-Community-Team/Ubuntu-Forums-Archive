### Post by patlkli on 2015-09-08
Do those tabs open randomly or just while browsing on certain websites? Some websites open ads and other annoying stuff without being asked.
If you don't want that, but still have to use these websites, you might want to use an Adblocker from the [Chrome Web Store]("https://chrome.google.com/webstore/category/apps?hl=en").

If these pages open on pretty much every website, maybe even on Ubuntu Forums, some installer might have injected an unwanted extension into Google Chrome.
You can open the Chrome extension manager (Menu > More tools > Extensions) to investigate and deactivate or remove it.

There are some differences between Google Chrome and Chromium, which you can find [here]("https://code.google.com/p/chromium/wiki/ChromiumBrowserVsGoogleChrome").
You can also get binaries of Chromium for Windows from various sources and varying degrees of stability and age [here]("https://www.chromium.org/getting-involved/download-chromium").
