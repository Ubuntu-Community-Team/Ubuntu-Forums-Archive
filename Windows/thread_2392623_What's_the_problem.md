### Post by bijayalaxmi1808 on 2018-05-24
I think you have to test this in step by step.

What is the source of file? Is the video recorded on your phone?
If yes then see if the video file is in sync with the audio in the phone or not before sending to the pc?
If yes then see if the transferred file (to your pc) is ok or not?
If you have problem in plating in vlc then try playing the video file on other media players and see if you do see the problem or not?
If not then VLC is the problem. If you see problem in other media player also, then the problem might have happened while sending the video to the pc.

VLC has a setting under Tools > Track Synchroziation
there in the Audio/Video synchrization setting the Audio synchronization value should be 0.0000s
