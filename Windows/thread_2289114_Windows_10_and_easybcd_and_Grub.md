### Post by yancek on 2015-08-01
You might start reading about UEFI as newer systems for windows and Linux use it to boot.  It's still possible to use the older MBR system to boot and you should have that option during the install, at least you will with Ubuntu?  I've seen a number of posts here and elsewhere of people who have done the upgrade and it has overwritten their boot files.  Not sure how it will affect EasyBCD?  You might check the neosmart web site for info as to how it will work with windows 10.
