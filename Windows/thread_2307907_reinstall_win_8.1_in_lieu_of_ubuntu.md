### Post by Bucky Ball on 2015-12-29
*Thread moved to **Windows***.

You need to boot from the BIOS and set the machine to boot from the USB stick. On some machines you can hit F12 at boot and that gets you to a boot device menu. Boot the USB. Sounds like you are trying to boot the Windows install USB while you are booted into Ubuntu. That won't work to install Windows. 

Did you try a Ubuntu flavour (Ubuntu Mate, Xubuntu) that looked more like Windows or Ubuntu with the Unity desktop interface only? Anyway, try the above and that should do it.

### Post by Bucky Ball on 2015-12-29
Like I say, [Lubuntu]("https://help.ubuntu.com/community/Lubuntu/GetLubuntu"), [Xubuntu]("http://xubuntu.org/") or [Ubuntu Mate]("https://ubuntu-mate.org/") might be a better fit. I haven't used Ubuntu since it changed to the Unity desktop environment (some years now). Never liked it. [Kubuntu]("http://www.kubuntu.org/") is also a nice looking OS. There are lots of desktop environments you can try. One of the beauties is that you are not stuck with one way to work and that's it with Linux. There are choices and lots of them. I have created quite a number of 'Franken-hybrids!'.

For instance, before trashing your current install, if you haven't already, try another DE. With Software Centre, look for and install 'xfce4'. That is the desktop environment used by Xubuntu. Once done, log out, choose 'xfce4 session' from the 'Sessions' menu at the login screen. Log in. Any better? :)

PS: I've heard [Ubuntu Gnome]("http://ubuntugnome.org/") is very similar to old-school Win look, and pretty much the same as Ubuntu pre-Unity.
