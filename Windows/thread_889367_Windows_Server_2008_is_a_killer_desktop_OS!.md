So far, it works just as fine as in XP (a little bit faster, maybe) using AviSynth in VirtualDub to encode Lagarith and Xvid, and in MeGUI to encode x264. Sadly, the license I have is x86 only so obviously, the performance can't be increased much (I do have a x64 Vista license though, so I'll probably try it out to see if I get a good performance boost with it).

Oh, and for those who want to try it, download a Windows version of Firefox and put it on a USB key or something before. IE is über-paranoid in Win Server, it actually asks you to put every website you're visiting in your Trusted list before letting you browse it. Downloading Firefox from it took me like 15 minutes :p
