### Post by MaxIBoy on 2008-08-10
It might be problems with the graphics card drivers. If he has Aero effects enabled, and graphics drivers aren't working, the Aero effects might not work and this would cause the desktop environment to fail. Safe mode probably disables these effects.


See if you can disable these effects while booted into safe mode so that Windows won't try to use them in "unsafe" mode.


Or try installing different graphics card drivers while booted into safe mode, that could also fix the problem.
