### Post by Battie on 2009-01-22
You're looking at [Alternate Data Streams]("http://support.microsoft.com/kb/105763").  It's a little-known feature added to the NTFS file system to help compatibility with Macintosh, which makes heavy use of them for metadata storage.

I've never seen ADS abused but there is the potential for an attacker to hide data in them.  They're a fork of the main file and their size and data are not evident in Explorer.  The stream must be accessed directly by opening file.ext:streamname.  

In short, I don't know why you're seeing this, or if its malicious, but you've exposed an interesting little facet of NTFS. :-)

You will see this error when copying data from NTFS to another file format like FAT32 (on a USB key) which doesn't support alternate data streams. Most frequently, Anti-virus software makes use of alternate data streams in order to improve scan times. So most likely, all you'll be loosing is a tag from your anti-virus.
