### Post by wtheron on 2013-11-23
It would be better to leave Windows 7 on the laptop.  Installing Windows 8 is like getting rid of XP to install Vista.  :)

I have an HP laptop with a similar setup.  As far as I know, the partition containing the recovery files are write protected, so you will have a really hard time getting rid of that partition, if it is even possible.  So, if you install Windows 8, and it asks you where to install to, you merely pick the partition that you know is not the recovery partition, and it will install to that partition.  If you do not like Windows 8, you simply reboot into the recovery partition, format the drive containing Windows 8 and reinstall Windows 7.
