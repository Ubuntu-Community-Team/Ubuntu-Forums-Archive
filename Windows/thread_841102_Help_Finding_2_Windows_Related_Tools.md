### Post by dca on 2008-06-26
To reset profile passwords, ERD Commander.  However, you have to pay for it because it runs on Windows platform.
 
To recover deleted files (even though I hate the company) Symantec/Norton make a heep of useful tools for Windows platforms.  Same caveat, you have to pay for it because it runs on Windows platforms.
 
If you don't wish to pay for a solution, you can have a go on 'download.com' OR 'sourceforge.net' and look for free/shareware equivalents but a (again because of the Windows platform thing) bunch of the hopefuls you'll find will find the deleted files in question but not let you recover them until you buy a valid license...
