### Post by blackraven36 on 2008-11-06
The problem of integrating a fully different GUI into windows is because its impossible. The windows GUI is integrated into the kernel, meanings it part of the core of the system. On linux, on the other hand, the GUI runs on top of the Kernel, and thus can be replaced very easily. The best I have seen before is GUI running on top the original Windows GUI, which hides it.

The reason you have such a hard time switching to blender is because its very different from standard 3D modeling software like 3ds Max and Maya. Thus, the learning curve of the program is very big, taking you lots of time to get used to new menus, and the general way it works.

If your class is teaching you 3D animation with 3ds Max, then use 3ds Max.  I was in the same exact boat, I went to school for animation too and learned in on 3ds Max (which will not work in wine, nor in a VM, I've tried).
