### Post by ilago on 2007-03-06
The volume icon in the system tray is an option in Sounds and Audio Devices Properties found in Control Panel. Tick the "Place volume icon in the taskbar.

The "Remove Safely" icon only appears when you have a "hot-pluggable" USB device in use. There is no option to turn it on when there is no device in use. It should come up automatically when you insert a USB thumb drive or USB external hard drive for example.

### Post by Donshyoku on 2007-03-06
If you want to know what startup process is which, you can view a name and blurb about each on in Start->Control Panel->Administrative->Services.  All of your computer's processes are listed there with the option to turn off/on and everything in between.  It also provides information as to what each process does.

If I were you,  I would turn all the defaults back on and go through that panel and pick and choose which ones you don't want.  There are quite a few that you do need to have running for a fully-functional computer.  Ironically, a hanful of those processes actually make your computer faster than turning them off.
