### Post by fiddledd on 2008-10-27
I don't know if Open Office portable is the same, but with Clamwin Portable I just had to extract the files to a USB stick. 

Provided it's not distributed as an SFX archive you can use ubuntu to extract the zip archive to a USB Stick. Even if it is SFX you can use Wine to do the same. As I said, no installation is needed, it's just an archived folder.
