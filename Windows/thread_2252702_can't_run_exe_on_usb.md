### Post by QIII on 2014-11-13
You should not be trying to install Windows with Ubuntu running.

If this is a bootable USB you should ensure that your machine is set to boot from USB and boot directly to it.

Before that, though,  you may want to boot a live Ubuntu disc or USB drive to run gparted and delete all the partitions on the drive.  You may leave all the space unallocated or create one large NTFS partition.
