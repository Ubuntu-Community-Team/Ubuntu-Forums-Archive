### Post by seanc7 on 2008-02-10
There's a freeware program for Windows called BGINFO. We use it at my work for easily seeing what system is what. Especially systems on KVMs. Basically it creates a wallpaper image that is updated everytime you run BGINFO. So if you place it in the startup group, it will update the information as you need.

Now this isn't realtime information. It doesn't do realtime processor or RAM utilitization or network bandwidth usage or anything like that. it's all static data - IP, total HDD size, total RAM, Processor type and speed, network info, host name, version and type of windows (ie: XP Pro SP2)

For Linux, give Conky a look.
