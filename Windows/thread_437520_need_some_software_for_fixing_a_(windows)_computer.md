> **Fittersman said:**
> im not a pirate, im using the product key that they already have been using for their current install (unless they pirated it in the first place)
Well, I didn't call you a pirate as such, but the problem is that even with a legit product key, it's not entirely legit to download a torrented copy of XP. It's sort of a gray area; I think MS would prefer you to buy another copy of XP, but I'd feel vindicated downloading my own copy so long as I used my legit key. Up to you.

> and, if i back up any files, im sure the viruses will follow, wont they? i know when i backed up my music and documents viruses came along before
Viruses are useless unless they can be executed. Music files are just blocks of non-executable data that another program reads and interpretes. Viruses will therefore generally infect files that have the potential to be executed. Word documents CAN be infected with macro viruses, but most other media/document files cannot.

When you get Windows back up and running, you can always install AVG and run a full scan on all the files you are concerned about. So long as you don't attempt to actually open the files before scanning them, you'll be safe.
