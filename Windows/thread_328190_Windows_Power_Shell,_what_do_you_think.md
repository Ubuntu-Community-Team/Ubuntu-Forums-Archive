The Windows Power Shell is new name of the 'Monad' shell announced a while ago has been released, I think it is quite good, it mainly uses .NET features, it's not like the Bash shell we're all used to at all, completely different concept, more a .NET shell.
Anyway, you can download it [here](http://www.microsoft.com/technet/scriptcenter/topics/msh/download.mspx),
