### Post by rockface on 2008-09-01
After days of trying to get my Netgear WG111v2 USB adapter working under multiple versions of Windows (98/2000/XP/2003), and different flavours of Linux and *BSD...we have a winner.

The problem is that it is not the one I was hoping for. In fact it was the last one that I tried...Vista. 

*slaps forehead*

The USB card is an internal Via PCI (don't know the version) designed to bring USB 1.1 machines up to USB 2 standards. No device has worked with it well until now, go figure. I was going to throw this card away, maybe now I'll keep it.:)

Printers, scanners, mice, keyboards and modems never functioned correctly with the Via card. All versions of Windows (previous to Vista), Linux and *BSD recognise the USB 2 chipset (Solaris does throw a wobbler though) but refuse to work with it as they should.

I have a retail copy of Vista that I was going to sell on Ebay. But the combination of Vista, the Netgear Wi-Fi dongle and this Via USB 2 PCI card has many possibilities. Downloading all my usual Open Source favorites (VLC, Open Office, Firefox, Thunderbird etc) and this old beast of a computer (1400 with 512mb ram) has a purpose again.

It is nearly as slow as a dog with quadriplegia (lets face it, Vista can crawl on more modern systems), but it works.

*takes a deep breath and eats a little humble pie...then whispers* That Vista install DVD has more uses than just a good beer coaster you know.
