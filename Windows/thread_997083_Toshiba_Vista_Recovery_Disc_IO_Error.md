### Post by Grant A. on 2008-12-28
The Windows Recovery disk assumes that you have an NTFS partition available. Unfortunately, even if you have an NTFS partition available, it always tries to install itself to your ext2/3 partition. And if you get rid of that partition, it fails. Unfortunately you must have Windows already installed to use it. So you must buy a new copy of Windows.

Anyways, that is my experience with Toshiba Satellites.
