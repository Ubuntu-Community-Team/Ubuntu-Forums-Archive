Microsoft has lagged behind Apple in terms of "Fluidity" of the interface since I can remember, I haven't used Vista enough to notice any issues. I am however, quite disheartened and worried about this particular problem.

Linux so far has failed to feel quite as well integrated as Windows or OS X, Windows and OS X apps feel like they are running in the same environment, share most of the same shortcuts, and work together easily... You still get problems with some Linux apps in terms of cutting and pasting between each other.
