### Post by RAV TUX on 2008-03-11
It's made in Japan, but...
 
It's not Open Source
It's not Linux compatible
 
It uses the Trident Engine by default but you can use a Gecko Engine also by enabling a plugin upon install.
 
So why use it?
 
...because I like experimenting.
 
My thoughts thus far:
 
1. Of course it is not as good as Firefox when it comes to the ease of use of extensions, and the vast library of both skins and extensions.
 
2. Inherent vulnerability by using default Trident Engine.
 
3. Where's Adblock!
 
4. Admittedly it is fast, it uses tabs and the default thumbnail tab option is very nice(I'll be searching for the Firefox extension that mimics this).
 
5. "Open in new window" actual means "Open in new tab".
 
6. The "Viola" skin is very nice if you like a dark theme.
 
7. "Close rightward", "Close leftward" tab options is also very cool.
 
8. I've had my fun with it I am now removing it.

I am curious to what others who have tried it think of it?
