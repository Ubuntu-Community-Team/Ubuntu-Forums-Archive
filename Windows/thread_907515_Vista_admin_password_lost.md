### Post by zmjjmz on 2008-09-01
It's pretty simple really.
Pop in a recent Linux LiveCD (BackTrack works), mount the hard drive, and find the file named Utilman.exe
Name it Utilman~.exe, then find cmd.exe, copy it, and rename it Utilman.exe.
Reboot into Vista, and hit Super-U at login.
You should get a command shell; type explorer to get the panels and the menu and stuff.
