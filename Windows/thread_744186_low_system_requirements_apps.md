> **jgrabham said:**

The graphics card and sound WILL NOT work with Linux, (don't think I, and other people from my LUG have'nt tried)

What is the graphics card? If you don't want Linux on it, thats cool too.

First, remove any antivirus you have. It eats RAM. 

Create a limited account for use, don't use the admin account for non admin work.

Use this firewall: [http://www.ghostsecurity.com/ghostwall/](http://www.ghostsecurity.com/ghostwall/)

Opera is the best browser (shameful of me, I know, but you asked for it). I would recommend against Firefox.

This browser is also good, probably faster than Opera for that machine: [http://www.avantbrowser.com/](http://www.avantbrowser.com/)

As mentioned before, get another PDF reader, Sumatra is good enough.

For media, I use VLC. [http://www.videolan.org/](http://www.videolan.org/) I don't know its requirements, but it can handle anything that I have ever seen.
