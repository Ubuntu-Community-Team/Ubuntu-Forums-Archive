### Post by tamoneya on 2008-08-31
well actually legally you cant transfer it.  Part of the OEM agreement says that the key cannot be transfered between computers.  When installed windows looks at the motherboard serial number and sends it to the activation server essentially locking the cd key with the motherboard.

Id say try xubuntu.  Youll be surprised how quickly kids can learn.
