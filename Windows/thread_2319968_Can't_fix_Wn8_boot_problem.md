### Post by yancek on 2016-04-09
There is noting in your boot repair output indicating any Linux other than your flash drive with boot repair on it.  The boot repair is basically designed to repair Linux bootloaders and although it can make some minor repairs and possible get a windows system to boot, I doubt it will work in your case where you only have windows installed.  I would suggest that you go to a windows forum and post the question there including the exact error you posted above.  Also, it would be helpful to inform them of any changes made to the computer just before this problem occurred or you could do an online search posting the specific error which will get you links such as the one below. 

[http://answers.microsoft.com/en-us/windows/forum/windows_8-update/error-code-0xc0000034-in-windows-8-boot/15e63631-1f94-473f-83a8-8d7af48a83f7?auth=1](http://answers.microsoft.com/en-us/windows/forum/windows_8-update/error-code-0xc0000034-in-windows-8-boot/15e63631-1f94-473f-83a8-8d7af48a83f7?auth=1)
