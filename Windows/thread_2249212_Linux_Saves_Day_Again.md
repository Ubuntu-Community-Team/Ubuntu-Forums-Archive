### Post by cmcanulty on 2014-10-20
Our main library desktop computer wouldn't boot Windows 7 after a recent update. System restore, system repair, safe mode, none would work. After being about ready to reinstall win 7 and all the programs (a tremendous pain) I tried the Xubuntu 14.04 live system. Mounted the windows partitions, searched for the errant update number which I got by searching for bad updates. Deleted that .cab file. The rebooted and was able to access system restore, restored to 2 days previous and rebooted and system is fine now. See article link to avoid this issue.
