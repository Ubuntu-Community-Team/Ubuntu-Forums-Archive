### Post by Spr0k3t on 2007-10-31
In windows you can't create a directory called CON as it's an element in the extended shell.  CON is part of the system calling mechanizm introduced when Explorer was created as the shell.  Programmers will understand what a CON element is similar to what RAW and AUX are.  Try creating a new file called AUX and see what happens.  For Linux only users like me, you can't create those folders in Windows.  I think you can create a file called RAW but CON and AUX are both used by the system's OSI layer.
