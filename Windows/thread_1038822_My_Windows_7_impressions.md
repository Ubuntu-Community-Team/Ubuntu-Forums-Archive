### Post by shatteredmindofbob on 2009-01-13
As I discovered, I was wrong in the previous thread, clean installs are possible without Vista, but man was downloading this thing a pain. For some reason, the download kept timing out in Firefox, so I finally tried downloading it in IE through VirtualBox and saving it to a shared folder to I could burn it with GnomeBaker. 

Once that was over with...wow. The install was just as painless as Ubuntu and it even configured my wireless card right away (It usually takes me about 3 hours to get wireless running on a clean XP install...of course, that's counting the hour it takes me to get it to recognize that the ethernet port is there so I can go online to get drivers..)

When I took the plunge into Ubuntu about six months ago, I found myself realizing how bad Windows shortcomings were. Maybe it's just the shock of a new OS, but Windows 7 is now doing the reverse.

The first thing I really noticed (aside from the task bar, obviously) was that the annoying right-click bug in Firefox wasn't there. 

Actually, scratch that. The first thing I noticed was how good the fonts look out of the box. It was two days before I remembered that I need to run ClearType (included out of the box, though it may have been there in Vista too.) 

As for speed, far as I can tell, Windows 7 and Ubuntu boot in about the same amount of time on my computer, though I haven't sat in front of my laptop with a stopwatch yet. 

That said, boot up time isn't something that bothers me. What bothers me is the length of time from when the desktop is displayed to when I can actually USE it. That was probably the biggest factor in my decision to switch to Ubuntu, as I watched XP gradually crawl up to the 10 minute mark on that front. 

Obviously, I won't know if the gradual slow-down becomes a problem for some time, heh, probably around the time the beta expires. 

(Speaking of beta and on a light note, this is probably the first time I haven't immediately changed the default wallpaper on an OS.)

It's also been nice to get reacquainted with some of my favourite Windows apps like Evernote. Don't get me wrong, I like Tomboy, especially since it lacks Evernote's bloat and it was upon discovering it while trying to a live session of Feisty Fawn more than a year ago that I even realize I needed a stand-alone note-taking app but it is missing some features I feel are key. Like being able to drag a spreadsheet into the program and have it indexed right away. 

And using Digsby for IM is also pretty great, much as I do like Pidgin...though I'm waiting on the Linux version of that one. 

Oh, and Jing, a rather nifty screenshot application. 

Oh, oh, and Windows Live Writer, which is the reason I virtualize Windows XP (seriously, Linux really needs a decent blogging client.) 

All this said, there are plenty of downsides. I was looking forward to smooth gaming again but can't find any drivers that support OpenGL for Windows 7, so it's back to Ubuntu if I want to play anything based on the Quake engine. I guess I could moan about how a kernel update broke my video drivers, :p

Windows 7 also doesn't seem to want to play nice with my Blackberry for some reason. It recognizes that it's connected...for about five seconds. Perhaps I have to do some additional configuring to get it working properly but for now, I'll have to load up Ubuntu to charge it through the USB port. 

Finally, for some reason, Windows Media Player doesn't recognize the tags on my ripped MP3s, even though every player on Linux handles them just fine. If anyone has any ideas on how to fix this, it would be much appreciated. 

Ohhh, yeah, and the task bar. ...I love it. I'm not trying to start a fight with this statement, but it feels to me like they took a bunch of the ideas from Compiz Fusion and finished them. 

Plus it's pretty enough that I may just become a KDE convert upon returning to Linux. 

Yeah, returning to Linux. As much as I am digging Windows 7, I'm trying not to get too attached and remind myself that once it's released, it'll probably run me about $300 PER computer. 

To end off almost completely off-topic, I do really wish Microsoft would ease up on licensing restrictions. I don't expect them to go GPL or anything but I don't think it would be unreasonable for say, a retail copy to be installable on three computers. 

Overall though, if money wasn't a factor, I'd probably be one of the people converting back. Except maybe on a netbook. Microsoft keeps saying they're slimming it down to run on small flash drives with the Atom processor, but unless they're going to add in a different interface (Windows 7 Netbook Remix?) I'm not sure I see it working all that well. 

...well, one last thing, not entirely converting back. I'll always have a Linux partition due to the fact that I'm insanely paranoid about security but can't live without online banking.
