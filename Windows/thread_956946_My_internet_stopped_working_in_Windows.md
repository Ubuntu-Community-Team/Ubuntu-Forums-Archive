### Post by smoker on 2008-10-28
a couple of suggestions: reset your modem/router, there's usually a reset button somewhere, or else turn it off and unplug it from power supply for 5 mins or so. uninstall and then reinstall the windows drivers for the belkin usb adapters, use them plugged straight into a computer usb port, not via a hub, use the windows wireless wizard to set up the wireless, rather than belkins own software, make sure the windows service for this is 'on', (wireless zero config service, i think it is called, or something like that!). is the wireless signal strong enough? set up the computers with a strong signal, next to the wireless modem/router if you have to.

maybe you've done all this, but all i can think of, at the moment, best of luck,

### Post by smoker on 2008-10-30
hi, what operating system are you trying to setup the wireless on, i assumed it was windows xp, can you let me know exactly what? if it is a customised version of xp, do you have service pack 2 installed? you need sp2 for windows wireless setup to work, i believe.

from the pics, you are using the belkin software, check if the windows wireless config service is enabled by going to start, run, and type,  services.msc   a page should open containing a list of services, scroll down and check for windows zero configuration (or wireless zero config, going from memory here!). if you find it, it should show whether it is running, or disabled, you can right click on it and set the attributes and you want to set it to run at startup.

when/if, you've done that, restart the computer and you should have the option to use windows to configure your wireless connection, use this instead of the belkin application. have a look at this link also: [http://www.microsoft.com/athome/moredone/wirelesssetup.mspx](http://www.microsoft.com/athome/moredone/wirelesssetup.mspx)

if you're not using windows xp sp2, or you're using some customised veriosn of windows xp, then post back exactly what version of windows you have and i may have other ideas!

best of luck
