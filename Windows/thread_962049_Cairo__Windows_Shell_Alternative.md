### Post by Giant Speck on 2008-10-28
[http://www.cairoshell.com/](http://www.cairoshell.com/)

Has anyone heard of this new shell alternative for Windows?

It's a closed-source desktop environment currently in development for Windows Vista and XP, and is meant to replace Windows Explorer and offer Windows users a customizable, free alternative to the current Windows look and feel.

They currently do not know when this program will be fully completed, but they hope to be at Milestone 1 by December.

I personally cannot wait to actually try this!  It looks beautiful!
