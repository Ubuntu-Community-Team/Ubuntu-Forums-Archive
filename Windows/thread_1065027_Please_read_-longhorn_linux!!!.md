### Post by bornagainpenguin on 2009-02-10
Unless I completely misunderstood the OP, (which is completely possible given the clarity of his writing) I think you're misunderstanding what he's saying.

He doesn't want to hack or reverse engineer Windows Longhorn, what he wants to do is theme and pull together the various applets, file systems, and search daemons until the combination resembles what Microsoft promised in those early builds and mockups of Windows Longhorn, and never managed to deliver.  Much of the bitterness having to do with Vista was they made the mockups look so good, promised so much, etc etc that when they delivered Vista the average geek felt like they'd been the victim of a bait-n-switch operation.  Meanwhile as the tech sites reported daily how some feature or another was being cut from the final release, Linux was adding those things and making them work on common hardware!

What the OP is saying (again, if I'm understanding him correctly) is if we can pull together all the various bits and bobs that were promised in Longhorn and implement them all together in the same desktop theme it would impress all the people who still pine for what could have been.

Personally I don't think it would be too dificult a task if someone were to work hard at integrating the various parts.

Compiz+ Emerald+Beagle+ GNOME Launch Box+Sidebar Screenlet...

If someone took the time and effort to pull these all together I imagine it could look quite good!
