### Post by coffeecat on 2007-10-30
**Claus7**, what exactly do you want to do? Windows only on one box and Windows/Ubuntu dual-boot on another? Or Windows/Ubuntu on one and Windows/OtherLinux on another?

Have you thought of fitting a removable hard-drive caddy to your machine? When I first started with Linux I only had one desktop and wanted to try out various distros and didn't have the experience/self-confidence to set up a multi-boot. So I bought a caddy holder (which goes in a spare optical drive bay) and a number of caddies, and a couple of spare HDs. I cloned Windows from the original HD to a spare one - which is quite legal since I was using the one licensed copy on one machine. I had something like Windows/LinuxA, Windows/LinuxB and LinuxC single-boot. If I was tired of Fedora and wanted to use Ubuntu (Breezy. :) ) I simply pulled out one caddy and put another in. You can use them how you want.
