### Post by r76 on 2008-02-20
> **(((X))) said:**
> No, My documents are still in there.
But when I click on them, windows says permission denied.
Does anyone know where opera bookmarks located in windows?, I need them back too.
Documents and Settings > (windows username) > Application Data - you may need show hidden and system files to see this one > Opera > Opera > profile > opera6.adr

Backing up the whole Opera directory (including mail if you use it and profile folders) regularly can be useful.  It takes much less space if you clear your cache and trash before doing so.

As regards the Windows files you don't have access rights to, can you access them in Linux e.g. from dual-boot or live-cd, say copy them on to FAT32 external media or something which doesn't have  user 'rights', then copy them back into a new working Windows install?
