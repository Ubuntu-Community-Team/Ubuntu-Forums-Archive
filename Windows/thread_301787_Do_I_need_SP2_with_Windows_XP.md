### Post by izalac on 2006-11-18
> **SnTholiday said:**
> Does anyone know the size of the file for SP2? Since I have only allocated 10 gb for Windows I may not have enough room for it.

The size of the installer is 266 MB. The installer goes a bit disk-space crazy though, first it extracts installation files, than it backs up your current files in the case you might wish to uninstall it, then it installs itself which is quite lengthy procedure.

However, once it installs, you can remove those backup files if you wish so by deleting all \Windows\$NtUninstall* folders, which are created for uninstalling SP2 and all MS patches.

I have a business laptop running XP SP2, MS Office 2003, all necessary additional software, and several applications I couldn't do without (Firefox, GAIM, GIMP, Python, Frozen Bubble etc.), everything fully patched, and it takes just under 5 GB of space in total.

I'd definitely recommend to install SP2 if you plan on using MS Windows XP at all.
