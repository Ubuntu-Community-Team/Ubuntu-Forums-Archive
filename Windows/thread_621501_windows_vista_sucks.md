### Post by Midwest-Linux on 2008-05-19
I see this is a older thread dating back to November 2007, but thats OK. Vista has one outstanding feature. You can shrink the hard drive without using a complicated outside program. This is a really great feature as you can free up 10 to 20 gig and install Ubuntu on the freed space. It takes only a few minutes.

 Vista can be hacked down to the point that the startup times and shut down times equal XP and the Ram usage can be bought down from 75% to 51%.

Turn Off UAC.
Turn Off System Restore.
Turn Off Windows Defender. 
Minimize Startup programs to only one or two.
Delete all crapware.
Delete Norton and Symantec (Use AVG instead).
Turn Off Aero.

All things aside if one needs Windows, you are still better off with XP. However Windows 2000 is still a good and useful OS, though a bit dated and may not run more modern apps.
