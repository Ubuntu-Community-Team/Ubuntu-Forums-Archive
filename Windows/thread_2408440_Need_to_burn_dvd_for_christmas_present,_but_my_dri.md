### Post by coffeecat on 2018-12-18
*Thread moved to **Windows**.*

The community [Burning ISO Howto page](https://help.ubuntu.com/community/BurningIsoHowto), which is a bit neglected, lists a couple of free Windows burning tools.

1 - [Infrarecorder](http://infrarecorder.org/) for "Windows 2000 or newer".

2 - ISO Recorder for Windows XP or newer. The link in the ISO Recorder section is broken, but I found this: [http://isorecorder.alexfeinman.com/#/](http://isorecorder.alexfeinman.com/#/) The wording in that is a bit confused regarding the Windows releases, but it does mention XP.

In the past I have used Infrarecorder on XP successfully, but that was quite a few years ago. One or other of those will hopefully fit the bill for you.

### Post by lisati on 2018-12-18
[ImgBurn]("http://www.imgburn.com/") can be a useful tool if you've already got an ISO file ready for burning. The web page suggests that it can be used with XP

There's [DVD-Flick]("http://www.dvdflick.net/index.php") for preparing ISO images from a variety of file formats, which I think will work with XP (I've only used it with Windows 7 and Windows 10, in conjuction with [IMGBurn]("http://www.imgburn.com/"))

I've also used [DVDStyler]("https://www.dvdstyler.org/en/") to author DVDs, I believe it has an option to burn directly to disk. I've used it mainly with Windows, but believe there's also an Ubuntu-friendly version available.
