### Post by WiFi Ed on 2007-12-05
You can use the partitioning tool that pops up during the install process for Xubuntu to resize your XP partition to make room. It *usually* works pretty well with XP (not Vista!!) but XP might complain when you boot it up next time and want to run it's disk checking tools. At least that's what happened the first time I installed Ubuntu (Edgy Eft version). After XP checked itself out, all was well.

It might be safer to get a Windows compatible partitioning tool (Acronis makes a good one) and resize the XP partition with that first, then install Ubuntu.

Whatever you do, please backup all your important data files first just in case it all goes to heck. I don't want to discourage you, I just want you to be in a position to recover if XP gets all pissy about it...:)
