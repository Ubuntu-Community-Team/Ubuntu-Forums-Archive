I would think 99.9% of Vista users would find Vista Home Premium and Office 2007 Standard to be all they need. If you want maximum security for Windows, Vista Business or Ultimate editions provide full disk encryption and other security measures.

Right now my assessment is that almost nobody *needs* Vista yet; XP is a perfectly serviceable OS and with a little tweaking and free software is fairly secure. It's just an excuse to justify purchasing shiny new hardware. Linux is already as secure (or more) as Vista, it's free, and it runs on existing hardware. I'll be getting Vista only because I need a new system anyway and Vista will be pre-installed on pretty much anything I buy with dual cores.
