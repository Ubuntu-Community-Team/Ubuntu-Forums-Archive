### Post by kanati on 2007-08-01
I can't explain this one.  I have a dual boot of Vista and Ubuntu (7.04).  Evertime I boot into windows after first usuing ubuntu my clock is set exactly four hours forward.  However the clock in ubuntu stays the same.  When I reset the clock in windows and then reboot into windows the clock shows the correct time.  It's only when I boot windows after using ubuntu that my clock is set forward four hours.  Both OS's are installed independently on different hard drives.  Can anyone figure that one out?

### Post by ThrobbingBrain66 on 2007-08-01
right-click the clock and select preferences. Either tick or untick the box for UTC (make it the opposite of what it is now) and I think your problem will be fixed.

---

### Post by Dr. C on 2007-08-02
Use UTC should be ticked off in Ubuntu if one is dual booting Ubuntu and Windows. Also automatically adjust for daylight savings time should be deselected in Windows. This prevents Windows from adjusting for daylight savings time after Ubuntu has already done that.
