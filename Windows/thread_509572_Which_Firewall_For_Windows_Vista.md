### Post by Depressed Man on 2007-07-25
Depends, is she any good with networking (and has a basic understanding of what software runs on her computer)? 

I personally prefer the more strict rule based firewalls like Comodo which tend to be free. They're also more secure then the ones you pay for. But you have to spend more time setting it up. I think Sygate Personal Firewall (if they still give away free ones) is a good one for people who aren't that familar with networking. I was able to teach my dad and sister how to use it.
