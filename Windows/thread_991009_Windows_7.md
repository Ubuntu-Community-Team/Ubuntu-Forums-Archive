To hide an icon of the systray just drag it to the arrow where the hidden icons are, that is simpler and nicer. the jump lists on the taskbar (with right click) makes it faster to get to your work, changing resolution, appearance and adding gadgets is just one right click away now. ### Post by Cenotaph on 2008-12-22
Window grouping like the one in XP it does regardless, in the sense that the various windows of the same application will appear all under the same taskbar icon.

What you can have now is choosing to use the taskbar as it used to be in previous versions (quick launch and window list separated), but in the eventuality that there is too much stuff in the taskbar, it starts to behave like default, and combines the window list with the launch icons.

Its kind of a new version of window grouping, i guess.

I think i wasnt clear regarding Libraries.

Libraries is something Vista already had, the difference now is that its the default entry place in windows explorer. It links to pretty much any folder you want, so when opening windows explorer you can get quick access to all the regular folders you visit. Not that much of a change, its just probably nicer for ppl who have work in many different folders.

In a nutshell, its similar to Gnome's Places. And with windows explorer in the taskbar you get immediate access to those folders thru the windows explorer jump list.

Kind of like Ubuntu.  Each new version of Ubuntu runs on the Linux kernel.  The difference however, is that the changes to the Linux kernel are more fluid than the changes to the NT kernel.  Linux kernel upgrades occur between releases and the changes are usually gradual.

I think the big mistake Microsoft always makes when it creates a new version of Windows is that it makes each version *seem* drastically different, making many users have to relearn the Windows experience all over again.

Ubuntu version releases are much more frequent than Windows version releases, occuring every six months.  Each release is subtly different than the previous release, both in features and visual appearance.  You can pretty much do in Intrepid what you could do in Gutsy because the changes from Gutsy to Intrepid have been subtle enough that it's not dramatically different or difficult to adjust.
