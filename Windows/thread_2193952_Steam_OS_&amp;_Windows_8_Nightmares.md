### Post by slooksterpsv on 2013-12-15
Looks like Ubuntu is my primary and only OS on my system for now. Here's what happened:

SteamOS came out, I downloaded it, put it on a USB drive, read in the FAQ that it'll delete all data and wipe the drive. Thinking well, the expert option should let you command how to partition the drive I attempted to do it going through countless screens.

Well SteamOS wiped out the recovery partition, Windows 8, Elementary, etc. When I got into SteamOS I didn't have Wireless available, so I grabbed an Ubuntu ISO, put Ubuntu on my system and ran testdisk to recover the partitions or try to. Well the Recovery partition looked fine, but the data was corrupt so I can only access limited amounts of data on it.

With that, I ran Hiren's and tried to restore the recovery wim image to my drive and it failed at 20% each time, so I grabbed a Windows 8 iso Enterprise Trial from Microsoft, installed it and found my embedded BIOS key didn't work, so I tried getting the pro update that I had for my HP computer when I had it. Well that key is long gone, won't let me apply it to another machine so I tried updating to Windows 8.1. Key still invalid and the key in the BIOS won't work on any Windows 8 or 8.1 edition (except core oem). I tried creating an EI.cfg and PID.txt file to force it to install, but it kept prompting that the key didn't match the CD.

So now with a Windows 8.1 3 day trial on my system I'm torrenting Ubuntu going to install it and try one last thing. I'm going to grab the registry hives from the WIM image and blast them onto my Windows 8 partition, doubt it'll work, but it's worth a shot. Overall Ubuntu is going to be my only and primary OS.
