### Post by cogadh on 2007-10-07
I bought an XP SP2 disk (full install, not upgrade) after a PC I bought with XP pre-installed did not include an actual XP CD. It had an image disk (PowerQuest Deploy Center) that only worked if I didn't upgrade any of the hardware on the system. It also had a habit of taking over the entire hard drive when the image was applied, regardless of how I originally partitioned the drive. 

Since the image disk was too much of a hassle to use and didn't allow for things like a Windows repair installation, I bought the XP disk and used it whenever a Windows re-install or repair was needed. I also was able to use the license that came with that disk to install XP on a second machine.
