### Post by Erdaron on 2007-07-19
I think my first exposure came from a 486 laptop my dad bought for work. My friends and I were solid DOS / Norton Commander users, and we thought 3.11 was an overly difficult and clumsy system. We thought it was funny-lookin' and would never last...

I remember playing some game on it which involved a dwarf running around and throwing swords at people. This was in the early days of LCD, so pixel lag was pretty bad, and the whole game had pretty intense motion blur, which made it even cooler, somehow. We also set about hacking Windows, which always made my dad very, very nervous. I don't think we ever broke it, though.

Also, that was the first computer I had with real 16-bit sound. *That* was amazing.

### Post by cunawarit on 2007-07-19
First Windows I used was 3.0, in 1991. 

> I liked the fact that Win 3.1 used text in configuration files, and it was reasonably human readable.

I got a job editing those at the University network one summer :) We were deploying a network install of Windows over the whole network. 

They all used the same config files and resolution was set to 640x480 VGA. For personal use I wrote my own config files and a batch files to the right config files depending on what work station I used, that way I got the best resolution in SVGA for every machine.
