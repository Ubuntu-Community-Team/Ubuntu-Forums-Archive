### Post by theothersteve on 2007-01-23
The default address of these devices is usually 192.168.1.77, so you should be able to point your browser at [http://192.168.1.77](http://192.168.1.77) and the nslu2 web interface should appear.

If not, check how you're connected to it (ie. if you're going through a router or something then make sure you can ping that address etc.).
