### Post by jloehrlein on 2007-01-16
You're probably trying to defrag protected system files. I had this happen to me, it turned out to be the backup points.  You probably need to disable backup points. If that doesn't work you might need to go in a delete them manually which involves unhiding system files.

This happened to me when I deleted an account that had backup points.
