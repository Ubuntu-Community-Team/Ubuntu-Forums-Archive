### Post by AdamWill on 2008-06-20
Compiz features use the 3D bits of your card and driver; desktop drawing uses the 2D bits. The two are pretty distinct and it's certainly possible to have a situation where 2D rendering would be buggy, but 3D wouldn't. It might also have been interesting to see if the problem still happened if you turned 3D desktop off.
