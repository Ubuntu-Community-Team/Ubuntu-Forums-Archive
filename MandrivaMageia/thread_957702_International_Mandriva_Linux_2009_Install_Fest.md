### Post by AdamWill on 2008-10-24
Mandriva is co-ordinating an international Install Fest for the new Mandriva Linux 2009 release, on November 22nd, 2008. If you are involved with a LUG or other community group and would be interested in running a local event as part of the Install Fest, Mandriva will provide professionally pressed One CDs and other material. [Visit the Wiki page](http://wiki.mandriva.com/en/Worldwide_Mandriva_Linux_2009_install_fest) for details of how to organize an event in your area, and for a list of already confirmed events, if you'd like to see if there's one in your local area!
