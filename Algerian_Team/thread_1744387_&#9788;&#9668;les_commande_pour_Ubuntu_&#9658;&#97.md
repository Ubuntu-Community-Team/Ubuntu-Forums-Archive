L'utilisation d'ubuntu via des ligne de commande sous le treminal :
-------------------------------------------------------------------
 
(1). sudo apt-get update (mise à jour)
(2). sudo apt-get upgrade (installation des mise à jour)
(3). sudo apt-get install build-essential (installation des compliateur gnu gcc, g++, c++)
