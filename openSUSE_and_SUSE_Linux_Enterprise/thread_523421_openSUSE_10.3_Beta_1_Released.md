### Post by RedDwarf on 2007-08-12
I write from it.
Works fine for a Beta. I could use it like if it was an stable version.
About the problems everybody says 10.2 has:
- 10.3 Beta1 boots my machine in ~25secs while 10.2 boots in ~30secs. The kernel in 10.3Beta1 supports NCQ with my SATA controller while the one in 10.2 don't, so I don't know how much of the gain is from NCQ and how much from real improvements. Anyway I had no problem with the ~30secs of 10.2.
- "software management system"/libzypp/YaST is faster... but is by far the part of the distro withe more bugs. There are problems so obvious that I do not have even reported them, any developer using it for 30 minutes will be able to find them by himself. So I probably will also use 10.3 with Smart (you can also use apt/Synaptic...).

So I see 10.3 just like an updated version of 10.2. Is better, sure, but libzypp is still a problem.
