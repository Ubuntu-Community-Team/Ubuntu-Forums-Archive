### Post by Antman on 2008-02-09
The Alpha2 install went smoothly on my test machine. The only spot it hung on was when it tried to contact the suse server and test the network connection-- so I had to abort that part. 

Upon logging in on the first boot, the upgrade manager crashed... lol. But it hasn't crashed on successive boots.

The whole system is pretty fast. All the apps I tried launched fast and didn't crash.

My laptop uses a 3945 wireless card - and Alpha2 installs the iwl3945 driver by default (instead of the ipw3945). Unfortunately, I can't get the wireless to work in this build. I'll hunt on the suse site to see if a bug report already exists or not.

The openoffice version is 2.4 and seems pretty snappy. I'll spend some more time with the system this weekend and report back. :popcorn:
