### Post by endymon on 2008-02-23
All you really need to do is to make sure you hardware works under Linux.

You can easily do this by googling or searching this forum.
Also try to get an idea by reading the how tos on how to get the network going.
So when you install - you will have Internet connectivity.

If you a buy a network card that doesn't work under Windows you will be majorly screwed if you try to install Windows. So this is something to check first :-)

Once you have those basic stuff you build and improve your experience from there.
The first time I installed Linux it took me about 2-3 FULL days of tweaking just to get everything basically up and running (some incompatabilities and work arounds with gfx and network card).

Now when I reinstall I do it in like 1-2 hours. So it's a learning process - and if you pull through you will never ever look back !
