### Post by julian67 on 2007-02-22
[Additional YaST Package Repositories]("http://en.opensuse.org/Additional_YaST_Package_Repositories")

You should also visit the forums at [http://forums.suselinuxsupport.de/]("http://forums.suselinuxsupport.de/")

and  [http://www.suseforums.net/]("http://www.suseforums.net/")

You should be able to find everything you need at either forum.

You can also have a look at [http://software.opensuse.org/]("http://software.opensuse.org/") which is the the upcoming software repository of the openSUSE build service, you can consider this to be analogous to debian's unstable repos. This is a great place to find the latest and community packages for Gnome and KDE and if there's something you need which you can't find elsewhere it's probably in there. There is a good search tool.

Many people use Smart Package Manager in preference to the openSuse offerings. If you use KDE desktop then the opensuseupdater applet is good for patch/security notifications of official packages. Like with debian/Ubuntu you can only expect security fixes/patches for officially supported packages.
