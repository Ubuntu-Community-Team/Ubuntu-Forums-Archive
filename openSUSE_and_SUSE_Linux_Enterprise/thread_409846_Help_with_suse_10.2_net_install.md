You need to look this stuff up before you start the install.  I would seriously recommend that you not use net install though.  It's a huge PITA.  By downloading the DVD you save time (faster downloads from the mirrors than the repo's) and get much more convenience.  If you really want to try net-install still go to [www.suseforums.net](www.suseforums.net).  If you don't have a DVD drive, download the iso, burn the suse boot disk, point it to the iso file (assuming it is on a partition that isn't being formatted) and install away.
