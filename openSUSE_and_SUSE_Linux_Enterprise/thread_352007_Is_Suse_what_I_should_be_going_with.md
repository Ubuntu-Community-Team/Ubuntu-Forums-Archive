### Post by Hendrixski on 2007-02-02
I'm using OpenSuse now and I'm not to impressed with it.  Debian based distributions are a much better fit for me.  I think that if you buy a licensed version of enterprise Suse Linux it may be worth all of the hype that Suse has been getting. It does come with more stuff out of the box than Ubuntu, but also has a few things missing that Ubuntu comes with.

yast is also kind of annoying, it takes it a while to set up every time you run it.  I think that Ubuntu really is the #1 non-enterprise distribution.
