### Post by Growbag on 2008-05-28
To speed up the process dramatically in 10.3, change all the installed repos to manual refresh.

It's through Software Repositories in Yast, just deselect Automatically refresh for each one.

You will have to re-do this of course each time a new repo is added, as they are set default to auto refresh.

Each time you open Software Management from then on will be faster.

Then when you want to try for a system update, open a terminal and do it manually before running the update tool:

```
sudo zypper refresh
```

It speeds the whole thing up tenfold.

Still not as good as the Debian way, but it makes life a lot easier.

### Post by Erunno on 2008-06-01
> **happy-and-lost said:**
> RC1 is nice, but I can't get my head around the Software Manager. How do you install msttcorefonts?

[http://en.opensuse.org/Optimal_Use_of_Fonts_on_SuSE#SuSE_Linux_10.1_.2F_openSUSE_10.2_.2F_10.3]("http://en.opensuse.org/Optimal_Use_of_Fonts_on_SuSE#SuSE_Linux_10.1_.2F_openSUSE_10.2_.2F_10.3")

This should also work for openSUSE 11.0 as well. openSUSE doesn't ship proprietary packages although some (with harmless licenses) can be found in the non-OSS repository. For everything else (e.g. multimedia codecs) you'll have to resort to third-party repositories. Thankfully there are already one-click-installers for most things so it's pretty easy to fill those gaps. [http://opensuse-community.org]("http://opensuse-community.org") is usually a good starting point.
