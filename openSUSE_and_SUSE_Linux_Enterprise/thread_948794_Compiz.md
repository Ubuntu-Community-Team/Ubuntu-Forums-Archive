### Post by Growbag on 2008-10-17
From your main menu, select: Applications - Utilities - Desktop, and there should be an entry in there called "Desktop Effects".

Run that and simply click on the "enable desktop effects" box.

I always have a problem with the border manager though, ie I get borderless windows whenever I use Compiz, no idea why.  If you do get that problem, then I would suggest making a shell script to reload it on boot.

I'll give instructions on how to do that if you need it, too long otherwise :).

Good luck.
