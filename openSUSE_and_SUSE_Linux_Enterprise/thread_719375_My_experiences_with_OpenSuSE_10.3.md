### Post by ElTimo on 2008-03-09
My time spent using OpenSuSE can be described in a word as "phail."

The install actually went extremely well, the theme is gorgeous (I actually wish someone would put it up on kde-look.org, hint hint wink wink), and i can run as root, but what really turned me away was the incredible number of hoops i had to jump through just to get software to install. Honestly, is it just me or should a linux-based operating system come with repositories preconfigured (just like every other *nix based os)? I spent literally 3 hours just trying to install the nVidia restricted drivers from the repositories, and, failing that, another 2 hours trying to find gcc so that I could compile them. That being said, there is a rather nice feature in SuSE that allows you to add a repository just by clicking a hyperlink. I would really like to see similar functionality implemented in *buntu, even if i have to add it myself ;-)

Whew, that felt good. *is finished whining now*

### Post by NightwishFan on 2008-03-09
I had a few gripes, however overall I like the os. First was that the install, although very good looking graphically, took several hours. The YAST package manager is also slow. Auto updates failed on occasion and I was forced to run them manually. No biggie, but very confusing at first. Also the graphics hardware is insanely difficult to get right, and my desktop effects lagged like I had a weaker card. Overall it was very polished and had some cool integrated features. I use it for when I get bored of gnome and to show off a cool interface. I suppose come Hardy will determine what distro of kde I use. (I am using GNOME Hardy now).
