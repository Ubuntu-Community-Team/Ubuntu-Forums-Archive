### Post by freerick on 2008-02-15
I think it kind of depends where you're upgrading from. 
If you're coming from Vista, then yes, KDE 4.0.x is rock solid, much faster and a great improvement. 
If you're coming from KDE3 that was heavily customized (i.e. Superkaramba, etc) the performance and stability will be about the same, if not slightly better when you upgrade to KDE4.
If you're coming from Gnome or a clean KDE3 install, you'll see some performance decrease, since performance optimizations won't be scheduled until a later release. You may also notice some slight instability if you're trying to customize KDE4 and you'll notice some functionality will be missing, obscured or in a different location (i.e. printer support, sound settings, panel settings, quicklaunch), so some poking around will be in order.

The bottom line is that I definitely found KDE3 to be a bit cleaner and while KDE4 has potential, the average user will run into a few issues here and there from time to time until those are addressed in a later release. However, it's nothing like the hell Vista or XP will put you through.
