### Post by 67GTA on 2008-07-16
Wine is on the DVD. You can get the Nvidia stuff here:  [http://download.nvidia.com/opensuse/11.0](http://download.nvidia.com/opensuse/11.0) You will need x11-video-nvidiaG01 and nvidia-gfxG01-kmp-pae.

EDIT: The wine version on the DVD is still the RC version. The latest version is here: [http://download.opensuse.org/repositories/Emulators:/Wine/openSUSE_11.0](http://download.opensuse.org/repositories/Emulators:/Wine/openSUSE_11.0)
