### Post by 67GTA on 2008-03-06
Someone at the suse forums helped me figure it out. The new driver installation set the window manager to use XGL as default. I changed it to use xorg and all is well. I'm still not sure why my mouse config changed. I couldn't find it in sax, so I loaded the live cd and copied it's xorg mouse config. Everything is cool now.
