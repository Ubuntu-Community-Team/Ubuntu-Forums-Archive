### Post by MAFoElffen on 2023-02-28
Maybe one of the Forums Admins or a Moderator could move this thread to the Portuguese section, where you can get help in Portuguese... [https://ubuntuforums.org/forumdisplay.php?f=462](https://ubuntuforums.org/forumdisplay.php?f=462)

From what I would ask, after translating your post to English, is what version of Windows and Why as WSL? You know that usually, without a lot of hacking, WSL is just a Terminal Session type of interface, or using some graphical applications. You are not going to get the true Desktop Experience and how to use that...

Installing Ubuntu through WSL2 requires some user experience/skills and following a good tutorial to get it running right. I think the end-result is far short of Ubuntu installed natively.

I think you might be happier installing as a VM in Windows (Hyper-V, VirtualBox or VM Player) or alongside Windows as Dual-Boot so you can try the full version of Ubuntu, and give it a full chance to see if that is what you want.

Pelo que eu perguntaria, depois de traduzir sua postagem para o inglês, qual é a versão do Windows e por que é WSL? Você sabe que normalmente, sem muito hacking, o WSL é apenas um tipo de interface de sessão de terminal ou usando alguns aplicativos gráficos. Você não obterá a verdadeira experiência de desktop e como usá-la...

A instalação do Ubuntu por meio do WSL2 requer alguma experiência/habilidades do usuário e seguir um bom tutorial para que funcione corretamente. Acho que o resultado final está muito aquém do Ubuntu instalado nativamente.

Eu acho que você pode ficar mais feliz instalando como uma VM no Windows (Hyper-V, VirtualBox ou VM Player) ou ao lado do Windows como Dual-Boot para que você possa experimentar a versão completa do Ubuntu e dar uma chance completa para ver se isso é o que você quer.
