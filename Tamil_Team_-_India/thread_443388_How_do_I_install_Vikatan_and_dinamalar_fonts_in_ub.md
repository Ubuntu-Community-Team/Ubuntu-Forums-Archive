### Post by XProgrammer on 2007-05-29
Hi,

Just download the fonts from respective webpages and do the following steps.

1. open nautilus
2. Press Ctrl+L ( it will goes to address bar)
3. type "fonts://" ( without quotes) 
4. drag the downloaded font file to this fonts:// folder.
5. open a terminal.
6. execute "fc-cache -f -v" ( notice, everything without "sudo")....

close your firefox and reopen it.. you can use tamil fonts working properly.
