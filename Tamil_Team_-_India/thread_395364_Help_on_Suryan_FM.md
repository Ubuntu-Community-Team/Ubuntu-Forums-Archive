Hi Murli,
  Its not that we dont want to help. Its just that page has been designed especially for Internet Explorer, Windows Media Player. So they dont really care about other users. I looked at the website's source code. They have options for using real player, but ironically the presence of Real Player can only be detected using VBSCRIPT.
