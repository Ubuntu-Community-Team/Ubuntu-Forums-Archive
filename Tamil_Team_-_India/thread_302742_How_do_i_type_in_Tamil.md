### Post by Amitava on 2006-11-19
To able to write Tamil:
1. First install Tamil fonts package..
2. System -> Preferance -> Keyboard -> Layout tab -> Add -> India -> Tamil ->Ok.
3. Add a keyboard indicator on ur panel so that u can toggle.
4. Click on the keyboard indicator to select Tamil open OO and start writing Tamil---.
