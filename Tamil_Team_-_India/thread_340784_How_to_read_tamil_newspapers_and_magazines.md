### Post by jeyaganesh on 2007-02-02
Hi, Now I am able to read tamil magazines and newspaper in ubuntu after installing some fonts.
I downloaded fonts from [http://shyamk.blogspot.com/2006/12/installing-tamil-fonts-in-linux.html](http://shyamk.blogspot.com/2006/12/installing-tamil-fonts-in-linux.html) and from particular tamil websites like [www.vikatan.com](www.vikatan.com), [www.thatstamil.com](www.thatstamil.com).
Then i copied those fonts into location 'fonts:///'. After copying i didnt see new fonts there, but i could see my fonts /home/<username>/.fonts (in nautilus). 

In next step I rebuilt the font cache in terminal using the code 'sudo fc-cache -f -v'.

Now i am able to read tamil websites. Opera browser is working good than firefox in viewing tamil fonts. 

Hope this will help some people to view tamil websites
