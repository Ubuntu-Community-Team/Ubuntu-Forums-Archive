### Post by igknighted on 2007-07-29
Use the nvidia drivers from either the Livna repo or the freshrpms repo (if you use one of these already for multimedia or anything else then use that same... do not mix these two repos).  If you do not already use one of these, then I recommend Livna ([http://rpm.livna.org](http://rpm.livna.org)).  I use it for madwifi, nvidia, and all my multimedia playback needs.
