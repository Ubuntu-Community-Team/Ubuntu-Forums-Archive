It is fairly simple to get the madwifi package from livna (rpm.livna.org), you just need madwifi and the proper kmod package for your kernel.

Codecs in fedora are no more difficult than in Ubuntu (through 6.10).  With Fedora 8 you will get a codec buddy to install codecs as easily as Feisty as well.

It sounds like you did not know what you were getting into when you started with Fedora.  Not that it is hard, but they have legitimate reasons these things are not included, other than being a shortsighted, crappy distro.

### Post by FriedChips on 2007-10-03
if you want sudo it's really easy to set up. first use

```
system-config-users
```

and add yourself to the wheel group
then do

```
su -
visudo
```

look for the line that looks something like

```
#%wheel ALL:ALL   ALL:NOPASSWD
```
may not look exactly like that but all the notes explain it. Just remove the # and you're ready to go.

as far as graphics card drivers, make sure you have the livna repository enabled then do

```
yum install kmod-nvidia
```
or
```
yum install kmod-fglrx
```
for ati.
