1) Mandriva comes with more set up out of the box than Ubuntu.  Sun's java is installed, flash is installed, MP3 codecs are installed.  Yes, to get DVD's and some other codecs working you need PLF repositories... but for DVDs in Ubuntu you need Medibuntu, so how is this different?

2) The PowerPack version is really just an add-on pack of paid-for commercial software that Mandriva makes available in a cheaper and more convenient package.  LinDVD is the only legal DVD playback solution in Linux, Cedega is included for gamers, and there are plenty more that I am forgetting.  It's a bonus... if you want the software, buy PowerPack, but you don't lose anything compared to Ubuntu/Debian that you could get for free that Mandy is charging for.

3) Oh... about non-free stuff included... many non-free drivers such as Nvidia and ATI are installed by default, so there's no hassle trying to install those.

4) For adding the non-free repo.  In 2008.1 it is installed by default (like I believe universe and multiverse are in ubuntu now)... but only if there was internet connection when you installed (again, Ubuntu does the same thing... if you install with no internet all/most repo's are commented out).  In the 2008 version, simply click the update icon in your system tray or open the add/remove software app in the drake control panel and it will ask you if you want to install the distribution sources.  Click yes and it will add them all, including non-free.

Honestly... give it a shot.  I think you will find that aside from a few quirks, it is incredibly easy and straight forward for new users, while not cutting too many corners that will annoy veteran users (a legit backports repo, for example).  I would recommend Mandriva over any other current distro for a new linux user.
