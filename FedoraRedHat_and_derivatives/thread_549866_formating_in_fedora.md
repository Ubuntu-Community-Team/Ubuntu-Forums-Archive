### Post by igknighted on 2007-09-13
Same as Ubuntu, use the gparted app.  You could use "fidsk" from the CLI if you prefer.

If gparted is not installed, use "yum install gparted" and it should be there.

If you are using it to partition a disk that the OS is on, you might be better off using a liveCD, especially the gparted liveCD, but any with a partitioner should work.
