### Post by bigee1212 on 2007-04-03
the performance between the two is relatively similar. 

They both have excellent hardware detection. Sam is based on pclinuxos, the 2007 version is great and works awesome. The xubuntu also has good hardware detection.

Sam uses .RPM which I hate personally, and Xubuntu uses .DEB. Also, Sam's desktop is better becuase on xubuntu, they tried to make i look like gnome (but it actually looks terrible IMHO)

I would suggest SAM if you aren't going to be getting any programs outside the repositories.

### Post by igknighted on 2007-04-03
SAM FTW!!!  I have never been a big PCLOS or Xfce fan, but SAM is really really good.  The default graphics are really nice, there is a great control center that Xubuntu doesn't have, and both use apt/synaptic for package management.  Xubuntu has more packages, but SAM hasn't not had anything I've wanted yet that Ubuntu did, so I think the difference is with real odd stuff.  Also, SAM comes with codecs and flash and all that stuff installed already, so if you plan on installing them it saves you the work.
