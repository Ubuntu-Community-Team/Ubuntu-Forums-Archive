### Post by igknighted on 2008-04-13
"restricted" packages in Fedora are in the livna repository (or freshrpms).  Simply go to livna.org and install the livna rpm there (it will add and configure the repo for you), then go ahead and install the codecs and other restricted packages to your heart's content.

MP3 support is available through the normal gstreamer plugins through livna, or for a while in f8 they offered a link to the fluendo site to download the legal fluendo version (like ubuntu's automatic codec install in a way).  This was dropped for f9 IIRC.

Fedora has become WAY more stable since 5.  6, 7 and 8 have all been tremendous and fairly (major) bug free, although being a distro that lives more on the cutting edge, bugs are possible.  The difference is that if upstream fixes a bug mid-life of a fedora release, fedora will get that update, whereas Ubuntu will not.  You are stuck with the bugs that Ubuntu ships with in most cases.

Other thoughts:

1) Bitlocker is easily defeated if someone has access to your running machine (even if it is locked), so don't think it is the end solution of security.

2) There are linux solutions (not too difficult) to encrypt a drive better than bitlocker, so why pay the resource price of running in Wubi?  These solutions do share the same fundamental flaw the bitlocker does, but other than that the linux options are usually a little better.
