### Post by Antman on 2008-04-01
hmmm... first backup your grub.conf file:
```
su 
cp /boot/grub/grub.conf /boot/grub/grub.BAK
```

Then use your favorite editor to open the grub.conf file (as root). I use nano, but feel free to use gedit or whatever:

```
nano /boot/grub/grub.conf
```

It looks as though your Windows partition is the first one so try changing:
```
title WINDOWS XP
rootnoverify (hd0,1)
chainloader +1
```
to:
```
title WINDOWS XP
rootnoverify (hd0,[COLOR="Red"]0[/COLOR])
chainloader +1
```
