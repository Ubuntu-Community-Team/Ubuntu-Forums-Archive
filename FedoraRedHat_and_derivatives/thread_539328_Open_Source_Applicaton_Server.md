### Post by PaulFr on 2007-08-31
One possibility you might want to look into is the Little-G version of Geronimo (see **[http://geronimo.apache.org/2007/08/21/apache-geronimo-201-released.html](http://geronimo.apache.org/2007/08/21/apache-geronimo-201-released.html)**) if you do not want Java EE capabilities. If you do want the whole JEE stack, the full version should work for you.

Another place to look is **[http://labs.jboss.com/projects/download](http://labs.jboss.com/projects/download)** for the JBoss family of products.
