### Post by wlbi on 2016-09-23
There is not one way, to put Ubuntu Touch on a Android device. 
There is not that **one** Samsung device. Each Samsung device is different.
It must be adjusted to the device,
It's not like you can install Ubuntu on most Desktop Computer.
 Ubuntu Touch is different...
On ARM architectures, the kernel must be configured exact to this one hardware device. There is no hardware detection like on i368 or amd64
