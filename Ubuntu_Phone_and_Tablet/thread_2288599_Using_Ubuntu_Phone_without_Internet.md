### Post by howefield on 2015-07-28
> **veddox said:**
> So a question to those of you with an Ubuntu phone: could you imagine using your phone without Internet, or would that play havoc with the user experience?

Well, to answer the question no *I* couldn't. If I could I'd buy a cheaper phone, a dumb phone with a better camera :)

But you certainly could use it without mobile data or wireless, as in making phone calls, sms, camera functions, music, clock, alarms, file storage, many games and handy utilities like the torch, terminal, and a load more.  

If by user experience you mean the "Ubuntu user experience" then the answer is different as the scopes play a big part in that experience and do need for the main part, an internet connection.
