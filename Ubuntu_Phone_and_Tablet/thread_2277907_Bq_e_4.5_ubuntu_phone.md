### Post by grahammechanical on 2015-05-12
The only applications in the app store are those that have been written for the Ubuntu phone. The developers of utilities like unrar will need to convert their applications to Ubuntu phone apps to get acceptance into the app store. Apps in the app store are tightly controlled to prevent them doing bad things to the operating system. If a developer does not convert his application to the Ubuntu phone and comply with the security requirements of Ubuntu apps, then his application will not be accepted into the app store.

Ubuntu Web Browser is the browser for the Ubuntu phone. It is not meant to be a replacement for a full featured desktop browser. Phones have limited storage space. Large applications are not suitable to a phone device. I think that you will find that the browser on the Firefox phone operating system is not Firefox browser but a specially constructed browser application. In the same way that the Ubuntu Browser on the Ubuntu phone is specially constructed for that mobile device.
