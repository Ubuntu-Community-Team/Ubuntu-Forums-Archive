### Post by jyoti3 on 2016-06-05
You can install the extension “Font Size”. After that, go to customize options, make visible the Togglemenu icon and change the size of the menu bar fonts. They will be bigger. Maybe, there are more then just this way, but this works.

[https://addons.mozilla.org/en-US/firefox/addon/font-size/?src=search](https://addons.mozilla.org/en-US/firefox/addon/font-size/?src=search)
