### Post by Liam_Murphy on 2016-06-29
Current desktop Ubuntu OS's use Unity 7 and X as a display server. You can install whatever programs you want with Ubuntu Desktop.

Ubuntu Phones is using Unity 8 and Mir as a display server. Both are still in development. Mir and X will probably work together to an extent at some point in the future.

In either case, even if it did work properly (Inkscape has gotten working on Ubuntu Phone) it would not be stable at all, especially with Blender.

Hope that answers it, let me know if there is something I missed.
