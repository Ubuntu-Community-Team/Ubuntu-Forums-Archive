### Post by grahammechanical on 2016-07-24
The purpose of Ubuntu for Devices (phones & tablets) is for it to be pre-installed by Original Equipment Manufacturers (OEMs) on retail products. Canonical will work with the OEM to make sure that the Ubuntu phone or tablet model works "out of the box."

Because Ubuntu is open source it is possible to download the code and port Ubuntu for Devices to other models of mobile device. I do not think it is easy to do. Here is a link to some of the information.

[https://developer.ubuntu.com/en/phone/devices/installing-ubuntu-for-devices/](https://developer.ubuntu.com/en/phone/devices/installing-ubuntu-for-devices/)

These are the target devices

[https://developer.ubuntu.com/en/phone/devices/devices/](https://developer.ubuntu.com/en/phone/devices/devices/)

Some other devices listed here but the porting is a community project.

[https://wiki.ubuntu.com/Touch/Devices](https://wiki.ubuntu.com/Touch/Devices)

Here is the Do It yourself guide

[https://developer.ubuntu.com/en/phone/devices/porting-new-device/](https://developer.ubuntu.com/en/phone/devices/porting-new-device/)
