Okay, this worked for me:

```

sudo add-apt-repository ppa:phablet-team/tools
sudo apt-get update
sudo apt-get upgrade
sudo apt-get dist-upgrade

```

Download [http://people.canonical.com/~alextu/tangxi/recovery/recovery.img](http://people.canonical.com/~alextu/tangxi/recovery/recovery.img) (for Meizu devices, you'll have to find a different one for BQ, try [http://people.canonical.com/~jhm/barajas/]("http://people.canonical.com/%7Ejhm/barajas/")[FONT=Consolas] as suggested by Yaowang_Li)[/FONT]
Boot into fastboot (VolDown+Power for MX4; VolUp+Power -> selection for BQ I think).

```

ubuntu-device-flash touch --channel ubuntu-touch/rc-proposed/meizu.en --bootstrap --recovery-image ~/recovery.img 

```

or

```

ubuntu-device-flash touch --channel ubuntu-touch/rc-proposed/bq-aquaris.en --bootstrap --recovery-image ~/recovery.img

```
respectively.

And again (this time in recovery mode -> VolUp+Power),

```

ubuntu-device-flash touch --device arale --channel ubuntu-touch/rc/meizu.en

```

or

```

ubuntu-device-flash touch --device arale --channel ubuntu-touch/rc/bq-aquaris.en

```

In addition, before the second device flash I flashed the system.img that I thankfully copied before updating.
```

fastboot flash system system.img

```
I don't really think that was necessary though.

At the moment I can't upload my system.img because the file is quite big (184 MB) but I'll try again later.

Hope this helps.

@[**[COLOR=#000000]Yaowang_Li[/COLOR]**]("http://ubuntuforums.org/member.php?u=1982415"), I think you need to go into flashboot mode after ```
ubuntu-device-flash touch --channel ubuntu-touch/stable/bq-aquaris.en --bootstrap
``` otherwise your device won't be recognized.
