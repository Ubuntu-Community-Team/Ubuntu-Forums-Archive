### Post by sudodus on 2016-06-01
I have no Ubuntu Touch, but there is an on-screen keyboard in standard Ubuntu and the Ubuntu flavours (Kubuntu, Lubuntu ... Xubuntu) for desktops and laptops:

***onboard***

and it responds to different language settings.

When you change the keyboard globally (via the 'input method' or via a command line depending on the flavour of Ubuntu), it will change also in LibreOffice.

Is ***onboard*** available for Ubuntu Touch?

In order to make it work, the different keyboard software (including fonts for special characters) must be installed. If you have standard Ubuntu in a computer, you can compare, and check what is available in Ubuntu Touch.
