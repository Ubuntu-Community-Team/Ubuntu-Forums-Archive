### Post by grahammechanical on 2016-02-04
> I wondered if anyone could help me and let me know if there is a generic version of Ubuntu touch which would work on any tablet

No. Ubuntu developers have never had any intention of doing that. From the beginning of this project the intention has always been to produce an OS that would be pre-installed on retail mobile devices. There are 3 Ubuntu phone models for retail. At the moment a Ubuntu tablet for retail does not exist. It seems that there might be some information released at the end of February about a Ubuntu tablet being released for retail in April this year. We wait and see.

> it was cheap but the spec looked good!

That is not so good. Mid range and higher is the simple description of the Ubuntu phone/tablet specification. Ubuntu is open source code so if someone wants to take the code and port it to other devices, then they are free to do so. 

Please read the disclaimer

[https://developer.ubuntu.com/en/start/ubuntu-for-devices/installing-ubuntu-for-devices/](https://developer.ubuntu.com/en/start/ubuntu-for-devices/installing-ubuntu-for-devices/)

[https://wiki.ubuntu.com/Touch/Devices](https://wiki.ubuntu.com/Touch/Devices)

Compare the specs of what you have with what will be released.

[http://www.ubuntu.com/tablet](http://www.ubuntu.com/tablet)
