### Post by donquichot2016 on 2016-05-22
In the camera app you can select the default save location, internal or SD card. It's in the configuration screen you can open by swiping upwards at the bottom of the screen.

I think it's not yet possible to preview images in the file manger. You could use the gallery app or some other app. I personally would like thumbnails for images & videos in the file manager. A feature request has already been filed for this: [1466620]("https://bugs.launchpad.net/ubuntu-filemanager-app/+bug/1466620") & [1466625]("https://bugs.launchpad.net/ubuntu-filemanager-app/+bug/1466625").
