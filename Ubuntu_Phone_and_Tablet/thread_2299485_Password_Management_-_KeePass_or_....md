Unfortunately bud, you're outta luck. I just changed over to Ubuntu for Mobile Devices from Android and at this stage, the lack of a KeePass client is the only thing I'm *really* missing... Well, that and a decent Nintendo Game Boy emulator.

There is *one* application in the Ubuntu Store that claims to be able to import KeePass files (there's no indication on whether the developer is referring to KeePass 1.x or KeePass 2.x), however once the application is actually installed, there doesn't appear to be any way to import KeePass files... If you want to try it out yourself, this application is called ""password-shroud".
