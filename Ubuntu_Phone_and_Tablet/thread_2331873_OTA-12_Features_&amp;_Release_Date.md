### Post by howefield on 2016-07-27
Released today, you should have the update sometime over the next 24 hours.

And posted today...

Hello everyone!

Good news! OTA-12 passed final testing and will be released tomorrow
around UTC morning. We will of course proceed with phased updates as
always, with the update appearing on all user devices in ~20 hours
since the release. We're finalizing the release notes and will send an
official announcement once the images are copied to the stable channel
tomorrow. Big thanks to everyone working on this milestone!

In the meantime we're constantly looking into the emulator state. Pat
did some investigation and probably found one of the reasons for the
networking issues [1] which we might be able to fix in the upcoming
days. We'll then respin OTA-12 emulator-only images with those fixes
in place for a separate release.

As for OTA-13 landings, today we had a new ubuntu-system-settings with
some test fixes and visual changes, a bugfix for indicator-network and
a hotfix release for libertine (updating bash completions for new
configure syntax, installing software-properties-common before adding
the overlay PPA). More staged in silos for upcoming days.

Cheers!
