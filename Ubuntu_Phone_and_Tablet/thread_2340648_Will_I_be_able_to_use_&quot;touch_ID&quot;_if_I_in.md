### Post by ajgreeny on 2016-10-20
Hello **willylc** and welcome to the forum.

Firstly, I think you may be disappointed with ubuntu on your mobile phone, if it will even be possible to install it, and it will certainly not give you any real idea of how Ubuntu runs on a PC.

As you have Windows 7 and it is probably installed on a machine with legacy BIOS, not UEFI, I suggest you start with a dual boot setup, much easier on a BIOS machine than UEFI.

There are several guides as to how to do that but one of the best, in my opinion is at [http://www.dedoimedo.com/computers/ubuntu-14-04-install-guide.html](http://www.dedoimedo.com/computers/ubuntu-14-04-install-guide.html)
It is for 14.04 but the installation is much the same for 16.04, which is the version I recommend you use as 16.10 is supported for 10 months only.
