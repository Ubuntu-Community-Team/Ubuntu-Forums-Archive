### Post by coffeecat on 2015-10-09
Yes, it's very easy. I'm posting from my Ubuntu 15.10 laptop connected to my Aquaris E5 phone hotspot. Simply go to System Settings in the Ubuntu Phone -> Hotspot to set it up. 

Caveat: I've only just done this as a test as I was intrigued by your question, so I cannot speak from long-term experience, and I have both devices in the same room at the moment. But I will experiment with this to see how robust it is as a backup to my dsl. Particularly, I would like to see how mush the range compares with domestic wi-fi from a conventional wireless router.

**Edit:** just to add, and I'm going from memory here. I believe the hotspot utility came in with an OTA update. If you get a BQ phone and it comes with the same default 14.10 OS as mine came with, and there's no hotspot in system settings, do all the updates and it should appear.
