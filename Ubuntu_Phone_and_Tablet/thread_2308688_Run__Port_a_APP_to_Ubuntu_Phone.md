### Post by grahammechanical on 2016-01-06
It is my understanding that Android apps will not run on Ubuntu phone. It is entirely possible for an Android developer to port his or hers Android app to Ubuntu phone. How much re-writing of the code this would require I do not know but I imagine that the procedure would be the same as for all Ubuntu phone apps. And that is, Use the Ubuntu SDK to write the app, package it as a click app and then submit it for approval into the Ubuntu app store.
