### Post by grahammechanical on 2014-09-09
Ubuntu for devices has a set of core apps which includes a document viewer.

[https://wiki.ubuntu.com/Touch/CoreApps](https://wiki.ubuntu.com/Touch/CoreApps)

Ubuntu phones will have an app store that we can use to install apps. All these apps have been security checked to make sure that they cannot do anything bad. They will be tightly controlled to prevent them accessing data that they have no right to access. They will all run in a sandbox to isolate them.

If an application is not in the app store then it should be considered a security risk. I am sure that the phone makers will have a large say on what we can and cannot do to their devices. We should not think of Ubuntu phones as Ubuntu on a phone and that we can do with the phone OS what we do to Ubuntu on a desktop. That I do not think that is going to happen. Not without voiding the warranty.

Applications that are suitable for a desktop monitor are not suitable for a phone or tablet size screen.  Right now Ubuntu developers are working on the phone and tablet apps to make sure they scale for the desktop for when Ubuntu for devices is converged into Ubuntu for the desktop. It is the responsibility of application developers to port their applications so that they scale for mobile devices and then apply for them to be accepted into the phone app store.

See the features of Ubuntu for devices. I htink that you will find you will not need applications like Calibre. Swipe gestures will be enough as each instance of an application runs on it own surface and we can swipe through them.

[http://www.ubuntu.com/phone/features](http://www.ubuntu.com/phone/features)

Regards.
