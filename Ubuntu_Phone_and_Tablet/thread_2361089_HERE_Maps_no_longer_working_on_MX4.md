@andrew-q2: Support for ubuntu-touch is being moved to uBports, and it will be supported by the community as Canonical decided not to invest in it anymore. There is already an update for the telegram and unav applications there, and for uNav I'm using "mapbox" and it works as well as OpenStreetMap, I already drove a couple of times with it. You should install OpenStore and update these applications from there. Ubports is also saying that they will provide images to flash for the existing models soon, so that we can keep getting updates from there on.
