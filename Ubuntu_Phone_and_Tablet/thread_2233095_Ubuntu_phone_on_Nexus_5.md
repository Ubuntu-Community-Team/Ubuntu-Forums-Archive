### Post by csc2 on 2015-03-11
You will have worse battery life and it only supports GSM (bad for North America if you want an actual phone for phoning and such). As far as support goes there isn't much these were never supported like the Nexus 4/7 I haven't tried many apps on it but so far it did come with ssh server (file browser isn't attaching RSA keys in gmail for no good reason so i can only ssh from not into) and it gets a lot of 404 errors when you go to update via terminal which is generally a bad thing on a new install. I posted a question about USB OTG a couple days back and got 200 views and 0 replies so I think that you're going into developer territory if you want to do serious stuff oth these phones. 

I recommend two things:
1) Complaining until one of the dev gods notices this thread and enlightens us ;)
2) See if the nexus 5 is supported by multirom and keep droid on your phone unlike a nood like me that just flashed it with ADB and got rid of Android in the process 

It is cool to try and if you have an extra nexus 5 sitting around I'd recommend it just for hahas if you're into that kind of thing
