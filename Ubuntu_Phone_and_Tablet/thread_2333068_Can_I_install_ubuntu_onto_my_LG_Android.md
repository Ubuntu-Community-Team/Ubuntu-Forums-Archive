### Post by wlbi on 2016-09-24
The BQ M10 don't have ethernet or serial port. A regular serial port plug is double thick than the M10 tablet is. So how can it be? ;-)

The only port available is USB. If you like to connect ethernet, you need a USB to ethernet adapter.

[http://amzn.to/2d6XHs8](http://amzn.to/2d6XHs8) and also micro USB to USB adapter [http://amzn.to/2cKf2VY](http://amzn.to/2cKf2VY)
