### Post by grahammechanical on 2015-06-12
At present the Ubuntu phone is a phone. We should not expect it to be  anything else. I suggest that you wait until there is a Ubuntu converged  device on the market that can be used as a phone, tablet and PC if it  is connected to the right peripherals before recommending the Ubuntu  phone as anything more than a phone.

At lot of applications that  are at present in the Ubuntu Software Centre will not be included in the  app store of a converged device because they are not packaged as either  a Click or a Snap package. They also break the security model of the  Ubuntu phone.

By the time Ubuntu 16.04 desktop is released there  will be 2 variants of Ubuntu desktop. One variant will be as Ubuntu is  at present. That is running on X and with Unity 7 and applications  packaged with the Debian package format and upgdated/upgraded using  apt-get

The other variant will be running on Mir with Unity 8 and  applications packaged as Snap packages and receiving transactional  updates. It remains to be seen how many of the applications at present  in the Ubuntu Software Centre will make the switch to the Mir, Unity 8 and  Snap package variant. 

Regards.
