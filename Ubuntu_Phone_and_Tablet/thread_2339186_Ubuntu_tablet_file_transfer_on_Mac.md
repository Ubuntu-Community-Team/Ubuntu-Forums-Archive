### Post by mike_johnson2 on 2016-10-05
Well, I found a solution.
I saw the Aquaris M10 was originally an Android device, repurposed as an Ubuntu tablet, so...,
The [android file transfer]("https://www.android.com/filetransfer/") app happened to work. It opens up like a finder window.

It has the available attached drives listed as buttons at the top.
I was free to drag and drop right to the location I wanted. :D
