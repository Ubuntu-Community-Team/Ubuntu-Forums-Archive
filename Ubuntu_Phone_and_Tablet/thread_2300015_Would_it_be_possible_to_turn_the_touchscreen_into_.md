### Post by grahammechanical on 2015-10-23
I do not think so. Based on the discussions & videos I have listened to and watched as I have followed the development of the Ubuntu Phone OS I have never heard any talk of the phone screen being converted to a touch pad.

The aim, as far as I can tell, is for there to be a Ubuntu Mobile device that has hardware connectors where we can connect a monitor (touch screen or not), a keyboard and a mouse. On the monitor we will get a Ubuntu desktop user interface running the sort of applications we would expect from a desktop install of Ubuntu. It should also run the Ubuntu phone apps on the monitor but scaled for the monitor and it would include the software to continue to use the device as a phone. The Phone itself will remain as a phone. 

There is no information as to what sort of dock, if any will be available. Hardware specifications are the responsibility of the OEMs that have partnered with Canonical to produce Ubuntu phones and tablets.
