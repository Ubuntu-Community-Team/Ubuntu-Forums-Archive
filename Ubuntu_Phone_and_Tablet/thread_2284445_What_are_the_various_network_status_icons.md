### Post by CybaCowboy on 2016-02-20
In ascending order (from slowest to fastest)...
* "G" usually means that you are using data via General Packet Radio Service (GPRS);
* "E" usually means that you are using data via Enhanced Data rates for GSM Evolution (EDGE);
* "H" or "H+" usually means that you are using data via High-Speed Downlink Packet Access (HSDPA);
* "4G" or "LTE" usually means that you are using data via a Long-term Evolution (LTE) data connection.

The symbols are written into the operating system itself and will be the same for every mobile device running Ubuntu for Mobile Devices, but it is the network-in-use which will determine the icon that appears...
