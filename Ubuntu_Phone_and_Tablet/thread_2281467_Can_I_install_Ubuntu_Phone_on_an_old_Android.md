### Post by Bucky Ball on 2015-06-07
Ubuntu Phone (and tablet) are analogue, hardware objects in the 'real world', not software. I think you are after Ubuntu Touch or Ubuntu Snappy, which is the software designed for phones and tablets, and no, I don't think you can install either on old Android hardware. A bit of research on Ubuntu Touch and Snappy will shed further light ...

Snappy is still very much in development but it is getting there.
