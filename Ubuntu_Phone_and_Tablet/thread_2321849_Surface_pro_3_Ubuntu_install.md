### Post by oldrocker99 on 2016-04-25
It isn't impossible that you got a corrupted Ubuntu ISO, which certainly could have caused the problem. Did you check the /b]md5 sum?[/b] Copy the md5sum for your ISO and use this to find its sum:```
md5sum name_of.iso
```
That will, after a couple of minutes, return a long hexadecimal number, which should be the same as the one on the download page. That the only way to check for an uncorrupted download. If you torrent the ISO, the md5 sum is automatically checked.
