### Post by vasa1 on 2015-11-11
> **grahammechanical said:**
> I understand that work is being done to enable the document viewer app to read all the file formats that Libreoffice can read. ...
OP specifically wants to know about .docx:
>  Is it possible to open up.docx files on my Ubuntu phone?A more blunt answer would be that that is uncertain. docx is closed source. Developers without access to docx code can only play catch up by trying to decode or reverse engineer what's being done to make docx what it is. IMO, it's quite possible that one docx document may open with 100% fidelity; another may not.

If it's not too much trouble, you should try asking people you know to use a genuinely open format for exchanging information rather than use what some people claim is the *de facto* "standard".

Some links:
