### Post by grahammechanical on 2016-04-21
Page 37 of the user manual

[https://it-bqcom15-media.s3.amazonaws.com/prod/resources/manual/Aquaris_M10_Ubuntu_Edition_Complete_User_Manual-1460548936.pdf](https://it-bqcom15-media.s3.amazonaws.com/prod/resources/manual/Aquaris_M10_Ubuntu_Edition_Complete_User_Manual-1460548936.pdf)

Open the applications scope and you are looking for a large upright orange pillar of an icon with a stylised capital letter A. You will most likely need to create a Ubuntu One account to install any apps. But you need one of those anyway to update both the OS & the apps.
