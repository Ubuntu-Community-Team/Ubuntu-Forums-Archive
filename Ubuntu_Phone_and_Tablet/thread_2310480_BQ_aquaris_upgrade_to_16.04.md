### Post by grahammechanical on 2016-01-19
It is my understanding that all existing Ubuntu phone models will receive Over The Air (OTA) updates to the Ubuntu phone OS. And so all Ubuntu phone models will be kept up to date.

First, the phone code has to be upgraded to be based on Ubuntu 15.10 code. I am not sure if that has been done. Or if the developers are going to bypass 15.10 and move on to re-basing Ubuntu phone OS on the 16.04 code base sometime after it has been released.

Certainly, the design for upgrading the Ubuntu phone OS is not the same as the design for upgrading the Ubuntu desktop OS. Do not expect Ubuntu phone to have a 6 monthly release schedule like the Ubuntu desktop. It has not worked that way up until now.
