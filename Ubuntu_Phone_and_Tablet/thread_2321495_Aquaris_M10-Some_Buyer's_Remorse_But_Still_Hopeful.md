### Post by thufirhawat2 on 2016-04-22
So I got my M10 yesterday, and I definitely feel like either the advertising was misleading, or they sent the units out long before they had core functionality fully developed. For anyone considering getting one, please note that when they say "desktop experience," they basically mean you can resize the apps in windows; that's about the only difference. At this time there are a couple of apps like firefox and libre office that are designed to function with a keyboard and mouse, but they run kinda poorly. The lack of a native file manager is super lame; you have to download it from the app store, and it is severely limited. There is no way that I have found to connect to ftps or ssh for file-sharing other than using the (once again) not-native terminal from the app store, and manually using scp to copy files to and from a server. When they said "full desktop experience," I kinda thought it would mean it would be like sitting at a PC with Ubuntu OS installed where you could install programs that would run on Ubuntu such as chrome, grsync, veracrypt, etc. But what it really means is that there are two whole new divisions of programs you have to deal with. There are now tablet apps that work in "tablet mode," tablet apps that work in "desktop mode," and regular Ubuntu software that runs on a desktop or laptop with Ubuntu installed. To me, this sort of defeats the purpose of the concept of convergence, it should be easier to run programs that you have been running on Ubuntu computers, and then add the tablet aspect to it for easier access to certain programs when you are on the go. My big two beefs are not being able to use veracrypt and netflix, at least I have not found a way to get either one to work, I would be pretty stoked if someone out there who has found a way could enlighten me though. I was also under the impression that as it was built around android, you would have access to the android app store, but this is also not the case, the apps have to be written specifically for Ubuntu touch, so your selection of apps is way, way limited.

I am holding out hope that they are still getting the kinks worked out and they will develop the platform to better accommodate standard Ubuntu desktop software while in desktop mode, but I was kind of under the impression that was supposed to be the idea from the get-go. All around, I have found surprisingly few glitches which is encouraging. My location icon does seem to blink off and on nonstop since the update to version 3, but I just turned location services off for now since I don't use them anyway. 

If someone has found some hacked-up way to get netflix and FTPS/SSH filesharing to work though, please share!
