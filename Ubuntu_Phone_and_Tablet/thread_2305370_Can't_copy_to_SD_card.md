### Post by jtappin on 2015-12-10
You can access it with the file manager (and also allow some apps such as camera to access it). 

For the file manager, you need to:
1: Enable full access (from the pulldown in the top right)
2: Bring up the places list by dragging up from the bottom edge.
3: Select the SD card in that list.
