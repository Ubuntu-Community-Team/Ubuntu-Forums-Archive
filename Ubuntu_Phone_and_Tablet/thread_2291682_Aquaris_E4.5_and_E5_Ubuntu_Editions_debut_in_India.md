### Post by grahammechanical on 2015-08-22
This news report illustrates a problem with some popular applications

[http://www.bbc.co.uk/news/technology-34016658](http://www.bbc.co.uk/news/technology-34016658)

The Ubuntu phone OS has strict controls over what an application can access. A music app would not be allowed access to the address book and pictures folder. The request would have to go through what is called the content hub which will then force the user to authenticate. Owners of popular applications would need to make their applications comply with these security requirements. It is also another reason why it is best to install apps that are in the app store. Apparently, it is very easy to get an app accepted in the app store.
