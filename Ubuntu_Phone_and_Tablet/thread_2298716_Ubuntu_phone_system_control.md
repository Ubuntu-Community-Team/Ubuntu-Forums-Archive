### Post by grahammechanical on 2015-10-13
Ubuntu Phone and Ubuntu desktop are not the same. I think that it is also true that Ubuntu desktop does not offer the same level of modification that other user interfaces do. It would be a mistake to think of Ubuntu phone as a skinny version of Ubuntu desktop. It is not.

I do not think that it is similar to Android either. Ubuntu phone is not meant to be similar to Android. The developers are coming at a phone OS from a different direction. There is a lot of information at Ubuntu.com about Ubuntu phone and also at the web sites of BQ and Meizu. Make sure that a Ubuntu Phone is what you want before you buy. It is still very much an early adopter's device.
