### Post by grahammechanical on 2016-01-08
The full Ubuntu operating system is not designed to run on any smart phone.

Canonical has developed a Ubuntu OS to run on OEM supplied Ubuntu phones. But it is not the full Ubuntu operating system although the OS is being developed to become a full Ubuntu OS when the mobile device is connected to a monitor, keyboard & mouse. The intention is for this Ubuntu phone OS to a phone OS when pre-installed on Smart phone devices and a tablet OS when pre-installed on tablet devices and a PC OS when either type of Ubuntu mobile device is connected to a monitor, keyboard & mouse.

I do not think that is accurate to say that a Ubuntu phone is running the full Ubuntu operating system although all the code may be present. But that is yet in the future. A not too distant future we hope.

Canonical is not developing this phone OS to be downloaded and installed on any & all mobiles devices in the way we download desktop Ubuntu. But as the code is open source there are some private developments at porting Ubuntu phone to other devices.

[https://wiki.ubuntu.com/Touch/Devices](https://wiki.ubuntu.com/Touch/Devices)
