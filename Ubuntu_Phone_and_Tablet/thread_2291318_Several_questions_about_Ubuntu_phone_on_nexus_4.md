### Post by grahammechanical on 2015-08-19
Q1 = It is an experiment. Flashing Ubuntu Touch to any device was meant for developers only so that they could learn about the Ubuntu phone OS and write apps for it. See, the advice about this:

[https://wiki.ubuntu.com/Touch/Devices](https://wiki.ubuntu.com/Touch/Devices)

Q2 = No. Ubuntu for Devices is not that kind of Ubuntu. Ubuntu phones run on Mir and not Xserver. They do not use LightDM. And Unity 8 is the crafted UI for Ubuntu phones. I would be very surprised if any Canonical engineer was working on porting another desktop environment and user interface to Ubuntu for Devices. Unless it was in their own time and for their own amusment. There is a terminal app.

Q3 = No. The best option is to buy a Ubuntu phone from either BQ or Meizu or wait until either of those OEMs releases a Ubuntu tablet device. Later this year a device will be released for retail that will be a converged Ubuntu mobile device. It will become a PC if plugged into a monitor, keyboard and mouse. How much of the default Ubuntu desktop applications will be available we do not know. At the very least the UI will scale for the size of the monitor screen and the mouse and keyboard will be useable. And it can still be used as a phone. And it will get Over the Air (OTA) updates to the OS every six weeks. So, the OS will keep improving. But if you do not like the user interface then a Ubuntu mobile device is not for you.

Regards.
