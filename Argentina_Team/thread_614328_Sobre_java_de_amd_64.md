### Post by juanman on 2007-11-16
La forma más fácil de instalar java en amd64 es con automatix. Te lo bajas de [acá]("http://www.getautomatix.com/wiki/index.php?title=Installation") y después instalas el swiftweasel de 32 bits y los plug ins... Vas a estar instalando otro navegador (es un firefox optimizado), pero es lo más fácil. Creo q no se puede instalar java en el firefox de 64b...

Cuidado: si usas ubuntu feisty (7.04) solo usa automatix para instalar esto. No lo uses para instalar el driver de la placa de video! Si usás gutsy (7.10), no hay problema...
