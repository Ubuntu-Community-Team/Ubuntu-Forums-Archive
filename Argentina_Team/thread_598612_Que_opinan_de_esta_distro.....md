### Post by ubuntu27 on 2007-11-02
Otras distrbuciones GNU/Linux de seguridad que recomiendo son:

[Network Security Toolkit (NST)]("http://www.networksecuritytoolkit.org/nst/index.html")  - 
Network Security Toolkit (NST) es un Live CD basado en Fedora Core. Este toolkit esta disenyado para proveer lo mejos de las aplicaciones de seguridad opensource. Este distro corre bien en las palataformas x86. 
Lo que impresiona es que con NST, podemos transformar systemas x86 (de Pentium II +) en un sistema disenyado para analisis de trafico de red, deteccion de intrusos, generador de paquetes de red, monitor de redes inalambricos, servidor de servicio sistema virtual, e scaneador de host e redes sofisticadas.

[Traducido de [Distrowatch]("http://distrowatch.com/table.php?distribution=nst")]


[nUbuntu]("http://www.nubuntu.org/") -

nUbuntu es una coleccion de herramientas de seguridad para redes y servidores.
