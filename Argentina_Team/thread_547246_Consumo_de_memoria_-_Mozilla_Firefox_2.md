### Post by por100pre1 on 2007-09-12
¿Quieren un navegador liviano? Prueben links2, un navegador para usar en terminal:

```
sudo aptitude install links2
```

Para usarlo en terminal (con graficos pero sin estilos):

```
links2 -g www.ubuntu.com
```

Para usarlo fuera de X o sin graficos (puro texto):

```
links2 www.ubuntu.com
```

Puede serles útil algún día. :)
