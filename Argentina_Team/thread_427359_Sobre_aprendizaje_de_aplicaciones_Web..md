### Post by clasificado on 2007-04-30
Una que te recomiendo es crearte la necesidad de aprender ese tipo de cosas. (Uno nunca va a aprender a usar bien un rifle si nunca va a cazar)

Un servidor web (como apache) está hecho para servir sitios web, te recomiendo practicar instalando frameworks que sirvan para hacer sitios web.

En este caso te presento tres [CMS](http://es.wikipedia.org/wiki/Sistema_de_gesti%C3%B3n_de_contenido) que usan apache + algun lenguaje de programacion (asi te rompes la cabeza viendo como se instalan y configuran).

[Drupal](http://es.wikipedia.org/wiki/Drupal): Apache + PHP5
[skeletonz](http://orangoo.com/skeletonz/): Apache + Python
[movabletpe](http://www.sixapart.com/movabletype/): Apache + Perl
