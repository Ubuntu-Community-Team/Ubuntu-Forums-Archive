### Post by User21 on 2008-01-21
Tanto GIMP ( alternativo a Photoshop ) como Inkscape (alternativo a CorelDraw )  estan en los repositorios oficiales de Ubuntu. Esto quiere decir que puedes instalarlos desde Añadir o quitar programas, o desde el menú Administracion > Sistema > Gestor de paquetes Synaptic. Desde consola basta con hacer ```
sudo apt-get install gimp
``` o ```
sudo apt-get install inkscape
``` 
Lo mismo sucede con Virtualbox. ```
sudo apt-get install virtualbox
```
