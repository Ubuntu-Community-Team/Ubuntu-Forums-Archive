### Post by Mauro22 on 2007-11-08
Hola!

Podria por empezar posteando tu xorg.conf

```
script xorg.txt
```
```
cat /etc/X11/xorg.conf
```
```
exit
```

Todo eso genera un xorg.txt en tu home con el contenido de la consola (o sea el xorg.conf)
--Se que es mas facil copiar y pegar desde la consola, pero asi es mas pro :P--

Pega el contenido del archivo ese aca...

NOTA: nunca esta de mas hacer un backup de tu xorg

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```
Por si las dudas :)
