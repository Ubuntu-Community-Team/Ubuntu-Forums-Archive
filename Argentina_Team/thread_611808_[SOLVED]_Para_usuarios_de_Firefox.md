### Post by fscodelaro on 2007-11-13
Yo uso NoScript y puedo agregar unas cosas:

1) si es un poco molesto como dice Facundo, al principio habilitar el JS de todos los sitios de confianza. 

2) como dice Facundo, es cierto que no añade mucha más seguridad que por el hecho de usar ubuntu, de todas maneras es bueno el consejo porque muchos seguimos usando xp en el trabajo con firefox y es una buena extensión para agregar.

3) dicho sea eso, sí añade seguridad, aunque no sea mucha; en gnucitizen.org se la pasan encontrando fallas de seguridad en firefox que son multiplataforma, si bien es cierto que pueden aprovecharse más de computadoras con windows que con linux.

4) la seguridad no es la única razón para usarla. La extensión bloquea Javascript e incluso Java y muchas veces, muchas páginas tienen errores de JS/Java que hacen que el firefox se cuelgue (también por estar diseñadas para IE). En ese caso ahorra crashes. Segundo, bloquea muchas publicidades que usan javascript (el caso más tipico es adsense de google, pero muchos más también), hasta que uno los habilite. Tercero, la navegacion anda más rapido al no cargar todos esos elementos espurios.

Yo la recomiendo para todos los usuarios firefox en cualquier plataforma, y aunque no sea con el objetivo de la seguridad únicamente.

Saludos,
