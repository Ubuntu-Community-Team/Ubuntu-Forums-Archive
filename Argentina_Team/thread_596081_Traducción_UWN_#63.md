### Post by spg76 on 2007-10-29
Para todos los interesados en colaborar, ya está disponible para traducir la edición #63 del Newsletter Semanal de Ubuntu.
A partir de esta edición se realiza con un nuevo método usando Gobby, una herramienta colaborativa que permite la edición múltiple de documentos. 
Para instalarla en Ubuntu escriban en una terminal "sudo apt-get install gobby". Si quieren descargarla para otros sistemas visiten [http://gobby.0x539.de/](http://gobby.0x539.de/).
Una vez instalada, unanse a la sesión de traducción en gobby.ubuntu.com puerto: 6522, y busquen el documento "uwn-spanish_63.txt".
Lo bueno de este sistema es que la tradución puede realizarse por varios colaboradores de manera simuntanea e incluso permite la interacción entre ellos. 
Además cada uno puede traducir lo que pueda, desde una palabra hasta todo el documento.
Muchas gracias.
Saludos.
