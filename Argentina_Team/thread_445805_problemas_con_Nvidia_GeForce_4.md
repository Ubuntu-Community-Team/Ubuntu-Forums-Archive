### Post by ntetreau on 2007-05-16
Hola,
you'll have to make an effort to understand my english, perdon!  Ok, I doubt that you installation of the Nvidia driver worked.  If it did, the driver name would have changed from "nv" to  "nvidia" in your xorg.conf file.

Anyway, if I remember well, the geforce4 needs the "legacy" drivers, not the latest one.  

You can install them like this:

```
sudo aptitude install nvidia-glx-legacy
```

then,

```
sudo nvidia-xconfig
```

reboot, let us know if that worked.

buena suerte

### Post by Al_maverick on 2007-05-19
Con el tiempo vas a aver que los drivers de video son uno de los puntos debiles de linux, porque las empresas de video no hacen buenos drivers (o no hacen ninguno, directamente) y la mayoria tienen que hacerse por ingenieria reversa (un termino bonito para decir prueba y error)

Mas alla de eso, por lo que  veo estas bastante cerca de que funcione. Lo que podes hacer es entrar a Windows, y fijarte las resoluciones a las que anda tu placa, y los refrescos del monitor en cada resolucion. Una vez teniendo eso, se puede modificar el xorg para que tome esos modos. Lo que vas a tener que hacer es copiar las resoluciones y Hz de refresco para cada resolucion, y copiar tu xorg. Con eso te podemos decir como modificarlo.
