### Post by branau on 2015-06-25
1. Todos los sabores de Ubuntu están preparados para usar los núcleos de procesador, así por eso no te tienes que preocupar.
2. Yo personalmente uso el sabor estándar de Ubuntu, pero no me tengo que preocupar por el RAM ni nada de eso. No he usado los otros sabores, pero podrías investigar Lubuntu. Según es como la versión "light" de Ubuntu, y por lo tanto es mejor para equipos viejos. Todos los programas y paquetes compatibles con Ubuntu también son compatibles con Lubuntu, así no tendrías ningún problema con usar audacity.
3. Para decidir entre 32 bit y 64 bit, te recomiendo que busques la información de sistema desde el panel de control de Windows 7 para ver cuál dice ahí. Si tienes una versión de Windows de 64 bit, puedes instalar cualquier de las dos opciones, solamente que la versión de 32 bit no podría utilizar la máxima potencial del procesador, así te conviene instalar la 64 bit. De lo contrario, si tienes la versión de Windows 32 bit, tienes que instalar la misma de tu sabor de Ubuntu, ya que tu equipo no podría correr la versión de 64 bit.
