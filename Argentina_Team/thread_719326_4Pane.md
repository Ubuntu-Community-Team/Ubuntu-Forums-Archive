### Post by Sit3UserX on 2008-03-09
Hola!

Andaba buscando un file manager con doble panel, probé algunos mencionados en ubuntu forums y en varios sitios de documentación de ubuntu, pero al final, ninguno me dejó totalmente satisfecho.

Buscando en google y ya no recuerdo exactamente cómo ni desde que sitio, encontré el link hacia la web de [**4Pane**]("http://www.4pane.co.uk/index.html").

Lo primero que hice fue revisar las capturas de pantalla y me intersó. Luego revisé brevemente el manual y finalmente miré las instrcciones de instalación.

Me quedé interesado y decidí probarlo.

Sinceramente, para mi gusto y necesidad está perfecto.

Un detalle: para ejecutarlo hay que hacerlo desde la Terminall escribiendo 4Pane, aunque también puede hacerse con el diálogo Ejecutar aplicación (ALT+F2). Igualmente se puede crear un lanzador con el editor de Menú Principal (Sistema => Preferencias => Menú Principal).
