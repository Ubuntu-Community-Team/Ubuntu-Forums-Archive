### Post by marianom on 2007-04-28
Dos formas:
generalmente los comandos vienen con un ayuda al cual se accede con --help o -h
Ejemplo:
```

ifconfig --help

```

Una ayuda más amplia la tendrás si usás man:
```

man *comando*

```

Ejemplo:
```

man ifconfig

```
