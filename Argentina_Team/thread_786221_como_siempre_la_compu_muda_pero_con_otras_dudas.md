### Post by atatiducken on 2008-05-08
> después de buscar el "7z" travesti intenté abrir el archivo y me gané un "seguí participando" jaja muy gracioso, para descomprimir cosas yo uso el **unrar** que lo podes instalar desde synaptic, podes instalar **ubuntu-restricted-extras** que trae el unrar y otras cosas mas como flash, java y otras cositas... sino por consola ```
sudo apt-get install ubuntu-restricted-extras

```
Si tu procesador de textos es openoffice, para que este en español instala desde synaptic el **openoffice.org-l10n-es**, por terminal ```
sudo apt-get install openoffice.org-l10n-es
``` si tenes hardy creo q ya esta instalado, por lo q deberias abrir openoffice e ir a** Tools > options... > Language settings > Language** y alli seleccionar español en user interface y los que hay abajo, despues de esto reinicias oppenoffice y listo esta en español
