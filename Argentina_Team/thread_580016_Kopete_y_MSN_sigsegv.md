### Post by leo_rockway on 2007-10-18
Si estan en gutsy y al intentar conectar a la red msn desde kopete les tira un error sigsegv y se cierra, tienen que bajar e instalar este parche:

Para i386:
[http://kubuntu.org/~jriddell/kdelibs4c2a_3.5.8-0ubuntu3_i386.deb](http://kubuntu.org/~jriddell/kdelibs4c2a_3.5.8-0ubuntu3_i386.deb)

Para 64
[http://kubuntu.org/~jriddell/kdelibs4c2a_3.5.8-0ubuntu2_amd64.deb](http://kubuntu.org/~jriddell/kdelibs4c2a_3.5.8-0ubuntu2_amd64.deb)

Fue un problema con el upgrade de kdelibs. Para saber mas pueden leer el bug de launchpad:

[https://bugs.launchpad.net/ubuntu/+source/kdelibs/+bug/153500](https://bugs.launchpad.net/ubuntu/+source/kdelibs/+bug/153500)
