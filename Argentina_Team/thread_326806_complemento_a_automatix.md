Automatix2 Bleeder Para Ubuntu Edgy 6.10 
Automatix2Bleeder es una 'extension' que completa a automatix2, que nos da la opcion de instalar beryl y driver's para ati y nvida, y demas funciones.
primero instalaremos Automatix2,
después hacemos
udo apt-get install automatix2bleeder

Una vez instalado podemos ejecutarlo iendonos a:

Aplicaciones -> Herramientas de Sistema -> Automatix2Bleeder
