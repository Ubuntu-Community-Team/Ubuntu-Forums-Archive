### Post by leo_rockway on 2008-01-24
Hace algunos dias salio la version prealfa de Amarok 2... 

Aquellos que quieran instalarlo para probarlo y ayudar a reparar bugs (no a reportarlos, ya que no va a haber reporte de bugs hasta que salga la primera alfa oficial) agreguen la siguiente linea a sources.list

```
deb http://ppa.launchpad.net/kubuntu-members-kde4/ubuntu gutsy main
```

y despues ```
sudo apt-get update && sudo apt-get install amarok2 amarok2-phonon
```

yo todavia no lo instale, quizas despues le pegue un vistazo... si alguien lo instala que comente sobre los nuevos features y eso.
