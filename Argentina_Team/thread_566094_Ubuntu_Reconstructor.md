### Post by spg76 on 2007-10-03
Reconstructor es un programa que te permite modificar el CD de instalación (Desktop, Server o Alternate) para personalizarlo.
Para usarlo tenes que instalarlo en Ubuntu y después modificar el CD a tu gusto. Una vez hecho esto te genera una ISO para que la grabes en un CD y puedas correlo en cualquier PC.
