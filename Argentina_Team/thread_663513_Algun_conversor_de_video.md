### Post by erginemr on 2008-01-10
Hola,

Como una herramienta de CLI, puede utilizar el "ffmpeg" del repositorio de medibuntu. Como un conversor GUI a 3GP, puede probar uno de estos programas, que son gratis:

[http://www.miksoft.net/mobileMediaConverter.htm](http://www.miksoft.net/mobileMediaConverter.htm)
[http://ubuntuforums.org/showthread.php?t=567016](http://ubuntuforums.org/showthread.php?t=567016)

Para el primer programa, también necesitará instalar el paquete libstdc++5 con
"sudo aptitude install libstdc++5"

¡Buena suerte!
