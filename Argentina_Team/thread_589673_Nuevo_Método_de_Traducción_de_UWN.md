### Post by spg76 on 2007-10-24
Hola a todos.
Les quería informar que hay varios equipos de traducción del Newsletter Semanal de Ubuntu (UWN) que se están juntando para poder de alguna manera centralizar las traducciones.
Para esto se va usar un servidor con Gobby ([http://gobby.0x539.de](http://gobby.0x539.de)) para que más gente pueda colaborar.
Para quienes no la conocen Gobby es un editor colaborativo que soporta múltiples documentos en una sesión y chat multiusuario. Puede correr tanto en Linux como en Microsoft Windows y Mac OS X.
Para instalarlo en Ubuntu ingresen en una terminal "sudo apt-get install gobby". Para otras versiones visiten [http://gobby.0x539.de/trac/wiki/Download](http://gobby.0x539.de/trac/wiki/Download).
Los que quieran colaborar pueden anotarse en [https://wiki.ubuntu.com/ArgentinaTeam/TraductoresUWN](https://wiki.ubuntu.com/ArgentinaTeam/TraductoresUWN). Cuando se inicie la traducción tendran que ingresar con Gobby a gobby.ubuntu.com puerto: 6522 donde encontrarán el documento a traducir y podrán chatear con otros traductores.
Muchas gracias.
Saludos.
