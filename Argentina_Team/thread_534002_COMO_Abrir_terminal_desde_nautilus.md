Hola a todos!!

Bueno, les dejo este comando para agregar el "Abrir terminal aqui" en el menu contextual del nautilus


Ir a la consola y poner:

```
sudo apt-get install nautilus-open-terminal
```

y despues se reinicia nautilus

```
killall nautilus
```
