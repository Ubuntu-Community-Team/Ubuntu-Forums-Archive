### Post by gmunioz on 2015-04-02
Los archivos .CDA son simples accesos directos que proporcionan la información disponible de un CD de audio, para reproducir desde el CD la pista de referencia indicada en el archivo al abrirlo. 
El CD naturalmente debe ser un Audio-CD y contener el formato propio de un CD de audio,  para que este acceso directo funcione correctamente. 
**Los archivos .CDA no incluyen datos de audio**, pues las canciones se copian del CD de audio y se guardan en archivos sin comprimir.
Los archivos .CDA no contienen datos de onda de sonido PCM, solo le indican al dispositivo lector en qué lugar del disco empieza y termina la canción. 
Es posible que insertado un Audio-CD, y utilizando un administrador de archivos como Total Commander, se hayan copiado esos accesos directos desde el CD al Pendrive.
Para extraer el audio de un CD, es necesario un rippeador.
