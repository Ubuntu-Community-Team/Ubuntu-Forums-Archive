### Post by spg76 on 2006-12-18
> **pump said:**
> De donde lo sacaste? Estaria bueno tenerlo.
A lo mejor hay algun script para pasar de chm a html o a algun otro formato.

Según [este post]("http://www.ubuntuforums.org/showpost.php?p=1171575&postcount=5") se puede hacer así:

```
$ sudo apt-get install chmlib-bin
$ extract_chmLib tero.chm tero/
```

Donde dice "tero.chm" debería ir el nombre del archivo.

Para lo que esten interesados en libros de computación (en inglés) pueden ir [acá]("http://www.ebookshare.net/")
Y [acá]("http://www.ebookshare.net/2006/09/29/prentice-hall-ptr-the-official-ubuntu-book-aug-2006/") puede ver el libro oficial de Ubuntu.
