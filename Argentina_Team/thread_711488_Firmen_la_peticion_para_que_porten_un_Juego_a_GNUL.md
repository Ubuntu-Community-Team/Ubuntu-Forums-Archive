### Post by pante on 2008-03-04
Yo voté la petición.

Aunque apoyo totalmente el software libre, votar una petición acerca de un software no libre no tiene nada de malo. No significa que les esté dando mi apoyo, ni mi dinero. En realidad _con esto le doy mi apoyo a Linux que sí es software libre y que se acompaña de mucho software libre_.

Es lo bueno del software multiplataforma: no te ata a un sistema operativo. Sos libre de elegir. Cuando alguien usa habitualmente Firefox/Opera, OpenOffice, Gimp, VLC/Media Player Classic, etcétera, ¿qué importa qué sistema operativo uses? Mejor aún, ¿qué te cuesta pasarte a Linux? Y una vez en Linux te vas acostumbrando a usar software libre. Y un día te parece repugnante el software no libre.

A muchos les resulta chocante dar el gran salto de Windows a Linux, porque es terreno desconocido, todo es nuevo. Si la transición es gradual, a través del software multiplataforma, pasarse a Linux no es tan chocante.

Es lo que pienso.

Saludos:)
