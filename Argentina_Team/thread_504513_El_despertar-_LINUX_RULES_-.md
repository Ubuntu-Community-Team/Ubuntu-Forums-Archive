### Post by jpmorelli on 2007-07-19
UUUUhhhhh instalar el kde es RE - dificil !!!!!! JEJE...

Tenes que abrir una consola ( gnome-terminal ) y tipear:

sudo apt-get install kubuntu-desktop
o
sudo apt-get install kde
o
sudo apt-get install kde-core

Cada una de estas opciones corresponde a:

(i) kubuntu-desktop -- Este es el metapackage recomendado a instalar; la instalación full de Kubuntu, con todos los paquetes recomendados por Kubuntu. Incluye OpenOffice, Kontact, Konversation, amaroK, K3B, y otros.

(ii) kde -- Este instala los siguientes paquetes KDE: kde-amusements, kdeaccessibility, kdeaddons, kdeadmin, kdeartwork, kdegraphics, kdemultimedia, kdenetwork, kdepim, kdesdk, kdeutils, kdewebdev, kdevelop3 y el kde-core metapackage.

(iii) kde-core --- Este instala solo el núcleo de KDE -- Lo mínimo requerido para el entorno KDE. Esto es, kdebase, kdelibs, arts y fontconfig.
