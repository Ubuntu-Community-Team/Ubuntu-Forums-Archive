### Post by User21 on 2008-04-03
Vas a tener que instalarte emerald. Con el, podes cambiar el borde de ventanas bajo compiz fusion, y a su vez, obtener vistosos efectos, como transparencias, brillos, animaciones.
```
 sudo apt-get install emerald 
```
Una vez que instalaste Emerald, debe estar activado en CompizFusion el plugin "Decorador de ventanas" y ademas, debes bajarte temas de bordes de ventanas, desde [http://www.gnome-look.org](http://www.gnome-look.org) .
Suerte!
