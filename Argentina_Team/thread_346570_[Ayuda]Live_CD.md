### Post by ubuntu27 on 2007-01-27
Hola **Biasoli** Como te va?
Y abajaste [Ubuntu]("http://www.ubuntu.com/") y [Knoppix]("http://www.knopper.net/knoppix/index-en.html")?

Cuando bajes Ubuntu, baja el ISO (imagen de CD) que terminal con DESKTOP.

DESKTOP.- Es el LiveCD, podras probar Ubuntu GNU/Linux sin instalar. Es lento porque lee directamente del CD y RAM. Puedes instalar con este CD tambien.

ALTERNATE.- Es el CD de solo instalacion a modo de texto. Te viene con opciones avanzadas que no te ofrece en el Desktop.


Tengo algunas preguntas:

Cuanto de RAM (Random Access Memory) tienes?

Si es menos de 256MB (como 128MB) sera mejor que uses otro Distribucion o Ubuntu con Xfce u otro DE mas ligero. 
Hay una distribucion que se llama [Xubuntu]("http://www.xubuntu.org/"), lo cual es Ubuntu con Xfce. [Xubuntu]("http://www.xubuntu.org/") es bueno para maquinas con poca potencia en Hardware. Es usado tambien por gente que le gusta tener una makina rapida.



UNA COSA MAS:

Cuando bajes el ISO, te recominedo que lo bajes usando algun cliente de Bittorrent.
Bittorrent te permite bajar archivos desde distintos peers (otras comps usando el mismo cliente) conectados quienes tambien estan bajando el mismo archivos, lo cual hace que cunto mas gente este conectado, mas rapido la descarga es.

Ademas, bittorrent tambien revisa si tu descarga fue axitosa o fue corrumpida.

Cunado bajas directo si bittorrent, hay mucha posibilidad que que tu descarga se corrupete, lo cual te dara un mal CD.




Ahora, desdepues de bajar el ISO.
Tienes que quemarlo en un CD blanco con la opcion de IMAGEN.
No se que programas usas para quemar CDs, pero casi todos tienen la opcion de quemar un archivo como imagen de cd.

Si no tienes unsa, podras bajar 


[CDBUrnerXP Pro]("http://www.cdburnerxp.se/")

Link:
[Un tutorial para bajar y quemar ISO en ingles. Contiene imagenes.]("http://www.psychocats.net/ubuntu/iso")

### Post by Hex_Mandos on 2007-01-27
Mirá, Speedy es un desastre y no se lo recomendaría a nadie (lo usé por algo así como un año hace un tiempo), pero ahora estoy con Arnet y no es muy bueno tampoco. Yo probaría, y si no aceptan seguí molestando. Y cambialo por un ethernet SI O SI. Son mejores en muchos sentidos que los USB: desde razones taradas como que te ocupan un puerto menos, hasta el hecho de que procesan la información internamente, no via software. Y por supuesto, la cuestión de la configuración en Linux que es mucho mas simple: Un modem Ethernet lo podés configurar internamente desde un navegador, el USB usa un firmware instalado en tu máquina. Eso lo hace mucho mas barato (viste lo livianos que son?), pero peor para el usuario. Lo mismo que los WinModems dial up, que usaban (usan) tu máquina para procesar, en vez de funcionar autonomamente.
