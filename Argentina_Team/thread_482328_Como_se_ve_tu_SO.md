```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_respaldo

```

```
sudo gedit /etc/X11/xorg.conf
```

Busca la sección identificada como:

```
Section "Module"
```

y al final de ella agrega lo siguiente:

```
Load "dbe"
```

no olvides usar TAB en lugar de espacio para que no se dañe la estructura de xorg.conf
