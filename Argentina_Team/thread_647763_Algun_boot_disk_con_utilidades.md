Y respondiendo al primer mensaje:

Una buena "caja de herramientas" libre puede ser Knoppix (todo un clásico dentro de las distribuciones Live). Se ejecuta como "Live CD" (aunque puede instalarse opcionalmente) y tiene muchas herramientas (incluso de consola) para usuarios avanzados/técnicos. La última versión incluso incorpora en forma nativa la escritura en particiones no-linux (NTFS). Y para los exigentes, los que siempre piden más, existe una versión en DVD.

Para saber más: [http://es.wikipedia.org/wiki/Knoppix](http://es.wikipedia.org/wiki/Knoppix)

Esta distribución está castellanizada y se puede descargar desde aquí:

[http://www.knoppix-es.org](http://www.knoppix-es.org)

Saludos !

### Post by pante on 2007-12-24
Ahora mismo el que más a mano tengo (tengo el CD a la vista) es el System Rescue x86 0.4.1.

"System Rescue es un sistema Linux en un CD-ROM booteable para reparar tu sistema y recuperar tus datos. Su objetivo es proveer una manera fácil de llevar a cabo tareas administrativas en tu computadora, tales como crear y editar las particiones del disco duro. Contiene muchas utilidades de sistema (parted, partimage, fstools, ...) y herramientas básicas (editores, midnight commander, herramientas de red). Es muy fácil de usar: simplemente arranca el CDROM. El kernel soporta la mayoría de los sistemas de archivos (ext2/ext3, reiserfs, reiser4, xfs, jfs, vfat, ntfs, iso9660), así como sistemas de red (samba and nfs)."

La página en español de System Rescue es: **[http://www.sysresccd.org/Página_principal](http://www.sysresccd.org/Página_principal) **
