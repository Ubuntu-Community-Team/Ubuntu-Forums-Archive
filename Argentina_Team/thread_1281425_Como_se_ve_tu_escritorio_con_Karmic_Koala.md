Aprovechando que hay mucha gente que ya esta usando esta nueva version y dado el exito que tiene este tipo de threads, lo abro "a pedido del publico presente" con las siguientes pautas:

No seran permitidos desvios, off topics, flames, etc. Todo comentario debe versar sobre los escritorios, como se logran, que elementos se han utilizado y como configurarlos (conky, plasmoids, widgets/screenlets, docks, superkaramba, tipografias, iconos, themas, colores, etc., etc.).

[LIST]
[*]Tal como se hizo en su oportunidad y si estan de acuerdo, propongo reflotar el tema del concurso/competencia (ver thread similar pero sobre HardyHeron o Intrepid) para determinar el mejor escritorio en base a los siguientes puntos: Diseño general/armonia de formas y colores, creatividad e innovacion.
[/LIST]

[LIST]
[*]El mas votado sera recompensado con un Ububtu badge de aluminio (no es una Ferrari, ni una impresora laser color ni un monitor de 40" pero queda muy bien y se puede lucir con orgullo en la maquina que estes usando con Ubuntu).
[/LIST]

**Plasma:** Air, por defecto. Nunca me había gustado, supongo que por lo blancucho. Le dí la oportunidad. 
[B]
Plasmoides:[/B] [Smooth Tasks]("http://kde-look.org/content/show.php/Smooth+Tasks?content=101586"), [Now Rocking]("http://kde-look.org/content/show.php/Now+Rocking?content=106304"), [Customizable Weather Plasmoid.]("http://kde-look.org/content/show.php/Customizable+Weather+Plasmoid+%28CWP%29?content=98925") Si no les toma el instalador, o simplemente si los quierecompilar, les dejo este sencillo script. NO olviden copiarlo en /usr/bin y darle los permisos necesarios. Luego lo ejecutan con: **plasma-install** dentro del directorio del plasmoide. También sirve para compilar. 

```
#!/bin/bash

if [ ! -d build  ]; then
    mkdir build
fi

cd build
cmake -DCMAKE_INSTALL_PREFIX=`kde4-config --prefix` ..
make clean
make

ret=$?

if [ $ret -eq 0 ]
then
    echo "Compilación completa. Instalando..."
    if [ ! `whoami` = "root"  ]; then
          echo
        echo "La instalación requiere privilegios de Root. "
        echo "Ingrese la contraseña de Root (CTRL+C para cancelar):"
        sudo make install    
    else
        make install
    fi
    echo "¡La instalación se ha completado con éxito!"
else
    echo "La compilación ha fallado, lo siento >_<"
fi
```

**Iconos:** [Kicons]("http://kde-look.org/content/show.php/Kicons?content=109221"), pero la 0.4, la última que combina bien en conjunto. Si la quieren la subo, el autor no la quiere resubir.
