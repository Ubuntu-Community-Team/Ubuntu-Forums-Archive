### Post by Ricardo_Velasco on 2015-11-25
Saludos:

Bueno intenta con este comando.. pero una pregunta? Instalaste solo los controladores no el paquete nvidia-settings??..

Intenta aver si te sirve porque se supone que con este comando puedes salvar tu configuracion de pantalla y asi poder disfrutar de tu Ubuntu:

Te dire los pasos de instalacion:

Ejecuta permisos de superusuario ( no olvides poner contraseña)

> sudo su

Luego vas a descargar por PPA el driver:

> sudo apt-add-repository ppa:xorg-edgers/ppa

(Problema: si aparece un emoticon entre pa org.. es : x )

Luego pones:
> 
sudo apt-get update

Y luego:

> sudo apt-get install nvidia-current nvidia-settings


Y luego de esto ejecutas:

> sudo nvidia-settings


Con este comando se supone que aparece la Interfaz Grafica para establecer la configuracion de la tarjeta

Ojala que te sirva.. Investigue lo que pude.. :)

Muchas gracias.
