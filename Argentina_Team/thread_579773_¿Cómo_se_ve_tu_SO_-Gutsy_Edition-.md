1- posteen la(s) imagen(es) que deseen y una breve descripción de que están usando para que quede como queda (opcionalmente las especificaciones técnicas de su máquina).

Bueno, ¡a disfrutar la belleza de nuestras máquinas!

Según tengo entendido el DS Linux tiene 2 versiones muy diferentes. Uno es si tenés memoria extra y otro si no la tenés.

Lo que pasa es que tenés que cambiar o instalar el tema para el root también (creo que si los instalas en /usr/share/themes no hace falta, aunque no estoy seguro)
Para cambiar el tema de root hace "Alt+F2" y pone:
```
gksu gnome-appearance-properties
```
Pone tu contraseña y cambia o instala el tema que quieras para el root.
