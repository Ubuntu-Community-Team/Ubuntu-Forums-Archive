### Post by faktorqm on 2007-08-08
Para copiar en la terminal hacia cualquier lado, seleccionas lo que queres copiar, y vas a edicion -> copiar. lo mismo si queres pegar. si no queres usar el mouse, shift + ctrl + v para pegar, shift + ctrl + c para copiar.
