### Post by TheFridge on 2007-08-15
[Dot News]("http://dot.kde.org"), a KDE news site has recently written about the [new release of LinuxMCE]("http://dot.kde.org/1187201437"). [LinuxMCE]("http://linuxmce.org") is a free and open source add-on for the [Kubuntu operating system]("http://www.kubuntu.org"). The new release, 0704, provides an add-on CD for Kubuntu 7.04, and turns your computer into the world&#8217;s most complete media center.
 [INDENT]&#8220;The traditional PC user interface doesn&#8217;t work well on a TV.  So a different interface is needed, which is called the &#8216;10 foot&#8217; interface&#8221;
[/INDENT] LinuxMCE&#8217;s lead developer, Paul Webber, in an effort to address the &#8216;10 foot&#8217; interface challenge, has joined up with the KDE [Plasma]("http://plasma.kde.org") developers and accepted the challenge. The 0704 release is the first step in making a prototype showing how LinuxMCE and KDE can work together.
 To understand the power of LinuxMCE, which literally can run your home, read the [features list]("http://wiki.linuxmce.org/index.php/Features") and watch [the video walkthrough]("http://wiki.linuxmce.org/index.php/Video").  To try it out [download the CDs or DVD]("http://wiki.linuxmce.org/index.php/Download_Instructions").
