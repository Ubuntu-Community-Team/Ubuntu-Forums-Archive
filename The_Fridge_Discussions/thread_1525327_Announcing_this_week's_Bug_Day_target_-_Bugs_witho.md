This week&#8217;s Bug Day target is *drum roll please* **Bugs without a package!**

The task is to assign to the right package and triage those as well:

[LIST]
[*]100 New bugs without a package need a hug
[*]100 Confirmed bugs without a package need a review
[/LIST] Bookmark it, add it to your calendars, turn over those egg-timers!

[LIST]
[*]Thursday, 8 July 2010
[*][https://wiki.ubuntu.com/UbuntuBugDay/20100708](https://wiki.ubuntu.com/UbuntuBugDay/20100708)
[/LIST]Are you looking for a way to start giving some love back to your adorable Ubuntu Project?

Did you ever wonder what Triage is? Want to learn about that?

This is a perfect time!, **Everybody can help in a Bug Day!** Open your IRC Client and go to #ubuntu-bugs (freenode) the BugSquad will be happy to help you to start contributing!

Wanna be famous? Is easy! remember to use 5-A-day so if you do a good work your name could be listed at the top 5-A-Day Contributors in the Ubuntu Hall of Fame page!

We are always looking for new tasks or ideas for the Bug Days, if you have one add it to the Planning page [https://wiki.ubuntu.com/UbuntuBugDay/Planning](https://wiki.ubuntu.com/UbuntuBugDay/Planning)

*If you&#8217;re new to all this, head to [https://wiki.ubuntu.com/Bugs](https://wiki.ubuntu.com/Bugs)*
