### Post by TheFridge on 2006-03-08

	 <p>Michael Dell, chairman and founder of Dell, mentioned Ubuntu in a number of contexts in a <a href="http://www.desktoplinux.com/news/NS3822185143.html">recent interview</a> on <a href="http://www.desktoplinux.com/">DesktopLinux.com</a>. Of great interest was his comment that, &#8220;Ubuntu is now the most popular desktop distribution on Dell PCs&#8221;. Rock and roll!</p>
<blockquote><p>&#8220;If we say we like Ubuntu, then people will say we picked the wrong one. If we say we like and support Ubuntu, Novell, Red Hat, and Xandros, then someone would ask us, &#8216;Why don&#8217;t you support Mandriva? The challenge we have with picking one is that we think we&#8217;d disenchant the other distributions&#8217; supporters.&#8221;</p></blockquote>
<p>Dell offers an range of workstations and laptops called the nSeries which do not include Windows, opting instead for a very basic FreeDOS. These machines are intended for buyers who are keen to run Linux without paying for an operating system they won&#8217;t use.</p>
