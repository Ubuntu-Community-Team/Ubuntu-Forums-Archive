### Post by TheFridge on 2008-07-12
Pre-releases of Intrepid are **not** encouraged for anyone needing a stable system or anyone who is not comfortable running into occasional, even frequent breakage.  They are, however, recommended for Ubuntu developers and those who want to help in testing, reporting, and fixing bugs. 

 Alpha 2 is the second in a series of milestone CD images that will be released throughout the Intrepid development cycle. The Alpha images are known to be reasonably free of showstopper CD build or installer bugs, while representing a very recent snapshot of Intrepid. You can download it here: 

 
[LIST] 
[*][http://cdimage.ubuntu.com/releases/intrepid/alpha-2/ (Ubuntu)]("http://cdimage.ubuntu.com/releases/intrepid/alpha-2/") 
[*][http://cdimage.ubuntu.com/kubuntu/releases/intrepid/alpha-2/ (Kubuntu)]("http://cdimage.ubuntu.com/kubuntu/releases/intrepid/alpha-2/") 
[*][http://cdimage.ubuntu.com/edubuntu/releases/intrepid/alpha-2/ (Ubuntu Education Edition)]("http://cdimage.ubuntu.com/edubuntu/releases/intrepid/alpha-2/") 
[*][http://cdimage.ubuntu.com/xubuntu/releases/intrepid/alpha-2/ (Xubuntu)]("http://cdimage.ubuntu.com/xubuntu/releases/intrepid/alpha-2/") 
[/LIST]
 See [http://wiki.ubuntu.com/Mirrors](http://wiki.ubuntu.com/Mirrors) for a list of mirrors.

 Alpha 2 includes a number of software updates that are ready for large-scale testing.  Please refer to [http://www.ubuntu.com/testing/intrepid/alpha2](http://www.ubuntu.com/testing/intrepid/alpha2) for information on changes in Ubuntu.

 This is quite an early set of images, so you should expect some bugs.  For a list of known bugs (that you don&#8217;t need to report if you encounter), please see: [http://www.ubuntu.com/testing/intrepid/alpha2](http://www.ubuntu.com/testing/intrepid/alpha2)

 If you&#8217;re interested in following the changes as we further develop Hardy, have a look at the intrepid-changes mailing list:

 
[LIST] 
[*][http://lists.ubuntu.com/mailman/listinfo/intrepid-changes](http://lists.ubuntu.com/mailman/listinfo/intrepid-changes) 
[/LIST]
 We also suggest that you subscribe to the ubuntu-devel-announce list if you&#8217;re interested in following Ubuntu development. This is a low-traffic list (a few posts a week) carrying announcements of approved specifications, policy changes, alpha releases, and other interesting events.

 
[LIST] 
[*][http://lists.ubuntu.com/mailman/listinfo/ubuntu-devel-announce](http://lists.ubuntu.com/mailman/listinfo/ubuntu-devel-announce) 
[/LIST]
 Bug reports should go to the Ubuntu bug tracker:

 
[LIST] 
[*][https://bugs.launchpad.net/ubuntu](https://bugs.launchpad.net/ubuntu) 
[/LIST]
