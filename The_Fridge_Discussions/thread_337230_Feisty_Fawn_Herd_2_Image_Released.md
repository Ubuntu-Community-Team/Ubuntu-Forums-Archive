### Post by TheFridge on 2007-01-12
The latest development milestone release (Herd 2) for the Ubuntu project&#8217;s next release, Feisty Fawn, has been released. This release will become Ubuntu 7.04 in April 2007.
 There have been many improvements along the lines of bug fixes, feature implementations, and the latest versions of the most popular Open Source software currently available. We don&#8217;t recommend it for use as a stable environment, but it is great for [testing Ubuntu]("https://wiki.ubuntu.com/Testing") and contributing to the next release by [filing bugs]("https://help.ubuntu.com/community/ReportingBugs")!
