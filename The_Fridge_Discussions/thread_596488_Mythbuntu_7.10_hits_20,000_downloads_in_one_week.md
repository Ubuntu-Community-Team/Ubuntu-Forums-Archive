The Ubuntu based MythTV distribution, Mythubuntu 7.10, released on Monday 25th October 2007, and in less then a week tracked hit it&#8217;s 20,000th download.
 [IMG]http://fridge.ubuntu.com/files/images/7.10_mythtv-setup.preview.png[/IMG]
 **What is Mythbuntu?**
 Mythbuntu is an Ubuntu derivative focused upon setting up a standalone MythTV system similar to Knoppmyth or Mythdora. You can install it either as a stand alone Frontend, Backend, or combination machines. It is also easy to convert any Ed/K/Ubuntu machine into a Mythbuntu machine via the mythbuntu meta packages.
 For further information go to the [Mythbuntu website]("http://www.mythbuntu.org/") and if you want to try it out, check out the [downloads page]("http://www.mythbuntu.org/downloads").
