### Post by TheFridge on 2008-07-01
How would you like to help make the Ubuntu website a thing of great beauty? If so, here&#8217;s your chance. There is a newly created Ubuntu Web Presence team recruiting members. If you&#8217;re excited about web technologies now you can help out. Some of the tasks this team will tackle include: 

 
[LIST] 
[*]New countdown banners for the releases 
[*]Contribute to the start page that users see when they first load Ubuntu ([http://start.ubuntu.com/8.04](http://start.ubuntu.com/8.04)) 
[*]Allow localization of key portions of the Ubuntu website 
[*]Use your javascript, python or PHP coding skills to create exciting new features 
[*]Design a visually engaging website experience 
[*]Enable more people in the community to help out 
[*]And more&#8230; the possibilities are limitless! 
[/LIST]
 Presently, the Ubuntu website is managed by a very small group of people, mostly Canonical employees. We&#8217;d like to open this up so that a team of talented and trustworthy community members can help define the website experience. This is your opportunity!

 Sign up now by joining the Web Presence Team&#8217;s mailing list and introducing yourself: [https://lists.ubuntu.com/mailman/listinfo/ubuntu-website](https://lists.ubuntu.com/mailman/listinfo/ubuntu-website)
