The [Official Ubuntu Book]("http://www.phptr.com/bookstore/product.asp?isbn=0132435942&rl=1") is now available from booksellers, featuring everything you need to know about:
 [LIST] 
[*]Ubuntu and Kubuntu as desktop operating systems 
[*]Ubuntu server installation and basic administration 
[*]the Ubuntu community, how it works and how you can get involved. [/LIST] It also features a DVD with the latest Ubuntu version.
 It&#8217;s written by our very own Benjamin Mako Hill, Jono Bacon, Corey Burger, Jonathan Jesse and Ivan Krstic, with contributions from many of Doc Team and other parts of the Ubuntu community.
 You can order your copy now from your local book shop or:
