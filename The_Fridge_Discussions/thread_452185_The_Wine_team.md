### Post by TheFridge on 2007-05-23
[Stephan Hermann]("https://launchpad.net/~shermann") has [blogged about the new wine team]("http://linux.blogweb.de/archives/328-Announcing-the-ubuntu-wine-team.html") and has created an [Ubuntu-wine team]("https://launchpad.net/~ubuntu-wine") on Launchpad. The Ubuntu Wine Team will take care about the WINE application in Ubuntu and plans to support all wine related projects as well. 
 He has also created a [wine-doors]("https://launchpad.net/wine-doors/") launchpad project for this tool and pushed two bzr branches to the team directory and has plans to add a few upstream branches later this week. If you have a [Launchpad account]("https://launchpad.net/+join/+login"), you can join them using the Launchpad links provided above.
