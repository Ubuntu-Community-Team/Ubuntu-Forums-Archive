### Post by TheFridge on 2009-03-27
With the coming of Ubuntu 9.04, it was time for a change to this package. Our latest release now has a countdown timer.

 You can see available options for countdown images at [http://www.ubuntu.com/getubuntu/countdown](http://www.ubuntu.com/getubuntu/countdown). Unfortunately, this third option being an iframe means that we&#8217;re unable to include it as an option.

 The options available:
 * Pick from 3 for the 4 images
 * Set position
 * Set size
 * Set opacity
 * Set visibility
 * Even IE tested

 You can grab the latest package at [https://launchpad.net/ubuntu-drupal-releases/6.x](https://launchpad.net/ubuntu-drupal-releases/6.x).

 Once installed, all the available options are listed in /admin/build/themes/settings/ubuntu-drupal. You can of course see a working example at [http://sd.ubuntu-us.org/](http://sd.ubuntu-us.org/).
