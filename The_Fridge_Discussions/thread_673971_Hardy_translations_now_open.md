### Post by TheFridge on 2008-01-21
Hardy is now open for translation! If you want to help make sure it&#8217;s accurately translated into your language, you can start work straight away at:
[https://translations.launchpad.net/ubuntu/hardy/+translations](https://translations.launchpad.net/ubuntu/hardy/+translations)
If you&#8217;re new to translating Ubuntu and want to find out more, take a look at the [Translating Ubuntu]("https://wiki.ubuntu.com/TranslatingUbuntu") guide.
**Frequent lang pack updates**

If you&#8217;re running Hardy Heron during its development cycle, you&#8217;ll receive twice weekly language pack updates reflecting the latest work in Launchpad Translations. These updates are planned to start later this week.
This is a great way to see your translations in place before the official release.
