### Post by TheFridge on 2007-10-20
With the release of 7.10 yesterday, it is not terribly shocked we have already started to see reviews.
With the massive amount of interest in Ubuntu, this means some of the mirrors have been a little slow. CLICK picked up [on this story]("http://www.insidesocal.com/click/2007/10/trying_to_get_gutsy.html"), remarking that it was similar to the 7.04 release period.
Most of the reviews have been quite favourable, with Desktop Linux saying *All in all, the new Ubuntu appears at first glance to be a solid improvement on the last version and one that both young and old Ubuntu users will find a worthwhile upgrade* in their review: [DesktopLinux.com:   It&#8217;s Here! Ubuntu 7.10 Arrives]("http://www.desktoplinux.com/news/NS9663741090.html").
Similarly, polishlinux.org says *The new Ubuntu is more polished, more professional and in general, better than the previous one, which was already a great OS* while pointing out that *Minor glitches? Present, as always*. You can read more at [Ubuntu 7.10 Gutsy Gibbon: Critical review]("http://polishlinux.org/linux/ubuntu/ubuntu-710-gutsy-gibbon-critical-review/").
Lunapark joined in praising 7.10, saying *I’ll cut to the chase and say from the start that Ubuntu 7.10 (Gutsy) Desktop Edition is simply the best desktop operating system that I have ever used. From installation, to setup, to regular everyday use Gutsy is just a thing of beauty.* Check it out at [Ubuntu 7.10 (Gutsy) Desktop Edition - Review]("http://lunapark6.com/ubuntu-710-gutsy-desktop-edition-review.html").
Lastly, PCWorld wonders [Ubuntu Gets Gutsy, But Is Linux Ready?]("http://www.pcworld.com/article/id,138625-c,linux/article.html"), saying *The Litmus test for Ubuntu, and Linux on the desktop, will come when people try Gutsy as an alternative to Windows and Mac OS X. With more device driver integration and automatic configuration built in, Linux may eventually shred its command-line hacker moniker.*
