### Post by TheFridge on 2006-10-31
The [Xubuntu]("http://www.xubuntu.org/") community is happy to [announce]("https://lists.ubuntu.com/archives/ubuntu-announce/2006-October/000096.html") the release of Xubuntu 6.10, codenamed &#8220;Edgy Eft&#8221;.
 Xubuntu is lighter on system requirements than its better known cousins Ubuntu and Kubuntu  since it uses the Xfce Desktop environment, which makes it ideal for old or low-end machines, thin-client networks, or for those who would like to get more performance out of their hardware.
 Included in this new release are:
 [LIST] <li > Newer Xfce Desktop Environment  (4.4 RC1) which brings trashcan support in Thunar and the panel, accessibility settings for the keyboard, and other improvements and fixes.
<li > New artwork for the boot splash, login screen and wallpaper.
<li > The more mature gxine media player replaces xfmedia.
<li > New printer GUI: system-config-printer from Fedora Core 6.
<li > Easier installation of packages with gnome-app-install.
<li > A calculator application and a dictionary panel plugin.
<li > Better support for users with motor disabilities via configurable sticky keys, mouse keys and an on-screen-keyboard application.
<li > Newer versions of firefox, abiword and gnumeric.
<li > New LTSP version on the alternate CD for setting up thin client networks.
<li > Other changes common to all Ubuntu variants such as the new 2.6.17 kernel, Xorg 7.1 and the upstart init system. [/LIST] Xubuntu 6.10 is available for [download]("http://www.xubuntu.org/get") today!
