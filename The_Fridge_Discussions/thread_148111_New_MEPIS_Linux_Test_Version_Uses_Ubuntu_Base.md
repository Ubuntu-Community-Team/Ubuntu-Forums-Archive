### Post by TheFridge on 2006-03-21

	 <p>Great news for lovers of Ubuntu and MEPIS:</p>
<blockquote>
<p>MEPIS founder Warren Woodford has announced a test release of SimplyMEPIS 6.0, incorporating software from the Ubuntu Dapper package pools. This is the first version of SimplyMEPIS with an Ubuntu base.</p>
<p>Ubuntu has a 6 month stable release cycle that will enable MEPIS to offer its customers a dependable release schedule. Woodford states &#8220;The switch to the Ubuntu pools was made to provide our users with a more stable underlying system. Of course it&#8217;s important for our users that MEPIS remains true to its unique vision. I believe this release demonstrates that we can combine the magic of the MEPIS user experience with the goodness of the Ubuntu foundation.&#8221;</p></blockquote>
