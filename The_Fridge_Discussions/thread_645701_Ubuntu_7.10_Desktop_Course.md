### Post by TheFridge on 2007-12-20
With the help of community members who slogged over writing, reviewing, editing, proof reading and fixing the layout, the [Ubuntu 7.10 Desktop Course]("https://wiki.ubuntu.com/Training") is finally available. The 400 page book provides a thorough overview of the Ubuntu Desktop and is a great resource for Linux and Ubuntu newcomers.
 This is just the beginning of the project, which will live and evolve with each new Ubuntu release. The community has the power to develop, enhance and branch off as they see fit. The latest version can always be retrieved from the [Bazaar repositories]("https://launchpad.net/ubuntu-desktop-course").
 Mark Shuttleworth also has [an interesting post about this release]("http://www.markshuttleworth.com/archives/134"), and where it&#8217;s headed.
