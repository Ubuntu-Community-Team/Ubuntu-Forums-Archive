### Post by TheFridge on 2009-12-10
Launchpad&#8217;s web interface will be read-only for one hour from 22.00 UTC on Wednesday the 16th December 2009 for the release of Launchpad 3.1.12. During that time, other services, including PPAs, code hosting and the email interface, will be offline.

Starts: 22.00 UTC 16th December
Expected back: 23.00 UTC 16th December

This is the final Launchpad code release of 2009! We&#8217;ll post the 2010 release calendar to the Launchpad blog in the next few days: [Launchpad Blog]("http://blog.launchpad.net/")

Follow Launchpad&#8217;s official status feed for full status information:[Launchpad Status]("http://identi.ca/launchpadstatus")
