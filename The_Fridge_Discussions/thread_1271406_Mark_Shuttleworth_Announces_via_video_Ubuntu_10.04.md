Mark Shuttleworth spoke via video to UbuCon at the [Atlanta Linux Fest 2009]("http://atlantalinuxfest.org") and announced the name of the next Ubuntu release. Ubuntu 10.04 will be code named &#8220;Lucid Lynx&#8221; and will be an LTS release with support for the desktop for 3 years and for the server for 5 years.
