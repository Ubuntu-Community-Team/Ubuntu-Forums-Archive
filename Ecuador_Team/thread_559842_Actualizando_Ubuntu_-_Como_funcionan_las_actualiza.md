### Post by hubuntu on 2007-09-25
Esto se puede hacer de dos formas:

1. Se usa la red para actualizar la máquina (descargando la nueva
versión automaticamente) con el manejador de actualizaciones; o,

2. Se usan los CD's para actualizar el sistema, siguiendo las
siguientes instrucciones:
[https://help.ubuntu.com/community/DapperUpgrades](https://help.ubuntu.com/community/DapperUpgrades)

Desde la versión 6.06 (Dapper) de Ubuntu el CD de instalación es
también un LIVE CD, osea un CD que muestra como su computador será
después de la instalación y el mismo CD  se encarga de instalar el
sistema.

Ubuntu soporta actualización de una versión normal a otra (5.10 ->
6.06 -> 6.10 -> 7.04 -> 7.10) y de una versión LTS a otra LTS (6.06 ->
8.04).

La version 5.10 (Breezy) no es un LTS (Long Term Support - De Soporte
Extenso) y no puede desgraciadamente actualizarse directo a la versión
7.04 (Feisty).

En este caso es bueno una instalación completamente nueva.

En pocos trazos ingresar el CD, escoger Español como lenguaje de
instalación, esperar a que Ubuntu se cargué y una vez en el sistema
pulsar instalar y seguir las instrucciones

Un buen recurso para el iniciante de ubuntu es la documentación de la
comunidad hispanohablante:

[http://www.ubuntu-es.org/documentacion](http://www.ubuntu-es.org/documentacion)

Este es un esfuerzo común de todos los LoCo Teams de ubuntu-es.
