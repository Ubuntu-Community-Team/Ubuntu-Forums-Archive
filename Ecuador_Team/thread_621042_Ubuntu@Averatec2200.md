### Post by hubuntu on 2007-11-30
[http://www.averatec-forums.com/index.php?topic=158.0](http://www.averatec-forums.com/index.php?topic=158.0)

Yo creo que talvés deberías tratar el driver de windows con ndiswrapper:

Busca "Windows Wireless Drivers" en synaptic o bajo Aplicaciones -> Añadir y quitar...

Luego escógelo, aplica los cambios (te pide TU password normal) y ya está.

Anda a Sistema -> Administración -> Windows Wireless Drivers

Instala ahi el driver (necesitas normalmente solo el inf, pero es bueno tener el CD de instalación para WINDOWS o simplemente bajarte el driver del sisito de los que hacen la compu o la tarjeta) y disfrutalo! :)
