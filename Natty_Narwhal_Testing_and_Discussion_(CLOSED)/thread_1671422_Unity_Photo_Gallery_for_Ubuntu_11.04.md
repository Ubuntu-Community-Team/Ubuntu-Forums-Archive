### Post by seeker5528 on 2011-01-20
Have not done much to change the default other than pin applications and use docky. Will probably continue to use docky for the time being. Got rid of some of the indicator stuff. Sound in particular since I remove all the pulse-audio stuff I can. The volume icon is provide by kmix, for which I created a kmix.desktop file with the contents of ```
[Desktop Entry]
Type=Application
Exec=/usr/bin/kmix --keepvisibility
Hidden=false
NoDisplay=false
X-GNOME-Autostart-enabled=true
Name[en_US]=Kmix
Name=Kmix
Comment[en_US]=
Comment=
```

and placed it in '~home/.config/autostart', I don't think the '--keepvisibility' part is necessary any more, I think it always starts in the state (visible or hiddent) it was in when it was last running.

Looks like some of the launchers must have been pick up from stuff I had added to the gnome panel, the area that appears to be a blank area in the icons is the launcher for xkill, which I don't remember adding, it did previously have an icon though.
