### Post by ankspo71 on 2011-04-08
Hi,
That setting is in a location that doesn't seem very obvious. 

System Settings > Shortcuts and Gestures > Global Keyboard Shortcuts;  
Now choose "Run Command Interface" from the dropdown menu. 
Then change the settings to "Lock Session".
Hope this helps.
