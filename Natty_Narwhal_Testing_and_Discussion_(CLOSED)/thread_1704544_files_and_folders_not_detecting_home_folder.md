### Post by mc4man on 2011-03-11
> ya, but mine shows nothing at all...
assuming your install is updated and fairly new, then at first nothing is in 'files & folders'.
As you use files, open folders, download files they'll be added.
Sometimes they'll show up right away, sometimes only after a log out/in.

But in general I've found it shows entries based on use, not existence.

For ex. this install is only a few hr.s old, at first F&F showed nothing at all.
Now some things are showing up, files after  using, folders seemed to take a log out.

Just solely on the default 'all files' screen, atm now have 3 sections - 'recent' (files), downloads (I did one), and favorite folders which show the default ones.

I added 3 to that section by adding the folders as bookmarks, -  File System, bin and share.
Over time the files section will get broken into time based subsections, I guess based on last use.

As far as the dropdown subsections, pretty much the same, entries will be added, probably based on mimetype, but not until you use (open)  them

Try logging out and obviously try opening some files, folders ect. Add a bookmark or 2 and see if they show up (after logout/in
