### Post by rebootme on 2011-02-04
I really wanted to give Unity a try but with an nvidia card I can't get 3D unity yet (unless someone knows something I don't...).  So I decided to give 2D a try.

```
sudo add-apt-repository ppa:unity-2d-team/unity-2d-daily
sudo apt-get update
sudo apt-get upgrade
sudo apt-get install unity-2d-default-settings
```

Log out, then select Unity 2D as your desktop and log back in.

Note this does tie you to a PPA so use at your own risk.
