Hi everyone,

For Natty we're trying to make it easier for people to contribute to Unity. Right now we've got [instructions](http://unity.ubuntu.com/getinvolved/) on the unity homepage on how you can grab the code and get hacking, but as you can imagine it's like drinking from a firehose.

So we've started tagging bugs in Launchpad with a "bitesize" tag for people that want to get started on little bugs that want to start hacking on Unity. You can find the [list of bugs here](https://bugs.launchpad.net/unity/+bugs?field.tag=bitesize"). Here's the [wiki page](https://wiki.ubuntu.com/Unity/Bitesize) with more information. 

Feel free to ping me in this thread or on irc (nick is jcastro) if you have questions on how to get started!
