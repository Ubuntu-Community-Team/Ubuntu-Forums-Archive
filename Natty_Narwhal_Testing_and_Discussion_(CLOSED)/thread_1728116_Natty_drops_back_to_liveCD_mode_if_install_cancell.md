### Post by youbuntu on 2011-04-13
Hi. I just ran my Natty daily liveCD, and I found something odd. If I boot from the liveCD and choose install from the text-mode menu (immediately after boot, NOT in liveCD mode), it then launches the *graphical* installer, but if you cancel the graphical installer, instead of rebooting or shutting down (as you'd think it would) it just drops back to a liveCD Desktop :?

I am going to download the live *DVD* ISO now, to see if the text mode installer is built into it, as was suggested by a member last night.

Just wondered why it defaults to the liveCD Desktop, instead of shutting off or dropping back to the menu? Could potentially confuse users.

### Post by coffeecat on 2011-04-13
The live CD has a graphical desktop and a graphical installer. You have the chouce between the old-style text menu and the two icon choices for "install Ubuntu" and "Try Ubuntu" because the text menu offers you more choices and flexibility - such as adding boot options or doing a CD integrity check. It does not give you the text installer. You need the alternate CD for that.

It seems quite logical to me that if you cancel the installer it would drop you back to the live desktop. It's only a couple of mouse clicks to shut down from there, and how would you feel if you'd started the installer by mistake when you wanted the live desktop and the thing shut itself down when you cancelled the installer? Me? I'd be annoyed. :)

The DVD comes with a load of language packs - I don't know about the text installer. I think not - the alternate CD is best for that.
