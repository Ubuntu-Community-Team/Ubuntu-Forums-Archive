### Post by M4570D0N on 2011-04-11
I had to delete Virtualbox 4.0.4-70112 on my Mint 10 partition that I downloaded from the  Virtualbox website, and instead, download Virtualbox-ose 4.0.4-dfsg from [https://launchpad.net/~debfx/+archive/virtualbox](https://launchpad.net/~debfx/+archive/virtualbox) so that I could also install  virtualbox-ose-guest-x11 on the Natty VM. Some Xorg update in alpha3  broke guest additions in the other Virtualbox version rendering it  completely unusable. Vbox hasn't been able to keep up with the updates  to Natty. As such, the solution is to install  guest-x11 instead. Grab a daily live iso of Natty beta and run it from there. Do NOT install guest-sessions. Once you have booted the Natty VM, open synaptic and d/l and install the virtualbox-ose-guest-x11 package instead. Everything should work after that.
