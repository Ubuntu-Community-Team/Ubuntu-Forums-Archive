### Post by amsterdamharu on 2010-12-17
When you start Ubuntu desktop (not classic) and press control + alt + F1 you should get a non graphic login.

Then after you login type:
```
lsmod | grep radeon
```

Is there any output?

If there isnt any it looks like 10.04 or Natty is not supporting your ATI card, you might report the bug to Ubuntu if you want so they can fix it:
You have to create an account:
[https://help.ubuntu.com/community/ReportingBugs](https://help.ubuntu.com/community/ReportingBugs)
I think the command would be:
```
apport-cli -f -p xserver-xorg-video-radeon --save bug.apport
```Make sure you don't type that in classic desktop because everything seems to be working there

After that you might get lucky with xorg-edgers to install the latest unstable drivers, if that doesn't work I suggest getting 10.04 or 10.10 and re install
To get the xorg edgers drivers start the ubuntu desktop (not classic) press control + alt + F1 and type this:

```
sudo apt-get install xserver-xorg-video-radeon
sudo add-apt-repository ppa:xorg-edgers/ppa
sudo apt-get update
sudo apt-get upgrade
```Again make sure you don't type that in classic desktop because everything seems to be working there

Then control + alt + delete to reboot.

### Post by cariboo on 2010-12-17
Unity works with Intel 945 series graphics card out of the box, If all you get when logging into the Ubuntu Desktop is a spinning cursor, press Ctl-Alt-F1 to get to a console, login, then reboot:

```
sudo reboot
```

Once at the login screen select the Classic Desktop from the session menu, once at the desktop, install compizconfig-settings-manager. Once ccsm is installed, run it, System-Preferences->Compizconfig Settings Manager, and scroll all the way to the bottom and make sure the Unity plugin is enabled, click the the plugin to enable auto/intellihide, once you done that, log out and log in to the Ubuntu Desktop. Unity is still in the early development stages, so be aware that things will break, or are broken quite frequently.

apport-cli -f -p xserver-xorg-video-radeon --save bug.apport
```Make sure you don't type that in classic desktop because everything seems to be working there

sudo apt-get install xserver-xorg-video-radeon
