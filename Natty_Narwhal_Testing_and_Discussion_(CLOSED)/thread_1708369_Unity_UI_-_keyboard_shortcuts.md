### Post by xykivo on 2011-03-16
I tried unity (Natty alpha 3).
In general it worked pretty smoothly.

The main issue I have with unity is that it's mouse centric UI.
All the nice keyboard shortcuts available in gnome have gone.

I'll give some examples for things that bothered me:

1) When you hit the windows key, u get the search app window. After I type in the application name, I have to use the mouse to select an application - why can't it be done with the arrow keys?

2) When an application is maximized the Alt + <menu letter> shortcut no longer works. Again I have to use the mouse to move the cursor to the top of the screen to even see the menus.

3) No way to use keyboard to select an app to run from the app list in right bar (not sure what's the proper name for it).

There are more things like the examples above.
But the general thing that bothers me is the need to move my hand from the keyboard to the mouse all the time.

In this respect the traditional gnome desktop is better for me.

### Post by castrojo on 2011-03-16
If you're on A3 you need to update as keyboard stuff has been fixed since then.

Dash does keyboard navigation now. The menu shortcut is now F10 (in addition the normal keyboard shortcuts work now), and alt-f1 will take you into keyboard mode in the launcher (then use the arrow keys and enter to launch an app)
