### Post by clubsoda on 2011-03-09
Alpha software is safe* again! :-) I swore off alphas after Karmic but Natty is in another league.

+ The **installer** is quick & slick. Installation proceeds concurrently with the handful of questions which require a user response, so no more configuration delays once you've chosen your partitions.

- **Grub** menu text is fuzzy due to resizing and vbemode doesn't list my native resolution, 1920x1200. [This worked before].
= Grub background colour is maroon, maybe even purple. Expect some howls of protest if this colour makes it to the final release. :-)

**XFCE 4.8.0** is really coming along nicely!
+ Antialiasing and hinting are now adjustable without logging into Gnome or KDE.
+ Improved rendering, particularly with asian fonts
+ Much easier to add launchers and find launcher icons.
+ The default panel app's seem more useful too, such as the terminal.
- Mousepad still doesn't play nicely with ibus - using Leafpad instead.
- Thunar joins LibreOffice as an app which starts slowly due to unnecessary DNS queries. :-(


**Firefox 4.0 beta 12** still needs some work. The toolbars are pretty well broken.
- Bookmarks Toolbar doesn't appear - has to be toggled off and back on
- Tabs sometimes can't be closed
- Navigation buttons aren't working
- No padlock or greenbar confirmation of https (but security appears to be working)
- Sometimes the first tab can't be selected
- Address in address bar doesn't always match the visible tab
+ However, the rendering is stunningly crisp
Tried reverting to FF3 but it fails on startup.

Firefox aside, this is looking really good so far.


**Safe: Less than one segfault per hour*
