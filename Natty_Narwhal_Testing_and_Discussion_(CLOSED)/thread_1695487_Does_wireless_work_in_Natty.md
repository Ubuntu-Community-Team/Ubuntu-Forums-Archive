### Post by walt.smith1960 on 2011-02-26
I've tried 11.04 with 2 Wireless adapters. Both are new 150Mb/sec. One is a RealTek RTL8192SU (TrendNet TEW649UB), the other is an Atheros Ath9K (Netgear WNA1100). Neither device works in a LiveCD session because they required a tweak to work in 10.04 or 10.10. The TrendNet requires a firmware file to be added, the Netgear I used an installer from sourceforge.  Both devices work on a LiveCD session with 11.04 so I would expect them to work "out of the box" on a hard drive installation.
