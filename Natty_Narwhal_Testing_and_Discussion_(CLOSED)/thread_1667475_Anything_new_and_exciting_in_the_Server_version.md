### Post by cariboo on 2011-01-15
The server installation and setup really hasn't changed since the Server Guide was written for 8.04. The individual sections have been updated as needed. To me that is one of the advantages of the server version, it isn't constantly changing the way the desktop version does. Once you successfully setup a server, the steps you used work well for every other server you setup. 

Personally I prefer to stick with the LTS server versions, for the 5 years of support. I recently built a new server and installed 10.04. All my configuration files  were just transferred from the old server that was running 8.04 to the new one without having to make any changes.
