### Post by cariboo on 2011-01-13
I saw on Jorge's blog that a global menu is now available for Firefox. The ppa is located [here]("https://launchpad.net/~chrisccoulson/+archive/ppa")

Prerequisites:

[LIST]
[*]11.04 only
[*]Needs the latest Firefox Beta 9 in 11.04
[/LIST]

The package is called firefox-globalmenu, I had to do an update before the menu would work. Once you've run the update restart Firefox.

Please report bugs [here]("https://bugs.launchpad.net/globalmenu-extension/+filebug")
