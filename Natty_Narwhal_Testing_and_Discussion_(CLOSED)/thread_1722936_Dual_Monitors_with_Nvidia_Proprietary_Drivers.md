### Post by Scenemaker on 2011-04-06
After messing with my xorg.conf for a while, I decided to simply start over.  I uninstalled the nvida drivers with:

```
sudo apt-get remove nvidia*
```

Could also do this via the GUI by going to system settings>additional drivers>remove nvidia stuffs

Next, I checked my xorg.conf file and saw that it still had Nvidia stuff all over it.  I then rebooted and still found Nvidia all over it.   I rebooted into recovery and restored the original xorg.conf file.  I don't recall the steps to get to that point but if someone cant figure it out, I'll gladly retrace my steps.  Anyway, I rebooted and when I logged in, it said I couldn't load Unity, etc and I got the gnome desktop back.  Reinstalled nvidia drivers, rebooted into unity, setup my dual monitors and everything was fine.
