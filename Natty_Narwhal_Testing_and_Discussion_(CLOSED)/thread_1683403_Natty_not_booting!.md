### Post by cariboo on 2011-02-07
Boot from the Live CD, and press any key when you see the little man and keyboard, select your language, then press F6 and select nomodeset. Press esc, then enter, the Live CD should then boot. 

You have to be running vbox 4.0 or better in order to run Unity.
