### Post by Rubi1200 on 2011-02-12
Upgrading to alpha releases is not recommended whether using Wubi or not.

If you want to upgrade to 10.10, refer to the [Wubi Megathread]("http://ubuntuforums.org/showthread.php?t=1639198") as there are GRUB updates which cause problems with Wubi and you need to lock the grub packages in Synaptic.

### Post by bcbc on 2011-02-12
Alpha is flaky (what I mean is that it might one minute, but not the next - because the repositories change a lot.)

Natty fresh Wubi install doesn't work [https://bugs.launchpad.net/wubi/+bug/693671](https://bugs.launchpad.net/wubi/+bug/693671)
Natty upgrade from 10.10 worked great last time I tried it.

If you are planning on testing you need to go to the [Natty subforum]("http://ubuntuforums.org/forumdisplay.php?f=394") - there are stickies there that provide important information.

If you test Natty don't expect that your Ubuntu install will survive. It might, but odds are it won't.
