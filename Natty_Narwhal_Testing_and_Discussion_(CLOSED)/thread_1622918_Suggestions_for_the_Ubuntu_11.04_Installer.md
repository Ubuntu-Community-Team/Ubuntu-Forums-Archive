### Post by Toxicbits on 2010-11-16
I experienced that many users, who have just come to Linux have the same questions like "what is a kernel?", "Whats a distribution", "What do i need a desktop environment for?" and so on.

I think that it would be useful not only to present some features while installing, but also give the user a comprehension of how Linux works and how he can participate.

Maybe it would be useful to add some slides like these:
[SIZE=4]
What is Ubuntu?[/SIZE]
-Its an open source OS, everyone can get a "view behind the scenes"
-Its based on the Linux kernel, the "core"
-Desktop environments make you able to have a graphical interface at all
-A distribution like Ubuntu consists of the kernel, a desktop environment, a selection of applications and some pre-sets
[SIZE=4]
[/SIZE][SIZE=4]How to install Software?[/SIZE]
-You can find over 30.000 applications in the software-centre
-Many of them have star-rating and reviews (By december 2010)
-You can add other software sources, so-called repositories
-Alternatively you can install single packages by hand

[SIZE=4]Who is behind Ubuntu?
[SIZE=1]-Ubuntu is the work of a huge community spread all over the world
-Get support online, for example in forums like ubuntuforums.org
-Everyone can contribute to the project
[/SIZE][/SIZE]-Get involved and help improving Ubuntu by reporting bug, requesting features, sharing ideas or just spreading Ubuntu

it has certainly to be phrased a bit differently, but I think that would make the getting in easier and provides a better user experience

Do you think the same way?

toxicbits

The philosophy has always been to provide interested people with the tools to learn more, without hitting everyone else over the head with them. That is why there is a note about ubuntu.com/community in the end; it is a much better starting point for that stuff than a time-limited presentation with about 100 words per concept.
