If I use 'Alt'+'Tab', it's because I want to switch between open windows. Switching to a running application without being able to select which of that application's windows you actually want to switch to, is not equivalent functionality, so not good enough. 

Also I expect 'Alt'+'Tab' to only show me the windows on the desktop I am on, the task panel shows all running applications.

If I could click the workspace switcher, select a desktop, then use 'Alt'+'Tab' to cycle the windows on that desktop, select another desktop, use 'Alt'+'Tab' to cycle through the windows on that desktop, that's something the could be useful to me.
