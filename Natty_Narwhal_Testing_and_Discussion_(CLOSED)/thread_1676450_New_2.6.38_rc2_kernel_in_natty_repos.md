### Post by Harry33 on 2011-01-27
Today the new 2.6.38 rc2 kernel found its way into Natty official repos.
The correct version is 2.6.38-1.27.

Packages are still in new "state".
So if you need them right now, download the packages and install them together in terminal with dpkg.
You need these (64-bit):
linux-libc-dev_2.6.38-1.27_amd64.deb
linux-headers-2.6.38-1_2.6.38-1.27_all.deb
linux-headers-2.6.38-1-generic_2.6.38-1.27_amd64.deb
linux-image-2.6.38-1-generic_2.6.38-1.27_amd64.deb

See here:
[https://launchpad.net/ubuntu/natty/+source/linux/2.6.38-1.27](https://launchpad.net/ubuntu/natty/+source/linux/2.6.38-1.27)
