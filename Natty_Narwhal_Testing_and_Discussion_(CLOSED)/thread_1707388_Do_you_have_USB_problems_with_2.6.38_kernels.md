As r-senior said, the majority of the output has nothing to do with the usb problem, it because your monitor has a bad EDID. 

From the output your usb device is detected and given a device label, it's just that it doesn't auto-mount, which has nothing to do with the kernel

This bit:

```
[ 1398.257417] sdb: sdb1
```

tells us that the device does work, you should be able to mount it manually using the something like the following command:

```
sudo mount /dev/sdb1 /media
```
