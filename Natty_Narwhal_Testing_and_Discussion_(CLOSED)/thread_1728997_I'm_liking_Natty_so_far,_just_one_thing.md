### Post by kansasnoob on 2011-04-15
You might want to read the unity specific part of the release notes:

[http://www.ubuntu.com/testing/natty/beta#Unity](http://www.ubuntu.com/testing/natty/beta#Unity)

Specifically:

>     Ubuntu: it runs Unity. It requires 3D driver support.
    Ubuntu Classic: it runs GNOME with gnome-panel. It supports all video hardware and video drivers.
    Ubuntu Classic (No Effects): it runs GNOME with gnome-panel. It is in 2D mode only.

Classic GNOME panel applets are not supported in Unity, only indicators such as nm-applet.

Should you wish to use the classic desktop (or classic w/o effects) I made a few notes in post #1 and #2 here:

[http://ubuntuforums.org/showthread.php?t=1718631](http://ubuntuforums.org/showthread.php?t=1718631)
