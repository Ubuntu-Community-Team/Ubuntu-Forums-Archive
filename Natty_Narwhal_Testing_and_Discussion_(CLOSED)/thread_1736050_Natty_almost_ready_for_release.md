### Post by marl30 on 2011-04-24
> **ivanovnegro said:**
> Ok, its installed but how I can set it up correctly?

In System Settings, go to System Administration> Startup and Shutdown.

Click on add new program. From the opened menu, choose: Open file dialog.

Browse to Root>usr>bin

All the launchers for applications are in bin. Type this in the name area: gnome-settings-daemon. Click ok when it's highlighted. 

You can start gnome-settings-daemon either by typing it in the menu and hit run when it shows up, or wait for it to start automatically at the next boot.
