### Post by r-senior on 2011-04-17
> **slooksterpsv said:**
> I couldn't see the full window so I maximized it, well there wasn't a close button or restore button ...
I believe it's this bug:

[https://bugs.launchpad.net/ubuntu/+source/compiz/+bug/739316](https://bugs.launchpad.net/ubuntu/+source/compiz/+bug/739316)

> Secondly, the windows that are going to have the disappearing scrollbars with the orange, will there be a setting to disable that (easily)?
Depends how you define easily. ;) There's no configuration setting I have seen, but there is a way:
[http://www.webupd8.org/2011/04/how-to-disable-overlay-scrollbars-in.html](http://www.webupd8.org/2011/04/how-to-disable-overlay-scrollbars-in.html)

Hopefully more configuration options will appear for Unity as we move forward.

> Third, will the Unity bar be able to have a settings menu to resize the icons
Compiz Config Settings Manager -> Unity Plugin -> Experimental

Minimum size is currently 32x32.

That's Unity (3D), with Unity 2D changing the size of the launcher icons involves code changes. There is no simple configuration option.
