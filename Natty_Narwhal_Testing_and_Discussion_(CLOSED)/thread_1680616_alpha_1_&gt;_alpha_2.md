### Post by cariboo on 2011-02-02
> **arpanaut said:**
> To go from Alpha1 to Alpha 2, it is not necessary to do the [B]upgrade-manager -d  command ,
[/B]the normal update, upgrade procedure results in the correct upgrade path at this time.  See the linked Sticky.

The update manager command won't work because that is only used to update to a new version, alpha2 is not a new version. 

If you are keeping up to date there is no need to do a fresh install. There really isn't much difference between alpha1 and alpha2 at this point, as you've probably noticed, updates have tapered off, and they will until alpha2 is released. Once updates start coming again, then you'll see a difference between the two.

The only reason I would recommend a fresh install is if it would take longer to repair a problem, then it would to re-install and and finish setting the system up the way you like it
