Have you installed all the updates? I had problems with wireless with the initial install of 11.04 Alpha 3 but updates (specifically the kernel updates) seem to have helped. Do remember that this is still an Alpha release.

What kernel version are you running?

```

$ uname -r

```

Make sure you have all the software sources that you need enabled and then:

```

$ sudo apt-get update
$ sudo apt-get upgrade

```

EDIT: This will take a while if you got the Alpha 3. There have been lots of updates.

### Post by donniezazen on 2011-03-24
Broadcom is broken in Natty. Try removing all STA and b43 drivers and then install bcmwl-kernel-source from [maverick repo]("http://packages.ubuntu.com/maverick/bcmwl-kernel-source"). That's how i got my wifi working.

[http://ubuntuforums.org/showthread.php?p=10585304#post10585304](http://ubuntuforums.org/showthread.php?p=10585304#post10585304)

[https://bugs.launchpad.net/ubuntu/+source/bcmwl/+bug/712053](https://bugs.launchpad.net/ubuntu/+source/bcmwl/+bug/712053)

gksudo gedit /etc/modprobe.d/*

# blacklist b43
``` and then save the file.
