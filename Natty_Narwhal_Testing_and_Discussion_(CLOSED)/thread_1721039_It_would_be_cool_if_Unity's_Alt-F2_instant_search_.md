### Post by Zorgoth on 2011-04-04
Gnome-do is what I have used for Alt-F2 as long as I've known it existed, because apart from the eyecandy, it has a totally addictive instant search method.  Rather than simply matching the first few characters exactly, gnome-do allows you to skip characters inside application or place names -

e.g.:

1) ww would likely open World of Warcraft
2) fox would likely open firefox
3) out will probably go to log out (observe that log will probably not since lots of things start with log!)

- if you don't understand, install and run gnome-do from the repository and it will be immediately obvious (the letters you type are underlined)

This search method allows you to uniquely identify almost anything in 1-3 characters, and is very quick to get used to and easy to use.  It is in my opinion superior to the obvious method that Unity uses because, for example, if you want to run "system monitor" - there are other things that start with both system and monitor, so you might need to type the whole word system to specify it with Unity's Alt-F2, whereas with gnome-do you can do it with many different obvious combinations of three letters.

Also it would be nice if things like log out, shutdown, and other useful non-applications could appear from typing in Alt-F2 (again as in gnome-do).

I'm not proposing replacing Alt-F2 with a gnome-do clone - mostly I just think gnome-do style search would be really cool.
