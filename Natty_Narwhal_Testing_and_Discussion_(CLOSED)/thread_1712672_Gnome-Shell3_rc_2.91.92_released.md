### Post by Harry33 on 2011-03-23
Gnome-shell is one choice in Natty only if you install it from Gnome3 PPA.
The rc release is not yet there, however.

Also, installing the package gnome3-session, you will have the choice to start it from the login screen.
The packages gnome3-session_2.32.1-0ubuntu17 and GTK3 (version 3.0.3-0ubuntu1)
are already in Natty repos.
And so are mutter (2.91.90-1ubuntu2) and clutter (1.6.8-0ubuntu2) too.

Of course you may use jhbuild too.
