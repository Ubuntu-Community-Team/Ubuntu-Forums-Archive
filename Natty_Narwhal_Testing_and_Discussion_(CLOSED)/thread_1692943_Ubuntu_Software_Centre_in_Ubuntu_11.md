### Post by Shepherd X on 2011-02-22
I just have a question regarding the Software Centre that is packaged with Ubuntu 11. I haven't had the chance to download and try out Ubuntu 11, but I read that in the new release users will be able to add reviews and report inappropriate reviews, etc. I saw some of these features in Linux Mint, an Ubuntu based distro. The problem with how it was implemented in Mint is that add-ons and plugins for programs were not listed on the same page of the program they modified. In the Ubuntu Software Centre for Ubuntu 10.10, if I clicked on a program I was interested in, I could see all the related add-ons and plugins on the same page of the software it would modify/affect. This made choosing to add an add-on an easy procedure.

The problem in Linux Mint is they wanted to give users the ability to review individual add-ons/plugins, so they couldn't fit and organize all the reviews for a program and its add-ons/plugins on the same page. Now if you want to install a program you have to do a search for the program. Then to install any add-ons, you'd have to search for individual add-ons/plugins, assuming you knew that they exist and the name of their package file. It's frustrating.

My question is, does anyone know if this is also how it was implemented in Ubuntu 11?
