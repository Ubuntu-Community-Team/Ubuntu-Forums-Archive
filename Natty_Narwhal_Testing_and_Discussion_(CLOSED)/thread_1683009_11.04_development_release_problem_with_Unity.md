### Post by 3rdalbum on 2011-02-06
If there's a launcher for File Manager, you can use this to navigate to /usr/bin/gnome-terminal and then keep it in the launcher for running programs. Or you can use the File Manager to navigate to /usr/share/applications.

Or, as the earlier poster said, just use the classic desktop until Unity is sorted out enough for you; you might also need to install the unity-places-applications package again.

### Post by PGHammer on 2011-02-07
> **nm_geo said:**
> At login after you click your username and before you put in password, check the bottom of the page pick Ubuntu Classic Desktop.  You will get a more Gnome User friendly page to work in.

Part of the issues is that Unity is simply not as well-developed as GNOME, especially for use as a UI replacement for GNOME.

While Unity may have uses for resolution-challenged devices (netbooks and the like with screen sizes 14" and smaller), it's the users with larger displays that, to be honest, want more choices than are currently given by the default layout of Unity.


Currently, Unity is rather sparse in terms of configuration *and* configurability, and is way too new to have any sort of adjustment options for users.  Instead, I've found myself going all the way back to Kubuntu, as opposed to Unity (or even GNOME).
