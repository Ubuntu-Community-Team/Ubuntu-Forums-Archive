### Post by terry_gardener on 2011-03-23
i have just found an easy way to add documents to the unity launcher which might be useful to others. 

1. right click on the desktop and then click create launcher in the dropdown select application and then give it a name. 
2. click the browse button and then locate and click the required file.
3. the filepath should be in the command field, enter the program name at the beginning for example for pdf docs enter evince. 
4. then ok it and the icon is on the desktop. 
5. move the desktop icon to the launcher. 

done.
