### Post by d3v1150m471c on 2011-04-09
when you reboot hold down shift until your grub menu appears. At which point you'll want to boot into recovery mode and then go to boot into root shell prompt with networking. **have your ethernet cable plugged in**. while in the root terminal issue the following command:
```

sudo apt-get update
sudo apt-get --purge remove xserver-xorg-core
sudo apt-get install xserver-xorg-core

```

Now reboot and see if that works. if not do it again with the following:
```

sudo cp /etc/apt/sources.list sources.list.bak
sudo sed -i 's/natty/maverick/g'
sudo apt-get update
sudo apt-get --purge remove xserver-xorg-core
sudo apt-get install xserver-xorg-core
sudo sed -i 's/maverick/natty/g'
sudo apt-get update

```
