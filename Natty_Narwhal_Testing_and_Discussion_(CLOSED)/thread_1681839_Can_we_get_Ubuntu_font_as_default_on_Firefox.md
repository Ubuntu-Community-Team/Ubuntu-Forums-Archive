On [my blog]("http://freeparticles101.blogspot.com/2011/02/getting-most-out-of-ubunt-font.html") I have a detailed explanation of getting the Ubuntu font to be the default font used by web browsers to display websites. The experimentation was carried out on Maverick. 

I was blown away by how beautiful websites looked when they used the Ubuntu font. The font was fuller and smoother. I don't know why Canonical  doesn't change the default font settings to allow Ubuntu to be the font used by Firefox, after investing resources in developing the font. It will impress new users testing out Natty. 
Although my solution was a hack someone might be able to come up with something more elegant.
