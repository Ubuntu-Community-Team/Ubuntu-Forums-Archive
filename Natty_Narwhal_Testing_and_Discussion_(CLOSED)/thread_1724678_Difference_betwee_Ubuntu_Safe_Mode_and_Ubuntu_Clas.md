### Post by seeker5528 on 2011-04-08
Since the normal Ubuntu log in now gives you Unity for the desktop if Unity is able to run.

Ubuntu Classic was added to give you the old Gnome/compiz desktop, and the Classic no FX option for the old Gnome/Metacity desktop.

Ubuntu Safe Mode is the same thing it always has been, a basic Gnome/metacity desktop without any of the extra things the may be set up to auto start at log in, hopefully giving you the best opportunity to log in to a GUI environment to try to fix things instead of being stuck at the command line only being able to use CLI tools to fix things if your normal log in fails for some reason.
