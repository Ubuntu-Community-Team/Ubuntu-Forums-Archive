### Post by mc4man on 2011-03-21
Find or create a folder - right click 'open with - other application' - 
from the list choose "Open Folder"

The default now is that the 'remember this ..' option will not be checked - so in this case enable the option and then click open

In the future if using the 'open with - other application' don't enable the remember option when used on folders or on files unless you wish the default changed

edit - see you fixed
