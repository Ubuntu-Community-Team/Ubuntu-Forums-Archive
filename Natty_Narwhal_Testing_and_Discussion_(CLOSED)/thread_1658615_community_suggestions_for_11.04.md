### Post by fisch246 on 2011-01-02
I figured since the first Alpha for 11.04 is out, it's about time to start a thread for everyone to post their ideas in a centralized place :)

I will start out this thread by posting a couple of ideas since using the "Live CD" version of the Alpha. Being as I never installed it, I understand my opinion really isn't that helpful but hey, I used it, so it counts :P

Anyway, first I noticed that there was no organization in the drop down lists. Which is fine, but then I noticed all of the applications were all in one place. Which is also fine, but my problem with this system, is for people who aren't used to Ubuntu. Who have no idea what to enter into the search field. Such as if I came from Windows and wanted to change my background, I'd search for "Desktop Preferences" with no success. Now I do realize this is still Alpha, but I would prefer saying something, instead of it staying the same. My solution would to organize all of the apps into folders or some other form of archiving, and then have the search feature look into those folders. That way, if you wish to browse, or search, both are offered equally and intuitively.

The second problem I found was when I was screwing around with the "special effects" options. When I turned off all of the effects Compiz crashed, as because the dock is using special effects to run. Now that's not really a problem for me because this is the first time I've actually found a good reason to turn on Compiz, but I think it should be laid out that the dock is included in this, and probably should disappear entirely when the effects are turned off.

Glad I could participate in the production process of yet another distro for Ubuntu :D To be honest I think 11.04 is looking quite good, and I am very excited for it :) :popcorn:
