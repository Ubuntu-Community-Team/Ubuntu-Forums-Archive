### Post by cariboo on 2010-12-14
If you read the popup, it means you have to log into the classic desktop and install the drivers. 

To test unity with the Live CD, once at the classic desktop open synaptic, and make sure all the repostires are selected, then click the reload button, once that is finished, search for:

```
glib1-mesa-dri-experimental
```

and install it. then log out and log into the Ubuntu Desktop.

**Note**: the above package only works with the nouveau drivers.

### Post by mc4man on 2010-12-16
Even at this early date it's pretty straightforward and certainly clear. (and obviously if can be refined and or changed.

You log into the Desktop Edition on first boot, the pop up is direct if drivers are needed - install them if you want unity, if not then restart and login to the Classic Edition.

If you do need to install drivers then after rebooting the unity plugin is enabled for you automatically 
ccsm is only needed if you want to make changes from the defaults - whether for the desktop edition login or classic. (which have separate profiles for each login
