### Post by Odisej on 2011-04-17
Well, since there are so many posts about the problems of the new beta 2 release I decided to post a positive one as well. One may get a feeling you have to be nutty to install Natty if  reading top posts of this forum.

I did an upgrade from 10.10 and all worked well. The system rebooted and Unity loaded without problems. Impressions are very positive and Unity seems to be a step in the right direction. If you don't like it changing the default DE is so simple at login fetus can do it.

I noticed it is a bit hard to use and there are quite a few rough edges there. It will take two to three cycles before it is on par with KDE 4 or GNOME 2. At least that is my optimistic assessment. 

Some of the problems I encountered: 
1. Using Alt+TAB did not work well for me to switch tasks, I could switch between windows but only doing so blindly as I could not see the currently selected window.  

2. Pulseaudio still does not use the correct input source (same problem with all latest Ubuntus I had installed).

3. Software Center crashed, bug reported but was not crucial. 

4. When having quite a few windows opened it is difficult to navigate through them. This is probably due the new logic Unity introduces for dealing with this. It is hard to change old habits.

5. Sometimes the three buttons on the edge, including close and maximize, disappear. It happened a few times using LibreOffice Calc.

Positive impressions:

1. The size of the usable screen area is much better than other DE I used and compared to Windows Vista or 7 I am using. 

2. All applications that were installed under 10.10 work great under 11.04. 

3. It is fresh and it has great potential for development. If one is used to previous Netbook editions of Ubuntu it is that much easier to get used to new environment.

4. Very easy to launch the most used applications. 

There is much more of course. But just wanted to encourage users to try the new release and judge by themselves. The release is more stable than it would seem from problems reported.

I have Asus F5Z laptop.
