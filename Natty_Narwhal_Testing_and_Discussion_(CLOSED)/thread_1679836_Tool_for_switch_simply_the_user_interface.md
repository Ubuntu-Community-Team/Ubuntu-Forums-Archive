### Post by cariboo on 2011-02-03
To load a different desktop, you need to log off, it can't be loaded on the fly. During a desktop install you have to specifically choose auto-login. Most users doing an install for the first time, usually use as many default settings as they can, it's only after they have done an install, that they want to set auto-login to make their system more Windows like.
