Which netbook?  I've been running Maverick 10.10 (regular, not remix) and Natty 11.04 on my Acer Aspire one just fine.  Now this netbook has 1 GB of memory and a 250 GB hard drive and typical 1.66 GHZ Intel Atom.  It cost $248 at Wall Mart and was built I think November 2010.  I split the hard drive between Windoze 7 starter and Maverick 10.10 runs fine (well, pretty sluggish with Windoze..)

For testing Natty on the netbook I plug in an old laptop hard drive in an inexpensive USB case.  It's got Maverick, Natty, and a partition with stuff to save when installing.  Natty has been screwing around with Grub 1.99 which I don't want to hazard on the main hard drive with its Windoze 7 including some goodies from Acer.

Frankly, from the desktop response which is nice and crisp, you can't tell that the USB hard drive is any slower than the built in one.

I've tried a couple "remix" and couldn't figure out what they were for, so I just run regular Maverick & Natty.  Now Natty is an Alpha and for weeks has had a lot of trouble with install etc.  I chanced to do an install on the USB hard drive in early March CD Daily Build and have been doing updates since.

The Acer Aspire one will do Unity 3D which is what I use, as a tester looking for bugs.  On an older laptop Thinkpad T40 I run Unity 2D which in my opinion is faster and crisper than the muddy see-through 3D.  The difference is the "eye candy" shading and other tricks Compiz does on 3D.  Compiz's main feature is crashing which I've endured since Intrepid.  For me Compiz has nothing to do with function or using applications.

So when you ask a question on the forums you may get more of an answer if you say what hardware you're using.

Example I'm using a Microsoft blue tooth mouse because I can't stand the touchpad.  Sale for $24 at Staples.  Has a little USB plug in and the mouse runs on an AA battery, rechargeable in my case.  No freezes that I've identified other than Natty Alpha is not! stable.
