### Post by Mr. Picklesworth on 2011-04-12
> **searayman said:**
> Say I just type in a letter like h, I get different results each time. Also I get files that dont even have the letter each in their name...

It searches files based on their location (so if I have something under my &#8220;School&#8221; folder, the file is tagged accordingly) and some other properties. File name is a very small part of Zeitgeist's coverage.

Also, as mentioned, Zeitgeist indexes temporally instead of spatially. So, it watches files you have been working on recently instead of crawling through your entire home folder. It has rather different strengths and weaknesses compared to something like Tracker or Nepomuk.

Keep in mind the global search is a little different than accessing a lense directly. You can access the Files lense with the Find Files shortcut from the dash, the appropriate icon in the launcher, or Super+F if you want to get there quickly. From there, you can narrow your search to folders and specific types of files. It might give you happier results.
