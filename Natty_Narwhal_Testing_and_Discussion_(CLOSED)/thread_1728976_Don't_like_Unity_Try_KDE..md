### Post by jfernyhough on 2011-04-14
Out of interest I thought I'd have another go with Kubuntu. Turns out the new global menus tie into the existing KDE functionality (since pretty much forever QT applications within KDE have been able to have their menu bar separated, e.g. at the top of the screen).

Not only do menus work but the task manager can pin window launchers. If you want you could even set it up to look like a traditional dock.

If you have some spare minutes, have a go. You may be pleasantly surprised.
