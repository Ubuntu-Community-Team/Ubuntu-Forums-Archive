### Post by tlcstat on 2011-04-19
Greetings,
I have been using Nero Burning Rom in Windows since it was originally created. Back in the old days it was the only thing the "real" computer geeks used, primarily for some of the reasons given by The Hedge. Two things that I have used it for is 1. merely backing up data files without corrupting the disk. I get 100% results for this task. 2. for copying Video Vob files to a Video DVD. Same 100% result. The problem with comparisons is that Nero has a lot of capabilities that I have never used nor needed. I am sure the same goes for the repository software. Audio is a big part of the burning needs and I have never burned a single audio disk. So you can see the bias that can enter the opinion process. Of course I have K3B and some of the other highly rated native fair but have not been able to get sufficient results to keep me from using a paid solution. After all blank media costs money, its not just the software that should be considered. 

In addition I use a registered copy of Moneydance Financial since it is the best replacement for Quicken that my wife would accept. Hence no more Wine and no more dual boot. That to me is worth a $60 registration. It lets me get a headache out of the house. Will Moneydance do anything that the others won't? Absolutely not! It is just a better Quicken replacement for the Windows crossover user. And it does a great job importing Qif files which makes the transition easier. 

Lastly, there are a lot of threads on this forum decrying the fact that developers don't port to Linux like they do for Windows. Hence there are tons more software for the Windows environment. This is the biggest reason for the success of Windows. People put up with its failings because of the availability of software. So I think if someone is using a for fee software that has been ported to Linux I should be somewhat grateful that Linux is getting that consideration. That is what we have asked for and we should be open minded to those that use that for fee productivity. 
Hope this helps and I apologize for my (admitted) rashness. I love Linux and I owe you guys more than a fast ill conceived answer.
tlcstat
