### Post by ronparent on 2011-03-09
A pre-release like 11.04 is still under development. By definition, at any given point something won't be working. Someone else will probably be having the same problem and report it. At worse if you wait long enough and update it the problem will probably take care of itself.

If you are unwilling to be a guinea pig and help solve the problems you should not be trying to use any pre-release issue.
