### Post by terry_gardener on 2011-03-25
you would have got a quicker response in the natty development subsection since people there have used unity to some degree. 

put i will try and answer your questions because i have used unity for the past few months.

1. minimised windows.
when you open an app it appears in the launcher, and you get a white triangle on the left hand side of the icon, the number of triangles on the left indicates the number of windows you have open for this app. for example with you open 2 firefox windows you will get 2 triangles. 
to maximise the window just click the icon. it will goto the last active window for that app. double click the icon and it will open a scale effect with all the windows for that app and you click the one you want. 

note: the white triangle on the righthand side of the app icon shows the active window, should be only one of these in the launcher as there can be only one active window at a time, as it is used for the global menu.

2. i don't know about the minimised firefox window for gmail etc because i use email client for my email. 

if you use evolution or thunderbird for gmail and have it minimised then you can get notification using the notify-osd system and envelope turns blue. with thunderbird you can get it to display the number of unread messages direct in the unity launcher. 

note: for this to work in thunderbird you have to download the addon.

### Post by bash on 2011-03-26
Double clicking the application icon in the dock on the left should normally "un-minimize" all windows again. Clicking the app icon a third time brings you to the open window overview again where you can now select your previously minimized window.
