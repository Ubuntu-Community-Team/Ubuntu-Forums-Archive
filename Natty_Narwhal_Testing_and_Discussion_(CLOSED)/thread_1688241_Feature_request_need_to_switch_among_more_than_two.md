### Post by mlg7 on 2011-02-15
Problem:

I want to configure my computer to allow input in English, Russian, Chinese and Polish. (I cannot replace English by Polish because Ctrl+something combinations do not work in the Polish input mode.)
So I install all four and find out that I have to press Alt+Shift many times to switch between 2 of 4 languages. This is annoying.

So, the problem is that:
it is inconvenient to switch among languages if there are more than two languages installed.

Proposed solution(s) [I would prefer the last one among these three]:

a) For each language, allow a specific key combinations to switch to that language. For example, win+e for English, win+c for Chinese, etc. Alt-Shift still works.

b) Have two input layout rings.
Alt-Shift (the combination is configurable) switches within the first ring (most frequently used layouts), in my case, at work these would be Russian and English.
Ctrl-Shift (the combination is configurable) switches within the second ring (less frequently used layouts), in my case, at work these would be Polish and Chinese.

c) It is possible to combine a+b and allow multiple language layout rings.
For example,
Alt-Shift: ring1, switches between Russian and Polish
Alt-Win: ring2, switches to English (only one language in that ring)
Alt-Ctrl: ring3, switches between English and Chinese

If the current input ring is N, and the user switches to ring M, the first language from the ring M is selected (so that the user does not have to remember the states of all rings).
