### Post by jerrylamos on 2011-01-27
Aspire one D255E today's CD live Jan 27 that's 20110127 Unity was up and running from USB flash memory to the level of whatever is O.K. now for Unity 3D.  That's with intel integrated graphics N10.  That's the good news.

Bad news is, did a restart from Natty Unity 3D on the USB flash memory  and the Aspire went black. Period. Took me quite a while plugging and unplugging things in USB ports, mouse, the USB flash memory, a USB hard drive, nothing. Powered up and down endlessly. Blank.  Turning on the hard drive light would flicker then go out.

Unplugged all USB.  Nothing. Then put the USB mouse into a different USB port and Windows 7 booted.  Whew!  Then I put in my USB hard drive with Ubuntu 10.10 in and now it ran.  

Oof, not likely to try USB Natty flash again on this Aspire.  I'll install it on the USB hard drive using my desktop.
