### Post by rbrick49 on 2011-04-25
how many times have people been told in the past if you install a beta version and you do an sudo update everyday you will end up with the finished product. heres the code from a terminal type
sudo aptitude update {if aptitude isnt installed on your system do this in a terminal {sudo apt-get install aptitude}
then when that is installed do this {sudo aptitude update} it will go through a lot of repoitories updating then do this {sudo aptitude upgrade} it will give you a list of packages if any are available just type y and if you do this everyday you will end up with the final release
