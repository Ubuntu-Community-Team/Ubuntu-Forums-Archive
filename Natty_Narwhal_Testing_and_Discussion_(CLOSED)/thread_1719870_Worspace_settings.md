### Post by encefalocardia on 2011-04-02
I loved how on OS X you could select a specific workspace for an app. Is that possible with Natty? Also, if that's possible then I would like to increase the number of Worspaces.

---

### Post by r-senior on 2011-04-02
Make sure you have Compiz Config Settings Manager installed (the full one, not the "simple" one). You can set applications to go to specific workspaces in the Window Placement plugin and also fixed sizes if you wish. You want the second tab.

You'll find it right at the bottom of CCSM.
