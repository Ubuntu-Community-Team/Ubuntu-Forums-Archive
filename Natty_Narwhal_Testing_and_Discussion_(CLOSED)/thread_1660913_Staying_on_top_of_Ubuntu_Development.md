> **amano said:**

...
One might prefer AWN, one might prefer Docky, one might prefer GNOME Shell, one Unity. It is just a matter of preference and a default program can be easily replaced.

Well something like that, but not exactly.
ATM it is easy to switch between Gnome-Panel and Unity or add AWN or Docky to one of those.
But, Gnome-shell is not anymore so simple to switch to. It is based on GTK+3, while other UIs mentioned are GTK+2 software.
Packages that Gnome-shell depends on (like gnome-settings-daemon) will break some GTK+2 applications.

If also Unity and Gnome-Panel would be GTK+3 UIs, it would be simple again.
Then again Gnome-Panel will never be developed further. No one knows about Unity for sure.
