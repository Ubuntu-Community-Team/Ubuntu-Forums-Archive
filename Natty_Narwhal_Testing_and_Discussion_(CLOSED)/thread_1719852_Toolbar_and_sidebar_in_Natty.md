### Post by r-senior on 2011-04-02
The panel and launcher are fixed in the Unity desktop. If you want applications to appear in the panel and have a panel at the bottom (and possibly top), you need the Classic interface.

You can select this before you log in, or you can set it as default in your Login Screen settings. Hit the Ubuntu button in the top left corner and search for "Login Screen".

Of course, you could try Unity for a few days and see if you get to like it. ;)

### Post by r-senior on 2011-04-02
Make sure you have Compiz Config Settings Manager installed (the full one, not the "simple" one). Go to the Ubuntu Unity Plugin and the Experimental tab. Various options here, including icon size - minimum is 32px.
