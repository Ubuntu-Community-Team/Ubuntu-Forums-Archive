### Post by LordDelta on 2011-04-04
@cecilpierce

I'm not an expert at this, but in case this helps at all:

You might want to logout, log back in, but before you do, switch your session to ubuntu classic (there should be a drop down box somewhere in gdm login screen, its a blue arrow I believe in the default kdm login screen)...

Hopefully your desktop should start enough that you can hit Ctrl+alt+t, and run this:

```
nohup compiz --replace &
nohup gnome-panel --replace &
nohup ccsm &
```

That worked for me, I had to run the ccsm (compiz configuration settings manager) to turn on all the window manager options that let me do useful stuff.

Also, I had a rogue unity-window-decorator screwing with my windows, so if you can't move the windows all of a sudden, you might try
sudo killall unity-window-decorator

Yeah, ugly hack, hope it helps though.

There is a systray whitelist where you can add apps, either on a per app basis or "ALL"
I prefer per app
after adding one should test how it works and does it create any issues. Most i've tried are fine, a couple will interfere w/ the global menu (of that particular app

Can be added either thru dconf-editor or thru gsettings (man gsetting is quite helpful
If adding thru dconf-editor keep to the formatting

Ex. of gsettings - added vlc and exaile to default set
```
gsettings set com.canonical.Unity.Panel  systray-whitelist "['JavaEmbeddedFrame', 'Mumble', 'Wine', 'Skype', 'Vlc', 'Exaile']"
```
