### Post by Harry33 on 2011-04-17
There is no common issue with mouse or keyboard input system in Natty.
So waiting does not solve anything.
See that this package is installed:
xserver-xorg-input-evdev

What graphics card do you use and do you have Nvidia or ATI proprietary drivers installed?

If not delete the file (if it exists):
/etc/X11/xorg.conf

Check that you can log in recovery mode (choosed from grub menu).
