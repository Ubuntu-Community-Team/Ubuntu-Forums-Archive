### Post by handy on 2011-03-12
Which latest AMD/ATi drivers for Ubuntu?

There are the proprietary drivers "Catalyst" & the open-source stack. Which you are currently using.

If you want to install a later kernel & the latest open-source Gallium driver stack. Then you need use the xorg-edgers ppa.

There is a how-to down in the Ubuntu Stuff: section of this post:

[http://ubuntuforums.org/showthread.php?t=1238129](http://ubuntuforums.org/showthread.php?t=1238129)

If you follow the how-to, do it in the order listed.

### Post by cariboo on 2011-03-12
