### Post by SevenMachines on 2011-04-16
I think the files search thing of unity doesnt actually actually search for files but uses zeitgeist (correct me if i'm wrong here). You can install gnome-activity-journal to get an overview of what zeitgeist has indexed and to blacklist directories and so on, but essentially this kind of file 'search' is very different to how it traditionally works. zeitgeist, and unity in general are not very configurable at the moment

[EDIT] For unity config you'll want to install compizconfig-settings-manager
and edit the unity plugin settings in 'desktop'->'ubuntu unity plugin'. Not too many things can be changed yet though
