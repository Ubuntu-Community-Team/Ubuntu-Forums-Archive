### Post by daengbo on 2008-03-15
I get that, but the window may be opening to show the entire page, which could be set through zoom. Evince will generally remember the window geometry and last read page for a document, as you mentioned, but it will try to open the window so that ~80% of the page length is visible. Try setting the zoom to 50%, closing evince, then opening several new pdfs to test it. You'll see what I mean.
