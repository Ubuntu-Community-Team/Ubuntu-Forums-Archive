### Post by bdb on 2006-10-13
Finally got my 2 monitors showing one beautiful gnome desktop. This makes me very happy.  Thought I would put my 2 cents in and share what I did.

I've got a GeForce 6600 GT with 2 monitors, an AOC Spectrum and some generic gateway monitor.

1) I first upgraded my drivers to the nvidia-glx drivers.
I did this by clicking Applications > Add/Remove... > Advanced > Search for nvidia > check nvidia-glx > click Apply

*WARNING*:
These next steps scare me.  You should write these directions down (or print them) because it turns off (stops) the X server.

1a) then I typed "sudo /etc/init.d/gdm stop" KABOOM!

1b) then I typed "sudo nvidia-glx-config enable"

1c) finally I typed "sudo /etc/init.d/gdm start" (cross fingers and hope it turns back on)

2) Next I copied someone else's xorg.conf file :mrgreen:  and made some modifications.
Thanks to Audax321 for the xorg.conf sample at [linuxforums]("http://www.linuxforums.org/forum/linux-desktop-x-windows/23758-running-dual-x-servers-dual-monitors.html")

I also used [xorg-edit]("http://www.cyskat.de/dee/progxorg.htm") to make changes to the file.  Of course you can just use gedit or whatever.

Here's my version of the file:

	InputDevice "Configured Mouse"

Section "DRI"

3) I restarted X again
type "sudo /etc/init.d/gdm stop"
then type "sudo /etc/init.d/gdm start"

4) tada!  it works!
