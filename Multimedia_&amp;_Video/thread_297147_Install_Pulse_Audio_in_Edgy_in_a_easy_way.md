### Post by ttoine on 2006-11-10
Hey people from the UDS-MTV

If you want to install Pulse Audio to replace your ESD in Gnome, just add this repository :

"deb [http://ubuntu.ttoine.net/](http://ubuntu.ttoine.net/) edgy-music pulse"

Then look for "Pulse Audio" in Synaptic with "Name & Description" option, ad install every package starting "pulse" or "pa".

Lennart, the man who created Pulse Audio, has recompiled everything this morning on my laptop for Edgy. So feel free to use it, it is not as unstable as a backport.

If you want to make some feedback, please comme on the english section of [http://forum.ttoine.net](http://forum.ttoine.net), it is better than here for managing that kind of stuff.
