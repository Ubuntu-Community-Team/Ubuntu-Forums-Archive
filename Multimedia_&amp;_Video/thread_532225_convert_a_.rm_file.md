### Post by DrScum on 2007-08-25
After intense searching and reading I found a solution myself: it's the sound converter ( [http://soundconverter.berlios.de](http://soundconverter.berlios.de) ) which can be installed conveniently from the ubuntu repository. Just go through applications - add/remove, search for sound converter, select for install if not already installed and go converting.

In order to get Mplayer to save the rm file you need to use the commandline though, the GUI doesn't work in my hands. I can't load a rstp adress through the Mplayer GUI.
