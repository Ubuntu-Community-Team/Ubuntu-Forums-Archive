### Post by pljones on 2006-12-08
Try running under WINE.  One of the major "problems" with studio software is that there is no common format for project files.  Each application hs their own proprietry format that they don't reveal easily.

Your best bet is to load up under Windows, convert sequences to MIDI tracks and export them to a .MID, then pull the .MID into a project under, say, RoseGarden.  I'm still building up my arsenal of Linux audio tools so I don't know of a good streaming sampler that runs with JACK - Hydrogen is as far as I've got (I don't think it streams).
