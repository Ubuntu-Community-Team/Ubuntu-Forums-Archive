### Post by Romanovzky on 2008-02-03
As far as I know there are codecs for linux for all the major encoding protocols, such as DivX, Xvid, mpeg4, pm3, etc

To enable them, you just need to install the Gstreamer packages, to do so go to the add/remove application (or synaptic, do with which one you prefer), and install the ones with the codecs you need.

After the installation it is expected all the multimedia players (i.e., VLC, Totem, Rythmbox, etc) to play without problems the multimedia files.

This codecs can also be used by browsers to decode incoming streaming videos and audio files, as it seems to me to be your request.
