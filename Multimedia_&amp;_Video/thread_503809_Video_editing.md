### Post by rsambuca on 2007-07-18
I also think cinelerra is very good, but it is very difficult to use.  If you want "Easy-to-use", I would recommend kino and avidemux.  Both are in the standard repositories so you can install with Synaptic package manager or from the command line:
```
sudo apt-get install kino
sudo apt-get install avidemux
```

I am not sure that either one by itself will do exactly what you want, but between the two, you should be ok.

Good luck

### Post by Cryophallion on 2007-07-27
Kino is probably best for what you are looking for. Cinelerra is a little too complicated. Kdenlive is great, and has a good razor and points tool. I haven't had much luck getting a good workflow out of lives, jashaka, pitivi, etc.

EDIT: I missed when you said titles. Kino doesn't do that. kdenlive does, but doesn't have justification tools (this should be remedied in the next release, rumored to be coming soon), but you can easily import from gimp.
