### Post by oktagon on 2006-09-27

Earlier when I tried using the same busIDs, I did the Screens in the serverlayout section like this:


### Post by veloce on 2007-02-12
Found my solution on an old thread - gnome must keep some stuff that I don't want.  The solution:

Delete the ~/.gconf/desktop/gnome/screen folder.

Works for me - my desktop now covers both monitors seamlessly.
