### Post by sawjew on 2007-06-25
I have had a lot of trouble accessing the videos at this site [http://apex.vtc.com/ubuntu-linux.php]("http://apex.vtc.com/ubuntu-linux.php") but have finally managed to do it using Quicktime in Crossover Office using their Quicktime plugin for Firefox.  I had no success with mplayer, Totem (xine or gstreamer) or Vlc but this may be because I am using 64 bit Ubuntu.  I only managed to get it to work at all with a 32 bit browser and the quicktime plugin.

If you don't have access to Crossover Office you could try Wine and installing Quicktime.  you may find it easier to install Wine-doors [http://www.wine-doors.org/wordpress/]("http://www.wine-doors.org/wordpress/") and use it to install Quicktime.  I don't know if this will install the browser plugin though but you could install the Windows version of Firefox using Wine and I have tested this and it works.
