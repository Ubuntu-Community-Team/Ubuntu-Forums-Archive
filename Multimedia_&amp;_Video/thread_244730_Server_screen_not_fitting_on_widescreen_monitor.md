### Post by avmahajan on 2008-01-19
Hi, I too struggled a lot while installing the Ubuntu Server 7.10 on my older machine, but using a wide screen Viewsonic 20" monitor. Was not able to see the command line. 

Started looking at forums for answers. Nothing satisfactory. However, found an extraordinarily simple solution. There is a button on the monitor for "auto adjustment". On pressing this, it automatically adjusted the screen to fit the command line screen.
