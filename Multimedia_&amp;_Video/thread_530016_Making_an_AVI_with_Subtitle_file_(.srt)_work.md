### Post by RomeReactor on 2007-08-20
Hi. As far as I know, if the **.srt** file is named exactly like the **.avi** file, it will load automatically in either Totem or Mplayer when you start playing the video; I suppose the same is true for VLC, but it's been ages since I last used it. As a side note, the .avi and .srt files should be on the same folder.
