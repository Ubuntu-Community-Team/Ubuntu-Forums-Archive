### Post by wconstantine on 2007-05-13
Does it have any shortcuts and if yes how can I modify them to my liking? I'm so stuck with Winamps shortcuts so I can't live without them. I'm currently using amarok but I think it would be better with XMMS.

EDIT: MATTER SOLVED.

The solution was to download xbindkeys and I configured it to do the commands that I liked by running xmms --help.

Then I configured the command, lets say xmms -f for next song to ctrl + alt + pgdn. And voila. Works fine!
