### Post by Cultghost on 2007-10-10
Having struggled with this for about a week I post to this forum then figure it out in about 1/2 hour!

There was two instances of twonkymedia server running when I looked at the processes running. Killing one fixed the problem. Hopefully this will save someone else the frustration.
