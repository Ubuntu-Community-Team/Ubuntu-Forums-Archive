### Post by Sutekh on 2006-03-05
Make sure when you try to make mplayer the default you run through this procession

Right-Click a File, then choose **Properties**, then choose the Open With Tab, and choose mplayer instead of totem.

[People often get this mixed up with Right-Click and the Open With (forgetting the Properties bit), which opens the file with mplayer rather than sets it as default]
