### Post by djrobthaman on 2007-05-01
Amarok seconded.

If you want to play mp3s you're going to have to add an extra package.  It really should display a pop up the first time you try to play an mp3 saying something like "click here to play mp3s" (it did for me in edgy) but for some reason on feisty (on my machine at least) it freezes every time it tries.  

But, if this happens you can just go to synaptic and install libxine-extracodecs (i think that's the name of it... just search for libxine and download the one that looks like that).  After that you're all set and it works great.
