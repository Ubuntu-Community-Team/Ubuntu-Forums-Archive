### Post by amoore on 2007-08-09
go to:

system>preferences>session -Add- elisa.

go to elisa's config file and edit the section for full screen (If you want elisa to start in full screen). The config file can be edited by this command.
```

gedit ~/.elisa/elisa.conf
```
