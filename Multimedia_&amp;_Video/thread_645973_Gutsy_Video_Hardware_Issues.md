### Post by acoustibop on 2007-12-20
FWIW, I have a machine running an AMD 1400+ CPU and an old ATI Rage Fury card - it runs DVDs etc perfectly.

I use gxine for playing DVDs; see the [Ubuntu 7.10 documentation]("https://help.ubuntu.com/7.10/musicvideophotos/C/video.html") for this.  You'll also need to install libdvdcss2 and w32codecs (or w64codecs) from Medibuntu.  See [here]("https://help.ubuntu.com/community/Medibuntu") for adding the repository to your lists and installing those packages.
