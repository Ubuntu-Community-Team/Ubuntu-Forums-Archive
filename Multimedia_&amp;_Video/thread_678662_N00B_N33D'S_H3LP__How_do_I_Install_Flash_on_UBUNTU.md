### Post by gandaran on 2008-01-26
you can install the flashplugin nonfree, just look it up on the synaptic package manager and  install it, if it doesn't work, (some people have reported the link is broken) you can do a local firefox installation, to do that you you must remove/deactivate the ubufox firefox extension, (look it up on the  firefox menubar » tools » extras), close and start again firefox , go to any webpage that users flash, and click install when you get that flashplugin needed message.
