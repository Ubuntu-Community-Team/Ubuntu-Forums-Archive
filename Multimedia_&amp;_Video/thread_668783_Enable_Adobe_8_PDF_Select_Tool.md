I just figured this out and I figured I would share it with you all. I often use this to assist with quotes on my paper, so is why I find it important to have

As those who use Automatix 2, there is a new version of ADOBE PDF Reader 8. It took me a little while to figure how to get the cursor function enabled so that I can select texts, so I thought I would make a post about it.

1) Edit > Preferences > Accessibility

2) Tick the box saying "Always display the keyboard selection cursor."
