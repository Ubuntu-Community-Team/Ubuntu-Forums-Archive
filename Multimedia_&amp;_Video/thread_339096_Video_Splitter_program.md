### Post by cor2y on 2008-03-16
Netgooroo try Kdenlive, its the closest GUI friendly/newbie NLE you will find on linux and because it is using ffmpeg as its backend most videos can be imported into it with the exception of real media  files.

forgeuk if you don't mind using the terminal theres also commandline tools like ffmpeg and mencoder which allow you to actually join or cut up movie files of the mpeg4,asf, mpeg1 and mpeg2 variety. Of course if its necessary that you use an app with a gui theres Avidemux , i certainly have never had a problem with it taking a long while to load divx/xvid/h264/x264 files but i always compile it from source since i want all the support it can give with the codec outputs etc.
I suggest Avidemux , but i also suggest you compile it yourself theres info on how to do this in the tutorial section (might be outdated) or the avidemux official site wiki as relates to ubuntu.
