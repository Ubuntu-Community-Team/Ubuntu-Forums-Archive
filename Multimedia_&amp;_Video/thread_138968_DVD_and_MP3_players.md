### Post by 3rdalbum on 2006-03-03
If you mean software, then Rhythmbox and Totem can handle MP3s. You will, however, have to use Synaptic to download the "gstreamer" plugins (just do a search for "gstreamer" in Synaptic and download it all).

I think most people use XMMS to play MP3s; I sometimes do too. XMMS is a free download from Synaptic, and it even puts itself into your Applications menu. I'm not sure if it still needs the gstreamer plugins though.

### Post by ncappel1 on 2006-03-03
Personally, I use banshee for music viewing and playing, GnomeBaker for burning cds, and sound juicer for ripping (i've always been a little disappointed at the speed sound juicer rips at--Im still working on that one).

I don't know if you already have this or not, there is a great package install program, basically a watered down version of synaptic, that I thought was a great help to use. 
Find it under Applications-->Add applications, or System-->administration-->add applications

For a while I just used this one instead of the 'complex' synaptic package manager, which is very confusing to look at if you're new to everything. With the simple one you can just download the programs with out having to worry if you're doing something wrong. 

if you don't have it, open System-->Administration-->synaptic package manager. then enter your password and use the search feature to find "Gnome-app-install." Right click on it and select install. Then click on Apply, youll be set to go. 

Definitely try out Automatix, which installs some very popular programs: [http://ubuntuforums.org/showthread.php?t=138405&highlight=automatix](http://ubuntuforums.org/showthread.php?t=138405&highlight=automatix)

I hope this has been helpful!
