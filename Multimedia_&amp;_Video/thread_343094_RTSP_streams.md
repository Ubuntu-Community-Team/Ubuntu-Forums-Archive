### Post by dolphinholmer on 2007-01-21
Assuming that your clicking the link to the .ra file in Firefox, click edit, preferences then select the content tab and click on manage under 'file types.' Find RA under the extension column, select and click 'change action.' In the resulting dialog box select 'open them with this application' and enter the command to open realplayer (probably just realplay)
In some versions of Firefox file types did not appear in the 'download actions' dialog box. If this is the case install the latest version of Firefox
