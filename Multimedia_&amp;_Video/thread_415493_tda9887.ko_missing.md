### Post by yorktown on 2007-04-24
I hear a lot of static when I on TVTime, but it is okay now after I did the following and rebooted.

for Feisty Fawn gedit /etc/modprobe.d/alsa-base and add the following line:

options tuner qss=0


for Dapper gedit /etc/modprobe.d/alsa-base and add the following line:

options tda9887 qss=0
