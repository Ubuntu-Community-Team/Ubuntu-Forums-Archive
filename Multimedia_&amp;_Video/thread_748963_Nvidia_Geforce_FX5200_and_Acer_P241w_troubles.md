### Post by Dave_schoorl on 2008-06-29
You have to install the nvidia-settings program with which you can set the resolution for your monitor to 1920x1200 as described at [http://www.funnestra.org/ubuntu/hardy/#nvidia-driver](http://www.funnestra.org/ubuntu/hardy/#nvidia-driver). In short:

sudo apt-get install nvidia-settings
sudo nvidia-settings

Don't forget to press the button to save your changed settings to the configuration file.
