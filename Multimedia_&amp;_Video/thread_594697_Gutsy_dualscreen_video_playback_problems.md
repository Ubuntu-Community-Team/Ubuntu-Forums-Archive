### Post by BatPenguin on 2007-11-03
I'm having same problems, but only sometimes. Occasionally works fine, sometimes not.

I have Gutsy, Nvidia's latest driver (installed through Envy), 7950GT card. Using Compiz-Fusion.

You could try disabling Compiz-fusion (Preferences - Appearance - Effect Off) and see if that helps. Also, try adjusting the playback driver for video to make it OpenGL -- or if it is, try Xv.

I'm pretty sure this is a Nvidia+compiz fusion problem. Search around the forums and you'll see plenty of posts about similar problems, audio fine, video garbled. I get hard lockups occasionally too because of these...really, really annoying.

Lemme know if you figure out something that'll fix this for good. Thanks.
