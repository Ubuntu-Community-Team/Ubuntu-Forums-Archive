### Post by Haegin on 2006-10-15
[http://dvd-slideshow.sourceforge.net/wiki/Examples_0.7.5](http://dvd-slideshow.sourceforge.net/wiki/Examples_0.7.5)

That looks simple enough. The first command pulls all the pictures together and sets the duration. The second command names the slideshow and generates the needed files. The fourth command (third is a test) sets up some buttons and generates the dvd structure. The next few commands let you test it all and optionally copy the original pics onto the dvd as a backup. Finally the last 2 commands make the iso and burn it.
