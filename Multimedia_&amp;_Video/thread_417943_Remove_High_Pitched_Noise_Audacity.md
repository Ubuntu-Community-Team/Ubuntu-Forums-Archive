### Post by firefly2442 on 2007-04-22
So I have an MP3 file and at one point there is a really high pitched noise.  It's almost like the mic got too close to the speakers or something.  Is there a way to automatically remove this or at least minimize it using the effects in Audacity?  Thanks in advance. :)

---

### Post by tgalati4 on 2007-04-22
Import the track into Audacity.  Select the portion of the track where the noise is evident.  Run a Spectrum analysis.  Note with frequencies are peaking.  Run a spectrum analysis on a normal portion of the track so you can see what the overall spectrum looks like.  Highlight the bad section, select EQ and make a notch or notches for the offending frequencies.

Try medium-sized notches.  Only cut out the minimum necessary to clean up the audio.  Play the track so that you can determine how the transistion from good to bad sounds.  Control-Z to undo (up to several times) if you are not happy and try again with different notches.

Export track back to mp3.

Good luck.

### Post by tgalati4 on 2007-04-23
Yes, EQ is a common abbreviation for equalization.  You may only need to lower the high frequencies instead of a notch filter.  This usually is more noticeable though, you want to remove a little as possible, yet remove the annoyance.

Audacity is powerful, especially if you install additional filters.  There are hundreds.
