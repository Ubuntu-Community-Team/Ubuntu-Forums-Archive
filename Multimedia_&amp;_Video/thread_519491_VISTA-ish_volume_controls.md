### Post by kostkon on 2007-08-08
[Gnome is going to replace, some time in the future, its default sound server]("http://live.gnome.org/PulseAudio") from * Esound* to *[PulseAudio]("http://www.pulseaudio.org/")*. 

*PulseAudio* is more advanced and has more features compared to *Esound*, offering things like the one talked here, separate volume controls for every application in the system.

I think the Ubuntu team was planning to have *Feisty* offered with *PulseAudio* but I suppose they did not it as now it is going to be implemented by Gnome. Maybe I am wrong about this, but I am sure I remember that it was supposed to be a feature of *Feisty*.
