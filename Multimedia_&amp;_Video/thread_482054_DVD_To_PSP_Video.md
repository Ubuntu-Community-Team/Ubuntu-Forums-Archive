### Post by Adam_GUI on 2007-06-23
Yo!  I'm doing this already using my Ubuntu 7.x (whatever it is.  I'm being lazy as I type this.)

Using Wine and PSP Max Media Manager.
Picked up the Media Manager at Wal-Mart.

It runs great under wine.
Use your DVDrip program to go from DVD to *.avi.
Then you just connect up your PSP and move the *.mp4 file to your "Videos" folder.

Only problem I've had is the last episode of M*A*S*H, which the PSP reads as being too big for my PSP to play...I think.  But other vids play fine.  Probably just a firmware issue with the size of video files.  Firmware 3.10 I think...last I checked.

Only compatibility issue I've found is that you need to set wine where the window manager DOES NOT have control over the Windows program.
