### Post by Noah0504 on 2007-09-11
From the first day I started using Ubuntu, I discovered the default audio application Rhythmbox.  My first thoughts were that it was quite lacking, but it added many needed features quite quickly in my limited exposure to it.  I used Rhythmbox to provide quick access to my music library, podcasts, and sometimes Internet radio.

For my iPod, I found Rockbox.  It let me play all of my music that was in open-source formats such as FLAC.  However, I soon felt myself missing the default Apple software.  Sure, I could organize everything in a database, but I wasn't able to have quick access to my podcasts.  It wasn't a big deal and I learned to live with it.

Recently I decided I wanted to go back to the default Apple software on my iPod.  I heard that Banshee was a little better in the iPod management area.  So, I grabbed the latest version and was very pleased with how far it has come, and it worked with my iPod just fine -- well, for the most part.  I love listening to several podcasts, and I love how Apple gives quick access to podcasts through its default menu.

A few have mentioned applications like Foola and gtkpod, but I will finish by saying there is the need for a music application that can handle iPods with no problems.  It needs to make playing music fun and second-nature, and it needs to be able to sync with iPods and other audio players with no problem.

I want to try and tackle the problem myself, but I only have limited programming skills in Python.  If anyone wants to get their hands dirty, maybe we can try developing our own media player that can combine the great features of many applications to provide the user with an even better experience.  Please let me know if anyone is interested.

There are several programs that will allow you to transfer music to your iPod on Ubuntu. Most of them use the gtkpod libraries.

My favorite music app is Amarok. It was the clencher that brought me to Ubuntu from Windows. It has a bit of a learning curve, but I find that it does everything I want it to and more. It manages my iPod quite well, has a good infrastructure for smart playlists, and has a good system for rating music. Since it's also scriptable, I use it as my alarm clock, and control it with the remote control I have set up for MythTV. Plus it will automatically look up Lyrics, find Album Art, and give you information about the artist.

As far as iTunes for Linux, I have no interest in getting support for the DRM, but I do wish Apple would create an API for downloading from iTunes Plus, or offer a web interface so Linux users can buy the DRM free versions.

On a default install, there is one: Rhythmbox. I've never used it for this purpose, though, since I use gtkpod, which handles videos. I tried Floola, but ran into troubles. If you're a KDE fan, or like a good, feature-filled player, Amarok is the way to go. If you're a Gnome fan and like a good, feature-filled player, Exaile might be your cup of tea. If you like a one-stop shop for all your music needs (playing, ripping, podcasts, iPod support), then Banshee's right up your alley.
There's no lack of iPod support in Linux. From what I can see, the only thing people complain about lacking which we may actually lack is a sync option. I haven't looked into it, since I use gtkpod, but even so, there's probably some combination of options which could achieve this. If you're good with Python, you could write a Rhythmbox plugin to add this, or else put in a feature request.
