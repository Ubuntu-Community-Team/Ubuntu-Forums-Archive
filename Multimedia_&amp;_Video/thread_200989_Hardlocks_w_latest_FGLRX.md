### Post by ashrack on 2006-08-23
Found a temporary workaround from my bugzilla report:
[https://launchpad.net/distros/ubuntu/+source/linux-restricted-modules-2.6.15/+bug/50535](https://launchpad.net/distros/ubuntu/+source/linux-restricted-modules-2.6.15/+bug/50535)
```
This bug is also reported on ATI Bugzilla : [http://ati.cchtml.com/show_bug.cgi?id=239](http://ati.cchtml.com/show_bug.cgi?id=239)

It seems that there is a workaround :
- For GDM : add AlwaysRestartServer=true in section [daemon] of /etc/gdm/gdm.conf-custom (it works for me)
- For KDM : Add TerminateServer=true in section [X-:*-Core] of /etc/kde3/kdm/kdmrc (i'm not sure of the config file, i don't use KDE)
```
