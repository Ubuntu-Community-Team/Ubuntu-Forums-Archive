### Post by fakie_flip on 2007-05-02
Try doing this.

sudo dpkg-reconfigure -phigh xserver-xorg

Then to restart x for the changes to take place, push control alt backspace, or switch to a linux console and type "sudo /etc/init.d/gdm restart" After you have done that, it should work, but if it doesn't, reinstall your graphics driver. There may be a newer version available that you should install.
