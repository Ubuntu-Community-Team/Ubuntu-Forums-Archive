### Post by Hendrixski on 2007-04-09
I guess it's possible.  It's unlikely because Linux users generally don't accept anything that restricts them from doing things on their system.  It's not synchronous with freedom (AKA consumer rights).

DRM in its current form works like malicious software.  And we know that malicious and immoral applications can only work in places where they can hide (just like people, if you're going to do something wrong, you do it while hiding and not in plain sight).  If everything is transparent, if anyone can read all of the code, then there is no way for DRM to perform malicious tasks.
