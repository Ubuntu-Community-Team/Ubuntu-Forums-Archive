### Post by Bablefish on 2007-05-31
I had the exact same problem when I finally got some help on another board now all DVDs play. It is a 3 part tip
first install Medibuntu if you had trouble like me just go to the main Medibuntu site and get the command lines. Next you need a better player that's Kaffeine Player you can download it from the Add Remove secton of Application, Finally you need to install and run Automatrix you can get it at [http://www.getautomatix.com/wiki/index.php?title=Installation](http://www.getautomatix.com/wiki/index.php?title=Installation) there is a .deb file there which makes it easy. Once you go to the Terminal and run sudo apt-get update you will be able to run any DVD
