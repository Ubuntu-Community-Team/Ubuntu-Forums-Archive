### Post by ske1fr on 2006-09-15
Umm.  forcesofhabit, the Rage 128 is in my Kubuntu box.  The Rage 128 Pro was, I think, the model after.  I can't tell you if it's any good for games, but it's an old card, pre-Radeon, so I doubt it will be very fast.  Because it's pre-Radeon, I don't think those Fglrx drivers will support it.  Certainly my reading of the software release note suggests not.  

If gaming is the main reason for your interest then I suggest you read the software release note yourself at [https://www2.ati.com/drivers/linux/linux_8.26.18.html](https://www2.ati.com/drivers/linux/linux_8.26.18.html) and try to find a card that's mentioned in there.
