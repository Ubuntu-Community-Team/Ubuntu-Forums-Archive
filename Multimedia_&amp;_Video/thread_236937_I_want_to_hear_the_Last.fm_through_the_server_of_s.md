### Post by maubp on 2006-08-15
Have you got sound to work with Rhythmbox?

If you download the latest CVS rhythmboxe (i.e. after 0.9.5) then the AudioScrobbler/Last.fm plugin includes experimental support for playing Last.fm radio stations.

The plugin shipped with 0.9.5 only supports sending track information to Last.fm to update your profile.
