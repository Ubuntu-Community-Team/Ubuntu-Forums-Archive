### Post by christhemonkey on 2007-04-16
Change vesa to nvidia and reboot/restart X server.

This should "just work" if you have nvidia-glx installed...
but you may get xserver errors, if possible please post them here.

OpenGL support is available through the mesa library for software rendering with any driver,
i presume you mean hardware rendering?
In which case, yes the nvidia-glx driver is what you need.
