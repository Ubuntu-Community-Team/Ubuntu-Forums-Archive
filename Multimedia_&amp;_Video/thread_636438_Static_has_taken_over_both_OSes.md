### Post by claw_atticas on 2007-12-09
I recently installed Ubuntu Feisty (Ubuntu Gutsy kept crashing about halfway through the installation), and upgraded to Gutsy through Feisty's update manager, but ever since I installed Ubuntu, I heard constant static. When I play video, most of the time I can hear the audio fine enough to understand, while the static is still there, and other times the audio is so bad I can't make anything out (Sounds heavily muffled). I tried setting up the emu10k1 (I think that's it), but to no avail.

After deciding I couldn't handle the constant static, I decided to reboot and switch back to XP, but noticed the static is just as bad in XP now than it is in Ubuntu! I reinstalled SBLive! drivers, but that didn't help anything.

Oddly enough, I turn the Master Volume down to mute, and I can still hear the audio and the static, however very quiet.

Dual-Booting:
   Windows XP SP2 (On first hard drive, currently c:\)
   Ubuntu 7.10 (Gutsy Gibbon) (On second hard drive, used to be d:\)
Sound cards:
   C-Media CMI8738 (On-board sound)
   Creative Soundblaster Live! 5.1

EDIT: I've fixed the problem! First I went to XP, and clicked Advanced in Volume Control, unchecked "Digital Output Only" (Which stopped the static in XP), and then when I went back to Ubuntu (There was still static), the option "SB Live Analog/Digital Output Jack" had appeared in the 'Switches' tab, and I unchecked that, and the static stopped in Ubuntu then!

Only problem I'm having now, is that the sound in Windows is stuttering, but not in Ubuntu, so I'm not sure if that's an Ubuntu-related problem (Though it did start around the time I installed Ubuntu)
