### Post by opscure on 2008-03-27
create your ~/.asoundrc file```
 sudo gedit .asoundrc
```
and then enter the following 

```
pcm.!default {
type plug
slave.pcm “surround51&#8243;
slave.channels 6
route_policy duplicate
}
```

See if that fixes your sound issue.
