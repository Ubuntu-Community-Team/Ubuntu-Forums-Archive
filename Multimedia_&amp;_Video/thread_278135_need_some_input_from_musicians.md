### Post by maniacmusician on 2006-10-15
Hey guys/girls/musicians of all sexes,

I'm helping with an ongoing program for linux called the Dohickey project. Basically, to give an extremely short synopsis of what it does:
You download the Dohickey Client onto your computer, and run it. It detects all the hardware that you have plugged in, attempts to recognize some of it, and what it doesn't recognize, you fill in for yourself. You also include details as specific as possible, and talk about the level of support for that piece of hardware. Did it work right out of the box? If not, what did you do to get it to work? or does it still not work? After you have entered all the info, Dohickey will then send all the information back to a central server database, which will catalog and organize that information into a human-readable and searchable database. So, anyone will be able to go to our site, search for that piece of hardware, and have the information that you entered about it show up for them to read. The purpose of this effort is to raise hardware awareness in Linux (it is for all distros, not just ubuntu), and to help people find ways to get their hardware working. 

since Ubuntu Studio is growing to be so prominent, i thought it would be cool to include categories specific to pro audio. What I need from you guys is a list of subcategories we should include under "Pro Audio," and parameters that we should include for them. for example, one of our categories is Digital Cameras. parameters for that are CCD resolution and Optical Zoom. I'll start off with a small list, and i'd like some input as to what else to include.

*Pro Audio
    -Sound Card (parameters?)
    -MIDI instruments (Paremeters: exact model)


I can't really think of anything else that is directly hooked up to the computer and dohickey will be able to detect. any ideas would be cool

Thanks!

edit: please be aware that this is not yet a definite thing. I **want** this to be included, so i'm hoping it will generate some interest, but nothing is for sure. All feedback helps, though.

### Post by damu on 2006-10-16
You can think of another category than soundcard and midi, which could be pci/usb/firewire related hardware. For example, I have a Roland GI20, which is a box that translate into midi the notes played on my guitar ( with a special mic on the guitar). Even though it can work through its midi out port, on Windows, it would be used through usb ( faster, better...); Unfortunatelly, there are no drivers delivered by Roland for linux.
Some other hardware which don't fit in soundcards or midi :
.rack effects which are usb implemented
.mixing tables with usb/firewire (Yamaha 01X, etc)
