### Post by Fitzcarraldo on 2006-10-19
I am using ubuntu 6.06 LTS Dapper Drake.

I discovered yesterday to my delight that, if I move the mouse pointer over any MP3 file icon then, after a few seconds, the file contents start to play without any media player starting up. No doubt many people already know this, but it was news to me. It is a handy way of 'previewing' a music file's contents without having to launch a player.

However, when I move the mouse pointer over a WMA or M4A file icon then, although a music note appears over the icon in the same way as for MP3 files, no music plays. Does anyone know if it is possible to get ubuntu Dapper Drake to preview WMA and M4A files in the same way that it does for MP3 files? Or is this something that can only be done for MP3 files? (I should point out that I have all the WMA and M4A codecs installed and these files play perfectly if I double-click on them to launch a media player to play them.)

Just curious, really. It would be nice to have the same functionality for other types of music file as well as MP3.

EDIT: The following ubuntu Web page mentions that the ubuntu file manager Nautilus can preview music files if you hover the mouse pointer over the file, but it only mentions MP3 files.

[https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)
