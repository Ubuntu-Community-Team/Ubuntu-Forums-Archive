### Post by ron999 on 2008-02-16
Hi
To play the file in a stand-alone DVD player you have to 'author' the file into a DVD structure (a video_ts folder).
There's a program called 'DeVeDe' that will do this.

When it's done you can burn it with 'K3b' program.

Look in Ubuntu repos for DeVeDe and K3b.

You can tell DeVeDe to create an iso image if you prefer, then K3b will burn that also. 
The video_ts folder or the iso can be viewed with VLC player before you burn it, to check it out.

If you want to put more than one file on the DVD it's more complicated/difficult.
You use different software such as 'tovid' to make a menu.
