### Post by Huffalump on 2007-04-21
You're not alone with problems.  Seems to affect a lot of people.  I cannot solve your problem but I can get you a workaround for temporary use.  Boot into recovery mode (choose it from the grub boot menu).  

That will automatically take you into a command line terminal as root.  There, you can do the following:

```

nano /etc/X11/xorg.conf

```

It's a text editor where you can change  the driver "nvidia" to be just "nv" then save (WriteOut) and exit.  Next you can startx.  It should put you into a GUI at that point.  You'll want to solve the proper nvidia driver, but this should at least get you up on crutches.

### Post by shane634 on 2007-04-21
Valinski do as suggested and use the nv driver. Once you get to a gui go and download the latest version of Envy. 

[http://www.albertomilone.com/nvidia_scripts1.html](http://www.albertomilone.com/nvidia_scripts1.html)

  Then start up Envy in a terminal 

  ```
sudo envy -g
```  for graphical

  ```
sudo envy -t
```  for text mode

  Let it install the correct driver for you. Worked great for me. Good luck.
