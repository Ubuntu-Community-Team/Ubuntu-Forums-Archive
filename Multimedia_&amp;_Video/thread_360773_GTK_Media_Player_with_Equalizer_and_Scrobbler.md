### Post by Datalanche on 2007-02-13
Give Beep Media Player a try. It's based off of xmms and is compatible with a lot of the plugins, as well as Winamp skins. I can't recall offhand if it has a media library, but it has an equalizer and playlist.

Edit: Oh, and for scrobbler capability, there is a package for the plugin you can install as well: beep-media-player-scrobbler
