### Post by stooshbunutu on 2008-02-19
Rhythem Box Media Player should already be installed on your machine from the installation. You probably wont need any new drivers installed as ubuntu supports most. Rhythem Box has an iPod sync function but you may need to install some plug-ins such as gstreamer and lame to get your pc handling ipod friendly extensions.

You may also need a program to convert files to ipod friendly formats. I use a program called [winff]("http://biggmatt.com/files/winff-0.33-i386.deb") that uses the [ffmpeg]("http://packages.ubuntu.com/gutsy/graphics/ffmpeg") codec to covert the file.

hope this helps you, it works great for me.:)
