### Post by Denamite on 2006-09-01
Okay, I got this working using an alternative route thanks to abel7's post in this thread [http://forums.gentoo.org/viewtopic-t-319276-highlight-twinview+tvout.html]("http://forums.gentoo.org/viewtopic-t-319276-highlight-twinview+tvout.html")

Made an xorg.conf for my TwinView and then an xorg.conf.tv for the tvout, then bound them together by adding another Server in gdm.conf like this:

> [servers]
0=Xgl #(Original X server)
1=TVOut

[server-Xgl] #(Original X server)
name=Xgl server
command=/usr/bin/Xgl :0 -fullscreen -ac -accel glx:pbuffer -accel xv:fbo
flexible=true

[server-TVOut]
name=TVOut server
command=/usr/bin/X :1 -config xorg.conf.tv
flexible=true

I can now easily switch between TwinView and TVOut with ctrl+alt+f7/f8 while I'm logged in with the same user in both servers and thus keeping my running processes untouched while i watch a movie :)
