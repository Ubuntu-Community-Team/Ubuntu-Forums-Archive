### Post by rainwalker on 2008-01-05
I'm not sure what is causing your YouTube problems, but it's probably Flash.

As for Songbird, you don't actually install anything. After you extract the .tar.gz, go into that folder and there is an executable named "songbird". All you have to do is run that; You can also create a new launcher in your menus by right-clicking on them and choosing "Edit Menus", then create a new item with the path to that Songbird program.

### Post by wolfen69 on 2008-01-05
here [http://getdeb.net/search.php?keywords=songbird](http://getdeb.net/search.php?keywords=songbird) is a link for songbird. just click to install.

for flash, try going to synaptic and completely remove flashplugin-nonfree. if you dont have it, dont install it from there. go here to get flash [http://mirror.ne.gov/ubuntu/pool/multiverse/f/flashplugin-nonfree/flashplugin-nonfree_9.0.115.0ubuntu0.7.10_i386.deb](http://mirror.ne.gov/ubuntu/pool/multiverse/f/flashplugin-nonfree/flashplugin-nonfree_9.0.115.0ubuntu0.7.10_i386.deb)  then just click and install.
