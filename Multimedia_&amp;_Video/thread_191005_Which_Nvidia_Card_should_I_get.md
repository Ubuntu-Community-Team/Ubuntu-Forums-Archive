### Post by Jason_25 on 2006-06-06
That's a tough question.  What graphics bus do you need it for and what processor?  Most cards you would find for less than $80 would not be able to make full use of the 256MB memory.  For instance, I recently picked up a PCI BFG FX5500 on clearance that happens to have 256MB memory.  It is still pretty slow, and not the same speeds as other cards with 256MB memory.  This is because the architecture is slow, the pci bus is slow, and it has a 128-bit memory interface.  I have a PCI-express 6600 GT in another system that is so much faster and it only has 128MB memory.  So, in short, video memory doesn't matter much in your price category.  Anyway. try to go for a 6200 or 6600 with at least the 128-bit memory interface.
