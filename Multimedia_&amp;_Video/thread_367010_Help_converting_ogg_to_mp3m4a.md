### Post by vigleik on 2007-02-21
There are many ways to do this. Because I use KDE, I like soundkonverter. Then I can just right-click on a music file in konqueror and choose to convert with soundkonverter. I think soundkonverter is available via apt-get, but you might have to install some other programs to make it work well. Some of the following, probably: oggenc/oggdec, lame, gogo, faac/faad, mppenc/mppdec, flac, mac, mplayer, ffmpeg, shorten, ofr, ofs, wavpack, lac, lpac, speexenc/speexdec

I know this isn't the answer you'd want to hear but you should try to stray away from anything that demands mp3 use if you can as it's a controlled format and OGG is far better.  Try to get players and other things that support OGG or other open formats so that no one is at legal risk because of mp3 patent BS.  I assume there are OGG players for Mac?
