### Post by chewearn on 2007-02-24
If you are referring to subtitle for computer video file (.ogg, .avi, etc) as opposed to DVD, a text editor will do.  Create a file with .srt extension.  Example of a srt goes like this:

```
1
00:00:02,000 --> 00:00:05,500
This is the story of a little pig...

2
00:00:06,060 --> 00:00:08,800
Who lived in a small farm
```

Player (e.g. vlc) will be able to decode (and display) this file, as well as some set-top DVD players.
