### Post by Jose Catre-Vandis on 2007-09-08
Depends how much fuctionality you want :)

I use mplayer almost exclusively to watch dvb TV. Once you have a channels.conf set up in your /.mplayer directory you simply type

mplayer dvb://"Channel Name"  (Quotes needed if spaces in channel name)

This is very much a hands on approach as opposed to a dvr/pvr setup like myth tv

Have also done some scripting to automate things a bit, to also help with recording and watching and recording. Check here.

[http://ubuntuforums.org/showthread.php?t=369653](http://ubuntuforums.org/showthread.php?t=369653)
