### Post by kebes on 2007-01-07
The open-source drivers work great for typical usage. The proprietary nvidia driver is stable and has very good performance for running things like games, beryl, etc. So if you are doing graphics intensive stuff you will notice a performance advantage using the proprietary drivers.

The only reason not to use the proprietary drivers would then be because ideologically you don't want to use closed-source software. (Often times proprietary drivers are buggy and insecure... but the nvidia drivers seem decent.)
