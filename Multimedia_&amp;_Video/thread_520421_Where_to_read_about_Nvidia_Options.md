### Post by dabl on 2007-08-08
Depending on what you mean by "options", the nvidia-settings utility will set most of your resolution, refresh, and display related settings in xorg.conf -- why not use that?

In a console window ```
sudo nvidia-settings
```

then, on the utility that comes up click on the panel for X Configuration, tell it first to "detect display", then set the default resolution, then set the refresh rate, then click the "Save to X Configuration File" button in the lower right, then click "save" on the little window, and you're done.  Restart the x server with Ctrl-Alt-Backspace and it should come up with the default settings that you just applied. :popcorn:
