sudo dpkg-reconfigure xserver-xorg
```

### Post by jeffus_il on 2008-03-04
Yes, it causes a problem, you cannot be sure which version of fglrx is being loaded, the new or the old.

Do ```
find /lib/modules/2.6.22-14-generic -name fglrx.ko
```

Do ```
sudo rm
``` on each of the files found.

Rerun the install you did previously to make sure you have the unique and correct fglrx.ko module.
