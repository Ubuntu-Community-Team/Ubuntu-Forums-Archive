### Post by pedroleouf on 2005-11-26
Hello !
I have a GeForce2 MX card but no TVOUT, but I think I can help you configuring corectly your xorg.conf file to use the nvidia driver instead (it's better, but perhaps it will not resolve your problem).

Make a copy of your xorg.conf in case of crash:
```

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.back

``` Like this, if you have a problem when restarting X, just overwrite xorg.conf with xorg.conf.back

Install the nvidia drivers by typing:
```

sudo apt-get instal nvidia-glx

``` 
Edit you Xorg configuration
```

sudo gedit /etc/X11/xorg.conf

``` and find the Module section. Change or add theese lines to match (if not like this):
```

    #Load    "GLcore"
    #Load    "dri"
    Load    "glx"  #nvidia OpenGL driver

``` If you have other lines (not dri, GLcore, glx), do jnot remove them.

In the Device section of your video card (where you have : Identifyer "NVIDIA Corporation NV11 [GeForce2 Go]") change the lines to match theeses ones:
```

    Driver        "nvidia" # replace nv with nvidia, witch is the nvidia driver

``` 
Save your file and close gedit.

Close your session System -> Close Session (I'm not sure of the menu label because I'm french and have the french version)

When done, hit Control+Alt+Backspace (del backward key, I'm not sure of the english name), it will restart your X server.

If you have the login screen, it's ok, the configuration works, else, hit Control+Alt+F1, log on into the console, and type:
```

sudo cp /etc/X11/xorg.conf.back /etc/X11/xorg.conf
sudo /etc/init.d/gdm restart

``` It will replace your config file by the backup one, and the second line restart your X server.

Sorry if it's not helping you, but normaly it must work better with this driver, at least for video display and grow up your X rapidity.

PS: Sorry for my poor english
