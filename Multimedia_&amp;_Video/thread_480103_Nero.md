### Post by ubuntu27 on 2007-06-21
I will recommend AGAINST Nero products in Linux. There are BETTER alternatives.

You can try [k3b]("http://www.k3b.org/") or gnomebaker, both are in the repositories. 

k3b has more option than gnomebaker. 


You can install it using the Synaptic Package Manager in Ubuntu or you can use the Terminal which is in Applications Menu, Accesories.

If you choose to use the Terminal, type the following:

For[ k3b]("http://www.k3b.org/")

```
sudo aptitude install k3b libk3b2-mp3
```

For GnomeBaker

```
sudo aptitude install gnomebaker
```


Here is a Review on Nero Linux VS [K3B]("http://www.k3b.org/"):
[http://www.flexbeta.net/main/printarticle.php?id=86](http://www.flexbeta.net/main/printarticle.php?id=86)
