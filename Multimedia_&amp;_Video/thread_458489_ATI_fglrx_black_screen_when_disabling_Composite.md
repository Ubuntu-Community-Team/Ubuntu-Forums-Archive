### Post by eye208 on 2007-05-30
OK, here's what to do:

1. Boot Ubuntu 7.04 Live CD/DVD in safe graphics mode.
2. Enter "sudo dpkg-reconfigure xserver-xorg" in terminal.
3. Skip automatic recognition of monitor and graphics card.
4. Make sure there are only those resolutions selected that your monitor supports. Leave all the other options and values unchanged.
5. Open restricted drivers manager.
6. Mark checkbox for proprietary ATI driver.
7. Ignore reboot request after driver installation.
8. Run "sudo aticonfig --ovt Xv" to enable Xvideo overlay in existing xorg.conf.
9. Press Ctrl-Alt-Backspace to restart X.
10. Check the output of "fglrxinfo". Fglrx should be running with direct rendering enabled.
11. Copy the working xorg.conf file from the Live CD /etc/X11 over to your harddisc /etc/X11.
12. Stop blaming ATI.
