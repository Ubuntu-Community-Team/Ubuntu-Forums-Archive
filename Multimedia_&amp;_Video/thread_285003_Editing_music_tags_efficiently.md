### Post by munkydust on 2006-10-26
I find Easytag is pretty good. I think to install

sudo apt-get install easytag 

and that's it. It can work out tag fields from the file name or the other way around. It's sufficient for me. Hope this helps. For converting dbpoweramp is good and I've read reviews that it works well with wine.

sudo apt-get install wine

then download from [www.dbpoweramp.com](www.dbpoweramp.com) and then

wine [where ever you save the .exe]. I've not tried it through wine though. Good luck.
