### Post by dbott67 on 2006-10-26
This [website]("http://www.bohne-lang.de/spec/linux/modeline/") allows you to generate your own modeline settings.  After entering the setting you posted above, I get this:
```
  Horizontal Resolution:   1600 
  Vertical Resolution:     1200 
  Vertical Refresh Rate:   85.00 Hz 
  Horizontal Refresh Rate: 107.48 KHz 
  Dot Clock Frequence:     301.81 MHz 
```

Add this line to your /etc/X11/xorg.conf file:

 ```
# V-freq: 85.00 Hz  // h-freq: 107.48 KHz
 Modeline "1600x1200" 301.81  1600 1752 2120 2808  1200 1200 1204 1264 
```

Under the "Monitor" Section:
```
Section "Monitor"

	Identifier "Generic Monitor"

	HorizSync 30-64  [COLOR="Red"]*<--- you'll need to set this to whatever your monitor supports*[/COLOR]

	VertRefresh 50-100  [COLOR="Red"]*<--- you'll need to set this to whatever your monitor supports*[/COLOR]

	Option "DPMS"

	**Modeline "1600x1200" 301.81  1600 1752 2120 2808  1200 1200 1204 1264**

EndSection
```

Hope this helps.

-Dave
