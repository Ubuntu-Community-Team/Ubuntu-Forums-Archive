### Post by Dr. Nick on 2006-10-16
type 

```
about:plugins
```into the firefox address bar.

You need a video player plugin to see the video, I have never had luck with totem which is usually the default. If you see totem mentioned in about : plugins then you can open synaptic and search "totem" and their may be a totem-mozilla or similar package installed, If you remove it then you can install another plugin. I personally have the best luck using mplayer and mplayer-mozilla. If you install them then recheck the plugins in firefox and see if they are thier.

Also you could look into the extension called "media player connectivity"
