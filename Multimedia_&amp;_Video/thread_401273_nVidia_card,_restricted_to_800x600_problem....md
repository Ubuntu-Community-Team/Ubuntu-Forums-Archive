### Post by tseliot on 2007-04-04
Maybe you disabled the driver when you reconfigured the xserver.

Try typing:
```
sudo nvidia-xconfig
```

then log out and press CTRL+ALT+Backspace

Should you have further problems with the Xserver type:
```
sudo nvidia-settings
```

and set the resolution from there

### Post by BuhSnarf on 2007-04-12
Hi,

This has now been fixed in the latest update to Fiesty, do a full upgrade. It seems that the MX400 is now included back in the nvidia-glx driver and not the legacy one. So if you do an update it'll install the glx and remove the legacy one. Just go back to restricted-drivers and reinstall and all works fine.

Hope that helps.
