> But when I boot up Ubuntu with the new card what will happen? Should I install some drivers in advance, or will it get automagically detected at boot up?

The opensource drivers ideally should be automatically installed.   Once you have booted up, you will get an option to add restricted drivers.  If you want to use Compiz-Fusion, you will have to install them.
