### Post by Palmyra on 2007-10-22
I took this from wiki.ubuntu.com:

    *

1.  Go to Applications &#8594; Add/Remove...

2.  Set Show: to All available applications

3.  Search for ubuntu-restricted-extras and install it. Note that there is also xubuntu-restricted-extras and kubuntu-restricted-extras.

I don't know how this works without enabling extra repositories, but give it a try.
