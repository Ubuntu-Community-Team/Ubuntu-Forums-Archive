### Post by aysiu on 2006-10-28
From [https://help.ubuntu.com/community/RestrictedFormats#head-1340337f2ca1d0c54900935468515ba7630fcc59](https://help.ubuntu.com/community/RestrictedFormats#head-1340337f2ca1d0c54900935468515ba7630fcc59)

>     *

      Kubuntu 5.10 uses two different methods for playing mp3s:
          o

            If you are using the version of amarok that comes by default with Kubuntu 5.10, install gstreamer0.8-mad.
          o

            If you are using the [WWW] akode audio library, install akode-mpeg.
          o

            To add mp3 support for use with K3b, install k3b-mp3.
