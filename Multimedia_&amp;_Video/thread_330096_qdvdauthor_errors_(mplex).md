### Post by Universal specialist on 2008-02-02
Hi everybody,

here are my experiences about "ERR: No .IFO files to process".

I get this error when qdvdauthor messes up the dvdauthor.xml file.  The following xml file causes the error:

```

<dvdauthor dest="/tmp/dvd" jumppad="yes" >
 <vmgm>
  <menus>
   <video format="pal" aspect="4:3" resolution="720x576" />
   <pgc entry="title" >
    <pre> { if ( g4 gt 0 ) button=g4; 
	   if ( g2 gt 0 )  { 
		jump title 1;		}
	} </pre>
    <vob file="/home/carsten/dvd/testmovie/Main Menu VMGM_menu.mpg" pause="inf" />
    <post> { g4=button; g2=0; jump vmgm menu 1; } </post>
   </pgc>
  </menus>
 </vmgm>
</dvdauthor>

```

As you see none of the movie files appear in vob-Tags. Instead a Main Menu (which I never wanted) strangely appears.

Restarting qdvdauthor, creating a completely new project and then adding the .mpeg2-files generated by the first try as movies generated a useable config file. The file (see below) only lists my videos now, no menu stuff. dvdauthor was happy with that.

```

<dvdauthor dest="/tmp/dvd" >
 <vmgm/>
 <titleset>
  <titles>
   <video format="pal" aspect="16:9" resolution="720x576" />
   <audio lang="en" />
   <pgc pause="0" >
    <vob file="/home/carsten/dvd/testmovie/1.mpeg2" pause="0" />
    <vob file="/home/carsten/dvd/testmovie/2.mpeg2" pause="0" />
    <vob file="/home/carsten/dvd/testmovie/3.mpeg2" pause="0" />
    <vob file="/home/carsten/dvd/testmovie/4.mpeg2" pause="0" />
    <vob file="/home/carsten/dvd/testmovie/5.mpeg2" pause="0" />
    <post> { g2 = 0; call vmgm menu entry title; } </post>
   </pgc>
  </titles>
 </titleset>
</dvdauthor>

```

Hope this helps,
Carsten

### Post by Universal specialist on 2008-02-02
I found an easier way to get a correct dvdauthor.xml out of qdvdauthor:

After having added your videos, check the dvdauthor.xml in the 'XML Out' tab. If your videos do not appear, drag them onto the Main menu tab. By doing so you will create buttons for the main menu for each video, which will make the vob entries appear in dvdauthor.xml.

You can even do this if you do not want a main menu (as I do). As long as you do not specify a background image, qdvdauthor will ask you if you want to create a no-menu video dvd. Answer yes here and you should be done.
