### Post by fede on 2007-04-13
To play flac in amarok I had to install libxine1-ffmpeg. Still, amarok does not search (go to a specific position) within flac files properly, so you might be better off using rythmbox. I would if I were running Ubuntu, but I'm on Kubuntu. It's not a really big deal, anyway, since I rarely need to search within files.

BTW, I'm using Feisty, I don't know if the packages are the same for Edgy.
