### Post by pytheas22 on 2008-01-17
check out [http://www.arsgeek.com/?p=439](http://www.arsgeek.com/?p=439) , which shows you how to turn .avi files into mpg's and burn them onto a DVD using tovid.  It also has some links to other resources on the forum for tovid that are useful for more advanced stuff.  I haven't done anything fancy myself, but it looks like tovid is an extremely powerful program that lets you do anything want with a DVD.

Also, k3b is a great CD and DVD burner for most uses, but I don't think it is able to take video files in a format like .avi and convert them into something that can be viewed on a standalone DVD player.  But for creating audio CD's and data CD's and DVD's it's great.
