### Post by rjbaker on 2007-10-11
Try using Audacity to record.  I had to also install the Gnome Alsa mixer, then check the record box in Front, then in Audacity, go to Edit/Preferences, and set the record selection to ALSA Front.  Both Audacity and the Gnome Alsa mixer can be installed from the Synaptics package manager.
