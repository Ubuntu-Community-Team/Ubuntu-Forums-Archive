### Post by yagood on 2007-06-20
I see that you use Dapper, right? Well, in Feisty it's much easier to get video/audio codecs installed, because if you try to use codec that you don't have, you're asked by Ubuntu if you want to install it. In previous Ubuntus there was just this scary error message. So you may want to consider upgrading to Feisty.

Anyway, you may try opening "Add/Remove Applications" from "Applications" main menu, selecting "Show all available applications" and search for "gstreamer plugins" and installing some of those.
