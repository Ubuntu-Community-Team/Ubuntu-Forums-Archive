### Post by Xer0 on 2006-02-15
There's a lot of information around on dual head setups, but nothing that covered my situation. Just took me hours to get this working, so hopefully by writing it out I'll save someone else some time...

I've got an nVidia 6600 GT, a 24" LCD and a 17" CRT. I wanted a separate session on each monitor, with different resolutions (1920x1200, 1152x864).

I set everything up as per a normal dual head setup with the distro included drivers, but the LCD would only go to 1152x864 with no option to set 1920x1200. By looking at /var/log/Xorg.0.log I saw that this mode was not being accepted, something along the lines of: Not using mode "1920x1200" (Reason).

In order to fix this, you need to add a ModeLine to your monitor definition as demonstrated in the [Xorg FAQ](http://wiki.x.org/wiki/FAQVideoModes). Problem is that the nVidia drivers don't display your pixel timings, so you need to revert back to the default "nv" driver (you backed up your xorg.conf, right?), grab the timings, switch back to your "nvidia" configuration and add an appropriate ModeLine.
