### Post by Zymous on 2008-03-08
Download Orbit Downloader from [http://www.orbitdownloader.com/]("http://www.orbitdownloader.com/") 

Install it using WINE

Go to [http://mark.mclaren.googlepages.com/iPlayer.html](http://mark.mclaren.googlepages.com/iPlayer.html) and save the iPlayerURL as a bookmarklet (this requires Firefox I think).

Go to the BBC iPlayer page that you want to download. Use iPlayerURL to capture the URL of the rtsp stream. (The Grab++ tool which grabs the rtsp stream URL under Windows, doesn't appear to work with Ubuntu). Copy the URL and paste it into Orbit Downloader. Start the download.

-- 
Zym
