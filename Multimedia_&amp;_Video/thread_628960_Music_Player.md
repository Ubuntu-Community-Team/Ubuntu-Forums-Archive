Synaptic is the package manager, which you use to install almost everything; System > Preferences > Synaptic Package Manager

Amarok and Rhythmbox are the two popular ones (well, two OF the popular ones); Amarok has a lot more bells and whistles (which I like) while Rhythmbox is simpler but still fully functional
