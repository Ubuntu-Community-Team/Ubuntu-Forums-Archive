### Post by Shinobi326 on 2007-05-12
I had to make sure in my xorg file that I put in the exact resolution I want, first (2560x1600).  So I manually put in the "2560x1600" for each video mode just to make sure.  I would try to make sure that whatever resolution you want you put it first.  In fact, I've seen some xorg files that only have the specific resolution you want.  You can always go into **System Tools / NVidia X Server Settings** and switch it to something different if you want to change it from your loaded default.
