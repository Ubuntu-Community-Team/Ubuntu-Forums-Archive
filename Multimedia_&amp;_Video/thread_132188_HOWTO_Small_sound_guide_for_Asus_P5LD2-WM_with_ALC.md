Now back to the topic.
Here on my new pc there was no sound after the install so i tried to search for a solution but nothing help me, even had to reinstall ubuntu after i tried to install the latest realtek driver. 
By default the hda intel (Alsa Mixer) is selected and that's needs to be changed. Right click on the speaker icon next to the date & time on the gnome bar. Select preference -> select Realtek  ALC882 (OSS Mixer) and close.

Now the sound should work.
