### Post by SZF2001 on 2006-12-02
I found out how and will post for future reference for me and anyone else interested.

Download and extract the latest firmware, which can be found at the [Anything But iPod Forum](http://www.anythingbutipod.com/forum/forumdisplay.php?f=51). This is a rar file, so if you haven't installed unrar yet, you should.

In the file, there should be a file with a .mi4 extension (example: PP5022.mi4). Extract it and put it in Sanra's root folder (the very very very first folder you come to in your Sansa, the one leading to other folders like MUSIC and PICTURE)

Disconnect the player, which should refresh itself, and say it's putting in some firmware. Congrats if success!

Also, to not frusterate yourself again, change the USB mode, because in the firmware upgrade it switches back to the other one. Enjoy.
