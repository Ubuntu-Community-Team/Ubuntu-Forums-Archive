### Post by bsmith1051 on 2008-03-05
I miss how easy it was in Windows to install Winamp, then right-click songs and select "Add to queue".  Is there anything comparable for Ubuntu?  I've tried several different MP3 players (including the default Totem Movie Player as well as VLC) but didn't see any option for desktop integration.

Even if there were a program that allowed you to drag songs to the icon(?) would be fine.

SOLUTION: Installing the 'nautilus-actions' package, then adding the 'audacious' script, was the solution I was looking for!  And Audacious is a Winamp-lookalike to boot.

### Post by soga_genzo on 2008-03-05
I think [Audacious]("http://audacious-media-player.org/index.php?title=Main_Page") supports Drag-and-Drop, at least you can drop files on its application window. As for embedding an "Enqueue" action in Nautilus' context menu, you'd probably be best off using [nautilus-actions]("http://www.grumz.net/?q=taxonomy/term/2/9"). There's even an ["Enqueue in audacious" script]("http://www.grumz.net/?q=node/204"). Both Audacious and nautilus-actions are in the repositories.
