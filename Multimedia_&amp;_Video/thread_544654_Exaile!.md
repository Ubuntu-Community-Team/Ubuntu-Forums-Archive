### Post by vinnn on 2007-09-06
Although Exaile has a lot of potential as GTK/Gstreamer based Amarok style app it is a bit buggy, especially the version in the Ubuntu repos which is getting on a bit.

The problem you're getting is because after version 0.2.10 Amazon's sites changed in a way that Exaile had to release a patch to query Amazon's sites in the required way..

My suggestion is to add Exaile's Ubuntu repository to your repo list as shown at [http://www.exaile.org/downloads](http://www.exaile.org/downloads) and update, this will keep your version of Exaile way ahead of the Ubuntu repos, and fix the album covers problem.
