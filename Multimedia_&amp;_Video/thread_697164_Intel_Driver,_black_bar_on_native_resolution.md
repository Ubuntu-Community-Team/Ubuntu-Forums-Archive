### Post by bowsercake on 2008-02-26
I am so excited, I just fixed this problem right now!

I downloaded the Windows XP inf file from the ASUS website and I used the 'add' feature in Settings > Screens and Graphics.  It detected the same sync numbers that I had been inputting before, however, this time it gave me the option to run it at 75Hz instead of the 60Hz that I could only get before.  What do you know, 75Hz displays it perfectly. The months of anguish are all over now. Thanks for the help guys.
