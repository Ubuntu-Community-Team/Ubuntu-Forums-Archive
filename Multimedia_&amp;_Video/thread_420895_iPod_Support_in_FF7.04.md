### Post by BoneKracker on 2007-04-24
I don't know if you can do it from the live cd or not, but you can easily play the music that's on your iPod.

If you're using the main version of Ubuntu (the Gnome-based one), you can do it with the program called Rhythmbox.

If you're using Kubuntu (the KDE-based version), you can do it with Amarok.

Another Gnome-based application that is up-and-coming, but still has some bugs to work out, is called Banshee.  It's also got the ability to import, export and sync with the iPod.  In the meantime, I've the best luck doing that sort of thing with an older application called GtkPod

### Post by reclusivemonkey on 2007-04-24
From my experience with my girlfriend's iPod (Sixth Generation 4Gb Nano) I found the following works on Feisty;

[LIST]
[*]Automatic iPod detection and mounting on the Desktop
[*]Ejecting on right click from the Desktop
[*]iPod detected and plays from Rhythmbox
[*]GPixPod will transfer Pictures to the iPod
[*]GtkPod will transfer songs to the iPod
[*]Evolution (with the relevant plugins installed) will copy Contacts and Calendars to the iPod
[/LIST]
