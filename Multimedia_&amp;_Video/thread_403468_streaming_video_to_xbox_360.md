### Post by ronmonki on 2007-04-09
Hi m8,

my solution to stream to xbox 360 is to install windows media center in vmware or kvm

set up samba share from ubuntu host to windows in vm, map network drives through a login script in windows, more detail can be found on google about the login script to acess samba share under xbox media center.

I also use twonkymedia server for my music ( it will also play any wmv video to xbox too, but I have not been able to get it to play non wmv content i believe that it can be achieved but you have to link twonky with vlc via a transcond or something, more detial can be found on the twonky forums.

good luck and if you find an easier way pls let me know....
