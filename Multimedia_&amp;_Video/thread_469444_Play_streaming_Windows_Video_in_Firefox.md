=======================================
Play streaming Windows Video in Firefox (Windows Media Player) in linux.
 
*UPDATED, LINKS WORK AGAIN*
The below guide is for MEPIS Linux although it might work in Ubuntu since I see the downloads are Ubuntu.

First you must install the following for Kaffiene.
Click on the first "here" and then download the following files (Do NOT Install yet)...

libdvdcss2_1.2.8-1unofficialubuntu4_i386.deb
w32codecs_20060611-0unofficialubuntu1_i386.deb

Then after that open your Synaptic Package Manager do a search for "mplayer" without quotes and install the package that says mplayer (not mozilla-mplayer) Then install both of the files that you downloaded (simply buy opening them and then clicking install). After that go back to Synaptic and then do a search for "mplayerplug-in".
Install "mozilla-mplayer".
Now go to [http://www.cnn.com/US/](http://www.cnn.com/US/) and click on a video to test it.
