Hi folks ...

I had the same problem and solves it with the following procedure:

1) Verify if Enlightened Sound Daemon (esound) Package is installed. You can use Synaptic Package Manager to make a search with the key word "esound".

2) If  Enlightened Sound Daemon (esound) package is not installed ... please.. install the package.

3) Logout or reboot the system after the installation for esound where succeful.

4) Go to System/Preferences/Sound menu and select **ESD-Enlightened Sound Daemon** for the Sound Events options (the first one) and press the TEST button. You must to get a sound test.

5) Select the Sound Tab and be sure to activate **Enable software sound mixing (ESD) and Play System Sounds** options.

7) Configure system Sounds as you wish.

6) Reboot the system.

7) You are rock now !!!!

I hope to solve this issue with this procedure..... and sorry for my horrible english.
