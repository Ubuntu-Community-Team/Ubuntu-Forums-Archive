### Post by npc100 on 2007-10-28
Hi All,

a couple of years ago I wrote a small java application that allows you to enter (and store) a list of keywords or phrases of things that you are interested in. It will then search the MythTV EPG and find things that you might like to watch and allow you to set them to record.

You can also set it up to automatically do this from a cron job.

You can blacklist individual shows, or entire channels.

Anyway, I dusted it off this weekend, added a bunch of functionality, and its starting to become quite useful (well, it found a bunch of stuff that I probably wouldn't have found through manually scanning the listings!)

So I've put it up online. If you think it might be useful, you can get it here:  [http://homepages.nildram.co.uk/~chinnery]("http://homepages.nildram.co.uk/~chinnery")

Feel free to contact me with any bug reports, ideas for improvement, etc. etc.

cheers
