### Post by nick_h on 2007-11-28
You need to edit your /etc/X11/xorg.conf and change the Driver line.  Set it to "nv" for the free driver and "nvidia" for the restricted one.

### Post by nick_h on 2007-11-29
You are using the "nv" driver at the moment.  You have two Device sections each containing a Driver line.

I suggest you make a backup:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```
before making any changes.

### Post by realzippy on 2007-11-29
I also had trouble activating the nvidia-driver with the restricted drivers manager.
"envy" is a script by Alberto milone,it installs the nvidia-driver and sets up the xorg.conf correctly,
even with an older Geforce graphicscard:

[http://albertomilone.com/ubuntu/nvidia/scripts/ubuntu/envy_0.9.9-0ubuntu1_all.deb](http://albertomilone.com/ubuntu/nvidia/scripts/ubuntu/envy_0.9.9-0ubuntu1_all.deb)

After installing you can start it from the command line by:
 sudo envy -t

or use the GUI in Applications/System Tools.
After restart open a terminal,type:

sudo nvidia-settings

select desired refreshrate and resolution,apply AND save to X Configuration File,

what creates a Metamodeline like e.g. :
Option		"metamodes"	"1280x1024_85 +0+0; 1280x1024 +0+0; 1280x960 +0+0; 1152x864 +0+0; 1024x768 +0+0; 832x624 +0+0; 800x600 +0+0;

in your xorg.conf`s "screen" section...
