### Post by nick_h on 2007-11-26
To install mplayer type:
```
sudo apt-get update
sudo apt-get install mplayer
```

and to install the mplayer plugin for mozilla type:
```
sudo apt-get install mozilla-mplayer
```

Alternatively you can install these packages using Synaptic (System->Administration->Synaptic Package Manager)
