I have 2 related problems. I am a new Linux convert.

I think this means that dual monitors is working but the resolution Ubuntu has set for the first is not compatible with the second. In the System > Preferences > Screen Resolution utility, the supposed refresh rate is 0Hz. Seems as if Ubuntu setup or Xorg hasn't configured my monitors correctly. Is there a gui tool to do this for each monitor? I suppose i would have to change clone mode to. Any pointers would be great thanks!
