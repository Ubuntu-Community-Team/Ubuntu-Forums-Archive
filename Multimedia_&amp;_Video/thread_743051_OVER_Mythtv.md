### Post by warp99 on 2008-04-02
You can window the MythTV front end to what ever size you want, see the man pages for mythfrontend, then run the scripts in the terminal window as "always on top". You can also use use alt-tab and bring the terminal window to the front even in full screen mode, but the terminal window needs to be opened before the MythTV front end.

Edit:

On full screen mode that would only be for the main and sub-menus, not for Live TV.
