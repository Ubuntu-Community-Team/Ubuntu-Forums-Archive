### Post by shirshasin on 2007-12-10
Adobe Flash Player 9 Update 3, version identifier 9.0.115.0 was made available for Linux at the same time as Mac and Windows versions. It supports the new exciting H.264 standard.

Download the .tar.gz from the following link [http://www.adobe.com/shockwave/download/download.cgi?P1_Prod_Version=ShockwaveFlash&P2_Platform=Linux](http://www.adobe.com/shockwave/download/download.cgi?P1_Prod_Version=ShockwaveFlash&P2_Platform=Linux)

Installation instructions
.tar.gz installation

   1. Click the "Download .tar.gz" link. A dialog box will appear asking you where to save the file.
   2. Save the .tar.gz file to your desktop and wait for the file to download completely. Close your browser.
   3. Unpackage the file. A directory called install_flash_player_9_linux will be created.
   4. In terminal, navigate to this directory and type ./flashplayer-installer to run the installer. Click Enter. The installer will instruct you to shut down your browser(s).
   5. Once the installation is complete, the plug-in will be installed in your Mozilla browser. To verify, launch Firefox and check the link [http://kb.adobe.com/selfservice/viewContent.do?externalId=tn_15507&sliceId=1](http://kb.adobe.com/selfservice/viewContent.do?externalId=tn_15507&sliceId=1) , it will tell you about your flash player version number.
