### Post by ssam on 2008-02-24
You dont need to do 32bit firefox anymore. there are clever compatibility layers that get installed automatically when you install the flash package. 

Applications -> Add/Remove...

set Show to 'all available applications' 

search for flash, tick the box, and click apply
search for jave, tick the box and click apply 

or just install the ubuntu-restricted-extras package, which gives you flash, java and all the codecs
