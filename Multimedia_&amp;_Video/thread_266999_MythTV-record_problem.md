There are ways to interface mythtv with satellite boxes.  Some work via a serial connection (I have comcast digital cable which I channel change via a serial connection), or just about any other kind of set top box can be set via an IR blaster (programmable remote control your computer can interface with, sold seperately).  As long as you manage to flub the video source section of the mythtv setup, there is a manual scheduling option in mythtv for recording.  Just try and get something manually setup in the video source section.
