### Post by ausmuso on 2007-06-30
I wanted to add Skype to my fresh Feisty install.
Wanting to get the very latest I downloaded the Skype 1.4 beta deb package from Skype's website. I found I also had to download some QT4 libraries before the deb file would install on my machine.
    Once I had installed the package I kept getting error messages whenever I tried to make a connection: there appeared to be a sound playback problem. I suspected there might have been yet another library missing.
    I uninstalled Skype 1.4 and then used Synaptic to download Skype 1.3 from the Medibuntu repository. Sure enough I was told I also had to get the librte1 package on which it was dependent. Skype 1.3 then installed without a murmur and it runs perfectly (I also had to install the Gnome ALSAmixer before I could access all the controls to get my microphone volume right).
   QUESTION: Skype 1.3 seems to do everything I want to do with it. Is it really worthwhile to upgrade to 1.4?

Ausmuso
