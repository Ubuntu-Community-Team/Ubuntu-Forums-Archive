### Post by jim777 on 2006-12-27
I like to listen to air traffic control on liveatc.net and in Ubuntu the default application for m3u is
movie player which is no good for those streams.    So I installed XMMS  which is on the Edgy DVD I have.

Then I used a text editor to create a tiny text file that I renamed to sample.m3u instead of txt.    After that it was a simple matter to right click on the fake m3u file and change the default application to XMMS instead of Movie player.    Now I can go to the liveatc site and using Firefox 2.0 click on one of the feeds and XMMS starts up automatically and works fine.

P.S.   Properties    >> Open with        then select XMMS radio button thing.
