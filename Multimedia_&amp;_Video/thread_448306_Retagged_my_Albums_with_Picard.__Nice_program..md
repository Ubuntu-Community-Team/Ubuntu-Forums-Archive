### Post by Gen2ly on 2007-05-19
I compiled the new Picard 0.9.0 but Picard 0.7.2 is in the repos.  Its a nice program that uses the MusicBrainz ( an non-profit music community ) metadatabase.  Here's the tutorial if you ever like to try it:

[Weblink]("http://musicbrainz.org/doc/HowToTagFilesWithPicard")

Its a little different for 0.9.0, if you choose it.  It's was rebuilt with the qt library - works fine in Gnome (older PC's may see a slowdown.) First "Add Folder" and select your music directory, then "Cluster" them together.

Click the Album You want looked up on the left box and and select "Lookup Metadata".  If you click on the bottom Lookup it will show you whats contained in the database (on the website) for that album.  If it looks correct in the right box select that album then click "Save."

I put this here because I had issues using easytag.
