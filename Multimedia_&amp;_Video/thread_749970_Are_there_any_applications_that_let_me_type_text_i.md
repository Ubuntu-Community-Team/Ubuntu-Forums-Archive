### Post by Sharp on 2008-04-13
The above post probably solves this with Gimp but since you were trying with Inkscape:

Type your text with the text tool, create a circle with the ellipse tool (hold down Ctrl to get a clean circle). Then just select both the circle and the text and go to Text -> Put on Path.
