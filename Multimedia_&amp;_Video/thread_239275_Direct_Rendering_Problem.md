### Post by foolsh on 2006-09-03
this section should look like this
```
Section "Device"
Identifier "Intel Corporation Mobile 915GM/GMS/910GML Express Graphics Controller"
Driver "i810"
BusID "PCI:0:2:0"
EndSection

```

run this in a terminal
```
sudo gedit /etc/X11/xorg.conf
```
then scroll down and find the section above and change 
```
Driver "vesa"
```
to
```
Driver "i810"
```
then save the file

then press control + alt + backspace at the same time, this will restart the xserver without a reboot. save all other open documents first 

let me know how it works out
