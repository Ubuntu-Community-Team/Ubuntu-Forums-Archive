### Post by Audi.RS4 on 2006-10-28
I had the same problem too. I finally figured out the settings though and fixed it. Not sure why they default it the way they do but I'm sure they had there reasons. The steps are listed below to get Alt-Tab to work like it should. (or as I feel it should)

1. Open up the Beryl Settings Manager

   System->Preferences->Beryl Settings Manager

2. In the list select 'Application Switcher'

3. Click the Keyboard tab.

4. For the first 'Next Window' check the 'Control' box

5  For the second 'Next Window' uncheck the 'Control' box

6. Do the same for 'Prev window'

Enjoy!
