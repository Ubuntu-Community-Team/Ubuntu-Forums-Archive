### Post by jonnymccullagh on 2007-02-08
I don't think that would work as your dvd player is unlikely to have the codecs required for decoding the AVI file.
I think there is a bit of a shortage of programs like this but I did find this:
[http://www.kde-apps.org/content/show.php?content=36351](http://www.kde-apps.org/content/show.php?content=36351)
You would have to install Kommander(file manager) first, then install the kavi2dvd script and hopefully right-clicking should allow you to convert it to MPEG2 for burning.
DVDs use MPEG2 format video files.
I haven't tried this myself - I just took a look as I saw your post,

### Post by wxnker on 2007-02-18
I would suggest that you try DeVeDe if you want do do this the easy way without using several tools and the terminal.

Version 2.1 can be installed through the Synaptic Manager it demands that you install some other tools though. e.g Mplayer, mencoder etc. I guess the install of DeVeDe will take care of all this but I'm not sure since I had all dependencies installed before discovering DeVeDe.

When installed DeVeDe will do it all and if you like, it can create an ISO that you can burn with k3b.

I would suggest that you upgrade to version 2.11 to get new functions like subtitle support.
Get the source here: [http://www.rastersoft.com/programas/devede.html](http://www.rastersoft.com/programas/devede.html)

One thing though. I _don't think_ DeVeDe supports menus.
