### Post by ron999 on 2008-02-27
Hi
It depends which program you are using to rip to mp3.

With SoundJuicer use Edit > Preferences > Format > Edit Profile > Edit

There you'll see the instruction.

Mine is set to use lame encoder and the setting is vbr-quality=6.
You can alter the settings or add new profiles.

To see all the possible lame settings open the monitor and type **lame --longhelp**
