### Post by mystiq23 on 2007-08-30
Im not sure all about this, but some time ago, the same think has happened to me. I downloaded some film (about 700 MB) and the whole thing about the movie was one second short clip, that I need some super-duper codec, which can be downloaded somewhere on internet. After installation, my Win detected trojan. 

So I have searched forums and discovered this: the movie can be in the file, or not. The "codec" just rewrites the head of the file with its own, which prevents it from playing.Only if you download their "codec", it will fix the head of the file (and give you a nice present - trojan :). Some people advised to fix the head of .avi manually in some hex editor, but It was too much for me, so I cannot say if this works or not. 

Anyway, google a little, and youll find out more about this crap.
