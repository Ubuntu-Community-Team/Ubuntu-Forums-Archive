### Post by Hexydes on 2007-06-03
Well, for anyone that cares, I swapped my firewire cable for a USB cable, and Banshee found the iPod right away. Additionally, even though the iPod was showing up in file explorer before, it was called "Hexydes's iPod". Now, when I hook it in via USB, it shows up as "Apple iPod Music Player: Hexydes's iPod". So it looks like a) Ubuntu is detecting devices connected via USB differently than firewire, and b) something within that process is making it so that Banshee is not able to detect the device.

I guess I'm going to use the USB cable now (it had been in my car, so that I could charge my iPod at work if I forgot to charge it the night before, now I won't be able to do that), but that really isn't a real "solution", just a work-around. :(
