### Post by vik_777 on 2007-11-10
I'm having similar problems, on an intel 945GM.

Try using the Nov 1st release of the xserver if you use the intel xserver from here:
[http://people.ubuntu.com/~bryce/Testing/intel-Gutsy/](http://people.ubuntu.com/~bryce/Testing/intel-Gutsy/)
and run blender with compiz turned off. 

This seems to at least fix Blender, but I'm still having problems with Direct Rendering via wine, so it's not a *fix* but a *workaround* for now.
