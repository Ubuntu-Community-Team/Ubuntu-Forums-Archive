### Post by bernardfrancois on 2006-01-27
If you can't find any Linux drivers for your sound card, then you can try to find out which chipset is used on the card. A driver for another card which uses the same chipset might work as well (probably you'll have to specify values like IO port, DMA channel and IRQ number, which can be found in bios).
