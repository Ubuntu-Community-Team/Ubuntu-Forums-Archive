### Post by da_Seeb on 2007-12-05
Hello,

i want to present you a project of mine. It's an audio player I have written over the past years....

Cactus Jukebox is a mp3 player with integrated databse like iTunes or Amarok/Rhythmbox. Main focus during development was on speed and easy straight forward usability. 


Feature list:
+ Fast database
+ Audio CD ripper
+ Upload/synchronize to mobile players
+ Plattform independent(GTK, GTK2, win32)
+ Browse your music collection by artist, albums and titles
+ Save and load m3u playlists
+ ID3 tag support
+ Easy tagging of mp3-files

Above all Amarok for example was a very annoying experience... It is feature rich but dead slow. When you want to play just one song you have to wait 30sec for amarok being started. That's where I began creating my own application...

I would be very happy if you give it a try and post your feedback/bug reports here( or email me).

Homepage:
[http://cactus.hey-you-freaks.de](http://cactus.hey-you-freaks.de)
(.deb packages available)
You can also try beta release. Though it's named beta it is quite stable, but needs some more user feedback to become official release


If you have some programming knowledge, want to implement some missing features I would be glad if you want to take part in development. I'm always happy getting some help from other people!

Sebastian  :guitar:
