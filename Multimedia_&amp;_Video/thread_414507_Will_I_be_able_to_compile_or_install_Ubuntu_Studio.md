### Post by mojoman on 2007-04-20
I think the Ubuntu Studio released this April is going to be a derative of Ubuntu based on Feisty, i.e. an stand-alone operative system and not an application. In that case, no, you won't be able to run in on Dapper. You would either have to (a) switch to Feisty (i.e. use Ubuntu Studio, in which case all other Feisty packages should be available to you) or (b) Dual boot with Dapper and Ubuntu Studio. At least that's what I could gleam from this page.

[https://wiki.ubuntu.com/UbuntuStudio](https://wiki.ubuntu.com/UbuntuStudio)

At also contains a link for setting up Ubuntu studio for Dapper but I have no idea if it works. Ubuntu Studio looks interesting though...

### Post by mojoman on 2007-04-20
Yes, it's a set of applications but so is for example Xubuntu. As far as I understood, Ubuntu Studio is an OS in it's own right, aimed toward sound and video editing meaning it comes complete with a kernel and everything. Presumably, it will have out-of-the-box support for multiple soundcards and other stuff one might expect it's users will have. However, it is based on Feisty and I suspect that every package available for Ubuntu Studio will be available for other Feisty distro though I might be wrong here. Look at these threads:

[http://ubuntuforums.org/showthread.php?t=343193&highlight=ubuntu+studio](http://ubuntuforums.org/showthread.php?t=343193&highlight=ubuntu+studio)

[http://ubuntuforums.org/showthread.php?t=401620&highlight=ubuntu+studio](http://ubuntuforums.org/showthread.php?t=401620&highlight=ubuntu+studio)
