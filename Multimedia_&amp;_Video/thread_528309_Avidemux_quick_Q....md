### Post by RomeReactor on 2007-08-17
Hi. Are you running Feisty? if you're running an older version of Ubuntu, you need to enable the [multiverse repositories]("https://help.ubuntu.com/community/Repositories/Ubuntu"). If you **are** running Feisty, then the repositories should already be enabled; try:
```
sudo apt-get install avidemux
```
If for some reason that doesn't work, get AviDemux from [here]("http://packages.ubuntu.com/feisty/graphics/avidemux").
