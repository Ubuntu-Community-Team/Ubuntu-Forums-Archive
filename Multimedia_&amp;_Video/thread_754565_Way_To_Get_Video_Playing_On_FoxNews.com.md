### Post by teeteebahbah on 2008-04-13
GreaseMonkey Extension NO LONGER allows you to view News Videos on FoxNews.com.  However, the "User Agent Swicther" Extension does.

I got it from the Mozilla main website for FireFox.  It allows you to temporarily view the Video on FoxNews under the guise or  belief that you are using Vista even though you are NOT.  Once you install it in FireFox, select the Tools drop down window.  Select U.A.S. Ext. and select IE7 Vista

(Note:  Nothing has changed w/Grease Monkey.  FoxNews apparently made some changes to their website recently and left us Linux users out in proverbial dust, so to speak. )
