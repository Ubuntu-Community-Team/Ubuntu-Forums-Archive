### Post by alphacrow on 2008-03-03
A GeForce 6200 should work just fine with Ubuntu - as long as you install the correct drivers.   I believe the 6200 is no longer supported by the newer drivers so you'll likely need to install the legacy drivers (or just download it from nvidia's site)

The 6200 is a fairly old card, so I wouldn't expect it to perform very well with the latest 3D games.
