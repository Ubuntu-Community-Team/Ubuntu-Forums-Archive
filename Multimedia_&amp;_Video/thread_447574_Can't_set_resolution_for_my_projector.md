I have registered here to detail the solution I worked out because I couldn't find it anywhere else...

I have a fresh kubuntu install with a Nvidia MX 440 and a Benq W100 projector connected via DVI-D. Watching the xorg logs it showed that any ModeLines or resolutions I tried over 800x600 were not passing the validation tests despite the hardware being capable (I know this as it worked under windows). The solution... disable the validation tests.

In my "Screen" section I have the following (may be over kill):

    Option         "UseEDID" "False"
    Option         "FlatPanelProperties" "Scaling = native"
    Option         "ExactModeTimingsDVI" "true"
    Option         "ModeValidation" "NoDFPNativeResolutionCheck, AllowNon60HzDFPModes, NoEdidMaxPClkCheck, NoMaxPClkCheck, NoHorizSyncCheck, NoVertRefreshCheck, NoEdidDFPMaxSizeCheck, NoEdidModes"

Once you have that you can set any resolution (even if the hardware doesn't support it!), I prefer 720p on this projector so I set:

    Modes     "1280x720_60"

That works fine, no need for a ModeLine.

I hope this helps.
