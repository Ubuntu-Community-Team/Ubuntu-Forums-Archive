### Post by kevinatkins on 2007-11-15
Hi,

I'm not overly familiar with using an iPod.... but have you tried Exaile music player? It's a bit like Amarok, for Gnome - it isn't as feature-laden as Amarok, but in many ways that's a good thing because it isn't quite as overwhelming! If you try it, you'll need to enable the iPod driver plugin (Edit -> Plugins)

Amarok may be missing some of your music because it's in a format that by default Amarok cannot decode. It's probably using the xine engine by default, so you might need to add some extra codecs to the xine engine....
