### Post by fourchannel on 2006-08-02
```
sudo apt-get install gtkpod
``` 
Launch GTKpod, plug in your iPod, set up a few options, and you should be rockin' in no time. **just make sure to unmount your ipod before unplugging the device from the computer**

in GTKpod there is an option to unmount the iPod but I forgot where. You can also open File Browser and go to "Computer" and right click on your iPod and select "eject" or "umount" -- I can never remember which one it is.
