I have an Nvidia Quadro 3450 and two monitors, one 19" Dell and one 20" Dell, and have TwinView working.  However, the setup is nearly unusable.  The Nvidia driver is telling the OS that there is one screen 2560x1024.  This means that maximized windows bridge over the two abutting monitor edges.  Often, windows open in the center which means they're obscured by the inside edges of the monitors.  

[IMG]http://www.trilug.org/~jjtuttle/snapshot2.png[/IMG]

I've used nvidia-settings to configure Twinview, but using different x screens with or without xinerama doesn't work.  You can see the attached screen capture.  Imagine that there are two black monitor edges down the center of the screen.  Pretty annoying.  

I've also tried to manually add metamodes, but no luck.

### Post by spin2cool on 2007-04-12
Add these two lines to your "Device" section:

```
Option "TwinView" "True"
Option "TwinViewOrientation" "RightOf" 
```
