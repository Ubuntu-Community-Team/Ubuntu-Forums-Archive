I use SPDIF for sound.  In Windows, I used a program called AC3Filter which means that whenever I played a video file with (for example) 5.1 AAC sound, it would be trancoded to AC3 on the fly, and played back  as 5.1 AC3 over SPDIF.  ffdshow also had such functionality.  It essentially allowed you to get 5.1 from files which were not AC3/DTS.

Can this be done in Linux?  I'm sure ffdshow gets it's code mainly from ffmpeg/libavcodec anyway, so...
I know such a process is lossy, but it's the best we can really get over SPDIF.  This will also be useful when the DD+ and TrueHD decoders are added to mplayer SVN.
