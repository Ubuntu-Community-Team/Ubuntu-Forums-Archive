### Post by Yoooder on 2007-03-25
I have a single PC that I've been using as a combination Desktop/MythTV box for several months, and it has worked well--however it's had some downfalls.  The biggest was that I was using my TV-out port on my Hauppauge 350 card for MythTV, and it played well but didn't show the OSD of MythTV and also wouldn't do clean frame-stepping (it was really jumpy).

Here is what I am hoping to do, and am looking for suggestions (whether it be to my plans, or suggested technology to be used).

Available Outputs:
 - I have a GeForce 6600 video card with a standard d-sub 15 monitor plug, a DVI plug, and S-Video.
 - I also have my TV-tuner, which has VGA (RCA connector) video out, as well as S-Video out.

Available Screens:
 - 1x LCD Monitor
 - 1x HDTV (with inputs that match all of my outputs, however I would need a DVI-HDMI convertor to use my Graphics Card's DVI port).

I wish to maintain my machine as a combination Desktop/MythTV box, and specifically be able to use my small LCD screen as normal for browsing & etc... while using my HDTV as a Dedicated MythTV screen.

Ideally I will only be using only my Video card for output (as my tuner's output has the aformentioned issues with not displaying an OSD and also the ugly frame-stepping).

I was hoping to be able to setup possibly a second X session on my TV, so it could run entirely seperate from my GNOME desktop--however I wasn't sure that this would be possible (as I'd need both to run at the same time, and be able to switch my input devices between the tow).

I've looked at the Dual Monitor how-to, and am under the impressions Xinerama is the way I'll want to go, as I don't want the cloned-view that TwinView supports.

Is using a DVI --> HDMI convertor a bad idea?

Any guidance would be appreciated!

### Post by Yoooder on 2007-03-26
Update:  I've found that by using the proprietary nVidia drivers, I have been able to configure most of what I need with ease.

I'm currently running Feisty, and after enabling the nvidia driver through the restricted-manager I got an nVidia tool in my System Menu (in GNOME).  By playing with this I was able to setup each screen as an independant desktop, and am able to watch my movies on my big screen while still using my little screen for browsing.

The only thing left to do is figure out why my big screen isn't allowing me to put the res at 1280, and to get MythTV up and running again!
