### Post by Gremlinzzz on 2007-09-20
I use mplayer plugin if you install it you have to remove other plugins. like totem
cd /usr/lib/firefox/plugins/
sudo ln -s /usr/lib/mozilla/plugins/mplayerplug-in-dvx.xpt
sudo ln -s /usr/lib/mozilla/plugins/mplayerplug-in-dvx.so
