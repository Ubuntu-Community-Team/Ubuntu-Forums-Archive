### Post by Jovec on 2007-02-16
> **kinpower said:**
> Also I was wandering, how do I change the default music player from "movie player"

Right-click on any file of the specific file type you want to open with a new appliaction (i.e. any .mp3 file to change it for all mp3s).  Choose properties, then the Opens With tab, and from there, you can pick which application you want to use.  You can do this for all media file formats you want to change.
