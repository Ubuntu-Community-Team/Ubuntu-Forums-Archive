### Post by ColJep on 2007-11-30
I have a Sansa e280e to which I have been adding mp3 files ripped from my own CD's with Grip and setting id3 tags with EasyTag. All was fine until recently. I think there has been an upgrade to EasyTag and now any new CD's are not recognized by my Sansa.

My suspicion is that Easy Tag now uses ID3v2.4 as default rather than v2.3 and the Sansa does not recognize v2.4 tags.
Can anyone confirm this or give me some ideas? I have been unable to upgrade the Sansa firmware, I think because I only have access to a Win200 machine not XP.

My Sansa firmware is European version is 01.02.16E

Colin

### Post by foolsh on 2007-11-30
also there is a way to update the firmware with out windows simply copy the firware.mi4 or pp5020.mi4 file to the root directory of your sansa and reboot.
here is a link to some firmware images [sansa firmwares]("http://files.zefie.com/PMP/sansa/e200/firmware/")
note: do not use the latest beta 1.03.07 it does not include a usb setting in the settings menu so you will not be able to select MSC mode and in effect will not be able to access your sansa from linux.
