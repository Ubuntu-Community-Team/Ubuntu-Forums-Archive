When you go to watch video's at the Fox News website you get the following error message:"Windows 2000, Windows XP, Windows XP Media Center 2005, Mac OSX 10.3 or Mac OSX 10.4 is required to view this page"
If you install the User Agent Switcher and set it to IE 7\Vista then the video's seem to work OK in FF3 on 8.04 Hardy Beta. Make sure your flash plugin is up to date.
