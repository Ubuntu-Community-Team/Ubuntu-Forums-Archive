### Post by Existentialist on 2008-03-15
I know all of the Samsung monitors I have used have can be set to automatically set the input based on which input it sees being used, or manually.  You might try checking the menu on the monitor to make sure it is not manually set to use the vga input.  Also try manually setting it to digital in if you have not tried this already.
