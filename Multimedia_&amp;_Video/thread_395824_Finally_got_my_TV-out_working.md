### Post by Steve H on 2007-03-28
Finally I have managed to get my monitors set up in a way that is both useful and stable. I followed this page

[https://help.ubuntu.com/community/NvidiaTVOut](https://help.ubuntu.com/community/NvidiaTVOut)

I had a few mishaps (notably with the wrong PCI address), but once I paid attention to the line about commenting out the "ConnectedMonitors" line it all works beautifully. I can't drag from one screen to another, but at least I can watch avi's and stuff on my TV-out while using my CRT monitor for surfing. I have just set up the relevant mplayer icon on the TV-out desktop (as that is the only thing i use the TV-out for!, watching movies etc).

I did have lots of trouble with TwinView and Xinerama setups as they do allow dragging, but seemed to cause other problems, like causing my Primary Monitor (CRT) to go "Out of Sync" and cause a black screen. Also they would open windows on the wrong desktop, which is fine if i have both monitors running (but a pain if i was watching telly and couldn't see where the window had gone!!)

But at least it is all working now......woo hoo!!
