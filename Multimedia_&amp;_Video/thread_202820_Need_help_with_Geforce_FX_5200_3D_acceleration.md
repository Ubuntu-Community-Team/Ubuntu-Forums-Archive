### Post by sethmahoney on 2006-06-25
Change this section:

```

Section "Device"
    Identifier     "NVIDIA Corporation NV34 [GeForce FX 5200]"
    Driver         "nvidia"
EndSection

```

to say:

```

Section "Device"
    Identifier     "NVIDIA Corporation NV34 [GeForce FX 5200]"
    Driver         "nvidia"
    Option        "RenderAccel"     "true"
EndSection

```

And then reboot or hit control+alt+backspace.
