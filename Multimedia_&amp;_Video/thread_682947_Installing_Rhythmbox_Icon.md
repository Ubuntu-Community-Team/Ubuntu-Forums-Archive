### Post by pencilcheck on 2008-01-31
actually, if you are using custom icons in .icons folder under your home directory, you should look over them too.
It's important to replace any icons appearing in any folder which has the icon like 22x22 or scalable.

Also, if you want to change the icons on the panels, you need to go to either, /usr/share/pixmaps or /usr/share/rhythmbox/icons/hicolor or something like that.

If you found that the icon is not display correctly, check those locations and make sure everyicon is replaced properly. 
I think this should resolve your problem :)
