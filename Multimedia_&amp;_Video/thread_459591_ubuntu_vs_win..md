### Post by martinand on 2007-08-30
I also had prioblems with Gopchop  closing when I opened an MPEG - but I later found that I had to set the Video Output Driver (in preferences) to x11 and it worked fine - I also noticed that even when setting this driver to X11 where I opened an mpeg had failed to stay open previously it would fail again until I deleted the old file that gopchop had previously created when it first opened the mpeg  - it creates a file with the mpeg name plus a .gop extension.

Hope this helps.
