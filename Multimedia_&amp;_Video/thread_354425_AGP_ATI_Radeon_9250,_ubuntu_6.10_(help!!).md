### Post by barney_1 on 2007-02-06
Looks like this is a popular topic today.  The 9250 is no longer supported by the FGLRX driver.  You should try using the open source driver (edit your /etc/X11/xorg.conf file and change FGLRX to ATI).

Here is another post discussing the cards that are no longer supported:
```
http://www.ubuntuforums.org/showthread.php?t=262092
```

### Post by je1117 on 2007-02-06
djsubject, boot into the console and run
sudo nano /etc/X11/xorg.conf

Scroll down to your Device section and under where it says "ati" or "radeon" type
Option           "BusType" "PCI"

control-x, save, quit

sudo shutdown -r now

I run a 9200 and that fixed the black screen problem. But I haven't experimented with extended or cloned desktops.

Hope this gets you moving!
