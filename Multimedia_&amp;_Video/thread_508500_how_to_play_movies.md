### Post by deadgobby on 2007-07-24
Well there is a newer version of Ubuntu out there and you may want to go that route. That is 7.04 or know as Feisty Fawn. You'll need to a clean install with feisty. You cannot upgrade at this point to a new version for many reasons. OK with that said. You need to install restricted formates. Ubuntu does not come with them at default. The codecs are proprietary software
