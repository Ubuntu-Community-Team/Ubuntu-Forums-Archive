### Post by pogo07 on 2008-04-13
If you're just looking for a screen capture open Gimp.  Go to File - Acquire - Screenshot. Select the part of the screen you want to capture and set the delay to several seconds.  Minimize Gimp and wait for the crosshair.  Select the screenshot area and Snap.  Your image will open in Gimp.  You can manipulate it and save it or copy it to another program.
