### Post by jeffus_il on 2008-03-03
To open rar files you need to install at a minimum  the unrar package, then gui's like file-roller will automatically open them.
You can use VLC and other players wmv files, see this page:
[http://www.videolan.org/vlc/features.html](http://www.videolan.org/vlc/features.html)

### Post by jeffus_il on 2008-03-03
[LIST]
[*]Use the menu to run Synaptic Package Manager.
[*]Open Settings->repositories and make sure that multiverse is checked.
[*]Use the search box to find unrar.
[*]Right click on it and mark for installation.
[*]Click on the apply button.[/LIST]
