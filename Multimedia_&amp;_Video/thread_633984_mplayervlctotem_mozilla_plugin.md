### Post by coskierken on 2007-12-07
I had the same problems.  The normal Totem movie player and mplayer are built around gstreamer.  I don't know if it is still in the cafe, but my buddy complied a short utilities program to take care of the problem.  It is called uBackup!.  I know he is working on an updated program and it should be back in the cafe today.  In the utilities section there is an option to install all the files you need to get it to work correctly.  I stumbled upon the answer early one morning as I was pulling my hair out working on the problem.  I found you have to install Totem-xine from synaptic and associated codecs.  It is a little more tedious this way, but it will work.  :lolflag:
