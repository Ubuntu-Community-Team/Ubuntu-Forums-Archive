### Post by tpearson@alltel.net on 2007-07-18
I have found that nvidia x server setting is a good tool.  However i also determined it was not overlaying the /etc/X11/xorg.conf file when you get both monitors set up correctly.  So i save to working directory and then copied over /etc/X11.xorg.conf.

Dell 620  Nvidia Quadro NVS 110M
Distributor ID: Ubuntu
Description:    Ubuntu 7.04
Release:        7.04
Codename:       feisty
Linux AG-GWLM5D1 2.6.20-16-generic #2 SMP Thu Jun 7 20:19:32 UTC 2007 i686 GNU/Linux

load the following.
sudo apt-get install -y nvidia-glx-new nvidia-kernel-common
