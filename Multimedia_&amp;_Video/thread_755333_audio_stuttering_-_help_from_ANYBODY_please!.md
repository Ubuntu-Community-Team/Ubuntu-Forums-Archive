### Post by nicedude on 2008-04-19
It could be slow internet but there is an easy test to verify this or disprove it. Just open the video link and when it starts playing press pause and wait as you will see the files seek bar start filling up as the file is being downloaded and cached. Once the files bar is full it is done downloading so now press play and if it doesn't stutter then look at your connection speed as a probable issue ( try googling "speedtest" and run one ) but if your video still stutters with the full file cached it is something else.

Hope that helps you clear up what it is.
