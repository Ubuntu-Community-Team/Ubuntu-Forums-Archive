### Post by nowshining on 2007-09-07
system - preferences - session

Click New

in the name box input
```

nvidia-settings 

```

in the command box put
```

nvidia-settings --load-config-only

```

Click okay and go to save session, hit save and now ur done. However this will NOT show until AFTER you login into Ubuntu - This way it preserves each users choice. :) so you may see a bit of the old settings..  In other words to fully complete this restart the computer.
