### Post by patis on 2007-10-30
You can install Realplayer for Linux. Just visit [http://www.real.com/](http://www.real.com/) while running Ubuntu and you will see the download link right away. There are also simple installation instructions right below the download button.

Also, install VLC (from the command line type "sudo aptitude install vlc"). It plays lots of video/audio formats.

Finally, click Ubuntu's help button on the top bar and visit "Music, Videos and Photos" to learn how to install extra codecs and other applications.
