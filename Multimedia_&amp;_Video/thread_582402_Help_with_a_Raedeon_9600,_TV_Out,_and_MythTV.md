So the guts f the problem I have is as follows. 

I have a Raedeom 9600, plugged into the TV Out of a Ubuntu 7.04 box. I've tried different methods to get the card working and i've had mixed success. I can get the card up to run and display ok on the screen, but I am having trouble with getting it to run video files/DVB TV properly. 

Now the card runs fine under windows, and it also ran file under GeeXBox linux when I tried it (At least for displaying avi files and DVD's). So maybe there is a simple way to pinch the GeeXBox setup that I am unaware of. 

The first problem I had was after installing and setting up according to the [BinaryDriverHowto/ATI]("https://help.ubuntu.com/community/BinaryDriverHowto/ATI") with Xv enabled I only got half a screen. So I disabled Xv and tried OpenGl or just overlay-disabled mode (using ati-config to do this) which solved the problem with the size but ran in a very jerky fashion, so i'm guessing it was not accelerated. I've also tried using the automated install with Envy. 

After playing around with different setting including disabling compostie mode explictly (no difference in either Xv overlay, opengl overlay or overlay disabled) I discovered that
