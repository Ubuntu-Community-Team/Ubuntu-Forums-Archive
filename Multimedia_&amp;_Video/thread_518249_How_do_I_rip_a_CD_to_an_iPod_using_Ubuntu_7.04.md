### Post by lgambett on 2007-08-05
To rip a CD (mp3 format) the easiest way is to use Kaudiocreator (sudo apt-get install kaudiocreator). Then to interface your iPod you can easily use Amarok (create playlists, synchronize, etc...) or gtkpod. Should you wish to create and import videos into your iPod a fully working solution is thinliquidfilm. It works, guaranteed !
