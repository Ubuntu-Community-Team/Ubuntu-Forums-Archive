### Post by 3rdalbum on 2006-10-25
The ATI Proprietry Driver for Linux works with the Xpress 200. If you go to [www.ati.com](www.ati.com), then go to the Drivers and Software section, and click Linux, you will be given a set of links to download the driver.

Here is an Ubuntu-specific HOWTO:

[http://wiki.cchtml.com/index.php/Ubuntu_Dapper_Installation_Guide]("http://wiki.cchtml.com/index.php/Ubuntu_Dapper_Installation_Guide")

Make sure you use Method 2 (downloading the driver from the ATI website). Method 1, which involves getting an older version of the driver from the Ubuntu repos, can cause crashes with the Xpress 200 and a couple of other cards.
