Well for ripping the dvd to your harddrive uncompressed you can use dvd::rip (it's in the add/remove) or acid rip (also in the add/remove). If you want to actually compress the dvd these two do that too. To burn a dvd from a folder containing dvd files though I don't know how it can be done cause I haven't tried. ### Post by fenian on 2008-02-19
Mythtv can do most of those things and much more.Mythtv is one of the best,if not the best suite of  HTPC programs on any platform.Check out the Ubuntu specific documentation (Mythbuntu)[ here.]("https://help.ubuntu.com/community/MythTV").If you are only interested in ripping and burning DVD's I recommmend K9Copy to rip DVD's (You can rip to either iso for burning to DVD or to .avi) and k3b to burn DVD's,both programs are available in the repos....

> sudo apt-get install k9copy k3b

a guide for using k9copy to backup (rip) a DVD can be found [here]("http://www.dvd-guides.com/content/view/213/59/")
