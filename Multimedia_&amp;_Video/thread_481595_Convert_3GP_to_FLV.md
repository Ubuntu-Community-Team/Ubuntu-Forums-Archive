### Post by stooshbunutu on 2008-03-11
two steps, messy but effective

first you need software:

ffmpeg Terminal: *sudo apt-get install ffmpeg*
winff from [http://www.biggmatt.com/winff](http://www.biggmatt.com/winff)
Modile media converter from [http://www.miksoft.net/mobileMediaConverter.htm](http://www.miksoft.net/mobileMediaConverter.htm)

**flv to 3gp**

Use winff to convert the flv video into wmv format then use mobile media converter to convert the wmv file into a 3gp file
*NOTE: i believe the latest version of mobile media converter now handles flv files*

**3gp to flv**

use mobile media converter to convert the 3gp to video for computer (only outputs as mpeg) then use winff to convert the mpeg into  flv or other desired format

hope this helps
