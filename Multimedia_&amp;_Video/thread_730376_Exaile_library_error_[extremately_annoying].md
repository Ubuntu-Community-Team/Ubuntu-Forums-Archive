### Post by Kabezon on 2008-03-20
Oh, wow >.> Solution was pretty simple, and yes, it is a pretty silly bug: Exaile has problem dealing with some characters o.O Ever seen how when you copy a file from a PC running a different OS then some characters change? Well, that was the case here. The filenames had all the accentuation marks wrong, I had to change them and then Exaile recognized them ._. I'll leave this up just in case someone needs it lol
