### Post by tseliot on 2006-09-02
I guess it has to do with your card and the opensource driver (nv).

Boot in RECOVERY MODE from the GRUB Menu almost as soon as you turn on your computer (it will take you to the command line).

Then type:
```
sudo apt-get install linux-386 nvidia-glx
```

then type (this will disable composite and set the driver to nvidia):
```
sudo nvidia-xconfig --no-composite
```

then type:
```
reboot
```
