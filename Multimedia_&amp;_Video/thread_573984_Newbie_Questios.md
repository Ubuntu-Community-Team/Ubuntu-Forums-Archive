### Post by Cuppa-Chino on 2007-10-12
CD-DA extractor for ripping -- standard gnome is sound juicer as soon as you put a cd in it should start

mp3gain -- amarok (the kde mediaplayer that works under gnome) has a tool to normalise all the tracks during playback (I use amarok not rhythmbox personally)
-- audacity for audio editing, with all kinds
-- mp3gain works under linux as well [http://www.maazl.de/project/mp3/mp3gain.html]("http://www.maazl.de/project/mp3/mp3gain.html")
-- gnormalize as here [http://www.gnomefiles.org/app.php/gnormalize]("http://www.gnomefiles.org/app.php/gnormalize")

for playbay:

default in gnome is rhythmbox, in kde is amarok -- I personally use amarok (just install via add/remove program or amarok), others recommend exaile, xmms, etc but amarok is fairly comprehensive and has good plugins and library et al
