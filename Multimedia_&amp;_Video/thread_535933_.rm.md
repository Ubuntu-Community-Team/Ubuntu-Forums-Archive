### Post by orangey on 2007-09-14
I made a recent [post ]("http://ubuntuforums.org/showthread.php?p=3358937#post3358937")about this too.  Oddly, I finally got it to work by reinstalling the realplayer.

Get the player here:
[http://www.real.com/linux](http://www.real.com/linux)

Or just do:
1/ wget [http://www.real.com/realcom/R?href=http%3A%2F%2Fforms%2Ereal%2Ecom%2Freal%2Fplayer%2Fdownload%2Ehtml%3Ff%3Dunix%2FRealPlayer10GOLD%2Ebin](http://www.real.com/realcom/R?href=http%3A%2F%2Fforms%2Ereal%2Ecom%2Freal%2Fplayer%2Fdownload%2Ehtml%3Ff%3Dunix%2FRealPlayer10GOLD%2Ebin)
2/ chmod a+x RealPlayer10GOLD.bin
3/ sudo ./RealPlayer10GOLD.bin
4/ install it into /usr/local/realplayer (or wherever you wish), and take defaults for all other prompts (even the link prefix for "/usr")
5/ get aoss with: sudo apt-get install alsa-oss
6/ run realplayer with: aoss realplay
7/ edit preferences and set the quality to lowest setting for best cpu, and go to hardware and uncheck "XVideo"

That was the exact steps I got real media to work on my box.  Got it going just last night in fact.  Good luck.
