### Post by balticman on 2008-02-22
I think the problem you are going to have doing this is the format that videos from YouTube are in. In order to edit effectivly it is best to have videos in an uncompressed format.

 Kino for example is an excellent and easy to use editer.I know it can output files to MPEG2 but I dont think you can edit such files directly. You can also extract  sound files to edit seperatly in Audacity or anyother sound editer . Its then a simple matter to export the edited file back. The program is easy to use,the documentation is excellent and it works well on reasonably low spec machines.

However with compressed files your options are much more limited. If you can convert the file to ogg then this excellent thread contains a post on how to extract audio files and after editing put them back.

[http://ubuntuforums.org/showthread.php?t=430778&page=3](http://ubuntuforums.org/showthread.php?t=430778&page=3)

How to convert to ogg is another question.
