### Post by booe on 2007-03-02
Use **mencoder**, which is in one of the extra repositories. Make sure all the checkboxes are ticked in System -> Administration -> Software Sources -> (First Tab, in my case: Ubuntu 6.10)

Some information I found by googling "mencoder ipod":

[http://atomized.org/2005/11/converting-video-to-play-on-your-ipod-with-ffmpeg/](http://atomized.org/2005/11/converting-video-to-play-on-your-ipod-with-ffmpeg/)
[http://changelog.complete.org/posts/495-An-iPod-under-Linux.html](http://changelog.complete.org/posts/495-An-iPod-under-Linux.html)
[http://www.julien-oster.de/crash/122.html](http://www.julien-oster.de/crash/122.html)
[http://reviews.cnet.com/4520-10163_7-6279061-6.html](http://reviews.cnet.com/4520-10163_7-6279061-6.html)

Check out those links, or google it yourself for more :)

If you want to learn how to use mencoder/mplayer properly, I suggest reading the manual for it at:
[http://www.mplayerhq.hu/DOCS/man/en/mplayer.1.html](http://www.mplayerhq.hu/DOCS/man/en/mplayer.1.html)

I hope this helps you somewhat :)
