### Post by jefferybeene on 2008-02-07
I've been using Ubuntu Linux for about 3 months (it's the platform we use at work) and I have to say that I really like the OS.  Music players leave a little bit to be desired, though.

I've been using Rhythmbox, and for the most part it does what I want.  I've also installed Amorak, Kaffeine, Banshee, Decibel, Exaile, Quod Libet, XMMS and Songbird.  The main problem that I have is that I have an external hard drive with approximately 38,000 songs, and every single application wants to build a giant media library.  Ok, I understand that, but navigating within that library causes some applications to freak out.   I've never been able to get Amorak to work as it always freezes during the library build.

At home I've got an XP box that has WinAmp, iTunes, Foobar and Windows Media Player...I generally use WinAmp, especially since now I can use it to sync my iPod.  Initially Songbird looked promising, as I like the interface, etc, but it keeps scanning the library every few minutes and is basically unusable.

I realize this is open source software, and I'm willing to put up with a level of instability/lack of features as I prefer the open source model, but for me there isn't that one application yet that is the gold standard.  

So what do I want?

1. A player that handles all file formats (mp3, m4a, etc.)
2. Communication with iPod and other portable media players
3. Skinnable interface
4. Advanced tagging (song information, user defined album art)
5. Multi-platform application (Windows, Mac, Linux, etc.)
6. Ability to stream internet radio
7. Configurable interface that allows sorting by user-defined criteria
8. Equalizer
9. Multi-size player window functionality (full, smaller, mini, windowshade, etc.)
10. Easy playlist generation
11. Preferences that can be set and accessed regardless of OS/computer (i.e. set preferences once, then be able to have the preferences display as set whether on work PC, home PC)
12. Open source application - allows community to improve application without built-in hinderances
and most importantly
13. Ability to handle large media libraries on external drives

Don't want much, do I?  I think that with external drives coming down in price, and the fact that downloadable music isn't going away, eventually there will be something along the lines of what I want.

Until then, I guess, I'll use Rhythmbox and overlook the irritations.
