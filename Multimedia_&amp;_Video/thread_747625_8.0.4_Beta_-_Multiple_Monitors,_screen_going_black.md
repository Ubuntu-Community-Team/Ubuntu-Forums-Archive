### Post by IronArjen on 2008-07-15
I have the same problem with propietary ATI driver and a aticonfig DualHead setup. Since this is apparently completely screensaver related, I did the following. I saved my xorg.conf as xorgDualHead.conf. Then put a xorg.conf for single screen in function (save a backup as... ctrl-alt-backspace). Switched off the screensaver and (just to be sure) set to become idle after 2 h instead of 10'. Save, close. put back the xorgDualHead.conf as my xorg.conf ad went  back to work. I have it running for a day or so and the problem has not occurred yet, whereas I have been away from my screens for over an hour.

It is a bit quack but who cares as long as it works.
