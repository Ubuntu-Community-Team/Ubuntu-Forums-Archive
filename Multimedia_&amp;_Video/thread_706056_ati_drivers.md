### Post by ssam on 2008-02-24
it is best to use the drivers from the ubuntu repositories. This ensures that the package management system knows what is installed where, and can handle upgrades properly.

go to
System -> Administration -> Restricted Driver Manager

It lists what hardware you have that might need restricted drivers, and lets you tick to install and enable them.
