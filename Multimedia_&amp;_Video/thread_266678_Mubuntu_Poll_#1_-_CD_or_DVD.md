### Post by ropers on 2006-09-29
Actually, I wish to amend my vote:

I vote for *multiple* CDs.

After considering that there are a lot of people still stuck on dial-up, I can see the merit of providing a lot of data on disks. 
However, I can also see how DVD drive owners would prefer a single disk over multiple CDs.

That's why I propose creating a Live-DVD that's identical to the first, bootable CD (CD  number 0) except that it also includes all but the first of these multiple CDs (ie. CDs number 1 through *n*) as ISOs which it then automounts. This would ease maintenance of two seperate editions (CD/DVD) significantly while satisfying both camps.

Finally, the installer of the multi-CD set should work in such a way that you don't need to have all of the CDs to complete installation. It should just skip installing the additional software on the CDs that the user says they don't have. And the website or other online information should tell the user what software is on which CDs, so the user can decide in advance whether they wish to download all of the disks or just some.
