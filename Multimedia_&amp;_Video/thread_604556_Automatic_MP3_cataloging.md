### Post by jonsmirl on 2007-11-06
Recently I've looked at several people's machines with a lot of mp3s on them. The organization of their collections is chaotic.

A neat feature for Ubuntu would be a 'mp3/incoming' type directory where you copy new music to. Ubuntu would then process this new music through EasyTag, MusicBrainz, ReplayGain, etc to clean it up. The music needs to be clean to get album art and keep Last.fm from becoming full of junk. After it is clean it would get inserted into a standard library directory. If the tools can't get it clean they should ask for manual help identifying the music before inserting it into the standard library in unidentified form.

Does anything like this exist? It needs to be an automatic system. Users encountering this type of problem are not sophisticated computer users.
