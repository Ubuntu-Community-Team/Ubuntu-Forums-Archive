### Post by BobCFC on 2007-06-26
My xorg.conf is almost identical to yours except for the metamodes line.

First here is my setup in ascii art lol
```
 
 _______________ __________
|..............||.........|
|..1680x1050...||1280x1024|
|______________||_________|
......DVI.........ANALOGUE

```

```

 Option         "metamodes" "CRT: 1280x1024_60 +1680+0, DFP: 1680x1050_60 +0+0; CRT: 1280x1024_75 +1680+0, DFP: nvidia-auto-select +0+0; CRT: 1280x1024 +1680+0, DFP: nvidia-auto-select +0+0"

```


You can see I don't have **nvidia-auto-select** for the first pair but the resolution is set explicitly.  Maybe you could try something similar.  Although mine was set using *gksudo nvidia-settings*

EDIT:  Also I am using the 100.x.x series drivers from the [NVIDIA website]("http://www.nvidia.com/object/unix.html").  If you do switch, make sure you COMPLETELY REMOVE old Ubuntu restricted drivers using Synaptic, otherwise it will say kernel mismatch.  [Here ]("http://ubuntuforums.org/showthread.php?p=2914739#post2914739")is a post saying how I installed the latest drivers
