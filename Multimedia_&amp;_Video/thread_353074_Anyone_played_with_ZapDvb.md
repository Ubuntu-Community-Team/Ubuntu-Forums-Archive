### Post by steevc on 2007-02-04
I found [ZapDvb]("http://www.j-pfennig.de/zapdvb/") when looking for a better option than Kaffeine for recording digital TV and radio. One problem with Kaffeine is that you have to leave it running to do the recording. ZapDvb creates cron jobs for each recording. Whilst recording it uses hardly any CPU as it's not having to decode the video. It can also record DVB radio straight to OGG.

The DVB GUI is fairly limited. There's no EPG. You also have to work to get it set up for things like finding the correct set of channels, but it's all documented. There are utilities to edit recordings.

I've mailed the author to offer my help on fixing some of the English in the GUI and to ask about whether it's still being developed, but with no response as yet. I've got the source code, but I'm not too hot on C++. The last update was August 2005, but I haven't found any forum or mailing list to get further updates on the status. I even had to hunt around for a contact email.

I haven't tried MythTV yet. Is it viable to install it on a Kubuntu PC that's used for other things as well? I'd like to have a dedicated media PC, but can't justify it just now.
