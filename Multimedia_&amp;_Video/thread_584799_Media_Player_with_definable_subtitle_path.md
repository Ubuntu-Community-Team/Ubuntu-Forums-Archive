Is there a media player software where you can relatively define subtitle path (for .sub, .srt etc..)

Meaning, I have video1 in /clip/ folder and video1_subtitles in /clip/subtitles/ folder and same for video2 in /clip2/ and subtitles are in /clip2/subtitles/ and so on, you get the point?

So how can I specify a subtitle path into a progmran when it is like ./subtitles/

I haven't found these options from Mplayer, totem or Kaffeine, please let me know if there's some other usable programs for this matter. Meaning, it automatically displays the subtitles under the ./subtitles/ folder

And no, I do not want to keep my subtitle -files in the same folder with videofiles, it gets too hard to find anything from a folder if I have like 50 files in there in stead of having only 25 files.

### Post by Ardghal on 2007-10-21
I think VLC has that.

Under Options > Preferences, Video > Subtitles. Check Advanced Options, and there it is, you can set a path for the program to look for the subtitles.

I don't know how is it done on the other players.
