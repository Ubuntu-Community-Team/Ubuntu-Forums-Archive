### Post by joriad on 2007-06-11
When you start to add a file, a file properties box will open. Near the bottom you will see "advanced options" go there and select the "misc" tab. Now there will be a section for adding Extra parameters, Type in "-afm ffmpeg", this should fix the sound problem
