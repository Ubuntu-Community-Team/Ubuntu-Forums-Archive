### Post by DarkN00b on 2006-12-31
Intel integrated video chipsets use a slice of system memory, they don't have their own dedicated memory afaik. As far as how much memory they have, my 82852/855 intel system uses 32 megs. I don't know about the 945 or above.

### Post by d3v1ant_0n3 on 2006-12-31
I think the 9xx series chipsets (all the newer ones) all use dedicated memory- they're still not exactly great for gaming, but great for desktop use. My GF's HP has a 915chipset, it has 128Mb dedicated and manages to run some games fairly well (in Windows).
