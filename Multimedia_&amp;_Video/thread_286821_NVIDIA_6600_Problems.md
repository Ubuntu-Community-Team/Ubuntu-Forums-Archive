### Post by tool462rules on 2006-10-28
first do this
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup
```
This way if anything ever happens that X should not be able to start enter the command at the bottom (remember only if it fails).

---------------------------------------------------------------

Now if you did install the nvidia driver from the .run file provided on their website, save all your work and log out. Press Ctrl+Alt+F1. Log in. Now type ```
sudo nvidia-installer --uninstall
```Now restart ```
sudo shutdown -r now
```.

---------------------------------------------------------------

```
sudo cp /etc/X11/xorg.conf_backup /etc/X11/xorg.conf
```
That way you will be reverted to your X config file that works.
After you do this simply restart.

Tell me if this doesn't help, I'll be back in a little to help if it doesn't.

### Post by Abdo375 on 2006-10-28
I've solved the problem (it was a very strange one) for some reason the NVIDIA auto-configration tool that modifes the /etc/X11/xorg.conf file
forgets to add

```

BusID           "PCI:1:0:0"

```

instead it just write's
```

Section "Device"
    Identifier     "NVIDIA Corporation NV43 [GeForce 6600]"
    Driver         "nvidia"
EndSection
```

so after adding the line every thing seems to work well.
and for everyone that replied in this thread thank you very much.
