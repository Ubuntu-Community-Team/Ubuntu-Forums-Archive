### Post by Forlong on 2007-11-23
As far as I know, the only thing that *could* be illegal is [libdvdcss](http://en.wikipedia.org/wiki/libdvdcss) (which is not part of that package).
All the codecs incorporated in those gstreamer plugins are not freely distributable - that means you can install them for personal use but Ubuntu is not allowed to install them by default (same goes for the Acrobat Reader - which license is even more restrictive, Ubuntu can't even include it in it's repos anymore).
