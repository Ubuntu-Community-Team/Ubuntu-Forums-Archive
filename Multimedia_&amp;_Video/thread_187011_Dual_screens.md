### Post by Osamabingandhi on 2006-06-03
If you have nvidia start to install the right drivers...how to do that is not that hard. Remember to press ctrl-alt-backspace to make the drivers active.

you start with typing in terminal window: sudo gedit /etc/X11/xorg.conf and in there you change around, but save the file before you change anything, Save as... and add for example .restore at the end of the file name. Then edit the text. When you are finished you save and press ctrl-alt-backspace.

Search for twinview(if you use nvidia...) xorg.conf for the right things to change and add in the xorg.conf file. If you make a big mistake you have to start ubuntu in safe mode and type: "sudo dpkg-reconfigure -phigh xserver-xorg" or "cd /etc/x11" then "sudo cp xorg.conf cd xorg.conf.restore"

It's too bad it isn't just a program you can install to make it work...Hope this helps a bit.

The important part is under "Section 'Device'". You need to add your own Option "TwinView" "Yes" and an Option "Metamodes" and then add the resolution combinations you want to use. Tinker with Option "TwinView Orientation" "CRT-0 RightOf CRT-1" to tell it which is which.

Notice I do not have both displays configured (in fact only one is actually listed, the NEC monitor), however it still seems to work. Your monitors might be named something different from "CRT-0/1", so check your /var/log/Xorg.0.log to get their names.

Hope this helps.

Edit: Of course you need to have your nvidia drivers installed and working correctly first. Test with glxgears (should not look skippy at all).

Hm, I'm not sure about that. You mean you have a menu with checkboxes that lets you configure your nvidia card? If so, and you're saying it doesn't have options like TwinView etc, then you'll probably need to go into your /etc/xorg.conf file and do it manually. Dont worry, its not hard at all. Just follow my directions above and you should be set. Just sudo vi /etc/xorg.conf.
