### Post by diggity on 2008-04-15
>  		Audacious cant handle adding folders, i have to add the files one at a time.  Unless I am missing something!

On the playlist, click "Add Files" and select the folder and click "Add" It will import all the files in that folder for you. You can also drag a folder from Konqueror (kde) into the playlist and it will import them as well. I imagine this is true for Nautilus, but I don't use Gnome.
