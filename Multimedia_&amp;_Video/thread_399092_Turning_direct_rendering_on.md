### Post by crispy_420 on 2007-04-19
You can edit your xorg from rescue mode in grub boot menu. When prompted hit esc to get boot options. Select rescue. This is all command line. Login with usual username/password. Then issue this command:

nano /etc/X11/xorg.conf

Fix the offending lines, save and reboot.
