### Post by wooster on 2007-11-16
Sure I'll try to walk you through it. I chose a drumkit off of here [http://hydrogen-music.org/?p=drumkits](http://hydrogen-music.org/?p=drumkits).

First download the drumset you desire. It should come in an archive format like zip. You can use file roller to extract it where you would like or you can use the command line to extract it.  After you extract it will create a folder in which it stores the drumkit, remember where this folder is. The example i'm going to use is Millo-Drums v3. 

Command to extract archive:
```
 unzip MilloDrums-3.zip
```  (Note: this assumes that you downloaded it to your home folder, if you downloaded it to your Desktop make that "unzip Desktop/MilloDrums-3.zip")

1. Open up Hydrogen. 

2. you should see at the top left of Hydrogen a button that says "view" which you should click. A drop down menu will appear and the option you want to choose is "Show Drumkit Manager". Click it.

3. After you click that a new window will pop up with several tabs, click the import tab. 

4. Then click the browse button and navigate inside the folder where you extracted the drumkit.  Double click on the file in the folder. For my example it was DesktopMillo_MultiLayered3.h2drumkit.  

5. After that click the import button right below the browse button. A box will pop saying that it was imported just click ok. 

6. Then click the load tab where you should see the drumkit you imported listed.  Highlight the drumkit and click the load drumkit button.

 That's all! Hope this helps.
