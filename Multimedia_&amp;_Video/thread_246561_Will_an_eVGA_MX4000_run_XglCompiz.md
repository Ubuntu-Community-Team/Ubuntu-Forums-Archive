I would strongly advise you to get a AGP or PCI-Express graphics card if you have the appropriate slot. Any PCI graphics card will probably have problems. You don't need something ultra-high-end (the $200-$300 mentioned earlier is *very* excessive) for just Compiz, though. If you have an AGP slot, you can get an ATi Radeon 9600 Pro for relatively cheap, and it should run Compiz well enough. (Hey, it's pretty much the same thing that I've got in my laptop right now) (Note: The Radeon 9600 Pro is a better card than the 9800 SE). If you've got PCI-Express, you can get something a bit better for the same price. I'd go with a Nvidia GeForce 7300GS (or GT, if you can find it cheap enough). Any of these should be under $70 or so.

EDIT: And yeah, the difference between any of these cards and your Intel Indegraded Video should be, as they say, "night and day".
