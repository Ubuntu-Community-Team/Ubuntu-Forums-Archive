### Post by Darkriser on 2006-03-13
I have Radeon 9100 card (pretty old, huh:-) and succeeded installing the drivers following this thread: [http://ubuntuforums.org/showpost.php?p=423584](http://ubuntuforums.org/showpost.php?p=423584)

As I have two monitors, I editted my xorg.conf file to enable dual-head. And the result - when I boot ubuntu, both monitors are blank. Despite of that, I log in (username->ENTER->password->ENTER) and...after GNOME starts, everything is fine - dual-head working great. OK, I'll have to solve this little issue in the future, but an advice for you - try logging in even if the monitor is blank. Maybe you're lucky....

(Anyway, this configuration "bug" has an advantage - it is a great security enhancement which makes logging in pretty difficult:-)
