### Post by finferflu on 2007-02-13
Which version are you using? I'm using version 0.1.9999.1. 
To start with, my workspaces were not working properly, that is, all the windows were stuck to desk1, no matter what I did. I thought about the idea of the cube, and I came up with the conclusion that each desk represented a cube by itself, more or less. So I right-clicked on the desktop pager and set it to 1 workspace only. Then I went into Beryl general settings and set the number of workspaces and Vertical Virtual size to 1. After that I could see one very long rectangular desktop on the pager, without borders, but every window was in its proper place on the right workspace where I put it. It was like an open cube...

I hope this helps...
