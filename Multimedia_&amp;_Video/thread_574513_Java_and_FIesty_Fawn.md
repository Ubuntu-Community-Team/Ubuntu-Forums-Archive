### Post by RomeReactor on 2007-10-13
Hi. You can install the **sun-java6-plugin** through Synaptic (System->Adminsitration->Synaptic Package Manager); or to install it from the terminal (Applications->Accessories->Terminal):
```
sudo apt-get install sun-java6-plugin
```
