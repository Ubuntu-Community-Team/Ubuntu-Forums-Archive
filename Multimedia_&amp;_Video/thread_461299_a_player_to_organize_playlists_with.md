### Post by bom28 on 2007-06-01
I know quod libet and gmusicbrowser can search in comments too.
What you want seems perfect for using labels/tags/keywords, I know at least gmusicbrowser and amarok can set labels, gmusicbrowser has a nice cloud view to select labels and other things. Unfortunately I don't think either of them can save the labels in the song file, so you can't use them with another player :(
Also, quod libet can be configured to use custom tags in the song file.
