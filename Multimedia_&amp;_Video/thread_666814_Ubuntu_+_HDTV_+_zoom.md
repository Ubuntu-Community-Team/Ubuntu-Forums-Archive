### Post by hebetude on 2008-01-13
These gimpy little 3.2'' portable devices have all kinds of nice zoom features.  But my nice 32'' HDTV is stuck at tiny fonts.  Even Apple can show (probably extremely limited) zoom feature for various websites.  

However, I have yet to find one for linux that works as well.  The closest thing out there is probably Compiz Page zoom (not compatible with ATI) and Firefox 3's full page zoom takes an incredible performance penalty.  FF3's page zoom renders far better than IE7, but the performance hit on my ATI/fglrx is so bad I don't even bother with it anymore. My ATI card can do 10k fps in glxgears, but it completely disables 2d acceleration when I turn on any type of composite manager.  So 3d still goes at 10kfps, but firefox scrolls windows at 2fps.

If somebody was just waiting for a cool name to call a program that could do this I got one:
** Squint**

P.S. turning up the font is not the same thing, you just wrap text inside of DIV/table boxes on websites.
