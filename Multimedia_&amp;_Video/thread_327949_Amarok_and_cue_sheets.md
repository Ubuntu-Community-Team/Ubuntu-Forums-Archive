### Post by tyfius on 2007-11-17
Put the .cue and .mp3 file in the same directory and give them the same name. The files won't show up as individual files in the play list window but the track list will be visible in the context menu. The name of the track in the play list window changes over time and you are able to select individual tracks (or jump to them) from the context menu.
