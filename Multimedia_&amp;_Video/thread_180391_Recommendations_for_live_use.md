It was updated a long time ago. If there are still issues with the current package, then a Freewheeling user should really report it to Launchpad/Malone so that it has a small chance of being fixed in the next **week** before Dapper is released..

For live stuff, which I don't know about, other than FW, there is also seq24, and a sampler like LinuxSampler or Specimen might be able to do some of what you need. Also you might like to look into LMMS because the Specimen developer has stopped development in favor of LMMS.

Usually you would work with several apps linked together with JACK.
