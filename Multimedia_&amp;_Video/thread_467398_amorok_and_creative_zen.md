### Post by zzztownsend on 2007-06-08
Open amarok
Settings --> configure amarok
Select media devices tab
Click add device
Select plugin ----> choose MTP media player
Give the zen a name
OK. OK.

Bosh - should be working

If the MTP media player option is not there then you need to install package 'libmtp5' using synaptic package manager

Hope this helps

Cheers

Matt

PS I have found amarok slighlty unstable at times - if you get problems try installing and using Gnomad2 for transferring to zen
