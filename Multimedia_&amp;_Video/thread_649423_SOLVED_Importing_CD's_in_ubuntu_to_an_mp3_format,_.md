Sound Juicer CD extractor - extracts songs from CD and converts to one of several formats such as mp3, ogg, mp4, speechx, etc.  If not already installed, type the following command

sudo apt-get install sound-juicer

After installation, open program and select #EDIT#Preferences to configure what format you want the program to use.
