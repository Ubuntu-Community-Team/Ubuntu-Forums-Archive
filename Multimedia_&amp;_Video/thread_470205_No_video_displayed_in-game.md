### Post by dfreer on 2007-06-11
It is now using the OSS "nv" driver, which should be very stable. However, in most cases it doesn't support direct rendering/3D acceleration (you can check this out by running those two commands, glxgears and glxinfo | grep rendering), which means you won't be able to play advanced 3D games, such as America's Army. pSX might work, because it is more CPU bound than GPU bound, well... I'm guessing it is very laggy because of the above reasons. Still don't know why it would be "squashed" into the top of the screen.

If you have about 1 Gb's of RAM, you could try using the Live CD and installing the restricted driver manager's nvidia drivers IN the live CD enviroment, and then test pSX out from there with no need to reinstall :) save you from doing a reinstall just to be disappointed. Of course it will probably be too laggy to play America's Army and the like, but pSX should run, and you should be able to just test and make sure the video works.
