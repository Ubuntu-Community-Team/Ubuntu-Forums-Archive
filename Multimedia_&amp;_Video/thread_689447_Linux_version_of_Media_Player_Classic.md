### Post by MrClearPore on 2008-02-07
> I haven't been able to find a media player replacement for Media Player Classic on linux yet.

It can be quite difficult to find a nix program that behaves exactly like its Window counterpart :) I'd been using Media Player Classic myself in Windows a long time before switching to Kubuntu four months ago, and I was looking for a MPC-type player.  However, just like you, I couldn't find the right one.

However, one thing I noticed was that one does not *need* to have the same functionality in order to get an enjoyable experience with a program.  I use **xine-ui** now and I'm liking it.  All keyboard shortcuts can be customized, so your option to have "Freely resizable Aspect Ratio via keyboard/numpad" can be configured quite easily in xine-ui.  As I recall, there is no option to shutdown the program after playback, but hey, you can't always get what you want :)

Switching from Windows to nix is a huge deal, and chances are you're not going to find programs that are going to mimic Windows programs exactly (unless you use Wine), so my advice is to try different nix multimedia programs to see which one fits you the best.  In time, you'll learn to enjoy them, and forget about MPC hopefully :KS
