### Post by Abaddon on 2007-05-15
If you ask a KDE user which media player they use, the answer seems to be pretty much always Amarok, which is not surprising given how good it is - pretty fully featured, slick etc.

Not so with Gnome.   There's a lot of decent media players out there, but none of them have the full range of features - Rhythmbox isn't bad, but doesn't support MTP, Banshee supports MTP but is not up to speed yet, and then there's exaile, songbird and probably a whole heap of others that I don't know about.

I wonder if it would be helpful for some more sharing of resources (or maybe a de-fork, ala Beryl & Compiz) to end up with a combined effort to come up with an amarok-like player for gnome - the Unified Media Player.


My wishlist is a decent MP3 player with Musicbrainz +/- Wikipedia support, Podcasting, MTP support for my Zen Vision:M.  A more controversial idea is I'd like Vodcasting support, so I don't need a separate podcast manager to copy my vids to my player.

What does everyone think?
