### Post by francisco_athens on 2007-06-17
Assuming you are using the Gnome desktop, there was a [patched metacity]("http://ubuntuforums.org/showthread.php?t=242502") that placed all windows on the main screen.  Even more insight [here]("http://ubuntuforums.org/showthread.php?t=15298&highlight=twinview+metacity") However it has not been updated for Feisty.  You are not alone with this [complaint]("http://ubuntuforums.org/showthread.php?t=426962").  [Bug reports]("http://bugzilla.gnome.org/show_bug.cgi?id=145503") have also been made.
  I've since had to use the XFCE window manager, xfwm on Gnome which deals with twinview in a sane fashion.  I'm considering just using complete XFCE desktop, but I'd have to install nautilus as well.

Good luck,

Francisco
