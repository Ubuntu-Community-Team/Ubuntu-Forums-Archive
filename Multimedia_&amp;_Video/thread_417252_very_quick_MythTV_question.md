### Post by superm1 on 2007-04-21
If your on Feisty, install the "mythtv" package.  It will get you a quick up and running setup that includes a backend and frontend.  You will still need to run through all of the steps in mythtv-setup however, and set up a location for recordings to end up.

No, you need a backend for MythTV to work. You can have a frontend on another machine, but it needs to connect to a backend. IMHO, MythTV is worth taking a little time to read up on before the install. I am sure you will find it worthwhile.
