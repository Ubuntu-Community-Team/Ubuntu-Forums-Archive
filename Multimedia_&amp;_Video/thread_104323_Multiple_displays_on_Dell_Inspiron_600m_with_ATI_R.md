```

nick@ubuntu:~$ fglrxinfo
```

I think what I need to do is edit my /etc/X11/xorg.conf file, but I don't even know where to begin. I searched around these forums a bit and couldn't find anything really straightforward on how to do this, so can anyone point me in the right direction? Ideally I'd like to set it up so that it happens automatically when I connect another display, or just by clicking a button to run a script or something easily.
