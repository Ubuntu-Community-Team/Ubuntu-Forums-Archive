### Post by pressed on 2008-02-23
in amarok 1.4.8, you can go to *Tools -> Script Manager -> Get More Scripts* and find Gnome Multimedia Keys as one of the most downloaded scripts. 

This worked for me on my Dell Inspiron 6000. You might want to make sure you have the newest version of Amarok as I think the script manager changed recently...
