### Post by Ubuntwan on 2007-11-23
people on the rhythmbox IRC told me that this is the way the queue is supposed to work. It is not the same as a playlist. If you would like behaviour like the playlists used in winamp, you have to use playlists instead. For playlists to function properly (with shuffle and repeat) however, the queue must be empty. So I guess I will not be using this queue anymore, i don't like its function very much.
i will now mark this thread as solved
