### Post by RomeReactor on 2007-04-16
Hi. Have you tried reconfiguring X (**sudo dpkg-reconfigure xserver-xorg**)? If not, then running it may solve the refresh problem; be sure to make a backup of xorg.conf first:

```
sudo cp /etc/X11/xorg.conf xorg.conf.backup
```

then

```
sudo dpkg-reconfigure xserver-xorg
```

If that doesn't solve the issue, and it breaks your X server, then restore the xorg.conf file:

```
sudo cp xorg.conf.backup /etc/X11/xorg.conf
```
