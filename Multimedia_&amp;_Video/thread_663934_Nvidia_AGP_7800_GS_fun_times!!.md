### Post by Shazaam on 2008-01-10
I have a BFG 7800GS agp card too. I have full use of desktop effects.  I used the nvidia README to install their driver 169.07.........

[http://us.download.nvidia.com/XFree86/Linux-x86/169.07/README/index.html](http://us.download.nvidia.com/XFree86/Linux-x86/169.07/README/index.html)

To shut down X hit control+alt+f1, login and type sudo /etc/init.d/gdm stop. Then sudo sh the driver.

### Post by Shazaam on 2008-01-11
You MUST make sure nvidia-glx-new (or -legacy) is uninstalled. Try this (scroll down in the first post to Debian)......

[http://www.nvnews.net/vbulletin/showthread.php?t=72490](http://www.nvnews.net/vbulletin/showthread.php?t=72490)

Once you get everthing taken care of on that page (except installing the driver) hit control+alt+f1 (drops you to a prompt),go ahead and login then cd to the location of the driver (mine was on the desktop so I typed in cd ~/Desktop).
Enter sudo sh NVIDIA-Linux-x86-169.07-pkg1.run


P.S. as stated before it's quirky; same card different install procedures so keep trying you will have success.
