### Post by kerry_s on 2007-03-08
Install the nvidia-glx driver, then in terminal run> sudo nvidia-xconfig --twinview
reboot if it installs a kernel and make sure you select that kernel.
And it should auto set it up, than you can adjust it to your needs.
