### Post by dabl on 2007-05-21
When you run (in the console) ```
nvidia-settings
```, do you get the Nvidia driver panel?  If so, if you click "X Server Display Configuration" it should change the monitor resolution.  But I only see 1024x768 in your xorg.conf, so you need to have the Nvidia driver write a new xorg.conf.  First, I would use the Envy installer, and reinstall Nvidia driver, and follow instructions to set it up.  Then, you can open the console and type ```
sudo nvidia-settings
``` and choose your desired resolution.  Then in the lower right of that "X Server Display Configuration" panel is the button "Save changes to X configuration?", and you want to say "Yes" and save it.

:)
