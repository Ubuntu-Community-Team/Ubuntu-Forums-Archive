### Post by jordilin on 2006-07-28
streamripper, rips shoutcast mp3 audio streams. It rips every song in a separate mp3 file. So, if you have a huge mp3 file, it means it's a long song. I have mp3 files of more than 50Mb, and they take more than one hour to listen. In any case, If you wanna split mp3 files, you can edit this mp3 file by using **audacity**. It's in the repos.

### Post by mauro007 on 2006-07-31
Hi
some radio stations don't break song into separate streams.
Apparently there's a lot of data besides the simple audio stream, some of this data (I think is called metadata) if present would identify each song and separate it from the rest.
If you want separate a huge chunk of ogg or mp3 into single songs, and if your soundcard has hardware mixing present, you can use audacity.
If you get problems once audacity is installed, check this forum again (and perhaps be prepared to fork out a new souncard).
Hope it helps.
Good Luck
