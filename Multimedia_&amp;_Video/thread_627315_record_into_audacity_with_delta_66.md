### Post by tgalati4 on 2007-11-30
>envy24control

Set up the patches so that hardware input #1 is sent to the PCM/digital mixer.

Make sure Audacity's preferences show the Delta 66 card as the input device.

The Delta 66 is a sweet card and it works well with Audacity.  I use Dynebolic for it's real-time kernel for more critical recordings.

Also, make sure the microphone level is turned up in Audacity (using the slider under the microphone meter).  Don't forget to turn "Monitor Input" on the microphone pull-down box.
