Well, i'm using Edgy and when i plug my Zen player it automatically opens a new window with all the mp3 files i've got loaded, so maybe the difference could be if you're using Dapper or Kubuntu.

You could try opening "System-->Preferences-->Removable Drives and Media", and on the first tab (Storage), on "Removable Storage", make sure the first 3 boxes are checked; other than that, i don't know what could be wrong, sorry.

**Edit**: Try installing KzenExplorer or Gnomad2; look for them in Synaptic, or from the terminal

```
sudo apt-get install gnomad2
```
