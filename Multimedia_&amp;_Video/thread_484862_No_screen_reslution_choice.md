```
Section "Monitor"
      Identifier   "aticonfig Monitor 0"
      Option       "DPMS" "on"
      **HorizSync    30-81**
      **VertRefresh  56-75**
EndSection
```

### Post by scxtt on 2007-07-03
i don't think xorg.conf will have much to do w/ VNC ... you can specify the dimensions of the VNC session when you start it:

vncserver -g 1280x1024 -depth 24

also, you may be getting a black screen for VNC if your ~/.vnc/xstartup file isn't starting a desktop session (xfce,gnome,kde,enlightenment, etc.) ... are you using the "Remote Desktop" from an already logged in Gnome session?

assuming the **nv** driver is working correctly, you still don't have the VertRefresh and HorizSync values for your monitor section:
```
Section "Monitor"
          Identifier "CM771"
          Option "DPMS"
EndSection
```you should find out what they are for your monitor ... also, running **sudo nvidia-xconfig --no-composite** may help ...

also, you DON'T have to reboot after making changes to xorg.conf ... just hit CTRL+ALT+BACKSPACE to restart X ...
