### Post by Flamekebab on 2007-04-16
Well I've never really used Windows Movie Maker, but I've seen stuff made in it and I've watched others use it, so I have a vague idea.

However, in all honesty, I find Kino trickier to use. It doesn't seem to be as happy to play with non DV video, but it will if you persevere. What I mean is, I used it to edit a few videos from my mobile phone and from an old digicam and it worked fairly well, but now that I'm editing footage from my Panasonic camcorder, I find it works very well.

I'd recommend you install Kino 1.0, which was recently released, as it seems to be considerably more polished.

Unfortunately, I don't know how to insert images or audio into the footage in a time-line fashion. Adding images is a problem I'm currently grappling with. Audio can be mixed in via the filters menu in the effects tab.
