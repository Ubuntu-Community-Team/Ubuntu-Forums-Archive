### Post by frodon on 2007-08-06
I think you have the profile of someone who will prefer linux mint over ubuntu, linux mint is based on ubuntu and use ubuntu repository.
Linux mint has different art work and some additional tools to download packages and **it has all the codecs installed by default** which is why i think it will fit your needs.

Ubuntu don't include the codecs by default by choice, they don't want too much proprietary stuff by default. Things are easy to install when you have an internet connection however as you can see it becomes a little bit more annoying without internet connection. This is where projects like linux mint are filling the gap and offer codecs installation by default.

My advice, download and install linux mint if you won't have any internet connection in the future, you will save your time :
[http://linuxmint.com/cassandra-light.html](http://linuxmint.com/cassandra-light.html)
[http://linuxmint.com/download.html](http://linuxmint.com/download.html)
