### Post by wild_oscar on 2008-01-04
Well, apparently you can only have one 5.1 stream at a time.

I seemed to have solved the problem by configuring Amarok to use 2.0 stereo.

And I also removed the .asoundrc file which had the contents

```
pcm.!default {
   type plug
    slave.pcm "surround51"
    slave.channels 6
    route_policy duplicate
}

```

And was crashing my 2nd instance which tried to access an already taken 5.1 sound output.
