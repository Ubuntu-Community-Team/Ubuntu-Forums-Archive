Please do the following:
Open up /etc/fstab as root:
```
sudo gedit /etc/fstab
```
and make it look as follows:
> # /etc/fstab: static file system information.
#
# <file system> <mount point>   <type>  <options>       <dump>  <pass>
proc            /proc           proc    defaults        0       0
# /dev/sda1 -- converted during upgrade to edgy
UUID=043d09d1-7b0f-4398-bee0-e144dd17526f / ext3 defaults,errors=remount-ro 0 1
# /dev/sda5 -- converted during upgrade to edgy
UUID=226cc024-13b0-43e0-b6c2-6461cd65fd2b none swap sw 0 0
/dev/cdrom        /media/cdrom0   udf,iso9660 user,noauto     0       0
save and close the file and restart your computer. DVD's will now load fine
You possibly made an error last time you edited /etc/fstab or trying to remove Automatix mounted partitions. You do not want comments on the same line as a mounted partition. That is what was causing the error.

### Post by xeocub on 2007-05-20
Actually you might want to change this line in your fstab:

/dev/cdrom /media/cdrom0 udf,iso9660 user,noauto 0 0

to something like this : 

/dev/dvd /media/dvdrom0 udf,iso9660 user,noauto 0 0

Make sure you create a dvdrom0 folder in /media before you try to mount though!

( sudo mkdir /media/dvdrom0 if you didn't know )
