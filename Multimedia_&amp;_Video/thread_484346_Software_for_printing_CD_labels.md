### Post by eivind on 2007-07-05
A Synaptic search for "cd label" gives two results:

1. Old and trusty cdlabelgen, the oldest program of this kind for Linux.

2. Disc-cover, which seems a bit more tailored to your needs. This program can at least query the freedb database for cd info. Come to think of it, I think I'll install it myself. Give it a spin if you feel like it.

All the best,

Eivind


Edit: Disc-cover is available online at e.g. [http://disc-cover.vanhemert.co.uk/](http://disc-cover.vanhemert.co.uk/)
