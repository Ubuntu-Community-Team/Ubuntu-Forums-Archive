### Post by Hendrixski on 2007-04-24
I think that the idea of bittorrenting streams is pretty cool, and I know that the guys behind that big p2p file sharing network Kazaa are now working on a p2p stream sharing network, so I wanted in.  I haven't  tried Joost yet, but I looked at a few alternatives.  Please help expand this thread with knowledge or experience.

**PeerCast:** unintuitive but it works, and actually has channels. Though, it looks like audio only channels.  The configuration doesn't seem to work, you're supposed to be able to configure it through your browser after you run ./peercast but I couldn't. No worry, I just clicked on a few streams and they worked pretty well... They didn't have to buffer every other minute, just smooth playing.  Unfortunately the same is not true of streams that weren't well seaded.

**FreeCast**: has an intuitive GUI (not just through a browser like peercast), and is really easy to install via JavaWebStart.  Almost too easy (feels like downloading malware on Windows).  There are only about 3 channels total, and they're professional radio channels (from France, not that we should hold that against them).  The Software doesn't seem to want to work though.  If you can figure out how, please tell me!

OK.... I'll try out a few more and post what I find.  If you have any that you like then please share, I want to try them.  :-)
