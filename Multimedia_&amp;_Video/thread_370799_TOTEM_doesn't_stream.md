### Post by chewearn on 2007-02-26
For files on you local harddrive, you need to install the codec:
[https://help.ubuntu.com/6.10/ubuntu/desktopguide/C/codecs.html](https://help.ubuntu.com/6.10/ubuntu/desktopguide/C/codecs.html)

Alternatively, you can install VLC player using Synaptic.  VLC player comes with built-in codecs.

For youtube, you need to install Firefox flash plugin.  Go to a website that has flash, Firefox should pop-up a bar on top of the window, asking you to install the flash plugin.  Click on the button to the right of th bar.
