Hi this is my first post so forgive me if this is in the wrong place. I've been using ubuntu and smplayer for about a year know and am loving it. My only real issue is that I can't load videos in smplayer like I could in media player classic back on windows. On windows I just double click on a file and it plays, when I want to play another video I double click on it and it loads in the very same player. In ubuntu when I try to load another video while one is playing on smplayer it just opens in another smplayer window and there's no video only sound. My question is is there any way I can make it so that when I double click video files they will all open in the same player?

### Post by soga_genzo on 2008-03-06
Open the "Preferences" dialog from the "Options" menu. Navigate to "Interface" -> "Instances". There's an option "Use only one running instance of SMPlayer" that does exactly what you want. If this option (or the "Instances" tab) is missing, you should upgrade SMPlayer to a new version by installing the Ubuntu package from the official [website]("http://smplayer.sourceforge.net/downloads.php?tr_lang=en").
