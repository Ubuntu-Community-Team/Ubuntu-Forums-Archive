just share how I get flac support for bmp

download the source package from:[http://www.skytale.net/files/bmp-flac/](http://www.skytale.net/files/bmp-flac/)
sudo apt-get install beep-media-player-dev libflac-dev
extract the downloaded file
cd the extracted folder
make
sudo cp libflac.so  /usr/lib/bmp/Input

that is all ,enjoy it!:D
