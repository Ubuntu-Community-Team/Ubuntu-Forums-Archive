### Post by Hanj on 2007-03-02
I'm planning to take an old machine and turn it into some kind of all-round server/multimedia box. What I want it to do is mainly:
- Play video files from its own disc as well as from other computers on the LAN (via TV-out)
- Play music
- Act as a file server
- (Maybe webserver and FTP later on)
I'm also planning to buy a remote control for it, so I would like this to be supported.

Anyway, I would really appreciate some tips on how to set all this up.

My first thought was to use MythTV, but I've heard it's a pain to set up and configure. On the other hand, most of the trouble seems to be with TV cards, and I won't use one. All I need is something to keep track of audio and video files and present them on an interface that is readable on a TV screen. How hard would it be to set up MythTV for this? Are there any good alternatives that might be easier to use?

I downloaded the GeeXbox CD and tried it, and it was actually really nice. Worked perfectly out of the box and was really easy to use. However, it didn't seem to be able to do much more than play movies. Is it possible to use GeeXbox as a media player on another distro, e.g. Ubuntu? 

All sorts of advice, tips, comments etc. are most welcome! I would like something that is as easy as possible to set up. Preferably using Ubuntu, but I might also consider something more specialized, like MythDora.

### Post by barney_1 on 2007-03-03
I have high hopes for the developments that have been made with Elisa Media Center:

[http://www.fluendo.com/elisa/](http://www.fluendo.com/elisa/)

It's worth a look.  You should be able to use LIRC to control Elisa with a remote control.

For networking, I'd just put some extra drives in the box and set them up as either NFS or Samba shares.
