### Post by jingo811 on 2007-12-23
looking...

I was given a bunch of methods and tutorials when I was in your shoes 3-4 months ago.  This was the only solution that satisfied all my Realplayer needs.
[COLOR="Red"]Unfortunately that circa 5.5 MB, **RealPlayer10Gold.bin** file has been unreachable for 2 months now[/COLOR].  You should pray that someone can upload that big file to some site so you can easily download it.  Once you get your hands on that file the rest is easy peasy.

[http://ubuntuguide.org/wiki/Ubuntu:Feisty#How_to_install_RealPlayer_10_Multimedia_Player_.28RealPlayer.29](http://ubuntuguide.org/wiki/Ubuntu:Feisty#How_to_install_RealPlayer_10_Multimedia_Player_.28RealPlayer.29)

**1. Download**
[http://www.real.com/linux](http://www.real.com/linux)

**2. Add execute permissions to the installer and run it: **
```

chmod +x RealPlayer10GOLD.bin
sudo ./RealPlayer10GOLD.bin
```

**3. Firefox plugin:  MediaPlayerConnectivity**
Then download the Firefox plugin:  MediaPlayerConnectivity to let your browser handle various types of media files.
