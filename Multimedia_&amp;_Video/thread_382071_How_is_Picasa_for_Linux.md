There isn't really a linux version, they use a modified version of wine to run the windows version. Therefore, all the same features and probably just slightly less stable.

### Post by trippinnik on 2007-03-17
Actually it is missing a few features (mentioned on the google site).  It doesn't upload to Google pictures yet.  I love picasa, but there is an even better program I use now called F-Spot.  It's open source, actively developed for linux and not running under Wine.  It does a lot of good tagging and doesn't automatically pull images from all over you computer.  I don't really need icons in my album.  Also the tags and batch editing are nice as well as being able to upload to several different picture services: flickr and google are the ones I can remember.
