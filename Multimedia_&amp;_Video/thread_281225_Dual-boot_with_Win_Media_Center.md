### Post by joegumbo on 2006-10-29
I am dual-booting Windows Media Ed with Ubuntu Edgy Eft on an eMachine W3503.  On my setup from the factory, Windows uses two partitions:

One FAT32 restore  and drivers partition

and 

one NTFS Windows Media Edition partition.

Ubuntu went on just fine without any issues.  I was running Slackware on here with Win Med Ed, so the drive was already partitioned.  

The thing tom keep in mind is that there are  two windows partitions.  If When I  booted into the the marked Win Xp, I booteed into the Restore partition and the Restore Wizard starts automatically.  In order to stop it, I had to force a hard shutdown.  But, after I booted into the partition marked Windows Media Edition, everything was fine.  Basically, I now have three partitions.

I didn't try it, but I suspect that if I did try to do a Windows Restore, I would probably end up reformatting the Ubuntu partition.
