### Post by Erlander on 2007-03-18
Open your file browser, go to the "View" menu and check "Show Hidden Files" then navigate to your users home folder (in my case it is /home/Rob) and then into the .mythtv folder.  Double click on the mysql.txt file to open it and check the password there.  Do not alter the password in the file but you then use it when starting the mythtv-setup etc.

Hope this works.

### Post by majoridiot on 2007-03-19
if you installed mythtv from ubuntu packages:

on a seperate frontend/backend setup, copy /etc/mythtv/mysql.txt from the backend to ~/.mythtv/ 
on the frontend.

if it is a combined frontend/backend, do:

```
# more /etc/mythtv/mysql.txt
```

this will show you the password for the database (8 characters such as **fhjieslk**).  enter that in the
password field of mythtv-setup and make sure the address for the backend is set correctly (first
entry on the same setup page)

also- make sure your main user is in the mythtv group:

```
# sudo usermod -a -G mythtv USERNAME
```

where USERNAME is your main account.
