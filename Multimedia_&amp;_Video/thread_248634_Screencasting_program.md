### Post by celloandy on 2006-09-01
Personally, I quite like Wink ([http://www.debugmode.com/wink/](http://www.debugmode.com/wink/)).  It's very easy to use, and allows you to add subtitles, voiceovers, etc.  Strict F/OSS people don't generally care for it, though, since it's not open, and it outputs Flash animations which are also only viewable using proprietary software.  If you're looking for something comparably easy that's GPL-licensed (I think), take a look at Istanbul ([http://live.gnome.org/Istanbul](http://live.gnome.org/Istanbul)).  It's Gstreamer-based, and outputs Ogg Theora (note that there have been some significant releases since Dapper arrived, so you may want to look around for a more current package than the one in apt).
