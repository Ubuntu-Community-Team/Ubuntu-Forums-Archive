### Post by RomeReactor on 2007-09-26
Hi. Declaring one player to be the best has proven to be a very subjective exercise; opinions in this regard frequently vary.

However, for me Totem has been very good playing multimedia files; if you want to give it a try, [here's how]("http://ubuntuforums.org/showpost.php?p=3399733&postcount=8").

If you prefer totem-gstreamer over totem-xine, substitute the third command in the linked post for this one:
```
sudo apt-get install totem-gstreamer libdvdplay0 libdvdcss2 libdvdread3 libdvdnav4 mplayer gstreamer0.10-ffmpeg gstreamer0.10-fluendo-mp3 gstreamer0.10-fluendo-mpegdemux gstreamer0.10-plugins-bad gstreamer0.10-plugins-bad-multiverse gstreamer0.10-plugins-ugly gstreamer0.10-plugins-ugly-multiverse libflac7
```
