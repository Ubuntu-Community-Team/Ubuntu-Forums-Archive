This is mostly aimed at the linux n00bs trying Ubuntu Studio. ;)

If you have "upgraded" to Ubuntu Studio from a existing Ubuntu install with proprietary drivers you must know that this will install another kernel without the needed modules. We cannot install these drivers so you will have to grab them manually.

You can, after you grab our packages but before a reboot, install **linux-lowlatency** and be fine.


[COLOR="Red"]Please do not post in this thread about xorg issues from a disk install.[/COLOR]
