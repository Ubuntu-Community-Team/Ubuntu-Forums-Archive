### Post by eye208 on 2007-11-30
I have an Nvidia 7900 GS which works fine with Compiz, but I won't recommend it anyway because Nvidia drivers seem to have even more issues than ATI ones. For example, XVideo is half-broken, and when I run games with anti-aliasing enabled, I experience occasional crashes which affect the whole desktop (even without Compiz running).

If I had an ATI card now, I would just wait for the upcoming drivers. The latest fglrx has AIGLX support, i.e. it works with Compiz. ATI/AMD have released their GPU specs so you can expect the open source driver to improve a lot in the future. Unless Nvidia follows suit, this will make ATI the #1 choice on Linux.
