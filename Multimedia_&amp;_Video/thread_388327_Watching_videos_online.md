### Post by flossgeek on 2007-03-19
First set up your [[COLOR="Blue"]repositories[/COLOR]]("https://help.ubuntu.com/community/Repositories/Ubuntu"), then follow my guide on my [[COLOR="Blue"]blog[/COLOR]]("http://www.ossgeeks.co.uk/?p=79") to install all the codecs.

I would also advise you to download VLC Media player,

```
sudo apt-get install vlc
```

and also grab a firefox plugin called [[COLOR="Blue"]mediaplayerconnectivity[/COLOR]]("https://addons.mozilla.org/firefox/446/"), which will create a black box where media is situated, then when you click the black box it will launch the player you have configured.
