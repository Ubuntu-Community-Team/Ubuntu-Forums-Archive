### Post by magicfab on 2006-10-27
The closest I would see would be Digikam + Kipi, using the MPEG encoder plugin. See:
[http://docs.kde.org/development/en/extragear-libs/kipi-plugins/mpegencoder.html#using-plugin-mpeg_encode](http://docs.kde.org/development/en/extragear-libs/kipi-plugins/mpegencoder.html#using-plugin-mpeg_encode)

apt-get install digikam kipi-plugins

Please do write back here if / when you find anything better.
