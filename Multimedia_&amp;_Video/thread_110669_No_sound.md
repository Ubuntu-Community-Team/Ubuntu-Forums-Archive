### Post by anil_robo on 2005-12-31
1. Software? You mean Ubuntu? Or some package in Ubuntu like XMMS?
2. Regarding CD playback, there's an internal audio cable for CD playback. Did you check your computer's inside?
3. For mp3 playback, you will need the mp3 codecs. You will have to enable the "universe" repositories first, and then install the mp3 codecs from them. More info is found at [http://www.ubuntuguide.org/#codecs](http://www.ubuntuguide.org/#codecs). MP3 do work on Ubuntu, and I play mp3 almost the entire day! :)
4. File sharing - there are many programs in Ubuntu like amule, directconnect, nicotine but I don't use them. You can open the "add applications" from the applications menu and then try searching for file sharing. Then install the programs from there and use them! :)
