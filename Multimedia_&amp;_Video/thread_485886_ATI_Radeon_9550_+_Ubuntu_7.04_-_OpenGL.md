### Post by Septor on 2007-08-01
Looks like the kernel driver "fglrx" is not getting loaded, probably because it doesn't exist.

If you ran the ATI installer directly, then it installed stuff in a non-Ubuntu fashion that could be causing your problems.  The easiest option is to use the Ubuntu linux-restricted-drivers package (apt-get install xorg-driver-fglrx), or if you want the latest-and-greatest driver you can build the packages yourself from the ATI installer with the "--buildpkg Ubuntu/feisty" option from the command line.  That should create 4 .deb packages that you can then install via dpkg.

More details can be found here: [http://wiki.cchtml.com/index.php/Ubuntu](http://wiki.cchtml.com/index.php/Ubuntu)

Hope that helps!
