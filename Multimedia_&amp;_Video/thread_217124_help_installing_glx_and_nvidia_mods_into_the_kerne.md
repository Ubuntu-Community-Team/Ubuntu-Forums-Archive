### Post by Jagot on 2006-07-16
Assuming you've installed the Nvidia drivers correctly as below:

```
sudo aptitude update && sudo aptitude install nvidia-glx]

Then you need to run the following command:

[code]sudo nvidia-glx-config enable
```

And restart.
