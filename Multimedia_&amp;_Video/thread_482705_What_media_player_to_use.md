### Post by NilsE on 2007-06-24
Install the totem plugin for Firefox and the w32codecs from the package manager and you should be good to go for the WMP stuff. If w32codecs does not show up in package manager you may need to go into add/remove programs and look under restricted packages or search on this forum on how to install it.

Take a look at the sticky 

[http://ubuntuforums.org/showthread.php?t=413626](http://ubuntuforums.org/showthread.php?t=413626) 

and you will get just about everything you need.

I actually have installed everything I can find and just choose the one I like for each media type in the edit/preferences/content/mange options in Firefox.

### Post by Chappy7777 on 2007-06-24
I too listen to alot of net radio, and have found it pretty darn ez to do w/Ubuntu. I am a dapper dude, 6.06, btw. 

All you need is to use synaptic and let it download and install a little program called Streamtuner. Then use synaptic again for either Amarok or XMMS, or both. If Rhythmbox is on your install already (I can't remember if I downloaded it or it came w/Ubuntu). 

Anyhow, I find my stations using Streamtuner, click play, and XMMS comes up (w/a nice looking skin) and plays the station. I am limited by only having dialup, so I only look for stuff streaming at 32bits or less. XMMS is a clone of Winamp, or maybe the other way around, anyhow, if you use Winamp, XMMS will be no problem. 
Streamtuner is obvious. (Heck, I figured it out!)

Give these a try. You might not find the stuff you're used to listening to, but there's tons of stations there to check out.
