### Post by bodhi.zazen on 2008-01-16
You need to first download the nvidia driver.

Then remove nvidia-glx and nvidia-glx-new.

Then :

```
sudo rm /lib/linux-restricted-modules/.nvidia_new_installed
```

Then :

```
gksu gedit /etc/default/linux-restricted-modules-common
```

Add this to he DISABLED_MODULES line to make it look like this:-

> 	DISABLED_MODULES="nv nv_new"

Then install the nvidia driver :

```
sudo /etc/init.d/gdm stop
sudo apt-get install build-essential linux-headers-`uname -r`

sudo sh /path/to/NVIDIA*

sudo /etc/init.d/gdm start
```
