Alternative software suggestions would also be welcome, but must include built-in database management to allow for cataloging all media files and cover art. I was very fond of "Xlobby" (until recently a free Windows application). I'm looking for the Linux alternative to Xlobby. I suppose if worst comes to worst I could try running Xlobby in a Windows VM under Feisty, but I'm guessing that watching movies that way would be problematical.

Any help or suggestions would be appreciated.
