So many steps? If you load up Synaptic, you can install **a lot** of software with only a few clicks. You can also install things from a terminal, which I find a lot more convenient.

> **diddy47 said:**
> Is there onay hope of having compatibility to .exe files? there are a lot of great programs like VLC player, Veoh TV clent, VeoB subtle program etc all ubuntu users are missing on because of that
VLC is available for Linux. MythTV is a popular TV client (sudo apt-get install mythtv mythtv-themes). I'm not familiar w/ VeoB. Most Windows programs have Linux counterparts. The thing most former Windows users complain about is gaming. WINE is a bit limited on what it can do, but it can run many EXEs and install Windows programs in Linux. However, I spend most of my time playing Battle for Wesnoth, which can be installed using Synaptic or **sudo apt-get install wesnoth-all**.

> **diddy47 said:**

There are some predictions about what kind of support we'll see in Linux in the future. A lot of it deals with hardware compatibility (esp. wireless), some improvements to TV tuning, and support for the Blackfin architecture, which looks pretty sweet (I'm a big fan of RISC processors, like the ARM and PowerPC processors). I think RISC is the future os servers, as CISC processors are inefficient and complicated.
