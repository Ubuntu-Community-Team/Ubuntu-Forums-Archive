### Post by sowelie on 2007-12-22
The best way to solve this is to boot in to safe graphics mode (hit esc while grub is loading and pick the second option).  This will get you to the command line.  Then just run:
```
sudo dpkg-reconfigure xserver-xorg
```

This will bring you through a wizard to set up your display / keyboard.  It should be able to auto-detect your display settings.  You'll just need to restart after you're done.
