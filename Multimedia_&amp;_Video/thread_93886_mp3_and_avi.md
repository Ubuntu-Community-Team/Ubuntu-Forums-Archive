### Post by frodon on 2005-11-23
Yes you should install some codecs.
For video, ffmpeg, w32codecs, libdvdcss2, quicktime, realplayer, ...
For mp3 it depends on which player you like, there's : xmms, beep media player, amarok, banshee, muine, ...
I give you a link about restricted formats : [https://wiki.ubuntu.com/RestrictedFormats?action=show&redirect=Codecs](https://wiki.ubuntu.com/RestrictedFormats?action=show&redirect=Codecs)

If you look for an automatic way, i advice you to run automatix which will install for you all the codecs : [http://www.ubuntuforums.org/showthread.php?t=80295](http://www.ubuntuforums.org/showthread.php?t=80295)
