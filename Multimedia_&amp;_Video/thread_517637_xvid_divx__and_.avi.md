### Post by Gremlinzzz on 2007-08-04
smplayer plays divx  if your using ubuntu feisty get the package (Qt3; without KDE support)
if your using kubuntu get the one with kde support.
if you want to watch divx movies on line go to synaptic and uninstall the Mozilla plugin and install the mplayer Mozilla plugin then put these commands in the terminal one at a time and hit enter. 
cd /usr/lib/firefox/plugins/
sudo ln -s /usr/lib/mozilla/plugins/mplayerplug-in-dvx.xpt
sudo ln -s /usr/lib/mozilla/plugins/mplayerplug-in-dvx.so
if you need codecs
[https://help.ubuntu.com/community/RestrictedFormats/WindowsCodecs?highlight=%28codecs%29](https://help.ubuntu.com/community/RestrictedFormats/WindowsCodecs?highlight=%28codecs%29)
