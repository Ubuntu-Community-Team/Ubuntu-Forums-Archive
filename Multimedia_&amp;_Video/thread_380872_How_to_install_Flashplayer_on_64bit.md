### Post by Duffy on 2007-03-17
I found any easy way around this problem. I installed Swiftfox (a 32bit version of Firefox optimized for Linux as I understand it) using the Automatix installer and also installing then 32 bit version of Flash. Use Firefox for normal browsing and Swiftfox when I have to open Flash related web sites.
