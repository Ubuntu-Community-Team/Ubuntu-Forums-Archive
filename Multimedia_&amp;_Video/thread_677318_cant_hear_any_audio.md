### Post by gilgongo on 2008-01-24
Have you tried running the Volume Control application? If it's not on a menu anywhere, try opening up a terminal and typing "gnome-volume-control" (without the inverted commas). Play with File > Change Device too.

If no joy from that, you can try having fun with alsamixer. Open a terminal and type "alsamixer" - once you've stopped laughing at the 1908's computer screen style, try using the arrow keys to adjust volumes on stuff. Hit escape to quit.
