### Post by hyperair on 2008-02-11
GUI program eh? Try using "Mobile Media Converter". It's not in the Ubuntu repository, but there's a Linux version available. Google it.

For me, I usually use my own custom build of ffmpeg, built with AAC support, as well as 3GP and AMR (narrow band and wide band). But ffmpeg is a command line tool, and Medibuntu provides the same build of ffmpeg anyway (but I use Ubuntu hardy so it's different).

EDIT: If you're still not satisfied, some Windows conversion programs work in Wine.
