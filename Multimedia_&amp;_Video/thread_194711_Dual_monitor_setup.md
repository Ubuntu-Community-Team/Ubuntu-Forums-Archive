### Post by fluffington on 2006-06-11
You have the same nVidia card listed three times. You only need one Device section, unless you're using more than one card. Your screen sectios should then all refer to the same device.

Also, I've never gotten the open-source nv driver to work with multihead setups, so you may have to install nvidia-glx and change "nv" to "nvidia" (and disable dri in the modules section).
