### Post by stooshbunutu on 2008-02-11
I used AcidRip to do just this:

When it loads up there will be all the chapters individually. However there will normally be one much larger file to rip (about the running lenght of the film) Clicking this to rip takes a long time to rip but puts it all together as one file.

Alternatively once you have ripped each scene you can piece them back together using Aidemux (works best with the .avi output from AcidRip) and open the first file, then use the File >> +Append (Ctrl+Alt+A) to add in each scene in order. Then Export the project as one file with the correct  scenes all in one.

Hope this helps:)
