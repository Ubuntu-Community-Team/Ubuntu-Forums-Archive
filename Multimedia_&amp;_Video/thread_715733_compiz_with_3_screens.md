### Post by broncavfan on 2008-05-15
I tried for days to get compiz working on three monitors and never had any luck.  I ended up using twinview + compiz on my dual-head card and my third monitor ran metacity as a seperate xscreen.  It worked well enough for me.  Sometimes I'd have problems when I'd try to open certain programs on the metacity xscreen, though.  I would run the program and nothing would display, but the process would be running in the System Monitor.  If I shut down compiz on the other screen the program would then magically appear as it should have.  It was very strange, but it didn't happen all the time, and really I only used the third screen as more of a monitor than an actual workspace.

If you need to find the old xorg.conf I put together to get it working I know I have it saved somewhere.  Good luck!
