### Post by Porch on 2006-09-01
Sure you can have two desktops. Don't use xinarama. 
I have it setup on my laptop with the built in display and the VGA out. 
It works just like having two computers.  My mouse moves from one desktop to another just fine. The only downside to not using xinarama, is that I can't drag windows across monitors. The upside is that DRI does work on the main display.
