### Post by thefockinfury on 2007-07-13
this may be a dumb question of me considering that youre (obviously) an audio-savvy linux user and seem to have been around the forum for a good deal longer than i have... but have you read about ubuntu studio? its got loads and loads of free audio apps included, one of which (the name escapes me because i wiped my hard drive and haven't yet reinstalled it) is a program that's based on programs like soundforge and peak that provides in-depth editing and, i believe, several mastering effects. check out the link if you haven't already. "upgrading" from a standard ubuntu installation to ubuntu studio is also very straightforward and i (still a linux noob, relatively speaking) was able to do it the first time without a hitch. 

[http://www.ubuntustudio.org]("http://www.ubuntustudio.org")

cheers

EDIT: obviously if you have your current setup working and you're happy with it there's no need to install ubuntu studio (unless you have already). check out this package list [https://wiki.ubuntu.com/UbuntuStudio/PackageList]("https://wiki.ubuntu.com/UbuntuStudio/PackageList") and scroll down to the ubuntustudio-audio section. it seems like the app 'jamin' is what you're looking for. > jamin - Audio mastering from a mixed down multitrack source with JACK
