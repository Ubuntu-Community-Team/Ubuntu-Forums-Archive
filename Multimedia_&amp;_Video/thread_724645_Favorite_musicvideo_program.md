### Post by MCrittenden on 2008-03-14
If you're looking for something like Winamp, you'll probably get a lot of recommendations for Audacious and XMMS.  Of the two, I prefer Audacious and I believe (put I'm not positive) that XMMS isn't still in active development.

I use Exaile, though it's probably more bulky than what you're looking for.  Along the same lines, Listen Media Player is also pretty popular.

If you want a complete list, check out [http://en.wikipedia.org/wiki/Category:Linux_media_players](http://en.wikipedia.org/wiki/Category:Linux_media_players)

### Post by oskar2000 on 2008-03-14
I like Exaile too. I got talked into all the bling-bling - I was a winamp-style-player kind of guy before that too, now I won't take anything that hasn't a dynamic-playlist feature and last.fm support. :)
Audacious is probably what you're looking for.
Amarok is very cool too, if you use KDE... works on gnome as well, but takes some time to load. For video I use mostly vlc and m-player.
I mostly use vlc because it has a drag and drop playlist.
