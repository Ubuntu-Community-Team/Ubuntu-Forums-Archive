I'm in the process of backing up my LP collection to digital using Audacity on Gutsy 64. I'm using Audacity to generate .flac and .mp3 files from the project and all seems good. But Audacity won't play the .flac files. id3v2 will put an id3 tag on them, so I don't think it's the actual file. There is a note in Audacity that it's creating 16 bit .flac files. Are these no compatible with Audacious? Also, Totem does play them, but I prefer Audacious.

### Post by DantePasquale on 2008-05-23
Hi, To answer my own question, in case any reads this, the problem with Audacity not reading/play .flac files was that I added mp3 tags to the flac files. I found, somewhere on some board, this very problem. I regenerated the .flac files and did not add mp3 tags to them and they are now fine.

The only "problem" this causes is that on my jinzora2 site, I have to be careful to NOT ENABLE automatic mp3 tag updates because when you reload the metadata that will add mp3 tags and then Audacious will no longer play them :(
