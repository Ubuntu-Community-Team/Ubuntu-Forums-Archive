### Post by wallywally on 2007-11-13
I've got the same problem in that when I try and play flv files in VLC I get only see the top left of the picture and the rest of the image is blank (blue screen) whilst the audio is fine.

However, I have discovered that within VLC Media Player if you go to Settings and select  Extended GUI then Image Clone it provides a duplicate window for the file that works (the original still mostly blank).

This isn't a very elegant solution but it works.
