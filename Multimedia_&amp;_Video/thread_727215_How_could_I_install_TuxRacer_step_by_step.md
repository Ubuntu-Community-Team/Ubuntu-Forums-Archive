### Post by bmac on 2008-03-18
I assume your talking about planetpenguin-racer. It can be added directly from your Synapatic Package Manager. Open "System" "Adminstration" "Synaptic Package Manager" Search for planetpenguin, left click on empty box to the left of the file and click on "mark for installation" then click apply in the tool bar. Don't forget to add the "extra" for additional game play.
