### Post by mysticrider92 on 2006-10-30
For the first question, you should be able to open file browser (nautilus) and go to the /cdrom folder and see the files, or you can use sound juicer (I think) and rip them. For the second question, I assume you are using Gnome. If you are, go to System > Preferences > Multimedia System Selector (or something like that, I am not on my Ubuntu computer right now), and you can choose the application to open different thing like cd's, cameras, etc.
