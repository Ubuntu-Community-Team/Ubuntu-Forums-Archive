The problem I am having is getting a Trust Wireless Pointer Stick Keyboard (KB-2800 UK) working in MythTV. 

This is how far I have got so far:- 
The keyboard/integrated mouse works perfect in Ubuntu Feisty 
However when I run mythTV the keyboard fails to do anything-exit MythTV and it works again. The integrated mouse works ok in either Feisty or MythTV. 

The keyboard appears as 2 USB HID devices in device manager under one USB "channel". 

I have tried the USB part of the keyboard in all available ports and have unplugged any other USB devices.

### Post by wilko10 on 2007-05-13
Hi

Problem solved. It was fully solved by upgrading the motherboard bios,but we decided to do this because just by chance when Myth frontend was running we unplugged then plugged in the usb keyboard "dongle"  it started to work.

Maybe it had lost connection somehow when we started myth frontend and unpluging/plugging in restarted the connection-anyway it worked hurrah!!
