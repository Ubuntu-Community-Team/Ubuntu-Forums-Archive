### Post by kspn on 2007-07-02
I have been testing both Torvid and DeVeDe and I have found that overall Torvid is better than DeVeDe mainly because adding files is a simpler process. I have not yet been able to do anything with OGM or MKV files that have 'Soft' Subtitles. both also automatically put Chapters every 5 minutes on a video file.

I have found that I have had to edit the XML file that Torvid created because of a bug in the program (v0.5 so it is fine) but apart form that they both do the same thing in a similar way so I think I will be using Torvid.

I will experiment with QDVDAuthor next but unless there is something that stands out about it (OGM/MKV for example) then I will stick with Torvid and give up on making DVDs of the multi stream file types.
