Hi,

There are two such programs written for Gnome that allows editing mp3/ogg tags:

Cowbell Music Organizer:
[http://more-cowbell.org/index.php/Main_Page](http://more-cowbell.org/index.php/Main_Page)

Audio Tag Tool:
[http://pwp.netcabo.pt/paol/tagtool/](http://pwp.netcabo.pt/paol/tagtool/)

I am not sure about Tag Tool, but Cowbell's web site supposedly has the update feature from the web. Both are available in the repositories.

### Post by billynomates on 2008-03-24
Picard is by far the best tagger I've used, and I've tried Cowbell, Easytag, quodlibet, the freedb tagging tool in Rhythmbox, Amarok and god knows what else.

Add
```
deb http://ppa.launchpad.net/luks/ubuntu gutsy main
```
to your /etc/apt/sources.list 
Then
```
sudo apt-get install picard
```

Or, if you're using Hardy, I think it's already in the repositories.
