### Post by bogdan_5844 on 2008-02-04
I went mad trying to make some DVDs I made with my Sony Camera to convert to avi.After trying many softwares that took hours to complete,i found one wich does it in max. half an hour.


AVIDEMUX-Search for it in ->Applications->Add/Remove Programs and check it for install like so: 



[IMG]http://img220.imageshack.us/my.php?image=screenshotio1.png[/IMG]

Before we start using Avidemux,enter in the DVD folder and copy all .VOB files (except the one with the menu,it should have a different name than the majority of the files)to a temporary folder.
Then start Avidemux.
Go to File,then click Open.Locate the .VOB files,and open the first one(it should have (VOB filename)_1_1.VOB at the end).It will offer to index it,click yes.


Then it will ask you if you want to merge the multiple .VOB files,telling you that

 "It looks like you have multiple MPEG files.Append them?"(or a message like it)

Wait for Avidemux to finish indexing,then go to File->Save->Save Video..(or press Ctrl+S).

In the new window that appeared,in the corner select AVI(*.avi) and in the name field write something like youraviname.avi (for example vacation.avi or trouble.avi,make sure you have .avi at the end)

Click save,and you're done!
