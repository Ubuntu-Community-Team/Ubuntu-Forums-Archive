### Post by Sheik Yerbouti on 2006-07-09
Korg Minipops 35 Soundfont

This is a soundfont containing samples from a cheesy beat box from the 1970s called the Korg Minipops 35. It was used by early electronic music pioneers like Jean Michel Jarre. I created it in swami a soundfont editor that runs on Linux. It is not GM compatible because the samples don't really map to a full drumset I tried to make it GM compatible but the layout just didn't make much sense when I laid it out that way. So the samples start at middle c and go up from there. I will be using this in a tutorial I am working on. The tutorial will be titled "Sound Like Jean Michel Jarre" from Computer Music 89 adapted for Linux and Rosegarden. The point of which is not to entirely ape JMJ but rather to learn the tools by creating a track start to finish.

[Download Soundfont]("http://www.allenjeter.com/korgminipops.sf2")
