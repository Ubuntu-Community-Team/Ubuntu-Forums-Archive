### Post by hardyn on 2007-06-22
its not the easiest thing...

use "sudo nvidia-settings"

set up the dual monitors, and let the utility sugget what changes should be made.  but don't let it over-write your existing file, it does not append, but makes a new one.

save the file with an alternate file name, then use 2 gedit windows and copy the monitor, device, screen and server setting sections over to your xorg file... 

here is an example:
