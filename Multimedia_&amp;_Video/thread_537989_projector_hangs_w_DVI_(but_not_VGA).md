### Post by kriotbsap on 2007-10-31
I also have an Optoma ep910 and also could not get it into 1400x1050 mode under Linux with the nvidia driver.  I eventually figured out that the valid refresh frequencies reported by the projector (via EDID) are incorrect.  So the nvidia driver discards the 1400x1050 mode due to a "too high" horizontal refresh rate.

I was able to get it working by adding a line in my X config file monitor section like this:

option "UseEDIDFreqs" "FALSE"
