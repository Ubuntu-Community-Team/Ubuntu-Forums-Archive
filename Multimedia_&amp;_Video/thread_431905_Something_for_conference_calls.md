### Post by amoore on 2007-05-03
Try **the Gizmo Project** [http://www.gizmoproject.com/]("http://www.gizmoproject.com/"), you should be able to find a .deb on their web page [URL="http://www.gizmoproject.com/download.php"]http://www.gizmoproject.com/download.php
[/URL]
here are directions on how to set up a conference call [http://www.gizmoproject.com/conf-calling.html]("http://www.gizmoproject.com/conf-calling.html")

I believe that the Gizmo Project can have 2-20 users including real phones and computers all for free. The Gizmo Project's GUI looks very much like Wengo's and there is no need for a server. I believe the the Gizmo project is owned by Google.
