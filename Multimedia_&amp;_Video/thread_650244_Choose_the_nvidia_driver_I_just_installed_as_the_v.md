### Post by GSF1200S on 2007-12-26
Try changing the driver section from 'nv' to 'nvidia' and save. (Not in the failsafe section)

Youll need to restart X to put the change in effect (Ctrl+Alt+Backspace).

Be prepared that X may fail, at which point use nano to open the xorg.conf and modify it back to get X:

```
sudo nano /etc/X11/xorg.conf
```

This shouldnt happen though, as you have failsafe X, which will automatically switch to nv drivers...

If this doesnt work, reinstall the driver, and make sure to run this command:

```
sudo nvidia-glx-config enable
```

and restart X...

Good luck :)

### Post by GSF1200S on 2007-12-26
Another thing: what version of Ubuntu are you running?

You can try starting from scratch:

Go into Synaptic and remove the Nvidia driver (make sure to have the nvidia common kernel installed).

Download the Nvidia driver from the website. Put it in your home folder and rename it to: nvidia.run

Kill the GUI:

```
sudo /etc/init.d/gdm stop
```

Press Alt+F2 and login.

Regenerate the xorg.conf:

```
sudo dpkg-reconfigure -phigh xserver-xorg
```

Run the nvidia driver:

sudo sh nvidia.run

have it automatically update the xorg.conf- use the command above in previous post if it doesnt succeed in doing so.

Restart X:

```
sudo /etc/init.d/gdm restart
```

Go into Synaptic and remove the nvidia common kernel, but not the driver (this prevents an API mismatch error). 

You should be good then...

Good Luck :)
