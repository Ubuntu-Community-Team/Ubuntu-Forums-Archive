### Post by ubuntu27 on 2008-01-03
I bought[ iAudio7]("http://www.cowonamerica.com/products/iaudio/7/") by COWON. I absolutely love it! It is 100% compatible with GNU/Linux.

You connect it with USB, and just copy and paste your music, picture, or video files to the Drive using your favorite file browser (nautilus, konqueror, etc). that's it. No need to use any special application.

It supports the following formats: MP3, WMA, **OGG**, FLAC, ASF, WAV, and JPG.

More information on the official website:

[http://www.cowonamerica.com/products/iaudio/7/](http://www.cowonamerica.com/products/iaudio/7/)

The Playlist can be accessed by pressing the "M" button that is located at the rear end of the iAudio7. 
The PlayList display the file name and not the ID tag which is organized alphabetically. It does not have a search feature, but it is easy to find your music if your file names are correctly labeled.

It seems the "M" stands for Multimedia. Since when it is pressed the first time you can access the Music's Playlist, if you pressed it for the second time, it displays  a list of icons with "Mac bar" style.

The list of icons are: Music, Video, FM Radio, Record, Picture, Text, and Settings.
