### Post by misterhead on 2008-03-25
the biggest problems with our ATI cards is that Ububtu treats the " big desktop " as just that...
Instead of 2 ( in my case ) 1280x800 displays, it treats it as 1 2560x800 display. I don't know the specs on yours, but my Mobility x600 will only allow 2048x"whatever" with ANY desktop effects whatsoever.
try turning desktop effects off entirely, and see if doesn't work for you. then you will know you have a choice between effects and resolution.

I run full effects when undocked, but have to turn them off if i want to go big desktop and keep 1280x800 (or better) on each one. If I back it down to 1024x768, i can run full effects with compiz and the cube, but some GL screensavers overwork the card and I have had it lock up a few times doing misc. other things.
