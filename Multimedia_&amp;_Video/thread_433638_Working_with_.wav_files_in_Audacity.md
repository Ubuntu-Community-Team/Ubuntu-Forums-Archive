### Post by ratbagradio on 2007-05-05
Audacity converts your .wav file into its own working format. So if you can load it -- it will take several minutes to do so -- then you can edit it in Audacity by playing , listening and cutting as you desire by using the tools option on the left side of the top panel. You'll find the "**I**" Tool the most useful if you are a beginner. As you do so save occasionally as an Audacity "project".

I'm not certain what your difficulty is. When Audacity loads the file it should fill a track across the window.and your file will be visually represented as a series of  audio waves. You can edit by zooming in and out and deleting segments you want to discard.

Then when you are finished the edit  EXPORT it in your desired format: Mp3, Ogg Vobis or Wav. But if you want to save it to Mp3 you will need to have LAME installed.

Go to Sonic Stage [for tutorials on editing with Audacity ]("http://audacity.sourceforge.net/help/tutorials")


Here is a review of [installing Audacity]("http://www.ubustu.com/globe/2007/04/24/how-to-enable-lame-support-in-audacity/") and [one on running LAME on Feisty]("http://www.ubustu.com/globe/2007/04/21/compile-audacity-132-beta-with-feisty/").
