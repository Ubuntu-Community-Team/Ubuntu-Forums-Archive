### Post by koleoptero on 2008-02-19
Patience :)

I believe since VLC can play mp4 files, a good option would be to install vlc, and then install the vlc firefox plugin. I am using it to watch/listen to video/audio on various sites that normally want wmp to play.

You can find vlc in add/remove programs, and for the plugin open a terminal and run:

sudo apt-get install mozilla-plugin-vlc

Hope this solves your problem.
