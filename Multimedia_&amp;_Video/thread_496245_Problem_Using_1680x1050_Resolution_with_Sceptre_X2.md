### Post by ftblguy on 2007-07-08
Hope to get some help here, as I am really stuck. I'm using a Sceptre X20 monitor, which has a native resolution of 1680x1050. I don't have a graphics card installed. Just using the integrated Intel video chipset on my motherboard.

At 1440x900, the display is just fine. But when I try to use 1680x1050 resolution, it doesn't display properly. About 400 pixels go off the screen to the left and about 26 pixels go off the screen to the bottom. What I mean by this is that Ubuntu is displaying 1680x1050, but the monitor is only displaying 1280x1024. The monitor's OSD menu shows "1280x1024". It appears as though the monitor thinks I'm trying to run at 1280x1024 when in fact I'm really running at 1680x1050. So the result is this: in the top menu bar, "Applications", "Places", and "System" are nowhere to be found, as they run off the left side of the monitor; and the bottom panel bar is completely missing, as it runs off the bottom, if that makes any sense.

I have gotten 1680x1050 to work fine in Windows, so I know it should work. I have tried to adjust the ModeLines in xorg.conf and have also tried the XServer reconfiguration and a lot of the other suggestions posted at [https://help.ubuntu.com/community/FixVideoResolutionHowto](https://help.ubuntu.com/community/FixVideoResolutionHowto) to no avail. Any ideas as to what my problem may be? Thanks!

### Post by nutz on 2007-07-09
Search my posts and you will see a fix for this. You need to set a metamode...



```
Section "Screen"
    Identifier    "Default Screen"
    Device        "Generic Video Card"
    Monitor       "Generic Monitor"
    DefaultDepth    24
    Option         "metamodes" "LCD: 1680x1050_60_0 +0+0"
    SubSection     "Display"
    Depth                 24
    Modes         "1680x1050"
   EndSubSection
EndSection
```

---

### Post by nutz on 2007-07-09
The problem here is too high of a horizontal refresh. Your vertical refresh is set to 60 which the display likes but the horizontal needs to be lower... I had this same issue with my LG 22" LCD; this is why the picture is off center.
