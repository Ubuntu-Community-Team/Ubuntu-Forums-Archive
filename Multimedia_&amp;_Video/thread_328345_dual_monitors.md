### Post by EZ-man on 2006-12-30
Section "Device"
A device is a video card or video adapter, so you will need one device section for each card in your system. A video card with multiple outputs is considered a single device, but requires two Device sections to distinguishthe outputs. Each Devise section includes, at a minimum, entries for Identifier and Driver, but if you have more than a video card installed, you will also need to enter a BusID value so Xorg knows which section is associated with each card; if you have a card with multiple outputs, enter a Screen value so Xorg knows which section is associated with which output. Use 1spci -X to determine the BusID for each card in a format suitable for entry into xorg.config: 
                                               EXAMPLE
Section  "Device"
     Identifier "NVIDIA Corporation NV34M [Geforce FX Go5200]"  
     Driver      "nvidia"
     BusID      "PCI:1:0:0"
EndSection      

 If you have a single device with multiple outputs, just number the Screen entries starting from 0. You should also give each Identifier a unique name. simply append 0 or 1 to the end 
                                                EXAMPLE
Identifier "NVIDIA Corporation NV40 [Geforce 6800]   0 "  
     Driver      "nvidia"
     BusID      "PCI:3:0:0"
     Option      "RenderAccel"                        "true"
     Option      "AllowGLXWithComposite"       "true"
     Screen      0
EndSection

Identifier "NVIDIA Corporation NV40 [Geforce 6800]   1 "  
     Driver      "nvidia"
     BusID      "PCI:3:0:0"
     Option      "RenderAccel"                        "true"
     Option      "AllowGLXWithComposite"       "true"
     Screen      1
EndSection
                                                 

Section "Monitor"
You need one Monitor section per monitor you have attached to your computer. Give each one a sensible and unique Identifier value: 
                                        EXAMPLE
Section "Monitor"
             Identifier      "Main Monitor"
             Option          "DPMS"
             HorizSync      28-40
             VertRefresh   43-60
EndSection

Section "Monitor"
             Identifier      "Second Monitor"
             Option          "DPMS"
             HorizSync      28-40
             VertRefresh   43-60
EndSection

Section "Screen"
In X terminology, a screen is a combination of a device and a monitor. Sreen sections have Device and Monitor values that refer to the appropriate section by name:
                                     EXAMPLE
Section "Screen"
	Identifier	"Default Screen"
	Device		"ATI Technologies, Inc. Rage Mobility P/M AGP 2x"
	Monitor		"Generic Monitor"
	DefaultDepth	24
SubSection "Display"
	Depth		    24
	Modes		"1600x1200"
EndSubSection

Section "Screen"
	Identifier	"Default Screen"
	Device		"ATI Technologies, Inc. Rage Mobility P/M AGP 2x"
	Monitor		"Generic Monitor"
	DefaultDepth	24
SubSection "Display"
	Depth		    24
	Modes		"1280x1024"
EndSubSection
EndSection

Hopefully, that explains what your trying to do...
