### Post by 3rdalbum on 2006-12-16
The different media players use different backends, which use different codecs, and so they act differently.

I don't know which Totem you are using, but if you've installed Totem-Xine (Automatix does this) that would explain why it performs the same as Kaffeine (which uses Xine). Gxine shouldn't have any problem playing DVDs, but sometimes it crashes when playing Quicktime files.
