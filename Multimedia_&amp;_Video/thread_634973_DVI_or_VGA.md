### Post by Yellow Pasque on 2007-12-08
I have problems with the radeonhd driver detecting my Vx2025wm through DVI as well, so you're not the only one bitten by this monitor's poor DVI EDID implementation.

IMO, there is a noticeable difference between DVI and analog VGA at the native 1680x1050 resolution, but I'm not sure if you're going to be able to do anything about it. If you're going to use the analog, the best tip I can give you is to use the auto image adjust and then follow it with the fine-tuning control.
