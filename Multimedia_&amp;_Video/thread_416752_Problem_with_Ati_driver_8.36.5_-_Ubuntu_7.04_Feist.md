### Post by MindRiot on 2007-04-21
Under /etc/X11 there should be a file called either xorg.conf.original-0 or xorg.conf.fglrx-0.  Either one is a backup of the xorg.conf file made by aticonfig.  Simply sudo cp /etc/X11/xorg,conf.original-0 /etc/X11/xorg.conf to recreate the xorg.conf file. Open the xorg.conf file and make sure you see:

```
Section "Device"
	Identifier	"Generic Video Card"
	Driver		"fglrx"
	Busid		"PCI:1:0:0"
EndSection
```

and

```
Section "Extensions"
	Option		"Composite"	"0"
EndSection
```

You can also boot into a terminal session and reinstall the kernel modules and then run aticonfig --initial and aticonfig --overlay-type=Xv 
Make sure your not in an xsession.
