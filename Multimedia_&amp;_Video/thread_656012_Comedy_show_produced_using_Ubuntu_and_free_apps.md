### Post by nexu56 on 2008-01-02
Hello Ubuntonians

As a personal project I put together a comedy sketch show with my friends. I just posted the first episode, and it was produced entirely using Ubuntu and free software. The production values are pretty low, I'm certainly no Speilberg, but I reckon its not bad for a first attempt! Here's the link (NB: shameless plug): [http://www.midnight-haircut.com](http://www.midnight-haircut.com)

I used Kdenlive for editing, which has a superb iMovie-style interface, but the version in the Gutsy repos is not very stable. Maybe because its QT and I use gnome (where's the GTK equivalent? Whatever happened to Diva?) 

Luckily Kdenlive has an autosave feature, which I soon set to 1 minute :-) It also helped that I soon learned what kind of actions would cause it to crash (such as making changes to the timeline while playing video) and avoided them.

I looked at Kino but the interface just seemed very unnatural to me. It has some great effects and titling features though. Cinelerra seems buggy as hell.

I was wondering if anyone else had attempted a similar project using Ubuntu / Free software, and could share some tips? In particular I'm wondering if there's any effects processing software out there, like Adobe's AfterEffects?

cheers!
