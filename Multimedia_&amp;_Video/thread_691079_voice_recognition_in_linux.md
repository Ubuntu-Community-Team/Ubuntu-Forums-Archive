There are only a couple of possible options for you to get good voice recognition in Linux:

Install Dragon Naturally Speaking using either Wine or Crossover Office.
Use VirtualBox or a similar virtual machine to install a copy of Microsoft Windows within Ubuntu Linux. Vista (and I think XP) has voice recognition built-in.
