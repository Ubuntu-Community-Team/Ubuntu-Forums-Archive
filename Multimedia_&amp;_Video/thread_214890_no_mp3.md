### Post by zerwas on 2006-07-13
Open a terminal and paste this:
```
wget http://easyubuntu.freecontrib.org/files/easyubuntu-3.021.tar.gz
tar -zxf easyubuntu-3.021.tar.gz
cd easyubuntu
sudo python easyubuntu.in
```
