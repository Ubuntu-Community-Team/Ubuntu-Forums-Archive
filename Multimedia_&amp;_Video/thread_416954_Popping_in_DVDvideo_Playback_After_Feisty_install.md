### Post by omegamormegil on 2007-04-22
I don't know if it will help any, but it seems some other people (and me) were getting popping/cracking audio while using VLC after getting Feisty.  The fix was changing the default sound output module from ALSA to OSS.  It fixed it for me, but again, this is in VLC.  I don't know how to change it in your software.  Hope that helps.
