### Post by mmorgan27 on 2007-01-19
You can use the  **artsdsp**  application.  Make sure that you have artsd running: **ps axu | grep artsd**
Then you can run the application like '**artsdsp ** *application*'

Note: It looks like artsdsp may have a bug when parsing and passing arguments with spaces (even if you quote the arg).
