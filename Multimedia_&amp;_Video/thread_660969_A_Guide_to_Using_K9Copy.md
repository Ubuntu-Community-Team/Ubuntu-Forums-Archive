A guild to Using K9Copy


First, in most Countries it is Illegal to backup DVD movies. In some Countries it is OK to make ONE backup but you have to have the original.  

After almost two months of frustrating experience with using K9Copy I have come up with this guide. 

My problems started after a Gusty upgrade from Feisty.  K9 was working, I think. The reason for the wondering is I only did a few movies. After the upgrade I tried to backup a movie and it did not work. This started my quest!

So what broke after the Gusty upgrade. I don’t know! But, here is what I had to do to fix it. I did a NEW install of Gusty! Tried K9 1.1.0 and K3B and it did not work, at least not in my players. So with allot of help from people on this form I upgrade to K9 1.2.2. There was some improvement but still not right. So what I noticed was an OPC error during the burn. This is not good! I changed the DVD drive and now everything is working. I tried the old drive in a Feisty machine and it also has the OPC errors. DID Gusty break the DVD drive? I don’t know. The drive works with Windows just fine. This drive is a Plextor with the latest firmware upgrade. 

If you are having problems with K9Copy 1.1.0 and K3B here is what I would suggest you do:

Make sure you have a good working machine. Solve all other problems first. Have all the correct codices and dependancies installed so you can watch DVD movies on your machine.  There are post all over this form for that.

Make sure you have lots of free space on you hard drive. A double layer movie is over 8gig. To use a 4.7 gig DVD backup you may need 8gig fee space to get the job done! The more the better!

If you are using eye candy desktop, turn it off until you get everything working. Then you can try turning it back on. I'm not sure how you do this as I don't use it. 
    
If you are still having problems, try a new DVD burner. I used Light-On. If you are using a Laptop, sorry you may be out of luck.

DON’T jump to installing K9Copy 1.2.2!  Make sure you have covered all the suggestions listed here first. It could save you time and frustration.  If you need to install K9 1.2.2, then check for posts here. There are some really good guides.

Using K9:
To backup a complete disk, put a check in “Keep All Menus”
If you use K3B to burn, and I suggest you do, under K9 setup put a check in “Burn with K3B” and Uncheck “Use K9 to Burn” This wording may not be correct but you can get the idea.

When the Rip/Transcode is completed K3B will start and place the proper folders in a project. You need not do anything here. Right click on the desktop movie icon and eject it. Then put in the blank disk and wait for it to be detected (mounted). Say no to the question box that comes up about making a DVD disk. Wait about 10 seconds then click on the Burn Icon on the K3B project box. It will be about half way down on your left.  When the burn starts a Box will come up to ask if you want to Kill K9 (shut it down). It gives you three options. Be careful here. It depends on what Version of K9 you have installed. If you kill K9 it may delete your Ripped files. No files no DVD. I suggest you chose “Continue”. I have tested this and it seams to work. When the backup is completed and you have tested it. You should delete the Ripped files as this space will be needed for the next backup. Or you can move them if you want.
