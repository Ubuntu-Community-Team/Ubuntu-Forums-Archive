All the information you need is here: [https://help.ubuntu.com/community/CDRipping]("https://help.ubuntu.com/community/CDRipping")

SoundJuicer will work fine and once set up is simple and effective to use.  There are many alternatives such as Asunder, Ripit, Ripper-x, Grip, K3b etc all of which will do the same job with different degrees of complexity/choice in the user interface. Possibly the the nicest to use (clear iinterface, easy to set up, excellent default options) is Asunder, which to some extent resembles SoundJuicer but can do a little more (encode to multiple formats simultaneously)  while remaining supremely easy to use.
