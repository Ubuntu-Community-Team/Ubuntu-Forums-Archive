### Post by croak77 on 2006-07-10
> **Dubbayoo said:**
> I can't figure out how to edit multiple files at once. I have highlghted some files and I want to assign an artist to them all but it only does the first. whats the deal.

All you have to do is highlight the songs you want to change ( Shift-Left Click first to last song or Ctrl-Left Click each song you want). Edit the artist field. Then check the box next to it (there is a mouse-over balloon that pops up that says "Tag selected file with this artist"). Hit the save button or Ctrl-S.
