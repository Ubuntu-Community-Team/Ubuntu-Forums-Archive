### Post by garybrlow on 2007-01-22
You probably have the NoScript Plug-in installed on Firefox. Allow the site or any particular flash animation etc. via NoScript. By default NoScript blocks flash animation, javascript and java applets. You can do so by right clicking on the NoScript icon on the status bar. If not found, you can always hover your mouse over the spot where the flash animation should be, right click then access the NoScript options to allow the site/URL. :)

Cheers!
