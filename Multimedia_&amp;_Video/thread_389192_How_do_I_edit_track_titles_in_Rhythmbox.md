I began importing my files and naming them in Banshee, where editing track titles is very very easy (open a set of songs, and all you have to do is type in the song name and press enter, repeating for as much of the selection as you please). However, I prefer iPod support and stability that I get from Rhythmbox, so I've been ripping with Soundjuicer (works great) and auto-importing the tracks via my Music folder into Rhythmbox.

However, it seems that the only way I can edit track titles in Rhythmbox is to right-click and select "properties" which is very, very tedious. There has to be an easier way! How though?

### Post by aysiu on 2007-03-21
You can always edit the info in Banshee (since you seem to like it) or TagTool (an ID3 tag editor that does a lot of bulk operations). Any changes you make in Banshee or TagTool should also apply to Rhythmbox. The tags live in the song files themselves--not in the player.
