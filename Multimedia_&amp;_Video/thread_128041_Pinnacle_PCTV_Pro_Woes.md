### Post by deadgobby on 2006-03-18
Question
  	Linux driver for Pinnacle PCTV products
  	Answer
  	We get more and more requests on LINUX support for products of the PCTV line. Pinnacle does not develop PCTV drivers or applications for LINUX. However, we keep contact to several external developers and provide them with hardware samples and technical information. Of course, we can’t provide any technical end user support on this 3rd party software and we are not responsible for any potential malfunction or damage caused by the installation of these drivers and applications. Support for PCTV products can be found at the following URLs:
PCTV/pro/RAVE: <http://bytesex.org/bttv/>
· PCTV Sat <http://www.pctvsat.com/html/linuxtv.html>; [www.linuxtv.org](www.linuxtv.org) <http://www.linuxtv.org>
· Bungee & PCTV Deluxe („Dune“) Linux tools: <http://www.paranoyaxc.de/bungee/dirlist.html>
· Bungee Linux Tools Sourceforge: <http://sourceforge.net/projects/bungee4linux/>
· Bungee USB Linux Driver Sourceforge: <[http://sourceforge.net/projects/pvrusb](http://sourceforge.net/projects/pvrusb)
