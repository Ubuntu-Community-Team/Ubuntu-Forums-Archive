### Post by SeattleUser on 2007-07-01
I've had trouble burning a video dvd using video from my minidv camcorder.  I previously used Kino to create a raw dv file, DeVeDe to create an iso and file manager to burn the dvd.  The dvd worked OK on the PC but the video had pixelization and sound problems from a stand alone dvd player.

I upgraded from Edgy to Feisty.  Then I ran Kino and opened the previously created dv  file.  Before the upgrade Kino would crash at that point.  I used the help documentation of Kino to proceed.  I used  export function and selected the mpeg and dvdauthor options.  That generated an mpeg file and an xml script.  I then ran dvdauthor as instructed by the help document.  I then  tried to run growiso from the command line, but it couldn't access my dvd drive.  So then I tried using the file manager instead.  I reinserted the blank dvd and file manager asked if I wanted to burn a dvd.  It burned a dvd  that works fine in both a Windoze PC and a regular dvd player.

Now I just need to learn how to do editing.  I've got a bunch of family videos that I need to condense.

Ubuntu and Linux rules!!!
