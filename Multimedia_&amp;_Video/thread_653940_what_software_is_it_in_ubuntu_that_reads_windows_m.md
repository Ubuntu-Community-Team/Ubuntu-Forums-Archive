### Post by rainwalker on 2007-12-31
You need to install w32codecs to play windows media files. Applications > Add/Remove and install "Ubuntu restricted extras"
Amarok doesn't play vids, but works with both Kubuntu and Ubuntu.
I think VLC will play just about anything, as long as you have the codecs installed.
Flash is working for me, I believe it's installed when you install the restricted extras
I think Evince is the pdf program with Ubuntu, but I'm not sure
