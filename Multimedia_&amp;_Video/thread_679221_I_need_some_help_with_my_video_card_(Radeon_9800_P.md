### Post by Tatty on 2008-02-01
Hi mr helton,

I have never installed these ATI drivers before myself, but I will try to take you through this guide step by step: [https://help.ubuntu.com/community/BinaryDriverHowto/ATI](https://help.ubuntu.com/community/BinaryDriverHowto/ATI)

Im going to assume you are using a standard Ubutnu Gutsy install.  And im also going to assume you are using the Gnome desktop (the standard Ubuntu desktop - comes up looking brown)

1. Open a terminal by going to Applications->Accessories->Terminal

2. Type the following code into the terminal (or copy and paste if you want) :
```
sudo apt-get update
```
Then press enter.
This makes updates the package manager so it knows what is in the Ubuntu repositorys

3. Now type this into the terminal:
```
sudo apt-get install linux-restricted-modules-generic restricted-manager
```
Then press enter.
This installs the drivers that your graphics card needs.

4. Next we need to activate the drivers so Ubuntu knows to use them.  To do this click "System->Administration->Restricted Drivers Manager".  Then when the window loads up it should have listed the option "ATI accelerated graphics driver".  Simply click on the square box to the left of this (so the box is ticked).

5. I assume you will probably now need to reboot your computer.

You might be able to simply restart X instead of rebooting entirely (press ctrl-alt-backspace), but im not entirely sure, someone else may have to correct me on this.

To play it safe simply restart your machine.

6. Your drivers should now be loaded and working correctly.  To check this open a terminal and type:
```
glxgears
```
Then press enter.
This should display some coloured moving cogs which are turning.  every 5 seconds the terminal will display the framerate that they are being displayed at.  
Im not sure what kind of framerate you should be getting with your card, but if its over 1000fps then its probably fine.  if its something silly like 200/300fps then something has gone wrong.

As i say, i dont have any ATI cards in any of my machines at the moment so i cant test this out myself.  But as far as i can tell this should work.

**EDIT:** I just re-read your post and saw that you are already using the restricted drivers. Sorry! :-) But at least you could use this to check over all your steps to make sure you havent missed anything.
If it is still not working after this then could you post your xorg.conf file - found in /etc/X11/xorg.conf.
