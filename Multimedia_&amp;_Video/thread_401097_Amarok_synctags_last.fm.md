### Post by qurk on 2007-07-18
Heya.  I have been wondering the same thing myself.  Well, labels are a sortof new addition to Amarok, so not a whole lot with them.  But I have found a couple of useful ways to utilize the labels.  The first is with smart playlists and dynamic playlists.  You can set up a smart playlist using a particular label, is useful if you have a lot of songs with weird labels or you just want one of the myriad weird genres people have tagged stuff with on last.fm (if you are using synctags).  Secondly, if you have it set to show the labels in your context browser, as the song is playing you can click on a particular label, and it shows you all the other songs labeled the same in your collection, without stopping the current song.  Hope that offers some insight, and hope you can find other ways to use the labels too :)
