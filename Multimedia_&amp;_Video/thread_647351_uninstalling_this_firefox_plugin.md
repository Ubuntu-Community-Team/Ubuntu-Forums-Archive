How do I uninstall the xine plugins for firefox and make it use MPlayer instead?

---

### Post by z0mbie on 2007-12-22
1. Open **Synaptic Package Manager** via *System* > *Administration*

2. Click *search*, type in "**xine-plugin**" and *Right Click* for removal.

3. Search for "**mozilla-mplayer**" and *Right Click* for installation.

4. Click *Apply*. 

Now Firefox should use mplayer as a plugin instead of xine.
