### Post by mouseboyx on 2007-11-24
Install DeVeDe from the synaptic package manager in System > Administration > Synaptic package manager.

It is meant to combine video files to burn to a dvd but you can make it combine it all into a mpg file if you tell it to leave temporary files alone.

It should be in Applications > Sound & Video > DeVeDe
otherwise press Alt+F2 and type devede
