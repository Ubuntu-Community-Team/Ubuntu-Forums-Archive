I have a Sapphire ATI Radeon HD 3650. I'm using the fglrx driver. Easiest way to install this is "sudo apt-get install xorg-driver-fglrx".

I don't know how reliable this driver is on Feisty, but they've recently released a newer version of it. I'm using Hardy and it works without question.
