Hi!
To install it, you have to go to **System/Restricted Drivers Manager ** and check it's box. That should do it, although not to get Compiz.
If you want to enable Compiz, you'll need to install XGL too. To do so, go to **System/Synaptic Package Manager** and look for **xserver-xgl**, or open a console and type:
```
sudo apt-get install xserver-xgl
```
I hope this helps!
Good luck!
