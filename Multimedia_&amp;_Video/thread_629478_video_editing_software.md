### Post by Unterseeboot_234 on 2007-12-02
I've been very impressed with Cinelerra. I recommend you check out youtube, do a search for cinelerra and some of the other softwares. There will be some HowTos and lots of demos. But, I will also tell you I run AMD 64-bit hardware with a mid-level nVidia graphics card and I don't seem to have as many crashes I read about from others on these forums using cinelerra. The other thing is, I've found I have to have Kino, Avidemux, Kdenlive and some of the others as tools for file conversions. Cinelerra works best as DV(raw) film format. Also, to use Cinelerra takes some research (lots of reading up in the docs) about what codecs do what. You also need the command-line tools ffmpeg and mencoder.

Cinelerra is not in the 'buntu downloads. It is a little tricky installing. I used the advice from this forum thread

[**Install Cinelerra**]("http://ubuntuforums.org/showthread.php?t=596160")
