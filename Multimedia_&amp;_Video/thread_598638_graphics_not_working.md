### Post by Yellow Pasque on 2007-11-01
Add the following to the end of your xorg.conf
```
Section "DRI"
	Group	0
	Mode	0666
EndSection
```

Add this block after the "file" section:
```
Section "Module"
	Load		"fglrx"
	Load		"dbe"
	Load		"extmod"
	Load		"fbdevhw"
	Load		"glx"
	Load		"record"
	Load		"freetype"
	Load		"type1"
	Load		"dri"
EndSection

Section "ServerFlags"
	Option		"AIGLX"	"off"
EndSection
```

See if that helps.

### Post by Yellow Pasque on 2007-11-01
OK, I'm sorry. I forgot that the new ATI drivers are out (probably because I gave up on them and went back to 8.40.4).

Change to:

```

Section "ServerFlags"
	Option		"AIGLX"	"on"
EndSection
```

and

```
Option "Composite" "Enable"
```

Also, you need to edit your /etc/xdg/compiz/compiz-manager and add the fglrx driver to the list of whitelisted modules, i.e.:

```
WHITELIST="fglrx"
```

Another minor thing - your monitor probably doesn't support 1680x1680, so remove that res from your xorg.conf
