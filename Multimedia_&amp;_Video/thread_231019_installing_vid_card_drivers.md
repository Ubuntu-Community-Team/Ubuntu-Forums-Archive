### Post by Ziox on 2006-08-07
if you want to know what kind of card you have this command should tell you:

and to download, you need to find the file from this site:

[https://support.ati.com/ics/support/default.asp?deptID=894](https://support.ati.com/ics/support/default.asp?deptID=894)

after you found the download page, download the file:

ati driver installer
