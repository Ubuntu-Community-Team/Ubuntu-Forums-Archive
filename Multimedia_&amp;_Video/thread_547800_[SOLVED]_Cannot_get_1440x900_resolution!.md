### Post by elliotjreed on 2007-09-11
Wow! I got it to work!

Just did (one after the other):

[PHP]sudo apt-get update

sudo apt-get install linux-restricted-modules-$(uname -r) 

sudo apt-get install xorg-driver-fglrx

sudo depmod -a

sudo aticonfig --initial

sudo aticonfig --overlay-type=Xv[/PHP]

Then downloaded: xorg-edit from [http://sourceforge.net/project/showfiles.php?group_id=169700]("http://sourceforge.net/project/showfiles.php?group_id=169700")

Then changed all the settings to ATI and added 1440x900 in the monitor bit!

Restarted, then went to System > Preferences > Screen Resolution, *et voila!*

### Post by shakyone on 2007-09-11
I don't have an ATI card, but I got this resolution working on my Dell D620's intel Graphics card.  Maybe it will work for you.  
I did a fresh install via live CD.  

When I rebooted I ran the update manager to download all the updates to the system.  Nothing else.

Then I rebooted, since update manager asks you to.

Then I opened Synaptic and selected one package for install, the 915resolution package and "Apply"

Closed Synaptic.

Then I restarted the computer.

My X-server reconfigured the Xorg.conf file for me at the logon.  The resolution defaulted to the 1440x900 resolution automatically.  I don't know how this would work for you, but it is worth a try.  

I am guessing somewhere in there you need to install the fgrlx package and configure it.  I dunno for certain.  I never tried it on an ATI card.

Good luck.
