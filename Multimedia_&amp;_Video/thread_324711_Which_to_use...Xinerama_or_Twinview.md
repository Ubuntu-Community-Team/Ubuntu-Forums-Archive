### Post by Cene on 2006-12-24
TwinView is only for NVIDIA cards.
You can use Xinerama (2 seperate desktops, cannot move windows between them) or MergedFB, wich is similiar to TwinView (one big desktop, possible to move windows between them).

[This](http://www.ubuntuforums.org/showthread.php?t=221174) thread might give some extra information.
