### Post by nedecor on 2007-05-26
> **ratbagradio said:**
> The newly released UbuntuStdudio requires what?

(1)Feisty -- it's a Feisty package? Right?
(2) What hard drive reserve/space ram and whatforth can I get away with?
(3) what a preferred partitions if any?

1. You can convert a Feisty Install into an Ubuntu Studio Install, but it's not exactly one "package". Directions can be found here [https://help.ubuntu.com/community/UbuntuStudio/UpgradingFromFeisty](https://help.ubuntu.com/community/UbuntuStudio/UpgradingFromFeisty) Or you can just do a clean install using the DVD available for download at [http://ubuntustudio.org](http://ubuntustudio.org) I recommend teh clean install because it provides a better default configuration geared towards multimedia use that the default Feisty install does not have.

2. A clean install with all options selected I think takes up 1.5 gig. But I don't remember, I didn't check right after I installed. I'd recommend at least 512M of RAM if you plan on doing serious audio and video work.

3. At the very least, a root partition and a swap partition, just like any other distro. A lot of people like breaking their /home directory onto a separate partition, but it all depends on taste.
