### Post by hooplah on 2008-02-17
Which plug-in did you install? There are two available, you should get the Adobe one. If all else continues to fail, remove the installed plug-in, and then proceed to the official Adobe site and get yourself the tar.gz package for the flash plug-in. Extract the file, and run the installer. Upon restarting Mozilla Firefox, you should have the plugin properly installed. To check if it installed or not, either run another youtube video, or simply go to about:plugins on the URL bar.
