### Post by dolson on 2006-03-22
Here's a compilation of songs that were all done with Linux. You can get them in Ogg format:

[http://linuxaudio.org/en/music/index.html](http://linuxaudio.org/en/music/index.html)

Be sure to look through the README file there, as you'll see some of the conditions used to make these songs. For instance, track #3 was done with a Pentium 166Mhz! It also has links to artist homepages.

I had a listen, and some are very professional-quality, and others are very interestingly crafted. All in all, a very cool overview of many bands that used open source. :D
