### Post by ericesque on 2007-11-13
Don't forget there's always the virtualization option.  Keep a slim XP install virtualized so that you can start it up right inside Ubuntu.  From there, you can install programs like iTunes that aren't particularly Wine friendly.  Quick sync up your touch and shut the VM down.

For more information on virtualization, search Virtualbox or VMware on the forums.  There is a current thread discussing the best way to run XP inside Ubuntu out there.
better yet, here's the virtualization forum:
[http://ubuntuforums.org/forumdisplay.php?f=308](http://ubuntuforums.org/forumdisplay.php?f=308)
