### Post by opticyclic on 2006-02-26
After hours (days?) of searching, I have found a solution. :D 
The full DSSPlayer that comes on CD with the player won't run in Linux. 
However, the DSSPlayerLite (downloadable from the Olympus website  [http://www.olympus.co.uk/consumer/2590_4694.htm](http://www.olympus.co.uk/consumer/2590_4694.htm)) will install and run when using wine.

I also found an application to convert the DSS format.
It is called Switch [http://www.nch.com.au/switch/](http://www.nch.com.au/switch/) and it runs perfectly under wine.


Goes to show that if you look hard enough you will eventually find what you are looking for!
Thats one more thing I can cross off my list for needing Windows for! \\:D/
