### Post by steefjeqv on 2007-12-29
[http://gatos.sourceforge.net/overview.php]("http://gatos.sourceforge.net/overview.php")

This should be available through your Synaptic Manager. You'll need to enable the "universe" repo's.
Just search for "Gatos".
Note : I don't think it's available in the 64bit version of Ubuntu.
Don't forget to uninstall the 7.1 ati driver as the gatos project uses the open source driver.
I'm not sure if it'll work for your 9600, but it's worth a try.

Support for the AIW series from AMD/ATI is nearly zero for the moment. I hope (and wish) that AMD is going to give the gatos project all the backup it needs.
