### Post by Gigaplex on 2008-05-05
> Option "MonitorLayout" "LFP,DFP"
That syntax is valid for the i810 driver, not for the intel driver. The intel driver supports RandR 1.2, try using that instead for dual screen support. Currently the intel driver has a regression for Xinerama and will crash if you try to enable it.

[http://intellinuxgraphics.org/dualhead.html](http://intellinuxgraphics.org/dualhead.html)

[http://www.thinkwiki.org/wiki/Xorg_RandR_1.2](http://www.thinkwiki.org/wiki/Xorg_RandR_1.2)
