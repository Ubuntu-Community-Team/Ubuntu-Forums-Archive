### Post by Kace on 2006-01-17
You should have Ubuntu on a cd-rom, if you don't, I would suggest burning it to one, and if you want to earse windows and re-install both Ubuntu and windows 2000 then when you start the computer up, just make sure that the first boot device is the cd rom and whichever disk you have in there will load up and give you an option for what partition you want to install it on. If you want to keep your current OS of windows 2000 without erasing it and you want to install Ubuntu on another partition just do the same. Make sure the computer reads the cd-rom drive first, and have the Ubuntu disk in the drive. Some computers made need additional configuring to get this to work. Make sure you pick the partition that windows IS NOT on. Hope this helps.
