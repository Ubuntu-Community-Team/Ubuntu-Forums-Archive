### Post by evolutionx on 2007-05-17
On my machine I use EasyTag for all my ID3 tag editing.   It allows you to grab all the song information for each CD and update the tags and you can also rename files, create play lists etc.

I also use Exaile media player for playing my music collection.  It has a feature which grabs the CD cover art off Amazon and it also opens a mini Wikipedia page for more information on artists and albums, which I found to be a nice touch.

Happy Ubuntuing!
