### Post by meng on 2006-08-15
I understand it's important to have a compatible video card. Mine is an NVIDIA. Here's how I enabled the swivel/pivot:

1. Install xrandr
2. Install gnome-randr-applet
3. Add to the gnome panel > Utility > Display Geometry Switcher
4. Edit xorg.conf and add following line under Device (video card)
Option "RandRRotation"
5. Restart
