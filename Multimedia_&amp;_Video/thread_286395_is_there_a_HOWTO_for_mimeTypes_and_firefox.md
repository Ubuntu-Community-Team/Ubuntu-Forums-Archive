### Post by moini on 2006-10-28
firefox uses the normal mime type of the system. so if you use gnome you can change the mime type with the normal file-browser.

for example you want to start a mp3 stream with xmms, just create a dummy-file test.pls. then change the mime type with the file-browser.

this also affects firefox.
