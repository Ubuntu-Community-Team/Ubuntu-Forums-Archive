### Post by shirshasin on 2007-11-25
Install gxine with libxine. This is a complete multimedia package. You can effortlessly watch DVD/VCD and other popular audio/video formats. Open Synaptic and install the following packages(just search for libxine in Synaptic and the necessary list will come up!)

[B]gxine
gxine-plugin
libxine1
libxine1-console
libxine1-ffmpeg
libxine1-gnome
libxine1-plugnins
libXinerama1
libxine1-xvdr
[/B]
Enjoy :guitar:
