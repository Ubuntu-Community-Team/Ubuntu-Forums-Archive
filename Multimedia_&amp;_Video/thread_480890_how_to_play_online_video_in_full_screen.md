### Post by spartan777 on 2007-06-21
you can go get automatix here if you use the regular version of Ubuntu Feisty Fawn:

[http://www.getautomatix.com/apt/dists/feisty/main/binary-i386/automatix2_1.1-4.7-7.04feisty_i386.deb](http://www.getautomatix.com/apt/dists/feisty/main/binary-i386/automatix2_1.1-4.7-7.04feisty_i386.deb)

Once you install  it, go to application, then system tools, and click on automatix. tell automatix Yes or whatever it needs, then go under "Codecs and Plugins." Check the boxes by "Mplayer and FF plugin," and "Flash Player." Be sure Firefox isn't already running, then at the top tell it to "Start." You should then be able to play any youtube/google video, along with anything else that uses flash. 

Automatix is a great tool to install other things too. In "Codecs and Plugins" you can do the same thing as above to install "AUD-DVD Codecs" in order to watch DVD's or "Multimedia Codecs" to listen to mp3's, though if you try to play an mp3, ubuntu now can most likely get the right codec, so try that first. If ubuntu can't help, then use automatix.
