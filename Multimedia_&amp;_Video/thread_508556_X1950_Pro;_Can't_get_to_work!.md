### Post by bourger on 2007-07-28
Well, I was almost completely desperate to properly install the drivers for my X1959 Pro, killed heaps of time but... at last I've done it!
What did I do:
1. Made a clean installation of 7.04.
2. Made a full update (about 102 updates).
3. Installed the newest driver from ATI site - in automatical mode.
4. Enabled ATI proprietary driver from System>Administration>Proprietary Drivers Manager
5. Added to /ets/X11/xorg.conf these lines: ```
Section &#8220;DRI&#8221;
Mode 0666
EndSection

Section &#8220;Extensions&#8221;
Option &#8220;Composite&#8221; &#8220;0&#8243;
EndSection

Section &#8220;ServerFlags&#8221;
Option &#8220;AIGLX&#8221; &#8220;off&#8221;
EndSection
```
6. Did everything from [here](http://wiki.cchtml.com/index.php/Troubleshooting).

In general, all the process is described [here](http://wiki.cchtml.com/index.php/Ubuntu_Edgy_Installation_Guide).

Some hint: reboot as often as possible, to be sure - after each step.
I still can't get Beryl and desktop effects working, but hope to settle this matter somehow...
