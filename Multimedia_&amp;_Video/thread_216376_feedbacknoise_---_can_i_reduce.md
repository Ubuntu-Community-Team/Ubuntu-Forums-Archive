### Post by zettberlin on 2006-07-15
If the noise is a hum you should check your cabels both powersuplies and USB/Network (if there are any).
Rule of thumb is, do everything powered with either batteries or one single fuse - there is an effect of the DC-Powerline called humloop that is invoked often if you use two different mainlines for a setup with guitars, Mics etc.
Try to run everything from a single DC-Powerlineconnector and look if it helps.
