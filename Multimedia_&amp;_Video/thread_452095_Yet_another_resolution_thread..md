### Post by benanzo on 2007-05-23
What is the highest resolution you're currently able to use?  is it 1024x768 or 1280x1024?

Either way, here are the steps to fix it:

You need to install the package *915resolution*.
```
sudo apt-get install 915resolution
```
or search for 915resolution in Synaptic.

This package changes the BIOS modelines that the X server gets from your graphics chip.  Unfortunately a whole crop of bugs have appeared since the introduction of these new widescreen monitors that are being dealt with carefully (so until we get fixes, we need this utility.)

First, determine your current resolution by looking at **System** -> **Preferences** -> **Screen Resolution**

Now do this in a terminal:
```
/usr/sbin/915resolution -l
```

This will print the available resolutions for your graphics chip beside the current mode their bound to.  Find your currently running resolution, (whether 1024x768 or otherwise) and take note of the mode it is bound to.

Now you need to edit the */etc/default/915resolution* file to change the default resolution to the one you want:
```
sudo gedit /etc/default/915resolution
```

change:
```
MODE=auto
``` to the mode of your current resolution.
it might be 45 so this is what that line would look like:
```
MODE=45
```

Now change the XRESO and YRESO from:
```

XRESO=
YRESO=

```
to
```

XRESO=1440
YRESO=900

```

Leave the
```
BIT=
```
line empty.

Now reboot and you should be at 1440x900
