### Post by superm1 on 2007-05-22
I would say this is most likely a cause from USB1.1.  The system should be able to handle playback of the files.  If you would like to verify this to be the case, copy over a recording to your hard drive.  Launch playback like this on the command line:

```
mythtv FILE
```

This will launch the file directly into the mythtv viewer with all mythtv settings.
