If you go to youtube it will direct you to the plugin. You can check out this thread though that will set you up with all the codecs plugins and peripheral programs you need.
[http://ubuntuforums.org/showthread.php?t=661833](http://ubuntuforums.org/showthread.php?t=661833)
read carefully a lot of the stuff is in synaptic so if you are not familiar with the terminal you can find them there. also you will need all the gstreamer from add remove and I have all of these installed to read various types of programs.
[https://addons.mozilla.org/en-US/firefox/browse/type:7](https://addons.mozilla.org/en-US/firefox/browse/type:7)
