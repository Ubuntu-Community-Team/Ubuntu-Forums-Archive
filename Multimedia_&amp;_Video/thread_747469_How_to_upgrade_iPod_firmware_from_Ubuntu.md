However, all I have readily available to me is my Gutsy laptop. I haven't found any instructions on these forums or on the web for how to update the official firmware from Apple on a Linux machine. Response from Bose:

I had to borrow a Windows laptop to use iTunes to upgrade the iPod software, but afterwards it does seem to be working correctly. It'd be much more convenient to be able to upgrade directly from Linux, though!
