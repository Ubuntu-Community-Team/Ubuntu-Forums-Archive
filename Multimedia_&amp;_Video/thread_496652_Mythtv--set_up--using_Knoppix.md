I haven't used the newest version; however, I did try an earlier version that allowed me to set up mythtv when I was unable to do so on any other operating system. It was still an experience that involved a fair bit of user googling. Also, the picture quality was not the best; however, that has improved greatly on both MythDora and other versions of Linux due to improved driver support, so I'm sure Knoppmyth has followed suit. I would have stayed with this flavor of MythTV, but I was looking for something that afforded me the ability to do video editing easily. Googling certain issues brought me to Ubuntu and I have never regretted switching. I would like to say that if it wasn't for Knoppmyth's great following, I probably would have given up on MythTV as a PVR alternative. 

If you are having trouble getting MythTV working in Ubuntu, do not be afraid to test other flavors. This will only help to improve the PVR experience of all.
