### Post by gorgon on 2008-04-22
Hi,

KINO converts compressed files to DV when you open them, (including mpeg), but if you for some reason don't find your file supported you could always get an other program to do the conversion, since KINO will want the file in DV anyways.

There are instructions on adding Cinelerra to your repostitory and installing here:
[http://www.linux.com/feature/119570](http://www.linux.com/feature/119570)

Good luck!
