### Post by ccav01 on 2007-12-18
In the device section, the "screen" parameter has nothing to do with the screens defined elsewhere.  It's actually the reference to which output on a dual head card you wish to use.

Section "Device"
Identifier "0 Intel Corporation 82865G Integrated Graphics Controller"
Driver "i810"
BusID "PCI:00:02:00"
Screen 0
Option "DDCMode" "True"
Option "MonitorLayout" "CRT,CRT"
EndSection

Section "Device"
Identifier "1 Intel Corporation 82865G Integrated Graphics Controller"
Driver "nvidia"
BusID "PCI:01:01:00"
Screen 1
Option "DDCMode" "True"
Option "MonitorLayout" "CRT,CRT"
EndSection

Unless these are dual head cards, you don't need the Screen parameter.  Since each card has it's own BusID, I'm assuming they are single head cards.  Remove the Screen parameter and it should work.
