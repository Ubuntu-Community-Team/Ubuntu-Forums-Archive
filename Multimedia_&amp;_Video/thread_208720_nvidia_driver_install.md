### Post by Dr. Nick on 2006-07-04
OK so if you get one of them blue screens about "no useable screens" then you can just login at the command line and run

sudo nano /etc/X11/xorg.conf

Thier should be a section called "driver" and it may say "nvidia" change nvidia to "nv" and save and exit. Doing that should allow X to start back up, but with no acceleration. Then if you could post you /etc/X11/xorg.conf here it may help some.

I havent built the nvidia driver in a while, Ive had good luck just installing the nvidia-glx package to get the driver properly working
