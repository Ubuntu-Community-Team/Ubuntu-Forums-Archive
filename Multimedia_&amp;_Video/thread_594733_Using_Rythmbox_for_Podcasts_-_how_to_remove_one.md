### Post by moshuptrail on 2007-10-28
I'm using Rythmbox to gather several podcasts.  It's pretty simple to use, but I added a Podcast feed that I later realized was not really the one I wanted.  So I went to "remove" it from the list.
As best I can figure, there is NO remove function.  You can add feeds but never remove them.

Is there a reason for this, or am i missing something, or is there a trick?

---

### Post by moshuptrail on 2007-10-29
I guess I'm the only one using Rythmbox.  There are so many other excellent choices!

But I did figure out the answer.

You need to enable the browser view: Click on View -> Browser
Then enlarge the window so you can see all the podcast feeds.
Right-click on the feed and select remove.  Then remove the feed an all files.
