### Post by francky_51 on 2007-11-23
I had the same problem and now it seems to work.
I found a solution on the gentoo's wiki.
[http://http://gentoo-wiki.com/HOWTO_Dolby_Digital_Out_(AC3,_SPDIF)]("http://http://gentoo-wiki.com/HOWTO_Dolby_Digital_Out_(AC3,_SPDIF)")
I created a .asoundrc file in my home directory and put in "pcm.!default spdif" (without quotes of course)
Then I restarted alsa-utils ( sudo /etc/init.d/alsa-utils restart) and sound came back.
I really hope it will not disappear soon.
Hope it helps :)
