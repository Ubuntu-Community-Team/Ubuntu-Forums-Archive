### Post by fenian on 2007-02-06
It's a lot easier to install mythtv now than it used to be but it still takes some effort.This page...[https://help.ubuntu.com/community/MythTV](https://help.ubuntu.com/community/MythTV) ,will give you a good starting point for installing mythtv on ubuntu.Be sure to read the sections on hardware setup ,plugin setup and media playback.While I would not say that it is an easy thing to install ,I don't think it's as difficult as some people make it out to be.Some important questions to answer would be...
What tvtuners work with mythtv?  (I recommend Hauppauge pvr150 but there are other options)
How will you change channels? (IR blaster or serial control)
How much hardrive space do you have/How much do you need? (myth needs some hd space ,1 hr of recorded tv takes up at least 1gb)
