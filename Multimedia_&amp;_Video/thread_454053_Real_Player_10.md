I have downloaded Real Player 10 Gold. The download is suffixed .bin. How do I install the software?

---

### Post by RomeReactor on 2007-05-25
Hi. Open a terminal (Applications-->Accessories-->Terminal) and enter
```
cd Desktop
```
assuming the file is in your Desktop; then
```
sudo chmod +x RealPlayer10GOLD.bin
```
to make the file executable; and to install it
```
./RealPlayer10GOLD.bin
```
