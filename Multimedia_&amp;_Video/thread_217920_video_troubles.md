### Post by RudolfMDLT on 2006-07-23
The GUI interface may not be the right tool to use.

In the terminal type "aticonfig" and press enter. A whole list of possible commands appear. Scroll through the commands. Specifically look at the last bit:

> Examples:
  1. Setting up fglrx for the first time.
       Single head :    aticonfig --initial --input=/etc/X11/xorg.conf
       Dual head   :    aticonfig --initial=dual-head --screen-layout=above
                        This command will generate a dual head configuration
                        file with the second screen located above the first
                        screen.
  2. Setting up big desktop to horizontal and set overlay on secondary display.
                        aticonfig --dtop=horizontal --overlay-on=1
  3. Setting up modes for primary display.
                        aticonfig --resolution=0,1600x1200,1280x1024,1024x768
  4. Force primary CRT on and TV-out off.
                        aticonfig --force-monitor=crt1,notv

Please report bugs to [http://support.ati.com](http://support.ati.com)
rudolf@rudolf:~$


These are examples om how to use the above commands. Choose the set of commands that you think you need, write them down and restart your pc into recovery mode. Log in as root. here again you should be able to enter aticonfig and see the list of commands. Enter in your command that you chose while in the X Windows system. press enter and restart. Hopefully with X not running your configuration settings would be saved... good luck!
