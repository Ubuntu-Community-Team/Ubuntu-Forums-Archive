### Post by paradoxchild on 2007-10-21
Try adding this code to your screen section in xorg.conf,
use either

sudo nano /etc/X11/xorg.conf

if in terminal or 

sudo gedit /etc/X11/xorg.conf

if in gui

then go down to the 'screen' section and add in

Option "UseDisplayDevice" "DFP"

hope that helps
