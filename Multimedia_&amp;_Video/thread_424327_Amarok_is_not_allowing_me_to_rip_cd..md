### Post by petersjm on 2007-08-20
Install the kio slave audiocd - go to the files tab in Amarok, type audiocd:/ and select the songs you want from the CD, then drag them into the "collection" tab and hey presto! Needs to be less complicated, though, if you ask me. And it's not as fast as K3B, either. Make sure K3B's paranoia mode is set to 0 for the fastest rip speed. Hope that helps.
