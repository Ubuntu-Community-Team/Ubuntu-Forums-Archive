### Post by tilapia on 2007-10-19
I had this problem. My monitor (a 24" Dell 2407FWP) was using the wrong refresh rate. I was using this with a GeForce 6600 GT (recognised as 6800) and the nvidia driver. It was defaulting to 53 Hz rather than 60 Hz. I changed it to 60 Hz and the desktop suddenly became the correct size. 

I changed this using the Screens and Graphics GUI, but you could also hand edit xorg.conf if you can't do it that way.

### Post by DoodlesQ on 2007-10-22
I also got Compiz working on both screens. I used nvidia-settings to enable TwinView. I'll be honest, I'm not totally clear on what the difference between Xinerama/Twinview is, but that's what worked for me.

So, in case anyone finds this in the future I fixed it -

- Installing restricted drivers
- Using "screens and monitors" to enable dual monitors, and setting them both to "generic" for the resolution I wanted.
- Then using Nvidia-settings to enable twinview.

Hope that helps somebody. Thanks everybody who pitched in.
