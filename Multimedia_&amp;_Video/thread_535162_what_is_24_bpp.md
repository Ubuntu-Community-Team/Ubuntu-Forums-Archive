### Post by luisromangz on 2007-08-26
Color depth is the amount of memory used to represent a color. "bpp" stands for "bits per pixel". With 8bpp you can represent 256 colors, 24bpp allows millions of colors to be displayed.

You can change that in your /etc/xorg.conf file. If you need advice doing that, publish that file here so we can tell you what to change.

### Post by CowzRule on 2007-08-29
First thing you want to do is make a back up of the file
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak
```
Now open the file to edit it
```
gksudo gedit /etc/X11/xorg.conf
```
Look at this section
```
Section "Screen"
Identifier "Default Screen"
Device "S3 Inc. VT8375 [ProSavage8 KM266/KL266]"
Monitor "Generic Monitor"
DefaultDepth 16
```
Change the last line to read
```
DefaultDepth 24
```
Save the file then reboot.
