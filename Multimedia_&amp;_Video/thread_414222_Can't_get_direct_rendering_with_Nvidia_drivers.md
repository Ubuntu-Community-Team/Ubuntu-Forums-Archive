### Post by RomeReactor on 2007-04-19
Hi. Open your xorg.conf file

```
gksudo gedit /etc/X11/xorg.conf
```

and on the "Device" section, add this this line

```
Option		"RenderAccel" "true"
```

so the file looks like this

```
Section "Device"
    Identifier      "NVIDIA GeForce 6800 GT"
    Driver           "nvidia"
    Option          "RenderAccel" "true"
EndSection
```

Now press **CTRL+ALT+F1** to get to the terminal and enter

```
sudo /etc/init.d/gdm stop
```

then

```
sudo /etc/init.d/gdm start
```

to re-start GDM; now press **CTRL+ALT+F7** to return you to your desktop; check if you have direct rendering now. If that still doesn't give you accelereation, then perhaps you'll need to add

```
Section "DRI"
	Mode	0666
EndSection
```

at the end of your xorg.conf file also, and re-start GDM again.
