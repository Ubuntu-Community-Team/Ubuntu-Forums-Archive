### Post by NightwishFan on 2008-04-10
I will tell you how to use bittorrent software, but I will not give you links to trackers and I advise you edit and not ask about it, unless what you are looking for is free to download.

If you want a simple client:
```
sudo apt-get install transmission
```
If you want a more advanced one:
```
sudo apt-get install deluge
```

When you find a (hopefully legal) torrent, download the .torrent file and open it with your client.
