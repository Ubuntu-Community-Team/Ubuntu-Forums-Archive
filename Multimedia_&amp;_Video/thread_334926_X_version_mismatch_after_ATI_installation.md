### Post by inCursorated on 2007-01-30
i got past this error... xserver-xorg-air-core seemed to be the booger... i did this and the X version switched from 7.1.1 to 7.0.0

```
dpkg --force-all -P xserver-xorg-air-core xserver-xorg-core
apt-get update
apt-get install xserver-xorg-air-core xserver-xorg-core

```

u need to use the dapper repositories. i'm using dapper, but had upgraded X using the edgy repos :roll:
