### Post by Mastermind963 on 2008-03-13
You can simply download it from [Here](http://www.hyperionics.com/downloads/HC2Setup.exe) and install it with WINE program , WINE = An program that allows you to install windows-based programs such as .exe ... Now if you don't have you install it this way -> Open a Terminal -> Write these:
```
sudo apt-get update
sudo apt-get install wine
sudo apt-get install winesetuptk
```
Now after its installed write these codes to configure it
```
winecfg
winesetuptk
```

Now your WINE would work just fine!If something's broke ... Just search for a tutorial on Google

Now about hypercam , you can install it with WINE , after you installed it just go to Applications -> WINE -> Programs -> Hypercam 2 !

Good Luck!
Hope this helped!
