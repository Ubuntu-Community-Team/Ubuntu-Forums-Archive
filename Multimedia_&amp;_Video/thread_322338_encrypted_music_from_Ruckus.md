### Post by ph1 on 2008-01-21
I have a feeling that all DRM stuff is handled locally with Ruckus, because after downloading and validating licenses using Ruckus (in windows), I can play the files in any DRM-supporting media player (e.g. WinAmp, WMP), online or offline.  Ruckus even gives a "renew by" date that usually gives you about a month of license time before the program must renew the files.

Also, the methods of getting around the DRM from Ruckus usually involve stripping the licensing from the files using keys that are found locally on the computer by key-grabbing programs.  

I'm not really an expert on this, but if DRM support is built into Windows (which I think it is), then porting it into Ubuntu is going to be difficult.  If it's merely application-based, then there could be some way using Wine to play it.

Any thoughts?  I would love merely to be able to play my Ruckus music in Ubuntu.
