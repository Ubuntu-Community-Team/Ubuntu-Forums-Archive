### Post by w4ett on 2007-09-07
Your xorg.conf says you have the Intel 845 chipset so you should be using the 'intel' driver NOT the ATI restricted driver.  To verify your chipset, from the terminal type:

```
lspci
```

Post the results here so we can have a look.

In the future, to reconfigure the video driver and resolution sections only, use the command:

```
sudo dpkg-reconfigure [COLOR="Blue"]-phigh[/COLOR] xserver-xorg
```

If you use the command without the [COLOR="Blue"]-phigh[/COLOR] toggle, you have to reconfigure each section of the xorg.conf file UNLESS you use the <esc> key to skip the sections you do not want to modify.
