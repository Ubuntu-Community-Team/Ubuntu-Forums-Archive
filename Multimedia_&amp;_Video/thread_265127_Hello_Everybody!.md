### Post by dolson on 2006-09-25
1) CCRMA is only going to work on Fedora. Fedora is another distro of Linux, like Ubuntu is a distro. We here are going to be biased and say Ubuntu is better, since well, we use Ubuntu. Personally, I can't stand Red Hat or Fedora... So. Yeah.

2) VSTi is a Windows technology. You *MIGHT* be able to get it to work with certain VSTis, using Wine and a host like FST or DSSI-VST. I have never used it, and probably never will. I use Linux apps only.

3) Links: [www.ubuntustudio.com](www.ubuntustudio.com) - read the Studio Preparation page first, then read the tutorials. I wrote a tutorial on creating a basic song using seq24 on there. I prefer seq24 as my sequencer, you might not like it, I don't know. Just try it out.

4) Good specs. You should do fine.
