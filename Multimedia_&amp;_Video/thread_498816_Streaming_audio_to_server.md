### Post by mohnkern on 2007-07-13
Generally you need some form of streaming server, and then DJ software.  

For a streaming server, if you are comfortable with the command line, shoutcast is the fastest to set up, it's available at [http://www.shoutcast.com](http://www.shoutcast.com).   You download it, untar the files, edit one configuration file, and you're ready to go.

Then you'll need a DJ'ing package.  I haven't found any good Linux ones.  But I hear Livecast is good.    I use Sam broadcaster on a Windows box for the djing side.

It's important to remember that if you're behind a firewall, or a NAT, you'll need to open up ports, and set up the proper port redirects.
