### Post by MikeDK on 2008-03-04
Hi if you havent got the solution yet, the thing is to choose under System=>Administration=>Screen & Graphics generic instead of model and use 1280x1024 LCD display setting, then it will ask you to logout user and then it'll work.

got the same monitor and using that setting, also i got an Acer AL1916w widescreen-monitor, but that setting is implemented as a driver and works like a charm

Just got Hardy installed, and it seems that AL1916p does the trick for AL1916Cs as well
Go to Programs=>Others=Screen & graphics , and select model, now go down and select Acer some models will show up in the window beside now select the AL1916p (if you have a 1916cs that is) Close and select OK, now logout and in again and it should work
