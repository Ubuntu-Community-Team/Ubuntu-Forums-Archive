### Post by ajgreeny on 2007-04-02
I think you need to install *timidity,* *timidity-interfaces-extra* and *freepats*.  That is what I did and it all works great on (k)ubuntu.  The command to use a gui is then shown in the manual (man timidity).  I use **timidity -ik** but there are many possible ones to use depending on your own preferences.
