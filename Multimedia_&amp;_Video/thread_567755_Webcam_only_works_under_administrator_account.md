I've run into a problem setting up a Logitech QuickCam Chat to work with aMSN in Ubuntu Feisty.

I've run lsusb in the terminal and it shows the camera listed, the same way it appears under the admin session.

I've tried granting the user administration privileges, but the problem still persists.

I've run into this exact same problem on two different machines, one a Toshiba laptop and the other an Intel-based no-name desktop.

I've looked everywhere else on the net, but have found nothing related specifically to this issue.
