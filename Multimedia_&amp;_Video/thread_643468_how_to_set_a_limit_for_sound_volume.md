I would like to find a program to do this also. It seems like it would be quite possible and I think it would be a very popular app. It would sure be better than having to have separate plugins for every application, too.

Meanwhile, there are some programs that can help. Search Synaptic for 'normalize' and, at least on Gutsy, I see:

normalize-audio
xmms-volnorm (at least, if xmms is used to play audio on web pages, this could help)

Also there is mp3gain and mp3cd that can normalize audio cds.

There may be some other audio players that have a 'normalize' function built in that might could be set as your default audio player or that your web browser could be configured to use.

Hope this helps a little, anyway.
