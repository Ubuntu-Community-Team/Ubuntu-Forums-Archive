Just a general information for those who are interested in using Guitar Pro-files with Linux: Finally found out how to import gp3-files! If you don't want to install the Guitar Pro-software with WINE but nonetheless want to work with these Guitar Pro-files, try importing them with SONGWRITE. It's a small but quite useful program that's able to work with this type of file.
