### Post by pdub on 2006-11-28
Are you using the nv driver or the nvidia driver?

You can check this by typing:

**cat /etc/X11/xorg.conf | grep nv**

Check to see if it says "device" "**nvidia**" or "**nv**"

If you are not running the nvidia driver, check out this wiki for instructions.

[http://ubuntuguide.org/wiki/Ubuntu:Edgy#How_to_install_Graphics_Driver_.28NVIDIA.29](http://ubuntuguide.org/wiki/Ubuntu:Edgy#How_to_install_Graphics_Driver_.28NVIDIA.29)

Once installed, and if your monitor supports it, you should be able to select 1280x1024. If not, then post your /etc/X11/xorg.conf.
