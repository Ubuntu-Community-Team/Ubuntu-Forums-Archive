### Post by Unisted on 2007-02-06
Hi Eddie,

I'm not sure if I can help too much as:

a) I don't know what I'm doing!
b) I don't know what installation (amd etc..)/ hardware you have

However, I managed to get K9copy working eventually in Edgy on my AMD64 installation. BUT it didn't work the first time around.

So, I would personally try un-installing it and then re-installing and see if it works

Check your blank discs are fine too

Check you can play the DVDs in your machine and if not then ensure you have all the correct dependancies and restricted codecs and the like and then run in K9copy (this shouldn't matter but hey... worth a shot!)

Also, when backing up dvds I noticed that if you use one folder it polulates the Video_TS folder with additional VOB files so you can end up with *.vob_1 *.vob_2 which may be confusing k9copy when creating the .iso

This post might be useful
