### Post by cneil on 2007-03-04
You won't really have to "get everything ready for it."
All you need to do is follow the instructions here
[http://ubuntuforums.org/showthread.php?t=199250](http://ubuntuforums.org/showthread.php?t=199250)

You'll end up using a program called gnomad2 to transfer you music and media.  I haven't figured out the directory structure yet, or if there is one.  However, playlists seem to work well and uploading videos is not a problem either.

The Zen doesn't support ogg, but I only use unencrypted, uncopyprotected mp3s for music so that isn't an issue.  

I've been disappointed in the way that the Zen handles video.  It advertises a lot of formats, but I haven't been able to get that many to work.  Xvid seems to work the best.  Uncompressed mpeg downloaded from internet archive work well.  I haven't gotten it to play any file labeled mp4, in any format. Divx seems to work well too.  The box says WMV9 compatible, but I think that this means it only plays windows media 9 files, not 8 or 7, or anything else.  I tried to get a wmv file to play and the sound worked, but the video did not work.  Moreover, when I try to play certain video file formats like quicktime (I didn't expect this to work) the whole thing freezes.

I think I will enjoy the Zen once I figure out all of the quirks and what file formats that work the best. There are also some things about the user interface that are annoying and counterintuitive.  I don't know why I can't just slide my finger back to go back. 

I think the windows CD comes bundled with some conversion software.  I haven't tried to use this yet, I am trying to solve my problems using linux and only linux.

Also of note, I did manage to buy a player with the older firmware.  This is good, because the newer firmware disables the FM record functions.  

I've only had it a day and I haven't spent a lot of time experimenting with this device.  I'm sorry if my comments don't seem very conclusive or technical, but I hope they help.
