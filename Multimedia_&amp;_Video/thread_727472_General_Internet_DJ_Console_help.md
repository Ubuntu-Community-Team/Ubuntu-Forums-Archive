### Post by Cernunnos on 2008-03-31
Hi Pandababy, I had this same problem earlier, I don't know if you've already found the solution or not but in case you haven't here it is:

The version of IDJC in synaptic is not compiled with MP3 support, to set it up properly you will need to download the source code and compile it yourself, and may need to compile/install some of the dependencies also.

Don't panic, though, it's /much/ easier than it sounds :)

Here's the forum thread that I stumbled across which lead me to the solution, read through the thread and follow Steveway's instructions and you should be all set.

[http://ubuntuforums.org/showthread.php?t=522579](http://ubuntuforums.org/showthread.php?t=522579)
