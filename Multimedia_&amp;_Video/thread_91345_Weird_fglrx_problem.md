### Post by Cashel on 2005-11-17
I see no mention of AGP modules.. perhaps they arent loaded... for myself I have:

doc@wargod:~$ lsmod | grep fglrx
fglrx                    255716  7
agpgart                34888  2 fglrx,sis_agp

This is the sis_agp module included with the newer kernels.. the basic agp module should work fine for most people (myself included)..

You need not use the following installer, but the instructions themselves will have usefull info for you, in particular for your Xorg.conf file... 

[http://xoomer.virgilio.it/flavio.stanchina/debian/fglrx-installer.html](http://xoomer.virgilio.it/flavio.stanchina/debian/fglrx-installer.html)

if you decide to use the installer, note that you need to hack the makefile (or config file whatever it was, been awhile) and remove the control panel parts, since those dont come in the newer drivers (or he just screwed that bit up, not sure which it was)...

Good Luck,
Cashel
