### Post by dolson on 2006-09-19
The kernel is going to get better as time goes on.

If your system can handle it, then you're fine.

Some people still will want to patch.

At the minimum, enabling CONFIG_PREEMPT and CONFIG_HZ set to 1000 are probably going to be wanted changes for a lot of people, but in my opinion, you probably don't need to bother with patching a vanilla kernel on Edgy UNLESS you are having a lot of issues with the default or with a re-compiled default with tweaked config.
