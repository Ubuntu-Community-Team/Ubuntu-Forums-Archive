### Post by byte69 on 2007-03-06
Its because the MP3 codecs are not installed by default.   I would suggest installing this:

[http://www.getautomatix.com/](http://www.getautomatix.com/)   

Follow the instructions for your version.   Then once you have that installed start it up under system tools and go to the multimedia tab.   Install what you want and it will do the rest.   Or you can install it yourself following this:

[http://www.harecoded.com/2006/02/17/how-to-play-mp3-with-ubuntu](http://www.harecoded.com/2006/02/17/how-to-play-mp3-with-ubuntu)
