### Post by Waappu on 2007-07-13
Hi

K3b, even it is KDE program.
To install type to terminal```
sudo aptitude install k3b libk3b2-mp3 libk3b2
```

There is also other burning apps e.g. Gnome baker```
sudo aptitude install gnomebaker
```and Graveman```
sudo aptitude install graveman
```

Edit:

For K3b MP3 support you need extra repository. You can find how to from here
[http://ubuntuguide.org/wiki/Ubuntu:Feisty#How_to_install_K3b_.28CD.2FDVD_burning_software.29](http://ubuntuguide.org/wiki/Ubuntu:Feisty#How_to_install_K3b_.28CD.2FDVD_burning_software.29)
