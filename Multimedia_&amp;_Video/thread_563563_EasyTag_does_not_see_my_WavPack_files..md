### Post by julian67 on 2007-09-30
Possibly the deb has been compiled without support for wavpack. This is certainly the case for aac/mp4 which is not enabled in the ubuntu debs. You might want to try compiling easytag yourself. If you use checkinstall you can have it make you a .deb and install that for you.
