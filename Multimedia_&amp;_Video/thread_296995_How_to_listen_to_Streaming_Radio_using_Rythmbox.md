### Post by FyreBrand on 2006-11-10
**1.** Make sure you have the proper codecs installed.  Look at this [**forum**]("http://www.ubuntuforums.org/forumdisplay.php?f=159") and [**thread **]("http://www.ubuntuforums.org/showthread.php?t=186792")for instructions.  Make sure you read the instructions in the quick start thread carefully.

**2.** When you are at a website such as sky.fm, shoutcast, etc there will be media links to various media types.  The radio "streams" are url links to their servers pointing to specific media types.  So what you can play depends on the codecs you have installed.  Until you get familiar with what plays and sounds best go for the type that always works for you.  You can always change it to a better quality stream later.

  [indent]**a.** choose OGG links first.  These should always work.

  **b.** if you installed the codecs in step 1 you should be able to play most .pls streams.  These are usually labeled for MP3 or iTunes.

  **c.** if you have Real Media codecs installed you can play .RM files.  I can't remember if Rhythmbox has thoses codecs available offhand.

  **d.** avoid formats intended for Win Media Player.  These typically don't work and/or are tricky to get going.[/indent]

**3.** In Rhythmbox there is a section titled radio.  There is a menu item that says "add radio stream" or add "add url" something similar to that.  I'm at work on the Windows box so I can't look it up directly.  I believe you can also right click the radio bar and it will give yo a context menu to add a radio url.

  [indent]**a.** With your browser open to the site that has the radio stream, let's say sky.fm, right click a media format and select "copy link location" from the context menu.

  **b.** click over to Rhythmbox and select the create new radio stream function.  When the box opens that allows you to enter the url, right click and paste the url into the appropriate box.  Enter any other information you need and then click "ok" or "close".[/indent]

**4.** Test the radio stream to see if it works.  If not, try a different media type for the station you want (there are usually 3 or so).  If it does work, then you're done.

**5.** In Rhythmbox if the radio station you make has the url as the title then you can edit it and rename the title and still leave url the same.
