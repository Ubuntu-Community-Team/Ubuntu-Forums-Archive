### Post by NewEnergyWorks on 2008-03-08
To watch CNN Live Video without running W*ndows:

1) Install WINE (Synaptic Package Manager),
2) In Linux, download Firefox for W*ndows,
3) In Firefox for W*ndows, open CNN and try to view a video,
4) Install the Turner add-on,
5) Restart Firefox for W*ndows, go to CNN and watch your video.
