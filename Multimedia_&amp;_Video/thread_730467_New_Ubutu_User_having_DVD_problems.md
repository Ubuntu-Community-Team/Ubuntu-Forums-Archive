### Post by LeoSolaris on 2008-03-20
[https://help.ubuntu.com/7.10/musicvideophotos/C/video.html#video-dvd](https://help.ubuntu.com/7.10/musicvideophotos/C/video.html#video-dvd)

DVD playback is proprietary, which means that technically it is illegal to install a codex to play them without paying the owners of CSS software money, which media players like Windows Media Player and DVD player manufactures do. However as of right now, in the US, no one has ever been tried on this, so it is untested as a restriction.

### Post by cozmicharlie on 2008-03-20
Enable the medibuntu repository ([https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)) - install the codecs (libdvdcss2, w32codecs -  scroll down in the Medibuntu page and it has instructions) - install the Ubuntu Restricted Extras (from Synaptic or add/remove).  That should get you going.

If you have problems check out this [http://ubuntuforums.org/showthread.php?t=529759&highlight=gutsy](http://ubuntuforums.org/showthread.php?t=529759&highlight=gutsy)  and this [http://ubuntuforums.org/showthread.php?t=661833](http://ubuntuforums.org/showthread.php?t=661833)

Enjoy!
