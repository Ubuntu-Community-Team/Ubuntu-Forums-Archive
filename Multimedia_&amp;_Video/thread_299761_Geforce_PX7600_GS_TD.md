Try this:
```
sudo dpkg-reconfigure xserver-xorg
```

and press ENTER whenever you don't know the answer to a question.

then:
```
sudo nvidia-xconfig --no-composite
```

and
```
sudo /etc/init.d/gdm restart (or "kdm restart" if you use KDM)
```
