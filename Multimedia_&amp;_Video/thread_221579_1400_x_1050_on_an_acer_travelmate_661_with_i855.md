### Post by Jagot on 2006-07-23
You'll need to install 915resolution from the universe repo.  After installing the package and restarting it automatically picked up the native resolution for my laptop (Dell Inspiron 1300 with 1280x800).  If it does not pick up yours automatically then have a look at the readme file in /usr/share/doc/915resolution/
