### Post by chrome307 on 2007-09-23
I have a Technisat Skystar 2 PCI DVB-S card which is attached to a 80cm motorised dish. (45w to 45e)

I'm currently using this on my Windows PC with Alt-DVB (viewing application) and using S2emu (Software encryption emulator) with the appropriate softcam key files to watch encrypted television.

Apparently this card is meant to work with Ubuntu Breezy v5.10, straight out the box, but I haven't tried it myself.
