### Post by claydoh on 2006-11-26
If you have kaudiocreator set to autorun when a cd in inserted, then yes, it is going to open up a new instance of the app. If you remove that option, you won't get multiple instances. I don't see any way tio force a single instance, but if the app is open, it will detect the tracks when a new cd is inserted.
