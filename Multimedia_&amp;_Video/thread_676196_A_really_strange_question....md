### Post by Frumious Boojum on 2008-01-23
Those 3-D glasses were not polarized glasses.   Most likely, they were shutter glasses, which electronically sync with the monitor.   At the moment, it's not quite possible to use any kind of polarized glasses with monitors or TVs.

Polarized glasses require projection onto a silver screen (so, not just any screen will do) and for polarized filters to be placed over the projectors lenses to correspond with the lenses in the glasses.

The only way to possibly make polarization work in a computer environment is to project from the computer using a dual projector setup.

As for software, StereoPhotoMaker is the best, period.  It's capable of just about every stereo photo format ever.   It's always been free software, too.    However, it has not yet made it to another OS beyond Windows.   I've not tried to run it in Wine yet, so I should probably do that soon.

Linux:

[Plascolin]("http://www.schrammel.org/stereo-plascolin.php") - haven't used this one, but should be decent enough for anaglyphs.   I hate anaglyphs.
[URL="http://mitglied.lycos.de/PageOfMH/index.html"]KMovisto
[/URL] - Not for you if you're looking for photographic uses, but should be fun enough for molecules.
