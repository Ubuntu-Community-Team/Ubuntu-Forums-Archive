### Post by MentalNotes on 2007-09-08
Had same problem - just installed Ubuntu 7.04.

System --> Preferences --> Sound.
Changed default Sound Events / Music and Movies to USB Audio.
Right-click Volume Control on top panel --> Preferences.  Change device to UAC3553B (ALSA Mixer)

Works fine now.  Hope this helps anybody with same problem in future.
