### Post by sharperguy on 2007-10-03
For most (non-intel) graphics cards to work fully in linux you must install the restricted graphics drivers. In the current stable version of ubuntu - 7.04 (The Feisty Fawn) - you just need to run the restricted drivers manager and it should grab it automatically (asking you first obviously).
