### Post by krelverehan on 2008-02-02
Well, if you want to run ubuntu as fast and sleekly as possible, on older machines, it is my opinion that you try out the xubuntu flavour. ### Post by TeaSwigger on 2008-02-02
Well here's a good site for resource-conscious computing:
[http://kmandla.wordpress.com/2007/10/20/howto-set-up-gutsy-for-speed/](http://kmandla.wordpress.com/2007/10/20/howto-set-up-gutsy-for-speed/)
the blog of a forum member which has been helpful to me as well.

The eye-candy is generally the first thing you have to cut. One should be particular about what apps one uses. Don't forget to drop unneeded services and weed out some unwanted stuff - not everyone needs bluetooth support for example. Then there are lighter window managers one can use over ubuntu, like Fluxbox.

I'm using a "very old" PIII with 384mb ram and nvidia vid card. I find it performs very nicely with Kubuntu, especially after some trimming of unneeded stuff and has an attractive appearance. Stuff like transparency, fading menus, shadows everywhere, has to go or it can get a bit sticky. Once it's lightened up, as mentioned, those specs actually run Kubuntu briskly. :)

### Post by krelverehan on 2008-02-03
Main Menu >> Add/Remove... 
In the top right, Show: Installed Applications.
From there you can see what has been installed and remove what you find unnecessary. Not quite like a wizard, but a nice file space saving trick.

Also: [Minimal Ubuntu]("http://www.psychocats.net/ubuntu/minimal") [[www.psychocats.net]](www.psychocats.net]).
