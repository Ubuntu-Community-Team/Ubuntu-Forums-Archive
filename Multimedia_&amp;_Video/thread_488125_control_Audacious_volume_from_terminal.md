### Post by munkar on 2007-06-29
i was trying to write a bash script that would run a certain mp3 on Audacious every morning, but i also want it to adjust the volume in case i left it low or muted previously. is there a way to run Audacious from bash with the volume set to a specific level? thanks!

---

### Post by munkar on 2007-07-01
got the answer from someone on the audacious forum:

```
audtool set-volume 100
```
