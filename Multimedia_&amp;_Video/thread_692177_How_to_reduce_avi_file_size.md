### Post by gsmanners on 2008-02-09
The main thing you can do is to reencode the video. The two main ways to do this would be to reduce the size (downscale) or use a more sophisticated codec.

To do this, you can use avidemux or transcode or mencoder (or one of the many GUIs for those applications).

If you have softsubs, then if you resize the video you will need to reposition the subs.

The process can be complicated and very time consuming depending on how you go about doing this.
