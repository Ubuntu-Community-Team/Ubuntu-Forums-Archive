### Post by sojyujai on 2008-01-19
It looks like you have a problem with the sun java plugin.  Do you have this installed?
This is a known bug with Miro - see this bug report:
[http://bugzilla.pculture.org/show_bug.cgi?id=9064](http://bugzilla.pculture.org/show_bug.cgi?id=9064)

Hopefully it will be fixed by version 1.5 whenever that will be.
Right now the only solution I'm aware of is to replace sun java with icedtea java.  See this thread:
[http://ubuntuforums.org/showthread.php?t=588617](http://ubuntuforums.org/showthread.php?t=588617)

The problem is, if you actually use java for anything, this isn't much of a solution since most java programs don't work with the icedtea version.  At least in my experience.
