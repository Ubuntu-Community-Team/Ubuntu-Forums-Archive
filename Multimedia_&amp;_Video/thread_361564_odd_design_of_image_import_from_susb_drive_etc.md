### Post by Xebeque on 2007-02-14
I hope this is the right place to discuss here and it hasn´t been discussed before a hundred times.
It is a question of gui design concerning ubuntus image import from usb media.

Anytime I plug in a usb drive a dialog pops up asking if I would like to import images found on this media or ignore them.

Choosing the import option opens gthumb. But gthumb doesn´t give an easy to use possibility to import images for the unexperienced user. It displays images stored on the media pretty well. So imho the pop-up dialog should offer to display images - not to import them.

I think the pop-up should have three options: ignore, import, display. An import option should provide a gui similar to that used with digicams - choose place and images to import. That would make using usb drives and similar media much easier.
