### Post by philc on 2008-01-29
Some more details regarding the software you are using for video conversion would be useful.

However, in general, the video codecs you have installed in Ubuntu will not also be present in your Windows Virtual Machine.

So, for example, if you were to use [MPEG Streamclip]("http://www.squared5.com/") for video conversion you'd also need QuickTime plus Apple's MPEG2 plugin, or [QuickTime Alternative]("http://en.wikipedia.org/wiki/QuickTime_Alternative") installed for Windows on the virtual machine to be able to transcode video to QuickTime.
