### Post by Dave54 on 2006-11-26
I have the same problem, and this answer didn't work for me.

It seems that if you right-click, and under "open with" you click "open with other application", you can select the program and that will become the default program. **However**, if the program you wish to use is already displayed under "open with", this method will not make it the default choice.

In this case, **neither** of the following sets the program as the default program:
1. Right-click, and under "open with" click "open with other application", then select the program
2. Right-click, and under "open with" click the program

Ok, I understand what you meant now. In that case, to change the default application, do the following:
