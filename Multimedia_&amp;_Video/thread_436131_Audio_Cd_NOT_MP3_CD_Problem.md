### Post by drdont on 2007-11-21
I just had the same problem, serpentine would not write a bunch of mp3 files, it gave disc capacity exceeded.  It turns out that serpentine writes audio cds so it will go and turn the mp3 files into wav files.  So when writing a collection of mp3 files write them as data files instead.
