### Post by mephisto1982 on 2008-03-21
Sound juicer, rhythmbox and just about every other software cd player dont get your cd track titles from cdtext, but from a cddb on the internet. You can verify this by disconnecting from the internet and putting in a new cd that you've never played before on that machine (cause it caches info, so already-played cds will probly work).
This fails for your own compilations cause the cddb doesn't know about them. You'd probly need to try the cd in a cd player with cdtext support (like car radios) to see if your cd indeed has cdtext. I dont know any burning app that will work for sure though, cause i dont have any cd players that support cdtext, so i cant test it.
