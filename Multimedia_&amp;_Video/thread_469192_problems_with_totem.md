### Post by TravMan63 on 2007-06-09
Tombol,

Try this:

1) open add/remove
    search for totem, using the 'show all available applications'

    You should find 2 totems, one with xine, one without.  You want the one without xine.
    If the proper totem is already selected - skip to step 2.
    Select the one without xine (if the one with xine is already installed, it will be uninstalled)


2) in the search box - type codec
    install all available codecs - might as well get them all now...

3) cross your fingers and hope it plays...

good luck
