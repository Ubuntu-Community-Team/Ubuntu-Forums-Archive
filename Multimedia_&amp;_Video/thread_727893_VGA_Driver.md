### Post by wolfen69 on 2008-03-18
is it broken? or are you just hoping for more performance? if it's the latter, i wouldn't bother. integrated intel video is not a performance video chip to begin with. you will not see any difference with a new driver.

In general, many Linux* distributions already include Intel® graphics drivers. If you are looking to update your driver, Intel recommends checking on availability and obtaining precompiled driver packages from your Linux distribution vendor or computer manufacturer. Support for these drivers will be through your Linux distribution vendor or computer manufacturer.

Intel provides development drivers for Intel® integrated graphics to the open source community. These development drivers are intended to provide the Linux* developer community an opportunity to improve and provide feedback for the driver. Driver source code can be obtained from public source code repositories.

If you are looking to compile or upgrade the Intel graphics drivers for Linux on your own, information on obtaining the development driver source code can be found in the Download section of [http://www.intellinuxgraphics.org](http://www.intellinuxgraphics.org). Support for this driver is only available through the open source community, specifically via mailing lists and forums. These drivers are intended for use by technical Linux developers.
