### Post by ron999 on 2008-03-31
Hi
If I were doing the job with Ubuntu I would use
**vobcopy**
to copy the main title as a single file onto the hard drive.
Then use
**tovid**
to re-author the file as an ntsc DVD.

These are both command-line applications.
Perhaps other people may suggest other methods.
