### Post by Xian on 2005-11-12
That package is available in the Ubuntu universe repo.
I would suggest using that build.

[regionset_0.1-1_i386.deb](http://archive.ubuntu.com/ubuntu/pool/universe/r/regionset/regionset_0.1-1_i386.deb)

```
# apt-cache policy regionset
regionset:
  Installed: (none)
  Candidate: 0.1-1
  Version table:
     0.1-1 0
        500 http://archive.ubuntu.com breezy/universe Packages
```
