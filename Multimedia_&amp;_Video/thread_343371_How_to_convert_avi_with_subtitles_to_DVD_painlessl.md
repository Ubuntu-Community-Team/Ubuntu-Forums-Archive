### Post by chalimac on 2007-01-21
The program DeVeDe is very handy for this. Unfortunately the version in the repositories is outdated and does not support subtitles.

Get version 2.10 from the repository:

deb [http://www.debian-multimedia.org](http://www.debian-multimedia.org) sid main

You will be able to add subtitles in File Properties/Advanced options. They will be converted to true dvd subtitles (optional)

It's a great out-of-the-box util.

More info and dependencies here:

[http://www.rastersoft.com/programas/devede.html](http://www.rastersoft.com/programas/devede.html)
