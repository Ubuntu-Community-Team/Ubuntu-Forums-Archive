### Post by newlinux on 2007-05-15
A few questions. What video card do you have? do you have another computer than you can use to ssh into it? Do you get a splash screen before it goes black? Can you access a console login (Ctrl-Alt-F1)? If you can try:
```

sudo dpkg-reconfigure xserver-xorg

```

which can help to fix you xserver. Then try restarting gdm.
Maybe you can post the contents of your /etc/X11/xorg.conf.

Graphics/Video Subsystem

The Graphics and Memory Controller Hub (GMCH) includes a highly integrated graphics
