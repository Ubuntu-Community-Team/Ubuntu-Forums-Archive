### Post by Alexander2007 on 2007-07-30
Well there is a project called UbuntuMMC, geared to the media center: [http://www.ubuntummc.com](http://www.ubuntummc.com)
I haven't tried it personally but go check it out. There is also MythTV ( [http://www.mythtv.org](http://www.mythtv.org) ) for Ubuntu which can be installed on Feisty, using its built-in "Synaptic Package Manager". There is also an installation guide for MythTV available here: [http://parker1.co.uk/mythtv_ubuntu.php](http://parker1.co.uk/mythtv_ubuntu.php).  Ubuntu Linux can be configured to your specific needs or preferences. The choice is yours.
Hope I helped. :)

### Post by aquavitae on 2007-07-30
Have a look at [[COLOR="Sienna"]http://geexbox.org/en/index.htm[/COLOR]]("http://geexbox.org/en/index.html").  IMHO, 4G ram and a massive processor will be wasted on it - multimedia doesn't really use all that much resources - you're likely to use far more by simply turning on more desktop effects.  If you really want absolute control over your system, try [[COLOR="Sienna"]gentoo[/COLOR]]("http://www.gentoo.org").  Its a steep learning curve, but when you get there you'll be able to fine tune your system way more than you'll ever do in ubuntu.
