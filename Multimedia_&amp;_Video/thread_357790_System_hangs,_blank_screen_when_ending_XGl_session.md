### Post by ingo on 2007-02-11
Holding down the power button is not the gentlemanly thing to do, especially as Ubuntu uses a journaled file system. Try ctrl + alt + backspace instead next time you want to log out - that should take you back to the login screen from where you can shut down properly.

For more info you might want to ask in the beryl forum. They are very knowledgeable and helpful!
