### Post by ron999 on 2007-12-13
Hi
If you want to watch the DVD on a stand-alone DVD player then it's not so simple.
You have to 'author' the file into DVD format first.
Only when you've done that can you burn it with a program such as K3b.

Some DVD players will play an avi file if it's been made using 'DIVX' codec. If so, then just burn it as a data disc with whichever program you like (Gnomebaker, Brasero, K3b etc).

So check your DVD player to see if it's 'DIVX compatible' first.
Then find out if the avi file has been made with a  DIVX codec.
