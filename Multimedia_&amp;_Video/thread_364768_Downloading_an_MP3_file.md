Apologies if this has been covered before.

Is there a way to actually save an MP3 file in Ubuntu?  My [church]("http://www3.calvarychapel.com/ccy/teachings.html") archives it's sermons in MP3 format.  When you right click on the link, the only option I get is "Save Link As...".

I tried saving one of the links to my desktop, and then right clicked on it and went to "Properties".  It says "File Type: MP3 audio (streamed)" and the file size is "76 bytes".

Previous files that I saved when I was still using Microsoft the "File Type: MP3 audio" and the file size is 13.6 MB.  So in this case the actual file is being saved as opposed to just a link.

Thank you for any help you can give me.

### Post by moitio on 2007-02-18
Hey. The reason the file sizes are so small is because the files your church's website offer you are streaming mp3 files. They should be only very small in size but when you open them in a good media player, they should download and play. If you want toa actually store the mp3 file on your hard drive, you can open the m3u (streaming mp3 file) and copy and paste the link to the actual mp3 file into your browser, and save it.

A little research into how your church's website works tells me that to get the mp3 all you need to do is get the link to the m3u file and replace the m3u extension with mp3. 

Hope this helps,
