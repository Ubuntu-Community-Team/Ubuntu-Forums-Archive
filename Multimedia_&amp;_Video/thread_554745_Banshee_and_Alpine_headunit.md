### Post by Todd1561 on 2007-09-19
Well, since I like Banshee and didn't really want to switch to something else, I downloaded the latest version from [http://banshee-project.org/Releases](http://banshee-project.org/Releases)

The binary version in the repositories is version 0.12.1, the current latest stable release is 0.13.1.  So we're not talking about a huge jump, but it seems to have fixed whatever problem I had.  I added a few more songs to my library, ran another sync and all was well.  For good measure I added some more and sync'd again, still good.

So if you're having some odd problem you might try the latest version.  It's not in a nice .deb file, so you'll have to do some compiling and whatnot, but it was worth it.  Hopefully it'll keep working....
