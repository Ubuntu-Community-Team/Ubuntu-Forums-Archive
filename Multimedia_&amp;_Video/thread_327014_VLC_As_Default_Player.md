### Post by KenSentMe on 2006-12-28
I don't think there is a way of making a certain program the default player for all media formats. However, you can specify a default opening program for a certain filetype. You can do this bu richt-clicking on the file, select properties, go to the 'Open with' tab and select a default program by clicking the right radiobutton. You have to do this for all the filetypes you want VLC to be the default player for.
