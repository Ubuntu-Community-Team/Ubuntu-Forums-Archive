### Post by coolen on 2008-01-27
Just so you know, the thing you'll find with a lot of OSS software is that there aren't as many all-purpose software suites. There are few programs, for exmaple, which could take care of everything that Magic DVD could.

Instead, there are a lot of programs designed to take care of individual tasks (in this case, k9copy is good for DVD backup, while others such as acidrip would be used for ripping the DVDs).

This is part of the Unix philosophy (each program should do one task, and do it well). This will probably change in the near future. There are plans in the works to make Linux more attractive to third party developers by providing a common interface to the many packaging systems, so you may see some of these programs ported to Linux in the years to come.
