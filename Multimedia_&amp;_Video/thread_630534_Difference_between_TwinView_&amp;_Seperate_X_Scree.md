### Post by jondecker76 on 2007-12-03
Twinview creates one big desktop between both monitors, where separate x sessions create 2 separate desktops. The thing about separate x sessions is that it is like having 2 computers side-by side so you can't drag applications from one to the other etc.. I think you would be happier with twinview- i really like it
