### Post by barney_1 on 2006-12-04
SOLVED-

I'm embarrassed to admit that I just discovered I hadn't been using the most up-to-date version of firmware for the K8V SE Deluxe motherboard.  I had been using a March 2005 build when the latest is a June 2005 build.

1. Download latest version: [http://dlsvr01.asus.com/pub/ASUS/mb/sock754/k8v_se_dx/K8vsedx07.zip](http://dlsvr01.asus.com/pub/ASUS/mb/sock754/k8v_se_dx/K8vsedx07.zip)

2. Unzip the file (K8VSEDX.007) and rename it K8VSEDX.ROM

3. Copy K8VSEDX.ROM to a floppy disk and leave the disk in the drive.

4. Reboot

5. During POST (just when computer is starting up) hold Alt-F2.

6. Don't touch anything!  The bios will be reflashed and your computer will reboot by itself.

7. Enter BIOS setup when the machine restarts and make sure all the settings are as you want them.

Now I have no problem booting into Edgy with DRI enabled.
