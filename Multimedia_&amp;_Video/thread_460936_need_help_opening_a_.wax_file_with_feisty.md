### Post by steeleyuk on 2007-06-01
A WAX file is a redirector which points to a media file on the web. If you open the WAX file with a text editor, it should have the link to the full media file. You could then download the media file using that link.

Alternatively, you could try VLC.

Heres some information about .wax files. [http://filext.com/file-extension/WAX](http://filext.com/file-extension/WAX)
