After a few days of messing about I thought I would share a solution to getting the Sony Walkman NW-E005 to work with Ubuntu. There is a program called Symphonic that is a java based program that enables you to fill the player. You just have to paste it onto the player and away you go.

### Post by Ubuntooner on 2008-05-20
Be sure to install and use the Sun Java run time for this to work.  The gtk open source java would error out when I tried to run Symphonic (see [http://ubuntuforums.org/showthread.php?t=102936](http://ubuntuforums.org/showthread.php?t=102936) for a similar problem).

Once I installed java-6-sun (see [https://help.ubuntu.com/community/Java](https://help.ubuntu.com/community/Java) for instructions) Symphonic worked fine.  I had to set up a script to launch it from the desktop (see [http://ubuntuforums.org/showthread.php?t=281932](http://ubuntuforums.org/showthread.php?t=281932) for instructions on how to do this).

When you run it for the first time, you must input the path to the device and then to your music files on the PC.  Symphonic does the very basics of moving files from one device to the other, and that's it.  But that's enough to make it worthwhile.
