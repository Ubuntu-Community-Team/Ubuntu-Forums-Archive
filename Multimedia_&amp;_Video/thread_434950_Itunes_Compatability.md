### Post by spin2cool on 2007-05-06
The short answer is no.   You'd probably be best served by using one of the many alternative programs  (rhythmbox, amarock, jajuk, and many others)  There are also options for syncing your iPod.  GTK-pod is one of the best.

The slightly longer answer:  I have heard that it is possible to run old versions of iTunes (6.0) under WINE, but the setup is not for the faint of heart.  Another option is to install windows under VMware, but again, setup isn't very easy.

If you'd really like to see iTunes on linux, please write apple and let them know.  They chose to only release windows and OSX versions, and many of us are unhappy with that decision.
