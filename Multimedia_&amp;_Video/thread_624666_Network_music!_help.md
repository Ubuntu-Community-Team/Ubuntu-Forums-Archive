### Post by daengbo on 2007-11-27
When you say that you mounted network drives, do you mean that you used Places -> Connect to Server or that you used a "mount" command. The Places menu will connect via GnomeVFS, which isn't available to non-Gnome apps.

If you want the drives available from all apps regardless of whether they are default for Ubuntu or not, you should use the "mount" command.
