### Post by eggdeng on 2007-07-07
OK. Running any linux distro on a PC with no internet connection is going to be like having to fetch water from a well. It would mean downloading different source packages for each app, plugin, set of codecs you want to install, copying them over via usb key and then compiling and installing each one. It's a non-starter. You will be better off either working out some way to share your connection from the windows machine or dual-booting ubuntu with windows on the machine with the working connection.

### Post by eggdeng on 2007-07-07
Check that the PC in question has a network card. (Look at the back and see if there are 2 telephone jack type slots, one a little bigger than the other.) The smaller one, the one your telephone jack fits snugly in, is a modem; the other is the network card. It looks like the graphic at 
[http://www.coolnerds.com/Newbies/Ports/ports02.jpg](http://www.coolnerds.com/Newbies/Ports/ports02.jpg)
If you have these slots, go with the cable, just tell the guy in the store you want to connect a PC to your router and make sure the cable is long enough.
That's all the hardware you need. Then you will have to configure a local area network. It isn't difficult and you can post here for help.
