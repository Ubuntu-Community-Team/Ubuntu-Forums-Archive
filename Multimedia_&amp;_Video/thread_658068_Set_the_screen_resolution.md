### Post by lusis89 on 2008-01-04
You may try to reconfigure the X Server by running
```
sudo dpkg-reconfigure -plow xserver-xorg
```
from a terminal.

This will ask you a lot of questions.
If you don't know the mean of a question, select the default (or pre-entered) answer.

When it asks to select resolutions, select all you need (using spacebar) and confirm.

Then after you have answered all questions restart your X Server by pressing
```
CTRL+ALT+BACKSPACE
```

....
Then, **If the X Server doesn't start at all**, press
```
CTRL-ALT-F2
```
login, and type
```
sudo dpkg-reconfigure -phigh xserver-xorg
```
to regenerate the original xorg.conf.
and 
```
sudo /etc/init.d/gdm restart
```
to start the X Server.
....

Then select your resolution from the System->Administration menu.

### Post by perixx on 2008-01-05
perhaps you can do it the 'graphical' way and choose localization and keyboard layout in your system settings?

Btw. if you have a look into your /etc/X11/xorg.conf file now, there will surely be a line specifiying your new resolution now. It should have sufficed to simply add your resolution to the existing lines, along with adjusting H/V-sync, as proposed by Temüyin - if you don't know how to calculate the correct Syncs, run Windows and look in your monitor's OSD for them, when running your desired maximum resolution. 
> SubSection "Display"
		Modes		[COLOR="Blue"]"1440x900@60"[/COLOR]   "1024x768@60"	"800x600@60"	"800x600@56"	"640x480@60"
	EndSubSection I'm not familiar with the modelines-method, cannot tell you how to calculate them, but there are tools for that - so hacking an additional proper modeline should work as well. 
If you got problems getting back your keyoard, you might look for an backup of your xorg.conf in the folder I mentioned. You can (and should) first make a backup of your existing xorg.conf:
```
sud cp /etc/X11/xorg.conf /etc/X11/xorg.conf.std
``` 
Then you can switch back to an older version of xorg.conf (if you have a backup, that is), by replacing the current one (delete the current and make a renamed copy of the old one) like so:
```
sudo rm /etc/X11/xorg.conf
sudo cp /etc/X11/xorg.conf.original cp /etc/X11/xorg.conf
```
If that one isn't working anymore, you can simply revert to the new pretty much the reverse way...
