### Post by edonia on 2007-07-01
Here, an unusual but good working way:
install "wink" for making desktop movies (debugmode.com), running your sideshow with gthumb or similar program and record it with "wink".
After recording, you can insert sound, text, making buttons, resizing and a lot more.
Then you can export the movie as html with an embedded flash file.
