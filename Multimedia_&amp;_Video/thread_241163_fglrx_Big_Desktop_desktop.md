### Post by mzungumwaafrika on 2006-09-27
OK. 

 I have an ATI Mobility Radeon X1800, and wanted to use dual screens. It was pretty tough but after four days reading through here the solution for me was:

Install the latest drivers. Get the 8.29.6 from ATI website, but then follow the instructions on this [http://www.ubuntuforums.org/showthread.php?t=204910](http://www.ubuntuforums.org/showthread.php?t=204910) thread. Make sure it works. 

Then simply follow the instructions here about adding the 

Section "Device"
	Identifier  "ATI Graphics Adapter 0"
	Driver      "fglrx"
	Option	    "VideoOverlay" "on"
	Option	    "OpenGLOverlay" "off"
	Option	    "DesktopSetup" "horizontal"
	BusID       "PCI:1:0:0"
EndSection

line Option "DesktopSetup" "horizontal". 

Reboot - now I have dual monitors. Fantastic!

A.
