**The Remedy**
It seems the problem wasn't in X, but in the way that GNOME automatically made the resolution set to 1440x1024.  All I needed to do was go to **System > Preferences > Screen Resolution** and select.

### Post by aysiu on 2006-01-08
Try this. ```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup
sudo nano /etc/X11/xorg.conf
``` Then replace this ```
Section "Monitor"
	Identifier	"VA1912wSERIE"
	Option		"DPMS"
EndSection
``` with this ```
Section "Monitor"
	Identifier	"VA1912wSERIE"
	HorizSync	30-82
	VertRefresh	50-85
	Option		"DPMS"
EndSection
``` Save (control-x), confirm (y), exit (Enter). Then press control-alt-backspace. Any different?

I'm basing that on the link you provided--from that I went to [this PDF of the monitor's specifications](http://www.viewsonic.com/pdf/us_eng/products/VA1912wb-1_ds_lo_1005.pdf) to see the vertical and horizontal frequency ranges.
