### Post by dabl on 2007-06-10
Have you tried (in a console window) ```
sudo nvidia-settings
```?  Assuming the driver really is installed, this will open a utility panel, and the second item from the top is the X Server Display Configuration window.  Set your resolution and refresh rates there -- if you don't see a satisfactory, choice, try the "detect monitor" button, and see if it can read your monitor's capability.  When you've got it where you want it, click the "Save to X Configuration File" button in the lower right, and in the window click "Save", and upon your next boot of the xserver it should come up in your desired resolution.  To restart the xserver manually, just hit Ctrl-Alt-Backspace. :)

### Post by johnc4510 on 2007-06-10
Mike, here is the next thing to try:

First backup your current xconf.

In a terminal type sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup

Then in a terminal type: sudo dpkg-reconfigure xserver-xorg

This will launch a graphical interface to configure the x

Answer all questions, sometimes the default is the correct answer

When it asks you can put in screen size and resolution to fit your monitor.

Then restart

If for some reason x will not start after trying this you can always use your backup file to 

get back to where you started. 

To do this in a terminal type: sudo mv /etc/X11/xorg.conf.backup /etc/X11/xorg.conf

Good luck  :)
