i have the following problem:

I bought 2 Matrox Millenium G550 for my work PC both are for PCIe x1 slot. I put one of the into the PCIe x16 slot where a Radeon card has been before and the other card into the only PCIe x1 slot right next to it. The computer is a Dell Dimension 5150 with 2 GB Ram and 2.8 GHz Pentium D. And Ubuntu 7.10 with all current updates.

The PC works fine. If configured both cards in xorg.conf to display the screen on all four monitors by using Xinerama. On all four connecters are Dell 17" Flatscreens connected with DVI-VGA-adapters:

card 1 (PCIe x16), screen 0 => monitor 1
card 1 (PCIe x16), screen 1 => monitor 2

card 2 (PCIe x1), screen 0 => monitor 3
card 2 (PCIe x1), screen 1 => monitor 4

There are coloured lines all over the screens and the picture is blurry. But I can see the mouse cursor and move it over all screens.

If I configure card 1 with the "mga" driver and card 2 with the "vesa" driver all works fine but I can't use the fourth monitor because Dual-Head is not working with the vesa driver.

If I use a NVIDIA card instead of the first G550 the same problem occurs. I don't understand it, I tried a lot of different things but nothing helped. Somehow it must be possible to get both cards working together with 4 monitors.
