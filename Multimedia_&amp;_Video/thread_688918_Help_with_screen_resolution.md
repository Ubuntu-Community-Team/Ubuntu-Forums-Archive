### Post by Don Gatto on 2008-02-06
Hello,

Take a look at the (System->Administration->)Screens and Graphics. Click on the button captioned 'Model:' where you can set your monitor type. Then you shall set the proper resolution too.

(If you experience problems with the frequency - I mean it can't be set to a value high enough -, you'll have to use the 'sudo dpkg-reconfigure xserver-xorg' command in the terminal. Tell me if you need this step, I'll describe the details then.)

### Post by Giannis68 on 2008-02-07
hi,
edit xorg.conf
 sudo gedit /etc/X11/xorg.conf
on Section "Screen" add
Modes    "1680x1050"     "1440x1440"    "1280x1024"    "1280x960"    "1152x864"    "1024x768"    "800x600"    "720x400"    "640x480"

save and reboot
