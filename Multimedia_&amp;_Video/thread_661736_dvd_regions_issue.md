As far as I know, the region code is something that's put in a 'hidden' track on store-bought movie DVDs.

I think that when you burn your own DVDs they automatically **don't** have a region code because blank discs don't have any hidden tracks.

So you could make yourself a region-free backup using **K9Copy**. It's a program similar to 'DVD Shrink'  that Windows people use.
