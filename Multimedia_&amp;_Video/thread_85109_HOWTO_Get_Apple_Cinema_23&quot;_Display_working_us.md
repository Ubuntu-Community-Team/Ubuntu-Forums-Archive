### Post by wellmt on 2005-11-01
Here is a working xorg.conf file for the Apple Cinema 23" Display (The silver aluminium one, not the older plastic one - although it might work for that too). I can't be the only one trying to get this monitor working using the Nvidia accelerated drivers, so I'm posting this in case it helps anyone... 

Note #1 

The standard NV drivers work perfectly at 1920x1200 in Ubuntu Breezy so you only need to follow this if you want accelerated 3D or some other special feature.

Note #2
This xorg.conf file works using the Nvidia drivers from the Nvidia website version 7676. This config has not been tested with the 7667 drivers. I installed the 7676 drivers by following the procedures here:

[http://www.ubuntuforums.org/showthread.php?t=75074](http://www.ubuntuforums.org/showthread.php?t=75074). 

BTW: My Graphics card is a Geforce 6600 AGP.
