### Post by anoc on 2008-03-04
In ex falso, there is a button on the lower left, "Plugins". Check the "Download Album Art" plugin.

When in quod libet, change the view to "Album List" (enabling the radio button). Then you should be able to reverse-click and select "Plugins" > "Download album art".

This will popup a new window with your choices for the album art based on the search results from the associated internet sites (Amazon, etc.). You can select one and save it.

This will create a (hidden)  *.folder.jpg * image in the file's folder. Note that due to the single naming convention, ONLY ONE IMAGE  IS SAVED PER FOLDER. This means that unless you have all albums saved as subfolders, only one image is allowed for all songs in a given folder. 

You can also save any image as the .folder.jpg file manually.

I just started using this app today (using different music players to see what I like). If someone can expand on the above comments, it would be appreciated (the ex falso/ quod libet site has lost much of it's content).
