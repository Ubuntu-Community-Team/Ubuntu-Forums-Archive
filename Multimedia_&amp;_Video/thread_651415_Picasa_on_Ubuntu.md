### Post by SanskritFritz on 2008-01-24
I'm a Picasa fan. I used it on windows, now that I switched to Ubuntu, I was glad to see, there is a Picasa for linux as well. I also was skeptical about the wine solution, but for me, all features I need, work flawlessly. I dont burn CD-s from within Picasa, I dont import images, no avi support (or very glitchy after some fiddling with some windows codecs, not recommended), but there is the upload for PicasaWeb that I use extensively. The GUI with all of the animations work nicely. 
The greatest advantage of Picasa is that you can make changes to the pictures, and the changes are not written to the picture file, but are replayed everytime you view the picture (except for the redeye correction, dont use that!). An essential for me. 
Fspot also allows make modifications on the photo, but it changes the file, although it saves the original as well. A huge waste in my eyes. Also the structure of the files in Fspot is a mess, not usable from other applications.
 So after trying Fspot, I decided to stay with Picasa. Digikam i have not tried, as i use Gnome. Maybe I will some day, I heard its good.
