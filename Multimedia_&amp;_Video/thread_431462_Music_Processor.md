### Post by jessdog9001 on 2007-05-11
Check out [http://sound.condorow.net/notation.html]("http://sound.condorow.net/notation.html") There is a comprehensive list of the different software you can use. Some programs you might particularly check out: [Rosegarden]("http://www.rosegardenmusic.com/"), [NoteEdit]("http://noteedit.berlios.de/"), and [Denemo]("http://denemo.sourceforge.net/"). Most of these editors also support playback, so you can hear what you're inputting. Also, you should be able to find most of these programs in the Ubuntu repository under System->Administration->Synaptic Package Manager. Just do a search and see what you can find. Good luck with the music! :D
