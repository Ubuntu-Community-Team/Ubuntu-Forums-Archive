### Post by williammanda on 2007-03-20
Modify your xorg.conf file as follows:

Section "Monitor"
	Identifier	"Generic Monitor"
#	Option		"DPMS"
	# 1360x768 @ 60.00 Hz (GTF) hsync: 47.70 kHz; pclk: 84.72 MHz
#	Modeline "1360x768_60.00"  84.72  1360 1424 1568 1776  768 769 772 795  -HSync +Vsync
#	HorizSync	28-51
#	VertRefresh	43-60
EndSection

Section "Screen"
	Identifier	"Default Screen"
	Device		"NVIDIA Corporation NV43 [GeForce 6600]"
	Monitor		"Generic Monitor"
	DefaultDepth	24
	SubSection "Display"
		Depth		24
		Modes		"nvidia-auto-select"
	EndSubSection
EndSection

If your native resolution is 1360 x 768 then xorg.conf should read it from the monitor and nvidia will select the best resolution (which should be your native). Also I would upgrade to the latest driver.
Thanks

### Post by williammanda on 2007-03-20
One other thing....check your var/log/Xorg.0.log file to see if nvidia is probing the monitor for it's info....your monitor name should be stated. Nvidia by default will probe for monitor info but I have found sometimes I have to enter this statement to ensure that it happens:

Section "Monitor"
Identifier "Generic Monitor"
  Option "UseEDID" "True"
EndSection

UseEDID is the command to probe your monitor for it's info.
Thanks
