Update-----

I've been running CS3 on a virtualbox for three months with no problems.  I can't say enough good things about the guys at innotek.  Check out the program here:  [www.virtualbox.org](www.virtualbox.org) and download it using synaptic package manager.  If you are new to Linux, there are several guides on how to set virtualbox up within Ubuntu, just Google them.  Here are two:

[http://www.ubuntugeek.com/create-and-manage-virtual-machines-using-virtualbox.html](http://www.ubuntugeek.com/create-and-manage-virtual-machines-using-virtualbox.html)
