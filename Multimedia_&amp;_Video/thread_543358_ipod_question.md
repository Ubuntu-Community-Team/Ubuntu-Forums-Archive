### Post by Rhubarb on 2007-09-06
Compared to the ipod nano, iRiver's S10 2GB player competes very well.
(I have an S10 here, and use it in ubuntu with no problems)
[http://www.iriver.com/product/p_detail.asp?pidx=78](http://www.iriver.com/product/p_detail.asp?pidx=78)

Advantages of ipod nano: has a longer battery life, and a belt clip.

Advantages of iRiver s10: has a screen, is a mass storage device (you can use it as a USB drive, and can drag n drop ogg vorbis (up to Q10) / mp3 / asf / wma using nautilus / konqueror), an FM radio, a microphone, supports NO DRM what so ever, is AU$50 cheaper than the ipod nano, has an alarm that can be set to record FM radio at a certain time, has a clock, can view simple bmp images, can display song lyrics, has better DACs than ipod for better audio quality, is the same size as the ipod nano (including it's belt clip).

So, given you have less features on an ipod nano, and that an ipod nano costs more and supports DRM and non-free formats, why would you want to go for ipod when there are better, cheaper, competing music players out there on the market?
