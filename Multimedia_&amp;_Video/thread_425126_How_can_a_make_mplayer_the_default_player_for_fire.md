### Post by ubu-for on 2007-04-27
1. Start Synaptic and search for "totem-mozilla".
2. Uninstall and lock this package.
3. Search for and install "mozilla-mplayer".
4. Start Firefox and go to a [video](http://movies.apple.com/movies/dreamworks/over_the_hedge/over_the_hedge-tlr1_h320.mov) website.
5. Right click on the video and select the preferences.
6. Change the video output to "x11"

Have fun!
