### Post by tauko on 2006-12-12
I'm using also an ipod 2nd generation. I'm using Amarok. It's the best option I've proved. My advice (I suppose you already know how to mount the device):


[LIST]
[*]amarok: settings -> configure amarok -> media devices -> add (1)
[*]in playlists you can add podcasts (add -> podcast) (2)
[*]I suggest that you install the "podcastCleanup" script
[/LIST]


(1).
To disconnect correctly the ipod: eject *place_where_ipod_is_mounted*
To mount the ipod: mount *place_where_ipod_is_mounted*
I did check the two options (automatically delete played podcasts)

(2).
In each podcast options (right click -> configure): I've it not to automatically scan for updates, to download when available and to add the downloaded to the media device.

So, if I want to update the podcasts: I go to playlists, right click on podcasts and refresh all. Automatically it'll download all new files and put on the queue of the ipod. When all're downloaded I transfer them.


Hope it helps!
