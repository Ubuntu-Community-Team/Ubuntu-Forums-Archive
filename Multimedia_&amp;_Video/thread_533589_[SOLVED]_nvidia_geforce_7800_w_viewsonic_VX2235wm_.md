### Post by gustavson12 on 2007-08-26
Okay so here's how I solved it.  

1st I figured out that when I installed Ubuntu the first time, I was using the DVI cable so it was in digital mode and if I'm correct, digital mode is not compatible.  So since I'm a newbie, I decided to reinstall Ubunut in VGA mode or Analog mode.  
Once ubuntu finished installation, I installed this nvidia driver "NVIDIA-Linux-x86-1.0-9639-pkg1.run" from [here]("http://www.nvidia.com/object/linux_display_ia32_1.0-9639.html").  Here are the steps to how I installed the driver:

0. First made a backup of xorg.conf by executing this command in Terminal: sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup
1. Downloaded the driver.  --> [NVIDIA-Linux-x86-1.0-9639-pkg1.run]("http://www.nvidia.com/object/linux_display_ia32_1.0-9639.html")
2. Shut down X:  Ctrl+Alt+F1.
3. Logged in using my username and password
4. Then executed this command to stop X: sudo /etc/init.d/gdm stop
5. Located the driver and executed this command to start installation: sudo sh NVIDIA-Linux-x86-1.0-9639-pkg1.run
6. Followed the on screen instructions and finished installation.
7. Then executed this command to start X: sudo /etc/init.d/gdm start
8. X starts and I logged into Ubuntu like normal.
9. Then I opened xorg.conf by executing this command in Terminal: sudo gedit /etc/X11/xorg.conf
10. Made a change to Driver in [ Section "Device" ] to "nv".  It was "nvidia"
11. Then I restarted X: Ctrl+Alt+Backspace and logged back into Ubunutu.
11. The last thing I had to do was to go into the monitor menu on the LCD screen itself and execute the "Auto Image Adjust" function from the menu...

That's it, fixed everything.  Now I have a resolution of 1680x1050 and everything is perfect...    Ubuntu is SWEET!!!
