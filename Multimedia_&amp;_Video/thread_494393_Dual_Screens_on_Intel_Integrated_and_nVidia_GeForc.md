### Post by Kaimi on 2007-07-06
I'm running Ubuntu 7.04 on a Dell Dimension 2350, with a built in Intel Integrated graphics card. I had a lot of trouble installing an nVidia GeForce 5200 graphics card but I finally got it working.  The reason I wanted to install the graphics card is so that I could have two monitors, but now I can't figure out how to do that, since the Intel card interferes with the nVidia and I disabled it to make the nVidia work.  I'd really appreciate some help. 

I've been looking at these tutorials and people's setups and xorg.conf's all afternoon, so I figured I'd post what worked for me. 


First, I made a backup of xorg.conf:

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup
```

This was my lifesaver that I used maybe 5 times when the GUI crashed, by starting Ubuntu in &#8220;recovery mode&#8221; and typing:

```
sudo cp /etc/X11/xorg.conf_backup /etc/X11/xorg.conf
```

This is obvious, but it saved my computer when (not if) I made mistakes!

This excellent tutorial matched my situation perfectly and everything worked when I followed it:

[http://doc.gwos.org/index.php/Nvidia_Intel_Integrated](http://doc.gwos.org/index.php/Nvidia_Intel_Integrated)

This tutorial is how to install nVidia drivers on Feisty (I used this after following the first tutorial) :

[http://doc.gwos.org/index.php/Latest_nvidia_feisty](http://doc.gwos.org/index.php/Latest_nvidia_feisty)

The second tutorial configured xorg.conf for nvidia, however that overwrote the necessary settings in the first tutorial. I had to go back to xorg.conf and re-enter the device settings in order for it to work.
