### Post by starorbs on 2006-11-21
I'm trying to burn a 16:9 video to a dvd (using Devede), instead of it keeping the 16:9 aspect ratio and 720x480 which fills pretty much my entire widescreen laptop screen. It resizes it to 720x480 4:3 aspect ratio which defeats the whole purpose. Is there a way to change this in Devede or is there another program that won't change my aspect ratio. The thing I liked about Devede is I can change the bit rate which I needed to do because it was too big for 1 dvd lol. Thanks in advanced.

---

### Post by starorbs on 2006-11-21
SOLVED:
Thought I would reply to my own thread so if anyone else wants/needs to know how to do this.

If using Devede or mencoder add this line to the extra options in the advanced section.

-vf scale=720:480,expand=720:480 -lavcopts aspect=16/9
