### Post by 3rdalbum on 2006-12-21
This is what I'd do:

1. Transfer the file to a Windows machine.
2. Hook up the Sound Out of the Windows machine to the Sound In of your Linux machine.
3. Play the file on Windows, and record on Linux
4. Listen to the new copy of the audiobook, and set up an At or Cron job to automatically delete the file after 3 weeks.
