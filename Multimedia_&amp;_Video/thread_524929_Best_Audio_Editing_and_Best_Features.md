### Post by variona on 2007-08-14
> **JustinAlf said:**
> but it neglects to have key plugins that are good for music production ie loops and digital effects. 
I'm not sure what you mean by 'lack of digital effects'  - there are LADSPA plugins aplenty. If you like the loop feature you would probably love the program rezound. If you want to save your effect plugin setup you should try 'jack rack' - when you use jack as sound output you can combine plugins (LADSPA), DAW (ardour), editor (audacity) and whatever with a patchbay like interface giving you all the flexibility you want.
