### Post by dreadlord_chris on 2007-06-20
I would make the following changes to your Ubuntu xorg.conf:

remove this section (from Section "Module"):
```

Load	"dri"

```

take the following lines (from your PCLinuxOS file) and add to your Ubuntu xorg.conf Section "Monitor"
```

# Monitor preferred modeline (60.0 Hz vsync, 64.0 kHz hsync, ratio 5/4)
    ModeLine "1280x1024" 108 1280 1328 1440 1688 1024 1025 1028 1066 +hsync +vsync
    
    # TV fullscreen mode or DVD fullscreen output.
    # 768x576 @ 79 Hz, 50 kHz hsync
    ModeLine "768x576"     50.00  768  832  846 1000   576  590  595  630
    
    # 768x576 @ 100 Hz, 61.6 kHz hsync
    ModeLine "768x576"     63.07  768  800  960 1024   576  578  590  616

```

Add the following (in its own section):
```

Section "Extensions"
    Option "Composite"
EndSection

```

Save the file & restart X - see if that's better for you....
