I haven't used it myself, but I know some people like a program called "devede".  It is in the standard repositories, so you can install it with the Synaptic Package Manager, or via the command line (Applications -> Accessories -> Terminal) and enter:```
sudo apt-get install devede
```

### Post by nalmeth on 2007-08-31
Is Nero complicated to make DVD menus? I've always wondered what the process was like.

I use the Tovid suite to make DVD menus, but I imagine if you find Nero complicated, you will not like Tovid. It can make really good quality, impressive DVD's, but I would call it a bit complicated.

As suggested, DEVEDE is probably what you are looking for. The only drawback is that it doesn't offer you too many options as far as background images/video/audio, animated thumbnails, etc.

And another vote for K3B burning suite. It doesn't do DVD menus, but its good for *most* everything else.
