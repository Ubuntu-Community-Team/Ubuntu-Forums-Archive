### Post by pball on 2007-06-01
Using Feisty and Firefox 2.0.0.3

Like many others, I had trouble.  A friend e-mailed a WMV file, and when I asked firefox to open
got a message that CODECs were missing.  Tried various suggestions on forum with little success.

However, the following worked and installed the CODECs and Totem now plays WMV files directly from firefox

1) Saved WMV file to desktop
2) Right clicked on file
3) Click open with Movie player
4) Totem will give you option to install CODECS
5) Installed and it now works on all incoming WMV files

Note - I also tried opening Totem (before CODEC installation) and then
opening WMV file.  This would not work as in 2nd sentence above

Strange but true
