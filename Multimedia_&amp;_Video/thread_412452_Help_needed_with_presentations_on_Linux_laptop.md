### Post by mojoman on 2007-04-18
I can't help you with the first question but on the second my first thought is to set the projector up as a second monitor, using twinview or something similar (xinerama, for example). The nice thing about setting it up as a second monitor is that you can have the control bars or whatever in one screen (i.e. the monitor) and the application in another (i.e. the projector screen). I got a HD-projector set up that way and it works beautifully (I use it for watching movies). Exactly what setup probably depends on what type of graphic card you have, twinview is for nVidia (I think). 

/mojoman
