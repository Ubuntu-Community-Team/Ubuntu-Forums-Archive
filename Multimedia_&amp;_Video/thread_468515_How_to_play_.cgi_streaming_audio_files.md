### Post by 444 on 2007-06-09
cgi is a server side script. So it's not a file type.  The file type would be determined by what's actually in the file, I'm guessing it's some sort of text playlist file, like asx or pls.

I use mplayer for this stuff as it opens anything. It still needs binary codecs but once it has them it plays all.
