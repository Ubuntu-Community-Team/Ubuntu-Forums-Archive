weird... its manual says:
```
      --composite, --no-composite
              Enable or disable the "Composite" X extension.

```

Ok, you will have to do it manually.

type:
```
sudo nano -w /etc/X11/xorg.conf
```

get to the end of the file (using your keyboard arrows) and add this:
```
Section "Extensions"
    Option         "Composite" "Disable"
EndSection
```

then press CTRL+X to exit (save the file)
