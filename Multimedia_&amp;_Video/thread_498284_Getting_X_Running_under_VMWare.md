### Post by bernied on 2007-07-11
My understanding of vmware is that inside vmware X does not have direct access to your graphics card. Hence this:
> (II) ATI: Shared non-ATI VGA in PCI/AGP slot 0:15:0 detected.
Vmware creates a virtual card which you can access. I think you can install this with vmware-tools (my memory might be a bit off here). Be careful with running this though as it will make lots of other changes to your system as well.

I'd suggest you just try a different driver - say vesa - to test whether I'm right. Then you could set up alternate xorg.conf files to use for the two different ways of running the system.
