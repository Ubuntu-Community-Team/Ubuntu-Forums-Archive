### Post by eeried on 2007-07-26
hello,
you're advised to uncheck "burn iso" (or DVD) in the config window before using DVD95. So you can first check the iso file before burning a DVD.

After making sure your iso file runs fine in VLC for instance (no need to mount the file in VLC, just open the file) you can launch k3b in order to burn your iso file and all should be fine.
