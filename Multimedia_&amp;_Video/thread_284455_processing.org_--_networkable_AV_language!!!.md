### Post by capsid on 2006-10-25
hey there.  just wanted to share something my friend showed to me after I mentioned my interest in avisynth.
[http://processing.org](http://processing.org)

It's great! It uses java to create a multi-platform API for animation, video, audio (with the help of Ess, Sonia, or others), and network stuff.  It comes from MIT and it's being used both as an introductory programming language and as a tool for creating A/V experiments and installations.  

Look in **exhibitions** for what others have done
Look in **learning** for all of the examples (did i mention it embeds seamlessly into html pages?)
Look in **reference** to see the API list
Look in **libraries** to find the link to Ess and other libs that work with it.
