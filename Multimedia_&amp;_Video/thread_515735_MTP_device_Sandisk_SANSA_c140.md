### Post by adzik on 2007-08-05
My understanding about these sansa devices is that they have 2 modes for usb transfer, and you must set it to MSC (mass storage) and not automatic, before any linux system will detect it.  Also, it seems that any files transferred within windows using MTP will make the files inaccessible under Ubuntu / Linux.

Refer to this thread for more info on it, since I don't actually have a Sansa device of anykind.  :)
