### Post by blackpaw on 2007-09-11
just to be a fair opposition here:

MythTV is nice, very nice actually. 
But it is very much TV-centered. (timeshifting, recording, programming, etc...) 
If you are looking for a media center that plays TV, movies and music (and some plugins, like pictures and weather, you name it...) then MythTV is definitely for you.

If your main purpose for a media center is not watching/recording TV but having a frontend for various types of media (and other things like email, torrents, Network storage, server functions, etc etc.. ) and you know precisely what you want it to do and are not afraid of some commandline hacking, then you should give freevo a try. I admit, it's a bit more "technical" but it is designed to be versatile and flexible, meaning it can be customized to your needs/wishes/desires.

I tried MythTV and to me it looked like windows media center, which was odd (in my opinion) so I went for freevo and I have a nice media center now, with TV function being disabled entirely. (which MythTV didn't let me do)
