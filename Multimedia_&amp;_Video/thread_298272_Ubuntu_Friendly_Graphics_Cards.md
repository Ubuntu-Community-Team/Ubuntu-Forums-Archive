### Post by ajgreeny on 2006-11-12
If you should ever need proprietary drivers instead of the supplied open source ones, geta nvidia card rather than ati.  Ati can be very difficult to set up with their proprietary drivers (though it can be done) and nvidia are much easier.  Now that ati and AMD are the same company this may all change.

Does your current card not support 1920x1200 res or have you just not set up the xorg.conf file to display that resolution?  It could be that 1920x1200 is possible if you edit the modes or something in that file to cover the resolution you want.
