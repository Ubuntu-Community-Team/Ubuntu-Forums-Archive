### Post by AD-RS1600i on 2006-11-09
After some research i think i've found the problem,

From my understanding of the documentation, as standard Ubunto isn't set up with file formats which are commercial/copyrighted, because this goes against the philosophy of Ubunto.

Luckily we've been provided with a method of installing these codecs via this page

[https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)

Which has sorted 75% of my issues with my video files, the other 25% i can live with :-):D 

That's something I would never have considered being from a Windows background, to me simply there was a problem with the player....
