OK. this the way that i install my AT&#304; readon x200 to 6.10 edgy.

if you have try this method please forgive me.

first add this repos to your sources.list


deb [http://tr.archive.ubuntu.com/ubuntu/](http://tr.archive.ubuntu.com/ubuntu/) edgy main restricted
deb-src [http://tr.archive.ubuntu.com/ubuntu/](http://tr.archive.ubuntu.com/ubuntu/) edgy main restricted
deb [http://tr.archive.ubuntu.com/ubuntu/](http://tr.archive.ubuntu.com/ubuntu/) edgy-updates main restricted
deb-src [http://tr.archive.ubuntu.com/ubuntu/](http://tr.archive.ubuntu.com/ubuntu/) edgy-updates main restricted
deb [http://tr.archive.ubuntu.com/ubuntu/](http://tr.archive.ubuntu.com/ubuntu/) edgy universe
deb-src [http://tr.archive.ubuntu.com/ubuntu/](http://tr.archive.ubuntu.com/ubuntu/) edgy universe
deb [http://tr.archive.ubuntu.com/ubuntu/](http://tr.archive.ubuntu.com/ubuntu/) edgy-backports main restricted universe multiverse
deb-src [http://tr.archive.ubuntu.com/ubuntu/](http://tr.archive.ubuntu.com/ubuntu/) edgy-backports main restricted universe multiverse
deb [http://security.ubuntu.com/ubuntu](http://security.ubuntu.com/ubuntu) edgy-security main restricted
deb-src [http://security.ubuntu.com/ubuntu](http://security.ubuntu.com/ubuntu) edgy-security main restricted


then update your apt.

add this to and of your xorg.conf file

Section "Extensions"
Option "Composite" "0"
EndSection

than install the drivers.

sudo apt-get update
sudo apt-get install linux-restricted-modules-$(uname -r)
sudo apt-get install xorg-driver-fglrx
sudo depmod -a
sudo aticonfig --initial
sudo aticonfig --overlay-type=Xv

and than install the kernel module

---------------------------------
sudo rmmod -f fglrx

if this command do nothing pass this and write the next code.

sudo depmod -a
sudo modprobe fglrx

than restart your PC. it is all this i do.

control your installation with 

fglrxinfo

ok.
