### Post by d3v1ant_0n3 on 2006-12-27
It's one of those things about Linux- because anyone can have access to the source code of the OS, if someone doesn't like the way a distro works/wants more bleeding edge/wants to have a specific feature set, etc, they're free to go off and make their own distro. Some are more user friendly than others (Ubuntu!), some have a somewhat elitest user base (Not Ubuntu!), there are various package management systems (the way the OS installs new apps etc). It's ALL about choice in Linux Land:D  It's great, but can be somewhat overwhelming at first if you're coming from 'Any colour you like as long as it's black' type Windows world.;)

### Post by d3v1ant_0n3 on 2006-12-27
Yep- Its stunningly easy to erase and format the hard drive (check and option, click 'Next'). It's generally easier to start with Windows and then install Ubuntu rather than the other way round- Windows seems to like to think it's going to be the only OS on a system- Ubuntu (and many other distros) bootloader - GRUB - will happily recognise an existing Windows installation, but Windows own bootloader doesn't seem to like Linux installations (without a specific 3rd party driver, Windows won't even recognise linux partitions).

I'd give the live cd a try, make sure all your major hardware is recognised, and go from there.

I found [this ](http://www.psychocats.net/ubuntu/index) page to have a great deal of useful information for new users. And don't forget, any problems you run into, no doubt someone on these forums has the answer, or can at least point you in the direction of an answer. And you (probably) won't even get told to RTFM!
