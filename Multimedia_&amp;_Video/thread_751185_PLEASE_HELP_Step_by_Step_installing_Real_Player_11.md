### Post by viciouslime on 2008-04-11
Right click on it, select properties, go to the permissions tab and tick the box where it says allow executing file as program. Then double click it again and press run :)

Chances are though you'll need to start it as root, in which case, open a terminal, navigate to the folder you have downloaded it to, then run the command: gksudo ./<filenamehere>
