### Post by diepruis on 2007-09-16
This is for everyone using Intel HDA sound under Linux and the related kernel module snd-hda-intel.

I just downloaded and installed ALSA v1.0.15rc1 a few days ago and it feels like it makes a marked improvement to the sound quality of this device, which was really poor a few months ago. I'm now able to set PCM to 100% and front to about 90% in alsamixer without distortion. It's still nowhere near the performance of the chipset under windows... but it's getting there.

The change log does mention quite a few changes to the source for this module, but I'm still unsure as to whether it has really helped.

What are your experiences with this new version? Does this remove the stuttering sound problem in Unreal engine games?
