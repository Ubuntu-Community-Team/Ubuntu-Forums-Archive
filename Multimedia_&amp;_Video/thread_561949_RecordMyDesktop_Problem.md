### Post by eye208 on 2008-01-25
> **Kadrus said:**
> Alright i have installed recordmydesktop.
When i record something...I open the file to see the video but it's all black....
Can anyone Help?!!
If you want to capture the output of an OpenGL application (including Compiz), you need to enable "full shots at every frame".

Also note that you can't capture the output of a media player that is using XVideo overlay. In order to do that, switch the media player to software rendering mode (aka "X11").
