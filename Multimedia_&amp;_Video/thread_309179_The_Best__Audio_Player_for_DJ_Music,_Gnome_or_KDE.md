### Post by djlosch on 2006-11-30
tier 1 media players = windows media player and itunes (most media types, and smart playlisting)
tier 2 media players = mplayer, xine, and vlc (crappy interface/lacking smart playlisting but they play anything) and amarok (almost all audio formats, smart playlisting, but no video formats)
tier 3 media players = xmms, rhythmbox, winamp, and the list goes on.

personally i use amarok and mplayer.

### Post by matthias_k on 2007-11-06
Exaile! ist my favourite audio player right now. I'm no KDE user but I think it's an AmaroK clone for GNOME/Gtk2.

So, if you're using GNOME, get Exaile!, if you're on KDE try AmaroK. Both are feature rich and have an extremely well designed user interface IMHO.

If you're looking for a simple WinAMP clone, Audacious is your best bet.

An interesting ongoing effort is Songbird, which tries to augment the Mozilla family with a media player, but it's still alpha software.
