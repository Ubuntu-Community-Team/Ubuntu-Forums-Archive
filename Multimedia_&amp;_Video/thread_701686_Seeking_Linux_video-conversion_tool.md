### Post by stooshbunutu on 2008-02-19
There isn't an all in one I know of but here is a good two step way.

First use[ WinFF]("http://biggmatt.com/files/winff-0.33-i386.deb") a front end for the ffmpeg codec to convert the initial video format to a .wmv format.

Second use [Mobile Media Converter]("http://www.miksoft.net/products/mmc-lin.tar.gz") to change the .wmv file into a .3gp file for use with mobile phone.

It's not a pretty process but I use it and it does work. This process can be reversed for phone to computer media.

Hope this helps :)
