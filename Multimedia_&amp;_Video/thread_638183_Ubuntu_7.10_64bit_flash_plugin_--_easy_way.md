### Post by srmantis on 2007-12-13
there is a easy way to install flash, first uninstall gnash and flasplugin-nonfree (gnash is not good).
second: to go to direction web with flash available and above the windows there is a bottom install plugin, click the bottom:
two options:
1 flashplugin-nonfree
2 gnash

select flashplugin-nonfree and to install

flashplayer-nonfree: to use in your browser (firefox, swiftfox,epiphany, etc)
gnash: to use for game flash in your hard disk
