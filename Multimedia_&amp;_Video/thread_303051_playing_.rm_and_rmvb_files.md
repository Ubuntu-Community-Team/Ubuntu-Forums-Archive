### Post by Kalinda on 2006-11-19
Okay, you need to have w32codecs and libxine-extracodecs (if you're using xine) installed. I'm not sure about you, but I use Kaffeine player and it plays them rather well.

I dunno how well Totem will handle them, it should play them fine if you're using xine; gstreamer doesn't appear to work. Neither does VLC Player, oddly. I know MPlayer won't play them, either, at least not on my system.

If you haven't got w32codecs, I would suggest just getting [Automatix]("http://www.getautomatix.com"), as I'm pretty sure it installs the newest version. Otherwise it's a huge pain to install them.
