Well, After spending two days trying to find out if it were possible to rotate a wide flat panel, I found it almost painfully simple...

I have Xinerama enabled...not twin view.  dont know yet if it would work in Twin VIew....
I am using an Nvidia quadro 4 700XGL card...

according to the [Appendix D]("http://http.download.nvidia.com/XFree86/Linux-x86/1.0-8178/README/appendix-d.html"), X Config Options, available from Nvidia, you can add the 

Option "Rotate" "left"  (or right....) either in the screen or Device section.  I happened to enter my in the Screen section,  with the simple thought that it is the screen I want to rotate...
my entire xorg.conf file looks like this...
