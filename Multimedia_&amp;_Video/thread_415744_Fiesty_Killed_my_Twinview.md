[B]How to Fix:
[I][U]
In Gnome:[/U][/I]
```
sudo gedit /etc/X11/xorg.conf
```
Remove this line in the "Module" section:
```
	Load	"dri"
```
Save and ctrl-alt-backspace.
[I][U]
In single user mode:[/U][/I]
```
sudo nano /etc/X11/xorg.conf
```
Remove this line in the "Module" section:
```
	Load	"dri"
```
Save
Test configuration:
```
startx
```

If it works reboot
