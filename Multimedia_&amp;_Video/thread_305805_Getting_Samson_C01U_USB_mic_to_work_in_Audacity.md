### Post by k1001001 on 2006-11-24
UPDATE: I got the mic to work through JACK, which successfully integrates with Ardour. I think I would reccommend using Ardour anyways. The layout is pretty easy to use, especially if you are familiar with SONAR. So if you are having a similar problem, just try using Ardour instead of Audacity.
