### Post by philidox on 2008-02-28
k9copy is it!  Its a kde program but it works perfectly under Gnome. No magic, geek-talk, or sorcery needed.  Its pretty straight forward program that makes perfect copies of dvd movies.  You may have to install codecs before using it lucky I got the url to the web page that explains how to install those codecs.  [https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs](https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs).  Enjoy!
