### Post by mindwarp on 2006-06-10
no not really common settings, just found them on google.  That problem is because your X session is running as matoxxx, but you are trying to run the control panel as root.  This fails because the only user autherized to run programs on your screen is you.  You can get around this by running 
```

#xhost +

```

Be advised that this will open up your system severely as now anyone who can connect to the X port can open applications etc.
