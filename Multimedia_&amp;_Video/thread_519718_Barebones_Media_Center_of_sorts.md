### Post by Visti on 2007-08-07
Hi, guys! I'm getting real tired of changing my DVD players every few months. It seems no matter what I buy or how well I treat them, they never cease to fail in new and, quite frankly, intriguing ways. For example, my latest player simply refuses to run original DVDs now, but has no problem what-so-ever with VCD or any burnt DVD.

Anyway, before I get sidetracked here.. I was thinking that it would probably be better for me to build a really barebones setup in a small cabinet and use that for playing DVDs. As far as I see it, there shouldn't be a lot of drawbacks and it would have the added opportunities for USB Flash Disk playing and stuff like that and I can replace a particular item if it goes bad (not that I'm seeing a lot of that at all outside stand-alone DVD players). 
But my problem is that I'm not really sure how low-end I can go, before it would get annoying and be a hinderance. I mean, DVD and audio playback would have to be smooth, but other than that, I don't really need anything else. Anyway, maybe somebody has some insight into this.

Things I would like to have:
DVD Playback *(That's probably a DVD drive + some software)*
Audio Playback *(Mp3 + CD, probably not even contained on a drive)*
Hard Drive *(solely for OS installation and some overflow, maybe)*
Low-noise and small size would probably be nice.
TV-out *(So I probably need to shell some on the card there, I guess?)*
Remote *(I would like to minimize actual OS interaction as much as possible, I figured this can be achieved with "auto-play on insert" kinda functions and then some sort of remote or bluetooth controller device, perhaps? Am I way off?)*

I don't really care about TV-tuning, what-so-ever. So it seems pretty do-able, no? What I'm looking for is anybody to pitch something obvious that I might have missed, suggestions for hardware (especially the small-size cab, I have no idea what's out there) and maybe what I absolutely need in terms of RAM and CPU and such.

I hope somebody gives me some input or moves this if it's in the wrong place! 

Thanks in advance, Visti.
