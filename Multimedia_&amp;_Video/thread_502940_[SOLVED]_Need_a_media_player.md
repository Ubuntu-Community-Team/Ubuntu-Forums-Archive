### Post by Washington Irving on 2007-07-17
I think there should be audio and video players already installed, as default (look in the Applications menu). You might be wanting some multimedia codecs though which you can get by following the instructions at this page: [http://ubuntuguide.org/wiki/Ubuntu:Feisty]("http://ubuntuguide.org/wiki/Ubuntu:Feisty"). You are using Ubuntu Feisty Fawn right? otherwise there are links to older versions of the guide on the same page.

There are many players which you can install and uninstall from Applications>Add/Remove...
I use Amarok for music because I have a large music collection which it handles well and I prefer it to the others but you could try a few out to see which you like best.
