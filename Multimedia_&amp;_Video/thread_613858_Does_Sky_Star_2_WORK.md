### Post by steefjeqv on 2007-11-15
Hi,

If I'm not mistaken : AltDVB only is available as a Windows application.
But it should work with a Sky Star2 DVB-S.

The easiest program to watch digital TV on Ubuntu is Kaffeine.
It should recognize your Sky Star2. Scan for channels and watch TV.

You can install it using Synaptic (Ubuntu's software manager), or just type in terminal :

sudo apt-get install kaffeine

If your interested in more sophisticated applications try :

- VDR
- MythTV
- Freevo
- Klear

These apps aren't as easy to setup as Kaffeine !

Hi,

I don't think Kaffeine supports the use of a smartcard. This means you can only watch free-to-air channels.

VDR does support the use of a CAM in order to watch scrambled channels.
This application requires a certain (linux) knowledge to set up, but there's lots of info to be found on the internet (google).
