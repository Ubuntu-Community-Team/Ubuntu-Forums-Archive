### Post by aeiah on 2008-04-04
mythtv and linuxmce are really the only complete, all encompassing options. 

you have a myth backend and myth frontends. your server would have the backend on, of course. as mentioned, mythbuntu is a distro based on ubuntu that has mythtv on it.

linuxmce is supposed to be quite easy to get to grips with although ive never tried it.

if you're streaming to things you dont want to put myth on, you might consider vlc or xine for live streaming, and an UPnP media server for the rest, like mediatomb for example
