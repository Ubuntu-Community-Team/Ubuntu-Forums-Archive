### Post by fenian on 2007-02-11
If I get you right I think you want two seperate X screens.You can do this acouple of ways 

1)You can follow this guide (second part for seperate X)...

[https://help.ubuntu.com/community/NvidiaTVOut](https://help.ubuntu.com/community/NvidiaTVOut)

OR

2)If you have the newest Nvidia drivers (1.0-9746) you can use the command...
> 
sudo nvidia-settings

to bring up the nvidia-settings window.From here make sure both monitors are detected than choose
X server display configuration and configure both monitors to use separate X screens set the resolution you want and click the Save to X Configuration file button.Exit and restart X (ctrl+alt+backspace).
