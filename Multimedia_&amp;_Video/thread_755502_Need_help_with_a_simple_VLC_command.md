### Post by chewearn on 2008-04-15
> **Afkpuz said:**
> Ok, I need the command to set deinterlacing for movies to "blend"

Go to:
Settings > Preferences > Video > Filters
Enable checkbox for deinterlacing.
Make sure "deinterlace" appear in the textbox below all the checkboxes.

Then, go to:
Settings > Preferences > Video > Filters > Deinterlace
Select "Blend" for the deinterlace mode.
