### Post by Voynix on 2007-07-19
I guess you are running desktop effects (beryl, compiz or compiz fusion). It is a known bug in java that has been discussed in many threads. There are some workarounds. 

You could turn off desktop effects while running java applications, upgrade to java 1.6 update 2, or use some of the solutions mentioned in the following threads.

see [beryl thread]("http://ubuntuforums.org/showthread.php?t=362821"), [another beryl thread]("http://ubuntuforums.org/showthread.php?t=484852&highlight=beryl+java"),  or [ this one]("http://ubuntuforums.org/showthread.php?t=445344&highlight=beryl+java").
