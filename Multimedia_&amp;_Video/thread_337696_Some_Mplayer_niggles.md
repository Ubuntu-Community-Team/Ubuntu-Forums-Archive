### Post by srunni on 2007-01-13
Not sure about 1,2 or 4, but for #3, the problem is that by default, mplayer has no GUI. The GUI you get when you install the package "mplayer" is gmplayer, one of the various frontends that have been created for mplayer. I'm sure you could find other frontends, but you may have to compile them manually (which you don't want to have to do).
