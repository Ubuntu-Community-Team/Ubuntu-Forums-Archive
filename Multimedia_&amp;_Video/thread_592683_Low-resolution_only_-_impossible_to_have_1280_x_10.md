### Post by prshah on 2007-10-26
Why not post your /etc/X11/xorg.conf file here? That will give a pointer:

Before that, try this:

-> Open a terminal (Applications->Accessories->Terminal)
-> type "cp /etc/X11/xorg.conf ." (no quotes)
-> type "sudo dpkg-reconfigure xserver-xorg" (no quotes)
-> follow the on-screen instructions 
-> When you get to the screen resolution part, select only the resolutions you intend to use and clear the rest
-> Save and exit
-> logout
-> on the login screen, press Ctrl+Alt+Backspace (NOT del !!): this will restart X (your screen may flicker momentarily).
-> if you run into problems (no screen, black screen, etc) press Ctrl+Alt+F1: This will drop you to a plain text console
-> login as usual
-> type "sudo cp xorg.conf /etc/X11/" (no quotes)
-> Now you are back to square one and can shift from here only once we take a look at your xorg.conf

Cheers,
PRS

### Post by prshah on 2007-10-28
OK, repeat te gedit command, then File->Save as and save it in your home folder.

Then type the below command in the terminal:

sudo dpkg-reconfigure xserver-xorg

as suggested in the config file.

Walk through the screens, mostly choosing defaults, and once finished, restart X or your computer and check if you can get a higher resolution.

From your xorg.conf file, looks like you had a failsafe config only; did you boot use the live cd or the alternate cd? Or some other special option during boot?

In any case, try the above, and let us know...
