### Post by psi36 on 2007-02-11
When I click on a playlist or mp3, Vorbis Ogg, whatever file in Firefox it plays it with the mplayer plug-in.

I would however like those files to open in Amarok or some other media player. I actually only want to use the mplayer plug-in for embedded video, but I don't want to remove it altogether.

When I look into my Firefox preferences > Content > File Types I don't see mp3 or ogg or any of those. Where can I change these options then?

---

### Post by mcduck on 2007-02-11
type 'about:config into firefox's address bar, find key 'browser.download.hide_plugins_without_extensions'andtoggle it'svalue to 'false'. Now you can change the setting in preferences > Content > File Types.
