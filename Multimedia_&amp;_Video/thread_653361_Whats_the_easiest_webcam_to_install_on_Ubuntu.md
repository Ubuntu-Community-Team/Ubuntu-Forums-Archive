### Post by TTL on 2007-12-30
spca5xx is a kernel module as far as I know. It looks like Ubuntu does not ship it in the default installation. If you want to get your camera to work, look if there are some new messages telling you that a driver is loaded after connecting the camera. You can view your system messages with executing dmesg in a console.
