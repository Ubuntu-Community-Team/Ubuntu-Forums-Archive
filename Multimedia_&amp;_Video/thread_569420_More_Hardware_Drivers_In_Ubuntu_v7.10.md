### Post by Hugolp on 2007-10-07
This has to do more with the linux drivers developers than with Ubuntu. You could send an email to the company as well to make some presure so they realize the importance of doing the drivers for linux.

With each release of Ubuntu a newer kernel is included, meaning that more hardware is suported. You can check [www.linuxtv.org](www.linuxtv.org). There youll find all the info about the drivers development for linux for any video hardware. Also, there are instructions on how to update your machine to install the lastest drivers if you need to. Sometimes thats interesting cause some drivers take their time to get onto the kernel, because it takes some time to check if a driver is stable enough to be included into the kernel.
