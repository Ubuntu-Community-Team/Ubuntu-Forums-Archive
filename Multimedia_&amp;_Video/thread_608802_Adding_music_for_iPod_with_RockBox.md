### Post by deruberhanyok on 2007-11-23
I've found drag and drop copying to be the easiest way; just copy the files to your iPod wherever you like (I just created a "Music" folder) and copy everything to that. Then you can rebuild the database (or just run the update option from the menu, should be faster) and you're good to go.

If you want to remove the files that iTunes scrambles when it puts them on the iPod, you could, if you want, just delete them from the iPod_Control/Music folder, but if you do this you may also want to go into the iPod_Control/iTunes folder and delete the iTunesDB file. I *think* this will clear the default firmware's database without issue.

You may want to actually go into iTunes and remove the files from the iPod using that, though, that way you can be sure the default firmware won't freak out if you boot into it for whatever reason.
