### Post by nick_h on 2007-10-27
If using Gnome you can install it from System -> Administration -> Synaptic Package Manager.
Once installed you can select it from System -> Preferences -> Screensaver.
I haven't used it but I'd keep the Gnome screensaver and not touch xscreensaver.
