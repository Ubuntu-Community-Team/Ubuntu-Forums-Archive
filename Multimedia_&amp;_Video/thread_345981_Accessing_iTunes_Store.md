### Post by cam_tram on 2007-01-28
There's a few options I know of:

1. Banshee supposedly has a unofficial plugin (it also appears that its unmaintained) that allows accessing the store.  Its not available as a package, and I'm not sure if it works, but you can get it from [here]("http://banshee-project.org/Plugins").

2.  Run Itunes in wine, this has problems too, Itunes isn't fully functional under wine.

3.  Use VMware Server to run XP as a virtual machine in Ubuntu.  This is probably the only fully functional solution.  Note though that with this and wine, you will only be able to use Itunes to play the tracks you buy.
