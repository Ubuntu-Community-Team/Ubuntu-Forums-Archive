### Post by booe on 2007-03-02
Maybe try [here](http://xinehq.de/index.php/releases#binaries) at the official xine website. They have a bunch of different binaries ready for download for pretty much every major distribution except ubuntu. Therefore you'd need to use some tools like dpkg to correctly handle (install) the RPM files you download. If you're a little more advanced, you could also recompile the whole thing from source, and install it that way.

None of this is really recommended as it could break stuff, or cause a lot of headaches.
