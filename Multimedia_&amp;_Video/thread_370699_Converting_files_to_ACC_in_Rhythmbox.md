### Post by doclivingston on 2007-03-02
Is there any particular reason to do that? iTunes and iPods can play MP3 files fine, and converting between lossy formats loses quality.

Rhythmbox 0.9.8 can, provided you have the right GStreamer plugins installed (from gst-plugins-bad) and an encoding profile set up. There is a program called "sound converter" that can do this too.
