### Post by cor2y on 2008-01-16
The Extra libs/plugins things that enable dvd playback,wmv/asf,avi,mp3,flac etc those you could remove but i don't recommend the base install some programs you might later install or already have depend on it. So gstreamer-ugly, gstreamer-ffmpeg, gstreamer-bad could probably be removed with no problems just remember if a program that does video playback etc gives an error that could be the reason,those missing plugins.
