You are using Xinerama, but I think it is better to use TwinView, which is supported by you're driver.  So use the guide in my signature (Dual Monitor), scroll to the bottom of the post, and try TwinView...If you have any problems post in that thread...

just a few steps before the actualy howTo, back up your xorg.conf file:

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup

and then create a new xorg.conf file (w/ the second monitor detached):

sudo dpkg-reconfigure xserver-xorg

then then follow my instructions in the HowTo thread...
