### Post by heimo on 2007-04-15
Run:
```
xrandr --query
```At the end, you should see which orientations are possible, eg:
```
Current rotation - normal
Current reflection - none
Rotations possible - normal left right 
Reflections possible - none
```Some drivers may need option in xorg.conf to enable rotation. Some drivers don't support all rotations.
