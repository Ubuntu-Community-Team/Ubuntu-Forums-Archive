### Post by tseliot on 2006-09-04
1) Did you already set up your xorg.conf for the Dual Head?

If so, try this command (we should try to solve 1 problem per time):
```
sudo dpkg-reconfigure xserver-xorg
```

select the "nvidia" driver and press ENTER whenever you don't know the answer to the other questions.

Then type:
```
sudo /etc/init.d/gdm restart (or "kdm restart" if you use kdm)
```

If that doesn't help, try point 4 of the Problems Section of my guide:
[http://doc.gwos.org/index.php/Latest_Nvidia_Dapper](http://doc.gwos.org/index.php/Latest_Nvidia_Dapper)
