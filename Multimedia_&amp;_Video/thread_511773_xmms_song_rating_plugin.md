> USAGE:

There are two different plugin modes which depends on the shuffle-toggle key:

1. shuffle: off

   The plugin just catches data. If you skip a song away the plugin decreases

   the assessment of this song. If you have heard a song completely its

   assessment will be increased. You can configure the decrease/increase values

   in the configuration window.



2. shuffle: on

   The plugin catches data further, but also implements a new shuffle.

   Songs with high assessment will be played privileged.
