### Post by angryfirelord on 2007-12-16
I'm taking a shot in the dark, but try these two options:

First remove and then reinstall your drivers. For example, if I used the nvidia-glx package to get 3D acceleration, I would do this:
```
sudo apt-get --purge remove nvidia-glx
```
```
sudo apt-get install nvidia-glx
```
Then, reconfigure your xorg.conf
```
sudo dpkg-reconfigure xserver-xorg
```
(Remember to select nvidia as your driver)
