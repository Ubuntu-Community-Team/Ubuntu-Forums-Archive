> …ps: one more question.. how or where would i choose mplayer to be default player for vmw and other file types?…
I don't know if this is the official way, but you can change it for each file type was right-clicking on the file, going to properties and then to the 'open with' tab. There should be a list of media players there - simply just select the one you want to use as the default. If your media player isn't there, you can add it to the list by clicking on the 'add' button. Unfortunately you have to do this for each file type you want to change.

Hope that helps.
