### Post by ajgreeny on 2008-03-02
Go with audacity!  It is one of the best audio file editors, and the fact that it's open source is all the more amazing when you see what it can do. Get the LADSPA tap-plugins and swh-plugins as well and you can do amazing things to wav, mp3, ogg and other file types.  It's the greatest, I use it a lot for various stage sound effects, and with a bit of fiddling I can get just about anything I want.  Get a good microphone and you can also make terrific recordings which you can then edit as you wish.

### Post by ajgreeny on 2008-03-02
Both sets of plugins I mentioned are in the repos.  Just make sure you have all repos activated and you should find them.  Just install as with everything else on ubuntu, and they will appear in the audacity Effect menu.  There will be about 130 different plugins in total plus the 15 -20 that come with audacity by default.
