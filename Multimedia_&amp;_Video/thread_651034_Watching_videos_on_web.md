### Post by ray bot on 2007-12-27
Gnash is not able to play all Flash videos.  It has full support for Flash 7, but if the video you're trying to watch uses the new features introduced in Flash 8 and 9, then it probably won't work with Gnash.  You can install the proprietary Flash plugin from Adobe with this command: (make sure you have the multiverse repository enabled first) ```
sudo apt-get install flashplugin-nonfree
```That one should be able to play all Flash videos.
