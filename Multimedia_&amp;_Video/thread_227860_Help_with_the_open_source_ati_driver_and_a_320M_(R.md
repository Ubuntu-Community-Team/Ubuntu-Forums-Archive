### Post by Dev_Null on 2006-08-24
I'm running a HP pavilion with radeon 320M and used the xorg.conf settings from [http://weblog.zoup.org/index.php?op=ViewArticle&articleId=12&blogId=1](http://weblog.zoup.org/index.php?op=ViewArticle&articleId=12&blogId=1) There's also stuff about installing things but everything comes with ubuntu dapper, just changed my xorg.conf file and got +- 250 fps.

Device section
```

Section "Device"
	Identifier	"ATI Technologies, Inc. Radeon 320M (RS200 IGP)"
	Driver		"radeon"
	BusID		"PCI:1:5:0"
	VideoRam 	65536
	Option 		"DPMS"
	Option 		"AGPMode" "4"
	Option 		"AGPFastWrite" "On"
	Option 		"EnablePageFlip" "On"
	BoardName 	"Mobility IGP320m"
	Option 		"AGPMode" "4"
EndSection
```
