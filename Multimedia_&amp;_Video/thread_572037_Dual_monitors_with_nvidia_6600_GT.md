### Post by nzadLithium on 2007-10-10
run 

sudo nvidia-settings

click x server display configuration

adjust everything so its how you want.
mine is configuration: twinview
resolution 1024x768 Auto

Press detect displays if two monitors dont show up at first.

once you've finished press save to x configuration file then close nvidia settings.

Hit Ctrl-alt-backspace (this will log you out so save anything else you might have been doing) and you should have dual monitors.
