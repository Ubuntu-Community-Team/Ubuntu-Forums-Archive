### Post by Aikon- on 2006-10-04
Hal,

I recommend giving [Amarok]("http://amarok.kde.org/") a go.. I've had no problems whatsoever using it to sync with my iPod (30GB Video). Not sure how well transfering playlists work though.. (I don't use them ;) ). The only slight problem I've had with using Amarok is that you can't use Musicbrainz for tagging MP3 files out-of-the-box, as Ubuntu has compiled libtunepimp2 without MP3 support.

Hope this helps!
