### Post by damu on 2006-12-27
Well, there is an Ubuntu-studio project which is supposed to provide what you are looking for in few months.

Before there was an 'Ubuntu Studio' forum which aimed at answering your question, so that you could discover linux tools for audio production (jack as the core system, ardour, rosegarden, hydrogen, freewheeling, linuxsampler, fluydsynth, ZynaddSubfx, etc). It has been deleted for some reasons that I don't get, making the developpement of the Ubuntu-studio distribution closed from its base - ie its community of potential users / testers. 
There is still a wiki somewhere and Luneo, an Ubuntu user and dvelopper delivered some package for a real-time kernel (the central missing piece yet for audio prod in the repositories of ubuntu).

If you want to experiment/produce now with linux, I would suggest to go for other solutions than Ubuntu :

[Studio-to-go]("http://studio-to-go.com/") is your best bet if you want something working out of the box with Vst support, and that you are ready to put £40 /60&#8364;.
You can download a demo of their live cd to test it
The version 2.0 is in beta test and looks very promising with  the last KDE environment, optimization for multi-core systems, the best of audio softwares (the creator of studio-to-go also develop some of the core open source applications for audio like Rosegarden).

[URL="http://64studio.com/"]
64studio[/URL] is rock solid and free. It doesn't come as a live cd, doesn't support VST, and is primilarly developped for 64 bits systems. They have a 32 bit version.

Enjoy!
