### Post by dolson on 2006-03-18
Many people seemed to have misunderstood what my project was initially about. It was not a distro, and still is not a distro. All of my effort goes into Ubuntu directly (although I have been asked to work on Debian now).

HOWEVER, I have it on good authority that Mark has something in the works for the Dapper+1 timeline - a multimedia-oriented Ubuntu derivative distro.

There is no further official information at this time that I can give.

However, let me speculate that it will be called Mubuntu, since [www.mubuntu.org](www.mubuntu.org) is registered to Canonical.

I should know more about this next week, and I will announce whatever I can.

I really hope that it's a derivative just like Kubuntu and Edubuntu are, so that we can just install normal Ubuntu and then just apt-get install mubuntu to add in all the multimedia stuff, and that it is official, and has a realtime kernel package, but I don't know any of this info yet. If it's a separate distro that I have to install separately, then the future of The Ubuntu Studio Project will stick with standard Ubuntu. But if it's as I hope, it will be easier for all of us, and the Ubuntu Studio Project will embrace Mubuntu, although it will retain the current name.

As Mark would say, more in due course!
