### Post by fenian on 2007-02-08
Use the envy script to get the latest nvidia drivers...

[http://albertomilone.com/nvidia_scripts1.html](http://albertomilone.com/nvidia_scripts1.html)

Follw the instrucions being sure to remove all previous nvidia drivers before installing the new ones.After you have installed the new drivers and are back in the normal Ubuntu enviroment type in a terminal...

> sudo nvidia-settings

This will open a window from where you can detect an enable tvout.Go to X server display configuration (on the left hand menu).If your tv is not shown in the layout window click the detect displays button.Select the TV in the layout screen click the configure button and choose separate X screens click the Save to X Configuration button.Close the window and restart X (ctrl+alt+backspace).

### Post by fenian on 2007-02-09
You would want to change the section you posted above to look something like this...
> 
Section "Monitor"

    # HorizSync source: edid, VertRefresh source: edid
    Identifier     "Monitor" #CRT
    VendorName     "Unknown"
    ModelName      "WAN LCD MONITOR"
    HorizSync       30.0 - 60.0
    VertRefresh     55.0 - 75.0
    Option         "DPMS"
EndSection

Section "Monitor"

    # HorizSync source: builtin, VertRefresh source: builtin
    Identifier     "Monitor" #TV
    VendorName     "Unknown"
    ModelName      "TV-0"
    HorizSync       28.0 - 33.0
    VertRefresh     43.0 - 72.0
    Option         "DPMS"
EndSection

Section "Device"
    Identifier     "Device0"
    Driver         "nvidia"
    VendorName     "NVIDIA Corporation"
    BoardName      "GeForce PCX 5750"
    Option  "NoLogo"        "true"
    Option  "RenderAccel"   "true"
    BusID          "PCI:1:0:0"
    Screen          0
EndSection

Section "Device"
    Identifier     "Device1"
    Driver         "nvidia"
    VendorName     "NVIDIA Corporation"
    BoardName      "GeForce PCX 5750"
    BusID          "PCI:1:0:0"
    Screen         1
EndSection

Section "Screen"
    Identifier     "Screen0"
    Device         "Device0"
    Monitor        "Monitor"
    DefaultDepth    24
    Option  "UseDisplayDevice" "CRT"
    SubSection     "Display"
        Depth       24
        Modes      "1600x1200" "1280x1024" "1024x768" "800x600" "640x480"
    EndSubSection
EndSection

Section "Screen"
    Identifier     "Screen1"
    Device         "Device1"
    Monitor        "Television"
    DefaultDepth    24
    Option  "TVOutFormat" "SVIDEO"
    Option  "TVStandard" "NTSC-M"
    Option  "UseDisplayDevice" "TV"
    SubSection     "Display"
        Depth       24
        Modes      "1600x1200" "1280x1024" "1024x768" "800x600" "640x480"
    EndSubSection
EndSection


In the second "Screen" section you will need to set the tvout format to suit your setup,either SVIDEO or COMPOSITE (all caps for both).You will also need to change the TVStandard to suit your region NTSC-M for N. America PAL for Europe. 

You will also need to set up the section "ServerLayout"  like this...

> Section "ServerLayout"
        Identifier      "Basic Layout"
        Screen 0        "Screen0"
        Screen 1        "Screen1" rightof "Screen0"
        InputDevice     "Generic Keyboard"
        InputDevice     "Configured Mouse"
EndSection
