### Post by WinterWeaver on 2007-04-29
ok, first off, make a backup of your xorg.conf.

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak
```

next open your xorg file as root. I usually use the terminal for this.
```
sudo nano /etc/X11/xorg.conf
```

scroll down to this section: 
```
Section "Screen"
    Identifier     "Default Screen"
    Device         "NVIDIA Corporation NV36 [GeForce FX 5700LE]"
    Monitor        "Generic Monitor"
    DefaultDepth    24
    SubSection     "Display"
        Depth       1
        Modes      "1024x768" "800x600" "640x480"
    EndSubSection
    SubSection     "Display"
        Depth       4
        Modes      "1024x768" "800x600" "640x480"
    EndSubSection
    SubSection     "Display"
        Depth       8
        Modes      "1024x768" "800x600" "640x480"
    EndSubSection
    SubSection     "Display"
        Depth       15
        Modes      "1024x768" "800x600" "640x480"
    EndSubSection
    SubSection     "Display"
        Depth       16
        Modes      "1024x768" "800x600" "640x480"
    EndSubSection
    SubSection     "Display"
        Depth       24
        Modes      "1024x768" "800x600" "640x480"
    EndSubSection
EndSection
```

and change all the display modes to read (just add the extra resolutions in the Modes line:
```
Modes    "1280x1026" "1024x768" "800x600" "640x480"
```

when done,  hit ctrl + o, to save your file, and then ctrl + x to exit.

before restarting, make sure you know how to restore your xorg in case something bad happens :P
```
sudo cp /etc/X11/xorg.conf.bak /etc/X11/xorg.conf
```
now restart your X server with ctrl + alt + backspace

Hope that helps :)

### Post by Talon2 on 2007-04-29
> **sparticus06 said:**
> I installed my Nvdia drivers for my Geforce 5700, I dont notice anything different. Everything works fine. 

when I type
sudo nvidia-settings i get the settings window and it detects everything. 

if that means it worked, how do I make my resolution higher?

This might or might not be the fix but it is what worked for my box with an nVidia card:

- Make a backup of your xorg.conf

- Add the following to the Device section of xorg.conf:    Option "DynamicTwinView" "false"

Example:

Section "Device"
	Identifier	"Generic Video Card"
	Driver		"nvidia"
	Busid		"PCI:1:0:0"
	Option		"AddARGBVisuals"        "True"
	Option		"AddARGBGLXVisuals"     "True"
	Option		"NoLogo"                "True"
        Option          "DynamicTwinView"       "false"
EndSection

---
