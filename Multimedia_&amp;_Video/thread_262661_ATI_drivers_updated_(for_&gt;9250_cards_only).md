### Post by IanW on 2006-09-22
The 32-bit ATI proprietary video driver "fglrx" has been updated to version 8.29.6 and is available from ati.com as usual.

[COLOR="Red"]HOWEVER[/COLOR], This comes with the following warning for users of older cards:- 

ATI Proprietary Linux x86 Display Driver

As of driver version 8.29.6 support for the following products is [COLOR="Red"]no longer included[/COLOR]:

    * Radeon® 8500/9000/9100/9200/9250
    * Mobility™ Radeon® 9000/9100/9200
    * Radeon® IGP 9000/9100/9200

Users with these products should use driver version 8.28.8

The x64 version seems to be still at version 8.28.8
