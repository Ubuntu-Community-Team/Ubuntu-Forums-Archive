### Post by DirtDawg on 2007-06-09
HI Qix

Look for a folder named ".Trash" on your iPod. Often on external HDs, if something is deleted it is placed in a temporary .Trash folder instead of being deleted, so unfortunately you'll need to manually delete this folder often (note: there's an "include delete command" in Nautilus under the behavior tab of the prefrences menu that creates a "delete" command on right click. Use with caution).

That said, using an alternate manager like Banshee, GTKPod, or Amarok may eleminate the problem.
