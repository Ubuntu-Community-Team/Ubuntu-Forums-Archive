### Post by Joeb on 2007-01-23
Is the "not enough space" error stating that the destination disk doesn't have enough space or that there isn't enough space on the partition to create the temporary file?

If you only have one dvd drive/burner, it first must make an image file, which if the dvd is full, is about 4.7gb.  You will need that amount of free space and then some to copy a full dvd.

If it truly is the dvd giving the error, are you sure that it is blank?  If you had tried to write to it previously and there was an error, it may not be completely empty.  Also, is it possible that you have dual layer burner and the original is on a dual layer dvd and you are trying to burn to a single layer dvd?  If so, then it is possible that your original does contain more than will fit.  If that is the case, you might check out k9copy which can shrink it down without loosing too much quality.
