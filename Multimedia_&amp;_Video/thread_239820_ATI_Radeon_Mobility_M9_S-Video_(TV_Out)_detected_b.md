### Post by rscarriere on 2006-09-23
Recently I started to run into problems where X would crash every time I tried to play a video (with totem, VLC, mplayer).   Turns out it was related to a known bug ([http://ati.cchtml.com/show_bug.cgi?id=179](http://ati.cchtml.com/show_bug.cgi?id=179)) in the ATI drivers (not sure why it is only showing up now though).

So if you get crashing you need to add the "ForceMonitors" option to the Section "Device" in my config above:
```

Identifier  "aticonfig-Device[0]"
...
...
Option 	    "ForceMonitors" "tv"

```

Alternatively (instead of using the "ForceMonitors" hack) - you could comment out the line:
```
#Screen       1 "fglTV" RightOf "aticonfig-Screen[0]"
```

And then just enable it after you plug in the TV.
