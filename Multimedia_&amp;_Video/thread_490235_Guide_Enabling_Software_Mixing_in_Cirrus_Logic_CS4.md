The particular card doesnot support Hardware Mixing under ALSA so different applications are not allowed to access the audio device at the same time. Software mixing overcomes that problem. [Here ]("http://sbjaved.blogspot.com/")is a How-to for enabling Software Mixing via "dmix" for this particular card.
