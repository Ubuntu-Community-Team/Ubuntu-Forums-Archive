### Post by Kateikyoushi on 2006-10-13
What is your video card ? Are you using the right driver ?

You can reconfigure X with the following command.

```
sudo dpkg-reconfigure xserver-xorg
```

Check if the resolution exists in your /etc/X11/xorg.conf file.

If still no go post your xorg.conf

```
cat /etc/X11/xorg.conf
```
