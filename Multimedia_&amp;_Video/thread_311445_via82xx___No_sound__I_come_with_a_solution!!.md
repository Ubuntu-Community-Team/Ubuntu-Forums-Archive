### Post by G3N3SIS on 2006-12-02
Hello all, running a Via82xx?  I just spent 6 hours trying to get sound on it.  If you are like me and don't have sound, then don't worry!  I have the easiest solution!!  If you haven't had sound since the start of the install like me, chances are this will work, but hey it's still worth a try.  Go ahead, take your live CD.  Shut down insert your cd, and boot up.  Enter LIVE CD mode, (The place where you installed it from.) Notice: MAKE SURE YOUR SPEAKERS ARE ON!  If you hear a sound when it starts loading up the User Interface like some sort of music, try some other applications you know will have sound.  I used GAIM and imed someone, I also connected to IRC to make sure I could here a little beep type sound.  If you can hear all of this, go ahead, and reinstall (click the install icon on desktop.)  Once you have reinstalled, you should succesfully have sound.  You can tell if you can hear a tribal sound on boot up, and then on log in and loading of User Interface; You should hear music and the tribal sound.  If so, welcome to Ubuntu - full.
