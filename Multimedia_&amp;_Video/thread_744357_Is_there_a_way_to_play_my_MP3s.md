### Post by Afkpuz on 2008-04-03
Well, Ubuntu will not just convert your music files for the heck of it.  Only if you tell it to will it convert them.  Chances are that you just need to install the mp3 codec.  Goto Applications=>Add/Remove

Click the drop down box that says Show and select "All availible applications"

Then search for "gstream".  Install every package with the word "gstreamer" in it and that will basically install all the codecs you'll ever need.
