### Post by elspic on 2007-04-28
dreadu: I was messing with a 7.1 surround sound card myself and finally stumbled on this for an .asoundrc:
pcm.ca0106 {
type hw
card 0
}

ctl.ca0106 {
type hw
card 0
}

pcm.!dmix {
type plug
slave {
pcm surround71
channels 8
}
}
pcm.!default {
type plug
slave.pcm "dmix"
slave.channels 8
route_policy duplicate
}

give that a shot. It works perfectly for me so far.
