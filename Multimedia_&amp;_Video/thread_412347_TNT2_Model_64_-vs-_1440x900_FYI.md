### Post by smsmith050 on 2007-04-18
Using an old TNT2 Model 64 with a new SyncMaster 906BW.
The horizontal sync width @ 1440 x 900 is too large for the TNT2 card. 
The card only support a horizontal sync width up to 256 but the monitor needs 1880 - 1472 = 408.
I can make the width smaller by using interlaced but the monitor won't work interlaced.
You need a Geforce4 or newer to support a horizontal sync width > 256.

### Post by smsmith050 on 2007-04-20
Yeah it does work after all. 

$ gtf 1440 900 60

  # 1440x900 @ 60.00 Hz (GTF) hsync: 55.92 kHz; pclk: 106.47 MHz
  Modeline "1440x900_60.00"  106.47  1440 1520 1672 1904  900 901 904 932  -HSync +Vsync

Add modeline to xorg.conf
Change Screen section to use new modeline
EndSubSection
        SubSection "Display"
                Depth           24
                Modes           "1440x900_60.00"  "1280x960" "1024x768" "800x600 " "640x480"
        EndSubSection
re-start xserver

The modeline generated by XFree86 Modeline generator would not work.

Excellent.  I'm happy for you.  There's nothing more disappointing than taking some new piece of hardware out of the box, plugging it in, and finding out you can't use it.

I don't know how you generated the modeline, but if the size/shape aren't quite right you might also want to try tweaking with xvidtune .  It will let you push buttons (up, down, left, right, fatter, narrower, taller, shorter) that will tweak the modeline.  I don't think it comes with X anymore, so you might have to install it.
