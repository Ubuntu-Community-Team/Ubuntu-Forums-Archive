### Post by Sarisar on 2006-10-08
So you use sound juicer to rip to MP3 / OGG / something and then burn those back to a new CD?

I have ripped MP3s using sound juicer (I started my collection on windows and MP3 and not sure the easy way to convert but that's another story).  I have found it does _not_ rip the text properly, but will however create filenames properly.  So I use easytag (it should be in the repository) to fill in the tags from the names of the files.

If yours is also not grabbing the text, then of course when you burn the CD using K3b it won't have the names because the tags are blank... so see if your files have tags :)
