Found [this]("https://help.ubuntu.com/community/i915Driver") in Community Docs - It's designed to help configure i915 chipset driver, but also has a section entitled:
**Display Available Resolutions**
which describes using:[SIZE=2]
[/SIZE]```
[SIZE=2]sudo 915resolution -l[/SIZE]
```
to discover what the available resolution modes are, as well as adding custom resolutions.

This presumes, of course, that you've already done:
```
sudo apt-get install 915resolution
```
