### Post by FuturePilot on 2007-05-04
```
sudo apt-get install xmms
```
Just put that in a terminal and you're done.

XMMS is sort of a pain if you want to manage a library. I would recommend Amarok for that. Or if you're using Feisty Exaile is a great player.
