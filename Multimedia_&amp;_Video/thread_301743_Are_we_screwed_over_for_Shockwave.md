### Post by dearephesus64 on 2006-12-14
Yeah, that does work, I just tried it!  Download the MS core fonts package,
```
sudo apt-get install msttcorefonts
```
then install Firefox 1.5.0.8 (Firefox 2 for Windows didn't work for me), then download the latest .exe files for flash 9 and shockwave.  It gives you a fairly reliable winedows-based browser.  Make sure you've already gone into winecfg and selected a sound driver under Audio.

It only hangs so bad it has to be killed manually sometimes... be careful with 4 Wheel Fury on Shockwave.com!  :)
