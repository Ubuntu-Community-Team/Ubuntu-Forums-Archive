**Ubuntu 7.10 ati multiple monitors with  different resolutions with Virtual Desktop**

I've seen a lot of posts and also seen the frustration it takes to setup a multi monitor setup with different resolutions and spanning the virtual desktop on Ubuntu 7.10. It seems like there are 2 types of setups. One is the following  where you can have multiple desktops but you can't drag your windows across screens. The other is the ideal one we are all used to in windows, where you can drag your windows across screens. The first link addresses the first issue, my solution addresses the latter.

[HTTP://hobbylobby.wordpress.com/2007/09/08/dual-monitors-in-ubuntu-xorgconf-driver-ati-card/](HTTP://hobbylobby.wordpress.com/2007/09/08/dual-monitors-in-ubuntu-xorgconf-driver-ati-card/)

The problem with this setup is you can't have multiple resolutions and drag the desktop across the screen. So in essence you don't really have a virtual desktop.

After fiddling with this for a few days, I was able to get the following working

Using a Dell D610 with and ATI x300 Mobile Radeon Card (128MB) and an external Dell 17' LCD display, I was able to finally get the final result working

Dell Notebook Display 1400x1050
Dell External Display 1280x1024

Unfortunately, I was not able to use the fglrx drivers with the desktop display, it wouldn't recognize them. Since I don't really play games on my laptop, I used the default ati driver. 

The solution to this problem is that you have to use a combination of modifying the xorg.conf file and using xrandr command. All I did was modify the following to my xorg.conf file and then created a script on my desktop so that when I login, it automatically re-sizes the screens. After doing this It works EXACTLY the way it is supposed to where I have multiple resolutions, multiple desktops and I can drag my windows across the screens!
