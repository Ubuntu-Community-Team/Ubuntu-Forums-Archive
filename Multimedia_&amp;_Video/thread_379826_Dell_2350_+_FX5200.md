### Post by catanzag on 2007-03-09
You could follow [this]("http://doc.gwos.org/index.php/Latest_Nvidia_Edgy") guide to install *nvidia* driver instead of the default *nv*.

For dual monitor, I think that with nvidia-setting tool (available after driver installation) you can setup the other monitor, however in the enclosed README you can setup the /etc/X11/xorg.conf manually (though it is not so easy)

As far as I know, there should be the [Xinerama]("http://en.wikipedia.org/wiki/Xinerama") extension in X11 server to use multiple monitor, but I've never used it (if I well remember, it doesn't work with nvidia driver, only with nv, but I'm not really sure)
