### Post by orange2k on 2007-09-14
Dont know about Cubase, but you can install and run Reason with Wine. But it takes some configuring and finetuning. 

Rosegarden is an alternative for Cubase in Linux.

I also recommend Energy XT2 which runs nativly in Linux. But due to lack of support for VST-plugins and instruments, many people run the windows version of that program via Wine to be able to use VST-s.

You might also try Ubuntustudio, and see what it has to offer...
