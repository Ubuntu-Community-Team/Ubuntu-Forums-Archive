### Post by dilb on 2007-06-06
I have a 7600GS and had problems too. I had to install the nVidia proprietary driver, then reboot, then run dpkg-reconfigure xserver-xorg. Then make sure you select 'nvidia' driver, rather than nv.

Hope that's helpful.
