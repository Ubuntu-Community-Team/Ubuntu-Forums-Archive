### Post by RomeReactor on 2007-10-11
Hi. If you have desktop effects on, try disabling them and see if Totem now opens. If disabling desktop effects doesn't work, try launching Totem from the terminal (Applications->Accessories->Terminal):
```
totem
```
You could also do the same for Gxine. If Totem or Gxine crash, please post the output left on the terminal here.
