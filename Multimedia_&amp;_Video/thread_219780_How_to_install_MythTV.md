If you are using dapper, don't intall mythTV package using synaptic. Since mythTV on dapper is version 0.18.1 

Go to mythtv.org and using subversion get a source code of latest bugfix version 0.19.1 and compile it by yourself. mythtv.org has enough document you can reference. 

If there is other problem you can not solve with those documents, probably you can ask further questions.
