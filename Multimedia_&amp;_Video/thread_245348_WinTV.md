### Post by Stormbringer on 2006-08-29
TVtime would be worth a try.

Install it by issuing:
# sudo apt-get install tvtime

This program is only a TV viewer - no Teletext or recording functionality.

Note that TVtime won't work with Hardware MPEG2 Encoder like the WinTV PVR 150/250/350 - but as you said you're having a WinTV PCI Go it'll work for you just fine.

A teletext viewer is somewhere in the repositories too ... just open Synaptic and search for "Teletext" if you need that.
