mmmmk.  So ive had that problem for quite a while.  Any one with the intel chipset for a video card i have your solution and its easy.

go to:
system -> administration -> synaptic package manager.

while there, find the "915 resolution" file.

install it.

go to:
system -> preferences -> screen resolution

change it to 1440 x 900

Itll be messed up, dont worry.  Hit  ctrl+alt+backspace and then log in again.
