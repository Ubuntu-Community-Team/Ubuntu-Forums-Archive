### Post by aysiu on 2008-02-20
> **ToaStOmGi said:**
> How do i do that? When i save the profile there is a palce to do that?
Go to File > Save As > File Type > Microsoft Word 97/2000/XP (.doc)

If you want that to be the default format for saving documents, go to Tools > Options > Load/Save > Document Type > Text Document > Always Save As > Microsoft Word 97/2000/XP
