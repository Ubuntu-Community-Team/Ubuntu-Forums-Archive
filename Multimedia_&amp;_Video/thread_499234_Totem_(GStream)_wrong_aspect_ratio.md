### Post by thespinesplitter on 2007-07-12
cool.
look up online your monitor's model and figure out what is the display size in millimeters then add the following

make this part of the condig file look like this

```
Section "Monitor"
Identifier "Generic Monitor"
Option "DPMS"
DisplaySize x y      # this is where you want to add the displays size in millimeters
EndSection

```
you may have to tweak this a bit
