### Post by Lord Banshee on 2007-04-20
Thanks it works now this is what i had to do:

1) Go into old kernel, xxxx-14
2) Download envy from [http://albertomilone.com/ubuntu/nvidia/scripts/envy_0.9.2-0ubuntu1_all.deb](http://albertomilone.com/ubuntu/nvidia/scripts/envy_0.9.2-0ubuntu1_all.deb)
3) Double click it, let it install
4) reboot start in xxxx-15 kernel let it force me into text mode
5) login
6) sudo envy -t
7) From text menu, clear all instances of nvidia i think it was "6"
8 ) From text menu, uninstall nvidia driver (just to make sure) i think it is "2"
9) From text menu, install nvidia driver, i think it is "1", "y" when it asks to replace xorg.conf
10) reboot
11) Load xxx-15 kernel  WOW it works
12) open terminal
13) sudo nvidia-xconfig --add-argb-glx-visuals
14) ctrl+alt+backspace to restart X
15) Enable Desktop Effect from System->Preferences->Desktop Effects, click the ‘Enable Desktop Effects’ 

Everything is working now thanks everyone for the help and i hope this help somebody else

I got instructions 13-15 from [http://www.ubuntugeek.com/compiz-and-nvidia-on-ubuntu-feisty-fawn.html](http://www.ubuntugeek.com/compiz-and-nvidia-on-ubuntu-feisty-fawn.html)
