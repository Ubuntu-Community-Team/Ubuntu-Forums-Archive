### Post by Marsolin on 2007-03-19
There should be. It sounds like the movie is in DVD format on two discs now? You could try ripping both to a hard drive using a program like [dvd::rip]("http://linuxappfinder.com/package/dvdrip"), keeping them at a high bitrate, and then combine the two files using [Avidemux]("http://linuxappfinder.com/package/avidemux"). It's as simple as File | Open, select the first file, then File | Append, select the second file, and Save.

You could then use [DeVeDe]("http://linuxappfinder.com/package/devede") or a similar program to turn it back into a DVD.

Hopefully I understood what you are trying to do correctly.
