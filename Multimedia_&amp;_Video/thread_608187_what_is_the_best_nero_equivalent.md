### Post by Shay Stephens on 2007-11-09
I was using nero, but when I switched to Ubuntu, I wound up looking for apps that could do data verification.  The best one I have found is K3b.  It is a KDE app, but there has been no negative side effects from using KDE apps in gnome.  So all good :-)

Last I checked gnomebaker did not do data verification.  Brasero has data verification, but is not yet to 1.0 status but it might work for you if you don't want to use K3b.

For burning the image - K3b
