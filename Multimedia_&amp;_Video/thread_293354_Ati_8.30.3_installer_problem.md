### Post by Xemanth on 2006-11-05
fglrx installer package doesn't understand edgy's changes yet.

With this help you can get it working:
Do sudo mv /bin/sh /bin/sh.backup
sudo ln -s /bin/bash /bin/sh

And now install fglrx package

after installing package do:
sudo rm /bin/sh && sudo mv /bin/sh.backup /bin/sh

Should work :)

### Post by NemesisUK on 2006-11-05
Isn't it about time for a sticky post on this forum about installing ATI drivers, as it seems that every day theres a post asking howto do it.

For the majority of users all they need to do is download the drivers from ati's site.

then

sudo apt-get install debhelper build-essential module-assistant linux-headers-$(uname -r)
sudo sh ati-driver-installer*.run --buildpkg Ubuntu/dapper or edgy
sudo dpkg -i *.deb
sudo module-assistant build,install fglrx
sudo aticonfig --initial --resolution=0,1280x1024 or whatever resolutions you want seperated by commas.

then reboot.

This has worked for me on the 6 machines Ive built with ati cards. This wont work with R200 based cards however as the newer drivers dont support them anymore.

Hope this can help anyone out.
