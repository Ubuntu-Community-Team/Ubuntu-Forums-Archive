### Post by dearephesus64 on 2006-11-20
I didn't really feel like installing any more sketchy packages only to play back one or two albums, so I just ran foobar under wine and converted them to .wav files.  That seemed to work great for me, and it made files that I could convert into other formats relatively easily using native Linux programs.  

Foobar isn't too terribly dodgy under wine, but it doesn't work that well as an audio player because of playback issues (it skips and such).  However, I have yet to find one GUI audio player for Linux with mp3 masstagging, mass file renaming, mp3 stream rebuilding, and file format conversion all built in.  It can only convert to .wav in my experience as it has trouble utilizing flac.exe under wine.
