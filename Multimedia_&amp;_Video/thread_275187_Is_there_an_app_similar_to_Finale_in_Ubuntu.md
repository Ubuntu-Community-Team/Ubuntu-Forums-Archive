### Post by mssever on 2006-10-11
There's nothing like Finale. Lilypond does a great job of formatting scores for printing, but it's completely non-interactive. You type a text file using an arcane syntax and it lilypond typesets it.

There are several front ends to lilypond, but I haven't found anything I like--although if you don't carre about playback you'll have an easier time.

### Post by christhemonkey on 2006-10-13
There are plenty of apps like this for linux,

although most of them use qt, the kde toolkit.



Noteedit might be the easiest since it is in the repositories.
```
sudo apt-get install noteedit 
```


There is also:
    * Denemo (denemo) A GTK frontend for Lilypond
    * Lilypond (lilypond)
    * Musescore
