### Post by mozetti on 2008-03-04
First, my advice would be to use the restricted drivers manager to install the nvidia drivers. Uninstall the driver you installed from the nvidia site and try that.

If you'd rather use the driver d/l from nvidia, then make sure you have the linux kernel headers installed. I'm not at my Ubuntu machine, but jus search Synaptic for "linux headers" and it should come up. When you install the nvidia driver it needs the kernel headers to properly compile the driver. Also note, whenever you upgrade your kernel you'll need to make sure you have the new linux kernel headers and re-install the nvidia driver -- you don't need to do this if you use the restricted drivers manager.

Finally, if you still don't have the right resolutions available, open a terminal and run:```
sudo dpkg-reconfigure xserver-xorg
```
That will run you through the xserver configuration tool and you can select the driver and resolutions you want to be made available to you.

I have the 6200-LE (256MB) and it runs great with the driver from the restricted drivers manager.
