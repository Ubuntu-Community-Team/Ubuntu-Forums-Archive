Hi Player,

Don't be afraid to play around. just make sure you make a backup of your xorg.conf before you edit it:

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.working.backup
```

The worst thing that will happen is X Windows won't boot so you just use the text login to go and restore your backup:

```
 sudo cp /etc/X11/xorg.conf.working.backup /etc/X11/xorg.conf
```
