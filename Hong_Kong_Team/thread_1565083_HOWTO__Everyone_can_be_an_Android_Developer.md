### Post by samiux on 2010-08-31
You are not required to be a Java programmer to develop an Android application.  Almost everyone can be an Android Developer by using Google's App Inventor for Android which is a web based development envoirnment.  A Java enabled browser and Android device as well as internet connection are required.

[The tutorial is here]("http://samiux.blogspot.com/2010/09/howto-app-inventor-for-android-on.html").
