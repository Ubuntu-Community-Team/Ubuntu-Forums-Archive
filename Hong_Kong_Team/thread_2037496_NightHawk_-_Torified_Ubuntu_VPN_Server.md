Hi all,

I think you all have heard about [Tor]("http://www.torproject.org") (The Onion Router).  It allows you to surf internet anonymously.  However, setting it up is very painful and the chances of fail is high.  Therefore, Tor Browser solve this problem.  However, Tor Browser just allows you to surf the internet by browser.  If you want to use other programs to access internet anonymously, it is no hope.

[NightHawk]("http://www.debian-inside.com/nighthawk") - Torified Ubuntu VPN Server overcomes the problem of Tor Browser.  It allows you to do anything in the internet via the Tor network.

It is very easy to implement and it is portable.  It works behind a router, such as wired router, wireless router, Android wifi share via Android device or etc.
