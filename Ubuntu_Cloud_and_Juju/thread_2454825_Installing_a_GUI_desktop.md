### Post by ajgreeny on 2020-12-07
Decide on the GUI you want, but make it the lightest you can manage with as servers are usually managed by command line only, then run command ***sudo apt install <package-name>***

The system will then install what is needed and you should be good to go, though you may have to configure the GUI startup process manually, depending on what you choose.
