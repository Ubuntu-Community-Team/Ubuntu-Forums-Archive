### Post by kim0 on 2011-03-10
well, for java files, I would imaging a user can upload a war file for example through a web portal, then the cloud can spawn an Ubuntu server instance, install apache-tomcat in it, deploy the user's war file and get the service started. Yes that sounds possible

For .exe files, you can spawn a windows VM. For games however you'd need accelerated graphics and sound which is very hard to do.

All in all, what you describe AFAIK doesn't exist today, but could be done if you want to. Hope that was helpful
