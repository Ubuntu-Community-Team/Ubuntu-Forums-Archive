### Post by kim0 on 2011-08-17
Hi!

Cloud images are the blueprints, from which cloud servers are created. If you'd like to run the cloud image on your local machine, you can use the KVM hypervisor. This page has info on how to do that
[https://help.ubuntu.com/community/UEC/Images#UEC](https://help.ubuntu.com/community/UEC/Images#UEC) Images on Local Hypervisor

Landscape is a paid systems maangement tools from Canonical that enables you to manage and update ubuntu servers. and I'm not sure I understand your question about performance, you can measure a cloud instance performance in the same way you'd measure a physical ubuntu server!

Hope that was helpful
