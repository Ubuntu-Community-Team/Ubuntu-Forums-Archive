### Post by drs305 on 2009-09-11
> menuentry "Memory test (memtest86+)" {
	linux16	/memtest86+.bin
}
**[COLOR="DarkRed"]#[/COLOR]**menuentry "Memory test (memtest86+, serial console 115200)" {
**[COLOR="DarkRed"]#[/COLOR]**	linux16	/memtest86+.bin console=ttyS0,115200n8
**[COLOR="DarkRed"]}[/COLOR]**
Updated: Yes, the bracket at the bottom is incorrect since the menuentry line is commented. There really isn't an entry to act upon - only a title. This is the problem.

When Grub2 automatically generates an entry, there is an opening  bracket at the end of the title line and a closing bracket on a single line at the end. In this case, there is only the closing bracket since the first is on a commented line.

To disable memtest, I don't think there is a line (yet) to disable the memtest entry in /etc/grub/default  However, you can prevent the memtest entry by removing the executable bit in the /etc/grub.d/20_memtest86+ script.
