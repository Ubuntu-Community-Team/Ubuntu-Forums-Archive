### Post by smani on 2009-02-27
Running update-manager -d only will add the repositories of the current development release to your sources, i.e. it won't proceed automatically to 9.10 once the repositories get online.
For jaunty, on my PC it works fairly well, but I'd avoid directly upgrading your productivity system, but rather test it side by side. Especially if you need proprietary graphics drivers, you might run into some problems.
