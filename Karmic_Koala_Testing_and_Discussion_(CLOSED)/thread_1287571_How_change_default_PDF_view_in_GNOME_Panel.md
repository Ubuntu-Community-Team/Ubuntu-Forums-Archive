### Post by christian.convey on 2009-10-10
I'm running Karmic 64-bit, which doesn't yet have an "acroread" package available.

I've dragged a PDF file from Nautilus to the GNOME Panel.  My goal is to have the PDF come up in some reader (ideally evince) when I click on that file in the GNOME Panel.

The problem is that GNOME Panel, or GNOME, or some other piece of software thinks that when I click on that file in the GNOME Panel, it should launch acroread (which I don't have).

Does anyone know where I can change what application is used in this case, so I can specify evince rather than acroread?

(I know that there are various work-arounds.  I'd rather work it out this way if I can though.)

---

### Post by junior aspirin on 2009-10-10
right click on any pdf > properties > Open with Tab > Select "document viewer"

all pdfs will now open in Evince.
