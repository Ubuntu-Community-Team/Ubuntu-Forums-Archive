### Post by Longinus00 on 2009-09-19
All the backgrounds are in /usr/share/images/xsplash/ 
The throbber and ubuntu logo are transparent pngs so replacing the background would be trivially simple in a script. I assume this will eventually get integrated with the gdm login background or your user background sometime in the future. 

**Make sure you have imagemagick installed or else this won't work!** 
Put this code in shell script file and give it execute permissions. 

```
#!/bin/bash
imgdir=/usr/share/images/xsplash
resolutions=( 1024x768 1280x1024 1440x900 1680x1050 1920x1200 2560x1600 800x600 )
for ((i = 0; i < ${#resolutions[*]}; i++))
do
	convert $1 -resize ${resolutions[$i]}^ \
	-gravity center -extent ${resolutions[$i]} \
	-strip -quality 80% \
	$imgdir/bg_${resolutions[$i]}.jpg
done
```

**Backup your original bg_*.jpg files from the imgdir** and invoke this shell script on an image you want to have as your new background. You will have to run this program as root.
This converter is just a bare bones one and resizes images while keeping the aspect ratio by cropping off excess, just like the "zoom" option in your desktop background. You will get the best results if you use it with a high resolution source image.

### Post by Longinus00 on 2009-09-19
Here is a script to change the color of the throbber and logo.
[B]Once again you must have imagemagick installed!
Also don't forget to backup the original images![/B]

```
#!/bin/bash
imgdir=/usr/share/images/xsplash
color=000000	# hex value of color 
		# http://en.wikipedia.org/wiki/Hex_color
images=( logo_large.png logo_small.png throbber_large.png
throbber_small.png logo_medium.png logo_xtra_large.png 
throbber_medium.png throbber_xtra-large.png )
for ((i = 0; i < ${#images[*]}; i++))
do
	convert $imgdir/${images[$i]} -channel RGB +level-colors \'\#$color\' \
	$imgdir/${images[$i]}
done
```

The color of the throbber/logo will be whatever is given in the line
```
color=000000	# hex value of color 

```
Just like the last script this doesn't do anything too fancy.
If you want to change the value of the throbber or logo independently, you can edit the $images array to only include the files you want modified.
