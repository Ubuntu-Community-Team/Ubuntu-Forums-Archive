I've tried to install Karmic Koala, Alpha 6, and am
experiencing problems which I suspect are associated
with my NVidia Quadro FX 370 LP graphics card.
If I try just to "run" Ubuntu, the screen goes dark.
Also, if I try to "install" Ubuntu, there is a flurry
of console output but when the installion gui should
launch, the screen goes dark.

I've tried to trigger the "basic video mode" using F4.
The mode seems to highlight (something that didn't
happen with Alpha 5), but after that the behavior
is exactly the same as described above.

For what it's worth, in installing Fedora 10, I had a
similar problem but with that distro, toggling the
"basic video mode" during the install did the trick.

Also, for what it's worth, Ubuntu 9.04 installs without
a problem.  After the installation is complete, I get
the "restricted driver" and configure twinview and
all is well.   So the problem is definitely in the
move from 9.04 to the current state of 9.10.

Help/pointers very much appreciated.
