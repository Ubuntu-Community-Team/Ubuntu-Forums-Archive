### Post by Robin Nixon on 2009-09-16
I have a Vaio UMPC which came with Vista installed and was so slow I'd sit there for minute on minute waiting for it to boot, let alone be able to use it.

So I downgraded it (I call it an upgrade) to XP and saw some improvement, although the mouse was all jerky and all the windows were still very delayed in opening.

So I had enough and installed Ubuntu Alpha 5 today.

I must say it's a breath of fresh air. The operating system is lightning fast, the mouse moves as smoothly as a, well a very smooth thing, and setting up a dual display was simplicity itself compared with windows; a couple of clicks and I was there. Networking also worked straight 'out of the box'.

Until now I had been thinking Ubuntu wasn't quite  ready for primetime. But I've been more and more impressed lately and today I changed my mind. I now have it as the main OS on 3 PCs and dual boots on most others (I have a few as I'm a technical writer).

For me, Ubuntu is now the OS to beat.
