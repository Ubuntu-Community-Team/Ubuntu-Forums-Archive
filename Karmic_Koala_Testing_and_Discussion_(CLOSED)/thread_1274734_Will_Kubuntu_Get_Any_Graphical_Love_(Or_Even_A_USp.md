> fix the themes' background images' aspect ratio by using the new "crop"
scalemode.

From [http://websvn.kde.org/trunk/KDE/kdebase/workspace/kdm/kfrontend/themes/oxygen-air/oxygen-air.xml?r1=1005267&r2=1009280](http://websvn.kde.org/trunk/KDE/kdebase/workspace/kdm/kfrontend/themes/oxygen-air/oxygen-air.xml?r1=1005267&r2=1009280);
```
...
	<item type="pixmap" id="background" background="true">
		<normal file="1280x800.jpg" scalemode="crop"/>
		<pos width="100%" height="100%"/>
	</item>
...
```

It was apparently backported to 4.3.2 ([http://websvn.kde.org/tags/KDE/4.3.2/kdebase/workspace/kdm/kfrontend/themes/oxygen-air/oxygen-air.xml?r1=1005268&r2=1022821](http://websvn.kde.org/tags/KDE/4.3.2/kdebase/workspace/kdm/kfrontend/themes/oxygen-air/oxygen-air.xml?r1=1005268&r2=1022821)).
