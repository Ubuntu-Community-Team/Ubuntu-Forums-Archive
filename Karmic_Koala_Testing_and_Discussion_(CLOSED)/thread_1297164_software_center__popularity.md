Software popularity is not available in the Software Center. It might come in the future but that is not for sure.

There has never been any software popularity sorting available in the Software Center as I far I know. I think you are talking about "Gnome App Install" which has been replaced by the Software Center. Gnome App Install has software popularity sorting.

Gnome App Install is still available in the repo. Install it with:
```
sudo apt-get install gnome-app-install 
```
