It will tell you on boot up if CD is first attempt to boot. By just saying CD during

Check to see if you saved boot at CD in BIOS and leave it there. You never know when you

 If it does not say CD during boot screen at one point, it is not attempting to.

It is one of those two problems or it would boot.
