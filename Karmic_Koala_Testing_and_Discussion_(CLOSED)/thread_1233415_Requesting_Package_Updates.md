### Post by chrisccoulson on 2009-08-06
The maintainer mailing list is not the place to request new versions (in any case, the maintainer for most packages is ubuntu-devel-discuss, which gets enough traffic as it is).

If you want to request a new version, open a bug report against the package and tag it with "upgrade"
