### Post by sedawk on 2009-10-05
Try 
Ctrl-Alt-Backspace.

This "only" restarts the desktop and might be a workaround.

You can also try
Ctrl-Alt-F1 (or Ctrl-Alt-F2, or ...)
to get the text console.

If you shortly press the power button, there is
a shutdown dialog and a countdown starting at 60 seconds.
If you wait long enough the machine might reboot, so
the button worked, even while the screen was blank.

Does any of this work?
