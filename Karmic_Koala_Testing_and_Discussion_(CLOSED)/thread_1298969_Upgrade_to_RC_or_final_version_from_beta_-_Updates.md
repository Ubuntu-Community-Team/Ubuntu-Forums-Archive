Yes; continued updating will update you to the RC and then the Final Release.  Beware the "Partial Upgrade" prompt as it "could" screw up your system.

There are situations where a clean install is better.  If you upgraded from Jaunty (Ubuntu 9.04) then you probably don't have EXT4 and Grub2.  A clean install would be in order for that situation.  There are also rare instances where upgrading will leave old stuff around from earlier alpha/beta releases, but shouldn't be a problem.

IMHO you should just keep upgrading if you installed a fresh copy of the Karmic Beta.  I'd also keep the dual-boot Windows there for awhile if you need it.  Maybe shrink the size of the Windows partition?  I'm thinking of the same thing with Virtualbox and a slim install of Windows XP, but I found that wasn't the most ideal solution.  I go back and forth.
