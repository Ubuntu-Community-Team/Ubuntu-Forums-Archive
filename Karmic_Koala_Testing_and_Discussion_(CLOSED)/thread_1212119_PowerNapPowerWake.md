### Post by ghindo on 2009-07-13
Not sure how many of you are using Karmic for the server release, but Dustin Kirkland [just announced on his blog]("http://blog.dustinkirkland.com/2009/07/introducing-powernap.html") some interesting new programs:> PowerNap operates sort of like a screen saver for servers. Besides watching the console and terminals for keyboard activity, it also watches the system's process table for activity.> Which brings me to PowerNap's kid brother...PowerWake. /usr/bin/powerwake is another Python script. This script is designed to be a smarter, remote waking utility. Currently, it supports wake-on-lan, but it will eventually support other mechanisms, such as IPMI, and perhaps NUT.
