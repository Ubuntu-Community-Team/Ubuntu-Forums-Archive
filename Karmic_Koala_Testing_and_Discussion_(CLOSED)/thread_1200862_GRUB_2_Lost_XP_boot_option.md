### Post by Bucky Ball on 2009-06-30
Try this:

```

menuentry "Microsoft Windows XP Home Edition (on /dev/sda2)" {
    set root=**(hd0,2)**
    chainloader +1
```Like this:

```

 menuentry "Microsoft Windows XP Home Edition (on /dev/sda2)" {
     set root=**(hd0,1)**
     chainloader +1
```and change the other entry for XP. hd0,2 is wrong. sda2 = hd0,1. hd starts at zero, sda starts at one, if that makes any sense. :)

sda=hd0; sda1=hd0,0; sda2=hd0,1; sdb1=hd1,0 etc ...
