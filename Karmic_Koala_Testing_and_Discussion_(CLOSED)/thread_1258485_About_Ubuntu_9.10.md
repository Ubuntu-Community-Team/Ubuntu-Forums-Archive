### Post by QIII on 2009-09-05
I would advise against it.  As an Alpha, it has not even reached the Beta stage.  Unless you are planning to actually test it rather than using it for daily use, don't upgrade.  It is not for use as a production OS yet.

When it actually comes out, be advised that there are some significant changes to GRUB (it uses GRUB2), which can leave you in a "legacy" mode with regard to GRUB (for instance, GRUB2 does not use menu.lst) if you update rather than doing a clean install.

Also, if you are dual booting, there are some command line work-arounds you will need to use to get access to your other OSs from your GRUB menu.

There are other issues as well.

While I have usually used the upgrade path in the past without problems (I did a clean install of Jaunty because I bought a new machine for it), I've been doing clean installs at each Alpha of Karmic.  The GRUB2 issue still requires the work-around as of my last download of Alpha 5, but that will be cleaned up before the final release.

If you have to ask, then the answer is NO.

If you want to test 9.10 to make sure stuff works on your computer, then download and burn livecd/liveusb and see what works and doesn't work. Then see if others have same problems, or file bugs and create threads yourself explaining problems that should be fixed.

I would not recommend you upgrading to 9.10 until final (format/fresh install would work better as you get grub2 and ext4 by default). Unless you currently have a serious issue in 9.04 that you know for a fact (because you tested livecd/liveusb) is fixed in 9.10.

9.10 is not ready to be used by people wanting it to be safe. Stuff can easily break.
