### Post by tsger on 2009-09-30

[B][COLOR="Red"]
EDIT: SEE POST #3 BELOW[/COLOR][/B]

I think the easiest way to do it is to open a terminal and type:
```

ubuntu-bug [name of package here]
```

This should collect some info, then take you to the launchpad site, where you can fill in more info.

Also, you will probably have to register as a user on the launchpad site before you do this.

Hope this will get you started!
