### Post by FatalChaos on 2009-10-26
I recently installed office 2007 enterprise on Karmic via Wine. I used the latest beta version of wine. Instructions to install the latest versions of wine can be found here: [http://www.winehq.org/download/deb](http://www.winehq.org/download/deb). I installed it without any tricks or anything just to test it out, and overall it's okay. One note didn't work for me, but excel and word both did. Office seems to work better if I choose to open files by going to the menu and clicking open as opposed to double clicking the file or w/e. 

I will try and reinstall Office 2007 with wine tricks installed to see if that fixes One Note. 

PS: If you need better support for windows apps, you might want to try virtualization. Virtualbox is pretty easy to use, not sure about the other virtualization programs.
