### Post by Paqman on 2009-10-15
Karmic has mostly the same Nvidia stuff in it as Jaunty. The only difference is that Jaunty lacks the nvidia-glx-185 driver and Karmic lacks the nvidia-glx-71. So unless you're using the 71 driver you'll be fine.
