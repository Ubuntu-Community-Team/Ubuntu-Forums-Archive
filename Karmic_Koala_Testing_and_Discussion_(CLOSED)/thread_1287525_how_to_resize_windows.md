### Post by NightwishFan on 2009-10-10
You can unmaximize windows by right clicking on their taskbar entries.

To stop the fullscreen Windows entirely, you can disable or uninstall Maximus. To disable go to System -> Preferences -> Startup Applications and uncheck Maximus.

You can drag drop between Windows by dragging the content to the Window you want on the taskbar, and waiting a second. Then the Window will get focus and you can drop to it.

Please ask if you have any questions.
