### Post by Lightbreeze on 2009-08-29
Please submit lots of bugs ([https://launchpad.net/humanity/+filebug](https://launchpad.net/humanity/+filebug)) but first make sure you have the latest icons:

To download and switch to the Humanity icons run this in terminal ```
bzr branch lp:humanity ~/.icons/humanity && gconftool-2 -t str -s /desktop/gnome/interface/icon_theme humanity
```

After that, when you want to get the newest icons, run```
cd ~/.icons/humanity; bzr pull
```
