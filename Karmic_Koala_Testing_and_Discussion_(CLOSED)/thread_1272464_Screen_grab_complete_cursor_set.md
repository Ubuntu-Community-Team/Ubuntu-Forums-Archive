I need to grab the full set of default cursor images to accompany something I'm writing, including the drag and drop and drag and copy mouse cursors etc. I tried using screen grab while they are displayed but I only get the default mouse cursor.

---

### Post by Tibuda on 2009-09-22
Open the /usr/share/icons/<themename> folder in your file manager and enable thumbnails.
