### Post by tagbre on 2009-10-23
I noticed there's no more context menu option to open links in new tabs in Epiphany...I'll have to switch to another browser if I don't have that option...

Anybody knows why this feature is no longer available?

---

### Post by qamelian on 2009-10-23
The default version of Epiphany is now the one based on webkit instead of gecko. For some reason the webkit version of Epiphany has never had this feature, although other webkit-based browsers do. If you revert to the gecko-based version of Epiphany you will regain the ability to right-click and open in a new tab.
