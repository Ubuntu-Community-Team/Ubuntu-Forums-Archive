### Post by kansasnoob on 2009-10-03
Try installing gnome-alsamixer. It's in Synaptic or you can just run the command:

```
sudo apt-get install gnome-alsamixer
```

It'll then show up in Sound & Video:

[ATTACH]130663[/ATTACH]

Lots of toggles. Notice the via toggles to the far right?
