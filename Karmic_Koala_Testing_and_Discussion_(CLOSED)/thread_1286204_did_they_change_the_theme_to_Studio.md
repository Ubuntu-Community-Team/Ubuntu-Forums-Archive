### Post by psyke83 on 2009-10-08
It's not the UbuntuStudio theme on the login screen, it's called HumanLogin. Go to System -> Preferences -> Appearances. Click the Customize button and on the Controls tab, you can choose HumanLogin. 

I think that HumanLogin uses the Industrial GTK engine (it's the only one which supports button glow), whereas UbuntuStudio uses the Murrine engine.
