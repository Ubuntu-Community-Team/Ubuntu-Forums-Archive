### Post by Copernicus1234 on 2009-08-19
I like karmic so far. Will definitely start using it from one of the beta versions onward and make a fresh install.

**Stuff I liked:**

- Mouse back/forward buttons work both in Nautilus and Firefox. Good!

- System is much faster. Not sure if this is because its the default theme or because compiz is not enabled yet, but it feels much faster and it even starts programs faster. I started Open Office 3 and it took just 2-3 seconds! On my 9.04 machine, it takes 13-15 seconds just for the Open Office logo to show up, and then another 5 seconds for the program to load. So that was impressive. Maybe its preloading some stuff?

- If you right click your name in the top right corner, you have System Preferences which starts a Control Panel-like program with all the settings. I personally dont like this, but I have no problem with it. Some people will like it for sure.

- Installation procedure is smooth and good looking!


**Stuff I didnt like:**

- Hbox. The login screen. Looks like crap and is also confusing considering there is the user name and then where there is supposed to be the OK button, there is a Restart button. Not sure what they were thinking here...

- Ugly default theme for Gnome and ugly default theme for Firefox. Will NOT give a great first impression. People used to Windows 7 will wonder what the heck is wrong.

- When you find the Ubuntu One shortcut in Nautilus and use it, there is a Connect button. Unfortunately it does nothing.
