### Post by snowpine on 2009-09-30
Hi Spotdog, I would recommend applying all available updates using the Update Manager (or the corresponding terminal commands if you know them). Karmic Koala is the testing version (it won't be released until the end of October), so it is very unstable and there are new updates and bug fixes being pushed through every hour. It is quite likely your particular bug has already been fixed since Alpha 6 was released.

I would recommend sticking with the current version (Jaunty) for now, and not using an Alpha release unless you enjoy testing unstable software and reporting bugs to the development team. I've used Jaunty on my Mini 9 in the past, with great results.
