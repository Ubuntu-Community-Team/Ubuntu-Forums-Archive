### Post by Sand &amp; Mercury on 2009-10-06
I've noticed a lot of attention has been given, not unfairly, to the overarching, large-scale goals of Ubuntu's current development cycle such as boot experience and artwork. But ITT, we celebrate the teensy weensy little achievements made with Karmic's development so far that have made Ubuntu a more pleasant experience overall. Here's a little list of mine:

[LIST]
[*]Adding repositories from Launchpad is now heaps easier than before; it can now be done with a single line instead of one for binaries, one for dev files and copying PGP keys.
[*]Rhythmbox no longer has a ½-second delay when changing tracks.
[*]There's now an option to temporarily turn off the trackpad while typing, eliminating losing keyboard focus when I'm trying to write or accidentally clicking on other stuff.
[*]You can now scroll using two fingers with the trackpad.
[*]Compiz no longer draws panel shadows on top of window decoration; you can set the shadow radius to anything you like and it will never obscure the title of a maximised window.
[*]The wireless menu is a great degree more sensibly set out (er... if you can get wireless working. 9.10 for me, is the first instance of an Ubuntu distro where wireless works perfectly, out of the box, with no BASHing required.)
[*]Totem can remember where you were up to when picking up a movie you were watching previously.
[*]Volume control is now part of the system tray, and controlling volume now makes sense for normal people as well as developers.
[*]The indicator applet is actually quite cool and useful once you're used to it.
[/LIST]

I assume a lot of this is the result of the papercuts project -- so props to the Ubuntu and GNOME teams and all the other contributors.
