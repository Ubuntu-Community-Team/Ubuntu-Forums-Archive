### Post by Mike_IronFist on 2009-08-13
> **kestal said:**
> Forgive me for the question, I never used the DVD. (Bandwidth cap issues)

Is there anything thats added onto the ubuntu DVD's compared to the CD's other than extra language support? The dvd iso is 4.1gb so that seems quite a bit for just added languages.

Does it include any extra themes/programs/etc installed by default? or just more things to install from the dvd compared to cd via add/remove?

It includes almost all of the software in the Ubuntu repositories, meaning you can install the programs off of the DVD rather than off the internet.

This is recommended if you have a computer that does not have an internet connection, etc. and want to install software on it, or if you want to install Ubuntu for someone with a slower internet connection; providing them with the disc gives them a faster way to install the programs.

The downside is that you'll probably have to update a bunch of stuff you install from the DVD because the online programs are getting updated all the time, but this is a minor issue for most people.
