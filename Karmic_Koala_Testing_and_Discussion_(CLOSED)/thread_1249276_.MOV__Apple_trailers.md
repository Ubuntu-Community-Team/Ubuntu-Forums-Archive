### Post by eyelessfade on 2009-08-25
then you are wgetting the wrong url. you have to insert a "h" in the url.
```
http://movies.apple.com/movies/fox/avatar/avatar2009aug0820a-tsr_1080p.mov
```
becomes
```
http://movies.apple.com/movies/fox/avatar/avatar2009aug0820a-tsr_h1080p.mov
```

the useragent can be whatever as long as QuickTime/ is first. so ```
wget -U "QuickTime/Apple sux Quick Time" http://movies.apple.com/movies/fox/avatar/avatar2009aug0820a-tsr_h1080p.mov
```
will give funny stats for apple, and you wil get your trailer.
