### Post by adamhalliwell on 2009-10-02
You can edit the GDM Login by launching Appearance Preferences as gdm:

gksudo -u gdm dbus-launch gnome-appearance-properties

Try setting the controls and window border as HumanLogin. It looks pretty good. I have my background set the same as my xsplash and desktop background, and it's nice.
