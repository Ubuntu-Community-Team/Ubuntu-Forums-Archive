### Post by bcat on 2009-06-12
Hmm, that is strange. Looking at the make.log you posted, I think this is your problem:
```
make[1]: dpkg-architecture: Command not found
```
It looks like you don't have the dpkg-dev package installed. You could just install it manually, but I recommend doing a "sudo apt-get install build-essential" instead. That will bring in dpkg-dev, and it should take care of any other compile-time dependencies as well.

Post your results back here. If DKMS still fails after installing build-essential, post any new log files. If it succeeds, you've just found a packaging bug. :)

Hope this helps,
Jonathan
