### Post by murderslastcrow on 2009-08-18
Does anyone know the current state of the project statix? You know, the one that anonymously polls your PC as an active Linux user so that we can have a more accurate number for Linux users?

As far as I know, nothing like this integrated into any mainstream Linux distro.

This would be a really good idea to include with Ubuntu, in Karmic, for several reasons.

1. Increases visibility of Ubuntu as a distribution, since it would be the only one with actually numbers to present.
2. Gives us information to leverage hardware manufacturers and software developers.
3. Ego-boost.
4. It is key to the growth of Linux to understand how many people are using it. Even if everyone in the world bought a Windows PC and switched to Linux already, we wouldn't know.

Proposed ways to do this, assuming a package like statix were available.

1. Install by default (since it is anonymous and non-intrusive - this would also yield the best results of 9.10 users).
2. Provide an option during installation or first boot. This is more annoying, but gives them more choice.
3. Include a link to install the program from the Ubuntu start page in Firefox. This would provide people using previous versions of Ubuntu an opportunity to participate, as well as being completely non-intrusive.

So, any ideas? I think this would be crucial, especially with Windows 7 and Mac OS X Snow Leopard being released around the same time as Karmic. We can do a real comparison!

So, good idea? Bad idea? Give me your thoughts, since I'm considering presenting this at brainstorm and the such if it gets a thumbs up here.
