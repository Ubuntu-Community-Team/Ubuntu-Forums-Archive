### Post by JordyD on 2009-07-09
System > Preferences > Appearance

Make sure the 'Themes' tab is selected. Download a chrome-like theme. ([Here's one]("http://gnome-look.org/content/show.php/chrome-like?content=88988")) Drag the uncompressed tarball onto the Appearance window (or click 'Install themes' and then navigate to your theme). Wait. When done, you can select the theme you want from the Theme tab in the Appearance application.

Simple enough, yes?

Tell me if you have any problems.
