### Post by PGHammer on 2009-07-28
I'm currently installing the alpha 3 version of Kubuntu Karmic Koala (AKA "Klansman"; you figure it out!), and got a rather *welcome* surprise upon firing up the LiveCD; it actually supports the Sound Blaster X-Fi *natively*.  (As it is, I now only will need to add the ATI drivers to the mix post-install, since it snagged the X-Fi already.)  This is the first (and so far, only) operating system to support the X-Fi from the jump (even Windows requires me to dig out a driver CD or grab the latest drivers from the Web).

Also, there's an *alternative* to the usual *Firefox or Konqueror* browser choice with the 'buntus (and it's not Google Chrome, though the browser *is* hosted on Google Code); it's called Arora, and I'm going to use this build to give it a workout as an alternative to the Fox with KDE.  (I hate Opera, and I find Konqueror's bugs just as detestable, which is why I usually find myself using Firefox, though I would love an alternative.)
