I have been using Gnome for about a year, but two days ago I decided to try KDE. So I installed Kubuntu Karmic Koala Beta, but it didn't work well, so I did a new install using the Ubuntu alternate CD. I started with command-line only system and added kde-minimal and the kdeplasma-addons meta-packages. Then I have installed the applications I need, which are mostly kde, but I have a few gnome applications too.

When I report a bug using ubuntu-bug, my report says I'm using Ubuntu 9.10. Additionally, the kdm login says welcome to Ubuntu. So my question is, what do I have, Ubuntu or Kubuntu? ### Post by 23meg on 2009-10-16
[QUOTE=lovinglinux]So my question is, what do I have, Ubuntu or Kubuntu? [/QUOTE]

You have Ubuntu, with some KDE packages. That's still Ubuntu, and still supported.

[QUOTE=lovinglinux]Having a KDE environment makes it automatically a Kubuntu system? [/QUOTE]

No; either installing Kubuntu from one of its official installation media, or installing the kubuntu-desktop package, along with everything else it pulls in, does. 

As long as you explicitly state which desktop environment you're using, and what packages you've installed on top of ubuntu-minimal when reporting bugs, you should be fine.
