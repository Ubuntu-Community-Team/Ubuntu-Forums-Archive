### Post by 3rdalbum on 2009-10-05
How did you install Flash Player to begin with?

If you installed it from the repositories, then you need to remove the package:

```
sudo apt-get remove --purge flashplugin-installer
```

**However**, software only needs to be removed and reinstalled if it gets corrupted, and as "libflashplayer.so" is usually owned by root and not writable by the user account, it's extremely rare that it gets corrupted. Either you've made some other sorts of modifications to the system that have caused Flash Player to stop working (i.e. installed gnash or swfdec) or you need to remove the preferences files for Flash Player (which is inside ~/.adobe/Flash_Player, I think).

If you dragged the Flash Player plugin to /usr/lib/mozilla/plugins or ~/.mozilla/plugins, then you need to remove it from that location and then drag it back, but this probably won't solve any problems anyway for the reasons I mentioned above.
