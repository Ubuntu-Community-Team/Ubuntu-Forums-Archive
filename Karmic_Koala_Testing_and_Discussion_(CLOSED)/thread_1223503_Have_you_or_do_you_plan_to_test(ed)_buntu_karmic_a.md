### Post by MaxIBoy on 2009-07-31
I can't use the new kernel because FGLRX still has issues with anything more recent then 2.6.28.x (you can get it to run but 3D acceleration is broken. As always with unstable distros, I've found it's better to turn auto-update off and manually select updates in synaptic. That way, you can always hit "cancel" if you see that an update clashes with another package, instead of having that other package get automatically removed. That is obviously inherent when packages are being updated as fast as possible the way they are right now, it won't be an issue in the final release.
