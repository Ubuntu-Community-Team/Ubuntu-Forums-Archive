### Post by wayne_cat on 2009-08-17
> **frustphil said:**
> Will we be able to choose between gnome-shell or default gnome on boot?

Karmic's default desktop will be a standard Gnome but you can switch to Gnome Shell
if you want (add a start object "gnome-shell --replace" to your session settings).
