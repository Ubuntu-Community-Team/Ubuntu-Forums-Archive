### Post by kvarley on 2009-06-29
I upgraded to Karmic Koala via the update-manager.

Somewhere along the way screen-profiles didn't install properly.

This will cause crashes of xserver.

[COLOR="Red"]**.::.TO SOLVE::.**[/COLOR]

-Open synaptic. "System - Administration - Synaptic Package Manager"

-Quick search for "byobu", four packages will be displayed: byobu; byobu-extras; screen-profiles; screen-profiles-extras.

-Make sure byobu and byobu-extras are installed, if not then mark them for installation.

-Mark screen-profiles and screen-profiles-extras (If installed.) for removal.

-Apply the changes and your problems should be solved!
