### Post by mperry on 2009-10-16
Only different thing for me is installing from the medibuntu repositories and the fact that all the earlier howto's focused on changing sound card drivers in skype. It appears that this is no longer needed and on my thinkpad t60 skype only offers the pulseaudio drivers these days. Check out the mic gain or input value when you go into properties for sound. I found that the mic value was set very low by default and that callers could not hear me yet they were crystal clear.

Seems like it used to be skype required some bit of manual intervention to get working. This is... easier, better, faster. Thanks!
