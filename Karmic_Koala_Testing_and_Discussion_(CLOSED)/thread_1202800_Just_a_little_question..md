### Post by lindsay7 on 2009-07-02
Generally doing an update in the Ubuntu version will not affect the dual boot process. I think at some point grub2 will be incorporated in the vew versions so you would think that would still be the case. But it is a beta.

I have had and I know it has been an issue with others, trouble with Nividia cards and the new beta kernels.  So, if you have one of these cards, be careful, make sure you have a back up of your system.
