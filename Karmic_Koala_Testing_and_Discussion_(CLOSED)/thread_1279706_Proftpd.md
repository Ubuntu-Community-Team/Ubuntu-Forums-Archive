### Post by dinxter on 2009-10-01
the proftpd package is a transitional meta package, and it is outdated, so if you try and install that it will fail. the newly named package to install is proftpd-basic and then any additional proftpd-mod-* files you want to add functionality. proftpd-basic installs fine on up-to-date karmic
