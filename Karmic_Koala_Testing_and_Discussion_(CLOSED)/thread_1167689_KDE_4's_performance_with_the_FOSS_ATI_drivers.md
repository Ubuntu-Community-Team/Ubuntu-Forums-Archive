### Post by krazyd on 2009-05-23
@Mazza
You're right, EXA is enabled by default. A couple of options to try would be adding to the device section:
```
Option "MigrationHeuristic" "greedy"
```
... which may or may not help to improve performance.
OR, try reverting to XAA acceleration, also in the device section:
```
Option "AccelMethod" "XAA"
```

I don't think those options work together, however.

Report back how that goes! :D
