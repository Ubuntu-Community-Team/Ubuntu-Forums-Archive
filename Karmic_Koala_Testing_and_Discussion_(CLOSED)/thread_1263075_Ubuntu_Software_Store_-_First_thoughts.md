### Post by celticbhoy on 2009-09-10
I just wondered what everyones initial thoughts were on the new Software Store? Having had a quick tinker with it I think that it will serve newbies to Ubuntu & Linux well. The layout is pretty straight forward and easy to use and understand, with the Installed tab making it a doddle for someone without experiance to remove all those little apps that we all download and try, then forget about, and just leave taking up disk space.

That all said I don't see this a replacement for Synaptic for those more advanced. There are times that you want to install a library file to get an app from the web going, and I dont see how Software Store would be able to fulfill that role as well.
