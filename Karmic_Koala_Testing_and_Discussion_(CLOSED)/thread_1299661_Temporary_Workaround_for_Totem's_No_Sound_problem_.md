### Post by FirstByté on 2009-10-24
For those experiencing the no sound in Totem's Movie Player on Ubuntu 9.10 here is a simple workaround I did

[https://bugs.launchpad.net/ubuntu/+source/alsa-driver/+bug/456699/comments/5](https://bugs.launchpad.net/ubuntu/+source/alsa-driver/+bug/456699/comments/5)

This is my temporary solution and hope it would be fixed soon before release or shortly afterwards.

=================
Whenever I finish a new media file (song/video/flv etc.) I'll have to **QUIT** Totem Movie Player, **PLAY another media file** (like inside Rhythm Box), **PAUSE or STOP the media **inside the RhythmBox then **relaunch Totem Movie Player** with the next media or launch a new file inside Totem.

Hope this helps.
