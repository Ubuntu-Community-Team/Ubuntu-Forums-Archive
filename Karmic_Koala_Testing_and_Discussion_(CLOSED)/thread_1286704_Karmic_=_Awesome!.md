### Post by ctothej on 2009-10-09
I recently installed the Karmic Beta on my Dell D630 laptop and all I have to say is it is AWESOME. I am a man with particular needs. I need WIFI working, I need VPN working, I need VPN over WIFI working, I need the Card Reader working, I need 64-bit, I need full disk encryption, I want compiz, I want smart power monitoring, I want a fast boot up.

I GOT IT ALL!

Things that needed some configuring were:

_Cisco VPN:_
I had to install 'vpnc', but it worked first shot no problem from the command line.
Network manager's version did not though ('network-manager-vpnc').

_Smart Cart Reader:_
Smart card was detected straight away. No mucking with config files. I had to install and configure coolkey for the Firefox/Thunderbird plugins. Worked first time. There is no real getting around this setup.

_Full disk encryption:_
I went straight to the Alternate CD install and had special configuration needs, so there was a small learning curve to setup LVM and Crypto  correctly. ***But I did not have to leave the installer!***

Misc stuff that's awesome:

Dark color scheme "New Wave"... awesome
Loading screen... awesome
Gnome-do... awesome
Truecrypt... awesome
Terminator... awesome
Conky... awesome (no flickering!)
Notifications... just about awesome (some weird mouse over issues)
Open Office... awesome
Brasero... awesome
Grub detecting all OSes ('grub2-update')... awesome

I had Fedora 10 installed before this. I tried to upgrade to Fedora 11. I used the Fedora 11 dvd, which asked me if I would like to upgrade an existing install. I tried, **everything broke**. I figured since I had to do a full reinstall, why not go with the beta of Karmic... so thankful I did.

Thanks Ubuntu Team.

:guitar:

****Update 10/20/2009****
I have been using Hibernate and Suspend with great success as well. The best part is that it correctly locks and unlocks the encrypted partitions on resuming the computer from hibernation... great stuff.
