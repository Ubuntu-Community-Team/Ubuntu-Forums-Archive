That was changed a while ago, so if you have current Firefox 3.5.x packages from the Karmic repositories it should have been changed to 'Firefox Web Browser'.

The current packages for Firefox 3.0.x should create a menu entry the reads 'Firefox Web Browser 3.0'.

I didn't actually try to change it, but if you right click the applications menu and choose 'Edit Menus', it comes up like it can be changed. If nothing else you should be able to create a new entry and hide the old one.

I'm not familiar with Ubuntuzilla, so this is all assuming the stock default gnome panel and menu applet are being used.
