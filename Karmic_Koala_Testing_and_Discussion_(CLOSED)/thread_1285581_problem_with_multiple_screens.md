### Post by Longinus00 on 2009-10-08
That's because if you could do this then there's the chance you could lose a small window (like a dialog) in the gaps caused by the different sized monitors. If this really bothers you then make sure the screens are set up so that their tops touch, not their bottoms, in display preferences.

### Post by Longinus00 on 2009-10-08
I'm not sure exactly how Nvidia does it but my guess is that they just create one huge desktop in the driver which it then presents to X/gnome. I'm guessing nautilus just assigns icons to the default location (top left) and doesn't really know it's not visible. If you setup the monitors like I told you (so that their tops align) it should solve your problem with the icons and windows.

Dual monitor setups on the closed source drivers has never been very elegant.
