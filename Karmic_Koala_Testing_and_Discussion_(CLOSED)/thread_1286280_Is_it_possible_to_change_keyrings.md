### Post by FuturePilot on 2009-10-08
Do you mean the password for your wireless or the actual keyring's password? If you're talking about the password to unlock the keyring then all you need to do is Applications &#8594; Accessories &#8594; Passwords and Encryption, right click on the Passwords: login folder and select Change Password. Make sure it matches your user account password or else it will keep asking you for it when you login.
