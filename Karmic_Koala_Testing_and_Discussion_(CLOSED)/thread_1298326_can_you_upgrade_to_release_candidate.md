### Post by flipper9 on 2009-10-22
Are you running Ubuntu 9.04 Jaunty, or are you running Ubuntu 9.10 Karmic Alpha/Beta?  If you are running the alpha/beta of Karmic, then you already have the RC updates if you are running update manager.  If you are running Jaunty, you need to do a distribution upgrade as specified at the beta/rc page at the ubuntu.com homepage...

**Upgrading from Ubuntu 9.04**
To upgrade from Ubuntu 9.04 on a desktop system, press Alt+F2 and type in "update-manager -d" (without the quotes) into the command box. Update Manager should open up and tell you: New distribution release '9.10' is available. Click Upgrade and follow the on-screen instructions.
