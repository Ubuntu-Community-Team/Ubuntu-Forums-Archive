### Post by VMC on 2009-10-20
gfxpayload needs to be put in "10_linux" file. LP suggested it somewhere after the linux function, like so:

```
linux_entry ()
{
  cat << EOF
menuentry "$1" {
EOF
  if [ "x$3" = "xquiet" ]; then
    cat << EOF
	set quiet=1
EOF
  fi
  save_default_entry | sed -e "s/^/\t/"
  prepare_grub_to_access_device ${GRUB_DEVICE_BOOT} | sed -e "s/^/\t/"
  cat << EOF
[COLOR="Red"]# My add as vga=773 is depreciated
gfxpayload=1024x768[/COLOR]
	linux	${rel_dirname}/${basename} root=${linux_root_device_thisversion} ro $2
EOF
...

```

This worked for me but not exactly as vga= did in grub-legacy. So I removed it.
