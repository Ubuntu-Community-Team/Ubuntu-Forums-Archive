### Post by Jordan Uggla on 2009-10-23
If you know that failures are going to be recorded and you want to always have a timeout anyway you can add a file /etc/grub.d/01_keep_timeout and copy and paste this as the only contents:
```
#!/bin/sh
echo "set timeout=${GRUB_TIMEOUT}"
```

Then make it executable. This only applies to people who know what it means to have a failure recorded and want to ignore it anyway:
**Most users should not do this.**
