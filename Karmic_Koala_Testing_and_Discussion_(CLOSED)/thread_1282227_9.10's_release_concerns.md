### Post by desmane on 2009-10-04
Hi, 

you can convert but I wouldn't do it. Old files will use the new file system when they are rewritten or something. The best way to take advantage of the ext4 is to do a clean install. 

If you have home at a separate partition already, simply boot into karmics live cd and follow the instructions from here: 

[https://help.ubuntu.com/community/Installation/UpgradeFromOldVersion](https://help.ubuntu.com/community/Installation/UpgradeFromOldVersion)

(the Clean Install Upgrade (not recommended) section)

if not, maybe its a good time to create one. 

[http://embraceubuntu.com/2006/01/29/move-home-to-its-own-partition/](http://embraceubuntu.com/2006/01/29/move-home-to-its-own-partition/)

but be careful with every single command, check for typos and before you start, I suggest to read the whole thing first :) 

if you use partimage for backing up your home folder first, consider that it doesnt work with ext4 (but is fast and well, if something happens you can go back to ext3 to restore your precious files), or use dd instead!
