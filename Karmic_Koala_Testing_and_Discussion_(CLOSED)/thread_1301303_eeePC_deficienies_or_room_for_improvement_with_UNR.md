### Post by rbrueske on 2009-10-25
I have a 1000HE and am missing some features I got used to in XP. Maybe there are some workarounds, but I haven't found anything.

1.  Additional gestures are not recognized with the elantech touch pad. I grew to use the three-finger swipe (which isn't as naughty as it sounds) to go back or forward in an app. Zoom is useful but missing too.

2.  Function key combos are missing. Things such as switching between monitors,  slowing down the cpu to conserve battery power.

3. The 'special' keys atop the keyboard don't do what I want, such as changing the resolution. I like to go from 1024x600 to 1024x768 and back depending on what I'm doing. The additional horizontal lines are virtual, where they are below the screen and moving the cursor down shows them.I tried setting the keys to a key shortcut, but pressing the keys doesn't register them in the shortcut box.

4. I can't find a way to put two apps side by side or one on top of the other.

5. Key navigatioon is lacking. For example, you can't select a 'favorite' by using key commands. And if you are on one of the other tabs (such as files & folders and accessories) with an item selected, if you then go back to the list of tabs the tab you are 'on' isn't the active one, instead there is no active tab, you have to hit the down arrow to select "Favorites". Clearly the use of keys instead of a cursor doesn't work as well. I had to use the keys because I managed to screw up the touch pad.

6. I can't seem to drag an icon to the top of the screen for quick access. For example, if I want to drag the terminal alias to the top, it doesn't budge.
