80 GB disk:

**sda1 40 Gb** - this is where I store videos that haven't been watched yet, virtual machines and software packages (debs, ubuntu releases and other stuff). Currently using 50% of it.
**sda2 25 Gb** - Windows 7. I hate it, but I need it to test applications under development. I'm currently using 16 GB of it.
**sda3 8 Gb** - testing partition, where I install Ubuntu betas and such, with only 1.63 Gb used, running kde-minimal and kde-plasma-addons over a command-line only installation from Ubuntu alternate CD.

As you can see, I have enough room on all system partitions. I'm currently being conservative, but I could shrink root and home to increase the data partition if needed. Besides, the last 3 partitions can be easily rearranged to accommodate different scenarios, since one partition is just for data and the other two are disposable systems.
