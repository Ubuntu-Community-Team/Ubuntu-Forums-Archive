### Post by RAOF on 2009-06-15
This is because not all gnome projects have actually released any 2.27 tarballs - including gnome-about (a part of the gnome-desktop tarball).  See [ftp://ftp.gnome.org/pub/gnome/desktop/2.27/2.27.2/sources/](ftp://ftp.gnome.org/pub/gnome/desktop/2.27/2.27.2/sources/) for details.  As far as I can see, we're almost entirely up to date (evolution was the only package I could see that wasn't at the latest version).
