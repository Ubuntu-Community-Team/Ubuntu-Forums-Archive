### Post by dro0g on 2009-10-20
I'm using a Blackberry Curve as a Bluetooth modem on Karmic using Blueman. It seems a lot slower after some updates last week but it's still working which I guess I should be happy about.

Basically the process in Blueman is:
1) pair the phone with the computer
2) Open Bluetooth Manager and right click on the phone and select Serial Ports -> Dialup Service
3) The connection then appears in network manager (like a wireless network)
