Can someone tell me how to make a USB Live Version and installer of Karmic Koala? I assure you every help is appreciated. Thanks in advance.

System > Administration > USB Startup Disk Creator.

The rest is rather easy.

Some machines will not boot off this sort of USB startup disk, so you might need to use Unetbootin from the repositories, which is just as easy to use (but doesn't allow a "persistent" USB install).
