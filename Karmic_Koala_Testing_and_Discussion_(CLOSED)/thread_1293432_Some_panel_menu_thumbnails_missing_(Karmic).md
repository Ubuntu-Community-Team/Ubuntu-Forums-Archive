### Post by Locke Cole on 2009-10-17
[FONT=Trebuchet MS][SIZE=3][COLOR=DarkRed]I discovered the solution for this recently.

Go to Preferences > Aparence > Interface > Show Icons in Menus

Aparently this option is disabled by default in 9.10.
