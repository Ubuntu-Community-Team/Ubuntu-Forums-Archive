### Post by cdEWoozy on 2009-08-23
Xsplash has nothing to do with the user list, that is generated by GDM.
GDM 2.20 offered the ability to exclude certain users from the user list, I can't find this functionality in the 2.27 documentation:
[http://library.gnome.org/admin/gdm/2.27/gdm.html](http://library.gnome.org/admin/gdm/2.27/gdm.html)

Either it is not yet implemented, not planned to be implemented or just not documented. Maybe someone else knows more.

### Post by cdEWoozy on 2009-08-25
> **JohnJackson said:**
> I just noticed in the [changelog]("https://launchpad.net/ubuntu/+source/gdm/2.27.90-0ubuntu1/+changelog") for gdm 2.27.90-0ubuntu1 this point:



I imagine you could add to this list if you could find it

The list is hardcoded in gui/gdmsetup/gdm-user-manager.c. Adding users to this list requires editing this file whenever you want to add users to the exclude list and then compiling gdm. The first link DryChill posted describes how to do that.
The second link contains a patch that adds the ability to dynamically add users to the exclude list, which is much more sensible.
