### Post by nomentero on 2009-10-04
Same insue here....looking for solution also:confused:

Solution;Solution is very simple. Open gnome-keyboard-properties, select tab Accessibility and deselect "Accessibility features can be toggled with keyboard shortcuts".

Best from Spain

### Post by Peter09 on 2009-10-04
Great - thanks.  On opening the Gui the Toggle Accessibility Features etc box was not selected, but by selecting and then un-selecting this box I got rid of the applet.  I do feel that this applet should have a right click menu - its not obvious otherwise on how to get rid of it.
