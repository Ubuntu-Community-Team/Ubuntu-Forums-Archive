### Post by Paqman on 2009-09-14
If you've not been involved in testing before it might be better to wait until later in the cycle. Jump in with the Beta in a few weeks to get an idea. Then if you're still happy maybe try an Alpha next time.

In the meantime, brush up on some of the basic skills: get familiar with some showstoppers like Grub and X/graphics drivers and learn how to get at them from the command line if you have to, learn how to use all the little features of apt-get and/or Synaptic, get a Launchpad account, and learn how to file good bug reports.
