### Post by Sophont on 2009-05-11
Waiting for Gnome 3 makes no sense IMHO.
Most of the design is independant of a particular Gnome version. Sure - some effects need support by a theme engine - but wallpaper, color scheme etc... is portable from one version to the next even if some parts have to be adapted for shiny new transparency effects and/or new widget elements.

Also what I have seen so far (I did install Gnome 3 to have an interactive look) of Gnome 3 shell is not very convincing. I hope there stuff planned to make that shell much more awesome than it is atm.
Not to be misunderstood - it ain't bad. It's cute and has nice effects. But for now it's more of a desktop management alternative to cube+docky - it's not the great breakthrough that people will expect as alternative to KDE4.

IMHO Gnome is mostly fine as it is. What I'd like to see for Gnome 3 is under-the hood improvements mostly. Get rid of old cruft. Clean up and optimize interfaces and libs.
For cool effects we already have the impressive power of what compiz and tools like gnome-do or awn can provide.

Personally I don't need a new default theme. I'm ok with the old one (though IMHO the login screen in jaunty was a step back - but not terrible) and I can easily switch to others if I prefer that.

As a poster said above earthy/chocolate brown and/or orangy tones are now as iconic for Ubuntu as blue is for windows, green for Suse and silver for Mac. Recognizability is a good thing. Not using the same old blue (whatever it's color-theory advantages it might have - or not - people will debate that ad nauseam as much as the pro/cons between single menu bar on screen-top or on every window (I hate the former - others love it - shrug) ).

I like Dust a lot - together with the jaunty wallpaper you get a very elegant look for Ubuntu - that clearly is recognizable as an Ubuntu release.

I'm currently giving Balanzan a try. Also a very good and Ubuntu-looking theme. The wallpaper and login screen look awesome.

Hm - got a bit off-topic above - shrug. ;-)
