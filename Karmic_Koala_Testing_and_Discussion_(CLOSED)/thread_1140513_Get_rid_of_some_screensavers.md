### Post by sertse on 2009-04-28
Some of you need to read. Your supposed idea for a screensaver-extras package already exists!

It's called ***rss-glx*** and ***xscreensaver-gl*** that is installed alongside ***gnome-screensaver***

***gnome-screensaver***, the actual program that does the work *only* contains 6 screensavers, 4 of them real...

1. Cosmos
2. Floating Gnome
3. Floating Ubuntu
4. Popsquares
5. Slideshow from My Pictures folder
6. Blank Screen

This follows upstream. GNOME, the makers of gnome-screensaver only contain 3 screensavers. Debian, Ubuntu's upstream, added their own branded one (Floating Debian), based on GNOME's default branded one, (Floating Gnome). Ubuntu, itself changed the branding from Debian to Ubuntu.

The correct implementation of this idea involves putting ***rss-glx*** and ***xscreensaver-gl*** to *suggests* instead of *recommends*, so it won't be install automatically when including gnome-screensaver.  

Any other implementation, like changing what is included in the core package ***gnome-screensaver***. significantly changes from upstream, which is generally against Ubuntu policy. I suppose it is possible to Ubuntuify popsquares, comsos etc like Ubuntu did with floating debian, but in anycase you got within the limitations of those 6 screensavers. That's the other problem with some people as well, "I agree with this BUT (my favourite screensaver)"

I'm getting too worked up over a trivial issue. lol

tl;dr

apt-get remove rss-glx xscreensaver-gl 

gnome-screensaver-preferences 

look at what's left.

### Post by autocrosser on 2009-05-25
I returned your PM--but for anyone else that is interested:

The flurry.tar has all the needed files in it--you just unpack it whilst seeing where they are placed--then do: gksudo nautilus & place the files in the same place in your system. Next either logout/in or reboot to have Flurry available to your system---you will need all the "extra" screensaver bits BEFORE installing the flurry bits.

*****DISCLAMER*****

I know that Flurry works with the nvidia driver--I DO NOT KNOW if it works with ATI, Intel or any of the open-source drivers---I recall that it was pulled because Intel systems were being bolixed with this screensaver--If you install it & then have a problem--either go in via the rescue console or run a LiveCD & remove the flurry bits, then reboot.

Of course--if anyone wants to run Flurry for testing use & reports that it is working with all again--I'll contact the developers to see if it can be re-included again......
