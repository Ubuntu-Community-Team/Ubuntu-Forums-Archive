### Post by Tibuda on 2009-09-26
I have tried it for a week. Here is what I think:

It uses a search-based application launcher by default instead of the menu-based, which is a step in the right direction in my opinion. It could be a barrier for people used to the menu-based approch, but this happen with every change. I don't like how it displays all workspaces in the launcher, but at least it will get people started into workspaces. It is not very customizable right now, but I think this will come in the future.
