### Post by jbrown96 on 2009-10-23
I like Karmic; there could be some minor issues, but I've found that especially with KDE the newer versions of Ubuntu get substantially better. There isn't any reason not to install karmic if you were going to upgrade from jaunty in the next week or two anyways. 
There's definantly a way to upgrade. The thing is that since karmic is still under development, there are a lot of updates. Be prepared for a lot of update requests. If this bothers you, you can change some settings in kpackagekit to check for updates less often.

If you just do a clean install of jaunty, then it will be a lot more work. 30 minutes to install jaunty + 45 minutes of updates (could be longer depending on your network connection) + 45 minutes to upgrade to karmic in a week. Or if you just go with karmic you have 25 minutes to install (installation is faster) + more requests to update (which can take place in the background).
