### Post by stevetxt on 2009-10-08
I'm using Karmic on a netbook, HP mini 110 and when I try to print from acroread the print window does not fit in the 1024 x 576 display.  In particular the "print" button is inaccessible off the bottom of the display.  This seems like Adobe's problem, but does anyone know a workaround?

There is no problem with the printing from evince and other applications I have tried.  This came up when I had a "fillable PDF form", the kind where you can't save what you've typed in, that seemed to work best with Adobe reader.

---

### Post by Aearenda on 2009-10-08
It IS Adobe's problem. However, you can workaround it simply by holding down the ALT key, then left-click-and-drag anywhere in the window (not just the title bar) and it will move so you can see the hidden parts.
