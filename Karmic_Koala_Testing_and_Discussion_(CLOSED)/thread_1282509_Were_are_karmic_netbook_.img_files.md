### Post by beastrace91 on 2009-10-04
Use [unetbootin](http://unetbootin.sourceforge.net/) to make an iso bootable on a usb stick. On Ubuntu installing it using ```
apt-get install unetbootin
``` or on Windows use the previous link. Then just install the netbook launcher package to make it "UNR"

~Jeff
