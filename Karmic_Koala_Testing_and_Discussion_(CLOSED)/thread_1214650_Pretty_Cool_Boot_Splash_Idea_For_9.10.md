### Post by pelle.k on 2009-07-17
> **nerdopolis said:**
> Also every time a new kernel is installed, it will revert.

No, not necessarily.
```
## additional options to use with the default boot option, but not with the
## alternatives
## e.g. defoptions=vga=791 resume=/dev/hda5
# defoptions=quiet splash
```
Change that to what you're after, and run "sudo update-grub", which is run at each kernel update, automatically, as well.
Just a note here, for once, you shouldn't remove the hash (**#**).
