### Post by Brandon Williams on 2009-10-19
The version of GDM in karmic is completely new. It doesn't provide any way to theme it via a special graphical utility. However, [this thread](http://ubuntuforums.org/showpost.php?p=7576112&postcount=365) describes how to configure some elements using gconftool-2 from the command line. You could change the same settings using gconf-editor, I would think.
