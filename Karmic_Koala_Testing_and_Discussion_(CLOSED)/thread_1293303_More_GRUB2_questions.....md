```As you see, I don't have any of the stuff you mentioned.
You may want to try one of the custom menuentries;
```

echo "Adding Zenix on sda13" >&2 
cat << EOF
menuentry "Zenix on sda13" {
        set root=(hd0,13)
        linux /vmlinuz root=/dev/sda13 so quiet splash
        initrd /initrd.img
}
EOF

```edited to suit your conditions.  This is a reat menuentry because it never needs edited for your kernel or even OS.  It boots what it finds on that partition and under the newest kernel it finds.  I think it is neat.
