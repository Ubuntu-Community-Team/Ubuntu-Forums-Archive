### Post by konqui on 2009-08-06
The Ubuntu team wishes to achieve this for Karmic

The planned features and goals can and should serve as guidance and inspiration for artwork, especially for backgrounds. 


[B]On the desktop: 
A fast, flicker-free and beautiful boot process. 

Possibly a new login experience that complements the graphical boot, and works well for small groups as well as very large installations. (This quite likely refers to [https://wiki.ubuntu.com/DesktopTeam/Specs/GdmFaceBrowser](https://wiki.ubuntu.com/DesktopTeam/Specs/GdmFaceBrowser)) 
A serious push for a new look from Canonical's design team. 
"Brown has served us well" but other options will be considered for Karmic Koala.
[/B]

I came across this distro long time back. [http://xange.serdigital.com/](http://xange.serdigital.com/)
It was called vixta before and that's when I tried it. It used kde3 and looked like this image [http://pcworld.hu/apix/0710/vixta3_k.jpg](http://pcworld.hu/apix/0710/vixta3_k.jpg)

The version I tried had a boot exactly like karmic's plan. Cursor on start. Could select verbose mode with mouse. The os was lightning fast!
