### Post by Raouf on 2008-11-07
I prefer using Synaptic Package Manger or Update Manger, so as they say wait for Firefox being added

but there's many other cases that you have to install tar packages:
*.tar.bz2 means there's some files compressed. It can contain everything.
To [***[COLOR="DarkRed"]compile a program[/COLOR]***]("https://help.ubuntu.com/community/CompilingSoftware"), first decompress the source code into a folder, open terminal and go to the path where the source code is, then write
```
./configure
make
makeinstall
```
