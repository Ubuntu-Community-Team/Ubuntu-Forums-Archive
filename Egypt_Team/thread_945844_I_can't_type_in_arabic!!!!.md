Get to System> Preferences> Keyboard
"Layouts" Tab
there's a "+", so you can add layout
choose Country> "Egypt" for example
choose Variants> "Arabic Digits"

you'll have to know how to switch between your Languages:
so in Layouts tab, you'll find "other Option"
choose "layout switching"
Default is both alt, I prefer "alt + shift" as windows:)
take care
