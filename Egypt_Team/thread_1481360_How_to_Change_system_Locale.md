### Post by simosx on 2010-05-13
> **M.3zzat said:**
> I've just set up Lucid Ubuntu , i am  totally  new to linux ;
So , i need to change my system locale to be Arabic beside English to  manage to read text documents in a right manner ;
Thanks in advance .

You should be able to read Arabic in Ubuntu Linux. The ability to read Arabic has to do with having installed an Arabic font. Ubuntu comes by default with some Arabic font.

To make sure that Arabic works correctly for your system, you need to install the Arabic language pack (which includes additional fonts and font configuration). You do this at System &#8594; Administration &#8594; Language Support and you add Arabic.

If you want to change the menu text of programs to Arabic, you need to logout, select Arabic and re-login again.

Now, some Arabic websites may not look properly due to encoding issues. In this case, you need to provide examples.
