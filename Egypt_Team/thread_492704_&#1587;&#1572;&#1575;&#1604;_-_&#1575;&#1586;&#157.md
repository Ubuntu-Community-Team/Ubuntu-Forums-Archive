### Post by ghost_ryder35 on 2007-07-11
VmWare Player or Server (Just in case you have files that will only run on Windows, or your in to Virtualization)
(To completely remove VmWare Server or Player you need to run the command[SIZE="2"]**[B] sudo apt-get remove --purge vmware-server vmware-player **[/B][/SIZE]if you installed from the repositories or [SIZE="2"]**sudo vmware-uninstall.pl **[/SIZE]if you installed from a .tar package. You also need to run sudo rm -r /etc/vmware two times to make sure it removes the folder!)

run these commands to competly remove VMware products from your computer, then reinstall.  You still have some folders/files left thats why you are getting the error.
