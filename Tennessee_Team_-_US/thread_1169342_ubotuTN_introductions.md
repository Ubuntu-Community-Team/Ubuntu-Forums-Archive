**I am pleased to announce ubotuTN to the Tennessee Team**[/CENTER]

ubotuTN is an IRC bot designed for the Tennessee LoCo's IRC channel. #ubuntu-us-tn, on Freenode. It contains 
[LIST]
[*]a factoid database that is up to date with the original ubotu on Freenode. See [http://ubottu.com/factoids.cgi]("http://ubottu.com/factoids.cgi")
[*]an Ubuntu bug searching plugin
[*]hopefully more features to come :)
[/LIST]

Because ubotuTN is developed on top of rbot, this bot is fully functional. For a list of rbot's features see [http://ruby-rbot.org/rbot-trac/wiki/RbotFeatures]("http://ruby-rbot.org/rbot-trac/wiki/RbotFeatures")

Check out ubotuTN's homepage at [https://code.launchpad.net/~tennessee.team/ubotutn/ubotuTN]("https://code.launchpad.net/~tennessee.team/ubotutn/ubotuTN") You can grab the source by issuing this command ```
sudo apt-get install bzr
``` and then ```
bzr branch lp:~tennessee.team/ubotutn/ubotuTN
``` Please feel free to contribute bugs, patches, or anything else to help make this project better.
