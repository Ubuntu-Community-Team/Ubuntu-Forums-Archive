### Post by Luke has no name on 2009-08-20
The books out there now for Ubuntu server are, in no particular order:

Practical Guide to Ubuntu Linux, 2e (Prentice Hall)
Beginning Ubuntu LTS Server Administration (Apress)
Professional Ubuntu Server Administration (Apress)
Ubuntu Linux Toolbox (Wiley)
Ubuntu Server Administration (Network Professionals Library)
Ubuntu 9.04 Server: Administration and Reference (Surfing Turtle Press)
The Official Ubuntu Server Book (Surfing Turtle Press)

What are your thoughts on any or all of these books, if you've read them? Here's my take:

**Beginning Ubuntu LTS Server Administration (Apress):**
I bought this book, and have read through most of it. I would not recommend buying it unless you are relatively new to Linux administration in general. On a side note, I have yet to see a quality, easy to follow, ACCURATE virtualization guide (to my knowledge). I have not checked the "Official" book. [Note, I don't have the "LTS" version of this book.]

**Ubuntu Linux Toolbox (Wiley):**
I flipped through this book. It looks like it would be something useful to have on your bookshelf, though it is very similar to the other Linux 'toolbox' books, only moderately tweaked for Debuntu. If you find it for sale for cheap, get it.

**Ubuntu Server Administration (Network Professionals Library):**
I don't remember too much from this book, so don't go off my judgment alone. It had a lot of information in it, as does the "Official" book, but I don't know if I would buy it. There are newer, more relevant books out there in the same market as this one.(I got it at the library).

**Ubuntu 9.04 Server: Administration and Reference (Surfing Turtle Press):**
Haven't looked at it, but I really want to know how it stacks up to the "Official" book.

**The Official Ubuntu Server Book (Surfing Turtle Press):**
The format of the book was a bit wordy and confusing at points, though it is crammed with good information. I will strongly consider buying this book. It does cover a lot of info not covered in the Ubuntu Server Guide. If only we had more detail and timely updates to the Server Guide! Detailed RAID and LVM setup recommendations, tons of network configuration tips, system services setups, etc. were really helpful. 

So those are my thoughts. Post yours here, and if you can make a quality review, post it on Amazon! Thanks, guys.

the best ones (in my humble opinion, with my own reviews) in that order

-A Practical Guide to Ubuntu Linux (Versions 8.10 and 8.04) (2nd Edition) (Paperback)
-The Official Ubuntu Server Book (Paperback)
-Beginning Ubuntu Server Administration: From Novice to Professional (Expert's Voice) (Paperback) 

Last one, I got the 1st edition, so ... I'll talk about that particular version. I own all of them

1st one is the most complete. It explains things in clear way but is a bit wordy and its size make it unpractical (really thick) But if you are serious in learning, you will find almost everything here.

2nd one, I bought that one just 1 week ago. Easy to read, like the writing style. But not very in-depth. Topics like services (Web, FTP, Samba, etc) are not fully covered. But I like the security chapter and the UFW examples. the software raid portion is nice too.

3rd one ... explains things in a more detailed format than the "Official Ubuntu Server" but in my opinion, is not so easy to read. However, I like it a lot. I got the pdf.

I would like to mention Beginning Ubuntu Linux, Third Edition by Keir Thomas. It is not a Server book but it contains some useful information and concepts that still apply to server. The writing style of this guy "rocks"... wish he can write something about Server.
