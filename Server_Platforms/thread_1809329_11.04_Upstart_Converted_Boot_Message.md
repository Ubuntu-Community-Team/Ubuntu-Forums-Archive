### Post by Athen on 2011-07-21
After some time playing around, I was able to run the following command for each service complaining about upstart:

```
 update-rc.d -f (service name) i.e. mysql remove
```

This fixed my issue. I am just so used to looking at the boot screen and watching the services start, I guess you no longer get this using upstart? It will be missed :cry:
