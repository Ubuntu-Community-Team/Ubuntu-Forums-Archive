### Post by ade234uk on 2007-01-20
Our company has upgraded servers that run the auction site. It is my job to swap the sites over.

Our old server was running FreeBSD and had the following specs:-

   2Ghz Celeron Processor
   512MB Memory
   40GB Hard Disk
   No Data Transfer Limit
   PHP4 / MYSQL 4

The new server we have ordered has the following Specs:-

    2.8Ghz P4 Processor
    1GB Memory
    2 x 80GB Hard Disks
    Hardware RAID
    No Data Transfer Limit
    PHP5/FAST CGI / MYSQL 4

In your own experiences what is Ubuntu like compared to FreeBSD for servers. 

The FreeBSD server was a speed demon. The site was getting around 25,000 page views a day and also had to cope with querying MySQL as well and it done a brilliant job.

I love Ubuntu myself, I use it at home from own websites, but there is no pressure put on my home box, like there will be with the Auction site.
