### Post by RealPSL on 2008-08-10
Both the private and public keys are portable. However for security reasons you must ensure that the path to the your ~/.ssh directory is only writable by your user id or root. Your home directory cannot have group write access for example. The access for the .ssh directory should preferably be 700. This is the most common reasons why they setup does not work.
