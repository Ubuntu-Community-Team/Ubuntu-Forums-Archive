Is there an installed ssh-client (such as putty) you can run? Would it be possible to talk the IT-department into installing an ssh-client?

If that's the case you can run a webproxy on your homecomputer and connect to it through an ssh-tunnel. Besides using the ssh-client to setup a tunnel the only change you'll have to do is to tell your browser to use "localhost:portnumer" as a proxy.
