### Post by arrrghhh on 2010-05-13
Where you download any other version of Ubuntu.  Intel, x86 (32-bit) is called "i386"...

Go [here](http://www.ubuntu.com/getubuntu/download-server), and click the "Alternative download options" link and make sure you select 32-bit.  

Grab a [torrent](http://releases.ubuntu.com/10.04/ubuntu-10.04-server-i386.iso.torrent) of the disc image if you want faster downloads!
