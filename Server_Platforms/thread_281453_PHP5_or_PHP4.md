I think if you are going to use Joomla as a business solution (mission critical), you better go with php4. Usually more features, mean more bugs, and since php5 has not sinked-in completely as the new stable/standard, I would recommend going with 4.

On the other hand, if you just want to play around and learn about joomla, etc., I think either php4 or php5 would pretty much give you the same results. I don't use Joomla anymore, but if I remember correctly, you can use php5 with it without too many problems.
