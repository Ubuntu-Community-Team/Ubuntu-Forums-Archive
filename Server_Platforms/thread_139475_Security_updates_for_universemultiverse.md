### Post by Xian on 2006-03-04
[QUOTE=lean]Instead unstable just upgrades to new versions, which only sees the light in the next ubuntu distribution, and never in a stable one.[/QUOTE]

Which includes upgrades due to security reasons, which is why SID does not have a security update team. However, in between Ubuntu releases you would not have access to the newer SID builds, and unless they were covered in backports it would be a situation that would fall into the 'unsupported' category.
