### Post by clrg on 2010-08-08
Make a backup copy of /etc/udev/rules.d/70-persistent-net.rules:
```
cp /etc/udev/rules.d/70-persistent-net.rules $HOME/70-persistent-net.rules.backup
```

Remove the eth0 entry and rename the eth1 entry to eth0:

```
sudo vi /etc/udev/rules.d/70-persistent-net.rules
```

Restart

```
sudo init 6
```
