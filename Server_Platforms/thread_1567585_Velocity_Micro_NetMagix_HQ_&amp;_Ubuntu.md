### Post by BkkBonanza on 2010-09-04
Assuming you can boot from a USB flash drive then that would be your best bet. You would have to check the BIOS to see if that is possible.

You will need another computer with Ubuntu running either by booting LiveCD or installed. Insert USB stock there. Select System, Administration, Startup Disk Creator and use it to create a LiveCD boot disk on the given flash stick.

Bring that over to the target system and boot from it and choose install.
