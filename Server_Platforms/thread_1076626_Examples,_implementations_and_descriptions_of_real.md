### Post by cariboo on 2009-02-21
It really depends on what you want to use your server for. There is no defaults. If you want to see what options you have, at the prompt type:

```
sudo tasksel
```

This assumes you are running a standard server setup with no gui.

If you do have a gui and have synaptic installed open synaptic and go to Edit-->Mark Packages by Task.

Jim
