### Post by volkswagner on 2009-01-11
You can use [Webmin]("http://www.webmin.com/") for administration.

[LinPHA]("http://linpha.sourceforge.net/wiki/index.php/Main_Page") is one of several photo archive/share sites you can install.  I have tried this one.  It is pretty cool as you can easily set private and public folders and user permissions.  Although installing this prompted me to upgrade my server from pent2 400.  I believe 1Ghz is recommended.  :p
