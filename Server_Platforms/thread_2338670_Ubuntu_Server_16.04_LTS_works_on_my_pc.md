### Post by ajgreeny on 2016-09-30
Ubuntu server edition does not have a desktop environment by default; it is command line only which is normal for a server.

You can add a GUI to the server if you wish but choose a low resource version if you must have one, such as openbox.
See [https://help.ubuntu.com/community/Servers](https://help.ubuntu.com/community/Servers) for much more info.
