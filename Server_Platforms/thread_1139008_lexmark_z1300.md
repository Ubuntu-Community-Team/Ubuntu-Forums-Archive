### Post by HermanAB on 2009-04-26
You can either run WinXP on an old PC and hook the printer to that, or install Virtualbox and install WinXP as a virtual machine then install the printer in that.  Once Windows is running and printing, simply share the printer, then go back to Linux and use the print wizard to go and look for it.

If that sounds like too much work, sell the printer on Ebay and buy a HP printer.
