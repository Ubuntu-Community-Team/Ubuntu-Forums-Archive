### Post by nerdtron on 2014-06-30
Why is this thread marked as solved? Is it solved already? How did you solved it?


Anyway if it isn't solved yet.
Shutdown the bond0 and interfaces.
```
ifdown bond0
ifdown em1
ifdown em2
```
Then try changing the "bond-slaves em1 em2" to "bond-slaves none".
Then activate again.
```
ifup em1
ifup em2
ifup bond0
```
