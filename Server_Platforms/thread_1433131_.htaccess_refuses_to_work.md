### Post by cdenley on 2010-03-19
> **raptormanad said:**
> Tried this... I didn't get any error. I guess it's not reading it?

It must mean it's not looking for it. If it failed to read it, you would get an error. Either you didn't name it properly or "AccessFileName" isn't set properly.
```

ls -l /var/www/.htaccess
grep AccessFileName /etc/apache2/apache2.conf /etc/apache2/mods-enabled/*.conf /etc/apache2/httpd.conf /etc/apache2/ports.conf /etc/apache2/conf.d/* /etc/apache2/sites-enabled/*

```
