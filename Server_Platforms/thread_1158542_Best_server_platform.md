### Post by HermanAB on 2009-05-13
Depends...

If you Mom needs to use it, then you would probably be better off with Windows XP Professional.

If you are the only user and you intend to use it as a Linux learning tool, then Ubuntu is good.

If you are the lazy sort and just wants a Linux machine that works and has wizards like Windows, then use Suse or Mandriva.

### Post by a9k3d on 2009-05-14
Assuming your Mom got used to Fedora last time, you should be fine with Ubuntu. Don't go with the server install disk though. It sounds like you need the desktop version and then add apache. I'm assuming your "server" is to play with not to serve 100,000 hits per day.
If you use the normal install, you will get excellent printing support. I remember the old configure your cups with a text editor days. We've come along way and everything has gotten easier and smoother. You get more solitaire games than Windows too.
