### Post by rapiscan on 2008-03-01
I would argue Ubuntu server is your best choice, especially if you're setting up a linux server for the first time.  Ubuntu, in my opinion, has the best community and documentation for those that are just starting with linux.

Here is the detailed documentation that you're interested in:
[http://doc.ubuntu.com/ubuntu/serverguide/C/index.html](http://doc.ubuntu.com/ubuntu/serverguide/C/index.html)

When you're interested in restricting access to/from the internet, check out the firewall section.
