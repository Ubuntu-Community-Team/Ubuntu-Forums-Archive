### Post by nerdtron on 2013-11-25
I'm a newbie here too. But I was able to setup a mail server in less than a day. I used iRedmail.
It has a webmail interface based on roundcube and it has spam and antivirus filtering. The install instructions on the website is very straight forward. It is almost one click install.
Our server is now is production and I can say it is very good.
You can test it first on a VM and installation will take less than 30 mins.
[http://www.iredmail.org/install_iredmail_on_ubuntu.html](http://www.iredmail.org/install_iredmail_on_ubuntu.html)
