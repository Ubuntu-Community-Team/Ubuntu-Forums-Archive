### Post by Mia1990 on 2012-01-23
Back up Servers are just like file servers but they keep a copy on your desktop and your server.Just find a good back up software set the backup location to the directory on your server and you should be done with it.

I have also heard of people setting cron jobs to do just this kind of thing.You may want to look into this.

Edit: I found these how to's when i looked it up

[https://help.ubuntu.com/community/BackupYourSystem/SimpleBackupSuite](https://help.ubuntu.com/community/BackupYourSystem/SimpleBackupSuite)

[https://help.ubuntu.com/community/BackupPC](https://help.ubuntu.com/community/BackupPC)
