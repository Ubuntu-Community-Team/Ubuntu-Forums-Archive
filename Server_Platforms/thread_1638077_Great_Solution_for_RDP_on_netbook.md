Hey all.  I was having a lot of difficulty with rdp into a headless virtualbox windows machine running in an ubuntu server using krdp and rdesktop on my netbook - primarily  related to issues with resolution and scrolling.

Finally I have found a great solution called Remmina Remote Desktop Client and since I did a search here and found no mention of it I thought I would share.  It is gtk based so there was a lot of additional package installation onto my kubuntu netbook install but the result was well worth it.

Found through this blog entry:
[http://theubuntunews.blogspot.com/2010/08/remmina-remote-desktop-client-081-on.html](http://theubuntunews.blogspot.com/2010/08/remmina-remote-desktop-client-081-on.html)
