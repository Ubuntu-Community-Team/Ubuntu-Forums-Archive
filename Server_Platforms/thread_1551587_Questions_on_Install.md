### Post by cdenley on 2010-08-12
I'm not sure the server installer offers to do a dual-boot, since having Windows installed on a Linux server doesn't make much sense. It is possible to have it dual booted, I'm just not sure if the wizard configures it for you or if it defaults to that configuration if you already have windows. You can always use the desktop version, though. There is nothing the server version can do that the desktop version can't. The server version has no GUI by the way, so I hope you are familiar with Linux commands.
