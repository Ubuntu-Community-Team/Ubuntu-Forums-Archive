### Post by craigp84 on 2012-12-24
For this use case i'd use the built in shared folders functionality in virtual box.

To do this over the network, you will need to make sure your virtual machine is using either a bridged interface or a "host only networking" interface.

Just purely from a "reduce your attack surface" point of view, i'd encourage using the faster shared folder functionality as it eliminates the need to have any network services exposed at all which is always a bonus where it is possible.

Hope this helps
