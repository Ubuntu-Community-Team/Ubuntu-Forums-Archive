### Post by lykwydchykyn on 2008-11-02
I'm surprised no one has recommended Python.  Python is very easy to learn, yet quite powerful and you won't wear it out like some beginner languages (VB, ahem).  Not as straightforward as PHP for web scripting, but it can still do the job.  

PHP was the first language I really dove into, and it's a great "instant-gratification" language because when you're working with web apps, you get around the tedium that is GUI programming.    You just spit out HTML and the browser does the rest.  Unfortunately it's not much use outside of creating dynamic web pages.  For system administration scripts or client-side apps, it's the wrong tool.

Never tried Perl myself, so I won't make any ignorant proclamations about it.
