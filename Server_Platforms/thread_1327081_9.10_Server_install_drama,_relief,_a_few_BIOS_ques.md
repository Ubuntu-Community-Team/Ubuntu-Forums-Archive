You can't run a computer without the BIOS because the BIOS POST is the first thing to run. The BIOS also is the program that looks at the MBR for the boot-strap program. VirtualBox needs Hardware Virtualization to run a 64 bit OS. Just a few exaamples.

### Post by hessiess on 2009-11-15
> **theozzlives said:**
> You can't run a computer without the BIOS because the BIOS POST is the first thing to run. The BIOS also is the program that looks at the MBR for the boot-strap program. VirtualBox needs Hardware Virtualization to run a 64 bit OS. Just a few exaamples.

That's what core boot does, initialises the hardware and boots a kernel without the legacy 16 bit IBM BIOS. After the OS is booted, the BIOS is not used much, if at all. Which would mean that settings like the OP's `installed os' wouldn't have any effect.
