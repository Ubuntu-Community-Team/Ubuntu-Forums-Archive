### Post by drubin on 2009-01-20

### Post by mregister on 2009-01-21
Ok that worked. Thanks for the .htaccess tip. I had seen that before but wanted at least one more opinion. The awstats was an easy setup. I followed:
[http://www.debuntu.org/book/export/html/67](http://www.debuntu.org/book/export/html/67) with a few adjustments. I did not copy the awstats.conf file to awstats.[www.mysite.org.conf](www.mysite.org.conf). I just put my information into /etc/awstats/awstats.conf. I also had to set the permissions to 755 on /var/log/apahce2/access.log to get rid of the permissions error I was initally getting when manually running the stats for the first time. I may post my entire setup and config process after I do it one more time on another site.

But I do love the stats it generates.
