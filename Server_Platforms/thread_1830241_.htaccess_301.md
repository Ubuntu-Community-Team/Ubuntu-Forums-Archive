### Post by Dry Lips on 2011-08-21
Solved!
  

  I found a way to do it:
   Make sure you have an index.html in your root directory,
  then add this code to the .htaccessfile:
  ```
redirect 301 /index.html /path_to_your_folder
```


How this works in practice:
I wanted [http://10.42.43.20](http://10.42.43.20) to redirect to [http://10.42.43.20/test/script](http://10.42.43.20/test/script).
The code would be like this:

```
redirect 301 /index.html /test/script
```


That wasn't too difficult, now was it?
