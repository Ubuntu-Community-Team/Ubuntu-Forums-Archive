### Post by sp0nge on 2009-03-14
Any machine that you want to devote to the purpose can be made into a "server". As long as your hardware can handle the demand you put on it, there is little to worry about. 

I turned an old Sony Vaio into my home server.. this is older hardware that originally had Windows ME loaded on it (that will help date the machine). I upgraded to a bigger hard drive and maxed the memory out then I followed [this guide ]("http://www.howtoforge.com/perfect-server-ubuntu8.04-lts") which really helped me understand the process better. 

Once all was set up, this machine first became an internal media server and overall storage for myself and my family. I have also been experimenting with virtual hosting and have opened this to the outside world a time or two just to test the waters.. It was a great (read:sometimes frustrating) experience but I am happy to have done it. 

Feel free to PM me if you get stuck.
