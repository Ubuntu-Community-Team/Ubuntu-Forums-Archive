### Post by Wolfhound6 on 2008-04-08
I installed ossec server on Ubuntu Server 7.10 and then I loaded ossec agent on my laptop running Ubuntu Desktop 7.10.  I went back to the server to add an agent, but to fully add the agent you need to copy the Agent Key and transfer it to the agent system and paste it into ossec there.  

How do you copy and paste when my installation of server does not have a gui?  The key only appears on the screen and does not give you the option to save it anywhere. 

Thank you for help.

---

### Post by kwambus on 2008-04-09
Open an ssh session from your server to your laptop in a terminal window. Just mark off the key text, copy to clipboard, then paste into the laptop or server terminal window. Voila.
