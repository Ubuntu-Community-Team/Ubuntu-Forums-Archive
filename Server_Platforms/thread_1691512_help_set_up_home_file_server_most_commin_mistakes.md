### Post by aeiah on 2011-02-20
your searches may be coming up short for a number of reasons. as a general rule for a linux file server: samba if there are windows computers, NFS if there aren't.

you should be able to find loads of guides on the ubuntu wiki and beyond.

i dont know about any common pitfalls, but just make sure you make good use of the server's log files if things dont go as expected and make sure rudimentary things work (such as pinging your server from the desktop etc)
