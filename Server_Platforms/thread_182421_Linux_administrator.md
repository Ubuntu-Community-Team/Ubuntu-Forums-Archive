### Post by Endwin on 2006-05-26
It all depends on the network you are setting up and the people who will be using it. If it is just you and X computers (where X >1) then you probably don't need to do a whole lot unless you want to learn some sort of program or network setup. Now if you are setting it up for say your family, with say a file server for everyone to save and store backups of their personal files, when you may end up doing more. You may want to end up monitoring traffic to see what kind of iteration the machines, and consequently users, are up to. That is always fun. So what kind of lan, users, machines, services do you want to setup?
