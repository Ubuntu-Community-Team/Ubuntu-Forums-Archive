### Post by sveinh on 2009-01-31
Satega: The Open Home Server, is an open source project which aims to develop a Linux alternative to M$ Home Server. We are getting close to our first technical release to the public. In order to complete the work, we are in need of help and guidance in developing an intuitive and nice looking user interface for the administrators and users of the server.

The project is using an Apache/PHP based front-end for the administration interface. We are using Kohana as a framework for the PHP code. The front-end interface connects to a back-end process written in Python using xml-rpc. 

Our most urgent need it to get help from someone who can contribute in the following areas:

[LIST]
[*]Create a design/mock-up of the overall look-and-feel of the overall application
[*]Create a CSS with guidelines on correct usage
[*]Provide designs/mock-ups of the screens/functionality to be included in the first release
[/LIST]

We already have screens developed for user and group administration and file-sharing with Samba which could provide a starting point for the designer. 

By using Kohana we have completely separated our program control and data from the html presentation logic. This means that the GUI designer can focus on making a good design (no PHP coding required), and the programmers can quite easily take the designed screens and implement into the application.

If you are able to contribute, please visit us at [http://www.satega.org/forum](http://www.satega.org/forum) and drop us a line in the Participate forum.

In advance, thanks!
