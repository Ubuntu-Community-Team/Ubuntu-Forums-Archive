### Post by mixmaster on 2009-10-03
I use a tranquilpc T7.  About 8.5x8.5x2 inches, fanless.  The case acts as one big heat sink.  CPU is a dual core atom.  Max 2Gb memory plus one 3.5 or up to two 2.5 inch drives.  Mine has a WD caviar green 1Tb drive in it and is effectively silent (in a completely quiet room you can hear a "hollow" sound from the drive).

You can get cheaper if you build for yourself but you won't get anything much neater.
