### Post by Hablas on 2010-05-25
Apologies every one.

The problem was with putty. I found it out from here [http://www.kapsi.fi/ohjeet/irssi-utf8.html]("http://www.kapsi.fi/ohjeet/irssi-utf8.html")

The site is in Finnish but the short story is, change putty charset from Window &#8594; Translation &#8594; Character set &#8594; UTF-8.

After that it works like a charm.
Screen was not the problem here.

Just make sure the client, server, screen and irssi use the same character set.

Now at least there will be a note somewhere.
