### Post by VK7HSE on 2009-06-05
This really depends on what you feel is more important, for the Ubuntu 8.04 release this is a LTS, meaning that it will receive security updates for a longer period and should be considered "Stable" where as the Ubuntu 9.04 is the latest release that will include newer versions of default software, but may also introduce some bugs that will affect you experience. So again it comes back to what you feel suites your preference. Now to contradict myself! if you are not planing on a dual boot then I would go with 9.04 just as this is because I like to use the latest!
