### Post by Swiss2K on 2007-11-12
Yes there is! and better if you ask me.
Slimserver is serving my 30.000+ musicfiles perfectly.

add to etc/apt/sources.list
deb [http://debian.slimdevices.com](http://debian.slimdevices.com) stable main

run apt-get update

install the software by
apt-get install slimserver

run and configure slimserver
[http://yourdomain:9000](http://yourdomain:9000)
