### Post by ian-weisser on 2014-02-16
And x11* and gir1.2-*

A good starting point is to look at every package that is not a (direct or indirect) dependency of the ubuntu-standard metapackage. That metapackage provides a full-featured CLI-only system that you can add your server packages to.

If you try to uninstall a package, and it wants to uninstall the ubuntu-standard metapackage, don't do it. You probably need it.
If you try to uninstall a package, and it wants to uninstall the ubuntu-minimal metapackage, REALLY don't do it.  You definitely need it.
