### Post by cdenley on 2009-05-08
> **Marvin666 said:**
> Will a sever versoin of xubuntu jaunty be any faster thjan the standard edition? I've heard server versoins of wiondows are faste4r, so was wonderinhg if the same priciple applied to ubuntu as well.

There is no such thing as xubuntu server. The server version is just standard ubuntu with no desktop interface, the server-optimized kernel installed by default, and a menu that lets you choose server configurations during install. You can always start with xubuntu, install the server kernel, remove xfce and any desktop packages, then choose configurations from "sudo tasksel", then you have a server version.
