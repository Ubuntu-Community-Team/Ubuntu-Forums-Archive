### Post by LordHunter317 on 2006-06-10
Install the desktop distribution.  Install whatever server packages you please.  Alternatively, starting from a server install, install one of the '*-desktop packages.'  The LAMP install option simply causes apaceh2, mysql, php to be installed as part of the inital setup.   It doesn't do anything extraordinarily special.
