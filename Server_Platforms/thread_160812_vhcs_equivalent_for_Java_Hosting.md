**Upgrade package for VHCS 2.4.7.1 that enables JSP/Servlet Tomcat Application Server to be automatically configured. **

**Key features:**  
**1.** Automatic Tomcat Virtual Host Context configuration when new domain is created 
**2.** Automatic Tomcat Virtual Host Context configuration when new sub domain is created
**3.** Test Article at index home page generated for every domain and subdomain
**4.** Servlet Test accessed via home page Test Article 
**5.** JSP Test accessed via home page Test Article 
**6.** Tomcat Refresh at every domain or subdomain change via VHCS panel
**7.** Enable Apache - Tomcat connectivity using mod_jk module
**8.** Fine tunning Apache configuration, so PHP ,JSP and Sevlets projects to work together 
 
**Versions:**  
**1.** Java (JDK 1.5) + Tomcat5.5.x  
**2.** Java (JDK 1.4.2) + Tomcat5.0.x
