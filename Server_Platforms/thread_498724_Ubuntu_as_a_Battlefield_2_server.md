### Post by primski on 2007-07-12
empty window with a loading cursor?

There is no window in BF2 server mode. After you unpack it, run ./start.sh to run the server. Loading screen comes up, showing what map and settings etcetc...

I suggest using BF2CC, command & control software, check it out at bf2cc.com, basicaly its a daemon that takes care of your server which you can administer remotely from your windows machine.

Stop over at [my clan's site]("http://www.slo-gamers.com") and take a look around, ask a question on the forums, email me, whatever. I run a BF2 server based on linux too.
