Well,

Natty has been (still is!) a rough ride on my desktop

11.04 server, however, has 'finally' enabled me to run Ubuntu server on some hardware I bought over 6 months ago now.

Both 10.04 and 10.10 were very erratic on my (then) new Intel DH57JG motherboard.

Server would regularly crash during boot or at some point after on the rare occasion it limped into life.

Sure looked like either an overheating or a memory issue, but I managed to discount both of those.

Anyway, 11.04 64 bit is running like a dream at the moment.

Phew!
