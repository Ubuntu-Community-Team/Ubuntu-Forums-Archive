### Post by Aircooledmadness on 2007-04-06
You wouldn't need a hardcore system to run something like that.  Maybe ebay a used system to save some cash?  Another plus with getting some used hardware is that the longer its been around,  the better linux support it has.  I've had some troubles with the new SATA drives.

FYI I've got a AMD XP2000 system with half a gig of ram serving up 1200 gigs of data to about 9 computers,  plus streams vids to a modded xbox.  It also hosts a few small websites,  runs torrentflux and some other goodies.  I also ssh into it from class to take notes on.  Never had a problem with it being too slow.

Running linux headless will produce some amazing results on older hardware.
