### Post by fang0654 on 2010-05-26
Unfortunately, likewise-open is not compatible with Samba 3.4.x at this time.  You'll notice lots of winbind errors.  I found this out the hard way when I went to set this up for a client and ran into the same thing.

On the plus side, it is pretty easy to set up Samba talking to AD in 10.04.  The instructions are here:

[https://help.ubuntu.com/community/ActiveDirectoryWinbindHowto]("https://help.ubuntu.com/community/ActiveDirectoryWinbindHowto")
