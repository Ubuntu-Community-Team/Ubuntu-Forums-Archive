### Post by Alpha_toxic on 2006-01-11
If I understand, you want to install Edubuntu on the partition where Ubuntu was? If so, just start the instalation of Edubuntu and when you get to the formating dialogs, just do a manual partition. Format the Ubuntu partition in ext3 and give it mount-point "/".
Suggestion: don't use PQ-Magic for formating/partitioning, it will most likely screw up things (or at least it's more likely than if you're using the formating utilities in the install cd of any linux)
