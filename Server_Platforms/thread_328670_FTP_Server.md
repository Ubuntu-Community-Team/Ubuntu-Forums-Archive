### Post by eric_stewart on 2006-12-31
I use Pure-FTP.  Very solid with good logging tools.

I'm sure some will post about the security vulnerabilities of FTP.  My simple advice (simple is usually good!) is to make sure that the username/password that you set up meet some basic rules for length and complexity, turn off anonymous login, ignore everyone who tells you how unsecure FTP is.
