### Post by HermanAB on 2009-03-31
Yes, Ubuntu is as secure as any other Linux.  Doing auto updates is fine if you are not doing anything out of the ordinary.  If the machine installed flawlessly without any tweaks then auto-updates will also work flawlessly.  However, if you had to do a lot of work to get the machine installed, then auto-updates may screw it up - YMMV.

Ubuntu is *not* the easiest Linux version.  It is intermediate.  The easiest Linux is Mandriva, followed closely by Suse.  With those two old distros, you can literally do *anything* with the click of a mouse, including set up of complex servers.  Their wizards are simply incredible.

However, Ubuntu is a fun distribution and the help forums are excellent, so you will likely enjoy configuring a server the somewhat harder way.

I use Mandriva for work and Ubuntu for fun, since I cannot justify spending time on mundane things at work - using wizards is more productive especially if you have a zoo of machines.
