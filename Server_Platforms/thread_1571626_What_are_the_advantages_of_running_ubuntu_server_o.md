### Post by bodhi.zazen on 2010-09-09
> **waloshin said:**
> What are the advantages to runing ubuntu server over cent os and debian?

The answer to that question is IMO a matter of opinion.

Advantages of Centos : Selinux, some sysadmin tasks are a little more automated on the command line, making a certificate for ssl (https) comes to mind. To some extent it depends on what you are used to.

Advantages of Debian / Ubuntu: Debian does well server side and it tends to be lighter on resources then Ubuntu and (Debian) packages have less dependencies. Other than that, IMO, most differences between Ubuntu and Debian server side are rather trivial.

Seconded. I think the main point here is familiarity. If you're comfortable with ubuntu, apt, and such, it makes the transition to pure CLI a little easier to deal with. It's all taste - use whatever will make the server and administrator the most efficient.
