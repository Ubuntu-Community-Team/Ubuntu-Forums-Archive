### Post by rsw686 on 2007-12-16
Almost any linux distro can run a web server. It just depends on which distro is your personal preference. My one web server box runs Fedora (probably not the best choice but I built it years ago), CentOS would be better choice now for a redhat fans. I currently prefer Debian or Ubuntu LTS for server situations. Ubuntu's root as from Debian so they are very similar. I know a lot of folks who swear by Debian, but Ubuntu Server seems to be more user friendly for those just starting out with its LAMP configuration.
