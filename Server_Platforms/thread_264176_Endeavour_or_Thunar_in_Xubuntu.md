### Post by Psquared on 2006-09-24
Thunar comes with the default installation, but Endeavour2 is a better file manager as far as I can tell. Particularly if you start it using GKSudo which gives you the ability to move files. The left pane in Thunar does not open folders it only shows what you are looking at in the right pane.

I'm still messing with it to see what it can do, but it is more like Commander for Xubuntu.
