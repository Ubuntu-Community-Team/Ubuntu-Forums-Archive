Doesn't look like it's in the respositories, you will have to install it by hand. Looks fairly simple though.

[http://linux.softpedia.com/get/Programming/Version-Control/Codestriker-944.shtml](http://linux.softpedia.com/get/Programming/Version-Control/Codestriker-944.shtml)

Here is the whole manual. It looks like it's written in Perl and the install script is just a Perl script (install.pl). All you have to do is download the source code, unzip it, and run the installer.

[http://codestriker.sourceforge.net/codestriker.html](http://codestriker.sourceforge.net/codestriker.html)
