### Post by Randomskk on 2006-07-16
Debian's stable, sure, but I very much doubt you'd notice any difference.
Dapper has all the latest apps, repos FULL of good programs, great support, etc - and it is stable as well.
Debian may have lots of stability, but all it's apps are old.
Overall, if you're running some kinda mission critical server which absolutely must be on 24/7 and you've even got a second copy for spare hardware, debian may be an idea, but otherwise Dapper should be more than reliable enough.
