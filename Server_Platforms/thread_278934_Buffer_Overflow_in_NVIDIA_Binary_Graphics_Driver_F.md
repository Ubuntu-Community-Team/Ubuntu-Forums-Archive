### Post by l.tambiah on 2006-10-20
Yes it is disabled by default, if you dont see it in the xorg.conf it is disabled. Alternatively you can find the section "Device" section and add the line like below:-

```
[SIZE=2]Section "Device"
    Identifier  "Card0"
    Driver      "i810"
    VendorName  "Intel Corp."
    BoardName   "82852/855GM Integrated Graphics Device"
    BusID       "PCI:0:2:0"
    [COLOR=Red]Option      "RenderAccel"  "false"[/COLOR]
EndSection[/SIZE]
```At least this way you know its turned off for definate.
