### Post by liqwiz on 2009-10-05
Through the Direct Connect Wikipedia.org page I found the following 2 options.

First there's a text based client (DCTC, [http://ac2i.homelinux.com/dctc/](http://ac2i.homelinux.com/dctc/)). The creator states that it's not to be used by humans, but rather by other software. So if you're a programmer you can use this.

The other is a console based client designed For humans (LDCC, [http://www.softservice.com.pl/ldcc/](http://www.softservice.com.pl/ldcc/)). This will probably be the one that does the trick.
