### Post by Jimmy9pints on 2009-04-08
There is a great home server project called Amahi which aims to build a low maintenance system for geeks and non-geeks alike. Look at their home page [here]("http://www.amahi.org/"). 

I am nothing to do with the project myself, but I do like what they have done so far and I like where they are going with it. 

Currently Amahi is based on Fedora 9, but they would happily have it ported over to Ubuntu. However, they lack developers knowledgeable in Ubuntu. Please take a look [here]("http://wiki.amahi.org/index.php/Ubuntu") for more details. I wish I could do it myself but don't have the expertise so I thought I would post it over here with the hope of drumming up some interest in the Ubuntu community.
