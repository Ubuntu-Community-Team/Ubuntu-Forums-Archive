### Post by LepeKaname on 2009-06-17
When you edit your /etc/hosts file, it is only for your computer... In order other computers to access your server using www9, you can:

1. edit others computers hosts file (in windows is under system32/drivers/etc/).

2. Setup a DNS server and set the DNS of others computers to that one. (this is the method I use).

I'm not sure about other alternatives, I believe there is an auto-discovery function when using samba, but not sure... 

If you want to setup easily a DNS, you can use this link:
[http://www.alepe.com/about_computers/about_linux/simple_intranet_dns_server_in_linux.html](http://www.alepe.com/about_computers/about_linux/simple_intranet_dns_server_in_linux.html)

### Post by LepeKaname on 2009-06-17
If you edit the host files must work (not only in your server, but also the clients). Just write the IP and the name like:

192.168.0.2   www9

If you want to add more, add them with a space like:

192.168.0.2   www9 testhost [www.otherhost](www.otherhost) mypersonalhost.com

Something like that must work.
