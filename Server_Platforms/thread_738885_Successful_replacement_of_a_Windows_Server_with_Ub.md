### Post by tubezninja on 2008-03-31
You know, I didn't give proper kudos to ubuntu developers when I switched over, and I definitely should have.

I went from a windows/apache/php/mysql server to an Ubuntu LAMP setup in November of 07, not too far after Gutsy came out.  And it too was the smoothest experience I've ever had setting up a server.

My experience up to that point administering linux machines was in dealing with systems that had been preconfigured and didn't require much in the way of care and feeding (mostly SuSE), but the quirks that SuSE has kept me from running my own personal site off linux, and I absolutely dreaded the prospect of setting up a system from scratch.

But then, two things happened: first Windows VIsta came out, which completely soured me on Microsoft, and then I switched to Mac for my personal and work computers, which gave me the opportunity to play with a unix command line and get familiar with it.  Then, one evening, after an apache update on my Windows web server went horribly wrong (again), I decded that I'd had enough, downloaded the 7.10 server install image, and booted up.

Now, the only thing I regret was that I didn't take the plunge sooner.  It would've saved me months of aggravation. :)  And the speed and stability I've enjoyed make the experiences more than worth the 1 hour it took to get everything transitioned over (by comparison, my fatest time getting a windows web server ready to go has been 6 hours).

Of course, I've made a couple of boneheaded mistakes in the 5 months I've been running ubuntu full time, but thankfully ubuntu is actually quite forgiving, and I've been able to recover quite well each time.  These forums are also an immense help. :)

By the way: anyone who THINKS running Apache/PHP/Mysql on Windows is easier than learning linux is fooling themselves.  The aggravation just isn't worth it.
