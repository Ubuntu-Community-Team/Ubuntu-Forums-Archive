### Post by Wim Sturkenboom on 2006-12-06
I suppose you can just add the content of the hosts file to the file /etc/hosts. Use the following command on the commandline:
```
cat HOSTS >> /etc/hosts
```
Notes:
- make a backup of your original hosts file first
- you more than likely need root privileges (so use sudo)
- before an update, copy the backup back and run above command again
- you might have to modify a config file to make sure that your system first looks in the hosts file (not sure how)
