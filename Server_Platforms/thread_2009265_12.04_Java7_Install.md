Is there a way to install Java7 on Ubuntu Server 12.04? The reason I want it installed is to run a Minecraft Server with minimal requirements.

---

### Post by Cheesemill on 2012-06-24
Use the WebUpd8 Team PPA:
```
sudo apt-get install python-software-properties
sudo add-apt-repository ppa:webupd8team/java
sudo apt-get update
sudo apt-get install oracle-java7-installer
```For more information:
[http://www.webupd8.org/2012/01/install-oracle-java-jdk-7-in-ubuntu-via.html](http://www.webupd8.org/2012/01/install-oracle-java-jdk-7-in-ubuntu-via.html)

```
sudo apt-get install python-software-properties
```
