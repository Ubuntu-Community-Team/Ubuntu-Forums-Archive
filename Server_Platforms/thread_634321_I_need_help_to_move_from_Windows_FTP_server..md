### Post by umattu on 2007-12-12
I have a suggestion, but is is a bit involved.

[http://www.xoops.org/](http://www.xoops.org/)

You would need to have a LAMP server setup, ubuntu has a LAMP install option for the server release.

Xoops is simply to use, to add to the site you add modules, by dragging them into the modules directory and installing them in the xoops admin panel.

The user accounts are not OS, rather they are xoops accounts, there are also groups in xoops as well. 

Xoops gallery is an AWESOME picture module, you upload your pics to the server, it displays a thumbnail, once clicked you can view differant sizes upto the total size of the image. Not only that it has a built in shopping cart (that if you did not like, could easily be turned off) that family could add the pics they wanted, and from xoops can be sent to shutterfly, or others, and have prints delivered to their homes, or they could just DL the images themselves.

Then there is DMS Document Management Systems, this is also a great module, you load any files, pics, music, docs, etc and depending on how user rights are set, anyone can DL the files just by clicking the button.

I can say with the utmost of certainty that this will do everything you  mentioned in the initial post, and then some.
