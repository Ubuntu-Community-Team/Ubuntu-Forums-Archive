---

### Post by kalikiana on 2007-02-23
You can for one just 'startx' to run the X server, as usual.

If you want to use it as a desktop computer you should install gdm or kdm. Then you will be able to select a 'session' from a menu at the login screen, which will be kde or gnome in your case.
