### Post by tgalati4 on 2010-02-03
Servers are modular.  You start out simple, then add functions as you need them.  You will probably start with simple backups, but then you will want to host your own website.  For that you install LAMP.  Then you may want to run your own mail server--you install postfix.  Then you may want a web-based, to-do list tracker--so you install Tracks.

The list goes on.

So, start out simple and learn as you go.  The forums are always here to search.
