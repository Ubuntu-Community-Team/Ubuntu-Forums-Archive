### Post by MkDiR on 2007-11-05
It appears that you have remote desktop enabled in System->Preferences->Remote Desktop. By default it requires a user to obtain permission to view your desktop via VNC or something similar. 

I'm not on a linux machine so i cannot give you a more detailed (and i may be wrong where the remote desktop settings actually are) explanation. Your best and most secure option would to disable remote desktop when you are in a public place/wireless connection.

### Post by milton1 on 2007-11-05
I believe MkDiR is correct. You must have remote desktop enabled. Many windows machines have some form of this enabled without the same level of security found on linux machines. As such, libraries, internet cafes, etc have become hotspots for hacking people's laptops. Your choice to refuse was a good one.
