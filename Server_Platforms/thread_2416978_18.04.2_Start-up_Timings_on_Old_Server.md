### Post by TheFu on 2019-04-20
Servers shouldn't be using network-manager. They should be configured with static IPs.  Don't use DHCP to give out IPs to servers. 

No need for avahi either. That's for home networks with media servers and lazy admins.  Servers don't have wifi. No need for WPA stuff.  I don't like automatic updates on my systems, so no need for apt-daily anything.
```

$ systemd-analyze 
Startup finished in 6.300s (kernel) + 19.331s (userspace) = 25.631s
```

systemd runs lots of startup stuff in parallel, so really need to look at the critical-chain - **systemd-analyze critical-chain**
