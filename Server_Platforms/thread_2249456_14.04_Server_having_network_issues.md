### Post by SeijiSensei on 2014-10-22
"Oplocks" are a file-locking system used by Windows and emulated by Samba.  While you may have network problems, it seems more likely to be an issue with these locks.  Perhaps you should read this document: [http://www.samba.org/samba/docs/man/Samba-HOWTO-Collection/locking.html](http://www.samba.org/samba/docs/man/Samba-HOWTO-Collection/locking.html)
