### Post by zeronothing on 2008-07-25
If you plan to use BIND on ubuntu you can take a look at "gbindadmin." it is a package you can install from synaptic. It has a GUI frontend to manage DNS. or you might want to look at "webmin" which is a web based GUI frontend. to get webmin's [[COLOR="Blue"]website[/COLOR]]("www.webmin.com") and they should have a debian package or source package if you perfer it. hope this helps
