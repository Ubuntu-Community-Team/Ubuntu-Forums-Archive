### Post by HermanAB on 2009-04-29
Howdy,

Older versions of Quickbooks run on WINE.  New versions don't.  So if you can get something circa 2000, then it will likely work.  The version I use is from 1998 and it works well on WINE.

Otherwise, install Win2003 server in a VM on VMware or Virtualbox and use that.
