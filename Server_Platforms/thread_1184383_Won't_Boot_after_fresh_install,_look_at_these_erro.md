### Post by LepeKaname on 2009-06-12
Which version are you trying to install? 
What Hardware you have? 
Does the live CD works (or tried other linux versions/distributions)?

It may be possible that you have a problem with your memory card(s). If you have more than one memory card, try to use one "alone" and then the other one. Just to be sure it is not that.
What is almost for sure is a hardware problem.

You may search for the error messages like:
"Unable to handle kernel paging request Error", "Oops SMP", etc.
