### Post by Jerome36 on 2006-10-25
I didn't do a server install on my Linux Box, but I do know the server installation gives you Apache, MySQL and PHP right out of the box, so you have a LAMP server ready to go right away.  Just be aware the server install doesn't include a graphical desktop enviornment, though you can always install something like Gnome, KDE, or Xcfe afterwards.
