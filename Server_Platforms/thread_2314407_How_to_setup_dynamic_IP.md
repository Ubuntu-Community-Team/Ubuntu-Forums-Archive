### Post by SchnappiJedi on 2016-02-21
Tomato Router Firmware works great for dynamic DNS without needing to install a client on a machine in your netwrok: [http://toastmanfirmware.yolasite.com/](http://toastmanfirmware.yolasite.com/) or [http://tomato.groov.pl/](http://tomato.groov.pl/)

This site provides free dynamic DNS servers: [https://freedns.afraid.org/](https://freedns.afraid.org/)

A great UNIX dynamic DNS client if you need to install a dynamic DNS client on a machine in your network: [https://sourceforge.net/projects/inadyn-mt/](https://sourceforge.net/projects/inadyn-mt/)
