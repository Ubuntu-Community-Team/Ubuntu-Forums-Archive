### Post by LundeFul on 2006-12-06
Could any one please explain to me how I could add a new hard-drive to my system. I installed the server on a 40GB HD, but would rather widen my possibilities and install a additional 300GB HD. This drive is not partitioned in any way.

---

### Post by tturrisi on 2006-12-06
1. install the drive.
2. boot the system using the live cd.
3. manually partition the drive however you want using the partition editor.
4. give custom names to the partitions.
5. boot from the first hard disk.
6. mount the partitions.
