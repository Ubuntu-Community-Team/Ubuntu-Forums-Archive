### Post by dwanders on 2011-02-15
well this is what I did - lets hope it was correct-a-mundo:

shutdown virtual server
add second virtual drive
boot into server

use fdisk to create partitons (created /dev/sdb1)

create Physical Volume
sudo pvcreate /dev/sdb1

sudo vgdisplay (to get the name of the virtual group)
sudo vgextend vgname /dev/sdb1
sudo lvextend -L +3G /dev/vgname/root
resize2fs /dev/vgname/root

so far seems to be bigger
