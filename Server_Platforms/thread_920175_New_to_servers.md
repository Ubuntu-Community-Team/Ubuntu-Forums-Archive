Installing a gui would be a waste of resources. All server configuration is down by editing text configuration files. A better way to manage your server is to use a web based administration suite. [http://www.webmin.com/](http://www.webmin.com/)

### Post by volkswagner on 2008-09-15
As stated earlier, server hardware will depend on load.  If this is for personal use, limited traffic, and small database, than minimal hardware is needed.  Many folks here run capable servers on Pent. II and lower spec machines.  Ram is important.  I am happy with Pent II, 400mhz, and 384meg ram.

I would consider power consumption.  Pent II machines consume less than half the wattage of a Pent. IV.

My next server if I build, will be based on the Intel Atom.
