### Post by Brazen on 2007-03-12
Their is no need for Kubuntu desktop to run a server, and in fact, I would highly suggest not running any desktop at all.

You should know, though, that if you are on a home internet connection, your ISP probably blocks incoming port 80.  This means you have to host your websites on a nonstandard port, which gets kludgy.
