### Post by LHammonds on 2017-02-16
The absolute easiest mail server I got up and running has been a [NethServer]("https://www.nethserver.org/").  It runs on CentOS though.  I needed an internal mail relay for all my Windows/Ubuntu servers and this was the easiest to setup. Kind of a set-n-forget server for me since all I have it doing is mail relay but I could use it for personal email if I wanted to.

Keep in mind, it is easy from a mail novice point of view.  I'm sure if I were an expert in mail MTA and whatnot I might go for some other advanced system.

LHammonds
