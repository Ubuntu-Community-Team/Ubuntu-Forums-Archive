### Post by tgalati4 on 2013-04-18
I've used [http://sugarcrm.com](http://sugarcrm.com) to send email campaigns with gmail.  You need to schedule 50-100 per hour otherwise you will get shut down by either gmail or your ISP.  I have not sent SMS messages, but I'm sure there is a plug-in to do that.  If there is not a plug-in, then you could write a script to send SMS using the lists generated by sugarcrm.

It runs well on Ubuntu server, but there is a learning curve.

For scripting, there is* smsclient*.

You could also hook up a cell phone to your server and use any of the several gsm tools to send text messages through your phone.
