### Post by FakeOutdoorsman on 2007-12-21
Ubuntu will work just fine for what you need.  If you already have an Ubuntu Alternate disc choose "Command-line installation", which is considered a minimal installtion.  Then follow the [Apache MySQL PHP]("https://help.ubuntu.com/community/ApacheMySQLPHP") instrucitons from the Ubuntu Wiki to setup your server.

I have no experience with the [Ubuntu Server Edition]("http://www.ubuntu.com/products/WhatIsUbuntu/serveredition"), but you can look into that as well.

### Post by HermanAB on 2007-12-21
The easiest server to get to work is FTP, second comes NFS and following those two by a few miles comes Samba.

So, if you want a quick and reliable setup, use NFS.  If you have two weeks to waste, use Samba.
