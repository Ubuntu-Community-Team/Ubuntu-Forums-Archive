### Post by crouton on 2008-02-13
Rackmount cases are louder and are intended for environments with tight space requirements like datacenters; as such they aren't designed for expandability most of the time.  A full tower case might be a better fit for a standard home/office environment, and you can add things much easier (and cheaper) if necessary.  Things like standard-size power supplies, case fans, 5.25" adapter kits for 3.5" HDs, etc.
