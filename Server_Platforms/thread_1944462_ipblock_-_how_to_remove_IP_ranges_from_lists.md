Hello,

[SIZE=3][COLOR=Red]**Question**[/COLOR][/SIZE]

I use ipblock and basically I can add or remove lists like level1.php.gz or level2.php.gz

but sometimes these lists contain ranges of IP addresses I do not want to block,

The Windows version allows you to search for a specific IP address and find the domain and remove it.

There doesnt appear to be a function like that in the Linux / Ubuntu version.

**How would you modify the canned lists to remove domains or IP ranges from an ipblock list?**
(white listing one IP address is not the solution I need here, I need to white list an entire domain)

[SIZE=3]**[COLOR=SeaGreen]Answer[/COLOR]**[/SIZE] (found it)

Click on the IP address in question, then click whitelist in the pop-up that opens

Create a NEW line and add the range in the start and end boxes.  Make it permanent.
