### Post by alexlafreniere on 2009-08-27
Ubuntu, and most versions of Linux tend to work on most hardware the first time without any hiccups, especially servers. A server doesn't have any special hardware like graphics or sounds cards. Just a CPU, memory, hard drives and an amount of NIC's. So you shouldn't have a problem getting up and running on either one of them. Good luck with your shiny new Ubuntu server!
