### Post by hawkmage on 2011-02-14
If you have curl installed run the command:
```
curl -iv http://yourhostname.com
```
Where yourhostname.com is the domin name you are using in your browser.

Look for the "Content-Type" header that comes back, it should be "text/html" or at least "text/plain". but if it is an unknown MIME type you browser may not know what to do with it.
