### Post by p_quarles on 2007-07-24
Another workaround is to use NoScript for Firefox. Disables javascript on all sites by default, but you can enable it with ctrl-shift-\ or a couple mouse clicks. 

This is a good extension to have at any time, regardless of the status of newly-discovered exploits. NoScript disables javascript and XSS, and gets updated more frequently than Firefox itself.
