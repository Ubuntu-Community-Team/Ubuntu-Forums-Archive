### Post by cariboo on 2008-11-04
If you have a gui use Synaptic to install LAMP. Go to System-->Administration-->Synaptic Package Manager-->Edit--.Select Packages by Task and select lamp, or, at the command prompt type:

```
sudo tasksel
```

Select lamp.
