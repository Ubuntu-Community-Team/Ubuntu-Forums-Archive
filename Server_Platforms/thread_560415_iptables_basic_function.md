### Post by HermanAB on 2007-09-27
Note that the order of the rules in iptables are important.  They are evaluated in a top down fashion, so you need to drop incoming port 80 traffic from the WAN port before doing anything else.

You can insert rules in the beginning with -I and add them at the end with -A.
