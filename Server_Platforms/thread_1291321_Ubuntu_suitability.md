### Post by BQAggie2006 on 2009-10-14
According to its README file it recommends Red Hat EL4. I'm sure it would be possible to get Darkstar running on Ubuntu, but since you're new to Linux, it might be easier to stick with what's supported for now. There is a freely available clone of Red Hat called [CentOS (Community Enterprise OS)]("http://www.centos.org") that you can try so you wouldn't have to purchase Red Hat support. CentOS maintains compatible version numbers, so try CentOS 4.
