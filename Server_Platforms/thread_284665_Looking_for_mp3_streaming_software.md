### Post by NewbieNik on 2006-10-26
Highly recommend Jinzora2. Uses PHP, Mysql and Apache, installs with a web page which I thought was easy to follow, points to a folder of your choice, displays album covers and deals with more than just mp3s.
Adding music is a case of copying it into the right folder then telling Jizora to rescan.

[http://www.jinzora.org/pages.php?pn=downloads](http://www.jinzora.org/pages.php?pn=downloads)

had it running for 4 months now with no issue.
