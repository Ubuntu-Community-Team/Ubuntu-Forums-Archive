### Post by yabbadabbadont on 2007-02-28
Just because Mozilla isn't going to provide security fixes doesn't mean that Debian and Ubuntu won't.  In fact, I think that was one of the reasons for the whole IceWeasel uproar over at Debian.  Mozilla pretty much told them that they couldn't use the official branding if they didn't submit patches for "Official Approval".  The Debian guys then pointed out that they would still be providing security updates to older versions of Firefox long after Mozilla dropped support for them.  Short answer, I don't think that you need to worry.
