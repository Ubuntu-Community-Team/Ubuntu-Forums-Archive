### Post by noway2 on 2010-07-08
Installing LAMP would be the first step.  Are you able to access the page and get the default "it works" message?  If so, you are on your way.

Once you have that working, you should be able to start writing PHP programs and put them in a location accessible by your server.  The first thing you should try is getting the php_info() function to work.  This will verify that it is properly interpreting your scripts and serving web pages.
