### Post by GordThompson on 2013-02-27
A WordPress site is not just a bunch of files. It is a bunch of files (web scripts) that provide the basic structure AND a database that contains the actual content (stuff that you've added to the site). If you want to move a WordPress site you have to move them both.

Check out the following Go Daddy Support article for more information:

[Moving Your WordPress Site to Us from Another Host]("http://support.godaddy.com/help/article/6117/moving-your-wordpress-site-to-us-from-another-host")
