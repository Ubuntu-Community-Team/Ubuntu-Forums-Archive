### Post by LordHunter317 on 2006-05-09
It's because flushing hte rules doesn't change policy, which is set to DROP by default in this ruleset.  BTW, Ubuntu doesn't include any firewall policy by default, because it is not needed.  Anyway, issue:```
for i in INPUT FORWARD OUTPUT ; do
    sudo iptables -P $i ACCEPT
done
```And it'll work.
