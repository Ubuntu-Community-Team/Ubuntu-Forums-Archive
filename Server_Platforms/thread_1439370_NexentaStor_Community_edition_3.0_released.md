Hi All,

The NexentaStor project has released version 3.0 of the Community Edition. 

This is a distribution targeted specifically for your storage server. This project combines OpenSolaris kernel, Ubuntu Hardy userland, and an advanced and easy to use web administration interface.

Release Announcement:
[http://lists.sonic.net/pipermail/gnusol-devel/2010-March/001503.html](http://lists.sonic.net/pipermail/gnusol-devel/2010-March/001503.html)

Quick Summary of Features
-----------------------------
* ZFS additions: Deduplication (based on OpenSolaris b134).
* Free for upto 12 TB of *used* storage
* Community edition supports easy upgrades
* Many new features in the easy to use management interface.
* Integrated search

Grab the iso and the VMware releases from
[http://www.nexentastor.org/projects/site/wiki/CommunityEdition](http://www.nexentastor.org/projects/site/wiki/CommunityEdition)
