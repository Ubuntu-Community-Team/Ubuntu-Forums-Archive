### Post by srunni on 2007-08-23
You can do ```
apt-cache search foo
``` where foo is the application name to search the repositories for an application. I'm pretty sure Openbox in there. You really don't need a GUI for web server administration though, you can just SSH in and edit the configuration files, etc.
