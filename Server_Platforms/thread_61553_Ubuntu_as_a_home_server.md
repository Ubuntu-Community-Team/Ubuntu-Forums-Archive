 I have found Ubuntu easy to use, update and maintain. You should not have any problems setting up Ubuntu as a home gateway, dhcp and firewall on a DSL connection. There is a "Server"-option in the Ubuntu installation procedure, however it can't measure with the server-setup-options available in say SUSE or FC. Personally, I use Ubuntu for its easy to use interface, debian-based, the free software policy and the Ubuntu Community. ;)

Installing apache is as easy as any. The config is easy if you know what you're doing since you use apache's config files. I don't know of a GUI for configurating apache other than webmin....
