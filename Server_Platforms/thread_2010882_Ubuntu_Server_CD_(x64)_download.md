### Post by darkod on 2012-06-26
The standard Ubuntu Server image fits on a CD. You don't need a DVD.

If I am not mistaken, the DVD image is the Desktop version with many packages and languages included which don't fit on the standard Desktop CD.

But the server image is definitely a CD image, not DVD.
