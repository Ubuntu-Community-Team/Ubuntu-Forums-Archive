### Post by tiger.woods on 2007-04-29
This fixed the error being generated by proftpd:

There is a bug in proftpd currently that might cause you to get a IPv6 error when proftpd is loading:

 - IPv6 getaddrinfo 'hostname' error: Name or service not known

To correct edit your /etc/hosts file to include:

::fff:A.B.C.D hostname
