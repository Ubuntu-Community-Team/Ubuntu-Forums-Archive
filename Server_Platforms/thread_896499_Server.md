### Post by Iowan on 2008-08-21
If you download the server version (at least on Ubuntu), you can set up a LAMP server in  addition to SAMBA, SSH, and a few others. Pick the options you want and the installer builds a server for you. Be forewarned that the server version comes out-of-the-box without a desktop.  It also defaults to DHCP (if installer finds a server), but that can be fixed later.
