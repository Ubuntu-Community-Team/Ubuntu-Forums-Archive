### Post by houstonbofh on 2014-11-09
If you need a secure and private server for chat, look at Owncloud. [http://owncloud.org/](http://owncloud.org/)  You can essentially make this anything you want.

But, you may want to go decentralized and serverless, in which case RetroShare is the way to go. [http://retroshare.sourceforge.net/](http://retroshare.sourceforge.net/)  [http://en.wikipedia.org/wiki/RetroShare](http://en.wikipedia.org/wiki/RetroShare)

It is secure in ways that a server based system never can be.
