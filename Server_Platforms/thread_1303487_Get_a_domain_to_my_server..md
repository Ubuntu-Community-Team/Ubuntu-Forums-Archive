### Post by Lars Noodén on 2009-10-29
The only silly questions are the ones that don't get asked.  ;)

When you to shop for a domain name registrar, make sure to read the fine print before making an order.  You need to be able to 1) transfer the name to another registrar on demand and 2) be sole owner of the domain name (both Admin and Tech contact).  

Scams to watch out for include having the registrar be the owner of the domain name.  If that happens they can pull the rug out from under you or try to ask for a larger fee, say $500 per month, for the privilege of continuing to use the domain.  

Most are reputable, but check with the BBB first and, again, read the fine print.  There are 1000's of registrars to choose from so it's not a big deal to find one with the price and conditions you like.
