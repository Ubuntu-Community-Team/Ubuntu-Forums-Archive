### Post by BkkBonanza on 2011-03-28
There really is no way to tell just by looking at these simple items. The variations in design and technology are just too varied. In my opinion you should look at benchmarks. They're less than perfect but they at least give some relative indications for reference. 

There is a long running thread on WebHostingTalk.com (dedicated server topic) forums where people post their (standardized) benchmarks for various systems going back for years now showing the range from older hardware up to the latest 8 cpu servers (including many vps offerings too).

BTW I've never ever heard of a 3 CPU server. I'm pretty sure it's never been done due to the symmetry usually designed in. But I could be wrong.
