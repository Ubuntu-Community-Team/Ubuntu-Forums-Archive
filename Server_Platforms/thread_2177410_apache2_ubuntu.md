### Post by Lars Noodén on 2013-09-28
Hi again.  

The directive [DocumentRoot](http://httpd.apache.org/docs/2.4/mod/core.html#documentroot) tells the web server where to look for the files in question.  All the directories in the URI will be relative to the DocumentRoot.  So if you have a DocumentRoot with a value of /var/www/ then the URI [http://localhost/](http://localhost/) will look there.  The URI [http://localhost/foo/bar/](http://localhost/foo/bar/) thus will look in /var/www/foo/bar/  

The default Index file will be *index.html* and that can be changed using [DirectoryIndex](http://httpd.apache.org/docs/2.4/mod/mod_dir.html#directoryindex) but most people leave it the same.
