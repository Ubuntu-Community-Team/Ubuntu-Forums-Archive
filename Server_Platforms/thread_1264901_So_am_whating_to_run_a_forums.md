### Post by abaden on 2009-09-12
Unfortunately vBulletin, the software that powers these forums, isn't free, but there's always [phpBB]("http://www.phpbb.com/"). To run it you will need LAMP (that's Apache, PHP, and MySQL), which is pretty easy to install using apt-get. There's plenty of documentation over at the [Ubuntu Server Guide]("https://help.ubuntu.com/9.04/serverguide/C/httpd.html").

phpBB also has very good documentation, complete with screenshots at their [site]("http://www.phpbb.com/support/documentation/3.0/quickstart/quick_installation.php").

You should look into installing phpmyadmin to make managing your MySQL databases earlier. You can install it via apt-get, and a quick google should give you more info. ;)
