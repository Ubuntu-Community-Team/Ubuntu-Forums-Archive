### Post by Titan8990 on 2009-05-15
A) LTS is a more traditional linux distro. Kernels are generally patched, not upgraded until the distro moves on to a new version

B) applications are generally patched, not upgraded in LTS

C) LTS is what it sounds like... it is supported longer (3 years)

D) IMO LTS (or non-ubuntu) distro should always be used for production server platforms, if it is not mission critical, its not a big deal.
