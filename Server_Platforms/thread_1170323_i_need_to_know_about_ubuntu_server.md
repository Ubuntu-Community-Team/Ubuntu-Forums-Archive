### Post by antikristian on 2009-05-26
If you have only one harddrive, then you have to partition the harddrive so that each OS has a separate root partition. Then you can run both 9.04 server and 9.04 desktop on the same drive. Beware that you might have to manually edit the bootloader to boot both OSes.

The same thing applies with a system with two disks, allthough it's marginally easier to install since you do not have to partition the drives manually.

Just try it, and come back here if there are any problems:)
