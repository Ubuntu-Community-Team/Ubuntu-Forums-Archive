### Post by bertan2 on 2013-12-26
For Android app development, you need to learn the Java language and the Android "Activity lifecycle". Typically Android apps are developed using the Eclipse IDE, Java edition, together with the Android Development Tools plug-in for Eclipse.


Web servers are a different beast entirely, even though both are ultimately Linux. There are tons of written tutorials, plus videos on Youtube, on how to set up a basic LAMP server on Ubuntu. Web developers typically learn PHP and MySQL, though many variations on that arrangement are possible.
