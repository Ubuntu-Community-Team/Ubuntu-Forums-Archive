### Post by windependence on 2009-03-22
Sorry I was not able to get here last night. 

Some ideas.

First off, forget about accessing the server using a domain name if it isn't pointed at the correct IP address. You will either need to point the name at your server, or you can test it using just the IP address, that's fine.

You need to edit your Apache configuration file 			 /etc/apache2/apache2.conf, and add the line:

ServerName xxx.xxx.xxx.xxx    # Your server's EXTERNAL IP address

Later on, when you know it's working, you will want to change this to the fully qualified name of your server, but in this case I have a feeling this may be part of your problem. Remember to restart Apache after you change the config file.

-Tim
