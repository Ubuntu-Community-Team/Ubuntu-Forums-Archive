I am not 100% sure I understand your question but have you taken a look at putty for Linux. You can install it with ```
sudo apt-get install putty
``` or from synaptic package manager. If you do not like it you can just remove it. However it does allow you to have a list of connections. It is telnet/ssh based however it does not do remote desktop which is the part of the question that had me a little confused.
