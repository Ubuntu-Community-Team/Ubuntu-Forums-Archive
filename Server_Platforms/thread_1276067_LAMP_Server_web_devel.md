### Post by mcduck on 2009-09-26
Just install the LAMP stack directly from Ubuntu's repositories.

The easiest way is to open a terminal and run the following command:
```
sudo tasksel install lamp-server
```
(Or open Synaptic Package Manager, then select Edit->Mark Packages by Task, enable "LAMP server" and click "Apply".)

You'll find more information here: [https://help.ubuntu.com/community/ApacheMySQLPHP]("https://help.ubuntu.com/community/ApacheMySQLPHP")
