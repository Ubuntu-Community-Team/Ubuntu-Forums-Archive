### Post by Queue29 on 2010-05-10
"amd64" is the name of the instruction set used by popular 64-bit chips, such as those sold by AMD and Intel.

[http://en.wikipedia.org/wiki/X86-64](http://en.wikipedia.org/wiki/X86-64)

If you want the 32bit version of Server 10.04, you can download that too, which will also run on both AMD and Intel chips.



In short, the amd64 is the version you want, even if you have an Intel cpu.
