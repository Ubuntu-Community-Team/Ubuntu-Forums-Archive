### Post by linuxpenguin on 2006-09-15
I've got my WWW domain name registered with NameCheap.com.  Supposedly you can run 'ddclient' to have your IP registered in their DNS servers (which isn't my current setup but would be ideal).

I noticed that Ubuntu's servers have 'ddclient' and installed it with apt-get.

When I try to run 'ddclient', though, I get the following error:
```
 ddclient
WARNING:  skipping host: linuxpenguin.net: unrecognized protocol 'namecheap'
WARNING:  skipping host: linuxpenguin.net: unrecognized protocol 'namecheap'
WARNING:  unable to determine IP address

```
However, on [ddclient's website]("http://ddclient.sourceforge.net/?page=2#namecheap") it says that namecheap is a supported protocol.

Any ideas?

### Post by zilla1000 on 2008-07-18
Just got this setup on Ubuntu 8.04 with the following configuration file at /etc/ddclient.conf. It works with namecheap Dynamic DNS service.

Replace [yourdomain] with the domain name you have registered under namecheap, [yourhostname] with your host name registered under dynamic DNS in namecheap (this is not FDN - host.domain.com; just host).

# Configuration file for ddclient generated by debconf
#
# /etc/ddclient.conf

pid=/var/run/ddclient.pid
protocol=namecheap
use=if, if=eth0
server=dynamicdns.park-your-domain.com
login=yourdomain.com
password='yourpassword'
yourhostname
~
