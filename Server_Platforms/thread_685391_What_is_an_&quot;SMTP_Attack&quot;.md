### Post by p_quarles on 2008-02-02
SMTP = Simple mail transfer protocol. It's the standard protocol for sending e-mail to a wide area network. 

An "SMTP attack" could be a number of different things: unauthorized access, phishing, spamming, and so forth. Anything bad that can be done using an e-mail server.
