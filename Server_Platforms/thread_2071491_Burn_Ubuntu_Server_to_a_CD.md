### Post by darkod on 2012-10-15
For windows, you can use the free Imgburn software.
[http://www.imgburn.com/](http://www.imgburn.com/)

When you open the program, just select Burn image to CD, select the image file, the write speed (it should be 8x or 10x for better quality), and that's it.
