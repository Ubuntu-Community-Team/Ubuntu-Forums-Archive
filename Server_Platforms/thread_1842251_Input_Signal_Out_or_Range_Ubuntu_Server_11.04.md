### Post by hydruid on 2011-09-11
The problem is that ubuntu is automatically using a resolution that the monitor doesn't support....I'm guessing it's an really really old CRT.

The easiest thing to do is to grab another monitor, change the resolution to something like 800*600 or 1024*768 and then put the old monitor back.
