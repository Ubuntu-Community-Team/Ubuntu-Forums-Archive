### Post by QIII on 2017-02-27
You can find cPanel's OS requirements [here]("https://documentation.cpanel.net/display/ALD/Installation+Guide+-+System+Requirements#InstallationGuide-SystemRequirements-Operatingsystems").  Ubuntu is not supported.

As for other similar packages, I don't know.

Typically, a Linux server is administered via the command line over SSH.  In my opinion, cPanel just adds an unnecessary layer of vulnerability.

But if you need to learn how to use it, I would suggest installing CentOS in a virtual machine and trying it there.
