The error message you  posted is for your smtp server (outgoing) mail. Many mail providers do not allow you to send mail without reverse DNS set up or authentication on their servers. They do this so  that you don't relay SPAM through their mail servers. the first thing you should do is set up reverse DNS through your ISP.
