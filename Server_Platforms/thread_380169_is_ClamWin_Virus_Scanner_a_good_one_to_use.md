### Post by Mr. C. on 2007-03-10
ClamAv is an antivirus scanner, but does not detect much of the other type of malware in existence.  It wasn't designed to be used in that fashion - in fact, it was designed as an email A/V scanner.   I would not recommend it as a Windows all-encompassing scanner.
