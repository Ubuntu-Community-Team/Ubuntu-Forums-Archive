### Post by eric_stewart on 2007-05-19
I made a couple of posts of some things I've learned in the process of installing this very capable product on my site.  While I don't need another SSL gateway (I've already installed a Cisco ASA 5505 firewall with this feature) I'm always interested in seeing what the Open Source community has to offer!

Despite a few installation glitches, caused mainly be following an incomplete quick installation guide on someone's site, I was able to get SSL-Explorer up and running without too many hiccups.  Now I have a VPN portal that will allow me access to my webmail, X Server and other sites both internal and external.  I have installed a certificate on my VPN server which was generated from my own OpenSSL Certificate Authority.
