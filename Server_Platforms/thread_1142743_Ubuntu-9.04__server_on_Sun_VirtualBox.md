### Post by BkkBonanza on 2009-04-29
Shouldn't be a problem. I booted the live image and it worked fine. Create a new machine with whatever memory you think you want. You don't have to give it a hard disk but you can if you like. Mount the 9.04 iso image on the cdrom for the machine and power it up. Should start up just like if you had a cd on boot up. At least it did for me.
