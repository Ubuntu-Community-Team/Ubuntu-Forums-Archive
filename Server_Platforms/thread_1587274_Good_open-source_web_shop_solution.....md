Hi,

 Is there a good web shop solution out there, that works well with clothing (different sizes of products). Preferably something you've used extensively.

I've had to dig into the code, and repair bugs and change features to solve basic stuff. And the code it self is hard to figure out, and often repeats itself (e.g. the code that calculates an order, is in 3-4 different places). I'm sure it works fine for others, but for our purpose it's been problematic. The problem with finding a good solution, is that you don't really spot the problems with it until you've used the system in real life for a while.

- Something that works with different size and color of clothing.
- Has international support for currency, tax regulations etc.
- Support CVS import and export.
- Clean, well written code, that's easy to extend. 
- Preferably open-source, but not a must.

Any suggestions?
