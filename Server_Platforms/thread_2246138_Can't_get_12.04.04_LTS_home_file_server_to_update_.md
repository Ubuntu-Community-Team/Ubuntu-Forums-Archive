### Post by rubylaser on 2014-09-29
Unless you fileserver has a DNS service setup on it, you will want to change the nameserver portion of /etc/resolv.conf to something like this (Google's DNS servers).

```
sudo nano /etc/resolv.conf
```

```

nameserver 8.8.8.8 8.8.4.4

```

To make this survive a reboot, you will want to edit /etc/network/interfaces and add a dns-namerservers line to your interface.  Here's how an interface looks when setup with a static ip.
```
sudo nano /etc/network/interfaces
```

```

auto eth0
iface eth0 inet static
    address 192.168.1.20
    netmask 255.255.255.0
    network 192.168.1.0
    broadcast 192.168.1.255
    gateway 192.168.1.1
    dns-nameservers 8.8.8.8 8.8.4.4

```
