### Post by Titan8990 on 2008-06-08
Your havde to mount the disk before you can use it. The fdisk -l command will give you a bit more useful information than you have there. I can see that volume.is_mounted = false.

In order to mount the drive you would do something like this:

#mkdir /media/sbd1
#mount /dev/sdb1 /media/sdb1

You will then be able to write to your drive at /media/sdb1
