### Post by e79 on 2010-10-07
I did try the Community Edition and I found it great ! If offers you the ability to set up multiple domains on the same server, gives you PoP, IMAP (over SSL as well), it also included a Web Interface that IMHO is better than OWA on Exchange, you also have an embedded IM feature and it can be integrated into an existing Exchange topology. Other thing worth to mention is ability to create and share documents  etc etc These are only few advantages I can name.
 
It can be installed on a lot of different distributions (the ones I tried are Debian and Ubuntu). There is a ZIMBRA desktop client that can be used to set up multiple accounts (Exchange, Zimbra, PoP etc etc) and is as functional as Outlook (again, my own opinion) .
 
I personally found it a great replacement for Exchange. The paid version includes support and more stuff of course but overall, even the free version *can* accommodate small to medium businesses.
 
Just my $0.02
