I want users to be able to simply point to the IP address with whatever client necessary, and access the public chat room (no messing around with channels and whatnot).  If I can easily create usernames and passwords for my users, all the better.

Any ideas?  Remember that I am fine with IRC, I just need to know a basic implementation of it where I can simply edit the configuration file and be done with it.  Security is not a requirement, but if you can name me an application that can encrypt data being transmitted, it will be a plus.
