### Post by ectospasm on 2007-08-24
I'd suggest, if you don't actually have a need for a server, installing OpenSSH on your toy server and learning how to configure that.  It's probably already installed buy Ubuntu Server, and I know it's already installed with the flavors of Ubuntu Desktop.  Learn how to configure that for "remote" shell access, which should entail just connecting to the server via SSH from your workstation.  It would be good practice, and any more substantial service should require SSH access for you to configure it.  

Just my $0.02.
