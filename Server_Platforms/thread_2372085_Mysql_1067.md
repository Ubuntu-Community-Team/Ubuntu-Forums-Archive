### Post by SeijiSensei on 2017-09-21
Might I suggest an alternative to using date fields in the database?

I store dates using Unix timestamps, a 32-bit integer, and format them in the output when needed like with PHP's date function.  This method also enables you to do date arithmetic without complications. 

```

$time_now=time();
echo date("l, F d, Y",$time_now);

```

results in a string like "Thursday, September 21, 2017".
