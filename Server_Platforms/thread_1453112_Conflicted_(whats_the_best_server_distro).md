### Post by dudumomo on 2010-04-13
It is may be not in the best forum to ask for :)

But, I've been using gentoo for a couple years now and when I decided to host my server, I've firstly used gentoo (Very stable, light and fast) but so damn difficult to administrate... Even if I know Gentoo for using on my desktop and laptop, I had no idea on how to use it for a server...and well, that was tricky...
Finally, I've hesitated between Debian and Ubuntu. (Not Unbuntu ;) ). And finally I had a try with Ubuntu, and it is so damn easy to administrate and to install anything !

But Debian should be a bit more stable (Ubuntu server is quite stable actually...I didn't restart my server yet...)
but Ubuntu seems to be easier than Debian.

So...I don't think we can say that Debian or Ubuntu is the best distrib for a server. I rather think it will depends on the admin :guitar:

Some people prefer using Debian, some others Ubuntu.
In my case I don't like the Debian repository system (Basically 3 trees)
I prefer of course the Gentoo portage, but I'm fine with ubuntu so far !
