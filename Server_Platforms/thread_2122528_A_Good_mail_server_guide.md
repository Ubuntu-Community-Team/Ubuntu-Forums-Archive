### Post by Cheesemill on 2013-03-05
If you don't mind using Debian instead of Ubuntu then I've found the guide at [workaround.org]("http://workaround.org/ispmail/squeeze") to be excellent (all of my servers are VM's running Ubuntu *except* my mail server which runs Debian just so I could follow this guide).

Even if you don't end up using it it's still definitely worth a read as it explains how all the different pieces of software needed to make a mail server fit together.
I use this setup on my home mail server, and have [Roundcube]("http://roundcube.net/") running on my web-server to provide webmail access.
