### Post by aa.ivanov on 2006-08-24
According to [packages.ubuntu.com]("http://packages.ubuntu.com/cgi-bin/search_packages.pl?keywords=squirrelmail&searchon=names&subword=1&version=dapper&release=all") 1.4.6-1 is the latest version of squirrelmail currently available through the apt system. 

If you desparately want 1.4.8 you'd better download the tarball from squirrelmail's web site. Of course this means you will not be able to update your squirrelmail installation with apt-get.
