### Post by trogdor1138 on 2011-12-04
A quick Google search leads to the b4rt website: [http://tf-b4rt.berlios.de/](http://tf-b4rt.berlios.de/)

Also on that website is a specific README about the requirements and how to install: [http://tf-b4rt.berlios.de/README](http://tf-b4rt.berlios.de/README)

The documentation above also references the INSTALL file included in the archives available for download.

It seems like everything you need is available on the website; I just downloaded the recommended release and you don't need to compile anything. I would recommend following the instructions at the above link and reading the included INSTALL file.
