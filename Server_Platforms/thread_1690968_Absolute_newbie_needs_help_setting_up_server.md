### Post by Monotoko on 2011-02-19
Hello and welcome to the forums. After it is installed, you will just have a bare system console. It is now up to you to build your server as you require.

You may want to look up SSH, which will allow you have a console from any other computer that knows a user+password combination (be careful) From there, you can install various other services such as apache2 web services, FTP, IRC etc

The Ubuntu Server Guide has more information that might help you: [https://help.ubuntu.com/10.04/serverguide/C/index.html](https://help.ubuntu.com/10.04/serverguide/C/index.html)
