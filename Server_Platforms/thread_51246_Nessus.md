### Post by safecracker on 2005-07-26
Nessus is a Open Source vulnerability scanning program. It consists of a 2 part system, 1 package is the server and the other the client. It can do much more then test for vulnerabilities, it has the ability to do password auditing, patch level checking on Windows and more.

It is for use on only machines you have permission for (not always your own as you maybe doing a legal pen test). If you need help installing Nessus take a look here:
