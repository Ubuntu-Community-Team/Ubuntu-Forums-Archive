### Post by herrpoon on 2006-02-01
To your first question, yes, you can set up a domain name, and it's not even necessary to have a static IP (although it is desireable :p).  You could use [www.dyndns.org](www.dyndns.org) to register a domain name (limited choice) and use the following guide to set it up:[http://ubuntuguide.org/#assignhostnametodynamicip](http://ubuntuguide.org/#assignhostnametodynamicip).

Alternatively, there are companies, that you have to pay for mind, that will do the same sort of thing as above but with an actual proper domain name that you own, like [www.jimmey.com](www.jimmey.com).

To your second question, I suppose that depends on how confident you are in updating, securing etc. and how many people will be accessing your website.  I personally wouldn't feel confident doing something like that if it was a fairly big site with a reasonable number of visitors, which is why I have paid hosting with a company.
