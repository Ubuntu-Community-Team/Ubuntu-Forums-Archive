### Post by Thirtysixway on 2011-01-30
> **throese said:**
> Thirstysixway: But, doesn't XAMPP allow more types of server-side code/scripts to be run, such as ASP, etc etc?

I don't believe so.  By default, it's setup to have Apache, Mysql, PHP and python.  
> 
The distribution for Linux systems (tested for SuSE, RedHat, Mandrake and Debian) contains: Apache, MySQL, PHP & PEAR, Perl, ProFTPD, phpMyAdmin, OpenSSL, GD, Freetype2, libjpeg, libpng, gdbm, zlib, expat, Sablotron, libxml, Ming, Webalizer, pdf class, ncurses, mod_perl, FreeTDS, gettext, mcrypt, mhash, eAccelerator, SQLite and IMAP C-Client.

I suppose you could go ahead with it if you want to get it up and running quickly.  You'll get more or less the same thing by following a guide such as
[https://help.ubuntu.com/community/ApacheMySQLPHP](https://help.ubuntu.com/community/ApacheMySQLPHP)
