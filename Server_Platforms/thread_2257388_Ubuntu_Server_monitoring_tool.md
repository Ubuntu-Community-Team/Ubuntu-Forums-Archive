### Post by iKIT on 2014-12-19
Hey all,
I've created an Android app for server and its services monitoring. Unix Server Monitor is an app which will check out your servers state any time and any place. All you need to do is add your servers and configure settings. You will even be able to remotely start, shut down and restart the necessary services. And with online widget on the home screen of your phone you&#8217;ll be aware of how your server&#8217;s doing without even launching the app.

Lite version features:
- Not need to install additional programs on server
- Services monitoring
- Monitoring schedule control
- Notifications control
- Monitoring statistics
- Limited number of services
[https://play.google.com/store/apps/details?id=com.applurk.servermonitorlite](https://play.google.com/store/apps/details?id=com.applurk.servermonitorlite)

PRO version features:
- Online widget for tracking servers status
- Variety of themes
- Manage services (Restart, Stop, Start)
- Add unlimited number of services
[https://play.google.com/store/apps/details?id=com.applurk.servermonitor](https://play.google.com/store/apps/details?id=com.applurk.servermonitor)
