### Post by Iowan on 2008-12-31
I have a couple of the Apress Ubuntu Server Administration books by Sander van Vugt. The first (Beginning Ubuntu LTS Server Administration) is more of an introduction - introduces several concepts an administrator should know, but stops short of fully explaining them. The second (Pro Ubuntu Server Administration) is more for a professional - introduces concepts that are currently beyond me (performance monitoring, for example) - but is more How-To than the first book. I'm just beginning the chapter on Security, and Troubleshooting is still a couple of chapters away.  I think you can view the first chapter of each book at the Apress website. (Consider this a very brief book review - but not a sales pitch).

I'm also fond of online information.  If I just need help setting up an application, I don't necessarily need a book.  If I need a reference book, it probably won't be for the OS, but an application (like Samba).
