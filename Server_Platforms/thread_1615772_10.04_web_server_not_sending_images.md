### Post by wongo888 on 2010-11-07
The images folder has incorrect permissions set on it.

```
http://computercrasherz.com/images/bg.png
```

This file is returning HTTP 403 Forbidden

Try (from web root):

```
$ sudo chmod 755 images
```

Also, your logo is missing a valid src attibute:


```
<div class="header">
<a href="#"><img src="" alt="Insert Logo Here" name="Insert_logo" width="180" height="90" id="Insert_logo" style="background: transparent; display:block;" /></a>


```
