### Post by darkod on 2012-11-15
What do you mean by "starting it from /home"?

You can start the Task Selector in the command line with:
sudo tasksel

Don't forget that you make selections with the Space Bar, not Enter. After you have made the selections use Tab to highlight the OK button and then press Enter.
