### Post by TheFu on 2015-05-11
There are hundreds of different php web apps. Most will install on Ubuntu.
[https://en.wikipedia.org/wiki/List_of_free_software_web_applications](https://en.wikipedia.org/wiki/List_of_free_software_web_applications) - you'll have to look through each to see which use the same stack you have. There are many others not in the list - "gallery" and ampache are very popular.  There are many business webapps too - like CRM,  server monitoring, email, list-servers, content management, document management, server management, DB management, and the list goes on.
[http://howtoforge.com/](http://howtoforge.com/) has guides.
[https://www.digitalocean.com/features/one-click-apps/](https://www.digitalocean.com/features/one-click-apps/) - has lots of apps listed. Most will be php, but some will be perl, python, java, ... 

And don't forget - just because something is installed and works, that doesn't mean it is secure.
