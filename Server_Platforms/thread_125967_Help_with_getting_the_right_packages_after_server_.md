Well you're saying it for yourself .. you want a Ubuntu desktop install, so why don't you install a Ubuntu desktop? Why go through all this hassle with setting up X, getting all programs to work when it is MUCH easier to just install a fresh Ubuntu desktop and remove the things you don't need?
