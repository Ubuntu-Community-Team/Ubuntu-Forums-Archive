### Post by ChrisRChamberlain on 2011-02-08
Seem to have found an answer. 

The website was copied from the Windows XP server to the Ubuntu 10.04 server without any modification.

I've now run ```
sudo chmod 0777 /var/www
```
and [www.specbilda.com](www.specbilda.com) now appears to work as expected, so it seems to be a simple permissions issue.
