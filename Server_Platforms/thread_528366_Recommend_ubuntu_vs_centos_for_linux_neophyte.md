### Post by flatwombat on 2007-08-17
For a server, CentOS is easy to set up and run and is rock-solid.  It has a long life-span and is essentially RHEL without the paid support.  There is a good Forum and you can certainly search the RHEL Forum or FedoraForum.org when you need assistance.  It's certainly not a bad choice.

### Post by HermanAB on 2007-08-18
For a server, I would use Mandriva since it has waaaaay better wizards which allow you to set up a server in seconds, instead of hours, but Centos is OK once you have configured yum properly.

Ubuntu is a good desktop system and deep underneath all Linux distributions are the same, so there is no deep technical reason why you can't use Ubuntu as a server.  The only problem is the lack of server configuration wizards so you'll have to configure many things manually.
