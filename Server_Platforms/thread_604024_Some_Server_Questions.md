### Post by MJN on 2007-11-06
Hibernation is going to be no good - if you need it in the middle of the night then what you really need to be considering is how to make it quiet so when it's on it doesn't wake them up!
 
There are a million-and-one ways to reduce PC noise - get googling for such terms as "quiet pc", "pc noise", etc
 
My server is practically silent - it utilises quiet temperature-controlled fans (case and PSU), sound deadening linings, quiet hard drives and a large CPU fan with a Zalman copper heatsink. As it's only a PIII 1GHz it doesn't take all that much cooling so the cooling system runs at a very low level. Unused drives are set to spin down after a period of no use so most of the time only the main system drive is running (this runs constantly).
