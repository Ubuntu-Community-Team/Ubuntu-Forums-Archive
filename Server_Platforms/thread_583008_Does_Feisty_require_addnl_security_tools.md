### Post by dmizer on 2007-10-20
start by reading the thread at the top of this forum labeled [ Are firestarter and clamav really necessary ?? ](http://ubuntuforums.org/showthread.php?t=131616)

now on a normal desktop install, connected to the internet through a router ... no.  you do not need any additional security tools.  if you intend to do more (basically anything that allows you/someone to connect to your machine from the web), yes, you will need more security tools.
