### Post by Kirboosy on 2014-04-10
Welcome to the forums! :D

The update system in Ubuntu is incremental. So you will have to go 12.04-->12.10-->13.04-->13.10. The only exception to this rule is jumping from one LTS to another (12.04 --> 14.04). Since Ubuntu 14.04 is coming out next week why not wait until you can upgrade directly to the latest and greatest?

Hope that helps!
~Caboose
PS Make sure to backup your data before upgrading anything. Just in case something goes wrong!
