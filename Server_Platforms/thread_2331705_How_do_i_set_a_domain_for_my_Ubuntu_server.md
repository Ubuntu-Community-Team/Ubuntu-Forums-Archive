### Post by SeijiSensei on 2016-07-25
Do you mean, how do I assign the machine a name like host.example.com?  If so, take a look at "[man dnsdomainname](http://linux.die.net/man/1/dnsdomainname)".  

If you mean, how do I create a domain and assign hosts to it, you'd need to set up a DNS server.  

Rather than thrash around in the dark, why don't you tell us exactly what you're trying to do.  As it stands, your question is too vague to be answered effectively.

Note that "domains" in the *nix world have little to do with "domains" on Microsoft networks.
