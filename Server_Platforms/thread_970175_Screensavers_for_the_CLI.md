### Post by cmittle on 2008-11-05
If you have a monitor hooked up to the server I'd just press the power button when your finished.  

Better yet you can save yourself a monitor and ssh into your server for administrative things.  I have a desktop and two laptops at my house all with access to my server.  If for some reason I really need to see what comes up on my server (when I can't ssh in because of boot problems) I steal the monitor/keyboard/mouse from my desktop (~4 feet away) and hook it up.
