### Post by SeijiSensei on 2012-07-07
I use (Kubuntu) 12.04 on multiple clients, and none of them have problems connecting to an NFS server running CentOS 5.8.  So the direct answer to your question is, no, not all of us using 12.04 have these problems.  I'm running a plain vanilla setup though, with a PC as the NFS server and other PCs as the clients.  Once I install nfs-common on the clients, they connect right away.  The server's /etc/exports isn't complex either, with just no_root_squash, insecure, and async in the options for each share.
