### Post by gaten on 2007-12-10
As far as a flash alternative, what you've mentioned is the best I'm aware of. However, to increase your secrurity, you should install the NoScript firefox plugin. It's a life saver. 

Basically it forces you to whitelist what sites you want to allow javascript to run from. Also, FlashBlock could help you (you never see any Flash apps except the ones you specifically allow). 

Beyond that, perhaps an external firewall distro like Untangle, Smoothwall, IPCop or m0n0wall?


Good luck.
