### Post by ubuntu27 on 2009-04-10
If you have a 64-bit capable microprocessor, then you have to grab a CD or DVD image that says "AMD64". 

[http://releases.ubuntu.com/releases/9.04/](http://releases.ubuntu.com/releases/9.04/)

Like "Ubuntu-9.04-beta-alternate-**amd64**.iso.torrent "

or 

"ubuntu-9.04-beta-desktop-**amd64**.iso.torrent"

It does not matter that you use Intel. It says AMD64 because AMD was the first one to bring 64-bit processors for the desktop computers.

I recommend you to download the CD image using a Bitorrent client.

A Bittorrent client allows you to download faster, and it also prevents any file corruption by checking with the original.

For more information on how to download and install Ubuntu go to:
[URL="http://www.psychocats.net/ubuntu/iso"]
Downloading and burning an ISO of Ubuntu[/URL]
