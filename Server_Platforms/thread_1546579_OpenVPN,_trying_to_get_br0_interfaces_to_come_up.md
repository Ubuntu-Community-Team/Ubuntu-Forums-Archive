### Post by YesWeCan on 2010-08-05
Hi. I think you may need to use a bridge set up script as they recommend here: [http://www.openvpn.net/index.php/open-source/documentation/miscellaneous/76-ethernet-bridging.html](http://www.openvpn.net/index.php/open-source/documentation/miscellaneous/76-ethernet-bridging.html)

The bridge set up involves creating the openVPN tunnel and then bringing the interface up in "manual" mode before the two are bridged. The bridge then gets the IP assignment rather than the interface. You can put a command in 'interfaces' under the relevant interface to run the bridge script, 'up /path/bridgeStart' and you can make a bridge stop script to tear it down when the relevant interface is taken down, 'down /path/bridgeStop'
