### Post by benderisgreat on 2010-06-14
if it is a linux software raid try this:
```
mdadm --examine /dev/sda
```
copy the UUID and try to assemble it:
```
mdadm --assemble /dev/md0 --uuid=<insertuuidhere>
```
if that works you can then just mount it:
```
mount /dev/md0 /mnt
```

### Post by Joshua.martin on 2010-06-14
ISSUE SOLVED.

i dropped the live cd and went right in 9.04 server. used standard commands:

FIRST
```
mkdir home/name/usbdrive
mkdir home/name/SANdrive
```THEN
```
mount /dev/sda /home/name/usbdrive
mount /dev/sdc /home/name/SANdrive
```
