### Post by MrMinute on 2005-09-07
In theory to work on an encrypted partition or directory will be slower than on an unencrypted. I use an encrypted home directory for more than a year and got no problem or significant slowdown. I recognize a slowdown when I copy huge amounts of data only.

An encrypted file has about the same size as it would be unencrypted. I use about 4GB for my home directory to be able to do a backup on one DVD.
