### Post by adeh on 2005-12-18
Hi,

Yo are in China??

I am running an IT services company in Guangzhou, we should chat. 

Anyway, to answer your question, I have set up 3 ubuntu/debian file servers for clients.

You can run them ubuntu on very cheap hardware, as you mentioned, the most expensive (relatively) thing might be the box, if you want quiet.

Here's what I use for normal file server:

CPU: AMD Sempron 2500+
MOTHERBOARD: MSI KV8 
RAM: 256 KMX
HDD: 2 * 120GB

All this should cost about 3000RMB. This is recommended only for a files server/pdc, not a application. web or database server. Since the components are so cheap, you can go ahead and grab a nice, solid box + power supply to keep everything nice and cool and running in the corner. I am right now upgrading a box we installed in March that has been sitting in a closet the whole time, 24 hours on, most users don;t even realize it is backing up all their important docs and providing shared diskspace for group projects.

Install ubuntu server on that and you'll be a happy camper.
