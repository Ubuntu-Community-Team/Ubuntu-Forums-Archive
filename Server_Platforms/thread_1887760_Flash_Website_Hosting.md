### Post by ddarsow on 2011-11-27
sure. the page will still be a html or xhtml doc but will include some level of flash varying with the design. Not a fan of flash sites since they are extremely diffcult to promote on search engines, but nothing special is required on the server end. It is not server side scripted, just some flash content delivered like an image basically. Does require the viewer to have flash installed, but almost evryone does these days.
