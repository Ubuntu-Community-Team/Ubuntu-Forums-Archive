### Post by spynappels on 2010-03-08
It means there is a USB port inside the case, protected by the case security (lock etc).

This USB port is very useful if you want to boot from a small drive/USB key and use the HDDs for storage only, or want to boot a Virtualisation Hypervisor from a USB key and use the HDDs as storage for the VMs.
