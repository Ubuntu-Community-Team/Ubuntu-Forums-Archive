The Server Edition by default does not ship with a GUI.  

There are a number of posts within this forum which explain why that is
so (so I won't get into it here).

The short answer to your question, is no, without internet access 
configured on the machine, you cannot get the GUI.

Alternatively, if you have the desktop edition cd handy, it may be possible
to set that up in /etc/apt/sources.list and get it from there.  That is something
I have never tried so I couldn't say it would work.
