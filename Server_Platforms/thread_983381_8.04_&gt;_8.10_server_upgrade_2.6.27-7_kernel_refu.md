### Post by madtom1999 on 2008-11-16
I've had this on 3 machines I upgraded and the solution appears to be to modify /boot/grub/menu.lst 
and add for the current kernel stanza  (the line starting kernel)
rootdelay=90

This has worked for me but booting is a bit slow so I guess you can reduce the 90 until it stops working, and go back.
