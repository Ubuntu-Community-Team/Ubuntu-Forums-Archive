I am getting errors executing a command at the command line, but the errors scroll off the screen when the system prints out all the possible options for my command, so I never get to really see what my error is. 

Is there a way either to turn off the help portion when a command fails or scroll back up to view the error?

### Post by dFlyer on 2011-06-21
Start the program from the command line like so:

ProgramName >InfoProgramName

this will write all the startup information in a file called InfoProgramName
