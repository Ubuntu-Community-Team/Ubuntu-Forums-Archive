I have installed hoary a few months back but switched back to sarge because there were lots of issues in running typical server applications(it worked but needs lots of tweaking, unlike sarge that almost worked out of the box).

Lately, I toyed with vserver which I like very much and find this to be a perfect match that joins the two thing nicely together. 

For those who have problems setting server apps on ubuntu, you may give this a try.

That way, you can have the sexiest debian based desktop with Xorg server, GNOME 2.10 but at the same time running a rock solid server  with minimal performance penalty.
