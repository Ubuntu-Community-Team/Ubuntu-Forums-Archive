### Post by conradin on 2011-07-13
Ok, So I couldnt figure it out quickly, how to use drush to add a user to my drupal 7.4 server. I do have a solution, but Im going to leave the thread open just in case someone can figure it out. 
my solution to add users from my CSV database was to write a macro using imacros, and utilize the web interface. In production I may very well remove the add users function from teh main page, but for development I think its fine. 
heres my macro:
```
VERSION BUILD=3700331       
' Specify input file (if !COL variables are used, IIM automatically assume a CSV format of the input file
'CSV = Comma Separated Values in each line of the file
CMDLINE !DATASOURCE testUsers.csv
'Number of columns in the CSV file. This must be accurate!
SET !DATASOURCE_COLUMNS 2
'Start at line 1, there is not header file.
SET !LOOP 1
'Increase the current position in the file with each loop 
SET !DATASOURCE_LINE {{!LOOP}}
URL GOTO=http://127.0.0.1/?q=user/register
TAG POS=1  TYPE=INPUT:TEXT FORM=ACTION:/?q=user/register ATTR=ID:edit-name CONTENT={{!COL1}}
TAG POS=1  TYPE=INPUT:TEXT FORM=ACTION:/?q=user/register ATTR=ID:edit-mail CONTENT={{!COL2}}
TAG POS=1 TYPE=INPUT:SUBMIT FORM=ID:user-register-form ATTR=ID:edit-submit
```


This macro is lame, I know. In this case, Im porting uniqe usernames previously generated by my old server, with the corrosponding email addresses from a pre-existing database. The only goal is to add user accounts quickly.
