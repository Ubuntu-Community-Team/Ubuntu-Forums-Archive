I have used SAMBA for a long time and I have a better understanding of its capabilities. At present I hide behind a firewall so I use no security at all.

I see that SAMBA can be secured to user accounts which is relatively standard. Then LDAP can deal with the festering growth in servers.

I see no problems with running a bunch of servers, each has its unique FQDM so I can connect with a terminal to install any needed components.

Making a uniform large amount of disk space is still limited. Biggest controller I saw on eBay is 24 channels for PCI Express x8. I see it as granularity. :(
