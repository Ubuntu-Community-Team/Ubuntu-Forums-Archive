### Post by Thirtysixway on 2008-11-16
The government doesn't provide weather information...

Web apps are more complicated than getting weather data, something that holds connections to aim/msn require a good amount of bandwidth and server resources.  There is something called BlueTOC which is a way to connect to aim with php, although you're limited to the php max execution time.

I'm working on a meebo clone (although I started it in vb6 on windows so it would have to run under wine on linux) which allows people to setup their own web aim site.
[http://noobbox.com/projects/waimserver](http://noobbox.com/projects/waimserver)
