### Post by lykwydchykyn on 2008-11-27
The usual thin client setup in Ubuntu is called LTSP (Linux terminal server project).  In Ubuntu it's closely tied in with the Edubuntu project, so most of the info you need is in the Edubuntu handbook:

[http://doc.ubuntu.com/edubuntu/edubuntu/handbook/C/](http://doc.ubuntu.com/edubuntu/edubuntu/handbook/C/)

That should get you started.  You can do a lot of customizing with LTSP.
