### Post by KnottyMars on 2010-05-13
Maybe its just me, but I would be real nervous about putting up an FTP server that is wide open and writeable by anyone. 

Unless this is a home server only and you are just using it to move files back and forth on your home network. Even then I would be nervous. 

I would use a local user account to just make sure. I think you are running into permissions issues. Usually what I have done is setup new users with their account information and mod their home directory to somewhere. this way I can make the new user the owner of that folder which makes read and write issues go away. 

Im not sure how you would enable ownership or just allow permissions for the anonymous account. If you want to go the user route, just do this: 

```

sudo useradd [your username]
sudo passwd [your username] (then assign the password)
sudo chown -R [your username] [path to ftp home directory]
sudo chmod -R 775 [path to ftp home directory]
sudo usermod –d [path to ftp home directory] [username]

```

This will setup a user, assign a password, grant ownership of the folder you specify, make the folder writable, then change the home directory to the path you specify. 

Good luck, hope that helps, sorry i cant be of more help with the anonymous account.
