### Post by tomgibson on 2009-09-29
Do you mean you are trying to use a wireless card to create an access point of sorts? If you are trying to do this I would suggest that you will get better end performance by linking a box such as a media server into your network via ethernet. I have had much trouble in the past with trying to use wireless for such applications and have generally found it unable to cope with the bandwidth required for applications such as media streaming. Sorry this isn't a direct solution to the problem you posted but I think that ultimately you will get best performance without using wireless.
