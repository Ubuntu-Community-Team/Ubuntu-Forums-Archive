### Post by jazee on 2012-02-12
So I've been trying to get a static IP to work on a new VM(VirtualBox) I'm setting up. However no matter how many different ways I setup my /etc/network/interfaces files it never seems to work. Only when I revert back to dhcp does it work. 

/etc/network/interfaces
```

auto eth0
iface eth0 inet static
address 192.168.10.50
netmask 255.255.255.0
gateway 192.168.10.1

```

I've been restarting "networking" via init.d. Trying to ping 192.168.10.1 just returns Destination Host Unreachable. Ping google.com just timesout. I've also made sure network-manager was uninstalled as well since they yelled issue with others. It can't be something with my router since I have a DroboFS configured with static IP and it appears to be working fine.

What am I missing? Did something change in 11.10 that wasn't in 11.04.

---

### Post by jazee on 2012-02-12
Well I know feel like a massive idiot. After posting this it hit me. I needed to change the Network interface in VirtualBox to be Bridged. Restart the VM and bam everything is now better.
