### Post by rubylaser on 2011-12-19
Sounds like a hardware problem. If you've pulled the CMOS there's not much to do other than check the manual for the machine and try to find your error either by diagnostic beeps or status indicator lights.  Normally, disconnecting hardware to simplify the system can help (hard drives, optical drives, etc).

Finally, if the other box isn't in production, you could try to swap the PSU into this server to see if it's a power issue.
