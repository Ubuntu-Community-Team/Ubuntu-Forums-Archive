Presumably all his site needs to be able to do is to send email?
If that is the case, then you need to install postfix (if it isn't installed already)
and add a line in /etc/postfix/main.cf  with the following contents
smarthost = [111.222.333.444]
where you replace the numbers above with the actual IP address of your exchange server.
Then any email generated by the site will be relayed to the Exchange server for it to send to the intended receipients.
