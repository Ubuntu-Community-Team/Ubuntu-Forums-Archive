### Post by ajgreeny on 2010-08-08
The simple answer is, "no, and "you can't".

However, let's not be totally silly!

Nothing is backed up by default with the server version of ubuntu, and as it does not have a GUI by default, there will not even be an opportunity to set the preferences of things like gedit to backup automatically when editing files.

Tell more about what exactly you want to do and there will be plenty of advice available, I'm sure, probably involving cron and rsync, or something similar.
