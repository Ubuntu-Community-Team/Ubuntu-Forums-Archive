### Post by Wardazo on 2007-06-14
Generally the best place to look for documentation about forums, is in the forums' own web site. For example, phpbb:

[http://www.phpbb.com/support/](http://www.phpbb.com/support/)

Phpbb is a good starter choice as it's very easy to install, has lucid multimedia installation guides, and also runs on php & mysql. Many people claim that it has serious security problems, but I think a lot of this is due to phpbb users not patching quickly enough. This was certainly the case of the  infamous 2004 santy worm attack.
