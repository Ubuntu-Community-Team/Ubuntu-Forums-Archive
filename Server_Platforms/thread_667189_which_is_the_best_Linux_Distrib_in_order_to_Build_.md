### Post by HermanAB on 2008-01-14
Well, Windows 2003 of course...

First you have to realize that deep down, there is no difference between the various distributions of Linux.  The differences are purely cosmetic, so this kind of discussion is necessarily about mainly inconsequential things.

Sooooo, which distro is best depends on you.  It depends on how you wish to administer the box.  If you want to have a really hard time, then use BSD or Slackware.  If you want a few basic wizards to use when you get tired of the command line, then use Redhat, Centos or Ubuntu.  If you want a full set of wizards so that you can do everything at the click of a mouse, same as on Windows, then use Mandriva.

Cheers,

Herman
