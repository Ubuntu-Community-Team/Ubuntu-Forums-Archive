### Post by gazj on 2006-04-14
a server install, installs just the bare bones of ubuntu, there is no graphical desktop, just a command line, many server users, like this as the performance of there server is not pulled down by the demanding graphical desktop

a full install, installs the graphical desktop for normal desktop use, this desktop is  called the gnome desktop
