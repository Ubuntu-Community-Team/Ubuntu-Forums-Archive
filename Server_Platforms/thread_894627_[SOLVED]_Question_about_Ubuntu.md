### Post by Ximbiot on 2008-08-19
You don't need an OS to run Ubuntu, it **is** an OS!  If you print the ISO to CD/DVD, you can use the Wubi installer to install it *like* a program under Windows, though you'll still have to reboot to use it.  Alternately, you could also download the free VMWare Server and run it in a virtual machine.

Try reading [What is Ubuntu?]("http://www.ubuntu.com/products/whatisubuntu") for more.

Derek
