ive setup a little Server with EMBY; RSS, and other things. It has two NICs.
The goal is that the server should be reachable throuh and from the whole LAN.
But can only reach the internet throuh a safe connection NIC2.
My netplan config has both NICs setup by macaddress and called NIC1 LAN and NIC2 VPN.

can someone tell me what and where i have to sett things up so that everytime the box tries to reach outside of lan it uses NIC2?

### Post by darkod on 2018-09-15
If you set up the internet router IP as default gateway on NIC2, that all internet traffic will go there.

Your LAN internal traffic usually uses subnets that are not routable on the internet so it wouldn't try to go out NIC2 anyway. On NIC1 you just set the LAN IP and subnet mask (or CIDR) and no default gateway. That way all traffic detected for the LAN will be sent to NIC1 anyway.

You don't need to do anything special...
