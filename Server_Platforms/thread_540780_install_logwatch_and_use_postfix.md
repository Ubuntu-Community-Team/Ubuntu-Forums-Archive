### Post by Mr. C. on 2007-09-03
Postfix comes with a sendmail compatibility application.  You don't need to change the mailer variable in logwatch.conf.

The logwatch installation from tarball is pretty trivial; just use it.

Pick up the new postfix filter at:

 [http://www.mikecappella.com/logwatch](http://www.mikecappella.com/logwatch)

MrC
