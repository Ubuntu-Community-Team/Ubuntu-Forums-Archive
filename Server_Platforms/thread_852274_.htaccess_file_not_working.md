### Post by hrod beraht on 2008-07-07
You don't seem to be specifying which directory you want to protect.

Basically, */etc/apache2/httpd.conf** should read:

[INDENT][COLOR="SeaGreen"]<Directory /var/www/topsecret>
AuthType Basic
AuthName "THIS IS A SECRET SQUIRREL RESTRICTED ACCESS AREA"
AuthUserFile /etc/apache2/passwd/passwords
Require valid-user
</Directory>
[/COLOR][/INDENT]

In this example, */var/www/topsecret* is the directory you wish to protect.
*/etc/apache2/passwd/passwords* is your passwords file, which will contain passwords created using *htpasswd -c*. The password file can be most anywhere, but it's a good idea not to put it somewhere accessible from the web, e.g. /var/www/passwords
*AuthName* is what will show up in the password request box, and is just for fun :)

To restart Apache: *sudo /etc/init.d/apache2 restart*

Bob

[SIZE="1"]*In general, you should never use .htaccess files unless you don't have access to the main server configuration file.
[/SIZE]
