Well, not being an expert by any means, or an industry monitor, I would say it is just another interesting email encryption idea. I use PGP and GPG myself already, but I thought this had a nice easy install and works in the background with little to no user interaction which might make it better suited for some general users.  

PGP and GPG also require both sending and receiving parties to use it, but unlike Ciphire, they do not require (but can use) an third party server to host public keys. This reliance on the Ciphire servers themselves may not be the best way to go for some security minded. 

All -in-all I just thought some Ubuntu users might have fun trying it. I always like to "play" with new software...especially security stuff!  :D  I don't think it will revolutionize the email encryption market by any means. PGP and GPG are still the time tested standards for now IMO.
