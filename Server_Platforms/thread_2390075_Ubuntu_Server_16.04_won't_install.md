I'm trying to install Ubuntu Server on a old machine that I dug up recently. However, I'm not able to install. I downloaded the ISO and then created a bootable DVD using imgburn. The machine does boot up from the DVD and I can select the language. After that I see the screen where options to install Ubuntu Server and a few others are presented. When I select any option from here, all I see is a black screen and nothing happens after that. There is no DVD not HDD activity. Can anyone help me fix this?

Thank you.

### Post by carolog on 2018-04-25
I think I've figured it out. I needed the 32-bit install since my processor does not support 64 bit instruction set. Downloaded the 32-bit version. So far so good.
