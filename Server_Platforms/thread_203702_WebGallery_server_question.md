### Post by Maggot on 2006-06-25
Register with a free DNS host like [http://dyndns.org](http://dyndns.org) and download a client like ddclient to get your IP updated automatically.

Personally I registered my own domain and use dyndns to get my ip up to date because its dynamic.

### Post by Maggot on 2006-06-25
Yup, add Dynamic DNS host. You can make up the hostname but have to use the preset domainname.  I'm assuming you have a dynamic IP of course.

There is a program that will poll dyndns.org every 5 minutes to make sure you computer IP is right. You can manually set it or manually have it update to current IP.
