### Post by koenn on 2007-12-20
you get busybox as a shell to allow you to troubleshoot a problem, so apparently, there's a problem.

try installing from the  alternate cd -> install with text interface 
it won't fix the problem but might tell you ***where*** the installer fails, and then you can start looking for solutions.

If necessary, try the "advanced" option for more detail during the install (just accept the default values if you don't know what to answer to all the questions the installer asks
