### Post by Ramses de Norre on 2006-11-01
I configured ddclient to work with dyndns but I get errors when using it:```
ramses@icarus:~$ sudo ddclient start -v
WARNING:  skipping host: : 'login=' is an invalid login.
WARNING:  skipping host: web=dyndns: 'login=' is an invalid login.
WARNING:  skipping host: : 'login=' is an invalid login.
WARNING:  skipping host: web=dyndns: 'login=' is an invalid login.
WARNING:  unable to determine IP address

```
My /etc/ddclient.conf looks like this: (with values instead of "###" and "domain")
```
# Configuration file for ddclient generated by debconf
#
# /etc/ddclient.conf

pid=/var/run/ddclient.pid
protocol=dyndns2
use=web , web=dyndns
server=members.dyndns.org
login=###
password='###'
"domain"
```

The login is the one I use top login on dyndns.com and it works there..

### Post by Ramses de Norre on 2006-11-01
Ok, it seems like I've allready solved it, I made my config file to this and don't get no errors nomore:
```
daemon=600
mail-failure=ramses
pid=/var/run/ddclient.pid
use=web, web=checkip.dyndns.org/
#use=web , web=dyndns
login=###
password=###
protocol=dyndns2
server=members.dyndns.org
domain
```
