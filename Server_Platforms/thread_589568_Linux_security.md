### Post by az on 2007-10-24
Yes.  

Keep up-to-date with updates, don't install software from any source other than the repositories and you won't need any anti-virus software or firewall.

If you need to install stuff that is not already in the repos, obtain it from source and compile it yourself.  As a rule, binary-only software is not as secure as open-source stuff when the project has an active community.  If the project has a very small community (or none whatsoever) I would be cautious.
