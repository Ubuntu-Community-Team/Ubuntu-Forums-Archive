### Post by sikander3786 on 2010-12-21
> (pentium II, 200mb RAM)

With these specs, I don't think you would be able to boot an Ubuntu Live CD (desktop) and post the output of bootinfoscript. However, the boot script is the standard for troubleshooting problems with boot but it might not be in your case.

Try re-installing Grub from an alternate disc.

[https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows#Recovery%20Using%20the%20Ubuntu%20Alternate/Install%20CD](https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows#Recovery%20Using%20the%20Ubuntu%20Alternate/Install%20CD)

Or try the Super Grub Disc.

[https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows#Recovery%20Using%20the%20Unofficial%20%22Super%20Grub%20Disk%22](https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows#Recovery%20Using%20the%20Unofficial%20%22Super%20Grub%20Disk%22)

I think you missed the Grub install during installation somehow. Are you sure you installed Grub properly to the MBR of your HDD? You can also try a re-install of Ubuntu and see if you missed something previously.
