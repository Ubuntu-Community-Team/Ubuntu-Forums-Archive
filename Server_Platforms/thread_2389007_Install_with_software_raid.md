### Post by LHammonds on 2018-05-20
The default "live" server image does not support RAID nor LVM.  You need to download the "alternative" ISO which is just like the 16.04 installer.

There are [differences between the end-result server](https://ubuntuforums.org/showthread.php?t=2390785) depending on which server ISO image you start with (which is not clear when making your selection for which ISO to download).
