I don't quite understand your reasoning : for someone to get a login prompt, he'd need to be at the console, or connected over a network, in which case he'd probably already know the name of the system - unless he's connecting to random IP addresses, and then a DNS lookup would also reveal the  hostname.
You could always just change the hostname to something generic that doesn't reveal much about the system (eg 'computer01')

If you really want to change the login prompt : it's generated by getty (agetty on Linux/Ubuntu) and it looks like the prompt text is hard-coded in the source, so you'd have to modify that & re-compile.
```
#define LOGIN " login: "		/* login prompt */  
```
```

#ifdef __linux__
	{
		char hn[MAXHOSTNAMELEN+1];

		(void) gethostname(hn, MAXHOSTNAMELEN);
		write(1, hn, strlen(hn));
	}
#endif		
    (void) write(1, LOGIN, sizeof(LOGIN) - 1);	/* always show login prompt */
}

```
