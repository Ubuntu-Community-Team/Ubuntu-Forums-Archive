### Post by Master Chief on 2008-08-12
Content filtering can be done in several ways, and it all depends on your needs.

For example; Mozilla Firefox or SeaMonkey in combination with AdBlock can block content, but my guess would be something like OpenDNS which is free, can block websites that are considered to be inappropriate for schools or other public places like hotels or even the salvation army. Just have a look at:  [http://www.opendns.com/](http://www.opendns.com/)  and see if it fits your needs.

Web proxies, like Privoxy can block sites and certain content but requires a lot more input from you, the administrator.
