### Post by volkswagner on 2011-01-28
You can boot clonezilla via the network.  You first need to setup server edition with [DRBL]("http://drbl.sf.net/") (diskless remote boot in Linux).

I have never done this so I can't say what is the best tutorial.

All the info should be found at clonezilla.org, [how-to-setup]("http://clonezilla.org/clonezilla-SE.php#setup")

Each client would need the pxe boot option available in BIOS.
