### Post by v3xtra on 2009-05-31
Have you installed something alongs the lines of LAMP ( Linux Apache, Mysql, PHP )?  If not, that is the first step in creating a home server.  It should have given you an option to install that when you installed Ubuntu 9.04 if I remember correctly.

There are **tons** of tutorials on the internet if you server for "LAMP Server Ubuntu" or something along those lines in Google.  Here are some tutorials that I stumbled upon while searching Google:

[http://www.howtoforge.com/ubuntu_debian_lamp_server](http://www.howtoforge.com/ubuntu_debian_lamp_server)

[http://myubuntutech.blogspot.com/2009/05/installing-lamp-on-ubuntu.html](http://myubuntutech.blogspot.com/2009/05/installing-lamp-on-ubuntu.html)

Even better is the actual search:  [Google Search](http://www.google.com/search?hl=en&client=firefox-a&rls=com.ubuntu%3Aen-US%3Aunofficial&hs=ILG&q=Linux+Apache+Mysql+PHP+ubuntu&aq=f&oq=&aqi=g1)

If you ALREADY have an Apache server running and you can go to [http://localhost](http://localhost) and get a response from Apache ( I believe the default is a page that says "THIS WORKS!!!" ), then the question you may be asking yourself is where do you put the files to see them on the website, and I believe the answer is in the /var/www/ folder.

If you need help with some PHP, that is something I really enjoy doing and would be happy to try and help you if you are having trouble!  Happy coding!
