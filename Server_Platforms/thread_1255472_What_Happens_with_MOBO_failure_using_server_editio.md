### Post by trundlenut on 2009-09-01
I changed the mobo on a laptop running the desktop version and the only problem was caused by the built in NIC which became eth2 rather than eth0 (eth 1 was a wireless card which I didn't change).  Assuming you change it for the same model I don't see there as being any other problems.

With the server version you are unlikely to have any problems with either the sound or graphics as you're presumably not using them.
