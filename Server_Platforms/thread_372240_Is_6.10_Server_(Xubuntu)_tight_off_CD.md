### Post by magicfab on 2007-02-28
There have been many, many security updates since the CDs where published. it's always a good idea to update as soon as you have finished your installation.

To see what bugs and security issues are addresses by any give update, you can use the apt-listbugs package .

There are many ways to automate / delay / check updates, you may also search Debian resources for that. An example:
[http://www.debian-administration.org/articles/93](http://www.debian-administration.org/articles/93)
