### Post by wormyblackburny on 2011-02-10
Both are great languages and are used pretty often. If it were me, I would learn both, but focus on bash before you mess with perl. It is better to be "great" with one than mediocre in 2 languages. That way you don't confuse yourself in the early stages of learning. Much like regular languages, you will find that if you understand the basic construction of if-then, strings, etc... then you will find that learning the second language is the same only with different syntax.
