I have a test box running Ubuntu Desktop 7.10 as a server. Works just fine, the only downside I can see is that you are just wasting space resources by having a GUI and other pointless programs. You can install all the normal server stuff as you would with Server ED. I think they are basically exactly the same except that Server ED doesn't have a GUI and has auto install of a basic LAMP setup. You can use this [How-to]("http://howtoforge.com/perfect_server_ubuntu7.10") to install everything needed for a full fledged web/dns/mail  server
