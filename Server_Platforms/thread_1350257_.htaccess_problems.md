Hey!

I'm installing wordpress on my apache2 server and I want to use the .htaccess file to make clean urls lite [www.mydomain.com/start](www.mydomain.com/start) instead of urls like [www.mydomain.com/page.php?p=2387sd987123987c](www.mydomain.com/page.php?p=2387sd987123987c) and such.

So I've made sure that the

```
AccessFileName .htaccess
```

is uncommented in apache2.conf. And that 

```
<VirtualHost *:80>
        ServerAdmin morten@aztek.se

        DocumentRoot /export/data/www/
        <Directory />
                Options FollowSymLinks
                AllowOverride None
        </Directory>
        <Directory /export/data/www/>
                Options FollowSymLinks MultiViews
                **AllowOverride All**
                Order allow,deny
                allow from all
        </Directory>
</VirtualHost>

```

In the sites-available/default

But then I get error "500 Internal Server Error" when I browse to the page.

What might be the problem?

### Post by artheus on 2009-12-09
Hah! Figured it out!
I'm Newbie as usual...

```
a2enmod rewrite
```

was the hidden secret solution to it all.
