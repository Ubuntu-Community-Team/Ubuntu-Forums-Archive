### Post by mdlueck on 2010-09-25
I found a work around!!! :P

At the command line, with the USB HDD plugged in and powered on, issue lsusb... then wait a few seconds. Issue a second one, and the first one has made the USB HDD device show up.

This is working with the current kernel build for 10.04.

If you would be so kind as to "me too" if you can dupe this problem, please do so here:

"USB HDD and Flash Drives no longer recognized"
[https://bugs.launchpad.net/bugs/645211](https://bugs.launchpad.net/bugs/645211)

Thanks for your assistance!
