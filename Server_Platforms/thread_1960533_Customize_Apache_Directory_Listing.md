### Post by SeijiSensei on 2012-04-17
You'll need to write an HTML page yourself (called "index.html" by default) to change the background or layout.  There are ways to customize the presentation of textual directory indexes; see [http://httpd.apache.org/docs/2.2/mod/mod_autoindex.html](http://httpd.apache.org/docs/2.2/mod/mod_autoindex.html) for details.
