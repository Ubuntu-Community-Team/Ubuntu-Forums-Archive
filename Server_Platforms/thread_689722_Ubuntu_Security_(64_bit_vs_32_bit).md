### Post by p_quarles on 2008-02-06
Moved to Servers & Security.

There are currently no viruses that are able to infect a modern Linux system. This does not mean that Linux is 100% secure, but the fact remains that viruses are a problem that is unique to the way Windows is designed. 

There is no difference in security between the 64-bit and 32-bit kernels.
