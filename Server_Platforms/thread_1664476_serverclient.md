### Post by sj1410 on 2011-01-11
its Linux. If you use a server application it becomes a server. If you use a client application it becomes a client.

For example:

if you install Apache web server it becomes a web server. to use it as a web client you can use a text based browser like lynx or you can also install gnome to have firefox as the web browser. to install the X Desktop do the following.


```


sudo apt-get install gdm


```
