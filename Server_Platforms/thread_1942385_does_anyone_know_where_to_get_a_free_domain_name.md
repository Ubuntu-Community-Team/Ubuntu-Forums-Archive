### Post by d4m1r on 2012-03-17
[www.dot.tk]("http://www.dot.tk") provides free top level domains (yourname.tk) but I warn you that if you website is down or if it gets popular, they can just seize the domain as they own it.

Instead, I would recommend no-ip.org as someone previously mentioned and if you have a dynamic IP (changes often), run the no-ip.org software and it will constantly update yournamehere.no-ip.org with your current IP.

Hope this helps.
