### Post by smileypaul on 2008-04-10
Sure, if you have 2 hard drives, theres no reason you can't install Server Edition on One, and Desktop Edition on the other. Grub will even allow you to choose which you want at startup. This is also possible with one harddrive, just using multiple partitions. 

Now, if you want them both running at the same time, you'll have to looking into something like Vmware, or Qemu. 

Or just running some of the server apps on you desktop install.
