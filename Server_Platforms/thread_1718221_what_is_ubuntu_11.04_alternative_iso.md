### Post by jerome1232 on 2011-03-31
It's the same as the desktop iso but with no graphical installer. It does however include lvm and luks encryption support. You can use it as an upgrade source to upgrade to a new release of Ubuntu, and I'm sure there are other things it supports that I don't know about.
