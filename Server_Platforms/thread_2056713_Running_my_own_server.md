### Post by Lars Noodén on 2012-09-12
There are also three web (HTTP) servers to choose from: Apache, Lighttpd and nginx.  You might look around a little for those.  Apache is sort of the default, being the number one web server since it first was released back in the early days of the WWW.  nginx is on the rise for a while now and is also worth a look.

What do you want to do with your web server?  Do you want to serve static pages (with or without SSI) or do you want to try to figure out running a CMS like Plone or Drupal?
