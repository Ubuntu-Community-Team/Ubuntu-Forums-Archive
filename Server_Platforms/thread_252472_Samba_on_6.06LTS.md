### Post by sonic2000gr on 2006-10-12
You could try issuing

```
$invoke-rc.d samba start
``` 

and then check if it actually runs by
```

ps aux |grep smb 
```

You should get a listing like:

```

root   25588  0.0  0.5  9916 3072 ?        Ss   14:27   0:00 /usr/sbin/smbd -D
root   25590  0.0  0.5  9916 3056 ?        S    14:27   0:00 /usr/sbin/smbd -D 
```

Maybe invoke-rc.d runs in quiet mode?
