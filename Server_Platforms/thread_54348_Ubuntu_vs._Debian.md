Ubuntu is more up to date. This can be a pro aswell as a con. Depends on your needs. One important thing is that this means Ubuntu supports some hardware Debain does not. Correct me if I'm wrong, but I think this was, amongst other things, about RAID suport.

Debian is very stable, as in one of the most stable distro's I've ever used. But this brings up the same issue. Right now Debian has just released Sarge, which is at the moment fairly up to date. But it won't be that up to date in a few months time.

I suggest you search through the forums with Google and see the previous discussions, because they bring up some very good points of consideration.
