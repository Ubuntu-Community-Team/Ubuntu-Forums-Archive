### Post by gtdaqua on 2008-06-24
My suggestion: Learn CLI. Not only you will find it increasingly useful, but also you will find that you can administer a server without GUI- which is a very good option in case you run in to any problems. 

CLI in one way is must - not everything is possible in Webmin or eBox. You can use them for monitoring but for administering themselves, command line is best.
