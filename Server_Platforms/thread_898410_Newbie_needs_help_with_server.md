### Post by mellowd on 2008-08-23
This is what my setup is like.

I have a server on 24/7. It's pretty much my torrent/storage/web/backup box.

I'll ssh in from work or my normal pc to stick in torrents to download. Certain folders are shared with samba so I can reach it from windows. I use it to backup all my photos and music. I use it to test things for work where i do a bit of linux work.

so yes, if you want to use it to download, you can most certainly do it. I haven't shared a drive with the ps3, but I'm sure if the ps3 can read network shares it can be done easily.


what more can you do? well you can run a web server, streaming mp3 server, backup server, torrent server, virtulisation server (I've got a winxp, flux and jeos server virtualised for testing)

The list is endless really. With the server route you can make it do almost anything. The NAS is more limited than that
