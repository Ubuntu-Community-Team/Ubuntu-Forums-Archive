### Post by inclusivedisjunction on 2010-02-23
1. Mandrake is very old. I do hope you meant "Mandriva."
2. Yes. Ubuntu has two editions, one for desktops and one for servers. You can install and run server applications on the Desktop version, but it is not optimized for this purpose. The Server edition does not contain a graphical user interface to minimize resource usage.
3. Yes, Ubuntu is "good enough." Many sites run on Ubuntu servers, including Ubuntu's own website.
4. "Safe enough" depends on how vigilant the server admin is, regardless of which distro you are using. Ubuntu provides regular security updates, and the server can be configured to download and install them automatically. Obviously, if you do something dumb like using an easy to guess password on an account that can be access via SSH, it won't matter how up to date you are.
5. As stated above, many sites run on Ubuntu with high uptimes. On the server edition, many factors that could otherwise lead to a system crash, like proprietary video drivers, don't come into play. The applications you choose may have varying amounts of stability, but common ones like Apache or MySQL have been developed for over a decade and are known to be highly reliable.
