### Post by bodhi.zazen on 2011-04-10
You can, but, keep in mind Linux servers are typically command line only.

Depending on what you are using the server for, it makes little sense to dual / triple boot. An alternate might be to run the Ubuntu server in VirtualBox.

### Post by BkkBonanza on 2011-04-10
Running the server in VirtualBox is actually more useful because it will be like having another machine on your network simultaneously. So for testing this means you can still use your desktop for browsing, coding, configuring the server while the server is up and going. It's very convenient as you can just start/stop your server as you please while still using the desktop.

Actually it's a great way to run Windows too (under Ubuntu of course!).
