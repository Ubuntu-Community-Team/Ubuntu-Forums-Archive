### Post by VinceDee on 2006-03-06
I'm currently using ISPConfig. It's pretty easy to use. Creating sites is almost too easy; Resellers is easy. They use a folder system to keep everything organized. The interface is friendly, and allows you to setup DNS, FTP, email, etc easily.

I would highly recommend using something like Webmin in addition, though, because you're going to want to be able to make some other changes via browser to your system that you can't with ISPConfig (like creating MySQL databases, although I must admit I'm not even using webmin for that...I'm doing it by command line :shock: )

I haven't used VHCS (I couldn't get it installed the only time I tried. I'm sure I'll try it again someday, but I got ISPConfig working and that's the important part for a newb like me). I hear it's quite good, though.
