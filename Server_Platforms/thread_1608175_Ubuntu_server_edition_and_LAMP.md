### Post by arrrghhh on 2010-10-28
> **Jolicoeur said:**
> Are Ubuntu Server Edition and LAMP the same thing?  If not, which would be better for a beginner?  Thanks:)

For a beginner?  Ubuntu-desktop is probably the best.  I wouldn't recommend server for a beginner, as it comes without a DE (desktop environment) - as in a point-and-click GUI.  It only comes with a command-line interface which isn't so beginner-friendly.

LAMP is a stack - a bunch of tools basically built on top of Ubuntu.  Apache, MySQL & PHP (the L stands for Linux ;)).

See the official [ApacheMySQLPHP](https://help.ubuntu.com/community/ApacheMySQLPHP) doc here.

By the way, you can run ALL of the same services on a server as you can on the desktop edition.  The server edition is just purpose built for servers and mean to run lean and mean.  For beginners, I wouldn't recommend it at all since you can run all the same services from Ubuntu-Desktop.  So download a liveCD and check it out!
