### Post by brighty22 on 2011-08-09
I assume you are unfamiliar with the linux cli then. Being an admin you should probably dive straight into it. I bought [this]("http://www.amazon.co.uk/Linux-Command-Shell-Scripting-Bible/dp/1118004426/"), and can't recommend it enough. It's neither reference nor tutorial.. kind of a perfect mix between the two. I've now written loads of bash scripts that do everything from maintenance on mysql tables to backups with versioning. It's one of those top quality books that you don't come across very often - have a look at the reviews.

You're looking for a textbook on UNIX/Linux system administration.  To be quite honest I don't remember ever seeing one.  Usually books like that are offered by various software vendors like Oracle(Sun), RedHat, IBM, etc.  Most stuff I learned in my SysAdmin days was from HOWTOs, FAQs, man pages, etc.

My suggestion on learning would be think of common tasks that you perform on Windows and try to do the same on Ubuntu...
