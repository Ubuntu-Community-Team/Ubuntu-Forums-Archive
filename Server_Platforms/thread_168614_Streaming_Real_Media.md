### Post by jarrett.wold on 2006-05-01
If you want to setup your system to stream real media you'll probably need to setup Helix.
You can read more about that here:  [https://helix-server.helixcommunity.org/](https://helix-server.helixcommunity.org/)


Otherwise if you just want to stream MP3's I would suggest icecast, it is in Synaptic.  You can read more about icecast here: [http://icecast.org/](http://icecast.org/)
