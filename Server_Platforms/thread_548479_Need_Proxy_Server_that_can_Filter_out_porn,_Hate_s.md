### Post by KCPokes on 2007-09-11
I agree, Dansguardian, for an immediate solution.  

There are some "out of the box" solutions out there, as well, that include different options for proxy modifications.  I hate to mention it again, but I run Smoothwall (smoothwall.org), which has a large user base and a core group of developers building add-ons.  If you go with SWE 2.0, there is an add-on call "URL Filter" (you can find it in the forums) that allow you to block different site categories, subscribe to blacklists, etc...  It is pretty flexible and extremely powerful.
