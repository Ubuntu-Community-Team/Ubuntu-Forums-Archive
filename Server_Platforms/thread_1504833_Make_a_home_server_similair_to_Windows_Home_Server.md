> Ok, so I'm looking to make a home server to store all our music. files, etc. I like the idea of Windows Home Server. I was wondering if anyone knew how to make any version of ubuntu, kubuntu, or basically any linux into something like this. Any suggestions would be helpful thanks.

This is pretty standard fare. Ubuntu Server will work just fine here. You'll need to install samba to make it play nice with Windows (walkthroughs are everywhere). Given that you'll (I assume) be primarily be using the machine as a file server, getting Samba up and running will be your first, and frankly, only real task outside of the base install.

For the xbox360 part, you'll need a media server app (DLNA compatible). I personally use Twonky, but it's not free, so it's a less-than-popular choice. Others will chime in with more Stallman-esque alternatives, I'm sure.
