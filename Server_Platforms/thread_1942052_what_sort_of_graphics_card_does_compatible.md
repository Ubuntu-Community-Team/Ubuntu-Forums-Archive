### Post by Demented ZA on 2012-03-17
Since you are posting in the server section, and you'll be running server apps on it, I assume you'll be running terminal only. In this case, just about any card will do.

If you are still unsure, Google will help keep your server stable:
[http://www.google.co.za/search?q=ubuntu+hardware+compatibility&ie=utf-8&oe=utf-8&aq=t&rls=org.mozilla:en-US:unofficial&client=firefox-a](http://www.google.co.za/search?q=ubuntu+hardware+compatibility&ie=utf-8&oe=utf-8&aq=t&rls=org.mozilla:en-US:unofficial&client=firefox-a)

As far as stability is concerned, worry less about the graphics card, and more about what release you are going for.

I'd advise Ubuntu 10.04 (LTS) or 11.04 (LTS), as any long term release tends to be more stable and less experimental. You'll also find better support by the community, not to mention, older releases have had more teething time, and you'll find a lot more information (to search) about potential problems.
