### Post by Bugginout on 2012-10-25
I found the answer here:
[http://www.linuxforums.org/forum/servers/169215-perl-refuses-work-ubuntu-server-lamp-server.html]("http://www.linuxforums.org/forum/servers/169215-perl-refuses-work-ubuntu-server-lamp-server.html")

Here is a portion of the working Virtual Host file:
```

ScriptAlias /cgi-bin/ /usr/lib/cgi-bin/
        <Directory "/usr/lib/cgi-bin">
		AllowOverride None
		Options +ExecCGI -MultiViews +SymLinksIfOwnerMatch
		Order allow,deny
		Allow from all
		AddHandler cgi-script cgi pl
	</Directory>

# enable mod_perl
<Files ~ "\.(pl|cgi)$">
       SetHandler perl-script
       PerlResponseHandler ModPerl::Registry
       Options +ExecCGI
       PerlSendHeader On
</Files>


```
