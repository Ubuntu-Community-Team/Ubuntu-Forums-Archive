### Post by radiognome on 2011-02-09
I find the big difference between managing Windows server and managing Ubuntu to be:

[LIST=1]
[*]On Windows you use a mouse, menus and buttons. On Ubuntu you use a text-editor and configuration-files.
[*]On Windows I don't find any clear answers on the Interweb to my problems, while typing an Ubuntu question in Google mostly results in some handy tutorial. 
[/LIST]

A handy place to start for me was [HowToForge]("http://www.howtoforge.com/")

But you say you bought Ubuntu Servers for hosting your site, ask for do's and don't s and your third question is about ripping movies and seeding torrents...... :-k

Perhaps try to cancel the buy for a while and play around with Ubuntu server (on your desktop in a virtual environment) until you are comfortable with it. If you are desperately looking for RDP and such, there is a long way to go until you have your perfect Ubuntu secure multi-purpose server up and running from scratch.

Hope my thoughts gave you some guidance.
