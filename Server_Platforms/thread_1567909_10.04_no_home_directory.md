### Post by TwoEars on 2010-09-04
If you didn't have any personal files on it, you can either just add a new user, in which case I would advise you run ```
sudo mkdir /home/ && sudo chown root /home/
``` before doing so or run ```
sudo mkdir /home/$USER && sudo chown $USER /home/$USER
```
