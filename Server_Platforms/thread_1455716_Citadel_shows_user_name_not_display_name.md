When i recieve email from one of my citadel users, it is showing me thier user name (login name), not there display name.

i was able to fix this by **_un-checking_**:
Correct forged From: lines during authenticated SMTP

you can find this option by clicking the administration link > Edit site-wide configuration > Network Tab
