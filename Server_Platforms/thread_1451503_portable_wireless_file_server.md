### Post by Tylerjd on 2010-04-10
You could set up Samba on Ubuntu or CIFS on Windows. In Ubutnu right click on a folder, then goto properties, then goto the sharing tab. Its self explanatory from there.
In Windows, right click on the folder and goto sharing.
