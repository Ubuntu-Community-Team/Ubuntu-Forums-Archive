### Post by tomplast on 2006-09-25
When you are running a server with no WM you are most likely not using much of your available video ram. Now it has come to my knowledge that you can use the video ram (the amount of memory you can spare) and use it as swap.

If I'm not totally misstaken this would be alot faster then using normal swap on the HDD and give a boost to the server when dealing with much data.

This could possible also be implemented on desktops used for normal activites (writing and web browsing), there is also the possibillity of making it dynamic and smart so it know what and when to allocate memory from the video card.
