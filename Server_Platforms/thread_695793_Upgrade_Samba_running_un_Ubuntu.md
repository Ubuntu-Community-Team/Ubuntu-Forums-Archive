> **Ringworld said:**
> Is there a command line statement I can run or do I just download and install it?
No. There are no version upgrades of software in a stable release:
[https://wiki.ubuntu.com/StableReleaseUpdates](https://wiki.ubuntu.com/StableReleaseUpdates)
You could upgrade to the next stable release (7.10). That will probably give you a newer version. Or you could download the source and build it yourself, but I wouldn't reccommend it.
