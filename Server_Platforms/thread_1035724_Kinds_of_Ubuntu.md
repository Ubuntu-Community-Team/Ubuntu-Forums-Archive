Why do you ask this question in the server platforms forum? it has no real relevance here, a server normally has no desktop

All the distros you have listed differ due to the windows/desktop managers they run, i.e. Gnome, KDE, XFCE...

No one is better than the other, they serve different tastes...personal preference is all that will make the difference. You can install standard ubuntu and then install all the components from the others (kde/xfce etc) through the apt repositories to trial them. Or download all the ISO's and run them "live" to get an idea

I suggest any more info required should be sought through an alternative forum (i.e. desktops) or through some searches in google...

Hope that helps
