### Post by tgalati4 on 2011-02-22
Server monitoring tools and other admin goodies are well-developed in the Red Hat world.  Centos is the free version of Red Hat if you plan on doing self-support.

Ubuntu server has the server kernel and it runs well and has been stable for me for several years.  Some server tools (nagios, cacti, dell's OMSA, etc) have been ported from Red Hat and run OK under Ubuntu.  So you have to search a little to find the tools you need to do the job.  Also depending on the server hardware, some RAID cards and other server-specific devices may have better support (i.e. work better and faster) under Red Hat.

For new or small businesses setting up servers, Ubuntu server is a good choice and paired with Ubuntu desktop deployment makes for a consistent IT environment.
