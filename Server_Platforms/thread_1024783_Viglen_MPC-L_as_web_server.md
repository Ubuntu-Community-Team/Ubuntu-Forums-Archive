### Post by Johnsie on 2008-12-30
Yeah, should be fine. I guess the main issue with whitebox servers is having a contigency plan for hard disk failure. Evenutally all hard disks fail, sometimes without warning, so it's important to have some way of backing things up regularly.

More expensive server systems use raid mirroring, so if one disk dies you still might have the other to fall back on.
