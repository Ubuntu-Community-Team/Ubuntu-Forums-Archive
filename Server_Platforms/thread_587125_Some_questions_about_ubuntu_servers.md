### Post by HermanAB on 2007-10-22
No difference.  It simply doesn't install X and everything that goes with it.

In a data centre there are no screens or keyboards attached to the machines.  They are maintained via SSH.  So you can run X on your local machine and then do all the GUI schtuff you want as usual.
