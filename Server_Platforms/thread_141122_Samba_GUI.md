Incase you don't have a root user on your ubuntu installation then check this out [http://ubuntuforums.org/showthread.php?t=120533&highlight=Howto%3A+Webmin](http://ubuntuforums.org/showthread.php?t=120533&highlight=Howto%3A+Webmin)
