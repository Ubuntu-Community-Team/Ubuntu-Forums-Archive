I did a quick Google search (did you?) and came up w/ this blog post:

I only see one KDE specific - [KcmPureftpd]("http://freshmeat.net/projects/kcmpureftpd/"). There are a couple of Gnome frontends to Proftpd. The post also includes some browser based administration tools (like Webmin).
