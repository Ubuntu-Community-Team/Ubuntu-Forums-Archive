### Post by HermanAB on 2007-11-23
Install Citadel: [http://www.citadel.org](http://www.citadel.org)

It is the easiest system to set up, since it is self contained.  It takes about 20 minutes to install Citadel and it really 'Just Works'.  Adding Spam Assassin and ClamAV takes another 20 minutes.  Any other mail system will take you about 2 weeks to install for half the functionality.

There are a few threads here about Citadel, but you can also get an older guide here: [http://www.aeronetworks.ca/citadel-howto.html](http://www.aeronetworks.ca/citadel-howto.html)

Cheers,

Herman
