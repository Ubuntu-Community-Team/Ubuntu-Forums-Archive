### Post by terazen on 2010-04-13
Check out O'Reilly's DNS and BIND.  The first 4 chapters only take a few hours to read and that's all you need to get started (about 80 pages).  I'm pretty sure you can get to it from books.google.com, though it's worth owning.

There are some guides on the internet for setting up bind, but if you are going to take the time to read those, you may as well read it from a well written book.
