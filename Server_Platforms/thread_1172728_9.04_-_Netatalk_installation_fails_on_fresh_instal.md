### Post by spectre_25gt on 2009-06-07
For anyone having the same problem, I found a workaround at the following thread:

[https://bugs.launchpad.net/ubuntu/+source/netatalk/+bug/109917](https://bugs.launchpad.net/ubuntu/+source/netatalk/+bug/109917)


"For a quick fix, if you need only file sharing, setting ATALKD_RUN and PAPD_RUN in /etc/default/netatalk to 'no' should make it possible to finish configuring netatalk (run in terminal: dpkg --configure -a). You Mac should be able to see shares then."
