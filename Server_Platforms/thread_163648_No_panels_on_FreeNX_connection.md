Hi,

If you follow the freenx install from the wiki

[https://wiki.ubuntu.com/FreeNX?highlight=%28freenx%29](https://wiki.ubuntu.com/FreeNX?highlight=%28freenx%29)

Don't forget this one in your setup

" If you encounter problems with icons when loggin multiple times with the same user in a gnome session, run gnome-settings-daemon. Let it run at every login (gnome->preferences->sessions).  "

I had the same problems as you did ( with freenx that is :) ) and this solved it...
