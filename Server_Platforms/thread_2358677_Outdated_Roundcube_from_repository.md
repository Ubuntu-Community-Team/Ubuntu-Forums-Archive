### Post by darkod on 2017-04-16
1.2 does not sound that outdated compared to 1.2.4. The use of installing from the repo is that you will be able to automatically upgrade in the future when new version is in the repo.

If you install manually from source you will have to take care of any maintenance and upgrades manually. Unless the latest version has some features that you really need that are not present in the repo version, better to use the repo.
