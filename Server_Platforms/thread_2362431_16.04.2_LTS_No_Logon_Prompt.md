### Post by darkod on 2017-05-28
It might be that usually the server enters into "display sleep mode" rather fast. Most time servers are headless so this doesn't come up noticed. Have you tried hitting a key like the space bar? Does that show the login prompt?

That¡s what I usually do when connecting console of VMs. The prompt is not there (it's sleeping) but hitting space bar shows it. Don't forget to delete the space before continuing to type your username. The space bar will "wake it up" but at the same time it will type one space in the login prompt.
