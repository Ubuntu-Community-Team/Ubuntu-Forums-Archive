After an incident that almost cost my data on the server, I decided I need a means to backup.

here's my choice:
a) good old tape drive: cheap media, expensive drives, and that's provided you can find it
b) Blu-ray: relatively cheap media and relatively cheap drive (compar e to tape drive). Media is easy to find. Unknown compatibility with Ubuntu (or any linux for that matter)
c) Hard drive dock and hard drive: cheap "drive" (the dock), expensive "media" if going for new SATA (around $60 for 150GB to 250GB), cheap if going with Freegeek tested IDE hard drives ($20 for 80GB).
