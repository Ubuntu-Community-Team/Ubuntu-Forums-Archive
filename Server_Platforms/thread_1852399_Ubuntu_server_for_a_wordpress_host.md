### Post by Dangertux on 2011-09-30
I would say go for it. I believe there are a number of Ubuntu friendly hosting companies out there although I am guessing it would be more likely to find CentOS. 

As far as wordpress as a CMS it does fine. Just watch your widgets and plugins for security issues and stay on top of wordpress updates. Wordpress like most CMS is an easy target. 

As far as Ubuntu goes in terms of security and stability, if properly configured it is equal in respect to other server operating systems. If you are new to Linux I would definitely recommend a managed hosting solution though, a bad configuration could lead to gaping security holes.
