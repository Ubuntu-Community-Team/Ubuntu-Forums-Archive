### Post by Meatshield on 2009-12-01
If you're not tied to a specific distro, try SUSEStudio for an easy to use method of making an ISO for your install. It supports making custom changes after you compile it the first time, so you can go in and customize your disk how you want it and then make an ISO from that.

That's the easiest method I've seen.
