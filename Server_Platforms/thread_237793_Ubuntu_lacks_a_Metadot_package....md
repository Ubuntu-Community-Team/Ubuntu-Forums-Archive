### Post by Joespower on 2006-08-16
First off, Ubuntu kicks irresponsible amounts off ***!  I love this distro, so please don't assume I'm a cronic complainer...

I was just wondering if there are any plans to include a Metadot portal package for Sever LTS.  Ever since the developers included the LAMP stack, throwing together web app servers has been a breeze.  Metadot is the exception because the end user has to individually install extra perl modules and edit configuration files to get it to work (like http.conf, etc.).  However, Gentoo has a Metadot package.  All you have to do is "emerge metadot" and your done...

I think having something simular for Ubuntu Server would really make it an attractive choice for company intranets and corporate websites.  I know I would feel much better running Ubuntu in a production environment than Gentoo, and with a package like this, I could get an intranet running inside my company in an afternoon (minus some tweaking).  It doesn't even seem like it would be that much work to put this package together....

Just my 2 cents ;)
