### Post by Ric_ on 2005-03-23
Now yes I would love to see an all linux world and Ubuntu is definately the best choice of distro I have seen for a long time, and its the only desktop I use.

However I would consider if you are to go down this root creating a chroot'd enviroment for the user. Maybe even consider disabling the machines running shells. The last thing you want in a new business is some hacker kid using your connection for ill gotten gains. You could possible force firefox to loads and thats all.

Also consider lots of people like to use Internet cafes for viewing word docs etc. Openoffice is great for this but people dont like change!

Its swings and round about. Maybe consider a dual boot default linux but if all is going bad someone on site could easily change the setup.

Right suggestions for a easy firewall.

Smoothwall express 2.0 and if you check out the homebrew section there are some accounting plugins so you could set up different charging methods.

For E-mail Postfix everytime!

### Post by henla464 on 2005-03-31
I saw on sourceforge that there is a project called zeiberbude for timing management of internet cafés.

It consists of a client (exists both for windows and linux I believe) and a server for linux. The server manages the time and the client is used to look the computer so that it can not be used without paying.

Sourceforge page:
[http://sourceforge.net/projects/zeiberbude/](http://sourceforge.net/projects/zeiberbude/) 

Debian package
[http://debian.internet.gr/debian/pool/main/z/zeiberbude/](http://debian.internet.gr/debian/pool/main/z/zeiberbude/) 


I installed the server and I liked what I saw. I also installed the client on a windows computer but I was unable to control the client from the server. Probably beacause I don't really know how to start the client (which arguments to use). Please try it and if you find out how to use the client please post it here.
