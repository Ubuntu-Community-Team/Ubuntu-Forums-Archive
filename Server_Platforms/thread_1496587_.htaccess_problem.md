### Post by mondo1287 on 2010-05-29
Everything there looks fine.  I'd say you have a problem with your code.  I see you're running wordpress.  Ensure that wordpress is setup correctly to use the external hostname.  Look at the source of the main page and check the path to the css and image files.  

It could be that wordpress is linking to something like [http://192.168.10.30/style.css](http://192.168.10.30/style.css), or some other incorrect host name.  You can change that on the wordpress general settings page, under wordpress address and blog address.  This is what wordpress uses to link to your css and image files.
