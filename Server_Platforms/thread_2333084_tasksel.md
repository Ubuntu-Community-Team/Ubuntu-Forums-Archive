### Post by Vegan on 2016-08-06
I noticed with Ubuntu 16.04 the old school LAMP was not an option directly
[COLOR="#FF0000"]
sudo apt-get install tasksel[/COLOR]

will install the meta installer

then when you run

[COLOR="#FF0000"]sudo tasksel[/COLOR]

you get the old school menu where you can then install LAMP

or you can also install it with

[COLOR="#FF0000"]sudo apt-get install lamp-server[/COLOR]

another distribution is offering their package so I thought why not post it here for Ubuntu users who may want to up a server?
