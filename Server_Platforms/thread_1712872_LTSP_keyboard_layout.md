### Post by pjomara on 2011-03-23
I set up an LTSP server with 20 clients.  All work fine except the keyboard layout is set to German.  I need them set to US English.  When I log into individual users on the server, the keyboard layout is US English.  It is only when logged in on a client that the setting is German.  I've set up other LTSP servers and this was not an issue.  I don't know how to change this.  Is it a parameter in LTS.conf?

Thank you in advance for any and all help.

---

### Post by pjomara on 2011-03-24
I should have looked closer at lts.conf.  The keyboard parameter was set to German.  I changed XKBLAYOUT=de to XKBLAYOUT=enus and the problem is gone.
