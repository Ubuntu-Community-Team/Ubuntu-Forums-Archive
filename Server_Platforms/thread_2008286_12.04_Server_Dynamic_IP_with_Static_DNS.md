### Post by jasonkostempski on 2012-06-22
I need to configure an 12.04 Server to obtain a dynamic IP address via DHCP but manually specify a DNS server. I see that /etc/network/interfaces is the place to configure network settings in 12.04 and I've found how to configure static IP and dns but not dynamic IP with static DNS. I was expecting to be able to do something like:

```

# The primary network interface
auto eth0
iface eth0 inet **_dhcp-address-only_**
dns-nameservers 8.8.8.8 8.8.4.4

```

### Post by CharlesA on 2012-06-22
Worked for me:

```
charles@Tardis:~$ cat /etc/network/interfaces
# This file describes the network interfaces available on your system
# and how to activate them. For more information, see interfaces(5).

# The loopback network interface
auto lo
iface lo inet loopback

# The primary network interface
auto eth0
iface eth0 inet dhcp
dns-nameservers 8.8.8.8 8.8.4.4

```

```
charles@Tardis:~$ cat /etc/resolv.conf
# Dynamic resolv.conf(5) file for glibc resolver(3) generated by resolvconf(8)
#     DO NOT EDIT THIS FILE BY HAND -- YOUR CHANGES WILL BE OVERWRITTEN
nameserver 192.168.1.1
nameserver 8.8.8.8
nameserver 8.8.4.4
search lcl
```

Just remember to restart networking after changing the interfaces file:

sudo /etc/init.d/networking restart
