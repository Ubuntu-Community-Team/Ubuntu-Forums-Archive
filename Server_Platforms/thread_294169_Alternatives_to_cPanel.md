### Post by wpwood3 on 2006-11-06
> **ChantCd_com said:**
> 
I'd like to know why some people choose Plesk over cPanel, for example. It seems that everyone prefers cPanel.

I have used both Plesk and cPanel. Overall, I prefer cPanel because it is easier to keep my server software updated with cPanel. Plesk is very stable but a pain to work with if you like to update or install software. If you try to update PHP or MySQL or Apache in Plesk you can break Plesk.

On the other hand I have found cPanel to be less stable than Plesk. Sometimes the automatic updates in cPanel can cause problems.

If you want to set up your server and rarely change anything then go with Plesk. If you like to tweek then go with cPanel.
