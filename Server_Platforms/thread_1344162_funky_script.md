### Post by munky99999 on 2009-12-04
Wow. Totally now what I was expecting.

I notice it's running @ 1024x768 from the size of the pic. So I think the issue comes down to being graphics drivers.

Ctrl+Alt+F1 will give you a command prompt screen. The gui will essentially be gone. 

[This would be a decent place to start for that.]("https://help.ubuntu.com/community/Video")


What video card do you have? If you have some wonky one like matrox-sis-via or something. Then that page above will point you to the right thing.


An alternative to this might be. Go to System - Administration - Hardware Drivers.

See if you can activate any in there.


As a test measure. You can try to boot up a livecd and see if it worked correctly. If it did. Then perhaps it was indeed simply an installing buggerup.
