### Post by cariboo on 2011-03-29
All Ubuntu bugs should be submitted to [https://bugs.launchpad.net](https://bugs.launchpad.net), you need an account to submit them. You can't directly report bugs on the web site. the preferred method is to use ubuntu-bug. In your case, you would enter the following command from the command line, or if you use a gui, press Alt-F2 and type:

```
ubuntu-bug grub
```

For more info, have a look at man ubuntu-bug.
