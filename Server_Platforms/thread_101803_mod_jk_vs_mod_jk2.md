I've noticed that mod_jk2 is packaged for use with apache2, while mod_jk is packaged for apache 1.3.
As far as I know this is not how the mod_jk versioning is intended. In fact, mod_jk2 is now branded as 'deprecated', and the 1.2 version is the main development branch. (see [http://tomcat.apache.org/download-connectors.cgi]("http://tomcat.apache.org/download-connectors.cgi") )
Compiling it is easy enough, but I thought someone might want to change this. No idea who that would be, though, so I'm posting here:-)
