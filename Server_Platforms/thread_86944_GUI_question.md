### Post by adwait on 2005-11-06
I am not really sure what exactly the server edition contains (heck I didn't even know it existed before I checked the site today, after reading this post), anyway, you can install anything you want on the system once it is installed. The server edition will probably install all the server softwares like apache2 etc. After that if you want to add a GUI, simply run
```
sudo apt-get install gnome-desktop
```
or
```
sudo apt-get install kubuntu-desktop
```
or for xfce
```
sudo apt-get install xubuntu-desktop
```
