### Post by windependence on 2008-09-24
You can do it that way but VMware ESXi is FREE now. I am sitting in VMware training right now. AFAIK, ESXi will allow multiple snapshots, it runs on bare metal (no host OS), and it has a [COLOR="Red"]32MB[/COLOR] footprint. The ONLY way you can manage it is from another box so you don't need or want a GUI at all.
