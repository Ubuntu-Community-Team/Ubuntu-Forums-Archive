### Post by volkswagner on 2011-06-10
It may help if you describe in more detail what you want to do.

There is [MySQL Workbench]("http://dev.mysql.com/downloads/workbench/") which has windows installer available.

There is also [PHPMyAdmin]("http://www.phpmyadmin.net/home_page/index.php") which can be installed on the server and accessed with the web interface from any computer.

You can also use [Putty]("http://www.putty.org/") to access the server via ssh and administer MySQL from the command line.
