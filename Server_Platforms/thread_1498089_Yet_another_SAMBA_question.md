```
[global]

      workgroup = STJANSVILLA
      server string = MEDIACENTER
      encrypt passwords = True
      security = user
      smb passwd file = /etc/smbpasswd

      domain master = No
      local master = No
      preferred master = No
      os level = 20
      dns proxy = No
      name resolve order = lmhosts host bcast
      bind interfaces only = True
      interfaces = eth0

      debug level = 1
      create mask = 0644
      directory mask = 0755
      level2 oplocks = True
      read raw = no
      write cache size = 262144

      [Audio]

      browseable = yes
      read only = no
      invalid users = root bin daemon nobody named sys tty disk mem kmem users

      invalid users = root bin daemon nobody named sys tty disk mem kmem users

      path = /srv/Media/TVShow	
      comment = TV Series

      read only = no

      read only = no
      invalid users = root bin daemon nobody named sys tty disk mem kmem users

      comment = Software library
      browseable = yes
      read only = no
      invalid users = root bin daemon nobody named sys tty disk mem kmem users

      valid users = admin
      invalid users = root bin daemon nobody named sys tty disk mem kmem users
```

i've set the Domain master, Local master, Preferred master AND the os level to no and 20, because i have a samba PDC running allready, these settings are just there to avoid that any system other then then PDC can become a network browse master :)

It might be that im blind or something but I didnt see ANY share on your smb.conf, all the shares in share definitions are commented.

If you just want to have samba configured with no restrictions uncomment the "security=user" and change it to "security=share"

That will make samba NOT ask for username and password, anyone can see the folder. Then, to define your share you can put in your share definitions something like this:
> 
[myshare]
     comment = Files
     read only = no
     path = /path/to/folder
     create mask = 0777
     directory mask = 0777
     browsable = yes
     writable = yes

Hope it helps.
