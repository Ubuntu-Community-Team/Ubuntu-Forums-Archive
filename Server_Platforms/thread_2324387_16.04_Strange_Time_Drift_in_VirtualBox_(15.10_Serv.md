### Post by Franz_Schindler on 2016-05-14
Dear all,

I've placed a "Workaround".
Yes, to do the temporary fix I've added this Paramters to the VirtualBox Paramters of the VM.
Now i have the issue that the Time tick's like every 5 Seconds.
```

timesync-interval=1000, default is 10000timesync-min-adjust=100, same as the default
timesync-set-on-restore=1, same as the default
timesync-set-threshold=1000, default is 1200000

```
Otherwise the drift would get to high. As I've did it without this parameter and I've watched the time in parallel the drift got greater than 3 Minutes and that made me a little nervous.
The Virtual Box guy's found this in the Logfile of the Virtual Box. (Not sure what it's about)
