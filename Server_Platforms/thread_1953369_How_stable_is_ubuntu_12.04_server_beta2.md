### Post by hughr2005 on 2012-04-06
As far as I know if you run sudo apt-get update and sudo apt-get upgrade on a 12.04 beta install when the complete version is released it will update to the full released version. I haven't had a chance to test the beta yet, but I haven't heard of anybody having any problems.

### Post by Cheesemill on 2012-04-06
I've been testing 12.04 Server for a few months now and been perfectly happy with the stability.
I still wouldn't use it on a production system yet but that is only common sense, for your purposes though I see no reason not to.
If you update the beta version after the official release date then yes, it will update to the release version.
One word of warning though, avoid doing a partial upgrade:
