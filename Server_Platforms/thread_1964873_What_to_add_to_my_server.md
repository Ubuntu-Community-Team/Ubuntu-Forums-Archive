...or Drupal instead of Wordpress. Anyway you would have to start by installing the web server (i.e., Apache).

Other ideas:
[LIST]
[*]Install open ssh server so you can administrate it remotely.
[*]Create a backup system from your desktop or laptop to the server.
[*]Configure it so you can access your files from the Internet:
[LIST]
[*]Look for a DDNS provider and create an account.
[*]Install and set up ddclient.
[*]Forward the necessary ports from your router to your server.
[/LIST][/LIST]
