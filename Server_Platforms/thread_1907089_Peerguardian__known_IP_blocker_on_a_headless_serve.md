### Post by rubylaser on 2012-01-10
Peerguardian is frankly a waste of time.  Torrents provide your public IP address by design.  Anyone that connects to the swarm can see your ip as long as you're connected.  The only good way to anonymize bittorrent traffic is via a VPN proxy like [BTGuard]("http://btguard.com/").

Peerguardian basically provides a false sense of security imo.
