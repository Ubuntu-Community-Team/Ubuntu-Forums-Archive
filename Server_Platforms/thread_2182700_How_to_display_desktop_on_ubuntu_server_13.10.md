### Post by coldraven on 2013-10-22
Servers don't have a desktop for security reasons. After installation most people would even disconnect the screen and keyboard then control the server by logging in from another machine. That stops people in your house from messing with it.
You could install a desktop on it but you could equally install the server package into a desktop distribution.
