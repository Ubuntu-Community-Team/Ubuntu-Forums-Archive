### Post by Carleas on 2010-08-10
How can I redirect my URL after a site move.

I have phpBB forum software installed on a 10.04 server, and I recently moved the forums from mysite.com/forums/ to mysite.com/.

So, a thread that looked like 
mysite.com/forums/viewtopic=...
now looks like
mysite.com/viewtopic=...

Otherwise, the URLs haven't changed.  I've been trying to make the substitution using .htaccess,  but I'm not having any luck.

This is how my .htaccess file currently reads.
```
RedirectMatch 301 ^/forum/$ http://mysite.com/
```

### Post by Carleas on 2010-08-11
OK, for anyone having a similar problem, I figured this out.

The code that I needed to put in my .htaccess file was
```
Redirect permanent /forums/  http://www.mysite.com/
```

But the major problem I was having was that Apache wasn't checking the .htaccess files.  In my site configuration (/etc/apache2/sites-enabled/mysite), there was an option called AllowOverride.  It was set to 'none', so I changed it to 'all'.
