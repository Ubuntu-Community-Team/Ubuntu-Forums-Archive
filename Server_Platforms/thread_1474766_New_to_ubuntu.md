### Post by cdenley on 2010-05-06
Make sure you're actually burning the disc image, not simply placing the disc image on a data disc as a file.
[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

Then you boot to it, and follow the instructions. If you're having trouble burning and booting the install disc, you might also have trouble with the command-line-only default server configuration. You might want to start with a desktop install.
