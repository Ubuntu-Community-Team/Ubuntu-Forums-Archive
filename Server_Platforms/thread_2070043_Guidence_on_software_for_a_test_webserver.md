### Post by volkswagner on 2012-10-12
I never heard of a Pentium4 with only 440Mhz.

RAM is more important than CPU.  For getting started you will likely be fine with 256MB of RAM, but 512MB or more would be great.  This is of course provided you use the server version (CLI only, no GUI/Desktop environment).  My first test environment was Ubuntu 8.04 on a Pent2 400Mhz, with less than 400MB RAM.  I actually still use the machine as a backup fileserver (call me sentimental... it was my first PC ;)  )

There are many tutorials on the WEB.  [HowToForge]("http://www.howtoforge.com/") is a fine place to start.  I would use Ubuntu 10.04 or 12.04 if you hardware is compatible.

If you want to spend more time working on websites vs. configuring a server you could try a prebuilt image from somewhere like [TurnkeyLinux.org]("http://www.turnkeylinux.org/").  They have LAMP stack, plus other web goodies like wordpress, django, tomcat, etc.  They used to be based on Ubuntu, but now have moved to Debian.
