### Post by volkswagner on 2011-02-28
[Ampache]("http://ampache.org/"), [Jinzora]("http://en.jinzora.com/"), [SqeezeBox Server]("http://wiki.slimdevices.com/index.php/LinuxInstallationGuide") with Squeezplay, [tin can jukebox]("http://tincanjukebox.sourceforge.net/")

The list is huge really.  You may want to increase your criteria, like some have native apps for devices like Android and iPod/iPhone, or even Java.

Ampache is pretty simple, Jinzora is really cool, even does video I believe, biggest downfall is lack of support community activity/support.

Have fun!
