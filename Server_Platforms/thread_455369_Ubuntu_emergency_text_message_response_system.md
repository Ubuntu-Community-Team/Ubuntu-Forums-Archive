### Post by Syirrus on 2007-05-26
Before anything is said, I would like to thank the Ubuntu Community for all of the the time, energy and support.   It's amazing what we can get accomplish when we work together. 

I got to thinking, how cool would it be if we could set up some sort of text message "emergency" technical support system?  So, I'm looking for Ubuntu volunteers / gurus who would not mind lending a hand at  further extending our technical support capabilities in a group context. Keep in mind this group is only for "emergency" situations.  Our forums do a great job at addressing ordinary problems.  This emergency system would be used in situations like:  My Ubuntu production server is down, I have googled the error message I keep receiving and I have prayed to the penguin gods.   

I have created a preliminary text message group at [http://ubuntu.intextmessage.com](http://ubuntu.intextmessage.com).  I think it would be cool if one of the Ubuntu Forum moderators could moderated it in order to make it official and for quality control purpose. 

How it works:
Ubuntu volunteers / gurus and  Ubuntu members looking for answers join the text messaging group.
A Ubuntu user has a emergency that he or she needs instant emergency technical support.
The user sends a message from the text message group or from his or her phone requesting emergency assistance.
All group members receive this emergency message and respond to the individual or the group addressing the problem.

Concerns and considerations:
From what I gather the text message group page is all opt-in as well as the service
There are daily limits put on text messaging I believe it is 10 per group per day
No one's cell phone number is revealed, just a generic one.
You only receive text messages pertaining to the group (NO SPAM!).
At anytime you can turn off the service so that you won't receive text messages.
There is no additional cost, on the normal cost of receiving text messages.
I understand that it is possible for someone to ask for non-emergency questions. We can kick that person out the group permanent, or maybe we should make members pay to be in the group.  The proceeds could go back to the Ubuntu community or foundation perhaps?

Let me know what you think

If you are interesting in volunteering to be apart of the group, you can email @ [email]vyis@htomail.com[/email] or you can join the text message group at:

 [http://ubuntu.intextmessage.com]("http://ubuntu.intextmessage.com")
