### Post by cariboo on 2013-01-04
It looks like the UUID of /dev/sda1 has changed for some reason. you should be able to use a Live CD to chroot the partition, then check what the UUID of the partition is in /dev/disks/by-uuid. Use the following commands to chroot the partition after booting from a Live CD.

```
sudo mount /dev/sda1 /mnt
sudo mount -o bind /dev /mnt/dev
sudo mount -o bind /sys /mnt/sys
sudo mount -o bind /proc /mnt/proc
sudo chroot /mnt
```
