As a suggestion, I recommend against 18.10 as a web server.  18.10 is not a Long Term Support release and enjoys only 9 months of support.   You would need to upgrade to a new version every 9 months until the next LTS arrives -- and that is not a good practice for a web server.

Please consider using 18.04, which enjoys 5 (or perhaps even 10, if I read the announcements correctly) years of support.
