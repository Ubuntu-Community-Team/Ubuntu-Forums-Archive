> Does server platform mean the same as the 'operating system"':lolflag:

A platform consists of hardware (generally the cpu), operating system, and application software. So ...

A desktop platform will include a cpu (x86, x86_64, PPC, etc.), operating system (Windows, MAC OSX, or a linux distro), a gui (Windows, GNOME, KDE, etc.), refer to: [http://en.wikipedia.org/wiki/Desktop_Linux](http://en.wikipedia.org/wiki/Desktop_Linux)

A server platform will normally include a cpu, operating system (Windows Server or linux distro), and may include other things (RAID, etc.). In normal operation, a server may, or may not include a gui (depending on operating system and how it was installed). refer to: [http://en.wikipedia.org/wiki/Server_(computing](http://en.wikipedia.org/wiki/Server_(computing))
