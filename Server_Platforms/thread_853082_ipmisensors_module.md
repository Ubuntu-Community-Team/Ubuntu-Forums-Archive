### Post by dozycat on 2011-06-10
Long time        solution but it worked for conky:

```

$sudo modprobe coretemp
$ sensors
coretemp-isa-0000
Adapter: ISA adapter
Core 0:      +39.0°C  (high = +90.0°C, crit = +100.0°C)  

coretemp-isa-0001
Adapter: ISA adapter
Core 1:      +39.0°C  (high = +90.0°C, crit = +100.0°C)  

coretemp-isa-0002
Adapter: ISA adapter
Core 2:      +39.0°C  (high = +90.0°C, crit = +100.0°C)  

coretemp-isa-0003
Adapter: ISA adapter
Core 3:      +37.0°C  (high = +90.0°C, crit = +100.0°C)  



```

And in conky:

${hwmon 0 temp 1}

gets the temperature of the cores 0 cpu1, 1 cpu2 ,
