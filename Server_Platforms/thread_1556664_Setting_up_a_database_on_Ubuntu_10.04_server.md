### Post by lombaardcj on 2010-08-20
Here's my advice,

Look at: [https://help.ubuntu.com/10.04/serverguide/C/index.html](https://help.ubuntu.com/10.04/serverguide/C/index.html) to start off.
If you want a PDF version go to [https://help.ubuntu.com/10.04/serverguide/C/serverguide.pdf](https://help.ubuntu.com/10.04/serverguide/C/serverguide.pdf), download and enjoy!

I really learnt most of what I know about Ubuntu Server by using this guide as a reference.  Thumbs to the Ubuntu Help Community

After that, you can move forward and look at the development part of your question. 

 The setup of a database etc (MySQL, Apache Server, and PHP took me under  30minutes to perform).   This is only if you want full web deployment as an option.

Let me know if you struggle anywhere.
