### Post by volkswagner on 2011-08-18
Check out [WikiMatrix]("http://www.wikimatrix.org/") you can use their [search]("http://www.wikimatrix.org/search.php") function and add criteria.

Personally, I've been happy with [FosWiki]("http://foswiki.org/"), which is a fork of tWiki.  I would not consider the admin functions easy or intuitive, but the documentation is decent.

Since you specifically mentioned PHP, you may want to start some comparisons with PHPwiki.  You may also want to pay close attention to DokuWiki.
