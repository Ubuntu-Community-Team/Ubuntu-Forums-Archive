### Post by MJN on 2006-08-25
No need to install it from the repositories - just download it from [http://www.squirrelmail.org/](http://www.squirrelmail.org/) as it's all written in PHP4 and hence doesn't need 'installing' per se, more just dropping into one of your web directories!
 
(If you haven't already got SM on your system then you'll need to create the attachment and data directories (outside of your web tree) - see [http://www.squirrelmail.org/wiki/QuickAndDirty](http://www.squirrelmail.org/wiki/QuickAndDirty) for details)
