### Post by captain_171 on 2009-12-06
Yes no mater what ubuntu you download you can always add other packages on. Such in your case you need the package called the ubuntu-desktop wich includes Gnome, Nautilus, and other packages that are needed for a GUI interface such as you would have on the desktop CD.

Your command you will need to install the ubuntu-desktop is:
sudo apt-get install ubuntu-desktop
