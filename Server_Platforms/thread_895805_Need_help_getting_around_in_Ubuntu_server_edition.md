### Post by Ximbiot on 2008-08-20
googling "getting started on linux command line" brings up a handful of tutorials.

I might also recommend a desktop configuration for a beginner so you can access both the CLI and GUI tools until you get the hang of things.

Finally, googling specific tasks you wish to accomplish from the command line can be a good way to learn, as the HOWTOs you will often find often have command-by-command walk-throughs that you can pick up basics from as you follow them.
