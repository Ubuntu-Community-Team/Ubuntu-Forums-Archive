### Post by arrrghhh on 2010-10-27
Well you may want to tell us where you're at.  Do you have the server setup?  Is phpMyAdmin working locally?  DynDns is pretty easy to setup once you have everything else nailed down.  Once you have an account at dyndns, you install a little client on the server (I prefer ddclient, but inadyn works as well... or you can also do this on certain routers) that updates dyndns whenever the IP changes on the server.
