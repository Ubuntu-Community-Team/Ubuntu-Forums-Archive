### Post by komputes on 2009-11-20
Depends which tasks you chose to install.

Take a look at the tasks that are preset:
```
gedit /usr/share/tasksel/ubuntu-tasks.desc
```

For more info:
[https://help.ubuntu.com/community/Tasksel](https://help.ubuntu.com/community/Tasksel)


What is the size of your usb stick? What is your goal size for the OS? What utilities would you like to include on it?
