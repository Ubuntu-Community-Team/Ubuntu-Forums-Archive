### Post by volkswagner on 2015-04-11
From your smbclient machine, is /etc/resolv.conf showing the ADDC as primary DNS?

No logon server can be a DNS issue, your client may not be finding the server.

Can you ssh into the server and run
```
smbclient //localhost/netlogon -U administrator
```
