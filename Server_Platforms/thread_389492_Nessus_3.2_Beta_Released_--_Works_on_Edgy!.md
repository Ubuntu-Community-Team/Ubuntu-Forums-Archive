### Post by eric_stewart on 2007-03-20
Tenable has released a version of Nessus which installs properly in Ubuntu 6.10 Edgy Eft.  I was getting tired of using Nessus v2.2.8 with its old plug-ins.  Not to sound ungrateful, but it has been a known bug for the better part of a half year that version 3.04 of Nessus (the previous release) would not install properly on Edgy Eft but was fine on Dapper Drake (Ubuntu 6.06).  The beta version of 3.2 is called 3.1.x but will be renamed 3.2 when it comes out of the beta period.

Make sure you uninstall your previous version of Nessus first or the installation WILL fail. You can uninstall the old version of Nessus a number of ways.  I like using the Synaptic package manager.

Here's the download link:  [http://www.nessus.org/download/index.php](http://www.nessus.org/download/index.php)
