### Post by aatdark on 2009-09-17
Satega: The Open Home Server, is an open source project which aims to develop a Linux alternative to M$ Home Server. We have released our first preview just a few days ago.
But as always we are searching for developers and testers. If you are a developer and want to contribute to an amazing project please join us. (Leave us a post at our forum
[http://satega.org/forum](http://satega.org/forum)

We are coding in Python on an ubuntu 8.04 server

Download: If you just want to check out satega you can download an vmware image on our website.
[http://satega.org/index.php?q=node/6](http://satega.org/index.php?q=node/6)

You may also checkout some screenshots at our site.
