### Post by johnny.deathmatch on 2011-04-15
I'm in the same boat as you, here. I got a TonidoPlug recently, and I really love it, but after installing Transmission (the default BT client is rubbish), I've been adding more and more packages that replace the default TonidoPlug software, and I find that I don't use any of it anymore, really. I still love the hardware, but I wish there was an easy vanilla Ubuntu build. I was searching via Google for that exact thing when I found this post, actually.

The last bit of the TonidoPlug software that I found handy was the dynamic IP address auto-updater, but since that hasn't been working reliably, I've switched to noip2. Also, since I'm using Webmin as well, there really isn't any reason to run the Tonido software, just seems to be wasting CPU cycles. 

Still, I really love the hardware, and from everything I've seen, it's much better than the PogoPlug. It's great for a home server. I just don't understand why Tonido doesn't use more standard off-the-shelf software, or at least offer a distro like that as an option.

I may try to do some playing around to see if I can get a vanilla distro to run on it, or strip down the default Tonido distro to the point where it is basically vanilla Ubuntu.
