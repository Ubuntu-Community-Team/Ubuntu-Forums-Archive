### Post by koenn on 2008-06-30
In most (not all) bioses you have an option about what sort of errors should cause the system to halt. eg "Halt on all errors" will cause the pc to stop if the keyboard is missing. 
Usually you can set something like "all errors except keyboard/monitor", or some other setting that tells the bios to ignore a missing keyboard.
