### Post by TheFu on 2019-08-26
Your time zone doesn't necessarily mean that the system's TZ is set.  To set the system time zone, I think there's a console tz reconfigure tool.  I don't know the name on 18.04, sorry.  When I travel, I'll set my TZ= in my ~/.bashrc for my personal needs, but leave the system setup to my home location.

Google found this: [https://help.ubuntu.com/community/UbuntuTime](https://help.ubuntu.com/community/UbuntuTime)
```
sudo dpkg-reconfigure tzdata
```
