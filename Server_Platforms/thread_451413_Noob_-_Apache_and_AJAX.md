Javascript and AJAX do not need to be installed: you call to them from your webpage/webapp. For example, say you want to use [script.aculo.us]("http://script.aculo.us/") in your document: you add the link to the javascript in your head, and code away. 

Actually, the usage docs for script.aculo.us are pretty good, check them out here: 

[http://wiki.script.aculo.us/scriptaculous/show/Usage](http://wiki.script.aculo.us/scriptaculous/show/Usage)
