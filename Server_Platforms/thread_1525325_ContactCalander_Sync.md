### Post by arrrghhh on 2010-07-06
Google Sync is honestly the easiest to setup, but not really in the scope of these forums.

There are some like Zimbra and... OpenXchange, but I don't know how well they work, and I know they are complex to setup and manage.  Google Sync will provide basically an exchange server so all those devices can sync.  And yes, the device will save the contacts so when you're offline you can still access them.  The internet connection is only needed to update calendar/contacts.
