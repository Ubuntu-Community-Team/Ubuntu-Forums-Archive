### Post by SaintDanBert on 2010-06-26
You can always add the gnome or kde or whatever -desktop package to a "server" install.

You can also take a "desktop" install and add the various service applications (web server, ftp, email, database, etc)

Adding -desktop to the "server" will avoid loading the server with some of the eye-candy applications that are part of the "desktop" install.

### Post by Vegan on 2010-06-27
I have examined the Ubuntu desktop and its unimportant to me as I have a powerful Windows machine.

I use Ubuntu server for my web server. SAMBA provides easy access to make web development simple.

Webmin is an adequate tool to manage a server, change of pace from years of TELNET use. Only gripe is its a manual download and install. It should be on the CD.

If you have a bunch of old machines, use the server edition. Then keep some of the machines as spares in case something breaks.
