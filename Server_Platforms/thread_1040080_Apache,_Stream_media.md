### Post by R.Bucky on 2009-01-18
Unless you have specific needs, try out [Jinzora]("http://http://en.jinzora.com/") media player/streamer. It runs off port 80 and only requires MySQL. It is solid for me. I stream all day at work and you do not have to move anything into your Server Directory. Gnump3d is another streaming media option, but it installs another server. Jinzora really is great.
