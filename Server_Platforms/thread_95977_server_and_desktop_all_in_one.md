### Post by Glut on 2005-11-29
You can do all that. 
Websites are magnets for hackers, so you will make yourself a larger target. The server will also be harder to secure given the large amount of applications on it.
Performance will suffer with people using the website/ftp/vpn with you using the machine at the same time.
My opinion is that you can by a suitable machine for dirt cheap that will enable you to create a more secure server with better performance for both desktop and server.
