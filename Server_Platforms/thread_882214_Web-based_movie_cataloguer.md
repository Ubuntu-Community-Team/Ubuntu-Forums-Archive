I use "Movie Collector" from [www.collectorz.com](www.collectorz.com) and it works really well for the data download - I can use a barcode scanner to identify my DVDs and the software will download info about them off the web (and within Windows I believe that the database can be accessed from within Windows Media Center for locating items for streaming, but I have never used that).  I believe, however, that it is only available for Windows and Mac OSx -- I don't think they have a Linux version.  It works with Windows ME, 2000, XP and Vista -- if you happen to have a copy of one of those you could always create a dedicated virutal machine for the purpose but perhaps someone else knows of a native solution.
