### Post by bluefrog on 2007-09-12
If you are planning a basic web server for your students, any new machine will do.

we are not talking about a commercial web site with thousands of hits per hour so the best thing you can do is to start a normal machine.

Then if needs be, you will have time to consider buying a real small/big server and transfer the existing data to it.
