### Post by FrankAu on 2020-12-28
Hmmm - found the problem ... the ootb install for postfix enables chroot'ing ... and I guess the dovecot auth socket isn't reachable from inside the gaol.

I "freed" both smtp and submission in master.cf and voila it works!
