### Post by SeijiSensei on 2012-03-25
I just export the appropriate directories off the server with NFS or, occasionally, Samba, mount them on the clients, and play the files directly.  I've never really seen the need for some type of specialized software unless you want to use DLNA clients like a PS3.  In those cases, [ps3mediaserver]("http://code.google.com/p/ps3mediaserver/") gets high marks from its users.
