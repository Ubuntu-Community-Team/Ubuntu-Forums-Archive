### Post by LHammonds on 2020-09-18
I assume you did an in-place upgrade of Ubuntu 16.04 to 18.04 and now you are getting the high CPU usage...correct?

I do not do in-place upgrades so I have not seen this particular problem before.  But since you said it is MySQL that is angry, what version of MySQL are you running?   Was this part of a LAMP package or did you install it separately?

If I were you, I'd uninstall MySQL and install a fresh copy of MariaDB 10.5.5 which is a drop-in replacement of MySQL and enjoys more updates / improvements over MySQL.

**EDIT**: Also, if your MySQL version changed during the OS upgrade, you might need to run "mysql_upgrade" to update the schema information.
