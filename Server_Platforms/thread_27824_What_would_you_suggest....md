### Post by tonym on 2005-04-23
Firestarter seems to be the Ubuntu preferred GUI but my impression is that it only works for basic configurations.

A good halfway house is Shorewall.  There isn't a GUI - you configure it by a set of text files.  However,  it can help you understand the underlying processes,  it can cope with any configuration and there is good documentation.
