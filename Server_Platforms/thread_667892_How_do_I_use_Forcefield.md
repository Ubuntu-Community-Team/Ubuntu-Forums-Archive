### Post by bobbob1016 on 2008-01-22
You don't encrypt files and folders directly.  You basically make a file, which is encrypted, and put the files and folders in there.  You mount the file as a drive, and then have RW access to your data.

I can't help much with how to use forcefield as I don't have it installed now, but I know you do NOT want to hide the actual file (I remember forcefield can hide the file) unless you KNOW EXACTLY how to use it.  It hides the file so nothing can see it, but you still have that amount of the drive used.

I remember the tutorial for truecrypt's Windows gui helped a lot.  [http://www.truecrypt.org/docs/](http://www.truecrypt.org/docs/)

Again DO NOT HIDE THE VOLUMES unless you know what you're doing.
