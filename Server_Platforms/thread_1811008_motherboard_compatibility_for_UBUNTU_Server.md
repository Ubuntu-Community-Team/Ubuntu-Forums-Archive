### Post by Gyokuro on 2011-07-24
Hi,

your motherboard is fine and supported - take a look here: [http://www.intel.com/support/motherboards/desktop/dg41wv/sb/CS-031710.htm](http://www.intel.com/support/motherboards/desktop/dg41wv/sb/CS-031710.htm)

I never had problems with Intel products with a Linux distributions like RHEL, Debian/Ubuntu. However for a server I would recommend to go with an Ubuntu LTS release (Lucid lynx 10.04.3) not with the latest one.
