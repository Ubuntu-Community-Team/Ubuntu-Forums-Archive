Yes, this is possible.

You've two options.  Either set the Wireless card on both machines to "Ad-Hoc" mode, or get a WNIC for the desktop that linux can run as an access point, in infrastructre mode.

Then you'd need to configure your desktop as a router.  If all you want to do from the laptp is surf the net, your best bet would be to install a proxy server.

All of the above is quite involved and beyond my scope in a forum.  I'm certain there are loads of HowTo's out there.
