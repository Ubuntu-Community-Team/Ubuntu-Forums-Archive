### Post by 5-HT on 2005-12-19
Hi, just wondering are you saving those files in gedit?

If so, its default behaviour is to automatically make a backup of a file when you save it (it's a great feature if you make a mistake...but ya, if you don't want it to then it can become a nuissance).


To turn this feature off:

 open up gedit > edit > preferences > editor > and uncheck "Create a backup copy of files before saving" 

If you're using another program that's doing this, sorry I won't be of much help as I've only encountered gedit doing such, but there should be similar options in the programs preferences.
