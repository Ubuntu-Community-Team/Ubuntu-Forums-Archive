### Post by SeijiSensei on 2019-05-22
I don't know why that page constitutes a solution to your question.  

The best method for migrating a WordPress site is to use one of the migration plugins.  

[https://www.wpsuperstars.net/top-wordpress-migration-plugins/](https://www.wpsuperstars.net/top-wordpress-migration-plugins/)

I used the free version of Duplicator when I migrated a site a year ago or so.

[https://wordpress.org/plugins/duplicator/](https://wordpress.org/plugins/duplicator/)

> Migrate WordPress and Run WordPress Backups

Duplicator creates a package that bundles all the site&#8217;s plugins, themes, content, database and WordPress files into a simple zip file called a package. This package can then be used to easily migrate a WordPress site to any location you wish. Move on the same server, across servers and pretty much any location a WordPress site can be hosted. WordPress is not required for installation since the package contains all site files.
