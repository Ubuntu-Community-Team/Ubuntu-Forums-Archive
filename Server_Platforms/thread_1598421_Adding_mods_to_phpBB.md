### Post by James78 on 2010-10-16
Most phpBB mods come with an XML file (in MODX format), which you open in your browser. It shows you all the changes you need to do by hand (read them carefully or you'll make a mistake!)

If you can't be bothered to do it by hand, use the phpBB [AutoMOD]("http://www.phpbb.com/mods/automod/") which will do it for you, but if the changes aren't too many, you really should do it by hand. It creates experience + you know exactly what you modified; who knows what AutoMOD did, EVEN if it logs it. However, it's PERFECT for huge mods that there's no way you'll do all those changes by yourself (trust me, I've done it before, it was torture)

There's plenty of information on phpBB.com if you take the time to do a search. [Adding mods to phpBB tutorial]("http://www.phpbb.com/community/viewtopic.php?t=61611"). [Finding phpBB mods]("http://www.phpbb.com/mods/finding/"). [Opening mod XML files]("http://www.phpbb.com/mods/opening/"). [Installing mods]("http://www.phpbb.com/mods/installing/"). [Main documentation page]("http://www.phpbb.com/mods/?from=submenu").
