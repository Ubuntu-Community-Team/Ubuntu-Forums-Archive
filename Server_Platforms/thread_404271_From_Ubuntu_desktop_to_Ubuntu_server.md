### Post by nix4me on 2007-04-08
Ubuntu server is the same as Ubuntu desktop except the server doesn't have the desktop gui.  The server is command line only by default.

All server packages are the same packages that would be in desktop.

I am not aware of a comparison document.

Basically, you could install ubuntu desktop and delete gnome and xorg and end up with a server install.
