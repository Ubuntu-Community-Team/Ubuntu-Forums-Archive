### Post by Iowan on 2006-06-12
I "borrowed" this from another thread:> This is the dhcpd.conf file

# DHCP configuration generated by Firestarter
ddns-update-style interim;
ignore client-updates;

subnet 192.168.0.0 netmask 255.255.255.0 {
option routers 192.168.0.1;
option subnet-mask 255.255.255.0;
option domain-name-servers my.isp.dns.01 my.isp.dns.02;
option ip-forwarding on;
range dynamic-bootp 192.168.0.2 192.168.0.254;
default-lease-time 21600;
max-lease-time 43200;
}
 If the router is 192.168.2.1, I presume you'd want more like:

subnet 192.168.2.0 netmask 255.255.255.0 {
option routers 192.168.2.1;
option broadcast-address 192.168.2.255;
option subnet-mask 255.255.255.0;
option domain-name-servers my.isp.dns.01 my.isp.dns.02;
option domain-name "mydomain";
range  192.168.2.2 192.168.2.20;
default-lease-time 21600;
max-lease-time 43200;
}

There's  a **man dhcpd.conf** page that gives a few more options (2000+ lines of them).
