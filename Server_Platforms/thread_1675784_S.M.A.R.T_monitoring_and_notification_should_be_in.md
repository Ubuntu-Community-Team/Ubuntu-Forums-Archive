### Post by disabledaccount on 2011-01-26
Unfortunatelly meaning of most SMART values depends on manufacturer and firmware version. Currently there is no software that can properly detect problems. Besides, not every failure can be detected by SMART and not every SMART alert is really important. 

Just for example: C7 "UDMA CRC error rate" never fails in SMART report, because this is not HDD failure (usually), but such errors causes system to hang/crash or to drop hdd from array.
