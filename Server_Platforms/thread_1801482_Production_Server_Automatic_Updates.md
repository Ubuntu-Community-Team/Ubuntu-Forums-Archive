### Post by kevin11951 on 2011-07-10
I am setting up a Zimbra email server based on Ubuntu 10.04 Server Edition for my home based business.

For a production server that needs as close to 100% uptime as possible, is it better to enable all automatic updates, or only security updates.

---

### Post by karlson on 2011-07-10
In general production servers should have no automatic updates at all.  You should review updates before applying them.  And while I have not seen updates retracted for Ubuntu (or just haven't looked) but I have seen this for other OS' and Windows is not the only one in that list.

---

### Post by Dangertux on 2011-07-10
I agree with the above post. In terms of production machines you should touch it as infrequently as possible.  Research updates prior and only install those that effect you positively.
