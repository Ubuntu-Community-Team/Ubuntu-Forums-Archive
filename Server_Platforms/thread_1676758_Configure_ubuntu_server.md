### Post by amra.sonntag on 2011-01-27
Start the Ubuntu Server Install on your favourite test machine. During the Installation select: LAMP (Linux-Apache-MySQL-PhP) and probably OpenSSH (for remote log in) and maybe some other things you want.

Once you are done with the installation - put your favourite website into /var/www and see if it works, when you point your browser to your new server. After that - you will probably want to read up on configuring your installation. A good place to start, with some additional ressources pointed out, is the Ubuntu Server Guide.
