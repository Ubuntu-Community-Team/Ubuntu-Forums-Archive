### Post by darkod on 2012-07-15
Only if it's a virtual machine like in Virtual Box or similar. But in that case it will not be running on full resources.

I am not aware of a way to boot both OSs from a dual boot at the same time. It's either one or the other.

EDIT PS: If this was a server installed only for practice, it would have been better to use VBox. Otherwise, there is little point of a server in dual boot because while win7 is booted the server is not available for anything.
