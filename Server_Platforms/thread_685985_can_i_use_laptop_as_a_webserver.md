### Post by frek818 on 2008-02-02
Sure, any computer that you can install webserver software (ex. apache lighthttpd) and has a network connections, can be used to server web pages. 

The real question is, does you laptop have enough memory and a powerful enough CPU to handle the amount of load that you're web server will be serving? What kind of hardware does you laptop have in it and how any people will be using it to view webpages?
