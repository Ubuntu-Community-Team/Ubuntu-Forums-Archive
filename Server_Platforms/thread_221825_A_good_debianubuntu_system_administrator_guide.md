### Post by harisund on 2006-07-28
To be frank, I spent a long time searching for something like that. 

However, eventually you realise that it is almost next to impossible to come up with a single super-guide of sorts. 

What are your requirements? What is it that you want to do? Once you start answering those questions,  you will begin to search the internet (using whichever search engine you choose)... you will begin to use apropos and man commands. Tweak it, customize it, break it. You will be amazed.

My advise would be to start using your system and playing aroudn with it. As someone suggested, write bash scripts and scripts in other languages. 

I, nearly every hour, check the Ubuntu forums' networking and server sections, and what I know are all stuff that I have learned from here and hanging out on the IRC forums.

Make it a point to learn something new every day, that eventually you can write up a book yourself. 

The beauty is, it eventually becomes an addiction in itself. let's say you begin with creating SSH keys and automated login. You will want to try them out with cron. Each man page has towards its end a list of related commands. You keep jumping from one command to another. 

And don't forget to read [http://home.xnet.com/~raven/Sysadmin/ASR.Quotes.html](http://home.xnet.com/~raven/Sysadmin/ASR.Quotes.html)
