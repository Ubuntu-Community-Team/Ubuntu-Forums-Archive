### Post by submitpaddy on 2008-09-03
The Amanda core team is pleased to announce the release of Amanda 2.6.0p2. 
 
Source tarballs are available from 
 
* [http://www.amanda.org](http://www.amanda.org) 
* [https://sourceforge.net/project/showfiles.php?group_id=120](https://sourceforge.net/project/showfiles.php?group_id=120) 
 
Binaries for ubuntu distribution and other OSes (including Win32 client for Windows) are available from 
 
* [http://www.zmanda.com/download-amanda.php](http://www.zmanda.com/download-amanda.php) 
 
Documentation can be found at 
 
* [http://wiki.zmanda.com](http://wiki.zmanda.com) 

This latest version provides open file backup and recovery for Windows XP, Vista desktops and laptops and Windows 2003, 2008 servers, enabling system administrators to perform backup without impacting users and applications. Look at the ReleaseNotes and ChangeLog file for more details.
