### Post by jonobr on 2012-01-09
I strongly recommend not ftping to a machine as root,
FTP is a weak protocol, but using using root login over FTP is a bad idea.
Root ftp login is disabled by default.

If you want to use FTP, what you need to do is setup a new user on your system,
create a new account with a home directory and login with that user name.

I would also recommend you install openssh,  and using something like winscp to copy files, its a lot more secure

```
sudo apt-get install openssh-server 
```

After that, remove vsftpd
