### Post by cariboo on 2012-09-16
I have an old Apple G4 specifically setup to play tunes from my mp3 library. I use randomplay and mpg123 and mount the music directory from my file server using NFS. The randomplay configuration file allows you to set the interval, before it starts repeating. I have mine set to 30 days before it repeats a track.

Randomplay and mpg123 are both in the repositories. I'm running Debian stable on it, because I just don't want to bother fiddling with it, but it'll work the same running Ubuntu.
