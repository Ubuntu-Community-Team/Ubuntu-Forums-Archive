### Post by nerdtron on 2014-07-05
The guide that you are using is for older version of Apache and Ubuntu.

Screenshot 1, minor problem, you can still use daloradius even with that.
Screenshot 2, apache is running fine.
Screenshot 3, here is the real problem.

Please post the output of
```
ls -lah /var/www/
```

Also have you tried moving the daloradius to the /var/www/html folder? This is now the default apache folder in Ubuntu 14.04.

```
sudo mv /var/www/daloradius /var/www/html/
```
Then restart apache and point your browser again to [http://IPaddress/daloradius](http://IPaddress/daloradius).
