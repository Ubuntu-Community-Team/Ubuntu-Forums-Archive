### Post by bpalone on 2010-02-20
Just did a quick Google for Free Web Hosting and found these (and others):

[http://www.absolutely-free-hosting.com/]("http://www.absolutely-free-hosting.com/")

[http://www.free-webhosts.com/]("http://www.free-webhosts.com/")

[http://www.freewebspace.net/]("http://www.freewebspace.net/")

These are review sites and will give you some options and allow you to make a somewhat informed decision.


The other option is find a cheap (preferably free) old desktop and install a Linux server on it and have a cheap home based server.  It isn't free as it consumes electricity, but you have control over it.  Once you have it set up and going, you can run it headless (no monitor) and just ssh into it to do everything. Mine doesn't have anything hooked up to it but the LAN cable and power cord.
