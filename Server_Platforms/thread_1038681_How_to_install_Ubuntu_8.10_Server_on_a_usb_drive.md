### Post by cariboo on 2009-01-14
There is no server LiveCD, so you'd have to install the server on your computer first, then using something like [Remastersys]("http:///www.remastersys.klikit-linux.com/"), to create a LiveCD after you have set the server up to your liking, then transfer the iso to your usb device.
