### Post by Cheesemill on 2010-01-18
The current LTS is 8.04 and is supported until April 2013, the next LTS is 10.04 which is out in April and supported until 2015.

It's a common mistake that people think that LTS versions are inherently more stable upon release, it's just that there is a longer period of time for all the bugs to be fixed so 10.04 may not be stable enough for you until it's been out for a while (although I have Lucid Alpha 2 on a couple of test servers and I've had no major problems).
