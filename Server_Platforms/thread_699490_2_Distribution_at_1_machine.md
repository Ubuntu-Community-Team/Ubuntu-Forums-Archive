### Post by faulkes on 2008-02-17
> **Hatoum said:**
> Hello, I have fedora 8 and i want to install Ubuntu 7.10.
Can'I have this 2 distribution in my machine at the same time ?

You can repartition the drive and install ubuntu on the newly created partition.

Alternatively you could use a viritualization container (kvm, qemu, vmware) to run a virtualized instance of ubuntu.

You cannot however install ubuntu over fedora and have both work.
