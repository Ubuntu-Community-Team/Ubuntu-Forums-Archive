### Post by Queue29 on 2010-01-24
64 bit essentially means that your cpu can deal with bigger numbers, and can access more than 4 GB of RAM. Almost all open source apps are available in 32 and 64 bit flavors, especially on the server side, so you won't notice any difference.

### Post by Vegan on 2010-01-24
A 64-bit machine will be more helpful with ext4 and other 64-bit objects. Disks are larger these days and 64-bit processors can better work with the large number of allocation units. So a 64-bit CPU will be faster.
