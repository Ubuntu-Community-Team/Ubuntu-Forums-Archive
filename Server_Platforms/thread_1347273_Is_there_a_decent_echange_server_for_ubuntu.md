### Post by Dan1317 on 2009-12-06
I really like [[COLOR="Navy"]Zimbra[/COLOR]]("http://www.zimbra.com/").  You can use either your browser or Zimbra Desktop.  The opensource version is free, but you can get support and more features with different versions.  A college near me uses Zimbra over Exchange and loves it.

Zimbra has LDAP servers integrated, so depending on what you are using if for, you can make a concise network.  (Zimbra also supports multi-server installs [ie. one server is just Zimbra LDAP servers, one is MTA, etc etc])

With Zimbra paid you can get MAPI support for Outlook.  I only have used open source so I can't really say much more about the other versions.


I'm not sure what scale you need, but [[COLOR="navy"]eBox[/COLOR]]("http://www.ebox-platform.com/") works pretty good too.  

If you need a full tutorial for mail setup, [[COLOR="navy"]HowToForge[/COLOR]]("http://www.howtoforge.com/howtos/email") has a section devoted to mail server setup.

Zimbra's [[COLOR="navy"]wiki[/COLOR]]("http://wiki.zimbra.com/index.php?title=Ubuntu_8.04_LTS_Server_(Hardy_Heron)_Install_Guide") is also good too.

Hope this helps :)
