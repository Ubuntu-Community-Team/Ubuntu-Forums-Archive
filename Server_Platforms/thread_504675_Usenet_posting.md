### Post by meatpan on 2007-07-22
Two recommendations, but neither are perfect:

pine : [http://www.washington.edu/pine](http://www.washington.edu/pine)

The big problem with pine is the licensing.  Otherwise a great program.  Adding usenets and posting is a cinch.  I've been using pine for usenet reading/posting for about 10 years. 

emacs : [http://www.gnu.org/software/emacs](http://www.gnu.org/software/emacs)

Emacs can be run with the '-nw' option to use a terminal (rather than X).  The emacs mode/program you want is 'gnus'.  There is a bit of a learning curve to using emacs and gnus.

Hope this helps.
