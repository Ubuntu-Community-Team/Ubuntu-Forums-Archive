### Post by pytheas22 on 2009-06-30
Do you mean add a background image to the terminal that you use within Gnome?  If so, yes; check out [this page]("http://lirent.net/ubuntu/how-to-add-transparency-and-background-image-to-terminal-on-ubuntu.html").

Ubuntu Server Edition (unlike SUSE, I believe) doesn't come with any kind of GUI running by default; as a result, you can't customize that terminal, as far as I know.  If you're running text-line-only Ubuntu, I don't think you can do anything to make it look nicer (you could install a graphical environment, but that mostly defeats the purpose of Ubuntu Server Edition).
