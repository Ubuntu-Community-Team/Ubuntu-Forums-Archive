### Post by bemenaker on 2010-06-01
Did you install a desktop or server version of Ubuntu, only difference is that desktop gives you a graphical environment to use and installs some other stuff you won't need as a server, but it's not a problem.  I would say a dekstop would be easier for a beginner because you would be a little more familiar working in one.  If you have a desktop installed, go to System - Administration - Package Manager and install Apache2, PHP5, and MySQL (if you need a database).  That will give you the basic engines to run everything. [http://www.howtoforge.com](http://www.howtoforge.com) also has great tutorials to get you up and running.  If you installed a server version, since it will all be command line stuff, just go to how to forge and follow directions.
