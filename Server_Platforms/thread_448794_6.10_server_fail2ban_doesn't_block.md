### Post by fyl2u on 2007-05-21
have you edited your jail.conf file in the /etc/fail2ban/ directory to be enabled for your type of service?

i.e.

[yourcomputer]/etc/fail2ban/~:  sudo pico jail.conf

my ftp server is running vsftpd, so in my jail.conf there's a section that starts: "VSFTPD = Disabled" that I had to change to "enabled" then restart vsftpd: 

sudo /etc/init.d/vsftpd restart
