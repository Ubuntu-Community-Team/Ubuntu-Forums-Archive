Hello

I have been searching the web for a while now trying to find an online form creator for PHP. I host my own LAMP server, and this has become a holy grail for many of the things I'd like to do with it.

Here are the basic things I'm looking for:
A user can log on to the site, and use a web interface to create a form.
While doing so, the site defines a table in MySQL to store the data from the form.
The user can then disseminate a link to the form to whomever needs it, and those people can fill out the form.
The creating user can then log in and download the data collected in a common format (.csv, .xls, etc...)

User authentication or Captchas are an added plus.

I have found many sites on the web that will help you define a form and then let you download the HTML that makes it work, but they expect you to be able to define the database yourself. This needs to be easy enough that Joe Public can use it. 

The closest thing that I have found is MachForm ([http://www.appnitro.com/](http://www.appnitro.com/)). However, something that is free and open source would be best. Would anyone have a suggestion? If I am unable to find one, I may look at making one myself in the future. If you do not have one, would you be interested in one if it became available?
