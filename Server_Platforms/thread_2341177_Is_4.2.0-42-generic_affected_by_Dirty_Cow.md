### Post by deadflowr on 2016-10-25
4.2 kernels?
Yes, they are affected by dirty cow.
As they are unsupported kernels now.

If on servers then upgrade to the xenial kernel stack with
```
sudo apt-get install --install-recommends linux-generic-lts-xenial 
```

from here:
[https://wiki.ubuntu.com/Kernel/LTSEnablementStack]("https://wiki.ubuntu.com/Kernel/LTSEnablementStack")

### Post by vagarwal on 2016-11-07

```

```


Just in case if you want to go back to the original trusty 3.x kernel:
```
sudo apt-get install linux-generic-lts-trusty
```

You can then get rid of 4.x kernel tree:
```
sudo apt-get purge 'linux-.*wily' 'linux-.*4.2.*'
```
