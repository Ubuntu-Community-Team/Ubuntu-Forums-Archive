### Post by Strayfolk on 2011-06-11
You can set keyboard layout under keyboard setting. Choose the one you want to use and click apply to system.

However, on my 11.04 Ubuntu Studio 64 bit install, it kept reverting to USA keyboard layout, after reboot or in the middle of the session, randomly. I want it to be Swedish or Finnish.

So I did the following:

gksu gedit /etc/default/keyboard 

and put```


XKBMODEL="pc105"
XKBLAYOUT="se"
XKBVARIANT=""
XKBOPTIONS=""
```

instead of XKBLAYOUT="us". Now Swedish is my default keyboard layout. I can write åäö and it doesn't reset to US layout.

Hope this helps anyone,
Strayfolk
