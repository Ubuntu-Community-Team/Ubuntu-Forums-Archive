### Post by bluefrog on 2009-03-17
get rid of the .htaccess protection and include the protection in the directory declaration of your virtual host. you can put that in a different file if you wish as well.

For example:
 	```
<Directory /var/www/yoursite/>
		Options Indexes FollowSymLinks MultiViews
		AllowOverride None
		Order allow,deny
		allow from all
		Include /whatever/path/default-security
		RedirectMatch ^/administrator https://yoursite.com/administrator
	</Directory>
```

then default-security:
```
<Files configuration.php>
order allow,deny
deny from all
</Files>

... everything you have in your current .htaccess
```
