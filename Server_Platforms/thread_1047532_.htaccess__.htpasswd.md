### Post by hrod beraht on 2009-01-22
Do you have access to the main httpd.conf file? Putting the user authentication files there, [[COLOR="Blue"]as recommended by the Apache folks[/COLOR]]("http://httpd.apache.org/docs/2.2/howto/htaccess.html#when"), works fine with my multi-site server, with both basic and digest authentication.

etc/apache2/httpd.conf
basic authentication
```
<Directory /var/www/site>
AuthType Basic
AuthName "SECRET SQUIRREL RESTRICTED ACCESS AREA"
AuthUserFile /etc/apache2/passwords
Require valid-user
</Directory>
```

etc/apache2/httpd.conf
digest authentication
auth_digest.load in mods-enabled
```
<Directory /var/www/site>
AuthType Digest
AuthName "SECRET SQUIRREL RESTRICTED ACCESS AREA"
AuthDigestDomain /site/
AuthUserFile /etc/apache2/passwords
Require valid-user
</Directory>

```
