### Post by SeijiSensei on 2014-02-12
Have you ever tried compiling applications from [source]("http://www.squid-cache.org/Versions/v3/3.4/")?  Squid compiled pretty cleanly for me the last time I built it. Here's an introduction to compilation on Ubuntu: [https://help.ubuntu.com/community/CompilingEasyHowTo](https://help.ubuntu.com/community/CompilingEasyHowTo)
