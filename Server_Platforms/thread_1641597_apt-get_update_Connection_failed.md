### Post by ShadeS_07 on 2010-12-09
> I am in Germany, can it be a problem, since i am trying to update from US ? 

Well, it could be a factor... So try this as well...

First, make backup of your **/etc/apt/sources.list** file...
Rename the *sources.list* backup as **sources.list.bk**

Now, make a _new_ **sources.list** file and copy/paste the following:

```
#############################################################
################### OFFICIAL UBUNTU REPOS ###################
#############################################################

###### Ubuntu Main Repos
deb http://de.archive.ubuntu.com/ubuntu/ maverick main restricted universe multiverse 
deb-src http://de.archive.ubuntu.com/ubuntu/ maverick main restricted universe multiverse 

###### Ubuntu Update Repos
deb http://de.archive.ubuntu.com/ubuntu/ maverick-security main restricted universe multiverse 
deb http://de.archive.ubuntu.com/ubuntu/ maverick-updates main restricted universe multiverse 
deb-src http://de.archive.ubuntu.com/ubuntu/ maverick-security main restricted universe multiverse 
deb-src http://de.archive.ubuntu.com/ubuntu/ maverick-updates main restricted universe multiverse 

###### Ubuntu Partner Repo
deb http://archive.canonical.com/ubuntu maverick partner
deb-src http://archive.canonical.com/ubuntu maverick partner


```

... and save the file.

Now do

```
sudo apt-get update
```

again, and try downloading that package you were tryin' to download.

Btw, you might find this link useful:

[http://repogen.simplylinux.ch/index.php](http://repogen.simplylinux.ch/index.php)

It's an Ubuntu Sources List Generator... That sources.list was actually generated by this generator... -_+
