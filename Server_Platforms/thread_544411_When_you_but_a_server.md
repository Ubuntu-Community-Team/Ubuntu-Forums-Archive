### Post by HermanAB on 2007-09-06
First thing?  Configure the IP network settings.
Second thing? Scan it with nmap to see whether the firewall settings are any good.

If you intend to provide virtual web sites to clients, then I guess it will be fine. Plesk is good for that.  If you want to use it as a mail or file server, then you will probably be better off re-installing.
