### Post by cdenley on 2010-04-08
The output you actually posted indicates you don't have any DNS server configured. However, if that were the case, then you would get output like
```

cdenley@ubuntu:~$ host ubuntu.com
Host ubuntu.com not found: 3(NXDOMAIN)
cdenley@ubuntu:~$ nc -z -v -w 5 ubuntu.com 80
ubuntu.com: forward host lookup failed: Unknown host

```
I don't think you posted output from the first command, and you seem to suggest that the second command results in an open connection.
