### Post by albandy on 2008-12-18
Yes, it's named an heterogeneous cluster
The best Linux Distribution that I Know for doing this is rocks.
It works with MPI and PVM and has a lot of features as node authomatic installation via bootp and a very usefull cluster management interface.

Read the documentation. You can download the documentation and the isos from  [http://www.rocksclusters.org/](http://www.rocksclusters.org/)
