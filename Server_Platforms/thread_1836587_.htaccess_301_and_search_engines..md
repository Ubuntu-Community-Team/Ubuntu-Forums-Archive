### Post by Habitual on 2011-08-31
Google recommends the use of 301s actually, according to [http://www.google.com/support/webmasters/bin/answer.py?hl=en&answer=93633&ctx=cb&src=cb&cbid=-1x05xlchti4ye&cbrank=0](http://www.google.com/support/webmasters/bin/answer.py?hl=en&answer=93633&ctx=cb&src=cb&cbid=-1x05xlchti4ye&cbrank=0)

"If you need to change the URL of a page as it is shown in search engine results, we recommended that you use a server-side 301 redirect."

Specifics can be reviewed at [https://www.google.com/support/webmasters/bin/search.py?query=301&ctx=en%3Asearchbox](https://www.google.com/support/webmasters/bin/search.py?query=301&ctx=en%3Asearchbox)
