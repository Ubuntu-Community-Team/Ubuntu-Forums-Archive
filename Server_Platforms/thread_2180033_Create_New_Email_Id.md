### Post by nerdtron on 2013-10-11
Simple. Create an actual account in zimbra and then set filters to reject/forward mail all mail. Or set it to reply to all emails with a predefined message (I'm not sure about the this but I think it is doable in the account settings in zimbra).
If you are setting up a new server look at iredmail, zimbra was too heavy on resources on my testing.
