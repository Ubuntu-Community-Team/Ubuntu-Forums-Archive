### Post by memilanuk on 2011-12-10
Depends on what you have available.

If you have just a stand-alone mysql server, you can log into mysql and issue various commands to see the schema, tables, and table data.  A quick search on the web should get you headed in the right direction.

If you have Apache & PHP also installed on say, a LAMP server, you can see if PHPmyAdmin is installed (and if not, install it) which will give you a GUI interface via your web browser.  Again, a search is recommended as this is pretty standard stuff and has been covered lots of times - no need to re-hash it here unless you have troubles.
