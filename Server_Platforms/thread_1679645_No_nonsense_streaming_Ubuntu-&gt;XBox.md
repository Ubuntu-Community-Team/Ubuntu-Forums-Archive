### Post by rubylaser on 2011-02-01
Are you using an original XBOX or an XBOX360?  If you're using an original XBOX, then XBMC is perfect for doing this.  To use it you'll need a modded XBOX, and then you install XBMC on top of the Dashboard.  Here are some directions to do that.
[http://www.xbmc4xbox.org/wiki/index.php?title=Installing_XBMC_for_Xbox]("http://www.xbmc4xbox.org/wiki/index.php?title=Installing_XBMC_for_Xbox")

Caution: this isn't hard, but is not for the faint of heart.

If you have a 360, XBMC works with alot of work, but wasn't usable the last that I knew.  Twonky will still work, and if you've run into bugs on newer versions, you could always run an older version.  It's just an uPNP server, so older versions should work fine.
