### Post by Hanzerik on 2007-01-30
Lets see, I had Slackware 11, Ubuntu 6.06 Server, and now Debian Etch running on my server since Slack11 came out last year. I am sticking with Debian Etch for now, but had good performance with the others and am familiar with all of them from the CLI. It was a toss-up between keeping Ubuntu or pure Debian as my server host. Slack is a great distro, but I have fallin for apt-get, makes updates and software installs very easy.

I picked pure Debian instead of Ubuntu for no particular reason, they both perform pretty much the same, but Debian seems a little more "less bloated" then Ubuntu. Ubuntu Server does a great job as far as I can tell, easy to setup and configure if you know your way around a Linux CLI. 

I can't really tell you any pros or cons towards FreeBSD or Gentoo. I have used both, but it has been a couple years. Heh, I went from LFS -> Gentoo -> Slackware -> Debian Based, kind of getting lazy in my old age :-)
