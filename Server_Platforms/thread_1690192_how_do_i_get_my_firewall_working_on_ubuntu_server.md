### Post by elico on 2011-02-18
i wrote on this post nice rules
[http://ubuntuforums.org/showpost.php?p=10439802&postcount=4](http://ubuntuforums.org/showpost.php?p=10439802&postcount=4)

will rewrite them for you:
[COLOR=#000000][COLOR=#0000bb]
[/COLOR][/COLOR][PHP]
IPT=/sbin/iptables
# Accept all from localhot
$IPT -A INPUT -i lo -j ACCEPT
#add this line here to match first the established connections
$IPT -A INPUT -m state --state RELATED,ESTABLISHED -j ACCEPT

# Port 80 (www) open, and need add other like ssh or mail
$IPT -A INPUT -i ppp0 -p tcp -p tcp -m tcp --dport  21,22,80,10000,5900  -j ACCEPT
$IPT -A INPUT -i ppp0 -p udp -p udp -m tcp --dport  21,22 -j ACCEPT

you can use these:(or the next)
# Close the rest
$IPT -A INPUT -i ppp0 -p tcp -m tcp --syn --dport 1025:65535 -j DROP
$IPT -A INPUT -i ppp0 -p udp -m udp --syn --dport 1025:65535 -j DROP  

$IPT -A INPUT -i ppp0 -p tcp -m tcp --dport 1:1024 -j DROP
$IPT -A INPUT -i ppp0 -p udp -m udp --dport 1:1024 -j DROP

this is the next: to close all except the above
$IPT -A INPUT -i ppp0 -j DROP[/PHP]
