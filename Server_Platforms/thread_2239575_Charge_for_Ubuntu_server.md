### Post by JetStorm on 2014-08-14
There are paid support services but unlike RHEL they're completely optional (optional as in you can use the server version absolutely free and not pay for support subscription if you're able to manage the machines yourself).
You can find some more info here - [http://www.ubuntu.com/server/management](http://www.ubuntu.com/server/management) and here - [http://www.ubuntu.com/management/ubuntu-advantage](http://www.ubuntu.com/management/ubuntu-advantage)
