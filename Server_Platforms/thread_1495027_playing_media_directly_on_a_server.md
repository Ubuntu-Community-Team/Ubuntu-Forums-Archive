### Post by dudumomo on 2010-05-28
What you can do is,
for using your computer on your TV as a media center, I recommend you to install XMBC (Really nice and powerful mediacenter software), you will then, need to install ubuntu with an interface. You can either install Ubuntu Desktop with everything and much more (So, a lot of useless stuffs) or install an alternative version and then install gnome-core to get a minimalistic interface.

To share your music on other computer, you can still do the same thing (Samba + VLC) or directly use a Jukebox software to be able to share your songs and videos online through a web interface by example. But in this case you will need to set up a LAMP server.

### Post by spynappels on 2010-05-28
Do you want to hook the TV up as the monitor for the computer, or were you thinking of a different solution?

How well this would work depends largely on the TV, as the resolution you may get may not match the computer's graphics card. Also how would you get sound into the TV?

If the TV is a recent HD Ready TV, check whether it is a DLNA compliant player, if so, all you need to do is run a DLNA server on the server, plug the TV into the network, and away you go!
