### Post by Ryan Dwyer on 2010-09-12
There is no set limit. The busier the machine is, the more its performance will suffer. How well it performs depends on many things such as hardware specs, what it's doing for each user and how many users are actively using the system.

If you just want to serve files with Samba or NFS, then you can handle 200 users with an EeePC 701 with 1GB RAM running at 600 MHz...
