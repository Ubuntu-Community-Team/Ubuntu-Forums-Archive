I can describe this as a chicken and egg situation.  Also, it's seems to be an "us and them" situation, but it's not.

Many things in ubuntu are a community effort.  If there is less trafic in the server section of the forums it is because there are less people here.  The single best way you can make this section of the forums great is to be here and help make it great.

It's a chicken and egg situation, because there are not a lot of people who are attracted to the forums who are interested in the server aspect of the OS.  Is is primarily a desktop distribution whose server side is about the become more mainstream.

All the tools are here.  This place is not neglected by anyone - it's not like there are more employees manning the desktop support forums.  There are no employees.  It's all in the hands of those who visit.


As for things being buried in the forums, I have had a few ideas.  There have been a few mentions of the fact that the forums are really really big.  Tell me what you think:

What is there were customizable templates for different groups of forum sections?

For example, if someone were to visit

server.ubuntuforums.org

They would be shown a subset of all the forum sections which included only the server section, security issues and installation help, for example.

Maybe desktop.ubuntuforums.org could display all the desktop-related forums and not show the other developmental, server and third-party forums.

Likewise, you could have a beginner.ubuntuforums.org which would only show the absolute beginner section.  Community.ubuntuforums.org can only be for non-tech talk.    Tech.ubuntuforums.org could hide the non-tech support forums.

Each template could have search restrictions, so that if you search through the tech.ubuntuforums.org template, you only see things from the allowed sections.

Of course, there would always be the ubuntuforums.org page which would display all of the forum sections.  My thought is that the forums are so big that perhaps a lot of people are too intimidated to frequent them.  Perhaps they are so new to the linux and forums community that they can even post their question and not be able to find it again?

Perhaps this would also cut down on bandwitdh as fewer things are displayed?  I dunno.

Anyway, would this help with the issue at hand - the server section?
