### Post by BkkBonanza on 2009-12-05
Putty is capable of using a key for login. If I recall it has a key generation feature built in. This would allow auto-login then but be aware that anyone with access to Putty then would have access to the server. I'm pretty sure it's easy to tell Putty a script to run automatically after login as well. So everything is right there in Putty. It's been a few years since I used it on Windows so I can't tell you exactly where to look on the config screens.
