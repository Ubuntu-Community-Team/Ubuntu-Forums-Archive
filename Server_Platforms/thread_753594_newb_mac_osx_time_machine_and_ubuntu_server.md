### Post by tubezninja on 2008-04-12
By default, Time Machine only works with HFS+ formatted drives, or on networked drives on machines running OS X.

There are ways to [hack it up a bit though]("http://www.pherg.net/2007/11/29/mac-os-105-time-machine-with-linux-server/"), and do what you want.
