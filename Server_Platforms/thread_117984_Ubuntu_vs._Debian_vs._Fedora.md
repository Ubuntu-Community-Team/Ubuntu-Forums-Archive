I use Ubuntu 5.10 on all 7 of my computers and have not had a single major problem. I switched to Ubuntu from Fedora Core 4 because Ubuntu is much faster and apt-get blows yum away big time.

Fedora is what Red Hat uses to test new stuff that may find its way into Red Hat Enterprise Linux, so at times Fedora can be pretty bleeding edge and have lots of problems - probably not a good idea for a mission-critical solution.

I don't know much about debian except that it is widely used and a great number of people (even ex-debian users) say it is very stable.

Hope this helps.
