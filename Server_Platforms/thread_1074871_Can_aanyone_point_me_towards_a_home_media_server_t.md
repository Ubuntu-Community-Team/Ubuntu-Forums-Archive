### Post by volkswagner on 2009-02-20
Have a look at [Jinzora]("http://en.jinzora.com/").  Once you have a functioning web server, installing Jinzora as a virtual host is well documented.

It has a nice feature to allow streaming to  any pc.  You can also choose jukebox mode which will allow local playback on the server if it is connected to speakers/amp.

You may want to explain what your specific needs (layout/infrastructure) are for more relevant answers.
