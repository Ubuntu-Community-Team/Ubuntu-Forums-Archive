### Post by garfonzo on 2009-11-18
Hi all,

I've got a server in my garage with Ubuntu on it and about 7 HDD hanging off of it. So far, it has been rock solid and done all that I wanted it to do. Of course, all I want it to do is share files on the LAN so it's easy. 

All of the computers in the house have 3 mapped drives which correspond to a drive on the Server. We all share the same music, movie and TV files. One computer in the family room is connected to the stereo system and movie projector. When we want to watch a movie, we turn on that computer, open the mapped directory, find the movie, and watch it on VLC. 

What I'm wondering is, how is this (the process I have above) any better/worse than having something like, LinuxMCE or XBMC installed on that computer when it comes to watching/sharing content?

It just seems to me that they have a lot of eye candy to make it all look nice which is great if thats what you want. I used XBMC for a while, but just found it a hindrance rather than a benefit. 

Is it all just personal opinion? 

Just looking for some thoughts on why you do or do not use a Media Center type setup and if there are real benefits one way or the other.

### Post by afterwego on 2009-11-19
I'm with you on just setting up a share and doing it the way you are already doing it.

I think things like XBMC, Boxee, and similar applications are designed to be "easy" and add a pretty interface. Granted the interface is pretty and for the most part functional there are a lot of extra steps that need to be done to get to what you want.

Personally, I would stick with how you're doing it because it requires the least maintenance and the least amount of steps to accomplish watching a video, playing music, or doing whatever is you want to do.
