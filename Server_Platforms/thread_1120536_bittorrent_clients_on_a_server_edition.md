Yes and no. The client itself runs on the server. When you're in control of the server, you only have the console. The client(s) are therefore controlled in the console. A lot of clients though have the ability to control them via remote calls. This means:

1. Bittorrent clients run at server
2. Software to control (the GUI) runs at your desktop

The best one available: [rTorrent]("http://libtorrent.rakshasa.no/"). If you do a search on ubuntuforums.org, you will see that a lot of documentation is available.
