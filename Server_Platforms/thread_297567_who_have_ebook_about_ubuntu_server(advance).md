### Post by elst on 2006-11-12
If you install the "rutebook" package you get a copy of the RUTE guide, which is then available as file:///usr/share/doc/rutebook/html/index.html

RUTE is also available on-line.

"The Debian System" by Martin Krafft is also excellent. It's published by No Starch Press, who offer PDF versions (for a good price).
