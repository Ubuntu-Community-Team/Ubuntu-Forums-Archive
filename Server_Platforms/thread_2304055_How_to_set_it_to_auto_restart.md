Some computers come with a setting in BIOS that lets you restart your machine when power comes on after a power cut. My old COMPAQ machine, which I used as a server until the machine failed on me, had such a setting. A good place to start would be to look in the BIOS power settings menu item. No idea here if the UEFI system has a similar option.
