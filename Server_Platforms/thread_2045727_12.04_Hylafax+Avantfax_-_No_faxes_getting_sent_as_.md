### Post by jumping_snake on 2012-08-22
Ah, ha!! The postfix was part of the problem, but I forgot that AvantFax 3.3.3 doesn't overwrite /var/spool/hylafax/bin/faxrcvd, it instead puts it's faxfcvd.php file there and leaves the Hylafax version of the file as well. After renaming the original to a different name, I renamed faxfcvd.php (AvantFax's copy) to faxrcvd and viola I'm back in business.
