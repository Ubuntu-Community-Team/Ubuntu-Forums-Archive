### Post by HermanAB on 2009-02-15
The default install of vsftp and proftp should work out of the box.  Usually, a sure way to break it, is to change the default settings.

First see whether the server is running at all:
$ sudo ps -e|grep ftp

You should see the ftp server process.

If not, restart it.

Now try to connect to it:
$ telnet localhost 21

You should see the server banner.

Now check your firewall and ensure that it is either completely disabled, or opened for FTP using a FTP connection tracker.

You can flush all iptables rules with:
$ sudo iptables -F

Hope that helps!

Herman
