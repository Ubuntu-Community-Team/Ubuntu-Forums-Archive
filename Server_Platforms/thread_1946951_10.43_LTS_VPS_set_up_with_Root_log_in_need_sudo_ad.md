### Post by iiz on 2012-03-26
One of two thigns you can do.

1. As root you can (assuming your username is john):

```
usermod -aG sudo john
```

This adds you to the sudo group and gives you sudo access when the server has default sudo configuration.

2. Add your user account to the sudoers file:

```
nano /etc/sudoers
```

Add:

```
john    ALL=(ALL) ALL
```
