### Post by druhboruch on 2009-07-15
GUI is not necessary to install Virtualbox.
Everything can be configured via command line.
You can also start and shutdown VM's using init scripts or cron.

The manual is very comprehensive.  You may download it from 
[http://www.virtualbox.org/manual/UserManual.html](http://www.virtualbox.org/manual/UserManual.html)
