### Post by Lars Noodén on 2012-01-26
Secure Socket Layer (SSL) is being replaced by Transport Layer Security (TLS). 

TLS encrypts sessions end-to-end.  It's often used to secure old, unencrypted protocols.  HTTPS is one example: [https://help.ubuntu.com/community/forum/server/apache2/SSL](https://help.ubuntu.com/community/forum/server/apache2/SSL)

More can be found by searching.
