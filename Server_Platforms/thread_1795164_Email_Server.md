### Post by Lars Noodén on 2011-07-03
Citadel is in the groupware category.  If you need groupware options, then there is also [Kolab](http://www.kolab.org/), [Dingo Calendar Server](http://andrew.triumf.ca/dingo/),  or [OpenGroupware](http://www.opengroupware.org/)

If you only need [mail](http://bsdly.blogspot.com/2011/02/problem-isnt-email-its-microsoft.html) then there is [simta](http://rsug.itd.umich.edu/software/simta/), [Postfix](http://www.postfix.org/), [Exim](http://www.exim.org/), or [Sendmail](http://www.sendmail.org/).
