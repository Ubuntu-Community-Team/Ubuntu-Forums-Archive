### Post by zero244 on 2007-06-27
Most servers are closed box setups.......Most servers Ive seen have been at least P4's. Mainly because the P4's have the failsafe diode incase the cpu fan fails it wont fry the system.
For Personal computers I use AMD exclusively, but I can keep a closer eye on my systems.
If a server is going to locked away in a back room somewhere unattended the P4 would probably be the best choice. 
If your choice is between the Celeron and the AMD I would go with the AMD 2400.
Ive had several AMD Athlons with very good results.
If you use the AMD don't go cheap on the CPU heatsink. And for that matter dont go cheap with the Celeron either.
The AMD's are fast work horse CPU's. As long as you keep them within specs heat wise they will run very well for years.
I like them so well I bought a Athlon 2200 on Ebay not too long ago as a backup.
