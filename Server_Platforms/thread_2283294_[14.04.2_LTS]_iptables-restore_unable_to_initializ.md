### Post by sandyd on 2015-06-20
Bit of background:
The server you currently have runs on OpenVZ Virtualization (with Proxmox). OpenVZ servers share a kernel with the host, and as a result, you cannot load any kernel modules, and do some other tasks.

That being said, it is possible that the required iptables module is missing. Ask the host to add iptable_nat to your VPS.

If you have access to the host server, add the following to "IPTABLES" in /etc/vz/vz.conf
```
iptable_nat
```
