### Post by Iowan on 2010-01-26
If your server is set up via DHCP address, the "unknown" problem can (probably) be fixed by editing */etc/dhcp3/dhclient.conf * and adding the server name to the "send host-name" line. If you have access to the DHCP server, you can probably set it up to issue a "static lease" to the server (based on MAC address).
