### Post by FakeOutdoorsman on 2008-07-08
I've had good experiences with [Website Baker]("http://start.websitebaker2.org/en/introduction.html") for some local organizations who aren't very tech savvy. It gives them an option to edit pages using an interface that looks similar to OpenOffice Writer/MS Word, which most people are familiar with.  It also allows them to create/delete pages, manage users, and upload files.  Upgrading is usually easy and the templating system is straightforward.  One downside is that it isn't as robust as something like Joomla, but you may just be looking for something simple.

### Post by Pharaoh Atem on 2009-04-11
[Enano CMS](http://enanocms.org) has the kind of thing you are looking for. It has ACLs that let you limit what users can do what with certain pages, so you can set up an entire set of pages and have them restricted to a certain teacher. It's also secure and modular, so you can add in plugins to add support of certain things.

[http://enanocms.org/Features](http://enanocms.org/Features)
