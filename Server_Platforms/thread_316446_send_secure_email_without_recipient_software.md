### Post by tturrisi on 2006-12-10
Secured how?  You mean encrypted messages?  If so, the recipient will need a plugin for his mail client to decrypt them.  No way around that.  A workaround is to use a weaker encryption such as javascript encrypting that can be decrypted by pasting the message into an html form.  But that's inconvenient.  Best solution is the free PGP plugin for windows users.
