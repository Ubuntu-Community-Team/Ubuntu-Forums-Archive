### Post by kaelspencer on 2010-07-22
Absolutely. MythTV has a client-server architecture. If you run Mythbuntu and that is all, it will act as both the client and the server. If you want to add another client (frontend), all you need to do is install MythTV frontend (or Mythbuntu) on another computer and tell it where the backend (server) is.
