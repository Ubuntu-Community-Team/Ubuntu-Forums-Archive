### Post by Meelad on 2010-07-14
I've had this problem for more than a week, I searched for a solution everywhere but I couldn't find any.

The problem is when you install egroupware on ubuntu (server 10.04) you get a message "your session could not be verified." every time you try to log in. This problem seems to be caused by a variety of reasons, but what I had was a fresh egroupware installation on a fresh ubuntu 10.04 server installation, so I think many would encounter the same problem, and probably for the same reasons.

Well, now I still don't know the reasons, but came up with a simple solution.. It seems that there is a problem with the default version of egroupware that comes with ubuntu (1.6.001). I installed egroupware version 1.6.003-2 package from ([http://www.egroupware.org/download](http://www.egroupware.org/download)), and it worked like a charm without any hussles!

So for anybody who is facing the same problem, and still can't find a solution try installing a newer version (I have tested version 1.6.003-2 only), and it could very well solve your problem..

And is it possible to replace the old version of egroupware (1.6.001) with the new one (1.6.003-2) in ubuntu's default repositories because the current default package definitely has a problem.
