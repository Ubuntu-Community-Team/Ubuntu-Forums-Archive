### Post by rickyjones on 2007-06-23
On the computer with the printer you will go to Start > Settings > Printers... Right click your printer and click Properties. Click the Sharing tab. Check the box to Share this printer and give it a name. Apply. Now on all the other computers you will want to add a new printer, look for a network printer, and browse to the computer that has the printer attached (so you'll need to know the name of the computer). The printer should be listed there.

Should be pretty intuitive from that point on. Let me know if you need more help, but this should get you started. It's a relatively simple process.

-Richard
