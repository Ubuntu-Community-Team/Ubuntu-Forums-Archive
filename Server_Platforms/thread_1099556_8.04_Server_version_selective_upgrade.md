Is there a way to selectively upgrade packages using apt on Hardy ?
Let say I only want to apply a kernel update, or a samba update, and leave other (non dependant) packages untouched.
With the desktop version, it is a simple matter of graphically selecting which updates to apply, but how can I achieve a similar result with apt (or with another tool) ? What would you recommend ?

### Post by lykwydchykyn on 2009-03-18
Just use "install" to select packages individually.  "install" will grab the latest version of the package.
e.g.
```

sudo aptitude install linux-image-whatever

```
Of course you can use apt-get instead of aptitude if that's what you're used to.
