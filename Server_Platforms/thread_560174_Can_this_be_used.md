### Post by kyphi on 2007-09-26
Do you mean that you have a website that is being hosted on a server and that you want to transfer to and/or remove your files from that website?

Then go to your top panel, if using Ubuntu, to Applications, Add/Remove.  Then 'Internet' and you will find 'gFTP' waiting for you to install it.

gFTP is easy to use and is sanctioned by Ubuntu.
