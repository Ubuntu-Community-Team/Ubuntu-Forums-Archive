### Post by fang0654 on 2010-03-26
You either need to settle with an imap setup with your phone, or get hosted email with something that will work with ActiveSync, such as Zimbra.  Another option is gmail, which I believe works as push with Android.

From what I've found, on Android you can also use a mail client called K-9 Client, which using imap, is almost as fast as push.
