### Post by jimm on 2006-08-31
Samba has its own web based gui called swat. I think you can install it by doing:


```
sudo apt-get smbclient-swat
```

Then you just use your browser on por 901 to get to the Swat site:

```
http://<your_samba_host>:901

```
Swat is pretty good to.
