### Post by KCPokes on 2007-10-01
I'm not sure I understand.  You are looking for something to allow you to run an internet site locally, which could run perl, php, asp, etc...?  Other then ASP, Apache with the right plugins will allow you to run the site.  Sounds like you are looking for something to maintain it as well, such as an IDE?  There really isn't a "one stop shop" for all these languages.  Personally I have a server run all my sites and use a simple text editor to handle any PHP or Perl coding changes, test it locally and SCP them to the "Production" server when complete.  Otherwise there are some good alternatives, such as Eclipse for PHP, that you could pursue.
