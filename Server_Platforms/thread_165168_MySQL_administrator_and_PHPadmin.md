### Post by adamkane on 2006-04-24
Try them both. You'll find out which one you prefer soon enough. mysql-admin is a stand-alone program, whereas phpmyadmin is a web interface. They both do the same things, but phpmyadmin is probably easier to use.

It's difficult to install phpmyadmin, unless you install apache2/php4/mysql4/phpmyadmin all together:
[https://wiki.ubuntu.com/ApacheMySQLPHP](https://wiki.ubuntu.com/ApacheMySQLPHP)
