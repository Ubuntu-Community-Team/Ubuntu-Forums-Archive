### Post by Girya on 2011-01-04
shot in the dark. try changing the 5th field in fstab to a 0 from a 1. all the lvm volumes I mount at boot have a zero in this field. A zero keeps the system from dumping the fs at boot. see man fstab

another idea: use blkid to get the uuid for the first field and use that to spec the block device to mount
```
sudo blkid
```
