### Post by megastench on 2008-08-31
This worked for me:

from [https://help.ubuntu.com/community/HardyUpgrades](https://help.ubuntu.com/community/HardyUpgrades)

run "sudo do-release-upgrade -m desktop" in a terminal window

I'm not sure what the '-m disktop' does, but this did the trick.
