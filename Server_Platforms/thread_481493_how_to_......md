### Post by mattskent on 2007-06-22
I'm pretty sure that the pagefile won't trick the system into thinking it has more RAM though...games/OS's that wouldn't run before because of lack of RAM still won't run. I'm not aware of a way to do what you're asking. Also, the hard drive is many times slower than RAM, so it wouldn't be efficient even if you could find a way to make it work.
You mentioned Vista...tricking the computer into thinking it has more RAM definitely won't work for Vista...if you (or your friend) has less than 512mb of RAM already, you can't run Vista. You need at LEAST 1gb of RAM to run Vista effectively.
