### Post by Paradox_Wanderer on 2011-01-11
It's an Active Directory connector for *nix.  It works pretty well so long as you get all the bugs worked out.

This particular module, as far as I understand it, is command line only and acts as a negotiator between AD and *nix systems.  Like a BDC, but *nix based.
