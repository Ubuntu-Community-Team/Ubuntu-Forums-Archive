### Post by Lars Noodén on 2015-11-22
You probably mean the [Linux Terminal Server](https://help.ubuntu.com/community/UbuntuLTSP) which you can set up on Ubuntu as well as on other distros.  When it is running, it looks like a regular desktop and you can open as many programs and windows for those programs as you'd like, as long as the server has the capacity.  In a production environment it is best to have everything on the same gigabit LAN and to have two servers for redundancy.
