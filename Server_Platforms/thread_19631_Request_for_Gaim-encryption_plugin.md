### Post by dewey on 2005-03-13
Gaim-Encryption is NOT secure enough for sensitive information.  It was made to block casual sniffers from reading your information easily.  For example, if you're at an internet cafe, using their Wi-Fi, someone can just sit there, and sniff out all of your IM packets.  If in cleartext, they'll be able to read it easily, if in cipher, they'll have to decrypt it first, and many people simply won't bother with it.

It's not security through strength, it's security based on the laziness of the middleman.
