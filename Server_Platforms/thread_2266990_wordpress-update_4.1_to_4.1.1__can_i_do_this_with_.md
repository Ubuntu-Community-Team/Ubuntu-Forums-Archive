### Post by volkswagner on 2015-02-26
As @Habitual says, use the web frontend as an admin and run the upgrade.

Be sure to, ALWAYS, ALWAYS, ALWAYS, backup both the filesystem and database for the site, prior to any upgrades.

There's a great plugin for running backups, [UpdraftPlus Backup/Restore]("http://updraftplus.com/shop/updraftplus-premium/").
