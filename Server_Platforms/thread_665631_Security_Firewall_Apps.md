### Post by balaknair on 2008-01-12
Another page I strongly suggest you take a look at is [https://help.ubuntu.com/community/UnsafeDefaults](https://help.ubuntu.com/community/UnsafeDefaults)
[https://help.ubuntu.com/community/StrongPasswords](https://help.ubuntu.com/community/StrongPasswords)

If you're security conscious you should also get Clamav antivirus  and Firestarter firewall(use synaptic to install). they're both simple to set up and configure with GUI frontends, and add to your peace of mind.

Use Firefox as default browser with these two extensions-
NoScript([https://addons.mozilla.org/en-US/firefox/addon/722](https://addons.mozilla.org/en-US/firefox/addon/722)) and 
Adblock Plus([https://addons.mozilla.org/en-US/firefox/addon/1865](https://addons.mozilla.org/en-US/firefox/addon/1865))
for secure browsing. 
The ubufox extension (I think it's included by default with FF on Ubuntu 7.10) disables the default FF updater- not a good thing but you can use the ubuntuzilla scrip ([http://ubuntuzilla.wiki.sourceforge.net/](http://ubuntuzilla.wiki.sourceforge.net/)) to overcome that. But you should keep Noscript and Adblock updated. Other good extensions- 
secure login([https://addons.mozilla.org/en-US/firefox/addon/4429](https://addons.mozilla.org/en-US/firefox/addon/4429))
Flash block([https://addons.mozilla.org/en-US/firefox/addon/433](https://addons.mozilla.org/en-US/firefox/addon/433))

OK this is about all that comes to mind right now.
Hope it was helpful
