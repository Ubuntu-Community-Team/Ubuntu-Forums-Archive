### Post by jramshu on 2011-07-15
This is possible and a good start would be here.

This link is to 10.10 server, since that's the one I use. I use mine as a web server and to back up files from my laptops. There are other server versions available all with documentation on setting it up and some security. They discuss the different types of servers and their uses. It's quite the task for someone who doesn't know much command line stuff since the server tends to work better without all the gui stuff. But if you take your time, read the docs, and read up on navigating and manipulating directories and files you will be fine. Then there is always these forums that can help you if you have problems. 

[https://help.ubuntu.com/10.10/serverguide/C/index.html](https://help.ubuntu.com/10.10/serverguide/C/index.html)
