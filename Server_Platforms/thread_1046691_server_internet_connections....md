### Post by volkswagner on 2009-01-22
If you plan on running a hosting service, you may want to check out renting a virtual or dedicated server.  Places such as [Godaddy]("http://www.godaddy.com/gdshop/hosting/dedicated.asp?ci=9014&display=dedicated") have Linux dedicated boxes available.  This can allow you to keep the less expensive internet connection and still host.  You can also save on equipment for redundancy and backup power, etc.

I looked into this when my cable company (Time Warner) wanted to charge me an added hundred dollars per month for a static IP.  They are the only high speed provider in my area.  :(
