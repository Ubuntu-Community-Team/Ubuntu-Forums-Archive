### Post by windependence on 2008-06-24
It's not absolutely necessary to put the OS on it's own drive but it would be good practice incase things go wrong. I would say you could get by with 10 gigs for the OS. In actuality, it would be about a 4 gig install if you install everything under the sun, which you aren't going to do anyway. 20gig is probably as small as you are going to find anyway.
