### Post by HermanAB on 2011-03-12
Howdy,

If your 'server' has a keyboard, mouse and screen attached, then you should use regular Ubuntu.  The server edition is meant for headless servers hiding in dark corners of data  centres, that are managed remotely over SSH.

However, you could install webmin and configure it using a browser by going to [http://localhost:10000](http://localhost:10000)
