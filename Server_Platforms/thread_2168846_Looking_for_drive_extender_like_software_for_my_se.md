### Post by mcapaldo on 2013-08-19
mdadm is a software raid program for Ubuntu and is pretty straight forward to setup to pool drives.  You can install and use smartmon tools and a ssmp e-mail to send you e-mails on changes in your drives status (i.e. failures)

Or you can use webmin to get that GUI and more to admin your server box.  I use all of the above.

use this tutorial set for setting up mdadm, smartmon, mail relay, and webmin

[http://www.havetheknowhow.com/Configure-the-server.html](http://www.havetheknowhow.com/Configure-the-server.html)
