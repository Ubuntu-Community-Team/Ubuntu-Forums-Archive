### Post by tubezninja on 2008-11-20
Without putting the monitor into some kind of graphical mode, I don't think it's possible to change font size.  Plain old text mode pretty much relies on the graphics card's hardwire, and the default setting in most cases is 80 chars x 25 lines, regardless of the screen's resolution.

### Post by robertyou on 2008-11-20
actually...you can.

Install startupmanager:
```

sudo apt-get install startupmanager

```

run it:
```

sudo startupmanager

```
