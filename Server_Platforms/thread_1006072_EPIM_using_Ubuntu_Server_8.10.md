If you do a Google for EPIM, the first hit will probably be this one:

The SQL server that is needed is Firebird SQL at

[http://www.firebirdsql.org/](http://www.firebirdsql.org/)

That should be all :-)

Mind you: This is a solution for Windows clients using a Ubuntu server. For Linux clients, I would recommend using Kubuntu and the pim that comes with it.
