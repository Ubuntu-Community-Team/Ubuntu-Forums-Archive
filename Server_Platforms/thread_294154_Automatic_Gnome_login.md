### Post by Ziggurat on 2006-11-07
Possible scenarios:

If you have dual boot, then you could install a utility that reads/write ext, reiser, etc.. partitions on Windows or OSX.

If you have a bootable CD of linux (live CD), you can access any partition.


So, if you really want security you have to take more than just a password on your account, so it is safe to automatic login if you dont care this issues.
