### Post by SeijiSensei on 2013-11-23
Those are not logins.  They represent jobs being spawned by crond, the daemon that manages automated processes.  They are generated by entries in "crontabs" which are stored in /etc/crontab, /etc/cron.*/ and /var/spool/cron/root.  

Notice how the times are periodic.  One job runs every thirty minutes on the "nines"; the other runs at seventeen minutes after every hour.

Read this page for more details: [https://help.ubuntu.com/community/CronHowto](https://help.ubuntu.com/community/CronHowto)

This is a normal set of events; there is no security problem here.

### Post by bashiergui on 2013-11-23
