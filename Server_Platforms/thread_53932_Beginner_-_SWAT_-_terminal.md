### Post by XoloX on 2005-08-04
SWAT can be installed on a headless (*no GUI installed*) server. That's the way I'm using it. I'd recommend SWAT over webmin (another web based configuration tool, which also supports Samba) because once in a while webmin would trash my configuration file, while SWAT has no such problems. Also, from the SWAT web-interface, you can view active users/shares/files, which can be real handy.
