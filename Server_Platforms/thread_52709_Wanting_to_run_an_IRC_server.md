### Post by DJ_Max on 2005-07-28
Running an IRC server is no simple task. First you need to make sure the Datacenter your server is in allows IRC. If it does, protecting your server from abuse needs to be a top priority. [Google](http://dir.yahoo.com/Computers_and_Internet/Internet/Chats_and_Forums/Internet_Relay_Chat__IRC_/Software/Clients_and_Servers/) around for software. You will need to closely monitor your server, network wise also. As IRC servers are the main target of DDOS attacks, syn floods, etc.., which is why most DC's don't allow it.
