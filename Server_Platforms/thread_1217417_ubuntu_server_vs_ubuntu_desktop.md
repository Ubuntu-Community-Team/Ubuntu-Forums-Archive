### Post by Cheesemill on 2009-07-19
> **measekite said:**
> It appears that Ubuntu is new to the server arena where Red Hat is the leader in most of the larger commercial enterprises.
 
Anybody think that Ubuntu will get to the point where the business world will think they are comparable?
 
It depends on what you're using your server for.
I know alot of businesses that uses Ubuntu in LAMP, mail, and file-serving duties because they only need to run open source software. However, if you're going to run commercial software (Oracle for example) then you are going to use an OS and hardware that has been certified by the software developers (ie Red Hat).
 
So for Ubuntu to gain a larger share in the server market, it would have to aim for certification from the major software companies, something which Canonical isn't really pushing for at the moment.
