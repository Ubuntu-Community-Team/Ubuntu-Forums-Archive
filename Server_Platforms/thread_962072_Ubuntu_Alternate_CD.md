### Post by MJWitter on 2008-10-29
The Desktop and Server cd's are seperate.

The Desktop can come either as a Live CD which you can try before installing, or as the Alternate CD which is used mostly for more advanced installs.

The Server CD does not install a Graphical User Interface(GUI), but once installed it is possible to install a GUI through the use of commands.
