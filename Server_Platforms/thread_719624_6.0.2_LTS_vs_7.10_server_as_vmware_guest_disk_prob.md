### Post by JoeZilla on 2008-03-11
I am using a softraid on the host only. I originally thought that the 6.06 doesn't have the problems, but I stressed it a little more and I got the same error messages, just less, but still twice the disk performance compared to 7.10.

Here's something interesting: I put the following configuration entries in the vmware configuration to hopefully finally fix the clock skew:
[I]
host.cpukHz = 2300000
host.noTSC = TRUE
ptsc.noTSC = TRUE
[/I]

I haven't seen the disk problem ever since.

This make no sense to me unless the underlying cause was some strange timing issue. Or it's solar flares or it's going to crash again tomorrow.
