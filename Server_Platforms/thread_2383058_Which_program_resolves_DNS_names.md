### Post by SeijiSensei on 2018-01-20
If you only want the results, you can use "host -a ubuntuforums.org":

```

ubuntuforums.org.       3330    IN      SOA     ns1.canonical.com. hostmaster.canonical.com. 2017030602 10800 3600 604800 3600
ubuntuforums.org.       411     IN      A       91.189.94.16
ubuntuforums.org.       41681   IN      NS      ns1.canonical.com.
ubuntuforums.org.       41681   IN      NS      ns2.canonical.com.
ubuntuforums.org.       41681   IN      NS      ns3.canonical.com.

```

Using "host -a -C ubuntuforums.org" will recurse over the various nameservers and report the records each maintains.  Usually these should be all the same unless the domain owner has not properly configured zone transfers from the primary to secondary servers.


Another common DNS client is the program dig.  I can't find any combination of command-line switches to produce the output you posted though.  See "man dig" for the details.

### Post by SeijiSensei on 2018-01-21
Start with:

```

<?php
$host=$_INPUT['targethostname'];

# store the query result in the $dnsreply array
$dnsreply=array();
$dnsok=exec("host -a $host", $dnsreply);

if (!$dnsok) {
    echo "No records found for $host";
    exit;
}

$nlines=count($dnsreply);
$nslist=array();
$nsn=0;

# iterate over the lines in the reply parsing on the resource type
for ($i=0; $i++; $i<$nlines) {
     $lineparts=explode(" ",$dnsreply[$i]);

     $rrtype=$lineparts[3];
     
     switch($rrtype) {

          case "SOA":
              do stuff;
              break;

          case "NS":
             $nslist[$nsn]=$lineparts[3];
             $nsn++;
             break;

          case "A":
              $hostaddr=$lineparts[3];
              break;
      
          default:
              handler for records other than SOA, A, NS like CNAME
             
        }
             
}
?>

```

The host address will be stored in $hostaddr; the nameservers will be contained in the numbered array $dnslist.

PHP also has a DNS client function [dns_get_record()]("http://php.net/manual/en/function.dns-get-record.php") which I have not used myself.  It generates a numbered array consisting of associative arrays for each resource indexed by "type","ip", "class", etc.
