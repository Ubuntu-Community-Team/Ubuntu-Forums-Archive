### Post by sudodus on 2015-05-11
You can try Ubuntu Server. You can download iso files for both 32 bits and 64 bits. I suggest that you choose an LTS version. Start with 14.04.1 LTS.

If you try the 64-bit iso file and it works, your computer has 64-bit capability. Otherwise there will only be an error message and the computer will stop, so it is easy to find out.
