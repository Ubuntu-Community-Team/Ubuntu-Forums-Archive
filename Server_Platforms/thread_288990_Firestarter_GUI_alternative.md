### Post by skymt on 2006-10-30
You should ask a mod to delete the other thread for you. Regular users can't delete threads.

As for a firewall, there are a few alternatives to Firestarter. Sticking to GTK, there's [gnome-lokkit](http://packages.ubuntu.com/edgy/gnome/gnome-lokkit) and [fireflier-client-gtk](http://packages.ubuntu.com/edgy/net/fireflier-client-gtk) (make sure you install [fireflier-server](http://packages.ubuntu.com/edgy/net/fireflier-server) also). If you're willing to use KDE/QT applications, there are several other choices.

I have two questions, though: (1) Why do you need the Firestarter GUI to be open all the time (it works even when it's closed) and (2) why do you even need a firewall in the first place?
