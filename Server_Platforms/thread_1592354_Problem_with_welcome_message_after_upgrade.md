### Post by BkkBonanza on 2010-10-10
The message is generated by pam_motd module (referenced by /etc/pam.d/login) by using run-parts on the /etc/update-motd.d/ directory. It runs the scripts there in order and they cat output into /var/run/motd which is then linked to /etc/motd. 

Maybe there was a bug introduced into one of the scripts there. I haven't upgraded yet so couldn't comment on 10.10 (except I want the new kernel for my SSD arriving next week!)

You might look at /etc/motd.tail and make sure it's empty as anything there gets appended at some step, probably last.

Edit: checked mine. It looks like /etc/update-motd.d/10-help-text is being run twice, once before and once after the landscape stuff.
