### Post by steve.horsley on 2007-11-07
There are very few Linux viruses, and I don't think there are any out there in the wild that still pose a risk to Linux systems. On that basis, you don't really need a virus scanner (yet). But if you exchange files/emails with windows users, you may want to make sure you don't pass infected files or attachments on.

You may want to run a rootkit detector occasionally though. These look for anomolies that an installed rootkit might leave.
