To the OP: Depends on how old these computers are. To run a cluster, you'll need two basic things: ethernet NICs (easy to get), and at least an i386 CPU in each machine. If these old computers have those then, yes, you'll be able to cluster them as per Herman's suggestion. 

The other thing that could be a problem: some BIOSes will fail if not connected to a monitor and/or keyboard. Other than that, if you're willing to read up, this should work great.
