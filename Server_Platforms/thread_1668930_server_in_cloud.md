### Post by kim0 on 2011-01-20
Hi!
You need at least 2 physical servers (cloud controller + node) to build your own private cloud. After that, you can run as many virtual servers as you have RAM and compute power. Typical values would be around 8-32 virtual servers per physical server (node)

There is a dedicated cloud forums. Please post future questions there
