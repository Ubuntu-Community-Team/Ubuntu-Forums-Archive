### Post by bluefrog on 2008-12-09
installing vim and having a look at vimtutor would free you from a lot of hassle.

vim vs vi alone because the arrows keys works differently in both and are more friendly in edit mode in vim.

you could use nano also but you will need a bit of learning the help file to find it useful on a large text file.

vi being at 99% (not to say 100%) installed by default on any unix linux system, knowing your way with it still seems the wisest move if your are running a server.
