### Post by SeijiSensei on 2013-05-12
If you have control over the server's configuration files, you are better off using the [Redirect]("http://httpd.apache.org/docs/current/mod/mod_alias.html#redirect") directive in the configuration file than futzing with URL rewriting.  You cannot use Redirect in an .htaccess file, but you can place it in the server's virtual host configuration in /etc/apache2/sites-available/.

Can you give us the URL(s) you want to redirect and the URL(s) to which you want them redirected?
