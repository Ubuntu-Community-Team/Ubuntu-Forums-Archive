### Post by brian mcgee on 2010-03-20
Once I ran into similar trouble. Run ```
md5sum /path/to/ubuntu-9.10-server-whatever.iso
``` and make sure the output matches the corresponding entry here: [https://help.ubuntu.com/community/UbuntuHashes](https://help.ubuntu.com/community/UbuntuHashes)

Turns out the file I downloaded was corrupted. I downloaded again, burned a new disc, and it worked as expected. :)
