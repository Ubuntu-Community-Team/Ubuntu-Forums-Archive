### Post by cariboo on 2009-12-02
It depends on how you have the server setup. Do you have a monitor and keyboard connected? If so just type your user name at the prompt and enter you password when asked.

If you access your server remotely via ssh, then you don't need to log on to the server, like you would a desktop computer, just connect like you normally would using ssh.
