### Post by deadflowr on 2022-06-06
Try with --no-install-recommends
rmlint recommends the rmlint-gui package.
The default setup is for apt to install recommended packages. They are treated more or less as dependencies.
```
sudo apt install --no-install-recommends rmlint
```
