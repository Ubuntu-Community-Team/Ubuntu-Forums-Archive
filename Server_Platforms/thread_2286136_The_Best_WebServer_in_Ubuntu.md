### Post by QIII on 2015-07-11
Nginx is pronounced "Engine X", so instead of a LAMP server it's called a LEMP server.

I use Nginx.

If this is something you plan on exposing to the web, then I suggest you hold off for a while and set up a home server first.  Learn everything you can about security and optimization of your chosen setup.  Ask a lot of questions.  Learn to do penetration testing (but don't ask questions about that here since we don't allow those discussions).

And remember:  running a server is not an hour of setup and then ignoring it.  It takes days and weeks to get things tightened up and secure.
