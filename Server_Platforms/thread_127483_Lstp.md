### Post by fnjordy on 2006-09-09
I've been working on some stuff to help out this situation:

[http://developer.novell.com/wiki/index.php/miru_directory_server](http://developer.novell.com/wiki/index.php/miru_directory_server)

Basic requirements:

* internet connection, e.g. DSL
* router, e.g. [m0n0wall](http://m0n0.ch/wall/).
* NAS, e.g. [FreeNAS](http://www.freenas.org/).
* directory server, e.g. [Miru Directory Server](http://developer.novell.com/wiki/index.php/miru_directory_server).
* server, e.g. regular PC with Ubuntu.
* thin client or cheap regular pc for each desktop.

There's a VMware demo you can download and play about to get used to how everything works.
