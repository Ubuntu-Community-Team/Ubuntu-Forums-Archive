I've got the 8.10 server and control it from Webmin. Webmin Command line does dot offer dialogue i.e. if the system asks for a response to a command Webmin does not allow this.

How do I get command line access from my laptop on the local network? You've probably guessed I do not have a permanent console or keyboard attached.

---

### Post by kay-man on 2009-05-05
You should be able to get a command prompt using ssh, provided your server runs an ssh server.

If your laptop is also running linux you should have an ssh client installed. If it's running windows you'll need to install one, for example [putty]("http://www.chiark.greenend.org.uk/~sgtatham/putty/download.html")
