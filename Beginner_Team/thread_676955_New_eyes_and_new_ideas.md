That is not reallly a topic that comes up...

In Ubuntu, the Debian packages are used to install programs normally. (Go to [http://www.opera.com](http://www.opera.com) and download the Opera 9.50b and you will see how it works) These are called ".debs" and are like the installers of Windows. Double click, give password, and you are set.

You download things the same way, and depends on your browser. If you wanted to run a file you downloaded in Linux, you have to mark it as executable. (right click, Properties). The command for this is "chmod +x <filename>"

You can copy and paste text into the blog, but you can't, on any client, use Word documents for web pages.
