### Post by igknighted on 2007-05-03
We have decided to use a link in our signatures for identification.  Please copy this code and add it to your signature (you get to pick the color ;)) if you are a beginner team member.  Remove the space in "[ url=...", as is only there so you can read it here and add in the color you want.

```

[ url=http://ubuntuforums.org/forumdisplay.php?f=215][color=]Beginners Team[/color][/url]

```

This is mandatory for experienced team members, optional for newer/less experienced members.
