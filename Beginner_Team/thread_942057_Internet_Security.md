### Post by patrickballeux on 2008-10-08
It is in the repositories of Ubuntu.

If you open Synaptic (System-Administration-Package Manager Synaptic), you can search for Firestarter by it's name.

There are other firewall available, but I find this one easier to configure.

Remeber that installing softwares should be done from Synaptic (or apt-get from the command line).  It is not the same as in Windows where you download a setup from somewhere you don't know.  The files in the repositories are checked and validate to be safe.

There is a more friendly way to install or remove a software using "Add/Remove" from the "Application" menu, but I find Synaptic more complete when you are looking for a special software...

Good luck!
