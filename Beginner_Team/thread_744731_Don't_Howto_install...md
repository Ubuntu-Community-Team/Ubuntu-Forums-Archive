### Post by Michael.Godawski on 2008-04-30
When you install stuff via Synaptic with the command

sudo apt-get install something

you need a working internet connection. The advantage of this installation method is that all dependencies will be installed automatically with the actual package. So you do not have to search them by hand. 

[Basic Installation Methods]("http://godawski.piranho.de/installingapps.html")

Dependencies are packages which are essential for a specif application to run. They are the base of the software, so to speak.
