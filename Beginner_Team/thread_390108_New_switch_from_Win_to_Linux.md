### Post by NJC on 2007-03-21
I landed in this forum via the [The Beginner Team has started ]("http://www.ubuntuforums.org/showthread.php?t=378426") thread over in Absolute Beginner Talk. I am about 3 weeks into Linux and it has been a steep learning curve to transition from my Windows Ways. Like most who are beginners, I've spent countless hours scouring through Google, Ubuntu docs, this forum et al. This is the info I'd like to see for new Linux folks:

1/ Drive designations. Specifically how Linux drive nomeclature of hda1, hdb2 and hd0,0 works. 

2/ How to Dual boot in Win. The percentage of users who are going to dive head first into Ubuntu and ditch Win completely would be small IMO. 

3/ Folder/directory naming conventions. /lib /bin - how does this translate to Win?
[https://help.ubuntu.com/6.10/ubuntu/desktopguide/C/directories-file-systems.html](https://help.ubuntu.com/6.10/ubuntu/desktopguide/C/directories-file-systems.html)

4/ Ensure the Official Docs ([https://help.ubuntu.com/](https://help.ubuntu.com/)) are easy to find. This is a great resource but somehow I found hard to find ... or rather easily overlooked. 

I found the following link today for the first time. This needs to be front and center!

[https://help.ubuntu.com/community/SwitchingToUbuntu/FromWindows](https://help.ubuntu.com/community/SwitchingToUbuntu/FromWindows)

------------------
Thanks for all of the work and support to help keep this community going!
