### Post by labinnsw on 2010-04-25
If you are in a hurry, you can go ahead and install Ubuntu 9.10 it is the current stable version. The next release 10.04 is due on 29 April though. You could wait for that. It will be the most stable of all the stable releases for the next 2 years.

If you do decide to install version 9.10, the update manager will let you know when the next release is ready. To upgrade, you will only need to choose to do so and follow the prompts.
