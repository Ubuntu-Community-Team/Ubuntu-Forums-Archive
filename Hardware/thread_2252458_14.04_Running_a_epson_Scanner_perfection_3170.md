### Post by lindenb on 2014-11-17
Complete thread at: [http://lists.alioth.debian.org/pipermail/sane-devel/2014-November/032869.html](http://lists.alioth.debian.org/pipermail/sane-devel/2014-November/032869.html)
  what I've done:
  
[LIST]
[*]uninstall the sane/xsane/... packages installed by ubuntu 
[*]clone sane-backend from git & install
  ```
./configure --enable-libusb_1_0  --disable-translations --prefix=/usr --sysconfdir=/etc


``` 
[*]clone sane-frontend  from git & install
  ```
./configure  --prefix=/usr --sysconfdir=/etc


``` 
[*]edit /etc/sane.d/dll.conf : comment out epson and epson scan. Add epkowa that was missing in this file. 
[*]the file /etc/sane.d/epkowa.conf was missing . I've added the usb-id for my scanner 
  ```
$ grep -vE '^#' /etc/sane.d/epkowa.conf 

  scsi EPSON
  usb
  usb 0x04b8 0x0116

``` 
[*]install iscan and its plugin from the rpm available at epson 
[*]reboot 
[/LIST]
