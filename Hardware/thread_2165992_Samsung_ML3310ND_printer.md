### Post by lammergeir on 2013-08-07
I figured it out, with help from the web. Samsung 3310ND printer - download latest version 1.00xx driver from Samsung website.  Extract and search for PPD file for printer in "Share". Proceed with installation from System Settings.  Works fine now.. Nice network printer for my family who run Windoze. No need for "bucket and spade" via terminal. :lolflag:
