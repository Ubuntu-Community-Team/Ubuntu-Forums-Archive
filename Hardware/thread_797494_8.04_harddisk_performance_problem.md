### Post by sleepingdragon on 2008-05-17
I had this problem on an old desktop. Can you try this command for me and let me know the results? It will set the DMA on and should keep the settings.

sudo hdparm -k1 -d1 /dev/sda2

Then try the hdparm speed test

sudo hdparm -Tt /dev/sda2
