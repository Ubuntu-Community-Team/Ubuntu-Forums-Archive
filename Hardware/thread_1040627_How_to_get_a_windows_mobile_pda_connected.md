### Post by ldrn on 2009-01-22
I do not have that model, and mine supports file system mode for its microsd card, but I run an FTP server on your phone when I want to get full access.  Mocha makes a free (as in beer) one:
[http://www.mochasoft.dk/freeware/ftpd.htm](http://www.mochasoft.dk/freeware/ftpd.htm)

There's also these two, which I haven't tried:
[http://forum.xda-developers.com/showthread.php?t=318313](http://forum.xda-developers.com/showthread.php?t=318313)
[http://www.cam.com/vxftpsrv.html](http://www.cam.com/vxftpsrv.html)

The first one is also free, the second is pay.

It's a bit irritating to run it every time I want to transfer files, but not too much. I could keep it running in the background, but... * shrug *  

I like to play with the internals of my phone, too, so I sometimes do this on Windows so I can remotely mess around with /Windows directory on my PDA.
