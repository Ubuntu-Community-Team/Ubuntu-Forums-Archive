### Post by Peter_W on 2008-06-25
We've been trying to install Ubuntu on a Samsung R60+. Looking on the internet, there seem to be quite a few others with problems doing this on this particular laptop. Initially the graphical interface wouldn't load at all, but on checking the internet we found that most people thought the problem was with the graphics card (AMD Radeon X1250), and that the problem could be solved by downloading updated drivers for this.

However, none of the drivers downloadable at [http://ati.amd.com/support/driver.html](http://ati.amd.com/support/driver.html) are for the X1250. We installed drivers for the X1300 Series (as 1300 was the closest number to 1250...), which caused Ubuntu to run fine up to the log-in screen - after logging in, we got the "white screen of death".

So, to anyone who has resolved this problem, which drivers did you install to get the graphics card working properly? And does anyone have any other suggestions for getting Ubuntu to work properly on this laptop?
