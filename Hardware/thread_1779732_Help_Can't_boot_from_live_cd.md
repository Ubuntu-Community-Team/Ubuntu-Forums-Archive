### Post by ratcheer on 2011-06-10
Do you know how to get into your BIOS settings? You usually have to press a particular key or key combo after the system starts running but before it starts booting the OS. If you get into that, there should be a setting called something like Boot Device Order. You would move the CDROM to the top, save the settings, and reboot.

### Post by dandnsmith on 2011-06-11
It's fairly common to allow temporary change to boot order (in order to boot from CD or USB) by using F12 at boot. There's usually a message, but this can be difficult to see - manufacturers manual can help here.
