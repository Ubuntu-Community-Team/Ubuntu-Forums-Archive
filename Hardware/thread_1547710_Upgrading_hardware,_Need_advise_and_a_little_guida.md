### Post by vanquishedangel on 2010-08-11
LOL, I have all the parts and they are working fine, To answer my own question, LOL. After the install of a new Processor even with all the updated bios, and even if you have the microcode updated, you still may need to update the microcode for the new processor, even if it is the same version as you already have. I believe that this is because the update is for the old processor and needs to be redone for the new one.

For this model HP and previous models I belive, The microcode update only works from a floppy according to many forums, and that is all hp had available, also it was only available for windows. If you are having difficulties installing a floppy into this model look it up on-line how to install, but be warned it can be a headache.

I used another computer with windows xp installed to make the floppy disk(had to find them stored away and clean the dust off) and booted from the floppy to update the microcode, the bios maybe updated from a disk but that doesn't update the microcode. Also the intel microcode updater in ubuntu doesn't update the microcode in the bios either.
