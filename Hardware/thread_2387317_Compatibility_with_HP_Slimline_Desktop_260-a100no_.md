### Post by Mark Phelps on 2018-03-19
> **juzzlin said:**

[https://support.hp.com/emea_africa-en/product/hp-slimline-260-a100-desktop-pc-series/12079697/document/c05313991/](https://support.hp.com/emea_africa-en/product/hp-slimline-260-a100-desktop-pc-series/12079697/document/c05313991/)

There have been some serious issues with some HP desktops in the past so I'd like to be sure before purchasing.

The main problem with "slimlines", as there are with laptops, is that HP contracts out the Motherboards and ends up with versions that often need special drivers -- often for which, there are no Linux equivalents.

Their "standard" desktops tend to use more generic Motherboards and other hardware, and those are better news for using Linux.

As already said, the only way you will know for sure is to run Linux on it in Live Mode -- and you're unlikely to be able to do that prior to purchase unless you can find a retail store that has one and will let you boot it from a USB stick you have made.
