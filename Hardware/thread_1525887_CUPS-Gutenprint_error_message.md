I am getting the following Gutenprint error: Warning: Quadtone inkset only available in MONO when I print to my Epson Stylus R220 printer. The print output colours are all wrong. Can't find how to reset Gutenprint or anything.
64 bit Lucid CUPS 1.4.3 Gutenprint v5.2.5  Dell Dimension C521.
Any idea what is causing this error?

---

### Post by awam66 on 2010-07-08
I hate to reply to my own message, but just in case anyone else gets the above error it was because in the printer configuration the Ink Set had been set to MIS Quadtone. I don't remember ever setting that but I suppose I must have for some unknown reason. Set it back to Epson Standard Inks and all is well
