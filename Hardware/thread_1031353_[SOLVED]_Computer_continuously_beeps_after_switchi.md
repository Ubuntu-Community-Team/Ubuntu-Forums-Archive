### Post by ebmelle on 2009-01-05
The symptom generally signify memory (RAM) error.  Check to see that the memory sticks are properly seated. If they are, remove one stick from slot #2, to see if that was faulty. If problem persist, try the removed stick in slot #1.
Remember to disconnect the computer from any power source prior to touching the RAM sticks.
Cheers :)
