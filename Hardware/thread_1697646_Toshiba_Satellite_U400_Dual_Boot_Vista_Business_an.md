I had **no dramas** at all installing Ubuntu 10.10. It has been smooth sailing!

However Vista has given me some dramas with the boot menu.
I entered Vista as you would expect and it took me to Windows Recovery Environment.
After a few attempts of it never finding an issue I shut down and entered the Vista Windows Recovery Environment to find that Vista was working "perfectly".

This is a note for anyone to know that Vista is hiding in the boot menu.
