### Post by corsair67 on 2011-11-05
After a few weeks of frustration I stumbled upon a solution that has worked for me on multiple systems using different ati radeon cards.  I disconnect the secondary monitors. Do a clean install applying updates with the install (one of the first check-boxes during the install process when ubuntu checks your system).  After the install I choose not to use proprietary drivers. When the new system is up and running, plug the monitor cable into the monitor. That's it, it automatically recognized the monitor and made it an extended desktop.  I have done this with laptops and desktops, applied all updates that ubuntu suggests and it still is working on all machines.  I still have not yet applied proprietary drivers and all is working great.  Why this works I don't know but when I install with both monitors attached no success, with only one at install works every time.  Hope this helps.
