### Post by pbpersson on 2009-04-28
I built a computer recently - got the parts from TigerDirect.

The motherboard was an ASUS M2N68-AM, the CPU was an AMD Athlon 64 X2 4200+.

With the Motherboard, CPU, CPU cooler (I got the Thermaltake Max ORB) and 4GB of RAM you will be under $200.  The newer motherboards do not support IDE drives so get a one terabyte SATA drive for $100 and you will be all set.

The video, sound, and network controllers are integrated in the motherboard.

However, you can get a brand new nice computer for $400 and you will not need to build anything.

### Post by PatrikG on 2009-04-28
I just built my first computer from scratch using a Compucase 8K01, Intel D945GCLF2 MB, 2GB RAM, Samsung SN-T083A slot-in dvd and a SATA HD. It might be limited for high-end games, but if you or someone is looking for a small size computer then I definitely recommend it.

There are better mini-ITX motherboards out there if you want to get more CPU power, but I'm happy with it so far. Running Ubuntu 9.04 hasn't been any problem, except for the video drivers which needed a backport. They worked well with 8.10 however.
