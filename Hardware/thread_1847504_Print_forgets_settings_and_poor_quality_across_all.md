### Post by Alistair George on 2011-09-20
Hi I am doing a review on a Brother MFC-795CW. Had been blaming it for its very inconsistent print quality, but when I tried print from Windows, the print was excellent.
Print settings can be set in CP and they are forgotten in Print Dialog. But whats worse is that after one print, the previous dialog is forgotten as well.

This is using CUPS and I have proven this issue on several variants of Ubuntu. Even with Kubuntu the print and settings problem are consistently bad. Used several different PC across a network, tried variations of interface eg USB/Wireless/Ethernet.

First picture is a scan of Ubuntu print, note the bleeding on suit. Next is Windows print. The scan degrades the pictures but you can see what I mean.
Its also same performance with an HP printer; inconsistent dialogs and print quality when using photo setting. Normal paper setting/lo resolution is no problem for either machine.
This is an important issue, as I'd like to be fair in this review. 
Kind regards,
Alistair.
