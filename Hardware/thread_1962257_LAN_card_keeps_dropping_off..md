### Post by ahallubuntu on 2012-04-20
Ubuntu 10.10 is no longer supported.

No one is forcing you to use Unity in the newer versions.  You can go back to Ubuntu 10.04 LTS (supported for another year) or move up to 11.04 (only supported for another six months) and use Gnome 2 natively in either one of them.  Or make the big move to 12.04 beta (official release shortly) and use Gnome 3.  In any case, I'd definitely try a newer release and see if your LAN problem goes away.

### Post by ahallubuntu on 2012-04-20
There is a small chance that 10.04 LTS will work better than 10.10.  Sometimes issues become introduced in the newer versions that never get properly fixed.  The LTS release may get more attention to fix certain nagging problems (the server branch of 10.04 will be supported until 2015).  If you already have a 10.04 LTS CD, at least you have the option to try it and see if the network connectivity issue persists.

Ubuntu 11.04 still includes Gnome 2, not Gnome 3.  I have put 11.04 on a number of systems and it works quite well.  The first thing I do, if Ubuntu decides the hardware is new enough to use Unity, is to revert it to always login to Gnome 2 instead.  After that, it looks almost exactly like Ubuntu 10.10 .
