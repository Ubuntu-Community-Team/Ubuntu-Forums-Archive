### Post by TheFu on 2013-09-06
I think this is purely a marketing gimmick.  Which OS runs shouldn't matter .... unless there are firmware updates that can only be applied using WinXP.

The projector has a native resolution and update rate.  Ubuntu can almost certainly output that, especially if it is anything standard.  I've connected my Ubuntu-based XBMC machine to a low end projector and only had issues with HDMI signals being lost about every 20 minutes. The same issue happened from a DVD player, WD TV Live HD and from a Roku, so that issue wasn't Ubuntu specific.  The projector also had VGA inputs, which worked from the computer.
My projector has internal SD storage, which is seen just fine by Ubuntu through a normal USB connector.

So you need to read the manuals and find how firmware updates are performed.  I suspect most projectors will use a USB flash drive for this - so the OS doesn't matter there ..... *most of the time.*
