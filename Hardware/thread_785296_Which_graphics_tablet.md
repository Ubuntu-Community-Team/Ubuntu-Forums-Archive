### Post by neko18 on 2008-05-08
Differences between Bamboo and Bamboo Fun:
```
                   Bamboo     Bamboo Fun
Stylus features    Pen only   Pen and eraser
Sizes available    Small      Small, Medium
Colors available   Black      Black, Silver, White, Blue
Includes mouse     No         Yes
```

I personally suggest the Bamboo instead of the Bamboo Fun. I only occasionally use the eraser and almost never use the mouse.

My Bamboo has a Pen / Eraser combo stylus. The mouse is not necessary, as you should already own one of these, or use the trackpad or get really good at using the tablet to navigate.

I do recommend the Wacom tablets. There are others that can work with Linux, but they require a lot more steps and have a lot less support. For years the main thing holding me back from Linux was that I owned an AceCad Flair USB and I could only use it in Windows and WINE wasn't up to snuff.

I love my Bamboo, and I find it useful for my projects.
