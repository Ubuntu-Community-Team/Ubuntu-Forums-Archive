I'm also using the nVidia 9600GT, I recomend you just upgrade to Ubuntu 8.10. (Or higher)

After 8.10 Ubuntu has improved compatibility with nVidia cards, and if you don't want to install 8.10+ but still want a easy way:
```
sudo apt-get install envyng-qt envyng-core
```

And then run envy to install nVidia drivers.
