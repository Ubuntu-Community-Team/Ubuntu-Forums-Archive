### Post by Bucky Ball on 2016-07-18
From [here]("https://www.gpio.com.au/flirc/"):

> Works with **any** Remote

Because the FLIRC receiver is programmable, you do not need a programmable remote control. Just plug the FLIRC into a Windows, Mac or Linux computer and run the supplied software to program your existing remote control keys to your desired keyboard commands!

I suggest you do a search for 'flirc which remote' and you will find a ton of info if you dig through it. Not sure if this makes things any clearer, but Flirc and the remote you are using communicate directly, there is no input from the computer, as such. The Flirc itself is programmed, NOT the device you are using it on.

As far as I know all TV remotes are IR (infra-red). That is the universal protocol. Others with more technical knowledge may be able to give more info.

The Flirc site is not the only place to buy one. Have a look around (they are available from at least two sellers I know of here in Australia directly from the store).
