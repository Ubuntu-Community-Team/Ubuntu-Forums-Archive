If it's a USB keyboard, enable it in BIOS; look for options similar to: "Legacy USB support", "USB keyboard support", etc; you will need to use a PS2 keyboard when setting this option; subsequently everything should be OK. I can't be more specific because the exact option varies from BIOS to BIOS.
