### Post by nicedude on 2008-05-21
Definatly a hardware issue if it occurs in multiple OS's so I would suggest you start figuring it out by trying plugging some (known good) headphones into the output jack and see if it still persists. If so then when you experience the problem I would try gently spinning or wiggling the connector to your sound card devices output and see if it changes the output you are receiving ( if it does then its the jack itself that has problems you can try hitting it with duster canned air but they may not fix the problem if it isn't dirt particle related. If its the jack ( or if your not sure if it is the jack ) then maybe your computer has 2 sound outputs ( Like a headphone jack on front ) so you might be able to dodge this by using the alternate port. If that does nothing to change the behavior then it is the sound chipset more than likely and you will need to get a new soundcard to fix this. If its a laptop you can get a USB one, if its a desktop I would get a PCI one and if you are uncomfortable opening you PC case to work on it I am sure you have a friend that can help since adding a PCI card is an easy job.

Sorry to hear about your troubles and hope the info helps.
