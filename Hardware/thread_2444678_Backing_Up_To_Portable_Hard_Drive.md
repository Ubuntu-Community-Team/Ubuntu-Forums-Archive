### Post by TheFu on 2020-06-02
Does the external HD have NTFS as the file system?  if so, then the options are limited.  Start by formatting to ext4.

Next, what do you want to backup and how easy does the restore need to be?  Easy means that more storage is necessary and fewer versions can be retained and the restore won't be as flexible.

There are lots of threads here about backups. Have you looked through those?

The model of the external drive matters for performance.  Some have really bad performance - all 2.5 inch drives, for example. With external power, those disks can be almost as fast as a 3.5 inch. 3.5 inch drives with external power sources are very fast over USB3.  Flash media tends to be really slow.  I&#8217;ve destroyed expensive 64G flash storage in 1 yr of daily backups.
