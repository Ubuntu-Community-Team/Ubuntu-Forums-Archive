### Post by pi/roman on 2010-02-05
to Antony,
if you are having the same problem of too high of a resolution on the display a simple solution is to add a virtual command to your xorg.conf. If you do not have an xorg.conf, here is how to make one, in terminal type:
sudo X :1 -configure

It will place xorg.conf.new in your /home/(username)/ folder, so to edit it type:
sudo gedit ~/xorg.conf.new

then open another gedit window for the main xorg.conf file:
sudo gedit /etc/X11/xorg.conf

now you have both files open, if xorg.conf is empty just copy everything from xorg.conf.new into it, then look at the "screen" section of xorg.conf. There should be some subsections marked "display" similar to the first post. Inside each display section place the following line:
virtual 1280 768

That should force your laptops resolution to 1280 768 without causing any errors.
