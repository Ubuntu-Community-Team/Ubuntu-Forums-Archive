### Post by zeronothing on 2009-01-02
you are correct about the the overhead issue with regards to using swap space more. after looking up your particular PC the maximum amount of ram it will take is only 256mb. one website to look at to see how much ram your PC can take is [[COLOR="Blue"]Crucial[/COLOR]]("www.crucial.com"). Crucial is memory/flash memory manufacturer. On the other hand with regard to the operating system, the maximum amount of ram an operating system can handle depends on the architecture. The architecture I'm talking about is 32-bit operating system or a 64-bit operating system. a 32-bit operating system will be able to handle up to 4gb of ram in theory but its usually a little bit less. in a 64-bit operating system it can handle up to an amount that is out-of-this-world. basically at this point you couldn't really fill a desktop computer with enough ram to max out the computer with a 64-bit operating system. well I hope this utterly long post helped you..
