### Post by BinaryJava on 2018-06-03
Solved it,
1. For latest ubuntu lts 18.04 you can download gnome tweaks from the ubuntu software centre and mouse acceleration is included and can be switched off
2. Downloaded openrazer drivers and installed polychromatic, changed dpi to 400
3. I found that cs go takes raw mouse input so desktop sensitivity makes no difference, I did find that a sensitivity of half way is very similar to windows 6th notch, measure distance mouse travels from one side of screen to other in both oses to check
4. Not much I could find on this but I am pretty sure none exist since the game uses raw input
