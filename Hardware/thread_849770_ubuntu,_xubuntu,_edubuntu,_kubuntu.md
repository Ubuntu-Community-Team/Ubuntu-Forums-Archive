### Post by ProbablyX on 2008-07-04
> **JafaarNhh said:**
> can i have ubuntu, xubuntu, edubuntu, kubuntu in same harddisk:confused:

The main difference between these is that they use different display managers (interfaces).

If you mean that you want to choose between KDE, XFCE, Gnome etc there are packages you can install to have both KDE, XFCE, Gnome etc and choose between them before logging in.

For KDE3 (Kubuntu) install:
kubuntu-desktop

For KDE4 (latest high-tech Kubuntu) install:
kubuntu-kde4-desktop

For XFCE (Xubuntu):
xubuntu-desktop

For Edubuntu (standard);
edubuntu-desktop

For Edubuntu with KDE:
edubuntu-desktop-kde


You can install these from Synaptic or Adept.

After installation, logout and there will be a menu called "Session" from which you can choose which interface to use.
