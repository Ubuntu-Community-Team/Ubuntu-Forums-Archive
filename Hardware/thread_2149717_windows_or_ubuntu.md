### Post by Mark Phelps on 2013-05-29
I run an (old) XP system with a mix of 1TB, 1.5TB, and 2TB drives -- and they all work fine.

I suspect that your drive problems may be related to size if the BIOS doesn't handle large drives well.  That will be the same situation, regardless of which OS you have running on the PC -- the OS can't override the BIOS in device handling.

And, as others have mentioned, you really need to boot using an Ubuntu LiveCD to see how well it handles the hardware -- including the video chipset.
