### Post by greyfox1 on 2009-11-07
Ubuntu has a lot of online help documentation outside of this forum. Try this page: [https://help.ubuntu.com/community/Mount/USB](https://help.ubuntu.com/community/Mount/USB). These are instructions on things you can try, although you shouldn't need to do all this every time you use a pen drive :).  

I'll tell you now that it is VERY important that you eject the drive before unplugging it (click the eject icon in Nautilus). If you don't, you'll just keep running into more problems.
