### Post by agillator on 2011-10-27
Have you tried a different set of headphones? I don't know how your computer does it, but the 'muting' of the speakers is often a matter of the wiring of the jack. As you insert the plug it causes the jack to physically break the circuit to the speakers and establish a circuit to the headphones. If you do not have the problem with a different set of headphones then you have a problem with the headphones. If it does, check with a repairman and see how your computer is built unless you can physically see the jack and how it is built.

As to not having root access, when you installed your system you gave it a root password. That allows you to use sudo to run a command with root privileges. You rarely need to actually be root. If you have run into one of those situations, consider whether you really need to activate the root account. Remember, there are two types of people when running as root - those who have and those who will make a total reinstall necessary. If you really need to run as root, give the root account a password (see the man page for passwd). Then you can su into it, or log out and log in as root.

As to the repositories, what problem are you having? Your friend loading a bunch of repositories means absolutely nothing. What is it you have not been able to install and how have you been trying to install it?
