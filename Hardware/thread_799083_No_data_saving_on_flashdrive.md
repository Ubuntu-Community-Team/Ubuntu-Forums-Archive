### Post by seth1991 on 2008-05-18
I am running Ubuntu 6.06. Whenever I plug my jumpdrive into the USB port, it is immediately recognized and a file window with all of my files on it pops up. When I attempt to drag any files into the folder, it works, however as soon as I unplug it and replug it in, the files are lost. How do I fix this? It has happened on multiple drives.

Thanks!

---

### Post by teaker1s on 2008-05-18
on panel right click
select add to panel
select disk mounter

copy files to drive and then use disk mounter to unmount the drive and it should write to drive as it unmounts
