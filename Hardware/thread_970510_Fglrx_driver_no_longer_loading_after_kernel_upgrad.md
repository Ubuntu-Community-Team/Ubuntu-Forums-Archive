xorg-driver-fglrx is installed and the newest version.

### Post by hasanen on 2008-11-13
Solved , I don't know exactly how it worked ,but I removed the first installation of fglrx completly with the package manager , then removed the module from dkms [dkms remove -m fglrx -v 8.xxx]
Then I renabled the ATI driver from the Hardware Drivers ...and it worked after rebooting :D

I think the issue was in the dkms , after installation dkms -status must report something like this : 

fglrx, 8.543, 2.6.27-7-generic, i686: installe
