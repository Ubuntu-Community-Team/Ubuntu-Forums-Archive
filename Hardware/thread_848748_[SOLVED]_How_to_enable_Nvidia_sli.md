### Post by Enlitend on 2008-07-03
Hi,

To enable SLI:
```
sudo nvidia-xconf --sli=On 
```

The valid values for SLI are "Off", "On", "Auto", "AFR", "SFR", "AA", and "AFRofAA".

Try 
```
nvidia-xconf --advanced-help
```

for a complete list of option that you can configure, just be sure to know what you are doing:)
Hope this help.
