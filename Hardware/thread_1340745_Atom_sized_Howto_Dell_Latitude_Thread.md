### Post by opt1k on 2009-11-28
Hello, I hope this helps some people out. 
This is specifically for the Latitude CPi 366. But it covers a lot of the hardware and configuration that older laptops of this line have as well.
Please post any additional information you have for this laptop or line if you own one.
This is very basic information for now but it should be enough to help someone make this system more usable than the $99 ARM netbooks being old on ebay ;)


Part 1.  Software.
Believe it or not this old slow little laptop runs pretty well, with the right software.  My recomendations are as follows:
*** 1.  When installing the system download the Ubuntu SERVER iso and install only the base system.
*** 2. Install lubuntu-desktop meta package (or lxde-desktop/lxde) meta package.
*** 3. This system struggles with firefox.  But does not struggle with opera web browser.
*** 4. Dont waste your time with flash.  If you want to watch flash movies google for the youtube.sh script.
*** 5. For a media player gmplayer appears to work best.
*** 6. Stop and disable every non essential service that may be running
*** 7. Comment out extra tty#'s in /etc/inittab
*** 8. Recommend icedtea package over the regular sun java.
*** 9. Do the 256mb memory upgrade ;)
*** 10. Do not use openoffice.  Use abiword and gnumeric instead.
*** 11. Try alsaplayer to play mp3 files.
*** 12. pppdialer = wvdial or gnome-ppp front end
*** 13. gprs evdo 3g cecllular acces load module cdc_acm, connect with wvdial. or gnome-ppp front end
*** 14. attractive looking lightweight terminal = aterm
*** 15.  Recommended games.
          1. ace-of-penguins
          2. supertux
          3. The Mana World
          4. bastet tetris
          5. Rick Dangerous
          6. wordwarvi
          7. OpenTyrian
          8. Ninvaders
          9. Kobo Deluxe
          10. Freeciv





Part 2 --- Configuration

For now all I have is an xorg.conf and a list of modules.
To have the modules loaded at boot if they are not auto-detected.  

$ sudo nano -w /etc/modules

I am not using yenta_socket because I have no use for it. And the pcmcia interface appears to be physically damaged on this ancient laptop.
**hint-- unload any modules you are not using**
