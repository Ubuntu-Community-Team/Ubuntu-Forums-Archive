### Post by Bucky Ball on 2016-01-07

Welcome. If you don't need a Chromebook specifically, have you considered something like a HP Stream 11? I don't know how they compare price-wise but I do know that it has a 32Gb eMMC card, enough to run any *buntu, and you can install whatever flavour you like without having to jump through the same hoops as you do with a Chromebook. 

Xubuntu 14.04 LTS, everything worked out of the box on my HP Stream. Running any OS long-term from SD or USB drive is precarious. Make sure you keep a clone of the install somewhere for when the OS device dies, generally unexpectedly.
