### Post by pwabrahams on 2015-04-29
I have just gotten a new HP Pavilion 23" all-in-one, and I tried to run Kubuntu 15.04 on it.  The graphics performance seems to be unbearably slow.  My test case is the old XWindow Spider Solitaire program, and I'm comparing it to the same program on several other computers, notably my old Lenovo laptop.  On the Lenovo, card movements are instantaneous; on the HP, they take several seconds.  I assume that other graphics programs will be similar.  I don't know how the graphics performance for Windows compares.

Does anyone else out there have a similar HP machine?  Have you noticed a huge performance hit for Linux?

I also had a unrelated problem: when booting up from grub, the system would display a sequence of several solid-colored screens for about a second each, then turn black.  I was able to get around it by going through the system menu.  The grub menu displayed differently in the two cases.  For an unmodified startup it was in low resolution; for the modified startup it was in high resolution.

I'm probably going to return the machine before it's too late.
