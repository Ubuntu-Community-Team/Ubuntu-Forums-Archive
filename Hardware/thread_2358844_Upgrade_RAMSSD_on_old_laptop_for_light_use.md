### Post by Bucky Ball on 2017-04-18
2Gb of RAM will make a difference, but be aware that it may not be usable in any modern machine when you retire this one. An SSD on the other hand can be used in any machine for the life of the SSD.

The addition of 2Gb of RAM and an SSD will certainly speed things up and probably allow you to open more browser tabs, but there is no improving that CPU. 32bit system can only handle 3Gb of RAM anyhow, without some massaging, so fine there. 

Lubuntu still provides a 32bit version, but something to remember is that 32bit ISOs are slowly but surely disappearing. 

Hope that helps a bit and good luck.

### Post by frostschutz on 2017-04-19
> I also find that the CPU is frequently at 100% with firefox and a few (10) tabs

Even if you upgrade RAM/SSD you'll still feel the CPU. Or at least I do with my old 32bit netbook (Atom N270): putting in an SSD helped some, but today everything just eats into the CPU, animations on every website etc. If you're using adblock, try ublock origin instead, it seems to perform better for me, also disable all other addons you don't need, every bit helps. SSD is a huge performance boost to any system but you can't magick a ~10 year old system (which was low spec when it was new) into something that makes for a satisfying experience today, if you find the cpu is your bottleneck you just have to get something new.
