### Post by oldfred on 2015-11-13
Either upgrade will not improve system a lot.

You already are using an SSD. You can use HDD for data if SSD is not large enough. I normally install / (root) to a 25GB partition on SSD and have all data on HDD anyway. With my new desktop system and 128GB SSD, I do not know what to do with all the space. I have room for several / and have some data on SSD, but most data is on HDD.

My old desktop system from 2006 had 4GB of RAM and never used swap. While my new build includes 8GB I am not sure if it all is being used with my normal work load. My old laptop with 1.5GB of RAM will use some swap and if I load several large apps then I do notice significant slow down when swapping,
