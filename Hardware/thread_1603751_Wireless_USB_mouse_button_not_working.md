Everything worked perfectly before I upgraded to 10.10. So I can safely blame 10.10 for the following problem:

When I turn on my laptop, the mouse buttons work perfectly fine until I start using the keyboard. When I first press a key on the wireless keyboard, the left mouse button stops working. The right mouse button still works and mouse can be used to move the pointer. The only thing that fails is the left mouse button. If I don't use the keyboard then mouse works, no problem. If the USB receiver is not plugged then the laptop keyboard and touchpad are working fine. If the wireless keyboard is used, then the touchpad left button is also blocked and doesn't work anymore. 

So: If I use the wireless keyboard for typing anything, the system stops responding to left mouse button events. 

Anybody with a similar problem? Any solutions?
