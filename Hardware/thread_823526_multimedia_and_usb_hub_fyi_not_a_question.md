### Post by frrobert on 2008-06-09
Just an FYI in case someone else has this problem.

I have a Toshiba A70 with Realtek sound card running Ubuntu 7.04.  

I had an interesting problem this weekend.

Any multimedia file playback would start then lockup the application playing it.  This would include flash,ogg sound, ogg video whether on Firefox or totem.


I found the problem was an unpowered usb hub.  I recently was given the hub and was using for my external keyboard and mouse so I had more available ports.  Everything worked fine it seemed.

I removed the hub and plugged the keyboard and mouse on the on-board ports and everything works fine again.

So something about the hub was doing something to lockup mutlimedia playback.
