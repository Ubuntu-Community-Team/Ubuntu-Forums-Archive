### Post by paullinux on 2008-10-17
I'm typing this text with Ubuntu 8.04.1 on a MSI-Wind 100. So yes it works.  However I used Wubi to install Ubuntu. Which uses a dedicated virtual part of your windows-filesystem. For now I don't want to meddle with partitioning. Works fine.  Even Wireless (allthough not out of the box...) is working. Suspend-hibernate does not work.  Compiz works great, however ! 
About the netbook: keyboard is great, screen (backlight) is better then my Imac...It takes some getting used to a screenresolution of 600x1024. 
One thing that troubles me is the battery: the 3cell-battery works barely 2 hours... The touchpad is small and very touchy.  Most people I know with this machine will drop the 'tap to click' function immediately. 

Bottomline: It works fine, with some finetuning, but it still is a netbook. Don't expect to do wonders with it...

About vista: Some have installed vista on it (look at MSI Wind forums) for more information.  But since vista requires a fast computer and additional ram I do not think it is wise... It still is a netbook...
