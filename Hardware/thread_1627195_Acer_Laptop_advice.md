### Post by roger_1960 on 2010-11-21
Hi

Make sure that the USB ports are working -some of the 5000 series Aspires have a motherboard fault which causes them to stop working permanently.  If there is a little "dimple" where overheating has occurred on the top casing just below the windows key, don't touch it.  Otherwise, my aspire 5101 has been great with full Ubuntu (32 bit) including 10.04 (although dont expect the built in dial up modem to work unless you have saintly patience with command line setups!)
