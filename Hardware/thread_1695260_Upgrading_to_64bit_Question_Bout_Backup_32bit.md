### Post by oldfred on 2011-02-25
The drive should install and work just fine. If it is a bootable device, then you may have video issues if you installed a  proprietary driver and the new build uses a different driver.

I rebuilt my system with a new motherboard, and newer nVidia card and the drive rebooted without issue. My XP install complained and I had to get permission from BillG to allow it to run. And the permission screen was hidden behind an error about not getting on the internet which the permission screen was preventing. Many reboots later XP did finally work, but it was the final nail in the coffin.
