### Post by robertc36 on 2008-09-17
Hi I have the same Laptop. I used the HP utility to create backup in ISO format resulting in 2 dvd images (transfered to external drive) and after that deleted the recovery partition. I partitioned and installed Ubuntu within 2 hours of owning the laptop and played around with Vista for a while (pretty but so slow), and then erased that partition using Gparted boot disk. I have however left the OS tools partition for the time being( as only 1gb). As long as you have the recovery discs it can be restored.
