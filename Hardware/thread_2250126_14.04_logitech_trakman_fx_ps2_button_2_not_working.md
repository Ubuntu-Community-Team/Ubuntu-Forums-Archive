### Post by vector on 2014-10-26
Hi all,
Recently picked up a new pc (2nd hand) and in the process built it with 14.04.
Everything is working fine except the PS/2 trackballs button 2. (emulate wheel)

tried running xev and where as on the old machine it used to recognise button2 it now only seems to recognise ball movement, right and left click buttons. I know it never reported the red button(8)

I installed and checked gpointing-device-settings (emulate and use button2) as I did last time but this did not help.
I read thru some posts on the subject and created this file 

/usr/share/X11/xorg.conf.d/40-trackman.conf
Section "InputClass"
        Identifier "Logitech TrackMan scroll button support"
        MatchIsPointer "on"
        MatchProduct "PS2++ Logitech TrackMan"
        MatchDevicePath "/dev/input/event*"
        Driver "evdev"
        Option "EmulateWheel" "1"
        Option "EmulateWheelButton" "8"
        Option "XAxisMapping" "6 7"
        Option "YAxisMapping" "4 5"
        Option "EmulateWheelTimeout" "0"
        Option "Emulate3Buttons" "0"
EndSection

and rebooted to ensure it restarted x

What Im not sure on is how to invoke it or tell xorg to use this instead of the ImExPS/2 Generic Explorer Mouse that 
"xinput" reports is being used.
