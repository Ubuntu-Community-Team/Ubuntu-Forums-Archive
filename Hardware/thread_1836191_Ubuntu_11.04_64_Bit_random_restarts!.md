### Post by Antarctica32 on 2011-08-30
About the cd drive: Do you have windows on it as well? I would try creating a cd for a different linux distro (just pick one, i suggest mint) and boot from the cd and see if it recognizes the drive. 

About the random reboots: I had a similar problem once with a desktop and have heard about it many times. It reminded me of how sometimes in windows programs that are written poorly can sometimes just keep using ram and keeping using it until there is no more left and you get a blue screen. I didn't know if the same thing could happen in linux being as linux distros use ram very differently. But what i did was go through all my apps that I had downloaded from the software center (I had a lot) and remove the ones that I hadn't used in a while and i didn't need. And it hasn't happened since. So maybe thats a fix for ya.
