### Post by Rocket2DMn on 2008-08-07
You shouldn't need to manually install and restricted drivers, that is generally only needed if "all else fails".  Can you check under System->Administration->Hardware Drivers to see if you can enable restricted drivers from there?  If so, go ahead and it will ask you to restart the X server (this is the graphical interface software).
