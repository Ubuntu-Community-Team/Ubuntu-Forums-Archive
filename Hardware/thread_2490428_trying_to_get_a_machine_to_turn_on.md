Hi jgw

It is not a good idea to assemble / reassemble a system with the power enabled, that can cause damage to components and you are more likely to short something out.  Hardware eventually does fail.  Are you changing out motherboards (swaping) to try to get a single running system, or are you have multiple computers going down?  If it is one machine you are trying to get running, that suddenly stopped working intermittently, I would probably start with trying another power supply if I had one lying around.  Not sure I would go the route of swapping out motherboards because there can be compatibility issues, not all part will work together.

The one that worked up to plugging in the keyboard, could you try another keyboard, perhaps with a PS2 connection?  USB can sometimes be flaky if you are using a new USB 3.x fancy keyboard with an older generation USB port, at least I've had issues, not often, but can happen.
