Unfortunately or not, the user is still the main concern in dealing with PC's having OS Linux installed. Iris software is known for not only reducing screen emitted blue light at relevant circadian time when generally it should not be present, but also for reducing flicker (dozens of references here). However, flux software version for Linux is still a very feature-reduced version comparing with Windows version. Windows version of flux has a very cool feature which is absent in Iris (even for Linux) that is it has Darkroom mode in which all bright light colours are inverted so very convenient to use while reading black text on light background with inverted black-to-white colours so that the text is white on black background thus less unnecessary irradiation falling onto the retina. I find this feature very convenient for multiple purposes.
Maybe there would be a possibility to integrate this flux feature into Iris for Linux (Ubuntu) ? Or any other way to use this feature? (I may donate something)
I know that Linux is the best and ubiquitous in dealing with hardware, so...?
