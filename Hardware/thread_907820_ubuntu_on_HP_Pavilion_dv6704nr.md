### Post by sergiom99 on 2008-09-01
well I have the same video card and it worked out of the box in Kubuntu 8.04, so it should work in yours too.
Go to "System\ Restricted drivers manager" and check if the nvidia drivers are enabled.
(the menu might change, i use KDE)
Also you can check in Synaptics Package manager if 'nvidia-glx-new' is installed, if not install it.

and for the wireless, there should be something in the forums about your card. search for it.
