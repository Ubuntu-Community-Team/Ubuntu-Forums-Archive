### Post by TomEvans on 2010-08-24
Sure. I didn't need to manually install any drivers. Ubuntu 10.04 automatically set it up. I did have problems with the volume on the device. As it is a USB device it didn't use my usual sound card. Therefore using the usual volume settings didn't work. I have 'Ear Candy' ([https://launchpad.net/earcandy](https://launchpad.net/earcandy)) installed, from that I was able to adjust volume settings. Also unlike on the Windows Skype app, I wasn't able to choose which device to use for calls; but when I plug in the USB Skype goes over to using the headset automatically and immediately; so I leave it unplugged, then plug it in when I'm in a call.
Overall, I'd say the sound quality is excellent, both the headphones and the microphone. However I still wouldn't recommend using it to listen to music.
Let me know if you have any other questions.
