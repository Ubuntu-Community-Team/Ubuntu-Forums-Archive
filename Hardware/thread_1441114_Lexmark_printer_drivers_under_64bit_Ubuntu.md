### Post by efflandt on 2010-03-28
It depends upon which Lexmark.

64-bit 9.10 works fine with my C543dn, once I pointed Printer Configuration to the ppd file (or with the C534dn driver from install CD).  But that is a network color laser that does standard postscript and HP PCL.

Inkjet printers may be an entirely different story, especially if they are dumb raster printers.  But you might try searching posts for "lexmark" because they have been discussed in these forums before.
