### Post by map7 on 2009-06-23
I've just brought a Logitech S520 keyboard and the backtick key does not work.  The backtick character is the one on the same key as the tilde (~) key.  I can use the '~' which is on the same key but when I hit that key with no shift I don't get anything on the screen.

This only happens inside the gui also if I ctrl-alt-F1 to the terminal then I can use the backtick.

Is it my settings for keyboards as I have this on Generic 105.  I've tried all the logitech keyboards but all have the same problem.

Found that the problem was a 'Keyboard Shortcut' was linked to that key, once I removed that link it was fine.  I just binded that key to the Volume Mute then removed it.
