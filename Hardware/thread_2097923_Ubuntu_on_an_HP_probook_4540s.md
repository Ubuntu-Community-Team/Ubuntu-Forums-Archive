### Post by Mike Donovan on 2013-03-24
I have one, HP 4540s ProBook running Windows7, Debian AMD64 and Backtrack 5r3 AMD64 (Ubuntu).
Only problem I had was with the RaLink 3290 Wireless/Bluetooth module.
I was able to compile the RaLink 3290 module and get it running just to find out that it did not support monitor mode.
I replaced the RaLink 3290 with an Atheros module from another Laptop and everything is working perfectly.
Just had to disable Bluetooth since it was not there anymore.
