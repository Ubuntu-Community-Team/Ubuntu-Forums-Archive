### Post by Ralph_Sharpe on 2015-06-22
[DisplayLink]("http://support.displaylink.com/forums/287786-displaylink-feature-suggestions/suggestions/7988955-support-linux-on-all-your-devices") [a provider of multiple USB 3.0 docking station solutions] is currently entertaining a request for [better] Linux support. The question asked is which distos should be supported first; which sounds very hopeful. Voting for the support, and/or specifying a distro preference, is encouraged. I have a Lenovo W530 that attaches to peripheral devices via the[ Lenovo USB 3]("http://displaylink.com/shop/hot_products.php#thinkpadusb30dock") dock M/N DU9019D1 which is decent for supporting Windows 7 Pro using the Ethernet and external monitor capabilities. I don't know how well gaming configurations would perform, but business applications run well. I 'pop' into Linux distros once I a while, just not to be totally ignorant of other operating systems. The current hardware supported using Ubuntu is incredible, yet can be better when more hardware suppliers consider 'joining the fold'.
