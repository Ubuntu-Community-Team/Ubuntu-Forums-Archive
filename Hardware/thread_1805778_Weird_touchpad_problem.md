### Post by roololing on 2011-07-18
There has been some previous talk of a problem in Unity where an invisible window comes up somewhere on the screen. It is a known bug that causes an area of the screen to become a "dead zone," where the mouse can't interact with the desktop or any of the windows below this invisible window.

Some people have been able to get rid of this window by restarting, logging off and on, etc. But some have trouble getting rid of it. And for some people it seems to go away on its own.

[Here is the link to the bug report.]("https://bugs.launchpad.net/ubuntu/+source/unity/+bug/787056")
