I have a Dell C600 and it works "fine"-but if you have less than 512 meg of ram you have to use the alternate install CD.  Also after install you have to edit the xorg.conf to handle 1024x768 resolution.  There is another post that explains in detail how to do this.  But actually if you have less than 512 Ubuntu is not really usable except to play around with.  Windows 2000 on the same machine with 256 meg is much, much faster and can easily multitask several applications not so with Ubuntu.  I used to think Vista was a pig but now I see it's just the way it is with modern operating systems.  It's a bit of myth that Linux is better and faster than windows-it all depends....
