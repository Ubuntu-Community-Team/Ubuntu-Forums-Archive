### Post by sudodus on 2016-02-11
I suggest that you get a regular (naked internal) hard disk drive for continuous use (24 hours 7 days a week) and put it into an external box that has good cooling. You can select connection via USB 3 and/or eSATA. It will be more expensive to buy, but the lifetime will probably be much longer than if you get a standard external hard disk drive.

### Post by kurt18947 on 2016-02-11
Do you plan for it to be online full time as a mirror for example, or plug in run a backup routine and unplug? For a device active 24/7 , cooling is a consideration as sudodus mentioned. I think hard drives labeled for audio/video or surveillance system use might be more durable.
