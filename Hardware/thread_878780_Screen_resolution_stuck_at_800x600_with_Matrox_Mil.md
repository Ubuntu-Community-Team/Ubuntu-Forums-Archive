### Post by nadarockyraccoon on 2008-09-16
in your xorg.conf file where it says DefaultDepth 24 change this to read DefaultDepth 16, maybe even try making it 8, but I know 16 works. The card doesn't have enough memory to run any higher than 800x600 at 24 depth, so by changing the depth you will free up enough memory to run at higher resolutions.

### Post by Corrado on 2009-12-06
Just an update to this question for any future visitors.

The new Xorg stuff in Ubuntu 9.10 doesn't require a xorg.conf file anymore.  It builds one in memory each time you boot Xorg.  That is probably the reason it works sometimes and doesn't others.  If this file doesn't exist on your system, just create it and Xorg will use your file instead of creating a new one in memory.
