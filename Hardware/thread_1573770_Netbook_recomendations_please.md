### Post by omniuni on 2010-09-13
If CPU is an issue, check out the MSI Wind U210 with the Athlon Neo. It's a 12" sub-notebook, about three pounds and some ounces including the extended battery. I get around 4 hours with it, depending on how I use it. It runs very cool, and hardware support is nearly perfect. There is a PPA that provides the appropriate wireless driver, after which it works flawlessly and never needs to be touched again. Sleep and hibernate both work well on this machine, so does the webcam, microphone, etc. Powers an external monitor just fine once you turn off KMS (though hopefully they'll fix this in Mavarick; it's an Ubuntu thing, not a driver problem), and contrary to what you'll find online, the Neo DOES support virtualization via AMD-V. (No trouble running a Windows XP VM through VirtualBox). It runs cooler, faster, and is much less of a pain to install Linux on than the HP Mini I have. (HP Mini has a Broadcom wireless, which needs proprietary firmware).
