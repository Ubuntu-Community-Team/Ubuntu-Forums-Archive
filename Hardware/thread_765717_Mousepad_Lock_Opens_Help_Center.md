### Post by Ayuthia on 2008-04-24
Just in case you feel like it is just you, it is not.  I have the same problem in Ubuntu.  In Arch, it works fine.  I have found out that the mousepad lock button is mapped to the same number as the F1 key.  If you don't mind having the keyboard shortcut for the help be something else instead of the F1 key, you can change it in your keyboard shortcuts.  Unfortunately, I am a KDE user so I can't remember where it is located in Gnome.  It should be in your control center/system settings section.
