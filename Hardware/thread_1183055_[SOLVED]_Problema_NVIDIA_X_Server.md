### Post by staar on 2009-06-09
en linux los mensajes de error son bastante explícitos generalmente (a menos que no domines el idioma del Imperio :-D)

```
You do not appear to be using the NVIDIA X driver
Please edit your X configuration file (just 
run nvidia-xconfig as root), and restart the X server
``` traducción al criollo: ```
no estás usando el driver Nvidia, por favor editá tu archivo de configuración de las X (simplemente corré nvidia-xconfig como root)
``` osea que tenés que abrir una Terminal (Aplicaciones > Accesorios > Terminal) y escribir ```
sudo nvidia-xconfig
``` poner tu contraseña, y reiniciar la sesión para que se apliquen los cambios...

### Post by Mauro22 on 2009-06-09
Lo que haria yo es:


1) hace backup del xorg.conf

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```

2) Editar el xorg.conf y:

donde dice:

```
Driver	"nvidia"
```

cambialo por:

```
Driver	"vesa"
```

3) Reinicia la X:

```
sudo /etc/init.d/gdm restart
```

4) Ahora te tendria que levantar bien pero sin acceleracion grafica.

5) Instala envy-ng:

```
sudo apt-get install envyng-qt
```

6) Lanza envy-ng e instala los drivers que te recomienda. Envy ng se encuentra en el menu o sino:

```
sudo envyng
```

7) Creo que se reincia solo, sino hace el paso 3 otra vez.
