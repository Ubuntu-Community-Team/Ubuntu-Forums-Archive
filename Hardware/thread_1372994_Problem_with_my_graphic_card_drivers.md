### Post by vulle on 2010-01-06
I managed to solve this problem with editing my xorg.conf file
Section Device should look like this, others remains the same.
```
Section "Device" 
    Identifier    "Default Device" 
    Driver        "nvidia" 
    Option        "NoLogo"    "True" 
    Option        "ModeValidation" "NoTotalSizeCheck" 
EndSection
```
