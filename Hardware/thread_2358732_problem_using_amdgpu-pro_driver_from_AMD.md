### Post by gsahli on 2017-04-16
I solved my problem by reinstalling MATE 16.04.2.
I use ubuntu MATE 16.04.2 with my A10-7860K processor in a gigabyte F2A88XM-D3HP motherboard. I installed the AMD driver amdgpu-pro-16.60.379184. After installing this driver, the cursor becomes invisible after the screen blanks and I have to re-login. The cursor is still present, but invisible. I can press "control" key to show the location of the cursor, but that doesn't regain visibility. In addition, there is no "uninstall" script included in the package even though the online instructions say there is. Reinstalling linux which uses the amdgpu driver (not amdgpu-pro), eliminates this problem. I'm posting here to hear if anyone else had this problem.
