### Post by sanderj on 2013-07-07
This is my Ubuntu harddisk setup:

```

Filesystem      Size  Used Avail Use% Mounted on
/dev/sda7        19G   11G  6.7G  62% /
/dev/sda8       149G  125G   17G  89% /home
```

So I assigned 20GB for the operating system itself.
