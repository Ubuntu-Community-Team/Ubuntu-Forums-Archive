### Post by jaime256 on 2010-02-15
Hi guys. Well, there have been a few more updates today, there's a new Chrome version and Adobe update. As for the printing, I don't know why it stopped working, so I went ahead and took a look at this. The icon is the older one and the printer model had an extra number on it which I'm pretty sure was not there before. It would be hard to miss, but anyway, I went ahead and deleted the printer, add it a new printer and I noticed that the system is still putting a new :631 after the printer IP on my side. I don't know if it's because it's on a nas or what, but MAKE sure you take that out, then verify your connection and continue with the installation. The name will show you that :631 as well, take it off and then finish the install. I then noticed the icon is the new network printer icon. Not sure that it means anything, but now I got the printer to work again. Hope this helps. I'm already logged into the nas so I know that should work, but haven't tested the printing before logging on. I hope it still works that way too.

If you're administrating computers that would be painful, so make sure that stuff is out when you add a printer. I just did an ebay text test print and that seems to print okay. I still still two small black blocks on some text, but everything else seems to print fine including the picture.
