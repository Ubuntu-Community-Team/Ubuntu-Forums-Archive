### Post by Yellow Pasque on 2013-01-27
10.04 is old, so the version of fglrx that comes with it is probably older than the video card. You should install a newer version of fglrx (if not a newer version of Ubuntu). [http://wiki.cchtml.com/index.php/Ubuntu_Lucid_Installation_Guide#Installing_the_drivers_manually](http://wiki.cchtml.com/index.php/Ubuntu_Lucid_Installation_Guide#Installing_the_drivers_manually)
