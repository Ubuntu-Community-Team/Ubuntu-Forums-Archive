### Post by shadylookin on 2008-05-25
you can also go system>administration>network tools

set the network interface from loopback interface (lo) to Ethernet interface(eth0) or possibly wlan0 if you're using wireless

then down at the bottom you'll see hardware address: with a bunch of numbers and letters following it which is your mac address
