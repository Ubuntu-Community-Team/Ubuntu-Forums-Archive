### Post by papibe on 2011-11-07
If you did a regular install, go to Computer, then you'll see several disks. The Windows files will be in one called 'OS' or 'Windows'. If you double click on it, you'll be in your C:\

In case you did a Wubi install, open Computer, then on the left select 'File System'. There look for the directory called 'Host' and open it.

Hope it helps,
Regards.
