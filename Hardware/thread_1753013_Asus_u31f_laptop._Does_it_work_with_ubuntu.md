### Post by saschd on 2011-06-16
Sparx, the model i am using is the core i5 2.66ghz with 4gig ram and 500 gb hdd. It is very quiet, almost silent i would say;yet to hear a noisy fan kick in.

I connected it to a larger screen with hdmi. When the mirror option was enabled, the laptop screen was cropped but full screen on the external display, though not its highest resolution. I disabled the mirror option and setup the second display using the monitor tool in ubuntu and rebooted.
   
It worked great on reboot. full screen on laptop and full screen on external display. I was able to watch a movie at 1980x1080 on the big monitor while transferring 120gb of data on the other screen simultaneously. My home folder is encrypted as well, but the video was smooth.

Horizontal and vertical scrolling work once enabled on the mouse menu.My 3g modem works out of the box, which is handy.

Now for some things that don't work so well lol. The webcam image is upside down! There is a workaround for this here [http://radu.cotescu.com/flipped-images-ubuntu-webcam/](http://radu.cotescu.com/flipped-images-ubuntu-webcam/). Though the image is still a mirror copy. So some tweaking still needed.

I don't think the suspend/hibernate work properly;they never really do under linux. Not really a issue for me as i usually shutdown and ubuntu is super quick to boot anyway.

I am yet to get digital audio working. If anyone has please let me know.

The ubuntu os runs great on this laptop and i really like the new unity interface.
