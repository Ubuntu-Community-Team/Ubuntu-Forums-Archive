### Post by maharaja2 on 2008-11-05
I had finally found the solution. It was very easy actually, so i will post it in order that someone need it

Open in firefox [http://localhost:631/](http://localhost:631/)
Add new printer and select Lexmark Optra E321 drivers.

That's it.

### Post by Clay_Banger on 2012-02-11
Confirming this works on Ubuntu 11.10.
I selected the "Lexmark Optra E321 Foomatic/hpijs-pcl5e (recommended)" driver, and have not had any problems with it.

Also, if sharing the printer via Samba to Windows clients, they will need to select the correct model when installing the printer.
