### Post by Yellow Pasque on 2012-11-23
You should settle on a CPU before picking a mobo. As a longtime AMD user, I'd actually recommend Intel, especially if you're not into heavy gaming. The Intel CPU's just have better performance/Watt (important if using a small case) as well as a better IGP

The Core i3-3225 is a good choice (dual-core 3.3 GHz Ivy Bridge with Intel HD4000): [http://www.newegg.com/Product/Product.aspx?Item=N82E16819116774](http://www.newegg.com/Product/Product.aspx?Item=N82E16819116774)
as well as the Core i5-3570k if you want a quad-core [http://www.newegg.com/Product/Product.aspx?Item=N82E16819116504](http://www.newegg.com/Product/Product.aspx?Item=N82E16819116504)
