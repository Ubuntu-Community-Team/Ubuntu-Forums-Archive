### Post by chrisreichel on 2011-10-17
The fix is:

1 - sudo vi /etc/default/grub 

change GRUB_CMDLINE_LINUX="" to GRUB_CMDLINE_LINUX="enable_mtrr_cleanup mtrr_spare_reg_nr=1"

save the file

2 - sudo update-grub

3 - restart

4 - It should work... but it doens't

references:
