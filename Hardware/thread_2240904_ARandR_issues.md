### Post by kakashi_12 on 2014-08-26
Thanks brick wall... for talking back to me.

Fixed the issue of the TV being turned on by default.
Just went into ARandR and set DVI-2 as In-active. Saved the layout as an sh file. Made the file executable. Then Went to Applications, System Tools, Preferences, Start Up Applications. Then browsed to the file.
