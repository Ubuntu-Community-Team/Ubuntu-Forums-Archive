### Post by MateuszP on 2009-02-22
I've bought HP Pavilion dv5-1060ew and I'm using Ubuntu 8.10. It cost 3400PLN(1000USD)

**Video-Kamera**(Easy Cam2) works properly, **Wi-Fi** works properly, **multimedia buttons** work properly, my **laser printer Brother DCP-7010** works properly(except scanner, which is revealed by system, but not works yet), **Blutetooth** is revealed(but there is problem with connection via mobile phone, because in Ubuntu 8.10is another module supporting blue-tooth comparing with 8.04 version - You have to change this in Synaptic). Sound work properly without microphone in Skype(I suppose it is not so problematic to solve). **Finger Print / Reader** doesn't work yet. I didn't check **Blue-Ray Disc** and **HDMI** socket and **Card Reader**
