### Post by Quadunit404 on 2010-12-19
This is for Samsung NP-RF510 owners, like me. For whatever reason, Nautilus freezes Ubuntu every few minutes when Ubuntu is installed on the Samsung NP-RF510. Ubuntu installs and otherwise runs just fine, however. This is how to resolve these freeze ups. Make sure you have installed the Broadco and NVIDIA drivers before continuing!

1. Open a web browser and search for Ubuntu Tweak. The site is the first result. Download and install the .deb installer on the site.
2. Launch Ubuntu Tweak.
3. In the Source Center, enable the elementarydesktop PPA and reload.
4. When UT tells you there are updates available, install them. This includes Nautilus Elementary (it says nautilus rather than nautilus-elementary so if you see any updates to nautilus - which you should - select them.)
5. If Nautilus did not freeze Ubuntu yet, restart Ubuntu. If it did, hold down the power button until the laptop shuts off. Restarting is important especially if you updated your kernel (although if you did I don't recommend that you manually power it off as that may break something.)
6. Log in again.
7. ???
8. PROFIT!
