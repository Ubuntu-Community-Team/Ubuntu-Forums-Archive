### Post by zodiac09 on 2010-04-20
I use to have the same problem, except sometimes with windows the fan would run really fast from time to time if i was watching youtube or something. But just surfing the web and basic word processing with the fan in windows was fine.
 
So i took my laptop apart, to get at the fan. There was a lot of dust in there. I mean a lot!! After removing that dust, i never overheated watching youtube in windows. And i could run ubuntu fine, with a quite fan. 
 
All the dust was making it loud!
 
So check your fan for dust.
