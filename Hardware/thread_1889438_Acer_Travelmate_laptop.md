### Post by difiCa on 2011-12-03
I don't see any reason you wouldn't be able to run either distro on it, just take the one you prefer. As long as you back up all important files from the Windows installation, there should be no problem with removing W7 and installing Ubuntu, however, I personally would not remove Windows if it works fine, but instead install Ubuntu alongside Windows in a dual boot system. 

For example, my netbook I use when travelling has both due to the reason that sometimes projectors and displays I use for presentations don't seem to like Ubuntu and it's obviously easier to deal with applications for Windows through Windows than Linux emulators. I'd also have Windows on my home computer if it hadn't been a broken Vista installation because I still can't fathom the aforementioned display issue completely. 

But, if you want to remove Windows, go ahead, the Ubuntu installer gives you the option of doing so.
