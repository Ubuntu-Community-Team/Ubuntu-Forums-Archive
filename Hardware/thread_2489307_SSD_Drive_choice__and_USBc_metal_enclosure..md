### Post by oldfred on 2023-07-25
My first enclosure as SATA only and drive was a M.2 SATA drive.
My second enclosure is both SATA or NVMe and I put a Samsung NVMe drive into it. Not sure if using a NVMe drive thru USB3 really adds any speed over using SATA only. But now most of the adapters support NVMe and NVMe drives are a lot cheaper than just a few years ago. I also think NVMe runs a bit hot, so needs to be metal to help cool drive.

The 2280 is probably what you want, as that is the longer and more standard for larger drives.
They often then have another screw for the shorter drives.

[https://en.wikipedia.org/wiki/M.2](https://en.wikipedia.org/wiki/M.2)
Two types of M.2 SSD (SATA and PCIe)            
SSD's using the SATA bus tend to use B-keys
SSD's using the PCIe x2 bus can use both B-keys & M-keys (to maximize compatibility)
NOTE: SSD's using the PCIe x4 bus (very fast SSD's) use M-keys since it's the only one that supports PCIe x4 lanes
