### Post by louieb on 2009-07-28
In Ubuntu to run a program with root privilege  put **sudo in front ** for a command line program. 

or **gksudo **for a graphical program.

more here [RootSudo - Community Ubuntu Documentation]("https://help.ubuntu.com/community/RootSudo")

You can Google how to log into Ubuntu as root. But forum policy says we can't post that how-to here.   [forum policy on log-in-as-root tutorials - Ubuntu Forums]("http://ubuntuforums.org/showthread.php?t=716201")
