This is a little late...but I have a Razer Blade Pro (17") from 2013.  I was able to install Ubuntu on it but had some various problems which were detailed pretty thoroughly in [this thread]("http://forum.notebookreview.com/razer/729380-razer-blade-pro-under-linux.html") on notebookreview.com.

On the [last page]("http://forum.notebookreview.com/razer/729380-razer-blade-pro-under-linux-2.html"), however, there is a small customer driver/workaround package that I found corrected the issues.

I still have some problems with my keyboard and touchpad drivers unloading themselves periodically (temporary freeze/restart of keyboard and touchpad) but I believe it is because I installed Ubuntu through UEFI.  You should disable UEFI boot in BIOS, but that will break the Windows 8 boot.  It is possible to manually enable/disable UEFI boot in BIOS depending on which OS you want to boot into...but you have to be pretty dedicated if you want to live with that work around.

The original creator gives his e-mail to contact him in case of problems...so I might do that...but for now I can live with it.
