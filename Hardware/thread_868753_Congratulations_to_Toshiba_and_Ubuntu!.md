### Post by ibargureni on 2008-07-24
Hi, my brother has just bought a new Laptop, a Toshiba A200 with Vista Home preinstalled.

I "needed" :rolleyes: :rolleyes: to install Ubuntu in that PC as soon as possible, so I started investigating the partitions it had in its 200Gb Hd. It had two main partitions, each one with 100Gb, so I decided to leave the partition with Vista intact and install Ubuntu Hardy (8.04) in the other one.

I was expecting problems with Vista after deleting that partition and with Ubuntu (since this laptop is quite new) and Hardware recognition, but to my surprise everything went smoooooth. Ubuntu installed itself easily and without a problem. All the hadware was detected, even the integrated webcam and the card reader.

This was the easiest installation I ever made, in a few minutes I had the laptop running Ubuntu with compiz and Vista propperly accesible from grub.

If anyone needs a laptop to install Ubuntu I really recommend this one.

Regards.
