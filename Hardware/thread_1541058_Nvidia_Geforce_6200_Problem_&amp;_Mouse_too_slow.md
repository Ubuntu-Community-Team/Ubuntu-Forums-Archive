### Post by quixote on 2010-07-31
I had nvidia graphics in my last computer, and I was very impressed by the open source driver: nouveau.  It was actually *better* than nvidia's own drivers. :shock: You can install it via synaptic. xserver-xorg-video-nouveau.  I also have a file installed called libdrm-nouveau1, which is probably needed.

You may want to search on your exact graphics card and nouveau, just to see if anyone's had any experience using them together.
