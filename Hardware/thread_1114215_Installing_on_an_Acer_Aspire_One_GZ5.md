### Post by easybake on 2009-04-02
I have had good luck using the kernel compiled by other people. Here is the link to the one i used. Wifi right out of the box. [http://www.aspireonekernel.com/]("http://www.aspireonekernel.com/")

Windows 7 works surprisingly well on it also. Don't be afraid to set up a dual boot or triple boot situation. They aren't that difficult (if you have the HDD space).
