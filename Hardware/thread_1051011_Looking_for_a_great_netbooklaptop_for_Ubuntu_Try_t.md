### Post by luckyluke699 on 2009-01-26
Hello All,

This is just a post for anyone considering an ubuntu box, who might want something slightly bigger than a netbook (so they can use all the keys etc) but not a big 15 inch laptop for under £250!!

Take a look at the BG45-U-300. It has a 12 inch screen (and a good keyboard!), has a little 1.8 Ghz Dual Core CPU, 80Gb hdd, and although it only comes with 1GB Ram, for those that are interested, this can be upgraded to up to 4Gb Ram for very little cost! The build quality is excellent.

You can pick one up for under £250 at either Currys or PC World, which seems an absolute bargain for the spec.

This comes with Vista basic pre-installed, and doesn't have a cd-rom with it. To get round this for installing Ubuntu use a program called Unetbootin (the 8.10 usb key cd builder didn't work for me, only gave me a "boot error" on bootup) from another linux or windows pc to make the usb boot key - it will work fine.

Once built, the only other problem you will have is the wifi drivers will need updating. A little google search will lead you to an easy answer on that one too!

I will be buying a 2gb ram upgrade (to give me 3gb total) and a slip case all for £25 very soon from ebuyer :) 


Great price for a great little machine. Hopeffully this will help someone out :D

Luke
