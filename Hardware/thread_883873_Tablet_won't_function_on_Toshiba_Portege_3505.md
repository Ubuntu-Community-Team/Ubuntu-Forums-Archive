	Driver		"kbd"
	Option		"XkbRules"	"xorg"
	Option		"XkbModel"	"pc105"
	Option		"XkbLayout"	"us"

	Identifier	"Configured Mouse"

	Option		"HorizEdgeScroll"	"0"

there is one problem in giving you a correct answer to your question, because it is not complete.

you need to find out what kind of mousepad you have and then set the correct values for it in xorg.conf

you can find out what mousepad you have with the following:

in a terminal: "cat /proc/bus/input/devices"
[size=1](without the quotes ofcourse)[/size]

for synaptics touchpads:

[https://help.ubuntu.com/community/SynapticsTouchpad](https://help.ubuntu.com/community/SynapticsTouchpad)
