### Post by dominiquec on 2009-10-18
If you really want to use Ubuntu, you could try a minimal install (command-line) and just add a window manager like OpenBox.  There's plenty of resources on help.ubuntu.com about installing minimal Ubuntu.  [Here's my own experience.]("http://ubuntuliving.blogspot.com/2008/06/minimalist-ubuntu.html")

If you're open to other distros, you can try Damn Small Linux.  It's 50MB and pretty lightweight.

And if you really want something light, try Tinycore Linux.  You'll just have to get your head around its TCE installation system (just place the apps in a directory, actually).  Smaller library of apps, too, but quite functional.
