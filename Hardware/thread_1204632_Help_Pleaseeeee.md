### Post by lisati on 2009-07-05
You might find that Frostwire will work just as well, if not better, than Limewire.

Go to the download page for your choice of software - [http://www.limewire.com/download/](http://www.limewire.com/download/) for Limewire and click on "get basic", or [http://www.frostwire.com/download/?os=ubuntu](http://www.frostwire.com/download/?os=ubuntu) for frostwire.

This will download a "deb" file which you can save. Once it has downloaded, open the "deb" file to install the software.
