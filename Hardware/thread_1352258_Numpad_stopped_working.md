### Post by rajtendulkar on 2010-01-20
[http://www.uzbmaster.com/shingil/200...s-not-working/]("http://www.uzbmaster.com/shingil/2008/05/13/ubuntu-804-hardy-heron-numpad-is-not-working/")

Solution
Go to:
1) System
2) Preferences
3) Keyboard
4) Switch to tab “Mouse Keys”
5) Uncheck “Allow to control the pointer using the keyboard”
6) Done.
