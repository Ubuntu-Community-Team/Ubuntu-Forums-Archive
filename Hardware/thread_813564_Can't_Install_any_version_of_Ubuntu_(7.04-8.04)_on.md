### Post by oVIXx on 2008-05-30
Make sure you have the jumpers set correctly for Master/Slave and that the jumpers correspond to the ports on the ribbon (or round) cable.

The HDDs are on different channels, Both are set to Master with DVD+RW drives as slaves because thats the only way the motherboard will recognize 2 HDDs.
