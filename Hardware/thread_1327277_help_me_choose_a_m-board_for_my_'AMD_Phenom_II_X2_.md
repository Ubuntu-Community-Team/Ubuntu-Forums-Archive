### Post by Fale on 2009-11-24
Just like to add that Phenom II x2 550 has potential to become powerful Quad core processor by just using one switch in BIOS.

I managed to unlock two extra core from my Phenom II 550 x2 which is now Quad core processor.

My motherboard is: **Asus M4A785TD-V EVO** (BIOS update needed!)

**Guide:** [http://www.xbitlabs.com/articles/cpu/display/phenom-athlon-ii-x2_15.html](http://www.xbitlabs.com/articles/cpu/display/phenom-athlon-ii-x2_15.html)

### Post by jonnytb on 2009-11-24
> **Fale said:**
> Just like to add that Phenom II x2 550 has potential to become powerful Quad core processor by just using one switch in BIOS.

I managed to unlock two extra core from my Phenom II 550 x2 which is now Quad core processor.

My motherboard is: **Asus M4A785TD-V EVO** (BIOS update needed!)

**Guide:** [http://www.xbitlabs.com/articles/cpu/display/phenom-athlon-ii-x2_15.html](http://www.xbitlabs.com/articles/cpu/display/phenom-athlon-ii-x2_15.html)
I did the same with an Asus 790XTD Evo, no update needed, just enable Advance Clock Calibration (ACC) on the bios and set it to unleashed mode (also called hybrid on gigabyte mobos), And if your cpu is in fact a quad core it should unlock. Just make sure to run a torture test afterwards (prime95) to make sure none of the cores are defective. I ran prime95 for 24 hours without any errors or warnings.
hope this helps
