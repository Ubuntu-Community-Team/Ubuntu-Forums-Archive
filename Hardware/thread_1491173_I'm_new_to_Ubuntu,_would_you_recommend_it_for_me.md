### Post by cbecker78 on 2010-05-23
If you're lucky, most drivers will be present in the kernel... if not, you can probably find instructions for compiling/installing them here on the forum, or else ask for help.  Setting up the dual boot will give you the time to figure out how to get everything working.  You can search the forum for the model of the computer and see what sort of problems others have encountered.  But remember, different people can have different experiences depending on the specific hardware installed on the laptop and the types of customization.  Also, you have the flexibility of using the live cd of Ubuntu as a test run to see how well everything works.  The only real difference with the live cd is you can make no permanent changes and since it is running from ram, it will be somewhat laggy/slow.

Once and if you are completely satisfied, you can wipe windows off the drive and use that space you left for it as additional storage/backup.

Good luck!
