### Post by heimo on 2008-10-29
> **rmhamm said:**
> I need to know if anyone has had any luck with tv tuner cards. Which brands are the best for linux and work well with mythtv.

I've a [Hauppauge Nova-T 500]("http://www.mythtv.org/wiki/index.php/Hauppauge_WinTV_Nova-T_500_PCI") (dual DVB-T receiver), which works pretty well with mythtv. There are some issues with it, but I'm pretty happy. If you have mythtv and this card and you're using EIT (electronic program information embedded in the signal) it may crash sometimes and create empty recordings, but if your system boots every now and then, it works well. It's probably issue with its firmware.

At least check these links for hardware compatibility:
[http://www.mythtv.org/wiki/index.php/Category:Video_capture_cards](http://www.mythtv.org/wiki/index.php/Category:Video_capture_cards)
[http://www.linuxtv.org/wiki/index.php/Supported_Hardware](http://www.linuxtv.org/wiki/index.php/Supported_Hardware)
