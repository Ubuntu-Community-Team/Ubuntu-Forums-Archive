### Post by alvinmoneypit on 2013-07-04
So reviewing what I did to get a Lexmark X1150 printer working (in Black ink only), I downloaded the [z600 deb]("https://help.ubuntu.com/community/HardwareSupportComponentsPrinters/LexmarkPrinters?action=AttachFile&do=view&target=lexmark.z600-0.4.deb"), derived the PPD file from the deb by opening it with Archive Manager and navigating through the folders: USR>SHARE>CUPS>MODEL>.ppd and extracting it to the desktop, accessed the [CUPS]("http://localhost:631/") through Firefox and added the printer (Administration on top of the page, then "add Printer"), then, since I'm running a 64-bit version, I added the 32-bit libs by pasting the following code into the terminal window (this takes a while to complete):```
sudo apt-get install ia32-libs alien
``` It would probably be better if 64-bit users enter the above code first to get the libs.
If this doesn't work for your Lexmark model, [this thread]("https://help.ubuntu.com/community/HardwareSupportComponentsPrinters/LexmarkPrinters") is where I got most of the info I needed to fix my print driver, it is old and for older distro's, but it worked for me.
