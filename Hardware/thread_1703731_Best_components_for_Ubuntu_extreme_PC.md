### Post by HankB on 2011-03-09
Pretty much anything that makes a PC run Win7 faster will make it run Linux faster. (Only exception I can think of is that small flash drive that Vista and Win7 can use to cache the spinning HD.)

Speaking as someone who has been using Linux a long time, you may find it difficult to entirely give up Windows entirely. You may find things for which there are no Linux counterparts. I still run Windows in a VM when I need to run my Garmin S/W to develop GPS routes. I heartily encourage you to try, but it might be wise to keep Win7 as a dual boot option in case you really need it now and then.

HTH,

### Post by cascade9 on 2011-03-10
DDR3-2000 is not much faster than DDR3-1600. Like, less than a 1% in any real world case. Its not worth the extra expense, last I checked DDR3-2000 was a lot more expensive than DDR3-1600. 

You shouldnt have any problems with DDR3-1600, even allowing for the weird stuff that can happen with overclocking.  

BTW, watercooling isnt worth it unles you get a custom setup. The 'off the shelf' watercooling kits are no better than a good aircooler.
