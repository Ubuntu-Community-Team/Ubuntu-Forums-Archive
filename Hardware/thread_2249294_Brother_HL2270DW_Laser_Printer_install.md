### Post by Bucky Ball on 2014-10-21
Have you got hplip installed? That is in the repos. If not, install it. 

Anyhow, after that, go [HERE]("http://support.brother.com/g/b/downloadlist.aspx?c=us&lang=en&prod=hl2270dw_all&os=128") and follow your nose. Let us know if you get stuck. Once the drivers are installed there are a couple of ways of going about installing. You can use localhost:631 in a web browser and set it up that way (the correct 2270dw driver should appear there) or easier for the uninitiated, go to 'Printing' and 'Add Printer'. 

Good luck and let us know how you get on. ;)

PS: You should only need the ['Driver Install Tool']("http://support.brother.com/g/b/downloadend.aspx?c=us&lang=en&prod=hl2270dw_all&os=128&dlid=dlf006893_000&flang=4&type3=625") from that page I linked to.

PPS: I have that printer and have just recently set one up for a friend also. Great little machine. The original ink doesn't last long but that can be easily remedied by refilling yourself (NOT buying a new ink cartridge). This will give you a full cartridge, not the 'beginners' version which doesn't print many pages. It's easy to refill.

You can get the ink from many printer refill suppliers. Not sure where you are. I'm in Aus and use Tonerstop. You'll find them online if you're in Australia, too.
