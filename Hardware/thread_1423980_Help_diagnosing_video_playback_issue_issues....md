### Post by chaanakya_chiraag on 2010-03-07
Hmmm...so you're using the Nvidia driver?  Try this:  reconfigure the graphical interface - I'm trying to figure out if this is a driver issue.  To do this, do the following:
1) Press Ctrl-Alt-F1.  DON'T PANIC - this is a rescue console
2) login like normal - your password WILL NOT be shown - just enter it like normal and hit Return/Enter when you are done
3) Type in ```
sudo dpkg-reconfigure xorg
```4) Type in ```
sudo shutdown -h now
``` and then turn it back on.
4) Log in like normal.
5) Try playing a video now and see if any flickers occur.  If they do, it's a hardware problem.  Otherwise, it's a driver issue.
Hope this helps!
- Chiraag
