### Post by Crinos512 on 2010-09-12
It is common for [Dual Channel]("http://en.wikipedia.org/wiki/Dual-channel_architecture") memory motherboards to use different colored sockets for each independent channel.

That is probably what you have there... do yourself a favor and split the RAM up (one Blue, one Black) so you can take advantage of it.
