My understanding is that a faster machine with additional ram is good with any V.M. Anyway it works fine with my machine which is a Thinkpad T61p see details at [http://www.cheritononline.co.uk/product.php?xProd=6139&xSec=95](http://www.cheritononline.co.uk/product.php?xProd=6139&xSec=95) with 2gb memory upgrade.

With the VM you can obviously select the amount of ram, HDD space that the V.Machine will use and this is where the extra RAM is nice, I have 4gb installed and it's plenty to run the VM without any horrid delays.
