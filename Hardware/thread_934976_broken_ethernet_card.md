### Post by indefix123 on 2008-12-18
Hello,
I have the same laptop and I think the same problem. Before 15 days my Ethernet stopped working, occasionally it starts but it makes the computer to hang. Now I am using a USB to Ethernet adapter to connect to the internet. I am running Windows XP. I also think it is a hardware problem.

As I found out in the internet there is a possibility that you don't have a hardware problem. I checked mine and it certainly is. If you are dual booting, Ubuntu is not able to start the Ethernet. What you have to do is to select Wake on Lan from windows so they won't put the Ethernet in sleep during shut down. Not all the drivers haves the Wake on Lan Option so you must install the latest and activate it in the control panel. 
Check also this thread
