### Post by TheFu on 2019-09-24
> **DeadlyOats said:**
> What about Chromebooks?  Chromebooks are basically Linux, right?  Can I install Kubuntu onto a Chromebook, or did Google do something to it so that you can't?

Sometimes, yes.  Sometimes, no.  My prior 2 laptops were chromebooks.  At the time, there weren't any comparable laptops for the same price available.  That has changed since 2017.  I wouldn't push anyone to get a chromebook who didn't know they wanted ChromeOS.  There are just too many limitations and very poor quality in the hardware.  

If you have an old laptop less than 10 yrs old, you can install ChromiumOS on it to get the Chromebook experience. It should be extremely fast, regardless of the CPU, RAM, disk.

I agree on the Dell/Lenovo general recommendation.  Also agree that HP is to be avoided. Don't know if current HP laptops need that "avoid" recommendation, but we used to do annual Linux InstallFests at a local University and HP and some of the really cheap Acer laptops were always a challenge.  Some low-low-end Asus laptops too. I have an Asus laptop that works fantastic with Linux. Only the fingerprint reader doesn't work and will likely NEVER work. No drivers. I've never tested bluetooth, but everything else "just worked."

My Asus was $305 from eBay.  Mobo: ASUSTeK model: X510UAR  Core i5 8th Gen.  It has 1 blemish that cannot be seen, only felt. Besides that, it seems new.  I did pull the 1TB HDD out and put in a 500G SSD.
Previously I had a high-end Toshiba Chromebook which was more expensive (Core i3), before I upgraded the SSD.  And before that I had an Acer C720 Chromebook, Celeron 2995U.

ChomeOS is Linux, about like Android is Linux.  It is usually non-trivial to get a full Ubuntu onto it. To boot a normal Linux always requires a hardware change, often just breaking an electric circuit, so a more open BIOS can be installed.  I ran using crouton for a few months, before getting frustrated for the things it couldn't do that  a full Linux could.  And don't get me started on the chromebook keyboards. They suck.
