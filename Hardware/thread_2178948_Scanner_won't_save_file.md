### Post by NilPointer on 2013-10-06
If you can see scanned document on screen, then scanner definitely works. If you can't save file, perhaps you're trying to save it to directory where you don't have permissions to save files? Did you try to save file on desktop? If you will successfully save the file - try to print it from image viewer.

Personally, I prefer XSane over standard simple scan tool, but it's a way more complex tool.
