### Post by utnubuuser on 2010-12-13
Lucid Lynx 10.04
I'd suggest reading an install how to that includes a seperate /home partition.

If you're new to Linux, partitioning etc sounds somewhat complicated, but it really isn't - having a seperate /home partition allows you to install, re-install etc while retaining all your settings, documents, data, what-have-you.

There is a paragraph on installing with a seperate /home partition here: [http://sazeit.com/articles/manually-select-partitions-ubuntu]("http://sazeit.com/articles/manually-select-partitions-ubuntu")

Insofar as the best version goes, 10.04 is now extremely stable and pretty much state-of-the-art.
In my experience, if you can run the liveCD of any given version without problems, then that version will install equally well.

If you do the seperate /home partition, you can change versions/upgrade from a live CD very easily. You can even use different release-versions from the same /home partition. For example, you can have Lucid Lynx and Maverick Meerkat running on the same machine using the same /home partition.
