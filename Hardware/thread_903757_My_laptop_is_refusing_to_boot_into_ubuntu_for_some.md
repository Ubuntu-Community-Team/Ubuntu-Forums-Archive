### Post by lovinglinux on 2008-08-28
I don't know about your notebook, but I tried to install Ubuntu and Mandriva on my mother's Compaq Presario V6210 and it gives me install errors at the same point yours is "freezing". The only version that works is the Mandriva 2007 included in the rescue disk.

It seems in my case that the newer kernels are incompatible with the notebook bios version, which is the real culprit. I decided to keep Windows on her machine. Too much trouble to install Ubuntu on the notebook while my desktop is up and running smoothly with Ubuntu 8.04.

Try to google your notebook model number. In my case this is the way I found the incompatibility with my specific model.
