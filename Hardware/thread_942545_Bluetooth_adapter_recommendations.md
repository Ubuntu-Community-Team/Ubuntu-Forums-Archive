### Post by MystaMax on 2008-10-09
I've used the [Bluetake BT009SX Bluetooth]("http://bit.ly/2weiZV") Adapter w/o any troubles. I primarily use it for A2DP, but have connected a keyboard and mouse, again w/o any issues.

If you're looking to get another adapter, I would make sure its the Bluetooth 2.1 Spec. This will allow better battery life, Simple pairing (the ability to pair devices by setting them next to each other), and NFC (Near Field Communications). NFC will allow all types of [usable applications]("http://bit.ly/rVRZR").

I found an 2.1 spec adapter on engadget the other day, but couldn't find it for this post.
