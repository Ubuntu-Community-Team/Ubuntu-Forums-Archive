### Post by alenis on 2008-06-22
I would physically install the old drive, boot ubuntu - not vista -  and then:
a) use avast for linux to scan the drive if you want to keep the data;
b) repartition the disk with e.g. gparted if you don't need the data any more.
Of course, you must be careful to select in the BIOS the appropriate hard disk boot priority to avoid booting from the infected disk. This could harm your vista partition and, depending on the viruses you got, cause even more serious damage.
