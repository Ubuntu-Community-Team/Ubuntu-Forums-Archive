### Post by ordaide on 2008-09-12
If you are still having problems, reconnect (if possible) the drive to a window$ machine and then unmount the volume after detection (You should find a little usb icon in the system tray to do so), then reconnect to Ubuntu machine.  If it still doesn't work, initiate a chkdsk in windows by right clicking on the mounted usb drive and from disk tools menu select disk check.
