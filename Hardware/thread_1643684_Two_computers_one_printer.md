### Post by Fafler on 2010-12-12
Do she want the printer shared over the network or connected directly to the computer she wishes to use it with?

The easiest solution is to switch the printer, in which case you just need to install the printer driver on both machines. The printer doesn't care what it's connected to.

As for sharing the printer, with Linux as the host, install the printer and share it with samba. With Linux as the client, just go through the printers dialog and search for network printers.

Any combination of 32/64 bit for any of this will work.
