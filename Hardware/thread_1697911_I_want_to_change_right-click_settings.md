### Post by Zucriy Amsuna on 2011-03-01
I'm not sure how to do that (and I'm not sure if you want to; it hinders selecting multiple files, and also hinders the opening-in-new-tab Firefox ability). But you can use the Menu key that is on most keyboards. Usually, it is near the Alt key and Space bar (or the Windows key). The key's image looks like the actual menu, and some keys have a little mouse pointer on the menu image.

This key works just like the right click menu.
