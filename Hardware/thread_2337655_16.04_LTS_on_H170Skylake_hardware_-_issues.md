### Post by joker535 on 2016-10-06
Just in case anyone was following this, I built the machine using all the parts listed except the heatsink. I went with a noctua heatsink that has 4 heat pipes and a 90mm fan. I removed the fan and just let the 120mm case fan draw air through the fins since it is almost in line with the cooler. Temp monitoring shows the cpu is running 2 degrees above ambient at idle. Made a UEFI bootable usb stick using rufus on a Windows box and the install went fine. Everything works as it should. Letting it burn in for a night before placing it in service. The i5-6400t appears to be able to run circles around my old dual xeon setup. I am very pleased with the build.
