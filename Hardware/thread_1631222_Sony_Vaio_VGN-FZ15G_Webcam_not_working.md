### Post by gordintoronto on 2010-11-26
Go to Accessories/terminal and run the command: lsusb

When I do that, one of the lines says:
Bus 003 Device 002: ID 0ac8:303b Z-Star Microelectronics Corp. ZC0303 Webcam

The most useful part of that line is "0ac8:303b" which can be plugged into Google along with the word "linux".
