Hi there,

I'm new on this forum.
I've succesfully installed UBUNTU 9.04 on my Hp 2710p Tablet laptop.
I've also configured screen rotation by making a simple script that I post for other users.
This script executes when the user presses the Alt-F12 key setted up  thru gnome keyboard shortcuts.

```

#!/bin/bash

if [ -e ~/.pivot ]; then
     xrandr -o normal && xsetwacom set stylus Rotate none
     rm -f ~/.pivot
else
     xrandr -o left && xsetwacom set stylus Rotate CCW
     touch ~/.pivot
fi

```Is there any way to intercept the ACPI event generated by the screen rotation sensor and so, map the execution of this script in the /etc/acpi/events directory?

Thanks a lot for your kind reply
Marco
