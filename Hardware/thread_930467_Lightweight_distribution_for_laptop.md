### Post by snowpine on 2008-09-26
The absolute fastest distro I'm experienced with is SliTaz. I think it would fly on an eee (as it boots in 34 seconds and opens applications almost instantaneously on my old pentium 3 laptop with 256mb of ram). However, SliTaz might not be quite full featured enough for you; for example installing Openoffice isn't simple (though it does have Abiword and Gnumeric in its repositories) and configuring wireless takes a few steps.

If you're looking for something Ubuntu-based, I can heartily recommend Crunchbang. It's much quicker than Xubuntu. I read from the forums (at crunchbang.org/forums) that people have had success installing it on the eee. Fluxbuntu is really nice too, however it stopped receiving updates a year ago, so I would be concerned about support for the eee and other recent netbooks. A recent thread on these forums suggested it is difficult to install Fluxbuntu on a computer without an internal CD drive, obviously a concern with a netbook.

Definitely check out eeeuser.com; their forums have great information on trying different distros. From what I've read, hardware such as wireless is not supported out-of-the-box by most linux distros. There are distros such as eeexubuntu that solve that problem. 

My #1 suggestion (based on what I've read, not personal experience) would have to be the new Dell Mini 9 with a special Ubuntu remix pre-installed. Presumably it's pre-configured to work well with the hardware without any headaches, and you could certainly install a lighter windows manager (such as Openbox, LXDE, or Xfce) if performance is a concern.
