Que WinVista no te pida la pass para la conexion WiFi puede deberse a que la pidio la primera vez y despues quedo registrada o utilizaba la autenticacion automatica de Vista, si el router soporta esa funcionalidad).

Para saber que protocolo de seguridad tenes que usar con Ubuntu hay que averiguar cual es el que utiliza el router WiFi. Para eso tenes que ingresar en el setup del mismo y consultar el modulo de seguridad de red inalambrica (generalmente se hace usando un web browser apuntado a una direccion no publica indicada en la documentacion que te brinda el ISP) con un usuario de administracion (del modem/router).
