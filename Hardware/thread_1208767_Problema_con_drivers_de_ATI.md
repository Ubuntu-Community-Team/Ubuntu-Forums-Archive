### Post by Hei Ku on 2009-07-09
Podes desinstalar los drivers y poner los viejos. Pero posteá tu xorg.conf primero y lo vemos.

A mi me pasó algo similar y lo solucioné agregandole un par de opciones al archivo.
Lo posteo acá como ejemplo:

```


# xorg.conf (X.Org X Window System server configuration file)
#
# This file was generated by dexconf, the Debian X Configuration tool, using
# values from the debconf database.
#
# Edit this file with caution, and see the xorg.conf manual page.
# (Type "man xorg.conf" at the shell prompt.)
#
# This file is automatically updated on xserver-xorg package upgrades *only*
# if it has not been modified since the last upgrade of the xserver-xorg
# package.
#

Section "ServerLayout"
	Identifier     "Default Layout"
	Screen      0  "aticonfig-Screen[0]-0" 0 0
EndSection

Section "Files"
	FontPath     "/usr/local/share/fonts"
EndSection

Section "Monitor"
	Identifier   "aticonfig-Monitor[0]-0"
	Option	    "VendorName" "ATI Proprietary Driver"
	Option	    "ModelName" "Generic Autodetecting Monitor"
	Option	    "DPMS" "true"
EndSection

Section "Device"
	Identifier  "aticonfig-Device[0]-0"
	Driver      "fglrx"
        Option      "VideoOverlay" "off"
        Option      "OpenGLOverlay" "on"
        Option      "AccelMethod" "UXA"
        Option      "TexturedVideo" "off"
	BusID       "PCI:1:0:0"
EndSection

Section "Screen"
	Identifier "aticonfig-Screen[0]-0"
	Device     "aticonfig-Device[0]-0"
	Monitor    "aticonfig-Monitor[0]-0"
	DefaultDepth     24
	SubSection "Display"
		Viewport   0 0
		Depth     24
	EndSubSection
EndSection



```

Mi placa es una HD4850

### Post by Hei Ku on 2009-07-10
Si tenés problema con el video, yo le agregaría esto en la sección Device:

```

Section "Device"
	Identifier	"Configured Video Device"
        Driver      "fglrx"
        Option      "VideoOverlay" "off"
        Option      "OpenGLOverlay" "on"
        Option      "AccelMethod" "EXA"
        Option      "TexturedVideo" "off"
EndSection

```
Si eso ayuda, podes tratar de cambiar el AccelMethod de EXA a UXA.
