### Post by pihhan on 2008-11-04
Please specify, which graphics card you have. You can check that using command from terminal:

```

lspci | grep VGA

```

if you have intel card, you might have problem with known bug #173652 (see [https://bugs.launchpad.net/ubuntu/+source/linux-source-2.6.22/+bug/173652](https://bugs.launchpad.net/ubuntu/+source/linux-source-2.6.22/+bug/173652)). Having working backlight control is different from having working fn hotkeys. Ideally, they both should work, but that differs from model to model and equipment your model has.
