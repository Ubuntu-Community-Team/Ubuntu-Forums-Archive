### Post by cdtech on 2008-12-25
First you would need to create a directory for the share:
```
sudo mkdir /media/**share**
```
Then I would update the /etc/fstab to include:
```
# Entry for /dev/sdb5 :
/dev/sda5 /media/**share** ntfs defaults,umask=007,gid=46 0 0
```
> **BOLD** What ever you want to name it........
