### Post by pmcloone on 2011-06-30
Be careful of which version of the Precision M4600 you get - only the model with the nvidia card is certified for Ubuntu usage, and even then its certification page outlines that it can't use hibernate.

Currently, my M4600 has an AMD "unsupported hardware" watermark, can't hibernate, and has no sound. It was unstable initially, but the recent kernel update actually seems to have cleared the issue up (there's a first).

I only have this machine due to work - I've had much more success running ThinkPads with Ubuntu.
