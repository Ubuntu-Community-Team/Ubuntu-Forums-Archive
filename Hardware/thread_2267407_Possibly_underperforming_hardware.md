### Post by Mark Phelps on 2015-03-01
Wine is,at best, a "hack" to get Windows apps/games running (to some degree) in Linux.  It is NOT an Emulator -- so performance is going to suffer.

It's using the Linux video drivers, and from what I've seen on these forums, folks have had a LOT of trouble trying to get decent performance out of their AMD R9's in Linux.

If you want to do Windows gaming, then you need Windows.

### Post by Bucky Ball on 2015-03-01
Do a search for your games on the WineHQ apps database [here]("https://appdb.winehq.org/"). With Wine, some games will run great, others ok, some garbage.

Performance is going to differ from game to game depending on how well the game is supported by Wine.

Does the machine work fine when you are not using Wine apps, just regular, native Ubuntu?
