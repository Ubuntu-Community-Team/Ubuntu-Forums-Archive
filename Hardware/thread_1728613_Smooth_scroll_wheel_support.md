### Post by OliverN on 2012-01-28
hi there, I don't know if I'm committing thread necromancy here, but I just wanted to tell you that you could at least install a browser plugin that would emulate smooth scrolling there. It can't go beyond the limitations of the native driver, but it's still an improvement. I'm using Chrome with the *Chromium Wheel Smooth Scroller* plugin which is intended for OSX-like scrolling but you can easily configure it to just work as a "regular" smooth scroll plugin.
