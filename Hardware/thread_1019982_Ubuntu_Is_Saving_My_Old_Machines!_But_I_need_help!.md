It's fairly easy, during the Ubuntu installation you will get the chance to occupy the old Windows NT partition.
Just make sure you choose the right partition to install on.

Did you try the Ubuntu cdrom on it already ?
During the "live" session you will get an idea what will work and what might not work in Ubuntu after the Ubuntu installation.
Because the "live" session only uses the cdrom and the RAM memory things will be a bit slower, but it's a good option to test it.

By the way, you are using the recommended way of installing now.
First MS-Windows, and then Linux (not the other way around) :)

Xubuntu is usually recommended for older machines.

You can also try PuppyLinux [http://www.puppylinux.org/](http://www.puppylinux.org/)
or DamnSmallLinux to see difference in performance between those different Linux releases.
