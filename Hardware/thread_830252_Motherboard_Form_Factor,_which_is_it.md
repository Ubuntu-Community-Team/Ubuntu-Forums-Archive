### Post by Xyem on 2008-06-15
I seem to have hit a roadblock trying to find a replacement motherboard for a work colleagues PC as it no longer powers up.

I tested other ( known to work ) PSUs with the motherboard and none started it up. They did act strangely though, for example, the fan on the power supply would spin ( not very fast ) on one and twitch on another.

I tested if the switch was broken but that didn't show anything either.

I can only think that the motherboard is damaged ( there is a large shiny teardrop shape on a large chip.. ) but I cannot find what form factor it is!

It is about 220mm x 245mm and I can't find a single identifying mark on it. No brand, no model number..

The case seems store-brought and there is a label on the front which tells me some things like it is a 1.8Ghz Intel Celeron with 256MB DDR.

Is anyone familiar with the dimensions? Are motherboards this size a pain/expensive to acquire?

### Post by lswb on 2008-06-15
Google for motherboard sizes and you will find that a standard micro-ATX is 244X244 mm so your board is slightly smaller. You could measure the case and likely will find that a standard microATX size will fit, if not then there is a DTX size board with a size spec of 244 X 203.

### Post by logos34 on 2008-06-15
it's probably micro-atx.  But the case may accept either atx or m-atx (it depends on the mount holes). 

Check Newegg.com for an Celeron-compatible replacement (You need to know the socket type.  Maybe it's 478). You'll need to look around on the board carefully for tiny ID serial or model numbers.  Same for case.  Then google it.
