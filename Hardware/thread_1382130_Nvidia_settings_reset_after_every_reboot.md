### Post by ratcheer on 2010-01-22
> **the_mulletator said:**
> How do I edit my startup to run nvidia-settings? 

Here is one way to do it. I am not claiming it is the best way, but it should work.

Create a file in /etc/init.d named something like nvidia-start. Its contents should be one line:
nvidia-settings

Create a symbolic link in /etc/rc3.d named S85nvidia. This link should have a target of /etc/init.d/nvidia-start

Try that and reboot. If it doesn't work, you can delete the link and the file and back to the old drawing board. But I think it will work.

Tim

### Post by traverlaw on 2010-02-15
If the problem is that Nvidia settings do not kick in when you boot, but you have set your Nvidia X server settings, you may wish to try this.

Go to System > Preferences > Startup Applications  

Click on Add.  In the boxes for name use something like *Nvidia*

For command type *nvidia-settings --load-config-only*

For comment put in something like *Kicks Nvidia* to remind you why it is there.

Click save, click Close, and restart X sessions or reboot.  

Your Nvidia settings will now kick in during the final portion of the boot.  This worked for me using Ubuntu 9.10 (Karmic).  

Best wishes.

### Post by ZeroFossilFuel on 2010-10-04
I have had this problem too in virtually every flavor of Ubuntu. I am now using 10.04 Lucid. Here is what works for me. First, make sure you have the most current Nvidia drivers by doing a search ```
System/Administration/Hardware Drivers
``` Driver installation should place an entry in ```
System/Preferences/Startup Applications
``` called Nvidia X Server Settings. The command line entry in mine reads ```
sh -c '/usr/bin/nvidia-settings --load-config-only'
``` People swear up and down that you need it. I have found it doesn't even matter whether or not this entry is enabled. Here is what's important. Be logged in normally. Go to ```
System/Administration/Nvidia X Server Settings
``` Change your monitor settings to suit your needs. At the lower left highlight **nvidia-settings Configuration** and click **Save Current Configuration**. Let it save .nvidia-settings-rc to your default home directory. On the left highlight **X Server Display Configuration**. Double check to make sure your settings are still correct and click **Save to X Configuration File**. It won't let you save because you're not root. Click **Show preview...**  Copy the entire contents to your clipboard.

In terminal type ```
sudo gedit /etc/X11/xorg.conf
``` Save as **xorg.conf.username.bak**. Now paste the contents of the clipboard in place of the original contents. and save as **xorg.conf**. Restart and settings should stick.

I found that it did not work even running nvidia-settings as root then saving to X config file. For some reason I have to manually edit the xorg.conf as root in gedit to make it work. And there are STILL some settings that to not restore for me such as brightness and gamma settings tweaks I made until I launch Nvidia Settings from a shortcut I have created to it on my desktop because I got tired of navigating to it every time I restart. Other than that, basic settings do restore correctly for me so I guess that has to be good enough for now.

Hope I've helped lessen someone else's pain.
