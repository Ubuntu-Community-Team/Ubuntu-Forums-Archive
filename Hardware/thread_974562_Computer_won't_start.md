### Post by jerrylamos on 2008-11-08
Wild guess on my part.  8.10 Intrepid Ibex Ubuntu had a problem with certain video graphics chips flash memory being corrupted.  See
[http://ubuntuforums.org/showthread.php?t=927943](http://ubuntuforums.org/showthread.php?t=927943)

Perhaps somehow your video graphics flash memory got over written.  I wouldn't know how you get that refreshed/replaced on your hardware.  Perhaps it is a place to start if even CD Live doesn't start.
