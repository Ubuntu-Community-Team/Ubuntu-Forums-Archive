### Post by cos85 on 2009-08-30
The proprietary driver from ATI (fglrx) does not support the X1400 under Jaunty, unfortunately. You're probably using the open source driver (radeon), which can handle the 3d effects, but not any proper game graphics -- yet. I'm still using Intrepid which works fine.

New ATI driver releases don't support the X1400 under Vista either (I am told). It seems ATI really aren't very good at supporting their older hardware on any platform. Actually, they aren't very good at supporting their new stuff either. 8)
