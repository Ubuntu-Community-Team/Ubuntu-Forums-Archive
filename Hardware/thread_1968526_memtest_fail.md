### Post by Dngrsone on 2012-04-29
I'd say 98% of the, when you fail memtest, it's a bad or improperly installed RAM module.

On rare occasions, it will be a bad Northbridge on the motherboard (and/or a bad slot).

Try reinstalling your memory, making sure it is squarely seated.  If you have more than one module, then test each module individually; try changing slots and see if the modules run better in a different location.
