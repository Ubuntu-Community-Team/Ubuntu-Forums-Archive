Have you tried the "Hardware Drivers" option under the system tools (or whatever it's called. I don't have Ubuntu running right now)?  There is a broadcom 3rd party driver in there for ubuntu.  The trick is that if it is detected it may say that it is "enabled" but "inactive".  The cure is to "disable" then "enable" again,  and then reboot.  After the reboot you should be able to see wireless activity.
