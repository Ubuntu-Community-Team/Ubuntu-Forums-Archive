### Post by JustinR on 2011-04-17
Hey,

I have the same drive, at an average unencrypted rate of 251 MB/s, a fully encrypted disk runs at 200 MB/S, so you lose 51 MB/S, which might seem like a lot - but it's still 3-4 times faster than the average unencrypted HDD @ 45 MB/S. 

I've also been wondering how it would affect Garbage Collection protocols with the OS and device firmware and have had no problems so far - I've had it encrypted for around 4-6 months now without issue.

Take note though, TRIM doesn't work with encryption - but that's not a big issue.

Good luck!
