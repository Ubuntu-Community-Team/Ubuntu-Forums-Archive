### Post by tgalati4 on 2014-04-23
The problem with hardware that old is it uses a lot of electricity when running 24/7.  Put a kill-a-watt meter on it and measure the power consumption.  For Los Angeles power rates a 100-watt server running 24/7 will cost you $100 per year.  A 200-watt server will cost you $200.  You machine might push 300 watts with all of those disks spinning.

It's also hard to find those scsi disks and new ones are expensive.  Used ones are cheap but they have 50K hours on them.

freenas would probably work OK or a basic linux server.  

How much RAM can those machines take?
