### Post by jlh68 on 2009-05-17
You should use your WiFi WLAN and connect to it.

That aside, you should be able to connect to any that do not have WEP?WPA keys.  An unprotected site that is.  You will not be able to connect to the protected sites without knowing the password (key) to the site.

Second question is how does Ubuntu indicated that you WiFi is not working?  Especially if you can bring up a list of nearby WLAN sites.

### Post by jlh68 on 2009-05-17
Try RIGHT clicking on the WiFi Applet Icon.  Then select "Edit Connections" and click on that.  Next select the Wireless tab and go to your WLAN and use the left mouse button double click.  There is a box labeled "Connect Automatically", make sure it is selected.  Save and close.
