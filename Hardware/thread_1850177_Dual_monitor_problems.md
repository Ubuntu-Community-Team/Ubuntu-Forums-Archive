### Post by BlackRat90 on 2011-09-26
Aha! I found a solution and what I believe to be the issue, though I still have no idea what was wrong.

After rebooting I was able to configure the monitors properly via the ubuntu monitors program. Everything seems to run perfectly now with no issues, I will try running some of my games when I have time to see if this holds true.
