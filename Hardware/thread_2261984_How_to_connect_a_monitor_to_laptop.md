### Post by TheFu on 2015-01-22
Every time I connect a laptop to a monitor (VGA/HDMI) I run the GPU management tool to enable the 2nd monitor how I like it - either as "1" or "2" depending on which side of the monitor the laptop screen sits.

You can use xrandr to set this up, but there are slightly nicer tools based on the distro to control it too - LXDE uses lxrandr, others use arandr.  Sorry, don't know the name for the Unity tool - you can load any of these into any release and use them - that would make it distro-agnostic.
