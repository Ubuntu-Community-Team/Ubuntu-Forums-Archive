You can start by going to the major manufacturers, Hp.com, canon.com, lexmark.com, brother.com and the like and start downloading their linux packages and store them until you know what you need. You should know that Ubuntu supports many printers with cups program so you may not need them. But if you want to make sure everything is covered that would be your best bet.
