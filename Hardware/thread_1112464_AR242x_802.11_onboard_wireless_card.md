Ok, guys, here's the story.

In assisting a friend with his Ubuntu Installation on an HP Laptop, we found out that the onboard Atheros card would not work.  

After several tries, I finally got it working using the attached script which compiles and installs madwifi.

Just make sure that it is set to execute, then in terminal cd to the location you downloaded it to, and run ./install.sh

You need to do this in terminal, because there are certian prompts that you need to pay attention to.
