I would be suspicious of any material that said a mobo designed for a max of 8GB would be able to address 16GB.  The modules might physically fit into the slots, but half would probably not be useable.

Before you buy, you should consult with the manufacturer or thoroughly read the specifications of that board.

### Post by #&amp;thj^% on 2023-03-20
In a nut shell the 10600 is another designation for DDR3 1333 DRAM, PC3 indicates DDR3 and the 10600 indicates the theoretical number of M/Ts (roughly MegaBytes per second)
If it is ecc ram it is not compatible with your/that motherboard ECC ram is for a server.
