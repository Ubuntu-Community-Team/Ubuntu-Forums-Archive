### Post by Flompe on 2009-03-17
I found this on the net - had no time to test it yet:

...make sure you use the R8168 network module. The R8169 module loads by default which causes the network connection to "disappear" on the next restart unless you remove the power cord from the PSU. I loaded the R8168 module and no problems ever since.....
