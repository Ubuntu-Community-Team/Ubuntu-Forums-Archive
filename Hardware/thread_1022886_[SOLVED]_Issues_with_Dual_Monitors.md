**THE FIX:**
Dual-Monitors Different Native Resolutions:

Fix (Need Nvidia Graphic Card with Support Drivers):
 - Install Nvidia Graphic Drivers
	- /System/Administration/Hardware Drivers/ and tick the Nvidia Drivers
	OR
	- sudo apt-get install nvidia-glx (NOT CONFIRMED)
 - back up and edit the xorg
	- sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
	- sudo gedit /etx/X11/xorg.conf
*Under the module section, replace "nv" with "glx"
*Under the device heading, make sure that Driver says "nvidia"
*Under the screen section, add the following line:
Option "RenderAccel" "true"
 - Restart the X
	- crtl+alt-delete
 - back to edit xorg
	- sudo gedit /etc/X11/xorg.conf
*"Screen". Let&#8217;s add a few lines:
##This turns on NVidia&#8217;s TwinView
Option &#8220;TwinView&#8221;
##Here I&#8217;m setting the resolution to the individual monitors.
Option &#8220;MetaModes&#8221; &#8220;1280×1024 1680x1050&#8243;

*You may want to add these as well to the "Scree":
Option &#8220;TwinViewOrientation&#8221; &#8220;LeftOf&#8221;
**LeftOf can be LeftOf, RightOf, Below, Above, or Clone.
 - Restart your computer and your go to go
	- sudo reboot

Note: I had to do some tweaking with the actually gui within "Screen Resolution" to make sure everything worked correctly when the screens was clone mode.

EndSection

Section "ServerLayout"

  screen "Default Screen"
	Inputdevice	"Synaptics Touchpad"
EndSection
Section "Module"
	Load		"glx"
EndSection
