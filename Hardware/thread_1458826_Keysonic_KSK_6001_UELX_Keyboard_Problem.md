### Post by AGDownie on 2011-05-28
Had same problem myself recently and found that problem is due to "anti-ghosting" gaming keyboard feature.

You can switch this feature on and off using built-in hot keys:

Ctrl-Shift-Alt-N (normal mode - keyboard works in Ubuntu)
Ctrl-Shift-Alt-G (gaming mode - keyboard dead in Ubuntu, but works in BIOS/Grub/Windows). 

Hope this helps.
