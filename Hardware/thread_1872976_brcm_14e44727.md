### Post by coffeecat on 2011-10-31
What version of Ubuntu are you using? You have the BCM 4313 wireless card which works out of the box in Ubuntu 11.04 and 11.10 with the default open source driver. This is referred to as brcm80211 in 11.04 and brcmsmac in 11.10.

If you have tried to install the proprietary driver you have probably managed to blacklist the open source driver which is why it is not now working. If you want help unblacklisting it, post back with the output of:

```
ls -l /etc/modprobe.d
```
