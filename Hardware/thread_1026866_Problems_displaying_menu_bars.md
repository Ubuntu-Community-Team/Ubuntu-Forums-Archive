### Post by Therion on 2008-12-31
> **Texaskilt said:**
> When I run a program, such as OpenOffice, I am unable to see the menu bar (File, Edit, etc.).  ...  If I didn't know any better, I would say that this is a font issue.
Could be a font issue or a theme issue.  See if either of these things help:

Go to System/Preferences/Appearance "Fonts" tab and "Details" button.  Enable "FULL" Smoothing and Hinting.  If still no joy...

Go to System/Preferences/Appearance "Theme" tab and try using a different theme altogether.

### Post by Texaskilt on 2008-12-31
I found it with y'all assistance!

I needed to set the fonts to Subpixel (LCD) and that fixed it!

Thanks for y'all's help!!

TK
