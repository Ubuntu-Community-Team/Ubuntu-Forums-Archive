### Post by walt.smith1960 on 2011-08-18
Here's a decent source for memory information:
[http://www.crucial.com/index.aspx](http://www.crucial.com/index.aspx).  
Their memory advisor should tell you what you need to know.  It looks like you have 2 slots and can hold a maximum of 2 pieces (2 Gb.) of PC2700 or PC3200 DDR.  You may find that older memory costs more per Gb than newer packages.  I bought some pulled RAM off Ebay for an old notebook that has so far worked okay.  If you buy used RAM, make sure you can return it if it fails memtest.  Either way, I'd install it and immediately run Memtest for a couple hours at least, longer is better.
