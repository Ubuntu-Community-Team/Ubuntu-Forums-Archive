### Post by Grenage on 2011-05-16
If your TV has a VGA port and the PC has a VGA port, use a VGA cable. If the TV port has an HDMI port and the PC has an HDMI or DVI port, use an HDMI or HDMI-DVI cable.

It should be that simple; if you're using DVI or VGA, sound will need to come from a separate connection.  Hardware wise, I'd be surprised if it didn't have the balls to handle it - very surprised.  It might be slow, but it should work; go with a cut down desktop unless you're beefing the RAM.
