### Post by Favux on 2009-03-24
Hi chwu0410,

Welcome to Ubuntu!

The sound fix is well known.  To the file "/etc/modprobe.d/alsa-base" you have to add the following line at the bottom:
```
options snd-hda-intel index=0 model=toshiba position_fix=1
```
Then restart.  If you need more help let me know.

Several good TX2500 threads.  Search with TX2500 or TX25*.

Good luck!
