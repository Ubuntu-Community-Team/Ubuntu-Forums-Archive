### Post by silkstone on 2008-12-26
I've recently done a clean install of 8.10 on an Acer Aspire 5610 to replace 8.04. A couple of things don't work that did under Hardy.

1. I've set 'Acer Laptop' as the keyboard layout. If I go into Keyboard Shortcuts, I can select the Windows key (Super-L) to open my Home folder, and Super-L shows as the chosen key. However, it doesn't actually do anything. If I choose another combination (e.g. Ctrl+Alt+H) it works.

2. The wi-fi card on the 5610 is turned on and off by a spring-loaded toggle switch on the front of the machine. By default it is off, and I can't see a BIOS setting to change this. With Hardy I could boot up, then flick the switch and the wi-fi would be recognised and work. With Intrepid the wi-fi card is not picked up unless I flick the switch as Ubuntu is loading - while it says 'Starting up'. Everything then works perfectly.

Any suggestions very welcome!

EDIT/ In case it gives anyone a clue to the first issue, if I set Super-L to do something else like open a terminal, it works fine. But if I set it to open my home folder, it doesn't. Strange.
