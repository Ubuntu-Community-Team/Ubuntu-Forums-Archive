### Post by Camilia on 2009-06-23
My printer is a hp deskjet 3930v. 
I have bought new black cartridge for color cartridge is very low.

Tried printing with just the black cartridge in the printer. The result was a blank page. In HPLIP tool box print test worked. Got blank page from CUPS test page. 

I have the print out mode in HPLIP tools set for black cartridge. I have several time set the print out in CUPS to black. When I check it is reset to color printout mode. Thus decided to use the printer with black and color cartridge in.

I printed a page and it came out dark purple. 

Went to CUPS and changed the printout mode to black.
Then printed a page and it came out black

Printed another page and it came out green
Went to CUPS and changed the printout mode to black.
I printed another page and it came out dark purple. 

Basicly the color cartridge is almost empty. The black cartridge is new but not being used by the printer. I have bought 2. I hate to waste money on a color cartridge now, especially since I am drawing unemployment.

I a got the printer to work. I am ecstatic!

I read the instruction for testing printer in ubuntupocketguide-v1-1.pdf. Follow instructions to print a test. System > Administration > Printing, then double-clicked the printer&#8217;s icon and clicked the PRINT TEST PAGE button. Result was blank page. In printer settings changed print out mode to black cartridge. Now it prints. It seems printout mode has to be set at 3 locations, printing, HPLIP, and CUPS.
