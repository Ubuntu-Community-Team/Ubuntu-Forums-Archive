### Post by SpectrumDT on 2011-03-21
Hello. 

I am a fairly new Kubuntu user. I am running a fairly fresh installation of Kubuntu 10.10. I have an ATI Radeon HD 5750 video card. I am using the open source driver (because [the proprietary driver makes Kubuntu slow](http://ubuntuforums.org/showthread.php?t=1710021)). 

I have two monitors: A computer monitor (connected via DVI) and a TV (connected via HDMI). I would like to set them up like this:

1. A single desktop spread across both monitors.
2. The DVI monitor should be primary. I.e., I want the K Menu bar on this monitor, and I want all windows to show up on this monitor by default. 
3. The DVI monitor should run at its native resolution, which is 1920x1200.
4. The HDMI monitor should run at its native resolution, which is 1920x1080.

I have the following problems: 

**A** 
The machine forgets all my settings on each reboot. It sets both monitors to show the same display (instead of a desktop spread over both) and forces both to resolution 1600x1200.

**B** 
I can manually set up a multi-screen desktop by going to "System Settings -> Display and Monitor" and setting one screen to be "Right of" the other. When I do this, the system makes the HDMI monitor primary. 
I have tried using [the trick described here](http://ubuntuforums.org/showthread.php?t=486809) to make the DVI screen my primary. This trick does indeed move the K Menu bar, but all new windows still open up on the HDMI monitor. 

**C**
When (in a multi-screen desktop) I set the HDMI screen to resolution 1920x1080, it overscans, meaning that the top and bottom of the image are lost beyond the edges of the screen. (This does not happen at, say, 1600x1200 or 1600x1050.)

Can anyone help me with these problems?

(I should note that using the prioprietary ATI driver solves these problems, but causes a new performance problem which is worse, [described here](http://ubuntuforums.org/showthread.php?t=1710021).)

Thanks in advance.
