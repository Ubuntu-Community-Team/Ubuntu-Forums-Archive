Hi. I have a Toshiba Satellite L40-13G, which currently runs with Hardy Heron. I had to do some tweaking to get some things to work-- namely adding the line "options snd-hda-intel model=3stack" in alsa-base, and using the madwifi driver for my Atheros 5007EG (i think) WLAN card. So far, it's been great, with my only problems being having no control over screen brightness (the applet doesn't do anything), and having sleep and hibernate problems... (faster response, with no lines of error text from a cold boot).

I'm thinking of switching to Ibex. However, I do not know if my current workarounds still work, and I read somewhere the the hotkeys on Toshiba Laptop(example, the volume control) no longer work.

I'm still knew to linux, so I can't think of any workarounds myself if these ones stop working. So can anybody tell me if these will still work? Or if new tweaks are available?
