### Post by coffeecat on 2023-03-28
Are either one or more of the USB key, keyboard and/or mouse connected through a USB hub? According to a Google search you can get this error when drawing too much power from a USB port. If so, try with all USB devices and the pen key connected directly to a motherboard USB port. That is, if you have enough USB ports on your machine. Perhaps if you give us more details about the machine you are trying to install to, someone will be able to give more help. How many USB ports do you have on your machine? How many USB devices are you trying to connect to it? If you are using a hub, is it powered separately? If you don't have enough USB ports on your machine, experiment with as few as possible USB devices connected through a hub.
