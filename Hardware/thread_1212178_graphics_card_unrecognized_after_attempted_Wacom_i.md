### Post by Favux on 2009-07-14
Hi oki-vino,

I think you are almost there.  I'm not sure what you mean by activates.  If it's the stylus buttons I set the first up as a middle click and the second as a right mouse click.

You can configure them with wacomcpl.  It's also used to configure and calibrate the rest of the tablet.  You type "wacomcpl" (without the quotes) in a terminal.  See Section 3 here on how to set it up:  [http://ubuntuforums.org/showthread.php?t=1038949](http://ubuntuforums.org/showthread.php?t=1038949)  If it's the stylus tip activating near the tablet then we may need to adjust CursorProx.  Try wacomcpl first.  See if that fixes things.

The .xinitrc file it creates is a hidden file in "/home/username/".  It's basically full of xsetwacom commands.  To learn about them see section 9.0 (and also 5.1) here:  [http://linuxwacom.sourceforge.net/index.php/howto/main](http://linuxwacom.sourceforge.net/index.php/howto/main)
