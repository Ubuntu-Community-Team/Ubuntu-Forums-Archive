### Post by Ted3929 on 2010-03-17
Ubuntu on the Fujitsu T4310 tablet is not a difficult installation and actually improves system performance.  For the initiated, this tablet comes with 4GB DDR3 RAM, 320 GB HDD, b/g wireless, webcam, etc.

	Let me state from the beginning I intensely dislike Microsoft operating systems.  They just have this nasty tendency to start out all bright and shiny like a new penny and then end up like any other one cent piece that has been in circulation for a while.

	Windows 7 didn’t fail me in this respect.  I gave it a fair 3 month evaluation but after a couple gigs of downloads to correct various security holes and a progressively slower boot time as a result, I finally gave up and opted to install Ubuntu 9.10, Karmic Koala.

	First, let me advise any owners of this unit to be wary of the BIOS settings.  Apparently nobody has updated this area in close to a decade or so.  Upon checking I discovered it was set at the factory to boot from (in this order) floppy, CD and then HDD.

	While that may not sound odd, it really is.  Have you seen a new laptop with a floppy drive in the last 10 years or so?  And what about USB boots?

	Beyond this little bit of fluff (I merely reset the boot order to HDD, CD and then floppy) I installed the Super OS DVD and watched it load.

	Yes, I cheated and loaded Super OS instead of the standard Ubuntu only because the latter lacks codecs and drivers that are considered non-free.  I have a hunch that’s why the name changed from Super Ubuntu to the current Super OS.

	As typical, installation took all of maybe 20 minutes.  I opted to blow off the whole hard drive since Windows was proving problematic so that probably shaved a few minutes off the clock.

	I figured a few drivers and codecs would be lacking after installation but I was shocked to discover differently.  Everything worked with the exception of the tablet features.  No problem with my wireless, DVD burner, webcam or any other standard equipment.

	The tablet features proved especially difficult to install and I never did get it right.  Various forums recommended I install Wacom drivers for Linux but that did nothing but lock the OS to the point I had to reinstall it.

	If anybody has a sane resolution for this problem, let me know.

	And did all of this improve performance?  You bet it did.  Boot times average around 20 seconds from BIOS to POST and then final boot which is about 20 seconds quicker than Windows 7.  Not only is the boot time improved, but apps seem to work better.
