### Post by larynx on 2010-07-22
I'm running Ubuntu 10.04 32-bit on a Toshiba M305-S4910 laptop and once in every few reboots when I get to the login screen the resolution changes/stretches as if I have an external monitor hooked up and the login box appears in the external monitor so I don't see it.

The only way I found around this is to switch to another console (CTRL+ALT+F1) and then switch back to desktop mode (CTRL+ALT+F7) and that brings the resolution back to what it's supposed to be and the login box shows up.

The resolution also sometimes changes in desktop mode after rebooting (after logging in), it doesn't change while I'm working it just displays the wrong resolution after the login screen and then I have to switch to the console and back to desktop mode.
