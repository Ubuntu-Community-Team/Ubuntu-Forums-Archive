### Post by Mark Phelps on 2015-12-31
The problem you are having is the lack of native drivers for touchpads on laptops.

For a start, you should read the linked page about Linux on Laptops and then review the details in the Touchpads section: [http://www.linux-on-laptops.com/components.html](http://www.linux-on-laptops.com/components.html)

This is NOT a simple fix like in Windows where you just download a driver .exe file and run it.

Good Luck
