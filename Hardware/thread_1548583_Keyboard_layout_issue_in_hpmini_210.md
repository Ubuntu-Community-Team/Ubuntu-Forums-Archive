### Post by ljrmorgan on 2010-08-10
If you go to System > Preferences > Keyboard, and then click the Layout tab you can select a new layout. Click the "Add..." button and select the country where you bought your netbook, and click add. It sounds like "US" is the wrong setting - I'm in the UK and if my keyboard is set to US I get '@'s where I should get '"'s and all sorts of things are in the wrong place.

The layout tab also has a text box at the bottom labelled "Type to test settings" - type some stuff in there and if what appears isn't what you typed then you're on the wrong layout.
