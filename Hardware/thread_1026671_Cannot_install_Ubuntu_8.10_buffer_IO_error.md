### Post by brodde on 2009-03-12
It looks like the new kernel is more sensitive when reading CDs than earlier versions. 

Most threads about the error suggests burning a new CD at lower speed, and sometimes it helps. Some suggest replacing the reader and/or writer and trying again.

I think all of the people with this problem can burn and install 8.04 (or another distro/version/os) without problems.
