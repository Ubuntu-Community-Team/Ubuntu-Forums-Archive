### Post by fwrun2011 on 2015-06-25
I know that this thread is almost a year old, but perhaps someone will have something to add, as we still have the same issue with the Anker gaming mouse now that we had a year ago!

I've got the Anker 5000 DPI version of the gaming mouse and still looking for linux drivers. Apparently, Anker feels that because this is a gaming mouse - and most games run on Windows, that providing a Linux driver is not cost effective.
Interestingly though, I have macros assigned to the buttons, and they work in Ubuntu just as they do in Windows - so it is obvious that the mouse has some flash RAM where it stores the macros.
I do agree with you on the acceleration issue. It is quirky in Ubuntu. I can change sensitivity, but it isn't as effective as it is in Windows. My recommendation is to plug the mouse into a Windows box, set up the mouse, then move it to the Ubuntu box.
