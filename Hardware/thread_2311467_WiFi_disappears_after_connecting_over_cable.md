### Post by gogromat on 2016-01-27
Hello everyone,

I have a weird problem with my Lubuntu 15.10 setup.

I have a Lubuntu machine at home that I ssh into.
I tunnel VNC on port 5900 and PulseAudio on port 4713.

All was good for a week when suddenly I stopped having a Wifi on any of the devices at home.
I thought it was a problem with my router (Verizon Actiontec MI424-WR), but could not find what the problem was.
I even got a newer MI424WR-Gen3I router and the problem still persisted.
Only after unplugging ethernet cable from my computer going to the router I was able to get Wifi back.
So when my Lubuntu machine is connected to the router over ethernet cable kills the Wifi. 

So overall situation is as follows:
- My computer has 2 ethernet cards, both automatically configured with Lubuntu's default network manager.
- Wifi disappears if I plug any of the ports to these 2 ethernet cards.
- Wifi reappears after I unplug cable from my machine.

I have not yet looked into logs because I was busy but I will submit some soon.
