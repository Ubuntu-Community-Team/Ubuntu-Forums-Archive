### Post by kilee on 2010-01-01
Hi
My business require the use of a couple of programs that can only be run under windows. I've tried with wine without success (I'm not very good at custimising it) so I want to share the solution I found.

I have a amd athlon laptop, 2 gb of ram and enough disk space. I have one partition with windows vista and the other with ubuntu 9.10. So I had to switch from one to the other to use the windows programs. So I found the vmware converter: [http://www.vmware.com/products/converter/](http://www.vmware.com/products/converter/)
> 
Convert Microsoft Windows and Linux* based physical machines and third party image formats to VMware virtual machines

- I converted my windows machine into a virtual machine. It was a little bit large (30 gb) but everything was in it.


Then I downloaded the vmware player for linux [http://www.vmware.com/products/player/](http://www.vmware.com/products/player/)

and bingo. I just had to run my windows machine within ubuntu.

There is a particulary function that rocks: Unity. You use the programs of the virtual machine as if they were running in ubuntu, so I could work all the time in ubuntu.

Of course this is a solution for people who doesnt need all the computer capacity for the aplication of vmware, but I run all the microsoft office applications at once, music, internet etc, and my computer runs fine.
To work with a single folder for the files I make, I activated the shared folders, and I save all to a folder in ubuntu and not in the virtual machine.

One final tip. You can install windows from zero in the virtual machine with the vmware player, it makes the virtual computer smaller and it will use less resources.

You can also use itunes and your ipod this way!

I hope someone find this tips useful.

Wish you the best for the year.
