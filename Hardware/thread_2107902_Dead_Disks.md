### Post by Warren Hill on 2013-01-23
If the disk is faulty then replace it.  If the data is just corrupted then a format would fix it but the fact that you have tried installing Ubuntu suggests its hardware.  Diagnostic software will only confirm there is a problem it won't fix it.

As to computer is "too slow for Windows" you may want to consider Xubuntu or Lubuntu.  You can try each without installing as they will both run from the CD/DVD or USB stick, and they are also faster than Ubuntu with the Unity desktop.
