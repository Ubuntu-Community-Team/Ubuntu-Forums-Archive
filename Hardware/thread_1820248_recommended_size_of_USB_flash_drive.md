### Post by oldfred on 2011-08-07
If you just want an installer you only need 2GB. If you want a full install the minimum is 8GB as it now wants just over 4GB for the full install. I have used a 16GB flash and used 8GB for / (root) and created another 8GB for data but do not save anything in /home that still is part of /.

HOW TO Avoid Wubi & Install Ubuntu on USB Drive -amjjawad
[http://ubuntuforums.org/showthread.php?t=1650699](http://ubuntuforums.org/showthread.php?t=1650699)

It does not have to be encrypted BIOS based system:
Standard full install with screenshots to flash or SSD:
Ubuntu Encrypted Flash Memory  Installation
[http://members.iinet.net/~herman546/p19.html](http://members.iinet.net/~herman546/p19.html)

If a full install you will want to make some settings to minimize writes. A flash drive is slower both do the flash and USB port speeds, but reduces writes helps a lot. It is functional when set up correctly.
