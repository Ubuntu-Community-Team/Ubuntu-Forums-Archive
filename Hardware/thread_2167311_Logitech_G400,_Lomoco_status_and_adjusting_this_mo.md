Okay - I just bought G400. Wonderful mouse for gaming. But this mouse has one "feature" which is annoying - DPI changes with extra buttons, that can be accidentally pressed during playing.

So ofc I wanted to DISABLE this. Unfortunately, Logitech doesn't store mouse settings in mouse, but in their application, which is ofc Windows Only. 


There is a project aimed at Logitech compatibility with Linux, called [Lomoco]("http://git.lomoco.org/"). Problem is - last update was 2 years ago and it doesn't work with G400. I have googled a lot about this. Only one solution that gives some hope,[ is really hacky and tricky method.]("http://askubuntu.com/questions/117149/disabling-remapping-logitech-g400-mouse-dpi-buttons")

Does anyone know better method or know someone, who would be interested reviving Lomoco project?
