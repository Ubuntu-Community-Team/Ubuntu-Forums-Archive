### Post by joe74 on 2008-12-23
All of a sudden my audio got unchecked from services lists. It used to go off when my laptop overheated by use, and *only* using the dock base which has the cd-rw/dvd rom, ports and usb connections. But now it is not working with or without the dock base. It only works when I choose "Failsafe GNOME" session otherwise there's no way. The sound services start un-checked when I restart using the last session, or it only works with programs using Pulse Audio.

There's a friend in my local LUG that has the same problem with a Dell laptop.

Would you please give us a hand in this issue?

### Post by joe74 on 2009-01-05
Problem solved a week ago. First I had to login as "Failsafe GNOME", always. Then I had to set "Failsafe GNOME" as the default session in System >>Administration >>Login Window >>General(tab) checked the "Failsafe GNOME" as the default session and problem solved.
