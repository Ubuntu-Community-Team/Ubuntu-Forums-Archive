Hey, had a quick question. I've had a file server running successfully for about 3-4 months booting Ubuntu Server from a USB flash pen drive. However, attempting to clean up my case, I bought a supplemental PCI card with 2 external and 1 internal USB 2.0 ports. However, once I installed it and moved the pen drive over to the internal USB port, the server no longer recognized it. Moving it to one of the new external ports...same issue.

However, when using one of the ports attached to the motherboard, it boots as normal.

My question is, is there any way to make it so that it will read the PCI card and boot from it?

Thanks for your help!
