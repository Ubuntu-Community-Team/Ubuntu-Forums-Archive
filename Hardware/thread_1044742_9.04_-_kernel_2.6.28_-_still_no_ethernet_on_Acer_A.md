### Post by bobwyajnr on 2009-01-19
OK folks... Its working!!

I set the MTU to 1500 using ifconfig:
```
ifconfig eth0 mtu 1500
```


Then I edited /etc/network/interfaces adding:
```
auto eth0
iface eth0 inet dhcp
```

Internet/DNS fully working now under Jaunty Jackalope!!

Bob
