### Post by Detroit_Bad_Boy on 2009-09-26
I am not sure if this will last or not, but here is what I simply did to get my Compaq Presario M2000 laptop online wirelessly with kUbuntu:

 I opened a terminal and typed (without internet connection): sudo apt-get install ndisgtk.

It began to install what it could from files it has within linux. It had also basically said it couldn't draw the other files necessary to complete the install from the internet as I was not connected to it. 
  At this point I had been been working on it for several weeks with no luck.
This afternoon I plugged my laptop directly into the router and thought I would try to get online that way for a fix. I realized that with the connection, linux was downloading the files it couldn't from last nite as I was continuing my last session. 
  Now I am connected wirelessly and am hoping this connection lasts.
  I have the Broadcom BCM4318 chipset in my laptop as the wireless controller. I do not know if this will work for anyone else but I hope so. I figure that you would need an ethernet cable and a direct internet connection to get the install to complete for you. I am also assuming that you would need to restart your computer to make it take effect.

  Hope this works for anyone else who has the same laptop as me with the same wireless chipset built in!!!!:P
