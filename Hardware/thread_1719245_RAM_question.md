### Post by metalf8801 on 2011-04-01
The timing and cas latency are different. However, you will most likely be able to get away with using, but it may cause the faster RAM which you already own to under clock itself so that it runs at the same speed as the new RAM you are looking at. That being said I've heard that for the best results you want to use exactly the same kind of RAM when adding more.  

I hope this helps
