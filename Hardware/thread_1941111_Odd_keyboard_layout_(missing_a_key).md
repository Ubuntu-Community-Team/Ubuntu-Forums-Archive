### Post by dandnsmith on 2012-03-16
"same as a regular keyboard" - doesn't really help, as there are so many 'regular' keyboard layouts - even American ones (I used to work in computers at an age when most suppliers didn't realise that even golfball typewriters didn't have always the same keyboard layout, and then 'standard' terminal keyboards got even more diverse).

You may have to decide which theoretical layout will satisfy your needs most of the time, select that, and learn the sequences for the key(s) that that doesn't give you. You could also look to binding the one(s) you don't have to easy sequences (I don't claim easily to be able to supply the detail you need to do this).

### Post by winh8r on 2012-03-16
Can't tell you which layout to use for sure but you should be able to type a backslash by using:

```
control + shift + U005C
```

until you find the right layout at least.


I know this is a bit of a hassle but it beats not being able to type a backslash at all!

Hope it helps in the meantime.
