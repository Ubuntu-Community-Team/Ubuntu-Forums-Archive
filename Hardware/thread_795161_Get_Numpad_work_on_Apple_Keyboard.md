### Post by Tompalaz on 2008-05-15
I had a problem that numpad didn't work on my new *shiny* ;) Apple keyboard.

My keyboard is Apples new Aluminum Keyboard and I use Ubuntu 8.04

I asked Google how to get it to work, but no useful answers.
I checked around in the settings and found this:

#1 First off, I choosed "Apple" in the "Keyboard model" located at System - Preferences - Keyboard - Layouts.

#2 Press Layout Options

#3 Press down the "Miscellaneous compatibility options"

#3 Check the box that says "Numeric Keypad keys work as with Mac"

Then try your numpad keys: 0123456789/*-+, :)


My keyboard stop to answer if I press the numpad key, but starts to work again if I press F1 F2 F3 F4 F5 F6 F7, for some reason.
