Hi,

I'm trying to get my HP Deskjet D1668 printer to work on Ubuntu 10.4. I've installed HPLIP, which has successfully detected the printer, and allows me to print out one-page documents (which I couldn't do before installing HPLIP). 

However, if I try printing a document with more than one page, the first page comes out fine but then just after the second page is fed to the printer, it stops. The HPLIP status says the document has been printed, but with CUPS a "Print error" window pops up, saying "There was a problem processing document X (job x)". Before this happens, the CUPS status says "Processing - printer not connected?" even when the first page is being printed.

If I go though the CUPS printer troubleshooter, it tells me that the printer's state message is '/usr/lib/cups/filter/hpcups failed'.

I've looked at a few threads here and elsewhere to find solutions, but nothing has worked. I'm a bit confused by what the difference between HPLIP and CUPS is and how they're meant to work together (which they don't seem to be able to do very well). Also why HPLIP seems to want to install itself in my home directory rather than somewhere more sensible and less irritating.
