Here we are...

```

root@pc-V-h81-intel-4570-desktop:~# apt-get install hw-probe
root@pc-V-h81-intel-4570-desktop:~# hw-probe -all -upload
Probe for hardware ... Ok
Reading logs ... WARNING: X11-related logs are not collected (try to run 'xhost +local:' to enable access or run as root by su)
Ok
Uploaded to DB, Thank you!

Probe URL: https://linux-hardware.org/?probe=fac0a305d4
root@pc-V-h81-intel-4570-desktop:~#

```
