GPUs use more than just pure Mhz (core clock). Memory bus, shaders, texture mapping and render units have a large impact of performance. The 5770 has more shaders than a GTX 260, but the GTX has higher memory bus, more texture mapping units and render output units. 

BTW, the '4800' for the memory speed on the 5700 is a bit of a misnomer. IIRC, its 1200 (x4 cause it uses GDDR5 which is clock quadrupled, not doubled like normal DDR). Also, there is no GTX250- its  GTS250, or GTX 260. 

I'd get a GTX 260 over a 5770. Its faster (and linux with ATI is still an issue)-
