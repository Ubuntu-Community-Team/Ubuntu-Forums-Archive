Hi,

Try this, if you haven't already, open up configure printer from Administration, when the window opens click on the icon for your printer, from the drop down list select properties, in the new window select "Printer Options", in this window look for "Color Model" select the appropiate one, CMY as mine is, while your here check everything else out to make sure it matches your printer, hope this helps.

P.S The cups driver may not be good enough, if the problem persists, go to the "Avasys" site and get the driver from there.

### Post by wiresquire on 2010-07-14
In case anyone happens across this, from posts elsewhere, it appears that to get 'black' this model printer sprays a bit of each colour down (which makes black), and then sprays some black on top of it. To make it really black ? To be serious, that could be a good thing, as it could make the black run longer. 

But...Except....

**The printer will stop working if ANY of the colours has 'run out'.**

What it means, is that you really need to have a set of colours available always, so keep this in mind if you consider this printer.

I have changed the default printout settings to Grayscale. This still appears to suck some colours. I have also set the print quality to "Economy". For most of my printouts, which are basic text pages, it really doesn't matter.

I could buy a new printer for the price of replacement cartridges. I find that offensive, so I have been buying 'non-genuine' ink cartidges from ebay. So far, so good. I really don't really print that much anyway, and virtually nothing in colour, so YMMV.
