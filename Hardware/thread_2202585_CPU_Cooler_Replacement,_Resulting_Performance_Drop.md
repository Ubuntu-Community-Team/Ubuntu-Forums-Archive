I recently had a CPU cooler fan fail, and used my processor's OEM cooler as a backup. While it is a bit noisier, and my cpu temps get a bit higher. It works, for the most part, well. I've begun to notice though, that I seem to be experiencing a performance hit after having replaced the cooler. 

I get the temperature and noise differences, but I shouldn't, generally, be seeing an actual decline in system performance should I? By "performance" I mean, I seem to be experiencing interface pauses and slowdowns, window manager responsiveness, mouse and keyboard input pauses, etc. This machine is no hotrod, but I beleive there definitely has been a slow-down post cooler replacement, which I don't understand.

### Post by tgalati4 on 2014-01-30
Many processors will slow down when approaching dangerous temperatures.  It's possible that your temperature sensors are not accurate or there is something else going on.  I would replace the fan (they are cheap) and get some silver or ceramic (good quality) heat paste.  Watch some youtube videos on the correct way to prep and apply the cpu heat paste.
