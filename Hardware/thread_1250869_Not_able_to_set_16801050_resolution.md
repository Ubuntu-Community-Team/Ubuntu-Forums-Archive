Try this for the Screen section:

Section "Screen"
        Identifier      "Default Screen"
        Monitor         "Configured Monitor"
        Device          "Configured Video Device"
        Defaultdepth    24
EndSection

The 1440x900 resolution seems to be implying that you only have 4MB of video memory. so....

Also, make sure in your BIOS that you have shared enough memory to the video subsystem. With the 24 bit depth (3 bytes per pixel) at 1680x1050 you will need 3x1680x1050 bytes or 5.3 MB. So make sure you have at least 8 MB shared memory setup in the BIOS.

	HorizSync	40-80
