### Post by Crafty Kisses on 2008-09-10
Haha, no problem man anytime, there's also great beginner documentation at the following links:

[http://www.psychocats.net/ubuntu/index.php](http://www.psychocats.net/ubuntu/index.php)

[https://help.ubuntu.com/](https://help.ubuntu.com/)

Those links should get you started, and the video card drivers are usually click and install, with the program Envy and the Hardware Drivers option, it's really easy. I say after 1 month, you'll get the hang of Ubuntu, remember some stuff can be difficult but that's why the forums are here.

There was also a great tutorial on howto install anything on Ubuntu, but for some reason it's down, but these are the three main ways to install programs on Ubuntu.

[LIST]
[*]Synaptic Package Manager
[*]Command Line
[*]Compile From Source
[/LIST]

Basically with **Synaptic Package Manger**, you can install things quickly with a click of a button, and it will do all the work for you. (usually). It's very stable and easy to use.

Then there's the **command line** option, if you don't feel like going into Synaptic and you know the package name off the top of your head, you can do the following: ```
sudo apt-get install package name
```So for example if I wanted to install Wine, it would go like this: ```
sudo apt-get install wine
```Then Wine is already installed.

Then there is the option to **compile from source.** Some packages require you to do this, there is a great tutorial on how to compile packages on the Ubuntu documentation, but remember to have the package **build-essentials** installed or you cannot really do this. You can install build-essentials by doing the following: ```
sudo apt-get install build-essentials
```
After that you should be set, read the documentation, read up on how to install things, mess around with Ubuntu for awhile get used to it and remember if you have any issues please post here at the forums, the forums are a great way to learn about Ubuntu. Hope this helps. :)
