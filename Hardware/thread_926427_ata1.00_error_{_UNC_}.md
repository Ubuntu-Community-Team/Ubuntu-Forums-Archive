### Post by jpkotta on 2008-09-23
Yes, this does not bode well: [UNC]("http://ata.wiki.kernel.org/index.php/Libata_error_messages#ATA_error_expansion").

You should think about a new hard drive maybe.  Definately backup your important data.

You can install from a USB drive or the network if your BIOS allows it, although it is trickier than just burning a CD.  I've never done it, but a coworker made a bootable USB hard drive at work.  We had to jump though some hoops, but it wasn't too bad.  Start [here]("https://help.ubuntu.com/community/Installation").
