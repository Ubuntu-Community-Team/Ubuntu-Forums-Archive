Before [installing Ubuntu]("http://www.easy-ubuntu-linux.com/ubuntu-installation-606-1.html"), you should also check your computer against the hardware requirements.  The system requirements for Ubuntu are as follows:  

[LIST]
[*] 700 MHz or better processor
[*] 3GB of available disk space
[*] 256MB of memory (RAM)
[*] CD-ROM drive
[*] Ethernet interface
[*] VGA graphics interface
[*] [SIZE=3]If all of this sounds a tad high of spec, you may want to try Xubuntu. The system requirements for [Xubuntu]("http://www.easy-ubuntu-linux.com/other-ubuntus.html") are as follows:  [/SIZE]
[*] 500 MHz or better processor
[*] 1.5GB of available disk space
[*] 192MB of memory (RAM) for installation but only 128 MB of RAM to run from the installation CD
[*] CD-ROM drive
[*] Ethernet interface
[*] VGA graphics interface
[/LIST]
    Once installed, Xubuntu only requires 64MB of RAM to run.  Of course, installing it with 192MB of RAM and running it with only 64MB means that you either temporarily insert some RAM into the computer or swap out the harddrive to a computer with more RAM.  Either way, some IT shoehorning is required if you do not have 192MB of RAM. 
  Once you have checked your system against Ubuntu's system requirements, you are ready to [install Ubuntu]("http://www.easy-ubuntu-linux.com/ubuntu-installation-606-1.html").
