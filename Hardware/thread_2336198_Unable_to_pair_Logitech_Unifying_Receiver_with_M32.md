### Post by him610 on 2016-09-05
Objective: Pair previously unpaired Logitech Unifying Receiver with Logitech M325 wireless mouse.

Problem: Unable to meet objective of pairing Logitech Unifying Receiver with M325 mouse.

Resources: Ubuntu 16.04, Chrome Vers 53.0.2785.92 (64 bit), Chrome App Unifying Receiver v 1.0.1 Logitech.

Sequence : 

1.  Attempt to pair Unifying Receiver with M325 resulted in Unifying Receiver undetected by Logitech App. Can't pair when receiver is not detected; although, Low Mouse Battery message appeared on screen.

2.  Repeated #1 on two other computers configured as described in Resources.

Comment:  I have continued to maintain Windows 7 on a dual boot desktop. 

3.  Last resort, download/install Unifying Software vers 2.50.25 for Windows 7 from 
[http://support.logitech.com/en_us/software/unifying]("http://support.logitech.com/en_us/software/unifying")

4. Executed Unifying Software application from Windows 7. 

Results: Unifying Receiver detected and paired successfully with M325 mouse from Windows 7. 

From the beginning, I expected the two Logitech devices to successfully pair using Ubuntu/Chrome. I was disappointed when it did not work with that combination, but I was glad to finally get them paired. :)
