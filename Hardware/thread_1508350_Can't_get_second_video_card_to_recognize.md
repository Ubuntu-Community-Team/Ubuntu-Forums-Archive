If ya'll don't mind a brief indulgence though, this system is dual booted - the exact same configuration in Windows XP took <5min, which includes the boot time. I love Linux as a development environment - but despite how far it's come since I first started working with it, it has a long way to go before I ever recommend someone who's not an IT "professional" already ever use it as their desktop.
