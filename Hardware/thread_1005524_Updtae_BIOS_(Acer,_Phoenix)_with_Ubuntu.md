### Post by russo.mic on 2008-12-08
well, you need to first off check the manufacturers instructions for running a BIOS update.  every company does this differently. 

If there is a program you download to create floppies or a USB jump to update, you could do that in Ubuntu via wine, I suppose.  It would probably work.

However, ASUS for example does there BIOS update directly from the OS, with a small application that you run.  Although I can't be sure, I would instinctively NOT run that from Ubuntu if it's designed to be a Windows app.  I wouldn't even begin to try and predict the results of that.  But It's conceivable it could hose your bios...i would think...More likely it just wouldn't work at all.
