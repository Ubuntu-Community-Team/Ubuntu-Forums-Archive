### Post by zonemikel on 2009-11-21
I figured it out, after installing the driver in windows on boot it would tell me the "supplementary power cable" was not installed, so it had to run in low graphics mode or something like that. 

I tried to run a simple game and .... bam same as before black screen. Apparently this video card needs more power or it cant really do jack. 

Its a gf 6800 ultra overclocked btw .. not a GTO like i thought. 

I'll have to find the adapter because i cant figure out the pinout, when i get the adapter im sure it will work.
