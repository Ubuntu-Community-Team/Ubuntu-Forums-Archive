### Post by Jose Catre-Vandis on 2012-05-21
Well Linux to the rescue once again!

I downloaded Slitaz 4, shoved the CD in the FS Amilo, it booted up and lo and behold keyboard working (well nearly all keys CTRL doesn't work) (Slitaz 4 has a countdown boot so no need to interact early)

Installed Slitaz and well on the way to a digital photo frame.

Keyboard would not work with any other live Cd I tried, and I have not risked going into the bios in case the kb packs up again.
