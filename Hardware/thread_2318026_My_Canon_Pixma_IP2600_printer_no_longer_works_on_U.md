Problem now solved thanks to askubuntu : [http://askubuntu.com/questions/230039/installing-canon-ip2600-printer-drivers-on-ubuntu-12-10](http://askubuntu.com/questions/230039/installing-canon-ip2600-printer-drivers-on-ubuntu-12-10) first answer.

> sudo add-apt-repository ppa:michael-gruz/canon-trunk
sudo apt-get update
sudo apt-get install cnijfilter-ip2600series
