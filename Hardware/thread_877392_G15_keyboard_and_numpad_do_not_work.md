### Post by kcsx on 2008-10-05
I had a real problem with my numpad recently because I had accidentally set the numpad to control the mouse in the mouse tab in the keyboard settings. Took me forever to figure it out. Try unchecking and rechecking the "Allow to control the pointer using the keyboard" option in keyboard settings and see if that helps.
