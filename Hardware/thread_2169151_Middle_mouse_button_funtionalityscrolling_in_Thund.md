### Post by imstochastic on 2014-04-17
I was able to get middle-click scrolling working inside thunderbird by enabling autoScroll.

It can be found under preferences -> advanced -> General -> use autoscrolling

the "about:config" for thunderbird can be reached from that same panel (preferences->advanced->general), via the "Config Editor..." button. the "general.autoScroll" performs the same function as the autoscroll checkbutton.

*my* problem is that while that lets me scroll through the body of the message, it does *not* let me scroll through lists (like the list of messages)...
