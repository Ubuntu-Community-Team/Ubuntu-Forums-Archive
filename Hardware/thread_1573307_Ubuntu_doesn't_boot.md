### Post by Lateralis on 2010-09-12
I am afraid you need to be a little more descriptive as to what the problem is.  Ubuntu might not boot for a whole bunch of reasons, each reason requiring a different solution! 

When you start up the PC, do you get to the GRUB menu?  Can you select an operating system from the GRUB menu?  If you can and it appears to start booting, how far does it get?  What error message(s) do you see?  Do you get dropped to a recovery shell?  

As for trying the CD, when did you encounter the Input/Output error?  What were you trying to do?  Did the LiveCD boot properly?  Can you see your Ubuntu filesystem from the LiveCD and does it mount OK, i.e. can you go to the Places menu and select the filesystem?
