### Post by User21 on 2008-08-05
Toco de oido. Nunca usé placas de video ATI... 
pero, [**esto**]("http://ubuntu2.wordpress.com/2008/01/02/ubuntu-tip-1-dual-head-on-ati-radeon/") te sirve?.

```
 sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
sudo aticonfig --initial
sudo aticonfig --initial=dual-head --screen-layout=left
```

Reiniciar X con Ctrl-Alt-Backspace

```
 sudo aticonfig --dtop=horizontal
```

Reiniciar X con Ctrl-Alt-Backspace

Se asume que estan conectados ambos monitores al momento de hacerlo! 

Es una solucion para Gutsy ... Si no funciona, pues, obviar.. ¬_¬
