### Post by eskararriba on 2012-04-09
I'm not sure if I'm the correct person to answer to your question, as I got my e420s only two days ago. so testing has been limited... anyway, what I can tell you is that pretty much everything worked out very very smoothly, no problems whatsoever with all the things that cost me a lot of work to fix on my old hp, some four or five years ago (sound, video, microphone, function-keys...). BUT: there is one really annoying issue, which is that there seems to be a problem with the wifi-adapter. works out of the box, no configuration needed, but it's slower than the 14.4-modem I had back in the 90s. it's a little weird, cause they useed the same card my 420s came with (intel centrino n-1000) when they certified the computer on 11.10; and, as you can see in the bug-report, it works perfectly well with older ubuntu-versions: [https://bugs.launchpad.net/ubuntu/+source/linux/+bug/836250](https://bugs.launchpad.net/ubuntu/+source/linux/+bug/836250)
anyways, I read that this problem will probably be fixed with the 12.04 version, maybe you'll want to wait. 
cheers
ps: since my first contact with vista, I never again used windows, and I don't have any partition. some pople say win7 is better, but after starting it and being confronted with about fifteen popups I decided to delete it inmediatly.
