### Post by listener on 2008-09-13
Sounds like you're running in the BIOS setup's version of 'safe-mode'.  If the computer runs satisfactorily this way, it probably is ok.  Maybe better than attempting to make BIOS settings you aren't sure about.  I think if you'll enter BIOS setup and exit with save (making NO changes) the message will go away.

Maybe 'safe-mode' isn't a good analogy though, it probably just means the BIOS setup defaults which are probably fine for the system.  If you have ANY doubts about going into the setup, please don't.  Lots of stuff to really mess up.

good luck
