### Post by stchman on 2008-06-26
> **kaldor said:**

I'm thinking of getting a Dell Inspiron 1525 Notebook with core duo processor. Here is what I want to know:

1) How well do games work? Can it play modern games? (At least like quake 4)

2) Will Kubuntu or any other version of Linux for that matter work well? I want to dual boot Vista with Kubuntu.

First off that laptop has an X3100 video card.  Not exactly a gamers card.

Second, Quake 4 is a very modern game and will probably run terrible on that laptop.

Third, if you are going to do a lot of gaming then get a desktop or laptop with a dedicated PCI-E video card with at least 256MB.  Make sure to get an nvidia card.  For desktops get an 8800GT for laptop get an 8600GM.

Fourth, if you are a gamer I do recommend that you have XP or Vista as not all games will run well or at all using WINE.

My Toshiba A205-S5859 has the X3100 card and it can play Open Arena ok, but nothing to write home about and that is the 6 year old Quake III engine.  Quake 4 uses the newer Doom III engine.  Granted opengl games play better with Linux, but you will only be able to use Mesa on the X3100.

DirecX games can be quite problematic with WINE.  So Ubuntu is not the #1 choice for Windows games.

So to answer your questions:

1.  Maybe.  Depends on the game.
2.  Definitely Ubuntu Hardy OOB.
