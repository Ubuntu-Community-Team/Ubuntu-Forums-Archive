### Post by BenM on 2008-05-11
I have a Zune and would advise against getting one if you are running linux. From what I've read, the only way to rune the Zune software is by using [VirtualBox]("http://ubuntuforums.org/showthread.php?t=492816") ([and here]("http://ubuntuforums.org/showthread.php?t=646969")) It sounds like a linux alternative for Zune software unlikely to happen soon (if ever). 

In addition, Zune only supports AAC, WMA and MP3 files. If you want to use FLAAC or Ogg, you are out of luck. While the player itself isn't to bad, its reliance on proprietary/DRM software has really made me regret my decision to buy one.

### Post by RAOF on 2008-05-12
Of course, the iPod is only an option if you never use iTunes with it (or you never update iTunes).  Apple changes the iPod database format pretty much each time it updates iTunes, meaning that all 3rd party software breaks.  Even better, iTunes will silently update your iPod database, just by plugging it in.

You could also try rockbox on your iPod, although I don't know how well it supports new iPods (or old ones, for that matter).  This, of course, voids your warranty.
