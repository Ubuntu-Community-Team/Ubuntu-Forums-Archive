### Post by kpizzolatto on 2020-08-06
EDIT
***Putting the solution at the top to make it easier for the next person; please excuse if this is inappropriate***
Briefly: 
Problem:      After kernel update, new kernel loads no video driver; also a lot of other hardware doesn't work.  booting to old kernel works OK
Symptoms:  /lib/modules/$old-generic/kernel/drivers/gpu/drm has a lot of stuff that /lib/modules/$new-generic/kernel/drivers/gpu/drm does not have
Solution:      sudo apt-get install linux-modules-extra-$new-generic
Thanks to all who helped lead me to this solution!
