### Post by miskopo on 2012-08-26
Hi, at first, ensure your system is up-to date
> sudo apt-get update && sudo apt-get dist-upgrade
 from terminal. Then, try to check closed drivers (try to look for "drivers" in Unity menu). Or try Ubuntu software centre.
If you don't find anything, try these instructions, it should work also for 12.04
