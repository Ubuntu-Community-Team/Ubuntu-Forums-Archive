Yes, Ubuntu will recognise that you have a laptop during install and will add the additional bits it needs.

Of course, if you have a laptop that has just been released and it has cutting-edge hardware, it may be several months before Linux drivers are released.

### Post by Qinjuehang on 2008-07-09
Ubuntu can run badly on a laptop, and it can run very well, depends on your configuration. Out of the box, it drains your battery quite a bit. However, there is Powertop (sudo apt-get install powertop), and a bunch of advice on lesswatts.org. After slight modifications, you'll be able to laugh at Vista's battery life! I run Ubuntu on a Compaq V3600 (couldn't remember the model, and it failed magically 1 month before warranty ends :)...I'm not complaining, but its at the service center). However, you might have slight problems with Intel speedstep...but I'm sure if you encounter any problem the forums will provide plenty of help.
