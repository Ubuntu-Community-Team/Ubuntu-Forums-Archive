Welcome. Make sure you have HPLip installed and reboot. Install it from Software Centre or [have a look here]("http://hplipopensource.com/hplip-web/install_wizard/index.html"). You may need to enable the Canonical Partners repository in Software and Updates> Other Software if you can't find it.

HP are very Ubuntu-friendly.
