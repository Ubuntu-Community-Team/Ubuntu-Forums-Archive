### Post by Bucky Ball on 2011-09-20
You should probably be asking if Ubuntu will work on that machine cos if it will, so will Wine. The next question you should ask is whether your games will work with Wine and that is unknowable as they aren't out yet.

Your comment that performance drops with Wine is not really right. Some users report better performance with some games on some rigs. Other games don't work at all. You are best to check the WineHQ and do some digging.

[http://appdb.winehq.org/](http://appdb.winehq.org/)

I notice 'Guild Wars, all versions' is listed on the first page, top ten platinum list. Just search for your other games there and see what you find.
