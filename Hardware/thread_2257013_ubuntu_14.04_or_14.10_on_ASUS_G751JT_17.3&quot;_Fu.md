### Post by sudodus on 2014-12-16
At least nvidia claims it has a linux driver that works with that chip :-)

Most but not all computers have UEFI systems, that can be switched into BIOS (CSM) mode. But you may want to run the computer in UEFI mode to be able to switch conveniently between Ubuntu and Windows. Then you might want to run UEFI without secure boot and without fast boot. There should be entries in the UEFI menu system to switch those settings.

See these links

[https://help.ubuntu.com/community/UEFI](https://help.ubuntu.com/community/UEFI)
