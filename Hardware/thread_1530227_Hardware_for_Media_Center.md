### Post by alexandrum77 on 2010-07-13
I use a similar configuration except for the CPU AMD Athlon II X3 435 Triple Core which is more that enough for a HTPC or a Media Center. The MB that I use is a Gigabyte GA-MA785GM-US2H AMD785G (Radeon HD 4200 on board graphics) that works flawlessly. It has hdmi, digital out, firewire, e-sata and all works well in Ubuntu. ### Post by cascade9 on 2010-07-14
DONT get a WD SATAII Green 64MB (that would be a EARS). That is an 'advanced format' drive, and the setup for linux sucks. If you can find a EADS 1TB drive, they are good (BTW, I have both a EADS and EARS runnign here). 

Hmm....Id go for an nVidia card + a AMD770 chipset motherboard myself. ATI works, but for a HTPC nVidia + VDPAU is a better choice IMO. ATI XvBA is still a work in progress, VDPAU is pretty mature now. 

BTW, with VDPAU you could even use a sempron and get ful HD playback, so if money is an issue, I'd take a lower CPU + an nVida card.
