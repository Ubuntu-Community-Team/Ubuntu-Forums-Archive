### Post by tornadof3 on 2008-12-21
Hi

I bought a generic USB bar code reader from ebay for about £15 and it "just worked" with both linux and windows. When a bar code is read it just "types" the number into the current application as if you'd keyed it in with the key pad. Very useful. You can programme the bar code reader to work with different bar code languages - mine came with a code book where you scan different bar codes to change the settings on the reader.
