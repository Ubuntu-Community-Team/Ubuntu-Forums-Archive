### Post by Brian Vaughan on 2009-03-08
I'm considering whether to get a laptop, and I'm wondering whether folks here can offer some practical advice in how to choose one. In particular, I'm wondering what advice current or recent undergraduate computer science majors could offer, as I'm beginning that course of study, and that seems like a significant factor.

I've been using Ubuntu for about six months on my desktop PC at home, and am quite happy with it. I am also very strongly interested in the free software movement, so I'd want to use some flavor of GNU/Linux or other FLOSS-based operating system.

However, I'm finding that there are some downsides to a desktop -- my wife and stepson want to use it frequently; it's currently in the bedroom, which means I'll keep my wife awake if I have to work late, and if I move it to the living room, that means I'll be getting distracted frequently. Getting a laptop seems like it might be a good idea.

Laptops tend to be expensive, obviously. I looked at linux-certified.com and Dell, and thought I could just about afford the low-end laptops. But, those are netbooks, with little storage and other limitations. They're too expensive if they don't do what I need, and I'd hate to find out that I'll need something that can compile code and that they're not up to it. On the other hand, if all I need is something that can run emacs and let me connect to a university server via ssh, then they'd probably be fine.

I'm also quite happy to do a bit of tinkering to get a computer set up as I'd like, and I think I'd rather install the operating system myself and have some say about its setup from the beginning. So maybe a used laptop would be a better option. In that case, I'm not sure what to shop for.

So, in short, if I'm to get a laptop to help in my studying computer science, what would I need?

### Post by sgosnell on 2009-03-09
I think any of them would work.  I've abandoned big laptops for an Asus EEE, and while I don't do a lot of programming, it would work for that.  The storage isn't an issue for me, because I don't have tons of music, pictures, or videos, which are the biggest storage hogs.  I do have a nit.  I also have a 16GB SD card in the slot for additional storage, and for use in checking out new distros.  I can easily install an OS on the card, try it out, and if I don't like it, just remove it and install another.  A number of .iso files, though, and those are stored on a WD Passport USB HD, which was relatively cheap and is easy to carry around and attach when I need additional storage, or to transfer files to the EEE.  I also have SD cards for more storage, and for installing new distros.  I can easily install a distro to the card, boot and run it from there, and quickly erase it if I don't like it.  SD cards are cheap, and can stay in the slot full time, either for storage or booting.  Having a large HDD in a laptop is over-rated, IMO.

The only downside of a 'netbook' that I know of is the somewhat slower CPU, which might be a limitation for compiling large programs, but I don't think that would be fatal.  For the price, they're surprisingly capable.  I rarely even bother to turn my desktop on these days, because my EEE is always at hand, and I can be using it quickly wherever I am, not being tied to one place or having to lug many pounds of weight around.

You don't really need anything too powerful to write up and run a Java, C or Python algorithm (Or whatever they make you use). I've taken quite a few CS classes at my School (I'm a Computer Engineering Major)  And I see lots of people using these little net books in class. This guy I know in my Electric Circuits class uses this really small one and he gets by fine. However some of these people I talk about do have larger laptops or desktops for if they need to something their little net book can't, but you also have one too. 

I would get something simple, CS classes don't require computers with major processors speeds, large hard drives or major graphics cards. I'm in this discreet structures class right now and we're learning this logic database language called ProLog, which was created in the 70's (not very resource intensive). A 300-400 dollar net book might be good for you. I wouldn't get a large expensive desktop replacement though. It's so annoying when I'm in class and I see these guys with these huge Laptops that hang off the desk (Desks in college are small).

I Have to say though that you might not even need a laptop, if you're worried about bothering your family at night, you could probably use your CS departments computers. The one at my school has quite a few labs that already contain all the software students need and they stay open till midnight. I would look into your CS departments Lab policies to see if you could just work on there systems before you go out and buy a laptop.
