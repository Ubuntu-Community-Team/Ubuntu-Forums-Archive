### Post by kazdan on 2011-06-21
Ubuntu 10.04 on Dell Vostro 1710,  dual boot Windows XP
The Ubuntu software is up-to-date as of June18, 2011.

At the login window I uses "suspend".  When I attempted to resume all I got was a blank screen.  The hard disk light indicates that it is working.  I also heard the sound (a new notes) asking for login data.

This has worked in the past but not this time.  A cold reboot didn't help.  And a cold boot with both the Ubtntu CD and the Windows XP CD did not help.  Still a blank screen -- and no bios etc options that one usually gets on a cold boot.

In its current state, the computer is completely unusable.

Any hints?
Jerry

### Post by kazdan on 2011-06-27
Solved.  

I tested by attaching an external monitor and the computer worked, so the problem was a bad screen.  

Dell promptly replaced the screen (under warranty) and the computer now works.
