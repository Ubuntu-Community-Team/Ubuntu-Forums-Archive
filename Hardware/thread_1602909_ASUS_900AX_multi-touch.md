### Post by FlyingBuzz on 2010-10-28
multitouch features are for touchscreen, not a touchpad.

Touchpad features already work out-of-the-box: scrolling, right-click, middle-click.

But if you buy a multi-touch screen and install it properly on your  900ax, you'll get multi-touch gestures support and so on.
