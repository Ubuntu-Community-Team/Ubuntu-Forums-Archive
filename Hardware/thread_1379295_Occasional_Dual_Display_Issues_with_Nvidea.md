Hi There.

I am on 9.10 running NVIDEA proprietary driver 185. I successfully configured dual displays using Twinview.

I primary display is my laptop display, and my secondary display is a lower resolution display connected by a standard VGA cable directly plugged into the laptop.

This works great when I actually have the second display connected, but when I am traveling and I do not have the 2nd display connected, my desktop resolution remains at the "combined" resolution causing a lot of desktop stuff to display off the laptop display in the space that the 2nd display would normally see.

My workaround used to be to go into the Nvidea configuration program and setting the resolution from "Auto" to 1900 x 1200 to set the virtual desktop resolution equal to the laptop display which caused the items on the desktop to appear.  Today after installing some display enhancements, setting the resolution from Auto to anything else blacks out the screen requiring a reset of X Windows.  This is a pain.

I would actually prefer to not have to change the display configuration when the 2nd display is not connected.  Is there a way to have my display configuration detect when my 2nd display is not connected and to NOT use the combined resolution of TwinView?
