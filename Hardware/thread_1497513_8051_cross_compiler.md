### Post by gmargo on 2010-06-19
sdcc - Small Device C Compiler

[http://sdcc.sourceforge.net/](http://sdcc.sourceforge.net/)
[http://packages.ubuntu.com/lucid/sdcc](http://packages.ubuntu.com/lucid/sdcc)


Other related things in the repositories:
as31 - Intel 8031/8051 assembler
d52 - Disassembler for 8052, 8048/8041, and Z80/8080/8085 code
dis51 - Disassembler for 8051 code in Intel Hex format
emu8051 - Emulator and simulator for 8051 microcontrollers
s51dude - In-System Programmer for 8051 MCUs using usbtiny
