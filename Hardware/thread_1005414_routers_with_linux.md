### Post by AlanR8 on 2008-12-08
Never had any compatibility issues with routers under Kubuntu on Ubuntu. They are natural networking environments.

The difference between a router and a USB modem is that the router stores the username and password information, whereas with a modem the details are stored in software on the computer. 

So long as you can access the router's web address with a browser you'll be fine.
