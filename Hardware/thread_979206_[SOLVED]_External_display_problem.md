### Post by doas777 on 2008-11-16
ok, it looks like you are already using a proprietary driver, so you should be almost there. 

try running the command ```
gksu nvidia-settings &
```

if you get an error that nvidia-settings is not installed, then install it with
```
sudo apt-get install nvidia-settings
``` 

once you get into nvidia-settings, you shoudl be able to enable the secondary monitor.

good luck,
franklin
