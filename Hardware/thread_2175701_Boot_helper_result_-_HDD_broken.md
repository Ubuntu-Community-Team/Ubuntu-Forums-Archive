### Post by oldfred on 2013-09-20
Boot-Repair is not showing anything on sda.
If BIOS does not see drive then no system can as BIOS has to report drive to system to use it.
So if you can get BIOS to see it, I would copy/backup if possible, otherwise previous backups may be all you have.

You can check that connections - power & signal have not come loose or similar issues.
