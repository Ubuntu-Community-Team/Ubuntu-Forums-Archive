### Post by varunendra on 2014-04-22
Welcome to the forums Jeff !

Does this happen when trying to get into BIOS as well? In that case, this is definitely a hardware problem and you should try resetting the BIOS using jumpers provided on the motherboard for this purpose, or by taking out the battery for a few minutes (and shorting the battery contact pins to make sure all power on the board is drained).
