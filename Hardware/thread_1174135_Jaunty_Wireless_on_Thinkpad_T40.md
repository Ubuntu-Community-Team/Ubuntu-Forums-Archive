### Post by amandel on 2009-07-30
Just connected an old T40 to my new Linksys 310N wireless.  Was a brand new install of 8.10 with upgrades connected via wire.
 
One that was all done, and rebooted.  After it came back I selected wireless from the menu, disconnected the ethernet cable.
 
First panel asks some basic ssi, name encryption type.  This is similar to windows.
Entered my key for WPA2, with type AES.
 
Worked fine.  Hope this helps.
