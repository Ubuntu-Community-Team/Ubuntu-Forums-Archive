### Post by Yashiro on 2009-02-03
That only works if you're running Compiz (Desktop Effects).

If it is running, the reason it's stopped working will be because you've assigned the mouse wheel to perform another action and it's interfering with this setting.

Or you've disabled the Vieport Switcher Plugin in Compizconfig Settings Manager.

---

### Post by tuxxy on 2009-02-03
System > Prefs > Compiz > Desktop > Viewport Switcher 

Make sure that is enabled
