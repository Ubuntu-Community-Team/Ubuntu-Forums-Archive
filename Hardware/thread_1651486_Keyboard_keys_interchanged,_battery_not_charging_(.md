### Post by Zorael on 2010-12-23
No, realistically this could not be caused by Kubuntu or any software that doesn't directly modify the bios. This sounds like hardware failure. Notably the Fn key should be set in stone as Fn in the bios, making it impossible to change it via software.

Obviously bios corruption might cause this, though over several laptops I've never had that happen to me. More likely there's something (usually liquid) shorting out keyboard keys, alternatively the motherboard is kaput. The latter seems more plausible if it's no longer charging your battery.

If you still have warranty left on the machine I'd suggest you sent it in for repair.
