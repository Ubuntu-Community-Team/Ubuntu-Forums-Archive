Good evening all,

I've installed CUPS on a rasberry pi 3b, to use my Canon MG5650 as a network printer. Reason: just installed wifi6, printer doesn't like it....

From the Dutch Canon site ([https://www.canon.nl/support/consumer_products/products/fax__multifunctionals/inkjet/pixma_mg_series/pixma_mg5650.html?type=drivers&language=nl&os=linux%20(32-bit)](https://www.canon.nl/support/consumer_products/products/fax__multifunctionals/inkjet/pixma_mg_series/pixma_mg5650.html?type=drivers&language=nl&os=linux%20(32-bit)), 
i downloaded the "IJ Printer Driver Ver. 5.00 for Linux" file. After extracting the file, i imported the "canonmg5600.ppd" file into CUPS and accept the default settings.

So far, so good.

When printing a test page in CUPS (from [https://192.168.1.199:631/printers/Canon_MG5600_series_pi](https://192.168.1.199:631/printers/Canon_MG5600_series_pi)), i recieve the error message:
