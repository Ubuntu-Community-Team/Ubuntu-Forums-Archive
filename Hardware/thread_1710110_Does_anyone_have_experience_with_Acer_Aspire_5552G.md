I have an Acer Aspire 5552G with a Phenom II x4 P920(2.1GHz) and Radeon HD 6650m. This is my third time I've tried to install ubuntu(or ubuntu based Mint) on it, and sadly I'm having the same problem as always, I install the graphics driver provided from AMD exactly as instructed, it seems to work, but I have tons of tearing, and general laggy gaming in games like Nexuiz. I'm very interested in seeing other peoples experience if there are any. Personally I think it's because of the way Acer builds their laptops, you can't use the AMD windows driver either you have to get the one from Acers web page that never gets updated(Although if you find a newer model with newer graphics odds are it's newer drivers will work on this acer laptop). And Acer don't have any Linux drivers listed, my guess is you might be able to configure Xorg to work with this, but I have no idea where to start tbh.
