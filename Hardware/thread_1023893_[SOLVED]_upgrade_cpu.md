### Post by igknighted on 2008-12-28
There should be an arrow on one corner of the CPU, and a similar mark on the socket.  Make sure these line up, and it should drop in.  DO NOT PUSH IT.  If it does not go in, make sure the release (metal bar) is flipped up.  If this already was, make sure no pins are bent.

That is why you lock the CPU in the socket first, clean the top of the CPU and the bottom of the HSF with some isopropyl alcohol, then apply thermal compound to the CPU, then lock down the heatsink on top of the new thermal compound.

Trying to lockdown a CPU that is glued to a CPU cooler with thermal compound is an exercise in futility--as the OP is finding out.

CPU paste is only about $5, and can be found at any computer store.  In fact, if you asked one of the tech nicely and explained your situation he/she might give you enough for free, you only need a small pea-sized drop of it.  This stuff is so critical I really would not risk it, you can fry a CPU in seconds without it, and greatly reduce its lifespan if it isn't doing its job.  Not to mention it can break down over time, so having the CPU off is really just a great excuse to do some maintenance that should be done either way.
