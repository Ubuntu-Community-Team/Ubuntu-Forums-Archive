### Post by gaston770 on 2013-01-24
I recently purchased a Lenovo U310 and I've just installed Ubuntu 12.10 64 bit.

It works OOTB except for the trackpad, the gestures to zoom in/out and scroll up/down are not working yet, I came across your post looking for a solution.

If it comes with windows 8, chances are you'll have to disable UEFI and then go to your bios and change the device boot precedence to boot from a USB stick. You should find extensive documentation on how to do it for different windows versions.

If I find how to fix the trackpad I'll update this thread.
