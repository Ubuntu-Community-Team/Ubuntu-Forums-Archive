If "bin windows" means to wipe it, then yes, you can completely wipe windows during the install. When the partitioning options come up, say "choose something else" go into the partitioner and wipe all the Windows partitions, create new Linux partitions and have at it.
Before doing all of this, it is probably a good idea to read up on UEFI Ubuntu installs. Google will find that information easily.
