### Post by Marlonsm on 2009-03-30
It's normal. When the CPU isn't processing anything big the computer gets its clock down, so it will heat up less and use less energy.

Same happens here, My 2.13Ghz C2D gets clocked down to 1.6Ghz when idleing.
but if you really want to make sure, just add the "CPU Frequency Scaling Monitor" to any of the panels, it'll show you the options for the clock.
