### Post by Ck.asdf on 2008-08-29
Hello, I have an HP Pavilion dv2915nr laptop, with Ubuntu 8.04 installed.  The mouse touchpad has a button above it that enables/disables the touchpad, so that my thumb/edge of my hand doesn't interfere with the pad while I'm typing.  Whenever I push the button to reenable the pad, the Ubuntu Help Center opens.  Disabling the pad doesn't make it open, just enabling it.

Obviously, this is a strange action, probably caused by some misinterpretation of the button, and I would like to disable this function.  Any suggestions?

### Post by ladr0n on 2008-09-14
I'm having this same issue on an HP dv6500.

--edit--

Go to System >> Preferences >> Keyboard Shortcuts

Set the value under "Open help browser" to blank (or another key combo if you want).

### Post by serpens1 on 2008-10-02
You have to go to System->Preferences->Keyboard Shortcuts and change the key code for 'Launch Help Browser' to something like ctrl-H. The problem seems to be that in some HPs the lock keypad button is interpreted the same as pressing fn+f1.
