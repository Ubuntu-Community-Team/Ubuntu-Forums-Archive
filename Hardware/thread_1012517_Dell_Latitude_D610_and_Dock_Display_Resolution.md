### Post by DrJohn999 on 2008-12-15
Using a Dell-D610 laptop (screen resolution 1400x1050) in a Dell Latitude dock with a Samsung 1680x1050 LCD monitor. With 8.04 both displays were detected correctly, depending on whether docked or not. In default 8.10 this doesn't happen; the 1680x1050 resolution is not available when docked (or not docked).

Initially the LCD monitor's modes are not detected correctly by X, with the following defaulted /etc/X11/xorg.conf, also generated by running
```
sudo dpkg-reconfigure -phigh xserver-xorg
```

default xorg.conf:
```

Section "Device"
	Identifier	"Configured Video Device"
EndSection

Section "Monitor"
	Identifier	"Configured Monitor"
EndSection

Section "Screen"
	Identifier	"Default Screen"
	Monitor		"Configured Monitor"
	Device		"Configured Video Device"
EndSection

```

To setup this workaround, I first enabled the ATI/AMD fglrx driver from System, Administration, Hardware Drivers, which adds/replaces the following two new sections in xorg.conf:
```

Section "Module"
        Load    "glx"
EndSection

Section "Device"
        Identifier      "Configured Video Device"
        Driver  "fglrx"
EndSection

```

Then inserted a "Display" subsection into xorg.conf to give the following complete configuration:
```

Section "Monitor"
	Identifier	"Configured Monitor"
EndSection

Section "Screen"
	Identifier	"Default Screen"
	Monitor		"Configured Monitor"
	Device		"Configured Video Device"
	DefaultDepth	24
	SubSection "Display"
		Depth	24
		Modes	"1680x1050" "1400x1050" "1024x768"
	EndSubSection
EndSection

Section "Module"
	Load	"glx"
EndSection

Section "Device"
	Identifier	"Configured Video Device"
	Driver	"fglrx"
EndSection

```

Restared X with <ctrl><alt><backspace>.

Login screen and initial desktop are displayed at first as stretched 1400x1050 ==> 1650x1050. Now, however, a functional 1680x1050 choice appears in System, Preferences, Screen Resolution. Selecting and applying gives a full 1680x1050 screen with no stretch. Subsequent restarts go directly into the 1680x1050 mode.

Rebooting when undocked displays the normal 1400x1050 on the laptop screen and the 1680x1050 choice does not appear. Presumably the 1024x768 will work with a projector, but I haven't tried that yet.

Does anyone know how to get 8.10 to recognize the actual monitor type (i.e. as "Samsung...")? That would be much easier.
