### Post by Fafler on 2010-12-25
It can handle both OS's just fine. Ubuntu should be just as fast as Windows 7. The only possible issues could be the Intel graphics driver, which isn't he fastest thing around. But far from everyone experience problems with it.

The Netbook Remix is basically the same as the normal release, but uses another window manager, that makes better use of the low resolution on netbooks.  You can use the normal version if you want too. Everything else will work the same.
