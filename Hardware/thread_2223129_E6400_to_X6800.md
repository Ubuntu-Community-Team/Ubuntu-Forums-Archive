The machine I'm using is an old HP DC7700p (Vpro version) reclaimed from an office skip. I've upped the RAM to 6GB DDR2 800Mhz and added a basic graphics card to lighten the load a bit. 

The RAM upgrade was basically to reduce swaping - It prevously had 2GB and with three server guests running, even though RAM load would be around 1.4GB, swap was hitting 600-750 MB Since the update I'm able to run 4 guests each with 1GB assigned RAM and keep swapping nice and low.

This means that the maximum chip would be an X6800. benchmarks suggest a 50% single thread boost, but only a 25% multi thread boost.

it's also a tad more power hungry. Any ideas if this upgrade is worth doing on such an old system? or if there exists software to increase the multiplier like in windows?
