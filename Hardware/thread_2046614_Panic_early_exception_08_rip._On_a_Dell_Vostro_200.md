### Post by Soleft on 2012-08-23
Hi I was trying to add a GeForce 210 to my Dell Vostro 200, and I was getting this error on boot: Panic: early exception 08 Rip 202:10 error. 

I have 4 slots of memory, 2 with 1gb and 2 with 512mb memory. So I replaced all of the memory with the 1gb memory and it worked. So I believe all the memory has to be of the same brand and model. I had the same model spare at work, so I just took the 2 1gb memory from the other computer. 

Other notes:
I'm using Ubuntu 12.04

I don't know if this is protocol to post like this but I thought someone else might find it useful.
