### Post by IcarusR on 2010-12-15
I don't have this printer, so this is all untested but ought to work.

Download driver package from canon asia site

[http://support-sg.canon-asia.com/contents/SG/EN/0900772407.html]("http://support-sg.canon-asia.com/contents/SG/EN/0900772407.html")

File you need is    CAPT_Printer_Driver_for_Linux_V200_uk_EN.tar.gz

Then double click on it to extract archive.
Next navigate to > CAPT_Printer_Driver_for_Linux_V200_uk_EN/Driver/Debian/
You will find 2 .deb files, double click to launch Gdebi & install both, common first.
Now you should be able to go to System > Preference > Admin > Printing > Add with printer conected & switched on Ubuntu should see & install it.

Any probs post back & I'll try & help. But as said before I do not have this printer.
