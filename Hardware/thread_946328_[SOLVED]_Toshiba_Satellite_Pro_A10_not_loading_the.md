### Post by snowpine on 2008-10-13
Probably the laptop doesn't have enough ram (384mb minimum) to install using the Live CD. If you only have 256mb of ram, you can install using the Ubuntu Alternate CD, or try a lighter-weight distro such as Xubuntu.

Also, it never hurts to use the Search function to look up your particular laptop model on the forums. This will tell you if other people have had success with that model, or if there are problems (such as the video card, wireless, etc) you will need to work around.

Good luck!
