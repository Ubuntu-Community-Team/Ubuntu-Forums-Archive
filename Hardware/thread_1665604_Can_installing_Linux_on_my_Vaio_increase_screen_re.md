### Post by Zzl1xndd on 2011-01-12
This is (Probably) the native resolution of your screen. Installing Ubuntu (Or any OS) unfortunately wont change this. You maybe able to change some config files that will allow you to set it higher but it wont look right as LCDs have a fixed resolution.

This Wiki article might explain a little better then I can:
