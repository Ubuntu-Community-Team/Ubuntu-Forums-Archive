$ sudo vol_id /dev/sda2

and then open /etc/fstab to see if the UUID of sda2 in the fstab is coincident, if not, correct it and save, then reboot.

sudo rm /root/.nvidia-settings-rc
```

***reboot. X should now start by using the nv driver. We don't expect the 3d effects***
7. If everything ok so far go to System>Administration>Hardware Drivers and tick the Nvidia Driver to be enabled. Restart and if it doesn't work you may break it!!!!!!!!!!
