### Post by robz0rz on 2008-09-30
It sounds like a netbook may be an option for you, on of thos eultra-portable laptops. I know the eeePC can run Linux, albeit with a little tweaking (nothing too hard, and there is plenty of guides seeing the eeePC is quite popular).

I believe the MSI Wind can also run Linux. It seems very common for netbooks to be tested (or even shipped) with Linux, so you shouldn't have much trouble finding out if the netbook you'd want is able to run it.

Netbooks are generally cheap, small, have wireless and are definately able to run OOo :)
