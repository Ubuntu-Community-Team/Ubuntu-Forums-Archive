### Post by finito on 2010-11-06
If you goto System -> Administration -> Printing. 

Click ADD a new pop will show up and the default Others will be selected. 

Once there Where it says Enter Device URI type "http://" without quotes and press next.

You will see a list of all the compatible Printers there is.

Go down and select HP and scroll through the Printers You like.
