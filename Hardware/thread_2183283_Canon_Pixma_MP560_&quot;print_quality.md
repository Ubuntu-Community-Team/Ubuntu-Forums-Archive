### Post by ewm on 2013-10-24
When using this printer under WINXP you can select a "FAST" print option. I am now using Ubuntu 12.04 (up to date) and it seems as if the only print options you can select are "STANDARD" and "MANUAL" (using the Pixma MP560 driver). Does any one know if it is possible to enable this "FAST" function in Ubuntu and if so how? In WinXP this mode gives  an economical print function.

Simple solution, under printer settings set resolution to 300x300 dpi which gives the same result as "FAST" print in WinXP driver.
