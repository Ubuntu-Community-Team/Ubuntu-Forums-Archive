### Post by tgalati4 on 2014-10-18
One issue with operating a server on the floor is that you pick up a lot of dust.  So be sure to clean it out more often than you would if it was in a rack, 1 foot off of the floor.  Also, because server hardware is expensive and hard to replace, any flood would take out your hardware.  So putting it up higher is a safer bet.  Perhaps make a pedestal for it, and use it as a coffee table.
