### Post by Apollo87 on 2010-02-25
You need root permissions to access xorg.conf. The error "Unable to remove old X config backup file '/etc/X11/xorg.conf.backup" is what you get when you are running the Nvidia X Server Settings program as a regular user.

Try opening a terminal and typing:
```
sudo nvidia-settings
```

And try saving the x configuration file again.

### Post by Apollo87 on 2010-02-25
New X servers don't use the xorg.conf file as much anymore.  Its mostly used now for manufacturers whose drivers don't support the newer technologies (nvidia being one of them) or if the user wants to add something specific in.

You can check out this page if you want to learn some more about it though:
[http://www.x.org/archive/X11R6.8.0/doc/xorg.conf.5.html](http://www.x.org/archive/X11R6.8.0/doc/xorg.conf.5.html)
