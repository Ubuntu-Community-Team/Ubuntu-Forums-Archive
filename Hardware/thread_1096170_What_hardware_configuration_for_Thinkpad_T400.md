### Post by undoIT on 2009-03-14
Here's my two cents:

Go with the Intel 5300 wireless card. It will give better range / performance, is not that much more, works well in Ubuntu and will help future proof the laptop if N-wireless starts becoming more prevalent.

Doubtful you will have any problems with the media card reader.

Regarding the battery, unless she plans on spending a lot of time on the road, on airplanes, or somewhere else where an outlet isn't readily available, it is better to go with the 6 cell battery. I think most people, when working for an extended period of time, are near an outlet. So it is better to have the lighter battery that doesn't add bulk.

### Post by mperry on 2009-03-14
I've got the T500 which is a cousin I'm gathering. I've had relatively few problems with Ubuntu 8.10 32bit. I'm user the atheros drivers for wifi on this laptop. Suspend, video, sound all work across suspend events flawlessly which seems to be something people always want to know. Wifi works but the cute little light for wifi never lights up any more even though it does if I pop in the Vista business drive I have. The screen backlight is kinda wonky at times after a suspend or when the screen goes off. It will not come back at the same backlight level that it had before so I end up doing the Fn->HOME combo to get the max brightness. I run Virtualbox also to use Outlook for my company's exchange mail server and a few needs I have for Windows programs and the laptop handles Vista 32bit as a guest very well. I usually have it fired up that way most of each day.

I'm favorably impressed with about 95% of this laptop on Ubuntu. The 5% are the wifi light and the backlight issues which admittedly are not showstoppers unless you like the little green lights :).
