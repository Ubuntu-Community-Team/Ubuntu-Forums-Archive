### Post by edhgoose on 2008-10-08
I've managed to get some help form ping from the linux on wacom sourceforge project, so perhaps this will serve as help for anyone with the same troubles as me:

Basically, with my configuration the driver needs to be recompiled for this machine. To do this, download the latest linuxwacom drivers tar, and after unpacking it do these commands from the folder you extracted into:

```
cd ./configure
cd /src/
make clean
make
sudo make install

```

and then reboot.

However, I'm still stuck as now the tablet works - and the tablet registers activity - it will only jump to the top left corner and can't be used anywhere else.

If anyone has any suggestions on how to fix this, that would be much appreciated. My xorg.conf is still the same as it was in first post.
