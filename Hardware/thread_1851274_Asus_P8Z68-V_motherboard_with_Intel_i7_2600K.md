### Post by shakayoda on 2011-10-08
Hi, I've been assembling a system based on the deluxe version of this mobo yesterday. I've installed windows in a raid5 configuration with the fakeraid without any problem (I need dual boot possibilities). When turning to linux, the installer failed to install grub2 on the raid5 array. After removing the raid configuration, the installer finished its work without any hassle. So in conclusion: if you're not in need of fakeraid, I don't expect any problem.
