### Post by SteveDee on 2009-11-21
I was recently given an old Dell Inspiron 500m which was apparently way too slow to run Windows. Despite my recent problems upgrading a desktop to 9.10, I did a fresh install of Karmic on this new acquision, and its now my 'new favorite computer'. It powers up/down quickly and is very snappy loading & running Firefox, OpenOffice & so on.

One thing I don't understand is what happens when I close the lid. I can't wake it up, and the display is black apart from a white arrow which is fixed in the last position occupied by the mouse pointer. I have to power down, then up again to recover. Changing the Power Manangement Options has no affect on the outcome.

What does work is using the key combination <Fn><Suspend>

This puts the laptop to sleep, which I can wake as follows:-
- if I close the lid, simply opening the lid again wakes the computer
- if I don't close the lid, I need to momentarily press/release the main power button.

Its strange that the lid switch half-works the way you might expect. I've read many articles about this, but none provide the answer that will allow me to just shut the lid.

Not a huge problem, but has anyone with a similar Dell found a solution?
