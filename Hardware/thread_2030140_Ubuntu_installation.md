### Post by sudodus on 2012-07-21
If you have space enough on your hard drive, I suggest that you keep Windows until you are sure that [LX]ubuntu can replace all the functions, that you need. [LX]ubuntu can be installed in 8 GB, but if you have more space, it is better (for example 20 GB).

Many people (including me) start dual booting linux and Windows and then (maybe after a year) wipe Windows, or install Windows in a virtual machine (inside [LX]ubuntu).
