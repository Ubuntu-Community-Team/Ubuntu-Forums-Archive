### Post by krazyd on 2010-05-10
```
sudo apt-get install laptop-mode-tools
```

more info here: [http://samwel.tk/laptop_mode/](http://samwel.tk/laptop_mode/)

however it probably won't help if your machine is crashing because it just tweaks a few settings built into the kernel. 

If you are getting a crash, make sure you report a bug so that it can be fixed! :D
[https://help.ubuntu.com/community/ReportingBugs](https://help.ubuntu.com/community/ReportingBugs)
