Hi all,

Happy to say that I've got Ubuntu Server running flawlessly on some old HS22 blades in an IBM Bladecenter E chassis. 

The Ubuntu installation was simple. Exactly the same as an ordinary install onto PC architecture. The Bladecenter itself requires a little bit of network setting up before Ubuntu can be installed though. I didn't find much support for this online so I've written my own guide on integrating a Bladecenter into a network for anybody else who may be attempting the same.
