### Post by TBABill on 2011-12-14
As for Unity, you either like it or you don't. Tons of discussion on the forum about the pros and cons of it. However, Ubuntu offers you many choices. Particularly, you can install xubuntu-desktop, kubuntu-desktop, lubuntu-desktop, etc. What those do, instead of installing generic desktop environments of xfce, kde or lxde, is they offer you the desktop tweaked to match those releases. So you get the wallpapers, themes, configs, etc. And all you do is log out, click the little sprocket icon on the login screen, select what you want to boot into, enter password and enjoy. you can switch at will by logging out and back into another DE.
 
As for the heat, hopefully the link provided above helps with that. Which version of Ubuntu did you install? Have you run all the updates yet? I have a Dell with an i3-330m processor and using on-chip Intel graphics that flies in ubuntu so it must be a config issue somehow...or an older kernel issue...or a newer kernel trying to deal with the latest Intel chip you have.
 
Speed won't be an issue once you work through the heat issue. Whatever is causing that is likely causing the speed problem as well. 
 
If all you want is the fastest speed you can get, try Lubuntu desktop first. Look for it in Ubuntu software center. If you want more configuration with decent speed, Xubuntu desktop. If you want it most visually like Windows, Kubuntu desktop (but not in any way does it configure like it).
