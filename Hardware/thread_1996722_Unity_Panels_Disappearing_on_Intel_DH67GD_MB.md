### Post by rmcclatchey on 2012-06-04
12.04 LTS my panels would disappear randomly. I use a HDMI monitor and
thought that might be a problem, I tried a VGA monitor - no dice. I Upgraded the
BIOS and that solved the problem. Version BLH6710H.86.0155.EB.exe.
My system is dual boot so I could upgrade the bios in Windows 7. Never saw the
monitor problem in W7. The bad bios was version 7?, so there were lots of changes
to get to version 155.

Install is mostly default desktop,
