### Post by alberto ferreira on 2008-10-17
To run windows applications install wine:
[PHP]sudo apt-get install wine[/PHP]

But there are applications that don't work too 100% or simply don't work at all. For that you can "install your Windows operating system inside Linux". This is called virtualization. For that purpose you have 3 choices : Virtualbox, VMWare and Qemu. I recommend the 1st
