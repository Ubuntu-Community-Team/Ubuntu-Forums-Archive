### Post by Warthaug on 2011-07-21
The dual monitors worked perfectly using the intigrated graphics.  Hooked up 2 monitors with DVI cables (the czard has 2 DVI outputs).  After installing ubuntu the monitors were mirrored, but I simply opened the monitors dialogue box and reconfigured as an extended desktop.  Has worked without flaw for a week now.  I can even get virtualbox to run win7 in one monitor without hiccups.

Sound and network also work flawlessly.  The T1000 runs as fast as it should, unlike the last T1000 ethernet card I had.  I've had no stability issues, but I'm also using a HDD instead of a SSD.

Attachment shows my desktop, with virtualbox running win7 on the right monitor.
