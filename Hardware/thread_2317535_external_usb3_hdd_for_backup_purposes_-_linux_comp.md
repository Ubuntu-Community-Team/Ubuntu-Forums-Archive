### Post by weatherman2 on 2016-03-17
I have used many portable/external hard drives with Ubuntu.  I've also used a couple of USB docks and enclosures with many internal drives.  I have never had any incompatibility issues.  I have used Seagate and Western Digital portable/external drives.  I would avoid anything that uses software encryption - that is less likely to work with Linux.

I use ext4 for all of my Linux drives.  You can use NTFS if you really want to have Windows compability, but ext4 will have fewer issues in Linux, and the NTFS drivers for Linux require extra CPU compared to ext4.
