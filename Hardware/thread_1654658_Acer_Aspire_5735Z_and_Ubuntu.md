### Post by gordintoronto on 2010-12-28
If you download the latest version of Linux Mint, you could make a bootable flash drive and try it on the 5735Z. Mint can play videos using VLC, "out of the box." And it can play youtube videos.

The 5735Z's screen can't display 1080P, but it should be able to play 720P videos, and also stream them.
