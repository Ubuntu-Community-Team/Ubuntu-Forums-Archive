### Post by Redblade20XX on 2011-10-21
When changing distribution version, it's better to do a clean install then update. Maybe some configuration files haven't been updated correctly or it can also be many other problems.:confused: As for getting two hours of time on battery, there was a power regression bug in the 3.0 kernel for certain intel chips so if you have one it might be that.
