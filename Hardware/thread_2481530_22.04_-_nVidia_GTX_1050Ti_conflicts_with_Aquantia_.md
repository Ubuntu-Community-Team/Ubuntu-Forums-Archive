Asus Z170-AR motherboard
i5-6660k CPU
32GB RAM

Ubuntu 22.04

I booted and selected the "proprietary- tested " version 515 driver.
I lost my Aquantia NIC, unfortunately. There is no light on the NIC itself or on the switch it's connected to.
Really problematic for a NAS !

I booted to Windows 10 (system has a dual boot) and there is no issue - both the nVidia GPU and Aquantia NICs function fine simultaneously.

For now, I re-enabled the Intel 1Gbe NIC built-in to the motherboard, to gain network access back in Ubuntu. This is only a very temporary workaround, though. I really need the 10 Gig NIC to work.

Does anyone have any suggestion as to what to do to fix this ?
