### Post by snowpine on 2009-05-04
Your processor and ram aren't up to the Ubuntu challenge (I recommend Pentium 4 with 512mb of ram minimum for Ubuntu, maybe a little more for Netbook, as it is more resource-intensive than "vanilla" Ubuntu). 

I would recommend a lighter weight linux distro such as Crunchbang, Puppy, Antix, etc. There are some great "Thinkpad owners" threads on these forums you could search to see what others are using. If you have not purchased this computer yet, I recommend you shop for something with better specs. I purchased a Dell Mini 9 for $200 that would run circles around that thing (1.6ghz Atom, 1gb ram). :)
