### Post by KenUBF on 2020-03-24
Hi yofilter,

From what you listed I don't believe you should have any issues with most everything. A few years back I built my computer from scratch and I got brand new RAM that had just been released, Intel processor, etc. and it ran great.... with a few hiccups. My sound would sometimes cut out, my screen would sometimes flicker, I lost internet for a while (had to use a usb ethernet temporarily), but that was it. I did a quick search for any mention of your chosen graphics card and Ubuntu and it seems that it will probably work. To be doubly sure your system will be compatible will be to search your components with the search term Linux or Ubuntu and see what comes up. Any mentions of it not working under Ubuntu or any other issues? Or maybe it will work with certain tweaks? If you don't find any issues mentioned it would probably be safe. Good luck!

I would add a NVMe boot drive. With Ubuntu & larger drives for data, it does not have to be large.

RAID is not backup. It is used more for up time and then you have you data copied/backed up somewhere else.
