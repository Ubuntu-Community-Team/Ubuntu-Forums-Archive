### Post by electrogeist on 2009-01-12
If you have 2x 512MB modules you should have 1GB.  That's what I recommend as a minimum these days.  If your computer supports dual channel, always install memory in matched pairs (So next step up would be 2GB=1GBx2)

Watch how much swap you are using.  If you are typically waiting for things to swap to/from the hard drive, then it would be beneficial.  Either way linux loves more memory and will use extra as a cache to help speed up frequently/recently read data from the hard drive.

One downside with a laptop tho, is that twice the RAM will take twice as long to suspend!
