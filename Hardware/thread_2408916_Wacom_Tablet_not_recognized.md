### Post by betazoid26 on 2018-12-25
I also have a new Wacom tablet, it did not work properly with 18.04 but it does work well with 18.10. Making it work with 18.04 was very difficult. I had to compile the newest driver form source and additionally sign the driver. Eventually it worked but... very complicated... if you want to stick with 18.04, google for wacom linux, I think there is a web page dedicated to these issues. There you can download and compile the newest driver. It is not easy but not as difficult.
[https://linuxwacom.github.io/](https://linuxwacom.github.io/)
You can find good instructions there and there is also a mailing list where you can get help.
Alternatively, you might try 18.10.
Hope this helps
