### Post by Hedon James on 2013-04-26
**I'm your guinea pig!**  At the suggestion of the seller, further research revealed this machine is nearly identical (perhaps perfectly identical?) to an Acer Aspire AS7560.  Inasmuch as Gateway is owned by Acer, it's certainly plausible these are identical machines, albeit sold under different badges.  Further, the AS7560 is reported to be 100% linux-compatible.  Accordingly, I took the plunge and ordered the Gateway NV75S32u laptop with the understanding I had 14 days to return the machine for any reason whatsoever.  I have installed Ubuntu 12.04 on this machine with NO ISSUES whatsoever!  Using an ethernet cable to avoid potential issues with Wi-Fi, all hardware was detected and everything is working 100%, including wi-fi, webcam, and acpi controls, which are notoriously buggy on many machines.  Not this one, though; screen brightness, volume control, suspend/hibernate/resume ALL WORK with NO ADDITIONAL TWEAKS WHATSOEVER!  The were no workarounds for boot splash restarts due to proprietary driver installation.  This was the EASIEST Ubuntu installation I have ever performed, and to have everything working so well, out of the box with no additional work...this is what an Ubuntu installation SHOULD be like.  A first-timer could do this without ANY issues that would send them scurrying back to an OEM installation.  As an added bonus, this machine is FAST with Ubuntu/Unity and everything is so buttery responsive.  I am ECSTATIC with this machine and it's linux installation, as it has exceeded expectations.  **I can wholeheartedly endorse this machine as "linux compatible" to anyone else interested in this particular Gateway machine!**  Good luck!
