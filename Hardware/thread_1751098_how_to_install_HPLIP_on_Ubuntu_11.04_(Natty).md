### Post by arkonr on 2011-05-06
As a new guy, I tried to install HPLIP drivers/software for my HP PhotoSmart  C4283 printer/scanner.
After struggling for a while, here's the list of packages I had to install in order to allow the setup to run properly :
libsane-dev
python-reportlab
dbus
python-qt4-dbus
libsnmp-dev
python-qt4
pyqt4-dev-tools
xsane
pyqt4

Hope this helps anyone.
