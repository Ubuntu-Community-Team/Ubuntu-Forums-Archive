### Post by ellis rowell on 2008-06-13
I use an external USB drive at times and have no problems with it, however, it is not a propriety external hard disk. These may need special drivers. I would suggest you buy an ordinary non-propriety external hard disk caddy + a hard disk of the type and size you require. It should be recognised as a USB storage device, it should be formatted as FAT32 (VFAT). NTFS does present problems. As you use Windows as well as Ubuntu, I suggest you set it up in Windows (if you wish to partition the drive, it will be recognised as a separate drive for each partition).

Propriety devices are a problem, printers particularly so. I purchased several printers which I could not get to work with Ubuntu (Minolta Magicolor 2400w, Canon Pixma IP1600 and a Brother DSC135) I have now switched to HP Laserjet 1020 and Photosmart C4280. While the 1020 works perfectly the C4280 does not give the same extensive print range as it does under Windows (both run under HPLIP in Ubuntu). The CUPS drivers are even worse.
