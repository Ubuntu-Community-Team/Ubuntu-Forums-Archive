Hello,

I've successfully installed ubuntu on the following laptop Acer Aspire 5535. 

Everything installed during setup fine including the WiFi device and sound. I belieive its Atheros wifi device which is fully compatible with linux.

I installed the Kubuntu distribution of ubuntu.

[LIST]
[*][Kubuntu  10.04 LTS 32bit]("http://releases.ubuntu.com/kubuntu/lucid/kubuntu-10.04-desktop-i386.iso") [URL="http://releases.ubuntu.com/kubuntu/lucid/kubuntu-10.04-desktop-amd64.iso"]
[/URL]
[/LIST]
I have just one issue which is the temperature. During the installation my machine kept shutting down with no real warning or notification, I found out it was the temperature sensors hitting over, 100c, I'm not sure if its an issue to this laptop only or the whole model. However a laptop fan desktop (£14) will resolve the issue. 

I installed the lm sensors package with the graphics interface which now tells me the temp on the desktop as i'm using it.

Hope it helps!
