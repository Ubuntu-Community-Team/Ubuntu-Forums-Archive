### Post by bernie9998 on 2009-04-04
I am experiencing some odd behavior with my laptop.  When plugged into a power source, it is configured for the performance profile, which is setup to do nothing when the lid is closed (no sleep, hibernate, etc.)

This tends to work fine, with one exception: when this occurs, my network connection is lost.  The machine is in fact not going into any powersaving state, but when connected to a wifi network, the connection does not continue for some odd reason.

What is also odd is that the network will not resume again when the lid is replaced back up.  Despite the fact that the network manager is showing the laptop is connected, no network connections can be made through the network (no ping, etc.).

The network interface will not resume working again until the hardware kill switch (the wifi switch on the side of the laptop) is switched off, and then switched back on again.

Any help on troubleshooting this odd behavior would be greatly appreciated.
