### Post by IcarusR on 2010-11-05
First from reading the specs of the mobo as far as I can see it does not support IDE so the IDE - SATA will not be of any use.
It already has 5 sata ports.

Second the cpu (AMD Phenom Socket AM3)

>  AMD Phenom II X2 545 Callisto 3.0GHz 2 x 512KB L2 Cache 6MB L3 Cache Socket AM3

is not compatible with the fan

> Compatibility
    AMD Athlon 64 FX (Socket 939, 940, AM2)
    AMD Athlon 64 X2 (Socket 939, AM2)
    AMD Athlon 64 (Socket 754, 939, AM2)
    AMD Sempron (Socket 754, 939, AM2)
