### Post by oldfred on 2018-08-12
When a small SSD went below $100 back in 2011, I got my first SSD, not the latest NVMe type but still twice or more faster than my HDD.

It boots really fast, loads large programs into RAM quickly. 

But with lots of RAM once an app is used it is kept in RAM, unless you use enough other apps that space is then needed. So having a bit more RAM would be my first suggestion. Now, 4GB is considered the minimum and more is good.

I recommend SSD, but have not found I can type any faster, and Internet is not any faster. 
I believe failure mode is usually without warning, but life is now similar to HDD. Just have good backups whether you have SSD or just HDD(s).

### Post by Autodave on 2018-08-12
SSDs will dramatically shorten boot time.  Programs also load much quicker.

I use a 120G SSD ($40 US) for boot and programs.  I keep all documents, pics, music, videos, etc on a 1 terabyte spinner.

As oldfred already said, when an SSD dies, it is without warning.  I also have a backup 120G SSD that I make a copy to every couple of weeks to make sure that when thjis one goes that I will have a pretty recent backup to stick in quickly.
