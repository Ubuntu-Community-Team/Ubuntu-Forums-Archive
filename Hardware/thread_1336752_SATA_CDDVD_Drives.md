### Post by ramjet_1953 on 2009-11-24
Hi, there!

SATA drives work fine with Linux.

However, if you are using a desktop machine and your motherboard doesn't support SATA natively, you will need to buy a SATA card.

If you have a laptop, with IDE, I would almost certainly say that you will not be able to use SATA, because there will be nowhere to plug in a SATA card.
