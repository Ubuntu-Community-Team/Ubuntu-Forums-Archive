### Post by starcannon on 2008-07-08
Most games have a "preferences" menu of some sort where you can set the joypad up, occasionally they have an ini file or something like it that you may have to modify. Placement and method vary from game to game, explore the menu's and if all else fails explore the ~/.gamename folder for files that you can fiddle with.
