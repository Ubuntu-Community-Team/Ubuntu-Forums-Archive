### Post by Cope57 on 2008-11-20
Sylvania G Netbook runs gOS which is basically Ubuntu based.
The Acer Aspire is also good, but it all depends on what you are looking for in a notebook, the memory, hard drive space, video card and other hardware depending on what you are wanting to use it for.
If it is just for sending email and surfing the web, and instant message, any will do.

But if you possibly video chat, you may want one with a built-in web cam, or make sure it has enough USB ports to add one.

Does it have enough USB ports for a mouse, printer, portable flash or other device you plan on using?
Are you going to be watching DVD's? It may need a good video card.
Do you do graphic design? It may need a good CPU and RAM.
Are you planning on burning CD's / DVD's?
Does it have a multimedia card reader that you need?
Does it have CD RW / DVD RW capability to burn rewritable CD / DVD's?
If you are going to download quite a bit, does it have the hard drive spave you need?
Is it the color you want?
What company do you like the best?
Once you find the perfect laptop/netbook, does it fit in your price range?

As you can see, the questions I have of you, would outnumber those you ask here. Only **you** can determine if a certain laptop is what you need it for.
Determine what you need a laptop/notebook for, and find one that can handle the capabilities that you need.

### Post by caitlynmaire on 2008-12-25
The Sylvania g Meso is a repackaged MSI Wind with Ubuntu Netbook Remix.  It costs less with the Sylvania name :)  I've seen them for as little as $288 now with 1GB RAM and an 80GB HDD.  IMHO the Sylvanias give you the most for the money.

I actually own the original Sylvania g netbook.  As reported it comes with gOS.  However, it's beta gOS and the configuration is terrible.  An updated gOS 3.0 is available on the Sylvania website but it requires a USB CD drive for you to install it.  If you don't do the upgrade you can't update the system safely.  In addition the Realtek wireless driver they used in the beta is the pits and gives you really limited range.  Upgrading to 3.0 does fix that issue.

gOS 3.0 is basically a Hardy Heron (8.04.1) remix.  The problem with it is that the configuration hasn't been tweaked for the Sylvania machine so it basically leaves you crippled.  The g has a 1.2GHz Via C7-M ULV processor but the gOS configuration has it clocked down to 600MHz causing slow performance.  CPU frequency scaling is fully supported in kernels 2.6.25 or higher.  gOS 3.0 comes with a 2.6.24.x kernel just like Hardy did.  There is a kernel patch to enable scaling and to allow you to run at full speed.  That patch has been out longer than the machine has but it's not included!  Grrr...

In addition gOS comes with a proprietary Via graphics driver with broken 3D acceleration.  If you download the latest open source Via driver for Hardy and install it only then does it work the way it should.

gOS uses the Ubuntu repositories which is fine by me.  It hides synaptic and gnome-terminal, hacking the .desktop files in /usr/share/applications to make sure they don't show up in the menu.  You have to either edit those files or create launchers if you want a terminal or if you want to keep your system upgraded and secure.  That's sheer madness in my book.

Once you fix all the issues the Sylvania g netbook is even cheaper than the g Meso (as in well under $300) and is really a great buy.  I love mine.  I highly recommend it only to people who know Linux and aren't afraid to get under the hood to fix things.  If that isn't you it's worth avoiding.

The other differences between the g and the g Meso are 7" vs. 8.9" screen, 1.6GHz Intel Atom processor vs. 1.2GHz Via C7-M ULV, 80GB HDD vs 30GB HDD. 2.2 lbs, vs 1.9 lbs.  The original g is also smaller.  If you want inexpensive and very small and lightweight the original g makes sense.  If you're willing to go a bit bigger and a little big heavier and sacrifice on battery life then the g Meso is the better pick.
