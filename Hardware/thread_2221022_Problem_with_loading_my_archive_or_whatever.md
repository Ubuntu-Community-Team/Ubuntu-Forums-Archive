### Post by grahammechanical on 2014-04-30
That error message usually means that the archive, a repository on a server, does not exist any more. Are you using the software centre to install software? Are you using a PPA? Which version of Ubuntu are you using? It may be that the developer of that game that you wish to install has not provided a repository (archive) for your version of Ubuntu. This is likely to happen if we are now running Ubuntu 14.04. It takes some developers a while to catch up.

Please give a more fuller explanation of what you are trying to do. What file are you trying to open and what utility are you using to open the file?
