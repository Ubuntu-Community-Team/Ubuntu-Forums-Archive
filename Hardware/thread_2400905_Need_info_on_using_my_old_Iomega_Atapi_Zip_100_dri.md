Interesting question. Will they work at all? I have ZIP 100 external USB drive (originally used with Windows 98 and XP). I dug it out, plugged it into USB port, and it was recognized by Ubuntu 18.04. When I insert a disk, it mounts and I can access files and folders normally with the file manager. I had some text files from 1998 on the disk and I could open and read them in gedit, and I also could save new files to the ZIP disk. 

That said, this is not an internal model, so that, and the adaptor you have, may make some difference.

You might want to read this, and in particular, the first comment posted at the bottom which contains information that might be relevant to your model where it mentions IDE/USB adaptors:
