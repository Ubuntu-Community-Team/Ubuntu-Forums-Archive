### Post by RasterBurn on 2010-05-21
first question is whats your resolution and you video card stats??

the simplest answer i could give you to speed up your system is to disable un-needed applications from the startup, if its not required all the time, don't run it all the time, you will need to research which applications are needed and which ones aren't needed by the system all the time.

On a side note, the computer i am running which is stated in my sig was built about 2 years ago is currently running a resolution of 1280x720 @ 60Hz with full desktop effects enabled gets between 20,000 & 30,000 while the window is open but can get between 50,000 & 60,000+ with everything minimized, but on the other extreme i have seen it as low as 10...
 that is frames per 5 seconds of course... with the exact same settings with everything minimized i can get ~1,000 Frames/sec from compiz benchmark so my simplest answer is cut down on what is running, if you need to, loose the desktop effects, maybe disable automatic update checking, um.... anything that doesnt need to be running should be stopped, just make sure the app isnt needed to running all the time or you may find your system will either become unstable or just wont boot properly
