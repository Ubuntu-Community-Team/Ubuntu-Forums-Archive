### Post by dreperk on 2011-10-22
I have not seen this laptop mentioned anywhere on the entire ubuntuforums site, so I figured I would create a thread for it.

My first impression is that this is a great laptop.  I thought it was kind of odd that Samsung left out the option of replacing the battery (at least not easily), but that's my only negative (and it's barely a negative).

I started out running Linux Mint 11 and was pleased with the results.  My only problem was the use of the touchpad.  I decided to install official Ubuntu 11.10 (32-bit) to see if anything changed.  Nothing did (regarding the touchpad).  I'm finding out that Samsung has used some hardware for the touchpad that seems to emulate a Mac clickpad.  With that comes the inability to right-click, click & drag, and other things that require both clicking a mouse button and using the touchpad simultaneously.  With the recent implementation of multitouch gestures I'm pretty much using Mac-like gestures.  It's taking some getting used to, but I'm starting to like it.

Everything else works out of the box.  In fact, the one program that I almost abandoned because it would not work on any laptop I had running Ubuntu (because of Pulse) was Skype.  This time, works out of the box like a champ!  No problems, and able to video chat without any tweaking.

If you're like me and were brave (or crazy) enough to buy this laptop, please chime in with your experience!

Sorry for the late reply.  The right clicking works much like a macbook touchpad.  You don't have a very functional right "click", necessarily, but you do have the ability to tap the very right bottom corner of the pad.  That will result in the "right click" context menu.  Two-finger tap also works.

The battery life is actually considerably better with Windows 7, but it is still good with Ubuntu.  With Windows 7 you can squeeze almost 10 hours out of this puppy.  With Ubuntu, maybe 7 hours... but that's still great in my book!
