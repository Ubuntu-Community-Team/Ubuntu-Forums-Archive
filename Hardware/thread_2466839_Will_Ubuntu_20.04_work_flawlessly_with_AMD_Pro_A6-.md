### Post by mikewhatever on 2021-09-07
Asking for advice on something like that is pretty pointless. Different PCs with that same CPU have different motherboards with different components and BIOSes. Some work better, others less so.

Since you have "a device", why not test it? 
Download an ISO, make a Live USB, and boot from it, then run for as long as needed to convince you it works "flawlessly" or otherwise.
