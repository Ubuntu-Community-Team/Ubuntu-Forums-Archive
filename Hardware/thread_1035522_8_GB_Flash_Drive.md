### Post by cwbar_1 on 2009-01-09
The U3 functions of the flash drive are special programs that run from that drive without affecting the hard drive.  You can also use security features that will password protect the drive.    These functions are available in windows and Mac only!  If you wish to use the drive for Linux, it will work perfectly fine.  If you choose to use the drive for Windows only and enable the security function, Linux will not recognize the drive.  If you wish to use it in Windows, Linux, and Mac, you can.  Just don't enable the security.
