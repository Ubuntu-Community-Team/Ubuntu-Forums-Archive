### Post by annatar on 2009-01-11
Are you installing Xubuntu from the LiveCD (the CD directly boots into a desktop)? It is suggested [here]("https://help.ubuntu.com/community/Installation/SystemRequirements") that you should install using the *Alternate Install CD* if you have less than 192 MB of RAM.

Apart from Xubuntu, checkout [U-lite]("http://u-lite.org/")
Installation is a *little* troublesome, but it should even be snappier than Xubuntu :)

Also see if this page would help: [https://help.ubuntu.com/community/Installation/LowMemorySystems](https://help.ubuntu.com/community/Installation/LowMemorySystems)
