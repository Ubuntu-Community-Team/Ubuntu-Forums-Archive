Been looking around at some of the development taking place in the GPU world, especially with regards to DIY-"MainFrames" (see [NVidia Tesla]("http://en.wikipedia.org/wiki/NVIDIA_Tesla"); CUDA is another similar project)

By implementing multiple 3D GPU processors in PCI-express slots, a truck-load of RAM, and a bit of kernel-magic, it should be possible to set up a massively powerful multi-core server. 

The [OpenCL project]("http://brainstorm.ubuntu.com/idea/16362") seems to be addressing this, and I figured that this could be hugely beneficial for VM/cloud systems, where CPU's need to be assigned to clients.

Any thought on this or any idea if some solution is already in the works?
