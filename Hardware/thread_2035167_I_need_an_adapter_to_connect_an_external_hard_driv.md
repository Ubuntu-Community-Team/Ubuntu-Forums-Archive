### Post by kurt18947 on 2012-07-30
I suspect you mean an IDE connector, not PCI but yeah, that cable looks like it should work.  Things are kinda exposed so you'd need to keep away from cats & kids if those are considerations for you.  Make sure the circuity exposed on the bottom of the drive doesn't get shorted.  One thing I'd caution about re cheap USB drive accessories.  I bought a $5 2.5" drive enclosure from an Amazon vendor.  When I tried it after having it for a number of weeks,  the data transfer rate was 1.2 MB/sec. (USB 1.1 speed), not USB 2.0 speed as advertised.  And the speed problem was present on Windows XP as well so it wasn't the linux NTFS bug.  Just something be be aware of.
