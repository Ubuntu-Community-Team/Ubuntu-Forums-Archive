### Post by Sciezyna on 2013-02-13
Hi,

Toshiba Satelite 850 + Kubuntu 12.04

I bought Toshiba Satelite 850 pro. with the intention to install Linux on it. It had windows 7 installed and I put Kubuntu 12.04 next to it with no problems. There are few issues worth mentioning though:

1. I did not check it yet but there is problem with sleep mode while closing the screen - it works intermittently and at times (say once a week) it freezes the laptop with black screen. Need to be _unplugged_ and restarted. Possibly it has to do with the BIOS - BIOS update is not helping.

2. More annoying is that Kubuntu 12.04 does not have support for the wireless Realtec NIC. Little search solved the problem, however, the driver (rtl_92ce_92se_92de_8723ae_linux_mac80211_0006.0514.2012) needs to be compiled locally. This created another problem. Every Kernel upgrade the driver must be reinstalled ("sudo make install") freshly.

3. As just a hardware it heats up markedly while charging. It is good to have the machine sitting lifted a little for better air flow.

Regards
