### Post by ATXChris on 2013-02-15
Well, I was thinking if you want cheap to run Linux, a used Fujitsu or ThinkPad would be very affordable. Even though the processors are "old" they run Xubuntu just fine. My specs are:

Core 2 Duo @ 1.5GHz
2GB RAM
256GB Toshiba SSD
12.1" SXGA+ Screen
Xubuntu 12.10

And I am running Xubuntu really fast. Mine was only $200 shipped and only that much because of the SXGA+ screen. You could probably find a Fujitsu off ebay for ~$150. But these are SUPER portable, like "normal" tablets. It's still a laptop.
