### Post by xofer on 2011-02-15
Hello

i am trying to use ViewSonic VX2258WM with Ubuntu 10.10

This display includes a touch device by Quantas, which according to my searches should be basically the same as used in HP L2105tm and Acer T230H (among others).

The touch interface seems to be recognized as such and works as a mouse - dragging a finger along the screen moves the cursor and clicks are (reluctantly) recognized. But the problem i am having is this - 
clicks on a certain location on screen (button, menu etc) are not registered unless you drag the finger along the screen a bit first. This makes use of some interfaces (e.g. onscreen keyboard) almost impossible and extremely unintuitive. 

This does not seem to be a hardware limitation as the clicks are registered just fine when the same display is connected to win7. So it must be either linux (kernel or X?) driver or configuration issue. 

Can anybody share some experiences with the aforementioned monitors?
