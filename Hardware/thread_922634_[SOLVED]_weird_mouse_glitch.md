### Post by punkbohemian on 2008-09-17
My computer is an HP dv6700tse and I recently installed Ubuntu. Everything is working fine, but there's this one weird thing that happens with one of the mouse buttons. The button that turns off the touch pad works properly when I turn it off, but when I turn it back on, the Ubuntu help window pops up as well. I'm guessing this some kind of weird mapping error. Does anyone know how to fix it? Thanks.

---

### Post by punkbohemian on 2008-09-17
I figured it out. I was eventually able to find a thread that covered this. In case anyone else has this problem and comes across this thread, what worked for me was going to System>Preferences>Keyboard Shortcuts and then changing the shortcut for the shortcut for "Launch Help Browser" (or clear it out completely).
