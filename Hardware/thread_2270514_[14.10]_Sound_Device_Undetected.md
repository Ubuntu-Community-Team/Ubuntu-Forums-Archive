### Post by Yellow Pasque on 2015-03-23
Maybe you should use -R (recursive) flag with chmod and chown. Either that, or simply move/delete the directory and pulseaudio will generate a new config directory when restarted:
```
sudo mv /home/sgashepp/.config/pulse /home/sgashepp/
pulseaudio -k
```
