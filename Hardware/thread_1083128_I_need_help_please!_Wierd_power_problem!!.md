### Post by kestrel1 on 2009-02-28
There could be a number of reasons, but I would check that everything is connected correctly, reseat the memory modules. If you have more than one memory module remove all but one & boot up. If it continues OK, then put another module back in & try again. Do this until it doesn't work, then you will know that you have a faulty memory module.
There is always a slim possibility that the CMOS Battery is getting low on power & needs replacing. If this is a normal desktop PC, they mostly take CR2032 button type batteries. These cost next to nothing to buy. About £2.00 in UK.
