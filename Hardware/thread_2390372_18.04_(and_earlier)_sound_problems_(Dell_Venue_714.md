### Post by Yellow Pasque on 2018-04-28
Start here: [https://wiki.ubuntu.com/Audio/AlsaInfo](https://wiki.ubuntu.com/Audio/AlsaInfo)

> I tried to follow some troubleshooting guides, but they generally suggest editing /etc/modprobe.d/alsa_base.conf

You can put directives in any .conf file in /etc/modprobe.d (i.e. you can make your own file named whatever.conf)
However, a lot of the tips telling you to do this are either very old (especially model= tips) or only apply to specific laptops/mobos, so be careful. If you try something and it doesn't help, remember to delete the line/file you added so it does not cause problems down the road.

### Post by Yellow Pasque on 2018-04-29
Have you tried toggling/unmuting these settings in alsamixer?

```
Simple mixer control 'HPO L',0
  Capabilities: pswitch pswitch-joined
  Playback channels: Mono
  Mono: Playback [off]

Simple mixer control 'HPO R',0
  Capabilities: pswitch pswitch-joined
  Playback channels: Mono
  Mono: Playback [off]
```
