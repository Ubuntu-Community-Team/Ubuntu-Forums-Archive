### Post by ahallubuntu on 2012-04-05
How did you install Ubuntu?  Dual boot? Wubi?  If you installed as a dual boot, Windows 7 being installed would be irrelevant.

Did you do all of the updates after installation?  Updated kernels can fix hardware issues.

Did you install any Additional Drivers?  (Proprietary Drivers?) 

Not sure what model Samsung netbook you have, but you might try googling for that and "Ubuntu" to see what potential issues people may have had with it. ("N450" is the model number of the Intel Atom CPU used inside the netbook.)

FYI, Ubuntu 12.04 LTS (Long Term Support) will be officially released in a few weeks - that's the version you are going to want fairly soon anyway since 11.10 has a limited period of support.  12.04 may fix some of the problems you've experienced if you find no other way to fix them.

### Post by kaldor on 2012-04-05
Note: This isn't a solution, but just my $0.02 :)

WUBI is known to be a bit flaky and often has weird issues (I've never had good luck with it).

The best decision is to keep using WUBI to become familiar with Ubuntu. Once you're actually comfortable with using it, I'd recommend you use a "real" Ubuntu install. This involves booting from the USB/CD and selecting "Install alongside Windows" during the install process. It should split your HD in half and do all the partitioning for you.

The good news is that the new release of Ubuntu is coming out in under a month (April 26ish). It has a load of improvements, new hardware support, and refinement of the UI. I'd recommend you keep using WUBI until then and install Ubuntu fresh when 12.04 is released. 

I'd also welcome you to try out the Ubuntu Beta release and take a peek at the Ubuntu+1 forum. Lots of activity and testing going on :)
