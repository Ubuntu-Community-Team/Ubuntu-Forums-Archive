hace como 3 meses me compre el modem de banda ancha de movistar... y hasta hace poco lo podia usar solamente desde windows... googlee, busque en foros, intente usando el Gnome ppp... de mil maneras y no podia...hace unos dias encontre la solucion de una manera tan simple que me daban ganas de golpearme la cabeza contra la pared por no haberlo intentado antes...

aqui dejo una guia para todos los que tengan el problema que tenia yo...

Aclaro que la guia la hago para Ubuntu 8.10... no se si sera igual en otras versiones...

El modem es un Huawei E226 pero supongo que sera igual para otros diferentes...

primero se va a "Editar las conexiones" haciendo click secundario en el icono de red (normalmente en la esquina superior derecha)

despues aparece otro cartel que dice "Service Provider"

[[IMG]http://img155.imageshack.us/img155/3663/captura02.th.jpg[/IMG]](http://img155.imageshack.us/my.php?image=captura02.jpg)

yo pongo movistar porque mi modem es de movistar. Click en adelante

luego aparece otro cartel que dice "Summary"

[[IMG]http://img237.imageshack.us/img237/4046/captura03.th.jpg[/IMG]](http://img237.imageshack.us/my.php?image=captura03.jpg)

Click en adelante... YA ESTA!!! SE PUEDE CONECTAR!!!!

[[IMG]http://img237.imageshack.us/img237/8451/captura04.th.jpg[/IMG]](http://img237.imageshack.us/my.php?image=captura04.jpg)

pero ojo... no dije que anda...

los DNS que te da movistar estan mal...

[[IMG]http://img16.imageshack.us/img16/4931/captura05.th.jpg[/IMG]](http://img16.imageshack.us/my.php?image=captura05.jpg)

para solucionarlo cada vez que uno se conecta hay que modificar el archivo /etc/resolv.conf
para eso se va al terminal
[B]Comando:
        sudo gedit /etc/resolv.conf[/B]

se abre el archivo en un editor de texto... el archivo debe quedar asi

# Generated by NetworkManager
nameserver 194.179.1.100
nameserver 194.179.1.101

guardar y listo... ya podemos navegar por internet...

algo que yo hice para no tener que hacer esto cada vez que me conecto es crear un archivo con el contenido de como deberia quedar /etc/resolv.conf
[B]Comando:
        sudo gedit /etc/resolv.conf2[/B]

se abre otro editor de texto... el archivo debe quedar con el contenido:

# Generated by NetworkManager
nameserver 194.179.1.100
nameserver 194.179.1.101

despues se crea un archivo en el escritorio que sera un programa de comandos:
[B]
comando:
        sudo gedit /Escritorio/movistar[/B]

se abre un editor de textos y el archivo debe quedar con el contenido:

sudo cp /etc/resolv.conf2 /etc/resolv.conf

luego de eso se va a un terminal
[B]comando:
        sudo chmod a+x /Escritorio/movistar[/B]

listo... cada vez que se conecte hay que abrir el archivo "movistar" desde el escritorio (cuando ya esta conectado). En el cartel que aparece poner "Ejecutar en un terminal", escribir la contraseña del usuario y listo!!!
INTERNET INALAMBRICO DE FORMA FACIL Y RAPIDA

PD:algunas veces no reconoce la conexion y pide una contraseña... escribir wap... con eso la deberia detectar bien...

### Post by c4d0rn4 on 2009-03-04
para lo de los dns por ahí te sirva esto:

sudo gedit /etc/dhcp3/dhclient.conf

y agregás esta línea:
```
prepend domain-name-servers 208.67.222.222,208.67.220.220;
```
Separado por una coma están los ip de los dns de opendns cambialos por los que quieras.

Con ese cambio los dns serán permanentes.
