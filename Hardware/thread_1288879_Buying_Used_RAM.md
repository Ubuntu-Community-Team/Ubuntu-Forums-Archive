Sorry, I should have been a little clearer there. Having a bad stick of ram typically won't kill a board unless it has pretty big physical problems. So if you can easily see the stick is damaged or has been physically modified (ie: some idiot tried to solder something) then don't put it in the dimm. ### Post by viper250 on 2009-10-12
used ram is usually no problem unless it has been through a power surge and then it is usually seen by eye as burned circuitry or error message from bios.
memory sticks sold on ebay are usually working pulls.
 but in some cases you can find the ram cheaper from a reputable dealer because some people have been known to shill bid in order to drive the price up.
check out places like pricewatch.com or memory4less.
