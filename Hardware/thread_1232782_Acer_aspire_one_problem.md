### Post by starcannon on 2009-08-05
[LIST=1]
[*]Unplug it
[*]Pull the battery out
[*]Wait a while, say 15 minutes
[*]Turn the machine on.
[/LIST]
If you get to the Acer splash screen, then go ahead and open Bios Setup and reset Bios to default values (option is usually on last tab of bios menus), then go back through, set it to boot from usb/external device first, save and exit. 

Reboot CTRL+ALT+DELETE with the bootable media in place. If all went as it should, you should be confronted with an Ubuntu Menu Screen.
