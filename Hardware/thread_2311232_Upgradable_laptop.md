### Post by weatherman2 on 2016-01-25
If you can find them, there are a slew of old Dell Inspiron E1505 laptops out there (also 6400 - same thing basically).  I've picked up a few of them for only about $20 USD at thrift stores.  I have friends/relatives still using a couple of them - not very fast but very adequate for basic stuff.  They work great even with "regular" Ubuntu (but with Flashback to get the Gnome interface + Metacity).  You can easily upgrade the RAM, hard drive, and wireless card.  The CPU is upgradeable too, but it requires pulling the whole laptop apart to get to.  On the other hand, if you find an old one of these, you may want to pull it apart to clean out the CPU fan anyway, so just upgrade the CPU while you are at it.

Biggest downside: the RAM on this model can be upgraded only to 2GB maximum.

Also, some of these E1505s came with the 32 bit-only Core Duo series.  Upgrade the CPU to a Core 2 Duo to get 64 bit capability.  (I have found recently that Google Chrome has decided to end support for 32 bit Linux in a few months.  Otherwise, you don't need the 64 bit version to address 2GB of RAM.)

I also have somewhere a Dell Inspiron 1545 which can be upgraded to 4GB of RAM.  The CPU on this model is very easy to access: from the bottom panel, so you don't even need to need to pull the keyboard off.  I did upgrade the CPU to the fastest Core 2 Duo CPU I could find at the time on eBay.   Some models of the 1545 come with a webcam, others don't.  You can add a webcam yourself if you buy a bezel that has the hole for it.
