### Post by sgosnell on 2010-05-30
Try installing hplip, and make sure the proper drivers are being used.  I like to start from scratch, System/Administration/Printing, Add printer, then have it find the printer.  HPLIP isn't technically required, but it makes things easier.  Running it will also install the printer, and be more likely to insure that the proper driver is used.
