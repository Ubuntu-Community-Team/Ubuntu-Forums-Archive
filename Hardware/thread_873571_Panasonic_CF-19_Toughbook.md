### Post by acidd_uk on 2008-07-29
Hi,

I've got one of these (the touchscreen version). First off, props that most things work straight away (though the touchscreen calibration is off, but that's easily remedied).

Now, normally I use RedHat (at work don't boo me), which works ok with the CF-18 toughbook but doesn't work too well with the CF-19.

So I'm trying Ubuntu (yay). So far so good, but there are 2 major (to me) issues.

Firstly, the graphics support. None of the screensavers are working (just get a black screen), and I suspect it's due to poor/no OpenGL support in the driver. However, the CF-19 has the Intel 945GM chipset, which I think should support some (crappy) graphics acceleation.

Secondly, the brightness control. The tablet has 'blue'/'Fn' keys as well as a special keypad at the front edge, both of which include screen brightness control buttons. Pressing these causes a brightness dialog to appear in Ubuntu/X with a nice progress type bar showing the position of the brightness control. Unfortunately, no matter where the brightness control thinks it is, the brightness doesn't actually change!

This is all from a Live CD, I'll install it properly and do some more investigation. If anyone has any experience of the CF-19 or indeed any of the toughbooks then help would be appreciated!
