### Post by ajgreeny on 2013-07-08
You can set the machine to use hybrid-suspend by following the info at [http://www.webupd8.org/2012/11/how-to-use-hybrid-suspend-in-ubuntu.html](http://www.webupd8.org/2012/11/how-to-use-hybrid-suspend-in-ubuntu.html) and [http://daniel.hahler.de/use-hybrid-suspend-method-by-default](http://daniel.hahler.de/use-hybrid-suspend-method-by-default)  at a quick look, they seem to be exactly the same info.

I used it for a short time on my desktop machine with no problem, but decided it was much more appropriate for laptops so removed it.

Give it a try yourself; if there are any problems you just need to remove the file that is written in order to enable it and it will stop hapening.
