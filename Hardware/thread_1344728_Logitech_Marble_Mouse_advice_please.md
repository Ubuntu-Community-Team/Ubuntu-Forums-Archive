### Post by El_marsupial on 2009-12-11
Hi I have an acer aspireone netbook and I recently bought a Logitech Marble Mouse Trackball, I would like to tell you I find it very useful, but it is also huge, it has four buttons and the ball, I have not made any changes to my system configuration and I can tell you I can use the four buttons, the big button on the left allows you to left click and the smaller button on the left side allows you to go back while you are browsing on firefox or even on your personal files, the same happens for the right side, the ball is really really helpul and it allows you to move the cursor from one side to the other side of the page in less than a second, the only thing I did not like about this Marble mouse is that I have not been able to scroll through the pages with it, I was checking the information about how to make that work, but it seems to be complicated because you have to edit xorg.conf which is a file I have not been able to find at all, I am using 9.10 ubuntu, anyway, I hope this information is useful for you and if you have any questions, please let me know, if you get the marble mouse, please let me know so that you can give me some advices.
