### Post by CMOS4081 on 2011-07-06
So strange,
Try logging into Ubuntu Classic first and see what happens. 
Does the same problem occur? 


Another tip, make sure any reference to the nouveau driver is gone.
Then go to [www.nvidia.com](www.nvidia.com) and download the current driver.

Make sure you follow the prompts from the nvidia driver so it removes the old driver and updates xorg.conf.
