### Post by weatherman2 on 2014-08-26
Does the new router support WPS?  If so, the HP printer probably does too so you might use WPS to connect them.  You can usually find a WPS code from the printer using the menu buttons on the printer itself, then login to your router and tell the router to allow that WPS code to connect.
