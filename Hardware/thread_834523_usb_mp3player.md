a while ago, my redfish R10 usb player decided to stop getting mounted. Not only on ubuntu, but on all computers, and Operating Systems, it simply doesn't get detected as new hardware. The players batteries are charged through usb, and before the problem occured for the first time, I plugged it into an usb charger which works with 230V (that's how much the plugs have in germany). the reset button doesn't help either. The player still plays musik, though, so I figure the memory chip is undamaged.

EDIT: somehow the usb modus was set to MTP. I set it back to MSC, and now it's working :)
