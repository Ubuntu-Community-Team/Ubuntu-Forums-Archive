### Post by snowpine on 2011-08-25
Open the System Monitor and study your RAM usage. If you are using less than 1GB under full load, then you do not need 2GB. If you are approaching 1GB used, then increasing to 2GB is a good idea. :)

### Post by foresthill on 2011-08-25
FWIW, none of my machines I have run Ubuntu on have ever, at any time, used more than 1 gb of RAM. Though if you run the 64 bit version of Ubuntu, you'll see more RAM used, but I doubt it will ever exceed 1 gb.

Ubuntu just does not use that much RAM compared to newer versions of Windows. 

I have an Acer netbook with 2 gb of RAM, and it's so slow I refuse to use it. Pretty sure it's the weak 1.3 Ghz Atom processor. I have not tried an SSD hard drive, so I can't say whether that would help, but a $200 laptop does not really justify that kind of expense, IMO. Lack of RAM is certainly not the problem, I know that much.
