### Post by gdonwallace on 2010-05-29
Are you using wireless or standard wired connection?

If wired, or wireless for that matter, there is an icon on the upper panel for networks.  If wired click on it and choose open and it will open a window for you to set everything up.  If wireless, click on the same icon on the upper panel and it will give you a list of all networks that your card is able to find.  Click on your network, and it should connect automatically.  If it does not, you will need to edit the settings and set either the IPv4 connection type with the passphrase or the IPv6 settings.  Either way there is a tab for both, and You only setup one or the other.

After that you should connect fine.
