### Post by IntuitiveNipple on 2008-09-07
Try it out with a [Live Desktop CD](http://www.ubuntu.com/getubuntu) - this boots from the CD and runs without using the hard-disk so the existing Windows installation isn't affected.

It allows you to check support for the PC devices before making a decision to install permanently.

In general most Vaios are well-supported, but there might be an issue with older wireless LAN cards based on Orinoco Hermes chip-sets since the driver is no longer supported and can be a pain to build manually.
