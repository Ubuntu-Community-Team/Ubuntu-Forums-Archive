### Post by gabbman on 2016-10-01
My HP Pavilion is running AMD A10-8700P Radeon R6, Compute Cores 4C+6G 1.80GHz. with Installed RAM 12.0 GB.  Not really a big deal, but anyone who has attempted to install any Linux OS on this microsoft only laptop has soon learned it is not that easy.  There are a ton of posts on various sites on how to and each distro has it's own unique issue.  The culprit of all the frustration is the A10-8700P Radeon R6.

I don't know all the technical stuff but I discovered the only way to get passed a kernel panic screen was to insert "nomodeset" in grub.  I have run this procedure with a lot of different Linux OS, mostly with not so positive results (unless you happen to be a well educated terminal Geek....which I am not).

Any way, the reason for my post is to give hope to those who are unfortunate enough to own one of these computers, and to sincerely praise all of the Ubuntu team who helped get this hiccup fixed so we can enjoy this beautiful operating system.

Thank you, thank you, for putting secure, functional, computing back on this very fussy box.
