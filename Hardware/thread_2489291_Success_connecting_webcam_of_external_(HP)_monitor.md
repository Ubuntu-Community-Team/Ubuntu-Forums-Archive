### Post by cliverlong2 on 2023-07-25
Posting this in case useful to others. Running Linux on Mac Pro 5,1 (cheesegrater). Reading the HP manual (!) I found connecting the USB B end (square plug) of the cable to the HP monitor and the USB A end to the mac, Linux/Ubuntu automatically detected the monitor webcam. I can use the 720p webcam in OBS Studio and MS_Teams through a web browser.
