### Post by lightbricko on 2009-12-13
I will answer my own question.

The best test suite I could find is phoronix test suite at [http://www.phoronix-test-suite.com](http://www.phoronix-test-suite.com)

Although I didn't manage to get the graphical user interface to work, it was easy to work with in the console. It contains a lot of tests and the results can be compared with other systems.
