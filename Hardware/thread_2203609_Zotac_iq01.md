### Post by blackgoose on 2014-06-15
I just got one of these boxes. I'm running 14.04 64-bit and it works just fine. This is my 4th Zotac and 3rd Zbox running Ubuntu and it's all been plain sailing.

I'm using the Wifi, bluetooth, etc. and it all works fine. I'm not using it for gaming or HTPC, but I'm sure it would do that as well.

Runs nice and quiet - can barely hear the fan. Some Zboxes require retuning of the fan defaults in the BIOS to get the fan running at the appropriate speed and temp for quietness' sake, but once you realize that it's no problem.
