### Post by bluexrider on 2011-11-16
Sorry for your loss (sniff and a tear) but if no bios no boot then it is probably a goner.

Recommend reinserting all the PCI cards and checking connections. See if when you pull any cards out that they don't have burn marks or smell bad. If you suspect one, replace it before moving on. 
IE: if the video card smelled like it was burnt go get another one.

If you are really into this MB then take it out and check the bottom of the board to make sure that there are no hot spots. Double check the processor pins in the holder and the cooling fan.

The OS is probably still intact so that isn't a problem. 

If everything else checks out then move the hard drive to another box and boot from it.

You have your work cut out for you now. I hate doing the diag. Would rather build another system then messing around with old parts.


Good luck
