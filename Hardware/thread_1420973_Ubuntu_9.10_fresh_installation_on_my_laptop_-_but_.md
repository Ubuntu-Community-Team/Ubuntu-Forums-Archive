### Post by Endomancer on 2010-03-04
For DVD playback in ubuntu you need to set up Medibuntu,
Medibuntu set up is explained on 

[http://www.youtube.com/watch?v=NT7urt5UcQg](http://www.youtube.com/watch?v=NT7urt5UcQg)

and

[https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)

Also Ubuntu Restricted Extras is handy to have to and can be obtained through

```
sudo apt-get install ubuntu-restricted-extras
```
