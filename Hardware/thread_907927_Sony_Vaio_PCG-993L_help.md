### Post by IntuitiveNipple on 2008-09-02
First, the best way to test it is to [download and burn to a CD](http://www.ubuntu.com/getubuntu/download) one of the LiveCD images or to [request one be sent to you](http://www.ubuntu.com/getubuntu). That way you can run Ubuntu from the CD and test the hardware, check the results fo various system tools that report the hardware, and so on, without affecting the currently installed operating system or documents.

It seems the model number for the PCG-993L is PCG-FX340. It is hard to find specifications on it now since it is old in laptop terms, but I did find [this half-broken Sony page](http://www2.sony.com/SEL/consumer/ss5/portable/vaio/notebookmodels/pcg-fx340_specs.shtml). Is that the same model?
