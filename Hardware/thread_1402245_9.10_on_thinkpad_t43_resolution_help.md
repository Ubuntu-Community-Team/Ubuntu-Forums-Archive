### Post by pi/roman on 2010-02-09
The "displaysize 311 255" was a shot in the dark and it doesn't appear to work so you can remove that.

Go to system / administration / update manager and check for updates first then install them, then check for updates again. If you install updates then reboot and make an xorg.conf.new and see if there are any more options available in "device" section.

If no other options are available try adding this line to the "Device" section
> Option "monitor-LVDS" "Monitor0"After rebooting check for errors and warnings.
grep "^(EE)" /var/log/Xorg.*
grep "^(WW)" /var/log/Xorg.*

Also look on your laptop or in  documents if you have any and see if you can find what version of t43 you have.

[[COLOR=Blue]t43/p[/COLOR]]("http://www-307.ibm.com/pc/support/site.wss/document.do?lndocid=MIGR-58317#vid")
> Display                                                                                                           14.1inch, 16M colors, XGA (1024 × 768 resolution) TFT display
                             14.1inch, 16M colors, SXGA+ (1400 × 1050 resolution) TFT display
                             15inch, 16M colors, XGA (1024 × 768 resolution) TFT display
                             15inch, 16M colors, SXGA+ (1400 × 1050) TFT display
                             15inch, 16M colors, UXGA (1600 × 1200 ) TFT display [B]
                             Notes:[/B]                               
[LIST]
[*]Support for a DVI monitor is through the ThinkPad Dock II option and is limited to a resolution of 1280 x 1024 at 60 Hz. Machine types 1871, 1872, 1873, 1874, 1875, and 1876 do not support DVI monitors.
[*]                                 All displays are measured diagonally.
[/LIST]
I believe your maximum resolution is 1024x768 but if you want to try 1400x1050 you can generate a modeline by:
```
cvt 1400 1050
```then copy the modeline and paste it in your monitor section and change the identifier to "1400x1050" so it matches the mode in your screen / display section.

Also remove the "virtual 1152 864" line so it does not conflict with the 1400x1050 mode.
