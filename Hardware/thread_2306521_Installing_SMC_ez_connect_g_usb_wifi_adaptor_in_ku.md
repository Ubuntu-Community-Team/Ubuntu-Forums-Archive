Hi, i am going to explain how to setup this externel usb on a laptop already having a builtin adaptor.

You connect the usb to kubuntu, no notification 'll appear but it is recognized.

Right click on network manager icon in kubuntu, hover mouse over your existing network, click settings, change "_**restrict to device**_", you should see wlan0 and wlan1, done
no restarrt is needed

If you want to use the 2 adaptors, then create a new network .but apply restricting  to the other network device.

you can use the externel adaptor to extend wireless range for example.
