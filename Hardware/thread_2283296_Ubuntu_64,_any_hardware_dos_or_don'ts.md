### Post by efflandt on 2015-06-21
The only thing I can think of offhand is that if you are going to do any Steam gaming, nvidia graphics seems to be most compatible at this time. I am running a 5 year old PC that has 2 core with hyperthreading (looks like 4 cores to any OS) and the low end of high end graphics which suits me fine for now. Although, I wish I had known that the GTX 960 was going to be released shortly have I got a GTX 750 Ti (the bright side is that my whole PC is only using 150 watts max input). Note that Google Chrome browser can run Netflix now in Linux (maybe you need to set your profile on Netflix website to prefer HTML5), so there is no need for netflix-desktop with wine any more.
