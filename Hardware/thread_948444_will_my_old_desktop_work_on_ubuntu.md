### Post by nixscripter on 2008-10-15
The specifications look okay; Ubuntu will run on a lot. But the simplest way to find out: try it!

Burn a LiveCD, and see what you can do with it. It may run a little slower than usual, because memory will be tight (it caches the CD into memory), but if all the hardware seems to work, it should be good to install.

If you repartition your hard drive (instead of erasing the whole thing), make sure to back up all your data before you install.

Good luck.
