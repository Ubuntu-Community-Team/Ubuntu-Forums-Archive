### Post by Renée Jade on 2010-11-03
Hi. I'm not sure if you mean that this laptop is running Windows XP. But if that is the case then I think you might be in trouble. I once tried everything to get wifi working on my friend's laptop before I realized that someone had turned it off using the key. That is to say, you can enable and disable via hardware manager or the like until the cows come home, but it won't help. If you are on Linux, and possibly if you are on Windows, I would say that your best bet is to try and remap some other key to act as an Fn key and use that. There might be some other way to enable wifi via software, but I don't know it.
