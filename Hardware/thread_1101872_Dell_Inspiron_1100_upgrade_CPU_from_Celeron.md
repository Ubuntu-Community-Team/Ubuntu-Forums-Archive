### Post by dajasc on 2009-03-20
You can look here:  [http://www.bay-wolf.com/parts_insp1100.htm](http://www.bay-wolf.com/parts_insp1100.htm)

There are a number of supported processors listed.  

That laptop uses DDR  and both the processors you tried use DDR2.  I know the memory controllers are separate but it still isn't going to work.  This is the problem with them running the same pin out on that socket for a decade.  You need an older P-4 the newer ones are higher bus speeds and not compatible.
