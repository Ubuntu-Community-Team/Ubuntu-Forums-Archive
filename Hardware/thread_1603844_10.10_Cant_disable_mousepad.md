### Post by welshmike on 2010-11-18
> **Urichhai said:**
> I use a logitech laser mouse and really wish I could disable the pad. I set disable pad on typing but really dosnt work. Using a Dell Studio 1725. Anyone have a fix on this because it is really getting on my nerves. Dont want to but may have to reinstall M$ again.:(
I've struggled with this for a long time.
I read something like the following somewhere, probably on the Ubuntu forum: 

"you can disable the mousepad with the following terminal command:
sudo modprobe -r psmouse"

The above does not last over re-boots or possibly re-logins.

Today I was able to make a permanent disabling of my mousepad having seen today on this Ubuntu forum that commands in /etc/rc.local are run at Ubuntu startup or possibly logins.
So before the exit 0 command in /etc/rc.local I added the following command:
modprobe -r psmouse

You can enable it again with the terminal command:
sudo modprobe psmouse
