### Post by pirattrev on 2008-10-23
well, assuming that your graphics aren't working at all, ie xorg could be totally fried like when i installed feisty and my nvidia wasn't working, i'd boot into one of your tty lines and run:
```
sudo dpkg-reconfigure xserver-xorg
```
then just follow through all the steps and check the appropriate things.  If that fails, just wget the newest nvidia driver or
```
sudo apt-get install nvidia-glx-new
```
and then it should compile itself as the new driver.  I'm not entirely sure of the accuracy of these statements but they are definitely a direction you should start in. Hope this helps.
