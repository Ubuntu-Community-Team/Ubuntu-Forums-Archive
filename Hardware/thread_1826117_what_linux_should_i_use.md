### Post by snowpine on 2011-08-16
Older releases (pre-10.04) are not supported any more and probably aren't a good idea. 10.04 is worth a shot, it is the current LTS (Long Term Support) release and will be supported through April 2013.

If 11.04's "Unity" interface is too resource-hungry for your hardware, you might try Xubuntu (Xfce) or Lubuntu (LXDE). Either of these desktops is a good choice for older hardware. 

If you want to go outside the Ubuntu family, there are ultra-lightweight distros like Puppy that fly on old computers... :)
