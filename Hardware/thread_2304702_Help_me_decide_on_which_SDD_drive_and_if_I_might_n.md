If you're in the U.S. keep an eye on newegg.com, particularly now (black friday). A couple weeks ago I installed an SSD in a Thinkpad X201 (core i5 4 GB RAM. It certainly sped booting up; Windows 7 boots to usable in 30 seconds, Ubuntu-Mate in < 10 sec.The most obvious improvement was booting and using Windows - less flick - flick - pause - flick flick flick of the hard drive light.  *buntus don't exhibit that behavior so the improvement was less noticeable. 

A second advantage in portables would seem to be resistance to shock while running; no moving parts and no heads to crash. A 240-250 GB. SSD will likely run $60-$70 in the U.S. 120 GB. are somewhat less $$ but if you want multiple O.S.s or virtual machines I'm not certain 120 GB. would be adequate. I bought an OCZ Vector 180 240 GB with an AMD sticker on it from Newegg. One thing to be aware of - most SSDs are 7 mm, most older laptop HDs are 9.5 mm. Make sure there's a means to shim the difference.

As far as adding RAM, I installed Htop which is a lightweight system monitor that I find more informative than Top. You can run that minimized then check it occasionally to see how much RAM you're using in your day to day activities. Add RAM if you're using a goodly % of that available of if swap gets much activity.  I've bought RAM off Ebay with pretty good success. I just make sure I can return it if it doesn't test out then install it and run memtest on it ASAP after receiving it. This is a good time of year in the U.S. to be buying tech stuff.
