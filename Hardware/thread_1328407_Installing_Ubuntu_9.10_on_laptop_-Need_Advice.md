### Post by gdonwallace on 2009-11-16
My first suggestion is to boot with the liveCD for Ubuntu.  It will react a little slower than when installed; but it will allow you to determine if Ubuntu will run on your laptop.

There are some known hardware issues with 9.10.  Mostly with the intel-graphics and some issues with wireliess connections.  Other than that, its really hard to say what problems you might encounter.

As to drivers, Ubuntu (or any linux for that matter) does not drivers.  Yes, there are some propitiatory drivers for graphics cards (i.e. Nvidia) but by and large, there are no drivers to download.  That is built into the kernel.

Start with that LiveCD and see how the laptop reacts.  If you don't encounter any real issues, install for the duel boot and go for it.
