No. 
> 
will I get troubles due to OCed ram in any ubuntu DEs?

Your RAM isn't overclocked. If you're using an XMP profile then it means the RAM is rated for that speed. 

Any issues caused by overclocking would be no different for any desktop environment, since that would be a hardware issue. 

>  is there a "best" DE you suggest for my cfg stability, my graphics and friendlyness?

That is entirely subjective. Use whichever one seems appealing. It's pretty straightforward to try another one later if you want to.
