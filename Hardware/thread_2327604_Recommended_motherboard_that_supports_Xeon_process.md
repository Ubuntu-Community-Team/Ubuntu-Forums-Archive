### Post by jake-swensen on 2016-06-12
I want to build a Linux desktop for practical home use (file sharing, office apps. suite, internet access, basic gaming, MP3s, single stream 4K video) but I see that much has changed since I built three MS Windows oriented desktops about 15 years ago.  Since then I have purchased a couple of basic Windows desktops and a tablet/laptop convertible.

Some of the changes that I am aware of:
- Much faster internet access will be available (Google Fiber is shaking up the ISP industry)
- Multi-core processors
- SSD hard drives are affordable (I have an old Samsung SSD)
- USB 3.1/C, M.2, U.2, Thunderbolt 3 interfaces are available
- HDMI 2.0a, Display Port 
- (much) better graphics cards
- Wireless network bootup is possible (iPXE?)
- Bluray discs are mainstream
- a Linux system is now far easier to adopt and maintain

In the new system I want to avoid bottlenecks such as having 1 Gb internet connection but only 100 Mb ethernet ports, or a very fast external SSD hard drive/external wi-fi 802.11ac adapter but only USB 3.0 ports.  I am leaning toward a Xeon processor *without* on-board graphics support and ECC memory.

As a good motherboard would be the foundation of such a system for practical use and allow for a future proof system (minimal upgrades for 5 years), which Linux compatible motherboard would you recommend to maximize data throughput and component inter-operability?
