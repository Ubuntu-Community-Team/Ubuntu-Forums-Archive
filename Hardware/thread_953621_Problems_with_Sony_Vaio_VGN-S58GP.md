Also, you can go into your system's CMOS (or setup program) when you first turn it on and see if it has a hard drive diagnostic test to confirm whether this might be a hard drive failure. ### Post by ardvark71 on 2008-10-22
Hi...

Ok, now we're in business! :)

Here is the link to the tool you're going to want to use, assuming your notebook is a 32 bit system, if not, please don't continue, it won't support 64 bit...

[http://www.hitachigst.com/hdd/support/download.htm#DFT](http://www.hitachigst.com/hdd/support/download.htm#DFT)

When the page comes up, click on "Drive Fitness Test" in the menu towards the top. This will bump you futher down the page where the tools are located. In all likelihood, your notebook will not have a floppy drive, so you will need to click on "CD Image." It will probably be easiest if you downlaod the file to your desktop. Once you have downloaded the file, you can do one of two things...

1. You can try double clicking on it to see a default CD burning program will come up and you can burn the file to CD this way or...

2. You can choose a CD burning program if that doesn't work or you prefer another program than what is offered as the default or if don't already have one and need to download and install one. If that's the case, here is one that I've read is pretty good as well as free of charge...

[http://www.download.com/Ashampoo-Burning-Studio-Free/3000-2646_4-10776287.html](http://www.download.com/Ashampoo-Burning-Studio-Free/3000-2646_4-10776287.html)

Just burn the file to a blank writable CD and insert the CD into the Notebook's CD-ROM drive and switch it on. If you get Windows XP instead of this program, it's possible the boot order (in the CMOS) isn't configured to look in the CD-ROM before the hard drive (for a boot record.)

On the web page you downloaded the CD image file, you can also download the "Drive Fitness Test User's Guide" for futher information and instructions.

I'd be curious to see the results of this test but I'm thinking your notebook's motherboard is the culprit, if something isn't overheating. :(

Hope this helps... :)

### Post by ardvark71 on 2008-10-23
Hi...

One way that might tell you if your system is 32 or 64 bit is by going to "Start"--->"All Programs"--->"Accessories"--->"System Tools"--->"System Information." Once this windows appears, you should see a summary of system specs, look for one called "System Type." This might tell you. If it says x86, that also means 32 bit.

Yes, you can burn the file on your PC and then use the CD on your laptop. :)
