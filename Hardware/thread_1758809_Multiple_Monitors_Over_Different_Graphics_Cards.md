### Post by awells on 2011-05-16
I believe this is not possible.  I tried this a while back with on-board ATI and a GeForce 8600 GTS.  The binary drivers would conflict with each other as well as their open source versions (Radeon & Nouveau).

I tried both the Ubuntu packages and the drivers from the vendors.  I also tried this on Ubuntu 10.10 and another distro, Arch Linux with no luck.

You are better off using the same cards and preferably the same GPUs.

I ended up purchasing a second GTS 8600, and aside from a [TwinView issue]("http://ubuntuforums.org/showthread.php?p=10352191"), it works ok.
