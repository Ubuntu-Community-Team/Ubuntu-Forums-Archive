### Post by ianmillington on 2011-08-12
Assuming this is your only machine you can visit here
[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

You will need a USB stick

Download unetbootin and then from within the program you can either download 11.04 or use a cd you already have and create a bootable usb stick.

Once that is done boot up the machine with the stick in the usb slot (making sure you have set the bios to boot from USB first) and take it for a test run. You will then be able to find out for yourself whether everything works before you do an install.

### Post by ianmillington on 2011-08-12
I made fairly broad and general comments based upon having no information on what OS you are running. I assume you are running windows on the acer, and assumed that to be your only machine. In which case there is probably little difference in unetbootin and the one referred to at the ubuntu site - I didn't know they were advocating the pendrivelinux product, but both appear equally capable.

If you have another machine already running ubuntu then you can either download the iso (or use a disk you have already got) and install usb-creator in the package manager and create the bootable disk. 

Whichever way you go, the main principle remains - the live cd/usb is the perfect way of enabling you to try before you (don't) buy :)
