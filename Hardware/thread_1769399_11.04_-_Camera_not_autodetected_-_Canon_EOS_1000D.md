### Post by MKe2 on 2011-06-01
This is a known bug:
[https://bugs.launchpad.net/ubuntu/+source/linux/+bug/786458](https://bugs.launchpad.net/ubuntu/+source/linux/+bug/786458)
The only solution I found sofar is use GTKam to download the photo's, it's in the repos
But for this you'll have to solve another bug in Natty: You'll have to install halevt:
```
sudo apt-get install halevt
```
You'll get error messages (bug in Natty)
This you can solve by editing the halevt.postinst:```
sudo mousepad /var/lib/dpkg/info/halevt.postinst
```

Comment-out the following line by placing a # in front of it:
```
invoke-rc.d --quiet hal restart
```

After rebooting, GTKam should work.
