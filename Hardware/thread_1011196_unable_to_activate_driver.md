### Post by Cocotaso on 2009-01-14
Okay i fixed the problem!!!!

If this happens and it is a fresh install you may need to run the Update Manager from  System=>Admnistration=>Update Manager

There was about 200 updates available. I downloaded/updated ...and after i restarted i clicked on "Activate" and it actually downloaded and updated the driver..i had to restart...and voila! beatiful display now :)
