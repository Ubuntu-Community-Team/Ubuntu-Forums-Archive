### Post by harmlessdrudge on 2012-12-14
So... it only took 53 weeks. 

I finally have faster graphics (even Firefox is noticeably faster)

I did a clean install of Ubuntu 12.10 AMD64 and updated it
Next, I installed Linux kernel 3.7 (instructions on upubuntu.com), then I installed Catalyst 12.12

[http://www.upubuntu.com/2012/12/inst...om-ppa-in.html](http://www.upubuntu.com/2012/12/inst...om-ppa-in.html)

The instructions here are wrong. You need to enter

[FONT="Courier New"]sudo apt-get install fglrx[/FONT]

not [FONT="Courier New"]fglrx-installer[/FONT] (the manual installation instructions above fail with a warning about a missing header file).

The software sources show 

[FONT="Fixedsys"](*) Using Video driver for the AMD graphics accelerators from fglrx (proprietary)[/FONT]

but system settings indicates

[FONT="Fixedsys"]Driver: VESA: WRESTLER
Experience Standard[/FONT]

However, the frames per second reported by glxgears

[FONT="Courier New"]sudo apt-get install mesa-utils
glxgears[/FONT]

has jumped from 301 FPS to about 650. Qualitatively it feels more than that.
