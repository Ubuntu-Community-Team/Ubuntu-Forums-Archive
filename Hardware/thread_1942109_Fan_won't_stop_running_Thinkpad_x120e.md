### Post by Mark Phelps on 2012-03-19
You've done all the usual stuff -- and since none of that is working, there's only one more thing you can do -- try 12.04.

They have made kernel updates to that version that claim to fix the overheating problem with Sandy Bridge processors.  If your Thinkpad uses that processor, then you may be in luck.

If it does not, there is nothing else you can do to fix this kernel bug.
