### Post by pauna on 2009-06-15
I bought today a Thinkpad USB Portable Secure hard drive, which uses hardware based encryption (i.e. no software installation necessary). It has been reviewed [here]("http://www.h-online.com/security/A-secure-USB-disk-from-Lenovo--/features/113192").

I did not find any mention about the HDD's Linux compatibility by googling, only some reports that it does not work with Macs. Which is surprising considering it's basically a generic USB disk with some additional electronics. So I tested it with my Jaunty laptop. The result is that it does indeed work. I have not done any extensive testing, but basic functions such as mounting, reading, writing and unmounting work just fine.

The key issue with Macs seems to be that the HDD has been formatted with NTFS, thus they have no write capability.
