i have xubuntu 18.04 installed on an acer es1-132. i'm experiencing very erratic touchpad and keyboard behaviour: the cursor suddenly jumping up the page or back several words in a sentence while typing.

i've tried tweaking the cursor and keyboard settings but this hasn't helped.

### Post by jaskah2 on 2018-06-18
well, after a bit more research on the web (where i didn't really find anything conclusive), the best workaround i could come up with was the setting for the trackpad "disable while typing." for those of you with this model acer, i'd say this is the best solution, aside from completely turning off the touchpad and using an external mouse -- which wasn't an option i wanted to pursue.
