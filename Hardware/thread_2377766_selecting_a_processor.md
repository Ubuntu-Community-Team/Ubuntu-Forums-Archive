Both will work fine. Base your decision on price, performance, power consumption, etc.

> Any suggestions would be appreciated

Do your homework, especially on whatever specific motherboard model you consider.

> Also memory size

It depends on what you are doing. If you run multiple VM's or are doing other heavy multitasking (like 100's of browser tabs open), 16GB is not out of the question. 8GB is more realistic otherwise. 4GB if on a a tight budget.

### Post by sgian on 2017-11-17
I got unjustly flamed here once over AMD, including by one of the mods.  So this is a sensitive subject with some people.  

Personally, I use both.  Right now one of my computers has an AMD CPU with the Carrizo architecture, and that is not yet well supported on linux yet.  My Athlon X4 845 will not work with the older kernels in Ubuntu 16.04, and only works with the Ubuntu 17.10 kernel if I don't use the proprietary driver for my NVIDIA cards.  So whatever the brand, just research the model number and architecture name before buying it.
