### Post by marshall.robert on 2009-03-30
Basically, what you have is CPU Scaling, which is where the CPU scales down its running speed to match how much it has to do in order to save power.

In my case, with my laptop running on an Intel Core2 Duo, my CPU scales down to 800MHz when doing basic things like browsing the web, but then scales back up to its full speed of 2GHz when doing something resource heavy, such as a game or logging in.

There is a CPU Scaling Gnome panel applet which allows you to see what speed you are currently running at.

Hope that explains it.

### Post by x33a on 2009-03-30
Take a look at the processor usage when running a program, if the load on the processor is more, then it means that more processing power is being used, and hence more usage of electricity. e.g., running games, watching HD movies, etc.

if you are running a torrent program, normally the processor usage is very less. for example, i use rtorrent and it barely puts any load on the cpu, so less usage of electricity. 

i don't think you'll save electricity substantially by buying a less power consuming cpu, unless you use the computer continuously and at load.
