### Post by Brooster1 on 2010-11-24
I was experiencing the same 10.04 installation problem that bluegoldtom described and read through all the replies here.  The hardware that I was installing onto was a Zotac GF-9300-G-E ITX motherboard with integrated NVIDIA graphics (different from what bluegoldtom has).  Similar to what bluegoldtom discovered, my problem was also resolved by removing a wireless ethernet adpater (Via VT6656).  Fortunately, and with the help of this forum and this thread, I was able to quickly get to the solution and bypass all the grub boot parameter experimentation, which in my case would have never solved the problem.  Thank you Ubuntu Forums!  And thank you bluegoldtom for initiating this thread.
