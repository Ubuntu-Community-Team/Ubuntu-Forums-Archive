### Post by sudodus on 2013-03-05
Ubuntu 10.04 LTS desktop is soon at it's end of life, April 2013, while the server version (without graphics) will receive security updates for two more yearss.

I guess you have the desktop edition, so I suggest that you choose a new version, the current long time support version 12.04 LTS or 12.10. These versions have much better support for hardware and peripheral devices like graphics cards/chips and printers.

If you want more precise advice, please post the specifications of your computer, at least cpu, ram, graphics chip/card and wifi chip/card :-)
