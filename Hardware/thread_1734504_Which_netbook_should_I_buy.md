### Post by rg4w on 2011-04-20
I've had good luck with my EeePC 901, able to run both the NBR and 10.10 standard with no troubles at all.

I originally got the 901 because it was the smallest I could find with a 1024x600 screen, and was fully solid-state (no HDD).  I'm no longer as concerned about having a hard drive, and the 901's keyboard size has proven to be a big compromise for a touch typist, so if I were to shop today I might consider the 11.6" Acer Aspire to get a little more screen real estate as well.
