### Post by Fafler on 2010-12-01
You should stick with your E4300. It's a lot faster than any old P4.

Apart from that, CPU's without integrated memory controllers doesn't care what kind of memory they use. It's chipset dependant. And memory modules can always run at lower speed, if the motherboard doesn't support their native speed, eg. 800 mhz memory will run 533 mhz or even lower.
