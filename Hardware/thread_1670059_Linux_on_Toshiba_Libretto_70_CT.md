### Post by gordintoronto on 2011-01-18
How much memory? With 512 MB you can install Ubuntu, with 256 MB you can use Lubuntu, less than that and you're into Slitaz or Damn Small Linux.

I would be amazed if Wubi would work under Windows 98.

Hard drive space might even be an issue. I'm pretty sure I installed Windows 98 on a 1 GB hard drive.

Does the machine have the ability to boot from CD or from a flash drive?

You might be able to put the hard drive into an external drive adapter, install some form of Linux on it, then put it back in the Libretto.
