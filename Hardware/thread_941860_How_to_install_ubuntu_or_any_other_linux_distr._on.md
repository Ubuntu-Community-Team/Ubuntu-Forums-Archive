### Post by tagawa on 2008-10-20
Hello Yukio,

The Ipaq Hx2000 series doesn't seem very compatible with Linux.
The only work I know of is documented here:

[http://www.handhelds.org/moin/moin.cgi/HpIpaqHx2000](http://www.handhelds.org/moin/moin.cgi/HpIpaqHx2000)

It seems it's possible to boot the Linux kernel and run [GPE]("http://en.wikipedia.org/wiki/GPE")

Apparently it's very alpha and by the look of it no longer maintained.  You don't need to flash your device, however, so it may be worth a look. I intend to give it a go when I have time.

Hope this helps.
