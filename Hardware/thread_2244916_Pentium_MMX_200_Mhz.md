### Post by tgalati4 on 2014-09-20
Don't give up that easy!  Freenas (older version) will run nicely as will Tiny Core Linux.

[http://www.nas4free.org/](http://www.nas4free.org/)  I've had my freenas server running on a Pentium MMX 166 MHz overclocked to a rousing 188 MHz.  Uptime 608 days!  I maxed out the RAM to 256 MB, but you can run on less.  Now technically this is BSD (unix) and not linux, but it smells a lot like linux.

Uptime 	608 day(s) 6 hour(s) 45 minute(s) 2 second(s)

[http://tinycorelinux.net/welcome.html](http://tinycorelinux.net/welcome.html)  Stripped down Debian that needs only 12 MB and some patience.

Don't throw away 17-year old hardware, unless you don't know what to do with it.

For me, it's a 35-watt home server that is worth keeping, just on principle.  I put in a PCI SATA card and stuffed it with several, terabyte green drives.  I even have a ZFS pool running across some of the drives.
