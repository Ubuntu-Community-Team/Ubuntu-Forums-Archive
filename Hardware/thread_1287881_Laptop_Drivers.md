### Post by UpsideDownFace on 2009-10-10
I have a ThinkPad R50e with originally nothing but Ubuntu 6.06 on it.
Everything worked after install except for the wireless link.
Changing to Ubuntu 7.10 made the wireless work, and I now use Ubuntu 8.04 on all my linux computers, because it is "Long Term Support"

If you want to keep windoze, then reduce its partition size to about half, using a live CD and System>Partition Manager.
Then install Ubuntu into the rest of the hard drive.
I always make a root partition  "/" about 10 GB and a home partition "/home" about 20 or 40 GB, but this means choosing "Manual" for the partition method.

The way to learn about linux is just to get on using it. When I started I couldn't understand why it was different from windoze - now I have difficulty remembering to do things a non-linux way.
