### Post by hiko2 on 2008-09-28
no problem i solved the problem, it appears it's a a problem with some firmware for a ts-l32d disk from samsung, to anyone one out there having a similar problem, run the package hardinfo, check if your disk is ts-l632d and if it's go to [https://bugs.launchpad.net/linux/+bug/75295/comments/97](https://bugs.launchpad.net/linux/+bug/75295/comments/97) and follow the solution there, good luck to everyone.:D
