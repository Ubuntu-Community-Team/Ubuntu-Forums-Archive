### Post by drober05 on 2013-08-28
I have an Acer C7 Chromebook with Chrubuntu installed.  Everything has been working splendidly with one exception.  I cannot seem to write more than a few files to the SD Card without the system locking up.  Looking at the system log there are lots of write errors on the SD Card.  It IS possible for me to copy individual files to the SD Card just not copying a lot.  For instance, if I select 1 photo to copy to the SD Card it works fine.  If I select 20 photos then it writes maybe 4 or 5 photos then locks up.

This is a problem for me because I use a program called Synkron to keep data synchronized between my work PC (Windows) and my C7.  Synkron works fine as long as it is only reading data from the SD Card and writing it to the C7 hard drive.  Once it hits a series of files to copy from the C7 hard drive to write to the SD Card, it locks up Synkron and (eventually) the C7.  

The SD Card file system is vfat (not sure if this might be an issue).
