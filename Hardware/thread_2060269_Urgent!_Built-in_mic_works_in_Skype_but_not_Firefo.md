### Post by squaregoldfish on 2012-09-20
You may well need to allow Flash to have access to the microphone. The Flash Settings Manager:

[http://www.macromedia.com/support/documentation/en/flashplayer/help/settings_manager02.html](http://www.macromedia.com/support/documentation/en/flashplayer/help/settings_manager02.html)

will allow you to specify which URLs are allowed access to the microphone.

Note that, for me, the settings manager never remembers what you set between browser sessions so you'll need to do this every time.
