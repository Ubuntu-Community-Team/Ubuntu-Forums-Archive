### Post by mikewhatever on 2011-06-17
Linux distros, including Ubuntu, have a handy feature called Live CD. It allows the user to run and evaluate Ubuntu from a CD/USB without installing. This way you can test most of the hardware, not only audio. In Windows, people are used to installing multiple drivers to make things work, apparently even multiple drivers for one device, which is bizarre. We usually don't have to do any of that with Linux, especially for audio, unless the hardware is very very new. Sometimes, you may need to edit a config file, but most of the drivers are included.

Linux does drivers completely differently from Windows. Run a LiveCD, and you should find that everything "just works."
