1) Go to System \ Mouse \ *uncheck* "Enable mouse clicks with touchpad
**OR:**
2) Open Synaptic, do a search for "disable touchpad" (in Description and Name) and install something like touchfreeze, for example.

[COLOR="DarkOrange"][FONT="Comic Sans MS"]The newly installed application should appear in your menus [/FONT][/COLOR](most likely in "Accessories")
