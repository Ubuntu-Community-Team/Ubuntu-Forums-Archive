### Post by arubislander on 2010-09-07
You mention that your wifi died unexpectedly, which means it worked before. So it is very unlikely to be a driver issue unless you installed or removed some wifi related software before it died.

More probably it's just a matter of your machine going into hibernation and then waking up with the wifi disabled.

If you are using Gnome check that wireless is enabled. Go to the network icon in the notification area and right click on it. Confirm that Enable wireless is checked.
