### Post by PRC09 on 2010-01-14
You can find the driver in System > Admin > Synaptic Package Manager.Then type in hplip in the search box and install from there.If that one doesnt work the latest version is available from the following link. Usually you can install the printer by just turning it on and plugging it in and following the prompts as HP has pretty good support in Ubuntu....
