### Post by dkolars on 2009-11-22
OK -- was able to get into the BIOS and discover that there is no reference to the Touchpad there.  OK, fine.

Surfing the web, found a post in a Ubuntu newsgroup from 2005 that said where gsynaptic put itself when installed:  System, Preferences.  Sure enough, there it was, out of sight at the END of the listing, not alpha like everything else (!), and I was able to instantly disable the touchpad.

So, had Ubuntu bothered to sort the listing after I installed the program, I would have seen it when I went hunting.  And, the color scheme is such that the down arrow was virtually invisible on the menu, so I didn't scroll down...

Anywho, problem solved, and I'll be finding a new color scheme that SHOWS THINGS that you need to see!!!
