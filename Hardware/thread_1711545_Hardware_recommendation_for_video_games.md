### Post by Thund3rstruck on 2011-03-21
I built an full blown arcade a few years ago and I can say that if you're gonna go through the trouble to create a dedicated box like that then seriously consider a full-service frontend for it. 

I used Windows 2000 as the operating system for my arcade cabinet with MALA, which runs everything from Atari through CPS-2. There are some other fantastic frontends also like Hyperspin you may want to investigate.
