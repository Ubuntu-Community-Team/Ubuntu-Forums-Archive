### Post by coldraven on 2014-04-04
When you plug it in does it show up as a USB Mass storage device? If so you would be able to copy/paste your files. (Look for folders called F1, F2 etc.)
As for it's other functions I don't know if they would work without the correct driver which will not exist for Linux.
