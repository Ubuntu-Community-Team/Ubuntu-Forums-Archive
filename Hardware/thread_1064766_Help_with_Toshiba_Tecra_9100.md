### Post by RobsterUK on 2009-02-09
A colleague asked me for help because he couldn't read SD cards on a Toshiba Tecra 9100 using the built in slot. (Ubuntu 8.10)

According to a thread on the Toshiba forums the SD card reader on this model only supports up to 128MB cards, so for anyone else having issues with this, that might be your problem.
[http://forums.computers.toshiba-europe.com/forums/message.jspa?messageID=89743](http://forums.computers.toshiba-europe.com/forums/message.jspa?messageID=89743)

The reason I've posted this is because I came across an archived (closed) thread about it and thought this information might be useful.
[http://ubuntuforums.org/showthread.php?p=4390882](http://ubuntuforums.org/showthread.php?p=4390882)
