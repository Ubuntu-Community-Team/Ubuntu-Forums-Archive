### Post by sushrut91 on 2009-09-07
:P The Problem is solved, Just edited my xorg.conf file as suggested by the following link :

[http://www.krishnamoorthy.in/2009/05/ubuntu-904-nvidia-7300-gt-widescreen.html](http://www.krishnamoorthy.in/2009/05/ubuntu-904-nvidia-7300-gt-widescreen.html)

My xorg.conf file is blank. 

I made following changes in the monitor in the above link xorg.conf before it went blank.

It seems you just have to edit the values for your monitor.

**Section "Monitor"**
**Identifier      "Lg Flatron L177WSB"**
**VendorName      "LG"**
**Modelname       "L177WSB"**
**Horizsync       30-81**
**Vertrefresh     56-75**
**EndSection**


**Section "Screen"**
**Identifier      "Screen0"**
**Monitor         "LG Flatron L177WSB"**
**Device          "nVidia Corporation C51PV [Geforce 6150]  "**
**Defaultdepth    24**
**Option          "AddARGBGLXVisuals"     "True"**

**SubSection "Display"**
**Depth           24**
**Modes           "1440x900" "1280x1024" "1280x960" "1024x768" "800x600"**
**Viewport        0 0**
**EndSubSection**
EndSection


:) Now I get 1440 x 900 resolution running nice without any problem.
