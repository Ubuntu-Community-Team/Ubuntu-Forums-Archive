### Post by mtbsoft on 2009-08-03
Almost certainly not.  The firmware of the device will most likely be designed to work with that file system only - if you change it you will almost certainly stop it working and could even "brick" it if you're really unlucky.

It would be like trying to get Windows to read ext3 without being able to install a driver.
