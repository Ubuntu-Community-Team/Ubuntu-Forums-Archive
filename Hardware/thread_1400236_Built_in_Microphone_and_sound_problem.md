### Post by macabrem on 2010-02-06
I had this problem also.  See this thread that I started, but look at the link in the first post.

[http://ubuntuforums.org/showthread.php?t=1399448](http://ubuntuforums.org/showthread.php?t=1399448)

The only thing I got to work was to install the older version of Skype, and then you can adjust the settings.  Unfortunately, the older versions of Skype are no longer available on Skype.com, so what I did was google the exact version "skype-debian_2.0.0.72-1_i386.deb" and found a download location for that version.  (You might try "skype-debian_2.0.0.72-1_i386.deb download" as your exact search).  It was one of the first hits that came up on my search that I downloaded from.

I hope this helps.
