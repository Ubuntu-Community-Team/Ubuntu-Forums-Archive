### Post by mörgæs on 2017-05-11
It happens only rarely that support for old hardware is withdrawn. I know there are exceptions but main rule is 'if it worked once it will continue'.

I have installed Buntus on many Dimensions in the 500, 600 and 800 series, all working fine. I would recommend Lubuntu or Mate in stead of Ubuntu for a 620.

Always install with a wired internet connection.
