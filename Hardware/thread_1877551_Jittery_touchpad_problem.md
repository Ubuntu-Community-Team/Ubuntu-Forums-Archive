I'm somewhat new to Ubuntu. I installed it on a laptop with a touchpad. 

The touchpad works fine when I'm moving the mouse, and the pointer stays perfectly still when my finger is not on the touchpad.

However, when my finger is standing still on the touchpad the mouse pointer keeps moving sporadically by a small amount. It's a big enough jitter that it makes selecting small things a bit of a challenge..For example, it's very hard to select a window edge when trying to resize it because the pointer jitters more than the width of the window edge.

I did not have this problem when I was running windows on the machine, so I suspect it's not a hardware problem. 

It's almost as though there should be a dead-zone, or a limit for at least some minimum movement on the touchpad before the mouse actually moves.

### Post by killobyte on 2011-11-08
I have a very similar issue with the driver on my toshiba laptop, the touchpad jumps everywhere on certain distros of linux, one of the solutions to it (not real fix, just a bandaid) was to set the sensitivity to the lowest possible setting, which did correct it to the point where it was usable.

Maybe try messing with the sensitivity settings under the mouse menu.
