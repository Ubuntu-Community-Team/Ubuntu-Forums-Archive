### Post by Grenage on 2011-02-16
Try adding the modes manually:

```
gksu gedit /etc/X11/xorg.conf
```

```
Section "Screen"
    Identifier     "Screen0"
    Device         "Device0"
    Monitor        "Monitor0"
    DefaultDepth    24
    SubSection     "Display"
        Depth       24
        Modes       "1024x768" "800x600" "640x480"
    EndSubSection
EndSection

```

You'll need to restart X afterwards, but it's easier just to reboot.

This way cant help as well...

But I have now realized a way that worked for me: use "xorg.conf.failsafe"

```
Section "Device"
	Identifier	"Configured Video Device"
	Driver		"fbdev"
EndSection

Section "Monitor"
	Identifier	"Configured Monitor"
EndSection

Section "Screen"
	Identifier	"Default Screen"
	Monitor		"Configured Monitor"
	Device		"Configured Video Device"
EndSection

```

This file is automatically generated when we go to failsafeX Mode(recovery mode). Just copy and rename it to "xorg.conf".
Replace "fbdev" or "vesa" by "nvidia"

```
Section "Device"
	Identifier	"Configured Video Device"
	Driver		"nvidia"
EndSection

Section "Monitor"
	Identifier	"Configured Monitor"
EndSection

Section "Screen"
	Identifier	"Default Screen"
	Monitor		"Configured Monitor"
	Device		"Configured Video Device"
EndSection

```
