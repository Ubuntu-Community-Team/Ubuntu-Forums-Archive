### Post by Lars Noodén on 2011-12-10
There's no way to be sure without actually trying it on your machine.  

What you can do is save time by using the Alternate installation disc and installing a "Command Line System" instead.  That takes only a few minutes to install and then you can see if it boots.  If it boots then you can finish installing the desktop of your choice.
