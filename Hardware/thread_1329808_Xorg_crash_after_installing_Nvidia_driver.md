### Post by madengineer on 2009-11-17
I can't tell from the logs what version of the NVidia driver you're using; I'd actually suggest moving /etc/X11/xorg.conf to a backup (like /etc/X11/xorg.conf.bak) so that Xorg can autodetect things; it seems to be pretty good at that, unless you have a laptop like mine that needs things tweaked to prevent suspend/resume problems.\

Right now your Xorg seems to be not finding the driver at all; make sure that you have the packages nvidia-96-kernel-source, nvidia-96-libvdpau, nvidia-96-modaliases, and nvidia-glx-96 installed; remove all nvidia packages with other version numbers (like 173, 180, or 185). After the installation completes, run ```
sudo depmod -aq
```.

If this still fails, try installing package envyng-core, and then run ```
sudo envyng -t
```. This will give you a series of prompts to install your driver and set up all configuration stuff for you.
