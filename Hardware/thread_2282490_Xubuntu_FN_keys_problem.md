### Post by efflandt on 2015-06-19
Typically if you have a key with blue "Fn" on it that is for specific ACPI functions for the typically blue print on other keys like for enabling/disabling certain hardware like wireless, Bluetooth, switching displays, webcam, etc. So the set functions for those keys when used with the Fn key may happen in the background (if supported by Linux) and not show up as normal keycodes. I am no expert though, just tossing that out as a possibility.
