### Post by jeroen.kurvers on 2009-02-24
Fat32 works fine under linux and Windows. Only problem with Fat32 is a maximum filesize of 4GB. You could also use NTFS, but then you would need to install the nfts-3g package from the repositories ([http://www.ntfs-3g.org/](http://www.ntfs-3g.org/)).
