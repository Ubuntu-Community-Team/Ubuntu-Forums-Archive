### Post by jmfal on 2012-07-10
Welcome to Ubuntu

The only driver for epson in the repos is "printer-driver-escpr"
```
 sudo apt-get install printer-driver-espcr    
```

Their website has driver for Ubuntu if that one won't work:

[http://global.epson.com/newsroom/2011/news_20110331.html](http://global.epson.com/newsroom/2011/news_20110331.html)
