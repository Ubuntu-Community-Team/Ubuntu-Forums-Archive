### Post by mariner09 on 2008-12-27
You have compiz.  Go into Control Panel - Appearance - Visual Effects and set it to "None".  Then see if it locks up any more...

If you want to turn them back on, install Compiz Config Settings Manager, CCSM and then uncheck the setting mentioned previously and it won't lock up any more.

You need compiz if you want to use any eye-candy like AWN or Emerald.
