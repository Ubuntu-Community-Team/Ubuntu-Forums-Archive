### Post by pastalavista on 2009-04-10
My Atheros was shaky for a while but has been rock solid ever since I upgraded to Jaunty. (kernel 2.6.28-11) It installed a proprietary driver called 'alt2-source' which is a 'madwifi' replacement for ath_pci. You'll probably need to add Jaunty repositories to get it and try ```
sudo apt-get install alt2-source
```
