### Post by Rocket2DMn on 2008-05-07
I think the only thing to really avoid is HP laptops, I know lots of people have problems with them.  In general, Dell laptops tend to have pretty good support, but I think so do Sonys, Toshibas, etc...  You should shop around until you see something you like, at which point you might want to post back to see if other users have had problems with that make/model.
Compiz Fusion is likely to run better on ATI and Nvidia cards since they have dedicated memory and tend to be more powerful.  That being said, I think people with intel graphics cards have had pretty good luck with CF, I haven't seen to many requests for help with them.

### Post by Rocket2DMn on 2008-05-08
Computers that come preinstalled with linux can be pretty convenient.  However, some companies will sell you these computers for only a few dollars less than the same computer with Vista.  If you are interested in dual booting or think you will need windows, the few extra dollars for the windows version machine can be worth the investment to have an up to date windows operating system.  Then you can setup your dual boot after you get the machine (since you can get the linux OS of your choice for free anyway).
