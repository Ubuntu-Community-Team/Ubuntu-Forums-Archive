### Post by michaelb28 on 2011-09-11
I did a little searching on Google and found this:

[http://www.zoneos.com/zonescreen.htm](http://www.zoneos.com/zonescreen.htm)

It looks like you would install a ZoneScreen Virtual Display Driver on your Windows desktop.  Then you install a VNC server on your Windows desktop and VNC client on your Ubuntu laptop. Some users report stability problems on Windows 7 though.

There's also a commercial program:  [http://www.maxivista.com/](http://www.maxivista.com/)
No Linux client exists, but some users have gotten it to work in Linux with WINE.
