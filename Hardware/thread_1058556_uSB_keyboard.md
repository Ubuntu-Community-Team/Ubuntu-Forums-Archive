### Post by Ripose on 2009-02-13
I'm not sure this will provide any useful info but post your** /etc/X11/xorg.conf** keyboard section.

Mine looks this:
```
 Section "InputDevice"
    Identifier     "Keyboard[0]"
    Driver         "kbd"
    Option         "Protocol" "Standard"
    Option         "XkbLayout" "us"
    Option         "XkbModel" "microsoftpro"
    Option         "XkbRules" "xfree86"
EndSection
```
