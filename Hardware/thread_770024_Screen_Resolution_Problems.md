### Post by natrixgli on 2008-04-27
Ok, what you just posted is the configuration file for the X server, which handles among other things, video display.

You can add some lines to tell X what resolutions your monitor is capable.


To edit this file, open a terminal and do this:

*NOTE: when asked for your password, type it but you won't see it displayed on the screen... That is normal.*

backup the original:

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak
```

open the file to edit:

```
sudo gedit /etc/X11/xorg.conf
```

Now look for this part:

```
Section "Screen"
	Identifier	"Default Screen"
	Monitor		"Configured Monitor"
	Device		"Configured Video Device"
	Defaultdepth	24
EndSection

```

After "Defaultdepth    24" but BEFORE "EndSection" add this:

```
	SubSection "Display"
		Depth		24
		Modes		"1280x1024" "1024x768" "800x600"
	EndSubSection
```


The edited part should then look like this:

```

Section "Screen"
	Identifier	"Default Screen"
	Monitor		"Configured Monitor"
	Device		"Configured Video Device"
	Defaultdepth	24
        SubSection "Display"
		Depth		24
		Modes		"1280x1024" "1024x768" "800x600"
        EndSubSection

EndSection


```


When you're finished, save it. Then save any open docs and hit CTRL+ALT+BACKSPACE to restart your X server. You should then be able to select more resolutions.



Let us know how it goes!
