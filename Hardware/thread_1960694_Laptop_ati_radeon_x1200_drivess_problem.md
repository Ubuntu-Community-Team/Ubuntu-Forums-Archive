### Post by Mark Phelps on 2012-04-18
Since you are obviously getting to a desktop (you would say if you were stuck in a command line interface), the graphics drivers MUST be installed.

As mentioned, the System Info report is unreliable.  Your drivers are already installed -- as they were previously in 11.10.

There are no restriced drivers that will work with your radeon card and any Ubuntu versions newer than 8.10.  So, don't go searching around for newer drivers, there aren't any.
