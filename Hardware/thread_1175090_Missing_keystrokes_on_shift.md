### Post by mikezila on 2009-05-31
This behavior is by design.

Almost all keyboards have this "problem".  To save on costs, the manufacturer uses as few switches as possible, which means that some very unlikely key combinations won't work.  Things like holding both shifts, both alts, both controls, or similar sometimes produce missed keys.  Even some nicer keyboards do it.  It's common to do a test by typing "The quick brown fox jumps over the lazy dog." with both shifts held down to see how your keyboard fares.

Here's mine:
H CK BN FX JMS V  LAZY DG

That's on an aluminum Apple Pro keyboard, so it's not just cheap ones that do it.  The only keyboard I've ever owned that didn't miss a single key was a generic pack-in keyboard from an ancient gateway computer, so I guess you never know.
