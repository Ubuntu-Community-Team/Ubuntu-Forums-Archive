### Post by fergalm on 2010-12-31
I'm probably too late to influence the Original Poster, but here are my notes if anyone else is thinking of buying one. I have two issues.

I can't boot from usb key. The boot starts, and after the initial message, the load just hangs. I had to install using wubi instead.

As mentioned above, the internal microphone doesn't work. Actually it does, just very poorly. If you click on Applications (I'm using the Unity interface), and select Sound, then "Input", you'll see the mic is muted by default. If you unmute, and test it using the Sound Recorder app, you can hear yourself speak, but only very faintly. I have to whistle into the mic to be heard.

Other than that I've see no problems. Video works. Firefox is quite fast, although the general interface is comparatively slower. This is the first netbook I've tried so I don't know how fast it should be.
