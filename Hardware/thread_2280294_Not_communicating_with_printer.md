### Post by weatherman2 on 2015-05-29
Changing the cartridge should have no impact on the driver - I can't imagine how it could.  Printers do fail sometimes - it wouldn't be unheard of.

Is this a USB printer?  Obviously, check the cables.

You mention that the printer worked fine with earlier versions of Ubuntu. Are you implying that you've also upgraded to a newer Ubuntu version at the same time you fiddled with the cartridge?  If so, try booting an earlier version of Ubuntu via live USB or live CD, and see if you are able to print again with the older Ubuntu.  If so, then you obviously have some issue with the newer Ubuntu that has nothing to do with the printer cartridge or the printer itself.
