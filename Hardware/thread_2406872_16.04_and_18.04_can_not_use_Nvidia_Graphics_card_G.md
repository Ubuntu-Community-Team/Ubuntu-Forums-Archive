Upgrading quite often causes problems like this.  I learned a long time ago to only do clean installs.  You can probably save yourself a lot of grief and time by just clean installing 18.04.  You can use the install medium to boot the machine and then get access to the hard drive(s) to copy anything that you need to keep to an external source.

I actually have that same video card running quite well on 18.04.
