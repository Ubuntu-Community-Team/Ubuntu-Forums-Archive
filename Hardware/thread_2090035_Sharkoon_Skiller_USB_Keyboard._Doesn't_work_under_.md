### Post by QIII on 2012-12-01
If it needs a driver, that's up to the OEM to provide.  The place to find a Linux driver, if it exists, is the OEM.

It may work fine with Windows because the OEM has jumped through the necessary hoops to get its driver into the Microsoft driverbase so that it is PnP.  If the driver is in the MS driverbase, you don't need the disk.  But they are often included anyway.

If it needs a driver to support standard I/O, shame on the OEM.

If it needs a driver to support specific functionality that is specific to Windows or software designed for Windows, that probably isn't going to work.
