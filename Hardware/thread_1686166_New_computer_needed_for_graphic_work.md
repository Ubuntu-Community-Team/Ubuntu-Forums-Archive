### Post by Dark_Stang on 2011-02-11

However if you're going to use this thing as your main work macine (I have a monster desktop as my primary machine in the office) I would recommend going with the best video card you can afford at the time. USB3.0 is via addon-chips, and its easy to get 870 (and 890FX) motherboards with USB 3.0. SATAIII is via the SB850 southbridge, as long as you get a SB850 the board will do SATAIII. Some 880 chipsets use the SB7XX series, AFAIK all the 870, 890FX and 890GX motherboards use SB850. 

If you really want microATX then dont bother looking for a 870/890FX. They are all full ATX.
