### Post by DottorBartolo on 2010-08-05
I have had a similar problem (the right crtl key didn't work) and I solved it by going to:
System > PReferences> Keyboard
Then, in the layout tab, I set the keyboard model of my laptop, but that wasn't enough. In layout options, I clicked on "compose key position": the "right ctrl" box was checked, for some reason. Well, unchecking it solved the problem!
