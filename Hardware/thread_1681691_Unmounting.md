### Post by Afishionado on 2011-03-10
When you shut down an operating system like Linux, everything gets unmounted. So if Ubuntu isn't still running, I don't see how this could be an Ubuntu issue.

Windows was able to run from this drive before, correct? What did you change since then? Did you alter any jumper settings?

You might get more help on a Windows forum than here, because this really sounds like a Windows issue.

One more thought: What device are you booted from when you try to perform the format? If you are booted from the same drive that you are trying to format, that's your problem. Find some rescue disk to boot from.
