### Post by darrelljon on 2008-05-19
Both Xubuntu and XP would quickly take up almost the whole hard disk drive.
Xubuntu requires 1.5Gb hard disk space to install.
Windows XP requires 1.5Gb hard disk space to install.

Go with [Damn Small Linux]("http://www.damnsmalllinux.org/wiki/index.php/Minimum_Hardware_Requirements") which only requires around 200Mb hard disk space to install.
To be honest, you may not even be able to boot from CD on an old BIOS in which case you will need a boot floppy.
