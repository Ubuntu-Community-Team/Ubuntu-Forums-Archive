### Post by 2F4U on 2012-04-14
If it hangs already in the BIOS phase of the boot process it is most likely a hardware problem or some problem in the BIOS firmware. Do you get beeps? On some computer models, beep codes indicate what errors occurred and aid diagnosing the problem. You could also check if a newer BIOS version is available.
