The quick fix use to be installing Robert Gerlach's fjbtndrv PPA:  [https://launchpad.net/~khnz/+archive/ppa?field.series_filter=](https://launchpad.net/~khnz/+archive/ppa?field.series_filter=)  The fjbtndrv git repository is at:  [http://sourceforge.net/projects/fjbtndrv/](http://sourceforge.net/projects/fjbtndrv/)

But it looks like it has been a while since he worked on it and he may no longer be supporting it.  Certainly there's no Natty version I am aware of.  So it would be good if you could take it up.  Otherwise for rotation see the [Rotation HOW TO]("http://ubuntuforums.org/showpost.php?p=6274392&postcount=1").
