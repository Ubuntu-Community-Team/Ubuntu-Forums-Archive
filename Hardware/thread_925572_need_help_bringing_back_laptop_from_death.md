### Post by IntuitiveNipple on 2008-09-20
From the photo's I notice it has a rear VGA port. Have you connected a monitor to that in case it is being polite and outputting video to all connections until the OS starts?

With a monitor hooked up try booting a Live CD and even pressing the correct keys blind to get Ubuntu starting... it might then start using the external VGA port.

It also seems to have an external serial port. You could hook up a serial console link to it and create a boot CD with the kernel command-line including serial console output - just to get some clues.
