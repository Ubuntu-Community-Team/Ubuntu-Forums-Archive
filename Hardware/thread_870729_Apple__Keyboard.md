### Post by coffeecat on 2008-07-26
If you are using Ubuntu/Gnome, go to System > Preferences > Keyboard. Now click on the Layouts Tab in the Keyboard Preferences window. In keyboard model choose Apple Macintosh. If the layout pane doesn't show the country/language layout you want, click on the Add button and choose from Layouts.

If you are using Kubuntu/KDE, then you need Regional and Accessibilty (I think - I'm going from memory) in the KDE Control Centre, but I can't give you any more details.

If you are using Xubuntu/Xfce, then it's more tricky. I've never been able to find a comprehensive keyboard layout configurator in the GUI - or at least one that includes Apple keyboards. What I have been able to do to get Xfce to work with a UK Apple keyboard is to modify the InputDevice section of /etc/X11/xorg.conf.
