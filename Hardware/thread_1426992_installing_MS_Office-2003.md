### Post by khelben1979 on 2010-03-11
If you run [winecfg]("http://wiki.winehq.org/winecfg") you have the possibility to change the default settings for wine including which screen resolution it should use.

Also, you might consider testing using the beta versions of Wine which is found from [the WINE homepage]("http://www.winehq.org/"), it might give you better stability. And if it doesn't, you might be limited by too little RAM or your /home/[user] catalog might be a bit filled up which also can create problems with instability.

I have noticed that rebooting the system when Wine works bad have solved the problems with stability on my system, but it's nothing I'm very happy about doing.
