**question1**: can I install the drivers of my printer, hp 940c, in windows xp which is located in the virtual box?

Yes. You can just go to HP website and download the driver via your VB. Watch out that for XP you might not find the driver in your language, so you have to select another one (e.g. english).

Do not forget to allow usb devices to be seen from VB as described here: 
[http://ubuntuforums.org/showthread.php?p=12370366#post12370366](http://ubuntuforums.org/showthread.php?p=12370366#post12370366)  (posts #4 and #5)

**question2**: my printer cannot print photos due to restrictions to the linux driver. Is it possible to print a photo under windows xp via virtual box, using the drivers installed from question1?

Yes! Both photos and mirror images for ironing photos on T-shirts.
