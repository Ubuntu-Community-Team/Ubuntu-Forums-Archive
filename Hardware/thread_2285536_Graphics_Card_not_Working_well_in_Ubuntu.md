### Post by john409 on 2015-07-06
I have a  Gigabyte GV-R927XOC-2GD Radeon R9 270X 2GB graphics card.

When I run it on windows, all 3 screens function as expected. When I run it on Ubuntu I only can manage 1 display. The others are not active at all.

When I unplug one, and restart, Ubuntu will pick another to use, but still only that one. I'd just return the card and get another, but the NewEgg return policy doesn't allow it.

Suggestions?

---

### Post by john409 on 2015-07-06
I figured it out. (I hope) I had to download the AMD proprietary driver using the System Settings -> Software & Updates -> Additional Drivers
