### Post by cascade9 on 2009-12-23
1. Yes. They are probably only single core so more like a dual though. 

2. Yes, normally. There might be dual-cpu boards that need both CPUs to run, but I've never seen one. 

3. Yes. 

4. Depends. 

5. No. 

6. No. That's just for dual-head. 

I'm pretty sure you will need RAM in both banks, but I'm not 100% sure. BTW, its common for dual-CPU boards to need EEC/registers/buffered RAM, and its more than possible that your standard SD RAM wont work.
