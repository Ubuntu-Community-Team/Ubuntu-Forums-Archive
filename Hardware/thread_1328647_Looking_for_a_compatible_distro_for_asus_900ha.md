### Post by snowpine on 2009-11-17
Hi Masterblah, the reason you aren't getting a lot of answers is probably because you haven't really said what you're looking for in a distro. Try a bunch of them and see which you prefer. If you want a comparison of the top distros, here's some good reading material: [http://distrowatch.com/dwres.php?resource=major](http://distrowatch.com/dwres.php?resource=major)

I have an Asus 900ha too. I tried a lot of different Linux distros, and most recent releases support the eee very well. :) I'm currently using Cruncheee 8.10. It is an Ubuntu derivative, using the Openbox windows manager. The "lightest" distro I've ever used on my eee was SliTaz. It was very fast, but I switched back to Cruncheee because it is more comfortable for me. Good luck!
