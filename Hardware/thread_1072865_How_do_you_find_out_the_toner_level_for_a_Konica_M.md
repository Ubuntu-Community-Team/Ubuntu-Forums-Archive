### Post by cariboo on 2009-02-19
Usually the printer will tell you when it is low on toner. I would suggest the cups web interface to see if there is any indicators. I have have a really old HP 5P that doesn't have a toner level indicator, but an light (no led's here) starts flashing on the control panel when the toner is getting low. 

To access the cups web interface open firefox and type in the address bar:

[http://localhost:631](http://localhost:631)
