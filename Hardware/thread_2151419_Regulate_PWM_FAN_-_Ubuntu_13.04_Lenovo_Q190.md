### Post by gordintoronto on 2013-06-04
Which CPU do you have? If it's the Celeron, it's running flat out any time it's playing HD video, and it needs the fan working hard so the CPU doesn't burn up.

Install LM-Sensors and hddtemp, and check what the temperatures are inside the box -- especially after playing 15 minutes of HD video.

It might work better with the box open, but I don't know if that is possible.
