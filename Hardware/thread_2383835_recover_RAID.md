### Post by TheFu on 2018-01-30
Welcome to the forums.

For a non-specific question, the only possible answer is - "it depends."
[https://help.ubuntu.com/community/DataRecovery](https://help.ubuntu.com/community/DataRecovery) 
[https://raid.wiki.kernel.org/index.php/RAID_Recovery](https://raid.wiki.kernel.org/index.php/RAID_Recovery)

But this won't help if the RAID uses Windows-specific drivers, or HW RAID that you don't have the exact same model card or MB for.  Hardware dependencies are ... er ... hardware dependencies.

Having backups for all RAID storage is still a good idea and would solve this issue.
