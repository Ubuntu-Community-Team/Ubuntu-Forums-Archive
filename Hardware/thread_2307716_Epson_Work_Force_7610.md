### Post by steven-misshula on 2015-12-28
Well - as promised, I spoke with Tech Support which was as useless as expected.  They only provide Microsoft and Apple support via phone.  The rep tried to be helpful but that is about the most positive feedback I can give.

Solution seems to have been choosing, "plain papers-High" in the PRINTER OPTIONS>Media Type.  

Follow these steps after downloading appropriate drivers, network set up etc.

[LIST=1]
[*]Open System Settings
[*]Printers
[*]Right Click on your WF7610 icon
[*]Select Properties
[*]Select Printer Options
[*]Select Media Type
[*]Choose "plain papers-High"
[LIST]
[*]for me this is the option at the TOP of the list (list is  about 25 choices).
[/LIST]

[*]On your printer settings (on the actual printer), navigate to PAPER SETTINGS
[*]It should be set to "Plain Paper" - if not leave it as such - otherwise, you will wind up with error messages that indicate that the paper type selected does not match Printer Properties (or something to that effect - and your print job will be held up indefinitely).
[/LIST]
