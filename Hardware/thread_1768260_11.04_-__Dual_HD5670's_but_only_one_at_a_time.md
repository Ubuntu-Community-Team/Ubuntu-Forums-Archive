### Post by Oliver_FF on 2011-05-28
Ok so without Xinerama I've managed to get my second display to be at least usable. I had to start up a window manager on it:

```

DISPLAY=:0.1 compiz --replace ccp &

```

I've now got two displays, each connected by the mouse down the middle, each has it's own set of workspaces and each is running its own setup of Compiz.
