### Post by TheFu on 2016-02-27
TL;DR - too much info is just too much.

It isn't about the CPU - it is about the GPU, using the correct GPU drivers and telling mplayer to make use of HW decoding in the GPU. That is only possible for MPEG2 and h.264 videos usually. Any other video codec will have to be decoded in software.

[https://askubuntu.com/questions/101948/command-line-flag-for-mplayer-to-use-hardware-acceleration](https://askubuntu.com/questions/101948/command-line-flag-for-mplayer-to-use-hardware-acceleration) says to enable HW acceleration in the options for mplayer.

* vlc --ffmpeg-hw video.mkv
* mplayer -vo vdpau video.mkv
> 
** place the following in ~/.mplayer/config:
```
[vo.vdpau]
vc=ffmpeg12vdpau,ffwmv3vdpau,ffvc1vdpau,ffh264vdpau,ffodivxvdpau
```
but there are several different ways of doing this. This method allows you to use different video out devices with MPlayer but will give proper -vc values when you specify -vo vdpau from the commandline.  
Of course, mplayer needs to be built with vdpau support (which I don't think is the default from the ubuntu packages). Definitely look at the console output to validate the HW support is working.  720p videos here are using 30-50% of the CPU on an i3-5015U. Couldn't trivially get HW support working for mplayer or smplayer.  I don't "do" 1080 resolution videos.

For nvidia GPUs: [https://launchpad.net/~nvidia-vdpau/+archive/ubuntu/ppa](https://launchpad.net/~nvidia-vdpau/+archive/ubuntu/ppa) might help.
For intel GPUs: [https://askubuntu.com/questions/614320/how-to-enable-hardware-acceleration-of-video-decoding-in-mplayer-with-intel-hd30](https://askubuntu.com/questions/614320/how-to-enable-hardware-acceleration-of-video-decoding-in-mplayer-with-intel-hd30)

On my box here with intel CPU, ** mpv  --vo=opengl-hq --hwdec=vaapi video.mkv ** worked perfectly. 7% CPU use and gorgeous video playback.  Did only "apt-get install mpv" and that was it. Worked without logout/in or reboot. Nice.  Dropped those settings into ~/.mpv/config and it worked perfectly by default. Very nice.  Thanks for your question. Got me to fix this for myself too!  Thank you!

sudo add-apt-repository ppa:mc3man/trusty-media
