### Post by amanchesterman on 2009-09-03
I have an **old** laptop (Acer Aspire 1300) which I have recently converted from Windows XP to Linux. I installed Xubuntu (9.04 Jaunty Jackalope) because of memory limitations (only 256 MB RAM) and I'm generally very pleased with it.

One small thing that doesn't work (and is a bit niggling) is volume control via the keyboard. When it was a Windows machine, I could alter playback volume by holding down the Fn (=function) key and pressing the 'page up' or 'page down' keys (the volume symbol is printed on them). That doesn't work now, and the only way I can alter the volume is by clicking on the Mixer button at the top of the screen and dragging the sliders up or down.

(There's a similar key combination that used to alter the screen brightness, and that doesn't work either. But I'm not bothered about that!)

I've searched through some posts on here and can't find any mention of this problem, but I can't be the only person with it? I've no idea how to go about solving it and would be grateful for any pointers. Please bear in mind that I'm not very technical, and I'm new to Linux!

Btw in *Applications > Settings > Keyboard* the keyboard is correctly set to Acer (GB version).

### Post by arcull on 2009-09-03
Hi, if it is just volume that you need, I think you can set it up in settings of kde, just go to system->setting->keyboard->shortcuts (not 100% sure if I typed the path correctly, cause I am on windows pc right now) and find volume shortcut. However if you need some other hotkeys, you should try to install acerhk driver (search for it in synaptic), it should support most models.
