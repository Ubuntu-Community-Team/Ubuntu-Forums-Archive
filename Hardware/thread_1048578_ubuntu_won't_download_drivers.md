### Post by taurus on 2009-01-23
You don't happen to use the Canadian server (living in Canada)?  Go into System -> Administration -> Synaptic Package Manager -> Settings -> Repositories and switch to either one of the US servers or the Main Server.  That should speed it up tremendously.
