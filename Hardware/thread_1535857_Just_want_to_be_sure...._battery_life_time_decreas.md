### Post by mikewhatever on 2010-07-21

### Post by moore.bryan on 2010-07-21
Only writing anecdotally here...

When I first fired-up my Asus EeePC 1005HA with Windows XP, it told me I had seven hours battery life. ### Post by moore.bryan on 2010-07-21
> **bobsta63 said:**
> Wow... That is really good to know then... I was kind of worried but to be honest it sounds like its pretty hit of miss... I think I'll just take the chance :)
Like I wrote, this as just anecdotal... I didn't run any kind of scientific experiment on my nettie.

> Also.... Did you install the netbook remix or standard Ubuntu?
I installed both and used GDM to choose which one I felt like using. Since then, I've moved away from UNE to using GNOME Shell. GNOME Shell is a hit-or-miss thing; some love it, some hate it. I love its simplicity.

> Do you think it's definetly worth installing the netbook remix or it really doesnt matter? - I know that the netbook remix improves the display but changing the menu's slightly but is there anything else that makes it worth installing the netbook remix?
I'd install both. Chances are your hd will be huge, so you don't need to worry about space. Then, play with both and see which one you like better. 

> Will I miss out on anything by not installing the standard version of ubuntu?
This is kind of moot if you install both. I guess the "simplest" way would be to install one and then add the other through Synaptic; i.e., install the desktop Ubuntu version, fire-up Synaptic, and install ubuntu-netbook.
