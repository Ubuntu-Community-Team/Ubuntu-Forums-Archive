### Post by TenPlus1 on 2009-01-18
I booted using the Ubuntu 8.10 live CD and installed Envy and managed to get the mx400 nvidia card to work perfectly...

1. Install Envy
2. Run Envy, select recommended driver
3. Dont reboot, Logout instead
4. Log back in, card active and working.
