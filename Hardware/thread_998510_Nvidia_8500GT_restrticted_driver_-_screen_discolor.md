### Post by polygone on 2008-12-04
OK.  In case anyone else has this issue, this is what I did.  It's not exactly a 'solution', but it will make the purple go away(temporarily).  I am running DVI to HDMI via one cable....to an HDTV.  I had to unplug the cable and plug it back in.  It fixes everything.  Get this, though.  It goes back to purple on login.  So, I can make it go away, but only by getting up and doing that.  So, it seems hardware related, yet a software change(logging out then in) seems to trigger it.
