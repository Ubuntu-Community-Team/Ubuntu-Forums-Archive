### Post by Bringsom on 2012-11-15
Hi, 
I had the same problem (Dell Studio 1735) and here is how I manage to make it work. First I press Fn + Right arrow and then my keyboard backlight is switched on but the keyboard is locked. So then I press the Fn + Esc or the Fn + F1 keys which both put my laptop on sleep mode, then I just switch on (awake) my laptop and the keyboard is unlocked with the backlight working.I have no idea what's going on but it works. So it's not ideal as  if I pres the Fn+ Right arrow again (to adjust or switch off the backlight) it locks the keyboard again but at least it allows me to use the backlight function. . I hope it helps.
Cheers
