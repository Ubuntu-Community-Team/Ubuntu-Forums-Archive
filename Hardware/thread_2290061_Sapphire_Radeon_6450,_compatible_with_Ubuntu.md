### Post by Mark Phelps on 2015-08-09
The Sapphire HD 6450 will work fine in Ubuntu with the default Radeon drivers -- which will be installed automatically as part of the initial setup.

You could later, if you want, choose to install the AMD proprietary drivers, but unless you plan to do intensive 3D gaming, you don't need them.
