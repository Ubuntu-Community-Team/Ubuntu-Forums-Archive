### Post by Fingel on 2008-06-17
Just download the live CD and put it in the laptop, restart. If ubuntu doesnt start up, you need to change the settings in your BIOS. Restart the computer again, and watch for the load up screen for something that says "press esc for setup" or f1 or f9 or something like that. Once you are in the bios screen, go to boot order, and put CD/DVD drive before internal HD. Restart the computer again, and ubuntu should load from the CD.
Hope you like Ubuntu!

### Post by issih on 2008-06-17
To boot from the cd, you need to enter the system's bios during boot, and set the boot order so that the cd is tried ahead of the main hard drive.

Once that is done the system will boot from the cd whenever a bootable cd is present in the drive on restart.

As to whether it is likely to run, how old is old?

Most importantly, what processor, ram and graphics card does the laptop use.

Failing that, tell us the exact model, and we can hunt the details out for you

Hope that helps

### Post by tangibleorange on 2008-06-18
As far as I know, these are the system requirements for Ubuntu to run smoothly (and install from the Live CD):

700MHz processor
384MB RAM
8GB hard drive

If the previous OSes are Windows 98 and Windows 2000, it is possible you will not be able to run Ubuntu at a reasonable speed. You could then try Xubuntu, which required about half the RAM and diskspace:
[http://www.xubuntu.org]("http://www.xubuntu.org")

There is really no requirement difference between 7.10 and 8.04.

Hope that helps!
