### Post by oldfred on 2014-08-21
One the same page you linked to.

[http://www.sysads.co.uk/2014/07/install-nvidia-340-24-driver-ubuntu-14-04/](http://www.sysads.co.uk/2014/07/install-nvidia-340-24-driver-ubuntu-14-04/)

That installs the ppa with the pre-compiled version, so you install with apt-get.
Be sure to completely purge your current nVidia driver. Otherwise you get conflicts.

I have an older nVidia card and the current version in the repository works for me.
