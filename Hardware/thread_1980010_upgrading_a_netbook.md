### Post by lukeiamyourfather on 2012-05-14
Upgrading a netbook is probably not worthwhile. That one has an Atom processor which is typically soldered to the board so that's out. The memory could be upgraded but not by much since it has only one memory slot. The hard disk could be upgraded but would be half the cost of a new netbook. There are barebone notebooks but not netbooks that I'm aware of.

### Post by ahallubuntu on 2012-05-14
Most likely the netbook Atom processor is soldered onto the motherboard and therefore not upgrade-able.  I have an Intel Atom-based motherboard for a desktop computer, and that CPU is soldered on as well.

You can upgrade the RAM most likely (I did on my Acer netbook) if you are not already maxed out.

Netbooks in general are slow.  The newer ones may be faster than the one you have, but it will still be slower than a laptop.  If you want a fast small laptop, look into an ultrabook and/or something with a low power i3 or i5 CPU.
