### Post by jocko on 2010-11-22
I've got a Razer keyboard, that came with a "driver" disk. The "driver" is actually not a driver but a windows program for recording macros and assigning to key combinations, and to make switchable profiles to use different set of macros in different programs.

If you have no use of such functions, the keyboard works just fine without it (even in windows). If you want such functionality in linux, there probably are programs around that can do it...
