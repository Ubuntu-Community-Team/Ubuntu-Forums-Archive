### Post by elyon225 on 2008-07-30
I am running Ubuntu 8.04 Hardy Heron (fully updated as of this posting).  I also have the Microsoft Wireless Comfort Desktop 4000 (wireless keyboard and mouse combo).

The keyboard and mouse work just fine for the basics.  However, the keyboard's function keys do not work.  For example, nothing happens when I press Alt-F2 (to launch the "Run..." dialog) or press Ctrl-Alt-F3 (to change terminals).

Strangely enough, the F1 key works just fine on it's own (loading "Help").  None of the other keys seem to work at all either by themselves or in combinations as noted above.

I have tried selecting several different keyboard models from the "Keyboard Preferences" dialog to no avail; I've even tried the "Microsoft Wireless Multimedia Keyboard" model.

Interestingly enough, the volume keys, play/stop/etc., and the "Mail" and "Web" multimedia keys work as expected.  The "Calculator" button even launches the GNOME calculator :).

Does anyone out there have any ideas how I can fix this;  Google seems to be clueless.

### Post by CyanideSun666 on 2008-08-27
Hello =) I use this same keyboard, and for the f keys, you have to punch the "f lock" button near the calculator key.Maybe you tried it, not sure.With f lock off, on mine, f1 works bringing up help, but the functions you described, did not.Punching the f lock key, the functions you mention, work.Hopefully that helps =) And if you already tried it, and it won't work, sorry I couldn't help more then =/
