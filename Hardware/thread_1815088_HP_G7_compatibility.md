### Post by zero244 on 2011-07-31
I used Windows 7 for about a week, then made restore disks with the HP software.
The problem I had is between HP and Windows they took up 4 primary partitions which wouldn't allow me to create the partitions needed by Ubuntu.
The Diskmanagement software that comes with Windows 7 does almost nothing.
I tried to install a real diskmanagement program but it was 32 bit and wouldn't install.
So I wiped Windows 7 away and installed Ubuntu.

Something you may want to do, there is a little Ubuntu program you can download that will disable the touchpad completely or just when your typing and is very handy.
It is much easier to use one of those very cool nano usb mice than the touchpad.
When I am using the laptop at home I use a regular usb keyboard as well.

Windows 7 is gone and I dont miss it.....I like XP better anyway.
When you first boot Windows 7 up.....it uses about 1.3 gigs of memory just sitting there.
