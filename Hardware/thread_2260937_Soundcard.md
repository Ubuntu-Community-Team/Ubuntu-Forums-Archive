### Post by Mike_Walsh on 2015-01-15
The terminal command

```
uname -a
```

will tell you what version kernel you are running. If your computer is up-to-date, you should have just recently received the 3.13-0.44 kernel in the last few days.

You do not need to enter this command as root (i.e, no "sudo" in front of it).
