### Post by Bucky Ball on 2016-04-06
Try using the cups interface. Open a browser and in the address bar type:

> localhost:631

... then hit 'enter'. Then Administration tab> Manager printers> choose your printer> Maintenance. Any luck?

Ink levels are not reported using the Printer GUI for most Brother printers in Ubuntu so nothing unusual there ...
