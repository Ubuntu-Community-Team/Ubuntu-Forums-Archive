### Post by gdville on 2013-03-05
Not yet, but I have one and would like to try. Found this thread in my research for how to, but was hoping I wouldn't be the first!

UPDATE 6MAR13:
Have successfully USB booted 12.04. Docked, using dual monitors, the touchscreen/ wacom pen doesn't track right but USB kb and mouse work well. Undocked, ubuntu default multi touch seems to work ok.  WiFi works. Volume buttons working OOB. Home button and rotate button not mapped yet. Camera not set up.

UPDATE 10MAR13:
Went ahead with the complete install. Considered setting up as a dual boot and keeping Windoze for contingencies, but figured that since I never used it on my last tablet set up that way (an HP TC1100), that there was no point in using up 50GB of my hard drive for an OS I'll hardly ever use (after I spent half a day unsuccessfullly trying to shrink the partion).  I did keep the 20GB MS recovery partition.  Still getting used to Unity, which seems to run a bit sluggishly so far.
