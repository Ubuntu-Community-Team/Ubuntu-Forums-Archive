### Post by Dark_Stang on 2008-09-03
I can think of a few things...
1. Use some low pressure compressed air to try to clean out dust from inside the laptop.
2. Flash the bios to an updated version. I must warn you though, if this goes wrong you may end up creating a very expensive paper weight.
3. Do you have a lot of desktop effects turned on? Your GPU might be causing the heat.
4. Have you altered your hard drive's power saving features? If it's always running it may run hotter.

### Post by Dark_Stang on 2008-09-04
I really doubt that Ubuntu could cause your touchpad to run hot. It sounds like there may be a short in touchpad. I'm not sure if you've seen a touchpad that's really worn out, but it's basically a big circuit underneath that tracks your finger. If it's still under warranty I'd give Toshiba a call or take it back to the retailer you bought it from to get the touchpad replaced if they have a service center. 

Don't forget to backup your home directory before taking it in if you are going to do that. Some manufacturers will reimage the drives with a factory install of windows so they can put a guarantee behind it.
