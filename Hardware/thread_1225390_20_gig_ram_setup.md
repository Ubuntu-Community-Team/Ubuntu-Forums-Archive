### Post by NoReflex on 2009-07-28
Both Ubuntu 32 bit and 64 bit will recognize 20 GB of RAM (the 32bit version will achieve this with PAE).
Anyway if your CPU is 64 bit it's best to go with Ubuntu 64 since using 32bit with PAE will impose some limitations like max memory accesible by a single process.
