### Post by snowpine on 2009-06-29
Ubuntu is available as a "Live CD." You can test the Live CD with no changes to your computer. This is the best way to tell if all your hardware is supported. Once you are confident it is, you can install. I think you will find Ubuntu has excellent support for Dell hardware. You can also (if you want) set up a "dual boot" so you can choose between Ubuntu and Vista each time you start up; if you can't figure something out in Ubuntu, you can fall back on Windows. Good luck!

### Post by evp55555 on 2009-06-29
This laptop is supported by ubuntu, I have ubuntu on the same model.

however, consider installing ubuntu 8.04 instead of ubuntu 9.04 in case you have an intel graphics card, which the last version is not dealing with it well.

you may have to connect it to the Internet wired in the first time, so ubuntu could download the proprietary firmware to the wireless card (only if it is the dell 1390, or broadcom 43xx wireless card).

I recommend you to move to ubuntu, it will work very well, especially if you do what i said.

if you encounter any problem write here and i would be happy to help.
