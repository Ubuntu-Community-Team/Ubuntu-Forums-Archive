### Post by Bucky Ball on 2015-09-01
From the information you've given it looks the same to me.

If all you want to do with it is web stuff, why not do a [minimal install]("https://help.ubuntu.com/community/Installation/MinimalCD"), install a desktop environment (perhaps xfce4 or lxde) and Firefox, a terminal program like lxterminal, and that's it. That will run like a rocket.
