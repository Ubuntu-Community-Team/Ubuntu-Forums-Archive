### Post by dabl on 2013-05-02
Linux support for printers is shown in the database [**[color=green]here[/color]**](http://www.openprinting.org/printers).  The Epson TM-U295 is unfortunately not listed, meaning no one has given a report on it.  The TM-U220 is listed as a "paperweight".  :(

Another possible source for a linux printer driver is Zedonet/Turboprint.  But I checked their supported Epson printers [**[color=green]here[/color]**](http://zedonet.com/en_p_turboprint_driver.phtml?&mfg=Epson) and they also do not list the TM-U295, so I'm afraid you will not be using Linux to print with that printer.  :(
