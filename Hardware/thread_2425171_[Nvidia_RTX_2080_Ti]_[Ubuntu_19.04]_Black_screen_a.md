### Post by oldfred on 2019-08-21
How did you install the nVidia driver? From ppa which is the recommended way.
If you installed directly from nVidia with .run file, you have to run reinstall or use dpkg to add driver into every new kernel. With ppa that is automatic.

Can you run this, or will it only be from live installer?
       dkms status 

If UEFI fast boot is off, you should be able to press escape key between UEFI screen & grub menu to get grub menu. Then either older kernel or recovery mode may work. If fast boot on, you may be able to "cold" boot from full power down, not "warm" reboot.

### Post by CatKiller on 2019-08-21
It's pretty clear that the nvidia driver is failing to load. There are a number of things that can cause that, unfortunately.

The most straightforward way to fix issues like these is using the live cd: mount your hard drive and chroot into it. That way you have all the support structure you need, crucially including a web browser.

You appear to be using the 18.04 kernel, rather than the HWE branch that's used for the point releases, or the 19.04 kernel. That's odd, and may be the source of your problem. The graphics drivers PPA doesn't have a particularly robust testing regime: lots of failures get let through that affect people on older kernels, simply because they don't test those configurations.

I would start by booting the live image and running the chroot. I'd have a look at the logs that the error message suggests, but it's probably just going to say that it couldn't load the nvidia kernel module. Then I'd probably run an upgrade to get a more appropriate kernel version: you may be missing the linux-generic package that would have upgraded the kernel when you did your upgrade to 19.04. If that wasn't sufficient, I'd purge the existing nvidia driver and try again, either with the same version or a different one. It's odd that you're on 430.14 with the nvidia driver rather than 430.40, which is in the PPA. Be aware that with that card you can't try driver branches earlier than 410, since that was the branch that had initial support for Turing.
