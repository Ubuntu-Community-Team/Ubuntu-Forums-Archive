### Post by mrintegrity on 2016-01-03
Hi Idzi, 

I realise this is a quite old post so excuse me bumping it wiht an answer. Anyway, i got the fit-pc4 pro barebones and added 8gb ram and a very fast intel ssd. I am using the ubuntu-gnome spin and performance is for the most part acceptable, 1080p BD rips play easilly with no issues. 

The only issue so far is that, strangely, browsing in chrome or firefox is extremely bad.. some sites hang the browser completely with 100% cpu usage (regardless of using hardware accelleration or not). With radeon driver general performance is poor as this chip is not very well supported yet, with fglrx everything is fine except the browser issues (it may be related to how ff / chrome draw the window decorations conflicting with fglrx and gnome-shell).

I am still testing and hopefully get a fix at some point.
