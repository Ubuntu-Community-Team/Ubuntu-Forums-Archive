### Post by D-Sider on 2009-11-08
Honestly, I've had isssues.

I'm not sure if it's caused by my monitor (a 27" Samsung HD TV) or the ATi Card, but I didn't have this issue in 8.10 with a GeForce and the same monitor.

Basically, using the ATI prop driver causes a "Mode Not Supported" monitor error on restart. Sometimes I can fix it by tinkering with Xorg.conf and it works... until I restart again.

I haven't tested the same machine with a GeForce card again, and I haven't tried using an older ATI driver. But that's where I'm at at the moment.  If I was you I'd probably go with nVidia... but the grass is always greener on the other side.
