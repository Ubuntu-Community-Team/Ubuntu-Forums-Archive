### Post by wojox on 2010-08-14
Okay you need to boot off a Live-CD. 

Mount the drive and navigate to /boot/grub.cfg

Then

```
sudo someLongNumber/boot/grub/grub.cfg
```

Look for **splash quiet** and add next to them **nomodeset**

Save and close. Reboot.

When you've logged back on go to

```
gksudo /etc/default/grub
```

Do the same thing. Then run

```
sudo update-grub
```
