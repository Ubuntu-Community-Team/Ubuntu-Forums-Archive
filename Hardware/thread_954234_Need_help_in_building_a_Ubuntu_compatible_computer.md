Sounds like a cool project. First thing I would do is check to see if your Internet Service Provider allows you to run a server. Some ISPs block ports that need to be open in order for a website to be served.

You can also buy hosting and build a website that way. If your primary objective is to learn how to build websites, you can buy some shared hosting and start experimenting right away. If you opt to actually construct a computer it may be a while before you get everything up and running.

I'm building a mythbox (to run Myth TV) myself, and I've been annoyed at the difficulty of finding information about hardware compatibility with Linux. Many hardware manufacturers don't make drivers for Linux, leaving it up to the community to create drivers themselves which can be time consuming.

Do you have an idea of what processor and motherboard you want to get? Likely you're going to be choosing a processor made by either Intel or AMD. Make sure your motherboard is compatible with the processor you select. There are tools that can help with this. If you're buying all-Intel, Intel has a database where you can check the compatibility of processors and motherboards. 

There is a handy database for checking compatibility of printers with Linux: 
[http://www.linux-foundation.org/en/OpenPrinting/Database/DatabaseIntro](http://www.linux-foundation.org/en/OpenPrinting/Database/DatabaseIntro)

and another for scanners:
[http://www.sane-project.org](http://www.sane-project.org)

Also, check out [http://www.linuxhardware.org/](http://www.linuxhardware.org/) for reviews of many hardware components.

Do your homework before you buy a case and power supply. Measure the dimensions of the space where your computer will go in your office, bedroom, of work area. Search Google, Newegg, etc. for reviews before you buy anything. 

When you decide to make a purchase, if it is online, comparison shop so you can get the lowest price.

### Post by dazzled on 2008-10-21
I've built a few low-cost Ubuntu boxes this past year, and organised a couple more that friends built.
 
The simplest, low cost approach I have found is to base the system on an AMD Athlon processor mounted in an Asus motherboard. Some of these boards come with Nvidia graphic chipsets built in, and with a good range of media outlets.  Chose the fastest processor you can afford. Low end Nvidia graphics give few problems, and are excellent for most media.

You don't need so much memory, and 32 bits can't address it anyway. Small web servers and media machines are not usually memory hungry. I wouldn't exceed 2 GB of DDR2 RAM.

Get a case with a decent built-in power supply of about 400w. I have used Coolermaster Centurions, which are well made and well ventilated, with plenty of storage and i/o outlets, but the choice is large. 

PIck whatever SATA hard disk size suits you. Check with your vendor that an IDE DVD drive will work with Ubuntu+the motherboard (I had a problem once)  otherwise use a SATA DVD drive.

These machines have all had Gutsy or Hardy installed without any problems. One runs MythTV. These machines were well within your budget in Australian dollars, so even better if you are quoting $US.
