### Post by curts on 2009-12-12
Check out this link:
[http://forums.linux-foundation.org/read.php?25,6117](http://forums.linux-foundation.org/read.php?25,6117)

I was able to print to my network attached Canon PIXMA MX850 following the advice on this page.  In summary:
- download cups-bjnp-0.5.4.tar.gz,
- using the Synaptic package manager install libcups2-dev and its dependencies,
- build and install cups-bjnp-0.5.4 per the instructions in the README,
- select "new" in the printing control panel and the printer shows up in the network printer list,
- select the printer and then manually choose the ip5300 driver.

Hope this helps!

Curt
