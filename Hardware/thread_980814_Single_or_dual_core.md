### Post by felker2 on 2008-11-13
The Pentium D is a dual-core processor. See [http://en.wikipedia.org/wiki/Pentium_D](http://en.wikipedia.org/wiki/Pentium_D)

Ubuntu will automagically use those two cores. You can see how the cores are used via CLI "less /proc/cpuinfo" and via the GUI "system monitor"
