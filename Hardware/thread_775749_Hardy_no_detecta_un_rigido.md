### Post by faktorqm on 2008-05-02
Buenisimo, te olvidaste del tail,  pero todo bien :D

lo que tendrias que probar es lo siguiente, instalar el ntfs-3g y luego ver si te lo monta, con algun comando.

```
sudo apt-get install ntfs-3g
```
```
sudo mkdir /media/ntfsprueba
```
```
sudo ntfs-3g /dev/sda1 /media/ntfsprueba
```

A partir de aca, si te tira error el ultimo comando, postea la salida a ver que paso. (si se cierra "mal" el windows, te puede tirar error en el journaling, creo que lo podes saltear con fsck pero no recuerdo exactamente el comando, podes probar con:

```
sudo fsck -pny /dev/sda1
```

y volver a hacer el ntfs-3g blabla a ver que paso) Salu2!
