I have a new keyboard and when I boot into Ubuntu, the keymap is not right. The keyboard works fine in winodws, but the model does not appear in the list of models under system > preferences > keyboard > layouts > keyboard model. How do I go about getting this keyboard working?

### Post by LMHmedchem on 2011-12-14
I figured out the issue, this is a mini keyboard and it has a Fn/NumLk combination that turns the right end of the keyboard in to a number pad. Somehow that function was enabled when I logged into Ubuntu. keying Fn+NumLk solved the problem, so it wasn't the keymap.

LMHmedchem
