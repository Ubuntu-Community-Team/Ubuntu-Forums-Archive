### Post by Mark Phelps on 2014-08-18
Unless you really hunt around, any Windows laptop you buy today is most likely to have Win8 on it by default, and then, is also going to be UEFI with Secure Boot -- two things that give Linux installs a lot of problems.  You can read more about UEFI in oldfred's link: [http://ubuntuforums.org/showthread.php?t=2147295]("http://ubuntuforums.org/showthread.php?t=2147295")

Also, any high-end laptop is likely to have dual/hybrid graphics -- and those pose problems with Linux as well.

I notice you mentioned Evolution and LibreOffice, so if you're planning on using Ubuntu as your primary OS, you should consider getting a preloaded Linux laptop -- in those cases, all the hardware works.

### Post by oldfred on 2014-08-23
I had to finally shut down my XP when I got my SSD a couple of years ago. I had said I would stop using XP for several years but had one app that I kept using.

You have to have AHCI on in BIOS for trim to work with SSD. And XP does not work with AHCI.

I was able to install Ubuntu to SSD with AHCI on, turn off AHCI and reboot into XP. But that was the straw on the camels back. Just too much hassle for the one app so I converted to a Linux one that did most things.
