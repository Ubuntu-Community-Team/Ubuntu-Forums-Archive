### Post by MyR on 2011-07-31
I'm guessing "CanonCaptDrv190" is version 1.9, in which case you installed an old version. 2.3 is the latest.  I would give the newest driver a try before giving up.
Link: [http://support-asia.canon-asia.com/contents/ASIA/EN/0900772407.html](http://support-asia.canon-asia.com/contents/ASIA/EN/0900772407.html)

I know printing can be frustrating.  That's why I made a site to help Linux users find a good Linux-compatible printer to buy: [http://linuxdeal.com](http://linuxdeal.com).  You can help by submitting a review of any printers you have used with Linux.

Hope this helps!  Post here if you need assistance installing the driver.  It should be the same process as the older one you installed. You may also want to uninstall the older one first; not sure how to do that though.
