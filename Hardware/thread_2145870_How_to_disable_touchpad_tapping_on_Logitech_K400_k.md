### Post by MarjaE on 2013-05-16
I picked up an external keyboard to help with my arm injuries. I can only type one-handed, so a smaller keyboard seemed better, and the Logitech K400 seemed like the best available option. It comes with a built-in touchpad.

I am currently using Ubuntu 11.04 but planning to switch to Mint in the near future. Under System > Preferences > Mouse, "Enable mouse clicks with touchpad" is off, because I don't have the coordination for it. But the touchpad still registers tapping and accidental touch as mouse clicks.

How can I disable this? Logitech only provides support for Windows and (sometimes) Mac. I know I had to install a special patch to solve a similsart problem with a built-in Alps touchpad.
