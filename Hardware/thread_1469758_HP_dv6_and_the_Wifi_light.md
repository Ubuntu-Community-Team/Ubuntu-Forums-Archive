### Post by iamdigitalman on 2010-05-05
I have the same thing on a DV7-1232nr. I think it flickers to indicate activity. It is odd, as it does not do that in Windows, or in 9.10. I am not worried about the lifespan, as they are LEDs, and are supposed to last forever, but it is rather annoying. The off color is actually more orange for me, and I think I see bits of white in it when it's flickering.

I know some older dells used to use the lights for the Ethernet port to indicate wifi activity. not nearly as annoying, but still odd to see a empty RJ-45 connector with lights lit and flickering.

By the way, this does not happen on my HP nc6400, which has a LED for the wireless on/off button.

Oh, both machines have Ubuntu installed via Wubi through Windows (XP on the nc6400, and 7 on the DV7).

What is really odd is that is a hardware thing, and I did not know the software had access to it. If so, that means it is using an undocumented/unused hook that Windows (with all drivers installed) can't access.

I like the fact it is there, but there should be an option to turn it on and off, in both Ubuntu and Windows.
