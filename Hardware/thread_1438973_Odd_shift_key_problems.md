### Post by underquark on 2010-04-02
If you open a text editor and try pressing both [Shift] keys simultaneously and then type a character then certain ones will appear as normal - "A" etc. - but others will show as if no key had been pressed.  I believe this varies from computer to computer.  It may be, therefore, that your left [Shift] key is stuck (hence the CAPS on login) and that when you press the right [Shift] key it is being interpreted as if both keys were being pressed.

"Stuck" in this case could be real, physical, sugary-coffee-in-the-buttons or virtual. i.e. an ubuntu bug.
