### Post by Frogs Hair on 2013-04-20
Open system> printing on the global menu > Pinter  View Print Que . This displays print status and has command option. If you have an HP printer there is the HPLIP Tool Box in the software center.

### Post by tgalati4 on 2013-04-20
Or open a webpage:  [http://localhost:631](http://localhost:631) if on your machine or substitute the IP address for the remote machine to control.  Requires setting up "Allow remote admin" on the remote machine.
