### Post by phishStix! on 2009-11-23
I have a Gateway P-7801u FX Edition Laptop (May 2000). It has an Intel Wireless WiFi Link 5100 Mini PCI-Express NIC. I notice that Intel gives Windows users the option to use "Throughput Enhancement" with the card (From the device manager). 
 
Is there a way to activate this feature in Linux? 
 
...There is a noticeable difference. When I copy a large file from my NAS upstairs in Windows, there is about a 3.0 MB/s transfer rate, just like in Linux. When I turn on the 'Throughput Enhancement', the transfer rate in Windows jumps up to about 8.0 MB/s.
 
I use 802.11n mode.
