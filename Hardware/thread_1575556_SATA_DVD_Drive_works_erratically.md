### Post by janneym56 on 2010-09-15
I have a dual boot system (Win XP SP2 and Ubuntu 9.04). I recently installed a SATA DVD R/W drive [an Optiarc AD-7241S-0B].

Under Windows the drive works perfectly. I can play both audio CDs and DVD movies using the Windows version of VLC without any problems.  However under Ubuntu the drive performs eratically. Audio CD playback (with VLC and Rhythmbox) is choppy as if the drive is not responding quickly enough to requests for more data. When playing the very same DVD that worked flawlessly under Windows, VLC simply quits about 10 minutes into the movie.

I also have an older IDE (PATA) DVD ROM. When I use this device under Ubuntu, the exact same media (audio CD and DVD movie) play perfectly using VLC.

So the evidence all points to a problem with Ubuntu's driver for the SATA DVD drive. However I have no idea how to diagnose the problem further. Any help would be greatly appreciated. 

[One further data point: I booted from the 9.10 live CD and tested the SATA drive using that version of Ubuntu, with the same results. 9.10 actually did worse than 9.04]

[I also tried using the 10.04.1 live CD and tested the SATA DVD drive using that version of Ubuntu, with similar results. The DVD playback went much further than before but eventually crashed. Audio playback was worse than in 9.04.  So it appears that this problem has not been fixed in the evolution of Ubuntu since 9.04.]

Thanks in advance.
