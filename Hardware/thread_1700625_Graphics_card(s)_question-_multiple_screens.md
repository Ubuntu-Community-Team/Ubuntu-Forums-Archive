### Post by cascade9 on 2011-03-06
The "4 pin circular port" on the FX5200 would be for TV-out, its probably a s-video port. It could be a break-out box attachment as well, depends on the model/brand. 

Its easy to have 2 different screens with nvidia cards. 

Adding the x1300 would just complicate things, lots of playing with the xorg file involoved, and it might not be possible to run the nvidia drivers + the open source ATI/AMD drivers. 

BTW, you dont have to hook up a DVI monitor to the DVI port. You can get DVI-> VGA adapters. But I'd use DVI if you have the option.

### Post by cascade9 on 2011-03-06
> **Antarctica32 said:**
> OK. So I can't hook 2 different screens up to the FX5200 and have them not be clones, right? 

Nope. 

You can easily run 2 screens from the FX5200 and not have them cloned (nvidia-settings-> xdisplay configuration-> 'configuration' button-> twin view). 

You wont be able to set up twinview with 2 mnitors attached. 

As for the low profile thing, you can always just remove the backing plate. Not that it matters really, the FX will do what you want.
