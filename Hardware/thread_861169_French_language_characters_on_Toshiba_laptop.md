I hope this is a suitable forum to post this in.  I am running Ubuntu (newest release) on my Acer desktop and Toshiba laptop.  I've been using the desktop for over a year and whenever I need to type French characters i.e. é, the way to do so is START-' then the letter i.e. e ... (START means the Windows "Start" button on the keyboard).  

However on my laptop, which has both French and English on the keys, this does not work.  Nor can I figure out how to use the special French language keys.  I consulted the laptop's user manual but there is no information about it.

### Post by latanerp on 2008-07-16
Ah ha, I fixed it in Keyboard layout settings.   I changed the "Compose" key to the Left Win-key, and now it works as my desktop does.  I did not ever have to do that on my desktop.
