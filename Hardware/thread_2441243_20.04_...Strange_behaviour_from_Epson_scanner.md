### Post by rubberduck1968 on 2020-05-01
[FONT=arial]Solved.

Further research I have done shows, that new updated Epson drivers don't now work with some models including mine a Epson Perfection V550 Photo and causes [/FONT][COLOR=#000000]'[/COLOR]*[COLOR=#000000]Access denied (insufficient permissions)could not open USB device'[/COLOR]*[FONT=arial] when [PHP]sudo imagescan l[/PHP] or [/FONT][PHP]imagescan l[/PHP] [FONT=arial]is entered into the terminal.  The only way to used the full iscan program is to install the previous drivers for 18.04 making sure all backend dependencies are satisfied and Sane backends drivers for the previous LTS distro.

The iscan uses the sane epkowa: interpreter:001:006 as seen below.

[ATTACH=CONFIG]285732[/ATTACH]

[FONT=arial]If you just want a flat bed scanner and not for photo negative scanning, then use Simple Scan, [/FONT][PHP]sudo apt-get install -y simple-scan[/PHP][FONT=arial] then in your terminal run  [PHP]sudo simple-scan[/PHP] as a root. I have found for scanning documents or photographs this little program is completely satisfactory I found for most needs.[/FONT]

[FONT=arial]For advanced users then Vuescan is the answer on Ubuntu 20.04. Steps needed are to install Epson plugin only, link;[/FONT] [imagescan-bundle-ubuntu-19.10-3.62.0.x64.deb]("http://support.epson.net/linux/en/imagescanv3.php")[FONT=arial] package[FONT=arial] then install Vuescan deb file downloaded from the Vuescan site. The down side the programme does cost $50.
