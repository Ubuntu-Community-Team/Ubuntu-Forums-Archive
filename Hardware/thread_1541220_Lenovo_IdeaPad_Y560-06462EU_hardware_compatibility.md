### Post by rafadev on 2010-08-31
I have a Lenovo Y560 that I bought straight from them. Exactly that model.

It's a really nice machine, beautiful design, and looks sturdy. The speakers sound REALLY well, except when the music hits a particular frequency and the speaker have some resonance that produces a noise. ATI drivers are not very good. I'd recommend installing the latest version directly from ATI. If you use dual monitor, be aware that cool desktop effects will only work with single monitor (they do work with both using nvidia cards). The network card has some issues with wireless n, some random times I have to disable networking and wireless and re-enable it in order for it to work (not many times though). Webcam and sound card work perfectly.

Hope this is helpful, feel free to ask anything else...
