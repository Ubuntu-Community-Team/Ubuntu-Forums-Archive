### Post by efflandt on 2010-02-07
Dell Inspiron 1545 works out of the box, except for Broadcom STA wireless which can be activated from Hardware Drivers (or bcmwl-kernel-source package from installation CD if you do not have temporary wired network).

I bought one of those for someone and test drove 64-bit Ubuntu 9.10 on it before turning it over.  It worked right off the bat from full installation on USB hard drive that already had Broadcom STA enabled for an older Dell.  But the BCM4312 wireless on the new Dell works much better (stronger signal) than the BCM4311 on the old Dell.
