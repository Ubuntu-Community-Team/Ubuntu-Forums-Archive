### Post by madverb on 2010-02-14
If you are just replacing PSU, RAM and GPU the GPU is the only thing that might cause a problem. If you currently have an ati GPU it shouldn't make much difference.
If you have an nvidia you might run into some problems.
Also you might have more problems with an ati on Linux. I always go for nvidia to be on the safe side, although lots of people seem to be fairly successful with ati cards.
