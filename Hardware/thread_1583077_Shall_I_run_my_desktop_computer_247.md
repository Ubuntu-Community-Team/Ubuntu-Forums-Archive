### Post by lisati on 2010-09-27
I normally leave my main machine switched on 24/7 because I have it set up as a web server. It was built as a "desktop" machine rather than specialised server hardware, and I haven't noticed any problems to date. 

I wouldn't worry too much about your hard drive spinning all the time, unless it actually does: as far as I know, and as already mentioned, most machines these days don't run the hard drive continuously if there is no activity.
