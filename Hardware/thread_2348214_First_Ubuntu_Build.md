### Post by Keith_Helms on 2017-01-02
From hanging out on the forums, my impression is that the following 3 areas give the most setup or compatibility problems:
* UEFI boot installs, specifically dual boot with Windows
* Graphics, particularly dual GPU setups such as Nvidia Optimus
* Some wifi chipsets

Formatting and partitioning hard drives/ssds gets a lot of posts.  Make sure you understand MBR, GPT, and UEFI BEFORE you try and install OSes on the new system.

Basic networking often crops up as a problem, but I get the impression that it's often more of a user doesn't understand how to set up a router thing and less of a linux doesn't support it thing.

Most of the other stuff - processors, memory, mice, keyboards, ethernet - don't seem to generate much in the way of complaints.
