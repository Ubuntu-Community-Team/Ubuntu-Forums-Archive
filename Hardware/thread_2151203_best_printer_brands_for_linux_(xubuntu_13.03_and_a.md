### Post by kurt18947 on 2013-06-04
I've had good luck with Brother.  They do use the command line for installation but do offer examples.  A hint for those new to the command line - copy/paste works just fine on Brother's site ;).  Printer drivers are 32 bit and I've used gdebi to install the printer portion of 32 bit systems.  Installing on 64 bit systems require either following Brother's instructions on their web site or downloading the installer.  The installer is found here:

[http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/faq_prn.html#f00104](http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/faq_prn.html#f00104)

The scanner drivers are available either 32 or 64 bit.  There are a couple issues with scanner installation on 64 bit systems, both are addressed on Brother's Linux web pages.  Brother's ink seems to be a bit less expensive than H.P. IMO.  I get the impression that H.P.s linux install is more polished though I haven't installed an H.P. device in quite some time.
