### Post by heir4c on 2013-11-11
Lubuntu and Xubuntu are the same as Ubuntu. Just the Desktop manager is different. 
Install Ubuntu 13.10 and if you want you can add the LXDE or the XFCE desktop to Ubuntu from the UbuntuSoftwareCenter and than you can choose a "LXDE" or XFCE session in the login screen. Than you have the same interface like Lubuntu or Xubuntu.
ZorinOS is also a good and quick distro (based on Ubuntu): [http://zorin-os.com/](http://zorin-os.com/)
Or: Makulu LInux: [http://www.makululinux.com/](http://www.makululinux.com/)
Or: Madbox: (Based on Ubuntu 12.04) [http://madbox.tuxfamily.org/](http://madbox.tuxfamily.org/)
Or: Bodhi: [http://www.bodhilinux.com/](http://www.bodhilinux.com/)

If you use Multisystem for boot up with a USB. You can add ZorinOS, Madbox, Ubuntu, Lubuntu and Xubuntu, Bodhi on that in ones, to try out all these from one usb. (Makulu linux is not supported, for that use Unetbootin)

Multisystem: [http://www.pendrivelinux.com/multiboot-create-a-multiboot-usb-from-linux/](http://www.pendrivelinux.com/multiboot-create-a-multiboot-usb-from-linux/)
Unetbootin: [http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/) (If you have Ubuntu on a other computer you can install Unetbootin from the UbuntuSoftwareCenter).
