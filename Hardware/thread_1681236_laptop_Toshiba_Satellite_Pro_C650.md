### Post by Trevor 42 on 2011-03-07
I cannot advise specifically on the Pro C650 model but....... I installed Ubuntu 10.10 on our Toshiba Satellite L450 last Friday. 

The were no problems with the install (I actually went and made a cup of tea and had a bath whilst it was installing !). 

Configuring the Wi-Fi was actually very easy - just doubleclick on the icon in the top panel, select your router, enter the WEP / WPA code etc and away you go ! We are using a Netgear (wireless) router to connect to the internet but judging by the ease with which Ubuntu detected several of our neighbour's routers I doubt that other makes would be a problem.

Personally I think Ubuntu runs a lot quicker than Windows 7 ever did on this laptop. I have not encountered any issues with the hardware etc.

In any case - you will be offered the opportunity to run Ubuntu "live" when you boot the CD - select this option and see how it works with your particular laptop's hardware before commiting to a hard disk install.

Hope this is of some interest / use / help.

Good luck !
