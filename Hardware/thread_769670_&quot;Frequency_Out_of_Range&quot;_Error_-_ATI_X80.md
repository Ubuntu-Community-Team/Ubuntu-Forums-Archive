### Post by Yellow Pasque on 2008-04-26
Is your card PCIe or AGP?

First, you should try generating a modeline for your monitor with the cvt tool. For example, if you wanted 1680x1050 at 60Hz:
```
cvt -v 1680 1050 60.0Hz
```
Paste the output into the Monitor section of your /etc/X11/xorg.conf file and delete all other mode lines. (gksudo gedit /etc/X11/xorg.conf) You'll also need to change the underscore to a '@'

If that doesn't work, I'll have you retrieve your X.org log with this procedure: boot with the restricted driver. When it finishes loading and you can't see the screen, press Ctrl+Alt+F1, login as your normal user, and run this:
```
cp /var/log/Xorg.0.log ~/Desktop
```
Now do the sudo dpkg-reconfigure xserver-xorg and restart. The X.org log should be on your desktop. Copy and paste that log here with [ code ][ /code ] tags. Do the same with your /etc/X11/xorg.conf file.
