### Post by gfunkera on 2013-01-13
I am running 12.10 on my Macbook Pro (dual booting with Mountain Lion). My Magic Mouse stopped working, it does not get picked up by the bluetooth preferences. I also tried using blueman, which is another bluetooth manager with no luck.

The mouse used to work and was picked up just fine by the bluetooth preferences prior to me running these commands:

```
sudo rmmod hid_magicmouse
modprobe hid_magicmouse scroll-speed=45 scroll-acceleration=1A
```

I am assuming that I removed the modules for the magicmouse and have no idea how to get them back.

Does anyone know how I can fix this? I really want to use my Magic Mouse again!

---

### Post by geedew on 2013-03-08
You added '1A' to the end. It needs to be just '1'. The code checks for an integer, fails because of the A and doesn't reload the mouse.
