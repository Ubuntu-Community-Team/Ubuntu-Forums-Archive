### Post by TheFu on 2021-08-01
Do other computers work with the same cable and monitor?
If so, then it is probably the video card failing.
If not, then it is probably the cable or monitor with the problem.

If the computer is the issue, try booting into a "Try Ubuntu" environment to see if that fixes the problem or not.  If it does not fix it, we are back to a hardware issue. If it does fix it, then it is likely a software configuration or driver problem.

### Post by Autodave on 2021-08-01
Those cables fail quite often.  Also, the cheaper ones do not have the extra wire in them to communicate the monitor's specs to the PC.  Try another cable.  Short of that, disconnect and reconnect both ends of the cable in case of a possible bad connection.
