### Post by lykeion on 2010-10-05
HP Printers should work very well in Ubuntu.
1. Install HPLIP (Applications > Ubuntu Software Center > search for "hplip" > Install HPLIP (might already be installed)
2. Connect the printer (USB)
3. Run HPSETUP (Alt+F2 > enter "hp-setup") and follow the wizard to setup your printer
