### Post by kgudger on 2012-02-09
I could not find anywhere how to add support for the Cannon ip4920 printer, but did find helpful advice on this forum.  I finally found a .deb file for the PIXMA iP4940 on the canon uk website:  [http://www.canon.co.uk/Support/Consumer_Products/products/printers/InkJet/PIXMA_iP_series/PIXMA_iP4940.aspx?DLtcmuri=tcm:14-863337&page=2&type=download]("http://www.canon.co.uk/Support/Consumer_Products/products/printers/InkJet/PIXMA_iP_series/PIXMA_iP4940.aspx?DLtcmuri=tcm:14-863337&page=2&type=download")
It is a deb for all ip4900 printers (the IJ driver).
Note:  I could not make the install.sh shell work on Ubuntu 10.10, but was able to use the package manager to install the debs.  After that it showed up in CUPS and everything worked great!
