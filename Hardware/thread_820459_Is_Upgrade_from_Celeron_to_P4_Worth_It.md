### Post by Sef on 2008-06-06
> My budget is very tight right now. Would it be worth the upgrade? Does P4 have any built-in virtualization capabilities? 

Since 2005, P4 has the capability, but not all do.  It would be best to check out the P4 that you want and see if it has virtualization capability.

As to whether you should upgrade or not, it would depend on how much you use Quicken and if you are willing to deal with the slowness.
