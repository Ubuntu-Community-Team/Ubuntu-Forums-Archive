### Post by kurt18947 on 2020-07-05
How did you install the printers? If you used the driverless printing, you might try installing the drivers supplied by Brother. I have an old inkjet MFC-6490CW and HL-3170CDW. The HL-3170 works okay with the driverless printing, the old inkjet does not. I've had very good success using Brother's installer script, others have no had as good luck. If you're using mainstream Ubuntu, you'll have a utility, system-config-printer. I find that useful for checking connectivity issues or installing other drivers.

Regarding the HL Brother Printer. 
Go to Brothers Support and download the two driver files. They will look similar to these &#8211;


Install 1st  hl2270dwlpr-2.1.0-1.i386.deb
Install 2nd  cupswrapperHL2270DW-2.0.4-2.i386.deb


These are for the HL2270DW so find the proper files for your printer.  They have to be installed in the 1, 2, order given. 


Install the printer with the system-config-printer support.
When the printer is is installed go to Properties and in the DEVICE space you will need to paste a string that look like this 


URI: dnssd://Brother%20HL-2270DW%20series._pdl-datastream._tcp.local/  (be sure that every dot dash and space in this string  remains intact or it won't work)


Naturally you will need to change the printer name to match your HL printer. 
Then click the install driver link and browse to 
&#8220;usr/share/ppd &#8220;      and choose the Hl&#8230;...PPD driver that you installed earlier. 


Done You should be printing.
