### Post by drobe011 on 2010-10-02
Just got mine to work today.  Using 10.04.

Go here for iscan: [http://www.avasys.jp/lx-bin2/linux_e/spc/DL2.do](http://www.avasys.jp/lx-bin2/linux_e/spc/DL2.do)

install the iscan-data package first, then the iscan-2.26.0-3 (select your architecture).

After that a menu item is made Graphics->Image Scan! for Linux is created.

I had to reboot for it to work.  It also added a scan to iscan option in GIMP file menu.
