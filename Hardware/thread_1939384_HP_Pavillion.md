### Post by Helen McCall on 2012-03-17
> **cssutto said:**
> I am going to purchase a HP Pavillion.

I plan to run Ubuntu 10.4 dual boot.

Have there been any major conflicts with the Pavillion?

The HP Pavillion computers each have completely different designs using a viariety of onboard graphics adapters from both nVidia and ATI, and using different processors from AMD and Intel and using totally different chipsets etc.

What you need to do is to look up the data on the HP website for the model number you are considering buying, and then check each component listed to see if it is supported.

The good news is that most Pavillions tend to be Linux friendly.

You will need proprietory drivers for things like the graphics. And also if you want to use things like the lightscribe on your dvd drive you will need to look up the latest linux driver for lightscribe.

The only bad news I can give you on Pavillions is that my one only lasted 2.5 years before it suffered major motherboard failure. I've gone back to building my own computers!
