I downloaded "epson-inkjet-printer-escpr_1.2.2-1lsb3.2_i386.deb" expecting that I would get a file of that name. I didn't. Instead, I got a file of about the right size that was named,
"Unconfirmed 13726.crdownload". It's a binary file and I suspect that it is the driver that I am seeking.  However, I have no idea what to do with it.  Suggestions?  Thanks for any input.
