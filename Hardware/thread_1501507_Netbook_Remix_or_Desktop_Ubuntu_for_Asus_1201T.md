the remix is really ideal if the computer is your secondary machine. it offers all the functionality of a normal computer with a different interface. it really uses the screen well. ### Post by myownserver on 2010-06-05
I've been using Ubuntu since 8.04 and on my Netbook, since Netbook 9.04 ( I think ).  I installed Netbook 10.04 and I personally dislike the changes to the menu system and can't seem to find how to change it back to the toolbar or if that's even possible.

I guess for people new to Ubuntu, it wouldn't be that big of a deal unless switching between Desktop/Netbook Remix, but I guess I don't understand why make the versions so different, especially considering many people will have a Desktop PC along with a Netbook.

I've found the changes in Netbook Remix awkward since it's definitely unique and somewhat limiting and restrictive in the fact it's hard to find out how to change it.

Besides that, it DOES however work good performance wise and is very newbie friendly.  I guess the best way to figure out is run it from a live version to see if you like it and if not, go with the Desktop version.
