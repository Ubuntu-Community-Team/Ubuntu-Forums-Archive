### Post by abriano on 2012-08-18
I have been using the Brother HL-4150CDN for a couple of months now, printing from Ubuntu 12.04, 11.10, and 10.10. I agree that the paper tray is on the flimsy side; I make sure to load the paper tray myself.

I set the default color setting to Mono so that it will use the (cheaper) black toner (only) for most pages. I change the setting to Color if I specifically want color. If the printer is set to Auto, color toner will be used even if only black text or diagrams are used. (Brother provides a note with the printer giving this specific advice.) Therefore, unless one is printing with color all the time, I recommend a default toner setting of Mono.

I saw on one forum (not this) a comment that the printer would not duplex in Linux, but that is not true. Duplexing works just fine in Linux. However, if you want to print two copies of a single page when the default setting is Duplex, the two pages are printed to the two sides of one sheet of paper. Once you know about this you can avoid that operation using several remedies, such as by having a blank second page or by changing the setting for that printing to Single-sided.

We're happy with the printer. As a laser printer, it's not for photographs. But otherwise, it's a decent color printer. I would recommend it to others.

I installed Brother's Linux drivers (available from their website) before connecting the printer.

And I use CUPS. Once I learned how to authenticate on CUPS, I have come to really like it. Neither Samba nor NTS is necessary for network printing with CUPS.
