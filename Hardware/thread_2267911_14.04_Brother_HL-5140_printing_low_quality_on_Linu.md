### Post by pdc on 2015-03-08
thanks Punk_a_cat

what about installing the drivers that Brother provide for this printer; and if you use the Driver Install Tool that Brother provide to help you; it should do all the work for you?

If you go here [http://support.brother.com/g/b/downloadend.aspx?c=nz&lang=en&prod=hl5140_all&os=128&dlid=dlf006893_000&flang=4&type3=625](http://support.brother.com/g/b/downloadend.aspx?c=nz&lang=en&prod=hl5140_all&os=128&dlid=dlf006893_000&flang=4&type3=625) and click to download, you will be getting [COLOR="#008000"]linux-brprinter-installer-2.0.0-1.gz[/COLOR] and it you select to **SAVE** when you start the download, it should end up in your [COLOR="#0000FF"]Downloads[/COLOR] folder

if you open a terminal; copy the commands below; line by line; and paste each line into the terminal; hitting ENTER after each paste ............ in a terminal, the keyboard shortcut for paste is control and shift and v (whereas in most programmes it is control and v ............)

```
cd Downloads
```

```
gunzip linux-brprinter-installer-2.0.0-1.gz
```

```
sudo bash linux-brprinter-installer-2.0.0-1 HL-5140
```

..........so that should run the install script and get the drivers installed; 

.........if you are not happy with the existing driver you have, why not delete it before you carry out the above? If you want to keep it as a comparison, why not rename it slightly so it is clear which driver is which: eg rename it Brother_Foo as it uses the foomatic driver 

if you click on this link [http://localhost:631/printers/](http://localhost:631/printers/) it opens CUPS, printer admin, in another way ......... column 4 lists Make & Model so you can see which driver is installed .............
