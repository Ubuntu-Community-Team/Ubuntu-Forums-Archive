Es fácil, bajás wvdial
```
$sudo apt-get install wvdial
```
Después creás en tu home el archivo .wvdial.conf (así, con el punto adelante) y le ponés lo siguiente:
```
[Dialer Defaults]
Modem = /dev/ttyUSB0
ISDN = off
Modem Type = Analog Modem
Baud = 460800
Init = ATZ
Init2 = ATQ0 V1 E1 S0=0 &C1 &D2 +FCLASS=0
Init3 = 
Init4 = 
Init5 = 
Init6 = 
Init7 = 
Init8 = 
Init9 = 
Phone = *99#
Phone1 = 
Phone2 = 
Phone3 = 
Phone4 = 
Dial Prefix = 
Dial Attempts = 1
Dial Command = ATM1L3DT
Ask Password = off
Password = internet
Username = internet
Auto Reconnect = on
Abort on Busy = on
Carrier Check = off
Check Def Route = on
Abort on No Dialtone = on
Stupid Mode = on
Idle Seconds = 0
Auto DNS = off
;Do NOT edit this file by hand!
```
Guardas  y cerras.
Después hacés un
```
$wvdial Defaults
```
Y ya estás conectado.
Si en una de esas no podés navegar es porque toma mal los DNS, entónces haces
```
$sudo gedit /etc/resolv.conf
```
Borrás todo lo que haya y ponés
```
nameserver 208.67.222.222
nameserver 208.67.220.220
```
Guardas y cerras y listo, ya podés navegar.
Abrazo.
