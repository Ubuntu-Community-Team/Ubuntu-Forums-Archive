### Post by vxj9219 on 2009-09-06
Has anyone encountered issues with running ubuntu 9.04 on an acer aspire 4530 laptop? I installed ubuntu 9.04 (64 bit, then tried 32 bit, then later tried kubuntu 64 bit) on my acer aspire 4530 (amd athlon 64, nvidia 9100M G, 3 GB ram) and as I use the system it gets progressively slower and slower. Every window takes a long time to fully load / open; scrolling on a page moves frame-by-frame like it is an old system. the mouse pointer's movement is not smooth either. I tried searching online and could not find an answer. I have used ubuntu in the past on an inspiron 6000 and have not any problems. windows vista ultimate 64 bit works fine on this laptop. can someone please help? thanks.

---

### Post by vxj9219 on 2009-09-07
Resolved it by doing the following:
1. Refreshed synaptic package manager
2. Activated the nvidia driver from "hardware drivers". Step 1 had to be done to even see the nvidia drivers 177 and 180 in the list.
3. Restarted laptop
