### Post by I&#9829;HTML5 on 2011-02-25
It is true; on boot there is a key that you can push (usually esc or f12) that brings up a menu listing the various devices you can boot from (hard disk, CD drive, network, etc.). You can then select the CD drive and hit enter and Presto!, you have a working Linux live cd.

Although, at least on my computer, it boots CD drive by default. If it can't boot CD, then it tries HDD, then USB floppy, then network.
