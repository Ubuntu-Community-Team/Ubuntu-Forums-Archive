### Post by tgalati4 on 2013-09-20
Some Calcomp plotters used  Zenographics drivers.  The only one that I know of is similar to an HP Laserjet 1000, that used the foo2zjs driver.  Since yours is a Color Calcomp, it might only drive a monochrome mode.  Otherwise, dual boot into Windows and use Windows to print.

It's worth a shot:  [http://www.stchman.com/foo2zjs.html](http://www.stchman.com/foo2zjs.html)
