### Post by Rhubarb on 2010-04-06
If you can boot up an Ubuntu 64bit live cd / usb, then you've got a 64bit compatible system.

I'm very very sure that all core2duos are 64bit compatible.

So your system is obviously 64bit ready, so install 64bit Ubuntu for some fun :)

Just beware that Ubuntu 10.04 hasn't been formally released yet, so in its present beta state things may break and go horribly wrong.  Make sure your important data is regularly backed-up just in case.
