### Post by Mark Phelps on 2010-04-01
Unless you're a seasoned, experienced Ubuntu user, you should not be messing with Beta versions of the OS.  These versions are released primarily for testing feedback.  They are known to be buggy and unstable.

If you want to continue messing with the beta versions, please go to the following forum for information: Development & Programming, Lucid Lynx Testing.  You'll find lots of threads there dealing with beta-related issues.
