### Post by crip720 on 2020-03-24
Would think this might be more of a hardware problem.  What speakers are you using(internal/external) and can you try either headphones/earphones or external speakers.  If using external try playing with cable.  Should be a sound setting in system settings, but usually can only change volume/balance and set which speaker to use.  The test sound is for balance, but try it and see if sounds right.
