### Post by Naggobot on 2010-04-04
May be a stupid question but have you tried the 

System->Administration->Hardware Drivers

Alternatively

System->Administration->Synaptic Package Manager

search for ati. In repository there is on open source driver for some ATI cards you need to check out the compatibility first. If this was not installed during system setup it probably is not compatible. I have old ATI card (do not remember what) and I had open source driver ATI installed during the system setup. 

I am not sure of the package but I suspect it is xserver-org-video-ati. 

Alternatively I think that ATI has some proprietary driver for some models. Try

[http://support.amd.com/us/gpudownload/Pages/index.aspx](http://support.amd.com/us/gpudownload/Pages/index.aspx)

and see if there are any instructions. 

Can someone say what specific steps are needed to install the proprietary driver?
