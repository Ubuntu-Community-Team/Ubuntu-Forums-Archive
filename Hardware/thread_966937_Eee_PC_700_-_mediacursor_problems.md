### Post by tghe-retford on 2008-11-01
I managed to install Ubuntu 8.10 on my Girlfriends 2G Surf using the alternative install with a USB pendrive. It works very well with the generic kernel after a few tweaks but my Girlfriend has noticed a few problems.

a) Browsing the web or playing simple games are fine, but if you watch a flash animation or video, listen to music or play on Gens every four seconds or so the video/graphics and sound will stall and stutter for a split second. Playback and Gens runs fine except for these stalls. The system is using ALSA for sound playback. I have checked Gnome's system monitor to see if a rogue process is hogging the resources but I can't see anything untoward happening.

b) Using Gens 2.15.5, when you use it on fullscreen there is a black cursor (the default X11 cursor) which stays in the middle of the screen and doesn't respond to movement. In windowed mode, the cursor can move freely. It's strange because I installed Gens for my girlfriend on the default Xandros install and this problem did not occur.

Can anyone suggest if there is any way to fix these problems. I have had a look around Google but to no avail. I've asked on the Eee PC user forums as well to no avail so far.

Thanks in advance.

### Post by EndOn9 on 2009-01-11
Answering b), (probably a bit late, but in case anyone else finds this thread having the same problem):

I had this on my eee too using zsnes, I found if I set the screen resolution to 800x480 (the highest the screen supports and what I was using) the cursor would no longer appear.
