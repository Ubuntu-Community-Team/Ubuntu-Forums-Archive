The Gateway EC1436u is an 11" netbook/notebook that is fairly similar to the Dell Inspiron 11z.  This version has an Intel ULV dual-core SU2300 processor, Intel 4500MHD integrated video, Intel WiFi Link 1000 wireless, 1366x768 display, SATA hard drive, etc.  I don't know the origins of the sound card.  Price is $399.

If purchased, I would wipe the Windows install in favor of Ubuntu 9.10.

Gateway is such a small player these days, there is very little info about Linux compatibility on the Web.

Does any know, or even have a feeling, as to whether there would be any obstacles to installing Ubuntu on this computer?  If not for any possible sound issues, the other hardware seems like it would be fine.

Any thoughts would be appreciated.
