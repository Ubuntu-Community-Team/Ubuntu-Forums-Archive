### Post by gradinaruvasile on 2009-08-27
If it works in 75Hz it means the panel supports it. If u set higher than supported settings LCDs usually wont even start, only send a message that resolution/refresh rate x is not supported.

Another thing would be pressing the auto calibration button while in 60 Hz mode, it usually helps in these situations.
