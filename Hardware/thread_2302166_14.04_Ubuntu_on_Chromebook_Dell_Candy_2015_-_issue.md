Hi

Started posting in the wrong thread so starting new one here.

Steps I have take with my Chromebook Dell Candy 2015:

1) Unscrew the back cover
2) Unscrew WP screw
3) Boot into developers mode
4) Use [https://johnlewis.ie/custom-chromebo.../rom-download/](https://johnlewis.ie/custom-chromebo.../rom-download/) to override BIOS
5) Screw WP screw back
6) Install USB 14.04 from USB stick

Feedback to date:
A) Everything works out of the box APART from sound
B) Battery works very very well I would probably judge it at about 8h (not measured) - it displays 10h when fully charged
C) Touchpad needs some getting use to as it does not have buttons. Right hand area which supposedly triggers the standard right click only works from time to time
D) One major disadvantage as far as I am concerned is SD card sticks out. So if you want extra storage added and buy SD card it does not go all the way in about 1.5cm sticks out.
E) Function keys F1, F2 etc are combined with Mute, Back, Brightness up/down on Chrombook and on Ubuntu they only work are function keys. I do not know how to active their other features.
