I found it! Very fast. By googling "kde temperature celsius" I found that it is in Regional and Language settings in KDE 4.
It's coherent that units must be set there but the thing I can't understand is that I must change temperature units there by searching for a tiny small change button top left near Local country/origin label to first change my country, then logout and return to see if units change!
