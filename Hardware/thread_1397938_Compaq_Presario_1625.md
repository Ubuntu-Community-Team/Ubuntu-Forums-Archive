### Post by pizza-is-good on 2010-02-03
Just go for it!

Yes, problems will arise, and when they do, please start threads here so that we can all help you out. That is what we are all here for.

You should be careful with the installation if you want to keep any of the data already on the computer. If not, then have fun.

I just recommend that the computer is connected to the internet while you are installing Ubuntu, so that it can get everything that it might need.

Here is a little something that you might run into. Many say, and I do **not **know from experience, that Ubuntu is not really linux, that it is too easy and so forth to be linux.
Ubuntu is very easy to use and is intended to be used without having to learn much about linux/gnu.
The most linux 'thing' if you would of any linux OS is the command line. I have lately been using [this book]("http://linuxcommand.org/tlcl.php") to learn more about it, and let me tell you, it is amazing. What you learn from it can be used in any linux-based OS. It is written for people that know little of linux/gnu, and it gets to some very advanced levels.
Have fun with it.

Good luck, and happy ubunting

~pizza
