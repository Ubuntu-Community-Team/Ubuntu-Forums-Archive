### Post by ghostborg on 2011-02-01
Checkout System>>Administration>>Hardware Drivers, I don't remember what it's called in your version and make sure the video driver is installed and active.

As for the cube I always install the compizconfig-settings-manager
which allows you to tweak the cube desktop to your liking as apposed to System>>Preferences>>Appearance>>Visual Effects Tab which just gives you None,Normal,Extra.

I also install fusion-icon which allows you to reload your window manager. This is useful if say your desktop loads without title bars or other mysterious issues. Once installed it will be located under Applications>>System Tools. You can also have it load at startup but I just launch it if I need it. It will appear as a blue icon in the upper panel and right-click to see the menu.

You can type compiz in the Synaptic Package manager search and see other things like compiz-fusion-plugins-extra...

If you need more help from more knowledgeable people you might want to give more detail like did you originally install the video driver through Ubuntu Hardware provided drivers or did you go to say ati/nvidia and install the latest and greatest packages. What Video card/platform do you have, etc.

Hope this helps.
