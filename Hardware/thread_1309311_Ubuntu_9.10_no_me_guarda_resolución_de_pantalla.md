### Post by Darth Trix on 2009-11-08
Ya se que este Thread esta [SOLVED] pero encontré otra solución para el problema con nvidia-settinegs y xorg.conf

Esta solución fue posteada por el usuario *aljoriz*:
> this assumes that you have install the nvidia driver.

at the terminal type:
sudo rm /etc/X11/xorg.conf
then:
gksu nvida-settings
(write password)
go to X server display configuration adjust to the desired resolution>APPLY>SAVE TO XCONFIG

Now write this on the save X box:
/etc/X11/xorg.conf

press save and you are done ... found this on the web upd8 blog. Hope this helped someone

Traducción:
esto asume que tenes instalado el driver Nvidia.

escribí en la terminal:
después:
gksu nvida-settings
(pone tu password)
anda "X server display configuration" y ajusta a la resolución que quieras >APPLY>SAVE TO X CONFIGURATION FILE

Escribí donde dice guardar:
/etc/X11/xorg.conf
Guardar y listo!
nvidia-settings crea un xorg.conf como el de antes :)

Espero que sirva para alguien que tenga el mismo problema.

PD: Si Karmic no usa xorg.conf ¿qué usa?
