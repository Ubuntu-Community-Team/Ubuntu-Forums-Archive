### Post by Yellow Pasque on 2009-09-30
If you have a current 32-bit install, try enabling PAE: [https://help.ubuntu.com/community/EnablingPAE](https://help.ubuntu.com/community/EnablingPAE)

> Also, a core 2 processor, it can handle 64 bit instructions right?
Yes, but beware that using a 64-bit OS doesn't guarantee that you'll see all of your RAM unless the motherboard/system supports it. Double-check your manufacturer's technincal specs.
