### Post by sea_dawg on 2011-05-02
You will need a way to connect the new HDD while the old is still in your system.

I have successfully used Vantaec's SATA / IDE to USB adapter to connect the HDD from a Thinkpad R61 to a USB port.  The model # I have is CB-ISATAU2.  It is an older model and my no longer be available.

I don't know if you can use Clonezilla across a USB adapter, I've never tried it.
