### Post by Mark Phelps on 2012-08-09
Depends on what you want to do ...

The Radeon HD will produce better graphics and may be able to manage the GPU temps as well.

The Intel will be fine for nearly everything you want to do except 3D acceleration for high-end gaming.

Problem is ... if this is a Hybrid Graphics system (which most are these days), you won't have any way to actually choose which one to use.  That is done by the system automatically.

And, Hybrid Graphics doesn't work well in Linux.
