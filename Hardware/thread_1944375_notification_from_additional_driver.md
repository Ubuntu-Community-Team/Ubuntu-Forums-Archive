### Post by delsus on 2012-03-21
You don't need the driver to complete the installation, however if you require 3d (for games) or any desktop effects you will need it.

However if you have hybrid graphics (graphics switchable between 2 graphics controllers) using fglrx can cause boot problems and HP will not give us the option to disable one graphics controller in the bios.
