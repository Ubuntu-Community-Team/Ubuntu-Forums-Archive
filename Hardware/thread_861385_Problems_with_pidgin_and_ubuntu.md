### Post by mattze on 2008-07-16
icq recently changed their protocol which caused pidgin to stop working. there is an update available now.
running ```
sudo apt-get update; sudo apt-get upgrade
``` should bring your system up to date.

there is also a somewhat unclean fix here: [http://ubuntuforums.org/showthread.php?p=5308651](http://ubuntuforums.org/showthread.php?p=5308651)
