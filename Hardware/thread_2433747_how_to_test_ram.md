### Post by guiverc on 2019-12-25
If you boot a Ubuntu ISO, there is a * TEST MEMORY* option, and is what I'd use.  I just booted a handy Ubuntu 20.04 LTS option (yesterday's daily) & second from bottom is the TEST MEMORY option (below 'Check disc for defects' and above "Boot from first hard disk").  How the menu appears can vary on your hardware (I had to press a key to see the menu) OR the version (or flavor) of Ubuntu you use; but most have the test memory option.

I would boot INSTALL media (ie. 'live' environment) and do it from there; I'd also use the '*check disc for defects*' option to ensure it's valid media (problems occur, you don't want false reports) first, but that's me.

fyi: * The version I mention is only example; used only because the thumb-drive sitting on this keyboard had Ubuntu 20.04 LTS on it. *
