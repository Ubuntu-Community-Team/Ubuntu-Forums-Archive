Normalmente lo que trae es CD es un asistente de configuracion basica, como para que cualquier "ama de casa" pueda poner en marcha y utilizar el equipo.

Tambien se ha convertido en un standard que estos aparatos tengan una interface web para su administracion y configuracion.

Si lees, que seguramente esta en formato PDF, el manual que esta en el CD vas a saber como hacer para entrar al equipo y configurarlo completamente.

Ovbiamente para esto no necesitas Wine, solo un lector de PDF y un navegador web.
