### Post by N00b-un-2 on 2011-12-16
VIA processors are severely lacking in power even compared to Atom processors with similar clock speeds.  If you want to use Ubuntu I would recommend a light version like Xubuntu (light) or Lubuntu (even lighter).

That being said, VIA chips are practically designed to run mini OSes like MeeGo/Moblin, Android, Jolicloud, etc.  I would check out Android X86 project (because I'm an avid Android hacker :wink:).  I think for a device like yours, you'd be much more satisfied with a less feature full OS than a full fledged Linux distro.  Not that you can't accomplish amazing things with Android!

Don't worry, the different 'buntus are the same operating system, they just vary in the desktkop environment or "skin," and you can customize the desktop environment after install if you like.

If you've used Linux before then maybe you already have a favorite desktop environment (Unity, Gnome, KDE, Xfce, LXDE, etc). If not then you can visit the websites and look at the screenshots, or better yet test-drive a Live USB. 

It is possible to switch desktop environments at a later date, or even to have multiple DE's and choose each time you log in, so don't feel that the K/L/X/Ubuntu decision is a long-term commitment. ;)
