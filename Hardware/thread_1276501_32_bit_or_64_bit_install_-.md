### Post by y0diggity on 2009-09-27
While this may not have any relevance, I'll tell you that I went ahead and installed Jaunty 64bit on my Dell 1737 laptop a couple days ago and I am thoroughly impressed. I had honestly expected all sorts of problems with hardware and software incompatibilities, but I have not found one issue so far that I haven't been able to overcome with very little work. Webcam, mic, sound, wireless, power management, flash player, just to name a few, all work wonderfully. I've heard about issues with Skype and video, not a problem here. I figured that since I was reinstalling anyway, I might as well go for 64bit to try it out and I don't believe I'll be undoing it. 
32 bit apps generally include their dependencies in the package, so they run fine as well. Give it a shot. I'm not going to tell you that 64bit has made my laptop some screamin' machine, but it's the future of computing and there's no sense putting it off any longer. If you can run it, try it out.
Good luck.
