### Post by 2hot6ft2 on 2010-04-23
You might try moto4lin
> file manager and seem editor for Motorola phones

This application allow to upload/download files from Motorola P2k Phones,
upload/download seem records, doing seem backup, and editing seem using
simple hex editor.
or opensync-plugin-moto
> OpenSync Plugin for syncing with Motorola phones

This is a VERY EXPERIMENTAL plugin for syncing with Motorola phones
using the AT command set. It supports syncing phonebook and calendar
entries (ie. the contact and event object types). It will most likely
mess up the data on your phone and any other members of the sync group,
due to the limited data that can be stored on the phone. Use with caution
and make backups!

SUPPORTED HARDWARE:
Motorola L7 ("SLVR"), RAZR V3x (read only), RAZR V3i (read only),
V635 (read only)
They are both in Synaptic which is where the quoted info. is from.
Just search Synaptic for **motorola**
System > Administration > Synaptic Package Manager

I have a RAZR V3m but I just plug it in to the pc to charge it so I haven't used them. In windows I used to use Motorola Phone Tools and manually set it to be seen as a V3mm so more things would work in the Tools.
Mine has also been flashed to Portuguese firmware to unlock it's full potential. Then I changed the language to English so I could use it. It's been a couple years since I did it so if you want to know how I did it you'll have to go to hacktherazr.com
