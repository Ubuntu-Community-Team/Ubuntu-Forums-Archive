### Post by mIk3_08 on 2021-08-21
Updating the BIOS is very risky thing. It might break your system hardware If you messed up
the update or if the update is corrupted. This is very serious problem.
But if you can still get into the BIOS/UEFI setting; If possible you should get to
complete the update without any interruptions because small interruption may break
your system hardware. So Good Luck.

### Post by tzion on 2021-08-22
Thanks mIk3_08 for your reply

Eventually i found the solutions
1. First make sure you update your drivers (Don't assume that your drivers are updated because your laptop is newly bought)
2. There are few ways to update your drivers:
      2.1. Dual boot windows to install drivers using appropriate software by your vendor like for me is Lenovo Vantage.
      2.2. flash the bootable update CD into USB by following this [way]("https://workaround.org/article/updating-the-bios-on-lenovo-laptops-from-linux-using-a-usb-flash-stick/")
3. for me, i erased windows and installed Ubuntu as main OS, in this case, i suggest you to dual boot windows back to install drivers because i have tried both ways and the second way not worked for me :
[IMG]https://ubuntuforums.org/attachment.php?attachmentid=289029&stc=1[/IMG]
4. After updated all the drivers, in my case Ubuntu should not have the erratic cursor movement issue anymore, but in Windows, the issue still exist, i solve it by following this [way]("https://youtu.be/dYnePF6pBFg").
5. Now all issue have solved, but in my case, touchpad left & right click button not working on Ubuntu, i solve it using this [way]("https://askubuntu.com/questions/1235067/touchpad-stopped-working-20-04").

Hope this help for further comers.
