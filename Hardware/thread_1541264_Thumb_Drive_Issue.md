### Post by Apollo87 on 2010-07-29
Hey,

Most likely your USB drive has been split into two or more partitions.  You should be able to set it back with the program gparted.

Just make sure you change the drive in the program to your USB key.  You wouldn't want to mess with your main hard drive's partitions and break everything.

Also, I'm not too sure about this, but you may need to format everything on the USB key, so back up any data you have on it

Look into that program and if you need some more help with this let me know.
