Ubuntu 8.10 The Down Arrow and Left Arrow Keys don't
repeat (not the keys in the keypad, they're fine, the stand alone arrow keys keys in the inverted "T").  The Left Arrow Repeat is especially missed when moving back and forth on a line correcting spelling in a text document.

Keyboard Setup at System/Preferences/Keyboard 

      Keyboard Model - Generic 105-Key (Intl) PC
      Layout - USA

all "Other Options" are default settings

SUGGESTION:  It would be easier to configure a keyboard if the "Generic" option was listed at the top of the Vendor option list.  i only found it in the alphabetically sorted position by accident, and it is probably by far the most used option.
