### Post by BenAshton24 on 2008-09-26
Firstly install wine, do this by opening terminal: Applications --> accessories --> Terminal: and typing the following commands and then pressing enter
> sudo apt-get update
sudo apt-get install wine
then head over to [http://appdb.winehq.org/](http://appdb.winehq.org/) and search for the application that you want to install. if it has a silver or above average rating then it will probably work alright. then double click on the exe that you want to install and it should start to install like windows.

hope this helps.

### Post by issih on 2008-09-26
Ok...one step at a time.

Linux does not natively support .exe files, those are windows programs...We do often have ways of running them however.

The best solution is usually to find a linux based equivalent of the program you want to run rather than trying to keep the programs you used on windows.

Try opening up the main menu and going into add/remove programs.. In there are hundreds and hundreds of programs, just waiting to be installed (yes thats right you can ADD programs..shocking huh). Search for something that sounds like it might be what you need...click on the tick box and hit apply...Bingo one application is installed.

Just shop around till you find one you like, they are all free. This website will help as it lists possible alternatives for various windows programs:

[http://www.linuxalt.com/](http://www.linuxalt.com/)

If you simply can't find a linux application that can replace your windows application you need to se the add/remove facility to install wine, which allows many windows programs to be run within linux (it is often called an emulator, although technically that is an incorrect description). Once wine is installed you should be able to just double click on an exe, and it will be installed under wine in the menu's. Note that the emulation is not perfect, and not all programs work, and even those that do are sometimes fiddly.

I urge you to try and find linux native apps first...wine should be used only if you can't find another way..not just because you are used to the programs it allows you to use. That being said, wine is actualy pretty impressive, and a lot of stuff does just work nowadays.

Generally I recommend reading some tutorials e.g. :

[https://help.ubuntu.com/](https://help.ubuntu.com/)
[http://www.psychocats.net/ubuntu/](http://www.psychocats.net/ubuntu/)

Anyway, good luck and just ask if you get stuck :)

Hope that helps
