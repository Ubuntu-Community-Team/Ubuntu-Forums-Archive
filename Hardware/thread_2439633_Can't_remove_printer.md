### Post by SeijiSensei on 2020-03-30
Open the file /etc/cups/printers.conf in an editor.  You'll need to be root to edit this.

```
sudo nano /etc/cups/printers.conf
```

Do you see a stanza for the printer you want to remove?  Something like
```

<Printer Brother_HL_3170CDW_series>
stuff
</Printer>

```
Try deleting that.  First you have to stop the CUPS daemon with
```
sudo systemctl stop cups
```
When you're done editing, run that same command substituting "start" for "stop".

> **zkab said:**
> Running Ubuntu 18.04 LTS and have a problem.

How big a problem is this? Does it prevent your printing from any application? What is shown in the print dialogs of applications you use? Please give the outputs of ```

``` and ```

```

>

That's a bug in s-c-p. localhost:631/printers/ shows only local print queues.

/etc/cups/printers.conf shows local print queues only. What you see enumerated is not a local print queue but a printer.

> 
It is only shown in Settings/Devices/Printers or when I try to print something ... then the printer is an option.

You can't, not without crippling your printing system. Why not remove the other entry?

### Post by brian_p on 2020-03-31
Thanks for the outputs.

> **zkab said:**
> ```
forsete@balder:~$ lpstat -a
HP accepting requests since tis 31 mar 2020 10:51:46
forsete@balder:~$ lpstat -l -e
HP permanent ipp://localhost/printers/HP hp:/net/HP_LaserJet_M402dn?hostname=printer.local

```

HP is a print queue established automatically by cups-browsed. HP_LaserJet_M402dn is the actual printer, discovered by CUPS from the printer's Bonjour (DNS-SD) broadcasts.

> 
Attached is print dialogs.
I suspect that the printer I want to delete is the HPePrint ... [https://printer.local./hp/device/info_deviceStatus.html?tab=Home&menu=DevStatus](https://printer.local./hp/device/info_deviceStatus.html?tab=Home&menu=DevStatus)
and I can't print on it only maintain it.

cups-browsed is not required software. You can remove it with ```
systemctl purge cups-browsed
``` and let CUPS handle everything.
