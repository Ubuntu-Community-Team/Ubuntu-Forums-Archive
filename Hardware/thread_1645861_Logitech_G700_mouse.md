### Post by andyvy on 2011-05-12
I just got G700 and to get all the buttons to work you pretty much have to have access to a Windows system with SetPoint (Logitech Driver Software) installed to get it setup.

There are profiles you can save to the mouse, so that when you set it up in Windows under SetPoint the mouse will memorize your settings when you plug it into your Ubuntu system.

For Generic Profile I left everything default except for the number of DPI levels, default was set to 5 ranging from 400 DPI up to 5700 DPI, I changed it to 3 DPI levels, 800 DPI, 2800 DPI and finally 5700 DPI. I left polling rate @ 1000 and disabled "Use OS native drivers for pointer speeds & acceleration".

Under Profiles tab, I deleted Productivity default profile, and modified Gaming profile to have all the thumb / top G buttons to be set to "Generic Button" option. Saved everything, and plugged mouse into Ubuntu. 

Now, the General Profile has DPI switching, Mouse Meter, and General Browsing buttons operational. When you push Change Profile button, it goes to thumb buttons working for forward / back in browser, rest don't really do anything until you're in game. I was able to bind all the other keys to World of Warcraft keybinds as long as you have the mouse in Gaming Profile when entering game everything works.

Friend mentioned that he set macros to the keys under Gaming Profile to F6-F11 keys, and was able to bind skills to F6-F11 in game, makes it a bit easier to read your action bar later. Instead of "Mous....." (Mouse Button G6) it shows "F6".

Best mouse ever.
