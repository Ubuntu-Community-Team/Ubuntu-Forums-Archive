### Post by coffeecat on 2011-01-26
> **Rotafia said:**
> If I follow this instructions [https://help.ubuntu.com/community/HpAllInOne](https://help.ubuntu.com/community/HpAllInOne) when I chose the USB option nothing is being found.

That link seems unnecessarily complicated.

I've checked in the list of drivers that come in a default install of Ubuntu and the driver for the Photosmart B109a is there. Your machine is supported out of the box. You don't have to install anything.

If you connect it via USB and switch it on it should be automatically detected and configured without you having to do anything at all. If not, as gordintoronto says, simply open System > Administration > Printing, click on the Add button (with the printer switched on) and follow the wizard.
