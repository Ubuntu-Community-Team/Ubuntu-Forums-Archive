### Post by CyberAngel on 2013-03-02
I found the solution to the problem eventually!

I just used the command "cat /dev/zero > tempfile" to fill up all of the empty sectors on my partitions.
When the command tried to write to the "Pending Sector" it stopped and smartctl does not report any Current_Pending_Sector now :D
