### Post by weatherman2 on 2016-04-24
I'm using an Acer Travelmate B113 with Ubuntu 12.04 (have booted newer versions - will upgrade to 14.04 soon).  I'm using "regular" Ubuntu distro but have installed Gnome Fallback which is probably a little faster.

This just replaced a nearly identical Acer AO756 which has a slower CPU (Intel Celeron 847 1.1GHZ, B113 has a Celeron 1017u 1.6GHZ).  This is a 11.6" screen.  It came with a 320GB 2.5" laptop hard drive.  I replaced the hard drive with a 500GB Crucial SSD.  Upgraded both the RAM (from 4GB to 16GB) and the wireless card (to an Intel 7260 dual band AC + bluetooth).

Ubuntu compatibility has been excellent.  Keyboard is OK, not amazing (I am a touch typist and that is important).  Build quality is cheap, which is why I replaced the AO756 after a few years, but on the other hand, they are cheap to buy.  The B113 refurbished from Acer was about $180 USD via eBay.  Battery is also removable - I have four batteries for it now, useful only when taking a long overseas flight, otherwise I never carry a spare battery.
