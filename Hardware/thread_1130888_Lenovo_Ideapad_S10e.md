### Post by thedigger on 2009-04-26
Hi,

Took delivery of my S10e yesterday. It came with XP, so first steps were:

1: Install anti virus
2: Install firefox
3: Repartition the hard disk
4: Install Ubuntu netbook remix

I only used steps 1-3 because my wife uses an iphone which needs itunes... it doesn't seem to be compatible with gtkpod.

Netbook remix is better for smaller screens as it doesn't use a top and bottom toolbar like a conventional Gnome desktop.

### Post by Hoora on 2009-04-26
I installed dual-boot XP and Ubuntu 9.04 netbook remix on my Lenovo s10e today and everything went very well, not a single problem! Wireless (with WPA2) and everything ok. I'm not a fan of the darkish theme on it and not shure if I'm gonna stick with the "favorites" etc. thingy on the desktop but it's only a matter of customization now. Performance is very good and the boot time is extremely fast. 

Btw, I partitioned my 80GB disk as following: 
40GB XP, NTFS 
10GB Ubuntu / EXT4 
1GB SWAP 
22GB for files, FAT32, prolly gonna reformat to NTFS 
2GB Lenovo system recovery FAT32

On overall, very happy with the Jaunty!
