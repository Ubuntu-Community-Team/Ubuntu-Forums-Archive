Just an FYI-

I have been attempting to get the ALC268 sound running on my ACER 5420-5038 running.  It was a little tricky but I got it working using a few old threads here.

I essentially followed the instructions detailed by vigyani in this thread:

[http://ubuntuforums.org/showthread.php?t=558069](http://ubuntuforums.org/showthread.php?t=558069)

The things I needed to do to get it to install was apt-get G++ and gettext.  I kept getting errors until they were installed.  I hope this helps everyone.

Of course you'll note the file name is different from the realtek site.  Use the new file name with vigyani's instructions.
