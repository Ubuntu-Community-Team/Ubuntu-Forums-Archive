### Post by *Geoffrey* on 2010-06-23
I'm new here too, I also have problems with my logitech headset, I found that It needed to be plugged in before you boot to be detected.

I have mine set to analog mono duplex and it shows in the output, so I set it the same as you have and I get the same result as you,

no output, when I change it back again to analog mono duplex it still didn't show, must be a bug, I use mine with teamspeak2 and

have to set the options in teamspeak to get it to work properly, I set it to " /dev/audio1 " this seems to bypass the sound preferences,

doing it that way I found that it frees up the onboard sound card for system use

hope that of some help
