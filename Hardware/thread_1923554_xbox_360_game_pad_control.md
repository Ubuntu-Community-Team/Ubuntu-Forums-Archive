### Post by jadechessink on 2012-02-10
hey guys, this is for those people who would like to use an xbox 360 controller on ubuntu. i didnt write this, i just found it. this ran on 11.10 for me: 

sudo add-apt-repository ppa:grumbel/ppa
sudo apt-get update
sudo apt-get install xboxdrv
sudo apt-get install xboxdrv-stable

remember to update and upgrade, then restart.
