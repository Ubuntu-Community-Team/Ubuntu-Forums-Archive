### Post by h.f.raven on 2014-09-22
Been browsing through some builds lately myself, seems like you have some good stuff picked out. Ubuntu is pretty wireless-friendly, though I always find it more challenging to setup wireless (make sure you have the right card/usb adapter) than it's worth if you're stationary.

Some of the AMD FX Vishera processors have excellent performance for the money. Not quite as fast as an i7 or some of the high end i5 CPUs, but they are considerably more affordable. I would personally split the hard drive between a solid state and a normal hard drive, though that's because I'm always worried about the read/write lifespan on solid state drives. You could get more (and more affordable) storage that way too.
