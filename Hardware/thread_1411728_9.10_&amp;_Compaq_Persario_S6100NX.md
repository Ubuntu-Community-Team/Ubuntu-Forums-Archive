### Post by williamts99 on 2010-05-10
I just installed 10.04 onto a s6100NX with 512MB which is the min that I recommend(original configuration for this model is 256MB) for ram.

The only issue that I have is on shutdown it doesn't turn off all the way, I have to push the power button once Ubuntu has shut down.

If you only have 256 you should use the text installer [http://www.ubuntu.com/getubuntu/downloadmirrors#alternate](http://www.ubuntu.com/getubuntu/downloadmirrors#alternate)

Or if you want to stick with 256MB of ram, you might want to check out Xubuntu which is a lightweight derivative of Ubuntu


Also, you should make sure that you have the latest BIOS if any of the issues affect you.
