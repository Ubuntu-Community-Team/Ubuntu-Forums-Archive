### Post by coffeecat on 2008-07-14
Dell sell laptops with Ubuntu pre-installed, so if you go for the right model you'll be pretty sure of it being compatible with Ubuntu. If you want Windows XP, you could buy a model that they sell with Ubuntu, but get it with Windows and then install Ubuntu yourself as a dual-boot. I don't know whether Dell do dual-boots or not.

And I don't know whether Microsoft is still letting them sell machines with Windows XP, but I'm sure that they can sell you a Vista machine together with the "upgrade" (cough, cough :wink:) to XP.
