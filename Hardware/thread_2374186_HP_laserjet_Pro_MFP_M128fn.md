### Post by Bucky Ball on 2017-10-13
Welcome. Have you got HPLip installed? It may be by default, but check in 'Software' or whatever package manager you're using. If not installed ... 

install HPLip> reboot> open 'Printers'> remove any printers you've added and click 'Add Printer'> when you get to the section to install a driver, select the appropriate one for your machine and try printing again.

Haven't used a HP printer in awhile, but most work with some cajoling. You do need HPLip installed, though.
