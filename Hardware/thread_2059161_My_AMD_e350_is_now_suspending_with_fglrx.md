My Aspire One AO722 suspend is working with AMD latest fglrx driver!

I think the APU is a  E-350

The version that I used was 9.0 (follow the link)
BTW - if this post is over a month old you might want to just go to there website, this driver will probably be superseded 
[http://support.amd.com/us/gpudownload/embedded/Pages/embedded_linux.aspx](http://support.amd.com/us/gpudownload/embedded/Pages/embedded_linux.aspx)

This driver is for embedded devices but works with the e350 chip set.
