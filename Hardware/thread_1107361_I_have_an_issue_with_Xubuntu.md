### Post by chessnerd on 2009-04-04
Are you installing using the live CD (it loads into a GUI interface and then you double click on an install icon) or the alternate CD (it loads into a DOS-style interface)? If your computer has low RAM you might have to use an alternate CD to install (the alt CD only needs 64MB of RAM to install). Go back to [http://xubuntu.org/get](http://xubuntu.org/get) and when you pick your install site, scroll down to alternate CDs and pick either Intel or AMD. Hope this helps.
