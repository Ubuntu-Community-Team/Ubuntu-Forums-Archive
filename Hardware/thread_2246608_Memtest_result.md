### Post by tgalati4 on 2014-10-02
Try cleaning the machine and moving the RAM sticks around.  Try declocking the RAM in BIOS, add wait states.  It's possible that the errors will go away.  If so, then it could be a motherboard issue--problems with capacitors and clock timing stability.  If you get the same errors in the same locations with heavy declocking, then you can be reasonably sure that one of your RAM sticks is bad.  

Run the test with one stick at a time.  I would replace the RAM in matched sets, so that you have all the same size and production run.  Mixing RAM can cause subtle issues.
