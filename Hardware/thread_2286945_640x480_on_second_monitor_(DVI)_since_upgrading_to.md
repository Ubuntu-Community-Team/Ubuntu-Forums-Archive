### Post by efflandt on 2015-07-15
It looks like you manually installed NVIDIA-Linux-x86_64-352.21.bin without purging nvidia-current instead of using Ubuntu packages. If you use Ubuntu packages there is no need to do anything about nouveau and everything is coordinated and updates automatically when kernels update. For example I am using **nvidia-352** package (same 352.21) from xorg-edgers ppa.

Years ago I gave my old 19" 1280x1024 monitor to my nephews with an old PC w/Qimo for Kids (version of Ubuntu with learning games), so I cannot test that on DVI as a dual setup with my 1080p HDTV. Since DVI is digital (unless using an adapter to VGA) it should be able to communicate its resolution to the video card.

What does **xrandr** show for DVI-D-0?
