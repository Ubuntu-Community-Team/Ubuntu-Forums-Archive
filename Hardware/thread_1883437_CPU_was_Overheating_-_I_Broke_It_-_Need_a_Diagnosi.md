### Post by Mark Phelps on 2011-11-22
Burn marks on the top are a strong indication that the CPU itself is damaged and if that is the case, you'll likely get no response when you try to turn on the PC.  If the board has an LED, that will probably light up, but you'll get no fans running.

I had an old ASUS motherboard that used the same chip (939 series?).  Board stopped working.  Was unable to find a replacement CPU anywhere.

If you can find one, it's NOT going to be returnable -- CPUs generally are not.  So, you can try replacing the CPU, but with that much damage, I wouldn't be surprised if some trace lines got burnt through as well -- rendering the board useless.
