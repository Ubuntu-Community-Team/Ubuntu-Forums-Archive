### Post by Rednic on 2012-07-24
> **tacgiaviet said:**
> what is GDDR5 ? sorry i'm newbie :(
Hi, see this, [http://en.wikipedia.org/wiki/GDDR5](http://en.wikipedia.org/wiki/GDDR5)
It is a type of high performance DRAM graphics card memory designed for computer applications requiring high bandwidth. Like its predecessor, GDDR4, GDDR5 is based on DDR3 SDRAM memory which has double the data lines compared to DDR2 SDRAM, but GDDR5 also has 8-bit wide prefetch buffers similar to GDDR4.
