### Post by kurja on 2014-10-16
Wacom tablets generally do work well in Ubuntu. I have a Wacom pen and touch and I use it in Gimp, the device is autodetected and works 'out of the box'. The wacom tablet settings gui in Ubuntu system settings doesn't allow button configuration though, so I needed to script that using xsetwacom. Having done that, both pen tips, touch, buttons and pressure sensitivity work as expected in Gimp.
