### Post by farleylai on 2010-05-20
I've bought this machine for several months.

HPLIP works well except that it doesn't support to receive fax on Linux.
BTW, the open source USB/IP project is not mature enough to offer production quality in daily use cases while commercial implementations require a normal user to build the kernel driver module and not every client platform is supported for free.
