### Post by emtdan on 2011-02-20
Thanks for the help everyone!

Haha -
Anyway, I fixed the issue.  If anyone else has the problem - which I am sure some do - the fix is a bios update.  Since the problem occurred both in windows and linux I knew it was a bios issue.  I have a thinkpad so I went to lenovo's site, downloaded the latest bios and bam - everything works.

### Post by pze7 on 2011-04-09
I just had the exact same problem with my thinkpad edge 15. The bios update from Lenovo worked, and all keys have become responsive again.
