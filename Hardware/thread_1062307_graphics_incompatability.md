### Post by Yashiro on 2009-02-06
Do 
sudo gedit /etc/X11/xorg.conf

> Section "Device"
	Identifier  "aticonfig-Device[0]-0"
	Driver      "fglrx"
	Option	    "UseFastTLS" "1"
	Option	    "VideoOverlay" "off"
	Option	    "OpenGLOverlay" "on"
	Option	    "TexturedVideoSync" "on"
	Option	    "TexturedVideo" "On"
	Option	    "Textured2D" "on"
**	Option	    "XAANoOffscreenPixmaps" "true"**
EndSection

Add the bolded line into your device section. Save and log out. Should be fixed. You can also try adding the FastTLS option too. The rest won't necessarily apply to your issue.

While you're there, report back on what your Driver line says.

### Post by Yashiro on 2009-02-06
[http://wiki.cchtml.com/index.php/](http://wiki.cchtml.com/index.php/)

Follow the guides there. I'd probably read through the Hardy guide and make sure you have ran 
*sudo apt-get install build-essential cdbs fakeroot ia32-libs dh-make debhelper debconf libstdc++5 dkms linux-headers-$(uname -r)* anyways.

Basically:
Make sure your have disabled Desktop effects in the appearance panel.
Download the driver you want from Ati.
Run the command line for your OS. intrepid in your case.
*sh ati-driver-installer-**9-1**-x86.x86_64.run --buildpkg Ubuntu/intrepid*
I'd recommend driver 8.9 or 8.10 as I know they both work.
This creates a few .deb files.
Install these deb files, terminal is faster. See the guide, use tab completion, don't be scared :P
Run aticonfig --initial -f
Reboot. Might need to mess around a little with xorg.conf, then Reboot again.
Job done. Test effects out.


Not quite like Windows, is it? :D
