### Post by Mark Phelps on 2012-04-04
Not to be facetious, but "proprietary" means just that -- no source code available.  So, without that, don't know how any debugging tool is going to do what you want.

You should check with Broadcom and tell them you're willing to invest your time to troublshoot a driver problem.  Maybe they'll do something to help -- like letting you have the source code as long as you sign an NDA (Non-disclosure Agreement).
