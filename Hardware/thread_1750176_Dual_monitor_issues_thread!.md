So I'v noticed that there ar lots of issues with Dual monitors since 11.04 Natty. So why not try and get one big thread to gather what works and what dosnt!

My issue is running two seperate desktops (xservers) with unity or gnome classical with compiz. This is due to compiz breaking dual monitor functionality. This has as far as i know been done on purpose from the compiz dev team for some reason.

I solve this by running Ubuntu-classical (no effects) session from login.

Also for some reason unkown randr wich is xorg tool for multiple monitoprs handling fails to detect both my monitors so Xrandr is for me also usless together with unity and compiz.

I can only hope that developers will put their heads together and figure out whats going on and how to solve this. and I'll happily by em lots of coffe :)

My issue is also been reportet at launchpad so if more penguins have same issue as me the can go there and let the devs know :)
