### Post by hackb0y294 on 2010-04-14
When you boot the live CD, it should show a language selection screen. Select your language and press enter. Then press F4 (for boot options), and select Safe Graphics Mode. This should work if you are using 9.10 or earlier (10.04 doesn't have this option as far as I know).

Hope this helps.
