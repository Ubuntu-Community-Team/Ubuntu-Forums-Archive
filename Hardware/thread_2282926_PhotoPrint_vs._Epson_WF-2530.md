### Post by Mike_Walsh on 2015-06-18
Hi, Terry.

Can you be a wee bit more specific? Which 'pull-down' are you referring to? 

You won't find you can select a different printer that will make your one work; it doesn't happen like that! I assume that you're referring to the Printer Install utility, yes?

Can we have a little bit more information? What your machine is; what CPU, how much RAM.....and what version of Ubuntu are you running? We can no doubt help you with all this (Epson's are not that hard to install), but it helps to know what we're dealing with.

You will need to install the appropriate driver (from Epson) for your printer, and then, you will have two choices; either to install it via the Printers utility, or to install it directly via CUPS (in your browser).
