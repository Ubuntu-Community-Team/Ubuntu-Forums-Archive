**Key Features of Cisco 1721 Modular Access Router** 

Cisco's 1721 is a modular access router. It has a 64MB DRAM which can be expanded up to a maximum of 128MB. It has a plastic chassis and a flash memory of 32MB. The *[Cisco 1721 modular access router ]("http://www.digitaldevicesonline.com/cisco/cisco-routers/cisco-1721-router.php")*has one console and auxillary port each..
