### Post by Claus7 on 2022-09-30
Hello,

I was facing problems with my laptop monitor when I attempted an upgrade, and I found a solution by changing the color profile. I have to admit that I haven't tampered with this option before. The thread where I had the aforementioned problem is here:
[https://ubuntuforums.org/showthread.php?t=2478607](https://ubuntuforums.org/showthread.php?t=2478607)
Usually, I do not remember having any available options in order to change the color profile, yet I think that always only the automatic one was used. At the moment that I had the problem and went there, I saw a pretty descent number of different options, along with the automatic one. Now that I went there again while checking your thread, I once again see that I have no option other that the automatic one. I do not see any pink "hue" on my screen, which at that time had led me to go and change this option.

By hovering on top of the lamp displayed at the right hand side of the selected automatic profile, the message displayed in my case is: Full-screen display not possible with this color profile.
I can see that you have these "lamps" as well. I would advice you, if you have no problems with the colors displayed on your screen, to leave these options as they are, since these I think that are dictated from the manufacturer in order to set how colors will be displayed on screen.
If your would like to change some colors, then you could check out, if you haven't come across it, the night light under Displays. 
Maybe in what you seek, there might be no support in order for these profiles to work. Just some thoughts.

Regards!
