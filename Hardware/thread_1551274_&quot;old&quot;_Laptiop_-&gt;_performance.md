### Post by Fafler on 2010-08-12
Xubuntu is just what you're looking for. But you should look for some more RAM. 256 is just barely usable. It uses 144 pin SO-DIMM PC-100 modules and the maximum size for each module is 256. If you already have a 256 mb module, just get another, but chances are you have 2x 128 mb and need to get 2x 256 for optimal performance.

Useful info about your machine: [http://www.thinkwiki.org/wiki/Category:T20](http://www.thinkwiki.org/wiki/Category:T20)
