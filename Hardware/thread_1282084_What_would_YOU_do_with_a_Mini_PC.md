### Post by bircoe on 2009-10-03
[LEFT]I've just completed a similar project, I've had a Intel D201GLY ITX motherboard sitting around for years... I finally decided to do something with it.

It's purpose is to play music via MPD, 24 hours a day 7 days a week, sound card line out is connected to a 100w PA Amplifier which pipes music to ceiling mounted speakers through my house, controlled via a PHP interface, nice simple reliable, before the last black out it's uptime was 126 days (4.5 months).

Specs:
[Intel D201GLY ITX Motherboard]("http://www.intel.com/support/motherboards/desktop/d201GLY/")
Integrated Intel® Celeron® 215 CPU @ 1.33GHz
512MB DDR2 RAM
[WD Scorpio Blue 250GB IDE 2.5" HDD]("http://www.westerndigital.com/en/products/products.asp?driveid=346")
[PicoPSU 90w 12v DC-DC Power Supply]("http://www.mini-box.com.au/purchase/browse-detail.asp?start=1&menuId=35&laCode=ENGAU&class=four%20four-two&subclass=&productId=115")
60w 240v to 12v Brick Power Supply
[Morex 5677 ITX Case]("http://www.morex.com.tw/products/productdetail.php?fd_id=40")

Software:
Ubuntu 9.04 i386 Desktop - Command Line Install
MPD - Music Player Daemon
LAMP - Apache MySQL PHP
Pitchfork - PHP MPD frontend
Mediatomb - DLNA UPnP Server
[/LEFT]
