### Post by AmerNewbieInDE on 2009-08-14
are you talking usb hard drive or mem stick.. i am running my ubuntu from an external sata usb hard drive and am suprised on how quick it is, i expected things to be alot slower.. but remember, externals are not really made for sustained use, dont forget they create alot of heat. keep that in mind, also, opening and running many programs at the same time will be slower.

when you goto install, i think it is step 7, goto advanced setupand choose the grub loader to be put on the usb also.. 

i set my bios to boot from the usb first, so if i have my external plugged into the pc, i boot automatically to ubuntu, if it is not plugged in, my pc boots to vista...

Ohh... if you are copying a current install from pc, i guess you will also have to change the boot loader.. so it knows where to find the install..
