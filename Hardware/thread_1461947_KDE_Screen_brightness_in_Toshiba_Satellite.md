I have a Toshiba Satellite T135 running dual-boot Win7/(K)Ubuntu 9.10.  For a long time, I could not get the screen brightness to change, either via the Fn hotkeys or via brightness applet.  

The solution in eOgas's post #6 here: [http://ubuntuforums.org/showthread.php?t=1321403](http://ubuntuforums.org/showthread.php?t=1321403) worked for Gnome, but KDE (3.5) does not seem to recognize this fix.  I still cannot change screen brightness in KDE (which is what I use most often).  On boot, the screen is set to the maximum possible brightness, which is way, way too bright. And I do tend to tweak the brightness rather frequently.

Can anyone suggest a way to do brightness changes directly from within KDE?  (And on a related note, is there a way to change the default brightness used on startup?)
