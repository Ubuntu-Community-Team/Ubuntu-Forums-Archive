### Post by jarrah-95 on 2010-03-20
hi 
ubuntu when built properly and used properly can be fast on any machine (i get 5-6 second boot times on a duel 1.6 cpu, with 2 gb of ram, and an intel chipset for my gpu) so i whould suggest that a duel core should be fine although im not sure if you can use a quad core properly with ubuntu yet.
gpu, almost any decent gpu on the market will do but you'll have to check witch ones work better with ubuntu.
also to get snappy boot times and load times some of the performance things in this [http://ubuntuforums.org/showthread.php?t=1209904&highlight=quick+snappy+ubuntu+from+scratch](http://ubuntuforums.org/showthread.php?t=1209904&highlight=quick+snappy+ubuntu+from+scratch) thread will speed it up a lot. the ones i use are prelinking, profiling witch is done at the grub bootloader, and editing the rcS file. these for me made a fast stable system on an old machine so yours should be awesome. hope this helped.
