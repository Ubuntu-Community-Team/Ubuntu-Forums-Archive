### Post by walt.smith1960 on 2009-08-15
I have a MFC7820N. Network printing and scanning. Printer install was a dream, it's in the FooMatic database. Installing the scanner function was a little tricker but still not bad.  The only glitch I found is DO NOT enable ICC color profiles (I think).  I downloaded those packages and broke the scanner function.  Removed the ICC color profiles and the scanner function returned.
