### Post by frankwakeman on 2009-05-31
But is there a sign that the computer is on, like my green power indicator?  If so, things are hopeful.  Look up on the manufacturers' website where to obtain the BIOS on that model of computer and make the BIOS flash disk.  If the machine is on but doesn't reach the BIOS stage the BIOS is corrupted somehow and is simple to fix.

Do you mean you reset the BIOS by removing the CMOS battery for fifteen minutes, or do you merely mean that you had hoped to reset it but couldn't?

If it's a desktop pc and not a laptop, you could also try booting up without the graphics card in, so that the on-board graphics are used, remembering to plug the monitor into the pc's VGA slot and not the card's connector.  If you had reset the BIOS with the battery-removal method I mentioned, the graphics would have been set to onboard rather than the card's graphics, and you might find things start up.  If you can see what you're doing and what's gone wrong finally you can work from there, reinstalling the operating system if need be and then put the graphics card back in.

This might seem like a 'stab in the dark' but is without expense and better to try that first than go on to the next step of replacing blown processor or motherboard.
