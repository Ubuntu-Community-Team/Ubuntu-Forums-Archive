### Post by ahallubuntu on 2012-02-12
Unfortunately, if you do some googling, it appears the DV4 has known problems with Ubuntu and the touchpad in particular.  Some models of computer simply do not work well with Linux - thanks to the manufacturer(s) not due to the Linux community.  Driver support from hardware manufacturers is crucial to allowing things to work in Linux.

If you tried Ubuntu 11.10 and got these problems, I'd ask you to consider Ubuntu 12.04 Alpha which will probably be buggy at this point but is due for final release in a few months.  If that fixes your touchpad problem, perhaps you'd be better off living with the bugs for a few months - or running Linux as a virtual machine (Virtualbox) inside Windows 7 for a while...
