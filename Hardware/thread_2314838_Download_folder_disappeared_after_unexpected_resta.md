### Post by howefield on 2016-02-24
If it really has gone, then simply recreate it.

in a terminal..

```
mkdir ~/Downloads
```

or via Files.. launch Files which should open in your /home folder, right click in the right hand pane and select New Folder.. name it Downloads (with a capital D).
