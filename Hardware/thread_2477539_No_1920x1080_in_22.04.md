### Post by libertyspike138 on 2022-07-29
I have 2 computers here side by side. One is running 20.04 & the other is running 22.04 they are both using nVidia driver 470.129.06. They are both connected to the same monitor via a KVM switch. 20.04 is displaying in 1920x1080 but in 22.04 if I select 1920x1080 the monitor says it is an unsupported mode and it switches back to the next highest option of 1440x900.

---

### Post by libertyspike138 on 2022-07-29
Come to find out that there is no issue with 22.04 in regards to this. The difference is that the 20.04 machine is using DVI-I and when using DVI-I on the 22.04 machine it works. The problem I got into was the fact that I also have MacOS installed and for some reason once you reach the GUI in MacOS it disables the DVI-I port leaving me stuck using the DVI-D port on this new machine. Ultimately I solved the problem by replacing my monitor with one that is a little smaller and it will accept 1920x1080 over DVI-D & DVI-I.
