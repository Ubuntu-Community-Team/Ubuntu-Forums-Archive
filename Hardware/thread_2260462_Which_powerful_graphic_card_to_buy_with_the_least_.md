### Post by Yellow Pasque on 2015-01-12
I consider myself an advanced user, so I wouldn't hesitate to get a Maxwell-based GTX card, even if it took some hacking initially. If I didn't mind an older card, I might look at something in the RadeonHD 6000 series because I like the open-source radeon driver for all but the latest cards.

As for GTX 900-series not working out of the box, Ubuntu 15.04 should fix that (assuming it gets the 3.19 kernel), at least to the point where you can boot to a GUI with the correct resolution using the open-source nouveau driver. Hopefully, Ubuntu will do a better job of making current nvidia proprietary drivers available in the main repos and easily installable (they've been slacking off and forcing users to install from PPA's).
