### Post by japhyr on 2009-05-24
When I first started using linux a year ago, I had an old laptop that needed replacement, so I had nothing to lose when I installed linux.  I am ready to buy a new computer now, and buying new with the intention of installing linux is a very different shopping experience.

I know the basics of what I want, so there is a reasonably small set of laptops to look at.  But the possibility of spending $1000-$1500 on a computer and then find that the graphics don't work well is pretty frustrating.  It seems to be, do a little research to make sure you're not buying a lemon, then buy the computer you like and hope you can configure it to make it work.

This makes me think seriously of something like System 76.  I also really like the idea of not buying a laptop with Windows and contributing to the notion that we all want Windows.

Have other folks had similar experiences where your initial installation of Linux is an easy choice, and then your first new purchase for Linux is more difficult than you imagined?

### Post by vincent orange on 2009-05-24
I've been using linux for more than 10 years now. I've been lucky enough to try hundreds of versions and incarnations of linux flavours. I must say though from my experience I've not had many if any problems with laptops. Many of my problems and head scratching moments have been with desktops machines. 

I have installed linux on several laptops ranging in age and performane. 

Toshiba 1300 (circa 1994)
HP pavillion z1800 (circa 2001)
IBM Thinkpad (circa 2005)
Acer travelmate 4300 (circa 2008)

The slightest obstacles for those were wireless and RAM configuration. But both are well documented and easily solved. significant improvements can be seen with a increase in usable RAM. Swap space is handy, but considering that RAM is running ata  faster pace, not worth the harm in buying more.

System 76 is a great provider in hardware for acquiring a linux machine without the hardware headache. Although if you want to go it alone I would consider either IBM laptops as they were all using supported hardware, Lenovo doesn't seem to officially support linux anymore but you may find that the components used are supported, HP as many laptops are compatible with linux.

now for the links
[http://www.linux-laptop.net/](http://www.linux-laptop.net/)
[http://www.linuxemporium.co.uk/products/laptops/](http://www.linuxemporium.co.uk/products/laptops/)

both are useful in determining if the laptop of your choice can support linux.

Finally if in doubt, burn a LIVE CD of Ubuntu (or whatever) go down to the computer show room and try and get the live cd to run on the machine. Test out all the important functions; wireless, display and etc and see if any issues occur. If it seems that everything works, it is a great indicator that the full installed distro will work too (including the upgrades). Lastly, I would recommend to have a dual boot install. I've been using such a setup on my machine for quite some time and it works great, just make sure you have a HD larger than the standard 80Gb...around 250Gb is better suited.

I hope that helps in come way, happy shopping :)
