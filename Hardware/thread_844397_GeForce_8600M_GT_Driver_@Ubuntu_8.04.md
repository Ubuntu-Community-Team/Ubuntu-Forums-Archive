### Post by thedoorguy on 2008-06-29
To install the EnvyNG package goto Systems>Administration>Synaptic Package Manager and do a search.  The package is in one of the universe or multiuniverse locations. There are three parts to select... mark all three for installaton.

You may have to uninstall the legacy package.  Check out Help on envy for more and probably more useful info.
