Steps to get Wireless WiFi wireless Broadcom 4320 working
1. create linux subdirectory  \wifidrivers
2. download the dell broadcom drivers using windows and extract to \wifidrivers
3. go through the blacklist procedure on the ubuntu how-to be sure to black list all of the b43 files mentioned
4. on the menu Administrator chose wireless setup up at bottom of menu.
5. when asked for the dll file navigate to the \wifidrivers folder also give your computer an ID/name for ssid
6. restart the machine and the blue light should be on then click on upper right signal icon and select wireless connection to use.
