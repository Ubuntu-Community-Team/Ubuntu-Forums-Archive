### Post by kvarley on 2009-03-01
I have the same webcam as you.

To use your webcam you can just install a viewing application called Cheese.

In terminal type:

"sudo apt-get install cheese"

It will install and appear in the "Graphics" section of the Applications menu.

Just plug your cam in and start cheese, should work fine, reply if you need help or something goes wrong.
