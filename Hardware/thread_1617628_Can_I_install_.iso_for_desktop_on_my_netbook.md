### Post by TheDuskrider on 2010-11-09
I'm much more impressed with the Ubuntu Desktop for netbooks than Ubuntu Netbook Edition. It's way, way, way faster. Also, the netbook edition still has some problems displaying things on the smaller screens. You won't notice any screen loss with the desktop edition (and you actually gain a lot of widescreen space). The desktop version has a few problems with displaying some Linux specific programs, but not too bad. Either way, the Desktop version is literally 10x as fast. I've also noticed less screen flicker and text entry problems. I can also access applications in the desktop edition that I can't in the netbook edition.

I'm running an ASUS EeePC 1005PEB Netbook
1.6ghz
1gb RAM (Upgrading to 2gb soon)
250gb HD
