### Post by LepeKaname on 2009-07-01
Hi UNIXkillz, 

There are plenty of models with that description... I usually build my own computer so I'm not familiar with Dell/HP/etc. products. I can suggest you to go to a computer shop and buy a barebone system, which usually is the box and the MB (with integrated network, sound, and video: Nvidia for example), without HDD, DVD or memory. 

Then at the same shop, you can ask for 3GB memory and a 400GB+ HDD. They will know exactly which type of memory is best for that PC and they will recommend you some HDD brands for sure. Also is very common they will install those in that moment. You can select any keyboard, mouse, monitor you want and I can bet you will get a very nice piece for $300-$600 (depends on what did you choose). 

The advantage is that it will come with no OS (which means you are not paying windows and a bunch of friend's software), so its perfect to install Ubuntu there. 

> Also what do u mean by make a list? 

If you want a brand computer then search on Internet for some models with those features (mainly at manufacturer web sites). And make your choices. Create a small list (up to 3 models) and then post it here (I recommend you a new thread as more people will see it). 

That way, we can suggest you which one may be better for you. It will be helpful if you add a brief description of what is your use intention, for example, if you want it to use it as workstation, for gaming, as a server (file server, web server, mail server, database, hosting, etc.), and so on.

Good luck!
