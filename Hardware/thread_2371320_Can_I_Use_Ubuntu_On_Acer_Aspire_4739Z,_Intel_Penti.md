### Post by oldfred on 2017-09-13
I have a similar spec Toshiba but dual core Intel chip with  1.5GB RAM.
With 12.04 the Unity gui was too slow, so I used fallback/flash/back/gnome-panel which has changed names with Ubuntu versions, but is still available as another lighter weight gui.

I have retired system, but recently installed 16.04 just to see if it worked and was surprised that Unity was functional, just a little slow. I do have to only use one larger application with maybe a couple of smaller apps. When I try to load two larger apps it would go to swap and be noticeably slow, screen turns gray for 5 or 10 sec.

Most would suggest one of the lighter weight gui to start with.
       Lightweight flavors
Lubuntu, xubuntu, Ubuntu mate, Budgie
[https://wiki.ubuntu.com/UbuntuFlavors](https://wiki.ubuntu.com/UbuntuFlavors)
