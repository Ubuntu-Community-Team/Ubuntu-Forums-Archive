Just upgraded my system to 9.10 and when all was done, there was a minor glitch with the display. The display config was set to 1920x1200 in X and Nvidia said it was 1600x1050, as it should have been. This caused a panning display, which is nice, but I'd rather just use the display natively. :D

I found the workaround, which is to say YES to the error I got about NVIDIA not supporting all the stuff required, and then adjusting the X config.

But there were several errors. You can see the details here:
[http://www.windyweather.net/wp/2010/01/21/karmic-koala-update/](http://www.windyweather.net/wp/2010/01/21/karmic-koala-update/)

The good news is that All is well with file sharing with my windows, and other Ubuntu systems. Hurray.
And the display is fine now that I fixed the Xconfig using the standard dialog as well as the NVIDIA panel.

No reboots like last time to fix things up. It all just worked.
