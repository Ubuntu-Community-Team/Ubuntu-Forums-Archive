### Post by Fafler on 2010-08-13
It should be okay. Pretty much standard hardware. You don't need 64 bit to use 6 cores, but you do, however, need either a 64 bit kernel or a 32 bit kernel with PAE to use all 4 gb memory. Which you choose is up to you. It doesn't make much real world difference. Downloading and installing the 64 bit version is probably the easiest.
