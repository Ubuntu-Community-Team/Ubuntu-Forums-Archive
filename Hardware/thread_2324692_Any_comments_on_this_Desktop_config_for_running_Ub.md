### Post by houstonbofh on 2016-05-16
I think more memory would give more benefit then the NVMe drive.  Normal SATA3 SSD is fast enough if you get a good one.  (Intel or Samsung)  Also, the Quadro is more expensive then consumer grade, and will not benefit you unless you have software with specific GL needs. (Autocad)

Also, if you are going to bother with spinning disks at all, put in a BluRay burner that will also do DL/DVD.  At the current prices, there is no reason not to, and you may need to burn something big one day.
