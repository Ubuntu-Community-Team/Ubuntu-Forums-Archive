### Post by kurt18947 on 2013-08-03
There are typically two problems with Brother scanners.  If you're using a network connection, I don't know if you'll need to do the 'scan as a normal user over USB connection' though it probably wouldn't hurt.  You will likely have to do the copy some files from one destination to another fix.

Here's one fix:

[http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/faq_scn.html#f00101](http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/faq_scn.html#f00101)

Here's the normal user USB fix:

[http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/instruction_scn1c.html](http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/instruction_scn1c.html)

If you're not aware of it, here are the directions for installing a networked Brother scanner.

[http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/instruction_scn1b.html](http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/instruction_scn1b.html)

I've found with recent Ubuntu releases I don't need to do the 'pre-install' stuff like the earlier releases e.g. 8.04, 8.10
