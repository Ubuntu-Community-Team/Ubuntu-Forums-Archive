### Post by ugriffin on 2009-04-26
Tehehehehe... I have a geforce 7000m too.... and I've learnt to never trust the options Ubuntu gives you. Here's what to do, write this down somewhere since we'll work from a command line. If you run Kubuntu change "gdm" rto "kdm"


Download the Linux drivers from nvidia.com
Rename those drivers to "drivers.run", and move them to your home folder.
CTRL+ALT+F1
Type "sudo /etc/init.d/gdm stop"
Type "sudo sh /home/yourusername/drivers.run"
Install the driver, respond yes to everything.
Type "sudo /etc/init.d/gdm start"

Enjoy.
