### Post by mike4ubuntu on 2014-04-02
Has anybody gotten the Acer T232HL touch screen feature to work with Ubuntu?  The Acer T230h seems to work just fine with display and TouchScreen feature working.  However, the Acer T232hl uses a new chipset, evidently, and isn't even recognized in Ubuntu 13.10.  The T232 has an integrated USB 3.0 interface that it sends the touch commands through.  It also has an integrated USB hub that allows you to connect USB devices to the monitor that then get passed through to the PC.  I'm getting ready to test it with the Ubuntu 14.04 Trusty in a day or two with the Daily build since 14.04 not officially released yet.

Also, the display looks fine.  It's just the Touch Screen and USB hub that don't seem to work.
