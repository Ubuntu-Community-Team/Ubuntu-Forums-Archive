### Post by jualin on 2008-08-03
Install the package "ubuntu-desktop" using Synaptic or using the terminal > sudo apt-get install ubuntu-desktop. Then you will have both in your computer, and every time you login you can switch by clicking on sessions and changing it to gnome or to kde. Hope this helps!
