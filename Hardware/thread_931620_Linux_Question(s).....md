### Post by jpkotta on 2008-09-27
> 1. Can I play Age of Empires II on Linux?
Probably not.

> 2. Its a free OS, so I take it updates are online somewhere?
Yes.

> 3. Is it suspect to all the virus/spyware like XP/Vista is?
No.

> 4. Is general software like Nero etc work on Linux?
Nero makes a Linux port, IIRC.  Usually there is an equivalent application, in the sense that it does the same thing, maybe worse, maybe better.  For example, I use k3b for buring CDs/DVDs, and I like it better than Nero.

> 6. Do you have to have a 64-bit processor to run it?
No, but it will run on them.  It runs on just about everything.

> 7. Can you download with it the same as Windows?
You will have to be more specific.  You can download things from the web with browser.  You can download torrents with a bittorrent client.  You can transfer files on a network with ssh, samba, nfs, etc...
