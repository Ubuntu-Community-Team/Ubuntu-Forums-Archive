### Post by staar on 2009-06-05

Section "Monitor"
# HorizSync source: builtin, VertRefresh source: builtin
Identifier "Monitor0"
VendorName "Unknown"
ModelName "CRT-0"
[B]HorizSync 30.0 - 82.0
VertRefresh 50.0 - 75.0[/B]
Option "DPMS"
EndSection

``` y estos en la sección Screen ```

Section "Screen"
Identifier "Screen0"
Device "Device0"
Monitor "Monitor0"
DefaultDepth 24
Option "TwinView" "0"
Option "TwinViewXineramaInfoOrder" "CRT-0"
[B]Option "metamodes" "1440x900 +0+0"
Option "UseEdid" "False"[/B]
