### Post by pancallo on 2013-10-08
This answer fixed the problem for me: [http://askubuntu.com/questions/57079/xubuntu-make-shiftnumpad-work-like-windows](http://askubuntu.com/questions/57079/xubuntu-make-shiftnumpad-work-like-windows)


[LIST=1]
[*]Search for Keyboard
[*]Click Layouts tab
[*]Click Options button
[*]Expand miscellaneous compatibility options
[*]Check "Shift with numeric keypad keys works as in MS Windows"
[*]Close and you're good!
[/LIST]

I'm using Ubuntu 13.04
