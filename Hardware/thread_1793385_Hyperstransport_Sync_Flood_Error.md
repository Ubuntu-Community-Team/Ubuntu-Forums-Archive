### Post by grunge09 on 2011-09-25
I have an MSI ddr2 MB with AMD 965 Quad Core CPU running 11.04 and Mythtv.  I started getting that error after I put in a new stick of RAM.

Turns out it was not the ram.  It was heat buildup.  I got a couple of Scythe Ultra Kaze 3000 RPM 120mm case fans and installed them and no more sync flood errors.  

Granted it is as loud as my 16 drive rackmount RAID file server now.  But its a small price to pay.  Case is cool, temps are down all is well for 7 days so far.  

Easy thing to cehck is to pull the side cover off the case and let the heat out, see if that helps.  Once I did that I know the MB was getting too hot.  Then I got the fast case fans.  

$10.88 shipped each on eBay for the fans.
