### Post by Favux on 2011-09-29
Hi GrindBastard9000,

Welcome to Ubuntu forums!

Sure you can boot from an USB stick.  See:  [http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

A thread that might be somewhat helpful is:  [http://ubuntuforums.org/showthread.php?t=1483623](http://ubuntuforums.org/showthread.php?t=1483623)  Pay attention the the HP security bloatware overwriting the MBR if you decide on dual boot.

For rotation you could also use Magick Rotation:  [https://launchpad.net/magick-rotation](https://launchpad.net/magick-rotation)  Because the TC 4200 and 4400 BIOS' reverse the normal hinge switch signal in Setup check the BIOS hinge switch values reversed and Save before rotating the first time.
