I have a Thinkpad W500 and it also had the LED light.  I *really* wanted a backlit keyboard originally, but having seen some since then, they're not as desirable as I thought.  Kinda spastic and also you can't really SEE where you're typing b/c your hands cover up the lights!  The top LED has its benefits, too... like being able to read printed text using it, for example.

If you STILL want a backlit keyboard, I found that the Dell Studios look great.  The Macbooks look terrible.  Just my opinion.

Oh, and my Thinkpad LED has worked fine with the function keys in Ubuntu from Hardy through Karmic.  Dang, ALL my function keys work... out of the box.  The W500 is built like a tank (and kinda looks like one) but the Ubuntu support is killer.

Good luck.
