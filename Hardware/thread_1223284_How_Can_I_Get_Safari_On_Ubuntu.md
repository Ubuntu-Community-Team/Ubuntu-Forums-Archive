### Post by Yanatsu on 2009-07-26
You can try Wine, which allows some Windows programs to be ran under Linux.
To get it, open console, then
```
sudo apt-get install wine
```Also, wrong board for this, but hopefully Wine will work. Otherwise, emulation, or get OSx86, which is MacOSX for normal PCs.
Or, you know, Windows.
Also, to use Wine, just download Safari's Windows installer and double click it, just like a normal Windows program, and it should install, then if it doesn't make a shortcut on your desktop, look under the applications menu and Wine, then Programs or whatever it's called.
