### Post by marcopolo1981 on 2009-11-11
For best results it is recommended that you use the vendors drivers where possible. 

These instructions work for me, and my card is correctly recognised and configured when I restart X.



Go to [http://www.nvidia.co.uk/Download/Find.aspx?lang=en-uk](http://www.nvidia.co.uk/Download/Find.aspx?lang=en-uk) and download the suitable driver for your graphics card.

Once downloaded, close all open documents to be left with nothing but your desktop and nothing in the panel. 

Press ctrl+alt+F3

Login as yourself on the command line.
Type 
sudo su 
 enter the password

Type 

/etc/init.d/kdm stop
to stop the kde -desktop environment

Type 
ls

This should show you what the nVidia file you just downloaded is called

Type 
sh (enter_the_name_of_the_file).run      NOT INCLUDING THE BRACKETS ()

follow the instructions to install the driver.

Once completed, type
Exit 
to return to normal user

Type
/etc/init.d/kdm start
to start the kde -desktop environment

and press ctrl+alt+F7
to return to GUI

Or alternatively, Type
Sudo reboot -n
to reboot the pc and log in as yourself with the driver correctly configured.

Like I say, this works flawlessly for me. Hope it helps you.

Mark
