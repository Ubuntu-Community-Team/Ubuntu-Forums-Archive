### Post by lekremyelsew on 2013-04-24
Make sure you have changed the setting on the phone to connect as an MTP device rather than a Camera.
If you still can't get that to work, you can try to install the android SDK and a program called aafm. That should allow you to transfer files to and from the phone through a GUI file management tool.
