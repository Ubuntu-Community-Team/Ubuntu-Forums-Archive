### Post by TheFu on 2014-02-12
There are many different Celeron CPUs. You'll need to know the exact model for any comparison.
[http://www.cpubenchmark.net/cpu_list.php](http://www.cpubenchmark.net/cpu_list.php) might be helpful, but CPUs are not the only consideration for performance. Sometimes the CPU is not nearly as important as the MB chipset, disk performance, and backplane throughput.
