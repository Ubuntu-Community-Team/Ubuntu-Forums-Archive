### Post by bcschmerker on 2013-06-15
I'm currently considering, among other keyboards, a custom 122-key Emulator version of the Model M from [Unicomp®]("http://pckeyboard.com/") based on the layout originally adopted for IBM® 6110744, a Model F that shipped as part of the Type 5251 microcomputer package (based on the Type 5160 Personal Computer XT; popular name: 3270 PC, as it was intended to replace certain Types 3277, 3278 and 3279 terminal flavors), as I find the current PC122 model too much of a departure from historical International Business Machines practice.  Unicomp® still offers the Classic 101/102 in a dozen languages.  Don't know if Unicomp® has a "blank keys" version of the Classic or Ultra Classic 104/105, much as Metadot offers in the form of the Cherry® MX Blue™-equipped [Das Keyboard Ultimate]("http://www.daskeyboard.com/").
