### Post by aurelieng on 2009-04-15
1/ if you do not have an external CD reader, you can use unetbootin to create a bootable installation USB key (see [http://unetbootin.sourceforge.net/]("http://unetbootin.sourceforge.net/"))

2/ yes. Just resize the NTFS partition upon partitioning instead of wiping it, and GRUB should recognize you windows.

3/ 8.10 is running perfectly. Make sure you install adam's kernel from [http://www.array.org]("http://www.array.org") (alternatively, you can go for Eeebuntu, which have it built-in [http://www.eeebuntu.org/]("http://www.eeebuntu.org/"))
