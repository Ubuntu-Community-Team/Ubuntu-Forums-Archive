### Post by wookieshaver on 2010-10-27
Well my first thought is about the setup of the laptop itself. HP does support on many intel chipsets with a seprate graphics card a feature called switchable graphics. This means (if unaltered) that the laptop switches to onboard video when unplugged from power and switches back to the discrete graphics when plugged into ac power. If the laptop was not plugged into ac power when you attempted to run the install cd, I would try plugging in ac power. If it was plugged in and attempting to use the graphics card - unplug the power when you are in windows to get it switch to the intel graphics on board (assuming windows is even there anymore) and then reboot to the Ubuntu install cd. Just a thought.
