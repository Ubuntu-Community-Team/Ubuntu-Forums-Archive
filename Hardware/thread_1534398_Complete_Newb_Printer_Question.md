### Post by ajgreeny on 2010-07-19
In 10.04 and I think in earlier versions of ubuntu you need to install the **brother-cups-wrapper-extra** for the printer part of the machine from the repositories, either with apt-get or synaptic.

The scanner driver is usually available as a .deb file from brother direct and all you need to do is download it and double click to install it.  Go to [http://www.brother-store.co.uk/category/brother-solutions-center_MjQyOA==.html](http://www.brother-store.co.uk/category/brother-solutions-center_MjQyOA==.html) for the scanner driver.
