### Post by Felix-the-Cat on 2010-04-20
That would be a kernel panic which could be caused by several issues. 

You shouldn't have to reinstall after kernel panic even if it did occur during the installation of a new kernel. You should be able to select the old kernel in the boot screen.

If you have a support package with Systems76 I would contact them or if you have a warranty. This could be a hardware problem.

More information of what happens when would be nice, i.e. what it does after shutting off during a update.
