### Post by DmitrySU on 2013-04-24
Let's create an Ubuntu on VAIO SVT support thread since USB is not only one problem.
All the Ubuntu problems on this laptop are(by majority):
1. USB 3.0 port works but 2.0 port doesn't.
2. Brightness hotkeys work but brightness doesn't change. However writing to intel-backlight file works.
3. Touch screen works for a few minutes after boot but then clicking, dragging stop working, only cursor and multitouch work. 
4. There must be a way to turn wireless module off to save power. In windows its done by enabling Airplane mode, need to find way for Ubuntu.
5. No hibernation opportunity is present.
6. Sometimes lightdm fails to start (fixed by editing lightdm.conf - add sleep 2 command before exec-lightdm, line 47.

### Post by DmitrySU on 2013-04-24
The problem number 7 for this laptop is:
It charges up only to 80% and no more. This is done in Sony Battery Care in Windows. To adjust that limit level in linux you need to edit file:
/sys/devices/platform/sony-laptop/battery_care_limiter (I found out that myself, really works)
