### Post by Mark Phelps on 2011-12-12
First of all, the CD is probably only Windows SW and drivers -- and those can't be used in Linux.

Second, you can't use Wine to install Windows drivers.  Linux needs Linux drivers.

Third, suggest you check out the manufacturer's website and see if they have any driver download for Linux.  Some do.
