### Post by TitanusEramius on 2010-04-20
Well, it's not much to go on, but I've got a few suggestions. The first is to check the drive with another Operating System, do you have Windows installed as well? Or maybe you can try to move the DVD-drive to another computer and check if it actually works.

The other thing you have to check, even if the drive works with another OS, is if the DVD-drive is connected correctly to the machine and if BOIS is showing it right. If the DVD-drive is SATA (connected with a thin red wire) you don't have to check the following, but if it's a IDE (connected with a very wide and probably a white wire) then you have check if the MASTER/SLAVE jumper is set right on the DVD-drive.
