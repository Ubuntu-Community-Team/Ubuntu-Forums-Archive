### Post by adam814 on 2010-01-01
I'd go with the second one.  The P4 is probably the fastest processor honestly, but since either is kinda low-end these days you'll get more out of the one with 1GB of RAM.  Either way I've had my share of headaches with ATI cards.  The first one has a 9200, which will work fine with the open source radeon driver, but won't be a strong performer.  The X1300 as I understand it still falls into the "limbo" situation where ATI doesn't support it with fglrx and the r500 development in the radeon driver isn't that mature.

Either way I'd go with the second one and see where you end up.  I have an r600 series ATI card in my laptop and have all but 3D acceleration working with the radeon driver.

Of course other factors might come into play (i.e. I wouldn't be willing to pay much more for the second than the first, but all things being equal I think it's your best bet).

### Post by mkoehler on 2010-01-01
#2.  The processors will roughly run at the same speed - however, the 1 GB of ram will provide some significant advantages.  It is true that ubuntu doesn't use 800 MB of ram to run the OS...however, start up a few programs and you'll surely be using upwards of 600 MB.  When the RAM is fully utilized, the OS utilizes swap space as a replacement for random access memory, which is very, very slow.  That's really the main dealbreaker for option #1.

My $0.02...good luck :)
