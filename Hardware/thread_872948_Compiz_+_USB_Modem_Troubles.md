### Post by blabbersoft on 2008-12-12
What version of Ubuntu are you using? Did you enable ICS on your Blackjack II or are you using it in modem mode? 

I have no problem connecting in Intrepid Ibex through ICS on my Blackjack. Initially, I was using Hardy Heroin and was encountering similar errors. After installing 8.10 opened ICS on my phone and pressed connect then plugged in the USB cable and 8.10 just connected with no errors. Additionally, I found that it connects as a removable drive if I (on my phone) open Settings => Connections => USB and set it as a mass storage device. When set as a mass storage device mode ICS can still be used, however, you cannot use ICS and browse the phones memory card at the same time. 

Currently, I have found no way to sync the Blackjack II with Ubuntu, but all other features appear to work with no problem.
