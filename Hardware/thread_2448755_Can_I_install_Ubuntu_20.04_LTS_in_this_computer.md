### Post by CelticWarrior on 2020-08-13
Hello again. :p

Yes, you can install Ubuntu 20.04 and it'll work acceptably **if the computer has the Nvidia GeForce 8500 GT**. If it has the old and very weak Intel GMA 3100 (integrated) then only the lightest of the desktop environments is recommended. That means Lubuntu or, even better, something else lighter and not in Ubuntu's family.

Furthermore, the "56K WinModem" likely won't work and I personally wouldn't use the original WiFi card. I would use instead a modern and supported USB WiFi.
