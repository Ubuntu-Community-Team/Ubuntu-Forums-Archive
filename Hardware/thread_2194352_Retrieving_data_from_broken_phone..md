### Post by efflandt on 2013-12-18
Another option to see the phone screen without it would any MHL adapter (micro-USB to HDMI to HDTV). However, if it is like my S3 (or a Note2) it would also need a micro-USB 5-pin to 11-pin adapter. I used this one which is currently $5.59 US [http://www.amazon.com/gp/product/B009ZA7XW4/ref=oh_details_o05_s00_i00?ie=UTF8&psc=1](http://www.amazon.com/gp/product/B009ZA7XW4/ref=oh_details_o05_s00_i00?ie=UTF8&psc=1)

I would not know what would need to be done to copy or transfer SMS, because the e-mail I use most is on an Exchange server and somehow text messages I receive end up there too. So the only important things are photos or videos that I save right to microSD. Your mobile provider may be able to transfer other data.
