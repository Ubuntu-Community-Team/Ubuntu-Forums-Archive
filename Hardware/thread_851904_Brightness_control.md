### Post by Bugu on 2008-07-07
Hi, I'm using Ubuntu 8.04.1 on a Samsung Q45 laptop. I can't use the brightness control. It doesn't work either with the Fn keys and the gnome-power-manager.
When i unplug the AC adapter the screen backlight is too low, and I can't do nothing to adjust. I've tried also with the brightness-applet but it doesn't work...

Thanks...

---

### Post by Bugu on 2008-07-07
SOLVED: the brightness control is disabled from the Bios, it's a BUG!

To run correctly the brightness control I downgraded my bios from the 15ST versione to the 09ST version.

That's all...
