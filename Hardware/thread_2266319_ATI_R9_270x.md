### Post by QIII on 2015-02-22
I have been having trouble getting an R9 290X to work with 14.04.5 myself and have been working on it with a second boot on this machine.  There seem to be some package conflicts that are going to be difficult to get around until they have it sorted out.  If you try to install xorg-video-abi-15, fglrx will be removed.  If you install it first, fglrx won't install.

Unfortunately, I've had to stick with 14.04.4 for the moment.

The odd thing is that Canonical bought the adapter for me to test and I recently blogged about the proprietary driver working with it in the development version, 15.04, just fine.  So something may not have been backported completely from Vivid to 14.04.5.

Don't know how much more testing and bug reporting I'll get done this weekend ...

Anyway, I have confirmed that it works perfectly with 14.04.4, so if you would like to run that until 14.04.5 is fixed, you can download the 14.04.2 .iso [here]("http://releases.ubuntu.com/14.04/") and then do updates and dist-upgrades to get it up to 14.04.4.
