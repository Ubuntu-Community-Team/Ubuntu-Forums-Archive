### Post by cnkbrown on 2008-11-10
My previous solution for dual head stopped working. I did this;

disconnected the 2nd monitor, 
opened System > Preference > Screen Resolution 
reconnected the 2nd head
clicked "Detect Displays" - which found the second display
set the resolutions and applied.

this gave an xorg.conf file that looks like this;

```

Section "Monitor"
	Identifier	"Configured Monitor"
EndSection

Section "Screen"
	Identifier	"Default Screen"
	Monitor		"Configured Monitor"
	Device		"Configured Video Device"
	SubSection "Display"
		Virtual	2800 1050
	EndSubSection
EndSection

Section "ServerLayout"
	Identifier	"Default Layout"
	Screen		"Default Screen"
EndSection

Section "Device"
	Identifier	"Configured Video Device"
EndSection

```

thanks guys, i have found another solution

i had Virtual with smaller numbers due i had used external monitor

i ran the command "sudo dpkg-reconfigure -phigh xserver-xorg" and it emptied my xorg.conf to reset to default values

Ctrl+Alt+Delete to restart X then logged in again and went to resolution and found my lovely 1440 x 900 :)
