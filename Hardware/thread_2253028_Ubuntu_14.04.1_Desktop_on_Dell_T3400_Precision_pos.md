### Post by weatherman2 on 2014-11-16
As long as it has 1GB of RAM or more it should be fine.  That doesn't sound too old for Ubuntu - most likely has a dual core CPU.  You might have issues with slow graphics performance on that chipset.  You can always use Gnome Flashback to get an old-style desktop interface that doesn't require fast graphics - or try Lubuntu, which is designed as a low-resource option for Ubuntu.

Or try booting a live USB Ubuntu on this machine and see how it works.
