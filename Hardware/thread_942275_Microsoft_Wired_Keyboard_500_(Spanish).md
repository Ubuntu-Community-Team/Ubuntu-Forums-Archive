### Post by HBit on 2009-04-07
May be too late for the post, but in case it's useful to someone else.

I've got the same Keyboard Wired Keyboard 500 Model RT2300 and it's working flaulessly under Ubuntu Hardy Heron.

The keyboard preferences:


Layout:  [COLOR="Navy"]"Latin American"[/COLOR]
Keyboard model: [COLOR="Navy"]Generic 105-Key (intl)PC[/COLOR]

Under Layout options...
>[COLOR="Navy"]Third Level choosers[/COLOR]
>> [COLOR="Navy"]Press right ALT key to chose 3rd level[/COLOR] [COLOR="Red"]**<---use this to enable AltGr funcionality[/COLOR]

Hope this helps
