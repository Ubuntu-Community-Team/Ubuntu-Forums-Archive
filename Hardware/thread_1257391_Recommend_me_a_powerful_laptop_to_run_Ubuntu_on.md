### Post by rodrigo.toste.gomes on 2009-09-04
Right now I'm running ubuntu on a clevo, and haven't had any performance/compatibility problems.
Everything works as it is supposed to do.
And best of all: it didn't even have windows installed originally, so no licensing fees.
I'm currently using the clevo M860TU.
Mine has a 9800m GT card, Intel Core 2 Duo CPU P9500 2.53GHz, wireless 802.11b/g/n wifi link 5300 card and the screen has 15.4" but supports 1680x1050, and has 4GB Ram DDR3 1066MHz, blu-ray and it has eSATA.
But my laptop is almos a year old, so there are better M860TU out there (since it's a barebone, everything is changeable), and you might find them with quad core, 260M GTX and 15.4" screens that support 1920x1200 resolutions and 8GB ram DDR3.
If you want more punch than that, you have the clevo D900F which comes with 12-cell battery, 17" screen that supports resolutions up to 1920x1200, comes with core i7, up to 12GB of DDR3 ram, GTX 280m.
Of course these will cost a lot more (although the total cost depends on the specs you choose. The D900F can go from 2500$ to about 4500$.
And the M860TU can go from 1500$ to 2500$.
Beside these two there are also very powerful notebooks from that same brand.
But I can only certify that mine is compatible with Linux, the other ones, I don't know :/ You can use some compatibility list.

EDIT: PS: Their disks are all 7200 RPM I believe ... but if you want real performance you can either buy one with several disks (the D900F can support up until 3 disks, but the M860TU only 1) and use RAID, or you can buy an SSD.
In terms of mobility, I believe the D900F isn't the best for that (or any 17" and up laptop from what I've been told) ... I'd advise you a 15.4" screen, since you can get great performance on those, and they're much more mobile.
