### Post by Rodney9 on 2008-11-25
I answered myself with a little searching, this is a full review done this year -

[http://www.everythingusb.com/logitech-illuminated-keyboard-15319.html](http://www.everythingusb.com/logitech-illuminated-keyboard-15319.html)

>  " We also conducted testing with the keyboard under MacOS X 10.5.4 and Ubuntu Linux 8.0.4 which both appear to operate correctly with the keyboard albeit that you don't have the luxury of using Logitech's SetPoint software under either operating systems. The function keys also seem to work and launch the operating systems default assigned internet browser, email client, etc. " 

> 
By default the media keys are accessed by holding down the Function (Fn) key and simultaneously selecting the media key associated (F1, F2, F3...F12) with the function you are looking for. This doesn't sound to bad, the function key is next to the right hand control key, it would seem to be easy to press with your right hand and use your left hand for the media key.
The sound volume, mute and backlight control have there own separate buttons above the numpad.

*> The Logitech Illuminated Keyboard is quite simply a pleasure to use,*

Rodney.

### Post by Bartender on 2009-12-28
Just happened to stumble on to this thread while looking into how hard it would be to make all the special function keys work. 

I picked up one of these Logitech Illuminated boards a week or two before Xmas at Staples.  Regular price $80, marked down to $30.  But honey, just think of all that money I saved!

I ran it in XP for the first few weeks.  Installed the SetPoint software, which seemed to add another few seconds to the XP boot time.  I really like the feel of the keys, the lighting is cool, and it's got a distinctive, sleek look.
