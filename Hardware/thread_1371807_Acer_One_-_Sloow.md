### Post by jlewisfl on 2010-01-04
Which model is it?  On my AO751H-1279, 2GB Ram, 250GB HD, the correct GMA500 video driver was not auto-installed, and as a result, repainting the screen was VERY slow, especially since the default Remix desktop is a full-screen menu, which has to be completely repainted when you simply scroll down on that full-screen menu.  I got that fixed through help on this Board, which sped up video considerably.   Then I added a complete KDE install, and and run a KDE desktop instead of the default - KDE seems to be hugely faster.  Part of that, I'm sure, is that it doesn't use the entire display for the menu, so there's a lot less info to be repainted.
