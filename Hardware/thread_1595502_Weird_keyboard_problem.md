### Post by Sum Guy on 2010-10-13
Just a little while ago, I was typing a post on the forum and my laptop keyboard stopped working. I rebooted, logged in, and after logging in, it stopped working again. The internal mouse works, and an external USB keyboard works fine. The strange thing is, all the function keys work perfectly, which is especially strange because the same keys don't work if I don't press Fn. Any ideas?
Thanks.

EDIT: The external keyboard just froze up too, but reconnecting it fixed the problem.

---

### Post by Sum Guy on 2010-10-15
Never mind, found the problem. For some reason, when I unknowingly activated slow keys, it disabled my internal keyboard, so when I wasn't logged in, the setting turned off, and the keyboard reactivated.
