### Post by grossogrossum on 2011-11-06
Sorry for my english.
I'm affected for this bug too. In my Acer Aspire plugging and unplugging AC power work properly after I adjust the brightness manually, but doesn't save the change when I reboot, screen is always dim when I start the system in 11.10. I solved this problem installing laptop-mode-tools by synaptic and configuring like this:

First I must apply a change in laptop-mode-tools script as told here (just the first one about kernel 3.0): [http://askubuntu.com/questions/74338/unhandled-kernel-version-error-when-trying-to-install-laptop-mode-tools](http://askubuntu.com/questions/74338/unhandled-kernel-version-error-when-trying-to-install-laptop-mode-tools).

To keep it working when I plugging the AC power:
```
sudo gedit /etc/laptop-mode/laptop-mode.conf
```and set
ENABLE_LAPTOP_MODE_ON_AC=1
(and ENABLE_AUTO_MODULES=0 because I don't really want to save power) 

To save changes in brightness after reboot ```
gedit /etc/laptop-mode/conf.d/lcd-brightness.conf
``` and change [value] as you wish. The last line must to be changed to /sys/class/backlight/.../brightness as the instructions in module file explains. This is how I set:

BATT_BRIGHTNESS_COMMAND="echo 4"
LM_AC_BRIGHTNESS_COMMAND="echo 8"
NOLM_AC_BRIGHTNESS_COMMAND="echo 8"
BRIGHTNESS_OUTPUT="/sys/class/backlight/acpi_video0/brightness"

I take de idea from [http://ubuntuguide.net/ubuntu-laptop-how-to-save-current-screen-brightness-settings](http://ubuntuguide.net/ubuntu-laptop-how-to-save-current-screen-brightness-settings) but this guide is for (I imagine) a previous version of laptop-mode-tools.


To force laptop-mode works on AC power ```
sudo /etc/init.d/laptop-mode start 
``` as is told in [http://samwel.tk/laptop_mode/faq](http://samwel.tk/laptop_mode/faq).
I only need to activate lcd-brightness.conf module but there are a lot of modules in /etc/laptop-mode/conf.d you can try.

I hope you can understand my poor, poor english and this can help you. I'm a newbie but after a month looking for a solution I find something that works to me so I decide to share.
