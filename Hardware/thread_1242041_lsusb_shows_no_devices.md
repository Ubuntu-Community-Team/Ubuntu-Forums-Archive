### Post by scottgun on 2009-08-16
I'm running dapper drake 6.06 LTS 2.6.15-29-server.  I resolved to host a USB HP Officejet 6110 aio printer off of the LAN.  Installed HPLIP, CUPS and Sane via apt.  HPLIP recognized printer.  CUPS configured successfully.  Printer visible and printable from LAN.

Problem: lsusb does not show the listing for the printer or for any other USB device.
```
$ sudo lsusb
Bus 005 Device 001: ID 0000:0000
Bus 002 Device 001: ID 0000:0000
Bus 001 Device 001: ID 0000:0000
Bus 003 Device 001: ID 0000:0000
Bus 004 Device 001: ID 0000:0000
```
I've run this command as super user, regular user and sudo.  Same result.  I also plugged in my ipod nano.  The nano screen powers up and connects but no listing.  I've rebooted.  Same result.  Unplugged, replugged said devices.  Changed cables. Changed ports. Same result.

Again, the printer works.  Why can't I see it with lsusb?
