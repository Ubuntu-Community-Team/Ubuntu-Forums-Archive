### Post by Matt__ on 2011-02-20
It all looks good to me.

ATI can be a pain but the open source drivers work well with 4250. No need for better graphics for surfing/open office/watching movies and even lower spec games like UT2004/second life.

One cautionary word, Ive used the ebuyer extra value ram many times, its the only memory I ever had problems with. It just dosnt like some motherboards...can you go with the almost as cheap Corsair?

Might also be worthwhile dropping the case fans and getting a better cpu cooler ..perhaps [_this one_](http://www.ebuyer.com/product/192753) with a side set fan to promote air flow through case? but thats just my personal opinion.

good luck with the build :)
