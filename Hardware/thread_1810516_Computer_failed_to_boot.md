### Post by Oldhacker on 2011-07-23
Computer would not boot this morning. Award-Phoenix BIOS gave one beep followed by three short beeps. Pulled the graphic card. Same result. Put back graphic card. Pulled the BIOS battery. Tested 0 V! Put in new 2032 BIOS battery. Booted but gave error in file check - no swap detected. Let Ubuntu repair file structure. Working again! :-)
