Due to problems with mouse I had to start over with ubuntu CD. 

Before starting over I had problems using printer, HP deskjet 3930v, with black carthridge. I changed the settings at CUPS for black cartridge and it printed.  

Now I change the settings at CUPS for black cartridge, click setting printer options, and print a test page. The test page is blank. I go back to the setting and I see it is set for color cartridge. I know that is the problem for test page under HPLIP tool box prints out.

What can I do to get the settings to remain set for black cartridge in CUPS?

Reset the settings and then typed selection of document and it worked.

June 22 found that the setting reverted back to colored cartridge.

Decided to leave color cartridge, which is very low, with the black cartridge in the printer. Now it is printing black. I also get notice that the ink level is low. 

Printed black 1 time. Now printing purple.

I a got the printer to work. I am ecstatic!

I read the instruction for testing printer in ubuntupocketguide-v1-1.pdf. Follow instructions to print a test. System > Administration > Printing, then double-clicked the printer&#8217;s icon and clicked the PRINT TEST PAGE button. Result was blank page. In printer settings changed print out mode to black cartridge. Now it prints. It seems printout mode has to be set at 3 locations, printing, HPLIP, and CUPS.
