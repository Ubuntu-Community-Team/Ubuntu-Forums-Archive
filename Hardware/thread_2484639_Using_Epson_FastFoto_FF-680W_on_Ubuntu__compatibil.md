### Post by jmax123 on 2023-07-11
You may have discovered this already but here goes anyway ... I have an epson fastfoto and an epson e2710. I find that if you install synaptic package manager thru the software center and enter "epson driver" it will work with both my epson printer and the fastfoto. As to scanning with both machines, I find that the simple-scan works fine with both machines to scan whatever. Epson have problems tho with photos. If you are trying to scan an image that is on slightly ruff paper, you get lines all thru the image. The best results come from images that are on smooth paper or glossy photo paper. Hope it helps if you haven't discovered this already. My system is ubuntu 22.10
