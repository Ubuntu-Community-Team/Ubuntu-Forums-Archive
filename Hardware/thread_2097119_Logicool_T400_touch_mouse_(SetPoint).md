Using: Xubuntu 12.10 64bits

I didn't find any post of anyone using this mouse in Linux, but I bought it anyways. Surprisingly, the mouse works without any problem in Ubuntu. It works the middle scroll, the middle click and even the launch-menu button (which binds to SUPER_L). Of course you need to bind that key to the menu in order to work. 

As I use much more the middle click, I would prefer to set the SUPER_L binding in the mouse to emulate middle click too (perhaps just change the buttons order). In windows it is supposed to be able to change that behaviour through software, using what they call "SetPoint". I wonder if any knows how to change that behaviour in linux?
