I'm investigating a more mild issue with heat, and I expect mine for the main part is based on high cpu or memory usage, so I would recommend checking that out with system monitor or some other cpu/memory monitoring application for a while, perhaps it's a problem with some app buggily stealing 100% of the cpu alot.

I'm also not sure that installing a different operating system will void the warranty on the hardware.  It may void the warranty on the operating system or something, but if there's a problem with the hardware then they might well still be responsible for fixing that, so they may still provide support.

Periodically, after heavy use, the system will start running unusably slowly. Even with nothing running, the system takes 30-60 seconds to respond to basic mouse clicks. Windows Task Manager shows taskmgr.exe hogging 50-100% of the CPU.

I contacted Dell support many times and they were no help. I ran through the diagnostics, ran various virus/malware checking apps, and I did a full reformat/reinstall of Windows Vista and this issue kept happening.

After this problem kept happening again and again (and it is maddeningly frustrating when you're in the middle of something and your laptop just stops working like this), I noticed that the system was unusually hot during these times. I think the problem is that I often work with the laptop on my lap, and when it gets hot, I put a pillow in between my lap and the laptop so that my legs don't burn. However, I suspect that pillow prevents the laptop from cooling, hence it overheats.

Dell tech support said this is probably not a heat issue since a heat issue would cause a system shutdown, not a slow-CPU type issue, but I think the support guy is wrong.

I have a full warranty, but I suspect this is just an overall issue with the laptop design rather than a defective unit. Personally, I'm going to try one of those laptop cooling pads.
