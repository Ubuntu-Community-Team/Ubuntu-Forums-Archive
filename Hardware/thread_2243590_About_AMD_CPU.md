### Post by hndrcks on 2014-09-11
Hello satimis

The A10 is a hybrid CPU and graphics processor designed for desktop environments. 

The FX series is just CPU, with no built-in graphics. 

Neither is designed for server use (the Opteron line is for servers).  The A series is for entry-level and mid-performance desktops, while the FX series is designed for higher performance desktops. A desktop with an FX CPU will require a separate graphics card, and will cost more for a complete setup.

### Post by grahammechanical on 2014-09-11
CPU = Computer Processing Unit. GPU = Graphics Processing Unit. It is the bit that does the work in a graphic/video adapter. Bring them together in one integrated circuit and they become APU = Advanced Processing Unit

[http://en.wikipedia.org/wiki/AMD_Accelerated_Processing_Unit](http://en.wikipedia.org/wiki/AMD_Accelerated_Processing_Unit)

There are 4 cores for the CPU side of the chip and 8 cores for the graphic/video adapter side of the chip.
