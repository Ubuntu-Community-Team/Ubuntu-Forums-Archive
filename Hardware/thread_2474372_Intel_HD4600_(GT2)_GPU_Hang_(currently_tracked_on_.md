### Post by doombotqq on 2022-04-27
Folks, This is just an awareness post that if you are experiencing ANY hanging issues with the Intel HD 4600 (GT2), there is an open critical bug issue over at this link: [https://gitlab.freedesktop.org/drm/intel/-/issues/5432](https://gitlab.freedesktop.org/drm/intel/-/issues/5432)

Feel free to contribute information if you are experiencing GPU Hang's on the Haswell iGPU's, without a discrete gpu attached.

This affects both Ubuntu 20.04 and 22.04, and many other linux distros as well.
