### Post by illusionate on 2010-07-29
I'm sure alot of people know Toshiba has ACPI issues with Linux. The trick is not to disable ACPI (you will ruin that new laptop). I have found a way to get around this the easy way no scripts ect. I installed this on my L505-GS5035.


Heres what I did.

Installed 9.04 (instead of the new 10.04 wouldn't even boot with ACPI)
Upgrade to 9.10 (via software update)
Install Toshset and Toshultils (Toshiba Support) (install realtek driver via terminal) (need help email me)
I have yet to upgrade to 10.04

So far I have yet to get the webcam working but I haven't tryed yet.
I've got Native Resolution, Sound, Voice, Wired, Wireless, CPU control, IDE, 
Anyone needs help email me I have instant email notification on.


UPDATE:

When Installing 10.04 LTS.
It is not a matter of Ubuntu with the ACPI.
It is the linux kernel 2.2.24 that causes the ACPI problems.
You need to select the 2.2.22 kernel when booting 10.4 LTS

So far I'm loving Ubuntu.

Good Luck Toshiba People!
