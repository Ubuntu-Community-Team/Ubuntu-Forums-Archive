### Post by cariboo on 2009-03-13
If you are running XP I would suggest running defrag at least twice, then using the LiveCD you can resize your partitions using System-->Administration-->Partition Editor.

If you are running Vista use Vista's disk management software to resize your  Windows partiton, then use the LiveCD Partition Editor to resize your Linux partitions.

Remember to back up important files from both installations, before doing anything.
