### Post by Richard Carlson on 2012-01-29
It seems to me somebody could get this fixed in some of the Linux Distro's. I use SDHC memory cards with one of my Dslr camera's and the rest of them use Compact Flash memory cards. On
two different desktop 64-bit systems one running Ubuntu 'Unity' 11.10 and the other Fedora 16 KDE with two different card readers I can't get the SDHC memory cards to even be seen. The Compact Flash memory cards have no problems whatsoever. They pop up and are seen in the /dev  and /mnt directories. The SDHC cards don't exist. In a Netbook (ACER) with a built in SDHC slot running DreamLinux v5.0 (i386) the SDHC card is seen and visible and even pops up the photo's. Why is this a reoccurring problem in Linux and hasn't it been fixed in the last 4-5 distro versions? It seems to me more time is spent re-writing the Distro's instead of fixing the problems with hardware. Any help would be appreciated. SDHC memory cards store photo's, mp3s ete. etc. etc. Such a small item yet seems to be overlooked time and time again.
