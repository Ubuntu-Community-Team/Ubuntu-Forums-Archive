### Post by mastablasta on 2017-01-17
with this much RAM and old buil-in intel video you might also want to try the lighter Xubuntu. you can try the linux OS before installing by either booting it from DVD or USB and selecting the try it option. it will load the whole OS first in ram, which means longer boot time but it will work a lot better when installed. the so called live session is good for testing hardware compatibility. note that sicne the OS works in RAM all changes are lost on reboot (unless you save somethign to the hard disk).
