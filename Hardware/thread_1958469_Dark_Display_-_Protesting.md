### Post by Toz on 2012-04-14
Hello and welcome to the forums.

A couple of suggestions/comments:

1. Are you using any command line parameters?
```
cat /proc/cmdline
```

2. Have you blacklisted any modules?
```
lsmod
```
...to see which modules you are using.

3. It appears that you are using the proprietary AMD drivers. Are these the ones that you installed from the Restricted Drivers app or did you manually install them. Also, what does this return:
```
sudo lspci -vnn | grep -A10 VGA
```

4. Its a little confusing from your post, but when exactly did the brightness stop working? Did it ever work with 11.10 (oneiric)?

And, do you have a LiveCD or bootable usb of 11.10 that you can fire up and see if the brightness works from there? If it does, it would be interesting to compare the results of the above commands between a live run and your installed setup.
