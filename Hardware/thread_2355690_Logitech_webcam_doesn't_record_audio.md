### Post by TheFu on 2017-03-15
Logitech makes a number of different webcams. The exact model would be useful to share. Not all logitech devices are supported under Linux. Contact Logitech if yours isn't and complain.

If you have audio issues, running through the "ubuntu audio troubleshooter" would be recommended too.

Have you tried using a different program?
Have you tried using a different userid? Sometimes old settings aren't compatible, so the easy way to test this is to create a new userid and login using it.

That's all the ideas I have. Sorry.
