### Post by InspiredIndividual on 2009-04-03
As far as I know, this is perfectly normal behaviour for the Caps Lock key. If you want Caps Lock to toggle Shift so all keys are effected, numbers included, go to System > Preferences > Keyboard. In the "Layouts" tab, click "Other Options". Now, select the radio button in front of "Caps Lock toggles Shift so all keys are effected" in "Caps Lock key behaviour".
