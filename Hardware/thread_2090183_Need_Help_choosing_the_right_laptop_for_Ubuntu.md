### Post by SeijiSensei on 2012-12-01
One problem with HPs, and perhaps other brands, is that the disks have all four primary partitions in use.  This makes it very difficult to set up a dual-boot system without jumping through a lot of hoops.  If you plan on ditching Windows and installing Ubuntu, it's not a big deal.  

Another issue to consider concerns the newer "hybrid" graphics offerings that include both an Intel and an NVIDIA or ATI adapter.  While there have been some advances in this area like the [Bumblebee Project]("http://www.bumblebee-project.org/"), I'd do a bit of reading here to see how well-supported these machines are.  When I [configured]("http://ubuntuforums.org/showpost.php?p=11299287&postcount=5") my daughter's HP dv6t, not only did I have to deal with the disk-partitioning problem, I had to upgrade the BIOS to use the machine's ATI card.

I avoid Sonys.  In the past they have had a lot of proprietary features that made installing Linux on them problematic.  I don't know whether that's still true.
