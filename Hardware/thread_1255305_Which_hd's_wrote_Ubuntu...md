I can't answer which HDs were used to write Ubuntu but generally I find that Seagate hard drives run MUCH cooler than similar WDs for instance under any OS.
I've also noticed that they're much shorter which would explain it as this might mean that there is more space for better air flow...

On the other hand, I find that even though Seagates are quicker at random data retrieval WDs are faster with sustained reading (i.e. reading a large continuous file or several big consecutive files)
