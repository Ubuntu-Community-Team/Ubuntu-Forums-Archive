### Post by taurus on 2009-03-15
Unmount it first.

```
sudo umount /dev/sda1
```
Now, edit /etc/fstab

```
gksudo gedit /etc/fstab
```
and replace the mount point for /dev/sda1 (last entry) with the new mount point, /media/Internal3.

```

/dev/sda1 **[COLOR="Blue"]/media/Internal3[/COLOR]** ntfs-3g defaults,locale=en_GB.UTF-8 0 0

```
Save it and close down gedit editing window.  Now, you need to create that new mount point and mount it.

```
sudo mkdir /media/Internal3
sudo mount -a
df -h
```
