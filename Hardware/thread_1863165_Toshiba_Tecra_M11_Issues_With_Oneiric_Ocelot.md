### Post by GreatAshGoblin on 2011-10-17
Hey!

So the Toshiba Tecra M11 was touted as being very compatible with Natty Narwhal, and for the most part it was. I had a few issues with brightness control and hibernation, but otherwise everything was just peachy.

Ubuntu kept asking me to upgrade to Oneiric Ocelot (the options to not upgrade or upgrade later didn't work in the GUI for some reason, only the update button did). So I eventually decided, why not? I think I might revert to Natty...

Here are some of the issues I have found so far, please feel free to add more and any fixes for the issues:

- The touchpad stops working after an inconsistent amount of time and starts working again if the computer is put into standby. The little blue ball at the center of the keyboard (forget what it's called) still functions as a mouse and the set of mouse buttons for it also do. Mysterious!

- The function key (Fn) doesn't seem to be recognized anymore, which means that I can't test brightness control at all.

- Issues with hibernation seem to still be around.

On the plus side, I love the new UI!
