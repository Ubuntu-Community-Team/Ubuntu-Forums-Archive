### Post by joshedmonds on 2009-07-29
To install a package such as LTSP (linux terminal server project) and have it survive rebooting the live USB environment, you need to make the live USB persistent.

Try this page of the Ubuntu documentation:

[https://help.ubuntu.com/community/LiveCD/Persistence]("https://help.ubuntu.com/community/LiveCD/Persistence")
