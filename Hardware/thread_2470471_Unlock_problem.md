### Post by grahammechanical on 2021-12-31
Please give clarification.

Is this happening at the Xubuntu login screen? The motherboard has a settings utility called UEFI (Unified Extensible Firmware Interface). We can alter the settings to require a password before the motherboard will look for an operating system. I doubt if the mouse is recognized until after the password has been accepted.

I see "unlock my laptop" as a different problem from "login to Xubuntu." It is possible to set Xubuntu to login automatically.

This problem may be caused by the enter keys on the keyboard or the mouse keys not working correctly. What happens if you use the enter keys on the keyboard instead of clicking a mouse key?
