### Post by Lars Noodén on 2010-08-28
You can install ubuntu bare bones from the alternate install cd and choose 'command line system'.  

From there  you can install a window manager (fluxbox, openbox, fvwm, oroborus) instead of a heavy desktop environment (gnome, kde, xfce).  After that, try which programs use the least amount of resources.  

If you can inexpensively add RAM to the computer, that will be well worth the effort.  [Fluxbuntu](http://www.fluxbuntu.org/) is one pre-packaged distro making an attempt at being lightweight, but by cherry picking the applications and adjusting the configurations you can make any distro like any other.

> **Writh said:**

I don't know how much experience you have with Linux and how uncomplicated you want the installation to be, but Linux Mint is good and easy to start with. [Crunchbang]("http://crunchbanglinux.org/") is a good too and will be lighter. Zenwalk, which uses XFCE, is also good and fairly light, but uses a different package management (used to install applications) which may confuse you a little initially if you are used to Ubuntu and Debian's way of doing things.

But I would go with Crunchbang if you want something simple, light and elegant.

(And in response to what Lars Noodén wrote: Fluxbuntu is seriously outdated, though, and the project has been abandoned. I would not install that. Xfce is also fairly light, though Xubuntu's use of it is.)
