### Post by hawaii-dave on 2009-04-03
I was given the job of retrieving a client's important data from a 'dying' Windows laptop.  I was able to save the data but Windows was terminal.  I reformatted and attempted to install a fresh copy of WinXP from Masters.  Got about 60% installed and the computer just shut off - no warning, no symptoms, nothing, just flat dead.  Restarted and tried again - same result!  Ok, dead computer, put it in the dumpster. Over two hours wasted!

But wait!  I thought "Why not try to install Linux, what can it hurt?"  Since Ubuntu is my favorite version, I put the CD in and started up.  Ubuntu installed perfectly within 30 minutes and has worked flawlessly for the last 72 hours.  I can't break it!

Once again proof (as if we needed it) that Winders is a POS and Linux is the answer!

Yeah Ubuntu!  Now the client has a useful computer that he is going to donate to a local youth group.
