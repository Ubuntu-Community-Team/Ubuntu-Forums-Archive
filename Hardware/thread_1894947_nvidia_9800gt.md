### Post by Mr. Shannon on 2011-12-13
The 1 short 2 long beep code is usually a video error.  Therefore, the card could either not be seated correctly or could be bad.  I guess it could be a problem or incompatibility with the motherboard.

Try re-seating the card (while the machine is off and unplugged).  If that does not work then I would put your old card back in and see if that still works.  If it does it is either a compatibility between the motherboard/BIOS, and video card or a bad video card.

This is not a software issue so there is no need to try booting from a live CD or anything.

I have only had something like this happen once so take my advice with a grain of salt.  When this happened we sent the computer in to the company which replaced the whole machine after months of delays, but that is another story.
