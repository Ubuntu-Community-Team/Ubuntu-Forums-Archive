### Post by tgalati4 on 2009-06-18
PIII will use more power 66 watts vs 46 watts.  PIII has more cache and "feels" faster.  Max out RAM in either case.  256 MB or more.  It will run on 192 MB (Linux Mint XFCE) for instance, but it will be slow and painful to use.

This class of machine makes for a decent server though.

Slackware and Arch run well on these machines.  Of course DSL runs really well.
