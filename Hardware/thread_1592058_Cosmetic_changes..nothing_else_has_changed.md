### Post by chrisplymouth on 2010-10-10
I had a few hours to spare and decided to download and install Ubuntu 10.10.

As an OpenSUSE 11.3 KDE user, I do not see any difference between the new release and the last two! The changes I can see are cosmetic; the same unstable core and ugly Gnome desktop lurks on the system.

Within 5 minutes, I noticed that my Broadcom wireless card still wasn't supported out of the box, the touchpad driver was still temperamental and there is a bug in the gnome sudo forcing users to physically shut down the authorisation window from the taskbar.

Has the Openoffice 3.2 memory leak problem been fixed yet? I don't even want to try.


Marks: 4/10
