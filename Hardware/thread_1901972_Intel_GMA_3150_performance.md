Hi. I recently bought intel motherboard D525MW with dual core cpu d525 and intel GMA 3150 graphics card. I wanted this hw as a replacement for my "old notebook NAS, torrent, web server for private use". I was curious about the performance of this hw on some desktop distribution. I was surprised by the performance compared to old P4 3GHz Dell Optiplex. I started thinking about replacing this P4 by this motherboard (power comsumption is ~30W against ~100W). But the graphics performance looked worse. I tried linux mint 11 there - but the graphics performance did not look fine - a bit laggy, but usable. Then I tried Xubuntu and Lubuntu as a lightweight alternative. I like Xubuntu - because it feels not as minimalistic as Lubuntu. But - when I compare these 2 distros, then in Xubuntu scrolling of some webpages is laggy or delayed (by moving scrollbar). Scrolling by mouse wheel looks better or fine. In Lubuntu scrolling is almost perfect (still can see some delay). In Xubuntu it is quite annoying. When I change the width of the window to 1/2 of the screen (1680x1050) then the scrolling looks fine.

Is it normal that performance of this intel gma 3150 graphic card is so poor? I always thought the best vga for linux is intel one. I have also other pc (P4 3GHz) with Nvidia GT 8500 vga and graphics performance there is absolutely super.
I also have old notebook with pentium m 1.6GHz cpu and integrated intel vga (815 I think) and even there the graphics performance is better than on this intel D525MW board.

So what do you think - simply forget about using D525MW as a desktop and use it only as a NAS box? Or is there some solution for the integrated intel vga?
