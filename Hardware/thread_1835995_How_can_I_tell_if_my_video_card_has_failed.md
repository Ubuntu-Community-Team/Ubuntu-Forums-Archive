### Post by diasf on 2011-08-30
Go on the terminal and try an lspci . It will list all connected hardware. Find the exact model of you VGA, go find and install the drivers (both OS).

I don't think there's anything physically wrong with your board. Just lacking drivers.
By the way, the "update driver" of windows never really worked. Once you know the manufacturer, go to the website. If it's a laptop, go to the laptop's manufacturer site. (eg, my old laptop's X1400 drivers existed only on dell's website, not on ATI's).

### Post by .... on 2011-08-31
The proprietary driver does not support this card, so I'm not sure how you fixed it, unless you borked your graphics stack and it now uses the CPU to do rendering acceleration (which will work, but slowly).
Post output of glxinfo:
```
glxinfo
```
As for testing the card, I would use atitool to check for artifacts: [http://www.techpowerup.com/atitool/](http://www.techpowerup.com/atitool/)
First, make sure the fan is working and/or the heatsink isn't clogged with dust, because that might cause overheating and artifacts. If your card is passively-cooled (my X300SE is), make sure it doesn't get too hot and install a quiet 120mm fan (zip-tie it to the drive cage or something) if necessary.

### Post by grahammechanical on 2011-09-03
From my own experience, I would warn about overheating as a danger to video cards. Make sure that there is a clean air flow.

In Ubuntu if you are using a proprietary driver it may come with its own configuration utility that displays the graphic card temperature. I have this with my Nvidia driver.

You could also install xsensors. It gives a visual representation of CPU and motherboard temperature as well as fan speeds. It will depend on the information that your motherboard BIOS makes available. Does the BIOS measure temperature?

I had an overheating video card that lost the monitor configuration and would not load Ubuntu. I do not have any other OS so I cannot say how it would have affected a Windows OS. Something knocked out the Windows drivers from the OS. What was it? It will happen again. Possibly.
