Not on your list, but I have the Acer Aspire One A110L with 8GB SSD.

It works great! The only thing that is a little iffy is the card readers; the one only works if a card is present at boot time.

The best thing about this netbook is that it's so popular, so is very well documented, especially on the Ubuntu front.

Wireless, video, webcam, network, sound works out-the-box.

You might, however want to make a few modifications in order to make things work even better. These are detailed at 

[https://help.ubuntu.com/community/AspireOne110L](https://help.ubuntu.com/community/AspireOne110L)

Basically a few things that make life easier on the A110L, like disabling access times for files on the (relatively slow) SSD.

Of course, this all also applies to the AAO with the hard drive, you just won't need to make the optimisations for SSD. And it'll run even better :)
