Video Card:           AMD Radeon R5 M230 2GB 

Hi. 
I bought a '**Dell Inspiron 15 3541**' ( *DI3541A2-A66310-4GH50D4BK-11* ) notebook which have the '**AMD Radeon R5 M230 2GB**' second graphics card. I've tried find their driver on dell-support but I couldn't find it. 
So, I moved ATI/AMD-Support website and I found this: [AMD Catalyst™ 15.5 Proprietary Ubuntu 14.04 x86_64 Video Driver for Graphics Accelerators]("http://support.amd.com/en-us/download/desktop?os=Ubuntu%20x86%2064") 

I tried to install this and other results. But all those didn't install because of errors. 
So, I installed the 'fglrx' of the main repository. But after that the '**Catalyst Control Center**' says there is '**ATI Radeon R4**' only and the games didn't run well. About between 5-15 FPS. 

So, I don't know what should I do now for I can play my games. 
P.S.: I also tried install Ubuntu 14.04.2 - AMD64 but there was a same problem. 
Thanks further the help and comments. 
Sincerely, Atlaimond
