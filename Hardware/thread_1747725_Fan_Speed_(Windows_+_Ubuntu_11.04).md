I just set up a dual boot system, now in Windows 7 & Ubuntu 11.04 the fan on the GPU & CPU is constantly on max.  Does anyone know what is causing it and how to revert it back to what it was at?  Thank You.

### Post by DdlyArcher on 2011-05-03
Ok I figured it out.  On this laptop maybe others if you hit the function key + 1 key it puts the laptop into max fan speed instead of automatic.  I guess when Ubuntu installed it enabled max fan speed.  Everything is back to normal now.
