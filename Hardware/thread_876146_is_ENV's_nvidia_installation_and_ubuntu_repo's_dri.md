### Post by evets25 on 2008-07-31
Using envy will generally install a slightly newer version of the nvidia drivers, but unless you have a good reason to need the newest bleeding edge version (i.e.: it fixes a specific bug that you know you have on your specific card), it is always HIGHLY recommended that you use the standard drivers from the ubuntu repo, via the restricted drivers manager. Installing the drivers the standard way is dead simple, and is far less likely to break your system.
