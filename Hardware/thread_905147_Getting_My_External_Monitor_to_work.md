### Post by kagashe on 2008-08-31
Try xrandr which is a command line to displayconfig-gtk.

This command tells you what is available:
> $ xrandr -q

If you see that 1600x1200 is available you can set it up with this command:
> $ xrandr --output VGA0 --mode 1600x1200

I am assuming that your external monitor is indicated as VGA0 in the output you get for the first command "xrandr -q" if it is slightly different change it.

kagashe
