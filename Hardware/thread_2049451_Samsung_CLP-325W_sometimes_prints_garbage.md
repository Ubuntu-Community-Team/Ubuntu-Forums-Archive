### Post by darfed on 2012-12-19
I'm using the CLP 325W on ubuntu 12.04.  It's all good, USB, wired or wireless until you try to print images.  These come out really gloomy and dark; really, too dark.  I've had some success in redressing this but it's still far from perfect and well short of what the printer is capable of via other OSs (let's not go there).  
This is a shame as otherwise it's a pretty decent printer.  It's possible that the official samsung 'unified' printer driver would get better results but in order to install it neatly (to create a deb so that it can be installed via package management) it looks like a faff and I've seen posts that suggest it's not all plain sailing then either.  I may try this out though as it's pretty important to us to get images printing nicely.
