### Post by realzippy on 2009-08-20
...get rid of that driver.Uncheck it in restricted drivers,reboot.
Download this one (assuming you are on 32 bit):

[http://us.download.nvidia.com/XFree86/Linux-x86/185.18.31/NVIDIA-Linux-x86-185.18.31-pkg1.run](http://us.download.nvidia.com/XFree86/Linux-x86/185.18.31/NVIDIA-Linux-x86-185.18.31-pkg1.run)

save it to Desktop.
Log out,press: Ctrl+Alt+F3
Log in,type:

**sudo /etc/init.d/gdm stop**

Go to Desktop. 

**cd Desktop**

type:

[B]sudo sh NVIDIA-Linux-x86-185.18.31-pkg1.run
[/B]
(just need to type "sudo sh N" and press TAB)

If asked,say "yes"

**sudo reboot now**
