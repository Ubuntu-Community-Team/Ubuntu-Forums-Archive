### Post by zo3adams on 2009-10-01
Sounds like it might be an issue with the very basic power management in the Server edition not being configured correctly for the laptop hardware.

Can you SSH (or use putty from windows PC) into the server to troubleshoot?

This site can be helpful, recommends using  "the i810 module" for your laptop, might be a good starting point for your search: [http://www.linlap.com/wiki/dell+latitude+d510](http://www.linlap.com/wiki/dell+latitude+d510)
