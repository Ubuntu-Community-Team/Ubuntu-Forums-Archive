### Post by yorakal on 2023-04-08
Support for Windows (Arm) AARCH64 laptops with Linux is only in very early stages, there are [experimental images]("https://github.com/hexdump0815/imagebuilder/blob/main/systems/snapdragon_7c_woa/readme.md") of Debian for some AARCH64 laptops which use the Snapdragon 7c.

Might be a while before you can install stock Ubuntu Arm on these machines.
