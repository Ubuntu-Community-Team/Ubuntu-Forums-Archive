### Post by sleeper86 on 2009-01-03
I have this same laptop and the same problem.  I took off a panel on the bottom of the laptop and found the Intel wireless chip.  

Typing the model of the chip into Intel's website brings up a list of drivers.  There are some Linux drivers available from the site.  However, it requires you to rebuild the kernel, which is a bit beyond my current level of expertise.  As a quick fix, I set up a dual boot so I can get wireless in Windows.  I'm still working on getting the wireless to work in Ubuntu.
