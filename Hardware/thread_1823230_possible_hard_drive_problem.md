### Post by militantduck on 2011-08-11
Hard drives have a system of keeping track of bad sectors so they don't get used again. So the value is probably accurate. But hopefully total failure is not imminent.

Make sure you regularly back up data of course.

Problems can arise when the bad sectors are in certain areas of the disk that have boot information or kernel stuff. As it only takes one bad sector in the wrong place to stop boot up.
