### Post by Linuxer on 2010-03-23
I have installed Ubuntu 9.10 on a Lenovo G 450 with 4gb ram and and a 320gb hard drive.  I purchased it refurbished from Lenovo for US$ 450 (About 300 pounds) It came with Windows 7 installed on the hard drive.  I installed Ubuntu 9.10 on a 8gb flash drive.  Everything worked fine until I installed the latest kernel update about 3 days ago.  Now I get Grub boot menu and I select the latest kernel (I am not near my computer so I don't have the version number) and I briefly get Ubuntu symbol and then my screen goes completely black.  Prior to the the latest kernel update, it worked flawlessly.
