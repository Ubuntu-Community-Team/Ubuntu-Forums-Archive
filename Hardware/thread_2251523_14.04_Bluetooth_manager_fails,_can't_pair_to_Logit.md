### Post by RCheesley on 2015-10-16
Did you have your headphones in pairing mode?  It will fail unless you use pairing mode during configuration:

You need to enable pairing mode by sliding the next track slider (beneath the off/bluetooth/USB switch) up, and pressing & holding the volume up button. Wait until the green light flashes rapidly, indicating that it's in pairing mode.

Once the headphones are added to your device, you'll need to set up the headphones as an audio device (sometimes you need to re-do the pairing mode above, if it's taken a little while to get the headphones paired in the first place, before you can set them up). 

**The headphones must be in pairing mode to do set up the device,** when going through the setup, use a custom code of 0000 and it should then pair successfully.
