### Post by bayvista on 2009-06-18
I have the same problem with an LBP3100. It installs correctly but does not print. If there is anyone out there that understands printers, please help.

I eventually solved the problem. Whilst trying different drivers and printers, I had left another printer tying up the USB socket. I deleted that using System/Sdmin/Printing and reinstalled the correct drivers. Bingo. It worked.
