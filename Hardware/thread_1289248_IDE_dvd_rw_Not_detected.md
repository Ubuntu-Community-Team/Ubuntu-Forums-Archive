### Post by dabl on 2009-10-12
Start with making sure the IDE cable* is connected securely at both ends, and then check BIOS and make sure the optical drive shows up there.

Do you have any old Live CDs from different versions of Linux?  It might be instructive to try booting a couple of them, to see if it works in ANY distribution.

*IDE ribbon cables can fail, btw.  Do you have another one that you could swap in for testing?
