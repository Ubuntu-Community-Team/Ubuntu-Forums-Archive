### Post by MrSpike16 on 2011-12-04
The logs tells us that you have the wrong Nvidia driver installed.  Also it appears you have a GTX 400 series card.  Ubuntu 10.04 doesn't have Nvidia drivers for newer Nvidia cards so you probably need to use the x-swat repository.

Try these commands:
```

sudo apt-get purge nvidia*
sudo add-apt-repository ppa:ubuntu-x-swat/x-updates
sudo apt-get update
sudo apt-get install nvidia-current nvidia-settings

```

Hopefully the first command and booting using failsafe will be enough to allow correct drivers and Nvidia kernel module to install correctly.
