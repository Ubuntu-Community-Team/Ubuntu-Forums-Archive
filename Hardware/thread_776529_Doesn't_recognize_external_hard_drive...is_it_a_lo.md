I believe you have to 'mount' the drive (volume), before Linux sees it. This is like partitioning the drive in Windows. Check both the Seagate website and Ubuntu Tutorials. Seagate will have the software you may need to partition the drive, and Ubuntu Tutorials in Documentation should tell you how to do it.  

As for upgrading from Gutsy to Hardy-Hardy comes with a beta version of Firefox 3. Firefox 3 has a problem-big time-with Adobe Flash. It won't work without serious patching. It CAN be fixed, I fixed mine. There are tutorials right here under 'Multimedia' on how to fix it. But be sure you can work Terminal OK, you'll need it. 

If not, stay with Gutsy until Firefox3 is officially released.
