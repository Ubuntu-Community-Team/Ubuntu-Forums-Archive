### Post by javispedro2 on 2014-08-07
Basically, you need Windows to install updates as they come in. Albeit as of now it is uncertain if there will be more interesting updates.

Before wiping Windows be sure to create the "USB recovery disk", there is plenty of Surface-specific instructions around.

Also ensure you install the UEFI CA key before wiping Windows: [http://uefidk.intel.com/blog/using-microsoft-uefi-ca-tool-surface-pro](http://uefidk.intel.com/blog/using-microsoft-uefi-ca-tool-surface-pro) . Otherwise you will have to disable Secure Boot to boot Ubuntu.
