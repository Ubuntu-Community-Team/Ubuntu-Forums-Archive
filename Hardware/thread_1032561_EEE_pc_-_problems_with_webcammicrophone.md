### Post by tonywhelan on 2009-01-16
For cheese:
System menu, Administration, Synaptic Package Manager.
Click Status button then show All packages.
Find cheese in the list of packages, rightclick and select Mark for Installation. Press the Apply button on tool bar to install it. 

Not sure about the microphone yet, I find sound mixers on all computers to be contrary and confusing. My asus 900 won't record sound yet either but haven't yet had a chance to search a bit more for fixes....
