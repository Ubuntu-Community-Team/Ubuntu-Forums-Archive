### Post by StephenF on 2010-10-30
That small screen and keyboard could cramp your style a bit. My programming rig has two large monitors, one of which is tilted sideways so I can see a hundred lines of code at once.

I use my netbook primarily as a bittorrent download box configured for RSS and to browse the net for train times when the main machine is off. It makes a handy second computer for testing networked stuff. When it comes to administration I prefer to VNC into there rather than pull the machine out and use its actual trackpad and keyboard.

I fear that any unpleasantness in your learning curve would be amplified by the restrictive machine and they are too easy to throw.

Edit: my machine is an MSI U-100 that I installed Ubuntu netbook remix onto using a flash drive and the Ubuntu startup disk creator program. Problems: the screen is still too small for some dialog boxes.
