The 20GB is a limitation of the windows installer. If you want more, you will have to make a full install (partition) or run off a USB.  

As a general rule of thumb, Ubuntu only requires graphics and sometimes network drivers, so no, you don't need any others unless something is obviously not working or wrong. :)

Mice...I'm not familiar with the hardware, but have a quick google around, see if you can find anything out. If not, it may be worth digging an old one out (I mean, everyone has old mice, right?)

Games...take a look in the software centre, I quite like hedgewars (worms clone).
