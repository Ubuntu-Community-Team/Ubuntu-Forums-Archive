### Post by TheFu on 2015-10-05
Just be aware that if you do need a mixer to have different audio channels input, that USB microphones do not usually work with the $50-$150 mixers.

For a single audio input, these USB microphones are very convenient and work well. 

If this is for a commercial audio production, get the yeti. $120 isn't much and that last 3% quality improvement may matter.

If this is for a hobby audio production, get the Samson. Read the reviews about it on Amazon. I wrote one too. 

There are lots of youtube recordings using both microphones, so be certain to see of you can hear the difference before spending the extra $80.  I couldn't, but I'm old and attended too many Van Halen concerts.
