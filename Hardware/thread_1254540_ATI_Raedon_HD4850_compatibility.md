### Post by 00arthuryu on 2009-08-31
I've been using 4850 for a long time now on ubuntu 8.10 and 9.04 on 2 computers, it works fine, as long as you install the latest ATI fglrx drivers from AMD. Since the ones that come default with ubuntu are outofdate and playing Video/Movies are a pain. (They flash continuously)
But installing the latest one solves this.

[http://support.amd.com/us/gpudownload/Pages/index.aspx](http://support.amd.com/us/gpudownload/Pages/index.aspx)
