### Post by Xinh on 2008-05-06
I have had a very strange problem crop up.   I downloaded and burned the iso of Ubuntu 8.04 using IrfanRecorder.   The burn process seemed to go fine.   I went to test the disk by rebooting my PC so I could boot from the Ubuntu LiveCD.   I have been able to do this in the past with no issues with Edgy (6.10) Feisty (7.04) and Gutsy (7.10).   However this time I have run into a serious issue:

Upon reboot, my system should first boot from CD, then if the system does not find a bootable disk in the drive, move onto the hard drive.   However at the stage where the system should attempt to boot from CD, there is zero DVD/CD ROM activity.  I can select my DVD/CD drives in my systems Bios and later they appear in Windows Explorer, yet I can do nothing with the drives.    Prior to burning 8.04, I could use my drives with no issue.   However after burning 8.04 they do not "want" to function.   

To rectify this issue, I have tried rebooting my system.   I have also tried swapping DVD/CD drives making sure there was only one drive connected to the system and that the jumper was in the Master position.   No dice.   I have even hauled an old drive out of storage and tested.  Nothing.   No boot from CD.  I cannot read any CD/DVD in Windows.   I can select the drive however Windows does not find any media (whether it be the Ubuntu 8.04 disk, a game Disk (tested with Homeworld 2, Diablo 2 and Warcraft 3), a standard audio CD, or a data disk).  

When I installed the old CD drive (an old HP CD Writer Plus 9300), Windows XP detected the drive, installed the drivers however and it appears in Explorer.   However as before Windows cannot detect any media in the drive nor can I boot from the Drive.   It does appear in my system Bios and I can set it as a boot drive, however again no disc activity.   

As I stated before these issues only started when I burned Ubuntu 8.04 on 04 May 2008.   Before that date I could use the CD drives with no issues.  After, nothing.

System specs

Vendor: Self built
OS: Windows XP SP2
CPU: AMD Athlon64 3000+
Mainboard: ASUS KV8 MB
RAM: Corsair 1 GB
Video Card: ASUS ATI 9600SE (AGP)
Hard Drive: Maxtor 120 GB HD (IDE)
DVD/CD Drives: LG DVD-RW ("Super Multi DVD Drive"), LG DVD-ROM, HP CD Writer Plus 9300 connected via IDE.

This is a home PC, used primarily for web surfing, gaming, some graphic design and some office work.

I have been trying to track a fix to the issue as if my system dies I am pretty much screwed as I will not be able to boot from CD or install a new OS if needed.
