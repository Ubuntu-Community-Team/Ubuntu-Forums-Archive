### Post by Bucky Ball on 2017-07-21
I'd also suggest Xubuntu, but would suggest you may want to go lighter. This approach would need a bit of a learning curve, but unaware of how au fait you are with Ubuntu already. You'll know plenty if you go this way. 

You could install [Xubuntu-core]("https://xubuntu.org/news/introducing-xubuntu-core/"). This gives you the desktop and a few default Xubuntu things, and that is it. You then install whatever you want need. For instance, you can install what you want through the terminal or install Synaptic Package Manager and use that GUI. I usually install Firefox, Synaptic and take it from there on a fresh Xubuntu-core install.

The benefit of such and install is that you get nothing that is not useful to you. No bloat, only the apps you use. A standard Xubuntu will come with some stuff you have no need for and other stuff you'll probably never use or be aware of. This gets rid of that.

I suggest 16.04 LTS, supported until April 2019. You [download the Ubuntu 16.04 LTS mini.iso]("https://help.ubuntu.com/community/Installation/MinimalCD") and take it from there. (Follow instructions on previous link. Very straightforward.) Don't worry that the mini.iso is 'Ubuntu'. It ONLY installs the Ubuntu kernel which is used by ALL flavours of Ubuntu. When you follow the Xubuntu-core process, you'll end up with a minimal, or 'core', Xubuntu. :)

Food for thought. Good luck, however you go with it. ;)

(PS: The mini.iso is a text-only installer and has NO 'Try' option, so if you are wanting to have a look at a full Xubuntu then yes, as suggested by others, burn standard, full Xubuntu install media and 'Try Xubuntu' from that.

Incidentally, I use Xubuntu-core on all my machines, from a dinky 11'' HP Stream - which flies - to my AV workhorse with an i7 and 16Gb of RAM. Why install stuff I'm not ever going to use?)
