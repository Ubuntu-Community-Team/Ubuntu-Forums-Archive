### Post by booyootoo on 2011-01-12
I bought the 1215N this weekend and installed Ubuntu Netbook remix on it (Dual boot). Everything is working for me. Had to install Ubuntu to download and install the broadcom wireless network drivers (via ethernet cable), this would not work with the "try ubunu" option.
 
Did not install the Nvidia drivers as I had read that others had problems with them.
 
It was late, but it seems everything is working. Though the netbook is responding slower then I expected (Both in Windows as in Ubuntu).
 
Example:When I click on "applications" the applications don't show up instantly as I expected, but I had to wait for maybe a second or less. This second makes me feel the netbook is unresponsive. I can't compare to other netbooks though... Just my thoughts on it.
