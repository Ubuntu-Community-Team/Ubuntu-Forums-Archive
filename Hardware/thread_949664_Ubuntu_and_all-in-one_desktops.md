### Post by jlochhead on 2008-12-13
I have no experience of all-in-one desktops... but I shouldn't think it would be too hard. They are basically just a screen, speakers and a cpu, basically like any laptop.

The only problem might be if you get a touch screen one. I don't know if Gnome has any touch screen support, however I do know that Kubuntu comes with touch screen support installed by default. 

On the other hand though, I have tried Kubuntu on KDE4, and I have noticed that KDE4 seems to have very bad support for my newish graphics card (GeForce 9600 GT), so that might be a problem if you're considering using Kubuntu.

### Post by haiku88 on 2009-02-22
> **jamesey said:**
> anyone figure this out?

FINALLY got my Vaio VGC-JS110 to work properly today using Jaunty 64 bit and the following hacks to my xorg.conf file, now have a sharp, clear display at the native resolution of 1680x1050, sound, webcam, and wifi work too. However, use at your own risk, these settings may not be correct for the display as I was unable to find the horiz sync spec on the web AND I basically don't know what I'm doing;)

# xorg.conf (X.Org X Window System server configuration file)
#
# This file was generated by dexconf, the Debian X Configuration tool, using
# values from the debconf database.
#
# Edit this file with caution, and see the xorg.conf manual page.
# (Type "man xorg.conf" at the shell prompt.)
#
# This file is automatically updated on xserver-xorg package upgrades *only*
# if it has not been modified since the last upgrade of the xserver-xorg
# package.
#
# Note that some configuration settings that could be done previously
# in this file, now are automatically configured by the server and settings
# here are ignored.
#
# If you have edited this file but would like it to be automatically updated
# again, run the following command:
# sudo dpkg-reconfigure -phigh xserver-xorg

Section "Device"
Identifier "intel"
Driver "intel"
EndSection

Section "Monitor"
Identifier "LVDS-1"
HorizSync 30-75
VertRefresh 60
EndSection

Section "Monitor"
Identifier "VGA"
Option "Disable" "true"
EndSection

Section "Screen"
Identifier "Default Screen"
Monitor "LVDS-1"
Device "intel"
DefaultDepth 24
SubSection "Display"
Depth 16
Modes "1680x1050"
EndSubSection
SubSection "Display"
Depth 24
Modes "1680x1050"
EndSubSection
EndSection
