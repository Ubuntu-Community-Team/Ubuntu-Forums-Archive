### Post by Favux on 2009-11-24
Hi razor180,

It could be a couple of things.  Which version of linuxwacom do you currently have installed?  If you have remnants of a previous version there can be conflicts.

Most likely it is one of two things.

1)  You don't have wacom.ko installed or auto-loading, or you have the wrong version.  So check lsmod:
```
lsmod | grep wacom
```
and see if we can determine the version:
```
modinfo -d wacom
```

2)  You do not have the wacom.rules for symlinks in “/etc/udev/rules.d”.  See Appendix 3 in this [HOW TO]("http://ubuntuforums.org/showpost.php?p=6546012&postcount=1") and check to see if the rules are installed.  If not install them.  Check both locations, Intrepid and earlier and the Jaunty and later location before you install anything.
