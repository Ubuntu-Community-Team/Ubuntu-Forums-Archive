So far I've only had everything work in LMDE and Fedora 20/21 alpha. I'm wondering if there's some driver or module or something that I'm needing? I belive it uses the CMI 8788 driver. Any way of getting this to working properly at all? Major let down to not have my dedicated audio work properly.

### Post by Yellow Pasque on 2014-12-01
So it works with 3.13 kernel but not with 3.16? That seems kind of odd. There were a bunch of commits for the DG(X) that landed in kernel 3.14 (the ones on Jan 29): [https://github.com/torvalds/linux/commits/master/sound/pci/oxygen](https://github.com/torvalds/linux/commits/master/sound/pci/oxygen)

My next experiment would be to try a 3.16 (or later) kernel with the Mint install and see if that breaks the speaker out. If it does, it's very possible you're looking at a regression introduced by those commits. In that case, the best place to ask for help would probably be: [https://lists.sourceforge.net/lists/listinfo/alsa-user](https://lists.sourceforge.net/lists/listinfo/alsa-user)  . Someone on there may have an idea by comparing alsa info's, or they may be clueless and ask you to bisect the kernel to find the specific commit that caused it.
