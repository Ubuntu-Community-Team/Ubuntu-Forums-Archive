### Post by aphirst on 2009-06-07
To be perfectly honest, nowadays you might as well install 64-bit Linux. There was a time when there were serious compatibility issues, but everything works now (the only thing that comes to mind is Adobe Flash seems to be a bit slow), and as a general rule stuff runs faster.

Celerons aren't *amazing* chips (I assume you're on a budget, or limited by what your motherboard supports), but you should be able to get pretty good performance out of it with Linux x86_64. Certainly better than just with Linux x86. ^_^

EDIT: Ah, it seems that the Celeron you got is basically just an under-clocked cache-crippled C2D. Which makes it leaps and bounds better than just a Celeron. Pretty damn good deal as well. ^_^

### Post by aphirst on 2009-06-07
Yes, indeed it will. AMD came up with the 64-bit extention to the 32-bit architecture first (afaik), so Ubuntu has opted to call their 64-bit release "amd64". The release will work on processors made by both companies, however, since they are built on compatible specifications.
