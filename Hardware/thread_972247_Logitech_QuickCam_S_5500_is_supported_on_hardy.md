### Post by bpl07 on 2009-01-24
I use this camera in ubuntu.  It should just be plug and play.  I use skype ([medibuntu repository]("https://help.ubuntu.com/community/Medibuntu#Adding the Repositories")), others use Cheese.  You don't get a response from the camera until you go in to settings for the program you want to use it in and select the S5500 as the video device.  I've had issues with it only showing a black screen at first, but all you have to do is shine a light in it and it works normally. I haven't had this issue for some time though, and I've had the problem in windows too.
