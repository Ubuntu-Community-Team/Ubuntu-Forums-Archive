### Post by snowpine on 2008-10-16
Hi there, Pentium 3 computers like your laptop are just fast enough run Ubuntu or Xubuntu (I'm typing on one now), but you should definitely max your ram out at 512mb before proceeding. A lot of laptops from that era came standard with 128mb, and that's just not enough.

Ubuntu 8.10 is still in testing, so if you're having any install problems, you might want to roll back to 8.04, which is the current "stable" release. I find that Xubuntu is a bit lighter on resources than Ubuntu, so it would be my suggestion out of the two. You can install any Ubuntu program in Xubuntu, and vice-versa, because they share the same "repository" of applications. Both are good distros to learn Linux.
