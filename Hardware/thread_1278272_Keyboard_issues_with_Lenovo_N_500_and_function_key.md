### Post by mohnkern on 2009-09-29
I've got a Lenovo N500 that I recently replaced the keyboard on, and I'm seeing a problem.  

This laptop has a numeric keypad that's available via pressing the Fn (function) key in the lower right hand corner.   

The problem I'm seeing is that when I type, it's behaving like the Fn Key is down.  When I hold the Fn key down it behaves like its up.  (In other words the Fn key is behaving "in reverse")

Has anyone ever seen this and been able to fix it?

---

### Post by mohnkern on 2009-09-29
I think I have this fixed.  The "numlock" key was down. Holding down Fn and pressing the Delete key turns on numlock).  Pressing it again solved the problem.
