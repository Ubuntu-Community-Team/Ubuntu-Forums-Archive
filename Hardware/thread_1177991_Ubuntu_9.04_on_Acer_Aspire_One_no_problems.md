### Post by Steelmourne on 2009-06-04
For anyone considering installing Jaunty on the Windows XP version of the Acer Aspire One, I've had no problems. This was a slightly older model with only a 120GB hard drive but with the same N270 Atom processor and 1GB RAM. Used another laptop to create a usb startup disk to install from, then a quick change in the BIOS boot menu to boot from usb and jaunty installed flawlessly.

Wireless and sound work out of the box although the Aspire One speakers are on the underside so sound quality is pretty poor without headphones. The resolution was detected perfectly (1024 x 600) and all that needed attention was the LED light for wireless. Otherwise all went well. RAM usage hovers around 35-40% with 1GB memory installed, and effects are set to normal which is all I'd expect from Intel GMA 950 drivers.

In short, Ubuntu is a great replacement to old and dull XP.
