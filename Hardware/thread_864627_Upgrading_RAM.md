### Post by silkstone on 2008-07-21
Many online sellers have guides to tell you what type of RAM is compatible with your machine - they don't want to sell you the wrong sort!

With many laptops you need to match the RAM modules in each socket. e.g. 2 x 1GB. There really is no need for more than 2GB to run Ubuntu, and if you have the 32-bit version it won't recognise more than just over 3GB anyway.
