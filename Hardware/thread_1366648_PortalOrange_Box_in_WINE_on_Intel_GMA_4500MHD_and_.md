### Post by femngi on 2009-12-29
First of all that's a pretty old version of wine. If you're going to try running more modern games then you're going to want the beta version by adding the repos from their site [http://www.winehq.org/download/deb ]("http://www.winehq.org/download/deb")[]("http://www.winehq.org/download/deb")
Secondly you shouldn't really be looking here to see how to run Windows games. Instead check out [http://appdb.winehq.org/appview.php?iAppId=5936](http://appdb.winehq.org/appview.php?iAppId=5936) 
There you should find instructions on the best way to run it.
Having said that I have the same card and have found it to be flaky at best even running natively on Windows. Don't expect immediate success, or even any at all.
