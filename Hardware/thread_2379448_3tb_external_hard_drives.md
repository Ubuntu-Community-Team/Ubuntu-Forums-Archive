### Post by sudodus on 2017-12-05
If you use the GUID partition table, GPT, it works well with hard disk drives with more than 2 TB.

### Post by oldfred on 2017-12-05
Issues in past often were using an older USB caddy that did not support larger drives or gpt partitioning. 
And some were pre-configured to allow use with XP, which only worked with MBR. And MBR has a max of 2TiB, so used non-standard sector sizes.

Most now work without issue.
