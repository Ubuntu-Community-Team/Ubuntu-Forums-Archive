### Post by fullmoonguru on 2015-01-29
I am running 14.04 on an Intel HD77 mb with an i3-3220T chip (HD2500 on-board graphics).  I have two monitors with DVI connections.  The MB has one DVI connector and one HDMI so the HDMI connector has an HDMI-DVI converter on it.

OK so both monitors work, but the DVI side regularly drops and reconnects.  This has been an issue for a long time.  Even with 12.04 I believe, but it seems like it's getting worse.  Anyway, looking for an answer.  It usually reconnects within a second or two, but sometimes it gets pretty bad with it going on-off-on-off-on...  I have switched monitors & cables both, and it's always the DVI side that's the problem so I know the problem lies with the board.  Any ideas?

### Post by lukeiamyourfather on 2015-01-29
In my experience this is caused by EMI and can be fixed with cables that have better shielding and ferrite cores. Use a cable no longer than what is necessary to make the connection. It's possible there's some sort of hardware problem with the graphics adapter or the monitor but I'd start with a high quality cable before trying anything else.
