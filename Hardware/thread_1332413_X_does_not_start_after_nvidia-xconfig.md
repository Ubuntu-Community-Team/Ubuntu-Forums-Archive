### Post by gradinaruvasile on 2009-11-21
Well for that u have to have vdpau enabled drivers. Here is the repo u need to install:

[https://launchpad.net/~nvidia-vdpau/+archive/ppa](https://launchpad.net/~nvidia-vdpau/+archive/ppa) - follow the directions there, reload the package archives.
First, install (or upgrade to) the latest video driver. It is automatically done if you already have it installed.
Then install xine-ui and mplayer. These two players have vdpau (hardware acceleration) support for some codecs (x264 and some mpegs). U MUST select in their configuration the output driver specifically "vdpau" - xine's ui is a bit confusing, first off u will have to select the experience level "master of the known universe" to make selections in the video tab.
But the fastest if mplayer launched from the command line (mplayer -ao pulse(or alsa) -vo vdpau /path/to/movie/nameofthemovie)

### Post by realzippy on 2009-11-21
Install it manually.
32bit:
[http://www.nvidia.com/content/DriverDownload-March2009/confirmation.php?url=/XFree86/Linux-x86/190.42/NVIDIA-Linux-x86-190.42-pkg1.run&lang=us&type=GeForce](http://www.nvidia.com/content/DriverDownload-March2009/confirmation.php?url=/XFree86/Linux-x86/190.42/NVIDIA-Linux-x86-190.42-pkg1.run&lang=us&type=GeForce)
64bit:
[http://www.nvidia.com/content/DriverDownload-March2009/confirmation.php?url=/XFree86/Linux-x86_64/190.42/NVIDIA-Linux-x86_64-190.42-pkg2.run&lang=us&type=GeForce](http://www.nvidia.com/content/DriverDownload-March2009/confirmation.php?url=/XFree86/Linux-x86_64/190.42/NVIDIA-Linux-x86_64-190.42-pkg2.run&lang=us&type=GeForce)

Uninstall current driver first.Maybe it can be done also in /system/administration/restricted driver.
Install manually,assuming downloaded driver to Desktop:

Log out,press alt+ctrl+F1,log in at shell:

**sudo /etc/init.d/gdm stop**

**cd Desktop**

**sudo sh NV  **             *(hit TAB to autocomplete filename)*

"yes" when asked.

**sudo reboot**
