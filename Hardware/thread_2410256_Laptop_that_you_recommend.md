### Post by wheatpenny2 on 2019-01-13
I have a Lenovo Ideapad 320 (8 GB ram, 1 TB hard drive) which cost me $500. It runs Ubuntu with no issues whatsoever. With a dual boot, unlike many other computers, on this computer, I don't have to move Ubuntu to the top of the boot order, because the computer does that automatically when I install Ubuntu.
I highly recommend it.
