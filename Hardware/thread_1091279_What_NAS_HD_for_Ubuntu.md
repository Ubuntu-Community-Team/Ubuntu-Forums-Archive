### Post by mperry on 2009-03-09
I've generally had good luck with the Buffalo Linkstation systems. I've also hacked a few of them and put Debian on them to expand their functionality. There are community websites which also tell how to "mod" these things to do more. I've done the same thing in a mixed linux and windows network using a smaller form factor system like a shuttle and then getting a tb drive in it. Just need to add things like NFS, Samba, etc. I even have firefly media server running on it to share my mp3s to the house.

This probably cost me about $400 in total to build out. Did all the shopping at Newegg.
