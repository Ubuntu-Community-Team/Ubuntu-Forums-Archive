### Post by sudodus on 2013-06-05
If the computer has a VGA output port and the TV has a VGA input port, it should be possible to use the TV as monitor. I have done it with several TVs. You need to select input from VGA in the TV and the computer needs to send a signal to the VGA port. Check in Ubuntu's settings menu that the screen is recognized and that a working resolution can be selected.

### Post by jellyini on 2013-06-05
Also look for NTSC vs PAL settings.  NTSC is for N America and some parts of  S  America.  PAL is the color encoding system for most of the rest of the world (unless they are using SECAM or PAL/SECAM).  

I recall a time when I was setting up a laptop to use a television to watch a movie.  I got all the resolution and refresh rates synced up and couldn't get it to show color.  It was the NTSC/PAL setting that fixed it.

### Post by aromo on 2013-06-05
Yeah, I have it done as well, but my TV doesn't have VGA port, only S-video.  I plug the TV S-video port to my laptop S-video port, turn on the TV and through its menu I select the S-video input, then I turn on the laptop and the desktop is mirrored on the TV, then I use XRandr to un-mirror the screen and set up a suitable resolution on the TV.  Hope this helps.
