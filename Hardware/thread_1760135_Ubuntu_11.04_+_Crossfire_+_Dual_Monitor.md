### Post by gigamonk on 2011-05-19
Ok, it seems I've solved it. I doubt anyone will run into this exact same problem, but just in case, here's some things to try:

1) Swap the monitor connectors around.
[COLOR=RoyalBlue]*--This fixed the issue where my secondary screen was appearing "on top / within" my primary monitor. Not sure why this fixed it, but it did...*[/COLOR]

2) Set the primary monitor in your ~/.config/monitors.xml file.
[COLOR=RoyalBlue]--This resolved any issues with desktop area / launcher position / snapping windows / grids.[/COLOR]
