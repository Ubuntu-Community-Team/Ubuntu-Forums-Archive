### Post by wilee-nilee on 2010-06-17
I have been looking on the web for the key prompt to choose the boot from screen, on my acer-aspire it is f12. I have seen computers not boot from a cd even if it's 1st in the read list. I found this description through bios though.
> 
go into BIOS then go to startup then boot then put the cd drive first and the hard drive second on the list. it asks you to press any key to boot from cd. follow instructions and it's fixed.

I have a much older thinkpad and it will boot with a key prompt to the boot from list.
