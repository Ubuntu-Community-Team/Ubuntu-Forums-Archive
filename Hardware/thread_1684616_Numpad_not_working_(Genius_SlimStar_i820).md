### Post by drcuacua on 2011-02-09
Hi there! This is my first thread, i've read your posts a lot of times when solving some problems, but i couldn't find a solution for this one.

I've recently bought a wireless keboard/mouse kit (Genius SlimStar i820) and i'm having a very rare issue: the keypad is not working. It's not that the keys don't work at all, it's more like if the Numlock key was never activated. In fact, when i press the keys, they just move the mouse pointer in the direction they show ("4" moves left, "6" moves right, "2" downwards, "8" upwards, even "1", "3", "7" and "9" move it diagonally!!) Furthermore, "0", "dot", "slash", "asterisc", "plus" and "minus" don't do anything at all!! The only key working is the Enter.

Along with the kit came an Installation CD, which is only supported for Windows, so i tried installing it with Wine unsuccessfully, nothing changed. I haven't tried it yet in my sisters computer which has Vista to see if there works.

Is there a way to solve this? Is there a way to install manually a regular "Multimedia keyboard" driver?

Thank you all in advance!

---

### Post by drcuacua on 2011-02-13
Got it working! It was just a configuration problem.

I went to System -> Preferences -> Keyboard -> Mouse keys.

There you have to uncheck where it says "Pointer can be controlled using the keypad"

Bye!
