I am not running Karmic,but it is in "synaptic package manager" for Jaunty.

Open synaptic and search for DCP-330C and you should find ```
brother-cups-wrapper-bh7
brother-lpr-drivers-bh7
```
Install both and "Add new printer" and you can select the correct driver.


OR look [here](http://solutions.brother.com/linux/en_us/download_prn.html#DCP-330C)


Good Luck
