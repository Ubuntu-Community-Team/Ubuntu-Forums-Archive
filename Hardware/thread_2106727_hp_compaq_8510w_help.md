### Post by tlhIngan on 2013-01-19
This specific laptop model maxes out at 4GB of RAM, so there would be no point in buying say 8 GB. You would only be able to use the first 4 GB of it.
As far as hard drives go, it's a matter of personal preference. For the same price, you can get a solid state drive that is 20x faster, or a spin drive that can store 20x more. Many people go for one of each, using the spin drive for data and media storage while using the solid state to house their operating system and software.
