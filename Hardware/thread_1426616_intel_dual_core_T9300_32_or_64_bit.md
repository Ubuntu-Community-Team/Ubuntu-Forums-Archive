### Post by Fafler on 2010-03-10
Both will work. The reason its called AMD64 is because AMD came first with 64 bit support in x86 processors, so they got to name it.

Some things are faster in 64 bit mode and some things, like in my case, home banking and windows games under wine, work better in 32 bit mode.
