### Post by dandnsmith on 2011-02-01
What you describe is a common problem.
First you have to determin whether the drive 'notices' the insertion of a CD or DVD (should run up speed, and possibly flash lights - my LG drives do)

If it does, then it is likely that the index track is not being read - very unlikely because of a stuck head, but possible. More likely that the reading system is giving trouble. As a first step try a CD/DVD lens-cleaner disk (available in lots of places).

Another possibility is a failing laser - but these drives usually have seperate lasers for CD and DVD, so you usually notice failure to read one type before it happens with the other.

Finally, any of these drives can fail - usually when it is safely out of the warranty period.
