### Post by jneusteter on 2010-03-23
First, have you installed the graphics driver?
Search the Main Menu for driver, should come up with Hardware Driver utility.

Many times the software that is used for your graphics card work better for adjusting screen resolution.

If all that fails you can edit your xorg.conf file.
