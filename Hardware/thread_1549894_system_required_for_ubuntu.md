### Post by snowpine on 2010-08-10
The Ubuntu System Requirements are very well documented:

[https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

Your RAM is lower than the "Recommended Minimum for Beginners" (1GB) but meets the recommendation for "Advanced Users" (512MB) and exceeds that for Xubuntu (256MB). 

Other than RAM, your system meets all the requirements. If you can get another stick of RAM you should be able to run Ubuntu with no problem at all, if not, Xubuntu might be the better choice.
