Ok, what version of Ubuntu are you running?

If running Hardy then do this:

```
sudo apt-get install nvidia-glx-new
```

The nvidia-glx-new for Gutsy and earlier is the 100.xx drivers that DO NOT support the 8xxx series cards.

If running Gutsy or Feisty then download Envy Legacy 0.9.10 and do the following:

[http://albertomilone.com/nvidia_scripts1.html](http://albertomilone.com/nvidia_scripts1.html)

1.  Remove the nvidia driver
2.  Manually install the nvidia driver and select the 169.12 driver
3.  Let Envy configure your xorg.conf file.
4.  Reboot the computer.

Let me know if this works.

### Post by supersonicdarky on 2008-07-02
Actually I got it to work!

For those interested:
- run the usual way to configure X (sudo dpkg-reconfigure -phigh xserver-xorg)
- now it loaded for me in normal res (1680x1050) but w/ vesa
- edit xorg.conf and add in the things from the nvidia-xconfig one (if none availible, click [here](http://en.wikipedia.org/wiki/Xorg.conf))
- copypasta things like Driver "nvidia", the whole composite and module sections and the subsection in Screen
- restart X (ctrl+alt+backspace)
- enjoy your shiny new gpu ;)
