### Post by z37a on 2012-10-17
Si queres comprar algo para un server mi consejo es, por compatibilidad TODO Intel, chipset y demas. Para durabilidad, el procesador es lo de menos, pero el motherboard es fundamental, busca algun mother (en lo posible marca Intel o como mucho ASUS) que tenga todos sus capacitores y componentes blindados, esto te asegura un mayor MTB (Tiempo estimado de vida), siendo un equipo 7x24 es fundamental esto, asi vas a tener menos problemas y se lo va a aguantar mas!

La temperatura ni idea, nunca controle ni se como se hace.

Saludos.

Para hacerla rapido, el mismo BIOS generalmente tiene una opcion de monitoreo de temperaturas y tensiones de la motherboard y sus componentes (CPU). No es muy preciso ya que el sistema no se encuentra trabajando en regimen, pero es mejor que nada.

Luego en casi todos los escritorios graficos hay gadgets para exhibir temperaturas, tensiones y otras mensuras del hardware.

Por otra parte, seria importante determinar el estado de los capacitores electroliticos que rodean la CPU y que generalmente se "pinchan" cuando trabajan mas alla de su temperatura maxima (se hinchan y pierden el electrolito que les permite funcionar correctamente).
