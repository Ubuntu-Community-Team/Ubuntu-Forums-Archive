### Post by kurt18947 on 2015-09-30
HP printers have been an excellent choice for some years. HP-LIP, HP's linux printer software is included in *buntu distros. New model printers may require downloading the latest HP-LIP from HP. I've had good success with Brother and it's easier to get Brother printers working since they released installer scripts. This involves terminal use but I download the script then open a terminal on the same screen as the directions. I often copy/paste to prevent typos.  Run the script, select the connection and that should do it. I do something similar with a Samsung laser multi-function. Manufacturers tend to use scripts run in a terminal I think because that takes care of differences between distros and desktops. One terminal to rule them all, so to speak :).

Here are the instructions and download link for Brother. There is an FAQ page for printers and one for scanners.

[http://support.brother.com/g/s/id/linux/en/faq_prn.html?c=us_ot&lang=en&comple=on&redirect=on#f00104](http://support.brother.com/g/s/id/linux/en/faq_prn.html?c=us_ot&lang=en&comple=on&redirect=on#f00104)

Here is Samsung's driver download site for my machine:

[http://www.samsung.com/us/support/owners/product/SL-M2870FW/XAC](http://www.samsung.com/us/support/owners/product/SL-M2870FW/XAC)

There's nothing that says you have to own the machine in order to download the driver and get a feel for what's involved in installing it. That may help you decide on the best machine for you, and the more you do the easier it becomes.
