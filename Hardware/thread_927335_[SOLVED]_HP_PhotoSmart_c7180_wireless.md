Sorry about that. I was able to resolve the issue by just installing the HP Linux Imaging and Printing software [<click here>]("http://hplipopensource.com/hplip-web/index.html"). I downloaded the .run file, executed it and it found my wireless HP printer. It works great.

In case you don't know how to execute a .run file...(I didn't know how)

Launch a terminal window
navigate to the directory that the file is in,


chmod +x nameoffile.run


./nameoffile.run
