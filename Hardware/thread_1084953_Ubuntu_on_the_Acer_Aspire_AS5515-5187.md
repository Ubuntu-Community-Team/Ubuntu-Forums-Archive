### Post by MARAUDER2003 on 2010-01-29
Ubuntu works fine on this laptop. Jaunty and Karmic do. In jaunty the only thing that does not work is the light for the wireless card, not a big issue though. In karmic everything works. I found out that the cpu frequency scaling thing does not work if you enable the dual "concurrency" feature (for booting faster using dual core cpus). The laptop boots fast enough so no need for concurrency feature to be enabled.
