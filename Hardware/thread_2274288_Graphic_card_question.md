### Post by Mark Phelps on 2015-04-19
>  If I install the graphic card how would I install the drivers and software that cam on the CD with my new Graphic card?

Basically -- you don't!  The stuff on that CD is for Windows, not for Linux.  You can't use it.

Ubuntu should automatically install the default Radeon drivers for that card when you first boot with the card installed.
