Hi.

You CAN get a non-PAE version of Puppy 'Tahrpup'; there's more than one version. As long as you are able to download, you can get it from here:-

[http://distro.ibiblio.org/puppylinux/puppy-tahr/iso/tahrpup%20-6.0-CE/](http://distro.ibiblio.org/puppylinux/puppy-tahr/iso/tahrpup%20-6.0-CE/)

You want the 2nd entry up from the bottom. That will give you 6.02, which is the current state of play. 

I've been using the PAE version since it was released, back in October; it's a brilliant little lightweight distro. You can run the entire thing from a USB stick; 'Puppy' is designed to work this way. You can also install directly TO that same USB stick...no need for a CD/DVD.

You don't say what your system specs are.....specifically the amount of RAM you have available. Ideally, you need at least 512 MB of RAM to run 'Tahrpup' properly.

Hope that helps!

EDIT: I've just been looking at your lappie (if it IS a 600**M**); it appears to have 512 MB of RAM as standard. It's not so different to my old Inspiron 1100; this is the machine I run 'Tahrpup on.....

[http://www.notebookreview.com/notebookreview/dell-inspiron-1100-review/](http://www.notebookreview.com/notebookreview/dell-inspiron-1100-review/)
