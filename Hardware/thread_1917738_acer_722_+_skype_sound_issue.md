another temporary fix for acer 722 when using skype and built in mic. Firstly I had to run terminal, type: alsamixer, select correct card sound using F6, rise the mic input to 100%.

then every time i run skype and have a video chat, I open sound preferences and disable than reanable mic input for it to work.

bit annoying, but a simple workaround
