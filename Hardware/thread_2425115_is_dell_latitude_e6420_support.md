### Post by TheFu on 2019-08-20
Dell is one of the most compatible vendors when it comes to Linux.

I don't know of any computer that will run Linux that also requires Windows. Ubuntu is a full operating system.  Any Linux distro is a full operating system - no Windows needed.  Often, having Windows is a complexity that isn't desirable.

No way to know which works the easiest, but you should always install only LTS releases if you aren't a developer or don't have some very specific need that lasts under 6 months.  LTS releases have 5 yrs of free support.  Non-LTS releases have 9 months of support from the release date.  18.04 is the current, newest, LTS.  For an older laptop like that, I'd steer you towards Ubuntu-Mate which has a nice GUI and should work nicely with not-high-end GPUs.

The best way to see how easy an specific computer will be is to create a bootable flash drive with the versions of Linux you'd like to try out. Then boot off it in the "Try Ubuntu" mode which runs entirely off the flash drive and won't save anything.  See how it goes. Check the networking, sound, video playback, mouse, touchpad, whatever. See what works.  Most computer stores don't have an issue if you want to do this with their floor models.
