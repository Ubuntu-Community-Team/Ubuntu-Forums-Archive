### Post by Yellow Pasque on 2008-08-24
Well, you're using a generic VESA video driver. You'll need the VIA display driver for widescreen resolutions. The easiest way to do that would be to use the last repo version made for Feisty here: [http://packages.ubuntu.com/feisty-backports/xserver-xorg-video-openchrome](http://packages.ubuntu.com/feisty-backports/xserver-xorg-video-openchrome)
Make sure you get the dependency packages as well.

Alternatively, you could build the latest version from source. Open the terminal (Applications -> Accessories -> Terminal) and paste the following commands:
```
sudo apt-get -y install build-essential cdbs fdupes intltool libdrm-dev libpixman1 libpixman1-dev x11proto-fonts-dev x11proto-gl-dev x11proto-xf86dri-dev xserver-xorg-dev libxvmc-dev libxrandr-dev libxrender-dev libgl1-mesa-dev libgl1-mesa-dri libgl1-mesa-glx libglu1-mesa libglu1-mesa-dev
wget http://www.openchrome.org/releases/xf86-video-openchrome-0.2.903.tar.bz2
tar -xjf xf86-video-openchrome-0.2.903.tar.bz2
cd xf86-video-openchrome-0.2.903
./configure
sudo make
sudo make install
```

Then, you should be able to reconfigure your xorg.conf and choose the openchrome driver with:
```
sudo dpkg-reconfigure -phigh xserver-xorg
```
