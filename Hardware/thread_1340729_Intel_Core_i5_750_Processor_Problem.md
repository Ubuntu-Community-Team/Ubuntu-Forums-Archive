### Post by prshah on 2009-11-29
> **marryjohnson said:**
> all of a sudden it started taking forever to upload files. Now the problem my computer get freeze up during startup. 

I suggest you check the processor fan, Internal fan, case cooling fan, smps fan and GPU fan (if you have one). I think you have a problem with overheating. 

I doubt that it is a processor problem, and I would not suggest you swap processors.

You can also run a memory check by using the memtest option in the grub menu.
