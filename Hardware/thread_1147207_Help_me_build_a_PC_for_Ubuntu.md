### Post by soldar79 on 2009-05-03
Although, ATI drivers have been developing quite well so far, Nvidia drivers are better. So, only thing you'll must get is an Nvidia Card.
I'm using a very old one on my Ubuntu Box (Nvidia fx5200 128mb) and what can I say... it's amazing. I can use compiz full fx activated and is smooth as silk.

So far, I've experienced some issues with a particular ASUS mother (m2-mx or something like that) which has been solved since 8.04, the one i've got installed these days. Anyway, ASUS, GIGABYTE, MSI are my choices. There's no big deal about compatibility speaking about desktop boxes. The big issues come when we speak about notebooks and netbooks where things get (sometimes) really messy.

Speaking about RAM... well, I only own 1 gb of ram, which is the minimum these days and I can use Ubuntu with compiz full fx and everything works flawlessly, including running sometimes a Virtualbox with virtualized XP in it. And some games, linux native and W$ native through WINE.

So, go and get the best PC you can afford (no need to say that... hehe). What is amazing about Linux is that every buck you spend goes to POWER instead of "system requirements" (you know what I mean.. M$).

### Post by pbpersson on 2009-05-03
I just built three computers for Ubuntu

The motherboard and CPU (AMD X2 64) together was $100.

The CPU cooling fan was $50, 4GB of RAM was $40 and you can get a one terabyte drive for $100.

The video, sound, and network interfaces are all on the motherboard.

That is my kind of system.

Be sure to check the [Ubuntu hardware guide]("https://wiki.ubuntu.com/HardwareSupport/") so you have components that run with Ubuntu
