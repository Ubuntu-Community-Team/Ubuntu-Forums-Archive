### Post by oldfred on 2016-01-27
With nVidia you just about have to use the proprietary graphics. 
Some Linux users do not like binary blobs and want pure Open Source software. 
Older nVidia cards usually work with the open source drivers and nVidia is starting to help the open source developers a bit more.  

And the very newest nVidia may have restricted drivers.
There was some nVidia cards that  could support 4 cards, but since Windows nVidia driver only worked with a max of three on those cards, nVidia had to modify Linux driver to only allow 3.  This was a while back so may not be an issue now.
