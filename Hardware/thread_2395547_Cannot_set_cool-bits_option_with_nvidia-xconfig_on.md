### Post by alexdumi29 on 2019-06-18
Thank you so much! I finally figured it out. (I have a prime enabled laptop) You have to modify /usr/share/X11/xorg.conf.d/11-nvidia-prime.conf to contain option "Coolbits" "28". And now you can overclock or do other stuff. (I find green with envy to be the best oc tool (I use the flatpak on mint 19.1))

P.S: If you have a desktop i think you should do a sudo nvidia-xconfig -a --cool-bits=28 and then copy the xorg.conf from /etc/X11/ to /usr/share/X11/xorg.conf.d/ and change it's name to 20-nvidia.conf (make sure to delete xorg.conf from /etc/X11 ) (To work with these files I find it easier to do a sudo nemo or sudo nautilus and then go to /)

Hope I've been helpful
