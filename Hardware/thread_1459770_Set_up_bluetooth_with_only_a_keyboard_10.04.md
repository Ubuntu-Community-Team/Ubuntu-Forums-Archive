### Post by gannggstaz on 2010-04-21
I recently upgraded to ubuntu 10.04, and the new bluetooth manager requires granting access to devices. Both my keyboard and mouse are bluetooth, and I could not find a regular usb mouse. I have a keyboard, but when I tab over to grant access it doesn't connect and soon shows me the notification again. I have connected these devices before and had to go into the bluetooth menu to configure them. 
Does anyone know how to start the bluetooth managers interface? When I run it through terminal, I get a message saying that it is already running.

### Post by ponderz on 2010-05-02
I had the same issue on an upgrade, go to the bluetooth menu **System -> Preferences -> Bluetooth**

Then click on **Make Computer Visible** checkbox it probably did not get checked on upgrade scenario.

You can also setup new device here too, and then the mouse should appear.
