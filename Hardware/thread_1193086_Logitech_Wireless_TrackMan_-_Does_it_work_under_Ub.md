I have an older model Trackman; the Trackman Marble FX Cordless. Before my upgrade to Jaunty I got all the buttons working by editting /etc/X11/xorg.conf, but since the upgrade this configuration has been taken over by HAL. 
I've done some research on how to get this working again, and finally did it with some help from a bugreport in Launchpad. 
I've written a small howto on my blog, not sure if it could help you because you have a different model, but it may point you in the right direction.
