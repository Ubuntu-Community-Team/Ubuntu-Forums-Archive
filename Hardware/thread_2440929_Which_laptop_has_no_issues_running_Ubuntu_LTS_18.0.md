### Post by ping-wu on 2020-04-17
Although Ubuntu 20.04 is still in the daily-build stage, it is much more stable than 18.04.  The 20.04 version is much more hard-compatible, especially with respect to AMD Ryzen hardware.  Once the official release becomes available, you can quickly update to the release version.  

We have recently purchased two new laptops: HP Envy x360 15.6" Touchscreen 2-in-1 Laptop (AMD Ryzen 5), and Lenovo IdeaPad FLEX14 (also AMD Ryzen 5), both work flawlessly with Ubuntu 20.04.
