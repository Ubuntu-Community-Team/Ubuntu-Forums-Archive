### Post by Ad@m on 2010-05-03
You wont find any.

Linux in general is different to windows. The Linux kernel includes the drivers for most hardware, hence there is no need to download drivers.

However you may find certain hardware isnt supported, in which case you will have to wait until it is added to the kernel or file a bug report at [https://launchpad.net/ubuntu](https://launchpad.net/ubuntu)

Before installing Ubuntu, use the live cd feature to test it out.
