### Post by Flohrian on 2011-06-11
Maybe it's not helpful for you anymore, but for others, like me, who want to get this keyboard: I bought the Logitech K350 today and most functions work out of the box. Just inserting the unifying adapter und the keyboard will work; using ubuntu 11.04 with unity:
what IS WORKING:
+all functions that are also on a normal keyboard + numpad
+some special features like calculator, hibernating, web browser, mail client, media keys (including launching banshee, volume up/down/mute, next/previous song/stop, play/pause)

what IS NOT WORKING out of the box:
-camera button
-the three window switching buttons on the left side
-the fn+f-key combinations(word, excel,calendar,programmable keys)
the keys described as do not work, cannot be displayed using xev, so they will probably not work.

### Post by lile001 on 2013-02-27
I also have teh MK550, which is a combination of the Logitech K350 keyboard and M510 mouse.  Things work pretty well out of the box.  

1.  I booted up in Windows first, and installed the software ont he CD.  Apparently there is some other sofftware to download but I didn't bother.  

2. Booted into Ubuntu 12.04.  Basic keyboard functions work, along with some others:  volume control, play, stop, and next/prev song works in Banshee and Quod Libet.  special :search www" key brings up a "search file" dialog, not the internet.  Email key brings up thunderburd.  all the other special blue keys don't do anything I can tell.  

Special zoom key on the left doesn't work, nor did it work in Windows.  This might require the software DL, don't care anyway.  Paginate key doesn't work.  Camera isn't installed on my computer so I don't know if that key works or not.  Media Center and other special key on the top left (shows a "gear") don't do anything in linux or windows.  

Calculator key works!  Haven't tried the power \button yet.  

I boughht the keyboard for being wireless, and for the shape, not the special bells and whistles.  Worked out of the box in Ubuntu 12.04.  Nice!  Mouse also has no issues.
