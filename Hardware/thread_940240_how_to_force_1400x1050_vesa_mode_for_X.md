### Post by vanwarantion on 2008-10-06
did you try envyng utility? it is agreat simple gui tool to automated install ati or nvidia drivers. You can simply install envy by typing:

```
sudo apt-get install envyng-gtk
```


you can also try this howto: [http://ubuntuforums.org/showthread.php?t=246746]("http://ubuntuforums.org/showthread.php?t=246746")


if these ways doesn't solve your problem, copy-paste your */etc/X11/xorg.conf* file here.
