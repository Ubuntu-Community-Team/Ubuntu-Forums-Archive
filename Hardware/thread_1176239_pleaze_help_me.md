### Post by coffeecat on 2009-06-02
I think you mean the 1692 WLMI, not 1962. If you've upgraded the RAM to 1024MB (if that's what you mean) it should run the standard desktop version of Ubuntu just fine. In fact, with the original 512MB of RAM, Ubuntu will run.

Download the 32-bit 9.04 Desktop from here:

[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

Burn the ISO file to CD. Instructions here:

[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

Make sure the BIOS on your laptop is set to boot from CD/DVD first and boot up the desktop CD and run Ubuntu live from the CD to see if everything works. Wireless may or may not be a problem depending on the wireless chipset. Your laptop uses the ATI Mobility RADEON X700 graphics chipset, so that may or may not be a problem too.

If everything works OK, then you can use the CD to install to the hard drive. Be warned that everything is sluggish when running from the CD. It's much, much better when installed to the HD.
