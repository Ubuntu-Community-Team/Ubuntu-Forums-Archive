### Post by ibjsb4 on 2013-11-14
There are options like Clonezilla, Gparted, DD.

[http://www.googlubuntu.com/results/?cx=006238239194895611142:u-ocqbntw_o&q=clone+hdd&sa=Search&cof=FORID:9](http://www.googlubuntu.com/results/?cx=006238239194895611142:u-ocqbntw_o&q=clone+hdd&sa=Search&cof=FORID:9)

You can also make a copy of your installed packages and just add them to a new install with [Synaptic Package manager]("http://www.googlubuntu.com/results/?cx=006238239194895611142:u-ocqbntw_o&q=Synaptic+Package+manager&sa=Search&cof=FORID:9").

[http://www.googlubuntu.com/results/?cx=006238239194895611142:u-ocqbntw_o&q=markings+list+synaptic&sa=Search&cof=FORID:9](http://www.googlubuntu.com/results/?cx=006238239194895611142:u-ocqbntw_o&q=markings+list+synaptic&sa=Search&cof=FORID:9)

[In terminal]("https://help.ubuntu.com/community/UsingTheTerminal#Starting_a_Terminal"):

```
sudo apt-get install synaptic
```
