Spent hours trying to find an answer but I couldn't find something satisfactory, so I figured I could ask here.

Also, when I tried to check for proprietary drivers to see if they would fix it, the drivers window displayed nothing except "No additional drivers available." I was a bit confused because I couldn't find any information on what xorg edgers *is, *or whether it was something I actually wanted to connect to or not. It seemed to be maintained by Ubuntu devs, so I tried it. It's not a fix that inspires a lot of confidence, but it worked, for others with the same issue.
