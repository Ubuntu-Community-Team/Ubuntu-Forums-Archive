### Post by Kira Vakaan on 2011-10-24
Hi there,

Just today, actually, I was able to fix my Battery (not present) problem on my Acer Aspire 5745G by updating the BIOS.  If you haven't already, I recommend going to Acer's site and downloading the most recent BIOS updater for your computer.  Make sure it's the right one, as I'm sure it's easy to brick your computer by doing it wrong.  I had to run the updater from Windows.  Good luck!

### Post by Kira Vakaan on 2011-11-06
@ajjw09: Well, for my model, Acer gave me two executable files: one for Windows, and one for generic DOS.  Since I was on Windows, obviously I just used that.  However, it should be possible for you to use Ubuntu to prepare the update using the DOS version, assuming Acer gives you the same options.  You should check out instructions [here]("http://ubuntuforums.org/showthread.php?t=318789").  Never used them, but they make pretty good sense to me.  My understanding is that you need to somehow (i.e. a cd) boot from a DOS image and run the DOS version of the updater from there.
