### Post by alexsah on 2008-06-10
Hi there,
I've been reading these forums quite regularly as I just have started using ubuntu at home, and it is mostly great!

I have been having one rather irritating problem, when I use the left and right arrow keys with my Logitech S510 keyboard, they jump two spaces two the left or right depending on the relevant key of course. Now this is where it gets weird, it doesn't do it with ANY other keys, just the left and right arrow keys, and I mean it doesn't even do it if I disable num-lock and use the keypad to move left and right, it is also fine. I tested it in the bios and it has no problems, only when logged in to ubuntu.

Does anyone have any ideas on what I could do? If I could clarify further please let me know. As one would imagine it can be stressful moving through a text document.

### Post by alexsah on 2008-06-13
I've found a fix to the problem. I don't know how well this would work, but it would work for any keyboard that has this problem, you just have to go to Keyboard Preferences -> Accessibility and then go down to Bounce Keys, and check the box "Ignore fast duplicate keypresses".

If anyone finds a better fix please do let me know.
