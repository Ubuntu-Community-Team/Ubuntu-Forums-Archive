### Post by cascade9 on 2011-01-25
Wow...

You dont need to install the BIOS. The motherboard should come from the factory with an installed BIOS (if it didnt, the motherboard would never boot, so you couldnt install a BIOS.....catch 22). You dont need to run any software for a video card to work in BIOS. 

You should be able to install a video card, close the box, boot it up, and start installing your OS.
