### Post by dclabaut on 2022-10-12
The issue is reproduced with the AX201 chip on kernels 5.15.0-50 (Ubuntu 20) and in Ubuntu 22.

Workaround found, for both the 9560 and AX201: It's possible to delete the recent firmware (> 55) files from /lib/firmware to fix the problem:

```

sudo rm -fr /lib/firmware/{iwlwifi-QuZ-a0-jf-b0-59.ucode,iwlwifi-QuZ-a0-jf-b0-62.ucode,iwlwifi-QuZ-a0-jf-b0-63.ucode,iwlwifi-QuZ-a0-jf-b0-66.ucode,iwlwifi-QuZ-a0-jf-b0-68.ucode,iwlwifi-QuZ-a0-jf-b0-71.ucode}
sudo reboot

```
