Recent versions of Flash player for Linux can also take advantage of these cards to use hardware acceleration for video.

### Post by MoreOrLess on 2012-02-08
I would go with an FX-4100: [http://www.newegg.com/Product/Product.aspx?Item=N82E16819103996](http://www.newegg.com/Product/Product.aspx?Item=N82E16819103996) along with corresponding AM3+ mobo (Either 970 or 990FX chipset).

Using 3 RAM sticks will disable dual-channel operation (triple-channel is only used for high-end Intel CPU's). Also, RAM's dirt cheap, so you can get an 8GB kit (2 x 4GB) for about the same price as the kit you picked out: [http://www.newegg.com/Product/Product.aspx?Item=N82E16820231428](http://www.newegg.com/Product/Product.aspx?Item=N82E16820231428)

Finally, I would pick a 7,200RPM drive if it's being used for the system disk. 5,400RPM drives are fine for storing media/photos, but they're noticeably slower for system tasks.
