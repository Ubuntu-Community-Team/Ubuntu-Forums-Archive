### Post by grahammechanical on 2014-08-21
Will Ubuntu run on an Intel Atom CPU?

[http://www.ubuntu.com/certification/catalog/component/dmi/dmi:Intel(R)Atom(TM)CPUN2600@1.60GHz/](http://www.ubuntu.com/certification/catalog/component/dmi/dmi:Intel(R)Atom(TM)CPUN2600@1.60GHz/)

Will Ubuntu be able to use all the hardware of a tablet? That is another type of question entirely.  That particular tablet has a 3D acceleration sensor and a gyroscopic sensor. We don't have those sensors on desktop machines. So, I would not expect Ubuntu desktop to be able to use them.

Ubuntu for phones and tablets is being developed for the ARM powered devices of Nexus 4, 7 and 10. We will not be getting the convergence of Ubuntu phones/tablets into Ubuntu desktop until the release of Ubuntu 16.04.
