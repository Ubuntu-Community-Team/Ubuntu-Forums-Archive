### Post by Mark Phelps on 2014-03-20
In my experience, as I have been doing this for years, the OS really has little or nothing to do with overclocking, so it really doesn't matter which OS, and which version, you are using.

Overclocking CPUs is generally done using either of two approaches:
1) Modifying detailed BIOS settings
2) Using an app provided by the motherboard manufacturer

Motherboard manufacturers like ASUS and Gigabyte have been released very sophisticated apps in recent years to support the second approach; unfortunately, these only run in MS Windows. Which means, you'll be forced to use the first approach.

Also, I would think that for Gaming, you would want to seriously look into overclocking the GPU, not just the CPU.

You should check the following sites for sound advice on overclocking: [www.hardocp.com](www.hardocp.com) , [www.overclock.net](www.overclock.net), [www.overclockers.com](www.overclockers.com)

Good Luck
