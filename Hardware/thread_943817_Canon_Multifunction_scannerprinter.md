### Post by phidia on 2008-10-10
If you have sane installed (which it usually is by default) then open a terminal and enter > sane-find-scanner

The sane site has a search engine for scanner models [here]("http://www.sane-project.org/cgi-bin/driver.pl"). 
You can check there to see if your scanner is supported and there will be howtos for getting it working if it is.

The Ubuntu wiki also has some [help on scanner set up]("https://help.ubuntu.com/community/ScanningHowTo?").
