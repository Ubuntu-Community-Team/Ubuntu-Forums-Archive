### Post by dandnsmith on 2011-04-17
I wasn't suggesting accessing the BIOS setup - that wouldn't (unless very unusual setup) help. What you need is the manual, for description on how to effect various changes via function keys (especially keys with different coulour than main keys) - all too often it is merely hints, which have to be read carefully, and interpreted.

Replacing icons on the top bar can be done by right clicking on the bar, slecting 'add to panel', and picking one from the dialog. You may well have to select 'create custon application launcher' and then find out what has to go in the subsequent boxes (I haven't researched it, but did it once when mine disappeared).
