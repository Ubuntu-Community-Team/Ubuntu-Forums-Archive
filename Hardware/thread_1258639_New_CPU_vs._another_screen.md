### Post by fela on 2009-09-05
I'm going to be adding something to my PC this christmas - so far I've got two options: one, a new screen, or two, a new CPU.

The new screen would be 24 inch, 1920x1080 (or 1920x1200) with an HDMI input. Right now I'm running two 19 inch monitors as a twinview configuration, and I have a spare HDMI output on my graphics card that's just dying to be used.  I don't know if you get diminishing returns switching from dual- to tri-monitors, in terms of productivity.

The second option, the new CPU, would be an AMD quad core Phenom II X4 940, vs my current AMD Athlon X2 6000 dual core. I often have things like compiling software running in the background while I want to do other stuff, and I know that the new CPU would be very much better than my current one.

This is what I do alot of, from *loads* to not SO much:
1) Web browsing and checking email. Of course my current system is fine for this :D
2) Editing _large_ images with the GIMP, the new CPU would have a big advantage with processing large images, but a new screen would let me have more images on screen.
3) Compiling large software packages, such as the KDE desktop environment and Linux kernels. Of course the CPU is the obvious candidate here.
4) Games that need lots of CPU power such as MS Flight simulator and Grand Theft Auto IV. GTAIV would definitely benefit alot from a new CPU.
5) Of course, all these things at once :lol:

So yeah, what do you think would be money better spent in terms of productivity and performance - a new screen so I have 3 monitors (and a virtual resolution of 4800x1080 :P) or a new, quad core CPU, keeping my current two monitors?

I'm leaning towards the CPU as of now. I don't even know if I'll be able to afford a new screen (more expensive than CPU), with the budget I'm on now :o
