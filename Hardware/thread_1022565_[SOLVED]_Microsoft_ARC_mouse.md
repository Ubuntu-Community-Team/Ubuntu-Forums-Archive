### Post by CJay554 on 2008-12-27
GOT IT! 

Apperantly the Microsoft Arc mouse automatically saves settings based on the OS being used. In this case i used this mouse on Windows then went onto ubuntu without unplugging the mouse. The settings from Microsoft totally dismantled my scrolling, but all you have to do is unplug it, and plug it back in into the computer when on ubuntu! The Microsoft Arc(and many other models of Microsoft mice such as [tags]: Microsoft Intellimouse Explorer 2.0 model 1007, Microsoft wireless optical mouse 2.0; found at: [https://answers.launchpad.net/ubuntu/+question/9200](https://answers.launchpad.net/ubuntu/+question/9200)) share this same configuration method.
