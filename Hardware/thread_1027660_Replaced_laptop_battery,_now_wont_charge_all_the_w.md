### Post by 2hot6ft2 on 2009-01-01
If you duel boot...boot into windows and check to see if it's the same there.

If it's reading right there it should do the same in ubuntu. If it's reading the same then it's not ubuntu but the battery or charging system.

While it's plugged in slide the battery out a little then stuff it back in. Worked on my Vaio
