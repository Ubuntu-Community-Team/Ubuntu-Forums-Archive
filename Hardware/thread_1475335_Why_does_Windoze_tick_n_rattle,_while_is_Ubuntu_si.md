I'm sure this is something to do with the ways the two different OS's access the hard drives - but I have wondered for a while:
When the PC is processing something why does windows XP make a loud rattling, ticking noise and Ubuntu is usually silent or occasionally a slight rattle or hum?

---

### Post by sgosnell on 2010-05-06
Probably because the Windows files are badly fragmented, so the heads have to move all over the platter to find the data, while the Linux files are mostly contiguous.  Fragmented files cause a lot of hunting and clunking around.
