There is an option to boot into, "Dell Utility Partition", but this is just an option to do some tests, and is not the same thing as the recovery partition, that wipes the rest of the drive clean, and reinstalls Windows, restoring the computer to the condition it was when it shipped from the factory.

### Post by saveanescapist on 2011-05-18
Generally, when you install linux from a live CD or something of that nature, it overwrites the entire hard drive, including the Dell recovery partition. I made the same mistake, but if that's the case it's gone forever. 

I recommend downloading the disk utility, if you haven't already, and doublechecking that it still exists. If it does, then I'm not sure how to help you, I'm afraid.
