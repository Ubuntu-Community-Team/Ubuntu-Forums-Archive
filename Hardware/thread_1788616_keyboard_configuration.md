I just bought a Logitech MK320 keyboard/mouse combo.  It works fine in 11.04.  It has some hot keys that don't all work.  I am wondering if there is any way for me to pull up a map of the keyboard and re-program the function of these keys.

### Post by SoFl W on 2011-06-22
I have the MK260.  (MK=Mouse/Keyboard)  My keys function, but I went to System-> Preferences-> Keyboard-> Layouts-> and selected "Logitech" and then just the basic wireless model.  (I don't know if yours is listed)

I also went to System-> Preferences-> Keyboard Shortcuts-> Desktop and that is where I adjusted the keyboard buttons.  (I disabled the power key)
