### Post by MugenDraco on 2011-02-02
I just built my first desktop, and it was working fine for about 2 weeks, then this morning I go to turn it on, and it starts for about half a second, then quits.  I quickly found that the cable connecting the power supply to the motherboard was loose, so I fixed it.  Now the computer comes on and stays on, but the monitor isn't getting a signal, and neither is the keyboard.  I have the monitor connected via HDMI to the graphics card, and the keyboard via USB.  It worked perfectly before this morning, in both Ubuntu and Windows.  If it helps, I've posted my specs below.

CPU: Intel Core i7 950
Motherboard: ASUS Sabertooth X58
Graphics card: MSI TwinFrozr II GTX 465
Memory: Ripjaws 12GB (3 X 4GB)
Monitor: ASUS VH222H 21.5"
Keyboard: Saitek Eclipse II
PSU: RAIDMAX HYBRID 2 RX-730SS 730 watt
Case: Xion Predator
