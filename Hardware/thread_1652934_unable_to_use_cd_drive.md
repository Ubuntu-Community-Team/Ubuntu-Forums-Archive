### Post by Habeouscorpus on 2010-12-25
Hi there! Old people rock :)

Try looking in System> Administration>Additional Drivers to see if maybe you are missing one. If there are ones that have grayed out circles next to them, enable ONE. ONLY. (I made the mistake of doing them all. It's messy.) If one doesn't work, try another. If none work, see if you can download one from the HP website.

Now about the programs. A lot of programs don't work on Linux, sad to say. A program called Wine can fake Windows so some programs can run. It's a bit of a hit and miss. [http://appdb.winehq.org/](http://appdb.winehq.org/) has a list of programs that work, and how well they work. If all else fails, try scoping online for a Linux program. Linux users have to pay taxes too, you know. Someone's probably made one.
