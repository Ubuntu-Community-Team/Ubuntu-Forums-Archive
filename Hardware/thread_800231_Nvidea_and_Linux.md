### Post by RAOF on 2008-05-19
You should probably be aware that there isn't yet a release of an nVidia linux driver that supports the 9600 yet.  You would need to use a beta driver, so all the nice tools we have to make using proprietary drivers easy won't work for you.

This is likely to be the case for **any** cutting edge card you get; either nvidia or ati.  ATI might be a better bet for a system you'll want around for a long time - they've released the specifications (and are providing other support) for their cards, so the open-source drivers should improve fairly rapidly.
