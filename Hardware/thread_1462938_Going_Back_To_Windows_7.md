### Post by razy60 on 2010-04-26
if you need to reformat then you can use your ubuntu install cd(live cd) start up your laptop and select in bios to start from cd then when the live cd loads select gparted in system-administration, choose your hard drive and reformat to ntfs, remember this WILL delete all your data and settings, once you have completed this you can then load you windows 7 disc and install.
If all you need it for is the ipod and a few other progs then i would dual boot.
Once you have installed 7 then reinstall ubuntu split your hard drive to something that is acceptable so 50/50 or 70/30 you dont need to make the ubuntu partition huge if you dont want to as it can easily access the windows partition, that wy you have the best of both worlds.

hope it helps
