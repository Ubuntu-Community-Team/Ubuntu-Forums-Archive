### Post by Apollo87 on 2010-02-23
I have an HP mini 1000 running Ubuntu 9.10 desktop and it runs perfectly fine.  It may not boot as fast as the netbook remix, but thats a small price to pay.

As for resolutions, I believe the max resolution the HP mini can handle is 4096 x 4096, so you're in the clear. You may need to use the xrandr terminal command to manually set it though:

[https://wiki.ubuntu.com/X/Config/Resolution](https://wiki.ubuntu.com/X/Config/Resolution)
