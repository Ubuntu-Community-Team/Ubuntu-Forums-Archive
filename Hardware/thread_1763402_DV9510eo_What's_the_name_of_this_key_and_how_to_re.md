### Post by jramshu on 2011-05-20
In windows, and from what I can tell has the same effect on Ubuntu.

[IMG]http://www.seoconsultants.com/windows/tips/images/windows-application-key.gif[/IMG] The Application key has an image of a mouse pointer on a menu (between the Alt and Ctrl keys ([IMG]http://www.seoconsultants.com/windows/tips/images/windows-application-key.gif[/IMG])  to the right of your Space Bar). Depressing this key will display the  selected item's shortcut window. This is the menu that is displayed by  right-clicking.
