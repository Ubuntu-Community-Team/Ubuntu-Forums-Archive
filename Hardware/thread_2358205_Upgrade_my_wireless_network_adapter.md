### Post by kurt18947 on 2017-04-12
> **stribor45 said:**
> I need to upgrade  my laptop network adapter so I can connect to 5Ghz network not wireless router :)

If you're planning to replace the internal card, are you aware of Lenovo's BIOS white listing where only 'approved' wifi adapters will work? If you did want to replace the internal card, you should be able to download the maintenance manual which will list the wifi cards that will work. If you're planning a USB adapter, there's no BIOS white listing issue. I've seen some indications that  manufacturers are going away from the white listing but check before buying. Supposedly BIOS/WiFi device pairings came about because some manufacturers felt it was necessary to get/maintain FCC approval. 

I'm no help as far as which models are best.
