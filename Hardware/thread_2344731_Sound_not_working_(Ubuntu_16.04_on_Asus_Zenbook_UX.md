### Post by leoliberti on 2016-11-28
I just bought an Asus Zenbook UX390U, and installed Ubuntu 16.04 alongside the existing installation of Windows 10 (dual boot). As far as I can tell, everything substantial works out of the box but for the sound -- I just tried playing an MP3 file using mplayer and heard nothing but weak static, with speakers and headphones.

Referring to [https://ubuntuforums.org/showthread.php?t=2340639](https://ubuntuforums.org/showthread.php?t=2340639), which raises the same issue, I'm posting a new thread following Bucky Ball's suggestion. He also suggests,

> As for sound, what are you using to tweak it, what have you done to  experiment to fix it? Have you got Pulseaudio Volume Control installed?  If not, install it. Get some music playing with 
> PAVControl open. In the  'Playback' tab, can you see the audio stream bouncing up and down near  the slider?  Go to the 'Output' tab. See the stream there? Above the stream you will  see 
> 'Port' with a drop-down menu. Start selecting devices from there or  select the right device to output if you know it. Matter of  experimenting because you've given no idea of your setup. 
> Would certainly help if you did

I ran the described experiment. I have two "output" tabs: internal speakers and headphones. With both I could see the audio stream bouncing up and down near the slider, but heard only weak static on both. Absolutely no music at all, even raising the volume at maximum. I observed that the volume of the static noise appears to be independent of the volume slider level.

I considered myself a Linux expert up to version 2.2. I have long ceased being an expert system user, so I wouldn't know what further experiment to devise to help understanding the issue. On the other hand, I can certainly run some experiments (even advanced ones) if they are explained in a sufficiently detailed way, and report the results back.
