Thanks a bunch! However, i think the **LCD on/off switch ( Fn+F8 ) ** you mention may be a monitor switch, meaning it probably cycles between lcd and vga/hdmi outputs. If so, i'm very interested if you could try connecting a hdmi monitor (or tv) and make sure it works without the nvidia card activated (i.e. running on the integrated IGP only).
The reason ask is because that many hybrid graphic solutions (especially using ATI GPUs) hardwire the digital outputs (hdmi/displayport/spdif audio) directly to the discrete GPU, which is problematic since most people tend to (or must) run linux off the intel IGP :/

I intend to run any noteook i purchase, connected to a external monitor via HMDI as my primary monitor. If you don't have the oppurtunity to test this, maybe someone else will at least :D
