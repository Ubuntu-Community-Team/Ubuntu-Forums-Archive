### Post by snowpine on 2008-06-17
Older computers, unfortunately, do not have the option to boot from a USB device. You'll need to use the internal CD drive (if you have one). However, you can use Wubi ([http://wubi-installer.org/](http://wubi-installer.org/)) to install Ubuntu from within Windows! Lots of threads here on the forum if you need help with Wubi (I've never used it myself).

Good luck.
