### Post by tea for one on 2020-09-28
A [COLOR="#000080"]live[/COLOR] session is pretty straightforward when you have done it a few times.

The first time can be a bit daunting.

Can you enter into your UEFI set up pages to organise your boot priority?

When you are in a live session, it is advisable to check:-

[LIST]
[*]Access to your internal drives via the file manager
[*]Sound output
[*]Ethernet and wireless connection
[*]Screen resolution
[/LIST]
if you have difficulty with preparation/booting of your usb device, please start a new thread with a succinct title because you'll attract more support from some very knowledgeable and experienced forum members.

Good luck
