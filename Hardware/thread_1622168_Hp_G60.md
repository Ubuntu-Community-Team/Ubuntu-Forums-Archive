### Post by pytheas22 on 2010-11-15
Please post the output in Ubuntu of:
```

lspci -nn
lshw -C Network
uname -a
dmesg | grep -e ath -e wlan -ie radio
ifconfig
ifconfig -a
sudo iwlist scan
```

That way we will have all this information in one place and can hopefully start figuring out what's not working for you.

It's unlikely that the device would work any better in Ubuntu than it does in Xubuntu, since they both have the same backend (the difference between all the different *buntus really just comes down to the interface they use).
