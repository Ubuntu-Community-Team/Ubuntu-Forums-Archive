### Post by pricetech on 2010-03-15
Sounds sort of like a "frankenproject".

You'll need some form of device that you can dial from your Ubuntu box that will send a ring to the fax machine.  At least that's the way it's designed to work.

You might be able to manually start receiving the fax, but you'd still need talk battery.

Beyond the physical layer connection, I know Ubuntu has some fax software, but I've never used it so I can't say how well it works.  You'll need that also.  Hopefully it will just create a virtual printer, but again I've never used it so I don't know.
