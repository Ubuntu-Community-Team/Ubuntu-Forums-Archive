Hi.. I have a dual screen setup running on ATI 5970's. The default restricted drivers don't handle dual screen setups very well.

I'm trying to have one wide desktop similar to how you would get if you had a windows setup with two screens side by side and one long desktop of 3840x1200.

I installed the new ATI drivers that comes from the ATI website (11.4) and got them sitting side by side...

### Post by PhonicLynx on 2011-04-29
OKAy.. I found the bug... If you some how install the "Restricted Drivers" first then install the ATi drivers you break the system because by the looks of it the install package does not uninstall the restricted drivers correctly.

I found this out by wiping the partition and reinstalling from scratch, copied the 11.4 drivers back to my home folder, making it executable, stopping GDM, and running the package.

Dual screen with Unity is quite nice too :)
