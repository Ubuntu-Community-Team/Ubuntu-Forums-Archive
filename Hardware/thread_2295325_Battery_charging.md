### Post by Bucky Ball on 2015-09-19
Those batteries need to be charged and discharged to stay alive. It has been flat for a year or two, I'd send it to the recyclers and get another. Looks dead. 

Easiest way to tell is to see if it is charging. If it is, how long til fully charged? Once charged, then take the cable out. How long til flat.

You would need to fully charge this, if possible, before pulling out the cable. If you pull out the cable and the battery dies then, dead.
