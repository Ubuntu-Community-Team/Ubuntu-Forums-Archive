### Post by mgranet on 2009-08-08
It should be good to go. I can't see how an operating system would have trouble with the AM3/DDR2 combo, since the memory controller is hardware. If you can see it in the BIOS, the OS can use it. 

For a budget build, you did very good. Ubuntu should run faster than a jackalope (pun intended) on it. XP too; even better if you have XP 64bit.

EDIT: The RAM choice you have made is for Intel chipsets only. :(
