I have a T42 thinkpad with 1 GB RAM and I want to upgrade. The plan is to simply replace the current memory stick with a 2GB stick.  Does it matter whether I install DDR or DDR2? If it matters, how can I see which I should go with?  Thanks! :D

### Post by ironclaw on 2010-02-17
DDR and DDR2 memory modules are not compatible with each other - you always have to buy the correct type that your computer supports. From searching online, it seems that the T42 has DDR memory, so don't buy DDR2 memory sticks.

Also make sure that you buy DDR memory modules in the 200 pin SO-DIMM form factor, which is physically smaller than the DIMM form factor memory modules found in desktops.
