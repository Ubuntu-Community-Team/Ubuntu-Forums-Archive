### Post by hyper_ch on 2009-04-12
best to download the live cd and test it with that. If it runs on the live cd then it runs on the real install also. You should have at least 512 mb ram to for running kde 4.x at an acceptable speed (IMHO). Pure requirements are lower.

And pretty everything is ported to 64bit. There's an adobe flash 10 64bit player but you need to install it manually. You ahve also a java 64bit plugin (icedtea).

Best, download 64bit desktop cd and test it.
