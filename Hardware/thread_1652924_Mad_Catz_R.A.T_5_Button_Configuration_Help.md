### Post by anduweb on 2011-04-07
So far (April 2011), there is not support for linux concerning their ST (Smart Technology) programming software.You need that software in order to program the buttons on the mouse.

That software is only available for windows and in order for your RAT to work with the ST you also need to install the drivers which are also just for Windows...
So unless you can crack up something else...you won't be able to 'map' anything....just the default will work... Right+Left+Center(scroll wheel) clicks. Scroll wheel and back and Forward buttons. DPI change switch..

Another thing you can do if you're not satisfied by the DPI values you can plug it into a windoze machine and chance the DPI values to what you want and save them (in ST off course)...and they remain the same because those values are written in the firmware of the mouse and will stay like you set them even if you un-power the mouse..

Hope it helps...
