### Post by adijumi on 2010-11-03
I found the solution - it's dumb simple. Yesterday I tried to open 2 files stored on that disc using windows 7. I couldn't, so I ran chdsk and it found them to be corrupted and deleted. And now Ubuntu reads it fine.

I think there should have been some way to do it from inside Ubuntu - probably there is, but I did not find any.
