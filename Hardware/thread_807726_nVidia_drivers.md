### Post by Abu_Dayya on 2008-05-26
You have to install propietary nvidia drivers. You can do that by

1 - Go to System -> Administration -> Ristrected Drivers. You'll see a list of hardware on your computer that require a ristrected driver. Then enable the driver for your graphics card. This will download and install your driver.

2 - OR, you can try an application called 'envy' that you can use to download nvidia drivers. Google it for tutorials on how to use it.
