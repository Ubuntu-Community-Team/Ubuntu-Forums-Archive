HP 1020 LaserJet printer
Here's how to install the driver of a B/W LaserJet printer made by HP.  Just follow the steps in the terminal

Install printer driver
Connect your printer to your PC, and make sure to switch it on.  Open the terminal and copy/paste the following command :  hp-doctor

Ubuntune automatically to HP via the terminal: Follow the steps.  Need an active Internet connection; Answer Yes and all.  That's it !
Make a print test page: Printer Test Page UBUNTU will be print with Job ID and driver version: F002ZJS-.PPD / 1.1
Should work for everyone using the latest version of Ubuntu.
