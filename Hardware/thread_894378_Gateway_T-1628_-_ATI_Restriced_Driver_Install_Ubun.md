### Post by CowboyPecos on 2008-08-19
Got it!!! After several days of playing with the various 'versions' of installation instructions scattered over the net all resulting in failure w/WSD & BSD, I finally decided to give the [http://wiki.cchtml.com/index.php/Ubuntu_Hardy_Installation_Guide]("http://wiki.cchtml.com/index.php/Ubuntu_Hardy_Installation_Guide") a try. I had read several places about using it and my response was, "Oh what the heck, nothing else had worked."

I'm happy to report that following the pertinent instructions exactly as given (no 64bit, errors:not needed, no tweaking or other fluff) produced positive results. The laptop is now running on the ATI 8.15 driver. 

I can not report on any software which uses it. My reason for posting is simply to let you T-1628 with built-in X1200 series ATI card users know that the driver does work, and the above link will get you there.

The instructions declare that you might see under Admin-->Hdw Drvrs where the ATI driver shows to be "In Use" but not ticked. Upon checking, I found mine In Use AND ticked. "fglrxinfo" reports everything as ATI.

Hope you have the same positive results!
