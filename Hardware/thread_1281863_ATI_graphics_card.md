### Post by boballen55 on 2009-10-05
I currently use an HD4650.  So long as you get the latest driver from ati's site it works pretty well.  I've missed the last couple updates but the driver I have now does everything pretty well.  There are minor issues such as when entering and leaving the full screen setting in a movie the desktop image blinks for a half second and then it fixes itself.  So it works but just isn't really pretty.  The newest version might have fixed those things...  If you haven't done it before get the linux driver from:

[http://support.amd.com/us/gpudownload/Pages/index.aspx](http://support.amd.com/us/gpudownload/Pages/index.aspx)

And follow the directions on this page:

[https://help.ubuntu.com/community/BinaryDriverHowto/ATI](https://help.ubuntu.com/community/BinaryDriverHowto/ATI)

All that has worked well for me.  Good Luck!
