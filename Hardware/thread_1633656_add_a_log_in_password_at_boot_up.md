### Post by sisco311 on 2010-11-29
Go to System -> Administration -> Login Screen and disable automatic login. 

If you forgot the password, boot in *recovery mode* and set up a new one:
[http://psychocats.net/ubuntu/resetpassword](http://psychocats.net/ubuntu/resetpassword)
