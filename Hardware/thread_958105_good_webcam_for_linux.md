### Post by ruien on 2008-10-25
The best webcam is probably a Phillips SPC900N wich is supported out of the box on any recent linux distro. It has a CCD sensor wich makes it much better than other cam's in poor light. Some use it for recording star activity at night. It is not a cheap webcam but if you want on off  the best affordable cam's arround then this is it. 

Some review's here: [http://www.amazon.com/review/product/B000EIIGYU/ref=dp_top_cm_cr_acr_txt?%5Fencoding=UTF8&showViewpoints=1](http://www.amazon.com/review/product/B000EIIGYU/ref=dp_top_cm_cr_acr_txt?%5Fencoding=UTF8&showViewpoints=1)

A few spec's

90 frames per second video
Takes 1.3 megapixel (1280x960) resolution snapshots

Tip: Get a camerafoot for it if you buy it.
