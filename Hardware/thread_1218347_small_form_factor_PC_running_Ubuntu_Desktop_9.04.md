### Post by kendoori on 2009-07-30
I have an MSI Wind ([http://www.newegg.com/Product/Product.aspx?Item=N82E16856167032](http://www.newegg.com/Product/Product.aspx?Item=N82E16856167032)) that I'm using with 8.10, but it would have no issue running 9.04 (I'm just too lazy to upgrade). I have it Ubuntu running 100% off of 4GB CF card. This is a very low power machine (and quiet), and pretty small, but not tiny. I'm using this box as a streaming media server and as a NAS.

I also have a an ASUS EEE Box ([http://www.newegg.com/Product/Product.aspx?Item=N82E16883220012](http://www.newegg.com/Product/Product.aspx?Item=N82E16883220012)), which I initially had 9.04 on, but now am running Mint. This is quite small, and I actually have it mounted to the back of a 20" LCD TV in my kitchen.

You might be able to buy the EEE Box at a local store, but the MSI box probably not.

You could build the MSI machine with a 40GB Hard drive and RAM for less than $200. The EEE Box will cost you more, but it's a complete system and comes with a 160GB drive.
