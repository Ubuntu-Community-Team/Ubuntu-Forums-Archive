Hi there. Got heaps of questions? Fantastic, you've come to the right place. These fora are a wealth of knowledge, and I'v found everyone here is willing to help a new Ubuntu user along the path.

> 
How to access and download new drivers and especially my internal webcam and my wireless printer does not work, is there any tweak or work around for that?


You are likely going to need to download drivers for your hardware. Simply go to System > Additional Drivers, and it will show you what drivers you need to download to optimise performance. A list of hardware compatibility lists can be found [here.]("http://www.linux-drivers.org/")

> 
For a new ubuntu user can someone give me some tips and tricks to customise this OS, and do I need to write a lot of coding.


Like to customise? Then Ubuntu is for you. You can customise this OS to almost any specifications. I would recommend looking at [this thread]("http://ubuntuforums.org/showthread.php?t=1772639&page=22") which has a bunch of users' screenshots uploaded. If you see something you like there, as pm the user and ask how they got it set up. I'm sure they'll happily tell you. With customisation, you probably want to look at installing Compiz Cinfiguration Settings manager, and maybe Ubuntu tweak. These let you customise little things about Ubuntu. As for needing to write code? Not at all, atmost all customisation is done through a graphical user interface (GUI), meaning it's just clicking. Some customisation does require you to use command line interface (CLI), but if you need help, most people here are ready to give you the commands to type.

> 
Where can I find the list of applications like windows start menu as I would like to know what application are installed and the way to open them? And is there any main menu?


There is indeed a menu in Ubuntu. I've not yet used 11.04 (I prefer 10.10, and it runs a lot smoother on my laptop), but I know that you can access installed programmes from the Dash when you click on the Ubuntu logo in the top-left corner.

> 
I'm trying to access ubuntu documentation to check what I can do with this OS especially anything different from windows and how I can switch gradualy to this OS.


Like I said, this forum is an excellent place to learn about Ubunutu. Reading [tutorials and tips,]("http://ubuntuforums.org/forumdisplay.php?f=100") or [other peoples experiences]("http://ubuntuforums.org/forumdisplay.php?f=103") can be very helpful. Other than that, check out [this site]("http://omgubuntu.co.uk/") for some cool Ubuntu stuff, or the Ubuntu homepage itself.

> 
Thank you.

No worries, welcome to Ubuntu and the Forums!

### Post by Bucky Ball on 2011-06-24
For printer, if you are using 64bit, go here:

                                       [B]                                                              [                                                                                                                                    Printer Driver without JRE for x64-bit Debian Package Manager based Distros                                                                                                                   ]("http://support.lexmark.com/index?page=answerlink&userlocale=en_US&segment=SUPPORT&locale=en&url=http%3A%2F%2Fsupport.lexmark.com%2Findex%3Fpage%3Dcontent%26id%3DDR20852%26actp%3Dsearch%26viewlocale%3Den_US&productCode=&answerid=16777225&searchid=1308891750243")                             

[/B]If you click 'More Details' you will see this driver works for Ubuntu (as Ubuntu is based on Debian, a point you should remember for the future).

If you are using 32bit, go here:

[http://support.lexmark.com/index?page=answers&startover=y&locale=en&userlocale=EN_US&question=pro900](http://support.lexmark.com/index?page=answers&startover=y&locale=en&userlocale=EN_US&question=pro900)

... and select the appropriate 32bit drivers. There is generally a 'Read Me' file in the download that should tell you how to go about installing. Good luck. ;)

Just a tip: Always check your device manufacturer's website to see if there are any existing Linux drivers. Many drivers are provided in the Ubuntu release also and one way to find out if you are already covered is to plug the device in, switch it on, and see if Ubuntu responds.

PS: You're welcome for the help; that's what it's all about 'round here. :)
