### Post by cascade9 on 2010-01-30
That would be an 'out of frequency' problem. 

I would guess that its set to use a higher frequency than the monitor can handle (eg its set to 1280x1024 @ 85Hz, but it can only do 1280x1024 @ 60Hz)
