### Post by Miguel on 2009-01-24
Exactly that one. As far as I know, there is no faster DDR memory than 400 MHz. I don't think your system would support DDR2 memory (faster and less power hungry), so if you want to get a fast memory you may be interested in getting two identical 1Gb sticks (or two 512 Mb sticks) and run them in dual channel mode.
