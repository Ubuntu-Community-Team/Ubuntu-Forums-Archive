### Post by DapperMe17 on 2008-08-30
Solved!

[COLOR="Red"]Added...[/COLOR]

/dev/sda5 /media/External1 ntfs-3g defaults,[COLOR="Red"]users[/COLOR] 0 0
/dev/sda6 /media/External2 ntfs-3g defaults [COLOR="Red"]users[/COLOR] 0 0

I made sure I [COLOR="Red"]added[/COLOR] directories for External1 & External2 ([COLOR="Red"]sudo mkdir /media/External1[/COLOR]
) & ([COLOR="Red"]sudo mkdir /media/External2[/COLOR])

Then made sure to [COLOR="Red"]give permissions[/COLOR] to both ([COLOR="Red"]sudo chown -R USERNAME /media/External1[/COLOR]) & ([COLOR="Red"]sudo chown -R USERNAME /media/External2[/COLOR])

I also [COLOR="Red"]edited[/COLOR] ([COLOR="Red"]sudo gedit /etc/rc.local[/COLOR])...[COLOR="Red"]adding[/COLOR] 
[COLOR="Red"]mkdir /media/External1 && sudo mount /dev/sda5 /media/External1  
mkdir /media/External2 && sudo mount /dev/sda6 /media/External2
[/COLOR]


Seems to have solved the problem & hope this may help others.
