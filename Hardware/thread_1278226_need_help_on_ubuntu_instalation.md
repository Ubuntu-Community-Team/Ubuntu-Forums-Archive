### Post by AlanQ on 2009-09-29
The alignment of the read-write heads varies from one CD/DVD drive to another.
This is rarely a problem with professionally created discs like an XP install disc, and usually only shows up when you try to play/boot a disc created on another drive.

Annoying as it is, the next thing to try is to take a CD-drive from a machine that does boot from your Ubuntu CD-R, and install it in the problem machine.
