### Post by shatterblast on 2009-07-07
You might want to find out if you have the Microphone enabled.  You should have it listed in the Sound window if it exists.  You can find that in:

System -> Preferences -> Sound

The bottom window should list the various devices.  Something should say "Capture" or "Microphone" there.

I haven't tested the Remix version of Ubuntu for notebooks, but it might possibly have the Microphone disabled by default for saving battery power.

Also, you might check out the following for DVD and digital media playback:

[http://www.medibuntu.org/](http://www.medibuntu.org/)
