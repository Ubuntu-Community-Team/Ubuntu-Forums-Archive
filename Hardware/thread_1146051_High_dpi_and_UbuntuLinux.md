### Post by apocalypse80 on 2009-05-02
I'm using a 15.4" laptop with 1680x1050 resolution (~128 dpi unless my math is wrong) and all works well in gnome and kde (haven't tried xfce).

All you really need is size manipulation, particularly for fonts.
Any os and any desktop manager will give you those.
It takes a few more changes than in windows (separate for gtk,qt3 etc), but it works just as well and I find the font smoothing to be much better looking in ubuntu than in windows.

The sole linux-only issue I have is web page viewing.
100% zoom = tiny pages, I need ~120% for comfortable viewing yet anything between 100% and 200% results in VERY ugly images - no matter the desktop, no matter the browser.
