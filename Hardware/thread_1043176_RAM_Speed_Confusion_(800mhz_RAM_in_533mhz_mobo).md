DDR2 type ram is backwards compatible with other types of DDR2 ram. So it should work, as long as the motherboard supports the amount of ram.

### Post by mcduck on 2009-01-18
> **cb951303 said:**
> it will probably work but at 533 MHz of course.

Definitely not at 533MHz.. :D

The internal speed of DDR2-533 is 133MHz, the external speed is 266MHz. The number in the name comes from the fact that it can do 533 million data transfers per second.

DDR2-800 runs internally at 200MHz, with 400MHz bus speed and 800 million data transfers/second.

But yeah, you can usually replace DDR2-533 with DDR2-800 without any problems.
