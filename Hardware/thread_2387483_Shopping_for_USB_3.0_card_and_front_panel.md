### Post by rbmorse on 2018-03-19
IF the motherboard has a socket for a USB 3 front panel, just about any should work with Linux. Dell sell's 'em if you want to go that route. 

The Windows driver that comes with the expansion cards I've seen is for versions of Windows before Win 7 (?) that did not support USB 3 in the original file set.
