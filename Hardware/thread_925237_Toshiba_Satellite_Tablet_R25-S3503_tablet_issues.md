### Post by spiritedviolinist on 2008-10-18
It's really simple.

Edit you x-server configuration file. On the command line execute the following.

sudo gedit /etc/X11/xorg.conf

Scroll down and you should see something like this:

```

Section "ServerLayout"
	Identifier	"Default Layout"
	Screen		"Default Screen"
#	InputDevice     "stylus"	"SendCoreEvents"
#	InputDevice     "cursor"	"SendCoreEvents"
#	InputDevice     "eraser"	"SendCoreEvents"
	InputDevice	"Synaptics Touchpad"
EndSection

```

Remove the pound signs (#) as they comment text out and there you go.

### Post by Favux on 2009-12-29
Hi Jagosix,

Welcome to Ubuntu forums!

In Jaunty and Karmic you no longer use a xorg.conf.  What you use instead is a .fdi file.  In your case a 10-linuxwacom.fdi.

Make sure in Synaptic Package Manager that xserver-xorg-input-wacom is installed.  It should be by default.  Also install wacom-tools if it isn't already installed.  If you search 'wacom' both packages will show up.

Go to the [linuxwacom HOW TO]("http://ubuntuforums.org/showpost.php?p=6546012&postcount=1") and download one of the two .fdi's attached to the bottom.  Instructions for installing are in Section 2 b).  Calibrating and configuring with wacomcpl is described in Section 3.

Hope this helps.  Good luck!
