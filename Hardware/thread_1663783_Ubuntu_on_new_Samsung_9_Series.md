### Post by syncopatod on 2011-04-22
I have a new 900X3A, and attempted to install Natty Beta 2 from a bootable USB flash drive with the live image. It didn't work. I had the following three problems:
1. It seems that you have to boot using the USB port on the right. I couldn't get it to boot using the USB port on the left.
2. If there are no partitions, install fails. Workaround: create some partitions and let the installer erase them, if necessary. It seems that the installer doesn't like too much free space.
3. This is a show-stopper: The installer kept crashing at "Installing the 'grub2' package". ([https://bugs.launchpad.net/ubuntu/+source/ubiquity/+bug/768051](https://bugs.launchpad.net/ubuntu/+source/ubiquity/+bug/768051))

I retried using a live image from the daily build from Apr 22, and it seems to work. I'm typing this from my new Natty system.

However, I would like to get the touchpad multitouch to work. I'm still looking into that.
