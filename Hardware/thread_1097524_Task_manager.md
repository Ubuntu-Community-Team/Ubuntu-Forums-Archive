### Post by cariboo on 2009-03-16
Top is a much better choice, as sytem monitor adds it's own overhead, so you are not getting a true reading of ram and cpu usage. To run top, open a Applications-->Accessories-->Terminal and type:

```
top
```

at the prompt. Press q to quit.
