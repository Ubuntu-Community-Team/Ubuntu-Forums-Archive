I have an Aspire 5740DG with an ATI Mobility Radeon HD 5650. Now, I've just installed ubuntu 10.04 and so far everything is silky smooth, I'm able to get my resolution to a nice 1366:768 but when I try to use the special effects, like the desktop cube, it tells me that it is unable to process. ### Post by Al.hefe on 2010-11-18
NVM found the solution. 
System > Administration > Hardware Drivers

It simply loaded up my ATI graphics chip and told me that I was fortunate enough to be able to download the drivers for it free of charge and gave me the option to do so. I've done it and my desktop special effects are running perfectly. Hope this helps someone else in the future...

Thank god to curious tinkering!
