### Post by grahammechanical on 2022-10-02
If we purchase a laptop with Ubuntu pre-installed then we get a standard version of Ubuntu with possibly the addition of specific hardware drivers for that machine. These drivers will be part of the Linux kernel. Such a modified kernel is know as an OEM kernel. Read about it.

[https://wiki.ubuntu.com/Kernel/OEMKernel](https://wiki.ubuntu.com/Kernel/OEMKernel)

The best thing for you is to put Ubuntu on a USB memory stick and boot Ubuntu on that stick and then choose Try Ubuntu. This way you can test Ubuntu to see how it works with your hardware before you install it. Keep in mind that there are different flavours of Ubuntu that use other desktop environments that are less demanding in hardware requirements than Ubuntu is with the Gnome desktop environment.
