### Post by wd4nzx on 2012-11-02
I suggest a wireless solution.  There is an app in the play store: WellFTP Server, that when run on the A500 turns it into an FTP server.  Now using any FTP (FileZilla?) software on your Ubuntu machine and following directions on the A500 screen, you can move files in either direction.  It isn't as fast as a USB connection but very reliable.  The one assumption is that you are both on the same network, not say one on the normal network and the other on the GUEST network.

I have an A500 and it has only been connected to a computer once in the past year, I was thinking of rooting but ended up resisting!:)

Give it a try, I find it quite freeing and it really takes less time to setup and transfer files.
