### Post by Mike_Walsh on 2015-12-14
I have been using SanDisk Cruzer 'Blades' with Linux file systems for quite some time now. They work exceptionally well with Puppy Linux , Knoppix, and Tiny Core; all of these use a delayed write sequence, to cut down on the constant read/write operations that tend to kill USB sticks eventually, when used with a traditional, 'full' install.

I'd recommend them to anybody. Most SanDisks are reliable with Linux filesystems (ext2, ext3 **and** ext4).
