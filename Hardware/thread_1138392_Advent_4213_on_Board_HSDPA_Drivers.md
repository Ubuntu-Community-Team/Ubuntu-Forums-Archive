### Post by anaz86 on 2009-04-26
hi. i'm one of new advent 4213 user as well, and i also use the onboard HSDPA. I think you need to make sure you turn on the built-in HSDPA by pressing fn+F2 together and the wireless led turned on. please note that the HSDPA & WiFi use the same led. personally I turned off the Wifi & HSPDA until the led is off & turn back on the HSDPA so that i can make sure that the HSDPA is on. Ubuntu will automatically recognise the HSDPA & will connect to the appropriate operator (Orange, Vodafone, 3, etc) automatically. I'm using Orange mobile broadband though, so i'm not sure about 3 mobile broadband whether they allow us to use the built-in HSDPA or not. 
Gud luck
