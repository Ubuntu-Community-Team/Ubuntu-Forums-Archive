### Post by Crucias on 2011-04-29
That illumination is because the default profile is stored on the mouse itself.

You will find that without the special drivers Ubuntu only recognizes default buttons common to many mice.

Please review this [Support page]("http://faq.steelseries.com/questions/19/Where+can+I+get+World+of+Warcraft+MMO+drivers+for+the+Linux+platforms.").

Also it may be possible to use [this page]("https://help.ubuntu.com/community/ManyButtonsMouseHowto") and i will be trying it in a few weeks after University exams conclude.
