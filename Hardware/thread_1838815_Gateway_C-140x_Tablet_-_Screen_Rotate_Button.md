### Post by Favux on 2011-09-04
Hi sjennings11,

Welcome to Ubuntu forums!

For a tablet PC you should look at CellWriter (has onscreen keyboard mode), Xournal, and EasyStroke.

The buttons depend on whether you can detect key codes from them.  See the [Rotation HOW TO]("http://ubuntuforums.org/showpost.php?p=6274392&postcount=1") for some more information.  Things changed with Lucid and I haven't quite finished figuring everything out.  But there should be clues there that are helpful.
