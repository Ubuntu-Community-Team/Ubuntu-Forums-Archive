### Post by Knightwise on 2015-02-14
I installed 14.04 on my surface pro. (I got rid of the windows install altogether)
With the latest 64 bit version everything is detected out of the box (even the wifi)
I've installed the latest updates and hardly have any problems.
My issues
- Sometimes my type cover isnt recognised at bootup. A quick reboot fixes that but its a little annoying.
- My wifi card works but sometimes it just won't connect to my router (or any router for that part) also : A reboot mostly solves the problem.

As for the rest , dont forget to install lmsensors (sudo apt-get install lmsensors) to make sure you get the proper cooling for your cpu.
