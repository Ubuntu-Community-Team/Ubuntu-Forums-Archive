### Post by Yellow Pasque on 2014-04-02
> I was planning to upgrade to the stable upcoming trusty tahr, but the Ubuntu one, instead of the Lubuntu
Your RAM is more than enough and the processor should be okay if it works for your current install. The main difference is that Unity needs 3D power, so you can test that out right now with a Live USB of Trusty. I don't think I would go through the trouble of doing a full install in a separate partition (especially if you're not comfortable with resizing partitions), but it would be a more thorough test.
