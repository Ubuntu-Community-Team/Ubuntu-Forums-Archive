### Post by InspiredIndividual on 2009-01-21
> **pumber said:**
>  How can I check I have the above fonts installed ? and any other in my snaptic that are Chinese ttf ??


In Synaptic, just type the name of the font in the quick search box. For example, type ukai.ttf and press enter. Now, check the search results. The first search result is the ukai font, which can be seen quite easily from the explanation listed. If the small square next to the package is green, it's installed. If it's empty, it's not installed. You can then install the package(s) by ticking the checkbox and clicking 'Apply' afterwards.
