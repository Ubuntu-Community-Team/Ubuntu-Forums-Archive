### Post by dannyboy79 on 2012-11-05
shift should bring up the grub menu, then you need to edit the grub line and add nomodeset after quiet splash. not sure if you can make this a permanent change since it's a usb stick install so you may have to do it after each boot unless you made it a persistent boot and if that's the case then you need edit /etc/default/grub and look for the line GRUB_CMDLINE_LINUX_DEFAULT="quiet splash" and make it read 
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash nomodeset"
and then run sudo update-grub

good luck
