### Post by newbie-user on 2015-01-22
Ubuntu Server is Ubuntu, but without the graphical user interface. So you would rely on command line to do all your stuff. Most server tutorials will have you working in the command line anyway. I suggest you give it a try. If you don't like it, you can always install the full desktop on top of the server install. Here's a tutorial on installing the GUI after Ubuntu server is already installed: [http://www.htpcbeginner.com/install-gui-on-ubuntu-server-14-04-gnome/]("http://www.htpcbeginner.com/install-gui-on-ubuntu-server-14-04-gnome/")
