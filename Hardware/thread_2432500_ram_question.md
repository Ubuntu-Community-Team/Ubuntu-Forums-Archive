### Post by Frogs Hair on 2019-12-03
2rx8 means that the memory modules are on both side of the card and not compatible with 1rx8 for voltage reasons. Staying with the same mhz is also a good idea unless you know minimum and maximum range for the device. You might find this on a site that has detailed specs for that computer. Some sites that specialize in selling memory may have a list of equivalent memory cards.
