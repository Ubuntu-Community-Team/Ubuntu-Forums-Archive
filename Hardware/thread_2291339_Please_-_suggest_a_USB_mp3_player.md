### Post by CantankRus on 2015-08-22
Any USB mass storage class(msc) player should work with linux.
I recently bought a couple of cheap mp3 players off Ebay from China. $AUS 1.95 ea(free post) needs sdcard.
Battery lasts about 3hrs and you can only manually cycle through songs/podcasts with no other features.
ok as a backup player but wouldn't use all the time.

Currently use a [**_[COLOR="#B22222"]sansa clip sport[/COLOR]_**]("http://www.sandisk.com/products/music-video-players/clip-sport/") that sells on Ebay for about $US 50.00
I use for listening to podcasts which I sync using gpodder.
You don't need to use software with an msc player. You can just drag and drop files onto the player. 
Features of sansa clip sport:
[LIST]
[*]fast-forward rewind
[*]remember last position
[*]radio
[*]timer
[*]25 hrs battery 
[*] good in-ear headphones supplied
[/LIST]

Worth the cost in my opinion if you use an mp3 player a lot.
