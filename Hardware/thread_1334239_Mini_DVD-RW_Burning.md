Hello,

maybe, I say maybe... the mini dvd-rw is already burned as an iso in such a way that it cannot be opened | viewed. If is that so, first it has to be reformatted, meaning to become blank. For this reason open k3b, choose Tools and then Format/erase.

There is also one other possibility: that the dvd tray is old enough so as to be able to open and read the contents written on the dvd-rw, which in turn were written to it from a newer dvd tray.
