### Post by dandnsmith on 2011-04-18
A crossover cable is advisable, in case of port intolerance (most routers and switches don't care, and adjust themselves)
Then you need to set up the connection between them, and something to handle routing of the traffic (since I assume the unstated bit is that you want to access the internet via the one machines wireless?).

I usually do this by hand, but I've been at it for a long time.
If you search the forums, I seem to remember that there was a posting covering setup of such connection a while back (probably since feb)
