### Post by smani on 2009-02-26
Which mirror you choose is not really relevant, those closer to where you live might be faster. Concerning which version, it depends what type of CPU are and how much ram you've got. Essentially:

x86 (32bit): Supported by practically all cpus, maximum of 2^32 bytes =4GB of address space can be addressed, which means that usually up to 3 GB of RAM are recognized by the operating system (the rest is simply "invisible" to the OS)

x86_64 (64bit): Supported by the newer CPUs, look at the specs of your CPU to be sure. 2^64 bytes can be addressed, i.e. you won't lose any memory if your laptop has more than 3 GB of memory, on the other hand, some proprietary software such as skype or other need some extra compatibility libraries to run as they are only released compiled for 32 bit systems.

### Post by thtrgremlin on 2009-02-26
A nice difference with Gnu/Linux distros, because there is nothing silly like a CD Key or other such things that make it hard for people that actually bought the software to install it, they can make things easy, like let you use the whole system from the CD. This lets you try the system, try it out, or just be productive while it is installing.

The two things that are sometimes problems (and these are problems with any OS install, Gnu/Linux or otherwise) is wireless. Specifically with Ubuntu, SOMETIMES sound, and in both cases, only on certain laptops where the OEM is really trying to save money, but I digress...

My recommendation, download and burn an iso as recommended above, and when it starts up, start firefox and go to youtube.com. It will tell you to install flash player, and follow their directions for ubuntu... assuming you connect to the Internet. If you can connect to the internet, and flash player installs and you get video with sound, you are likely very well set. The one other minor thing worth checking (but certainly no deal breaker) is recognition of and 3d acceleration. 

From the System menu at the top:
System->Preferences->Appearance. The fifth tab is "Visual Effects". Select the "Extra" option. As it mentions, you need a fast video card, but if you check it, and everything works, and it doesn't switch back to "none" or "normal", then you are very well set to go.

While Ubuntu / Gnu/Linux are "awesome" and have great potential for both system and user, and while I find it much easier to learn, it is very much like needing to learn everything you ever thought you knew about computers over again. Its worth it, but it is different, even more so than Mac and Windows from a learning perspective, in my opinion.

If you want to wipe your system and just go with Ubuntu, there is an option for that, but, no offense, if you are pretty new to this, and particularly if new to installing OS's, pick the dual boot option. Just as there can be problems installing Ubuntu, Windows has its own mess of problems installing too that are different. If for whatever reason you might still need Windows, it will be there, however often it might be necessary, but dual boot instead of wiping everything will certainly mean that if you have a problem with your Ubuntu install that you would need to turn around and deal with a Windows install. For me, I used a dual boot for about a year before I managed to clean my brain of old bad habits and was comfortable with knowing how to do things the Ubuntu way. Then, with the release of 6.10, and the changes it brought, I now only have Ubuntu installed (plus some virtual machines, but that is another adventure).

Have fun, and good luck!
