### Post by sergiom99 on 2008-10-18
well you can install cheese to use the camera:
```
 sudo apt-get install cheese
```

and to see the model/maker of your webcam, type this command in a konsole shell: ```
sudo lsusb
```

with that info you can search for your webcam specific data in the forums.
