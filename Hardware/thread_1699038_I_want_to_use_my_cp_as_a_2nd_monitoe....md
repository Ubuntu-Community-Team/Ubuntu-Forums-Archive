### Post by Trooper_Max on 2011-03-03
I'm not familiar with the HP All-in-One hardware, but I'm guessing there isn't a VGA cable for the monitor... and breaking one of those open probably isn't fun either, for most people anyway :D

I'd recommend trying out Windows XP Mode on the Windows 7 machine in order to run your old Windows XP games.

[http://www.microsoft.com/windows/virtual-pc/download.aspx](http://www.microsoft.com/windows/virtual-pc/download.aspx)

Alternatively, you could setup a VNC server on your Windows XP machine (like TightVNC). Then use your Windows 7 machine to connect to it. This way you would be able to view your Windows XP desktop in a window (which can be made full screen) on your Windows 7 machine. Or intead of this you might just be able to use Remote Desktop to connect, but not sure if this works between 7 and XP, but I wouldn't be surprised if it does.
