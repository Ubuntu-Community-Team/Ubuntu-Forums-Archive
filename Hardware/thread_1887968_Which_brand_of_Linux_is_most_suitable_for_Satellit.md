### Post by kurt18947 on 2011-11-28
If you have the capability, you could create some LiveCD/LiveUSB installs.  I would be primarily interested in two things.

- which interface feels best to me and which distros offer which interfaces.  Some people abhor Unity for instance.  A live install may not permit you to use any restricted drivers but they may show up as available in 'additional drivers' and you'd know restricted drivers are at least available.  The inability to use restricted drivers may not let you use Unity 3D or Gnome 3 shell for instance but Unity 2D may still work.

- perhaps more importantly, how is hardware compatibility?  Many distros have multiple desktop environments/windows managers.  For instance, in addition to Ubuntu there is Xubuntu and Lubuntu.  Linux Mint is a current darling and PCLinuxOS is not in the *buntu family but works pretty well.  If one distro has your WiFi and most function keys working from a live install, that would be my leader.
