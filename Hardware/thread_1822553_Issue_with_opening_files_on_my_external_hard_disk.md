### Post by MG&amp;TL on 2011-08-10
No problem (the connection (USB) was what I was asking about) 

If your machine is old, Docky is very unlikely to run as it requires a decent graphics card to composite the images and provide the 3d effects.

Dual booting is very simple. You simply set aside some space for Ubuntu (realistic minimum 5GB), then you are presented with an option on boot of whether to boot windows or Ubuntu. If you choose this option, report back for help. :)

If you are getting a new pc, don't throw this one away. Backup your files on the 40GB to the external, give the old pc a new lease of life with a minimal linux distribution. (I recommend Lubuntu, it runs literally 4.5 x as fast as Windows XP on my Pentium4)

Plug the external into your new pc, download the files across.

What model is your pc (mine's built around 2000 too)?
