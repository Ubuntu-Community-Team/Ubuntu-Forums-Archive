### Post by sanderd17 on 2011-03-02
A pentium II is a 32-bit processor, that won't start with a 64-bit OS.

btw, you have certainly not enough RAM to install Ubuntu.

Maybe Lubuntu will run on it, but I doubt it. You can always try [Damn Small Linux]("http://www.damnsmalllinux.org/") or [tiny core linux]("http://www.tinycorelinux.com/")
