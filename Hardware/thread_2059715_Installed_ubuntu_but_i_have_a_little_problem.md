### Post by ShadowVegan on 2012-09-18
Did you just close the video call window, or click end call? If you just closed the window perhaps that didn't end it. Otherwise maybe Skype is still using your webcam even if it is not being used in a video call. Try exiting Skype completely and see if it turns off, just to troubleshoot. If that doesn't work, try logging off of your user account (but keep your computer powered on) and see if that works. This could help pinpoint the problem. And like the previous poster said, Skype is proprietary so it will be harder to pinpoint the issue.
