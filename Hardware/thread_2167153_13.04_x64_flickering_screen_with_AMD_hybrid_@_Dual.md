### Post by claudiovcastro on 2013-08-19
Found a FIX, but i dont know how to turn it to set automatically.

To fix just type: " xrandr --output VGA1 --auto --left-of LVDS1 --output VGA1 --auto --scale 1.0001x1.0001 "


use HDMI1 if HDMI
after that command you can adjust the screen @ Displays ,  no more flickering.
