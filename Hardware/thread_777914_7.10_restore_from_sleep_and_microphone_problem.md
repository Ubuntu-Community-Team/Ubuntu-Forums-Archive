### Post by horton287 on 2008-06-13
I don't know if this is the same problem, but I am running Ubuntu on an Inspiron 1420. When I resume I often get a completely white screen. It turns out that I am actually at the screen lock screen and if I just type my password and hit enter the desktop resumes. Also, it may help to check out 
[http://ubuntuforums.org/archive/index.php/t-781153.html](http://ubuntuforums.org/archive/index.php/t-781153.html)
It contains some settings for the /etc/default/acpi-support file, significantly reduced the number of times I get a white screen on resume.

As for the problem with the microphone, have you run alsamixer from a terminal/command line. Hit F4 to see the inputs and adjust the input levels to get the best results. Cranking up the input levels on the digital input was a big help for me on skype.

Hope this helps.
