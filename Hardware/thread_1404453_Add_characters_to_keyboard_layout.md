After having finally managed to install Xubuntu on an old Sony Vaio with only limited RAM resources, here comes my next problem:

The laptop has a US keyboard, but it should be used in a German language environment, so it should be possible to write the German special characters (äöüÄÖÜß) on it. 

I have read some instructions about using a "compose" key , but I'd rather like to have these characters accessible using the "alt" key (<alt>-a for ä, or <alt>-<caps>-a for Ä, for example). Can this be done using a modified us keyboard layout file, and if so, how?

### Post by Enigmapond on 2011-04-16
Navigate to Administration/Language Support and add German to the system languages. Make sure you make the changes system-wide and check that the Keyboard layout also is set for German. You will need to reboot after. Hope this helps.
