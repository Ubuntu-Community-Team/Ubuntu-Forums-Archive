### Post by IcarusR on 2010-02-04
Normally one can do this, but 'Fn' key does not output normal keycode that can be read & so can not be modified.
Try with xev. Inastall & run from terminal. Press any key & compare results with those of 'Fn' key. You will see what I mean.

Lucid is a beta Ubuntu version, could always post on the dev forum.
