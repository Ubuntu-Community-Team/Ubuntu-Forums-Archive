### Post by PatrickVogeli on 2009-05-15
Are you going to use Virtual Machines? Like KVM, WMware,Virtualox?

I have an Intel T5450, which is a very low budget CPU, and I can use Virtualbox just fine... but then again I don't do much stuff there.. just Windows XP with microsoft Office when I get files which won't display that well in OpenOffice.

So, if you aren't planning on heavily using Virtual Machines I wouln't care much about the VT extensions....

Also keep in mind that some CPUs do support intel VT but the BIOS won't, so it's a useless feature. Check that first
