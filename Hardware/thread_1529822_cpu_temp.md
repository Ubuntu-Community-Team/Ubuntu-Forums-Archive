### Post by walle1357 on 2010-07-12
For the newest ubuntu version thing, try "update-manager -d" without the quotes. Keep in mind that update-manager is a graphical program. If there is a new version, you have the option to upgrade. If not, you only see packages that can be upgraded.

For the CPU temp sensor, try looking at [http://www.cyberciti.biz/faq/howto-linux-get-sensors-information/](http://www.cyberciti.biz/faq/howto-linux-get-sensors-information/)
