### Post by Kalanac on 2012-08-09
Ubuntu also comes on a minimal version:

[https://help.ubuntu.com/community/Installation/MinimalCD/](https://help.ubuntu.com/community/Installation/MinimalCD/)


This doesn't have the usual graphical install and requires a net connection during the install so it can fetch packages from the web.  You can select a the components you want to install (like the desktop).

Another alternative is lubuntu

[https://help.ubuntu.com/community/Lubuntu/Alternate_ISO](https://help.ubuntu.com/community/Lubuntu/Alternate_ISO)

There are torrent and web download options there.  Lubuntu is a "light" ubuntu.  Make sure to plump for the low ram options.
