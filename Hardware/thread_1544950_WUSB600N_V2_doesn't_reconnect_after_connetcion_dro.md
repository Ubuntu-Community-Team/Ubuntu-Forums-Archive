Netgear WNDR3700 appears to be supported by dd-wrt:

[http://www.dd-wrt.com/site/support/router-database](http://www.dd-wrt.com/site/support/router-database)

If you decide to try it, do not use the router database recommended version with first double checking dd-wrt forums to make sure it will work, the router database often recommends bad builds..
