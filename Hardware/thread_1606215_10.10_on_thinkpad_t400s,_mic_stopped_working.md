I updated recently to 10.10. I now noticed, that the mic doesn't work anymore. The funny (or lets say weird) thing about it is, as soon as I boot with the old kernel (i.e. 2.6.32-25-generic #44-Ubuntu SMP Fri Sep 17 20:05:27 UTC 2010 x86_64 GNU/Linux) the mic works again! Any ideas?

### Post by kktm on 2010-10-31
all right... solved.

open a terminal and run the following
```
echo  options snd-hda-intel model=\"thinkpad\" | sudo tee -a /etc/modprobe.d/alsa-intel.conf
```

taken from here (and slightly modified):

[Thinkwiki - Install_Ubuntu_9.10_%28Karmic_Koala%29_on_a_ThinkPad_T400s]("http://www.thinkwiki.org/wiki/Install_Ubuntu_9.10_%28Karmic_Koala%29_on_a_ThinkPad_T400s#What_doesn.27t_work_out_of_the_box.3F")
