### Post by tommcd on 2008-12-16
Before plugging in the usb hard drive run:
```
sudo dmesg -c
```
This will clear the dmesg file. Then plug in the usb hard drive and run **dmesg**. Hopefully you should get some info indicating that the drive was detected. Also, after plugging in the usb hard drive, run **sudo fdisk -l**. You should see the usb hard drive listed as /dev/sdb1, or something like that. If you do, you can mount it manually by making a mount point for it and mounting it like this:
```
sudo mkdir /media/usb_drive
sudo mount /dev/sdb1 /media/usb_drive
```
If this works you can make an entry for it in /etc/fstab and it should mount automatically on bootup.

If the system doesn't detect the drive as indicated by dmesg and "sudo fdisk -l" I'm not sure what else you can do.
