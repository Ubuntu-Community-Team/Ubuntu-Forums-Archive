### Post by lars-albertsson on 2016-11-24
Hi,

I have a Precision 7510 running Ubuntu 16.04. It has a 4K screen, which works ok. It flickers with KDE plasma, but I always disable plasma features anyway, so it does not bother me. I have the NVIDIA card, but use the internal Intel graphics. My impression is that NVIDIA has better Linux support than AMD, but I have never tried AMD, so take it with a grain of salt.

If you want to drive an external screen, that screen should be 4K capable as well, or you will never get sane fonts (I tried hard).

I bought the machine before you could get Ubuntu preinstalled, and the configuration that I have is not possible to buy with Ubuntu, but it works sufficiently well from 16.04. Earlier distributions worked poorly. I tried 14.04 once, and it barely worked at all for me. 

I still have some trouble with the LTE modem (also not officially supported by Dell on Ubuntu), but have found workarounds.

If you wonder why I bought a non-supported configuration, Dell claimed from the day the 7510 was launched that Ubuntu is supported: [http://www.dell.com/support/home/uk/en/ukbsdt1/Drivers/SupportedOS/precision-m7510-workstation](http://www.dell.com/support/home/uk/en/ukbsdt1/Drivers/SupportedOS/precision-m7510-workstation). Since there was no option to get Ubuntu preinstalled, I assumed that they would support if I installed a standard Ubuntu 14.04. They did not however, gave me no value for the support I paid extra for, and did not offer any compensation. I am not happy with the customer relation. I'll go back Lenovo next time.

If you can find a Lenovo P50 for a similar price, it might work better with Ubuntu. Thinkpads have traditionally worked well, although I have not heard any reports on newer models.
