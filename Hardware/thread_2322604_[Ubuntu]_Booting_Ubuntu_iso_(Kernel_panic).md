### Post by sudodus on 2016-04-29
Welcome to the Ubuntu Forums and welcome back as an Ubuntu user :-)

Maybe there was an error when you downloaded the iso file. Please check it with md5sum (and use the built in check at the boot menu, if you reach that far).

Maybe there are problems with a hardware driver. In order to suggest possible causes and solutions, we need to know about your computer. Please specify your computer

- Brand name and model
- CPU
- RAM (size)
- graphics chip/card
- wifi chip/card

-o-

You can try different versions of Ubuntu. Instead of 15.10 I would recommend the two versions with long time support, 14.04.1 LTS and 16.04 LTS.
