### Post by prshah on 2009-02-16
> **zy_guy said:**
> 
      (WW) VESA(0): No valid modes left.  Trying less strict filter...


That's because the openchrome drivers are probably not loaded (maybe not even installed). Currently you seem to be using VESA (generic) drivers.

Perhaps you can first try installing the drivers:```

sudo apt-get install xserver-xorg-video-openchrome
``` (or just [click here]("apt://xserver-xorg-video-openchrome")) then restart; hopefully the new drivers should be picked up automatically, otherwise post back on how to manually enable them. Also post your /etc/X11/xorg.conf file and the version of Ubuntu you are using.
