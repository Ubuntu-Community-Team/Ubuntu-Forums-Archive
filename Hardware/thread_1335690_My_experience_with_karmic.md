### Post by barrieluv on 2009-11-24
You don't say which Eee you are using, so it's difficut to comment.  9.10 woks perfectly on my 900 and has done since alpha 6 without fail.

Now, the Chromium browser isn't finished, so don't expect it to work perfectly.  Compiz, as nice as it is, doesn't perform well on these (somewhat underpowered) netbooks and running compiz and streaming Flash video at the same time is a bit of a no-go.  Personally, I always remove Compiz, but that's not for everyone.
A common thing, windows not fitting properly.  Hold the Alt key and drag the window to see what's off the screen.
And the Eee Control website doesn't mention a package for Karmic, so it's unfair to expect it to work.

Oooh, that's a difficult one for me to answer.
The Eeebuntu team are currently working on the new Eeebuntu which will have a new name and will be based on Debian Unstable.  I would strongly recommend waiting until the new release (late Dec.).
Read here:  [http://forum.eeebuntu.org/viewtopic.php?f=25&t=4492](http://forum.eeebuntu.org/viewtopic.php?f=25&t=4492)

The 1005ha seems to be problematic with the current release, although there are workarounds which work well if you don't mind a bit of fiddling with the command line and replacing the kernel and stuff.
Read here:  [http://forum.eeebuntu.org/viewtopic.php?f=7&t=3866](http://forum.eeebuntu.org/viewtopic.php?f=7&t=3866)

The Eee-control issue is best read up on starting here:  [http://forum.eeeuser.com/viewtopic.php?id=43975&p=30](http://forum.eeeuser.com/viewtopic.php?id=43975&p=30)

The intermittent flashing on streaming Flash isn't something I've come across and I would put it down to either Compiz or dodgy videos as my brother (1005ha + Karmic) doesn't have this problem (he doesn't have Compiz either).

So, I'm avoiding Jaunty, EasyPeasy (built on 9.04) and Eeebuntu 3.1 (also built on 9.04) and looking forward to Eeebuntu 4.0, Mint 8 (I'm running RC1 from SDHC) and Lucid Lynx.
I will start using Eeebuntu again when the new release arrives.  For now, I am more than happy with my Karmic install.
