### Post by MonkeyPaw on 2008-12-09
If it's a desktop, then you should be able to replace the fans yourself. You will need to measure the width of the old fan so that you can buy the right fan to replace it. Case fans come in sizes ranging from 60mm up to 120mm (yes, measure in metric). However, if the fan inside the power supply is dead, that's not an easy change if you're not pretty experienced, especially since electric shock is a real possibility when working with PSUs. You might be better off finding a replacement PSU for the machine. Dell uses a proprietary PSU, so it could take some searching.

Also, be careful using vacuums around electronics, as vacuums can produce static electricity. It's better to use compressed air cans to get the dust out.
