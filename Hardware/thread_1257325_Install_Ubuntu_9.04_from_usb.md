### Post by ujodarko on 2009-09-03
> **Cygoku said:**
> I guess I need to check the bios settings and version ?? 

Cygoku

You need to have USB memory stick with bootable Ubuntu.

First download Unetbootin.
Second download Ubuntu .iso file (around 700MB - from Ubuntu download area but choose ubuntu-9.04-desktop-i386.iso)

Now, start Unetbootin and there you select that .iso file and click on create or whatsoever...

After that you will have bootable USB that is able to install system on internal disk of yout laptop.

In BIOS you need to select USB device to start first to boot. It is usualy in "boot" section and there you must put your freshly made USB with Ubuntu on first place! Than save that change and reboot.

System should start from USB and after Ubuntu desktop appear - there will be "Install Ubuntu" icon. Double click on it and you may start with installation on internal HDD.
