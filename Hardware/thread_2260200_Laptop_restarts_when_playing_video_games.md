### Post by weatherman2 on 2015-01-09
70C is extremely hot for a laptop CPU, unless it is say a gaming or performance laptop designed for high performance.  It's bad for the laptop to run it that hot for a sustained period even if it can handle it for short periods. The heat can damage the components.

What model HP laptop?  Specs?  How old?  If more than 4-5 years old you should clean the heat sink and fan area to clean out the dust and dirt.  It's very common for dirt to get stuck in the heat sink area and cause the CPU to run hot.

### Post by weatherman2 on 2015-01-11
Not all games put the same amount of load on the CPU and graphics card.

An overheating CPU could either shutdown or restart.  It's not always the same symptom.  I have a little desktop box with a lousy fan and it occasionally overheats - rebooting the computer.

Are you saying it does sometimes shut off due to overheating?

Are you sure the fan is now working correctly?  

It appears this model is known to run hot.  Some people have found that applying new CPU paste (cleaning the CPU and heat sink first to get the old stuff off) like Arctic Silver helps lower the operating temps considerably:

[http://www.tomsguide.com/forum/62589-35-envy-heat](http://www.tomsguide.com/forum/62589-35-envy-heat)
