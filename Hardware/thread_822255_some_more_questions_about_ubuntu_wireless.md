### Post by avtolle on 2008-06-08
Compile means the same in Linux; essentially, making the text or whatever runnable. Under Ubuntu, after installing a program, the same is run in general either from a menu under Applications, for example (or a launcher on the desktop/panel), or by typing the name of the program at the terminal (can use Alt+F2 to get to a special window to launch an application from the terminal; if permission is needed to run the program, e.g., sudo, if the program has a graphical interface, gksudo is used in place of sudo). If a launcher is created in the panel, often a single click will run it; if the launcher is on the desktop, a double click on the icon is, as I recall, the default to run it.

Wifi radar is available in the repositories for Ubuntu; no need to run the Windows version under Wine.
