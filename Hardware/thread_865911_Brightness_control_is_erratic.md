### Post by tommcd on 2008-11-11
Try going to: system > preferences > power management. Set the brightness levels to where you want for both AC power and battery. You can also try unchecking "dim display when idle".

 It seems that, at least on my laptop, that Ubuntu wants to reset the brightness according to what is set in the "power management" section even after I set it manually with the function + arrow keys.
