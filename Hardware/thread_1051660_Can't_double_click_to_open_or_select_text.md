### Post by nonzer0 on 2009-01-27
I finally figured out how to change the single/double click options!  This is great to know, even if you're not stuck in my boat, but you can edit all kinds of preferences using this.  

Open nautilus by opening a terminal and typing in "sudo nautilus", enter password.  Then go to 'edit' and then 'preferences'.  This will open up the nautilus file manager preferences.  Click on the 'behaviors' tab and there you can change from single to double click and v.v.  

By the way, the file browser options really should be located in the control center.  Or better yet, the click options should be put under the option in the control center called "MOUSE."  Just a thought... Sometimes linux makes things a lot harder than they should be.](*,)
