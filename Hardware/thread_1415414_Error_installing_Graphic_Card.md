### Post by pgodw4 on 2010-03-06
If you open up the Ubuntu Software Manager, do a quick search for ati, or radeon, install the one driver support thing u see there, and it should work.

I know this because I have the same video card, and it worked for me. THe only downside atm is that there is the AMD unsupported watermark on the screen, which isnt really a big issue for me, I was more pleased about having compiz working and having native res going.

There are some issues with the driver, HDMI output causes my mouse cursor to disappear, but im sure all this will be fixed in due time as more drivers are released.
