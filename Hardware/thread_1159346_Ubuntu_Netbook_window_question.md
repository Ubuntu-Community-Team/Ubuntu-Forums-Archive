I think that's the whole point of the Netbook Remix! Simplify Windows so you don't have to minimize, maximize, etc. Everything maximizes automatically by default because a netbook's screen is so small.

If you want to multitask, just click on the Ubuntu symbol on the upper left hand side. That's like Netbook Remix's way of minimizing Firefox, so you can run another program. 

You see little icons on the upper left indicating that the programs are running, but are "minimized" (not actively displayed on the screen) until you click them which will "maximize" it. The small icon system practically replaces the traditional window functions (which I have found to be very innovative) 

Honestly, if you want a more traditional desktop, you really should be using the standard i386 Ubuntu 9.04 :)
