### Post by houseworkshy on 2012-09-04
I don't have a fix or an explaination of this but seeing one original post that's two weeks old followed by nothing but the OP's own bumps on this usually rather responsive forum bothered me so I'm bumping it for you.

I have a few idea's about the sort of info more knowlageable people happening by may like to see.  A sysinfo report would give an idea of hardware in one go, sysinfo can be installed from the software centre and it will put a link in your system tools menu. It's a program which is worth having anyway, I usually end up installing it sooner or later.  If you choose "save to file" it will produce a text document which has all your main system spec's.  Reduces answering questions about your hardware to a simple attachment. 
You could try out another user interphase such as LXDE, which can also be installed from the software centre, and see if that does/does not work.  Unity and Gnome3 share many dependancies which LXDE doesn't use so it'd be a valid test which could provide clues.  Installing LXDE wouldn't remove what's in already, it simply becomes another GUI choice at start up.
Not astoundingly helpful I know but a
