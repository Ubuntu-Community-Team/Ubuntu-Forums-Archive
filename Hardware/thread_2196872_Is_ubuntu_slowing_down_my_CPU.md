### Post by grahammechanical on 2014-01-01
I installed a little utility called GKrellM system monitor (it is in the Ubuntu softeware Centre). It shows me that my 2 core CPU rarely runs at full speed. In fact it shows that often only one core core will run at full speed and then only briefly.

Now, how long would a laptop battery last if its multi-core CPU ran at full speed all the time? It is by design of the CPU maker that they do not run at full speed all the time. It is done to reduce electricity usage and to reduce heat output.

By the way, my CPU is 2.4GHz but right now it is running at 1596 MHz. Call it 1.6GHz.
