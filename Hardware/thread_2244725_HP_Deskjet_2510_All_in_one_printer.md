### Post by Bucky Ball on 2014-09-18
As a matter of fact, HP give a very loud hoot about Linux users and have done for years. You need to install HPLip. You can even install that from the Software Centre with a couple of clicks. It supports literally thousands of HP printers. 

Once you have it installed, please go here if you need more info:

[http://www.hplipopensource.com/hplip-web/downloads.html](http://www.hplipopensource.com/hplip-web/downloads.html)

You will also find a ton of clues here:

[https://duckduckgo.com/?q=install+HP+Deskjet+2510+ubuntu+14.04](https://duckduckgo.com/?q=install+HP+Deskjet+2510+ubuntu+14.04)
