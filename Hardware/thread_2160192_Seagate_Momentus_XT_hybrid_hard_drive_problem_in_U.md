### Post by i2000s on 2013-07-05
Hi,

I am using a hybrid hard drive which has a SSD section and a normal hard drive section. I am experiencing some problem suspected to be related to this specific hardware. First, I was suffering a failure of installing Ubuntu beside a preloaded Windows 7 system in the hard drive. It is finally solved by change the property of some partitions. I don't know if anyone else also have this problem in this specific hard drive. Second, there is a file corruption problem which has been reported in another thread. I report this problem again, because the original thread is closed (if I remember it correctly), and I am using the lasted firmware (SD28) which has few follow-up report so far.

On the first problem I was sufferring, I summarize it here. I was attempting to install Ubuntu 12.04 in my Seagate Momentus XT hybrid hard drive which has a windows 7 installed already. Turns out, the installer cannot recognize my windows partitions! The hard drive has a 4G SSD cache to make the hard drive different from normal ones. I suggest our community to put some attention on this kind of hard drive, as I found I am not the only one who had similar problem on installing dual system with this hybrid hard drive. And more people are buying this hybrid hard drive, because it has a speed of SSD but a price close to normal hard drive. 

More details of the problem I have encountered is posted here:
[http://ubuntuforums.org/showthread.php?t=2160168](http://ubuntuforums.org/showthread.php?t=2160168)
This problem was finally solved, but I don't know if this problem can be avoided in the future by improving the compatibility of this type of hardware.

On the second problem, I cannot give any solutions. Maybe let us collect more feedbacks in this thread, and see who should we talk to to solve it.
