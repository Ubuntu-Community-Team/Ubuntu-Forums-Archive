Just thought I'd mention some issues I had with 9.10.

Secondly the intel graphics drivers have some kind of issue; when using 9.10 the max res is 800x600 and the monitor is not identified. The only refresh rate available is something like 60hz too.
The intel graphics work perfectly with 9.04, however the Gigabit eithernet is not recognised.
The motherboard is an ASUS P5G41T-M LE (intel X4500 for graphics, only using builtin devices - no pci cards).
Any helpful hints would be appreciated - it's not a big deal however, as I'm using a USB Wi-Fi card for networking.
