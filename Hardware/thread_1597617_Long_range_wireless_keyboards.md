### Post by grindboy on 2010-10-15
Bluetooth is out of the question because it isn't picked up as an input device until after the OS has loaded the relevant drivers. This makes a bluetooth keyboard useless for editing the BIOS or more importantly choosing options from GRUB. I dual boot Win7 and Ubuntu and although 7 gets use very rarely I would prefer not to have to dig out a usb/ps2 keyboard everytime I need to do some OS switching.

I have heard that there are some USB bluetooth adapters that will present a BT keyboard as a Human Interface Device (HID) so that it can be used for BIOS etc but I haven't seen any user reviews that categorically state that these work.
