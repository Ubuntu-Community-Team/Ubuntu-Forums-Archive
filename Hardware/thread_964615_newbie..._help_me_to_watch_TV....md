### Post by MichaelSwengel on 2008-10-31
snkr,

First of all, welcome to Ubuntu! I hope you like it and find the community helpful!

The first route I would try is WINE. It's a program that allows Linux to run some Windows programs. It is certainly worth a shot.

You can install WINE by using Add/Remove Programs from the Applications menu at the top of the screen.

To install something inside of WINE, open Terminal (Accessories > Terminal) and type "wine programname.exe/msi/whatever.

So, if I wanted to install or run a program named "bob.exe" that is on my Desktop, I would type in the Terminal...

wine ~/Desktop/bob.exe

WINE should take over and get you going.

This may or may not work for you - as I am not familiar with that specific brand/model of TV tuner. However, I hope this will give you a good starting point. I'll be watching this thread in case I can help.

Let us know how this works out for you.

And hopefully someone else will have other ideas. That's the beauty of Ubuntu/Linux. :)
