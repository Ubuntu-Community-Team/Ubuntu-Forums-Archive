### Post by Mark Phelps on 2019-02-12
Personally, I think getting any Linux distro onto an actual tablet is not going to happen.

I have been interested in this myself, since I have some older Android tablets that I would like to run a different OS on -- but everything I've read indicates that it's not going to happen -- anytime soon.

One really serious problem is that you have to be able to boot the tablet from media plugged into the USB port in order to install an OS, and tablets won't let you do that.  So, without that, there is no way to do the OS installation in the first place.

A second serious issue is the lack of Linux drivers for tablets -- and without them, your touch screen is simply NOT going to work, rendering the tablet as useless.

So, like has been already said, until some tablet supplier gets serious about preloading Linux onto their boxes, this is not going to happen.
