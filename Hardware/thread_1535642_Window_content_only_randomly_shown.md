Hi,
I am using Ansoft's Maxwell (a 3D electromagnetic simulation software), which is running on a central linux server, and windows are displayed on my notebook using setenv DISPLAY my.notebook:0.0 (after xhost + on my notebook).
This was the way I always used it, and it worked fine on my previous notebook, and it still works fine on other computers. 
However, on my recent 64bit DELL laptop with an ATi Radeon HD4330 512MB DDR3, and the proprietary fglrx driver from ATI, the 3D panel of the software is not showing anything, only if I move the mouse within this window, when the 3D model starts to flicker. I have absolutely no idea where to start searching for the problem. Everything seems to be ok with the graphics card.
