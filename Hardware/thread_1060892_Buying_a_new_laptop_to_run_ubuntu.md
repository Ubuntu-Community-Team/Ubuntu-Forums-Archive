### Post by shadowhacker on 2009-02-05
Mine runs it real good. It's a HP Compaq Presario CQ50-115NR. I paid $400 for it, pretty good considering the specs.

Turion X2 64 @ 2GHZ
3 GB DDR2 800 RAM
Atheros 802.11G WIFI
10/100/1000 LAN
Nvidia 8200M G Video (shared)

It runs 8.10 i386 out of the box. AMD64 takes some tweaking for the WIFI driver, but it's not bad.

This thing is pretty fast with Vista which came with it, but it flies with 8.10 AMD64. It's also durable as hell; I abuse it frequently and haven't had a bit of a problem with it (except of course when I mess with something I shouldn't ;))
