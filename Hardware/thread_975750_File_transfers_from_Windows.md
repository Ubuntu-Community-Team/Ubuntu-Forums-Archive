### Post by NIT006.5 on 2008-11-08
Personally I have never come across any Windows files that couldn't be opened on Ubuntu. However, it also depends on whether you've got any special/unusual software installed which stores its files in a non-standard format. If files are stored in some strange format by a program that only runs on Windows, you might have issues opening those files. However, all your standard file formats are well supported, including Excel, Word, PowerPoint... and of course Ubuntu handles pdf files way better than Windows :) I'm not sure if there is any support for Microsoft Publisher files though... I've never tried.

### Post by heretik_85 on 2008-11-08
You'll be able to read and write to the Windows file systems just fine, but not necessarily the other way around. If you want to read and write to your Ubuntu partition from Windows, you can install the driver from [http://www.fs-driver.org/](http://www.fs-driver.org/).
