### Post by tL0z on 2021-08-25
Hi everyone,

I'm looking for some help with an issue that I have been facing since upgrading to 21.04, please.

I am running Ubuntu on a Dell G3 laptop that has a GeForce GTX 1050 Ti graphics card. I connect the laptop to a LG display via HDMI, as well as to a Plugable dock which I then connect to another Dell display, also via HDMI.

Following the 21.04 upgrade, every time I start Ubuntu with the dock and the additional screens connected, I get a black screen after logging in. I did some trial and error and things work fine if I have only LG display connected directly, or only the Plugable dock with the Dell display connected. It's when connecting both simultaneously that things go wrong. 

Has anyone experienced this issue and know how to fix it?

Thank you

---

### Post by tL0z on 2021-08-25
Nevermind guys. Upgrading the NVIDIA driver from 390 to 470 fixed it. Cheers :)
