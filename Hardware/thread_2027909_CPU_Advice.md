### Post by TheFu on 2012-07-17
I think your CPU choices are overkill for any of those activities, so it doesn't matter.

I edit HD video on a Core i5 mobile CPU. Compiling can be done on an Atom or Arm CPU - it will just take longer.

A CPU comparison site says the Xeon E5345 is faster, but they didn't have a quad-core version, just a dual core in their list.

BTW, there are lots of Core i5/i7 CPUs for much less money that are listed as faster. Like the[[COLOR=Black]** Intel Core i5-2380P @ 3.10GHz for about $200.**[/COLOR]]("http://www.cpubenchmark.net/cpu_lookup.php?cpu=Intel+Core+i5-2380P+%40+3.10GHz")  That would be a great CPU if you switched to a compatible MB.
