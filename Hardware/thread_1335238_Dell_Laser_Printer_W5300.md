### Post by charrington on 2011-10-07
I have it working on 11.04.  Add Printer did not automatically find it so I selected Appsocket/HP jetDirect and put the printers IP in the host box.  For the driver I used the Dell M5200.  

Not sure if it is a coincidance but the IP is now showing up when I go to Add Printer
