### Post by westie457 on 2021-10-02
It is possible to install a Linux OS on to one of those however there might be a lot of work involved. I had an Asus Transformer about 5 years ago and managed to install Lubuntu on it.
You will need a SD card to go in the expansion slot for storage of data files, 64GB minimum size.
You will also need a powered USB hub and adaptor to connect to the charging port (this port is the only one that supports USB booting) plus a USB keyboard and mouse.
A USB stick of at least 8GB for the ISO. Another computer to create the ISO will be necessary.
If Windows is still on the machine you must disable or uninstall Bitlocker.
Also you must disable 'Fast Start up' from within Windows. These instructions are from Asus. [https://www.asus.com/support/FAQ/1008831/](https://www.asus.com/support/FAQ/1008831/)
The windows must be adjusted using Windows Disk Management.
The ISO has got to be customised before being burnt to the USB stick. Instructions here. [https://askubuntu.com/questions/1168036/creating-uefi-bootable-live-usb-using-custom-iso](https://askubuntu.com/questions/1168036/creating-uefi-bootable-live-usb-using-custom-iso)

If you understand all of the above and are willing to go ahead with this. Come back here with any problems we will try to help.
