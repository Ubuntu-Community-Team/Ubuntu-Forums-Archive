### Post by bnjmnwroberts on 2010-12-03
when I installed 10.10 desktop,( I have an e-machines m-250 netbook) I noticed the same thing. Here's what you should do-I did it and it worked for me.
Re-install ubuntu.
Make sure your laptop is plugged in.
Make sure you have it connected to a high-speed internet source via ethernet cable.
Do NOT select download updates.
Do NOT select install 3rd party software.
you can get this crap later, if you need it at all.
when you are done the installation, you will be prompted to re-start.
do it.
when you turn it back on, you will see an icon on your top panel. click on it, DO NOT SELECT B-43 WIRELESS DRIVER! Instead, select the STA WIRELESS DRIVER.
download and install it.
restart.
after you restart it will find your wireless, and look for modems to connect to.
I hope this helped.
