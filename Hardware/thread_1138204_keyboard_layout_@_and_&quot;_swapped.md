Hi,

I am German using a british laptop and just set up the keyboard myself after installing Jaunty.

Try this:
System > Preferences > Keyboard
> Layouts > Add...

And there is the "images of the keyboard"  you are looking for.
You can add a new layout, make it default and delete the old one after restart.
