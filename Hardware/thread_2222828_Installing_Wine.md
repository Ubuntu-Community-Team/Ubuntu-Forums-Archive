### Post by grahammechanical on 2014-05-08
Are you not connected to the Internet? Are you not installing through the software centre? I do not have Lubuntu installed but it will have something like System Settings and in the setting utility it will have something similar to Software and Updates or Software Sources. There should be a tab called Other Software. In that tab you will see the Cdrom listed as a software source. Untick the box and you will not get that prompt. While you are in Software Sources go in the Ubuntu/Lubuntu software tab and makes sure that all the repositories are ticked. Especially, Universe.

Regards.
