### Post by Zorael on 2008-05-05
I can't say I see the problem right away, but the standard blanket solution would be to reconfigure xorg and see if it helped. You'd need to reenable ati/radeon drivers afterwards.
```
$ sudo mv /etc/X11/xorg.conf /etc/X11/xorg.backup0805
$ sudo dpkg-reconfigure xserver-xorg
```
