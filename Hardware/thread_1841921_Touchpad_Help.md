### Post by al1x on 2011-09-11
If you found the right driver and it's a tarball if it's a tar.bz2 go to the directory of the file and run from the terminal the command: 
```
tar -xjvf yourfile.tar.bz2
```
If it's a tar.gz file type:
```
tar -xzvf yourfile.tar.gz
```
Then just consult the file INSTALL which is inside the archive for the installation.
Good luck searching for the right driver!:mrgreen:
