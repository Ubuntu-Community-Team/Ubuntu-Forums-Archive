### Post by retiefdv on 2010-05-03
After upgrading to Ubuntu 10.04 I had display problems with application windows appearing without any changeable edges, squashed into the top-left area of my laptop screen. I was able to fix this by selecting "System" -> "Preferences" -> "Appearance" and on the "Visual Effects" tab change the settings from "None" to "Normal". After this has been done, it searches for drivers and then asks me if I want to keep the settings. The display then functions normally. The big problem is that I must repeat this process every single time I switch on the laptop. This laptop uses the "Intel Integrated Graphics Media Accelerator X3100" graphics card, which seems to be part of the problem. It seems that there are not Ubuntu compatible drivers and it reverts back to the "none" settings for the appearance visual effects. How can I retain the "none" settings with the rest of 10.04 still in working condition?
