### Post by Andrew_P on 2010-01-22
I, too, am considering getting an external USB hard drive, but I'm not sure which ones are Linux-compatible.  A few weeks ago I was at my local Costco warehouse, and one of the drives actually mentioned Linux compatibility on the package, but most don't.  It's likely, but no guarantee, if the package says it's Mac OS X compatible that it will work with Ubuntu, too; OS X is based on FreeBSD and NextSTEP, which are Unix-like operating systems.

If you happen to be a Costco member, you could purchase the drive and try it out; if it doesn't work, return it within 30 days for a full refund.  There may be other big-box stores that have a similar generous return/exchange policy.

I've read in threads elsewhere that certain Seagate drives have caused Linux problems because they have "auto spindown" after 15 minutes to save power, and Linux typically doesn't allow enough time for the drive to restart and reconnect. Some users were able to get the drives to work by fiddling with Linux device parameters, or with help of a Windows-based configuration utility from Seagate to modify the drive's firmware.  See [http://www.nslu2-linux.org/wiki/FAQ/DealWithAutoSpinDownOnSeagateFreeAgent](http://www.nslu2-linux.org/wiki/FAQ/DealWithAutoSpinDownOnSeagateFreeAgent).

For more discussion in this vein, see the archived thread from 2007 at [http://ubuntuforums.org/showthread.php?t=606675](http://ubuntuforums.org/showthread.php?t=606675).
