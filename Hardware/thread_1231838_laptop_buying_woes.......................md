If you don't want Windows on your laptop at all, there are companies that make laptops with Ubuntu already installed. So all the hardware works in Ubuntu, and you can get tech support if things go wrong:
[http://system76.com/](http://system76.com/)
[http://www.zareason.com/shop/home.php](http://www.zareason.com/shop/home.php)
[http://linuxcertified.com/linux_laptops.html](http://linuxcertified.com/linux_laptops.html)
The Ubuntu forums has a system76 support section:
[http://ubuntuforums.org/forumdisplay.php?f=341](http://ubuntuforums.org/forumdisplay.php?f=341)

[http://linux-laptop.net/](http://linux-laptop.net/)
