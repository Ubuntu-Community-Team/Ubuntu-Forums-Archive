### Post by ajgreeny on 2016-11-02
You do not need to uninstall the open-source driver for the now missing AMD (ATI) card; if the system does not find that card when booting it will ignore the fact that its driver is still on the system.

I have only an integrated Intel graphics system but both the nvidia and AMD (ATI) drivers are installed by default, so you have no need to do anything.
