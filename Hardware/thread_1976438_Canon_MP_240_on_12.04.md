### Post by computerwiz908 on 2012-05-26
You might not be able to get that PPA working in 12.04, but the following steps should work to get your printer working:

[LIST=1]
[*]Download [http://gdlp01.c-wss.com/gds/0/0100001590/01/cnijfilter-common_3.00-1_i386.deb](http://gdlp01.c-wss.com/gds/0/0100001590/01/cnijfilter-common_3.00-1_i386.deb)
[*]Install it using Ubuntu Software Center
[*]Download [http://gdlp01.c-wss.com/gds/5/0100001595/01/cnijfilter-mp240series_3.00-1_i386.deb](http://gdlp01.c-wss.com/gds/5/0100001595/01/cnijfilter-mp240series_3.00-1_i386.deb)
[*]Install it using Ubuntu Software Center
[/LIST]

You should then be able to find your printer when adding it using the Printing subsection of System Settings.
