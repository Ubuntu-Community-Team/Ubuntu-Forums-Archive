> if you have the 98se cd just upgrade from that. while windows is running load the cd, you'll be asked if you want to upgrade. then drivers should not be an issue. and if they are, you can go back to the older version of the driver.

I have the FULL version of Win98SE and if you try to do an upgrade (even though the option is there to select) it will stop the install saying that you MUST install from a blank hard drive.

However, I will say that since I made this post I did take some action:

1) Considering that I couldn't find the drivers posted on line, BUT I had a fully functional system, I looked into one of those "Driver Scanning" programs.  I downloaded two (Driver1_0 and Driver Magician) just to be on the safe side.

2) Later on I found out that many of the drivers were already a part of the Windows 98 OS.   For instance, the built in sound card is made by Crystal Clear.  The card was recognized by Windows 98SE and installed drivers for it!  The same was true of the video card.  (The drivers were older, but nevertheless a part of the Windows operating system).

I believe only about 3 or 4 drivers I needed to put back on to the system.

As we speak I it is now running on Windows 98SE and it has full USB 2.0 support as such.  I finally got it to read/write to SansDrive memory sticks.

So, as it stands, I am not going to covert this machine to a Linux box.

At any rate, thanx for chiming in.
