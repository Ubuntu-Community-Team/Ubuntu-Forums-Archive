### Post by aidenscool09 on 2010-08-15
DDR3 IS a great deal better than DDR2 if your motherboard and CPU can support it. Also, make sure the voltages are compatible. Check with your Motherboard and CPU Manufacturer's website before buying anything. Also, you're better off keeping any RAM modules as you may encounter a problem and not be able to use the DDR3 RAM.

DDR3 has a number of technical improvements over DDR2, most of which are way over any normal users head. The mian differences for normal people are that DDR3 uses less power, runs cooler, is faster and will have a better upgrade path in the future.
