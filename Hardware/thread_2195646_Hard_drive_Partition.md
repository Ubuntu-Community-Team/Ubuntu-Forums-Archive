Only the server version of 10.04 is still supported. You should upgrade. If system is limited you may prefer Lubuntu or Xubuntu as Unity & latest full Ubuntu needs more RAM and decent video card.

Windows has to have a primary partition (sda1 thru sda4) formatted NTFS with the boot flag.
