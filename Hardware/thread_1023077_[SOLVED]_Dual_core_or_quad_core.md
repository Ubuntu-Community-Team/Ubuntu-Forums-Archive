### Post by Rocket2DMn on 2008-12-27
PCs do spread tasks between cores, but not all processes are designed to run in multiple threads on multiple cores.  So the main advantage to having multiple cores is if you do a lot of multi-tasking or are using software that is designed to work efficiently between multiple cores.  Programming multithreaded applications is very difficult, so many products don't do it or do it very well.  It is true that videogames don't tend to use this technology very much, so two cores is likely enough for you.
That dual core processor you are looking at is pretty beastly, seems to be pretty top of the line, so make sure you get other hardware to match (for example, a motherboard and RAM that can match that FSB speed).
