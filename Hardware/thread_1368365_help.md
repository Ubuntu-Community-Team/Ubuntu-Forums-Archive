### Post by tacantara on 2009-12-30
There's a few ways to run Ubuntu and Windows on the same machine:

1.  Dual-boot:  (I'm assuming you already have a Windows DVD and an Ubuntu LiveCD) Install Windows, then install Ubuntu off the LiveCD.  You will be given an option to put Ubuntu on a separate partition from your Windows install.  Once you've done this, restart your computer.  You will see a screen that will give you the option of starting either Windows or Ubuntu. 

2.  Virtual Machine:  Install Ubuntu, then install the Virtual Box package.  From your post, I'm guessing you don't have an internet connection, so this may not be an option for you.

3.  Wubi Installer:  Just like the first option, this requires you to install Windows first.  The Wubi installer is a version of Ubuntu that is designed to install within Windows, as a program.  Again, you'd need an internet connection to download Wubi.

WINE is good for running some Windows applications, to include Internet Explorer.  However, you can access the internet in Ubuntu. The default web browser in Ubuntu is Firefox, and it does a really good job.

I hope this is helpful info for you.
