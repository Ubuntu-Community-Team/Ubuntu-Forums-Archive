I'm running 64-Bit 14.04 on a HP Elitebook 8440p using the NVIDIA Driver version 331.113 (the x.org driver didn't cut the mustard on this machine). 

I have been trying to use two external monitors through VGA from my computer (both with a HP docking station and without), but have been frustrated. First, hooked up an HP W1907 19" monitor, which worked fine for about four hours, until it inexplicably refused to notice the VGA connection and would subsequently go to sleep. Since then, I've been able to "jumpstart" the monitor by turning it on or off or holding down the menu button, but then it only works for maybe 5-10 minutes before going black. I then switched to DVI-D, which had the same results: The monitor works fine for five minutes before cutting out. 

 Thinking it was a problem with the monitor (which I'm about to return on eBay), I switched to an old and trusty monitor of mine, a ViewSonic VP201b, connected though VGA. For about 8 hours, it worked great. Get up the next morning, and the horizontal control is all messed up. The screen kept flipping and was totally unreadable. Attached it on DVI and had the same issues. This went on for several hours, but right as I'm writing this, it slowly corrected itself. 

Both monitors seem to function correctly when connected to other, non-Ubuntu machines. The VGA cable seems to be fine, and as it was also a problem when connecting through DVI, I think I can rule that out as a culprit. 

Any ideas?
