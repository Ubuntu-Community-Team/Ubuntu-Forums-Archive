### Post by alex2399 on 2011-05-25
If all functions well under the hood of your Ubuntu installation then they are used automatically when you remove the restricted drivers. I recommend to just use the restricted drivers in your case since you wish to play games. The restricted drivers offer better performance at the moment for this situation.

Regarding games, it will play games that are availabe in the software center ok, but don't expect to run a game like Crysis and have the same experience as in windows.

You have three options:
- Play games from the Ubuntu software centre (i.e. OpenBVE, Flightgear, Sauerbraten, Nexuiz etc...)
- Use a Windows emulator like Wine in Ubuntu. Windows games may, may not or partially function.
- Keep a seperate Windows install (dual boot > best option in my opinion)
