### Post by Gardrek on 2012-08-03
> **smashweights said:**
> Two straight up noob questions really.
1- When Ubuntu loads up, the initial brightness is set so low the backlight is off.  I can turn it up using the hotkey, but was wondering if there's a way to adjust the load up brightness setting so i don't have to do this each time.

If you select System > Preferences > Power Management (in 11.04) you can adjust your brightness then press Make Default to have it start with that brightness on the log-in screen. Sorry for bump, but I had the same problem and this thread comes up on Google. Hopefully, this will at least help some future Googler.
