### Post by legendre17 on 2008-10-26
I'm running Ubuntu 8.04.1 on a Dell Latitude D620 laptop.

I'm trying to use a different keyboard layout (Romanian vs. US), but nothing seems to work: the hotkey doesn't switch between layouts, and the keys behave like the normal US keyboard no matter which layout I choose (even if I completely remove the US layout). I remember this having worked fine at some point, possibly while I was running Ubuntu 7.10, but right now it looks like the layout settings are just completely ignored.

Does anyone else have this problem and / or have a clue how to solve it?

### Post by legendre17 on 2008-10-29
Never mind, it works now. I had to add the keyboard indicator to the task bar and install language support for Romanian.
