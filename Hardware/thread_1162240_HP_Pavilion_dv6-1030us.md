<<First response>>I have been using ubuntu with the hp dv6 1030-us laptop model for a while now, I have 
had the least trouble with the new release (9.10 64bit), the only problems I have found are 
with wifi , and sound comming out of both speakers and headphones at the same time. if 
you are considering buying this laptop the simplest fix for wifi is install wicd
"sudo apt-get install wicd", don't know about sound. you can try ubuntu 8.10 which I used 
briefly without problems.
