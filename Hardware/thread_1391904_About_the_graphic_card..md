### Post by doas777 on 2010-01-27
> **BlindSoul said:**
> Sorry about the bold, I'm just always used to use in Bold. And if the driver is existed where can i find it?
ubuntu will point it out and install it for you, using the hardware drivers manager. the card must first be installed however.

alternately you can download it from nvidia, but that can be a pain when kernel updates come down, as you may need to reinstall it, if you are not using the ubuntu bundled driver.
after you install the card, boot it up, and then reboot it. hopefully you will see a message in the notification area, and if not, just open and close software center once, to get it to notice the driver if it has not already.
