### Post by wolfwood7 on 2008-05-23
Recently I bought a new computer and I've installed both the 32bit and 64bit versions of 8.04 but for both systems the keyboard support for both my ps2 and usb keyboard have been horrible. In every case the system either doesn't respond to a key-press or assumes the key has been held down to fill the entire text box, taking anywhere from 0.5 to 2 seconds to respond to a quick press. 

At first I thought it might be my KVM, but I've taken it out and plugged both my mouse and keyboard directly into the system without effect. Next I swapped for a PS/2 keyboard which only made it slightly better in it would occasionally interpret a single key press as a single key press after a second delay. Lastly I tried to alter my usb settings for my motherboard (EVGA 780i), but the only options I could find were to either disable usb keyboards, limit myself to usb 1.1, or eliminate usb all together.


I really hope someone knows of something I could do to fix this as I've always loved using ubuntu as my chosen distro, but without my keyboard it's unusable. 


system specs:

EVGA 780i motherboard
Natural Ergonamic Microsoft Keyboard
