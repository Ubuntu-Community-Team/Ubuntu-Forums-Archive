Hi there, I'm very new to ubuntu as well, so i may be of no help, but from what i have understood the power management is actually done on two fronts.

1) Screensaver
2) Power management

if you open up the screen saver settings uncheck the activate screen saver when idle. I had an issue whereby when idle the screen saver would activate (blank screen saver in my case) but the screen wouldn't actually shut down meaning it was still drawing lots of power, drawing a black screen.

Because there is no setting such as turn off screen, i think you have to select blank screen in power management, and if you have no screensaver activated it should turn the display off.

Well that is how my laptop works. Maybe it will for you as well.
