ASRock Z790M-ITX WiFi Bluetooth Issues.

Installed Ubuntu 23.10 and later on a custom built ASRock Z790M-ITX WiFi motherboard. Could not get bluetooth to work. Finally added a DP-Link bluetooth dongle and bluetooth worked as expected. I can now see devices for paring. 

Checked the ASRock website and there are no drivers specific to Linux or beyond the version I have presently installed.

Suspect the driver for the ASRock Z790M-ITX WiFi Bluetooth hardware is not compatible. Posting this here to bring attention to the issue.

If anyone is aware of a software fix or if Ubuntu is working on a driver would appreciate a heads up.

Spent nearly a day looking for possible fixes. Did not find any software fixes. However for a hardware fix,  the TP-Link Bluetooth dongle works great.
