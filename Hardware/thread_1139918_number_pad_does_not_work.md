### Post by ronaldprettyman on 2009-06-30
System>Preferences>Keyboard
Mouse Keys Tab
Uncheck Pointers can be controlled using the keypad


Layout
Keyboard Model

Set to your keyboard model, Under brand, then generic if nothing else

Then click Apply System-Wide

Had the same problem after upgrading to 9.04, thought keyboard was going bad at first. :lolflag:
