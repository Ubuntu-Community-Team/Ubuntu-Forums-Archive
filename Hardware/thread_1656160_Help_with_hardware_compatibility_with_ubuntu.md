### Post by cchhrriiss121212 on 2010-12-30
**General rules on buying new hardware:**

[LIST]
[*] CPU and RAM you don't need to worry about
[*]Nvidia GPUs will work fine with the non-free driver
[*]ATI GPUs will work but the non-free driver is not so good
[*]The motherboard is the thing you should spend the most time researching, as it handles networking, sound and other things that can be problematic
[/LIST]
