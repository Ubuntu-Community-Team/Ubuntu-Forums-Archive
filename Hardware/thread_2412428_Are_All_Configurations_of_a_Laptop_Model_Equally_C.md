### Post by scribner on 2019-02-12
If a laptop model (say a Lenovo ThinkPad X270) is said to be fully compatible with an OS (say FreeBSD), will all configurations (RAM, storage, processor, etc. [note: I think other things like Wi-Fi and graphics are more important in this regard]) of this model be fully compatible with the OS? I've been trying to get a response on the FreeBSD Forums but have had no luck. I thought Ubuntu users might be able to help (Ubuntu is my current OS). The more people who can confirm, the better; I was planning on ordering very soon. Refer to this page: [https://wiki.freebsd.org/Laptops](https://wiki.freebsd.org/Laptops)

### Post by CatKiller on 2019-02-12
No.

RAM, processor, hard drives: no problem. Anything in those categories is going to work.

Graphics, WiFi, sound: you're going to have to check compatibility with the components that are in the specific configuration you're interested in. Manufacturers love to make those non-standard to "add value."
