### Post by HappyFeet on 2009-12-08
If you are using 32bit vista, it will only see a little over 3gb of ram. The only time ram is used for virtualbox, is when you are actually using the guest OS. When the guest OS is not in use or deleted the ram you allocated becomes available again for the host OS.
