### Post by Mark Phelps on 2014-12-16
> Is there any sort of hardware that I should be taking into account when selecting a laptop for Ubuntu or will any do a job.

Many current laptops have some form of Dual/Hybrid/Switchable Graphics -- which work poorly, if at all, in Linux.

Older laptops might not have processors that support PAE -- which is needed for recent Ubuntu versions (I believe).

Also, if you get a Win7 laptop that uses UEFI, you will have more trouble installing Ubuntu than if you get an older XP laptop that simply uses BIOS.

### Post by oldfred on 2014-12-16
Laptops with XP cover a very wide range of specs. I would avoid the very old ones.

My laptop from just before Vista came out has 1.5GB of RAM and Intel Core2 duo. That does run Ubuntu but not with Unity as Intel video chip just does not have enough horsepower. I change to fallback/gnome-panel. Others would suggest Lubuntu or Xubuntu.

AMD chip based systems may be a lot cheaper, but AMD does not support older video well and you have to install legacy drivers or use the open source drivers which may work for some models better than others.
