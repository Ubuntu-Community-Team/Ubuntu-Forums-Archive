### Post by figgs on 2008-12-18
I figured out the problem.  In case anyone else has the problem, the description from the other thread should probably read:

   1. Open Volume Control (double click speaker icon upper left in gnome)
   2. go to Options (File > Options
   3. Make sure "External Amplifer,0" is checked
   4. close
   5. Click the "Show Switches" arrow
   6. and UNCHECK External Amplifier

Hopefully that will work for anyone else using this computer with xubuntu.
