### Post by wheeze on 2012-11-07
Most often, references to "Virtual Box" are aimed at the software made available by Oracle, [https://www.virtualbox.org/](https://www.virtualbox.org/)

It allows you create a complete virtual PC that you can install an OS and applications in. Many people use it to evaluate new distros without requiring a free partition to do an actual install on.

### Post by drmrgd on 2012-11-08
My understanding is that it's a file, whose size is either set to be dynamically determined (i.e. it'll grow as your system grows), or statically set at a predefined size (e.g. 100GB).  That file is then represented as a virtual hard drive to Virtual Box.  In fact, you should be able to take your VB file and move it to another computer running VB and it'll be exactly the same as on the computer you've set it up on (barring any special hardware issues).  

The RAM is real, though, and uses system RAM just like any other program.
