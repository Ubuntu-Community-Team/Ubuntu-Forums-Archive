### Post by angelsguitar on 2009-06-29
I installed Ubuntu Studio on my desktop. Since I have an Nvidia GEForce 7300 LE, and noted the graphics performance a little slow, I decided to install the restricted drivers for it.

Now, after a while of using the computer, suddenly the mouse becomes very slow and erratic, as if it were delayed. It gets solved when I reboot, but then it will happen again after several minutes of using the computer. I really don't know what is causing it or how to solve it.  So, if I stay with the open source nvidia drivers, the graphics performance is slow.  But when I change to the propietary drivers, I get the slow mouse symptom.

The same is happening with 64Studio 3 Beta 3 (the slow mouse), which is based on Ubuntu Hardy, installed on the same machine.  Only in this partition I did not install the propietary NVidia drivers, but decided to use the open source ones, as they were not showing any problems.

Any help would be greatly appreciated.

---

### Post by angelsguitar on 2009-06-30
Solved: changed the mouse to a PS/2 port using an adapter (it was originally connected to USB).  Maybe this was cause because I have two mice connected to my system, and both were USB.  Changing one of them to the PS/2 solved the problem.
