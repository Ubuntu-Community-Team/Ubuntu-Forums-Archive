### Post by martinrame on 2013-07-15
Whell, I've fixed the issue.

First tried "ghostscript --version", and its result was:

ghostscript: error while loading shared libraries: libijs-0.35.so: cannot open shared object file: No such file or directory

So, I did "sudo apt-get remove libijs-0.35" to remove libijs. Then, as it deinstalled ghostscript and cups, I did " sudo apt-get install libijs-0.35 ghostscript cups cups-filters".

Now everything works as expected.
