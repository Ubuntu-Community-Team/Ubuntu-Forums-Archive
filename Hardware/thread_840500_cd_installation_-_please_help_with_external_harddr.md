### Post by mikjp on 2008-06-25
It is not a good idea to install Ubuntu to an external drive. You should defragment your laptops harddrive and shrink windows' partition. If you install Ubuntu on an external harddrive, you might not be able to boot your laptop if the harddrive is not connected to the laptop. Just take a full backup from the files in your Windows drive to the external harddrive to have everything backed up in case of some major disaster.

If you use a normal Ubuntu live CD for the installation, you don't need any command line stuff. Don't worry!

You should read [https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall) to see what to do.

Greetings & welcome to the world of Linux!
