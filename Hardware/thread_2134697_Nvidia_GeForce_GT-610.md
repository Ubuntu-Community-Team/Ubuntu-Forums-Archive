### Post by gazzrandall on 2013-07-11
Hi. I have been using geforce gt-610 video cards for some time now in Ubuntu, Linux Mint
and even Netrunner with no problems. I always install the nvidia drivers using Synaptic
Package Manager. If your using Ubuntu you will have to install Synaptic (I usually open
Ubuntu Software Centre and type in Synaptic and then install it) as Ubuntu does not have it
pre-installed.
Then.....open Synaptic Package Manager and firstly hit the "reload" button loacated on tne
upper left. After Synaptic has reloaded everything then just type in the search box at the top
"nvidia" .  All of the "nvidia" drivers etc should now appear in the window below the search
box .
Look for "nvidia-current" and then select it - it will then be highlighted. then select the square
checkbox to the left. The words "mark for installation" will appear so then select the "mark for 
installation" text. When you do the before-mentioned a box will then appear with the words "Mark
additional reqired changes ?  Then select the "Mark" checkbox.
Next at the top of the Synaptic window select "Apply" and then one more window will appear that
says "Apply the following changes". Then select the "Apply" button and sit back and wait until 
Synaptic has installed all of the required drivers. When Synaptic has finished installing re-boot your 
computer and everything should work with your GT-610.
