### Post by kpkeerthi on 2008-10-14
1. Completely uninstall NVIDIA using envy or nvidia-installer (google to find out).

2. Boot with Ubuntu Live CD and copy the xorg.conf (yes, from the live cd) to a flash drive. Lets call it **xorg.conf.live**

3. Boot your installed Ubuntu. Now replace (back it up just in case) the xorg.conf with **xorg.conf.live**
```

cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
cp /media/MyFlashDrive/xorg.conf.live /etc/X11/xorg.conf

```
* This is just for illustration. Change paths accordingly.

4. Reboot.

5. You should now be on Ubuntu desktop just the way it was when you installed it for the first time.

6. To enable NVIDIA, go to System -> Hardware Drivers (Preferred way!)
