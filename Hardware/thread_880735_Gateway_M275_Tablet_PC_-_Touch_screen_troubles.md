### Post by ichthus on 2008-08-15
I use the same tablet, and I can tell you what's wrong with the configuration:
1) two Option lines I see in each of the InputDevice sections,  > Option "ForceDevice" "ISDv4" 
and
> Option "USB" "on"
is an either/or proposition.  The former is for an internal device, like the graphics tablet used in the tablet PC, and the later for a USB tablet device.  You want the former and NOT the later.

2)the InputDevice identified as "pad" is superfluous.  This is again an external, like USB, graphics tablet.  You don't need this in the xorg-conf.

That is, if the xorg.conf above is yours.  If it is different than the listing above, then could you provide the current version?
