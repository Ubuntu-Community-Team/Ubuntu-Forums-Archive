### Post by Edubuntu on 2011-01-22
Solved it myself by experimenting.

If anyone wonders the Wireless drivers were disabled. 
What i did was:

1. Go to System > Hardware Drivers
2. When i got there, it automatically brought up a window mentioned B43 something drivers are not activated. I activated them, restarted and it works now.

Note: If you have Software manager, or update manager up, close them, you will get errors while installing/activating the drivers. The error was: installArchive() failed.

Hope it helps anyone.
