### Post by JBAlaska on 2009-11-23
Pulling out the card without using the 'Safely remove Hardware feature' risks corrupting data on the card.
Reformat the card in the camera.

Also with compact flash it is a simple matter to damage some of the pins in the card slot.

X2 Also : HP printers only read 1 gb memory cards.
