### Post by oldfred on 2014-04-28
I usually suggest 20 to 25GB and actually use 11GB in my working install. But that includes about 2GB for /home mostly .wine. I have all data in data partitions. With a 1 TB drive does having enough extra space make much difference. I find just writing a DVD uses another 4GB of tmp space.
Windows always works best if you have 30% free space and at 10% free just about is so slow as to be unusable. 

If installing with two drives consider Windows on one drive with a smaller system partition and larger data partition. And Ubuntu on the other drive with smaller / and larger data or /home. You then can also backup Windows data to Ubuntu drive and Ubuntu data to Windows drive. But still should have image backups. I backup some of my most important data to DVDs and archive them, so I can go back and get some data if I modified it and need old version.
