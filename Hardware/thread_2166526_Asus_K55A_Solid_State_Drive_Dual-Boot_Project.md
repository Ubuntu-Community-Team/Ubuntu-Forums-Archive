### Post by oldfred on 2013-08-10
How much space is moderate amount of storage. A few videos can consume a lot of space.

I have two Ubuntus installed in my 62GB SSD, but have all data in two 100GB data partitions on another hard drive. But I find I have used about 9GB with a fair number of applications installed including 2GB for wine with Picasa in my /home.

With Windows you may need 30GB as a minimum as it needs about 30% free space in the NTFS partition to work well. One of the main reason Windows gets slow is full NTFS partitions. If down to 10% it is just about unusable. Of course even Linux file systems need some space.

So I think with 60 to 90GB you can have two systems and some data. Then how much more data may you want? Or can you add an external USB drive for some data?
