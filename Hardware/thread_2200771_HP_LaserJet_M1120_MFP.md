### Post by tgalati4 on 2014-01-21
Open a terminal:

```
hp-info
```

Cut and paste the version information and any printers that show up.

Try searching for the device using:

```
hp-toolbox
```

Make sure the printer is on, awake, and connected to the network.  Double-check your network settings from the front panel of the printer.
