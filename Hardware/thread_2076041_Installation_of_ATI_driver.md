### Post by myromance123 on 2012-10-25
viktorjano, you should try and download a more recent version of Edubuntu. I suggest going with Edubuntu 12.04.1 since it is an LTS.

There is however Edubuntu 12.10 if you must be using the latest and greatest.

You can download it here, and it is free.
[http://www.edubuntu.org/download]("http://www.edubuntu.org/download")

It would help us to help you if we knew what AMD/ATi card you actually have.
Are you installing this on a laptop or maybe a desktop?

In newer versions of Ubuntu & Edubuntu (like in 12.04), to install a .run file is pretty easy.

Right click it -> Go to Properties -> Permissions Tab -> Make sure "Allow execute" is ticked.

Then just double click the installer (the .run file) and when prompted, click the "Run" button. After that it should be pretty self explanatory. 

However it might be better if you used the driver from Ubuntu/Edubuntu's repository, since the one from there will properly set itself up on your system.
