### Post by troymius on 2016-05-23
My zenbook ux305 started having issues with wifi (very flaky behavior). Xubuntu 16.04. 
I tried many suggested solutions from around the web. In the end, this did it: install wicd, remove network-manager.
Now the connection is solid.
