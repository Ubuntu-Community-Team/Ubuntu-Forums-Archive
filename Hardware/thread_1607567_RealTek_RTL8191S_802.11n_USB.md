On my 9.04 karmic laptop.

Didn't recognize it when inserted directly - but I used Synaptic to load "ndisgtk" and the ndiswrapper libs.

Ran ndisgtk, pointed it at the .inf file in the Windows XP directory on the driver disk - and it worked right after a reboot!


Using it now to post this... :)

Just thought I'd share.

This little adapter was $11 or $12 (USD) on eBay.
