### Post by grahammechanical on 2013-11-13
! am running Ubuntu 13.10 on a machine that has an Intel Core2 Duo CPU running at 2.40GHz. The machine has 1GB RAM. Right now I am playing a BBC radio program and using Chromium to access this web site and the System is using 712 MB.

The video adapter is the component that you should give careful consideration to. The power of the GPU and the amount of memory that the graphic adapter has will make a big difference to your user experience. Get the most powerful video/graphic adapter you can afford. I would say that video of 1GB would be a minimum amount. I have an Nvidia GT220 with 1GB of memory.

Ubuntu 12.04 is fine but I would also recommend upgrading to Ubuntu 14.04 when it comes out next April. Why? All the work being done on Ubuntu for phones and tablets is filtering down to Ubuntu desktop. When I was running Ubuntu 12.04 memory usage of 1GB was just under 70%. Moving on the Ubuntu 13.04 memory usage went down to a little over 50% and now on Trusty Tahr (Ubuntu 14.04 under development) memory usage is a little under 50%. So, I would recommend moving on to Ubuntu 14.04 when it comes out.

Regards.
