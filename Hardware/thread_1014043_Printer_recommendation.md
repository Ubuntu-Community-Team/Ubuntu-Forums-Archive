### Post by kaibob on 2008-12-17
I have an HP Deskjet 6988 and like it a lot. You can get the very similar HP Deskjet 6940 for $80.00 at Micro Center, HP online, and other places. The cartridges are worth $55.00, so you're paying $25.00 for the printer.

A few comments:

* I plugged the printer into the computer and it was recognized and worked without further action.

* The printer is well built and print quality is good.

* The cost per black-and-white page is about 3.5 cents and lower with some competitive shopping. 

* The printer is a bit annoying in regular mode because it works so fast that it actually shakes the table--fortunately there is a quiet mode that eliminates this problem. 

* The 6988 has wireless built in and isn't all that much more expensive.

[http://h10010.www1.hp.com/wwpc/us/en/sm/WF05a/18972-18972-236251-14438-3328074-468001.html](http://h10010.www1.hp.com/wwpc/us/en/sm/WF05a/18972-18972-236251-14438-3328074-468001.html)
