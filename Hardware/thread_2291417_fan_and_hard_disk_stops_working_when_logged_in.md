### Post by weatherman2 on 2015-08-20
Test the hard drive.  Check the S.M.A.R.T. Attributes with GSmartControl - any highlighted in pink are the potential trouble spots.  You may need to do this from a live USB Linux distro.  If you use Ubuntu live USB, you'll have to install GSmartControl after enabling the "Universe" repository in Ubuntu Software Center.
