Hi there,
In answer to your question, yes you can. I am writing this via my Justop using Lubuntu. It isn't fast but it is usable.
Apparently Ubuntu can be used too but it is slower still.

I am booting from a 32Gb Class 10 SD card (4Gb Minimum), using the image and instructions found here;

[https://www.miniand.com/forums/forums/2/topics/1]("https://www.miniand.com/forums/forums/2/topics/1")

I'm using the 1Gb 1080p version.
            
I wasn't that impressed with the standard Android interface (it is a bit clunky compared to ICS on my tablet and phone).

Just tried this for fun, I am waiting till XBMC is released for Android or an Openelec for A10 SoCs, which should make it slick.
