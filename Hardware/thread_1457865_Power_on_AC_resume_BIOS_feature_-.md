### Post by Jethro_uk on 2010-04-19
I've had to replace my dead fileserver with another machine.

Unfortunately this machine is based on a Gigabyte GA-8IDX motherboard, which although it has APCI features, and wake-on PME/LAN settings, doesn't have a "State after AC resume" option.

This means that if I have a powercut, I have to go up the ladder to press the button ...

does anyone know if it's possible to flash the BIOS to support this feature ? Or if there's another way to allow me to get the machine up & running if the power goes ?

It's an AWARD BIOS, btw

---

### Post by cascade9 on 2010-04-19
It does have the settings you want- "AC back function". Set this to "full-on". 

Intergrated Peripherals-> AC Back Function.
