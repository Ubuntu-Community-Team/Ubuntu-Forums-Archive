### Post by redwaldmcmanus on 2010-10-13
Hi All,

I've used Ubuntu for a few years now on old hardware, and finally have myself something new. One day, I'm sure things will be supported from the off, but until then, I have to tip my cap to all the wonderful developers out there working on compatibility issues.

The Sony Vaio VPCS12V9E has a few issues. I've overcome most on 10.10, and I'm working on the rest. Here is my guide thus far. Please post any workarounds you have. I'll do my best to keep this first post updated. Thanks.

(I've included more steps than some may find necessary, to try and make it simple for new users.
I'm sure there are better methods than the ones I've used, but they worked for me. Please submit improvements! :-) )

[SIZE="5"]Booting Live CD[/SIZE]
There are two boot options that need adding.
1) Boot from CD
2) Press a key when you see a picture of a keyboard.
3) Press F6 to add boot options
4) Press ESC to type in your own
5) Add "nomodeset" and "i8042.nopnp"
6) Press enter to boot

[SIZE="5"]Booting After Install[/SIZE]
1) At Grub, press "e" to edit the default option
2) After "quiet splash" add "nomodeset i8042.nopnp"
3) Press CTRL + X to boot
**Making these changed permanent**
1) After startup, hit ALT + F2
2) Type "gksu gedit /etc/default/grub" and press enter, enter password and press enter.
3) Line 9 GRUB_CMDLINE_LINUX_DEFAULT, add nomodeset and i8042.nopnp after "quiet splash"
4) Save and exit
5) Press ALT + F2 and enter "sudo update-grub", select "Run in Terminal" and press run
6) Enter Password
This will then update grub with the new options, and will do it on future kernal updates also.
