### Post by papibe on 2011-05-05
I had a laptop with a similar problem. I never solved it, but I learned a few tricks that can help you. Here's 2 ways to get your screen back:

[LIST=1]
[*]Press Ctrl + Alt + F7 to force X Windows to refresh your graphical session.
[*]If that doesn't work, try going to text mode first by pressing Ctrl + Alt + F1, and then pressing Ctrl + Alt + F7.
[/LIST]
I hope it helps.
