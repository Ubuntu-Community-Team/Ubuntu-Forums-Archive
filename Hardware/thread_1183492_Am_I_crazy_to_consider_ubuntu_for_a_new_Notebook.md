Hey Jolo

I can't answer all of your points, but a couple of comments from my point of view (not an IT pro, but use my lappy quite a lot).

Ubuntu works great for me.  I chose my laptop partly based on information from the Ubuntu Laptop Testing team, which was a useful resource (google turns it up quite readily).

There's also a Wine app database with loads of info on which apps work or not, so another great resource if you're interested in using slightly unusual software.

Don't know about Edubuntu, but as for the other 2, I think it's a matter of personal preference.  Take a look at some screenshots to help decide.

Good luck!

### Post by yarko on 2009-06-10
A note from my experiences:

-- I've been using ubuntu (no windows) for > 6mnths;  I weaned myself in on a corporate pc (so dual boot, overwriting the MBR was not a good option).

Here's how I started:

-  Got a (large, cheap, fast) mini-USB drive, boot from it at home (NO Windows, thank you!);

Wine has limitations, but for the few things (like, if you have a self-running director file, and you have no other choices) you can deal with it just fine.   You could also try running your windows inside something like VirtualBox (I've talked to people who say it works; I just never had motivation / need).

IF you are considering going dual-boot, be sure to backup your MBR (so you can recover if need be).  (see, for example, [http://www.tuxation.com/mbr-tricks-with-linux.html](http://www.tuxation.com/mbr-tricks-with-linux.html) and [http://en.wikipedia.org/wiki/Master_boot_record;](http://en.wikipedia.org/wiki/Master_boot_record;)  I have successfully copied back the first 446 bytes to restore original boot info).

Going USB boot mode is a good long term way to convince yourself.

Enjoy!

### Post by jward3010 on 2009-06-11
If you're thinking of having some kind of DATA partition, where you save commonly used files for both OS's like Music, Videos, Pictures, Downloads etc. etc. then make it an NTFS partition - this isn't because NTFS is better, it's not, it's because Ubuntu has absolutley no problem reading and writing to NTFS partitions whereas Windows has no default method for accessing Linux filesystems like ext3 / ext4. You have to download a third party driver for ext2 and the developer doesn't want to release it as open source so you can't guarantee it will be upgraded as the "ext" filesystem progresses. In fact it's probably incompatible already with ext4, it also expects an ext3 filesystem to be formatted with a particular block size which is quite advanced.

Once again, old Hardy was the savior. It is my opinion, based only on my successes and failures, that if Hardy will run well from a Live-CD the laptop should run very nicely under Ubuntu Linux. To date, I have not encountered a laptop that would not do so -- excepting machines with too little memory, of course.

If you have another source to run Windows from, so that this laptop doesn't *really* have to run it, my suggestion is to wipe Windows from the drive (leaving any hidden OEM installation partitions intact) and install Ubuntu in two partitions: one for "/," the other for "/home." If you find a particular release of Ubuntu doesn't run well on the laptop you have bought, all you need do is delete all the unnecessary configuration (dot) files in your home partition (as root) and reinstall with something different. Don't reformat "/home" and you are back in business with a new lease on Linux and with your data ready to use. In my opinion, the best way to allow Linux to work for you is to be committed to it. Dual-boots are like training wheels on a bicycle: for some, the sense of security helps them learn to move on, but for others they never do make the transition as long as they lean on the 'status quo.' My experiences over the years, not gospel of course.

As for the rest, life is always tough for the underdog. Linux has made great progress over the years. You and I are mainly the reason for this. Without us to use Linux and preach its benefits, no one or company would invest the time to make it better. Time is our ally. Patience is the catalyst that makes our efforts worthwhile. Good luck with your pick of laptop. Welcome to the exciting world of Linux!
