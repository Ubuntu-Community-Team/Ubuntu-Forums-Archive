Both the amd 64x2 dual core cpu and intel c2d cpu are 64 bit cpus, so they are completely compatible with 64 bit ubuntu. As both of you have 4 Gb of ram, you will both benefit from being able to use all of your ram if you install a 64 bit os.
For more benefits and possible drawbacks, [see this post]("http://ubuntuforums.org/showthread.php?t=765428") and check out the [64 bit section]("http://ubuntuforums.org/forumdisplay.php?f=343").

@Ac1ds0ld13r: As the cpu is 64 bit there is no reason why your computer wouldn't "handle it well enough". Either it does handle it or it doesn't, there are no "almost 64 bit but not really" cpu's.
@notfound123: A dual core should be detected as a dual core, irrespective of which os architecture you install.
