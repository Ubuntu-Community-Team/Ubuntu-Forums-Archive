The easiest thing would be to look up your laptop and get the right screen for it. A quick Google for laptop screen replacement will pull up some places were you could get the right display for your computer. (assuming Ubuntu displays correctly if it had the right screen.) If that is not an option for some reason, there is one thing I did when this happened to me a few years ago, but you would have to be running Ubuntu 10.10 or older for my old method to work. Fundamentally I think it's a pixel mapping problem.

Also, if you just want to use the laptop and don't need to move it around a lot, you could always plug in an external screen. I know that's not a "fix", but it would be a good work around for the problem if you have to use it right now.
