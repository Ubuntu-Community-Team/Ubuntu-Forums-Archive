### Post by sanderj on 2014-01-03
Very probably not.

Reason: the VIA 8650 800Mhz processor seems to be a ARM 9, which is ARMv4 or ARMv5 architecture ([http://en.wikipedia.org/wiki/ARM_architecture](http://en.wikipedia.org/wiki/ARM_architecture)). *ubuntu needs at least ARMv7. And even for the ARMv7 only very limited set of hardware platforms can run Ubuntu.
