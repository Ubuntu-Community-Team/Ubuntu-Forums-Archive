### Post by Hukei on 2010-10-16
How do you know you lost all your partitions?

In any case, try using Hiren's Boot CD.

Download it from here:
[http://www.hirensbootcd.net/download.html]("http://www.hirensbootcd.net/download.html")

Just don't try to repair the partition table or the mbr with it. Do it only if you know what your are doing.

And try booting the Mini Windows XP (if you have a valid windows license that is) to see if you can get into any of your partitions to recover the data. If not, try using some of the recovery software withing the "Dos Programs" menu or within the Mini Windows XP.

Most probably you can recover everything.

### Post by cbemerine on 2010-10-17
> **slkamath said:**
> Now it's already formatted and installed fresh copy of vista and ubuntu. So i want to know how to get back those data's???

If you are positive that you formatted the disks when you reinstalled, than it is extremely doubtful that you will get any of the data back.   I would chalk it up to a lesson learned and never make that mistake again.

However if you are not positive that you told the installs to format (you would have had to select the options to reformat) your disks, than you could try the suggestion above and see what you can get.  It would be a great learning experience for you regardless if you can afford the time.  At this point you only lose time to give it a try.  Worst case scenario, you still do not have your data and you have to reinstall everything again.  At this point you can not make it worse.

In the future, always backup your data before upgrading and/or installing an operating system, kernel, hard drives or anything that could potentially impact your data.  Just one reason to loath automatic upgrades, though they are suppose to prevent this. *(I acknowledge that you post did not say anything about automatic upgrades.*)  You should always test on a non-critical PC first to be safe and sure.  

If in doubt, back up your data.

Applications can be reinstalled, but that data needs to be backed up regularly if you do not want to lose it.
