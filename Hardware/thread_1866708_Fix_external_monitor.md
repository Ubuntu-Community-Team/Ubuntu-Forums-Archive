### Post by mopringles on 2011-10-21
The monitor of my netbook broke down. I connected it with a external monitor successfully. But now everytime I start up, I have to manually select the external monitor and deselect the monitor of the netbook (3x fn & F4).
I thought of changing xorg.conf, but since I missed some versions of Ubuntu, I find out xorg.conf is not there anymore. I wonder how I can make my netbook start up for all the users on the external monitor with the required resolution. Can anyone help?

### Post by mopringles on 2011-10-22
It was easier than I thought. It was simple to do in the monitormenu. Disable mirroring, deselect the netbookscreen, select the right resolution on the external screen and that was that.
