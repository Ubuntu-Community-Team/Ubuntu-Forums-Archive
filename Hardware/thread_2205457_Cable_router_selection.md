### Post by tfrue on 2014-02-14
If you are getting cable, you will need either a cable modem and a router, or a cable router combo. There are many debates as which one to get, but for "generic" use it doesn't really matter.

I decided to get a really nice modem and a really nice router so I could have more bells and whistles on the router rather than being stuck with something.

There are many good brands like, Belkin, Cisco, Linksys, Motorola, NetGear, Buffalo, etc. Also, DD-WRT is another thing to keep in mind, here is a link to DD-WRT:
[http://www.dd-wrt.com/site/index](http://www.dd-wrt.com/site/index)

I hate to say it like this, but it depends on what you need. Do you need better wireless support? or LAN? Maybe both?

If you want to purchase with wireless in mind, then I would buy a router that supports the new 802.11 AC since that is becoming the new standard. The good thing about 802.11 AC is that it supports Wireless N mode as well. If you need a lot of LAN connections then maybe you should be interested in purchasing a router that supports 1Gb LAN ports and possibly an additional 5 or more port switch.

Here is the router and modem that I purchased:
[http://www.belkin.com/us/p/P-F9K1113/](http://www.belkin.com/us/p/P-F9K1113/) (Router)
[http://www.newegg.com/Product/Product.aspx?Item=N82E16825390001](http://www.newegg.com/Product/Product.aspx?Item=N82E16825390001) (Modem)

Good luck,
