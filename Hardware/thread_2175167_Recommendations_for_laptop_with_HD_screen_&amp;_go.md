### Post by sanderj on 2013-09-18
"good keyboard" is very personal and subjective. I think that only you can determine that hands-on.

Personally I would advice an Intel GPU, as Intel is completely open about it's GPUs and thus you can use OSS drivers (which are included in Linux and Ubuntu). 
I would choose an Intel Core CPU
Furthermore I would advice an SSD: once you have experienced an SSD, you won't go back because of the speed it causes.
I would advice 4GB RAM as minimum
Screen size minimum 15 inch (unless you're very mobile)
I would say a GigE-ethernet port is advisable.
I would go for Lenovo, HP or Samsung. Not Asus, MSI nor Dell. All based on my own experience (N=1)

Using that as input on Tweakers Pricewatch ([http://tweakers.net/categorie/496/laptops/producten/](http://tweakers.net/categorie/496/laptops/producten/)) the choice is easy:
