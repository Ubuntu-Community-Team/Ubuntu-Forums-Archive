### Post by than2 on 2013-08-10
Found out few unsolved, closed threads with this problem so I'm sharing my solution.

Suddenly my Lenovo stopped charging. Recognised plugged adapter, but battery percentage wasn't rising.

Only solution I found out was to make windows dual boot, install Lenovo batery management and disable battery saving (that enables battery to charge only to 50% under windows) that got itself enabled somehow. Now the battery should be charging normally.

Hope it helps.
