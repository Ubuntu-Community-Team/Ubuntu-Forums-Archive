### Post by dandnsmith on 2012-03-03
If I got the model details correctly, your beast is a 'notebook' (unfortunately). It looks as if the Optical device is not functioning correctly so:
first I would try a CD/DVD cleaning disk kit, just in case the lenses are sufficiently dirty to cause the problem,
second replace the device or plug in an external CD/DVD using USB.

Notes:
1. I fake the USB connected CD/DVD device by using an ordinary one with a USB converter - this also allows me to quickly connect an extra HDD
2. There is a [useful page](http://h20000.www2.hp.com/bizsupport/TechSupport/DocumentIndex.jsp?lang=en&cc=us&prodClassId=-1&contentType=SupportManual&prodTypeId=321957&prodSeriesId=1842153) with tech support manuals to help changing a drive etc.
