### Post by Kubunteando on 2009-06-19
Many people starts to be interested in having a computer on all day for many tasks: browse the Internet without waiting to boot it up, chat, use Skype, watch movies on TV, as media center, watch youtube...

To have this all day on computer the power consumption is an issue, so the components need to be selected so that they eat as less power as possible. But on the other hand you may require that it is powerful because you may want to watch HD movies, or even game from time to time.

So you can post here your solution running Ubuntu, you can add the specs and if you know the power consumption of the it (for desktop computers please do not include the external devices like monitor, speakers...)

### Post by Kubunteando on 2009-06-19
My current computer is a light desktop (I can carry the CPU around in a backpack) which consumes when on but at idle the amazing figure of 35W measured on AC current consumption.

It is built with a mobile mini-ITX motherboard from MSI, the MSI GME965, a 2.5GHx dual core mobile processor, 2 GB of RAM, and a 320GB of 2.5inch HDD.
It could consume less, but I want to play games from time to time and the motherboard has a PCIEz16 (amazing for a mobile motherboard, eh? :-)).
The GPU is a passive ATI HD 4550. I have also a DVB-T tunner Nova-500 to watch and record TV.

It only needs a laptop power supply of 90W, which has no fans so the computer is very silent.

I use Kubuntu 8.10 with KDE 4.2.2 and ATI proprietary drivers.

I works great!

### Post by Kubunteando on 2009-07-30
Does anybody else have any suggestions or ideas?

My thought are so that when your computer become old you can find some other use for it, like home server, media server or media center.
But many of those applications require to have the computer on many hours and who wants to do that when the consumption is 100W?

That is why I think that low idle consumption computers are the only interesting to be build. But at the same time they have to be powerful enough to play games or run your applications.

For example with mine I can Quake Wars at high settings.

I think these kind of computers are the best balance: they are small, silent, low idle consumption but powerful.
