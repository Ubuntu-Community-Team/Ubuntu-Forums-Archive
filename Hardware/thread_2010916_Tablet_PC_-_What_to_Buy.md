### Post by efflandt on 2012-06-28
Originally I went to cdw.com intending to buy an Android tablet for around $400, but on a table they had B stock Acer W500P for $350.  I could not pass that up because I sometimes need to field program/monitor electronic controls with Windows software. About size/weight of a netbook closed, but can run without included keyboard.

10.1" 1280x800 touch screen, AMD C-50 1 GHz 2 core, 2 GB, ATI HD graphics, 32 GB SSD.  W500P includes Win7 Pro (W500 Win7 Home).  I used iso on USB stick to install 64-bit Ubuntu 11.10 (and grub) to internally USB 2.0 connected memory slot (up to 32 GB SDHC). Not real quick, but everything I tried so far works including touch screen.

Getting into CMOS setup requires keyboard, holding Windows button under touch screen while turning On and hitting F2.  From there you can enable F12 on BIOS splash to select boot device.  Once installed, On button boots Windows, or Windows button+On boots grub on SD, so it is possible to boot Linux without keyboard attached.

They have had W500 for $314 and W500P for $348, but not sure if or when those are available. _[http://www.cdw.com/shop/search/Computers/Tablets-Tablet-PCs/Acer-America/result.aspx?w=C7&b=ACE&a1607=660308&SortBy=TopSellers&outlet=1](http://www.cdw.com/shop/search/Computers/Tablets-Tablet-PCs/Acer-America/result.aspx?w=C7&b=ACE&a1607=660308&SortBy=TopSellers&outlet=1)_
