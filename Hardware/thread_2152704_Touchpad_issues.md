So I read some good things about ubuntu on Samsung Series9 hardware, and since I was a little frustrated with some Win8 issues I figured I could kill the recovery partition to test Ubuntu 13.04.

So far in testing most stuff seems to work, but something is making me bang my head against the wall. I've read from more than one person that the touchpad was working better in ubuntu than in windows (and I have some issues in windows, like unintentional clicking etc). So it's frustrating to find usability even worse in ubuntu.

I don't have the same issues as in windows (big plus) but I find the "sensitivity" or lack of extremely frustrating. Mainly I can describe 2 issues:

#1 - I have issues hitting the stuff I want, url's on websites, buttons etc, I use tapping and when I move to the link/button and lift my finger to tap, I usually miss, because the movement of pointer isn't precise enough or is "skipping" along. The pointer is skipping along when moving, it's like the screen has 1600x900 res, but the mousepointer is moving in a 400x300 resolution grid layer on top. 

#2 - When using 2 finger scrolling it's not smooth at all (again probably because of same as issue1 )  When I slowly move my 2 fingers down the pad, it's like "scroll-pause-scroll-pause-scroll..."

I've been looking at synclient, but I'm clueless where to start and man page doesn't say much. Is it even possible to tweak ?

Also I'm missing features such as 3 finger guestures (ex. back) and pinch zoom.
