### Post by TheFu on 2024-01-27
> **Autodave said:**
> HP would be my first choice and Brother my second choice.

HP has been caught, yet again, being mean to their customers.  [https://arstechnica.com/gadgets/2023/12/hp-misreads-room-awkwardly-brags-about-its-less-hated-printers/](https://arstechnica.com/gadgets/2023/12/hp-misreads-room-awkwardly-brags-about-its-less-hated-printers/)  Life is too short to give money to companies who treat their customers poorly.

Laser is good. Look for a "driverless printer". If you have to install **any** drivers to make it work, then it isn't driverless. Apple lead this effort because they wanted iPhones to be able to print easily.  This has gotten most of the printers to recognize "driverless printing" and Chromebooks, Android and Linux fully supports it.  You only need to be careful to choose the right printer.

[https://openprinting.github.io/driverless/](https://openprinting.github.io/driverless/) has more details and a list of printers.  [https://openprinting.org/printers/manufacturer/Brother](https://openprinting.org/printers/manufacturer/Brother) has a list of Brother printers that are driverless.  Not the headings - avoid the "partial" and "paperweight" ones.

### Post by SeijiSensei on 2024-01-27
I have a Brother HL-3270CDW color laser. It's connected to my network with an Ethernet cable and has a static IP address. I can print to it from Kubuntu using "driverless printing" and from other devices like my Android phone using the Brother app. 

If you go this route, I strongly recommend hard-wiring the printer to the router if possible. I've read reports here that getting the printer to wake up from sleep when using wifi is more iffy.

### Post by MAFoElffen on 2024-01-28
On Ink jets. I ran through that many times. Sitting there forever, then the cartridges drying out when I did need to print something...

Someone gave me an Epson ECO Tank (ET 2760). It does dry out a bit... But I run a couple cleaning cycles, and away it goes. I have to say I am very happy with it.

I do have a very old HP LaserJet that still prints whenever I need it to. Every once it a while I shake the toner cartridge side to side to free up the toner. It just needs to use 24 pound paper through it. Any lighter paper weight and it is prone to jamming, but other than that... It has run for, jeeze, I just realized I've had it since the mid 90's. Has the extra memory and a wired Ethernet port. I guess it really was a very good investment.
