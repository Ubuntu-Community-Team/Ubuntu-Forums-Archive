### Post by sforces on 2009-12-28
This started happening with 9.10 (I'm running x64 but don't think that matters). When dock my X61 ThinkPad or undock it, my theme (all the icons on the top panel and on the desktop) change.

I found that if I just run Display app in System menu, the theme mostly goes back to normal. I don't actually change anything in Display either, just run it. But the icons on Desktop don't change back to normal until a reboot.

Anyone else having this issue? Any thoughts on how to fix it?

Update: Looks like it's related to this possible issue: [https://bugs.launchpad.net/ubuntu/+source/xserver-xorg-video-intel/+bug/444448](https://bugs.launchpad.net/ubuntu/+source/xserver-xorg-video-intel/+bug/444448)
