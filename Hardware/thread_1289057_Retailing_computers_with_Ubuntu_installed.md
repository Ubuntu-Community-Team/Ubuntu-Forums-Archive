As for "some computers not for resale" - US law recognizes what's known as the First Sale doctrine, which basically says that once they sell you the computer, you can do anything with it you like, including resell it.  *Software* companies have been trying to evade this, by claiming you're *licensing* the software rather than *buying* it, but I've never heard of this even being attempted by hardware companies (Note: the courts have been smacking down the "licensed not sold" theory on a regular basis anyway).

As for Ubuntu - under the GPL, if you distribute a binary that you have modified, then you must make the source code available.  But if you're just doing a default Ubuntu install (no modifications), then *you* don't have to provide the sources - you can just point them at the Ubuntu repositories if they want sources.

The only issue I see is that the word Ubuntu is trademarked (by Canonical), so you probably can't use it in advertising without Canonical's permission.  You should contact them about it - they *will* grant the right to use the trademark, under certain conditions.  Here's the [page]("http://www.ubuntu.com/aboutus/trademarkpolicy") with more info on the subject.
