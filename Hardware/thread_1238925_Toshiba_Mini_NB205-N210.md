### Post by oracle2b on 2009-08-15
Although I have yet to own a netbook the method I believe your inquiring about can be achieved with [http://unetbootin.sourceforge.net/]("http://unetbootin.sourceforge.net/") . *UNetbootin allows you to create bootable Live USB drives for a variety of Linux distributions from Windows or Linux, without requiring you to burn a CD*.

 If this is to much of a hassle you can use [http://wubi-installer.org/]("http://wubi-installer.org/") which is far easier to install if you have a high speed broadband connection available. *No need to burn a CD. Just run the installer, You keep Windows as it is, Wubi only adds an extra option to boot into Ubuntu. Wubi does not require you to modify the partitions of your PC*.

I just bought this model of netbook (well, it's actually the blue one, but same hardware) and I'm running the Ubuntu Netbook Remix, which is the only Linux distribution I've found which will work with almost all of the hardware.  The only thing that doesn't work are the netbook speakers, but the sound will come out of the earphone jack (don't ask my why this is).

Before you install anything else, I highly recommend that you at least borrow a USB DVD burner and use the Toshiba backup utility which is on the Windows desktop.  It makes 2 DVDs which will allow you to reinstall Windows if you change your mind later.  Otherwise I'm not sure that there is a way to get back to the factory configuration.

Anyway, you can get the Ubuntu Netbook Remix by following this [link]("http://www.ubuntu.com/getubuntu/download-netbook").  On that same page, you'll find a link about the netbooks supported.  That page will talk about how to enable the wireless and the audio through the earphone jack.  Read that before you install anything and make sure you understand the instructions.  They're very clear and the process is very simple, but you should take a look first.

Finally, as far as installing the Netbook Remix on from a thumb drive, that's actually the only way to install it as far as I've found.  You need a 1GB drive I believe.  When you download the install image, it will show you the instructions on how to do that.  It's not hard, but it does wipe the thumb drive, so make sure any data on that drive is backed up if you need it.

Hope this helps.

FYI, the netbook is awesome!  Even if you don't put Ubuntu on it, I highly recommend buying it.  The keyboard alone is enough of a reason.
