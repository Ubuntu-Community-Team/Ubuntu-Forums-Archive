### Post by sisco311 on 2010-02-26
Scanning should work with SANE using Epson's IScan.

You can download the iscan .deb package from:
[http://www.avasys.jp/lx-bin2/linux_e/spc/DL1.do](http://www.avasys.jp/lx-bin2/linux_e/spc/DL1.do)
then simply double click the .deb file to install it.
