### Post by TheFu on 2018-04-19
$500 is hardly a "budget PC".  That is a gaming rig in my book.  If you reuse parts (case/PSU/mouse/keyboard/HDD), you can build a pretty incredible desktop for $450.  You might be able to get below $350 with the way Ryzen prices have been dropping.

Just stay away from the high-end GPUs and more than 16G of RAM to keep the costs in-line.

PCPartpicker.com has examples and checks for compatibility.
 
You might find: [https://www.phoronix.com/scan.php?page=article&item=ryzen-2600x-2700x](https://www.phoronix.com/scan.php?page=article&item=ryzen-2600x-2700x) interesting.

As reference, my last "budget" build was $126.  I reused components, but bought RAM, CPU, MB all new.  CPU+MB was $99. 8G of RAM was $26.  For an office computer, for a typical office worker, it is fine.  Video, audio playback works great using the onboard GPU.  Pentium G3258 CPU (3900 passmarks)
I look at passmark scores to get an idea as to how capable any CPU is. For a slow, but acceptable desktop, 1500 passmarks is my lower limit.  A Ryzen 1600 is 13500 passmarks for reference.  My main VM server is an i5-750 with 3700 passmarks. Also have an old i7 with about 5000 passmarks. .... so  3700+3900+5000 = 12600. Performance doesn't really work through simple addition, but it is close enough since passmark workloads don't match what I do anyway.

Anyway, the point is to use *some method* of comparison based on consistent testing. Benchmarks provide that. I work from the CPU to the motherboard and almost always go with the cheapest RAM possible.  A 3% change in RAM just doesn't matter to me for 2x the cost.  If I could, I'd use cheap DDR3 RAM on my new system.

I've been looking to build a new Ryzen 1600 system since July.  This new machine would easily replace 3 of my current systems on CPU performance alone.  I do not game, but I do have virtual machines and more storage than most people. That means the GPU doesn't matter to me at all. Built-in GPUs have all handled everything I needed the last 5 yrs.
