### Post by xph8ter on 2010-09-06
I just attempted a wubi install of ubuntu 10 on my panasonic cf-18 toughbook (cf-18dhazxkm). The install seemed to go ok, but it seems to die on the reboot, as follows: when I reboot and choose ubuntu on the boot menu, it shows a screen with a text only message stating that it is completing the ubuntu installation. It then goes to a graphical display of the ubuntu logo on a black background. Then the logo goes away and I'm left with a blank screen, and it looks like nothing is happening. That's it. I left it sit like that for a few minutes, and then gave up.
 
I'd love to do a normal cd rom installation (instead of wubi), but the toughbook has no cd rom drive, and I do not yet have a usb cd rom drive. I can not boot off a usb pen drive because the bios does not support it. It only supports usb cd rom. (I've tried).
 
Any ideas?

### Post by xph8ter on 2010-09-06
I solved my own problem.  I tried booting again.  This time, after I chose "Ubuntu" from the boot menu, I noticed that the next screen gave me 5 seconds to hit a certain function key to get to more boot options.  So I hit the function key (F2 or F8 maybe ... not sure, but it said which one to hit).  Anyway, it then gave me a menu with a "Ubuntu - Safe Video" option.  I chose that one.  It then finished the install just fine, and I am now using my wubi install of Ubuntu 10 on my clunky old panasonic cf-18 laptop!  Awesome!  Hope this helps somebody someday.
