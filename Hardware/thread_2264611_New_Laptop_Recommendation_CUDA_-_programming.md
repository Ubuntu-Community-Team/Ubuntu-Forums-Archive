### Post by oldrocker99 on 2015-02-08
Actually, nVidia's own proprietary GeForce drivers are pretty good, and I have had very good results using nVidia, using their drivers, over the last seven years I've been a Linux user; most Linux forums for any distro will advise using nVidia over ATI. 

There are pretty good FOSS ATI graphics drivers ('radeon') that some ATI users prefer over ATI's proprietary drivers. The FOSS drivers for nVidia are, at present at least, inferior to nVidia's own drivers.
