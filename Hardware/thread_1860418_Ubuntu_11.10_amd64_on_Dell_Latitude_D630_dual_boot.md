### Post by RBarak on 2011-10-14
[FONT=Arial, sans-serif][SIZE=3]**UBUNTU 11.10**[/SIZE][/FONT]
 [FONT=Arial, sans-serif][SIZE=3]**On Dell Latitude D630, side by side with Windows.**[/SIZE][/FONT]
 [SIZE=3]
[/SIZE] 
 [FONT=Arial, sans-serif][SIZE=3]Hi all,[/SIZE][/FONT]
 [FONT=Arial, sans-serif][SIZE=3]I am a dumb newbie who wanted to stop using windows.[/SIZE][/FONT]
 [FONT=Arial, sans-serif][SIZE=3]And I did. Two years ago. And it was easy![/SIZE][/FONT]
 [FONT=Arial, sans-serif][SIZE=3]Ok, let's get started:[/SIZE][/FONT]
  [SIZE=3]
[/SIZE] 
 [FONT=Arial, sans-serif][SIZE=3]**A. Preparation**[/SIZE][/FONT]
 [FONT=Arial, sans-serif][SIZE=3]1. Download the new Ubuntu at:[/SIZE][/FONT]
 [SIZE=3][[COLOR=#0000ff][FONT=Arial, sans-serif]_http://www.ubuntu.com/download/ubuntu/download_[/FONT][/COLOR]]("http://www.ubuntu.com/download/ubuntu/download")[/SIZE]
 [SIZE=3]
[/SIZE] 
 [FONT=Arial, sans-serif][SIZE=3]2. Get an empty USB. It must have at least 2 GB.[/SIZE][/FONT]
 [SIZE=3]
[/SIZE] 
 [FONT=Arial, sans-serif][SIZE=3]3. If you have an older Ubuntu and want to get rid of it first, click here:[/SIZE][/FONT]
 [FONT=Arial, sans-serif][SIZE=3][http://ubuntuforums.org/showthread.php?p=11344067#post11344067](http://ubuntuforums.org/showthread.php?p=11344067#post11344067)[/SIZE][/FONT]
  [SIZE=3]
[/SIZE] 
 [SIZE=3]
[/SIZE] 
 [FONT=Arial, sans-serif][SIZE=3]**B. Making the installation USB**[/SIZE][/FONT]
 [FONT=Arial, sans-serif][SIZE=3]1. Format the USB:[/SIZE][/FONT]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]My Computer - Right-Click on the USB – Format[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif](make sure you choose filesystem fat32 or fat16)[/FONT][/COLOR][/SIZE]
 [SIZE=3]
[/SIZE] 
 [FONT=Arial, sans-serif][SIZE=3]2.  Download the program that prepares the USB:[/SIZE][/FONT]
 [SIZE=3][[COLOR=#0000ff][FONT=Arial, sans-serif]_http://www.pendrivelinux.com/downloads/Universal-USB-Installer/Universal-USB-Installer-1.8.6.8.exe_[/FONT][/COLOR]]("http://www.pendrivelinux.com/downloads/Universal-USB-Installer/Universal-USB-Installer-1.8.6.8.exe")[/SIZE]
 [SIZE=3]
[/SIZE] 
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]3.[/FONT][/COLOR][/SIZE][SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]Put Ubuntu on the USB:[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]Open Universal-USB-Installer-1.8.6.8. (the program we've just downloaded)[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]In the next window:[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]In “Step 1” choose Ubuntu 11.10;[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]In “Step 2” browse and find the Ubuntu file (for example: ubuntu-11.10-desktop-amd64.iso)[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]In “Step 3” choose your empty USB.[/FONT][/COLOR][/SIZE]
 [SIZE=3]
[/SIZE] 
 [SIZE=3]
[/SIZE] 
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]**C. Partitioning the Hard Drive, using Partition Magic**[/FONT][/COLOR][/SIZE]
  [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]If you've never done it, click here:[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif][http://ubuntuforums.org/showthread.php?p=11344144#post11344144](http://ubuntuforums.org/showthread.php?p=11344144#post11344144)[/FONT][/COLOR][/SIZE]
 [SIZE=3]
[/SIZE] 
 [SIZE=3]
[/SIZE] 
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]**D. Installing Ubuntu**[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]1. Restart the computer.[/FONT][/COLOR][/SIZE]
 [SIZE=3]
[/SIZE] 
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]2. Boot from USB:[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]Press F12 at startup - then choose “&#8235;Boot from Usb&#8236;&#8236;”.[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]If you don't have that option, here's a quick and easy howto:[/FONT][/COLOR][/SIZE]
 &#8235;&#8234;[SIZE=3][[COLOR=#0000ff][FONT=Arial, sans-serif]_http://pcsupport.about.com/od/tipstricks/ht/bootusbflash.htm_[/FONT][/COLOR]]("http://pcsupport.about.com/od/tipstricks/ht/bootusbflash.htm")&#8236;&#8236; [/SIZE]
 [SIZE=3]
[/SIZE] 
 [FONT=Arial, sans-serif][SIZE=3]3. Start Ubuntu and Install:[/SIZE][/FONT]
 [FONT=Arial, sans-serif][SIZE=3]Click “Try Ubuntu without any change to your computer”. This allows you to have a quick look around. When you're ready, click the icon on the desktop and go through the installation.[/SIZE][/FONT]
 [SIZE=3]
[/SIZE] 
 [FONT=Arial, sans-serif][SIZE=3]Don't forget:[/SIZE][/FONT]
 [FONT=Arial, sans-serif][SIZE=3]Make sure to install alongside windows! [/SIZE][/FONT] 
 [FONT=Arial, sans-serif][SIZE=3]Also, in the window, “Preparing to installUbuntu”, check “download updates while installing” and “install this third party software”. This will allow you to play media files.[/SIZE][/FONT]
 [SIZE=3]
[/SIZE] 
 [SIZE=3]
[/SIZE] 
 [FONT=Arial, sans-serif][SIZE=3]**Optional Steps:**[/SIZE][/FONT]
 [SIZE=3]
[/SIZE] 
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]**E. Getting updates:**[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]in Terminal type:[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]sudo apt-get update[/FONT][/COLOR][/SIZE]
 [SIZE=3]
[/SIZE] 
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]Where is Terminal?[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]Dash Home (first icon on the left sidebar) – type “Terminal” - click Terminal[/FONT][/COLOR][/SIZE]
 [SIZE=3]
[/SIZE] 
 [SIZE=3]
[/SIZE] 
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]**F. Installing the Restricted Extras:**[/FONT][/COLOR][/SIZE]
  [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif](if you didn't do so at installation). This gets your mp3 and videos working. [/FONT][/COLOR][/SIZE] 
  [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]In Terminal type:[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]sudo apt-get install ubuntu-restricted-extras[/FONT][/COLOR][/SIZE]
 [SIZE=3]
[/SIZE] 
  [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif](Use the Tab key to click on the <ok>)[/FONT][/COLOR][/SIZE]
 [SIZE=3]
[/SIZE] 
 [SIZE=3]
[/SIZE] 
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]**I Setting the Wifi:**[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]I had many problems with my wifi. My wireless card is Broadcom. One of these always works:[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]1. Plug to cabled internet. Then:[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]System Settings (on the left sidebar) - Additional drivers – choose driver – activate[/FONT][/COLOR][/SIZE]
 [SIZE=3]
[/SIZE] 
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]2. Plug to cabled interne. (If you tried the previous, go back and deactivate it). Then:[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]In Terminal type:[/FONT][/COLOR][/SIZE]
  [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]sudo apt-get install firmware-b43-installer[/FONT][/COLOR][/SIZE]
  [SIZE=3]
[/SIZE] 
 [SIZE=3]
[/SIZE] 
 [FONT=Arial, sans-serif][SIZE=3]**J. Get Hebrew (or other languages):**[/SIZE][/FONT]
 [FONT=Arial, sans-serif][SIZE=3]1. To install Hebrew:[/SIZE][/FONT]
 [FONT=Arial, sans-serif][SIZE=3]System Settings - Keyboard Layout - “+”[/SIZE][/FONT]
 [SIZE=3]
[/SIZE] 
 [FONT=Arial, sans-serif][SIZE=3]2. To be able to change from English to Hebrew:[/SIZE][/FONT]
 [FONT=Arial, sans-serif][SIZE=3]System Settings - Keyboard Layout - Options - Key(s) to change layout[/SIZE][/FONT]
 [SIZE=3]
[/SIZE] 
 [FONT=Arial, sans-serif][SIZE=3]3. For Language support[/SIZE][/FONT]
 [FONT=Arial, sans-serif][SIZE=3]System Settings - Language Support - Install/Remove Languages – choose your language[/SIZE][/FONT]
  [SIZE=3]
[/SIZE] 
  [SIZE=3]
[/SIZE] 
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]**K. Get WINE**[/FONT][/COLOR][/SIZE][SIZE=3]
“[/SIZE][FONT=&quot][SIZE=3]Wine” allows Linux to run Windows programs.[/SIZE][/FONT]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif][FONT=&quot]In Terminal type:[/FONT][/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=&quot]sudo apt-get install wine[/FONT][/COLOR][/SIZE]
 [SIZE=3]
[/SIZE] 
 [SIZE=3][COLOR=#000000][FONT=&quot]After this you can open programs by right clicking and opening with wine.[/FONT][/COLOR][/SIZE]
 [SIZE=3]
[/SIZE] 
 [SIZE=3]
[/SIZE] 
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]**L. Installing additional programs: For example “Skype”**[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]1. Finding the program we want:[/FONT][/COLOR][/SIZE]
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]open “Ubuntu Software Center” (the magic suitcase on the left sidebar) – type “skype”[/FONT][/COLOR][/SIZE]
 [SIZE=3]
[/SIZE] 
 [SIZE=3][COLOR=#000000][FONT=Arial, sans-serif]2. Choose from the list and install.[/FONT][/COLOR][/SIZE]
 [SIZE=3]
[/SIZE] 
 [FONT=&quot][SIZE=3]This is the best way to install programs. If you don't find what you want: find a “.deb” file, then right click and open it with the Ubuntu Software Center.
Alternatively find a tutorial that begins with “sudo apt-get install”
[/SIZE][/FONT][SIZE=3]
