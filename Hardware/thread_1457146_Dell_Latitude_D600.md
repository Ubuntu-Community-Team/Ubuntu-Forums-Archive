### Post by cascade9 on 2010-04-18
That system should run well with ubuntu (xubuntu would work as well, but its not 'lighter' than ubuntu). 

The only issue would be your video card- ATI Mobility Radeon 900. ATI dropped support for that card, so you can only use the free drivers. As far as I know you cant run compiz on the free ATI drivers. 

If you want to use 10.04 wait for it to stop being a beta release, like Jose Catre-Vandis said its not that long untill it is released.

### Post by Blackcamaro8 on 2010-08-28

I know this thread is a few months old, but I would like to clarify that I am using a laptop of similar configuration. It's a Latitude D600 with 1GB of RAM, and a 1.6Ghz Pentium M.
