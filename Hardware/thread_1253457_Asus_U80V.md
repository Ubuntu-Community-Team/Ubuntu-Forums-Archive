### Post by Emdaer on 2009-09-26
Hi Mark

I recently installed ubuntu on this ASUS U80V and it's working okay.

1. I made the ethernet controller work, see this thread [http://ubuntuforums.org/showthread.php?t=1266267&highlight=asus+u80v](http://ubuntuforums.org/showthread.php?t=1266267&highlight=asus+u80v).
2. I haven't got the keyboard backlight working, but i don't care.
3. The screen brightness works.
4. The microphone does work.
5. The sound control does work.
6. Only 3 GB of the 4GB installed recognized.
7. I haven't tested the webcam.

Try maybe enabling Ubuntu Backports if it still doesn't work for you. Backports will get you some drivers, that aren't with ubuntu initially.
Remember, that you can test if ubuntu backports works for you before you install the OS on your computer. Just insert the install CD and choose "try ubuntu without any changes to my computer".

I hope this helps.
Regards
Emdaer
