### Post by sudodus on 2012-07-19
An easy but expensive way to make any distro boot faster is to replace the HDD with an SSD ;-)

But seriously, what is more important: fast boot or fast start of applications? The extreme case is to boot the computer without any graphic desktop environment at all, and with very few services running. But then it will take longer to start browsing the internet, to start checking email, start printing a text etc.

If there is some program running in the background providing a service, that you don't need (at least need very seldom, then you can skip that from the boot or log in tasks. But ***backup*** your system before starting to tweak it!!!
