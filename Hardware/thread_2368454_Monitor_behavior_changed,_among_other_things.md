### Post by QIII on 2017-08-12
Hello!

I'm pretty sure what you are talking about is a hardware problem.  No matter that your machine is an AIO, if your monitor is turning back on after power has been turned off then the issue has nothing whatsoever to do with either OS, but with the design of the hardware.  The monitor power switch, for instance.

This is confirmed by the fact that it happens with both OSes -- that indicates that the issue is OS independent.  One thing that is often done to differentiate between a hardware fault and an OS fault is to run another OS to see if the issue persists.  If it does, the diagnosis forks to hardware.

You say that the issue was caused by installing Ubuntu because it was noticed after installing Ubuntu.  The assumption that some condition is caused by an event simply because the condition appears after the event is known as the *post hoc ergo propter hoc* (after this, therefore because of this) logical fallacy.  Despite the fact that there is a temporal relationship between the two, there may not be a causal relationship between them.  So unless you can establish that there is a causal relationship, you cannot assume there is one.

This is something you should pursue with the hardware vendor.

As to your other problems, please start a new thread for each of them in General Help for now.  If we decide that they need to be moved, we will do that.  Please don't lump them all into one thread.
