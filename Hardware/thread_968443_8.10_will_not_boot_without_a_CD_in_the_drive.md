### Post by kcy29581 on 2008-11-02
Hi all,

I am having a very bizarre problem with a laptop - Gateway 7811-FX. I have installed Ubuntu 8.10 via the alternate disc (so I could install LVM) and now when the laptop will not boot unless I have a disc in the CD/DVD drive.

Detailed explanation below:

** Unsuccesful boots:**
 - If I allow the laptop to boot without a CD in the drive, the initial loading bar appears and after a few seconds I see a blinking cursor on the top-left of the screen. By pressing CTRL+ALT+F1 the test appears as follows:
     Boot from (hd0,0) ext3 *numbers*
     Starting up ...
     Loading, please wait...

 - If I try to boot with the recovery option in GRUB (again, without a CD in the drive), the boot process stops at this line:
     [16.634032] Uniform CD-ROM driver Revision: 3.20

** Successful boots:**
 - If I try to boot with the Ubuntu 8.10 alternate disc in the drive, and select "Boot from first hard disk", the boot process continues without a problem

 - If I try to boot with any disc in the drive, the boot process completes without a problem

 - If I try to boot with the CD drive tray ejected (no disc), the boot process completes without a problem

Any ideas?
