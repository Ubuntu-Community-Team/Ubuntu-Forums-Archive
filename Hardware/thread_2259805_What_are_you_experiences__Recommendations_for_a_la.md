### Post by weatherman2 on 2015-01-07
There are USB video adapters  that allow you to add another screen to any computer with a USB port.  Of course these are supported in Windows.  Not sure about Linux.

Depending on the laptop design, upgrading the hard drive to an SSD of any size may be very easy.  Some laptops allow easy access to the hard drive to remove/replace it.  You can then clone the old drive to the new one using a USB hard drive adapter or enclosure plus cloning software like Clonezilla.  Some laptops bury the hard drive under the keyboard and/or motherboard so check that out before assuming you can upgrade the SSD.
