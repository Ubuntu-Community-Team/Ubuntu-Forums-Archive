### Post by scragar on 2008-10-24
I think it's easier for you to just test your confg than to actualy try checking hardware listings(which are updated by users, so there's always a chance of it being wrong or out of date), you can download the disk and install it inside vista using "wubi" which is an EXE that can be run from inside windows. From the information available elsewhere on the forums it looks like your wireless card will work if you have a wired network you can connect it to while it downloads the first set of updates and scans the repositories for drivers available.
