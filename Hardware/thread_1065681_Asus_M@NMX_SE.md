### Post by martinbaselier on 2009-02-10
The iso you download is both a live-cd/installer cd. So you'll find out if you boot from it. If your computer can boot from usb, I would advise you to install unetbootin under windows, select ubuntu and install it on the usb device. (it will download the iso, put it on the stick and make the stick bootable)

About software: There's an add/remove item under system, where you can choose your program and it will automatically download and install it.
