### Post by Jive Turkey on 2009-11-26
Check the manufacturer's website to see if they offer an upgrade.  Get the specs on it (ie size speed latency voltage etc).  Then go looking at other vendors you trust for better prices on it.  Crucial (crucial.com) has a windows only analyser that might be able to tell you how many slots you have and offer some upgrade options.  The RAM itself is located under one of the removable panels on the bottom of your laptop.  You can probably find some tutorial videos on changing laptop memory.

According to [this CNET review]("http://reviews.cnet.com/laptops/hp-compaq-business-notebook/1707-3121_7-31950813.html") you are using:

# Technology: DDR2 SDRAM - 533.0 MHz 
# RAM configuration features 2 x 512 MB 

While the size is not mentioned, it is probably SODIMM. You have 2 memory slots, so you can probably just add memory to your existing RAM.

I suggest you get an additional 1GB SODIMM DDR2 SDRAM 533 Mhz (or higher).

Installing it should be pretty easy; usually just a matter of opening one screw, popping the lid, and inserting (carefully!) and latching the RAM (By slowly and gently pushing down until you hear a click).
