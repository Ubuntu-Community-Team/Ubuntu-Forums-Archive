### Post by Jupp3 on 2009-03-28
I have MP610, and it's been working well enough for me. Earlier I had to use the separate "ScanGear" program for scanning, but at least with Ubuntu 9.04 alpha (probably 8.10 too, I skipped it), the scanning works with Sane aswell, which makes things more straightforward.

On the printing side, it's automatically detected, and open source drivers are used. However, they don't support printing beyond 600x600 dpi. You can download the official Canon drivers, but they don't support it either (but you can manually edit config files to support higher resolutions).

OS issues aside, I like the fact it has 5 separate ink tanks, which means you only have to replace the ones that don't have any ink left. Also, the tanks on new printer weren't "half empty", as they are on seweral budget models.
