### Post by IcarusR on 2010-03-06
I don't think you are going to be able to achieve this with software. 
If the drive is active in the bios & the OS 'sees' it then it is using power, & resources, even if it is not doing anything.
You should be able to physically remove it from laptop quite simply. 
Probably a couple of screws in bottom case below it, undo them & it should just pull out. They are modular.
