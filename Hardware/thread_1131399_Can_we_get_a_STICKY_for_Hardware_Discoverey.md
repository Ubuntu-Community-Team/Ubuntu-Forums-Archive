### Post by TBerk on 2009-04-20
I'd like to see a sticky up at the top of this forum that goes toward helping find, discover, and test hardware. 

Along these lines would be some common to some, obscure to others Command Line er 'commands'. 

I've been compiling some commands in a text file but who knows what they don't yet know?

An example:

```
$ lspci -vvnn
```

from Wikipedia:
[http://en.wikipedia.org/wiki/Lspci](http://en.wikipedia.org/wiki/Lspci)
> lspci is a command on Unix-like operating systems that prints detailed information about all PCI buses and devices in the system. It is based on a common portable library libpci which offers access to the PCI configuration space on a variety of operating systems.

lspci(8) - Linux man page
[http://linux.die.net/man/8/lspci](http://linux.die.net/man/8/lspci)
> lspci is a utility for displaying information about all PCI buses in the system and all devices connected to them. 


Perhaps this is already been created, compiled, and appended somewhere in a FAQ but having readily accessed and refered to in the Forum itself might help Newbies, Intermediates, and perhaps cut down on the exasperation of those lurking Grand Wizards.
