### Post by beanerjooe on 2009-02-07
I have the same computer (Presario C700) and was having the same problem yesterday. At first I thought it was kinda creepy, lol. I haver Ubuntu 8.10. First my computer did a huge driver and system update. After this driver and program update my Wifi still did not work. So I played with the Wifi drivers. Under System>Administration>Hardware Drivers> I set it up so the **Broadcom B43** wireless driver was on but the **Broadcom STA** had to be deactivated. My Wifi works but if it does not I have to play with this setting to make it work. Hope it works.
