### Post by Mark Phelps on 2015-01-14
Combination of falling off the desk while running and Windows showing it as RAW is a pretty good indication of hardware failure on the drive, most likely, a head crash -- and if that is true, there is NO software that will recover anything from the drive, as it would need to be repaired before it can be used again.

Repair requires disassembly and reassembly in a "clean room" environment and is VERY expensive -- minimum estimates START at $1000 USD.
