### Post by cascade9 on 2010-01-02
It should work. In 99.99% of cases anyway, the only reason why you might not be able to run one 533/xxxx CPU but not a different 533/xxxx CPU is if the motherboard doesnt have the right multiplier.

Be careful of how much power the late p4s use though, if you currently have an early P4 533 like a 'northwood' 2.26 @ 56watts TDP, moving up to a prescott 2.80 @ 89watts TDP could put a bit more strain on your power supply. 

BTW, northwood = 512K cache, prescott = 1024k cache.
