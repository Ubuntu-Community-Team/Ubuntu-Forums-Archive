### Post by Super Jamie on 2012-10-05
I downloaded the Windows driver for the touchpad and found

```
; Localizable Strings
Provider            = "Sentelic"
DiskId1             = "Finger Sensing Pad Driver Installation Disk"
AVC.DeviceDesc      = "Finger Sensing Pad"
AVC.DriverDisplayName = "Finger Sensing Pad Driver"
AVC.DriverDisplayVersion = "9.2.1.6"
AVC.DriverReleaseDate = "12/16/2011"
AVC.DriverCfg       = " "
fspad.SvcDesc       = "Finger Sensing Pad Driver for Windows 2000/XP/Vista/Win7"
fspadsvr.SvcDesc    = "Finger Sensing Control Service"

```

A Google search for "finger sensing pad linux" turned up this project who make drivers, distributed in a deb file:

[http://sourceforge.net/projects/fsp-lnxdrv/](http://sourceforge.net/projects/fsp-lnxdrv/)
