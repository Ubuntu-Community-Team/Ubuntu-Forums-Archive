### Post by ahallubuntu on 2012-02-18
Did you try using the previous kernel in Grub?  If you don't get a Grub boot menu when you start up, you can usually get one by holding down the Shift key.  Then, choose "previous" to get a previous kernel.  If that fixes it, you can remove the new one though I'm not sure how to do that in 11.10 .
