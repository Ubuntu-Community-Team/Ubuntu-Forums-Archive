### Post by Yellow Pasque on 2017-01-02
First, get rid of this mess:
```
snd_hda_intel: model=dell-vostro enable=1 index=0
snd_hda_intel: model=auto
snd_hda_intel: model=,alc256-chrome
```

Next, I think I would file a bug against Sound(ALSA): [https://bugzilla.kernel.org/enter_bug.cgi?product=Drivers](https://bugzilla.kernel.org/enter_bug.cgi?product=Drivers)
Make sure you restart your system after removing the above lines so they don't show up in the report.

Normally, I would suggest trying the latest sound driver code before reporting a bug, but the PPA is borked right now: [https://wiki.ubuntu.com/Audio/UpgradingAlsa/DKMS](https://wiki.ubuntu.com/Audio/UpgradingAlsa/DKMS)
Also, I don't see any specific fixes for your PCI ID (1043:12f0) that would make me hopeful that the issue has been fixed upsteam.
