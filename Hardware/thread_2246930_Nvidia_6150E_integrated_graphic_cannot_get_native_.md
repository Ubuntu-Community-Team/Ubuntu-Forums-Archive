### Post by Vladlenin5000 on 2014-10-04
Hi, welcome.

You may need proprietary nvidia drivers.
If you're using standard Ubuntu just go to System Settings > Software&Updates. Select the rightmost tab - Additional drivers -, wait a few moments and check whether or not there are offers. You should see different versions and among them *probably* the 304 version (recommended for your chip). Select it and allow it to do its thing (downloading and installing), reboot when finished and try to change the resolution.
