### Post by sudodus on 2013-07-12
Try several of the Ubuntu flavours. A new computer can run them all, but you might need to boot with nomodeset and then install a proprietary driver for the graphics.

If you don't want to download them all and try live, you can install one and then install the various desktops and select session at the login screen. See this link

[http://www.psychocats.net/ubuntu/index.php](http://www.psychocats.net/ubuntu/index.php)

If you start with standard Ubuntu, you can also install the other desktops with

```
sudo apt-get install kubuntu-desktop
```
```
sudo apt-get install lubuntu-desktop
```
```
sudo apt-get install xubuntu-desktop
```

and remove what you don't need according to Psychocats.
