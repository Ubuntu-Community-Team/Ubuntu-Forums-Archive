### Post by Autodave on 2021-07-19
Sounds like either a sticking key or a bad keyboard.  First, try disconnecting ALL USB devices from the machine and checking if the keyboard works.  If still no good, turn laptop off and start rapidly tapping every key many times.  If you have a can of compressed air, blow under keys and repeat. Also bang on keys while holing laptop upside down.  (I know this is kinda difficult.)  If that all fails, get an external keyboard and try it.
