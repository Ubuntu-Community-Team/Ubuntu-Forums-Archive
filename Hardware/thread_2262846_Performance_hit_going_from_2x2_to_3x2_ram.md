### Post by Yellow Pasque on 2015-01-27
The answer to your question depends on whether you really use more than 4GB of RAM. If you regularly use VM's, have a lot of browser tabs open, and/or edit large images, more RAM might be helpful. If not, then it's a waste of money.
Dual channel RAM doesn't make a huge difference, and I think you'd be pressed to notice it in anything but benchmarking and maybe demanding gaming. 

If you're really curious about the effect of dual channel, you can probably do an experiment and disable it by moving the second RAM stick. I'm not sure if it's still the case with current boards, but when I built a lot of systems a few years ago, the mobos required the RAM sticks to be in particular slots for dual channel operation,.

> Would it matter that there would be different brands of ram?
Theoretically, it shouldn't matter, but RAM and motherboards can be finicky.
