### Post by mikezila on 2009-05-30
There's a livecd somewhere that has fatx support.  It's what I used to backup my Xbox's HDD when I upgraded it.

I found it was easier to make a xboxpad-USB cable and use my computer to upload the savegame to an official memory pack.  You can find the memory pack at pretty much any gamestop for $3, and you can make an xbox-usb cable with any usb extension cable and any xbox cable.  Just buy a cheap extra controller and go at it.

I had to use windows software to write to the memory unit, but you only need to do it once.
