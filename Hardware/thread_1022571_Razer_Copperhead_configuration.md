### Post by tommcd on 2008-12-27
See this page from the Ubuntu wiki:
[https://help.ubuntu.com/community/RazerCopperHeadMouse](https://help.ubuntu.com/community/RazerCopperHeadMouse)
As noted, that page is a bit out of date. The xorg that comes with Intrepid does not need a complete xorg.conf anymore because input-hotplugging is used. It should still honor entries made in the xorg.conf file though.
You can try editing your xorg.conf as indicated in that wiki page to see if it helps.
First, back up your xorg.conf like this:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak
```
Then edit xorg.conf and put in that section about the razer mouse right above the **Section "Monitor"** part of your xorg.conf. Leave a space between sections. Save the file, then restart X (ctrl + alt + backspace) or reboot, and see if it improves the mouse. If it don't you can replace your original xorg.conf like this:
```
sudo cp /etc/X11/xorg.conf.bak /etc/X11/xorg.conf
```.
