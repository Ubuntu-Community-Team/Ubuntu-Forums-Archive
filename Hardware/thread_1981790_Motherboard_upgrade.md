### Post by nariub on 2012-05-17
You should be cool
you can even use clonezilla or something to move the image of the old harddrive to the new harddrive if you want..

ubuntu doesn't tie the installation to the present hardware.

there are a few things you may run into.. such as the network control center may create a new interface if you have your current config tied to a MAC address for a network card that does not exist in the current setup..

you should be cool

Standard disclaimers should follow here:
Your mileage may vary,
if your original is 32bit it will not be magically 64bit on the new one even if the new hardware supports 64bit,
etc,
and so on.
