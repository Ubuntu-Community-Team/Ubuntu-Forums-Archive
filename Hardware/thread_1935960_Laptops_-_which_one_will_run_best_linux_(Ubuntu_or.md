### Post by bcarlowise on 2012-03-19
I've used Dell, IBM and HP laptops for the last 5 years and all have run Ubuntu just fine......this includes Intel intergrated graphics, ATI and Nvidia graphics. My current laptop has the ATI HD4250 graphics and it will work using the stock Radeon drivers, the open source ATI drivers and the ATI proprietary drivers. I use the proprietary drivers because they provide the best performance and keep the machine running coolest. I thhink nvidia has typically better support for Linux but ATI, in the last 6 months, seems to have gotten better about providing current and working drivers. Unless you're into gaming or video editing/processing I'm not sure you would really need a system with a discrete graphics processor....the intel graphics machines would work perfectly fine.
