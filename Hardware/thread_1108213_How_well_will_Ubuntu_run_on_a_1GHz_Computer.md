### Post by jespdj on 2009-03-27
The speed of the processor is most likely not the limiting factor; 384 MB is the minimum required amount of memory needed to install the regular version of Ubuntu.

It will probably run OK, but if you can put more memory in it, it will probably run a little better.

You could also try Xubuntu, which is an official Ubuntu variant especially for older computers with less memory and slower processors. I have installed Xubuntu 8.10 on an old 1 GHz Pentium III laptop with 256 MB memory, and it runs quite well on it.
