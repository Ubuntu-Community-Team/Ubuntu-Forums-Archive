tardis0011,

you may need to play with synclient, here is the link to the ubuntu documentation page...

[https://help.ubuntu.com/community/SynapticsTouchpad](https://help.ubuntu.com/community/SynapticsTouchpad)

and here is a linux mint thread that may be of interest....

[http://forums.linuxmint.com/viewtopic.php?f=49&t=40277](http://forums.linuxmint.com/viewtopic.php?f=49&t=40277)

### Post by theDaveTheRave on 2010-03-31
tardis0011,

from this forum thread on mint8 it seems that you are right about there being no xorg.conf by default.

[http://forums.linuxmint.com/viewtopic.php?f=59&t=42279&start=0](http://forums.linuxmint.com/viewtopic.php?f=59&t=42279&start=0)

The guy in this thread is offering his as a sample xorg.conf, so I'm going to do the same here for you. it should then be a relatively simple (ha ha ha) matter of implementing the solution on the page I previously mentioned.

here is my very simple xorg.conf

> 
/etc/X11/xorg.conf
# xorg.conf (X.Org X Window System server configuration file)
#
# This file was generated by dexconf, the Debian X Configuration tool, using
# values from the debconf database.
#
# Edit this file with caution, and see the xorg.conf manual page.
# (Type "man xorg.conf" at the shell prompt.)
#
# This file is automatically updated on xserver-xorg package upgrades *only*
# if it has not been modified since the last upgrade of the xserver-xorg
# package.
#
# Note that some configuration settings that could be done previously
# in this file, now are automatically configured by the server and settings
# here are ignored.
#
# If you have edited this file but would like it to be automatically updated
# again, run the following command:
#   sudo dpkg-reconfigure -phigh xserver-xorg

Section "Device"
	Identifier	"Configured Video Device"
EndSection

Section "Monitor"
	Identifier	"Configured Monitor"
EndSection

Section "Screen"
	Identifier	"Default Screen"
	Monitor		"Configured Monitor"
	Device		"Configured Video Device"
