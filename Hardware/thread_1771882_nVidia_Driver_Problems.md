### Post by kacperpl1 on 2011-05-31
Nvidia Optimus linux support is not there yet. You should go to bios and check if you can disable integrated gpu so you will run on discrete nvidia card only or try to find out how to disable nvidia card by software. 

Your sandy bridge gpu is fast and you could do without nvidia card if you're not going to run some high end games. In the end you must choose one of them.

### Post by adam.webb on 2011-05-31
Thanks very much for your reply.

I had discovered the same thing via lots and lots of googling, but "BumbleBee" seems to be part way to Optimus on Linux.

I am not a gamer, and 99.9% of the time the onboard graphics chip will be more than adiquate for my games of freecell and procrastinating on facebook, but Ubuntu would force me to run in "Classic Mode", not allowing me to use Unity and all the new shiny stuff.

As soon as I installed BumbleBee and rebooted, It's allowing me to log in to the normal ubuntu mode, and use Unity as usual. So this seems to have solved my problems for now.

Thanks again for your response!
