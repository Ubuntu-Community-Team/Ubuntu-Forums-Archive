I've been using Ubuntu for about a year now after migrating from Windows 7. I find Ubuntu and Linux great (enjoying unity as well!). But one thing has always surprised me about Ubuntu - why aren't laptops and simple common sense power management supported out of the box?

Here's a couple of simple things I mean that MS Windows does really well:

[LIST]
[*]When booting up on AC default screen brightness to high
[*]When AC us unplugged dim screen brightness down

[*]All the above brightness levels, settings, etc. configurable via GUI
[/LIST]

Windows has always done this stuff really well. ### Post by Cheesehead on 2011-06-19
Different hardware manufacturers implements those features differently. They tell Microsoft how. They don't tell us. Sometimes they refuse to answer even if we ask politely.

Linux and Ubuntu developers take the software as far as they can. But if the hardware manufacturer refuses to tell anyone how to detect 'On battery power' for certain models, then obviously the software won't work.

Conversely, if you find a way to detect such events that the software isn't using, then please file a bug report!
