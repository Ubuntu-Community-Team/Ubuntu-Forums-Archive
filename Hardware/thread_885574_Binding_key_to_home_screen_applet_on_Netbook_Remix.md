Hey all,

I just got an Acer Aspire One and put Ubuntu 8.04.1+Netbook Remix on it. It&#347; working well so far. 

What I would like to do is bind a key to the Home Screen applet which serves as the main menu in Netbook Remix. In the Aspire&#347; native OS (Linpus Lite), the Home key (equivalent of the Windows key) is bound to their main menu. Is there a way to duplicate this?

Allan

### Post by Suvarin on 2008-08-11
Yes. Preferences -> Keyboard shortcuts -> Scroll down to "Hide all windows and focus desktop" (the last item) -> Select the shortcut column -> Press the home key. Done!
