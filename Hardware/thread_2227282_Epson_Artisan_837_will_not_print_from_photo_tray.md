I have an Epson artisan 837 printer that I'm using w/ 14.04lts. I've tried the driver that comes from epson (installed from epson-inkjet-printer-201112w_1.0.0-1lsb3.2_amd64.deb), and that appears to be recommended at openprinting.org, and another one that seems to be included by default in the ubuntu printing system. I would appreciate suggestions as to whether there is a way to get photo printing to work.

Edit:
The driver from epson prints from the photo tray. There is no separate option to select the photo tray as the source, but in the paper size list, there are entries that read e.g. "4x6 (Photo Tray)".
