### Post by donniesito on 2010-01-01
Hey all! I've run into a problem that I can't seem to figure out for myself, though I have tried, so I decided to come here and ask some advice :-)

I have a new laptop with Windows 7 preinstalled. I used a utility to non-destructively resize the Win partition and created 2 new partitions for linux (one for the OS and the other for swap).

I downloaded Ubuntu 9.10 x64, and did the whole "try Ubuntu" thing when I booted it from CD and everything worked fine. While using the Live CD, the "restricted drivers" thing came up, and obviously I ignored it since I was running from the live CD. 

I installed the system onto the HD and it works fine in that I can boot to Windows or Ubuntu, and the screen resolution is fine, and desktop effects work.. However, since install, it no longer detects any hardware that needs the restricted drivers, and it doesn't detect my wireless OR wired NICs..

Any ideas?

### Post by donniesito on 2010-01-01
Well, I have wireless networking working now. Fortunately a friend let me connect to his router via ethernet. After I let the system apply all the updates and restarted, it finally detected the wireless card & allowed me to install the drivers. 

So it's fixed, thanks to having hard-wired internet access ;-)
