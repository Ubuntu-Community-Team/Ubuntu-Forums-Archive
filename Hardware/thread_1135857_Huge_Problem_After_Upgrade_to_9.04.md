### Post by MichaelPalin on 2009-04-25
Not a whole day, just an hour in the worst case and if you have the CD already.

Install Ubuntu 9.04 mounting "/" in the same place of the hard disk you have it now, and all the configuration files will be there when entering Ubuntu again, you will only have to reinstall the programs that don't come with the base installation (which is piece of cake with Synaptic) and that's it.

Still, if the problem persists or you want to guarantee solving it, delete all gnome configuration files in your home folder (as the problem seems to be related to gnome) and let the installation create a new, cleaned ones. Of course, you'll have to reconfigure Gnome afterwards, but it takes only some minutes.

Good luck!
