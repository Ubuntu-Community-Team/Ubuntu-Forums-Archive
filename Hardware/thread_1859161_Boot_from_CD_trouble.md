Hi,
Welcome to the Ubuntu forums!

I came across [the user manual](http://redhotinternetmarketing.com/dl/ToshibaUserGuide.pdf) for this machine, and it said this:
> While holding down F12 key on the keyboard, turn on your computer - when the TOSHIBA Leading Innovation >>> logo screen appears, release the F12 key.

This should get you to a menu where you can select the boot media.

Hope this helps,
Ezra.
