### Post by snowpine on 2010-06-23
I believe you can log out and choose Session: Gnome Desktop from the GDM login screen. If this is the case, reinstalling is not necessary.

But yes, Ubuntu Desktop should run equally well on any system capable of running UNE; they are the same core system underneath. Try using the Search feature to read other users' experiences with the HP Mini; I have no personal experience with that hardware.
