### Post by Themotorman on 2011-01-20
I have just installed Ubuntu 10.04 onto a ZE5700 laptop. It runs great and even found the driver for the broadcom  wifi card. To run this you need to increase the RAM to max for the laptop. Also make sure that your HD is fully functional. An 80 GB is now only about $40 so it is not worth messing with old ones. I did use a 9.04 CD first then updated but it would be quicker to just download and burn a CD with 10.04.
