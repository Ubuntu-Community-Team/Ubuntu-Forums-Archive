I'm a new Ubuntu user running 9.10.  I dual boot with XP until I can get comfortable with Ubuntu. Recently I picked up a new SATA internal hard drive a 2tb WD 20EADS which I formatted in windows in NTFS format so I could view/transfer files to that drive in both windows as Ubuntu. The only problem is that Ubuntu doesn't seem to always find the drive.  On about 10% of the boots it finds the drive, the rest of the time it's like it doesn't exist.  Any suggestions/ideas?  Thanks!

Edit:  I resolved this issue by running checkdsk in windows, apparently there were some read errors that needed to be fixed.
