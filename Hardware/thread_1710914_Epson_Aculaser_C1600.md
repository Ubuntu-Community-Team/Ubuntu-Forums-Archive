### Post by ceminino on 2011-04-11
I reply to my own question in case others are trying, 
The Aculaser C1600 works with the following driver: Komica Minolta/ Magicolor 1600w. 
Use the ICM color profile in the Color Mode settings or you'll get strange blurry results.
