Is there something you want to do, which this system won't do? It's not all that fast, and the GMA 950 is not wonderful, but 2 GB of memory is excellent for Mint 13. Some time ago, I bought an ATI PCI graphics card on eBay, (for $10) and it worked just fine.

If you find the computer to be slow, you might try Xubuntu or Lubuntu.

If you don't want the computer, I'm sure you have neighbours who would be delighted to have it.
