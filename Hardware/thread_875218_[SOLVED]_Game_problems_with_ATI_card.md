### Post by Rocket2DMn on 2008-07-30
Have you turned off compiz while you play?  Using desktop effects while trying to play 3d games can cause a lot of flickering issues as well as lag.
```
metacity --replace
```
To re-enable:
```
compiz --replace
```
Also check out the program fusion-icon
