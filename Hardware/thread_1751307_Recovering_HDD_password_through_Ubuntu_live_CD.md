### Post by ajgreeny on 2011-05-06
What do you mean by a "HDD password"?  Is the drive encrypted?  If so, I don't think you will get anywhere with a Ubuntu live CD, though there are some system recovery linux distros which might help, though I doubt it.
[http://ubuntu-rescue-remix.org/](http://ubuntu-rescue-remix.org/)

If you mean that you have simply forgotten your user password for a ubuntu linux install, you can get it back with info from:-
[http://www.psychocats.net/ubuntu/resetpassword](http://www.psychocats.net/ubuntu/resetpassword)
