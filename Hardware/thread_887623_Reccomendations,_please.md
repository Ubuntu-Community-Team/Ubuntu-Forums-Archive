### Post by tamoneya on 2008-08-12
as for k/x/ubuntu it doesnt really matter.  Use what you prefer.  I personally use ubuntu.  As for 32 bit and 64 bit you can use either.  You have a 64 bit processor so you can run OSes that are 64 bits or less.  In your case I would actually recommend that you use the 32 bit version unless you plan on upgrading your RAM.  The reason being that until you get 4GB or more RAM the gains of the 64 bit OS over the 32 bit OS are usually not noticeable.
