### Post by r57 on 2008-05-30
Hi! Since I have nVidia graphics, I cant help U so much because I set all this in nvidia-setttings. But, i had recently found myself without nvidia drivers and X.org handles external monitor very well.

In my case, I had 1280*800 on laptop an 1280*1024 on ex.LCD. So on LCD there was full desktop and on laptop there was bottom missing. But it didnt mind me since I only want to use LCD.

What I want to say is that it depends on how U want to use it. I think, no config will be a problem. Try borrow monitor from friend and try settings in Preferences>Monitor resolution settings.
