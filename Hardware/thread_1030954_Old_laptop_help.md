Debian actually seems to get lighter over time - I seem to remember the version before Etch saying that the minimum amount of RAM is 32MB. Etch says 24MB.

This is to say nothing of the desktop environments available on debian, which probably get bloatier over time, but I don't use and cannot comment on those authoritatively.

My setup consists of Debian Lenny+Openbox+lxpanel. You might want to do openbox+LXDE if it's for your friend, as I'm sure you'll want a file manager and such. With two CLI IRC clients, one CLI torrent client (all in instances of screen), five terminal emulators, Transmission, Pidgin, Firefox, Akregator and Sylpheed running, I use 244MBs of 512MBs of RAM. Kill akregator, the unused terminal emulators, and it's akregator's associated daemons, and I'm down to 201.

That'd probably drop by another 50 if I killed this browser, but then I'm not sure I'd be motivated to log back on to this forum to finish this post. My name is awfully long and takes a while to type.
