### Post by IcarusR on 2010-08-26
Some windows games work under wine. Wine lets you run Windows software on other operating systems. 
Don't play games myself so don't know if runescape will work.
Youtube is a web site & works with Firefox.
Yes windows .txt .doc .xls files all work with openoffice in Ubuntu.
If your webcam works with the live version of Ubuntu then it should work when installed.

I would suggest shrinking your windows partition,creating a second partition & installing Ubuntu alongside windows, then you can grab whatever files you need from windows onto Ubuntu. 
Also if your games do not work with wine then you can boot into windows to play them.
