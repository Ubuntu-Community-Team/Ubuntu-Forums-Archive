### Post by PABlanche on 2014-10-08
Hi,

Today 10/08/14, I run a regular update like each week. At restart, the login screen did not have the correct resolution. When logging in, I could see my mouse pointer and my usual desktop wallpaper but no launcher, no menu bar.

Long story short, I removed the nvidia driver via terminal (crtl-alt-F2): sudo apt-get remove nvidia*
restart the computer
Graphic looks ugly but the graphical interface started at login (relive)
Reinstall nvidi via the "additional driver" gui interface
restart the computer
reset the display preferences
Done

So, the problem is solved and I am posting it for further reference, and in hope of helping some poor soul.

Also, if a developer sees this PLEASE, work on those drivers nouveau vs nvidia, not the first time I experience conflic with NVS5100M graphic card :(
