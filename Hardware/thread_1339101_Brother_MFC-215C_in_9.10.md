The MFC-215C uses the MFC-210C driver which is packaged in Synaptic Package Manager Universe repository.

Open Synaptic and search for "Brother".(without the "")

Select and install the "Brother-cups-wrapper-extra" and "Brother-lpr-drivers-extra" which will install the MFC-210C driver.

Connect and power up the printer to the computer and select the MFC-210C driver for the printer as it should now be installed.

For further information,check out the [Brother Solution Centre](http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/index.html)

Good Luck
