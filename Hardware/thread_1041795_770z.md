### Post by cariboo on 2009-01-17
The trident driver should be installed by default when you installed Ubuntu. You could always use the vesa driver to see if you can get better resolution

Edit the device section in /etc/X11/xorg.conf

```
Section "Device"
	Identifier	"Configured Video Device"
        driver           "vesa"  
EndSection
```

to edit /etc/X11/xorg.conf press Alt-F2 and type:

```
gksu gedit /etc/X11/xorg.conf
```

Remember to make a backup copy of the config file before editing it.

Jim

Section "Monitor"

         Option           "DPMS"
         HorizSync      31.5 - 90.0
         VertRefresh   59.0 - 75.0
