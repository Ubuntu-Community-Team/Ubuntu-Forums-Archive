### Post by Mrmental on 2008-08-04
Turn off screenlets by going into System > Preferences > Sessions and un-ticking everything to with screenlets.
I know this sounds like a flippant answer, but I've been using screenlets since installing Hardy. My laptop has been so slow since then that I've been considering buying a new one. Today, one of my screenlets crashed so I de-activated all of them for testing.
The speed increase is phenomenal. Openoffice writer used to take twenty seconds to start. Now it takes three. Everything about the system is now snappy and responsive, even with loads of compiz fusion effects enabled. I've removed all the screenlets from my start-up profile now, and I'm leaving them off.
Screenlets are a lovely little luxury, but this speed increase has to be seen to be believed. It's like having a new computer.
