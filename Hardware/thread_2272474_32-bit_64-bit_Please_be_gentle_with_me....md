### Post by weatherman2 on 2015-04-07
A likely source of confusion: Intel put out a CPU called "Core Duo" in 2006 that was only 32 bit.  But it was soon replaced by "Core 2 Duo" which were (I believe) all 64 bit.  Dell used both types of CPUs in some of the same model laptops that were marketed for a while.  So, some laptops with the same model name may be 32 bit and some may be 64 bit, depending on what CPU was used.

Another complication: you can still install a 32 bit OS on a 64 bit machine.  (But not the other way around.)  You can install a 64 bit OS yourself on a 32 bit machine, if you choose to.

If it's "Core 2 Duo" not "Core Duo" then it should be 64 bit.
