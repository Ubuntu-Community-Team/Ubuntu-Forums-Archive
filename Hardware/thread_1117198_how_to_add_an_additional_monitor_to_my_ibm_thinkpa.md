### Post by Cybie257 on 2009-04-05
What model is the laptop and what OS and version are you using? 

Most laptops have a FN + F5 option to enable the external monitor. You might be able to get it to boot up with the screens being mirrored and then you can disable the laptop screen, or disable it via the FN + F5 option by hitting it an extra time. 

an easy way to test if the external monitor will work is to boot up with the liveCD while the extmonitor is connected and powered on. this way the initial hardware is found without any xorg.conf issues blocking it. You willalso have an idea it is working because the BIOS post should show up when you power the laptop on. 

-Cybie
