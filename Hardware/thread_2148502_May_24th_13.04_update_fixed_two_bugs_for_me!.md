### Post by whyisdaskyblue on 2013-05-25
After updating to 13.04 I noticed a few things broken from 12.10. UPS detection broke - it would see a UPS was there but not read any values and displayed a red "UPS not present". But APC tools would work fine.  Fixed in 5/24 update. UPS (APC and CyberPower) correctly detected and display battery meter.

Second bug was suspend on my Lenovo T61p laptop - suspend would work.. but waking up it would always crash in some ugly way. 12.x to 13.04... 5/24 update fixes that! suspend + awake now work. Time to wake is fast and no ill effects seen.

happy camper here.
