### Post by kaldor on 2011-10-14
Been using Linux since Ubuntu 7.10. I've never had any issues with this until I installed Oneiric last night, so this is completely new and Ubuntu-specific. Fedora 16 beta worked fine (same kernel version, etc).

Multimedia keys are sluggish in their reaction time, and the colours do not change. Normally, if I press the touch-sensitive Mute button, the light on the keyboard turns from blue to red in order to signify that it's muted. On 11.10 everything stays blue.

Also, adjusting volume or backlight brightness isn't working properly. It works but it's sluggish and buggy. For example, if I press the volume down/up button for more than a split second it seems to "stick"; it will go all the way down or all the way up without me pressing anything. It just keeps making the *blip* and goes up each notch until it can't go higher or lower. The same issue exists for adjusting the backlight brightness.

Overall, I think this issue is Ubuntu-specific. The issue happens on GNOME Shell as well. Before running 11.10 I was running Fedora 15 and Fedora 16 Beta. My keys worked just fine around 11.10 Beta 2, and on daily builds as late as last week.
