I have recently bought an acer aspire one d250 and installed the regular ubuntu-desktop on it. I chose not to install UNR, since 9.10 doesn't include a regular desktop mode anymore. The machine runs fine, except for one thing.

The machine originally came with windows 7 starter, which I can still use though dual booting. The problem is that when I am in Windows 7, the wifi reception is very good, I get 90-100%. However, when I remain seated at the same location and reboot to ubuntu, the wifi recpetion drops to 48-60%. It also occasionally drops out, especially when I try to download large files or when I want to open a website with a lot of pictures on it. When the wifi drops out, I also get a crash report. However, I am not able to send that, since my wifi has then dropped out on me.

### Post by Objekt on 2010-02-23
For what it's worth, I also have an Acer Aspire One D250, and I have noticed no difference whatsoever in wireless reception between Windows XP and Ubuntu Netbook Remix 9.10.  The only small problem I've had is that the wireless on/off switch doesn't do anything under UNR.  If I want to turn off wireless, as when on an airplane, I have to do it in software.

You might try resizing one of your partitions to make room for an UNR 9.10 install.  Since you already have a dual-boot setup, it shouldn't break anything to have UNR alongside your regular Ubuntu install.  Let us know if you have the same problems in UNR vs. Windows 7.
