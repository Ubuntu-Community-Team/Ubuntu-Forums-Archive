### Post by aacero on 2010-04-07
I just went through this yesterday.

Download the firmware upgrade ISO file (CD image) from the Intel web site: [http://downloadcenter.intel.com/Detail_Desc.aspx?agr=Y&DwnldID=18363&lang=eng](http://downloadcenter.intel.com/Detail_Desc.aspx?agr=Y&DwnldID=18363&lang=eng), and burn it to a CD. I used brasero's "Burn image" button.  Boot off the CD and you're done.

I didn't have any data on my SSD when I burned it, so I don't know if updating the SSD's firmware will erase it.
