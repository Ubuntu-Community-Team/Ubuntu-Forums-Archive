I need to get printer drivers for a Brother MFC0-5840CN printer for the 64 bit architecture. I found some x86 drivers on their website but these won't work for me as I get the "Wrong Architecture" error. Is there any way that I can install these drivers or that I can convert them into 64 bit?

### Post by kaoskorruption on 2008-08-04
Ahh just kidding I figured it out :P

They were the "brother-cups-wrapper-extra" and "brother-lpr-drivers-extra" packages on Synaptic Package Manager if anybody happens to need them.
