### Post by biodojo on 2012-06-13
When you boot up the netbook press F2 to enter the BIOS. Use the arrow key to go over to the Boot menu use the down arrow to select the network boot option and use F5 or F6 to move it up to the very top. Then press F10 to save the settings and boot. You will see the system try to use network boot, give up and then go to regular boot. This solves the freeze issue.
