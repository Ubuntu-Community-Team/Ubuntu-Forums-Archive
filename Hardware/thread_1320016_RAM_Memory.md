### Post by 73ckn797 on 2009-11-08
> **techm3 said:**
> Hi, I have a computer with two partitions (Windows & Ubuntu) and I have a 6GB RAM, but Ubuntu is only using 2.9Gb. How can I change that to about 5Gb ???

If you are running 32 bit system then that is about all that the system sees. If the mother board is 64 bit then I would install the 64 bot Ubuntu and all RAM will show up. There is no PAE (physical address extension) in Ubuntu 32 bit unless you use the server version or 64 bit. 64 bit will see all memory. I tried installing the server version one time with a Gnome desktop but was not happy with the results.

Search the forums and you will see this has been discussed many times.
