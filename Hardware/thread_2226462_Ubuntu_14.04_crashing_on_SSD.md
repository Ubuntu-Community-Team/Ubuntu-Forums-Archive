### Post by QIII on 2014-05-28
Given that many, many of us run 14.04 on SSDs,  I submit that it is not Ubuntu per se that is the problem.

If you find that everything is fine after you fiddle with the cables, I suspect a hardware issue:  the cable connections.  Next time this happens, don't switch cables.  Instead, disconnect and reconnect the same cable at both ends.  If the problem persists, try a different header on the motherboard.

Let us know what happens and we can move on from there if need be.
