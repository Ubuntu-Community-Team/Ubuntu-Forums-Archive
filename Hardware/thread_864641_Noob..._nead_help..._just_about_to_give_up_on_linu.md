### Post by Dark_Stang on 2008-07-20
For your wireless problem, check out the thread in my sig for the hp dv6636nr laptop. We have the same wireless cards.

For compiz settings manager... Open synaptic package manager. System > Administration > Synaptic. Look for the package compizconfig-settings-manager. That will provide you with a program to tweak compiz.

### Post by Sef on 2008-07-20
> As far as the second part though... all of the compiz packages I have are installed... I see a compizconfig-backend-gconf, but not a compizconfig-settings-manager

Applications > Add/Remove > Show: 'All Available Applications' > Search: ccsm > click on ccsm box > apply changes > apply > close.

Read [Forlong's Blog]("http://forlong.blogage.de/article/2007/8/29/How-to-set-up-Compiz-Fusion") for information on how to set up compiz-fusion.
