### Post by pansz on 2009-02-05
Usually, programs does not have "unique" installation method in Linux world.

You need to find ways for any particular application.

For example: download the Firefox from the official Firefox website and you can place it in any folder in your USB Drive and run it.

There's anther issue you need to consider then: typical X applications will not write anything into the directory where the application reside, it will assume that the only writable directory is your /home/username, so your user data will always be on your /home file system.
