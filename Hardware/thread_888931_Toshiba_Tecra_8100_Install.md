### Post by djom on 2008-09-24
Hello kellogs,

If it can help I have the same laptop (without wifi and with 384 MB RAM) and after some tries and errors finally managed to install Xubuntu (not Ubuntu - too much for such a small config).

For this I 1st had to update BIOS to version 2.50 (otherwise the fan was not starting, the tecra eventually froze during install because of overheat).

Once done I have been able to install Xubuntu 8.04 Alternate CD.

Only the alternate CD worked, the normal desktop was extremely slow and never went to completion of install.

Once installed, Xubuntu is nice & neat and the laptop not too slow, I am happy with it, a second life for this forgotten laptop ;-)

Hope it can save some of your time.
