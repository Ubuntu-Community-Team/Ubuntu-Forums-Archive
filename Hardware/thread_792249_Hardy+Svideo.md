### Post by jocko on 2008-06-09
Not sure that this is needed, but after you applied the changes in nvidia-settings, did you press "Save to X Configuration File" and restart the x server (ctrl+alt+backspace)? To be able to save the configuration you need to start nvidia-settings with "gksudo nvidia-settings".
The only way I can use s-video is if it's properly configured in xorg.conf and the xserver is started with the s-video cable connected.
