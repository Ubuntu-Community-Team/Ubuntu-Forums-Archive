### Post by kpatz on 2009-04-15
I have 4 of them in my Intrepid file server (in a RAID 5 array).  No issues so far, with load cycle or anything else.  In fact, my load_cycle_count is 16, with a power cycle count of 14, so they don't load cycle at all while idle.

They aren't my boot drives though, so I can't relate any issues with booting off them.
