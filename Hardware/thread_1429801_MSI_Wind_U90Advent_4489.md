### Post by chrisj802 on 2010-03-14
There have been a lot of problems reported with Ubuntu 9.10 not recognising the WebCam or the USB ports. 
MSI recommend the installation of the latest BIOS which appears to fix the USB recognition problem. The only pain is that you have to re-install MS Windows to apply the BIOS update. (I had the foresight to get Currys.digital to send me a a recovery CD - makes putting Windows back a doddle)
The webcam and USB ports all appear to work properly on my Advent 4489 after the BIOS update - they did NOT work before so that would appear to be the fix.
