Well after having some help from ManDay on #xorg (freenode,irc)

I fixed the issue :)

For all that come here lost, here's how to FIX it:


first put a bare minimum valid xorg.conf in your system:
[INDENT]sudo gedit /etc/X11/xorg.conf
Paste this into it and save

```
Section "Device"
	Identifier    "Configured Device"
EndSection

Section "Monitor"
	Identifier    "Configured Monitor"
	HorizSync      30-96
	VertRefresh    50-160
EndSection

Section "Screen"
	Identifier    "Single Screen"
	Device        "Configured Device"
	Monitor       "Configured Monitor"
EndSection
```

[/INDENT]


Then do some search on how to insert custom resolution in xorg.conf :)

For example my xorg.conf now looks like this:


```
Section "Device"
	Identifier    "Configured Device"
EndSection

Section "Monitor"
	Identifier    "Configured Monitor"
	HorizSync      30-96
	VertRefresh    50-160
	Modeline 			"1360x768_60.00"   84.75  1360 1432 1568 1776  768 771 781 798 -hsync +vsync
	Option				"PreferredMode" "1360x768_60.00"
EndSection

Section "Screen"
	Identifier    "Single Screen"
	Device        "Configured Device"
	Monitor       "Configured Monitor"

	SubSection    "Display"
		Depth       24
		Modes  	    "1360x768" "1024x768" "800x600" "640x480"
	EndSubSection

	SubSection    "Display"
		Depth       32
		Modes  	    "1360x768" "1024x768" "800x600" "640x480"
	EndSubSection

EndSection

```
