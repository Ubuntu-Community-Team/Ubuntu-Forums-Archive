### Post by Alfons81 on 2009-08-14
First of all (and very important) I would recommend you to run an Ubuntu livecd, for example, and when the system boots go to your Places and mount your hard drive. Find everything you want you have saved under Vista. Save all your stuff on an external hard drive.

Then you have two options as I know:

1.- There is a tool called Supergrubdisk, as I think your problem seems to be with the grub. Grub is the boot menu where you choose wich operating system to boot, and as You explain it seems that Your new Ubuntu studio "overwrite" your well running grub.

[http://www.supergrubdisk.org/](http://www.supergrubdisk.org/)
 
Find this free software on internet, download it and burn it on a cd or usb. In their web page they explain how to use it and I think they have forums where you could ask.
I used these tool on last week on a friend's laptop and it works, also you colud choose different languages and it provides help while you're using it.

2.- The second option is using the rescue cd of Vista:confused:. And I could not help you with that. Anyways there is full information around goggle.

Hope this is the solution for your problem. Have a nice weekend
