### Post by y@w on 2008-08-11
I've been using synergy for some time. Just like x2x it doesn't actually extend the desktop, but will allow you to share a mouse, keyboard, and clipboard across machines. It's also cross-platform so you can use it with Linux, Windows, or OS X. You can use Quicksynergy to configure it via the GUI. 'sudo apt-get install quicksynergy' I believe.

[http://synergy2.sourceforge.net/](http://synergy2.sourceforge.net/)
