### Post by markbuntu on 2008-05-23
More ram definitely. 512 is a bare minimum. You could upgrade the processor but it will still be restricted by the lack of ram. ram will speed things up better as it will reduce the need to use the swap space on the hard drive which is far slower than ram. You will notice a much bigger difference with more ram than a faster processor with so little ram as you have now. ram now, processor later.
