### Post by lev-unr on 2010-01-17
Just fixed this - if anyone is having this same problem fix it in a few easy steps. 

1. Go to to terminal type in alsamixer and turn up the mic stuff. 

2. Go to sound settings under system preferences. 

-> For some reason there are two mics listed on this netbook. By deault it is set under mic 1. Switch it to mic 2 and works like a charm!

easy fix for a very frustrating problem. :D
