As for Ubuntu compatibility, so far I've only run 10.04 from a USB key. I ran System Testing and fiddled around and my results are:
Wireless works, but occasionally disconnects. 
HDMI doesn't work.
Webcam, sound (in/out), card reader all work.
Didn't try gpu switching.

Eventually, I'll try some suggested fixes from [ASUS U/UL Series laptop page]("https://launchpad.net/~asus-ul30").


Oh, and one thing I find odd. It comes with ExpressGate (instant-on limited Linux), but the resolution is set lower than ideal for the screen. I haven't figured that out yet.
