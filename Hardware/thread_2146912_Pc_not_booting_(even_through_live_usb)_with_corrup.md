### Post by 2F4U on 2013-05-20
When connecting the damaged hdd did you try to change the boot order in the BIOS to boot from USB or CD first? Are you sure the hdd was connected properly (cables etc.?
If the hdd is logically damaged, you should be able to recover some if not all data. However, if it is physically damaged, data recovery is most likely much more difficult depending on which parts of the hdd are damaged. As a starting point, here is some information which deals with data recovery:

[https://help.ubuntu.com/community/DataRecovery](https://help.ubuntu.com/community/DataRecovery)
[http://ubuntuforums.org/showthread.php?t=1655771](http://ubuntuforums.org/showthread.php?t=1655771)
[http://ubuntu-rescue-remix.org/](http://ubuntu-rescue-remix.org/)

If you are not familiar with this there is of course always the possibility to give the hdd to a professional data recovery company.
