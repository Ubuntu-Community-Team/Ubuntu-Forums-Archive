### Post by mikewhatever on 2011-07-29
The 4GB flash is probably used as a caching buffer to make the files you use the most more readily available. Most all modern hard drives use this technology, although cache is usually much smaller - 8 - 32 MB. I very much doubt it's supposed to announce to the user that cache is being used.
