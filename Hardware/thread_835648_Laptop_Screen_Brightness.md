### Post by nickdbliss on 2008-06-21
That is the issue of ur graphic driver not supporting the function keys. I had the similar issue with proprietary drivers. With default drivers it would use function keys.
As a work around, try pressing Ctrl+alt+f5 and try changing ur screen brightness and then again press Ctrl+alt+f7 to get back to GUI. Hope that ll work for u.
