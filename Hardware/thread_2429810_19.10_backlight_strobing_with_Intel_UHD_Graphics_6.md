### Post by jeremy31 on 2019-10-27
Edit grub
```
gedit admin:///etc/default/grub
```
Go to the line that has "quiet splash" and make it "quiet splash intel_idle.max_cstate=4"
Then do ```
sudo update-grub
```
Reboot

### Post by greedylizard on 2019-10-30
For future reference, this issue was resolved by changing the "quiet splash" line to "quiet splash i915.enable_psr=0", then running "sudo update-grub" and rebooting.
