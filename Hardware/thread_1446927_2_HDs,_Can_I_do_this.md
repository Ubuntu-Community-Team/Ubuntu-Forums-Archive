### Post by lindsay7 on 2010-04-04
I would buy an external case for a 2.5 inch hard drive and not worry about putting it inside the new computer case.  You will need to verify if it is an older IDE (has pins on the back) or a newer SATA drive before you buy an external case since the cases are specific for one or the other.  I do not know if there is a ribbon cable to mount a 2.5 inch drive to the motherboard but there are converters to connect a 2.5 inch drive to a usb port.

### Post by lindsay7 on 2010-04-04
Windows will see that it is running on a new motherboard and it may or may not start up and run.  If it does start up there will be a lot of work to do to reinstall drivers and maybe some programs and other things that windows will not be happy about. You most likely will have to call in to MS to re-authorize the key...

Bottom line is it may not be worth doing all of this if you do not need to run vista.
