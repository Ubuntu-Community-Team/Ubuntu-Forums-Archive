Some of you may have already voted, Thanks.

For those of you who have no idea what this is, it's a suggestion to Ubuntu developers to start working on a mouse configuration for the scroll up/down. I have to roll my mouse wheel a lot of times in order for me to reach my destination at all pages. I have seen many others suffering through enough of this as well, for Ubuntu NOT to come up with a solution by now.

[[IMG]http://brainstorm.ubuntu.com/idea/27631/image/1/[/IMG]](http://brainstorm.ubuntu.com/idea/27631/)

Click on that image to vote for this solution and to have Ubuntu Developers see what can be done about this, I offer a solution to have configurations to increase/decrease the mouse scroll speed, and if possible they may also add a threshold for the mouse to scroll faster if more rotations are detected from the mouse scroll wheel.

But please vote, Ubuntu needs to stop working on creating new icons and images, and start working on something more imperative that should be relatively fixed in a small time.

Thank you for your votes! :)
