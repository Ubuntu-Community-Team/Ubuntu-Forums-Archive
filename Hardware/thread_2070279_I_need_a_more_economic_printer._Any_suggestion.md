### Post by coldraven on 2012-10-13
Before you buy a printer do a Google for "<printer-model> CISS"
CISS stands for continuous ink system.

I have an Epson R300* and for several years ran it with one similar to these
[http://www.ebay.co.uk/sch/i.html?_trksid=p5197.m570.l1313&_nkw=epson+r300+ciss&_sacat=0&_from=R40](http://www.ebay.co.uk/sch/i.html?_trksid=p5197.m570.l1313&_nkw=epson+r300+ciss&_sacat=0&_from=R40)

When it eventually died after several **litres** of ink I bought refillable cartridges which are still working.
[http://www.ebay.co.uk/sch/i.html?_sacat=0&_from=R40&_nkw=epson+r+300+refillable&_nkwusc=epson+r300+refillabe&_rdc=1](http://www.ebay.co.uk/sch/i.html?_sacat=0&_from=R40&_nkw=epson+r+300+refillable&_nkwusc=epson+r300+refillabe&_rdc=1)

Ink costs me £12 for 600ml, the regular price if I had bought cartridges would be more like £1000 per litre

Now the paper is more expensive than the ink :)

*It's a six colour printer

ABSOLUTELY!!!  Here is a source I use:

[http://www.inkowl.com/](http://www.inkowl.com/)

This is why I like Brother printers; the ones that use LC 61, 65, 71, 72 have no electronics in the cartridge.  The refillable cartridges hold a lot more ink than OEM cartridges because there are no sponges in them.  The only downside I've seen with bulk ink is if the printer sits unused for more than 10 days or so there may be some nozzle plugging.  I created a document with small colored rectangles, each a different color.  If I'm not using the printer regularly, I print this document.  Just enough to 'exercise' the print head.

You *can *refill the HP cartridges but I've had mixed success at best the times I've tried it.  The refillable cartridges - or CISS - seem to work well.  If you search Youtube for "refilling HP ink cartridges" or similar you can find info.
