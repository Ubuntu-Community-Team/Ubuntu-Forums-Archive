### Post by Odrodzona-Sarmacja on 2008-06-16
Try some other program (less graphic heavy preferably) to copy over these pictures.

Uninstall all internet using apps on your ubuntu, which you don't use with synaptic manager plus gnome vpn, if you are not using vpn that is. Check your screensavers and if you've got more then 1, then uninstall overflow.

Disable internet connection when coping pictures over.

Don't run other programs when coping over with screensaver included.

Freezes should not be so imminent any more... I used to have once 2 days without any freezes. Also I noticed here someone who had full 512mb graphic card and was experiencing freezes also much less frequently, then i did.

Good luck!
