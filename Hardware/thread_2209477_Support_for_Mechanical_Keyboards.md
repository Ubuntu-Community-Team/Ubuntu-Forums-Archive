### Post by bcschmerker on 2014-03-05
I had previously used a Unicomp® Customizer™ 104 (PS/2 Version, with its onboard Motorola® MC68HC05P programmed compatible with the IBM® 5170/85xx/95xx firmware and a mini-DIN6 for connection to the host system) on my hot rod LinUX box under Ubuntu® 8.04.x-LTS before the spacebar switch failed - the Model M technology (buckling-spring-actuated sealed membrane switches) is not quite so robust as International Business Machines Corporation's earlier Model F technology (buckling-spring-actuated capacitive sensors); the Cherry® MX™ switches on many competitive flat keyboards (e.g. the Logitech® G710+ USB 2.0 keyboard, which packs the MX™ Browns) split the difference between the IBM® single-curve keyboard generations cited.  The onboard microcontroller is the primary issue for whether a particular keyboard will function under LinUX in general, as Ubuntu® supports both IBM® 5170/85xx/95xx and USB Human Interface Device Class keyboards in all available Kernels.
