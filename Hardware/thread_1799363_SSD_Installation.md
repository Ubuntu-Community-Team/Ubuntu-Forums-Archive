### Post by oedsrm on 2011-07-15
Hi,

I just bought an L502X and have dropped in an SSD.  I'm not dual booting but this info might be useful...

[LIST]
[*]Opt for the Windows 7 install DVD for an extra couple bucks when ordering the laptop
[*]Use the [service manual]("http://support.dell.com/support/edocs/systems/xpsL502X/en/sm/hdd.htm") (on Dell website) to learn how to change hard drive.  Use a credit card if you don't have a 'plastic scribe'... what ever that is.
[*]You can boot from the original hard disk via eSata... which is how I'll keep the bios and firmware up to date (among other things)
[/LIST]
You should be able to do a completely fresh install using the Win7 install disk if you want.

Other tips when ordering the laptop:

[LIST]
[*]I saved 10% by doing a simple coupon search
[*]Read and consider each option available EG: back lit k/b, VGA dongle, touchscreen, setup assistance
[/LIST]
Cheers,
Pete
