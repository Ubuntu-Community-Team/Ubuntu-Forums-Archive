### Post by C.S.Cameron on 2010-01-23
Stick unetbootin and the Ubuntu iso on a thumbdrive.
Start Windows
Plug the thumbdrive into the b2620.
Start Unetbootin.
Choose Diskimage and point to the Ubuntu iso.
Select Type: = Hard Disk and Drive: = C:\
Select OK
When finished restart the computer.
Select UNetbootin at the menu.
Install Ubuntu

Unetbootin installs a virtual "Live CD" on the hard disk, from this you can install Ubuntu.
