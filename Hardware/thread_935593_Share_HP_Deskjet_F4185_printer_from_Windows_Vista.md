My ubuntu 8.04 PC connects to router via ethernet cable.   Router then connects wirelessly to laptop running Windows Vista (home basic).   Laptop has "shared" HP Deskjet F4185 USB printer.   I am trying to print from ubuntu to HP printer.   Have installed SAMBA.

Have installed printer via System/Administration/Printing
Click "New Printer" select Windows Printer via SAMBA.   Browse to locate HP printer on network, ok.   Select printer driver from list, HP, Deskjet F4100 series, ok.   Verify printer is shared, ok.

When I go to print test page, parts inside the printer start to move, but the page does not actually print.   Test page document does appear in print queue accessible from control panel on laptop.   No error messages other than green power light on printer flashes.   Any ideas?   Printer driver as seen in CUPS web based interface is HP DeskJet F4100 Foomatic/hpijs, hpijs 2.8.2

Printer is reliable when used directly from laptop.   Would switch to Linux on laptop too but would have to convince owner/family member first!   Thanks.
