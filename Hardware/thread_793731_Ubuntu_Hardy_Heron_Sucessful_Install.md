### Post by fmartinez on 2008-05-14
So I've heard a lot of mixed opinions of Hardy vs Gutsy. I was hesitant to install, but so far I have installed Hardy Heron on: 
Laptop:
Toshiba Satellite A215-S4747
CPU: AMD Turion 64 bit
Graphics: ATI Raedon X1200
Ethernet: Realteck Semiconductor RTL8101E
Wireless: Atheros AR242
- All functions worked out of the box. The only modification was to the wireless because I had to use the Windows driver with ndiswrapper. After that my wireless has been flawless. 
- Compiz-fusion works out of the box. No difference in CPU consumption. Flawless
- Virtualbox: the only thing I had to do was manually install the kernel modules through synaptic; after that flawless. 

Desktop
HP Pavilion Desktop Elite
CPU: QuadCore 
Graphics: nVidia Geforce 8500 GT
Ethernet: Realtek RTL8111/8168B Gigabit 
Wireless: Intel wireless chip
- All functions worked out of the box. Especially the Ethernet which was not detected under Gutsy! Now it flies.
- Compiz-fusion works out of the box. Flawless
- Virtual box: same as above. 

Summary: 
This new upgrade is one of the best for the hardware that I currently have. I am definately one statified Ubuntian user! As of yet I have had no issues. And to top it off I actually only had to install it 1 time; which is unheard of in the Linux world.
