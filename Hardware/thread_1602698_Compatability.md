### Post by QIII on 2010-10-21
I don't think that you can naively think that all drivers are open source, nor that all applications installed are.

While Canonical is committed to open source software, and makes a great effort to be sure that is what is included in the repos,  third party and closed source software is included in the "partner" repos.  If you want Java to run properly in all instances, you may want to replace OpenJDK with Sun Java, for instance.  Essentially, Canonical does not force you to use open source software, but only includes that by default in the "standard" repos.

Can you expect that the OEM you buy a machine from will only include open source software and drivers?  No.  They will include what they will.  Some jokingly refer to "Dellbuntu", for instance.  They may install proprietary drivers, bloatware, or any number of strange and wondrous things.

I would say, however, that you are most likely to get a true system from System76.  They may include proprietary drivers, but you can always disable them.  As far as I know, System76 sells Ubuntu machines exclusively.
