### Post by KdotJ on 2010-11-07
I know that this is an ongoing issue with Ubuntu and the 'newer' synaptics touchpad that have no physical buttons but allow you to click the pad. The issue people have having is no right click and erratic pointer movement. I run Arch on my netbook, because of this problem. In the Arch User Repo (AUR), they have a user built package named "synaptics-clickpad". After installing this, my clickpad works perfectly... left click, right click, side scroll etc. I know supposedly a "fix" is in the pipelines, but this has been an issue from 10.04. I know that Arch is more "bleeding-edge" than ubuntu is, but I just put 10.10 on my netbook for dual-booting. Mainly because I wish to use Unity. 

How come there hasn't been a port for this yet? Surely Ubuntu, having one of the largest package bases for a Linux distro would have had some sort of package by now? And we can see it's possible, it's already been done.
