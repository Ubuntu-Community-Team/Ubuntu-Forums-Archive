### Post by snowpine on 2009-01-16
Hi Santos,
I recommend using an external CD drive, USB thumb drive, or SD card to test out the different distros you have mentioned. They all use basically the same Ubuntu "core" system, so it's mainly a question of which user interface you prefer. Personally I like eeebuntu the best out of the choices you mentioned, because it comes in 3 different flavors (base, regular, and netbook remix) depending on your preference. I personally am using a distro called Cruncheee on my eee 900ha. You should definitely check out the forums at [http://forum.eeeuser.com/](http://forum.eeeuser.com/) if you haven't already done so, lots of good info there. Good luck!

Eeebuntu NBR sucks, imo. 

It's a very... unpolished, unprofessionaly looking distro. The array kernle it uses has issues with using a webcam as well - and can only use it when it's set to 300x200 or some such.

I think easy peasy is ubuntu eee not to be confused with eeebuntu. I installed it as soon as I got my 904. It has a 'simple' interface or yoou can change to a 'normal' desktop. It needs a little setting up on the desktop but nothing major. It's just like using Hardy. go here for more info.

### Post by snowpine on 2009-01-16
Also keep in mind you can install "regular" Ubuntu, Kubuntu, or Xubuntu on your eee. This custom kernel will allow you to use wireless and other hardware features: [http://array.org/ubuntu/](http://array.org/ubuntu/) In fact it is the same kernel used in Easy Peasy, eeebuntu, Cruncheee, etc...

Prior to switching to Cruncheee, my favorite eee distro was plain old Ubuntu Intrepid Ibex with the eeepc kernel from the link above. (This was however before eeebuntu and easy peasy were released, so I haven't tested those extensively.)
