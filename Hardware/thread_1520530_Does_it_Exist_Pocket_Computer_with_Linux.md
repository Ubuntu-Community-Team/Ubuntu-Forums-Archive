### Post by sgosnell on 2010-06-29
Nokia N800/N810.  They come with Maemo installed, which is a Linux variant optimized for the hardware.  You can put Ubuntu on it, using an SD card, if you want.  There are detailed instructions on the Maemo forums, [http://www.internettablettalk.com/forums/](http://www.internettablettalk.com/forums/),  I do think you'll be happier using the default OS, but that's up to you.
