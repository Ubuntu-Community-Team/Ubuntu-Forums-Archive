### Post by weatherman2 on 2015-06-21
What happened to the old battery?  Is it completely dead?  If it still works at all (even if it barely holds a charge), does it register with Ubuntu when it is discharging?

Any chance you could create a Ubuntu 14.04 live USB that you can boot on this laptop, to see if the battery works any differently there?  Maybe a newer version of Ubuntu would fix any Linux issue regarding the battery. If 14.04 doesn't work any differently, then perhaps there is no Linux fix to your issue.

Perhaps the battery is a third party battery that doesn't communicate properly with the laptop the way a "genuine" Acer battery would?  (I rarely buy "genuine" batteries myself but have had good luck with them; I have an Acer netbook running Ubuntu 12.04 and it sees the generic battery discharging correctly.)

As for battery life: there are a lot of variables there.  There are many different version of an Acer Aspire laptop with different hardware.  Some hardware is better supported in Linux than other hardware.  Sometimes the Linux drivers aren't great for some laptops and will cause more power drain than say under Windows.  My Acer netbook works well, though - I put it on a "watt meter" to see how much power it consumes when plugged in and it's the same as under Windows, ore or less, so battery life should be about the same, too.

Some laptops drain the battery faster than others.  Some "new" batteries have been on the shelf a few years and may not retain a charge as well now as when first manufactured.  Some laptops drain the battery faster than others - and two hours could be the absolute best you would have achieved even when everything was new...but again, it depends on the laptop, the battery, etc.  - as I said, many variables...
