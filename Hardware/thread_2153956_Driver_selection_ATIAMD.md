### Post by Mark Phelps on 2013-06-13
Unless you are planning on doing graphics-intensive Gaming or need to use Catalyst Control Center to control the GPU speed and/or temps, you do not NEED the AMD restricted drivers installed.  You already have Radeon drivers installed, or you would not be seeing a desktop.

I've run across this before -- where all you are being offered is Beta versions of the AMD drivers, and folks who have installed any of these have often run into problems.

Unless you really NEED them, you're better off staying with the default Radeon drivers.
