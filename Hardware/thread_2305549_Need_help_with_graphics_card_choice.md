### Post by Yellow Pasque on 2015-12-25
Crossfire is a bad idea on Linux (unsupported). Crossfire on any OS is a bad idea when your PSU is a base model without any PCI-e connectors and all the power is coming from the slots.

As for the power through the PCI-e slot, it looks like 75W (through either color slot) for a single GPU and 35W in each slot for dual GPU's (which makes sense) on page 22: [http://www8.hp.com/h20195/v2/GetPDF.aspx/c04313388.pdf](http://www8.hp.com/h20195/v2/GetPDF.aspx/c04313388.pdf)

I think you will be very happy with the 750Ti. It's the best card you can get without upgrading the (non-standard) PSU. Happy gaming.
