According to the model's specs, it should have come with Windows 7 pre-installed. If yes, does the touchpad work correctly in Windows 7?

For the OS choice, if your version is an 'LTS' version (Ubuntu 10.04.2 LTS), I'd say it is already pretty good and stable. If the wireless is working in it, then I'd suggest to just stick with it and try disabling your touchpad from BIOS if possible.

Alternatively, you may try Linux Mint or Ubuntu 11.04 if you wish. But be informed that 11.04 is not quite stable yet.
