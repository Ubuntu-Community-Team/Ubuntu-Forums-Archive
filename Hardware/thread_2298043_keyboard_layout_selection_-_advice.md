### Post by roland_burley on 2015-10-08
Hi,

I am using a UK spec/layout CM Storm Quick Fire Pro keyboard w/ Ubuntu 14.04 LTS but I could not select any keyboard layout except US layout. I went into System Settings -> Keyboard - > text entry and selected the correct 'English (UK) (+ a ton of other options to test) but it would not use the correct layout. I solved it by deleting every other layout but English (UK) and now it works correctly. 

There is obviously some sort of bug that is forcing the a default option of English (US) regardless of your own selection. This was driving me nuts, hope it helps others.
