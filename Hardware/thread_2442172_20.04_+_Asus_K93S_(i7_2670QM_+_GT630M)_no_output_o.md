### Post by CelticWarrior on 2020-05-01
Yes, 390 is the right choice and it seems to be installed. But then there's also
```
ii  nvidia-settings                            **440.64**-0ubuntu1                             amd64        Tool for configuring the NVIDIA graphics  driver
```
I don't know where this came from but it's certainly a problem.

You can use additional drivers to change to the open-source driver (temporarily) as this should purge all Nvidia drivers and software installed. Then reboot and re-select the recommended Nvidia drivers in the same tool. This should install the 390 drivers as well as the proper nvidia-settings (Nvidia X Sever Settings) utility and then, hopefully reagain the HDMI output support.

Purging can be alternatively done by running ```
sudo apt-get purge nvidia*
```
