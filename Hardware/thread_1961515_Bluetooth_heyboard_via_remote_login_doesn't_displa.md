Here's an interesting one, to me at least. I have a small bluetooth Keyboard that bought to use with my Android tablet, so that I can login to Ubuntu remotely. 

I use the Android app "Connectbot" to login. When used with the tablets default software keyboard the login and characters seem to work. When the bluetooth keyboard is on and paired. I lose the use of all of the punctuation (shift + 1-0) keys in the Ubuntu tty.

### Post by promet on 2012-04-19
Speaking of hacks...

I've discovered that logging in with the bluetooth keyboard produces this problem, but logging in with the software keyboard and enabling the bluetooth *after* having logged in, regains function of all of the keys that had not been registering prior.

It's ugly, but I will take it. Marking this one solved (as obtuse as that is...).
