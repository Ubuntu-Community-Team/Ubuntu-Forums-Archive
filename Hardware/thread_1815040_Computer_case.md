### Post by zero244 on 2011-07-31
Looks like a decent case and the price is right.
You might want to save up if you want to do a lot of mods or run extra cooling gadgets.
I like to find a case where both sides and the front open up for better access.
Also if a case has a removable motherboard tray, it is much easier to install heat sinks outside the case on a tray.
The call is yours.
