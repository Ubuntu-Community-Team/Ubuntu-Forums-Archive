### Post by newton21989 on 2009-09-03
Are you trying to do OCR? It looks like you're scanning it as an image, in which case you should save it as a *.jpg, *.png, etc... You won't be able to copy and paste text, but you can still read and re-print it. If you are wanting to scan it as text, set Xsane (or whatever you're using) to do OCR.
