### Post by ajgreeny on 2009-12-30
Try adding this line to your /etc/fstab file
```
/dev/scd0       /media/cdrom0   udf,iso9660 user,noauto,exec,utf8 0       0
```
but make sure the mountpoint folder is present with ```
sudo mkdir /media/cdrom0
```and also check in /dev that you have a dev/scd0, as it may be different in your case, in which case you will need to edit the fstab line.
