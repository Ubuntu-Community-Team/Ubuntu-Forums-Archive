### Post by jerrrys on 2011-08-01
yes, this question has been asked a couple of times.  but in short, here's the deal:

10.04 is LTS (long term support) and basically built to be highly stable.

10.10 is based on 10.04 and shares this stability.

11.04 is now the new ubuntu breed, including things like Unity and is still a work in progress.  when 11.10 comes out things should start running smooth again. 

last cycle (releases between LTS versions) it was release 8.10 that was a big headache.

and by the time the next LTS (12.04) comes out; everything should be fixed and solid.

EDIT:  check out googlubuntu for more info
