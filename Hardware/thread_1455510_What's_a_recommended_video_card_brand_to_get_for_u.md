### Post by Linuxforall on 2010-04-16
Do a search and you will see the consensus to be nvidia, stay off ATi for now. For PSU, any good brand 500W is sufficient for a single video card, for SLI it has to be 700W, no less.

### Post by cascade9 on 2010-04-18
A lot of motherboards with AGP slots work fine with PCI cards. 

If you live in a major city, I would consider going to a 2nd hand computer store (or 'computer upgrader') and finding a different card. The FX5200 will work, but for desktop use its pretty much the same as an older GF2/GF4MX/GF4 which would be cheaper.
