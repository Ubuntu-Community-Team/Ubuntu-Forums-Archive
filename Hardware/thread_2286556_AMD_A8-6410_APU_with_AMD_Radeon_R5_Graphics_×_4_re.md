### Post by psfal on 2015-07-13
Just a head's up, I don't know where to report bugs/fixes, but hopefully someone can redirect this where it needs to go so it's a known issue...

I bought a new Lenovo G50-45 laptop with AMD A8-6410 APU with AMD Radeon R5 Graphics x 4 and installed Ubuntu 14.04. The default open-source graphics driver (Gallium 0.4) didn't work with the external display output. I had to use the proprietary AMD fglrx driver to enable it so I could use both displays. The issue is resolved. Hopefully someone can make good use of this info [COLOR=#000000] [/COLOR]
