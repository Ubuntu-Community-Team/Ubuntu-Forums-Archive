### Post by nixscripter on 2009-02-01
Okay, it's a little confusing, because the second card has the same name as the first one.

Make a backup copy of your xorg.conf file, and then try replacing the two Device sections with this:

```

Section "Device"
    Identifier     "Device0"
    Driver         "nvidia"
    VendorName     "NVIDIA Corporation"
[B]    BoardName      "GeForce MX200"
    BusID          "PCI:2:8:0"[/B]
EndSection

Section "Device"
    Identifier     "Device1"
    Driver         "nvidia"
    VendorName     "NVIDIA Corporation"
    BoardName      "GeForce FX 5200"
    BusID          "PCI:1:0:0"
EndSection

```

If it doesn't work, then I'll probably look at the "metamodes" part of the Screen configurations.
