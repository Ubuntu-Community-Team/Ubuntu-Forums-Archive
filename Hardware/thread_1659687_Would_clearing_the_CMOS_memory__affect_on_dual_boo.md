### Post by Fafler on 2011-01-04
Not in any way. The only exception is if you have two physical harddrives with XP on drive one and Ubuntu on drive two. In that case you need to select the second drive as boot drive.

But if you have a normal one-drive setup with multible partitions, it won't care what you do to your CMOS data.

### Post by IcarusR on 2011-01-04
Clearing the cmos will not effect any OS or data on the hard drive. 
All it will do is reset BIOS to default values & you will need to reset things like time & any non default settings within the BIOS.
