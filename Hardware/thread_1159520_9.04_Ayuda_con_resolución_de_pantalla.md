Para revisar permisos podes usar el file manager de tu gusto, Nautilus, Dolphin, etc. Inclusive la consola/terminal tambien te sirve.

Si usas un file manager tenes que pasar el modo lista y configurar que se vean las columnas de dueño (owner), grupo y permisos (no en modo octal asi es mas "human way").

En una consola podes usar el comando ```
ls -al
``` en cada home para ver las columnas que mencione, en ese orden.

Cualquiera de las dos alternativas, tendras tres ternas de permisos (hablando resumidamente y para no entrar en detalles e irnos por las ramas).

La primera terna indica que permisos tiene el archivo para el propietario, la segunda para el grupo y la tercera para "el resto del mundo".
