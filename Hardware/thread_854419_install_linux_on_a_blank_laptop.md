### Post by tamoneya on 2008-07-09
1. Go to computer with OS and burn the Ubuntu CD from the ISO which can be downloaded from Ubuntu.com
2. Place CD in new laptop
3. Enter Laptop BIOS and select "boot from CD"
4. Exit BIOS and Restart
5. At ubuntu Loading screen select "install"

ubuntu is identical for laptop, desktop and for the server it is very close to identical.  The standard OS would work on a server just it would be a little be slower but not much.

As for installing both windows and linux I highly recommend that you install windows first.  Windows is not smart enough to notice that ubuntu is on the laptop and will mess things up.  Ubuntu however knows how to work with windows and will be able to setup what is called a dual boot.
