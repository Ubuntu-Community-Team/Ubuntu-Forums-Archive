### Post by gordintoronto on 2009-12-13
You should not need any software from the CD which came with the camera.  If you attach the camera to the USB cable, then turn the camera on, it should appear on your desktop as if it were a hard drive.  You might also have a window pop up suggesting you use F-Spot to gather pictures from the camera.

The CD contains Windows software, which is useless in Ubuntu.
