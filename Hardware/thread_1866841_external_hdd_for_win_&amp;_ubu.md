(on multi-user systems that might be tricky, but for common desktop user, it's perfectly fine, I have 2 of them myself :P )

### Post by diasf on 2011-10-22
It comes formatted as NTFS. I have 2 WD elements, just plug and use.

And as long as each OS supports NTFS, you will be fine. Or if you don't have windows, then ext3 would be the choice. There are programs that add support to ext3 on windows, but IMO is better to let NTFS if you have windows around. The linux support to ntfs is way better than the windows support for ext3....
