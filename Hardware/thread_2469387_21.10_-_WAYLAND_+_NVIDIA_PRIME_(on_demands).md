Hi,

Since more than a year, I managed to use Nvidia prime on my Laptop (Dell XPS 15 9570) with the "on demands" options that allow you to use discrete NVIDIA graphic cards for a specific app. It's was working fine with ubuntu 20.04, 20.10 & 21.04 that used Xorg as default graphic server.

I've been happily surprised when I upgraded to 21.10 and realized that it works out of the box on WAYLAND ! This was really cool, but a week ago after an update, prime stopped working properly: it's like the NVIDIA graphic cards is no more detected (see pictures attached).
When I log with Xorg, it's working again.

I did some research but I didn't really find any solution for this problem and seems that maybe Wayland is still not suppose to work with prime ? 
I really use this feature for graphic and gaming purpose and I'm also really addicted to gesture with my touchpad (that need Wayland), so is it a way to make prime working back on Wayland/xWayland ?

Also /usr/lib/udev/rules.d/61-gdm.rules file:
```

# disable Wayland on Hi1710 chipsets
ATTR{vendor}=="0x19e5", ATTR{device}=="0x1711", GOTO="gdm_disable_wayland"

# disable Wayland if modesetting is disabled
IMPORT{cmdline}="nomodeset", SUBSYSTEM=="drm", KERNEL=="card[0-9]*", GOTO="gdm_disable_wayland"

# Disable wayland when nvidia modeset is disabled or when drivers are a lower
# version than 470, in any case always prefer Xorg
KERNEL!="nvidia_drm", GOTO="gdm_nvidia_drm_end"
SUBSYSTEM!="module", GOTO="gdm_nvidia_drm_end"
ACTION!="add", GOTO="gdm_nvidia_drm_end"
# disable wayland if nvidia-drm modeset is not enabled
ATTR{parameters/modeset}!="Y", GOTO="gdm_disable_wayland"
# disable wayland for nvidia drivers versions lower than 470
ATTR{version}=="4[0-6][0-9].*|[0-3][0-9][0-9].*|[0-9][0-9].*|[0-9].*", GOTO="gdm_disable_wayland"
#GOTO="gdm_prefer_xorg"
LABEL="gdm_nvidia_drm_end"

GOTO="gdm_end"

LABEL="gdm_prefer_xorg"
RUN+="/usr/libexec/gdm-runtime-config set daemon PreferredDisplayServer xorg"
GOTO="gdm_end"

LABEL="gdm_disable_wayland"
RUN+="/usr/libexec/gdm-runtime-config set daemon WaylandEnable false"
GOTO="gdm_end"

LABEL="gdm_end"
```
