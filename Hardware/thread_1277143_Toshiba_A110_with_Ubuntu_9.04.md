### Post by Thalarse on 2009-09-28
Hey folks. My HP tx1000 died recently - another story I guess - and so I resurrected my wife's old toshiba A110, donating the 2gb of ram from the HP. 

Everything seems to work fine, except that after a while the computer freezes. Firefox, xchat, or whatever program I'm running greys out and the computer becomes unresponsive. It might be firefox causing it, but it has happened with other programs - it's just that firefox is what I run most of the time. :P 

I haven't managed to get temp sensors working, so I don't know if it's temperature related. The laptop does feel quite warm, though. I have the system monitor working up the top, and usually (though not always) the CPU ramps up to 100%, and the system load slowly rises. 

I'm not sure, it's a fresh install but I might try and uninstall the programs I recently put on one by one and see if that helps. 

Any advice is greatly appreciated! Thanks
Thalass.

### Post by Thalarse on 2009-10-07
Solved the problem - the CPU was overheating. Or, at least, after I cleaned and replaced the thermal paste on the cpu and gpu, I haven't had this trouble again. Now if only the CPU was faster, I could play X3 Reunion on this machine. :(
