### Post by MrChezedoodle on 2008-06-26
Putting ubuntu on the eee pc was a piece of cake trust me.  you just install the live cd (either through an external cd drive or a live USB) and install the eee pc scripts found [here ]("http://code.google.com/p/eee-ubuntu-support/") As for performance you can expect it to run smoothly.  I've had no problems with performance despite the size.  

As for video files, they run fine with Compiz Fusion off, but when it is on you may run into a bit of trouble.  

Web browsing was tricky for me at first.  With firefox some webpages are a bit too big even in full screen mode.  However i found [this page ]("http://wiki.eeeuser.com/opera"). This explains how to use Opera and it's "fit to width" feature.  

personally i'd say its worth the investment!  The EEE pc is worth the price and is quite convenient.  I would recommend it any day.  I hope i helped.
