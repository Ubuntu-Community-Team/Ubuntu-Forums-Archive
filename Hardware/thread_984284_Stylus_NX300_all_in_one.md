### Post by x1a4 on 2009-01-03
Epson doesn't provide Linux support.  You can try [this Gutenprint driver]("http://www.openprinting.org/show_printer.cgi?recnum=Epson-Stylus_NX300")

EDIT: ensure you download the DEB and install like so:
```
sudo dpkg -i /path/to/driver.deb
```
