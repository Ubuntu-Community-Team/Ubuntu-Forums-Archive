### Post by cariboo on 2009-01-06
THe first thing I would suggest is to set everything, including your video card to the default settings, there is no need to add anything that may become a problem during installation. THe next thing is to start up in safe graphics mode, to do this at the LiveCD menu screen press F4 and select safe graphics mode, the continue on. This will have no effect on your installation, as the restricted drivers should not be installed until you have done all the upgrades after installation.
