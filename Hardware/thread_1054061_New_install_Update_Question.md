### Post by pritamps on 2009-01-29
The old kernels are stored just in case the new one doesn't work for whatever reason (Wireless cards, graphics cards, sound....), and you need to boot into the old kernel. If everything works fine, you can remove the old kernel. Each kernel you remove will give you about 100 MB.
