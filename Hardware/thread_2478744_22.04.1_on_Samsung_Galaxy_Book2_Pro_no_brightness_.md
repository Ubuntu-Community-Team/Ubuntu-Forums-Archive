### Post by him610 on 2022-09-04
I have a monitor on one of my systems that always opens up on the too-bright scale of brightness. I use *xgamma* to dim it to more acceptable levels.
When you first log on, try this from the command line ....
```

xgamma -gamma 0.8

```
I don't know if will work for you, but it may. You can play around with the numerical factor to brighten (1.+) or dim your screen as shown above.

### Post by mathieu-tarral on 2022-09-04
So, I just found a solution from this [bug report]("https://gitlab.freedesktop.org/drm/intel/-/issues/6466") on Intel's Gitlab.

You need to set a parameter for i915 driver;

Create /etc/modprobe.d/i915.conf:
[FONT=courier new][SIZE=4]options i915 enable_dpcd_backlight=3

[SIZE=2]And regenerate the initramfs:
[/SIZE]sudo update-initramfs -u -k all

[SIZE=2]Enjoy :popcorn:[/SIZE]
[/SIZE][/FONT]
