### Post by pricetech on 2011-02-11
The only way to get a firm answer is to check the manufacturer's website for information on the MoBo in question.

All of the ones I've dealt with so far would work with an add on card which would at least imply that failed video would not necessarily turn the MoBo into a paperweight.

The best way to know for sure is to just try it.

Reset the BIOS to defaults first, which should insure it finds the add on card at post.
