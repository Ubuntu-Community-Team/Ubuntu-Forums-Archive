### Post by him610 on 2016-06-12
If the OP knows the model numbers of the parts, the OP should be able to go to each manufacturer's support webpage to determine, for instance, if the memory he has chosen is compatible with the chosen motherboard. The OP can also search on the internet for any problems related to the chosen parts. Oftentimes it is a combination of different manfacturer's parts that may lead to unforeseen, unpredictable problems. 

A couple of weeks ago on these forums there was someone who complained about Ripjaws memory that caused problems with his/her system - it turned out the memory was incompatible with the motherboard.

Due diligence is on the buyer.
