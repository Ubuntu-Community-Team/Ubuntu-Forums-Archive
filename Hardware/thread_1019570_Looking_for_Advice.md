### Post by iggykoopa on 2008-12-23
It depends on what you are using windows for, if it's for gaming you should dualboot because direct rendering doesn't work in virtualbox(they have opengl rendering they just added but I'm not sure how well it works). If your just using a few random windows apps virtualbox is nice because you don't have to reboot (and you can get it set up how you want then take a snapshot, if you get a virus or the system is hosed for some reason you can just go back to the snapshot).
here is a hp laptop I configured that should be pretty nice for you [http://www.shopping.hp.com/webapp/shopping/load_configuration.do?destination=review&email_id=619493&jumpid=in_r329_emailconfig](http://www.shopping.hp.com/webapp/shopping/load_configuration.do?destination=review&email_id=619493&jumpid=in_r329_emailconfig) it will have 3d rendering but if your looking to use it for gaming you may want to upgrade to one of the nvidia cards.
