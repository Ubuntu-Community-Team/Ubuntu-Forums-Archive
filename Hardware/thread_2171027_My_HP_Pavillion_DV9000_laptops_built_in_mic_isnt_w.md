### Post by David D. on 2013-08-28
On my HP Pavilion dv7 the input volume for the built in microphone was set at 0 by default.  Try opening the sound settings and select the input tab.  Check the input volume.  Move the slider to the right to obtain more input volume.

It would also assist in getting more detailed help if you state the version of Ubuntu you are using.
