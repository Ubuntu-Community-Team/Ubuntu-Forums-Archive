### Post by markbuntu on 2009-01-06
I have recently upgraded to a Biostar TA790GX A2+ which is pretty much the same mobo, ATI3300, ALC888, etc and have had zero problems. Just plugged in the old hard drives and everything worked automagically, everything. I am running Hardy i386, amd64 and Intrepid amd64 and Mandriva 2009 One.

One thing, the bios will only support hybrid crossfire with a HD3450 but no matter as I plan on getting another HD3650 to go with the one I have.

A little advice. When you install Intrepid get all the updates and reboot before getting the restricted driver. If you already have Intrepid installed, remove any restricted driver and switch to the VESA driver before changing the gpu. 

This will save you a lot of pain.

### Post by Blue Beard on 2009-01-06
I haven't found anything in Ubuntu land that will use more than 2 or 3 GB of memory.  8GB will be unused.  

As for the quad core, most applications have a hard time using more than 2.

Get the highest bus speed that is reasonably priced.

The AMD overclock program is awsume.

I would water cool (Quiet) an unlocked dual core with a high performance video card(s) and multiple good/high quality monitors.  Multiple video cards are not well supported in any OS.
