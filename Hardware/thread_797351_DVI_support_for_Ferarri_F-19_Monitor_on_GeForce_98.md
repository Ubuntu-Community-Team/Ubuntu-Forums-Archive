### Post by carljokl on 2008-05-17
My Ubuntu installation is running the display in a mode which my Acer Ferrari F-19 Monitor does not like. Athough I do have a display I get a box in the middle of my screen saying "Input Not Support" (good English). I am familiar with this being displayed when the Graphics card is trying to display in some display mode which my monitor does not like. I see the problem being that there is something not quite right with the DVI mode. I am running at 1280 x 1024 at 75hz which is a mode which I know works fine on the Windows installation. I have NVidia drivers installed and have access to the Linux NVidia control panel but that has not fixed the problem. I know the NVidia driver is working as I have full graphical effects and OpenGL 3D acceleration working great. I have been able to use the computer but it is anoying having the box in the middle of the screen with the monitor's error message blocking my view.

I know I could just use VGA and this might well fix the problem (though the GeForce 9800 I use has 2 DVI connectors so I would have to use an adapter to make it work). The problem I have is that I use the VGA connection from the monitor to connect to a second PC which I use for something else. It would be a real hastle to have to keep powering down and swapping cables around just to use Ubuntu properly if there might be a way of getting DVI working properly. 

I used to use an ATI Radeon X550 card on DVI in the same computer before I replaced it with the GeForce one for better performance and Linux compatibility. I have noticed the difference between the two is that the Radeon worked fine while booting up using the DVI connection. The GeForce on the other hand causes the "Input Not Support" message on the screen during boot up and only goes away once Windows starts up (with the NVidia drivers). This makes me think that by default the NVidia card uses a DVI mode which my screen does not like.

I am sure many people have used NVidia Cards but perhaps not so many have used a GeForce 9800 with a Ferrari F-19 TFT screen using a DVI connector. Could anyone suggest a resolution to this problem or do I just have to live with it?
