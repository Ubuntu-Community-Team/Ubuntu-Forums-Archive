I've recently updated to a Dell XPS 9500 (Core I7) with NVidia Drivers (GeForce GTX 1650 Ti).  I'm running latest Ubuntu with latest NVidia drivers set to "NVIDIA On Demand" mode dual-booted with Windows 10.  I've noticed that sometimes the brightness kind of pulses darker and lighter.  Are there controls around this?  Is this some kind of bug?  It is not terrible, but it can be distracting.  Is there any way to fix this?

### Post by csete on 2021-04-23
For anyone that might stumble on to this "issue", it turns out to be related to the "Automatic Brightness" setting and the fact that this new laptop has a brightness sensor.  I had looked for a setting like this in Display, but did not know that it was hiding on the Power settings.
