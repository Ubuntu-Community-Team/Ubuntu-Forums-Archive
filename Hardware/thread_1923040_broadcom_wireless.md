### Post by ahallubuntu on 2012-02-09
Which version of Ubuntu?

What have you tried so far?  Did you go to "Additional Drivers" and try to have Ubuntu install a proprietary driver for the wireless card?  Does it detect your card there at all?

If not, which Broadcom card is it?  To find out, you can open a terminal and type

sudo lshw

to find out more information about the exact Broadcom card you have.  There are often different instructions for different cards and different versions of Ubuntu.
