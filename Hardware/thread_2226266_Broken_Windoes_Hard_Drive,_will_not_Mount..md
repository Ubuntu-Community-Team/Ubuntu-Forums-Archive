### Post by Mark Phelps on 2014-05-26
> the hard drive brokeNot a very useful description of the problem -- "broke" could mean a number of very different things.

If the drive was in use at the time, a likely problem is that it suffered a "head crash" -- a serious hardware failure, and if that is true, there is NO OS that is going to be able to mount it, period.  It would have to be sent to a recovery Lab, be torn down and rebuilt, and even then, only SOME of the data could be retrieved.  And ... services like this are  VERY expensive -- read that as thousands of US dollars.
