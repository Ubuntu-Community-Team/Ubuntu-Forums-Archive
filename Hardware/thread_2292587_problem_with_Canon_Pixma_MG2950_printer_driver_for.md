### Post by Bucky Ball on 2015-08-29
Welcome. Go [HERE]("https://www.canon.co.uk/support/consumer_products/products/fax__multifunctionals/inkjet/pixma_mg_series/pixma_mg2950.aspx?type=drivers&language=&os=LINUX") and download and install the driver. Delete the printer you've added and add it again. This time the driver should be available when you are asked to choose one.

Choose the (debian) packages from the link, not the rpm ones. You will need the printer driver and probably the scanner driver. Double clicking the .deb files once downloaded should open Software Centre which will install for you. Look for these two:

ScanGear MP Ver. 3.00 for Linux (debian Packagearchive) 
IJ Printer Driver Ver. 5.00 for Linux (debian Packagearchive)
