### Post by Mark Phelps on 2011-12-26
You're NOT going to find a Linux-oriented manual for the Biostar motherboard because the manuals are written for the most prominent community - MS Windows.

As to drivers, the Ubuntu installer automatically does hardware detection and installs the proper drivers.  It's not like Windows where you use a driver CD or download drivers from a website.

Best thing you can do is boot your system from an Ubuntu CD, select the Try Ubuntu option, and see how well it detects and handles the hardware.
