### Post by az on 2008-10-18
It depends on your motherboard, not on the type of ram you are installing.  Some newer motherboards require dual-channel ram modules, which means that both modules need to be identical.  This may not be the case for your motherboard.

Try and see.  If you already have the ram, put it in.  The worst thing that will happen is that the computer won't boot while you have the new ram in.  If this is the case, take it out and reboot.

If both modules don't work together, you can try the single 1Gig module.  If it boots properly, make sure you run memtest (hit escape to see the grub menu if it is not shown by default and then pick memtest.)  Let it run a few tests to make sure that the ram works fine.  I have had computers that detect the new ram in the Bios, but the ram is not usable.  The computer boots and crashes in that case.

Make sure the power cord is disconnected when you insert new components into your computer.

It shouldn't be very complicated.
