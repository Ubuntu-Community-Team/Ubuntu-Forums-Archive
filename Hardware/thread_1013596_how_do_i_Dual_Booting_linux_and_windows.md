### Post by MonkeyPaw on 2008-12-17
Install windows first, then boot via the Ubuntu live CD (the install CD) and see if the distro will automatically load on your hardware. If all is well, run the "install" app found on the desktop. In the process, Ubuntu will let you decide how much of your hard drive it can use, and it will install from there and automatically configure the bootloader. When you reboot, you will be given a time period (5-30 seconds) to decide which OS you wish to boot. It's really quite painless if your hardware is all supported.
