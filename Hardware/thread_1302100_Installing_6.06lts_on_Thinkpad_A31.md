### Post by noelvh on 2009-10-27
The one this I would suggest is you pick up a bit more ram for that computer. 512 will work, but with the newer OS 1gb would be much better.

With my Dad's I T30 I found that 9.04 would not support his wifi card, and I tried all the work arounds to get it working but was unable to. That is why I suggest 8.04 or 8.10. I would download and run the live CD to test the hardware support.

Now to the BSOD you are getting in windows!!! You should not have these issues with Ubuntu!!!!! I can count on one hand the number of time I got a full system crash using Ubuntu, and I have been using it for 3 years now. I have had apps crash, but the way Linux works when an app crashes it dose not take the hole system down, only that one app. Now if the BSOD are hardware related you will see some issues. The great thing about Ubuntu amd other distros of Linux is the live CDs. You can run the OS right from the CD with out installing any thing to the hard drive.

Noel.
