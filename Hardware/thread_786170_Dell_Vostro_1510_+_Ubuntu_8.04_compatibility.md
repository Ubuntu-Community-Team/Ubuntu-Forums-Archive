### Post by andrewshalin on 2008-07-28
Yes, Ubuntu if quite compatible with Vostro 1510. One thing you'll have to do is use ndiswrapper with windows xp drivers for your broadcom wireless card. Other than that, everything works out of the box.

### Post by karl_eller on 2008-07-28
I actually haven't had any trouble with the wireless (I'm using the standard Dell A/B/G wireless card), although I did have some some trouble with the wired card (dual booting Vista + Ubuntu, and Vista kept turning the card off), but that was fixed with a driver update.  One thing that has been annoying me in Ubuntu, however, is the touchpad. I like the sensitivity (as in movement rate) up really high, without having a very sensitive touch-pad click, and in Ubuntu it doesn't seem to want to be set up as nicely.  I've also been having some issues with connecting to the Uni's wireless, since Ubuntu doesn't want to detect the WEP2 Business (or is it Enterprise) security encryption that the network uses. But that's more of a software problem than anything else.  Eller
