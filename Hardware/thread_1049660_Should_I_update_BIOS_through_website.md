It would probably be best to update through the BIOS flash utility. I recently updated mine. Initially did it through Windows but that created some issues. I then went through the BIOS utility. You may run into a problem of the utility wanting a floppy drive with the update on it. Some will allow you to use another removable device source.

Download the update from the manufacturer. Copy the file to a floppy (if you have one installed) or to a USB flash stick or possibly burn it to a CD. Follow instructions in the BIOS flash utility.
