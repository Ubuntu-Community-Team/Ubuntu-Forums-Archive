This behavior seems to change randomly, as I do not change any hardware setting nor connection from boot to boot when the behavior switches.  The problematic variation is when I have my internal sata drive installed at boot time: it randomly decides to either boot from the cdrom, or I get a blank screen with a flashing cursor in the upper left corner.  Unfortunately, the hotplugged sata drive is not recognized (the system does not respond at all), so it must be connected at boot time in order to use.

I have tried all six boot sequences available in the bios (Esc, then F1 into the bios menu), explicitly choosing the boot device at boot time (F12), and enabling and disabling various other features.  Does anyone have a suggestion for something that I might have overlooked.  I want to get rid of the "Press any key ..." prompt so that it boots automatically.  I'm not sure what this means, and it's probably just my imagination, but I think that sometimes I can get back to the automatic boot if I eject the cd, pop it off the spindle, and then reinsert it.  I'm looking for a more elegant/reliable solution, though.
