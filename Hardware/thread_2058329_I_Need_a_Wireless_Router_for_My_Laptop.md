### Post by cjhabs on 2012-09-15
I am using a Linksys router which I configured via the web interface -you can do this for most routers - just check what the default IP, user and password are from the docs - set your PC IP to something on that IP range and connect the pc and router via a wire - login via the web browser and configure it as desired - no need for the install cd

### Post by digitalzy on 2012-09-15
I recommend a refurbished LINKSYS E3000. It supports 2.4G and 5G dual-band simultaneously as well as Gigabit wired network. I don't think you need to install the software shipped by manufactures. You may simply go through web interface with default IP address, eg. 192.168.1.1, and configure it.
