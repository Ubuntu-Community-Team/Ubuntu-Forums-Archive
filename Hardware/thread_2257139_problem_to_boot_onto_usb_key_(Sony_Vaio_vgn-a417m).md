### Post by oldfred on 2014-12-17
Have you used flash drive to boot before on this Sony? Many old XP systems were built before booting from flash drives was an option. Only some just before Vista, or those XP released after Vista typically have the BIOS updates to boot from a flash drive.

If DVD tray is locked, you should have a tiny hole that you can use a straightened paperclip to release lock and open CD tray.
