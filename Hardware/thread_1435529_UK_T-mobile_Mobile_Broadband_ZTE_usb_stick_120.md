### Post by little_penguin on 2010-04-01
I have just got this T-mobile 120 mobile broadband dongle working. I had to change the username and password the Network Manager wizard originally created for the T-mobile UK entry. Here are my settings:

Number                *99#
Username             user    (instead of User)
Password              pass    (instead of mms)
APN                     general.t-mobile.uk

Hope that helps,

Regards LP
