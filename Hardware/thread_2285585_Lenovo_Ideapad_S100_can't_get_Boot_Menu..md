### Post by kagashe on 2015-07-07
I am trying to boot from usb on my friend's Lenovo Ideapad S100 which has Windows 7. On boot the key to get into BIOS settings is F2 which was working. I could see the usb on Boot Tab but there is no provision to move the entry up and bring it on top.

On boot the F12 key is supposed to bring up the Boot Menu but it did not bring up anything. There is one more problem with F12 key. Actually there is no separate F12 key, F12 is Fn+F11.

After reading about Supervisor Password in the Hardware Maintenance Manual I thought that I need to set up the Supervisor Password to alter the Boot Menu entries but I could not do this even after setting it but it brought one more problem.

Now pressing F2 does not ask for Supervisor Password for getting into BIOS but pressing Fn+F2 does. I don't know how it got altered just by setting Supervisor password.

### Post by kagashe on 2015-07-07
In the BIOS setting Boot tab there is a provision to disable booting from hard disk by putting a "x" by pressing "x" key. I disabled booting from hard disk and could boot from usb. This machine does not have UEFI. I also disabled Supervisor password by changing it to blank password. The F2 key is also working easily by keeping it pressed while pressing the power button.

This thread is sort of solved although pressing F12 does not bring any Boot Menu.
