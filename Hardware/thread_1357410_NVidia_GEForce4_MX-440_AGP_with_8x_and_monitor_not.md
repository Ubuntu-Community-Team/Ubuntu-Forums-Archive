### Post by moddie666 on 2009-12-19
Hi.

Firstly:
That card is ancient, replace it while there are still a few rather cheap AGP cards available.

Otherwise:
Did you try this? Replacing "nvidia" with "nv" because this card is most likely not supported in the new driver.

```

Section "Device"
    Identifier     "Device0"
    Driver         "nv"
    VendorName     "NVIDIA Corporation"

```

That should be the old nvidia driver, if it is still there. Also try to find out what other generic drivers you might use.

Alternatively

When the low-graphics mode error pops up you should have the option to change some settings, before loging in.
On that note, I think you get this error because your card is not supported by the "nvidia" driver.

Just play around and hope for the best.

greetings
/moddie
