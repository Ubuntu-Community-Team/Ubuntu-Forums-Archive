### Post by TAZ46 on 2013-07-02
I just purchased a new 3 tb RED WD30EFRX, formatted it EXT4.  I installed it in a Vantec HX4 and am using a USB cable.  When mounted the hdd activity lamp flashes every second; when unmounted the acticity LED is solid like the 3 other drives (2 NTSF and 1 EXT4). The 3 other drive are fine mounted or unmounted.  I'm afraid this activity may ruin my new drive.  I would appreciate any help troubleshooting this problem.  I have found some similar threads that are about 3 years old but have been unable to find any solution.

I am running Ubuntu 12.04 64 bit.

Thanks.

Update:  Well since the activity LED was flashing, I figured I would give it (the drive) something to do so I copied 1.2 tb of data to overnight.  Now it seems ok, the LED is no longer flashing.  It is on steady like the other drives in the Vantec HX4 and only flickers when I read or write to it.  I have no idea why it was acting the way it did.
If nothing else, maybe someone else with the same problem could try copying data to see if they get the same result.  I see there were several hundred views of my post so maybe my update comments can help someone else.  Anyway, if others are looking at the Vantec HX4 4 drive bay enclosure...it works with Ubuntu 12.04
