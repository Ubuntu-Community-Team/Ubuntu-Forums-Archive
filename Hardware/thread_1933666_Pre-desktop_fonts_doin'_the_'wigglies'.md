### Post by winh8r on 2012-02-29
I would say that the initial splash is good because it is a motherboard/bios generated image, but then the GRUB screen goes wobbbly because the correct graphics card drivers are not fully loaded in the pre boot environment which then resolves itself after boot due to the correct drivers being fired up by the OS.

As everything is working once it boots, I would not worry about it too much.

Start worrying when you start wiggling and there is no display at all!!!
