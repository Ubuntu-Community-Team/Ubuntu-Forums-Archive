### Post by sudodus on 2022-08-12
I think the crucial hardware item is the graphics chip/card. If it is a very new nvidia chip, there is a risk, that there might be no working linux kernel driver for it. If the graphics chip/card was released to the market around one year ago, Linux has probably caught up and there is a working driver for it. Also brand new graphics chips/cards of other brands may have problems with Linux.

So please tell us about the graphics: brand name and model in those two computers. (Often there are several alternative graphics to choose when configuring and buying a computer.)
