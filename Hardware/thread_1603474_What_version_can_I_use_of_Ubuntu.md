### Post by Fafler on 2010-10-23
Besides getting a kernel that actually boots on your system, you should also get something that uses a lighter user interface.

Try this:
[http://cdimage.debian.org/debian-cd/5.0.6/i386/iso-cd/debian-506-i386-xfce+lxde-CD-1.iso](http://cdimage.debian.org/debian-cd/5.0.6/i386/iso-cd/debian-506-i386-xfce+lxde-CD-1.iso)
Or if you really want Ubuntu and not Debian, then try an older version of Lubuntu or Xubuntu like 8.04.
