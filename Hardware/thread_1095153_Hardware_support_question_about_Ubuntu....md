### Post by pytheas22 on 2009-03-13
Ubuntu definitely supports RAID.  You can set it up during installation if you use the [alternative CD]("http://www.ubuntu.com/getubuntu/downloadmirrors#alternate"); it's also possible, although more difficult, to convert an existing system to use RAID.  I've only set up an Ubuntu system with RAID once, so I can't offer much insight, but you could check out the [wiki page]("https://wiki.ubuntu.com/Raid"), or do a Google search for something like 'ubuntu raid'.
