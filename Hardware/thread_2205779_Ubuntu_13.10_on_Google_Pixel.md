Greetings! I purchased a Google Pixel and after about 47 seconds came to the conclusion ChromeOS isn't for me. To be fair I suspected as much before I ever ordered the Pixel but the hardware reviews were THAT good that I ordered one anyway. To be honest I'm not disappointed. The hardware really is THAT nice. There is one problem that is keeping me from truly enjoying my Pixel.

Evidently there is a conflict between the Linux kernel and the TPM chip inside the laptop that causes it to reboot after every second standby cycle. Linus himself experienced this problem and apparently a patch/update/change has been made to fix this but despite running the latest kernel on 13.10 trying 14.04 and Fedora 20 which had 3.12 I STILL have this TPM problem and I'm beginning to pull my hair out.  I've posted on the Google Pixel's G+ page but so far no help. If anyone has any advice I'd be forever grateful.
