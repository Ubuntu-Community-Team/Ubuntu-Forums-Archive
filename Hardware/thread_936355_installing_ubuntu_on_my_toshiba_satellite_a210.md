### Post by snowpine on 2008-10-02
I would recommend, Ubuntu, of course! :)
My suggestion is to burn the Live CD and test it out for a while before deciding to install to your hard drive. That way, you can try Ubuntu with no changes to your computer (unless you click the Install button, or run the Disk Partition editor, or something drastic like that). If you run into any problems while you're using the Live CD, post the details here, and we'll try our best to help! As far as the laptop vs. desktop issue, Ubuntu is smart enough to auto-detect your hardware, so there aren't really any extra steps you need to take because you're using a laptop. 

Here is a good place to start for information: [https://help.ubuntu.com/](https://help.ubuntu.com/)
And of course, these forums. Good luck!
