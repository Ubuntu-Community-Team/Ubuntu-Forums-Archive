### Post by yahooshua on 2009-04-04
Had the same problem with 2 Ubuntu pre-installed Dell laptops.

You need to right click the sound control. By default usually in the top right corner.

Select 'Open Volume Control'
Select 'edit'
select 'preferences'

make sure 'capture' is selected.

If that doesn't work try selecting some others. I ended up selecting them all. I don't think that is necessary.

By the way, I tried a fresh install to fix this, only to find out the touch pad mouse does not work with a fresh install.
