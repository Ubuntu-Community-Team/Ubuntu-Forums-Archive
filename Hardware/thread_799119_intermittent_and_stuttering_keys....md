### Post by bodhidharma on 2008-05-18
Has anyone heard of a problem like this?

When typing on my Lenovo Thinkpad R61e (running Hardy Heron, although
I think this was also a problem before upgrade!), sometimes I hit a
key and the symbol does not appear.  Not very often, but often enough
that I am fairly sure this is something real and not just my imagination
as I type quickly (which I do) and perhaps do not hit the key completely
enough.

Worse, though: sometimes (maybe once every hundred keystrokes) when I
hit a key it starts repeating, it will repeat hundreds of times if I
don't quickly hit that key again (or another).  It is getting to be
often enough that I find it annoying ... I type well enough that I
don't always look at what I'm typing for a while, and then there can
be huge blocks of some repeated character.

Note both of these problems will easily happen with any key I hit:
letters, numbers, symbols, arrows, enter, etc.

(This key skipping has happened several times already in this
message, the key repeating, twice.)


Anyone have any idea what might cause this?  Is it an intermittent
hardware glitch (something in the key-debouncing circuit, maybe?),
or somehow even a software thing?  Has anyone had this problem, too?
Does it get worse (the laptop I'm speaking of is about 8 months old,
but I usually get 3 or more years out of a laptop before it dies)?
Is it the kind of thing someone might be able to fix -- at a price
which is better than replacing the laptop (maybe just replacing one
chip associated with the keyboard)?

Thanks in advance?

### Post by teaker1s on 2008-05-18
similar happened to me I got the error activating xkbd error and I then found out that this could be cured by creating new keyboard layout scheme in System>Preferences>keyboard I did this and then my symbols took two presses and an annoying system beep.

I would suggest you try specifying a new keyboard layout and deleting the existing one, if need be several times as I believe it's a bug.
