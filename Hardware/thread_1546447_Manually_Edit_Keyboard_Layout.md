### Post by dv3500ea on 2010-08-05
You can edit keyboard layouts in System -> Preferences -> Keyboard.

Keyboard shortcuts can be modified in System -> Preferences -> Keyboard Shortcuts.

To remap certain keys, you can use xmodmap:
```
man xmodmap
``` for usage.

There is a graphical program for this which you can install from the Ubuntu Software Centre called [xkeycaps]("apt:xkeycaps")
