### Post by shag00 on 2014-06-23
I just built a new system running 14.04, Intel I5, 8GB RAM, LG DVD drive, Logitech keyboard and mouse, Gigabyte MB Z87X UD5H, on-board video(MB). I run 1 x SSD and 9 x HDD so unless you need a lot of storage you can down spec the MB. Also SSD is unnecessary for Ubuntu I think as it's so much faster than Windows. Used as a entertainment center/server but also runs some office apps as a general purpose PC. Works a treat and pretty cheap to boot. Easily handles playing 1080p movies and built-in office app so unless you are gaming a suggest a video card is unnecessary..
