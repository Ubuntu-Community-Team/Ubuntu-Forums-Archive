### Post by mivo on 2009-11-09
There are some guides around, but I would use 12 GB for / and the rest for /home. If you plan on installing really, really, really many programs, you can up this to 15 GB, but Linux software is not like Windows software, so even 12 GB is probably too much for most people  (but leaves room for expansion). For swap, about the same size as the laptop's RAM, plus 10-15%, if you want to hibernate.
