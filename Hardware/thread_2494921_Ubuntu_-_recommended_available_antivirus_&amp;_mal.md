### Post by TheFu on 2024-01-31
The only reason to run AV on Linux is if you are legally required to do so.  For example, my professional *Errors and Omissions* insurance mandated that I run "industry standard", current, AV software on fully patched OSes using supported versions of all software. This insurance was required by a client.   When I fired that client, I didn't need the insurance any more.   

On Linux, the default AV software is ClamAV.  You can pay others for theirs, if you like, but it won't be any better.  AV on Unix is next to useless.  It isn't like you are using root or running with elevated privileges all the time. If you are, don't. ;)

Go read the sticky posts at the top of the "Security" sub-forum here.  Do the _Basic Ubuntu Security_ things in those posts and in the links they have.
