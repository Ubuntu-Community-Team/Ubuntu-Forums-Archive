### Post by jerrrys on 2012-10-22
It is my understanding that Classic is lighter on resources than Unity.  Its simple enough to add to your current install just to see if it would give you a performance improvement.

There are other options.  The Lubuntu desktop or LXDE.

sudo apt-get install lxde

Or the Xubuntu desktop called xfce.

sudo apt-get install xfce

These desktops can all be installed on your current ubuntu installation and like Classic, you can choose which one at login.

If you decide on lxde or xfce you can of course install the complete Xubuntu or Lubuntu system.[ ]("http://lubuntu.net/blog/lubuntu-1204-now-available")
