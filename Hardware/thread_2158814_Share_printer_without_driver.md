### Post by dandnsmith on 2013-07-01
It would be possible to share the printer and scanner, certainly, but where you come adrift is that Windows doesn't like to use printers without having drivers for them.
In addition, remote scanning nearly always gives problems, however you try it!
BTW, I wouldn't worry too much about support for XP being dropped - the only things you are really able to get now are updates for msoft software which no self-respecting user should be using anyway (firewall, browser, email tool)
