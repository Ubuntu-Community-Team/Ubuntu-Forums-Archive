### Post by guiverc on 2022-03-27
You've not provided release details, so I'll assume the *latest stable* release manual for my links (*adjust as necessary; the word "stable*" *in URL is adjusted to match release; use "lts" if using the 20.04 or current LTS release of Lubuntu)*

See [https://manual.lubuntu.me/stable/F/keyboard_shortcuts.html](https://manual.lubuntu.me/stable/F/keyboard_shortcuts.html)

I'm not sure what shortcuts you mean though; as F1-F12 are not programmed by Lubuntu, being available for use by applications; thus it's on a per-application basis they are changed.

If you're talking about Ctrl+Fn, or Meta+Fn, the prior Lubuntu Manual will send you to the following page [https://manual.lubuntu.me/stable/3/3.2/3.2.14/shortcut_keys.html](https://manual.lubuntu.me/stable/3/3.2/3.2.14/shortcut_keys.html) where they are easily seen (*in 'Global Actions Manager'*) and when you click MODIFY you'll see the ENABLED checkbox.

(Again no release details were provided - so adjust if not using Lubuntu 21.10).

I've assumed Lubuntu as that's where this question was placed on UbuntuForums, though note `qterminal` doesn't offer the menu options provided in 1fallen's answer, but if using Lubuntu I'd just follow what is covered in the Lubuntu manual.
