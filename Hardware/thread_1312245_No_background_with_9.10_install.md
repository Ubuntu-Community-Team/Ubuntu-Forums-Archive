### Post by JBAlaska on 2009-11-04
Before you give up on compiz completely, Try installing the compiz config settings manager:
```
sudo aptitude install compizconfig-settings-manager
```
And enable the "Wallpaper" add-on in the utilities section, See if loading your wallpaper through that will draw it.
