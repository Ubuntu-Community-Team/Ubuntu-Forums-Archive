### Post by MonkeyPaw on 2008-12-10
Well, if you've already managed to make a bootable XP USB drive, then you should just boot to the USB drive and go through XP's standard install process. Make sure you do a clean install (delete all partitions and then let XP do its thing). Once complete, you can then install Linux and let lilo do the bootloader.

Just out of curiousity, are you planning to install one of the OS's to an SD card? I ask because XP is going to consume a large portion of the SSD, leaving little room for anything else. If you're looking to trim down XP's size, use nLite to make a custom install disc.
