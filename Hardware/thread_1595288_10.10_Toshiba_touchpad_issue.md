### Post by IcarusR on 2010-10-13
Not sure about 10.10 but on my 10.04 install doing this in a terminal as user disables touchpad..

```
synclient touchpadoff=1
```

&

```
synclient touchpadoff=0
```

enables it again.

Many synaptics setings can be set with synclient.

To see a list of current settings

```
synclient -l
```

See manpage for synaptics & synclient
