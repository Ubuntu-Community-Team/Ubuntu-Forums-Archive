### Post by Nick Payne on 2013-09-12
On 13.04 amd64 I have a Logitech bluetooth mouse talking to a Belkin USB bluetooth receiver plugged into my desktop PC. Using Blueman I can setup the mouse and it works, but then after each reboot, the mouse no longer works until I go back into Blueman, remove the mouse from the known devices list, search for and add it again, then run the setup assistant and choose "Proceed without pairing", after which it is functional again.

Any suggestions on how to avoid having to re-configure and setup the mouse after each reboot?

### Post by Nick Payne on 2013-09-22
After further playing around, what I find is that if I have a second (non-bluetooth) mouse connected, and use that mouse to momentarily move the mouse pointer after login, then the bluetooth mouse then functions. But with the bluetooth mouse on its own, it doesn't work after login unless I go into Blueman and set it up again...
