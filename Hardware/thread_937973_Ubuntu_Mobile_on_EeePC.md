### Post by michaelkahl on 2008-10-06
Here are the resources I used.

[http://www.ubuntu-eee.com/](http://www.ubuntu-eee.com/)

The above site has an iso image for you to download as well as a wiki attached with some documentation, but read carefully and make sure it isn't mark out-dated.  With this distribution I've had great success, all the tweaks and repos are setup as you need them.  I ditched the netbook remix UI in favor of XFCE4.  This distribution should have everything working out of the box.

[http://wiki.archlinux.org/index.php/Installing_Arch_Linux_on_the_Asus_EEE_PC#Avoiding_Pitfalls](http://wiki.archlinux.org/index.php/Installing_Arch_Linux_on_the_Asus_EEE_PC#Avoiding_Pitfalls)

I know the above is for arch linux, but I read the pitfalls section and applied that information.  More detailed information on avoiding the pitfalls is listed further into the documentation but just use the find option in firefox to skip ahead.

Finally
[http://www.eeeuser.com/](http://www.eeeuser.com/)
Great community support there as well as here.
