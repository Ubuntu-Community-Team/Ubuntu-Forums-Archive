### Post by 00b00nt00 on 2008-10-25
Hmm, SIMMS haven't been used in quite a while. Do you have a 486 system? Pentiums can take 144 pin DIMMS, I'm quite sure of that.

Anyway, to answer your question Single Inline Memory Modules (SIMMS) and Dual Inline Memory Modules (DIMMS) generally have a different pin count, so no, you won't be able to use a DIMM in a SIMM socket.
