### Post by tiom33 on 2012-06-19
I finished the installation of 12.04 LTS on my laptop 'Lenovo E320',
and it works well. But note the things below:



**1. Wi-fi on-off button doesn't work.**

                     it's already noted on the Canonical official website. 



[B]2. Installation of the Catalyst driver is not easy.
[/B] 
                    As like the other machines with ATI graphic cards, especially the one with
                   'hybrid graphics', installation of the Catalyst driver is very subtle problem.
                     I tried to install ATI driver by myself.
                     But things got worser and worser...I finally reinstalled Ubuntu.

It is recommended to refer to this thread written by Alexislavie, which is very helpful :
                
                **     [[COLOR=#008C00][B][all variants]**[/COLOR] AMD/Intel Hybrid Graphics works !]("http://ubuntuforums.org/showthread.php?t=1930450")[/B]


**3.**  Always-on USB port doesn't work -** _It _****_Does Work Properly._**

      E320 is equipped with Always-on USB port, but it doesn't work.
      I think it can be fixed..but I don't know how to fix it right now. 
    
---->> It's fixed by change settings in BIOS.




[B]4. For Koreans
[/B]      
     Local server is not good for download the KR language pack.
     Plus, "ftp.daum.net" server doesn't work properly, either.
