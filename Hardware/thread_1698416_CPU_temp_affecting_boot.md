### Post by grahammechanical on 2011-03-02
> It seems like the cpu needs to literally "warm up" to a certain temp before it will work properly. Is that possible?

Only if the machine has vacuum tubes/valves. Heat is the killer with solid state electronics. Transistors and integrated circuits do not need a warm up time.

Perhaps some components are not seated correctly and only begin to get a proper electrical connection after they have heated up and expanded a little. Make sure everything is seated fully.

Regards.
