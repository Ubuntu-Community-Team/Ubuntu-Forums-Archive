### Post by Rooster2000 on 2014-03-24
I have this old motherboard Abit KT7 which doesn't boot at all with Athlon 1200MHz. Based on manual it should support both Duron and Athlon. I have cleared the CMOS memory several times but still no luck. The power goes on and fans and hard disk starts but after that it gets stuck. The speaker doesn't beep and there is no picture on display. I didn't find on the manual that any jumpers should be changed, installed or removed in order to run Athlon/Duron. The processor itself should be fine because it worked ok in MSI K7T Pro2.

Abit KT7 seems to have some difficulties recognizing other processors too. With Duron 800MHz everything goes fine, but with Duron 1100MHz it says the processor is unknown and 1100MHz. When I run *System Profiler and Benchmark* in Lubuntu, it says the processor is

```
Name: Unknown CPU Type
Family, model, stepping: 6,8,1 (AMD AthlonXP/MP (Throughbred))
Vendor: AuthenticAMD
...
Cache size: 64kb
Frequency: 1100,00 MHz
```

All these processors were recognized without efforts by MSI. How come Abit is having such problems? Anything I could try to get my Athlon work with it?
