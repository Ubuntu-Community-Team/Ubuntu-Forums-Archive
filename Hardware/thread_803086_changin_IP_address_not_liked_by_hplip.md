### Post by tylerjroach on 2008-05-22
I'm sorry I dont have an answer for this one. In fact i asked this question about 2 weeks ago and it was never answered. What I have done when it changes is go to add printer in hplip and see the ip address of the "new" printer. And then go to the regular printing item on the administration menu. From there you can make changes to the ip address. This only keeps you from having to reinstall it every time.
