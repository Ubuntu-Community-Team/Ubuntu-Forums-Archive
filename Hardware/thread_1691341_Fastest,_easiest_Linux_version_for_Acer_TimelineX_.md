Jolicloud is good for more of the cloud-type experience - yes you can still install apps, but more or less I'd only recommend jolicloud for a computer that has like a 16GB SSD.

Puppy is good for systems that don't have much RAM, it has Openoffice, mtpaint, and other programs on it. It's not bad. Probably won't do HD video though.

OpenSuSE is good, I had problems with it overheating my laptop. I like OpenSuSE for easy LDAP Server setup, flash usage (seems faster on OpenSuSE). Some of the programs I require I had to compile on my own.

Others, personally I'd say Ubuntu, install the ATI/AMD Drivers directly from AMD's site. - I'm using them they're working a lot better than the ones that jockey gets (open source fglrx). It will play HD Video... I haven't tried on this machine, but I play games, watch movies, etc. All on a gateway nv53 - 4GB RAM, ATI Radeon HD 4200, 500GB HDD, AMD Athlon II X2 2.0 GHz etc.

Ubuntu's probably the best and the community is amazing.
