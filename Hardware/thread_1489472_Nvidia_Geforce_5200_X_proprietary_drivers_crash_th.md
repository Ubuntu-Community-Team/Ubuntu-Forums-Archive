### Post by hsoulen on 2010-05-24
Hi There,

If you are having problems getting the driver to load from Hardware Drivers, have you tried a direct download from Nvidia?

[http://www.nvidia.com/object/linux_display_ia32_173.14.25.html](http://www.nvidia.com/object/linux_display_ia32_173.14.25.html)

You will need to download to a folder (not your desktop!), stop gnome (sudo gdm-stop) then run the script to install (I recommend you run it with "sudo su" to switch to root), you may get a warning or two during the installation but it should work.

Good luck,

Hank.
