### Post by Chibone on 2009-03-08
I'd normally go for the 2 GHz machine since there is space for more memory and better graphics card potential.  I think AGP is a better interface for graphics cards as PCI gets swamped rather easily if I remember correctly.  Modern processors are all quite fast.

There is an exception though: if your processor is a Pentium IV paired with slow RAM, in which case I would go for as much GHz as possible.  I have an old desktop which used to be equipped with a P4 1.5 GHz, 1.25 GB memory, and a 4x AGP slot with an NVIDIA Geforce 5200. It was slow until I upgraded to a 2 GHz P4 for $25 and I still felt like it would have been better with a faster processor.  It might be that the memory is a bottleneck though as it was only PC100 SDRAM.  Funnily enough it became quite fast in Windows XP (I had Ubuntu 8.04 on it which was slow for some reason.)

Just some anedcotal evidence to help you in decision making.
