### Post by hugokra on 2009-06-16
Problema resuelto asi:


1. Abrimos el Terminal.
   Nuestro primer paso sera sera abrir el terminal, esto dando clic en Aplicaciones ->
   Accesorios -> Terminal.
2. Abrimos Nautilus desde el terminal.
   Luego de abrir el terminal debemos escribiremos
          Sudo nautilus
   A continuación, damos clic en enter. Y seguidamente se cargara una nueva ventana. En la
   cual tendremos permisos de administrador del sistema. Eso quiere decir en otras palabras
   que podremos modificar datos y archivos.
3. En busca del archivo Xorg.conf
   Dentro de la ventana debemos buscar una barra donde dice /root. En lugar debemos escribir
   /etc/X11/ y seguidamente dar enter.
4. Archivo Xorg.conf
   Una vez que hayamos entrado debemos buscar el archivo Xorg.conf.
   Dicho archivo contiene todas la configuración posibles que se pueden dar al monitor que
   estamos usando. Así que es necesario que lo abramos para modificarlo.
5. Abrimos el archivo Xorg.conf
   Cuando ya hayamos ubicado el archivo daremos sobre el mismo doble clic para abrirlo. Y a
   continuación procederemos a modificarlo.
# xorg.conf (X.Org X Window System server configuration file)
# This file was generated by dexconf, the Debian X Configuration tool, using
# values from the debconf database.
# Edit this file with caution, and see the xorg.conf manual page.
# (Type "man xorg.conf" at the shell prompt.)
# This file is automatically updated on xserver-xorg package upgrades *only*
# if it has not been modified since the last upgrade of the xserver-xorg
# package.
# Note that some configuration settings that could be done previously
# in this file, now are automatically configured by the server and settings
# here are ignored.
# If you have edited this file but would like it to be automatically updated
# again, run the following command:
# sudo dpkg-reconfigure -phigh xserver-xorg
Section "Device"
          Identifier "Configured Video Device"
          Driver "vesa" (Este es el driver de Linux para la tarjeta de vídeo)
EndSection
Section "Monitor" (Este nos muestra el tipo de monitor que tenemos)
Identifier "Generic Monitor"
EndSection
Section "Screen" (Este nos muestra la resolución de nuestra tarjeta de vídeo)
          Identifier "Default Screen"
          Monitor "Generic Monitor"
          Device "Configured Video Device"
EndSection
   6. Inicia la modificación de Xorg.config.
       Para iniciar la modificación lo primero es buscar la sección Section “Monitor” y en ella
       agregar las siguientes lineas de texto:
                  Section "Monitor"
                  Identifier "Generic Monitor"
                  Option "DPMS"
                  HorizSync 25-75 (esta es la taza de refrescamiento horizontal del monitor)
                  VertRefresh 40-150 (esta es la taza de refrescamiento vertical del monitor)
              EndSection
Nota importante: si no sabes de donde debes sacar la taza de refrescamiento horizontal y vertical
de tu monitor. Te recomiendo que busque en la parte de atrás del mismo, pues normalmente traen
un etiqueta del fabricante en donde están esas especificaciones.
   7. Seguimos modificando Xorg.config
      Ahora debes buscar siempre dentro del archivo Xorg.conf la linea de texto que dice: Section
      “Secreen”.
      Una vez ubicada en la misma deberás agregar estas lineas (las cuales he subrayado en
      amarillo).
                  Section "Screen"
                           Identifier "Default Screen"
                           Monitor "Generic Monitor"
                           Device "Configured Video Device"
                           DefaultDepth 24
                           SubSection "Display"
                           Depth 1
                           Modes "1280x1024" "1024x768" "800x600" "640x480"
                           EndSubSection
                           SubSection "Display"
                           Depth 4
                           Modes "1280x1024" "1024x768" "800x600" "640x480"
                           EndSubSection
                           SubSection "Display"
                           Depth 8
                           Modes "1280x1024" "1024x768" "800x600" "640x480"
                           EndSubSection
                           SubSection "Display"
                           Depth 15
                           Modes "1280x1024" "1024x768" "800x600" "640x480"
                           EndSubSection
                           SubSection "Display"
                           Depth 16
                           Modes "1280x1024" "1024x768" "800x600" "640x480"
                           EndSubSection
                           SubSection "Display"
                           Depth 24
                           Modes "1280x1024" "1024x768" "800x600" "640x480"
                           EndSubSection
                   EndSection
        Luego de agregar las lineas que te he subrayado deberás tener presente que si quiere lograr
        la resolución máxima de tu monitor. Este es el punto indicado donde debes agregar la
        misma.
Ejemplo: en el caso de mi monitor la resolución máxima del mismo es “1280x1024” por tal
razón la he agregado en la lineas de Modes.
Así que, sin importar cual sea la resolución máxima de tu monitor. No te olvides de agregarla en
estas lineas dentro de “Modes”.
        Al terminar de hacer este paso solo nos resta salvar o guardar el archivo. Y luego reiniciar
        nuestra computadora.
        Una vez esta reiniciado te encontrarás con la gran sorpresa de que ya puedes cambiar la
        resolución de tu monitor.
        Claro si no te recuerdas donde. Has clic sobre Sistemas -> Preferencias -> Pantalla.
        Hecho esto todo estará listo para que disfrutes al máximo tu Ubuntu 9.04 Jaunty Jackalope.
