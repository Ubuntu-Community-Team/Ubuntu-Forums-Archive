### Post by Baggyeyes on 2009-04-08
The sdhci is not the only issue with this machine, I'm not a linux expert but it seems the network adapter also has problems loading ubuntu

By no means a new problem and in future it may be best to do a thorough search on the forums before posting. I managed to install ubuntu on a freevents x55, however since all the freevents X5* series are very similar the issues can be solved by blacklisting some default drivers. This website has a torrent of a modified ubuntu distro:

[Ubuntu on Averatec 2460 and Philips freevents X5* series]("http://www.fitzenreiter.de/averatec/index-e.htm")

And in answer to your question a desktop cd/dvd is also known as a livecd as the OS can be run from the disk before installing. An alternate disk is one where a very basic GUI is used to install ubuntu to the system, can be superior in certain circumstances.

Hope that helps
