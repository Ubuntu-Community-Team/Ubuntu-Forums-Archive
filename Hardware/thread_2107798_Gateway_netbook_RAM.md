### Post by ahallubuntu on 2013-01-22
Ubuntu will utilize as much RAM as you have installed.

However, if you use the 32 bit version of Ubuntu, it may not automatically recognize more than 4GB of RAM (it's possible but not always automatic).  If you use the 64 bit version of Ubuntu, it should recognize all your RAM.

Best Buy is probably the most expensive place to buy laptop RAM unless it is on sale, but it is very convenient for some people who don't mind paying a little more to get their RAM quickly.  Try Amazon, NewEgg, even eBay to save money.  The important thing is to get the exact type of RAM you need, as there are numerous types of RAM for laptop computers.  Crucial.com's website is good at finding the exact type of RAM you need.
