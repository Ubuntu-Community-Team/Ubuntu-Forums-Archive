### Post by G_man on 2008-10-27
I would recommend that you try to boot Ubuntu from the CD before you install it
Then, test your hardware (Wireless, sound, VGA ...)

A useful tool for that is "Hardware Testing" which can be found in System tab under Administration

But I would recommend you to test the networking devices personally to make sure that they are working properly (wireless, bluetooth, and eithernet) 

Note: Some drivers may not work on LiveCD as with the restricted drivers

If the drivers are working fine then everything would go smoothly.

Hope you enjoy it
