### Post by grahammechanical on 2011-06-27
Is this a laptop? Do you have to press a combination of keys on the keyboard to switch on the wireless adapter? If so then you need to do this as the operating system will not use the wireless adapter if it is switched off in hardware. What does the computer operating manual say about using wireless?

Also it is sometimes possible to switch off the wireless adapter in Windows. If you do this then Ubuntu cannot switch it back on. So, make sure that the wireless is working in Windows before you shutdown and then boot into Ubuntu.

ARe you running Ubuntu inside Windows or did you do an install alongside Windows? ARe you using Wubi to run Ubuntu? I will make difference to the advice that is given.

You may need to connect using an ethernet cable and then go to Additional Drivers to activate any addition drivers that are available. This may install a driver for your wireless adapter.

Regards.
