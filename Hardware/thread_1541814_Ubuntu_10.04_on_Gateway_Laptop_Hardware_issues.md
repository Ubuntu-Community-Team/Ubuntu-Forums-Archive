I am having quite a bit of hardware issues on the Ubuntu side of my computer, which is currently the only side working properly at the moment. **Microphones don't recognize. **I'm not really sure what the problem is, but since I've been using computers, microphones have always given me trouble. This laptop on Ubuntu is no exception, but it *did* work on Windows. I've tried two different headsets, and even the trick where you plug a pair of headphones into the microphone slot and it captures sound as a Line-in. None of these worked, and it seems to be a problem that is linked to the OS's mic functions itself. I've downloaded at least three microphone utilities, turned the volume up on the microphones, and checked line-in and microphone boxes, and no matter what the combination of settings, I just can't get the microphone to pick up any sound.

**Video card isn't 100% functional.** The laptop came preinstalled with an ATI Radeon X16 something or other graphics card that worked flawlessly on Windows. However, Ubuntu has managed to pick up the full resolution of the computer (1280x800), as well as been able to pick up the 720p HDMI colors that the monitor can output. It's also been able to run the wiggly full-blast graphics that are able to run on the computer. But for some reason, on games and full-screen programs, and much of anything I'm running through Wine, I don't get the full brunt of the graphics card's power. I get color levels restricted to a low number like 256 or just a little bit above, I get the .gif dither all over most of the graphics. 3D and OpenGL programs either won't run at all or will have huge problems drawing the 3D graphics from this EASILY capable graphics card. And anything in 3D or 2D that does run at full graphic capacity is REALLY slow to render and has poor framerate. I've downloaded every driver, support, and ATI program to try and make this graphics card be recognized fully, but no matter where I go, it doesn't seem to even notice the existence of the card. Even the ATI Catalyst Control Center for Linux doesn't see my graphics cards.

**Ubuntu is _obviously_ capable of running faster than it is.** I have a computer that came preloaded with 4GB of RAM and an AMD Turion(tm) 64 X2 Mobile Technology TL-60 processor with about 2.0 GHz of processing power give or take a tenth of a GHz. Despite this, Ubuntu only recognizes 3.3 GHz of that speed, and even then, it is much slower than my processor-heavy Windows side of the computer. Flash applications on websites cause the computer to try and overwork itself, video and music takes up much of the System's power, and programs that could run on Windows at full power faster than the Linux versions of them seem to be the norm. 

**DVDs (movies) don't read in my DVD drive.** I have a DVD-RW drive that was preloaded into the computer, and each time I've tried to put a DVD movie into the disk drive, on any player I get an error saying something along the lines of the data not being able to be read on the disk.

**Browser-embedded JAVA apps don't run properly.** Every now and then an application on a website which was coded in JAVA doesn't load properly. Most of the time the issue tends to be only the upper left corner is all that shows up of the application on Ubuntu where it worked fine on Windows.

These are just the first few major problems that come to mind which I would really like to find solutions to. I've installed the restricted extras, I've installed utilities and programs from the Synaptic Package Manager, and I've done all that is in my limited knowledge of Ubuntu to solve the problems that my laptop never gave me on Windows. If there is anyone who knows good solutions to these problems, please please _***please***_ tell me, as it would be very nice to get this system running like it should.

Thanks in advance.
