### Post by nicedude on 2008-08-29
Put in the ubuntu non-region dvd and thinks it is a blank disc, having previously installed off it 

At this point right here you say it won't boot a Live CD that you know is good , well that has nothing to do with any OS or Fstab settings as none of that would come into play at this stage ( or any stage since your booting the OS from the CD not your Hard disk ). You should know this if you know much about how PC's work :-(

 put in bought music CD and doesn't find it

Most likely because it is broken

run wininfo thru wine and recognizes drive, as it did before it stopped working.

So that is just that it sees the drive not that it works, you might try looking in your BIOS and see if it shows up properly there and maybe try changing settings, Like make sure your DVD is set as the boot drive before the hard disk is and once you verify that try booting from a live CD again. But if that is still a no go then your drive is sunk, sorry.
