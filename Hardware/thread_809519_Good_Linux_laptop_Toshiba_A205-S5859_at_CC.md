### Post by stchman on 2008-05-27
Hello all, if anyone is looking for a Linux laptop I can recommend the Toshiba one in the title.  I purchased it over the weekend from CC and everything works OOB on it using Hardy.  CC has more of these

Intel Core 2 Duo
2GB DDR2 RAM
DVD burner
15.4 LCD
4965 wireless
10/100 Ethernet


Intel X3100 works using Compiz.
Intel 4965 wireless works OOB.
Webcam works OOB along with bezel microphone.
Wired ethernet works OOB. 
Realtek HD sound works including headphone jack sense.
Ti 5 in 1 works with SD but not with XD.

I booted up the LiveCD and everything worked.  CC has quite a few of these for $600.  A great buy.  Toshiba's new laptops use the Marvell ethernet which is not Linux friendly.  Not to mention the new models are pretty overpriced for the fancy new black finish.

Just wanting to let all the folks know that want a Linux compatible laptop.
