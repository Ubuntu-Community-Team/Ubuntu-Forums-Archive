### Post by roger_1960 on 2010-06-06
Hi

On the phone, (assuming you have not yet done this) before you plug it in, go to:
settings>connectivity>USB>SetDefaultMode and select USB mem device file transfer.

Then plug in and nautilus should see the memory card rather than the phone memory.

When you have finished, change it back to the previous setting.
