### Post by bloodorange on 2008-05-15
I've got the same fantastic laptop.  I'm dual-booting Ubuntu Studio 8.04 and Vista and it works fine.  Although, to begin with, I had trouble with the touchpad on Ubuntu but, after a long search, found that it's easy to fix.

There are a few issues with 8.04, but I'm sure they'll get fixed soon enough.  For one thing, Compiz doesn't work very well.  Or at least not as well as it did on Gutsy.  But I can live with that...

I reduced the size of my Vista partition dramatically because I hardly ever use Vista.  In fact, the only real reason I've kept it on is because of the slightly annoying Media Centre button on the laptop.  If I could stop that button working, I would take over the whole HDD with Ubuntu.  I've also got Virtual Box on Ubuntu running Vista, so I've got the best of both worlds - I actually think Vista's OK.

### Post by bloodorange on 2008-05-16
Sounds OK.  My HDD is 250GB.  I've left the hidden Media Direct partition as it is, then I shrunk the Vista partition to about 30GB.  I made a root partition of about 15GB for Ubuntu, a swap of about 4GB, and the rest as a separate home partition.

I think a separate home partition is essential; keeps your stuff safe.
