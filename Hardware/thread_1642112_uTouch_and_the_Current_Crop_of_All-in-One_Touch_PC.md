### Post by KoRnholio on 2010-12-09
I've been interested in both the Vaio and HP Touchsmart "All-in-One" touch pcs. Using the Vaio at a Sony Style store, I was very impressed and would consider one for a purchase if the touch functionality worked with Ubuntu.

Searching the forum for compatibility, there doesn't seem to be much experience with this. Here's what I found:

In a thread about the Dell Studio One, it looks like touchscreen functionality works in Linux if you run the NextWindow drivers under Mac mode. Not multi-touch, however.  Looking at the NextWindow site, HP Touchsmart pcs also use their hardware, so I think it would follow that mileage with those would be similar to the Dell.

Onto Vaio's: I've seen one thread about this, and it looks like you can't even get touch functionality with this. That majorly sucks. The Dell's are butt-ugly and I have no intention of getting one, but the HPs are attractive enough so the lack of Vaio support isn't as terrible.

So that's what I can tell about the current status of support for these machines. But I also want to ask: Does the work that Canonical has been doing on uTouch affect these? I just read an article in an Ubuntu magazine about this, and it looks like Canonical has really been going headfirst into multitouch and gestures. Would this uTouch stuff apply to PCs like the HP Touchsmart at all?
