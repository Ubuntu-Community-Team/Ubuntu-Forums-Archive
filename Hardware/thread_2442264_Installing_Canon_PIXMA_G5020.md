Hi everyone, 

I just wanted to leave a note here on how to install a canon printer (Canon PIXMA G5020)

You just go to this page 

[https://www.usa.canon.com/internet/portal/us/home/support/details/printers/megatank-inkjet-printers/all-megatank-inkjet-printers/pixma-g5020?tab=drivers_downloads](https://www.usa.canon.com/internet/portal/us/home/support/details/printers/megatank-inkjet-printers/all-megatank-inkjet-printers/pixma-g5020?tab=drivers_downloads)

download the software for ubuntu, decompress it:

tar -xvf cnijfilter2-5.90-1-deb.tar.gz

and run the installation file, 

bash install.sh

You follow the directions on the screen and it works!
