### Post by aikishugyo on 2010-05-08
Hi,

Probably the reason you cannot detect the scanner with Debian packages is that there *are* no Debian/Ubuntu packages for the latest SANE, 1.0.21. So you have to build it from source in order to get support for your MP250 scanner capability, since it is only available in the latest pixma backend (0.16.1). Note that SANE 1.0.20 is two years old!

If you need help compiling SANE, PM me. It is easy, and installs in /usr/local so as not to clash with your system SANE. When Ubuntu eventually packages the new SANE, you can run "make uninstall" to get rid of the source-installed one.

HTH,
Gernot Hassenpflug
