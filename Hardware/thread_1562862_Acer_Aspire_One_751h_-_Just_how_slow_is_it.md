### Post by pirx82 on 2010-08-28
Hi!

I own this exact model and I run the Netbook 10.04 version on it.

The OS works surprisingly fast and so does the software. I can run Firefox, Thunderbird, OpenOffice, Transmission all at the same time and it still has room for some more. I was *very* surprised how smooth everything works. Even Google Earth runs quite decently. But you have to download the GMA500 chipset "patch" ([https://wiki.ubuntu.com/HardwareSupportComponentsVideoCardsPoulsbo](https://wiki.ubuntu.com/HardwareSupportComponentsVideoCardsPoulsbo)) - otherwise it's quite unusable.

However, I also had Win 7 Ultimate installed before that and the OS was so slow, that I lost my will to use it. Everything took ages to run. If I wouldn't found Ubuntu, I would probably sell it.

By the way, I still have an ongoing nasty problem with booting up... Sometimes it freezes during the bootup and I need to reset it by turning it off. I'm still working on this issue... But, the "good news" is, that I haven't seen anyone else report this problem. ;)

I use the Ubuntu 10.4 Netbook edition on an ACER Aspire One 751h.
Even with the GMA500 driver, I found it very slow (I often type quicker than the letters appear, and it takes several seconds to display the launcher when I click on the upper-left Ubuntu logo).

### Post by PendragonUK on 2010-10-28
I'm running 10,10, I did try to install the netbook version but found that Unity dose not work at all. It's requirement for 3D may be the issue. This site is a must:
[https://wiki.ubuntu.com/HardwareSupportComponentsVideoCardsPoulsbo](https://wiki.ubuntu.com/HardwareSupportComponentsVideoCardsPoulsbo)

With Lucid the 3D worked but with 10.10 it does not. 

I have used may different distros on this netbook. I have found that the GUI's designed for netbooks are not best for the screen size. The screen may be small but the rez makes the standard desktop feel a lot better. Jolicloud runs very well but as I mentioned before I'm not impressed by the GUI. PCLinuxOS runs nicely so does Fedora. 

As for Windows, XP is a dog! yes everything works but all so slowly as to make it horrific. Windows7 however is excellent. It all works and feels a zippy, but everything works where it does not with Ubuntu.

May I suggest Ubuntu 10.04 LTS, the standard desktop edition, If you insist on dual-booting and using Windows I strongly suggest Windows 7 Home Premium. I fear if you do install Win7 you will never boot to Ubuntu unless you are using Ubuntu for it's own sake. 

Clean install 10.04LTS (*), update, then run the "Poulsbo" install found on the site mentioned above, all will be right with the world of running a 751h


(*)if you get the black screen of death on first boot then restart using recovery and go from there...
