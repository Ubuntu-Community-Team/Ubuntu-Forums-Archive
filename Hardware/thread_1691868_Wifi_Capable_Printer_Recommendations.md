### Post by deragon on 2011-02-20
Greetings.

The Brother [MFC-J615W]("http://www.brother-usa.com/MFC/modeldetail.aspx?PRODUCTID=MFCJ615W") is fantastic.

I run Ubuntu 08.04 LTS (Hardy Heron) and the drivers of Brother works marvelously.  Wifi works for printing, but not for scanning.  For scanning, you need to connect the device through a USB cable.  This is because XSane, at least the version shipped with Ubuntu 08.04, does not allow you to setup a scanning device through the network.  BTW, for XSane to run under 08.04, you must run it as root.

The other beauty of this printer is that there are no electronics preventing you to insert 3rd party cartridges, and you have on cartridge per color.  Cartridges go for less than $10 on eBAY.

The only feature I have not tested yet is FAX, as I have no use of it.

So you have a cheap per page printer that is very compatible with Ubuntu, at least version 08.04.

Best regards,
deragon
