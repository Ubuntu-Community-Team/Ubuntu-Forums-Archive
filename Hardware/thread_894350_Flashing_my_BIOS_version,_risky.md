### Post by linux_tech on 2008-08-19
Flashing bios is easy as downloading file.  Then either make a floppy, boot to it or run a executable that handles the bios flash. Just make sure you go to the website for your computer and download correct bios, then make sure

You can also try using command-line parameter acpi=off at the install menu  with the alternate text-based install cd.
