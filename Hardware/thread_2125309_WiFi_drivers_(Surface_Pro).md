Without an internet connection, you're likely to face a very difficult time installing to a Surface Pro -- because Canonical can not possibly include all the drivers for all the hardware out there on a single disk, so where do you think the installer goes to get the drivers it needs?

And Yes, you CAN install without an internet connection -- if you're using older equipment for which all the drivers are already on the disk.

As to what will be included with 13.04, probably the best way to find out at this point is to ask in the Raring Ringtail Development section.
