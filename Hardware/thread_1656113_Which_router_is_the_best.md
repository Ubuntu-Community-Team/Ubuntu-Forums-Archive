### Post by groovomata on 2010-12-30
Hello All, I've recently bought an external hard drive and I'd like to purchase a wireless N router to attach it to, so it's available through my network and so I don't need to plug it directly into my laptop. Two models I'm considering are: the D-Link Wireless N Router DIR-825, and the Netgear WNDR3700-100PAS. Both are the same price - $100 (CAD). The only real differences between them that I can see are that the Netgear specifically says it works with linux, D-Link makes no such claim. However, the D-Link allows various usb devices to be connected to it such as printers while the Netgear only allows a hard drive. 

Does anyone have any thoughts on this?
Thanks,
Erik.

As you are talking about routers you don't really need it to say its Linux compatable as you can access it from any browser as long as you are connected to it.
