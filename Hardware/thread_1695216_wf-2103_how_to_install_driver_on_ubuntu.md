### Post by houseworkshy on 2011-02-25
Just a thought, try your wireless anyway it may work already (mine did).  Then try hardware drivers from System>Administration, it may do it for you.  If not go to Applications>Ubuntu Software Centre and put in sysinfo.
This will put a link in Applications>System tools, run that and "save" this will generate a text file with your hardware specs.  Posting that will help others to help you.
