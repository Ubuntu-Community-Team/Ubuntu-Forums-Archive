### Post by thomaswp on 2012-09-23
Solved.  Found this thread:
[http://askubuntu.com/questions/175933/why-did-my-bluetooth-headset-disappear-from-sound-settings-and-how-can-i-get-it](http://askubuntu.com/questions/175933/why-did-my-bluetooth-headset-disappear-from-sound-settings-and-how-can-i-get-it)

And did this:
```
:~$ pactl list cards short
0	alsa_card.pci-0000_00_1b.0	module-alsa-card.c
2	bluez_card.00_1A_7D_11_34_A1	module-bluetooth-device.c
:~$ pactl set-card-profile bluez_card.00_1A_7D_11_34_A1 a2dp

```

### Post by thomaswp on 2012-09-27
I had more problems tonight.  This thread helped:

[http://askubuntu.com/questions/145733/a2dp-connection-not-shown-in-pulseaudio](http://askubuntu.com/questions/145733/a2dp-connection-not-shown-in-pulseaudio)

I installed pulseaudio-module-bluetooth
```
sudo apt-get pulseaudio-module-bluetooth
```

Then
```
pactl load-module module-alsa-sink device=bluetooth
```

Then a reboot and it all started working again.
