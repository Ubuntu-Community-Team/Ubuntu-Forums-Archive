### Post by #&amp;thj^% on 2018-06-22
Do you wish/want to still use Bluetooth after a boot-up?
This "should" stop it from automaticaly starting at a boot level.
Run 
```
sudoedit /etc/rc.local

```
and add this **before** the line with **"exit 0:"**
```

rfkill block bluetooth
```

You should still be able to enable Bluetooth through the top bar applet.

This "should" work for most systems but also read the below.

NOTE: But for some Thinkpads you will need to add the following to **"/etc/rc.local":**
```

echo disable > /proc/acpi/ibm/bluetooth
```
For My ThinkPad T430 the first command works for me.
