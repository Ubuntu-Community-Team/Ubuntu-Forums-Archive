### Post by porikonut on 2008-06-07
I read other tutorials,but they didnt worked for me so I decided to write this simple tutorial.

1.Download these and install them
[http://rapidshare.com/files/120820772/z600cups_1.0-2_i386.deb]("http://rapidshare.com/files/120820772/z600cups_1.0-2_i386.deb")
[http://rapidshare.com/files/120821155/z600llpddk_2.0-2_i386.deb]("http://rapidshare.com/files/120821155/z600llpddk_2.0-2_i386.deb")
2.Go to System>administration>Printing>New printer
When it asks for model click install driver and select /usr/share/cups/model/Lexmark-Z600-lxz600cj-cups.ppd (it may be gziped if so just extract it), finish the wizard
3.You're done ENJOY :)
