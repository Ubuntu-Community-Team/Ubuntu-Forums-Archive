### Post by joshedmonds on 2009-06-14
RE: fan running non stop

[https://help.ubuntu.com/community/AspireOne]("https://help.ubuntu.com/community/AspireOne")

Scroll down to 'NOISE (FAN CONTROL)' and follow the instructions to install the acerhdf kernel module (not the acerfand daemon).

This worked on my AA1 110L with bios v3309. FYI I have seen mention (on these forums) of possible issues if using bios v3310.
