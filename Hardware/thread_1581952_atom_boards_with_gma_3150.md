### Post by Lanser on 2010-09-28
Drbisio. the S10-3 with this chipset uses shared memory, so the more memory the better. The cost of memory is also so low, that 2Gb would be a min spec.

If you are going to buy a mATX MB for a media centre build, there are probably better chipset choices..... have a look at the ASUS web site.
