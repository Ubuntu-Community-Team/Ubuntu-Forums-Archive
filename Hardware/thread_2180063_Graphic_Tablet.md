### Post by Favux on 2013-10-11
Hi The_Aegidius,

What release of Ubuntu does your girlfriend have?

A lot of graphic tablets are supported.

Wacom tablets tend to be well supported but it takes a while for support for newer models to appear in the distributions like Ubuntu.  For example a Wacom Bamboo Pen and Touch from last year would be supported in Raring (13.04) but the new Intuos Pen and Touches are not yet supported since the patches for their support are still waiting to be included in the current linux development kernel.  The Intuos Pro models just got included in the kernel but it will probably be a while before a Ubuntu release catches up to that kernel.  The nice thing about Wacoms is you can compile input-wacom (for the kernel driver) and xf86-input-wacom (the X driver) from the linuxwacom site ([http://sourceforge.net/apps/mediawiki/linuxwacom/index.php?title=Main_Page](http://sourceforge.net/apps/mediawiki/linuxwacom/index.php?title=Main_Page)) before a Distribution officially supports them.

Non-wacom tablets tend not to be quite as well supported.  For some of them see [http://sourceforge.net/apps/mediawiki/digimend/index.php?title=DIGImend](http://sourceforge.net/apps/mediawiki/digimend/index.php?title=DIGImend).
