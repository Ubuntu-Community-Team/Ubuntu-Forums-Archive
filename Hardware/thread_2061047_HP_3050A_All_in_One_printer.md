### Post by sid1950 on 2012-09-21
I used to have one of these, and never did get the configuration to work from Ubuntu. I was using a laptop with Ubuntu 11.04, but I had Win XP installed in VirtualBox, so I booted that, and used the tools that came on install CD. Once it was set up in Windows everything worked fine from the rest of my network.

I gave up on that HP after a couple of months. It is very flimsy. The paper feed stopped working and the cost of shipping it back under warranty was more than buying a new one! I ended up throwing it away and buying a Brother MFC-J430W which has worked fine ever since. Again I had to run the Windows install disk from my VirtualBox XP but it was much easier to setup.
