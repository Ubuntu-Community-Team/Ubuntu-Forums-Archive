### Post by saggz on 2011-02-02
For anyone having this problem, I figured out what it is. The newer drivers make it crash, if you have an Acer Aspire 9300 with a Geforce GO 6100, use driver version 96 in Ubuntu. Not the newer ones because they cause it to crash.
