### Post by coffeecat on 2012-01-20
> **lonekingc4 said:**
> Are there any open source Ubuntu supported drivers for this card? If so, how can I install it?

Yes, the one you were using before you installed the proprietary driver, and the one you are using now (if you are getting a GUI). :)

Ubuntu comes complete with the open source drivers for ATI cards. Is there a problem with the driver for you? So long as you are not a gamer, the open source driver is usually more than adequate for giving you 3d for compiz/Unity and for high-definition video playback. I don't have experience of your card, but my Radeon HD4350 and HD5450 cards work just fine (for my purposes) with the default open source driver.
