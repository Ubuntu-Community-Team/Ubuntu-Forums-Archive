### Post by daveloh on 2011-09-27
Just got my new Dell Latitude E6410 notebook and install Ubuntu 11.04 AMD64 version.

After install was leaving patching running unattended, so I close my lid. After a while, I come back----then my screen had FREEZE!!!!

no matter what I do, it does seem to recover. I wait for 5 mins still the same. 

My setting is disable all screen saver, and set close lid and "Blank Screen".

Reading some article but haven't find the solution yet.

Any suggestion?

---

### Post by daveloh on 2011-09-27
Got it!!!

just install "compizconfig-settings-manager"

then go to control panel configure Open GL -> uncheck "Sync to VBlank"
