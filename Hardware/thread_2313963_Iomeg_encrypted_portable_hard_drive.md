### Post by ajgreeny on 2016-02-17
The encryption software used for that disk was obviously proprietary and for Windows, IomegaEncryptionSetup.exe, so will not run in Ubuntu.

Was the previous machine (the Sony) running Windows in which you installed Ubuntu using wubi, ie, you installed Ubuntu while Windows was running?  Wubi is no longer supported or used by anyone here, so you will get very little help if that was the case.

If you still have the old machine from which you can read that disk and backup the files on the encrypted disk to another unencrypted disk, you might then be able to format the encrypted disk and make it as new, so to speak.

However, I do not, and never have used encryption on any disk, so I may be wrong and it may be impossible to format an encrypted disk without knowing its passphrase.
