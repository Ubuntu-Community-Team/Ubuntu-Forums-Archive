### Post by alexandari on 2012-12-07
I wouldn't recommend using a virtual desktop for playing games,especially with your configuration. As you know a virtual desktop takes half or sometimes more than the half of the computer's memory,CPU (depends on how you set it up). Have you tried running Ryzom with Wine? As much as I can see,there is a wine appdb page for that game [http://appdb.winehq.org/objectManager.php?sClass=version&iId=20325](http://appdb.winehq.org/objectManager.php?sClass=version&iId=20325) . Upgrading your machine would affect your performance,but I take it as unnessacery because you can set up your things the way you want them to work but just not like that. Have you considered making the opposite - running Windows XP and running Ubuntu via a virtual desktop? Seems like you need more resources for the Windows than for the Ubuntu
