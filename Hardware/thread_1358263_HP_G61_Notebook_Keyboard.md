### Post by Eben Fourie on 2009-12-18
Hi,
 
I've installed Karmic on a HP G61 notebook. It has a numeric keypad type keyboard.
 
The keypad part works when you switch off "Pointer can be controlled using the keypad" in System ? Preferences > Keyboard (Mouse Keys tab) + switch ON numlock.
 
The keyboard layout for the QWERTY part of the keyboard is not 100% correct.
 
When I type SHIFT + 1,2,3 I see "!@#" on the screen, but the keys themselves are labelled:
 
1 - exclamation mark
2 - double quote
3 = GB pound sign
 
To the right of the J, K & L keys are:
 
:; then @, then ~#
 
Any idea which keyboard layout I should use ?
 
Thanks in advance,
Eben

### Post by prshah on 2009-12-18

"!@#" is the USA layout; you probably need to use the "United Kingdom" or "Great Britain" layout.

### Post by Eben Fourie on 2009-12-21
Thanks, got it working with the following:
 
Keyboard: HP SK-250x Multimedia Keyboard
Layout: United Kingdom
 
To get the | (pipe) character working - third character on the key to the left of the 1, you need to press "alt gr" + the "|" key
 
"alt gr" is to the right of the spacebar.
