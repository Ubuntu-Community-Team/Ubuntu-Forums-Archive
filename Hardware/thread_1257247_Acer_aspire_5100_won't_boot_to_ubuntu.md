### Post by JHan816 on 2009-09-03
I have the same model. The USB CDROM setting is for an external USB CD drive. I set ---> "IDE:0 Slimtype DVD A DS8A1P-(PM)" at the top of the boot order on mine.
 
Your DVD/CD drive name may be different but you should be able to find it on the list. It should say "DVD". There should be devices in the boot list for the HD, should say IDE, SATA, etc.
