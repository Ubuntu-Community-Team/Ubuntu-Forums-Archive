### Post by Yellow Pasque on 2011-03-08
The P4m's are long out of production, so you're looking for used ones. They're mobile variants of Northwood-core P4's. Unfortunately, the numbering scheme was very confusing then and the ordering code is probably the best I can give you: 
[http://ark.intel.com/Product.aspx?id=27363&processor=&spec-codes=SL6WZ](http://ark.intel.com/Product.aspx?id=27363&processor=&spec-codes=SL6WZ)
