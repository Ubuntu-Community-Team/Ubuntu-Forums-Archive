### Post by pastalavista on 2009-07-09
the application you want to run is called 'system-config-printer'. try running it in the alt-f2 run dialog. If it runs, make a new entry for it in the menu manager. If it doesn't run, you need to re-install it. ```
sudo aptitude reinstall system-config-printer
```
