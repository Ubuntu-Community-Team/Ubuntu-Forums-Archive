Try searching and using **Synaptic Package Manager** to install the drivers for the MFC-210c printer.

I am assuming you have the MFC-210C Brother printer.

For the scanner,you have to download the .deb file from the [Brother website](http://solutions.brother.com/linux/en_us/download_scn.html) and double click on it to install the driver.

Your printer uses the brscan2 .deb file.
