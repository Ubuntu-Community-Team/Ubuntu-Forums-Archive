### Post by squigish on 2012-09-19
The difference between PCIe 2.0 and PCIe 3.0 is that 3.0 is a newer standard, but one that is fully backwards compatible with 2.0.  You should have no problems using a PCIe 2.0 graphics card in a 3.0 motherboard, or vice versa.  

For PCIe, the "x16", "x8", "x4" or "x2" refers to the number of "lanes" the device uses.  The x16 slot is physically the longest, and almost all graphics cards these days require a slot that is physically an x16.  On some motherboards, if you're running more than one graphics card, the motherboard will reduce the number of lanes available to some of the x16 PCIe slots.  Your graphics card should still run fine. 

In summary, if you're only planning on using 1 graphics card, and your motherboard supports PCIe 2.0 x16 or 3.0 x16, you can use a graphics card that supports either of those standards, and you should be fine.
