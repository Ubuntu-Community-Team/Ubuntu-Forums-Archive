### Post by TheFu on 2022-03-30
Updates below:

[https://frame.work/](https://frame.work/) should be THE most upgradable laptop. Pretty much everything can be customized.
LTT did a video-cast on it.

I'm more of a $300 laptop person, so it is doubtful I'll ever get one of those.
As for Linux compatibility, if we stay away from the VERY low-end, and stick with Lenovo, Dell, and perhaps HP ... only minor things usually don't work - like fingerprint readers. Of course, there are some models that have bad NICs or bad wifi and going too new on hardware is always a danger for drivers.  But that applies to all hardware and Linux.  Avoid the bleeding edge.

I have an Asus laptop now that works well with Ubuntu too. Bought it used ($305).  Swapped the 2.5in HDD for a 2.5 SATA SSD before I ever booted it. It came with 8G of RAM, which I find just fine for a laptop.  Core i5 8th-gen. On a desktop where I need to run containers and VMs, 16G is my minimum RAM these days ... or 32G if prices are cheap.  

I cannot imagine wasting 16G on a laptop, but I don't program in Java either. If I was doing java development, I'd want a Core i9 and 128G of RAM to fight the bloat.  Alas, I do perl development and that works just fine on a 2G chromebook running a light Ubuntu. If I need to do video editing, the short answer is use a desktop with a $250+ CPU and pretty much choose the GPU based on performance needs.  For my video editing needs, a $70 GPU (nvidia 1030 or AMD Ryzen 5xxxG APU) have been fine.  The onboard Core i5 iGPU feels 20x slower.  Read this morning that the USA is removing a 25% tariff on GPU cards and that some AMD cards have dropped 25% already. I haven't verified this.  The top-end GPUs will likely never have discounts - people chasing high performance always overpay.  

People chasing light/tiny always overpay too.  A typical mid-tower desktop is about the cheapest answer for computing, though a boot-box-size system isn't obscenely expensive. All the NUC systems are over $100 too costly for what is provided and don't allow upgrades.

For laptops, many things can be added as external USB options. A $25 USB3.2 hub can handle nearly anything we'd add to a laptop. Only CPU, RAM, and GPU wouldn't be included and GPUs are probably coming.  RAM upgrades are easy on most mid-range laptops. Just check that it isn't soldiered on first.  Same for storage - avoid eMMC storage. Look for m.2 NVMe or SATA or 2.5inch SATA inside the laptop.  For me, the keyboard fails long before the CPU usefulness ends, about every 24-36 months. Then it is time for a new keyboard, which runs $80 +$100 installation fee.  In the old days with thicker laptops, swapping a laptop keyboard was 5 minutes of effort and $30 in parts.  That's changed with the thin laptops.  My last 2 laptops used plastic rivets to hold the keyboard to the face and removal of everything inside was required to access it - that's the storage, ram, motherboard, CPU, heat pipe ... all wifi ... everything just to swap the keyboard.  And because they use plastic rivets, the replacement never is attached as firmly as the original keyboard was.

With a desktop, we can use nearly any keyboard we like for decades.  3 months ago, I retired an IBM 101M keyboard and started using a newer mechanical USB keyboard with less loud clicks, but still a great 'feel'.  We don't get that choice with laptop keyboards.  I'm partial to Dell laptop keyboards and dislike most other brands. Perhaps I'm a snob and hard on keyboards?  Hard to tell.  The IBM only failed after about the 15th cleaning over the last 25 yrs - and only a few keys died.  The replacement keyboard isn't nearly as cheap, but still cheapish compared to other mechanical keyboards. Mass produced, nothing too fancy.

I've never heard of anyone swapping the CPU in a laptop with a different model before the framework. Interesting.

If you want upgrades that are cheap-ish, stay with desktops.  Get a cheap laptop and setup remote access from the laptop to the desktop.  With 2 systems like that, it is still $800 cheaper than 1 high-end laptop. No comparison.
