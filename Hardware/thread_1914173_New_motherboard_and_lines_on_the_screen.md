### Post by grahammechanical on 2012-01-23
Just stop and think for a moment. You had a motherboard with an ATI Radeon x300 video card/adapter. Did you install a proprietary driver for it using Additional drivers?

Now you have changed motherboards and this board has an intel video adapter. Are using the same proprietary ATI driver to work an Intel adapter?

Try using Additional Drivers to de-activate the ATI driver and may be activate an Intel driver. All this is just a guess based upon you not telling us that you have already done this.
