### Post by jimv on 2008-04-30
Kinda odd...I didn't get any updates today in Hardy.  So I don't think it was anything system related...

If it's beeping when you try to boot up, those are POST codes.  One beep means everything is peachy.  More than one beep is an error code.  If you know what kind of bios you have, you can look up what the code means on Google by typing your bios name and "post codes".
