### Post by TheFu on 2017-07-08
If you already have a laptop and don't intend to do much gaming, then why not just run Linux inside a virtual machine?  If you post the config of your current laptop, we could advise whether doing it would work and recommend lighter distros that will work nicely.

When it comes to used laptops, build quality is the most important consideration - that and replacement parts.  Expect delicate and moving parts to wear out.  Screens, hinges, GPUs, keyboards, and fans all tend to break.  And batteries. Getting a replacement battery would be good.

But if you just want to learn Linux, use a virtual machine to start.  If you want to run a small home server, a raspberry-pi v3 can do amazing things provided you don't need network or disk I/O. Cheap ARM systems just aren't built for that.
