### Post by MyR on 2013-04-02
[Linux-compatible network printers]("http://linuxdeal.com/printers.php?connect=lan")

[Linux-compatible wireless printers]("http://linuxdeal.com/printers.php?connect=wifi")

Also, you can [share any printer connected to your Linux system over your network]("http://linuxdeal.com/how-to-share-printer-over-network-lan-or-wireless").  Ubuntu is really great with printer sharing; if the system is set to show shared printers, it comes up automagically.

I don't have one, but I think any network drive you get should have no problem with compatibility.

[ps3mediaserver]("http://www.ps3mediaserver.org/") is what I use to share music with my ps3.  it's pretty easy to use... and it also works with xbox (but we don't like to support that company).

Hope this helps!
