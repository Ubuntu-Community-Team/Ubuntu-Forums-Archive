### Post by Dark_Stang on 2010-12-24
The iPod touch uses a standard spinning hard drive (1.8" I think) where the iPod nano uses flash based storage. This is where the speed difference comes from. I don't think there is anything that can be done about this, unless you want to try to put a solid state drive in your iPod touch. I wouldn't recommend attempting that.
