### Post by SneakPeak on 2011-10-18
Hi,

I recently installed Ubuntu 11.10 on a Lenovo B570 laptop.  When the laptop goes into some sort of power save mode (not sure if it is sleep or hibernate the screen just goes black) moving the mouse or typing on the keyboard or pushing the power button will not wake the machine.  Initially when this happened I had to hold the power button in until the machine switched off and then I had to reboot.  (Not sure if this is relevant but my machine is on a Windows 7 dual boot).  

Anyway I searched the web and the forums for a solution to this problem and have not found one but..... I discovered that if I pushed the Fn F1 key combination on the laptop there was some hard disk activity and the power button started flashing.  I could then push the power button to wake the machine.  It is a bit irritating that the machine doesn't wake on mouse or keyboard activity but at least this way a full reboot is not necessary.

I post this because I hope it is useful to someone else with the same problem.  Of course if anyone has better solutions I am happy to have them.

Cheers

Sneak Peak
