### Post by cascade9 on 2010-08-10
Nope, you dont need AMD64 )x64-64) to address 8GB of RAM. You can also see 8GB with the PAE kernel. I'd just go for 64bit myself,  use 64bit on machines with 2GB ;) 

Recommended number of cores? 2 or higher these days. You wont need 2 cores to run ubuntu, but try finding a single core CPU these days thats not an atom. 

You should be able to run up to the current top of the line nVidia card (GTX480) but that would be pointless for WoW. The recommended GPU for WoW is still amn nViida 7600GT or better, using a GTX4XX on wow is like using a sledgehamer to open peanuts.
