### Post by oldfred on 2017-02-27
I also do not know LVM, and is it also encrypted?
You need to know passphrase to mount it if encrypted.

A standard install does not have the lvm2 & cryptsetup drivers.

Someone posted this which I saved:
 To get Ubuntu to see different encrypted install:
sudo apt-get update && sudo apt-get install lvm2 cryptsetup
sudo modprobe dm-crypt
sudo cryptsetup luksOpen /dev/sdb2 my-crypt

You probably do not want this, and if new install is UEFI and old install BIOS it will not work anyway.
sudo os-prober
sudo update-grub
