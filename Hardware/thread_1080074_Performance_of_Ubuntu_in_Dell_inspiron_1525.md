### Post by happycube on 2009-02-25
Actually, intel integrated graphics work quite well in Ubuntu for desktop work - you can use desktop effects using free drivers unlike the current situation with nVidia and AMD chips.  So in a lot of respects it's the best supported graphic setup.

You should have no problem using desktop effects in most cases, although there may be cases (playing video?) where you'll want to turn them off.  (I don't have a laptop with an x3100, just the older GMA950)

Overall your new laptop should run ubuntu excellently.  Enjoy!
