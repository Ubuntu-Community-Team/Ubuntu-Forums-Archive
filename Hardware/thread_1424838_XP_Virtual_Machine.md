### Post by [Shadow] on 2010-03-08
Hello!

It is possible to install ubuntu as a program in windows xp if you just want to generally mess around with it by sticking the live cd in after you have already booted xp.

You can also use virtualization software like VMware Player or Sun Virtual Box for free.

As far as connecting your computer at work, I havnt tried using projectors or external screens yet but i know that the function/volume buttons on my laptop keyboard have integrated perfectly so far. I have been able to connect to "windows" wifi networks with the internal wireless so you should have no problems with a CAT5/Ethernet.

Hope it Helps :)
