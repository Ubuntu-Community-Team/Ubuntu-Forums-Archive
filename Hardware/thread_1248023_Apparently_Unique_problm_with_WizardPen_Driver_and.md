### Post by Favux on 2009-08-23
Hi GothPanda,

Doesn't make any sense to me.  Why would you get "Suspicious buffer size 511" when your .fdi doesn't have anything like:
```
        <merge key="input.x11_options.zMax" type="string">1023</merge>
```
instead of:
```
        <merge key="input.x11_options.zMax" type="string">511</merge>
```
You might want to see if "xinput --list" tells you anything.

I noticed:
```
linux.sysfs_path = '/sys/devices/pci0000:00/0000:00:13.1/usb6/6-3/6-3:1.0/input/input3/event3'  (string)
```
doesn't match:
```
[ 1073.660277] input: UC-LOGIC Tablet WP8060U as /devices/pci0000:00/0000:00:13.1/usb6/6-2/6-2:1.0/input/input9
```
But I suppose that's because you replugged.  Although wouldn't just the event number change?

Before you reinstall why not give up on the .fdi first and try xorg.conf?  Just comment the .fdi out or remove it and use your old xorg.conf settings.
