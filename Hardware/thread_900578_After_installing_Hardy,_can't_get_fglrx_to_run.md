### Post by markbuntu on 2008-08-26
This is a problem with the upgrade. I got stuck in mesa hell once due to a kernel update that broke fglrx and eventually, apt and a whole bunch of other stuff when I tried to remove the broken fglrx driver with synaptic. But for you, the first thing to try is this:

sudo apt-get remove xserver-xgl


Try this if you want the latest fglrx:

[http://wiki.cchtml.com/index.php/Ubuntu_Hardy_Installation_Guide](http://wiki.cchtml.com/index.php/Ubuntu_Hardy_Installation_Guide)

### Post by markbuntu on 2008-08-27
If you do not have the Catalyst Control Center I suggest you get it from the repos. You can try running it from a terminal:

amdcccle

The Catalyst Control Center lets you set up some fglrx options, detect your monitors, etc. which may help clear up your problems some. If you want to stop the flickering windowed video playback you can set Desktop/Visual Effects to none while watching them and turn it back on after.

You can also try some of the suggestions here to make fglrx run better:

[http://forum.compiz-fusion.org/showthread.php?t=6794](http://forum.compiz-fusion.org/showthread.php?t=6794)
