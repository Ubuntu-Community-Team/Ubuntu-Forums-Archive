### Post by stinger30au on 2008-10-20
there is a few ways around this

you can install the gnome desktop if your think that ubuntu had the printer cleaning stuff

this will tell you how

[http://www.howtogeek.com/howto/ubuntu/install-kde-kubuntu-on-ubuntu/](http://www.howtogeek.com/howto/ubuntu/install-kde-kubuntu-on-ubuntu/)


or you can install turbo print 2 (must be purchased)

[http://www.turboprint.info/printers.html](http://www.turboprint.info/printers.html)

as it supports it


or


you can install virtual box closed source (so long as the printer is a USB) printer and install a copy of windows xp and put the drivers on xp for the canon printer and share the printer on the lan.
if you do this you *MUST* uninstall the printer from ubuntu. you can ot have the printer shared on ubuntu and virtual box at the same time.
another way to do it via virtual box is to setup a shared directory and save all print jobs from ubuntu to the shared directory as a pdf. install a pdf reader like foxit pdf reader in to xp and then print from xp inside virtual box

[http://www.foxitsoftware.com/pdf/rd_intro.php](http://www.foxitsoftware.com/pdf/rd_intro.php)
