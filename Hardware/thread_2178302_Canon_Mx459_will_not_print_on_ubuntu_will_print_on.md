### Post by aikishugyo on 2013-10-03
If you want to use gutenprint, you shoudl download 5.2.9 since I think for reasons unknown Ubuntu only has an old crippled pre-release 5.2.8 version. You will need gutenprint from CVS (search for the webpage) and compile the latest code. You should find the MX450 family, and also the individual instances (MX459) for easy reference (warning, it is a long list of printers).

Alternatively, you can search on this forum how to install the Canon drivers.
