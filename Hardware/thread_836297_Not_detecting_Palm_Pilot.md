### Post by chewearn on 2008-06-22
Right-click on the gnome panel, select "Add to panel..."
Search for "Pilot Applet", click "Add".
A gnome-pilot dialogbox will pops-up.  Follows the instruction given.

Note:
You might need to experiment different settings in the Device Settings page, to find the correct one.
If you already have data in the palm, make sure select "Yes, ..." in the PDA Identification page.
