### Post by egadgeportalcom123 on 2022-10-31
Most programs only use one CPU core. So if you use 4 OC'd cores in your system, the result will be about the same as a single CPU for your system. The only time that 4 cores really matter is if you are using 4 cores intensively. So if you have 4 cores and 16GB of ram, you might be better off with 2x 8GB. Also, quad channel is only useful if you are using 2 or more high end graphics cards (such as 7970s or 680s) or if you are using a high end consumer CPU like the 3930K. So if you want to build a system with 4 high end graphics cards or a 3930K CPU, then a system with quad channel is golden. If neither of those apply to you, dual channel will be just fine.If you want to run quad channel, you need to buy a compatible motherboard and quad channel memory kit. The difference between the two is that quad channel memory has 4 memory modules on one channel, which means you will have 4 slots available. Dual channel has 2 channels, so you will have two slots available. Quad channel is supported on a PCIe 2.0 x8 interface. PCIe 2.0 x8 is a newer version of PCIe 2.0 x16. If you have an older motherboard, you will want to upgrade to a PCIe 2.0 x16. Newer CPU's also have a new feature called Turbo Boost. Turbo Boost is the ability to make a CPU run faster than its normal clock speed. The Quad workstation CPU's are built for quad channel. Older CPUs with the FSB on a PCIe 2.0 x8 interface will not work on a quad channel motherboard.
