### Post by gordintoronto on 2016-12-06
It can be useful to say exactly what version of Linux you are using, such as, Ubuntu 16.04 64-bit.

You probably have an application called Printers or Printing. It should have a button labelled "+" or Add. Then select "network printer". Give it a few seconds, and it should find your printer, highlight it and select OK, Yes or whatever positive option is provided. Keep going and then you can use the printer.
