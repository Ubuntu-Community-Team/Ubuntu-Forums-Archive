### Post by Sergius14 on 2012-03-07
Because several things...

First: It is very desirable and sometimes a requirement to install software over a Live CD (for ex., to install 'Boot Repair' and so on).

Second, because exists a virtual HDD in your RAM and the OS think you have a small drive. Read more about ramdisk.

Third (not completelly sure, I used this long time ago) you can have a File in your guest OS (a Windows file in a NTFS) and use it as a drive/partition in your Live CD Session to store/persist data and never have Ubuntu installed.
