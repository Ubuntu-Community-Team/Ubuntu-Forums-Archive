### Post by mustaghattack on 2009-06-12
I just went through the hardware "compatibility" thread and the [https://wiki.ubuntu.com/LaptopTestingTeam#Canonical%20Supplied%20Laptops](https://wiki.ubuntu.com/LaptopTestingTeam#Canonical%20Supplied%20Laptops) page and I'm quite disappointed. It's full of post with "80% stuff working" or completely out-dated laptops. In my point of view a 100% smooth hardware compatible laptop is something that gives you the same comfort than a mac. 

For instance :
* I want the laptop to resume when I close the lid
* I want my external screen to be **easily** pluggable (no painful Xorg configuration or crippled interface that never works the same after each reboot)
* I don't want to log out to plug an external screen
* I want my TV / HDMI connector to work
* I want the wifi card to work out of the box (detect connections, resume lost connections ...)
* I want the screen resolution and frequency to be optimal
* I want the laptop to be quiet (no noise if it's able to be quiet)
* I want the sound to work out of the box. By sound I mean speakers and integrated microphone.
* I want the webcam / trackpad / keyboard ...

Most of those are already listed on the laptop testing page (hardware testing section), and some others are problems related to the X server.
But I'm insisting on the graphic aspect because to me that's Ubuntu achilles tendon.

Does anyone know if this kind of laptop exist ? Of course pre-2000 laptop don't count ;). I'm just teasing but there should really be a page that lists laptops you can buy at the time you read the page and work with the latest version of ubuntu (not 6.04).

IMO instead of spending time to get a nice distrib like Ubuntu working 80% on every single machine that has been created since 2000, it would be better to focus on a small subset of recent laptops and achieve a true 100% hardware compatibility with the current version of Ubuntu. Ideally this subset would cover a reasonable price range. 

In short, please update the Laptop Testing page with relevant info ;)

Thanks, Bruno

### Post by mustaghattack on 2009-06-12
I've seen Ubuntu pre-installed on Dell laptop. Honestly booting up on a CD / USB stick and following the install wizard isn't such a big deal for me.

What I was complaining about is what you get once the install phase is done. On the Dell laptop (which aren't available anymore with Ubuntu I guess) I heard people complaining about the graphic card using a non-native screen resolution (1024x768 instead of 1280x800, for example) ...

So system76.com looks like a good website, selling pre-installed laptop with well supported hardware, but, have they checked the laptop before shipping it so it supports everything I listed in the first post ?

About OS X, I have to admit that I had problems running the system on a non-apple hardware ... and that's exactly my point ! These people are not trying to build an OS that will work with all the hardware on earth.
When I start a brand new mac and plug in my external screen, the desktop is automatically expanded to the new screen using the right resolution and without shutting all my apps down. I call that working out of the box.
It's not even remotely imaginable to do that with Ubuntu. Apple is able to do it since years ... 
Ok you'll tell me that's because of the graphic card driver and the X server and the gnome session manager or maybe the window manager or ... but not Ubuntu's fault. Fine, so I must conclude that Ubuntu is a great distrib when you're lucky enough to get the perfect hardware combination (that you must guess or digg through tons of boring forums and actually might not even exist !). Ubuntu is also a great distrib that let you (in the 9.04 release), amongst other things :
* choose between a filesystem able to handle up to 50 million files or a filesystem able to handle up to 4 billion files
* adjust the 3D windows effect so you can choose between 1232 effect with an average of 35 settings per effect
* boot up quickly (useful I must say when a suspend doesn't resume or when my graphic card crashes)
* provide a photo manager that performs about 10 times slower than picasa with half the features and twice more bugs
* ...

Honestly, if someone, who doesn't have 1h per day to spend on forums about missing drivers, ask you what kind of laptop she/he should buy, what would you say ? Frankly ? Apple and if that's too expensive a cheap dell with Vista, right ?
