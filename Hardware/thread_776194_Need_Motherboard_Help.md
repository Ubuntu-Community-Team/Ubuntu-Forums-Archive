### Post by DandyDon on 2008-04-30
> **Squint1987 said:**
> I'm looking for a Asus mobo to build a desktop. I don't know what out there is issue free. Has to be an Intel socket.

Try newegg.com and read user reviews. Remember, simpler is better. Don't buy a motherboard with stuff on it you don't need. And compare the motherboard hardware with the hardware compatibility list in the forums above.

Asus and Gigabyte are two good brands that are well made and supported by the manufacturer.

Personally, I'm running an AMD64 processor on an Epox Board at default speeds, with a Nvidia Nforce4 Ultra chipset and an Nvidia 6800 graphics card. I'm running Ubuntu 8.04 with NO issues. I AM using a soundcard, but most boards have RealTec sound chips;(mine does, ALC880) which are supported.

I can't speak to Intel's chipset support for Linux, but Nvidia's is very good. You might want to consider an Asus board with an Nvidia chipset for the Intel Core2 Duo.

Hope this helps
