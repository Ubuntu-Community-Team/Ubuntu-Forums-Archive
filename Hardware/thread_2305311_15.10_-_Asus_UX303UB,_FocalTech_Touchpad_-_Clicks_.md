### Post by lubricus on 2015-12-17
I just got the same machine, installed Mint 17.3 Rosa. Almost everything worked out of the box,  except trackpad scrolling and two-finger right click.

I found the following:

[http://askubuntu.com/questions/609892/touchpad-not-recognized-on-asus-n550jk-tp500l-focaltech](http://askubuntu.com/questions/609892/touchpad-not-recognized-on-asus-n550jk-tp500l-focaltech)

In short, this worked for me:

```

sudo add-apt-repository ppa:hanipouspilot/focaltech-dkms
sudo apt-get update
sudo apt-get install focaltech-dkms
reboot

```
