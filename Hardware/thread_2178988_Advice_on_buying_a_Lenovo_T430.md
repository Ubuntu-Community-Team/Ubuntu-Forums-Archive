I want to buy a new laptop. I'd like to run Ubuntu and Open Source software but for work and study reasons I also need to run Windows. So I'm looking to have a dual boot machine.
I'm finding it a bit bewildering trying to find a machine that will do both easily and not require me to do any tinkering. I'm an average type of user who isn't that skilled when it comes to messing about with IT.

After some research the machine I'm looking at getting is a Lenovo T430. This machine is certified on the following site [http://www.ubuntu.com/certification/hardware/201112-10223/](http://www.ubuntu.com/certification/hardware/201112-10223/). However,  I'm hoping this machine will keep me going for a few years so I'm looking at making the following the changes to the standard Lenovo build, these include:

- Intel i7-3520M processor
- NVIDIA NVS 5400M Graphics with Optimus Technology, 2GB DDR3 Memory

I'm worried that I end up buying the machine and it then doesn't work as a dual boot machine. Can anyone reassure me that the spec I'm thinking of is okay? Also is there anything else I should be taking account of in specing the machine?

### Post by qunungnauraq on 2013-10-06
If you order  T430 with windows 7 pro, a dual boot installation should be pretty strait forward. Before installation I believe you need to  set your bios for discrete graphics mode and set operating system detection to off.

### Post by kdoherty on 2013-10-06
IF you want the most straight forward installation, one with fewest obstacles, then you should consider forgoing the discrete graphics, and stay with the on-board Intel graphics(HD 4000).
The other path to avoid is the Windows 8.  Opt for a version of Windows 7.   The previously mentioned obstacles may be overcome during an install, but can take some persistence and could induce some frustration.
