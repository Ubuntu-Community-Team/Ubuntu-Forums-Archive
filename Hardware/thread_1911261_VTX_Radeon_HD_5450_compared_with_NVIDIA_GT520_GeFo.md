### Post by coffeecat on 2012-01-18
In fact the Radeon HD5450 is quite a good choice, imo. I use one in my main desktop and it works very well with the default open source driver in Natty and Oneiric. Enough 3d acceleration for compiz and Unity and I can play HD videos without stuttering with it.

If you're not a gamer then I think it's fine since the ATI open source driver is more than adequate.

There is a thread somewhere else in the forum where someone is having issues with a setup that includes a Radeon HD5450, but I rather think the problem lies elsewhere. My experience has been good.
