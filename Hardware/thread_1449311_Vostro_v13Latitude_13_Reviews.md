### Post by lobotomy42 on 2010-05-20
It's good.  The only major nuisance I've had has been the touchpad issue here:

[http://en.community.dell.com/support-forums/software-os/f/3525/t/19324786.aspx](http://en.community.dell.com/support-forums/software-os/f/3525/t/19324786.aspx)

This was very annoying at first because I would keep brushing my hand against the touchpad, which would register as a "click" and interrupt me when typing.  However, even that I've been able to get used to - I wrote a script to enable/disable the touchpad; most of the time I used an external mouse anyway.  

The hardware itself is fine - very thin, very light, which I like.  Despite being a low-end Celeron, the machine feels very fast, and I have bunches of programs open at once.  Only when running Windows under virtualization does it become slow. ;) 

Speakers are a bit quiet, and not centered. 

I'm considering upgrading to 9.10, but there've been some bug reports on these forums with that version, so I'm holding off.
