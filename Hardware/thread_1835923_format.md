### Post by nomko on 2011-08-30
> **j d said:**
> how to format hard drive with ubuntu
 
operating system
 
Install Gparted (in terminal type: **sudo apt-get install gparted** and follow the instructions). Gparted will appear under System > Administration> Gparted. When you start Gparted you have to type your password since Gparted only works with root priviliges. At the right topside you can select in a pull-down menu the drive which you want to format. If the drive is still mounted, you migth get a warning to unmount it. On the menubar you can find the command to unmount the drive, don't know under which drop-down menu you can find this but just take a look and you will find this option.
