### Post by gmargo on 2010-06-22
There is no 64-bit netbook .iso, but do not despair.  Just install one of the other 64-bit releases (desktop or alternate or server).  Then install the "ubuntu-netbook" package.  If you have both the desktop and the netbook versions installed (and they will co-exist), then you can select which you want from the login screen.

3GB of memory will be fully utilized by a 32-bit install.  You run into problems when you get to 4GB.
