### Post by Kubunteando on 2009-07-30
microATX you can find many of them.
All the ones from Intel or AMD not having an integrated video card will work with Ubuntu. And they will fulfil your requirements.

With mini-ITX there are also some options, one can be:

[http://www.zotac.com/index.php?option=com_wrapper&Itemid=483](http://www.zotac.com/index.php?option=com_wrapper&Itemid=483)
You will need to install the Nvidia drivers for the graphic card..

With the quads the main issue is the heat dissipation, mainly the TDP for the SFF cases are limited to 65W. But personally I consider it very optimistic.

My system is based in a mini-itx with a mobile processor with 35W of TDP and I still had some over-heating issues. Although I have to admit that the case I used is quite small since it fits in a backpack. But there are 3 fans on it.

If you want a SFF PC with a Quadcore don't underestimate the heating issues. It is the most important thing in SFF PCs. Another point related to this that I found out is that desktop processors have much lower maximum temperatures than mobile processors. That can explain also why mobile processors are much more expensive because nowadays both have almost the same features. My system is based in a Core2Duo mobile CPU (max temperature of 105 Celsius, while equivalent desktop processors only support till 72 Celsius)..
But I am looking to build the next one using a desktop processor because they are cheaper. So I am following if Intel brings some desktop processor with lower TDP that 65W.

Well, now you got some talk from the experience.
