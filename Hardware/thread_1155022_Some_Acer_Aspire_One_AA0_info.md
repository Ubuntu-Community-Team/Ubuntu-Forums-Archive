### Post by Aearenda on 2009-05-10
How to flash the BIOS: [http://macles.blogspot.com/2008/07/flashing-bios.html](http://macles.blogspot.com/2008/07/flashing-bios.html)
Latest BIOS: [http://macles.blogspot.com/2009/04/acer-aspire-one-bios-3310.html](http://macles.blogspot.com/2009/04/acer-aspire-one-bios-3310.html)

macles* is doing good stuff, even if most of it is about Linpus.

In UNR, the assumption is that everything runs maximised, since the screen is so small. The minimise/maximise functions are still available by keystroke or by right-clicking the icon in the window picker. The 'close' button should still be showing to the right of the window name in the window picker.
