### Post by MaindotC on 2009-03-23
Ubuntu will just locate what drivers it can to utilize your motherboard.  A lot of these drivers, called modules, are included by default and are loaded when necessary.  Think of it this way - suppose you boot from the livecd.  Unless it's a really whacked out motherboard the Live CD will still be able to boot and install an operating system on the hard drive, right?  Rest assured you won't have to do any registration or anything like you would with Windoze.
