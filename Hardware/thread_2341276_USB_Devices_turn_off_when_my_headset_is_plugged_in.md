### Post by awsumatt on 2016-10-26
I recently purchased a *HyperX Cloud II headset*. It comes with a singular 3.5mm jack that plugs into an external sound card dongle which provides simulated 7.1 surround sound. If I boot up my computer with the dongle plugged in it cuts power to the keyboard and mouse (the only 2 USB devices I have plugged in). The odd thing is that unplugging them and plugging them back in does not solve this problem, nor does changing the USB port. If I unplug the dongle and plug it in once I've booted up and logged into my desktop the headset, after a few seconds, works fine, until a couple minutes later power is again permanently cut to my mouse and keyboard making it impossible to even try to troubleshoot. This issue does not occur when connected using the 3.5mm jack but one of the main reasons I bought these was for the external sound card which several users reported to have been functioning with zero work on Ubuntu.

Specs:
[LIST]
[*]Headset: HyperX Cloud II
[*]Mouse: Razer Deathadder Chroma
[*]Keyboard: Logitech K120
[*]OS: Ubuntu 16.10
[/LIST]
