### Post by sudodus on 2013-09-19
Welcome to the Ubuntu Forums :-)

Maybe people have been hesitating to give you a review, because it sounds too complicated. Anyway if you have the computer already, the best method is to try without installing anything. (If you consider getting the computer and want to know if it works well with linux, that is another problem.)

A netbook is usually not very powerful, yet quite modern. I think there should not be too big problems with drivers, and it would be worthwhile to download the light-weight Ubuntu flavour Lubuntu or the medium weight Ubuntu flavour Xubuntu. Of course you can also download Ubuntu and Kubuntu, particularly if you like the eye-candy, and you do not use any applications that need fast graphics.

The long time support version 12.04 is the most polished and debugged version. (Lubuntu 12.04 has no LTS.) The version 13.04 is the newest one, but needs to be upgraded to 13.10 during January 2014.

[Make USB boot drives]("https://help.ubuntu.com/community/Installation/FromUSBStick") from the iso(s) and try them. Do not install, only try them 'live' to find out how they work, and what you like :-) You should also decide if you want to install the Ubuntu flavour OS as the only one, or if you want to install it alongside Windows.

And when you know, you can go ahead and install.
