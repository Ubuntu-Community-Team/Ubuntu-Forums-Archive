### Post by TheFu on 2015-02-20
I'd be surprised if any chromebook, netbook, or laptop that had an SDHC reader couldn't boot from it ... provided secure boot were disabled.  It may be necessary to go into legacy boot mode too - I dunno. So really the issue becomes whether the vendor has locked those functions out of the BIOS or not.  

My C720 chromebook can boot from either SDHC or USB, but I'd never suggest it be used for a general use OS without extremely careful thought first. The keyboard is just too odd and missing important keys to be use. I ended up spending $130-ish more to upgrade the chromebook to be useful - getting a Dell would have been less and came with DEL, Home/End/PgUp/PgDn keys.
