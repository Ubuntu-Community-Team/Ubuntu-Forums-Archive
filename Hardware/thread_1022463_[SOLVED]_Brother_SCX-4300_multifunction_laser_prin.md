### Post by hardyn on 2008-12-28
Well i was looking at alot of Bothers, and the SCX-4300 is actually a Samsung; oops!

anyway... i did end up buying it; the copy function is too handy to get just-a-printer and the price was right.

googling around the closed source brother driver works, but is buggy; requires root permissions to work.  There is a hacked driver that fixes this but will not be developed futher.

The splix open source driver works and seems to work pretty well; however it is a printer only driver thus no scanner.

the scanner does work apparently with the closed source driver, with root permissions (or without with the hacked driver) but i didn't explore much.  the 2.00.95 driver downloaded from the samsung website untars with errors; so i don't know if the file is simply not closed correctly or is corrupt.  don't use the scanner much so not a big concern.
