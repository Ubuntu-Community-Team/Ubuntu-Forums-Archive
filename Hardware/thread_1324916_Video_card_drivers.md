### Post by Anarci on 2009-11-13
By full potential I assume you mean the latest nvidia drivers. You have to download these yourself from the nvidia driver website. 
[http://www.nvidia.com/Download/index.aspx?lang=en-us](http://www.nvidia.com/Download/index.aspx?lang=en-us)

There is more info in the community documentation
[https://help.ubuntu.com/community/BinaryDriverHowto/Nvidia](https://help.ubuntu.com/community/BinaryDriverHowto/Nvidia)

Remember these are closed source propriety drivers

### Post by Anarci on 2009-11-13
If ubuntu installed the drivers for you you can check through System->Adminisation->Hardware Drivers. The version number should be listsed there.

Also running nvidia-settings should display your driver version, if you have it installed
