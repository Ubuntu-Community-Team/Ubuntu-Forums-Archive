### Post by Crandom on 2008-10-24
You have set your Xorg.conf file with only 640x400 as a mode. Save it as xorg.conf.bak to back it up, then change the following line:
```
		Modes		"640x480@60"
```
to
```
		Modes		"640x480@60" "800x600@60" "1024x768@60" "1280x1024@60"
```
Adding the resolutions you need need in this format:
```
"<height>x<width>@<refresh_rate>"
```
Save it then press Ctrl + Alt + Backspace to restart X, or if you are in a shell, type startx.

### Post by Yellow Pasque on 2008-10-24
NO, do not use aticonfig/fglrx/Catalyst for a Radeon 9200. Any recent version only works with Radeon 9500 and later; old versions aren't worth running.
