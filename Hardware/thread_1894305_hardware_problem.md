### Post by Mark Phelps on 2011-12-12
[Rhetorical question mode ON]:  Why does everyone think a BIOS update will automatically fix laptop heat problems? [/mode off]

But seriously ... the BIOS on purchased laptops was designed to work hand-in-hand with the OS preloaded on the laptops -- in nearly ALL cases, Windows.  There is no reason to suspect that upgrading the BIOS will suddenly make the laptop hardware compatible with some "other" OS (i.e., Linux).

This is a known bug with recent Linux kernels that causes some PCs to run their CPUs a high rates, causing the PCs to overheat.

The BIOS version on the PC is unrelated to this problem.
