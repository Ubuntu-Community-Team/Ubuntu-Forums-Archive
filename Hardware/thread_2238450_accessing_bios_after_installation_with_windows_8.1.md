### Post by slooksterpsv on 2014-08-08
As soon as you press the power button on the computer, start madly and rapidly tapping the F1 key (usually 3 times works for me, but some systems won't accept it unless you just keep pressing it). Try that to get into the BIOS. You can also press F12 to get into the Boot Menu - I'm wondering if you have EFI, which if you do, sometimes in the Boot Menu (F12) you can select what to boot from - Windows, ubuntu, etc. So try  that.

### Post by oldfred on 2014-08-08
Some also find that a total cold boot. 
Or turn off power, unplug system and hold power switch to drain residual power.
If laptop also remove battery.

Then try the reboot and getting into UEFI/BIOS boot keys.

Always best to keep fast boot off as that setting is to make the time that UEFI is in control or the time you have to try to get into UEFI as short as possible to have system boot very quick.
