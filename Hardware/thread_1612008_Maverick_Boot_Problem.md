Hi All

My Maverick system has crashed; hangs on boot. Using LiveCD I can still see the folders on the hard-drive, but cant copy anything over to my USB drive due to "You are not the owner..." Is there any way to change owner/permissions from LiveCD so as to be able to copy data across?

---

### Post by ajgreeny on 2010-11-02
Just open nautilus as root with ```
gksudo nautilus
``` in the live CD, and you will be able to copy it all to your USB.
