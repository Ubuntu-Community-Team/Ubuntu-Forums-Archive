### Post by DrSurge on 2012-03-07
The hardware in the CF-18 is incapable of driving multiple screens. Remember, this laptop is a low powered passively cooled utility computer, Not a netbook despite it's similar size and features...
Windows XP Pro forces the toughbook to do this but Ubuntu is more friendly to the hardware than this... Just dual boot to XP if you want to run an extended environment.
