### Post by KarT on 2010-12-29
Hello,

I'm trying to get my Dell U2711 to work with my Asus G1—which has a Geforce Go 7700—running Ubuntu 10.10 but I can't seem to use its native resolution (2560x1440).

I can get to 2048x1152 using the VGA connection, but when I switch to DVI-D to try to use 2560x1440, the maximum resolution available becomes 1680x1050.

I've read [this thread]("http://ubuntuforums.org/showthread.php?t=1606704&highlight=u2711"), but it didn't help.

I would assume that my graphics card doesn't support these resolutions but my problem seems to be elsewhere. I can use higher resolutions through VGA and I can connect my laptop to my TV and output fine at 1920x1080 (my TV's resolution).

Any hints on why I'm stuck at 1680x1050 using DVI-D?

Thanks!

### Post by lenooh on 2012-01-04
Just to be sure, are you using a dual link DVI-D cable? With a single link, it will not go up to 2560x1440.

Some info:
[http://en.wikipedia.org/wiki/Digital_Visual_Interface#Connector]("http://en.wikipedia.org/wiki/Digital_Visual_Interface#Connector")
