### Post by coffeecat on 2011-08-18
> **Drovakoy said:**
> I'm quite sure it's the AMD graphics causing the problem here.

Indeed, but not because the graphics are from AMD. AMD is very well supported in many pre 6*** cards. The Radeon HD 6630M GPU in the more expensive Mac Mini is probably too new to be properly supported with the default open source driver. Whether it's supported by the proprietary driver is another matter.

Out of interest, which version of Ubuntu did you try? Anything prior to 11.04 would be a waste of time because of the newness of the GPU, I would guess, but it would be useful to know whether or not 11.04 works. If it was 11.04 you tried, it would be worth trying again in a month or so when Oneiric/11.10 is nearer to final release. Newer version of Ubuntu = newer version of video driver.

My Mac Mini dates back to about 2006, so I'm keeping an eye on what the latest model is capable of in case I need to (am tempted to! :)) replace it.
