### Post by ejn114 on 2010-10-09
My desktop (the machine in question) is dual-booting Ubuntu 10.04 and Windows 7 Professional; both are 64-bit.  I use Ubuntu for almost everything and rarely boot into Windows.

My graphics card is an XFX Radeon HD 5650 which, after a bit of initial setup difficulty, I was able to configure beautifully under both Ubuntu and Windows using the Catalyst driver.

My monitor is an Acer AL2216W with a native resolution of 1680x1050.

Earlier this week, one morning I turned on my desktop and the display resolution was set to 640x480 in Ubuntu.  I checked the monitor settings, and the monitor did not seem to be properly detected.  However, in the Catalyst control center, the name of the monitor was properly displayed and detected.  In both the monitor control and Catalyst, the only resolution option was 640x480.

I tried booting into Windows to see if this was Ubuntu being strange or something else.  Booting into Windows gave the same 640x480 resolution, though the name of the monitor was still detected properly.

Since that failed, I tried plugging in my old graphics card, an Asus Radeon HD 3850 which I knew to still work.  The monitor received no video.  I swapped the cards back so my current card (the 5650) is in, and I'm still getting the 640x480 resolution.

At this point, I can't figure out if my problem is a graphics card issue, monitor issue, a software issue, or a motherboard issue.  So my question is this: how can I tell what my problem is?  When I'm able to figure that out, then I can ask how to solve the problem.

Thanks,

EJN

### Post by e79 on 2010-10-09
Based on the fact that you've try 2 different graphic cards and 2 different OSes, this rules out the software and the graphic card issue IMO. I would be inclined to think it is a monitor issue but could also be a motherboard's.... What I would do is either :

1. Plug any other monitor to your comp (swapping graph cards to make sure if you want) and see if the problem persists.
or
2. Plug your monitor to any other comp and see if you can play with the resolution.

Maybe it will help you ruling out the mobo or the monitor...

Just my $0.02
