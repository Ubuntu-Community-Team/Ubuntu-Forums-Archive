### Post by IcarusR on 2010-03-11
This scanner is not supported by the current stable version of sane.
It is supported by the development version.

You have two options, install the development version of sane or download iscan, which supports it, from here...

[http://www.avasys.jp/lx-bin2/linux_e/spc/DL1.do]("http://www.avasys.jp/lx-bin2/linux_e/spc/DL1.do")
