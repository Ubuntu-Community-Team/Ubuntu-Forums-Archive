### Post by louieb on 2010-01-07
There is no one way to upgrade BIOS.  You'll need to know the motherboard manufacturer  and the model number of the board. Open the case and look around it should be printed somewhere. 

BIOS is usually customized to for a particular motherboard model.  If there is an update available it should be found in the support section of the computer / motherboard makers web site.
