### Post by kagashe on 2015-01-04
I bought Lenovo Notebook G50-45 with Windows 8.1 pre-installed. I installed Ubuntu 14.04 in dual boot without disabling Secure Boot and both Windows and Ubuntu are running well. I updated Windows 8.1.

Since yesterday I find that Windows (Super) key is stuck up (not physically) both on Ubuntu and Windows. If I connect another USB Keyboard and press the Windows key on it the stuck up Windows key gets released.

I can remove the USB Keyboard and work normally using the Laptop Keyboard till I press the Windows key once again.

It seems there is some kind of bug.

Kamalakar

Update on March 13 2015
The above workaround did not work after a few days. I went to Lenovo Service Centre and they replaced the Keyboard under warranty.
