### Post by snowpine on 2010-06-21
Ubuntu Desktop and Ubuntu Netbook are the same operating system; the only difference is the user interface. So, yes, you can use "regular" Ubuntu on a netbook, you don't have to use the Netbook Edition unless you prefer the "launcher" interface.

I have never used an Asus 1001HA but I do have an older model (the 900HA) and can honestly say that Ubuntu is one of the best netbook distros. If you use the Search box at the top right of this page, you can read about other users with specific experience on the 1001HA.

One tip, you might consider the current release, 10.04, rather than 9.10. Using the current release will give you newer applications, better hardware support, and updates through April 2013.
