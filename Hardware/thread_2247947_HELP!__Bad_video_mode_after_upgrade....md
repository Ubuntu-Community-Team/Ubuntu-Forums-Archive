### Post by gwalborn on 2014-10-11
I upgraded a friend's system to Ubuntu 14.04.  He is running an eMachines computer with an nVidia chipset.  Unfortunately, the system set the default video mode to 1280x1024 which seems to be unsupported in his configuration.  When he boots, the system completely boots and then the video crashes.  If he boots into "recovery" and uses "resume boot", the system boots up OK with 1024x768.  Unfortunately we have found no way to make a resolution change permanent, so he has to boot using "recovery" and "resume boot" EVERY TIME.  There has to be a way to fix this.  Any ideas?

Gary Walborn

### Post by Vladlenin5000 on 2014-10-11
The way to fix is installing (reinstalling) the nvidia proprietary drivers.
System settings > Software&Updates and find the rightmost tab, Additional Drivers.
