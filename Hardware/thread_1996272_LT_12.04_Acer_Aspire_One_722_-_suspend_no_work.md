### Post by Toz on 2012-06-13
Based on this: [http://forums.linuxmint.com/viewtopic.php?f=191&t=104875]("http://forums.linuxmint.com/viewtopic.php?f=191&t=104875") it looks like the system may be resuming from suspend, but the screen is turned off.

Can you try this script to see if it helps?
1. Create the hook:
```
sudo leafpad /etc/pm/sleep.d/00blank_screen_fix
```

2. Copy/Paste the following content:
```
#!/bin/sh
#
case "$1" in
   suspend)
   ;;
   resume)
      sleep 5
      vbetool dpms off
      vbetool dpms on
   ;;
   *) exit $NA
   ;;
esac
```

3. Save the file and make it executable:
```
sudo chmod +x /etc/pm/sleep.d/00blank_screen_fix
```

4. Test.
