### Post by sudodus on 2014-03-31
1. I have a USB to SATA & IDE to USB 'cable' which works as a temporary adapter. I don't know if it reliable for continuous use.

2. I have an 'external box' where an IDE disk can be connected to USB. If such a box allows for enough cooling, I think it can last a long time.

But USB 2 is slow, so if you want high speed you should have USB 3 or connect directly from IDE to SATA.

3. Browsing the internet I found several adapters, that are interfaces so you can connect an IDE disk to a SATA connector on the motherboard. I have no own experience of them, but I think they are worth trying since you have many IDE disks.

-o-

An alternative is to use an old computer with several IDE connectors as a file server via the local area network.

### Post by Mark Phelps on 2014-03-31
I've tried all the SATA/IDE adapters that MicroCenter sells and none of them are what I would call reliable.  The worst reliability comes with the boot operation -- which is why I bought them in the first place.

What DOES work, in my machine that has no IDE ports on the motherboard, is using a PCI IDE controller card. That's a LOT more expensive than one of the little SATA/IDE adapters -- think it was around $70 US.  But, it's reliable and works all the time.
