### Post by kurt18947 on 2012-03-31
> **siepo said:**
> The pae 32-bit kernels support more RAM.

Right.  In fact aren't newer 32 bit O.S.s coming with the PAE kernel by default?  I think PAE is considered a bit of a kludge and I've been using 64 bit exclusively for the past couple years. Flash was a pain on 64 bit systems up until recently.  Another reason I can think of to use 32 bit would be hardware support.  Quite a bit of hardware doesn't have 64 bit drivers available.  32 bit drivers should work fine on 64 bit O.S.s but if you want to be safe ................

  I don't know that there's a huge advantage to 64 bit over 32 bit at this point for common apps.  I believe some CPU intensive tasks such as video transcoding perform quite a bit better on 64 bit.
