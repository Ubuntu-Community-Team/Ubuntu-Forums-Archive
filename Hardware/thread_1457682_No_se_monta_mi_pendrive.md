### Post by Carlos C on 2010-04-18
Si no tienes información importante en ese pendrive te recomiendo darle formato nuevamente. Una forma simple es instalar Gparted, de esa forma utilizarás una interfaz gráfica. Para instalarlo puedes usar el comando:
```
sudo aptitude install gparted
```
