### Post by kosumi68 on 2008-09-15
We are working on improving the default behavior of the Xorg synaptics driver, and would like to get a picture of what the default behaviour should be, given all the various systems using it. By default here means what you would consider a pleasant experience when you load a live CD of the latest Ubuntu release, for instance.

Assume that all parameters that can be automatically scaled, such as size, width, mouse speed, etc, are taken care of. Also, multi-finger clicks will be enabled for those devices that have multi-touch capabilities but lack a right or middle button, like the Macbooks. This leaves us with the following parameters to share opinions about:

Tapping
------------------------------------------------------------

* Single-finger tapping - should it be enabled by default?

* Two-finger tapping - should it be enabled by default? Should it emulate a right-click or a middle-click? Does the usefulness of this feature depend on whether one has the right/middle buttons or not?

* Three-finger tapping - should it be enabled by default? Should it emulate middle-click or right-click? Does the usefulness of this feature depend on whether one has the right/middle buttons or not?

* Corner buttons - should these be enabled by default? If yes, which buttons and which corners? Does it matter how many buttons the trackpad has?

* Palm detection - should this be enabled by default?


Scrolling, paradigm 1
------------------------------------------------------------

* Vertical edge scrolling - should this be enabled by default? Does it matter if the trackpad has multi-touch features or not?

* Horizontal edge scrolling - should this be enabled by default? Does it matter if the trackpad has multi-touch features or not?


Scrolling, paradigm 2
------------------------------------------------------------

* Vertical two-finger scrolling - should this be enabled by default, for the trackpads that support it? If yes, should edge scrolling be disabled?

* Horizontal two/three-finger scrolling - should this be enabled by default, for the trackpads that support it? If yes, should edge srolling be disabled?


All comments and suggestions are welcome :-)
