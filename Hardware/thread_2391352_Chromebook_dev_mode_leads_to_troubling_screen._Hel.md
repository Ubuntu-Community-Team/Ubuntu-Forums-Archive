### Post by TheFu on 2018-05-08
You are using crouton.... so others don't need to follow the link.

Crouton is a chroot method, so not really an Ubuntu install, though it might be close enough for many users.

I stopped using crouton a few years ago, because every forced upgrade by google of ChromeOS broke my crouton environment, forcing an update.  There wasn't any way to stop google from forcing the OS upgrades, so for my stability and sanity, I wiped ChromeOS and did all those warranty-voiding things to install real Ubuntu on my chromebook(s).

Video issues are usually fixed by updating the crouton stuff.  Do that first.
