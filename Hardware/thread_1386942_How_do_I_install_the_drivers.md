### Post by ajgreeny on 2010-01-21
If you are talking about **System -> Administration -> Hardware Drivers** and none are available, it is most likely that you don't need any other drivers than those already installed with your system.  That utility is for any proprietary drivers, usually for graphic card or modem, that are not already installed.

If your system works, and if you can get compiz working, there is probably nothing more to do.
