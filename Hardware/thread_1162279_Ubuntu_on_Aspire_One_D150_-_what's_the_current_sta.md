### Post by CDR Services on 2009-06-03
I have an AA1 with the tiny little 8gig ssd drive that came preloaded with windows and less than 2 gig free hd space!!  So I went looking for an alternative I think I have tried 20 different Linux dirivitives.  I used EEEBUNTU for a while it being one of the best builds for the Acer Aspire 1 that is until the release of Ubuntu 904!!  It loads faster and pritty much works out of the box!  True that the built in Mic and the wifi leds do not work properly but that is minor compared to the positives.  I have pretty much everything installed that I need and I still have better than half the hard drive free, the WiFi works way better than with Windows and if I plug in real speakers the sound is quite good! I have found 904 is the best OS by far in this little machine!!

### Post by jagg on 2009-06-07
I have the AA1 D150 (1GB RAM, 160GB harddrive) that came loaded with XP so I partitioned it to install ubuntu 8.10.  Had issues with wifi and the inbuilt microphone and  speakers which needed some tweaking - had to install new driver for the wifi, had to change some of the settings for the speakers... still working out the microphone grrr! (plugging in an external mic/speakers works fine from the get go btw) Everything else worked straight away.  Interesting to note that startup time for ubuntu 8.10 is about the same speed as for XP.. not that its getting used all that much :)  
Great little machine though - cannot recommend it highly enough.  Have taken it on vacation and had to do some work remotely on it and its handled everything no problems; the other benefit is that its just so small/light you can throw it into your bag and take it anywhere.  :D
