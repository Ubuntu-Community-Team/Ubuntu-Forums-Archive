### Post by brucenduane on 2008-10-19
I have a HP PhotoSmart C5200 which works with Ubuntu 7.10 Gutsy, 8.04 Hardy without any problem.  It use USB cable to connect and the scanner works with Xsane - the scanner software.

I connect the USB cable to the computer with the printer already powered up and Ubuntu recognizes the printer and is already configures.  If I start Xsane, it sees the printer and scanner and self configures to use it.  That is true "Plug and Play" with the computer doing all the work.

The C5200 is ~$150 USD and is single sided printer, 100 Sheet capacity.
Will scan a document and save it on FlashCF, SD card, etc as well.

There are printers in the series that will double-side print for about $100 more.  I didn't need that so saved the money. 

I looked at printers/scanners for a year before choosing this one. Good luck!

-Bruce.
