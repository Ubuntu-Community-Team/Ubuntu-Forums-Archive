### Post by gmunioz on 2011-05-07
1- En principio tienes que conectarte por cable para configurarlo, asi es en todos los router wireless. 
Como su IP address es 192.168.1.1, el ordenador debe
tener la IP en ese rango.
2- El User name va en blanco y el password es admin
3- Lo demás es como los otros routers.
4- Una vez configurado el resto, recien te puedes conectar por wi-fi.
5- Colocar nombre de usuario, clave nueva, y habilitar clave para wi-fi son medidas de seguridad necesarias.
6- El manual esta aqui:
 [http://www.linksysbycisco.com/APAC/en/products/WRT120N](http://www.linksysbycisco.com/APAC/en/products/WRT120N)
