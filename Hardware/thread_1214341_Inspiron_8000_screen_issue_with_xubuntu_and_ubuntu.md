I have the same problem on my Dell Inspiron 8000. Use the Fn-F7 key combination to change the display resolution. The first time will make the display smaller. The second time will make it back to the full screen...but not chopped up.

It seems that if you continue hitting the Fn-F7 combination, the screen will toggle between the small size and the full size.
