### Post by sdennie on 2009-08-14
The 4GB is because 32bits is 2^32 which is 4 billion.  In theory, 64bit will allow you to access 2^64 bytes which is about a quintillion bytes (16 exabytes).  More information can be found here: [http://ubuntuforums.org/showthread.php?t=855511](http://ubuntuforums.org/showthread.php?t=855511)
