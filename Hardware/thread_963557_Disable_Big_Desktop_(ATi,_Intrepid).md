I got that to work using the ATi Catalyst Control center and using Big Desktop to the right. I had to disable compiz to get anything do display on it though.

Now every time my computer boots, it extends my desktop off the right side of my screen (i.e. I can run the cursor off the screen and cannot see it anymore.). I can change my resolution to something else and then back to 1280x800 (native laptop LCD res) and it works like it should, with only the built in display. But as soon as I reboot (or restart X) it goes back to the Big Desktop mode!

### Post by ionman on 2008-10-30
Nevermind. I solved my own problem. Just go to System->Preferences->Screen Resolution. Change it there, and it saves the setting.
