### Post by chettyk on 2010-09-26
I need some advice. I have bought a HP Pavilion dv6 laptop with an Intel i3 core processor and Windows 7 Home Premium pre-installed. I am planning to install Ubuntu 10.04 as dual-boot on the laptop (to my delight, 10.04 ran beautifully on the laptop when I tried it out with a live CD).

But before I install Ubuntu, I want to check something. Currently, as soon as the laptop starts up, I get a message: "Press the ESC key for the Startup menu." The Startup menu has options for system information, system diagnostics, boot device options, bios setup and system recovery.

If I install Ubuntu and GRUB, would I still have access to these options? Without access to these options, I will not, for instance, be able to boot from a CD/DVD or memory stick.

I get the impression from this HP technical support page [http://h20239.www2.hp.com/techcenter/startup/index.htm]("http://h20239.www2.hp.com/techcenter/startup/index.htm") that the Startup options are firmware based and therefore should not be affected when GRUB is installed?

Am I right on that? 

Thanks.
