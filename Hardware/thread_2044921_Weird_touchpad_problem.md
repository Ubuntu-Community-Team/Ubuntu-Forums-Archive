### Post by istiska on 2012-08-20
When running standard Ubuntu 12.04 live usb the touchpad doesn't work. The pointer is there in the middle of the screen but it doesn't respond to moving or clicking. 

Went to the mouse&touchpad settings and checked the option to indicate where the pointer is unpon hitting the "Ctrl" key. Hit the Ctrl key and it showed a red circle around the pointer. Does this mean that the touchpad is recognised?

I'm on a Toshiba Satellite L500-1WH with Intel Core i3 64-bit and 4Gb RAM. Tried the unbuntu live usb both in 64-bit and 32-bit with the same result.

The interesting part is that the touchpad works on Ubuntu (with Unity), and on Xubuntu in virtual box on Windows 7 AND when booting using a Lubuntu live usb. Seems like it's a Unity problem...?

My intention is to dual-boot Ubuntu alongside Win7 to use Ubuntu as my main OS, leaving Win7 for need-to-use situations. But I'm kinda wary of going through the install process without a working touchpad.

### Post by istiska on 2012-09-06
UPDATE:

I eventually installed Xubuntu. The touchpad worked but the brightness level could not be changed either from the keyboard or from the applet icon.  Eventually I came across how to change the brightness from the terminal as explained here:

[http://ubuntuforums.org/showthread.php?t=2029629#7]("http://ubuntuforums.org/showthread.php?t=2029629#7")

But I never got the keyboard hotkey combo (Fn + F6/F7 on my laptop) to work.

Eventually I removed Xubuntu and installed Linux Mint Maya 13 (Cinnamon DE) and both touchpad and brightness worked out of the box.
