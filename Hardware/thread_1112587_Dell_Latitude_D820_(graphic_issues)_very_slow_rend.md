### Post by hashable on 2009-04-01
Solved.

The issue was that I was running it on an external "vertical" monitor. Apparently xorg has some 2D drawing issues when display is rotated. I changed to regular horizontal display mode and it worked fine. My xorg currently reads:
```

Section "Device"
        Identifier      "intel"
        Driver          "intel"
        Option "AccelMethod" "XAA"
        Option "XAANoOffscreenPixmaps" "true"
EndSection

Section "Monitor"
        Identifier      "Configured Monitor"
EndSection

Section "Screen"
        Identifier      "Default Screen"
        Monitor         "Configured Monitor"
        Device          "Configured Video Device"
EndSection

```
