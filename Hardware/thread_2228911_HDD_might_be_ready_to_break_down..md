### Post by Vladlenin5000 on 2014-06-10
Hi, welcome.

Almost all HDD manufacturers provide OS independent tools, usually an ISO for making a bootable CD, that often are more reliable than anything you can run from within an OS session. It won't hurt trying it just to be sure. Of course it might fail as well if your suspects about the motherboard's controller. If so, test the drive using the same method but in a different computer, if available. 

I'm almost 100% sure it is indeed an hardware problem.
