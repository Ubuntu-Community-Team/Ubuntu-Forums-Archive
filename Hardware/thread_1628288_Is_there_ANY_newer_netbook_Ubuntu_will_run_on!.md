### Post by Acibeb on 2010-11-22
Now, there is a whole edition dedicated to netbooks - but what netbooks? 

Virtually all newer netbooks come with Intel Atom N-type processor and therefore GMA graphics chip, which Ubuntu does not, and judging by the progress, never will support to the full extent. 

For me this means, shitty video playback, no (or awful) full-screen video (with or without Poulsbo support), no web-cam, no favourite apps, no microphone, no suspend-resume, usb memory mounting problems, card reader problems...

So can someone please tell me, if I went to a store today, to buy a brand new netbook, will I find at least one that Ubuntu is fully compatible with?
