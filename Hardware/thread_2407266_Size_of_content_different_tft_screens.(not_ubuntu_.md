### Post by CatKiller on 2018-12-01
A larger screen is physically bigger.

The resolution is how many pixels there are in each direction.

A larger screen with the same resolution will simply have bigger pixels.

Historically, text and icons and the like were simply bitmapped images: the icon was 16×16 pixels, say, and that letter x was exactly 5 pixels high, say, always on every display.

That meant that, yes, bigger pixels made everything bigger and smaller pixels made everything smaller.

That's really limiting, though. As displays have been able to be made with smaller and smaller pixels - which makes things like text, especially, much less blocky-looking - the size of things on-screen has been decoupled from the size of the pixels. You can set a scaling factor for your display that will let you have tiny pixels, but still have the overall size of things comfortable.

I hope that clears things up a bit.
