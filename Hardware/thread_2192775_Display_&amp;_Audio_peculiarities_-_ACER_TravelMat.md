### Post by ipb on 2013-12-09
Folks,

I have rescued an ACER TravelMate 3004 WTMi laptop by installing Ubuntu 12.04 LTS on it - _most_ things work well.  However, there are two hardware issues.

Hardware problems: 

a)  Video controller performance; and

b)  Sound controller not working.

VIDEO:
The video controller is peculiar.  It works ok with my external display at 1024x768 which is acceptable BUT it does not work properly when the lid is shut, switching on and off every 8-10 seconds.  I have tried the various power management settings available from the standard settings menu (aka control panel) but these appear to make no differennce.

The display does the same thing if operated through the exDock Lite port expander which otherwise seems to work well.

The curiosity is this switching off and on - obviously it makes the machine unusable when the lid is closed.  With the lid propped open the machine fan turns on more frequently, I supposed because air-flow is disripted through the keyboard.

I should be grateful if anyone has any experience of this and can offer a fix or, alternatively, suggest any work-arounds.  I should like to operate this laptop with the lid closed when in the office.

AUDIO:
The OS does not seem to recognise the built in Intel HD Audio controller device.  Which is odd because it is reported.  Standard programs therefore have no sound.  However, when audio is played out via an external device, in my case a bluetooth speaker, the sound comes out in two-second bursts followed by 3-4 seconds of silence.  This is odd and makes teh sound useless.

_However_ when running a Win98 program under Wine, which looks for the audio hardware (in this case an SDR decoder) the sound is audible, albeit at a rather low volume, without any interruptions.

Prior to the rescue, the machine ran as well as can be expected on Win XP Pro and all hardware was working, save that the machine was pathetically slow and the WiFi was restricted to US channel plans.  Now running Ubuntu 12.04 LTS and with 2GB RAM, the machine is in all other respects running extremely well.

Grateful for thoughts and suggestions.

Best Regards

Ian Beeby
