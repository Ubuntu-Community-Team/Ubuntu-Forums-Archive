### Post by SeijiSensei on 2018-09-04
I have a color Brother HL3170CDW.  It has a built-in network server and wifi which I don't use.  I connected it to the router, gave it a static IP address, and installed the relevant driver in Ubuntu.

It's now down to $175 at [Amazon](https://www.amazon.com/Brother-HL-3170CDW-Wireless-Networking-Replenishment/dp/B00BQU141C).  The [monochrome]("https://www.amazon.com/Brother-Monochrome-HL-L5100DN-Two-Sided-Replenishment/dp/B01BBL2TRS") printer there is only $30 cheaper, though of course you only have one toner to replace, not four.  I've used [off-brand toners]("https://www.amazon.com/gp/product/B07BDGXDDF/") for this machine with no issues.
