### Post by Leppie on 2010-02-01
check your /etc/network/interfaces**, **it should only have the lines "lo" and "auto lo". if it's already like that, check the /etc/NetworkManager/nm-system-settings.conf as well and change the following line like this:
```
managed=true
```
then restart network services:
```
sudo services networking restart
```

### Post by SedaliaSteve on 2010-05-16
I had the same problem with Xubuntu 9.10. It just showed device not ready.

I went to [http://linuxwireless.org/en/users/Drivers/b43]("http://linuxwireless.org/en/users/Drivers/b43") and downloaded legacy drivers. I then went to System -> Hardware Drivers and activated the Broadcom B43legacy wireless driver and rebooted.

It saw the wifi as soon as it came up.
