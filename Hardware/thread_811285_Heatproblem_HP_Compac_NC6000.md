### Post by cdtech on 2008-05-29
First, you'll want to run top and see what is running your proc at 100%.

I've seen tracker (search and indexing service) do this. I've had it heat my laptop up pretty good and run my cpu load to max. Top wont show tracker, you can shut it down to see if its the cause, go to system > preferences > sessions and uncheck it.

Hope this helps......
