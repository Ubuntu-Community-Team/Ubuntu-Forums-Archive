### Post by markmckee601 on 2009-06-18
If you can, boot from a live cd or usb flash drive and see if you get the same problem.
If you do get the same problem then you know it's a hardware problem and not software related (keyboard layout etc..)

I came across this site but i think the instructions are a bit too much.
[http://www.computing.net/howtos/show/cleaning-laptop-keyboard/12.html](http://www.computing.net/howtos/show/cleaning-laptop-keyboard/12.html)

I find that the best way to clean a keyboard is to remove each key and clean it with a baby wipe which are alcohol based or you can clean each key with isopropyl alcohol and a soft cloth. A word of caution if you choose to remove any of the keys - the keys pop off and you can usually use a small screwdriver to pry the key up from one of the corners at the bottom.
Also make sure to clean underneath the keys.
