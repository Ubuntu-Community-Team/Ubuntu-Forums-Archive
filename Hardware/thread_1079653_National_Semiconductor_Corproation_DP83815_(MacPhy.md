RE: National Semiconductor Corproation DP83815 (MacPhyter) chip

I was having trouble with this chip on a computer someone brought in to me today.

The problem was it was actually disabled in bios, but was still being detected but unable to be used.  So it showed up, but with no address (couldn't get one via dhcp).  As soon as I enabled it in bios it worked immediately.

Not sure if this is a common problem but I figured I should post it somewhere in case anyone else had the same problem.
