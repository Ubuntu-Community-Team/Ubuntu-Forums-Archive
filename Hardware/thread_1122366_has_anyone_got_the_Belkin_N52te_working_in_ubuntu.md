### Post by second.exodous on 2009-04-12
I don't know, back when I had the n52 I used Gentoo so there was probably an e-build, I only recently started using Ubuntu.

Did you search with the package manager for the package?  If it is in the Ubuntu repositories then that is the best way to get it installed because it takes care of all the dependencies automatically.

[Here](https://help.ubuntu.com/8.10/add-applications/C/gnome-app-install.html) is how to do it.  I would first try to search for n52te and see what it comes up with.  Just like a search engine you'll get a lot of things that aren't what you're looking for.  If you don't get what you need then do n52 and see what happens.

At first this all seems more difficult than the OS you were using before, be it windows, osx, or whatever, but it's just a different way to do something.  After a while you'll get used to it.  In fact I like going to one place first to find software/packages then look other places.  This is probably the second to the main reason I use Linux, it's easy to find things.

I hope this helps, if it doesn't just ask here again and I'll try to help you more.  If you own the n52te and get it to work then let me know and I'll buy one, I miss my n52.
