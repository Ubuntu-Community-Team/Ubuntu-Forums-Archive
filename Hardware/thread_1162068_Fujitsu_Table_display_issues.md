### Post by Enter t'Ibex on 2009-06-28
just for documentation sake.

I got the LCD display, tablet and pen working in openSuse11.0 tonight.

There are two problems.  First you need to use an older distribution like 8.04 (maybe 8.10) or opensuse 11.0 since the newer dists do not support the 830MG intel video hardware correctly. opensuse11.2 and 9.04 will not work (IMHO)

The second problem with the tablet functions seems to be with the driver load order in xorg.conf; I found that by moving the input driver functions BELOW the video driver load function, allowed all parts to start working correctly.

As well the network install in opensuse worked better for me with out having a drive to load from.

The platform I am using is a Fujitsu st4120.  Things that dont work: side buttons, xournal (on the second page) will work on that now.
I will post my before and after xorg.conf files within a week or so as Im out of town now.
Cheers and thanks to all
