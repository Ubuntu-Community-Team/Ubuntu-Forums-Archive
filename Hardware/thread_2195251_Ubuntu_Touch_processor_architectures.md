### Post by grahammechanical on 2013-12-23
First, there is a section for Mobile specific discussions and the available information has been posted in that section before. 

Ubuntu Touch is officially being developed on four devices; Galaxy Nexus; Nexus 4; Nexus 7; Nexus 10. and that does not include the new Nexus devices. Only those that were available about a year ago.

Community developers are working on porting Ubuntu Touch on other devices. How successful they are is anybody's guess. Officially, Ubuntu Touch is at version 1.0 and is successfully optimised for phones. Over the next six months Ubuntu Touch will be optimised for tablet devices as it is brought up to version 1.5. Following that the Touch code base will be converged with the desktop code base so that there is one Ubuntu code base.

[https://wiki.ubuntu.com/Touch/Devices](https://wiki.ubuntu.com/Touch/Devices)

The intention of Canonical from the start was for Ubuntu Touch to come pre-installed on mobile devices. We have just heard that Canonical has signed an agreement with an OEM for the OEM to bring a Ubuntu phone to market. Ubuntu Touch tablets may appear sometime in the new year. It is questionable that we will be able to install Ubuntu Touch on mobile devices in the way we install Ubuntu on desktops and laptops. I doubt very much that Canonical will take up that challenge. But Ubuntu is open source and community developers may attempt it but there comes the risk with these kind of things of causing the device to become unusable.

Do you not think that if Canonical meant Cortex A series in general, then they would have said so? They say "Cortex A15" and that must be what they mean. Yes?
