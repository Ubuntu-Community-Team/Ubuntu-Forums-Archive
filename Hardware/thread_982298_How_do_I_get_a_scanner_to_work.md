### Post by ajgreeny on 2008-11-14
Do you have xsane installed?  If not get it in synaptic or Add/Remove Applications, start it from your **Applications->Graphics** menu, and if it is supported it should find the scanner.  There are a number of scanjets that are supported and I suspect your should also be, so give it a try.
