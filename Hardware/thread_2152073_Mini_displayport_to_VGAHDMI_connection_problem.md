### Post by cephalopods on 2013-06-06
I have a Thinkpad T530 running 13.04, with the integrated Intel 4000 for graphics. My goal is to be able to connect two external monitors in a usable configuration. I have a 24 inch Insignia HDTV which accepts HDMI and VGA input (among others). If I connect directly to the VGA port on my Thinkpad, the monitor behaves perfectly. However, I will need the direct VGA connection for the second monitor (an actual monitor, with only VGA in), so I've been trying to connect via the mini displayport. I have new mini displayport to HDMI and mini displayport to VGA converters. When I use either to connect to the TV (no other external monitors attached), it is correctly detected and an image is displayed, but the response has a lot of lag. There is a noticeable delay typing into a terminal, and if I move the mouse in a circle, I see a circle of mouse cursors on the screen (as if I had turned on cursor persistence - I see a trail of cursors where the mouse was). This does not happen on the main laptop display, or on the TV when connected directly with VGA, but it does happen using either converter on the mini displayport, so it seems to me that the problem is in the mini displayport connection itself.

Any insight into what might be causing this would be appreciated.
