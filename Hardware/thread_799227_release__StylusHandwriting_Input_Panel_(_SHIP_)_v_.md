### Post by bcw on 2008-05-18
Available for download at:
[https://sourceforge.net/projects/ship-project/]("https://sourceforge.net/projects/ship-project/")

The Stylus/Handwriting Input Panel (SHIP) is a system for gesture text entry for Tablet computers using an X11 User Interface.

Text may be entered from an on-screen keyboard or by handwriting &#8211; either printed or cursive handwriting - but this requires a server application (included) installed on a copy of Windows Tablet XP or Vista.

The provided "SHIP User Manual" explains the installation and use of the system.

This is Free Software. License details are included for each component.

I hope you find this software useful.
