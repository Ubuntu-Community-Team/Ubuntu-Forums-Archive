### Post by lukeiamyourfather on 2009-10-27
Ubuntu has a proprietary drivers tool that will install third party drivers like ATI video drivers automatically, and using packages so they can be easily uninstalled later. The tool is in System > Administration > Hardware Drivers. If it doesn't show up there you might have to install the proprietary driver yourself or use an open source driver if one is available. Cheers!
