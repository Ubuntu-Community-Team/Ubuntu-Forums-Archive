### Post by Mark Phelps on 2011-12-12
Yes ... there is a known bug with recent Linux kernels that causes the CPU to run at high rates, causing the PC to overheat.  This bug is still being worked.  

BTW, as far as I know, there is no "BIOS upgrade" that will fix this (before you ask).
