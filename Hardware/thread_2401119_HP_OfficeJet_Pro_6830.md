Hello

I have been given the subject printer as a 'gift'. It is a little over 3 years old. I have updated the printer using the onboard web services update function. The printer cartridges in it were start up versions. I have replaced all four cartridges with new full ones. 

It shows an error message on start up which says something is wrong with the printhead.I have used the onboard 'clean printhead' function. It ran this and then declared the same error message. When I tried to run clean printhead again nothing happened.

I suspect than it may have dried ink in it.

Is it possible to clean the printhead in a manual way without using the onboard clean printhead function?

### Post by ajgreeny on 2018-09-13
Not very easy, I suspect, as that machine uses cartridges that do not have the inkjet integrated; the jets are in a separate printhead. My HP has cartridges that replace the inkjet with each new cartridge; more expensive to buy, but I always use replacements from a compatibles manufacturer which reduces costs to a sensible level.

You may have some luck with bringing the head to the middle as if changing cartridge, then removing the power so it stays there.  Now see if you can get a wet lint free cloth beneath the printhead to soften any hardened ink and let it soak for some time.  You may find that the clean process works better, though it may take several attempts to clear it properly.

It is likely to be an uneconomic repair should the printhead need replacing unless you can do it yourself with a replacement that comes from another faulty machine you can find (different fault, of course) as printheads seem to cost as much as a new printer in my experience.

### Post by Autodave on 2018-09-13
As ajgreey suggested: lots of warm water soaking the head for a long time.  Soak it, use paper towels to draw out the ink. Repeat and repeat and repeat.
