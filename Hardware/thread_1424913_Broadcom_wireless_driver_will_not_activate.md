so I decided to remove windows for a while from my netbook, to give ubuntu a fair shot. I have been using a live USB of ubuntu for a while, and was happy with it. the live usb had no issues finding and activating the driver needed for wireless on this netbook, and everything went without a hitch.
so I installed ubuntu. I went to System>administration>hardware drivers as usual, and it detected the required driver. this time however, after I entered my password for it to install, it did nothing. I figured, "ok, I'll plug it into the network via a wire". I got the same result, though I have a net connection now.

as this is a netbook laptop, wireless is essential for me, any insite on how to fix this is much appreciated.
