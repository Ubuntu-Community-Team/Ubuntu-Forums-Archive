### Post by sakishrist on 2011-06-17
I am not very familiar with that technology but Ubuntu automatically "Underclocks" my nVidia when there is no need for it. So if I use my laptop for browsing the Internet, the graphics clock remains at 169 Mhz, but when I start a game it jumps to 500 Mhz.

There are also stages in between those two I described so my guess is that if you are on battery it will try to balance things a bit.

I also notice that there is a setting that allows you to chose if you want max performance or to be adaptive.

Anyway, all that is about my laptop which has an nVidia GeForce 9600M GT and ... my battery is long gone so I can not really test to see how the laptop reacts when on battery only.

EDIT: I am also not sure about the rest so ... I'll stay out of it. ;)
