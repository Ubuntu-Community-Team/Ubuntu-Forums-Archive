### Post by kurt18947 on 2013-02-04
Some hardware that requires 3rd party drivers in Windows - especially XP and earlier - 'just work' in linux.  Support for quite a lot of system hardware is 'baked into' the kernel.  But unless someone is using the same or very similar hardware and participates in this forum,  it might be wise to check into return policies.   I've bought some PCI USB cards that were just a matter of plugging in.  If you can determine the chip set used in the device you're interested in, you might have better luck determining compatibility.  Several manufacturers might use the same chip set.  For instance, PCI USB expansion cards that use NEC chip sets generally work very well.
