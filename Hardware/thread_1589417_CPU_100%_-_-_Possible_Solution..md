### Post by PCTinker on 2010-10-06
My CPU has been running flat out 10 minutes or so after launching Ubuntu, with the fan roaring away to dissipate the heat. This is even without a single app running.

My 'fix' was to add the CPU SCALING MONITOR to one of the toolbars. When added it showed my CPU set to PERFORMANCE. I changed that to ON-DEMAND and everything now runs cooler, and oh so much quieter.

As I type this I have Chromium browser open with 5 tabs, Ubuntu One syncing some files and music playing. My CPU is at 52% with no fans running at all.

Hope that helps somebody.
