### Post by Favux on 2009-06-11
Hi Toy Machine,

Your info. says you are using Gutsy.  Starting with kernel 2.6.26 and newer the Linux UVC driver was included in the kernel.  But Gutsy's kernel is earlier (I don't remember the number) so you'll have to install the UVC driver.  Info. about it is here:  [http://linux-uvc.berlios.de/](http://linux-uvc.berlios.de/)

Then you can install Cheese or webcam or webcamd with Synaptics to test it out.
