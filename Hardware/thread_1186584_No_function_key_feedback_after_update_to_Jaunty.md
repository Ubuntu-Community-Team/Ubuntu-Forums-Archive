### Post by pwaring on 2009-06-13
I've recently upgraded my Thinkpad to Ubuntu Jaunty from Intreprid. Almost everything works fine, and in many cases better than before, but one thing I have noticed is that the function keys (Fn + F3, F4 etc.) no longer provide any visual feedback on the screen. In previous versions of Ubuntu, the current volume would appear in the middle of the screen when I changed it using the function keys - same goes for anything else on a scale such as brightness and contrast. In Jaunty, the function keys still work, but I no longer get the visual feedback in the middle of the screen, which is a bit annoying because it means I can't see what the volume and brightness settings are when I'm adjusting them.

Is this a configuration option which used to be enabled by default but was disabled in Jaunty, or have this feature been removed entirely? I'm not sure whether to report this as a bug or not, as I don't expect things to stop working when I upgrade, but it might have been a deliberate design decision for all I know.
