I believe most people who are having the fan issues with the Inspiron 3521 are the ones with the i5 processor which is equipped with the ATI Radeon graphics cards. I have the Pentium processor which--like the i3-- uses the integrated graphics. I haven't had any issues with 12.04 with fan issues. It fan runs quiet (if not silent/off) when it is idling or when web browsing. Video content such as YouTube is running moderately, sometimes it is running at full blast when it gets too warm but shuts off when it is cool again. Most BIOS drivers are Windows only, but if you can get your hands on it then try to update to the latest BIOS. Something that should have been done as soon as you switched to 12.04

### Post by forestG on 2014-07-18
Although this is a very old thread and possibly most people have addressed the issue, the solution that worked for me is to do a bios update. Unfortunately to do that you either have to install windows (at least for the bios update) or to make a bootable freedos usb stick. I used windows, updated the BIOS and now it is totally silent with ubuntu 14.04.
