### Post by viper250 on 2010-02-21
you need to find the info for your mobo and your post card and find out what the error code means.
if the post card is showing any error code you have a hardware issue 
It may be a failed component or a hardware conflict.
if all hardware is working correctly you will hear a single beep from the bios
 not having a display is pointing to a video card issue. 
the power led on most monitors will usually indicates if it is receiving any signal from the video interface by changing color.
ubuntu detects hardware easily and usually has no problems with video
also you can go to the ami. website and research the post error code. you just need to have the bios revision number
this will tell you exactly what the error is
