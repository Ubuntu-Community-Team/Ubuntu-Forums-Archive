### Post by tgalati4 on 2010-12-31
If the laptop has a poor cooling design, then I doubt that Ubuntu would be any better.  If you installed Ubuntu, you would have to modify CPU speed settings (on-demand vs performance) using the cpufreq panel add-on.  You can also undervolt the CPU but that gets into some advanced techniques.

The graphics chip also creates a lot of heat.  Sometimes proprietary drivers help, but often, high-performance graphics chips run hot regardless.  The splash screen problem may be related to KMS--kernel mode setting--setting the graphics resolution during boot.  You would have to search the forums for settings to modify.  Only you can judge if the proprietary drivers work better for the things that you use the laptop for.  For playing games, you often need the proprietary drivers to get decent 3D framerate performance.

In the meantime, get a USB laptop cooler.  The are designed to make your laptop thicker and heavier, with less battery life.  Of course if the computer maker designed it that way in the first place, it would run cool, but then it wouldn't look cool.  I'm going to guess that this laptop is glossy and bright colors and contains the letters "H" and "P".
