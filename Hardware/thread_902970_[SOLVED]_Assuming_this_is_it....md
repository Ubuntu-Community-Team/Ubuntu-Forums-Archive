### Post by ilrudie on 2008-08-28
Its pretty accurate I think but scaling may not be the best measure of how taxed your CPU is.  It may scale up even when cpu usage is low to meet demands from lots of processes quickly.  Mine does.  I also have the system monitor gadget (or whatever you call them) on my bar so I can see when the proc is really getting worked.

### Post by falcon61102 on 2008-08-28
Yeah, as ilrudie says, your CPU scale guage is just to let you know where the CPU is currently set.  As you know, when not being used the CPU is scaled down but it will be raised back to normal in anticipation for usage even if it's not currently being used to the fullest capacity.  Keep an eye on you System Manager or you can install some other aps that will more accurately show the actual CPU Usage.
