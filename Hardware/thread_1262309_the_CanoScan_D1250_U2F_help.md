### Post by ronparent on 2009-09-09
I know it all too well. Great scanner for situations where windows driver are available. I spent two hour yesterday researching how I would get it back in service. What I found? Appear to have unique driver not in common with most other canon scanners. Does not have linux drivers. Does not have 64bit drivers. Is unsupported by Vuescan which otherwise supports virtually everyscanner ever made in both windows and linux (in this case vuescan requires a functioning windows driver). I plan on using mine as a doorstop unless canon actually releases a 64bit driver for wind 7.
