### Post by gideond on 2011-04-04
I got Ubuntu 10.10 setup on my HP DV4-1281us about 2 weeks ago. It's been running good with one issue. During normal operation the battery seems to discharge at about the same rate that Vista did. However, once I shut the laptop down, complete shutdown, the battery continues to discharge. I get as much as a 20% loss overnight. This is with a relatively new 12-cell battery. Vista never had any such issues. I've read a little about some repeated kernel issues for the past several versions of Ubuntu that have shown the same behavior, but nothing concrete. It's a real pain. Anyone know of any fixes for this? The only thing I can think is that it is letting power remain active to WLAN, LAN, and USB. I can shut off the wireless manually before shutdown and it seems to help some but not much. I have a touch button for the WLAN and it automatically comes back on when the computer next starts so it may not be doing much to turn it off manually anyway. Outside of removing the battery every time I shut down I'm not sure what else to do. 

Thanks.
