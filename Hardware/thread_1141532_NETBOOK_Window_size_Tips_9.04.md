### Post by dpblank on 2009-04-28
Since I use my netbook mostly just for surfing the web, F11 is my friend for making Firefox full-screen. If you let your cursor hover near the top of your screen, a panel with some basic navigation tools should automatically show/hide itself.

Not sure what other applications you are using, but Alt+F10 is the default shortcut for maximizing current window.

See below for a list of more keyboard shortcuts:
[https://help.ubuntu.com/community/KeyboardShortcuts](https://help.ubuntu.com/community/KeyboardShortcuts)
