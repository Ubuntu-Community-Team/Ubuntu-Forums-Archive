### Post by oldfred on 2015-09-29
It is my understanding a ppa is just another source repository. Then apt-get finds the newest version of software and installs from whatever source it finds.

Probably bests not to have two ppa for same software, but in effect you do when you add a ppa for new nVidia and Ubuntu repository still has many older versions.

Ubuntu ppa is new so that may be why ZaReason suggested not to use it. But when the ppa they suggest now says to use the Ubuntu one, I would go with the suggestion from the ppa.
