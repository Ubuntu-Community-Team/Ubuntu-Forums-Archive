### Post by saint-takeshi on 2008-10-19
> **varnil said:**
> Better yet could someone go through and tell me the parts that are safe to delete



depending what version of ubuntu you're using, you can pretty much delete all of it, certainly with intrepid (the beta, anyway) it's all automagical, and you can boot without even having a xorg.conf.

might even be worth deleting it and letting X autogenerate one

```
 # sudo rm /etc/X11/xorg.conf
```

then ctrl+alt+bspace (kill X) log back in and you should either have

a) the result you wanted in the first place, everything working, etc :guitar:

or

b) a nice, clean xorg.conf without all the craziness in it :popcorn:

or, possibly, if you're using pre 7.04;

c) a lovely "X is fail" screen :confused: 

in which case simply 

```
sudo dpkg-reconfigure --phigh xserver-xorg
``` and follow the sexy, sexy prompts to fix it.
