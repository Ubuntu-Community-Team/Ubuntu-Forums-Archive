### Post by Polaris96 on 2009-06-22
1. Go to an internet Cafe with fast machines.  Bring a Blank CD or DVD Download a live install cd. Burn it.  voila.

2.  No fast connections?  Download Damned Small Linux ([http://www.damnsmalllinux.org/download.html](http://www.damnsmalllinux.org/download.html)).  install it.  Get online.  Download the live installer 

3.  Don't like DSL?  Go to Debian.org.  Download the web installer.  burn it.  Hook up your laptop and install debian.  Use Debian to download ubuntu liveinstall cd.

4. Go to internet cafe.  Get on ubuntu website.  Buy CD from canonical or some such.  ship it next day. install.
