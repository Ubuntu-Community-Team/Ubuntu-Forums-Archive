Unless, you use an 'interpreter' software to convert the joystick output to keys. Steam does that already. It isn't restricted to the Steam controller, and they don't actually sell it any more, anyway. Steam has a framework (called Steam Input) for devs to support; Steam then translates the input from any controller supported on Linux (which is quite a lot) into Steam Input, which the games understand. With a controller connected you can go into Big Picture Mode to configure them exactly as you want, including setting the controller to be keypresses and mouse for games that *don't* support a controller.
