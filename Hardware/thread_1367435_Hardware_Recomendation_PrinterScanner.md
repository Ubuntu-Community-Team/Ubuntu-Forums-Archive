### Post by ajgreeny on 2009-12-29
Most of the HP machines are great.  Install hplip and hplip-gui and you should be ready to go.
Some Brother MFCs are great as well, though the scanner drivers need to be downloaded from Brother direct for some of them.
Many Epson work well, also, but I have always had blockage problems with Epson printers, which have permanent ink-jet heads, unlike the HP machines.

Overall, I would go with HP.

### Post by jml on 2009-12-29
I like HP printers as well.  One caution, if you are getting a multifunction printer/copier/scanner.  Linux sees those devices as two separate devices.  The printer is usually not a problem, but some scanners can be a bit of a problem getting the needed drivers.  Here is a link to a database that should be of use.

[http://www.linuxprinting.org/show_printer.cgi?recnum=Epson-Stylus_CX6400](http://www.linuxprinting.org/show_printer.cgi?recnum=Epson-Stylus_CX6400)

This database is a great resource because it lists many devices and describes real world experience with them.  It list both printing and scanning functionality where appropriate as well.  It kept me from making a costly mistake when I was considering the purchase of a new multifunction device.
