### Post by Gramps on 2009-03-19
It will help if you say what version of Ubuntu you used for your headless box.

You will also need to go to System/Admininstration/Printing then Server/Setting and Check Publish printers connected to this system.

This works for other Linux systems, not sure if it will be the same When connecting to Windows. I think you have to install the printer in Winodws to print to a network printer.
