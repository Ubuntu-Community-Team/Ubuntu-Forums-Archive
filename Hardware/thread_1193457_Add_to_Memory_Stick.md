### Post by ASJboarder on 2009-06-22
I wouldn't see why not, I imagine your system would simply see it as removable storage anyway (which is what a USB stick would show up as). Yes.

It will behave exactly like a USB Flash Drive.

The NBR comes as an img file, which can be written directly to the Memory Stick; no need for Unetbootin.
