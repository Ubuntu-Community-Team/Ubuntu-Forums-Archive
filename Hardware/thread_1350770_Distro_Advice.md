### Post by snowpine on 2009-12-09
I recommend either "regular" Ubuntu (if you prefer Gnome) or Ubuntu Netbook Remix (if you prefer the launcher interface). The only major difference is the user interface.

Eeebuntu is in a period of transition right now (switching from Ubuntu to Debian) so I would wait and see what happens to that project before recommending it.

Use the forum Search feature for the specific netbook model you're considering. The different EEE models all have slightly different hardware, so results may vary.
