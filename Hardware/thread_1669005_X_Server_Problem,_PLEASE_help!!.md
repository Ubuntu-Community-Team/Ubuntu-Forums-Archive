### Post by drewsus on 2011-01-23
I have the same card. 
Do you have the latest Nvidia drivers? 
If not:
```
sudo add-apt-repository ppa:ubuntu-x-swat/x-updates
sudo apt-get update 
sudo apt-get upgrade
sudo apt-get install nvidia-current nvidia-current-modaliases nvidia-settings
```
**REBOOT**

Now, disable Nautilus from drawing the desktop:
```
gconftool-2 --type Boolean --set /apps/nautilus/preferences/show_desktop FALSE
```

Backup your current xorg.conf file to something that the next command will not overwrite:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.mybak
```

Plug in second monitor. 
```
sudo nvidia-xconfig
```

Logout and in. 
```
nvidia-settings
```

Go to "X Server Display Configuration"
Select your secondary monitor. 
Configure
Seperate X screen
Ok
Apply
Apply what is possible
Save to X Configuration File (uncheck merge)
Save (authenticate) 
**REBOOT**

### Post by drewsus on 2011-01-24
there are ways I have worked around this. 

you need to have the wmctrl package installed

```
sudo apt-get install wmctrl
```

then run these two commands:
```
gconftool-2 --type String --set /apps/metacity/keybinding_commands/command_11 "wmctrl -r :ACTIVE: -b toggle,fullscreen"
gconftool-2 --type String --set /apps/metacity/global_keybindings/run_command_11 "<Control><Alt>f"
```

Now, ANY active window will go into/out of fullscreen (maximised with no window title bar) when you press the key combination CTRL+ALT+F.
