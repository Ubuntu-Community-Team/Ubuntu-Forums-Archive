### Post by westie457 on 2013-10-09
Hi
Ubuntu is really friendly like that. Install on one HDD and move that HDD to another machine and it will run without complaining.

Windows on the other hand AFAIK will complain even from inside a VB.

Also you should use a different original Windows CD\DVD for each installation otherwise you will be breaking the terms of the EULA.

### Post by oldfred on 2013-10-09
You say they are identical.

The main issues of using on another system is usually video drivers and Internet drivers. If system are truly identical then you could even install the proprietary drivers. But normally if using on more than one system we suggest not installing the proprietary drivers.
