### Post by Marzata on 2011-07-05
Some days ago I bought brand new Lenovo ThinkPad X200. For a 3 years old model I expected everything to work from beginning. In the shop I checked that with the a bootable USB flash with Ubuntu 11.04. Everything seemed to be ok. Later in the office when I installed Ubuntu 11.04 on the hard drive, the system started to overheat and the fans running all the time at ~3500 rpm. I decided to install Ubuntu 10.04.2 which is the current LTS release but it was the same. I tried also 32 and 64 bits versions - the same. I installed Linux Mint - the same. I installed Fedora 15 - again the same. Because I really like that ThinkPad X200, I decided to read which laptop from Lenovo X series works fine with Ubuntu. It happened that most of the owners of X series machines complained  from the same issue. I also have Lenovo IdeaPad S10-2 netbook which is fantastic machine and works cold and quiet without any problems. Is there a Lenovo ThinkPad laptop, not only from the X series, which works fine with Ubuntu Linux? Which one?

### Post by An Sanct on 2011-07-05
Welcome to the forums!

Well, fan control is disabled by default for safety reasons. You are one of many to have this problem.

Take a look at this "[thinkwiki]("http://www.thinkwiki.org/wiki/How_to_control_fan_speed")" it is an article about fan control ... make sure to read all of it and do not ignore the warnings! (You want to have a working laptop after all ;))
