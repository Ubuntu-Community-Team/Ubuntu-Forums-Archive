### Post by ajgreeny on 2010-09-27
Definitely go for HP; great printer/scanner/fax machines.

Go to [http://hplipopensource.com/hplip-web/supported_devices/index.html](http://hplipopensource.com/hplip-web/supported_devices/index.html) for info on the models and support of hplip, showing the version of hplip needed for the printer.  If Lucid's hplip version 3.10.2 is not recent enough, you can download and install the latest version from HP very easily.
