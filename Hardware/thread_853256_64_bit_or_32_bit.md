### Post by dogscoff on 2008-07-08
I would go with 64bit. I've been running Gutsy & Hardy 64 on my laptop since about Feb with zero problems (well, no 64bit-specific problems, anyway!) Even the flash player (which used to be the bugbear of 64bit linux) works fine for me with no tweaking required. I just went to youtube, got an option to install a player, picked one and it works. 

Seriously, I don't think you're likely to run into 64bit problems in linux - nearly all software in the repository seems to have a 64bit build available. If it doens't have one, the source is usually available which means you can build a 64bit friendly version for yourself.

If you've got 64 bit hardware, why not get the most out of it?
