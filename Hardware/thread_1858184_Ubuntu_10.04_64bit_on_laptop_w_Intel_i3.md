The i3 should be compatible with either 32 bit or 64 bit Ubuntu 10.04. Booting Ubuntu 10.04 from a live cd or usb will allow to test the functionality to be certain. Live cd method: download the Ubuntu iso image file, burn it to a cd, set your bios to boot from cd, reboot with cd in the try, and select try before installing option.

Live usb method, download the Ubuntu iso, download unetbootin, plug in usb drive (usb > 1gb in size), open unetbootin and select
your usb stick, browse for your Ubuntu iso image, proceed with button at bottom, set bios to boot from usb, try Ubuntu without installing to test compatibility.
Hope this helps.
