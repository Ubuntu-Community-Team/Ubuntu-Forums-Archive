### Post by taurus on 2009-02-05
Try to include the filesystem.

```
sudo mount -t ntfs-3g /dev/sdb1 /media/sdb1
```
And if that still doesn't work, boot into windows and run scandisk and defrag of that drive, /dev/sdb1.

Then, reboot into Ubuntu and see if Ubuntu can mount it now.
