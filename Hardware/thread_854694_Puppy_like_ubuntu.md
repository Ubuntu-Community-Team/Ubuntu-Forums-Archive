> **atomkarinca said:**

+1; Aysiu's guide is excellent. Fluxbuntu is kind of like a pre-bundled minimal install. If you like the choices the developers made, it can be a real shortcut for a newbie like myself. But if you're going to end up changing everything, you're better off starting from the minimum install and building up exactly what you want. 

I do think that Xubuntu is too bulky for your 96mb ram notebook, Fluxbuntu or a minimal Ubuntu install would fit (though heavy apps like Firefox or OpenOffice might be a stretch); also, there are even lighter non-buntu options--DSL for instance.
