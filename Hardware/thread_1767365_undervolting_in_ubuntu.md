### Post by tgalati4 on 2011-05-25
You need to recompile your kernel in linux with P-States Voltage Control set to "on".  Takes about 2 hours to recompile my kernel on Jaunty using a thinkpad T43p.  Then you can use a panel widget PHCTool to undervolt.  There are several threads in the forums on how to do it.  Only certain processors support undervolting.
