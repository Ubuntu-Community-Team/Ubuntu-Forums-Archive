### Post by jvance492 on 2011-09-20
I just resolved mine. Pretty sure its a flash problem.

When I installed flash i used:
sudo apt-get install ubuntu-restricted-extras

This installs flash player, an mp3 codec, and java I think. 


This is my solution

sudo apt-get remove ubuntu-restricted-extras
Y
sudo apt-get install ubuntu-restricted-extras
Y

Good luck, Also while not a solution a feasible work around if that didn't work for you. If you right click on the video and hit Pop Out its full screen. This is Chromium browser.
