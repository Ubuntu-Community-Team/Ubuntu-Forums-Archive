Is it the same machine as the one you posted about [here]("http://ubuntuforums.org/showthread.php?t=2306363")?

If not, start by posting the hardware specs.

Without further informations the best we can do is to suggest the *nomodset *boot parameter -> As soon as you see the "keyboard and human" splash screen press any key, select default language and press F6 then select "nomodeset". Then - and only then - you can use the main menu options to either select "Try..." (or directly "install").
