### Post by cottonswab on 2010-05-08
I had a similar issue with my T60 Lenovo Thinkpad. I wanted to use only external monitor with higher resolution (than 1024x768). The following worked for me:
1. Go to System>Preferences>Display
2. Uncheck Mirror screens
3. Select Thinkpad 14" laptop screen and turn it off. (This enables higher resolutions on the external display)
4. Select the highest (best) resolution.
But simultaneously I cannot have the best resolution on both screens. Thats still a problem.
