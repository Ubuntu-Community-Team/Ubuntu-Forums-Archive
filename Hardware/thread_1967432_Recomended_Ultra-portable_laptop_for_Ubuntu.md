### Post by natgab on 2012-05-02
Don't know if you are still looking. 

But one place to start would probably be Dell laptop, since they do offer some with Ubuntu preloaded.  I would think that most would use the same components even if they are MS Windows models.  

System76 also sells some with Ubuntu, so it would hurt to see if any of them can fit your needs. Might be good to check the System76 section of the forums.

Personally, I use a Thinkpad X40 (Ubuntu 12.04 LTS), its an ultraportable 12" laptop.  My X40 is an older model, but Lenovo still makes some in the same ultraportable size.  IBM / Lenovo laptops are pretty linux compatible.  My laptop has worked with whatever Distro I throw at it (and Haiku too).
