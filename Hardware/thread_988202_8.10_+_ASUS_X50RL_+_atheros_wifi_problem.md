### Post by andy_cowman on 2009-02-17
Might be a bit late!! But just used the info you provided... to edit that file you need to open the file as a superuser. One way to do this is to open the command line and use something like

sudo gedit /sys/devices/platform/asus-laptop/wlan

Then change 0 to 1 and save the file.

It might also be worth noting that I set the bluetooth file in the same location to 0 as despite having a bluetooth light I don't think my laptop actually has bluetooth!!
