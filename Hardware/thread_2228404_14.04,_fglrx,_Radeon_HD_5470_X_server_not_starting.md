### Post by TheFu on 2014-06-07
I'm not an expert, but have noticed that installing GPU drivers outside the Ubuntu repos doesn't turn out well long term.  If you stay in the repos or with a highly trusted PPA, then when a new kernel gets installed, the GPU drivers are relinked. If you install from source, I haven't seen that - always.  After every new kernel install, I --reinstall the fglrx drivers and reinitialize the settings.

```
sudo apt-get  --reinstall install fglrx

``` This seems to fix many little and large video issues here.
Don't know if it will help you or not.  Doesn't ATI have a GUI to manage the video card settings?  Shouldn't need to touch the xorg.conf manually anymore.  I know that nvidia works that way with nvidia-settings.
