### Post by jbicha on 2011-05-12
This is a very old [bug]("https://bugzilla.gnome.org/show_bug.cgi?id=325150") in Gnome. In the past it has seemed that the Gnome/GTK developers believe users should just use their File Manager to manage files and use the "file-chooser" to choose files for opening or saving, but not renaming or deleting.
