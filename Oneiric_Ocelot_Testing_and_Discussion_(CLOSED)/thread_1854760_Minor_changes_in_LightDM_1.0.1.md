### Post by Harry33 on 2011-10-05
The newest version of LightDM (1.0.1-0ubuntu1) does possess a new feature.

If the setup has only one user and the guest session is disabled
(with the line allow-quest=false in /etc/lightdm/lightdm.conf)
you do not have to hit enter anymore to choose your user account.

Just directly start typing the pw.
