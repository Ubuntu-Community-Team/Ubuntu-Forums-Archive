### Post by coffeecat on 2011-09-18
This is a FTR (for the record) because it probably won't affect anyone much.

A [few weeks ago](http://ubuntuforums.org/showthread.php?t=1831283) Synaptic was removed from the default list of apps. I discovered today when using a live USB from today's daily ISO with persistence that Synaptic has moved from the main to the universe repo. sudo apt-get install synaptic gave "unable to locate package synaptic" which threw me for a bit - until I enabled the Universe repo.

I doubt whether this will make much if any practical difference in permanent installations, but I thought I would point this out so that people who have been active here and who will be helping those using Oneiric for the first time when it goes final might need to know.
