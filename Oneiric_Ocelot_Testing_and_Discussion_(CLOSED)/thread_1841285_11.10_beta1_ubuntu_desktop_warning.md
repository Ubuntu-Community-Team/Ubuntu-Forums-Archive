### Post by Martin Marshalek on 2011-09-09
I'm not sure if it makes a difference here, but I've heard before that you need to have the ubuntu-desktop metapackage installed in order to do a dist-upgrade. Sometimes people remove this when they remove some of the default apps and I've read before that it's pretty much useless **unless** you're doing an upgrade to a new version. Just run:

```
sudo apt-get install ubuntu-desktop
```

Then try to update again and see if it works.
