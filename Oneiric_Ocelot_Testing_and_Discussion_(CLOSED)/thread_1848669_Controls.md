You are using gnome. Oneiric is based on gnome 3, and the gnome devs have been reducing configurations options for the last couple of years, the ubuntu devs have been adding back some configuration options, such as changing themes from the Appearance window, but we'll have to wait for other developers to come up with tools to help configure our desktops. There is a tool available, that depends on gnome-shell, which isn't installable from the repositories at the moment, called gnome-tweak-tool that does allow you to change more configurations.
