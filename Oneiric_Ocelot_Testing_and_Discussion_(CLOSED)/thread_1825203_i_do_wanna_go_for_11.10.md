### Post by cariboo on 2011-08-16
Running a testing version in a VM doesn't really allow you to completely test it, as it is running on virtual hardware. The idea behind a testing release is to test it on as many different types of hardware as possible, so that as many bugs as possible are reported and hopefully fixed, using a VM defeats the purpose, as you are more less just seeing bugs in the program you are using, be it vmware or vbox.
