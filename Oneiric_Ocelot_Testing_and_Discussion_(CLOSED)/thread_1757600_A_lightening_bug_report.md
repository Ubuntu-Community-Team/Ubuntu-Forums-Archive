### Post by kansasnoob on 2011-05-14
I see that quickly earned an "invalid" status.

What one must understand is that Natty uses gnome version 2.32, but Oneiric will use gnome version 3.* which implements the "classic" gnome-panel desktop in quite a different manner.

That said I have a gnome3 desktop running fairly well on Natty with the use of the gnome3 PPA (via UGR) using the forced fall-back DE and the only actual DE draw-backs ATM are the lack of functionality related to two panel applets that I normally use (specifically inhibit-applet and gnome-sensors-applet).

All of this fretting is a total waste of time and effort, over the next few months I expect gnome3 development to move forward quite quickly based on recent experience. We just need to be a bit patient and see how things shake out :D

I see Unity as just one more option, not my #1 preference, but if one DE fit everyones needs there wouldn't already be so many options.
