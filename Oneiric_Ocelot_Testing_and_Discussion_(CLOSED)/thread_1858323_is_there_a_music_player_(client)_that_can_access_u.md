You can try VLC, it has the option, but don't know what the compatibility is like.

When you open VLC you have to go to the 'View' menu and select 'Playlist', if the playlist window is not already being displayed. Then in the 'Media Browser' pane on the left go to 'Local Network --> Universal Plug'n'Play'.
