### Post by apollothethird on 2011-10-09
I recommend version 11.04.  You don't have to practise Windows these few days and have a lot of relearn next week.  Since you're interested in Ubuntu, why not start out with it.

After for days you can upgrade to 11:10 with a simple apt-get commandline.  With this, you won't skip a beat.

I wouldn't recommend the beta on a main machine.  It's stable, but the developers are still working on it.  There might be a chance you'll run into support issues with the beta which might be something on your end or it might be with the beta which is still being fine tuned.

Install the release.  You'll have plenty of support for any issues that you might experience.  Then when you move up to the release version of 11:10, you'll still be a member of a large support system for any problems you might have.  Thousands of people will be using the same apt-get commandline to upgrade.  You'll most likely see many tested resolutions to any issues you might have before having to ask questions.

Why put yourself into a position to ask a question that has been responded to countless times in the beta section, that it's a bug that is being removed by the release version.
