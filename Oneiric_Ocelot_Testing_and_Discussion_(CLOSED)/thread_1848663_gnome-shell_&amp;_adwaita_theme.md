### Post by Krytarik on 2011-09-23
The "Adwaita" theme is included in the package "[gnome-themes-standard]("http://packages.ubuntu.com/oneiric/gnome-themes-standard")", it should be the original one.

As it's only 'recommended' to install complementary to "[gnome-shell]("http://packages.ubuntu.com/oneiric/gnome-shell")", make sure it's installed:
```
sudo apt-get install gnome-themes-standard
```Greetings.
