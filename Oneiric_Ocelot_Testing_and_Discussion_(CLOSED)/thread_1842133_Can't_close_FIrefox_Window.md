### Post by effenberg0x0 on 2011-09-11
Try any application. Gedit, calculator, firefox, etc.

- Open
- Minimize
- Restore (click it in the launcher)

There. You can't move it, minimize, maximize or close. Thats a documented bug for which there already is a proposed patch pending approval.

Workaround: Restore the app (click on launcher), immediately press alt+space
n = Minimize
x = Maximize
c = Close (or use the good old alt+f4)
m=move (or click anywhere on the app, except the titlebar area, while pressing left alt key and move)

Effenberg
