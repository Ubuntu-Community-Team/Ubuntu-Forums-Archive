### Post by Peter09 on 2011-05-02
I've posted this in a couple of places so apologies if you've seen it before.

UI designers have long recognised that the mouse/menu system is not the best way of driving a computer. Navigating a mouse to a small target, moving over multiple other targets before clicking requires concentration and coordination. Just watch a beginner try to find a menu target from a big menu.

Power users have for a long time deferred to keyboard shortcuts, command lines and searches to perform these tasks, even in MS Windows. They do it because it is quicker and easier - it allows the user to concentrate on the task in hand.

What Unity does is move focus away from the mouse/ menu towards an easy to use and easy to remember system of shortcuts.  Allowing casual users to easily adopt the practices of Power Users.

If you are going to enjoy Unity you have to change you way of working somewhat to fit in with this concept. Once you do so it all comes together.
