### Post by MARP1961 on 2011-09-09
For those reading who might be confused with all this:

Oneiric Ocelot (11.10) will be the first Ubuntu to run with Gnome 3 instead of Gnome 2. I like to think of Gnome 2 and Gnome 3 being like different engines in a car. Although cars need an engine, they also need a body shell and** Unity is the new default 'shell'**. The 'shell' is what you see on your desktop.

Now some people, (people like me) don't like body-shells that cannot be customized easily. If you like Unity as it is then fine; but for the rest of us there are alternatives.** Gnome Shell **(which some people are still incorrectly calling 'Gnome 3' is a fine alternative. Some people have already created **extensions** for Gnome Shell which mean that you can alter it's behaviour and appearance much more substantially than with Unity.

You need to make sure you have Gnome Shell installed via either Synaptic or the Software Centre then choose it at login. You will be greeted with a simple, attractive and symmetrical desktop. Clicking on 'Activities' will reveal an attractive 'Overview Screen' with it's own dock/side panel. Clicking on 'Applications' will reveal a much slicker dash than that awful Unity one. You simply start typing and an extremely efficient search will find your application instantly. It is truly wonderful whilst Unity is plain and boring in comparison.

The way it handles multiple workspaces is great fun! Actually, my advice is use both Unity and Gnome Shell until you find the one you prefer.
