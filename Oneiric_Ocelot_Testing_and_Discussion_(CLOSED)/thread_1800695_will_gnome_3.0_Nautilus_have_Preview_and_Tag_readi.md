### Post by jbicha on 2011-07-11
If by "Preview" you mean, will hovering over an audio file in Icon view start playing a preview of it, then yes, that works. Nautilus 3 is mostly the same as Nautilus 2.32: the biggest change for me is that the Sidebar has been reorganized.

No, mp3 tags are not shown in list view. I suggest you use a music app for that functionality as the "List Columns" feature in Nautilus only allows displaying metadata that all files have.
