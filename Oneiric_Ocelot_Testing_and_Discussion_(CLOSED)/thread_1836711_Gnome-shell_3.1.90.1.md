### Post by tekstr1der on 2011-08-31
Gnome development versions are odd numbered. Releases are even.

2.29 dev became 2.30 stable, 2.31 dev became 2.32 stable, 3.1 dev will become 3.2 stable, etc.

The .9x suffix denotes betas and RC.

It's the same every release cycle which is nice and consistent.
