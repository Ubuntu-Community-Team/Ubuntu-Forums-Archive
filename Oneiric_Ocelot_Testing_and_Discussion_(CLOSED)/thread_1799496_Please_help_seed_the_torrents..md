### Post by jbicha on 2011-07-11
I just use zsync to get the latest daily build.

I make a copy of the latest Ubuntu ISO that I have and rename the copy to oneiric-desktop-amd64.iso (for instance). Then I run the zsync command and it will update the parts of the ISO that have changed but it won't need to redownload the parts that haven't changed.
