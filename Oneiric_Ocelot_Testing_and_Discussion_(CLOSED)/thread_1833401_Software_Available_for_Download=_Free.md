### Post by jbicha on 2011-08-26
There are 64-bit apps for sale. Click the All Software drop-down arrow and select For Purchase.

Yes, nearly all Linux apps are absolutely free. There are some commercial apps that cost money.

Perhaps a future Software Center will allow donations but that gets a bit complicated. Some developers believe that introducing money into a volunteer-based system will severely hurt volunteer development contributions. Others appreciate donations.
