### Post by jbicha on 2011-08-20
Bug filed: [https://bugzilla.gnome.org/show_bug.cgi?id=656982](https://bugzilla.gnome.org/show_bug.cgi?id=656982)
GNOME has significantly decluttered/simplified the screensaver. Setting alternate screensavers is no longer supported in GNOME Core. But it's unclear whether this feature should be removed so at least there's a tracking bug now.

> **ventrical said:**
> is that 'lockfile-progs' because if that is the file, it is in synaptic now.

Lockfiles are different. Firefox sets a .lock for instance because it doesn't support having multiple instances of Firefox open using the same profile.
