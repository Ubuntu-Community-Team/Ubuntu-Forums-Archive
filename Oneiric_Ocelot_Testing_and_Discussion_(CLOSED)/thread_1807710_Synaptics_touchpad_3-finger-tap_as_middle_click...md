### Post by rubengs on 2011-09-15
Well I found a really ugly workaround for this, I installed the kde-config-touchpad, this installed a lot of kde based dependencies, after that I ran the touchpad configuration utility with the command "synaptiks", had to do it twice because it needs to load a library previously, in the last tab of this kde dialog box I set the middle click to be activated with two fingers and the right clik with three, so I basically lost right click to be able to use middle click because the three finger is still not working because unity, so the right click is done via touching the bottom right corner (you can set any corner for this). 

It's far from perfect but I can use oneiric now.
