### Post by ranch hand on 2011-07-14
The alt-install CD uses the Debian installer.  It is not as pretty but has years of stable performance behind it.  Pretty reliable.

Many folks use it in preference to the LiveCD.

It is used when the Live CD is not updated.  Keeping the Live CD working is a major job.  Some think that time could be better spent working on the OS itself.

The Live CD will be back when they get the time and certainly before A3 as ISO testing for that is critical.
