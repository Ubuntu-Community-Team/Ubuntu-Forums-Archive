### Post by kansasnoob on 2011-07-08
I know this is largely OT for the true purpose of Oneiric development, but many of you know I've been trying for several months to get a setup where I can more efficiently perform iso-testing.

Finally I got my home router set up today with an inexpensive KVM switch and all went very well, almost too darn easy :D

Now, I did used to perform LAN wiring as part of my last job so that helped but I still can't believe how easy this went.

The reason this is important is just how many installation options are now truly available. The live-installer (aka: ubiquity) now allows a "replace existing OS/upgrade existing OS", a default to Wubi if Windows is using 4 primary partitions, and "install alongside" now includes a "use free space" option that's still a bit buggy.

This will allow me to run more tests at one time I hope, and hopefully once Lubuntu get's added to iso-testing I'll have a third old PII-335mhz w/256MB of RAM hooked up constantly so I can potentially run three tests at the same time, how cool is that?

Sorry if this is out of place, but I'm just sooooo happy. Also, for this being Alpha 2 I thought the team did very well with handling iso-testing, bug reports, release notes, etc.

The biggest draw back to this is getting Windows XP to behave on the second computer :(
