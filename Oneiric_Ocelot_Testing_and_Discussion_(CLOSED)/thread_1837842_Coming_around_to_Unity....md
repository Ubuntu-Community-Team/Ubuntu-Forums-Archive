### Post by cariboo on 2011-09-02
IF you have quite a few applications open, try alt-tab, if you have several windows for the same program opne after pressing alt-tab, press alt-tick, the key right above the tab key.

The first screenshot is alt-tab, just keep pressing tab to cycle through the apps, the second screenshot is alt-tab, then still holding down the alt key, press the tick key to show the other open windows, press the alt-tick to  cycle through the windows.
