### Post by jbicha on 2011-09-08
The missing minimum/maximum buttons are because the GNOME designers don't think they're useful. Try GNOME Shell for a few weeks and see if you agree. You can minimize by right-clicking the title bar. If you can't stand the missing buttons, just install gnome-tweak-tool; it has an option to show the extra buttons. While you're at it, change your window theme at least to Adwaita; Ambiance won't display correctly in GNOME Shell [yet]("https://bugs.launchpad.net/light-themes/+bug/800315").
