### Post by cariboo on 2011-09-10
Dual displays work for me without any problems using nouveau and nvidia-current. I'd suggest removing /etc/X11/xorg.conf, and rebooting. I have run into a problem when re-installing nvidia-current, but removing xorg.conf and rebooting solves the problem. Use nvidia-settings to create a new xorg.conf.
