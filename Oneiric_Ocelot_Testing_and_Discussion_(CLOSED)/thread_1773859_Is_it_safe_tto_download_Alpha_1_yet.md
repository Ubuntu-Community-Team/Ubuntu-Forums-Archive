### Post by jerrylamos on 2011-06-02
[https://wiki.ubuntu.com/OneiricOcelot/TechnicalOverview/Alpha1](https://wiki.ubuntu.com/OneiricOcelot/TechnicalOverview/Alpha1) has the link to the Alpha 1 version of Oneiric.  I just downloaded it with zsync.

At least I can get Firefox up with it as I use it here.  The current daily build wouldn't do anything.

Do note the current list of bugs on the Technical Review.

By the way, I put oneiric-desktop-i386.iso into a directory /boot/iso on /dev/sda1 then altered /etc/grub.d/40_custom as follows;

#!/bin/sh
exec tail -n +3 $0
# This file provides an easy way to add custom menu entries.  Simply type the
# menu entries you want to add after this comment.  Be careful not to change
# the 'exec tail' line above.
menuentry "Ocelot iso on sda1" {
set isofile="/boot/iso/oneiric-desktop-i386.iso"
loopback loop (hd0,1)$isofile
linux (loop)/casper/vmlinuz boot=casper iso-scan/filename=$isofile noprompt noeject
initrd (loop)/casper/initrd.lz
} 
#! exit

then did 
sudo update-grub
reboot and select the above entry to get Oneiric Ocelot Alpha running Live.

Pretty quick.  Do note, living somewhat dangerously with an Alpha on my main hard drive so back up everything....

Enjoy.  Jerry
