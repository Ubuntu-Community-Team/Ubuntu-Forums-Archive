### Post by gnomeuser on 2011-05-01
> **benjamimgois said:**
> Wow i didn't knew about muinshee (banshee interface), it's mockup looks awesome !


Muinshee has existed for a while, it is a Banshee backed version of the classic Muine player. You can run it today using muinshee from the standard Banshee package (the meego interface is split out in banshee-meego). The mockups have not been implemented, so the look is more like the original Muine interface. The experience is intended to be very album centric. I suspect given the workload for 2.1.x-2.2 is to hefty to see the mockups implemented by the core developers but patches are naturally welcome. I would suggest waiting till the GTK#3 port has happened though.

One reason Banshee rocks, multiple UIs on the same database without losing any of the power of a full library oriented player.

### Post by Johnsie on 2011-05-05
Evolution may be ugly and buggy, but it's the only email client for Linux that even tries to implement exchange/global address book. Exchange compatibility is necessary for most existing corporate environments. Certainly every company I've worked at over the last 6 years and my University before that had an exchange based system. If you want to plug a Linux computer into a Windows network then it's important to have exchange compatibility for emails and the corporate address book.  

My suggestion is that Evolution be radically improved so that it's easy for anyone who wants to have a Linux workstation on a Windows network (and looks better that outlook)\

Companies are less likely to change their server than they are likely to change a workstation. Having a better exchange email client would really help with that, especially in the office environment.
