### Post by mc4man on 2011-10-02
There are 3 cli's for deja-dup, for gui
```
gnome-control-center deja-dup
```

The gui desktop is named deja-dup-ccpanel.desktop, should be seen in /usr/share/applications along with 2 others

Edit: try the one found in System Settings
