### Post by Quackers on 2011-09-17
From live cd desktop open a terminal and run ```
sudo mount /dev/sda5 /mnt
sudo grub-install --boot-directory=/mnt /dev/sda
``` which should report no errors.
This will over-write Windows bootloader in the MBR of the hard drive.
When run, reboot and you should get Ubuntu booting directly. If so, open a terminal and run ```
sudo update-grub
``` and watch as grub.cfg is run. You should see the Windows Loader recognised. If it is reboot and you should then see a grub menu giving you the choice of which OS to boot.
