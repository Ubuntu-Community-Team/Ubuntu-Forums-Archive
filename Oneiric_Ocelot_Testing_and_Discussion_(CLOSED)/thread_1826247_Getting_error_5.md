### Post by Catharsis on 2011-08-17
If I understand correctly, there's something wrong with either A) your USB drive, B) your ISO image file, or C) the partition you're trying to install to.

First, exactly which image did you put on your USB drive, and what program under which Operating System did you use to put it there?

Diagnostically, first you should use the "Check disc for errors" option on the LiveUSB menu and see if it checks out. You should also check the MD5SUM of the ISO you downloaded and make sure it's correct. Then you should make sure you can mount the partition you're trying to install onto from the live session of the LiveUSB. This is all assuming that you have used the USB stick before and know that it works fine.
