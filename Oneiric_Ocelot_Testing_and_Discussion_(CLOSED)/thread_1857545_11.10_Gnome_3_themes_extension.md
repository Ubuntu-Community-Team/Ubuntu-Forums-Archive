> **cecilpierce said:**
> @fillmont

Seems like when I try anything new it messes up something else, so I'm not sure what to do anymore.
How would I get gnome user theme extension working ?

Honestly, the first time I got any extensions to work was after reading this [webupd8 post  ]("http://www.webupd8.org/2011/10/official-gnome-shell-extensions.html")from a few weeks back. Before that, I had tried, and failed, like, 4 times.

Basically, you have to make sure your gnome 3 is up to date. If you've been doing regular 11.10 upgrades, it should be. You can make sure by going to system tools and checking your gnome info. It should be 3.2

Then sign up with the webupd8 ppa to get access to the extension packages. Once you update and install the ones you want, it's just a matter of restarting gnome shell to get access to the extensions. You use gnome tweak to use them.

After you get user theme extension up and running, add the theme folders for the ones you want to /usr/share/gnome-shell/theme. You don't need to rename anything. Once you plop those in, you should be able to switch easily using gnome tweak.

But really, you should just read the webupd8 post. It explains a fair bit better than I could!

Good luck!
