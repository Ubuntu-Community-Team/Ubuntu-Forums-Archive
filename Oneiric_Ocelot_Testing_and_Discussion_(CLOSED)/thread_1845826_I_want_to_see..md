### Post by mc4man on 2011-09-17
You'd probably get a better idea by booting the current daily and experimenting if possible  - there are numerous possibilites.

There has been some progress on the blurs & over light backgrounds has been improved but it's still possible to get hard to read dashs

There are 3 blur options - active, static & no blur
The active blur can change every time the dash is opened, hence active, more likely to get readable issues
The static blur is created the 1st time the dash is opened and lasts for the session
The no blur now uses the desktop background to create a pretty much solid dash color (though I see a graphical glitch in no blur as soon as the cursor is moved in a lens
(I'll probably use no blur even if the glitch remains, have no interest in seeing thru the dash at all
