### Post by budluva04 on 2011-08-17
if you guys are using 64bit...try this out, worked flawlessly for me..

```
sudo add-apt-repository ppa:sevenmachines/flash
sudo apt-get update
sudo apt-get install flashplugin64-installer
```

thanks to jbicha@#ubuntu+1 for this info
[https://launchpad.net/~sevenmachines/+archive/flash]("https://launchpad.net/~sevenmachines/+archive/flash")
