### Post by ventrical on 2011-08-18
Got it! No global updates for new 'accounts'. This is pretty crafty (for now) because the Compiz settings also will not update in new accounts - so - if one user gets borked by Compiz the 'new' users will not be affected. 

 The same goes for updates .. if one user get borked by the updates , the new users will not get borked unless synaptic is run on that particular "new'user.
