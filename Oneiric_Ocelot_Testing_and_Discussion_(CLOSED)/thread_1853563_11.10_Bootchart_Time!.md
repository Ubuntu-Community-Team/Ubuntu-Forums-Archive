### Post by MacUntu on 2011-10-03
> **xyzzyman said:**
> It was 2 minutes when I had apache and mysql installed. I was only ~40 seconds with 11.04.

Well, you have some custom stuff in there like uvesafb and Preload. It's not easy to tell where problems are that way.

What's never wrong, is to refresh the pack files for Ureadahead:

* Disable Preload for the procedure. It would be kinda fatal for the boot time if Ureadahead caches the stuff that Preload is touching when starting the session.
* Go to '/var/lib/ureadahead' and remove all files (not the 'debugfs' folder).
* With autologin enabled, do a full restart.
* Once you are at the desktop, wait for a minute or two (don't do/start anything).
* Make sure there are pack files in '/var/lib/ureadahead'.
* Reboot with Bootchart enabled.*

*) Personally, I have Bootchart installed all the time and added 'bootchart=disable' to 'GRUB_CMDLINE_LINUX_DEFAULT' in '/etc/default/grub' (if you add that too, don't forget to run 'sudo update-grub' afterwards). When I want a chart (which is not every boot obviously), I just edit the GRUB menu entry when booting, i.e., pressing 'e' on the selected entry, going down to the 'linux' line, removing the 'bootchart=disable' part, pressing Ctrl+x to boot.
