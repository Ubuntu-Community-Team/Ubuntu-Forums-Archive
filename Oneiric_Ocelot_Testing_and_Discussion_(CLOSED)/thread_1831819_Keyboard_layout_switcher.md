### Post by Yumi on 2011-08-29
Keyboard layout and input method are two different things. If successful you will find eventually two icons on the top-bar. One for keyboard layout switching, one for input method English/Hebrew.

In my case with Chinese I would not need to switch keyboard layouts as Chinese computers use US-layout, Just toggle between English and Chinese writing methods. But for my German language input I need a German keyboard layout. There is no German input method just a different keyboard layout with our special characters.

The Chinese input method is installed from Synaptic or the software centre. As an example Arabic is written from right to left and has a input method in synaptic. I do not know anything about Hebrew but there are many packages in the repository, you have to find out from a different source. 

Michael
