### Post by cariboo on 2011-05-12
Edubuntu, is designed for a totally different type of user, it actually assumes you know what you are doing, whereas the Ubuntu desktop install assumes you have no idea what you are doing :).

The Edubuntu install allows you to set up a server/client system, which most new Ubuntu users either have no idea what it is, or don't have the skill set that allows them to setup a working server/client network.
