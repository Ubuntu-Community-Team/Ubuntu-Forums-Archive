### Post by Framli on 2011-09-16
The lenses API doesn't deal with the appearance of the lens, it just receives the search string, defines its filters & categories and provides data sources. There is one exception : it has the choice between using vertical (icon on top, title on bottom) and horizontal (icon on the left, title & commentary on the right) rendering. Everything else is handled by the Dash.
