### Post by garvinrick4 on 2011-09-20
Everyone has an opinion about upgrading or fresh install. For a new user it is moot, wait until the
new version is released before doing either one. Being new and attempting to keep a Alpha or Beta
running is not worth an attempt and certainly not for a main install. When the new version is released.
[Download Ubuntu | Ubuntu]("http://www.ubuntu.com/getubuntu/download") 

If your /home is on a seperate partition at original install can just fresh install, download, burn cd, install, set up and done, nice and clean.

If /home is not on seperate partition back up your /home files and keep them. Download image and
Burn CD and install in the same partition and put your /home files back in place, set up and done.

If you choose to update have over 1000 packages to upgrade from one version to next. Takes time
and you do not have a clean as install in this users opinion as a fresh one.

You are going to want to have a Live Cd (install cd using Try Ubuntu) handy anyway so the time it takes downloading and burning the CD is also moot, need to do it anyway.

If have gotten a better skill set in Ubuntu just make a new logical partition and keep your old version and install the new version.

Over time it will be a nice constant rotation of Ubuntu versions coming in and going out when no longer
supported.

Learn to keep your /home in its own partition so all can use and any fresh install does not effect your
personals. Always keep personals backed up if you value them.

Only takes about 10 gig partition to install Ubuntu, 5 gig is a large / so an extended partition with some
logicals inside and you are off and running, 40 gig gives you 4 partitions and whatever size you want
your /home. With drives being the huge size they are nowadays it is nothing. So make an extended

### Post by JMFTheVCI on 2011-09-20
Is it good? No, not for me.

I use a Samsung NC10 netbook and have been running every version of Ubuntu from 8.04 with great success facility and enjoyment.

On Sunday I upgraded from 11.04 to 11.10. Poor. It took over six hours to perform the full upgrade. Then would not boot into Unity until I had cured the /var/run and /var/lock issues.

Then came the nasty realisation. Unity. OMG that is poor on a netbook. You cannot resize, move or tailor it whatsoever. Ubuntu-classic was not installed by default but when I did install it also does not work as designed as my netbook does not have the higher level of graphics support required for Gnome3.

I have always liked Ubuntu as you could tailor the whole interface to give you as much screenspace as possible on a netbook which made the whole thing worthwhile. Now you can tailor nothing. You cannot even remove the bottom panel or move it's contents around. 

With things like Gnome-Do and Docky the whole OS thing was fast. And I mean fast enough to run Lotus Notes on for email. Lotus Notes on the same hardware under Windows 7 was dire. Now Ubuntu 11.10 is slower than Windows 7. What a dire state of play. 

However I am fully aware that 11.10 is in beta state but have enough experience of previous beta and even alpha releases to know that the whole experience was one of total failure, inflexibility, poor performance and rejection.

Instead of developing an excellent platform further the Ubuntu team have gone completely off the rails and created a plaform for large screen dummies with big push-buttons and no flexibility.

And to demonstrate my disappointment? I performed a /FixMbr and restored Windows 7 as the only OS on my Netbook.

Ubuntu, you should be thoroughly ashamed of yourselves.
