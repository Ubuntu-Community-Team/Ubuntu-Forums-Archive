### Post by ranch hand on 2011-06-07
My son uses Ubuntu on a new box and the first thing he does is remove OOo or LibreOffice (depending on the packages installed on any OS).  Abiword is all he needs for document creation.  Why have all the other stuff?

I have had installs of both Lubuntu and Xubuntu in the past and I install LibreOffice because I really like it.  There really is no need for it is Xubuntu at all.

The only xfce and ldxe install I have now is on one Wheezy install so I have it on there but I am not sure I would install it if I had a clean install of either of them as neither would be my Main OS.  In that case a full office suite is just bloat.

I suspect that most home users would get by quite fine with Abiword and Gnumeric for their home office needs.  Without the big suite any DE is going to be somewhat more responsive no matter the age or power.

A number of people I know of like a stripped down install on new gaming computers for the speed it gets them on high power boxes.  They do not need an office suite to play WOW.
