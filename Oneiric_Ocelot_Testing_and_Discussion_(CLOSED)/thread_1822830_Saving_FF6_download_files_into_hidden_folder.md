### Post by coffeecat on 2011-08-11
> **dino99 said:**
> Wonder where i can change this setting.

Firefox menu Edit > Preferences > General Tab. Click on Browse (next to "Save files to" field). In the "Choose Download Folder" window simply press ctrl-H to show, and select, hidden folders.

Alternatively, you can type the path with the hidden folder directly into Save file to field.
