### Post by jbicha on 2011-08-02
There are definitely no plans for an official Gnome Shell edition of Ubuntu.

But anyone that wants to can create their own live CD remix. I believe Ubuntu Gnome Shell Remix or possibly Ubuntu Gnome Remix could be acceptable names but you'd definitely need to read the [Trademark Policy]("http://www.ubuntu.com/aboutus/trademarkpolicy"). If I were going to do this, I'd also submit a detailed [request]("https://forms.canonical.com/trademark/") to Canonical to get their approval for the project scope and naming. It looks like to use the Ubuntu ____ Remix name, you're limited in the degree of changes you can make (you should use packages in the official repositories).

I use Gnome Shell and for me it's an easy install in Oneiric. I don't see the need to do all the work necessary to make it a separate live install CD.
