### Post by rigobot on 2011-09-11
I've have encountered some CPU usage problems with Firefox and Chrome, that together with the server X process, were running the CPU all the time.
In particular Firefox with the "continuous scrolling" enabled was very luggish.
I tried everything and the only thing that seems to work is disable the dual screen, after deactivating the secondary screen my cpu usage was returned to the normal percentages and Firefox is smooth again.
I have the lastest nvidia driver available in the repos.
In the previous version of Ubuntu I never had a second screen so I don't now if it is actually a regression, is there anyone that has a similar problem?
