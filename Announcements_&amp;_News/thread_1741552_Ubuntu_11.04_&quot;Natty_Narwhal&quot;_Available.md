Rolf   The Ubuntu team is pleased to announce Ubuntu 11.04, code-named "Natty Narwhal". 11.04 continues Ubuntu's proud tradition of integrating the latest and greatest open source technologies into a high-quality, easy-to-use Linux distribution.  For PC users, Ubuntu 11.04 supports laptops, desktops and netbooks  with a unified look and feel based on a new desktop shell called "Unity". This version supersedes Ubuntu Netbook Edition for all PC netbooks.  Developer reference images are provided for select Texas Instruments (TI) ARM platforms, specifically the "PandaBoard" and "BeagleBoard".  Ubuntu Server 11.04 has made it easier to provision servers, and reduce  power consumption.
