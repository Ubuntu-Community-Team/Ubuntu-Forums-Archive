### Post by jdong on 2005-02-24
Community Members:

The moderation team has received numerous reports from users of spam and such in the Mailing Lists. Note that we, UbuntuForums.org, are **not associated** with the official Canonical Ubuntu Mailing Lists -- we simply provide a simple, one-stop interface that makes the list appear like a forum.

We cannot moderate posts or delete threads in the mailing lists -- that's beyond our control. It's normal to get some spam in a mailing list -- please bear with it! Please don't report stuff found in the mailing list forums.

(In case you are confused, the following are mailing lists:)
[http://ubuntuforums.org/forumdisplay.php?f=17](http://ubuntuforums.org/forumdisplay.php?f=17) -- Official Ubuntu Announcements (Including Security Announcements Subsection)

[http://ubuntuforums.org/forumdisplay.php?f=6](http://ubuntuforums.org/forumdisplay.php?f=6)  -- Ubuntu Users Mailing List Thread 

[http://ubuntuforums.org/forumdisplay.php?f=29](http://ubuntuforums.org/forumdisplay.php?f=29) -- Ubuntu Developement Mailing List
