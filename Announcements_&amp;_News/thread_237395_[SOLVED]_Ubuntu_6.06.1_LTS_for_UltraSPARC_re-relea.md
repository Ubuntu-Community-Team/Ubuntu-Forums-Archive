### Post by Adam Conrad on 2006-08-16
Due to an unfortunate hiccup in the CD build system, the ISO image for
Ubuntu Server on UltraSPARC had to be regenerated.  The original image
featured a set of mismatched kernel and module packages which would have
prevented the installation from completing fully.

We would like to make it clear that none of the other CD images have
changed, and this is not a new point release.  Only the Ubuntu Server
image for UltraSPARC had the defect mentioned above.

Our thanks go out to David Miller for noticing the problem with the
UltraSPARC image and for testing the regenerated image to be sure that
it works as advertised.
