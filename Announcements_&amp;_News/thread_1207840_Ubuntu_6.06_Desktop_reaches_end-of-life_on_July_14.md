As a general analogy, I suppose you could say that.

What a LTS point release is is a rebuild of the installation media using all of the current software updates. This allows for minimal software updates in the future, whereas if one was to install 8.04.0 at this point in time, there would probably be ~1000 package upgrades to do in order to get the system up-to-date.  If the same user was to install 8.04.3 right now, there would maybe be 25 updates - much less.
