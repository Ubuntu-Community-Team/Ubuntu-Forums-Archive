Kubuntu Breezy Release Candidate

[06 October 2005] [http://www.kubuntu.org/announcements/breezy-release-candidate.php](http://www.kubuntu.org/announcements/breezy-release-candidate.php)

The Kubuntu developers are pleased to announce the Breezy Release Candidate.

Kubuntu is a user friendly operating system based on KDE, the K Desktop
Environment. With a predictable 6 month release cycle part of the Ubuntu
project, Kubuntu is the GNU/Linux distribution for everyone.


Adept Package Manager
---------------------

This release features our new package manager, Adept, making searching for and
installing packages a breeze. The Adept Updater keeps your system up to date
with the latest changes and security fixes at the click of a button.

Adept is the first package manager to make use of debtags, an advanced
categorisation system for collections of packages, making it easy to find the
packages you are interested in.


Downloading Kubuntu Breezy Release Candidate
--------------------------------------------

The Release Candidate is available on install CD, live CD and dual install/live
DVDs for PCs, 64-bit AMD64 PCs, and PowerPC (Macintosh). The final Kubuntu
Breezy will be released in just one weeks time, so please test it out and send
us your feedback.

You can download the CD and DVD images from:

  * United States: [http://us.releases.ubuntu.com/releases/kubuntu/5.10/](http://us.releases.ubuntu.com/releases/kubuntu/5.10/)
  * Europe: [http://se.releases.ubuntu.com/kubuntu/5.10/](http://se.releases.ubuntu.com/kubuntu/5.10/)
  * UK and Rest of World: [http://releases.ubuntu.com/kubuntu/5.10/](http://releases.ubuntu.com/kubuntu/5.10/)

Please download using bittorrent if possible.

Note that some of these release candidate CD images are larger than 650
megabytes, and require CD media with sufficient capacity.


New in Kubuntu Breezy
---------------------

  * KDE 3.4.2
  * X.org 6.8.2 - with modular packaging
  * Adept and Adept Updater - Package Manager
  * KDE System Settings - Easier Control Centre
  * KDE Language Packs - now on the CD
  * KDE Bluetooth Framework
  * OpenOffice.org 2
  * GStreamer for Kaffine and AmaroK - media framework
  * Krita - image editor
  * KLaptopDaemon - now with hibernate support
  * Locate File Search - find files in Konqueror
  * Automounting - for CDs and memory sticks
  * Katapult - alt+space for program launching
  * Simplified Konqueror
  * Graphical startup with usplash

Off the desktop Kubuntu Breezy boasts Linux 2.6.12.6, PHP5, 64-bit PowerPC and
many other improvements.
