As we've been upgrading and bug fixing, many of you have been puzzled by an apparent bug in the posting order.

Well, it's not a bug.  The current default is now set to show the last post first, or, reverse chronological order.

If you don't like the order or style of how the posts/threads are appearing, you can change the defaults in your User CP.  :)
