The Ubuntu team is pleased to announce Ubuntu 10.04 LTS (Long-Term Support). This release incorporates the Desktop Edition and the Server Edition.  The Server Edition can be used on physical servers, on Ubuntu Enterprise Cloud (UEC), and on Amazon's EC2 public cloud.  Codenamed "Lucid Lynx", 10.04 LTS continues Ubuntu's proud tradition of integrating the latest and greatest open source technologies into a high-quality, easy-to-use Linux distribution.  We are also pleased to announce Ubuntu 10.04 Netbook Edition, which is not a long-term support release.
