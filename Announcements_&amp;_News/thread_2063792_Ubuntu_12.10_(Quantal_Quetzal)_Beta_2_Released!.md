The Ubuntu team is pleased to announce the final beta release of Ubuntu12.10 Desktop, Server, Cloud, and Core products.Codenamed "Quantal Quetzal", 12.10 continues Ubuntu's proud tradition of integrating the latest and greatest open source technologies into ahigh-quality, easy-to-use Linux distribution.  The team has been hard at work through this cycle, introducing new features and fixing bugs.With Ubuntu 12.10, Kubuntu, Edubuntu, Lubuntu, Xubuntu and Ubuntu Studio also reached Beta 2 status today.   These images will continue to have daily updates for the remainder of the release.Ubuntu Changes--------------Some of the new features available since Beta 1 are: * Quantal Beta-2 includes the 3.5.0-15.23 Ubuntu Linux kernel which    is based on the v3.5.4 upstream Linux kernel.  * Unity has been updated to version 6.6 which contains the new    default web application in the launcher, a new shopping lens,   improvements to the dash and multiple bug fixes. * GNOME has been updated to 3.5
