### Post by lloyd_b on 2009-09-11
Let's give this a try.  Assumptions:
x = multiply
/ = divide
+ = add
- = subtract
> = decimal point (period)
( and ) represent 9 and 0, respectively, not their mathematical equivalents.

adding some spaces around the operators for readability:
%&@ / ## x !# - &@( / $# x !>&$ + ^*) {=}

convert:
572 / 33 x 13 - 729 / 53 x 1.73 + 68

Applying standard order of precedence:

(572 / 33 x 13) - (729 / 53 x 1.73) + 68

225.33333 - 23.7957 + 68 =

269.5376
