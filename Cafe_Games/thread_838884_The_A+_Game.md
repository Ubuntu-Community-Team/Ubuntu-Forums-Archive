### Post by Th3Professor on 2008-06-23
Okay, I have no idea if anybody has ever played this version of this game... as far as I know I'm "inventing" it right here on the spot... though it's probably been done before. Here's the objective:

Round 1:
Go through the alphabet, first posting a single word (noun) that starts with the next letter of the alphabet (after the previous poster's word). When it gets to Z, start round 2.

Round 2:
Select 2 words, each with the same first letters and the next letter in the alphabet (after the previous poster), the first word is an adjective describing the second word, which is a noun. When it gets to ZZ, start round 3.

Round 3:
Select 3 words, each with the same first letters and the next letter in the alphabet (after the previous poster), the first and second words are adjectives describing the third word, which is a noun. When it gets to ZZZ, start round 4.

Round 4:
(etc.)

Example:
Round 1...
Post 1: Alligators
Post 2: Building
...etc.
Post 26: Zit
Round 2...
Post 27: Angry Ardvark
Post 28: Big Booger
...etc.
Post 52: Zealous Zebras
Round 3...
Post 53: Ancient Allocatable Apples
etc.

Make sense? :) No need to be too strict with type of word (adjective, noun, etc.)... most important thing is the 1st letter and # of words. :D
