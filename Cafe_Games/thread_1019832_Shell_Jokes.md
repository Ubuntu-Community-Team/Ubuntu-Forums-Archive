### Post by jenkinbr on 2008-12-24
From a windows command prompt:
```
C:\>windows
'windows' is not recognized as an internal or external command,
operable program or batch file.
```

On a linux prompt:
```
$ whatis
Whatis what?
```

```
$ whatis politics
politics: nothing appropriate.
```

**the above works for anything that is not a man page.
Speaking of man pages,...
```
$ man woman
No manual entry for woman
```

Didn't get this next one to display verbatim in bash, but:
```
$ drink <bottle; opener
bottle: cannot open
opener: not found
```

Remember to never remove apt, or you might get this if you run apt-get:
```
The program apt-get is currently not installed. You can install it by typing apt-get install apt
```

```
$ cat "can of food"
cat: cannot open can of food
```

```
$ make dead
make: *** No rule to make target `dead'. Stop.
```

That last one actually killed the facebook thread I started on this subject - hopefully it doesn't do that here. :)
