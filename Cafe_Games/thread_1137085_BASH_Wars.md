### Post by cheapie on 2009-04-25
OK. The object of this game is simple. It is to attack each other using BASH commands. You are currently in a yard surrounded by an electric fence, and on the other side of the fence is a symbolic link to /dev/null. The yard also has a pile of cookies in it.

For example, if I wanted to attack SlickRick, I would post something like this:

cheapie@ubuntu~$sudo apt-get install cable cable-end
[sudo] password for cheapie:
Done.
cheapie@ubuntu~$ln /dev/electric-fence /dev/cable-end-1
cheapie@ubuntu~$ln /home/SlickRick /dev/cable-end-2
cheapie@ubuntu~$
