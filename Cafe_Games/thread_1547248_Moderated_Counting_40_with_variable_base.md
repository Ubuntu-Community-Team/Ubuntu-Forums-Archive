We start at base 10.

Every user must post a number and the base it is in.

If a moderator resets the count when it is past 10 (base n), the base INCREASES BY ONE :o

If we complete a count, the base decreases by one :popcorn: and the count restarts

We win by getting the base to 4 (so 40 is not a number)!  Let's go!
