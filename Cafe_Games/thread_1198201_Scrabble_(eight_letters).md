### Post by HappinessNow on 2009-06-27
Rules:

    1. You can change two letters.
2. English only. 
3. You can add two letters.
4. The total letter count needs to remain eight.

    For example:

    Academic

    American

    (I took away letters D, C ; and added R, N.)

    Let the game begin with the word:

American
