1st  **gconf-editor** last  **./lmms** 

Obviously if they contain something you don't want to post, skip to the next/previous as the case might be.

For newer Ubuntu users who may not be able to find the .bash_history file, press Ctrl h key combo when in your /home/username directory and open the file with gedit (double left click should do it).

There is probably a command to run from the terminal that will output these entries, so someone may come up with that ;)
