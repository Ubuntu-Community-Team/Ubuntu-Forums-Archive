### Post by Excedio on 2009-08-20
This game was recently ported to Linux and it's pretty awesome. I would check out the demo if you're interested. Heck the game is only $20 too!
 
I love the way the game plays, seems more real with how you pick things up and such. Also, the game is great if you play with the lights off... ;)
 
Demos can be found here:
[COLOR=#ff0000]**[Penumbra: Overture Demo for Linux]("http://www.gamershell.com/download_24986.shtml")**[/COLOR]
[COLOR=#ff0000]**[Penumbra: Black Plague Demo for Linux]("http://www.gamershell.com/download_24990.shtml")**[/COLOR]
 
 
Just have to download them to a folder, any folder:
 
Right click on the file and select "Open With another application."
Then click the box near the bottom so you can do a Customer Command.
Type in "sh" (without quotes)
Click OK and the game will start to install.
 
Hope you guys like it as much as I do. :)
