OK, this is a forum game which is pretty simple.  The idea is you revive this thread from time to time, thus performing thread necromancy.  How does it work?  So glad you asked!

When the thread has been dead for longer than the previous time, you have shown that your necromancy powers are greater than the previous necromancer and therefore have the honour of being dubbed, THE UBERMANCER!  If it turns out the thread HASN'T been dead for longer than the last time, you're just a filthy corpse botherer.

This game slows down a lot :)

Make sure to note in your post how long the thread was dead for, mostly for simplicities sake.
