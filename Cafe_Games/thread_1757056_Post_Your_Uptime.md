### Post by Metallion on 2011-12-09
> **dargaud said:**
> All Kubuntu. I really wonder how the user count is done... It's probably more the tty count.

It's all users logged into a shell. That means it accumulates all gnu screen windows and all tabs in whatever terminal emulators you have open.

```

metallion@kimchi  (~) > uptime
17:24:28 up  6:37,  6 users,  load average: 0.05, 0.04, 0.08
metallion@kimchi  (~) > users
metallion metallion metallion metallion metallion metallion

```
