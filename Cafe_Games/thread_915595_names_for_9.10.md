### Post by elmer_42 on 2008-12-17
My personal favorite choice for the 9.10 release is Kleptomaniacal Kookaburra. In the spirit of having easy finals tomorrow, I just wrote a Ruby script to generate names for 9.10. I call it 910.rb. I'm pretty sure I did this a long time ago, but I can't find the topic it was in. Anyway, it will ask you how many names you'd like and then it'll generate that many names. Hope you enjoy it!
Example output:
```
$ ruby 910.rb 
How many names would you like?
3
Kaleidoscopic Komodo
Kingly Koi
Kinetic Kodkod
```
Script:
```
ka = %w(katie kafkaewque kaleidoscopic kamikaze kanny karmic kayoed keeking keen keepable kempy khaki kicking kicky killer kilted kind kindhearted kingly klutzy kinetic king-sized kinky kissable kissy kitschy kleptomaniacal knavish knightly knobby knotty kooky kurt kvetching)
kn = %w(kittie kakapo kalong kangaroo kaydid kea kelpie kestrel kid killdeer kingfisher kinkajou kite kitten kittiwake kittyhawk kiwi knight koala kodkod kodiak koi komodo kookaburra kouprey kowari kudu kraut)
puts "How many names would you like?"
ip = gets.chomp.to_i
ip.times do
		  puts "#{ka[rand(ka.length)].capitalize} #{kn[rand(kn.length)].capitalize}"
end
```
