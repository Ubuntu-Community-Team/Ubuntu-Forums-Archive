OK, so everybody has very likely seen the cool Scrabble thread where one letter is to be changed each post to make a new word.

Never mind that: <snip> I had to edit the mess I am replacing...


1. Take as many of the letters from the last post in any order you like.

2. Add **one** letter (restrain yourself to English alphabet please), or discard letters, if you want.

3. Post a new word, less than four letters may not count.


If you are positive the last poster posted an invalid word then (after, seriously, checking a dictionary) maybe you should work from the last valid English post.


Scoring is simple: Plus one point for every new word *you* form. Minus ten points if you post a word that has already been posted.

So if you want to score points as well as just play a word game then go nuts.
