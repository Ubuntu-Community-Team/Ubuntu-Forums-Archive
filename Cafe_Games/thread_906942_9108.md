### Post by tommynz1975 on 2008-09-01
from the playdeb site.

[HTML]www.playdeb.net[/HTML]

**introduction**

  This is a beta websie for the [Playdeb]("http://www.playdeb.net/") project. Playdeb is a gaming repository for Ubuntu - aimed to provide titles already available on  [getdeb.net]("http://www.getdeb.net/category.php?id=3") in an easier to install and update format.

**Note**: The look and layout of the website are temporary - we'll be getting a more serious website up soon.
  **install playdeb**

  To install Playdeb, you can: 

[LIST=1]
[*]Download and install updated [ apturl package]("http://www.playdeb.net/apturl_0.2.6-0%7Egetdeb0_all.deb").
[*]then,
[LIST]
[*]install **[ the playdeb package]("http://www.playdeb.net/playdeb_0.1-0%7Egetdeb1_all.deb")** (if you'll get a dialog asking you for confirmation, press **Y**)
[*]or add this line to System-Administration-Software Sources, Third-Party Software tab: [INDENT]deb mirror://www.getdeb.net/playdeb-mirror/hardy/// hardy/[/INDENT]
[/LIST]
 (three trailing slashes are intentional) 
[/LIST]
  **available games**

 See the [available games]("http://www.playdeb.net/available_games.html") section for those!

[/B]
