### Post by Poser55 on 2011-01-05
1. Bring up a terminal window
2. cd /var/games
3. sudo nano mahjongg.easy.scores (enter the root password when it asks)
4. delete the lines you don't want
5. ^x
6. y
7. enter
8. Repeat for any of the other layouts you want to erase
