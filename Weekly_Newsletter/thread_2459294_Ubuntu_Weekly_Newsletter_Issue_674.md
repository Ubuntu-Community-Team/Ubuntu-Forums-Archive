***[SIZE="4"]Extended Security Maintenance for Ubuntu 16.04 (Xenial Xerus) begins April 30 2021[/SIZE]***

&#321;ukasz 'sil2100' Zemczak reminds us of the release of Ubuntu 16.04 LTS which will soon reach the the end of it's five year standard support period and transition to Extended Security Maintenance (ESM). WIth a reminder to upgrade to Ubuntu 18.04 LTS before April 30, we are also told briefly of the Extended Security Maintenance option if we need to continue using Ubuntu 16.04. Supporting links for upgrade processes are provided.

***[SIZE="4"]First Look at Ubuntu’s New Desktop Installer[/SIZE]***

Marius Nestor reminds us it was only recently a new installer for Ubuntu Desktop was announced, and now we can actually have a look at what it may look like. Referring to Sebastien Bacher's new design we can see it as "web-based pages". With a description of how it is mostly similar to the current installer, we are also told of new pages and options we can select. We should see the new installer as a "technical preview" in Ubuntu 21.10.

[https://9to5linux.com/first-look-at-ubuntus-new-desktop-installer](https://9to5linux.com/first-look-at-ubuntus-new-desktop-installer)
