***[size="3"]Simos Xenitellis: Playing around with LXD containers (LXC) on Ubuntu[/size]***

Simos Xenitellis writes about LXD, another version of the LXC  (Linux containers) on his blog. Linux containers are an operating system level VMs used for running multiple linux systems on a single kernel. He explores the various containers available in LXD  through the LXC commands in the Ubuntu environment.

[https://simos.info/blog/playing-around-with-lxd-containers-lxc-on-ubuntu/](https://simos.info/blog/playing-around-with-lxd-containers-lxc-on-ubuntu/)

***[size="3"]Timo Aaltonen: Intel Graphics Gen4 and Newer Now Defaults to Modesetting Driver on X[/size]***
