**[size="5"]General Community News[/size]**

***[size="3"]The Ubuntu Online Summit starts [this] week[/size]***

David Planella reminds us that the Ubuntu Online Summit will run from 3-5 Nov 2015 from 2pm-8pm UTC which is three days of Ubuntu and Open Source discussions, tutorials, presentations, demos and Q&A's. Sessions include detail on app and scope development, cloud, community, convergence, core and more. The summit includes a keynote by Mark Shuttleworth, and a Q+A session with Canonical's CEO Jane Silber.

[http://davidplanella.org/the-ubuntu-online-summit-starts-next-week/](http://davidplanella.org/the-ubuntu-online-summit-starts-next-week/)

***[size="3"]UOS Session Roundup for Ubuntu Snappy Core[/size]***

Daniel Holbach writes about the Ubuntu Online Summit, running from 3-5 November, providing a schedule (time and synopsis) of sessions relating to Snappy Ubuntu Core. He notes that there might be last minute changes which could affect the schedule.

[https://daniel.holba.ch/blog/2015/10/uos-session-roundup-for-ubuntu-snappy-core/](https://daniel.holba.ch/blog/2015/10/uos-session-roundup-for-ubuntu-snappy-core/)

***[size="3"]Xenial Xerus is now open for development[/size]***

Matthias Klose writes that Xenial Xerus (16.04LTS) is open for development. In addition to discussing changes in 16.04, Matthias mentions that Python3(.5) is standard as Xenial is hoped to be Python2 free, a glibc update and more. Matthias also highlights some common errors made with Ubuntu development, such as necessary builds and the need to test in both Xenial and Wily versions.

[https://lists.ubuntu.com/archives/ubuntu-devel-announce/2015-October/001157.html](https://lists.ubuntu.com/archives/ubuntu-devel-announce/2015-October/001157.html)

***[size="3"]Canonical Promises Easy, Secure Containerized Apps for Ubuntu with LXD[/size]***

Christopher Tozzi, The VAR Guy, writes about the bundling of LXD, Canonical's open source containerized app framework with the latest release of Ubuntu 15.10. He shares that with ease of use and security in mind, Canonical promotes the solution as making it "easier than ever to set containerized apps in motion and step away" and promising "hardware-guaranteed security" for applications running in LXD. He concludes by writing, "Put together, LXD and Autopilot--both of which are homegrown tools from Canonical itself--offer streamlined orchestration and management for people using Ubuntu as a hosting platform."
