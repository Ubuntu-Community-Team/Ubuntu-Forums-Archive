***[SIZE="4"]Plasma Mobile 21.07 Released with Improvements for Kasts, Dialer, Shell, and More[/SIZE]***

Marius Nestor tells us the KDE Project has announced the release of Plasma Mobile 21.07, and provides us with details of many of the improvements contained therein. A link is provided to the release announcement, and suggests if we're using Plasma Mobile, that we update asap to enjoy the benefits of this upgrade.

[https://9to5linux.com/plasma-mobile-21-07-released-with-improvements-for-kasts-dialer-shell-and-more](https://9to5linux.com/plasma-mobile-21-07-released-with-improvements-for-kasts-dialer-shell-and-more)

***[SIZE="4"]VirtualBox 6.1.24 Released with Support for Linux 5.13 and Ubuntu Specific Kernels[/SIZE]***

Marius Nestor informs us of Oracle's release of VirtualBox 6.1.24, with support for the Linux 5.13 kernel series. A few Linux host fixes are mentioned, as well as other all-platform fixes, and a Solaris guest fix. A link is provided to the official website.

[https://9to5linux.com/virtualbox-6-1-24-released-with-support-for-linux-5-13-and-ubuntu-specific-kernels](https://9to5linux.com/virtualbox-6-1-24-released-with-support-for-linux-5-13-and-ubuntu-specific-kernels)

***[SIZE="4"]Canonical Pushes New Ubuntu Kernel Updates to All Supported Releases[/SIZE]***

Marius Nestor alerts us that Canonical have released Linux kernel updates for all supported Ubuntu releases that fix many security vulnerabilities. Listing the releases (which include 16.04 & 14.04 ESM) we are then provided a summary of the releases impacted and the CVEs fixed. Presented is how they could be mis-used, and why we should upgrade our systems asap.

[https://9to5linux.com/canonical-pushes-new-ubuntu-kernel-updates-to-all-supported-releases](https://9to5linux.com/canonical-pushes-new-ubuntu-kernel-updates-to-all-supported-releases)
