***[SIZE="4"]Ubuntu 22.10 Has a New Default Text Editor[/SIZE]***

Joey Sneddon tells us the daily builds of Ubuntu 'kinetic kudu' now include the GNOME text editor instead of gedit. We are reminded of this new GTK4 text editor app, and what the advantages of it are when compared with the 23-year old gedit. If we're not using Ubuntu 'kinetic kudu' we are told how we can add the app to our Ubuntu 22.04 LTS systems for a look.
