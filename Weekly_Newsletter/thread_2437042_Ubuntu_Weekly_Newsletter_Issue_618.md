***[SIZE="4"]KDE Plasma 5.18 LTS Released After A Lot Of Polishing, New Features[/SIZE]***

Michael Larabel reports the KDE community has released Plasma 5.18, which is a LTS release. It brings better integration with GTK/GNOME applications, a new opt-in feedback feature, an emoji selector has been added, and more. A video and link to KDE.org is provided.

[https://www.phoronix.com/scan.php?page=news_item&px=KDE-Plasma-5.18-Released](https://www.phoronix.com/scan.php?page=news_item&px=KDE-Plasma-5.18-Released)

***[SIZE="4"]KDE Applications Start Conquering the Windows 10 World[/SIZE]***

[https://news.softpedia.com/news/kde-applications-start-conquering-the-windows-10-world-529149.shtml](https://news.softpedia.com/news/kde-applications-start-conquering-the-windows-10-world-529149.shtml)

***[SIZE="4"]MATE Desktop 1.24 Release Arrives with a Slew of Improvements[/SIZE]***

Joey Sneddon alerts us to the new MATE desktop 1.24 release. After a brief history of the MATE desktop, we are soon led into MATE's new features. We are also told we'll see it in Ubuntu MATE 20.04 LTS. A link is provided to download the source code.

[https://www.omgubuntu.co.uk/2020/02/mate-desktop-1-24-release](https://www.omgubuntu.co.uk/2020/02/mate-desktop-1-24-release)
