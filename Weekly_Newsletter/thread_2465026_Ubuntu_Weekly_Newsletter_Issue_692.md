***[SIZE="4"]Ubuntu Touch OTA-18 Released - Still Using Ubuntu 16.04, 20.04 Migration Ongoing[/SIZE]***

Michael Larabel tells us that UBports have released Ubuntu Touch OTA-18, which is still using the Ubuntu 16.04 LTS base. UBports continues with their work on the Unity-derived Lomiri environment on the Ubuntu 20.04 LTS base; alas it wasn't ready for OTA-18. We are told of a number of improvements found in this release, and given a list of some of the devices supported.
