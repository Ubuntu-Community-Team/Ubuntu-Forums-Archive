***[SIZE="4"]KDE Developers Kick Off 2019 With More Fixes & Polishing[/SIZE]***

[https://www.phoronix.com/scan.php?page=news_item&px=KDE-Kicks-Off-2019-Improvements](https://www.phoronix.com/scan.php?page=news_item&px=KDE-Kicks-Off-2019-Improvements)

***[SIZE="4"]Intel Graphics Driver Working On FP16 Visual Support For Handling Wide Color Gamut[/SIZE]***

Michael Larabel  writes about some interesting work conducted this weekend in the Intel open-source Linux graphics driver space. The FP16 visual and framebuffer configuration support, together with other extensions should allow for Android wide-color gamut support and other wide-color scenarios. It's at an initial stage and currently under review, with Kevin Strasser, leading the work.

[https://www.phoronix.com/scan.php?page=news_item&px=Intel-FP16-Wide-Color-Gamut](https://www.phoronix.com/scan.php?page=news_item&px=Intel-FP16-Wide-Color-Gamut)

***[SIZE="4"]Bash 5.0 Release Packing Many Changes & A Lot Of Fixes[/SIZE]***

Michael Larabel reports that the GNU Bash 5.0 shell is now available. Bash 5.0 debuts  with quite a lot of bug fixes and brings a number of new features and improvements to better conform to POSIX specifications.

[https://www.phoronix.com/scan.php?page=news_item&px=Bash-5.0-Release-Candidate](https://www.phoronix.com/scan.php?page=news_item&px=Bash-5.0-Release-Candidate)

***[SIZE="4"]‘Cast to TV’ Lets You Stream Media From Ubuntu to Chromecast[/SIZE]***

Joey Sneddon reports of the GNOME extension that allows to cast content.  Joey provides an overview of the extension,  gives install instructions with links/screenshots, and usage pointers.

[https://www.omgubuntu.co.uk/2019/01/chromecast-tv-gnome-shell-extension](https://www.omgubuntu.co.uk/2019/01/chromecast-tv-gnome-shell-extension)

***[SIZE="4"]AMDVLK 2019.Q1.1 Radeon Vulkan Driver Build Now Available For Ubuntu[/SIZE]***

Michael Larabel advises on this first update from AMD for the VLK Vulkan 1.1.96 headers which adds GPU memory references, clean-ups, fixes, optimizations, and other changes. The package is now available for just the Vulkan driver and validated for at least 16.04/18.04 and possibly works for 18.10 also. Michael provides the download link to GitHub tag for those wanting to install this driver.

[https://www.phoronix.com/scan.php?page=news_item&px=AMDVLK-2019.Q1.1-Driver](https://www.phoronix.com/scan.php?page=news_item&px=AMDVLK-2019.Q1.1-Driver)

***[SIZE="4"]Ubuntu Touch OTA-7 Rolls Out With Keyboard Improvements, Browser Fixes[/SIZE]***

Michael Larabel reports that OTA-7 has been officially released. OTA-7 introduces themes for the on-screen keyboard, plus many fixes and updates. An update to libhybris will also aid the porting of Ubuntu Touch to more Android devices. More details are provided on the UBports link.

[https://www.phoronix.com/scan.php?page=news_item&px=Ubuntu-Touch-OTA-7-Released](https://www.phoronix.com/scan.php?page=news_item&px=Ubuntu-Touch-OTA-7-Released)

***[SIZE="4"]GNOME 3.31.4 Released As A Big Step Towards GNOME 3.32[/SIZE]***

Michael Larabel reports on this development snapshot, GNOME 3.31.4. Highlights include additions to the Epiphany web-browser, supports within the Evince library, supports for mouse gestures, and added zoom levels, additions to the WebKitWebDriver, and support for narrow/mobile mode support. There are also improvements for GNOME File Roller, GNOME Boxes, GObject Introspection, Vala and a new applications panel. More details are available from the provided release announcement and a roadmap to the 13 March release of GNOME 3.32.0.

[https://www.phoronix.com/scan.php?page=news_item&px=GNOME-3.31.4-Released](https://www.phoronix.com/scan.php?page=news_item&px=GNOME-3.31.4-Released)

***[SIZE="4"]PHP 7.3.1 Lands A Bunch Of Fixes For This Fastest PHP Release Yet[/SIZE]***

Michael Larabel reports of the first bug-fix release for PHP 7.3. In the convenient improvements and language additions in 7.3 is of note the improving PHP performance. Michael advises that with "PHP 7.0 now being end-of-life, it's a good time to upgrade". The many bug fixes are detailed in the PHP.net announcement.

[https://www.phoronix.com/scan.php?page=news_item&px=PHP-7.3.1-Released](https://www.phoronix.com/scan.php?page=news_item&px=PHP-7.3.1-Released)
