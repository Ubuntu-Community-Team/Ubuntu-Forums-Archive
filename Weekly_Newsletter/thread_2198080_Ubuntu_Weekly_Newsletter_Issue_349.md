**[size="5"]Other Community News[/size]**

***[size="3"]Ubuntu Kylin: Released the first Linux client for Chinese cloud storage services[/size]***

Jack Yu from Ubuntu Kylin team announces the official release of Kingsoft Kuaipan client. Kingsoft Kuaipan is one of the largest cloud storage service providers in China and provides 2TB free storage for each user. It supports clients for Windows, Mac, Android, iPhone, iPad, etc, but not Linux. Since August 2013, Ubuntu Kylin team started to cooperate with Kingsoft Cloud Cooperation and develop the Linux client. Now, Ubuntu/Ubuntu Kylin users can use it freely!

News: [http://www.ubuntukylin.com/news/shownews.php?lang=en&id=176](http://www.ubuntukylin.com/news/shownews.php?lang=en&id=176)

Download: [http://www.ubuntukylin.com/applications/showimg.php?lang=en&id=34](http://www.ubuntukylin.com/applications/showimg.php?lang=en&id=34)

More information about Kingsoft Kuaipan: [http://www.kuaipan.cn/](http://www.kuaipan.cn/)
