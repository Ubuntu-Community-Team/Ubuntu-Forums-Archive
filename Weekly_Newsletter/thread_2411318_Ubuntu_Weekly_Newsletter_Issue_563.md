**[SIZE="5"]In the Blogosphere[/SIZE]**

***[SIZE="4"]KDE Now Has Virtual Desktop Support On Wayland[/SIZE]***

Michael Larabel writes of the KWayland/KWin resolution for KDE virtual desktop support on Wayland, and as well a new UI for X11. The improvements include the digital clock widget, KDE Information Center's USB devices, wallpaper chooser, and more. A link is provided to Nate Graham's blog page for more details and  screenshots.

[https://www.phoronix.com/scan.php?page=news_item&px=KDE-Wayland-Virtual-Desktop](https://www.phoronix.com/scan.php?page=news_item&px=KDE-Wayland-Virtual-Desktop)

***[SIZE="4"]Ubuntu Gets Snappier Video Playback With Chromium Snap For VA-API Acceleration[/SIZE]***

Michael Larabel advises on the availability of a testing Chromium Snap that features VA-API video acceleration support for GPU-based decoding. The caveat is that you must have the supporting hardware/driver and for YouTube videos one must have the h264ify extension. Links are given to Olivier Tilloy's patch and to the binary snap.

[https://www.phoronix.com/scan.php?page=news_item&px=Ubuntu-Snaps-Chromium-VA-API](https://www.phoronix.com/scan.php?page=news_item&px=Ubuntu-Snaps-Chromium-VA-API)

***[SIZE="4"]Qt 5.13 Slated To Deliver Many WebAssembly Improvements[/SIZE]***

Michael Larabel reports that the WebAssembly for Qt 5.13 in the next few months is bringing many improvements. Features planned are "optional multi-threading support, system clipboard access, local file access, binary packages, and platform documentation.

[https://www.phoronix.com/scan.php?page=news_item&px=Qt-5.13-WebAssembly-Improvement](https://www.phoronix.com/scan.php?page=news_item&px=Qt-5.13-WebAssembly-Improvement)

***[SIZE="4"]Wine 4.0 Officially Released with Vulkan & Direct3D 12 Support, HiDPI on Android[/SIZE]***

[https://news.softpedia.com/news/wine-4-0-officially-released-with-vulkan-direct3d-12-support-hidpi-on-android-524647.shtml](https://news.softpedia.com/news/wine-4-0-officially-released-with-vulkan-direct3d-12-support-hidpi-on-android-524647.shtml)

***[SIZE="4"]Debian & Ubuntu Fix Man-in-the-Middle Attack in APT Package Manager, Update Now[/SIZE]***

[https://news.softpedia.com/news/debian-ubuntu-fix-man-in-the-middle-attack-in-apt-package-manager-update-now-524662.shtml](https://news.softpedia.com/news/debian-ubuntu-fix-man-in-the-middle-attack-in-apt-package-manager-update-now-524662.shtml)

***[SIZE="4"]GNOME 3.32 Desktop Environment to Launch with a "Radical New Icon Style"[/SIZE]***

The GNOME 3.32 release of the open-source desktop environment is used by numerous Linux-based operating systems, including Ubuntu.

[https://news.softpedia.com/news/gnome-3-32-desktop-environment-to-launch-with-a-radical-new-icon-theme-524663.shtml](https://news.softpedia.com/news/gnome-3-32-desktop-environment-to-launch-with-a-radical-new-icon-theme-524663.shtml)
