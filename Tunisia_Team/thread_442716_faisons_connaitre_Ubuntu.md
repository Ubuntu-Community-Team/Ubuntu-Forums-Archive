Ubuntu = beauté

Une expériense personnelle pour tous les Windowsiens, j'ai un core 2 duo avec Windows Vista, c'est vrai par rapport à XP c'est plus beau et plus développé, mais sur ma machine, on a l'impréssion que c'est un pentium 4,  Vista allourdit énormement l'ordinateur, en plus les virus qui arrivent d'à peu prés partout rendent le travail impossible. Personellement, je dirais que Linux est l'OS qui exploite le mieux le matériel. On devrait l'utiliser dans les écoles pour apprendre l'informatique sur de bonnes bases. En effet pour des gamains il fudrait un système sécurisé et performant sans messages d'erreur incompréhensibles, alos que dans windows les messages imprévisibles de plantage laissent les enfants impuissants et le flux énorme de virus pousse les grands à effectuer du formattage inutil tous les mois. 

Pour presque toutes les personnes, (Linux=difficile) comme l'a mentionné mon ami "Dark_Prisoner".
