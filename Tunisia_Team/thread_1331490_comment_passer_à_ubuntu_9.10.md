### Post by RachedTN on 2009-11-20
Bonjour,

Si vous avez une bonne connexion internet, vous pouvez tout simplement faire une mise à jour, voici la procédure :
1- Système -> Administration -> Gestionnaires de paquets Synaptic -> Mise à jour -> Mise à niveau de la version : choisir : Versions normales 
2- Fermer la fenetre :)
3- Système -> Administration -> Gestionnaire de mise à jour : La nouvelle version 9.10 est disponible : juste cliquez sur le bouton Mettre à niveau.

Voilà :)
