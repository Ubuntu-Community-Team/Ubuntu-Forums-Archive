### Post by alaya.zied on 2010-03-24
[SIZE="4"]En s'inscrivant dans notre ML ([page de la ML]("https://lists.ubuntu.com/mailman/listinfo/ubuntu-tn")) quelques membres se sentent parfois gêné par le nombre de mails.
Une solution simple à appliquer avec tous ML est de creer des filtres pour les mails.

Voici comment faire sous Gmail:
[INDENT]1- cliquer sur le lien qui affiche les détails de mail (voir l'image)
2- cliquer sur filtrer les messages
3- faite suivant, puis choisir l'option archiver (skip inbox) + appliqué le libellé "Ubuntu-tn" (il faut l'ajouter bien sûr)[/INDENT]

Ainsi, tous nos mails ne seront plus afficher dans la boite principale.

Pour les utilisateurs de Yahoo, juste selectionner le mail, cliquer sur Action, puis faite 'filtrer les mails semblable'.

Voilà, j'éspère que ça aidera à mieux exploiter notre ML (et les autres ML aussi) :popcorn:[/SIZE]
