### Post by Ibidem on 2010-01-01
Remove Compiz--And you like Windows 7?

Be aware that transparency requires desktop effects (compiz).
That said, you can uninstall Compiz in Synaptic.

You might try wbar or AWN (slows things down on a few systems) for a Dell Dock replacement; try looking [here]("http://ubuntuforums.org/showthread.php?t=1335974&highlight=wbar")
I would suggest that you post/look in [Absolute Beginner Talk]("http://ubuntuforums.org/forumdisplay.php?f=326")  if you have only added one program.
I can't tell  how far you've gotten with Ubuntu, so I apologize if this is for a diffferent level of user.
Hope this helps,
