### Post by mark.a.nicolosi on 2007-02-25
The start menu you refer to is actually the boot loader menu. Memtest isn't a DOS program, it doesn't run under an operating system. Check out [http://www.memtest86.com]("http://www.memtest86.com") for more information on it.

You might have use for [FreeDOS]("http://www.freedos.org") to run your DOS programs under, but this has nothing to do with Ubuntu or Linux. Good luck.
