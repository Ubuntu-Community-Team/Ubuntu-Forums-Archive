### Post by jpietari on 2007-10-18
Are you running Ubuntu 7.10 (RC, Beta)?

I ran into this problem in Ubuntu 7.10 Beta (number?).  When I clicked on the menu bar it would appear.  The problem seems to have been fixed when I installed the updates.

If you are still having the problem, try re-creating your taskbar.  This can be done by right-clicking the taskbar and selecting "add panel".  After that, you can right-click the new panel and select "add to panel".  

Hopefully this helps!
