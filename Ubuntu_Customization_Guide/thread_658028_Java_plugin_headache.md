### Post by NilsE on 2008-01-04
The Java that is in the repositories will work so un-install what you have and install the Java6 from the repositories.

As a matter of fact you should be able to just install the ubuntu-restricted-extras and it will install Java and a few other restricted packages like ms fonts, gstreamer plugins etc. If that does not do it then install anything with java6 in it and it will also pick up other dependencies.
