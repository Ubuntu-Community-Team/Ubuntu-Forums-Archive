### Post by zedmaster on 2007-02-05
Hello all!  I just installed Beryl on my AMD 64 Xubuntu machine.  Needless to say, it's really quite amazing, especially given it doesn't crash now that I've switched back to my onboard ATI.  After reading the various Ubuntu forums for a while, I got the impression ATI would be much less reliable than NVIDIA, so I picked up a GeForce 7300GS before I tried Beryl.  Anyway, believe it or not the GeForce froze up after about a minute, never fail.  Played with it a for a few days, no love.  Anyway, believe it or not the ATI X200 works flawlessly!  No crashes yet.  Can't say I'd suggest the 7300 GS to anyone who wants to run Beryl.

Anyway, here's my problem:  When I'm running Beryl on Xubuntu, I get strange behavior on my desktop.  I'm not talking about the windows wobbling like Jello, either!  What happens is when I click an icon on the desktop, the entire desktop actually shifts up or down a few inches.  Wallpaper and everything.  I believe this is a setting somewhere in Beryl.  Does anyone have any idea which one it is?

### Post by aidanr on 2007-02-05
[http://bugs.beryl-project.org/ticket/586](http://bugs.beryl-project.org/ticket/586)

right click -> desktop settings -> uncheck/check allow xfce to manage the desktop
