### Post by TheFu on 2018-08-22
Without the email headers, I won't believe it came through email.
Anyone can put any "FROM" address into an email.  Validation of that isn't part of the specification for a number of good reasons. For $5, you can run an email server and send messages from GOD if you like.

If it was a private message, you should use the *resolution center* to deal with that specific login.  The mods here are excellent at cleaning up spammy accounts, but there's no way to be inviting for new users AND block every spammer before they get a chance to send a few messages.

So - if it was email, please post the full headers.
