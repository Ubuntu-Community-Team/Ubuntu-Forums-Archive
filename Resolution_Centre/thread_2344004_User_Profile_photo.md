### Post by g33zr on 2016-11-21
My user profile photo appears in my profile but not when I post to a forum. The photo is less than 100 kb and is only 100x80 pixels. Please advise.

---

### Post by coffeecat on 2016-11-21
Profile pictures are not the same as avatars. Profiles pictures only appear in your profile page. If you want an image to appear in each of your posts you need to set an avatar.
