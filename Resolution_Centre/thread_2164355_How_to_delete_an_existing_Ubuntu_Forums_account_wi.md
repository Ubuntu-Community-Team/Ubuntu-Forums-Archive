### Post by coffeecat on 2013-07-31
As far as I can see, you are asking a question for someone in irc who does not have a forum account but who is, in turn, asking a question for yet another person.

First thing: the Resolution Centre is for asking an admin to deal with issues concerning **your** account. If you need any help with your account, please say so and I'll see what I can do.

The second thing is that we do not delete accounts. If they are not needed we disable them, which involves removing personal details and adjusting settings so that the disabled account cannot be logged into. Perhaps you'd like to pass that message down the line! If the person with the account doesn't want to set up an Ubuntu One account in order to login and make their request, they can contact the forum admins directly on the Forum Council mailing list. The email link is in this page:

[https://wiki.ubuntu.com/ForumCouncil](https://wiki.ubuntu.com/ForumCouncil)
