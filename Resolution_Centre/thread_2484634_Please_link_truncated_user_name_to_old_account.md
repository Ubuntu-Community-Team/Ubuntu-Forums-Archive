The 25-character limit for usernames has been around for many years, probably since the forums were first set up. Also, there is no account on the forum database with the username i41bktobiu5q-launchpad-net-hpgu96fjbrq6, nor is there any username in our database that contains the string hpgu96fjbrq6, and there is only one account that starts with the string i41bktobiu5q, namely the one you are posting from: i41bktobiu5q-launchpadnet

The forum software only allows a particular email address to be associated with one forum account. Therefore, if you had an older forum account with the email that you have set as preferred in your Ubuntu One account, you would have logged into that account, and not into a newly created account.   

So, without accurate identification of any earlier forum account, there's nothing I can do.

### Post by coffeecat on 2023-03-05
Google? You don't log into ubuntuforums with Google. And, "Launchpad still recommended?" Certainly. But for bug tracking, etc, but you don't log into the forum from launchpad.


If you are logged out of the forum and go to our main page you'll see a button top-right that says, Login with SSO. Click that and you'll be taken to [https://login.ubuntu.com/](https://login.ubuntu.com/) Canonical's site for SSO. From there you log into your forum account. Your forum account is linked to a logon.ubuntu.com account and that is what you logged in with when you created your forum account.


Perhaps you are confusing us with askubuntu.com, which is a Stack Exchange site.
