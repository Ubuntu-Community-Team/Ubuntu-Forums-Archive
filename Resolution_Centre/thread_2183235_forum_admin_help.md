### Post by coffeecat on 2013-10-24
OK - I see what you've done now. You accidentally created a duplicate forum account from the Ubuntu One account with the gmail. We don't normally edit Resolution Centre posts but I've snipped the gmail from your last post for your protection, so that it isn't picked up by spammers, but it is still readable by forum admins. I've done the following:

[list][*]Removed the gmail from the duplicate account, disabled the account and unlinked it from Ubuntu One.
[*]replaced the email in this account with the gmail, and unlinked it from the hotmail Ubuntu One account.
[/list]

If you now log out and then log in from the gmail Ubuntu One account, you should log back into this account from the gmail Ubuntu One account, achieving (I hope) what you wanted.

This will still leave you with the older hotmail Ubuntu One account. If you want that deleted you will have to contact the Ubuntu One admins. We do not administer Ubuntu One.
