Maintaing the obvious and easy-to-use interface. Our primary source of inspiration is currently Python, Google's Dart, Ruby and a number of elements from jQuery which we want to implement into a real language in a "real" way, instead of just having a layer on top.

This is exactly why we are spreading these polls on forums and message boards around the internet. In order to see what the most people would like to see in a language like this, and how we can do the best to achieve this optimal, easy-to-use scripting language!
