### Post by coffeecat on 2013-10-03
You have two Ubuntu One accounts, one linked to this forum account, and using the me.com email, the other linked to the Euanbuntu account and currently using the gmail. You have managed to log into both forum accounts today using each of the Ubuntu One accounts. So you are able to log into the Euanbuntu forum account. The only problem for you is that both the Euanbuntu account and the linked Ubuntu One SSO account have the "wrong" emails for your purpose.

First - remember that once an Ubuntu One account and a forum account are linked it really doesn't matter what emails are used in them, neither do the emails have to match. Email matching is only necessary for initial linking of the two accounts to take place on first login. This is all explained in [this sticky]("http://ubuntuforums.org/showthread.php?t=2164051") if you need more detail. 

So - I suggest you use the Ubuntu One account linked to the Euanbuntu forum account to login. If you still really need the me.com email registered to that Ubuntu One account, you'll need to remove it from the other Ubuntu One account before you can use it in the Ubuntu One account you need to use. You may need to contact an Ubuntu One admin if that doesn’t work - I really don't know. We don't administer Ubuntu One. Once you've sorted out your Ubuntu One accounts, contact me by pm - you won't be able to post to this thread as Euanbuntu - and I can help you sort out the email registered to the Euanbuntu account.
