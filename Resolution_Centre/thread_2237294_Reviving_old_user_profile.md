### Post by coffeecat on 2014-08-01
We can help you get back into your old account, which was registered with a hotmail email rather than the gmail you used in Ubuntu One when you created this duplicate account. Set the hotmail as preferred in your Ubuntu One SSO account and post back to confirm that you have done this. A forum admin will then disassociate this account from Ubuntu One, which will enable you to log into the Deer Hunter account.

If the hotmail is no longer active, and you cannot set it as preferred in Ubuntu One, post back and we'll do this another way.
