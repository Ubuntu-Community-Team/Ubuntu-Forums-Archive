I am not under any obligation to allow Kilz to distribute Swiftfox.  My only obligation under the MPL that governs Firefox source code is to provide him a source package which is freely distributable.  I will be providing a link to Kilz for the Swiftfox source package within 12 hours and have requested that he remove the binary from his web server.  Any assistance the Ubuntu forum staff can provide in assuring this request is followed through on would be greatly appreciated.
