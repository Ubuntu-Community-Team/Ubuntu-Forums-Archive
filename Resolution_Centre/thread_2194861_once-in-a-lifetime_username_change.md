There already is, but it only applies to people creating a new account from SSO login. The Ubuntu One Personal Data request page permits unticking the Full name field. If a new user keeps the Full name field ticked, the system will base the forum username on the Ubuntu One name. This doesn't apply to you because you created your forum account in 2007, long before SSO login, and you chose the original forum name, which does not get changed by using SSO login.
