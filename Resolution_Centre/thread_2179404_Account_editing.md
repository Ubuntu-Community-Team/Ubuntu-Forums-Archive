### Post by Elfy on 2013-10-08
Not at all.

The script that runs to move people from one  usergroup to another doesn't read posts, and certainly wouldn't differentiate between one that said 'bump', 'this is not a love song' or indeed 'anything else' even if it could.

What it does do is run once an hour - so if you're 10th post was just after the script had run you'd need to wait till the next run. 

If you make the 10th post and then the script runs - but you'd looked just before - well I'm sure you get the picture :)
