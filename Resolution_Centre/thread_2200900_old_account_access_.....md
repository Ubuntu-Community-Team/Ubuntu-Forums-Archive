### Post by coffeecat on 2014-01-21
The email registered to the eudemus account did not match the one you used in Ubuntu One, hence the new forum account. Both email addresses have the same domain, but different local parts. If the email you used in the eudemus account is still active, please set it as preferred in your Ubuntu One account and the post back to confirm you have done this. An admin will then disassociate this account from Ubuntu One, which will enable you to log into the eudemus account.

If the email is not still active and you cannot set it as preferred in Ubuntu One, post back anyway and we'll do this another way.
