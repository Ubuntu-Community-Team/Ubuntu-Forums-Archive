### Post by coffeecat on 2013-08-04
I've disassociated this account from Ubuntu One - this needs admin action so it wasn't a thing you could do yourself.

> **sage-sageth said:**
> One additional question: On login.ubuntu.com, I added all three emails that I use for forums.  Do I need to select the proper "preferred" email that I used for the forum before? Or as long as one of the emails is the valid one here, can they still be merged?

The preferred email on Ubuntu One needs to be the same as the email in your old account for the two accounts to be associated. If you can remember which this is, change the email in Ubuntu One and login again and you should be taken to your old account. After which you can change the email in either account without breaking the association.

If you can't remember which, make sure the preferred email in Ubuntu One is the one you had set when you logged into this sage-sageth account - we can disassociate again - and post back so that we can look into this further. If you do need to post back as sage-sageth, please let us know the username of your old forum account.
