### Post by Bucky Ball on 2008-12-23
Love your work. You might want to start here.

[https://wiki.ubuntu.com/LiveUsbPendrivePersistent](https://wiki.ubuntu.com/LiveUsbPendrivePersistent)

These methods could apply to 8.10 version of any ubuntu distro (xubuntu might be a good one for lower end/older hardware) but best experiment. You could try compiling a kernel with just the packages you need and the kernel also, though this is a bit more complex and would be a learning curve if you're up to it. That would be one way of creating a version specific to your exact needs using existing open source software packages.

Good luck and have a look on distro watch, do a google search for 'open source distibutions distros' or somethings along those lines.

Another appoach might be to search for packages specific to your needs on an existing install in 'Applications->add/remove' and synaptic package manager, customise that install removing the packages you won't need, then work out how to get that onto a usb dongle. There will be a way.

You could also use AptOnCD to save your packages to usb or cd.
