### Post by coffeecat on 2014-04-25
Yes, it does have to be via Ubuntu One SSO as described here:

[http://ubuntuforums.org/showthread.php?t=2164051](http://ubuntuforums.org/showthread.php?t=2164051)

The email used in the davidmohammed account was a different gmail from the one you used in Ubuntu One. Set the different gmail as preferred in your Ubuntu One account and then post back to confirm that you have done this. I or another forum admin will then disassociate this account from Ubuntu One which will enable you to log into the davidmohammed account from your Ubuntu One SSO account.

If for any reason you are unable to set the correct email in Ubuntu One, post back and we'll find another way to do this.
