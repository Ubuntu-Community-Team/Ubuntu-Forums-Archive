### Post by bapoumba on 2011-07-19
Yes, we know.
The Archives have been restricted to logged in users because we had complaints that old fixes from old threads were breaking user systems.
[http://ubuntuforums.org/showthread.php?t=1799098](http://ubuntuforums.org/showthread.php?t=1799098)

This is a good compromise. Indexing bots will eventually forget about the Archive section now they are not allowed to index it. It'll take some time before these threads disappear from searches, they will, eventually.
