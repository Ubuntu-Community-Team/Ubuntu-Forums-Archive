### Post by Mark Giesler on 2007-09-06
I just wanted to introduce myself. I'm from Madison (way down on the Ohio river), although I'm currently working in Indy (Quality and FDA regulatory consultant). I toyed briefly with RedHat and Damn Small Linux (very, very briefly), but Ubuntu is the 1st Linux system with a small enough learning curve to really hook me (and the eye candy possible with Ubuntu isn't bad either). 

In a past life I was very briefly a Windows programmer (MCSD), but quickly became frustrated with the Microsoft meglomania. I haven't done any C programming in years, but the whole open license thing is so appealing that I may have to dust off the old books and start trying to learn something about the kernel.

Anyway, I'm a raw green user, but very little scares me (I've already had to reinstall Feisty Fawn once). I'm glad to see a Linux package that has enough of a GUI interface that the average user won't recoil in horror.

I currently run a Vista laptop, an Ubuntu 7 laptop, a Windows 2000 server, and a very, very old laptop running Win98, which will probably get converted to some flavor of Linux soon.

I'm looking forward to getting to know everyone here! :)

Mark
