### Post by scrooge_74 on 2007-02-25
if you are loosing the DNS you can prepend the DNS servers in 

/etc/dhcp3/dhclient.conf

add a line which says prepend domain-name-servers vvv.xxx.yyy.zzz

for network interfaces check what you have in /etc/network/interfaces

and set your correct network card or comment out does that you dont have or use

Hope it helps
