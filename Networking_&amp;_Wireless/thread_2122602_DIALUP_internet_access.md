I've never had to resort of dial-up networking so have no experience.  The biggest problem I've seen reading posts and such is hardware support.  I believe the common internal "winmodem" is supported poorly if at all.  People with external "hardware" modems using Rockwell chip sets seemed to have the easiest time.  There is some community documentation available but those I'd found with a quick google search were somewhat dated.  Good luck, I hope you're able to work it out.
