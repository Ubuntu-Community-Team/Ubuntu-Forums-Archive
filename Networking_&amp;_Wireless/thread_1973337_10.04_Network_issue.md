found it  in case anyone else has the same problem...
its really stupid...

```

auto lo
iface lo inet loopback
	post-up iptables-restore < /etc/iptables.up.rules
auto eth0
iface etho inet dhcp

```

Make sure you  dont forget **o**  and **0** are not the same
