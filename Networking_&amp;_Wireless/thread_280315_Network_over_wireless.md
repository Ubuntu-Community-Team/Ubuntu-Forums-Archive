### Post by wieman01 on 2006-10-19
You have to install either "NFS" or "SAMBA" while "SAMBA" is more used if you need to connect with Windows clients in the network. In a pure Linux environment "NFS" will be the better choice as it has a better performance. 

SAMBA: [http://kubuntu101.blogspot.com/2005/11/how-to-share-files-using-samba.html]("http://kubuntu101.blogspot.com/2005/11/how-to-share-files-using-samba.html")
NFS: [https://help.ubuntu.com/community/SettingUpNFSHowTo]("https://help.ubuntu.com/community/SettingUpNFSHowTo")
