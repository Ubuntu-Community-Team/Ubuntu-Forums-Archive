### Post by noob12 on 2007-10-01
Follow this path of menu selections: System | Administration | Network

A panel pops up.  

On the general tab select your wired connection.  Hit Properties. Set the address, subnet and gateway there.  OK.

On the DNS tab, set your name servers.
