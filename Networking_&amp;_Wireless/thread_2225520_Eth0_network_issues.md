### Post by papibe on 2014-05-21
Hi izzydojo.

I don't see the gateway field. Without it there's no default route to the rest of the world.

Set it to point to your modem or router that connects the Internet. For instance:
```

iface eth0 inet static
    address 192.168.1.140
    netmask 255.255.255.0
    network 192.168.1.0
    **[COLOR="#000000"]gateway 192.168.1.1[/COLOR]**
    broadcast 192.168.1.255
    dns-search example.com
    dns-nameservers 8.8.8.8 8.8.4.4
```
Then, restart the network
```
sudo service networking restart
```
Hope it helps. Let us know how it went.
Regards.
