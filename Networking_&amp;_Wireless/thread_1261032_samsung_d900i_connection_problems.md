### Post by prshah on 2009-12-20
> **folabiklan said:**
> samsungmobile puk code which apparently is four digits or i dunno.

Apologies for the thread necromancy:

the PUK (Pin Unblock Key) is usually 5 digits; and has to be got from the provider whose SIM is in the phone. It is to prevent misuse of the SIM. It is usually only required when the PIN of the SIM is entered wrongly 3 (5?) times.

Change the SIM, and you will not need a PUK code; though some phones ask for a security code when the SIM is changed.
