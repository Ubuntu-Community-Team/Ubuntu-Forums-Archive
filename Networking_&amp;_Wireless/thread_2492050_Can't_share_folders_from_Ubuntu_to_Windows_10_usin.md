### Post by SeijiSensei on 2023-10-29
Get rid of the space in the folder name for starters. Replace it with an underscore. Any better?

Unix treats spaces as delimiters. To use a file name with an embedded space requires wrapping the file name in single quotes or "escaping" the spaces with backslashes. It's easier to avoid the problem entirely by not using spaces in file names.
