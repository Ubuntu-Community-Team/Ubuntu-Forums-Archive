### Post by zika on 2009-06-06
If You want to get NetworkManager out of the loop Ypou have to change managed= line in /etc/NetworkManager/nm-system-settings.conf file:```
[main]
plugins=ifupdown,keyfile

no-auto-default=**aa:bb:cc:dd:ee:ff**,

[ifupdown]
managed=[COLOR=Red](false[/COLOR] to [COLOR=DarkGreen]true[/COLOR], as far as I remember)
```and add lines in /etc/network/interfaces like (choose if You want DHCP or static):```
# The loopback network interface
auto lo
iface lo inet loopback

# The primary network interface DHCP
auto eth0
iface eth0 inet dhcp

# The primary network interface static
#auto eth0
#iface eth0 inet static
#    address **xxx.yyy.zzz.www**
#    netmask **yyy.www.zzz.xxx**
#    gateway **sss.ppp.qqq.ttt**  
#    broadcast **aaa.bbb.ccc.ddd**
#    mtu **1444**
``` and /etc/resolv.conf:```
#search www.google.com
nameserver **xxx.yyy.zzz.www**
nameserver **www.sss.ddd.eee**
nameserver **208.67.220.220**
nameserver **208.67.222.222**
```(bold is machine dependent stuff)

But, You can achieve all of this through NetworkManager also.
