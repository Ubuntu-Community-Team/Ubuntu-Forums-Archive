```
        dns-nameservers 150.164.24.5 150.164.255.232

=============

cat /etc/resolv.conf
# Dynamic resolv.conf(5) file for glibc resolver(3) generated by resolvconf(8)
#     DO NOT EDIT THIS FILE BY HAND -- YOUR CHANGES WILL BE OVERWRITTEN
nameserver 150.164.24.5
nameserver 150.164.255.232
nameserver 127.0.0.1
search icb.ufmg.br
================

```


The address 127.0.0.1 should not be listed last but (if you have a local nameserver running) first. When that address is registered by a local nameserver such as dnsmasq it does appear first; that it appears last in your resolv.conf file indicates that there is some misconfiguration. Perhaps you have "nameserver 127.0.0.1" in /etc/resolvconf/resolv.conf.d/tail. Remove it from there.
