### Post by herbster on 2008-04-02
Enter the user name and password of your Ubuntu install.

You will need the FS driver to view ext3 (linux) partitions: [http://www.fs-driver.org/download/Ext2IFS_1_11.exe](http://www.fs-driver.org/download/Ext2IFS_1_11.exe)

Install that (in Windows) and restart your computer.
