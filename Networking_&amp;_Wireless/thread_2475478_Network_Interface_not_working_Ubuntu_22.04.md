### Post by glenp2 on 2022-05-28
My network interface keeps breaking on my Librem Mini V2 running Ubuntu instead of PureOS.

It appears to break if I suspend the computer and then wake it up or otherwise leave it running overnight.  Or if I insert a suspected malware USB before turning Auto Run Software Off.

This can be fixed with the following work around.

Settings (top right) --> Cog --> Search bar (top left of Settings window) --> Search and find Network --> Under the "Wired" heading on the right click the Cog --> Now click the big red button that reads "Remove connection"  -->  DO NOT ATTEMPT TO RECONNECT UNTIL AFTER A REBOOT --> Reboot

It should now be fixed.
