### Post by monton on 2009-11-21
I went to Synaptic Package Manager and downloaded wicd to replace network manager. SPM removed the network manager and installed wicd. Rebooted and went to places - network - windows network - and there was my home network/workgroup was there! Was able to access my wife's Mac and my Windows computer. So far, all is well.
