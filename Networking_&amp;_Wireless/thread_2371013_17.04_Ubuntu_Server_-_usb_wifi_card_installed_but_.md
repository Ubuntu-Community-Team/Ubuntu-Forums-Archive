### Post by chili555 on 2017-09-09
Please detach the USB wireless. Next, amend your /etc/network/interfaces file to:```
auto lo
iface lo inet loopback
 
#auto enp0s31f6
iface enp0s31f6 inet static
        address 192.168.10.9
        netmask 255.255.255.0
        gateway 192.168.10.1
        dns-nameservers 8.8.8.8, 8.8.4.4
 
auto wlp61s0
iface wlp61s0 inet static
        address 192.168.10.10
        netmask 255.255.255.0
        gateway 192.168.10.1
        dns-nameservers 8.8.8.8, 8.8.4.4
        wpa-ssid #####
        wpa-psk wxyz*
        
 
```...where wxyz* is the WPA2 password in clear text, not any sort of scrambled obscuration.

Reboot with the ethernet and the USB disconnected and check:```
ifconfig
ping -c3 www.ubuntu.com
```
