### Post by OrelEagle on 2012-02-03
Hello,

I own a usb 3G dongle which I was using with my laptop under Ubuntu 10.10 succesfully. Since 11.04 I couldn't make it work anymore, it wasn't recognised at all (not even as usb stick). 

I tried a lot of things and I finally got the solution! I thought I would share here, as I didn't find any documentation about that.

The fix was very simple: re-install modemmanager! I used Synaptic for that, although it can probably be achived with the command line.

After that, when I plug the stick in, I get an additional section in the network menu to connect to mobile broadband.
