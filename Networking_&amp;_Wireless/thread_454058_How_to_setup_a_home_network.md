### Post by jetsam on 2007-11-14
I came across this link somewhere on these forums and found it pretty useful as a template for setting up a home server.  I didn't follow it exactly; I've probably set up about a third of the services the article talks about:

[[COLOR="RoyalBlue"]http://www.bit-tech.net/bits/2007/06/05/build_your_own_server/[/COLOR]]("http://www.bit-tech.net/bits/2007/06/05/build_your_own_server/")

Samba was the hardest part to get working the way I wanted it to; expect to spend some time on this if you need mixed networking with WIndows.  SWAT makes configuration a little bit easier; if you use SWAT, you need to enable the root account on your server and log into SWAT as root in order to use it fully.  

The process has been tricky in places, but never horribly so, and what hurdles I've had so far were solvable by resorting  to forum searches or documentation.
