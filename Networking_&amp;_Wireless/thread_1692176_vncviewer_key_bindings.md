Hi,
        I am trying to view my ubuntu desktop using tightvnc. I have installed tightvnc on the desktop and started a vncserver on display 1. 
         
        I am able to connect to it using another ubuntu machine, but the problem is whenever i access the server the key bindings are messed up, which is very annoying.
for example.. whenever i press the character D the show desktop operation is performed.

I would like to change that and make Win+D as my show desktop key in vncviewer, but I do not know where to change these settings.

### Post by ultimateabhi on 2011-02-22
Well  I found the answer and I'm posting my solution in case someone finds it useful...

when logged in using vncviewer go to System>Preferences>Keyboard Shortcuts and disable/change the "Hide All normal windows and set focus to the desktop" action key in Window Management to something else.. :)
