### Post by tagra123 on 2006-08-22
If you are connecting with an ethernet cable then this should be simple.

Plug in the cable.

From the Panel Menu choose: System > Administration > Networking

Click on Ethernet connection and then Properties.

Enable this connection should be marked

Under configuration select DHCP.

Click OK

Click Activate if its not already activated and then click on OK.

You should be able to use the internet browser now.
