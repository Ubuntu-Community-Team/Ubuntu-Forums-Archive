### Post by dgriff2 on 2007-01-02
Just thought I would post the problems I had with getting my wireless to work. I am dual booting on a Toshiba m45 laptop with the Pro Wireless 2200 built in. I scoured these pages looking for help. I tried every sudo under the sun, at least those I could find within the last two days. I was thinking about giving up on Ubuntu until I tried something, and I don't know anything about this (translation n00b) but my wireless worked fine in Win XP Pro. Auto detected everything. No problems. But Ubuntu was a different story. This may not work for anybody else, but in window, I have my SSID and password set to certain text and my security is enabled. I was using the same text in network settings in Ubuntu, but to no avail. I also tried disabling the security settings online and using my windows ssid and password in Ubuntu as well, and that didn't work either. 

          My fix was, at least for me again (please don't flame me if this doesn't work for you) was to set my ESSID to linksys and password to admin, (defaults) and disable the security settings online,.  That's it. Now, I connect everytime. I'm sure there are some changes I have to make for a secure connection, and if someone knows what I need to do, by all means; but I'm not really worried about it with Ubuntu, and this is a mostly stay at home laptop. Anyways, that's the jist of it.
