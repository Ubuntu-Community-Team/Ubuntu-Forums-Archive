### Post by wildmanne39 on 2011-08-01
Hi,which SSID are you wanting to connect too? you have three access points listed.

Also your mode is set to master and you need to have it set to manage so let change that and see if you can connect.
```
sudo ifconfig wlan0 down
```
```
sudo iwconfig wlan0 mode Managed
```
```
sudo ifconfig wlan0
```
then see if you can connect.

Also change the SSID to the network you want to connect to by clicking on the internet icon in the top right corner of the screen then click on edit connection the on wireless and edit, then change the SSID to the correct name, and if it does not connect let us know the name you changed it too so I can see if it matches what is in your information.
