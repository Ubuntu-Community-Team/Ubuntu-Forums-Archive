### Post by netztier on 2008-02-22

> cannot connect to lan with 3c905b-tx pci card....

Not needed. The correct kernel module is already are in the kernel.

You might have a misconfigured card that has strange settings in it's internal configuration registers. It seems that the Windows drivers can override these settings while the Linux kernel module can't.

There is a DOS-based tool available from 3com that can reconfigure these registers.

See these threads

[http://ubuntuforums.org/showthread.php?t=387839](http://ubuntuforums.org/showthread.php?t=387839)
