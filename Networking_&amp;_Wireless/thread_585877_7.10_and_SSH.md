### Post by ezak on 2007-10-21
To solve your problem make sure you run this:

sudo apt-get install openssh-server

sudo apt-get install ssh

Make sure you are allowed to do "sudo" commands and it will ask for your password, after that, you should have absolutely no problem with ssh.
