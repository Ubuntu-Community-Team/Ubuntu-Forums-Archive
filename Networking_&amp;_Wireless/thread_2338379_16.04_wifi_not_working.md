### Post by jeremy31 on 2016-09-27
Welcome to the forums

Use the Iphone hotspot to connect and run ```
wget -N -t 5 -T 10 https://github.com/UbuntuForums/wireless-info/raw/master/wireless-info
```

Disconnect from the hotspot and attempt to connect to your home router, then run the wireless diagnostic script
```
chmod +x wireless-info && ./wireless-info
```

Then reconnect to the Iphone and post the contents of the wireless-info.txt using code tags or attach the wireless-info.tar.gz if the file is too big

Please remove the [SOLVED] from the topic until it is solved as it might keep people from reading your post
