### Post by praseodym on 2011-10-19
Hi,

in 10.04 you can install a backport-kernel from Natty, which contains the driver (firmware installation attached):

```
sudo apt-get install linux-image-generic-lts-backport-natty linux-headers-generic-lts-backport-natty build-essential dkms
wget http://media.cdn.ubuntu-de.org/forum/attachments/2640681/ath_htc_firmware-1.tar.gz
sudo tar xvf ath_htc_firmware-1.tar.gz -C /lib/firmware 
```
Reboot.

Other solution: Compile the driver on your own from linux-wireless.
