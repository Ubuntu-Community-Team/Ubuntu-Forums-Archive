### Post by Brandon Williams on 2009-10-29
tsclient is just a GUI interface for running rdesktop. The rdesktop man page says:  "Fullscreen mode can be toggled at any time using Ctrl-Alt-Enter." This works for me.

### Post by mister_doctor on 2009-11-04
During a remote desktop session, in the start menu, Shut Down is replaced with Disconnect. It is the same as if you were to exit the client.
