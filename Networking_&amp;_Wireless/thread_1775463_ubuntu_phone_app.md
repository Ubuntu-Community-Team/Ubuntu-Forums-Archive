There were some types if modem that could do this... (and I'm using the past tense - haven't actually seen a POTS modem outside of a second hand store for a while).

Your best bet is to track down the tech specs for the laptop and see if you can identify the particular type of modem and see how the modem audio relates to the sound hardware for the rest of the computer.
Some of them are "hybrid" setups - the main CPU actually does the signal processing; some have dedicated hardware that doesn't do anything more than dumb data and fax... and some are inbetween.

I presume you've tried looking at software like Asterisk and perhaps tried an install to see if it already knows about this kind of device?
