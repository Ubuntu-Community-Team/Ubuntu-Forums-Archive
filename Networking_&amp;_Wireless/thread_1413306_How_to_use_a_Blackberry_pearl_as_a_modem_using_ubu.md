### Post by noway2 on 2010-02-23
Try berr4all, [www.berry4all.com](www.berry4all.com).  I use it to tether my BB under linux.  The wifi I usually use was down this evening and I used it to get a connection, while listenig to internet radio at the same time.  It isn't the speediest connection, but it is decent and sure as hell is a lot better than none.  

Simpyl download and unzip.  You will need to scan once and select your provider.  The application will also put your USB port into charge mode.  You will either need to run the script as sudo or change the USB permissions.  Also, note, it does not seem to work on 3G networks.  You need to change your network to 2G (edge) only.
