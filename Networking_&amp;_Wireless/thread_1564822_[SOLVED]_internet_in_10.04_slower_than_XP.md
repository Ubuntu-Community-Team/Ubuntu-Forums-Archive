### Post by srini_yes88 on 2010-08-31
One possible reason for slower Internet in 10.04 compared to XP could be the use of different DNS.
**solution:** If u are using the DNS provided by the firewall services like COMMODO FIREWALL in XP, then make sure that u set the same DNS in 10.04 too.
I had the same problem earlier and this solution made my internet faster in 10.04 too..:P
