# This file was automatically generated by the /lib/udev/write_net_rules# program, run by the persistent-net-generator.rules rules file.

### Post by steeldriver on 2013-08-16
It sounds like you don't have an appropriate eth1 stanza in your /etc/network/interfaces file - based on your previous posts it could be something as simple as

```

auto eth1
iface eth1 inet dhcp

```

but if you need help with that, post back including the current contents of the file, plus what kind of networking you are trying to set up (static versus DHCP, what DNS servers etc.)
