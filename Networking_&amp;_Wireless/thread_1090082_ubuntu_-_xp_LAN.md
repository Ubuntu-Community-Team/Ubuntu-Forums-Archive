### Post by Iowan on 2009-03-08
Networking between Windows and Linux will probably involve [Samba]("https://help.ubuntu.com/community/SettingUpSamba").  The client comes pre-installed - so you should be able to view Windows shares on an Ubuntu machine - but going the other way (sharing FROM Ubuntu TO Windows) will require installing the server component.  The link should provide details - I have a few more links if you would like to see them.
