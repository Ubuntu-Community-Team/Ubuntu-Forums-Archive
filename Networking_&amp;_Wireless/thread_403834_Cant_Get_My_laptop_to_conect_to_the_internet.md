### Post by Hendrixski on 2007-04-07
If you're doing a wired internet, then it should just work.  Make sure that you have DHCP enabled on your router, or whatever network you're on.  It's usually on, but sometimes not.

If it's wireless then you MAY need NDISwrapper (it's a little thing that uses windows drivers on your wireless card if there are no linux drivers for it) but chances are that you don't.  Make sure that you're connecting to a network.  Go to Add-Remove programs and install "Network Manager", when it runs, it runs on your start menu, and you can use it to switch between networks.  My laptop lives on that.

If you have any more questions there's TONS of documentation about it.  Try the little question mark on your desktop.  :-)
