### Post by NIN101 on 2011-01-05
You need samba. Run:

**sudo apt-get install samba**

You probably have it already installed. In nautilus, click with the right mouse button on a folder and then configure it for sharing. 

This is probably the easiest way. You may also want to read:
[this](https://help.ubuntu.com/community/Samba) etc. There is a lot of documentation about this topic.
