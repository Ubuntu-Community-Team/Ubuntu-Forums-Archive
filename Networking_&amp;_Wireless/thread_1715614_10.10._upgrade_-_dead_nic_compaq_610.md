### Post by Zli Zeka on 2011-07-21
This sequence worked for me:

1 go tu system - administration - synaptic package manager - and remove all b43 drivers from the list.

2 reboot

3 again go to system - administration - synaptic package manager - searche for bcmwl, and select both of the options to be installed.

4 reboot 

After reboot, my wirlees came to life!

I have a BCM4315 chipset.


if that fails for you, try this [https://help.ubuntu.com/community/WifiDocs/Driver/bcm43xx/](https://help.ubuntu.com/community/WifiDocs/Driver/bcm43xx/)
