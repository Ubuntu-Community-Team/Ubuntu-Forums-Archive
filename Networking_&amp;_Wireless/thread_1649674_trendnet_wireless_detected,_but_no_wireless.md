### Post by grahammechanical on 2010-12-20
You will not need to disable the wired network driver. It would be very unusual for there to be a conflict.

If you left click the network icon you might see a list of available wireless networks. Is Trendnet among them? If so, select it and put in the pass phrase or key to connect to the wireless network.

If you cannot see any available wireless networks, then right click the network icon and see if Enable Networking and Enable wireless have tick marks against them. If they do not, then select them. If they both have a tick mark, then select Enable Networking to disable networking and then select it again to re-enable networking. In this way you switch networking off and then on and it may activate the wireless adapter and get it scanning for wireless networks.
