### Post by jerome1232 on 2008-09-04
You could use pidgin with the bonjour service, it's a lan based instant messaging solution, so if every machine is signed into bonjour via pidgin (no internet needed) you can IM each workstation.

My old job used a similiar approach using psi, it's another open source instant messaging product.
