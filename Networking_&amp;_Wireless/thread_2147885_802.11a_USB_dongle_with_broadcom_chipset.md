### Post by praseodym on 2013-05-23
Hi,

unfortunately, Broadcom does not support its own USB-wireless-dongles in Linux-systems. The respective Windows driver needs to be installed via Ndiswrapper. Please show the terminal (CTRL+ALT+T) output of the command

```
lsusb

```
with the dongle plugged. Normally, Ndiswrapper and the Win-driver do not support the 5 GHz region. Therefore, I strongly recommend another supplier, e.g. **FRITZ!WLAN USB Stick N v2-B** with Ralink rt5572 chipset.

Please check this site and search for "Dualband" and check if the respective device is flagged with "USB"

[http://wiki.ubuntuusers.de/WLAN/Karten](http://wiki.ubuntuusers.de/WLAN/Karten)
