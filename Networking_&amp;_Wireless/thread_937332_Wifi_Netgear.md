### Post by lisati on 2008-10-03
I haven't actually used an ad-hoc network using my Netgear router and a Zydas-based dongle, but have successfully used the Netgear router as an access point and the dongle as a station, without the need for installing any extra or replacement drivers for the dongle. 


A check of the [Netgear website]("http://kbserver.netgear.com") has this suggestion:
>      Be sure all computers in your Ad Hoc network are configured with static IP address in the same subnet.



"Dongle" isn't quite the right word for my Zydas-based gadget (it has 3 x usb ports, 2x PS/2 ports, a serial port and a printer port in addition to the wireless) - but since it uses the same chipset, it **sort-of** comes close.
