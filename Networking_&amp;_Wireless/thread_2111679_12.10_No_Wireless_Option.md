### Post by praseodym on 2013-02-02
Hi,

please execute the following commands:
```

uname -a >> wlan.txt
lspci -nnk | grep -iA2 net >> wlan.txt
lsusb >> wlan.txt
lsmod >> wlan.txt
iwconfig >> wlan.txt
rfkill list >> wlan.txt
```
This creates a file called wlan.txt in /home/$USER. Upload this file or its content
