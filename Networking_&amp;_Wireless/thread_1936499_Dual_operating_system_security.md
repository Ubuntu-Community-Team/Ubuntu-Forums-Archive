### Post by mikewhatever on 2012-03-06
If someone from within the network downloads malware, knowingly or unknowingly, and runs it on Vista, then yes, that would be a security compromise. If you open an infected .pdf or .doc file, Vista could become compromised as well. That said, the impackt would probably be minimal, provided it doesn't have internet access.
