### Post by annoyingrob on 2010-09-08
I'm not sure if it helps, but I had a very similar problem with my Acer Aspire One netbook. The problem ended up being the wifi card in the machine wasn't the best quality (Atheros 5xxx or something), and would overheat/shutdown randomly. The more bandwidth I was using, the faster it would reset itself. My solution was to buy an Intel 4165 A/B/G/N wifi card on Ebay for 15 bucks and swap it out.

If you have access to another wireless card, you might want to try swapping it out, and see if the problem persists.
