### Post by jbergfel on 2005-10-19
A patch is available on [http://www.linuxant.com/drivers/hsf/downloads-patches.php](http://www.linuxant.com/drivers/hsf/downloads-patches.php). Download this file and download from [http://www.linuxant.com/drivers/hsf/full/downloads.php](http://www.linuxant.com/drivers/hsf/full/downloads.php) the file hsfmodem-(version-nr).tar.gz

then type

tar -xzvf hsfmodem-(version-nr).tar.gz
cd hsfmodem-(version-nr)
sudo make install
sudo hsfconfig --patch /(your-folder)/(filename).patch

sudo make install (you need to reinstall the hsfmodem, because the patch is applied to the install files)

This will solve the uname error. Then 

sudo apt-get install gcc-4.0 
export CC=gcc-4.0

this will solve the gcc error. You might need to install linux-headers and or linux-source, both available with apt-get.

and sudo hsfconfig again
