Well, solved it by myself after extensive reading and spending a full day behind it. 

If you don't see the network drive (like me) the first thing you want to install is Konquerer. On the front page itself you see network folders icon and clicking on that takes you to network where somehow it found the network drive for me and asked for the relevant password and all. 

Then I just copied the command line from the address bar and put it in nautilus and hoorrraaa... got it there too. Have bookmarked it since so easier to open every time.
