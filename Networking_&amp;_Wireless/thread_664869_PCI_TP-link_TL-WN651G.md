### Post by mikeand on 2008-03-26
I use this card with 7.10 64 bit. It works out of the box.

After installing the card, the only setup required was to select my wireless network from the network manager icon and enter my passphrase (I use WPA2).

It does use a Restricted Driver (Atheros Hardware Access Layer (HAL)), so presumably it's not completely open source or uses firmware or something. The box it came in mentions it uses the Atheros chipset.
