### Post by MelArcher on 2009-12-11
Much of the time for powerline devices when they say support for PCs and Macs only, they are referring to the software administration utility on the CD that ships with the units. The powerline adapters themselves will work fine with Unix. You just can't run the software for changing the network password, etc.

I saw some new adapters from Plaster Networks that have a built-in web app for administering the network - no need to load any software on your system because all you need is a standard browser (PC, Mac, or Linux). The concept is similar to the management and administration tool in most routers these days.
