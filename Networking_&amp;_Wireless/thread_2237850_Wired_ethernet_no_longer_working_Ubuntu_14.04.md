### Post by varunendra on 2014-08-05
> **J Blain said:**
> ```
....driver=e1000e .... **[COLOR="#0000CD"]link=yes[/COLOR]** .... speed=100Mbit/s
```
The "link=yes" and speed=100Mbit/s suggest that you are physically connected, and communication link has been detected and negotiated. Perhaps it is only the configuration part that is having some minor trouble.

Easy fixes first -

Disable the wifi, then -

Disconnect the cable > reboot both the PC and Router > reconnect the cable. Reboot the router once more if this doesn't automatically get you connected.

If this doesn't help, try manual IP configuration in Network Manager. Something as suggested here (change the IP to suit your network) : [http://ubuntuforums.org/showthread.php?t=2237120&p=13089144#post13089144](http://ubuntuforums.org/showthread.php?t=2237120&p=13089144#post13089144)

Save and close Network Manager settings, and retry to connect.

### Post by lnappa on 2015-01-01
You probably solved the problem long ago, but in the interest of others viewing this post, I found that for my Intel NUC D54250WYKH the problem was [COLOR=#000000][FONT=Ubuntu Mono]driverversion=2.3.2-k[/FONT][/COLOR] . Updated driver can be found here [https://downloadcenter.intel.com/Detail_Desc.aspx?DwnldID=15817](https://downloadcenter.intel.com/Detail_Desc.aspx?DwnldID=15817). I found the script on [https://gist.github.com/mrapple/8249889519abd93dfba9](https://gist.github.com/mrapple/8249889519abd93dfba9) very helpful.
