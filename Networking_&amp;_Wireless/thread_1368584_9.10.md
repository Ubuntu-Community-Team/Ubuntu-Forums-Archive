### Post by pdc on 2010-01-08
you can help by telling what you have done; even your title does not help anyone help you;

here is a description of what worked for one person

[https://help.ubuntu.com/community/CableDialup](https://help.ubuntu.com/community/CableDialup)

you need two programmes: wvdial and netenv

he says how to install them

then in a terminal one looks for what port the computer will use by issuing this command

> sudo wvdialconf /etc/wvdial.conf

and for him: the port created was called

> /dev/ttyACM0

[I]yours may be different; it might be /dev/ttyUSBsomething ............ possibly ...........
[/I]
you then modify the config file for wvdial

> sudo gedit /etc/wvdial.conf is what I use

the file he suggests looks like

> [Dialer Defaults]
Modem = /dev/ttyACM0
Baud = 230400
Init1 = ATZ
Init2 = ATQ0 V1 E1 S0=0 &C1 &D2 +FCLASS=0
Init3 = AT+CGDCONT=1,"IP","web.omnitel.it"
ISDN = 0
Modem Type = USB Modem
Phone = *99#
Username = ''



you need to change the 

> web.omnitel.it

with whatever Airtel GPRS uses

I suspect it is 

> airtelgprs.com and I got that from here

[http://www.flexispy.com/Mobile%20APN%20Setting%20to%20use%20GPRS.htm](http://www.flexispy.com/Mobile%20APN%20Setting%20to%20use%20GPRS.htm)

so I guess your wvdial.conf file should be

> [Dialer Defaults]
Modem = /dev/ttyACM0
Baud = 230400
Init1 = ATZ
Init2 = ATQ0 V1 E1 S0=0 &C1 &D2 +FCLASS=0
Init3 = AT+CGDCONT=1,"IP","airtelgprs.com"
ISDN = 0
Modem Type = USB Modem
Phone = *99#
Username = ''

so you would copy and paste the above text into the file and save as 

> /etc/wvdial.conf

then wvdial knows where to go and look for instructions when you tell it to dial

if you get that done: try

> sudo wvdial

if it works, control-c stops the connection

if no joy, let the forum know
