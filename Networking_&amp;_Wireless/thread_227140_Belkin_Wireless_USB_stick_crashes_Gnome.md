### Post by bodycoach2 on 2006-08-21
This has happened on several Ubuntu or Xubuntu machines I've set up, but so far I've been able to make each one a success.

I found that the system froze anytime I tried to make more than one change at a time. When using Network Manager to configure the wireless USB (usually is stays rausb0), make only one change, click okay, and close Network manager before making another change. 

Even if it froze a couple of times, with enough patience, and one-step-at-a-time configuring, I've been able to get all the Ubuntu boxes (10 so far) set up with wireless. I've NOT been successful with the old Strawberry iMac 333 Mhz machines, though.
