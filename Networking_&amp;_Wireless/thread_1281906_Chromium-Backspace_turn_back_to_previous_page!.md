Since yesterday there's a new version of the extension. About the security concerns:
It offers access to file:/// protocol to give backspace keybinding on local files as well. The newer versions of Chrome / Chromium have a special option on extensions page to not allow an extension to read files by file-protocol. There are no other special permissions requested.
