### Post by trustnoone on 2009-02-18
Hi this is my first post i am probably one of the most noobest persons on the planet but i figured i would describe how to set up the maxon modem BP3 (for australia), the one that looks like a toaster design that run through the next G network for Ubuntu 8.10 (soz soz if im posting this in the wrong place or if it has already been posted)

setting this up is actually fairly easy although it took me hours to figure out, this is mostly for complete noobs that have this net, and want to run it on Ubuntu

Firstly you plug in you maxom modem BP3 into the USB of your Ubuntu computer. (it might have to be set up on a windows computer before hand though)
on the top right hand of the screen there is an icon of 2 computers one behind the other, click on it and it should automatically be found (in ubuntu 8.10) as "auto mobile broadband (CDMA) connection" firstly click on it, you might get some error? if so go to system>preferences>network configuration then click on the mobile broadband tab.

"auto mobile broadband" should now be in the list, click edit on the right.
click on connect automatically check box, the number is *99# (in australia) the user name is [email]_____@bigpond.com[/email] with ___being your username and password is your connection password.

hit okay and close, then go to the icon with the two little computers and click on it, you should now be able to click on "auto mobile broadband" and it should load fine :D
