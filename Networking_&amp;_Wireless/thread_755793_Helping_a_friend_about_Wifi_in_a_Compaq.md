### Post by chili555 on 2008-04-15
I have some advice. Please give us some information about what kind of wireless card it is.```
sudo lshw -C network
```If you find out it's the dreaded Broadcom, you might check here: [https://help.ubuntu.com/community/WifiDocs/Driver/bcm43xx/Feisty_No-Fluff](https://help.ubuntu.com/community/WifiDocs/Driver/bcm43xx/Feisty_No-Fluff)

If it turns out to be something else, please use the search function and look for the SOLVED qualifier. Unless a card was built five minutes ago or fifteen years ago, it has been installed a few hundred times here.

Post back if you get stuck; we'll be happy to help.
