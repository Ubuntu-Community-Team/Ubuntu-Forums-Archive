### Post by Grulk on 2009-11-02
If it is just updating that is slow i suggest that you change where you download it from. 

**Go to Administration -> Software Sources** 

Under the  **Ubuntu Software tab** you should see something that says "**Download from:**"
with a drop down menu beside it. Click the** drop down menu -> Others** and change where you want to download it. Once this is complete, close and it should prompt you with something that says "**The information about available software is out-of-date**" Click on the **reload** button and it should update it and hopefully sort your problem.
[B]
Note: [/B]It is best not to use the main server because a large percentage of uses might have that set and the speeds might be slow try choose something that is close to where you live.

Hope this helps :)
