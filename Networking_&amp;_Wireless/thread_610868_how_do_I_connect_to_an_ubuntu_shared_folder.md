### Post by MrFSL on 2007-11-12
It depends on what type of share you are setting up.

You can read here:
[https://help.ubuntu.com/community/Servers#head-80dd5f86d31988e45e15caedce1d515e991a0f34](https://help.ubuntu.com/community/Servers#head-80dd5f86d31988e45e15caedce1d515e991a0f34)

for more information about file servers.

My personal suggestion is to share files using ssh. The remote machine can be accessed in nautilus by typing:

ssh://username@IP.Of.File.Server
