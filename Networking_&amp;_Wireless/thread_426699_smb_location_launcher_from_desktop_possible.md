### Post by SendDerek on 2007-04-28
It's going to be easier than a script...

Just create a launcher on the desktop by right-clicking on the desktop and selecting Create New Launcher.  It should bring up a "launcher wizard" so-to-speak that ask you what to name your launcher, how to run it, and the command needed to launch it (in this case it would be smb://192.168.0.2).

If that doesn't work, you can goto the "Places" menu on the 'task bar' and select "Goto Location..." and that will guide you through the process as well.  Then after it's finished it will place a shortcut on your desktop.

Hope this helps...
