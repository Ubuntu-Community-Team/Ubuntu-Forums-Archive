### Post by nils234 on 2007-12-27
Well maybe if you would be more specific in what you want to know people might reply ;)

To set up your router, just plug it in and wait till it´s started up. Then either connect to it´s wireless or wire it to your computer. Normally it will automaticly set up a non authentication wireless so that souldn´t be a problem. Then surf to your router´s ip, probably 192.168.1.1. Log in, standard password and  username are probably admin/admin or user/user. Then just go to the security tab and set it up.... If you want a decent security type choose wpa2 or wpa2-psk, and choose a decent length passphrase. Make sure your clients support it though.

thats it realy.
