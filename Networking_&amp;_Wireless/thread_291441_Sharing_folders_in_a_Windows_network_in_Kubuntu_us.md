### Post by ceros on 2006-11-05
Is Samba installed? If it is, to administer Samba in Kubuntu, key in ALT+F2, then in the run command dialog box, type in **kcontrol**. Go to Internet & Network > Samba. This module allows you to do what you need to set up Samba. The Shares tab is where you go to share folders.
