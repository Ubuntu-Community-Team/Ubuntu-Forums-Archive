Right,
I'll include my smb.conf, it works on my pc.

You copy/paste it to a new file named smb.conf in your home dir.
Then run this in a terminal (substitute <username> with your username):
```

sudo bash
mkdir /myshare
chown -R <username> /myshare/
cd /etc/samba
mv smb.conf smb.conf.old
cp /home/<username>/smb.conf .
chmod 644 smb.conf

```
reboot
now copy some files to /myshare
