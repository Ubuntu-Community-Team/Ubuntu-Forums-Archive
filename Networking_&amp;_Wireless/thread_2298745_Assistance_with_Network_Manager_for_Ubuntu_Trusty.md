### Post by grahammechanical on 2015-10-13
Ubuntu has what are called app indicators. They are little icons in the top panel to the right. The network icon looks like two arrows going in opposite directions when we have a connection to a router. And it looks like an inverted cone when there is no connection.

Clicking on an app indicator will bring down a menu. The Network Manager icon menu includes an option for VPN connections.

Another method is to open System Settings (power/cog menu) and load Network. Select the network that is connected and there should be a way to access a VPN connection tab. I am using a version of Ubuntu newer than Trusty (14.04). I am not sure if things have changed. Any descriptions I give about dialog boxes may not be accurate. If there is an Options button try that. Look for the General tab.

You did say Ubuntu, did you not? Xubuntu will do things differently.
