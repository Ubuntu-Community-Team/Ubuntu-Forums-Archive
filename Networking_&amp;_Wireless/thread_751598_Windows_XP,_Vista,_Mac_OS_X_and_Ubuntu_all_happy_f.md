### Post by Eiríkr on 2008-04-10
Provided that your desktop computer with all your files on it is Ubuntu or some other Linux flavor, this is do-able using the Samba share server.  You'll need to install the **samba** package on your Ubuntu machine.  If your desktop is Windows, this is still do-able, but the setup is different.  Let us know which OS is on the machine that will be the share server, and we'll go from there.
