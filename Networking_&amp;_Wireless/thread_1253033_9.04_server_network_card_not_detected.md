### Post by Iowan on 2009-08-29
> **Fnugus said:**
> 
This looks to me like an adress and netmask is missing...
Address and netmask are not usually specified for dhcp - the server generally provides them (as well as gateway and DNS information). 
**ifconfig -a** will show all interfaces - including those that have not been configured... it will probably show that eth0 has no IP address.  
**lshw -C network** will *probably* show the same (limited) information about the ethernet interface.
