### Post by ugm6hr on 2007-05-26
Not sure exactly where you are trying to type the "network key" - but I suspect it is a 64-bit HEX key for WEP if it is 10 digits (0-9,a-f).

Try selecting HEX key / WEP and entering.  

If it doesn't work, give more details about exactly where you are trying to enter the key (I would recommend using roaming mode in the manual config, then left click on Network Manager to select network).

Other questions is - does the laptop connect with Windows?
