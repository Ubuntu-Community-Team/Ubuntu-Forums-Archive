### Post by viet94 on 2008-08-18
Firefox 3, the latest version of firefox family. Generally, it works well, but there is a big problem in this new firefox : flash seems not working as good as it was. Every time I using firefox to browse a site that have a big amount of flash, firefox crash right away:(.
I had found a solution for this problem. It seems that Ubuntu cause this problem (hope the new version Intrepid will fix it). So just uninstall all firefox content out of your ubuntu by synaptic (include all plugins : mozilla-mplayer.....). Next, backup the folder .mozilla in your home folder in order to save all your preference and addons. After that just go to firefox homepage and download tar.gz version and run it by create a launcher to firefox file (like flock). Forum now, you firefox will work completely well with flash.
