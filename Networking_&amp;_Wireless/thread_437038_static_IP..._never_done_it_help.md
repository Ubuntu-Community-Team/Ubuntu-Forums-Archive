### Post by renzokuken on 2007-05-08
you dont necessarily have to have a static ip for port forwarding to work. if there is only one or two pcs on your network more often than not they get assigned the same ip as before anyway. you can just change which ip address to forward to on your router if it does asign you a different ip.

but thats a moot point. it is still much better to assign a static one though so you dont have to keep changing your router settings.
