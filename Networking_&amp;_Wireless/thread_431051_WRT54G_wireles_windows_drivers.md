### Post by Floppyjoe on 2007-05-02
You can set up the router using the linux computer and some of the instructions that came with it.
Connect your router to your modem and to the wired Linux computer. Then open firefox and enter the IP address that came with your documentation.
This should bring up the web interface of the router.
Then you can enter your ISP details and what not.

### Post by chili555 on 2007-05-02
Yeah, what Floppyjoe said...the URL to type for a WRT54G is [http://192.168.1.1](http://192.168.1.1), the user name can usually be skipped and the default password is admin. As soon as you get set up, I advise changing the password.

I have administered my Linksys equipment from Linux for many years.
