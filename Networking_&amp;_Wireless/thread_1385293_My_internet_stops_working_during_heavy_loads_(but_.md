### Post by ant080808 on 2010-01-20
OK, so I managed to fix this by going back to the native linux wireless driver. The reason I moved to the windows one was because I was only getting a network speed of 1mb/s instead of 54mb/s. So this was making my streaming really slow. Anyway I found out on this forum how you can manually change your network speed using

```
sudo iwconfig wlan0 rate 54M
```
and now when I check my connection info it says 54mb/s and my streaming seems OK now.
