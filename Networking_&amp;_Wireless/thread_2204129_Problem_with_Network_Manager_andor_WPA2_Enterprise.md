```
bssid=00:1A:1E:27:5E:A0
```
..where the MAC ID "00:1A:1E:27:5E:A0" would be the one which was closer to you and you found its signal strength to be strongest.

Try binding a particular MAC ID again, and check the file again. Does the line change to look like above? I believe you can also do vice-versa. That is, changing the line manually in the keyfile should make Network Manager reflect that setting when opened AFTER saving the change. You'll need to open that file with root access *(gksu gedit /etc/NetworkManager/system-connections/<the keyfile>)* to be able to edit it (but before making changes, make sure to save a backup copy of it to be able to revert back quickly if required).
