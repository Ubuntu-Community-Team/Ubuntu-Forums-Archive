### Post by UtopiaTheory on 2008-10-29
Hello,

Try clicking on System > Administration > Hardware Drivers.

Check the box that says "Enable" for the B43 Wireless Driver.  When it asks you if you want to fetch the firmware from the internet, make sure you have a wired connection and click Yes.  This should enable the card!
