### Post by TheFu on 2014-06-20
Is this a trick question?  If you installed it like every other program using the package manager and repos, then do the normal system maintenance stuff that everyone should do every week: [http://blog.jdpfu.com/linsysmaint](http://blog.jdpfu.com/linsysmaint)

If, OTOH, you installed using a .deb file or from source, then ... well, you are on your own. You'll need to manually maintain it. That is the main reason why installing software this way is discouraged.
