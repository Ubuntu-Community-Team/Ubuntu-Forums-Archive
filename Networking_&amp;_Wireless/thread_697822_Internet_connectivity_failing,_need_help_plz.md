### Post by nixscripter on 2008-02-21
Please be more specific about the symptoms. I am assuming, by the way that the "nic card" is a wireless card.

1. Does the Ubuntu machine find the wireless network of the adapter? If not, then it might be a problem with the adapter.
2. Does it successfully connect to that network? If not, then it's probably a configuration problem.
3. Can you see (e.g. ping) the XP machine? If so, then it's a problem with the switch (unlikely, since the XP machine works).
