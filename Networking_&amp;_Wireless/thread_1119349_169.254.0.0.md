### Post by Iowan on 2009-04-07
169.254.X.X is an address range used by **avahi** > Avahi is a fully LGPL framework for Multicast DNS Service Discovery.
It allows programs to publish and discover services and hosts
running on a local network with no specific configurationInformation is from the package description in Synaptic.
