### Post by Brandon Williams on 2009-08-02
What you're describing is a repeater, not just an access point. I use a WRT54G as an access point, with a cable from the WRT54G to the gateway router. However, it's also possible, from what I've read, to configure it as a WDS repeater. [Here](http://www.linksysinfo.org/forums/showthread.php?t=47118) is a page with some configuration notes that might help.
