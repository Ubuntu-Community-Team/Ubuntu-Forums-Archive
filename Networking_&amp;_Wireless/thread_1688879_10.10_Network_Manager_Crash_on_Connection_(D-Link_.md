Hello if you are having problems connecting to your D-Link DIR-655 (or other router) on Ubuntu 10.10 and are frustratingly having your Network Manager crash and freeze your computer, I think I have found the solution.[INDENT]1.Log in to your router via [192.168.0.1]("http://ubuntuforums.org/192.168.0.1") or whatever you have specified.
[/INDENT](Log in default is **User**: Admin **Password**: (blank) )[INDENT]2. Navigate to Setup > Wireless Settings > Manual Wireless Network Setup
[/INDENT][ATTACH]183587[/ATTACH][INDENT]3. Click the drop down box for 802.11 Mode and select 802.11g ONLY as shown above.
[/INDENT]Hope this works for anyone with the same issue as me. :D
