### Post by spd106 on 2007-04-26
This is a very difficult question and it gets asked a lot. The problem is that manufacturers change the chipsets in their cards every few months so while some version work well, the next iteration might not.

The best strategy is through trying before buying, though this is rarely possible. If you can find a supplier who advertises support for Linux then buy from them.

You may also find the subforum sticky about supported cards to be useful resource.
