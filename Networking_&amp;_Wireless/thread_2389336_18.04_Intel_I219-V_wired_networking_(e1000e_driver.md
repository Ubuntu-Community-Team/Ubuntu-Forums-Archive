### Post by dbd on 2018-04-16
Sadly, the latest driver doesn't build for the kernel that ships with 18.04.

I had to add [FONT=Courier New]-fno-pie[/FONT] to the [FONT=Courier New]EXTRA_CFLAGS[/FONT] in the Makefile to even get it to attempt to build, as described here: [https://askubuntu.com/questions/851433/kernel-doesnt-support-pic-mode-for-compiling](https://askubuntu.com/questions/851433/kernel-doesnt-support-pic-mode-for-compiling)

But even with that it doesn't build, because the interface for the timer in the kernel has changed and they've not updated the driver to implement the new interface yet: [https://sourceforge.net/p/e1000/bugs/600/](https://sourceforge.net/p/e1000/bugs/600/)
