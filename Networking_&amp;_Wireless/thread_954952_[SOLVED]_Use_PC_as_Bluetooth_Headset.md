### Post by camoanimal on 2008-10-22
That is a good question.  I have done this with my LG VX9900 under both M$ Windows and Ubuntu, But I gave up on Kubuntu a while ago.  In theory it is possible but you must first make sure you computer is bluetooth enabled.  Once you have done this make sure the drivers are correctly installed for that particular device.  Finally put your phone or wireless device on brocast mode and use the bluetooth GUI under you settings to find you bluetooth device.  Click on it and it should automatically set it up for audio if it is capable of using that feature.

I know this is vague but I hope it helps some.  Remember if all else fails try Google.  I have found a world of answers on there when I was first trying to setup my system.

Good luck :)
