### Post by chili555 on 2013-11-17
> E: Could not get lock /var/lib/dpkg/lock - open (11: Resource temporarily unavailable)That usually means some other software installation process is sitting open; often it's Software Center or Synaptic. Find it, close it and try again.

In most recent Ubuntu versions, if you have a temporary wired ethernet connection, I'd actually prefer:```
sudo apt-get install linux-firmware-nonfree
```
