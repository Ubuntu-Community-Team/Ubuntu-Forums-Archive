### Post by nzadLithium on 2008-05-26
Ok your network driver is up and your network card is setup to a degree. Just needs ip etc and to be started :)
try this:

sudo ifconfig eth1 down
sudo dhclient -r eth1 (if this fails dont worry)

if your using dhcp
sudo ifconfig eth1 up
sudo dhclient eth1

if your using static
sudo ifconfig eth1 (ip addy) netmask (netmask) up

then you will probably need to add your gateway:
sudo route add default gw (gateway address) eth1

hopefully that will get it going.
