### Post by shairozan on 2010-01-07
Actually, it's even easier. Go to applications -> add remove and search for "Active Directory Membership" which is the likewise open system. 

Once you've installed it, it will be listed under system -> administration. There you can join the domain like your normally would, but you'll have to have an account with domain admin rights to do so, same as you would in Windows.
