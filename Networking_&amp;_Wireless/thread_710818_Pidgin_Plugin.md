### Post by kevdog on 2008-02-29
I would suggest either both or one of the following plugins:

pidgin-guifications - [http://plugins.guifications.org/trac/wiki/Guifications](http://plugins.guifications.org/trac/wiki/Guifications)
pidgin-libnotify - [http://gaim-libnotify.sourceforge.net/](http://gaim-libnotify.sourceforge.net/)

With the guifications plugins there are screenshots.  I believe you have to compile this package also.  I personally prefer the libnotify plugin for appearances, although the guification plugin has a lot more options.

The libnotify plugin you have to compile and install.  This is really easy if you have ever compiled anything before.
