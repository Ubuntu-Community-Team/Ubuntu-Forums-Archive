### Post by Sandevil on 2012-09-17
*Is there any problem with this type of devices ??*

Yes, http traffic didn't work with the generic driver (rt2800usb).

I solved using the official drivers from ralink(RT8070 /RT3070 /RT3370 /RT5370 /RT5372 USB):

[http://www.ralinktech.com/en/04_support/support.php?sn=501](http://www.ralinktech.com/en/04_support/support.php?sn=501)

You have to set some options in os/linux/config.mk:

** Build for being controlled by NetworkManager or wpa_supplicant wext functions
'HAS_WPA_SUPPLICANT=y' and 'HAS_NATIVE_WPA_SUPPLICANT_SUPPORT=y'.

and then:

sudo make  && sudo make install
(check the name of the driver in os/linux/ directory, in case of rt5370sta.ko)
echo rt5370sta >> /etc/modules
reboot

and problem solved.
