### Post by ugoboom357 on 2008-08-20
Basically DHCP lets your router assign ip addresses automatically as opposed to using static IPs. DHCP is usually the easiest way to go, but in my experience it can be annoying if your computers happen to start up in a different order (meaning the IP addresses would then be assigned in a different order).

Here's more info on DHCP:
