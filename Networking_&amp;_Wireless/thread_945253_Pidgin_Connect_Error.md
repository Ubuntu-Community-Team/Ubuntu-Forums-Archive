### Post by spcwingo on 2008-12-11
I had the same problem.  I'm now using a different server to correct this.  Open Pidgin and open accounts.  Then, select your Yahoo account and press modify.  A new window will pop up.  Select the advanced tab and change the Pager Server to:

cn.scs.msg.yahoo.com

Then try reconnecting to Yahoo.  I hope this gets the problem solved for you.
