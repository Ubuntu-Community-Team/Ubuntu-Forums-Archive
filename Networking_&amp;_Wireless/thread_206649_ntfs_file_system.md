### Post by tturrisi on 2006-06-30
Actually, you CAN write to ntfs volumes, but this is not enabled in a default install of Ubuntu.  You must install some restricted packages.  Caution, linux writing to NTFS is NOT completely without risks.  See:
[http://doc.gwos.org/index.php/Mount_NTFS_volumes_with_write_support](http://doc.gwos.org/index.php/Mount_NTFS_volumes_with_write_support)
