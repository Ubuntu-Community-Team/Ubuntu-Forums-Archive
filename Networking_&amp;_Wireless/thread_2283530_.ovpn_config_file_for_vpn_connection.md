### Post by PaulBx on 2015-06-22
So, you don't have your old computer any more? If you do, just do 
sudo find / -iname *.ovpn

If you don't have it, and you are using vpn to connect to work, just get the file from the admin. I don't think it matters what OS they are using; the file should be the same (with the possible exception of references to file locations in the file, which should be easily detected and modified).
