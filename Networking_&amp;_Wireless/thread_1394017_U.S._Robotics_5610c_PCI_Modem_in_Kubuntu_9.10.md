This is a shot in the dark coz I don't use Kubuntu. To bypass the 'missing /etc/resolv.conf' error, try entering the following command in a terminal:

```
sudo touch /etc/resolv.conf
```

This should create a blank /etc/resolv.conf file. In theory, kppp would update this file to the appropriate DNS server IP addresses of your ISP once it successfully connects.
