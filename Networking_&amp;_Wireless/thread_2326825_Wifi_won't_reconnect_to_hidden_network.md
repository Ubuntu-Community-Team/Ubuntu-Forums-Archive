### Post by arie-i on 2016-06-05
Hello, I have a hidden network that I connect to with my laptop. I set up the network by adding it in network manager. The first time I connect it works fine, but as soon as I close the connection I am not able to reconnect to that network. I can go to 'connect to hidden wifi network' and my network is listed, but I can't connect. The only way to get wifi back to to create a brand new connection. But then again, the same thing happens to that new connection when I leave it and try to come back to it. I have network manager set to auto reconnect to this network when available.
I am running Ubuntu 16.04 on an Acer E 15 laptop.

### Post by jeremy31 on 2016-06-05
You will have to make the wifi visible to make it work.  Once it is visible, use network manager to enter the password, click automatically connect and save the connection.  You should then be able to hide the network and automatically connect
