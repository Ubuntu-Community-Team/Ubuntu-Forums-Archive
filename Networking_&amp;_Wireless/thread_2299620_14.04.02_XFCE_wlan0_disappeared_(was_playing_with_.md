### Post by Bucky Ball on 2015-10-20
With this command:

```
curlftpfs#ftp://[user]:[password]@[ip_address]:[port]/ /mnt/[folder] fuse _netdev,auto,user,allow_other,uid=1000,gid=1000,umask=0022 0 0
```

... are you actually inserting the correct details for [user], [password], etc. or you are using that line as it is??? Example, presuming the user is 'you' and the password is 'yours', your IP is 192.168.0.100 and you want to mount to /sdb2 (second partition, second drive) in /mnt.

```
curlftpfs#ftp://you:yours@192.168.0.100:22/ /mnt/sdb2 fuse _netdev,auto,user,allow_other,uid=1000,gid=1000,umask=0022 0 0
```
