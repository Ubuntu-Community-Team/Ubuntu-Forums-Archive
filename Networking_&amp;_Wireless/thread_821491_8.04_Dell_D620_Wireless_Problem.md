### Post by Ayuthia on 2008-06-07
You can try and see if this works:
```
sudo modprobe -r b43 bcm43xx ndiswrapper
sudo modprobe ndiswrapper
sudo iwlist scan
```
and see if it finds any access points.  If it does, you can try the application at this post:
[http://ubuntuforums.org/showthread.php?t=779754](http://ubuntuforums.org/showthread.php?t=779754)

If you are not comfortable with using it, you can post the following:
```
cat /etc/modprobe.d/blacklist* |grep -i -e b43 -e bcm43xx -e ndiswrapper
cat /etc/modules | grep -i -e b43 -e bcm43xx -e ndiswrapper
```
