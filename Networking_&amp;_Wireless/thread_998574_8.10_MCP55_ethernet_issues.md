### Post by cariboo on 2008-12-01
After you made the changes to /etc/modprobe.d/config did you reload the module, and then restart networking. You will probably have to remove the module and then reinsert it. In a terminat type:

```
sudo rmmod forcedeth
```

then to insert

```
sudo modprobe forcedeth
```

If you don't get anything echoed back the command worked. To restart networking, in the same terminal type:

```
sudo /etc/init.d/networking restart
```

and just in case, in the same terminal:

```
sudo dhclient eth0
```
