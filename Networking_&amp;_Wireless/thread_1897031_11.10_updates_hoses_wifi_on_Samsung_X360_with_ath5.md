Hi,

try this:

```
sudo modprobe -rfv ath5k
sudo rfkill unblock all
sudo modprobe -v ath5k
```
Check also the BIOS settings and/or reset the BIOS

### Post by praseodym on 2011-12-18
Any buttons/switches/keys/key combinations (e.g. Fn+F2)? Try the Samsung-PPA from here:

[https://launchpad.net/~voria/+archive/ppa](https://launchpad.net/~voria/+archive/ppa)

```
sudo add-apt-repository ppa:voria/ppa
sudo apt-get update
sudo apt-get install samsung-wireless samsung-tools
```
Reboot after that
