```
sudo ethtool -s enp0s31f6 speed 1000 autoneg off

Driver version 3.2.6-k may be too old. Try the newer one attached

[https://superuser.com/questions/1104537/how-to-repair-the-checksum-of-the-non-volatile-memory-nvm-of-intel-ethernet-co](https://superuser.com/questions/1104537/how-to-repair-the-checksum-of-the-non-volatile-memory-nvm-of-intel-ethernet-co)

Alternatively, try the native one, version 3.3.5-10 from the Intel homepage

[https://downloadcenter.intel.com/de/download/15817](https://downloadcenter.intel.com/de/download/15817)
