### Post by aomlives on 2008-12-21
Because it should work with Linux, I'm guessing you have the DWL-G122 rev. C1. I have the same and it works straight out of the box without problems here. So for starters, you could check whether you really have a C1 version (look on the back of the adapter). Then check if the lights of the adapter are turned on when you plug it in. If this is the case, does a new icon appear in the upper right corner of the screen indicating wireless connection where you can check all the access points? 

If so, you can start connecting.

If not, then there might be a little more work to get it working.

In the meanwhile [this]("https://help.ubuntu.com/community/HardwareSupportComponentsWirelessNetworkCardsDlink") might help.

Good luck.
