### Post by him610 on 2024-04-03
Yes, provided the Cat5 cable is connected from your machine to the same router (wireless) that your other devices connect to using wifi.
I have been using a mixed environment such as you want for years. 
Connection by cable is preferable to connection using wifi. 
Only use wifi connections for mobile devices, or where there is no Cat5 cable.
