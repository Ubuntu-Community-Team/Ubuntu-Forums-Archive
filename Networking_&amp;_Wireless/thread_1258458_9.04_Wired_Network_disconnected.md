I see you have no eth0 in /etc/network/interfaces
If you have static IP, you should have something like the following there
```
auto eth0
iface eth0 inet static
address x.x.y.x
netmask 255.255.255.0
network x.x.y.0
broadcast x.x.y.255
dns-nameservers comma-delimetered DNS 
gateway your_gateway

```If you use DHCP, you have to write there auto eth0, if you want it started automatically. 

And I cannot see IP configured for eth0, i.e. inet addr. You should set IP for it, e.g.
```
sudo ifconfig eth0 down
sudo ifconfig eth0 inet 192.168.140.1
sudo ifconfig eth0 up

```After all configurations restart the daemon:
```
sudo /etc/init.d/networking restart  
```
