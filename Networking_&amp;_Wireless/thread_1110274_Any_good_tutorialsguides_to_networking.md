### Post by PhrankDaChickenGeek on 2009-03-29
I'd maybe start with learning how to setup a DHCP & SAMBA server on ubuntu. [http://www.howtoforge.com/](http://www.howtoforge.com/)
Search wikipedia for any unfamiliar terms.

As far as making your network run better, I'd stick with wired and just use wireless on those that _need_ it. Wired is faster, more reliable, more secure and easier to maintain. Get a 100/1000mbs 12 to 16 port switch for your network so you have room to expand. You can always plug in a wireless access point and use it as well.
