### Post by noob12 on 2007-10-01
Had you installed wireless drivers yourself on Feisty ?  If so, they won't carry over.  You need to redo the installation of those.

If you did not install drivers previously and you have a card that uses the orinoco_cs driver (like the Dell TrueMobile 1150) you  are probably bitten by this bug where they forgot to include that driver in Gutsy; a fix is committed, so it should get in before Gutsy final, but I think the current pre-release (Tribe 5) is affected still:  [https://bugs.launchpad.net/ubuntu/+source/linux-source-2.6.22/+bug/125832](https://bugs.launchpad.net/ubuntu/+source/linux-source-2.6.22/+bug/125832)
