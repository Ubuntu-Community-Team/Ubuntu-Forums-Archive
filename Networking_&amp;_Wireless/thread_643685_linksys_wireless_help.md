### Post by d_in_Conduct on 2007-12-18
First, you will need to find out which version the adapter is.  I believe that Version 4 is the only one supported.  If you have V4, it should work.  There are a number of approaches to making it work and plenty of HowTos explaining how.  You can start with the sticky at the top of this list regarding RT2500 and just start following the various links.  Yes, I know it refers to Dapper right at the start.  

I got the source code from Ralink, the maker of the chip used, and compiled it myself.  That will require adding some software to your system if you haven't gotten all the programs needed to compile source code already.  There are HowTos for that, too.

If you are running a 64-bit version of Feisty, it will be much more difficult.  I haven't gotten it to work on any 64-bit Linux system.  That doesn't mean it can't be done, I just couldn't do it.

You've got some research ahead of you.
