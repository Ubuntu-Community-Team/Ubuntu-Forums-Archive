### Post by oleink on 2010-03-19
Ok I was struggling with ndiswrapper and basically everything else.  I found a great way to install wireless on my laptop (although some people may not have these drivers to install.)  First I plugged into my router for wired internet, updated ubuntu and got started:
1) Go to System > Administration > Synaptic package manager
2) type in ndiswrapper (do not hit enter.)
3) install the top file (which will install the other two with it)
4) go to Applications> Ubuntu Software Center and install wine
5) Go System >Administration>Hardware Driver and select the correct driver for your wireless card and hit I believe it is "enable" if not it is the button near the bottom that will enable your driver to work with ubuntu and will download and install it for you.
6) Use wireless (you just have to connect to a wireless network and get started)


HAVE FUN:D SORRY IF THEY DONT HAVE YOUR DRIVER IN THERE
PS it only took like 5 minutes tops
