### Post by JimLS on 2023-03-27
Followed the example at [https://netplan.io/examples](https://netplan.io/examples) for "Connecting multiple interfaces with DHCP" which uses overrides to set port metrics.  Set the main nic to 100 and the isolated nic to 110.  Applied it and tested by booting with the main nic disconnected and connecting later.  Works as desired!  Thanks!  BTW, did this on my 20.04 install.
