### Post by mihaiv on 2008-10-03
To explore windows shares use nautilus. Under ubuntu menu use Places->Network. Also you can write smb://computer_name in nautilus address bar.
You can use network printers. Go to System->Administration->Printing and add a new printer. Chose the "Windows Printer via SAMBA" option.
