### Post by gonzalov on 2007-04-15
Yes, you will need samba to connect, unless your Windows box has FTP or SSH installed. What are you running? Dapper, Edgy or Feisty?

In the "Places" menu you can find a "Connect to network server" (or something like that, I'm translating since I use a Spanish talking Ubuntu...). There you have a very easy option of connecting to a windows share.

Of course, you must have file share active on your Windows box and a share on the folder you want to give access to.

Hope this helps.
