### Post by ITAndrew on 2008-10-20
In order to share files from you're PC to another using windows files sharing (sharing from you're PC to a windows PC) you would need to install samba.

The installation and setup is fairly easy and there are a lot of great resources..

I.E.. [https://help.ubuntu.com/community/SettingUpSamba](https://help.ubuntu.com/community/SettingUpSamba)

### Post by Warren Watts on 2008-10-21
I recently set up my Ubuntu server as a file server using NFS so I could put all my MP3's in one centralized place and have them accessible to all the Linux computers on my home network. It was quite easy, and I managed to get NFS set up and working in about an hour.

Unfortunately, after I proudly demonstrated my accomplishment to my wife, she wanted to know if she could have access to the files on her windows PC. My wife stubbornly refuses to adopt Linux, so I have one dual boot computer running windows and Ubuntu.

So, off I went to get Samba up and running on the file server. After reading several complicated guides to setting up Samba, I came across this guide:

[Howto: Quickly & Easily Setup Samba filesharing with windows for Ubuntu]("http://ubuntu-unleashed.blogspot.com/2007/08/howto-quickly-easily-setup-samba.html")

Using this HowTo, I had Samba up and running on the server in less than an hour and-a-half! Perhaps it doesn't get into all the nitty-gritty details of exactly how it all works, but for a quick-and-dirty guide I found it to be quite excellent. I would definitely recommend it for anyone trying to set up basic filesharing with a windows box.
