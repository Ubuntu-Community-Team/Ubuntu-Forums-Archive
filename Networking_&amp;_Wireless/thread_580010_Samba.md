### Post by mpokrywka on 2007-10-18
I guess there's no samba package on CD (probably because size constrains it is not included on live cd).
You can download ubuntu DVD, it should contain all usually needed packages.
One thing: this computer "doesn't have internet", but will work on some network (samba). I strongly advise you to arrange some kind of temporary internet connection (maybe through proxy) to check for security updates once for a while. You would not want your box to be compromised by other network peers - maybe this may seem very unlikely, but better safe than sorry.
