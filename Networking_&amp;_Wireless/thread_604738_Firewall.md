### Post by JawsThemeSwimming428 on 2007-11-07
You can install a graphical front-end for iptables (firewall) by installing Firestarter. Go to Applications -> Add/Remove and find Firestarter in the list. Once it is installed you should be able to start it up and it will walk you through the initial setup.

On a default Ubuntu installation, you really don't need a firewall because there are no world-facing servers or open ports. However, there is already one installed, it's called iptables. You can install a gui front-end (firestarter or guarddog) to help you configure it if you want.
