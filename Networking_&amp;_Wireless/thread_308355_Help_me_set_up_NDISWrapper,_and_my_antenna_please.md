### Post by FrodoB on 2006-11-28
You can never be sure until you try these things sometimes.  Publish the output of lsusb with the device connected. Get the USB ID and search for it and the word Linux on Google. That should show some other folks experience and wheter is it really a ZyDas or not.

This site claims it is a ZyDas zd1211 or zd1211b, if so these instructions should help:

[https://help.ubuntu.com/community/WifiDocs/Device/Belkin_F5D7050_ver_4000_%28ZyDas_zd1211b_driver%29?highlight=%28WifiDocs%2FDevice%29](https://help.ubuntu.com/community/WifiDocs/Device/Belkin_F5D7050_ver_4000_%28ZyDas_zd1211b_driver%29?highlight=%28WifiDocs%2FDevice%29)

if you complete the instructions and it still does not show up then the site was wrong.
