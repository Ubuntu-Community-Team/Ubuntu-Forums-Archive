### Post by t0mppa on 2009-04-09
If everything's working, there should be an icon on your Notification Area (just like System Tray in Windows) located in the upper right corner of your desktop, which you can click and then choose which network to connect to (just like in Windows). If you can't see that icon though, your dongle doesn't work with the default drivers and you need to install new ones (just like in Windows).
