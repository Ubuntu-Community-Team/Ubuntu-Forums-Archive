### Post by davidmb on 2008-09-17
Hi,

It's Alt + Left Arrow.
Alt + Right Arrow goes forward in history.

Other useful key combinations are Ctrl + T to open a new tab and Ctrl + F4 to close it.

### Post by Denestria on 2008-12-08
Type about:config into the location bar, type backspace into the search
Change browser.backspace_action to 0.
