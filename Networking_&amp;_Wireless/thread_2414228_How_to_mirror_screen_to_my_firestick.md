### Post by SeijiSensei on 2019-03-08
You can cast the desktop with a Chrome or Chromium browser and a Chromecast device. Amazon generally refuses to support Google technologies for competitive reasons. If the firestick supports Chromecast, you can try that.

I don't suggest using this method to cast streaming videos though. There's too much latency in the link to support video casting.

### Post by vijayanmsm on 2019-10-16
1. Install AirScreen app ([https://www.amazon.com/AirScreen-AirPlay-Google-Cast-Miracast/dp/B07CPZ698R](https://www.amazon.com/AirScreen-AirPlay-Google-Cast-Miracast/dp/B07CPZ698R)) in your FireTV stick. 
2. Start AirScreen in FireTV
3. Launch Chrome browser in Ubuntu -> cast -> sources -> cast desktop -> click on selected device (e.g. AS-AFTT[cast]).  The downside is it doesn't stream audio. 
There is an option to cast only particular application.
You can also cast the chrome tab or particular file. 
I tested. It works :-)
