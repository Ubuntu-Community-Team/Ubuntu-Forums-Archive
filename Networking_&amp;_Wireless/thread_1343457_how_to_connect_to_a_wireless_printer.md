### Post by teward on 2009-12-01
scan for network shared printers.  Go to (in Jaunty 9.04) System > Administration > Printing.  Then go to the Server menu, then click Settings.  Now, click "Show printers shared by other systems."  This will show you all the printers on your network.  This works in the University I'm at (and I can see every printer on-campus), so maybe this will work for you.
