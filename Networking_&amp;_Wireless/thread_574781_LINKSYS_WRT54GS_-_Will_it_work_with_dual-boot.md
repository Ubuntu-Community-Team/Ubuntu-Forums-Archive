### Post by Xenaco on 2007-10-13
Routers are not operating system dependent.  You can have multiple computers all running different operating systems on the same router simultaneously.  You access your routers setup via your browser.

So the short answer to your question is YES, but you will have to configure each operating system to use that computer's WiFi network adapter and connect to the WiFi network via the router.
