### Post by spd106 on 2007-12-26
If you are using a version of Ubuntu later than 7.04 then Network Manager will prefer to use only one interface at a time. This is a good thing because it can prevent routing problems.

If you want to use multiple network cards then open System -> Administration -> Networks and turn off Roaming Mode on one or both of them.
