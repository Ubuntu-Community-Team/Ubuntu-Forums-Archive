### Post by gali98 on 2008-09-20
Easiest way:
Plug into ethernet
Update
(System->Administration->Update Manager)
first click the button check button then install all the updates (this may take a bit)
Then go to System->Administration->Hardware drivers, and make sure all the boxes are checked. If they are already checked, then uncheck then check them again. Hit okay then after it closes out, restart and wireless should work...
If you have a problem, post back here.
