Friends,
I was trying to configure Internet on 7.10 Desktop version
I was using 240nline service of Beam cable in Hyderabad 
It gave me nearly 25 sleepless nights
(I will recommend this route to any one who wants to go deeper
into various aspects of Linux!)
Finally I could find a solution
It goes as follows
Log into the gateway as follows
[http://172.23.0.1/myaccount.html](http://172.23.0.1/myaccount.html) (replace your GW No)
This will work if you are able to ping the GW
Enter your Username and Password
The home page should open
This is ths page where the password and other particulars can be edited
Go to Client > Download > HTTP
It will again ask for UName and PWord
Follow the instructions
Users of Cyberoam, please try and let me know
