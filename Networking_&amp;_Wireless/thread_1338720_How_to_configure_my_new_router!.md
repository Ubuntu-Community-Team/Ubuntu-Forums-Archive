### Post by inobe on 2009-11-26
hi you can start by typing **192.168.0.1** into your address bar then hit enter.


a password prompt will appear, in the drop down menu select **admin** 

leave the password field at bottom **empty**

here is the step by step instructions from d-link, it's a pdf [ftp://ftp.dlink.com/Gateway/dir615/Manual/dir615_manual_100.zip](ftp://ftp.dlink.com/Gateway/dir615/Manual/dir615_manual_100.zip)

ultimately you want to do a manual configuration and avoid automatic because automatic requires it to be run on windows.
