### Post by 98174 on 2010-08-23
1) Connect to ethernet
2) Make a ad-hoc, use a WEP 40/128-bit key, not the passphrase
3) Connect with iPhone
3.5) If connection has stopped working on both iPhone and computer, disconnect from both ethernet and wireless, then join ethernet first and create the above network second
