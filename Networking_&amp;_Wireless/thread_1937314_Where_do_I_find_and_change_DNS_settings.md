### Post by lykwydchykyn on 2012-03-07
Typically it's default setting.  

If click the network indicator and select "edit connections", then go to the wireless tab and either add or edit a connection, the "IPv4 Settings" tab allows you to switch from automatic to either manual or "automatic (IP addresses only)" so that you can specify a DNS server.

The default is to obtain all the information automatically, and most of the time that's what you want.
