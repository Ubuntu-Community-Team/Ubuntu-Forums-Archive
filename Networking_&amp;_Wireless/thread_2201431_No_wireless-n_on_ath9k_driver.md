### Post by praseodym on 2014-01-25
Deactivate the hardware encryption of the driver:
```

echo "options ath9k nohwcrypt=1" | sudo tee -a /etc/modprobe.d/ath9k.conf
sudo modprobe -rfv ath9k
sudo modprobe -v ath9k
```

### Post by K3N8 on 2014-02-01
The last couple of days I took the time to invest in a new wireless network. I bought a wireless access point and replaced my wireless repeater. I read that a repeater is not nearly as good as an access point. I bought an Asus EA-N66. Set it as an Wirelss N access point an a gigabit network. Next I set it up as a 2.4Ghz wirless network and set the bandwidth on auto. This wireless card only supports 20Mhz bandwith on a 2.4Ghz frequency. The result is that I now have 150MB network connection!
