### Post by Floppyjoe on 2007-02-11
The recent kernel update probably borked your wifi. Do you know if you were using ndiswrapper drivers? If you were using the latest non-repository ndiswrapper version it will likely not work now untill you reinstall ndiswrapper. There are other reports of some chipsets being broken by this upgrade.
