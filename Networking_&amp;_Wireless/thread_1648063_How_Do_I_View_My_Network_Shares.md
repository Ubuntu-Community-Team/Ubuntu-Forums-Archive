### Post by k4rizma on 2011-01-14
I love simple solutions but hate when it takes forever to find the answer. 

To launch a File manager open a terminal and type in: 
```
nautilus .
```

Once the File Manager has opened there should be a Network option right above the trash on the left hand side.
