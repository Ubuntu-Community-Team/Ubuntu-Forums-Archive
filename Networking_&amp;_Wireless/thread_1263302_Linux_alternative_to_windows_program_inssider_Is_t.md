### Post by hal10000 on 2009-09-10
Here are some: wicd, wifi-radar, wlassistant, knetworkmanager. I like wicd most, you can install any of them with your package manager or on a commandline with: [COLOR="Red"]sudo apt-get install wicd[/COLOR], [COLOR="Red"]sudo apt-get install wifi-radar[/COLOR] etc...
