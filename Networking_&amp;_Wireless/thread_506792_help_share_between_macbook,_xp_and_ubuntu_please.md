### Post by happy-and-lost on 2007-07-22
You'll want to be setting up a Samba share. This is relatively easy to do. In the "System-->(Can't remember which, I'm on Xubuntu atm)" You'll see a "Sharing Folders" menu item. When you launch it, it'll prompt you to install all necessary Samba packages. From there, setting up a Samba share should be self-explanatory. There's plenty of Samba howtos out there if you get stuck.
