### Post by buttertoad on 2008-11-16
Yes it does. I've been using it for awhile now.

It does require Hamachi to be running on both ends however so I don't think that can work with the iPhone. If you are just on your local network you can use your iPhone with VNC and not worry about it. But to use hamachi you would need a hamachi app for the iPhone.

There are SSH apps for the iPhone that work well. Can you do what you need to do from the command line?


This is the great guide I followed for setting hamachi and VNC up: [http://ubuntuforums.org/showthread.php?t=135036&highlight=hamachi+vnc]("http://ubuntuforums.org/showthread.php?t=135036&highlight=hamachi+vnc")
