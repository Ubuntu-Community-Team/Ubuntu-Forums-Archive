### Post by sarojroy on 2011-01-06
This method worked for me -
1) Install these packages using software manager: wvdial usb-modeswitch-data usb-modeswitch
2) Download and install package from [http://sourceforge.net/projects/lg-evdo-reva-us/](http://sourceforge.net/projects/lg-evdo-reva-us/)

After installing these packages plug-in the Huawei EC122 device. It should be auto-detected.
