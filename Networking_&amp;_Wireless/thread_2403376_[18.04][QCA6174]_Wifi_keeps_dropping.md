### Post by praseodym on 2018-10-10
Try
```

sudo cp /lib/firmware/ath10k/QCA6174/hw3.0/firmware-6.bin /lib/firmware/ath10k/QCA6174/hw2.1/firmware-6.bin
sudo modprobe -rfv ath10k_pci
sudo modprobe -v ath10k_pci
dmesg | grep ath10k
```
