### Post by blackbird34 on 2020-02-17
Most computers only have one wi-fi radio. So you should be able to create a hotspot, but then you won't be  able to connect to the internet. There's a tutorial [here]("http://ubuntuhandbook.org/index.php/2016/04/create-wifi-hotspot-ubuntu-16-04-android-supported/") that explains all the details 

It's also possible to buy an ethernet adapter so you can plug a wired network connection into a USB port, such as [this one]("https://www.amazon.com/AmazonBasics-1000-Gigabit-Ethernet-Adapter/dp/B00M77HMU0").
