### Post by chili555 on 2016-02-29
> 148f:7601 Ralink Technology, Corp. MT7601U Wireless AdapterIn Ubuntu 15.10, the built-in driver mt7601u covers your device. It needs firmware that is not installed by default.

In earlier Ubuntu versions, you'd need to compile the driver from source code. Ooooh! My keyboard is tingling already!

Which do you have?```
lsb_release -d
```If you have 15.10, let's load the driver and check the log:```
sudo modprobe mt7601u && dmesg | grep mt76
```

### Post by Evil Wayz on 2016-02-29
I did the following, applicable for 14.04 - 15.04

sudo apt-add-repository ppa:thopiekar/mt7601
sudo apt-get update
sudo apt-get install mt7601-sta-dkms
sudo modprobe mt7601Usta

and wireless appeared.  Thanks.
