### Post by chili555 on 2020-07-06
Please try:

```
sudo apt update
sudo apt install build-essential git dkms
git clone https://github.com/aircrack-ng/rtl8812au.git
cd rtl8812au
sudo ./dkms-install.sh
sudo modprobe 88XXau
```
