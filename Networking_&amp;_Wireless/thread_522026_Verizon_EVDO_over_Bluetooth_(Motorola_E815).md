### Post by Mach1US on 2007-08-10
The problem you are haveing is that your pc's request for your own ip address is invalid .
The part where it says :
```
sent [IPCP TermReq id=0x4 "Could not determine local IP address"]
 rcvd [LCP TermReq id=0x3]
 LCP terminated by peer
```
explains that IPCP which is a module of NCP (network control protocol) which in turn is part of ppp protocol used for this WAN connection having problem with your local ip is receiving request to terminate a link.
You can google those terms with linux key words to find a fix or you may want to try my how to.
[http://ubuntuforums.org/showthread.php?t=343989&highlight=motorola+evdo](http://ubuntuforums.org/showthread.php?t=343989&highlight=motorola+evdo)
Hope this helps.
