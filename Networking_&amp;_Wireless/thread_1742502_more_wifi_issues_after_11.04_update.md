I have the same issue with my wireless from a clean install of Natty on an Inspiron 1501. I tried several things posted but they did not work. 

The Solution: Remove the STA driver that jockey-gtk installed and use Ndiswrapper. Try searching "Windows Wireless drivers" in the Software Center to install it. I can post a guide if needed.
