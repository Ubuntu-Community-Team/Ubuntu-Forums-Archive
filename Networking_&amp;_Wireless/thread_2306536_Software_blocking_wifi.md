### Post by jeremy31 on 2015-12-16
```
sudo modprobe -r acer-wmi
echo "blacklist acer-wmi" | sudo tee -a /etc/modprobe.d/blacklist.conf
rfkill unblock all
```
And it should work.  Reboot if needed
