Wireless Network troubles have been common over the past week or so with fresh installs on Ubuntu Karmic and Lucid install hangs at 85%. Needing to get back up and running in a preferred Ubuntu based distro I have easily installed LinuxMint8 which gave me my wireless, codecs and familiarity. 

Copy out Home/user directory to external USB drive (alway have a current copy) then after the distro is installed on your Linux partition then copy Home/usr back in. This should be common practise for those of us that "play" to much with updates.
