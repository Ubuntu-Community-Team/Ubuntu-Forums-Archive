Which driver is that? Broadcom-STA? That one can be installed via

```
sudo apt-get install --reinstall broadcom-sta-dkms dkms build-essential linux-headers-$(uname -r)
```
Edit: Reboot and deactivate SecureBoot, that driver is not signed
