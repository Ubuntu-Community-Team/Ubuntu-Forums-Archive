### Post by scorp123 on 2008-04-30
A client for Red Hat Enterprise Linux 4 can be found further down in this list:

[http://www.juniper.net/customers/support/products/aaa_802/oac_demo.jsp#oc](http://www.juniper.net/customers/support/products/aaa_802/oac_demo.jsp#oc)

The user guide is here:
[http://download.juniper.net/software/aaa_802/public/oac/docs/OACLIN43MAN.pdf](http://download.juniper.net/software/aaa_802/public/oac/docs/OACLIN43MAN.pdf)


That being said, you will have to try "alien" (read its manual page: "man alien") and convert the Red Hat *.rpm package into a *.deb package for Ubuntu. Or you will have to change distros. To start with Red Hat you'd have to go here:

[https://www.redhat.com/apps/webform.html?event_type=simple_form&eid=871](https://www.redhat.com/apps/webform.html?event_type=simple_form&eid=871)
