### Post by Iowan on 2009-02-21
In the top right corner should be a Manual Networking Configuration icon. You can verify that "Wired connection" is enabled.  (Should have option for wireless there, too). You might also check how the connection is configured - static/DHCP.

8.04 *would* let you have wired/wireless interfaces both active - but you need to edit **/etc/network/interfaces** file.
