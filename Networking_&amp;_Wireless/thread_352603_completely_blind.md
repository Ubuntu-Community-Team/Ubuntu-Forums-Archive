### Post by gradedcheese on 2007-02-04
Some PCI cards will work, some might not.  I usually start by looking here:

[http://www.linuxquestions.org/hcl/index.php/cat/10](http://www.linuxquestions.org/hcl/index.php/cat/10)
(pick a company, then look at some models and see if it's a PCI wireless card, and see what people say)

My usual technique, though, is to go to a store with a no-questions return policy and just buy any card that I feel like.  Then plug it in and see if it works.  If it doesn't, I return it and try another one.  The store I usually use has a 14-day return without any problems.  

The trouble is that with some manufacturers, a card may work fine but then they revise it slightly and change the chipset, rendering other cards (with the same make/model) useless for some time.
