### Post by mpokrywka on 2009-11-01
I use 8.04 kubuntu, but I got 9.10 installer to try newer network manager. I have one idea:

When you right click on Network Manager applet and select "Edit connections"
you probably have: Wired -> eth0.
When you click "Edit" button on the right,
there comes "Edit" window with "IPv4 Settings" tab.

On this tab there is "Method" drop-down list. Which setting is selected:
"Automatic (DHCP)" or "Automatic (DHCP) addresses only"?

If **Automatic (DHCP) addresses only** is selected, change it to "Automatic (DHCP)", apply, click Close on Network Connections.
Try reconnecting (left click on Network Manager and click on eth0/wired) and check if problem is fixed.

If **Automatic (DHCP)** is selected, then change it to "Automatic (DHCP) addresses only" and in "DNS servers" input box set: 4.2.2.1
Apply, close, reconnect, check if network is working.

(instead of 4.2.2.1, there should be addresses of you provider's DNS, but that can wait)
