### Post by SeijiSensei on 2014-02-03
You don't allow any UDP traffic as far as I can tell, so DNS queries will be blocked.  At least add a rule to permit UDP traffic to remote port 53, or to port 53 on the DNS servers you use.

Also I strongly recommend turning on packet logging while in development.  I suggest adding
```

    $IPTABLES -A ALLOW -j LOG 
    $IPTABLES -A ALLOW -p TCP -j DROP

```
right above the DROP rule.  You'll see which packets are blocked in /var/log/syslog.
