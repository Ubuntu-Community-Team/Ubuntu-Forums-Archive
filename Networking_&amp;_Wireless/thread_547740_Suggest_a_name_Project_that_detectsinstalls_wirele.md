### Post by bmartin on 2007-09-10
**[COLOR="Red"]WiFix is the name of the project. If you're interested in the development or would like to contribute ideas, please see [this thread]("http://ubuntuforums.org/showthread.php?t=549491").[/COLOR]**

A new project is starting. We'd like the community's help in naming it. Pick a name, or post a suggestion for a different name if you don't like any of ours.

Here's a description of what the program will do:
[LIST]
[*]Detect network devices that are present (wireless and potentially wired)
[*]Locate, download, compile and install drivers
[*]Configure kernel modules (i.e., /etc/modprobe.d/blacklist and /etc/ndiswrapper) as necessary
[/LIST]

EDIT: I created a logo for [WiFish]("http://blakecmartin.googlepages.com/wifish.jpg")... it's not my first pick of name (actually, it's my least favorite), but I thought it was a funny concept. I'm not much of an artist. We can update the image after we get past the "betta stage".

EDIT: Here's a [WiFix]("http://blakecmartin.googlepages.com/wifix.jpg") logo.
