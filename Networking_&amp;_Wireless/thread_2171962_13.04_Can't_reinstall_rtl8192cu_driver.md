### Post by chili555 on 2013-09-03
Let's take one step at a time:>  I have to open up a terminal and type "sudo modprobe rtl9182cu" in order to turn on the wifi dongle. That's probably because you did:> I blacklisted the built-in rtl8192cu driver,Le's undo that:```
gksudo gedit /etc/modprobe.d/blacklist.conf
```Remove the line 'blacklist rtl8192cu', proofread, save and close gedit. Now let's set the country code automatically:```
gksudo gedit /etc/rc.local
```Right above exit 0, add a new line:```
iw reg set US
```Of course, substitute your country code if not US. Proofread, save and close gedit. Reboot. Is there any improvement?
