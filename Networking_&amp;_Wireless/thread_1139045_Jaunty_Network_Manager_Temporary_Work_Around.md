For those having problems with the Ubuntu 9.04 64-bit Network Manager Bug: (see http://bugs.launchpad.net/bugs/191889)> I have discovered a temporary workaround while the program is being corrected in order to get onto the internet.  Another configration program: ***_xnetcardconfig_*** is a GUI program that can be installed and used via synaptic.  Although you have to manually configure everytime you boot up,  It is a good alternative for now.
