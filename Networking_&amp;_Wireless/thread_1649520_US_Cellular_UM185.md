### Post by Barry53 on 2010-12-20
I was able to get US Cellular's Pantech UM185 modem working on Lucid in two steps.
1 Install package usb-modeswitch. This will switch the modem from a CD-ROM device to a usb serial modem.
2 Using the network connections applet on the Mobile Broadband tab click add. The modem will show up as a Pantech device. Next select country, United States. Next select carrier from drop down list, US Cellular and hit apply. You will be asked to fill in the phone number, accept the default and leave the user name and password blank. The modem has to be activated on a Windows machine, so the user name and password are already om the modem itself. Be sure to check Automatically connect and hit apply. In a few seconds a ppp connection will show up in network properties sheet. Select it and your on the Internet.

The modem needs to be activated on a Windows machine to work.

What I don't know is how to access the modem to do things like power down the modem or keep track of my data usage like the US Cellular Connection Manager in Windows. Anyone know how to do this or know of a applet that will work with this modem.
