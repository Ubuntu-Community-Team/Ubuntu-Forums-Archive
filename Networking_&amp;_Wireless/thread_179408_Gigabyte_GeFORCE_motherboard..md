### Post by ve6ani on 2006-05-19
The motherboard uses an AMD Athon +3000.
Installed Ubuntu 64 n the box.
Had no network or internet working. 
Tried for hours to get it to it working. 
The nic is a NVIDIA nForce network controller with an 1394 NetAdapter.
Obviously UBUNTU doesn't support the Gigabyte motherboard.
Finally went into CMOS BIOS and dis-abled the on-board lan and then plugged in a D-link PCI card that I bought for $14.
That did the trick. UBUNTU booted up with the network and internet running.
Thought this might be of interest to someone trying to bring up a Gigabyte GeFORCE  motherboard.
