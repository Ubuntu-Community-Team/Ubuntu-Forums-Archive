Agreed. 

On the rare occasions I've used my Ubuntu machine over WiFi, all I've normally needed to do is to click on the network menu, click on the name of the network I want to connect with, enter the network's password if asked, and I've been ready to go. 

One thing to keep in mind is that if the router has access filtering based on MAC address activated, the router's settings will need to be updated to allow your device to connect.
