Hi, all!
I have 2 PCs weith shared LAN Internet connection. But sometimes the Internet stops and I need an alternative Internet provider. So I managed to configure a PC to use GPRS Internet, e.i. I connect my mobile phone and start the Wvdial. I read that I should restart the Firestarter after I dialed-up (there is an option in the Firestarter to avoid the restarting).
What I read everywhere was that I should be able to browse the Internet, but I cannot!
I figured out that I have to stop the Firestarter (from the STOP button) and only then I can browse the Internet.
But I need the Firestarter up because of the shared connection!
Can anybody help me, please?

P.S. The mobile phone (Nokia E50) is connected via USB sync cable - /dev/ttyUSB0

P.P.S.The dialup connection is only for alternative Internet until the LAN Internet is up. So I added to Panel 2 Network Monitor icons - one for eth0 and one for ppp0 - and when the LAN internet is down - I see an orange triangle on the eth0 icon. And when LAN Internet is back - the triangle disappears and I would just stop the wvdial. If anyone knows a better solution to this, please post it!
Thank you in advance!
