### Post by BETELGEUSE58 on 2009-10-26
First step to make them communicate is to make sure they are on the same network. 

Use these IP's if you are not sure what is and isn't the same network.

ip 10.1.1.30    subnet mask 255.255.255.0
ip 10.1.1.40    subnet mask 255.255.255.0

Set one ip for one computer and the other ip for the other.

Alternatively you can use these ip's with the same subnet mask as above.

192.168.0.30
192.168.0.40
