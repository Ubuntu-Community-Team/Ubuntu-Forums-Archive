### Post by thyriel on 2011-10-29
Hi,

after a long morning id finally managed with a minimal Installation if 11.10 to get my Wifi working :)
To let it autoconnect id made it like described in the Wiki for WPA2 PSK WLANs (german [http://wiki.ubuntuusers.de/WLAN/wpa_supplicant](http://wiki.ubuntuusers.de/WLAN/wpa_supplicant) )

So basically id created a /etc/wpa_supplicant/wpa_supplicant.conf with the connection Infos, and added into /etc/network/interfaces
```

wlan0
iface wlan0 inet dhcp
wpa-conf /etc/wpa_supplicant/wpa_supplicant.conf

```

Works great so far, even connects automatically to the Wifi after some seconds when i turn it on later, but what really disturbs is the long time its now taking to boot when the Wifi is not present.
Its now trying to connect to the AccessPoint, if its not finding it it takes around 30secs or so, then a message appears "trying to connect 60 more seconds" or so...

Is there any way to configure that so it just tries to connect one time, and if that doesnt work just leave it ?
Or maybe if theres no connection after 10secs just continue ?

regards

### Post by thyriel on 2011-11-01
got it meanwhile...
If anyone else is having that problem, changing the wpa-conf to wpa-roam in /etc/network/interfaces makes the connection wlan0 optional
