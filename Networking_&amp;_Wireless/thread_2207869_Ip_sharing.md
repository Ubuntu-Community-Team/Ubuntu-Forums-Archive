### Post by zybersun on 2014-02-25
The more computers that share the same internet connection and use it at the same time, could create problems with lag and such when streaming or getting online. When computers are sharing the same internet connection, the speeds/bandwidth is also being shared with all computers/devices using that same connection, if they are using it at the same time.
