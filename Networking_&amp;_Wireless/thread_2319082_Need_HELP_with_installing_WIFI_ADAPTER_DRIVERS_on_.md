### Post by grahammechanical on 2016-03-31
You cannot use those drivers but what you can do is go to System Settings>Software & Updates>Additional Drivers tab & see if the utility offers to install a WiFi driver for you. We need to be connected to the internet and allow time for the utility to find the drivers.
