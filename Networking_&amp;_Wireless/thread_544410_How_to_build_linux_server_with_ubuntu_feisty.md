### Post by Vince00 on 2007-09-06
Just remember the server distro comes with "no" gui... only command line.

If you think you might want a gui install the desktop version and then you can add the server packages as you would normally...

For a new user Webmin is a great addition also.

I would install the desktop version, and then install webmin, and then pick one thing and start with that... Example ProFTP for a ftp server is a easy quick 10 min setup... then move on to something like samba for file server, and then apache for the web server etc.. pace yourself and enjoy it don't get frustrated.
