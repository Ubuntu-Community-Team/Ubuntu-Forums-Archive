### Post by stimpack on 2007-05-27
KGet is a nice gui download manager.

wget is a command line download manager and can be added as a custom in flashgot, however its super simple to use standalone.

wget -c [http://somedomain.com/somefile.avi](http://somedomain.com/somefile.avi)

the '-c' means continue, so if you stop it and want to restart you just use the same line again, it carries on from where it left off. You can resume ftp or http doesnt matter.
