### Post by sudodus on 2014-06-21
1. Are you running a live session with an Kubuntu install drive, or is it an installed system (installed into USB)?

2. Please specify your wifi chip/card. Some work 'out of the box' with 'buntu, some need some tweaking, some need a proprietary driver, some do not work at all.

3. Please notice that the 'buntu version 12.10 has passed end of life and is no longer supported. There is no 12.11 version. If you are running 12.04 LTS, you are running a supported version. See this link

[Forums Staff recommendations on EOL Ubuntu releases, in particular 10.04 desktop.]("http://ubuntuforums.org/showthread.php?t=2229730")
