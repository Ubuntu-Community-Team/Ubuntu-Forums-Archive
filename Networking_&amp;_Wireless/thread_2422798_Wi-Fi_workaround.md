### Post by oldrocker99 on 2019-07-13
Ubuntu MATE's welcome screen enables easy firmware installation on its Getting Started buttton. You can't, however, install it in a live session.You can install it alongside your current installation and see if it works, and a live session allows removable of a partition, so you can install the firmware, see if you like this Ubuntu flavor (as someone who started with 8.04, and fell in love with the GNOME 2.3 desktop. Then GNOME devs decided to fix what I considered not broken, and it ceased development. A user in Argentina, Perberos, got the source code and started fixing it. It was first released on Mint something or other, and a PPA soon followed. In 2015, a community Ubuntu MATE distro was released, and then Ubuntu MATE became an official Ubuntu flavor. You'll see that I use it. At idle, with no other startup programs running, uses <500MB, which I consider lightweight. 

MATE (pronounced MAH-tay, after the caffeinated Argentinian herb, not mayte) is very suitable for beginners and old hands alike. It is amazingly configurable; you can make it look like Windows 7 or OSX, As an Amiga lover back in the day, I love top-down menus, so I keep it the way it is. You have your choice of menus, too. You have complete control over the system. 

After some distro-hopping in a VM, this old hand considers Ubuntu MATE the very best distro he ever has used. [https://ubuntu-mate.org](https://ubuntu-mate.org).
