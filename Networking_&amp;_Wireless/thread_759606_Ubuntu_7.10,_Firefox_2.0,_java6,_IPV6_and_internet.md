### Post by djronh on 2008-04-19
Over the last 12 months or so the Ubuntu internet experience for the average user has been destroyed. Doing a search for IPV6 will reveal that firefox on many computers was slowed down to the point of not working. Java6 is not ready on either of my desktops so I had to return to Java5, complex Java applets are crashing firefox. Then we have the network manager which seems to be buggy, there are alternatives in the repositories. There are problems assigning DNS addresses, which you should get from your ISP and configure in the network manager. Configuration in a network manager should be done using automatic DHCP. This is all simple stuff that can be done using a gui. Things are not going backwards however, once you have set up Ubuntu it is superb! Internet connection setup is no longer user friendly and 'out of the box'.
