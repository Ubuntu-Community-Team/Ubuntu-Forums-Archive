### Post by grahammechanical on 2010-08-23
Did the router come with a set-up guide? Perhaps not. My ISP provided the router and the set-up guide. The information is standard. If your computer has wireless and it is working then all you need to do is tell the router how to connect to your ISP and what your ISP user name and password is.

You should be able to use network manager to select this wireless connection from those that are available. To use the connection you will need a network security key or pass-phrase, also called wireless key or WPA-PSK. This information should be written on the bottom or side of the router. Once you have entered this information network manager will remember it and if your connection is set to connect automatically it will do so every time you start the computer.

Once you are connected you can open a browser. It may automatically direct you to the router/modem's set-up page. You can follow the instructions to put your ISP information into the router's memory. You will need the router's user name such as admin and a router password. This should be the serial number recorded on the bottom of the router.
