We can deploy two or more APs to solve the signal coverage problem.
These APs has different BSSID (aka, MAC address), but are configured with the same SSID.

### Post by R4kk00n on 2011-06-19
The one with the better signal strength/quality.

I hope you'll manage to set the APs to non-overlapping channels (like 1, 6, 11). Or use 5 GHz range &#8212; they all are non-overlapping there.
