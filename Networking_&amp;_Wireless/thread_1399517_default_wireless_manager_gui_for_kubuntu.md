just so you guys know it is not called knetworkmanager in the repository. instead it is called network-manager-kde which is out of date and is now called
```
plasma-widget-networkmanagement
```

so if you ever accidently remove the kubuntu wireless manager, 
```
sudo apt-get install plasma-widget-networkmanagement
```
