### Post by abn91c on 2009-04-09
if the network notification icon is missing from the top toolbar, right click panel>add to panel> find the network notification widget and add it. then reset the network from there. Also in a terminal you can run a couple of commands
sudo dhclient eth0  for wired connection
sudo ifconfig
then post the results here
