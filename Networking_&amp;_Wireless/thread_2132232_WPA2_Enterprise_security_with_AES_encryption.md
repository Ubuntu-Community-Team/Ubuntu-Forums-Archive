### Post by Cheesemill on 2013-04-04
Go to System Settings > Network > Wireless then click on the arrow next to the network name, then on Settings.

On the Wi-Fi security tab you should find all of the details you need.

[ATTACH=CONFIG]240932[/ATTACH]
