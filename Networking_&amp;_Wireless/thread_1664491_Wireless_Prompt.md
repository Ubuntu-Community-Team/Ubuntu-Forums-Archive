### Post by grahammechanical on 2011-01-11
I do not know the answer but I imagine that what you are trying to do is no different from what anyone does when they want to access a web site. The computer connects to the router/modem by wireless. The web browser then connects to the web site whose address it has been directed to go to.

When I get a server not found message it is because I have either entered the wrong address in the browser address bar or my computer is not connected to the router. This could be because the router is switched off or disconnected from the telephone line or the wireless adapter in the computer is switched off. This is my set up at home. Is it not similar for you at school? You have to assume that the school network is working correctly. 

So, are you directing the browser to the correct web page? Is the computer connected to the network? Confirm these things and then you can look further.
