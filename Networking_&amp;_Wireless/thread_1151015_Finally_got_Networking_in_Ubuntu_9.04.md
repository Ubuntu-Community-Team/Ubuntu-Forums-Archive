### Post by quntumleap on 2009-05-06
I wanted to share my experience with my home network. I don't know why this worked but maybe it will help someone else. I have used Ubuntu since Dapper and always had a problem with my windows network. I had it working under 8.04 but it did not work with 8.10. I tried all the tutorials and could not get anything to work. 

So when I did a new installation of 9.04 I decided to start from scratch. I initialized my router, changed the SSID and gave each box on the system a new static ip address. I then tried to access my network from the Ubuntu box and it still would not let me access MSHOME, my network. It kept giving me a message that it could not obtain the file list from the server. But I noticed that there was a new group, called WORKGROUP, and I was able to access it and it showed the Ubuntu box print share and my NAT drive. Voila! So then I changed my windows network name from MSHOME to WORKGROUP, rebooted all the machines and sure enough I can see everything from every computer, including my Acer Aspire with EasyPeasy.

I don't know why and I am not that good with networks to be able to figure it out. If someone understands what I did I would be grateful for why it worked as I am always trying to learn more.
