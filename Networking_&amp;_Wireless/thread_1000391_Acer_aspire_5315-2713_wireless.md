### Post by entercomp on 2008-12-02
After quite a few hours in the forums and searching for drivers I found a combination that got my wireless card working. I installed ndiswrapper and used System-> Administration-> windows wireless to set a atheros driver 5.1.1.9 this is the xp driver and not the vista driver of the same name on the acer support site. The driver from the support site did not work. If you are having this problem google atheros 5.1.1.9 and download it.
I hope this save someone else some time.
