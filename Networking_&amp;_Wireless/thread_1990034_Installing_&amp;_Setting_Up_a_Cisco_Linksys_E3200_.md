### Post by chili555 on 2012-05-29
I haven't used that specific router before, but I have set up several other routers for wireless the manual way. It involves connecting by ethernet and logging on to the administration pages of the router with a web browser; for example, Firefox. Type in the router's address, usually something like 192.168.1.1, in the address bar and the log in window will appear. The user name, password and specific to the Cisco address are covered clearly in the manual.

I suggest you then navigate to the wireless section. Change the SSID or name to suit you; it's probably now set to Cisco or default or something generic. I suggest a name with no spaces, such as oldtree22. Set the encryption to the strongest possible, WPA2-AES. Select a strong, non-guessable password; that is, not containing your house number or similar. Save, close and done.

You will probably see a number of other features you can set. The firewall is probably on by default and I suggest you leave it on. You might study MAC address filtering which allows only the computers you specify to connect. These details and many more are fully described in the manual.

I have no hesitancy to set up any consumer router with Linux.
