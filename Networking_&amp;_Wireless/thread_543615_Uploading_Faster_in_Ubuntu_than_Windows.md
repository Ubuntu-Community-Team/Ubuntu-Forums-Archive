### Post by shewbox on 2007-09-05
Using my laptop to upload some larger files via the campus wifi to my shared hosting account, I noticed something quite interesting.  I was first uploading via FTP on my windows partition, and I was getting an average upload speed of around 95KB/sec.   I then booted into my Ubuntu partition, and uploading the same files to the same server is going around 330KB/sec!  I was using Filezilla in Windows and gFtp in Ubuntu.  Is this a sort of fluke, or does Linux really handle packets that much better than XP?  The speed difference is astonishing!  Now if only I could get some decent upstream speeds at home with comcast  :neutral:
