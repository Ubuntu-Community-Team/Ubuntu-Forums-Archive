### Post by bwhaley on 2007-07-29
Well, the question doesn't make a lot of sense. You don't exactly "broadcast an IP address" as you surf. Rather, your computer is identified by an IP address. That said, what I think you are asking is can you mask your IP address such that nobody can tell where the request is coming from. The answer is "sort of." I'm assuming you're asking about web browsing, and the closest thing is using an anonymous proxy server.  See the [Wikipedia article on proxy servers]("http://en.wikipedia.org/wiki/Proxy_server"). The end of the article has links to lots of free anonymous proxies.
