### Post by MartinChir on 2009-12-05
I'm a Networking student & thought I would start my first post with the following data:

```
NUMBER: *99***8#
APN: preconnect
AUTHENTICATION: PAP
COMPRESSION: ALL OPTIONS ON
IPv4: AUTOMATIC (PPP)
```The above input will allow Optus customers to access the mobile broadband service using ubuntu.

APN can also be set to **internet** for all users or **preconnect**/**connect** for prepaid/plan.

**Also if your modem connects as a wired connection, disconnect it & dial using your Optus profile.**
