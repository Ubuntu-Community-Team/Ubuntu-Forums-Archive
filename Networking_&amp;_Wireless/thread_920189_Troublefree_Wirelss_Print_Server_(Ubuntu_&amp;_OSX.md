### Post by ecosprog on 2008-09-15
Although I don´t have the Linksys print server you have been struggling with, I do use a combination of Linux and OS X computers to print to a remote printer hooked up to a NAS driver with a print server function.

All of the print servers I have seen so far rely on SAMBA to share the printer and I suspect that this might be the root of your problem. Try enabling SAMBA on your linux computer and see if you have any better luck.

If you already have SAMBA enabled then this probably isn´t your problem and you´ll have to give a bit more detail about what is happening with your setup.
