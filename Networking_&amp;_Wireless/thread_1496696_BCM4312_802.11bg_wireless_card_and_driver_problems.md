### Post by DavePlummer on 2010-05-29
From the menu, select System ->  Administration -> Hardware Drivers. Then select and activate the Broadcom STA driver. You should then be good to go. I'm sending this wirelessly from a Dell Inspiron 1545 with the same wireless adapter. Hope this helps.
