### Post by walt.smith1960 on 2011-06-01
You can't access the configuration pages?  Any idea why not?  I would reset it to factory defaults and see if you still have the startup problem.  Of course resetting to factory defaults will reset all passwords and remove any encryption so don't leave it like that longer than necessary.  If the problem persists while running with factory defaults, it seems like either new firmware or new hardware (new router).  I assume you've checked the manufacturer's web site for know issues?
