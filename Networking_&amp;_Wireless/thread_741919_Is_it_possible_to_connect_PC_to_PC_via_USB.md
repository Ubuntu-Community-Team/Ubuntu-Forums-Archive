### Post by Kevbert on 2008-04-01
Hi again.  It looks like this is possible - I believe that someone has already done this with Fedora (See [http://forums.fedoraforum.org/showthread.php?t=51097]("http://forums.fedoraforum.org/showthread.php?t=51097")
The cable I have is quite old as I used to use it with Win98.  The cable is a USB PC-Linq data transfer cable.  I've just connected it between my XP laptop and Ubuntu desktop.  The XP laptop doesn't recognise the Ubuntu machine but the Ubuntu machine recognizes the XP one as a 'network connection.  Unfortunately I'm not conversant with Samba (see link).  You may be able to find more info if you google 'USB PC-linq linux' with quotes.
