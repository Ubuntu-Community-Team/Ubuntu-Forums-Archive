### Post by mssever on 2006-11-27
As long as the router is able to access the Internet, then any computer that's connected to it can, as well. Generally, you don't have to do any configuration, unless you're using static IPs. (If you don't know about static IPs, then you probably aren't using them.) But, you can easily make the computers talk to each other if you want them to.

By the way, most routers are configured using a web browser. Just type in your router's IP address (the default on Linksys routers is 192.168.1.1--other brands vary) from any computer connected to it and configure to your heart's content.
