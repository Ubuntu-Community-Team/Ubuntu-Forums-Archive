### Post by 12quality on 2005-08-11
Linux support for wireless cards can get pretty complicated as vendors add/withdraw support for drivers for their cards.  Also, many manufacturers keep changing their chipsets on the same model cards, thus making it hard to determine if the card is compatible. Generally, 802.11.b chips are easier to find support for than the newer 802.11.g cards.

The RT2x00 open source driver project for Linux supports cards based on the Railink RT2500 chipset.  See [Ralink RT2500 chipsets based WiFi wireless IEEE 802.11g 54 Mbps 2.4 GHz devices](http://ralink.rapla.net/).

A useful site about wireless cards on Linux is provided by [an open source project sponsored by Hewlett Packard](http://www.hpl.hp.com/personal/Jean_Tourrilhes/Linux/Wireless.html) .
