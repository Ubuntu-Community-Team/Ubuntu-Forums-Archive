Hi,

I have a USB bluetooth adapter on my edgy desktop, and I can send + receive files between this and my Sony K750i and my wife Nokia something.

The software you need it called ´bluetooth file sharing' or gnome-obex-server.  Install using Ádd/Remove or synaptic, then just start it and you will get a blue antenna icon in your taskbar, then you can send files from your phone.  A message will pop up asking where to save them.  To send files from your computer to the phone, right click on the file and choose ´send to´, then bluetooth.#
