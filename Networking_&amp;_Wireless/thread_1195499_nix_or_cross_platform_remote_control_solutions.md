I am compiling a list of remote control solutions (*nix only) (preferably unattended host possible) Can you contribute any others? Please and thank you!

-X over ssh (open source, available in any distro, not a complete solution, not for remote assistance...)
-VNC and variants(open source, cross platform, widely used, similar to RDP on Winders)(x11vnc,tightvnc,ultravnc,xvnc,realvnc,...)
-freenx (free but partly closed source, supposed to be fast but I have not seen anything conclusive myself) I WAS using this until I couldn't overcome a keyboard issue a while ago...
-XDMCP (I don't know much about this, terminal server-type remote control?)
-webex (commercial/subscription/web based)
-showmypc (webex clone sort of only some free options)
-pcAnywhere (not free, originally only w32, and linux client not impressive)
-pcxware and exceed (commercial windows > unix remote control)
-gotomypc (commercial, web based)
-yuuguu (web based)

wrappers
-gitso
-krdc
