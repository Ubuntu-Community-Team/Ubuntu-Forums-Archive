### Post by spd106 on 2005-10-19
As long as you are buying a 802.11 'wi-fi' compatible wireless router there should not be any problems. Most of the consumer level routers ie belkin, d-link, etc all work independantly from your PC. Often they're even running an embedded/custom version of linux. Make sure that if you want to use 54 Mbps ie 802.11g then the router and nics must support that standard. 

It is usually better to buy a router from the same manufacturer as your nics, this is because they implement the wi-fi standard differently. So those 'extra special, super dooper, fantastic' speeds that their marketing dept wants to push on you, will only work with their products. Stuff like pre-n and 108 turbo modes.

Basically any 802.11b/g router will happily work with any 802.11b/g nic, if it doesn't, then it's faulty. Wow, standards that actually work!:eek:  

If I felt thet it would make any difference I would suggest you buy from a manufacturer that is linux friendly in all their products. As a reward so to speak. But the choice is yours.
