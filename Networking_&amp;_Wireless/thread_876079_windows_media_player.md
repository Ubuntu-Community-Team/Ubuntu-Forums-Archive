### Post by jualin on 2008-07-31
You don't need to run Windows Media Player to share your music from Windows. In fact I dont think that using Windows Media Player will solve your problem. If your "windows music" is in the same computer then you can access it by going to Places and then selecting your Windows Partition. If you want to share the music from another computer, then you need to set up a network so you can share your files using the network using Samba. Hope this helps!
