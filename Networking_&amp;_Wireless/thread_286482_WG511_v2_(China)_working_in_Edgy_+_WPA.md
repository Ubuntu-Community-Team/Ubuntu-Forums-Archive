### Post by acegolfer on 2006-10-27
Finally got it to work.

The key is to use ndiswrapper 1.8 (not 1.1) with WG511v2 (updated) driver. I gave up network manager as it takes 30 seconds more during booting. I'm now using wpa_supplicant.

I'm using a pretty old laptop (>8 years). My bootup time is 1 minute 45 seconds vs 2 min 30 sec for XP.
