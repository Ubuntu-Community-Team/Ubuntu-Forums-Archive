### Post by Claus_T on 2007-05-28
I'm a noobie to Ubuntu also - Feisty Fawn, so I'm not sure i can help.  However, first of all, I'm wondering if you didn't get your router's model number wrong, I couldn't find any reference to a DG8**43**G - but Netgrear does have a DG8**34**G model.

I'm guessing that your router may not be set up right.  When you first installed it, did you use the default install?  You may not have it's DCHP setup correct.  I would recommend reading the Router Manual.  You may have to re-setup the router defaults (using your Windows computer).

I have a Netgear WGR614 v6 router and it works fine (but I'm using a wired connection from my "test-bench Ubuntu" laptop - the wireless USB adapter that I had wasn't Linux compatible, so I bought a Startech USB1100S USB to 10/100 adapter - which is Linux compatible and cheap).  When I got the router, I did a default setup from my Windows XP desktop, which sets the router to be DCHP (and not the computer), if I recall correctly.
