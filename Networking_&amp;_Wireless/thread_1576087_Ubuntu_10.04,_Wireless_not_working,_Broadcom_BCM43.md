### Post by MaindotC on 2010-09-17
Dineshs did post a good link but in the future please consult the [Ubuntu Wireless Troubleshooting Guide](https://help.ubuntu.com/community/WifiDocs/WirelessTroubleShootingGuide) and let us know if you have any difficulty and precisely where so we can better help you.  It describes in detail the process of installing modules which are drivers.  Some users get this device working using the wl proprietary broadcom driver, and some use ndiswrapper.  Either way, we want your wireless device working so please check the guide.
