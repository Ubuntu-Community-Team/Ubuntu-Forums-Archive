### Post by amohanty on 2008-05-09
Sorry for the late response. Also I think I made a few assumptions about your familiarity with the network config files. 

First a few questions:
1. When you say you can connect with the first method does that mean you clicked on the network applet icon and tried "Connect to another network"?
2. When you get prompted for a password every so often, does it prompt for **your** password or the **router key**?

Finally if you want to try the manual method you need to do the following:

Approach A: Edit the config file
1. In a terminal type in:
> sudo gedit /etc/network/interfaces
2. This will fire up the editor and show you the file. In the file enter the following:

> iface wlan0 inet dhcp
wireless-mode managed
wireless-essid "Ruth Ann"
wireless-key insert_your_wireless_hex_key_here

save the file then in a terminal try:

> sudo ifdown wlan0 && sudo ifup wlan0

If it does not succeed please post the output here.

Approach B: Use iwconfig
1. Open a terminal and type:
You got this *almost* correct
> sudo /sbin/iwconfig **wlan0** mode managed retry 16 essid "Ruth Ann" key your_hex_key

you missed out the wlan0 part

2. Then bring up the interface:
> sudo ifdown wlan0 && sudo ifup wlan0

If this fails please post the output and also the output of the following command:

> sudo /sbin/iwconfig wlan0

Mark out your key with X's.

AM

PS: re wicd I think you need to run the following form the command line:

/opt/bin/wicd/tray.py

I might be off on the exact path but I think it installs wicd in /opt so you can look for it there in the file explorer.
