### Post by praseodym on 2014-07-24
Please try:
```

echo "blacklist brcm80211" | sudo tee /etc/modprobe.d/blacklist-broadcom.conf
```
and reboot. If its not working anymore, reverse it via

```
sudo rm /etc/modprobe.d/blacklist-broadcom.conf
```
and another reboot.
