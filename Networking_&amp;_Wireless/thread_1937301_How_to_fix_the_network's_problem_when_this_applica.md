### Post by varunendra on 2012-03-20
Sorry, I gave you unnecessary command, I've added the correct one (cat /etc/resolv.conf) in my previous post. Please post its output as well, just for confirmation.

However that mistake on my part seems to have happened for good, since now, by its output, I know there is at least one more thing you need to change/correct (actually I am just following praseodym's advice at this point, since he is the expert). That correction is to change "manual" to "dhcp" in 'interfaces' file, as he had earlier asked you to do: 
> **123ahah9 said:**
> ```
..
..
auto eth0
iface eth0 inet [COLOR=Red]**manual**[/COLOR]


```
see that "[COLOR=Red]**manual**[/COLOR]" entry in /etc/network/interfaces file? Let's change it to "**dhcp**" now. Open the file as root by:
```
gksu gedit /etc/network/interfaces
```

Then, change the above line so that it looks like:
```
auto eth0
iface eth0 inet dhcp
```
Don't touch any other line, leave them as they are. Save and close the file. Then retry:
```
sudo /etc/init.d/networking restart
```
Then retry to connect with pppoeconf.
