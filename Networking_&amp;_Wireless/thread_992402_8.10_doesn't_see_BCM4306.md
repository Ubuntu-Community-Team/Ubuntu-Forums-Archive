```

This is showing that the b43 and ssb modules are conflicting with ndiswrapper.  You can try the following:
```
sudo modprobe -r b43 ssb ndiswrapper
sudo modprobe ndiswrapper
sudo iwlist scan
```

You will want to also check ndiswrapper -l (lowercase L) to see if the device is present first.  If it is, you should be able to run those commands above.  The commands only work for this session though.
