### Post by kevdog on 2007-07-06
You need to unzip this file.  gz is an extension for gzipped.  

At the terminal:
cd ~/Desktop
gunzip scanModem.gz

I think this either creates a directory or file with the name of scanModem.  I believe its a file.

To use type
./scanModem

Good luck.
