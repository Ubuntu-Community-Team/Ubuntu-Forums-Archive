### Post by spd106 on 2007-06-20
I think I understand what you mean. There are several possibilities.

1) Download the official Ubuntu DVD, this contains more packages than the alternate cd and can be used as a live cd as well. It's about 4GB in size, [http://cdimage.ubuntu.com/releases/7.04/release/](http://cdimage.ubuntu.com/releases/7.04/release/) or find a local mirror. Though there are many of the popular packages on this DVD, it doesn't contain every single one that can be found in the on-line repositories.

2) A project called [APTonCD]("http://aptoncd.sourceforge.net/"), which aims to allow you to effectively backup the packages you have downloaded on to a CD along with all of their dependencies.

3) You could download the entire repositories using something like rsync on windows, then write them to several DVDs.

4) The repositories are available to purchase on DVD from some on-line stores.
