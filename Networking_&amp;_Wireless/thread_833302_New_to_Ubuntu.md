### Post by diaa on 2008-06-18
A simpler and more friendly method for exchanging files between connected PCs:
1- [Using Woof]("http://nothickmanuals.info/doku.php/blog:woof")
2- [Using HFS]("http://www.rejetto.com/hfs/"), it requires installing WINE because it's a windows application but it works very well under Wine, you can even drag-n-drop files into the program to share them.
