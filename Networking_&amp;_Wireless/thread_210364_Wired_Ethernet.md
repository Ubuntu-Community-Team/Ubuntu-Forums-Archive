### Post by woedend on 2006-07-09
if you are using a static IP address(and at times, even if not), you must specify your dns servers in the file
/etc/resolv.conf

if you open that file and find it empty, or just having comments and no servers, you must add your ISP's dns servers.  If you do not know them, there are two which should work (albeit probably slowly)...you would add them like this :
nameserver 4.2.2.1
nameserver 4.2.2.2

### Post by woedend on 2006-07-09
I see...you are right and I apologize for the misinformation(my linux is different, understand).  A little searching and apparently resolvconf autogens it every reboot using /etc/resolvconf/resolv.conf.d/base, so try using
sudo gedit /etc/resolvconf/resolv.conf.d/base
and adding them there.

if it works let me know please! this way I will edit my original reply as not to confuse people.
