### Post by l0b0 on 2010-06-21
When traveling, it would be great to have a painless way to keep the connection up and running automatically. In effect, a program that would try to keep you connected to the access point with the best signal at all times. Some useful features:
[LIST]
[*]Only try to change the connection if the quality is below some threshold
[*]After connecting, check that web pages don't redirect to a pay/registration page
[*]Keep track of bad and good APs
[*]Remove APs from the list after some time
[*]Allow the user to blacklist an AP
[/LIST]

If someone knows a piece of software which already does some of this, it would be nice to know. If not, it would be interesting if we could compile a set of commands to fulfill individual features, which could then be made into a nice script.

PS: I am aware that this is illegal in some backward countries. Fortunately, I don't live in a backward country :-)
