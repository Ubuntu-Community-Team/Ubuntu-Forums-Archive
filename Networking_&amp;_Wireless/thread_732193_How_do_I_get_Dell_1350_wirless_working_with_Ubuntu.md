### Post by dstew on 2008-03-22
I assume you downloaded the Ubuntu Debian package, with the filename ndisgtk_0.7.2-1ubuntu1_i386.deb. Did you download it to a Linux, Windows or Mac computer? If you have a CD burner, usually when you put a new blank CD-R in it, a window opens. Drag the file into the window. Then, you can choose "Burn these files to CD" from a menu, or sometimes when you close the window, you get a message box with "Burn CD?" or something like that on it.
