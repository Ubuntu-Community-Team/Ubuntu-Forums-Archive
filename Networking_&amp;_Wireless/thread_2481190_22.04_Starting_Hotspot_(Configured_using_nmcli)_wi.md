### Post by romeotangosiera on 2022-11-25
Well I answered one of my questions.
To controll logging use:  
```

nmcli general logging

```
I remove dbsmasq and used the embeded dbsmasg, just to reduce the number of things that can contribute to the problem.
I had to build my onw WIFI adapter for a [B]Realtek Semiconductor Corp. RTL8811AU 802.11a/b/g/n/ac WLAN Adapter
[/B]I did this for both Ubuntu 20.4.5  and 22.04.1
Wifi Access worked on both (wifi mode=intfasructure)

With wifi mode=ap I had no problems with 20.4.5, but it does not work with 22.04.1
(Connection made with:
```

nmcli dev wifi hotspot
```
