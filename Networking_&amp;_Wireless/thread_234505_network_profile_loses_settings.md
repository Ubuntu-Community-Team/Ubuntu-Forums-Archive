### Post by srt4play on 2006-08-24
Hello,

I too had trouble with the network profiles feature a while back. The way to create new profiles is not intuitive at all. What is does is when you choose "create location" from the dropdown menu, it takes a snapshot of your current settings. 

So delete your 2 existing profiles.  Then configure the first connection. Click "create location" from the menu and type in the name. Then configure the second connection. Click "create location" again and type in the name. 

HTH
