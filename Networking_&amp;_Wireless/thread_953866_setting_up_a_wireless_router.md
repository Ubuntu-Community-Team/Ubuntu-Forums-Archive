### Post by Agent Smith on 2008-10-20
You dont need any software for a wireless router itself. I myself use a Netgear 834GT and so long as you have wireless enabled on your laptop / pc, whatever software you run, it will see the router. Its just then up to you to put in the network key. I use Ubuntu 8.04. 
As soon as i installed the operating system it told me wireless networks were available. 2 mins after that i was on the net.
I have never used one of those usb dongles on any of the laptops i have had, as the wireless card has always been built in, so i dont feel im the right person to advise on that part of your question.
I have just picked up a wireless Belkin router on Ebay as a spare for £8. This also works fine. Hope this helps Jon.

### Post by cariboo on 2008-10-20
The only thing I use the CD that comes with the router for is the user manual. On a windows system they want you to use it to set the router up, but it is really not needed. There is usually a quick setup sheet that comes the router. Just find out what the ip address of the management interface is, and then you set it up.

I've only had experience with usb wired ethernet adapters. The one I have just plain works, you just plug it in and go.
