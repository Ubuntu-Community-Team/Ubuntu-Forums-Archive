### Post by dmizer on 2009-02-24
First of all, I suggest disabling network manager (it allows users to manage network settings without the need for administrative access): [https://help.ubuntu.com/community/NetworkManager#Disabling%20NetworkManager](https://help.ubuntu.com/community/NetworkManager#Disabling%20NetworkManager)

Reboot, and make sure it is disabled:
```
ps aux | grep NetworkManager
```
If it returns results other than "grep", use this method to disable network manager:
```
sudo update-rc.d -f NetworkManager remove
```

Log in with your administrative account and remove the user's ability to manage networking, and whatever else you want to disable (under: system > administration > users).

Configure wireless according to this howto: [http://ubuntuforums.org/showthread.php?t=571188](http://ubuntuforums.org/showthread.php?t=571188)
