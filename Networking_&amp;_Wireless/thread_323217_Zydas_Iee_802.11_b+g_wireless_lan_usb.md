### Post by Rosskat on 2008-03-10
Hey Zydas

The easiest way to install anything in ubuntu is to use "Synaptic Package Manager". There is no .exe equivalent in linux.
Go to:
System => Administration => Synaptic Package Manager.

Then click on "Search" and type in the name of the driver you found. If you can't find it in synaptic then you'll probably have to install it using the terminal.
