### Post by Munk3y on 2011-11-29
You're /etc/resolv.conf should look like this:

```

domain private.local
search private.local
nameserver 10.2.10.10

```

I'm not sure what hso0 is, that IP listed is one of the OpenDNS Server Addresses. The only DNS server that the machine should be using is 10.2.10.10. Then on that machine (The DC), run the "DNS" MMC and setup "Forwarders" to point to external DNS Servers. For OpenDNS, that would be:

DNS 1: 208.67.222.222
DNS 2: 208.67.220.220
