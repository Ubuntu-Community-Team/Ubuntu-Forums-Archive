### Post by coffeeaddict22 on 2009-05-06
If you can see the router, you've probably got an OK driver installed.  

From what you've written, I assume you're talking about the gnome network-manager applet- four bars in the panel at the top of the screen.  If you right-click on it, then select edit connections, you can see what's been entered.  Try putting in your password there; post back if it doesn't work.
If you like reading, there's a good wireless troubleshooter at [https://help.ubuntu.com/community/WifiDocs/WirelessTroubleShootingGuide]("https://help.ubuntu.com/community/WifiDocs/WirelessTroubleShootingGuide")
