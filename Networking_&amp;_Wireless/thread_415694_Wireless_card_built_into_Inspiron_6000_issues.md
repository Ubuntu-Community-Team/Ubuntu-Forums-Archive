### Post by Xipher on 2007-04-21
Well the first thing I should let you know is the wireless notification LED doesn't work in Linux, it's controlled by software and not hardware so you can't really use that as an indicator of the cards status. I do know that one works though, I have that model laptop (on it now, using wireless). If you go into the networking configuration (System menu -> Administration -> Network) you should see a wireless connection there that you can configure.
