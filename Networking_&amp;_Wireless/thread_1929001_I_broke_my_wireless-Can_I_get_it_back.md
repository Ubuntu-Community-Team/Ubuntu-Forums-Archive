### Post by westie457 on 2012-02-22
Hi, not knowing your particular router here is a rough guide for wireless setup.

Plug a cable from the modem to the router and reset then modem so it can connect to the router.

Plug a cable from the router to the computer. Open a browser and type in the explorer bar 192.168.0.1 (the default address of the router), hit Enter. The home page of the router should open and all settings can be done from there.

The wired connection should not need any setting up at all. If the home page of the router does not show or you cannot log in to the router then you will need to get the manual to check what the address is.

Any further problems post back here.
