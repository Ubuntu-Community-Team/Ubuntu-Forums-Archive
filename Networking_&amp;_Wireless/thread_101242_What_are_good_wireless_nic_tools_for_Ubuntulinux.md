### Post by blueplazma on 2005-12-09
From a command line you can use 'iwconfig' to get information about your connection.  If you're looking for easier to use tools, Kismet is a good scanner and NetworkManager is a good GUI tool to switch between wired/wireless networks.  It displays signal strength and available networks.
