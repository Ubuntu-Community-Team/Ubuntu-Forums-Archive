### Post by t-mo_ on 2009-01-29
On the Panel of your Desktop (by default in the upper right corner in the Notification Area) you should see a launcher for the Network Monitor. If not, right click on a empty space at the panel => add to panel => Notification Area

Right click and it gives you Informations about and let you edit the Connection, left click and it shows all existing Networks. Hope this helps.

### Post by Iowan on 2009-01-29
1) A direct connection will require a crossover cable.  You'll need to set up both machines with similar IP Addresses, and you'll need Samba to share files from Ubuntu to Windows.  Technically, you also need Samba to access shared Windows files, but the client comes pre-installed.

2) I presume there is a DHCP server to manage addresses.
