### Post by spd106 on 2009-08-03
That card has been supported out of the box for many years now. I suggest you try out a desktop (live) CD, just to be sure. It won't make any changes to the hard disk, it runs purely from the CD (or USB key), but you don't have the emulation layer in between.

See
[http://linuxwireless.org/en/users/Drivers/ipw2200](http://linuxwireless.org/en/users/Drivers/ipw2200)
