I doubt you'll need terminal commands, at least initially.  I had a 1005HAB if that matters.  Assuming a default gnome install, you should see 3 arcs in the upper right corner. If you right-click those curved lines you should see boxes labeled "enable networking" and "enable wireless".  Make sure both boxes are checked.  Sometimes one or both of these boxes are deselected.
