### Post by gerrymoth on 2007-04-23
I had trouble setting up wireless connection with 6.10, but upgraded to 7.04 and the built-in Network Manger worked great. All I had to do was click on the network icon at the top to see the available internet connections (routers), put my SSID & Password in and away I was surfing.
Why not download the 7.04 iso and put on a cd, then reboot and you'll be able to see if the network manager will work before having to install 7.04 if you don't want it.
