### Post by grahammechanical on 2016-06-22
I have no idea what 14e4-4315 means. I have no idea what script you have run.

If you are using Ubuntu 14.04 there should be a Network Manager icon in the right of the top panel. If you are connected to the router by ethernet (wired) connection then the icon will look like two arrows going in opposite directions.

Click on the icon and a drop down menu will appear that will list wireless access points that are in range. Select your wireless access point or WiFi network, as it is also called. A dialog will appear saying; "Authentication required by WiFi network." Type the wireless passphrase or key. It is also called WPA-PSK.

And that is all there is to it. Unless the WiFI adapter is not switched on. Unless the driver module for the WiFi adapter has not been installed. Which we do not know is the case.
