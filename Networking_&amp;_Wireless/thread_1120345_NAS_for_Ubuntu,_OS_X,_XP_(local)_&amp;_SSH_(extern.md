so basically it's possible to connect from outside to a NAS via SSH, right?

Yes, with a bit of hacking. The NSLU2 runs a Linux OS, and you can install things like an SSH server. For example: [http://www.nslu2-linux.org/wiki/HowTo/UseOpenSSHForRemoteAccess](http://www.nslu2-linux.org/wiki/HowTo/UseOpenSSHForRemoteAccess)

The NSLU2 also supports sftp by default.
