### Post by Tyconic on 2008-12-12
Addenum for anyone who is experiencing a similar problem:

The problem is fixed! 

Actually there was never really a problem. 

All I had to do was restart my machine and internet began working right way... Seems like restarting cleared out all the janky things network-manager did to my network settings. 

Now I am able to stop wireless with 

```
sudo ifdown wlan0
```

and start it back up again with

```
sudo ifup wlan0
```

If you're interested in trying wpa_supplicant as an alternative to network-manager you should check out this excellent guide: [http://ubuntuforums.org/showthread.php?t=263136](http://ubuntuforums.org/showthread.php?t=263136)
