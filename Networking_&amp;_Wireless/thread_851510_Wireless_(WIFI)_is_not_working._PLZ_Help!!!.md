### Post by lisati on 2008-07-06
There are different options for entering the WEP password which can make a difference to whether it is accepted, ASCII or HEX, and 64 bit/128 bit - make sure you have the correct combination.

There's also the "Keyring password" (relevant if you have roaming switched on) - have a look here: [http://ubuntuforums.org/showthread.php?t=463621&highlight=keyring](http://ubuntuforums.org/showthread.php?t=463621&highlight=keyring)
