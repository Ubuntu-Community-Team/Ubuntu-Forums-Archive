### Post by OfMacandMen on 2008-04-10
I love Linux -- I have only one compliant that is the network connection...

Here is what I mean, to connect to a AFP in all you hit Apple+K and enter the server..you are connected 

In Windows, you have the Remote Work Place (blah)

But in Ubuntu you would connect to server using SSH it will connect but not limit the user to their home folder and the share folder. User will have the ability to stomp all over the file system. Even though can not change anything vital it still sucks!

The other network connection issue is the implantation of a true network server. Hours will go into setting up LDAP and configuring client to join the network. As well as authenticate to the server and auto mounting  User & Shared folder. As IT professional this is easy for us to figure it out but for the lay person it is frustrating. I have 15 Clients on small Linux networks (2 -25 users) and after some training we are running fine, a few people still have a hard time grasping the concept. I believe in the KISS method (Keep It Simple Sucker) (the Shaft Version).

I will be open to any suggestion tho help facilitate these issues. I would like never to install another M$ server if possible.
