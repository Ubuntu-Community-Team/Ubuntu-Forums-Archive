### Post by 505 on 2007-10-05
Isn't that the same as setting up a Samba server to share with Windows? Once you have a server running, and you want to connect to it from Ubuntu, goto Places->Conntect to server and select Windows share, and fill in the whole window, and click connect. Now you have the connection on your desktop. You can also use smbmount to mount it to any folder.
