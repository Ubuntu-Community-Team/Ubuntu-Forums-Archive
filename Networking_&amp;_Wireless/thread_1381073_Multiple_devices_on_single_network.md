### Post by pirateghost on 2010-01-14
you can set up the older linksys wrt54g routers as bridges after they have been flashed with dd-wrt, and although i have never used them that way, i have heard that they work great...you might look on ebay for wrt54g routers and hit up the dd-wrt website for compatibilities on versions and router revisions....flashing new firmware adds a ton of new features and makes the router actually worth a crap.

i do, however, dislike wireless for so many reasons, and prefer to have a wired connection where possible.  if it is possible, maybe look to do a run of cat5 to the locations that need wired, it might make things more simple in the long run.
