For reference, I managed to get this USB adapter to work using the following:

```
sudo mkdir /lib/firmware/RTL8192SU

sudo wget http://svn.debian.org/wsvn/kernel/dists/trunk/firmware-nonfree/realtek/RTL8192SU/rtl8192sfw.bin
```

Followed by a reboot. I hope this helps someone else :)
