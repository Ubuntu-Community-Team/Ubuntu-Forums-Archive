### Post by madverb on 2009-06-23
You only need a firewall if you aren't already behind a hardware one i.e. ADSL Modem.
You might want to have an AV scanner if you will be sending files or emails to Windows users. It's not really necessary. Just do a quick search through these forums. You should be able to find a bit about 'anti-virus and linux'.

### Post by yota on 2009-06-23
It's not a silly question, it's just a "recurring" question.

Please search the forum for answers, you can start with:

[http://ubuntuforums.org/showthread.php?t=847087](http://ubuntuforums.org/showthread.php?t=847087)
and
[https://help.ubuntu.com/community/Antivirus](https://help.ubuntu.com/community/Antivirus)

in short: 
ubuntu is already secured out-of-the-box, and has a firewall built-in, so you don't have to worry expecially if you are a new user. Just don't authorize (giving your credentials) a program you don't trust.

If you are a more advanced user you can add an antivirus mostly to spot windows-virus (ubuntu is an immune carrier for them) in files you exchange with windows users.
Also the firewall can be tuned to better suite the user's need, there is specific documentation on that subject.

Hope this helps.
