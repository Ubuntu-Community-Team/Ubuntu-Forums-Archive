### Post by chili555 on 2011-06-22
10.04 handles WPA perfectly well. However, not every setting in the router against every driver and every device handle WPA perfectly well. In particular routers set to mixed mode WPA and WPA2 are very troublesome. If possible, set yours to WPA2 only or WPA only; not both.

As well, there is sometimes interference from other electric devices on the default channel 6. Please try channels 1, 2 or 11 in your router. If you are in a country that allows 12 and 13; try them, too.

Is your router set on N speeds only? Some driver and device combinations have trouble there, too. Try mixed B, G and N.

Also, there are two ways to do encryption; in hardware or in software. Let's switch. Open a terminal and do:```
sudo ifconfig wlan0 down
sudo rmmod -f iwlagn
sudo modprobe iwlagn swcrypto=1
sudo ifconfig wlan0 up
```Does it perfom as expected?

This is a temporary change; it goes away on reboot. If it is helpful, we'll write one short file to make it persistent.
