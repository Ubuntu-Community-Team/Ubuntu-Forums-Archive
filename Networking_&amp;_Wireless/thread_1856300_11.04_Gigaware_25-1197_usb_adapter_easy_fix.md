### Post by baxter33 on 2011-10-08
I came across this easy fix in the forums dealing with ubuntu showing adapter but keeping it disabled. These Gigaware adapters are really RaLink 3070 series wireless adapters(chipset?) The 25-1197 being a Ralink 3071.  Check Your exact adapter type by opening a terminal and type the code:  lsusb
 ex;  baxter@baxter-desktop:~$ lsusb

You should see similar info displayed:

Bus 005 Device 001: ID 1d6b:0001 Linux Foundation 1.1 root hub
Bus 004 Device 001: ID 1d6b:0001 Linux Foundation 1.1 root hub
Bus 003 Device 001: ID 1d6b:0001 Linux Foundation 1.1 root hub
Bus 002 Device 001: ID 1d6b:0001 Linux Foundation 1.1 root hub
Bus 001 Device 004: ID 148f:3072 Ralink Technology, Corp.   RT3072 Wireless Adapter
Bus 001 Device 001: ID 1d6b:0002 Linux Foundation 2.0 root hub

Notice 5th line, this lists your adapter info. If its a Ralink 307? series, then go  [HERE]("http://ubuntuforums.org/showthread.php?t=1769669")  "** Ubuntu 11.04 Realtek RTL8192SU firmware/driver issues"  **and scroll down to Box #9 and follow his directions (Says Ralink Technology, Corp. RT3072 Wireless Adapter but it works for 3071 & others too).                      This worked for my gigaware adapter. hope this post saves someone some time.
