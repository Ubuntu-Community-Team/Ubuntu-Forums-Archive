### Post by plehman on 2007-11-13
I was able to solve my problems by doing two things:

1)  I let the key manager automatically go when I log in (no more typing in the password every time)
2) I updated the firmware on my router which still had the original firmware.

As far as I can tell, everything is working better, and all my devices (Nintendo Wii, Windows Vista machine, Ubuntu machine) have been able to connect multiple times without issue.
