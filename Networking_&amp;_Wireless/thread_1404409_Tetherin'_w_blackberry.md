### Post by Whoopie on 2010-02-11
Googling for "blackbarry tethering linux" gave a lot of results.

You should have a look at the [barry project.]("http://www.netdirect.ca/software/packages/barry")

Packages for barry are available in the repository:
barry-util - Command line utilities for working with the RIM BlackBerry Handheld
barrybackup-gui - GTK+-basierte Oberfläche zum Sichern des RIM BlackBerry Handhelds
barrybackup-gui-dbg - GTK+-basierte Oberfläche zum Sichern des RIM BlackBerry Handhelds
libbarry-dev - Development files for libbarry
libbarry0 - Library for using the BlackBerry handheld on Linux
opensync-plugin-barry - Opensync Blackberry plugin, based on the Barry project
