### Post by bob-r on 2007-08-07
Ubuntu 7.04 successfully installed on my Micron Transport ZX Laptop with one exception; it did not recognize my U.S. Robotics Cardbus 10/100 Ethernet PC Card Model USR7901A.

The disk provided by U.S. Robotics does include a Linux driver; however, some knowledge of Linux programming is required in order to install it. Not being a Linux programming genius, I was hoping to find a card that Ubuntu would recognize.

My next choice was a D-Link Model DFE-690TKD. Ubuntu instantly recognized the card and Firefox was up-and-running without a hitch.

Perhaps U.S. Robotics will someday be included in the list of "plug-and-plug" devices recognized by Ubuntu?

That's my story and I'm sticking with it LOL
