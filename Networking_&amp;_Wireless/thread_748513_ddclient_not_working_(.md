### Post by darkazurka on 2008-06-16
Which wiki instructions? I don't find anything.

But no worry! My problem has been solved! I have just one domain name, and it's enough for me.
Just having the following in /etc/ddclient.conf file is enough and it works. It just updated! Horray!!! :) :)

> # Configuration file for ddclient generated by debconf
#
# /etc/ddclient.conf

pid=/var/run/ddclient.pid
protocol=dyndns2
use=web, web=checkip.dyndns.org
server=members.dyndns.org
login=myloginname
password='mypassword'
logigtk.home.dyndns.org
