Here are general directions for setting up an FTP server (you want option 1 or 2): [https://help.ubuntu.com/community/FtpServer](https://help.ubuntu.com/community/FtpServer)

As for the router, you will have to enable port forwarding on the router, which depends on the model you have. Also, make sure that whatever Cable/Modem/ISP you have won't mind you running a server.
