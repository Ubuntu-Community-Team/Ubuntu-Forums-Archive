### Post by jonobr on 2012-02-10
Hello


I notice your ip address is 192.168.1.x and your router is 192.168.2.x
These are on two different networks and wont talk to each other if they are put into the same network.

Your laptop would have to have a 192.168.2.x address to be able to talk with the router.

This assumes there is no router in between routing between the 192.168.1.x and 192.168.2.x networks.

### Post by jonobr on 2012-02-10
Depends on how you configure your interface.
If your using the nm applet to manage the ethernet interface,
change it in there.

Or , if y9our not using the nm applet then change the IP settings in the configuration file
```
/etc/network/interfaces
```

What you did is only temprorary and will be overwritten on reboot.

You could test by doing what you did
and then
```
sudo /etc/init.d/networking restart
```

Then test, but after a reboot it will always go back.

Cheers

Jonobr
