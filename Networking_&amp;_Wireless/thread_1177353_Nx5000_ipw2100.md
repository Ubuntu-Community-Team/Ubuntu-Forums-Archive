### Post by chili555 on 2009-06-03
Open a terminal and do:```
sudo cat /var/log/messages | grep -i kill
```That will tell us some information about the kill switch. The driver module is called, appropriately, *ipw2100*. Let's manually insert the module:```
sudo modprobe ipw2100
```If the terminal prompt just comes back with no message or error, then we know the module is, or already was, inserted correctly. Now does your network appear ready to connect when you click the Network Manager icon?
