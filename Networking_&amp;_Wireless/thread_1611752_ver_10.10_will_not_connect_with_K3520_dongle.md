Fixed by reference to dj-toonz fix as follows

open a terminal up

Applications , Accessory , Terminal

then copy & paste these

sudo rmmod usb-storage

sudo modprobe -r option

sudo modprobe -r usbserial

sudo modprobe usbserial vendor=0x12d1 product=0x1001

sudo pppd ttyUSB0

Did not think it had done anything but then when I plugged in the dongle and right clicked on the network symbol top right it was there and connected instantly.. Well done that man!!!
