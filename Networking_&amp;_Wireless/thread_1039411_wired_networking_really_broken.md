### Post by parminides on 2009-01-14
A couple of weeks ago a weird sound problem appeared with my Ubuntu 8.10 installation (cf. [http://ubuntuforums.org/showthread.php?t=1026031](http://ubuntuforums.org/showthread.php?t=1026031)). I was never able to fix the problem. 

Today I decided to reinstall Ubuntu to fix it, but this time I used Ubuntu-Studio, which is supposed to be geared towards audio applications. If a fresh installation of an audio-geared disto doesn't bring my sound back, then what would? Right?

During the installation of Ubuntu-Studio, some networking stuff (I presume Ethernet things) configured itself, seemingly without a hitch. At least it didn't complain about anything.

I had hoped to reinstall the Ubuntu-Studio in the same partition as the old Ubuntu, writing over it. However, I screwed up the partitioning and had to reinstall both Windows Vista and Ubuntu (Studio)!

Other problems were encountered, and I ended up reinstalling Vista and Ubuntu Studio a second time.

I distinctly remember that the wired network in Vista worked after the first reinstallation yesterday.

However, an installation of Ubuntu Studio later complained that it couldnn't configure the network, and from that point on, wired networking hasn't worked in EITHER VISA OR UBUNTU! (To add insult to injury, the sound never returned in Ubuntu.)

It doesn't seem possible that a software installation would "jump the partition" and break something in both partitions, but it's a very big coincidence otherwise. The computer, an HP TouchSmart, is less than a month old.

I know that wired networking was working in the first fresh installation of Vista, and I know that after an Ubuntu Studio installation complained that the network couldn't be configured, it hasn't worked since in either operating system.

To check that the wired network problem exists inside the TouchSmart, I plugged the ethernet cable into a separate computer (my laptop), and wired network showed up right away in the Vista Network Sharing Center.

Is it possible that my Ubuntu Studio installation broke wired networking in Ubuntu and Vista? That goes against my computer world view, but that certainly seems to be what has happened.

By the way, I currently have only Vista on the computer, reinstalled to hog the entire hard drive. Still no wired networking!
