### Post by SpaceTeddy on 2008-08-11
i have a different icon set installed, but as far as i remember in the "classic" icon sets, the network does not blink. If you are connected via cable, the icon will change (without the red x on the bottom) once it is connected - if you connect to wireless it should show four bars (blue and white) which indicate how good your reception is. 

Other than that, i don't think they do anything. If you want to see your network activity, i would suggest you use the systemmonitor applet. That one can show network activity.

hope it helps :)
