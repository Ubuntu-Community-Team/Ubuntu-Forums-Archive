### Post by Kobalt on 2007-03-23
You will need to download the scanmodem tool in order to determine which winmodem type you're using. 
[Here]("https://help.ubuntu.com/community/DialupModemHowto/ScanModem") is how to get and use scanmodem.
Once you know you modem type, [here]("https://help.ubuntu.com/community/DialupModemHowto") is how to install some models. If yours is not in the list then please post the modemdata.txt file generated by scanmodem so that we can help you.
