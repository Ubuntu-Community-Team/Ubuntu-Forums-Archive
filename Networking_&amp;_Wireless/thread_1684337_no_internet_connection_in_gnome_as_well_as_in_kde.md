### Post by ryan andrews on 2011-02-09
:guitar: well well i solved my problem on my own
it was so simple

ok if someone has problem like me follow the following instructions

go to 
preferences--network connections 

go to wired connection add 
name it something like auto etho 1
then uncheck the connect automatically
now left clik on connection symbol there you will see the new contion name 
click it 
and you will access to internet once again
