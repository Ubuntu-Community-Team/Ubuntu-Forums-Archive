### Post by tgalati4 on 2015-03-02
Network cards fail from time-to-time.  If it does not work with a LiveUSB, then chances are it is broken.  Is this a laptop?  Remove the battery for an hour and try again.  If it is a desktop, clean out the machine and put in a new network card.
