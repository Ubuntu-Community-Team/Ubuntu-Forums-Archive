### Post by ajgreeny on 2016-09-11
There is no real point in trying to use 11.04 as it lost all support a very long time ago and will be impossible to update in any way, therefore OS security will be very dubious to say the least.

I have no experience of using it, but forcepae may be your saviour and allow you to use a later version, eg 14.04 or maybe 16.04.

See:-
[https://ubuntuforums.org/showthread.php?t=2211590](https://ubuntuforums.org/showthread.php?t=2211590)
[https://help.ubuntu.com/community/PAE](https://help.ubuntu.com/community/PAE)
or
[https://help.ubuntu.com/community/Lubuntu-fake-PAE](https://help.ubuntu.com/community/Lubuntu-fake-PAE)
