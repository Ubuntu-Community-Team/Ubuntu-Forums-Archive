### Post by praseodym on 2014-11-07
Hi,

first: re-set the "interfaces" (do never comment out the "lo"-lines"):
```

echo -e "auto lo\niface lo inet loopback" | sudo tee /etc/network/interfaces
```
Reboot, reset the modem (some tiny button pressed with a pencil?!) and run:
```

sudo pppoeconf eth0
```
