Hello,

I'm using Ubuntu Server.
RA stands for Router Advertisements which are sent over WiFi by my router.
My Ethernet p3p1 interface gets all its IPv4 and IPv6 settings, but wlan0 gets only its IPv4 settings.

_**/etc/network/interfaces**_:
```
# This file describes the network interfaces available on your system
# and how to activate them. For more information, see interfaces(5).

source /etc/network/interfaces.d/*

# The loopback network interface
auto lo
iface lo inet loopback

# The primary network interface
# I've commented out the following line to allow the WiFi interface to come up when there's no cable connected
# auto p3p1
allow-hotplug p3p1
iface p3p1 inet dhcp
iface p3p1 inet6 auto

# Wireless Interface
iface wlan0 inet dhcp
iface wlan0 inet6 auto
```

_**p3p1 network settings**_:
