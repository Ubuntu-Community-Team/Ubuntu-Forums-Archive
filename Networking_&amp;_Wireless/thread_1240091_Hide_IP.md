### Post by dlmarti on 2009-08-14
This has nothing to do with windows vs. Linux, its a network issue.
Numerous http proxies exist, the setup is the same, just go into the browser and set the http proxy value to point at a proxy server you know of.

There are IP vpns, such as itshidden, setup it through the network manager.
