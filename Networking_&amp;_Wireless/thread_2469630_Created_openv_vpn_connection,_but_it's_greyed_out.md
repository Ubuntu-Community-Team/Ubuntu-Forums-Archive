### Post by TheFu on 2021-12-05
Support for Ubuntu-Mate 18.04 ended last April. That distro gets 3 yrs of support.
Run **ubuntu-support-status** to see the current state of your system support.  I bet over 1/3rd of the packages aren't supported.  Basically all the GUI and "desktop" programs are not supported, only the core programs - those used by non-GUI "Ubuntu Server" - have support.  Not every flavor of Ubuntu desktop gets 5 yr support. [https://askubuntu.com/questions/1161195/do-i-need-to-upgrade-mate-18-04](https://askubuntu.com/questions/1161195/do-i-need-to-upgrade-mate-18-04)
