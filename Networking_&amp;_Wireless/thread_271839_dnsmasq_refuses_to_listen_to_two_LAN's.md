### Post by appelblauwzeegroen on 2006-10-05
I'm using a pc (with dapper) as a NAT-router.  Internet is connected and succesfully configured to it with a USB-ADSL modem as ppp0.  Also connected is a PCI wifi card with a Ralink chip, working with the legacy rt61 driver in ad-hoc mode, this interface is called ra0.  This is the WLAN one which one ore more laptops are logged in for networking and internetting.  For the setup I followed more or less this guide: [http://www.gentoo.org/doc/en/home-router-howto.xml]("http://www.gentoo.org/doc/en/home-router-howto.xml")[http://www.gentoo.org/doc/en/home-router-howto.xml](http://www.gentoo.org/doc/en/home-router-howto.xml) 
Both interfaces can't be setup at boot-time, so this is done by a self-made script launched from rc.local:

```
# iptables setup:
iptables-restore < /root/netwerk/iptables_nat_060813
echo "1. firewall with NAT"

# LAN setup:
ifconfig ra0 192.168.0.1/24 up
echo "2. LAN (ra0)"

# WAN setup:
echo "3. internet (ppp0)"
echo -n "       "
while true; do
        if test -a /proc/net/atm/cxacru:0; then
                echo " modem connected "
                while test -a /proc/net/atm/cxacru:0; do
                    sleep 15;
                    ifconfig | grep -q ppp0 || ifup ppp0;
                done;
                echo " modem disconnected";
                ifdown ppp0;
        else
            echo -n "-";
            sleep 15;
        fi;
done
```

with iptables_nat_060813 (made with iptables-save following the ruleset in the gentoo guide)

First I tried to brigde ra0 and eth0 to br0 using brctl and replaced ra0 by br0 in the scripts above. I gave up this idea.

### Post by appelblauwzeegroen on 2006-10-07
I've found the ridiculously simple solution.  In dnsmasq.conf I've rearranged the lines as shown below and now it works.  Why is this not mentioned in the comments or the man-page?  ](*,) 

dnsmask.conf:
```
interface=ra0
dhcp-range=192.168.0.10,192.168.0.250,45m

interface=eth0
dhcp-range=192.168.1.10,192.168.1.250,45m

```
