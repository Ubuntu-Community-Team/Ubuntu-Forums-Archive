### Post by bswilson on 2006-10-23
Bill, the ip6 information you're seeing is a reference to version 6 of the Internet Protocol stack, called [IPv6]("http://en.wikipedia.org/wiki/IPv6") for short.

The post popular protocol for Internet communications today is IPv4, which is _heavily_ in use.  Whenever you see an IP address such as 192.168.1.1, that is an IPv4 address reference.

You didn't say where you're seeing this ip6 reference, but I believe that IPv6 is enabled by default on Ubuntu when you install it.  There are several tutorials and tips here in the forums on how to disable it if you want.
