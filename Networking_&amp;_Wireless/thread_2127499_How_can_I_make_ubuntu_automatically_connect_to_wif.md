### Post by varunendra on 2013-03-20
Welcome to the forums Ibraheem!

In the Network Manager > Edit Connections > Wireless tab, double click the connection name and put a tick mark in the "Connect Automatically" checkbox.

If it requires a password, fill it with correct encryption type in the "Wireless Security" tab of the same dialogue box. Then Save > Close.

Hope that's what you want.
