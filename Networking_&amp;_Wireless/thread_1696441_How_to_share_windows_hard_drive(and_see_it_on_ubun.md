### Post by JRV on 2011-02-27
First set-up the share on the XP machine.
I believe you will need to create a folder and share it.  If I remember correctly Windows will not let you share the whole drive. 

Now on the Ubuntu machine click Places==>Connect to Server 

In "Service type" select Windows share.

In "Server" type the name of the XP box.

In "Share" type the name of the shared folder.

In "User Name" type your user name on the XP machine.

Click "Connect"

After you have it working correctly do it again and check "Add Bookmark".

Type a name for the bookmark in "Bookmark Name".

Now the bookmark name will be listed in the Places menu next time you need it.
