### Post by Plumtreed on 2011-05-16
After setting up an Ad Hoc wireless connection on my laptop, that wireless connection can be seen by any wireless computer in range. That range appears to be at least equal to that of my router. 

I use a WEP connection as security because there is little chance of an unwelcome intruder where we are. It appears that other forms of security are available but I have not tried them.

To all intents and purposes this allows the laptop to act as a router so that any other computer that knows the security code can connect.

I suggest to you, that I may not need a router at all.

When we travel, we connect to the internet by the laptop via 'prepaid mobile broadband' and we have had up to two netbooks hanging off the laptop, wireless, Ad Hoc connection very successfully.

As far as file sharing and things, I seem to recall that we used Dropbox but since we can print I am sure file sharing can be incorporated.....but can't say for sure.

This all works very simply 'out-of-the-box', the only disclaimer may be that all my connections have always been Linux based.
