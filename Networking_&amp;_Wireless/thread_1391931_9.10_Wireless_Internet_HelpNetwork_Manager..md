### Post by UbuNoob001 on 2010-01-27
SOLVED!

So I assume this is pretty basic but I figured it out.

1. I had to download the Broadcom STA Driver (as stated in main post)
2. Using System>Admin>Hardware Drivers: activate the STA Driver
3. Restart
4. Upon looking Network Manager, no wireless networks were found....so 
            A. I knew the name of an open wireless connection so click "connect to hidden connection" 
            B. Type name/connect
            C. connected...then allowing me to see all other wireless connections! 

Not suggesting this is the way i should have solved it, but that it solved my problem in the way I had it.
