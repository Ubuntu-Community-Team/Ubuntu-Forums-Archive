### Post by steveneddy on 2008-07-13
To interact with any Linksys router, type this address into the location bar in Firefox, or your choice of browser:

**[COLOR="Blue"]192.168.1.1[/COLOR]**

hit the Enter key and then enter the router password.

Usually the default password is **admin**

and remember, only input the password as there is usually not a user name.

Most routers are accessible by this default address and can be administered via any internet browser.

**[COLOR="Blue"]The router doesn't care what the operating system is connected to it. Set it and forget it.[/COLOR]**
