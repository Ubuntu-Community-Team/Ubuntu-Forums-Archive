### Post by ubu_can on 2009-12-15
For the Linux computer, you should not have to do anything (ssh most probably will be installed. you may have to edit the sshd.conf file and initiate the service).

For the Windows machine, look here: [www.openssh.org/windows.html]("http://www.openssh.org/windows.html")
If you want to have ssh server on the windows, you may have to install cygwin.
