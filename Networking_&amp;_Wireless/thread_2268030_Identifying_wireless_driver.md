### Post by jeremy31 on 2015-03-05
The iwlwifi driver supports quite a few chipsets with either iwldvm or iwlmvm.
[https://wireless.wiki.kernel.org/en/users/Drivers/iwlwifi](https://wireless.wiki.kernel.org/en/users/Drivers/iwlwifi)

If you want to see the pci ids of all devices ```
modinfo iwlwifi | grep alias
```
