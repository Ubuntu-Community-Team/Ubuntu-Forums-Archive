### Post by jeremy31 on 2017-01-06
Network disabled, please check terminal results for
```
rfkill list all
```
If acer-wireless is part of those results do
```
echo "blacklist acer-wmi" | sudo tee -a /etc/modprobe.d/acer-wmi.conf
```

Reboot

### Post by jeremy31 on 2017-01-07
The acer-wmi module is loaded and causing wifi to be blocked.  A HP laptop should just have hp modules loaded like hp-wmi or hp-laptop
```
echo "blacklist acer-wmi" | sudo tee -a /etc/modprobe.d/acer-wmi.conf
```

Then reboot and see if wireless works
