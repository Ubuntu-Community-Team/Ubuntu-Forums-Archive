### Post by coffeeaddict22 on 2009-10-11
Wireless can be a challenge, although it's got a lot better in the last 12 months, and you're less likely to have problems now than you would have been a year ago.  

Try running Ubuntu off a live USB and see how you get on; if the wireless is going to be difficult it'll do so on that.  If it works, you can then go on and install.

If you need help setting up a live USB, The wiki has a page on it [here.]("https://help.ubuntu.com/community/Installation/FromUSBStick")
