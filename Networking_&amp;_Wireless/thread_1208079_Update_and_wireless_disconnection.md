### Post by ewan86 on 2009-07-10
Well just in case anyone has the same problem I installed an alternative program called wicd after searching forums to handle wireless networking. This uninstalls the standard network manager. Wicd worked after first then the same problem. The next time I booted into ubuntu Wicd did not work at all. So I connected to my router via ethernet cable and reinstalled the standard gnome network manager (which uninstalls Wicd at the same time), now the wireless is working better than ever, fingers crossed forever!
