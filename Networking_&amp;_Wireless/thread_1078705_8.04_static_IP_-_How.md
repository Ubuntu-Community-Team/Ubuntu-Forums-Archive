I have problems with the NM as well, so I just edit the interfaces file manually. /etc/network/interfaces. Add the following and you should be good to go after a restart.

auto eth0
iface etho inet static
address 192.168.1.234
netmask 255.255.255.0
network 192.168.1.0
broadcast 192.168.1.255
gateway 192.168.1.1

Don't forget to add your dns to the resolv file
