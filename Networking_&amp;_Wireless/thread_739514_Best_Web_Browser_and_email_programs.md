### Post by Eiríkr on 2008-03-31
Again, what is "best" totally depends on who you are.  Since we have absolutely no clue who you are, none of us can answer that for you.  I'd recommend that you try several different browsers and mail clients, and then decide for yourself.  

Possible browsers (mainstream packages in bold):
[list][*]**Firefox**
[*]**Konqueror** (KDE browser)
[*]Opera (not listed in Synaptic, you'll have to download it separately)
[*]Iceape (similar to Firefox)
[*]**Galeon** (Gnome browser)
[*]Epiphany
[*]Amaya
[*]Lynx (text-based)[/list]

Possible email clients (mainstream packages in bold):
[list][*]**Thunderbird**
[*]**KMail** (KDE mail client, also included as part of Kontact all-in-one Outlook replacement)
[*]**Evolution** (Gnome mail client, Outlook replacement, compatible with Exchange servers)
[*]Alpine (text only)
[*]Mutt (text only)
[*]Sylpheed[/list]
I'm sure there are other packages that I haven't heard of or forgot to list.  This should get you started.  Note that if you require compatibility with Exchange mail servers, Evolution is likely your best bet, though I understand that there are plug-ins for KMail / Kontact that might provide similar functionality.  

Cheers,

Eiríkr

### Post by r76 on 2008-03-31
Opera and Opera - works great on PC, Mac, linux - so you can use it anywhere.  It's really fast, not just the browser engine but because you have email in the same program cuts down on system overheads.  The email is structured more like gmail (views rather than folders), and the find-as-you-type search is so fast I don't bother making views.

You can port all your settings to any platform by copying one folder, and the current beta/ upcoming 9.5 release has shared bookmarks which you can sync between any computer you use AND your mobile (running the free and also great OperaMini) over the internet.  Loads of great skins, tango ones look nice in Ubuntu.  No need to find and update plenty of plugins - I fiind a default install has everything I need (e.g. mouse gestures), and a great user community like this one. Very customisable interface, mine no longer looks anything like the default, and like I say you can copy that to any future install on another system in seconds (no need to move buttons and bars around, and set preferences to get comfortable every time).

If there is a downside it's free (as in beer) but proprietary.  CAN be downloaded from synaptic manager - enable the "partner" repository ([see here]("http://tuxicity.wordpress.com/2007/10/18/howto-install-the-latest-opera-browser-on-ubuntu-gutsy-gibbon/")).  Or grab a .deb off the [Opera website]("http://www.opera.com/download/index.dml?custom=yes").
