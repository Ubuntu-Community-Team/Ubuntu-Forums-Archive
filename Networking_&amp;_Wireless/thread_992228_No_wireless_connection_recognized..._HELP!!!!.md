Sometimes in Ubuntu, the network configurator struggles to get the ip from the router. This leaving you with a invalid ip. If your router is DHCP enabled, this should'nt be a problem.

My advice would be boot into windows. Check the connection information and write down the Default Gateway and ip. Use this in Ubuntu as manual configuration.
