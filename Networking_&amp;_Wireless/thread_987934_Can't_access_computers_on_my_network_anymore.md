### Post by spcwingo on 2008-11-23
> **sirlizzie said:**
> Thanks, but I have no idea how to do that...

You type your router address in your browser.  System>Administration>Network>DNS to find the router address.  Open Firefox or whatever browser you use.  Instead of a web page address type the address you got from the steps described above (it will be a bunch of numbers and periods).  Then a window will pop-up asking for your username and password.  If you haven't changed your username and password check the router documentation for default username and password.  Inside of the router setup utility there should be something to the effect of "DHCP" or "addresses".  Find the computers that you're trying to connect to and there should be an option for assigning addresses via static or automatic with DHCP.  Choose static for those computers.  Write down the addresses assigned to those computers and their aliases.  Log out of the utility.  Open system>Administration>Network>Hosts>Add.  Add the hosts' names and addresses.  Then try to access them again.  Please let me know if this works for you.
