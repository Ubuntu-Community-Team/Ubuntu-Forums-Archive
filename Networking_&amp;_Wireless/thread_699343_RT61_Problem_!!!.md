So, what exactly is the problem? Isn't RT61 recognized automatically during installation? I am asking because I use the DWL-G630 Rev.C with the same chipset in it, the installation process ended perfectly, no further installation and/or confuguration was neccessary -- except ESSID and passphrase, of course. Please check the following in that order:

1. Both NICs up and working?
2. Same subnet, different IP adresses?
3. Ping possible in both directions?
4. Firewall(s) active?
5. Filesharing allowed on the XP box?
6. Same workgroup?
7. Same user and same password on both machines?

Check these things, and reply where you get lost.
