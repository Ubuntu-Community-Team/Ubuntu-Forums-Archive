This time, I've decided to simplify things and use an Ethernet cable and a vanilla desktop computer.

I've chosen an emachines T3256 that has an onboard ethernet adapter.  

I'm running the installation for the lastest Ubuntu Studio, and everything is going great.  My hardware is being detected and the installation is flying by.  It gets to the Network Adapter Detection part and it tells me that "No Network Adapter was detected".  I don't get it.  I don't believe there is anything special about the onboard internet adapter on this emachines system.

### Post by WaxwingSlain on 2007-09-13
Never mind.  I just remembered I had the NIC turned off in the BIOS because I was using a wireless adapter previously.

Ubuntu Studio is now working beautifully.  It recognized my pro USB audio adapter (M-Audio) and I'm off and running.  I'll try capturing video from my camera via Firewire later.
