### Post by toasterofirony on 2006-12-26
I'm at university at the moment and the halls I'm staying at has just initiated a wireless network that is giving my Ubuntu install a headache.

It runs from multiple access points on various channels, so if I look at it in Wireless Assistant I will see several entries, each of varying signal strengths. If I connect to one of them it will be working quite happily then die as the connection drops. I get the feeling that somehow the signal strength is cycling through the various APs, as the Windows wireless manager has exactly the same problem (except it only sees one AP). 

The curious bit of it is that the 3rd party windows software that came with my USB dongle (3Com OfficeConnect 11g) it's happy as Larry. As much as it's nice that *something* works, I'm not too joyed about having to use Windows for the next few years, just because my uni runs a crappy network.

Can anyone suggest anything to help? Oh, I should mention that this was with Dapper and a Wireless Assistant that was last updated in September.

### Post by jml on 2006-12-26
You might want to download and install network-manager and network-manager-gnome.  It deals with multiple acces points better than the other utilities.  It allows you to see all available access points and choose which one you want.

Another alternative would be to download and try the Edgy Live cd.  It seems to handle wireless better than Dapper does.  Just my two cents worth.
