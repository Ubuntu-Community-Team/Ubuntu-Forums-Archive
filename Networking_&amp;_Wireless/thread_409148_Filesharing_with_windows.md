### Post by Moiramses2 on 2007-04-14
I personally experimented the following : in order to visualize the content of the hard disk on my Windows computer from my Linux computer, I use SAMBA. There is no problem to copy data from the Windows PC to the Linux one.

But if I want to transfer data from the Windows PC to the Linux one, I only have  the following solution : under Kubuntu, after having created a web serveur, I type the URL of that server in the URL windows of the Windows computer. And then I can download the content of the Linux web server. Under Ubuntu, I am obliged to create an Apache server or similar to do the same.

There is maybe another solution but I don't know what it is.
