### Post by prshah on 2008-05-14
> **SilverGoldBronze said:**
> 
I tried the BLKWGDv7.inf file in the directory C:\Program Files\Belkin\Driver, but NdisGTK tells me that this is not the right one, so I beg of you, Ubuntu Forums, help me~

The file you have tried should be correct, but if it isn't, look in the "/windows/inf" or the "/windows/inf/other" directories for suitable INF files. Remember, the /windows/ directory will be available through /media/something/windows, where "something" is probably sda1 (your windows c: drive).

Or you can try a command-line approach, based on my (solved) thread (with expected outputs) ([http://ubuntuforums.org/showthread.php?t=756260](http://ubuntuforums.org/showthread.php?t=756260)), in which you can use ndiswrapper on the command line to install the INF file that you have mentioned above.
