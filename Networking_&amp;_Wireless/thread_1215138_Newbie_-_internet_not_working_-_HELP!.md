### Post by lukeiamyourfather on 2009-07-16
Saying there are bars in the top I'm assuming its a wireless connection. Make sure you're connected to your network and not the neighbor's. Also make sure the security is setup properly for the connection, click the icon to get a list of the networks in range and the configuration options (like WEP, WPA, etc.). If all of those are proper make sure you're getting an IP address. In the administration menu there's network tools which should give you an IP address of whatever network card you select from the drop down. Cheers!
