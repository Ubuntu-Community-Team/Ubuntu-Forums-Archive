### Post by spd106 on 2007-03-10
Please check that you have the linux-restricted-modules-generic (Edgy) or linux-restricted-modules-386 (Dapper) package install. You can do this in the Synaptic Package Manager. The packages you need should be on the install CD unless you have updated since install.
