### Post by kilps on 2007-01-14
How to connect would depend on what sort of connection you have, you mention a Ethernet cable - so I presume that that does into your DSL modem? (perhaps you should check that it does not go into a router - which would require a different sort of setup...)

If that is the case my guess is that you are needing to use a PPoE connection (again check this so you don't break anything by mistake ...) - take a look at [https://help.ubuntu.com/community/ADSLPPPoE](https://help.ubuntu.com/community/ADSLPPPoE) for that.

Perhaps if you could tell us what the ethernet connection goes into someone might be able to help you - unfortunately there are a few different ways for this to be setup - so you need to know which ...

Hope this helps
