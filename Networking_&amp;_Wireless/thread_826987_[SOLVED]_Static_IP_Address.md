### Post by Mako Eyes on 2008-06-12
This is confusing me. I'm trying to set up a static IP address for proper port-forwarding, but it never works. I put in the same information that I would in Windows:

Chosen IP address, Subnet Mask (which is input automatically, but still the same value) and my Gateway Address (which I assume is the same as what Windows calls the "Default Gateway"). The I move over to the DNS tab and put in the preferred and alternate DNS addresses. The numbers I put in all of these fields are the same as I the ones I input in Windows and it's always fail-proof in that operating system.

However, I think I'm doing something wrong because this always kills my connection in Ubuntu. Can anyone offer some advice?

Thanks.

EDIT: All fixed. I realized that inputting the DNS addresses *before* everything else allowed the connection to refresh with these new DNS entries and now my connection is up and running.
