### Post by t0mppa on 2009-11-10
You can find it on Synaptic, if you go to Settings / Repositories, then choose the Updates tab and enable the *Pre-released updates (karmic-proposed)*. After changing the sources, you just need to reload and then you can find the package with the help of the quick search field.
