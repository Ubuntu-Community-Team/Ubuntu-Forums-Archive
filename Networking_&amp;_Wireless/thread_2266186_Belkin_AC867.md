Hey All,

I just wanted to share my experiences with installing the drivers for this on 14.04... 

I'll just start off with a short story... I needed a dual band 2.4 & 5 GHz USB WiFi adapter. After some researching some as*****le on Amazon said that Belkin's AC867 USB WiFi adapter was completely plug & play for Ubuntu 14.04 ()in the reviews section), so I bought it. Unfortunately, it was not the case. My machine would recognize the adapter, but they would not play nice.

After some digging around, I found a Github project that you can download and build via make... saved me a lot of time and money. So, many, many thanks to blogger Daniel Scrivano [http://blog.danielscrivano.com/installing-rtl8812au-on-linux-for-wireless-dual-band-usb-adapters/]("http://blog.danielscrivano.com/installing-rtl8812au-on-linux-for-wireless-dual-band-usb-adapters/"), he also created the GitHub project.

GitHub URL: [https://github.com/scrivy/rtl8812AU_8821AU_linux](https://github.com/scrivy/rtl8812AU_8821AU_linux)
