### Post by PsychoFox on 2009-01-08
I can help with the "Invalid Permission" thing.  Try typing "sudo" before any command you type in the terminal.  Since you can't log-in as the root in Ubuntu, you can use the "sudo" string before each command to elevate your permission to admin status.  If it asks you for a password, enter YOUR password and press enter.  Hope this sorta helps :P
