### Post by changlinn on 2006-05-16
Which file manager are you using is it rox, or XFFM? AFAIK rox doesn't support natively browsing smb shares that aren't mounted to a folder.
XFFM however does have an icon in the left pane called "SMB Network" which will scan for domain and workgroups, it can be a bit slow however to browse the network this way. I would suggest you are better mounting a network share to a folder using fstab.
