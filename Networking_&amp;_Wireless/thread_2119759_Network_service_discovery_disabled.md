### Post by steeldriver on 2013-02-25
AFAIK having a disabled avahi service will only affect access by name to local resources (e.g. other LAN hosts / printers etc.)

It shouldn't affect access to LAN resources by numeric IP, and it shouldn't affect internet browsing

You could try changing your DNS server(s) (to use google or openDNS instead of an ISP-supplied DNS server for example)
