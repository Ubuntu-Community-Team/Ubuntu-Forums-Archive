### Post by FrodoB on 2006-12-03
I just bought one of those about a week ago and if you setup it up per the instructions it just may work fine.

I returned mine because it did not want to work with my laptop that has a Broadcom bcm4311 chipset running through ndiswrapper, but it did work with my Ralink rt73, Netgear WG111v2, and ZyDas zd1211b devices on Linux and my wife's Windows XP laptop. 

As my existing Linksys router worked with them all, the WGT624, which I got for $20 on sale for a backup was returned.

Set it up and give it a try I suppose.
