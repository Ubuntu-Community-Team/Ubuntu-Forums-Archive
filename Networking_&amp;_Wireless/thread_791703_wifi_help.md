### Post by Het Irv on 2008-05-12
broadcom is good enough, and all you need is an wired connection for a few min.

Hook up to wired and run updates, (System-Administration-Update Manager)
After Updates are done go to the restricted Drivers (System-Administration-Hardware Drivers)

You should see a line that has broadcom in it, with an unchecked check-box.  Check the check-box, and follow the steps on the screen. That should fix it.
