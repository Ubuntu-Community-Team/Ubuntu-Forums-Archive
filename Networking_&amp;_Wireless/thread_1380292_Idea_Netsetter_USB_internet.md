### Post by shredder12 on 2010-01-16
> **losersgiveup said:**
> ping 137.222.253.92
PING 137.222.253.92 (137.222.253.92) 56(84) bytes of data.
64 bytes from 137.222.253.92: icmp_seq=1 ttl=52 time=7504 ms

64 bytes from 137.222.253.92: icmp_seq=106 ttl=52 time=6135 ms
64 bytes from 137.222.253.92: icmp_seq=107 ttl=52 time=5128 ms
64 bytes from 137.222.253.92: icmp_seq=108 ttl=52 time=4123 ms
^Z
[1]+  Stopped                 ping 137.222.253.92


Good news.. this means that you are able to ping the IP addresses i.e. you are connected to the Internet but aren't able to resolve the website names. 

You will have to add some Open DNS servers to your /etc/resolv.conf file. 
try adding these 2 ips in that file
```
208.67.222.222
208.67.220.220 
```

So your file should look something like this
```
# Generated by NetworkManager
nameserver 208.67.222.222
nameserver 208.67.220.220 
nameserver 10.4.42.15
```

then restart your network and see if you are able to open websites now.
```
sudo /etc/init.d/networking restart 
```
