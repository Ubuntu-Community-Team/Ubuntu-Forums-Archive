I also recently got the Toshiba A205-5804, but since I see that others have had wireless trouble also, I have not opened the box yet, so as to avoid a big "re-stocking fee" if I have to return it.  But I will if I have to, as wireless is not a luxury on a laptop, It's mandatory.  

Consider this: if the internal built-in wireless doesn't work with Ubuntu, have you tried using a usb or "express card" wireless adapter that's known to work with Ubuntu 7.10?  I know the Belkin F5D7050 usb wireless G adapter works out of the box, but ONLY for sure if the box says 

"ver. 4000" on the bottom.  (ver. 5000 will NOT work correctly; other ver. #'s I don't know).

Check your local retail stores, and if you can find one, it should almost certainly get wireless connectivity.  You might have to disable the built-in wireless on the Toshiba, but that shouldn't be hard to do.
