### Post by jeremy31 on 2017-10-21
I would move the wireless router to channel 9 and disable power management with
```
echo "options 8192eu rtw_power_mgnt=0 rtw_enusbss=0" | sudo tee /etc/modprobe.d/8192eu.conf
```
Reboot

### Post by jeremy31 on 2017-10-21
Network Manager might be trying to enable wifi power management so try
```
sudo sed -i 's/wifi.powersave = 3/wifi.powersave = 2/' /etc/NetworkManager/conf.d/default-wifi-powersave-on.conf
systemctl restart network-manager.service
```

You can also find a good post by chili555 [https://ubuntuforums.org/showthread.php?t=2354328&p=13614520&#post13614520](https://ubuntuforums.org/showthread.php?t=2354328&p=13614520&#post13614520)
