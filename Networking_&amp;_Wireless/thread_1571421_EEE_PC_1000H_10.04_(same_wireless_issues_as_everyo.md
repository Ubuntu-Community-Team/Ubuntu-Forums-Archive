### Post by utilitytrack on 2010-09-12
Sorry me, actually I don't pay attention that you have a 2.6.33 kernel. Beta driver for RaLink RT2860 already present in this kernel.

This driver require some firmware that should be installed into /lib/firmware/. Check that it's right:
```
# ls /lib/firmware/* | grep 2860
```


So you can try to load it:
```
# modprobe -v rt2860sta
```

Now post here output of 
```
# ifconfig -a
```

and also 
```
# tail -f /var/log/kern.log
```

We should to see something unusual :)

blacklist rt2800usb

blacklist rt2800lib
