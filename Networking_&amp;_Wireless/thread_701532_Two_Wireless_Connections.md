### Post by himikeb on 2008-02-20
Not sure if you could get a speed-boost, but if you have different providers (or even if not), you could get a reliability boost.

I suppose, if you both had TWO wireless cards, you might be able to get a speed boost - configure each card to talk to a different wifi station...
