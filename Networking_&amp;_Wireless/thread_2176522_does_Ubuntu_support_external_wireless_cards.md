### Post by kurt18947 on 2013-09-25
I've had pretty good luck with two N150 USB 'dongles'.  One is Negear WNA1100 which uses an Atheros 9171 chipset &ath9k driver.  Another is Trendnet TEW-649U B, a RealTek RTL-8192SU powered device.  I find the RTL-8192SU powered device faster but it does have a suspend/resume problem.  That problem is fixed with the addition of a simple one line file.
