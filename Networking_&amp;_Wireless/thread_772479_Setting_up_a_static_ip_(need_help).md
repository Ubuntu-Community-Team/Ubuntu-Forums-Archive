### Post by rturner on 2008-04-28
You can set it up in Administration/Networking.  Choose static ip from the drop down box, type it in and type in the ip address of your router under gateway.  Then you have to go into your router and point the forwarding to your static ip address.  I haven't used azureus in a long, long time, so not sure of the port to forward it to in the router.
