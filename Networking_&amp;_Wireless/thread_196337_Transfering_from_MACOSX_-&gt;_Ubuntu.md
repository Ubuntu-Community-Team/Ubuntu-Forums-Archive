### Post by Drobbins on 2006-06-14
I have got it working after looking at [this]("http://www.ubuntuforums.org/showthread.php?t=159388") thread it is for a older version of ubuntu but the principle seems the same. 

Just thought i would post the soloution here incase anyone was in need of it.

**Soloution**

I had to open the terminal and create a smb user password for my account to do this you have to open the terminal and type > sudo smbpasswd yourusername you will then be prompted to type the samba password you require into the terminal and then confirm it once that is done make sure the folder you need to access is shared and you should be good to go.
