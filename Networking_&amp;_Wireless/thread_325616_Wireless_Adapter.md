### Post by SkimWear on 2006-12-28
i use a trendnet 54g usb wireless adapter

it's portable, small, and was not a big pain to install..

theres a lot of tutorials out there to install it. it's moderately easy.

mostly you will need ndsiwrapper to load the windows drivers. and if you use WPA-PSK you'll need to edit the conf files to include your SSID and your WPA-PSK.

if you use WEP its even easier.
