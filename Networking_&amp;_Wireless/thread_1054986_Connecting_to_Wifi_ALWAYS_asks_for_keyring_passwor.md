Is there a way that my Wifi connection will just connect when I boot my machine?  Everytime I boot up, I am asked for my keyring password.  For everything else, this is fine, but I would like my Wifi to just connect when I boot up without asking me for a password.  Is this possible?

### Post by mossman44 on 2009-01-30
I had this same problem as well. I fixed it by replacing gnome network manager with wicd. Follow the instructions at [http://wicd.net/](http://wicd.net/). It automatically uninstalls network manager and you have to reboot for the change to take effect. The interface is slightly different but not hard to work with. The best part is the keyring password prompt never appears again!
