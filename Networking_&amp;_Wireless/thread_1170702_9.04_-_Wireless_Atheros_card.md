### Post by snowman4839 on 2009-05-26
Try installing madwifi. Go to [the download page]("http://snapshots.madwifi-project.org/madwifi-trunk/") and either get madwifi-trunk-r4023-20090520 or the newest one (I'm using madwifi-trunk-r4023-20090520). You should be able to go to the main directory after you untar it and make it. If you cant then go to the command line and type "sudo apt-get install build-essential" and type "sudo apt-get install update" just for good measure. Then try to make it again. If you don't understand make then just google "compiling from source" or "make in linux" or something like that. Thats worked for me on my Asus EEE 900HA. I have an Atheros AR5007EG but it shows up as a "product: AR242x 802.11abg Wireless PCI Express Adapter" just like yours. Hope that helps :-)

### Post by hbetx9 on 2009-06-16
Thanks superprash2003! I just upgraded a IBM T60 Thinkpad with Atheros wifi chipset from 8.10 to 9.04 and after a couple hours of trying to "manually update network-manager settings" because of another post, this link solved the problem in about 2 min. 

For any using Atheros and upgrading to 9.04. It seems that the driver does install correctly but is not active out of box for some reason. Go to System -> Administration -> Hardware Drivers and check for the Atheros/Madwifi driver and make sure that this driver is active before digging really deep. Could save you time and trouble.
