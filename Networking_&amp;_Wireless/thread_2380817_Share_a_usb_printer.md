### Post by Morbius1 on 2017-12-22
> I added my printer in Settings->Devices->Printers Add printer but there is no Server option.

At the bottom of that page under Printers there is a button called: Additional Printer Settings.

If you click on that it will launch the old printers application which has the "server" button.
