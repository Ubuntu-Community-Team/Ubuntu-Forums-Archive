### Post by csoler on 2010-03-22
We're proud to announce the new version of RetroShare 0.5.0.

RetroShare is a friend-to-friend file sharing application, based on SSL for encryption and GPG for managing the friend web of trust.

Retroshare offers multi-hop chunk-based file transfer and search. Transfers are anonymous between non direct friends. In addition, RetroShare offers useful services such as public/private chat and messages, as well as internal forums. We provide Ubuntu install packages for jaunty/karmic 32b/64b here:

[http://retroshare.sourceforge.net]("http://retroshare.sourceforge.net")

This alpha2 version is very close to the upcoming 0.5.0 release. We consider it fairly stable. However, we'll be happy to receive detailed bug reports.
