The only exception for this that I have found is the 'Bluetooth Manager' which does a good job of handling pairing and file transfers but that is pretty much all it will do.

I found pairing some headphones and then getting the sound to output through the headphones an absolute nightmare of editing configuration files and changing sound driver settings. I've also looked at using my mobile phone as a remote control, and using the phone as a network device to connect to the internet but these both also require a lot of fiddling with various configuration files and terminal commands.

As I mentioned, various tutorials are available to do these kind of things, and if you follow them step by step I've found that I do get a fair bit of success. However these are likely to put of newbies and above all else are time consuming and require a lot of patience.
