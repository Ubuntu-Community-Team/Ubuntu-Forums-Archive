### Post by gtc on 2010-05-27
I use logmein free to control an XP box out in the shed. Lets me control XP from Ubuntu or any machine. Browser based. Does require the remote user to install a program and have it running, but they can turn it off to deny access. Does not require any changes to firewalls etc - as long as the remote machine can surf the net it can be controlled. Sort of think of it as skype for remote control :)  Free version has some limitations, no file transfers and no remote printing.
