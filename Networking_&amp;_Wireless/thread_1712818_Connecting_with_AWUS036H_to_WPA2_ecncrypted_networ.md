My system is Ubuntu 10.10, 64 bit
My home wireless network is encrypted with WPA 2
My laptop has an internal Centrino Ultimate-N 6300 using the r8169 driver.  My wireless router is some distance from the computer.  If I take the computer close to the wireless router, I can connect easily using Centrino Ultimate-N 6300 with the Network Manager. 
I then bought an Alfa AWUS036H wireless antenna in hopes of logging into my network easily.  This device uses the rtl8187 driver.
When using this device, the Network Manager sees my network but does not log in.  It keeps asking for my WPA2 password.
How can I log in using this device?

### Post by volkerbradley on 2011-03-23
I bought another one of these devices.  
The second one connected to my WPA2 encrypted wireless network immediately using NetworkManager.
Interestingly, both of them are Model No.AWUS036H.  The one that does not connect is described as 802.11g High Power.
The one that does connect is described as 802.11b/g Long Range.  It also has a symbol of the number 1 surrounded by a circle on the front.  The one that does not connect does not have this symbol.
There must be slight differences between the two.
