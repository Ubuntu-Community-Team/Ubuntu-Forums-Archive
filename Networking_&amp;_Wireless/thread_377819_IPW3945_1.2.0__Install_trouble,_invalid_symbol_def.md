I used the latest stable from the ipw3945 web site, which was 1.2.0 and then I just did: "IEEE80211_IGNORE_DUPLICATE=y make", IIRC. I could modprobe ieee80211, which means I already have that driver and don't need to compile it separately.

It didn't break my system, but even if you experience anomalies, the driver won't be autoloaded until you place the compiled ipw3945.ko in /lib/modules/your.kernel/kernel/drivers/net/wireless/ipw3945 .
