### Post by Claus7 on 2007-12-18
Hello,

Go to System->Preferences->Remote Desktop and tick the first two options. Have in mind that if you tick the option :
Ask you for confirmation

then you have to be in front of your ubuntu machine (the one you want to connect to), and you have to reply positivly so as to be able to connect. In addition giving a password it will be far more secure to attempt remote desktop.
