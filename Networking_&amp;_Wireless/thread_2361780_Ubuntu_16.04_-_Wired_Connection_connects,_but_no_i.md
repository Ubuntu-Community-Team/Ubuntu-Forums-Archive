### Post by foxly on 2017-05-20
This sounds like an issue I had this week with 17.04, [https://bugs.launchpad.net/ubuntu/+source/systemd/+bug/1682499](https://bugs.launchpad.net/ubuntu/+source/systemd/+bug/1682499)

The problem is solved after editing: /etc/systemd/resolved.conf 
set DNSSEC=off and reboot

Try that and see if it helps any, if it does, do an update afterwards as it should be fixed
