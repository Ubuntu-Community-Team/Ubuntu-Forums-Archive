All I can offer is that you try turning off IPv6 in Firefox:

Enter **about:config** in the Firefox address field.

Then enter **ipv6** in the newly displayed **Filter:** field.

Then double click on the one & only line displayed in the Firefox view window to change it's boolean value to **true**.

Then restart Firefox.
