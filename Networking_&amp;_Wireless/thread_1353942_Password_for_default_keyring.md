### Post by deathbyswiftwind on 2009-12-13
Did you use unsafe storage? If not try removing all the saved info for your connections and then remove the connection. Then reconnect and after you enable the new connection if it comes up asking you for a keyring password leave it blank and continue on. Youll get a warning about unsafe storage yada yada and accept it. Ive done this everytime and havent had a problem hopefully this will work for you.
