I just installed Maverick, and after rebooting for the first time my network manager vanished.
In case anyone experiences this problem, it happens because of your ICON THEME, of all things. KDE icon themes will make it fail, and possibly other non-default themes.

To fix this issue use the GNOME default icon theme, and it will work again.

I hope a fix is released soon, because this is ridiculous.
