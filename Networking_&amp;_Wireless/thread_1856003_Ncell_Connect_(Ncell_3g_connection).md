### Post by razorxpress on 2011-10-07
If you have Ncell data card, upon inserting the card into usb your network manager should list the new wireless connection. All you have to do next is follow the wizard and you should establish connection in no time. For those of you who want to do it manually go to Edit Connections -> Mobile BroadBand -> Add. After you press forward you can select the country (E.g Nepal) from country and region list. Next select the provider. Currently it lists only two providers i.e Mero Mobile (for Ncell) and Nepal Telecom. Select Mero Mobile and press forward. You should have mero as APN. After you apply this setting you should be able to connect. If you get stuck you can always edit the settings in Mobile Broadband section as in the picture attached.
