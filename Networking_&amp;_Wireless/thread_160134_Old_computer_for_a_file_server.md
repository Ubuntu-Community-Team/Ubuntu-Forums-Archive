### Post by rmjokers on 2006-04-14
That would be pretty simple to do.  I would install the server edition of Ubuntu to avoid all of the unneeded applications like X windows.  Then, install samba-server and an ftp server like vsftp.  It will take a bit of effort to get the samba install set up properly but there is plenty of help on the forums and wiki to get you started.
