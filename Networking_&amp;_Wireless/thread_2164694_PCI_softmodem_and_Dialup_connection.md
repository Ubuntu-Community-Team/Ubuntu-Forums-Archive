### Post by joseph2 on 2013-08-04
i have good news. by editing the grub.cfg file (located in /boot/grub). i was able to edit the following line:
```
linux    /vmlinuz-3.8.0-26-generic root=/dev/mapper/lubuntu--vg-root ro   quiet splash $vt_handoff 
```
by adding "pci=routeirq" to the end of the line. so the line now looks like this:
```
linux    /vmlinuz-3.8.0-26-generic root=/dev/mapper/lubuntu--vg-root ro   quiet splash $vt_handoff pci=routeirq
```
this has allowed lubuntu to change the IRQ of my modem from "255" to "17". but i'm afraid that this still may not be able to allow me use the modem in lubuntu as the network settings do not have any dialup options. but a friend of mine has suggested that i directly connect my lubuntu machine with my win2k machine via ethernet/lan port and attempt to access the internet through the dialup connection on my win2k machine. this may take some time to try as i don't have the proper cable. but as soon as i'm able to try it, i'll post the results. :)
