### Post by Mark_in_Hollywood on 2007-07-24
The various wireless & WiFi How-To's need the following added at the front of each one, so that newbies can make sense of what they are attempting:

Determine the chipset of the wireless device.
Install the driver for your device, if it's not already included.
Install firmware, if needed.
Load the module.
Set the wireless parameters (SSID, mode, security)
Set the network parameters (gateway, DHCP or static IP, DNS)
Automate the above to establish a connection on boot.
