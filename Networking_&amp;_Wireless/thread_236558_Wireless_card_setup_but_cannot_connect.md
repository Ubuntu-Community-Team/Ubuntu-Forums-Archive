### Post by Darkhack on 2006-08-15
The passphrase is not the actual WEP key.  You see, there are passphrases and WEP keys.  A passphrase is something that the user makes up and a WEP key is generated from it.  It is to make it easier to setup networks instead of retyping a very long network key over and over again.  For example if my passphrase is "doggy", I just need to type that in on all of my computers.  The "doggy" phrase will generate the exact same WEP key on each computer because the encryption algorithm is the same.  If you don't know your passphrase, leave that section blank and look for a WEP key field and enter it there.
