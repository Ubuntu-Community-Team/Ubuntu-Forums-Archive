### Post by weatherman2 on 2015-08-04
What kind of router?  How is it configured?  What security settings?  I prefer WPA2 + AES.  Avoid the obsolete "WEP" security, and also avoid "WPA2 + WPA mixed" mode - modern devices can handle WPA2 just fine.  Sometimes the mixed mode causes connection issues with only some devices.

Does the router manufacturer offer an updated firmware for the router?  If so, try installing it.

We're assuming here that your Ubuntu machine connected well to your old router and other routers?  If not, run the wireless script given in the "sticky" post at the top of this page and post the results here.
