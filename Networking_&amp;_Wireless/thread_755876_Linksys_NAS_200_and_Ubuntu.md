### Post by AntoMan on 2008-04-16
Well,
after some [**[color=green]further[/color]**](http://www.nas-central.org/index.php/Main_Page)  [**[color=green]searching[/color]**](http://www.smallnetbuilder.com/component/option,com_nas/Itemid,190/) about NAS devices working with Ubuntu, I've considered those NAS as possible choices:

I'll put it here ... *your opinions are welcome*!   ;)

[**D-Link DNS-323**](http://www.dlink.com/products/?pid=509)  **[color=blue]Pros[/color]**: firmware linux; good [unofficial support](http://wiki.dns323.info/); fast transfer speed rate.  **[color=red]Cons[/color]**: price; support to Linux is not officially given.

[**Linksys NAS200**](http://www-it.linksys.com/servlet/Satellite?c=L_Product_C2&childpagename=IT%2FLayout&cid=1175238051978&pagename=Linksys%2FCommon%2FVisitorWrapper&lid=5197810715B01) [color=blue]**Pros**[/color]: firmware linux [customizable](http://www.nslu2-linux.org/wiki/NAS200/HomePage); expandible   **[color=red]Cons[/color]**: slow transfer rate (not sure if this might change by modifying the firmware); it seems that Linksys wants to support only  Windows Vista.

[**Buffalo Linkstation Pro Duo**](http://www.buffalotech.com/products/network-storage/linkstation/linkstation-pro-duo/) **[color=blue]Pros[/color]**: web access; good speed rate. **[color=red]Cons[/color]**: Linux & file system Ext3 aren't supported (*it seems to me*); bundled applications work only with  Win & Mac Os.

[**QNAP TS-209 Pro**](http://www.qnap.com/pro_detail_feature.asp?p_id=82) **[color=blue]Pros[/color]**: it is exactly the device I wanted, and more; **[color=red]Cons[/color]**: price ... but,  I believe [[COLOR="Green"]it worths[/COLOR]](http://www.smallnetbuilder.com/content/view/30181/75/) all the € / $ they ask.
