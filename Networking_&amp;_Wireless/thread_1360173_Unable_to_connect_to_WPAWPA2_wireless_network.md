### Post by Pasdar on 2009-12-20
I was unable to connect to WPA/WPA2 wireless network in Kubuntu, I thought it might have to do with Knetworkmanager, so I installed gnome network manager (nm-applet)... that didnt work either... I then installed wicd and now it works.

If you have the same problem you can install wicd:
sudo apt-get install wicd

it will prompt you with two questions, answer yes to both... It will remove your other networkmanager frontends...
