### Post by mfox on 2009-02-11
I decided to try KDE 4 on my Ubuntu 8.04 installation so I added the KDE files to my present installation.  Booting up with KDE works fine except that wireless support is gone.  What's odd about it is that if I start up in Gnome, log out and log back in with KDE, wireless support is maintained.  I'm guessing that the problem is that network-manager doesn't work in KDE, and I would need the KDE equivalent, although this doesn't explain why wireless is maintained by logging in first with Gnome.  But getting the KDE equivalent from the repositories requires uninstalling network-manager, so I would probably have the reverse problem if I want to use Gnome.  Is there a wireless solution that works with both Gnome and KDE?  I'm using an Atheros wireless card which has native support in Ubuntu, so I didn't have to install any drivers.
