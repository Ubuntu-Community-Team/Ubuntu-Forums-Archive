### Post by drumone on 2023-05-19
After trying numerous code fixes found in various forums, to no avail, I tried a different approach. I installed a number of Bluetooth packages with Synaptic Package Manager. It looks like the Blueman graphical Bluetooth manager seems to have done the trick. I now disconnect and re-connect my headphones and all is back to normal.
