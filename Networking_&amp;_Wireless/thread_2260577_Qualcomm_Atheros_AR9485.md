### Post by jeremy31 on 2015-01-13
For the wifi, try the keyboard combo FN+F? to see if the results of ```
rfkill list
``` changes to soft blocked: no

For the bluetooth, if you want it to work ```
echo "blacklist ath3k" | sudo tee /etc/modprobe.d/ath3k.conf
``````
gksudo gedit /etc/rc.local
``` and add ```
modprobe ath3k
``` above the last line that needs to be ```
exit 0
```, then save, exit program, and reboot.
