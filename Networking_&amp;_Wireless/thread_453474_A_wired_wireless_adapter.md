With all the issues with different wireless adapter cards I thought I would pass along a tip for desktop users that want wireless.

This week CircuitCity has the Buffalo AirStation G54 Router (WHR-G54S) on sale for 24.99(after rebate).  This router is as cheap as any desktop wireless card you will find.

Buy this router for 24.99 and replace the original firmware with DD-WRT firmware and you will have an option for a wireless adapter with 4 ports.  Simply plug one of these 4 ports into you wired network card. No Linux configuration/drivers needed.  Checkout the links below.
