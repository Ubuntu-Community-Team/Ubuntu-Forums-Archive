### Post by Ayuthia on 2008-10-01
> **trmiller said:**
> I installed ubuntu in a dual-boot configuration with Windows. For the wireless to work with ubuntu, I need to install new firmware for the card (broadcom sb43). Will that card still work with Windows after the firmware update?

Yes.  The b43 driver just uses information from the firmware, but does not update anything on the card.  I have two laptops with Linux and Windows that use the b43 driver for Linux and the Windows driver for Windows.  I have never had any problems with either.
