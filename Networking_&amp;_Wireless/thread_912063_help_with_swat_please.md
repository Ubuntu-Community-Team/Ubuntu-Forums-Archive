### Post by bab1 on 2008-09-07
You have to login to SWAT as the Linux root user.  I have configured the login for root.  My root login to SWAT works.

I know that everyone feels that you should not do that.  I agree with that policy for casual or beginning users.  I'm not sure what others do.

Since SWAT is a Web based, no form of CLI sudo will work.  I tried gksudo Firefox, but still was presented with a login -- foiled again!

The SWAT documentation is [[COLOR="Green"]**here**[/COLOR]]("http://http://www.samba.org/samba/docs/man/Samba-HOWTO-Collection/SWAT.html")

I do not use SWAT at all.  I don't see any advantage to using it.  It has one **BIG limitation**.  You can't generate a smb.conf file with comments.  I comment my conf files for reference use later on.  It's just as easy to use a text editor when modifying the smb.conf file.
