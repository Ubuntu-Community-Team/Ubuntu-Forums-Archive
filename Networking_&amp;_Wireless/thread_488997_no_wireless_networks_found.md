Well I don't know AP broadcasting is, but there is good news, I just rebooted back into Windows, and it seems that windows wasn't shut down properly last time I used it, so it resumed and it even didn't connect to my network at first, it took windows quite a while to get working properly, then I closed it down and rebooted into Fiesty and PRESTO! the wireless networking worked straight away! :D  So it seems that it was something to do with Windows and it not closing down last time it was used.
