### Post by MaindotC on 2010-09-12
The driver on the cd should also have a text file on it that tells you how to install it.  Typically you type```
./configure
make
make install
```  Please consult the [Installing Software Official Documentation](https://help.ubuntu.com/community/InstallingSoftware#Source%20or%20Binary?) for future installtions.

With the TP-Link TL-WN321G you should consult the [Wireless Troubleshooting Guide](https://help.ubuntu.com/community/WifiDocs/WirelessTroubleShootingGuide).  That has specific steps for troubleshooting your connectivity issue.  If you need further help, please post precisely where in the troubleshooting process you are having difficulty.
