### Post by MeanRoy on 2012-11-13
I have been trying to bring up NDAS on my 10.04 and couldn't get a response from the server at  [http://ndas4linux.iocellnetworks.com...wToBuildTheDeb](http://ndas4linux.iocellnetworks.com...wToBuildTheDeb).

I put in a call to tech support at iocellnetworks and explained the problem.
The tech explained it will be *some* weeks (months?) before the servers will be back up due to hurricane sandy.

In the meantime it appears the only way to proceed is to compile from source which may be obtained at [https://github.com/iocellnetworks/ndas4linux](https://github.com/iocellnetworks/ndas4linux).

The good news is the [wiki]("https://github.com/iocellnetworks/ndas4linux/wiki") has some directions etc.
```
How to

    export NDAS source for different architectures
    build and install NDAS modules from source
    build, install and remove .deb ndas packages
    build, install and remove .rpm ndas packages
    use ndasadmin

```
