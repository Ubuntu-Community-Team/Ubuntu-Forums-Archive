### Post by shanix on 2008-09-23
Not very much familier with Vista, but might share some thoughts here. I know Vista set up more restriction when you try to share folders on Vista. So, I think it's easier to go another direction, Vista -> Ubuntu.

On Ubuntu, you right click on the folder and share it. Simple as that. It will ask you to install some program called SAMBA, but that's fine.

Remote desktop is easy. You can go to System> Preference> Remote desktop and enable the services. After that, you should be able to RDP to it.
