### Post by conjur3r on 2007-10-05
Right click on the panel you wish to add the icon to, select Add to Panel.  Scroll down until you can see Network Manager.  Add it and then it's just a matter of specifying which network interface you wish to monitor.  If it is a wireless one, a strength bar will display next to it.
