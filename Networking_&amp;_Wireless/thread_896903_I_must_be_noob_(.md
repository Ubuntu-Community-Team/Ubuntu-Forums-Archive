I'm guessing OCPL stands for something like Orange County Public Library - that doesn't really help.

I think your problem is that you need wireless drivers for Ubuntu?  Are you able to connect wirelessly at all when using Ubuntu?  If not, check out threads on ndiswrapper - you will probably need to install it.  Ndiswrapper is located on the liveCD, so you don't need internet access to get it, if you have the CD in the drive.  You will need access to your windows wireless driver - should be able to get it off the windows partition, although there's a question of whether or not you'll know where to look for it.

Anyway, check out one of the threads on ndiswrapper and see if there's any help there.
