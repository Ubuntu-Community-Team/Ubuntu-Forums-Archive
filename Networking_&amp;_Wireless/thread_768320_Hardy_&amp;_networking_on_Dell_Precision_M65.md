### Post by steinarcarlsen on 2008-04-26
I have a clean wubi-installation on a Dell Precision M65. It will not connect to the network with a cable, and wireless networking is very slow. With an earlier alpha version of Hardy on the same computer, and on a Gutsy system I got 12 Mbit/s on my home network; with the Hardy release I get only 600 kbit/s. Also, connecting to the network takes a long time compared to earlier. Even worse, network throughput becomes slower after using the system for a while.
