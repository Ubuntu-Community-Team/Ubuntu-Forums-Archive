### Post by tamagotono on 2006-10-08
If your NIC supports WPA than the easiest way to get it working is to install knetworkmanager.  It is very simple to use, just install it, select it from the menu then click on the icon in your application bar.  you should see your network, select it and it should ask for the WPA passphrase if it is using WPA.

Good luck
