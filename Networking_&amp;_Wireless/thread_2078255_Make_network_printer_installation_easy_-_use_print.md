### Post by Rezliw on 2012-10-30
Installed Ubuntu 12.04  (and also Pupeee and Easey Peasey on an old Asus 2Gb Surf) and they all found my el cheapo D-Link  print server.  There were also drivers for my older HP6122.  Bought the D-Link a while back with rebate and it was practically free.  :)

You may want to set up print server ahead of installation and manually force a static IP from your router.  

Didn't need samba and in fact works better to not Samba, just click add printer, give it a few to find the print server and then follow the pages.

I see a lot of traffic on networking printers and thought I'd put in my $0.02

PS windows 7 found it too but no printer drivers for old HP6122, finally found something that worked
