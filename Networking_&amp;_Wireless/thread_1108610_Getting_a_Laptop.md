### Post by lisati on 2009-03-27
I don't have any experience using the Sony Vaio range of laptops, but suspect that setting up wireless will be fairly straight forward. 

When Ubuntu detects any kind of network, there's usually a network icon on the top panel, at the top right of the screen. Clicking on that will show the networks you can currently connect to. If it's encrypted (e.g. WEP or WPA), all going well, you will be asked for the proper key, which will be remembered for the next time you want to connect to the network.

Other forum users more familiar with Sony's laptops might be able to fill you in with more details if there is any more work to get things connected - on my Toshiba laptops with built-in wireless cards, Ubuntu connected wirelessly as described without any hassle.

Sharing files between my various pieces of hardware took a little more work (about roughly 5-10 minutes on each machine, with a little tinkering to fine tune things as required).
