### Post by alasdair3 on 2015-02-23
OK, so after a lot of hunting around I finally solved this one.

For some reason my "/etc/nsswitch.conf" file appears to have been modified and was missing the "DNS" entry. It read:

```


hosts: files mdns4_minimal [NOTFOUND=return] mdns


```

So I changed it to:

```

hosts: files mdns4_minimal [NOTFOUND=return] dns mdns4


```

And everything worked fine. No idea how that file got modified.
