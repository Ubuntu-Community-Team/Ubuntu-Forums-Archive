I have recently installed Ubuntu in a dual boot configuration with xp. I need to set it up for internet access using a cdma dialup modem which is *not detected automatically under windows. This is the procedure i follow in Windows:

1. Plug in my FTDI chipset-based USB cable. The system will ask for drivers for 'USB-Serial Converter' which i point it to and then it appears under 'Universal Serial Bus controllers' in the Device Manager.
2. After above step, windows automatically installs a 'USB Serial Port(COM X)' under 'Ports(COM & LPT)'.
3. Now I install 'Standard 28800bps Modem' by checking 'Don't detect my modem; I will select it from a list', into COM port X installed earlier.
4. I setup a dialup connection manually with a username and password and '#777' as the dialup number.
