I have been ubuntu (7.10) for a few months now, and have been generally very happy about it. One thing which I do miss from my old windows system was the network status icon. It provided two very valuable indicators :

a) Send/Receive notification on the tray icon. So If you are making a request, the send monitor icon in the tray would become blue, and on receiving the other one becomes blue. This is very useful to know what's going on if you have an network issue.

b) Windows will notify you if for example your gigabit card  drops down to a megabit link due to a bad cable, incompatible switch, or auto negotiation issues. 

None of these are present in ubuntu, and i was wondering if there was a way to get similar functionality or not.

### Post by Fixman on 2008-01-02
On one of the bars, you can click:

right button->add to panel
system monitor

On the new bar the appeared you put
right button->preferences
check network and uncheck processor

There you got it.
