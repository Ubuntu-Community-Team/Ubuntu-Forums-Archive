### Post by Wamoc on 2008-05-01
It generally isn't installed separately, either as part of the driver set or it came with it preloaded. How you can check what you are using is read the description that is given from holding your mouse over an icon on the taskbar. Make note of any that could be a wireless controller program. If you see multiple, I recommend using the one from your wireless card, not the microsoft one. 
Does your wireless use encryption? If so, you might be using the wrong key. Try to get it to reask for the wireless key and reenter it. I am trying to remember how to do this, it has been awhile since I have done wireless with windows (my windows machines are wired desktops). If I remember how to do this I will let you know.
