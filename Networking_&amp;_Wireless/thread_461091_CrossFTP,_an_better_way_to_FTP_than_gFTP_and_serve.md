### Post by morningboat on 2007-06-01
Always looking for a great GUI ftp client in Linux, but the result is quite disappointing. However CrossFTP is different. [CrossFTP]("http://www.gnomefiles.org/app.php/CrossFTP_Client") is a free multi-tabbed FTP client for Linux and more. The latest version is 1.36. It is a good replacement for people who used to work on FlashFXP or CuteFTP.

You can [Download here]("http://www.crossftp.com/crossftp.jnlp") and open it by Java Web Start (javaws)

FEATURES

    * Web Starting
    * Web Updating
    * International CJK Word Support
    * Local File Search
    * Multi-Tab
    * Archive Browsing
    * Extraction
    * Filtering
    * Transfer Compression
    * Web Searching
    * Auto Reconnect
    * Anti-Idle
    * Sound & Event Triggers
    * Much More

The CrossFTP Server, a free secure-enabled GUI FTP server, is also available by [Web Starting]("http://www.crossftp.com/crossftpserver.jnlp"). 

  * System requirement: Java 1.4+
  * Available at: [http://www.gnomefiles.org/app.php/CrossFTP_Client](http://www.gnomefiles.org/app.php/CrossFTP_Client)

This combination solution is definitely one of the easiest solutions for people to share and transfer files among different machines. How do you think about it?
