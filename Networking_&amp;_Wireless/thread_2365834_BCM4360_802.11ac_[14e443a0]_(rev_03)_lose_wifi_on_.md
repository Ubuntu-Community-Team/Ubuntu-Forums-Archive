### Post by BenginM on 2017-07-11
Salutations!

According to the [wireless wiki]("https://wireless.wiki.kernel.org/en/users/drivers/b43") you should use the wl module instead of bcma:

  PCI-ID    Supported?    Chip ID     Modes      PHY version    Alternative
14e4:43a0   no          BCM4360    a/b/g/n/ac        AC?               wl

  To load the wl , you need install the [broadcom-sta-dkms]("https://launchpad.net/ubuntu/xenial/+package/broadcom-sta-dkms") 

```
sudo apt update
sudo apt upgrade
sudo apt dist-upgrade
sudo apt-get install linux-headers-$(uname -r|sed 's,[^-]*-[^-]*-,,') broadcom-sta-dkms

sudo modprobe -r bcma 
sudo modprobe wl 
```
