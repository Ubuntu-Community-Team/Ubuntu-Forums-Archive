### Post by Skidpad on 2007-03-01
I believe I'm in the minority here, because I don't use Network Manager, but rather Wi-Fi Radar and System>Administration>Networking to manage my wireless connections.

I created two separate locations in Networking (just click Save and give the location a name.  I use Home and Work since that is where I primarily use this laptop). 

Once I did that, my laptop connects automatically each time I boot - *but if I've changed locations since the last boot, I must open Networking and then select the new location to get it to connect *.
