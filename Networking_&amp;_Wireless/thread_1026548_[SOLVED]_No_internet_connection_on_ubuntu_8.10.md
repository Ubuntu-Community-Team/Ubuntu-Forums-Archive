### Post by zika on 2009-01-04
let's try with the simplest solution ...

```
sudo nano /etc/network/interfaces
```should look like (assuming You want dhcp. if You want static IP address, comment lines for dhcp and uncomment lines for static providing necessary addresses) :
```
# The loopback network interface
auto lo
iface lo inet loopback

# The primary network interface DHCP
auto eth0
iface eth0 inet dhcp

# The primary network interface static
#auto eth0
#iface eth0 inet static
#    address ***.***.***.***
#    netmask ***.***.***.***
#    gateway ***.***.***.***
#    broadcast ***.***.***.***
``````
sudo nano /etc/resolv.conf
```should look like
```
search google.com
nameserver ***.***.***.***

```once You've entered all this do
```
 sudo /etc/init.d/networking restart
```do not worry if You get red or yellow triangle on Network Manager icon, You can sort that later, it doesn't matter now.
