### Post by timrichardson on 2010-07-31
This is a very, very frustrating problem. 


Check this: 

changing the file /var/lib/NetworkManager/NetworkManager.state to networking enabled may fix the problem.

That is, that text file should look something like:
[main]
NetworkingEnabled=true
WirelessEnabled=true
WWANEnabled=true

then you'll need to do
sudo stop network-manager
sudo start network-manager

(I think's that's the correct syntax, I'm away from my Ubuntu installation at present)
