### Post by jlhughes on 2007-10-25
Under Ubuntu 7.10 with Visual Effects turned on, the vpnc command to exit full-screen -- ctrl-alt-enter-- does not work. 

When you press the key sequence, you see the local desktop for a brief moment and then the remote screen snaps back into full-screen.

If I turn off visual effects, then vpnc full-screen exit works as expected -- dropping the remote screen to a window on the local desktop.
