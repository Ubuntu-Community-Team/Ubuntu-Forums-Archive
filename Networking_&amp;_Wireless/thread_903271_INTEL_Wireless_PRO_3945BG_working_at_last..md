### Post by eddieishavingago on 2008-08-28
After nearly a week of failed attempts I managed to get the wifi working. For all those who are having problems, this is what I did:

I downloaded the an .exe .msi extractor called uniextractor.exe. It is freeware, and can be downloaded from various web pages, I got it from softonic I think. I installed this under windows xp. I used that to extract all the files in the setup/install provided by Samsung, the manfacturer of my laptop. I needed the file NETw3x42.inf. The program extracts all the contents of the setup file into a separate folder, just search for the .inf file.

I then rebooted into ubuntu. I have Ubuntu 8.04 hardy. I downloaded and installed using a wired connection the WINDOWS WIRELESS DRIVERS applet(which I think is a version of ndiswrapper)using the add/remove manager in Ubuntu. Then I ran it, gave it the NETw3x42.inf file, and rebooted. Voilá. It seems to work as I write this now. I hope it lasts too. 

I hope this is helpful. I cannot guarantee that it'll work for everybody.
