### Post by Iowan on 2008-08-12
Welcome aboard!  Yes, you'll need Samba to share between Linux (Ubuntu) and Windows.  (OK, there may be other options...) The client should already be installed.  The Windows machine will need to have something shared, and will probably need to have it's firewall "adjusted".  Past that...
[this]("http://ubuntuforums.org/showthread.php?t=870330") thread:> **superprash2003 said:**
> to access windows shares from ubuntu go to terminal and type nautilus smb://x.x.x.x whre x.x.x.x is the ip of the windows machine...
Also, [here]("https://help.ubuntu.com/community/SettingUpSamba") is a good intro to Samba.
