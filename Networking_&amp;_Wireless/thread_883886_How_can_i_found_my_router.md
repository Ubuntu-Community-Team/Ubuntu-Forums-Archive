### Post by wirespot on 2008-08-08
First choice would be to set your network connection as DHCP and see if it picks up the router. If the router is in a default state it would most likely offer DHCP.

If this doesn't work and you're sure there's a router on your LAN, try using ZeNMap as root to scan around and see what you find.
