### Post by ajgreeny on 2011-08-03
I have just found **giver** and it looks as if it could be useful to send files/folders between two network attached machines, mine and my wife's.

Is there a way to add giver to startup applications, but have it start minimised to the system tray and not as a window, just like skype?  If there is, it seems to be a well hidden configuration option.

---

### Post by ajgreeny on 2011-08-11
OK, folks.

I've found a way that works, for anyone else who is interested.

1.  Install **alltray**
2.  Start giver with command "alltray giver" instead of just "giver"
3.  Done!
