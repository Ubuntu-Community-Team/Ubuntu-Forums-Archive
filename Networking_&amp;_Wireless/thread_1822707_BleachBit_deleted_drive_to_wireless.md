### Post by Andrew.Z on 2011-08-11
[BleachBit](http://bleachbit.sourceforge.net) version 0.8.8 doesn't do anything with your network or wireless.  It does clean apt, but that's the same basically as running 'apt-get clean' or 'apt-get autoremove' and, again, won't cause any network problems.

So I suggest basic network troubleshooting starting with these
[LIST=1]
[*]Does your web browser connect to Google?  
[*]Have you tried rebooting the machine?  
[*]Have you tried rebooting the wireless router and modem?  
[*]Some laptops have a physical switch on the case to disable the wireless: make sure this is not off.
[/LIST]
