After I upgraded from Hardy (8.04) to Intrepid (8.10), I began to experience a weird problem. My wired network connection is working fine on Intrepid but after I shutdown, the network adapter is also turned off (which normally doesn't happen with Windows and previous versions of Ubuntu). The connection lights of the network interface are turned off, and when I boot to Windows or even another version of Ubuntu (Live CD), there is no network connection and the network adapted is still turned-off. The only thing that can "turn-on" the network adapter is booting Intrepid again.

I'm using the following network Adapter:
Attansic Technology Corp. L2 100 Mbit Ethernet Adapter

Note: I experienced this same problem before with Fedora which is why I changed to Ubuntu.
