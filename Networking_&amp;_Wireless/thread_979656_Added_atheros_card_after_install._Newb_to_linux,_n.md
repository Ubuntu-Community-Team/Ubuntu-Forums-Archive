Hello tikibomb. I would at least upgrade to Hardy. It is still supported for at least another year. I'm letting them work out the bugs in Intrepid before I move. If you want to keep 6.10 then go to [http://snapshots.madwifi-project.org/madwifi-hal-0.10.5.6/](http://snapshots.madwifi-project.org/madwifi-hal-0.10.5.6/) and download the driver. Remember most commands start with sudo if you're going to be installing things into your system. Good luck hope this helps.
