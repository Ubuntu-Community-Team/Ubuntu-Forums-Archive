### Post by praseodym on 2011-09-25
Check the mtu-value of your ISP and add it as a number instead of "Automatic" in the network-manager applet. Via terminal (example):

```
sudo ifconfig eth0 mtu 1492
```
What card is that?

```
lspci -nnk | grep -iA2 net
route -n
uname -a
lsmod
cat /etc/resolv.conf
```

### Post by praseodym on 2011-09-25
The driver **forcedeth** often needs additional module parameters:

```
echo "options forcedeth msi=0 msix=0" | sudo tee /etc/modprobe.d/forcedeth.conf
```
Reboot, and check:

```
ifconfig eth0
cat /etc/network/interfaces
cat /etc/resolv.conf
dmesg | grep force
```
