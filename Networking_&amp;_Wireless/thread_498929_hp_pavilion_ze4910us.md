### Post by drhavanger on 2007-07-12
This really isn't a question just wanted to note how easy it was to setup wireless network on my hp pavilion ze4910us.

[LIST]
[*]downloaded the newest version of ubuntu, 7.04, and burned to a disk.
[*]installed with out a problem.  
[*]updated system with out a problem after install was complete.
[*]downloaded and installed fwcutter-006 from System->Administration->Synaptic Package Manager
[*]entered root password
[*]clicked on search and enteredd "fwcutter".
[*]it found bc43xx-fwcutter 
[*]selected it, downloaded it, and installed it.
[*]i could then goto the network icon in the upper right hand of the screen (by date and time)
[*]saw several other wireless networks, choose my network
[*]entered my wep key and then entered my root password
[/LIST]

everything worked like a charm!
this has been far the easiest install for linux i've ever expierenced
