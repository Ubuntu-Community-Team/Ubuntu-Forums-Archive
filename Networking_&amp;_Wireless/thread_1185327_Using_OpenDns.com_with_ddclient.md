### Post by itang sanjana on 2009-06-22
I assume that you have follow the instruction at [https://www.opendns.com/start/device/ubuntu](https://www.opendns.com/start/device/ubuntu).

Now move on to [https://help.ubuntu.com/community/DynamicDNS](https://help.ubuntu.com/community/DynamicDNS), follow the ddclient instructions part.
At the point #3. Modify the /etc/ddclient.conf .. here is mine:
```
# Configuration file for ddclient generated by debconf
#
# /etc/ddclient.conf

daemon=300
syslog=yes
pid=/var/run/ddclient.pid
ssl=yes
use=web, web=checkip.dyndns.com/, web-skip='Current IP Address: '
server=updates.opendns.com
protocol=dyndns2
login=your-opendns-login-name
password='your-opendns-pswd'
Home
```

And then follow the rest of the instructions.
Hope this helps ..
