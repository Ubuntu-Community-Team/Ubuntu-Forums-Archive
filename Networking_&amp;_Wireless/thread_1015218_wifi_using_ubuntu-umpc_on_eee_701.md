### Post by snowpine on 2008-12-18
Hi John, no version of Ubuntu supports wireless on the eee pc "out of the box." There are a couple of different methods to get it working. One (which I've never personally tried) involves downloading the appropriate madwifi drivers. The "hack" I personally use is to install the custom kernel from array.org, which adds support for all of the eee's hardware (wireless, wired, webcam, etc.) and works with Ubuntu/Xubuntu/Kubuntu hardy or intrepid: [http://array.org/ubuntu/](http://array.org/ubuntu/)

However, I absolutely no idea if the array.org kernel will work with umpc. You're on your own there. :)

FYI my main OS on my eee 900ha is an Ubuntu derivative called Crunchbang that uses the Openbox windows manager, plus the array.org kernel. I find it to be a nice fit for the eee: [http://crunchbanglinux.org/forums/topic/11/crunchbang-on-the-eee-pc/](http://crunchbanglinux.org/forums/topic/11/crunchbang-on-the-eee-pc/)
