### Post by PaulM1985 on 2012-10-23
I finally managed to solve my problem.  I opened the file /etc/network/interfaces and removed everything and replaced it with:

```

auto lo
iface lo inet loopback

```

I then restarted networking by typing the following in the terminal:

```

sudo service networking restart

```

Ubuntu managed to find my wired connection and I am now back online.

Paul
