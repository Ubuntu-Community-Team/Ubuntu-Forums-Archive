### Post by dandnsmith on 2011-07-21
The facile answer is YES.
However,what are you connecting to - internet (via router) on each? - if so, then you'll have a problem using it unless you're prepared to solve some routing problems.
If you're using the connections to two separated networks, and have suitably chosen IP addresses and routing, then you should have little trouble.
