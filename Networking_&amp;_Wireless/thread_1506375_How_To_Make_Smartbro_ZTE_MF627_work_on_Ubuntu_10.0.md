### Post by lod2 on 2010-06-10
1. Plug Smart Bro ZTE MF627 usb modem (a message might appear saying GSM network not connected, or something similar)
2. Click the "network connection" icon
3. Select “New Mobile Broadband GSM Connection” and Click "Forward"
4. Select "Philippines" then click "Forward"
5. Select "Smart", click "Forward"
6. Select "my plan is not listed", type smartbro
7. Click "Apply"

In my experience, the modem automatically connected and the LED started blinking (and a "GSM network connected" message appeared).Dont worry if this didnt happen to you. If your modem LED is not blinking, just skip step 11. Anyways, just continue with the following steps:

8. Right click network connections and click "Edit connections"
9. a window will appear, click "mobile broadband" tab, select "smart connection" and click "edit"

10. select "ppp settings" tab and click "configure methods" button, uncheck all
except pap. Click "apply"

11. If the modem is still blinking, left click the network connection icon
and click "disconnect "

12. then click the network icon again and click smart connection. 

At this point, you may now surf the web.

hope this works for you...
