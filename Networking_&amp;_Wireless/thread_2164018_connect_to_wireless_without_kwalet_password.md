### Post by itlarson on 2013-07-20
Is there a way to make KDE connect to WIFI automatically without being insecure?  The two ways I have found both seem to have security problems.  the first way is to leave the KDE wallet password blank, which obviously makes any password in it accessible.  The second way is to go to "network management settings->other->network management secrets" and choose "in a file (unencrypted)", which is what I have done.  The first method makes all KDE wallet passwords readily accessible, the second, only the WIFI logins.  Fortunately, both these methods (I think) require physical access to the machine.  

It seems like KDE needs make "connect automatically" an option which can be chosen the first time you log into a network.  Other desktops do this.  I am thinking of reporting this as a bug, unless someone convinces me it isn't.
