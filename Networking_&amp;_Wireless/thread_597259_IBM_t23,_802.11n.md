There are no native linux drivers for 802.11n cards at the moment. There is some success with cards using airgo and marvell chipsets with ndiswrapper.



Some resources for you.

Cards listed as working by community
[http://ubuntuforums.org/showthread.php?t=370108](http://ubuntuforums.org/showthread.php?t=370108)

General linux wireless device
[http://linux-wless.passys.nl/](http://linux-wless.passys.nl/)

NetworkManager is the default app to handle network connections. Still a buggy application but chipsets that work with it listed here.
[http://live.gnome.org/NetworkManagerHardware](http://live.gnome.org/NetworkManagerHardware)

And a last thought, after you narrowed it down, try to do a search on the forum for troubles that anybody has had.
