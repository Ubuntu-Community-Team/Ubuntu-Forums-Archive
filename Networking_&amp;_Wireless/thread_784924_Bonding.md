### Post by elamericano on 2008-05-07
Page not found?

This was useful for modems, but it's less useful today. Your eth0 link speed certainly exceeds your internet speed. If this is to improve transfer to a local server, then just be aware what you hope to achieve. An 11g connection will get 20-25 Mbps throughput if you're the only client connected (less than half that if you're not), but you can count on the wired connection for 80Mbps. Not worth the hassle for a 30% increase, if you ask me. Keep in mind that all computers you plan to transfer from need to be set up for bonding in the same manner as yours, which is usually impractical or impossible.

Sorry if some of these comments don't apply, but you were really short on details.
