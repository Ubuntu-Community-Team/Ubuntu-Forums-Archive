### Post by p1t0u on 2009-09-15
If they use the same IP subnet that you use at home and the IP address you use on your computer is not in use at the hotel when you connect then you have no issue using your configuration.

For IP to work you need a valid unique IP address, a valid default gateway and valid name server(s).

Most home routers point to themselves for the name servers so as long as the hotel's router acts
as a name server relay you did not have to change anything in your configuration.
