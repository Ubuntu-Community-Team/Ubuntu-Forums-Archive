### Post by xadloki on 2008-06-27
I just installed this app myself to sync between a mac and ubuntu for my bookmarks, I've been able to make the sync to local folders so far, the way it works is you drag an item to the canvas, if it's a file like bookmarks select the file node on the left panel and drag it in. Then you get a gray box around the file node. Next drag a location to where you want to sync the file to let's say a folder from the left panel into the gray area. You can change the settings by right clicking the nodes in the canvas. When you're ready to sync you right click in the gray area and select sync. Now you should have a copy of that file in the location you selected.
