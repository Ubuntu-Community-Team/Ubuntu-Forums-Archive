I have a big problem with my internet connection on Ubuntu 6.06.
I recently switched from XP to Ubuntu.  Everything works way better/faster than XP.  The only problem is that my internet is not working correctly.  I can download files at the same speed as on XP but when I browse web sites (MSN.com for example) they take up to a minute to load.  There is a blank screen for about a minute and then in a split second all the text loads and a little later the graphics.  My internet connections speed is 1.5mb/s and I never had this problem on XP.  My router is a Cayman 3341 Ethernet DSL router.  I have tried to look up/correct this problem but since I'm new to Linux in general, I don't know what to do.  I would greatly appreciate your help.

### Post by Bartisimo on 2006-08-26
YEEEEAAAHHH!!!!:D =D> 
I fixed it! I spent about 5-7 hours on this and i fixed it.  It was especially hard because I was completely new to Linux and though that i had messed something up or didnt have enough ram.  Wow, persistence does pay off!  If  you have a similar problem all you need to do is find the DNS addresses for your ISP and type them in under SYSTEM>ADMINISTRATION>NETWORKING>DNS.  At least that is what did the trick for me.  If this is not the corrrect way of doing things please tell me!
