### Post by Perkins on 2008-04-16
I have an old laptop, and its internal wireless doesn't support WPA.  Which is fine, since I have a pcmcia card which does.

The problem I am having is that if I remove the card after I have started using it, and then reinsert it, I have to reboot my machine to make it work again.  The card is detected upon the second insertion, but it does not function properly and the network manager app does not detect any networks.

Do I somehow need to tell the system that I am going to remove the card so that I can put it back in again later without rebooting?  Or is it supposed to do that automatically and I need to figure out why it's not?

---

### Post by Perkins on 2008-04-16
Never mind, figured it out.  Painfully obvious.  Disable wireless networking, and then reenable it, and it works again.
