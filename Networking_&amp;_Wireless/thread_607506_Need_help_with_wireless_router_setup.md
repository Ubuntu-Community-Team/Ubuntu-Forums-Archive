### Post by jrz on 2007-11-09
If wireless connections are not working at all, be sure it's enabled in the router under the wireless tab. I have the same router and it works fine with 2 wired WinXP connections and a wireless WinXP and Ubuntu notebooks. The WLAN light on the front should be on if the router has wireless enabled, and there are some other options in the router you may need to look at.
If I can be of any help let me know as we had some difficulty getting wireless to work at all under Ubuntu, but finally seem to have it working very well. Thanks to help received on these forums.
