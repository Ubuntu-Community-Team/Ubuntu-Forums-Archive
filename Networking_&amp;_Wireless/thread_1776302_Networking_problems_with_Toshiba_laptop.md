### Post by Atirag on 2011-06-05
Hello everyone, I been having quite a few issues with the network connection on a Toshiba Satellite C655D-SP4131L.

First of all the installer of Ubuntu 11.04 64 bit kept getting stuck but after some tries it worked and I managed to install it. Most things worked from the go but everytime I tried to connect to the wireless network the computer freezes completely. So I tried with the 32 bit version of 11.04 and the same happens.

I figured that maybe it was an Ubuntu problem so a friend tried installing Fedora and it also freezed when trying to connect to the wireless.

I tried OpenSUSE and it gets stuck midaway installation everytime, I don't know if it happens trying to connect to the wireless to dl something but it wouldn't surprise me.

So since the freezing is happening with several distros I figured that it wasn't a distro specific problem but either a kernel issue, a firmware issue or both.

So I went back to Ubuntu but I tried to install Ubuntu 10.10 64 bit. I managed to install it and everything seemed to work fine except it didn't recognize the wireless card. So after searching a bit I found the solution here: [https://answers.launchpad.net/ubuntu/+source/gnome-nettool/+question/132667](https://answers.launchpad.net/ubuntu/+source/gnome-nettool/+question/132667)

After that it recognized the wireless card and I was able to connect BUT...

Sometimes it freezes again. It doesn't freeze all the time like it happened with 11.04 but it happens usually when I go from wired to wireless. Another issue is that sometimes the wired connection disconnects and it refuses to connect again no matter what I do. So I have to restart the computer for it to connect again.

Even with this issues 10.10 has been the one with less problems so I want to stick with it but I would like to see if I can fix these other problems.

I would really appreciate if someone can help me with this since I'm pretty frustrated at this point and I have no idea what it might be. I wanted to update the firmware to see if that was the problem but Toshiba doesn't provide a firmware upgrade for this specific model so I'm a little afraid to try some other firmware and harm the computer.

So anyway the summary of the 2 main issues I want to fix in Ubuntu 10.10:

1. Sometimes when I try to connect to wireless network the computer freezes completely.

2. Sometimes with the wired network the computer disconnects suddenly and refuses to reconnect with both wired or wireless connection. The only way to connect again is to restart the computer.

I have looked for answers in this forum but I only find people that had similar issues with 11.04 or 10.10 but not with both so their solutions don't work for me.

Thanks for any help with this issue.
