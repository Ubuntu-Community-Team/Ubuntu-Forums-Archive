### Post by tormod on 2006-08-29
The support for the prism2_usb driver, necessary for USB wireless adapters like D-Link DWL-122 and Netgear MA111, is not perfect in Dapper. At this point in Edgy (close to Feature Freeze) there is no support at all.

If you are using this driver, please have a look at the bugs:
[https://launchpad.net/distros/ubuntu/+source/linux-wlan-ng/+bugs](https://launchpad.net/distros/ubuntu/+source/linux-wlan-ng/+bugs)

- please confirm the bugs if you can
- add useful information if possible
- show the developers that this support is wanted

Some of the bugs have proposed patches. Please try them out and report the results. This would help to get the developers consider incorporating them, and most importantly, keep supporting these devices in Ubuntu.

Thanks,
Tormod
