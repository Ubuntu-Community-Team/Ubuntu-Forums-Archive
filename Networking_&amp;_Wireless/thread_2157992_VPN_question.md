1. What is a VPN? 

One popular technology to accomplish these goals is a VPN (virtual private network). A VPN is a private network that uses a public network (usually the Internet) to connect remote sites or users together. The VPN uses "virtual" connections routed through the Internet from the business's private network to the remote site or employee. By using a VPN, businesses ensure security -- anyone intercepting the encrypted data can't read it.

2. How do I connect to it? 

That depends entirely on what kind of VPN it is. Which style of VPN is used (Cisco, OpenVPN, Freeswan/OpenSwan, Juniper Dana) will depend on what you use to connect it with. I suggest you find an OpenVPN provider, its the easiest to use and configure, client wise. 

3. Can I connect more that one computer to a VPN? 

Traditionally speaking yes. Some VPN providers however will limit the number of connection you are allowed to connect with. Be dilligent of this fact.
