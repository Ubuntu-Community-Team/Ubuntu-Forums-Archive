### Post by labinnsw on 2010-03-22
All these use the same service. Some of these require wine. 

[Transmission]("http://www.transmissionbt.com/")
[Deluge]("http://deluge-torrent.org/")
[Vuze]("http://www.vuze.com/")
[uTorrent]("http://www.utorrent.com/")
[BitTorrent]("http://www.bittorrent.com/")

You can search Youtube for tutorials relating to your choice.

Just a note. To keep it private, create a torrent file and therefore a seed.
Load the seed and send it to your friend.
You will not need to both use the same application. It is kind of like E-mails in that respect.
