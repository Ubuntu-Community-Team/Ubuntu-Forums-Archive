### Post by AJL on 2007-04-30
Just wanted to post an FYI about this USB wireless access point.  I have been building a MythTV box and had this MA101 usb wireless device. 

I originally did the install with Edgy, but could  never get wireless to function using ndiswrapper.   I could see the ESSID but could never authenticate.

Installed Feisty, did the whole MythTv setup and the Netgear MA101 worked great right from the start.
