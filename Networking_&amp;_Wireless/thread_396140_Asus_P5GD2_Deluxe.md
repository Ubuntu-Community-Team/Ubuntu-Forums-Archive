Isn't this the famous Marvell Yukon? Or, as some here call it, the Marvelous Yu kan't? You can search these forums for Yukon. The short version is the kernel module needed is buggy until kernel version 2.6.20. You can install Feisty and get 2.6.20-13 (easy) or install a Feisty kernel on top of your Edgy installation (less easy).  Feisty is only a short time from official release and works perfectly here, but your computer is different from mine and your mileage may vary. Post back if we can help.
