See this thread for some background info on the different versions:

[http://ubuntuforums.org/showthread.php?t=111746&highlight=belkin+f5d7000](http://ubuntuforums.org/showthread.php?t=111746&highlight=belkin+f5d7000)

Assuming the driver is right, would yours be either the version 1000 or 2000 of the f5d7000? If so this maybe be your problem:

[http://ubuntuforums.org/showthread.php?t=16591&page=2](http://ubuntuforums.org/showthread.php?t=16591&page=2)

See also: (scroll down to the entry for Belkin F5D7000)
