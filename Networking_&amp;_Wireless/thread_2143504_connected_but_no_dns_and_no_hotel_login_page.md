### Post by SeijiSensei on 2013-05-09
I didn't read all the details in the other thread, but have you tried spoofing your User-Agent header?  If you use Firefox, install the [User Agent Switcher]("https://addons.mozilla.org/en-US/firefox/addon/user-agent-switcher/") add-on, then tell the browser to advertise itself as Internet Explorer.  That can sometimes help with poorly designed servers that expect everyone to be running Windows.
