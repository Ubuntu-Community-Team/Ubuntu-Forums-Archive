### Post by tjandracom on 2010-07-11
install samba package in your ubuntu, then u will be able to share your ubuntu folders to windows.
or, install samba-client to your ubuntu, then u will be able to see your windows sharing folder. it is assumed that your two computers have already connected using LAN and you can ping to each other.
