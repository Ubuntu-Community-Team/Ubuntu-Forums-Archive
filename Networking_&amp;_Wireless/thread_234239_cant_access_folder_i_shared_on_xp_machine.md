### Post by DotComFactory on 2006-08-11
With XP, you need to use the administrator account or other account with sufficient privileges to access some subdirectories of the root drive.  The Program Files folder and Windows folder are a couple examples.  

I run into the same thing trying to access my XP machines through Ubuntu 6.06.  In fact, I had the same problems accessing from one XP machine to another.  Disabling simple sharing and specifying the user names authorized for that directory was the answer.

You can also try a service such as LogMeIn ([http://www.logmein.com]("http://www.logmein.com")) that will allow you to have a PC Anywhere type of control over a remote computer.  Their basic service is free and should do the trick for most tasks except for tranferring files.  Just set up an FTP server for that.
