### Post by Kris_M on 2017-10-29
I picked up a TP-LINK TL-WN725N - box says ver 2.2 - no ver info on dongle - plug and reboot 17.10 - log on to my router - perfect.

I intentionally picked the old 725 - even though they update the versions, I hoped it would be plug and play. It was. RTK 8188 per below.

lshw -c network
```
  *-network       description: Wireless interface
       physical id: 2
       bus info: usb@3:2
       logical name: wlxc025e9cd0fc8
       serial: c0:25:e9:cd:0f:c8
       capabilities: ethernet physical wireless
       configuration: broadcast=yes driver=r8188eu ip=192.168.1.211 multicast=yes wireless=IEEE 802.11bgn



```

dmesg shows
```
Chip Version Info: CHIP_8188E_Normal_Chip_TSMC_D_CUT_1T1R_RomVer(0)
```


lsusb shows
```
Bus 003 Device 004: ID 0bda:8179 Realtek Semiconductor Corp. RTL8188EUS 802.11n Wireless Network Adapter
```
