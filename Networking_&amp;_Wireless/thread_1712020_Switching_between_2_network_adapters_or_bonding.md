### Post by sedawk on 2011-03-22
If you are using DSL the speed limit really is the DSL line.
WLAN and wired network are much faster than DSL under
normal circumstances.

I do not see the point of bonding in that situation and
with wireless devices. If all devices talk to the same
WLAN router I would expect speed to drop, not to increase!

If your EEE PC has RJ-45 network port it would be the
best way to connect it to wired network for some
big download. If that still is slow the limit is not
the network.
