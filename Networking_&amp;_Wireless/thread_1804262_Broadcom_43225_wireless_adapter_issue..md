### Post by TBABill on 2011-07-14
Something failed in the setup of the wl module I think. Can you ```
sudo apt-get install --reinstall bcmwl-kernel-source
``` and then go back and reactivate it again? Are you doing this from the LiveCD or from an installed system with wired internet access?
 
Edit: those blacklisted items are correct. That shows you the system set up the system to ignore those items because they can conflict. The b43 series is open source and does not support all Broadcom adapters. It's normal to have them blacklisted when you use the proprietary driver.
