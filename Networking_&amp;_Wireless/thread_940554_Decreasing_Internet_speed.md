### Post by lisati on 2008-10-07
Like other downloads, it's possible that the servers for the updates are busy. My copy of 7.04 has an option where I can automatically choose a "better" (read "quicker") server. What I usually do is start the System->Administration->Select software sources applet, click on the "download from" source, and choose "other" to be able to use the "select best server" button.
