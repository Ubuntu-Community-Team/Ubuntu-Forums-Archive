### Post by sudodus on 2011-11-25
Welcome to the Ubuntu Forums

Ubuntu 8.04 LTS is a good but old version without support. It is important to have at least the security updates. The present long time support version is Ubuntu 10.04 LTS. Because it is newer, it has better support for wireless LAN too.

If that would still not work, I suggest that you try Ubuntu 11.04 or the present version 11.10. This last version is new.

You can also browse the internet for wireless LAN and linux. See for example
[https://help.ubuntu.com/community/WifiDocs/WirelessCardsSupported](https://help.ubuntu.com/community/WifiDocs/WirelessCardsSupported)
