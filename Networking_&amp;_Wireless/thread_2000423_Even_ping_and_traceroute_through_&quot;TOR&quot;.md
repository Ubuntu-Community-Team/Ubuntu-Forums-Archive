### Post by lucb1e on 2013-01-11
The torify command should do this. So "torify ping google.com" for example. If you want to encrypt *all* traffic, you might find the Whonix linux distribution interesting.

Yes, I know the OP might not be looking for the answer anymore, but for future reference for any googlers like me...
