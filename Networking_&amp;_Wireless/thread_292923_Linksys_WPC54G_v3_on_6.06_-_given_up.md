### Post by nickbaker77 on 2006-11-04
Unfortunately I've given up trying to get this wireless card working under 6.06. I did eventually get it working on the previous release of Ubuntu, but I just can't be bothered to sit here **again** for literally hours, trying all the different things from the multiple forum posts in order to get the card working only *most* of the time. This is a pretty disappointing result for me, as I need the wireless working in order to be able to use my laptop at a table without it burning a hole in my knee! So unfortunately no wireless = no Ubuntu.

Here are the reasons for my frustration:

1) Ok so the card manufacturers don't support linux (maybe I should just bin this card and get a different one, especially as its already been returned to  the manufacturer once due to connection problems under windows and when I previously had it working with Ubuntu). This one is not Ubuntu's fault, fair enough.

2) In my opinion the Gnome network settings manager is rubbish. I've never got it to do anything useful as far as I can remember. Usually changing any settings just results in an extremely long wait followed by no perceivable results. No error messages, nothing. In fact under Breezy it seemed to have trashed the interfaces file (it was only after manually fixing this that I got it to work). 

3) The process to install ndiswrapper seems to be different on every forum post I see. Ok so loads of people have posted helpful information, and spent time helping others get their wireless connections set up, but the prevailing theme seems to be something like "run these commands and post the results". In general no explanation of the thinking or summary of what the commands actually do. This seems a bit like "you can give a man a fish, or you can teach him to fish" or however the proverb goes. I'm not expecting a full-on linux lesson here, but allowing people to see your reasoning, and explaining the steps and options would be a lot more useful.

So what am I trying to say here? Well I'm not sure really, except that I think Ubuntu is a brilliant product and philosophy, and I'm disappointed to have run out of patience...

Maybe I'll try again next week.

Nick
