### Post by grahammechanical on 2011-01-23
It is said that we cannot be in two places at once. Yet, that is exactly what we are trying to do if we are downloading and visiting a website at the same time. The poor little machine does not know whether to following the instruction to download the file or the instruction to download the web page. It tries to do both, a little bit at a time. So, there is a slow down as the machine tries to keep more than one connection active.

Download speeds are a measurement of data received for each second. Time spent connecting to a website would interrupt the measurement and so a slower time is shown.

I think that it is a technological marvel that we can do all of this over two copper cables that were originally designed for using a telephone. An operating system has to work with what it has got.
