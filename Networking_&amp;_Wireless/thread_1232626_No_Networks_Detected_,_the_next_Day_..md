### Post by quixote on 2009-08-08
Your wired connection is still working, is that right?

One thing I've done on occasion is turn wireless off by accident either with my laptop's hardware switch or function keys.  If your computer has that option, check it to make sure.  I know it sounds really stupid, but honest, it can happen.

Is your dual boot XP machine the same one as your ubuntu machine?  If yes, it's obviously not a hardware switch.  But then maybe it's some flag XP is leaving set that Ubuntu interprets as "no wireless."  Try enabling or disabling wireless before shutting down XP and see if that makes a difference.

When you say it's "not showing networks" do you still have the network manager icon in the panel, but when you click on it, it says "no networks"?  Or do you mean the whole network manager is gone?  If the latter, right-click on an empty part of the panel, and choose "Add to Panel".  Then select "Notification Area".  The network manager is part of that, for reasons known only to gnome developers.

If it's none of these things, we'll move on to the next stage of toubleshooting.  Fun, huh ? :D
