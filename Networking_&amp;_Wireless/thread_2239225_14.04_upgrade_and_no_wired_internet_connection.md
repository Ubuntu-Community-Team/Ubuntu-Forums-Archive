### Post by varunendra on 2014-08-14
Doesn't look like your router is forwarding the DNS queries properly. Please try explicitly defining one that works in Network Manager in Ubuntu.

In Network Manager settings for your ethernet connection, select "Method" under "IPv4" to be "Auto (DHCP) address only" (if it is currently set to "Auto") to be able to edit the DNS field. If it is set to "Manual", it should be already editable.

In the "DNS servers" field, type -
```
8.8.8.8, 8.8.4.4
```
..or any other DNS server address that you know is working and better than above (the above are DNS servers from google, usually good, but not the best for everyone). Click "Save", then close. Disconnect > reconnect and see if you can browse normally now. If not, please post the report Hadaka asked for (following the link he gave).
