### Post by atryus28 on 2008-10-29
There is a lot more you will need to do in order to get through both his router and yours.  A better and easier route would be to try and use hamachi so you can vpn to his machine and then use remote desktop.  Otherwise you are talking about port forwarding and opening up access on both routers which you may not want to do.  

There is always the possibility of setting up a DDNS for the router so you can always use the same "address".
