### Post by ankspo71 on 2013-09-24
Have you tried having the administrative user (the original user on the notebook) go to Start Menu > System Tools > Users and Groups, Then select a user, then click on Advanced Settings, then click on User Privileges, then place a check by "Connect to wireless and Ethernet Networks". Then repeat for each user needing access to the wireless. I'm only using a single user setup so I hope this helps.
