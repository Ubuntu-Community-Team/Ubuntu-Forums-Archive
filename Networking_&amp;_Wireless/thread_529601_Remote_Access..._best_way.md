### Post by Robbie Pence on 2007-08-19
Well, there is a few ways to do it.

There's a REALLY easy way to do it,


This is the way I do it. I'm running a laptop in my living room,
and am connected through default remote desktop to
Ubuntu in my bedroom.

Go to System>Preferences>Remote Desktop

This is pretty straight-forward, if you are using Evolution for email.

Just set your password, disable "ask me for confirmation"
and click the button with the name of your computer.

Done! :)
