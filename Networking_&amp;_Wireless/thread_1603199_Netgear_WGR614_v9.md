### Post by ajgreeny on 2010-10-22
The router seldom makes any difference to the success of wireless in Ubuntu or any other linux distro; it is usually the wireless adapter in the client laptop or desktop that matters.

Setting up most routers does not need any specific software either as you can normally log into the router with a cable to start with, and do all the setup using that.  Netgear even says you can use linux now on the box my new router came in, and give you instructions for linux and Mac systems exactly the same.

The outcome of all this is that you should not worry; even if they said it does not support linux, it would still probably work OK.
