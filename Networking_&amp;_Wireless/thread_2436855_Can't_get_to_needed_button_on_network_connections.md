### Post by Holger_Gehrke on 2020-02-14
On XUbuntu (and quite a few other Desktop environments) you can move a window by alt-dragging (move the mouse pointer to anywhere inside the window, hold the alt-key on the keyboard, hold the left mouse button, move, release mouse button and alt-key). Using this technique you should be able to reach all parts of a dialog, so no need to use the terminal (although there are of course ways to configure the network that way ...).
