okay so i know there are other ways to do this. 

but the way i figured out  is pretty straight forward. 

it requires an active internet connection already. so if you dont have an ethernet card already plugged in and working this probably wont help you. 

okay what i did was download the automatix program [http://www.getautomatix.com/wiki/index.php?title=Installation](http://www.getautomatix.com/wiki/index.php?title=Installation)

after that you go and find the program called wine. this is the what will allow you to run this windows .exe files. 

okay now that you have your emulater of sorts. you need to download the driver on Netgear.com. or whatever adapter your using. after you find your driver (which will most likely be in the support section). after you download the driver. unpack it to the desktop and open the folder find the setup file. most likely will be .exe. and right click open with. wine. you will probably have to enter the name of the program but its in there somewhere. not exactly sure will. and a very familiar looking will pop-up allowing you to install the driver. after that you should be good go.
