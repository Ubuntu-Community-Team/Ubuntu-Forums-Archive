### Post by gkestrel on 2007-11-24
I had the same problem.  First check to see if you have nautilus share installed. you can find out by searching synaptic for it, or by checking to see if you have a sharing options listed in nautilus when you right click on a folder.  If so nautilus share needs to be setup first to stop the error message from happening.  Full instructions can be found at

[http://gentoo.ovibes.net/nautilus-share/](http://gentoo.ovibes.net/nautilus-share/)

don't forget to run the commands as root or sudo or they will not work.  Hope this helps you out.
