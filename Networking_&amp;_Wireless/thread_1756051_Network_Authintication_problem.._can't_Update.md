### Post by Osberox on 2011-05-13
I had the same problem. 
First I insert username and password to the system network proxy. You can find it for 11.04 at the top right corner - the power switch - then system setting. There is a button "Details" where you can enter username and password.

But this does not help for me!!!
I opened etc/apt/apt.conf with sudo and a editor and changed the setting
from: 
```
Acquire::http::proxy "http://10.40.10.100:8080/";
Acquire::ftp::proxy "ftp://10.40.10.100:8080/";
Acquire::https::proxy "https://10.40.10.100:8080/";
```to:
```
Acquire::http::proxy "http://user:password@10.40.10.100:8080/";
Acquire::ftp::proxy "ftp://user:password@10.40.10.100:8080/";
Acquire::https::proxy "https://user:password@10.40.10.100:8080/";
```Now I can update my system.
