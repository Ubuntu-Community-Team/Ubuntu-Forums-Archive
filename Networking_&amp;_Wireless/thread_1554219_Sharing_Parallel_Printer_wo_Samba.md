I have a parallel printer working on a desktop. I want to share it with my laptop, but I don't want to have to use samba. Both have ubuntu 10.04 installed.

I'm sure it can be done, but I'll be darned if I can find anything in the skimpy admin. tools.

---

### Post by bilkay on 2010-08-16
Ahah!

On the Desktop: System->Administration->Printing

Server->Settings - Checked "Publish shared printers connected to this system

On Laptop: System->Administration->Printing - Click on "Add+" , "Network printer", "Find Network Printer" Voila!
