### Post by wylfing on 2005-06-28
Browsing for Windows shares is a hit-or-miss kind of thing. There are bugs (or at least very, very touchy settings) in Gnome that make SMB browsing work badly. Try connecting directly to the Windows shares instead:

1. Open a nautilus window (e.g., home).
2. Press Ctrl-L.
3. Type smb://machinename/sharename into the box and click Open (where 'machinename' is the Windows computer name and 'sharename' is the name of a shared folder).

You may be asked for a username and password (ignore Domain). Does it work? If not, try the Windows machine's IP address instead of its name. Does that work better?

Also, are you trying to ping their IP addresses or their network names?

Are you absolutely sure both computers are in the same workgroup?
