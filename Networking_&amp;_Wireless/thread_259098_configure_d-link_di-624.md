### Post by neouser99 on 2006-09-17
this probably isn't the place to be seeking this kind of help, but because we are all wonderful people here....

most, if not all consumer wireless routers have a reset button on the back. usually a combination of unplugging the router, then holding down the button while plugging the router back in, and continuing to hold the button for at least 10 seconds, will do the trick. this will set the router back to its default settings at the time you purchased it.

at this point, i believe the dlink routers, the default user is admin and there is no password, but i'm probably wrong here.
