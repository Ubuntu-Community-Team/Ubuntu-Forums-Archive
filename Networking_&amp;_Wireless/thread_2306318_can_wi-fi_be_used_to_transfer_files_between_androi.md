### Post by SeijiSensei on 2015-12-14
Install the [Wifi File Transfer Pro]("https://play.google.com/store/apps/details?id=com.smarterdroid.wififiletransferpro&hl=en") app on your phone.  The free version has a limit of 5 MB per file.  I think I paid $3 for the "pro" version without limits.

If the laptop is running Samba, you can use [AndSMB]("https://play.google.com/store/apps/details?id=lysesoft.andsmb&hl=en") to transfer files.

In general, I prefer to use a USB cable.
