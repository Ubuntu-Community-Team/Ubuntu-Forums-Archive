### Post by praseodym on 2012-01-21
You can deactivate the hardware encryption of the driver via:

```
echo "options ath9k nohwcrypt=1" | sudo tee /etc/modprobe.d/ath9k.conf
sudo modprobe -rfv ath9k
sudo modprobe -v ath9k
```
Check the connection again:
```
dmesg | egrep 'ath|country'
iwconfig
iwlist chan
```Which country do you live?

Otherwise start the NM-applet from terminal if its still greyed out:

```
gksu nm-connection-editor
```

### Post by praseodym on 2012-01-21
Now the power management is activated, try to:

```
sudo iwconfig wlan0 power off
```

Also the country code settings dont seem to be proper:

```
sudo apt-get install iw
iw reg get
sudo iw reg set US [COLOR="Red"]#US for the United States, change it to your country[/COLOR]
```

Remove the current wireless profile and try to set up a new one

### Post by praseodym on 2012-01-21
Did you uninstall the network-manager completely, when using Wicd? Otherwise its not working properly:

> sudo apt-get remove --purge network-manager*
sudo service wicd restart
BTW: Wired connection is always preferred
