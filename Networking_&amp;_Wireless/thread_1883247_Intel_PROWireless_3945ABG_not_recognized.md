### Post by praseodym on 2011-11-18
Does your router send in N-mode, too? This card doesnt work with it. 

The card uses its own country settings:

```
wlan0     IEEE 802.11abg  ESSID:off/any  
          Mode:Managed  Access Point: Not-Associated   Tx-Power=[COLOR="Red"]15 dBm[/COLOR]  
```
You can try:
```
sudo iwconfig wlan0 txpower 20
iwconfig
```
Error messages can be ignored. This is a bug [No. 884605]("https://bugs.launchpad.net/ubuntu/+source/iw/+bug/884605") in kernel 3 at the moment, the **crda** (central regulatory domain agent), which settles the country settings (frequencies, band width, etc), doesnt work properly at the moment. Which country do you live? Try also to set the country settings manually (example for [COLOR="Red"]US[/COLOR]):
```
sudo apt-get install iw
sudo iw reg set [COLOR="Red"]US[/COLOR]
```
Which channel does the router send? 
> [    8.774174] iwl3945 0000:02:00.0: Tunable channels: [COLOR="Red"]11[/COLOR] 802.11[COLOR="Red"]bg[/COLOR], 13 802.11a channels
Tried channel 1? Router fimware is up to date? System updated, too?
