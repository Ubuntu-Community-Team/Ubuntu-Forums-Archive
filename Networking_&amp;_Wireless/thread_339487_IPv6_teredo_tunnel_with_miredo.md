### Post by madblueimp on 2007-01-15
> Teredo is a tunneling protocol designed to grant IPv6 connectivity to nodes that are located behind IPv6-unaware NAT devices. It defines a way of encapsulating IPv6 packets within IPv4 UDP datagrams that can be routed through NAT devices and on the IPv4 internet.
[INDENT][http://en.wikipedia.org/wiki/Teredo_tunneling](http://en.wikipedia.org/wiki/Teredo_tunneling)[/INDENT]

[Miredo](http://www.remlab.net/miredo) is a teredo implementation for Linux.

I wrote a little miredo (teredo) howto, available on

[INDENT][https://blueimp.net/linux/howto/teredo.txt](https://blueimp.net/linux/howto/teredo.txt)[/INDENT]

You need the package "checkinstall":
```
$ sudo apt-get install checkinstall
```


As an alternative, you might as well use the provided attachement package.

Have fun on the IPv6 network! :)
