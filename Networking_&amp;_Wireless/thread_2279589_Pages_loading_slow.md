### Post by Vladlenin5000 on 2015-05-24
You can try the script anyway. It also gives useful info about other connections.
However, I would start by ruling out the obvious like cables, etc.
Yes, it can also be something graphics related but in that case other symptoms are to be expected. Examples: Slowness in opening apps, slow/freezing mouse movements, slow scroll, issues with video, etc. can be a result of a problem with the graphics drivers.
Everything else working fine (even considering the nvidia error message) and only web pages loading slowly is definitely something network related.
