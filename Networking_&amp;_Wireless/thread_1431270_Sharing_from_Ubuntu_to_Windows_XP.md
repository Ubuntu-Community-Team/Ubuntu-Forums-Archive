### Post by TRG1 on 2010-03-16
I ran into this right away the first time I wanted to share to my XP computers. The first thing is to install the Samba sharing, but that was easy. Once that was done my XP computers could see the shared folder, but could not access it. To fix this I simply checked on the allow guest access to the folder I wanted to share. I was then able to see the files and to open them.
