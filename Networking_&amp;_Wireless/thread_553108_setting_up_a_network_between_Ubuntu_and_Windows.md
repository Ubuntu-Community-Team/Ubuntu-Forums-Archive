### Post by dcwilliams on 2007-09-17
If I understand correctly, you want to share your wireless internet connection through your laptop which means you would have to run a cable between the laptop and the desktop.  

If you wanted to use the router, you would have to use it like a hub.  Alternatively you could just buy a crossover cable and plug the cable directly into the two ethernet cards of  your desktop and laptop.

If you can do that, then yes, it can work using Windows internet connection sharing (ICS) or 3rd party routing software such as:
[http://www.nat32.com/](http://www.nat32.com/)
