i recently installed ubuntu studio 9.04 on my laptop and came across with a problem on the network manager --- when i am in a wireless hotspot with several wireless sources and i attempt to change network through network manager, the main menu stops functioning, even the hover effect on all the icons, and then the laptop freezes. the only way to get out of it is a hard reboot (as in power off and then power on).

fortunately, i found a workaround --- i installed ***wicd***. now i can change network without freezing. ;)
