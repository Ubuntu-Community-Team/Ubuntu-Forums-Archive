### Post by meww on 2008-07-06
I formatted my laptop many times before I could figure this out. It actually is a mixture of two different tutorials.
1) First got to [http://ubuntuforums.org/showthread.php?t=766560](http://ubuntuforums.org/showthread.php?t=766560) thanks to Alex Kent
2) Follow all those steps and remember to download the 2f drivers.
3) Now go to terminal and type in wget [http://ftp.us.dell.com/network/R115321.EXE](http://ftp.us.dell.com/network/R115321.EXE)
4) then type in unzip -a R115321.EXE
5) now go to applications and go down to add remove programs and install windows wireless drivers and then go to system administrator windows wireless drivers and you remove previous drivers and then you reboot.
6) now you go to windows wireless drivers and you got to browse and find the drivers folder and then you install the .inf file and install and then reboot the computer start it and you should have it.
REMEMBER every time you restart the laptop it will not automatically connect so you have to right click on the wireless connection and click on existing connection and go from there thats the only set back.
ALSO REMEMBER TO DOWNLOAD YOUR CORRESPONDING DRIVERS FOR YOUR CARD JUST REPLACE R115321.EXE WITH WHAT YOU HAVE AND THAT SHOULD BE IT. POST YOUR FEEDBACK TELL ME HOW IT GOES.
