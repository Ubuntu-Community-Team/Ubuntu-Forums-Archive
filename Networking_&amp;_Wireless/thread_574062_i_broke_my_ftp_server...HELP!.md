### Post by gautada on 2007-10-12
On a client that can see the server use nmap to check if the ftp port (21) is open.
 ```
%>nmap [ip address or domain name]
```

---

### Post by gautada on 2007-10-12
You can also use System > Administration > Networking Tools.  Click the "Port Scan" tab. Enter the external IP/Domain Name.
