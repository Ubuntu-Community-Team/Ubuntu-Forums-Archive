### Post by wildmanne39 on 2012-06-16
Hi, first make sure you have your wired connection unplugged or it will override the wireless, and you probably should switch ethernet drivers because you are using the wrong one and it makes your wired connection slow but let's deal with the wireless since that is the title of the thread.

Please do:
```
echo "blacklist iwlwifi" | sudo tee -a /etc/modprobe.d/blacklist.conf
```
Then:
```
gksudo gedit /etc/modprobe.d/rtl8192ce.conf
```

A new empty file will open, paste this one line into the file.

```
options rtl8192ce swenc=1
```

Proofread, save and close gedit, then reboot.

There maybe be some other settings that we need to change but let's start here.

Please post the output of:
```
nm-tool
```
Thanks
