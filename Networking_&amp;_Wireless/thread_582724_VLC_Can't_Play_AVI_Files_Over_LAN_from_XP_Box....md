### Post by VladoVNS on 2008-01-12
I have the same problem. I use wireless connection to share my XP box drives and i store the movies there. I found that if you paste the link to the XP shared file you want to play in  VLC player you should be able to play the movie.

I use these steps.
1. Open the shared directory and copy the file you want to play (to get the network path)
2. Open VLC player then choose File->Open File, choose Network tab, choose HTTP/HTTPS... radio button and paste the link that you copy in the URL box.

that should be able to help you to play over wireless network.
