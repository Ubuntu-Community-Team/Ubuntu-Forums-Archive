### Post by pseudonym on 2007-06-25
I've never used Vista but ICS in XP works fine with Ubuntu (as far as networking goes, the OS on any given machine is generally not that important). The easiest way to do it is to have IP addresses assigned automatically via DHCP, which is the default setting in both the XP Network Wizard and the Ubuntu network configuration utility. It's something that will more than likely 'just work' (hopefully :) ).
