### Post by Mike_IronFist on 2009-10-06
Hey guys, I want to remote view my friend's desktop via VNC, but apparently the Remote Desktop Preferences thing only allows local connections. How do I (and/or my friend, for he's going to be the server in this case) make it so that the Remote Desktop thing allows connections beyond local ones?

---

### Post by SoftwareExplorer on 2009-10-07
If he is behind NAT, or a router (which will probably use nat) then he needs to go to the router's config page and forward port 5900. [http://portforward.com/](http://portforward.com/) is a place that does a somewhat good job of telling you how to forward ports for all the different routers.
