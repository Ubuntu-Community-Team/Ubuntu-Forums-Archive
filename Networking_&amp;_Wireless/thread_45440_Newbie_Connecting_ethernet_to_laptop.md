### Post by kamstrup on 2005-06-30
Is the connection activated? Check in System->Admin->Networking and select your net card. Make sure that the "Active" button is grayed out. If not, click it. You should probably also make sure that it is set to DHCP in the properties section...

Good luck

PS: If it is already activated, you might try deactivating it and activating it again... Just a thought... :-)
