### Post by moates1989 on 2010-08-01
You probably don't have your drivers installed. I have the same laptop. Every time I install any distro, I have to install the drivers. You need a ethernet line. After you plug it in, go under system, preferences, and find hardware drivers. It will automatically find the driver if theres more than one I chose the Broadcom STA wireless driver.
