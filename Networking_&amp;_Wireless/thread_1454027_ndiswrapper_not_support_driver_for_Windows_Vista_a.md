### Post by arubislander on 2010-04-14
You are assuming that in the open source world thing remain as they are... while it is often so that necessity mediates change, i.e. as long as there are XP drivers that work with ndiswrapper there will be no need to support the Win7 ones. If and when these drivers become unavailable support will be added to newer ones.

What can we do in the meantime? If you have coding skill you could join the effort and bring forward, or speed up support for the newer drivers, if not, then we do what we did before ndiswrapper: either make sure we buy only supported hardware, or agonize in the meantime...
