### Post by RobGoss on 2016-06-27
Hello and welcome, I think what you're wanting to do is create a **Persistent USB** installation [https://wiki.ubuntu.com/LiveUsbPendrivePersistent](https://wiki.ubuntu.com/LiveUsbPendrivePersistent) this will allow you to save changes and even data right on the USB drive. I would recommend using a USB drive with enough storage space preferably a **32 or 64** and even higher

Can you connect using a wired connection after loading the live session?

You can also dual boot Windows and Ubuntu, if interested here's a great starting point [https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)
