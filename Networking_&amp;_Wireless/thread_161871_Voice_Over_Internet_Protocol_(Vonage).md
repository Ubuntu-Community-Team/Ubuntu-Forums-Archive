### Post by opus on 2006-04-24
I have been using Vonage for a while, and I love it.  I've had the same problems with service when downloading though.  If you have a better router that has traffic shaping/QoS you can configure the Vonage ports (5060 & 5061) to have dedicated bandwidth.  It will essentially make your BT downloads go slower if you are using your phone.

You can also do this with [IPCop]("http://www.ipcop.org"), [pfSense]("http://www.pfsense.org/"), and [Endian]("http://www.efw.it/") firewalls.  IPCop requires the Dan's Guardian plugin (easiest way is by using [http://firewalladdons.sourceforge.net/]("http://firewalladdons.sourceforge.net/")).
