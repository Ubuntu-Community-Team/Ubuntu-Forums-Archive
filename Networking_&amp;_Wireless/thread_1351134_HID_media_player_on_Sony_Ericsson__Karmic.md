### Post by puppywhacker on 2009-12-10
Basically what you see in the menu is a HID profile which can be seen as a very limited keyboard. It sends key-commands to the applications. It is not OS specific, but ofcourse you need the correct key-combination. So you need a package specific to your favorite linux application

[http://www.kde-apps.org/content/show.php/Sony+Ericsson+HID+profiles+package?content=60288](http://www.kde-apps.org/content/show.php/Sony+Ericsson+HID+profiles+package?content=60288)
