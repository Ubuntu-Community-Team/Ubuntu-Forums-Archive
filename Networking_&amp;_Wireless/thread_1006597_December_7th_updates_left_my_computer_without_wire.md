From what I have been experienced so far in Ubuntu, the two main things that generally affect the wireless card is either network manager or else a kernel update.  The easiest thing to test first is the kernel because you can usually fall back to a previous kernel without making any changes.  If that does not work, the next thing to try is to turn off network manager and connect manually.  If you are able to connect manually, then you know the issue is with network manager.

In this case, it looks like it might be the kernel that cause the problem.  Can you post the results of:
```
lspci -nn
lshw -C network
```so that we can see what wireless card you have?  The first command will list out your PCI cards to identify your wireless card.  The second command will show the wireless card along with the driver that is being used.  From there we can try and identify which package is causing the problem.
