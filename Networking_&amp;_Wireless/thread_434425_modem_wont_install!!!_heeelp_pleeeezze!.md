### Post by helmut_hed on 2007-05-06
Hi Charjon,

When you say your "modem" does not work, are you talking about a regular 56K dialup modem, or an ADSL modem?  I think it's probably the latter.  In that case, you actually have a problem with your network connection.  When the installation refers to a "modem", it normally means a dialup modem.  So if you don't have one, or aren't using it, that's not a problem.  Instead, see if "Ethernet" is working for you (probably the term the install will use).

Good luck,
