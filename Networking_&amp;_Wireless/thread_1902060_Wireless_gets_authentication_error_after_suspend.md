### Post by praseodym on 2011-12-31
Hi,

check all modules which should be unloaded and reloaded during SUSPEND/HIBERENATE:

```
lsmod | grep -iE '802|ndiswr|forced|8139|hci|usb|1394|hid|..tv' 
```
and add the driver **iwlagn** of your wireless card to the list. After that write all these modules (left column!) into a file:

```
gksu gedit /etc/pm/config.d/00sleep_module
```
Like this (example):
```
SUSPEND_MODULES="$SUSPEND_MODULES ehci-hcd uhci-hcd usbcore forcedeth" 
```
save, close, and set the executable flag:
```
sudo chmod +x /etc/pm/config.d/00sleep_module
```
Check S/H
