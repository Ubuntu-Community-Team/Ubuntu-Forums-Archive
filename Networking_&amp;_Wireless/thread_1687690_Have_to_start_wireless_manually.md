### Post by thefasterblueone on 2011-02-14
You are probably using NetworkManager so here is a link to a guide for NetworkManager.

[https://help.ubuntu.com/community/NetworkManager0.7]("https://help.ubuntu.com/community/NetworkManager0.7")

Follow the guide carefully and ensure everything is setup correctly.

In the guide look for "Creating/Modifying a User connection" there you will see how to enable it to automatically connect.

Also I suggest using "WICD" as an alternative to Network Manager. It is easier to use and works better in my opinion.

[https://help.ubuntu.com/community/WICD]("https://help.ubuntu.com/community/WICD")
