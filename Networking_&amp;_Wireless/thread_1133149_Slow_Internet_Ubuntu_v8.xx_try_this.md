Hi,

I recently installed Ubuntu v8.10 on my laptop and I always seemed to have a problem browsing the internet was very slow internet, Downloading was fine and the same websites were fine in XP too.

I tried the disable IP version 6 link here

[http://ubuntuforums.org/showthread.php?t=87798]("http://ubuntuforums.org/showthread.php?t=87798")

That didn't resolve it.

I tried the firefox disable IPv6 that didnt work

about:config in address bar

The next step I tried was a basic ping

  ping [www.google.com](www.google.com)

PING [www.l.google.com](www.l.google.com) (209.85.227.99) 56(84) bytes of data.
64 bytes from wy-in-f99.google.com (209.85.227.99): icmp_seq=1 ttl=247 time=33.6 ms
64 bytes from wy-in-f99.google.com (209.85.227.99): icmp_seq=2 ttl=247 time=38.3 ms



this was slow to resolve once I got the ip address I tried that

ping 209.85.227.99 (this can change)

The response was instant. Hmm ok.

check your /etc/resolv.conf

and make sure this has your primary DNS server first

I had 3 in mine the top two were random servers the 3rd was my router ip

192.168.0.1


> 

 cat /etc/resolv.conf
# Dynamic resolv.conf(5) file for glibc resolver(3) generated by resolvconf(8)
#     DO NOT EDIT THIS FILE BY HAND -- YOUR CHANGES WILL BE OVERWRITTEN

nameserver 10.1.1.1
nameserver 192.10.10.1
nameserver 192.168.0.1



I removed the top two from the file and tried again this seems to of resolved it

> 
 
sudo gedit /etc/resolv.conf




I think this is set in the network manager but it was also hardcoded in one ofmy interfaces here

/etc/network/interfaces

Find out what file and removed the odd ones from there and that seems to removed it permanently.

No more slow internet. 

Hope that helps
