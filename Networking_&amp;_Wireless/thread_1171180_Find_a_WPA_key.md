### Post by spd106 on 2009-05-27
That is the hex value of your key. It will be accepted by any decent network configuration program just as it is. 
However because it was generated using a hash algorithm (with your ssid) you won't be able to get the original back without some serious hacking tools.

For an explanation see the following
[http://www.xs4all.nl/~rjoris/wpapsk.html](http://www.xs4all.nl/~rjoris/wpapsk.html)
