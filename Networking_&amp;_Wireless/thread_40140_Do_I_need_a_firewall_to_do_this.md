### Post by byen on 2005-06-07
well.... i have firestarter(a firewall). Its pretty decent and It has an option where you can see the Active connection list.

How to install firestarter?  
sudo apt-get install firestarter

and after install you can find it here: Applications -> System Tools -> Firestarter

Goodluck!

PS- if you do not want a firewall... im not sure how to get around looking for active connections...but with a firewall...the above mentioned is the way to go!!
