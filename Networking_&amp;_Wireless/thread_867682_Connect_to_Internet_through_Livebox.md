I have seen several questions on the web concerning difficulties in connecting to a Livebox using Ubuntu etc.

By far the easiest way is to use the USB connection.  No password is required and Ubuntu recognises the connection almost immediately.

Not as convenient as WiFi but much quicker and just as efficient as Ethernet.

I hope this is helpful to someone
