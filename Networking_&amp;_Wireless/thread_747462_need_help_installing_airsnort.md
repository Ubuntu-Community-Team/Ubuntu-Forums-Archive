### Post by spd106 on 2008-04-06
Airsnort is in the Universe repository, so you can find it via Synaptic.

It's a rather old utility, so you may have problems getting it working with newer hardware. Times have moved on and I don't even think Airsnort is being maintained any more.

FYI the main focus is now on Aircrack-ng or perhaps Kismet if you just want a simple wifi scanner with logging capabilities. There are already a few guides around this forum if you don't mind digging a little.
