### Post by cmittle on 2009-07-13
I had to add some stuff to my /etc/network/interfaces:
```
auto lo
iface lo inet loopback

iface wlan0 inet static
	address 192.168.1.129
	netmask 255.255.255.0
	broadcast 192.168.1.255
	gateway 192.168.1.1	
	wireless-key 30F2E4A8977F71E56113CFC0E6
	wireless-essid tomcanshrap

auto wlan0
```
It previously looked like this:
```
auto lo
iface lo inet loopback

```

I also added to my /etc/resolv.conf
```
# Generated by NetworkManager
nameserver	208.67.222.222
nameserver	208.67.220.220
```

Formorly only this:
