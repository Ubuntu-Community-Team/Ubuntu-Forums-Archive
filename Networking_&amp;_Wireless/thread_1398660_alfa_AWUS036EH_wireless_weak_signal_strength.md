### Post by almufadado on 2010-02-04
Read the manual to see how to access the router (probably an IP (web page) -> user and password) 

Check if the broadcast signal is on, the linking protocol is the same (G or N) in both router and adapter.

Do not place the router near to the monitor nor to the computer. They can absorb/interfere  with the signal.

Don't place the adapter and the antenna too close to each other.
