Just wanted to let anyone interested that i figured how to make this work.

after installing the beta drivers from the website on the windows partition. 

Then i copied the files in the driver folder from the installtion cd that came with the USB adapter into the folder of the beta installation and install the .inf file with ndiswrapper.


So far the card has been very very stable and is looking like a great buy for me.
