### Post by ffaker on 2013-06-03
I’ve got a rather strange problem on my hands here, hoping someone can figure it out, because I’ve tried everything I can think of. Internet connection is working, but slowed right down to between 200 Kbits/sec and 20 Kbits/sec (less than dial-up!).

I’m staying overseas with a friend who has broadband connection via satellite, but his wifi router has broken (so we thought we’d try mine so we could share internet while we’re here). It works via an ethernet cable that comes down from the roof, to a small box (he says it’s a filter) out of which comes another ethernet cable. Not sure exactly how it works, but the short version is: when you plug this second ethernet cable into a laptop, it connects to the internet (via DHCP). The connection is fast: almost 2 Mbits/sec both down and up.

Now: when I set up my Billion BiPac 7800N router (which I brought with me from the UK) and set the WAN interface to the EWAN port (Billion's name for uplink, I guess: the second ethernet cable is plugged into this), it does connect to the internet -- but the download speed slows right down, sometimes it’s barely usable. The problem is just as bad when connecting to the Billion either via ethernet or via wifi.

Reasons I’m pretty sure it’s a problem with the Billion: a) the cable plugged straight into a laptop works just fine; b) another friend brought over her T-Link router that she uses at home, which also has an uplink port, and it all worked fine with that, the speed was right up there. At one point, I remembered I might need a crossover cable and maybe that’s why it’s slowing it down, but then I read the Billion has MDI-X, which if I understand it correctly means it should work just as well with the straight through cable (also my second friend’s router worked fine with the same cable).

I’ve tried everything I can think of in the settings, also did a factory reset and a firmware upgrade to 1.06g -- all to no avail. (though I notice today there is a 1.06h firmware, so I might try that, if I can get it to download). 

If anyone has any clue what on earth could be causing this, please HELP! This is driving me crazy, and I need the connection for work while I’m over here!

One last weird point: the upload speeds are usually (but not always) unaffected: they are often just as high as the cable alone (up to 2 Mbits/sec).
