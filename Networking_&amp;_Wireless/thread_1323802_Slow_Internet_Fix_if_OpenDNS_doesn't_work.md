### Post by squidmaster on 2009-11-12
WORKED on 9.10

OK so after (way too much) time going with some terrible internet speeds, I was able to come across a fix. Though I wasn't expecting this to fix the speeds for me, it happened to do just that.

[LIST]
[*]Alt+F1 (or main apps menu)

[*]System -> Administration -> Users and Groups (User Settings)

[*]There will be a "click to unlock" icon at the bottom of the new window
[IMG]http://i24.photobucket.com/albums/c31/kakor/Screenshot-UsersSettings.png[/IMG]

[*]Unlock it.

[*]Click on your user profile and then click "properties"

[*]I RECOMMEND enabling everything under the use privileges, just to make some things easier, because 9.10 has a more annoying verification system than Windows Vista.
[IMG]http://i24.photobucket.com/albums/c31/kakor/Screenshot-AccountsquidProperties.png[/IMG]

[*]Go to Advanced and change Main User Group to Root
[IMG]http://i24.photobucket.com/albums/c31/kakor/Screenshot-AccountsquidProperties-1.png[/IMG]

[*]Close it out and go back to the User Settings window.

[*]Click Manage Groups

[*]Manage Root group and make sure your account has been checked off in the box.
[IMG]http://i24.photobucket.com/albums/c31/kakor/Screenshot-17.png[/IMG]

[*]Close out everything and see if it works! It may require a restart, but who knows.
[/LIST]

If it works for you, please post up and let me know! If not, then tell me too!
