What type of problem are you having?  You should just be able to plug the ethernet cable from you broadband modem into your Ubuntu computer and it will detect an IP address (that is if your ISP is providing a dynamic IP).  If you have to use a static IP then you can click on "System" > "Administration" > "Network" and set everything there.
