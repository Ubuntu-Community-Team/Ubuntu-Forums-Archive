### Post by mrsteveman1 on 2008-02-10
While i don't know offhand which specific models you should look for, you may want to be careful choosing one since the actual chipset used in the devices changes a lot.

For instance, linksys cards have used many different chipsets even within the same model. I have a belkin USB stick that was a realtek in V3, zydas in V4, and now V5 is back to realtek, the model number never changed.

Now, as for which CHIPSETS are good in linux, there are good references available, the wikipedia page [here]("http://en.wikipedia.org/wiki/Comparison_of_open_source_wireless_drivers#Linux_2") seems a bit old, but the linuxwireless site [here]("http://linuxwireless.org/") has some good information available on newer drivers.
