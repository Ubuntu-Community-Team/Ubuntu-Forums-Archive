### Post by jimmccool on 2008-01-23
I installed 7.10 [dual-boot] on my year-old Acer 5101 as peformance on Windows XP MCE had degraded to the point where watching videos had been become an impossible jerky mess. Updated ATI drivers, upgraded RAM to an excessive 4GB and spent months checking for spyware, bad drivers, etc. No change, except for the worse... Presumed the hard drive was faulty - and just thought I'd check with a Ubuntu install whether it really was a hardware or software problem.

However, just installing Ubuntu was the solution. Now my laptop springs into rapid life and plays every sort of video and DVD perfectly. However, when I try to use the proprietary ATI drivers - then video playback gets choppy again. So, I don't simply use them - and everything runs fine. Beautiful. Best OS I've ever used - congratulations to the Ubuntu team.

Everything worked more or less, straight from the install [one problem with the bootup screen - solution elsewhere on this forum] including the Atheros wifi card. Fast and reliable network connectivity.

One small problem though - I'm using WPA encryption and everytime I boot up, I have to go into Network settings, remove the long encoded password field - and replace it with my own password. Obviously, password encryption is getting it wrong somewhere. No big deal, but I would prefer not to have to replace the WPA password string every bootup.  Anybody got any solutions? And should I report this officially as a bug?

And again, can I say thanks and congratulations to the Ubuntu team. This is the Linux system I've always wanted. Usability is excellent and everything just works. What's more, my laptop actually works better now than it did with Windows - it runs cooler, plays videos without stalling or crashing, boots up and loads applications much, much, faster.

Thanks!
