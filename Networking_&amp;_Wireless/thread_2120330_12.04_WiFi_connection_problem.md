### Post by grahammechanical on 2013-02-26
There could be several things happening here. In a terminal type

```
rfkill list
```

If the printout is this

> 0: phy0: Wireless LAN
	Soft blocked: no
	Hard blocked: no

Then we have to look elsewhere. If it is this

> 0: phy0: Wireless LAN
	Soft blocked: no
	Hard blocked: yes

Then the wireless adapter has been physically turned off by some key combination. if the printout says this

> 0: phy0: Wireless LAN
	Soft blocked: yes
	Hard blocked: no

Then you do not have wireless enabled.

Regards.
