### Post by johngreth on 2010-04-07
If Amy has six tenths of the file and Bob has seven tenths of the file and Cameron has eight tenths of the file, then the availability is the sum of those parts, so .6 + .7 + .8 = 2.1. This may have an effect on the download speed, but it isn't really a good indication of how available the file is because there may still be two tenths of the file that no one has.
