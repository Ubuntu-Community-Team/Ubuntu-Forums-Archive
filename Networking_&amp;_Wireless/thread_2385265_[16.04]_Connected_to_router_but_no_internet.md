### Post by wildmanne39 on 2018-02-18
Let's turn off power management:
```
sudo sed -i 's/wifi.powersave = 3/wifi.powersave = 2/' /etc/NetworkManager/conf.d/default-wifi-powersave-on.conf
```
We need to set your location on the router, it is set for a generic U.S. country code and the file shows you are in Europe/Paris, this very well keep you from connecting so if you are in Europe/Paris please do.
```
sudo iw reg set EU
```
```
sudo sed -i 's/^REG.*=$/&EU/' /etc/default/crda
```
If there is a better country code to use if you are in Paris then EU for Europe them look it up and use it instead.

For 2.4Ghz, if you aren't in an apartment with 10 other APs, it is best to use channel 1, 6, 11. All others overlap and reduce performance and add interference for 3 channels above and below.  Unless you are in Japan. Japan legally allows a channel above 11 that doesn't interfere with any other 2.4Ghz channels.
