### Post by lisati on 2010-06-10
[SSID]("http://en.wikipedia.org/wiki/Service_set_(802.11_network)#Service_Set_identifier_.28SSID.29") ("Service Set Identifier") is the name of your wireless network, and is usually set in your router. If broadcast of the SSID is switched on, you should be able to see it with Ubuntu by clicking on your network manager icon. On 10.04, the icon looks like a pair of arrows, and in older versions it looks like a pair of computer screens.
