### Post by netztier on 2007-12-17
> **wirelessmonkey said:**
> 1) Library ;) [http://www.pantz.org/os/ios/ioscommands.shtml](http://www.pantz.org/os/ios/ioscommands.shtml)


That link refers to Cisco IOS. To a certain level of certainty, a Catalyst 4000 will run Catalyst OS (CatOS), which is fundamentally different in terms of command syntax and config file handling.

To the OP:
Not the worst of places to start is: [http://www.cisco.com](http://www.cisco.com), then enter "Catalyst 4000 Configuration Guide" in the search box, top right of the page.

Pick the search result that best matches the CatOS version you're running on that 4000. In most cases, you can download the configuration Guide as a single PDF on the subsequent page - no login or registration required.

Or use this link directly to get to the [overview of Cat4000 Family Configuration Guides]("http://www.cisco.com/en/US/products/hw/switches/ps663/products_installation_and_configuration_guides_list.html").
