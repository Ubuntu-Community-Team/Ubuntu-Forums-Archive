### Post by Rogeriototh on 2009-11-22
Hi, this is my first post here. I was searching for an answer on how to configure my 3G modem on Ubuntu Studio 9.10 , and I saw your message. 

I've configured my wireless connection by going to the Ubuntu Studio Menu > *System* > *Administration* > *Network*

You will have to enable and configure the *Wireless Network* in a similar fashion as you were doing with Ubuntu.

Hope it helps.

Rogério

### Post by Rogeriototh on 2009-11-22
One additional information. While trying to find a solution to my 3G configuration issue, I've found that in order to have the same network manager on Studio that we are used to see on Ubuntu, you just need to install the *Network Manager* from Add/Remove Applications.

This has fixed the the issue I was having to configure my 3G modem.
