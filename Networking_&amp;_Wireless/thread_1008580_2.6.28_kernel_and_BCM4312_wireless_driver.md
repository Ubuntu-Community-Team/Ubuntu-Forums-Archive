You don't really need to compile a new kernel to use the new module.  You need to go to [Broadcom site]("http://www.broadcom.com/support/802.11/linux_sta.php") and compile the module.

The wl module is a proprietary module made by Broadcom so it will never be included in the kernel.  Ubuntu bundles the module in linux-restricted-modules so it looks like it comes with the kernel.
