### Post by fdrake on 2011-09-07
the eth0 shows un-managed because you previously changed the interface file.

see post # [29]("http://ubuntuforums.org/showpost.php?p=11213704&postcount=29") of this thread: 

turn these lines from :
```

auto eth0
iface eth0 inet static
address 121.165.98.219
gateway 121.165.98.1
netmask 255.255.255.0
network 192.168.3.0
broadcast 192.168.3.255

```

to 

```

#auto eth0
#iface eth0 inet static
#address 121.165.98.219
#gateway 121.165.98.1
#netmask 255.255.255.0
#network 192.168.3.0
#broadcast 192.168.3.255
[COLOR="Red"]iface eth0 inet dhcp
auto eth0[/COLOR]

```

make sure you have (iface eth0 inet dhcp , auto eth0) showing somewhere.

then;
```

sudo /etc/init.d/networking restart

```
