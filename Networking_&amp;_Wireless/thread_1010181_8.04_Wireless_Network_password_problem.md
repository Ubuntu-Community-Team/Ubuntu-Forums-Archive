### Post by Infinite Indefinite on 2012-07-05
I had a very similar problem, and I found an answer here:

[HOWTO: Wireless Security - WPA1, WPA2, LEAP, etc.]("http://ubuntuforums.org/showthread.php?t=318539")

The lines in my **/etc/network/interfaces** file were in the wrong order, (see point 3 in the first post of the above link).  Putting them in the correct order solved the problem.
