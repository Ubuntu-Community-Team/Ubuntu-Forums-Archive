### Post by Morbius1 on 2010-05-02
I know nothing about how to set up a shared folder on a Mac but to create one on Ubuntu:

Open Nautilus ( your Home Folder )
Right click on say .... /home/your_user_name/Public
Select "Sharing Options"
Select all three boxes
Select "Create Share"

You should see the "Public" share in Finder.

Note: When you first try to use Right Click-Sharing options you will be prompted to download samba because for some odd reason Ubuntu does not install samba by default.
