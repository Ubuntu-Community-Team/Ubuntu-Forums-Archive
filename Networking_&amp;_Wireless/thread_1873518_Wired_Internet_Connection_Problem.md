### Post by praseodym on 2011-11-02
If you are using the NetworkManager (see /etc/resolv.conf) remove _anything but_

```
auto lo
iface lo inet loopback
```
from the "interfaces", remove your profile in the NM applet and reboot.

### Post by praseodym on 2011-11-02
Yes, right-click->Edit connections, choose the wired (and DSL) connection(s) if any, and remove via

```
gksu gedit /etc/network/interfaces
```

anything but the two "lo" lines, save, close, and reboot.
