### Post by WritersBlock on 2008-12-28
ok - back in ubuntu - from SYSTEM > ADMINISTRATION > Synaptic Package Manager.
Oncce this opens select NETWORKING > select the NDISGTK package >  and Mark for installation. 

THis will install the wrapper (you'll probably need an internet connection for this to work so connect your wire). 

Once this is installed you should now have an additional menu option in the SYSTEM > ADMINISTRATION called > Windows wireless drivers. When you select this you will be asked to find (good luck) and install your driver - which must have an INF extentension. 

That is as far as I got!
