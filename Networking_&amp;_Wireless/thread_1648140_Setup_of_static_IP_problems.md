### Post by chili555 on 2010-12-18
I suspect that either Network Manager or Wicd is interferring with your efforts. Please remove them with Synaptic.

Next, your *interfaces* file looks very busy. I'd change it to:```
# The loopback network interface
auto lo
iface lo inet loopback

# The primary network interface
auto eth0
iface eth0 inet static
address 192.168.1.16
netmask 255.255.255.0
gateway 192.168.1.1
```Assuming your router has valid, working DNS servers in its configuration, it's safe to change /etc/resolv.conf to:```
nameserver 192.168.1.1
```Then restart networking:```
/etc/init.d/networking restart
```Note any errors or warnings. Check to see if you have the correct IP address:```
ifconfig eth0
```Check to see if you are correctly connected:```
ping -c3 192.168.1.1
ping -c3 www.google.com
```
