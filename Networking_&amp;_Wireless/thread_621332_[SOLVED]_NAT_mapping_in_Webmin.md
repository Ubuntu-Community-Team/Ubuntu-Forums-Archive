### Post by Zer0d on 2007-12-25
I searched on Google for a while and found a working command don't exactly remember the web page but think I did something like this:

```
iptables -A PREROUTING -t nat -p tcp --dport 1001 -i eth0 -j DNAT --to-destination 192.168.0.100:1001
```

This should send all incoming port 1001 connections to internal IP-Address 192.168.0.100.
