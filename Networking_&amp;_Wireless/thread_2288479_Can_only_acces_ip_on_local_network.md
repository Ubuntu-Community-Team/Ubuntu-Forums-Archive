### Post by SeijiSensei on 2015-07-28
Yes.  Can you ping 192.168.1.1, but not 8.8.8.8?  If so, the problem is with the router.  Try turning off the power, waiting 15 seconds, and turning it on again.  If that doesn't fix the problem, you need to contact the ISP.
