### Post by adam_kimber on 2008-07-15
The "taskbar" is the name that has been adopted by many users for the bar that runs along the bottom of the screen in operating systems like Windows and desktop environments like Gnome and KDE. Ubuntu uses Gnome and thus this bar can also be referred to as the Gnome panel,which is the official name for it (in Gnome). So unless you are using Kubuntu you will be using Gnome. 

For the vanishing icon, right click next the the date on a blank bit of bar and go to add to panel. You will then be presented with a box. Have a look for Network Manager. Click add to panel and the network tools are back. 

Roaming mode can be found by clicking on this icon.
