### Post by xmastree on 2005-12-02
If you're using DHCP, it should work right away. If not, you'll need to manually enter your computer's IP address, and the router's address (for the gateway).

Either way, it should be simple. Assuming you're using windows at the moment, take a look at the tcp/ip settings. If it's set to automatic, you're set. If it's manual, make a note of the settings and enter them when prompted in ubuntu setup.
