### Post by kingfu on 2013-04-13
I had to reinstall ubuntu 64bit on my Acer 5750 laptop and now my Broadcom BCM43227 wireless drivers are not working. First time I installed Ubuntu I had the same issue and had to google a lot to find a solution, sadly I haven't been able to find solution as last time.

I've tried a lot of things I've seen in other threads regarding this series of wifi card but so far the only way I can get it to work in anyway is to go to Software Sources -> Additional Drivers and select "Do not use this device" then after 10 mins reselect "Using Broadcom 802.11 Linux STA wireless driver source from bcmwl-kernal-source(propiertary)", once done my wifi springs to life and the hardware led turns on. Obviously though this isn't going to be something I can live with long term.

Any ideas?
