### Post by jpl888 on 2010-03-10
Your question doesn't make a lot of sense.

usbserial is a standard module and is included with every version of Ubuntu for I don't know how long.

It is a generic module for connecting HSDPA modems, amongst other things. Generally speaking most HSDPA modems now are supported without using this module. For instance my Huawei E220 and Novatel X950D both use the "option" driver.
