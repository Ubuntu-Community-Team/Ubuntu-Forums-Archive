### Post by flash63 on 2009-05-22
Your Router use mixed Encryption WPA1/2 ...
```
Group Cipher : TKIP
Pairwise Ciphers (2) : CCMP TKIP
Authentication Suites (1) : PSK
IE: WPA Version 1
Group Cipher : TKIP
Pairwise Ciphers (2) : CCMP TKIP
Authentication Suites (1) : PSK 
```
... but both types are recognized with the same properties. That could not be right. Change to pure WPA1 (or WPA2) encryption and try again.
