### Post by tfrue on 2014-01-09
If someone has a router with WiFi that is broadcasting, then their SSID will show. The people who have their SSID's hidden, will not show up using the default wireless network manager in Ubuntu.

If you have a wireless NIC in your computer, and have an appropriate drive installed, then you should be able to connect to any wireless network, the only compatibility errors I've ever had with wireless in Linux is incorrect drivers.

Now I will say that if you learn BackTrack5, or the new Kali Linux, you can see all the networks hidden or not using the aircrack-ng suite. Anytime you connect to a network, you will recieve an IP address, so you will not be anonymous to the router, but you can do tor routing or other masquerading and be anonymous online. So if you are asking can you connect to a wireless network and not be seen, then no, but you can monitor all wireless traffic in a proximity and see what's going on.

Be careful though because the aircrack-ng suite is supposed to be used as a pen testing software, not hacking. You've been warned!

Hope I answered your question!
