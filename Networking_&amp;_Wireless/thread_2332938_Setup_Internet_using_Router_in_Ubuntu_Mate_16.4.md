### Post by ajgreeny on 2016-08-05
You normally configure the router by going to its IP address, probably shown in the manual for the router or even on its label, but probably something like 192.168.0.1, using a web-browser.

That usually shows a configuration page with a help panel on one side.

Lots of info at [http://setuprouter.com/router/tp-link/tl-wr740n/login.htm](http://setuprouter.com/router/tp-link/tl-wr740n/login.htm)
