cd mt7601/src

sudo make install
sudo mkdir -p /etc/Wireless/RT2870STA/

thanksAwesome! Glad it's working. The only issue is that you have compiled the driver against the current kernel only; in this case 3.13.0-39-generic. When Update Manager installs a later linux-image, after the requested reboot, recompile:```
cd mt7601/src
make clean
make
sudo make install
sudo modprobe mt7601Usta
```Now I realize that this is probably a bit above the ability of a charity group that know nothing about such matters. Therefor, I suggest you set Software and Updates to check for updates Never. Then, from time to time, make a friendly visit and ask how the computer is running. Sit down and check and install any updates:```
sudo apt-get update && sudo apt-get -y dist-upgrade
```Reboot, re-compile and you're all set.

Please retain the files and these instructions for that time.

Please use thread tools at the top to mark Solved. The MT7601 searchers will appreciate it!
