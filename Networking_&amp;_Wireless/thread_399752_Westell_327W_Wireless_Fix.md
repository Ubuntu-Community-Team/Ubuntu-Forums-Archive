### Post by Luigi Boccabella on 2007-04-02
To everyone unable to make a wireless connection to your Westell 327W router, here's what solved the problem for me:

1) Downloaded the latest firmware for the 327W here:

[http://download.verizon.net/webdownl...R_03_02_01.exe](http://download.verizon.net/webdownl...R_03_02_01.exe)

2) Removed an IPSEC ALG entry that was in the routers custom-defined services list.

I'm not sure if both of these were necessary but I can now connect running the Feisty Beta. If you have a different version of the 327W you might find the proper firmware here, but proceed with caution:

[http://www.fastaccess.drivers.bellsouth.net/#westell](http://www.fastaccess.drivers.bellsouth.net/#westell)

### Post by tuesday20102001 on 2008-04-19
5 minute solution

Model: Westell 327W D90-327W15-06
firmware version: VER:4.04.03.00

I started experiencing problems lately with this router the more I used it. Connection would drop completely and lock me out for several minutes at a time. Turning the power off and on, 15 seconds to a minute did not resolve the issue. Wireless feature is turned off in the router. Problem must exist in the router itself. Suspected solution: firmware upgrade. I downloaded my firmware package from Verizon located here: [http://netservices.verizon.net/portal/link/help/item?linkflag=guestsso_dsl_west_verizon_consumer&case=VersaLinkFix](http://netservices.verizon.net/portal/link/help/item?linkflag=guestsso_dsl_west_verizon_consumer&case=VersaLinkFix) 
.Assuming you already have the the correct package for your router (found underneath the router). Login to the router, click on the advanced tab, then click firmware upgrade. Select upgrade now button found at bottom of page. Browse to the place where you downloaded the file, then click upload. The rest should complete itself, Be sure not to browse the web, as noted by the router while upgrading (I only had my router page open while doing this upgrade).

As for wireless feature on the router, I have not been able to get this feature to work correctly with gusty. I was not really concerned with this because I feel it is just another thing I would be calling verizon about every time The router goes on the fritz. Strangely enough, this has been a ongoing issue for several hundred users for the last few years, as noted here: (solution page: [http://www.dslreports.com/forum/remark,13747078~start=80](http://www.dslreports.com/forum/remark,13747078~start=80)). Perhaps this is a complaint for the public service commission to lean on verizon to fix these problems before sending out the router to customers.
