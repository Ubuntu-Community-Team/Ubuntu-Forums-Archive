### Post by coolglobal on 2009-09-05
Solved: Select Optus 3G, in Preferences > Network Connections, enter the APN as dodolns1, no passwords required, set to Automatic (PPP), save & exit.

I suspect the microphone will work after installing the latest stable version of ALSA. Will try this and update this thread accordingly.
