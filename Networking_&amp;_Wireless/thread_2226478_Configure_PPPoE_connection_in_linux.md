### Post by Sufian_Ahmed on 2014-05-27
Now I am going to discuss how to configure PPPoE connection in linux in a very simple way.
**Process to configure PPPoE connection in linux :**

First of all, make sure that you have connect the internet cable into the port of Enhernet card.
Now,
Run the Terminal simply by pressing Ctrl+Alt+T  Keys of keyboard.
Terminal will appear.Type the following command :

sudo pppoeconf

Press Enter Key then input the password of your ubuntu/linux pc and press Enter Key again. 
 


[COLOR=#ff0000]SCANNING DEVICE[/COLOR] dialog box will appear.
[IMG]http://1.bp.blogspot.com/-8judmfv2XqM/U4S0iLjMQHI/AAAAAAAAADA/SMwZCUxghO0/s1600/pppoe+linux+(2).jpg[/IMG]

then
[COLOR=#ff0000]
[/COLOR][COLOR=#ff0000]OKAY TO MODIFY[/COLOR] dialog box will appear.
[IMG]http://3.bp.blogspot.com/-NiOm_Rkajrk/U4S1Dg_Oo1I/AAAAAAAAADI/Ax1R3Y-OX_g/s1600/pppoe+linux+(3).jpg[/IMG]



Press  the Tab or Down Arrow Key from the keyboard and select <Yes> then press Enter Key.



[COLOR=#ff0000]POPULAR OPTIONS[/COLOR] dialog box will appear.

 select <Yes> and then press Enter Key  as previously described.



[COLOR=#ff0000]ENTER USERNAME[/COLOR] dialog box will appear .

It will ask to enter a username. Type the username which provided by your ISP
select <ok> and then press Enter Key.



[COLOR=#ff0000]ENTER PASSWORD[/COLOR] dialog box will appear .

It will ask to enter a password. Type the password which provided by your ISP
select <ok> and then press Enter Key.



[COLOR=#ff0000]USE PEER DNS[/COLOR] dialog box will appear .

 select <Yes> and then press Enter Key.

[IMG]http://1.bp.blogspot.com/-ZaTNFL2QC6E/U4S32ONsBZI/AAAAAAAAADw/cucCRoCEYvc/s1600/pppoe+linux+(8).jpg[/IMG]

[COLOR=#ff0000]LIMITED MSS PROBLEM[/COLOR] dialog box will appear.

select <Yes> and then press Enter Key.

[IMG]http://4.bp.blogspot.com/-UgebO-2TcqY/U4S4XV9UdtI/AAAAAAAAAD4/WU2czlyEyMo/s1600/pppoe+linux+(9).jpg[/IMG]

[COLOR=#ff0000]DONE[/COLOR] dialog box will appear.

select <Yes> and then press Enter Key.

[IMG]http://1.bp.blogspot.com/-mTZCQ274Vdw/U4S40x4lXZI/AAAAAAAAAEE/y8JCi6uHCkg/s1600/pppoe+linux+(10).jpg[/IMG]

[COLOR=#ff0000]ESTABLISH A CONNECTION[/COLOR] dialog box will appear.

select <Yes> and then press Enter Key.

[IMG]http://4.bp.blogspot.com/-JXDGHDXztsc/U4S5RpCTNPI/AAAAAAAAAEM/BZn0xPn7Lnw/s1600/pppoe+linux+(11).jpg[/IMG]

[COLOR=#ff0000]CONNECTION INITIATED[/COLOR] dialog box will appear.

select <ok> and then press Enter Key.

[IMG]http://1.bp.blogspot.com/-BXwnhKA_Wxg/U4S5q-LWnLI/AAAAAAAAAEU/QWSafMc24e8/s1600/pppoe+linux+(12).jpg[/IMG]
