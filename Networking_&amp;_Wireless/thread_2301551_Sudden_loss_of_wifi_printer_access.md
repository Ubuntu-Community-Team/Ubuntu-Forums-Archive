### Post by rkstarr on 2015-11-03
Had wifi access to home Canon MP560 printer from my 14.04 LTS laptop until a couple of weeks ago.  After an update it suddenly could not print to the printer by Wifi.  I uninstalled/deleted the printer and tried to re-install but the printer could not be found by the print manager.  I can still print to it using a USB connection and my daughter can still print to it using wifi from a Mac laptop.  Any suggestions as to what is going on?

### Post by nknwn on 2015-11-04
Find the printer's IP in your router control panel. Use that to find the Printer and to install the driver.
For example my printer is @ 192.168.1.5

[ATTACH=CONFIG]265347[/ATTACH]

### Post by rkstarr on 2015-11-07
Thanks for that.  I had tried that and it did not work - still could not find it.  But the problem has now been solved anyway!  I installed the latest updates this morning for 14.04 LTS and the printer control panel could now automatically find and identify the printer on the local network again.  It installed the driver (a cups one?) but that did not work but I then manually selected another driver I had for it and it now works fine - again - hooray.
