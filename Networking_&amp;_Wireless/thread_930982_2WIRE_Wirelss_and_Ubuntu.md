### Post by doug-M on 2008-09-27
I was having trouble with hardy connecting to one access point until I realized I was given the HEX version of the key.
The default dialog box in Network Manager on 8.04 (hardy) was asking for as passphrase as in text not HEX.
After I changed the security drop down from 128 bit passphrase to 64/128 bit Hex and re-entered the key it worked.

Probably not your issue but worth a shot.
