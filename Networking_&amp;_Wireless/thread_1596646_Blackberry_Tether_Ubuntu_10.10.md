### Post by night_sky83 on 2011-01-24
Just registered to post a follow-up for the thread. Yes, using the Network Manager applet you can indeed tether your BlackBerry. The set-up has been tested using BlackBerry Curve 3G 9300 running OS 6.0.0.380 on Mobilicity network (a Canadian HSPA provider). Make sure you know what settings you need to use, though, as many Telus Mobility customers have been burned using an incorrect APN. You can peruse through iBluever for settings.
