### Post by BkkBonanza on 2010-10-04
It should work with settings in system proxy but you have to also tell your program (Firefox?) to use the proxy. So you have to select "Use System Proxy" in the Network settings in Firefox. Similarly for other apps. Not all apps can work with a proxy but the main ones do.

Also, anonymous proxies are notoriously unreliable and come and go constantly - it's always possible that the one you tried wasn't even working any more.
