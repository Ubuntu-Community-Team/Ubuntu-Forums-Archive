### Post by Tony221268 on 2010-08-08
Using a curve 8900, uk vodaphone and a 10.4 netbook samsung nc10 with usb tether. 

Having tried XmBlackberry (the link here) and Berry4All, the latter seemed much more straightforward to me. Less terminal (note you can do the Python installs using Synaptic Package Manager if you hate terminal), more GUI and a fairly straightforward app to play with. The guide is ugly (seemed daunting) but worked easy.

Just make sure your package manager includes the universe (default seems not to, so until that was spotted, getting some Python libraries failed).
