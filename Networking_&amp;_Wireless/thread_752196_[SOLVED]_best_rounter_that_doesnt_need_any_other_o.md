### Post by Eiríkr on 2008-04-11
Pretty much all routers that I'm aware of come with some pre-installed embedded OS.  In fact, the only time you ever want to do anything with the router's OS is if the firmware is ancient or buggy and the manufacturer has released an update, or if you're an inveterate tinkerer and just want to (perhaps using the [DD-WRT]("http://www.dd-wrt.com/") or [OpenWRT]("http://openwrt.org/") Linux-based OSes).  

So basically, *every* router you could buy at regular retail doesn't need any OS installation.  

Does that help?  Or have I misunderstood what you're asking about?

### Post by hyperair on 2008-04-11
Just get a router that uses an Ethernet port. USB routers tend to work only with Windows, though there are some exceptions which work in Linux too. 

Anyway as a personal recommendation from me, the Aztech DSL600EW, or Solwise SAR600EW (they're the same product under different manufacturer brands). For more information head over to [http://www.routertech.org/](http://www.routertech.org/). They've got a forum there, and many users which have much knowledge about routers.
