### Post by servicetech on 2010-04-10
If there a setting that will tell ubuntu 10.04 to automatically login to MY network (instead of the neighbors) wirelessly when the computer is booted?

---

### Post by chili555 on 2010-04-10
Sure. Right-click the Network Manager icon, select Edit Connections, select the Wireless tab, highlight your network, Auto Whoever, and select Edit. Make sure the Connect Automagically box is checked.
