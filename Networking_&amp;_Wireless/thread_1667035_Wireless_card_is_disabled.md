### Post by grahammechanical on 2011-01-14
Yes it does. It is called network manager. There should be an icon in the top panel and on the right. If you are using Ubuntu 10.10 then right click and select Enable Networking. Click the line once to disable networking and click it a second time to activate network manager to scan for wireless networks and connect to one if you have set one up to connect automatically.
