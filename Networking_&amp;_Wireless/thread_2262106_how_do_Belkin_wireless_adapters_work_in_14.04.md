### Post by Unterseeboot_234 on 2015-01-23
I have no problems with the Belkin N300 Wireless USB other than I have to use the computer's front panel USB connector and such set-up has been that way since Ubuntu 11.10. Currently using Ubuntu 14.04 LTS on two boxes -- one box is hard-wired to a NetGear Router and the other box is the Belkin wireless USB-dongle. The wired node connects instantly after boot. The box with the USB dongle has a 1-second scan to find the router and then connects. For some reason if I use the rear USB connections for the dongle I have issues staying connected to the router.
