### Post by cleverusername on 2010-03-22
Hello all,

I've been using Ubuntu 9.04 with GNOME as the (only) desktop environment installed since the summer and have had little or no problems with the Broadcom wireless card since I fixed the problems I had at first. My problem is that the wireless card, which after a reinstall I have gotten working perfectly in GNOME will not work under KDE (installed via Synaptic). I can turn on the laptop, log into GNOME, use my wireless card, log out and log back into a KDE session and it does not seem to work - it's like GNOME was before I got the drivers working at all.

I'm not sure if it's of any relevance, but I have tried Linux Mint 7 KDE Edition which is based on Kubuntu 9.10 and Fedora 12 to no avail, which leads me to believe that this is KDE-related more than anything else.

Any help would be greatly appreciated.

Thanks,
Scott.

EDIT: I fixed it myself, apologies for posting. All I had to do was install knetworkmanager in GNOME, then start a KDE session and start knetworkmanager via a terminal. Sorry if I caused any inconvenience!
