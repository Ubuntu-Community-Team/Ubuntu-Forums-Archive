### Post by tarverator on 2009-09-03
Failed finding end of line(timeout) for:
Error: [Errno 11] Resource temporarily unavailable

...occurs when there there is an unclosed quotation.  I made this error when I modified the <carrier>-chat script.

I edited this stanza of att-chat:

'' 'BBT_OS'
'' 'ATZ'
SAY 'Setting APN\n'
OK 'AT+CGDCONT=1,"IP","wap.cingular"'

and made this mistake converting it to fido-chat:

OK 'AT+CGDCONT=1,"IP","internet.fido.ca"
# Wrong!

the correct line is:

OK 'AT+CGDCONT=1,"IP","internet.fido.ca"'
# Close the single quote at the end of the line!!


Here are the complete fido and fido-chat scripts which work with my Blackberry 8220 flip (unlocked from Rogers) on Ubuntu Jaunty:


# fido
115200
noipdefault
defaultroute
#nomultilink
ipcp-restart 7
ipcp-accept-local
ipcp-accept-remote
lcp-echo-interval 0
lcp-echo-failure 999
nopcomp
noaccomp
pap-timeout 20
pap-restart 20
lcp-restart 10
novj
#noauth
nomagic
noccp
crtscts
usepeerdns
novj
user "fido"
password "fido"
#debug debug debug
# does not exist in all pppd versions (osx)
#replacedefaultroute

connect "/usr/sbin/chat -f conf/fido-chat"


============================


# fido-chat
TIMEOUT 10
ABORT 'BUSY'
ABORT 'NO ANSWER'
ABORT 'ERROR'
SAY 'Starting GPRS connect script\n'

'' 'BBT_OS'
'' 'ATZ'
SAY 'Setting APN\n'
OK 'AT+CGDCONT=1,"IP","internet.fido.ca"'

ABORT 'NO CARRIER'
SAY 'Dialing...\n'
OK 'ATD*99#'
CONNECT
# Without ~p it does NOT continue passed Connect !!
~p
