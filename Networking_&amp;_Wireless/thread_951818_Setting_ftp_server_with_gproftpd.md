### Post by cariboo on 2008-10-18
For any service, a static ip address is better, as with dhcp the ip address changes occasionaly, then you have to find the new ip address.

To stop from needing a password with your windows shares, make sure, Allow network users to change my file is checked in the sharing dialogue.
