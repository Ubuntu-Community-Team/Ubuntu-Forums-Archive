### Post by wkulecz on 2009-04-16
When researching which draft-N wireless interface to use for 8.04, I gave up and bought a Linksys WGA600N "wireless gaming adaptor".

You program the network security settings using a web interface, much like you would set up an Access Point and then reset the box and use Ubuntu's wired interface to get wireless access through the WGA.  Slick, and if you have a windows box you get full manufacturer's support setting it up on windows, after which you can move it to your Ubuntu box and expect it to "just work".

Only about $20 more than the USB or PCI alternatives which seemed to offer a lot more headaches.  I'm using it with a Linksys draft-N router and am definitely getting better than G speeds, it also worked fine with my Airlink G router before I got the draft-N router update.

This lets you use the much more solid wired Ethernet drivers on the Linux side and leave the wireless headaches to the WGA's firmware.

--wally.
