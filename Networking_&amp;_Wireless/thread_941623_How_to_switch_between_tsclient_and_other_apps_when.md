### Post by dcviana on 2008-10-08
I'm using Terminal Services Client (tsclient) and Gnome-RDP to access a Windows 2000 machine in work.

When using a Windows machine to access another one and opening the remote session in fullscreen, I can position the mouse on the top of the screen and a yellow bar appears. I can use this yellow bar to minimize the session and use my local programs.

In tsclient and Gnome-RDP I haven't found a way to do that. I have to use a windowed session or logoff  if I want to switch between remote session and a local program.

Anyone know how I can do that in tsclient? Or another RDP client that can do this?

Thanks.

---

### Post by dcviana on 2008-10-08
I ended solving my own problem. You can press Ctrl + Alt + Enter to switch between fullscreen and windowed display.
