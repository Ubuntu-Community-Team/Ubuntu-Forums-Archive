### Post by MrHorus on 2006-07-14
> **oliwally said:**
> 
So you're saying it's the ethernet card in my computer that's important? Am I understanding correctly?

Sort of.

A router just routes packets between one network (your LAN) and another (the Internet). So long as you have a working Ethernet card (wireless or physical) and a working external connection such as cable or ADSL, then any old router will work.

Of course, if you pay more then you will get better features, maybe a bit more range on the wireless, perhaps a pretty HTML interface to configure it.

I have a Netgear 54g Wireless ADSL router and I am perfectly happy with it. To configure it I just plugged my laptop into one of the Ethernet ports on it and brought up the HTML interface in my browser and after I had set it all up it was plain sailing.
