### Post by rob356 on 2009-04-19
Go to System -> Preferences -> Remote Desktop. Then check "Allow other users to view your desktop" and then you should be able to configure it the way you want from that. 
On windows you have to download a VNC viewer, such as RealVNC, TightTNC, there are alot, you only need the viewer though.
You should be all set!
