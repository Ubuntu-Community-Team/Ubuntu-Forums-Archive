### Post by jeremy31 on 2014-06-22
Ok, If you are in the Netherlands the country code is NL, if you are somewhere else look up the 2 digit country code from here [http://en.wikipedia.org/wiki/ISO_3166-1#Current_codes](http://en.wikipedia.org/wiki/ISO_3166-1#Current_codes)

First ```
sudo iw reg set NL
``` and ```
sudo gedit /etc/default/crda
``` and add NL so the last line is REGDOMAIN=NL
Save, exit gedit and reboot
