### Post by smashin on 2011-04-29
Hello,

I've just working this drivers in N mode really fine. They just broken in the upgrade.

My USB dongle is ALFA AWUS051NH

These are the steps:

**1** - Download **RT2870USB(RT2870/RT2770)** from: [http://www.ralinktech.com/support.php?s=2](http://www.ralinktech.com/support.php?s=2) 
It's version 2.4.0.1 (07/09/2010)

**2** - extract the package and edit /path/extracted/..**/os/linux/config.mk**
and set:
```

# Support Wpa_Supplicant
HAS_WPA_SUPPLICANT=y

# Support Native WpaSupplicant for Network Maganger
HAS_NATIVE_WPA_SUPPLICANT_SUPPORT=y

```

**3** - Because new kernel has this wireless drivers in other path, edit /path/extracted/../**Makefile**

Look for that:

```
LINUX_SRC_MODULE = /lib/modules/$(shell uname -r)/kernel/drivers/net/wireless
```
and change to
```
LINUX_SRC_MODULE = /lib/modules/$(shell uname -r)/kernel/drivers/staging/rt2870
```

**4** - make

**5** - make install

**6** - edit the file **/etc/modprobe.d/blacklist.conf**
and add

```
blacklist rt2800usb
blacklist rt2x00lib
blacklist rt2x00usb
```

**7** - Restart the computer.

**8** - OPTIONAL: Config file in **/etc/Wireless/RT2870STA/RT2870STA.dat**
