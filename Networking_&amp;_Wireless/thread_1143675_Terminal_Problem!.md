This was not a solution, but a workaround that helped me taking back my notebook keyboard control.

### Post by Kevbert on 2009-04-30
It may be that your terminal package is corrupted. Try going to Synaptic Package Manager and search for gnome-terminal. Select both gnome-terminal and gnome-terminal-data and in each case right-click and select 'Mark for Re-installation' and then 'Apply'. Please post back any errors.
