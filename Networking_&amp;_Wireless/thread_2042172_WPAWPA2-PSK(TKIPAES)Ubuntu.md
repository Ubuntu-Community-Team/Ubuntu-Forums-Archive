### Post by miho on 2012-08-14
I was wondering if other users were having a hard time using WPA/WPA2-PSK(TKIP/AES)on their modems in conjunction with Ubuntu 12.04 LTS. The reason I ask this is because I was suffering from some problems which were actually solved when I switched to WPA-PSK(AES). I was not able to successfully download or install anything from command prompt or the Software Center, and connecting to websites was hit or miss at best. In fact, what caught my eye was that X-Chat was working fantastically, although I did have problems downloading it to begin with.

Anyways, so, if anyone else has this issue and they're using WPA/WPA2-PSK(TKIP/AES)with their modem, switch to WEP or WPA-PSK(AES) and see if that solves the issue for you.
