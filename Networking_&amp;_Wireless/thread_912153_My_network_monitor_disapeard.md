### Post by spd106 on 2008-09-06
nm-applet (Network Manager) sits in the Notification Area, so you need to add this to the panel. Also open System -> Pref -> Sessions and check that Network Manager is ticked in the Startup Programs tab.

You should also open System -> Admin -> Network and check that the network interface is set to roaming mode.
