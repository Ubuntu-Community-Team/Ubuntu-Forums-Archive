### Post by donpaolo on 2007-08-11
You have to set the IP of every computer as same family IPs:

Put 10.0.0.1 to the one which is going to be the router to the internet, 10.0.0.2 and 10.0.0.3 on the others.

In the network configurator of 10.0.0.2 and 3 put gateway 10.0.0.1
