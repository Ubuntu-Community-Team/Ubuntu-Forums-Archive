Do you need the firewall with Ubuntu? Try:

```
sudo dhclient eth0
ifconfig eth0
```

Router-firmware is up-to-date? Checked the BIOS for those cards?

Checkbox all user rights in Users&Groups, remove your wired profile and reboot.
