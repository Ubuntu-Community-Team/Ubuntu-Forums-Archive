### Post by raja on 2007-05-04
Using the internet from both computers is pretty straightforward. You use a router and both the computers connect to it. If it is a wireless router, the laptop can connect with the wireless card. For networking, you can set up shared folders in windows and be able to see them from Feisty with Samba which can access window shares. 
Hope that is useful for starters.
