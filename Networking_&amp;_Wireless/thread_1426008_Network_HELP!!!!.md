### Post by gmjs on 2010-03-09
Your ifconfig output should display the IPv4 address you have assigned it in /etc/network/interfaces.

Can you check the contents of /etc/resolv.conf?  For your setup, it should contain the following:

```
nameserver 192.168.1.1
```

Then, also check the /etc/hosts file.  This needs a localhost entry and the static entry for your machine

e.g.

```
127.0.0.1        localhost
192.168.1.100    *<your hostname>*
```


Finally, have you restarted the network service?

```
sudo service networking restart
```

or

```
sudo /etc/init.d/networking restart
``````

### Post by gmjs on 2010-03-09
I'd keep both the **auto lo** and **auto eth0** in, yes.

The nameserver line should be the IP address of your DNS server (probably the router/gateway in this case) rather than your machine (unless you've installed DNS).
[B]
/etc/network/interfaces[/B]

```
auto lo
iface lo inet loopback

auto eth0
iface eth0 inet static
  address ...etc...
```

**/etc/resolv.conf**
```
nameserver 192.168.1.1
