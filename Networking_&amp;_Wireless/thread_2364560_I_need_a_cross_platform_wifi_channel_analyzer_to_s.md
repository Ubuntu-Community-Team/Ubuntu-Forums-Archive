### Post by johndough2 on 2017-06-25
Hi

The best channel is likely to be the one least used by those around you.  Since WiFi can be turned ON/OFF it can be difficult to select one for the longer term.

NONE OF THESE CARRY ANY WARRANTY OR GUARANTEE FROM ME.
Using tools/utilities that are Lo-Cost/No-Cost, like:-  
Microsoft Network Monitor,  inSSIDer,	TamoSoft Throughput Tester,  Angry IP Scanner,	NetStumbler & LizardSystems Wi-Fi Scanner  can help to identify problem areas.

I have used a free version inSSIDer and liked it, but personal preferences are personal.
