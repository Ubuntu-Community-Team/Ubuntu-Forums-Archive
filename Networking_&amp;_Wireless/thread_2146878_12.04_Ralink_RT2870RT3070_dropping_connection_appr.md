### Post by praseodym on 2013-05-20
Try to use pure WPA2-AES (CCMP) encryption and deactivate the power management
```

sudo iwconfig wlan0 power off
```

### Post by saracen666 on 2013-05-24
PROBLEM SOLVED : Set IPV6 to ignore in EDIT CONNECTION. System Settings-> Network -> Options ->Wireless tab -> Click on network -> Edit -> IPV6 Settings set to ignore
