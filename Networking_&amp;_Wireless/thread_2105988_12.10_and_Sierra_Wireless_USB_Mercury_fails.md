### Post by bmork on 2013-01-31
> **silverbullet007 said:**
> That's my main problem.. it can't connect if no device is seen.  Unless there's a way to make NetworkManager use attached devices from the CLI?  It never appears in the gui so I can't attempt to connect.

I have no idea how this will appear with an older NetworkManager, but I believe you should be able to see the modem (as disconnected) using the nmcli CLI utility:

```
bjorn@nemi:~$ nmcli dev
DEVICE     TYPE              STATE        
modem0     gsm               disconnected 
wlan0      802-11-wireless   unmanaged    
eth0       802-3-ethernet    unmanaged    

```
"nmcli dev list" may or may not display more detailed information.
