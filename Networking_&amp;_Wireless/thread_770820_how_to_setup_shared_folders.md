### Post by Monicker on 2008-04-27
> **HakerDemon said:**
> I have hardy heron and i want to set up a shared folder with my apple laptop which is running leopard.  Can any one help me with this one??  it needs to be able to read/write too.

Right click on the folder that you want to share.  Select Sharing Options towards the bottom.  You should get prompted to install the sharing service.  Once that completes, log out and log back in, then adjust the permissions.  If you try to save changes before logging out you will get an error message, which is a known bug.
