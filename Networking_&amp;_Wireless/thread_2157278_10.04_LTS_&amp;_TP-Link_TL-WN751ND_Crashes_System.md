### Post by praseodym on 2013-06-25
Install the mainline-kernel 3.9:

[http://kernel.ubuntu.com/~kernel-ppa/mainline/v3.9-raring/](http://kernel.ubuntu.com/~kernel-ppa/mainline/v3.9-raring/)

You need all possible packages for 32 or 64 bit, respectively (plus the "all.deb" files).

Save them in "Downloads" and install via
```

sudo dpkg -i ~/Downloads/*.deb
```
Reboot.

I am using this kernel here with 12.04:

```
modinfo ath9k | grep 002D
alias:          pci:v0000[COLOR="#FF0000"]168C[/COLOR]d0000[COLOR="#FF0000"]002D[/COLOR]sv*sd*bc*sc*i*

```
