### Post by DFlame on 2008-10-29
That driver CD simply contains a small setup guide. The router is driven through ethernet so there is no need for any kind of real driver.

First of all, connect everything up in a powered down state (ADSL/Cable leads to the modem, modem to the routers WAN port, Computer to routers LAN port(s)). Switch on the modem, then the router, and finally the computer.

Once you've done this, you should be able to browse the the router set up page by via [http://192.168.10.1/](http://192.168.10.1/) . Log in as the admin user with a blank password. And there you have the set up page. This might provide you with more information: [http://techgage.com/article/trendnet_tew-631brp_wireless_n_router/2](http://techgage.com/article/trendnet_tew-631brp_wireless_n_router/2) . It becomes relevant to your problem about 1/3 down the page.
