### Post by changingstate on 2015-04-05
The problem is with your mydestination field in main.cf, that's why you're getting relaying denied errors in the mail log. The mail server isn't recognizing it's the destination for the mail. 

```
mydestination = ldap:/etc/postfix/ldap/mydestination.cf
```

Either change this line to read "mandelkind.ch" or check that file contains the domain.
