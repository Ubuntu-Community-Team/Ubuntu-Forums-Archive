### Post by d.j.schroeder on 2007-04-10
Ndiswrapper is actually on the install CD but isn't installed by default.  You can go to the synaptic package manager and add the CD to your repositories and from there you can install ndiswrapper.

If you don't know how to use ndiswrapper you can check here [https://help.ubuntu.com/6.10/ubuntu/desktopguide/C/connect-to-internet.html#ndiswrapper]("https://help.ubuntu.com/6.10/ubuntu/desktopguide/C/connect-to-internet.html#ndiswrapper")
