wifi probably simpler, but as long as you can run a cable to your router or switch wired would work just fine.

if your buddy is using windows you will need to use samba, if ubuntu samba would work also but open-ssh might be a better choice. In either case you will need to setup your machine as the server, for both the client functions are installed by default but the server function will need to be installed (easily done via Synaptic) and then setup to match your requirements.

yes he will be able to retrieve and place data on the drive, provided you set him up with a user account AND proper permissions on your network.
