### Post by gordintoronto on 2011-08-09
On Ubuntu "places," Home Folder, right-click on Documents, and select "Sharing Options." Click on "Share this folder," and go ahead with what is suggested. I usually allow everything possible.

Vista should see the folder under "network."

It can help if you say what version of Ubuntu you have installed.
