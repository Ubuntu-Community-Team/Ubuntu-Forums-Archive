### Post by JBAlaska on 2009-11-28
Is there more than one broadcom driver showing in "Hardware Drivers"? You will want to use this one: "Broadcom sta wireless driver"

Just to make sure what chipset your using, please post the output of this run in a terminal:
```
lspci
```

The native driver works very well for most broadcom chipsets, so you might want to try some more before using ndiswrapper.
