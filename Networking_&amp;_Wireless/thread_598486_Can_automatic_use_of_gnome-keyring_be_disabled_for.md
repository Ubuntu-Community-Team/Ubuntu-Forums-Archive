I just switched to 7.10 (clean install) and now the network manager automatically stores my wifi password/key in the keyring which results in automatic network login without a password prompt.  I came from 6.06 - where I had the option to deny access to the keyring (a little annoying for some but fine for my needs).

I actually liked it the way it was.  That is, **I want to type my password for each and every wifi network that requires one.**  I do not want passwords stored anywhere.

I've deleted the keyring and unchecked the read, write, delete checkboxes for nm-applet but it seems that after a reboot, the keyring is back and nm-applet has full control again...  If I dare select gnome-keyring for removal in apt/synaptic, I get a list of other software to be removed that works out to be basically the whole OS...

Other than that, 7.10 is freak'n awesome...   :)
