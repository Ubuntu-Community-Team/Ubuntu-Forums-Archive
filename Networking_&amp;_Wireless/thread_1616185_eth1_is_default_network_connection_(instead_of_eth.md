### Post by cipherboy_loc on 2010-11-07
If this is in NetworkManager, then it names different ethernet cables different things, but they all go through eth0. On my laptop I have 3 or 4 names for different cables. Look under Edit Connections->Wired (from the right-click menu on the NetworkManager applet) and delete all the connections there. Reboot.  That might fix your issue, assuming that you are using an ethernet cable and not Wi-Fi.


Cipherboy

### Post by watchpocket on 2010-11-07
This worked -- thank you!

It connected after reboot and put an "Auto ethernet" (with a white circle dot at its left) under "Wired Networks" in the nm-applet dropdown menu.

I edited that, changed its name to "auto eth0" and put in the MAC address that had been there previously.

I also created an "eth1" with the same MAC address (they both had the same MAC addresses previously).  Thanks again.
