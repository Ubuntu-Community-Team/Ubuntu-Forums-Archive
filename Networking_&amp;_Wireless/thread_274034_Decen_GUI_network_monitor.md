### Post by srf21c on 2007-03-21
Try the the System Monitor Gnome network applet.

Right button clock on the top toolbar, select "Add to Panel" and then pick System Monitor" under the "System & Hardware" category.

Once you add it to (IIRC) the "Gnome Notification Area", you can double click it, or right click to bring up the application window.  

The middle "resources" tab has some nice graphs of CPU, memory, and network activity.


Update:  I just found some more good resources for monitoring network activity in the [Ubuntu desktop guide]("https://help.ubuntu.com/6.10/ubuntu/desktopguide/C/networking.html")

[Another good howto]("https://help.ubuntu.com/community/HowToMonitorInternetTrafficTotals") in the Ubuntu community docs detailing the ipac-ng bandwidth accounting package.
