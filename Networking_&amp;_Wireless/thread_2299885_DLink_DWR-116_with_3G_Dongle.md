Hi, 

                                  After messing around with the DWR-116 and IDEA net-setter 3G for the last 8 hours and finally got it working.


 Nowhere in any instructions could I find what I've needed to do to get it working, finally found a post on internet at [http://forums.whirlpool.net.au/archive/2349248](http://forums.whirlpool.net.au/archive/2349248) by [Craig]("http://forums.whirlpool.net.au/user/86511")


 Earlier I was trying to set 3g dongle and the router to different IP addresses but setting wrong ip to router, then according to this post changed it with corrected one. 

By default both products have the same ip address of 192.168.0.1  . When changed the router to 192.168.1.1 everything started working.

thanks Craig.

I have also successfully tested DWR-116 with tata photon+.
