### Post by Matt You on 2008-02-21
I am very new to linux, but managed to share a folder over the network through samba.  My question is: is there an easy way to "unshare" this folder? So that users won't see it anymore?

---

### Post by Matt You on 2008-02-21
For anyone with the same problem:
  Go to [ System ==> Administration ==> Shared Folders ==> *Click the share that needs removing*, click delete ]

Yes, it was that easy, and yes, I'm that new to linux.
