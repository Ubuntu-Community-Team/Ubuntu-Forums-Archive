### Post by wadesmart on 2009-02-22
There are web cams that can work wirelessly and wired. A networked cam can plug right into the network like a computer. A wirelessly one can be secured to send to the network.

A local school just purcahsed a linksys wireless router and a web cam - allowed the router access to the network - it serves 2 mac address (school room laptop and wireless camera) and thats it. 

NewEgg.com has a lot of wireless camera - but in the end there are a lot of considerations before just doing this. 

Cost - of the camera, of the router, of wiring, etc
Security - can the IT Admin set this up without voiding the school systems..

etc.
