### Post by dsluser on 2011-09-20
Hi All, 

Just a quick note based on my saga with this router which is supplied by both Eircom and Digiweb.

I was having issues with 11.04 unable to route to the net or the router after 10 mins even though it was still connected. The same issue occurred with a viewquest slate 2 tablet. Windows XP, Vista and 7 all had no issues.

After weeks spent reconfiguring my wpa_supplicant, network manager, network interfaces etc. I swapped out the router for an old Thomson gateway and the problems were immediately resolved for both linux-based devices.

I had thought that routers would not be OS dependent but it seems that there is a problem with the implementation on this router. 

I'm posting hoping that I might save some other users a few weeks of hair-pulling, cheap routers can be picked up for around €40 and I would recommend something by Linksys based on past experience(most can be hacked with Linux and use open-source firmware)

Belkin may also be a good choice but I had problems previously with the 7051 usb wireless adapter and ndiswrapper.

Happy networking.
