### Post by FrodoB on 2006-12-09
After you system is booted up and you have logged in, insert the install CD into the CD drive.

You hopefully will get a message in about 10-15 seconds that says:

A volume with software packages has been detected.

Click on the "Start package manager" option rather than Cancel and the Synaptic tool will start.

Select "All"

type the first few characters of the package you are looking for and it should take you to the right line.

Select the package you want and click on the large Apply icon in the top bar and it should load what you need.

Good luck.
