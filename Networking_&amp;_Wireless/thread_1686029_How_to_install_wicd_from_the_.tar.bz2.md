### Post by IcarusR on 2011-02-11
In a terminal cd to directory containing tarball extract it with

```
tar -xjvf your_tarball.tar.bz2
```

See the tar man page

```
man tar
```

Then read the install.txt or readme.txt for specific install instructions.
