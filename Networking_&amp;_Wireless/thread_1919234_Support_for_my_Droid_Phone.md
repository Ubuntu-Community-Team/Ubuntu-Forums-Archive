OS X and iOS are not Linux.  OS X is a derivative of the BSD flavors of *NIX.  BSD was branchec off ito NextStep which then came to OS X.

Getting your phone to "work" on Linux depends on what you mean by "work".

You should be able to hook up a usb cable and have connectivity to your pjone like any other usb type storage.  Morks for my Infuse on Ubuntu.  There are some steps you need to do on the phone to activate this (responding to some prompts, so be sure to log onto your phone).  After that, you should see it in the file manager.

Syncing and such would need to be done via the Google wordls.  Calendar, contacts, and such sync to the Google counterparts.  Not sure what other type of syncing you want to do.  But unfortunately we still live in a Windows (and MAC) world.
