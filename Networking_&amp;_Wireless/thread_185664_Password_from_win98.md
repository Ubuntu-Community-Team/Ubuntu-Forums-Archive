### Post by Iowan on 2006-06-01
If you logged into win98 box as **Sugyi**, Ubuntu is expecting the password for the Linux user **Sugyi**.  The password may (or may not) be the same as the one you used for Windows.
   Samba is the usual method of file sharing between Linux and Windows.  [http://www.howtoforge.com/samba_setup_ubuntu_5.10]("http://www.howtoforge.com/samba_setup_ubuntu_5.10") You could (should?) also search the forum for other threads about Samba.
