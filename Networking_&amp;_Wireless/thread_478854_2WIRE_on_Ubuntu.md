### Post by p_quarles on 2007-06-19
I use the same service, and I can promise you that if you've configured the gateway to connect with Windows, it will also connect to Linux.

The problem is most likely the wireless device. Many of those require you to install a Windows driver through the NDISwrapper software for Linux. My wireless card worked fine with Linux, so I have little advice on this, but if you look up "NDISwrapper" in the forums, you'll find tutorials and how-tos.
