### Post by eggroid on 2005-09-10
After running ubuntu's update manager and installing the latest kernel image patch my system was unable to connect to the wireless network through WPA, which I had previously set up.

In order to get wireless networking with WPA working again I rebuilt the ipw2200 driver and ieee8020111 driver and rebooted - and now it's working perfectly again.
