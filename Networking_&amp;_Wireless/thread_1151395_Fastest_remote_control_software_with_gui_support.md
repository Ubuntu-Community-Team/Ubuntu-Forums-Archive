### Post by finite9 on 2009-05-15
yes it is slow.  I had the optimistic intention of video editing via SSH and X forwarding.  What a joke :)

I use vnc at the moment (vncserver on server and Terminal server client on desktop) and this is better but nowhere near console speeds.  I have heard that FreeNX is a better remote solution, but it's not free software (they do have a free software version but you have to build it yourself and it is crippled compared to their commercial version...no usb i think).  But that was the status a couple of years ago, so I dont know if the free version is any better now.

regarding vnc, there are a few different versions, tight vnc, realvnc ultravnc etc, but the realvnc version on linux is very adequate... you dont gain anything by fighting trying to get the other versions working under linux.
