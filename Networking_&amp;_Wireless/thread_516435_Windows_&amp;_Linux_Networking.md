### Post by Wharf Rat on 2007-08-03
I am experimenting with Ubuntu on a laptop while responding to you on an old PC with Xandros loaded. But, I am no means an expert here.  
Havng said that, I have used a Linux server at work for over seven  years. I have a Nitix box running for the past three years  with only about 3 forced reboots.  Pretty good record, I'd say.    :)

I am using a Nitix box.  [www.nitix.com](www.nitix.com)     It isn't free but it is configured out of the box for: Windows users, Vista, Mac; auutomatic backups; firewall; anti-virus (for the user data), mail, web services, database services (MySql), and it mimicks an Exchange server so you can use Outlooks range of options.
Shared mail, calendar, contacts, etc.

I understand they now support Domino.

It insn't free.  I invested about the same as I would have for a starter Novell or Windows system.  I started with a   simple 6 user setup and now support 16 users.  About my only interaction with the server is to change backup drives.

Someone here will likely tell you that all this can be done with freeware -- it might. Assuming you have the time and skill.  My server was ready to have users log in 1 hour after unpacking it.  My time is worth something.  Especially in a small business.  I am not in the IT arena.
 I love my Nitix box.:-({|=
