### Post by openaddict on 2007-10-15
You will need to configure the network connection with your specific wireless network settings.  If it's an encrypted WAP (and it should be, for security purposes), you'll need to know the key along with the SSID (the ID your access point broadcasts).  One of the easiest ways to get this information is to boot back into Windows, go to your wireless network settings, and write down all of the connection information.  You could also save it to a text file on your hard drive since Ubuntu can read NTFS partitions.  Boot back into Ubuntu, click on the network tool in your toolbar or go to System - Administration - Network and you can edit the properties of your wireless connection there.
