### Post by cariboo on 2009-07-08
To set a static ip address for wifi, right-click on the network-manager icon in the top panel and select edit connections, in the window that pops up select your device and click edit, you will be asked to enter your password. Next in the Edit settings window, click the IPv4 settings tab. Click the method dropdown and select manual, then click the add button to enter your ip address, netmask and gateway. Scroll down and enter your DNS server address. Then click apply and you're done. Your new static ip address will be automagically applied, no need to restart.
