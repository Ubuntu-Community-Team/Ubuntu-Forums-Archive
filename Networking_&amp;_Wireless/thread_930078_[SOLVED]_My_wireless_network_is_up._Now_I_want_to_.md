### Post by lisati on 2008-09-25
Both your router and your Linux installation need to be set to use security measures with wireless. The setup process is sometimes easier if you are able to use a wired connection to your wireless router.

Have a look [here]("http://ubuntuforums.org/showthread.php?t=202834") for a long but potentially useful thread - most of the information you'll need is likely to be in the first post.
