### Post by nixscripter on 2008-10-26
A router is only as good as the software that runs it. A really cheap machine (like a Pentium II!) can do a sufficent job, limited only by its I/O bus and Ethernet cards.

I would suggest buying a cheap PC and using a couple of Ethernet cards. Here's a Live CD (don't even have to install anything), which you can boot off of: [http://www.wifi.com.ar/english/cdrouter/](http://www.wifi.com.ar/english/cdrouter/)

Hope this helps.
