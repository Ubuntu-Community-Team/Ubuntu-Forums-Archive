### Post by d3v1150m471c on 2010-03-18
When you launch network-manager navigate to the wireless tab and add a new wireless network. There should be a scan button that you can use and then set it to show details. It will show both the bssid and the essid. If you can't figure that out:
```

sudo apt-get install aircrack-ng
sudo dbus-send --system --type=method_call --dest=org.freedesktop.NetworkManager /org/freedesktop/NetworkManager org.freedesktop.DBus.Properties.Set string:org.freedesktop.NetworkManager string:WirelessEnabled variant:boolean:false
sudo airodump-ng wlan0

```
