### Post by jml on 2006-05-30
An easy way to tell if your hardware will work would be to boot up the Ubuntu Live CD.  It does not do anything to your hardware, the OS resides in RAM and in temp files.  Once its booted, open up the network application and see if your card is listed.  If it is, then configure the card, and then activate it and see if you can connect.  If it works with the live CD it should work when Ubuntu is installed.  Its also a good may to make sure things like video resolution, sound card and other hardware works.  Fair warning, getting wireless to work in Ubuntu can be a challenge.  Here is a link to a wireless troubleshooting guide you might find helpful.

[https://wiki.ubuntu.com/WirelessTroubleshootingGuide](https://wiki.ubuntu.com/WirelessTroubleshootingGuide)
