I'm not familiar with Windows Directory Service, so I'll give an example of what I like to do in the Windows NT environment.

In windows we have the Primary Domain Controller (PDC) with all the users access control, then a client/ workstation can join to the domain to access all the shares. A user can log into any workstation and see the same profile and have the same access right as if they were on the same system. 

Of course, I like to have this done in the Linux environment.
