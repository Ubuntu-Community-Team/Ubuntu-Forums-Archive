### Post by lukeiamyourfather on 2013-09-05
Move the virtual machine window to the second display output (projector). The multiple virtual monitors work only in certain configurations and I'm pretty sure it doesn't work with a Linux host. See the VirtualBox documentation for the full details on the supported configurations.
