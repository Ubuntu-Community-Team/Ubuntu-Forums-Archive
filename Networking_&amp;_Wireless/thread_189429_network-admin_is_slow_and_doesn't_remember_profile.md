### Post by zaiyon on 2006-07-16
I think I have to excuse, since I finally got how profiles work.

If you have this problem too:
Do not do anything up there on the dropdown menu, just set everything up.
Once you're finished, select "Add profile"

Your current settings will be added as a profile - think about the name, because there is no other way to change it then to delete and resetup the profile.

Once created, you cannot make changes to a profile, delete and resetup for that.

But it is still slow ;)
