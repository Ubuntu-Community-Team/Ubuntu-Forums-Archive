### Post by Lambert on 2007-11-09
N devices currently don't have native drivers in linux. You can look into using ndiswrapper. There is some success with devices that use the airgo or marvell chipset but I haven't read about any others.

you can search this forum or the wiki.ubuntu.com site for info on using ndiswrapper.
