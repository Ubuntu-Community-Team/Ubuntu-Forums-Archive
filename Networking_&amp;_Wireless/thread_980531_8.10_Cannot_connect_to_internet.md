PROBLEM SOLVED

From JFo NZ, solved my problem. Needed both code lines.

Re: No Ethernet in Intrepid Ibex
After upgrading to 8.10 my ethernet stopped. You might have the same problem. This is how I fixed it.

Edit /etc/network/interfaces

Code:

sudo gedit /etc/network/interfaces

Add 2 new lines to "/etc/network/interfaces"

Code:

auto eth0
iface eth0 inet dhcp

Then restart machine. Fixed.

### Post by pbubenik on 2008-11-25
I too was unable to connect to the internet with Ubuntu 8.10 and this solution worked for me.

For some reason, on installation, Ubuntu 8.10 did not give the (only) user (me) the privilege to connect to wireless and ethernet networks.

Click on System >> Administration >> Users and Groups
Then click on your user name.
Click Properties and then the User Privileges tab.
Check to see that Connect to wireless and ethernet networks is checked.

After this change, everything worked perfectly!
