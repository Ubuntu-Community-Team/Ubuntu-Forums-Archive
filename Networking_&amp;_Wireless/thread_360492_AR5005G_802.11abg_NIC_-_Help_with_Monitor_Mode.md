### Post by spd106 on 2007-02-13
You need the madwifi tools to be able to create a VAP in monitor mode. For more information check out [http://madwifi.org](http://madwifi.org).

Although the driver is include with Edgy, the tools are not. Feisty has a madwifi-tools package ported from debian testing, but since Feisty won't be out for a couple of months yet I've asked for it to be backported. If it is successful, then the tools should be available within a week or two.

Incidentally, the madwifi-old driver in Dapper doesn't need the extra tools.
