### Post by jasontan6 on 2007-11-04
Are you using Gutsy? You need to select which repositories you want Ubuntu to search for software.

Goto:
System > Administration > Software Sources

Tick the types of software you are interested in downloading. Basically you should tick "main" and "universe". If you have graphic cards or hardware which only use manufacturer drivers, tick "restricted". If you are looking for non-open-source software, tick "multiverse". Lastly, you can also tick "Source code" to download uncompiled programs and let Ubuntu compile when installing.
