How do you install a Netgear N300 Wireless USB Adapter on a Dell Latitude D830?

---

### Post by Bucky Ball on 2012-07-15
Start by plugging in the dongle, plug in an ethernet cable, boot the machine and make sure you are online, do an update (Update Manager), then check 'Additional Drivers' if your card is not already up.

Once all this is done, reboot. ;)
