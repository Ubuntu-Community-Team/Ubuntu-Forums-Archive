### Post by gordintoronto on 2012-09-01
Generally, the only thing you really need for configuring a router is information from your Internet Service Provider and the router's manual.

The manual probably tells you the default IP address, admin account and password. Once you log in as admin, you should be able to set a more private ssid and a stronger password, as well as a WPA password.

If the router has been used previously, you might need to reset it to factory defaults. Once again, that's in the manual.
