### Post by brigadoon on 2008-06-01
I had similar problems. Most of my neighbours have wireless networks. I ended up using manual configuration for networks. I manually joined my home network and then saved the configuration as a location. I also did the same for my office location and several other places I haunt. My wireless card is automatically set to roaming. Since I saved my preferred locations I haven't logged onto the wrong network when roaming. I hope this helps.
