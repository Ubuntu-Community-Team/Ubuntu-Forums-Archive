### Post by luisdavila on 2009-10-13
hi, i don't know what do you want to do with backtrack's program, but i'm not sure that "all" fonctions featured by backtrack are 100% available for a standad ubuntu installation.  The repositories are specific for backtrack and there are a lot of patches for kernel or others programs for better hardware detect (at least for wireless cards) and other additional features like injection packages, monitor mode, etc.. 
Not because is "ubuntu based" is simply "another ubuntu with other programs"... there are another specifics features.  In ubuntu you have a lot of software for sniffer network, injection packages, or other networks functions like kismet, aircrack-ng collection software, nmap, etc etc but is a really big list...  I think that it easier to burn your BT iso image and test in LIVE mode or install in other partition and only in function that you are need or want to do, install and test these software over ubuntu 'classique'.
