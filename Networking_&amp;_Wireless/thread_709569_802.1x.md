### Post by spd106 on 2008-02-27
wpa_supplicant, which should be installed by default, will handle 802.1x.

See [https://help.ubuntu.com/community/Network802.1xAuthentication](https://help.ubuntu.com/community/Network802.1xAuthentication) 

It might be worth asking the network admins for the specific settings or maybe even a Linux compatible script (wpa_supplicant.conf).
