### Post by RandomJoe on 2007-01-20
Have you had the cablemodem working before, with this or any other machine, or is this the first attempt?

If you are connecting directly to the cablemodem, and normally have it on another computer, did you cycle power on the modem?  Mine only allows one MAC address to get an IP (unless I pay more for extra IPs), so to let another computer (with different MAC) to connect I have to cycle the modem so it resets.

There also used to be a number of companies that required you register your _computer's_ MAC address with them as well as the cablemodem - a royal PITA.  Very few seem to do this now, but it's something else to check.
