### Post by clausrei on 2012-03-01
It doesn't seem to be a problem of the OS or driver at all. It is your Internet connection, but you are obviously connected to the Internet - so thats fine.

>  
I checked the network activity graph and noticed that as soon as I clicked on any of the photos the activity dropped to zero.
When a Internet page is loaded the received and send packages naturally drop to zero, because there is nothing new requested or have to be send. 

> 
I click on any of the photos to expand them the loading icon appears and nothing happens.
 If the browser never completes loading the picture, then this could be because your browser session times out after a while and stops loading.

> 
As soon as I refreshed the page or opened a new window one the activity would come back.
I have also tried this with Chromium with the same out come.
Of course, because the browser sends a new HTTP request to the server: Please send me the website  [www.yours-etc.com]("http://www.yours-etc.com").

So everything is working fine.** It must be your Internet connection, which is very slow. **I would suggest to check your Internet connection for speed.

### Post by winh8r on 2012-03-02
Try this :

open firefox

in the address bar enter the following:

```
about:config
```

it will open a page with a warning, click on "I'll be careful"

in the search box type:

```
ipv6
```

you will see an entry that reads:

```
network.dns.disableIPv6 
```   you need to click on the entry, then right click and select "toggle" to make it read :

```
network.dns.disableIPv6  user set    boolean     true
```

then close the window 

and restart firefox.

check your speed again and see if you get any improvement.

There should not be any adverse effects by changing this setting , but if you do experience anything odd then you can just go back in and revert the setting by doing the reverse of the above.

Hope this helps.
