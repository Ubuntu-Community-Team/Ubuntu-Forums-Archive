Synaptic Package Manager (or Software Centre), b43-fwcutter and firmware-b43-installer. You are probably missing the firmware perhaps. You will need to uninstall the sta driver first. 

Sure you haven't got something wrong in your wireless details on the local machine? All details match the router? Tried putting the connection in Network Manager Manually? Using static IP address?
