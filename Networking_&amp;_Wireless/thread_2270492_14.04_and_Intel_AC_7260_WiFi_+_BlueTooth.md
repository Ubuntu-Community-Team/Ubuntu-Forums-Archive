### Post by jeremy31 on 2015-03-23
You can find finmware [http://git.kernel.org/cgit/linux/kernel/git/firmware/linux-firmware.git/tree/intel](http://git.kernel.org/cgit/linux/kernel/git/firmware/linux-firmware.git/tree/intel) that is only a month and a half old, it may or may not help.  It needs to be in /lib/firmware/intel

It might also be an issue with the bluetooth coexistance coding of the module, you could try ```
echo "options iwlwifi bt_coex_active=N" | sudo tee /etc/modprobe.d/iwlbt.conf
``` Reboot and if this makes things worse just ```
sudo rm /etc/modprobe.d/iwlbt.conf
```
