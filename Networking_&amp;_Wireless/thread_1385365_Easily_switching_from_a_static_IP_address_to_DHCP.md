### Post by pricetech on 2010-01-19
I can't give you command line that would help, but changing the setting in Network Manager shouldn't take long.

Here's an alternative though:  See if your router's DHCP server has a setting to reserve a specific IP for a specific MAC.  If so, you can leave your laptop set for DHCP but always get the same IP when you're plugged in at home.

Hope this helps.
