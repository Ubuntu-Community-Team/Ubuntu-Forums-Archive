### Post by papibe on 2016-04-07
Thanks.

It looks fairly OK. If pings to outside IPs is not working, I would test access to the gateway: 192.168.1.1
```
ping -c3 192.168.1.1
```
If it responds, it means the machine has network access to the gateway, but the latter is not routing its traffic.

Is proxmox NATing and serving as a gateway?
Are the VMs bridge to the rest of the LAN?
If so, Is 192.168.1.1 then your modem/router to the Internet?
Does it have firewall rules blocking/filtering any traffic?

Regards.
