### Post by OoooMatron on 2008-05-09
I use one pc for storage and then set up a permanent smb connection in fstab on Ubuntu.

I run mpd as the music server on each machine and Ario as the interface. I can have all my media collections on the different computers this way but only stream from one.

Another solution i have used is gnump3d and streamed it via a web interface direct to Totem. That works nicely for audio and video but you sacrifice some playback options.

The first solution is my favourite now.

Lastly, for video i have an xbox running xbmc and stream direct from the main computer.
