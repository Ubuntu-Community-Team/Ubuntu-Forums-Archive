### Post by blackened on 2008-11-15
Most consumer level routers are about the same in my experience. You won't need any software to set them up, but you will need to read up on how to secure the wireless portion of your router, and then how to connect to it using those security settings.

If you plan on going this route, then expect to have to troubleshoot something at one time or another. Networks are almost never trouble-free if only because of power-outages, poor coverage, and the like.
