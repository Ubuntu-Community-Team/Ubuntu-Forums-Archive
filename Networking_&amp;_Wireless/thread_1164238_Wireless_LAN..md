### Post by puppywhacker on 2009-05-19
wireless and wired are only providing lower link access. so yes you can do anything you could via wires, the only differences is that setting up wireless is a bit more work and the speed of wired is generally higher.

both of you have ip-addresses, so to share files you have a few different options. The easiest is to setup an SSH server on one system and using the nautilus browser on the other one to browse the files.

you can connect over SFTP by connecting to the server using it's IP-address and connect using the username and password of the server

goodluck
