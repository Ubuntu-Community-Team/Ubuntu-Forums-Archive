### Post by amsterdamharu on 2010-12-26
> Reported to work in 32bit with drivers that came with the device but not in 64bit
[http://sourceforge.net/apps/mediawiki/ndiswrapper/index.php?title=Linksys_WUSB300N](http://sourceforge.net/apps/mediawiki/ndiswrapper/index.php?title=Linksys_WUSB300N)

I think this is your XP driver:
[http://homesupport.cisco.com/en-us/wireless/lbc/WUSB300N](http://homesupport.cisco.com/en-us/wireless/lbc/WUSB300N)
Open with archive manager and extract

Then use the WUSB300N_20071204/Drivers/XP_2K/netmw245.inf
Let's say you extracted to desktop then you can use the following command:

```
sudo ndiswrapper -i ~/Desktop/WUSB300N_20071204/Drivers/XP_2K/netmw245.inf
```

To check if the driver was installed:
```
ndiswrapper -l
```

If there are no errors so far you can try the following commands:
```
sudo depmod -a
sudo modprobe ndiswrapper
```

See if there were any erros:
```
tail /var/log/messages
```

Please post any output you think is an error message.

### Post by amsterdamharu on 2010-12-27
I didn't have any experience with the ndiswrapper but it looks like you have to do the following:
Press alt + F2, type the following command "gksudo gedit /etc/modules" and press enter:
Now add the following line at the end of the file:
```
ndiswrapper
```Save it and it should be loaded by network manager.

[https://help.ubuntu.com/community/WifiDocs/Driver/Ndiswrapper](https://help.ubuntu.com/community/WifiDocs/Driver/Ndiswrapper)
Under 3.7

Under 3.6 it explains how to check out if it works.
