### Post by randlieb on 2011-07-18
Am using fresh install Ubuntu 11.04

I recently bought a new Zoom modem and Hawking wireless router. I plugged them in according to the installation guide and they worked right out of the box. I get a wireless connection but am unable to set up a WPA password.

Here's what I do:
1. network manager > edit connections > wireless tab
2. highlight the connection and click edit
3. click wireless security tab
4. at the security dropdown select WPA & WPA2 personal
5. enter my password
6. make sure 'connect automatically' and 'available to all users' are both checked
7. click save (ubuntu asks for authentication and I enter my login password)


When I click on network manager the icon for my connection does not show that it is secure.

I restart the computer but the connection does not automatically start up. The connection is in the list when I click on the network manager. I click on my wireless connection name and it connects but it creates a new listing in the edit connections window with no security enabled.

I have googled and looked through these forums for an answer but to no avail. A number of searches led me to answers that said to install WICD and uninstall network manager. Did that but also no go.

I am glad wireless is working but REALLY want a secure connection!

HELP!!! (and THANKS in advance to all who post here)

### Post by |{urse on 2011-07-18
You need to set that password in your wireless router, one generally does .that by typing its address in your internet browser. It's generally something like 192.168.2.1 

Be sure to set a password on your wireless router's login page as well. Else some nice person might login and set the password for you and not tell you what it is.
