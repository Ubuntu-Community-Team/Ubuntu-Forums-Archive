### Post by squaregoldfish on 2010-06-30
What do you mean by 'unstable'? Does the connection drop, or is it just slow?

If it's slow, that's understandable - torrent clients are generally bandwidth-hungry, especially when uploading, so you'll often find that everything else struggles for bandwidth. It should be possible to set limits on your bittorrent client so everything else has a chance.

I tend to run my torrent stuff overnight while I'm not using the connection.
