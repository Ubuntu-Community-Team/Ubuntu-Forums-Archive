### Post by wildmanne39 on 2014-12-02
Your wired connection is using the wrong driver that is why it is so slow:

```
sudo apt-get install linux-headers-generic build-essential dkms
wget http://ftp.de.debian.org/debian/pool/main/r/r8168/r8168-dkms_8.038.00-1_all.deb
sudo dpkg -i r8168*.deb
echo "blacklist r8169" | sudo tee -a /etc/modprobe.d/blacklist-r8169.conf
sudo modprobe -rfv r8169
sudo modprobe -v r8168

```
Reboot, it should download the files eventually if it does not time out.

As for the wireless  the driver here should fix your issue.
