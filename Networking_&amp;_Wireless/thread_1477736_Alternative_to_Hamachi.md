### Post by Gaming4JC on 2010-05-28
Hey, just bumped across your topic while googling for the same thing myself.

It seems there are quite a few alternatives, though you should consider if you actually need to use them. It is possible to install Hamachi on Ubuntu.
1) Follow this old, but still good, tutorial:
[http://supware.net/HamachiUbuntuHowto/](http://supware.net/HamachiUbuntuHowto/)

2) When you get to the part "sudo modprobe tun", you'll need to run the script by cdenley on another post for fake tun:
[http://ubuntu-ky.ubuntuforums.org/showthread.php?t=1459559&page=6](http://ubuntu-ky.ubuntuforums.org/showthread.php?t=1459559&page=6)

3) Choose a GUI.
Haguichi: [http://www.haguichi.net/download/](http://www.haguichi.net/download/)
Quamachi: [http://code.google.com/p/quamachi/](http://code.google.com/p/quamachi/)
gHamachi: [http://linux.softpedia.com/get/Communications/Conferencing/gHamachi-30358.shtml](http://linux.softpedia.com/get/Communications/Conferencing/gHamachi-30358.shtml)
YAHG: [http://sourceforge.net/projects/yahg/](http://sourceforge.net/projects/yahg/)


Concerning alternatives, in all honesty they don't compare to hamachi. Remobo got some poor ratings, here's two with a little better:
Wippien: [http://www.wippien.com/linux.php](http://www.wippien.com/linux.php)
Tinc: [http://www.tinc-vpn.org/](http://www.tinc-vpn.org/)


Hope this helps some one. I'm still compiling my GUI for hamachi on Lucid, if I get one to work I'll blog it. :)
