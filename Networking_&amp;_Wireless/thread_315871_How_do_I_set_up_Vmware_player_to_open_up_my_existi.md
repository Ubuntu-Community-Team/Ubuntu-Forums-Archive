### Post by susantha on 2006-12-10
Hi,

  I think you've misunderstand the questions you've post. To my knowledge you can't run VMware player and boot a os in a harddisk. Vmware player can run vmware os images. If grub asking which one to boot then you've the option to selecg which one to be the default os. It has detected you've 2 operating systems in your machine and giving you the option to select the default one. 

   If you like to run windows xp within the linux then you need to download the vmware player for linux. You'll also need a windows XP image. To do that actually you'll need vmware workstation. One option would be download the VMware server and try to create and image then open it from the vmware player. Hope this will help you to identify and solve some of your problems.
