### Post by Iowan on 2009-07-30
Depending on NIC(s), it'll probably be either a crossover cable, or via hub/switch.  You will probably need to install a server of some sort on the "source" machine.  Jaunty comes with clients for several protocols (FTP, Samba, SSH, etc.), but the server usually must be installed.  [https://help.ubuntu.com]("https://help.ubuntu.com") has several help pages available.
