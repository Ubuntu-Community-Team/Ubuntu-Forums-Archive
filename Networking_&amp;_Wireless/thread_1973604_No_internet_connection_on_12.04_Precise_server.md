sudo /etc/init.d/networking restart

Off-Topic: The forcedeth driver from LAN often needs additional parameters:

```
echo "options forcedeth msi=0 msix=0" | sudo tee /etc/modprobe.d/forcedeth.conf
sudo modprobe -rfv forcedeth
sudo modprobe forcedeth
sudo /etc/init.d/networking restart
```

### Post by praseodym on 2012-05-24
Try it with
```
iface wlan0 inet static
	address 192.168.5.11
	netmask 255.255.255.0
	broadcast 192.168.5.[COLOR="Red"]255[/COLOR]
	network 192.168.5.0
	gateway 192.168.5.1
```
You can also configure it additionally for WPA2-AES with
```
     wpa-driver wext
     wpa-ssid YourSSID
     wpa-ap-scan 1
     wpa-proto RSN
     wpa-pairwise CCMP
     wpa-group CCMP
     wpa-key-mgmt WPA-PSK
     wpa-psk "yourKey"
```
