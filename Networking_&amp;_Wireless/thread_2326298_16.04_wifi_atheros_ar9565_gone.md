### Post by jeremy31 on 2016-05-30
```
systemctl restart NetworkManager.service
```
See if that helps, sometimes the renaming of the wireless interface messes up Network Manager

Why do you need the 4.6 kernel?  Using an upstream kernel may leave you vulnerable to security issues
