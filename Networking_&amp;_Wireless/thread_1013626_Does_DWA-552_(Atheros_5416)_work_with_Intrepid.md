### Post by kortsen on 2009-02-14
I came here to ask this too.  I have been searching for answers for a D-Link DWA-642 (AR5416+AR2133).  There is one in the bargain bin at my local hardware store and I hoped to find the answer before buying it.

I was given hope when I read [http://wireless.kernel.org/en/users/Drivers/ath9k](http://wireless.kernel.org/en/users/Drivers/ath9k) and found > Any distribution shipping a kernel >= 2.6.27 will have ath9k present. Below is a list of the minimal distribution requirements to either use ath9k from 2.6.27 or from compat-wireless. 

**Ubuntu Intrepid (8.10)** 
Fedora Core 10 
OpenSuse 11.1 
Mandriva Linux 2009.0 (formally known as Mandrake Linux) 
ARCH Linux 
Gentoo Linux 
Red Flag Linux 7.0 (based on Fedora 10)

The tubes are also littered with success stories involving building the latest MadWifi and using the BIN files from the manufacturer's XP drivers.
