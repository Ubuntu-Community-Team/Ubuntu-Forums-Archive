### Post by wat3r on 2007-03-06
Hello fellow Ubuntu's/Ubuntuers or whatever!
Lets start at the beginning, trying to explain very thoroughly.
When I made "The switch", as it is populary called, I copied all my important files over to a Windows computer. Yesterday evening, I decided that I was going to bring them back to my good, nice working Ubuntu laptop. 

Previous that day, I experienced a problem with my wireless network card disappearing from the system. I don't remember exactly what I did, but I do remember it being found, but saying I did not have permition to use it. I found out that the last time I turned off the computer, the kernel was upgraded from 2.6.17-10 to 2.6.17-11. After some looking around the forum, I saw that it was a common problem with this kernel, so I stopped using it. Everything ok.

But anyway, back to my problem. I found this guide to use Samba (Very good guide, by the way): [http://www.ubuntuforums.org/showthread.php?t=202605&highlight=samba+windows+linux+network+howto](http://www.ubuntuforums.org/showthread.php?t=202605&highlight=samba+windows+linux+network+howto)
Of course, it worked very good yesterday evening. I just test copied a file to see if it worked, and it did. So I went to bed, thinking that I would copy the files today.

But when I booted up the computer today, the same wireless problem occured. It is like the wireless network card has disappeared. I don't remember exactly what commands I used to find out that it actually was recognized by Ubuntu, since there was a lot of terminal-tapping going on that day, using different howto's and guides.

So my question to you gurus out there is; Reading the howto I linked to earlier, can you see a reason why the wireless card would "disappear" again? I am using the 2.6.17-10 kernel now. That I have quadruple-checked. And I have rebooted so many times, I am starting to get sick of the startup-tune in Ubuntu.

I hope there aren't too many typos in this post. If there are, bare with me.
