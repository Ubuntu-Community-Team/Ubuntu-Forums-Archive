### Post by Iowan on 2009-12-28
I doubt the [ICS]("https://help.ubuntu.com/community/Internet/ConnectionSharing") How-To will help - since the sharing is on a Windows machine. You'll need either a crossover cable between the two computers or a hub/switch (or I hear some of the gigabit NIC's are smart enough to figure it out). Then you'll need to set up either zero-conf networking - or set up addresses in the same subnet for the two machines.
