### Post by e64462 on 2013-02-08
Hello all,

I'm running Kubuntu 12.10, with the D-link DWA-556 network adapter

As the title suggests, I'm using the ath9k driver, and the speeds are just abysmal, often less than 100 kbps. I've tried several fixes... and nothing has worked for me. My attempts have been:

```
nick@desktop:~$ cat /sys/module/ath9k/parameters/nohwcrypt
1
nick@desktop:~$
```

```
nick@desktop:~$ cat /etc/pm/power.d/wireless 
#!/bin/sh

/sbin/iwconfig wlan0 power off

nick@desktop:~$ 
```
(This temporarily helped, but not much)

```
nick@desktop:~$ cat /etc/modprobe.d/ath9k.conf 
options ath9k nohwcrypt=1
nick@desktop:~$ 
```

I've tried changing the wireless channel to avoid interference from nearby networks. I've tried choosing WPA2-only mode. I've tried using wicd instead of Network Manager. 

All of this has resulted in only marginal (and QUITE intermittent) increases in speed. When I place my laptop (running Ubuntu 10.10) using a different driver / chipset right next to my desktop (using the ath9k driver), i get max speeds in excess of 5mbps... 

Please, any advice is welcome because I'm truly at a loss here!
