### Post by drbongo on 2008-11-29
Solved! The command needed is nm-connection-editor, this brings up a gui which enables you to specify the network name, enter an encryption key and then connect to a wireless network. If anyone can work out the terminal command to scan for available wireless networks and select one please let me know.
