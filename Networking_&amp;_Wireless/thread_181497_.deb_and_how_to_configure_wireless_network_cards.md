Yeah, I've been trying to make the switch as well and reinstalled WinXP on my desktop yesterday. Very frustrating stuff.  I made some progress on my own.  After start up has finished place the ubuntu CD in your drive.

I think dpkg is the tool you need for unpacking files with the .deb extension.  If not, sorry I didn't write all this down :) , then try reading the references for appropriate tools at the bottom of the dpkg manual.

After installing ndiswrapper, insert the CD that contains the drivers for your card and save them to a folder on your desktop.  In a terminal, go to the folder with your drivers, and type           ndiswrapper -i [drivername].INF

Now, I think you have to give a command so that the kernel can use your driver, try         ndiswrapper -m [drivername].INF    or try          modprobe ndiswrapper

Now, if you haven't already, insert your card and restart your computer.  Go back into a terminal and type              ndiswrapper -l          this should return   driver present, harder present        if everything is ok.

Now, to turn the card on type      modprobe ndiswrapper            you should see the LED come on.   This is as far as I got, I couldn't get network-admin to even recognize wlan0.  So, I said screw it.


Hopefully this helps.
