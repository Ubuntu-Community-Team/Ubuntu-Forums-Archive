### Post by paker on 2008-10-22
GOAL: switching from wired to wireless
PC: Wired, but has wireless card inserted and driver installed.
ENVIRONMENT: PC & router in the same room. 8.04 32 bit. Broadcom 4318.

Click on the network connection icon @ top right corner.
Wireless network is listed. 50% signal strength. 
Select wireless.
A circle runs around and around.
Put cursor on the circle.
Message reads: Waiting for network key for the wireless network xxxx.

Is it waiting for me to enter my secret wireless passphrase that I setup when I first installed wireless router?
If the circle runs without stopping, how can I enter the passphrase?

---

### Post by Peter09 on 2008-10-22
Yes its waiting for you to enter the network key. You can do this by setting up the network parameters for your network. Try (right or left click - cannot remember which)  on the icon. It should give you the option to set up the parameters for the network by clicking on the name of the network you want to attach to.
