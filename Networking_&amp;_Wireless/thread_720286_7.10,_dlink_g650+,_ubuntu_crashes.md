Is this a wireless adaptor. If it is, then u might be using default driver ,which comes with Ubuntu. U need to install serialmonkey drivers. It solved my connection problem. If it is rt73, here is the thread to install rt73 driver [http://ubuntuforums.org/showthread.php?t=400236]("http://ubuntuforums.org/showthread.php?t=400236")
