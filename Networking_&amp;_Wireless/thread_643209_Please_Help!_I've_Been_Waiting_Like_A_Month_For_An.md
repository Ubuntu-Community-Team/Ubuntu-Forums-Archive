### Post by Junglizer on 2007-12-20
That is because this forum is made up mostly of Ubuntu users. Ubuntu is "noob friendly, user friendly, beginner Linux," whatever you want to call it. This means that aproximately 90% of its registered users are relatively inexperienced in regards to the use of a Linux system, therefore, expected good responses on often in-depth technical issues is merely an exercise in futility. In this case I usually recommend the user go to a generic linux forum, where the users registered are from various Linux distribution backgrounds and experience levels. Or I also like to direct them to the Gentoo Linux forums, a distro that has been for quite some time rather difficult to install, as it is a source distribution and system files such as fstab and the bootloader config must be written by hand during installation. This forum is located at:

[http://forums.gentoo.org]("http://forums.gentoo.org")
