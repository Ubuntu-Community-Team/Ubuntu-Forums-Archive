### Post by lorenzo on 2005-07-08
Have a look at: **[http://ubuntuguide.org/](http://ubuntuguide.org/)**

I have shared a "public" folder in read/write like written there, and it works. Search for:

[I]"Q: How to share public folders with read/write permissions (Authentication=Yes)?"
[/I]
You probably have to add a network user like explained under:
[I]
"Q: How to add/edit/delete network users?"[/I]

in the same guide.
