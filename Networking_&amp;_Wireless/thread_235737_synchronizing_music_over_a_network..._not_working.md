### Post by spd106 on 2006-08-14
It should work in rhythmbox quite easily. Try navigating to the folder in nautilus (file browser) then add a bookmark, there should now appear an link with the folder name and a folder icon with smb written on it.
Open rhythmbox then import a folder from the file menu, select your bookmark and you should be good to go.
