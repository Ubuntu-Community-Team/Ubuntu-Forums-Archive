### Post by DustyMP on 2009-06-08
I just installed Jaunty remix on my shiny new netbook and realized I couldn't remote into Windows Server without adding the terminal server client package that's installed by default on the desktop version. If you need this ability then enter: 

```
 sudo apt-get install tsclient 
```

Enter your password and this will get you everything including the pretty GUI the first time around. Hope this saves someone out there the time it took me to find it.
