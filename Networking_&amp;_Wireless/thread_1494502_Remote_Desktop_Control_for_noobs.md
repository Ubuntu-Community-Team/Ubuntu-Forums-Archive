### Post by julio_cortez on 2010-05-27
I've tried TeamViewer and I found it nice (at the extent that me and my UL asked for it to be installed on our PCs so we can easily remotely check machines that report troubles).
Never tried LogMeIn honestly.

TeamViewer has a "free" (non-commercial, so I think it can suit your needs) version which you should try out. It also offers an "un-attended login" option that allows you to login also if there's no-one confirming the session on the remote PC (you may be provided a "special" password for that, but since it's you that set it up it shouldn't be a problem).
In addition, it has whitelist and blacklist management (so you can expressly decide the IDs that can connect to your machine and the ones that cannot).

I suggeest you to install the "full" (for personal use, of course) version of TeamViewer on both machines, even if you may want to only install the TV-Viewer module on the PC you want to control.

As I said it's something I'm very comfortable with, at least for remote troubleshooting.

I'm looking for something different on my home PC (something more like "Remote Desktop" for Windows) and if this is what you also want I think TeamViewer isn't enough, but from what you described it may suit your needs.

### Post by howefield on 2010-05-27
Teamviewer for me, mainly because it easy for non tech users to set up.

It has many features such as text/audio/video communication, file transfer, remote rebooting and much more and now comes with a Linux version, albeit a beta.

You also have some included options within a default install of Ubuntu, Terminal Server Client supports RDP.
