thanks got more probs?
try to go in a terminal
> sudo pppconfig
Create new connection:
Change the connection name to 1xEVDO
for DNS Server use Dynamic
type nothing in ip
use PAP
$$$$$$$$$$@VZW3g.com (note: use the user name your provider has provided you)
password /for verizon vzw is default password)
changing the speed of your modem can mess with your connection speed. ( upload and download speeds) mines 115200
tone
#777 (this is the phone number for the modem to dial)
ttyUSB1 (caps MATTER) Normally the computer reads it as a modem so change it to ttyUSB1 for most commonly used usb port. for port info  type this in a terminal with your card plugged in.
> lsusb

after you detected usb port
> dmesg

> sudo apt-get update

Encountered errors?
> sudo apt-get wvdial

in a terminal make sure caps are right
> wvdial 1xEVDO

then the modem should connect with the terminal. but dont close terminal unless u wanna disconnect.
