### Post by Alabamaschalk on 2010-04-02
Hi!
I use VPN to connect to my windows machine at home while traveling. After the vpn connects I use the option places -> connect to server, chose windows share and type in the ip address of the machine.
Then I can browse the shared files with nautilus.
