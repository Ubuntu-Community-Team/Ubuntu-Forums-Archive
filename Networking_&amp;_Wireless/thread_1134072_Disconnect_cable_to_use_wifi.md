Hi Antonio,
In the notification area (normally upper right) there should be a small icon that looks like two screens overlapped or WiFi signal bars depending on which network interface you are using. If you click that, you should see entries for both wired and WiFi networking. Clicking on the entry will change state (disable -> enable or enable -> disable) so you should be able to select which one you want.

I haven't actually tried this, but the only thing I can think of that would cause problems is if an interface is set as automatic. You can right click on the icon in the notification area and select 'edit.' if you want to something other than just enable/disable the interface.

I think that's what you're asking and I hope this helps.

(this is fairly recent - I know it's there for Intrepid and Jaunty - not sure about earlier releases.)

-hank
