### Post by chili555 on 2016-10-14
Wonderful. You followed incorrect advice and overwrote the useful information already in the file. Moreover, the misguided suggestion adds a parameter that is not available. 

Please do:```
gksudo gedit  /etc/modprobe.d/iwlwifi.conf
```Use nano or kate or leafpad if you don't have the text editor gedit. Change the file to read:```
# /etc/modprobe.d/iwlwifi.conf
# iwlwifi will dyamically load either iwldvm or iwlmvm depending on the
# microcode file installed on the system.  When removing iwlwifi, first
# remove the iwl?vm module and then iwlwifi.
remove iwlwifi \
(/sbin/lsmod | grep -o -e ^iwlmvm -e ^iwldvm -e ^iwlwifi | xargs /sbin/rmmod) \
&& /sbin/modprobe -r mac80211
options iwlwifi power_save=0
```Proofread carefully, save and close the text editor. Reboot, test and report back, please.
