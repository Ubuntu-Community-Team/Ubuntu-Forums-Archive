### Post by zuke on 2009-11-01
A new feature in windows media player 12 is the option to have remote access to your media from anywhere using your live id.[http://windows.microsoft.com/en-us/windows7/products/features/remote-media-streaming]("http://windows.microsoft.com/en-us/windows7/products/features/remote-media-streaming")

When this feature is installed it breaks samba and Ubuntu cannot access windows 7 shares, a password box will pop up and no username/pw combo will work. Windows 7 is still able to connect to ubuntu linux shares.  
The suggested fix is not to turn this feature on/uninstall the live id program , I have looked and cannot find documentation for what changes to the networking protocol this feature causes.
