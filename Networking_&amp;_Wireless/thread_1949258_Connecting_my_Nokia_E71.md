### Post by Irihapeti on 2012-03-30
I've used a Nokia E63, which I think is fairly similar, as a data modem. From memory, when connecting the phone (via USB cable) to the computer, choose the PC Suite option, not data modem (or whatever it calls it). Then you can set up a new connection using Network Manager -> Edit Connections... -> Mobile Broadband -> add.

It's been a while since I've done this, and I was using 10.04, but I think it should work on 11.04.
