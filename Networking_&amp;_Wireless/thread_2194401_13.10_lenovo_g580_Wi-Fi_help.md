### Post by squakie on 2013-12-18
First check "Software and Updates" and the "Additional Drivers" tab to see if it shows any drivers available.  You would probably need to be connected (can you hard-wire temporarily to your router?) to install them.

If not, please post back the output of:

lspci | grep Ethernet

lspci | grep Wireless

lsusb | grep Wireless
