### Post by darrelljon on 2008-09-16
You need to go into your browser and type in the IP address as the url. Linksys routers, for example, use 192.168.1.1 for their internal IP address. D-Link and Netgear routers typically use 192.168.0.1. Some US Robotics routers use 192.168.123.254, and some SMC routers use 192.168.2.1.
