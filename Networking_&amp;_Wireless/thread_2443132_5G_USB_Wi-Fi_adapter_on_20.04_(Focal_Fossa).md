### Post by stafio on 2020-05-11
This is mainly an information post, as I had a hard time deciding which USB Wi-Fi adapter I was going to get. 

My situation was that I have an [older Zotac machine]("http://old.zotac.com/mx/products/mini-pcs/zbox-dvd/product/zbox-dvd/detail/zbox-blu-ray-id34.html") and wanted to be able to use my 5GHz Wi-Fi connection on it, running Lubuntu 20.04. 

After some research, I found a post elsewhere that [led me here]("https://github.com/MaxG87/DWA-182_Rev-D1") "The project started with official versions downloaded from D-Link's download page for the DWA-182 Rev D. Unfortunately, D-Link's version stopped do work on Linux v5.3." I followed the DKMS installation and was up and running pretty quickly with 5G Wi-Fi from the USB dongle.
