### Post by guyvertoon on 2007-12-24
I use dansguardian and LOVE its' word filtering capabilities. Much easier to manage than a whitelist IMHO. I have tried both ways.  I happen to use smoothwall with a mod, but I would imagine that you can do it pretty easily with Ubuntu. I chose Smoothwall because it is a very thin distro that is dedicated, rather than Ubuntu that I would have had to substantially mod to arrive at the same place. If you want to have a look, check out the 3.0 homebrew mods for Dansguardian + Clamav in their forums.
