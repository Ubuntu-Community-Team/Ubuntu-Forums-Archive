### Post by mustatommi on 2009-02-24
I wanted to install Ubuntu Studio (8.10 amd64 version) but i needed the wifi.
I don't know anything about compiling and whatever is written code.

I tried every way to install standard Ubuntu on my laptop to add later the metapackages of the Studio vers. (realtime kernel hasn't been implemented in 8.10 as long as i know), but it never ended right*.

Ubuntu Studio Cd installer worked flawlessly tough...

**So...** after dealing with many methods found over the forum that i don't understand (very first time in linux) and i wasn't able to follow None of those worked. here's my way:

1. Install Ubuntu Studio via cd WITH a working internet connection via ethernet port

2. update Ubuntu Studio with the Update Manager program (found via Start System > Administration > Update Manager) and restart if needed

3. Open Synaptic Package Manager (System > Administration > Synaptic Package Manager) scroll the right column to "Meta Package" and on the right column select "ubuntu-desktop"

4. with Add/Remove Applications (Applications > Add/Remove Applications) remove all the apps you don't need

5. rerun Update Manager (just in case :) )

6. enjoy the wireless freedom!

I know is not the best way, but this way was the only one i found that work without dealing with terminal and string command.

Hope this will help!


*= i tried 3 different cd burned with different burner and burning softw, i tried net install and install via usb. None of those worked. Don't ask...
