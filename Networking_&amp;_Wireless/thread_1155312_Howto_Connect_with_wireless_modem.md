Finally got a wireless broadband connection on Jaunty: 
To connect to Three prepaid:
run pppconfig as root,
User is "a"
Password: "a"
Speed 115,200
Com      /dev/ttyUSB0
Method   PAP
You should be able to connect by right-clicking the network manager icon near the top right of screen. click "Three prepaid".
Good Luck
