### Post by noob12 on 2007-09-08
OK.  Here is something to keep in mind.

Most of these devices available today are gateway routers themselves; they "expect" to be hooked up direct to a modem without an intervening router; this means that the default setups and instruction guides are oriented towards that.  It is usually possible to use these devices as access points, but you need to understand your network a bit better to do that.

It will in all cases be important to understand whether your Speedtouch modem also has a gateway router built in or not.  If it does, it suggests that you should either get an access point device (not a gateway router) or that you will need to do one of these somewhat non-standard setups of a wireless gateway router to behave as an access point.
