### Post by cariboo on 2008-08-17
It really shouldn't make any difference if you have a static ip address or use dhcp. As far as checking your transfer speeds, you could use iptraf to monitor your connection in real time. Iptraf is available in the repositories and you can install it using Add/Remove Programs, Synaptic Package Manager and of course the command line.
