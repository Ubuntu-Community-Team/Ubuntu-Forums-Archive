### Post by ibutho on 2008-07-16
> Hi all, hopefully you'll be able to help me. I've just installed Ubuntu onto my PC which originally had windows on it. now I'm trying to get it hooked up to to the net, but from what I've heard you can't get dial-up to work with Ubuntu, is this true?
Not true. If you are using a proper pci hardware modem, an external serial or usb modem, then you should be able to use dial up. If you are using a software modem (winmodem) then chances of getting that to work with non Windows OSes can be impossible. You can check at [Linmodems]("http://www.linmodems.org") to see if your modem is supported.
