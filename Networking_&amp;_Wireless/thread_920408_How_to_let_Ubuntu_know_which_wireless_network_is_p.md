### Post by PocketDog on 2008-09-21
I don't think you can in roaming mode. Left click the network manager icon > manual configuration > wireless connection > properties. (You'll have to unlock and enter your password to do this bit). Untick roaming mode and put in your router's ESSID. Leave connection type as automatic (DHCP). Click ok. Give the location a name and click the save icon. It should connect to this network automatically in future.
