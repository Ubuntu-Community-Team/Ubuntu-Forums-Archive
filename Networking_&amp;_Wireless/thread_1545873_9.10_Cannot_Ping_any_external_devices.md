### Post by cjames1968 on 2010-08-05
Solved.  Went back to what had changed which was the router Cisco WRVS4400N.  Looked at other threads and saw the comments about NOT using mixed-mode security.  Played aroud with security, etc.  Finally chose 'B' for network on router.  Security Mode WPA2-Personal (NOT Mixed!)  Connection came up and has stayed up.  

Behavior was odd initially b/c Ubuntu box would connect, but only for a brief second.
