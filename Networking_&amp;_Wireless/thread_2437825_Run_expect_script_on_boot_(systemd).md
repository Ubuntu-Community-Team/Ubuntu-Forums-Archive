### Post by TheFu on 2020-03-01
What is this?  /root/surfshark.sh

Usually, scripts that work from an interactive session but don't work as cronjobs or other startup scripts have 2 issues:
* minimal environment - add a line to the script that shows the **env** to see what's missing.
* not specifying the full path to all programs, config files, etc.  Never trust the PATH or use relative paths. Always use absolute paths.
* assuming a CWD/PWD.
