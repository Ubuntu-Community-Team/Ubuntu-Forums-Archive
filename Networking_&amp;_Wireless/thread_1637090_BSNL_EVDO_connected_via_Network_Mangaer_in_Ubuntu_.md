### Post by arundracula on 2010-12-04
At last BSNL EVDO is working with Ubuntu 10.10.
I have been using Ubuntu since 9.10 and my BSNL EVDO usb modem was not working till today through Network Manger. 
Today I've upgraded to 10.10 from 10.04 and it is working. Simply I created the account details and Selected Automatic PPP Addresses Only and put DNS addresses.
Logged out and In and it is working.. Thanks for the people behind this..

I can now use Ubuntu One/Gwibber etc.
I posted this if anyone searches for this problem they can upgrade to 10.10 to make it working..
