### Post by ibates on 2012-01-21
I am running 10.04 LTS updated,  and have a successful WiFi local network.  I have recently installed a Netgear ReadyNAS network "server".  The Ubuntu computer is connected to the ReadyNAS by ethernet, but all other computers on the network connect to it by WiFi.

Everything is operationally 100% however a weird problem has emerged.

When opening my pictures on the ReadyNAS drives, I can only see icons of the photograph files, and all my previous preview thumbnails are no longer available.  This is inconvenient because to find a particular photo one must open each one in the folder (sometimes hundreds) in order to find the one you want, instead of being able to scan the thumbnails to find the pic.

What is frustrating is that when opening the same folders on the same ReadyNAS device in Windows XP through the same network on the same ethernet, bingo!  There are the thumbnails.

To make it even more annoying, when I open the folders in Windows XP running as a guest in VirtualBox hosted on the Ubuntu OS, once again, there are the thumbnails.

Ubuntu is the only OS which is not showing the thumbnails.  Even my Mac, opening the files via WiFi using AFP gives me thumbnails.

So it seems that this inconvenience exists only in Ubuntu.

Any thoughts as to what can be the problem and how to overcome it.
