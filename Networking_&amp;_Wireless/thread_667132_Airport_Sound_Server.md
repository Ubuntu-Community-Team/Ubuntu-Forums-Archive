pre-7 versions onf iTunes allow this: their iTunes shares can be accessed by Rhythmbox' DAAPD-Plugin on Ubuntu. Newer versions allow browsing of iTunes' shared music - but you can't access it. The other way round of course, it works. Music shared by rhythmbox' DAAPD-Plugin can be accessed from iTunes.
