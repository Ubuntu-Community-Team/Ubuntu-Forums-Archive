### Post by SeijiSensei on 2017-02-25
The IP address doesn't match the network and broadcast.  If the machine has 192.168.1.104 for its address, the network needs to be 192.168.1.0 and the broadcast address 192.168.1.255 with a netmask of 255.255.255.0.
