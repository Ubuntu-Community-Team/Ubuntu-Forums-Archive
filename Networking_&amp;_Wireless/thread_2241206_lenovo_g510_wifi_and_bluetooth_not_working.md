### Post by Vladlenin5000 on 2014-08-25
The Lenovo support drivers download page provides a single Windows installation file for the WiFi but mentions *Atheros, Broadcom, Realtek*... This means the Windows exe will run a script that detects the WiFi device and install the proper driver. In Linux you need to know what your hardware is because the solution almost certainly is hardware specific.
