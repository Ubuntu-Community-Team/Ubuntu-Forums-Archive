### Post by HologramX on 2006-10-20
This is probably obvious but I had some trouble with this trying to set up wireless networking with a Rt2500 card on Ubuntu for the first time so I thought I'd point it out in case it hasn't been before!

The ESSID and key are both case-sensitive so you need to enter them exactly as you have them set up on your access-point.
