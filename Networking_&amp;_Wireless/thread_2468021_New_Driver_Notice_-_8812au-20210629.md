### Post by morrownr on 2021-10-16
News: A new version of the Realtek driver for the 8812au chipset for USB WiFi adapters is now available. Testing has shown it to be an improved driver over earlier versions of the driver. Managed, master and monitor modes perform well and performance is good. Monitor mode supports packet injection for those involved in security analysis.
