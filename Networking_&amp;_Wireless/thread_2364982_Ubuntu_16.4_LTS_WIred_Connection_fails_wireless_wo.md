Please note that the notions of "slow" or "fast" are extremely subjective if you don't prove a certain speed by measuring. It depends on bandwidth, number of programs and computers using the network, type of data transmitted, signal strength, obstructions, interference, etc.

Please give us as much information as you can. What have you tried? What guides have you followed? What do you think the problem is? How experienced are you in Ubuntu?

Just saying "it doesn't work" is not helpful. Please see the sticky thread on how to start troubleshooting [https://ubuntuforums.org/showthread.php?t=370108](https://ubuntuforums.org/showthread.php?t=370108)

The name of the game is drivers. Many network cards are intended for Windows, and thus have drivers intended for this operating system. This is not a problem with the Linux community, but rather with the hardware manufacturers who ignore Linux users by not releasing drivers for their equipment. In many cases you can install better drivers, and suddenly you see an improvement. Other times there is just no good driver in Linux, and thus the hardware remains with problems throughout its entire life.
