### Post by rfkrocktk on 2009-07-02
I just changed my user password using System -> Administration -> Users and Groups. Now, whenever I log in to my desktop, I get an annoying popup that is basically saying that I need to grant access to NetworkManager to the keyring. It only works if I input my old password, the one I had before I changed it. How can I make this go away?

---

### Post by Boondoklife on 2009-07-02
press Alt+F2 and run seahorse, from there goto the passwords tab, right click on passwords and change it to match your new one.
