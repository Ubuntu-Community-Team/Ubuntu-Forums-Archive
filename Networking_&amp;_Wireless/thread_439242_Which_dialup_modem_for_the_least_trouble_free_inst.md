### Post by phidia on 2007-05-10
IMO external modems are the best. So if they have a serial port in back and don't mind the extra box that's the way to go. If you want to use an internal pci modem USR does make a controller type modem. One hint if you're shopping online is that winmodem (some can be made to work) are very cheap but modems with controllers are usually not.
I don't know if there is any support for isa slot modems but you can check at [www.linmodems.org](www.linmodems.org)
