### Post by ohayle on 2008-06-10
Can you provide some detail on you wireless card. You should see if you have Ndiswrapper installed, check synaptic for installed programs or use "sudo ndiswrapper -l" on the command line.
    If you have the Ubuntu installed you should have a help file available in Firefox , look under wireless connections. The help file will walk you through the process, you will need the Windows version of your *.inf and *.sys for you wireless card, available from you installation media.
