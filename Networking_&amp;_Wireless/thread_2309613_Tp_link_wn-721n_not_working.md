### Post by Bucky Ball on 2016-01-12
You might get Lubuntu running on that. Give it a try. If you create install media, USB or CD/DVD, you can boot from that and choose 'Try Lubuntu' at the options screen. That will let you try it out without changing anything on the hard disk. See how it runs. 

Simple fact of the matter is you are going to get little to no help with a release that old so you need to look for an alternative. If you could throw another Gb or two of RAM into the machine you could run Xubuntu or Lubuntu without an issue. The processor is not an issue (I have Xubuntu running with a P4 on my desktop, but it has 4Gb of RAM). 

PS: 9.04 has not had updates, and that includes security updates, for over five years. Probably not a good idea to get that machine online even if you could.
