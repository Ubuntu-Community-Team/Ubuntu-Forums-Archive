### Post by ajgreeny on 2014-10-15
I think on a new install of Lubuntu there is no network icon in the panel; it's a bug if I remember correctly.

If that is your problem you must make sure that you have the system-tray in your panel and that network is included in the autostart applications which you get to from **Preferences->Default applications for LXSession->Autostart tab**;just make sure there is a tick mark in the network item.
