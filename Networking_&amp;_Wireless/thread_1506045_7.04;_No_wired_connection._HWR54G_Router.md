OK back to basics, and a sanity check
 
1. Check the actual wire between the machine and the switch.
2. Check the lights on the connector on the switch and the machine, are they both green, do they change when you unplug the cable?
3. When you run the LiveCD do you get an internet connection?
4. the command -
```
lshw -C network
```
should identify the network device you are using - does it?
 
Are you able to get a more upto date version of Ubuntu downloaded?
