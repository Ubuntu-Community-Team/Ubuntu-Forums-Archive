This is probably not help, but as a general rule, you should avoid usb modems with Ubuntu.  Connecting to modems with an ethernet port is much easier.

However, if you enter "usb modem" into the Ubuntu Forums search box, you will see there are a few discussion threads dealing with same problem and it seems that some people have been able to get their usb modems working.
