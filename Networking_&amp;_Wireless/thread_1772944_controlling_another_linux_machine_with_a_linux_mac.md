### Post by RichardG891 on 2011-06-01
For cross-platform (i.e. seeing a linux desktop remotely on a windows machine) I've found NX terminal server OK.... it's free for personal use, but the server side (the computer you want to control, not the one you're sat at) is only free for Linux. See below for details.
[http://nomachine.com/]("http://nomachine.com/")

As for streaming video to a computer connected to your TV, you just need to share folders to the network via Samba (should be pre-installed). Just right-click on the folder you want to share and select "sharing options".
