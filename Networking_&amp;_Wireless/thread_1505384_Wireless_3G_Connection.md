### Post by pdc on 2010-06-09
plug it in;
hover mouse over network manager; (top-line menu bar and is left-most of the right-hand items;

RIGHT-CLICK; select EDIT CONNECTIONS;

select MOBILE BROADBAND;

ADD;

select country; AUSTRALIA ?

network ; telstra?

next screen: check you have the correct plan selected: do you see the option to select plan?

once done, apply and close;

now LEFT-CLICK on NM;

you should see your telstra entry; click; should connect

no worries
