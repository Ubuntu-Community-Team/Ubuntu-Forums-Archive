### Post by shredder12 on 2011-03-13
It could be an IPv6 issue. The system tries IPv6 before IPv4 and hence the lag. Use the steps mentioned [here]("http://www.webupd8.org/2010/05/how-to-disable-ipv6-in-ubuntu-1004.html") to turn off IPv6 in Ubuntu 
One more thing, disable it in firefox too.  Type "about:config" in the URL bar and hit enter. Search for
```
network.dns.disableIPv6
``` and set it to true.
