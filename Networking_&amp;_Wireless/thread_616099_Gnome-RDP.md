### Post by jshurst on 2007-11-20
Is this a home network?  If so then you probably are not running a domain.  Usually you will put the name of the computer (in windows it is like this  "MyComputer\username"  - where "MyComputer" would be the name of your computer or it would be the domain name if you were on a domain.

If right click on My Computer and go to properties then go to Computer Name then you should be able to see the name and domain if there is one (also you might try putting "WORKGROUP").
