### Post by jetsam on 2008-04-04
Your plan is a good one.  There isn't any server software that can't be installed on the desktop versions.

Servers exposed on the web and in professional contexts need to be optimized and secured by running only the few services they are designated to run.  So the server edition is a minimal ubuntu system that comes without a GUI and with just enough services to run linux and install more software.  After installing, they get customized to serve their designated function and nothing else.

Since you're still exploring, you'll be much better off with a desktop edition.  You can later uninstall stuff if you want to designate it as a server only, or leave the desktop on and use it as a mixed use machine if you don't run into any performance trouble.  You probably won't on a small home LAN.

I started off using Edgy I think, just to see what this 'linux' thing was all about.

I first set up a simple mp3 server, so I could listen to my tunes anywhere. (Gnump3 I think it's called)
