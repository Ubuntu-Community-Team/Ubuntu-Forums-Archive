### Post by Iowan on 2008-03-08
Under the "Manual Network configuration" icon in  upper right corner (or System>Administration>Network) should be a list of network setting options. Mine has only  "Wired connection" and "Modem connection" - yours should have a wireless option, too.  You should be able to set up the wired network from there.  I usually tweak settings in **/etc/network/interfaces** directly - that's another option.
