### Post by rad_sci_guy on 2008-03-27
Hello everyone

I have a acer aspire 5040 laptop which has Ubuntu 7.10 installed.  I installed the acer_acpi controller software to activate my wireless card (atheros).  The card gets activated each time I turn on the laptop (the wifi light always comes on).  The problem I'm having is that when the latop is initially turned on, the wireless connection does not appear in the Network Manager.  I cannot connect to the internet, even though the wireless card is activated.  If I then restart the computer (not full shutdown), the on the following start up I have access to my wireless internet.  This is the only way I can get Ubuntu to access the wireless connection.

I've seen others post this problem as well, however I've never been able to find a solution in the forums so that I don't have to always reboot twice.  From the other post in the forum it looks as though Ubuntu is searching for a wireless device before the wireless card is turned on.  I figure if I could move the wireless card turn on sequence up in the start up or If Ubuntu were to able to search for a wireless device a second time, then the card would be detected on first computer start and I wouldn't have to reboot each time.

Does anyone know how I can make either of the 2 changes I've mentioned above?  I've looked on other forums as well and am unable to find a solution to this issue.

On a side note I also have a thinkpad x30 notebook with Ubuntu 7.10 install on it as well.  This laptop is 5 years old and doesn't have a wifi card build in.  I purchased a D-link WNA-1330 PCMCIA card which worked out of the box and didn't require any setup except to install a software modem driver from the restricted drivers tab.  This card works really well and I would recommend it to others who are trying to get Ubuntu to work wirelessly on a laptop without a built in wifi adapter.
