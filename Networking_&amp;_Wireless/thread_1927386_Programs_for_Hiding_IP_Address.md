### Post by tomsn2 on 2012-02-17
It is in beta at the moment, but [www.torproject.org](www.torproject.org)


It will only change the IP in the tor browser though unless you make a proxy server through the local machine.  I'm pretty new to Linux so I can't tell you how to do that on Ubuntu.  Just FYI, if you set up Tor using a proxy you must have Tor open for the internet to work since it is using the proxy to route all traffic through Tor.

Just be sure to read the site's TOS carefully to make sure they don't specifically prohibit proxies or otherwise hiding your IP address, or using a false address.  If they do then it's technically unauthorised access to a computer network, which is technically a felony.  They don't often chase after people for it on social sites, but they could if somebody took a disliking to you.
