### Post by jolan on 2013-04-03
It's actualy quite unbelievable, but I tried to connect to a WiFi network and accidentaly entered a wrong password. I also had the "remember password" option checked. Now it's impossible to change the password of the password. It's quite ridiculous because:
1. I can only change settings for the network, to which I'm connected.
2. Since I put in a wrong password, I can't connect to the network, thus can't change the options of the network. And the system doesn't ask me for the password, since it's stored in the system.

Is there any way to get out of this mad circle?

---

### Post by coldraven on 2013-04-03
Press the windows (super) key to bring up the dash, start typing "network" and then click on "Network connections".
Go to the wireless tab and edit the password.
