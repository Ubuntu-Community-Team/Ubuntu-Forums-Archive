### Post by greenstar on 2007-10-17
Yes, it saves the settings. Firestarter isn't the firewall itself, iptables is the actual firewall, as you know. Firestarter is only a GUI tool for configuring iptables. It doesn't need to run in the tray. It took me a while to get used to that when I first started using Firestarter.
