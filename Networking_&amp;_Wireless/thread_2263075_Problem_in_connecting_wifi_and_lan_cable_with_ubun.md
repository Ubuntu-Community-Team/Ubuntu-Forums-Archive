### Post by SeijiSensei on 2015-01-29
Are you trying to connect both the wifi and the Ethernet to the same router?  Just pick one or the other.  There are no performance gains to be had using both unless you use an advanced technology called "bonding."  If anything, connecting both can degrade performance because of routing issues.  If both connections are equally convenient, use the Ethernet.

If you are using the Ethernet port to connect to a *separate network* and want to use the machine as a router, that's a more complicated issue.
