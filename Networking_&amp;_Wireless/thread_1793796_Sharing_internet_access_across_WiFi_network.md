### Post by helgman on 2011-06-30
Try to find information regarding:

1) Setting up the computer as a wireless access point or how to use wireless ad-hoc mode

Google: [http://www.google.com/search?q=ubuntu+as+wireless+ap](http://www.google.com/search?q=ubuntu+as+wireless+ap)

Nice result: [https://help.ubuntu.com/community/WifiDocs/WirelessAccessPoint](https://help.ubuntu.com/community/WifiDocs/WirelessAccessPoint)

2) Find out how to "share internet connection" in Ubuntu

Google: [http://www.google.com/search?q=ubuntu+share+internet+connection](http://www.google.com/search?q=ubuntu+share+internet+connection)

Nice result: [https://help.ubuntu.com/community/Internet/ConnectionSharing](https://help.ubuntu.com/community/Internet/ConnectionSharing)

Just remember that you'll need static IP or a DHCP server if you're not bridging between the wired and the wireless interface (then you have a number 3 that you'll have to research).

Check those links (and maybe something more from the Google-results) and let us know if you have any questions or run into any problems.

(I promise to check back so please tell if it all works out as well.)
