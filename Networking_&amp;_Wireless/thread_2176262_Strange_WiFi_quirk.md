### Post by varunendra on 2013-09-24
If I remember correctly, it is the default behaviour in Power Management that the system goes to suspend as soon as the lid is closed. Unless I had changed settings earlier and forgot, I had to change this behaviour manually.

Please goto System Settings > Power, and make sure the action on "When the lid is closed" is set to "Do nothing" as shown below -
