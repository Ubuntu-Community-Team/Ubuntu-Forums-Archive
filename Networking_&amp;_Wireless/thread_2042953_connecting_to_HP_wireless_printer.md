### Post by ajgreeny on 2012-08-15
Open the HP toolbox, go to Device ->Setup Device, and then choose the middle of the three options available **Network/Ethernet/Wireless network (direct connection or JetDirect**

You should then be able to use the Select from detected devices page.  You need to have a static IP address for the printer, which you can probably setup through your router's setup webpage.

### Post by irv on 2012-08-15
I just setup my HP printer from the printer setup in settings.
Just select Network printer and let it find the printer. You need to have it setup on your network before hand with an ip address.
