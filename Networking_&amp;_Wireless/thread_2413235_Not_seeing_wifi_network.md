### Post by lathamio on 2019-02-23
Hello,

Have you tried to use Network Settings to login to the WiFi network of your choice?

1. Navigate to Settings.
2. Click Wifi Category.
3. Next to the Wifi On/Off Switch will be a "Hamburger" or three line menu, click on the menu.
4. Check to see if Known WiFi Networks contain the WiFi Network you need.
5. If the network is not shown, try to connect to a "Hidden" WiFi Network using your SSID and password, using the "Hamburger" menu.
