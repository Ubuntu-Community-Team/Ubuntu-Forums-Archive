### Post by jeremy31 on 2017-11-12
```
echo "options iwlwifi 11n_disable=8" | sudo tee /etc/modprobe.d/iwlwopt.conf
```
```
sudo modprobe -r iwlwifi && sleep 20 && sudo modprobe iwlwifi
```
```
sudo sed -i 's/wifi.powersave = 3/wifi.powersave = 2/' /etc/NetworkManager/conf.d/default-wifi-powersave-on.conf
```
```
systemctl restart network-manager.service
```

The first 2 codes enable aggressive TX on the wifi and the last 2 turn off wifi power management
