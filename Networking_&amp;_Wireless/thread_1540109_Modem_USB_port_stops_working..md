Hi, I have a D-Link DSL modem, with an ethernet and a USB ports. Both ports are working perfectly with Ubuntu, until a few days ago when the USB port suddenly stops working. I can still connect to the internet using the ethernet connection, however I need both of them working as the modem is shared between 2 computers.

The USB cable is perfect, I can print using the cable.
The USB light in the modem is on.
If the cable is connected to the modem, lsusb does not return anything (it works with other devices).

The setup has been working perfectly for a few months. I wonder whether the modem is broken or the software is broken after an update. Any help will be greatly appreciated, thank you.

Solved. The cable is somewhat broken. It works on the printer, but not always. Changed the cable and everything is fine.
