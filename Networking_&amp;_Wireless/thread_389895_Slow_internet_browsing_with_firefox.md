### Post by k1w1 on 2007-03-22
If you are still using Firefox, try disabling IPv6.

Press Alt+C or type **about:config** in the address bar.
In Filter bar type in **IPv6**.
Set network.dns.disableIPv6 to true, by right click then toggle.

Has worked for me in the past, although mainly on dial-up.
