### Post by gfunkera on 2012-05-29
My W500 Lenovo has the following wireless network adapter (obtained by doing lspci):
```

Network controller: Intel Corporation Ultimate N WiFi Link 5300
```

I initially thought that this machine would have broadcom drivers and attempted doing the b43-fwcutter procedure that most of my other machines needed in order to get the wireless working. I remember installing 11.10 on this machine and found out that i had to disable the "N" by doing:

```
sudo rmmod iwlagn
sudo modprobe iwlagn 11n_disable=1
```

the problem is that when I run the first command, it says:

```
ERROR: Module iwlagn does not exist in /proc/modules
```

Im goin nuts tryna get this thing to work.. anyone got any ideas?

### Post by praseodym on 2012-05-29
Hi,

from Ubuntu 12.04 the module is renamed to "iwlwifi":

```
echo "options iwlwifi 11n_disable=1" | sudo tee /etc/modprobe.d/iwlwifi.conf
sudo modprobe -rfv iwlwifi
sudo modprobe -v iwlwifi
```
