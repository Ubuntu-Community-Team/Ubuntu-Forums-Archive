### Post by Holger_Gehrke on 2017-12-27
ActiveX is a proprietary MS thing. Even on Windows Firefox could (past tense; the addon needed to do this used an API which has been deprecated) only use ActiveX by embedding IE. On Linux ActiveX does not exist. There are some reports about getting ActiveX to work by using WINE and running IE in WINE (which is not easy to do, there's some seriously tricky stuff involved).
