### Post by huygens on 2007-04-14
Hello,
To share printers and files, a possibility (as it is supported by both Windows and Linux - and even Mac OS X) is to set up Samba under Linux. This will make your Linux look like a Windows computer on the network. You can then just exchange files or use the printer much more like you were doing it with Windows.

There a few wiki articles, here I give you a few of them:[LIST]
[*][http://doc.gwos.org/index.php/Category:Servers](http://doc.gwos.org/index.php/Category:Servers) check the 2 articles named "[Samba print server quick and dirty]("http://doc.gwos.org/index.php/Samba_print_server_quick_and_dirty")" and "[Share files using Samba]("http://doc.gwos.org/index.php/Share_files_using_Samba")".
[*]The official Ubuntu documentation: [https://help.ubuntu.com/6.10/ubuntu/serverguide/C/index.html](https://help.ubuntu.com/6.10/ubuntu/serverguide/C/index.html) check chapter 5.[/LIST]Perhaps, those articles are a little bit too technical (and using a lot the command line). If you prefer to set this up using graphical user interface (GUI), you could check this article: [http://www.berthon.eu/ice_and_fire/?p=9](http://www.berthon.eu/ice_and_fire/?p=9)
I also duplicate a post I had written on this forums to help a new user setting up his Samba, I'm perhaps explaining more in this article: [http://www.berthon.eu/wiki/foss:wikishelf#samba](http://www.berthon.eu/wiki/foss:wikishelf#samba)

I hope this will help you start :-)
If you have further questions do not hesitate to post them back here.
