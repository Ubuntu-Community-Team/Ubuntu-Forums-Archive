### Post by acan2 on 2014-06-09
It looks there is a problem with keys. Is it OpenVPN connection? Try to invoke this connection directly from shell:
1. Check what config files do You have in NetworkManager:
adam@laptop:~$ ls -l /etc/NetworkManager/system-connections/
-rw------- 1 root root 277 cze  3 09:54 WIFI
-rw------- 1 root root 482 cze  9 15:32 lk-vpn

2. If Your vpn connection is OpenVPN (as mine), I would do:
adam@laptop:~$ sudo openvpn /etc/NetworkManager/system-connections/lk-vpn

and look for errors.

3. If You will see there is a problem with keys/certs, check if root can access key and cert files (paths to that files shown in NM vpn config file)
