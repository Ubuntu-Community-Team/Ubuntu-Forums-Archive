There seems to be a problem with ndiswrapper on Ubuntu 9.04. Finally I don't think the problem is due to my router (Linksys WRT54G) or my wireless network controller (RaLink RT2561/RT61 802.11g PCI) because with Windows XP on the same machine I don't have this problem.

Hope this rings a bell to anyone who can give me some directions of what I need to do to resolve this issue. Any help would be greatly appreciated!
