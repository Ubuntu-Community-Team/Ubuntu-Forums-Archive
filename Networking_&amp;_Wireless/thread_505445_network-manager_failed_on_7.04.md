### Post by Kralizec on 2007-07-20
Hello,

When you have network manager uninstalled, do you still fail to shutdown? You could try finding an alternative to network manager if it is indeed the problem, or you could try a different version: [http://linuxappfinder.com/package/network-manager]("http://linuxappfinder.com/package/network-manager") has a list of deb repositories. If you only use network manager for wifi connections, an alternative is WiFi Radar, available through Add/Remove Programs.
