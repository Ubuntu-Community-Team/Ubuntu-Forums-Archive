### Post by gadgetman1 on 2009-05-23
This is resolved!!

Just FYI, if you install vsftp or any other module that need to use /var/run/<module dir>, which is the default for vsftpd, you may run to similar issues and according to some documentation on the web this is not a good location to use.
I just created a directory (/var/vsftpd) and changed the vsftpd.conf file to use this directory instead of /var/run/vsftpd.
