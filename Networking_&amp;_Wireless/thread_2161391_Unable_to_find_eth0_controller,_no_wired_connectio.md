### Post by praseodym on 2013-07-10
Try to reset the BIOS to default settings. Check:
```

lspci -nnk | grep -iA2 net
cat /etc/udev/rules.d/70-persistent-net.rules
ifconfig -a
cat /etc/resolv.conf
lsmod
```
