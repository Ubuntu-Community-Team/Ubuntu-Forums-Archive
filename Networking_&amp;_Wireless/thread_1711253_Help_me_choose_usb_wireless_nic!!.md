### Post by walt.smith1960 on 2011-03-21
I've used a TrendNet 424UB rel. 3.1 since starting to use Ubuntu.  It gets some poor reviews on Amazon but I've had very good success with mine.  It uses a RealTek 8187B chip and does connect out of the box with any recent Linux release, Windows XP requires a driver.  I'm using a Netgear 1100WNA but that requires a little work in Maverick-atheros 9K installer from Sourceforge.  The Netgear works out of the box with Natty A3.
