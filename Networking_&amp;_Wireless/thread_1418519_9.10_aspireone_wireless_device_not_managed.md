### Post by cchhrriiss121212 on 2010-03-05
Have you successfully used wireless with any other ubuntu versions with this machine? A live cd of 9.10 desktop would be a good start, if you have not tried it. 
Wireless is not recommended if this is primarily a music machine for the reasons I mentioned. In order to use wireless internet without interference, you need to disable it and reboot every time you want to work on something.  

Just noticed that you are using a generic kernel which means this is a plain install right? You might find this useful then: [http://ubuntuforums.org/showthread.php?t=1309605&highlight=AR5001](http://ubuntuforums.org/showthread.php?t=1309605&highlight=AR5001)
