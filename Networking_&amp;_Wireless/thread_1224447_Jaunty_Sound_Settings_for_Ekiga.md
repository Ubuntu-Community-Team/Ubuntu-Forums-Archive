When you right-click on the small speaker icon in the tray at the top of the screen and then click on Open Volume Control, you are presented with four options for each sound device: i) device, ii) capture device iii)capture monitor of device and iv) playback device.  Which do I leave working and which do I mute when using Ekiga?

The main Sound settings (at the large speaker icon) that I use for Audio Conferencing are:
input--my usb mike and
output--PulseAudio Sound Server.
These main sound settings are the only ones with which I have had intermittent luck with the Ekiga echo test.

Within Ekiga, I leave the sound devices set to the named devices used rather than using any default settings.
