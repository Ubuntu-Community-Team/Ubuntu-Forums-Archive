### Post by spd106 on 2007-09-13
You said in [another thread]("http://ubuntuforums.org/showthread.php?t=549961") that you have an external Wifi button, is this set to on? 

If you run these commands does it say that the interface is disabled or that the radio is off?

```
sudo lshw -class network
```
```
iwconfig
```

Make sure that you have the multiverse and security repos enable in Synaptic/Software Sources. You can download the Flash installer from Adobe's website too. The instructions are inside the tarball.
