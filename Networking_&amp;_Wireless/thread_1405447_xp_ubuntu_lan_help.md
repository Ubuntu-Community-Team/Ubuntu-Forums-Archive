### Post by Iowan on 2010-02-12
Without a hub or switch, you will need a crossover cable (although some 1000M cards can auto-detect). XP (pro) has the connection sharing option (though I don't know how to set it up). You will need to set the Ubuntu machine in the same subnet and use the XP machine as the gateway.
