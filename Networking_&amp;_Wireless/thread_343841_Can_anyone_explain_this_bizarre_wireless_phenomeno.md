### Post by tturrisi on 2007-01-22
things to t-shoot the xp comp:
1. only use the windows wireless zero for wifi connections, don't use 3rd party apps that come w/ the wifi adapter.  This done by viewing the wifi connection properties and putting a check in box "let windows manage my wifi connection".  Some 3rd party apps may do aggressive scanning  for available wlans in the background.
2. folder options > view tab > uncheck "automatically search for network folders & printers".  This function sends out regular queries to the lan and failures to get responses will result in more  queries.
3. clean comp of spyware-malware; disable or pause unnecessary background legit processes and services that use the net.
