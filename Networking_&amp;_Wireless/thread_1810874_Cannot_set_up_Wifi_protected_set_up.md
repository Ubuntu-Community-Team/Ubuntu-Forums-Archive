### Post by thefasterblueone on 2011-07-23
You don't need a disk to set up your router. Just type 192.168.1.1 in your web browser address bar, hit enter, when you see the login box enter admin for user name and admin for password and click login button.


If you need a User Manual here is a link:

[User’s Manual Model # AR670W Wireless Router](http://www.airlink101.com/download/download_links/m-ar670w.pdf)

The manual explains how to setup router manually on page 21.
