### Post by Floppyjoe on 2007-04-01
In Edgy Eft you can click on System/Administration/Synaptic Package Manager.
Then click on Edit/Add CDrom and follow the directions using the Ubuntu Install Disk.
Then click reload.
Then search for ndiswrapper and install ndiswrapper-utils-1.8 and ndiswrapper-common.
