### Post by seismicmike on 2008-04-16
I work in an office where everyone else uses a program called "Message Pal". You've probably not heard of it, so here's some [screenshots]("http://images.google.com/images?q=messagepal+screenshots&ie=UTF-8&oe=utf-8&rls=org.mozilla:en-US:unofficial&client=firefox-a&um=1&sa=N&tab=wi").

MessagePal does not use a server, but still uses an instant message like interface. There is a list of all other users on the system (in reality all other computers on the system with MessagePal installed, but each computer is identified by the name of its primary user) and you can message that person individually (like AIM). You can also set up groups and send messages to all online users.

I have seen others suggested including Trix and glanchat, but they provide an IRC like interface and a group chatroom. I need to see everyone else on the system and be able to send and receive messages between specific users.

Thanks.

PS: Yes I have tried to wine it, and it works as far as giving me the window with the list of everyone online, but that's as far as it goes. For much of the functionality it opens new windows (sending/receiving messages etc). I tried to send a message using the wine version and the program crashed when it tried to spawn the new window. No dice.
