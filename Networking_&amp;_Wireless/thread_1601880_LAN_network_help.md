### Post by Iowan on 2010-10-20
Depending on what you want to do...
Will machines connect directly to each other, or use a hub/switch/router?
Will you want to share files? Samba client is pre-installed - Ubuntu *should* be able to see Windows shares, but to share from Ubuntu to Windows will require the Samba server to be installed.
