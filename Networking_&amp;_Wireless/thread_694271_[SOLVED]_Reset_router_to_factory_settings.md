### Post by linuxmagick on 2008-02-12
When you press the reset button on the router, try holding it down for about 10 to 15 seconds before releasing. The router will power off for a moment and then back on.  After that it should be set to the default IP address and the default username and password (probably admin with no password or admin with password of admin) or something along those lines.
