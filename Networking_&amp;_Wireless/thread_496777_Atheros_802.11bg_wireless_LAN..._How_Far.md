### Post by t4thfavor on 2007-07-09
Thats a pretty broad question concidering all of the variables. 

First of all you need to know the chipset you are using. Atheros has quite a few that they are using today (52XX, etc. different chips different ranges). Three miles is kinda far, but easily doable if the correct conditions exist. 

First of all you will need a clear line of sight. (Nothing in the way!!)
Then you go get some sort of directional antenna for each end of the link.  
Set up your end devices at a close range so that you know they can talk to each other, and then do the same using the directional antenna.

This is an extremely general aproach to this question, and It most likely will not work (legally).

But it will get you started. 

If you have any more specific questions feel free to ask them.
