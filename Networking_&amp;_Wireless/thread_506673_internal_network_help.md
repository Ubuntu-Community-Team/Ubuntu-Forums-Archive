### Post by lisati on 2007-07-22
Place the file in a "shared" folder on the XP machine. On the Ubuntu machine, go to Places->Network and click on the XP machine's icon and then open the shared folder. You should then be able to copy the file from the XP machine to the Ubuntu machine.
