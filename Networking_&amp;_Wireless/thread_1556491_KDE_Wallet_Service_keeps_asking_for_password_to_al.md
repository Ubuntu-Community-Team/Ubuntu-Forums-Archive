### Post by grahammechanical on 2010-08-23
Is your system set for automatic login? If so then for security reasons wireless and network require authentication. I am using gnome. You can edit the network connections and mark each connection to connect automatically and/or to be available to all users. Also under Users and Groups you can allow or disallow users from accessing networks and the internet. I think that network manager over-rides this. In this way you can control not only who can use your computer but also what they have access to. 

If you set the system to require a log-in this need for authentication is removed and if your network connections are set to automatically connect then you are not asked for authentication as the keyring or wallet service has been unlocked.

I am not an expert. I am just passing what I have learnt. I may be wrong.
