Hi,

first I'd like to say that I'm very impressed by the very simple and user-friendly installation (this was my first Ubuntu).

I used the Gutsy Gibbon, and ran into a small but cumbersome problem with my wireless network:

I use WEP with 128-bit encryption. Being a bit suspicious about generating keys from ascii strings, I decided to punch it in as a hex number.

And used lowercase letters.

The UI doesn't state that uppercase letters are required, neither does it complain when encountering lowercase letters in the key. (Like e.g. a1b2c4...)

So it took me a while to get to the point of trying uppercase letters...

So if anyone else got the same problem (and some other OS/computer to surf here on), you can always try uppercase letters (Like e.g. A1B2C4...)

Maybe this is actually to be considered a bug, but I haven't got the energy to figure out how to submit a bug report...
