### Post by t0mppa on 2009-10-07
You'd probably be best off installing a Linux distribution that's specifically built for this kind of use, unless you really want to spend a few days studying iptables, etc. to be sure to make the machine secure.

[Distrowatch]("http://distrowatch.com/dwres.php?resource=firewalls") keeps a pretty good list on these. Personally I have an old 486 running IPCop and that was very simple to install & setup and can be configured through a web admin interface, but there's plenty of other options too.
