### Post by SoftwareExplorer on 2009-09-06
It wants to remember your wep key for you, but it wants to password protect it. It therefore saves it in the keyring, a place to store multiple passwords protected by a single password. If you already set up a keyring password, it is asking for it. 
Otherwise, it might be having you set up a new password, but it sounds more like you already put one in and it's asking you to enter it.
