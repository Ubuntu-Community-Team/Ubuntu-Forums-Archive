### Post by aesis05401 on 2009-08-24
Hello, 

You should not need any software to use your router.  My home LAN is all linksys, both wired and wireless.  I have not installed any special software. 

Linksys routers have a default IP address and administrator login.  The address should be something like 192.168.1.1, the username should be 'admin', and the password should be either 'admin' or nothing (just hit enter).  You should be able to put the address into your browser with the router powered up and see the admin login.

Be sure to change the password before doing anything else.  You will need to log back in after resetting the password.

Does this help?
