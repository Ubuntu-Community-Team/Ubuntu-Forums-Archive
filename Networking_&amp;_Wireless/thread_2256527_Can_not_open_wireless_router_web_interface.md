### Post by yc2 on 2015-01-04
If someone else finds this thread, I just report my findings. Two ways of trying to solve the problem are:

1. One reason the phone does not connect to a wifi router that is not connected to the Internet could be that the phone checks for an Internet connection. (If this does not exist,  the phone rejects to connect.) In some Android versions it is possible to disable this check. Not so in mine though, unfortunately.

2. To connect to the Web interface of a router (1) connected via another router (2) you can try to forward port 80 in router (2) to router (1).
