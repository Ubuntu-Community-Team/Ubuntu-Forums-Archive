### Post by CUrmudgeondoug on 2009-08-27
There are a lot of threads going on here about Network Manager repeatedly asking for the password while trying to log on to a wireless network in Ubuntu 9.04. It appears to replace the password with a long string of hex characters.

My fix was to reconfigure my wireless router from WPA2 to WPA.

Hope this helps.
