### Post by Drezard on 2008-10-05
A firewall is only necessary if you are running any servers such as Apache, MySQL or BIND. If you arent running any servers then a firewall is useless to you because all ports are closed by default. As Jmbargar said, make sure your system is up to date and you should be fine.
