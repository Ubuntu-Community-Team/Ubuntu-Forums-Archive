### Post by varunendra on 2014-02-16
You are currently using the sta driver for your internal card while the native one is usually better for the card you have. Accordingly, please purge the sta driver to try the native one -
```
sudo apt-get purge bcmwl-kernel-source
```
Reboot after this. The native driver (brcmsmac) will automatically load at next boot. See if it lets you connect and browse any better, I hope it should.

Apart from the above change in the driver, you should also change the encryption in your router/access-point. Currently it is using WPA/WPA2 mixed mode, and to make it even more weird, it is somehow using CCMP (AES) for both WPA and WPA2 parts. As per my knowledge, WPA essentially needs TKIP which should be avoided at all costs.

So.... regardless of the fact that your usb adapter is able to connect to the same router, be aware that your router is currently using one of the weirdest encryption settings. Change it to **pure WPA[COLOR="#006400"]2[/COLOR] with AES (CCMP)**. No mixed mode, no TKIP.

After saving the changes in the router, reboot it to make sure the settings take effect properly.

Try these two changes (driver + router encryption) and let us know the result.
