### Post by tturrisi on 2006-07-19
No wifi connection is ever secure!
All wlan traffic can be sniffed using Kismet or Ethereal.
So...all you can do is take reasonable precautions in public wlans such as;
- setup a new web based email account at yahoo or similar
- don't send-receive confidential data
- if send confidential data then encrypt & compress it
- no need to worry about viruses, etc.
- unshare any presently shared dirs
- avoid logging on as root
- if paranoid, keep private-confidential data on a usb pen drive or cdrw
