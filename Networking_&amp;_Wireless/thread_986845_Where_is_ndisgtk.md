### Post by aysiu on 2008-11-19
You have to enable the Universe repositories to get *ndisgtk*

Close Synaptic

Go to System > Administration > Software Sources

Check all the appropriate boxes

Reload when prompted

Go back to Synaptic and search for *ndisgtk*
