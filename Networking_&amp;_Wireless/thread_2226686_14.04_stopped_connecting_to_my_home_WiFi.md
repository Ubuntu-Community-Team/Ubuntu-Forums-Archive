### Post by Bucky Ball on 2014-05-28
Could you please open a terminal and paste in these commands, one at a time, please:

```
sudo lshw -C network
sudo iwconfig
sudo ifconfig

```
Post the results of each here.

Did the router switch to default settings when you switched it on again? Can you ping the router? (You need the router's IP for this.)

Presuming the router IP is 192.168.0.1, you would issue this in a terminal:

```
ping 192.168.0.1
```
