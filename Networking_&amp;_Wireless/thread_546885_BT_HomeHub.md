### Post by Steve1961 on 2007-09-11
81.155.244.98 doesn't look like an ip address from a bt homehub.  As the routers default address is always 192.168.1.254 your ip address should be 192.168.1.? unless you've changed things of course.  But even then you should have a non-routable ip address such as:

10.0.0.0 - 10.255.255.255
172.16.0.0 - 172.31.255.255
192.168.0.0 - 192.168.255.255

I'm not sure what you've done here.  Have you tried to setup a static ip by entering your external IP address in the network settings?  If so that won't work.  You need to access your router if possible and ensure the dhcp server is switched on.  In network settings you should then make sure everything is set to obtain an ip automatically.

### Post by noob12 on 2007-09-12
You must run /etc/init.d/networking restart as root (e.g. using sudo) as follows:
```

sudo /etc/init.d/networking restart

```
Then you won't get those errors.  But you've got other issues.

Is your windows host able to reach the internet that way?  I can't understand how that could work at all.  The address is 81.155.244.98 with subnet 255.0.0.0 and the default gateway is set to 217.41.165.122 which would not be on your subnet (!?) at all.  Meanwhile your DHCP server is at 192.168.1.254 which is a local private net.

That last address looks like your BT HomeHub, so what you've probably got is a jumble of odd settings there which are being provided to your local boxes from the DHCP server in the hub.  You might want to set things on the BT box back to factory defaults if it provides a way to do that, and reconfigure the box from that point.

When things are normal you should be getting offered an address starting with 192.168.1., your subnet mask would be 255.255.255.0 and gateway 192.168.1.254.

If you ever need to reset the router do this:

   1.  Push and hold the Wireless Association button (on the back panel) for 15 seconds, until all the lights light up - then release it.
   2. The reset process may then take several minutes.
   3. The Hub is reset and ready for use when the Broadband and Internet lights are steady green.
