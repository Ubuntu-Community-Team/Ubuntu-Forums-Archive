### Post by philipgm on 2008-02-21
Hi Everyone,

I just got this smartphone, and I've been trying to get it to connect my ubuntu box to the internet so that I can take it to work. Our network guys are worried about me having a box they don't understand on the network.

So far, when the phone is connected it appears as eth1, dmesg reports that it's an RNDIS device and that's about it. ifconfig gives an ip address for eth1 and route gives a gateway address on the same network. Looking at the ip address I can ping some other nodes in the network,  but I can't connect to them. I thought that my problem was not having a dns address so looking inside the mobile internet set up in the phone, I found two dns addresses and put these in to my network config - still no joy. 

I'm a bit stuck for ideas now - I've checked out these forums and the xda-developer forums and googled it - perhaps I'm not asking the right question. 


If anyone has an idea please let me know

TIA

Phil

### Post by philipgm on 2008-02-22
Done! 

It turns out to have been easier than I thought. There's this little program on the Orbit 2 that allows you to share the internet called Internet Sharing - open it and click connect and it just works...

I had been thinking about the usb modem from vodafone, 3 an T-mobile. No need now with my unlimited web bolt on! There is a fair usage limit, I'll have to find out what it is...
