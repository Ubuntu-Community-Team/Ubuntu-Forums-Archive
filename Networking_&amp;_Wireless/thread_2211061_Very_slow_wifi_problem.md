All the parts highlighted in Red above are points of prime concern.

**1)** Ubuntu 10.04 has reached its End of Life (EOL), means it is no more supported. You should upgrade to the currently supported LTS - 12.04.4

**2)** You are using WPA/WPA2 mixed mode in router, with TKIP algorithm. These should be avoided at all costs. Change the encryption to pure WPA2-PSK with AES (CCMP) in the router > Save the setting > Reboot the router. This alone may help dramatically improve the speed. Even if not, keep it as suggested.

**3)** Try changing the channel to 1 or 11. Usually they offer better signal quality. Besides, many of you neighbourhood access-points are using the same channel (6), which may be causing mutual interference.

**4)** If the change in the router doesn't help, try this -
```
echo "options ath9k nohwcrypt=1" | sudo tee /etc/modprobe.d/ath9k.conf
```
This will take effect after a reboot, or after manually reloading the driver as follows -
```
sudo modprobe -rv ath9k
sudo modprobe -v ath9k
```

Please try these for now and report back. And if it is not a problem for you, please consider doing a fresh install of 12.04.4 which would be supported till April 2017.
