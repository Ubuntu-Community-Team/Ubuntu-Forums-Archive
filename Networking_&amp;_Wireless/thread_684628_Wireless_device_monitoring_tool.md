### Post by tgalati4 on 2008-02-01
kismet with gps monitoring is pretty cool.  I put it on a powerbook  when I found that someone ported it to OS X (called kismac).  You can make some neat wireless coverage maps.  It's more difficult to set up in linux than OS X, but it basically works the same.  Use a USB-gps dongle and run kismac or kismet and you will have a complete wireless monitoring tool.
