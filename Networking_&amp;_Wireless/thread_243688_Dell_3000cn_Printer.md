### Post by BenHines on 2007-02-13
I accomplished this by configuring my router.  Basically, the router I have (DLink) allows you to set a particular MAC ID to always get the same IP address through DHCP.  Anytime my 3000cn requests an IP address, the router recognizes it by its MAC ID and assigns it a consistent IP address.  

I looked through the menus of the 3000cn, but didn't see anything that allowed the IP address to be explicitly set there.

Hope this helps.
