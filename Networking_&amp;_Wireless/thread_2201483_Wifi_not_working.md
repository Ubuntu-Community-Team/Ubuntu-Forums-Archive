### Post by Bucky Ball on 2014-01-25
Don't know about all the manual stuff, but the b43 packages you need are already in the repositories. No need to do any installing manually. If you checked Additional Drivers they were possibly already there. 

First thing to do is get online with a cable if you aren't already, do an update and then check Additional Drivers. If nothing, install b43-fwcutter and firmware-b43-installer (from Software Centre or Synaptic Package Manager) for a start and reboot. You may have problems if the drivers you have already installed get in the way. There may be some blacklisting to be done.

praseodym, I'm sure, can add more clarity. ;)

PS: This worked for someone on another thread:

```
sudo apt-get remove --purge bcmwl-kernel-source
sudo apt-get install linux-firmware-nonfree
```

... From here:

[http://ubuntuforums.org/showthread.php?t=1997880](http://ubuntuforums.org/showthread.php?t=1997880)

### Post by praseodym on 2014-01-25
The driver is not loaded:
```

sudo modprobe -v b43
```
Check:
```

iwconfig
dmesg | grep b43
grep b43 /etc/modprobe.d/*
```
