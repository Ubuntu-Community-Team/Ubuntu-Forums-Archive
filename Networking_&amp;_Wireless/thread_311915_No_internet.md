### Post by koenn on 2006-12-03
> The DNS address is the same as the routers.Is that right?
No, unless your router does more than routing, eg. act as a proxy-DNS. 

Put the address of a DNS server in network settings. Usually your ISP will have DNS servers you can use, but in fact any DNS server will do
(here's one : 213.224.126.242)

Are you using static IP addresses ? most ISP's pass the addresses of DNS servers and other network configuration by DHCP
