### Post by jbrown96 on 2010-05-19
Please post your /etc/samba.smb.comf

You probably just need to add something like 
```
[Public]
path = /SHARED/PATH
read only = yes
browseable = yes
guest ok = yes
```
you will want to change the name (Public), path, and read only as you see fit. Then restart samba with ```
sudo service samba restart
```

### Post by jbrown96 on 2010-05-21
My first post was to get your shared files working. The second is to allow your Windows boxes to print to your Ubuntu-attached printers.
Editing smb.conf is the best way to configure it. You can use gedit, which is a graphical editor, by running ```
gksu gedit /etc/samba/smb.conf
``` but it's better to use a command line editor. I like emacs because it's easier to enter text. If you want to up your Linux-fu, then you can install it with ```
sudo apt-get install emacs23-nox
``` and edit with ```
sudo emacs /etc/samba/smb.conf
``` use the arrows to navigate, normal keys to edit, and Ctrl-x-c to save and exit.

It's easier to use the gedit command, but it's better for learning to use emacs (like I suggested) or vi (tutorials available on the net).
