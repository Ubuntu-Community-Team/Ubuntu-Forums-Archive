### Post by Cheeese on 2011-12-13
I had the same problem with the same laptop.

Firstly, if your wireless was turned off with the hardware switch (Fn+F7) in Windows, you have to boot up Windows again and turn it on. I couldn't find any way to do this from either the BIOS or in Ubuntu.

Once you know that's been sorted out, you need to edit /etc/modprobe.d/blacklist.conf (I used: sudo nano /etc/modprobe.d/blacklist.conf) and add "blacklist acer_wmi" to the end. 

That did it for me, hopefully this helps you out.

Also, I made a [blog post]("http://www.lagserv.net/wp/2011/12/wireless-and-ubuntu-linux-on-the-lenovo-ideapad-u400/") about the issue which goes into a bit more detail.
