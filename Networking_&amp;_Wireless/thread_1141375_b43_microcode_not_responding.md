### Post by wildtiger23 on 2009-04-29
For those who try to use an hp mini 2133 with ubuntu or kubuntu 9.04 with the following issues :

Problem with the video driver.
Unstable wireless connexion.

Use this information to install the correct video driver :

[http://weeklypedia.blogspot.com/2009/01/openchrome-ubuntu-intrepid-810.html]("http://weeklypedia.blogspot.com/2009/01/openchrome-ubuntu-intrepid-810.html")

And use this information to apply a patch to the driver before compiling and installing it.

[https://bugs.launchpad.net/ubuntu/+source/linux/+bug/331952/comments/36]("https://bugs.launchpad.net/ubuntu/+source/linux/+bug/331952/comments/36")
