### Post by amrahsjar on 2014-05-19
Solved and connected bluetooth. :D

**Step 1:** followed [http://forum.ubuntu.cz/index.php?topic=70909.0](http://forum.ubuntu.cz/index.php?topic=70909.0). Still no connection.

**Step 2:** (a) created bootable usb and booted with it (b) downsized ubuntu partition and reclaimed 40gb on hdd (c) created 20gb primary partition and installed windows xp (d) downloaded broadcom driver from lenovo support site and installed on xp, it worked (e) created another 20gb primary partition and installed ubuntu from bootable usb and let grub automatically create boot records for all os.

Now, I have a fresh ubuntu installation, my previous ubuntu installation with working bluetooth without any further configuration (don't know if step 1 above had any role to play here), and windows xp. 

I have no idea why this worked but was desparate enough to try anything out and on one blog (forgot which one) it mentioned on dual boot systems with windows and ubuntu, if you installed bluetooth on windows first and booted ubuntu later bluetooth was automatically recognised.
