Every mac address filtering issue i have ever seen is due to incorrect MAC address input or incorrect formatting.

make sure all 0 are zero's and not the letter 0 and if the filter requires : or - use them, some require : seperators and some - and some require you to remove the seperators altogether.

connect with ethernet cable to double check the mac address, also are you using a static IP on the client or wanting a DHCP, does the router have DHCP setup, IP can often prevent connection thought it shouldnt
