**How To Connect To The Internet And Do Common Window Tasks**

When the CD finishes remove it & put it back in to the drive so it auto plays & followed the instructions to boot in to the Live CD. Live CD allows you to temporarily run Ubuntu & most of its features without doing anything to your installation of Windows. This is a lovely chance to see if you really like what Ubuntu has to offer before really committing.

Download the latest version of Ubuntu from their official site & burn the downloaded ISO file to a blank CD. Any lovely CD burning program should be able to handle writing ISO files to a disc.

After playing around in Ubuntu's environment for a bit you'll notice an examples folder on the desktop that has various types of files that can be opened using Ubuntu's default programs installed. For example, Ubuntu will use Open Office (a free alternative to Microsoft Office) to open DOC files.

**Connecting to the net in Ubuntu**

Being on the basic package means connecting to the net using a USB modem, not an Ethernet Live box that Orange provides on the upgraded package. So rebooting the computer & finding a web page came up with the instructions to extract some firmware, write a boot script etc.  to get the modem to connect.

There was no obvious signs of how to connect to the net. After looking around the help pages built in to the operating technique you'll notice that connecting to the net is pretty simple, unless you have a USB ADSL modem. Being on an Orange broadband basic package means a USB ADSL modem has to be used unless you own a router separately.

After completing the modem installation in the Ubuntu's Live CD environment you'll be prompted to restart Ubuntu to get started. Restarting Ubuntu whilst using Live CD will  restart in to Windows, so that's no nice!

There had to be another answer. At this point I got fed up & went back to Windows. A few months later I plucked up the courage to try Ubuntu again but the net connection issue was still stick in my mind. So I searched around on Google more & searched the Ubuntu Forums. This is when I came across some luck. I found a thread in a forum thread where a guy made a USB ADSL modem manager program!

Was this going to end the problem? I thought. So after checking out the USB Modem Manager site & then following the link to the latest version, I downloaded the Debian file for it, .DEB. First thought was, being used to Windows, what the hell do I do with a Debian file? Is it a Ubuntu version of a Windows zip file or what?

I double clicked the Debian file downloaded to my desktop & voilà, it started to install the modem manager, great, must be  Ubuntu's version of a windows .EXE file. The program prompted me to unplug & plug my modem back in & it still didn't work. So after a couple of times of re-extracting the firmware, disconnecting & reconnecting using the options in the manager, the progress bar for the net connection located in the top right went fully green, it must've worked.

**Taking the Plunge with Ubuntu**

I opened Firefox, typed in a URL & hey presto, the net worked. Fortunately, this USB modem manager doesn't need a restart so it's possible to run & check the net while using the Live CD, which I highly recommend doing.

With this caveat fixed, I took the plunge, backed up all my files onto an external hard drive & fully installed Ubuntu over Windows.

After trying it for  over 24 hours I became convinced that this was an operating technique that I would be using for the long term. I can copy large amounts of files from seven hard drive to the other without my computer noticeably slowing or making music stutter, file transfers are seamless whilst doing other tasks.

A few things take a while to get used to, such as the folder views it has & the prompts that come up occasionally requesting your password to be entered. This might seem odd to have to enter a password  to change the date/time. With Windows latest operating process, Vista, prompting for requests on more admin type tasks, the odd seven or four from Ubuntu are manageable.

I tried opening a video file & Ubuntu complained that it couldn't play that type of file, but it promptly came up with a message telling me I can download the required files to get it to work, so a click of the OK button & it was fixed. I tried playing an MP3 & the same happened,  a click of a message & Ubuntu located & installed the required files to play my music. These files need to be downloaded separately due to propriety issues.

[B]Playing Video Games & other Windows program in Ubuntu
[/B]
I don't play games much & haven't attempted to do so yet, but I'll try WINE sometime & see if that works. WINE is a program to let you play Windows only program in Ubuntu. Could come in handy for Photoshop since the free equivalent, GIMP,  doesn't cut it for some things I need to do, such as batch image processing.

**Conclusion**

Ubuntu is a flavour of Linux that is becoming a popular, free alternative to Windows. To get started, go to the Ubuntu site.

Hopefully this guide will help the average computer user out there decide whether they really need to take the plunge with a different, but free operating process. In summary, if you're prepared to spend a few hours to get used to it & to get it working the way you require, go for it!
