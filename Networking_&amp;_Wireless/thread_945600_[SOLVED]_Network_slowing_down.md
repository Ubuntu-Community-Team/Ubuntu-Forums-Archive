### Post by Jackp90 on 2008-10-12
I recently set up a network consisting of 3 routers (all 3 Linksys wrt54gl) linked to one modem/router (a Zoom adsl x5) and have been experiencing some network difficulties since then. The problem is that every so often the connection just cuts out. The network monitor says that the connection still exists but I cannot open a web page on any computer.

Any suggestions as insignificant as they might seem would be very much appreciated.

### Post by Jackp90 on 2008-11-07
It turns out that it was a problem with conflicting IP addresses.  It was just a setting in the routers.
