### Post by moshuptrail on 2007-02-11
Is that a wireless device?  USB or PCI, or built-in?

If so, you will need to read up on someting called "ndiswrapper" (most likely)

Ubuntu only recognizes a few vendor chip-sets.  But there seems to be a way to get other wireless devices to work using  ndiswrapper.  

For me, I fired up Live CD after Live CD from all kinds of Linux distros until I found one that recognized my Wifi card (Belkin).  That was my number one selection criteria because getting Wifi to work can be tricky and always  it's ugly.

p.s. ndiswrapper is for situations when no native diver exists.  Looks like you found one.
