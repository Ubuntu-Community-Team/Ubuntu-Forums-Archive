### Post by tturrisi on 2006-06-16
That card has a realtek chipset & should work.  Go to menu System/Administration/Networking &:
- give the connection a name such as 'Home' or 'Office'
- select Wireless connection
- select Default gateway device
- Properties button
- checkbox Enable connection
- select your SSID (if router set to broadcast it)
- ok out of both windows
