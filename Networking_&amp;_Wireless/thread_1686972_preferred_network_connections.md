### Post by Kreindhild on 2011-03-10
Hello!

I looked into this for you.

Right click on the networking icon, and go to edit connections. You can add multiple ethernet configurations in there. When you need to connect to each one, you can specify by left clicking the networking icon and selecting which profile you want to use. Auto eth0 is the automatic one, so use your most common config there.
