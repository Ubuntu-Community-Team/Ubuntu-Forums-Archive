### Post by profipix on 2009-07-15
**Hi all,**
I'm pretty new to ubuntu, but already figured out a way to set-up WAP on Ubuntu laptop through SE W580i.
Hope this tut will help.. Here it goes::::

==================================================

**Things you will need before starting:**
**1.** DCU-60 cable (Simply, USB cable for Sony Ericsson phones)
**2.** An actual phone with preferably unlimited Internet access on the plan
**3.** And ofkos, ubuntu laptop =)
==================================================

**1.** First off, you'll need to turn of the network manager on the laptop. You can do that by physically turning off Internet connection.
**2.**Now we'll need to set-up the phone first, in order for it to allow i-net connection.
- First, go into main menu:
[[IMG]http://img177.imageshack.us/img177/7522/dsc7639.th.jpg[/IMG]](http://img177.imageshack.us/i/dsc7639.jpg/)

Then go under "Connectivity" tab:
[[IMG]http://img441.imageshack.us/img441/9616/dsc7629.th.jpg[/IMG]](http://img441.imageshack.us/i/dsc7629.jpg/)

Now go into "USB" sub-menu:
[[IMG]http://img210.imageshack.us/img210/9616/dsc7629.th.jpg[/IMG]](http://img210.imageshack.us/i/dsc7629.jpg/)

Then go into "USB Internet":
[[IMG]http://img210.imageshack.us/img210/8370/dsc7632.th.jpg[/IMG]](http://img210.imageshack.us/i/dsc7632.jpg/)

Turn it "ON":
[[IMG]http://img441.imageshack.us/img441/5750/dsc7633.th.jpg[/IMG]](http://img441.imageshack.us/i/dsc7633.jpg/)

Now go back once, and go under "USB data accounts":
[[IMG]http://img441.imageshack.us/img441/2797/dsc7635.th.jpg[/IMG]](http://img441.imageshack.us/i/dsc7635.jpg/)

Now, MAKE SURE YOU HAVE some kind of MediaNet Connection. If you don't have it, follow this link: [http://www.sonyericsson.com/cws/support/phones/detailed/phonesetupwap/](http://www.sonyericsson.com/cws/support/phones/detailed/phonesetupwap/) and then follow instructions on how to get correct WAP connection set-up on your phone.
[[IMG]http://img149.imageshack.us/img149/7775/dsc7636.th.jpg[/IMG]](http://img149.imageshack.us/i/dsc7636.jpg/)

Now go back to "Connectivity" tab and go under "Bluetooth" menu. Once there, select "Bluetooth Internet":
[http://img401.imageshack.us/img401/83/dsc7637.th.jpg](http://img401.imageshack.us/i/dsc7637.jpg/)

Finally, and again, MAKE SURE YOU HAVE data account set up.


This concludes the WAP set-up portion of the phone. Now we'll head for the computer and do the hard work :) jk
==================================================

### Post by profipix on 2009-07-15
-------------------------------------------------

**1.** So now what you need to do is to connect the phone to the computer through USB cable.
**2.** A menu will pop up, giving you an option of using the phone for file transfer, phone mode, or print pictures. Select "**Phone Mode**"
**3.** Now click on the "Wired Network connection.." (Those 2 little monitors by volume and date..) icon and select the 1st "AT&T" button
[[IMG]http://img404.imageshack.us/img404/1220/screenshot1x.th.png[/IMG]](http://img404.imageshack.us/i/screenshot1x.png/)

**4.** So once you do that, it's probably (I don't know, it did on my phone, I assume it's because I have a PIN enabled on my SIM card) gonna ask you for a password. If it does, the just insert the PIN number your SIM card is protected with, 4 digits :)
**5.** So once you do that, (in my case, I use AT&T SP), something lik this will pop up...
[[IMG]http://img216.imageshack.us/img216/579/screenshotnfk.th.png[/IMG]](http://img216.imageshack.us/i/screenshotnfk.png/)

**6.** And this pretty much concludes the tutorial. Thx for following through :)
