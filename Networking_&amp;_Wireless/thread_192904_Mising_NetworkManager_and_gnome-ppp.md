### Post by zxee on 2006-06-09
IMO and experiance, I use dial up on my home computer, gnome-ppp does not work in dapper. That is it doesn't work at all. If you're looking for an applet to connect using dial up you could try modem monitor which is one of the applets you can add to your panel. Be warned though modem monitor has some problems although it works well enough for me-considering that there arn't many options. Modem monitor sometimes fails to dial particularly if the connection drops. When that happens I have to delete the modem monitor I'm using, restart the computer and create a new modem monitor. Dial up in dapper is not well supported.
