### Post by frankos44 on 2007-11-13
I had similar issues with my laptop running wireless. In the end it all worked out once i made sure that the laptop was set to roaming both on wireless AND on wired connections despite not actually using the wired connection. I assume you would be using WPA. When making changes to Network settings it might be wise to restart network on the pc.

My router is set so it supports static IP for  range 192.168.0.1 - 192.168.0.64 and DHCP for address range 192.168.64 - 192.168.0.255

This gives me the best of both worlds .eg. My server is on a static IP as is my desktop PC but the laptop is on DHCP (All running Ubuntu i might add) 

Hope this helps
