### Post by cdot85 on 2005-11-01
When I try to connect wirelessly to my school's network I get a message saying SSL version 2 is disabled. I have connected to other wireless networks fine. How do I "enable" it?

---

### Post by 23meg on 2005-11-01
If you're using Firefox, type about:config in your adress bar, type "ssl2" in the filter box and make sure the "security.enable_ssl2" preference is set to "true". If any other ssl2 variants are listed as disabled, try enabling them.
