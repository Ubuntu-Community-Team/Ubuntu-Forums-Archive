### Post by Phrank_Da on 2014-01-22
You're not the only one! I've been working on this problem as well... I have a Dell Inspiron E1705 running xubuntu 13.10 with a Conexant D110,HDA,MDC,v.92, modem.

It seems that Linuxant's modem drivers are no longer being developed and they stopped support at 2.6 something... So kernels 3.+ won't be able to compile the driver because linux-source has a different folder and file layout, so it doesn't know where to find the files. I was able to link some file that had moved folders, but it was completely missing others....

I'll post more links later, but it looks like at $15 usb modem off of Amazon or Ebay is the best solution...
