### Post by Claus7 on 2014-07-24
Hello,

I was trying to find out how to connect to remote server via nautilus from my trusty tahr ubuntu box, so as to have access to my files and folders graphically, and not just via command line.

The thread that I came across:
[http://ubuntuforums.org/showthread.php?t=2116588](http://ubuntuforums.org/showthread.php?t=2116588)

was not so clear for me so as to be able to figure out how to provide the address, until I saw in addition this post:
[http://ubuntuforums.org/showthread.php?t=1976517&p=11918980#post11918980](http://ubuntuforums.org/showthread.php?t=1976517&p=11918980#post11918980)

So, in order to do it you go to Places->Connect to Server... (if you are using ubuntu classic)
or else you can open your home folder (nautilus) and click under Network the: "Connect to..." option.

In there you will be prompted to type the server address like:
sftp://<user_name>@<IP_address>:<port>/home/<user_name>

and then you will have to provide your password.

Next time you select Connect to Server you will be able to see the option in order to connect again under: Recent Servers, where the address is bookmarked there for you.

Hope this helps,
Regards!
