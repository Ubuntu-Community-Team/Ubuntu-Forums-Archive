### Post by sigixv on 2009-06-07
Do the other computers connect to the router directly or is your desktop hosting their connections as a (proxy)server?

My setup at home every connection is directly on the router. When i fill in the router's IP i can configure it in a web browser: the router sends back information in the form of a webpage and thus allows to set several variables (emitting power, password configurations,...) and see log information. This program might also be able to limit bandwith for given MAC adresses (haven't checked).

So be sure that it is your desktop that has to be configured and not your router.
For the desktop i have no solutions at hand, the router probably can be set by connecting to it's IP.


Hope this helps

### Post by 3rdalbum on 2009-06-07
The router is handling all connections, not your computer (your computer is just a client), so installing tracking software on the computer will not help.

It might be possible to flash the firmware of your router to an open-source equivilant, that may provide the functionality you seek. Search Google for the model number of your router plus "replacement firmware".
