### Post by yesdup on 2007-12-11
Solved :

I needed a firmware update for my router so no issue with Kubuntu :). 

If you are in NZ and using kubuntu with a Dlink DSLG604T wireless router then get the latest firmware to solve any problems with crashing when connecting to IRC. Worked for me - just make sure you copy a backup of router settings to hard drive before hand - just to be safe.
