### Post by henriette_holm on 2007-07-31
I don't have it running right now, so you'll have to excuse if some of the menus are called something else.

Anyway, somewhere in the program there's a button called something lilke "accounts". Click it and choose setup new account. Choose a msn account and fill in user name, password etc. Click connect and you're ready to go.

If you used the MSN Messenger within the last year or so it should download a contact list for you. If not, you'll have to add the contacts yourself.
