### Post by pytheas22 on 2008-12-27
The next time the card crashes, please immediately open up a terminal from the Applications>Accessories menu, run the following commands and post the output here:
```

dmesg | tail -50
lspci -nn
uname -mr
sudo iwlist scan
lshw -C Network
lsusb
```
