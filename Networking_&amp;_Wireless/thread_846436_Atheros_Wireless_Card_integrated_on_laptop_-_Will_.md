### Post by cpetercarter on 2008-07-01
Look at [https://wiki.ubuntu.com/HardwareSupport]("https://wiki.ubuntu.com/HardwareSupport"). 

Before you commit to installing Ubuntu, boot Ubuntu from the live disk. Then test whether Ubuntu recognises your various system components, cameras, any other plug-in kit; and if it doesn't all work "out of the box", see if you can find any drivers, work-rounds etc that you might need. That won't completely guarentee a problem-free installation, but it certainly helps.
