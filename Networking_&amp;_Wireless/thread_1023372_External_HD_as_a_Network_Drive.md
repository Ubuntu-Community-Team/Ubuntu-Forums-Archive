---

### Post by lswb on 2008-12-27
If it is just a plain USB connected hard drive then you will need to have it plugged in to a computer (or other device capable of acting as a file server) and have that computer configured to share files from the drive. There are some routers that can do this in addition to their primary function.

---

### Post by Madduck on 2008-12-28
You can download and install the program "Samba" to create and manage your shares, to include the external drive.
