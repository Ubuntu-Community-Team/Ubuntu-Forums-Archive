### Post by steffen on 2006-08-29
For some reason the bittorrent client shipped with Ubuntu-Desktop is very old, and has an awkward user interface. If you want anything that comes with Ubuntu, download Ktorrent or Qtorrent instead. It's for KDE, but has a nicer interface than the Gnome equivalent IMO. Some people favour Bittornado, but I don't think the interface is much better. The problem with the Gnome bittorrent and bittornado is that they split each torrent into individual windows, which is annoying.

If you fancy the original bittorrent client (updated version) go to [www.bittorrent.com](www.bittorrent.com) and download the DEB package. Be sure to open Synaptic and remove 'bittorrent' *before* installing the DEB from bittorrent.com. It's simple and got a nice interface.

If you fancy Azureus, download EasyUbuntu or Automatix and install Azureus that way.

uTorrent is reported to run well under Wine, as far as I've heard.

Another option is to download Opera (in Ubuntu through Add/Remove Programs). Opera comes with a built in Bittorrent client. If you'd like to use Opera visit the Ubuntu Wiki for Opera and follow the steps to have Opera get the proper look and feel for Ubuntu.

In addition gTorrent, Transmission, Tribler and G3Torrent looks promising.
