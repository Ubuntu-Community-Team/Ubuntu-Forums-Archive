### Post by noob12 on 2007-10-03
If you can get the output of these commands posted, it is likely to help.

With no net on the box, you may need a USB jump drive (FAT32 formatted) to allow you to transfer stuff between the Ubuntu and Windows boots, unless you set up a shared partition.

Let's try to get your wired interface working first; generally that is easier and once online you will have much less trouble downloading whatever is necessary to set up your wireless.

```

lspci -nn

sudo lshw -C network

ifconfig -a

route -n

cat /etc/network/interfaces

```
