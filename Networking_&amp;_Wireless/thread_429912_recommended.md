### Post by obrient on 2007-05-01
One for about $70 is a linksys WRT54GL which runs linux and people have installed OpenWRT and DD WRT on it. You can usually get them from Newegg or Dell. I have one of these that is similar and flashed it with new firmware and it works well. I have another linksys router WRK54G which worked okay, but would take a little persuasion to get the wireless to jump on. 

From my experience I would think that it would be more critical for the wireless card you are using in the computer to be linux compatible, since it will be the main thing talking to the computer. For finding information on this stuff a good site is the ndiswrapper site at [http://ndiswrapper.sourceforge.net](http://ndiswrapper.sourceforge.net) for compliant cards.
