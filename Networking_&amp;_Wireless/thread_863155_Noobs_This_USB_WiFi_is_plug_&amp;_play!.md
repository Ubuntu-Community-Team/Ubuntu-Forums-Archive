hello,
       I have set up about 5 computers now all with different Wifi PCI,USB,PCIMA. 

Personally I have never found it hard to use NDISwrapper and have had 100% success with it. 
However people seam to find Wifi a real pain with linux (although wifi recognition has come on loads).

I would like to tell you that for me following USB WiFi dongle worked out the box on xubuntu 8.04LTS & ubuntu 8.04LTS:

BlueNEXT BN-WD54G

Other good news about it was it cost me £8.50 off ebay (new) and postage was free! -- It also arrived the next day! -- So couldn't be happier with it.

For those of you intersted how its listed in lsusb:

```


Bus 003 Device 002: ID 148f:2573 Ralink Technology, Corp.


```

-- So i presume its using Ralink chipset which is obviusly well supported in linux.

[http://www.bluenext.co.uk/wifi-adapter/bn-wd54g-wireless-usb-adapter.html]("BlueNEXT website")
