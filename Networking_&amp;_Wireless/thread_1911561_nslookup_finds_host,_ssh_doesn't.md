### Post by Doug S on 2012-01-19
I tried SSH using CNAME records and it worked fine. I did locally, and from two other computers, one windows and one ubuntu, on my network.
 
I would have expected your nslookup to be a little different for a CNAME record. Example:```
doug@doug-64:~$ nslookup bla.smythies.com
Server:         127.0.0.1
Address:        127.0.0.1#53
bla.smythies.com        canonical name = ns1.smythies.com.
Name:   ns1.smythies.com
Address: 192.168.111.1

```
In your nslookup there are two periods in a row, which, to me, seems odd. "foo..local"
Note: I am still using server version 10.10, so maybe there are differences.
