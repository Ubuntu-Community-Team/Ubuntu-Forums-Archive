### Post by spd106 on 2007-11-12
In this case I usually try to watch the connection status in one of two ways.

First try opening a terminal and run
```
iwevent
```
then use Network Manager to connect as normal. You should see a what stage the connection attempt fail.

For more verbose output also look in the /var/log/syslog file. You can watch that by using this command.
```
tail -f /var/log/syslog
```

or just simply open it in the log viewer, System -> Administration -> System Log.
