### Post by mikewhatever on 2008-10-19
This is a common problem when installing wireless drivers. To put it simply, you need to download stuff you want to install, but there is no way to do that because the wireless device is not working. Obviously, the device needs firmware to work which can't be downloaded.
I've no idea if there is a way to use your particular device without ndiswrapper (you may want to provide more specs), but if there was, you'd most probably hit the same issue of needing to download stuff.
