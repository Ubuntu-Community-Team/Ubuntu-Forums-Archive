### Post by gnomeza on 2008-02-25
Hey

I recently bought a Thomson Speedtouch 516 and upgraded the firmware using a combination of bootp and tftp installed on my Gutsy laptop. It was surprisingly easy.

Instructions are here:
  [http://markferry.net/Network.SpeedtouchUpgrade](http://markferry.net/Network.SpeedtouchUpgrade)

It should be noted that while doing this my laptop had a static IP address.
Also, make sure you get the correct firmware for your device.

Instructions assume you're comfortable with the command line. And, as always, don't run commands unless you understand what they'll do.

Hope this helps.
