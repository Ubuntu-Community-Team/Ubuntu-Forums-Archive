Wireless is very much hardware dependent. It can vary from working out of the box to working with ndiswrapper to working with limitations to not working at all. It all depends on if the wireless manufacturer has written drivers for linux and if not if the open source community has done so.

In addition there can be issues with WPA, sometimes it is easier to install wicd.
