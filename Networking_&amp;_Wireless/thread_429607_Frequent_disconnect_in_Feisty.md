### Post by whayong on 2007-05-02
I just wanted to update my own thread in case someoen else runs into the same problem.  All I did was install WICD.  Yes, you have to remove network manager to install it if you're using GNOME.  If you're on Xfce, just install it.  It works.  No more disconnecting.
