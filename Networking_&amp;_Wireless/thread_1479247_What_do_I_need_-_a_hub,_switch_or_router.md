### Post by Iowan on 2010-05-10
You need a router. If you only get one IP address from your ISP (like most of us), you can have only one device "connected". If that device is a router, it can NAT (Network Address Translation)... make several devices look like they all have the same (external) address. Most routers (that I'm familiar with) also have some firewalling capabilities.
