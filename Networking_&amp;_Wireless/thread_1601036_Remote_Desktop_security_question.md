I've had Remote Desktop running on 3 local machines for a while.. usually it's simply for me to update all of the PC's, etc.. w/o having to physically go to them.

After installing 10.10, I set up VNC again, just as I did with 10.04.  The problem is, when I click on Remote Desktop preferences, it shows my PC's as being accessible from outside the network, despite the fact all ports on my router are closed.

So just to test this, I used VNC to come in from my IP address(as opposed to my network IP) and sure enough, I can use my desktop after entering the password I set up.
