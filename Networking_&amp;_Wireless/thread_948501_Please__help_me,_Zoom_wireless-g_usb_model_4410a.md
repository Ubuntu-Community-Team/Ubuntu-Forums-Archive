And hire is my glide how to install drivers for zoom 4410a :)
1 - First copy from original zoom cd drivers (WlanUIG.sys and WlanUIG.inf)to any location.
2 - Now go to add/remove programs find "Windows Wireless Drivers" program. (you need connection to the internet)   
3 - open program and find wlanUIG.inf (if icon "present=yes" then you succeeded !)
