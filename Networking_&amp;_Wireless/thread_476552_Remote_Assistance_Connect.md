According to [this]("http://www.cyberciti.biz/tips/linux-remote-desktop-for-controlling-windows-xp-desktop.html") article, you can control a windows machine using almost any unix system.

It's even in the repositories, so you can simply type:

```
sudo apt-get install rdesktop
```

Read more about rdesktop [here]("http://www.rdesktop.org/").
