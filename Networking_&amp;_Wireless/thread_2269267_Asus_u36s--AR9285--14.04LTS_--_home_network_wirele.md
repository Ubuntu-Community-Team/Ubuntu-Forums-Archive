### Post by jeremy31 on 2015-03-14
The one thing you can do is change the wifi routers encryption to WPA2 only, without TKIP as I am sure it is part of your issue.  And it wouldn't hurt to set the router to one of the unused channels- 2, 3, 8, 9, or 10

### Post by jeremy31 on 2015-03-25
Also try ```
sudo modprobe -r ath9k
``````
sudo modprobe ath9k nohwcrypt=1
```

If that makes an improvement then ```
echo "options ath9k nohwcrypt=1" | sudo tee /etc/modprobe.d/ath9k.conf
```
