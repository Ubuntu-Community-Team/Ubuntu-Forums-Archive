### Post by Kernelingus on 2010-11-30
If you can, find yourself an older version (Version 4 or prior is ideal) of the **Linksys WRT-54G** and flash the firmware with the awesome, Linux-based **[DD-WRT]("http://www.dd-wrt.com/site/index")**.  You can probably find one on Craigslist.  I've picked up several there.  I think the last one I got was a version 4, and I paid $20.

Even the newer versions of the WRT-54G will work with DD-WRT.  But the older ones have more flash memory, so you can do more with them.  :)
