### Post by hellion0 on 2009-10-16
It helps to know what hardware we're dealing with. I have an Optiplex as well, but it could have a different network card than yours (mine's a GX260.) 

Run lspci in a terminal and post back what ethernet card is in the machine. The entry should look similar to this:

```
01:0c.0 Ethernet controller: Intel Corporation 82540EM Gigabit Ethernet Controller (rev 02)
```

...Alternately, it could be "Network controller" instead of "Ethernet controller."
