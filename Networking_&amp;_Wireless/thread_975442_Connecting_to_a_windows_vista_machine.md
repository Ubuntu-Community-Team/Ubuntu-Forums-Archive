### Post by doppis on 2008-11-10
If your using vista...

1. click start
2. click network
3. click network and sharing center

Under sharing and discovery you have a list of items like...network discovery, file sharing, public folder sharing ect...

You can open those up and turn them on or off.
Under password protected sharing is where you might want to turn off to be able to log in on the network with ubuntu unless you would rather use passwords.  It will still ask you for your password...just tell it to forget it.
