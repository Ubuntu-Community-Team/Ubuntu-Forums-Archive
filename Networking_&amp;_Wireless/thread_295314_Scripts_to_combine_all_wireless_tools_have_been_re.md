### Post by enigma_0Z on 2006-11-07
Hi all,

Just wanted to officially announce the alpha release of wifisetup, a collection of scripts that combine the functionality of the various wifi tools. This was written mostly because the tools available do not support manual and/or ad-hoc configurations. Additionally, this script can launch a script (for instance, mounting filesystems) upon connection. The script is also modular. See the docs in /usr/local/share/doc/wifisetup. The script comes with a wpa_supplicant module already installed. Modules go into /usr/local/lib/wifisetup.

Oh yeah, the install script assumes that /usr/local/lib, /usr/local/share/doc, and /usr/local/bin have been created. you may need to manually.


Finally, you can get these awesome scripts at [http://sourceforge.net/projects/wifisetup](http://sourceforge.net/projects/wifisetup)
