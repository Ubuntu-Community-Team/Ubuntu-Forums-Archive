### Post by Sergei_K on 2011-04-18
try adding these lines to /etc/network/interfaces
 
```
 
auto eth0
iface eth0 inet dhcp

```
 
after that change user to root in terminal and restart network:
 
```

sudo su -
/etc/init.d/networking/restart

```
