### Post by newbie-user on 2011-07-27
You'll need to set up a mail server first of all.  Check out postfix for that.  You'll also need to obtain a domain name from a registrar such as GoDaddy or Networksolutions so that you can actually have @yourdomain.com.  You will need a static ip, preferably, otherwise sign up for dynamic DNS.  There's actually a lot of stuff to go through, but it's not so bad with a bit of research and help.
