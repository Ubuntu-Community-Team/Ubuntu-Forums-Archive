### Post by Carl H on 2009-05-18
At the weekend I bought a T-Mobile "Web 'n' Walk" USB modem, and have got it working fine on Ubuntu 9.04. Most of the posts/howtos etc that I've found relate to the Hauwei E220, but as mine is an E160 I'm posting this on the off-chance that it's useful to somebody. 

It almost works OOTB - plug it in and Network Manager fires up a wizard to set up a new connection. I found that the only change I needed was to set the User and password to 'ppp'. The default settings didn't work. I got the 'ppp' from an example I found on the net, which sadly I now can't find again. (It will be on my home PC, I'm at work at the mo). 

It's refreshing to see that T-Mobile **[acknowledge Ubuntu as an OS](http://tm.feedbacklab.com/display/support/Copy+of+How+to+setup+and+connect+a+web%27n%27walk+USB+using+Ubuntu+as+your+operating+system)**, even if they don't officially support it. 

Some other snippets of info that I wish I'd known before I started....

1. There is a Sim card hidden in the box. This needs to be fitted to the USB stick before you start. 
2. The Sim card has to be fitted the right way around. :D
