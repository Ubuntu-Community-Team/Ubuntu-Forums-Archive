### Post by kiirokurisu on 2012-04-10
Is your smartphone an Android phone? If so it does indeed not support ad-hoc wireless connections (been a shortcoming of Android for a long time now). As for running your laptop as an access point, this is only possible if your wireless adapter supports going into the 'master mode' as you discovered. If it does not support this, you're totally out of luck (trust me, I've tried).
