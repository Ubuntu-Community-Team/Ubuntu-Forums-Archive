### Post by Topher88 on 2009-08-06
> **strAlan said:**
> Right-click the panel and select "Add to Panel" then choose the Network Monitor icon.

I think what he wants is Network Manager, as Network Monitor will not show available networks.

Instead of clicking on "Network Monitor," try "Notification Area."  If Network Manager has been turned off and nothing happens, push Alt + F2 and type "nm-applet" without the quotations.
