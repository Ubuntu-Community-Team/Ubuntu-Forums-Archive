### Post by praseodym on 2011-08-21
Try the Madwifi-Trunk-driver (Module: ath_pci):

```
sudo apt-get install linux-headers-generic build-essential
wget http://snapshots.madwifi-project.org/madwifi-trunk-current.tar.gz
tar xvf madwifi-trunk-current.tar.gz
cd madwifi-trunk-r*
make
sudo make uninstall
sudo make install 
```
After that change the blacklist file:

```
gksu gedit /etc/modprobe.d/blacklist-ath_pci.conf
```
from
```
blacklist ath_pci
#blacklist ath5k
```
to
```
#blacklist ath_pci
blacklist ath5k
```
You may change the country-code via
```
echo "options ath_pci countrycode=276" | sudo tee /etc/modprobe.d/ath_pci.conf 
```
according to [this]("http://www.davros.org/misc/iso3166.html#existing") list (276 is for Germany).
After that reboot your system.
