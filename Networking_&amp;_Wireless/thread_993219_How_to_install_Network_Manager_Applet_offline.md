### Post by spd106 on 2008-11-25
Don't bother with that as you'll have to build from source.

Get the deb file from the Ubuntu archive here (for Ubuntu 8.10):
[http://packages.ubuntu.com/intrepid/network-manager](http://packages.ubuntu.com/intrepid/network-manager)

Just copy the .deb file over to Ubuntu and double-click it to install. You might need one or two other dependencies, but I think that's unlikely.

Once it's installed just hit Ctrl+Alt+Backspace or reboot and it should be back up and running.
