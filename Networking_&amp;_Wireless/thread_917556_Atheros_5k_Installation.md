### Post by jdguzman on 2008-09-12
I'm sure there's alot of guides out there for getting this wireless chipset to work on Ubuntu but I figured I'd put up the info that finally allowed me to be writing this post through my wireless nic card :).

First this is for the Ath5k drivers.  The first thing you need to do is download the latest HAL tar from:

[http://snapshots.madwifi.org/madwifi-hal-0.10.5.6/](http://snapshots.madwifi.org/madwifi-hal-0.10.5.6/)

You should find the latest tar packages there just download and untar the one you choose. next cd into the folder that you just created and from here just follow these instructions:

[http://madwifi.org/wiki/UserDocs/FirstTimeHowTo](http://madwifi.org/wiki/UserDocs/FirstTimeHowTo)

They should be pretty self explanatory.

Should anyone need some help I'll offer what help I can :).  Hope this helps.
