### Post by Hadaka on 2016-10-02
Hi, You have two SSIDs named TheDen. One is 5 ghz and the other 2.4 ghz. I  assume part of the problem is that the card is roaming among them and  also the 5 ghz may be set to 802.11a, b, g and n. Your old-school 3945 is  an a, b and g device and so I wonder if the router is trying  and then  failing to negotiate up to n. 

I'd give the SSIDs different names; something like TheDen2.4 and TheDen5  and see if the connectivity is improved at 2.4. You might also try  turning off the n on TheDen5 and see if it helps.

You also have Power Management set to ON an that needs to be set to OFF.

Thanks

### Post by Greylopht on 2016-10-02
Set the  router up differently and did.

```
 sudo iwconfig wls3 power off 
```

And that seemed to work.  How do I make the change permanent.  IE which file do I edit.  All the information I find is for 15.04.

Never mind.  Did.

```
sudo gedit /etc/network/if-up.d/off-power-manager
```

Inserted 

```
#!/bin/sh
/sbin/iwconfig wlan0 power off
/sbin/iwconfig wls3 power off
```

Saved and then

```
sudo chmod +x /etc/network/if-up.d/off-power-manager
```

That made things permanent from what I can see.

That got things working, thank you very much Hadaka!
