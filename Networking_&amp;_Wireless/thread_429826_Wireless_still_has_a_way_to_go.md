### Post by sashazur on 2007-05-01
Just some comments on trying to get my wireless card working w/7.04, and why it's still too hard:

- Why does the network icon let you try to connect to a wireless network even if there's no wireless network card configured on the PC? Nothing provides any hints that the hardware you need isn't there.

- Why do different parts of the GUI show different wireless security options? In one place, the only option is WEP. In other places, you see WEP and WPA.

- Why isn't ndiswrapper preinstalled when you're doing an upgrade install from Windows?

- Why isn't the respository for ndiswrapper pre-configured so that ndiswrapper is listed as an installable option without having to first figure out what repository to add?

- Why does ndiswrapper indicate that a driver is correctly installed but the rest of the system still won't recognize the wireless card?

- When ndiswrapper shows that an alternate driver is installed, how come the troubleshooting instructions I found don't let me uninstall it?

Getting wireless to work should not be this hard!
