### Post by Werewolf486 on 2011-02-27
So I had posted that I had 2 Ethernet ports and neither of them were working on a fresh build a friend was struggling with. After reading the suggestions to fix the issue and thinking that there had to be an easier way I had an epiphany. I would go to my spare parts box and bring out my old Netgear GA311 and pop it into the slot. So after searching for 10 minutes I found it and installed it. Booted up the rig and it found that right away (SWEET) now that's not the fix anybody can do that, once I updated Ubuntu the on-board ports started working (that's what I'm using now). I can't explain what the update did but for a few bucks a used card in the tool box might not be a bad idea! I truly hope that this is a help to others, while not real technical I hope my simple brain provides somebody with an easy fix to an aggravating problem.

Mother board is a Gigabyte GAX58A-UD5 Rev.1 Personally I'm an ASUS guy.
