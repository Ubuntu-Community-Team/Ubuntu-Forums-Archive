### Post by joshua__lim on 2007-12-24
You can either setup your own SMS Gateway using a GSM modem connected to a PC, or you can connect to an SMS Gateway provider like [http://www.clickatell.com](http://www.clickatell.com) through the Internet, or you can use a combination of both. There are open source gateways like [http://www.kannel.org](http://www.kannel.org) and also commercial gateways like [http://www.visualtron.com](http://www.visualtron.com).
