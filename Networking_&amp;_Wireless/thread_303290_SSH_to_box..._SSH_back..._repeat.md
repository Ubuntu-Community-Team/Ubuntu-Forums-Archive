A few weeks ago I SSH'd into my home computer from a friends house via [Putty]("http://www.putty.nl/") and I realized the file I wanted was also at my house but on a different machine. I then had the crazy idea to try and SSH from my box that I was currently on into the machine where the file was located. AND BAM!!!! it worked... I then played around with SSHing from box to box at my house... between 4 machines... and I was quite happy with how it all worked...

sorry about the rant... just wanted to spread the love that I'm feeling now that I think back to that night :-P

### Post by reid on 2006-11-20
That is a cool feature.  I have seen it used for security measures.  You can set ssh to only allow connections from certain IP addresses.  Then you can ssh from a remote machine into the one that has access to the final destination, then ssh into there through the intermediary machine.
Kinda neat!
