### Post by PatchesTheCaveman on 2011-01-03
*linux-image-2.6.35-24-generic* is the faulty update.

There is an earlier update for that package that works fine.  You can install it by running:
```
sudo apt-get install 2.6.35-23-generic
```

Also, what sound card do you have that broke with that update?  I'll add it to the bug I have filed against the Linux kernel in Ubuntu.
