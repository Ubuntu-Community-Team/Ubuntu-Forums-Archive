### Post by him610 on 2007-09-10
I recently upgraded to a Belkin N Wireless Router, F5D8233 from a Linksys WRT54G that got zapped by lightning. I use a mixed network consisting of machines running WinXP, freeBSD, and Ubuntu 7.04. The F5D8233 is a good router, in MHO, much better than the Linksys model. Although I setup my router from one of my  WinXP machines, the whole setup process seemed to be pretty straight forward to me. The setup guide for the router was written for a Windows-based machine; however, if you read it carefully you should be able to figure it out for a Linux-based machine since the setup is done using one's web browser interface. The documentation for the router is included on the CD that came with it.
Good luck.
