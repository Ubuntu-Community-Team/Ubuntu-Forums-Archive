### Post by secdroid on 2006-08-26
Sorry, but you really won't get any help until you know manufacturer, model, and version.  Sounds too picky, but version is very important.

Best bet is to power down, pull the power cord, remove the adapter and note all of the info on it.  (Assumes you are comfortable with working with PCs.)

If you could check out the hardware info on Windows, it could tell you the first two, but not the last.

Many manufacturers use the same model number for multiple different chips.  From a Windows user/store SKU perspective, they are identical because they are shipped with the right driver and are functionally equivalent.
  
Unfortunately, we need to know this info to go find the right drivers for Linux.
