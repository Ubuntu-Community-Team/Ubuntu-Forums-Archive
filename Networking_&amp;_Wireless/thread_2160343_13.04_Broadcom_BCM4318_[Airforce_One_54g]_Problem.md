### Post by chili555 on 2013-07-06
Your device 14e4:4318 reqquires firmware that is not installed by default. Here is a post that describes the process to install it without an internet connection: [http://ubuntuforums.org/showthread.php?t=2134058&p=12597332#post12597332](http://ubuntuforums.org/showthread.php?t=2134058&p=12597332#post12597332)  See post #9.

### Post by chili555 on 2013-07-06
Ah ha!! Please do:```
sudo apt-get remove --purge bcmwl-kernel-source
```After it's done, reboot and tell us if it's working beautifully.
