### Post by cmnorton on 2008-11-30
You'll need to fiddle around with Network Manager and choose either a secure method (like a wallet) or save as plain text. At least a year ago and using Kubuntu, the secure wallet method was not well liked. I use a plain text save method, not as secure, but it works.
