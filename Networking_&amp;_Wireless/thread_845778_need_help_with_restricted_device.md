### Post by ad_267 on 2008-07-01
I don't think you can install wireless drivers from the live cd, you have to install them after you have actually installed Ubuntu.

What wireless card to you have? Check that it is supported here: [https://help.ubuntu.com/community/WifiDocs/WirelessCardsSupported](https://help.ubuntu.com/community/WifiDocs/WirelessCardsSupported)

I think that's a good thing that it is saying it is restricted as it has recognized the device. If you install Ubuntu then you should be able to go to System - Administration - Hardware Drivers and just tick next to the drivers to enable them. If it's supported and you want to install Ubuntu you can resize your Windows partition so that you can dual boot both operating systems.
