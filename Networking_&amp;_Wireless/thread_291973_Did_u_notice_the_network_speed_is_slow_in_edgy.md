### Post by Bretls on 2006-11-09
I had the same problem with edgy. I was also connecting to verizon dns servers which aren't that fast to begin with. 
I set up my computer as a dns server and looped back to it for my dns lookup. This made a huge difference in page loading.
Here's the tutorial I used to set it up  [http://www.ubuntuforums.org/showthread.php?t=236093]("http://www.ubuntuforums.org/showthread.php?t=236093")

Also, do a search for disabling ipv6 in the forums, this should be an easier way to help your speed.
