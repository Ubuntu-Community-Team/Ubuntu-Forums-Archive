### Post by snowpine on 2008-11-10
Hi there Ghostlove, unfortunately not all of the eee models are 100% supported by Ubuntu out-of-the-box. Since you didn't mention which eee you have, we'll have to guess. :)

I suggest visiting the eeeuser.com sub-forum for Ubuntu: [http://forum.eeeuser.com/viewforum.php?id=43](http://forum.eeeuser.com/viewforum.php?id=43)

I have an eee 900ha. I was able to get everything working in Ubuntu 8.10 by installing the custom eeepc kernel from array.org. There is a big thread about it over on the eeeuser.com forums--check it out!

If you don't want to mess around with custom kernels, there is also a distro called Ubuntu eee which is Ubuntu Hardy + the custom kernel + netbook remix interface = Everything Just Works!  [www.ubuntu-eee.com/](www.ubuntu-eee.com/)

I don't know much about it, but there is also an eee-specific Xubuntu called eeeXubuntu.

The only wrinkle is that my ethernet worked out-of-the-box, so that made installing the new kernel a lot easier. You might have to come up with a clever plan such as installing Xubuntu (with the custom kernel) onto a USB stick on a different computer, then copying it over to your eee. But definitely ask over on eeeuser.com because they are a knowledgeable bunch. :)
