### Post by Talon2 on 2011-06-16
I have a netbook with a Broadcom shipset.  Here is what I do to get wifi going:

1) Connect the netbook to my router via cable.

2) Make sure I have internet access.

3) Run "Additional drivers" >

System > Administration > Additional drivers

4) Install Braodcom STA.

5) Reboot.

6) Live happily ever after.

Now, if that isn't the problem, we will need to know what specific wifi chipset you have.
