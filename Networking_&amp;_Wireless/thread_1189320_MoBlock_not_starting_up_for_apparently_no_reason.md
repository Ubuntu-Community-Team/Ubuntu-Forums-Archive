### Post by jre on 2009-06-19
The daemon crashes while it´s loading the blocklist. As far as I can see, this is because you use OpenDNS, which does not play well with the blocklist update (the blocklists are corrupted). I´ll investigate that sometime, since I just got another report for this topic.

In the meantime I can only advise you to "sudo aptitude purge blockcontrol", disable OpenDNS, and then install blockcontrol again. If you want to keep OpenDNS, then you have to disable blockcontrol´s automatic blocklist update. Whenever you want to update the blocklists manually, you´d have to disable OpenDNS
