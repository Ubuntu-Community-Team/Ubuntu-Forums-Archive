### Post by cbanakis on 2012-09-14
It is extremely rare for someone to actually need a print server.

If the printer is connected to your network, Ubuntu should detect it.
Wether or not it will find a driver is another story.

If the printer is connected to a computer on your network, you would need to share the printer through that computers printer settings.

If ubuntu does not find a driver, it could get tricky, but usually you can find a driver somewhere for common printers.

### Post by cbanakis on 2012-09-14
Start by going into the printer settings on your computer.

Click add printer.

Then select network printer, and see if the printer pops up in the list.
(Might have to give it a few minutes)

If it does, select it, and install.

Ubuntu will try to locate and install a driver automatically.

It the printer is not detected, then I have no further advise.

If Ubuntu cannot find a driver, you can try the manufactures website.

Or just google "Lexmark 6500 ubuntu driver"
