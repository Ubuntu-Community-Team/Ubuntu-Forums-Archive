### Post by puppywhacker on 2008-11-15
Hi,

I don't fully understand your situation. There are a few things you could do depending your exact situation

1. start firefox from a terminal window with specified W x H settings
firefox -height 600 -width 800

2. start firefox from a terminal window in debug mode
firefox -debug

3. undo any full screen mode by pressing the F11 key

4. resize and move the firefox window
right click firefox on the bottom program bar and click on
a. resize
b. move
c. unmaximize

goodluck
