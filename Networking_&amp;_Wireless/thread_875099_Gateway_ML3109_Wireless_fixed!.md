### Post by nickgaydos on 2008-07-30
Hello everyone! I am Nick. I am new to the Ubuntu forums but have been using Ubuntu since the summer of '07. I am one of the few who have a Gateway ML3109 Laptop. I have read that a lot of you had problems with the Wireless internet. Well, I am here to fix that for you. You will not need to use the Terminal for anything with this process. The first thing you need to do is go to Add/Remove Applications in your Applications menu. The next thing is to search for Windows Wireless Drivers. Check off that and then search for Rutilt WLAN Manager or any other wireless network manager. The next thing is the most important, the drivers. I'm sorry I cant put the direct link so your gonna have to Google Realtek 8185 (I'm not exactly sure) Windows Drivers. Once that is completed, go to System>Administration>Windows Wireless Drivers. Next click install new driver. Navigate to your Realtek Driver folder untill you see Windows 2000. Enter that folder and choose the net8185.ntf and it should install. The final step is to search for a network using Rutilt WLAN Manager or the WLAN Manager of your choice. This was all done on Ubuntu 8.04 Stable Release.
Thanks! :popcorn:
