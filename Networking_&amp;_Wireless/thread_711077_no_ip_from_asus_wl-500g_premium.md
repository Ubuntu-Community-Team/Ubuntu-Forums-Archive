### Post by superprash2003 on 2008-02-29
try this.. go to system->administration->network.. you should find wired connection(eth0 or eth1).. click on properties and make sure its set to DHCP.. incase its set to ROAMING.. if it still doesnt work.. try manually entering a ip there...if it still doesnt work.. go back to DHCP.. and type ifconfig in the terminal and post results!!
