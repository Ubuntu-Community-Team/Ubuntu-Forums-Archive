### Post by dmizer on 2011-02-12
If you want a chat room, you'll probably need a messaging server of some kind attached to the network. I assume that would be impossible in your situation.

If you're using your own computer on the network, you could try installing an xmpp server on it. Here's a good tutorial: [http://www.waveprotocol.org/federation/openfire-installation](http://www.waveprotocol.org/federation/openfire-installation)
