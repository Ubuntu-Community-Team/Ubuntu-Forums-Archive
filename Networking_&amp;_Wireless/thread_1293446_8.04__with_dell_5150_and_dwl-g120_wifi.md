### Post by hal10000 on 2009-10-17
if you card is an usb device, then it makes no sense to post the output of lspci, use lsusb instead

[EDIT] your Broadcom Corporation BCM4401 100Base-T ethernet driver may run if you use the b44 kernel module driver.
Try
```
lsmod |grep b44
``` to verify if it's already loaded and if it's not, try
```
sudo modprobe b44
```
then do
```
sudo lshw | less
```
scroll down to the ethernet card section and look to the last lines of that section to see if the b44 driver is in use.
You may then configure your ethernet card with your network-manager
