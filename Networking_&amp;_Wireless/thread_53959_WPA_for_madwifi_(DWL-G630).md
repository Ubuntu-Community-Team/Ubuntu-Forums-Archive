### Post by luca_linux on 2005-08-03
Try to run wpa_supplicant with this command:
```
sudo wpa_supplicant -B -iath0 -c/etc/wpa_supplicant.conf -dd
```
(With "-w" taken out).

If it doesn't work, try to add the "-D" flag.
```
sudo wpa_supplicant -B -iath0 -c/etc/wpa_supplicant.conf -D -dd
```

You could also think about compiling and installing the latest wpa_supplicant version (0.4.3) from [here](http://hostap.epitest.fi/wpa_supplicant/) or giving [xsupplicant](http://www.open1x.org/) a try.
