### Post by coffeecat on 2007-07-11
Sorry I didn't get back to you before - we're probably in different time zones.

I can only make a few suggestions - I only use Windows infrequently and although I can usually find my way around I can't remember the various settings utilities well enough.

Yes - check that your firewall in Windows is not blocking anything. Did you make sure that you haven't got MAC address filtering enabled in the router? Have you tried connecting the laptop to the router with an ethernet cable? If you find you get internet access that way at least you've narrowed it down to a wireless problem.

Lastly - one day I found my laptop wouldn't connect wirelessly. It really baffled me. It had been working fine the previous day. I've set this laptop up as a multi-booter with Windows and several versions of Linux. I tried booting into one, then the other but nothing I could do would make it connect. Then I noticed that there is a little slider switch on the front marked 'WLAN OFF- ON'. It was switched off - I must have caught it without noticing. :oops: :roll: . You might want to check that. :lol:

I don't want to shoo you away - far from it - but if none of this works, I suggest you try a Windows forum. Someone there should be able to walk you through the various configuration windows.

But before you go :) - check out [this site.]("http://www.ubuntu.com/") Go on. You know you want to. :wink: Once you get your internet connection, download the Ubuntu iso from there and then don't hesitate to come back here if you need any help.
