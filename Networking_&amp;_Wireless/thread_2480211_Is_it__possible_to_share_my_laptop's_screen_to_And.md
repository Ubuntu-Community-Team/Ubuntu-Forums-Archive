### Post by TheFu on 2022-10-23
What is possible depends on the Android side. Not all android systems support screen sharing.
Similarly, not all Ubuntu Desktop Environments make sharing a screen easy.

'KDE Connect' is the normal Android tool [https://kdeconnect.kde.org/](https://kdeconnect.kde.org/) .
> The KDE Connect team has applications for Linux, BSD, Android, Sailfish, iOS, macOS and Windows.

'GS Connect' is the Gnome version [https://extensions.gnome.org/extension/1319/gsconnect/](https://extensions.gnome.org/extension/1319/gsconnect/)

The Android versions are in popular app stores.

I think Google Chrome browser supports screen sharing, though I'd never use Chrome myself.

So, the only answer is "it depends."
