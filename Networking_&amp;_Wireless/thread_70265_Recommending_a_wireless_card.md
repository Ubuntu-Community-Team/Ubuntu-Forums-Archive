### Post by az on 2005-09-29
Many of them have linux drivers, so they are automagically set up for you.  The one's that do not have linux drivers can be used with the windows drivers and ndiswrapper.

You would need to install ndiswrapper-utils and configure your card with that.   There is a howto page about it on the wiki.

It is quite easy.  Breezy has newer drivers so I would reccmmend starting with Breezy.  It will be stable in a few days (weeks) but you can run it now...
