### Post by Paddy Landau on 2011-05-11
You will need [LAMP]("http://en.wikipedia.org/wiki/LAMP_%28software_bundle%29") -- Linux, Apache, MySQL, PHP/Perl/Python.

See the [community help]("https://help.ubuntu.com/community/ApacheMySQLPHP") (a bit dated, but useful information).

If you can go for a clean installation, [download Ubuntu Server]("http://www.ubuntu.com/download/server/download") instead of the default Ubuntu Desktop.

Search these forums, because I don't know much about it, but the question has been asked before.
