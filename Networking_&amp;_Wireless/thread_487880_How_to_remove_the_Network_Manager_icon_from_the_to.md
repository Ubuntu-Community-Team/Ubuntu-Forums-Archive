### Post by RPG405 on 2007-06-29
I have a network manager icon in my toolbar that is currently doing nothing except wasting space. I only have one profile, and it is always enabled (ethernet), so I have no use for it. How do I get rid of it? \\:D/

---

### Post by reacocard on 2007-06-29
Under System -> Preferences -> Sessions -> Startup Programs, uncheck the box next to 'Network manager'. Log out and back in, and the icon will be gone.
