### Post by Bil-E-daKid on 2007-02-10
Hi there,

If you go to ubuntuguide.org, read and follow the General Notes section:
[http://ubuntuguide.org/wiki/Ubuntu_Edgy#General_Notes]("http://ubuntuguide.org/wiki/Ubuntu_Edgy#General_Notes")

Then, follow the Samba server install section:
[http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_install_Samba_Server_for_files.2Ffolders_sharing_service]("http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_install_Samba_Server_for_files.2Ffolders_sharing_service")

This will install the software you need to participate on a Windows network/workgroup.

Then, to change the workgroup settings, follow this bit:
[http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_change_computer_Domain.2FWorkgroup]("http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_change_computer_Domain.2FWorkgroup")

Then you should be fine.

Remember, the ubuntuguide.org site also has a wealth of other information available on it - it's definetely your friend!  :-)
