How do I join a Windows XP Workgroup named HOME?

---

### Post by pricetech on 2010-05-11
You don't really "join" a workgroup like you do a domain.

System - Administration - Samba
Type in your password
Preferences - Server Settings
Change the workgroup there.

That should help browsing, but makes no difference when it comes to connecting to a windows share.  Nor does it make any difference when connecting from windows to a Samba share.
