### Post by Jeroensum on 2010-05-29
In Evolution there are no standard options to do this, however you can add a 'search folder'. A searchfolder is a user specified filter which can also contain dates. I've just added one for my own use which shows all the emails older than 7 days.

Here's how I did it:

1.) Goto Edit -> Search Folders
2.) Add a folder
3.) Set conditions: Date sent -> is before -> 1 week ago
4.) Select All local folders
5.) Close and check the botom of the Side Bar on the left
