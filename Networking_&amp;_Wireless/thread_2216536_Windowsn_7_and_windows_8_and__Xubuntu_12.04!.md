### Post by gordintoronto on 2014-04-12
File sharing uses Samba, and I'm not sure if it is included in Xubuntu 12.04.

The file manager in Xubuntu is Thunar. In the bottom-left of my Thunar window it says "Browse Network". I click on it, and my wife's (Windows 7 Ultimate) computer appears in the right, along with some other things. I double-click on it, and the Shares on her computer appear. I double-click on one, and the list of files appears. I can copy from her computer, or paste to it.

One thing which might be helping: I have the same username and password set up on both computers.

I also have a shared folder set up on my Linux system, and her Windows system sees it using a very similar set of clicks.

It's not complicated at all!
