### Post by Placidia428 on 2013-06-03
I just had the same problem. Realtek has a driver - rtl8191su. Download it [here ]("http://www.realtek.com.tw/downloads/downloadsView.aspx?Langid=1&PNid=48&PFid=48&Level=5&Conn=4&DownTypeID=3&GetDown=false&Downloads=true#RTL8191SU"). Unzip the folder and run the install script (install.sh). You need to change the permissions on the sh file to allow execution.
