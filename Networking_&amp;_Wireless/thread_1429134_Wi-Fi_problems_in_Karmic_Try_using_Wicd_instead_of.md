### Post by Rick Deckard on 2010-03-13
If you're wi-fi connection was working fine before you upgraded to Karmic and now it randomly disconnects and forgets your password, try replacing network-manager with the wicd package. 

I did and now my Wi-Fi connection is rock-solid again - the Wicd configuration interface is nicer than Network-Manager's too! :D
