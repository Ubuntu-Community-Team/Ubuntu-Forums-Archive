### Post by slickymaster on 2014-02-26
I would advise you to use either one of the following:
[LIST=1]
[*][vnStat]("http://humdi.net/vnstat/") - It's a console-based network traffic monitor, which is quick and simple to install and get running. It gathers statistics persists through system reboots and can monitor multiple interfaces at the same time. Also it has several output options and summary (hourly, daily, monthly, weekly, top 10 days) and allows to configure months to follow billing period.
[*][download-monitor]("https://apps.ubuntu.com/cat/applications/precise/download-monitor/") - It displays the amounts of data transferred over the network with graphs showing historic monthly, daily and hourly usage. Allows you to set usage quotas and warns you when you exceed them or look like you will exceed them.
[/LIST]
