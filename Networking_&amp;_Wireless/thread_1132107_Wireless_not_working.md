### Post by StuartN on 2009-04-21
Is there a network manager connection icon (a little computer display, or a bar-graph) somewhere in the top right-hand of the screen? If so then you left-click and either select a network or connect to network.

If not, then there is a detailed how-to [http://ubuntuforums.org/showthread.php?t=370108](http://ubuntuforums.org/showthread.php?t=370108) listing the details needed to sort out networking. In brief, you need to identify your network card or usb dongle by typing "lspci" or "lsusb" in a terminal.
