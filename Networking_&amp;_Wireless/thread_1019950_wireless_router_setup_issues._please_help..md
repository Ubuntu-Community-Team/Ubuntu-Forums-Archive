### Post by dmizer on 2008-12-23
It would be helpful if you provided the model number of the router.

Most (if not all) routers come with a web interface. Just browse to the router's ip address, log in, and configure at will. You'll, most likely, have to physically connect (wired) to the router in order to accomplish the setup.
