### Post by rbradt on 2008-10-14
Both Hamachi and gHamachi come with README files that give a decent explanation on how to properly install the applications.

As far as filesharing, I've had luck with that by creating shared folders.  I simply just right click the folder and click on "Sharing Options."  I haven't had much problems with systems on my Hamachi server seeing and editing those folders.
