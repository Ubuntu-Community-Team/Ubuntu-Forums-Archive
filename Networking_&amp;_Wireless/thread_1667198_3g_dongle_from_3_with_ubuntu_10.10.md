### Post by David_UK on 2011-01-14
Just to document this for anyone querying it:

Ubuntu 10.10
Mobile internet dongle from '3' (in UK).

Very simple.  Buy the dongle, insert SIM.  Plug into laptop.  Wait a moment.  Nothing appeared on the desktop when I did this - other users report they had to unmount the dongle.
Right-click the network manager icon and 'edit connections'.
Select mobile broadband tab.  Click 'Add'.
Under 'Create connection for...' the device is listed.  If it says 'any device' it is not recognised.  Click forward.
Select UK, 3, and then keep the default 'internet' at the next 3 stages.  
No need to enter username or password.  Basically just accept all the defaults.
The connection is now listed when left-clicking the network manager icon.
