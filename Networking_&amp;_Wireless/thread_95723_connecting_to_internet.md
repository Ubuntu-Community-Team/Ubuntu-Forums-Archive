### Post by greenway on 2005-11-27
First thing you should do if your NIC is up and running; go to System>Administration>Networking and check if your network card is listed here.

If so, check if it is set to "active" and if it has the right ip settings (static ip or DHCP).

start a terminal and run the command "ifconfig" and post the output in this thread.
