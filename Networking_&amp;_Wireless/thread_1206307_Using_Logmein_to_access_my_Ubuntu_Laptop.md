I found this on LogMeIn's site: [https://secure.logmein.com/labs.asp](https://secure.logmein.com/labs.asp)

> The Linux browser plugin allows you to access LogMeIn computers from any Linux-based, LSB 3.1 compatible distribution, including:

    * Asus Eee laptops
    * Ubuntu 7.10 and above
    * OpenSUSE 10.3 and above
    * gOS
    * etc.

A 32-bit x86 version of Firefox 2.x or 3.0 is required.

It's on the "labs" page so they say: > this is pre-release software and is in no way meant to be used in a production environment.
