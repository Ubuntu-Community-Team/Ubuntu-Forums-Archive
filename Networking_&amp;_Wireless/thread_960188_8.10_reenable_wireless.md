### Post by Sylchat on 2008-12-11
Hi,

Same problem here, when the switch is turned off on Ubuntu start (8.10), I can get the wireless up even after switching to ON.

The "Enable Wireless" menu in the Network Manager was greyed out.

But I could solve it by doing:
```
sudo ifconfig wlan0 up
```

Hope this solves the issue.
