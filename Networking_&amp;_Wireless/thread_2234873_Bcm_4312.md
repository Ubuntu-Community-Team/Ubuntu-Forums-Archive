### Post by kc1di on 2014-07-17
you need to install the following driver bcmwl-kernel-source  you can do that by running the following commands in a terminal.
```
sudo apt-get purge firmware-b43-installer
sudo apt-get install bcmwl-kernel-source
```

The first one may come back not found, just go on to the second command if it does. when it's finished installing reboot, Wireless should work ;)
