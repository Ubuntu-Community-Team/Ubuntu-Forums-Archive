### Post by chili555 on 2012-08-19
If you are using Network Manager, the best way is to click the NM icon, edit connections and change IPv4 settings to DHCP (addresses only). You will then be able to specify your unique DNS nameserver addresses. Please see attached.

I recommend you research faster nameservers with *namebench*, available in the Ubuntu repositories.> DESCRIPTION
       namebench  searches the fastest DNS servers available for your computer to use. namebench runs a fair and thorough  benchmark  using  your  web browser  history, tcpdump output, or standardized datasets in order to provide an individualized recommendation.You can check the DNS nameserver used with:```
nm-tool
dig www.google.com
