### Post by TheFu on 2016-09-02
Using ssh or something else?  BTW, ssh is the best choice.
Did you setup external DNS, so you'll be able to find the WAN IP from anywhere in the world?
Did you open a firewall port, so traffic can get from the outside to the inside?
Did you forward that firewall port to an internel static IP where the new system is?

If using ssh, did you secure the connections so that only key-based authentication is used and so that brute force attacks are blocked after 3 attempts?

In short, have you setup anything to make this work?
