### Post by AVanover5 on 2011-01-10
First, try using a different device or OS to determine if it's the router or Ubuntu. Then, try using a device closer to the router to see if it makes a difference. Some routers have an "extended range mode" which is disabled by default. Also, check whether your authentication is WEP, WPA, or WPA2. Does your device connect when it's set to open authentication? Maybe a hard reset on the router as well.
