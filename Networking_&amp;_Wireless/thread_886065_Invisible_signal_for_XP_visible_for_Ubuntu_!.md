### Post by pytheas22 on 2008-08-10
Probably the wireless drivers that Ubuntu is using give a stronger signal strength than the ones on Windows (which is actually rare, but is becoming more and more common as Linux wireless drivers improve).  There could be other explanations, though--maybe the networks that you see in Ubuntu are operating on a frequency that the Windows drivers don't support.  Or the networks could use hidden ESSIDs, which the Linux utilities tend to be better at detecting automatically (not broadcasting your ESSID doesn't really hide much).
