### Post by Autodave on 2017-08-30
I am assuming that this is a laptop. If it is not, ignore this post.

First off, have you tried connecting to another wifi network to make sure that yours is functioning?

Secondly, it is not real uncommon to see that failure in laptops. :-)

Again, if this is a laptop, try this: it is easy and certainly won't break anything else: (I have fixed MANY laptops with this procedure.)

  	 	 	 	   1. Power down and disconnect power cord.
2. Unplug all peripherals.
3. Remove battery.
4. Wait 10 minutes.
5. Reinstall battery.
6. Connect power cord.
7. Boot computer and test to see if condition is gone.
8. If good so far, now you can reconnect peripherals.
