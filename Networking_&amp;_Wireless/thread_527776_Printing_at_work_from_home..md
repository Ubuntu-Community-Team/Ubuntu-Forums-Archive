### Post by ihristov on 2007-08-17
Depending on the protocol the printer supports you might be able to publish it via the Internet. You need to give more specific details to determine if this is possible. Security becomes an issue in this case.

I would recommend VPN since it's much more secure and has many more uses. Setting up VPN can be quite easy if you use IPCop or a Linux box for your firewall. OpenVPN is the way to go if you ask me.

BTW: Ease of use is almost always in contradiction with security.
