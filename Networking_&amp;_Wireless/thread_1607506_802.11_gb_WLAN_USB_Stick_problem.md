### Post by chili555 on 2010-10-28
> Bus 001 Device 003: ID 124a:4023 AirVast Your device is supported by the module p54usb that is already built in to the kernel. It requires firmware which is *not* built in. If you can get an ethernet connection and go on line, you can download and install the firmware with:```
sudo apt-get install linux-firmware-nonfree
```Then remove and re-load the module and your stick should come to life:```
sudo rmmod p54usb
sudo modprobe p54usb
```Post back if it doesn't go as smoothly as expected.

### Post by chili555 on 2010-10-29
Ooops! Sorry about that. I forget that the module is not installed by default. It's been a few years since I did a clean install.

Let's find out if the needed software is installed. In a terminal, do:```
ndiswrapper -v
```It will report a version that's installed or report that it's not installed.

If it is not installed, drop your Ubuntu installation CD in the tray and do:```
sudo apt-cdrom add
sudo apt-get install ndiswrapper-utils
```A few dependencies will be added. 

Now, do:```
cd Desktop/cookie6
sudo ndiswrapper -i AWLGTNIC.inf
```Next, check that it's installed correctly:```
ndiswrapper -l
```(l is for 'list.') If it is working correctly, you should see:

```
Installed ndis drivers:
		AWLGTNIC  driver present, hardware present
```Next, do:```
sudo depmod -a
sudo modprobe ndiswrapper
sudo ndiswrapper -m
sudo iwlist wlan0 scan
```Any improvement?
