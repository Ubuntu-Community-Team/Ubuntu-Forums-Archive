### Post by i-Buntu-2 on 2007-03-24
I was getting a soft lockup error on my Ubuntu 6.06 Dapper server.

After removing all my peripherals (memory, CD / DVD drives, floppy drive etc.) I was still getting the same error on boot.

I re-installed Ubuntu on a spare hard drive to see if this would rectifiy the problem but nothing changed.

Finally, I went into the BIOS and applied the "Set-up Defaults" (Pressing F9 in my BIOS control screen) and re-booted.

This solved the problem for me and my server has been up and running ever since ... sweet as a nut.
