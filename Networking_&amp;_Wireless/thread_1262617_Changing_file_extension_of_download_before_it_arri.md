### Post by t0mppa on 2009-09-10
Right clicking the link and choosing *Save Link As...* will let you specify some other extension for it, but it will still be in cab format. Size you can't really change, as that would simply mean either not downloading the whole file or then adding some additional data to it, which would probably end up breaking the format of it and then you'd have a lot of trouble opening the file.
