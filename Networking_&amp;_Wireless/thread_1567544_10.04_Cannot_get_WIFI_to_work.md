### Post by DeanHeathen on 2010-09-04
Wow, so lots of views and no replies. Good news, though! I did find a workable solution. 
 
open the Package Manager:
Make sure you uninstall network-manager, and if you installed this one, network-manager-gnome, you have to uninstall that as well. 
Install wicd.
And the problems end there. 
 
Some say the connection randomly drops, but I've been using mine for at least 8 hours and haven't run into that yet. 
 
I hope this helps someone else. Now on to getting dual boot to work in GRUB... (that's my current nightmare).
