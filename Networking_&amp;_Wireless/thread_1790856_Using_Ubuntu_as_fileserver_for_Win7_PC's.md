### Post by southkhaki19 on 2011-06-26
Hey! Well I am actually interested in this exact same theory. I know that in the past it was a bit more complicated to set this up but with the new v11 Ubuntu there is a simpler way. What you are looking for is SAMBA share. There should be some options for it within the network settings I believe. I know that the server version supports SAMBA share (simple file sharing to Windows Clients) since I actually seen the description in the "Windows Compatible" on the Ubuntu Site. I have not actually played with Ubuntu before but I do like to read a lot online. I finally downloaded it and will be loading up tonight. The only thing that I am unsure of if it is only the "Server" version or both of them come with the SAMBA built in.
