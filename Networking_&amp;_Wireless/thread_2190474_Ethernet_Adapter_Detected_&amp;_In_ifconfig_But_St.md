### Post by steeldriver on 2013-11-27
You have a typo in your /etc/network/interfaces file, which is likely what is stopping the interface from receiving a IPv4 address from your DHCP server

```

# The primary network interface
auto eth0
iface **eth[COLOR=#ff0000]o[/COLOR]0** inet dhcp

```

See if it works after fixing that and restarting the networking service - if not, post back and we can do some more troubleshooting
