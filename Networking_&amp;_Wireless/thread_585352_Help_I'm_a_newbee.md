### Post by bswilson on 2007-10-21
If your Network Manager icon shows a red X, I suspect that means one (or more, if you have more) network interface is not configured.  When you click on that icon, do you see the Network Manager box appear, which allows you to configure the interface?  See my attached sample screenshot.

Assuming you do, is at least one interface enabled?  Is it configured to use an Automatic Configuratino (e.g. DHCP)?
