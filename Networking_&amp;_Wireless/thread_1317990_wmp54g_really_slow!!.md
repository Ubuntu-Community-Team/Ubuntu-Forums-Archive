### Post by papangul on 2009-11-07
After installing it you will find a tray icon. Click on it and you will get the configuration options, including dns entry fields. Here are some relevant info:
[http://ubuntuforums.org/showthread.php?t=1150797](http://ubuntuforums.org/showthread.php?t=1150797)

On karmic you install it in usual way: sudo apt-get install wicd

My suggestion is not to use the hidden essid option in your router when you try it for the first time.
