### Post by leandromartinez98 on 2008-05-15
I changed from NM to wicd because I had several problems with the former. However I found the icons of wicd horrible. So I made some NM-like icons for wicd, with ubuntu colors. They are very simple, but I like them better.
To use tem:

tar -xvzf wicd-icons.tar.gz
cd wicd
sudo cp * /opt/wicd/images
