### Post by klemes on 2010-10-07
All you got to do usually to connect is plug your windows phone int a free usb port and after that activate in your phone Internet Connection Sharing via USB (you have to specify that).Your Linux box will see it as an other ethernet connection (ie eth1)through rndis driver and you will be connected to the Internet.At least this is the case with my mobile phone if this is not the case for you you would better provide some more information in order for others to be able to help more.:popcorn:
