### Post by az on 2005-03-24
"hi i am new to ubuntu and would like some help configuring firewall."

Firestarter, Shorewall, Guarddog.  There are many more. 


"i also would like to know how to get a static ip address."

You pay for one from your isp.  If that is not what you mean, look into dyndns.

[www.dyndns.org](www.dyndns.org).  
Install the ddclient package (apt-cache search dndns, since there are other dyndns clients available)  to use your dynamic dns.  You can sign up for free.
