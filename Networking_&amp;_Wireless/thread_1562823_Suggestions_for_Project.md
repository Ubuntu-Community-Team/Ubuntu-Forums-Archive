### Post by BkkBonanza on 2010-08-28
If gaining useful practical experience is the goal, then why not setup a web server / mail server.

It's not very esoteric but the skills gained would be of solid use in future. 

Apache (or Nginx is much cool), PHP, MySQL. For bonus points, install and setup OpenVPS to learn how to config and run a VPS hosting solution.

Give everyone a web site dev project after and they can learn html/js/css/php etc.
