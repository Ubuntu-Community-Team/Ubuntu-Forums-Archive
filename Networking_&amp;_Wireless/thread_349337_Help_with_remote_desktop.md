### Post by bustanil on 2007-01-30
You can control your Windows XP desktop from your ubuntu using **vncviewer**. Download and install [vnc server]("http://www.tightvnc.com") on your Windows XP. Then run this command from your ubuntu:
vncviewer <xp_ip_address>:5900.

Hope this helps.
