### Post by plunder on 2010-01-19
With the growing complexity of the Aircrack Supplement Script, I've decided to release it under a new name "Airbourne". The name coming from the fact that it basically scripts aircrack-ng tools and it being scripted for the Bourne-again shell, otherwise known as BASH.

Some of the new features that are currently implemented:

    * Remembering target information including

        * ESSID
        * BSSID
        * Channel
        * Associated Station
        * MAC Randomization
        * MAC changer

    * Working on a session based environment - all of your data is grouped for a particular session allowing you to continue where you've left off
    * Partial implementation of modes - modes will be tty and x11, currently x11 is the only fully implemented, tty mode will come with v1.5
    * Completely restructured code to allow module based add-ons

More features will be implemented in coming versions, as of now some bugs will need to be squashed. Help test Airbourne by visiting our [repositories]("http://codecaucus.com/downloads/repository/scripts/shell/airbourne-v1.0/") and reporting any bugs in the Airbourne support forums. Lastly, I want to thank those who've clicked my ads, made donations and help support further development of this project.
