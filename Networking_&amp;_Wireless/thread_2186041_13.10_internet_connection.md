### Post by Rob Sayer on 2013-11-09
I'm not one of the network experts here by any stretch of the imagination, but I can't see any reason to assume you needed to remove bcmwl-kernel-source installed or had it in the first place.  That package AFAIK is the proprietary broadcom drivers, which you'd install in "additional drivers", which I actually don't do that way anymore.  I use synaptic instead.  

And, believe it or not, the proprietary drivers don't necessarily work as well as the open source linux ones.  That's been the case in 13.10 for my broadcom wireless adapter, which is a funny little beast.   It says it's a bcm4313 but it's actually 4727 (rev 01).  The "14e4:..." is what I mean.  Yours is 14e4:4311, which is different from mine.

To see what open source driver is actually in use post the output of:

```
sudo  lshw -C network
```

The wireless driver is under wlan0.

Hopefully someone who can actually diagnose this will read it.
