### Post by Brzhk on 2009-02-20
Hello,
i had quite a lot of problems trying to connect to one of my invisble networks. I discovered that if you have saved multiples configurations for invisibles networks using network manager, it fails to use the right one:

if i select to use the second network, it will still try to connect to the other one.

the workaround is to have only one wifi configuratiopn...

WARNING: you'd better be able to remember your details to write your configurations before deleting them. Don't rant if you forgot your WPA password and stuff !

here's the bugreport if you can add any info:
[http://bugzilla.gnome.org/show_bug.cgi?id=572539](http://bugzilla.gnome.org/show_bug.cgi?id=572539)
