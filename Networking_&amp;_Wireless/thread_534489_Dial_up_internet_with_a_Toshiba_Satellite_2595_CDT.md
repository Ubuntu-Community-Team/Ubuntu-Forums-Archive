### Post by jml on 2007-08-25
A little more information would be helpful.  What model of Toshiba do you have and can you list the technical specifications of the modem.  The reason that this is important is two fold.  On older laptops not all modems are supported in Linux so which one you have is important.  

Second, most modern laptops don't actually have true hardware modems in them, but have the so called "Winmodem"  This is actually not a true modem, but simply a connector between the phone line and your computer.  The modem functions are carried out in software (Windows software,) not in hardware.  If you have one of these, then search this forum for the term winmodem, for some help.  Also, here is a link to a site that is dedicated to making Linux work with Wnmodems:

[http://www.linmodems.org/](http://www.linmodems.org/)

Hope this helps.
