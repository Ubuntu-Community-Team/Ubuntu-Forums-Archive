My builtin wireless card does not work all that well.
I installed a pcmcia (wlan1)card which works great. I used it without a hitch with wicd. However, when I reinstalled network manager, it only wants to use my built in card wlan0. 
How do I use network manager to select wlan1 over wlan0?
Thanks

### Post by crossedout on 2011-03-21
If you still have the applet on, you can right-click it and select Edit Connections.  From there you can manually set it up, choose 'Add' and setup what you need.

From the start menu -> Preferences -> Network Connections -> Add

Good luck,
-Xout
