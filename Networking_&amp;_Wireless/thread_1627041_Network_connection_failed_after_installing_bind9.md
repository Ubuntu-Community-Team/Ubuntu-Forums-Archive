### Post by TuesdayDowns on 2010-11-21
Ok, apparently having 192.168.0.1 as a nameserver WAS the issue, ubuntu was replacing it as soon as I removed it, I solved the issue with:
```
sudo chattr +i /etc/resolv.conf
```
