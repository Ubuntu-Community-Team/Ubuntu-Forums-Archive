### Post by maestrobwh1 on 2009-09-03
This package is supposed to be for Ralink devices, but I have found it works for other wireless interfaces (i.e. intel, zydas, etc) which show up as wlanX or ethX.  The GUI is simple, yet seems to be as powerful as wicd or network manager.  The package is very small and didn't seem to want to pull in extra dependencies so that is a bonus.
