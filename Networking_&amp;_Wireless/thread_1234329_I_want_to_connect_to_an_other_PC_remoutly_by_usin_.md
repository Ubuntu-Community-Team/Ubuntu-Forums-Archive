### Post by cgb on 2009-08-07
If you want to use the built in VNC server you can enable it on the computer you want to remote into by going to System/Preferences/Remote Desktop and switching it on.  

Then on the computer you want to connect from you will go into Applications/Internet/Remote Desktop Viewer and you should see the other computer in available connections.
