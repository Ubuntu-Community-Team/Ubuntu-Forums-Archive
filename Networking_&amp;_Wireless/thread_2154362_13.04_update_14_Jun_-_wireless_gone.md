### Post by praseodym on 2013-06-14
Obviously, the wrong driver is loaded, iwlwifi for Intel cards. Blacklist it and install the missing firmware for the native driver of your Broadcom device:
```

echo "blacklist iwlwifi" | sudo tee -a /etc/modprobe.d/blacklist.conf
wget media.cdn.ubuntu-de.org/forum/attachments/56/18/5007272-Broadcom_Firmware_070513.tar.gz
sudo tar xvf 5007272-Broadcom_Firmware_070513.tar.gz -C /lib/firmware 
```
Try to load the driver by hand:
```

sudo modprobe -rfv iwlwifi
sudo modprobe -v bcma
sudo modprobe -v brcmsmac
```
Check the black- and whitelists:
```

egrep 'bcma|brcm' /etc/modprobe.d/*
cat /etc/modules
```
