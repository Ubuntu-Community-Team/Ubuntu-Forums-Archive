This is Bubba: [http://www.linux.com/article.pl?sid=07/04/05/143233](http://www.linux.com/article.pl?sid=07/04/05/143233)

Basically this is a very small (think two hard drives stacked on each other) LInux (Debian Sarge) printer server, email server, etc., networked storage device that can do lots and lots of things. It plugs right into a router and is fully configurable from a web browser. It can handle FTP and BitTorrent downloads (although I don't know how it would do that, being the Linux noob I am).

It also starts at 240 Euros or about 200 pounds (roughly 375 dollars?), so is pretty expensive!

Apart from the fact it'll be really really hard to replicate this at the same size :) how would someone go about doing basically this same thing with some old PC parts? It only has 64mb of RAM and has only a 200mHz CPU, so theoretically my leftover P3/350 could handle the job...

Any ideas?
