This is a print of /etc/network/interfaces which worked fine earlier today when I was happily using 7.04
Then I decided to update and did just that, rebooted and poked around before trying to use the net, I like some of the improvements already!
BUT soon discovered for whatever reason that the inclusion of WPA support ootb has paradoxically killed my setup!
I've spent the entirety of the day trying to get it back up using the new WPA settings but to no amends.
There are rumours that removing network-manager will help, but of course without the interweb i can't get it back if it fails!


auto ra0
iface ra0 inet static
address 192.168.0.13
netmask 255.255.255.0
gateway 192.168.0.1
pre-up iwconfig ra0 mode managed
pre-up iwconfig ra0 essid "Network 65"
pre-up iwpriv ra0 set EncrypType=TKIP
pre-up iwpriv ra0 set AuthMode=WPAPSK
pre-up iwconfig ra0 essid "Network 65"
pre-up iwpriv ra0 set WPAPSK="anoushka"
