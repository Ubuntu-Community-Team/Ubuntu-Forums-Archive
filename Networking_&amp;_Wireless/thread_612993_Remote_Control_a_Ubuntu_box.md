### Post by MakotoTheKnight on 2007-11-14
Yes, there is.

Controlling a Ubuntu box from a Windows box can be done by using [Ultra VNC]("http://www.uvnc.com/").  Of course, you have to have it enabled in System->Preferences->Remote Desktop.

My advice is to put in a password as well, just in case.  You can never be too safe.
