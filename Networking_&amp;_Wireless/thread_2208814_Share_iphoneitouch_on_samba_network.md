Hello,

I'm using Ubuntu 12.04 and can see my jailbroken iPod Touch connected as a "device" from the unity bar. I am able to read and write files to the ipod file system with no issues. What I'd really like to do, though, is include it in my samba home network so I can access the file system from other computers on the network. I am able to configure "drives" from the samba gui, but there doesn't seem to be a way to add a "device."

Is there a way around this? I tried using 3rd part software to mount the ipod as a usb drive, which works, but that drive is not accessible from ios.

### Post by eagle3ye on 2014-03-02
I still haven't found a fix for this, but I found another solution that works for my purposes. I simply downloaded the Samba app by Big Boss from Cydia (requires jailbroken ipod/iphone). This app allows you to share your entire ios device's filesystem over your home network via samba share. Once the app is installed, you just have to remember to make it visible over your home network from your idevice's general settings under "Samba."
