### Post by praseodym on 2013-05-13
The LAN driver is already part of the kernel. Load the driver and make it permanent:
```

sudo modprobe -v b44
echo b44 | sudo tee -a /etc/modules
```
Check:
```

lspci -nnk | grep -iA2 net
lsusb
lsmod
cat /etc/resolv.conf
ifconfig -a
```

### Post by Hadaka on 2013-05-14
Hi, you have loaded a wireless driver that is
blacklisting your wired connection, please do,,

```
sudo apt-get remove --purge bcmwl-kernel-source
sudo modprobe b44
```

your wired connection should now be working..

to load the correct wireless driver...please do..

```
sudo apt-get install linux-firmware-nonfree
sudo modprobe b43
```

report back any errors,
thanks.
