### Post by newlinux on 2007-05-17
Well, if you must have the USB print server feature between these two get the MSI. I have 3 Linksys WRT54G (one is a WRT54GL). All flashed with dd-wrt firmware which makes them very feature rich and robust. Never have a problem with them. The earlier versions of the WRT54G and the current WRT54GL are easy to flash with better third-party firmware. Just my two cents since I do own one of the models you are looking at. I don't know anything about the MSI, but the USB print server feature is useful.

You could just make one of your computers a print server. To save on energy costs, I bought a cheap used print server off ebay for $15. I flashed my router for some of the features that the stock Linksys firmware didn't come with:

[http://www.dd-wrt.com/wiki/index.php/Main_Page](http://www.dd-wrt.com/wiki/index.php/Main_Page)
[http://www.dd-wrt.com/wiki/index.php/Supported_Devices](http://www.dd-wrt.com/wiki/index.php/Supported_Devices)
[http://www.dd-wrt.com/wiki/index.php/Linksys_WRT54G/GL/GS/GX](http://www.dd-wrt.com/wiki/index.php/Linksys_WRT54G/GL/GS/GX)

There are others third party firmware packages that can be flashed. Make sure to look at the instructions for flashing different routers (and different versions of routers) and which routers are compatible.
