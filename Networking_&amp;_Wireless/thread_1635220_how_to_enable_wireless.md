### Post by grahammechanical on 2010-12-01
How did you enable the wifi in Windows? I have heard that when Windows shuts down is deactivates (switches off) the wifi adapter. I think that not having the wireless adapter powered up all the time extends battery usage. So, you need to press the switch on the keyboard or the combination of the Fn key + a function key. (The ones with F and a number). It takes a few seconds for the wireless adapter to power up and make a connection to the modem.

If this happens again or you want to experiment and the enable wireless is grayed out, try disabling networking and then enable networking. Switching networking off and then on might activate the wireless side of things.
