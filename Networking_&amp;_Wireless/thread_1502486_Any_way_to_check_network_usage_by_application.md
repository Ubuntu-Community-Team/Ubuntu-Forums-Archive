### Post by scorp123 on 2010-06-05
> **googleworldblog said:**
> Just wondering, is there any way to check how much bandwidth is being used by each application in Ubuntu? Look into **ntop** ... It can monitor your host's network activities and generate nice graphical charts so you get an idea what process has been talking to what host on the network and by what protocol ... 

The "ntop" web site has some nice screenshots so you get an idea what this thing does and how its reports look like:

[http://www.ntop.org/overview.html](http://www.ntop.org/overview.html)
