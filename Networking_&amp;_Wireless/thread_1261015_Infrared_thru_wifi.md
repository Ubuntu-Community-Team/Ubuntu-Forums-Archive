### Post by t0mppa on 2009-09-08
Infrared is (invisible to human eye) light for transmission and reception of data, also used in TV controllers. Thus in order to catch the data, you need an infrared port in the computer. Wireless chips aren't capable of catching the light itself. Older laptops used to have these ports, but they've been mostly dropped since it's a rather slow way to transport data and you need to have that unobstructed connection between the two devices all the time for it to work.

If you really need the data though, you should be able to find a IRDA device that you can hook up through an USB port.
