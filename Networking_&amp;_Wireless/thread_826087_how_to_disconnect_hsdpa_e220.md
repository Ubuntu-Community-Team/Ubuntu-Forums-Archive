### Post by aquavitae on 2008-06-11
There should be a network icon in the top-right corner of the screen (in the system tray). If you click (right-click? I can't remember which) you should be able to disconnect.

To see how much you've loaded, add a network monitoring applet to the panel. Right-click on the panel, and go to Add Applet. Look through the list it brings up for something that monitors network traffic and select it.

Hope this helps!
