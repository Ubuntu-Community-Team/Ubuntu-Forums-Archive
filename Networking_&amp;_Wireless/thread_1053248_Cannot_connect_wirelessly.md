### Post by Metaljaz on 2009-01-28
This is what worked for me and my Dell Inspiron 9100 Laptop:

You need to be wired for a short period to do this. If you can't
get a wired internet connection the file is on the install CD.

Go to System>Administration>Synaptic package manager
Do a search for b43-fwcutter click to install and install it. After you
install reboot and the firmware should now be available under 
System>Adminstration>Hardware drivers.

Post back if you have problems.
