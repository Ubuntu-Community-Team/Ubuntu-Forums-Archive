### Post by SeaJayEmm on 2009-12-30
To the original poster, you need the rt2870.inf file as well as the rt2870.sys file from the winxp 64 driver for it to install properly in ubuntu. First you will need to hard wire into your router and update your system and install ndisgtk via synaptics package manager or in terminal by typing "sudo apt-get install ndisgtk". Then install with ndisgtk, reboot and you should have wireless connection. To the other guy the rt2870sta does not work with this and you have to blacklist it, and it should work properly afterwards. If anyone needs the rt2870.inf and the rt2870.sys email me and I will send them to you.
