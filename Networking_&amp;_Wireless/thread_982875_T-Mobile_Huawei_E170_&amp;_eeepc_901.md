### Post by Plumtreed on 2008-11-15
I use *vnstat* which is in system>administration>synaptec package manager 

Google *vnstat* for more info. 

You have to use the terminal once installed but I found I only use it very occasionally so that part is not a problem. It is very flexible and user friendly. I also find I rely on my 'provider' 3, they tell me how much I use.
