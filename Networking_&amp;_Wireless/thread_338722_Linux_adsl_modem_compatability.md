### Post by handy on 2007-01-16
The worst thing that can happen (famous last words :) ) is that your modem/router is windoze centric & doesn't deal with IPv6 properly, which can cause it not to be able to browse the web, & you can have a DNS problem which stops you from being able to use the Ubuntu repositories & other internet facilities such as gaim.

Both of these problems are realy easily fixed, so don't stress.

If your modem/router gives you either or both of these problems (like mine ;) ) then have a [_**look here**_]("http://ubuntuforums.org/showthread.php?t=282034") for the very easy fix = IPv6, & the not much harder DNS fix.

Linux natively supports far more hardware than windoze or any other OS! :KS
