### Post by grahammechanical on 2011-02-15
Right click the icon and select Edit Connections. Select the Wireless tab. Select your wireless connection and click Edit. Remove the tick mark in the box Connect Automatically.

So, you have two ways to do this. Clicking Enable Wireless will active the wireless adapter and it will start to scan for a network. Should you need to connect wirelessly.
