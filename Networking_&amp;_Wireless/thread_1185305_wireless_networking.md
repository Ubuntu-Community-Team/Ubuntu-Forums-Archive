### Post by jobsworth on 2009-06-12
Hello All,

Thank you to the powers that be for fixing a very big bug in Ubuntu.
Since the latest upgrade of Ubuntu from 8.04 to 9.04 wifi cards ceased working. They all worked great with Ubuntu 8.04.
Wifi would install with great difficulty and sometimes appear to connect but then it would be impossible to connect to the internet.
Either the startup would freeze or the system would freeze later.
I have 3 wifi cards ,Broadcom, the generic name for Intel Centrino, on one laptop, Dlink on one PC and Zyxel on another PC.
I have been downloading the Ubuntu updates regularly.
I have spent a lot of time, effort and money on LAN cables to get my systems to work.
We have been told repeatedly that NDISWRAPPER is no longer necessary and 
that NetworkManager is "Pain-Free Networking".
No matter which windows driver I installed none of my wifi cards would work.
Now they all do.
Thank you to whoever did this but please could you not have told us all that there was a problem.

John
