### Post by tturrisi on 2006-08-26
First thing to consider for wifi is NOT what card to get but what chipset the card uses.  Then decide on what card.  Easiest best most feature packed chipset in use today for linux is the Atheros chipset.  This chipset is available in may different cards.  Trouble is that manufacturers have a tendency to change chipsets while retaining the name of the card.  All they change is the version # of the card.

To use Atheros chipset cards with Ubuntu you will need to install the linux-restricted-modules package that matches the kernel you have using synaptic package manager.

Here's a great deal on a real good wifi card that works with Ubuntu:
