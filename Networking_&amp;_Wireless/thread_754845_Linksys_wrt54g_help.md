### Post by kstella on 2008-04-14
Hello. I've moved back home for the summer and want to hook my Gutsy-running PC up to the Internet. My parents' computer, running WinXP, is hooked up to a Linksys Wireless-G WRT54G router. I've connected my computer to the same router using a LAN cable, and the signal's detected.

How do I configure my computer to access the Internet? A step-by-step would be greatly appreciated. :)

---

### Post by kstella on 2008-04-14
Regarding IPs, the Windows PC is set for "Obtain IP automatically," while Firefox is set for "Direct connection to the Internet."

---

### Post by chili555 on 2008-04-14
System-->Administration-->Network. Tick the box next to 'Wired Coonection.' Select 'Properties' and tick 'Enable This Connection,' select DHCP. Save and close and you should connect.
