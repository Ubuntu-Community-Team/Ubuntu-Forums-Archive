### Post by zachbb on 2009-01-20
Is there a way to force ubuntu to connect to a specific network at startup? My computer keeps connecting to a network at the other side of my house, when I want it to connect to the one in the same room as the computer (which has better signal strength)!

I have been told that Ubuntu remembers the last network that it was connected to. However this is not true.... It ALWAYS connects to the wrong network and I change it every time I turn on the computer!

---

### Post by Kobalt on 2009-01-20
Did you try to right-click NetworkManager, select Modify Connections. Go into wireless, and from here specify only the wireless network you want to connect to automatically, or modify other networks by un-selecting the "connect automatically" option.
