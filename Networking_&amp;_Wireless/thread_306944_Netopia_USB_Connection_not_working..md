### Post by marianlibrarian on 2006-12-03
I never got the Netopia USB connection to work. I wnt to walmart and bought a linksys network card for 14 bucks. I plugged it in and got no errors. Then I had the book "Beginning Ubuntu Linux" and on page 81 it tells how to configure a network card via DHCP. I followed the instructions and got no errors. The rest of my problems were resolved when I realized that I had to log into my centurytel account by going to 192.168.1.1 and logging in. As soon as I did that, it tested my ethernet connection and it passed along with all the other tests. 

It is working and I am posting this using the old computer ready for the trash heap that would never run windows xp but is now running with Ubuntu 6.06 and doing wonderful. - Thanks to all the patient keepers of this forum for helping me get this far. :)
