I've just been re-reading my posts and thought I would add to this one (hence the long time between thread creation and this reply).

Anyway,  the software I use for scanning wireless networks is called

inSSIDer

I strongly recommend this bit of software.  It displays all the wlans in range and shows them in graph format so you can see the overlapping channels.
