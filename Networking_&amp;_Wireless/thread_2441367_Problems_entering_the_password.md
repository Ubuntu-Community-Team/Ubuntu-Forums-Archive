### Post by CelticWarrior on 2020-04-23
Yes, there is. 
Just follow the best practices for all OS families and do NOT use autologin. In any desktop Linux distro if you enable autotlogin you'll be asked the password anyway later for anything that requires access to the passwords keyring like when you try to connect to an AP that is password protected.
