I've been troubleshooting my LinksysWMP54G hardware for weeks.  My card requires the RT61 driver, and I've had no luck at all.  Below is a link to ralink website that provides support for the rt2x00 and rt61 drivers.

[http://www.ralinktech.com/supp-1.htm](http://www.ralinktech.com/supp-1.htm)

The software has only been tested with I think it was Fedora.  Anyway, there still might be hope for you there if your card uses RT2x00 drivers.  I've given up the fight with ubuntu and internet.  I repartitioned my hard drive and found software that allows windows to exchange files between fat32 and ext2 file system.  This way I can enjoy gnu software, practice with Linux, and hope that some day the guys working on ralink drivers will include working linux drivers for rt61.
