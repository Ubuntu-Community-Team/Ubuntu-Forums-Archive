### Post by donpaolo on 2007-08-11
You connect the two machines (via network cables connected to the same network switch, or with a crossover cable), give them the respective IP address belonging to the same family, and then on the desktop you enter the network configuration, and set the gateway as the IP of the laptop.

All should work.
