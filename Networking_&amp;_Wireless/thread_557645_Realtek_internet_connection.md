### Post by MadMac on 2007-09-22
I have a Realtek 8139 network card (on motherboard) and just did an update to the new driver while I was working in Windows.  When I went back to Ubuntu, I was not able to connect to the internet.  As there are three computers connected to my router, I was easily able to determine that it was my computer and not the internet connection.

While in Windows, I did a search for "internet connection" in the forums and found a message that told how the updated drivers messes up the 8139 Realtek network card.
(Of course, now, I can't find that message to save my life!  ](*,)   Par for me, anyway. )

So, I went back to Windows, rolled back the driver update (to a driver from sometime in 2006), rebooted back into Ubuntu and, waa-laa, I now have internet in Ubuntu.

Maybe, if anyone has a problem with internet connection and uses a Realtek 8139 card, this will help them.
