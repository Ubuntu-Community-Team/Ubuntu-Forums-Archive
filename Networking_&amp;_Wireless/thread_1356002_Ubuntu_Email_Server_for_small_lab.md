### Post by shairozan on 2009-12-15
Hey there, 

If you're comfortable with bind, I would recommend setting up Zimbra. Once you have Bind working and it recognizing your mail server as being the valid MX record, then the install is a breeze on the 8.04 LTS. 

From there it's all web ui. It's a lot easier for most people to deal with than working on postfix configs.
