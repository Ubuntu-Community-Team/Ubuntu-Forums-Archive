### Post by SoftPops on 2009-08-24
> **Kaninepete said:**
> I don't have the password for the router I'm using written down, but I know that it is still stored in my computer (as I can still access the internet). Is there any way that I can display my password?
 
If this is to get the WII online I would have thought the password required is the WEP/WPA key rather than the router password. Assuming you are using a wireless connection to the internet at the moment the you should be able to get this by right clicking the network connection icon on the "applications, places, system" panel. Select "Edit Connections", followed by the "Wireless" tab. Edit the connection you are using (you will need admin password), select the "Wireless Security" tab and tick the show key box. This should display the key/password you are looking for.
 
Setting the WII up with the same settings as displayed should let you in UNLESS there is additional security set on the router in which case you will, unfortunately, also require the router password.
 
Hope this helps.
