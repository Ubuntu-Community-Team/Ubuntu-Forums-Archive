### Post by varunendra on 2014-03-26
Please try these, one at a time, try next only if required -

**1)** Turning "Power Management" off on wireless interface -
```
sudo iwconfig wlan0 power off
```
Check with "iwconfig" command to make sure it turned off. If it seems to help initially but problem returns again, check "iwconfig" again and report back if Power Management seems to automatically turn on again.

**2)** Trying a parameter available for your driver -
```
echo "options ath9k nohwcrypt=1" | sudo tee /etc/modprobe.d/ath9k.conf
```
Followed by a reboot or manually reloading of the driver with following commands -
```
sudo modprobe -rv ath9k
sudo modprobe -v ath9k
```

**3)** If the above two don't help, try compiling a newer driver as suggested in this post : [http://ubuntuforums.org/showpost.php?p=12963298](http://ubuntuforums.org/showpost.php?p=12963298)

Let us know how it goes.
