### Post by alphacrux on 2008-03-26
I don't know enough about networking to help you here with the real problem though I might be able to give you some workarounds. You could try adding the code: 
```
auto lo
iface lo inet loopback


auto eth0
iface eth0 inet dhcp
pre-up iptables-restore < /etc/iptables.rules
pre-up ifup
pre-up ifdown
post-down iptables-restore < /etc/iptables.rules

auto eth1
iface eth1 inet static
address 192.168.1.1
netmask 255.255.255.0
```
OR you could write a startup script which is what I did to get my wireless working. Follow tictacman's instructions in this [[COLOR="Blue"]post[/COLOR]]("http://ubuntuforums.org/showthread.php?t=352568")
Its about halfway down the page, and your script will just be ```
#!/bin/sh
ifdown eth0
ifup eth0
```
