Hello,

I have just installed Ubuntu server 11.10 on a computer module mounted in a 19-inch rack. The module has two integrated wired Gigabit Ethernet interfaces, eth0 and eth1. eth0 is connected to a DMZ and eth1 to a LAN.

During the installation I followed the manual configuration for the eth0 setting up its IP address, gateway, netmask and DNS server.

Right after the installation a network connection was not present as expected. 

I suspect a hardware failure but I am not sure. Can anybody confirm that?
