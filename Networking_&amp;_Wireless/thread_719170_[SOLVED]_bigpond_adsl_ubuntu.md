I tried to connect to bigpond broadband by adsl, with and without a router in place. Sometimes the UU1.7 DVD found the LAN connection, now and then the installed system would also. I don't know why, but the randomness is no good. Now I have removed the network cable and reinstalled the wg311v2 wireless card, for both Windows XP and ubuntu. After getting help from the Post Install Page site I now have the internet, with the the router and PC a few cms apart. Conceptually I'd have thought the network link would have been straightforward rather than using ndiswrapper. Perhaps Telstra are too obsessed with Microsoft?
