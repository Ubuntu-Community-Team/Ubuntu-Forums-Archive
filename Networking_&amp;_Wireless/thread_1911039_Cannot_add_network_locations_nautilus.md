### Post by Morbius1 on 2012-01-18
>  For the moment I have made a copy of .gvfs to my home folder but that's not the best way i think..gvfs is already in your home folder.

Open gedit
Select File > Save as
Right click an empty space in the "Save As" dialog box
Then select "Show hidden files"

Then you will see the .gvfs folder in gedit and all the other applications.
