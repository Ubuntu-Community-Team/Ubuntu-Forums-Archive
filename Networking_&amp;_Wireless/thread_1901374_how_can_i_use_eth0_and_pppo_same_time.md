### Post by bvkmohan on 2011-12-28
PPPOE is not a physical interface, it is a virtual interface used for regulating data traffic. PPPOE uses existing physical interface to connect. If you say I am using PPPOE it means you ARE using eth0 (considering you have only one physical eth0).
