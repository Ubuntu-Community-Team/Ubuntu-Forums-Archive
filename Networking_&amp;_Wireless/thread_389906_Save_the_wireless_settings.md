### Post by Skidpad on 2007-03-21
I saved two different locations (Home & Work) in my Network Settings interface.  It will remember my settings for both, but if I change locations, then I must select the new location manually.

Assuming you aren't using Network Manager, go into your Network Settings as I described above.  Once you current location is configured, click the little icon with the blue arrow on it to **"Save current network configuration as a location".**  It will remember your settings for your next boot.
