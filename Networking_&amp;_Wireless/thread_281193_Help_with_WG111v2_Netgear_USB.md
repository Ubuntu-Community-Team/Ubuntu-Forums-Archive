### Post by chewbunny on 2006-10-21
About to embark on the installation of a wireless adaptor in Dapper myself.... which sounds like it may be fun. No specific help I'm afraid, but a general pointer from my experience with wireless networks - if things are not going well - turn off ALL security on the wireless router and broadcast your SSID. Then start building up the security layers one at a time. 

Are you using MAC address filtering in the router? If so have you added your MAC for the USB adaptor?

:-# If there's real issues with the security being down. Just disconnect all other machines from the network, and configure the Ubuntu machine up.
