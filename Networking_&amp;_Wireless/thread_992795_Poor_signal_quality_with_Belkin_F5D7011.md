### Post by Phosphoric on 2008-11-25
Finally got my elderly Inspiron 1100 loaded up with 7.10 and it works OK if a little slow.

Installed a Belkin F5D7011 wireless card using the driver that came on a cd with the card, bcmwl5.inf, via ndiswrapper.

Got it all working eventually, but the signal quality is really poor, about 25% just 2m from the router. :(

The same card in the same laptop running Windows XP using the same driver gives 75% signal quality in the same situation.

Any thoughts?  Is it the ndiswrapper that's downgrading the performance?  
One obvious difference is that under Windows XP the green power light is on all the time and the green data light only comes on during transmission.  Under Ubuntu, both lights are on all the time and the data light flickers on and off during transmission.
