After looking for solution to get my D link usb adapter to work i got tired and decided to experiment.
This is how to get your D-link 125 Ubs Wireless adapter working:


1. First you want to download virtual box in Ubuntu

2. Now you must have windows in virtual box to work, put your D-link 150 Wireless adapter installation cd inside your cd drive.

3. (Mount the Cd to Virtual box) Install the software (make sure the Adapter is plugged into the computer) Now after you install the software your Adapter should activate if you followed the installation process correctly.

4. You adapter should connect Via your windows OS in Virtual Box, not you can access the internet with your Adapter on Ubuntu. (You may turn off virtual box) 

5. Enjoy and spread the word :) of this easy solution
