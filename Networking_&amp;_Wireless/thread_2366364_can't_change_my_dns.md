If you are using default Ubuntu installation with GUI, then you can change DNS information in Network Connection Manager Settings icon shown besides the time and battery. After changing it you need to restart the NetworkManager or disable / enable networking through the same icon settings.

### Post by SeijiSensei on 2017-07-18
You can add servers permanently by editing the file /etc/resolvconf/resolv.conf.d/head like this:
```

# Dynamic resolv.conf(5) file for glibc resolver(3) generated by resolvconf(8)
# DO NOT EDIT THIS FILE BY HAND --
# YOUR CHANGES WILL BE OVERWRITTEN 
nameserver 8.8.8.8
nameserver 8.8.4.4

```

Save the file and reboot.  Now this header will be placed atop /etc/resolv.conf, and the Google servers will be contacted first.
