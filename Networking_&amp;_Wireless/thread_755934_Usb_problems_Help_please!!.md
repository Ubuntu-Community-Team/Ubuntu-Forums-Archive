### Post by dstew on 2008-04-15
> when you install something and you have to manually set it up by typing some sort of code into a terminatorHi, and welcome to Linux and Ubuntu. You are referring to the "Terminal", which comes from the old computer days when the computer itself was far away, and the user had only a keyboard and screen in front of him or her, and that was called a computer terminal. Now, the Ubuntu Terminal is a screen you get by opening the Applications --> Accessories --> Terminal program.

As to setting things up using the Terminal, Ubuntu is working toward eliminating that, by developing automatic and graphical setup tools. The idea is to use the terminal rarely, if at all, in the future.

Your problem sounds like a bug in the software that runs the USB bus. I am not sure you will be able to fix it by reconfiguring things, but I don't know for sure. Maybe someone else will post an answer for your disconnecting issue.
