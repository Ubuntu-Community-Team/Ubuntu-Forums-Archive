### Post by grahammechanical on 2010-12-22
First make sure that the connection is actually set to Automatically Connect. Right click the network icon and select Edit Connections. Select the Wireless tab and then your wireless connection and make sure that the little box at the top left has a tick mark against it. If it does not, then click it. If this is obviously not the answer, someone else might have a suggestion.

Regards.

### Post by folli88 on 2010-12-23
I have found a solution. The connection must be set for all users in the check box on the bottom left corner. Now it will reconnect automatically after a router down! :D
