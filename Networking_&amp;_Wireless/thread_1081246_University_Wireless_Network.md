### Post by m_duck on 2009-03-06
If you search for "wireless linux" on the uni site, one of the top couple of links shows a page with images with settings to use to connect and also a link to the certificate you need. If you end up having no luck using the regular network manager (as I did in Intrepid) try installing [wicd](http://wicd.sourceforge.net/) and setting it up in that. That's how I got me eee connected recently.
