### Post by chinaski on 2008-02-06
I have installed Ubuntu Gutsy server edition on a machine but I am having issues to connect to the internet

1 ) during installation process a warning stated no suitable network interface was found

2 ) after installation finished, the command ```
sudo lshw -C network
``` showed ```
eth0
*- network DISABLED
``` which I fixed with ```
sudo ifup eth0
```3 ) the file resolv.conf was not present in /etc so I created it myself

4 ) the file /etc/network/interfaces only had those lines in it ```
auto lo iface lo inet loopback
``` I added ```
iface eth0 inet static address 192.168.1.70 netmask 255.255.255.0 gateway 192.168.1.1
```5 ) I _can_ ping the localhost, I _cannot_ ping the router nor other clients in the lan or anything in Internet

6 ) I installed Ubuntu Gutsy desktop edition and it works perfectly...

additional info:

router has disabled DHCP

I installed a printer + samba server, that's it

PC is brand new, mobo is Asus P5KR with onboard realtek gigabit ethernet adapter

thanks in advance for any help on this issue

### Post by Iowan on 2008-02-07
My **interfaces** file:```
auto lo
iface lo inet loopback

auto eth0
#iface eth0 inet dhcp

auto eth1
#iface eth1 inet dhcp

auto eth2
#iface eth2 inet dhcp

auto ath0
#iface ath0 inet dhcp

auto wlan0
#iface wlan0 inet dhcp


iface eth0 inet dhcp

```
