### Post by mlho on 2008-11-15
For some reason, my system (Ubuntu 7.10) doesn't automatically route the newly found IP.

Had to enter:
sudo route add default gw xx.xx.yy.zz

(where xx.xx.yy.zz was the local IP address as specified in the Wvdial logs)
