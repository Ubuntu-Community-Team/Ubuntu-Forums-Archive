### Post by TheFu on 2018-08-19
If the systems are on the same LAN (100 Mbps connections or faster), using remote X/Windows is easier and much more convenient.  It has been built-into X/Windows for many decades.  The remote system is the X-client and the local system must run an X-Server.  It is easiest if there is an ssh connection between these two machines.  I'm using it now to connect to about 8 different systems running different programs.
