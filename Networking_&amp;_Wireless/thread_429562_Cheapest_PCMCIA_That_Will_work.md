For the benefit of others here is a positive report...

I purchased the Linksys Network kit for NoteBooks which includes a WRK54G router and PCMCIA card.
The PCMCIA is Atheros AR5005G 802.11 abg as reported by lspci.
Purchased from Staples for $30 after rebates so not a bad price for a card and router (good spare).
I did have to attempt using Network Manager a couple of times but did not have to install anything
else so I'd consider this basically working out of the box for a fairly  low price giving a good
price to pain ratio.

My T30 is loaded with Edgy and I'm using WPA-TKIP

I hope this helps someone.
