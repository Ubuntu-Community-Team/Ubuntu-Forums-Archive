### Post by FryerFox on 2006-11-26
You really don't provide much information - for example, have you configured your router?

If not, I would suggest that you read the little book that comes with it. Most routers come preconfigured (from the factory) with an address of 192.16.1.1 or 192.168.0.1

Try typing those addresses into your browser and see if it  connects you to the router. If it does, you should be able to proceed from there by using the default username and password (should come in the router documentation).

Also, there is usually a small button somewhere on the router that resets it to factory defaults (so that you can still use the router if you forget the password).
