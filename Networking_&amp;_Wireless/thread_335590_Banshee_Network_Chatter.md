### Post by idlewild on 2007-01-16
Theres several plugins in banshee that might be using the network, but if it is regular traffic I would suspect it is daap (music sharing over the network). Theres no reason to be worried by the traffic but you can go about disabling plugins and removing banshee-daap (via synaptic) if you're that intrigued.
