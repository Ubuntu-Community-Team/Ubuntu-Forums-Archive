### Post by pytheas22 on 2008-12-26
I don't have any personal experience with this kind of stuff, but based on the output you posted, I think that encryption is your problem, or at least part of it.  Note this error message:
```

"MPPE required but peer refused"
```

I know you said you played with different encryption options, but did you try every one?  Did you check to see whether the error message regarding encryption changes if you change the encryption options?  It's possible that there are other issues that need to be resolved as well, but the encryption stands out to me as the most immediate problem.  You need to make that error message go away.

Also, you probably need to specify your DNS servers (10.101.10.5 and
10.101.10.2) under the DNS field.  It looks like those are blank below.

A final note: since Ubuntu 8.10, Network Manager is supposed to be able to handle this kind of stuff.  You may want to try playing around there--under the DSL section, there are options for configuring a ppp connection.  Network Manager is the icon in the top-right corner of your screen that looks like two computer monitors.  Right-click on it and select 'edit connections' to configure a new connection.
