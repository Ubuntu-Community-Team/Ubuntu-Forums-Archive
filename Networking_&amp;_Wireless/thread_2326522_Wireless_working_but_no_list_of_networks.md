### Post by Keith_Helms on 2016-07-14
I'm having a problem where my wireless just halts working, but also had a secondary problem with the network list disappearing at some point.  As part of trying to fix the first problem, I turned off power saving mode for the wireless adapter and that seems to have fixed the network list disappearing problem (regretfully, it *didn't* fix the first problem).

Anyway, for whatever your wifi adapter is known as in your system - wlan0, wlp3s0, etc - try this:

```
sudo iwconfig wlp3s0 power off
```

If that seems to work, you can make it permanent multiple ways.  The one I used was to add a line to /etc/rc.local before the exit command

```
iwconfig wlp3s0 power off

```

I don't know if this will work for you or is applicable to your system, but it's worth a try.
