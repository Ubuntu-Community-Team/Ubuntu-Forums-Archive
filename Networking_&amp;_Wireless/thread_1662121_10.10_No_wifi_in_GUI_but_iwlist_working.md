### Post by PatchesTheCaveman on 2011-01-07
> While the above solution enables me to ping to external ips, I'm still unable to use Firefox or other appilcations as they appear to indicate that I'm offline.

That's an easy fix.  File > Work Offline.  Unchecking that should get Firefox working.

> The only problem is that on my ubuntu desktop, the wifi icon if clicked does not display the option of wireless connection (only wired network) as if no wifi adapter is connected.

You can probably get a working network icon by switching from the default NetworkManager to wicd.  To do that, run these commands on a terminal:
```
sudo apt-get remove network-manager
sudo apt-get install wicd
```
