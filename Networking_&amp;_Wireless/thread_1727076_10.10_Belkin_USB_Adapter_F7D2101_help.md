### Post by chili555 on 2011-04-12
First of all, autorun is not an .inf driver file, let's erase it:```
sudo ndiswrapper -e autorun
sudo rm -r /etc/ndiwrapper/autorun
```Next, let's remove the RT61 stuff:```
sudo ndiswrapper -e net8192su
sudo rm -r /etc/ndiwrapper/net8192su
```Now, let's straighten out some of the .conf files:```
sudo rm /etc/modprobe.conf
sudo gedit /etc/modprobe.d/blacklist.conf

```Remove this line; proofread, save and close gedit.```
blacklist rtl819xU
```There is no such module, so there is no need to blacklist it.

Reboot and post:```
ndiswrapper -l
iwconfig
```We may have more to do.
