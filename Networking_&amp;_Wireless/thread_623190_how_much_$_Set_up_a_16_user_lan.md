### Post by intelligentfool on 2007-11-26
depends on your needs. if you/the customer has the money, get a proper Cisco switch. if you want a high end setup, get something with PoE & QOS... it will push power down the ethernet line to keep VoIP phones online in the event of a power outage (assuming network gear is on UPS). QOS will ensure phone convo's stay clean even while a server is pulling updates, etc. 

unfortunately, such a high end switch will run around $5k US including software. but thats the rolls royce of LAN setups,.... its all way cheaper as you start taking features out. 

if you want a specific recommendation, get the Cisco 3750-24PS. The uplink ports use GBICs, which are hot swappable and upgradable to 10Gbps, along with the PoE and QOS features, and more features... read the site :)
