### Post by jml on 2006-05-31
I agree with the recommendation for a NetGear WG511T.  I use one as well and it is recognized out of the box by Ubuntu.  It is also compatible with WPA encryption.  If you need WPA encryption, you will have a little more work to do.  While the card supports WPA encryption, Ubuntu does not.  To get WPA encryption working you need to download, install and configure an application called WPA_supplicant.  It is available from the Universe repository.  If you plan to use an unencrypted network, or one using WEP encryption, you should be ok.  Here is a link to the Ubuntu wireless troubleshooting guide.

[https://wiki.ubuntu.com/WirelessTroubleshootingGuide](https://wiki.ubuntu.com/WirelessTroubleshootingGuide)

and the Ubuntu WiFi docs Wiki.

[https://wiki.ubuntu.com/WifiDocs](https://wiki.ubuntu.com/WifiDocs)

hope this helps.

Joe
