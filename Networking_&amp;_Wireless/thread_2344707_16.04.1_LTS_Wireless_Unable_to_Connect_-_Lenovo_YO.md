### Post by jeremy31 on 2016-11-28
A hard block on a Lenovo with ideapad_laptop loaded
Try ```
echo "blacklist ideapad_laptop" | sudo tee /etc/modprobe.d/ideapad-laptop.conf
```

Reboot and it should work

Can you post result for ```
sudo dmidecode | grep -i version
```

I would suggest filing a [bug report](https://ubuntuforums.org/showthread.php?t=1011078) against package linux so that others with your model laptop won't have to deal with this.  Blacklisting the ideapad_laptop module will get your wifi working but some FN combos will fail
