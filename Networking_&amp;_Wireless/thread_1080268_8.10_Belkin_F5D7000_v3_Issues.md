### Post by Atrophy on 2009-03-02
I solved my wireless issue by connecting to the internet with a wired connection, running the package manager an installing all updates, then running terminal and using the command 'sudo apt-get install linux-backports-modules-intrepid-generic' and rebooting. Now I can see all wireless networks in my area and connect every time. Before this, I was getting the same results you have posted as far as iwscan, iwconfig, and lshw -c network goes.
