### Post by ihristov on 2007-08-17
Do the following:

With the USB adapter unplugged.
1. open a terminal

2. type "lsusb" and hit Enter.

3. Plugin the adapter

4. again type "lsusb"

5. Copy and paste the results from the terminal into a response.
