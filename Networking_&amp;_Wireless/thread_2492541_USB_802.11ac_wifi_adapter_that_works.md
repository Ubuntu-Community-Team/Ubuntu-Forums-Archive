### Post by fahrbach on 2023-11-14
Finding an ac wifi adapter that works can be tough but if you need one check out BrosTrend (Amazon).  Compatible with Linux Kernels up to 6.5, Ubuntu 23.10~16.04.  One reviewer claims only Debian based distros work but that is quite a few flavors, e.g., Ubuntu, Mint, etc.  Comes with a CD ROM that has a .pdf file for Linux installation that has a simple single command.  Simply open your terminal, copy and paste the command, hit enter.  The command will download the necessary driver and all of a sudden your adapter activity light will begin flashing.  No reboot necessary.
