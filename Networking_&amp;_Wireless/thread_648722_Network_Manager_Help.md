### Post by Kobalt on 2007-12-24
You cannot install a package out of the LiveCD (aka you can't use it as a source for Apt). You need the Alternate CD to do so.

Otherwise you can download the packages manually from [packages.ubuntu.com]("http://packages.ubuntu.com")
