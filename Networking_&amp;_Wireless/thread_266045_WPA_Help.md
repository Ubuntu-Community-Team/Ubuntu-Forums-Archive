### Post by wieman01 on 2006-09-26
"wpa-psk" is normally based on TKIP, so don't worry. Unless it says wpa2-psk (AES) you are fine to give those tutorials a try. If those do not work out, let us know & we help.

WPA(1) uses TKIP (RC4 cipher - same as WEP), WPA2 uses AES which is way more secure.
