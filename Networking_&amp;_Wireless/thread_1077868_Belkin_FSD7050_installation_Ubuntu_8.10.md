Hello,

I have seen so many how to guides on installing the Belkin FSD7050 and though I should make clear of one thing before I start this guide. 

This installation guide is for the Belkin FSD7050 Version 4002 Uk, it may work on other version of the FSD7050, but this is all I have known it to work on. 

To find out what version of the FSD7050 you have you will need to look on the back of your wireless USB stick and there is a small sticker on there about half thumbnail size or baby thumbnail size. On that sticker should be a version number. 

The drivers of different FSD7050 versions are different because they run different firmwares, so therefore run different drivers. Just to confuse matters they decided to keep the same model number. 

Anyway lets start, click Applications> Add/Remove, in the search box type wireless and press enter.

Select Network Selector and RutilT WLAN Manager and then apply changes. 

Wait for them to install and when they are both run together you should now get a smooth connection. You won't have to open them manually each time you load Ubuntu either.

Again I would like to note that this was tested on version 4002 UK and I don't know what versions of the FSD7050 it is compatible with.
