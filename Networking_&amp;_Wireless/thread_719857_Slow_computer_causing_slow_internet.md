### Post by ajgreeny on 2008-03-09
Try disabling ipv6 in Firefox as this may speed up your connection speeds.  Go to **about:config** in the address bar and then in the filter box type "ipv6".  Then if the value is "false", which it is by default, double click on the text line that appears to change the value to "true"  Now restart Firefox and you may find it is much faster.
