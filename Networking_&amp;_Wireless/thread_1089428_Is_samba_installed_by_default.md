### Post by Iowan on 2009-03-07
Ubuntu comes with Samba's client installed - that means you should be able to see shares on a Windows machine.  To share from Ubuntu to Windows machines requires installing Samba's server component. The newer versions seem to have files stored in different places (/var/lib/samba/usershares), but I hadn't (yet) heard about "remote".
