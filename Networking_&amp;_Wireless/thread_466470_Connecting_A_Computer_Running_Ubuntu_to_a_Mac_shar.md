I recently overhauled my old windows pc with some new memory and an ethernet card and installed Xubuntu on it.  It was able to connect to the internet fine when directly plugged into a router at my parent's house, but is unable to find a connection when it is connected to the G4 via its ethernet port, while the G4 is sharing its airport connection.  I've used this feature with other macs, and it usually works swimmingly.  I tried setting the IP address manually with the G4's IP as its gateway, but that didn't seem to work.  Any attempts to ping the gateway (G4) result in 100% packet loss.  Any suggestions?
