### Post by InspiredIndividual on 2009-02-27
GCC is the GNU C Compiler, a C compiler that's installed in Ubuntu by default if I'm not mistaken. Qt is a C++ application framework. Those references in the filenames probably indicate the versions used of GCC and Qt respectively. 

The standard package format in Ubuntu is .deb, so you probably want to download the .deb file. Just grab the one with the newest GCC and Qt versions, unless you have a specific reason to use a different one. After you've downloaded the .deb file, just double-click the file and it will be installed. I tried the Opera 10 alpha version for amd64 architecture and it's working fine for me. Hopefully my short explanation will help you as well.
