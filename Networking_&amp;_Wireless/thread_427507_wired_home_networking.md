### Post by Kevanx on 2007-04-29
As a disclaimer: I'm sorry if this information is inaccurate - I'm just speculating based on my own experience with wired networks, and I've always used a switch/router to connect.

The main problem I forsee is that most computers only have a single ethernet card. If you have a cable modem, this becomes a problem, because you wouldn't be able to connect the second computer to it. But if you use dial-up, then the ethernet cards could be used strictly for networking, which should be very straightforward. I've used shared internet connections over a network with Windows, but I haven't tried doing it with Samba, so I don't know what it would involve, but there is no reason I know of that a computer couldn't piggyback its internet connection off of the first one, provided that the connection scenario mentioned above works.

In terms of hardware - a LAN (ethernet) cable might be all that you need, or just a LAN cable, and an additional Ethernet card.

Them's my two cents. Hope it helps.
