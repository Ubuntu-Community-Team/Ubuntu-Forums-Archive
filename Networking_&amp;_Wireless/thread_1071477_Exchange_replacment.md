### Post by nixscripter on 2009-02-17
I don't know if there is an M$ Exchange server equivalent for Linux, but Mozilla Thunderbird can pretend to be an Exchange client. More on that is here: [http://www.bauer-power.net/2008/08/how-to-get-thunderbird-to-work-with.html](http://www.bauer-power.net/2008/08/how-to-get-thunderbird-to-work-with.html)

If you want to get rid of your Exchange server, I would suggest using IMAP instead: [http://linuxgazette.net/issue35/jao.html](http://linuxgazette.net/issue35/jao.html)

Hope this helps.
