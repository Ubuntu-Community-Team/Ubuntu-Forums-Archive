### Post by dmizer on 2008-12-08
With transmission, you will need an open and forwarded port for each torrent file you'd like to download.

For example, if you want to have 10 torrent files downloading at the same time, you'll need to have ports 6881-6891 (for example) open and forwarded to your computer.

### Post by dmizer on 2008-12-08
With transmission, you will need an open and forwarded port for each torrent file you'd like to download. You'll also have to configure transmission's settings to use the same port range.

For example, if you want to have 10 torrent files downloading at the same time, you'll need to have ports 6881-6891 (for example) open and forwarded to your computer.
