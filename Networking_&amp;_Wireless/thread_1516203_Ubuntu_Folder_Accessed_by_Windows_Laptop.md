### Post by dozdozez on 2010-06-23
I'm running debian and have this entry on the toolbar menu:

System-->administration-->Shared folders

in ubuntu it should be the same, but i'm not sure.

This display this message to me: "you need to install at least either Samba or NFS in order to share your folders."

So i should install samba or NFS. Have you one of them installed?

I can't test it cause i haven't any windows machine, but i think it should work for you. 

If you don't find this menu entry, type shares-admin in a root shell.

Good luck!
