### Post by SeijiSensei on 2022-02-18
Is your router configured to pass along DNS server information when the network card requests an address?

One alternative is to edit /etc/systemd/resolved.conf and enter your preferred DNS servers there. Put the primary on the DNS= line and any secondaries on the FallBackDNS line. You need to be root to edit this file, e.g,  "sudo nano /etc/systemd/resolved.conf". Remove the hash marks ("#") from the beginning of those two lines.

Then restart the resolver with the command "sudo systemctl restart systemd.resolved".
