### Post by superprash2003 on 2008-03-27
you need to configure DNS properly.. go to System->Administration->Network then go to the DNS tab and add the following 2 ips 208.67.222.222 and 208.67.220.220 
 you can alternatively configure DNS in your modem or router using this tutorial [http://prash-babu.blogspot.com/2008/03/how-to-configure-dns-server-in-your.html](http://prash-babu.blogspot.com/2008/03/how-to-configure-dns-server-in-your.html)
