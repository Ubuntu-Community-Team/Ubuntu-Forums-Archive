### Post by Lambert on 2007-10-13
Make sure the interface is not up

sudo ifdown eth0

replace eth0 with whatever the devices interface is.

Also try to use this command instead

sudo modprobe -r r8169

You did lose me though saying you installed the driver and then want to unload it from running?
