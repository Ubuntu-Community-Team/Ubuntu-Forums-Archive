### Post by EvilSupahFly on 2014-03-26
The real name is most likely the one Ubuntu has picked up because Windows uses [FONT=courier new]desktop.ini[/FONT] files to manage folders in special ways, like custom names, Folder Icons, etc, although, if you're looking at the folder over a Windows-hosted network connection, it could go either way. If possible, I would use a Live CD to boot and check things out that way.
