### Post by chili555 on 2013-03-07
Ubuntu 10.04 supports WPA2 very well. However, many older wireless cards do not. You can check with this command from the terminal:```
sudo iwlist wlan0 auth
```This assumes your wireless interface is wlan0:```
iwconfig
```As well, if you are trying to associate with a wireless access point that is encrypted with WEP, Network Manager will pop up an authorization box with only the method the access point uses; i.e. WEP. Please see attached.
