[FONT=Arial][SIZE=3]On Ubuntu 9.04, if you should get asked to "Enter password for default keyring to unlock" when inserting your USB wifi stick, then just type your login password:) Make sure you click on "Allow allways".  

Set up your connection to your AP(Access point) or Router.

When done,  rightclick Network Manager in the notification panel, and choose "Edit Connections ....", Click on Wireless.
Mark your connection and click on "Edit".

If you want this connection to be available for all users then add a checkmark in the box next to "Available to all users". If you want it to connect automatically,  then add a checkmark in "Connect automatically".
 

Thats it:)
