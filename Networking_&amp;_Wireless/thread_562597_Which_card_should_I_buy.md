### Post by Thunar on 2007-09-29
My personal experience with network cards tells me that, for one, you will have a much easier time setting up a pci card than a USB card because when you use a USB  network card you not only have to set up the network card itself, but you may also have to set up your network to interface with USB, and secondly the latest and greatest is not always the best choice, as it is much more likely that Linux will have the drivers for an older card built right in.

You can, however, get a newer USB card to work, but it may make you work, too. :P
