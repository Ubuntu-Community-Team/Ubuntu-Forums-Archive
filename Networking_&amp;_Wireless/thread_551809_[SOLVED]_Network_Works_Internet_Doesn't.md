### Post by noob12 on 2007-09-17
I don't know of anything other than the firewall that can cause this particular message.  There may be other tables that we need to flush as well.  If you can turn it off from the fiirestarter GUI, that will probably flush all of them. 

```

sudo iptables -t filter --flush
sudo iptables -t nat --flush
sudo iptables -t mangle --flush
sudo iptables -t raw --flush
sudo ifdown eth1
ping 192.168.1.254

```
