I have a really old computer that I've been using as a mythtv box with an hdhomerun connected to the ethernet port of the computer and the local network conected through a wifi dongle.    It works ok but is pretty laggy.    (I actually have a usb device set up as swap or it doesn't work at all but that is a different story)

I already use plex media server on another computer (ubuntu) in another part of the house and I got to wondering if I could just use it to record over the network and (mostly) bypass the old computer.

I'd have to run new cables and might not get as good a tv single because of the longer cable run and our TV reception is iffy anyway since we are about 90 miles from the towers.    I'd like to leave it connected to the old computer, as it is now, but still access it on the network through the wifi connection.   If I try to set the network connection for the hdhomerun from locallink to make it available to other computers, the computer tries to use the HDhomerun for internet which is obviously not what I want.

### Post by alforddm on 2018-04-15
Well, I finally realized I was being stupid and went to the store and bought a 50ft cat 6 cable and ran it under the house to the router.   It was a bit of work but I'm sure I'll get much better performance since the recording computer and the HDHomerun are both connected by cable instead of wireless.
