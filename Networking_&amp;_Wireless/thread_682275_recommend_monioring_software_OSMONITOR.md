### Post by wangya on 2008-01-29
OsMonitor is a kind of user friendly management software designed for LAN environment. It adopts C/S modal with one server monitoring all client computers. Its features are as follow: 
1.Forbiddance, restriction, surveillance and monitoring. The settings on all client computers take effect automatically following the server. It’s much easier for you to manage and monitor your employees’activities by setting up everything at one time. You won't do settings on the Clients one by one. 
2.OsMonitor has no specific any requirement for the target computers, network structure and distribution and is fully compatible with all versions of operating system. High security with the hidden running in client background and cannot be closed by Windows task management. .  
3.Block any chatting program like ICQ, MSN, any downloading program like BitTorrent and any P2P software which occupy company’s bandwidth. 
4.Forbid any USB ports and any USB storage device, such as portable hard disk and flash memory disk, etc., to avoid confidential information leak. Record all of the ICQ conversation and MSN messenger conversation. 
5.Forbid surfing on the Internet. It can assign which particular computers and employees can access to Internet, when the employees can access to Internet, and which particular Websites cannot be accessed, such as sex sites, stock sites and more. It also can grant access to or block some particular websites as well.
6.Forbid installing illegal software like games. Only permit working related software, for example, you can allow your employees’ computer only run Word or Excel, in that case, other programs are not allowed running. 
7.Do you want to know who is downloading? Which computer is infecting the worm virus? OsMonitor allows you to view the downloading and uploading stream of every employee’s computer. All of the data are saved for your later review.
8.The LAN administrator can control the whole LAN without any difficulty due to the user-defined feature of the forbiddance and surveillance functions.  
9.Record the employee’s windows opened, program run, web sites visited and conversation in chatting program.
10.Surveil files operation, record the files copied, pasted and deleted, and report the information of USB storage devices plugged in.  
11.Besides the instant forbiddance of the employees’ appropriate activities, it also saves these records for later review with indices. All of the report forms can be printed in the form of Word or Html.
