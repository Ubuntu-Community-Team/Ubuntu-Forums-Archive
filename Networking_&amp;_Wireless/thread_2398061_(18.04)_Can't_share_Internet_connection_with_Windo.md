### Post by TheFu on 2018-08-07
Did you edit:
>     edit /etc/sysctl.conf and uncomment: 

```
#net.ipv4.ip_forward=1
```

... so that it reads:

```
net.ipv4.ip_forward=1
```
Either forcing the update or a reboot is needed after.
BTW, this change has huge security implications that you want to understand.
