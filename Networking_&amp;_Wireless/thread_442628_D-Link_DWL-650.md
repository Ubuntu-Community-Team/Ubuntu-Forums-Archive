I have the PCI version DWL-G520+ which uses the ACX111 chipset. I have tried the native ACX100 linux driver and found it to be ok, but am now using NDISWRAPPER in Edgy which I am finding to be more reliable, and will also allow me to use WPA when I get round to changing it.

If you have the 650 with the Texas Instruments ACX100 or ACX111 chipset you have both options. To install the native driver search the forum or if you want to use NDISWRAPPER I can post my own How-To which I wrote recently when installing the card on my server, or again you can search the forums.
