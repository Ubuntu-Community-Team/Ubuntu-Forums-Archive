### Post by chili555 on 2010-06-01
Network Manager is designed to not allow a wireless connection if you have wired, which you do:  [https://help.ubuntu.com/community/NetworkManager](https://help.ubuntu.com/community/NetworkManager)> The computer should use the wired network connection when it's plugged in, but automatically switch to a wireless connection when the user unplugs it and walks away from the desk. Likewise, when the user plugs the computer back in, the computer should switch back to the wired connection. The user should, most times, not even notice that their connection has has been managed for themPlease detach the cable before you try to connect.

Also, I have read several cases on this forum about trouble connecting when the router is set to mixed WPA *and* WPA2 mode. Connection is immediate when the router is reset to WPA2 only. Please check your router.

If you are still unable to connect, we might look at the last 20 or so lines of:```
sudo cat/var/log/syslog | grep -i network
```
