You probably aren't affected if you are running Linux, but you router could be or someone else on your network.

Here is something I found

If the infected PC is on a network, you'll have to check every other PC  on the network for signs of infection, and then check your router's  settings to ensure that it isn't affected (DNSChanger is programmed to  change router DNS settings automatically, using the default usernames  and passwords of most modern routers). To do this, copy down your  router's DNS server IP addresses (located in your router's settings  menu; read "[How to Set Up a Wireless Router]("http://www.pcworld.com/article/249185/how_to_set_up_a_wireless_router.html")"  for more information) and check them against the FBI's IP address  database mentioned above. If your router is infected, reset the router  and confirm that all network settings are restored to the manufacturer's  defaults.
