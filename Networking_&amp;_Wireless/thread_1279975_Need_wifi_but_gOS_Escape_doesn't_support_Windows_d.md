### Post by geno1956 on 2009-10-10
Hi, I may be a little late but I bought one of these too. The gOS distro on it is a slimmed down version of Ubuntu and doesn't allow you to do much. Do not try and load 9.04 or 9.04 remix because the book locks up every time you try. I finally installed Xubuntu 8.10 and it seems to be pretty stable. It took a couple tries to get it done though. You can load the updates when it asks, but DO NOT upgrade to 9.04 because it will lock up. And the on board wifi is very weak. I have my book 3 feet from my wireless router and I barely get 20%. I plugged in a USB wireless adapter and get 100%. Hope this helps.
