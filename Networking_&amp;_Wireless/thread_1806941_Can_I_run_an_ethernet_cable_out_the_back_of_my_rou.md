### Post by sanderd17 on 2011-07-18
If I'm right, you need to disable DHCP of the second router, pick a fixed ip address and you have to put the ethernet cable in a computer port, not in the input port.

You'll better read the router specific documentation.

Oh, and if you want to play with port forwarding, this will touch the home network. If you don't want to touch that network, you need to pay an extra internet connection.
