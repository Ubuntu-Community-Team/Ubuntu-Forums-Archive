### Post by buchalka on 2006-06-04
Hey all,

I got this working, I'm now running the Dell Inspiron 9400 wirelessly using WPA!

Did not need to  download any scripts, etc either which is a bonus.

This is what I did to get it to work.

1.  Install Dapper 6.06 LTS from scratch WITH regular cabled network cable plugged in with my router allocating it an IP address via DHCP.

Step 1 is really the key to everything, without this I think you will be struggling to do anything, mainly because Network Manager (which is an important part of the install is not available until you download some updates).

2.  The Live CD found my dell network card and I was connected to the net (via cable).

3.  I then did a complete install as per normal and rebooted into the new installation.

4.  Click Applications,Add/Remove, Internet, select Network Manager and install it.  When prompted about being out of date choose to download the latest updates.

5. from a terminal sudo gedit /etc/network/interfaces and comment out everything but the lo stuff e.g. put a # in front of it, save the changes

6. Reboot (yes you could just stop/start networking, etc but what the heck).

7. When you have rebooted you should still be connected to the net via cable.  Now unplug the cable and click the network manager image (top right near the date/time).  

8. Enter your wireless details and passphrase and your good to go!!  

It worked for me and it's all good. 

Hope this helps someone else out there.

Good luck



Tim:mrgreen:
