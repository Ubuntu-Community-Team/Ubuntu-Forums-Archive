### Post by bwtranch on 2008-01-26
I don't know if there is an easy solution to a wifi problem unless it just works. Most things involve some sort of driver problem, either with having the right one installed or a config or both. The output of ifconfig is the starting point and lspci and lsusb output can be useful also. This can sometimes be difficult if you don't have any other way to connect, I understand. But, try to post as much info as possible and we can try to figure it out. I've never seen one that is impossible, but some are harder than others.
