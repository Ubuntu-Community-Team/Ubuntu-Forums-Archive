### Post by ellisdee on 2007-06-27
Try vpnc
shan@schlappy:~$ sudo apt-get install vpnc

Then run vpnc
shan@schlappy:~$sudo vpnc

Should prompt you with the following:

shan@schlappy:~$ sudo vpnc
Enter IPSec gateway address: 
Enter IPSec ID for gateway address: 
Enter IPSec secret for ID@gateway address: 
Enter username for gateway address: 
Enter password for @gateway address: 

Hope this helps ;)
