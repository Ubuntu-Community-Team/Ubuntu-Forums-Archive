I use gnome-ppp and it does a good job. You will have to make sure the modem is installed properly first. Once you have it set up and running fine, you can get gnome-ppp from the repository and it is pretty easy to configure. I even added a icon on my toolbar and desktop for easy access.

If you have problems getting it configured, I would be willing to help as much as I can.
