### Post by wildmanne39 on 2012-05-21
Hi, please do:
```
gksudo gedit /etc/modprobe.d/rt2800pci.conf
```

Add one line:

```
options rt2800pci nohwcrypt=1
```

Proofread carefully, save and close gedit.

Also set IPV6 to ignore in network manager then Reboot.
Thanks
