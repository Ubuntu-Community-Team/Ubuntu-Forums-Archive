### Post by fbuchele on 2008-02-15
So I found out on my own that my problem is with my IP rules, typing ```
$ sudo iptables -L
```

gives

```
Chain INPUT (policy DROP)
target     prot opt source               destination

Chain FORWARD (policy DROP)
target     prot opt source               destination

Chain OUTPUT (policy DROP)
target     prot opt source               destination
```


instead of 

```
Chain INPUT (policy ACCEPT)
target     prot opt source               destination

Chain FORWARD (policy ACCEPT)
target     prot opt source               destination

Chain OUTPUT (policy ACCEPT)
target     prot opt source               destination
```

since it looks like the ubuntu forums doesn't care, I will work on fixing this on my own, then post my solution so others with the same problem can fix this.

### Post by fbuchele on 2008-02-15
So i fixed it, once i found the problem the solution was pretty simple.

I put in (I think) ```
sudo iptables INPUT --policy ACCEPT
sudo iptables FORWARD--policy ACCEPT
sudo iptables OUTPUT --policy ACCEPT 
```

I don't remember the exact thing i put in, but it was something like that. If someones trying to fix this same problem and i got the code wrong, just look up the correct commands in ```
$man iptables
```

and it should be pretty clear... i think...
