### Post by IntuitiveNipple on 2008-08-09
**For Hardy** I've just built the Intrepid version of Network Manager 0.7 and all the required supporting packages so that it is possible to use Network Manager and the panel applet to create, edit, and manage GSM mobile-phone connections amongst [a raft of other great improvements](http://live.gnome.org/NetworkManagerToDo). The updated packages are:
[list]
* libdbus-1-3 1.2.3
* libdbus-glib-1-2 0.76
* dbus 1.2.3
* dbus-x11 1.2.3
* lsb-base 3.2-14
* wpasupplicant 0.6.3
* dhcp 3.1.1
* network-manager 0.7
* network-manager-gnome 0.7
[/list]
My wiki has an [article on how to install them](http://tjworld.net/wiki/Linux/Ubuntu/Hardy/NetworkManager0.7). The packages are [all available from my PPA](https://edge.launchpad.net/~intuitivenipple/+archive).

If used together with the [latest version of Blueman Bluetooth Manager (0.6)](http://blueman.tuxfamily.org/forum/viewtopic.php?f=5&t=74) it is incredibly easy to create dial-up connections on mobile 'phones, use multiple interfaces, and more.
