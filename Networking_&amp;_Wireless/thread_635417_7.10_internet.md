Make sure you don't have the proxy set to 0.0.0.0
Go to System => Preferences => Network Proxy and make sure "Direct internet connection" is selected.

If that doesn't work, refer to [this post]("http://ubuntuforums.org/showthread.php?t=435851") to renew the IP address.
