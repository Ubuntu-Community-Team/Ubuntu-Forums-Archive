How do I go about setting a static IP on my Edgy system? I have a router and would like to remote to my computer, etc.

---

### Post by merlinus on 2007-05-22
System/Administration/Network.  Select the checked Connection (Wired or Wireless), then Properties.

Select Static IP address from the dropdown Configuration list.

You can use any range of IP addresses that your router supports.

I use 255.255.255.0 for subnet mask, and 192.168.1.1 for gateway address.

Good luck!
