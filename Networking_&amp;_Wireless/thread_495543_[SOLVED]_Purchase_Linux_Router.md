### Post by tturrisi on 2007-07-08
> **Flashgordon20 said:**
> I'm in need of a new wireless router and was curious if there are any that run the initial set up and administration on Linux. Anybody hear of such a thing or is windows required? I have wine installed but I shy away from installing any windows set up files. Thanks for any input you can provide.
Vendors provide those router setup cdroms as a "no brainer" method of setting up the router.  However, those setup cds are completely unnecessary.  All you have to do is access the router control panel via the web browser and set it up manually, usually takes about 2-3 minutes & is quite simple to do.

For example, if get a linksys model, open the browser and put [http://192.168.1.1](http://192.168.1.1) in the address bar & Go.  Login at the prompt and access the router controls.   There is no username required and the default password is "admin".
