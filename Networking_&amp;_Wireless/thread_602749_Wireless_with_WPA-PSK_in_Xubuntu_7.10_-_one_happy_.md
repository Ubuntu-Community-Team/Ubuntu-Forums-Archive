### Post by ChrisG_UK on 2007-11-04
Just thought I'd share my success story.

I first installed Xubuntu 7.10 on my Thinkpad T23 without a wireless card. After the install, I plugged in my Netgear WG511v1 wireless network card. Xubuntu recognised the card and installed the Prism driver and everything worked fine -  with no encryption. But then I tried to configure WPA. What a nightmare. I worked for two days trying to get it to work with no success.

Then, for an unrelated reason, I replaced the hard drive, so I had to re-install and start over. Only this time I did the install with the wireless card in the machine. I don't know if it makes a difference but this time I also had the machine connected to the router via the wired connection, and I installed all the available updates when Xubuntu first came up. This time, to my total amazement, I was able to configure WPA from the Network Manager in the system tray. It worked first time, and I was literally up and running with WPA encryption within about 5 minutes of starting Xubuntu. Absolutely no fiddling around at all, and a heck of a lot easier than Windows. Now I am really, really impressed - I'm a 100% happy bunny!

Xubuntu is really a brilliant operating system. I have the desktop looking exactly like Windows (sorry, but it takes all sorts...) only it's a lot, lot faster. Anyway, a big pat on the back for all the people that made this possible. Now we just need to make it work as well for the case where you add the wireless card later on. I'd be interested in comments on this issue. Might this be a solution for other people with WPA issues?

Anyway, while I'm here, one final quick tip that doesn't belong in this thread but is so hot everyone should know it. Go to User Interface Settings and tick "use sub-pixel hinting". The improvement to the display quality is so great I just had to spread the word.
