Open terminal(CTRL + t) and enter the following
```
lspci -nnk | grep -iA2 net; rfkill list all; dmesg | grep iwl; uname -a
```

Post the output

### Post by jeremy31 on 2016-04-12
This should be the easiest
```
sudo apt-get install build-essential linux-headers-generic
wget https://www.kernel.org/pub/linux/kernel/projects/backports/stable/v4.4.2/backports-4.4.2-1.tar.gztar -zxvf backports-4.4.2-1.tar.gz
cd backports-4.4.2-1
make defconfig-wifi
make 
sudo make install
```

Reboot

After updates are installed, you may have wireless fail after reboot, in that case
```
cd backports-4.4.2-1
make clean
make defconfig-wifi
make 
sudo make install
```
