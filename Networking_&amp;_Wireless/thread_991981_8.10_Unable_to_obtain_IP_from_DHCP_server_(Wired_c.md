### Post by HallStu on 2008-11-28
I had the same problem as this connecting to my router. 

The problem was fixed by adding the following lines to /etc/network/interfaces.

```
auto eth0
iface eth0 inet dhcp
``` 

see[ here]("https://help.ubuntu.com/8.10/serverguide/C/network-configuration.html#ethernet") for more information.
