Did you install from an Ubuntu Live Disk?  If so, it looks like from the link you supplied that it must be some form of Broadcom wireless device.

If you go to System>>Administration>>Software sources and put a check in Install from CD. Hit reload and ignore any messages about connectivity. You should be able to install the necessary driver from there. See this thread for additional information.

[http://ubuntuforums.org/showthread.php?t=1368699](http://ubuntuforums.org/showthread.php?t=1368699)

Good Luck!
