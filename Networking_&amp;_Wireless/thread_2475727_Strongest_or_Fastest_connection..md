I have a home network with two, sometimes three different access points.  Each extra AP is connected using a powerline adapter with main one being part of router.

### Post by TheFu on 2022-06-10
Always use wired connections when possible. wifi will always be worse than a wired connection.  Not all powerline is the same.  Test the throughput.  I tested the 600 Mbps powerline here in 3 different locations
a) same room, different wall plugs
b) room and hallway on the same level
c) room and downstairs where a media playback would happen

a)  220 Mbps
b) 200 Mbps
c) 62 Mbps

That 62 Mbps is stable, not all over the place like wifi connections are.  Humans don't notice the massive fluctuations in wifi throughput, but the computers definitely do see those fluctuations.

As for wifi, there **is** a reason that wifi-mesh networks exist.  Those sense the output from each of the APs involve and adjust the power to prevent much signal overlap.  In theory, we can manually adjust the power of each AP to get close to what the mesh software does automatically.
