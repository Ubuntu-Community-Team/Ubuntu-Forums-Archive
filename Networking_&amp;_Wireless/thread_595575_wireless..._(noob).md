### Post by khughitt on 2007-10-28
Try checking the restricted drivers manager (system -> admin -> restricted drivers manager), and see if there is a driver for your wireless card available there. If so, and the box is not already checked, try installing that driver. To see the networks available at your location, once you have the driver installed you simply click the network icon in your system tray, and a list of networks should pop up.

Goodluck!
