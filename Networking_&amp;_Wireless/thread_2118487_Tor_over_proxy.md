I want to use tor over a wifi network that requires web-based authenticaton.

How can I go about finding the IP, proxy type and port of this proxy so that I can enter this information in tor settings?

### Post by Oranos666 on 2013-02-22
I figured it out. In the Tor network settings I ticked the box which reads: "My firewall only lets me connect to certain ports" - "Allowed ports 80, 443".

Tor connects to its network after this is ticked. :guitar:
