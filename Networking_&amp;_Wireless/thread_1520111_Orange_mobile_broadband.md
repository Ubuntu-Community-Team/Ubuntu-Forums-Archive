Thanks - the link you gave was very helpful.

I spoke to Orange Help desk again. This time I got somebody who knew Linux well and he told me that they had tested Linux on an Asus EEE using both Xandros and Ubuntu.

He was able to talk me through the installation - settings for my particular contract are as follows -

Username = user
Password = pass
APN = consumerbroadband

When I set up the connection initially he told me to choose "Contract with VPN".

The only thing I discovered was that I needed to install a couple of progs from the repository to make my PC recognise the USB dongle ok. These are discussed in the following link (mentioned in the thread given earlier by the previous poster) -
