### Post by HermanAB on 2007-12-08
There are various ways to share data:
1. FTP
2. SSH
3. NFS
4. Samba

Samba is standard Windows networking and is horribly complicated and error prone.  Any of the other methods are a lot simpler.  For example, if you search these forms, you'll find oodles of problem threads on Samba and pretty much none on NFS - that should tell you something.
