I have Ubuntu 10.04 LTS installed in a HP Pavillion dv1000 laptop, it has a Intel Pro 2200 wireless adapter
After installing WICD to manage wireless connections I found that my laptop couldnt connect to the internet via wireless
After some research I discovered that Network Manager had to be uninstalled completely before installing WICD, so I did it using terminal
Now I can connect to the web via wireless
Hope it helps
