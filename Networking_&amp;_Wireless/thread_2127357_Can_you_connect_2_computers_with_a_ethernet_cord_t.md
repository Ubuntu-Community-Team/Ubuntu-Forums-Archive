### Post by SeijiSensei on 2013-03-19
You can connect them together with a "[crossover]("http://www.newegg.com/Product/ProductList.aspx?Submit=ENE&N=100016774%20600026306&IsNodeId=1&name=6%20%28Crossover%29")" Ethernet cable, but merging them into a single processor requires some specialized software.  A better arrangement might be to dedicate one of them to being a file server and use the other for normal computing.
