### Post by Argus on 2007-05-03
I just got mine going again the other day.

Go to System-printer-add new printer.
   Select network, at this point I was asked for a user name and password for the computer that the printer was on, since it was Win98 is just hit cancel.
  
Next select the computer the network print is on and the type and that was it.

### Post by somersetsimon on 2007-05-03
> **heathduncan said:**
> the printer is not pluged into a pc it has a mini print server plugged into the router via cat 5e cable

I have a Netgear PS111 print server.

On the "new printer" setup I selected Network Printer, then Unix Printer (LPD). I was then asked for the Host, so I puit in the IP address of the print server (in my case 192.168.1.7). I was also asked for the Queue. This was P1 for me.

This worked fine. Hope this helps

Simon
