### Post by PaulFr on 2008-02-03
I would suggest you start Firefox (same version) in Windows XP and in Ubuntu and go to **[http://www.speedtest.net/](http://www.speedtest.net/)** in each case and test your speed. Then you will have an objective idea of the difference in speed.

Various things to try based on output from above confirming your report : upgrade to latest Ubuntu, upgrade to latest patches / drivers for your card, monitor your DNS resolution speed (and perhaps replace your DNS entries if required), install _**Fasterfox extension ([https://addons.mozilla.org/en-US/firefox/addon/1269](https://addons.mozilla.org/en-US/firefox/addon/1269))**_**** for Firefox if required.
