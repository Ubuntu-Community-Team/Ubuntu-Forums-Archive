### Post by lisati on 2010-04-16
"Internal address" is a good term which helps us know what you're talking about.

The thing about *dynamic* addresses is that they're not permanent, and *can* change. If you want your server to have the same address all the time, you might need to set up a static address instead of a dynamic address. My preference is to do that with my router, which has an option "use same address" when I'm looking at the information for one of the devices attached to it.

Wikipedia has an article on IP addresses here: [http://en.wikipedia.org/wiki/IP_address](http://en.wikipedia.org/wiki/IP_address)
