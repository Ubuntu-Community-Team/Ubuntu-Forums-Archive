### Post by sbnwl on 2014-08-30
Go to system tray, look for network manager icon, click on it.

Make sure "Enable Netwroking" is selected.

Click on "Edit Connections"

A new window appears, click on "Add" then select "Ethernet" and then click "Create"
Another window will appear.
Set the connection name, default is "Ethernet Connection 1"
Click "Save" and you are done.
Go to system tray again and click on the Ehternet connection to connect.
