### Post by koszta on 2010-03-10
Open a terminal and type 
sudo apt-get install pidgin

Then open pidgin from applications-> internet -> pidgin

You can add yahoo accounts as well as gtalk as well as 1000 of others...
If you want to use linux then you should for the very most part run linux-native programs. If you want to use linux and just use wine for everything then you should have stayed with windows. I recommend pidgin above any IM tho, it is extremly stable and works perfectly well
