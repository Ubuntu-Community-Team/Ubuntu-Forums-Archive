### Post by wormyblackburny on 2011-03-02
Ok, I am experiencing ridiculously slow wired and wireless speeds. We're talking .01mb/s or less. So using the power of Google and after reading a bunch of articles here, I found this little tidbit:

sudo iwconfig wlan0 power off

Sweet! Speed jumped instantaneously to an acceptable 1.8mb/s. After a few reboots, its back to slow, great....try the same command, nothing. I see there are some updates to run and a new kernel available, so I give that a try... Still horribly slow. If I boot back into the old kernel, still slow. Ok, so maybe just wireless driver is garbage, so I switch to wired....same speed. Now after a reboot (as if to add insult to injury) the speed is even SLOWER.... Please help :(
