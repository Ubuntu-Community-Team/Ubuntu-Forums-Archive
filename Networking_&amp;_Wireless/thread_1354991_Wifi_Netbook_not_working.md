### Post by benjaminsuman on 2009-12-14
Make sure the wireless adapter is enabled in the BIOS.  Also, if it is a Broadcom wireless adapter you will need to install the driver for it using the Hardware Drivers tool.  You may need to plug it into a wired connection to download the drivers initially.

### Post by MaindotC on 2009-12-14
The FN + F8 command is a hardware function that will turn on/off the wireless card regardless of the operating system installed.  You may not see a litle light indicating it, but it's a hardware feature not a o/s specific feature.  Try following the [Ubuntu Wireless Troubleshooting Guide](https://help.ubuntu.com/community/WifiDocs/WirelessTroubleShootingGuide) and explain precisely where you have having trouble if you can't complete the troubleshooting.
