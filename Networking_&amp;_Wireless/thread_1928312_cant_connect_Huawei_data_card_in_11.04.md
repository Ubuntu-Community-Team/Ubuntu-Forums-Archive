### Post by p1977p on 2012-07-01
If your modem was detected correctly it should show a 'New Mobile Broadband connection' in NetworkManager applet. It show prompt you with default values for India --> Tata Docomo --> tata.docomo.internet (for GPRS -- use 'tatadocomo3g' as APN  for 3G).  Accept them to create connection. you don't need 3rd party software.
