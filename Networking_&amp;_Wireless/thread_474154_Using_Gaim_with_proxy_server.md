At the moment under  'edit account'-'advanced tab' I have ticked the boxes 'use gnome settings' and also 'use http method' with all the server details, password etc but neither way works. Ive spent hours trying different settings but I cant get it to connect through msn.
 It is coming up with  'xxxxx@hotmail.com disconnected: Failed to connect to server.' Can anyone help me? Any help is greatly appreciated.;)

### Post by hannah009 on 2007-06-25
Just thought Id put up a reply since I worked out the problem myself. I had to open up 'telnet' and put in the command to open up 'ienabler', which allows you to directly connect to the university connection. After it asked for my username, password etc, I could then tell it to allow certain programs/url addresses to be accepted by the proxy server. However, I actually needed to go into the network settings of Gaim and tick 'enable http method' and enter the proxy server, which I got from one of the University computers in the library. So all is well now! I can chat at Uni even though I probably shouldnt be since thats what the proxy server is for!
