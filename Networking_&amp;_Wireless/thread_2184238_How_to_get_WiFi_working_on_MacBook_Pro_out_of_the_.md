### Post by phocks on 2013-10-28
This might not work for everyone, but getting wireless working in Ubuntu had always been a messy annoying issue for me that usually always required an external WiFi adapter. But I accidentally came across this method while booting the Raring Ringtail 13.04 live CD on USB and thought I might post it here to see if anyone else has any luck.

Anyway, here's what I did on my MacBook Pro mid-2012 model:

[LIST]
[*]Boot into Ubuntu (I used the [Mac linux USB creator]("http://penguintosh.com/tag/linux-usb-creator/"))
[*]Open "Software Updates" from the Applications menu search
[*]Click on the "Additional Drivers" tab
[*]The Broadcom Corporation BCM4331 Linux STI wireless driver source... will show up
[*]Click the radio button to enable it, then click "Apply Changes"
[*]All done! You should now have wireless in the upper right corner.
[/LIST]

Please let me know if this method works for you. Thanks.
