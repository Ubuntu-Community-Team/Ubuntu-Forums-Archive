### Post by ajgreeny on 2012-06-23
If the wireless is protected with wep or wpa encryption you will need to supply a password before you can connect, and the system will normally ask you to enter the password.

If you do not get to that point it is possible that the wireless adapter is incapable of connecting to encrypted wireless, as I used to have to deal with when using a netgear pcmcia adapter in an old laptop.  The newer versions of ubuntu have overcome that problem completely for me, thank goodness.

Do you see any available wireless networks if you click on the wireless network icon in your panel?
