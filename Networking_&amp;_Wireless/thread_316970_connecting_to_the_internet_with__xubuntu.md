### Post by goatflyer on 2006-12-12
You shouldn't need to, not really.  Unless you are planning to run a mail server on your Xubuntu box, to switch mail to and from windows boxes, in which case you would install ClamAV to check for windows viruses.

You also don't need any firewall provided:
1) you only run software from trusted repositories
2) you aren't running any services that haxxors would try to crack into (by default in Ubuntu there aren't any services started)

Enjoy safe computing!
