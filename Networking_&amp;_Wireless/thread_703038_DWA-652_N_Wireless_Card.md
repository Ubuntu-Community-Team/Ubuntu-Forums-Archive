Well just a quick note. I did install 7.10 of Ubuntu on my laptop, and could not get the DWA-652 Xtreme N Wireless card working at all. So I opted for other options. I  downloaded Xandros Linux,
and the card is now working. It seem that Xandros already had the ndiswapper software all installed and ready to run. All I did was install the INF file and that was it. Workiing like a champ. No problems no hassles no reprogramming or writing any scripts or anything like that. It just works out of the box no problems at all.

Just thought  I would let you know.
