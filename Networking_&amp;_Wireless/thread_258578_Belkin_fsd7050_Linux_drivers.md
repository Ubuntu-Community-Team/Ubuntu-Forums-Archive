Thank you for contacting Belkin Technical Support.
 
We understand that you want to know which versions of the wireless USb adapter supports Linux Operating system.
 
Make sure you check for the version of the wireless adapter before downloading the drivers. You can find the version of adapter on the box or on the adapter itself. If the model number is followed by F5D7050 2xxx (2000 series), it is version 2, version 3 will be mentioned as F5D7050 3xxx (3000 series) and version 4 will be mentioned as F5D7050 4xxx (4000 series). Anything other than these three would be version 1.

We are sorry to let you know that we don't have the drivers for Linux Operating system on our website as of now. We don't support open source Operating systems. However we can provide you a link where you can search for appropriate drivers and install the wireless USB adapter on your computer.
 
The F5D7050 version 2000,3000 and 4000 supports Linux Operating system. The chipset used in F5D7050 version 2000 and 3000 is "Ralink" chipset and the chipset used in F5D7050 version 4000 is "Zydas".
 
You may refer to the link given below to download the drivers for the F5D7050 version 2000 and 3000 wireless USB adapter.
 
[http://www.ralinktech.com/supp-1.htm](http://www.ralinktech.com/supp-1.htm)
 
You need to use the search engine to find the drivers for the F5D7050 version 4000.
