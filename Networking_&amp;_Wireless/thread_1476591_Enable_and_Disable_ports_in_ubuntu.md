### Post by royleith on 2010-05-09
The 'Network' application in Administration no longer brings up the display of the network interfaces that are installed. That pane has the features necessary to enable and disable the interface and also to decide on static or dynamic addressing and the wifi passwording amongst other features.

Running the program in 10.04 just brings up the DNS and other settings for the interface currently selected in the Network Monitor applet in the system tray. If that interface is not running there is no way to start it.
