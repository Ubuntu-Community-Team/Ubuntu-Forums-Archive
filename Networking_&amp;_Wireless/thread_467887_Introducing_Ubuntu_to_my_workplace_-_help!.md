### Post by EndPerform on 2007-06-08
For starters, how is the network printer accessed in Windows?  Is it via a printer share?  If so, it's possible you might need to authenticate to get it to work correctly, same with the SMTP server.  Although, before you go romping around the network, you might want to check with your IT guys (unless are one of those) and make sure they're ok with it.  Last thing you need are some sysadmins upset at you.  Additionally, they might be able to give you some information on how the printer and email server are set up. :)
