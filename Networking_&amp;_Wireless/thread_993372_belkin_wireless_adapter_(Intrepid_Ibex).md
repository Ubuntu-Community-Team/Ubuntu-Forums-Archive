### Post by spcwingo on 2008-11-26
Install ndisgtk through synaptic then pop in the belkin install disc.  Then go to system>admin>install windows wireless drivers.  Choose install new driver and a window will pop up.  Choose cd rom on the left and find the xp driver .inf file and choose it.  I hope this gets you up and going.  Let me know how it turns out for you.
