### Post by spd106 on 2007-01-03
Plug it in through the ethernet port (RJ-45), you may need a crossover cable, but try a normal one first. Also consider using a switch or a hub to connect to the router.
If you want to use the USB port then you will need to search for a linux driver, which isn't likely to be easy.

Make sure that the router has DHCP turned on, it should give you an address automatically. Try navigating to it's web interface through firefox.
