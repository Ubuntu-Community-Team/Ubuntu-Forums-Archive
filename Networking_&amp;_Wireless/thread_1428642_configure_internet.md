### Post by Iowan on 2010-03-13
From a terminal (Applications>Accessories>Terminal) enter ```
sudo lshw -C network > hardware.txt
``` Copy the file (hardware.txt) to a floppy, USB stick, etc., so it can be transferred to whatever computer you're using for internet access - post it here.

MY DSL modem acts like a router, so my workstations are set up for ethernet access.  Not all do - that's why more details on your setup would be helpful.
