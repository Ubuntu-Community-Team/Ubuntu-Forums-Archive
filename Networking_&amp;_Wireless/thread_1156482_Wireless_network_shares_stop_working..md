I have completed my testing of the Ubuntu networking problem on my Asus Eee PC1000HE.   Guess what the problem is definitely in Ubuntu's wireless functions.   I have tried the desktop 9.04 installed on this netbook and it does exactly the same thing.   The shared devices and printers work for a few minutes ie. maybe 15 minutes and then quit and become unavailable and then will disapear from the networked items panel.    If the netbook is connected via cat 5 cable and the wireless is turned off/disabled everything works and continues to work with no problems.    But as soon as the wireless is turned on/enabled the networked items stop working and become unavailable even with the Cat 5 cable still connected.    So the problem has to be with Ubuntu's wireless driver interfearing with the networking device contol panel.

Remember all networking items work perfectly with Windows XP wirelessly so I think there has to be a coralation with the driver Ubuntu is using for the wireless on this netbook.
