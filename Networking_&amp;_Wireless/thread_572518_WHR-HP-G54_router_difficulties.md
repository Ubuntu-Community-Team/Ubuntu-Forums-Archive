Are you talking about the admin password to access your router and configure it? If so you need to figure how to do a hard reset. This will restore all settings back to default factory settings.

for some routers there is a button that does a hard reset, some are only a soft reset. For routers with a soft reset, you unplug and immediately press the reset button when plugging back in. Hold for 30 seconds. 

See if you can find your manual or an online manual for your router.
