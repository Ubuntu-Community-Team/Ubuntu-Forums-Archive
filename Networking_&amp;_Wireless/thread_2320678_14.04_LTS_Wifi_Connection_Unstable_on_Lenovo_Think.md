### Post by Hadaka on 2016-04-16
Hi, This may be the root cause of some issues with your computer..
```
"I have a non-genuine Windows 7 and when I tried to upgrade to a genuine Windows 10 there were some problems"
```
Since your windows 7 was of "questionable origin" and windows 10 would need a legitimate source to upgrade to, I suspect that is
the reason windows is no longer functional.Windows also applies EULA "End User License Agreement" to its system, that conflicts
with Ubuntu at times. Hopefully you did a full windows system backup before you decided to load Ubuntu.
 This command in linux removes a module..
```
sudo modprobe -rv rtl8723be
```
This needs to be done to change parameters in the module. So to change the antenna selection from 1 to 2 
the module is removed, changed then re-actvated. When changing setting to wireless, those changes should be done on a
wired ethernet connection, not wireless because you "turn off" wireless when you remove a module it is using to function.
So, please use a wired connection and try the commands again from the first command in the rtl8723be link.
I hope this gives you a better understanding of why you are having a difficult time with this.
