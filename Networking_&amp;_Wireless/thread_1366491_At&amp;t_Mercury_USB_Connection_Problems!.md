### Post by GeorgeVita on 2009-12-28
Hi **Command+Z**, welcome to this forum!

There is a similar thread with a solution for Ubuntu 9.10 at:
[http://ubuntuforums.org/showpost.php?p=8539368&postcount=18](http://ubuntuforums.org/showpost.php?p=8539368&postcount=18)

The idea is that DNS are not managed well and you have to setup them manually. Try also enable/diasble IPv4 settings Automatic PPP and/or addresses only (from network manager icon, editconnections, mobile broadband, edit, IPv4 tab).

In any case it is better to update first your Ubuntu 9.10 via ethernet or wifi.
