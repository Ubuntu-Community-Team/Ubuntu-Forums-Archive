### Post by TheFu on 2015-03-29
I appears you are connected via wifi.  For beginners, it is best to only have 1 network connection active at a time.  So - turn off the wifi connection and enable the wired connection.  I didn't look for issues specific to your driver (seems fine) and I don't use network manager. Basically, go into the GUI for NM and disable the wifi, then enable the wired port.  

It just looks like a configuration issue to me - not related to drivers.

### Post by michi1983 on 2015-04-07
alright, try this please:

edit the file ```
/etc/network/interfaces
``` and add the following two lines:

```
auto eth0
iface eth0 inet dhcp
``` 

I suppose nano editor for this. In other editors the save and exit procedure my vary.
After filling in this line hit CTRL+X, type y for yes and hit enter to save the file.

After that type in ```
sudo /etc/init.d/networking restart
```

Then please type ```
ifconfig
``` again and post the output here.

greetz
