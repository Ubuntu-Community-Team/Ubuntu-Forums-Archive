[http://www.realtek.com.tw/downloads/searchView.aspx?keyword=rtl8187](http://www.realtek.com.tw/downloads/searchView.aspx?keyword=rtl8187)

I would download the B, save it to your desktop then unzip it to the desktop.  Go to systems, admin, then the wireless drivers at the bottom.  Open and add driver point it to the .inf file inside the RTl8187 folder (you need the whole folder though) if you install it and it says yes you are good to go if it says no you may need one of the other letters.  Usually ndiswrapper needs re-booted.
