### Post by capscrew on 2009-02-09
Netdisco is a network management application targeted at large corporate and university networks. Data is collected into a Postgres database using SNMP. See: [http://netdisco.org/]("http://netdisco.org/")

This is definitely for managed switches in large dynamic (always changing) networks where the network administrators need to monitor and control switch port usage.

The list of switches known to work is available [**here**]("http://netdisco.org/DeviceMatrix.html")
