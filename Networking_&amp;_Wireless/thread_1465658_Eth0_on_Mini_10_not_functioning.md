### Post by chili555 on 2010-04-30
Your system wants to use */etc/network/interfaces* OR Network Manager but not both. NM will manage interfaces which are NOT listed in *interfaces*. If you want to use NM, revert the interfaces file to the lo stanzas only and restart networking.```
sudo /etc/init.d/networking restart
```Does eth0 reappear in ifconfig?
