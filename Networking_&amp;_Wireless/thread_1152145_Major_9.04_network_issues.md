### Post by runebinder on 2009-05-07
Installed Ubuntu 9.04 yesterday, so far not impressed.

I'm used to Ubuntu not recognising my BCM4318 card, happened in 8.10 as well. However in Intrepid was a nice easy fix, set my girlfriends computer up for internet sharing, connect the 2 together via LAN and then after a while of being online it would give me the option to use the proprietary driver for it.

So did the same with Jaunty, network connection has the loading icon, then a message "wired network, you are now disconnected and offline" Going to try lugging it all down 2 floors to connect it to the cable modem tomorrow, needless to say I'm not impressed.

You would have thought the developers would have sorted this by now, the Broadcom Airforce 1 chip set is pretty old, why not include the necessary files and driver in the install package? 

I read some where that network device driver support had been improved with Jaunty, so far I'd say it's gone backwards :(
