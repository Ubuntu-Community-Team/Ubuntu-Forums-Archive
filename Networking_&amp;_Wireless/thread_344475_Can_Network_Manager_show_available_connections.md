### Post by randomnumber on 2007-01-23
what the link  shows pictures of is a prgm that is called gnome-network-manager.

you can install it with this how to: [http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_Configure_Ubuntu.2FKubuntu_with_WPA_using_Network-Manager](http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_Configure_Ubuntu.2FKubuntu_with_WPA_using_Network-Manager)

also if you want to use the software that is installed by default go to System-> Admin.. -> Networking.

Select the connection device you want to use and go to the properties. If wireless, there should be a drop down menu you can select the network you want to connect to.
