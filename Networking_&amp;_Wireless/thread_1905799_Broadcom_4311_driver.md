### Post by westie457 on 2012-01-08
Hello, as stated in my first post here remove ndiswrapper, and also remove any other drivers that are installed. The gui way is open synaptic and search for 'bcm'.

In the list mark the ones that have a green box for complete removal and mark b43-fwcutter for installation (see screenshot).

Then open a terminal, type and run```
sudo modprobe -r b43 wl
```

ignore any error messages.

Then do ```
sudo modprobe b43
```

wait a few seconds then try connecting.

The two commands are for testing and to make the change permanent a reboot is usually sufficient.
