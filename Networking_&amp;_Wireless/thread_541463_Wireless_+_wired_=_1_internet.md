### Post by darkwyrm on 2007-09-04
There are quite a few NICs that work well. I personally use D-Link DWL-G510s in the network I administer. They're cheap, reliable, and work well in my experience. Working out-of-the-box in Feisty. I have personally gotten a wifi + wired server together. It's not that bad - setting up Samba is about the hardest part, provided that the wired and wireless networks have different addresses and/or different subnet masks (which I'm still learning about). Adding a second wifi card is something I'm fighting with right now.
