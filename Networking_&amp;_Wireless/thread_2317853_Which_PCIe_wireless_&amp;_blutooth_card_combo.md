### Post by weatherman2 on 2016-03-20
I've had great luck with Intel mini-PCIe wireless cards in Ubuntu.  I've used the Intel 6235 (dual band + bluetooth) in a several laptops.  I've also got an Intel 7260 (dual band AC + bluetooth) in my current laptop; it has some issues, actually, but I suspect it is related to incompatibility with the laptop.  WiFi works fine, but bluetooth is unreliable.  Using a 6235 card in the same laptop, WiFi worked great and bluetooth was reliable.

I did have to build the latest iwlwifi driver for Ubuntu 12.04 for the 7260.  I don't think I would have to for the newer versions of Ubuntu, but even so, it wasn't that difficult.
