### Post by grahammechanical on 2016-04-15
Is there a tick mark against Enable Networking? There should be a tick mark against Enable Networking otherwise the ethernet adapter will be switched off. If you do not see an entry for Enable WiFi, then either you do not have a wireless adapter or it is disabled in the BIOS or it is switched off by a keyboard combination.

When Enable Networking is off the Network Manager icon in the top panel & on the right looks like an upside down cone. When connected by WiFi the icon becomes an upside down cone with concentric arcs radiating upwards. When connected by ethernet the icon becomes two arrows pointing in opposite directions.

If we have disabled networking then clicking the icon of an upside down cone will bring up a small menu that says Networking disabled (greyed out); Enable Networking; Connection Information (greyed out); Edit Connections.
