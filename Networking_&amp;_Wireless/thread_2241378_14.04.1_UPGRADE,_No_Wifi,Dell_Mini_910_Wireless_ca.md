### Post by varunendra on 2014-08-26
You need the firmware for the driver to work -> **joewx said:**
> ```
....
[   12.508599] b43-phy0 ERROR: Firmware file **[COLOR="#FF0000"]"b43/ucode15.fw" not found[/COLOR]**
[   12.508804] b43-phy0 ERROR: Firmware file **[COLOR="#FF0000"]"b43-open/ucode15.fw" not found[/COLOR]**
[   12.509002] b43-phy0 ERROR: You must go to http://wireless.kernel.org/en/users/Drivers/b43#devicefirmware and download the correct firmware for this driver version. Please carefully read all instructions on this website.

```

If you can connect to internet via Ethernet cable or other means, install it with -
```
sudo apt-get install linux-firmware-nonfree
```
Then either reboot, or manually reload the driver with -
```
sudo modprobe -rv b43
sudo modprobe -v b43
```

If you have no way to connect to internet, follow this post to install the firmware offline : [http://ubuntuforums.org/showpost.php?p=13062169](http://ubuntuforums.org/showpost.php?p=13062169)
