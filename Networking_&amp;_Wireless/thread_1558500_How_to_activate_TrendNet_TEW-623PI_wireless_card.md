### Post by finkey on 2010-08-22
_Step One_
You may have to do this every time you start Ubuntu.

Start the wireless card in Ubuntu 10.04:

Open a terminal window.
Type this:  

**sudo dhclient**

Type your password.
Exit the terminal window.

Just in case,
Right click on Network Connection icon in the system tray.
Open Edit Connections
Open Wireless tab.
You may see a duplicate of your wireless connection.  
Delete the one starting with **auto**.

The network icon in the tray will show as unconnected.
Ignore the network icon in the tray.

_Step Two_
You may have to do this every time you start Firefox.

Start Firefox browser:

Open browser Firefox.
Ignore the Off line Mode page.
Open **File** on top menu.
Uncheck Work Offline in the drop down list.
Hit Try Again button.
Browser will connect to your home page.
