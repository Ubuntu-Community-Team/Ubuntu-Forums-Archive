### Post by jeremy31 on 2014-08-13
Here is something to try
```
sudo gedit /etc/modprobe.d/iwlwifi.conf
```
and add this line
```
options iwlwifi 11n_disable=8
``` and reboot

---

### Post by skiironmask on 2014-08-13
That fixed it, thank you very much. What does the 8 parameter do? I had tried 0 and 1 and it did not resolve the issue.

---

### Post by jeremy31 on 2014-08-14
It enables TX AMPDU, and it was disabled by default because it causes issues for some chipsets
