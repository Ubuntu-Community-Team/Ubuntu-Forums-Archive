### Post by spd106 on 2008-11-25
The network configuration tool can be quickly accessed through the applet on the panel in the top left side, the one next to the sound applet.

When you click on it there should be a list of networks to join. If not try right-clicking it and see if Enable Wireless is an option.

See [https://help.ubuntu.com/community/NetworkManager0.7](https://help.ubuntu.com/community/NetworkManager0.7)
