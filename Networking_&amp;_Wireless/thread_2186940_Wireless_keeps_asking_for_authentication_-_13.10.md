### Post by dentaku65 on 2013-11-10
Hi,
on my Atheros I had to install and configure crda
```
sudo apt-get install crda
```
Then:
```
sudo nano /etc/default/crda
```
then change the line REGDOMAIN= to your country code, eg.
REGDOMAIN=IT
