### Post by dineshs on 2010-11-27
MT880 modem has DHCP disabled by default.You may try this
Step-1 Manual IP configuration
[http://ubuntuforums.org/showpost.php?p=9467538&postcount=5](http://ubuntuforums.org/showpost.php?p=9467538&postcount=5)
Note:You may use the same IP addresses given as examples
Step-2 Modem configuration (for always ON internet)
Open browser (E.g. Internet Explorer, Opera , Firefox)
Type 192.168.1.1 in Address bar. click GO
Enter both Username and Password as admin
Click OK
Click the + sign left to home
Click WAN settings 
select PPP as WAN type
Enter  Username and Password as allotted by BSNL 
Select max idle time -Always on
Apply the settings
The system will ask for reboot .Tick Yes and Click OK.
Wait till reboot complete (3 minutes).
Open browser and see if pages are loading
