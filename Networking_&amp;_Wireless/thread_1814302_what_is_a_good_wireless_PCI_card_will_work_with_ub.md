### Post by the_chink_between on 2011-07-29
Hi, I recently replaced my old wireless card (a Belkin "Wireless G") as it had been flaky for some time.  However it's worth saying that the Belkin had worked with the Ubuntu version that I'm using (11.04), albeit flakily (but then it was flaky under Windows XP too).

The new card is a TP-Link 150Mbps Wireless N PCI Express Adapter, Model No TL-WN781ND.  It's also working fine, right out of the box.

Hope this helps.
