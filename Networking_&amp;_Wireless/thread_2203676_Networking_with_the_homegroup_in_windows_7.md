### Post by TheFu on 2014-02-04
"Homegroup" is proprietary and little use to Linux.

You need to use old-style Windows Sharing.  Right click on the folder in Windows that you want to share, choose "Share with ..."  then Specific people.
On the Linux side, after the folder is shared from Windows, use any file manager you like and "browse" the network.

If you need/want to go the other direction, we need to know which version and release of Ubuntu you are running. Newer releases make sharing easier.
