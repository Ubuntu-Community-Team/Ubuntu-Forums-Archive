### Post by Denver Dave on 2011-09-26
I had quite the time getting my acer xp to connect to my own wireless network, so I was not surprised that I also had trouble connecting my new acer windows 7 to my wireless network.  Despite attempts for several months, I could not do it, gave up and used a wire.

With ubuntu I initially brought up the network connections window, didn't see anything and game up.  Then I fooled around with the network network connections icon on the top right of the screen and was floored that I was able to connect.

In fairness, it is easier when we temporarily broadcast the ssid and have the router settings screen visible, but ubuntu is definitely easier than windows 7, partly because ubuntu displays the hidden wireless name once setup.

An internet search for windows 7 hidden wireless connections, shows that I wasn't the only one that had trouble.

I an now connected to my hidden wireless network with both windows 7 and ubuntu, but ubuntu was definitely easier to setup and modify. Without ubuntu, I'm not sure I would have ever had the persistence to get my windows 7 to connect.   

Thanks ubuntu.
