### Post by mike71ghia on 2007-05-09
It was in the smb.conf. The line with socket options = TCP_NODELAY  
Needs socket options = TCP_NODELAY SO_RCVBUF=8192 SO_SNDBUF=8192
Works fine now.

It's located in /etc/samba. To edit it, press Alt-F2 to get the run application dialog and type ```
gksu gedit /etc/samba/smb.conf
```
Enter your password when requested..
