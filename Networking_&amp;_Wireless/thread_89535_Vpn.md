### Post by chiefofthejojos on 2005-11-13
I guess it depends.  I've read many good things about freeswan/openswan, but I don't think they will work for certain configurations.
At my company, we use a checkpoint firewall, and our network administrator asks us all to use the Checkpoint SecuRemote client from their website.  Unfortunately, their linux client was written for Redhat 7.1 and hasn't been updated for years.  I attempted to get it to work with Ubuntu, but gave up.  I've also attempted to run the windows client with wine, but that didn't work either.
The "solution" I've been using involves port-forwarding with ssh.  More about that in this thread: [http://www.ubuntuforums.org/showthread.php?t=21105](http://www.ubuntuforums.org/showthread.php?t=21105)
