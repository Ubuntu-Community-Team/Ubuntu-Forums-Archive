### Post by rvbhute on 2009-01-17
Network Configuration :)

1. Go to Preferences > Network Configuration. Select the type of network (I use ADSL, have a ADSL router connecting to my LAN card. So for me, its Wired).

2. Click on the Add tab, then the IPv4 tab and off you go!

You should see this connection come up in the Network Manager applet.
