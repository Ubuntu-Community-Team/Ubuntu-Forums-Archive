### Post by mtalexan on 2007-05-04
Follow the directions on the DialupModemHowTo on the Wiki.  Basically you use a program (exactly as directed) to detect your modem and tell you what type it is.  Next you have to install the module (directions included in readmes) and then compile the drivers.  It takes some work to understand if you don't know anything about Linux but there's enough information there to help you figure out how to do it.
