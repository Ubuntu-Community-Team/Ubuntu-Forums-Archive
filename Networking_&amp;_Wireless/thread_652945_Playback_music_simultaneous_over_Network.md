### Post by tgalati4 on 2007-12-29
Yes and no.

You can set up several machines to play the same network stream.  The asyncronous internet protocol does not allow time-alignment of music playback, so each machine will have a slight delay.  This makes basic mp3/ogg streaming unsuitable for a whole-house sound system.

There are some concepts in development to allow music collaboration over the network--having several muscians playing at the same time and have the signal mixed and played at each location.  

What are you trying to accomplish?
