### Post by multi on 2007-09-19
Hi,

I was able to share my printer connected to a Ubuntu machine to 2 computers running Vista and XP using  CUPS instead of Samba printing. 
I basically followed this howto: [https://help.ubuntu.com/community/NetworkPrintingWithUbuntu](https://help.ubuntu.com/community/NetworkPrintingWithUbuntu)
I've been told that CUPS is considerably easier to setup and use in comparison to Samba printing.

Some more info on Cups and Samba printing: [http://ubuntuguide.org/wiki/Ubuntu:Feisty#Print_Server_.28cupsd.29](http://ubuntuguide.org/wiki/Ubuntu:Feisty#Print_Server_.28cupsd.29)
