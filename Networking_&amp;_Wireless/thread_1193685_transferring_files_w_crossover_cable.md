### Post by dmizer on 2009-06-22
Is this a one time transfer, or is this going to be a permanent arrangement?

For one time transfers, you're best off just assigning static IP to both boxes and using SCP via the command line. That's your fastest and most pain free way. More information here: [https://help.ubuntu.com/community/SSH/TransferFiles#Secure%20Copy%20(scp](https://help.ubuntu.com/community/SSH/TransferFiles#Secure%20Copy%20(scp))

For a more permanent arrangement, see the 4th link in my sig.
