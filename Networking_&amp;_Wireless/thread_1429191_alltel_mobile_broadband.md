### Post by d3v1150m471c on 2010-03-13
You don't use a username or password. When you plug in the usb modem, create a mobile broadband connection, set it as alltel(should be the first option). Name the new network and you will find it in the menu of your network manager icon. Click on it and allow it to connect. If it doesn't connect from there then it could likely be alltel's well-known awful service.
