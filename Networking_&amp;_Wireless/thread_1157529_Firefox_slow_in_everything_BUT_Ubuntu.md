### Post by Iendicis on 2009-05-12
**EDIT:** For those of you who arrived to this post via google or the forum's search function, I've nailed down the speed issues to the about:config settings in Firefox. Just type that into the address bar and google around for some precious speedy settings.

This is largely an academic question if anything, but I'm curious.

When running Ubuntu (any release since 8.04, I can't remember 7.10 that well) Firefox runs great. Google takes five to ten seconds to load, which matches Windows XP/7 on this machine.

But everything outside of a default Ubuntu or Xubuntu install in the Linux world is a different story. Debian 5.0 (Gnome, KDE, Xfce, and LXDE) takes around thirty seconds for Google to load. This should not happen. Debian runs much, much faster than Ubuntu on this machine, as seen by the stunning OpenArena performance.

I even tried making an LXDE installation from scratch in Ubuntu 9.04 the other day. Same story. Very, very slow web browsing experience. Pages take twice to three times longer to load.

The following phenomena has been seen in the following distributions:
-Debian (and other friends, such as MEPIS and AntiX)
-Custom Ubuntu's made with the alternate install disc
-CentOS 5.3
-Absolute Linux 12.2 releases

I have used a variety of filesystems, from ext3, ext4, jfs and xfs, to no avail.

Other results:
-Fedora (performance was better than the above, but this computer doesn't play nice with Fedora for some reason, so the program's performance was visibly poor.)
-Puppy Linux (Seamonkey was reasonably fast; around Fedora's performance.)

I realize this is an older computer (2.0 Ghz Northwood Celeron processor, 1 GB of RAM, a few selection of small hard drives), but Ubuntu runs fine, again. So I feel this is a software issue. Other distros work fine in the LiveCD when I use my laptop, however, but I'd like my desktop to work better online with other distros. I have a Motorola Surfboard modem and a Realtek ethernet port integrated into the motherboard, with a Belkin Wi-Fi router in between the two.

I would use Ubuntu on a regular basis anyway, but I'm curious to know why my web browsing is so slow outside of a default Ubuntu installation.

Are there any ideas?
