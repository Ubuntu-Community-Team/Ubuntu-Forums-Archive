### Post by alfplayer on 2009-03-23
You could create and use a new firefox profile to know how fast it can be (profiles are created without addons) using Firefox's profile manager.
If you find a big speed difference between the new profile and your regular profile you can disable the addons that you use the least, restart firefox and check if it is faster. You can disable more addons until you are satisfied.
The addons that you disabled can be reenabled again if you want (be careful because you may reenable the ones that were slowing down firefox).
Maybe someone that knows about firefox and addons knows which addons slow down firefox.
