That "linksys" network is almost for sure not yours because that's the default network name that Linksys routers all use. You should be able to access your router's configuration web page and see what your wireless network name is, also known as the "SSID". If you haven't done so all ready, you should change the default name to something else. Can you connect to your router by an ethernet cable? That is the safest way to set up your router.

Wait until you get your network name (SSID) setup, and make sure you can connect and use your router OK before you turn on encryption. And as far as encryption goes, WPA is the best and is very secure if you use a good password, WEP is crackable but is better than nothing.
