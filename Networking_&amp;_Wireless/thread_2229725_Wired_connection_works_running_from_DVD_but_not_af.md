### Post by Bashing-om on 2014-06-17
john.quinn; Hi !

As the floppy drive is present, there is no harm at all in leaving as is. 

As to that external hard drive. How is this for a thought, the file system has 
a) Become corrupted ;
b) At some point was not properly UNmounted, and there is a lock on the file system ?

Check for a locked condition, with the hard drive plugged in:
and find the mount point ( in /media or perhaps /mnt directories) ->
```

sudo lsof /<mount_point>
sudo fuser -m /dev/sdb1 ##assuming the mount_point is known as

```
What is the format imposed on the HD ?
```

sudo fdisk -lu

```

If it turns out the Format is fat16/32 or NTFS; recommended procedure is to sic Windows on it to repair the file system. 
Else we can sic e2fsck on it and see what results.
