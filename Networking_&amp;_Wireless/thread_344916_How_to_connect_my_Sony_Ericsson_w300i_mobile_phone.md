### Post by v4q3r0 on 2007-04-22
I bought a SanDisc memory card (M2) which came with an adapter.

Then I bought a SONY card reader.

it was plug and play from there amigo.

By the way - if you're trying to use an MP3 file as a ringtone - you'll need Audacity to chop the file down to around 30 seconds -- then just drag and drop your new ringtone/mp3 file into your "ringtones" folder in the card.

If you have trouble with that...email me...i had some trouble because Audacity needs 'libmp3lame.so' to convert the file into an mp3.  all you need to do is install lame via sourceforge. and look in usr/local/lib for the .so file.
