### Post by praseodym on 2011-09-07
Only in Firefox? Only with downloads? In FF you can turn off IPv6. Type

```
about:config
```
into the URL-line, search for the entry
```
network.dns.disableIPv6
```
and change it via double click to "true". Restart Firefox.

```
sudo su
