### Post by chili555 on 2014-11-10
You have compiled the driver for your currently running kernel only. When Update Manager, or whatever is used in solid linux, installs a newer linux-image, after the required reboot, re-compile:```
cd ~/RTL8188EU/rtl8188eu
make clean
make
sudo make install
sudo depmod -a
sudo modprobe 8188eu
```Please use thread tools at the top to mark Solved. The searchers will appreciate it.

Glad it's working!

-----

Note to searchers: This is included in Ubuntu 14.10 and probably 14.04, so no need to git clone for you!
