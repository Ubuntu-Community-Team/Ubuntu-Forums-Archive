### Post by tfrue on 2014-01-11
Since you're on the local network, it shouldn't slow down bandwidth per se, but it will make using applications on either computer slower especially transferring larger files.

Any time you use wireless to do anything, it will be slower than using wired. Just because you have an N router, doesn't mean that your hardware is N compatible. I assume that it is, unless you have an older laptop, but make sure nothing else is going on like torrenting, or gaming because that will slow down your network.

Routers today, and for a while now, are smart enough to prioritize data on the network know as QoS. So see if your router supports it.

So if you want to speed up your transfer speeds, hook your laptop and your desktop to your router through ethernet, it will speed up your speeds significantly.

Also, if you want to speed up wireless, make sure that you don't have any viruses, or using any other resource intensive programs. Also, distance from your router and the devices like microwaves and cordless phones that operate on the same frequency as N, will cause network issues.
