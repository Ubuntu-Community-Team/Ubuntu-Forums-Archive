Network manager does not show active networks. It does give you the option to Disconnect from the connection that you are connected with. Other connections will show Disconnect in grey letters. It will also show available wireless networks. 

If you click Disconnect on a wired connection, then network manager will try to connect to an unsecured wireless network. It will usually find the network with the strongest signal.

If your wireless network is set to Automatically Connect, then the manager will try to make a wireless connection to your modem. What has changed in the last two weeks?
