### Post by Bucky Ball on 2013-01-29

What isn't showing any wireless networks? You mean the name of the modem/access point (I'm presuming the modem is also a wireless router) is not showing on your local machine?

Are you sure you have the modem set up correctly? In the manual you should find its IP address (commonly 192.168.0.1). You will probably need to use an ether net cable directly from your machine to the modem/router to set it up. Insert that IP address into the Firefox address bar when you have a cable connected. That should take you to the modem's configuration page.

PS: If you have the old modem/router you could cable into that, check the configuration, scribble down the details and then replicate them in the new one. You don't need to be online to do any of this, incidentally.
