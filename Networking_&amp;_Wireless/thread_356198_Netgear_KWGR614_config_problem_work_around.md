### Post by fdimmling on 2007-02-08
The Netgear KWGR614 WLAN (and LAN) DSL router is running an open source Linux system. However configuring the router by use of a Firefox browser which is standard in Ubuntu systems does not work. You cannot proceed in the configuration dialog past the first page, Firefox simply does not respond to the Continue button clicked. Even if you manage doing all the settings by entering the address [www.routerlogin.com/basicsetting.htm](www.routerlogin.com/basicsetting.htm) for the next page directly (as suggeted by the Netgear hotline) you cannot use your nicely configured router because you are directed to the first configuration page every time you try to reach an address on the internet.

Work around is to install and use the Konqueror browser. :)
