### Post by alime on 2008-09-04
We can create wireless mesh routers using Open software. (Like the [OLPC  ]("http://wiki.laptop.org/go/Mesh_Network_Details"))
What would it take to create an educational network for low income families? 
Think of this, you build or buy devices that send and receive information on this network. You want to be on the network, you build or get a device that can communicate both ways. The ability of self routing devices, if more people joined the better the network would be. 
We would not need the Cable, Telco, or other networking powers that be. This could be done with a complete peer to peer arrangement.
Just a thought what do you think? Making a complicated problem to simple?
