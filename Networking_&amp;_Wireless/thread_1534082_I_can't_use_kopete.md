### Post by ankspo71 on 2010-07-19
Hi,
Are you using the Yahoo chat service? If so there are a few bugs at KDE that fit your problem's description.

[https://bugs.kde.org/show_bug.cgi?id=228410](https://bugs.kde.org/show_bug.cgi?id=228410)
[https://bugs.kde.org/show_bug.cgi?id=226699](https://bugs.kde.org/show_bug.cgi?id=226699)

Try opening kopete from the terminal/konsole to see if you get any error messages when you click on the send button too. That might give you an idea on what the problem might be.

I personally haven't seen this problem in kopete yet... I chat using google and facebook chat services though.

PS. there are several more chat clients to choose from too. 
Pidgin and Empathy are good multi protocol chat clients.
Hope this helps.
