### Post by Waveridr85 on 2010-02-10
Just a heads up to anyone who is having issues with this device.  What you need to do is:

Install Ndiswrapper

Download rt2870.inf driver 

Use Ndiswrapper to select the .inf file and install it

Reboot and it works.
