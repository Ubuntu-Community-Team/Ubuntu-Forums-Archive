### Post by praseodym on 2011-09-06
Just load the wired module and force its loading at boot:

```
sudo modprobe -v b44
echo b44 | sudo tee -a /etc/modules
sudo service network-manager restart
```
Wired should work now. After that you only need the firmware for you wireless device, STA driver is not needed:

```
sudo apt-get install b43-fwcutter firmware-b43-installer
sudo modprobe -rf b43
sudo modprobe -v b43
sudo depmod -a
```
