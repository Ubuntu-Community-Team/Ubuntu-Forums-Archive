### Post by badger_fruit on 2009-09-20
Checklist:-

1. Can you ping other computers on your local network or your default gateway by IP address?
2. Can you ping other computers on your local network or your default gateway by DNS name?
3. Can you ping the IP address of a remote computer (for example 209.85.227.103)?
4. Can you ping the DNS name of a remote-computer (for example [www.google.com)?](www.google.com)?)

Please answer those and then we may be able to help more :)
