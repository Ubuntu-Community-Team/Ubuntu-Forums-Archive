### Post by praseodym on 2013-05-06
You should be able to connect via the network-manager in "DSL". For this, open the file
```

gksu gedit /etc/network/interfaces
```and remove anything except
```
auto lo
iface lo inet loopback
```
Reboot and connect via NM.

### Post by praseodym on 2013-06-21
