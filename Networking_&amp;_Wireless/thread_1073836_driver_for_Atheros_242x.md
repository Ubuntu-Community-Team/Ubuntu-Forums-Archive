### Post by theDaveTheRave on 2009-02-18
try [here]("http://madwifi-project.org/wiki/About/ath5k")

it is the madwifi project page.

The link is for the ath5k project, it is the replacement for the ath_pc that used to run the atheros cards.

You'll need to confirm that the ath5k is the one you need on the help page, but there should be a link to the ath9k that should work otherwise.

There are plenty of threads out there for instalation, but the instructions on the ath5k are quite easy to follow, just post a message to one of the may threads if you continue to have problems.
