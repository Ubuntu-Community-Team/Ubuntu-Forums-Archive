### Post by maestrobwh1 on 2013-09-12
This stopped working after kernel 2.6.35 with the release of Natty.  I am curreently running precise with that kernel (Maverick) because of this issue.  Sound also stopped working shortly thereafter (Ocelot).  This issue gets no attention really which is somewhat understandable because this machine is not all that common.  

Actually, the Precise iso doesn,t boot without a whole lot of blacklisting.  To get the iso to boot (and it wont via usb, and no 3.x kernel in Ubuntu will on this machine), one has to add

```
nomodeset xforcevesa via_agp.blacklist=yes via.blacklist=yes pcspkr.blacklist=yes pcspkr.blacklist=yes snd_hda_codec.blacklist=yes snd_hda_intel.blacklist=yes psmouse.proto=bare 
```

Oddly, the sound icon appears and there are some pcm modules loaded in lsmod.  I did not test sound.  Wifi is still nill though.
