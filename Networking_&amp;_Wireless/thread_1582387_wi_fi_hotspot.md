### Post by louis--taylor on 2010-09-26
You can create a quick wifi hotspot using nm-applet (the applet in the notification tray which helps you to connect to wireless networks). Left click on the applet and select "Create New Wireless network". A window should then appear, into which you can configure your new wireless network. Enter a short name and choose an encryption type (use WPA if you want it to be secure) and key.
I think that's all there is to it.
