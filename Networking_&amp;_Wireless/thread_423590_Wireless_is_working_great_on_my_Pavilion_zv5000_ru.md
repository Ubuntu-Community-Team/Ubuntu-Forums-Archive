### Post by timmahhny on 2007-04-26
I like many have had a time of it getting my built in wireless (Broadcom BCM4306) to work running Feisty. I have been reading every post, downloading many different things, trying everything, but could not get it to work.
 I ran the gui front end to ndiswrapper
```
sudo ndisgtk
``` for the gui for ndiswrapper. The driver that I used was already installed, but did not show in ndiswrapper. I then reinstalled the bcm43xxx driver, went up to the network manager to see what I could do, and selected  eth1, and it just started to work. 
 I truly don't know what I did, as I have done so much, that I am now really confused on what I did to make it work, but it is now working. 
 I am glad that I installed Feisty. As a "newbie" only 18 months or so using Ubuntu, I have learned a great deal from this forum. 
 So for those that are having problems, just stick with it, and keep reading. You will find the answer. I will switch over to Windows and grab my bookmarks for all of the posts and places I went to get information; post it here, so that it may help others. 
 Ubuntu Rocks!!:lolflag: 
 Feisty is running very fast.
