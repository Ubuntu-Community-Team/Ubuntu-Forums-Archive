### Post by t0mppa on 2009-09-17
> **gtoro said:**
> 
   *-network UNCLAIMED  
        description: Network controller  
        product: AR9285 Wireless Network Adapter (PCI-Express)  
        vendor: Atheros Communications Inc.  
        physical id: 0  
        bus info: pci@0000:05:00.0

        clock: 33MHz  
        capabilities: pm msi pciexpress bus_master cap_list  
        configuration: latency=0  


The unclaimed bit means that no driver is getting associated with the interface and thus you can't use it. The Atheros native Linux driver for this card (ath9k) isn't included in the version of kernel that Jaunty (9.04), let alone any of the older versions, is installed with and thus the problem. See [here]("http://ubuntuforums.org/showthread.php?p=7409939") how to get the driver installed.
