### Post by spd106 on 2007-04-22
You can't use Network Manager without the nm-applet, so you might as well turn it off. You can use wpa_supplicant to connect to a wireless network it's just less convenient as you have to edit text files when you want to switch networks.

 This wiki page will help [https://help.ubuntu.com/community/WifiDocs/WPAHowTo](https://help.ubuntu.com/community/WifiDocs/WPAHowTo)
