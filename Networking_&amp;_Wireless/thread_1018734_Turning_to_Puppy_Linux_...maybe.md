### Post by spidermaniac on 2008-12-22
Since my using Ubuntu almost exclusively now instead of Windows, I must say that Ubuntu has a couple things to learn from Puppy Linux -- which I am now considering as more user friendly.

1. Whenever I install Ubuntu clean on a laptop or desktop with wireless card, I always have to go through all the ndiswrapper hoops, etc, needing a wired connection to start out with to get all the updates and then trying to find the wireless card's drivers to get the darn thing working...never does it work out of the box. With every Puppy install all I had to do is follow a couple steps and wireless is working. Nice job Puppy!

2. File sharing really sucks in Ubuntu. I can't for the life of me get passed the "255" error when trying to share files in Ubuntu even after following available help tutorials. At least Puppy has something called "pnethood" which automagically searches for all shares and finds my Windows shares right away and allows me to at least drop files to my Windows box.

These two areas really need cleaning up as my preferred Distro is turning slowly to Puppy Linux.
