### Post by Nebby on 2008-04-01
If you're wiring your pc up to the livebox via ethernet you don't need ndiswrapper or anything, as that's only for if the pc is using wireless. You should just be able to plug the cable in and be off (mine worked like that).
As for security, yes, the DS only supports WEP or no security, so you're best going for WEP encryption, which the Wii will also work fine with. All the wireless security is configured via the livebox itself.

Good luck :)
