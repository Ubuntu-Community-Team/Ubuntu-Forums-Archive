### Post by jeremy31 on 2017-11-30
First idea, download attached file, right click on file, choose extract here, then click on show files, then right click in window and chose open in terminal ```
sudo cp BCM.hcd /lib/firmware/brcm/BCM20702A1-0a5c-21e8.hcd
```
```
sudo modprobe -r btusb && sleep 20 && sudo modprobe btusb
```
Test pairing, if you have the same issues, then
```
sudo -H gedit /etc/default/grub
```
Find the line with GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"  about line 11
make it ```
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash pci=nocrs"
```
Save and exit the text editor, then
```
sudo update-grub
