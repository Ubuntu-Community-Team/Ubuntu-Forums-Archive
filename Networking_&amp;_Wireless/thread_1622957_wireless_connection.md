### Post by tacantara on 2010-11-16
If you know your router's IP address, just open a terminal session and run the ping command.  Here's an example:

```
ping 192.168.1.1
```

If you don't know the router IP address, you can find it by opening the Network Tools program (Main Menu > System > Administration > Network Tools).  Click on the traceroute tab, and enter a commonly used website (i.e. [www.google.com](www.google.com)) in the input box.  With a traceroute, the first IP address that comes up should be your router, as all traffic has to go through there before it gets to the outside.  Take that first IPA that comes up, and run a ping as explained above.  You can also ping using Network Tools (there's a tab for it).

I hope this helps :)
