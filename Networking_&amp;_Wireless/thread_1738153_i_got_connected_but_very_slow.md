### Post by majorawesome on 2011-04-24
Oh, and try to disable IPv6
1. open firefox
2. in the address type about:config
3. search for (CTRL + F) IPv6
4. change value of "network.dns.disableIPv6" to "true"
5. if that doesn't work try upgrading from dial-up to Verizon fios (It's fast!)
