### Post by Dobro_player on 2010-07-24
I have a desktop computer that does not have a wireless card and a netbook that has a wifi and bluetooth card. I want to share an internet connection between these computers and allow them to share access to the hard drives and dvd drives on these computers, and I want to encrypt the connection in some way so the neighbors can't use my internet. To do this, should I buy a wifi card for the desktop computer and plug my dsl modem into the desktop computer's ethernet port and then create a wireless network, or should I buy some sort of wireless router with ethernet ports and plug my dsl modem into the router and attach the desktop computer to the router using an ethernet cable?

---

### Post by linux1534 on 2010-07-24
The second option seems best. Otherwise your first computer has to be always, even only for the netbook to have internet access.

So yea, buy a wireless router with an ethernet port and connect your computers to it and have the router kinda do the sharing.
