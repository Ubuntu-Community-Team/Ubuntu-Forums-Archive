### Post by praseodym on 2011-12-30
The module acer_wmi is wrong on a Lenovo, blacklist it via
```
echo "blacklist acer_wmi" | sudo tee -a /etc/modprobe.d/blacklist.conf
```
Reboot and:
```
sudo rfkill unblock all
```
