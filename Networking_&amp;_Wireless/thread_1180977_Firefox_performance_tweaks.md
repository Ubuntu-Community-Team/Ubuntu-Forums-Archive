### Post by Crafty Kisses on 2009-06-08
Some things you can do is make sure you have any desktop effects turned off. You have the latest version of Flash and all the plugins of course. If you don't have all the plugins/codecs and you're on a 32-bit machine you might want to install this package:
```
sudo apt-get install ubuntu-restrcited-extras
```Now you might also be interested in a lighter browser, now when I say lighter I'm meaning less resource hungry. You may want to look into Swiftfox or Opera. In my experience Opera runs a bit faster than Firefox. Swiftfox is just an optimized version of Firefox, and it works great.
