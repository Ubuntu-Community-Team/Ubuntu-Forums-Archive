### Post by ian-weisser on 2015-12-02
If you want to merely *administer* the system, good old ssh is a fine, safe, unobtrusive way to go.
If you want a *remote desktop session*, which you really don't need for mere admin stuff, many fine options are available, including RDP and VNC.
Remember to use a secure method!
