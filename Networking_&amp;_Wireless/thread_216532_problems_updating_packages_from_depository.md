### Post by triviaseeker on 2006-07-16
Fixed the problem by finding the primary and secondary DNS that modem router was using and then setting that DNS on the DNS tab under network.

I came to this solution by scouring the forum and seeing what others had done.  Setting the DNS to the gateway did not work and removing the line from etc/resolv.? (can't think of its proper name at present) did not work.  All ok now though!!
