### Post by tgalati4 on 2008-02-17
Please elaborate "connects to the Desktop via USB", I am not aware of a 2-wire DSL modem that uses USB to communicate with Desktops.  Do you have 4 ethernet ports in the back of the DSL modem?  If you only have one ethernet port in the back of your modem, then you will need to hook up your desktop using an ethernet cable.  If other desktops need to share the connection then you will need a 4-port hub.

Using one of the wireless laptops, log into the modem using Firefox with the following address:

[http://192.168.1.1](http://192.168.1.1)

or 

[http://192.168.0.1](http://192.168.0.1)

You should get a login page.  Use the username and password printed in the instruction booklet (typically *admin* and *password*.  Go to the tab that says show connected devices.  You should see a list of IP values and MAC addresses of all the machines on your network.
