### Post by puppywhacker on 2009-11-13
IP Forwarding is when a device is not an end-point but routes to another device. You would need IP forwarding on all systems that serve as a gateway (for the route).

by default desktops have ip-forwarding disabled, to prevent it taking part in routing incoming ip packets with a destination which is it's own.
