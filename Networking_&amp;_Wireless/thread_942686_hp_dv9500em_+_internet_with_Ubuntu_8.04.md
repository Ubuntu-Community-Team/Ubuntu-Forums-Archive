People have wi fi problems with this laptop let me show you how to get it working...

Ok first download this!

[Click Hear]("http://ubuntuforums.org/attachment.php?attachmentid=70815&d=1211259015")

save the zip to somewhere in the system then right click and select Extract Hear you will get a file that ends in .ini

now go to Applications then click on Add/Remove then on the show drop down list then select All Available Applications then on the search next to it type "NDISWRAPPER" without the quotes then click the box you see you might get a message just say yes or OK or whatever. Then restart and go to system administration then click on Windows Wireless Drivers then click on install new driver click (none) then browse for that .ini file on select it and open it then click install. TA DA your wireless is working if it says "bcmwl5" and says underneath "Hardware Present: Yes". If not A) Do you even have this model most of the HP laptops look the same! B) Try Restarting or Removing Restarting and reinstalling the driver. C) Uncheck Roaming Mode.
