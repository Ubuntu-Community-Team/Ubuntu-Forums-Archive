### Post by stefangr1 on 2008-04-02
Yes, if you have enough network cards you can. You just connect one card to the internet, and the others to the machines at home. Then you have to set up a dhcp server, an optional firewall, etc.

EDIT: If you're only going to use it as a router, you are better off with a hardware router, since that can easily save you 100 euro or more annually. If you'd like to have your own server to play with, however, it's a different story.
