### Post by SoftwareExplorer on 2009-10-22
If both of the computers are connected to the router, all you have to do to share files on the ubuntu computer is right click on the folder you want to share and go to sharing options. If you have never used network sharing, the computer will want to install it. After that, check share this folder and it is up to you wheather you want guest access or want to allow people to write to it. Then click create share.
