Wireless *can* be tricky - especially on laptops.  I got a bit confused with the transition to network cables. The new Network Manager seems better than previous versions, but "Guaranteed to work without a hitch" doesn't come printed on the CD sleeve.  Frequently, Ubuntu is set up to dual-boot with Ubuntu.  If something doesn't work on Ubuntu, you can (usually) reboot into Windows.
