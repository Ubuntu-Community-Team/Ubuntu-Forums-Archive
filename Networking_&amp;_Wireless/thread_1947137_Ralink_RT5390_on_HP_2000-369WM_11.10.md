### Post by wildmanne39 on 2012-03-26
Hi, welcome to the forum! Please open a terminal ctrl+alt+t copy and paste the following commands into it one line at a time:
```
sudo apt-get install --reinstall linux-headers-$(uname -r) build-essential dkms patch fakeroot unzip
[wget http://media.cdn.ubuntu-de.org/forum/attachments/35/07/3208747-angepasster-2011_0406_RT5390_RT5392_Linux_STA_V2.5.0.3_DPO.zip]("wget http://media.cdn.ubuntu-de.org/forum/attachments/35/07/3208747-angepasster-2011_0406_RT5390_RT5392_Linux_STA_V2.5.0.3_DPO.zip")
unzip angepasster-2011_0406_RT5390_RT5392_Linux_STA_V2.5.0.3_DPO.zip
cd angepasster-2011_0406_RT5390_RT5392_Linux_STA_V2.5.0.3_DPO/
sudo make
sudo make install
sudo modprobe -v rt5390sta
sudo depmod -a
sudo update-initramfs -u 
```
then unplug your wired connection and reboot.
