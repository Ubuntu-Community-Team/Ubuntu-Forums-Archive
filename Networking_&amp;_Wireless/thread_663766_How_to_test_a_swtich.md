### Post by Fire_Chief on 2008-01-10
There's not really a script to do that. You will probably just have to plug in two systems and test file transfer or pings (file transfer would be a more reliable measure). Then leave the first system in it's port and move the other system to the next open port and test again. Sorry that it is so tedious but that's the only way to be thorough in testing the switch.
