### Post by bmartin on 2007-08-29
You're probably using 8139too kernel module (driver) already. Try **lsmod | grep 8139** to see if the kernel module is already loaded. Any lines that follow the command indicate that the kernel module is already loaded.

You can insert the 8139too kernel module using **sudo modprobe 8139too**. I have both 8139cp and 8139too loaded right now.
