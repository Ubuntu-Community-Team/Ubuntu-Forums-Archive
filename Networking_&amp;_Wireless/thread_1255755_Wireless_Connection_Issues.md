I have an Acer Aspire 5100. It has a AR5100EG wireless network card. I have tried for months to get a stable version of Linux on my computer. This would include repartitioning my computer several times. Losing my O/S twice. 

I have learned a lot about Suse 11.1, PCLinux, Several versions of Ubuntu. I thought the Wubi Ubuntu was the best.

But you know what...My computer will never work with Linux because it can't. It can for a while, but it is temporary. 

So before you put yourself and your machine through hours, perhaps days and weeks ( Like me ) I suggest you find out if your wireless network card is compatible with Linux before beginning an installation.
