ahhh, you're using resolvconf....
```

echo "nameserver 69.1.30.43" | sudo resolvconf -a eth0

```
if you have 2 servers to add - create a file with each server on a seperate line:
```

nameserver 69.1.30.43
nameserver 69.1.30.42

```
save it as something like dns.txt - then do:
sudo resolvconf -a eth0 < dns.txt
[/code]

btw... you might need to change eth0 if your interface is called something else. 
should do it for you...
