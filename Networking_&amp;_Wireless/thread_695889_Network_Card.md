### Post by Iowan on 2008-02-13
Hello,
Unless it's really strange (or new), Ubuntu should pick up the NIC automatically. There are several good tutorials on setting up a network - far better than what I can provide.  In brief, under System>Administration>Networking is a Wired Connection option. OOPS!!! that's on a Gnome desktop... To connect between two computers, you'll need either a hub, switch, or crossover cable.  You'll need to see what network address the Windows box is using - if it hasn't been set up for networking yet, it, too, will need an IP address.  Both addresses will need to be in the same subnet.
