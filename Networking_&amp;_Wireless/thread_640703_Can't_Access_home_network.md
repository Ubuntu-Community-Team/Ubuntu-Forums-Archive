### Post by spd106 on 2007-12-15
If you are using WEP make sure that you have selected the correct password type. This is usually in hex rather than ascii. There should be a drop down box to select the type from.

Some cards have problems with connecting to particular encryption types, so would be useful if you could let us know what kind of chipset it has. This wiki page will help you [https://help.ubuntu.com/community/WifiDocs/WirelessCardsSupported](https://help.ubuntu.com/community/WifiDocs/WirelessCardsSupported)

The chipset will also be named in the Hardware Information utility under the System menu.
