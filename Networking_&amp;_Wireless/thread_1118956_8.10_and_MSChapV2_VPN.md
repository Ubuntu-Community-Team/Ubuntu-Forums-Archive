### Post by steeleyuk on 2009-04-07
You can access the configuration settings through gconf-editor > system > networking > connections. 

You'll have to be careful with NM, I've found that settings don't appear to be set through the GUI but in reality are. For example, the VPN I use needs 256-bit encryption, and the option is correctly set, yet the GUI doesn't show it as set.
