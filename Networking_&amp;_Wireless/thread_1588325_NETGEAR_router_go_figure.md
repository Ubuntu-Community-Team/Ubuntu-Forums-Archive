As I wrote earlier the default username is 'admin' and the default password is 'password'. If that doesn't work you should do a complete reset of the router. There is a little recessed button on the back of the router that you will have to press with something like a paperclip. I'm not sure what the procedure is for net gear, but ot should be something like unplug it, press the button and hold it, plug it in while holding it and keep holding it for thirty seconds. Then let go and log in with the above username and password. Be sure to set a new password and enable WPA.
