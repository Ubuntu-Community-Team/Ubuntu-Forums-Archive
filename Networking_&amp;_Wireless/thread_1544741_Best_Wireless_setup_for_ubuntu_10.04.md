### Post by arewethereyet on 2010-08-02
I recently bought the D-Link DIR-655 and DWA-140 wireless setup just before converting completely to Linux from windows.

As a lot of you I'm sure know there are issues with the ralink rt2870 chipset. I did get this to work after a few days of trying various things, but I could not get Wireless N to work, it only operates in G mode. This seems to have something to do with needing to use AES for N-mode. My wireless setup will only work with Ubuntu if I have TKIP set. This of course requires my entire network to work at G speeds.

Consequently I am returning my D-Link setup tomorrow, and am looking for the community opinion on what Router/adapter setup is truly out-of-the-box compatible with Ubuntu 10.04.

This Router will need to be compatible with other windows based computers on the network running the DWA-140 USB adapters.

I would also like to keep it in the same price-range of $100 for the router, and $30 for a USB adapter(I have no PCI slots available).
