### Post by Beaker Boy on 2008-02-13
The problem with connecting to most Liveboxes is that the security on the box is set to dual WEP/WPA.  (Weird I know, but true!)  The way around this is to select WPA only on the Livebox 'Home page'.  This is really clearly explained on the links below.

This first one is great in that all you do is connect an Ethernet cable (supplied with the Livebox) between your computer and the Livebox, log on to the homepage as described, make the change and log off.  Don't forget to 'pair' the Livebox again on completion by pressing Button 1 on the Livebox.

[http://zxon.notnet.co.uk/orange/manual_setup.html](http://zxon.notnet.co.uk/orange/manual_setup.html)

Changing the security to WPA is dealt with more generally in this link.

http://help.orange.co.uk/documentDisplay.do?resultType=5002&docProp=$solution_id&groupId=1&directSolutionLink=3&gotoLink=0&page=&clusterName=DefaultCluster&docType=1006&docPropValue=kb12331

Ultimately, you are simply trying to change the security to WPA.  Don't worry that you only have a WEP code, it'll still work.

Best of luck

Beaker Boy
