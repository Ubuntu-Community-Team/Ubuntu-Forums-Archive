### Post by Autodave on 2018-01-11
Desktop or laptop?  It is always a possibility that your wifi card died, especially if a laptop.

If a laptop, this is the first thing that I always try (and it fixes a lot more of them than you would think):

   1. Power down and disconnect power cord.
2. Unplug all peripherals.
3. Remove battery.
4. Wait 10 minutes.
5. Reinstall battery.
6. Connect power cord.
7. Boot computer and test to see if condition is gone.
8. If good so far, now you can reconnect peripherals.
