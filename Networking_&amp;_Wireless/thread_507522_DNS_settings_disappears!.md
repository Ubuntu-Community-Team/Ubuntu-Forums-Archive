### Post by djdicbob on 2007-07-23
I set my dns myself. 
As root, sudo, of gksudo....edit the file
```
/etc/resolv.conf
```
add the ip address of dns servers, I am not sure if their is a limit.  But, I have 10 or 12.  All you need is 1.  That being said, add this to resolv.conf.  If there is already an entry, leave it and add the static  dns address.
```
nameserver *ip.add.re.ss*
```
I change the permissons on resolv.conf so the dhclient can't change the address.
