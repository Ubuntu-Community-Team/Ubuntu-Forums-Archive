### Post by DarkStarDeity on 2006-05-20
I recently installed Ubuntu Breezy and I was using a D-Link G520+ card (previously used in a windoze box) which was detected and working out of the box ... sort of. The connection would sometimes drop and the only way to get it back up again was to reboot the system. So I replaced it with a Netgear WG311T which is not only stable but faster :)

There is a list of supported hardware devices [here]("https://wiki.ubuntu.com/HardwareSupportComponentsWirelessNetworkCards"). When I was looking for a card to replace the D-Link, I was advised to buy one  supported by Madwifi listed [here]("http://madwifi.org/wiki/Compatibility") as that gives you native linux drivers without having to faff around with ndiswrappers or reverse engineered drivers.
