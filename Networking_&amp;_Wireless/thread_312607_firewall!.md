### Post by lloyd_b on 2006-12-05
One obvious question: How are you connecting to the internet?  Many DSL modem/routers and cable modem/routers have an inbuilt firewall/NAT.  If you have one of these, you'll need to forward the gnutella port that you're using from the router to the computer in question.

The specific method for doing this varies with the particular router - as a rule, try "http://" and the IP address of the router from a web browser, and you'll get access to the admin controls on the router...
