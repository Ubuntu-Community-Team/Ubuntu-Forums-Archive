### Post by mahapatrasworoop on 2012-08-19
Hello all :)

I installed Ubuntu 12.04 on mu Dell Inspiron 1464. This was a fresh install. On previous installs I used the STA driver. This time out i thought of trying out the fw-cutter drivers. 
I have successfully installed the drivers using terminal
```
sudo apt-get install b43-fwcutter 
sudo apt-get install firmware-b43-lpphy-installer
```

Now the problem is that while using the wireless it gets disconnected and i am unable to connect until a reboot or a hardware switch reset of wireless. And it keeps asking me for password to my wifi. 

Is there a method to solve this problem or to uninstall the drivers and install STA

Any help will be appreciated.

PS : Linux Newbie here.

---

### Post by mahapatrasworoop on 2012-08-19
Solved.
```

sudo apt-get remove b43-fwcutter 
sudo apt-get remove firmware-b43-lpphy-installer

```

and activated STA driver from additional drivers
