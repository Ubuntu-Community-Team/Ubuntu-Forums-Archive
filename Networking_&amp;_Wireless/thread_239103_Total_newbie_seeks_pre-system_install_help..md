### Post by spd106 on 2006-08-19
It is definately worth investing in an ethernet connected router. If you can afford the upgrade then most ISPs will provide a free router (often wireless these days). 

If you don't want to be locked into another 12 month contract paying an extra £10-20/month then it is probably cheaper to buy a simple ethernet/ADSL router from PC World or similar. Shouldn't cost much more than £35-40, probably less if you look around. Wireless ones are rather more expensive, but worth it if you need to share the connection, don't bother with pre-n yet though.

You just need to connect the PC to the router via ethernet cable and the router to the microfilter. The router will provide DHCP, DNS and NAT services. The PPoA connection will be done on the router through a web interface. This is all explained in the manual. 

You could always try out this thread to keep you going [http://www.ubuntuforums.org/showthread.php?t=190359](http://www.ubuntuforums.org/showthread.php?t=190359)
