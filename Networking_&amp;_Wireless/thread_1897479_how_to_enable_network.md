### Post by stephenbbb on 2011-12-19
I have a desktop with ubuntu 10.04 with a USB printer and laptop with ubuntu 11.10. laptop has wireless, desktop is fed with a cable from a 2wire 2701HG-B modem/router from Bell. desktop also has win XP and laptop has Win 7.
neither machine sees the other. I used to see a windows network from the laptop, access files and even print. it is gone now after installing ubuntu on the desktop and setting up the modem to allow ssh. I think I changed sth I should not have in the modem.
the modem (192.169.2.1 in a browser) shows a home network with both the laptop and desktop and shows the desktop supports ssh. I cannot find any place where the network is shut down. 
the XP is setup to share some folders and the printer, but sees nothing except itself. same for Win 7 on the laptop, there is a home network, but sees only itself. both ubuntus can see the windows files on their own box.

### Post by stephenbbb on 2011-12-20
The ridiculous can sometimes work in your favor !!

I spent half an hour yesterday with Bell support only to learn they offer no support for file sharing or printing or any kind of home networking.
I had left the modem/router setup page open in a browser window presuming they would want to connect to my desktop. at the end of it my network was working, both in Windows and in Ubuntu !! without making any changes ! Hallelujah :-)
