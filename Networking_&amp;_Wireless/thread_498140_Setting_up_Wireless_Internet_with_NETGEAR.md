### Post by lisati on 2007-07-11
The setup disk that came with the Netgear router I use was designed for Windows, so the disk won't work too well with Linux without help from a Windows Emulator. I'd already run the disk for my router on a Windows machine before hooking up to it with an Ubuntu installation, no special changes at the router end were needed.

Netgear has some information available on the internet. You might want to visit [http://kbserver.netgear.com/](http://kbserver.netgear.com/) to see if there's some helpful information.
