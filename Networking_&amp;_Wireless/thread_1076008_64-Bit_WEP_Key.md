### Post by elishock2420 on 2009-02-21
Per netgear's website - 

"For wireless devices to communicate, all of them must use the same WEP setting. (40-bit and 64-bit WEP encryption are the same thing — 40-bit devices can communicate with 64-bit devices.)"

Maybe this url will help...

[http://kb.netgear.com/app/answers/detail/a_id/1141](http://kb.netgear.com/app/answers/detail/a_id/1141)

So I would say that you should select 40-bit wep key and enter in your 64-bit wep key from the wireless router.  Good luck and let us know if it works!
