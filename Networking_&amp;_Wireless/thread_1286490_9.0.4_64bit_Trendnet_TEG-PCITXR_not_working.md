### Post by nixscripter on 2009-10-11
A few things could be wrong.

First, you might need the driver. You should see if there is a kernel module loaded for it with: ```
lshw -C network
``` If there isn't, you can download it here: [http://www.trendnet.com/asp/download_manager/list_subcategory.asp?SUBTYPE_ID=208](http://www.trendnet.com/asp/download_manager/list_subcategory.asp?SUBTYPE_ID=208)

Second, when you connect, see if you can run **mii-tool -v** or **ethtool** on the network interface of the card (as root). This will indicate if there is a physical link (or if it's a networking problem).
