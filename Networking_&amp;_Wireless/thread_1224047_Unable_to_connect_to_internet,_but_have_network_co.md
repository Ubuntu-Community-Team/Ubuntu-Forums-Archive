### Post by x33a on 2009-07-27
first backup the resolv.conf file
```
sudo cp /etc/resolv.conf /etc/resolv.conf.bak
```then to edit resolv.conf:
```
gksudo gedit /etc/resolv.conf
```delete everything in the file and add your favourite dns servers or opendns servers to the file:

for opendns
```
nameserver 208.67.222.222
nameserver 208.67.220.220
```

save the file.
