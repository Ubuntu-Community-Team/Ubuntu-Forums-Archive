### Post by gordintoronto on 2011-12-03
When I clicked on the link, I was asked, in German, for a username and password.

You don't "open" a .sh file, you run it in a terminal window. CD to the folder where it is, then I think the syntax is ./vpn-setup.sh (could be wrong...)

Ubuntu has a VPN client already installed, just waiting for you to configure it. Click on the network manager icon and select VPN Connections.
