### Post by MyR on 2007-08-07
first: make sure your wireless card is turned on. Fn + F2
the light doesn't work so give it a little while to see if network manager finds your network AP.
(you can get the light to work somewhat but it's a little tricky and probably not worth the effort.)

if this resolves the issue:
i believe there may be settings in the bios regarding when the wireless card turns on and off if the problem recurs.

still not working?
i assume you have the intel pro/wireless 2200 chipset.
you can find drivers here:
[http://ipw2200.sourceforge.net/](http://ipw2200.sourceforge.net/)

if *that* doesnt work perhaps networkmanager need to be reinstalled.

hope this helps!
peace
