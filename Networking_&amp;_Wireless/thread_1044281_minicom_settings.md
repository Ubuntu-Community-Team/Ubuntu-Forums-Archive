### Post by Kebabman on 2009-01-19
To configure minicom you just need to type :

CTRL-A

followed by 

Z

Then to get to the configuration type

O

Then go to serial port setup.

There is also a shortcut of just typing :

CTRL-A

followed by

O

Hope this helps
