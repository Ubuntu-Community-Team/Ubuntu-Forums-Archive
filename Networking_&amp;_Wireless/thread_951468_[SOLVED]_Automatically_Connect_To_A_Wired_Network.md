But...

How do I get Ubuntu to automatically connect to my wired network?  Each time I boot-up to the desktop I have to left click the Network Manger icon and select wired network, then it'll connect.

---

### Post by HockeyPlayer on 2008-10-23
Ok.  I solved this by going into the Manual Config for the Network settings, selecting the Wired Connection, and within the properties I turned off 'Enable roaming mode' and selected 'Automatic Configuration (DHCP)' from the Configuration drop down menu.

Seems to work fine now.
