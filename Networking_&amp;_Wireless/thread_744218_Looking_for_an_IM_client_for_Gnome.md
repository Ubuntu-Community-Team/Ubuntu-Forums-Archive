### Post by ubuntu27 on 2008-04-03
The only IM that **I** know that can connect to the Yahoo! and MSN protocol, and has webcam support is [kopete]("http://kopete.kde.org/").

Kopete is designed for KDE, but can run in GNOME just fine. Try it.

```
sudo apt-get install kopete
```
