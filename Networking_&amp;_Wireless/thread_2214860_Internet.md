### Post by sudodus on 2014-04-03
Ubuntu desktop 10.04 has passed end of life. Ubuntu Server still has support and receives security updates. Many program packages and libraries are common, but the desktop and application packages of Ubuntu desktop receive no security updates. So I recommend that you use a current version, 12.04 LTS or 13.10. In a couple of weeks a new long time support version, 14.04 LTS will be released.

If you have old hardware, please consider a flavour of Ubuntu with a lighter desktop. Xubuntu (12.04 LTS and 13.10) is medium light and Lubuntu is ultra light (only version 13.10).
