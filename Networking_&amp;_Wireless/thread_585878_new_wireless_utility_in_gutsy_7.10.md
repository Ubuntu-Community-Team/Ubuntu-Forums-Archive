### Post by spd106 on 2007-10-23
The applet you have seen is for Network Manager and is only available when your interface in set to roaming mode. To switch to roaming mode you have to open System -> Administration -> Network and click on the interface properties.

At the moment Network Manager does not support static address settings, but it is being worked on for the next release.
