### Post by clowncoder on 2009-08-01
Hello, this is not a question on networking but valuable information: if you wish to have a whole virtualized network based on User-Mode-Linux machines and QEMU-KVM machines, have a look at [http://clownix.net]("http://clownix.net/") there is a plug and play drag-and-drop network creation tool.

Create machines, links between machines and visualize your network's activity only through mouse-clicks. The name for the tool is cloonix, it is GPL-V3, source included in tar.gz, last version is 2.1.
It works on ubuntu!
