I run Windows inside of VMware and recently decided to try the Windows version of Skype instead of the Ubuntu version because the ubuntu version doesn't have the Skype Prime Beta feature implimented yet.

Upon testing out skype in windows (inside VMware, running on Ubuntu), my friend told me my voice sounded like a robot.

Now, in the past, when I'm uploading a file via FTP or HTTP from Windows to a server, I notice that in my networking activity bar graph (using system monitor) is very jumpy.  There are gaps equidistant of each other for long time spans...as though VMware is only allowing windows periodic access to my Wireless network adapter, causing a lot of data transmission to be screwed up.

After logging off of the Skype in Windows, and using the native Ubuntu version, we tested it again, and the sound quality was top notch.
