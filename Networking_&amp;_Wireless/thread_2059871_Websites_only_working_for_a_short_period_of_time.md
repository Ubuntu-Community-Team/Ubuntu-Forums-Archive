### Post by spjackson on 2012-09-19
Welcome to the forums.

I don't know what is causing this or how to fix it, but if I had the problem I would definitely try another browser first. If you are using Firefox then try Chromium, and vice versa. If it works fine in the other browser, then I'd consider removing the entire directory trees for the browser configuration either ~/.mozilla/firefox or ~/.config/chromium ~/.cache/chromium and start from scratch. N.B. this would throw away any bookmarks, saved passwords etc.

If switching browser results in the same symptoms, then that's a different ballgame.
