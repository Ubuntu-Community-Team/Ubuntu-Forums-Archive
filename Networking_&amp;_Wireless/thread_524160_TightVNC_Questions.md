### Post by ADT on 2007-08-13
You need to setup a desktop environment with tightvnc. That means that when you connect to a tightvnc server you are viewing it's console because no desktop environment has been loaded.

This tutorial explains how to setup tightvnc so it displays a desktop environment when you log in.
