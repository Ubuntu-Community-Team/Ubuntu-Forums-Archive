### Post by benander on 2008-09-22
Hey, 

Just set up something like this I think.  If you're using a network printer it should have an IP. If you know it great, otherwise check your router. On your linux machine go to "System -> Administration -> Printers". Add a new router and then select host router.  Enter the IP then when it goes to the next menu select the make and model. For me it was an HP laserjet 4000, and it installed fine, hopefully it will for you too. 

Not sure if this helps or if I'm way off, I'm somewhat new to linux. 

Good Luck.
