### Post by Jim_in_Omaha on 2008-08-31
I have the same problem on my desktop. It took me a while to figure it out. I was so disappointed in Ubuntu because of it. But now I know what it is for me. It is the wireless PCI card. There is information out there about it but for me I have not been able to fix the wireless PCI card issue. 

The way to test it is turn off the wireless, and CAT5 cable right into the router. If it is like mine it will run for days without a hitch. 

We live out in the country with DSL. I was running with my cable to the router and lightning struck the phone system somewhere, knocked out the router, which in turn killed my motherboard.

So when my new motherboard comes in, I will probably be getting an Edimax EW-7318USG USB wireless adapter and this company supports Linux on several of their products. Of all things I found some info on Youtube. Search for "ubuntu wireless" there are some helpful vid's out there and one reviewing this adapter.
