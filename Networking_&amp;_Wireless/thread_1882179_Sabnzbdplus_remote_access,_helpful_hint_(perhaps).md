Hi,

I've just had a heckuva time re-enabling remote access to my Sabnzbdplus server, via my Android devices (Sabcontrol, etc.). I found myself checking and rechecking, firewall, port forwards etc. when, as it turns out, I had, at some point, configured the server setup option "Host SABnzbd should listen on" to "localhost" when, in fact, it should be set to "0.0.0.0" to accept remote connections. which is, of course, clearly stated in the setup and I had mistakenly changed it in my frenzy to correct problems.

Silly mistake, but may be of some use to someone out there, eventually. Now, back to media consumption! ;)
