### Post by kebes on 2007-01-10
Does it happen for all sites or only some sites? If some websites load fast but others do not, you are probably experiencing a problem related to IPv6 compatibility. In that case you can just turn off all the IPv6 support and it should become fast again.

EDIT: see, for example, [this thread ]("http://ubuntuforums.org/showthread.php?t=87798")for info on how to disable IPv6.
