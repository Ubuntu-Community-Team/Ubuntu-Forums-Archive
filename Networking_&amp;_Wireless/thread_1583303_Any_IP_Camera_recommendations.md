### Post by Cameraguy on 2011-01-22
> **SarAmyKay said:**
> I need to have it work with Linux and keep the cost down.
 
 
No problem...I have my self-networking cameras running fine on Ubuntu...on a netbook as well as a desktop.   Here are some details.  The camera set up is automatic...no OS is required...with CAT5  cable.  If you want to use wireless models..they have a High Definition camera that has a WEP button that synchronizes security words with the WEP wirless router.  No PC is required.  Now what if you want to see the video?  Most software is for Windows but there are some tricks around Windows.  Here are five tricks:
 
First---these self-networking cameras also have a webserver built inside...NO PC is required for set up.  They have 5-6 different client software  see below tricks.
 
1) Use the iPhone 3 or later, iPad, or iPod Touch...they have free software that can 
render and record video and sound LIVE from the camera. Also Pan-Tilt controls of P-T camera.
 
2) Each camera in addition to 10 separate video streams---sockets---there is an RTSP stream that is EASY to render and record video.  Email me and I will send you the exact url description you need to see the video with any browser---including seamonkey-Linux.
 
3) Some of the self-networking ip cameras (ipcams) have a MICRO SD flash card in them for in-camera recording.  The Flash card videos use the .crf video file format.  The recorder and player are FREE for Ubuntu and other Linux distros.  Visit "VLCLan" and download VLC player FREE for the OS you are using....Debian and most Linux Distros including Ubuntu are available.
 
4) These little self-networking ip cams also run on Tim Hillones Free software at h.264.net   
 
5) The self-networking ip cameras  also have their own NAS with embedded Micro Linux instruction set.  All you need is any Linux distro to access the NAS.  As soon as the crf video file is written...you can see it on the NAS using Ubuntu.  It works as well as the Micro SD cards on those type cameras.
 
Send me an email and I can send you step by step instructions...Its very easy.  After you set it up the first time.
 
The only feature you don't get is the mini program embedded in the Internet Explorer browser Active X controls---which download to Windows OS only.
