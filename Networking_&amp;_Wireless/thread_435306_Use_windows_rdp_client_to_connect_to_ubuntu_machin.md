### Post by thelinuxguy on 2007-05-07
> **johnnyphive said:**
> Is there anything out there that will allow an rdp connection from and xp machine using port 3389?

I doubt if Ubuntu allows logins via RDP since RDP is native to Windows. It has RDP clients though. The native protocol for X is XDMCP. From Windows, you can use XMing to log into a Ubuntu system with XDMCP enabled. My post about XMing at [http://ubuntuforums.org/showpost.php?p=2302970&postcount=36](http://ubuntuforums.org/showpost.php?p=2302970&postcount=36) may help.
