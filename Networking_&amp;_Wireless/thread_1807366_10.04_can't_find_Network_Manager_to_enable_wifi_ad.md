Hi, network manager is installed by default.  The top right corner of the screen there is a little icon and if you click on it you should see your network connections, make sure wireless in enabled.

If you still have problems run these commands in a terminal.
```
lspci -nn | grep 0280
```
```
rfkill list all

```
lsmod
```
```
iwconfig
