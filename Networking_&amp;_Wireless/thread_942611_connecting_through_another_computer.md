### Post by cariboo on 2008-10-09
There a couple of ways to accomplish what you want, if your laptop has a wired network  port you can connect your computer to the laptop via an ethernet cable and setup internet connection sharing, because you want to do connection sharing from a windows based computer there are so many howtos available including windows help that it doesn't need to be covered here. Here is a link:

[http://support.microsoft.com/kb/306126](http://support.microsoft.com/kb/306126)

The other way to do it is to buy a well supported wireless adapter and connect via wireless.
