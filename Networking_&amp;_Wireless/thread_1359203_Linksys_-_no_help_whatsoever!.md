> **geomcewan said:**
> If anyone's interested in setting up this router, boot into windows, run the setup program and when you go into Ubuntu it picks it up straight away.

And if the anyone doesn't have Windows? Or is a Mac user? Or you want to change settings using Ubuntu?

I have a Linksys router and I have never used the Windows setup wizard.

If anyone is interested in setting up **any** router, simply use a web-browser to log into the web configuration interface. Then you don't have to worry about which OS you are using. The router manual will tell you what network address you need for logging in (192.168.1.1 or 192.168.0.1 or 10.0.0.1 or similar depending on make/model) and the admin username and password.
