### Post by Stryder64 on 2008-02-17
I finally figured it out after calling my ISP provider.  If anyone needs help with setting up Comcast...get the Automatic DHCP set up with the ISP address and the default gateway, and then make sure you are in roaming mode.  Also, make sure you reset your cable modem.

Hope this helps!
