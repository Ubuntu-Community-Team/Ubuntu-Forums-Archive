### Post by wildmanne39 on 2013-10-30
Please do:
```
sudo apt-get install --reinstall linux-firmware-nonfree
```
watch for errors then reboot.
Also it says you have a hard block which usually means the wireless switch is off, you can try to turn it on with a button or key combination like fn +f5.
Thanks
