### Post by mizwete on 2007-12-01
You must edit your wvdial.conf file !

copy this and paste it into wvsial.conf

you MUST change this line:
Init5 =AT+CGDCONT=1,"IP","internet.vodafone.pt";
and replace "IP","internet.vodafone.pt";
by the IP address of your IPS


then in order to make a connection, open a terminal and type 

wvdial hsdpa


[Dialer hsdpa]
Phone = *99***1#
Username = vodafone
Password = vodafone
Stupid Mode = 1
Dial Command = ATDT
Modem = /dev/ttyUSB0
Baud = 460800
Init2 = ATZ
Init3 = ATE0V1&D2&C1S0=0+IFC=2,2
ISDN = 0
Modem Type = Analog Modem
Init5 =AT+CGDCONT=1,"IP","internet.vodafone.pt";
