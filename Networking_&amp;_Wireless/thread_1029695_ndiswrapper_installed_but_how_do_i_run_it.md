### Post by Tim Sharitt on 2009-01-03
Several apps don't have a gui and don't have a menu entry. Those apps are used from the command line. To figure out how to use them open a terminal and type 
```
man <application>
```
In the case of 7zip, if you have it installed, you should be able to work with 7zip files using the archive manager.
