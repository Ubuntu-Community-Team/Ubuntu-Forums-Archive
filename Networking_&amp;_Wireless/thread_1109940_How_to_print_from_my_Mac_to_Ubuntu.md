### Post by sgosnell on 2009-03-29
Perhaps.  I know nothing about Macs, so I can't help you with anything there.  On the Ubuntu machine, you should have samba installed, and the printer marked as shared.  

Do you see the server on your Mac?  If so, can you see the printer?  If you can see the server but not the printer, that would seem to point to the printer not being shared from the server.  If you can't see the server from the Mac, the problem could be on either machine.  Make sure the server is configured properly, then you may have to go to a Mac forum to get it configured, but there may be some Mac experience by here soon.
