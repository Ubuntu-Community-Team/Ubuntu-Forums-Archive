hi guys,

i have just finished setting up a small office network.

LAYOUT: 
All computers (15 pc) are connected to a router via wireless connections. the router is connected to an internet service provider, thus all computers have internet.

PROBLEMS:
The idea is to convert one of the computers as the "main computer" so that the following can be achieved:

- file sharing is possible
- the other computers can access files on the main computer
- the main computer can monitor, restrict the other computers.

### Post by dandnsmith on 2011-05-28
With the setup you describe, you won't be able to restrict the other computers from the main computer, unless the router has the facilities built in, because you won't be 'in the loop' for their 'conversations'.

I'm not sure that you'll be able to monitor either, without some advanced hacking software.

To achieve your stated aim, you'd need to set up one computer as a router, and have all the conversations flow through it - conceptually, to be certain, connect the router PC to the internet modem/router via wires, and turn off wireless on that router, and make the PC router a wireless point for the rest.

Good luck

### Post by jaime256 on 2012-08-27
With this setup you can just add a nas in your network for file storage and use the nas for restrictions on files and who can access them since you don't have a server. This takes care of your files, restrictions and gives you the space in one place instead of different computers that you can connect to for sharing files. Just don't forget to add the antivirus from Microsoft to each windows pc if that's what you use to keep the workstations healthy in that respect. It's free. If you use Ubuntu, that may not be the case, but you can still add one in ubuntu. Now you can still keep that set up for that small amount of computers and call it a day. You'll keep it running and still be organized. So even if something happens to one or all your computers you still have files in the nas. A second nas for backup wouldn't hurt, but this is just a suggestion. I hope this helps.
