Hi All,

I did a fresh install of 10.4 on my HP DV7-1132nr Laptop last night.  I am not dual booting or anything.  Everything worked great after the install.  Wireless worked fine, ATI video drivers installed just fine...no problems at all until I closed the lid and put it in suspend.

I am able to hook up a ethernet cable and get access that way, but wireless is completely shot.  

Note even if I manually put the computer in suspend (without closing lid), it will still freeze and force me to shut off and manually power back on.

I don't know what else to do, I was hoping to ditch windows completely but now I'm not so sure!  Everything worked fine right after install so I don't know why putting it in suspend caused all these issues!

### Post by bford16 on 2010-10-25
Please try```
rfkill list
```  If you don't have rfkill installed, please install with ```
sudo apt-get install rfkill
```On my dv7-1232NR, I get this;```
0: hp-wifi: Wireless LAN
	Soft blocked: no
	Hard blocked: no
1: phy0: Wireless LAN
	Soft blocked: no
	Hard blocked: yes
```If you get the same result, try```
sudo modprobe -r -f hp_wmi
sudo modprobe -r -f ath5k
```Then reboot.  The wireless should come back enabled.
On my machine, the key combination to disable/enable wireless is```
Place finger on wireless button, and tap the Quickplay button
```Let me know if this works for you.
