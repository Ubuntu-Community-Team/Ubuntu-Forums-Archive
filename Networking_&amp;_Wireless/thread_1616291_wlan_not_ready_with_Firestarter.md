### Post by grahammechanical on 2010-11-08
It sounds like you want to set your computer as a wireless Access Point (AP). You may need to change the mode of the connection. I do not think you can do what you want if the connection is set to Infrastructure mode. Network manager does not give you the option to set the mode to Access Point Mode but you could try Ad-hoc mode.
