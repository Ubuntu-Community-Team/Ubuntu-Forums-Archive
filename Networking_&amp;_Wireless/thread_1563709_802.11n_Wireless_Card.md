### Post by MaindotC on 2010-08-31
Fantastic!  Yes those are the directions for compiling and installing the driver file.  If you look in the source code that you downloaded, there should be a file called Makefile (or something similar to that) and if you open that file with a text editor you will see the list of commands that are passed to the "make" command.  That's where "make" got all it's instructions.  "Make install" also followed similar instructions to place certain files in certain directories, and those are also defined in that source code you downloaded.  Just something I thought I'd add in case you were wondering how "make" and "make install" knows to do what it just did :D
