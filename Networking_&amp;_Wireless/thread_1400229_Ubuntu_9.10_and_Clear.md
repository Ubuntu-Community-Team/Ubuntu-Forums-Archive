### Post by p110011 on 2010-02-06
I have the home/mobile modems and I get good service (8M/!M) with the home modem. I have it connected to a linksys router and use wireless, pretty basic set-up.

As far as the mobile usb modem, it works with Windows and Mac machines, no drivers in Linux yet. An option for using Linux loaded on a laptop is to use a little usb router, like the clear spot. I think it runs around $150.00

Hope that helps.
