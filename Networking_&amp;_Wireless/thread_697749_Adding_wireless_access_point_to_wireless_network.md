### Post by lawentzel on 2008-02-15
Adding a second wireless router would potentially cause more problems then it would solve.  Now, getting a better wireless router with a stronger antenna and placing it in a good central location would.

As to adding an additional access point I talked to my network guru.  They wire directly to your router.  So you would have to run cable to it.  After that, you would need to have what ever port it is on, on the router configured to issue IPs... ah la DHCP.

Hope this helps some.
