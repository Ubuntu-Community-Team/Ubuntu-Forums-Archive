### Post by spd106 on 2007-11-09
Make sure your wired interface is set to roaming mode in System -> Administration -> Network, then there should be a wired option at the top of the drop down list in nm-applet.

Also if you right-click nm-applet and untick enable wireless, it should switch straight to the wired connection.
