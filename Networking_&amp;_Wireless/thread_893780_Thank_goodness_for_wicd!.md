wow well let me tell you my adventure with linux wireless. Ok I started off with networkmanager automatically installed and the default driver for the rt2500usb. Well that only worked with wep so I installed ndiswrapper and installed the windows driver. Needless to say I had wpa working but every two minutes while I was playing OA I'd get a huge lag spike that lasted 2 seconds in roaming mode. switched to manual mode and it smothed out but for some reason I couldn't start my wireless automatically each time I rebooted...what a pain. I remember hearing about wicd and blam it works like a charm. smooth and consistent connection but for some reason it is not in ANY of the ubuntu repositories. Why is this? It is all under GPL doesn't break any patent laws either. Either way I urge any ubuntu or any linux user to try it if they have issues with networkmanager. Also you need to go to their site to find out how to download it:guitar: rock on
