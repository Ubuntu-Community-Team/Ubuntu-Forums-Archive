### Post by zekica on 2009-02-06
Nokia 3110c is a phone.

This functionality should be built in NetworkManager, but for some reason not all phones are detected.

If you cannot setup a connections using network manager (applet in the tray, right click -> edit connections) then you can try these instructions:
[https://wiki.ubuntu.com/NokiasN70Modem](https://wiki.ubuntu.com/NokiasN70Modem)
If you are connecting via Bluetooth, it is a little bit more complicated:
[https://help.ubuntu.com/community/BluetoothDialup](https://help.ubuntu.com/community/BluetoothDialup)

Hope it works.
