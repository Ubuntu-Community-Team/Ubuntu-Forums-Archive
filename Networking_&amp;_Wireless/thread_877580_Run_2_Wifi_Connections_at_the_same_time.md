### Post by mb_webguy on 2008-08-01
What you're talking about is called network bridging, and while it's fairly common using a wired and a wireless connection, or multiple wired connections, I've never tried it using multiple wireless connections.  There shouldn't technically be any reason you shouldn't be able to, since you have two separate wireless cards.
