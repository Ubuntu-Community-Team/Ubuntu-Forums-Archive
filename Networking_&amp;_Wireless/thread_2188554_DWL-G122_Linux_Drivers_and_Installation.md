### Post by varunendra on 2013-11-21
Like praseodym recommended earlier, we strongly recommend to upgrade to latest LTS, that is, Ubuntu 12.04.3

Download its ISO via a torrent to ensure the download's integrity as well as faster speed : [http://www.ubuntu.com/download/alternative-downloads](http://www.ubuntu.com/download/alternative-downloads)

Burn a CD from this ISO or create a live USB from it, test it in a live session and if satisfied, do a clean install from the same cd/usb.

Most probably your device would work out of box with the latest LTS. If not, troubleshooting it on a supported version would make much more sense.
