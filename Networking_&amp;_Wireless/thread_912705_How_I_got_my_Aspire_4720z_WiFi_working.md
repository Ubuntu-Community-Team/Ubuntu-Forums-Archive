### Post by mishathegoat on 2008-09-07
This is what I did to get my Acer Aspire 4720z internal wireless NIC to start working properly. I'm not sure how good of an idea this is.. but I tried all the posted methods with no success.

System > Administration > Software Sources > Updates

I checked "Pre-released Updates (hardy-proposed)" and performed another update which resulted in my internal wireless NIC working fine.

[EDIT] It works out of the box with Ubuntu Desktop 8.10
