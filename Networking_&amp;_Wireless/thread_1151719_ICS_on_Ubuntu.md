### Post by dream_coder on 2009-05-07
Best way will be to learn and use iptables, there are many guides around the internet just google, secondly you can use a program called firestarter that will do it for you! type "sudo apt-get install firestarter" it works pretty well but learning iptables is obviously an advantage!

I have done both and Firestarter is really easy to manage the firewall and set up ICS using a GUI. :)
