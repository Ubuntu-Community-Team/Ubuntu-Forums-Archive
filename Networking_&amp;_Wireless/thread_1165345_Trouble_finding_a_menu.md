### Post by stefangr1 on 2009-05-20
If you choose "Places" in the main panel and go to the "Network", it should find your windows network automatically. To search a shared printer in the network you go to "System" -> "Printing" -> and add a Windows printer with samba.

To add a shared folder on your ubuntu machine you should install some additionall software: Samba.
