### Post by lsp432 on 2009-09-10
In the deluge [FAQ]("http://http://dev.deluge-torrent.org/wiki/Faq#DoesDelugehaveanequivalenttoForceStart") it says if you set a torrent to not be auto managed it's essentially like force start; the torrent will start uploading and downloading regardless of queue position.

Custom grouping can be implemented through the "label" plugin, you can add "labels" (groups) in the left sidebar, and then add torrents to the various labels.
