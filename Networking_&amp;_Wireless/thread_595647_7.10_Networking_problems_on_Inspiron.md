### Post by spd106 on 2007-10-29
Your wired connection should work out of the box.

Try this
```
lsmod | grep b44
```
If you get nothing then enter this command.
```
sudo modprobe b44
```

For the wireless card you need to download the firmware as suggested in the Restricted Drivers Manager. You can download it via another PC from here [http://downloads.openwrt.org/sources/wl_apsta-3.130.20.0.o](http://downloads.openwrt.org/sources/wl_apsta-3.130.20.0.o), then just transfer it over or pull it from a Windows partition.
