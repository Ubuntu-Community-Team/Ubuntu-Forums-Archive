Welcome to the forums.

Usually when someone requests help with something like this, there are better solutions. Not always, but usually.

So, won't make a bunch of assumptions, since there isn't any data provided.  Data like:
* which OSes and versions are running on both systems AND all the clients?
* which subnet is each different server and client on?
* what sort of network connection does each server and client have?
* you did already try to do what you described and it didn't work?  Can you post the smb.conf and fstab files please? Use "code tags", so they line up correctly.
* did you consider going with the native NFS methods, which "just work" and perform faster for Unix-to-Unix storage?

We don't have any real idea about your level of expertise - all I see is 1 bean which often means someone is very, very, new to Linux. That isn't always true, so please help us out if you can.  Describing solutions to someone with 10+yrs of Unix is very different than trying to walk someone through with less than 1 yr.
