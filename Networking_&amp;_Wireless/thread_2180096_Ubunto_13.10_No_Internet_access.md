I've just spent half a day in the same situation. I'm replying only because I am so pissed off. Apparenty, probably within 48 hours Ubuntu released some screwy update for 13.10 that gets downloaded while you're installing it. There is only one fix for this: clean reinstall Ubuntu 13.10 and physically DISABLE (UNPLUG) the network while you do that. After you reboot you can enable the network, it's fine. I've just tested that myself - so far so good.
