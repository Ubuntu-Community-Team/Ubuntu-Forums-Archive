### Post by lsutiger on 2007-12-02
Yes. Yo don't go to network places. You set it up in the VirtualBox itself. 
Select your VB, then on the menu bar, select Machine-->Settings
In the window that pops up, Select "Shared Folders". Add a shared folder and it will show up in 'My Computer'.

This is all for a windows based VB.
