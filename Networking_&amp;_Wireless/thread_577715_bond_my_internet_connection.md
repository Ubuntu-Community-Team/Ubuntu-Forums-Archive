"bonding" is a feature for LAN and is not suitable for Internet connections. To work properly, the Ethernet LAN switch must also support it.

Here's a few words and some links about using two internet connections:
[http://ubuntuforums.org/showthread.php?t=555044&highlight=dual+WAN](http://ubuntuforums.org/showthread.php?t=555044&highlight=dual+WAN).

Other links:
[http://www.linuxforums.org/forum/linux-networking/33074-dual-wan-router.html](http://www.linuxforums.org/forum/linux-networking/33074-dual-wan-router.html)

It's not going to be a quick + easy setup, but it can be done. "dual WAN" is a good search keyword for Google and the other search engines
