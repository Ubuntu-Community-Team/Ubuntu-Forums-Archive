### Post by papibe on 2012-06-13
It looks like dnsmasq is not working properly. Check if disable it solve the problem.

Follow this [steps]("http://ubuntuforums.org/showpost.php?p=11923702&postcount=40"). After that, make sure the file /etc/resolv.conf contain a line like this:
```
nameserver 192.168.0.1
```
Hope it helps, and tell us how it goes.

Regards.
