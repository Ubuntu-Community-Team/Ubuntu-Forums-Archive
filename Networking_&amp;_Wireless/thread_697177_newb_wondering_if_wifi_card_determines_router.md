### Post by jrusso2 on 2008-02-14
Routers use TCP/IP so they should work with any brand of Network or Wireless card.  That being said YMMV as some people have reported problems when using certain combinations.  Some I have heard of is using an Intel wireless card with a router that uses atheros.  But I have this combination and it has not been a problem for me.
