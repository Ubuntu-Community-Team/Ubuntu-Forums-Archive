### Post by underquark on 2010-02-01
Most likely, but a bit more detail would help if it's more complex than the following:.

If it's just your standard wireless network setup then right-click on the little steppy icon in your top panel ("Wireless network connection").  Choose Edit Connections, Wireless tab, your network and Edit.  Assuming you have permission on the PC you are using then it ought to allow you to insert password etc.  Make sure you have the correct SSID, security settings etc.
