### Post by DumpsterCat on 2010-01-01
ok, i've done this only once, using the liveCD on the wifes laptop.  There's a "sockety" looking symbol, on the upper right, to the right of the speaker symbol, near the date.  You'll need to enable networking and/or edit connections.  Within those menus you'll provide network name, ssid, etc.  when you've provided that, you should get a connection.  You wont however, see all existing local networks, as you might see using xp.  There's another tool for that.  Dont recall what it is.
