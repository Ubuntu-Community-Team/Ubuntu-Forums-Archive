### Post by keithcausey on 2007-05-24
Hi Keith, this is Keith. I saw in one of the posts that there was a user that made a revision to the file etc/network/interfaces file so I typed into the terminal window after navigating to that directory "sudo gedit interfaces" and noticed that not all of the interfaces that I had were setup to automatically run. I edited the one that was not automatic so that it became automatic, hit ctrl-alt-backspace, logged in again and suddenly my system alerted me that it could connect to Black Water Cafe! I'm in. Thanks Keith...
