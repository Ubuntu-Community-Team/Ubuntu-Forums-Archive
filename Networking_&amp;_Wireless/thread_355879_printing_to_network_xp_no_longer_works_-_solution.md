### Post by randomnumber on 2007-02-07
I have had my networked Ubuntu systems. printing to a networked XP machine. I recently had trouble printing. I found that I my problem was that I had installed firestarter (firewall). It runs even if the applet is not visible. You can just allow the service through the firewall. 

I thought this might help someone if they saw it and had the same trouble. I thought that I was lucky to think of this and I considered how much time I would have spent figuring it out if I did not. That was when I decided to post this.
