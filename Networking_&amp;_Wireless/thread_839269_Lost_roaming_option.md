My wireless has been working beautifully, until I tried to set a manual configuration to connect to a work wireless network. Now my network icon is set for manual configuration with no roaming option available any more.

Does anyone know how I can get the network manager back to its normal state, showing the available wireless networks. Thanks.

---

### Post by issih on 2008-06-24
Click on icon and select manual configuration.

Select the wireless connection and edit properties.

Tick roaming in that dialog and you are done.

Note that you may need to unlock the networking dialog if you using hardy.

Hope that helps

---

### Post by issih on 2008-06-24
One thought, you may want to save the manual configuration as a location (e.g. work) and the roaming set up to (e.g. home) then you can just switch he location to change the settings.
