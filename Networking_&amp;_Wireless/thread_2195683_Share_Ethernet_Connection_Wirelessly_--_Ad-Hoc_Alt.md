### Post by grahammechanical on 2013-12-26
In Ubuntu 13.10 we can go to System Settings>Network and we will see a button labelled "Use as Hotspot." I have never used this button. I have no need. But that should set up the wireless adapter in Access Point (AP) mode if the wireless adapter is capable of working as an Access Point. I do not remember this function being available in 12.04. Although 12.04.3 may have been given the upgrade to the Network utility in System Settings. I do not know if this is so.
