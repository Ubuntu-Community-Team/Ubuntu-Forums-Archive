### Post by espressobeanie on 2010-12-06
Try a TRENDNET TEW-644 UB. Using a wired connection, download the linux wireless backports on your computer to help stabilize the random disconnects. You could also try downloading the trendnet wireless driver for windows xp and install ndiswrapper and install the driver on that. Gotta warn you, ndiswrapper drivers can make your kernel panic on boot if something goes wrong.
