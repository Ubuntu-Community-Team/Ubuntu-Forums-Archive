### Post by varunendra on 2014-03-10
> **tom62 said:**
> ```
***** rfkill *****

0: phy0: Wireless LAN
    Soft blocked: no
    **Hard blocked: [COLOR="#FF0000"]yes[/COLOR]**

```

Your wireless is physically turned off. If there is some Key or Key combination (e.g., Fn+F2), please try that and check with -
```
rfkill list
```
Both Hard and Soft blocked should be "no". If there is some physical switch, please make sure it is in "On" position.

Also, some BIOS have this feature to automatically turn the wireless interface off if a cable connection is detected. So if the above suggestion to try the switches doesn't seem to work, please disconnect the cable and see if wireless gets enabled. If not, reboot with the cable unplugged, go to BIOS and make sure wireless is enabled there.

Please check this thread out if having trouble with turning it on : [http://ubuntuforums.org/showthread.php?t=2107871&p=12489467#post12489467](http://ubuntuforums.org/showthread.php?t=2107871&p=12489467#post12489467)

**PS:**
The wireless takes a few seconds (can be upto 4-6 seconds) to change its state from "off" to "on", so give it some time after trying the switch/key-combo.
