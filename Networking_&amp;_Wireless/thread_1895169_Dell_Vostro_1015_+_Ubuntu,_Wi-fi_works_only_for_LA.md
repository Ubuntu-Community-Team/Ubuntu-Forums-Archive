### Post by tossshik on 2011-12-14
Hi everybody!

I've got this weirdest bug after installing 11.10 on Dell Vostro 1015 laptop (Broadcom BCM4312 Wi-fi chip). I can see all wireless networks, I can connect to any of them, have IP address assigned and everything, but after that I can access only local network. When I try to load any web site or otherwise access any WAN server it just doesn't work. Domain names are resolved, but that's it.

I tried to install Ubuntu 10.10 and 11.04 and got the same problem. Wired connection on the other hand works flawlessly. When WinXP was installed on this machine I had no problem with Wi-fi. 

I tried to use both STA and b43 drivers. I also tried all solutions from [this]("http://ubuntuforums.org/showthread.php?t=1604868") thread. 

Any help is appreciated!

### Post by praseodym on 2011-12-15
Blacklist the following module:

```
echo "blacklist brcm80211" | sudo tee -a /etc/modprobe.d/blacklist.conf
```
and reboot.
