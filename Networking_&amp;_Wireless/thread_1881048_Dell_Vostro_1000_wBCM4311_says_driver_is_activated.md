### Post by diablo75 on 2011-11-15
Okay, here's what I did.

1.  I checked out the link that bkratz posted, which calls for the b43-fwcutter and firmware-b43-installer packages to be installed.  However, they were already installed.  Skip to step two.

2.  Below this are a couple of commands suggested if the above doesn't work.  The key one was:

```
sudo apt-get remove --purge bcmwl-kernel-source
```

It should be pointed out that the double-hyphen in front of the word purge had to be manually typed/replaced because the HTML formatting wasn't entirely removed when posted to the web page.

The above command removed some stuff, including a few blacklist files in /etc/modprobe.d/

3.  I opened up /etc/modprobe.d/blacklist.conf and inserted a # in front of the line that says "blacklist bcm43xx".

4.  Rebooted.

It works!!  You guys are great!  Thanks for your help.
