### Post by Rem0te C0ntr01 on 2009-03-07
I am not sure what type of router you have, but most are all similar to set up. You access the router configurations by typing in its IP into your web browser. It will prompt you for an administrator password that you may have already set up, but if this is a new router, or recently reset, it will have a default password like admin or something. After you get into the router configurations you should be able to navigate to the right place to set up a password. I recommend setting up the password with WPA or WPA2 encryption. Don't use WEP encryption as it is not very secure, and now outdated. After that, save your settings and try connecting. You should be able to see your SSID (network name), and connect by entering in your new password. I hope this helps you.
