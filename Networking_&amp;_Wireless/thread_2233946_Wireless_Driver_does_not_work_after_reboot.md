### Post by Hadaka on 2014-07-11
Hi,Let's verify that you are loading the correct driver with this guide
[http://ubuntuforums.org/showthread.php?t=2214110](http://ubuntuforums.org/showthread.php?t=2214110)
The bcmwl-kernel-source is also known as the wl driver. If the above
guide verifies that wl...bcmwl-kernel-source is correct then boot your
computer and enter this command..
```
sudo modprobe wl
```
**If that works it is a temp fix and will be erased on next boot
To make it permanent please COPY and paste this command
```
echo "wl" | sudo tee -a /etc/modules
```
thanks.
