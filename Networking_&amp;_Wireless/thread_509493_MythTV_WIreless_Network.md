### Post by newlinux on 2007-07-25
The best place to start for ubuntu is here:

[https://help.ubuntu.com/community/MythTV](https://help.ubuntu.com/community/MythTV)

Which covers various scenarios (frontend/backend, frontend/backend/desktop, frontend only, backend only, etc.)  and tuner cards and other hardware (e.g. remotes). Read through a lot of it before beginning.

As far as wireless goes if you get it working with ubuntu then myth will work with it. If you are using wireless G then high definition will probably be a problem. You also might run into problems if you are streaming  multiple frontends at once over wireless, but it depends on the quality of your wireless connection the and the bit rate of your recordings. One of my frontends has a wireless connection and everything works great except for HD, which sometimes stutters.

Also, you may run into issues if you use your network for a lot of non-myth wireless traffic at the same time as you are using a wireless frontend.
