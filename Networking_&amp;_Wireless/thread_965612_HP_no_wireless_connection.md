### Post by bobmct on 2008-10-31
You need to have the correct wireless drivers.  Your system most likely has a Broadcomm 43xx wireless adapter.  

Use google or this site to search for the following:
hybrid-portsrc-x86_32_5_10_27_6.tar.gz

Then download it and use your network configuration to reconfigure the wireless adapter WLAN and elect to change the driver.  Point to this file where ever you stored it.  Upon completion you should have a working wireless adapter.

Hope this helps - Good luck.
