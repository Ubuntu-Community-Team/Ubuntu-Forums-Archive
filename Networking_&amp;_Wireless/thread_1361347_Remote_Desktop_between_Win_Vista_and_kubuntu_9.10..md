### Post by KeLa on 2009-12-22
Ask the person to go [http://www.teamviewer.com](http://www.teamviewer.com)
and download and install TeamViewer version 5 to windows computer.
Then you go to address [http://login.teamviewer.com](http://login.teamviewer.com) and create account
there and login and get remote connection to that person's computer.

I recommend that the TeamViewer is installed so that it won't start automatically
(only from menu or icon in desktop)
And you don't have to install anything on your linux box.

If there is firewalls (one or more) between your and that person's computer
it doesn't matter at all, everything will work great.

There is "support" module of the TeamViewer for Windows and MAC but not to Linux
And only extra what that browser access needs is Flash plugin in your browser.

TeamViewer includes remote control, file transfer and chat.

I've used it for long time now and it's great.
