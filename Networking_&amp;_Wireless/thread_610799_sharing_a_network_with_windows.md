### Post by vvnoob on 2007-11-14
If you want to share internet connection without router, beware that one ( main computer) have to be online all the time otherwise second will not have a internet connection. If you go windows main computer, you have to have two ethernet cards, first for ISP connect and second for ubuntu box. Configure network on win box ( this computer is connected directly to internet and other computer connect over this computer.... something like this ). connect win and ubuntu box with crossover cable and configure ubuntu box. That's all. If your win box is shut down, ubuntu will not have internet conection.
