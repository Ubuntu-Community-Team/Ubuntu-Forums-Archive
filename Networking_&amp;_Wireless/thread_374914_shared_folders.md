### Post by kidders on 2007-03-03
Hi there,

Since you need Windoze compatibility, I would opt for Samba (an implementation of the MS filesharing protocols used by Linux/MacOS/etc.).

There are plenty of HOWTOs here (and elsewhere) to help you get started. Playing around with Samba requires a certain level of familiarity with how Microsoft's filesharing works, but you should find the basic setup straightforward.
