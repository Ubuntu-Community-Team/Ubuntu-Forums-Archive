### Post by EngineerGuy on 2007-05-31
The Linux Firware upgrade to Linksys routers does offer more flexability and user control. Some people actually have done throughput tests a nd have had good sucess in increasing it.

I used to run a BSFR41-V2 Linksys (4 port wired) with an external Xircom (now Cisco) Wireless 11b Access Point (AP).

but I was tired of the dropped connection and other issues with the router.
I also wanted to add a firewall to block all those Spider sites snooping my IP.

I decided to build my own router/firewall.
I had an old K6-2 550Mhz Computer with 256Meg of ram. What I did was removed all the unnecessary stuff like Modem and USB cards.
I put two network cards in the unit (I used two different types to distiguish them apart)
One is a 3Com 3C905TX (I believe) and the other a realtec 8139C.

The system had a 20 Gig HD, so that was plenty for what I wanted to do.

I downloaded Smoothwall Express 2..0 and installed it.

Configuration was not dificult and is explained well in the documentation that is part of the ISO (if you so download the one with docs)

[http://www.smoothwall.org/]("http://www.smoothwall.org/")

My Throughput tests were wonderful

The best I could do with the Linksys on a 600 MB file downloaded from work lab (OC-3 connection speed -  Approx 135 MB pipe). Tnroughput average was 572Kb/s.

I hooked up the Smoothwall box and the throuput average was 731Kb/s

I ran this test 5 times and the results were always the same. the Router/Firewall I built outperformed my linksys and did better than my D-Link 604 (throughput average 607 Kb/s).

So if you have an old computer around, revive it to nmake a very nice router and firewall.

The added bonus is that you can Block IP Address (single or Host or Network wide)
Also there is Intrusion detection. plus DHCP, Web Proxy and more.

If you do decide to use the smoothwall firewall as your router make sure you do all the updates and reboot it after each one.

I have built quite a few of these for friends and neighbors, and the funny part was that the computers I used were bought for around 30-40 dollars at computer shops and a few were found on the curb for trash pick up after the Christmas holidays.

Enjoy

### Post by tahuya_rat on 2007-05-31
My 2 cents:  I have a Linksys WRT54GL (the "L" stands for Linux, BTW, as in Linux Firmware).  *All* GL's can be easily flashed, regardless of version number, and in fact, v5 of the WRT54G can now be flashed, as well.  I live in an RV park, and have a RadioLabs 8db omni outdoor antenna attached to the router, via a Hawking 500mw amplifier.

I am running DD-WRT v23sp2 firmware on it, it has been up for months with *no* issues whatsoever, and at any given time, there are 8-10 active wireless clients from around the park active on the DHCP. (You can see all active connections in the Status page of the firmware).
