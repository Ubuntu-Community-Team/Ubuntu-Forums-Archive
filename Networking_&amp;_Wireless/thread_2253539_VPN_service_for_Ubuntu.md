### Post by weatherman2 on 2014-11-20
I use OpenVPN.  The software is free to install and use in Ubuntu, and there are free OpenVPN clients available for Mac OS and Windows too.  You can setup an OpenVPN server in Ubuntu on your own server or pay for an account to use an OpenVPN service somewhere else, if you are simply trying to hide your IP address.
