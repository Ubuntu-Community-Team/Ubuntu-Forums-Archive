### Post by jml on 2006-05-10
Getting USB based wireless netwok adaptors to run in Linux is a real challenge.  I haven't been able to get one working either.  If you do not need mobility, just wireless access, I've read that you can use a wireless access point connected to your computer's ethernet jack  and use it as a wireless bridge.  This is useful if you can't run an ethernet cable to your computer.  It won't help if you want mobility, however.

If your computer has a PCMCIA slot, you are more likely to get wireless working with a wireless card.  Just be sure to check the hardware compatibility link on this site to be sure you pick the right card.  Here is a good resource for wireless informatiom.

[https://wiki.ubuntu.com/WirelessTroubleshootingGuide](https://wiki.ubuntu.com/WirelessTroubleshootingGuide)

Hope this helps.
