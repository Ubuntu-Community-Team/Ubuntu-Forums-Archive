### Post by chili555 on 2013-10-26
There are several things you might try. Some may be effective and some not. We hope we find the one trick that fixes the issue. First, I notice this:> *TALKTALK-C64320:Infra, 34:08:04:C6:43:20, Freq 2422 MHz, Rate 54 Mb/s, Strength 58 [COLOR="#FF0000"]WPA WPA2[/COLOR]If you have administrative privileges over this router, I suggest you set the router to use WPA2 *only*; not mixed mode WPA and WPA2. I also suggest you experiment with 802.11N turned off.

Next, please run:```
dmesg | grep CRDA
```Is your correct country code set, like this?> cfg80211: Calling CRDA for country: USOr is it blank; that is, one size (maybe) fits all? If it is blank or incorrect, set it:```
gksudo gedit /etc/rc.local
```Right above exit 0, add this:```
iw reg set US
```Of course, substitute your country code from here if it isn't US. [http://en.wikipedia.org/wiki/ISO_3166-1_alpha-2](http://en.wikipedia.org/wiki/ISO_3166-1_alpha-2)  Proofread carefully, save and close gedit.

Reboot and let us have your report.

> **djwalk2 said:**
> Thanks chili555. I have checked the country code and it is correctly set. Before I set the router to WPA2 only, could that have repercussions for other web users in the house? There are several.
 Would it be wise to check with my broadband provider first?It could cause repercussions in two instances. First, when you actually change the setting, and ask the router to 'Apply,' I believe on-line users will be momentarily disconnected and then reconnected. Second, if anyone in your home is using an ancient wireless device that lacks the capability to use WPA**2**, then they will no longer be able to connect. I suspect any device manufactured in the last 6-7 years does WPA2.

As to whether or not you should contact your ISP, I have no idea what the arrangement you may have. In my locale, the USA, most ISPs deliver a working connection to the doorstep. Within few limits, you are free to do what you will inside the home. If you wish, you may run no encryption, WEP, WPA, WPA2 or even another encryption scheme you invented yourself!
