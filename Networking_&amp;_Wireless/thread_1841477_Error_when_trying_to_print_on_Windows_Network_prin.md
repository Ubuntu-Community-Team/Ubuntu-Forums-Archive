### Post by markcoronado on 2011-09-09
I'm getting the Idle- Connection Failed: NT_Status_Bad_Network Name error when trying to print to a HP DeskJet D2620 PRINTER that's connected to my Windows 7 Desktop.

I also have a HP OfficeJet 6110 connected to the same desktop & I have no problem printing to it.

Both printers were installed on my Ubuntu 10.04 LTS 64 bit Laptop at the same time using the same process.
System->Administration->Printing
Click "Add"
Expand "Network Printer"
Click "Find Network Printer" and enter the IP address of the machine the printer is shared on and click "Find".
Select the printer you want to add from the list that appears and click "Forward"
Follow the wizard to finish adding the printer.

Why does the 6110 work & the 2600 not?
