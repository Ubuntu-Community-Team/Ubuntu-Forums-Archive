### Post by harner on 2007-06-06
Most routers have a 192.168.1.1 address.  If you type that into your web address bar you should be prompted for a username and password.  If it is the default, it is probably something like "admin" for username and "password" for password.

From there you can set encryption, which allows you to generate a pass phrase.  Your wireless devices will notice the network is encrypted and prompt you for the code.  Enter the code and the wireless devices are online!

You can always call Dlink tech support to get help.  Personally, I use a Linksys WRT54G.
