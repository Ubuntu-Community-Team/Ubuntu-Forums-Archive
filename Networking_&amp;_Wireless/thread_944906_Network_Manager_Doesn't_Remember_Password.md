### Post by Patrick793 on 2008-10-11
I have my wireless card working fine, and I am connected, but I have one problem.

My connection is WPA2.

When I reboot my laptop, the wireless connection gets set back to WPA, and the password is gone. Is there any way I can get it to remember the information, or any way to disable network manager and use something else? It gets annoying having to enter my wireless password and choose my security on every boot.

EDIT:

Fixed my problem. Installed Wicd.

Wicd should be the default network manager.
