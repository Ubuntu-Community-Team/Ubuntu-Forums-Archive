### Post by oldsoundguy on 2010-11-06
> **cracker89 said:**
> Right. how do i get that working?

I just plugged in the cat 5, ran it through the wall and plugged it into the existing router.  Then powered up the access point .. when I re booted the wireless computers, the access point was there.  Instructions for hooking it up usually come with the units.  You MAY have to hook it to the main computer first to give it an address (instructions will tell you how).

I use a D-Link DL-624 router, a D-Link DWL-7100AP access point and D-Link DWL-G520 wireless cards (in all of the desktops that are in the block portion of the building. (ALL equipment was bought off of eBay .. CHEAP!!)

Using all of the units from the same maker makes getting support a lot easier if you need it.
