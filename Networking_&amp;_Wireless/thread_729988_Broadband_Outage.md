### Post by dstew on 2008-03-20
An outage is when their machines go down for some reason. A server can be attacked, overwhelmed with requests, or just crash. It can be a power glitch, or a cable cut or rain leaked into the server building or whatever. If it happens on a weekend, when techs are not around, the server is usually set up to page somebody. But often you just have to wait until Monday for the thing to get fixed.
