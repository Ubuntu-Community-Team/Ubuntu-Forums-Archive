Hallo Everyone,

I have an Acer Aspire One which I recently converted to Ubuntu (Lubuntu, 15.10, all updates installed).  It seemed to work fine for a while but now I have sporadic failure of the wireless connection:  It appears and disappears and sometimes all connections (and detected signals) disappear completely.

I thought it might be hardware (a dodgy connection somewhere) so tried and external USB wireless adapter.  But the same happened with that.

Has anyone got any ideas/tips for me?

### Post by asearle on 2016-01-14
Hallo Everyone, I found the solution here:  I had installed Lubuntu 32bit and the computer turned out to be 64bit.  I reinstalled and the wireless is now stable.  Sorry for the false alarm but maybe this posting will help others too.  Regards, Alan
