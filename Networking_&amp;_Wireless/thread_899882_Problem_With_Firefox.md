### Post by spin2cool on 2008-08-24
This is not a ubuntu issue, but a well-known limitation of Adobe's current implementation of flash on linux.  (no element can overlay a flash item) 

Supposedly, this will be fixed in the next version of Flash, which is in alpha testing right now.  Look for it in the next few months.
