### Post by aesis05401 on 2009-07-24
> **philcamlin said:**
> right click save page as 

i think thats what you want

Would that give the functionality to play the games?  

I am not going to post a script, because most websites have terms of service that prohibit bot-scraping.  A simple website is nothing but a marked up bit of text that command line utilities can easily break down into lists by tag (or any number of other factors).

From there you can run gets on each item in the link list... 

An evening reading man pages on command line text manipulation/command line web tools will yield all the secrets you need.  

I will warn you now that good sites monitor for scraping activity and you will get IP banned from a lot of places for generating bot traffic.
