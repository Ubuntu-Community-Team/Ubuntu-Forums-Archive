### Post by ci9her on 2015-10-06
SO me and friend were using ubuntu 15.04 when suddenly noticed that wifi doesnt work in 15.04(well in our case it didnt work)
so we experimented and found a way to work th wifi.
This is for broadcom wifi adapters:-
go to [http://linux.dell.com/dkms/](http://linux.dell.com/dkms/) and download dkms*.deb package.
install it in ubuntu 15.04
now go to \pool\restricted\b\bcmwl of the live ubuntu media.
install the package inside bcmwl.
This works for broadcom wifi adapters.
Others can try can tell me if it works or not
