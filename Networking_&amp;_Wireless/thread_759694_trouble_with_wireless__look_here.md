### Post by linuxnoob18 on 2008-04-19
if your using wireless and you are having trouble setting it up do this

1> download ndisgtk form [http://linuxappfinder.com/package/ndisgtk](http://linuxappfinder.com/package/ndisgtk) NOTE : make you select the right one for your distribution of Linux i.e 7.10 or 7.04   when downloading save to destop this will make it easy to find

2> insert ubuntu cd it will ask you to do so 

3> proceed to install the application

4> insert wireless driver cd and browse for the .inf file

5> once that is done it will tell you if the hardware is present if above steps have been carried out correctly it should say yes

6> you need to click configure network you will get a window showing you network ESSID that is simply the name of your router and any others you may find

7> once you have slected your router you need to enter other information i.e WEP key make sure key type is set to hexadecimal  lower down on that screen you should see wireless configruation slect automatic (DHCP)

click ok and reboot this may need to be done a few times if you still cant connect redo these steps and make sure your WEP key is correct and other settings
