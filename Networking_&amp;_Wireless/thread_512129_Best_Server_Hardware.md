Anything better than a Pentium one works fine.  A pentium one will work fine, but you will need to install the OS using the alternate cd.  That's because the server cd installs the server kernel and it only works on a 686 or better processor.

The desktop kernel (installed by the alternate cd) will run fine on a pentium one.  It should even boot a 486.
