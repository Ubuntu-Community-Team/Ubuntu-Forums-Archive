### Post by spd106 on 2007-11-22
There are several applications that can do this, they are usually a variant of vnc, though there are a few others such as xdmcp.

Ubuntu has a built-in vnc client called Terminal Server Client, under Applications -> Internet.

See [https://help.ubuntu.com/community/VNC](https://help.ubuntu.com/community/VNC) for more.
