### Post by Cheesehead on 2013-06-11
As long as both are connected to the same router/gateway, they will work in parallel.
However, wired ethernet tends to have much greater capacity, so any performance improvement by adding WiFi may be marginal.
For example, copying a one-hour HD video within my LAN takes 50 minutes by WiFi, and 4 minutes by ethernet. And still 4 minutes using both together.

If wired and wireless networks seek a different router/gateway, then the system will choose only one route to the internet. You can define a second route to internet addresses in your routing table (using the 'route' command), and any IP address can have only one route. For example, either gateway can to programmed for YouTube, but not both. I generally find it to be much more work than the improvement is worth.
