### Post by klemes on 2010-04-27
A quick glance in google did not reveal any Linux support for this specific wi-fi adapter model.
Issue the following commands in the console in the hope that they may help to reveal the type of chipset your dongle uses and then try to see if it is supported in Linux with native drivers:

```
sudo lshw -C network
sudo lsusb -v
```

If not you can always use ndiswrapper and use the windows drivers that came with your wi-fi dongle I guess...:):popcorn:
