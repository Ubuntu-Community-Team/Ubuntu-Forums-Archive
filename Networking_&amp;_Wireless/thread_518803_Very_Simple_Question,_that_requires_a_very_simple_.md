### Post by EndPerform on 2007-08-06
If your box is set up for DHCP, there shouldn't be much to it:

- Connect the modem to the router

- Connect the machines to the router

- Configure the router (it may need something set up for your ISP; this varies)

Most modern routers come with DHCP set up by default, so you should be pretty well set to go.
