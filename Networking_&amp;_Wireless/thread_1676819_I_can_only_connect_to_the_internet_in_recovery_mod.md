### Post by grahammechanical on 2011-01-27
Go to System>Preferences>Startup Applications and make sure that there is a tick mark against Network Manager and then re-boot. Try that.

If there is a tick mark then right click the top panel and select Add to Panel and select Notification Area. This where Network manager should reside.
