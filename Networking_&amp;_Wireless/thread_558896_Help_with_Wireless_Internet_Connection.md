### Post by Kilz on 2007-09-30
Ok Ill help with editing files in the system. vi is a text editor, but you may find gedit or kate easier to use.

If you are using Gnome open a terminal. Type in 
```
gksudo gedit /path/to/the/file
```
This will open up the text editor in admin mode. You will be able to save the documents. You could also use 
```
gksudo nautilus
```
This would open up the nautilus file manager in admin mode. Then when you open up a file in it it is still in admin mode. This is also helpful for renaming and moving files in the system. But be careful. It is possible to mess up your system in admin mode.

If you are running kubuntu with the kde desktop
```
ksudo kate /path/to/file/to/edit
``` is the command to open up the kate text editor in admin mode and 
```
ksudo konqueror
``` opens up the konqueror file manager in admin mode
