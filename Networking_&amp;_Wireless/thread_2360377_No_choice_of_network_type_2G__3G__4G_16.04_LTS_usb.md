### Post by Bucky Ball on 2017-05-04
Welcome. Why don't you log in to the router and set it up there? Not a Ubuntu 'thing'. You won't be able to fiddle with router settings from your Ubuntu desktop or Network Manager that I know of.

Open a web browser and type in the IP address of your modem then hit return. Should take you to the log in page for your router config. Look it up in the manual if you don't know it, but it is usually something like 192.168.0.1. Try that (no period at the end, just '192.168.0.1').

You can probably set 3 or 4g from the router config. Despite the fact hardly anyone reads them, manuals often have a lot of useful information! If you haven't checked yours, have a look. It might fill in some gaps. ;)
