### Post by rotomano on 2014-03-10
The problem seems to be with gvfs-dav (a component of Nautilus = Ubuntu's file manager), so until it is fixed, if ever, you can use a different file manager that does not rely on gvfs, like **Krusader**, in which dav shares work fine.  See: [http://www.krusader.org/](http://www.krusader.org/)
That is the only workaround I could find, for this very annoying issue ...
