I'm trying to connect from a Mac to Hardy with JollyfastVNC client and Chicken of the VNC but Ubuntu will not accept the password I have set up. If I remove the password I can connect. Is there a solution for this?

### Post by xadloki on 2008-06-25
Solved. 
To make it work:
1. erase the password field but keep it checked
2. log out
3. log in and type you're new password
4. log back out and in the next login it works
