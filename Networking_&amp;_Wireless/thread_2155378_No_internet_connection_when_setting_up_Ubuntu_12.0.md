### Post by praseodym on 2013-06-18
No problem. Install them in the Live-system via

```
sudo apt-get install build-essential dkms linux-headers-$(uname -r) linux-headers-generic
```
Then copy all ".deb" files from the live system's folder /var/cache/apt/archives to a stick. After the OS installation to the hard drive copy them to "Downloads" (Edit: of the installed system) and install them via
```

sudo dpkg -i ~/Downloads/*.deb
```
Download the file

[http://media.cdn.ubuntu-de.org/forum/attachments/55/19/4987097-alx_dkms-cw20121003.tar.gz](http://media.cdn.ubuntu-de.org/forum/attachments/55/19/4987097-alx_dkms-cw20121003.tar.gz)
and save it in "Downloads",too. Installation via
```

cd Downloads/
sudo tar xvf 4987097-alx_dkms-cw20121003.tar.gz -C /usr/src
sudo dkms add alx_dkms -v cw20121003
sudo dkms build alx_dkms -v cw20121003
sudo dkms install alx_dkms -v cw20121003
sudo depmod -a && sudo update-initramfs -u 
```
Check the installation via:```

dkms status
modinfo alx
sudo modprobe -v alx
```
