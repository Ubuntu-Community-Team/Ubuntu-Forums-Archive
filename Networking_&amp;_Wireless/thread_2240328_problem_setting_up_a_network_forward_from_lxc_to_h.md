### Post by nayeem3 on 2014-08-19
1. intention: 
to forward all requests coming to 172.31.23.17:40000 to go to 10.0.3.95:80

2. way of accomplishing it:

> sudo iptables -t nat -F lxc-nat
sudo iptables -t nat -D PREROUTING -j lxc-nat
sudo iptables -t nat -X lxc-nat
sudo iptables -t nat -N lxc-nat
sudo iptables -t nat -A PREROUTING -j lxc-nat
sudo iptables -t nat -A lxc-nat -d 172.31.23.17 -p tcp --dport 40000 -j DNAT --to 10.0.3.95:80

3. what i expected
to be able to get a response from apache which is running on 10.0.3.95:80 when running
> curl 172.31.23.17:40000

4. what i observed instead:
`curl: (7) Failed to connect to 172.31.23.17 port 40000: Connection refused`
