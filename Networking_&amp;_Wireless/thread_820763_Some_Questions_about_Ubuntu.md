### Post by diablo75 on 2008-06-06
What is the make and model of your wireless adapter?  ndiswrapper is sort of a Plan B solution.  The first thing you should check is System>Administration>Hardware Drivers to see if any are listed as needing to be enabled.

Also, if this computer has not yet been able to talk to the Internet and check for updates, take it someplace where you can hook up a Ethernet cable and check for updates real quick so you can make sure your system is as up-to-date as possible.  Many times, I've been able to get wireless adapters working just by updating after a fresh install and checking the Hardware Drivers section.
