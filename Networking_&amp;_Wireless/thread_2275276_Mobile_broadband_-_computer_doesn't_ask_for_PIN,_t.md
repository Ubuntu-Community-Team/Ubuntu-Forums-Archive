### Post by Mike_Walsh on 2015-04-25
Odd.

This obviously changes from release to release. I'm using a Huawei K3565 (it's about 8 years old, but still works fine on VodaFone UK). When using it in Vista a few years ago, you had to plug it in AFTER you booted-up, otherwise the system only saw the built-in mini-SD card reader. Then you got the set-up window, and just followed the guide. 

Under Ubuntu 14.04 (actually, Xubuntu 14.04.2 as I'm now using, on a very elderly Dell Inspiron 1100 laptop, from 2002) you plug it in BEFORE powering on. Having got it set up initially, you click on the Network icon on the top-bar, and select 'Enable Mobile Broadband'. The system then gives me a pop-up to tell me I'm registered on the local network. I then get the log-in window; I click on OK WITHOUT entering anything, the box pops up again, I repeat this procedure, and it's up & running. No PIN or password apparently required.

I appreciate that Gnome probably has a different user GUI for this, but since the 'buntus are all the same 'under the hood', I would have thought things ought to be pretty much along the same lines.....

Unless, of course, the log-in procedure has been significantly altered in 'Vivid'.

Mike.
