### Post by pcolbeck on 2008-10-22
Hi

I work with Cisco kit all day and sometimes HP too. You can use telnet or shh from within any terminal (Gnome Terminal, Xterm etx) to connect over a network or if you need to connect via a serial (console) cable then install minicom which is a serial terminal emulator like Hyperterm on Windows excetp you run it from teh command line.
There is also a Linux version of Putty now if you prefer that.
