### Post by doobiest on 2008-12-16
That chances of you finding a new wrt router that can take 3rd party firmware is slim to none.  You have two options. 

1) find out the hardware version of the g/gs router, should be written on the box. then go to dd-wrt's website and check the compatibility list.  I think anything over 5.0 wont work

2) Find the wrt-54gL model which is a new version which specifically allows third party software.

I think the GS has more memory and a speedbooster which from my usage has been a useless item.
