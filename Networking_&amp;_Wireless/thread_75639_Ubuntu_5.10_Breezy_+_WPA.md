### Post by matthew on 2005-10-14
During setup it will try to connect to download the latest versions of software. Since Breezy was just released you will already have the latest versions. You can just install and choose "do not configure network at this time" or something similar when running the installer (after it doesn't work the first time) and go through the full installation. Then you can go back and set up networking after all is running.
