I  am running an HP Pavilion DV5000 with a Broadcom BCM4311 Rev 2.  I was really excited to see that Gusty supported this almost out of the box with the new Restricted Drivers tool.  The one problem I am running into I have boiled down to this...

At home I run WPA with DHCP on and it worked perfect.  
I got to the office this morning which is also running WPA but we use static IP's with DHCP off and I could not get it to connect.  I read in the syslog indicated the network manager was not able obtain an IP...  I turned DHCP on in the router and BOOM!  It is working great.  Unfortunately, I would really like to give this machine a static IP for VNC and various other things.

I am running my wireless card in roaming mode, which I am playing with now to see if I can turn that off and still connect.

Has anyone else seen this?  Any ideas?
