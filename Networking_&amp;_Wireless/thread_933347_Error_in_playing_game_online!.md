### Post by noerrorsfound on 2008-09-30
Make sure you've got the latest version of Sauerbraten, which is currently the CTF Edition.

[http://downloads.sourceforge.net/sauerbraten/sauerbraten_2008_06_20_ctf_edition_linux.tar.bz2](http://downloads.sourceforge.net/sauerbraten/sauerbraten_2008_06_20_ctf_edition_linux.tar.bz2)
Also the patch: [http://downloads.sourceforge.net/sauerbraten/patch_2008_07_11_linux.tar.bz2](http://downloads.sourceforge.net/sauerbraten/patch_2008_07_11_linux.tar.bz2)

Older protocol should mean the server is for an older version, and newer should mean they're running a newer version. I say "should" because it's possible to modify your protocol number and register it to the official master server list, but you aren't supposed to register any modified servers on it.
