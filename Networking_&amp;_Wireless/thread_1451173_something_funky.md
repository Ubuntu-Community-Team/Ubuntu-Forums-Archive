### Post by quixote on 2010-04-12
That computer chip-looking icon gives you the option to install proprietary drivers, if I remember right.  For some reason it's skipping over that after the actual install.  

However. you can start the thing manually by going to System > Administration > Hardware Drivers.  It'll trundle for a while, searching for drivers, and then it should give you the option to install the ones you need, like it did when you booted the first time off the LiveCD.
