### Post by dineshs on 2010-04-16
Hope you have only one ethernet port and you connect only to internet(no local network)If so try this
```
gksudo gedit /etc/network/interfaces
```
The file should contain only this
```
auto lo
iface lo inet loopback
```
Now configure NM as follows
follow this guide
[https://help.ubuntu.com/community/NetworkManager0.7](https://help.ubuntu.com/community/NetworkManager0.7)
ie
Right-click on the nm-applet and then click on Edit Connections.
Select the tab, wired 
select the connection and click edit(you can add the ethernet connection if not listed)
select ipv4
method-manual
click add
address 192.168.0.110
mask 255.255.255.0
gateway 192.168.0.1
then hit enter
DNS 8.8.8.8
then click apply
Now connect broadband using pon dsl-provider
Does this help?
If not```
 ping 8.8.8.8 
```
What is the result?
