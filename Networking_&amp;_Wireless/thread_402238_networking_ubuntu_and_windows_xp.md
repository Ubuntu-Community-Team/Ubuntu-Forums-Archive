### Post by drmbogo on 2007-04-05
Just an addendum: I recently set up a wireless network and a wireless print server for a family member. Both units were Dlink. The print server kept crashing. After trying all the obvious things (reinstalling the printer, etc) I finally updated the firmware in both boxes.
The router's (sorry, I don't remember off hand the model #'s) firmware was some 6 or 7 revisions behind! And it was brand new! The print server was only 2 rev's old. Overall, though, it seemed to fix the problem.
Best of Luck
