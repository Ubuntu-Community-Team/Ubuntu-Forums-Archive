Hi, this seems to be a common problem after a lot of research. I gave a Ubuntu another chance in a shared computer with Windows 10.

Everything was going fine.

Windows was in one hard drive and Ubuntu in another one, so pressing the boot menu (F11) while booting would let the people select with OS would they like to run. Default boot drive was Windows so people would not have any problem.

But after a few reboots I noticed something strange, if I boot to Windows AFTER using Ubuntu, computer won't connect to the Wifi. No matter what you do, it just won't connect. Luckily with this PC the WiFi is a USB dongle, so unplugging it and plugin it again works. 

After a little research, I noticed this has been a problem from YEARS. And all the advised solution don't work for me.

I have tried:

Disabling secure boot in the BIOS
Disabled "Fast Startup" in Windows 10
Disabling energy saving for USB ports

NOTHING works.

Can someone give some insight? How a 5 years problem hasn't been fixed yet?
