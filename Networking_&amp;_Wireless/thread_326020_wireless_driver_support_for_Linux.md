### Post by FLPCGuy on 2006-12-26
We've suffered through the modem era with almost no Linux compatible dial-up modems.  Now many users have wireless notebooks with broadband but still can't use Linux because there is no driver support.  Linux desperately needs a software boundary layer interface for drivers of all kinds to make it easier to write drivers for softmodems, wireless nics, etc.  But ultimately we will need manufacturers willing to do whatever it takes to support Linux.  Perhaps we should be petitioning Taiwanese computer makers to provide Linux drivers letting them know how many people will make their next notebook purchase based on it's Linux hardware support.
