### Post by sudodus on 2013-06-04
1. Check very carefully all the differences in the settings! If you find no difference, copy the virtual machine that works, so that you have two instances of it.

2. The problem might be the operating system that you have installed into the second virtual machine. Try some other settings. Sometimes NAT works instead of Bridged network.
