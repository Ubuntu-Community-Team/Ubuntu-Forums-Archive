### Post by scorp123 on 2010-08-22
Use TeamViewer. It's very noob-friendly. All your friend has to do is to read the numbers (which represent Login ID and password!) to you that will show up on his screen .... and baaaam, you're connected.

[http://www.teamviewer.com](http://www.teamviewer.com)

TeamViewer is commercial software but it can be used freely for non-commercial + private purposes.

When you launch it for the first time you will get this (I know: this is a Windows screenshot ... but it does not matter. Apart from the window decorations the program looks absolutely the same on Windows, Mac OS X and Linux!):

[IMG]http://www.teamviewer.com/en/res/img/screenshots/win_mainwindow.jpg[/IMG]

So when your friend starts his TeamViewer session, all he has to do is to read those numbers to you ... you type in those numbers in your window and taaadaaaaaa ... you get connected.

And you don't need to worry about NAT, firewalls, reconfiguring routers and what not. This thing just works.
