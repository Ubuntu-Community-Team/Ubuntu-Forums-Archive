### Post by virgo17 on 2008-05-13
I have the WAG200G. Setup of the router is independent of the O/S.

I have Hardy on a HP NX6110 Laptop which has a Broadcom WiFi chip. It links fine via WiFi to the WAG200G using WPA.

I am very pleased with the WAG200G's. I have two, one in my home in UK and one in my home in Spain. Both work fine with laptops running Hardy. Make sure you have the latest firmware on them. Early firmware had a problem restoring an internet connection if it was lost for any reason. 100% reliable with latest firmware.

HTH.
