Hmmmm...

I hate to change things without full knowledge of the system.  In this case we are going to make a very small change as it is easily repairable if it doesn't work.

You will need to edit the /etc/nsswitch.conf file with a text editor using sudo. ```
gksudo gedit /etc/nsswitch.conf
```

The secton of the code we are looking for is```
**[COLOR="Red"]#[/COLOR]**hosts: files mdns4_minimal [NOTFOUND=return] dns mdns4
hosts: dns
networks: files
```

We want to move the comment sign (**[COLOR="Red"]#[/COLOR]**) to the line below where it is.  It will look like this when you are done```
hosts: files mdns4_minimal [NOTFOUND=return] dns mdns4
**[COLOR="Red"]#[/COLOR]**hosts: dns
networks: files
```

Then you need to reboot the system and again try ```
hostname -i
```
