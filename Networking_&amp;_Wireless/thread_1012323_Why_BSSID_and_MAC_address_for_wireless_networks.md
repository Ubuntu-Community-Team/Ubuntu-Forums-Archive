### Post by Meson on 2008-12-15
Why does NetworkManager have a field for BSSID and MAC address for wireless network settings?  Shouldn't they be the same thing?  Should I set both to the same thing? Or pick one?

---

### Post by sloggerkhan on 2008-12-15
While technically a BSSID and MAC address appear identical, a MAC address is in theory set and tied to your hardware while a BSSID is randomly generated for the network in question. In practice you can set your MAC to whatever you like so the difference isn't really existant.
