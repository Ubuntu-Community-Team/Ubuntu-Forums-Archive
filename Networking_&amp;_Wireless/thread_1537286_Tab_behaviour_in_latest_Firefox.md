### Post by bravo sierra echo on 2010-07-23
My Firefox has just updated itself to 3.6.7, and the update seems to have changed the behaviour of new tabs being opened.

They used to appear on the end of the list of tabs - now they appear next to the parent tab, like Windows Explorer does.

I have to use Explorer at work and for some reason that I can't explain, I prefer the Firefox way of having them appear at the end of the list.

Is there any way to change it back?  No obvious settings present themselves in the Preferences menu.

---

### Post by dca on 2010-07-23
*DO THIS AT YOUR OWN RISK*
 
type in new opened blank Firefox tab *about:config*
 
...in the search area, type in *tab*
 
Don't quote me but the line you're looking for should say 'browser.tabs.InsertRelatedAfterCurrent'...
 
Highlight, right-click (what-ever) and select 'Toggle'.. It should set it to 'True' which I think is what you're looking for.  Using Chrome right now, or I'd test it on mine...
