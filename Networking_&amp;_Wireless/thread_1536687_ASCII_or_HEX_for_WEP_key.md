### Post by ajgreeny on 2010-07-22
If you are talking about the entry of the password/passphrase when you try to login to a wireless network, the answer will depend on the dropdown box you try to fill.  My router is set to a 128bit password and that is exactly what is needed when loging in if I choose the **WEP 128bit encryption** dropdown box when asked for the password/phrase on logging in to the wireless.

Interestingly I have always found that the full encrypted hex key is needed when doing the same on windows boxes;  don't ask me why, I have no idea, but I have never found the actual password to work in the windows boxes of my friends, and I don't have any windows to check it.
