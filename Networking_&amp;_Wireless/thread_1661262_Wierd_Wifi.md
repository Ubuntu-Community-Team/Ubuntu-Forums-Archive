### Post by PatchesTheCaveman on 2011-01-06
A MAC address is a unique identifier assigned to every wireless network adapter (and wired Ethernet network adapters as well).

It's not very secure because you can spoof them, so if you can figure out a MAC address of another machine that has been authorized, you can get on the network.  Also, since no other security is used, anyone can snoop on all the traffic on the network.
