### Post by Claus_T on 2007-05-28
I was looking for a wireless adapter for my test install.  I discovered that if the adapter or card is [COLOR="Red"]not[/COLOR] listed as having Linux support, or is [COLOR="Red"]not[/COLOR] on the "supported" list, then it won't work.  It seems that almost all of the new wireless models are using 3rd party drivers and chips that are usng MS Windows OS propritary code to safeguard their property.  Most of these, interestingly, are being made in China.

This, generally means that we are relegated to older models, that do have specific Linux support.  What makes it even harder is that many of the wireless manufacturers seem to have 2 different versions of chips and drivers (only one of which is supported) with the same model numbers - so there is no way to tell (a priori) if what you are buying is supported or not.
