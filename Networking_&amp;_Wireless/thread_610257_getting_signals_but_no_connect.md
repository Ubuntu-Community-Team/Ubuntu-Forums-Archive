### Post by ub9876 on 2007-11-11
got it fixed

finally figured it out...this might help some others with new ubu..
goto adm. then network,not network tools.
it will show 3 things. wireless,ethernet, and modem/phone.
there should be a check in the wireless box. if there isnt put one in
and wait for a process box to pop up and it will show a MOVING
bar for 10 seconds. if it doesnt do it again. I have already noticed in Linux that you have to repeat actions to get things moving.
Take the check out of the box and in the box until you see the movement in the status bar. You might have to do it 2-3 times.When the check is in there and you saw the install process in motion, your wifi should show a little connection strength bar under the network monitors icon.:popcorn:
