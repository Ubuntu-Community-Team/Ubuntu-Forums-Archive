### Post by stevecaldwell on 2007-07-05
I've been able to get Windows - Ubuntu printer sharing to work using a wireless networked Windows XP Professional desktop PC in my home.  My Ubuntu laptop and my son's Ubuntu desktop both can use the HP LaserJet 1300 attached to the Win XP Pro desktop PC.

I haven't been able to get this work at a friend's home -- no networking and no printer sharing.  The only significant software difference that I've seen is she's running Win XP Home (NOT XP Professional).

IIRC, the XP Home Edition has less robust networking support than the XP Professional version.
