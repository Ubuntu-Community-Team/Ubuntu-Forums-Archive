### Post by varunendra on 2012-10-06
Interesting indeed! May we see some basic configurations? -

```
ifconfig -a
route -n
cat /etc/network/interfaces
cat /etc/nsswitch.conf
cat /etc/hosts
cat /etc/hosts.allow
cat /etc/hosts.deny
```

if you are also using Network-Manager in Unity, then also -
```
cat /etc/NetworkManager/NetworkManager.conf
cat /var/lib/NetworkManager.state
```
