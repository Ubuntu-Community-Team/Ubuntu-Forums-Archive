### Post by B!aine on 2009-10-30
I was finally able to "fix" my wireless.  Like the title says, nothing was actually broken this time around.  In 9.04 the wireless on my Toshiba A75-S211 didn't with the encryption I had on my network.  I use WPA-PSK and a hidden SSID and need to use Wicd before, so when I decided to do a fresh install of 9.10 one of the first things I did was to install Wicd.  
    For hours I was unable to get it to work trying just about everything I came across on the internet, and these forums.  Unencrypted it worked, but not with the encryption and hidden SSID turned on.  I ended up deciding to reinstall the default Network Manager and putting back everything I had changed just to see what would happen.  Unencrypted it worked so I reconfigured the security and to my surprise it worked.  
    So as it turns out, if I would have just done the fresh install and configured Network Manager from the beginning it would have saved me several hours of frustration. 
    
    Hopefully this may help someone else.
