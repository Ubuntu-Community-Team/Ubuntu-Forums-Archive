### Post by Tomarq on 2008-10-31
Yes - go to Applications -> Add/Remove.

From the Show dropdown "All available applications", and then enter "vpn connection" into the search box.

Install the flavour of VPN manager you want (e.g. openvpn), and then the VPN connection panel should be more helpful :)
