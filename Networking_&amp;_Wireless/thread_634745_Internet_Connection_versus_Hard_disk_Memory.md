### Post by jpkotta on 2007-12-08
192MB is not much RAM these days.  RAM is cheap, and installation is easy.  It is one of the easiest and most noticable ways to improve performance.  OTOH, internet connections are usually so slow compared to other things in a computer that it doesn't matter (unless you have something like an 8 Mbit connection, which I'm guessing you don't...).  

The acid test is this: if *everything* slows down, it is something else; if the loading of web pages slows down, it is the internet connection.
