### Post by JonnehD on 2010-11-05
I've got a recently fresh install of 10.10 on this computer and I want to share files to my laptop- something I was already doing on this computer with 10.10 before doing a fresh install. 

The problem is, when I go to System > Preferences, the "Personal File Sharing" option isn't even there.  It's there on my laptop, which I upgraded from 10.04 to 10.10, but on this computer with a fresh install of 10.10 it isn't.

I've installed Apache2.2.bin and lib-apache2-mod-dnssd as I've seen them mentioned in other places and remember installing them on my system a while ago, but without the "Personal file Sharing" option in my preferences menu I'm at a loss as to what to do. 

Is there maybe some way to open "Personal File Sharing Preferences" via terminal?  

As it stands, my Desktop (new install) can see my Laptop and I can copy files over from my laptop's public folder, but not the other way around.

EDIT:

After a few minutes more of research I found out that gnome-user-share wasn't installed.  Installed it and it works now.  ^_^
