### Post by SeijiSensei on 2011-09-09
> **petersk said:**
> How do you assign different subnets in Ubuntu to different interfaces automatically at startup?
Kurt

You can't do it if they both use DHCP and talk to the same server.  You'd need to set up static addresses and routes.  You do this by editing /etc/network/interfaces as described [here](https://help.ubuntu.com/10.04/serverguide/C/network-configuration.html), in particular the section on "Static IP Address Assignment".
