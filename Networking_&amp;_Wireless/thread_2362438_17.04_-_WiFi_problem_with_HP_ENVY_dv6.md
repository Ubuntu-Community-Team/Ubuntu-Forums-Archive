### Post by jeremy31 on 2017-05-28
```
sudo sed -i 's/wifi.powersave = 3/wifi.powersave = 2/' /etc/NetworkManager/conf.d/default-wifi-powersave-on.conf
systemctl restart network-manager.service
```

This will disable wifi power management and it might help, you might want to change wifi encryption on the router to WPA2 only as it seems it has WEP or TKIP in use now and that usually creates issues

### Post by jeremy31 on 2017-05-28
I don't know what Kali uses for network management but it might be more tolerant of WEP/TKIP

The USB adapter should work after
```
sudo -H gedit /etc/NetworkManager/NetworkManager.conf
```
Add this
```
[device]
wifi.scan-rand-mac-address=no
```
Save and exit the text editor and
```
systemctl restart network-manager.service
```

```
$ sudo gedit /etc/default/grub
```

```
$ GRUB_CMDLINE_LINUX=""
```
To
```
$ GRUB_CMDLINE_LINUX="net.ifnames=0 biosdevname=0"

Then
```
$ sudo update-grub
$ sudo reboot
