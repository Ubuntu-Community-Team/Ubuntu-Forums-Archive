### Post by zero-9376 on 2007-10-12
network manager is probably what you want. its installed by default on ubuntu 7.04

if you have it installed there should be a network icon on your panel which you simply left click on and select the wireless network you wish to use.

if its not there then you should be able to start it by pressing alt-f2 to bring up the run dialog and enter nm-applet
