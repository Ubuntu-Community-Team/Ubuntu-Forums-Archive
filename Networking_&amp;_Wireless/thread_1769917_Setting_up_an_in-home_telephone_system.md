### Post by sandman887 on 2011-05-28
I have a bit of an odd situation. I want to set up a telephone system in my house. I don't want it to connect to any outside lines. But I still want it to have a dial tone when I pick it up. I really only want to be able to have 3 phone numbers on the phone system: 1 downstairs so I can dial up on my desktop down there to my desktop upstairs, and a phone in another room.

I run Ubuntu on all my computers, and I am wondering if there is some kind of software for Linux that can set up a working phone system, that you dial a phone number on, and it switches you to the phone number you're trying to reach (in my house). 

The problem is, I don't know what kind of software I would need. I have heard about asterisk, but I don't know if it would act as a telephone provider (dial tone, and then switch calls to the correct phone line). 

Does anybody have any experience with this?

### Post by Artemis3 on 2011-05-29
This sounds like a job for: [Asterisk](https://wiki.ubuntu.com/AsteriskOnUbuntu/Current)

Its a GPL PBX that also allows you to connect with physical land lines with the [appropriate hardware](http://www.asterisk.org/hardware).

It is an enterprise ready solution, but I'm sure it can accommodate your needs cheaply ;)
