### Post by GeorgeVita on 2009-08-20
Hi **jrev**,
usually you must be root user or the user dialing must be into dip and dialout group. Try 'sudo gnome-ppp' or 'sudo wvdial' using an **/etc/wvdial.conf** file or try to add user to the groups:```
sudo adduser USERNAME dip
``` ```
sudo adduser USERNAME dialout
``` (USERNAME is 'PC username' and not the provider's username).

'Carrier detected.  Waiting for prompt.' can be bypassed using an extra line to /etc/wvdial.conf file: ```
Stupid mode = Yes
``` (or select 'ignore terminal strings' at gnome-ppp >Setup > Options tab)
