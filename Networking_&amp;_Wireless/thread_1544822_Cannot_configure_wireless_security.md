I am using Ubuntu Lucid Netbook Edition. When I used the Karmic version of the same, I could connect to my university wifi simply by adding it in the wireless tab of Network Connections (System -> Preferences ->) and configuring it. 

In Lucid however I can't use the same wifi because of a peculiar issue. I can create a new connection and add it, but whenever I try to edit the stuff in the "Wireless Security" tab, the button to commit changes ("Apply") turns grey and I am forced to cancel all changes - I can't get to configure my wifi. ### Post by akshaypande1989 on 2010-08-04
Problem solved! :)

I signed in my Ubuntu Netbook Edition session (earlier it used to be GNOME sessions, where the network indicator wasn't visible on the IndicatorApplet in the taskbar.) 

In a Netbook session, I simply clicked the now visible network icon in the IndicatorApplet which could detect the available wifi connections. I then chose mine and configured it in the next window. Now the "Connect" button didn't grey out after I filled the appropriate data. It also created this wireless connection!

Thank you!
