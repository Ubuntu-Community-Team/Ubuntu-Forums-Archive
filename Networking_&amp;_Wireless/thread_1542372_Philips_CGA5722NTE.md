### Post by gordintoronto on 2010-07-30
Most routers are set up using a web browser. Initially, you would use an Ethernet cable between your computer and the router, start a browser and type the Router's address into the address bar on the browser. It is usually 192.168.1.1

The router usually sends a logon screen, and you log on based on the information in the manual. Sometimes the "factory default" is userid admin with password admin.

If the router was used by someone else, it might be a good idea to reset it to factory defaults. The manual should tell you how. On one router I have, you poke a pin in a hole in the router, then turn the router on, and keep the pin there for several seconds.

Because you use a browser to set up the router, it doesn't matter what OS your computer is using.
