### Post by kleeman on 2006-06-15
The just released 2.6.15-25 kernel solved two networking problems I was having. I have a Yukon II nic which is handled by the sky2 driver. Previously a remote freenx connection to this machine (it is the nxserver) would not work. Also the gnome-cups-tool on the remote machine showed the sky2 machine printers but printing froze when used.

ALL FIXED WITH 2.6.15-25.

Kudos and thanks to Ben Collins and the other kernel hackers. Shuttleworth needs to give them more help (another Ben) IMHO. 

Many kernel bugs were only fixed very close to Dapper release meaning that others were not found because fewer test machines were available.

I hope Mark is listening......
