Do NOT buy an ADSL modem. Make sure you buy an ADSL ROUTER with integrated modem that has an ethernet port and NOT USB !

Any brand should work as far as i can tell.

From what I can tell your laptop uses Broadcom AirForce One G54 BCM4318 chipset which means you will have to use ndiswrapper.
