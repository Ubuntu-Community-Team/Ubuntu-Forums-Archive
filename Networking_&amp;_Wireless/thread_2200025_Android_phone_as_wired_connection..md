### Post by Tomtompiper on 2014-01-17
I haven't seen this on any post, and it was very useful. I was given an Asus 1200 USB WiFi dongle as a birthday present and it wasn't recognised by Ubuntu. All of the ports on my router are used and the computer was too messy to move easily. I was considering using my phone to try to troubleshoot the problem but I am near my data limit and don't want a huge bill. I wondered if the phone would work as a USB tether with the WiFi on and data turned off. It did, Ubuntu saw it as a wired connection and I was able to find out that the dongle was not compatible. I plan to keep on using the phone as my connection and will swap the dongle for a graphic graphics card.  So if you are stuck with a WiFi problem use this to get connected while you sort it out.
