### Post by bukwirm on 2007-05-10
Make sure, in the DNS Server options in your router's configuration, that the "Use these DNS Servers" option is set (instead of the "Get Automatically from ISP" option).

If that doesn't work, try [disabling IPv6]("http://ubuntuforums.org/showthread.php?t=6841&highlight=disable+ipv6").
