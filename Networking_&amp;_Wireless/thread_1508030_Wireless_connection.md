### Post by scraginagubu on 2010-06-12
Hi,
I have had a problem where when I logged out of an "old" wireless network and went somewhere and tried to login to a new wireless network, it would recognize that the new network was there, but still try to log in to the old network.
 
The only solution I found was to go in to System - Network Connections - Wireless and delete the old network connection, and then it suddenly would recognize the new one and log into it properly.
 
That worked fine for a while. However, I developed a problem in that if I deleted all the wireless network connections, it unloaded the Hardware Driver as well, and would not reload it, so in order to make the solution work, you have to keep at least one wireless connection registered in the System - Network Connections - Wireless section
 
Wish I could give you a better solution, but...hey...it works...:-)
