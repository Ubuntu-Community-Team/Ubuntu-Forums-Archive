### Post by bscbrit on 2006-06-21
First, please don't try to use 'clever' abbreviations in your questions - many of us find them very hard to read.  And many others, for whom English is not their first language, will find it almost impossible to understand. :wink: 

I do not know how much you know about networking so I will give you a quick summary and, if you have any questions, you can come back and ask for more information.

On the computer that has the connection to the broadband cable modem, you will have to have an ethernet socket spare.  You might already have one but, if not, then buy/obtain a cheap Network Interface Card (NIC) which should cost as little as £5/8$ or so.  Once it is installed and set up, you can install a software package called 'firestarter' which will allow you to share the connection easily.  If you just want to connect to a single computer then you can use an ethernet 'cross-over' cable direct to the ethernet socket on the second machine.  The second machine must be given the address of the NIC in the first machine as its 'Gateway'.

If you want to share the internet connection with more than one additional computer then you will need a standard ethernet cable which can feed a switch or router.  That will allow you to connect a number of computers via the switch / router to the first computer and then to the internet. 

It sounds difficult but it isn't really.  If you take things one step at a time it can be done quite simply.  Let us know exactly what you want to do and we can give you more precise instructions.  Good luck.
