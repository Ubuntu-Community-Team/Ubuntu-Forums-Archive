### Post by xnx on 2010-04-06
Hello.
I'm a newbie.  I used to be under linux all the time 10 years ago.  I moved to other systems and now i'm back trying out Xubuntu 9.10.

I feared that the 3g internet stick i had would be complicated to set up.  Didn't find a lot of information on the internet.

Here is how i got it to work.  It was the easiest of things.

I'm using Xubuntu from a live usb key install (installed on another usb key, not the rogers one).  Working out all the little details before deciding to install it.

I booted with the rogers internet stick in the computer, it is a ZTE 668.  It was configured before under mac os x and windows.  I don't know if this changes something.

I clicked on the network connection manager (NetworkManager Applet 0.7.996) and created a mobile broadband connection.  It detected the modem without any work from my part.  I then selected Canada as the country then Rogers as the provider.  Didn't change anything else.

First connection didn't work.  I looked at the connection information and no dns were shown.

Went back and edited connection settings, bascially selecting Canada and Rogers again.

Tried to connect again.  Worked!

Wow.  This is impressive.  It works very fast and very well.

This ain't the linux i had 10 years ago.  It's quite polished now!

Only remaining big problem is the sound.  I'll post in the proper forum if the need arises.

Why did i post this even if it wasn't a question ?  To share the experience.  In case some other people are wondering and don't feel like spending a few hours to install Xubuntu then realizing their internet doesn't work.

Yay.
