### Post by Iowan on 2010-06-24
The (wired) interface appears to be getting an address - hopefully a different subnet than the wireless (check **ifconfig -a**). Another possible problem might be routing - check **route -n** - the default route is usually on the bottom (line contains UG). Also check contents of */etc/resolv.conf* - it should have the address of your nameserver. You can double-check by pinging the internet by IP address (74.125.95.147).
