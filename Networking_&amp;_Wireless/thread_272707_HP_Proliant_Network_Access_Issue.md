I have installed Xubuntu (6.06) on two Proliant DL360's and am having network issues.  The install picks up the NIC's fine and allows for manual setup.  After the install in complete, the systems have access to the internet and can ping any computer on the network.  

The tricky part is that computers not on the same switch as these two systems, can not access them (ping/ssh/http).  I have tried several different switches and the same thing still happens, no connectivity.  If I go to the one of the Proliant systems and ping a client not on the switch, then that client can access the server for a limited time.

As I have installed Xubuntu on Dell systems without issue, I am convinced it is an HP Hardware problem.  I also had one of our network guys dig into the Cisco switch they are connected to and he came up empty.  

Has anyone run into this before?
