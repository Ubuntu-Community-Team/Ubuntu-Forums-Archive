### Post by Blind-Summit on 2006-06-20
in my experience - no

I had to use the cutter tool to extract the firmware (which was my problem) but you need to get hold of the package. The only way around this is to get someone to send you the firmware files to go in the /lib/ folder (I think) or you would need to download each deb file under windows to install the firmware cutter. Remember that each file has dependencies (sp) so this could mean getting a whole load of files (not good!)

Try and go to a friends house with a lan or a PC with shared internet and use a crossover network cable. You really need a wired lan to get the working. Sorry

I know how bad it is - took me weeks to first work out that it was firmware and not a driver issue! Then I had to plug my router in to one room with a telephone wire, and then back to my room with a network cable! Just for this one damn file!

Good luck
