### Post by lisati on 2009-04-18
On the top panel there should be a network manager icon - if you're connected via ethernet it will look like two little computers, and when you're connected by wireless only it will look like a group of little bars. If you click on it you should be given a list of networks that Ubuntu currently recognizes.
