### Post by sanderd17 on 2010-09-20
Android does not support ad-hoc networks. You'll have to root your android to do that. If you have an infrastructure network, it behaves the same as internet, so yes, files can be shared but you'll need file viewers.

A lot of office apps are really heavy and slow on android (at least on smartphones, didn't try any on a tablet). They are also limited in functions. The best files to share would be html files I guess (or plain text).

So in fact, it's like setting up your own little internet.
