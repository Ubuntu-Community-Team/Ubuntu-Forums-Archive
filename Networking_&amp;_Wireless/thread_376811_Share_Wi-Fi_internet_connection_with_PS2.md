### Post by pjman on 2007-03-14
This was actually pretty easy to setup. I used Firestarter which has a straight forward wizard when you first run it.

[http://www.fs-security.com/docs/connection-sharing.php](http://www.fs-security.com/docs/connection-sharing.php)

The only problem I had was that my client (PS2) needed to have the same DNS setting I had for my wireless internet connection in Ubuntu. When I was testing this using ICS with my Windows XP laptop I had to have my local LAN IP set as DNS (LAN IP of my laptop). Other than that it worked like a charm!
