### Post by nixscripter on 2008-02-21
First of all, I am assuming:

1. Linux box has wireless and ethernet.
2. Windows box has ethernet only.

Generally, for what you stated you want to do, what you will need is this:

1. Windows machine connects to cable box via ethernet. You have that already.
2. Linux machine connects to windows box via wireless. You will need a wireless card for the windows machine.

In this setup, the windows machine will have to act as a bridge. I don't have any suggestions on how to configure that, unfortunately. That will require research, and I don't even know if it's possible if you have XP Home Edition (might be too advanced).
