### Post by ajgreeny on 2020-08-05
I am not completely sure exactly what you're asking, nor why, but I will be  very surprised if there is any loss of performance for either machine, more probable is the opposite.

The wifi connected machine could get slightly higher network speeds as the wifi will no longer be shared between the two machines, though in normal use I doubt you would notice any difference.
The cable ethernet connected machine will most likely get faster network speeds as cable connections are generally much more reliable, and less likely to suffer from all the interferences that can happen with a wireless connection.  This is why servers are recommended to use ethernet, not wireless.

If cable connection is in use I believe you will temporarily lose wireless activity on that macine only (not totally sure about that) but even if that is so it should not matter as the cable ethernet should be even better and probably faster than wireless.  The wireless connected machine will not lose its connection just because another one on the network is cable connected, so there is no problem there; this is how my network is set up all the time, my laptop using wireless and my desktop using cable ethernet.
