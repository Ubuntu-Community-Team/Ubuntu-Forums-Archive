### Post by cecilieaux on 2014-10-12
Without getting an answer I managed to set it up in Linux. This is how I did it:

The file manager saw it under Networks. There were several server icons (a computer box with cable through it) some friendlier than others. One that said Seagate Central Shared Storage allowed me in and there was a link (a file called "Manage Seagate Central Seagate.url"). This got me into the setup screen through Firefox. I named the unit, gave it the name of a user (turns out to be the default superuser) and an email.

I still have loads of questions about the 6 (count 'em) connection options via the file manager I now get, including the afp:// (???), smb:// (3 flavors), [ftp://,](ftp://,) sftp:// protocols. Plus I managed to get into the filesystem of the device and discover its very Linux directory structure.
