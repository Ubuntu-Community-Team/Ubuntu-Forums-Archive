### Post by AlexDudko on 2011-09-18
Where did you do
> iwlist scan
Do it not on the netbook, but on an other device.
If it's seen, then I think it won't be difficult to connect to it.

P.S. As far as I understood ppp0 interface is facing Internet.
To make your netbook work as a router run these commands one by one as root:
> iptables --table nat --append POSTROUTING --out-interface ppp0 -j MASQUERADE
iptables --append FORWARD --in-interface wlan0 -j ACCEPT
iptables-save
If there isn't a problem devices, connected to **qwerty** should have Internet access.
