### Post by thatguruguy on 2012-06-07
Sorry, wrong info.

open a terminal and type:
```
mupen64plus
```
... with the name of some rom that will run on it.

So, if you were going to run a rom named "Mario64.n64", you'd go to wherever that rom is stored, and type:
```
mupen64plus Mario64.n64
```
To get more information about the program, you can type:
```
man mupen64plus
```
