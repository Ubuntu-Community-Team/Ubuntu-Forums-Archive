### Post by burek on 2007-01-26
How to play Captive Mindscape with Linux 
[http://en.wikipedia.org/wiki/Captive](http://en.wikipedia.org/wiki/Captive)
[IMG]http://upload.wikimedia.org/wikipedia/en/f/f0/Captive-InAction.png[/IMG]
  
1/ You just need an emulator, but the problem is that hatari is not too great for this game
so :
linux can offer you the possibility of downloading and having this program, that s free emulator program : 
[http://www.zophar.net/unix/Files/xsteem_v3_2-i486.tar.gz](http://www.zophar.net/unix/Files/xsteem_v3_2-i486.tar.gz)
you unpack it with for instance with file-roller
 then 
```
./steem
```
  
2/ then you can run your favourite game !
(or dumped to msa file : you need the original file floppy disk that you bought the game, You should have the original game,.in that case you may run  med_097.st  with tos 1.06 works  )
   
3/ steem offer you too fullscreen (just press on PAUSE key)
  
Enjoy this very famous Game !!! :D
