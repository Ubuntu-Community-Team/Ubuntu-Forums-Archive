### Post by LillyDragon on 2013-10-21
Project64 usually works under Wine. I've actually never seen it break for as many Wine versions as I've went through, the worst issue being sound. (Which was PulseAudio related. Pulse is horrible on my hardware, and crackles with extensive sound playback even for native Linux applications, so I disabled it for ALSA.) You shouldn't need any extra libraries to get it working. Were you seeing any error messages?

Also, there is Mupen64 Plus in Ubuntu's Software Center, and it works great on 64-bit, but the UI it was packaged with for previous LTS's has been gutted, leaving you to use only the command line. It's very usable despite that, but the learning curve is annoying, and different files you need to configure are scattered everywhere.
