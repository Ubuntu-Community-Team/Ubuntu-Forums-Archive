### Post by WienerWuerstel on 2012-01-13
It depends on which System you want to emulate but there is an Emulator for almost every System available for Linux anyways. Normally I would tell you to use a Search Engine for that but I'm in a good mood Today so I'll make a List for you. Maybe that List can be used for a Sticky in the near Future too.

Nintendo Entertainment System: [fceux]("http://fceux.com/web/home.html"), [halfnes]("http://code.google.com/p/halfnes/"), [nestopia]("http://nestopia.sourceforge.net/"), [fakenes]("http://fakenes.sourceforge.net/svn/")
Super Nintendo: [zsnes]("http://www.zsnes.com/"), [snes9x]("http://snes9x.ipherswipsite.com/"), [bsnes]("http://byuu.org/bsnes/")
Nintendo 64: [mupen64plus]("http://code.google.com/p/mupen64plus/")
Gamecube and Wii: [Dolphin]("http://code.google.com/p/dolphin-emu/")
Gameboy (Color): [Gambatte]("http://sourceforge.net/projects/gambatte/")
Gameboy Advance: [vba-m]("http://sourceforge.net/projects/vbam/"), [VisualBoyAdvance]("http://vba.ngemu.com/")
Nintendo DS: [desmume]("http://desmume.org/")
Sega Master System: [meka]("http://www.smspower.org/meka/"), [kega fusion]("http://www.eidolons-inn.net/tiki-index.php?page=Kega"), [osmose]("http://bcz.asterope.fr/")
Sega Genesis: [kega fusion]("http://www.eidolons-inn.net/tiki-index.php?page=Kega"), [gens/gs]("http://segaretro.org/Gens/GS")
Sega Saturn: [yabause]("http://yabause.org/")
Dreamcast: [lxdream]("http://www.lxdream.org/news/")
Sega Game Gear: [osmose]("http://bcz.asterope.fr/"), [meka]("http://www.smspower.org/meka/"), [kega fusion]("http://www.eidolons-inn.net/tiki-index.php?page=Kega")
Playstation: [pcsxr]("http://pcsxr.codeplex.com/"), [epsxe]("http://www.epsxe.com/")
Playstation 2: [pcsx2]("http://pcsx2.net/")
Playstation Portable: [jpcsp]("http://jpcsp.org/")
Atari 2600: [stella]("http://stella.sourceforge.net/")
Atari 5200: [kat5200]("http://kat5200.jillybunch.com/"), [atari++]("http://www.xl-project.com/")
Turbografx16: [hu-go!]("http://www.zeograd.com/")
Commodore 64: [vice]("http://www.viceteam.org/")
PC: [dosbox]("http://www.dosbox.com/"), [scummvm]("http://www.scummvm.org/")
Arcade: [sdlmame]("http://sdlmame.wallyweek.org/")
Multi Machine Emulator: [mess]("http://www.mess.org/"), [mednafen]("http://mednafen.sourceforge.net/")


Let me know if I missed any important System.
