### Post by Toz on 2014-01-14
Mupen64plus is in the repositories, why not just install it from there? You don't mention which version of Ubuntu you are using, but even Precise (12.04) has a newer version in the repositories than what is being displayed in your screenshot.

To install the application from the repositories, search for it in the Software Centre or run the following command in a terminal window:
```
sudo apt-get install mupen64plus
```

### Post by deadflowr on 2014-01-14
You should delete what you've extracted and install the version from the repos.
The one thing of note is that mupen64plus, is a command line program.
It wasn't always this way. The older version1.5 had a really nice frontend, but the developers dropped it.:confused:

It may take some time to get used to the way it works, but when you do, it runs beautifully.

On the other hand, there are several 3rd party frontends
[https://code.google.com/p/mupen64plus/wiki/ThirdPartyPlugins](https://code.google.com/p/mupen64plus/wiki/ThirdPartyPlugins)
