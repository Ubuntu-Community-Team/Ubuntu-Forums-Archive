### Post by LillyDragon on 2012-06-09
It is quite odd how it's all scatter about, but they're all important I do believe. You can go with the dummy package that installs each in one sitting.

And just encase you're out of the loop on this detail, it's a terminal-based app with no front-end GUI. (I do remember a GUI being included with a previous version of Mupen64 on Hardy Heron, so that's why I mentioned it.) _[Section 4 of the readme has all the commands you'd want while making launchers for each of your games.]("http://code.google.com/p/mupen64plus/wiki/README")_ Personally, it's only a minor inconvenience, but it does make configuration a bit confusing.
