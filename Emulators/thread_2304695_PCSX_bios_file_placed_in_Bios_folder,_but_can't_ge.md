### Post by deadflowr on 2015-11-29
While your bios file is in the bios folder, the configuration page shows you're using the default simulated bios, from inside the Downloads folder.
You'll need to re-navigate that configuration section.

Go to the bar that says Downloads and toggle it.
At the bottom of the toggle should be an option for other, click on it.
Since you are in the Downloads folder chances are you'll need to go up to your main folder (the one that has the downloads and documents and pictures folders; the one with your username for it)
And also, since you're in the Downloads folder you might need to toggle the show hidden folders, so you can see the .pcsx folder.( just right click in the main window area to get the show hidden option)

After that, simply navigate to the .pcsx/bios folder and click open.
Now you should see the bios file you put there as the bios file selected.

hope it helps.

As long as the bios file you want is in whatever folder is listed in the bios and plugins configuration settings page.
It won't matter if you copy or duplicate the file over and over.

So while it works best for organizational purposes to have it the .pcsx/bios folder. It really does not matter if it is.
