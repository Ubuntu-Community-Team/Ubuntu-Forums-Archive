### Post by Dennis N on 2013-12-14
> **michael-piziak said:**
> I have Ubuntu 11.x
I installed the Atari 800 emulator, using Ubuntu Software Center; however, I can't find where the download put the emulator.

It is in /usr/bin, but you would start it from the terminal by typing **atari800** as a menu entry may not be created automatically. Once started, F1 get you the menu. Nothing is obvious here. try in terminal: **info atari800**. Once set up (and you figure out how it works) it runs well. Will run maximized.

The Windows version, **Atari800win Plus** runs well under Wine and can also be used. Has mouse-driven menus. Size of window is limited, though.
