### Post by General_Faliure on 2013-11-02
Hello.

My arcade cab is currently based on a socket 754 Athlon 64 3200, one gig of ram and a Nvidia Geforce 6800 (it also worked fine withe a Geforce 5200)
I can play all but the heavier mame games on this, i also have the Mess emulator, Daphne for the laserdisk games.
There is also FS-uae for Amiga, Mupen64Plus for nintendo 64 (some games are a bit too much for this system)
For the best result you might have to add some PPA's. (see this thread for some links: [http://forum.arcadecontrols.com/index.php/topic,114917.msg1220873.html#msg1220873](http://forum.arcadecontrols.com/index.php/topic,114917.msg1220873.html#msg1220873))
I am currently setting up a newer mainboard for my cab: a core 2 duo 8400 (@3.6 Gh) 4 gig of ram, and a Geforce 9500 and Lubuntu, which works better for the newer emulators and should last me for some years to come.
Here is a link to my cab: [https://picasaweb.google.com/102745845720316376753/ArcadeCab?authuser=0&feat=directlink](https://picasaweb.google.com/102745845720316376753/ArcadeCab?authuser=0&feat=directlink)

Have fun.

### Post by deadflowr on 2013-11-02
If you have old playstation one games, PCSX is pretty good.
Though, like any other emulator, it is only as good as the system it's running on.

Mupen64plus is good, though the version for 12.04 upgraded and they took away the default gui, so it uses the cli instead.
There are some guis, but I've found none as good as what they had before.

I've also run Desmume for nintendoDS games, not too bad, though I've never gotten the mic to function right.

I'm sure there are a couple others I've used but these are the three I seem to use the most.

OOH, there's also visualboy advanced for gameboy games.

And of course dosbox, for your old school old school gaming.

So these are the few I run somewhat regularly
Never ran mame or the like so don't know how well they run.
