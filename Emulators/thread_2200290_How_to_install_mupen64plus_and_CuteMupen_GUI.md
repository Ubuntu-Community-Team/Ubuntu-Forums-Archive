### Post by deadflowr on 2014-01-18
Did you try installing cutemupen from here
[http://www.playdeb.net/software/cutemupen](http://www.playdeb.net/software/cutemupen)

and what have you done, efforts-wise, in terms of trying things out?

But if you want to try something, which seems a little hard at first, simply run
```
sudo apt-get install mupen64plus
```
and install mupen64plus from the repos.
Then run mupen64plus -h
to get an overview of the commands to run games.
Basically though, the common command would be like any other command to run something.
```
mupen64plus path-to-game-file
```
