### Post by adec2 on 2015-10-18
Use Snes9x-gtk much better

[https://code.google.com/p/snes9x-gtk/downloads/list](https://code.google.com/p/snes9x-gtk/downloads/list)

As for mess that is now incorporated into mame so to run snes game (although snes9x is much better) you'd have to find the snes bios zip for mess then put it in the home/<yourname>/.mame/roms folder then execute the snes emulation with:

mame snes -cart "/path to your snes roms.smc"

Snes9x-gtk is much better though
