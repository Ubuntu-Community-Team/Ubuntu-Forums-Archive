### Post by Tea4all on 2008-11-27
Try getlibs. Download the package from [ here.]("http://www.boundlesssupremacy.com/Cappy/getlibs/getlibs-all.deb") Then install. Next open a terminal and type "sudo getlibs /usr/bin/doom3" Try doing this 3 times. Then try to run the game.
