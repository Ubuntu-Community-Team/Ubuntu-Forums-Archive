### Post by mcloser on 2009-04-18
Hello everyone, just wanted to tell the world that I finally got Xubu running on my home built AMD x64.  And the most amazing part is I got it to dual boot with Vista.

The secret to the dual boot ( at least what finally worked for me ) is to load the disk while in Vista and select install boot files before rebooting.  This has the magic of setting up the GRUB for the restart.  I had tried many things in the past to get Xubu to work with XP or Vista, super grub - everything I could find.

And now it just works ! 

like I said above it is a home built AMD 9850 Blk on a Gigabyte Mobo - MA78GM-uS2H, 4 gig Kingston 800 Mhz ram and WD640 HDD. went together like butter and booted up first time.  this was my first home built and I was nervous.

I was hoping to find an easy answer to my dual boot problem here but it eluded me, so hopefully if some lost soul finds this and it helps, then great.

not sure if the difference with xubu dual boot is 9.04 beta , but it works and I'm posting this on it now.

good luck to all brave hearts.:o
