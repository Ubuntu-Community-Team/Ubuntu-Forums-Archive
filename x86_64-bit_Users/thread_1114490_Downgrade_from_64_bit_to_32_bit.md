### Post by ptn107 on 2009-04-02
Switching between architectures requires a clean installation. +1 to you if you kept your /home on a separate partition so at least that stuff can be saved.

### Post by ynyus75 on 2009-04-02
So can I uninstall my 64 Bit Ubuntu which sits along Vista with Dual boot option without disturbing Vista. I.e. Can I restore Vista after I unstall ubuntu 64 bit?

---

### Post by ptn107 on 2009-04-02
You won't uninstall Ubuntu per se.  You will install Ubuntu [32-bit] and point it to the same partitions Ubuntu 64-bit was on.  Vista will remain untouched.
