I downloaded Beta CN896/P4M900 Linux Graphics Driver via-chrome9.83.40558 for my Ubuntu 8.04 running 24-24 kernel on P4M900-M7 SE.

I get a strange login asking me to locate the driver, and then when I do login, my 1280 x 960 Philips monitor only has resolution up to 800 x 600. Please help. I really would like to access the Directx9 capabilities of the IGP Chipset.

This is a known issue according to release notes:

*Default xorg.conf doesn't set display mode in order to let the driver to choose the best one according to the DDC reading of each individual monitor. However, for old monitors that do not have DDC, a Display subsection needs to be added to the xorg.conf file manually to set a mode else it won't display.*
