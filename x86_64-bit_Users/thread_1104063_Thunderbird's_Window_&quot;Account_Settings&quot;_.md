Hello everybody!

I have a very peculiar problem in Thunderbird. I installed it and everything seemed nice. When I run it for a first time it asked me to create an account so I did it. The account was created and then I wanted to edit its settings. When I open the Account Settings window ... I can edit nothing as the window is blurred. I can't see/read anything and I need this to finish editing my account so it can work properly. All other windows in Thunderbird work fine. I tried to reinstall the program several times with no success. Got any ideas what's wrong? I really appreciate your help.

### Post by julian.b on 2009-03-23
OK, I found a solution, at least a kinda.
What I noticed is that sometimes windows in KDE 4.2 are blurry, but generally it goes away in about a second or two.
In thunderbird it did not go away itself, so I couldn't see anything.
Remedy: play with the Account Settings window i.e. resize it/maximize it. Everything should be visible then.
