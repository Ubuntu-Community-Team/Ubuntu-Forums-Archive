### Post by Arup on 2008-11-16
Gmail allows POP or IMAP and you can retrieve mail from Gmail via any mail client, I have been doing so for years with Opera's excellent M2 mail client.

To retrieve mail from web based Hotmail and Yahoo, try Freepops which is in the repos, it acts as a proxy and retrieves mail via your client. You just put the login in Freepops and set the pop server to localhost.
