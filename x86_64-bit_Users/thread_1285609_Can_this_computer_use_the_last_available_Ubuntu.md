### Post by Lukios on 2009-10-07
Let me tell you something ironic. I used to be a Windows user and often times when I would go to a Windows based forums you would see people who would post their system specs. Generally these specs were top notch as they had to be in order to run the latest Windows. Now coming over to Linux I have noticed in the last year, that people who post their specs generally are those who don't even have a dual core, and often times are running nothing more then a P3 or less. It's like they are sending the message "I bet you can't get vista to run on this, but I can get the latest ubuntu will all the eye candy I want on my computer" So to answer your question, yes, you will be able to run the latest ubuntu on your setup, and with it being a dual core, you can most definitely run the 64 bit version as well.
