### Post by jespdj on 2009-05-29
I guess you are using Sun Java 6 update 13, which is the version of Sun Java 6 that's in the Jaunty repository.

It's really nice that Sun finally has a 64-bit browser plug-in, so that Java should not be an issue anymore for 64-bit users.

However, as far as I know, that browser plug-in is only for Firefox. Opera doesn't use it and doesn't need it, because it uses the Java runtime environment that's on your machine directly (without needing a plug-in). This should have worked on earlier Ubuntu versions with earlier versions of Java as well as it does on Jaunty (although I don't know if there were any specific problems with it in the past, I am not an Opera user).
