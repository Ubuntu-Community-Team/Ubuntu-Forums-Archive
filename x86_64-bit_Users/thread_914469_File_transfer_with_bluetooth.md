I never tried a dongle or cradle for my bluetooth devices. I did discover that blueman is a great GUI tool for pairing up bluetooth devices. It's not in synaptic, but you can find it on the net and download the .deb file. Install it with gdebi.

Once you have the device paired up all you have to do is double-click on it in blueman and a file browser window will pop up. Then you can drag and drop files as you please.

### Post by Thelasko on 2008-09-10
Ubuntu has a pretty nice GUI for bluetooth support.  I think you need the bluez-utils package installed.  If that's installed and it still doesn't work, you might need obexftp.  I think obexftp was all I needed on my machine, but I can't remember.  (I'm sure obex was in the name)
