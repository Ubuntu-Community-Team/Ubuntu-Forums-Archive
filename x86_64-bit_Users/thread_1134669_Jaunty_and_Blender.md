### Post by Thelasko on 2009-04-24
If you installed it from the repositories, it should automatically install the 64-bit version.  Yes, there is a [64-bit version in the repositories.]("http://packages.ubuntu.com/jaunty/blender")

Try to update your system and see if that fixes it.  It may be a bug that the developers fixed before releasing the final version of Jaunty.

P.S. [Here is the list of know bugs in Blender.]("https://launchpad.net/ubuntu/+source/blender/+bugs")
