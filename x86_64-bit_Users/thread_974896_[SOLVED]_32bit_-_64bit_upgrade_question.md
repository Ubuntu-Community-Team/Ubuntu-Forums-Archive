### Post by dabang on 2008-11-08
As far as I know, there is no way to "update" 32bit to 64bit Linux. You have to reinstall, as this will be a "different" OS. However, you can backup the contents of your home directory and recover it later on.

---

### Post by m_l17 on 2008-11-08
You'll have to do a fresh install of 64 bit version. If you have a separate /home partition, you can use it on a reinstall. Just use the same username, create the mountpoint and don't format it. Ubuntu will use your old /home dir with everything intact. But to be on the save side make a backup of you /home dir.
