### Post by insane_alien on 2009-11-16
depends how memory intensive your workload is.

if your swap is heavily utilized then yes, you will see a big difference(mine is fairly regularly but my RAM is maxed out so i'm stuck till i get a new PC)

if your swap is barely touched at all then you will see no difference as you already have plenty of RAM.

upgrading would allow you to have more virtual machines active as well.

### Post by 3Miro on 2009-11-16
Linux uses all the free ram as hard drive cash. Thus if you are doing hard drive demanding jobs (such as miving large files around), more RAM helps.

Otherwise, if you are using 3GB out of 4GB you will see no significant difference.
