### Post by 3Miro on 2009-04-29
I would stop right there and take a step back. What does all of this stuff do and do I need it.

First I would read carefully on what this anti-virus software does. If it is in the repos, it is not malware, but why would I need antivirus software. I have not done so, but I will see what it does to make sure.

Download managers, tools for compiling and so on ... Do I need that. Am I going to use it. Read and ask yourself that, don't just install something you will never use.

Why do you need Adobe Reader. The default Ubuntu document reader reads pdf files and is hundred times lighter and faster than the windows version of Adobe (and judging by the flash, Linux is not Adobe's strong point). I suppose Adobe Reader would support some exotic stuff (such as videos imbeded into the .pdf), but do you really need that.

If you need all of that software then af course install it, but don't if you are not going to use it, then why would you bother. I saw somewhere a list once, it gave a list of possible software and how to install it or set it up. Then I read through it, picked several things and installed them. Definitely not everything.
