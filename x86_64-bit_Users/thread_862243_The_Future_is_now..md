### Post by phlembob on 2008-07-17
Hi,  the people using AMD 64 bit computers should try a 64 bit OS.  AMD has true 64 bit capability and a 64 bit OS should produce a near double speed computer with the same operating frequency.  This isn't quite the result because system boards haven't made the move to 64bit registers for processing, however your CPU is ready for the next command or has processed the next command waiting for a catcher for the information.

The 64bit processor has out stepped the production of inexpensive chips needed to update the motherboards.  64Bit computers will operate with 32bit programming appended one time.  It is taking 2 programming statements at a time.  Yes, this limit will not allow 16 or 8 bit programs.  The CRC or Parity is messed up with more than one appendage.

64 bit is now and the future.  Other limiting factors such as operating oscillator frequency, and perfect DC supplies will come around with new developments also.  A 64bit programming language is needed also!  This will encompass the average English of most users.

Get familiar now, and you will step into the future with ease.
