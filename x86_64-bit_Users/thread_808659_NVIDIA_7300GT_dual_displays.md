### Post by mitchej123 on 2008-05-29
Take a look at my config file and try adding stuff like 
```
Option         "AddARGBGLXVisuals" "True"
```
in.  I think nvidia's need that to work with compiz.  There are a few other options I threw in and I forget exactly which are needed to get it to work.

You might also try adding 

```
Section "Extensions"
    Option         "Composite" "Enable"
EndSection
```
at the end.
