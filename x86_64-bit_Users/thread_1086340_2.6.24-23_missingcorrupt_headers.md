### Post by clarknova on 2009-03-03
I would try deleting the .deb file from cache, i.e.,
```

sudo rm -rf /var/cache/apt/archives/linux-headers-2.6.24-23_2.6.24-23.48_all.deb

```
Then try installing again. It's a shot in the dark, but I'm wondering if the package was corrupted somewhere in transit.

You could also try purging the linux-headers package and install fresh.
