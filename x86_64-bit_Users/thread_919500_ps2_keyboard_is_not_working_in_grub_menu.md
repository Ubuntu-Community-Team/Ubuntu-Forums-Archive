### Post by IntuitiveNipple on 2008-09-15
I've seen this occur as a result of a motherboard CMOS back-up battery having very low charge. Replacing the motherboard battery solved the issue. There were other symptoms too, such as the timer running 20x as fast usual, so the 5-second GrUB menu pause would be over in a fraction of a second :)
