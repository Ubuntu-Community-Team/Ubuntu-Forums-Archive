### Post by jocko on 2009-03-18
The kernel (package linux-image-2.6.24-23-generic) in the hardy repos are at version 2.6.24-23.48.
As long as you have "hardy-updates" and "hardy-security" activated in your sources.list (check in System --> Administration --> Software sources under the "Updates" tab) you should have that version, irregardless of which architecture you run.

Maybe you have removed some meta package?
Make sure you have the package "linux-image" installed to keep the kernel updated, for the rest of the system ensure that the packages "ubuntu-desktop", "ubuntu-standard" and "ubuntu-minimal" are installed.
