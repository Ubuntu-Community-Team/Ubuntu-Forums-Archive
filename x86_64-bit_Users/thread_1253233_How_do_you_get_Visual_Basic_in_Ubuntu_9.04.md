### Post by yabbadabbadont on 2009-08-29
Install vmware, virtualbox, qemu, or some other virtualization software.  (virtualbox is probably the easiest)  Then install your legal copy of Windows inside it.  Then install your legal copy of MSVS 2008 inside of that.

### Post by PatrickVogeli on 2009-08-30
you can't install Windows applications in Linux, at least not natively. Some apps work using wine, but that's not optimal. You can also install Virtualbox and virtualize windows.

But if you installed ubuntu thinking you would be able to install all of your windows apps, you're wrong. Linux is not windows, read this [http://linux.oneandoneis2.org/LNW.htm](http://linux.oneandoneis2.org/LNW.htm)
