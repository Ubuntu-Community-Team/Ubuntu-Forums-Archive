### Post by raven128 on 2009-06-19
solved this. found solution on one of russian ubuntu forums.
there is a beta version of Adobe Flash Player 10 for Linux x64, it could be found at [Adobe Labs website]("http://labs.adobe.com/downloads/flashplayer10.html"). 

For Firefox solution is:
1. Uninstall flashplugin-nonfree package from your system via Synaptic or "apt-get" command. Ensure that there is no other such plugins for Firefox left on your system.
2. Create a folder by name "plugins" in a "~/.mozilla".
3. Unpack archive with plugin you've downloaded from Adobe Labs website to that location.
4. Restart Firefox. After that on "Plugins" tab in "Add-ons" dialog you should see "Shockwave Flash" item. 

this tested by myself on Ubuntu 8.10 x64 & FireFox 3.0.11.
