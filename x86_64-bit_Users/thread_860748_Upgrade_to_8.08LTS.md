### Post by philinux on 2008-07-17
Ok.

I'm assuming you only got the one hard drive. Forget the OS thats running on it for a moment.

Whats happens when it dies. **Not if when**.

You loose all your stuff. eg photos, docs etc etc.

Wise precaution against this would be to buy an external hard drive or even a usb stick and copy the contents of home to it. Or just burn a DVD.
I've got a usb and dvd backups.

This is very easy, not sys admin and necessary. I worked as an admin and I'll tell you that when hard drives fail, usually it's when the user has not done a backup for ages.

Hope this helps.
