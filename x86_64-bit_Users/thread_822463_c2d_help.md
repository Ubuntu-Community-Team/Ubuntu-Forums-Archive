### Post by jespdj on 2008-06-09

Installing 32-bit Adobe Flash is not different nor more difficult on 64-bit Ubuntu than installing it on 32-bit Ubuntu. You just need to install the package **flashplugin-nonfree** - that's all. No special steps needed. (On 64-bit Ubuntu 8.04, at least).
