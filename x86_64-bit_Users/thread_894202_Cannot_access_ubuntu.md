### Post by VeeDubb on 2008-08-20
A live CD should work if you have two separate drives.  Boot the live CD in your non-CD/DVD burning drive, and use whatever burning app is on the live CD to burn your files off to disk.

I don't know for sure about doing this with current live CD's, because the last time I had to do this was about 4 years ago with knoppix.  I'm sure if you google linux live CD data recovery backup you'll get all sorts of hits.

If you only have a single drive, you'll need to have a UBS hard drive to copy them off to, or use a pen-drive live linux distribution.  As far as permissions, I know that there are a few live CD's out there that either have open read permission or can be set up that way, for just this sort of situation.

Best of luck to you.


[tough love] I'm actually working on starting my own wedding photography business, so I certainly understand being afraid of losing images.  But, and I'll only say this once, if it is important data, do NOT expect it to be safe in magnetic storage without a backup.  

Hard drives fail.

Flash drives fail.

Houses catch on fire.

Lightening does hit power lines, and more surge protectors really won't save you.

External hard drives are very cheap these days.  I picked up a 250GB external for only $100 on sale at BB a couple months ago.  And DVD media is so cheap these days, that there is no excuse to have important data with no back-up. [/tough love]
