FYI the AMD64 designation is a historical reference to the 64 bit instruction set that AMD pioneered and now Intel also support with their 64 bit CPUs.

Intel created the i386 CPU so that instruction set (used by many CPU manufacturers) name is still used today, so it is only fair that AMD have the name that they created also used in the same way.
