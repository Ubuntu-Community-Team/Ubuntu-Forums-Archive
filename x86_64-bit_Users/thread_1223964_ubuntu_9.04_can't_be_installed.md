### Post by Raffles10 on 2009-07-27
If you want to do a clean install I would suggest setting up your partitions before you start, I find this always works best. Boot with gparted live and create or modify the partitions as you desire, I would suggest one partition for windows and three for linux: root, home, swap. Install windows first, then linux.
