### Post by PRMan on 2008-10-16
To my knowledge, there is no way to use Access databases in Linux.  There are a few utilities to convert them to other things.

Another good tool for you to check out is to use VB.NET on a PC platform and then run the program on Mono on .NET.  You would get easy access to multiple database types (including SQL Server if the data was on one).

Most people on Linux use MySQL or Postgres, which are both free.
