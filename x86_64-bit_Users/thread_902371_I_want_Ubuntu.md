[http://albertomilone.com/nvidia_scripts1.html](http://albertomilone.com/nvidia_scripts1.html)

### Post by Martje_001 on 2008-08-28
That means it trys to start X, but fails. It problably tries to load a too high resolution for your monitor. You have to edit **/etc/X11/xorg.conf** with nano.

```
sudo nano /etc/X11/xorg.conf
```
(CTRL + O to save, CTRL + X to exit)

In the section 'screen' add this:
```
SubSection "Display"
		Viewport   0 0
		Depth     24
		Modes    "1024x768" "800x600" "640x480"
	EndSubSection

```

Edit: Didn't see it was a problem with Sync. Add these lines:
```
HorizSync    31.0 - 48.0
	VertRefresh  50.0 - 100.0

```
to the section 'Monitor'.
