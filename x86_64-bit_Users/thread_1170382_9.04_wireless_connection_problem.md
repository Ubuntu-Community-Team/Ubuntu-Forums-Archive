### Post by pro003 on 2009-05-26
try this:

```
sudo ifconfig wlan0 down
```

wait for sec, and then

```
sudo ifconfig wlan0 up
```

in right upper corner of your desktop (if gnome) there is a network manager icon and if you right click it it should show you network a.p visible in your area, then chose one and if asked enter the passphrase...etc
