### Post by gonesolo on 2008-04-29
Had a simular cdrom issue when installing 8.04 myself.

got an error - invalid install media or simular - this cd can not be used to install.

I rebooted and verified the CD and it passed. So I tried install again and got the same error.

Worked it out though. 

I have 2 cdroms on my system. When I have a CD in the other drive then I get this error.

So suggestion if you have a second cdrom/dvd drive make sure you have no media in the other drive.
