### Post by bumanie on 2009-01-08
The best thing to use would be testdisk to try to recover the entire partition. > sudo apt-get install testdisk to start testdisk > sudo testdiskor get a live cd of testdisk from [here]("http://www.cgsecurity.org/wiki/TestDisk"), where there is a tutorial on how to use it. If that doesn't work, get photorec from the same site (tutorial there also) or > sudo apt-get install photorec to start > sudo photorecPhotorec looks for 'raw' data and works independent of the filesystem. Good luck.
