### Post by becho on 2009-06-14
I have the same card and it runs perfect. You need to download the drivers from the Creative website for the X-Fi Extremegamer. Make sure u download the Linux version. Once downloaded u have to right click and extract the file tar.gz then using console make yourself root by typing "sudo su -" and hit enter,put in your password, next u  CD into the Xfi file it created and type "make" and hit enter then u type "make install" and reboot. There will be a readme file inside the xfi folder that tells u what to do. That's how I got mine workng.
