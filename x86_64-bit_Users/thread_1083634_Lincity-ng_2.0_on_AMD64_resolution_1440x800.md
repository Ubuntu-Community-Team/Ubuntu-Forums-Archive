### Post by AutiCoder on 2009-03-01
Hello everyone , 

This is a little how to , to get Lincity-ng 2.0 working on a AMD64 with a wide-screen ( mine is 1400x800 ) using Intrepid .


This is where you find the game :

[http://www.getdeb.net/release/3895](http://www.getdeb.net/release/3895) 

Download both   lincity-ng  (361.5 kB) and  lincity-ng-data  (39.8 MB)

Remove previous installed versions

Install the data package first and then the lincity-ng

If you have some troubles with the screen , check out your resolution .

I've done this to fix the graphics : 

```
lincity-ng -S 1440x800
```

Now it works like it should be .

I hoped this will help someone ...

Greetings to you all
