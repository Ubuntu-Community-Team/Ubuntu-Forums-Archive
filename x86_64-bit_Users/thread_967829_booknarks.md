### Post by Kilz on 2008-11-02
> **gretarsson said:**
> Hi there
I like to know in what folder is bookmarks for firefox?

The actual file that holds them is in ~/.mozilla/firefox/RandomNamedDir/bookmarks.html. The .mozilla folder is a hidden folder in your home directory. The RandomNamedDir is actually a random named directory, there is no way to know what it is. There should only be one directory anyway, and if not only one should have a bookmarks.html.
If you just want a backup of the bookmarks you can click Bookmarks > Organize Bookmarks >Import & Backup > Export.html and then save it. You can also restore the bookmarks this way, you just import the file.
