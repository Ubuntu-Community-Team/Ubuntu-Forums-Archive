### Post by drs305 on 2009-08-06
The easiest is to install and use Startup-Manager. It's a gui app that tweaks the menu.lst so you don't have to edit it manually. I wrote a guide a while back on how to reduce the number of kernels on the menu. But SUM can tweak a lot more settings.

Click on the link "SUM" in my signature line for information on StartUp-Manager and a bit on manually editing menu.lst if you prefer to do it that way.
