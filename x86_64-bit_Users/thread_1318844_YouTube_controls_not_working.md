### Post by jhulf on 2009-11-08
Hi, I had the same issue with flash on 64bit karmic.

First, I uninstalled the existing Flash completely via synaptic.
I closed firefox.

Second, I downloaded the 64bit flash from 'Adobe Labs'.
I unpacked the file and moved it into ~/.mozilla/plugins. (If the 'plugins' folder is not there, make one.) 

Since then everything works fine. Except the CPU load is relatively high during flash playback.

For further information search 'karmic + flash' or read this thread:
[http://ubuntuforums.org/showthread.php?t=1259102](http://ubuntuforums.org/showthread.php?t=1259102)
