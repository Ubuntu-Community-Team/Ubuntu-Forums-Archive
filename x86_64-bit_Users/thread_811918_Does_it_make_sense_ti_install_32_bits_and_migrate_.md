### Post by Cotopaxi on 2008-05-29
I bought my computer with two removable Hard Disk drives. On the first HD I had installed Windows
and on the second HD I installed Ubuntu 64 bits (Kubuntu in my case) and fell in love with it.

I wrongly decided to install 64 bit (Kubuntu 7.10) and managed to fix most of the problems I encountered, except some smaller problems that that are not actually critical.

Recently I gave the Windows Operative System on the first drive the boot, I actually needed that for self-esteem reasons; I wanted to prove myself, that I am not a Microsoft slave anymore.:-D

Now, given that this drive is available, I am having a crazy idea:

Does it make sense to install first a 32bits (K)Ubuntu version (v.7.10), install EVERYTHING under this Ubuntu
version and then upgrade to 64 bit (K)Ubuntu (Hardy)?

In my case, a list of items (software & drivers) to be installed would be: Opera, Real Player, Flash Player, Skype, Open Office Suite, View Your Mind (mind-mapping program). These programs actually work fine so far.

Programs I do not manage to install under 64 bits: Free Mind (mind-mapping program), a CAD-CAM program for beginners, VMware Windows emulator and the scanner driver.

Actually, the only reason I even think about the idea of a 32 bit installation, installing all the programs and then upgrading to 64 bits, is because I have a HD available for playing.

Now, opinions please:

1) Does it make sense?
2) Would it generally make sense for a (K)Ubuntu beginner to install first the 32 bit version with ALL software & drivers and then upgrade to 64 bits? Would this newcomer be on the safe side by going this way?

Thanks.
