### Post by Kilz on 2008-10-19
The one specifically designed to run on that hardware is the amd64 version. In this case amd64 refers to the 64bit chip, intel calls them emt. The core 2 duo is a 64bit emt chip. The i386 version will run on it, but will make it run in 32bit mode only.

### Post by jespdj on 2008-10-20
