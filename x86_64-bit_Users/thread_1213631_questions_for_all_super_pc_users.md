### Post by jocko on 2009-07-15
> **vinutux said:**
> If u have more than 4 GB ram sure there is improovements over 32 bit...

[COLOR=Red]othervice not....[/COLOR]
Wrong. The ability to use more than 4 GB of ram is **[COLOR=Black]not[/COLOR]** the only benefit of 64 bit, and you **don't** need 4 GB to notice the benefits. With a 64 bit os your cpu can process more information per cpu cycle, so it has the ability to do things faster than the same cpu running 32 bit code. As it can process more information, it will also use more ram. As long as the amount of ram is not the bottleneck in your computer, things *will* be faster with 64 bit (and the amount of ram you need to avoid a bottleneck is higher with 64 bit than 32 bit, but it's well below 4 GB).
But as long as you don't do things like video encoding/ripping, kernel compilation, 3d image rendering or other serious stuff that requires processing of a lot of information, you may not notice the difference.
