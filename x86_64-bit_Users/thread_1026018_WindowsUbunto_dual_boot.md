### Post by wtwhitey on 2008-12-30
Had the same problem.  If your bios is like mine you have to go to boot menu (f-10) and select your dvd drive each time you want to boot from it.  My bios would not allow me to put it in boot order first with hard drive next even though that was an option in settings the boot options menu overrode it.  This is actually nice for me as I share a computer with my non linux wife.  my computer brings up vista first after Ubuntu install unless I stop it during the boot.  If I hadn't told her it was there,  she wouldn't even know Ubuntu was there.
