### Post by QIII on 2009-09-05
While there are two ways to run Windows "inside" Ubuntu, you may decide you don't want to do either and would rather just dual boot and keep Windows native.

To run Windows in Ubuntu you can use something like Wine (which is problematic and does not run all Windows applications well) or use a virtual machine (which can tap your system resources.)

This gives a pretty good rundown on dual booting ...

[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)

Despite the rumors, Ubuntu is very good at detecting hardware and using appropriate drivers.  If you want, you can install the restricted (read "proprietary") driver for your video.  I think most people have good luck with nVidia.  I have.
