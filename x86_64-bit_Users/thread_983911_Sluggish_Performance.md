### Post by Musicalmidget on 2008-11-16
Hi.

I've been trying for a few weeks now to make the full switch to Ubuntu as my primary operating system for pretty much everything except gaming.  However, it seems that no matter what I try some applications are really running quite sluggishly and I've been subsequently unable to commit to the switch as yet.

I'm trying to run Ubuntu on a fairly decent system.  Some vital specs are:
[LIST]
[*]AMD 6400 X2 3.2GHz (64 bit)
[*]3GB DDR RAM (PC6400, 800MHz)
[*]Nvidia 8800GT 512MB
[/LIST]

In particular, it's the applications I want to use the most that seem to be performing the worst.  Firefox is particularly sluggish in general, even when I'm only browsing with one window and one tab open.  Similarly, I seem to be suffering from the grey box issue when there is flash on a page, although I gather I'm not alone in this.

The other application that's really blocking my switch is Eclipse.  At the very least, I'd really like to use Ubuntu for web development.  I have a Kubuntu machine at work (using 8.04) and I really enjoy developing on it.  However, Eclipse (with PDT) seems to run so slowly on my machine that it's practically impossible to use.  For instance, for each key I press there's a delay of anything from 0.5 to 5 seconds before the key press is displayed on the screen.  Of course, this gets worse the faster I type.

Initially, I attempted to use Ubuntu 8.10 and then noticed the issues as mentioned above.  After a few days battling, I then tried Ubuntu 8.04 which improved things to an extent.  Firefox seemed a little more responsive and I don't recall having the flash issue (although I can't be certain).  Eclipse however remained as slow as ever.  I've recently tried again with 8.10 and the same issues with Firefox and Eclipse are present.

I've seen a number of threads relating to slow performance on Eclipse which point to the java environment in use on the system.  Many such threads suggest using a version of Sun Java which I've tried to no avail.  I verified that Eclipse was indeed using the correct environment and it still didn't increase performance.

I suppose the big question I'm asking here is whether there's any particular reason I should be getting such bad performance on my particular machine.  I have an identical hard drive in the machine which holds a vista installation and that runs fine.

Any suggestions that could increase the performance I'm getting would be most appreciated.  I'd really like to make this switch happen.

Just to clarify, the current version I'm using is 8.10 (with Gnome) and if possible, it would be my preference to stick with this version.

I certainly will.

My setup includes two identical monitors running from the same Nvidia graphics card, but you can probably tweak this for your own setup.  I would advise keeping a copy of your working xorg.conf somewhere safe, although I'm sure you're already aware of this.

```
Section "ServerLayout"
   	Identifier     "Default Layout"
	Screen		0  "Left Monitor"  0	   0
	Screen		1  "Right Monitor"  RightOf  "LeftScreen"
	InputDevice    "Keyboard0"	"CoreKeyboard"
	InputDevice    "Mouse0"		"CorePointer"
	Option		"Xinerama"	"Off"
	Option		"Clone"		"On"
EndSection

Section "InputDevice"
	# generated from default
	Identifier     "Keyboard0"
	Driver         "keyboard"
EndSection

Section "InputDevice"
	# generated from default
	Identifier     "Mouse0"
	Driver         "mouse"
	Option         "Protocol" "auto"
	Option         "Device" "/dev/psaux"
	Option         "Emulate3Buttons" "no"
	Option         "ZAxisMapping" "4 5"
EndSection

Section "Monitor"
	Identifier	"Primary Monitor"
	VendorName	"HannsG"
	ModelName	"HU196D"
	HorizSync	30.0 - 80.0
	VertRefresh	50.0 - 75.0
	Option		"DPMS"
EndSection

Section "Monitor"
	Identifier	"Secondary Monitor"
	VendorName	"HannsG"
	ModelName	"HU196D"
	HorizSync	30.0 - 80.0
	VertRefresh	50.0 - 75.0
	Option		"DPMS"
EndSection

Section "Device"
	Identifier     "Device0"
	Driver         "nvidia"
	VendorName     "NVIDIA Corporation"
	BoardName      "GeForce 8800 GT"
	BusID          "PCI:7:0:0"
	Screen          0
EndSection

Section "Screen"
	Identifier	"Left Monitor"
	Device		"Device0"
	Monitor		"Primary Monitor"
	DefaultDepth	24
	Option		"TwinView"	"1"
	SubSection "Display"
	    Depth	24
	    Modes	"1280x1024"	"1024x768"	"800x600"	"640x480"
	EndSubSection
EndSection

Section "Screen"
	Identifier	"Right Monitor"
	Device		"Device0"
	Monitor		"Secondary Monitor"
	DefaultDepth	24
	Option		"TwinView"	"1"
	SubSection "Display"
	    Depth	24
	    Modes	"1280x1024"	"1024x768"	"800x600"	"640x480"
	EndSubSection
EndSection

Section "Extensions"
	Option		"Composite"	"0"
EndSection
```
