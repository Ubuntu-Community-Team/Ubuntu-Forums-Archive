### Post by bford16 on 2008-09-08
It is fairly easy to share a printer from Ubuntu to Windows, provided you have already installed Samba and set up your Workgroup.  See the Printing section on this page. <http://www.funnestra.org/ubuntu/hardy/#ipp>  Do all the steps for "On the Server" in Ubuntu. Then install the printer drivers under Windows, and connect to the printer in the usual way.
