### Post by cariboo on 2008-10-10
Which program are you looking for the source code for? Most of it is available in the repoistories if you have the src repos enableb. Check System-->Administration-->Software Sources, uncheck the packages and leave the src enable then in a terminal type:

```
sudo apt-get update
```

Then go to System-->Adminstration-->Synaptic Package Management, all the source packages should be listed for you.
