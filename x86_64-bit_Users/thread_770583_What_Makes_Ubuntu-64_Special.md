EMT64 (Extended Memory Technology 64) chips allow a P4 chip to
use the same registers that a 64 bit chip accesses, so that they can use

But in the end it makes no difference. Intel's chips with EM64T (which has been renamed to Intel 64) are in no way inferior to AMD chips with AMD64.

Intel 64-bit x86 processors and AMD 64-bit x86 processors have the same instruction set (with some very minor differences), the same registers, etc.

The actual hardware implementation may be different, but the Intel implementation is in no way inferior to the AMD implementation.
