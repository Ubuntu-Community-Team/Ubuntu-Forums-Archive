### Post by Kilz on 2008-07-18
> **dystorteddream said:**
> What would you suggest for running Warcraft III? WinE? 

Thanks for the info!

D~D

Wine will run Warcraft 3, but so will plays on linux with a few edits. If you go the Wine only way, [look here for all the required settings.]("http://appdb.winehq.org/appview.php?iVersionId=1177"). If you go the plays on linux rout, you install then open up playonlinux, highlight Warcraft 3 in the menu, click tools, then registry editor, and add the key HKEY_CURRENT_USER\Software\Blizzard Entertainment\Warcraft III with new DWORD value called "Gfx OpenGL" with the value set to 1. Then rename the movie folder in ~/.PlayOnLinux/wineprefix/WarcraftIII/drive_c/Program Files/Warcraft III.
