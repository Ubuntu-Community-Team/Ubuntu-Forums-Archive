### Post by cybrsaylr on 2009-05-23
I did a clean install going from 32b to 64b.

When I got to that install step that asks if you want to save or import settings from your previous OS, I clicked 'yes' and all my Home folder files were carried over to the 64b clean install of Jaunty with no problem.

 I use Opera brower.
When I installed 64b Opera to replace 32b Opera, all my 70 bookmarks were carried over also to the new 64b version of Opera. That was a pleasant surprise.
