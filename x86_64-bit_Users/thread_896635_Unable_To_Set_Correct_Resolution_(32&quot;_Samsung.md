### Post by soxs on 2008-08-23
Replace the whole last part named Section screen wwith this one

```
Section "Screen"
    Identifier     "Screen0"
    Device         "Videocard0"
    Monitor        "Monitor0"
    Option         "TwinView" "1"
    Option         "TwinViewXineramaInfoOrder" "DFP-0"
    SubSection "Display"
         Modes		"1360x768"    "1024x768"	"640x480"
    EndSubSection
    Defaultdepth	24
EndSection

```

Note: use ```
sudo gedit
``` in a terminal to be able to edit and save it

### Post by soxs on 2008-10-03
```
Section "Device"
...
        Option      "ForceMonitors" "crt1"
        Option      "EnableMonitor" "crt1,tmds2i"
...
EndSection
```

add these 2 options to your device section, which forces connected Monitor or LCD to be considerad as NON-TV
