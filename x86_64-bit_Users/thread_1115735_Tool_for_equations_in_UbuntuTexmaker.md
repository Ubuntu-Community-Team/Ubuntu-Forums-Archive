### Post by 3Miro on 2009-04-05
If you want to just type couple of equations, then use some graphical program (actually I think you can write equations in open office). If you have a complicated document to write, learn LaTeX. I know it may look hard in the beginning, but no graphical program can make an integral faster than \int

The expression:
```
\int_0^\infty e^{ A^T t} Q e^{ A t } dt
```
takes about 5 seconds to type, try it in open office or something like that, it may take a minute.
