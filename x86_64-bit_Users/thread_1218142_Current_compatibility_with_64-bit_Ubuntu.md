### Post by automaton26 on 2009-07-20
I've not had any 64-bit specific problems at all, but then I don't use your applications. I think there were issues with Flash-plugins in the past, but this all seems solved now - well, it automatically works on mine :).

You could try googling for each application along with "64-bit ubuntu", to see if anyone else has had problems first ?

If you're not sure, just stick with 32-bit (unless you want to address more than 4 Gb memory) because the performance benefits (from having extra CPU registers?) might improve benchmarks, but doesn't make things much faster when running real-world software.

### Post by edantes on 2009-07-20
I have been on 64bit since 8.04.  All problems with software installation are gone.  My only two remaining tweaks overriding the official solutions  are:

**Flash 10:**  I am using the alpha 64bit version 10.0 r22 downloaded directly from Adobe and installed by hand.

**Java for Firefox:** Sun Java (my default Java environment) still lacks a 64bit plugin, I installed the IcedTea Browser plugin, which requires the installation of OpenJDK.
