### Post by Barquero on 2009-07-22
The bios is where I'd start, I've had experience of a barebones rig through a friend who has no patience for troubleshooting, and the bios default for memory voltage was way off the spec for the RAM involved, even though the timings were relaxed, causing crashes.  It's good advice to flash the bios if an update is available and boot into memtest to make sure the RAM is good.
