Hi,
first of all what do you mean by "very slow"? I don't think you will benefit from Ubuntu so much to increase internet speed. If you have slow internet connection you will not benefit at all. Computers today are usually not the problem but connection can be.

Try running Windows XP first and if you still have a slow connection you probably have slow internet connection to your ISP.

If XP works way better then Vista, there is possible you have installed Vista on old PC, which consumes all RAM and your computer is swapping.

What I am trying to say is: Ubuntu is great software but it will not solve your hardware problems if there are present (I don't know that). Something like if roads (internet connection) are busy no car (Windows/Ubuntu) can drive fast.

The simplest thing is to download Ubuntu and burn ISO image on CD. Then boot up CD it will start live CD (running Ubuntu full system without installing it on hard disk). Then try running Firefox.

If you have an old hardware then Xubuntu will be running faster.
