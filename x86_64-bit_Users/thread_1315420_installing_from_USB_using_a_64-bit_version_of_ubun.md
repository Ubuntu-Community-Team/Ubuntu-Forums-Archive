### Post by sanderj on 2009-11-05
I tried Unetbootin to create a live USB Ubuntu stick to install Ubuntu to my (cd-drive-less) Netbook. It worked, but the boot process looked slow and looked different than de Ubuntu boot process. Disclaimer: this is just a guess, I have no facts for it.

So, I did it the old-skool way: boot the Ubuntu CD in another machine (with cd-drive), create a live-USB-stick in that machine, then use the live-USB-stick on my netbook to install Ubuntu there. And that worked for me.

And: 32-bit or 64-bit shouldn't matter, as long as both machines are 64-bit if you want 64-bit.
