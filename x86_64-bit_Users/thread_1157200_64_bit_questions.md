> **Yomny said:**
> First thing this AMD 64 does it necessarily mean to be used with AMD processors or boards?

No, the 64-bit iso's are not for use only with AMD processors.

The iso's are for Intel users with EM64T processors eg: Pentium D / Pentium Dual-Core / Celeron D/ Core2 series including Duo, Quad, and Xeon processors, as well as certain Pentium4's, As well as AMD 64 users.
