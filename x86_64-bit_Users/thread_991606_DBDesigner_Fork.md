### Post by Yellow Pasque on 2008-11-24
The i386 package seems to work.
```
sudo apt-get install ia32-libs
cd ~/
mkdir DBDesigner
cd DBDesigner
wget http://downloads.sourceforge.net/dbdesigner-fork/DBDesignerFork-1.4-bin-i386-linux.tar.gz?modtime=1185910135&big_mirror=0
tar xzf DBDesignerFork-1.4-bin-i386-linux.tar.gz
cd bin/
./startdbd_usingAMD64
```
