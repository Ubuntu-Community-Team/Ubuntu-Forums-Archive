### Post by jocko on 2008-10-26
To install drivers for your graphics card, just start the restricted hardware manager (in gnome you find it in System --> Administration --> Hardware Drivers, not sure about kde).
If you don't find it in the menu, you can probably start it from a terminal by:
```
sudo jockey-kde
```
It should tell you that there are drivers available and let you activate them.
