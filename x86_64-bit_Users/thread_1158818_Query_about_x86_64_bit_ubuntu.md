### Post by krendar on 2009-05-14
> **Ajit Wadekar said:**
> But can 64 bit version of ubuntu can get installed on a 32 bit system

Windows is showing your system as a 32-bit system because you probably use the 32-bit edition of Windows.

64-bit Ubuntu/Linux and 64-bit Windows can be installed on most modern desktop processors (except possibly some processors used on mobile devices like Netbooks). Don't be fooled by the AMD64 designation you sometimes see on 64-bit Linuxes. It will work on modern Intel processors as well :)
