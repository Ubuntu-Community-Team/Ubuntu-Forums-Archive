### Post by toemos on 2008-12-18
I had this same problem,

solved by going into Tools>Options>System Tray>Shutdown Immediately.

for some reason frostwire won't ever close with "shutdown after transfers" enabled, whether transfers are active or not.
this can be annoying as im used to the 'X' button minimising to system tray when im downloading...
