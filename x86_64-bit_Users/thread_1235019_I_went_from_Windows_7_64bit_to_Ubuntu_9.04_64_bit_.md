### Post by edwardtilbury on 2009-08-08
I had to buy a new hard drive, since mine was too small and too slow.. These P105-S9339 laptops come with a pretty small drive... Now I'm happy with my new 500gb drive that I got for $89.00 from Fry's in Fullerton.  

No windows partitions!!  I copied all my stuff over to a share on my desktop server, and connected to it with Places to copy everything back to my new user directory... 

I'm RDP-ing to all my windows servers and desktop machines, I have a virtualbox (it's slow) but oh well, just incase I need to run my iphone sync.  

What an experience.. Compiz is enabled.. I have some fish swimming in my background and a I added a shark!  I got a new skydome and configured the "forward" button on my death adder mouse to invoke the cube globe thing.. it's like a fish bowl floating in the sky, lol..  It's soo coool. I close windows and there's this rainbow fire effect.  I have the awn mac toolbar that's actually useful, unlike the one I tried in windows.

Cons..
No more photoshop or iTunes
No IE... darn...  Actually I only need it for testing.. but oh well I can rdp to my windows box
No more excel.. I'll have to learn open office..
Can't do windows stuff, like create a new folder or copy / cut / paste in an "Open dialoge" box etc..
No default xbox360 controller support, I shouldn't play games anyways.


Cool stuff..
FREE!
Not bloated! No AV needed etc.. I don't have 10 billion things embedded and running in my sys tray anymore.
Still have pidgin and it doesn't crash now!
Still have filezilla!
Switched to kompozer from Dreamweaver, uhhh Kompozer is super buggy.. I gotta ditch this thing.
Switched to Thunderbird from Outlook
Now I have the proper environment to learn how to setup a linux server /w apache & php, no more wamp...
Installs are soo damn easy.. I just go to the synaptic thing and type in "compiz" check a box or two and bam.. I guess it also updates if I install it that way.. 
Insane desktop effects
Installed Emerald aero glass, but I cant find how to make it Gaussian blur, oh well.
I can still watch movies etc.. 


Ok, I gotta get back to work, but I'm amazed that I could get my Nvidia card to work and my sound, mouse , etc.. to work on this laptop, 9.04 is really something..  No thanks to Toshiba of course.
