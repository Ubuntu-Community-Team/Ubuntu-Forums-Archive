### Post by Darael on 2009-04-20
If you ordered a 32-bit CD from shipit, that's all that will be on the disc.  If you ordered a 64-bit CD, you'll get a 64-bit only disc.  If you don't know what you ordered or don't remember being given the option, you'll probably get 32-bit discs.

Unfortunately, there's no easy way to switch post-install.  I can only suggest using torrents to get the other CD as fast as possible - though I must also suggest that you torrent the Jaunty release on Thursday, as you do  not appear to need an LTS release.

If you don't use BitTorrent, it's worth trying - the http and ftp download servers will be flooded for the Jaunty release, but more people will only speed up the torrents in the long run.
