### Post by mpsii on 2008-12-24
My $.02 is to stay 32-bit. There really is not a reason to go 64-bit unless you use applications (photo or video editing, code compilation, physics calculations, etc) that would take advantage of the 64-bit processing. With only 1GB (I love that I just said "only" 1GB) of RAM, it is not really worth it. You would be better sticking with 32-bit until you find an application that you just MUST run in 64-bit to get the bang out of it.
