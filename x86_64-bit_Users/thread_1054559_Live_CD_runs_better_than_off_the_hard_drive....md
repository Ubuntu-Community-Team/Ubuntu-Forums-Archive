### Post by Jorge_C on 2009-01-30
Hi Neodynamics,

Your problem is that downloading apps from the repos using synaptic is too slow, right?
If so, open up Synaptic, then Settings->Repositories. In the "Ubuntu Software" tab, select download from "Other" and then "Select Best server". This will run some tests and you will probably get better download speeds.

By the way, the partition size shouldn't affect download speeds.
