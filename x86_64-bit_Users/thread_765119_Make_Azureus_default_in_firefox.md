### Post by hodge24 on 2008-04-30
Go to Edit -> Preferences in Firefox, and select the Applications tab. Find "torrent", and change the action to "azureus". If Azureus isn't in the list, choose "other" and point it to the Azureus executable - probably /usr/bin/azureus

Hope that helps :)
