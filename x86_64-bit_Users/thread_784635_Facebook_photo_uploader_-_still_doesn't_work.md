### Post by gnimsh on 2008-05-17
Hey I just thought I'd throw in my 2 cents on this problem, as I just "solved" it.  I also cannot load the java applet, but can only use the simple uploader.  I tried it in firefox, and it stalls, and in swiftweasel it keeps telling me that it needs to download the plugin.  I compared the about:config java settings between firefox and swiftweasel and they were the same, so that left me stumped and needing to get creative.  

What I did was search for a facebook photo uploading extension, which I found: [http://www.fireuploader.com/#fupHome](http://www.fireuploader.com/#fupHome)

Choose facebook from the dropdown menu, and then enter your username and password.  From there you can log in and it presents all your albums as folders.  To create a new album simply click the new folder button.  

I think this is the easiest workaround for this problem, if not the most elegant, but it requires no java fixing or anything complicated.
