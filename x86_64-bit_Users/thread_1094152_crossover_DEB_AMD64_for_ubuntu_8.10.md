Yes, jusst go to Codeweavers.com and you can get the .deb file for 64-bit Linux. If you qualify for the academic version you can get a discount.

But bear in mind that if Wine will run the programs you need it may be a  better alternative, as it is free and open source. Codeweavers adds features and functionality to Wine and markets it commercially. Most people don't have a problem with this because Codeweavers contributes back to the Wine project. Nevertheless, I would suggest checking out Wine first at winehq.org. Both Codeweavers and Wine have a database of supported applications that you can browse.
