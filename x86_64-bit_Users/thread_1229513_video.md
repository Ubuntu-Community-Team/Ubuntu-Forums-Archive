### Post by niw_uk1964 on 2009-08-02
I had a similar problem with a Radeon 3300 on my desktop motherboard.

The solution was to install the Radeon drivers. 

System - Administration - Hardware drivers.

Select the ATI-AMD proprietary driver.

If it is not available then download from here:

[http://support.amd.com/us/gpudownload/linux/Pages/radeon_linux.aspx?type=2.7&product=2.7.4.3.3.3.1&lang=English&rev=9.7&ostype](http://support.amd.com/us/gpudownload/linux/Pages/radeon_linux.aspx?type=2.7&product=2.7.4.3.3.3.1&lang=English&rev=9.7&ostype)

Good luck!
