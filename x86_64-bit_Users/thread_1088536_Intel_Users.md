### Post by gagnon88 on 2009-03-06
AMD64 here is just the "generic" name for the x86_64 architecture... Intel 64-bit processors fully support this architecture. For example, I run Ubuntu Intrepid 64-bit on a Intel Dual-Core processor, and everything works like a charm!

EDIT: In other words, you can surely find help here... I already have had some.
