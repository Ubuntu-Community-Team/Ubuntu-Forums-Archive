### Post by Yellow Pasque on 2008-08-16
Why don't you give your kids a separate user account, and then use the Unix/Linux file permissions? Or, if they already have a separate user account, lock the screen. There's a shortcut for this. Right click on an empty part of a panel and select 'Add to Panel'. Scroll down and you'll find a Lock Screen utility.
