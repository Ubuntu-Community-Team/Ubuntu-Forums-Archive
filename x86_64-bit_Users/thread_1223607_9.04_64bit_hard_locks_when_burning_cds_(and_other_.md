Sometimes it will lock when trying to install a program one time and then work fine the next. Lately however it seems to hard lock whenever I try to burn a CD (DVDs burn fine). Also installing large apps in wine would freeze it but that seems to have gotten better with recent updates.

When it locks up I can't use the keyboard to switch to a different shell and it also drops off the network so I can't ssh in either. The only thing I can do is power it off.

I've found posts of other freezing problems [here](http://ubuntuforums.org/showthread.php?t=1186997&highlight=ubuntu+64+freeze) and [here](http://ubuntuforums.org/showthread.php?t=1163584) but my problem doesn't seem to fit either case (haven't had any file system corruption). 

As far as it being a hardware issue, it will run memtest86 and prime 95 for hours without errors. And also XP (32 bit) runs fine with no lockups.

Is there any specific place I can look for a log file that might give me a little more insight as to what's causing this?
