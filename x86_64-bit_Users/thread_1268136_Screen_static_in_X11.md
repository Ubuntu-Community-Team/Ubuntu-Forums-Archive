### Post by lightsaber on 2009-09-16
I have an issue with a fairly new desktop and the 9.04 version of Ubuntu running a 64 bit kernel.  The system is running a ASUS M3A78-EM AM2+/AM2 AMD 780G and an AMD Athlon X2 4850e.  First off, it doesn't recognize all 4gb of memory I have in there, which may just be a bios issue.  Secondly, I have a major issue with static on the screen when X starts.  I have no problem running the live disk and no problem in any of the startup screens.  I do not believe it is an issue with hardware acceleration because I turn off the ati drivers and I still have the problem.  I even had it running initially without the static, turned it off and back on and got it again.

Any ideas or suggestions would be helpful.

EDIT: I do believe now that it is an ATI problem.  I turned off the ATI drivers and logged out but didn't restart the machine. After disabling the ati drivers, when I restart, the static is gone.  So for now, no eye-candy until this gets fixed. :(
