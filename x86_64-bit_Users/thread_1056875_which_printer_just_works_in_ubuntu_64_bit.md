same here...bought it last week.

```
cd Desktop

sh hplip-3.9.2.run
```followed the prompts on screen and a few minutes later had a perfectly functioning printer/scanner.  

repeated on the other 3 ubuntu desktops + one ubuntu laptop in the house....same results, flawless.  :)
