### Post by Rog-Mahal on 2008-10-22
The Core 2 Duo is a 64 bit processor. I have one in my Macbook, and I'm currently running 64 bit Hardy. You can run either 32 or 64 bit versions of Ubuntu on your computer, however. Check the stickies at the top of the forum to read about the advantages and disadvantages.

Rog-Mahal is correct.  Additionally, if your processor was 32-bit the 64-bit version of Ubuntu wouldn't work at all.

### Post by jespdj on 2008-10-22
All Intel **Core 2 Duo** processors are 64-bit. As Thelasko says, if it wasn't, you would not have been able to install 64-bit Ubuntu at all. You cannot install 64-bit Ubuntu on a 32-bit processor, and there is no way that this would affect your hardware. The people who told you this don't know what they are talking about.

Note that the older **Core Duo** (without the 2) processors are 32-bit.
