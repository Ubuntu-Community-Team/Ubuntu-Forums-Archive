### Post by jbrown96 on 2009-01-14
Unfortunately the ATI drivers for Linux are not very good. There is a serious problem with "tearing," which sounds like your problem. There isn't a whole lot you can do, but using OpenGL or X11 output has improved the situation for me (HD3200). I use VLC, which is available in Synaptic, and it has lots of settings. You can set the mode in the Video Preferences. 

You may also be having a problem with Compiz. Try disabling it System-->Preferences-->Appearance-->Visual Effects.
