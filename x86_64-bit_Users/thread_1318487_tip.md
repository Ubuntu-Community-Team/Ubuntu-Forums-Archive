### Post by lavinog on 2009-11-08
> **necromonger said:**
> If you are using 9.10 64 bit DO NOT use the ati drivers if you have an older card it will cause big problems not ubuntu's fault it is all on ati there support for linux sucks!

ATI drivers don't support older cards. This was done to improve support for the newer cards.

Older cards should be using the open source drivers...if there is a problem with the open source drivers, you should file a bug report.

The hardware manager should not offer the ati driver if your card is not supported.  The driver from ATI will let you install the driver, and could break your system if your card isn't supported.
