### Post by Sef on 2008-12-30
I have the same graphics card, so here is the way I can do it.

1) System > Administration > NVidia X Server Settings

2) highlight X Server Display Configuration

3) Set Left Resolution button to desired resolution

4) Set Right Resolution button to desired refresh rate.
