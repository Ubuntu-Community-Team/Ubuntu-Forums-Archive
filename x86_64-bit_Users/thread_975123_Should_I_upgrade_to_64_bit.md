### Post by Artemis3 on 2008-11-08
I have been running 64 bit for almost a year, and there is nothing to worry anymore.

Flash works by installing flashplugin-nonfree (just like in 32 bit), but i simply install **ubuntu-restricted-extras** to take care of the rest.

The only 32 bit binary game i can think of is Enemy Territory and i don't notice any significant performance difference.

I also happen to be running Eve Online for almost a year. With premium content it goes smoother, eg: no slowdowns when approaching gates (Billboard?). With the recent server fixes, response is even snappier.

Note: i go without swap because 4g is a lot :)

### Post by Yellow Pasque on 2008-11-08
I would not wipe a working 32-bit install unless I was mainly doing some task that benefitted from 64-bit (Photoshop, VirtualBox/VM's, video encoding, heavy math apps, etc.) If your main focus is gaming and web browsing, it's not worth it, unless you have some really RAM-intensive games and have more than 3GB of RAM installed.
