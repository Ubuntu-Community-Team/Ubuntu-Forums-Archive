### Post by Yellow Pasque on 2009-05-17
For future reference:
- You need root persmissions to edit /etc/X11/xorg.conf (i.e. use *sudo nano* or *gksudo gedit* )
- If you're using Ubuntu 8.10 or later, X.org will ignore your input devices configuration in xorg.conf and use evdev unless you explicitly tell it not to: [http://wiki.archlinux.org/index.php/Xorg_input_hotplugging](http://wiki.archlinux.org/index.php/Xorg_input_hotplugging)
