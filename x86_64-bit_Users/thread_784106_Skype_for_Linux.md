### Post by jespdj on 2008-05-07
There is indeed no 64-bit Skype, but the 32-bit version works also on 64-bit Ubuntu.

If you add the [Medibuntu](http://medibuntu.org/) repository to your system, then it's very easy to install - the necessary 32-bit libraries will be installed automatically along with Skype.
