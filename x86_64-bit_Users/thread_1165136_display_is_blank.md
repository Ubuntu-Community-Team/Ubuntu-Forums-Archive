I upgraded from 8.10 to 9.04 and lost my display. All I have now is a blank sreen, and if I leave the sound on during startup, I hear the "drum" sound of an error.

I've followed all the advice listed in Ubuntu Forums an other linux forum and cannot get my display back.

I have a 64-bit Gateway, with a 19" LCD (FPD1960) monitor; and an Nvidia GeForce 6100 video card.

My final attempt to get it working involved the following steps:

1. removing all nvidia packages:
   ```
sudo apt-get remove --purge nvidia*
```

2. Stopped X server:
   ```
sudo /etc/init.d/gdm stop
```

3. I had previous downloaded the latest nvidia driver: NVIDIA-Linux-x86_64-180.51-pkg2.run. I installed it using:
   ```
sudo sh NVIDIA-Linux-x86_64-180.51-pkg2.run
```
accepting all defaults. It didn't find a compiled kernal so it supposedly built its own, and I also saw a message stating it couldn't find libGL or something like that, and it assumed it was installed.

   ```
sudo /etc/init.d/gdm start
```

Looking at the log, it recognizes my monitor and display card...
