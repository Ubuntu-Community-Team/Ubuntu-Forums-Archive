### Post by Ninjapirate on 2009-08-13
I'm using an Nvidia 8700M GT and I had a similar problem.

Adding 

**Option         "ModeValidation" "NoExtendedGPUCapabilitiesCheck"**

to /etc/X11/xorg.conf under the screen section, restarting, and then running gksudo or kdesudo nvidia settings change the settings to max resolution and merging it when I saved fixed the problem for me.

Here is the link to the thread that I got this info from -> [http://www.nvnews.net/vbulletin/showthread.php?t=126169]("http://www.nvnews.net/vbulletin/showthread.php?t=126169")

Hope this helps.
