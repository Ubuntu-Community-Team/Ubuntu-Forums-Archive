### Post by jespdj on 2009-02-17
Yes, 32-bit Ubuntu will work on a 64-bit AMD or Intel processor.

However, why would you want to do this? 64-bit Ubuntu works just as well. Almost all software is available as 64-bit native executables, and if it isn't, then there's no problem, because 32-bit software also runs on 64-bit Ubuntu.

### Post by bodhi.zazen on 2009-02-17
In general if you have a 64 bit processor you should use 64 bit Ubuntu.

If you have some personal reason or you *know* you *need* the 32 bit version for some task or application => go with the 32 bit.

You can install and run 32 bit apps on teh 64 bit version if needed.

The 32 bit version should run without any major problems on a 64 bit processor although to utilize more then 3.2 Gb RAM you will need to use a PAE enabled kernel.
