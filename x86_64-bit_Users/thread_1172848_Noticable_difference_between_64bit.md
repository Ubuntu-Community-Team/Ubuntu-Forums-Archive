### Post by jespdj on 2009-05-29
> **Goddard said:**
> ... but since I switched from Windows I know how hard it is to find drivers and supported software ...
The fact that 64-bit drivers and other software are hard to find is a problem with 64-bit **Windows**. The same is not true for 64-bit Ubuntu, or  64-bit Linux in general.

Since Ubuntu and most of the drivers and software are open source, it's much easier to compile or adapt drivers for the 64-bit version. There is little difference in this regard between the 32-bit and the 64-bit version of Ubuntu. Fortunately, the manufacturers that supply closed-source drivers (such as nVidia) also have 64-bit versions of their drivers available.
