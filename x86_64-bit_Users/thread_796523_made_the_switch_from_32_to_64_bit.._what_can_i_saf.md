### Post by Yellow Pasque on 2008-05-16
> Do I need to reinstall everything?
You can save your /home and other personal data on a separate partition. You can also run the 64-bit system from a separate partition, but yes, you need to reinstall everything else.
> Can I just copy my home folder and bin folder over and whalla I get my old system with a 64 bit processor?
No, not /bin. You can copy your home folder and personal data, but you'll want to run the 64-bit version of the Ubuntu/Linux apps.
> Are there some config folders I can copy over to change my desktop to the custom one I built when running 32 bit Ubuntu.
This will probably be taken care of by copying your /home, but you'll need to be more specific if you want clarification. What part of the desktop did you build? Background? Icons?
> Finally, how is it that Firefox found my old bookmarks?
Firefox and other Mozilla-based browsers use user profiles to keep track of favorites and prefs.
