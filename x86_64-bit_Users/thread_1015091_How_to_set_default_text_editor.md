### Post by wangjiajun on 2008-12-18
I have bluefish editor installed in my Hardy. 

Each time I try to open *.txt by double-click that file, there is always a dialog asking me if I would like to display its content or run it. I always want to say display, how can I avoid the dialog.

Second issue is, after bluefish is loaded, it does not automatically load the contents of the *.txt file I just double-clicked. How can I address this?

Thanks for help!

---

### Post by ajgreeny on 2008-12-18
For the first use **nautilus > Edit > Preferences - Behaviour** tab and set to display executable text files; the default seems to be to ask.

For the second right click on a txt file, go to **"Properties > Open with"** tab and set your preference there.
