### Post by pixel :-) on 2008-05-24
this line in your fstab is a CD driver, you have only one cd drive?
/dev/hdb /media/cdrom0 udf,iso9660 user,noauto 0 0

put a cd in and try 
mount /dev/hdb

normally in /media/cdrom0 you should find the countent of your drive,do you?

now i'll assume you actually have 2 drivers, and one for some reason didn't registered in the fstab

put  the cd in your burner/driver and try something like

sudo mount -t iso9660 -o ro /dev/hda /media/cdrom0
sudo mount -t iso9660 -o ro /dev/hdc /media/cdrom0
sudo mount -t iso9660 -o ro /dev/sr0 /media/cdrom0
sudo mount -t iso9660 -o ro /dev/sr1 /media/cdrom0

see in /dev if ther is stuff called hd?,try those names

if it doesn't complain at one check if you can see your stuff in /media/cdrom0

send us /var/log/dmesg (attachment please)
