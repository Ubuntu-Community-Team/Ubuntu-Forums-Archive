### Post by Sef on 2008-04-29
The current version of OpenJDK is compatible with Sun Java 1.6.0 .   However, some websites will not currently work with OpenJDK  because it is not 100% complete.   If 100% website compatibility is needed, then [Sun Java 6]("http://java.sun.com/javase/downloads/?intcmp=1281") will need to be installed. Once there,  click on this link: Java SE Runtime Environment (JRE).
