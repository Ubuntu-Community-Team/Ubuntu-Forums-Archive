### Post by sjbaugh on 2008-10-28
You might try EnvyNG (it's in the repositories). Download and run EnvyNG. EnvyNG auto-senses nvidia and ati video cards and builds an appropriate video driver and links it to the kernel.

When I first used it (I have only used it on 8.04 Hardy 64 bit), it sorted out the X settings as well. If it works for you I would recommend you save a backup of xorg.conf in /etc/X11 as this contains the resolution settings available.

Best of luck,

Steve
