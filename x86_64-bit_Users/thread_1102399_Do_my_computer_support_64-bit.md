### Post by jespdj on 2009-10-08
> **realizee said:**
> I have a pentium x2 2,5 ghz that could run 64 bit.
A "pentium x2", what's that? As far as I know there is no Intel or AMD processor called "pentium x2". Please find out what the exact model name or number is of your processor and then look it up on [Intel's processorfinder](http://processorfinder.intel.com/) (if it is an Intel processor). The specs you can find there will tell you if it has 64-bit extensions (look for "EM64T" or "Intel 64").

Don't be confused by the "amd64" name in Ubuntu's 64-bit version; this is for both AMD and Intel processors that have 64-bit extensions.
