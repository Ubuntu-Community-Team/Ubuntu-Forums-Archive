### Post by cariboo on 2009-01-06
Go [here]("http:///help.ubuntu.com/community/Medibuntu"), to add the Medibuntu repositories, then scroll down to get the gpg key. Once the repositories have been setup go to System-->Administration-->Synaptic Package Manager and do a search for **acroread**, mark it for installation, install it and your ready to view pdf's.
