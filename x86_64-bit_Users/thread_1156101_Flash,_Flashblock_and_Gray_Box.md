Upgraded to Jaunty yesterday and now where I used to get a gray box, I now get Flash!

Back when we used npiviewer and it was such a resource hog I installed Flash Block so that I could control which animations would launch.  I like having it around as it does away with some annoying ads as well.

What I'm finding is that if I click on a Flash app to allow it to play, it will go to the gray box I used to get shortly afterwards.

The quick fix for it apparently is to click somewhere in the Flash app once it has been activated from Flash Block.

Hopefully someone finds this useful. :)
