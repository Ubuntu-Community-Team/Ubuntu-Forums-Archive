### Post by jespdj on 2009-03-29
There are several threads with tutorials. Look at this one, for example:
[http://ubuntuforums.org/showthread.php?t=1063635](http://ubuntuforums.org/showthread.php?t=1063635)

Note: You need to be really careful and follow the tutorial exactly, don't make any typing mistakes.

The problem with Java on 64-bit is that Sun until recently did not have a Java plug-in for 64-bit systems. They did release a 64-bit plug-in in January, and it is included in the next version of Ubuntu that is going to be released in April (Ubuntu 9.04, a.k.a. Jaunty Jackalope).
