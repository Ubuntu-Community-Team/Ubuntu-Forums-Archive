### Post by exploder on 2008-05-10
My system is booting up much quicker with the 64 bit version of Hardy. Applications seem to start faster. This is my first time running a 64 bit OS. I just recently purchased a system that could support it.

The only disadvantage I have see is that more memory is used than the 32 bit version, but I have 1GB of RAM which is not a lot these days and the system runs fine.

I will be sticking with the 64 bit version of Hardy on the new system. Your best bet is to compare the two versions on your hardware and the decide which is best for you.
