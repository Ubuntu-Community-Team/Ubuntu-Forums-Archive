### Post by jespdj on 2008-12-09
> **cacycleworks said:**
> I'm running a Dell laptop with a core duo processor.
Me too (a Dell XPS M1530). Mine has a T9300 (2.5 GHz, with 6 MB processor cache). 64-bit Ubuntu 8.04 works great on it.

> **cacycleworks said:**
> From /proc/cpuinfo:
Intel(R) Core(TM)2 Duo CPU     T7300  @ 2.00GHz
each core has a 4G cache
No, that processor has a shared 4 MB cache for both cores (not "4G cache for each core"). It's actually a good thing that the on-processor cache is shared between the cores, because that means that both cores can see what the other core writes to the cache (which means the processor has to go to the relatively slow main memory less).
