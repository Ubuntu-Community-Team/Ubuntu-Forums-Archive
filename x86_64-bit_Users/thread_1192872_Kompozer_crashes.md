### Post by Francewhoa on 2009-06-23
KompoZer 0.7.10 is unstable with Ubuntu. **Try the latest version 0.8 alpha it's more stable.**

1. Download and extract the new version: 
[http://downloads.sourceforge.net/kompozer/kompozer-0.8a4-gcc4.2-i686.tar.gz](http://downloads.sourceforge.net/kompozer/kompozer-0.8a4-gcc4.2-i686.tar.gz)

2. Open the folder and double click on the file 'kompozer'. Kompozer will run as if it is installed.

That's it. The new version has views/tabs built in and spit window like Dreamweaver.

**To report bug** click on the following link then click on ADD NEW: [http://sourceforge.net/tracker/?group_id=170132&atid=853122](http://sourceforge.net/tracker/?group_id=170132&atid=853122)

**To request a new feature **click on the following link then click on ADD NEW:
[http://sourceforge.net/tracker/?group_id=170132&atid=853125](http://sourceforge.net/tracker/?group_id=170132&atid=853125)

**To request support** click on the following link then click on ADD NEW: [http://sourceforge.net/tracker/?group_id=170132&atid=853123](http://sourceforge.net/tracker/?group_id=170132&atid=853123)

-------------------
Note that this is an Alpha version. Use at your own risk. Alpha mean it's still under development and not stable. Visit the following page from time to time and download the final stable 0.8 release when available: [http://www.kompozer.net/download.php](http://www.kompozer.net/download.php)
