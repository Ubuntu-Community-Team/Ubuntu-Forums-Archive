### Post by Bucky Ball on 2008-10-17
Download the image and burn a cd of:

[www.supergrubdisk.org]("http://www.supergrubdisk.org")

This will add XP to your grub menu list.

Are you able to boot into Ubuntu from that screen? It is all as it should be BTW (choose the first entry to boot Ubuntu). There are other ways of adding Windoze to this list, but for now, supergrubdisk is your easy option.
