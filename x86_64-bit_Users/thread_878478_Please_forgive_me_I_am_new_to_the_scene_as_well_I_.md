### Post by bford16 on 2008-08-03
Ubuntu and Red Hat work in slightly different ways, but they are both "GNU/Linux."  There are two important things to learn. First there is the graphical interface, which is GNOME, or KDE or Xfce (Google those names to learn more). Second is the command line, which is called "bash" in Ubuntu.

In my opinion (whatever that's worth), the command line is what makes Linux so powerful. With it you can do some surpisingly cool things.  You can search for and obtain updates for every installed program on your computer, by typing a simple phrase, "sudo apt-get update && sudo apt-get upgrade." If you get tired of typing that, you can create a script to type it for you. Or you can use a 'history' command function to retype it.

My advice to you is to go poking around an finding out what you can do. "The truth is out there."
