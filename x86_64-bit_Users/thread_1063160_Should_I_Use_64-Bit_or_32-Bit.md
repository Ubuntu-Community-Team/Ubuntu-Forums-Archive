### Post by Yellow Pasque on 2009-02-07
You can read the (somewhat outdated) sticky on the subject, but the general rule is, start with 64-bit.
IMO, users should skip the versions of Java and Flash offered in the repos.
A lot of users report the native 64-bit Flash "Alpha" plugin works better than the 32-bit version offered in the repos (which requires a wrapper to work in 64-bit browsers)
[http://labs.adobe.com/downloads/flashplayer10.html](http://labs.adobe.com/downloads/flashplayer10.html)
As for Java, Sun just officially released Java 6 Update 12 (first version to include the 64-bit Java browser plugin) a few days ago.
[http://java.sun.com/javase/downloads/index.jsp](http://java.sun.com/javase/downloads/index.jsp)

> difference in performance?
Yes, most noticeably with tasks that take advantage of large integers (video encoding, graphics, DVD burning, heavy math apps, virtualization, etc.)
