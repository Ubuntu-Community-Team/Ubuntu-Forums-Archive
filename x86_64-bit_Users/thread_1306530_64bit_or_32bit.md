### Post by khamil8686 on 2009-10-30
if your processor can handle 64 bit id go 64 bit, all 32 bit apps can run on a 64 bit system but just cant access over 3gb of RAM, the 64 bit apps can access over 3GB ram. ive never had a problem running 64 bit ubuntu, just install IA32-libs package so you can run the 32 bit applications on your 64 bit system, sometimes installing new programs have weird errors when installing 32bit apps on a 64 bit system, but these problems have to do with missing IA32-libs, the libraries for 32 bit applications, so just install that and you will be fine, its smooth sailing, i just installed karmic koala 64 bit and everything went more smooth than any other os installation ive ever done :) 64 bit gives you a slight speed increase and you can also add more than 3gb of readable memory (many computers have 4gb of ram in them but a 32 bit OS and most people dont realize that 3gb ram is what is used by 32 bit os, unless you have a PAE-enabled OS) so heres a summary, go 64 bit and install IA32-libs :) :D
