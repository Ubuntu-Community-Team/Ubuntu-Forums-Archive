### Post by jespdj on 2008-11-01
You cannot install a 64-bit operating system on a computer with a 32-bit only processor. If you try to boot 64-bit Ubuntu (from a live CD) on a 32-bit system, you'll quickly get an error message that says the system can't be loaded.


Also, you cannot run 64-bit applications on a 32-bit operating system. The other way around does work (32-bit apps on a 64-bit OS).

### Post by stmiller on 2008-11-01

It depends. :) Software written to take advantage of 64bit will be faster on 64bit. Software which is written to take advantage of 32bit will be faster on 32bit. So it all depends on the exact program, and varies wildly.

For the end user there on the desktop there will be little to no perceivable difference with the setup you described. 

But if you are running mega-calculations with some specific software that is optimized for 64bit, you would need 64bit.
