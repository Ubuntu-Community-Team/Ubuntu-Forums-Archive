### Post by Justin Trouble on 2009-02-13
Thunderbird 3.0 beta 1 is really awesome (Deb in [https://launchpad.net/~fta/+archive/ppa](https://launchpad.net/~fta/+archive/ppa)).

David Ascher says "I feel that even for a first beta, Thunderbird 3 is much better than Thunderbird 2" [[http://ascher.ca/blog/2008/12/09/thunderbird-3-beta-1-a-platform-for-innovation-shapes-up/]](http://ascher.ca/blog/2008/12/09/thunderbird-3-beta-1-a-platform-for-innovation-shapes-up/]).
I don't mind living a little on the edge so I've taken the plunge.

The only problem I have is installing the Lightning calendar extension.

Thunderbird 3.0 beta 1 requires a version of Lightning higher than the current version 0.9 - which at the moment means a nightly.
32 bit nightly builds can be downloaded directly from Mozilla, but a 64 bit version simply doesn't seem to exist!

Has anyone over come this problem?
