### Post by Ken_Shabby on 2008-05-18
After trying many times to install Ubuntu 64 on a Gigabyte MA790FX-DQ6 Motherboard, and having the install lock up every time, I finally worked out a simple solution and couldn't find a reference from my many searches on the net recommending this easy fix, so hope this post might help someone.

From the Gigabyte web site I downloaded the latest version of the BIOS, and installed the new BIOS via USB memory stick, remember to edit the BIOS after install. I just did a load optimum defaults, untill after the install. 

The next install worked perfect, gpart completed  for the first time, and it's all up and running.

Long live UBUNTU
