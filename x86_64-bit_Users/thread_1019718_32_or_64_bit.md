THe quick and easy way is to open a Applications-->Accessories-->Terminal and type:

```
uname -a
```

If you have the 64bit version the output will look like this:

```
Linux jack 2.6.28-3-generic #4-Ubuntu SMP Fri Dec 12 22:47:31 UTC 2008 x86_64 GNU/Linux
```

Note the x86_64.

Correct: the "i686" means it's a 32-bit OS.
FWIW: i686 is a nickname for the sixth generation Intel x86 microprocessor architecture, which is 32-bit.

If you want to check if your *processor* is 64-bit (which is needed to run a 64-bit OS), you can use

```
grep lm /proc/cpuinfo
```

If that gives a line of output, your processor is 64-bit and you can (probably) run a 64-bit OS.
