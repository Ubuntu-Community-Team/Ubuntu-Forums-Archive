### Post by edonn on 2008-10-28
I was suffering from this exact same problem (mouse sort of jams or lags whenever I press something on the keyboard) but found the solution here:

[https://wiki.ubuntu.com/MacBookPro/Feisty](https://wiki.ubuntu.com/MacBookPro/Feisty)

On that page search for the term "mouse lag" you'll find the instructions. From that page:

[INDENT]You may notice that the mouse or touchpad becomes unresponsive when holding down a key (especially annoying in games.) This is due to a program called mouseemu. To fix temporarily, run the following command:

[INDENT]sudo killall mouseemu[/INDENT]

If that works and doesn't cause any other problems or disable any other features, you may want to disable it permanently. To remove mouseemu, run the following:

[INDENT]sudo apt-get remove mouseemu[/INDENT][/INDENT]

Hope it works for you.
