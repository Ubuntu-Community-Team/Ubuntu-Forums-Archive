### Post by gjoellee on 2008-07-28
Amarok works very good (I use it)
```
sudo apt-get install amarok
```

How to use the  ipod with Amarok:
open Amarok and go to Settings->Configure Amarok->Media Devices
now press add device, set the pugin to Apple ipod, add a name and to find the path:
plug in the ipod, when Ubuntu have detected it Rythmbox start (you don't need rythmbox) now go to Places->(Ipod (mini, nano)) something liek that. to find the path click on the "document+pen" icon and copy the path and paste it into the mount point field in amarok...
