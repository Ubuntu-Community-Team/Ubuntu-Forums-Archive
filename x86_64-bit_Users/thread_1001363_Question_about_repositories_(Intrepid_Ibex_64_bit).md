### Post by Ng Oon-Ee on 2008-12-03
The repositories are just that, repositories. The correct version for your x64 will be downloaded, for all programs. There aren't any separate 64-bit repositories and 32-bit repositories. However, some 3rd-party repos may be 32-bit only, you can still use them though if you manage to get the dependencies satisfied.
