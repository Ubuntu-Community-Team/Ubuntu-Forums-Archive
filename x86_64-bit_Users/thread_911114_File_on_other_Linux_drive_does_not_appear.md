### Post by Oldhacker on 2008-09-06
To every problem, there exists a back door solution. 
Unable to see the file to transfer on the drive which holds my other
Linux distribution (openSUSE 11.0) I went back into that distribution 
and decided to copy the file onto a CD-RW. Unfortunately, I had overlooked
the fact that files on a CD-RW become owned by root, and become read only.

Undaunted, I went back and extravagantly used up a CD to copy the file.
From there, I returned to UBUNTU and managed to copy the file where I wanted it to be.

I have a usable program running on UBUNTU now, but I would still like to
know why I couldn't see that file from UBUNTU.:confused:
