### Post by alex2399 on 2009-06-23
You are using ATI propietary drivers (FGLRX) and enabled Compiz(Desktop effects) probably. Maximizing and minimizing is very slow (1-2 seconds delay)while the rest is snappy and you use Ubutnu 9.04 (Jaunty Jackalope), right?
If this is the case, then there is nothing you can do, ATI FGLXR drivers just don't mix with a comforatble Linux desktop experience. The only thing that really works is disabling desktop effects and if you do not need 3d acceleration (which is unreliable at best) go back to the open source drivers. Furthermore it solves a lot of (for me atleast all) video display problems.

There is a PPA which circumvents these problems seemingly, but you get other problems for it in return as I have read. So if you wanna have a comfortable nightrest without feeling the need of running into brick walls the next day, pulling your hear out or so because it is more satisfying then getting these drivers to work decently then just ditch the FGLRX drivers till there finally is a suitable ATI driver in your lifetime.
