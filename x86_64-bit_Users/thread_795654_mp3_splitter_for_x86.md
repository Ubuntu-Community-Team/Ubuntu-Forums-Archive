### Post by Yellow Pasque on 2008-05-15
The first place you should search is Synaptic Package Manager. The Ubuntu/Debian repo is **HUGE**
mp3splt is provided. It's an excellent way to break up CBR and VBR mp3's without losing audio quality (as with Audacity). The only catch is that it's a command-line app, but don't let that scare you. IIRC, the manual is excellent.
