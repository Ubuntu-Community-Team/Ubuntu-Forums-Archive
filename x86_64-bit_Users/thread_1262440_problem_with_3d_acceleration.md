Yes, they use different 'drivers' so to speak. Make sure Ubuntu is completely updated, then reboot, and check : System > Administration  > Hardware Drivers. It should give you options to install one, then after that is 'green' enabled, restart again, then go to System > Preferences > Appearance > Visual Effects and enable them. 

That should do the trick, good luck. :KS
