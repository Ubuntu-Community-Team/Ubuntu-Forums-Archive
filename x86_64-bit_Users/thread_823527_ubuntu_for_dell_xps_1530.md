What do you mean with "certified"? Guaranteed by Dell? I don't think so. As far as I know, Dell does not sell the XPS M1530 with Ubuntu out-of-the-box, so there is no official Dell Ubuntu image for the M1530.

However, Ubuntu 8.04 works perfectly on the M1530. See the link in my signature.

Also see the [Dell Linux Wiki](http://linux.dell.com/wiki/index.php/Wiki_Main_Page) and the [Dell Ubuntu Support]("http://ubuntuforums.org/forumdisplay.php?f=342") forum.
