### Post by Kevbert on 2009-11-08
Please run memtest from the boot menu (it may be that the memory is poorly seated).  BIOS only performs a quick check as well as POST (when you first turn on the PC). What do you get with the terminal command
```
free -m
```
I get 3896Mb reported for my 4Gb of memory.
You may need to set memory remapping in the BIOS to use the memory hole.
