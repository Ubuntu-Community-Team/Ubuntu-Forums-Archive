### Post by brayden13 on 2009-08-02
On my 64-bit Ubuntu 9.04 the gnome sensors applet doesn't pick up my CPU temperature, only the hard drive temperature and GPU. I have a Core i7 920, and there is nothing wrong with the temperature sensors on it as I was able to get them readouts in Windows. But on ubuntu it just doesn't show. any reason why this might be happening?

---

### Post by 67GTA on 2009-08-02
You might try installing lm-sensors. Afterwards run ```
sudo sensors-detect 
```in a terminal. Make sure to answer yes to the last question about loading modules at boot. Reboot and see if it works.

---

### Post by brayden13 on 2009-08-02
Thanks, it worked fine. Now it shows the CPU, chipsets,voltages, fan speeds etc.
