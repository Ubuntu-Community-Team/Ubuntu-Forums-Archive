### Post by piratejake on 2009-01-12
During boot your BIOS should have a key that allows you to enter a Boot Menu (it's either ESC or DEL in my BIOS). From there you should be able to choose the HD containing vista and boot from there. Once in vista go to the msconfig (hit WINDOWSKEY+R and type in msconfig then hit enter). You should be able to select which OS boots first or give yourself the option of choosing under the BOOT.ini tab.
