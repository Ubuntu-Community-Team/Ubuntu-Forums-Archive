What Sef tried to explain is this. AMD64 is the name for the architecture of a 64bit cpu. It has nothing to do with AMD or Intel. AMD made the architecture when they first made dual core cpu's. Now that Intel has abandoned their EMT64 architecture, both AMD and Intel cpu's run AMD64 architecture. Get it? Basically what we are telling you is, if your Intel cpu is a dual-core (i.e. Core 2 Duo), it will run Ubuntu AMD64.

The next problem you ask has to do with graphics. Intel is a maker if intergraded graphics cards. AMD does not make graphics cards. The only graphics cards you are likely to run into are, Nvidia, ATI, and Intel. I believe you have a graphics problem but because your post lacks technical detail we can not tell.
