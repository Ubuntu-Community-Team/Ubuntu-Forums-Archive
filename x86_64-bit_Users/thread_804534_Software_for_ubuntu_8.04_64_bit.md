Do you mean beryl?  That is unsupported now.  Compiz-Fusion has replaced it, and it works just fine on my hardware.  However, it may or may not work well on your hardware.

If you really need a 32-bit program, then you can install most 32-bit programs on a 64-bit system with getlibs.  I have installed the 32-bit version of Firefox because currently Sun Java does not have a 64-bit plug-in.  One is due this summer.

### Post by jespdj on 2008-05-24
Almost all software that is available for 32-bit Ubuntu is also available for 64-bit Ubuntu. And if you find a program that's only available in 32-bit, then that's no problem, because you can run 32-bit software on 64-bit Ubuntu too.

See [http://packages.ubuntu.com](http://packages.ubuntu.com) to see which Ubuntu packages are available.
