### Post by peakshysteria on 2008-05-31
If you mean the dreaded so called ***awesomebar*** try:
Go to [about:config]("about:config") and set **browser.urlbar.maxRichResults** to **0**.

This will remove Awesomebar (the dropdown history bar). If you then *uncheck* history and *remember what i write in the search bar* you will no longer get the irritating dropdown every time you search for something.
