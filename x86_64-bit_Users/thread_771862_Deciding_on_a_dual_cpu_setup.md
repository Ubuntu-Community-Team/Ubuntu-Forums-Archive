### Post by koshcu on 2008-04-28
For dual processor boards especially if you want to use a lot of memory I would go with an nforce professional board. The Tyan S2915-E board is probably what you would want. You can run two quad cores opterons in it and it has 8 DIMM slots for ram.

I have the S2915 board with 2 dual core 2218 opterons in it and 8G of ram and it works great under ubuntu. I really like the S-2915 and S-2915-E boards because half of the hardware is hooked to each socket and you can get some very nice IO throughput on them. If you want to push your system hard that is what I would go for.
