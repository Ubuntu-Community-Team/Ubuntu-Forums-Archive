### Post by ctarwater on 2008-11-12
If you're running Gnome (as I am) and you installed kTorrent using the "Add/Remove" programs feature (as I tried) then it seems to crash a lot and eventually stop loading.

I found that if using apt-get instead it installs and runs perfectly.

Also, compiling 3.1.4 solved the problem as well.

Hope that helps.
