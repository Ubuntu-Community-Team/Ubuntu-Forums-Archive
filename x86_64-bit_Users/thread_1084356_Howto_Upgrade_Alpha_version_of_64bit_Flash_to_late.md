### Post by cariboo on 2009-03-02
It is pretty simple to install the updated plugin. I went [here]("http://abs.adobe.com/downloads/flashplayer10.html") and downloaded the new version, double clicked the file and used file-roller to extract it, then copied libflashplayer.so to /usr/lib/mozilla/plugins. Restarted Firefox and it's done.
