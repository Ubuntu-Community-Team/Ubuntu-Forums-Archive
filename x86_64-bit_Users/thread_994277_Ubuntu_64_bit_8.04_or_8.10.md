### Post by Unterseeboot_234 on 2008-11-28
For me the issue on what version has been centered around my nVidia card. The card worked fine in Drake (before Xorg, Compquiz, all of that) without any adjustment necessary. With 8.04, I had to use the thrid-party Envy to gain a usable Desktop size. Funny, the default desktop with LiveCD is just fine, but after install the desktop with the hidden buttons is an extreme pain.

Vers. 8.10 is a delight to install and get going. I did install on a clean partition 8.10 to gain the libraries necessary to build a drawing program I wanted, and then on reboot discovered that conflicted with my nVidia that was working on the other partition.

As far as I'm concerned, the only reason to upgrade is when particular software I want is unavailable for my current setup. The choice of libraries is a big structure like a model house made of sticks. Adding or removing a library in Linux can explode everything. That's why I really advocate Ubuntu -- 'buntu gives you the big picture on available software. IIWY I would go with the 8.10. Personally, I keep a partition of 7.10 just to run one video editing program, but I see myself migrating to 8.10 in two weeks for everything else.
