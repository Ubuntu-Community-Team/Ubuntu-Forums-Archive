### Post by saru411 on 2008-05-07
I've learned in the past that you shouldn't fix what ain't broke. If your 7.10 is running well then you have no real reason to upgrade to the newer version. normally I wait for the first update cycle to upgrade if I decide to upgrade to that kernel version. The best thing for you to do if you need a stable system for now would be to wait for the 1st major update in the summer and to search for your hardware on the forum/launch pad to check for possible bugs you might encounter. If you do decide to install the newest bestest best kernel be ready for setup issues. The bleeding edge can cut you......AND ALWAYS USE BACK-UPS!
