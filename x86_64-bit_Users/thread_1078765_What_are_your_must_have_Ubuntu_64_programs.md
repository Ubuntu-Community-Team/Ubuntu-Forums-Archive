### Post by Yellow Pasque on 2009-02-23
See the sticky in this subforum on Flash if you have issues with the one provided by the Ubuntu repos.

You can also install Sun's version of Java (which includes a 64-bit browser plugin), but again, I'd recommend sticking with the Java that Ubuntu provides (Openjdk) unless you experience issues with Java web apps in your browser.

Some closed-source/proprietary programs only have 32-bit versions. If you come across these, the first thing you should do is search this subforum to see if other users have found solutions on making them work. Most 32-bit programs can be made to run with the help of getlibs: [http://ubuntuforums.org/showthread.php?t=474790](http://ubuntuforums.org/showthread.php?t=474790)
