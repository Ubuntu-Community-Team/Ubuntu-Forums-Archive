### Post by Cappy on 2008-06-16
Gnome doesn't show the correct refresh rate for nvidia cards, however you can see it with
```
gksudo nvidia-settings
```

I would backup your xorg.conf before you decide to change anything, just in case
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.changing_refresh
```

You'll want to goto "Screens and Graphics Preferences" (that's what it is on Gutsy anyway) and set your monitor type, such as 1280x800 LCD.

You may also need to run
```
sudo nvidia-xconfig
```
and press Alt+Control+Backspace which will end any programs you have running and restart X before you can use nvidia-settings.

### Post by Xyem on 2008-09-13
Chances are that you've either given up or fixed it but might this be where you have to add

Option "DynamicTwinView" "false"

in the "Device" section to get the settings dialog to give you the true refresh rate?

The above was from memory but it will give you an idea what to look for ( for example, without that option, my settings say "61Hz" when it is in fact at "75Hz", adding that line corrects it ).
