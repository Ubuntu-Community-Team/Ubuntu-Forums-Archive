### Post by Forrest Gumpp on 2008-02-26
When I entered this topic in the Title pane of the Post New Thread page the first message I encountered was "Notice: When you enter a thread title the system will search for similar threads which have already been posted. That should help you to find answers.".

I then clicked in the Message pane.  After a short delay, several potentially similar thread titles displayed.  A second message saying "Please take a look at those threads before you start a new one." also displayed.  I moused over all to get as much indication as possible about each before selecting one to read.

Selecting one took me to the thread, which I read.  Not what I was trying to find out.  Back to the previous screen.

Upon arriving back at the Post New Thread page, all the other choices were gone.  My own proposed title was still in the pane, and clicking within the message pane did not get the other titles back.

Is this only a one shot feature, or is there some not-so-obvious way of getting the other possibly similar topics back for checking before posting?
