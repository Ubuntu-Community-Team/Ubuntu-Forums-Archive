### Post by Jason Kuzhively on 2013-05-02
Hey Guys,

The Signature is the perfect place to give pointers for your work but long links are ugly and unappealing. I'll tell you how to turn *any *site address into a Hyper Link with the actual address hidden. Many people already know this but some newbies don't so I decided to post this here.

For this example, I'll use the link of Ubuntu Forums.
If I wanted to say "Visit UbuntuForums", my Hyper-Link would be: [Visit UbuntuForums]("http://ubuntuforums.org")

Now, the question is, How to do that?
The answer is pretty simple, here's the dummy format:

[url= "the site address"]"message"[ / url]

Just use the above format; replace "the site address" with the link and replace the "Message" with what should you want it to say, that's it.

Note: Make sure that there are NO spaces in the URL commands, but you can have spaces in the "Message".

Note II: After you type up the Hyper-Link code, when you type something else, if it also starts acting like a Hyper-Link, then use a couple of spaces, i.e. after typing up the URL command, insert some spaces *after* the command. This can also be used in your posts.[URL="http://ubuntuforums.org"]


[/URL]
