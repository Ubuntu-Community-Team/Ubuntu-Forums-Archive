### Post by coffeecat on 2014-10-18
I'm not sure why you would post this in New to Ubuntu but, although you are asking about logging into Launchpad, I've moved this to Forum Feedback & Help because it affects ubuntuforums as well.

When dealing with login queries, I tend to refer to Ubuntu One as Ubuntu One SSO - Single Sign On - as a way of emphasising its function, although I am not sure whether this is an "official" description or not. The Ubuntu One SSO login page:

[https://login.ubuntu.com/](https://login.ubuntu.com/)

Does not mention SSO, but at the bottom of the page it says:

> Ubuntu One. One place to log in to everything on Ubuntu.

All of the Canonical-funded Ubuntu sites that I can think of use Ubuntu One for sign in. Askubuntu slightly confuses things - for me at least - by asking you to login via Launchpad, and then takes you to the Ubuntu One login page anyway.
