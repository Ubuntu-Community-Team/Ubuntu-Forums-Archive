### Post by cariboo on 2012-10-31
It's perk only for staff members, and even we have to be discreet when it comes to images in signatures.

Historically images in signatures have not been allowed, and my personal feeling is that images in signatures can be very disruptive to page flow, and in many cases confusing to other users.

From the forum code of conduct:

> Signatures: Forum signatures are limited to four lines of text, 10pt maximum font size. Signature content must conform to the same guidelines as all content in this code of conduct.

If you forgot the rules, you can read the Code of Conduct [here]("http://ubuntuforums.org/index.php?page=policy")
