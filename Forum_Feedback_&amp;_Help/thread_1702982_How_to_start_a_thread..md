You've already started a thread on the forum: that's a start. Starting a support request is similar.

[LIST=1]
[*]Go to [http://ubuntuforums.org](http://ubuntuforums.org)
[*]Choose one of the headings that covers the area you're interested in, and click on it.
[*]Click on the "New Thread" button (illustrated earlier in the thread.)
[/LIST]
Keep in mind, however, that some sections (e.g. the archive) don't have an option to start a new thread.
