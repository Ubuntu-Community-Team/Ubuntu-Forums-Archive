### Post by Elfy on 2014-10-10
Attachments jpeg/jpg/png all have the same file size limit and image size limit.

I've just added a .png that's 175kB - it's changed it to .jpg

I've added a .png that's 38.5kB - it's left it as .png

All set to 1000000 bytes.

So I would say from that - file size too big - expect a change to .jpg
