### Post by matthew on 2006-07-12
Hi, guys and gals!

We have had some great input regarding the new forum theme. Thank you to all who have given suggestions and ideas. Your input has been helpful. At the moment we are feeling a bit overwhelmed by all the detail work that has come up as a result of the forum software upgrade and the subsequent need for a completely new theme. 

For those who missed it, what happened was we first upgraded the operating system the forum server is running on to Ubuntu 6.06, then we upgraded the actual forum software to vBulletin Version 3.6.0, and now we are in the process of making a totally new theme for the forums, using what we can from the old theme, but making changes along the way--partly out of necessity because the old themes are not compatable with the new vBulletin version, partly to bring the forums look more in line with the main Ubuntu web site, and partly to do a few customizations along the way. We have also had some pretty typical bumps and surprises along the way.

We want to say thank you to everyone who has given suggestions. At this point we are going to ask that everyone please refrain from commenting further on the forums' look, theme, and functionality for the moment with one exception: please continue to report bugs and glitches [in this thread]("http://www.ubuntuforums.org/showthread.php?t=211210") as/if you find them. All other forum upgrade related threads are going to be closed starting now with a referral to this post. 

Please understand, we are not doing this because your input isn't valuable, it is simply that we are overwhelmed with data and suggestions (and a few complaints...) and need some time to let the dust begin to settle. We may open things up again for comment in the future. If that happens you will see a new thread in this forum making the request. Thank you all for your great patience and understanding.
