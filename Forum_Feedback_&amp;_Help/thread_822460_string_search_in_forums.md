### Post by nhandler on 2008-06-09
I also use google to search for a specific string on the forums. However, I use a special query to restrict the results to ubuntuforums.org

> site:ubuntuforums.org "STRING TO SEARCH FOR"


The results aren't as up-to-date as the normal forum search, but I still tend to get good results.
