### Post by lisati on 2012-11-30
The usual approach for "regular" users here is to mark the thread as "solved" using the "Thread tools" drop-down menu, which doesn't prevent new replies in situations when you might want to come back with extra information. It also changes the thread title. This is something different to actually closing/locking threads.

When an admin or staff member closes a thread here, the usual "New reply" button gets replaced with a "Thread closed" button, further replies are blocked, and a "locked" symbol shows up: [IMG]http://ubuntuforums.org/images/rebrand/statusicon/thread_hot_lock.gif[/IMG]

When I move a thread, I usually post a reply in the thread stating that this is what I've done.
