No need to use github at all the stylus add on is in the chrome web store where you get any extensions for chrome/chromium from.

Once installed you will need to go to userstyles.org (or other sites like it) and install some themes for ubuntuforums.org.
 Then when you are on ubuntuforums you can use the stylus icon on your browser tool bar to set a different theme.
The new theme when activated will only apply to the ubuntuforums site not any other site you may then visit.
