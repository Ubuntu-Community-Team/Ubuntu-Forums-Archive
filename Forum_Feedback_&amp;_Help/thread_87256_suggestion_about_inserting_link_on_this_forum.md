### Post by 23meg on 2005-11-07
It's not an error; in the textbox that appears, http:// is already there, so when you paste a link with an http:// already in it you're inserting two instances. The first http:// appears in the box so that you can easily type in a URL without having to type http:// ; if you'll paste something and don't want it, you should double click the text box so that the URL will replace it.
