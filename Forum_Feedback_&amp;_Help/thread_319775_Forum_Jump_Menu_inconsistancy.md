Issue:
When one navigates to any forum main page where a table of individual threads are listed, the Forum Jump menu is displayed in it's own table <table align="center"> on the left side of the browser window, while when one views an individual thread the Forum Jump menu is displayed in a nested table <td class="smallfont" align="right"> on the right side of the browser window.

IMHO it is best to have all "static" content displayed the same way on all site pages.  (static content in this context meaning elements that are used on all pages, such as title images, menus, etc).  It makes it faster for users to navigate the site and one need not "remember" what kind of page one is currently viewing to know where the Jump Menu is on that page.  It should be constant on all pages, very easy to implement.
