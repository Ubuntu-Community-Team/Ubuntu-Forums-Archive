### Post by naaman on 2006-01-27
Hello all,

The Ubuntu Document Storage Facility (UDSF) was raised again as a point of discussion at the last Ubuntu Community Council that had taken place on the 24th of January 2006.  The main outcome of the dialogue was that there is a need for a review of all the documentation resources available to the Ubuntu users community.

The best way to conduct the review is to invite the main interested parties who are driving the current documentation resources to an #ubuntu-meeting.  Having all parties at the same place will allow for open discussion and allow agenda items to be raised in an orderly fashion.  Discussion, not posting, allows for all sides to be heard in a more formal setting.

The agenda for the meeting is available, for all to edit, on the Ubuntu Wiki - [https://wiki.ubuntu.com/UDSF/MeetingAgenda](https://wiki.ubuntu.com/UDSF/MeetingAgenda)


Naaman Campbell,
Ubuntu Documentation Team.
