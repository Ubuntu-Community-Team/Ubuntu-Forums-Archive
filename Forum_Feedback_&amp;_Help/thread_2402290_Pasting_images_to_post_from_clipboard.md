### Post by oldos2er on 2018-09-28
If all you are posting is text, couldn't you simply copy and paste and put it all inside [code tags]("https://ubuntuforums.org/showthread.php?t=2171721&p=12776168#post12776168")? ### Post by ajgreeny on 2018-09-28
And if you're really talking about images, ie pictures, that you want to include in the text of a post here, please do not do it that way.

Images should be added using the Attachment icon paperclip, in the "Reply to Thread" box.
If you want text please do as oldos2er says and do not post an image of the text; it is impossible to copy any of that in order to carry out searches.
