### Post by yetimon_64 on 2019-09-23
On the forum itself there are only 2 themes available "Ubuntu vB4.x" and "Ubuntu mobile style". They are available from the bottom of your browser window; no dark themes are provided unfortunately. Assuming those two don't suit you then the other option would be with a browser add on.

If you use chrome/chromium you may be able to change the theme here with the add on "stylus".
 I'm not sure about Firefox anymore, I used to use the stylish add-on with it, but apparently it was removed by both Google and Mozilla for collecting user data in a way that could be identifying. [[COLOR=#0000ff]--Here--[/COLOR]]("https://ubuntuforums.org/showthread.php?t=2400684") is a discussion from this forum from about a year ago that may give you some further ideas.

I doubt it will happen, but it sure would be nice to have more options added to the bottom of the forum window's drop down menu for themes; even only 1 more "dark" theme would make the site much easier on the eyes.

Regards, yeti.
