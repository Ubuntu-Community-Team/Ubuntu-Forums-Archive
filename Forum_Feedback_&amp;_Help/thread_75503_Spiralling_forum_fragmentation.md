### Post by kassetra on 2005-10-13
Previously, the Kubuntu area was jam-packed with information all over the place, and it was difficult to find help or even information.

We have made them mirrors because while the underlying code is the same, they have very different ways of handling user interactions.

The games are split because installing games on either platform can be very different.

If you use both KDE and Gnome then yes you would need to read both; if you only use one you would only need to read one.

We are comfortable with the layout changes we have made for now, but if we see a need to change them we will.
