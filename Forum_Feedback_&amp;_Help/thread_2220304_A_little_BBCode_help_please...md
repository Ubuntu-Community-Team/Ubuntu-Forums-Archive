### Post by coffeecat on 2014-04-27
Just to add:

It's best if you have your forum text editor in source mode when typing up a post with BBCode - not WYSIWYG mode. You can do this by clicking on the message editor toolbar icon that looks a bit like A/A to switch the editor into source mode if necessary. Or for making this the default, go to settings -> General Settings -> Message Editor Interface, and select Standard Editor. Then, before you post, be sure to click on the Preview Post button to make sure it has come out right.

As a bonus, click on the Reply With Quote button under bapoumpa's or Iowan's post to view how the noparse tag is used. Make sure your editor is in source mode otherwise you won't see all the BBCode between the quote tags.
