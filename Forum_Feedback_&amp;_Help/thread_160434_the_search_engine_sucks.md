### Post by 5-HT on 2006-04-14
[quote=jpazindustries]how do i go into advanced search?[/quote] 
If you have javascript enabled, you can just click on the search link (near the middle in the 'Forum Navigation' section at the top of the page) and it'll be the second option on the little menu that pops up.

If you don't have javascript enabled, clicking on that search link will bring you to the advanced search page. 

To only search thread titles, select that option in the pull-down menu just below the keyword input box on the advanced search page.
