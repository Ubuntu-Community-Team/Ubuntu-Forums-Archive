### Post by LaRoza on 2008-01-30
If it doesn't have real benefit to the community, I suggest not making a howto. Perhaps your own personal blog would be better for cool, yet not entirely useful, ideas.

If this is something that is not illegal, and a possible security or monitoring tool for your own networks, it may have merit. At the very least, you could post it in the Cafe or Programming Talk if it is a program you developed.
