As many of you know, the difficulties we had been having with the performance of the Ubuntu Forums have been very nearly fully resolved. Canonical significantly upgraded the server hardware used to host the Forums and we have enjoyed nearly problem free operation since. These things happen. Please bear with us for a short time longer!

Thanks!
