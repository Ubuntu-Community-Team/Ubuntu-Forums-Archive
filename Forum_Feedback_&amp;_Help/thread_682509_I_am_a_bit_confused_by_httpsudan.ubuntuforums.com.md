### Post by matthew on 2008-01-30
It's an automatically created pointer to the main site. You can do it with just about anything...I just tested matthew.ubuntuforums.org and it works the same. I would imagine ravtux.ubuntuforums.org would work the same as well.

It would probably NOT work with the forum cookie on your machine, so you could end up with login difficulties if you use an address like that, just like if you are using ubuntu-forums.org or any of the other pointers that exist.

It is best to stick with the main address, ubuntuforums.org... but now you know what is up with the <name>.ubuntuforums.org stuff. :)
