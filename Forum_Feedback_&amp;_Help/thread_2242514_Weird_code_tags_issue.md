### Post by coffeecat on 2014-09-02
A useful tip if you are using BBCode in any of your posts is **not** to use WYSIWYG mode in the message editor. WYSIWYG mode simulates the effect that any BBCode has rather than showing the actual BBCode, and if any of your BBCode tags get misplaced, it's a nightmare to sort out. If you use a source mode in the message editor, you can see the BBCode, and then use the "preview post" button in the advanced editor to preview your post before posting it to make sure it does look as you intended.

Two ways to change from WYSIWYG editor mode:

One time only - click on the icon in the message editor toolbar that looks something like [SIZE=1]A[/SIZE]/_A_ to switch to source mode. 

To set your preference to default - click on Settings (top-right) -> General Settings -> Message Editor Interface. Choose either basic or standard. I prefer standard.
