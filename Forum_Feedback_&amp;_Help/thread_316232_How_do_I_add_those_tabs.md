### Post by Soarer on 2006-12-31
It looks to me like it's done in CSS, but it may use a Content Management System (CMS) like Mambo or Joomla! and PHP.

If you right-click on the page, you get a menu. 

Select 'View Page Source' and, guess what, you can see the HTML of the page.

Select 'View Page Info' , then the 'Links' tab and open the CSS file ([http://www.ubuntu.co.in/style.css)](http://www.ubuntu.co.in/style.css)). All the details are in there (but are probably copyright to the site owner/designer so can't be used without permission).

There is an FF extent ion that helps with all this called 'Web Developer' : [https://addons.mozilla.org/firefox/60/](https://addons.mozilla.org/firefox/60/)
