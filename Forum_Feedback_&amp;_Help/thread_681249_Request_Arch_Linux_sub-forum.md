### Post by Rumor on 2008-01-28
Greetings:

I am posting to request a sub-forum for Arch Linux in the Other OS Talk forum panel. As a former Ubuntu user turned Arch Linux user, I think such a sub-forum would be very beneficial.

The current Arch Linux Talk thread has grown to the point where it is no longer a useful thing for those who are curious about Arch. At close to 950 posts, it is not something any person is going to read through page by page.

There is a fairly active group of Arch users who began their journey into discovering Linux through using Ubuntu. Some of them dabble in other distros, some throw themselves in full scale. A new user would find it a daunting task to search for help using Arch by going to the above mentioned Arch linux Talk thread.

I am certainly not the only Arch user who still frequents the Ubuntu forums offering help to those who ask questions. The sheer size of the Arch Linux Talk thread has brought about a large number of repeated questions . . . how do I Install this? What do I do about that? What are the benefits of Arch, etc., etc. These questions and others have been answered several times within this thread, but it is simply either easier to just ask them anew, or it is too difficult to search the thread for answers to questions that may or may not be what a curious user is looking for.

I don't want to see the Arch Linux Talk thread degenerate into a place where these questions are greeted with snarled answers, "We've already answered that in post 126, 231, and 480!" or such the like. Such answers go against the helpfulness of the ubuntu forums and should not be encouraged. 

In an Arch Linux sub-forum, a faq or some general answers could become sticky posts to assist the curious user in quickly finding the information they seek.

As you are aware, Arch is different, not based on Slackware, Debian or Red Hat, and offers a very good learning experience for those who wish to know more about the nuts and bolts of Linux and how things work. The current thread is more a hinderance than an assistance in helping those who view the Ubuntu forums as "home" and are more comforatable seeking assistance here than in the Arch forums where they don't "know" anybody. I think it would be beneficial to those users to have a place here in the Ubuntu forums community where they could ask their questions, without those questions becoming buried in a gargantuan thread.

That there *is* interest in Arch is evidenced by the daily activity in this thread. Some days have seen a dozen or more posts to it and some conversations quickly become lost with the other posts piling on. The thread has grown to the point that were it not for the fact that I subscribe to it, I'd be hard pressed to find where the new posts begin since my last reading.

So I ask you to please consider this request to create an Arch Linux sub-forum in the interest of furthering the friendly, helpful community that makes up the Ubuntu forums.
