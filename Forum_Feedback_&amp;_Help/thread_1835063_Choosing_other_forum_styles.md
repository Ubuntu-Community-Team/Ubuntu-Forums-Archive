### Post by TeoBigusGeekus on 2011-08-29
If you're using Opera, extract this CSS style and put it in ~/.opera/styles/user.
Then right click anywhere in the startup page of ubuntuforums, Edit Site Preferences>Display>My Style Sheet and choose the css style you've just extracted.
Reload the page and you're done.
