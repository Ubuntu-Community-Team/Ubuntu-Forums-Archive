### Post by coffeecat on 2013-09-24
If you are talking about editing problems with the forum message editor, then I would suggest switching to Source Mode or one of the non-WYSIWYG modes. I don't follow your description of the cursor problem, but not being able to see the BBCode for bold text (or other BBCode) can cause confusion when editing a post. I long ago abandoned the WYSIWYG editor as being more trouble than it's worth, and more likely to lead to the cursor problems you describe.

Try one or both of two things:

In the advanced editor click on the button top left of the editor toolbar that looks something like [SIZE=1]A[/SIZE]/_A_ to toggle between WYSIWYG and source mode. My guess is that you are using WYSIWYG. Source mode will let you see the BBCode. It doesn't preview how the text will look as you type but at least you get to see if you've put the cursor in the right place when you are editing, and you can use the preview post button to make sure your formatting looks OK. Or...

Settings -> General Settings -> Message Editor Interface. I find "Standard Editor - Extra formatting controls" to be the most usable but personal preferences vary.
