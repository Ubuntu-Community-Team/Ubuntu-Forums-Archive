### Post by Woodgar on 2006-08-29
What are your cookie settings in firefox?

I have mine set to "allow sites to set cookies", but I have the "keep cookies" option set to "until I close firefox". This means that I can avoid the nagging from sites that want to set a cookie before they let me in, but I ensure that the cookie is deleted when I close firefox rather than when the cookie expiration date is met. 

To ensure that forum sites work correctly, ie I'm not forced to log in every time I visit, I manually set up exceptions in firefox to allow the cookies from the forum sites to remain. Sometimes this doesn't always work, as you are experiencing, but it usually turns out that I have set the wrong domain name in the exceptions list. Maybe this is something you could check as well.

Good luck. :)
