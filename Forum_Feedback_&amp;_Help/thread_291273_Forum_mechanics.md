### Post by PriceChild on 2006-11-02
If you use the advanced editor, the interface with all the buttons appears, where you may highligh text and press the "code" button etc.

Alternatively, you can do this anywhere on the forums:
[ code ]this is code[ code ] (without the spaces) becomes:```
this is code
```&

[ quote ]this is a quote[ /quote ] (without the spaces) becomes > this is a quote

Pressing the quote button on a post will open up a new reply for you, with the quote inside, and a ready made link so that you can press a button on your quote to go to the original post.
