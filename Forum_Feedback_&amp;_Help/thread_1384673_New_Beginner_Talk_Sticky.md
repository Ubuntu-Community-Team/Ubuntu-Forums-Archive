### Post by presence1960 on 2010-01-18
I don't suppose it would hurt to create a sticky for that topic. My view is that the sticky will not really help though because if these "new" green linux users will not listen to the people who are successfully using linux & taking their personal time to try to help then the ones who don't like terminal still won't like it.

If someone doesn't like Linux go back to the OS you used previously, but remember the reason you took a look at linux in the first place.

Sometimes I feel that we in the community believe that we are supposed to overtake windows and make linux be for everyone. ***_Linux is not for everyone_*** and if it ever achieves that dubious distinction it will be at the expense of the things that are it's greatest strengths. Those things BTW are the things that set it apart from windows in a stark contrast of how tasks are accomplished.

Some of us dual boot or even multiboot, some of us dislike or even hate windows. But let's not get caught up in the pseudo-movement or belief that Linux has got to overtake windows. If that ever does happen it will happen without any input by us. Our repeated efforts to entice windows users by making linux more windows like is going to have negative or even dire effects in the long run. Who are we trying to impress? This is not a business venture.
