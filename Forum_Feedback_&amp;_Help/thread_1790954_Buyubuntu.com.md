### Post by lisati on 2011-06-26
Ubuntu is an operating system, i.e. its a collection of programs that are intended to help you run your computer. To install a new one, particularly one with which you are unfamiliar, is a big step.

As papagray suggested, you might want to try booting from the CD, and selecting the "Try Ubuntu" option before you get anywhere near installing Ubuntu. Although it will probably run a bit more slowly off CD than if it was installed, this will help you get some idea if it's for you and also if it will work with your hardware.

If you want some help putting the ISO file you downloaded (Windows is sometimes kind enough to 'pretend' it is another kind of file, which can confuse newcomers), you might want to have a look here: [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

If you decide to take the plunge and actually install Ubuntu, don't forget to make backups of important data (including recovery partitions) - we will try to answer any further questions you might have.
