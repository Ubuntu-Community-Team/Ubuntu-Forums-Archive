Yes "Wubi" should be the name, the project is in reality composed of 4 subprojects, but vast majority of our users will come from the windows world and "wubi" will be the name they will be familiar with. Since the main objective is to make it as braindead as possible for them to install and use Ubuntu, the main forum should be called "Wubi". As for the parenthetical note, it will be disclosed later in a marketing stunt, so for now we should just have "wubi".

The backend is called lupin (loop-installer) and you can see the launchpad page here: [https://launchpad.net/lupin/](https://launchpad.net/lupin/)

The windows frontend is called, surprise, surpise... ..."wubi": [https://launchpad.net/wubi](https://launchpad.net/wubi) There is not much on launchpad yet for wubi itself, but it will be filled up later on. 

Later on there will be a linux and a mac frontend.

The original project pages are: 
[https://blueprints.launchpad.net/ubuntu/+spec/windows-installer](https://blueprints.launchpad.net/ubuntu/+spec/windows-installer)
