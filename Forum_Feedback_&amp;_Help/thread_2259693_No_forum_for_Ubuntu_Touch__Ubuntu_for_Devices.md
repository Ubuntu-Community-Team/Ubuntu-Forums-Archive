### Post by I.Bun.Tu on 2015-01-06
I have quite a difficult time getting support for Ubuntu Touch or Ubuntu for devices, whatever it is called. I have it installed on a Nexus 4, which is officially supported according to Canonical's website, but where is this support? So far, the best place I can seem to find is the xda developers forum.

If Canonical actually hopes to succeed in bringing Ubuntu to mobile devices there needs to be a convenient and obvious place for people to ask for help and get support.

Canonical suggests that Ubuntu should have core functionality stable and usable for the supported Nexus devices but I haven't even been able to connect to my mobile network for over a week. And I don't mean for data, I can't even make calls. Good thing I didn't throw out my old phone, but I am using the latest 'stable' version or the release to market version and it is beyond buggy. I wouldn't mind so much if there was somewhere in the Ubuntu forums I could actually get some troubleshooting help though.

As it stands, if I post for help in the General Help forum, my thread will get buried within an hour or two. There needs to be a support forum dedicated to Ubuntu for devices or Ubuntu Touch, whatever it is called (personally I think it needs a specific identifying name like Ubuntu Touch in order to be able to easily google/query information for Ubuntu for phones/tablets as opposed to desktops).

What are your guy's thoughts? I love Ubuntu, and in having to revert back to android for the last week, its making me realize even more how great the Ubuntu OS is on phones, but we need support and at this point I feel the ball is in the court of the Ubuntu Forums.
