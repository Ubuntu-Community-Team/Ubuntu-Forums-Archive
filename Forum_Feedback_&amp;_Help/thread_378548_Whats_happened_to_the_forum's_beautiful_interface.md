I love the effort, I really do, but this is rather backwards. If this is unfinished, why not have it as an alternate forum theme that is not default? I believe we can change themes via preferences, so the experimenting could be done live via an optional "testing" theme, thus not giving Ubuntuforums.org an ugly look for the time that the testing takes place.

### Post by reacocard on 2007-03-17
For those whose fonts are still small, I have a fix!

Run
```
sudo dpkg-reconfigure fontconfig-config
```
and select 'Native', 'Always' and 'No'. (the last two might be unnecessary, the first one is essential.)
Then do
```
sudo dpkg-reconfigure fontconfig
```
and finally restart firefox. Fonts should be normal size again.
