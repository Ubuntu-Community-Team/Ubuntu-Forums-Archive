### Post by az on 2006-08-22
A forum site is database-driven.  When you click on a page, the html code is dynamically created by the server.  It uses the LAMP stack.

[https://help.ubuntu.com/community/ApacheMySQLPHP](https://help.ubuntu.com/community/ApacheMySQLPHP)

LAMP is usually Linux, Apache, Mysql and PHP, but it has also been called "Linux, Apache2, Many different languaged which begin with "P" (Python, Perl, PHP) and Postgresql".  The point is that there is a web server which uses a database for content and the programming language is what is used to make it all work.

Here are some examples of other forum software you can run on your ubuntu system:

[https://help.ubuntu.com/community/PhpBB2](https://help.ubuntu.com/community/PhpBB2)

[https://help.ubuntu.com/community/PunBB](https://help.ubuntu.com/community/PunBB)

Other web applications:
[https://help.ubuntu.com/community/Servers#head-ed0a847767fb61cd99eed5e6b641c2ccf11cdea4](https://help.ubuntu.com/community/Servers#head-ed0a847767fb61cd99eed5e6b641c2ccf11cdea4)

A forum is also successful or a failure dependant on the community around it.  Ubuntu and the spirit of community which it encourages tends to attract good, friendly and helpful people.
