### Post by tashirosgt on 2009-06-22
I find that the "Username" field for logging into to theforum works fine with Firefox provided I use the default font.  However if I pick the Firefox "View" option and increase the font size, the "Username" field becomes unresponsive and I can't enter anything in it.  On a moderately larger font, the "Search" field is misplaced and it covers the username field.

### Post by jenkinbr on 2009-06-22
This is a known issue that has appeared in many other FF&H threads.  The current fix is to use the default font for logging in, or log in at the user control panel ([http://ubuntuforums.org/usercp.php](http://ubuntuforums.org/usercp.php)) *the latter will place the login fields in the middle of the page, which sometimes solves the problem)
