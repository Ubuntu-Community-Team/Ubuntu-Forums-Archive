### Post by coffeecat on 2014-09-27
Link to thread: [http://ubuntuforums.org/showthread.php?t=2245916](http://ubuntuforums.org/showthread.php?t=2245916)

The solved prefix is set either by choosing it in the prefix field when you first create the thread, or by editing your OP and going to advanced editor mode, or from the thread tools drop-down menu at the top of the thread. I can't tell which you did but since only forum staff and the OP can change the prefix on a thread, and since there is no record of any staff activity on that thread, you must have done one of the three. Perhaps it was an accidental mouse movement that set the solved prefix in this case. Easily done.

Have a look in the thread tools drop-down and see if you can unset the solved prefix. I'm not 100% sure whether you can because admin accounts see certain things differently from other users. Or try edit post and go to advanced mode. If neither of those work, post back, and I'll remove the solved prefix for you. If this or something similar happens again with another thread, you are welcome to use the report post button on your post to send a request to the staff area.
