### Post by COKEDUDE on 2010-12-11
You gotta search for whatever you wanna see then add this to the end:
**&pp=100&page=1**

For example I have these 2 links. 
```
http://ubuntuforums.org/search.php?searchid=78003315
http://ubuntuforums.org/search.php?searchid=78007671
```

They would turn into this. 
```
http://ubuntuforums.org/search.php?searchid=78003315&pp=100&page=1
http://ubuntuforums.org/search.php?searchid=78007671&pp=100&page=1
```

You can't just copy my links or you will get a message that says: 
Sorry - no matches. Please try some different terms.
