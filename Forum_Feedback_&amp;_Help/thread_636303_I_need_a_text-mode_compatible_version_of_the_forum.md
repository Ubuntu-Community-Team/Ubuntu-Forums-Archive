### Post by matthew on 2007-12-09
lynx is pretty old. I believe the default install includes links2, another text browser that I just tested and can confirm to work with the forums. You might try it.

To run from a terminal, type:

```
links2 http://ubuntuforums.org
```

and you should be all set.

If it isn't installed, just use

```
sudo apt-get install links2
```
