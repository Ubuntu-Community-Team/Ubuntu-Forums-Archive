### Post by deadflowr on 2020-10-01
You mean how to set attachments?
When posting a new post, any reply to thread option except the Quick Reply option should have an advanced toolbar in the editor.
One option is the paperclip icon.
That's the attachment utility.


Basically when inside the attachments utility you want to click on the Add file in the top right corner and then select the file you want from your computer and select it to  upload it and it will automatically get attached to the post.
I think you can attach up to 5 images per post.

You can see a quick list of max file sizes and what file types are acceptable clicking a little blu(ish) question mark icon right above the Add file button in the attachments utility.
most image file types have a max of 976KB.

Hope that make sense.
