### Post by coffeecat on 2018-04-29
You do not state, but my guess is that the bad bot message is being delivered by the Ubuntu One site. Have a look through this thread:

[https://ubuntuforums.org/showthread.php?t=2273359](https://ubuntuforums.org/showthread.php?t=2273359)

Is the page that is giving you the bad bot message similar to or the same as the screenshots in posts #11 and #12 of that thread? Specifically, does the URL in the address bar include "login.ubuntu.com"? If so, you would be best advised to contact the Ubuntu One admins about this, as we do not administer Ubuntu One. Here is a link for you:

[https://forms.canonical.com/sso-support/](https://forms.canonical.com/sso-support/)

If you do use that form, you need to provide comprehensive information, far more than you have done so here so far, for the Ubuntu One admins to be able to help you. In the meantime, you can do some of your own troubleshooting. What browser extensions are you using in Chrome that you are not using in Firefox, and which might be the cause of this? Interestingly, in the older thread, the problem was seen with Firefox. Also, are you using the LastPass plugin, or other password manager, in Chrome? According to this AskUbuntu thread, that could be the problem:

[https://askubuntu.com/questions/650544/login-launchpad-net-tells-me-bad-bot-go-away-request-aborted](https://askubuntu.com/questions/650544/login-launchpad-net-tells-me-bad-bot-go-away-request-aborted)

Although that refers to the launchpad site rather than Ubuntu One, it is likely to be relevant.
