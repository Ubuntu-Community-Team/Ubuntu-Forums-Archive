### Post by lovinglinux on 2010-07-13
You can make the web the way you like with Stylish.

[Stylish for Firefox](https://addons.mozilla.org/en-US/firefox/addon/2108/)
[Stylish for Google Chrome](https://chrome.google.com/extensions/detail/fjnbnpbmkenffdnngjfgmeleoegfcffe)

Install the extension, download the attached archive, extract it, open the file with a text editor, copy the contents and paste on a new Stylish script.  

This is an incomplete version, but should be enough to get you started.

[IMG]http://ubuntuforums.org/attachment.php?attachmentid=163271&stc=1&d=1279003697[/IMG]
