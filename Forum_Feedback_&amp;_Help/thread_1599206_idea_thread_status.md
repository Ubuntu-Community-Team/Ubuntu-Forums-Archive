### Post by bprins on 2010-10-17
Hi,

Would it be helpful to extend a thread with a "status" field, so that forum users are able to see if a thread discussion is new, open, probably solved, or solved.

Now I often find myself reading threads with the intention to help out, and find that somebody else already gave the answer. In order to find that out, I have to read all posts which can sometimes be time consuming.

In particular the status "probably solved" would help, so that I can see that somebody else gave a possible solution. The starter of the thread can then accept or decline the proposed solution bringing the thread to the state "solved" or "open". Having that feature I can search on the "open" status and see if I can help other users, instead of digging through the threads and trying to find open issues.

Would this help others too, or just me?

Regards,

Bas
