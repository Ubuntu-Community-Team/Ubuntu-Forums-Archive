### Post by mattheweast on 2006-12-16
The forum theme is looking really good at the moment. One thing - you can't see links in the brown bar near the top of each page, unless you mouse-over them.

For example, on the Feisty forum ([http://ubuntuforums.org/forumdisplay.php?f=179)](http://ubuntuforums.org/forumdisplay.php?f=179)), you can't see that there is a link there.
> 
Feisty Fawn Discussions Discuss the development of Feisty Fawn here.
Please note developers are not very active here. If you wish to file a bug report please do so here.

In that quote, the word "here" is a link.

Also, if you thought that was a minor complaint, check this one out - it's a bad idea to use the word "here" as link text; instead you should consider using some descriptive text, such as "please do so on the Ubuntu bug tracker".
