If you see post the violate the COC please use the report button. Thanks :)

### Post by Elfy on 2011-07-29
IF the only post is a LMGTFY or similar then it's likely to be looked at askance.

IF the post has help and refers to the OP's original request - and then goes on to give a link to a google search it'll be looked at somewhat differently - in that case it's showing people that the results are out there.
