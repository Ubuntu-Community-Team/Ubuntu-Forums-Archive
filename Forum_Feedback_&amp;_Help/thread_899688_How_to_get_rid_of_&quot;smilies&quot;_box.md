### Post by bthoward on 2008-08-24
I assume you are talking about in Pidgin.  Try going to Tools > Preferences then to the smiley themes tab and selecting None.  The button will still be there but when you click it you'll just get a small box saying there is nothing to display.

### Post by raphaelr on 2008-08-24
I think he's talking about this forum. Go to the User CP, select 'Edit Options' on the left Sidebar and change the last Option, Message Editor Interface, to 'Basic Editor - A simple text box'.

### Post by coffeecat on 2008-08-24
Your post made me realise how one could disable them one by one, not only from the message box, but from the whole damn forum. So I thank you.

Install adblock plus in firefox and then add the urls for each smiley that you never want to see again. To be honest, I find some of the small ones useful. In a multi-national forum where there are many members whose first language is not English, it is very easy to have one's intentions misunderstood with plain text. Discrete use of a limited range can reassure the reader that one's intentions are friendly. One picture is worth a thousand words and all that.

However, I cannot stand the guitar and the waving placard ones. They are childish imo. For convenience, here are the URLs for the blasted things:

http://ubuntuforums.org/images/smilies/smiley-faces-75.gif[/noparse]

I'll leave you to discover the rest. (Just right-click on each and select Properties.)
