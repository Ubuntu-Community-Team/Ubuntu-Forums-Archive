### Post by matthew on 2006-02-22
I was recently asked to change my signature as it didn't conform to guidelines. I immediately did it, removing all the helpful links and so on that I had been using. Then I looked again at the guidelines and realized my signature ***DID*** conform as it was--it was within the allotted length and font size restrictions and well within content guidelines. The problem arose from the font size check. If you adjust the font size currently using the drop down menu in the forum editor box you will get this error message:
```
1. The font size you are using is too large. You are limited to using a maximum font size of 10 in your signature.
``` So this is legal:
```
Why waste time learning when ignorance is instantaneous? --Hobbes (the tiger)
``` but this is not:
```
[SIZE=1]Why waste time learning when ignorance is instantaneous? --Hobbes (the tiger)[/SIZE]
``` If you get a chance you may want to look into this.
