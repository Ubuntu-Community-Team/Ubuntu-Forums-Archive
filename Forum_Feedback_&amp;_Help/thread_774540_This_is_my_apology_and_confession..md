No need to apologise! There are two types of bans. The kind an admin does (which I am not sure of the specifics, but it can include IP bans) which results in burnt beans, and the kind moderators can do. 15 active infraction points results in a ban. Infractions can expire, but they can be permanent. (15 points that do not expire is the infraction for spam, which we call a "spam ban"). If infraction build up to 15 and are active, it is a ban. If they don't expire (and are not reversed) it is more or less permanent. It used to be, way before I was a moderator, that only admins could do any sort of bans and mods had to request it.
