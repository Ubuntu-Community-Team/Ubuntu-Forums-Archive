### Post by louis_nichols on 2006-05-10
:) there is a whole discussion around that!

[This]("https://wiki.ubuntu.com/TransparentTerminals?highlight=%28transparent%29") can make transparent terminals. [Here]("http://ubuntuforums.org/showthread.php?t=115974&highlight=knome") you can see how to enable kwin, which will later allow you to set windows really transparent. The only thing is, it won't be like the panel in the sense that even texts will be made this way, so if you set it to 100% transparency, you won't see anything of it anymore, unlike the panel.

Of course, the fuss right now is around [xgl/compiz]("http://www.youtube.com/watch?v=-CgqWlX_GsI&search=compiz")
