### Post by coffeecat on 2014-10-31
There are about 114100 tags in the database and growing. Many are quite obsolete. If the tag you want doesn't appear in the list, just define your own. The only thing that makes a tag unacceptable is if it doesn't conform with the forum [Code of Conduct]("http://ubuntuforums.org/misc.php?do=showrules"). So feel free to make up your own if you need.

Just one thing to help you not to waste time. The character length of a tag is minimum 3 and maximum 18. It used to be maximum 25 and there are tags >18 chars in the database. If you choose one of these you will get an error message.
