Yes.

If you look on the home page of the forum, you will see four main sections, the first two are for technical support and for all the official Ubuntu flavours:  Ubuntu, **Kubuntu**, Xubuntu, Edubuntu, Lubuntu, Ubuntu Studio, Mythbuntu, Ubuntu Mate, Ubuntu Budgie and Ubuntu Kylin. The list is there under each section heading.

There is also the [Other OS Support and Projects](https://ubuntuforums.org/forumdisplay.php?f=446) sub-forum, where you are welcome to post questions about non-Ubuntu distros, but please bear our caveat in mind if posting in this section:

> Please be aware this is not the best place to receive tech support for these OSs/Distros and users should look for official support channels for those systems.


But to return to Kubuntu, yes, very much so. It is one of the official favours and questions about Kubuntu are welcome. Please remember to choose the Kubuntu prefix for any thread you might post.

Good luck.
