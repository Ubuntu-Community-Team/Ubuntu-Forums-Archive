Hi.  When editing a post which I've made and wanting to add additional information I have to add the Code or Weblinks manually.  Would it be possible to include these (icons) with the Edit screen ?

---

### Post by jpeddicord on 2008-07-12
If you single-click on the Edit button, you get a basic inline editor. It does have a link button on the toolbar (globe) but it does not have the code tags. It's something ubuntu-geek would have to implement/activate.

If you double-click or middle-click on the Edit button, it will open up the full-featured post editor on a new page, which has all of the normal toolbar buttons for posts.
