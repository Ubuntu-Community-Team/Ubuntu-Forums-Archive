### Post by yetiman64 on 2010-12-15
> **Spyderkid said:**
> I went on Ubuntu forums and then a window popped up and said ubuntuforums.org wants to open something in a new window i clicked ok and nothing happened is this strange?

I get this exact behaviour when receiving a private message on these forums (using Firefox). If you have firefox set to block popups you will not get the message immediately.

In the top right hand corner of your browser window under the search box and your username check if you have any unread messages. You can click directly on the words "Private Messages" to see them, if there are any "unread".

### Post by Quackers on 2010-12-15
Add ubuntuforums.org as an exception to your Firefox pop-up exceptions (in the Content tab of Preferences) and private messages will then open up in a separate window if you ok it.
