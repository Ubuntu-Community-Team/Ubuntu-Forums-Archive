### Post by indytim on 2009-01-03
I have noticed a dramatic up tick in what I would classify as "non-valued" thread responses in the last year or so.  This is especially true in the Absolute Beginner's forum.  

The situation I am referring to is a posting of a question.  You will find a good response posted which resolves the question quite adequately.  From that point on, the "point hungry" post basically the same reponses with minor variations.  The net result is a thread that lacks any value content wise generally from the second or third thread on.  

To regain some of the value of the forums, I would suggest getting rid of the coffee cups etc. and just display only the number of postings by an individual.  This would minimize the "trophy hunting" that is so common on the Ubuntu Boards.

Just a friendly suggestion....
