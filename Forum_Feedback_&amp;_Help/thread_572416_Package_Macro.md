### Post by gautada on 2007-10-10
I have begun using the forums on an almost daily basis since switching distros to Ubuntu.  On a regular basis when asking and answering question many times I need to reference a specific package.  Currently I just create a URL such as [pysdm]("http://packages.ubuntu.com/feisty/admin/pysdm"). 

It would be better if it were unified into a macro (my term) similar to quote or code like:

```
[PKG="pysdm,feisty"/]
```

In the format [PKG="package name,[version OPTIONAL]/]

This would produce an html table that references:
 [LIST]
[*]packages.ubuntu.com
[*]the repository (universe, metaverse, etc.)
[*]links to other versions if the package exists (dapper, feisty, gutsy...)
[*]link to launchpad for the package if exists
[*]link to other posts and threads with this package mackage macro
[/LIST]

This macro? would be able to easily integrate forums with other resources via cross refrencing launchapad, wiki, packages, etc.  And would look better to boot.
