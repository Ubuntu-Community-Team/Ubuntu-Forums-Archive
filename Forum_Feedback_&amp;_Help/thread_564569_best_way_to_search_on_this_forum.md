We would like to [enable a better search mechanism]. For a while we used a custom search system that ubuntu-geek came up with that was much better. The problem is that it creates a nightmare for forum software updates/upgrades and had a couple of other silly issues that I can't remember, but were less important. In any case, what we are using now is vBulletin's built-in search just because it makes keeping the forums running substantially easier. We agree, it isn't what we would like it to be.
