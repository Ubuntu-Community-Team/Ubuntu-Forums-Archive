### Post by aysiu on 2008-07-28
Somewhere in your User CP is a preference about automatically subscribing to threads you post in. If you don't have that set, you won't be subscribed to any threads except the ones you explicitly ask to subscribe to.

If you're looking for the threads you started, go to Search > Find All Your Threads instead of going to User CP.
