### Post by coldcritter64 on 2013-03-29
> **Frogs Hair said:**
> Excuse me if I overlooked the obvious, but as I upload screen captures using manage attachments there seems to be a collection of all the uploaded Images. Is there a way to clear this list ?

That "collection" are your files on the server and are there so you can add a previously uploaded image to a post, the top "Add File" button is for new images needing uploading.

I suspect to clear that list will actually require removing the files off the forum server, any not "utilized" within 1 hour of uploading are deleted according to the preview window. In your user profile "Settings",  > My Settings > Miscellaneous > Attachments, is where attachments can actually be deleted by a user.
