For sake of clarity, here's what I have planned (the full content of the proposed page, omitted one image due to 8 image per post restriction) :-
===================================================================
[COLOR="#800000"]**_[SIZE=3]How to use [noparse]```
[/noparse] tags in your Posts[/SIZE]_**[/COLOR]

**_Why ?_**
Output posted as plain text loses its formatting and parts of it sometimes turn into smileys making it difficult to read and understand.

The use of 'Code' tags makes the post clean, compact and more appealing, thus getting more attention from readers.

**_How ?_**
There are three ways to put your code within [noparse][CODE][/noparse] tags -

[indent]**A.** Manual method (quick, and applicable everywhere)
**B.** Advanced Edit box (New post)
**C.** Advanced Edit box (Edit post)[/indent]

**[COLOR="#000080"]A. _Manual method_[/COLOR] (applicable everywhere - Advanced, Edit and Quick Reply boxes) :**

[indent]Add **[noparse][code][/noparse]** in the beginning and **[noparse]
```[/noparse]** at the end of the text that you want to put into the 'Code' box. For example -
----------------------------------------
[noparse]```
[/noparse]
[COLOR="#000080"]My gibberish output code here[/COLOR]
[noparse]
```[/noparse]
----------------------------------------
After submitting, the post will look like this -
```
[COLOR="#000080"]My gibberish output code here[/COLOR]
```[/indent]

**[COLOR="#000080"]B. _Advanced Edit box_[/COLOR] (useful when Creating New Posts) :**

[INDENT]**1)** Click the **#** button at the top of the edit box to automatically generate the opening and closing [CODE] tags pair -
[IMG]https://ubuntuforums.org/attachment.php?attachmentid=284515&d=1574952410[/IMG]

**2)** Copy-paste your code in-between this tag pair -
[IMG]https://ubuntuforums.org/attachment.php?attachmentid=284516&d=1574952398[/IMG]

Type the rest of the text as usual, outside of this tag-pair and submit your post. It will look like this -
[IMG]https://ubuntuforums.org/attachment.php?attachmentid=284520&d=1574952251[/IMG][/INDENT]

If the code is too long to fit in the box, it will automatically create scroll bars within the box to keep the post compact without affecting the formatting of the pasted code.


**[COLOR="#000080"]C. _Advanced Edit box_[/COLOR] (useful when Editing existing Posts) :**

[INDENT]**1)** Click on "**Edit Post**" button at the bottom of your post, then click again on the "**Go Advanced**" button -
[IMG]https://ubuntuforums.org/attachment.php?attachmentid=284517&d=1574952312[/IMG][IMG]https://ubuntuforums.org/attachment.php?attachmentid=284518&d=1574952294[/IMG] [IMG]https://ubuntuforums.org/attachment.php?attachmentid=284519&d=1574952278[/IMG]



**2)** Using your mouse pointer, **highlight** the part which you want to put into code box, then **click** on the **#** button at the top of the edit box to wrap the highlighted part within the [CODE] tags pair -
[IMG]https://ubuntuforums.org/attachment.php?attachmentid=284521&d=1574952262[/IMG]

Click on "**Save Changes**" button at the bottom of the edit box to submit the modified post *(You can use "Preview Changes" button to see how it will look after submitting, then "Save Changes" when you are satisfied)*.[/INDENT]
=================================================================
