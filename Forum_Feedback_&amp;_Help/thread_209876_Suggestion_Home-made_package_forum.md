### Post by gingermark on 2006-07-05
I'd like to suggest a forum specifically for people to post homemade packages. I was thinking it could work like the HowTo forum, with each package being checked (obviously we'd need some volunteers for that).

I can see the downside of this suggestion - bandwidth, possible conflicts with repos etc. Yet if you were to limit the size of the packages, and restrict them to new apps not in the repositories I think it would prove most useful, especially for those GNOME users who may want to try the odd new KDE app, but don't want to download all the KDE dev libraries, and vice-versa for Kubuntu users.

Of course there is also the security issue, so some restrictions might need to be placed on those allowed to post packages. Nevertheless I think the community would benefit from a central area where people can post their own packages to share with others.

I know I picked up a Kaffeine 0.8 package here, and that was very much appreciated.
