### Post by -kg- on 2009-01-02
As everyone says above.  If you mouse over any icon (meaning put the mouse pointer over the badge or any icon without clicking) a little text box will pop up saying what that icon does.  That is true for any icon or button on this page.

You can also tell which one the "Thanks" icon is by the fact that you cannot thank yourself for a post, so the icon won't be displayed in your own posts.
