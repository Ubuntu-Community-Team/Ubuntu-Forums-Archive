### Post by coffeecat on 2012-11-16
*Thread moved to **Forum Feedback & Help**.*

This is known about. Yes, the font rendering for this forum in Chrome in Windows is dreadful. I did some investigating myself and can confirm this. If it's any consolation, it's a little better in Chrome  in Windows 8, if I remember correctly.

One problem is that Technoviking had to do some creative coding to get the Ubuntu branding working well enough in the version of Vbulletin that we are running. For some reason the embedded font isn't playing well with the Windows version of Chrome. I believe things may be better when the long-awaited upgrade happens so, realistically, nothing will be done until after the upgrade. If Windows/Chrome font rendering is still bad after the upgrade, that is something I will try to ensure gets looked at.

In the meantime, I can only suggest you use Firefox or IE when using Windows. Ubuntuforums font rendering in Windows/Firefox is acceptable although the bold font still looks a bit gritty to my eye. This can be improved by installing the Ubuntu font family locally, which you can download from here:

[http://font.ubuntu.com/](http://font.ubuntu.com/)

You could see if installing the Ubuntu font locally helps with Chrome, but as far as I can remember the improvement was minimal.
