### Post by cariboo on 2010-12-16
The names are generally only used during the development cycle, once the version is released it is usually referred to by release number. 

Usually there isn't that much difference between interim releases, most solutions that work for Lucid (10.04) also work for Maverick (10.10).

### Post by cariboo on 2010-12-16
As for searching, I personally have better luck searching by version name.

Once an Ubuntu version is released, there are only security updates after the first couple of months. The only drivers that I have seen that are Ubuntu version specific are dialup modem drivers, the rest of the drivers, other than closed source drivers are included with the kernel, this assumes of course that you have well supported hardware.
