### Post by cariboo on 2009-11-12
Recurring Discussions are just that, fairly often we have quite a few threads on the same subject, for example Windows 7 and Google Chrome.

The choice is to merge all the threads on the same subject and create a megathread, or just move them to recurring. They both have their merits, creating a megathread puts all the posts in one huge thread, the downside is that posts get lost in all the noise. Putting all the threads on the same subject in Recurring puts them all in one place where it is easy to find them. The downside is you have to search recurring to find a thread if you are looking for it.
