### Post by ardchoille on 2005-12-25
I have been using Ubuntu 5.10 for three months, having switched from Fedora Core 4, and I have to say that this is the best distro that I have used thus far. It's only one CD and installing apps is as easy as using synaptic or apt-get install.

The repos are huge. I have found things in there that I always had to compile on other distros.

I used to run yum in Fedora and that required me to run it and go get a cup of tea because yum is horribly slow. The first time I ran apt-get install appname, I was amazed that it was half done before I could get out of my chair. I don't even get the chance to switch to another window when I run "sudo apt-get update && sudo apt-get upgrade" because it is blazingly fast.

I am similarly amazed at these awesome forums. I come here with a question and it is usually answered in five minutes and the folks here have much better attitudes than those of other forums. These forums ROCK!

I would like to say:

To whomever started these forums: Thank you so very much, you are appreciated :)

To the Ubuntu dev team: You people ROCK! I love this distro!

To the folks who post answers to these forums: I'd be lost without you :)

To all: Happy Holidays from a very happy camper :)
