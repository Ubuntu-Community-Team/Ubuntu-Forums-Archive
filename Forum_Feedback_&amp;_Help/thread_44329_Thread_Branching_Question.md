I am not sure how best to describe my question, but in long threads, sometimes I see replies that look like this:

```
start
   |
   + Response 1
   |     |
   |     Sub response 1-1
   |
   + Response 2
```

Whenever I reply to a post, it staircases down like where I have labeled "Sub response 1-1".  What controls how a reply like "Response 2" is made?

The controlling mechanism is how your post is originated - 

If you click on "quote" down at the bottom of response 1's post - it will become sub response 1-1...

If you click on "reply" at the top of the thread, it becomes response 2.  :)
