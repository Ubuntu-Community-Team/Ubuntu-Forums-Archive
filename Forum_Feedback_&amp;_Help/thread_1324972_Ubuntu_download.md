I see what your saying. The ubuntu.com site is quite poorly designed in my opinion. The torrent links can be found by going to the standard desktop download URL: [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

Then click the drop down link called "Alternative download options, including Ubuntu installer for Windows" underneath the download location box. In this list is a link to the torrent files: [http://www.ubuntu.com/getubuntu/downloadmirrors#bt](http://www.ubuntu.com/getubuntu/downloadmirrors#bt)
