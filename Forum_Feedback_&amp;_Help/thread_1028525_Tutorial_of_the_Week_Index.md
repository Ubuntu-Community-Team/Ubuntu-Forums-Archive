### Post by Elfy on 2009-01-02
Open the post for editing - in the bar at the top with the font tools etc is an icon that represents the Earth with a staple sort of thing - that is the Insert Link tool.

Highlight the whole of say 
2 - 12-31-07: Compile & Install Osmo Organizer

Click the insert link tool and you will have a box to insert the hyperlink into. You can add the [BBcode directly]("http://ubuntuforums.org/misc.php?do=bbcode") 

There is a way to do so quite quickly using the middle mouse button - 
1 - highlight the text as above
2 - have the relevant page open - highlight the url in the address bar
3 - go back to edit thread page - click the insert link tool, backspace and then click middle button in the box, it should paste the address for you.
