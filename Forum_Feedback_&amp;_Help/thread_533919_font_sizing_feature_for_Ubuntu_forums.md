### Post by angryfirelord on 2007-08-25
Ubuntu forums doesn't have that feature, but since you seem to have trouble reading smaller print, you'll also run into that issue with a lot of web pages.

To adjust fonts in FF:
Go to Edit-->Preferences-->Content-->Fonts & Colors-->Advanced. Look for "Minimum Font Size" and adjust it as you see fit.
