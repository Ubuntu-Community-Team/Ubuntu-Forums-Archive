### Post by yetiman64 on 2011-10-03
> **CTBuckweed said:**
> ...again i added spaces so you can see
If you wish to show code tags without using the space and having to explain it is there, you can use noparse tags around the bb code tags. For example you can put in the editor,

> [noparse][noparse]```
 your code goes here 
```[/noparse][/noparse]
Note that there is a second set of hidden noparse tags in use here for the purpose of turning off the bb code to show you the first noparse tags and the code tags :).  The tags I have used to display the above quote go in the order [QUOTE][noparse][noparse][CODE].

What you see is what you put in the editor and reduces the likelihood of someone accidentally copying the added space (more as a convenience for the person you are helping / demonstrating something to).

The link bodhi.zazen posted has the full list of tags available including [noparse]. The quote and code tags each have a button in the editor (go "advanced" if the code button is missing from the simple editor). You can simply insert a block of text/code in the editor highlight it and click the appropriate button and the tags will be inserted as you need them. Edit: code button has a # symbol, quote button looks like a text "balloon" from a cartoon.
