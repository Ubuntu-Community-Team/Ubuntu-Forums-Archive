### Post by QIII on 2015-06-27
Text from the terminal can be enclosed in code tags, which presents code in a box with a limited size.  Others can use the scroll functionality to read the entire text.

Code tags also make your posts more readable and they preserve the formatting.  You can have a look at one of your earlier posts to see how it looks after I edited it to include code tags.
