### Post by bonzodog on 2007-01-02
Generally, no, they don't. 
These forums have only recently become part of the 'official' structure of the Ubuntu community, and is mostly staffed by experienced linux using volunteers. There is next to no communication between 'Ubuntu Proper' (as the offical structure calls itself) and the forums. The main structure, including developers, uses the Mailing lists as the main form of communication.

We know that some of the developers occasionally look in, and we even get the odd post off mark Shuttleworth himself. I would personally hope that this situation will improve over the next year,and we start to see interaction from the main core of the developers and team leaders. It would be nice.
