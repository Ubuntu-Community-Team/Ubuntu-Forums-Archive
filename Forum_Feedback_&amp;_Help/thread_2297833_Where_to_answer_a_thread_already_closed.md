### Post by tgalati4 on 2015-10-06
When posts are over a year old, they are generally closed.  I presume this is to keep the information current, as Ubuntu and Linux in general are constantly changing.  However, to reopen an old post,  there is a "Report Post" in the lower left corner.  Normally this is used to call attention to bad language, bad behavior, or not following the Ubuntu Forums Code of Conduct.  But you can use the "Report Post" button to flag the Post (and the thread that it is in) and explain that you would like to reopen it and add a solution.  If a moderator agrees and reopens it, then you can add your solution.

Another way is to just make a new Thread in the appropriate subforum (perhaps Programming), and make a Descriptive Title for it and post your solution.  Then put a link to the old thread in it and then send a Private Message to the Original Poster (OP) that you found a solution to his 3-year old problem.  It's possible that the OP already found a solution or a different way of solving the problem.

Regardless, welcome to the forums.
