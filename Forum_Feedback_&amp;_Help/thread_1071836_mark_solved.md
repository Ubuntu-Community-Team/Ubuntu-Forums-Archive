### Post by drs305 on 2009-02-16
'Solved' is currently disabled while the folks work on the servers.

There are two workarounds, neither of which is great. You can edit the text and change the first word of the post to "Solved". It won't show in the title but if someone holds the cursor over the thread they will see it in the text.

The second thing you could do it Edit, then Edit again. This will allow you to edit the title. It won't show when someone is looking at the titles in a particular forum but will show once the post is opened.

Once 'Solved' is activated again, it is accessed via the Thread Tools link at the top right of the first post.
