### Post by DeMus on 2009-03-17
Hi,

I use Firefox to browse the net and therefore also this forum. I have set FF to use a new TAB when a page opens.
Unfortunately on this forum this does not happen automatically. I always have to use the CTRL-key and the LMB to open a post in a new TAB.
Now why do I want to open a post in another TAB? Glad you asked.
I use an FF plugin called "Reloadevery". I switch it on in the TAB that houses the index of a forum, to have this page reloaded every 5 to 60 seconds (depending on what I choose). This way I automatically see the newest postings. Now opening a post and maybe even answer it requires a new TAB which has not switched on the Reloadevery plugin. This way I can read the messages in a relaxed way, type the answer and when done close the TAB to switch back to the index, which has reloaded several times already showing the latest information. Reloading also shows me when a post has been answered.
On the Google search pages you can set a preference which automatically does that. Is it also possible to have it here on this forum? Or does it exist already and don't I know about it?
