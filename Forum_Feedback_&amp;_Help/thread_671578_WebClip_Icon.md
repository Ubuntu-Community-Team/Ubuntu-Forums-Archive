Could the admin please put an image of the Ubuntu logo named apple-touch-icon.png in the root directory of this server?  It is used for the WebClip feature of the iPhone/iPod touch and is similar to favicon.ico.  The image size should be at least 57x57.  More information on the feature is at [http://developer.apple.com/iphone/devcenter/designingcontent.html](http://developer.apple.com/iphone/devcenter/designingcontent.html).  

I know this is a Linux forum, but owners of those devices might multi-boot with Windows or Mac OS X to sync the players.
