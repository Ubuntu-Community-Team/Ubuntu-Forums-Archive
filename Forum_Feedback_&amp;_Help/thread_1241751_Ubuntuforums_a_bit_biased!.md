### Post by .Maleficus. on 2009-08-16
Ubuntu Forums is not biased towards any one OEM over another.  Dell ships some computers with Ubuntu, and not vanilla Ubuntu (Dell has made changes), so a Dell-specific board only makes sense.  If a user with a Dell/Ubuntu machine came into the regular support categories for Dell-specific problems, it would get hard to manage.  Just like why we have the Apple board and the System76 board.  They have their own specific problems not related to vanilla Ubuntu.

As for the .org status, Ubuntu Forums started as a 3rd party website and was acquired (given to?) by Canonical after it grew to the massive site it is today.
