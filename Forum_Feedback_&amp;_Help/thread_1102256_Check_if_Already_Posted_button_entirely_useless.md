### Post by Mr. Picklesworth on 2009-03-21
There is a potentially quite nifty feature here that checks if a thread's topic has already been posted. It works by running a quick search for the thread title that has been written.

Big problem: 99% of the time it fails because people tend to use words in these titles that are "either very common, too long, or too short" for the search engine, thus causing it to return an error. Not helpful for a feature that is meant to be a quick and easy measure for precision.

The feature knows what the offending words are, so the problem could be fixed if it just stripped them automatically in this case.
