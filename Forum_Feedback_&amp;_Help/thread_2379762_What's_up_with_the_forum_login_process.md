### Post by coffeecat on 2017-12-09
I've already asked you to tone down the melodrama. Please do so.

The Ubuntu One SSO login procedure was imposed on us. We didn't ask for it. Most of what you describe is to do with Ubuntu One login. We don't administer Ubuntu One. If you have any problems with Ubuntu One, please contact the Ubuntu One admins.

As far as being logged out of the forum, you will remain logged in for 24 hours, so long as you don't clear cookies, deliberately log out, or return from another browser or system (obviously). Another thing that will log you out (it is a security feature built in to vbulletin) is if you return from a different public IP address. If your ISP assigns you a dynamically changing IP address and you find you are being logged out without intending to because of this, there is absolutely nothing we can do about it. 

Another occasional problem that some users have reported is if browser addons are deleting cookies when you close the ubuntuforums tab. That will log you out as well.
