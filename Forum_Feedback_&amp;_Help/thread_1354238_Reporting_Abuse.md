### Post by matthew on 2009-12-13
The usage statistics on reported posts are not public, so you have no way of knowing how often or seldom that function is used. 

The only thing that you personally could see is whether posts you report personally end up having anything done to/with them or not.

If there is content that violates our code of conduct, please report it using the button and a staff member will always investigate to determine if action is needed. Please remember that we are looking for specific CoC violations, simply disagreeing with content is not adequate for posts to be removed, edited, etc.
