### Post by imagine on 2006-03-26
The site has a mininmal width of 500 pixel. So if your browser window is less than 500 pixel wide, horizontal scrollbars appear. In that case either the CSS code of the site needs to be changed (can't help you there) or you could use a browser which supports small screen rendering, like Opera.


A website signature would help you (and other people) also, if this isn't too much work for the administrator. I mentioned it already in the "Forum Layout Organization" thread, but maybe it got lost there.

All that needs to be added is an id-attribute for the body-element:
<body id="ubuntuforumsorg">

[http://www.rdrop.com/~half/Creations/Writings/TechNotes/css.tip.2.html](http://www.rdrop.com/~half/Creations/Writings/TechNotes/css.tip.2.html)
