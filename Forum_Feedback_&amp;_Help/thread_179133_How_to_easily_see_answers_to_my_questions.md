### Post by 5-HT on 2006-05-19
[quote=Arndt]When I want to see whether there is a new answer to any post of mine in these forums, I do Quick Links -> My Profile -> Find all threads started by Arndt. Is there a shorter route?[/quote] 
You can subsribe to threads you've posted in, or just threads you'd like to keep track of.

You can reach your subscribed threads via the 'Quick Links', and those with new replies will be boldface.

There is an option in 'User CP' that you can set to automatically subscribe to threads you've posted in. 
Additionally, you can subscribe to a thread manually by going to the 'Thread Tools' link in each thread.

HTH
