### Post by notjustanotherone on 2015-06-21
Hello! When I registered for this forum, I saw a message that told me I wasn't allowed to edit my profile until I have 10 posts. When registering for an account, I was never asked for a username, so I wondered what forum name I would be posting under. Then I noticed that my forum name is simply the username of my e-mail. Isn't this a potential privacy concern? If new users aren't allowed to edit their profile yet, they can't change their forum name to something else, which means people have a pretty good chance of guessing their e-mail. They could try the popular services like Gmail, Yahoo, and so on. Maybe it would be better if new users were assigned a random name like guest12345 so it's not tied to their e-mail. Just a thought.

### Post by yetimon_64 on 2015-06-21
> **notjustanotherone said:**
> Ok, so aside from asking an admin to change my username, my only options initially are to either tick the box to give my real name or change my email under my settings page, right?...
No. When setting up a new account tick the box but provide the user name you want on the forums to the "real name" box on Ubunu One sso log in page.

Ubuntu one knows me only as "yetimon_64" put into the "real name" box, this is how I set my username here without needing the admins to change it.
Then set your email address as the "Preferred email address" etc.

From          howefield's link to the "SSO registration guide" by Bapoumba, 
>  5. This name will be displayed on the forums as your username,  so make sure you are comfortable with it. **Your forums username will be  your Ubuntu One account name** with a number appended if the username is  already in use in the forums database.
 My emphasis with the bolded text, not bolded in the guide itself.
Note: I did a search on the ubuntuforums site for my chosen username _*first*_ so as not to have my choice altered automatically by the ubuntu one site thus avoiding needing admin assistance for any changes.
