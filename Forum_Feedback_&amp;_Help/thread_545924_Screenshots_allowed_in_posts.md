### Post by lisati on 2007-09-08
Just below the "submit post" button you will find a section "Additional Options" with a "Manage Attachments" button. As long as the screenshot is in good taste and relevant to a question, I don't see any problem.

### Post by southernman on 2007-09-08
lisati is correct, with the notation that you have to use the advanced editor when posting a thread or reply, to have the option of adding an attachement.

In your user cp you can set that up as your default editor, or just use the "Go Advanced" button, below the quick reply box.

### Post by bonzodog on 2007-09-11
if you upload the screenshot to an images website, you can use forum code by inserting (IMG) and (URL) before and after the web address, only with square brackets -- I used round ones here so you can see the code.

Say, for instance you upload the image to [http://xs.to](http://xs.to). It has a very useful tool that gives you a list of codes to allow people to see the image how they would like. One of them is to display the image as a thumbnail in a forum, and wraps the link in the correct tags.

If you look for my screenshot posts in the community cafe thread, then use the 'quote' button at the bottom of it, it will put you in reply mode, but also showing the code I used to display the post.
