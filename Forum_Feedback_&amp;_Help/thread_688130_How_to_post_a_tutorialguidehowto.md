### Post by p_quarles on 2008-02-05
You can post it here:
[http://ubuntuforums.org/forumdisplay.php?f=100](http://ubuntuforums.org/forumdisplay.php?f=100)

It shouldn't be a .PDF, as it's quite common for tutorial authors to revise or clarify things in response to comments. If you wish to edit it offline, a text editor is all you need. Please read the sticky note in that forum for a list of guidelines to consider when composing the tutorial.
