To those who don't know, youtube has in the last few days displayed faults for many Linux users. 

Given the huge impact of this I think a sticky thread would help, as there are multiple threads still being created around the forum. The contents should be simple:
1. This is a youtube problem, not related to any recent OS updates
2. Simple fixes: disable hardware acceleration; install [flash video replacer firefox plugin]("https://addons.mozilla.org/en-us/firefox/addon/flashvideoreplacer/"); or switch to html5 version of youtube

Multimedia & Video seems like the best place for it.
