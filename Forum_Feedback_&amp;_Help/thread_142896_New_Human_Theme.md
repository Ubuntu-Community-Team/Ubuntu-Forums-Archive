### Post by kassetra on 2006-03-11
For the most part, the forums try to match [http://www.ubuntu.com](http://www.ubuntu.com) as closely as possible.

If/When they change the home page, we also will change.

The tango icons reflect the icon theme in dapper... so that part has... mostly... already been changed.

We try to keep the themes universal to the forum software - so that when we add plugins, updates, and extras to the forum software, we don't need to spend three weeks updating the theme code as well.

Unfortunately, no matter what choices we make to the theme here, someone will hate it.
