### Post by lavinog on 2009-07-15
The Absolute Beginner Talk section gets alot of posts from Wubi users. Many times they are about uninstalling Ubuntu or increasing the partition size.  Since the users are new to Ubuntu, they don't seem to realize that the wubi install is not the default installation method, and fail to provide this detail. Other users provide information assuming the user has a default installation 

This lack of information on both ends can lead to frustration and push new users away from the ubuntu community.

Some wubi users need to be informed of what a wubi install is, and should provide this information in there request for help.

A sticky in the Absolute Beginner Talk section could address this. Something titled: 'If you installed Ubuntu from Windows'
The contents should explain that installing Ubuntu from windows is a new feature, and behaves differently from a installation from the live or alternate cd.  When requesting help on the forums it is recomended that you inform others that you installed Ubuntu with Wubi to avoid confusion.

Also in the sticky, provide a link to the wubiguide :[https://wiki.ubuntu.com/WubiGuide](https://wiki.ubuntu.com/WubiGuide)
