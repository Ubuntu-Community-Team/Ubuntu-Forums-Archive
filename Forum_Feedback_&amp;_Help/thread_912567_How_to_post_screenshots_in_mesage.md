### Post by ByteJuggler on 2008-09-06
Click Applications->Accessories->Take Screenshot.

Save your screenshot to disk.

In the forum, when posting, click the paperclip icon, upload your file with the browse button.

In addition to the guides for it (posted by someone above), you can use several methods of taking a screenshot and hosting it. I prefer imageshack.us because you have more flexibility, a central host, the ability to have the site resize it to various sizes and make thumbnails (with or without the dimensions of the original printed in it) and it generates all the code for the forum. You don't need an account to use it, but an account is handy (and free).

[http://imageshack.us](http://imageshack.us)

For taking a screenshot you can use the PrtScr key, or you can use various applications. I use scrot.
