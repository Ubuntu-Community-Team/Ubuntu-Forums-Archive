### Post by kassetra on 2006-05-12
The mailing list areas of the forum are not a regular part of the forum; they are the actual mailing lists displayed on the forum so that forum users can be part of the mailing list discussions.

Since the mailing lists are simply displayed here, we sync with the actual mailing list every few minutes, so, any spam that we try to remove from this area, will reappear when the mailing list area is synced.  It's magically reappearing spam.

The only way to remove the spam is to remove it from the actual mailing list, of which, we are not in charge.
