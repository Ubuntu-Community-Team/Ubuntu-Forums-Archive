Under the *Forum Tools* drop-list, the option *Unsubscribe from this Forum* is not even listed, while it is listed in the open Forums.

### Post by coffeecat on 2014-08-11
Click on the Settings link at top-right of any page. Then in the left pane, click on whichever folder you saved your subscription to under My Subscriptions. If you haven't made any custom folders, it will probably be in Subscriptions. Find the thread; that could be somewhat challenging if you have thousands of subscribed threads like me. Tick the box to right of the subscribed thread. Near the bottom, click on the Selected Threads button -> Delete Subscription -> Go.

### Post by Dragonbite on 2014-08-11
Ok, I got a solution ...

[LIST=1]
[*]Copied the URL from the Programming Talk unsubscribe icon ( http:// ubuntuforums.org/subscription.php?do=removesubscription&f=39 )
*Note the space between http:// and the rest of the URL address. This way the forum doesn't turn it into a link and you can see the whole address*
[*]Noted the "f=" number value for Ubuntu One (367) by hovering over the forum title link
[*]Paste the URL from Programming Talk into another tab
[*]Replaced the old number with the number for Ubuntu One
[/LIST]
Once I hit entered, it removed the subscription.
