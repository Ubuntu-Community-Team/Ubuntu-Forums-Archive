### Post by coffeecat on 2015-06-13
I've had a look at your raw text. The forum software has not deleted the extra spaces you typed after each full-stop (period). It merely displays them as one space. This has been happening with the vbulletin software we use for this forum for at least the last 10 years. As QIII says, this reflects modern preferences for the typed or printed page.

**Edit:** I've just tested this on a phpBB forum. The same happens there. Also on the IMDb discussion boards, where IMDb appears to use custom software. I think you'll find this is standard on most forums and bulletin boards.

### Post by coffeecat on 2015-06-14
> **roy27 said:**
> It just started happening to me recently, before I was fine.

The loss of visible spaces between sentences that you describe in your first post is not new - there has been no recent change. Please see my earlier post. If you believe there is some other recent change in forum display which might be due to browser addons, please post more details and we'll try to help you solve this. 

But to repeat - the forum software automatically renders the display of several adjacent spaces as only one space in the ordinary text area. It does not delete the spare spaces - this is simply the way they are displayed. You can see this most dramatically in posts where members post output from the terminal which is formatted into columns by means of spaces. If this is posted into the main body of the post, the formatting is lost, but if posted between BBCode [code] tags, the formatting is preserved.
