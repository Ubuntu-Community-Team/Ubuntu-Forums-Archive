### Post by Dr. Nick on 2005-12-01
This is a quick reply, It looks just like a new reply :)
The main difference is that a new reply just gives more options for formating the text, like clickable smilies and easier access to "code tags" Once you post enough you memorize most of the formating syntax and can use quick reply efficiently.

One thing to watch out for is that when using quick reply dont hit the "back" button in your browswer before submitting post then try to go foward, it will erase your post.

If you type a new reply going back then foward doesnt hurt any
