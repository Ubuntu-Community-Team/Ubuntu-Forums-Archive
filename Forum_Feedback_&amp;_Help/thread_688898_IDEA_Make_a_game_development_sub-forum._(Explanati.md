### Post by some_random_noob on 2008-02-05
Hi everyone, I have an idea:

We should make a sub-forum which is dedicated to game development. I know that some people here will say "Well, if you want a game dev forum, why not go to pygame.org or something?". My response would be that the majority of game dev sites out there are made for cross-platform projects. I think that we should have our own game dev forum here which is **dedicated to UBUNTU game development** (And nothing else).

This would be a great idea, and we could use the new sub-forum for:

- Sharing our projects with the community.

- Suggesting ideas for new Ubuntu games.

- Discussing all aspects of game design (Because all we currently have is a **development** forum, which is ONLY about code - no discussion of sprites, sound, music, fun... etc).

------------------------------------------------------------------------------------------

A new sub-forum would immensely benefit the Ubuntu forums community, and it would be a great place for hobbyists. The "development" forum which we have here disgusts me - it's so boring! We have officially succeeded in making programming look like a nerdy activity for people who do nothing but crunch numbers all day.

What happened to fun and creativity? Let's put the fun back in programming by making a new sub-forum!!!!! :biggrin:

**_[COLOR="Red"]Who's with me?[/COLOR]_**
