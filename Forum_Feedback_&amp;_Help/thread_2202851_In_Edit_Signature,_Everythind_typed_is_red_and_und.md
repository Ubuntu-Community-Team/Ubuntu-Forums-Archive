### Post by coffeecat on 2014-01-31
It sounds as though you are using WYSIWYG mode in the message editor. If your settings are giving you WYSIWYG mode for the post message editor, you will get the same for the signature editor. WYSIWYG mode is very confusing when trying to incorporate BBCode into a post or into a signature. My guess is a misplaced [noparse][/URL][/noparse] BBCode tag was causing the problem you describe. You can't see the BBCode tag in WYSIWYG mode and it's easy to be typing with the cursor on the wrong side of it. 

You can change your default editor mode from settings -> General Settings -> Message Editor Interface. If you want to change mode for a single post or just for editing your signature, the icon in the message editor toolbar that looks a bit like [SIZE=1]A[/SIZE]/_A_ will toggle between WYSIWYG and source mode.
