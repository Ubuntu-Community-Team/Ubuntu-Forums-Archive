### Post by 1clue on 2014-08-19
When the default is to open in the same tab, you can always use your browser's modification keys or right click to open in a new tab.  There is no browser feature to facilitate staying in the same tab.

Likewise, pretty much every forum I have used for years has behaved in the 'single tab' mode.  This includes what non-technical people would call a forum such as Facebook, twitter, gmail, <your-favorite-webmail>, pretty much everything.  I design web applications, if you're staying in the same login session the default best practice is to stay in the same tab.

Thanks.
