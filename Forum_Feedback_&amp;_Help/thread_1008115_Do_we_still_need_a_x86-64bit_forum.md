### Post by Rocket2DMn on 2008-12-11
I think there are still a fair number of differences between x86 and x86-64 that warrant keeping that forum area around.  Although most stuff works out of the box, people still run into problems with 3rd party software like virtualization, skype, java and flash, etc.  Users still also have questions about the differences between 32 and 64 bit.  Also, since we still support Dapper and Gutsy which don't work as smoothly with 64 bit as newer versions do, we still provide help for those releases.
