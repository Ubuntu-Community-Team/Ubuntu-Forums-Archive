### Post by Wim Sturkenboom on 2012-10-17
type [noparse]> [/noparse]
paste the text to be quoted after this
type [noparse][/noparse] after the pasted text

The disadvantage is that it omits the name of the original poster in the quote. To include that, you just use the 'quote' button at the bottom of the post that you want to quote.

### Post by grahammechanical on 2012-10-17
Just select the portion of the text that you want to put in a quote box and click the quote icon.

Just select > the portion of the text that you want to put in a quote box and click the quote icon

You can also do it by hand by enclosing the word quote in square brackets at the start of the portion of text and then ending the selection by enclosing /quote in square brackets.

Square brackets enclosing code starts code or command text. Square brackets enclosing /code marks the end of the code or command text.

It is called markup. The backslash marks the end of the portion of text to be formatted as a quote or a code.
