### Post by nola504 on 2007-10-21
I seem to remember a few month back that the forum search returned a lot more targeted/accurate search results.

These days the search feature seems more like Microsoft's search whereas a Google search seems to be more effective at searching the Ubuntu Forums than the forums search itself.

I can't seem to figure out how the search here works.

For example, if I tell the search to ONLY search the title and I type "Gutsy TV", I don't see any results that have those two words in the title.  Seems like the results are all over the place.

If I search Google for the same phrase I find, as the 9 result returned:
Tv-out with gutsy + ati driver using only TV - Ubuntu Forums
Tv-out with gutsy + ati driver using only TV Multimedia & Video.
ubuntuforums.org/showthread.php?t=580796 - Similar pages

Clearly the words "Gutsy" AND "TV" are in the title so why doesn't the forum search here find it?

I looked in the Forum FAQ and don't see a search tips section.  Is there one?  I don't see one on the search page itself.

This site is nicely designed but seems to have several usability issues.

Another issue I'll through out there is the lack of a "New Post" link in the main forum nav.  You have to click to a specific forum category before you are presented with a "Make Post" link.

Basic usability states you must provide multiple navigation paths to important functionality.

Just to even get to the forums from the Ubuntu home page takes at least 3 clicks that are all located on different locations on the screen.

All of this is very frustrating when you are already frustrated with something that has gone wrong with your system.

Is this the appropriate forum for this sort of conversation?  If not, it might be a good idea to start a specific "Usability" forum which would help the developers of this site get direct and targeted feedback on usability issues.

Sorry for the tangent.  This site IS one of the best Linux sites out there and I am consistent impressed with the level of community involvement, if we could just tackle some of these key usability issues I think it would be all the more better.
