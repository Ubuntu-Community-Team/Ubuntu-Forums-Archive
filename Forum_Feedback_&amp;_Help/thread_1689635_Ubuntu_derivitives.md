This thread seems to have taken some interesting turns.

To answer the question, it depends.

If the question is generic enough then support is easy / trivial.

But, once a spin / distro matures there will be more and more differences and eventually the differences merit separate support forums (mailing lists, bug reports, etc).

For example, while it all Linux, there are sufficient differences between Fedora and Ubuntu that we should send Fedora specific questions, selinux to name one of many, to the Fedora forms.

Same is true for Debian. Although Ubuntu is based on Debian, there are sufficient differences we send Debian questions to the Debian forms.

IMO the same is true for Mint.

So while it is all Linux, there are differences between the various projects and forks and once a project is large enough and different enough they maintain their own forums as is the case with Ubuntu/Debian, Ubuntu/Crunchbang (which is now Debian), and Ubuntu/Mint.

If you use Fedora / Ubuntu / Debian / Mint you should participate in the those communities, so in referring people to the appropriate forums we help those communities grow as well.

It is also a matter of "netiquette". Just as you (I hope) would not ask for Ubuntu support on the Gentoo forums and you should not ask for Mint support on they Ubuntu forums, use the Mint forums.

There are generic linux forms if you prefer, ie justlinux

[http://www.justlinux.com/forum/](http://www.justlinux.com/forum/)
