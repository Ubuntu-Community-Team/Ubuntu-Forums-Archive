### Post by ubuntu-geek on 2006-09-29
We now have a much improved thread tagging system. When creating a new thread you can tag it with "keywords" that will help better track your post.

For example..

Scenario.
You are posting about a specific issue with "KDE" fonts. When creating a new thread a box will now appear to enter tags,  in the new tag box you could specify "kde, fonts" this will then update the cloud of tags.

You can view the tag cloud [here]("http://ubuntuforums.org/tags/"), as more people work on tagging threads our cloud will grow and will allow users to drill down on specific searches based on keywords or tags.

If you wish to make comments please do so here.

### Post by matthew on 2006-10-02
Tags are key words that can be attached to posts or threads in a smaller index area. When doing a search it is substantially faster to search through that small index than it is to search through all the words in all the posts. What the tagging system does is enables more people to search faster using far less system resources thereby speeding up everyone's forum experience.

The feature is still pretty new. The idea of the tag cloud is still catching on but its usefulness is yet to be determined. However, you can search tags using the search menu and this is extremely fast because it doesn't need to access all the data in the main database of posts.

The system is still being set up, but I think it has real potential.
