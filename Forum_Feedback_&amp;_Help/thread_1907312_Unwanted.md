### Post by coffeecat on 2012-01-11
Also - check your User CP setting. User CP -> Edit Options -> Messaging & Notification -> Default Thread Subscription Mode. My guess is that you have one of the email notifications set there. I have mine set to "No email notification", so that I'm automatically subscribed to threads I post in, but not sent emails. In order to see if there are new posts in subscribed threads I simply click on User CP.
