### Post by dmizer on 2009-06-17
Enclose the output in [bbcode markup](http://en.wikipedia.org/wiki/BBCode) code boxes like so:
[COLOR="Red"]**[noparse]```
[/noparse]**[/COLOR]code in a box[COLOR="Red"]**[noparse]
```[/noparse]**[/COLOR]

Which will give you something that looks like this:
```
code in a box
```

Please don't use the [noparse]> [/noparse] tags for long code outputs!  It makes pages run too long and takes forever to scroll by them.

However, there is also the following:[noparse]
[php][/php] -- Formats and colors the text for php code, also works resonabally well for bash scripts
[HTML][/HTML] -- Fomats and colors the code for html[/noparse]
