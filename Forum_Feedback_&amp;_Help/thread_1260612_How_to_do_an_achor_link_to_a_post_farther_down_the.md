### Post by lisati on 2009-09-07
One way is to click on the post number for the post you wish to link to. This will open up the single post in a new tab. You can then edit the original post, and copy the url for the single post.

### Post by drs305 on 2009-09-07
When I reviewed the page, the bottom post was #13.

Take the post URL and append #13 to the end:
[http://ubuntuforums.org/showthread.php?t=1240210](http://ubuntuforums.org/showthread.php?t=1240210)  >>
[http://ubuntuforums.org/showthread.php?t=1240210#13](http://ubuntuforums.org/showthread.php?t=1240210#13) 

[http://ubuntuforums.org/showthread.php?t=1240210#13]("http://ubuntuforums.org/showthread.php?t=1240210#13")
It will open to the specific post but on the original thread page.
