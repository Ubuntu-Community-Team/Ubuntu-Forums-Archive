### Post by T C on 2012-12-29
Thanks for the replies. I see now that there are many forum threads addressing this. Acting on the advice I found there, I installed the Ubuntu fonts, and now everything looks great.

A note: Some posts recommended ClearType. I personally do not like ClearType, and I would be reluctant to recommend it to anyone as a solution to this problem, since it may produce undesirable results elsewhere.

Also: I'm glad that installing the Ubuntu fonts fixed things for me, but I'd like to point out that this is a solution to my problem, not the webmaster's problem -- you still have a website which, to many first-time visitors, looks ugly and amateurish because the fonts don't render correctly.
