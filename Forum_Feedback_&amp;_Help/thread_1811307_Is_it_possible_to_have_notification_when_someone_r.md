### Post by coffeecat on 2011-07-24
Click on User CP near the top left, and then on "Edit Options". Under Messaging and Notification you can change the setting for Default Thread Subscription Mode. You have "do not subscribe", or four email settings which include automatic thread subscription. If you are subscribed to a thread and there has been a new post since you last looked at it, the thread will be listed for you when you next click on User CP.

My preference is not to have emails, but to subscribe to all threads I post in, and I click on User CP whenever I first log in and periodically thereafter to see if there has been any activity on threads I am subscribed to.
