### Post by pickarooney on 2005-11-02
In the short time I've been on the site, I've been really impressed with the volume of traffic and in particular the number of quck, helpful replies.
If this has a downside, though, it's this - threads disappear really quickly from the front page and can get overlooked easily. Would it be an idea to make a Solved Threads sub-forum to the main help forums. The idea is that once the thread opener is satisified with the response, he/she edits the thread title, adding {SOLVED} and a moderator then moves this thread to the Solved forum. In this way, threads might not appear and disappear so fast and it could help users with searches for solutions to their problems, by allowing them to search first in the archive of solved issues.
