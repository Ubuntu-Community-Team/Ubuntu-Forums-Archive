### Post by leejkennedy on 2010-02-16
After using the ubuntuforums for some time I feel a useful tool for post creators would be to allow them to hide/delete any replies which they themselves deem pointless, without necessarily having to report it in any other process first.

By 'pointless' I mean a reply such as "[FONT="Courier New"][COLOR="Red"]Yes, I have the same problem but managed to fix it. I don't know how I fixed it but you will be able to resolve your issue too if you search the forums.[/COLOR][/FONT]" or some other such similar nonsense.

By giving the post creator more control of the content in the thread they have created, I believe this will help to reduce the number of troll posts and keep the thread itself focused on resolving the issue at hand - which i think is the whole purpose of the forum!

I would appreciate any feedback comments esp. from the team(s) that could make this happen :)
