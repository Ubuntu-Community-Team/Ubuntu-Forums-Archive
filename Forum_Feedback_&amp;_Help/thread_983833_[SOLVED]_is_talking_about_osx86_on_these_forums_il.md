### Post by pp. on 2008-11-16
[QUOTE=wikipedia]OSx86 (a portmanteau of OS X and x86) is a collaborative "hacking" project to run the Mac OS X computer operating system on non-Apple personal computers[/QUOTE]

Since the license of OS X is restricted to running the OS on Apple hardware, OSx86 is in violation of the license, and that in turn is against the CoC.

Iow, yes, it is.
