### Post by bodhi.zazen on 2009-08-25
The reasons we ban people are simple, failure to follow the Code of Conduct. Nothing more, nothing less.

Some of the interaction is public, ie what is posted in the Resolution Center. The rest is private. Along those lines, please give these users some respect, there is no need to publicly embarrass or ridicule these people.

The intention of moderation is to bring people into the fold. Some need more guidance then others, and again there is no need to do this in any kind of public or humiliating venue, so infractions are also private.
