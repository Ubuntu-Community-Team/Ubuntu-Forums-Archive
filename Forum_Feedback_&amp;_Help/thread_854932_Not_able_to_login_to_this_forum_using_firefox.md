### Post by lisati on 2008-07-10
Try clearing your browser's cache. From Firefox, click on Tools->clear private data, make sure cookies and offline web data are slected, and click "clear private data now"

You might also want to check Firefox's "work offline" mode on its file menu. If it's selected, unselect it.
