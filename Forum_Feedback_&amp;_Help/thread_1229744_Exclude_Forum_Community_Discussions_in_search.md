### Post by urukrama on 2009-08-02
As this is mainly a support forum for Ubuntu, it would be very handy to have an option to search only in the support forums here. If I search for some terms now, often almost half the results are posts in "Forum Community Discussions" forums, particularly the Cafe and Recurring Discussions. 

I can select which forums to search in on the advanced search page, but can only select one support category at a time ("Absolute Beginner Talk", "Main Support Categories", "Forum Archive" and "Other Community Discussions"). It would be nice if there were an option to search only in all these forums.

I am not suggesting to exclude all the forums in "Forum Community Discussions" from searching, as is the case with the Community Cafe Games forum, but merely to have the option to search only for support.

I don't know how feasible it would be to implement this, but it would help forum members (old and new) better find what they are looking for.
