### Post by Quackers on 2010-12-04
Two ways to do it.
The first is to click on New Reply instead of Quick Reply and then click on the # symbol in the toolbar. This creates code tags and puts the cursor in between them so whatever you have copied to the clipboard (right-click/copy) can be pasted there.
The second method is to type in to the reply [code#] (without the # - if I hadn't put that in it would have treated the rest of this message as code) and then put in the bit of code, then follow that by typing in [/code]
