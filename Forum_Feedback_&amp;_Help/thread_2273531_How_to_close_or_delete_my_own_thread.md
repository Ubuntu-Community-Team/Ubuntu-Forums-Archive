### Post by QIII on 2015-04-13
Only Staff can close or remove threads from public view.  You can use the "Report post" button to request that, but a Staff member will make the call.

To mark your own threads solved, click on "Thread tools" in the menu and select the mark as solved option.

But before you mark it solved, please describe your solution so others with the same issue can benefit.

Cheers!
