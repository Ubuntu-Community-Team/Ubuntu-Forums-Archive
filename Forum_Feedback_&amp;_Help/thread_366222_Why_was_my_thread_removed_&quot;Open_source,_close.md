This is not the case. Samba is not _just_ Windows filesharing. Samba allows you to setup Windows DOMAINS and all kinds of crazy configurations.

Since you probably wont read the documentation, here's a tip. If you want seemless filesharing in the same config change the permissions to SHARE and use the "force user" setting so all files on that share are accessed with the permissions of whatever user you want. Then it will not ask you for a password to access the share, and everything will be happy.

But please, next time read the documentation or take 10 seconds to google. It's on the first page of a search for such things.
