### Post by Hawkeye52 on 2011-12-01
The Ubuntu Forum is one of the most extensive and active forums in the Linux community. It does a wonderful job of helping both Ubuntu users, users of Ubuntu's children (e.g. Kubuntu, Xubuntu, Lubuntu, Ubuntu Studio) as well as countless (and unnamed) users of derivative distributions.

One thing that seems lacking to me, to make it easier for a user to find an answer for their specific desktop, would be to provide discreet categories for Ubuntu, Kubuntu, Xubuntu, Lubuntu, Ubuntu Studio, and (maybe) Ubuntu Derivatives. These categories might be well placed within the 'Desktop Environments' Category, as some type of sub-set. This would go a long way toward helping end users to search for answers that might be more 'desktop environment' specific than 'Ubuntu' core specific.

If this topic has been brought up in the past, beaten to death, and discarded, I apologize for the redundancy. If not, I look forward to your insights regarding this possibility.

Hawkeye52

### Post by kio_http on 2011-12-02
While I don't mind the present system, the seperate K/X/L Ubuntu branding has really spoiled things for varriants. 99% of new users don't even know about the existence of the variants until after they start using Ubuntu.

It would have been much better if all varriants had the same homepage. And if there were called Ubuntu Unity, Ubuntu KDE etc

People tend to stick with what they try first and I have seen many users leave linux because of dislike for the Ubuntu interface.

Its human nature, since I use Kubuntu, that is what I introduce to new users and when I show them the other variants later they all prefer KDE to it. Had I shown them Ubuntu first, I'm guessing they would tell me the same thing about KDE.

The fact that the main forums (Canonical donates £0 to kubuntuforums), brainstorm etc are all branded Ubuntu with ubuntu colours makes people feel that using a variant means using something inferior.
