### Post by cariboo on 2010-03-30
Right click the post number in the upper right of the post and select **copy link address**, then paste it in to your post, for example the link to your post in this thread looks like this:

[http://ubuntuforums.org/showpost.php?p=9052555&postcount=1](http://ubuntuforums.org/showpost.php?p=9052555&postcount=1)

### Post by 2hot6ft2 on 2010-03-30
You see the #1 in the top corner of your post at the top of this thread?
Right click on it and select Copy Link Location, then you can paste that link to another post and it will ONLY show that post when clicked on.

For the code box.
Highlight the text you want in the box and click on the # sign at the top of the reply box (you have to be in Advanced Edit mode for it to be there).
