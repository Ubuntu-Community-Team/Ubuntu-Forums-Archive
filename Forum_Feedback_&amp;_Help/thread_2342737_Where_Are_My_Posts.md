### Post by slow-speed on 2016-11-09
One way that works for me is to use the Advanced Search function and search for one's own user name.  Then, if one edits one's post and chooses Go Advanced, simply make sure to subscribe to that post (option shown below edit window).  Doing this to each one will make them appear in a list under the Settings when one selects "View All Subscribed Threads".

Hope that helps others.  :D
