### Post by coffeecat on 2016-02-12
Are you sure you are typing in the answer correctly? I've checked all the questions and answers in the setup and there is no error there. I've tested all the possible questions in a search while not logged in, and I was able to do searches if I gave correct responses.

In particular, make sure you don't use capital letters where the possible answer is all lower case. Type out the answer exactly as it appears in the list of possible responses - don't copy paste anything. Also, if you get a question asking you to choose a number, something like this:

> What is the second number in the following list? 12 88 17 48 34

The answer would be 88, not 2. (The spacing is better in the actual search page than it appears above in the quote box, so it should be clearer.)
