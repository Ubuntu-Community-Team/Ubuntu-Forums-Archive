### Post by doas777 on 2010-03-02
I think you are confusing support with documentation. the forums are not really here to be logical documentation. they are people asking questions. each thread belongs to the op, and each post to the poster. 
personally I would not be back if some mod had removed one of my on-topic posts, because they didn't get how it fit into some organizational paradigm. 

secondly, these threads get very unhelp as they get longer (your answer is on page 49, but how would you know...), so there would be no benefit to an single omnibus thread about everyones GDM issues.

when you start troubleshooting an issue, google it, rather than coming straight here, until you are sure that you need to ask the question. if there are good threads on the topic here, google will turn them up for you better than the vBulletin search features will anyway.
