### Post by kassetra on 2006-02-18
[quote=ndhskp]The Firefox password manager does not use cookies to remember logons. Also I just tried it with cookies set to the cookies expiration date and it still woud'nt work.

Here's what I want, I want the Firefox password manager to work with Ubuntu forums. The problem lies entirely with the Ubuntu forums configuration as well as the web coding I am sure.[/quote]

It has nothing to do with our configuration of the forums.  If you are accepting the cookies that this forum gives out, then you are only logged out when you actually click on the 'Log Out' button.

Firefox being unable to remember any vbulletin-software based forum password has been discussed numerous times before.  Vbulletin uses a technique that removes the password before logging in after it has been confirmed, as a security precaution.  Firefox sets the password when you click on the button - so it essentially sets a blank password.

You have three options:
1. Go into your Firefox password manager and hand-set the password for the forum.
2. Accept the forum cookie without altering it.
3. Type in your password every time.

This is most likely not a piece of the forum software we will be editing.
