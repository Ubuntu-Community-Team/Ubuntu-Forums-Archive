### Post by coffeecat on 2015-10-06
The Resolution Centre is for problems with your account, and this sounds like a local browser problem, so I've moved this to FF&H, where others can offer their ideas as well. Only the OP and staff can post in RC threads.

I'm using Firefox, so I don't think there's a problem with the forum software and Firefox itself. Have a look at your browser addons to see if anything is interfering with rendering the forum. Also, make sure you aren't blocking any scripts at ubuntuforums.org.

To troubleshoot Firefox, you could start in safe mode:

[https://support.mozilla.org/en-US/kb/troubleshoot-firefox-issues-using-safe-mode#w_how-to-start-firefox-in-safe-mode](https://support.mozilla.org/en-US/kb/troubleshoot-firefox-issues-using-safe-mode#w_how-to-start-firefox-in-safe-mode)

Or create a new profile:

[https://support.mozilla.org/en-US/kb/profile-manager-create-and-remove-firefox-profiles](https://support.mozilla.org/en-US/kb/profile-manager-create-and-remove-firefox-profiles)
