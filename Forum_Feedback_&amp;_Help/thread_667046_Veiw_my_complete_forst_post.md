### Post by matthew on 2008-01-14
Go to Search->Advanced Search
--Enter your name in the "Search by User Name" box.
----Select "Sort results by" "Thread start date" in "Ascending order."
-----Click Search Now.

FYI, [here's mine]("http://ubuntuforums.org/showthread.php?t=30166") and [here's yours]("http://ubuntuforums.org/showthread.php?t=552779").

(Note: this assumes your first post was to start a thread, probably with a question. If your first post was merely contributing to an existing thread, this might work, but I won't guarantee it.)
