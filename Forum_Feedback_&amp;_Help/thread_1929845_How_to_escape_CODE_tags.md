"Please use [noparse]```
 and

### Post by coffeecat on 2012-02-22
@Cheesemill, I have a standard snippet of text which goes something like this:

> ....please post ..... between [noparse]```
 and 
```[/noparse] tags for clarity. The simplest way of doing this is to highlight the output and then click on the [img]http://ubuntuforums.org/images/editor/code.gif[/img] button in the message toolbar.

Bare BBCode is:

> [noparse]....please post ..... between [noparse]```
 and 
```[/noparse][/noparse][noparse] tags for clarity. The simplest way of doing this is to highlight the output and then click on the [img]http://ubuntuforums.org/images/editor/code.gif[/img] button in the message toolbar.[/noparse]

The [img]http://ubuntuforums.org/images/editor/code.gif[/img] button is useful and you would think easy to use, but even then some people ignore the suggestion. :|
