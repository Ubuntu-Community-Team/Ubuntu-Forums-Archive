### Post by coffeecat on 2013-02-08
*Thread moved to **Forum Feedback & Help**.*

If I'm reading you correctly, you are confusing threads with posts. If you wish to create a new thread, you simply click on the orange "New Thread" button near the top left of the sub-forum that you wish to create a new thread in.

If you wish to quick reply to an existing thread, you need to activate the quick reply message box with the [img]http://ubuntuforums.org/images/rebrand/buttons/quickreply.gif[/img] button you'll see near the bottom right of the last post in the thread.
