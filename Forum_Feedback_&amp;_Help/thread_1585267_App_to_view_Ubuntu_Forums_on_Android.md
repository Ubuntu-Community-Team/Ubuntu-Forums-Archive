### Post by samalex on 2010-09-30
> **whiskeylover said:**
> They should do what the folks at XDA have done. I love their app for Android.

The pro with using something like NNTP is you have one standard aggregation app to tie all the forums you participate with into one nifty place.  Granted you wouldn't have access to things like creating subscriptions, viewing user details, etc, but to read and respond to messages it'd be awesome.  Very similar to having an RSS aggregation app to view content from many websites at once.

The problem though is NNTP uses a non-standard port, and PHP (and similar languages) don't do too well stepping away from serving just web content.  It'd almost have to be a stand-alone application that hits the database directly, but even if some other messaging standard was created outside of NNTP with these added features it would definitely make it easier to read message forums on mobile devices.

Heck I'd even go for a theme that's formatted for mobile devices ;)
