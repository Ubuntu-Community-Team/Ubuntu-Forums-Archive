### Post by coffeecat on 2018-05-26
The forum uses vbulletin software to run on, which does not have native markdown support. To provide that, a plugin or modification would be needed. Plugins are potential security issues and any proposal to add a plugin to the forum software requires a necessarily extensive and bureaucratic technical audit by Canonical IS before it can be installed. Prior experience suggests that such an audit has a low chance of succeeding. Also, implementing such a plugin would require a significant amount of time and effort on the part of several people, some of whom are volunteers. The Forum Council would need to be convinced that there is sufficient demand from the forum membership for markdown support.

But that all pre-supposes a suitable plugin already exists. For the webpage of the only published vbulletin mod for markdown that I have been able to find so far, I find this comment:

> Please note: In a Markdown Enabled forum BB Code will not work so don't turn a forum with existing posts into a Markdown forum- you'd be best off making a new forum for this mod.

No, we won't be creating a new forum.

And since there is zero likelihood of finding a talented coder who could and would be prepared to do the necessary coding work for an in-house mod, and then have that successfully audited by Canonical IS, and since (as far as I can make out) you are the first person to even mention markdown in relation to forum features, I wouldn't bet on seeing markdown support any time soon. 

But let us see if any other forum members have anything constructive to add to the conversation.
