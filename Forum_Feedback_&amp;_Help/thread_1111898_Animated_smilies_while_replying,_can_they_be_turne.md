### Post by sisco311 on 2009-03-31
in Opera is easier :)

to disable animated images(global): Tools -> Quick Preferences -> untick Enable Animated Images 

to disable animated images (/site): Tools -> Quick Preferences -> Edit Site Preferences -> Content tab -> untick Enable Animated Images

to disable the smilies: Tools -> Advanced -> Block content -> Add.. *[http://ubuntuforums.org/images/smilies/*](http://ubuntuforums.org/images/smilies/*)*

or right click menu -> Block Content...
