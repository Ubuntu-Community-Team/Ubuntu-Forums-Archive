### Post by 5-HT on 2006-02-23
Hi, how can we get options like that for this forum?

Well, you're in luck because we already have them!

There is the subscription option (under 'Thread Tools') for every thread (regardless of if you have posted) whereby subscribed threads are put in a folder for you (under 'Quick Links', as well as 'User CP') and get bumbed to the top of that folder (as well as being bolded) when new posts are made to them. You can always unsubscribe by deleting the tread from the subscription folder.

You can also set up in User CP the option to automatically subscribe to threads when you post in them/or create them, as well as getting an email notification sent when new posts are made.

I don't know if you can get email notifications for new posts in your subscriptions if you haven't posted in them, but if you look around User CP there should be an option for it if it's possible.
