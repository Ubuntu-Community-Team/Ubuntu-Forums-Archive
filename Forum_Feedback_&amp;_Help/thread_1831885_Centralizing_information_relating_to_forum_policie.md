### Post by dmizer on 2011-08-24
Double post to add:
If you google the word "as", you will return about 3 billion results. If you google "root login" you only return about 50 million results. Including very common words like "as" "to" "in" "on" "at" "with" etc. in your search terms actually hinders your ability to get good results. When you use a search box, you should not use correct grammar. Rather, you should select your key words very carefully.

More search help here: [http://www.google.com/support/websearch/bin/static.py?hl=en&page=guide.cs&guide=1221265&answer=134479&rd=1](http://www.google.com/support/websearch/bin/static.py?hl=en&page=guide.cs&guide=1221265&answer=134479&rd=1)
