### Post by Rocket2DMn on 2008-11-24
I don't think I fully understand either, but let me try to outline:
- There is a development version forum under Development & Programming
- The main forum areas are shared with all supported versions of Ubuntu - this is primarily because not that much actually changes between versions, and many beginners may not know what version they are actually running.  I'd say 99% of questions are version-independent
- Many different flavors of Ubuntu do have their own forums, but we support them here as well.  Again, most stuff is independent, and you can mark your threads with [ubuntu], [kubuntu], [xubuntu], or [all variants].
- Having different forums for different packages just isn't feasible.  However, there is a Wine forum area, and common packages have some forum areas, like Virtualization, Desktop Environments, Multimedia
- There are different forums for common hardware platforms as well - Apple, Dell, System76, servers, 64 bit architectures, SPARC.  x86 is assumed to be the default.

I would not be opposed to having a separate forum for LTS versions of Ubuntu, though.  This could be especially useful after the LTS version is no longer the latest (like now, since Intrepid is now available).
