### Post by McRat on 2010-06-04
Many boards auto-lock threads after a certain time.  Pretty sure it's now a normal optional feature in vB, it used to be hack.  I'd say the average time 1 year.

When the subject of a board is a very dynamic topic, like computers, locking is perhaps a good thing, because computer advice is not like a fine wine; it does not get better with age.

But if the board is say Classic Cars, locking old threads can be very counter-productive, since sometimes that data DOES improve with age...
