Someone else should do this, because I use the text only editor on the forum (no buttons or anything). Here is how I do:

To get a quote, in which the words are in a box but the words wrap use the [noparse]> Quotes Material Here[/noparse] tags.

So:

> 
I am quoting myself.


To quote a post, you click on the "quote" button or the little button next to it (for multiquote) and make a post. 

For code, use the [noparse]```
put code here
```[/noparse] tags.

```

~$ls
2008-06-11-222539_1280x1024_scrot.png  Music     p.cpp     p.php
2008-06-11-222611_1280x1024_scrot.png  off       p.cs      p.pl
alpha.py                               operabk   p.d       p.py
a.out                                  out.pdf   p.exe     Program.cs
Desktop                                out.text  p.f       p.sh
Documents                              p.asm     p.hi      p.text
HyperGlavin1b.jpg                      p.bf      p.hs      Public
info.sh                                p.c       Pictures  Templates
MannyCalavera1.jpg                     p.c~      p.js      unix.text
marinehealth.png                       p.ch      p.o       Videos
~$

```

For images I can't give a great answer. You can attach an image, I believe the icon is a paperclip (click on it).

For hotlinked images, you have to have the image on a server somewhere. I use [http://imageshack.us/](http://imageshack.us/)

That site will give you several options (even without an account). It gives you the code to paste in, it creates a thumbnail and it allows you to resize the image. If you have an account, you get more function.

Here is an example of me using it:

[[IMG]http://img88.imageshack.us/img88/2219/cimg0030in9.th.jpg[/IMG]](http://img88.imageshack.us/my.php?image=cimg0030in9.jpg)

The code is:
```

[noparse]
[[IMG]http://img88.imageshack.us/img88/2219/cimg0030in9.th.jpg[/IMG]](http://img88.imageshack.us/my.php?image=cimg0030in9.jpg)[/noparse]
```

It has to be an internet address, not a path on your computer.
