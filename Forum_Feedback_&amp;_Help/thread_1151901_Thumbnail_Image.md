### Post by John.Michael.Kane on 2009-05-07
When posting click--> Manage Attachments, and you should be able to upload the file you want which should be automatically scaled down according to the maximum file-size Width Height allowed.

The other option is to use an image hosting site which should offer the option of a link you can paste in your post, which would give you a thumbnail image that points to a larger version.
