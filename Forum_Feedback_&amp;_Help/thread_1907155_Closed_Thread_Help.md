### Post by Iowan on 2012-01-10
There is no way for a user to directly reopen a (closed) thread. You can use the [Resolution Center]("http://ubuntuforums.org/forumdisplay.php?f=123") to request a thread be re-opened.

I can't say for sure, but I suspect the thread was closed due to it's age. From the Code of Conduct:
> If a post is older than a year or so and hasn't had a new reply in that time, instead of replying to it, create a new thread. In the software world, a lot can change in a very short time, and doing things this way makes it more likely that you will find the best information. You may link to the original discussion in the new thread if you think it may be helpful.
