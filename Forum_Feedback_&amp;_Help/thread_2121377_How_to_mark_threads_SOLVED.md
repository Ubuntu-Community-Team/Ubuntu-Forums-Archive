### Post by coffeecat on 2013-03-01
*Moved to **Forum Feedback and Help***

The plugin that provided this function is unfortunately not compatible with vBulletin 4. We are looking to replace this as soon as we can, but this may be some time yet. In the meantime, a workaround would be:

Go to the first post in your thread. Click on "Edit Post". Now click on the orange "Go Advanced" button. In the advanced editor change the prefix to "SOLVED". Now click on the orange "Save Changes" button.

Please note that there is a [s]7-day[/s] 60-day limit on being able to edit posts, and this would include changing the prefix.

**EDIT**: I've extended the edit time to 60 days. We'll review that again once we have a workable plugin in place.
