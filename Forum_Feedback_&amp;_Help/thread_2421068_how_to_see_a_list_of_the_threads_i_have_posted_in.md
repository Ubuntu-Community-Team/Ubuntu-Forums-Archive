### Post by coffeecat on 2019-06-16
Click on the "Advanced Search" link top right of any page. Type your username in the User Name field. Make sure the Threads radio button is ticked by "Show Results as". Now click the Search Now button. Done. That will include threads you have started and threads you have contributed to, even if not started by yourself.

You can fine tune the search results with the other fields.
