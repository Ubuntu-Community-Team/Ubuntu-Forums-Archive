### Post by giddyup306 on 2010-08-02
The forums move fast enough here that it may be on another page before you go to check it again.  What I do is add a subscription (thread tools -> subscribe to thread) to it and check for updates in user CP.  If you forget to do that you can go into user CP and search for any thread that you started or for any post that you made.  Those two options are under statistics.
