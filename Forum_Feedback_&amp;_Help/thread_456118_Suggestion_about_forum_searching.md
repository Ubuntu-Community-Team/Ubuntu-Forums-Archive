### Post by Nehvrook on 2007-05-27
Just going to mention a few suggestions regarding the search feature in the forums. It's not that important but it bugs me a little so I thought I'd post about it here.

The first thing is you can't search for a number. If your search includes a number it claims it's below the minimum word length required. This has got in the way quite a bit while I'm trying to find help before making a new thread.
For example I had problem with Quake 3's sound, but I can only search for Quake sound, because it won't accept a search with just 3 in it. And I had a problem with fsck dying at exit status 4, but I couldn't include the 4.

I just don't see why there is a minimum word length, especially for numbers.

Another thing is the fifteen second's between search thing. I guess that's there to reduce the load on the server from too many searches, so I guess nothing can be done about it. But if you make a spelling mistake in your search waiting fifteen seconds gets irritating sometimes XD

Anyway it's not overly important like I said, but I thought I'd mention it.
