### Post by erfahren on 2007-12-08
I don't know how many will see this, but it's not a good idea to recommend to newbies that they install the newer proprietary ATI drivers just to enable desktop effects. It gets involved and oftentimes unnecessary.

For the current version (8.37.6 I think) in Gutsy all they usually need to do is install xserver-xgl 

When I was using Gutsy it I thought prompted me to install it, but maybe I just knew what to look for.

In any event, the current driver in Gutsy is (fairly) stable, and for the purposes of desktop effects XGL works fine (for the most part). Installing and configuring the newer drivers is a little advanced and pointless just for the purposes of desktop effects.

I just think that newbies should get a handle on the basics before tackling something like that. There are many others who have more pressing issues, ...thats all I'm saying.
