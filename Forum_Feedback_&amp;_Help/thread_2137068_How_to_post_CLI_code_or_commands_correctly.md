### Post by sudodus on 2013-04-19
You can write like this manually

[noparse]```
this is the code
```[/noparse]

to make it look like this
```
this is the code
```

or mark (high-light) the code text and use the # icon at the top of the editing window to create the code tags automatically.

### Post by oldfred on 2013-04-19
You can just copy & paste code from a terminal. If more than just a few lines you want to include it in code tags.
While you can manually add code tags, it is often easier to use advanced mode editing which has a lot more options on edit panel than quick reply. Highlight code again like copying and click on the # icon to auto add code tags.
Sometimes now that I usually am in Quick Reply mode, I click on the quote icon and manually edit QUOTE to CODE at beginning & end.

This is older from when we have vB3, but I assume most is still the same with the new forum vB4.
