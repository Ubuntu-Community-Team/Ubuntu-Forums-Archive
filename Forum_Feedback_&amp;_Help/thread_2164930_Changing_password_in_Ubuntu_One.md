### Post by coffeecat on 2013-08-02
The Ubuntu One password is the only one that matters and you change that in your Ubuntu One account. Your forum password is now an irrelevance, at least to non-staff members. You would not know your forum password anyway, since these were all changed to random ones shortly after the forum was taken offline as an initial measure. If you are wondering why there is still a reference to passwords in "Edit Email and Password" in your profile settings, this is because staff still need access to password change for a legacy function as a temporary measure only. As soon as this becomes an irrelevance too, it is intended that the reference to password in this setting page will be removed. This is in the hands of Canonical sysadmins, so I am not sure what the timescale would be.
