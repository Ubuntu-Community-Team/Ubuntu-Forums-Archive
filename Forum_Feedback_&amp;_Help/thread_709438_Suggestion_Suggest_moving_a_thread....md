### Post by lloyd_b on 2008-02-27
A suggestion for improving the forums:  Have a way (similar to the "report" mechanism) whereby ordinary users can suggest moving the thread to another forum.

The reason - I've seen *many* posts lately that are obviously in the incorrect forum.  Programming questions in "Absolute Beginner".  Grub related questions in "Desktop Environments".  Questions on installing software in "Forum Feedback & Help".

Having such a mechanism should make it easier for the moderators to identify which threads need to be moved.  And if the mechanism allows the reporting user to specify the proper forum, hopefully all the mods will have to do is read the thread topic (and possibly the original post) and then click an "accept" or "reject" button.

Lloyd B.
