### Post by BackInTimeMan on 2006-12-23
> **ayoli said:**
> the new theme is not so bad, it just could be nice to have ability to choose font size. font size is acually a bit to big for me.

The font size problem, that is, they are too big or too small for different people, is because the font size attribute is being set using px, and even worse, pt, which is a measurement for the physical world, ie. DTP. Relative font sizing works best for screen based design, ie. em or %.
