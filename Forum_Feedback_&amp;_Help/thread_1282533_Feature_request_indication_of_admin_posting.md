### Post by falconindy on 2009-10-04
This is a direct ripoff from the WoW forums, but I rather enjoyed the feature. Any time a Blizzard employee (a "blue") posted, the thread would be tagged as such, and an emblem would be visible in line with the thread when viewing the forum. Clicking on that emblem would bring you to the first blue post in that thread. Furthermore, if there were successive blue posts in the thread, each prior blue post would have a link (if i recall, it was the top right of the post) to jump to the next blue post.

I find that often, admins have good (informative) things to say, and having this functionality could be very useful when browsing for solutions. The WoW community took this to the next level and created "Blue Trackers", which were essentially parsers that created RSS feeds of blue posts.
