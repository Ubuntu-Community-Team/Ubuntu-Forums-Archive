### Post by TheFu on 2020-12-21
Server email programs are usually stopped from sending email whenever the OS is 90% or more loaded. Between all the searches and little forum spamming games, plus all the good reasons for these forums, I can easily believe that happens on this forums server multiple times a day.

If you click on the "Settings" link in the upper right-hand corner, any subscribed threads with updates will be displayed.
Also, while I don't have any proof, but perhaps if there was an automatic unsubscribe for all threads over 3, 4, 5, 6 months old, then the DBMS wouldn't be nearly so busy. The notification function of the forum software could limit searches to only threads under 6 months old.  But I've never setup or run forums software like this and don't have any clue how it really works underneath the GUI.  I do know that every few weeks, a dead thread from last year will get a necro-post and it is a reminder to unsubscribe.  

I'd love an easy way to unsubscribe from all old threads. I don't want to unsubscribe from all threads, just those over a few months old, en masse.
