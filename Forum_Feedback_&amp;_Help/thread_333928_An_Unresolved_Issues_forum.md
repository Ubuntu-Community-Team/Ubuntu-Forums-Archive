### Post by MindRiot on 2007-01-08
This has probably been suggested before, but...

It would be nice if there were a section devoted to threads that contained unresolved issues.  Such as the missing restart and shutdown options from the log-off window, whether this is related to XGL/Compiz, or not. 

It would be the responsibility and decision of the moderators as to which threads to  move to the unresolved issues forum. 

Members would not be permitted to create new threads in the unresolved issues forum, but members could post replies.

This would help focus attention on threads that need it the most, since this is a very active forum, many threads go unresolved because they are buried to quickly (by many threads that are resolved) and the thread starter might not have specified the issue well enough in the thread title to draw the needed attention.

Just an idea.
