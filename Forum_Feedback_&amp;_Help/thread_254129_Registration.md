### Post by max.diems on 2006-09-09
Please be sure to register your ubuntu installation. This is done at two places (register at both places)
1. [The Linux Counter ]("http://counter.li.org./")(this is older and has registration for all distributions of Linux)
2. [The Ubuntu Counter ]("http://ubuntucounter.geekosophical.net./")(this is newer and is only for Ubuntu and derivitives)
Also be sure to register YOURSELF, the user and the MACHINE at both sites
See my sdig for ways to use this (note that images are avaliable, but are not usable in sigs)
