### Post by crip720 on 2020-10-01
If you have a good answer for a tread that was closed, then you can make new post with an answer to the linked post.  Some closed posts might give someone looking for an answer ideas to make a better search.  Think it would be difficult for a forum to have only certain posts show on search sites.  Some search sites might give better listings to your own searches than google, but you will need to try them.

### Post by TheFu on 2020-10-01
if you include the 'ubuntu release' in the query, then old answers won't show up.  Also use "-closed+solved" in the query to avoid or include those types of threads.

When you post, be certain follow up with the solution in a way that can be found next year and mark SOLVED using the thread tools. Encourage others.

Strong google-fu is a life skill.
