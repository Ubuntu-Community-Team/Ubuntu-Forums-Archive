### Post by yetimon_64 on 2020-01-17
If you are using firefox the next link to the stylus addon may be of help to you, I THINK it is available for chrome/chomium also; you would need to check in the chrome store, I've only linked to the firefox addon here ...
[[COLOR=#0000ff]https://addons.mozilla.org/en-US/firefox/addon/styl-us/[/COLOR]]("https://addons.mozilla.org/en-US/firefox/addon/styl-us/")

You add the addon to the browser then go to userstyles.org to download/install themes for whichever site you wish to change, here is the direct link on userstyles.org for ubuntuforums.org...
[[COLOR=#0000ff]https://userstyles.org/styles/browse?search_terms=ubuntuforums.org&type=false[/COLOR]]("https://userstyles.org/styles/browse?search_terms=ubuntuforums.org&type=false")

Hope this helps. Regards, yeti.
