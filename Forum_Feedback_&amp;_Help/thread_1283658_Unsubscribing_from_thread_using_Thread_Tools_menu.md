### Post by Donalb on 2009-11-06
Ok, I also realised I'd subscribed to too many and couldn't see an obvious way to unsub.
Fiddling around though...

Pull up your User CP and go to List Subscriptions on bottom left.
Then against the thread you wish to unsub, tick the Notification box on the right.
Then in the Drop Down Box at the screen bottom you can select "Delete Subscription".
Done.
