### Post by po0f on 2007-01-14
arnieboy,

I think that the search functions are trying to be "too smart", in that they are trying to put more relevant threads towards the top of the list.  The only way around it that I see is to set the option for "Find posts from" to however far back you want to go.  I see there is also an option to sort results by relevancy or last post date as well.

Using these functions requires you to go to the [search page](http://www.ubuntuforums.org/search.php) or click "Advanced Search" (just a link to the same page) from the search's drop down menu, though.  

HTH
