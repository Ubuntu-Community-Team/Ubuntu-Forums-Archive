### Post by matthew on 2007-12-20
> **Kilz said:**
> That is an even worse analogyYeah, you are right. It was.

I really am/have been listening to your concerns, and I understand them completely. We have decided to deal with threads on these topics the way that has been described. It is a tenuous balance, but there really are legitimate and good uses for this knowledge and I'm not willing to say to those that need it, "You can't have it because of what someone else might do with it."

You know, this is actually an old argument in the software world, with a slight twist. There are those who do not believe that source code should be open because they feel that doing so is inviting naughty types bent on destruction to cause damage more quickly and easily. The counter argument there is that the more people that see the source code, the better your chances are of having a friendly person discover a possible exploit so that it can be fixed earlier. (I think that's a much better analogy, isn't it?) We haven't been able to convince everyone of the value of open source yet, so the discussion continues with everyone choosing to follow what they think is best, and hopefully remaining engaged in a friendly dialogue.
