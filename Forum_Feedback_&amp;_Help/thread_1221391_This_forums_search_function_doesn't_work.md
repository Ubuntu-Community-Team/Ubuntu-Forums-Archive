### Post by bodhi.zazen on 2009-07-24
Search (near the top of the page, in the horizontal menu) -> Advanced search

Put your user name in the "search for users" box on the left
Go with "find all posts by user" (default) => search away.

If you are searching for a specific topic, go with google.
If you are searching for a specific topic by a specific member, use the forums .
