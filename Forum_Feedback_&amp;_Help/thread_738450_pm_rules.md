### Post by LaRoza on 2008-03-28
They are the same as general posts [Ubuntu Forums Code of Conduct]("http://ubuntuforums.org/index.php?page=policy")

You can opt to only receive PM's from buddies, if you don't want to receive them from everyone.

If you have a problem with PM's, spam, harrassment, etc, you should forward them to an admin (or possible a mod if no one else is around).
