### Post by bapoumba on 2014-06-16
One and only recommendation : *Ubuntu One and ubuntuforums accounts are linked based on email string addresses*, so make sure you have set the correct email in the Ubuntu One account _BEFORE_ login into the forums.

Special case for gmail/googlemail accounts :

[list]
[*]Google treats accountname@googlemail and accountname@gmail as the same
[*]Google allows you to add periods in the account name: thus accountname@gmail, account.name@gmail, acc.ount.name@gmail, and so on, all go to the same gmail inbox
[/list]
When you first log in with Ubuntu One, please ensure that the string you use for your email address in Ubuntu One is exactly the same as that registered with your pre-existing forum account. 

**If you have no Ubuntu One account and no ubuntuforums account**
You need to create a Ubuntu One account before you create a forums account
[LIST=1]
[*]Go to [https://login.ubuntu.com/](https://login.ubuntu.com/)
[*]Click &#8220;I am a new Ubuntu One user&#8221;
[*]Enter your email address
[*]Choose a name and a password
[*]This name will be displayed on the forums as your username, so make sure you are comfortable with it. Your forums username will be your Ubuntu One account name with a number appended if the username is already in use in the forums database.
[*]Answer the captcha and accept Ubuntu One ToS
[*]Click &#8220;Create Account&#8221;
[*]Log into the ubuntuforums by clicking &#8220;Login with SSO&#8221;
[*]Verify this email from the link you will receive in you mailbox to be able to post on the forums
[/LIST]

**If you already have a ubuntuforums account but no Ubuntu One account**
You need to create a Ubuntu One account before you login your forums account
[LIST=1]
[*]Go to [https://login.ubuntu.com/](https://login.ubuntu.com/)
[*]Click I am a new Ubuntu One user
[*]Enter the same email address you used in your forums account
[*]Choose a name (it does not have to match your forums account nickname) and a password
[*]Answer the captcha and accept Ubuntu One ToS
[*]Click &#8220;Create Account&#8221;
[*]Log into the ubuntuforums by clicking &#8220;Login with SSO&#8221;
[*]Check your email address on the personal details page and proceed
[/LIST]
Note : if the emails on the forums and Ubuntu One do not match, you will end up with a newly created forums account with a username derived from the Ubuntu One name.

**If you already have a Ubuntu One account but no forums account**
[LIST=1]
[*]Login to Ubuntu One 
[*]Your forums username will be derived from the Ubuntu One name, so make sure you are comfortable with it displayed. Your forums username will be your Ubuntu One account name with a number appended if the username is already in use in the forums database.
[*]Log into the forums by clicking &#8220;Login with SSO&#8221; 
[*]Check your email address on the personal details page and proceed
[*]Verify this email from the link you will receive in your mailbox to be able to post on the forums
[/LIST]

**If you already have a forums account and an Ubuntu One account**
[LIST=1]
[*]Go to [https://login.ubuntu.com/](https://login.ubuntu.com/)
[*]Log into Ubuntu One
[*]Make sure your preferred email address is the same as your forums email address
[*]If not, change your Ubuntu One email address to match the ubuntuforums account before the first login to the forums, otherwise you&#8217;ll end up with a newly created forums account
[/LIST]

**Please note:**

If you un-tick your name in your Ubuntu One  Personal Data Request page, the system uses the local part of the email to create a forum username.

The Ubuntu One preferred email address can be changed after the initial login to the forum without affecting the link between the two accounts.

Only a ubuntuforums admin will be able to sort you out of a Ubuntu One and forums email mismatch. To get help, please create a thread in the ubuntuforums [Resolution Centre]("http://ubuntuforums.org/forumdisplay.php?f=123") after you have read the stickies at the top of the forums section.
