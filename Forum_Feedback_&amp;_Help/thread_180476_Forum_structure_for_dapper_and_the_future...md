### Post by ubuntu-geek on 2006-05-22
With the ever growing number of Ubuntu alternatives such as Kubuntu, Edubuntu and Xubuntu we have decided to implement a new posting system using thread prefix's. We curently use these in the Gaming and Howto section and they work really well.

Below are a few screenshot's from the new dapper category showing what the prefixes are.[IMG]http://ubuntuforums.org/screen1.gif[/IMG] 

Notice we have added a new Wireless support category. The Howto and Gaming forums will be included in the dapper section they are just not reflected here.

[IMG]http://ubuntuforums.org/screen2.gif[/IMG]

When posting in the General Support area you can choose which operating system you are using. This will be reflected next to the post.


[IMG]http://ubuntuforums.org/screen3.gif[/IMG]

Also when posting in the Desktop Support category you can select which supported window manager you are using. Gnome, KDE or XFCE.

[IMG]http://ubuntuforums.org/screen4.gif[/IMG]

When performing a search you can choose a thread prefix to search in. For example you want to search for kde support but only want results from kubuntu 5.10 not 6.10 you will be able to fine tune your search like that.

This is just a peak of what's in store. :)
