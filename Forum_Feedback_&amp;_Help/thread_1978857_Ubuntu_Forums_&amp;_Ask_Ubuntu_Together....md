### Post by Carborundum on 2012-05-12
I don't think AskUbuntu is intended to in any way replace the Ubuntu Forums, as the two sites have rather different purposes. Unlike UF, AU really isn't meant to be a discussion forum. It aims to have no duplicate posts at all, and as few answers per question as possible. Rather, if you think you have something to contribute to an existing answer, you are supposed to edit that answer instead of posting your own. This makes for a very good environment for questions that have a definite right answer, but the system does not lend itself well to lengthy discussions. For example, a question like "how to I install a .deb-file without Software Center?" is probably better asked in AU than in UF, but something like "which desktop environment is better and why?" most definitely should go in UF.

As for having a "+1"-button on posts here on UF, I can really only see advantages. It would avoid cluttering up threads with posts without content, and it would be much better at determining which posts the community considers to be insightful than the current system.

In short, I say "aye" to integrating such functionality in the forum.
