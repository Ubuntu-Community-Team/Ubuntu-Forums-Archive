### Post by coffeecat on 2012-10-03
This is to give advance warning that we will be making some changes to the layout of the home page of the forums. Nothing will be lost, but we hope to make it easier to find specific sub-forums. We are planning to make the main changes on Monday and Tuesday, 8th and 9th October. When finished, and barring minor changes, the layout should look like this:

[img]http://ubuntuforums.org/attachment.php?attachmentid=225014&stc=1&d=1349264337[/img]

Our main goals are to reduce the amount of space that the Forum Archive is taking up on the home page (but still make it accessible), and to separate technical support and discussion forums into their own sections as much as possible.

Please note the following:

[list][*]Prepare for inconsistencies for a while. The two servers, lingonberry and bilberry, can take a while to synchronise, so changes may appear on one server but not the other for some time. We will try to limit this as much as possible, but you may see elements of the old layout re-appear when refreshing the page.[*]Partly because of the above, we will probably do the changes in two or more bites over the two days.[*]The Loco Teams section needs to be the last item in the last section for technical reasons. It is an important part of the forums and deserves a place high on the home page, so a new link to it will be created in the “Ubuntu General Support” section.[*]Some minor changes or section name changes may appear before those two days.[/list]

Thank you for your patience, and apologies for any inconvenience or confusion that this may cause.
