### Post by gmxgeek on 2008-08-31
This becomes especially important in the Absolute Beginner section, as many times there are several people looking for the same thing. If an issue has been resolved, please remember to mark the thread as solved so that you can let people know that:

1. The givers of the solution knows that all problems posted have been solved, and that everything is working fine.
2. The solutions posted worked for you so they should try and see if it resolves their problem as well.

If you need more information on marking solved threads, check out
[https://wiki.ubuntu.com/UnansweredPostsTeam/SolvedThreads](https://wiki.ubuntu.com/UnansweredPostsTeam/SolvedThreads)


It makes the lives of the givers of advice so much easier, and really helps out those with the same problems as well. So mark those threads!
