### Post by RedSquirrel on 2009-10-01
If you're referring to the idea of having a button to click (**#**) as you would in the full editor, I think that came up a while ago and the answer was "No".

You can add code tags *manually*:

[[COLOR=DarkRed]_[/COLOR]code[COLOR=DarkRed]_[/COLOR]]sudo apt-get update[[COLOR=DarkRed]_[/COLOR]/code[COLOR=DarkRed]_[/COLOR]]

(Remove the "[COLOR=DarkRed]_[/COLOR]", of course.)
