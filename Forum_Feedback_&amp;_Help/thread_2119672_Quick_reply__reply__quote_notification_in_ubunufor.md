### Post by sisco311 on 2013-02-24
Thread moved to Forum Feedback & Help.

You can set the Default Thread Subscription Mode to No e-mail notification under User CP (top left corner of the page) > Edit Options > Messaging & Notifications > Thread Subscription Mode

To access the Subscribed Threads Folder go to User CP > Subscribed Threads Folder.

See: 
[http://ubuntuforums.org/showpost.php?p=4537471&postcount=9](http://ubuntuforums.org/showpost.php?p=4537471&postcount=9)
