### Post by sudodus on 2016-09-18
Welcome to the Ubuntu Forums :-)

Ubuntu is free. You can download an iso file without paying anything. In some places, they ask for a donation, but you need not pay anything. Also the source code is free for you to use and develop. But there is no longer a distribution of free DVD disks. It is possible to buy DVD disks with Ubuntu from 'third parties'. (The Ubuntu image, the ISO file, is too big for a CD disk.)

I suggest that you download an iso file, and burn it to a blank DVD disk or flash it to a USB pendrive. If you cannot do it at home, maybe you can borrow a computer with a good internet connection.

See this link for more details: [http://www.ubuntu.com/download/desktop](http://www.ubuntu.com/download/desktop). See also the links from it.
