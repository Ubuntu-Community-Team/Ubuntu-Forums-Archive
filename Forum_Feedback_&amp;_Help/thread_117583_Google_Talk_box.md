### Post by kassetra on 2006-01-17
This subject has been asked a few times (as we do not have a jabber box setup either) - and the answer is this:

Currently, the vbulletin version of software that we are using for the forums only has four protocol plugins: ICQ, AIM, MSN, and Yahoo IM.  These are actually tied to functions that can technically IM someone from the forums.  Vbulletin does not have (that I have seen) any protocol plugins to support Jabber/Google Talk, so adding those options to the list of IM protocols isn't feasible.

Given Google's popularity, I am sure we will see a Google Talk plugin available soon, and when one is available, we will most likely try and install it.

In the case of adding more IM protocols - it's not about what we choose to add - it's about what's available for us to add, that's all.
