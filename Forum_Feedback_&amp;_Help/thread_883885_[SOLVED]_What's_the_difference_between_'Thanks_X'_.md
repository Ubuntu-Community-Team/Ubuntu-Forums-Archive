### Post by pmlxuser on 2008-08-08
```

Thanks X

```
means you were the one that thanked somebody for a useful post they posted

```

Thanked X times in Y posts

```
X means the number of people who thanked you for writting a useful post
Y just shows you for how many post you got those thank yous from, you can be thank youd several times for the same post.
