### Post by Irihapeti on 2011-03-03
I think that the time-out might be less than that, lisati. 10-15 minutes maybe?

There are two options I use to deal with that.

One is to compose the reply in a text editor. Or, if I've already written it out in the reply form, copy it to a text editor before submitting. Then if I've timed out, I haven't lost it.
