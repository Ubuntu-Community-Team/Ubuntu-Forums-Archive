Hello,
I was able to capture a possible double-post inducing bug in the site.
I attached a image where you can see that after refreshing the page to see whether someone answered yet or not, it is possible that the editor keeps the text of the last posting (see [https://ubuntuforums.org/showthread.php?t=2435369](https://ubuntuforums.org/showthread.php?t=2435369) )
Note that an answer is present, but the editor proposed me the same text. I would be induced to think that the posting was not successful.
In this case my post was visible, but also another case is possible where it seems that the post was not sent. It happened to me recently.
I hope this is useful.
