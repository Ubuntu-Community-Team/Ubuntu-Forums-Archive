### Post by The Toxic Mite on 2010-03-07
> **JoeGoe said:**
> How do you add code from the terminal to your posts?
 
If you want to copy snippets from the terminal to a forum post, paste it here then wrap it around ```
 tags like so:
 
[noparse][code]
$ echo "Hi"
Hi
$

```[/noparse]
 
This would render it as:
 
```

$ echo "Hi"
Hi
$

```
