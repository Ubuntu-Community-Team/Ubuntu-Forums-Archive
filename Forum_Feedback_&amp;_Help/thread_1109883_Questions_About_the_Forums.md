### Post by bapoumba on 2009-03-29
> **Penguin Guy said:**
> Q: Is it possible to change the thread title after you have made a thread? For example to add [SOLVED] to the front?

You cannot change the main thread title, only the first post title. The solved plugin was doing just that (change main title by a user), but corrupted the database..
Staff can change a thread main title, but using tags is far more efficient: you can do it on the spot, no need to report a thread and wait for a Staff member to do it. Even better: anyone can add a tag, not only the OP :)
