**_Current list of Prefixes_**

> 
Ubuntu
Kubuntu
Xubuntu
Edubuntu
Gobuntu
Mythbuntu


Okay, these probably are for referring to specially to problems with *those* distributions, witch is pretty pointless since there are some environment prefixes, what makes them address a very small minority of the problems. Besides of the *Ubuntu* tag, that is generally used for general questions, these 5 tags address less than the 1% of the posts.

> 
KDE
GNOME
XFCE


These are probably for help on the environment, but why do we need 3 of these prefixes when we can just have 1 "Environment help" that could address these 3 and more desktop environments? These 3 tags address 5% of all the posts.

> 
All Variants
Other


Now, these are more general. Very general actually. **All Variants** addresses questions about all Ubuntu variants, being it pretty pointless, since there are not many questions only about Ubuntu variants that addresses ALL variants ( some question like "what are these Ubuntu variants" is the only thing I can think of ). On the other side,the "other" prefix addresses all questions not contained in any of these prefixes. If you think about it, most questions should go here.


**_More correct prefixes:_**

**Installation**
Questions about downloading/installing Ubuntu **OR UPDATING IT**. Here questions like "How do I install Ubuntu" or "Hardy screwed up my disk"

**Applications**
All problems with Linux applications should go here. Questions ranging from "Nautilus doesn't show files" or "Wine doesn't open GTA" to "How do I create units in FreeCiv" should go here.

**Desktop Enviroments**
This would be the second group of old prefixes. Questions like "How do I change my GNOME theme" or "How do I download KDE4" should go here

**Networking**
Since Networking is such a big issue on Ubuntu, it deserves its own prefix. Questions like "How do I set up a wireless connection" or "How do I change the name of a Samba Domain" should go here.

**Coding**
Lets face it: even with synaptic, .debs, and nautilus doing most of my Samba work, most Ubuntu users must use the Terminal. Bash-related questions like "How do I use wget" should go here, as well as programming-related questions like "How do I open a file in Python"

**Other**
All questions whose theme isn't on any of the previous categories shuold have this prefix, or questions where you don't know what prefix to put.

**Non-technical**
Prefix for all non-technical posts, like "check my video" or "OMG LOOK A PINK PONIE".


Suggestions are not only appreciated but required for the competition of this idea!
