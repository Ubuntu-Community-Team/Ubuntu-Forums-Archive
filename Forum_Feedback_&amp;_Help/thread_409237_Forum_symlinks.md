### Post by caffienefree on 2007-04-14
I was contemplating the ups/downs of an idea like this while I was eating this morning. 

In reality, there are some threads that really belong in multiple forums. Sometimes, the threads are moved to the more appropriate forum, others are not. This is usually not a problem because you can search across multiple forums. However, for a newer user, they may not be fully aware of that capability, and only search in one forum where the thread does not exist. The ability to add references to different forums would a) cut down on cross-posting, as the thread would exist in one place with multiple references, and b) solve the above mentioned search issue.

The capability theoretically exists, as symbolic links are created when a thread is moved. What I'm suggesting is the ability for the poster to create those links.
