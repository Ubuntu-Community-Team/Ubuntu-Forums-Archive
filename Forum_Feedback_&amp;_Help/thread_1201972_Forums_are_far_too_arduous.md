### Post by rsindlin on 2009-07-01
All I wanted to do was find out if there was a way to put a comment in a source file being read in fdisk.  Goggling brought me to this page.
[http://ubuntuforums.org/showthread.php?t=732332](http://ubuntuforums.org/showthread.php?t=732332)

I already knew that, but it seemed the answer really just wasn't out there, so I just started tinkering around in fdisk.  It turns out that apparently if you enter the command 'm' in the fdisk menu, all other text after it is simply ignored.  Since m only prints the menu, it's a perfectly safe character to use to put a comment in the file being read in by fdisk.

So I thought I'd be helpful and add that to that thread quick.

20 minutes later I'm posting this.  I can't reply annonymously.  OK, fine, I understand the concerns, but I can annonymously edit Wikipedia.  So I jump through the hoops, and I'm registered.  I try to reply again.  Still denied!  So I wonder, "is it because I clicked back through my browser and followed the old link as an unregistered user?"  So I decide to search for it in ubuntuforums.  Wow.  How is it that typing in the exact text of the subject of the thread *didn't* bring up that thread as the first link, let alone in the first page of results?  Finally searching for "fdisk 489" got me the thread halfway down the 1st page of results.

So, now I'm definitely viewing this old thread as a registered user.  Now I click on reply.  I get the *same* message that I can't edit it.  Finally, I after looking very carerully at the old thread, I see it's read only because it's technically "archived".  I'm not crazy about that policy, but not everyone will agree with every policy.  But it would have been extraordinarily helpful to have the denial message explicitly say "Hey, it's read only!" or just have no reply button on the archived page at all.  What I thought should take me only 2 minutes took me 20, just to realize I can't really do it (add to the thread).

And now, aside from creating a new post, where and how should I contribute this idea?  Quite frankly, issues like this tend to keep me from posting to forums at all.  I think this is quite unfortunate, because formus often tend to have a lot more folks with questions than folks with (good) answers, from what I've seen.
