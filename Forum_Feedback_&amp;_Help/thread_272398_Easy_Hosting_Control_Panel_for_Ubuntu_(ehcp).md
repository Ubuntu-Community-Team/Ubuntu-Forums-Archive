### Post by bvidinli on 2006-10-06
i have built Easy Hostig Control (ehcp) Panel for Ubuntu.
you can review and download it at 
[http://freshmeat.net/projects/ehcp/](http://freshmeat.net/projects/ehcp/)
or 
[http://www.sourceforge.net/projects/ehcp](http://www.sourceforge.net/projects/ehcp)

features:
Easy Hosting Control Panel designed for hosting of multiple domains on single machine.It uses LAMP(LinuxApacheMysql).Its aim:easily installable,e.usage, non-complex,functional.


version 0.17 released !
many improvements:
* php and apt-get based simple install,
* automatically installs and works: dns, apache, mysql, ftp, email, domains,
* most easy install: copy & paste in console and watch
* ready to use after install - at ubuntu
* adodb and phpmyadmin included


future plans:
* reseller implementation,
* subdomain implementation,
* server/services status,
* plugin system using php

ehcp hosting control panel primarily supports ubuntu, tested on kubuntu,
but virtually work on other linux distributions with apt-get installed as well.
