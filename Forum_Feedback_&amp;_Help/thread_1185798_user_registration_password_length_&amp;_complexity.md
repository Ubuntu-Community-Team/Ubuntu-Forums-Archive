### Post by knomaly on 2009-06-12
Regarding the Ubuntu forums user registration process: I do not remember seeing any password rules that define either the password complexity or minimum-maximum character string length. 

If no rules are defined, it might be a good idea to include them below the respective text box sections which are located adjacent to one another.

A brief FYI: Perhaps one of the best user registration password processes I have seen is that created by the WordPress team. Besides defining their rules in detail, they promote the use of KeePass, and provide excellent information regarding what constitutes a strong password. Hence, they educate users about the critical nature of using unique, strong passwords for each site, including theirs, and they suggest the use of one superb, cross-platform password manager tool. Practical and educational, all rolled out on one page.
