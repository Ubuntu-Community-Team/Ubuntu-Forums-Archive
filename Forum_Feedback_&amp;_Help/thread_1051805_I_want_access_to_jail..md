### Post by matthew on 2009-01-27
It used to be open to anyone, a very long time ago. We closed it because leaving it open allows search engine spiders to crawl through it, which meant they indexed all the spam and other garbage in there. We didn't want that, and honestly, the stuff in there is not what we want the forums to be associated with; spam, grumpy or rude behavior, etc., but we want to keep some of it around in case we need an evidence trail.
