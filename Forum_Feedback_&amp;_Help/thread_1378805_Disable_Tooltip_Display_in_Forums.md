### Post by cariboo on 2010-01-15
If you are using Firefox try this:

>  Firefox provides a way to turn tool-tips off. Type &#8220;about:config&#8221; in the address box and hit the enter. It shows a list of configurable feature, find &#8220;browser.chrome.toolbar_tips&#8221; item and double-click to turn it value to false.

There doesn't seem to be a way to do it in Chrome.
