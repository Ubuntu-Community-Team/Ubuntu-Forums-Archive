### Post by cariboo on 2011-07-28
We aren't putting up barriers to any of the information on the forums. We made a decision to make the archives available to registered members only because all of the information in the archives is out of date, and could cause more problems than they solve.

The threads that were archived are for versions that are no longer supported. Making the information available to registered members only, will make the obsolete information disappear form Google queries in the next few weeks.

I would suggest that you include the version you are using, when making a Google search for answers to your questions.

### Post by lisati on 2011-07-28
The problem with old threads is that sometimes the way to use or configure programs changes.

For example, when I first started using Ubuntu 4 years ago, the Linux version of AVG free came with a GUI interface, which was good for people who were comfortable with using the Windows version. I remember seeing a lot of bad advice relating to using this product that could have been avoided if the people had bothered to read the documentation. More recent versions seem to be command-line only, so any advice given in old threads relating to the GUI no longer applies. Two problems: bad advice given in the past, and any good advice was given that is no longer applicable. (Remember this is just an example, and this thread is not the place for discussion of the merits of AVG and other AV products.)
