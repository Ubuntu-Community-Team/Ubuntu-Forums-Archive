### Post by dhave-dhave on 2011-05-25
I log on to the Ubuntu forum several times a day. I usually start by clicking "New Posts" to see what's come in since my last visit.

My question: I'd like to omit some forum sections ("Community Cafe", for example) from the New Posts that are displayed. I'd rather see only the technical sections that actually have to do with using Ubuntu.

Is this possible?

Thanks.

### Post by Joeb454 on 2011-05-26
Following [http://ubuntuforums.org/search.php?do=getnew&exclude=11](http://ubuntuforums.org/search.php?do=getnew&exclude=11) will do what you ask :)

Note that exclusion of forums is based from the forum ID (you can get this from the URL), so the above link excludes the cafe, with its ID of 11. If you wanted to exclude the cafe AND this particular forum, you would use [http://ubuntuforums.org/search.php?do=getnew&exclude=11,48](http://ubuntuforums.org/search.php?do=getnew&exclude=11,48)

Hopefully that will provide you with the answers you need. It'd probably be easier to bookmark the URL though, to save you having to type it in each time.
