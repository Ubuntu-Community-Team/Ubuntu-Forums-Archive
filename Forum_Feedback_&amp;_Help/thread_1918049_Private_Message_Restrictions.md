Hi, all. Just to add. as already mentioned there is no restriction on PMs for new members, but there is still a 75-post restriction on VMs (visitor messages). This was an anti-spam measure (visitor message spam) that predates the 50-post restriction for editing the user profile introduced to combat signature link spam.
