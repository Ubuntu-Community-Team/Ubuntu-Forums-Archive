### Post by coffeecat on 2013-10-31
> **Luc_Lacombe said:**
> Hello  I have FireFox and yes...javascript is enabled.

I'd like to double-check that with you. With recent versions of Firefox, you can't disable javascript globally - javascript is enabled by default. You have to rely on addons such as noscript or ghostery for disabling scripts. If, by chance, you are using either of those add-ons, make sure that you haven't disabled something specific to this forum. Don't block yahooapis.com, although I'm not sure if it's related to this issue.

Another possible cause for a lack of carriage returns in posts is the use of one version of Internet Explorer - I can't remember which. It doesn't play nicely with vbulletin. If you are using IE for the forum, switch the message editor from WYSIWYG mode to source mode using the icon in the toolbar that looks something like [size=1]A[/size]/A. Or in your Settings -> General Settings - Message Editor Interface, try either basic or standard editor.
