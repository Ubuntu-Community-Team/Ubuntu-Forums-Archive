### Post by ajgreeny on 2016-03-22
I have removed your thread that "went wrong" so it is now out of sight to users.

There should be no problem using either chromium or firefox for replying to threads or adding attachments; it is a forum property, not browser setting.

You need to use the Reply to Thread button and in the toolbar of that there is a paperclip icon which will take you to a dialogue to add images from your hard disk.  If you use the quick reply text entry box you will not see that icon nor many others, so just click the **Go Advanced** button and the full range of toolbar icons will appear.
