### Post by codejunkie on 2006-08-01
open up firefox and in the adress bar enter ```
about:config
``` copy and paste this in the Filter bar ```
browser.chrome.toolbar_tips
```then double click on the item to set it from true to false completly close all open firefox windows and restart firefox and they will be gone.

this also works in epiphany as well

hope this helps.
