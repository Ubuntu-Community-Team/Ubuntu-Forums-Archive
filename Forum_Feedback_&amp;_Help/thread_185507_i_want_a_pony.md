### Post by sudomania4 on 2006-05-31
The green icon is part of that weather stuff. It is the firefox extension forecastfox. The green icon is a doppler radar. The others tell current temp/conditions/wind and temp/conditions/wind for the day and he next day. There are three versions available: original- [https://addons.mozilla.org/firefox/398/](https://addons.mozilla.org/firefox/398/)
localized- [https://addons.mozilla.org/firefox/2024/](https://addons.mozilla.org/firefox/2024/)
and enhanced- [https://addons.mozilla.org/firefox/1978/](https://addons.mozilla.org/firefox/1978/)
I use the original forecastfox, and thats what is in the screenshot. I may switch to forecastfox enhanced, though, it looks nice and has a superior doppler radar. 

To use the extension, simply input your area code in the yellow box. That's all you need to do. Ignore the other windows and tabs in the extension settings dialog window unless you want to change the icons or get really deep modifications.
