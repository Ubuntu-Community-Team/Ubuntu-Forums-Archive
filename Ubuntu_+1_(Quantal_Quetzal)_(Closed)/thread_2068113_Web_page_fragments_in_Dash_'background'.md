### Post by VinDSL on 2012-10-08
I got rid of this ill-effect by disabling most of these features in CCMS -- features that are now enabled by default -- features that my setup cannot even use to good advantage.

This will result in a solid black background, e.g. no transparency in the Dash, which I didn't care for anyway.


[INDENT][[IMG]http://vindsl.com/images/vindsl-desktop-2-oct-2012-2(650x520).png[/IMG]]("http://vindsl.com/images/vindsl-desktop-2-oct-2012-2.png")[/INDENT]


These settings, when enabled, were cause tearing, ghosting, artifacts, and lagginess, et cetera.

You might want to play around with these, and see if it makes a difference.  If it doesn't make any difference, just return them to their default settings.  ;)
