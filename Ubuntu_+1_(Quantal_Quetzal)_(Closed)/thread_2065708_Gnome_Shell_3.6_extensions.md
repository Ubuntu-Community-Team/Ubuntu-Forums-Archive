Ok, I'll re-phrase:  a minor weakness and a big nuisance :)

I think the gnome way of adding stuff via extensions is just how it SHOULD be done. Give users a streamlined desktop and let them add the features and functions they want themselves via extensions.
However, I still see it as a weakness too as it leaves those users at the mercy of the developers of the extensions.
I understand they aren't going to be updated during alphas and betas and may take a while to be updated after release but there are a fair amount that haven't been updated to work with older versions of shell so the user has to make do without or hope someone else takes it on.
Of course users can just do it themselves...
