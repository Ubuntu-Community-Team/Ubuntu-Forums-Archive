I've been trying to use HUD a bit more and recently noticed that it behaves differently on the 12.10 partition.  

For example, I was renaming a few files in 12.04 using HUD and it worked like this: I'd arrow down to the file in the file manager, hit Alt for the HUD, start typing "rename," the "Edit>Rename" option would come up, I'd hit enter and it would work. 

When I tried to do the same thing in quantal HUD would throw out a bunch of suggested commands related to networking but not the edit>rename option I was expecting.  

While I was experimenting with this I also noticed that the menu in 12.10 shows more shortcuts than in 12.04: for example, if you go to the edit menu in 12.10 it shows that the rename shortcut is F2 while the 12.04 menu does not (one reason I was using HUD, I couldn't remember which fn key was supposed to trigger the rename command).  No idea if the two are related but it seems that 12.10 at the moment has less HUD but more fn keys functionality and vice versa in 12.04.

Has anyone else noticed that?  It's not a major issue, I'm just being curious and slightly bored before the alpha1 release :)
