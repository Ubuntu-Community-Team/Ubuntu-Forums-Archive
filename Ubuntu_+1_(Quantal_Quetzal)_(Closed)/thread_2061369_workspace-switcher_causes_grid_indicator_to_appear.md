try this:

[LIST=1]
[*]Open any window on any workspace
[*]Press super + s or the workspace-switcher button in the launcher.
[*]Select the workspace with the opened window in it
[*]Move the mouse to the left or the right edge
[*]The Grid indicator will now appear
[/LIST]
This issue has appeared before in 12.04, but I cannot find the bug report any more.

Edit:
Even easier, try this:

[LIST=1]
[*]Open the workspace-switcher
