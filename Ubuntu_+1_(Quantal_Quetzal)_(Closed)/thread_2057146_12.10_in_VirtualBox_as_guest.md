### Post by ergo-proxy on 2012-09-14
As far as I can see the latest virtualbox 4.1.22 doesnt come with virtualbox guest additions supporting X 1.30... so how did somebody got it working (screen resizing, higher resolution etc) ??

Here is a part of a config file:

```
case $x_version in
        1.*.99.* )
            echo "Warning: unsupported pre-release version of X.Org Server installed.  Not"
            echo "installing the X.Org drivers."
            dox11config=""
            ;;
        1.12.* )
            xserver_version="X.Org Server 1.12"
            vboxvideo_src=vboxvideo_drv_112.so
            test "$system" = "redhat" || setupxorgconf=""
            ;;
```

### Post by ergo-proxy on 2012-09-14
Ok I got it solved :)

What **I did wrong** (might be a bug too): 

- installed guest additions via virtualbox (didnt work)
- later on I installed guest additions from a repository, it did work but I wasnt able to pick and apply a driver from "an additional dirvers" tab.

I got it working by doing a clean installation but removing all installed virtualbox packages would work too, updated the system,** install the additions from the repository**, enable the driver from 'additional drivers' tab, reboot and voila :KS
