### Post by VinDSL on 2012-07-08
[s]This might work too[/s]...

```
[s]From cli:

gconftool-2 --type string -s /apps/nautilus/preferences/click_policy "single"[/s]
```

**EDIT**

Just tested it -- didn't do jack!

Next, I used dconf (as mc4man suggested above)

```
Using dconf gui:

dconf-editor -> org/gnome/apps/nautilus/preferences/click_policy -> single
```

Worked fine!  ;)
