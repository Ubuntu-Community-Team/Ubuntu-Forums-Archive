### Post by jerrylamos on 2012-07-03
Here on the development testing forum we share technical problems, "heads up", fixes & workarounds, what's coming, what needs to be looked at, useful hints, etc.

Now we on occasion do vent some spleen which generally doesn't do much good.

Of course the idea is to test out unstable often broken code and enter bugs on "Launchpad" so development can get some feedback.  Development doesn't follow this forum but they do keep track of bugs - some I've entered do get fixed, some don't.  That's their choice.  

There's a bunch of us with a lot more hardware and setups than development has so it is advance warning on what might happen when code is released.

Have fun.  Don't get frustrated....

Jerry
