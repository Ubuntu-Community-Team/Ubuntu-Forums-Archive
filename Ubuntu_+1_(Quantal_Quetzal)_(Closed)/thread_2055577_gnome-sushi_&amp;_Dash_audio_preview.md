### Post by mc4man on 2012-09-09
If wishing to use gnome-sushi in 12.10 then one would need to make sure that this package is installed as nothing will install it other then the current sound-juicer
gstreamer1.0-plugins-good
[https://bugs.launchpad.net/ubuntu/+source/gnome-sushi/+bug/1048356](https://bugs.launchpad.net/ubuntu/+source/gnome-sushi/+bug/1048356)

If you want fuller playback support then - gstreamer1.0-libav 
and for mp3 - gstreamer1.0-plugins-ugly

The unity Dash now has an audio preview thru the Previewer, it uses RB
Not exactly obvious, clicking on the little # to the left of track name will expose an arrow, click to start. screen
Currently has bug that when dismissing the Dash the audio continues, should be fixed shortly
[https://bugs.launchpad.net/ubuntu/+source/unity/+bug/1045298](https://bugs.launchpad.net/ubuntu/+source/unity/+bug/1045298)
