The package that allows you to have appmenu for Libreoffice is lo-menubar. It wasn't that stable for the previous releases but has been improved upon for this release. It isn't currently installed by default but you really want to have the menu support, you can go ahead and install it. It mostly works without much of a problem. 

If in case they make it default, you should be notified by either of the blogs.
