### Post by jfernyhough on 2012-09-09
The Cinnamon nightly PPA has builds for quantal (plus Cinnamon 1.5 is a fair bit better than 1.4):

[https://launchpad.net/~gwendal-lebihan-dev/+archive/cinnamon-nightly](https://launchpad.net/~gwendal-lebihan-dev/+archive/cinnamon-nightly)

If you want, you can use the stable PPA version but you'll have to edit the source (e.g. in Synaptic or Software Sources) and change it from quantal to precise.
