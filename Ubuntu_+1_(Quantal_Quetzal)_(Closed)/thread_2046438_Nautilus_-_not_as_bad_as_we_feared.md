The easiest way is to go to this [page]("https://launchpad.net/~gnome3-team/+archive/gnome3"), and add the ppa to your sources list. The command to do it via the command line is:

```
sudo apt-add-repository ppa:gnome3-team/gnome3
```

Then run:

```
sudo apt-get update && sudo apt-get upgrade
```

You can also of course use graphical tools to install it, just make sure your software sources have been updated before hand.
