### Post by twipley on 2012-10-09
Just to note that the current state of Quantal permits going to:

Edit > Preferences > Desktop Integration > Untick "Prompt integration options for any website"

in order for the Web Apps feature to be annihilated.

Alternatively (and for power users!), settings can be tweaked over at dconf > com > canonical > unity > webapps.

Hope that helps someone!
