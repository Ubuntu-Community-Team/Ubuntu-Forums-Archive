### Post by funicorn on 2012-10-06
look here [https://github.com/ali1234/fullscreenhack](https://github.com/ali1234/fullscreenhack)

only works for xinerama, not for twinview

```

git https://github.com/ali1234/fullscreenhack.git
cd fullscreenhack
make

```

then use it
# for firefox
env LD_PRELOAD=/path/to/libfullscreenhack.so firefox %u

# for google chrome
env LD_PRELOAD=/path/to/libfullscreenhackppapi.so google-chrome %u
