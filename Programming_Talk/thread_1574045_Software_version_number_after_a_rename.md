### Post by techgeek314 on 2010-09-13
I have a project written in C++ (hosted on github if you're interested) called "Class Preview".  It uses SDL, SDL Image, SDL TTF, and SDL gfx.  The basic idea is that it makes images and pieces of text appear on the screen (either moving and bouncing off of the sides of the screen or standing still) for a time and then disappear.  You can also control other stuff, like the average font size and a background.  The user writes a configuration file that tells the program what he/she wants it to display.  It is intended for use by a teacher at the beginning of a class, like when the students are arriving and sitting down.  The idea is that it would have images and pieces of text on the screen about the stuff that will be discussed/done in class.

I understand that this is a frill; it is not going to revolutionize education.

I would like to rename this program; "Class Preview" isn't a really exciting name (and it might make it sound like something different to a programmer initially).  The latest release is 1.1.  If I were to keep the name the same, the next number I would use would be 2.0.  What number should I use?  Should it be __NAME__ 1.0 or __NAME__ 2.0?
