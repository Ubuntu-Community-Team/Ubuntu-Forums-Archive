### Post by WW on 2007-04-02
To get started? Nothing. Python is already installed.
For example,
```

$ python
Python 2.4.3 (#2, Oct  6 2006, 07:52:30)
[GCC 4.0.3 (Ubuntu 4.0.3-1ubuntu5)] on linux2
Type "help", "copyright", "credits" or "license" for more information.
>>> x = 100
>>> print x
100
>>>

```
(I am using dapper; you might see some differences in the initial message if you are using edgy.)
