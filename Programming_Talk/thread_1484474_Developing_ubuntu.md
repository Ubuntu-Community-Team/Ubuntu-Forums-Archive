The kernel is written in C with some assembler code, most of the other software is written in C, C++, Python, Perl and C#. There are a lot of sh scripts, too.

If you want to help a good way to start is to help triage and fix bugs, see [https://wiki.ubuntu.com/Bugs/](https://wiki.ubuntu.com/Bugs/)
