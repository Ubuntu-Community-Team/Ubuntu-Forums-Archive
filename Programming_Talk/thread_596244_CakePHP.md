### Post by dataw0lf on 2007-10-29
It certainly is impressive for a web framework written in PHP (which is a pseudo web framework in and of itself).  I've done some work in CakePHP (mostly pen testing existing applications), and it seems to be the best possible way to implement an MVC/RAD framework in PHP.  I'd recommend it. 

CodeIgniter isn't too bad either.  I'd steer clear of Symfony, though.
