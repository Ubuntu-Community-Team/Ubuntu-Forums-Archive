### Post by MicahCarrick on 2007-10-06
You can write the files for Gedit. They are XML files. You can create your own based on an existing one. It's very easy. Then you drop the .lang file in ~/.gnome2/gtksourceview-1.0/language-specs/ (assuming it's gedit 2.18 or earlier which uses gtksourceview-1.0).

The original .lang files which will give you a starting point are probably in /usr/local/share/gtksourceview-2.0/language-specs/c.lang
