I'm working on a plugin for wireshark and I want to put it on a git repo. But the developers who are gonna be using and testing it out don't want to have to compile wireshark with the plugin.

What I want to implement is a situation where the developers, who already have a copy of wireshark installed, simply pull a file from the git repo(whether it be a rpm, or a patch or whatever) and said file updates their copy of wireshark to include the plugin. This would make the process much faster as everytime an update is committed they just grab a file and install it.
