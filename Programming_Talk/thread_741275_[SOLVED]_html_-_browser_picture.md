### Post by mssever on 2008-04-01
> **stooshbunutu said:**
> can gimp make an .ico file from a png?

An .ico is an archive of (differently-sized) .bmp images. You can rename foo.bmp to foo.ico for basic functionality, and there are websites that will convert several different-sized icons to an .ico file.
