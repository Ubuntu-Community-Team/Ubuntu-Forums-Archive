### Post by rootkowski on 2009-07-22
HI everyone!

My response perhaps is a bit late but it still might help someone. I installed Flex Builder on Mint 6 (Ubuntu 8.10) some time ago and had some issues too. What I found out was that the version of Eclipse you need to use was 3.3.2. It just wouldn't work with 3.4.

You can also use NetBeans 6.5 with a plugin for Flex development. Because of issues with Flex Builder and java performance on my machine (NetBeans) I nowadays use Komodo Edit plus Flex SDK. Runs fine and compiles as it should. For details on setting up Komodo for Flex you can take a look at my blog: [http://blog.rootkowski.com/get-started-with-flex-development-on-linux/]("http://blog.rootkowski.com/get-started-with-flex-development-on-linux/")

Hope this helps someone eager for some Flex development :-)
Cheers!

EDIT: just found a post about setting up Flex Builder with Eclipse 3.4. You can read it here: [http://ubuntuforums.org/showthread.php?t=1103588]("http://ubuntuforums.org/showthread.php?t=1103588")
