As I recall, GPL v3 prevents your code from being put into places where you're legally allowed to change it, but is practically infeasible. This defeats the purpose of GPL'ing it in the first place, so they added this clause.

GPL v2 didn't have this, which is (I think?) the reason Linux didn't adopt it, as it is used in these scenarios.
