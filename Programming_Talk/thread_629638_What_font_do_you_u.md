### Post by wolfbone on 2007-12-02
Bitstream Vera or Deja Mono have good brackets and good distinction between 0 and O. Liberation Mono doesn't have the diagonal through the 0 but it's very good too. Other good programmer fonts are ProFont, Ti92Pluspc and Monaco - if you can get hold of them - but they have limited character sets and no italics etc. (This isn't a problem for program code but it makes them less suitable as a primary font for Emacs (Xft, unicode-2 branch) because of all the other stuff Emacs needs to display).
