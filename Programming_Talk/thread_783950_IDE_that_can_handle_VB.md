### Post by samjh on 2008-05-06
What kind of VB is it?  Classic Visual Basic or the .NET version?

If you are using VB.NET, then install the monodevelop package.  It is an IDE very similar to Visual Studio, and can import Visual Studio projects.  It uses the Mono project instead of .NET.  Beware that Mono does not implement ALL of .NET's features, although it does implement MOST of it. :)
