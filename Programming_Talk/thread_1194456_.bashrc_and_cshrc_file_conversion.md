### Post by .Maleficus. on 2009-06-23
Try adding this line to your .cshrc:

```
set prompt='\n\[\e[01;30m\]\h:\[\e[34m\]\w\n\[\e[00;32m\]\u\[\e[0m\] > '
```
Being a zsh user I have no idea how that will work but from what I gather about the .cshrc, that is the syntax.
