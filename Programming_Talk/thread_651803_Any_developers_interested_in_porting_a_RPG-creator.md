### Post by bepe86 on 2007-12-28
Hello. I'm in the process of planning a port of a popular Windows-only RPG-engine / creator to other operating systems.

At the moment, the [RPG Toolkit Development System](http://toolkitzone.com) is a GPL'd system for creating and playing Role Playing games. The system itself is very flexible, and has an active community. I think that porting this to other operating systems can provide both the toolkit community and the linux community with both talent and great experiences.

The engine itself is written in C++, using DirectX for input and graphics, and using Audiere for audio.

If anyone thinks this sounds interesting, please stop by the toolkit website, [this](http://www.toolkitzone.com/forum/viewtopic.php?p=426245) forum post, and [this](http://bepe86.com/rpgtk) wiki.

Let's give Ubuntu-users an excellent, creative tool which can be used for both education and entertainment!
