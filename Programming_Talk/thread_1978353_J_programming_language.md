### Post by r-senior on 2012-05-11
The place to download is here:

[http://www.jsoftware.com/](http://www.jsoftware.com/)

The "install" just unpacks to a directory under your home directory IIRC. Then you run the jwd program from the bin directory. You can make a .desktop file if you want to run it from the launcher.

You'll also find links to guides on that site.

Amazing programming language but if you don't use it for a couple of weeks, you just forget it all. It's so terse. But great fun.
