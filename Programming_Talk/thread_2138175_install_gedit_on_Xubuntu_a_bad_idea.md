### Post by Elfy on 2013-04-23
Go for it - other than installing a few packages it should be fine.

If you want to only install the bare minimum, you could install from a terminal with 

```
sudo apt-get install gedit --no-install-recommends
```

I've got gedit installed here aas well as nautilus.
