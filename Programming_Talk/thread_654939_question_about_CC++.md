### Post by wh0rd on 2007-12-31
There are a plethora of apps in Linux for C/C++. I don't know the names of all but my favorite is Geanie it has a GUI, it's light weight and has command line options. Also supports html, python...

Another popular IDE is Eclipse. 

Other than IDEs there are command line for [gcc]("http://gcc.gnu.org/") (the recommended compiler for c/c++)

You can use command line editors also like nano, kino, and popular ones like vi and emacs. Save you files in .c and just run gcc to compile it.
