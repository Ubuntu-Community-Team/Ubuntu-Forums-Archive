### Post by pmasiar on 2007-08-11
You need to learn programming first. Best language is Python, see wiki in my sig.

After that, you will need to pick the project and learn the language it uses. Some are in Python, but many are in C/C++ or other languages(Perl/Java/PHP/Ruby etc).

Python is comparatively easy (easier then most other languages, like C/C++/C#/Java) and fun. Has great library for games (pygame), but for start, just do plain text programs - GUI is hard.
