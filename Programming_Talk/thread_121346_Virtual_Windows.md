### Post by Seq on 2006-01-25
There is a version of VMWare for Linux. I used it to do windows programming for two semesters at school ;)

However, it can't do hardware accellerated 3D like OpenGL or DirectX. You would have to dual boot to play windows games (I keep windows around on my desktop for this purpose). 

You could also try Transgaming's Cedega, which is basically an extension of the Wine project, and it implements DirectX API calls. Compatability varies from game to game, however.
