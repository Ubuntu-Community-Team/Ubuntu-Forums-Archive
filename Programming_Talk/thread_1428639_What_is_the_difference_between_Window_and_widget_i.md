### Post by km0r3 on 2010-03-13
From Wikipedia:

[Widget]("http://en.wikipedia.org/wiki/GUI_widget"):

> In computer programming, a **widget** (or **control**) is an element of a [graphical user interface]("http://en.wikipedia.org/wiki/Graphical_user_interface") (GUI) that displays an information arrangement changeable by the user, such as a [window]("http://en.wikipedia.org/wiki/Window_%28computing%29") or a [text box]("http://en.wikipedia.org/wiki/Text_box"). The defining characteristic of a widget is to provide a single interaction point for the [direct manipulation]("http://en.wikipedia.org/wiki/Direct_manipulation") of a given kind of data. In other words, widgets are basic visual building blocks which, combined in an application, hold all the data processed by the application and the available interactions on this data.

[Window]("http://en.wikipedia.org/wiki/Window_%28computing%29"):

> In [computing]("http://en.wikipedia.org/wiki/Computing"), a **window** is a visual area containing some kind of [user interface]("http://en.wikipedia.org/wiki/User_interface"). It usually has a rectangular shape.[[1]]("http://en.wikipedia.org/wiki/Window_%28computing%29#cite_note-ArsTechnica3-0") It displays the **output** of and may allow **input** to one or more [processes]("http://en.wikipedia.org/wiki/Process_%28computing%29").
