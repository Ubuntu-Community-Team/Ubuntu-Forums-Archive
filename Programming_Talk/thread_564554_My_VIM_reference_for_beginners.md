### Post by samjh on 2007-10-01
I've been learning to use Vim lately, and it has grown on me as a very quick code editor.  The only downside was the steep initial learning curve, and time it took to become accustomed to using only keyboard commands.

During the learning phase, I had a look around the internet for quick reference cards for Vim.  Most of them were overkill, too sparse, or too verbose.  So in true FOSS community spirit, I made a quick reference just for my own and attached it here (zipped pdf).

This is pretty basic stuff, but these are 95% of features I use in Vim.  I hope it helps out the beginners here.

To install Vim (the proper version, not the default stripped-down version):
```
sudo apt-get install vim
```

Enjoy. :)
