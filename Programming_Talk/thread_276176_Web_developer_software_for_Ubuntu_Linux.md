### Post by Jolly Roger on 2006-10-13

### Post by almahtar on 2006-10-13
As one person mentioned, Nautilus is great for sftp and ftp.  Just hit ctrl + l, type in [ftp://user@servername](ftp://user@servername), hit enter, type your pass, and it mounts the ftp server just like a local folder.  You can drag a file from it onto gedit (or other programs that use Gnome's VFS), and just hit "save" from that program and it'll connect to the server, log in, and upload the file completely transparently.

I love it.
