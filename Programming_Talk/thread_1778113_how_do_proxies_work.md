If proxies work by letting you connect to the website you want to go to through another server.
For Example:
The website you want to go to is Server A, but it is blocked. The proxy would be Server B and is not block. You would basic connect to server B and tell it to download information from server A, then server B now with the information you wanted will be sent you.
More Simply put: The proxy is a unblocked middleman.

By the way Ultrasurf is one of the best proxy I know of that is free, but it is many times detected as a virus by different anti-virus software. Many people say it is a false positive but I go head and use it to watch youtube videos at school (without login). Also if your school does not allow exe files use a usb stick and disable the anti-virus from removing it if it does.

If you want to go on to Facebook I would recommend typeing the url as [https://www.facebook.com](https://www.facebook.com) .
