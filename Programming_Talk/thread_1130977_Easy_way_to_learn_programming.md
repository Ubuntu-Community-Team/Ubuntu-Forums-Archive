the best way to learn is to work towards a goal or project of some sort

goal 1. create a python/gtk application that adds 2 numbers together

sounds really simple and dull, and you'd be half right. it's really dull. but in learning to create that app you will learn, assuming you have no knowledge about programming atm:

basic programming concepts such as
variables
methods
types
type casting

you'll also learn
python
gtk
generic window programming (signals and callbacks)

and whats more it will be a graphical application which is rewarding. there is nothing less rewarding than working for hours on an application that just spits out console text.

once you've completed your project you'll know so much more and be able to choose a new project/goal that is more interesting to you and makes you learn a new set of skills
