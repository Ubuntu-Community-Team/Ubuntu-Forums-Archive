I have written a python program which hopefully makes using your DVB-T TV  card with Gxine easier.
It asks you for your region, generates a channels.conf file in the .gxine directory and overwrites the media marks in Gxine with the channels found for easy channel selection.

It requires Python, PyGTK, libglade and the scan tool.
