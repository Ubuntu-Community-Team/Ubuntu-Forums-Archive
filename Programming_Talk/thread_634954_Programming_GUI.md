### Post by MicahCarrick on 2007-12-08
Personally, I like GTK+ with C or Python programming. Using Glade is like using VB or VC++ designer and then code can be written using any number of languages including C, C++, Python, Ruby, PHP, and more. Applications written for GNOME are typically using GTK+. If you have VB experience, you may want to read [VB Programmer’s Intro to Linux Programming with GTK+]("http://www.micahcarrick.com/08-19-2006/vb-in-linux-with-gtk.html")

Qt is the toolkit used by most KDE applications and also has many languages available, though C++ is probably the most common.

Mono is a C# language for Linux which uses GTK+ and comes with a nice IDE, monodevelop.

wxWidgets and the Java environments mentioned are also worth checking out.

I would recommend you look at Qt, GTK+, and Mono and make a decision from there.
