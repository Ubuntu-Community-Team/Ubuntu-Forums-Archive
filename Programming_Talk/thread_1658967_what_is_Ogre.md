### Post by trent.josephsen on 2011-01-03
No offense, but based on this and some other posts you have made, I doubt you have anywhere near the necessary experience to start mucking about with 3d rendering.  Don't try to run before you can walk.

If you're relatively new to programming, I suggest learning Python.  If you already know a language, learn how to best use what you know and consider expanding your skill set (C++, Java, and Python are all popular in game development).  Start with simple terminal programs and put a few months-to-years in before you start bothering with 3d stuff.
