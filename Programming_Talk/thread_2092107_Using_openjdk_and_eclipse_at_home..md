### Post by r-senior on 2012-12-07
You have rights to do whatever you like with software you create with Java and Eclipse (or Netbeans) using Ubuntu. If you want to sell it as a product, that's OK, if you want to make it open source, that's OK. You own any software you create. You don't need to register anything.

If you start including third-party libraries you need to check their license terms.
