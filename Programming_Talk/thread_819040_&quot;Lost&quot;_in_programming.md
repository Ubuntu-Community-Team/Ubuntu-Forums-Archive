### Post by Cappy on 2008-06-05
My opinions:
If you want to work with the kernel learn C with specially focus on pointers, memory allocation, malloc, etc. Most programs are written in C/C++ obviously.

If you want to write general programs, especially GUIs or text processing, I would learn python [instead of Java]. It's considered a good beginner language.

If you want to automate system tasks then learn BASH scripting.

You'll end up learning all 3 eventually if you stay in Linux territory.

Java is decent, I've had to use it for 4 classes but I don't use it for anything else.

Final advice:
Just learn something you can use immediately and have fun with. Nothing else really matters imo.

### Post by descendency on 2008-06-05
I strongly recommend buying/finding a book or taking a class at a local college (that's the best option). While many of the languages listed have adequate tutorials across the internet, you (as a novice programmer) probably won't be able to determine the quality of the tutorial. Bad tutorials often lead to bad programmers or infinite amounts of confusion.

The easiest way to learn is to do all of the programming exercises from a book. If you complete them all and still do not know the language, then it was a poorly written book.

### Post by pmasiar on 2008-06-06
> **reidms said:**
> I know a fair amount about commercial UNIX operating systems, Linux, BSD, hardware, internet security etc. etc.,

Looks like you are sysadmin/security guy?

So next step for you could be to start writing programs helping you to automate tasks you do, because you know what you want to do, and automating them will free you time to learn more.

Python is excellent tool for that. And with generator expressions, you can build extremely flexible pipeline to analyze huge files in sophisticated ways without wasting too much memory.

Don't waste time on Java or C++: they are not suitable to automate work you do. Another option might be Perl, but it is becoming obsolete (even if widely used by legacy scripts), being replaced by Python.

Python is easy to learn by your own (see wiki in my sig), colleges teach Java for code monkeys, with your skills you do not want to compete for jobs with them.
