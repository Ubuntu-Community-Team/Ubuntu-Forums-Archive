### Post by Ripfox on 2009-01-17
Well ATM, I figured out something about the Flex Builder plugin and Eclipse under Ubuntu 8.10.

You have to install Eclipse 3.3  "Europa" (3.4 doesn't seen to work) 

The 3.2 that is in the repos doesn't work with it either.

If you have this version  installed which is basically just a folder you unzip, then you can install the new Flex Builder for linux by pointing the installer to that folder. 

Hope this helps someone b/c it took me like 3 hours to figure it out. :)
