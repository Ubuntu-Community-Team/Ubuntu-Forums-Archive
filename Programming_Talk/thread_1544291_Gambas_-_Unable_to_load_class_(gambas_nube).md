### Post by kalaharix on 2010-08-02
Hi

10.04 software centre installation is annoyingly incomplete.

Try either:

'sudo apt-get install gambas2' without apostrophes from Applications, Accessories, Terminal

or:

System, Administration, Synaptic. Search on Gambas and install components that were left unistalled by Software Centre.

Many of the examples will say 'read only' so save them to a sub-directory in your home directory so that you can mess around with them.
