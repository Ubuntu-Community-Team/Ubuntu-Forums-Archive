Well the reason I haven't spilled the beans is because the mockup isn't quite finished, but I suppose I can post what I have so far.  

Now, my idea is not a panel.  It's more like a multipurpose display area for everything from calculators, to messengers, to a callender, to a favorite applications dock, to a folder browser.  The folder browser would be nice to avoid having a separate window when looking for something.  It could all be in one space.  Now, all this would be able to fit in this one space by replacing what was there previously.  It would start out with the main display bar menu ([for which I have made the mockup here]("http://i79.photobucket.com/albums/j131/TypeL/sold-color.png")) and you can select one of the options.  Then the main menu will go away and be replaced by whatever you choose.  Then you can hit a small button on the side of it to go right back to the main display bar menu.  You could also have 2 of these display bars on the same "panel", and you could different apps open on different workspaces.  

I think people would like this because it would look nice on the desktop and would put many commonly used applications and resources in one place, which I think would make everything a bit easier and organized.  And since it is just an alternative way of doing things, it wouldn't have to take away from anything else.  It can also limit what you'd need to put on the panel.  If I had it, I would just use the panel for window management, and everything else would be up on the display bar.  You also wouldn't need a dock, or any screenlets on the desktop.

I know it isn't terribly original.  I mean, it's kind of like a souped-up sidebar + panel + and dock all in one...plus a few other things.  So, if anyone is interested in this, please let me know.
