### Post by Ankur Dave on 2007-02-23
In Ubuntu, the Meta keys are already mapped to Alt. Emacs accepts the Alt key as a Meta key. For example, typing Alt-X doctor is the same thing as typing M-x doctor.

You can see this if you go to System -> Preferences -> Keyboard. In the Keyboard Preferences window, click the Layout Options tab. Click Alt/Win key behavior to expand it. You will see "Alt and Meta are mapped to the Alt keys (default)." As it says, this is already the default.

Hope that helps!
