### Post by Treutwein on 2009-10-06
There is the BlackBox Component Framework, which runs nicely with Wine under ubuntu LTS. It implements Component Pascal, which is a the heir of Oberon/L ...

It is available at [http://www.oberon.ch/Blackbox.html](http://www.oberon.ch/Blackbox.html)
There is a slightly newer release candidate available here:
[http://www.oberon.ch/zip/SetupBlackBox16-rc6.exe](http://www.oberon.ch/zip/SetupBlackBox16-rc6.exe) 
If you are interested in the changes for the last release candidate, please mail me at bernhard (at) wildwein de
