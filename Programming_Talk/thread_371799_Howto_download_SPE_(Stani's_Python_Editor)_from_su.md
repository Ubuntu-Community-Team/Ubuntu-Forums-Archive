SPE is a free python IDE for Windows, Mac & Linux with UML features, PyChecker source code docter, Debugger (WinPdb), GUI designers (wxGlade and XRC editor) and Blender (3D graphics) support. After some hibernation, development has continued and here is how to get the latest version before it will be released:
