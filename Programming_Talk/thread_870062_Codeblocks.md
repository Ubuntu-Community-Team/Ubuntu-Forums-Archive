### Post by GreenN00b on 2008-07-25
Go here: [http://www.codeblocks.org/downloads/5#linux](http://www.codeblocks.org/downloads/5#linux)

Download codeblocks_8.02-0ubuntu1.deb.tar.gz or codeblocks-8.02-amd64.tar.gz depending if your are running on x32 or x64.

Right click on file and select extract.

Right click on the deb file and select install.
