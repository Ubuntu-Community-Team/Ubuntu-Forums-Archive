### Post by Some Penguin on 2010-09-20
> **lordpancho said:**
> So, software like cvs, svn, bamboo, git, mercurial or any other can work with any porject that i have in fortran????

They're storing files.  There's no need for any such thing as "language support" for version control systems, any more than closets or wardrobes need to support particular types of shirts --  all they need to do is store data and deltas.  They're not compiling files, interpreting them, formatting them, or doing anything at all like that.
