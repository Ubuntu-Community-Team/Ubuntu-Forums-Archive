### Post by LaRoza on 2007-05-04
Why don't you start by making things that would be useful to yourself? I agree, to learn programmers need projects to tackle, but I think making something you can use is more fun. Of course, you don't get solutions, although since the user (you) is the judge, you can do anything you want.

If you are really just starting out in programming, try making a script to print out the lyrics to "100 Bottles of Beer on the Wall", remember, 1, is singular and 0 is plural. If you want a little more challenging project, make a script to check if an XML document is well formed and look for errors.

If you are just new to Python, try doing projects you did in other languages, using Python.
