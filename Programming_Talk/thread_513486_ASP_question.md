### Post by pmasiar on 2007-07-30
There is fork in your road:
1) float with the flow - learn "Linux way" to accomplish the same tasks like ASP does: LAMP stack, use Perl/PHP/Python/Ruby.
2) be contrarian - keep using ASP (proprietary MSFT technology) and try to find ways to run it on Linux: Mono and friends

AbyssWs web server is something contrarian again, you may have better luck with help with more common servers like Apache or (for Python) CherryPie
