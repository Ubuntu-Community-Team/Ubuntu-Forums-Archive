### Post by kaelspencer on 2010-07-19
I use [Aptana]("http://www.aptana.com/"). It can either be standalone or a plugin for Eclipse.

As for FTP, consider mounting the FTP share: 

Places->Connect to a Server->FTP[ with login]

I find it much easier, as it acts like local files.
