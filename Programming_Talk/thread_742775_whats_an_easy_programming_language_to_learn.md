### Post by tseliot on 2008-04-02
I started with shell scripting, then I learnt Python (which I warmly recommend). I think you can start with Python or any other (very) high level language as LaRoza suggests.

When you have learnt (or even better, mastered) Python, if you feel that you need your programs to run faster you can learn C++ (with QT4, yes, even for command line applications, since it will make your life a lot easier), C, etc.

Of course it depends on what your planning to do with these languages.
