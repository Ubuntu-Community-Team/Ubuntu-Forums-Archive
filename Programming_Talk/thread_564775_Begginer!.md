### Post by Hoff-Z on 2007-10-01
You can use Eclipse for almost any language, it's just the matter of finding a good plug-in. Plug-ins for Eclipse can be found [here]("http://www.eclipseplugincentral.com/Web_Links+main.html").
I use Eclipse for Perl, PHP and HTML, and I love it.

I can't promise that it rocks with Ruby, but you can try out an Eclipse/Ruby plug-in here: [http://rubyeclipse.sourceforge.net/download.rdt.html]("http://rubyeclipse.sourceforge.net/download.rdt.html").
(jump to the "Using the Eclipse Update Manager" section)
