Where would you like to put it? A control-letter combination? Actually, long ago, C-h used to be equivalent to Backspace, because the original meaning of ASCII Ctrl-H is backspace, but then they put Help on C-h.

Delete used to be a synonym to Backspace in Emacs, but that's even further away from the home row. Hmm, I can't have used it lately - it deletes forward now, like in text windows in Windows. These old conventions of course come from how physical terminals behaved.

I don't know if there is any alternative place to put the backspace function that has become particularly popular.
