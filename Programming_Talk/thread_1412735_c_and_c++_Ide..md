### Post by GregBrannon on 2010-02-21
Another vote for Code::Blocks.  It provides beginners with advanced IDE features and the flexibility to quickly create simple file applications that can be run outside a project.  I've learned the value of programming inside projects, but the speed and simplicity of creating small, beginner console applications outside a project is a nice option.

Easy to install, easy to run and learn, and able to grow with you as you move from the simple to more complex programming exercises.
