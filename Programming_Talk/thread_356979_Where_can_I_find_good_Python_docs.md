### Post by christhemonkey on 2007-02-09
Or you can view the default python docs on your system.

Type:
```
python
```

Then to search for a suitable module:
```
help("modules foo")
```

Or to look up the docs on that module:
```
help("foo")
```


If you want to view the docs for the builtin commands you need to install python-doc then:
```
help("__builtin__")
```

You can also view documentation outside of the interpreter by doing this:
```
pydoc foo 
```
