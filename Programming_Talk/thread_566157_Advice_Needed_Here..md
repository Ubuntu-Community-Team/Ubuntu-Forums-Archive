### Post by Wybiral on 2007-10-03
Open source usually works like this:

1. Are there any projects that you like and want to help out?
2. Are there any bugs in the software that you use that you might be able to fix?
3. Do you feel a particular application is missing from the community (or one exists, but isn't very good)?

If you can answer one of those, then you should be headed in the right direction.
