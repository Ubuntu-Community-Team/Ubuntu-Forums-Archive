### Post by MeanEYE on 2009-12-18
It's been a while since this program showed its face but here it is again. Unless you tracked changes on one of sites ([Ohloh]("https://www.ohloh.net/p/sunflower"), [FreshMeat]("http://freshmeat.net/projects/sunflower"), [Google Code]("http://code.google.com/p/sunflower-fm/"), [AlternativeTo]("http://alternativeto.net/software/sunflower/")) here's a short list of things finished in this program since last update:

- copy, move and delete are fully functional
- progress dialogs can be hidden 
- bookmarks menu with active mounts
- new mounts are detected automatically
- configuration is easy now with options dialog
- fullscreen mode

and many other things I can't really remember now. :)

Take a look at newer [screenshots]("https://picasaweb.google.com/MeanEYE.rcf/Sunflower#").

If you have feature request, bug report or anything you want to let me know, please do! :)
Oh, and if you are willing to share your desktop screenshot with Sunflower on it, I'd really like to see those. :)
