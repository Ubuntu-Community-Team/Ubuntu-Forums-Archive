### Post by Zugzwang on 2012-01-31
Don't expect to find a tutorial that includes both libtar and gedit - this combination is too special. As far as I know, gedit is only an editor, so there is nothing extra that would need to be covered in such a tutorial anyway.

Having said that, look at this page: [http://stackoverflow.com/questions/813223/how-to-compress-a-directory-with-libbz2-in-c](http://stackoverflow.com/questions/813223/how-to-compress-a-directory-with-libbz2-in-c) - Apparently, libtar has no extra documentation or tutorial, but rather a documented header file you should have a look at. The page also contains an example to get you started, including the command to compile it.
