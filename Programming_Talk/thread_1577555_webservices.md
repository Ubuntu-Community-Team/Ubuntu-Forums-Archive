### Post by dv3500ea on 2010-09-19
What programming language(s) do you know. There are lots of web frameworks and servers in lots of different languages. If you tell us your favourite programming language we could give you a list of resources for that language.

There are some things that will be useful to know for any programming language, even if it is just to get an understanding of hoe the frameworks work:
[LIST]
[*] [http]("http://www.jmarshall.com/easy/http/") - the protocol used by most web services.
[*] [sockets]("http://en.wikipedia.org/wiki/Internet_socket") - the low level details of how networking works.
[*] [JSON]("http://www.json.org/") - A simple, flexible, human readable data format that can be used in pretty much every programming language.
[/LIST]
