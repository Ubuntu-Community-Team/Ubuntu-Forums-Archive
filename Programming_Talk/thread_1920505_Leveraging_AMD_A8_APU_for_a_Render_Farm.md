### Post by Richard 2112 on 2012-02-04
**Challenge:** 
Building a render farm is an expensive endeavor involving thousands of CPU cores which also requires enormous amounts of electric power and cooling. 

**Potential Solution:** 
AMD has created a new line of low cost processors called APUs. One in particular is the A8. It is a quad core CPU with 400 APU cores. More info on the processor can be found on AMD's website.

[http://www.amd.com/US/PRODUCTS/DESKTOP/APU/MAINSTREAM/Pages/mainstream.aspx#3](http://www.amd.com/US/PRODUCTS/DESKTOP/APU/MAINSTREAM/Pages/mainstream.aspx#3)

**Question:**
Is it possible to write a driver for Ubuntu to harness the processing power of the AMD A8's 400 APU cores and future core sizes to allow multiple mini-ATX motherboard with an AMD APU processor to operate like a massive render farm with the fewest number of physical nodes?

If yes, are there any developers interested in creating such a driver or application?
