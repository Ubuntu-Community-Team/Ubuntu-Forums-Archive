### Post by kalaharix on 2010-04-09
Hi

I use Gambas a lot as an interface to mySQL and SQLite. I have commercial applications running all day every day and find Gambas very reliable. I too prefer an integrated gui and like the fact that I can build on my VB6 experience (though Gambas and VB6 are NOT the same). Have you looked at the game examples that come with Gambas?

I don't know much about gaming but suggest you also look at Purebasic. It is blindingly fast, compiles to a true executable and interfaces very well with the API. Unlike Gambas it is also cross-platform. There is a one-off (very reasonable) cost which guarantees all future upgrades.
