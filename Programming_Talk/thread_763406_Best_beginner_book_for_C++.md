### Post by Chayak on 2008-04-23
C++ Primer Plus, Fifth Edition
It's also used as a textbook by some colleges so if you ever take classes in programming it'll give you a head start.

C++ is not a language to 'get to the cool stuff quickly' that would probably be something like Python (which a lot of games are written at least partially in now and you seem to irrationally hate despite never really using it.)  

A big question is where do you eventually want to do?  If you want to be a software engineer then your job field will direct the choice of language a bit.  If you're interested in programming for NASA/JPL doing flight software then C and Ada are a must. Game development uses C/C++, Python, and Java in the majority but that's not to say thats all they use. etc...

Researching your final goal in programming may help you decide on the best languages.
