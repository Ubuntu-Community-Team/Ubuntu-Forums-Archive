### Post by Simian Man on 2009-11-11
You don't make it yourself, it's generated by a program called autoconf (which usually goes along with automake).  These tools are a lot harder than they should be and, if you're a beginner, I wouldn't bother trying to learn them.  I personally hate the autotools, and use scons instead.

If you want to learn them anyway, just google for "autotools tutorial".
