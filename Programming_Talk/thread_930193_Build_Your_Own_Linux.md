### Post by JupiterV2 on 2008-09-25
Building [Linux from scratch]("http://www.linuxfromscratch.org/") is a worthwhile task for those of you who are new(er) to Linux and/or C and C++ programming. Why?

1. Understand the directory tree, devices and all the nitty-gritty details under the hood that Ubuntu typically hides away.

2. Understand manual package management and gnu tools like make (automake), configure (autoconf), etc. for those who wish to program in C or C++.

I still like Ubuntu but building a system from scratch is a sobering experience and makes you appreciate the ease of use Ubuntu provides. =)

I highly recommend it.
