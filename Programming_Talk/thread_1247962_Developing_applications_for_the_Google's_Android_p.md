### Post by WitchCraft on 2009-09-16
It's even more simple:
Delete eclipse
install the Android sdk
add the sdk's tools folder to your system path variable
install JetBrains Intellij IDEA, for Linux, Windows or Mac.
Install the IntelliJ Android plugin (don't use the latest version, because it is buggy)
Point it to the android sdk folder

Then create a new project, and in the module dialog, select android.

Then it creates a hello world application, and when you hit run, it starts an emulator,  uploads installs and runs the program.
