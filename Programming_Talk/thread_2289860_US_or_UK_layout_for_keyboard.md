### Post by trent.josephsen on 2015-08-08
Just so you know, you generally can't switch keyboard layouts by plugging in a different keyboard. The keycodes are the same for the keys that are the same, so if you plug a UK keyboard into a computer that uses the US layout, Shift+2 will give you [@], not ["]. If you want to use a different keyboard layout at work, you have to change it in software. This is generally easy.

A friend gave me a Polish keyboard (which has the same number and placement of keys as a UK one), and I used to use it with the US layout. I got used to moving my hand for Enter and stretching a bit farther for Shift, which are really the only pain points as far as key placement is concerned, but I wouldn't buy one myself unless a US version wasn't available. (I also had to get used to the new location of [\], but if I had learned to touch type the other way it wouldn't have mattered.)

If you want to optimize your touch typing experience, consider alternative layouts that reduce typing effort. I use Colemak because I find it more comfortable to type with. I first tried it in 2012, switched back to QWERTY for work, then switched to Colemak again in 2013 and I've been using it ever since. When I first started it was rough going, but now I can switch almost effortlessly between the two (although I have to look at the keys for QWERTY now). US and UK layouts are essentially identical -- there isn't a substantial cost associated with switching between them and so it really is just a matter of preference and what you have on hand.
