### Post by Severun on 2007-12-03
I found one called Paros that seems to work even better than Fiddler, plus it's written in Java so it runs on Ubuntu.  I also got quick response on a support request I made.

It's at [http://www.parosproxy.org]("http://www.parosproxy.org")

I also posted a short review a screenshot and some basic instructions on how to get it working on Ubuntu on [my blog]("http://geektravelling.blogspot.com/2007/12/paros-proxy-web-debuggingtesting-at-its.html")

You've got to use Sun Java as opposed to gcj Java, other than that, it's a slam dunk.

Enjoy..
