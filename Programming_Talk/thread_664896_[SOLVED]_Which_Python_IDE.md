### Post by pmasiar on 2008-01-12
vim has steep learning curve, but it is optimized for hardcore file editing (full time). You can also write vim macros (custom extensions) in Python, a big plus. But it is hard to learn, so vim is still on my to-do list :-)

If all you need is to play with Python a little IDLE is fine.

I personally do simple edits in IDLE, big programs I prefer to edit in SPE because it has very smart code marks and code outline, and all other edits (non-Python, like HTML and KID templates) in SciTE.
