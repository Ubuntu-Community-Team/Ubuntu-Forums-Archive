### Post by dawynn on 2011-09-29
Linux is a fantastic environment for programming in many recent languages.  One can find freeware or open source compilers for a number of different languages, however my search for COBOL resources on Linux has not been fruitful.

That doesn't mean you have to give up hope.  If the professor provided a compiler program, even if its meant for IBM PC platforms, you may still be able to run it.  Dosbox can run just about anything built for a DOS environment, and Wine can often run Windows-based programs.

Find out what environment your professor's program is aimed at, and let us know if you encounter any issues using it in Linux.
