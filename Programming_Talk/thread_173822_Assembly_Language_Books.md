### Post by biocrasher on 2006-05-11
A nice book that I bought,is 
Professional Assembly Language by Richard Blum,wrox press.
Major advantages of this book, is that it uses AT&T style syntax, 
it has a linux specific section about system calls and 
also he uses only GNU tools throughout the book. 
That is, you learn exactly the same tools that you will come to use later.
It shows you also how to integrate assembly programs, 
with your regular C programming.


And don't be scare by the title. He goes through the basics in a very comprehensive manner.
