### Post by ankursethi on 2008-09-26
You can use a content management system like Joomla and customize it to your heart's content. It's easy for non programmers to customize, but since it's written in PHP it should be easier still for you to get under the hood and play with it. The second option would be to learn Rails or Django. Both of them are dead easy to use once you get to know the basic MVC architecture they're built upon. It's time well spent, if you ask me.

For assistance with hosting and maintaining a web server, you might want to check out the Apache mailing lists or IRC and ask for help.
