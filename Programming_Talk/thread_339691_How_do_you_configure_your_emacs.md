### Post by mc^2 on 2007-01-16
Hi, after couple years of using emacs I still find new interesting features. So, I'd like to say a couple of words about the extra packages I use and which I find very handy. 

I usually use emacs to do some c++/java (sometimes also perl and python) programming, so I found ecb a nice little code browser very useful. I also use elscreen to get the firefox-like tabs. A nice feature is also integration with my Subversion repository with psvn (even though it's not as feature-rich like the cvs one).

How about you? Which emacs packages do you find the most useful in your common tasks?
