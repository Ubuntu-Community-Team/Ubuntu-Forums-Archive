### Post by martinfierro on 2006-08-28
Complete and utter Ubuntu/Linux/programming n00b here with a suggestion for a Linux screen saver:

An astonomer at [http://www.fourmilab.ch/skyscrsv/](http://www.fourmilab.ch/skyscrsv/) (John Walker) devised a very cool Windows screen saver that shows the overhead location of the stars and planets (and former planets like poor Pluto) as seen from any specified GPS location on Earth, based on the computer's clock.

The (Windows) Sky Screen Saver is in the public domain. Walker has also made the source code (for Visual C 7.0 / Visual Studio .NET) freely available for download at the above URL.

I can't help but think that this screen saver would be a big hit if ported over to Linux by someone with far better programming skills than mine.
