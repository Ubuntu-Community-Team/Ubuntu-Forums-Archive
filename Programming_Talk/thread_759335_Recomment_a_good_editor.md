### Post by x1a4 on 2008-04-19
Hi,

By far the best editor I've used is what I'm using now: [jEdit]("http://www.jedit.org/").  It's very well thought out and developed, and with tons of plugins (easily installable from within jEdit itself) you can set it up anyway you wish to work any way you wish.  

jEdit has syntax highlighting for more than 130 languages and is even able to correctly highlight different languages in one file (ie: HTML, CSS, JS, PHP)  It can switch bufferes using tabs or buttons (located on top or bottom).  Built-in macro language and extensible plugin architecture.  Code folding, and just about anything you want from an editor and then some. It's highly customizable and configurable.  Check out its long list of features [here]("http://jedit.org/index.php?page=features").  I highly recommend it.
