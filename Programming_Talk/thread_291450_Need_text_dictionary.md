### Post by hod139 on 2006-11-02
If english is ok, then there is this site I found after a quick google search:
[http://msowww.anu.edu.au/~ralph/OPTED/](http://msowww.anu.edu.au/~ralph/OPTED/)

If you can use a binary file, you can always use aspell's dictionary, which is already installed on your system: 
/usr/lib/aspell/en_US-w_accents-only.rws  (I believe, I could be wrong)
The benefit of aspell, is that you can have many more languages available.
