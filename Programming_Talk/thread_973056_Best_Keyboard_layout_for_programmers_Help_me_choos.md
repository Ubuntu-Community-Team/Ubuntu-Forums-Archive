### Post by Cracauer on 2008-11-14
My remapping is basically oriented on a Symbolics Lisp machine.  The important part of a Symbolic Genera setup is the "SELECT" key.  The select key is a prefix that allows you by preffix PREFIX-<someletter> to switch directly to a critical application or sourcefile. The most important stuff *must* be available directly with these strokes.  

Incomplete List: [list] 
[*] SELECT-s - last shell 
[*] SELECT-c - last *.c *.cpp file 
[*] SELECT-h - last *.h file or if none, load the *.h file of the same name as the last C++ file 
[*] SELECT-l - last Lisp file 
[*] SELECT-L - Lisp REPL 
[*] SELECT-4 - last m4 file 
[*] SELECT-i - emacs *info* buffer 
[*] SELECT-m - last manpage 
[*] SELECT-M - last Makefile, GNUmakefile or if none, load ./(GNUm|M)akefile 
[*] SELECT-return - accept regular expression of buffer name 
[*] SELECT-SELECT - flip back and forth between two buffers 
[/list]  

That are the important ones.  

The SELECT key is F12, but I remapped F12 to the right ALT key. So one flip of the thum like you hit the spacevar, but a little right and one char gives you all this stuff. Of course it doesn't beat the original Symbolics location which is a row of killer keys left of the main block (why the %^&*@ did the PC designers leave the space unused?).

[img]http://www-2.cs.cmu.edu/~tss/sc.jpg[/img]
[img]http://farm4.static.flickr.com/3182/2602577869_a2c57399b3.jpg?v=0[/img]

The power of this setup is incredible. If you haven't seen it you can't imagine how fast I flip back and forth between different source files, a commandline window, a manpage and whatnot. And putting in a new shortcut is two lines of code and Control-L for load config file.

In other remapping, I of course have control above shift, whatever key is left of space must be ALT/META, the right control key is emulating a left mouseclick, that dummy windoze key next to it a right mousekey (control-cursorkeys move the mouse pointer) and the whole num block is remapped to control my mp3 player and give special chars such as Umlaute. On Thinkpads and some other small keyboards I exchange ESC and ~.
