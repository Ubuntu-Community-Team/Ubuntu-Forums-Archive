### Post by hessiess on 2010-02-09
Don't create websites using photoshops/gimps image slicer and HTML generator. They both generate table based layouts which are a very bad way to do things. Instead learn to create layouts using CSS, then incorporate image slices where appropriate. With CSS you can normally get rid of most background images.
