### Post by kostkon on 2009-11-08
> **mvalviar said:**
> Where can I find/invoke these?
Using Synaptic or in a terminal ;)
```
sudo apt-get install python-gtk2-doc python-gtk2-doc
```
Also, to easily have access to these docs, I would recommend you to install a documentation browser, i.e. *Devhelp*. Thus, the above command should be :P
```
sudo apt-get install python-gtk2-doc python-gtk2-doc devhelp
```
