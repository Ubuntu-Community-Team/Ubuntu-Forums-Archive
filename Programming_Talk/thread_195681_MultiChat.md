Hello,
MultiChat is an IRC client with the following features:
- Multiplataform (wxMSW/wxGTK)
- System tray integration
- Minimalistic login screen (IM style)
- Graphical smileys
- Private Messages Photo (for networks supporting it)

Ubuntu install package:
[ftp://ftp.sunsite.dk/projects/ptlink/multichat/multichat_0.8-1_i386.deb](ftp://ftp.sunsite.dk/projects/ptlink/multichat/multichat_0.8-1_i386.deb)

Source code:
[ftp://ftp.sunsite.dk/projects/ptlink/multichat/multichat-0.8.tar.gz](ftp://ftp.sunsite.dk/projects/ptlink/multichat/multichat-0.8.tar.gz)
If you intend to install the application from the source you will need to download and install wxGTK2.6.3 (the wxGTK version that comes with dapper is old and buggy).


Please note that the client does not support network selection, network developers are expected to customize and repackage the installer to use for their networks.

This is my first wxWidgets app and I have not programmed C++ for a long time so any feedback is welcome.
