### Post by spjackson on 2017-02-20
Welcome to the forums.

In order for us to help you, we need to know precisely what you are doing. Some of the causes that lead to this error are, in no particular order.
[LIST]
[*]Trying to run the source code (e.g. ./myprog.cpp).
[*]Trying to run what has been compiled but not linked (e.g. ./myprog.o).
[*]Trying to use a folder on a non-Linux filesystem, e.g. USB stick or Windows partition.
[/LIST]

There are many threads on similar problems already in these forums, but it isn't really practical to steer you in the right direction without some more information.
