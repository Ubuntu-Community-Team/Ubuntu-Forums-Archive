### Post by ankursethi on 2007-10-28
You could try making -
1. A text editor.
2. A note taking app (like Tomboy).
3. A web indexer (I'm making one in Python, so the idea just popped up :))
4. A media player (you'll need to read some more for doing this).
5. A SHOUTCast server.
6. A direct-to-IP chat application.

The above projects are smallish, yet you will need to use all available language features to get them working. They're pretty good for practice.

By the way, it's unlikely you will find anybody in this community who will help you with C#. Many Open Source guys seem to dislike the thing, and those people are in majority here. Most of them simply don't see anything special about it.

Anyway, best of luck with your project :) Mono is not evil. Learning something will only help you in the future.
