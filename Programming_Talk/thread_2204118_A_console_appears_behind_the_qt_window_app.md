### Post by spjackson on 2014-02-07
It's nothing to do with the source code you have posted. On Windows, whether you get a console window or not is determined by the link command. How are you building your project? If you use qmake + make, then post your .pro. If you are using QtCreator, there might be an option to click - I don't recall.
