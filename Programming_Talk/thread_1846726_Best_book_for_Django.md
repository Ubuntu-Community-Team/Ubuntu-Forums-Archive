### Post by littlemog on 2011-09-19
No better place to start then [The Django Book]("http://www.djangobook.com/").

After that, head towards the [Django documentation]("https://docs.djangoproject.com/en/1.3/") and read as much as you can. :)

Once you've done that, try out a simple project or two, and then head to #django on freenode if you still encounter any problems (please do read the docs first)
