### Post by Corvillus on 2006-05-29
Ruby on Rails itself isn't an AJAX framework. However, Rails provides a nice Ruby based API for the script.acluo.us ( [http://script.aculo.us/](http://script.aculo.us/) ) and Prototype ( [http://prototype.conio.net/](http://prototype.conio.net/) ) frameworks. However, both of them can be used seperately from Rails. Also, the Dojo Toolkit is another popular framework ( [http://thedojo.net/](http://thedojo.net/) ).
