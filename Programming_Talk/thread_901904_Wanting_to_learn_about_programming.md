depends on where you wanna go. if you want a job in a corperate IT shop, learn java and .net. if you want to get into the technology itself, start with C. if you wanna be a linux app guru, go with the P langagues (perl/python/php).

if you wanna play, I recomend java + netbeans or eclipse. that way you can do guis easily and see what your writing. 

in terms of how to start, it depends on where you are. if you've taken 2 classes in any langague, then just download the dev tools and pick somthing you wanna do. then jsut google until it works. rinse and repeat. 

if you've picked a langague that is horribly differant from what you know, buy a beginners book and read it cover to cover. google some more. gain experience. solicit peer review.

I can't understate the need for school if you plan to do this professionally. first it gives you a good basis on how programming really is, and to be honest, theres a whole lot of non-IT stuff to working in IT.
