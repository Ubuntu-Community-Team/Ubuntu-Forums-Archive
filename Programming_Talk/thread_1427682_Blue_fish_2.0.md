### Post by soltanis on 2010-03-12
I don't use bluefish but in most editors, to indent/unindent a line you would use a key combo. For example, in Geany it's ctrl+i for indent and ctrl+u for unindent, and in GEdit it's ctrl-t and ctrl-shift-t (I like Geany better). Using the tab character would in almost every editor I have ever used overwrite the highlighted text with a single tabulator (or 4 spaces).

Probably not in Emacs but that's not a real text editor.
