### Post by Paul41 on 2006-11-17
I use Eclipse with the phpEclipse plugin (I also use the jsEclipse and css plugins). Eclipse is available in the repository so can just search for it in synaptic and install it there. You can get the phpEclipse plugin here [http://sourceforge.net/project/showfiles.php?group_id=57621&package_id=70009](http://sourceforge.net/project/showfiles.php?group_id=57621&package_id=70009). There is also an address you can put into Eclipse and it will get it for you (this way is easier). I am at my other computer right now so I don't have the address, but I can give it to you later if you want it. If you are going to use Eclipse I would also use the Sun JVM because it runs much faster that way. Here are some instructions for doing that. [https://help.ubuntu.com/community/EclipseIDE](https://help.ubuntu.com/community/EclipseIDE).

Some other options are Bluefish or Scream. These are both available in the repositories also.
