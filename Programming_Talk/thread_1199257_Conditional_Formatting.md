### Post by fr4nko on 2009-06-29
> **Mbengi Bongi said:**
> Is conditional formatting possible in HTML?

I have a web page which features a large table (4 columns, 10,000 rows), what I'm wondering if there's a way if I can make the text in Column B appear red if the value is greater than 16 characters (including spaces), otherwise default to black if 16 characters or less.
HTML does not support conditional formatting because HTML never cares about the *meaning* of the text. The thing that you want can be obtained if the code is generated by PHP. You can instruct the PHP code to select a red or green backgroud depending on the values of the cell. More generally you can do it if you generate the HTML code by using any scripting language like PHP, Perl, Python or any other scripting language.
