So, I am not sure that you will have a business problem, but I am sure that there is something that you do regularly that follows the sames steps over and over again.

A book XML parser:
```

from xml.dom import minidom
doc = minidom.parse("books.xml")
for node in doc.getElementsByTagName("book"):
	booktitle = node.getAttribute("title")
	bookauthor = node.getAttribute("author")
	print booktitle + " is written by " + bookauthor + "."
for node in doc.getElementsByTagName("series"):
	sertitle = node.getAttribute("title")
	serauth = node.getAttribute("author")
	serbooks = node.getAttribute("books")
	print sertitle + ", a series with " + str(serbooks) + " books, was written by " + serauth + "."

```
