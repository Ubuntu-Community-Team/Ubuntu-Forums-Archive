### Post by OutOfReach on 2008-08-29
This should be moved into Programming Talk, but any way a module is just any other .py file, so if you make a .py file with uhh let's say something you need for your main file, you can just save it in the same directory as the main file and use
```
import MYMODULE
```
to import it, notice you don't use the .py extension when importing modules.
