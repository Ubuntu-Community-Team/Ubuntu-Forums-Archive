### Post by t3tech on 2009-04-20
> **smitty92 said:**
> Ok, so i need to learn a language, im guessing to start with python? and so i also need a coding program? if theres so many languages what makes one compatible with windows and not linux, im so confused

As sinkingships7 stated, most languages could be used on any platform there's just some caveats involved depending on which combination. 

As for a 'coding program', that could be anything from a basic text editor (pico, kedit, gedit) to an advanced text editor (vi/vim, emacs) to a full IDE (KDevelop, Eclipse, etc.). I think I've used all of those examples, and then some, at one point or another for coding. Lately I've grown accustomed to using Eclipse for the most part, though I'll still use whatever text editor is handy on occasion for certain things like quick changes to a particular source file.
