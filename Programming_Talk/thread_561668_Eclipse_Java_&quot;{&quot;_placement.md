Is there a way to make eclipse put the opening curly brace on the line underneath the method or class name instead of directly after on the same line.

---

### Post by BrumleyGap on 2007-09-27
Go to Window > Preferences > Java > Code Style > Formatter

In "Select a Profile" choose Java Conventions and press Show

Choose the "braces" tab and save the edited profile under a new name

Have at it!
