### Post by jblebrun on 2007-01-28
I like deskbar applet! I wrote a simple handler that lets you set a basic countdown timer, and then pops up an alert when the timer is up. So, you can enter something like:

45m Check Laundry 

In the deskbar applet search box, and it gives you an option to set a 45 minute timer. When the timer is up, a dialog box saying "Check Laundry" will pop up.

When timers have been started, you can check how much time is remaining, or cancel the timer, by just typing anything into the deskbar applet search box. Any running timers will show up, and their remaining time will be shown. If you click on them, they'll be cancelled.
