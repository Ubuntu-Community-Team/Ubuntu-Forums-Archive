### Post by kostkon on 2007-08-22
In order to get the latest version of *Anjuta* you have to add its repository to your sources list.

Follow the [instructions here in the Download section of the *Anjuta* site]("http://anjuta.sourceforge.net/downloads") and then you will be able to install the 2.20 version from *Add/Remove* programs or *Synaptic*.
