### Post by Lux Perpetua on 2008-04-23
For what it's worth, I got started with Cairo through cairomm and gtkmm. After reading this: [http://www.gtkmm.org/docs/gtkmm-2.4/docs/tutorial/html/chapter-drawingarea.html](http://www.gtkmm.org/docs/gtkmm-2.4/docs/tutorial/html/chapter-drawingarea.html), I got a feel for how the system worked, and that along with the documentation was enough to get me started. I definitely prefer C++ to C, and that affected my route; if you prefer C, then I don't think there's any particular reason to go that way.
