### Post by Zugzwang on 2010-12-14
On their web page, they have "application notes", which describe the functions you need to use to implement your application.

In order to program something, you need to download a copy of their API (an archive which contains the header files and so on). This archive also has examples. Try to compile & run them and then using the API documentation to get write your application. Good luck!
