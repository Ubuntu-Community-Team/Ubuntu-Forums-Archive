### Post by r-senior on 2012-03-22
In my opinion, the most complete IDE is the Eclipse platform. Well-known as a Java IDE, its pluggable architecture also supports C, C++ and Python. It runs on Windows and Mac.

Having said that, I like the simplicity of Geany for languages other than Java. If you want something even simpler, the standard Gnome editor GEdit deals with syntax highlighting for lots of languages and is a decent programming editor when combined with a terminal window.

Emacs is like Marmite. ;)

### Post by r-senior on 2012-04-03
> **llanitedave said:**
> Jut curious:  strictly as a Python/Javascript/HTML coder, what does NetBeans or Eclipse give me that Geany doesn't?
For Python I think it's refactoring mainly. Geany can do search and replace but not intelligent refactoring like the full-blown IDEs. So renaming functions, extracting statements into functions, pushing up class members into a parent class, etc, etc.

Depending on language, there are all sorts of plugins in Eclipse and Netbeans for template code, projects, infrastructure generation, testing, build tools. I've not used them for Python so I don't know how much of that there is but there are stacks of them for Java.

### Post by na5h on 2012-04-03
Eclipse and NetBeans are both great IDEs. If you're into developing Android Apps, the Android SDK is really easy to set up with Eclipse.

According to "Linux User & Developer" magazine, Komodo Edit is also a very good choice. They also mention that NetBeans might be a "potentially better choice if you're concentrating on Java application development". Eclipse is also mentioned as a powerful, yet very resource hungry IDE. Both NetBeans and Komodo Edit need to be installed manually, but it's no big task.

If you want something more advanced than a simple text editor, yet lightweight and fast, then you may also want to consider Geany. It can be found in the Software Center.
