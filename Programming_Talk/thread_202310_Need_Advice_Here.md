### Post by Daverz on 2006-06-24
Python and Ruby both have vibrant user communities, but I would only worry about whether a language can do the task at hand and whether it fits your brain.

For Python I would first install ipython, which is a souped up interactive interface.  For an editor, you might try idle, but there are several Python IDEs  in the repositories like Eric, SPE, and DrPython, and lots of editors that have language support to varying degrees like jed, gvim, gedit, scite, etc.  I use xemacs + python-mode.
