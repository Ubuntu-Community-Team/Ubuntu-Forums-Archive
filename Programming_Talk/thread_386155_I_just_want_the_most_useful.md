Start with Python - especially if you are self-learner. In Java you will have to start with OOP (Object-oriented programming) from very beginning, and C++ is even more picky than Java. ### Post by mikeoh on 2007-03-18
I'm going to say something different then everyone else, learn C!  C is quite a small language, its low level so you learn about pointers and addressing memory and its only procedural so you don't need to worry about Object Orientation.  Then when you feel comfortable you can learn C++ or Java which have similar syntax but with object orientation.   

Its not as simple as Python but it will put you on a solid foundation to learn C++, Java, C# and plenty of other languages that are based on C syntax and learn alot about how a computer works.

### Post by pmasiar on 2007-03-18
Python does not force you to use OO either - you can program all your life using procedural approach only (and using objects from libraries). C development cycle is longer (you need to compile), syntax is pickier,  data structures are less flexible, and Python has command-line shell to try syntax one line at a time like BASIC - the most popular language of old-times, which allowed beginners to tinker with code.

But if you insist on C, Bruce Eckel (one of the best experts on teaching C++/Java - "Thinking in ... series) has free online course on C. BTW Bruce prefers Python :-)
