### Post by jespdj on 2008-10-22
It depends on the plugins that you have installed in Eclipse. For those formats, you probably need the WTP (Web Tools Project) plugins. If you downloaded and installed "Eclipse IDE for Java EE" from the Eclipse website, you already have this. Otherwise, you can install it via Eclipse's Update Manager (Help / Software Updates).

Note, I noticed that the original Eclipse 3.4 has some nasty bugs, which make editing JavaScript files very annoying: Eclipse frequently stops responding for a few seconds. The latest update, Eclipse 3.4.1 (which is what's currently on the Eclipse website) fixes these problems.
