### Post by epicoder on 2012-07-14
Both Geany and gedit are viable options for what you are doing. Both have plenty of features and customization options, and they should both be able to meet your requirements. Both have sidebar extensions you can use for ftp integration, and snippet toolbars that you can set up for HTML, PHP, etc. They both have an extensive library of plugins that can add just about any feature you can think of. Just try them both out and see which works better for you.

I personally like Geany better as it is similar to Notepad++, a program I use quite a bit. However, I have also used gedit and had no trouble with it. In the end, it just comes down to personal opinion.
