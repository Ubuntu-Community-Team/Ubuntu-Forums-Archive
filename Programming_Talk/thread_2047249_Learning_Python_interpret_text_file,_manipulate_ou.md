### Post by Vaphell on 2012-08-27
```
print( ..., file="testout.txt" )
```
what you say here: file is a string "testout.txt". You need to create a file object (just like with input file but in write mode) and use it there. Looking at your code that would be fout
```
print( ..., file=fout )
```

you dont need to reinvent the wheel with alignment

[http://docs.python.org/py3k/library/string.html](http://docs.python.org/py3k/library/string.html)
you need to use format() function
to right-align in fixed width:
```
"{0:>10}".format(var)
```
> = right, 10 = width
