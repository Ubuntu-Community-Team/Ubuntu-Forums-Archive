### Post by Can+~ on 2009-10-10
To get accustomed to the syntax, read the [lexical analysis]("http://docs.python.org/reference/lexical_analysis.html") (and [expression syntax]("http://docs.python.org/reference/expressions.html")) and then proceed to the language reference or tutorial (depending on your level of expertise).

[http://docs.python.org/index.html](http://docs.python.org/index.html)

The default doc is pretty good, if you want a more step by step, trying looking for python tutorials links on this forum, or books.

Notice that most of the things on the documentation are expressed on the Extended Backus-Naur Form.
