### Post by daou on 2006-09-05
Hi everyone,

I just finished programming the first release of my new app Color-Chain. If you are interested, you can get the .deb or source from here:
[http://colorchain.wiki.com/]("http://colorchain.wiki.com/")
There is a video of the program in action at the site.

Basically what it does is it changes the GNOME desktop background gradient colors gradually according to the time of day. It includes a configuration tool to change the colors and times when the colors appear.

I hope you like it. It's a first release with plenty of bugs and it's my first attempt at Linux programming so please be forgiving for its shortcomings ;) .
