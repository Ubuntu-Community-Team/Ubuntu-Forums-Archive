### Post by Gannin on 2009-07-15
A couple years ago I created a program to help generate the XML needed for a podcast's RSS feed.  You just fill in the fields, click the generate button, and there you go.  Some knowledge of working with such things is still needed as the user has to manually copy and paste the XML into their feed.  Having the program do it automatically for you is planned for the next major version.

The old version has been downloaded quite a few times and was even included on the CD with a computer magazine in the UK.  I decided recently to go back and work on the program some more, but I found working with SourceForge to be a pain in the butt, so I moved the project over to Google Code and released a new version.

Now, before I release version 1.0, the program could really use some stress testing.  It's over at:

[http://newpodfeed.googlecode.com](http://newpodfeed.googlecode.com)

The program is just a Java .jar file distributed in a zip along with a copy of the GPLv3 license.

If anyone finds any bugs it would be greatly appreciated if they could report them either in the Issues section of the project webpage or on this thread.  Thank you :).
