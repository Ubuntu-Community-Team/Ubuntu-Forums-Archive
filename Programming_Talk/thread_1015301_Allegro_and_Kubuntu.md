### Post by thetofucube on 2008-12-18
Open your Konsole or xterm. 

From the command line type the following:

sudo apt-get install build-essential

and next:

sudo apt-get install liballegro4.2-dev

see this link for additional information on compiling and running allegro programs:

[http://blog.kunugiken.com/?p=66](http://blog.kunugiken.com/?p=66)
