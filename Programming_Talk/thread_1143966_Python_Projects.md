### Post by Kilon on 2009-05-01
> **matmatmat said:**
> How do you know what language their in? On launchpad most don't mention what language their in & if you search for python it just comes up with the programming language.

Actually in case of source is very easy. 

These are the steps

1 ) go to sourceforge

2 ) Click "Find Software"

3 ) Click "Display all"

4 ) In the "Display : " click "Filters"

5 )In "Add filter to" leave "Require" as it is

6 ) Choose instead of "Files" , "programming language"

7 ) Choose "Python" 

8 ) hit apply and that is it all projects who use python even a little will display. 

9 ) You can click in the "Display" , "Details" to see what other languages are used and all the detailed info for each project.

10 ) If you are interested in a specific kind of app then in the left there categories you can choose , the filter will stay until you remove it.

11 ) if you want a python only project then go to the same process but this time choose "exclude" instead of "require" and exclude all the popular languages . That will show you project that are using only python. 

Try it and informs us what you found.
