### Post by iball on 2006-04-25
You could try the GNU Pascal compiler, gpc.  This is obviously the sister to the GNU C compiler, gcc.  You can install it using ```
sudo apt-get install gpc
```

I have never done any pascal programming, so I don't know hoe good gpc is.  If you install gvim (sudo apt-get install gvim), you can get Vi with a graphical interface.  Gvim will do syntax highlighting, which I always find handy.

I hope this helps
