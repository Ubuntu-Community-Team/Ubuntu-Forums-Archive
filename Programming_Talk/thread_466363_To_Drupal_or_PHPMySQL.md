### Post by Frosty Cold Drink on 2007-06-06
Drupal rocks, if you never want to get away from MySQL. (They claim PostgreSQL compatability, but most add-ons won't work, and even some of the core features break easily.)

Just go with Drupal and implement custom modules if you need additional features. Doing it yourself is great, and I wont knock it, but sooner or later you'll have special needs come up, and its better to focus just on those instead of the basic groundwork someone else could have done for you.

### Post by Mirrorball on 2007-06-06
Drupal is itself a PHP/MySQL application. You will need to know some PHP/MySQL to understand Drupal. If your goal is to learn, keep on learning PHP and MySQL, which is the basis, and have a look at Drupal's source code to see how it works. If you just want to build a community, install Drupal and learn to use it.
