### Post by victor.zamanian on 2010-03-13
If you run the program `alacarte', or alternatively right-click the menus and select "Edit Menus", you will be met with a program that can edit your menus. There is a category to the left with the name Programming, and in there is an entry called "Python" which might be unchecked. Check it, and it will appear in your "Applications->Programming" menu. If the "Programming" entry wasn't in your "Applications" menu before, it will appear when you select it.

All the best!

PS: You can also create your own desktop launchers, which you can find info about everywhere (Google, here probably). And you can also just run `python' from the terminal and that will give you the same effect as launching it from the Applications menu.

If you press ALT+F2 and enter `idle-python[version_number]` in the Text field then you can use the Tkinter based Python interpreter.
