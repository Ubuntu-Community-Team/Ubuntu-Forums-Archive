### Post by mehaga on 2012-05-11
It will be hard to find something better than Netbeans and Eclipse. [IntelliJ Idea]("http://www.jetbrains.com/idea/") (Community Edition) is another popular choice, but you'll probably have to register. If you want something really simple, you might like [BlueJ]("http://www.bluej.org/"). If you really want just an editor, you can try [jEdit]("http://www.jedit.org/").
