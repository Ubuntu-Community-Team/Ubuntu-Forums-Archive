### Post by oldfred on 2017-04-01
I add Geany as my editor, many others have posted in various threads alternatives they prefer. 
And in Geany set it up for python by changing to 4 spaces instead of tabs in preferences for editing  & saving. Having tabs creates all sorts of issues.

Then I can quickly test/run my python code using run (gear) icon. With python the compile button just saves file and checks for some errors.

Not sure if my version of Geany is python3, you may have to change settings to have it default to python3.
