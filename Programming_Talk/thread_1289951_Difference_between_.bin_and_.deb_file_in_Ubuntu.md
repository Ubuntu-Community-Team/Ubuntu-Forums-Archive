### Post by nvteighen on 2009-10-13
A .bin is usually a binary... though very few applications use this; the only one I can think of right now is OpenOffice.org.

A .deb is a Debian installer package... Like a .tar.gz, but which manages its own decompression into system folders, allows to perform dependency checks and also manages its own removal.
