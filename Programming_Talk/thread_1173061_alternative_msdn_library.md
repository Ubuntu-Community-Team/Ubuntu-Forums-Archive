### Post by tyfius on 2009-05-31
MSDN is just a documentation site. (You can integrate it with Visual Studio but it's still an online website.) You can access it at [http://msdn.microsoft.com/en-us/default.aspx](http://msdn.microsoft.com/en-us/default.aspx).

There are similar sites out there depending on the language you are currently working with. For Java you can use [http://java.sun.com/reference/docs/](http://java.sun.com/reference/docs/), for PHP you can use [http://www.php.net/manual/en/](http://www.php.net/manual/en/).

Some of these can (or are by default) integrated with the IDE you are using. Check the documentation to see how you can access it. Mostly using CTRL+Click will get you a long way.
