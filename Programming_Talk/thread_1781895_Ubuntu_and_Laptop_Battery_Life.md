### Post by IC2D on 2011-06-14
Ubuntu needs an edge over windows to make people try it out.  Yes, we already have Wubi, and benefits of stability and security with the Ubuntu OS. But what about Battery life? Many laptop users, especially university students (like myself) would love to have all day battery life, or as much battery life as possible. Although there is the option to buy a bigger battery (which costs money), the OS should make the most out the battery it is supplied with (for me, it's a 3 cell). I was gonna discuss this idea on here before I publish it on Ubuntu Brainstorm, so I would like as much answers as possible. Thanks :)

### Post by gyussz on 2011-06-14
I have positive experience about this. My ASUS K50AB (4 cells, 4400mAh),  could only run Windows for like an hour or less without charging (on power saver mode, aero/effects disabled). Ubuntu can run for approx 90 minutes. Not Much, but good enough for me. (On battery I only run a web browser, a text editor, and a terminal, so nothing really consuming.)
