### Post by Canis familiaris on 2007-04-21
Use the command:
```

user@user-pc:~/dev$  g++ filename.cpp -o program.bin
```
This compiles the file. Now:
```

user@user-pc:~/dev$  ./program.bin
```
This executes the program
