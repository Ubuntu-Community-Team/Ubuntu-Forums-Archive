Most open source projects have a formal bug reporting and tracking system, usual via third-party providers like SourceForge.

If you want to help fix bugs for Ubuntu specifically, you will need to learn C.  C++ and Python can help, but Linux is developed using C, as is Gnome, so learning C is essential.

You can work on non-Ubuntu specific projects, which include many projects using Java.  As mentioned before, Eclipse, Azureus, etc., are all prominent Java projects.  Java is one of the most popular programming languages for open source projects (it was second behind C last time I checked Sourceforge), so you should be able to find plenty of Java projects to choose from.
