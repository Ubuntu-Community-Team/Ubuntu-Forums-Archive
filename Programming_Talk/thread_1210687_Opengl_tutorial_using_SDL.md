I am not sure what the problem is, but looks like you have all the pieces. Please note that most (if not all) OpenGL tutorials will most likely use GLUT, because GLUT is perfect for OpenGL tutorials/demos.

I doubt you will find OpenGL tutorials that use SDL for the simple reason that GLUT is simpler and SDL does not win anything vs. GLUT (in fact, I would say that SDL loses) when used for OpenGL tutorials.
