### Post by r-senior on 2012-05-06
OpenJDK 7 is the reference implementation of the Java standard. In other words, if it works on OpenJDK 7 it is supposed to work on all conforming implementations.

You'd be better targetting Java 7, because Sun Java 6 becomes end of life in November this year.
