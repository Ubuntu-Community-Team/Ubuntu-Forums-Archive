### Post by Tomosaur on 2008-07-14
> **lucky.developer said:**
> hey.. ok i agree with you.. i have now reduced my font sizes. i want some resources from where i can learn with the project development process.

Generally speaking - people go to school to learn all about this kind of thing. Project management covers so many areas, it's difficult for one person to manage a large project. If your project is small, then I wouldn't really worry too much about anything but actually programming the software. Documentation can come later, and most people are happy with just a user guide.

If it's homework or something, I would imagine you'd need at least a design, documentation of your code (Which may amount to simply commenting your code as you write it. Java has Javadoc, which can produce a website based around your code comments for easy navigation), and a user guide.

You are wrong :-) It is "ra" by default, but I have the maatraa of a vowel there. 

Devanagari is a Syllabary, each letter makes one sound (an entire syllable). In Devanagari &#2352; and all consonants have an implied schwa (there is no maatraa for this sound, although there is a vowel, for when it is the first sound). Letters have no name, and are named after their sound (usually). It is hard to type this at the moment, so I can't give good examples. To change the default sound, you just add a vowel, in this case "oo" (long o). Adding maatraas like this makes it look like one character, but it isn't. (The very last "character" on my title is two of them, but the positioning makes it look like one) The maatraa's are "small", so they are easy to miss (if you are just looking at a list of letters)
