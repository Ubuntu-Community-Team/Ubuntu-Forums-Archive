### Post by 11jmb on 2011-10-31
Check out Wing. It is an ide for Python written in Python. The only time I might recommend PyDev is for people who already know their way around Eclipse. Wing does a much better job than PyDev when it comes to text editing features that you expect from a good ide. I will point out, though, that Wing is not free and open source if that is important to you. The free version works very well, and you can access the source with a professional license.

I think that the free version of wing is the best free Python ide that I have used (although I generally work on smaller projects where emacs/command line is better)
