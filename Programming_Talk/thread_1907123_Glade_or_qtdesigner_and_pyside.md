### Post by KdotJ on 2012-01-11
Are you asking which GUI framework toolkit you should use? Or which IDE you should use? Either way, my suggestion would be that if you are just learning how to programming a GUI, you should do so by hand in an editor and not use a "drag and drop" GUI creator, you will learn a great deal more.

Back to the frameworks, you should have a think about if you want to aim your apps at KDE or Gnome (either will work on Windows anyway)... and also which framework you think feels comfortable to code. For me personally, I prefer Qt... but others will prefer GTK.

However, aside from them two.. for Python I actually like wxWidgets the most.
