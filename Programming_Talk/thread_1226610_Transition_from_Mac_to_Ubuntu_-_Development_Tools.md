### Post by Cyanidepoison on 2009-07-30
As a current Mac and Linux user, I cannot stress highly enough that you should try using a text editor like Vim as your IDE.  Unless you're working on truly massive projects where an IDE would help with organization and compartmentalization, vim and the command line in general is much more efficient in my opinion.

If you can't make that switch, look into the QT suite of developer tools.  They have their own build script format, a GUI editor equivalent to Interface Builder, code completion, and the rest of the fancy stuff.
