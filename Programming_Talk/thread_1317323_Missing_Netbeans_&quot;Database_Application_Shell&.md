### Post by beaner11 on 2011-10-28
As of Oct-2011 in Netbeans 6.9/Mint11 and in  Netbeans 6.9/Ubuntu 10.10, the "Database Application" was still missing.  After I installed the following plugins it became available: 
Database 
Sakila Sample Database 
Netbeans Visual SQL Create 
Java Web Applications 
Java SQL Generator 
 
Not all the above may be needed, but that's what I did. 
 
Hopefully it will help someone, since there was no answer to this issue.
