### Post by azurepancake on 2008-09-14
This wiki is a wonderful resource for budding game programmers, [http://wiki.gamedev.net/index.php/Main_Page](http://wiki.gamedev.net/index.php/Main_Page)

I'm new to programming all together and my main goal is to create some fun and innovative 2-D games, such as RPGs and adventure titles. 

Through what I've researched, using Python with PyGame is probably my best bet. Python is a clean and interactive language, while PyGame is a powerful API / Library / SDK that assists with building some pretty professional looking games, [http://www.pygame.org/](http://www.pygame.org/)

If what I've learned is correct, the main disadvantage of Python is that it is just not fast enough with complex computations. But, when combined with C code and it's libraries, you have a very powerful team!
