..and well .. lol .. i guess it has something to do with timezones maybe, but generally we are just way too few regulars there

..and what is up with not hanging on freenode in the first place? -- i mean; if you are or want to be a programmer and/or have an interest in open source and linux it is *The Place* to be anyways! .. so get a clue and join #ubuntu-programming while at the same time you join channels such as:

#c++

#java
#python
#kernel (about the linux kernel -- kernel/driver hacking  .. etc.)
#lisp
#haskell
#ocaml

#mysql

#perl
#linux (linux in context as an OS)
#defocus (a "social channel")
#ubuntu (you know this name already probably .. veryvery noisy .. i do not hang there but many do)
#linux.xx (for instance #linux.no is for norwegian linux and open source users/hackers .. you'll probably find a channel where people from your country "hang out" also)
etc.etc.etc. .. there are so many!

in general you will find a channel about most topics related to open source and/or programming .. for instance, if you think Pidgin is a cool project and you have some questions about how to hack on it you can join #pidgin ..etc.

the server is chat.freenode.net .. it's up to us/you, but as an individual with an interest in open source / linux / ubuntu you should already be on this server
