### Post by ndefontenay on 2010-11-08
Hi.

The C/C++ compiler is cpp.

To install it, open the command line and type:

sudo apt-get install cpp

If you want a nice developing environment look for eclipse in your add/remove program application.

You will also need to download a c and/or C++ plugin for eclipse but that shouldn't be too hard to find with a google search.

That should make the ball rolling :)

The system scripting of choice seems to be bash (or sh for short).

You should try a bit of scripting too. It's really fun as well.
