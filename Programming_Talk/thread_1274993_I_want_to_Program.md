### Post by nvteighen on 2009-09-25
Read the Stickies and search about this topic in the forums... This has been discussed almost daily here... :p

I recommend you Python as it is a high abstraction level (aka high-level) language with tons of resources. High-level means that the structures and mechanisms the language offers to you are nearer to mental concepts (e.g. lists) than low-level languages, which offer you stuff nearer to the computer's internals (e.g. memory addresses).

For learning programming, IMO, you need to learn how to map your ideas into code... High-level languages are best for that; of course, there are drawbacks like being (a bit) slower and hiding away control of the computer's processes... but a beginner shouldn't need that control... Heck, unless you need it, you usually should try to avoid that control as it hands you a lot of more responsability, along with what you already have to do...
