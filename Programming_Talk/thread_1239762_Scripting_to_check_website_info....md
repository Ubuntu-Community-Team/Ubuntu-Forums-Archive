### Post by aesis05401 on 2009-08-13
You should be told that this is not such a popular topic.  Web scraping is against many site's TOS.  Generally, you can be banned from a site for harvesting information in an automated manner without a prior agreement. 

The program you are trying to write is known as a bot, and the additional traffic it generates while staying on top of changes to the pages you monitor is not typically appreciated by web admins.

Here is a wikipedia overview of the topic:[http://en.wikipedia.org/wiki/Web_scraping]("http://en.wikipedia.org/wiki/Web_scraping")
