### Post by alzaf on 2010-05-20
Hi

Just posting a wee tutorial that I have created which uses Gnome Seed which is in the Ubuntu repositories (although the newest version needs to checked out from Git to allow it to work) and SeedKit which unfortunately isn't in synaptic but needs to be downloaded. 

The links to get the source for both projects are as follows:

[http://live.gnome.org/Seed/Release](http://live.gnome.org/Seed/Release)
[http://gitorious.org/seedkit](http://gitorious.org/seedkit)

The proof of concept is a JavaScript/HTML based frontend of Gnome's Log Viewer. It doesn't replicate 100% of Log Viewers functionality but it produces the basics of it which is to display the contents of Linux log files. 

The tutorial and link to download the source files are as follows:

[http://alsaf1.wordpress.com/2010/05/13/first-seedkit-program-sort-of/](http://alsaf1.wordpress.com/2010/05/13/first-seedkit-program-sort-of/)

[http://alsaf1.wordpress.com/2010/05/20/seedkit-tutorial-sort-of/](http://alsaf1.wordpress.com/2010/05/20/seedkit-tutorial-sort-of/)

For more information about SeedKit and HTML based User Interfaces are in my blog, link for that post is as follows:
[http://alsaf1.wordpress.com/2010/05/07/275/](http://alsaf1.wordpress.com/2010/05/07/275/)

I would be grateful for any feedback.
