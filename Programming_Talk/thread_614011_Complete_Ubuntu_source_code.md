### Post by ssam on 2007-11-15
iso files are disk images (name comes from the cd format ISO 9660).

you can burn the iso to a cd, and look at the files on the cd.

a search for "linux mount iso" should show you how to mount the image.

or file-roller should be able to show the contents, and extract it to a folder.
