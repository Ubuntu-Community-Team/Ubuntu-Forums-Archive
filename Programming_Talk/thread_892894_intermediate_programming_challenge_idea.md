### Post by loganwm on 2008-08-17
Try a palindrome check program.

A palindrome is a string that reads the same forwards and backwards.

take a string: "racecar" for example and check to see if it is a palindrome by comparing the forwards to the backwards, the algorithm is quite simple but it's a nice change.

You can expand it by making it case insensitive, user input, etc.

It's a program my programming teacher let us do for a class.

Examples:
"Sore was I ere I saw Eros"
"Noel sees Leon"
"Dennis sinned"
