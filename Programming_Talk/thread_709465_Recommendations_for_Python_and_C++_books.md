### Post by themusicwave on 2008-02-27
I just finished Learning Python and started with Programming Python.

Learning Python was pretty good, I already knew several languages prior to reading, so it could be slow at times.  However, it is a great Python foundation.  It is esoecially good for those with little programming experience.

Programming Python is a hefty book but covers lots of topics.  I am reading the Tkinter section now and it is great.  I tried some online tutorials, but the book is much more comprehensive.

II also read most of Thinking Like a Python Programmer.  It is a free and open souce book you can download in PDF form.  Here is the link [http://www.greenteapress.com/thinkpython/]("http://www.greenteapress.com/thinkpython/")

Have fun learning!
