### Post by spupy on 2010-05-07
Keep in mind that Qt is also a full toolkit, not just GUI. The portability and native-looking-ness are also very nice.

Some words about PyQt, though. It is not developed by Nokia. It is GPL + commercial licensed, I don't know if that will bother you.
Last: while Qt itself is easy to be ported - just cross-compile or whatever, PyQt isn't so easy. Just today I read a tutorial by a guy who was trying to deploy a simple pyqt program to windows. After all the struggles the install file was ~25MB. But all these are not relevant if you only code for fun/learning/linux.
