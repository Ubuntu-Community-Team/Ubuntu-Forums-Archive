### Post by smartbei on 2007-11-29
For a dialog that simple I wouldn't bother useing a full-featured GUI builder such as Glade. Just use Tkinter. It is very simple, and there are excellent online reference and example websites. If you do want to try out a fully-featured gui building IDE, try Glade. I have never used Boa Constructor so I am unable to provide a comparison.

GTK stands for Gimp Toolkit, and is the default graphical user interface toolkit used in Gnome. QT4 is the default GUI toolkit used in KDE.
