### Post by TimeTraveler on 2008-09-08
_A tip about programming c++ in eclipse _

If you have a problem compiling a c++ hello-world program, after installing the Ubuntu Linux and the eclipse platform (c++) - you should know that Ubuntu 8.04 won't arrive with the headers you need, such as <stdio.h> etc.

so what you simply need to do is install these headers using a console:
just write down: 

**sudo apt-get install build-essential**

and it will install what you need on your Linux

now you can work with eclipse c++ on your Linux.
