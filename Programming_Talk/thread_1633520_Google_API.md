### Post by Zugzwang on 2010-11-29
Well, you will have to look up how your request is done correctly. Use the API documentation provided by Google: [http://code.google.com/apis/maps/documentation/webservices/index.html](http://code.google.com/apis/maps/documentation/webservices/index.html) The output by google for usage in your browser (the link provided) is only meant to be used in your browser - thus, you won't have much luck using it.
