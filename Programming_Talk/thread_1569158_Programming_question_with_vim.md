### Post by Bachstelze on 2010-09-06
vim-tiny, as its name implies, installs a stripped)down version of vim, with some features removed in order to make it smaller

vim is the normal vim

vim-gtk installs vim plus a GTK graphicla interface for it (you can run vim in either command-line or GUI mode)

vim-gnome is the same as vim-gtk, with some additions to make the GUI integrate better with Gnome

vim-nox installs a version or vim with Python, Perl and Tcl scripting engines, but no GUI.
