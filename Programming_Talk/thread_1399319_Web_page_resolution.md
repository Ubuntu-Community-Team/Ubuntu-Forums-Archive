### Post by Reiger on 2010-02-05
Tricky, but possible. What you could do is fix the dimensions (while testing) of the HTML element.

Then what you need is a window manager that allows you to zoom/scale in/out your desktop, accepts windows that are larger than the screen dimensions and is capable of displaying individual pages of a desktop/workspace or has a similar concept.
