Support for editing menu bars and tool bars (among other things) was added to designer in Qt 4.1 which is the current version.
Sadly the version in Breezy is the very old Qt 4.0.0 and and even in current Dapper has only version 4.0.1. If you are interested in doing some Qt4 stuff I recommend to download the source from TrollTech.com and compile it yourself.
