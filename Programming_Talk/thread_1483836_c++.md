
and eclipse for C++ IDE in the repos.  It has code completion, syntax highlighting refactoring, version control plug ins  etc.

### Post by Tony Flury on 2010-05-15
The Best compiler is the GNU complier - g++ - it should already be installed - but just incase : 

```

sudo apt-get install g++ build-essential

```

As for IDEs - people will recommend Ajunta, Eclipse and possibly even Emacs. When I use an IDE, I use IDLE (but then I program in python). For a general good editor, I use Kate (multiple files, code-folding etc).
