### Post by PaulM1985 on 2009-02-22
I would not say that it is a waste of time.  It is still worth reading through for reference.  Some things have been made simpler now as classes have been added to reduce the amount of additional programming.  It doesn't hurt to use it and buy a more up-to-date book later.
