    operating_system=-1
```
If you need to test the platform just test against sys.platform.

It's probably best to avoid this kinda thing:
```
os.system("/usr/bin/xvkbd -xsendevent -text \"" + key + "\"")
```
and use a library instead, like Xlib.

There were a few bits of code that were repeated, which isn't great, but you're getting there.
