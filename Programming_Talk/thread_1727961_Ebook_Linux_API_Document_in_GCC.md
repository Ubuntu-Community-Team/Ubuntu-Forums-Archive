### Post by dwhitney67 on 2011-04-13
If you comply with programming in ISO C, it should be equally difficult (or easy) to develop under Linux as in Windoze.

Under Linux, reference information for C library functions can be found in the man-pages.  You should have these installed on your system; if not, then use the Synaptic Package Manager to get them.

Alternatively, you can browse the man-pages online at this site: [http://www.linuxmanpages.com/](http://www.linuxmanpages.com/)
