### Post by -Rick- on 2006-12-13
I've made Lua support for my project. Lua is a very nice language, I recommend to read the [book]("http://www.lua.org/pil/").
Lua is not really a standalone language, while python is. This is why lua by default has far less libraries then python. This is ofcourse good for buildin languages since it keeps the size small.
On the wiki you can find the following comparisation for Lua vs Python: [link]("http://lua-users.org/wiki/LuaVersusPython").
