### Post by dwhitney67 on 2008-08-04
An IPv4 IP address consists of a quad of four numbers.  The minimum value for a number is 0, and the maximum is 255.  In hex, this would be 0x00 and 0xFF, respectively.

Thus an IP address of 192.168.1.100 is equivalent to C0.A8.01.64.

It seems that the values you provided are indeed representative of IP addresses, however I did not look at each and every value to verify this.

I would recommend that you write a script or other application (C, C++, whatever) to parse the file/data.  Collect four values at a time, to formulate the IP address, then print it out in human-readable format.

I presume the \x in front of all of the values denotes that the value that follows in in hex.
