### Post by shifty2 on 2008-08-01
There is a different mindset between vim and emacs users. Vim users have the unix bag of tools mentality where you have a lot of tools to do the job. You use vim to write code, terminal to compile it etc etc. Emacs users one one tool to do the job for them. In emacs I can read my emails, compile a document, browse the internet, run vim. Its just a case of choosing which side you fall on - both are very very good programs.
