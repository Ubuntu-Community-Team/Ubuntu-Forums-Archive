### Post by samjh on 2007-05-22
> **reidms said:**
> Do you think C++ beats Java in Game Development?

For AAA titles, C++ is the king of the hill, and it won't change for many years yet.

For indie-style titles, and especially for indie developers targeting the widest possible market by going multi/cross-platform, Java is becoming increasingly popular.  Check out [www.jmonkeyengine.com](www.jmonkeyengine.com) which is being used by several indie developers for some very promising 3D titles (check the screenshots).

JOGL is a lower-level 3D API that is a Java wrapper for OpenGL, and is now stable enough for mainstream use.  See [https://jogl.dev.java.net/](https://jogl.dev.java.net/) for examples.

Can't forget to mention Tribal Trouble: [http://tribaltrouble.com/](http://tribaltrouble.com/) :)

But if AAA titles are your goal, your best bet is with C++, and you had better be really good at using it!
