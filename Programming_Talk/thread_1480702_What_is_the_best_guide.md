### Post by trent.josephsen on 2010-05-17
Best introduction to a language for non-programmers:  *Learning Perl*.

Best introduction to a language for programmers:  *The C Programming Language*.

Best language reference manual:  *Programming Perl*.

Best introduction to programming in general:  probably *Head First Programming*.

Worst programming book ever:  *Jamsa's C/C++ Programmer's Bible*.

You can't go wrong with O'Reilly, but looking at just their library will limit your options.
