I would suggest books from this range: [http://www.wrox.com/WileyCDA/Section/PHP-MySQL.id-131383.html](http://www.wrox.com/WileyCDA/Section/PHP-MySQL.id-131383.html)

I was very impressed with the way [this book]("http://www.wrox.com/WileyCDA/WroxTitle/Professional-PHP5.productCd-0764572822.html") was set out. It is practical and follows a logic order of progression. I would recomment it to anyone wanting to learn PHP and specificaly OO in PHP.
