### Post by ziekfiguur on 2010-10-31
To do what you want i think you would have to install and use java-gnome. Take a look at [http://java-gnome.sourceforge.net/](http://java-gnome.sourceforge.net/)
However, I would actually advice on using python or C/C++ instead of java for desktop applications. 

Python has an excellent binding to gnome called PyGTK which is part op the ubuntu standard installation. Because of that users of your program will not have to install additional packages like java or java-gnome. And because it is python your programs will be readable and easily editable for users too.

C++ might be better when you have to do some heavy calculations or make extensive use of hardware accelerated graphics (although python has some great libraries for those things too). And, because it is linked to the same libraries other gnome programs use, your users would not have to install any additional packages as well.
