Sure, but it is still worth learning how to set up and run your own web server, understanding the platform which is serving your files/running your code is essential to understanding how everything works and fixing problems when they occur.

Professional hosting providers, especially free or very cheep ones are not necessarily very secure either as they try to cram as many sites as passable onto a single computer. As most hosts also offer PHP, ones trying to obtain the lowest overhead are more likely to run it as mod_php, meaning that all the code from every site on the server is running as the same user as apache, meaning that any user can read any other users files. Hacks like Safe mode don't truly solve the problem.
