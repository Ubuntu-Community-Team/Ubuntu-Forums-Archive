### Post by Firestom on 2009-04-09
As a lot of software is still developed only for Windows, you'd rather partition your hard drive to be able to reboot under Windows. This said, translating code from Windows-compatible to Linux-compatible could be very fast, or very long, depending on libraries and external resources used. If you really need it for your classes, partition your disk in order to install Windows XP or Vista for now, and the community will see what they can do! I really hate saying to install Windows, but if it seems you really need it, this would be the best option.
