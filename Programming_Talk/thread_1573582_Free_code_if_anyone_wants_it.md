### Post by soltanis on 2010-09-13
Since you technically hold the copyright to this code, it is up to you to release it if you wish under the terms of the GPLv2. Applying the GPL in earnest to your code is a bit of a pain but it frees up a lot of legal ambiguity. Read about it [here]("http://www.gnu.org/licenses/gpl-howto.html") on the GNU web site.
