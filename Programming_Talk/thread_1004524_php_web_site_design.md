### Post by Delever on 2008-12-07
Quick startup if you are using Ubuntu:

Install apache server and php5 on your computer:

```
sudo apt-get install apache2 php5 libapache2-mod-php5
```

Create "test.php" file in "/var/www" (you will need write rights, for example, you can run nautilus with sudo (gksudo nautilus) to access that directory).

Access your page by going to [http://localhost/test.php](http://localhost/test.php)

You can place html/php into that file and see results in browser.

Search for some tutorials, follow them, and good luck :)
