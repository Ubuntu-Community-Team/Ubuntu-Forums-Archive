### Post by shadow_code on 2009-02-27
Hi, I posted this already in another forum, but noticed this is probably a better place for it.

I've just released the 1.0 of a new plugin for gedit called Multi-edit. It allows you to have multiple text cursors so that you can simultaneously edit multiple parts of a document as you type. Kind of like an advanced column edit, and similar to what's available in e-texteditor and TextMate.

More details found at: [http://jon-walsh.com/journal/multi-edit/](http://jon-walsh.com/journal/multi-edit/)

Hope you like it :D

Cheers,
Jon
