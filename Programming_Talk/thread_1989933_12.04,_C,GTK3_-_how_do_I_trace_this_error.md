### Post by denago on 2012-05-29
Here's a breakdown of your problem.


```
font_name = gtk_combo_box_text_get_active_text  (panel_font_type);
    
```
This block of code returns a gchar* array of the text in the drop down.  Let's say the text is "Andale Mono".  This will return an array of 12 gchars witht he value of text followed by the null character which terminates the string.  You then call strcat like so:

```
strcat(font_name," Bold");
```

This combines the 11 characters in the array "Andale Mono", removes the null character, adds " Bold" followed by a null character to the end of the array going passed the 12 gchar's that make up the array.  That's your buffer overflow.  Try replacing it with something like so:

```

   GString font_name = new GString(gtk_combo_box_text_get_active_text(panel_font_type);

   if (gtk_toggle_button_get_active(panel_font_bold))
   {
      g_string_append_c(font_name, " Bold");
   }

```

etc...

You get the point

> **Barrucadu said:**
> Remember in C that a string is just a char*.

No, a string is not "just" a char*. A string is a null-terminated sequence of chars, that is accessed via a char* pointing to the first character of the sequence.

Saying "a string is just a char*" gives the impression that you can just do

```
char *s;
```

and get a string. That is not true.

### Post by Bachstelze on 2012-05-31
Here's how you would create a dynamic array of strings:

```
firas@av104151 ~ % cat strings.c                                           
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

void usage(char *s)
{
    fprintf(stderr, "Usage: %s [n] [string1 string2 ...]\n", s);
    exit(EXIT_FAILURE);
}

int main(int argc, char **argv)
{
    /* Parse the arguments */
    if (argc < 3) {
        fprintf(stderr, "Need at least 2 arguments!\n");
        usage(argv[0]);
    }
    char *p;
    int n = strtol(argv[1], &p, 10);
    if (*p != '\0') {
        fprintf(stderr, "First argument was not a number!\n");
        usage(argv[0]);
    }
    if (n <= 0) {
        fprintf(stderr, "Need at least one string!\n");
        usage(argv[0]);
    }
    if (argc < n+2) {
        fprintf(stderr, "Not enough strings!\n");
        usage(argv[0]);
    }

    /* Allocate space for the array */
    char **strings = malloc(n*sizeof(char*));
    if (strings == NULL) {
        perror("Could not allocate memory");
        exit(EXIT_FAILURE);
    }

    /* Allocate space for each string and copy it */
    for (int i=0; i<n; i++) {
        int ls = strlen(argv[i+2]);
        strings[i] = malloc(ls+1);
        if (strings[i] == NULL) {
            perror("Could not allocate memory");
            exit(EXIT_FAILURE);
        }
        strncpy(strings[i], argv[i+2], ls+1);
    }

    /* Print results */
    printf("Number of strings: %d\n\n", n);
    for (int i=0; i<n; i++) {
        printf("String %d: %s\n", i, strings[i]);
    }

    /* Clean up */
    for (int i=0; i<n; i++) {
        free(strings[i]);
    }
    free(strings);
}
firas@av104151 ~ % gcc -std=c99 -pedantic -Wall -Wextra -o strings strings.c
firas@av104151 ~ % ./strings 4 foo bar baz supercalifragiliticexpialidocious
Number of strings: 4

String 0: foo
String 1: bar
String 2: baz
String 3: supercalifragiliticexpialidocious

```
