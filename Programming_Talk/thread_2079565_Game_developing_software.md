### Post by jerome1232 on 2012-11-02
You can use eclipse and set it up for programing for android devices, there is a great how to on Androids Developers site to get you started with Eclipse on Ubuntu. Download the Android SDK here: [http://developer.android.com/sdk/index.html](http://developer.android.com/sdk/index.html)
 
 Follow the how to here: [http://developer.android.com/sdk/installing/index.html](http://developer.android.com/sdk/installing/index.html)

### Post by Brandonlayton on 2012-11-02
The only INTERACTIVE Game Engine for Android that I know of is Unity 3D, but there isn't a Linux release for it. I am not sure if that is what you are asking for. You can always use that with Wine.

Now if you're asking for stuff like graphic editor and what not then I recommend Blender for 3D and Inkscape for 2D Vector. Gimp is also good.

As the post above me says, AndEngine is good if you want a Game Engine (not interactive) for android, but I have always preferred programming from scratch, more flexibility :).
