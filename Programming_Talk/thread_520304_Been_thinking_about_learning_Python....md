### Post by pmasiar on 2007-08-08
and see my wiki too :-) - online books, guides, and training tasks.

Python is excellent language for both beginner and experienced programmer: easy to learn, and scales up well. It is first language designed to communication between humans as priority - being readable is important. Feature implementations are rejected on account of being ugly or not obvious enough.

Many editors are fine to use for Python, just need be careful and don't mix tabs and spaces for indent - Python uses whitespace for structure (correct indent instead of BEGIN... END or braces). Most editors will allow you to set "convert tab to spaces" and you are fine.

I recomment to start with IDLE as your IDE - install from repos.

Good luck!
