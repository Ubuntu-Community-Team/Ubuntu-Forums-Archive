### Post by TheOrangePeanut on 2011-01-19
It's becoming increasingly populate to refer to HTML5 as the complete stack of web technologies to build rich internet applications.  This technology stack includes the updated HTML markup, CSS3, and javascript with new functionality like audio/video tags, canvas tag, web workers, web sockets, etc.

With that said, I don't know any good IDEs for HTML5 development.  Hopefully as the technologies become more popular to build more interactive web applications, some better tools will come also.
