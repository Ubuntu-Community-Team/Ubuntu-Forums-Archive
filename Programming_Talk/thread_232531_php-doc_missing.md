Php-doc has been missing from Debian for a long time, too. I've used Debian for a long time and have had to download it manually from the PHP website. I see Lior Kaplan is the new maintainer, he is also responsible for gPHPEdit in Debian.

As you can see, the package has no dependencies. So, you can [download]("http://ftp.us.debian.org/debian/pool/non-free/p/php-doc/php-doc_20060408-1_all.deb") it and install it easily, without any problem!
