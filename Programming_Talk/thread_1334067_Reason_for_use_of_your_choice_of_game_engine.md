### Post by maximinus_uk on 2009-11-23
Having played with many many engines over the years, the most important considerations are:

1: Is it available in my preferred language?
2: Is the documentation good?
3: Would using this library make it hard for others to run my code?

3 is very relevant for some libraries such as PyOgre, which are known to hard to setup / install.

It is passes all that, then it comes down to:

4: Is it easy to image data and or models (most important in 3d)
5: Are the forums busy? Is it still in development?
6: Is it stable and non-hacky?

You'll note that IS IT FAST? and DOES IT LOOK COOL? are not mentioned at all. The former might have been important in the 90's but it's mainly irrelevant now.
