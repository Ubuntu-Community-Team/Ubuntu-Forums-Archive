### Post by Alasdair on 2009-01-06
Cabal (the library) comes with GHC, but if you are looking for the command line cabal-install utility you'll probably need to get it from hackage and build it manually.

[http://hackage.haskell.org/cgi-bin/hackage-scripts/package/cabal-install](http://hackage.haskell.org/cgi-bin/hackage-scripts/package/cabal-install)
