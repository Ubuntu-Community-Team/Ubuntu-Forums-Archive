### Post by thumper on 2006-04-06
Firstly you don't learn C/C++, you learn either C **or** C++.

For C++ I, and many others, recommend a book by Andrew Koenig and Barbara Moo called "Accelerated C++".  I believe that it has exercises througout to give you something to implement.

As far as [KX]?ubuntu goes, apt-get install build-essential will get the environment.  gcc for C and g++ for C++ to compile.  There are many posts in this forum about using these commands to build executables.
