### Post by Jamie Jackson on 2007-08-10
1. It should already be in your Applications >> Programming menu. I don't know of an easy way to update Eclipse along with the Ubuntu updater. They each have their own mechanisms.

In Eclipse, if you go to Window >> Preferences >> Install/Update, you have options there as to scheduling updates, etc.

2. If you know the name of the plugin, you can google "yourPlugin Update Site" and that will usually give you the URL to provide to Help >> Software Updates >> Find and Install >> Search for New Features to Install >> New Remote Site

Then, you put in the URL and label it, and the rest is fairly obvious.
