### Post by krypto_wizard on 2006-08-14
I was in a similar position as you are and I gave python a chance and I am glad I did that.

I recently wrote an app for a Company using wxPython. Creating GUI is really very easy using wxPython. You could use GUI designer tools like wxGlade to quickly make GUI. wxPython sample code gives you sample of every small widget and tells you how to use it. On top of that the mailing list is also very helpful for wxPython.

I would suggest you to go ahead with Python.
