### Post by boralyl on 2007-07-08
When I started learning Python I picked up the book, "Beginning Python: From Novice to Professional" from Apress publishing.  It was extremely helpful, starts off with some basic stuff then goes into more advanced items.  Highly reccomended.  And if you are planning on doing GUI stuff, I use wxPython, and the book wxPython In Action was an amazing resource.  For wxPython I also use [http://wxpython.wxcommunity.com/docs/api/wx-module.html]("http://wxpython.wxcommunity.com/docs/api/wx-module.html") which has the api.  Good luck with your learning.

Another good resource with a bunch of tutorials broken down by category: [http://www.awaretek.com/tutorials.html]("http://www.awaretek.com/tutorials.html")
