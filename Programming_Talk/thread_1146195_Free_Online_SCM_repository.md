### Post by wmcbrine on 2009-05-02
There are tons of these services nowadays. Of course, most of them are geared towards making your project available to the world, not just sharing between home and office. For the latter, if possible, I'd recommend running your own server. (This is really easy with git -- all it needs is ssh access, no special configuration. git itself doesn't actually run as a server, just works through sshd.)

If you do want to share with the world, see here:

[http://en.wikipedia.org/wiki/Comparison_of_free_software_hosting_facilities](http://en.wikipedia.org/wiki/Comparison_of_free_software_hosting_facilities)
