### Post by norobro on 2021-01-16
Found the following here: [https://snapcraft.io/kompozer](https://snapcraft.io/kompozer)> KompoZer has not been in active development since 2010, and the latest version is KompoZer 0.8 beta 3 using the Gecko 1.8.1 engine. It is still usable for mostly offline Web development and design, with partial support for more modern standards. This snaps preserves the last available version, allows KompoZer to be used on modern operating systems.Many moons ago, I used and really liked quantas plus on KDE3. For what ever reason development ceased at about the same time as development on KompoZer.

### Post by deadflowr on 2021-01-19
> **Tom_Carr said:**
> Maybe the only new thing is that I can once again install Kompozer directly from the Ubuntu Software Center, rather than downloading and installing from the terminal.  For several years it was not available in the Software Center.


It's now available as a snap which can install the legacy packages without compromising the rest of the overall system.
See: [https://snapcraft.io/blog/how-to-preserve-old-software-with-snaps]("https://snapcraft.io/blog/how-to-preserve-old-software-with-snaps")

Edit: Note that the Software Store enables snap package listing/installation/updating/management/removal options by default on Ubuntu.
(Though snap auto-update themselves so the odds of seeing updates through the Store are slim; I've never seen a snap update being available through it,
but I assume it can from what I've at least read about it.)
