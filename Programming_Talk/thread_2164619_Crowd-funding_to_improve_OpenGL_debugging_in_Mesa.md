### Post by t-fridey on 2013-08-01
Hi Guys,

I though some people here might be interested in my crowd-funding campaign to improve the OpenGL debugging support in the open source Mesa library.

Basically I'm raising money to implement the GL_KHR_DEBUG extension. This extension helps OpenGL become a more appealing API to developers. From the spec:
 "The intended purpose of this extension is purely to improve the user  experience within OpenGL development tools and application built-in  profilers and debuggers. This extension typically improves OpenGL  programmers efficiency by allowing them to instantly detect issues and  the reason for these issues giving him more time to focus on adding new  features to an OpenGL application."

I'm happy to answer any question you might have, so feel free to ask me.

My Goal is to raise: $2500
I'm currently at: $1080

For a full description of my campaign see: [http://igg.me/p/475220/x/2053460](http://igg.me/p/475220/x/2053460)

Thanks for your time,
Tim
