### Post by soltanis on 2009-04-10
Java is best done with an IDE imho. I suggest you try netbeans or eclipse, installed with the package sun-java6-jdk.

If you don't want to use an IDE, you can use the default editor in gnome (gedit, it's pretty good for coding, you can set some options in the preferences to make it more like a code editor) or you may choose to go with vi/emacs. If you are going to do that, by the way, you better pick on and stick to it, or risk a terrible fate (at the hands of command line editor fanatics).
