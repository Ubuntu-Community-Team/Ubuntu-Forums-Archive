Hi,

May be people here already know it, but just wanted to share this.

Ubuntu makes it real easy to install Perl Modules. 

All you need to do is open the Ubuntu Software Center and type the name of the module and it works !!

For example, say you need to install the LibXML module. You can install it from command line using CPAN, the usual way, or, you just open up the Software Center and type "XML::LibXML" (without quotes) and it lists the module(s). It also installs all the required dependent modules as well.

One more reason why I stick to Ubuntu. :)
