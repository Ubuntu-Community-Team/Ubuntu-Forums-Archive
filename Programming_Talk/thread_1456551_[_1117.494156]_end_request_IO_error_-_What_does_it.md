### Post by cdekter on 2010-04-18
This error is printed when the kernel can't access a given block device. In this case it's a CD drive, likely the one containing the live CD. I'm guessing the order in which devices are disabled during shutdown causes the CD device to go away before the kernel has actually finished reading from it.
