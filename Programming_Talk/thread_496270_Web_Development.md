### Post by Modred on 2007-07-08
You could set up a web server on your computer, but not open it up to the whole world.  Just use it for testing and such.

Some frameworks (Rails and Django come to mind) come with a development server you can use for local testing, so that would be all you needed to get your feet wet.  As far as I know, you'd have to go the route of actually installing and setting up a server to test PHP web sites, although I could be mistaken.

### Post by unixhead on 2007-07-09
I would recommend to go with [Rails]("http://www.rubyonrails.org/") and [Django]("http://www.djangoproject.com/") no matter what you choose to work with later. Rails and Django show you many interesting ideas that save your time and force you using best practices of web development, such as testing, true usage of OOP and metaprogramming.

Though this is not php frameworks and definitely may not be what you are looking for at the moment it's definitely too most interesting frameworks for newbies cause they learn you good stuff. And more languages you know, the better.
