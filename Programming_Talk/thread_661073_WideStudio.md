Why I'm looking at WideStudio is because it seems to really cater for a full cross-platform environment and also for various languages.

[COLOR="DarkRed"]"This enables you to develop GUI applications that can run on Windows95/98/Me/NT/2000/Xp, WindowsCE, Linux, FreeBSD, SOLARIS, MacOSX(w/X11), BTRON, T-Engine, mu-CLinux(wo/X11) in various programming languages such as C/C++, Java, Perl, Ruby, Python,Objective Caml

### Post by popch on 2008-01-08
I had a quick look at the web site.

It says right there on one of the first pages that the product is made in Japan. That might explain the language usage. IMO, the language is certainly not worse than that used by some forum members here, even some originating in English speaking countries.

The product is kept well up-to-date and supports an impressive number of platforms.

I don't share the opinion of others here that an IDE is useless. When a project reaches a certain size, getting some help in organizing code, other ressources and views can be extremely helpful. Also, some people are more comfortable with a GUI painter than with manually coding all widgets which make up any GUI.

Having said that, I still have no opinion about the actual product. I think I will have a closer look, but don't keep your breath until then.
