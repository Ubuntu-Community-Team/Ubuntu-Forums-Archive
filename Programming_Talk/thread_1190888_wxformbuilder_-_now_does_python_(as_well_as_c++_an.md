### Post by pelle.k on 2009-06-18
Just wanted to inform you people that [wxformbuilder]("http://www.wxformbuilder.org/"), which is a form builder such as wxglade/glade, now generates python code as well!
I love wxwidgets as a framework, but the only thing missing was a lightweight, no fuss, form builder for wxpython. wxglade never did work that well for me (if you felt it was/is enough for you - good for you).

I've been working on a pretty big project now, with over 500 lines of wxformbuilder generated GUI code, and it's really made a world of difference, since i can now concentrate on the actual function of the code instead of building the gui. Suffice it to say i couldn't be more pleased!

It's in beta as of today, but i haven't hit any bugs personally, though. Here's the PPA: [https://launchpad.net/~rjmyst3/+archive/ppa](https://launchpad.net/~rjmyst3/+archive/ppa)
I hope you'll enjoy it!
