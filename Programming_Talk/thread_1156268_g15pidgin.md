### Post by Schitso on 2009-05-11
So, I threw together this program for myself so that I can use my g15 keyboard to chat with my friends while I'm playing games. (I use my monitor for my computer and my consoles) It's basic and rough, but I figure it could be of some use to others.

Patch and install webpidgin, turn off color coded history and bold buddy names in its options, and set the userpass variable in g15pidgin.c

Your webpidgin userpass can be acquired with this:
```
echo -n "USERNAME:PASSWORD" | base64 
```

Have fun.
