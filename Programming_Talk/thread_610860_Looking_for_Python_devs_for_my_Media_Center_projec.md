I have been working on new Media Center application since last May. Recently I got a job and I can't spend as much time on this project as I would like to. That's why I'm looking for interested co-workers for the project. If you know Python and object-oriented programming that's enough!

[IMG]http://www.taimila.com/pics/entertainer-promo-small.png[/IMG]

My blog posts about this project: [_http://www.taimila.com_]("http://www.taimila.com")
Project homepage: [_http://www.entertainer-project.com_]("http://www.entertainer-project.com")
Screenshots of current version: [_Flickr_]("http://www.flickr.com/photos/9111235@N06/tags/entertainer/")
Launchpad entry: [_https://launchpad.net/entertainer_]("https://launchpad.net/entertainer")

Let me introduce **Entertainer**. Entertainer is a GTK-based media center application which uses Clutter library for beautiful OpenGL GUI. Multimedia is handeled with a great gstreamer backend. This combination provides support for large variety of codecs and eye-pleasing UI. Entertainer has seperated backend and frontend processes. Backend is responsible of indexing media, recording TV, updateing RSS feeds and stuff like that. Frontend is mainly just a GUI to media cache.

Currently the project is not ready for release, but you can run it and play audio, video and watch photographs from your harddrive. It also retrieves metadata from internet (album art, movie metadata, lyrics). Read more from my blog.

So, if you feel that you could help project like this, please contact me and we'll talk about it. You can reach me by email **lauri@taimila.com** and with MSN messanger **ltaimila@surfeu.fi**

Oh, and yes. This is an open source GPL licenced project. Code is available at Launchpad.
