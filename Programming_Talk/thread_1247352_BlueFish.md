Bluefish is a powerful editor targeted towards programmers and webdesigners, with many options to write websites, scripts and programming code. Bluefish supports many programming and markup languages, and it focuses on editing dynamic and interactive websites. See [features]("http://bluefish.openoffice.nl/features.html") for an extensive overview, take a look at the [screenshots]("http://bluefish.openoffice.nl/screenshots.html"), or [download]("http://bluefish.openoffice.nl/download.html") it right away. Bluefish is an [open source development]("http://bluefish.openoffice.nl/development.html") project, released under the GNU GPL licence.
