### Post by SunSpyda on 2009-02-23
> **jimi_hendrix said:**
> just curious...why korn?

Because it's the default shell in OpenBSD and other BSDs ;)

I use a terminal a lot, but I don't often shell script (I use C with system calls instead). Most of the big differences between shells is in the scripting I believe.

I use Bash on Debian, and only a few commands are different to Korn in OpenBSD, even advance stuff like | and > doesn't seem that different.

I suppose I should standardize all my PCs, and just install Bash onto OpenBSD, but I have never got round to it, seeing as I have never needed to.

So, it's just a force of habit (And I hate to admit it), since OpenBSD is my primary OS, and the Korn shell is default for this.

If big differences start to emerge between Korn and Bash, I will of course install Bash onto OpenBSD, and use that instead.
