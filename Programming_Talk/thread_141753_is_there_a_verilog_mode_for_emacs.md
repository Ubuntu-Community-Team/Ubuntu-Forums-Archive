### Post by toojays on 2006-03-09
Google gave me several hits for "verilog mode emacs". [Here's one](http://www.verilog.com/verilog-mode.html). I have used one in the past, but I don't remember which one exactly.

You can use customise to turn on font highlighting, but you have to drill down so far into the menus that it's ridiculous. An easier option is to put "(global-font-lock-mode t)" in your .emacs file.
