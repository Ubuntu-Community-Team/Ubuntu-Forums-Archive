### Post by mltucker on 2009-04-13
Hey Robin,

I actually do my C# development with visual studio on a windows machine (VS is Microsoft's best product by far IMO).  On the other hand, I've played with Mono a bit, and if you get Mono and MonoDevelop, you'll have a complete IDE and be able to write platform-independent C# code running on Mono.  Furthermore,

"The Open Toolkit is a free, cross-platform OpenGL and OpenAL wrapper for C# and other .Net languages."

This sounds like exactly what you'll need!

[http://www.opentk.com/]("http://www.opentk.com/")

Good luck!
