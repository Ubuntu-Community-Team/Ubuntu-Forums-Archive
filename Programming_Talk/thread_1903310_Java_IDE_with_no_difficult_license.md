### Post by lykwydchykyn on 2012-01-02
> **jaho22 said:**
> Could you write few lines about this OSI license, is it something like 'simply install it and use it and forget what you are using, simply write your code with it and dont worry about ide licenses or is there cat catches' ?

OSI isn't a license, it's short for "Open Source Initiative".  It's basically a group that (among other things) certifies various licenses as "Open Source".

In order to be certified "Open Source", a license has to meet the [open source definition](http://www.opensource.org/docs/osd).
