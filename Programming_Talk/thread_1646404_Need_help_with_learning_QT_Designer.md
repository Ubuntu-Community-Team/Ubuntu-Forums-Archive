### Post by dwhitney67 on 2010-12-15
Since it seems that you have no steadfast requirements, you might want to consider using the gtkmm (package: libgtkmm-2.4-dev), which is a C++ wrapper around GTK+.  There's also good [documentation]("http://library.gnome.org/devel/gtkmm-tutorial/stable/") for it.

As for Qt, you can find documentation [here]("http://doc.trolltech.com/4.3/tutorial.html").
