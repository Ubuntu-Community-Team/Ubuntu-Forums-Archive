### Post by laceration on 2008-12-28
To me that is a weird question.  That's is just because of the way I learn, I am not saying you are weird to ask it.  I learn by doing.  So I would ask what do you want to do?  If you want to make dynamic websites or web applications PHP absolutely rocks!  It is easy to learn and not more complicated than it needs to be.  [www.php.net](www.php.net) is a great resource, every function is documented, has tons of user notes and PHP has a giganormous contributing user base.  It will complement what you know about web design.

Lately I have been working on something to integrate programs and various systems commands, so I have been learning BASH, which is suited for that, though I not sure it qualifies as a programming language exactly.  I wonder if I should have used Python instead though.  I haven't messed around with Python too much but it is a well conceived language.  If I had a desktop app project I would use it.

If you want to get into the bowels of Ubuntu or Linux it would have to be C or C++.
