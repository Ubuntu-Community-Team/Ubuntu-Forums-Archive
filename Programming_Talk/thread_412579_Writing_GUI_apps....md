### Post by samjh on 2007-04-18
You will need to learn GTK.  The site for C++ bindings is [www.gtkmm.org](www.gtkmm.org)

Documentation is good, including this free online e-book: [http://www.gtkmm.org/docs/gtkmm-2.4/docs/tutorial/html/index.html](http://www.gtkmm.org/docs/gtkmm-2.4/docs/tutorial/html/index.html)
