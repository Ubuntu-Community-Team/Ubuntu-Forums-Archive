### Post by nhandler on 2008-04-23
With the recent forum update, we gained the ability to add tags to threads. I feel this feature will become particularly useful in this programming forum. Most of the time, if a user is trying to find the answer to a programming question, they only care about certain languages. This is where tagging comes into play. If we simply add a tag that contains the name of the language the thread is about, we can make it much easier to find the threads we are searching for. Adding a tag is quick and easy, so there is no reason not to tag every thread that you make. You can still help even if you don't make threads. Just click the "Edit Tags" link near the bottom of a thread, and add the appropriate tags. If we work together as a community, we can make this great archive of programming knowledge much more accessible.
