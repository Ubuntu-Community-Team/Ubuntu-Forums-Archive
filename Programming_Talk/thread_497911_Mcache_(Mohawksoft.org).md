### Post by CopaceticOpus on 2007-07-10
Here is how to install Mcache from MohawkSoft.org successfully. I didn't see instructions for this on Ubuntu anywhere, so I hope this helps someone. I'm on 7.04 (x86).

You'll first want to do this:
```
sudo apt-get install build-essentials libncurses5-dev
```

Now go follow the instructions at the following page, except for the "yum install..." command at the top:

[http://crazytoon.com/2007/05/23/mcache-install-and-configure-mcache-msession-to-be-used-for-session-caching-in-php/](http://crazytoon.com/2007/05/23/mcache-install-and-configure-mcache-msession-to-be-used-for-session-caching-in-php/)

There are a couple tricky undocumented steps in there that I never would have figured out. Make sure to follow it precisely.
