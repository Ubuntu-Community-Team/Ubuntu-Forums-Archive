### Post by hamzah2096 on 2009-02-03
few tips. 
1. install pydev from synaptic.
2. start eclipse
3. go to _W_indow menu.
4. under _W_indow go preference.
5. there should be Pydev in the list, if not it is installation prob

6. In Pydev pick Interpreter-Python
7. Click New and browse to /usr/bin then select python2.5
8. Click apply then ok (unless you want to add other lib).

9. Try debug with _D_ebug As > Python Run

Hope that works.

I dunno about code completion. Maybe you should try do this. Import os module. Next, type "os." only and press ctrl+space. I think maybe that is code completion.
