### Post by The Cog on 2008-11-08
It's predominanently a java question, I think.

I connect to MySQL using the driver downloaded from here:
[http://dev.mysql.com/downloads/connector/j/5.1.html](http://dev.mysql.com/downloads/connector/j/5.1.html)

The driver file is mysql-connector-java-5.1.7-bin.jar. I don't know which JVM you are using. I know that for the Sun jvm you just copy that file to the to the jre/lib/ext directory, then restart OOo. You'll have to search for your jre/lib/ext directory - I don't know where the Ubuntu devs hide it (I keep my jre in /opt).
