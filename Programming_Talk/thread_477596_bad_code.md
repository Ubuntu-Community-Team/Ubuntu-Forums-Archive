> 
Is possible that this user send me the  Malware  accidentally?

That's definitely possible. It's been a common way for malware to spread for a few years now -- it infects a person's computer, then finds an address book or friends list or whatever, and sends itself out over email or IM to the person's friends and acquaintances. It's very effective, because people are more trusting of a message that seems to come from someone they know.
