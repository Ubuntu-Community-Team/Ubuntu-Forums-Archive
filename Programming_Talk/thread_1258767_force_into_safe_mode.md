### Post by Volt9000 on 2009-09-05
Well first of all, you're in completely the wrong forum. This is not a programming question.

Second of all, to get into safe mode, you use F8. The best thing to do is right before Windows begins to boot, start punching F8 like a madman. Eventually the Windows bootloader will pick up on it, and let you choose Safe Mode.

If F8 doesn't work for you, then yes your keyboard is fubar'd. Better try another one.
