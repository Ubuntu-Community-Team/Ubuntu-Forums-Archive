### Post by Gordon Bennett on 2009-03-20
As we all well know, programming can be a fine creative mix of elation and self-flagellation, so here is a thread where one's ruminations on the matter can be expressed via the art of haiku.
The general rule for a [(link)haiku]("http://en.wikipedia.org/wiki/Haiku_in_English") is that it is primarily composed of three lines of 5-7-5 syllables each.  Of course, as with programming, rules can be bent/broken :P

Here's a humble programming haiku offering:


[I]stubborn line of code
why must you mock me with crash
we will see who's boss[/I]
