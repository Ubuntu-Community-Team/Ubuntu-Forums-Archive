Hi,
My world is highly limited to Oracle so when it comes to table design I can only suggest related products. My next try will be JDeveloper (yes, it's free): [http://www.oracle.com/technology/products/jdev/index.html](http://www.oracle.com/technology/products/jdev/index.html)

I post a quick and ugly screenshot for your consideration. You can model there and then export in .png the results. You can do UML too. My experience regarding importing schema definitions and let the soft build the model is poor but I'm sure you can access different vendors db just with the jdbc api.

I consider it might be a little too much for your requirement (downloads run from 55MB to 361 MB) but if you have no other option check it out.
