### Post by schauerlich on 2011-05-23
> **rpmp said:**
> can someone explain me how this works?

Pick a challenge, write some code to solve the problem given, and then post it. If you post to a challenge that's still accepting entries, you have a shot at winning. The winner gets to make the next challenge, or pick someone else to do it if they don't want to. The only challenge open currently is [this one](http://ubuntuforums.org/showthread.php?p=10723980), which just so happens to be the one I'm running. :)
