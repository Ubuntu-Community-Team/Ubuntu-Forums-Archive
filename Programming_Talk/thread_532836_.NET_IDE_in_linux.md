### Post by Dragonbite on 2007-08-28
With Edubuntu
If you "Install on Hard Disk" it installs a thin client server which I do not believe includes a LAMP server.

If you "Install on Workstation" it installs locally (like Ubuntu installs) and does not include the thin client component.

Ubuntu Server includes a simple means to install a LAMP server.

Yes, ShipIt sends Edubuntu CDs, but they are not Live CDs where you can "try before you buy", they are installation only.

For more information check out the [[COLOR="Blue"]Edubuntu Website[/COLOR]]("http://www.edubuntu.org").
