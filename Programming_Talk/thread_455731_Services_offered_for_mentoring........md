### Post by pmasiar on 2007-05-28
You can search sourceforge by language. But first complete at least 1-2-3 months of pythonchallenge, so you will have some skills tucked in your belt, to avoid asking too stupid questions. :-)

Many projects set aside simple tasks/bugs especially for new developers to train on, and offer mentoring. Developers are scarce resource, and every project competes for people with attitude: if you have proper attitude, gaining the skillset is just matter of time!

You can also train skills in Pygames, there is lot of sorce code to learn from and collaborate.

One of nice projects in Python is SPE - Stani's Python Editor, developed in Ubuntu now.
