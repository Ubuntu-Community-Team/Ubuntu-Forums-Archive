In my experience with it, you can install a lot of great extensions into the Lite version to get features similar to the Pro ones or even better than them. 

Of its kind, it's possibly the best.

Now, I'm not sure how much programming do such programs teach. IMO, GM suffers the same issue as Visual Basic. The best GM games I've seen are made by people that already knew how to program and coded the whole logic in GML scripts and used the GUI tools just for that... But you'll see that GM encourages beginners to start coding everything using the GUI tools, mixing interface and implementation, for the sake of "simplicity". In other words, GM isn't a bad idea, but it stresses the wrong features.
