### Post by jeffathehutt on 2009-08-08
All mediawiki requires is php and mysql support.  If you have access to both, you should be able to install mediawiki.  Check with your university to see what features are available to you.

There are many alternatives to mediawiki though.  Chech out this list [http://en.wikipedia.org/wiki/Comparison_of_wiki_software]("http://en.wikipedia.org/wiki/Comparison_of_wiki_software").  That list will show you which software requires which features, so once you know what your university supports, you can make a decision.  Some popular alternatives:

moinmoin
dokuwiki
pmwiki
twiki

and many others. :)
