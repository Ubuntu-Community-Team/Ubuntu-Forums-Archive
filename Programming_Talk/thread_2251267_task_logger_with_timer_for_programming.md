### Post by tgalati4 on 2014-11-03
I have always used *gnotime*.  At the end of the week, just generate an html report and print it out or send an email with your hour breakdown.  It has a small learning curve, but once you figure it out, it works well.  If you work on several computers (desktops/laptops, etc), you can run gnotime using a graphical ssh session to keep track on multiple devices on one report.

There may be an Android task timer and use your phone to track your programming activity.
