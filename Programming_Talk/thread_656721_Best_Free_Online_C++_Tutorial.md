### Post by samjh on 2008-01-03
For beginners:
[http://www.cplusplus.com](http://www.cplusplus.com)

For experienced C/C#/Java/Perl programmers:
[http://www.icce.rug.nl/documents/cplusplus/](http://www.icce.rug.nl/documents/cplusplus/)

Look up standard C and C++ classes, functions, and methods:
[http://www.cppreference.com/](http://www.cppreference.com/)

Useful collection of tips:
[http://www.parashift.com/c%2B%2B-faq-lite/](http://www.parashift.com/c%2B%2B-faq-lite/)

*The* C++ standard for pedants:
[http://www.open-std.org/jtc1/sc22/wg21/](http://www.open-std.org/jtc1/sc22/wg21/)

Last but not least, the C++ Wikibook:
[http://en.wikibooks.org/wiki/C%2B%2B](http://en.wikibooks.org/wiki/C%2B%2B)
