### Post by ofnuts on 2014-03-15
Never tried it but in konsole you can define profiles that can log you on another host. They also allow you to set what is displayed in the window tab and this included a remote host name. See [http://docs.kde.org/stable/en/applications/konsole/konsole.pdf](http://docs.kde.org/stable/en/applications/konsole/konsole.pdf)

Ofnuts is right. You can set the tab title to "%w" to display the titles you set in your application. [http://ubuntuforums.org/showthread.php?t=2084746](http://ubuntuforums.org/showthread.php?t=2084746)
