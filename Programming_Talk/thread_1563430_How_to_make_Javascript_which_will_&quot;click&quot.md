### Post by James78 on 2010-08-29
I hope you meant Javascript and not Java, because if you meant Java, you just changed the thread topic. You can redirect to another page using Javascript like so. This is the most basic way that I'm giving, you can get much more creative by doing things like making a function out of it, etc.

```
<script type="text/javascript">
window.location = "http://www.google.com/"
</script>
```
