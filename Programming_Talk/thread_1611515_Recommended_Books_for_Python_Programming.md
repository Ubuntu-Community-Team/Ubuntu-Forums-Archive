### Post by OgreProgrammer on 2010-11-02
An introduction to python by guido van rossum (who created python) is a good one. Cant get much more official than that. 

Its a word for word copy of the online version at the python site, but the nice thing is the purchase supports the python foundation. Plus I like having paper books.

It doesnt have a project associated with it, but most everything is shown in small examples... one to ten lines of code. Its excellent.

### Post by raydeen on 2010-11-02
I recently picked up Rapid GUI Programming with Python and QT.

[http://www.qtrac.eu/pyqtbook.html](http://www.qtrac.eu/pyqtbook.html)

Picked up a hardcover version from textbooks.com for $8! (found it through cheapbooks.com). The first section gives a quick but pretty thorough intro to Python and then moves on to developing GUI apps with QT.

For more indepth reading I'd recommend Learning Python by Mark Lutz.

[http://books.google.com/books?id=1HxWGezDZcgC&printsec=frontcover&dq=learning+python&hl=en&ei=90LQTOr5AoG88gb506TTBw&sa=X&oi=book_result&ct=book-preview-link&resnum=1&ved=0CDgQuwUwAA#v=onepage&q&f=false](http://books.google.com/books?id=1HxWGezDZcgC&printsec=frontcover&dq=learning+python&hl=en&ei=90LQTOr5AoG88gb506TTBw&sa=X&oi=book_result&ct=book-preview-link&resnum=1&ved=0CDgQuwUwAA#v=onepage&q&f=false)

It's really big and verbose but was what I needed to initially get my head into the language. Some people don't like his style but it works for me. The Google preview above should help you decide. It's also available for cheap on textbooks.com.
