### Post by river22_34 on 2007-09-13
I think that JMX technology will have a significant impact on the proliferation of distributed computing. Java technology is already leading the way in distributed computing. One of the greatest remaining challenges, however, is the inherent complexity of distributed computing. JMX technology addresses this particular issue head on. And because it is an extension of the Java platform, the tools needed to address these problems will be readily available to developers in this field.
