### Post by simeon87 on 2009-03-06
Examples can be found here: [http://lolcode.com/contributions/operators](http://lolcode.com/contributions/operators)

```
UP VAR!!7
```

Means: increment VAR by 7.

```
I HAS A num ITZ LEFTOVER 3!!2
```

LEFTOVER means modulo operator (see same page) so the above means: declare a variable num with value 3 mod 2.
