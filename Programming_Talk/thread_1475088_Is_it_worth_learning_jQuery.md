### Post by myrtle1908 on 2010-05-06
Learn JavaScript first, then jQuery.  However, jQuery will only get you so far.  It is useful for adding "bells and whistles" and "shinies" to web pages/sites etc.  If you want to create web applications then you should look to YUI or ExtJS.
