### Post by DolphinWeb on 2006-06-27
Well...this is the first of my series of python scripts that'll create PyGTK guis for ya'...

Menu_Maker.py is a python script, designed to be run from the command-line. You give it a few details, such as file name, the variable to use, and such, and it'll do 3 things: create the file contents, print it out the contents to the command-line, and create a file specified by you with everything filled in. After, you can run it and have fun! Comes with a default action handler. Doesn't support icons and the like. Nor re-arragement of the menus. Once I get some feedback, I'll make changes.
