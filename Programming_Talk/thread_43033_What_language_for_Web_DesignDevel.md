[QUOTE=hardwarder]I have some books on PHP, Javascript, etc...

First thing you need for a website is content.

Second thing you need it to work out what you want to have it look like.

Once you have those two, I'd suggest looking at cascading style sheets for formatting HTML.  The book "Eric Meyer on CSS" is really good for this as everything is done with examples to show you how to do stuff.

For starting out, I would definately go for an example driven book.  You can then take those examples and us it with your content.

HTH, Tim
