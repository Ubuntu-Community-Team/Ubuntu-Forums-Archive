### Post by vemon on 2007-09-24
Hi!

I just just finished my little project which I started to create LASH ([http://www.nongnu.org/lash/](http://www.nongnu.org/lash/)) support for Hydrogen. The patch is created against Hydrogen 0.9.3 since it's the only stable version at the moment. All development and testing has been done in Ubuntu Feisty.

Get the LASH-patched Hydrogen deb (Feisty) package from: [http://www.cs.helsinki.fi/u/jjsipari/public/hydrogen_0.9.3-2_i386.deb](http://www.cs.helsinki.fi/u/jjsipari/public/hydrogen_0.9.3-2_i386.deb)
The package is built using debian/ubuntu tools (dpkg-buildpackage etc.), not checkinstall :)

The source patch can be downloaded from hydrogen forums:
[http://www.hydrogen-music.org/forum/?action=show_thread&thread=657&page=1#2516](http://www.hydrogen-music.org/forum/?action=show_thread&thread=657&page=1#2516)
