### Post by kalaharix on 2009-05-12
Hi

I think you are right. Gambas does not install to the menu system from the repositories. It does though when you compile the latest version from the Gambas site.

Probably the simplest option is to create a desktop launcher (right-click on the desktop. The program is gambas2). Search first on gambas.png and copy it to a convenient location so that you can change that funny springy board thingy at the same time.

Once you have created the launcher you can (also) drag it to the top panel.
