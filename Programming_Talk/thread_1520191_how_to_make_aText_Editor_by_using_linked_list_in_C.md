### Post by Zugzwang on 2010-06-29
Your request seems to fall under the homework policy, so I'll just give you some general advice:

[LIST]
[*]Try to collect a list of functions your program must have.
[*]Look up details about linked list. What could the linked list be used for in your case?
[*]Try to devise suitable data structures for your task; check if the data structures allow to efficiently perform all necessary operations that you identified beforehand. If not, redesign the data structures such that they do.
[*]Then familiarise yourself with the "development" environment (programming language, GUI widget kit, etc.)
[*]Read examples that are close to what you want to do.
[*]Develop a program structure for your editor
[*]Implement all parts that you idenfitied in your program structure
[*]While developing, constantly write tests for the parts you have written.
[*]Do some alpha/beta testing of your application. Fix bugs.
[/LIST]
