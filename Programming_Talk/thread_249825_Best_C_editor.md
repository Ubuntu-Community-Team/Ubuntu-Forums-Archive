### Post by rplantz on 2006-09-04
Try several and see what you like. I constantly move between various editors. I just get bored and want to use something else for a while.

My experience goes back to the days of punch cards. Make a typo and throw the card away!

When the Mac came out, I immediately bought one because of the gui/mouse user interface. I continued to use a Mac almost exclusively until a couple of years ago.

Although I use vim and emacs from time to time, I find myself always returning to an editor that's based on a gui. Since I'm running gnome, I use gedit. I've also tried kate and like it.

Yes, I know that there are gui interfaces for vim and emacs, but I don't find them as nicely integrated with the editor.
