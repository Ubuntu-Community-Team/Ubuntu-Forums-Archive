### Post by CptPicard on 2008-04-19
Drupal is *otherwise* a great CMS but it's always been just totally appalling how tough it is to add pictures to stories. It always needs some extra hacking on top of the stock version or otherwise you just need to add the picture as a file attachment and manually add the HTML image link into the raw HTML version of your story... there are image plugins also, and the TinyMCE editor has some image attaching support. 

I am not quite sure what it is that you mean though ... theming is a somewhat different issue. :) Drupal's themes are just a collection of templates that let you change the "skin" of the CMS.
