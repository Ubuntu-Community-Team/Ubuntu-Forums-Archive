### Post by kalaharix on 2010-02-19
Hi,

Gambas really is good and rock solid. It is different from VB and has a learning curve.

I think the repository version is incomplete. It is safer to use 'sudo apt-get install gambas2' from the command line (I can't think why). Or install from repository and then install the missing bits by doing a gambas search in Synaptic.

The Ubuntu versions of Gambas are always old (unlike Mandriva and others which manage to keep things more up-to-date). Once you get the hang of the  old repository version, it is no big deal to compile the latest Gambas on Ubuntu. Instructions on the Gambas website,
