### Post by pmasiar on 2007-07-05
SPE - Stani's Python Editor is good IDE. Stani develops it in Ubuntu... Eric is another IDE. For really simple edits, even SciTE if fine, or any other text editor which does not mess with tabs.

Because in Python whitespace is significant, you don't want to mix tabs and spaces. Standard is, replace tab with 4 spaces. Any editor capable that should be fine for basic editing.
