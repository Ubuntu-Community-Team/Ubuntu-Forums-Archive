Uh... the usual way is to install python-mode from the repos (that installs pymacs automatically), but you don't have admin rights.

A first option that comes to mind is that you ask the admin to install it. Tell him your reasons, ask him politely and he would probably install it. I can't see any reasonable risk associated with installing those packages.

If not, there are ways to install everything manually. Look here: [http://www.emacswiki.org/emacs/PythonMode](http://www.emacswiki.org/emacs/PythonMode)
