### Post by Orion8 on 2010-08-13
Hi --

I found your post wondering the same thing.   A bit more looking and I found this page linked from the Geany theme page:

[http://geany.pdarko.co.cc/filetypes/](http://geany.pdarko.co.cc/filetypes/)

This is an online style generator for geany -- you don't have to install any programs or much around marking stuff up.  I needed a more attractive color-scheme for C++ and LaTeX.  It worked great.  You will have to imitate Kate's color scheme for python yourself but it should not be too hard.

That said, I likely will still use Kate, but I like to keep my options open!
