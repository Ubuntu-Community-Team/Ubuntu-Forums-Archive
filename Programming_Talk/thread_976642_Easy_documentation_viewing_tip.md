Ok so I was browsing information about using apache2 on ubuntu and I found out that the apache2 package by default hosts your documentation directory. This allows really easy access to the documentation of your packages without having to deal with switching between your file manager and browser. 

anyway to use it just install the apache2 package and whatever [library]-doc packages and then direct your browser to 

[http://localhost/doc/](http://localhost/doc/)

just wanted to share this little tip
