### Post by pmasiar on 2008-06-09
I don't think you will find someone to start project for you: people who are smart enough to do it, are too busy hacking on their own projects. So, as in life in general: if you want something to get done, do it yourself :-)

Learn programming. Start with simple language like Python. It has excellent libraries for many tasks, including something like you want, as is easy to learn.

Start your project. You will have hard time to get  anyone interested before you have at least crude usable code - it is not that hard and frankly rather than trying to decipher some non-running code, any competent hacker would prefer starting own code - if there are bugs, at least they are my own bugs :-)

Even better, try if there is such project already started, and join it, if possible at all.
