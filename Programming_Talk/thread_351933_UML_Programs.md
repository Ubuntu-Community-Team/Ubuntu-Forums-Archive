### Post by Greykrrr on 2007-02-22
> **kane77 said:**
> isn't starUML open-source?? that should mean that it could be compiled in linux right? but I havent found source download...

Not really. Open source really just refers to its licens. It is release under the GPL (I think) but it has been written for windows. It uses the window GUI toolkit the same way as a program can use GTK, QT or whatever. Besides it probably as some language specific things that will only work on windows.

On its webpage it says that a long term goal is a linux port but right now there isn't one and you need windows or wine (probably, haven't tried) to run it.
