### Post by Tombgeek on 2012-06-17
Personally, I started programming with Delphi. There is an open-source IDE called Lazarus (which uses the Free Pascal language), which does a really good job of recreating a Delphi-like programming experience (assuming you're not going to work with SQL. The last version I used didn't include SQL components).
Java and C++ are very complicated languages and not recommended for beginner programmers. Free Pascal is very easy to learn (and the syntax is more or less the same as Delphi's Turbo Pascal) and it will provide the guidelines and logical skills needed for more complex languages (especially with object-orientated programming).
