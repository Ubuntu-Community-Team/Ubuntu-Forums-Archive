### Post by Orestes on 2006-09-23
Java shares a similar syntax to C, so if you've programmed in C before, you'll find Java a piece of cake.

One icky point about Java though - it's non-free. It costs nothing, sure, but Sun won't release the source code (yet), so, if you are a GNU/RMS fan, I wouldn't touch it with a long stick.

If you fancy something different altogether, there is the most excellent Python (which is just plain *fun* to code in, dammit!) and the purely object oriented and much lauded Ruby.

For any of the above languages, there are loads of web tutorials and resources knocking about. Why not try all three?
