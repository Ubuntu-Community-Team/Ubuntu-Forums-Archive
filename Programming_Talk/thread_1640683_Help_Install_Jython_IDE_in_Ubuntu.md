### Post by Wistful on 2010-12-10
1) [Install Jython ]("http://wiki.python.org/jython/InstallationInstructions")

2) [Install PyDev plugin for Eclipse IDE]("http://www.jython.org/jythonbook/en/1.0/JythonIDE.html") and configure it for Jython or use [Jython in Netbeans]("http://netbeans.org/kb/docs/python/jython-quickstart.html")

Next time before you ask a question read [this]("http://www.catb.org/~esr/faqs/smart-questions.html#before") and try [this ]("http://tinyurl.com/34vuycq")

P.S. Installing from Synaptic or Software Center is the same thing just that Software Center has a nicer GUI for installing apps.
