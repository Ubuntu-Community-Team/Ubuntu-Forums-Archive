I'm glad to announce I wrote **Antenna DTV**, which is a free and open-source** dvb-t **(terrestrial digital tv) **tuner** for linux desktops. Antenna is a full featured tool to see what's going on in the sky. It enables you to scan, watch and record television, radio and services broadcasted on air.

It's written in **python**, using **libglade** and gtk and has been designed with gnome desktop in mind (the aim is to build something that looks solid like programs like rhythmbox are). It relies deeply on shell command, don't expect complex object programming here (kinda don't like it)! I use **ubuntu** 10.04, and this should run fine quite on a lot of different ubuntu versions!

Antenna DTV is intended to take over other tools such as Me TV, kaffeine, totem-dvb, with an approach more signal based, than tv itself focused. That's why one of the most notable feature (you'll see!) is that **big signal strenght indicator** with an innovative "quality level" as well!

Deb packages are already available.

All you need is a linux desktop and a digital tv card (such as an usb stick, I do personally use a WinTV-HVR).

Anyone interested? Tried it out? Found a bug? Got an idea? Post about it!
