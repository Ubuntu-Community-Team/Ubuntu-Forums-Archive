### Post by punkrockguy318 on 2007-05-15
Hey all!  I was bored in my C++ class in school one day, so I decided to write a space invadors clone.  In Python (ironically) :popcorn: .

Besides messing around with Multimedia Fusion/TGF when I was 11 or 12, this is my very first attempt at a graphical game.  It's very UNoriginal, but I just wanted something to work my skills.

It's written in python and uses pygame for the graphics/sound/input.  I wrote it to be pretty crossplatform, and it should work almost everywhere.

I wrote invade spacers to be pretty extensible; and I believe that is pretty true.  Creating a new enemy is as simple as drawing up a simple 32x32 bitmap and writing up a class definition which can be as little as five or ten lines.

I'm pretty excited that I finally have a beta together.  I've been working on it during class for the last month or so, and I'd like some advice and critique.
