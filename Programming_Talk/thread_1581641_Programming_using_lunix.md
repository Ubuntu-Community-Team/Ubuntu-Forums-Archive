### Post by worksofcraft on 2010-09-25
A good way to start is to set your self some project that you will enjoy doing and not make it too hard either... like don't go writing your own MORPG from scratch or you will likely be disappointed.

Anyway so once you have something you feel is achievable pick just one small aspect of it... like... pop a up window that says "Hello World!" or read from a file etc.

Then you make a little program to do just that one bit.
Repeat this for each and every of the bits you identify in your project and when you can make all of them work individually it's time to make them work together
:guitar:

Good luck and do come back tell us what project you set yourself ;)
