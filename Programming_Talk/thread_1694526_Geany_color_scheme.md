### Post by stchman on 2011-05-05
If you are referring to things like syntax highlighting for various programming languages then yes.

Go to:

Tools ---> Configuration Files ---> filetypes.common

This will allow you to set the colors for things like keywords, comments, preprocessor statements, etc.  You will need to use the hex color codes.

Hope this helps.
