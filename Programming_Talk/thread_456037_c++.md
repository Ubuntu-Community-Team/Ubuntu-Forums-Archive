### Post by harun on 2007-05-27
In System -> Administration -> Synaptic Package Manager

Search for and install any of these:

Anjuta ([http://anjuta.sourceforge.net/](http://anjuta.sourceforge.net/))
Code::Blocks ([http://www.codeblocks.org/](http://www.codeblocks.org/))
KDevelop ([http://www.kdevelop.org/](http://www.kdevelop.org/))

Those are probably more of what you are used to. Code::Blocks and Anjuta don't require a lot of the extra KDE packages KDevelop would, so I would start with those and see how you like them.

Without understanding make, the GNU autotools and how to use pkg-config and the rest of the tools GNU/Linux uses programming in gedit is going to be a bit rough for you. However learning them isn't as bad as you think. Here is a nice walk through:

[http://www.st-andrews.ac.uk/~iam/docs/tutorial.html#SEC7](http://www.st-andrews.ac.uk/~iam/docs/tutorial.html#SEC7)
