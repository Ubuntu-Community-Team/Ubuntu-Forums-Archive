### Post by knallkopf on 2008-10-28
To look up perl documentation from within Vim, I'd recommend manpageview ([http://www.vim.org/scripts/script.php?script_id=489](http://www.vim.org/scripts/script.php?script_id=489)). After you install it, you can look up Module Docs by pressing 'K' in normal mode or with the ':Man Some::Module' command.
