### Post by Valsodar on 2008-08-06
As a self learned programmer (for 8 years now) I suggest one of these alternatives.

1. Start with html (I use quanta and its easy it has nice code completion)
   Move to PHP (Quanta + Apache on local server)
   Move to C
   Move to C++ (After that its all easy)

2. Start with C
   Move to C++
   Learn html
   Learn PHP

I actually did path 2. As for C/C++ compilers I use Code Block (its a very very good one) and it is very easy to install because its cross platform (you will need wxWidgets).

If you want lesson plan you can message me and I might be able to devise something. (I am looking for an excuse to write a nice tutorial for C|PHP)
