### Post by pbrane on 2010-03-13
I don't know pygtk either but I do program in gtk/gtkmm and it looks like from the pygtk docs you need to define a label then call the set_text method.

```

label = gtk.Label()
label.set_text("whatever")

```

or you can just set the text when you create the label.
```

label = gtk.Label("whatever")

```

here is one source of documentation:
[http://www.moeraki.com/pygtktutorial/pygtk2reference/index.html]("http://www.moeraki.com/pygtktutorial/pygtk2reference/index.html")
