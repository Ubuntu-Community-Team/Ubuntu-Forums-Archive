### Post by agnes on 2010-04-20
Doesn't start on typing hello, and it is a collection of scripts, not 1 script.
More importantly, the stuff in "autin" is redundant and doesn't work. 
I assume the scripts are meant as bash scripts, for the bash shell, since that shell is default in Ubuntu. 
You don't need to install bash script to run it in a bash shell.
You only have ./configure; make; make install thing, when installing programs from source code - the kind of source code that the shell can't natively understand (like C).
Your friend would certainly benefit from a bash tutorial :) Good luck to him.
