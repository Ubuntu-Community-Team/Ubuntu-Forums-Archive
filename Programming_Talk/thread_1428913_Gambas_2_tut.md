### Post by kalaharix on 2010-03-15
Hi

It depends where you are starting from: if you have some programming experience in other languages there is a tutorial 'Gambas 2.8 on Ubuntu 9.04' on [www.kalaharix.wordpress.com](www.kalaharix.wordpress.com). It's not meant for somebody who is new to programming though. It does, at least take you through the process of creating a new Gambas project.

Remember that the repository versions of Gambas2 are incomplete. You have to go into System, Administration, Synaptic. Search on Gambas and install the components that have been left out of the repository install.
