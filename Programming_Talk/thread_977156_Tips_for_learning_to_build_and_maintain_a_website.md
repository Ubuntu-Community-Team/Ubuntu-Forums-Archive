### Post by ad_267 on 2008-11-10
If you don't want to learn html then try Kompozer from the repositories.

If you want to learn some html and css then look for some good tutorials on the internet (a good place to start is [w3schools]("http://www.w3schools.com/")), and check out how other people put their designs together at [http://www.opendesigns.org/](http://www.opendesigns.org/). Use standards compliant code (xhtml and css) and check your designs on different browsers and different operating systems, they won't all render the same site exactly the same.

If you want it might be a good idea to try out a content management system like [Silverstripe]("http://www.silverstripe.com/") or [Wordpress]("http://wordpress.org/"). You can install Apache + PHP + MySQL on Ubuntu to try these out on your own computer. It's good to have some idea about html and css as well as an understanding of what php and MySQL are before getting into something like that.

### Post by ankursethi on 2008-11-10
For web design, you've got two options.

**Do Everything by Hand**
Learn XHTML (for content), CSS (for presentation) and JavaScript (for logic), then write your website by hand like most professional web developers do. The good thing about this approach is that learning the above 3 technologies is *very easy* and you can make many mistakes along the way (and learn from them) without really hurting anybody. Moreover, the skills you learn while building your website can be easily carried over to other tech-related fields. For instance, JavaScript is a full featured programming language and moving from JavaScript to something like Python will be trivial.

**Use Hand-Holding Tools**
You can always use tool like Kompozer to quickly put together a website and be done with it. However, you learn nothing of lasting value through this approach. If you just want to put together a site to get the scholarship money ASAP, use a WYSIWYG tool, but make sure you learn the ropes of XHTML, CSS and JavaScript later on.

BTW, you can also use a content management system to build your site. A CMS is the easiest and quickest way to put together a dynamic website. Google for Drupal, Joomla or Wordpress to know more.
