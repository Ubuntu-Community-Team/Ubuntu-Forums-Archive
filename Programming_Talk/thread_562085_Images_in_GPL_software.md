### Post by DMills on 2007-09-29
That does not help if you are wanting to USE an image supplied as part of a GPL package in a non GPL application, for that you would need to contact the author of the image and get them to license it to you.

Using it in a GPL program of your own devising is, I think, fine. 

For some image types the distinction between image and code is vague anyway, XPM files for example are really C program fragments....
