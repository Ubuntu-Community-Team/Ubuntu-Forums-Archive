### Post by Dhiraj Thakur(Invincible) on 2011-06-19
> **AlexC_ said:**
> Guys, you can't just "learn" JavaScript and create an animated drop down menu.

Dhiraj, you first need to learn HTML. This is the markup language that will display your data. You then use CSS to style the HTML, this is where you add in your nice background images, borders, font colors etc etc.

Once you understand well how both HTML and CSS works, you can then start learning JavaScript to make your website more dynamic. It will allow you to interact with the DOM. This JavaScript runs in the clients browser.

You can then have a server-side language, such as PHP, to create more dynamic web applications. All the PHP script will mostly do is output the HTML.

So for this menu you want, you need to learn HTML, CSS and potentially JavaScript depending on what sort of effect you want.
