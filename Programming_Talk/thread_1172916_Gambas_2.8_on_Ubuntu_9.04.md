### Post by kalaharix on 2009-05-29
Hi,

Just a note to those who try to install Gambas 2.8 from Add/Remove Applications on Ubuntu 9.04. The installation package is currently incomplete and not all required components are automatically installed.

After you have installed from Add/Remove Applications: go into Synaptic (system, administration, synaptic), do a search on Gambas and install all the related packages.

The current stable version of Gambas2 is 2.13.1 and can be installed on UB904 from source. 2.8 is perfectly usable though.

There is also a development version of Gambas3 (currently version 2.99) which has to be installed from source.

Gambas rocks.
