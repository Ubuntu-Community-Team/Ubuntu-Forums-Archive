### Post by kahumba on 2010-06-28
IDEs become useful when:
1) You need refactoring (not just replace-all) in the whole project
2) Your project is big
3) You need someone to clean up your code, like telling you that a variable is not being used etc.
4) A sea of other stuff like plugins (real plugins, not amateurish scripts) which is not worth the time mentioning, you get it yourself when you start developing medium and large apps.
Yes, NetBeans is bloated but it's a lot smarter and more intuitive than Eclipse which is is worse at anything except maybe C/C++ support.
