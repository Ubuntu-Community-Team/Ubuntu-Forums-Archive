### Post by Reiger on 2009-07-27
What is -ultimately- the difference between "scripting" and "programming", "script" and "program"?

EDIT: As for the "why is Java strongly typed" ... what have type systems got to do with the previous question? As for Java's supposed strong-typed-ness: that really is a matter of perception, where you draw the distinction. For me Java isn't really strongly typed, just like C and C++ aren't strongly typed either. They have types, yes. They are explicit about 'em, yes. But it isn't strong in the sense of strict rules (consider the free-form conversion between bytes, integers, and characters); certainly nowhere as strongly-typed as say Haskell.
