### Post by jespdj on 2009-03-16
API means [Application Programming Interface](http://en.wikipedia.org/wiki/Application_programming_interface).

Many libraries and other pieces of software have an API. Programmers use the API of a library or other piece of software to write programs that use the library or other piece of software.

Google has APIs for many of its services, including the search service and for Google Maps. You can use these APIs from, for example, JavaScript in a web page.
