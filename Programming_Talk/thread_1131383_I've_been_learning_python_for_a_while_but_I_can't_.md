### Post by Sinkingships7 on 2009-04-20
Write a script that makes mirror backups of important files on your hard drive. If something is missing from the source, delete it from the destination. If something new is in the source, make sure you copy it over to the destination. Basically, just make sure that every time you run the script, the source mirrors the destination. 

Try writing a deck class which represents a deck of cards. Include methods to shuffle the deck, draw a card (randomly or from the top or bottom of the stack), and replace a card. Make it exception safe and dynamic, so it can be used in future projects such as...

Simulate the blackjack card game, human vs AI. Write simple rules for the AI to follow. Include a betting feature. Make sure to emulate splits and insurance. If you're not familiar, more info can be found on Wikipedia.

Write any other card games which are interesting to you. 

Write a small command line music player. This will teach you the basics of using libraries. 

Write something more complex using a third-party library, such as the 2D game Pong, using PyGame. This will give you a more in-depth experience with external libraries, as well as more experience with testing/debugging and algorithms. If this proves not much of a challenge, try adding options to change the color of the "paddles" and ball. Change difficulty level. 

Afterward, learn C. Practice searching and sorting algorithms. Learn all about Big-O notation. Learn how to manually control memory. Learn to appreciate Python's built-in string handling abilities and hiss at the thought of C input.
