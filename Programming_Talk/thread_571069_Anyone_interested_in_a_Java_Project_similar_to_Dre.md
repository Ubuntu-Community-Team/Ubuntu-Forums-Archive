### Post by brennydoogles on 2007-10-08
Hello, I am looking for someone who would be willing to help with a project very similar to Dreamweaver called Aptana. Aptana is a promising web IDE written in Java, but it is missing a very few features that would make it an ideal IDE. It started out as an eclipse plugin (and can still be used that way), and is almost mature enough to be a professional tool. The functionality that I am most interested in getting developed is the ability to refresh the design in html documents in a manner very similar to Dreamweaver's split view (right now you can preview your document in a split view, but you have to save to refresh the view, which is a pain while editing an external css file). Anyone interested post here, and maybe we can get a discussion going on how to get this working!!

p.s.-- [Here]("http://www.aptana.com/forums/viewtopic.php?t=608&unwatch=topic") is a link to the related thread on aptana's forums, and [here]("http://www.aptana.com/trac/ticket/1143") is a link to the feature request itself.
