Gidday all.

This is the first of a series of tutorials that cover OpenGL programming in C++.

This first tutorial covers creating an OpenGL window/fullscreen mode, setting up OpenGL, drawing a simple shape, event handling, rotation and translation!  Quite a bit of stuff there, but not too much code (130 lines- with comments).

It is recommended you have at least some experience with C or C++, but doesn't use anything too tricky and should be quite easy to follow.

This is pretty much the base code we will be working off, adding stuff to it as we go along, so spend a bit of time experimenting and getting familiar with it.  Future tutorials will along adding little bits to it (probably approx. 10 lines at a time), and won't be quite so daunting!

The executable is included, and should work if you have the required libraries, or you can build it yourself with "make"

Future tutorials will cover: textures (next), text, frame-rate control, transparency, reflections, particle systems, special effects, and anything else people want to learn about.

Any feedback is welcome!  (hope the code is ok- was a bit rushed, but does compile!)
