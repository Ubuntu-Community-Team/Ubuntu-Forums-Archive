### Post by lykeion on 2010-11-13
In Java you can use jogl as OpenGL bindings. Install the **libjogl-java** package in Ubuntu. 
There's example code and tutorials all around the web, like in this wikipedia entry: [http://en.wikipedia.org/wiki/Java_OpenGL](http://en.wikipedia.org/wiki/Java_OpenGL)
And some demos of jogl in action here: [https://jogl-demos.dev.java.net/](https://jogl-demos.dev.java.net/)
