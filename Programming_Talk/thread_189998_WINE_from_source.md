### Post by meng on 2006-06-05
The basics of compiling from source are the following three commands (run from the directory you've unarchived the source into):
./configure
make
sudo make install

To unarchive the stuff in the first place
tar xvf <name of archive>.tar.gz

More info here
[http://www.psychocats.net/linux/installingsoftware.php](http://www.psychocats.net/linux/installingsoftware.php)
