### Post by Jucato on 2008-10-17
Based on my compiled Qt 4, there isn't any man page for Qt API (unlike C or C++ API). The main source of offline documentation is the Qt Assistant program. Online reference would be [http://doc.trolltech.com](http://doc.trolltech.com)

### Post by snova on 2008-10-17
There are no manpages for Qt 4. But if you install the "qt4-doc" package, you can point your browser to /usr/share/qt4/doc/html/index.html for the full reference/tutorial/everything else.
