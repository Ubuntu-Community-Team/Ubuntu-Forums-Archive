### Post by ablegreen on 2008-01-05
I program in ASP.net, but I don't really like it. I love the Visual Studio IDE but programming in it and in ASP.net felt kinda bloated (I don't really know how to describe it). I felt lost. I just didn't feel comfortable for some reason.

Then I heard great things about python. People said it was simple and easy but mature, and it made programming a joy. I decided to give it a shot, downloaded "A Byte of Python" and learned the language within weeks. It was to easy to learn to be true, but it's true.

Now that I've learned python, I looked around for a web framework. Again people recommended Django, so I went ahead with it. I couldn't believe it. It's wonderful and it gets you programming sites extremely quick. Reasons why you should pick Django:

1) Documentation. Django's documentation on their site is so well written and easy to read that I never had a thought of "Wtf?". Also, their awesome book "The Django Book" is wonderful as well, and it tells you everything, from scratch to deploying.

2) Everything is consistent. There are no "magic" that leaves you clueless and puzzling of what's happening behind the scenes. Everything seems to make easy sense.

3) There's a lot of integration. Django comes with a database API (or mapper or whatever you call it) that works with most databases. It even includes the admin interface for your site, among many others.

ASP.net may be more powerful and of course faster than Django but Django is just a pleasure to code in. If you are making a personal site, go with Django and Python. ASP.net is more suited to enterprise and big sites I think, but of course Django can do that.

If you already know VB or C++, you will Python in less than a day.
