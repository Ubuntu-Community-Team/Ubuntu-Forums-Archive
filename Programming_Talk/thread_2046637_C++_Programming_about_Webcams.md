If you are looking for facial recognision and using webcams etc I think OpenCV is the right place to be looking.  OpenCV is an API which is cross platform, so not being used to Ubuntu should not really matter.

Are you new to programming?  Is that why you are thinking this is a bit complicated?  I don't want to sound too harsh, but facial recognision is a complex subject, so it isn't really something you can just jump into.

Here is a link for some documentation for facial recognition with video:
[http://docs.opencv.org/modules/contrib/doc/facerec/tutorial/facerec_video_recognition.html?highlight=webcam](http://docs.opencv.org/modules/contrib/doc/facerec/tutorial/facerec_video_recognition.html?highlight=webcam)
