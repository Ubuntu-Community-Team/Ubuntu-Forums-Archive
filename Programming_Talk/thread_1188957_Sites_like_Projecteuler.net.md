### Post by gnuman on 2009-06-16
If you're thinking of something that has a lot of small programming challenges, but not strictly centered on math-type problems, something like [JavaBat]("http://www.javabat.com/") is an example for Java programming problems.

I was recently involved with an attempt to make a [similar site]("http://pywhip.appspot.com/") for Python problems, but the team is pretty small and it's a lot of work.  Also, things that are appropriate Java exercises on JavaBat are sometimes almost laughable as Python problems, so you almost have to start from scratch to think of what categories of problems, and so on.  

As you said, a team is more appropriate for a task like that.  Just work on the idea a bit more and maybe have a few examples or guidelines so we can know more about your goals.
