### Post by geirha on 2008-01-13
> **Vinno said:**
> geirha: Yeah, thats what i thought but doing the database backup pauses for few seconds and still doesnt output the database echo still.

I don't quite follow. The database dump I can imagine takes a bit of time, so the script waits till that is done before proceeding to the rest of the script, which probably completes in less than a second. Do you want the commands to go in parallel or something?

BTW, you can add -v to mysqldump too if you want to display its progress. It's printed to stderr, so it won't be redirected to your backup-file
