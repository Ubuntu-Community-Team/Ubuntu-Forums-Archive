### Post by Mr. C. on 2007-07-14
There are so many different languages today, that such a broad question has no answers.  And each person is going to give you their own personal favorite, usually substantiated only by opinion.  Its like asking "What car should I learn to drive in".  The make/model matters less than the basic concepts such as rules of the road, accelerator pedal, break pedal, gear shift, key, mirrors, etc.

The same goes for programming - it is important to focus on fundamental design, algorithms, structure, problem analysis, etc. Which language you use depends on many factors.  Each language has strengths and weaknesses, and today, there are a ridiculous number of new languages, each promulgated by an NIH attitude.  Don't get bogged down in worrying about the language choice. Select one, and start learning how to use it to solve useful problems. 

MrC
