### Post by scribles on 2007-06-12
Eclipse PHP Development Tools (PDT) and phpeclipse ([www.phpeclipse.de](www.phpeclipse.de)) are not the same thing, but they both do essentially the same thing (PHP Editor / Debugger for Eclipse).

Phpeclipse is a completely open project that has been around for quit some time. Development for phpeclipse is often slow. I've had problems with it not working with new version of PHP and activity on the site is often nonexistent.

PDT is a project recently created by Zend. Its still early in development and as such has bugs and lacks some features. Documentation for it is completely nonexistent. I have found a tutorial on using PDT with XDebug (instead of the Zend debugger) which has some good information of PDT in general. The tutorial is included with the XDebug plugins here [https://bugs.eclipse.org/bugs/show_bug.cgi?id=169408]("https://bugs.eclipse.org/bugs/show_bug.cgi?id=169408")
