### Post by prensing on 2007-02-19
My company is doing most of its programming in C# on Windows. I am working on Linux/Mono and it is pretty good about compatibility. The low level code is quite good. Unfortunately, the GUI libraries (Windows.Forms) are still catching up, especially the complicated ones like the GridControl. So, if your code only uses 1.1 C# features or the less complicated 2.0 features you should be OK.

In general, I am impressed with Mono and each version has been a noticable improvement.
