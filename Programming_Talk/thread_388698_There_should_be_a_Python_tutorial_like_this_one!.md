### Post by pmasiar on 2007-03-19
It was dome for Python too, and is linked in sticky for this forum. I admit that Ruby guys have nicer design. :-(

Running Python/Ruby via browser has couple issues: 
(1) if many people starts hitting the site, cam be a lot of abuse..
(2) some statements are not safe to execute and need be filtered out Like writing files.
(3) you probably want to write little scripts and test them - so you want to save them on your own computer

Python needs less such website than Ruby because
(1) is installed on every Linux box,  just run it
(2) has command shell. where you can run all commands without restrictions
(3) even for windows, it is easy to install - or get Ubuntu LiveCD abd run it from there

But if anyone wants to try Python in browser, it is available.

[http://learnpython.pbwiki.com/HowToStart](http://learnpython.pbwiki.com/HowToStart)
