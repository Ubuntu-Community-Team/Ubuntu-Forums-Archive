### Post by namegame on 2009-02-11
> **spupy said:**
> Developing for iPhone/iPod Touch is done with Xcode on Mac OS X. And you need to pay some tax, i think.

That is mostly correct. The language for the iPhone is Objective C. Anyone can freely develop applications and run them in an emulator environment, using the iphone SDK and simulator. However, to actually test your applications on physical hardware you must pay for the iPhone Developer Program, which costs  from $99 (US) for a standard edition, to $299 for the Enterprise Edition.

Unfortunately, you must own an Apple computer to develop for the iPhone.


I only know all this because I'm participating in an undergraduate research class in which we develop pedagogical applications for either the iPhone or Google Android phone.

I've chosen the Google Android. Developing applications for this phone is completely free. It is programmed in Java and the SDK heavily utilizes the capabilities of Eclipse. Obviously, since it is Java/Eclipse, you can use any OS as your development environment.
