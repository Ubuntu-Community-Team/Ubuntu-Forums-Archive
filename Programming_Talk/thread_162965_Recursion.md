### Post by treak007 on 2006-05-07
Try drawing diagrams with psudeocode. First think of what tasks need to be repeated. label them, and draw arrows to help you see how the program will run, then go ahead and insert real code into your diagram.  Usually helps the people I tutor in computer science, plus drawing diagrams makes your job coding easier.
