### Post by Ferrat on 2008-10-22
Best way is either if you have enough knowhow to help on a bigger project or take a project that includes all that you have done and adds some. 

For ex. making a small game isn't that hard, simple apps? same there but make a bigger project of it. 

for ex. 
if you've made a small wargame with topview 2D sprits, try adapting it to 3D, try adding network gaming, make a simple and easy map editor, if you haven't then try making scripted AI etc elaborate on the work that you've already done, same goes for the apps.

If you made a lets say fancy clock, then add features to it, make it a calendar, make it linked to a server where you can sync you schedule with a friends, make it "smart" so that you can set variables for when a meeting if ok for you and let the program find the best time for all that should attend. 
Give it the possiblity to differenciate between work, family time, hobbies etc, make it so that it has different scales of how important something is, make a web interface where people can request a "date/meeting" and have your server send you the request and a suggestion if you should take it or not and so on.
This project could use many different languages and work on different platforms(Linux, MS, Mobiles, PDAs etc). (this project could actually use more or less everything "C++, Java, Python, PHP, Actionscript 2.0, MySQL" and then some :) )

and to answer what you do when you know all the syntax etc, you put it in use and link every thing together and try new stuff that you haven't done.

### Post by Rich78 on 2008-10-22
Firstly I doubt you've learned all there is to know about a particular language as you're always learning no matter how proficient you are.

Secondly a professional programmer is just that, someone who does it as a profession.  If you want to be a professional programmer, then I suggest working towards doing it as a career, study, get the qualifications and go for the interviews.

If you want to progress your knowledge/experience of programming, then try to think of a project that might be of interest to you, either start your own project or find an ongoing open source one which you feel you can contribute to.

Programming is all about solving problems, so in order to program you need find problems that need to be solved.
