XHTML is a rewrite of HTML in XML. HTML was based on SGML.

That won't make sense :)

Just learn to write standard XHTML 1.1 and CSS. Use the w3c validator. On my home page, I have a link which validated my page. Click on it to see it in action. View the source of my page to see what it looks like.
