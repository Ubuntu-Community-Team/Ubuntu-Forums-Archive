### Post by Bachstelze on 2012-04-24
Yes and no. It is a good choice as a first language becuase in addition to being readily available on virtually every modern Linux system, it it very easy to learn and will get you started quickly. That said, no one who knows only one language can be a good programmer, so depending on how "serious" you are about programming, you might have to learn other languages. Java is a good thing to know, you should probably also learn C (C++ is less important right now, as it would be somehow redundant with Java).

### Post by kevinharper on 2012-04-25
+1 on what Simian Man & Bachstelze said.

Note that Java is becoming more widely used because of phones and tablets. A grasp of C, Java, & Python would be a nice combo. Do Python -> C -> revisit Java.

Here's an MIT course on computer science that uses Python (free, got these from Bachstelze a while ago)
