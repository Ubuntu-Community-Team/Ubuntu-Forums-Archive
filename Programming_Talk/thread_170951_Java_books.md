### Post by yaaarrrgg on 2006-05-05
As for intro books, most O'Reilly books are pretty good.  I think "Learning Java" is one of the best intro books I've seen.

As for online resources, the "Java developer's Almanac" is good for short code examples.  It takes a cookbook approach:
[http://java.sun.com/developer/codesamples/examplets/index.html](http://java.sun.com/developer/codesamples/examplets/index.html)

If I only had one book, I'd want the online documentation from sun is probably the best for everyday use.  The syntax of the language is pretty simple, although the API is fairly huge.  I've used it for a couple years and still run across things I didn't know existed.  

[http://java.sun.com/j2se/1.5.0/docs/api/](http://java.sun.com/j2se/1.5.0/docs/api/)
You can also add a search engine to firefox called "Java Doc"

Java 5.0 isn't all that different.  The changes are summarized at:
[http://java.sun.com/j2se/1.5.0/docs/relnotes/features.html](http://java.sun.com/j2se/1.5.0/docs/relnotes/features.html)
