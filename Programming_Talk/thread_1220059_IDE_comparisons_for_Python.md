### Post by pokerbirch on 2009-07-22
I spent several weeks trying out various IDE's and text editors. My main criteria were good code completion and simplicity. Komodo Edit was pretty good but i finally settled on Editra. It's simple, fast and suits my coding style: write some code, press F5 to run it. Edit the code, press F5. Rinse and repeat. Code completion is the best i could find and saves a lot of time having to Google for suggestions. There are also plugins for various things like project browsers, CSS editors, various themes, etc, but they're not really necessary. It works "out of the box" for Python and many other languages.

If you try Editra, download and install it from the website. The repository is out of date.
