Ks Arch Alpha Build 2 is available: [http://kshoster.net/content/ks-arch](http://kshoster.net/content/ks-arch)

Ks Arch is some kind of script library for Arch Linux. Some scripts may work with Ubuntu as well, but they does not exist yet. Ks Arch is build up with a GUI in form HTML files which will be ran in Firefox. The installer and additional scripts are BASH scripts.

Ks Arch is indeed light and is just a few KB. Installation instructions is in the Readme. You launch Ks Arch from the system menu.

Please notice that Ks Arch actually have not reached Alpha yet, and does only contain very few scripts at the moment.
