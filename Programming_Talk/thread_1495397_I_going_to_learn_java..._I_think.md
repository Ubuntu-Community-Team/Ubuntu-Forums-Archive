### Post by simeon87 on 2010-05-28
Frustration levels can rise, even for seasoned programmers. But for learning programming itself, it's more important to be able to have a mental model in your head of what's going on and how to fix things when you get the wrong values on the screen. Being successful in programming largely depends on that, all syntax and tools can be learned through practice (and being slapped by the compiler if you make a mistake).
