### Post by jwbrase on 2011-12-28
> **machdohvah said:**
> This is on my wish list. I would like to have someone create a Linux version compatible with the Ubuntu 11.10 Gnome environment of the exact same program made for DOS called FoxPro 2.6 for DOS. When I worked in aerospace a while back there a demonstration from Fox Inc. that showed that FoxPro was written for all operating systems including Linux. What ever happened to the Linux version?

Well, according to [this]("http://en.wikipedia.org/wiki/FoxPro_2"), FoxPro 2.6 for UNIX will run on Linux, but wasn't programmed specifically for it (Linux was very new at that time). It's proprietary software and no longer in print (it was released in 1993, almost 20 years ago), so you'd have to find someone who owns a copy and buy it from them to run it legally (and they'd, of course, have to uninstall it from their machine after selling it to you). 

Even then it might technically be illegal, if the EULA specifies that the software is not being sold to the purchaser, only licensed, and that the license is not transferable. (It's a Microsoft product, and Microsoft is very fond of such abusive licenses, but I don't know how far back their use of such terms goes).

Once you had a copy, you'd most likely find that it was distributed on floppy disks, and as most modern machines don't have floppy drives, you might have trouble getting it installed on your computer.

So you can run it on Linux, but finding and installing a legal copy might be more trouble than it's worth.
