### Post by vintem on 2005-07-20
yes, there is. Open "Synaptic Package Manager" (via System -> Administration) 

Note that, in Synaptic Package Manager, you should have added "universe" to your repositories (settings -> repositories -> add).

If you click "search" button and you fill in "kDevelop", you will see the files you need.
Mark them for installation and click on "apply".

I suppose you need to have Kubuntu to get kDevelop to work, but I'm not sure (I'm also a newbie and I'm using Ubuntu - Gnome
