### Post by smartbei on 2007-11-24
I occasionally have KDE programs crash due to them failing to startup a plugin program they take for granted but is not available on a non-KDE system, such as Konsole. If you are getting related error messages, try installing the application they reference. If there is nothing, try running the program from the command line so that you can (hopefully) see the cause of the crash.

Is the behavior consistent (does it happen every single time you save a file)?
