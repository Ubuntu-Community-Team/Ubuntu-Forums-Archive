### Post by ZuLuuuuuu on 2008-10-20
It is the abbreviation of this book:

[http://www.amazon.com/Programming-Language-Prentice-Hall-Software/dp/0131103628/ref=sr_1_1?ie=UTF8&s=books&qid=1224522947&sr=8-1](http://www.amazon.com/Programming-Language-Prentice-Hall-Software/dp/0131103628/ref=sr_1_1?ie=UTF8&s=books&qid=1224522947&sr=8-1)

But it is not for complete beginners, so you should define your level more detailed; do you know any other programming language and want to learn C or will C be the first programming language of yours etc...
