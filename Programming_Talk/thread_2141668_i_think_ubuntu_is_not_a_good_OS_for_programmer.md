### Post by iamkuriouspurpleoranj on 2013-05-12
Ubuntu is fine for programming. Don't forget: it's built on top of Debian. The advantages of using something like CentOS are a) for learning CentOS/Red Hat b) stability c) developing a site in the same environment it will be hosted on, if your hosting provider uses CentOS.

It will tend to have older versions of software (PHP and MySQL, for example) but that's a good thing if your site will be hosted on a similarly old server. That said, in recent memory I've installed NetBeans 7.2 (from the NetBeans site), the latest VirtualBox (from the VB site) and even had Enterprise Architect running in wine but you might need 6.4 for a good enough wine. 

However, c) can apply to Ubuntu and b) can apply if you use an LTS. Ubuntu is also used as a server in the real world and can even help you learn Debian. So that's a) too, then.

What's nice about CentOS is that I'm always conscious it's a tool and so just get on with work with it. In fact, I haven't even bothered to customise my current CentOS box, not even a wallpaper, yet my website is complete. I just need to wait for the domain name to activate and everything will be dandy.

But there's nothing wrong with Ubuntu for programming. Of course, it's Linux so it's good for developing web stuff and for other Linux. If you're developing software to be installed on Windows and OS X boxes, you might do better to look elsewhere. Macs are probably a better bet for hedging your bets though, as you can develop comfortably for all three, as well as iOS and Android. Plus more people will want to be your friend if you have a Mac. No, really.

### Post by trent.josephsen on 2013-05-12
You've heard of comparing apples to oranges, but for programming purposes comparing different Linux distros is like comparing a Granny Smith apple to a slightly greener Granny Smith apple. They all work the same way and any particular version of a piece of software, be it an IDE, compiler, editor, debugger, whatever, will behave identically on all of them, assuming you can install it. And for all practical purposes there's no reason to aggressively update your development tools like you would a browser or anything that handles untrusted and potentially malicious data.

Pick any Linux distro you like, or pick Windows if you want to write programs for Windows specifically.

### Post by slickymaster on 2013-05-13
It depends what computer programs you want to write. If you want to write applications for Windows users, or to write programs that use the dot Net framework or that automate the creation or processing of MS-Office documents then writing them on a Windows 7 platform makes sense. On the other hand, Ubuntu costs you nothing, comes with Perl, and Python programming languages already installed, has a great, helpful user community and lots of free software you can download.
Depending on what your goals are, I feel Linux is a great platform to learn programming on. It was built as a unixlike system, and has a wide array of development tools and programming language compilers and interpreters available to it. It can also allow you to get a better lower level understanding of your computer, which is integral (IMHO) to being a talented computer programmer.
