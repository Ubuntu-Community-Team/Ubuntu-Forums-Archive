### Post by ankursethi on 2008-10-14
I have seen many people using slightly academica languages like Haskell or OCaml in these contests. In any case, you need something more than Java or C++ to win in the IOI, since you need to focus on your algorithms rather than twiddling bits and pointers. Java, IMHO, is still pretty low level compared to, say, Python.

The algorithm matters. Choose whatever language you want and follow this course : [http://ocw.mit.edu/OcwWeb/Electrical-Engineering-and-Computer-Science/6-046JFall-2005/CourseHome/index.htm](http://ocw.mit.edu/OcwWeb/Electrical-Engineering-and-Computer-Science/6-046JFall-2005/CourseHome/index.htm)

The book "Introduction to Algorithms" is a beast. You might want to just look at the introductory parts and basic data structures, then move on to the section on graph theory. I just got the book myself, and I'll be getting into it whenever I get some free time.

EDIT : "free tim" doesn't make sense.
