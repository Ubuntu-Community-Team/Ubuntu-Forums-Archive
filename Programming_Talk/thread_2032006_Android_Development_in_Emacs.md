### Post by mevun on 2012-07-22
Of course, you need to have the Android SDK and the JDK.  I don't think you need the NDK unless you plan to write some C code for your app.  

Just search online for "develop android without eclipse".  There are plenty of guides.

If your system is a bit slow, you aren't going to be happy with the emulator's performance.  I guess it is good you won't be running Eclipse at the same time, but I don't if that will help enough.
