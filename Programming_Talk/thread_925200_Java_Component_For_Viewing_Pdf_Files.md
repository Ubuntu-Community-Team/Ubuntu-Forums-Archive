### Post by jespdj on 2008-09-20
> **DBQ said:**
> I looked on google, and did not find anything free.
Really? What search terms did you use? If you would have searched simply for "java pdf", you would have found the following, among others:

[iText](http://www.lowagie.com/iText/) is one of the most well-known Java PDF libraries. I don't know if it also includes something to render PDF.

A library that can create images from PDF files (which you can display in your Java program) is [PDFBox](http://www.pdfbox.org/).

[PDFRenderer](https://pdf-renderer.dev.java.net/) is a free and open source 100% Java PDF renderer and viewer.
