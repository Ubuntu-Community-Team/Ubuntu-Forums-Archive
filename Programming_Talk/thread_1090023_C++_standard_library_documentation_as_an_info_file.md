### Post by BattlePanic on 2009-03-07
Does anyone know if there is any documentation to the C++ standard library that is available as an info file?  I do most of my C/C++ programming in emacs and love its ability to look up a C standard library function in the relevant info file by placing the cursor over the function and pressing Ctrl+h+Shift+s. ### Post by the_unforgiven on 2009-03-08
[http://gcc.gnu.org/onlinedocs/](http://gcc.gnu.org/onlinedocs/)
Go down to 'Current development' -> 'libstdc++-v3 documentation'.
They give the Texinfo sources.

### Post by merlinblack on 2009-05-09
You can also lookup libstc++ in Synaptic and grab the doc package.  Then browse to

file:///usr/share/doc/gcc-4.3-base/libstdc++/html/spine.html

I think its the same information as what the previous poster gave a link to.
