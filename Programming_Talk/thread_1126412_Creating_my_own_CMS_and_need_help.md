### Post by hessiess on 2009-04-15
Start simple, wright a CMS for a single website which is not extendalbe, eaven a simple CMS is quite complicated.

As an absolute minimun you will need to create a system to display pages from a database, a system with a frount end to add pages and update existing ones in the database and a file manager. Optionally you may also want a visual designer for formatting and/or a custom markup language.
