### Post by Corlyn on 2007-05-27
If you click on the add/remove button and type Wine it will pop up on as the first choice. Very simple to use. I usually do it from the terminal for example: "wine Frozen\ Throne.exe -opengl -window". I haven't came across a game yet that I haven't been able to play.
The other program is Cedega, but I don't have any experience with it.
