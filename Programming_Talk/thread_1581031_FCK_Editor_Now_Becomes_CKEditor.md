FCK Editor Now Becomes CKEditor:

FCK Editor is an HTML text editor having many of the powerful functionalities of text editor like MS Word. It is the most widely used implementable text editor in the world of web applications. It helps web programmers to produce content management systems that can produce dynamic content very quickly for almost any web development environment.

It is a very lightweight HTML text editor and does not require any kind of installation on the client computer. 

FCKEditor also popularly known as CKEditor, releasing a more impressive version of dynamic text editor. The FCK in FCKeditor stands for Frederico Caldeira Knabben and the CK in CKEditor stands for Content Knowledge.
