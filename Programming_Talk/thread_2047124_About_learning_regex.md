### Post by Lars Noodén on 2012-08-24
There are two main kinds, POSIX and Perl-Compatible Regular Expressions (PCRE).  If you're more into programming start with PCRE.  If you are more into systems administration, then start with POSIX.  

You'll have to look around the net for tutorials and HowTos like these: 

[http://www.cs.tut.fi/~jkorpela/perl/regexp.html](http://www.cs.tut.fi/~jkorpela/perl/regexp.html)
[http://www-rohan.sdsu.edu/doc/sed.html](http://www-rohan.sdsu.edu/doc/sed.html)

so you can find ones that match your interests.  The expressions can get quite complex, especially when dealing with substitutions.
