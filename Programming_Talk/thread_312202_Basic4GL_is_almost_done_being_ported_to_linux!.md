### Post by Wybiral on 2006-12-04
Basic4GL is a FREE BASIC programming language that offers the programmer BASIC syntax wrapped into an OpenGL shell. Up until recently it was only Windows compatible, but a project to port it to linux has emerged and made quite a bit of progress.

You can check it out here:
[http://b4gl-linux.wikispaces.com/](http://b4gl-linux.wikispaces.com/)

And the official forum is here:
[http://basic4gl.proboards20.com/](http://basic4gl.proboards20.com/)

The last release used GLUT to setup the window and everything, but the next one will switch to SDL. It still needs alot of work, but it is certainly comming along. So any BASIC programmers out there that cant find a good BASIC for linux, this should soon be a good program.

BlitzMax is a fantastic, extendable, game programming language that is.... awesome :P  Although I don't use Blitz Products anymore, BlitzMax is really good, and has a Linux port.

BlitzMax recently got a GUI Module and they are working on a 3D Module.  Their 3D Modules are always really easy to use, but powerful to create good looking stuff.  Also, the 2D Module available is AMAZING. :)

BlitzMax is also a write-once, compile anywhere (For the most part) with Win/Lin/Mac support and OpenGL/DirectX drivers for the 2D and hopefully 3D Engine.

The only bad part is that BMax is shareware, and not freeware, but it is worth it if you want to go game programming under Linux.  Easy, quick, powerful.  :)  Also, the community is FANTASTIC.  I should know, I was a part of it.  :)
