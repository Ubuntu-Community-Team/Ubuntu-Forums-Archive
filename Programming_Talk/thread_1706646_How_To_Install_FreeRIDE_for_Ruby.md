### Post by Darren Sparrow on 2011-03-14
Hello all, I'm not sure if this has been posted before so I thought I'd stick this up here and hopefully give some help to people not sure how to install this wonderful (although dated) IDE for using with the Ruby programming language.

I'm using Ubuntu 10.10 at the moment, so this should work with previous releases.

Ok, here goes...

Most times when you download something it's usually in your Downloads folder, so I'll be showing you how to do things from there.

at the terminal prompt type         CD Downloads
This takes you to the Downloads folder.


Then type dir to see what's in your Downloads folder, this should show   freeride-linux-installer-0.9.6.sh

Next whilst in this folder type 

 chmod +x freeride-linux-installer-0.9.6.sh

Then type sudo sh freeride-linux-installer-0.9.6.sh

Just press the Enter key when it suggests where to install.

If all goes well you should get this 

--------------------- MANIFEST ---------------------

This is version 0.9.6 of FreeRIDE, the Ruby integrated
development environment.

This version is built with the following components:

Ruby               1.8.4
Fox Toolkit        1.2.16
Fox Scintilla      1.62
FXRuby             1.2.6

----------------------------------------------------
FreeRIDE succesfully installed.
Start FreeRIDE with '/usr/local/FreeRIDE/freeride'


Or something similar.

Then it's just a matter of editing your Applications menu to point to the FreeRIDE program and it'll run.

--------------------------------------------------------------------------------------------------------------------------

Now I hope people don't mind me posting this here, it's just I've had a few emails from friends both new to Linux and to the Ruby programming language, and although there are some very good IDE's available for using with Ruby, they can be a bit overwhelming to new users.

I like using FreeRIDE, and although it hasn't been updated for a few years now, when I get better at using Ruby I will hopefully see if I am able to update the program and address any issues with the bugs that some people have.
