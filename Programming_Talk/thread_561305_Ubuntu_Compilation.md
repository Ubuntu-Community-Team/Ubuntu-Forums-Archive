### Post by Jussi Kukkonen on 2007-09-27
Source code for all ubuntu packages is available from the repositories. If you're on ubuntu just run *"apt-get source package"* for any package that you want to inspect. For single packages using any OS you can also use [http://packages.ubuntu.com](http://packages.ubuntu.com) (select "source package name" and some search terms)

For building a derivative distro, see: [https://wiki.ubuntu.com/DerivativeDistroHowto](https://wiki.ubuntu.com/DerivativeDistroHowto)
