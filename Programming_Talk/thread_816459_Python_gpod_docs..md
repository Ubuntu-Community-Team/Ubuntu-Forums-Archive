### Post by kalos on 2009-09-20
The best I've found for a reference is pydoc:

```
pydoc gpod
```

Works for any python code you have installed.

You can also run ```
pydoc -g
``` and click "Open Browser" to browse the documentation for all of your installed python modules in Firefox.
