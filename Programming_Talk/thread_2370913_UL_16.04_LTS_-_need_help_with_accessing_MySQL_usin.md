PHP: 7.0.22-0ubuntu0.16.04.1
MySQL: 5.7.19-0ubuntu0.16.04.1

I can verify that MySQL is currently running.

When I try to connect to MySQL server using PHP, I can successfully do it; I do not face any issues/problems.

However, when I try to access the same database using C language, I see an error message.

```

#include <stdlib.h>
#include <stdio.h>
#include <mysql.h>

int main(int argc, char *argv[])
{
    MYSQL *conn_ptr;
    
    conn_ptr = mysql_init(NULL);
    if (!conn_ptr)
    {
        fprintf(stderr, "mysql_init failed\n");
        return EXIT_FAILURE;
    }
    
    conn_ptr = mysql_connect(conn_ptr, "localhost", "username", "password", "db_name", 0, NULL, 0);
    if (conn_ptr)
    {
        printf("Connection success!\n");
    }
    else
    {
        printf("Connection failed!\n");
    }
    mysql_close(conn_ptr);
    return EXIT_SUCCESS;
}

```
When I try to compile this C file using:
```

gcc connect1.c -o connect1
gcc -I/usr/include/mysql connect1.c -L/usr/lib/mysql -lmysqlclient -o connect1

```
For both commands, I see the same error:
```

connect1.c:3:19: fatal error: mysql.h: No such file or directory
compilation terminated.

```
I tried to find where mysql.h is installed but I couldn't find any file  with that name. What package(s) am I missing? How do I make my C program  access MySQL database?

Any help on this is appreciated. Thanks!

### Post by norobro on 2017-09-08
You need libmysqlclient-dev [https://packages.ubuntu.com/xenial/libmysqlclient-dev](https://packages.ubuntu.com/xenial/libmysqlclient-dev)

Note also: [https://dev.mysql.com/doc/refman/5.7/en/mysql-connect.html](https://dev.mysql.com/doc/refman/5.7/en/mysql-connect.html)

---

### Post by la.khan on 2017-09-08
Thanks for the reply.

I have downloaded the package libmysqlclient-dev and changed code to use [mysql_real_connect()]("https://dev.mysql.com/doc/refman/5.7/en/mysql-real-connect.html") 

Now, my C file compiles with:
```

$gcc -I/usr/include/mysql connect1.c -L/usr/lib/mysql -lmysqlclient -o connect1

```

:smile:

Thanks for your help!
