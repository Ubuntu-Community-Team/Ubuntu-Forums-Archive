### Post by SeijiSensei on 2012-01-16
I don't see what "mode" you're in matters one whit.

MySQL is licensed under GPLv2.  That gives you the right to re-distribute the software to anyone free of charge as long as you distribute the source code as well.  Saying you need to buy an OEM license for MySQL sounds like a shakedown to me.  If that were true, no one could ship a system with RedHat or Debian and MySQL on it.

When I first started building websites with database back ends around 1997-98 or so, I chose PostgreSQL because it was the only freely-redistributable DB then available.  I've never regretted this choice; the fact that Oracle has ended up with MySQL only reinforces that decision in my book.

I strongly urge anyone running MySQL in a commercial setting to consider switching to PostgreSQL and let Oracle twist in the wind.
