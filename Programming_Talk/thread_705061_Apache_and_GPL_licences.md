---

### Post by dismal_denizen on 2008-02-25
After a bit more research I have found the answer to my own question. The Apache License v2.0 is [GNU compatible]("http://www.gnu.org/licenses/gpl-faq.html#WhatIsCompatible"), which means I can use GPL v3 for the whole project. Yay!
