If not emacs, here is my requirements:

[LIST]
[*]Can handle, colorhighlighting and preferably indent "mixed modes" correctly. I mean for instance mixed HTML and Javascript

[/LIST]

Nice to have:

[LIST]
[*]Autocompletion. I mean like Visual Studio where you can tab and complete a word/html element

[/LIST]

Also, another nice feature is regular expression support in search.

Cssed software has Css autocompletion support and it is good.
