### Post by brcre on 2009-01-22
1st DVD740 I purchased only recognized CDs never a DVD.
I sent it back and got a new one.  It correctly recognized CDs and DVDs or so I thought until I finally bought a DVD +R DL which was recognized as a CD of only 700MB.

Solution was to install the drive into a windows computer and then download and install new firmware from HP.  So far so good.

[http://h10025.www1.hp.com/ewfrf/wc/softwareDownloadIndex?os=228&softwareitem=pv-28739-13&lc=en&dlc=en&cc=us&product=502021&lang=en](http://h10025.www1.hp.com/ewfrf/wc/softwareDownloadIndex?os=228&softwareitem=pv-28739-13&lc=en&dlc=en&cc=us&product=502021&lang=en)

They didn't have a solution available for any other platform other than Windows XP.
Hopefully this helps anyone who also runs into this issue.
