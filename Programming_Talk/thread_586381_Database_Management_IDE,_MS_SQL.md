### Post by AZzKikR on 2007-10-22
OpenOffice has a database program which lets you connect to a database using JDBC. You can do queries on the database that way. There is also SquirrelSQL which is a Java based SQL tool, which can connect to any database, given you have the JDBC driver for it.
