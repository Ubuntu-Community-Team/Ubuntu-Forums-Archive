### Post by Lux Perpetua on 2007-08-15
The statement ```
typedef X Y;
```defines Y to be a synonym for the type name X. In your example, the first line makes "vec_sz" a synonym for the type vector<double>::size_type, and the second line declares a variable using the new name.
