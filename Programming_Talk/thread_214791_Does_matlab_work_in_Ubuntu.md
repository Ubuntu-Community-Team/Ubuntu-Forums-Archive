Many of the advanced functions are defined a little differently making it impossible to directly run the same code on both.  I also found functions which gave inaccurate output.  All in all the question I kept asking myself was "why come so close to mimicking Matlab but then change it a little?"  Anyways, for general/simple computations octave works but it is unfortunately not even close to a valid substitute for Matlab.
