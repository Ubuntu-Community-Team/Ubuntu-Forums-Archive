### Post by Hellkeepa on 2010-03-20
HELLo!

I reckon you're talking about the "PHP explorer view", which is where you see the projects listed?

When it comes to the speed issue, just close all but the currently active project. You can do this by right-clicking on the project you're working on, and selecting "Close unrelated projects".

As for the local file explorer, you can get it back. Though, Zend Studio 7.x (read: Eclipse) is really built for using the projects to load files. That said, just add the "Remote systems" view to the perspective. There you will find "Local files", and under it "My home". Clicking on this will open up your home folder, and you can also right-click and select "Go into" to make this the root folder which is shown in the "Remote systems" view.

Happy codin'!
