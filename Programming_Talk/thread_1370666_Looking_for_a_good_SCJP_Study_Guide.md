### Post by Axos on 2010-01-15
It's not a study guide but *The Java Programming Language* by Arnold, Gosling, and Holmes is absolutely mandatory reading for any serious Java programmer. If you already have that (and have read it cover to cover), never mind. :D

The only thing which annoyed me a bit about the Sierra and Bates book was that it isn't quite as precise with the nomenclature as *The Java Programming Language* is. For example, they are a bit loose with what they call the various kinds of nested classes.
