### Post by CptPicard on 2009-10-21
Considering that the two are rather different languages (although Javascript does have some lispish elements to it), I would say no, unfortunately... these kinds of converters generally do not exist, as different languages behave differently enough that a "mechanical" translation is not possible unless one goes the route of actually writing an interpreter for the source language in the host language.

Mind you, this is the reason why differences between programming languages are not mere "syntactics" but are deeper and more real, regardless of their theoretical equal strength.
