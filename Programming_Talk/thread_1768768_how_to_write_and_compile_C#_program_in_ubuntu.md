### Post by PaulM1985 on 2011-05-27
Install Monodevelop from the Ubuntu Software Centre.  It will allow you to create C# projects in the same way as Visual Studio.  If you want to use Windows Forms you need to add it as a reference on your project.

You should be able to take your solution and open it in Visual Studio ok for VS2008.  I am not sure about VS2010.

If you build it in Ubuntu it should run on Windows, but I would re-build it using Visual Studio if you have it available.

Paul
