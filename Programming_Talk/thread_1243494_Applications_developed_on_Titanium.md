### Post by irv on 2009-08-18
Has anyone tried Titanium yet. I read an article in Linux Journal this past issue and it was interesting. I downloaded the open source version of the Web developer package for creating desktop applications. I had some problems with it so I worked with one of there tech guys and he sent me a Drillbit tester so we could get some of the bugs worked out. It seems to work, but I have not done any developing with it yet. If any one is interested there website is:

[http://www.appcelerator.com/appcelerator-platform/titanium-architecture/]("http://www.appcelerator.com/appcelerator-platform/titanium-architecture/")

There plans are to release the fix in the next version of this package, so you may want to wait until this version comes out. They thought that the Compiz and transparency is where the problem was in the release that is available right now.

The thing that caught my attention was the fact you could call JavaScript, Python and Ruby all in one HTML file. 

If you are interested in this, just get this month's issue of Linux Journal (Sept. 2009) and read the article.
