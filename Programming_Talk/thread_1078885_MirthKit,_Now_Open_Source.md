### Post by curvedinfinity on 2009-02-23
All,
I've flaunted community leaders long enough, and I'm finally going to match my words with actions. I just pushed MirthKit's source code to Launchpad. [MirthKit](http://www.mirthkit.com) is an advanced web-based 2D game development and distribution platform. I've picked LGPL as the license. Its designed to be OSS's Steam, except with an emphasis on ease of development. Check the developer website for tutorials, API examples, and so on.

To download the source, use bazaar (via [http://www.launchpad.net/mirthkit](http://www.launchpad.net/mirthkit)):

```
bzr branch lp:mirthkit
```

You'll then have to grab all of the dependencies (in the readme file), and compile/install squirrel (included in the project).

If anyone is interested in helping the project, it needs make scripts and an OS X distributable. There is also a list of features that would be well worth putting in.

Lastly, I do have designs for an extremely simple 3D API in the works, but it needs to be very flush before it is included in MirthKit (which has a very very clean and condensed API).

-Chase

P.S. Please let me know if you spot any left-over legal anomalies on the website.
