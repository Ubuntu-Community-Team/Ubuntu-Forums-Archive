### Post by jespdj on 2009-04-10
Java EE version 6 is not yet out, as far as I know, and there is no Java EE container that implements Java EE 6 yet. Note that the version numbers for Java SE and Java EE are not equal. The currently released version is Java EE 5.

Some useful information:
[JSR 316 - Java EE 6](http://jcp.org/en/jsr/detail?id=316)
[Glassfish v3 Prelude](http://www.sun.com/software/products/glassfishv3_prelude/index.xml)

The standard place to start for Java EE documentation is ofcourse:
[http://java.sun.com/javaee/reference/](http://java.sun.com/javaee/reference/)
