### Post by SoggyCornflake on 2007-05-01
> **Seisen said:**

As with most things, what do you intend to do with it?  What's better a Hammer or a Socket Set?

Learning XML is useful if you're looking at working with applications and data.  Learning HTML would be useful for working with web pages.

If I were to choose to study only one, it would be XML because once you grasp how it works, then HTML can be seen as a subset of XML more or less.
