### Post by nmincone on 2006-12-22
wget is a very powerful command that has many uses and is extremely flexible. Through the command line interface (CLI) you can issue a #man wget for some information.
Detailed information can be found here [at the wget GNU project page.]("http://www.gnu.org/software/wget/")
Some sample scripting using shell scripts & wget are found here [Linux Journal Smart (Script-Aided) Browsing]("http://www.linuxjournal.com/article/5905")
Attaching the script to the wget command is possible but too detailed to document here, see the links I provided for assistance.
