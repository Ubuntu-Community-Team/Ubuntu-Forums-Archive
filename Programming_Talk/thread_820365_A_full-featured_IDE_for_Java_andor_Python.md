### Post by Jeff Johnston on 2008-06-06
I use Netbeans for Java. I also started using it for doing GTK+ development with the nice C/C++ plugin, which is part of the netbeans distribution. Sun also recently announced that they will be supporting Python in the IDE which utilizes the work done for the Ruby plugin. The nice thing about this kind of integration is no matter which language I use most (if not all) of the shortcut keys are the same and the editor is consistent.

I used to use Eclipse (for years) and it is nice as well.

Your best bet is to try them both out and see which one you like. A little bit of a learning curve for each but its worth it.
