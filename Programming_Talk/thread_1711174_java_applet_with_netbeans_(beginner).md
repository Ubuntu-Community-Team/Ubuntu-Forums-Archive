### Post by PaulM1985 on 2011-03-22
I ran this in a debugger.  The issue is that for some reason although the preferred size is 100 x 100, it is setting the size to be 0 x 0.  Because it is 0 x 0 it decides there is no point painting it.

I am not sure why it is setting to 0 x 0, but if you call:
```

setSize(new Dimension(100, 100));
setMinimumSize(new Dimension(100, 100));

```

It will set the size to be 100 x 100 and your paintComponent method will be executed.

Paul
