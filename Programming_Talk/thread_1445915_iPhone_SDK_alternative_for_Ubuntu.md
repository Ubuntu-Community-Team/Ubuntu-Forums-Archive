### Post by heikaman on 2010-04-03
the iphone sdk is based on the gcc toolchain so it works just fine on linux, there's a script here [http://code.google.com/p/iphonedevonlinux/]("http://code.google.com/p/iphonedevonlinux/") that downloads and builds the whole toolchain on linux, but don't expect anything fancy, like xcode, you'll have to write everything from scratch, and deploy with ssh, some examples are included too, have fun :popcorn:
