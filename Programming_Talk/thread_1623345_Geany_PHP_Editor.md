### Post by manuelb on 2010-11-16
> **KevzJD said:**
> how would i go about installing it from there?

If you want the latest version, you'll need to compile the plugin before using it: download the file [http://plugins.geany.org/geany-plugins/geany-plugins-0.19.tar.bz2]("http://plugins.geany.org/geany-plugins/geany-plugins-0.19.tar.bz2"), this contains various geany plugins and the treebrowser plugin too.

I don't know if the latest Ubuntu now includes it, but you could try to install if you didn't do it already:
```

apt-get install geany-plugins

```
