### Post by JupiterV2 on 2011-09-25
There are plenty of ways. Search for GNU autotools if you want to install on Linux. You'll need them to install via tarball/source, .rpm, .deb, etc. On windows you need to build an installer. There's lots. A popular free one is NSIS (NullSoft Install Script). Do a search for 'windows installer software' or something to that effect.

You need to do a little homework yourself before asking people here for help.
