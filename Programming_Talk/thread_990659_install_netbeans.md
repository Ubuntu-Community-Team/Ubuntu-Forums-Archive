### Post by jespdj on 2008-11-23
How did you install NetBeans, did you download it from the NetBeans website and run the installer (i.e. you did not install it via Synaptic or apt-get)?

See [NetBeans 6.0.1 installation instructions](http://www.netbeans.org/community/releases/60/install.html), it also explains how to uninstall the software.

What do you mean with "the next step to proceed"? If you've installed the JDK and NetBeans you can start using it (there are no next steps that you have to do to make it work). You'll find NetBeans in the menu Applications / Programming.

If you want to start with learning Java, then the [Sun Java Tutorials](http://java.sun.com/docs/books/tutorial/) are a good place to start.
