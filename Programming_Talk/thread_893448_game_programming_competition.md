### Post by mvaniersel on 2008-08-18
Hi there

For the 5th time, I'm organizing TINS, a game programming competition where you have to write a game using the Allegro library within 72 hours. It will take place from August 29 to September 1, so that's pretty soon already. The competition website can be found at [http://blog.bigcat.unimaas.nl/~martijn/tins08/](http://blog.bigcat.unimaas.nl/~martijn/tins08/)

The competition accepts only cross-platform entries. My hope is that this competition will contribute a little bit to a healthy Linux game development community.

You're allowed to use any programming language that has Allegro bindings, C and C++ are most common. There are a few more rules, like you have a maximum entry size of 400 kb compressed, and you have to submit the complete source code (for details see the website). Plus, there are always a few secret rules that are announced just before the start.

It's always a lot of fun, and it is a good way to tune your programming skills. You'll be surprised what you can achieve in 72 hours of dedicated hacking :)
