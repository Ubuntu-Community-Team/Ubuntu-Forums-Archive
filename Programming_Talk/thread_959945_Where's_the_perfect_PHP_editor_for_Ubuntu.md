[LIST]

### Post by ad_267 on 2008-10-26
Geany might be what you're after. It is very lightweight and fast and I'm pretty sure it can do all of those.

Gedit (the default text-editor) can also be very powerful with some plugins. See [http://live.gnome.org/Gedit/Plugins](http://live.gnome.org/Gedit/Plugins) and the gedit-plugins package in Synaptic.

Bluefish is another editor that is oriented towards php/web development.

**Edit:** My mistake, it doesn't look like geany can autocomplete brackets, although it does have code completion.

There's a plugin for gedit that does this though: [http://www.garyharan.com/index.php/2006/11/16/gemini-gedit-plugin-for-all-those-textmate-fans/](http://www.garyharan.com/index.php/2006/11/16/gemini-gedit-plugin-for-all-those-textmate-fans/)
So gedit can do all three things you're asking.
