### Post by Impavidus on 2016-04-09
Ubuntu (or any other Linux) is great for programming. If you want to build GUIs, I suggest you have a look at [GTK+]("https://en.wikipedia.org/wiki/GTK%2B") or [Qt]("https://en.wikipedia.org/wiki/Qt_%28software%29"). I never used those myself though. For 3D graphics as in games or data visualisation, I suggest you have a look at [OpenGL]("https://en.wikipedia.org/wiki/OpenGL"). Use your favorite search engine to find tutorials; there are plenty around on the web. To create a context for OpenGL and handle mouse and keyboard interaction I use [GLFW]("https://en.wikipedia.org/wiki/GLFW"), but there are other possibilities.

There is no best or worst programming language. There are just programming languages more or less well suited for different types of program or for different types of programmer.
