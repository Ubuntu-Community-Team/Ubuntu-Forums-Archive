### Post by geirha on 2009-06-29
I'm having trouble understanding what you want to do, but based on your example, I guesstimate that you want something like this
```
#!/bin/sh

datafile="data.txt"
grep '^cre eqplnk' "$datafile" | awk '{print $4" "$6" "$7}' |
while read nodebid lu vpi; do
    grep "$lu $vpi" "$datafile" > "$datafile.$nodebid"
done

```
First it grabs all lines starting with "cre eqplnk", then extracts the fourth (nodebid=), sixth (lu=) and seventh (vpi=) field of each line. 

For each of those tuples, it writes all lines from the datafile that has the same "lu=X vpi=Y" to a new file named "data.txt.nodebid=Z".

So the example output you provided will be in a file called "data.txt.nodebid=84"
