### Post by Can+~ on 2009-04-16
Before investing in any book, take a peek at the documentation:

[http://docs.python.org/](http://docs.python.org/)

If you still want to invest in book. I personally bought O'Reilly "Python in a Nutshell" which was quarter-language introduction and three-quarters-reference manual.

It's not a friendly book for beginners, more intended to mature programmers that want to catch up the features of python and doubles as a reference manual (which is more comfortable that having to open a tab on the python doc)

But that's just me.
