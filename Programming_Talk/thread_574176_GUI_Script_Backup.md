Hi,
    i used a program called zenity which creates dialog boxes from shell scripts and requires no knowledge of code writing to make the dialog boxes. In a terminal do:

sudo apt-get install zenity
zenity --help

That will install zenity and then show you all the things you can do with it. Also look  [here]("http://linux.byexamples.com/archives/259/a-complete-zenity-dialog-examples-1/"). hope that helps :)
