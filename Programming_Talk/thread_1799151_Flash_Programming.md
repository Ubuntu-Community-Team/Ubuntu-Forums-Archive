### Post by megabytemonster on 2011-07-08
There are a few options actually. You can use the open source Flex SDK directly: [http://opensource.adobe.com/wiki/display/flexsdk/Flex+SDK](http://opensource.adobe.com/wiki/display/flexsdk/Flex+SDK)

Or a library already geared towards making games: [http://flixel.org/](http://flixel.org/)

Or if you want to go super-easy mode, someone has made a very nice cross platform IDE which uses the flixel library: [http://www.stencyl.com/](http://www.stencyl.com/)
