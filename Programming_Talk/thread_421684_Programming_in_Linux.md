### Post by predaeus on 2007-04-24
Developing Java applications is well supported by the Eclipse IDE. For C/C++ you could use the Eclipse/CDT plugin, but it is not as fast as Anjuta which is very nice for coding C/C++. Also there is a new language around with great features called D. It is somehow a mix of C/C++ and Java and other languages, I am going to try and program my next bigger application in D for sure. IIRC all of this is in the Ubuntu repositories even a D compiler should be in there, probably the gcc one. You could also check out Python as many Linux applications support Python plugins or are Python applications.

For more info check out the links:
[http://java.sun.com/](http://java.sun.com/)
[http://www.digitalmars.com/d/](http://www.digitalmars.com/d/)
[http://www.python.org/](http://www.python.org/)

### Post by JT673 on 2007-04-25
Ruby's the best OO language that comes to my mind.  Java might be nice too.  PHP is supported, but I don't know about .NET.

Eclipse, EasyEclipse, or MyEclipse is my favorite IDE, but you can also consider NetBeans, jEdit, and many others from the sticky thread.  gEdit (Applications -> Accesories -> Text Editor) is a pretty good editor already built into the system.
