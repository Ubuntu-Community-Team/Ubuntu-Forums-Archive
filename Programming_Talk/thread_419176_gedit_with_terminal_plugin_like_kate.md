### Post by TheNumber42 on 2008-10-03
To get Embeded Terminal in gedit, open Synaptic Package Manager ('System' > 'Administration' > 'Synaptic Package Manager') and search for "gedit plugins". Select 'gedit-plugins' and then click 'Mark for Installation', finally click the 'Apply' icon to install.

Once it's finished, open up gedit and select which plugins you would like to use ('Edit' > 'Preferences' and click the 'Plugins' tab).

Have fun :)
