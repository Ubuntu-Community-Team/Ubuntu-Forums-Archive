### Post by Can+~ on 2008-01-11
I remember, my first interaction with programming, was with War3 editor, which had a kind-a pseudo language (after that I learnt ActionScript, Javascript, C, and now studying Python on my own).

I still remember how good was the War3 editor system, it had an easy interface with functions and keywords someting like:

> Event
Condition
Action

So when the event occurs, and a condition is met, an action is done. The way in which you could do this, is adding functions to each one of them:

> When _Player X_ enters _Region_ do _stuff_.

When you click on Player, you are given a list of possible players, when clicking a region, you could add a Location on the map, etc.

So, I was thinking, what if we could do this into a real Bash application, friendly with new users, I think MacOS has something called Automator? Sorry, I never used a Mac (and I won't use one).

> "When computer _loads_ run with a _x seconds_ delay:"
"Ask for root power"
"Delete files in _folder_"
"Do other stuff"

I still lack the skills for creating visual applications with python, but I give the idea for anyone to develop.
