### Post by samjh on 2009-04-16
They're still using 1.4.2?!

Support for that version is long gone.  If you really need it, you'll need to download the installer from Sun's website ([http://java.sun.com/j2se/1.4.2/download.html](http://java.sun.com/j2se/1.4.2/download.html)) and install it manually.

I wrote a tutorial for manually installing Java on Ubuntu a long time ago:
[http://ubuntuforums.org/showthread.php?t=319138](http://ubuntuforums.org/showthread.php?t=319138)
Ignore Step Five of the tutorial which installs the plugin for Firefox, as you'll want to use the up-to-date plugin from the repositories.

Good luck.  And tell your professor that they should be using a more up-to-date version of Java.  Even Sun has discontinued support for 1.4 last year!
