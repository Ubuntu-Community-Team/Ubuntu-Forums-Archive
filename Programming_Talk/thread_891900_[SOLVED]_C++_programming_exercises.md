### Post by Mr.Macdonald on 2008-08-16
write a data compressor that hopefully can expand the data
implement the merge sort, I consider it to be the best sort algorithm (quick and heap next)
write an efficient encrypting mechanism to encrypt a standard file
write a scripting language
write a game that uses AI like checkers, connect4 (I have done it), chess(harder), or battleship
during the process write a header library of neat classes that make life easier

these aren't super advanced or too easy, but will challenge you (provided you know how to program)

difficulty in programming should make it funner, just don't write all the code at once, because that will make it annoying to debug.

Test individual algorithms one at a time, then combine

For AI pretend that you are a computer and draw a scenario (literally), and scroll through the scenario like a computer and decide what to do like the computer. I have found AI way!!! easier in Python
