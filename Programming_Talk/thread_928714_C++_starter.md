Before anything, what's your programming experience? If you have some, go for it, but if you happen to be a newbie, then many here and myself will recommend you to learn something simpler and more intuitive. The most popular thing here proposed for that cases is Python (see the Sticky too)... also it comes installed by default in Ubuntu (open a Terminal and type python... there you have it: an interactive shell to play around).

(And if you have programming experience, but don't know Python, consider to learn it!)

Anyway, for C++ or any language, Gedit (GNOME's default editor) is a nice start; look at its plugins and you'll love it. Unless you need to do some advanced stuff, don't consider the "major" editors (vim, emacs). I don't know anything about Kate (KDE's editor).
