### Post by snova on 2009-03-05
> **Johnsie said:**
> I'm doing some projects for work and I would like a system like launchpad so that users can submit bug reports and make requests over the lan. Does anyone know of any such php-based systems that can be run locally off our Ubuntu server?

Launchpad is a lot more than bug reporting, and there are probably only a handful of things that can compare to its full functionality. For that, you could look at GForge, but such installations are difficult to set up.

But if you only need something for bugs, there are [countless]("http://en.wikipedia.org/wiki/Comparison_of_issue_tracking_systems#Bug_tracking_systems"). In particular, I might suggest Mantis. (BugZilla is Perl).
