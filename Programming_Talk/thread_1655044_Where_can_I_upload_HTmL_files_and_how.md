### Post by PaulM1985 on 2010-12-29
Hi

Once you have finished your HTML website and want to be able to show it to other people, you need to put it on a server that people can access (this can be your own server, or someone elses, look up "web hosting"), which can be accessed via a web name (eg [www.ThisIsAsSiteIMade.com](www.ThisIsAsSiteIMade.com)).

This is pretty much how I first got into programming.  I put my demo websites on geocities, which seems to have been taken down. If you look on google for "free web hosting", you will probably get lots of results, so scan through and try and find something which is actually free rather than claiming to be free.  I haven't done this in a long while so unfortunately I can't recommend anything, but other people on this forum might be able to.

I think these provide free hosting and I remember old friends using it when they were starting out: [http://angelfire.lycos.com/](http://angelfire.lycos.com/)

EDIT: Also, I am not sure how much of a beginner you are, so sorry if you already know this, but you do not need to actually upload your webpages to view them while you are creating your site, you only need to browse for them on your computer using your web browser, clicking "File->Open File" etc.  Uploading is so that other people can access what you have done.

Good luck

Paul
