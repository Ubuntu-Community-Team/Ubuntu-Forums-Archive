### Post by Corvillus on 2006-03-25
There's also the Eclipse WTP package, which has JavaScript, (X)HTML, XML, and CSS editors, as well as a bunch of J2EE related stuff, you can find that at [http://www.eclipse.org/webtools](http://www.eclipse.org/webtools) . Zend also just released a PHP IDE to use with the Eclipse WTP as well, which you can find at [http://www.zend.com/phpide](http://www.zend.com/phpide) . That said, Eclipse is probably a bit overkill, and you could easily get by with a syntax highlighted text editor.
