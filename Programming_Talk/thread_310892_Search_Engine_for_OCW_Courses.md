   Check out this new search engine which indexes OCW courses. For the uninitiated, OCW stands for OpenCourseWare - This is an initiative from MIT which has been picked up by many universities across the world. The objective is to put as much of open courses online for the access of general public.

Search Engines like OpenContentOnline provide an interface to search for OCW courses.
Example Query: Artificial Intelligence
[http://www.opencontentonline.com/search.php?query_text=artificial+intelligence](http://www.opencontentonline.com/search.php?query_text=artificial+intelligence)
