### Post by nvteighen on 2009-08-25
Hm... It depends on what you want to accomplish.

If you want to be a professional programmer, I think you should get a CS title in university... Not because of what you may be taught, but actually because of certification. If you want to program as a hobby, I guess tutorials and books will be o'right.

Of course, you need a mathematical mindset. This doesn't mean you have to be a maths guru, but have good logic skills and specially, know how to order ideas in a formal-logical way.

But you also need other skills. The one of learning English is a very important one if you're not a native speaker. Like it or not, English is the vehicular language of computer science... and to be a proficient programmer you need to be proficient at communicating with other programmers. Otherwise, you'll be pretty lost.

But also remind that programming is a technical skill, which is learned by doing things. So, to be a programmer, code, code, code! (and ask!)
