### Post by nvteighen on 2009-07-06
I've only used Obj-C... but of course I've seen and read and know about C++. IMO, you can't compare them. Obj-C is designed to be "C with Classes", while C++ currently tries to go very far beyond that. Also, in Obj-C there's the de facto dependence on the OpenStep libraries, which aren't standard at all but without them you're pretty much with C's Standard Library and some nice OOP syntax... To enjoy Obj-C programming, you're forced to use OpenStep.

Anyway, Obj-C is a really clean language. I enjoy it a lot.

And as Steve Jobs knows what he does, the Cocoa API is in Objective-C... so he can promote "his" language. Result: people tend to relate Obj-C to Mac.
