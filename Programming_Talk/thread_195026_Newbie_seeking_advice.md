### Post by Zed on 2006-06-14
Here are some links that may be of use:

[How to become a hacker](http://www.catb.org/~esr/faqs/hacker-howto.html) (pay attention to the Other Resources list, too)
[Teach yourself programming in 10 years](http://www.norvig.com/21-days.html) (don't let the title discourage you -- the point is that acquiring depth and breadth take some time, not that you can't do fun and interesting and useful things during that time)
[How to Design Programs](http://www.htdp.org/) (an intro to programming using Scheme)
[How to Think Like a Computer Scientist](http://www.ibiblio.org/obp/thinkCSpy/) (an intro to programming using Python)

### Post by IYY on 2006-06-16
Which languages do you use in Windows? What kinds of things do you want to do in GNU/Linux? 

If you want to learn the art of making graphical user interfaces, have a look at the gtk tutorials (on the official gtk site). 

If you just want to learn to code in C, grab a copy of "The C Programming Language", no doubt the best book out there on the subject.

Learn how to write at least basic shell and perl scripts. It's very easy, but also very handy.

Learn about make files. Again, very simple but you must know this.
