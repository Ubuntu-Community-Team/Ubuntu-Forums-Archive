What samjh said.

Really can't emphasize enough that you learn C or C++, *every* language you could possibly learn is built on top of one of those two languages.

I have used the same tutorial samjh linked at: [http://www.cprogramming.com/tutorial.html#ctutorial](http://www.cprogramming.com/tutorial.html#ctutorial)

It's incredibly easy to use, it teaches you the same concepts you already know in other languages:

- variable types, and making your own types
- functions, how to call them and how to write them
- user input/output
- flow control(if/when/for)
- file i/o

Those may seem like simple concepts, but once you learn to apply them in C or C++, your understanding of abstracted languages based on them(python,ruby,C#,etc) will skyrocket.  Samjh is correct, once you learn C, the rest will be simple and intuitive to pick up.

Best of luck to you in your projects.
