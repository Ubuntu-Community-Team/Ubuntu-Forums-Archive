### Post by Delever on 2008-10-31
I have said where you should start, it is very good to get to know simple concepts.

Now some more information:

Apache is very popular web server. For example, this forum is running on apache like many other sites. To install apache on your computer, simply type in terminal

```
sudo apt-get install apache2
```

it will be downloaded and installed.

To see how your computer now is able to serve you web pages, open [http://localhost](http://localhost) address. "localhost" is your computer. You should see some positive text in your browser window.

Now, to find where this text comes from, go to /var/www. This is default location where apache keeps web site. There should be file index.html, it is displayed by default, or if you enter exact link to it, like this: [http://localhost/index.html](http://localhost/index.html).

Open it, edit it, save, then reload browser page. You should see your changes in browser. This is called static content.

Now you should get familiar with html.

Thats all for now, good luck.
