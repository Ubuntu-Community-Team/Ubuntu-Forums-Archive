### Post by spjackson on 2012-09-09
I'll stick my neck out and say now is the time to learn Python 3, but I'd also recommend you to acquaint yourself with the differences from Python 2. Lots of Python 2 examples and useful snippets will exist for years to come.

### Post by trent.josephsen on 2012-09-09
I think that is bad advice. There's no reason not to learn Python 3 now; there are plenty of excellent learning resources (*Head First Programming* comes to mind) and Python 2 is surely on its way out. Many (most) of the more cutting edge distros like Arch and Gentoo already have Python 3 as the "default" Python. Python 2.7 was the last major release in the Python 2 branch and that was over two years ago. It's only a matter of time before your Python 2 skills are obsolete, and Python 3 can be used immediately.

That's if you are just learning and have no particular preference. There are still good reasons to use Python 2 -- you might find yourself writing programs for a platform that you can't control, and there are still a few Python libraries that don't work with the 3.x branch. But I see no reason to recommend 2.x across the board.
