### Post by r-senior on 2012-02-05
You can just download Indigo from Eclipse:

[http://www.eclipse.org/downloads/packages/release/indigo/r](http://www.eclipse.org/downloads/packages/release/indigo/r)

Unpack it under your home directory, e.g. under ~/Software and you are ready to go (assuming you already have a JDK installed). Just run the 'eclipse' executable.

Create a .desktop file in ~/.local/share/applications if you want it to appear as an application shortcut.
