### Post by hayati on 2009-01-26
Hi,

when i want to add qt4 code completion database for a qt4 project in kdevelop, it says it is created but in editor code hints don't appear either i push ctrl+space or while typing. This wasn't the case when i used ubuntu 7.04 or 7.10. I faced with this problem with ubuntu 8.04 (which is a lts). Only different thing is the distribution. All other things (compiler, qt version, enviroment, kdevelop version 3.5.4) are the same. As a symptom: code complete wizard rapidly finishes its work, which means it doesn't enter sub directories i guess.

Any suggestions are wellcome.

Ok i solved the problem.
Qt4 code complete database wizard sucks for ubuntu 8.04. i used custum pcs importer. And edited the pattern to include so that it can handle file names which don't ends with "h". then it's ok now.
