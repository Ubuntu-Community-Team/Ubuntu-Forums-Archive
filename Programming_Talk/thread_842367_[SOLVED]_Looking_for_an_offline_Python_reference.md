### Post by WW on 2008-06-27
Ubuntu also has the book "Dive Into Python" available as the package [diveintopython](http://packages.ubuntu.com/hardy/diveintopython).  If it is not already installed (it is not installed by default on my server; I'm not sure about the desktop), you can get it with:
```

$ sudo apt-get install diveintopython

```
