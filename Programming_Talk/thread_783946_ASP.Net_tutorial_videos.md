### Post by twothird on 2008-05-06
Found a website that caught my interest called [Code Chronicles]("http://www.microsoft.com/uk/codechronicles/default.mspx#"). The way it caught my attention was a bit false though – at first I thought it was a guide through the history of programming languages. Turned out to be something totally different – a fancy suasion for developers to learn ASP.Net.

What was rather annoying was that there’s no information what so ever about other languages, not related to ASP.Net. I think that’s a bit hypocritical, considering the site’s title. Anyway, as soon as one realizes what it’s about, one can skip the not important parts, because there are some actually useful bits. The download section is cool with Express Editions of Visual Web Developer 2008 and SQL Server. What I liked more was the support section. There are quite a few instructional videos, for all levels of proficiency in web-development, even some from a PHP developer’s perspective. The whole site is made in Silverlight, which caused some problems with me – had to watch the videos in fullscreen. Overall, I think the site is worth seeing, but only if one sticks to what’s worthy.
