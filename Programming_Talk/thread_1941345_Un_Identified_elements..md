Before you can use a library, you will need to install it (preferable using Ubuntu's packaging system) *and* make sure that when compiling, you tell the compiler and linker where to find the header files and libraries. You appear to have skipped one of these steps.

The Tutorials on the wxWidgets page should help you: [http://wiki.wxwidgets.org/Guides_%26_Tutorials#Linux](http://wiki.wxwidgets.org/Guides_%26_Tutorials#Linux)
