### Post by edd07 on 2008-11-25
Yes!

MySQL Workbench for Linux is finally out!! I used it on Windows for a while and have been waiting for the Linux version to come out. It's still an alpha, but it has worked very well for me.

Install it by adding this repo to your third party sources:

```
ftp://ftp.mysql.com/pub/mysql/download/gui-tools/ubuntu/ binary
```

and installing the package (I think its called mysql-workbench or something like that)

Hope this helps
