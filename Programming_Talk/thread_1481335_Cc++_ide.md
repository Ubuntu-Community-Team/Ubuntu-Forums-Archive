### Post by trent.josephsen on 2010-05-12
I would encourage you to start with just an editor and invoke gcc from the command line.  IDEs contain many abstractions that hide what's actually going on from the user -- useful for some enterprise level projects, but detrimental to learners.  But that's just my opinion...

I've heard good things about Eclipse, though.
