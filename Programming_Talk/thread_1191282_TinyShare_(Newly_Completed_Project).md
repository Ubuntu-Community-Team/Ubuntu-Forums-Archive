Hi All,
I wrote a small python web app which acts as a nicer/richer file index for Apache. Currently I have it hosted on my desktop:

[http://curved.ath.cx/tinyshare](http://curved.ath.cx/tinyshare)

Basically, if you're like me and have DDNS pointing at your WAN IP and have your router forwarding port 80 to your desktop, it makes it much easier to share pictures and other stuff with friends.

On an aside, can anyone mirror it? I'm going to turn my computer off tonight, and I don't really have anywhere else to put the demo site. It requires mod_python and .py set as an apache index file type.

Thanks!

Edit: TinyShare is a lot like Opera Unite ([http://unite.opera.com](http://unite.opera.com)).
