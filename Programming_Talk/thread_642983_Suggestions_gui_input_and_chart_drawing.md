### Post by RHTopics on 2007-12-17
Bob,

Have you considered using the macro and programming features available in OpenOffice.org?

It is relatively easy to get started with it.  The help documentation that comes installed with OpenOffice.org in Ubuntu provides some simple gui examples.

As you become more proficient with its programming environment, it is certainly possible for your programs to call the built-in objects in OpenOffice.org to present charts.

And since OpenOffice.org runs under Windows, the OpenOffice.org programs/macros you develop should run under Windows as well.

Just a thought.:)
