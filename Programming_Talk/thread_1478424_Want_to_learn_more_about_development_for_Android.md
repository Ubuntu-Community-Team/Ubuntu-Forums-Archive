### Post by Axilus on 2010-05-11
> **Kdar said:**

Does NetBeans have plug in as well?

Yes, It is called nbandroid. You can find the instructions on how to install it [here]("http://kenai.com/projects/nbandroid/pages/Install").

As much as I prefer to use Netbeans over Eclipse, it is only fair to point out that the ADT plugin for Eclipse has far more support that the plugin for Netbeans.

It really all depends on what IDE your most comfortable with using. I was just recently a windows guy so I'm more familiar with a Visual Studio type layout (which Netbeans provides); however if your all for the community support then Eclipse is where its at for development (although from my experience it is quite buggy).

I use both of them though depending on what I'm doing.

UPDATE:
