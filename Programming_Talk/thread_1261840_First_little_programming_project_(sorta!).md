### Post by FokkerCharlie on 2009-09-09
Hello!

A little while ago, I decided I'd have a go at learning some Python- as a programming exercise, I decided to write a small app to quickly disable and re-enable the screensaver; it's a handy thing to have for when I want to watch a DVD or something.

It also included a few elements of coding that I wanted to try- threading, a notification-area icon and menu, interaction with the OS, gtk window.  I also wanted to try out making a .deb.  So that's all complete, I thought that it might be useful for other Ubuntu users, too.

It's uploaded to:

[http://www.mediafire.com/file/kkmjkxyytnz/screensaver-inhibit_0.1.deb]("http://www.mediafire.com/file/2a2j1yzdknr/screensaver-inhibit_0.1-1_all.deb")

Source code here:

[http://www.mediafire.com/file/tjzmwximzgz/screensaver-inhibit.py]("http://www.mediafire.com/file/tjzmwximzgz/screensaver-inhibit.py")

I put the deb together on my Jaunty 64 system, but there's no reason why it shouldn't work on any architecture.  I'd be very interested to hear from anyone who fancies trying it out, whether it works smoothly, and feedback in general.

Cheerio
Charlie
