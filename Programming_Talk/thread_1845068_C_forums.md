### Post by Arndt on 2011-09-16
> **cap10Ibraim said:**
> Hey there 
Does any one know about any good quality (advanced topics) forums discussing C Language ?

There is still the old-style Usenet news group comp.lang.c, which you can read using Google groups:

[http://groups.google.com/group/comp.lang.c/topics](http://groups.google.com/group/comp.lang.c/topics)

It used to be the number one forum for C before the web. If you can stand the spam, which seems to make up half of it nowadays, it's probably still useful. I don't follow it anymore (last time I tried, there were some very opinionated persons dominating - this may have changed), and now when I looked, I see that one of the biggest authorities there, Chris Torek, doesn't post there anymore, but if you search for old posts about a certain topic and the author Chris Torek, you can't go wrong.
