### Post by r-senior on 2013-06-02
You won't find an exact match, but you should look at these:

Code::Blocks
Eclipse
Netbeans
Anjuta
Geany

You should also browse the Software Centre under the development and programming section.

Or you could just go with a good editor and build tools that you run from the command line, e.g. GNU Autotools or CMake. It's worthwhile for every C/C++ programmer to understand how to build manually, so do try and understand gcc/g++ compilation whatever you choose.
