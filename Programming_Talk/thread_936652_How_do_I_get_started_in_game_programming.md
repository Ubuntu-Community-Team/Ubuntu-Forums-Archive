### Post by Calmatory on 2008-10-03
Why don't you learn how to use sockets and write a simple text based multiplayer game? E.g. Game where the idea is to gain experience and advance in levels. Players would have stats and skills, could equip weapons and armor, use magic etc. It could be real time or turn based. Either completely text driven or with ASCII graphics like in ADOM or nethack.

This would be a complete game project for you. An programming exercise and it would teach you about sockets and data handling which could become handy in "real" game developing.

Suer text based game isn't attracting people like graphical games are, but eventually some people would try it out, and even some could stay.

Actually, my own long-time goal has been such a game. Not for people necessarily, but for myself. Something I want to do some day, may others like it or not.
