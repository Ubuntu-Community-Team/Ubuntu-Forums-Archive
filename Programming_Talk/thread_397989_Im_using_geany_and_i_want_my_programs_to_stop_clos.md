### Post by samjh on 2007-04-01
First, open up Synaptic, and use the search function to look for "geany" (without quote marks).  If it shows geany installed, right-click on it and select Remove.  Click the Apply button on the toolbar to do the removal.  Shut down Synaptic when finished.

Click this link to download the Geany 0.10.2 deb package (you can find this by Googling the Geany website):
[http://gauvain.tuxfamily.org/repos/dists/edgy/contrib/binary-i386/geany_0.10.2-0ubuntu1_i386.deb](http://gauvain.tuxfamily.org/repos/dists/edgy/contrib/binary-i386/geany_0.10.2-0ubuntu1_i386.deb)

After saving the file to your desktop or home directory, double-click it to open it using the Deb package installer.

Click the Install button.
