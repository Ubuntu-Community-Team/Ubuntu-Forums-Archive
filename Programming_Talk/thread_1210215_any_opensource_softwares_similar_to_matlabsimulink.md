### Post by ve4cib on 2009-10-27
For a while at work several years ago we ran into a licensing issue with Matlab, so I had to use Octave for a few weeks while we sorted that out.  The two languages are practically identical, and a lot of Matlab libraries can be used without modification in Octave.

Octave is missing a decent IDE (for those who want that sort of thing), though [Octave Workshop](http://www.unige.ch/math/folks/loisel/www.math.mcgill.ca/loisel/octave-workshop/) isn't too bad.  Some of the keyboard shortcuts are screwy (who decided that Ctrl-A should be "Save As" in a text editor!?), but that's easy to fix with a few tweaks to the code and a recompile.
