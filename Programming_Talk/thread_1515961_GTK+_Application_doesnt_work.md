### Post by dwhitney67 on 2010-06-23
I ran Glade to create a LibGlade project, with version 2.16 of the toolkit (I did not experiment with other versions).

Anyhow, I added a window and a button to the project, then saved it.  The resulting file was aptly named "mygui.glade".

From there, I created a "mygui.xml" file using the following command:
```

gtk-builder-convert mygui.glade mygui.xml

```

From there, my code:
```

#include <gtk/gtk.h>

const char* GENERATED_GUI = "mygui.xml";
const char* MY_WINDOW     = "window1";

int main(int argc, char** argv)
{
   gtk_init(&argc, &argv);

   GtkBuilder* builder = gtk_builder_new();

   gtk_builder_add_from_file(builder, GENERATED_GUI, 0);

   GtkWidget* window = GTK_WIDGET( gtk_builder_get_object(builder, MY_WINDOW) );

   gtk_builder_connect_signals(builder, 0);

   g_object_unref( G_OBJECT(builder) );

   gtk_widget_show(window);

   gtk_main();

   return 0;
}

```

Then I built the project using something like:
```

gcc `pkg-config --cflags gtk+-2.0` GTK_Example.c `pkg-config --libs gtk+-2.0`

```
If you are using C++, use 'g++' instead, and of course name your source file with a .cpp extension.
