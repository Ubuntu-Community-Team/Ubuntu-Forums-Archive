### Post by n~kf)}BW% on 2010-12-30
Bash scripting is in fact a bunch of Linux/Unix programs and their parameters put together in shell scripts.

If you want to learn bash scripting, you should learn about common GNU programs and basic shell commands.

I think you can learn to script in bash without reading books, it's just programming logic connected with all these Linux programs...

Just Google for some example scripts, there are plenty on the net.

This is a nice tutorial:
[http://linux.about.com/library/cmd/blcmdl1_bash.htm]("http://linux.about.com/library/cmd/blcmdl1_bash.htm")

And also some example sites:
[http://360percents.com]("http://360percents.com")
