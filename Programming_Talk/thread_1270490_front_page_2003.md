### Post by nvteighen on 2009-09-20
Hm... Those are two different and separate issues. One is the way you generate your page and another is the need to update your server.

Front Page and usually all HTML-related software by Microsoft use non-standard stuff meant to be optimized only for IE. Of course, using Front Page 2003 will lead to IE 6 optimized code... :P

The best is to learn XHTML 1.1 Strict. Whether to use an IDE or not, well, it's a dispute we have at these forums a lot. But what's sure is that you have to know (X)HTML in order to fix things manually...

Now, **my** preference is to write the HTML all by hand in a text editor.
