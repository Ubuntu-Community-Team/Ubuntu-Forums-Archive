### Post by lykeion on 2009-12-03
To start learning Java I'd suggest you just jump right into it and begin coding. Follow a "Hello World" example and compile it. Then move on towards more complex examples.

If you are more of a "reading" person I can suggest this free book: [Introduction to Programming Using Java]("http://math.hws.edu/javanotes/index.html")

If you need more help getting into it, just ask.
