### Post by kefler on 2007-11-15
just a suggestion, if you want to load a series of random/ordered text strings, take a peek at loading xml files into flash. Doing so, you'll avoid having to use one long line of variables and values, and the parsing code in flash if you were to have it on seperate lines since flash already fully supports the loading and parsing of xml files. I did something like this at work on the fly without even knowing how to program in flash and it was very simple to do.. of course I do have programming background in other languages but I'm assuming you do as well so it should be just as easy for you. The source code I referenced was very much like the code I had done in javascript so I had a head start on loading the xml. 

I know one of your requirements is to be able to change that text file through a php app, I took this into account as well, the flash object I made was a news ticker which read from an xml file, the xml file was generated by a php script which took a listing of all the PDF articles listed on the current news section.
