### Post by paulocoghi on 2010-12-07
XML Copy Editor is a fast, free, validating XML editor.

If you have tried and failed to install it on your Ubuntu, don't worry.

The version available on the official website XML Copy Editor use an old version of "libxerces" and this one, in turn, depends on an old version of "libcu".

So, what you should do is not to download the version available on the official site.

**Step 1:**
Open the Ubuntu Software Center

**Step 2:**
Type "XML Copy Editor" or "xmlcopyeditor" on the search bar.

**Step 3:**
Click Install.

That's it!

--

**[COLOR="red"]Alternative steps:[/COLOR]**

[COLOR="Red"]If you can not install[/COLOR] using the steps above, perform the following steps:

**Step 1:**
Access [COLOR="Blue"][http://packages.ubuntu.com/karmic/i386/libicu40/download](http://packages.ubuntu.com/karmic/i386/libicu40/download)[/COLOR]
Download the 'libicu40_4.0.1-2ubuntu2_i386.deb' from your preferred mirror and install it (double click).

**Step 2:**
Access [COLOR="blue"][http://packages.ubuntu.com/karmic/i386/libxerces-c3.0/download](http://packages.ubuntu.com/karmic/i386/libxerces-c3.0/download)[/COLOR]
Download the 'libxerces-c3.0_3.0.1-2_i386.deb' from your preferred mirror and install it (double click).

**Step 3:**
Access [COLOR="blue"][https://sourceforge.net/projects/xml-copy-editor/files/xmlcopyeditor-linux/1.2.0.6/xmlcopyeditor_1.2.0.6-2_i386.deb/download](https://sourceforge.net/projects/xml-copy-editor/files/xmlcopyeditor-linux/1.2.0.6/xmlcopyeditor_1.2.0.6-2_i386.deb/download)[/COLOR] to download the latest version of XML Copy Editor
And double click to install.

That's it!
