### Post by leg on 2010-12-06
I have used the Netbeans designer in the past and I think it is ok and I don't know of any other designer anyway. I am currently using eclispe for Java programs and do not bother with a GUI designer at all and just write the code. The biggest problem I had was which sections you shouldn't change as the designer will always change them back to its template. If you get the hang of it it isn't that bad.

Don't know if you found them but these are the [Netbeans Tuts]("http://netbeans.org/kb/trails/matisse.html") on their site.

Don't know what these are like but I have found two:

[GUIDE]("http://www.mindsilver.com/")
[JGuiD]("http://sourceforge.net/projects/jguid/")

A couple of quick searches seem to suggest that a lot of people have high regard for the Netbeans designer.
