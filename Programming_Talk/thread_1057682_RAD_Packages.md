I am afraid you wont be able to find something as easy and well made as Delphi, in Linux ,Windows or MACOS. DELPHI pretty much defined the real meaning of the word "RAD". 

But there some that come pretty close. Lazarus is still struggling to be an worthy Delphi-clone, NETBEANS is quite good as well, gambas have not tried it but seems very promising . I will add to the list Eclipse. None of the above reach the ease of use and efficiency of Delphi but they are quite close.

I have searching for Delphi replacement myself as soon as I started using UBUNTU. I have been a Delphi developer for a very long time but I recently abandoned Windows in favor of mainly MACOS and UBUNTU.  I considered many choices. The one that I personally think comes the closest to Delphi and I use right now is Jython with Eclipse.

Jython is basically python that can call Java libraries. Java has loads of competent libraries and is the most well behaved language for cross platform apps. It has also JAVABEANS which is rip off of Delphi components and also the python syntax of Jython ( which is basically python syntax) reminds alot of Delphi syntax .
