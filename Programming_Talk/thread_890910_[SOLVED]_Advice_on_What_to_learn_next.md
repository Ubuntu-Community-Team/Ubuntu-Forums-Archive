### Post by slavik on 2008-08-15
most commonly used languages in the open-source world?
C is by far the most used. But it isn't the only one, there's Perl, Python, C#, Java and many more.

Since you already got the hang of programming, my recommendation is to learn C, then Perl/Python/Ruby. There are people who recommend learning Python then C (I disagree with them).

If/when you decide to learn C, I suggest getting the K&R book (The C Programming Language, 2nd ed. by Kernighan and Ritchie). It has a tutorial introduction of C and is a very good reference for the C standard library.
