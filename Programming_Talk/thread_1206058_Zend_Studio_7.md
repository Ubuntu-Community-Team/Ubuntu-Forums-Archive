### Post by Mirge on 2009-07-06
I created a .html file, which includes PHP code within it. Is there a way to make it use syntax highlighting for the PHP code as well, not just the HTML markup?

Thanks...

---

### Post by Mirge on 2009-07-06
For those who may run across this issue and find it as irritating as myself:

Window -> Preferences -> General -> Content Types

Find "PHP", click "Add".. and add an entry for *.html. It will rebuild your workspace (well, mine did)... and then close/reopen any .html files you had that didn't have PHP syntax highlighting, and it will when the file(s) are re-opened.

Enjoy.
