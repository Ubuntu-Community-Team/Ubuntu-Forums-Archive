### Post by pmasiar on 2007-10-23
Python comes with many standard modules preinstalled. .tar,  .bz2 or .rpm are not python modules - they are compressed packages, might be source code of applications written in Python or any other language. 

Better don't use sources (unless you know what you are doing, and how to if it if it breaks): install applications using synaptic, so you can get updates, and it will not break dependent libraries.
