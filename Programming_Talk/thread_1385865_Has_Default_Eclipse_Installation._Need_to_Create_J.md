### Post by navneeth on 2010-01-20
I got help at the #eclipse IRC channel. 

There were no other sources listed in *Help->Install New Software...*, so I had to add this: [http://download.eclipse.org/releases/galileo](http://download.eclipse.org/releases/galileo) (my installation being 3.5 Galileo). Choose the site from the drop-down list and then check WEB, XML, and Java EE Develpment from the list below and proceed to install.
