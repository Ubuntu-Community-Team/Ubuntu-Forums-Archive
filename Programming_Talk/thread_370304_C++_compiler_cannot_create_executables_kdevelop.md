### Post by lnostdal on 2007-02-25
right, the problem is that you have somehow specified that the compiler/host to use is `i586-mingw32msvc-c++' which is a special version of gcc that produces windows-binaries/programs (yes, on or from linux) which you have not installed

you probably want to specify somewhere that you want to use gcc or g++ as host/compiler
