### Post by papibe on 2016-09-11
Hi  rakeshvz. Welcome to the forum ):P

You need to use no spaces, separate paths by colon, use $ for the current PATH, use $HOME instead of ~, and wrap it all up on double quotes. For example, to add ~/tools and ~/arm-linux-x64-bian to your path:
```
export PATH="$PATH:$HOME/tools:$HOME/arm-linux-x64-bian"
```
Here are more examples: [how-to-add-a-directory-to-my-path]("http://askubuntu.com/questions/60218/how-to-add-a-directory-to-my-path"), and  [EnvironmentVariables]("https://help.ubuntu.com/community/EnvironmentVariables").

For the changes to make effect, run this:
```
source ~/.bashrc
```
Hope it helps. Let us know how it goes.
