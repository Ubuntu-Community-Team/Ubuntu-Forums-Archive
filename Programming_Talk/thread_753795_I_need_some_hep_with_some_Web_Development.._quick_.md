### Post by Zugzwang on 2008-04-13
Dear OP, it's hard to guess what you're actually meaning since you're comparing two different applications. 

The guitar tuner would require capturing audio whereas what you're describing seems to me to be an application which shows a fingerboard on which you can click to hear the tone that playing the string with the finger put there would produce. Right?

Just a note: For a tuner, there's a lot of complex math involved!
