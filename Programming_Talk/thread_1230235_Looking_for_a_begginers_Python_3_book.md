### Post by Copernicus1234 on 2009-08-03
If you are the kind of guy who wants to read a book to learn, I can recommend Learning Python by Mark Lutz. Its really good but doesnt cover python 3.

My edition is a year old or so, but there is a new edition coming in October ([http://www.amazon.com/Learning-Python-Mark-Lutz/dp/0596158068/ref=sr_1_1?ie=UTF8&s=books&qid=1249296418&sr=8-1](http://www.amazon.com/Learning-Python-Mark-Lutz/dp/0596158068/ref=sr_1_1?ie=UTF8&s=books&qid=1249296418&sr=8-1)). It will most likely support python 3.

There is also a web based book called Dive Into Python in progress for python 3 ([http://diveintopython3.org/](http://diveintopython3.org/)). If you are a new programmer, it may be a tad too difficult however.

The changes between version 3 and earlier versions are not very noticable for the programmer, so I think you should start playing with version 2.6....
