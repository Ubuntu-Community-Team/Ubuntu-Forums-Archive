### Post by samjh on 2008-08-10
First of all, Java is not Javascript.  Despite the similarity in their names, they have practically nothing to do with each other.

The Ubuntu project itself does not use Java, as far as I'm aware.  However, there are many software packages which use the Java programming language and platform, which you can help by triaging bugs, submitting patches, etc.

Go to Launchpad.net, register yourself if you haven't already done so, and take a look at the various Java-related packages on there.  See if some maintainers will mentor you.

There are various teams which focus on Java too:
[https://launchpad.net/people/?name=Java&searchfor=teamsonly](https://launchpad.net/people/?name=Java&searchfor=teamsonly)

Lots of Java-related projects:
[https://launchpad.net/projects/+index?text=Java](https://launchpad.net/projects/+index?text=Java)

Mentoring available:
[https://launchpad.net/ubuntu/+mentoring](https://launchpad.net/ubuntu/+mentoring)
