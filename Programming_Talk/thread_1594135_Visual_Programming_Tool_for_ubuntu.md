### Post by DJonsson2008 on 2010-10-12
I think there are python and c++ development 
environments that approximate the 
visual programming experience. Wingware is a commercial 
version of this. There are open source equivalents. 
I've settled for gedit and ipython. Gedit is very flexible
and can be set up as a good programming tool with some effort.
ipython is a command line interpreter. 

I've looked at several more complex IDE's but they all come
with a learning curve hard to justify with my immediate 
productivity demands. As well some of the open source tools
I tried were abandoned not-maintained projects. 

There may be some new IDE's out there if you find something
elegant, scalable, sustainable and powerful for Python please
let me know. One thing I miss is a good object browser gui,
if I could find one that could be triggered from the python
command line, it would be a leap forward.
