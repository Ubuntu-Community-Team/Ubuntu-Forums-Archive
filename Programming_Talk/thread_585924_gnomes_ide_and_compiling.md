### Post by jespdj on 2007-10-24
Try this:
```
sudo apt-get install gnome-devel
```
This will install a bunch of tools useful for GNOME development, including the Anjuta IDE.

See: [Installing Gnome Development Tools](http://ubuntufs.wordpress.com/2007/05/25/installing-gnome-development-tools/)

> **slavik said:**
> there is also Kdevelop and ...
Kdevelop is for KDE primarily and is not optimal for GNOME development.
