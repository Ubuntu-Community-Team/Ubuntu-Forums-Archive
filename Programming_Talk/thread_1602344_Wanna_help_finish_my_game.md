### Post by dv3500ea on 2010-10-23
You can check out the whole repository with this command:
```
svn co https://la-vida.svn.sourceforge.net/svnroot/la-vida/ la-vida
```
or just the most recent branch with this command:
```
svn co https://la-vida.svn.sourceforge.net/svnroot/la-vida/branches/JDShu la-vida
```
Which I recommend because it reduces the size of the download. (Which branch is the most recent will obviously change in the future).
