### Post by tinny on 2008-07-13
It will be interesting to hear how you get on with this project.

If I where you I would start by having a look at the source code of a project like [Pidgin]("http://sourceforge.net/projects/pidgin/"). Pidgin is an IM application that can interface with MSN. It is not written in Java but should give you a sort of bottom up approach to understanding the MSN protocol.

Also have a look at this.
