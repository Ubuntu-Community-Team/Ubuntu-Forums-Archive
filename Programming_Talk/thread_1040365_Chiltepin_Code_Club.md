A good friend of mine and I have created a relaxing social site for Developers and Coders to meet and share ideas and code. The site is free for everyone. You are all  welcome to create your own tutorials using Courses and make forums, blogs and wikis as well as share images and videos linked from TAGS. The entire site is built on Moodle so its pretty sweet and fun to use. We are fairly new to the Moodle application so if you have any issues let us know. We plan to keep the Chiltepin Coding Club Open source and Dynamic for the users to Chiltepin Code Club utilize it as they see fit. Here's the link if you would like to give it a try:
