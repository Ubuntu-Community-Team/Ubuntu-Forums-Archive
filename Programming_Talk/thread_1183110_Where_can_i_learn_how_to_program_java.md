### Post by achelis on 2009-06-10
If you already know C# it should be fairly easy for you to learn Java. The Javadoc for the API is a good place to learn I think, since one of the big differences between C# and Java is the API's. Apart from that it's really just about learning to make good and well-structured object-oriented programs - and that applies to both C# and Java.

As for making games I'd recommend looking into LWJGL which is (among other things) an OpenGL binding for Java. Whatever OpenGL-specific stuff you learn you can also apply if you decide to learn C or C++ one day.
