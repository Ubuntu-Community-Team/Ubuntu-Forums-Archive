### Post by lnostdal on 2008-04-06
you're looking for comp.lang.lisp

you can either use an NNTP/NEWS/USENET provider and a news-reader (like, say, Thunderbird) to browse the forums, or you can use a web interface like Google Groups, here: 
  [http://groups.google.com/group/comp.lang.lisp/](http://groups.google.com/group/comp.lang.lisp/)

if you're on IRC join #lisp (on the freenode servers) .. i hang there (my nick is lnostdal) .. we are quite a few people there .. at the moment #java has 250 participants, while #lisp has 265 .. so "nobody" ain't exactly how it actually is .. lisp has a huge community consisting of (often) very smart people .. edit: also check out [http://planet.lisp.org/](http://planet.lisp.org/) .. there are some interesting posts there, sometimes
