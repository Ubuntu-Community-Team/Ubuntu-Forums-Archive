### Post by thornmastr on 2011-02-18
Copy your py files to Winblowze. Use py2exe to generate your distribution Windows Directory. Use Inno to generate your install program. Slap it into a Winblowze drive and install it. Then test to make sure it works nicely under M$ banner.
