### Post by noup on 2006-09-02
For anyone who's interested, the first version of [Gnome Subtitles](http://gsubtitles.sourceforge.net) has been released.

Gnome Subtitles is a subtitle editor (SubRip, MicroDVD, ...) for the GNOME Desktop that supports subtitle editing, conversion and synchronization. Screenshots are [available](http://gsubtitles.sourceforge.net/screenshots).

Ubuntu packages aren't available yet, so any support on that would be really appreciated.
