### Post by t1497f35 on 2011-06-16
Hi,
I just discovered Qt Creator (v2.2), so far it seems the most (and only) professionally looking (C/C++) source code editor for Linux, I'm impressed.

1) Is there a "delete current line" keyboard shortcut? usually it's either Ctrl+D, Ctrl+E or Ctrl+K.

2) Is there a "format source code" option? NetBeans (and I think Eclipse) have such options.

---

### Post by mourt on 2011-06-29
1. There is no default shortcut for it, but you can assign one by yourself: Tools -> Options -> Environment -> Keyboard, type something like "Line" in the filter field, select the "Delete Line" entry, and add any shortcut you like.

2. There is no "reformatting" of the code as a whole, but you can have it re-indented with Ctrl-A (to select all) plus Ctrl-I (re-indent selection)
