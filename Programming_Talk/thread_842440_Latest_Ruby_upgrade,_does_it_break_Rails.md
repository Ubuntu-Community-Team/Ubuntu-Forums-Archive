### Post by pdwerryhouse on 2008-06-30
As long as you're upgrading within the same version of Ubuntu (eg, you're running Hardy, and you're installing an update from the hardy updates repository), then there shouldn't be any operation changes to the package.

That is, updates fix bugs, but don't change functionality. Hence, you can reasonably expect updates not to break anything, and if they do, that's a bug in itself.
