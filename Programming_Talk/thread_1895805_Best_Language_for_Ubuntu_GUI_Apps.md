### Post by karlson on 2011-12-15
There is no such thing as best.  There are 3 major GUI toolkits available for Linux:  QT, GTK+, wxWidgets.

There are many languages that have the ability to use these toolkits.
The primary ones people use are: C++, Python, C

There are programs written in Haskell. [http://xmonad.org](http://xmonad.org)
There is perlQt if you choose to use it.
Off the top of my head I can't come up with Ruby Projects but I am sure there are some.
There is C# with Mono

So pick your poison and go forth...
