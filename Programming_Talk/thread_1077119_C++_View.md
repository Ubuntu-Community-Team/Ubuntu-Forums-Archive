### Post by geirha on 2009-02-22
If you open a new tab, the empty file it has does not yet have a name (just Untitled1 or something like that), so gedit doesn't know what type of file you want it to be. Once you've created a new tab, save (ctrl+s) the empty file with a name ending with .cpp, and it should start highlighting c++ syntax.
