I'm very bad at math,
can anyone please help me solving this:
100 / x = 75
what are the operations to get the x value?

---

### Post by MadCow108 on 2010-03-18
multiply both sides with x:
100 * x / x = 75 * x

divide both sides with 75:
100/75 = x
