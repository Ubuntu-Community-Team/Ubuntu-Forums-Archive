### Post by Vaphell on 2014-08-27
these functions should do the trick (in 2 flavors just for kicks: native bash and python lib)

```
#!/bin/bash

#native bash
uri_encode() {
  local length="${#1}"
  for (( i = 0; i < length; i++ ))
  do
    local c="${1:i:1}"
    case $c in
      [a-zA-Z0-9.~_/-]) printf "$c" ;;
      *)               printf '%%%02X' "'$c"
    esac
  done
}

uri_decode() {
  local uri="${1//+/ }"
  printf '%b' "${uri//%/\x}"
}

# call to python lib
py_uri_encode() {
  python -c 'import sys, urllib as ul; print ul.quote(sys.argv[1] )' "$1";
}


py_uri_decode() {
  python -c 'import sys, urllib as ul; print ul.unquote(sys.argv[1] )' "$1";
}


test="/home/zomg/u wot m8.jpg"

#bash
uri="file://"$( uri_encode "$test" )
path=$( uri_decode "$uri" )
path=${path#file://}
echo "bash: $uri -> '$path'"

#python
uri="file://"$( py_uri_encode "$test" )
path=$( py_uri_decode "$uri" )
path=${path#file://}
echo "python: $uri -> '$path'"
```

```
$ ./uri.bash 
bash: file:///home/zomg/u%20wot%20m8.jpg -> '/home/zomg/u wot m8.jpg'
python: file:///home/zomg/u%20wot%20m8.jpg -> '/home/zomg/u wot m8.jpg'

```

### Post by Vaphell on 2014-08-27
in case of functions you have to have them in your script or import them via *source uri_functions.sh*, which pretty much embeds contents of external file as if it was a part of your script. Once they are loaded, they can be used. That said your file is not really a standalone executable that performs some well defined task which */bin folders are for. In order to use them as standalone 'programs' without *source*'ing you'd have to promote each function to a separate executable script, preferably without extensions (system programs don't use extensions of their language, do they?) and dump them in either some system bin or ~/bin. System looks for stuff on a file level, it doesn't know that there are some functions inside.

uri_encode:
```
#!/bin/bash

length="${#1}"
for (( i = 0; i < length; i++ ))
do
  local c="${1:i:1}"
  case $c in
    [a-zA-Z0-9.~_/-]) printf "$c" ;;
    *)                printf '%%%02X' "'$c"
  esac
done
```

uri_decode:
```
#!/bin/bash

local uri="${1//+/ }"
printf '%b' "${uri//%/\x}"
```

etc...

alternatively you could dump these functions to your .bashrc so they get loaded every time you open terminal.
