### Post by cristoper on 2008-12-07
It's not an OpenGL implementation (whereas Mesa, for example, is), it's a media library (like SDL) that offers a consistent API across several operating systems for creating OpenGL contexts, drawing things, and playing sounds.

It certainly looks nice, though. I plan to use it for a simple cross-platform game I'd like to create.
