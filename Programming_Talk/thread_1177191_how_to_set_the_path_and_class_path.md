### Post by dwhitney67 on 2009-06-03
Your attempt to post the error that was generated by the compiler did not work; perhaps you should edit your post.

Nevertheless, the most likely culprit for not being able to find stdio.h is that you do not have the C library installed on your system.

Try running this to have it installed:
```

sudo apt-get install build-essential

```
