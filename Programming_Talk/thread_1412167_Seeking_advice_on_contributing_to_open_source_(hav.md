### Post by Simon17 on 2010-02-20
The best thing to do is find an project that you're interested in. Download the source and build it. Poke around, see if you can make some tweaks, get familiar with the codebase.

Many open source projects have irc channels and forums where you can go to ask questions about the code, if you need help.

### Post by matthew.ball on 2010-02-20
If you're ever on freenode Jono Bacon (Ubuntu Community Manager) has set up an IRC channel #ubuntu-app-devel - basically it's a meeting point for people developing applications for Ubuntu (and I would guess GNU/Linux in general). There's also #ubuntu-programming with some active projects, you might be interested in offering some help.

Good luck!
