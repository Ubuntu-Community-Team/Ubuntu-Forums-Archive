### Post by Dejai on 2007-10-30
Hello, I have been really interested in the unix and Linux operating system for along time and I am a strong believer in community sharing of knowledge and software. Now I have Linux on my Ipod, and had it on my Laptop, which I am reinstalling as I right this message (driver config blah blah blah).

Now as a C++ programmer (still a relative newbie compared with some people I know) I have been working with a API called Irrlicht, now the thing which is great about irrlicht is it can produce stunning 3d games, can be ported for hand held devices (linux on the pocket pc) (also if you have Linux on the Xbox you can port it their to :D). It also contains wrappers for Java, and infact all .Net languages. Now the irrlicht API is opensource and works on linux, mac and windows. 

I am hoping to developed a range of open source products to the Ubuntu community consisting of my work and compiling that of others who offer it from the irrlicht community so that you have access to a LARGE collection of indie games.

I myself hope to only make games for Linux for a while at least in a hope to show that it has the possibility to be the Best Game Platform out. (better than the new mac 10 with its blue screen of death or vista with its lag and shotty compatibility) (And how Irronic I thought they said Linux was bleeding edge ).

Anyway I hope you enjoyed this project I have undertaken, and if you want to get some links for game development engines open to the Linux Community...

IRRLICHT - 3d graphics realtime rendering engine
IRRKLANG - 3d sound engine 
NEWTON SDK - Physics Game Dynamics Engine
Raknet - Game and Application Networking :D
