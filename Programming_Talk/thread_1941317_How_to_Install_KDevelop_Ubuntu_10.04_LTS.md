Hey guys I was looking for this and I found out that in 10.04 LTS you can just call this in terminal to install KDevelop.

Open Terminal and type

sudo aptitude install kdevelop

It should ask for your password.

Then it should install no problem.

This may not work on older versions of Ubuntu.
