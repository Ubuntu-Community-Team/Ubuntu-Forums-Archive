The University of Waterloo (in conjunction with Google employees) is sponsoring a Tron AI challenge, where anybody can upload their own AI code and compete against everybody else. The code submissions opens today (Feb. 4th and closes on Feb. 26th), after which the final contest will take place. (You can update your code as much as you want until the 26th)

It's pretty easy to get working code going, as they give you a starter pack with a working AI right from the get-go, just follow the "Five Minute Quickstart" guide.

I'll be competing as "Queue29" if anybody here wants to join the challenge.
