### Post by durand on 2009-07-12
If you're making a web application, you can use the [django framework]("http://djangoproject.org"). They have a really good book as well that you can read online at [http://djangobook.org]("http://djangobook.org"). It makes python web programming really straightforward and fun. Note that most hosts will only support python cgi scripts so what I do is just use django templates with my applications. There are other frameworks as well but I've never tried them.

For desktop applications, I use wxpython for the GUI but I think pygtk is a nice choice for gnome apps. pyqt is apparently really good but I've never tried it.

Oh and try Dive into Python, its a really good online book: [http://diveintopython.org/](http://diveintopython.org/)
