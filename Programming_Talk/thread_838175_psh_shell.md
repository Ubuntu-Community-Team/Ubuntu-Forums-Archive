> What does the psh and what does it do?

psh is a Perl program which executes a read-eval loop.

psh is an interpreted, interactive, modular programming language that includes some features of a UNIX shell, with the power of Perl.  

The Perl Shell will use Perl syntax and functionality for control- flow statements and other things. 

Psh's power can also be extended by writing your own modules in other languages.
