### Post by JLTB on 2007-01-16
Textmate is worth every penny.  I bought it last year and have been using it to write all my code for work.

Hints:
[LIST]
[*] Learn how to use the tab triggers, it takes a bit of time but it is very worth it.
[*] Check out the macromates website for some great themes, the default colours get boring after a while.
[*] Learn how to use the subversion support (if you use subversion).  I've found it to be better than every subversion GUI on the OS X platform.
[/LIST]
