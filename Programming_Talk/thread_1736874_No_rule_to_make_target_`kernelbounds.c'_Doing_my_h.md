### Post by dwhitney67 on 2011-04-22
There's a couple of issues with your module, however they do not explain the errors you were getting.  Here's how your module should appear:
```

#include<linux/module.h>
[B]/*#include<linux/kernel.h>*/
/*#include<linux/init.h>*[/B]/


static int hello_init(void)
{
        printk(KERN_INFO"MODULE LOADED");
        **return 0;**
}

static void hello_exit(**void**)
{
        printk(KERN_INFO"MODULE UNLOADED");
}

module_init(hello_init);
module_exit(hello_exit);
MODULE_LICENSE("GPL");
MODULE_AUTHOR("me");
MODULE_VERSION("v0.1");

```

Please explain how/where you are building your module.  I can say that you do not need to build it in the kernel area; it can be built in user-space, and by a regular user (ie. you don't have to be root/sudo).

Your Makefile looks good, and aside from the minor corrections to your code, everything should have worked fine.  In fact, prior to the corrections, only warnings were generated by the compiler.
