### Post by Omnios on 2006-04-22
Hi I play mud games and have found a program that is doing a lot of cross platform developement called Mudmagic. Mudmagic is one of the best mud clients I have seen for Linux with the folling features.

 ```
The MudMagic Client Features List
Client Support 	Community interactive support for this client.
MXP 	Mud Xtension Protocol for font changing and image manipulation.
ZMP 	Zenith Mud Protocol For Client Interaction.
MSP 	Mud Sound Protocol & on demand music play/download
MCCP 	Mud Client Compression Protocol for faster connectivity.
Theme Manager 	Extensive gtk based theme support for all platforms.

I.  Interface
        1. ANSI color
        2. Input
           a. Multi-line input
           b. string argument seperator
           c. command history
           d. auto completion
           e. keep sent text
        3. Multi-Game Tabs
           a. Tab moving
           b. Tab detachment
           c. Tab alert on change
           d. Tab reconnection, disconnect, close
        4. Game Wizard List
        5. Logging
        6. Themes

II. Protocols
        1 MSP (sound protocol)
        2 MCCP v1/v2 (compression protocol)
        3 ZMP (Zenith Mud Protocol)
        4 MXP (Mud Xtension Protocol)

III. Alias, Triggers, Macros
        1. Alias
        2. Triggers
        3. Macros


IV. Modules
        1. Database Module
        2. Notes Module
        3. Recorder Module
        4. Automapper Module

V. Scripting
        1. MudMagic BASIC
        2. Python



```

 Needless to say there will also be more new features added and also a lot of work on thse features. From what I have been told Mudmagic is now in Debian and there is an Ubuntu package available. 

 Anyways if you want a project to work on this project may prove interesting.
