### Post by Reiger on 2009-06-06
Kate does (or will/can do) that. Of course any half a decent IDE with XML highlighting too. 

And Kate will do so not just with XML: anything it recognises as `block' scope (and you can apparently script that too).

EDIT: Actually Kate doesn't do the exact same thing as what you describe: rather it highlights the whole block enclosed by matching delimiters. So anything between {} in C-like languages, and <some-element></some-element> in XML...
