### Post by Andrew_P on 2013-03-06
Search on terms **gnome 2.32 source download** and follow links to [http://ftp.gnome.org/pub/GNOME/desktop/2.32/2.32.0/sources/](http://ftp.gnome.org/pub/GNOME/desktop/2.32/2.32.0/sources/).  It takes about an hour to get all the files with an high-speed Internet connection.  Yes, it's tedious, but it can be done.:)  If you are using Firefox with the FlashGot extension, it can download all the links automatically, unattended, although you only need half of them — either the .gz or .bz2 files.  If you're not in a hurry and have enough drive space, let FlashGot download everything, then delete what you don't need using wildcards.
