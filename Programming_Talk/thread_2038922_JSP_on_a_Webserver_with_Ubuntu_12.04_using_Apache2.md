You haven't really explained the problem that you are having.  Are you using Tomcat as your HTTP server as well?  If so, it really is straightforward.  If you're trying to run Tomcat as your Java server off of Apache as your HTTP server then yeah, it can get fairly confusing for a newbie.  I can certainly appreciate the rant / frustration, but I'm not sure what you are looking for here.  I hate to say it, but go back to one of your tutorials and work through a Hello World app.  If it doesn't work come back and let us know what errors you're getting, or where things went wrong.  From there it's really easy to trouble shoot.  Odds are it's something small and silly and with one little tweak you'll be up and running in no time.

Oh, and don't hate me for suggesting another book / tutorial but the Heads Up Java Servers & JSP is just about as good, clear, and easy as you will ever find on the topic.
