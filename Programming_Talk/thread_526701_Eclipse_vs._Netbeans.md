### Post by tenmillionmilesaway on 2007-08-16
I personally prefer eclipse, mainly because afaik it compiles on the fly whereas netbeans uses ant to compile when you run the project meaning that launching an eclipse project is faster than in netbeans (feel free to correct me though).  I also prefer the eclipse debugger and like the wide range of plugins for eclipse.  However the visual gui designer in netbeans is top notch, but if your prepared to pay for it (i'm not :) ) then myeclipse has ported the netbeans gui designer to the eclipse platform giving the best of both worlds.
