### Post by tinny on 2008-11-18
> **figo2476 said:**
> Hi:

I am using eclipse IDE.

Is there any way to make text easier to see in dark background? e.g. like textmate. I can make the background be dark, but the contract is not good.

Go to "Window" > "Preferences" > "General" > "Editors" > "Text Editors"

You should now see a section titled "Appearance color options:"

Select "Background color" and change the color to what you want. You will also need to change your foreground colour.
