### Post by randallecook on 2009-06-11
UML is probably the closest thing to an industry standard. Start with these:

[http://www.uml.org/](http://www.uml.org/)

[http://en.wikipedia.org/wiki/Unified_Modeling_Language](http://en.wikipedia.org/wiki/Unified_Modeling_Language)

UML has specifications for static relationships (like classes), dynamic relationships (like interactions), and I believe data relationships, although ER diagrams might be a better fit there. See [http://en.wikipedia.org/wiki/Entity-relationship_model](http://en.wikipedia.org/wiki/Entity-relationship_model) for more info.
