### Post by jondiced on 2009-08-18
Problem solved! Solution here, which I can't believe I didn't find earlier: [http://hypernews.slac.stanford.edu/HyperNews/geant4/get/runmanage/191.html?inline=-1](http://hypernews.slac.stanford.edu/HyperNews/geant4/get/runmanage/191.html?inline=-1)

Anyway, the following 3 items must be the same:
1. Executable name in the makefile
2. Directory name
3. Foo.cc (replace "Foo" with executable name)

Thanks for your suggestions. Hope this helps anyone else in the future.
