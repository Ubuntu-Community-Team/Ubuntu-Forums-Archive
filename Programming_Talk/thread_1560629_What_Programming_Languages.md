### Post by nvteighen on 2010-08-25
Python, definitely: it is high-level, very fast to code in and has a huge amount of modules available in its standard modules, in the Ubuntu repos, in the internet in general... and if needed, you can make it interact with C.

My second option would be Perl... It's more C-like, but also high-level, also fast to code in (maybe the syntax is a bit confusing at first, but you get it), and has an even bigger amount of modules usually distributed through a universal Perl library called CPAN ([http://cpan.perl.org](http://cpan.perl.org)).

My third option would be C# itself. Yes, you can use it in GNU/Linux using the Mono framework... The thing is that Mono may not have the latest MS .NET features and there are some legal issues some consider to be very dangerous about it (not to me, actually).
