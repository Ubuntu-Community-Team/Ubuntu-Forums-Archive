### Post by derekeverett on 2010-03-30
I can tell you two things from my limited personal experience.

1) Learning C can teach you a lot about the fundamentals of programming. The syntax you learn carries over into many other languages. It's also a fairly small language all things considered in that there aren't a whole lot of keywords and basic functions to learn. 

2) Java can become pretty fun fairly early on. It is quite challenging to understand a lot of the object oriented features but once it starts to click for you it's a blast! 

These are the two languages I chose two get a good grip on before expanding too much into other ones. I'm happy I did now that they are both starting to make some sense to me. There was some real struggle moments in the beginning for sure.
