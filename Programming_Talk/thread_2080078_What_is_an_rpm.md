### Post by zombifier25 on 2012-11-03
RPM files are software packages for Red Hat-based Linux distros (such as Fedora), as opposed to DEB files, which are for Debian-based Linux distros  (such as Ubuntu). So you create an .rpm when you want your package your application for Red Hat-based distros. As to how to create a .rpm package, Google should know how.
