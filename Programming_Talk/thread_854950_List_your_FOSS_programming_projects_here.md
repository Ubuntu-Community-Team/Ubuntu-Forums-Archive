### Post by mssever on 2008-07-10
[LIST]
[*][Net Responsibility]("http://ubuntuforums.org/showthread.php?p=5340787") is Internet accountability software for Linux. I'm the principal author. It's primarily written in Ruby with some Python mixed in, and there will soon be a tiny smattering of Bash, too. It's licensed under the GPL.
[*][BadgeEntry]("http://badgeentry.sourceforge.net") is attendance software for childrens' events. Key features are: a) The ability to print badges which include pictures and barcodes; b) Support for badge scanning stations; c) It's web-based, so it can scale as needed. There's no real access control, though, so it should be kept on a private network. BadgeEntry is written in PHP5 using the Code Igniter framework. I'm the sole author. All of the code is open source; most is under the GPL.
[/LIST]
