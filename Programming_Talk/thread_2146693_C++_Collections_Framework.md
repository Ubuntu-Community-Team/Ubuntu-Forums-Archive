### Post by dwhitney67 on 2013-05-19
Standard C++ does not have the variety of Collection classes as Java, but other library extensions such as Boost offer additional choices.  For standard C++, you could the peruse the following [site]("http://www.cplusplus.com/reference/stl/") (and I recommend you bookmark it!) for reference on the such containers as vector, set, map, etc.
