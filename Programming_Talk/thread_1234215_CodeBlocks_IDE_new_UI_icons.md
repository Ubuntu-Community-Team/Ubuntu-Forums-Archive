I've been configuring my Code::Blocks IDE so that it fits my needs, and in the process I decided I didn't like the Code::Blocks icons, so I changed them. I used the free (and free) iconset "Silk Icons" released under the CCA 2.5/3 License. I modified some of them, but most of them are used untouched.

As the author would like credit, or, to quote him:
> All I ask is that you include a link back to this page in your credits (although a giant link on every page of your website really isn't needed, contact me to discuss specifics). 
Here is his link: [http://www.famfamfam.com/lab/icons/silk/]("http://www.famfamfam.com/lab/icons/silk/"). This guy does great stuff, it's worth checking it out.

Alright, here we go for the actual package:

**Screenshot:**
[[IMG]http://img194.imageshack.us/img194/5574/screenshotmlu.th.png[/IMG]](http://img194.imageshack.us/img194/5574/screenshotmlu.png)

**Download:** (feel free to create mirrors and redistribute this)
[http://www.muboo.net/cbsilkicons.tar.gz]("http://www.muboo.net/cbsilkicons.tar.gz")

**Installation:**
[LIST=1]
[*]Extract the tar.gz.
[*]In terminal: sudo gnome-open /usr/share/codeblocks (note: I don't know the command for KDE, maybe someone wants to share it ;))
[*]Enter your password
[*]Drag the contents of the extracted archive to your Code::Blocks folder and overwrite if prompted
[/LIST]

Note: As Code::Blocks uses ZIP files for its icons, and as these ZIPs contain some parameters, overwriting these ZIP files may reset some of your parameters. I'm sorry if it does so. You can back them up before if you feel the need or don't want to reconfigure them after the install.

Please report any bugs or missing icon file. ;)
-Lazlo
