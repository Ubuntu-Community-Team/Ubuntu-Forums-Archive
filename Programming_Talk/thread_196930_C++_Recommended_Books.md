### Post by asimon on 2006-06-15
I don't know the C++ Primer, but "Thinking in C++" is good if it has to be free, otherwise "Accelerated C++" by Andrew Koenig and Barbara E. Moo is an excellent book. And the classical reference is "The C++ Programming Language" by Bjarne Stroustrup, but this is not a good book for learning C++.

If you already know C++ the books from Scott Meyers, "Effective C++" and "More Effective C++" are a valuable addition. Both offer good practices, design and programming tips.
