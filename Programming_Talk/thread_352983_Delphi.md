### Post by samjh on 2007-02-07
> **Rhaddad said:**
> can anyone tell me a program that uses delphi. thanks

Nothing (that I know of) for Linux, I'm afraid.

Delphi is Windows-only, but there are quite a few impressive programs written with it.  People have mentioned Dev-C++, which is the obvious one, but there is also wxDev-C++,  GEL editor, and Borland's own tools: Borland C++ Builder, Delphi itself, and Borland Developer Studio.

If you talk of Delphi, you're really talking about Object Pascal itself, which is a bigger thing.  The software side of Apple Mac was largely built on Pascal, and later Object Pascal, much like how *nix was built on C with later C++ influence.
