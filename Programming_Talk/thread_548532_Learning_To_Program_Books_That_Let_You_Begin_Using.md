### Post by LaRoza on 2007-09-11
See my wiki, there are many PDF books and tutorials available for download. (You can read them on screen, or print them.)

I would suggest Python or PHP. (The wiki has all the links, in my sig.)

Most books I have seen, not Windows specific, don't assume you use Windows or a Mac. PHP books will often use Apache as the server in the book, and Python is the same no matter what OS you are using, as far I can tell.

For books to buy, most O'Reilly books, the "Learning xxxxx" or "Programming xxxxx" are good, as are the the "Head First" series (I hear).

PHP [http://www.amazon.com/Programming-PHP-Rasmus-Lerdorf/dp/0596006810/ref=pd_bbs_sr_2/002-9379475-6117603?ie=UTF8&s=books&qid=1189529878&sr=8-2](http://www.amazon.com/Programming-PHP-Rasmus-Lerdorf/dp/0596006810/ref=pd_bbs_sr_2/002-9379475-6117603?ie=UTF8&s=books&qid=1189529878&sr=8-2)

Python: [http://www.amazon.com/Programming-Python-Mark-Lutz/dp/0596009259/ref=sr_1_4/002-9379475-6117603?ie=UTF8&s=books&qid=1189529916&sr=1-4](http://www.amazon.com/Programming-Python-Mark-Lutz/dp/0596009259/ref=sr_1_4/002-9379475-6117603?ie=UTF8&s=books&qid=1189529916&sr=1-4)

The same publishing company has other books on the languages, but these are good for almost any level, although the "Learning Python" and "Learning PHP" might be better if you are completely new.

I the Web to be very useful for programming. The documentation and tutorials that can't be found in books, is free on the Web, I suggest you try them.

### Post by pmasiar on 2007-09-11
If you want to learn only one language to cover most of what you want to do, Python is the best choice. Web apps, desktop apps, text file parsing, numerical apps (if not as fast as in C, but so what) with numpy, statistics (with RPy), file shuffling - Python can do it all, unless yo want do drivers or deep kernel hacking.

Plenty of free online books and also paper books: see wiki in my sig
