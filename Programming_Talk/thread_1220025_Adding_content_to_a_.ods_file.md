### Post by johnl on 2009-07-22
Hi,

I believe that all Open Office documents are simply compressed XML files.  

There might be some library out there that makes it easy to generate ODF files -- I don't know of one.

The other option is to do some reading on the file format, since it is an open standard.

Check out the technical works @ [http://www.oasis-open.org/committees/tc_home.php?wg_abbrev=office#technical](http://www.oasis-open.org/committees/tc_home.php?wg_abbrev=office#technical)

You'll probably be interested in the **OpenDocument v1.0 specification**.

Hope this helps!
