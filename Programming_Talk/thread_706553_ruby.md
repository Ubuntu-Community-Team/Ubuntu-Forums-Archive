### Post by pmasiar on 2008-02-24
Rails are for web applications - too complicated for beginner. Don't focus on GUI just now, plain text program in terminal is the simplest start.

Most language learners have problem: what tasks to solve to learn how to program? Tasks have to be simple enough to be solvable, but not trivial and boring.

Wiki in my sig has many links to traning tasks. It suggests Python (comparable to Ruby, but IMHO slightly better for beginners), but you can solve them with Ruby as well. 

Another good source of tasks to solve is PythonChallenge (of course in Python) - 32 tasks of increasing complexity, where solving level unlocks URL to next one. IMHO it is a way to lern language and have the most fun. It has also forums with help and hints for eacj level. ProjectEuler in language-agnostic, but math-only problems.
