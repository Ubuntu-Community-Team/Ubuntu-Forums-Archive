### Post by jespdj on 2008-10-27
This is going to be difficult to answer, because there are hundreds of web frameworks. It's very hard to say which one is "the best" or "the most popular". It also changes all the time. If you ask the same question in three months, the answer might be different. So, don't expect a clear answer that says "this is THE framework that you absolutely must learn because of these clear reasons...".

For Java, some of the well-known ones are JSF (Java Server Faces), Struts (which is getting a bit old), Spring Web MVC, GWT (Google Web Toolkit), Wicket, Seam, ...

See for example: [Which is the Hottest Java Web Framework? Or Maybe Not Java?](http://www.breakitdownblog.com/which-is-the-hottest-java-web-framework-or-maybe-not-java/)

Different companies use different languages, and they are NOT going to change it because some web framework is popular. Different frameworks for the same language are more or less productive, but companies will think hard to change framework they use.
