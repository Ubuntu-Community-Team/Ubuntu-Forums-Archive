### Post by Can+~ on 2008-03-27
Personally, I think you should get a book about python itself and then learn pygame or any other extra module using the documentation.

One of the cool things of understanding a programming language fully, is that you can go to any documentation site, read a few paragraphs, and start doing things inmediately, and if pygame teaches you how to use 1 specific, non-standard module, another book may teach you how to go fully on the language. Although, this is solely my opinion, I usually learn the basics and get books about complex things which aren't on the documentation, those kind of things that take time.
