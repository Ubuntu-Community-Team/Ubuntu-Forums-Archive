### Post by BeardlessForeigner on 2008-06-13
I'm a senior math major in University who got a late start programming about a year ago. I found Python to be a great first language. I learned from online resources linked to by others in these forums. [http://http://www.diveintopython.org/]("http://http://www.diveintopython.org/") was the biggest help. Object orientation was the biggest obstacle, I think because it doesn't come as naturally from mathematical thinking. I have since used Python with matplotlib and scipy/numpy to do some stuff for free that I might have wanted to do with Matlab.

I have using Haskell for the past month now, and really enjoy it. As a purely functional and lazy language, you must do things very differently from languages more commonly used. It is fairly difficult to learn (There is A LOT I haven't grasped yet), and of course not that common compared to C and Java (and Python in these parts).
