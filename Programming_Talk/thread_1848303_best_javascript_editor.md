### Post by aura7 on 2011-10-01
There is a different eclipse distribution for different needs.. 

[http://www.eclipse.org/downloads/packages/release/helios/sr2](http://www.eclipse.org/downloads/packages/release/helios/sr2)

for your case you should download the PHP version 

Then start eclipse ... go to Help -> Install New Software 

in that work with ... select All Available sites... 

install Javascript IDE from there ..  ( type javascript in the text bar -> Select JavaScript Developer Tools )
install C++ from there ....  ( type Eclipse C/C++ in the text bar -> Select Eclipse C/C++ Development Tools


After you install both .............. restart the IDE ........... then your C++ and PHP files will open properly


There is one more free IDE ... Netbeans ... which provide good PHP, C++ and Javascript support

[http://netbeans.org/downloads/index.html](http://netbeans.org/downloads/index.html)

Download the version in which all technologies are supported ... by Selecting All... 

Let the platform be Linux... which is displayed at the top of the page
