### Post by shawnhcorey on 2010-04-29
> **jetsam said:**
> Perl is stuck between versions

Perl has recently became unstuck.  A new version has just came out, incorporating some of the ideas for Perl 6 back into Perl 5.  And Perl is leading the way into genome processing with BioPerl.  Perl is going to be a lot more prominent in the coming years.  However, it's not going to do it fast enough for someone looking for a career change.

If you're learning to programming, a scripting language like Ruby, Perl, or Python will give you the basics.  If you're about to enter the work force, C, C++, Java or LAMP (Linux, Apache, MySQL, Perl/Python/PHP) and Javascript.

If you have some experience working in a language, then your best bet is to learn some management skills.  Programming is as much about organization and communication as it is about coding.  Learning how to organize and communicate better will increase your prospects faster than learning a new language.
