### Post by KaeseEs on 2006-11-06
I believe vim comes with support for syntax highlighting and indentation for Ada.  Install it, run 'vimtutor', and you'll be good to go as far as editing (just remember to set up ~/.vimrc).  If you prefer a graphical IDE, I've had good experiences with Anjuta, which supports Ada.

For a compiler, GNAT should work fine.
