### Post by scourge on 2009-03-04
I think it's awesome for C++/Qt projects. It's simple, fast, has excellent automatic code completion, support for SVN and Git, etc. What I love the most, is that I can just hover the mouse over any Qt class, variable, function, etc., press F1, and Qt Creator will show me the relevant part of their documentation.

> Seems good to me, but I didn't had time to test the svn and git plugins and some other things.

The Git plugin works great in Linux. I still have to do stuff like merges and rebases from the command line, but the plugin is good for simple commits, checkouts, and especially viewing diffs. On Windows I couldn't get the plugin to find my Git installation.
