I am working on a open-source project called Deimos, which is: a simple client program will gather data about your computer (specifically configurable which data) and sends it to a website where statistics, graphs etc are generated. The project is much like [http://whatpulse.org/](http://whatpulse.org/) except it will feature more statistics and both a linux- & windows-client.

People would use it for fun but also to get actually some useful data from it like how much is downloaded in a certain period of time with multiple computers (or else the ISP gets angry :P)
Personally I started this project because I wanted to see stats about my network/cpu/memory about different periods of time.. of both my computers.

Proposed stats so far:
- Up/downloaded of network
- CPU-usage
- Memory-usage
- Uptime
- Number of keys pressed
- Number of clicks pressed

I've already started coding on the website. I only need someone for the client (which is considerably less work then the website)

What do you think? Anyone interested? Or do you think this is retarded? :)
