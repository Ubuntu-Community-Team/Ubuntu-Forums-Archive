> Announced a new free educational project - Alpha (Alpha). This is a program for children from 3 to 6 years. In addition to already existing projects, it will teach children to read, and later, perhaps, to write, but in the more distant future. In the first release will be realized following functionality: learning the alphabet, read by syllables, reading the words, multilingual interface. The program will be written in C++ with Qt interface. For learning to write there will be handwriting samples, which can be printed on the printer. Since the program focuses on children, will be created a clear and beautiful interface, used many illustrations, and chosen good sound and musical accompaniment.I’m going to make a free educational project for children. This project will provide functions to help children from 3 to 6 to learn the alphabet and to study how to read words. This project will be cross-platform (created with QT libraries) and licensed under GNU/GPL license. I ask everybody who want to help me for alphabet illustrations and sounds(readings) of syllables, letters and words in his native or other language.
