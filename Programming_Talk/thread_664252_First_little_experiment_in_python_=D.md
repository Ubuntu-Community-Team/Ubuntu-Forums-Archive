### Post by ryanVickers on 2008-01-11
Some of you know me for my work on "mkrar" (the graphical split RAR maker in my signature), and that was written in bash as a shell script.  I decided to learn python now and this is my first coding experiment with it.  It is a program that goes from <number A> to <number B> and shows all prime numbers between them.  If a number is not prime, it shows the first multiplication equation it finds that makes it not prime.  It has features like value correction* and it goes right to the number you choose, not one short of it!

*if you enter a negative integer, it will offer to fix it (convert to positive).  Also, if the integers are entered in the wrong order, it will offer to fix this too.
