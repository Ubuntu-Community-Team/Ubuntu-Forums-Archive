### Post by gmargo on 2010-07-10
It looks to me that Ubuntu has provided everything that the original web site [http://freeglut.sourceforge.net/](http://freeglut.sourceforge.net/) has provided.  The api documentation on the web site is contained in the /usr/share/doc/freeglut3-dev/freeglut_user_interface.html file.  The index.html file contains this at the bottom: 
> 
Since freeglut is a re-implementation of the original GLUT API, you can find API documentation at [http://www.opengl.org]("http://www.opengl.org/").
