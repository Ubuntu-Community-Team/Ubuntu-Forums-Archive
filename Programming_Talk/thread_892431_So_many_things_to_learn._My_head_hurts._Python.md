### Post by Lster on 2008-08-17
I would recommend Project Euler or similar (like samjh posted):

[http://projecteuler.net/index.php?section=problems](http://projecteuler.net/index.php?section=problems)

The challenges start off quite easy and slowly get harder. By problem 200, some of the challenges are amazingly hard. It will certainly take you from beginner to intermediate, and if you keep on with it you should become very experienced.

From the "About" page:

[QUOTE=Project Euler]The problems range in difficulty and for many the experience is inductive chain learning. That is, by solving one problem it will expose you to a new concept that allows you to undertake a previously inaccessible problem. So the determined participant will slowly but surely work his/her way through every problem.[/QUOTE]
