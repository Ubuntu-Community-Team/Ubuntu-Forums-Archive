Is Java your first programming language?

I would not recommend learning two languages as first, even/especially if they are as close as Java and C++.

Excellent books to learn java is "Headfirst Java" - to understand basics ("how"), and then Bruce Eckel's book "Thinking in Java" - it is so good it is in 4th (updated) version. Will explain you "why" java is (as quirky) as it is: design decisions made, and consequences.

BTW in my opinion (and many other, including Eckel) best first language is Python. It will allow you to start writing working code way before you understand objects, classes, types, exceptions etc. To start with Python, check [http://learnpydia.pbwiki.com/](http://learnpydia.pbwiki.com/)
