Hi there,

I'd like to know if a GTK+ Theme creator existed, i.e. a GUI application that helps people visualize and create their GTK+ Theme and produce a gtkrc or gtk.css file. I'm talking about apps aiming at creating themes, not only visualizing them.
It would :
  - include on one side a set of most GTK Widgets, and on the other controls changing the appearance of the theme (and so of the Widgets of the first side), helping creative people visualize their theme without having to switch between the text editor and the visualizing application
  - produce the theme file automatically, so theme creators don't have to know how to write a theme. They could even just pick up someone else's theme, and tweak it out.
At the end of the day, it could help produce more and better themes.

I'd like to know if such an application existed, so I know if I might consider developing it in the future :)
