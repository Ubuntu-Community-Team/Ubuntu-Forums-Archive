I just came across some projects witch I think sounds interesting. So if your looking for a project to get involved with, maybe you want to contribute there. It would be great to see these project really take form - perhaps even be ready to include in 8.10 or more likely 8.10 +1

_**Epidermis theme manager**_
Epidermis combines wallpaper, GTK, metacity, icon, splash, usplash, cursor, grub and GDM themes in one GUI program for the GNOME desktop.
[https://launchpad.net/epidermis/]("https://launchpad.net/epidermis/")
- *Contributors: 2*

**_Gloobus, a quicklook for linux_**
It is a quicklook for GNU/Linux using OpenGL, the point is to preview the most common files in a very graphical and fast way! 
[https://launchpad.net/gloobus]("https://launchpad.net/gloobus")
- *Contributors: 8*
