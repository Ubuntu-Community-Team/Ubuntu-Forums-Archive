### Post by Verminox on 2008-06-02
If a Wiki is indeed what you are looking for, try [DokuWiki]("http://wiki.splitbrain.org/wiki:dokuwiki"). It has a very simple installation, very easy interface, very easy markup and best of all no MySQL needed. And it's still efficient. It also has a search bar :-) The only problem you might have is finding a theme to suit your RuneScape needs but if you know HTML/CSS you can make your own layout so no issues.

You can also go in for a Content Management System like [Drupal]("http://drupal.org/") if you want a lot of features like User Management, Visitor Comments, Polls, News, etc. However, this requires MySQL so you have to check with your host for that.
