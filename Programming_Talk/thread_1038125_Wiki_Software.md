### Post by drubin on 2009-01-12
Like slavik suggest each office would have an ip address range restrict access on a server level to those ip's.

Or give each user their own account on the wiki. Most wiki egines allow for different access levels they also allow for only logged in users to edit documents.

If having each user account is over kill you would set up an account for each department using the wiki.
