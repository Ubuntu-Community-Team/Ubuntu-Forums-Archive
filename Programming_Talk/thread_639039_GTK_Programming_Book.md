### Post by MicahCarrick on 2007-12-13
Granted my opinion is biased as I was the technical reviewer for" Foundations of GTK+ Development", I strongly recommend it. Not only is it up to date, it's much more thorough than some of the others. There are exercises and you even end up creating a text editor and file browser. Source code is free from gtkbook.com and you can checkout all the source code from the book. Some of the good things are it's coverage of some newer widgets including printing and notification icons, creating custom widgets, and entire chapters dedicated to text view and tree view widgets.

"The Official GNOME 2 Developer's Guide" is also really good as it covers (though briefly and somewhat out of date) the build tools, .desktop entries, GConf, and some other GNOME-specific issues.
