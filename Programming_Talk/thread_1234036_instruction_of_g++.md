### Post by nvteighen on 2009-08-07
Er... just type **man g++** in a terminal and you'll get enough information to have it compile your code.

But, if you're actually searching for a programming environment, well... if you ask me, I'd use a text editor + tools like Makefiles or a revision control. Of course, it depends on how much time you have to finish this project and having to learn this stuff may make you lose precious time.

Look at the Stickies. There's plenty information on Integrated Development Environments (aka IDEs) that give you features similar to MS Visual C++. I don't like such things and prefer the UNIX way of several tools, but in your case they might help you.
