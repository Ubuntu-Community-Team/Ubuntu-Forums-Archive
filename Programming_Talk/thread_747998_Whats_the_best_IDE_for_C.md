### Post by lnostdal on 2008-04-07
> **Mick8882003 said:**
> Whats the best IDE for C?

well, the one i use of course..

..but its mine; you're not getting it .. you'll have to create your own IDE, but here are the tools i've used to create my IDE:

* emacs + lots of shortcut keys and small utilities .. everything is connected
* gcc
* cmake / scons
* gdb / kdbg
* terminal windows

i use multiple desktops (ctrl-alt right/left arrow key) .. each filled with windows and tools that fall under a certain category or "task type":

* editing of code
* testing of application + debugger, or view of run-time debugging output
* reading of documentation .. searching for clues on the web .. etc.

..i use an extra screen connected to the laptop; this way i can view two desktops and do two of these "tasks" at the same time

a lot of programmers don't bother with "static IDEs" .. why should they? .. they either use flexible IDEs they can mold to their needs, or use no IDE at all, or well, build their "own IDE" - like i do
