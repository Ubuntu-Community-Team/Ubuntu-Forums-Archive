### Post by xytron on 2012-02-07
( x-1 ) ( x+2 )^2  -  ( x-1 )^2  (x+2)

Note that the common factors are;

 ( x-1 ) ( x+2 )

so that you have;

( x-1 ) ( x+2 )  [  x+2-x+1 ]  =  3 ( x-1 ) ( x+2 )
