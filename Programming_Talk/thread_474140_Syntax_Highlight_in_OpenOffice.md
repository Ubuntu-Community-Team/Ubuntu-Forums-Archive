### Post by jyba on 2007-06-14
Okay, you're going to laugh at this but it works...

1. Go to the "File" menu of Kate and "Export as HTML"
2. Open the HTML file up in a web browser.
3. Copy from the web browser and paste into an Open Office Word-Processor document.
4. Save with the '.odt' extension.

There's got to be an easier way. :D
