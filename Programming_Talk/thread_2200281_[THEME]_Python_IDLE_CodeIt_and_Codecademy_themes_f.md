Hi guys!

I have made two themes for Python's IDLE. One is CodeIt and the other Codecademy UI (based on [Codecademy]("http://codecademy.com")'s theme).

> **PLEASE DO NOT POST THIS THEME ELSEWHERE. IF YOU WANT TO SHARE THE THEME, PLEASE LINK TO THIS THREAD. **

[COLOR=#000000]Installing:[/COLOR]

[LIST=1]
[*]Close IDLE (if open)
[*]Add the lines below to : Ubuntu(**~/.idlerc/config-highlight.cfg**), Windows(**C:/Users/[username]/.idlerc/config-highlight.cfg**)
[*]Start IDLE
[*]Click **Options** -> **Configure IDLE**
[*]Click the **highlighting** tab
[*]Select **a Custom Theme**
[*]Select **CodeIt/Codecademy UI**
[*]Click **OK**
[/LIST]
