### Post by crdlb on 2011-12-10
It's part of xscreensaver, so you can get the code with 'apt-get source xscreensaver', or by looking at Launchpad: [http://bazaar.launchpad.net/~ubuntu-branches/ubuntu/oneiric/xscreensaver/oneiric/view/head:/hacks/substrate.c](http://bazaar.launchpad.net/~ubuntu-branches/ubuntu/oneiric/xscreensaver/oneiric/view/head:/hacks/substrate.c). As you can see, it is written in C.
