### Post by Antoni on 2005-05-09
An open source 32 bit compiler being developed for Linux, Windows, and MSDos. Syntax compatible with Microsoft's Quick Basic, runs graphics QB programs without DOSBox. It has enhancements as pointers, optional function args, and coming soon, objects.  Generates fast code, 75% of gcc speed, and it's compatible with gcc libraries and tools.

Give it a try:
[http://www.freebasic.net](http://www.freebasic.net)
