### Post by LordHunter317 on 2006-08-21
Just what he said, plus a serious study of operating systems at both the thereotical and practical level.  

OS X isn't a UNIX, both legally or in reality.  IT's based on NextStep, which happens to be based on UNIX.  I'm noting this because OS X's UNIX support isn't because Apple wants a UNIX, but because they want NextSTEP.

Windows NT is largely based on VMS, because Dave Culter designed that while he was at DEC.  The rest is largely MS' own creation with no real ties to anything.

You need to do some basic research on OS histories.  Wikipedia has some OK articles and a Google search will turn up some more.  You're grossly misusing terminology here, but I'm not even sure how to go about correcting it.
