### Post by WW on 2008-11-08
Hozza,  thanks for your generous offer.

The link provided by mentallaxative provides a good starting point for discussing which features to provide.  For what it's worth, here are a few features that my ideal host service would have:
[list]
[*] A source code control system, e.g. [subversion](http://subversion.tigris.org/) or [git](http://git.or.cz/).

[*] Something like [Trac](http://trac.edgewall.org/) for project management and issue tracking.

[*] A home page for the project that is completely customizable--let me hack around with the look and feel of my project's web site.

[*] A forum for discussing the project.

[*] A build system for creating installable binary packages for as many platforms as possible.  E.g. build a deb for debian/ubuntu, an rpm for redhat/fedora, an msi installer (say) for windows, etc.  This might not be technically feasible, but hey, we can dream, can't we?
[/list]
