Hey, I just have a simple question.  I like emacs keyboard shortcuts.  They all work well in the bash terminal except Alt-f for forward-word.  Instead of moving forward one word, it opens the file menu.  Is there any way to disable this?

### Post by Mister.Vash on 2008-09-14
In gnome-terminal, go to Edit -> Keyboard Shortcuts and check "Disable all menu access keys (such as Alt+f to open File Menu)"

Doing this will allow it to work
