### Post by Aeglius on 2017-01-26
[COLOR=#000000][FONT=&amp]In photo collections one encounters more and more often panoramas (taken by cellphones) or you are even yourself an occasional panorama photographer. A lot of web platforms automatically display them the "right way". But in Ubuntu's default image viewer it is all flat and distorted. How can I present such photos in a presentation without having to zoom in and out all the time?

[/FONT][/COLOR][COLOR=#000000][FONT=&amp]I decided to create a plugin that may be useful for everybody else:
[/FONT][/COLOR][https://github.com/Aerilius/eog_panorama](https://github.com/Aerilius/eog_panorama)
[COLOR=#000000][FONT=&amp]
Installation into user directory:
Copy the folder eog_panorama into ~/.local/share/eog/plugins/ and open Eye of Gnome preferences to activate the plugin.[/FONT][/COLOR]
[COLOR=#000000][FONT=&amp]
Maybe it is a feature that a "smart" image viewer should have built-in? 
Currently it is rather a prototype (using a JS panorama viewer in a webview), but possibly one could create a native implementation.
Feedback is welcome!
