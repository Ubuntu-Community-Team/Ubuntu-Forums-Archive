### Post by pmasiar on 2008-08-14
Which Python book did you liked?

Try solving problems: project Euler, and PythonChallenge website.

You were smart (or lucky :-) ) to start with Python - it is simple enough to be learned by yourself. You will be surprised how much more complicated Java is, but because you can see the "big view" from Python perspective, you will do fine in Java.

Consider also looking into Jython: Python for JVM, so you can use all java libraries.

Simplest IDE for Python is IDLE. On Windows, it comes with Python (ActiveState.com is better than python.org), synaptic will fetch it for you in ubuntu. 

Most people use any common editor and not IDE (so you can edit all your files: HTML, .ini, etc) in same editor, and many people go for vim or emacs. SciTE is excellent lightweight editor with syntax highlighting for many languages, for both Windows and Linux.
