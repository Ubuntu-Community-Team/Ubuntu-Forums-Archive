### Post by PriceChild on 2007-08-03
Ubuntu uses a stable release sytem.

After releases, only critical bug fixes and security patches will get in, no feature updates.

This helps keep stability, as big (unforseen) problems can appear which can't be prepared for.

If you want to be constantly updated, then go with a distribution with a "rolling release"
