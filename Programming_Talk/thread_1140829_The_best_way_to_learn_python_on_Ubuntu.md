### Post by StooJ on 2009-04-28
Point you in the direction of what, exactly? Could you be a bit more specific?

Python is installed by default. The python tutorial on python.org is fine, but there are plenty of others. Python is just another language, so the environment (Ubuntu/Debian/MovieOS) is pretty irrelevant when learning (especially when starting out).

[http://laroza77.wikidot.com/](http://laroza77.wikidot.com/) has lots of great information about learning various languages, including python. It will start you off here: [http://learnpython.pbwiki.com/HowToStart](http://learnpython.pbwiki.com/HowToStart)

Unfortunately, the pbwiki seems to be down just now, so here are the ones from that site that I remember:
[http://www.greenteapress.com/thinkpython/thinkCSpy/html/](http://www.greenteapress.com/thinkpython/thinkCSpy/html/)
[http://www.swaroopch.com/notes/Python](http://www.swaroopch.com/notes/Python)
[http://www.dickbaldwin.com/tocpyth.htm](http://www.dickbaldwin.com/tocpyth.htm)

As the wiki says, it's worth going through a few of them to clear up any confusion you might come across.
