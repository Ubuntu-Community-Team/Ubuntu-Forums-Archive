### Post by ssam on 2007-01-31
dont worry to much about versions of python. python 2.0 code will mostly run fine in 2.5.

if you read the [whats new in python 2.5]("http://docs.python.org/whatsnew/whatsnew25.html") page, you will see that it is mostly fairly advanced stuff. it matters more for people writing modules and libraries.

major version changes have more difference. python 2.x code may not work in python 3.0 when that is released (probably at least a few years away).
