### Post by v1cho on 2008-08-21
There is a GREAT open-source GUI for mencoder for Windows called viDrop and it has no equivalent for Linux. 
But its author, Krasimir Stefanov, has shared that he's having problems porting it for Linux, so I wanted to ask if there is anyone willing to port it (or possibly rewrite it, its source is in the AutoIt 3 language) :). There's everything in it - from downloading from video sites (like YouTube, etc.) to inputting subtitles and DVD encoding :) It is has a presets feature for MANY various MP4 players, PSP, iPod, iPhone... 
Try it - you will be amazed! And if you are able to port it - PLEASE, do so!

Link to the installer: [http://learnfree.eu/download/vidrop/vidrop-0.56.2-install.msi](http://learnfree.eu/download/vidrop/vidrop-0.56.2-install.msi)
Source code is attached to the thread and included in the installer. 

Thank you!
