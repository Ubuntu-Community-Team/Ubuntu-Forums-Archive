### Post by dawdler on 2007-02-01
I installed anjuta recently through synaptic which went smoothly, but the anjuta 2.0.2 package does not have certain very necessary dependencies linked such as the anjuta-dev package and C/C++ compilers gcc/g++.

I haven't fully tested the app yet, but I have run into a few random crashes.  Nevertheless, I was finally now able to compile and build my hello world program.

First impression, I like this IDE better than eclipse with C/C++.

My advice, go into synaptic, search for the anjuta package, and look at its dependencies list before installing via source.
