### Post by ofnuts on 2013-11-12
Bash is useful, but it's not really a general-purpose programming language. In most languages, programming is breaking the code into smaller units that you can write (or find written for you in a library)(and IMHO breaking the code up is what makes the good porgrammer). In bash you mostly have to find how to funnel the data to system commands, and avoid as much as possible handling the data directly....
