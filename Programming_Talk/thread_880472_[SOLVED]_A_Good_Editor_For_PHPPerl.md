### Post by scragar on 2008-08-05
quanta is a great tool in my opinion, although it lacks some of the power gedit has is some ways(for example saving a file doesn't change it's colour coding right away, you have to close the file and reopen first, strange I know), but it has a lot more power for advanced features like tag completion, page validation(plugin) and such like.


try not to get caught in the emacs/vi war, although both are great tools I wouldn't recommend them over a tool designed for especially for web development, which offers better features(although it is important that you don't get reliant on it, if your tool does all the work your in trouble if a tool breaks or doesn't support something new you want to use).
