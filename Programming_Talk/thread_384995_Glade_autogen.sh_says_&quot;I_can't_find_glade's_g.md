### Post by lnostdal on 2007-03-15
Do not generate source code using Glade! It is deprecated!

Use Glade to generate the .glade/XML-files (use `Save' instead of `Build'), then instead use the libglade-API to load those files (GUI-descriptions) in at runtime: [http://developer.gnome.org/doc/API/2.0/libglade/index.html](http://developer.gnome.org/doc/API/2.0/libglade/index.html)

I link to some writings about gtk+/glade/libglade in my signature; check them out.
