### Post by lswest on 2008-04-03
Well, a useful scripting language to know a bit of is BASH (which is what the terminal actually employs, with commands and such, tons of tutorials out there, but i actually have a book on it...called: "Linux Shell Scripting with Bash" by Ken O. Burtch)
