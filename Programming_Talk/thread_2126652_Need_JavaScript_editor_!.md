### Post by slickymaster on 2013-03-18
You really should give Netbeans a try. The last version, 7.3, comes with full support for HTML5, CSS and JavaScript, a JavaScript debugger, which works with either Chrome or the Embedded WebKit browser. It also supports ScriptDoc, ExtDoc and JsDoc in version 2, with better supporting for JQuery and pattern recognition.
