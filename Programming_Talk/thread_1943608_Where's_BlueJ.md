### Post by Alberonn on 2012-03-19
I was looking in the Ubuntu Software Center and I couldn't find the BlueJ ADT. Luckily, I knew where the site was and I was able to use their Debian/Ubuntu .deb package to install it.

BlueJ is a somewhat lightweight Java ADT that I was using in my CS-161 and CS-162 classes. It has some nice features, but it doesn't have all the bells and whistles as some of the bigger ADTs such as Eclipse. I do like how it highlights my code, making it easier to follow and catch some syntax errors. I also like how easy it is to create and test individual classes within BlueJ--especially how easy it is to create a test class to give your code a workout. The textbook that goes with it is pretty decent as well.

While it does go with the BlueJ textbook, it's still free software and it is very functional outside of the classroom. I was surprised to see it was not on the list. I know the main site is at bluej.org. Is there any chance to see this added to the Software Center?
