### Post by ofnuts on 2014-11-03
I use [Squirrel SQL]("http://squirrel-sql.sourceforge.net/"). And I don't seem to be missing much compared to my colleagues on SQL Developer or Toad. The support mailing list is fairly responsive and there are regular updates. It is also DB-agnostic, provided you have the Java JDBC driver.
