### Post by MadCow108 on 2010-04-09
visual c++ is just an IDE (integrated development environment) for C++, its no new language.

Similar IDEs (although not quite as powerful) for linux are netbeans or eclipse

just note that visual c++ uses a different compiler than the majority of the unix world uses (=gcc)
But if you write standard conforming code this does not matter much
