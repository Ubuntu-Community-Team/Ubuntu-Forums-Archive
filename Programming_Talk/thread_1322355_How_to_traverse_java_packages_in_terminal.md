### Post by DaithiF on 2009-11-11
also not sure what you mean -- do you mean something to cut down on the typing required for cd'ing around the package directory structure?

if so, then you could check cdbm ([http://perlmeister.com/scripts/cdbm.html](http://perlmeister.com/scripts/cdbm.html)), a handy perl script from Mike Schilli which allows you to bookmark directories you cd to frequently.

theres also a project called bookmark tools ([http://www.bookmark-tools.com/](http://www.bookmark-tools.com/)) which sounds similar, but covers more than just cd'ing.
