### Post by pmasiar on 2007-08-27
You forgot to mention your programming experience. Any? None?

You did not mentioned Java in your post title, so many Java experts around might miss it.

Will you be learning programming completely by yourself, with no help from teachers? They at least should recommend a book for study, no? How would you know what to study?

Very good and easy to read book is "Headfirst Java" from O'Reilly - most O'Reilly books are very good, above average, and this one is excellent for first read. Then you will need deeper book, more technical - Bruce Eckel book "Thinking in Java " is in 4th edition, but might be too advanced for you - maybe as third book?

That said, in opinion of many teachers Python is better intro language that Java. Syntax is simpler and less picky, data structures too. The guy wrote that he had 2 groups for a year: one Java only, and the other had 3 months of basic Python before diving into Java. Mixed group has better results, covered more material, and had more fun, than Java-only.

If you are adventurous, you may try it too. In 2 weeks you will see if Python is for you or not. Wiki in my sig has many links. Subscribe to python mentors mailing list, you will need all the help you can get, and quickly! 

Surely Java has specialized forums like that too, ask around, I know that "headfirst" guy and gal have javaranch website and forums.
