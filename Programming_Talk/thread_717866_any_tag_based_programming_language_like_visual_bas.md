### Post by CozmoTrouble on 2008-09-29
> **exneo002 said:**
> I want a tag based language

ColdFusion with out a doubt. You can uses it as a tab based language or like JavaScript using CFScript blocks.

There are 2 free/OS versions aside from Adobe's offering, Railo (v.3.1 will be released Open source later this year) and Open Blue Dragon. They are all very mature and robust.

All three run on the any J2EE server and can be used in conjunction with Apache or IIS (Adobe CF and Railo being the easiest to set up) on any OS that can run an J2EE server. Linux, Windows, Solaris, Macintosh and even AIX (RS/6000).

Either way it is an easy language to pick up and by far the most productive language that I have ever used. Even though it is easy to learn it supports OO constructs that wont leave advanced programmers wanting.
