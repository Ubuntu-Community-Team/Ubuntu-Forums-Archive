### Post by sudodus on 2014-02-05
I suggest that you browse quickly the tutorials about bash and shell-scripts that you find. And then focus on one tutorial that suits your way of thinking and work out the examples.

Then make some own scripts, that you would find useful (maybe only as exercises, even if there are already programs doing it). Don't expect to learn everything at once, hardly anyone knows everything that can be done with shell scripting. I know quite a lot, but I am still a noob compared to some of the gals and guys at the Ubuntu Forums. Finally, feel welcome to ask for help about specific tasks or problems :-)
