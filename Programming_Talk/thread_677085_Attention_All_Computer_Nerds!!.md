I had pleasure to play with OLPC laptop for a while, and it is incredible fun. Many people misunderstood it - it is NOT a laptop for business traveler. It is not even a windows laptop. But it is excellent learning tool for someone who has plenty of time and curious inquisitive mind. [http://en.wikipedia.org/wiki/Minimally_Invasive_Education](http://en.wikipedia.org/wiki/Minimally_Invasive_Education)  ("Hole in the wall") proved that children are able to learn using computer without **any** supervision. [http://www.hole-in-the-wall.com/](http://www.hole-in-the-wall.com/)

I really liked how XO Sugar is desktop without file manager. I liked also eToys, which lets you to build programs in SmallTalk wrapped in GUI: it is possible to build programs simulating physics experiments, and look at the source (very GUI and very visual) and learn how it is done.

So as computer geek, you can create new lab experiments for those kids, and let them learn!
