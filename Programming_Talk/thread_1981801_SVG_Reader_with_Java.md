### Post by PaulM1985 on 2012-05-17
I don't see why there would be an issue with you writing your own.  "The SVG specification is an open standard that has been under development by the World Wide Web Consortium (W3C) since 1999." ([http://en.wikipedia.org/wiki/Scalable_Vector_Graphics](http://en.wikipedia.org/wiki/Scalable_Vector_Graphics)).

Since it is an open standard, people shouldn't have to pay to use the standard.  There shouldn't be any copyright issues preventing you from writing your software.

This project (first thing I found on google) seems to do what you want: [http://xmlgraphics.apache.org/batik/](http://xmlgraphics.apache.org/batik/)
