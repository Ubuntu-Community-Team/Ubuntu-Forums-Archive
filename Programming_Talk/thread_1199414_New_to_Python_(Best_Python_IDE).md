### Post by ideaplan9 on 2009-06-29
> **Freckletoe said:**
> So I'm just beginning to get into Python and I have some experience with Visual Basic, so I was wondering two things:

1. What is the best python IDE in your opinion?

2. (optional) any good sources that could give me some beginners help and tutorials.

1.  *Geany* if you really need an IDE.  I prefer using a text-editor and console myself when I do things in Python.  Lately I have been using *Geany* because there is a terminal embedded in the interface in the bottom.

2.  *A Byte of Python* is a free book that can be downloaded.  It's only a 150 pages but packs information that is a really good guide for people who have never programmed before.  If you have, then a good reference guide [http://rgruet.free.fr/PQR25/PQR2.5.html](http://rgruet.free.fr/PQR25/PQR2.5.html) can be found here.
