### Post by NoReflex on 2008-09-24
Hello!

I don't know a way to pass an ftp server to nautilus as an argument nor a file where you could save this. What you could do is :
- connect to your ftp server
- go to the Bookmarks menu and add a bookmark to the ftp server
When you restart nautilus your bookmark will be there and if you selected the "Remember forever" option for you password it should connect immediately. 
I hope this helps you somehow.
