### Post by Hellkeepa on 2010-03-05
HELLo!

You could use any image editor/viewer that prints the coordinates, Gimp being one of these. Personally, the only times I've made an image map I've used CuteMAP. If you google a bit, you should be able to find an older version that'll work nicely with Wine without any bloat/adware.
CuteMAP just makes creating image maps a breeze, and this comes from someone who's swearing to hand-written code and markup. ;-)

Happy codin'!
