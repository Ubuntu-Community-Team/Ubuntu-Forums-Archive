### Post by evymetal on 2008-01-05
Are you looking to do this so you can install the same setup on loads of workstations or so you can get back to your old settings after an upgrade?

If it's the first then I think there are GPL systems out there for doing it quite easily. (packaging your entire system up for an install elsewhere).
