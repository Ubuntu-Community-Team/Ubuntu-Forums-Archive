### Post by ankursethi on 2007-05-26
I don't know about how to do it from within the IDE (because I haven't used it) but to do it manually using the compiler from the command line, do :
> gcc path/to/file.c -o output-file-name
Then, to execute the file :
> ./path/to/output-file
