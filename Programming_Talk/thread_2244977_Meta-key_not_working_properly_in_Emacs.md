### Post by William_Bruschi on 2015-07-05
I'm starting to use Ubuntu in Parallels and encountered your meta twice issue in emacs. I highly customized my keyboard layout with several remappings, but what finally worked for me was to open the keyboard shortcuts menu in Parallels, view the Linux profile and add an additional mapping from Mac Alt to Alt.
