### Post by moephan on 2006-03-30
You'll need to learn the c++ syntax, and the c++ "way of thinking". IMHO, the best online resource for the latter is probably the [C++ faq lite]("http://www.parashift.com/c++-faq-lite/"). The former is rather easier to find in various [online course]("http://www.cs.rochester.edu/u/leblanc/csc173/review/CppPrimer.html")s, etc...
