### Post by DaithiF on 2010-10-25
I would recommend Vim.

Very powerful in its own right, plus there are thousands of plug-ins / scripts to extend it to behave in particular ways you might want.

For example one of Textmate's nice features is Snippets, and there are a couple of vim plugins (e.g. [http://www.vim.org/scripts/script.php?script_id=2540](http://www.vim.org/scripts/script.php?script_id=2540)) which implement Textmate style snippets.
