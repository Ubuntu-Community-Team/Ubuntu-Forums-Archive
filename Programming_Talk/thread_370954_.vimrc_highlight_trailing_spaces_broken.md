public class Main

For some reason the tabs are showing up as #;;;.

### Post by x1a4 on 2007-02-26
Hi,

Why not just use *sed* to remove the trailing spaces?  

```

sed 's/[ \t]*$//' ./infile > ./outfile

```

If your version of *sed* doesn't recognize '\t' (as some don't), replace '\t' with an actual tab.

EDIT: I just wrote this tcsh script which will remove your trailing spaces.  The syntax is: 

script /path/to/file-with-trailing-whitespaces

```

#!/usr/bin/tcsh
#
# ~/bin/dtw
# delete trailing whitespace (spaces, tabs) from file
#
set infile=$argv[1]
set tmpfile=/tmp/dtw

sed 's/[ \t]*$//' $infile > $tmpfile     # strip whitespace and send to tmp file
mv $tmpfile $infile                      # overwrite original file with finished product

unset {infile,tmpfile}                   # free-up memory by unsetting variables

```
