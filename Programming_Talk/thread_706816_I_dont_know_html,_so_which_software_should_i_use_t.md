### Post by Alpinist on 2008-02-24
Kompozer is in the repos and is a fairly easy to use and pretty good graphical tool for creating web pages.   The next easiest graphical tool would be Open Office which has a pretty decent html designer itself.

Quanta is an in between program.  I has a graphical tool for creating pages but requires some programming to achieve some of the same results you can get by just using the layout in the other programs.

Bluefish is more advanced and is more designed for people who would rather just learn html and code and gives you some tools to assist you.

And of course html is just plain text so if you want to just learn code you can use any text editor.

I would recommend starting at first with Kompozer.  (NVU and Kompozer are pretty much the same thing)

### Post by scruff on 2008-02-24
Seriously - it is VERY easy to learn HTML + CSS and do some nice web design. I learned both in a week like 8 years ago before knowing any other programming languages and while I was no pro at first, I certainly learned to do things the right way. 

I did begin using Dreamweaver on Windows and to be perfectly honest it is easier to learn to write the code yourself than to learn to be effective with one of those do-it-all WYSIWYG applications. Not only that, but with those applications it is too easy to wind up with a giant amount of sloppy code, error prone code.

Check out the tutorials at w3c: [HTML]("http://www.w3.org/MarkUp/Guide/") [CSS]("http://www.w3.org/Style/Examples/011/firstcss")
