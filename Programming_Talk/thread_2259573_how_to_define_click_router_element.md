### Post by schragge on 2015-01-07
Well, an obvious start point is to install [wireshark]("http://manpages.ubuntu.com/manpages/trusty/en/man1/wireshark.1.html"). Or [tcpdump]("http://manpages.ubuntu.com/manpages/trusty/en/man8/tcpdump.8.html"). Or both. Then learn how to use them. Besides, the Click router distribution already comes with an element [ToDump]("http://read.cs.ucla.edu/click/elements/todump") (in package userlevel) that writes packets to a tcpdump file.

 Without you providing more information, it's hard to suggest anything else.
