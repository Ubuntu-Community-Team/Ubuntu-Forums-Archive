The coolest thing I've done with Linux has been a CAD/CAM weaving program called [ArahWeave]("http://www.arahne.si/"). Anyone interested in handweaving and especially writing one's own drafts would be intrigued. Although written for jacquard and dobby loom weaving, it can be adapted for handlooms as well.
