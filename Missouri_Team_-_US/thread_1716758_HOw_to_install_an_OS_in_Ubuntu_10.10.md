### Post by Dark_Stang on 2011-03-28
You can do what's called dual booting. Both operating systems will reside on the hard drive, but you can only boot one of them up at a time.

Or you could run Fedora inside of Ubuntu on a virtual machine. This will be slower and requires much more RAM, but allows you to be much more brave in your test environment as you can run both operating systems simultaneously.
