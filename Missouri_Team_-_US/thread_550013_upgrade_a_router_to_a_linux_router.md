### Post by tgm4883 on 2007-09-13
Your title is kinda misleading.  The router already runs linux.  It's just that the 3rd party firmware has different features in it.  The fact that it already runs linux is why it is so easily able to run the 3rd party linux firmware.

:EDIT: 

I guess though that I should add that the firmware works great.  I wouldn't buy that router though, as there are others that work (some even with better specs)  

Check the site directly at [http://www.dd-wrt.com](http://www.dd-wrt.com)  It should also be noted that v24 of their firmware will be released soon (the just released v24 RC2) and it has many improvements and new features over v23.  Also, I am running it on a Buffalo router.  Linksys seems to have made that version more expensive than it should be, as what they did with that is change the WRT54G into a WRT54GL and raise the price $20.  When they did that they also changed what the WRT54G ran, so if you wanted to run 3rd party firmware you had to buy the GL version (or a different router).  I'm not saying that the WRT54GL is a bad router, but you can get better for less.
