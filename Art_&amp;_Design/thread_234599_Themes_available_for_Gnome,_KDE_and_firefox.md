### Post by asimon on 2006-08-11
The Crystal icon theme is also available for [Gnome](http://www.gnome-look.org/content/show.php?content=26331), [KDE](http://www.kde-look.org/content/show.php?content=25668), and [Firefox](https://addons.mozilla.org/firefox/916/). But the Firefox theme is outdated and doesn't work with current browser versions but there are some similar themes like mostlycrystal.

Then [Tango](http://www.gnome-look.org/content/show.php?content=31261), the tango-icon-theme package is already in Ubuntu and there is [Tango for KDE](http://www.kde-look.org/content/show.php?content=32288), [Tango for Firefox](https://addons.mozilla.org/firefox/1565/), [Tango for Thunderbird](https://addons.mozilla.org/firefox/2258/).
