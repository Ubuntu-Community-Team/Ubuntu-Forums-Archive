### Post by RSingh on 2008-06-16
First off, I find it very strange that such a feature in Ubuntu is not used nor much info is available. Instead of flaunting it as a feature, it was just left alone. I'm talking about the Wallpaper slide show in system in Ubuntu i.e. GNOME. For people who don't know, Its where the wallpaper changes its color depending on the time of the day. It was available as a default wallpaper in Fedora 8 & 9.

Anyway, I really like the feature and have created 3 wallpaper sets. **I did not create the actual images**, just the XML file that defines the transition rules. The pictures were created by an awesome artist, Petercui on Deviant Art. Here is the link to his page: [http://petercui.deviantart.com/]("http://petercui.deviantart.com/")

How to setup:
---------------

1. Extract the file, open the folder. There will be an .xml file and a "folder with the images".

2. Press Alt-F2 and type "gksu nautilus"

3. Browse to /usr/share/backgrounds

4. Copy the "folder with the images" here.

5. Right click on the "folder with the images" >> Properties >> Permissions >> make sure all entries are set to "Read and write"

6. Open the "folder with the images" and select all of the images and Right click >> Properties >> Permissions >> make sure all entries are set to read and write.

7. Now all you have to do is Right click on the desktop >> Change Desktop Backgrounds >> Drag the .xml file into the window here. (Be careful to keep the xml somewhere safe eg: yourhomefolder/Wallpapers)

8. Thats it. The wallpaper slideshow should be set. It will automatically change its colors depending on the time of the day.
