### Post by Thee on 2013-04-30
That's because you need root privileges to copy files outside of your home directory.

The easiest solution is to open up a terminal and type:

```
sudo nautilus
```

It will ask for your password, type it and press enter. A file manager window will open with root privileges.
**WARNING this will allow access to all your system files, you can screw up something if you are not careful! Make sure to close that window as soon as you copy your themes!**
