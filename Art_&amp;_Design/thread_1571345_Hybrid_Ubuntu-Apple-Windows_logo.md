### Post by funkwurm on 2010-09-15
Ow, Kerry, dude, you remade the Apple logo from scratch? No need to:

What I did for all 3 logos was Google for the biggest possible image with the logo in high contrast (I found them all black on white, perfect). Then in Gimp you can select all the white background, invert selection (you now have the logo selected), then do selection to path and export that path to SVG.

The rest is done in Inkscape, where I tweaked the individual logos to make corners hard corners again in stead of rounded corners etc. The challenge was really to get them all together into one logo where you can still recognize the individual logos. I had to mirror the Ubuntu logo for that, there was no other feasible way that looked half-way decent.
