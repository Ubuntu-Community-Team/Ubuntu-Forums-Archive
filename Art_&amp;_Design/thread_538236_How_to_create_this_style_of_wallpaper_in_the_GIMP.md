### Post by vanadium on 2007-08-30
You could do it in Gimp, but then you could be using layers for the different circles. This way, you could change the transparency of the different circles, or change their size and shape. However, basically, this is indeed best done in a vector graphics illustration program such as inkscape.

---

### Post by PrimoTurbo on 2007-08-30
I don't use multiple layers, all I do is use 1 base background gradient. Then I make a new layer and set the transparency of the gradient tool to low, make some circles and just pull around with a color to transparency using 2 colors a lighter one and a dark one. With this method you can make that image in as little as 2 or 3 minutes.
