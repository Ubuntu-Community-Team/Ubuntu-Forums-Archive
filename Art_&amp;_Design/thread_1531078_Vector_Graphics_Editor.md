### Post by mcduck on 2010-07-15
Xara was pretty nice, but also rather buggy (at least the last time I tried it) and also stacked with not-so-professional features like "push this button to turn your image into a supercool shiny 3D button" instead of focusing on getting the basic tools to work correctly (I did like Xara's way of handling relative color definitions, though).

Sodipodi is OK as well, but since is no longer under active development and all the work is now being put into Inkscape there's really no reason to use Sodipodi.

Synfig is a good one, but primarily intended for vector animation instead of still graphics. You might want to try it even for other than animation work if no other program seems to do what you want, but otherwise I'd just recommend using Inkscape.
