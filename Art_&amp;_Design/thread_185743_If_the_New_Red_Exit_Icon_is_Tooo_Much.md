### Post by j_baer on 2006-06-01
I  found the new exit icon to be too much. The bright red color didn't really blend
well with the rest of my ubuntu desktop so I put together something different.

View the attached before and after images to decide for yourself and if you agree download the install package.

**Installation:**

1. Download exit_icon.tar.gz to a local folder.

2. Extract contents.

3. Run *sudo sh inst-icon.sh* to install.

Logout, login to enjoy!
