### Post by st0n3cutt3r on 2008-04-15
After I started using AWM I was a little disappointed in the icons, and decided to check if I could change any of them.  Turns out I could, so I found my favorites on wincustomize and started using them instead.

[http://wincustomize.com/skins.aspx?libid=29&cat=Dock%20icons](http://wincustomize.com/skins.aspx?libid=29&cat=Dock%20icons)  --  This link takes you directly to the Icon gallery, which for the most part is .png or .ico files (both of which AWM can handle fine).

Also the Windows XP icons on interfacelift.com work nicely: [http://interfacelift.com/icons-win/index.php?sort=ratings](http://interfacelift.com/icons-win/index.php?sort=ratings)  (sort for highest rated so that you're more likely to see nice ones)

I do like a lot of the [mac icons]("http://interfacelift.com/icons-mac/index.php?sort=ratings") on interfacelift too, but I have yet to find a set on there that I can figure out what the filetype is, and a few of the archives are .sit?  and other filetypes that I am unfamiliar with.  If you're smart (or determined) and manage to figure out how to view the files on the mac-side reliably, please share.  For the rest of us, there's still the other two sites, and I'll do what I can to find the original OSX icon set that I've been using.  (I really really dislike apple, but they do have nice icons)
