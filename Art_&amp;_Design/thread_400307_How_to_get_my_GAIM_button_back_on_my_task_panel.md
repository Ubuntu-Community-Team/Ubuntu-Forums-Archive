### Post by H3g3m0n on 2007-04-03
You probably want to add a "Notification Area" from the add to panel selection,, this is the widget that gnome-panel uses to keep system tray icons like Gaim, network-manager, and the update-manager etc...

Otherwise check in the Gaim options that the icon is enabled.

### Post by brownknight on 2007-04-21
> **Pilot-Doofy said:**
> Yesterday and today I've been reworking my panels to make them look better and more usuable by me. Well, today I accidentally removed the GAIM button from my task panel. It was over in the right hand column of the panel. It enabled me to X out of the GAIM buddy list but still keep the program open.

Now, when I click X on the buddy list and try to start GAIM again, it does that, starts it again. How can I get this little program link to be down in my task bar again? I know how to add things to the panel, but I didn't see it in the list.
1. Check if system tray icon is checked in gaim. 
Open gaim > tools > preferences > interface tab > show system icon tray (select always)
2. Add or make sure notification is added to gnome desktop panel
Right click on panel > add to panel > utilities section > click on notification area.

That should do it.
