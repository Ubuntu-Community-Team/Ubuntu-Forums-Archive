Ok had a quick go with it ...... so many shards it takes a while to render ........ for something that seems so simple ...

in Winds3d start of with a cube .... Doo sabien subdivision first ..... gives a rough shape for a rock .....
select all faces  .... tesselate ....... triangulate .... 
pick a few diagonal vectors and start to re shape it .... as you want it to look .... this is just to get a rough shape ,,,
[http://i45.tinypic.com/10idsb9.jpg](http://i45.tinypic.com/10idsb9.jpg)
Next save it as a 3ds file and import it into Blender
go into edit mode and select all .... do a sub division of all faces
deselect ,,, pick random faces ... I did 50 % .... then choose fractal
rotate and mess about with the faces until you get something that looks something like ..... add a coal texture
and then render .... I added a couple of lights just to the right and left .... and I slightly added opacity 
and transparency to the model .... ( hope this helps )
