Eh.
Firefox themes are somewhat complex to create from scratch (ok, complex and t-i-m-e consuming) - so if that's what you want to do, oooh boy do you have a lot of work on your hands.  [url="http://kb.mozillazine.org/Theme_development"]A Massive Ton Of Work.
[/url]
If you want to just edit a theme, however, now that's not as bad.
1. Download a theme you like and want to edit.
2. Rename the .jar extension to .zip (that's all it is anyway), and extract the contents into a folder.
3. Edit the icons/colors ...

Here are some pretty good starter type pages:
[first steps]("http://cheeaun.phoenity.com/weblog/2004/12/first-steps-in-theme-design.html")
[tutorial]("http://www.eightlines.com/neil/mozskin/index.html")
