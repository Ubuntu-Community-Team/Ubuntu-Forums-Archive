hello there dude. finally!

regarding using pixmap column headers, you can append your gtkrc file...

style "column-header"
{
    engine "pixmap" 
    {
        image
              {...
              }
    }             
}
widget_class "*List"             style "column-header"
widget_class "*GtkTree*"    style "column-header"
widget_class "*GtkCList*"    style "column-header"
