### Post by EvilEntity on 2006-04-25
The Ubuntu Documentation Team is going to have its documents printed and available cheaply though on line book publisher Lulu in many different languages. The price of these books will be very cheap, only the production price + postage (around only $US10.00). We are looking for someone to design the covers for these books, naturally we thought the Artwork team might be interested!

Books to be printed include the Ubuntu Desktop Guide, Kubuntu Desktop Guide, the Ubuntu Server Guide and the Packaging Guide in just about every language they are translated in. We would like something we can use for all these guides for simplicity (the Kubuntu Desktop Guide can use different colours) and be able to ask the artist to modify the covers on a per language basis (we can give you the translated text to insert) - or give us the necessary files to enable us to make these changes ourselves.

For more information see [this post on the ubuntu-art mailing list]("https://lists.ubuntu.com/archives/ubuntu-art/2006-April/001180.html").

If your interested please reply on here with some examples, or feel free to email [email]ubuntu-doc@lists.ubuntu.com[/email] or even drop by #ubuntu-doc on the Freenode IRC network (irc.freenode.net).

Thanks!

Robert Stoffers - Ubuntu Documentation Team Member
