### Post by ice60 on 2007-12-23
i've posted about cursor themes before and i know no one is interested in them lol. here are some good ones, they have **GREAT** animations - instead of the egg timer the whole cursor spins around in a 3D way not around the centre as you look at it lol, when something is selected the cursor changes shape slightly. i can't explain it, but it's worth trying out.

all you do is download the archive, extract it, then open (or download) gursormaker, click on File>Import CursorXP Theme, browse to the extracted cursor themes, then go File>Export X Cursor Theme and tell it the name to call it.

then when you have your exported theme, you can maybe use gcursor, or compiz to select the cursor theme. i'm not using ubuntu atm, i'm using an old suse so i'm just copying the new cursors manually.
