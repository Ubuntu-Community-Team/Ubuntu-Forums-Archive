### Post by sayakb on 2008-03-21
This is called the Shuffle Desktop. It has a circular shuffle selector which can be rotated like a disc, with the outermost ring having the installed applications, the middle ring being the System paths, while the inner ring left for a custom applet. It has a customizable panel at the bottom which might resemble the KDE panel w/o a menu. The log-off and shut-down buttons are there on a bottom right tray which may be drawn out/inflated or pushed back in/deflated. The desktop supports widgets like the KDE4 environment.

[[IMG]http://img380.imageshack.us/img380/7175/atgaaaasqsft3xekvhkfetpnp1.th.jpg[/IMG]]("http://img380.imageshack.us/my.php?image=atgaaaasqsft3xekvhkfetpnp1.jpg")

Please note that this is just a mockup (a concept) published at the Wiki by me.
[https://wiki.ubuntu.com/Artwork/Incoming/Hardy/Alternate/ShuffleDesktop](https://wiki.ubuntu.com/Artwork/Incoming/Hardy/Alternate/ShuffleDesktop)

### Post by k99goran on 2008-03-24
A few thoughts...

1. This looks like something that could be implemented in [Avant Window Navigator]("http://en.wikipedia.org/wiki/Avant_window_navigator") as an alternative to the dock.
2. It could be hidden by default and accessed if either the mouse-pointer moves to the corner in which the radial menu is positioned, or if a certain keystroke is entered.
3. The user should be able to create new rings and add whatever icons he/she wants.
4. Scrolling could be reduced and in some cases eliminated if:
4a. The order in which the rings are placed could be automatically tied to the number of icons in each ring (optional feature). More icons -> outer rings, less icons -> inner rings.
4b. The scale of the icons in a specific ring could be based on the number of icons in that ring.
5. Temporary outer rings could be created if for instance a person clicks on a 'docked' folder. The new temporary ring will display the files in that folder.
