### Post by Dragonbite on 2009-04-30
I ran across this application today and it intrigues me for two reasons;[LIST=1]
[*]It is another photo editing application for Linux
[*]It is a paid-for application for Linux
[/LIST]

First, the application is called [[COLOR="Blue"]LightZone[/COLOR]](http://www.lightcrafts.com/linux/index.html) and it states > Whether you’re a photographer by trade or by passion, LightZone enables you to retouch and enhance photos in a simple and natural way, without the need to learn a new editing language or mathematical representations that have no meaning to most of us.

It's always good to see alternative applications available for Linux.

The other part, that it is a paid-for program, is very exciting because if this were to take off then this may help other, traditional software makers to not write off Linux thinking that it HAS to be GPL'ed and Open Source.  

Imagine if Adobe realized that they could sell Photoshop and Dreamweaver to Linux users without having to give anything away? Of course there would still be Open Source applications but for some people they need (or want) the fully professional versions and that is what holds them back from being able to choose their platform of choice.

If this were to start happening, then people would be able to choose the operating system they WANT, to choose the one that fits them the best and not just what their applications run on.

I hope more companies start selling software to Linux users, and Linux users start buying some to make it worth while!
