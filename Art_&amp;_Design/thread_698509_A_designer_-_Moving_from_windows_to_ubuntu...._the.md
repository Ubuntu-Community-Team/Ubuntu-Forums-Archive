### Post by jrharvey on 2008-02-16
First of all I would check out UBUNTU STUDIO. Its an ubuntu based distro that is centered around art, design and music. It comes with all the apps that you could want for design (blender, inkscape, movie/music editors) and many more. As an architecture student I have been looking for all sorts of alternatives to windows apps. I mainly use Cad programs, 3D modeling programs and Vector based apps and linux has them all. I feel that I could move everything to linux if i really wanted but my school uses apple and windows based apps that are not available in linux. Yes, there are alternatives but its hard to change what you already know. For these programs I either Dual Boot or use virtualbox. Virtualbox is basically an emulator that can run windows while you are in ubuntu and its surprisingly fast compared to VMware. Its works really good unless you are building very complicated 3d models. Its hard for me to reach that limit so im sure you will be fine. If you are running your own company and it is small i dont think it would be all that hard to make the switch and it will save you tons of money.

### Post by dmn_clown on 2008-02-17
There is also ArtistX and 64studio though realistically all distributions can be used.

Many commercial apps have been ported to linux, like Maya (if you can afford it) and Corel Draw 9 (if you can find it, they dropped support long ago).

Your first order of business should be to see if you really need to switch, XP still receives security updates and realistically you shouldn't need to update all of your graphics apps to meet your customer's needs.  The second order is to see if the software you are already using and familiar with has been ported.  If those can't be met then you need alternatives, a fairly good list of graphic apps (both commercial and free) available for linux can be found [here](http://www.linuxmovies.org/software.html) in the sidebar.

### Post by mendozaro on 2008-02-17
Thank you all for you'r time...

Now... 

First i whant to say that a change like this can't come over the night. So i already have a virtual machine running with ubuntu. And i am seriously thinking of buying a box just for ubuntu. I did try other alternatives like PClinuxos, debian and fedora - top 4 in distrowatch.com. Out of all ubuntu seems the best. Or atleast it feels like more mature and more polished.

Out of the programs you all enumerated... i tryed:

- GIMP (fireworks replacement) - nice program, complex, can make all i need. But it is kinda slow at editing large pictures, many layers. Also all seems more complicated in gimp. I had to read a tutorial on how to add shadows or how to add outline to a text. So in conclusion this could be good for editing images...just needs about 1 month to get used to. Gimp has windows version so i installed it and start to complete some tutorials.

- Cinelera (sony vegas replacement) - some sort of unfinished student project... it feels to me. It took me alot only to install it... never did managed to actualy edit anything.

- kdenlive (sony vegas replacement) - much more advanced and in the same time easyer to use. But it doesn't render the videos as it should. Changes frame rates while rendering so in the end... i get a clip that has the sound track longer than the video part. This could be the best replacement for Sony Vegas ever. Needs more polishing from it's dev's. Also i should state that in order to get videos from a DV camera one would require to use KINO also (this one works great).

- flash replacement - there is none... but flash works well in wine. This is one of the programs i whant to quit using. Flash artwork is not what i intend to sell in the future. I am more oriented towards large mysql / php driven web sites. The problem now is that is still have clients that have flash web sites witch  still need to be maintained and updated. I'll turn those sites (sooner or latter) in php / html web sites.

- Blender - good replacement for 3d studio. This one (like Gimp) has windows version... so i'll start doing some tutorials.

- Corel Draw replacement - i didn't try InkScape... I did try Xara. Good replacemnt but i have a problem regarding the work space. I can draw in coreldraw huge plans in real scale... like using mm or m. I didn't find that in any replacement for corel draw i tryied.

GENERAL CONCLUSION: As i see this "move to ubuntu" now... it will take me (or any designer) about 4-6 months. Ofcourse after all this time i will still be slower using linux than using windows. But on the other hand... NO VISTA, NO HARD EARNED $$$ for proprietary software, NO FIREWALL, NO ANTIVIRUS... and the NO list could continue. I think all independent designers should think about this... The lost time for learning new ways is an investment into the future.

About UBUNTU STUDIO: It doesn't offer me nothing more than standard UBUNTU. I have the feeling that this is a lost cause project.

I forgot in the original post to ask about Dreamwaver... did anyone that used to work with this one find a good replacement for it?
