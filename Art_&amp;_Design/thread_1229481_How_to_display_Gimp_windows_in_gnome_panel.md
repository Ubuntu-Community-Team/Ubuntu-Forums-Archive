In new version of Ubuntu, the Gimp toolbox, layers etc. windows don't have their entries in Gnome panel. How can I revert this behaviour. ### Post by Giant Speck on 2009-08-02
> **GeoRW said:**
> In new version of Ubuntu, the Gimp toolbox, layers etc. windows don't have their entries in Gnome panel. How can I revert this behaviour. It makes me really angry when I cannot change focus to these windows with ALT+TAB or when I cannot move these windows to another desks. Please help. This new feature is rather bug than improvement. Thanks.

1. Go to Edit -> Preferences
2. Click on "Window Management."
3. Set both "Hint for the toolbox" and "Hint for other docks" to "Normal Window."
4. Click OK.
5. Restart GIMP (for good measure)
