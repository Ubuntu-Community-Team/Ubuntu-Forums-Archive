Sorry, but there's no such option in the latest Ubuntu versions (9.10 and later).

In older versions you could enable/disable menu icons through a gconf key (desktop/gnome/interface/menus_have_icons) but it doesn't work correctly any more. Now it only switches between all menu entries having icons and only pre-defined entries having icons instead of allowing you to completely disable the icons.
