If I get your question: Can a computer handle full CPU load for 24 hours or more? Then the answer is, if your computer is not particylarly dusty (not well ventilated / cooled) then it should handle it very well. If it runs one hour, it will probably run a week. I've done some video transcoding which took more than a week with 2 cores running at 90% or more, and there was no problem.

Modern CPUs are nice. They will protect against overheating by throttling (lowering it's speed) or even shutting itself down, if needed. You shouldn't worry.

### Post by meborc on 2008-04-19
usually desktop computers are better suited for those kind of tasks... notepads tend to heat up more just due to the size of the cooling bay and the fans etc... notepad manufacturers say that notepads are ment to be used on the road and mobile, so there is no need to run 99% cpu for 5 days :D ... lets just say that i wouldn't dare to try it with my notepad, even though i have a decent c2d 2GHz cpu... when i run some games like enemyterritory, my cpu heats up to 70C, which is way too high...

i would check the temp of your cores from time to time, the manufacturers have set a limit of around 85C for the cores to make the computer shutdown when overheated... but the cores will get damaged long before that :(
