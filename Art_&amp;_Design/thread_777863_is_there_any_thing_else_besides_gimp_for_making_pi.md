You are looking for a vector-based 2D drawing program, not a tool for image manipulation.

Bitmap-based image manipulation, photo retouching: GIMP
Film retouching: CinePaint
Vector-based 2D drawing: Inkscape, OpenOffice.org Draw
Vector-based 2D drawing, desktop publishing: Xara LX
Desktop publishing: Scribus
Presentation graphics: OpenOffice.org Impress
Vector-based 2D animation: Synfig Studio
3D modeling: Wings 3D
3D modeling, animation, video editing: Blender
Image format conversion: ImageMagick, GraphicsMagick
Video format conversion: FFMPEG, MEncoder, Avidemux

This is some of what free software has to offer in the graphics department. While there may be overlapping functionality in these packages, picking the right tool for the job will yield the best results.
