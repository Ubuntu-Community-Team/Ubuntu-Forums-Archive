### Post by Darkstar_444 on 2008-10-05
Couldn't find the script on a Google search, so i have uploaded the scripts on my Gimp version 2.4.5 to the file host below. B&W Film Simulation plugin should be in there.

[http://stashbox.org/250308/scripts.zip]("http://stashbox.org/250308/scripts.zip")

Go to Places/Home Folder, press Ctrl-h, add script to your .Gimp-2.6/Scripts folder.

Hope this helps
