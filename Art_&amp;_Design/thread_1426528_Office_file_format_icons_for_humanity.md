### Post by Merk42 on 2010-03-14
> **ElSlunko said:**
> I like your alternatives, but it hit me that the document icon just reflects the icon of the program that it's meant for so it KIIIINDA makes sense to copy the word logo but I guess it should copy the official logo.

Problem is there is no **one** logo. the logs for the Mac versions (currently used in Lucid) are very different from those in the windows version.  I'll use Word as an example:
**Word for Windows**
[img]http://upload.wikimedia.org/wikipedia/en/thumb/b/bb/Microsoft_Word_2010_Icon.svg/64px-Microsoft_Word_2010_Icon.svg.png[/img]
**Word for Mac**
[img]http://upload.wikimedia.org/wikipedia/en/thumb/8/87/Word_Mac_2008_icon.png/64px-Word_Mac_2008_icon.png[/img]
Since we can't have it be both the Mac and Windows versions, it should be neither (or arguably Windows since that's more market share).  Instead copying one or the other, the icons should play off the similarity of the two, ie blue W for Word, Green X for Excel, etc.

Show you like them by commenting in the bug or at least saying it affects you too :D
