### Post by kayosiii on 2008-11-30
I use Open Source Software to do payed design work so I guess I count. (for print even)

My primary design tool is Inkscape. I used to primarily use Corel Draw on windows as I prefer the bitmap in vector approach to the vector in bitmap approach that Photoshop uses ( I like to keep my designs as resolution independant as possible). 

I use the gimp specifically for doing pixel based manipulations that I can't do in Inkscape and in truth I would use it a lot more if it supported 16 bit per channel images.(even photoshop's 16bit support could be a lot better).

I don't find the gimp's interface particularly non-intuative but it is different enough from photoshop in some areas for people to have trouble.

With Inkscape being object based and having both layers and being able to group objects - makes layer groups unneeded.
