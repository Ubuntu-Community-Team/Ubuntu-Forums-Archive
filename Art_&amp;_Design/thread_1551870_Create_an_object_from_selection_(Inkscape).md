### Post by mcduck on 2010-08-13
Since it's a bitmap image you are working with, you really need to use some bitmap editor (like Gimp) if you want to break the image into separate objects.

Inkscape is a vector graphics editor, and doesn't have any tools you need for bitmpap editing.

You can of course import stuff back to Gimp after you have separated the things you want into different image files. If you did a good job separating the text in Gimp, you'll be able to use the Trace Bitmap function in Inkscape to convert it into a vector object.

(well, you *can* try to trace the image as it is, and then delete every unnecessary object, but depending on the image getting anything else than a mess could be very hard)
