### Post by diatribe on 2007-05-10
when you install through the theme manager, just drag the tar file onto the window and it will work, or select the tar file to install from.  also, you can extract the files (with the path, using file-roller or xarchiver) into your ~/.themes directory.  this works for tarballs with multiple themes in them
