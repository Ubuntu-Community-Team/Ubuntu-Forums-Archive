### Post by K-Man_22 on 2007-07-26
So far I have only tried the german Windows version of Qtpfsgui but this is what I can tell you. You want to make a (jpg) picture from your HDR file not by 'saving as' but by using the function 'Tonemap the Hdr'. Hopefully your version will not crash there as mine does. Will try with Ubuntu next.

Be advised to read the manual since you can vary the results of the tonemapping process by entering different parameters.

Pls post back if you succeeded or not.

### Post by Darxus on 2009-08-27
You can save to jpeg without tone mapping by doing File / Save HDR Preview.  This retains the entire dynamic range of your image, compressed by contrast reduction, with none of the artifacts or surrealism of local tone mapping.
