### Post by hikaricore on 2007-06-11
[http://appdb.winehq.org/appview.php?iVersionId=2631](http://appdb.winehq.org/appview.php?iVersionId=2631)

PS CS2 is kinda a so/so thing in WINE.  I believe there has been progress in CrossOver with it though.

Probaby your best and most free(ish) bet is to run a VM ontop of linux with something like [VirtualBox]("http://www.virtualbox.org/wiki/Downloads"), as it will run perfectly in the VM (where you'll obviously need to be running some version of Windows).  What's great about VirtualBox is it now has better support for using physical drives along side virtual dives which is huge when it comes to being able to access your files.  It's also not huge and combersome like VMware.
