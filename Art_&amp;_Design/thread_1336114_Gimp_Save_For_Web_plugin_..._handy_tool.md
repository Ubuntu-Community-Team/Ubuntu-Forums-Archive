### Post by newbie2 on 2009-11-24
> I used the Photoshop’s Save For Web feature extensively as it enables me to optimize my screenshots to the smallest possible size without affecting their quality to a great extent.

Luckily, GIMP supports plugins and there is this plugin available for this purpose.

If you are using Ubuntu (or any other debian-based distros), you can easily install the Save For Web plugin via the following command:

sudo apt-get install gimp-plugin-registry

Better still, click on this link to install.
[http://maketecheasier.com/enable-photoshop-save-for-web-feature-in-gimp/2009/11/22](http://maketecheasier.com/enable-photoshop-save-for-web-feature-in-gimp/2009/11/22)
