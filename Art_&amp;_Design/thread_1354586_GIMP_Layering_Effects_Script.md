### Post by Exodist on 2009-12-14
This could be one of the best scripts for new and experienced users of GIMP. It adds many functions very much like Photoshops layer attribute options. This is a must for all GIMP users.

(THIS NEEDS A STICKY)

> **Layer Effects**

                                                               January 24, 2008 - 21:43 — JonStipe           This is a series of scripts that implement various layer effects:
 
[LIST]
[*]Drop Shadow
[*]Inner Shadow
[*]Outer Glow
[*]Inner Glow
[*]Bevel and Emboss
[*]Satin
[*]Color Overlay
[*]Gradient Overlay
[*]Pattern Overlay
[*]Stroke
[/LIST]
 Some effects can have contours (a.k.a. curves) and noise added to them. Effects can either be applied on a separate layer, or be merged into the active layer (which in some cases gives better results).


Link = [http://www.pixel2life.com/viewtutorial/65633/create_a_transparent_glossy_lettering_with_the_gimp/](http://www.pixel2life.com/viewtutorial/65633/create_a_transparent_glossy_lettering_with_the_gimp/)

There're also 2 plugins i use really often for photography work :

* Lomo plugin (to create a [vigneting]("http://en.wikipedia.org/wiki/Vignetting") effect) :
[http://registry.gimp.org/node/7870](http://registry.gimp.org/node/7870)
* Exposure Blend plugin to create an exposure blending in few clicks :
[http://registry.gimp.org/node/6708](http://registry.gimp.org/node/6708)

I use the first one really often and it is for me one of the greatest GIMP plugin, very efficient and customisable.

In a general way you can find plugins for almost everything there:
