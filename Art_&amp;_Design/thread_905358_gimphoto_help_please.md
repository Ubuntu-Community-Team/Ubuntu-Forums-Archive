- Install GimPhoto 1.4.3
- Install GimPad 1.1
- Run GimPad (DON'T Run GimPhoto) because GimPad that will make GimPhoto appeared as single MDI window ala Photoshop.

sorry for Linux users, because GimPad depend too much to Win32 API then for now GimPad still not available for Linux :(
