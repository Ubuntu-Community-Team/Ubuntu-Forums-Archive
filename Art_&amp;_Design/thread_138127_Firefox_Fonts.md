### Post by bswilson on 2006-03-01
You need to install the **Microsoft TrueType Core Fonts** package.  You can do this with Synaptic Package Manager by searching for *msttcorefonts*, or you can do it via the command line:

```
sudo apt-get install msttcorefonts
```

This package installs these fonts onto your Ubuntu system:

[LIST]
[*]Andale Mono
[*]Arial Black
[*]Arial (Bold, Italic, Bold Italic)
[*]Comic Sans MS (Bold)
[*]Courier New (Bold, Italic, Bold Italic)
[*]Georgia (Bold, Italic, Bold Italic)
[*]Impact
[*]Times New Roman (Bold, Italic, Bold Italic)
[*]Trebuchet (Bold, Italic, Bold Italic)
[*]Verdana (Bold, Italic, Bold Italic)
[*]Webdings
[/LIST]
