### Post by beesthorpe on 2008-03-08
I agree, indeed I think all themes should support this.  As I understand it the implementation's quite trivial. To change the application background to allow it to be set with the appearance settings you would, for example, change "bg[NORMAL]= "#eaeaea" in the gtkrc file to "bg[NORMAL] = @bg_color".  There's a bit more to it than that but not an awful lot, if you were feeling brave you could have a go at it yourself :)
