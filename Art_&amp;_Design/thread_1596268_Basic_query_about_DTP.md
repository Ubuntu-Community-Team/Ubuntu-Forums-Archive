### Post by Newfoundlander on 2010-10-14
I'm about to go to bed so I'll have to keep this short for now... I've been using Scribus under Ubuntu to lay out books and magazines for the last three years.  Before that, I used PageMaker on Windows for about 14 years.  I've had far fewer problems with Scribus in creating proper PDF files.  There is no way to import templates or publications from Quark/PageMaker/InDesign so you will just have to create new templates in Scribus.  Search YouTube and you will find lots of tutorials and tips for doing layouts in Scribus.  Let me know if you have any questions.
