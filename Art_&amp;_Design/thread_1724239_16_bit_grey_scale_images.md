### Post by BcRich on 2011-04-08
Hi starbase1
nip2 is for working with large scale images on computers that don't have a lot of RAM. although it's not a traditional image editor as it's interface resembles a spreadsheet more than something like the GIMP or photoshop it still has many of the standard image editing tools. You might need to do a tutorial if you plan to use it. nip2 is available from the repositories it is actually a GUI for a library called VIPS the website is [http://www.vips.ecs.soton.ac.uk/index.php?title=VIPS](http://www.vips.ecs.soton.ac.uk/index.php?title=VIPS)
the [quick interface tour]("http://www.vips.ecs.soton.ac.uk/supported/current/doc/html/nipguide/nipguidese1.html") link provides a decent description of how to use nip2
