### Post by outofsync on 2010-12-09
I don't know where would be the place for congratulating the team responsible for the desktop design.

Today I upgraded to 10.10, from an older release which still had the older "human" looks. I've been using Linux since the days of Windows 3.1, beginning with Slackware, and today is the first time that I see the Linux GUI look like a really serious machine.

I don't remember this same feeling of "serious GUI" since the days of IRIX. As an IRIX user, I really loved its desktop because it gave you the feeling that you were using a very professional environment.

Today I feel a simmilar experience with the default look of Ubuntu 10.10. I never experienced this on Linux before (yep, I know there're millions of themes in the galaxy, but first of all, my work isn't searching for themes, and second, I guess this feeling isn't just caused by the new theme, but by the overall feeling of the current Ubuntu distribution).

My most sincere congratulations, and I also wish a huge success with Unity (didn't try it yet, but I like its philosophy and the first screenshots, and hope to see it soon working in desktop Ubuntu).

And, btw... please fix the Gnome panels so they can be used in vertical position without having to apply manual patches :D

Sincerely,

outofsync
