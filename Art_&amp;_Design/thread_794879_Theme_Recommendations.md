### Post by CarpKing on 2008-05-16
I too have looked for something like this with little luck.  I have found a few things that may be of interest, though.  

[Muku](http://gnome-look.org/content/show.php/Muku+GTK?content=59034) is a GTK and Emerald theme with a wooden look.  The screenshot uses icons from [here](http://www.mikworks.com/originalwork/edo/) and [here](http://en.meijitaisho.net/icon/).  

There's a really slick looking icon set called [Icons of the Library](http://aiolorwile.deviantart.com/art/Icons-of-the-Library-70721899) that hasn't been packaged for Linux, but it is available in 512x512 PNGs that could be resized.  A Windows version (which has some extra small icons) is [here](http://aiolorwile.deviantart.com/art/Icons-of-the-Library-Windows-C-79915833).  A different and much smaller antique set is [here](http://paradis24434.deviantart.com/art/Antique-Icon-update-72437894).  

Also, it might get this thread more attention if you could get the title changed to something more descriptive.
