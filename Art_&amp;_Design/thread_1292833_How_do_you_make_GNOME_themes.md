### Post by mcduck on 2009-10-19
> **s3a said:**
> How do you make GNOME themes? Is there a GUI application to help in the process? Is it difficult?

Any input would be greatly appreciated!
Thanks in advance!

Both GTK and Metacity themes are usually done by editing the theme description files by hand. There is no GUI for the purpose, although there are some apps that allow limited editing of colors in already made themes.

The best way to learn how to do it is to start with some existing theme you like, and start changing things.

What comes to the level of difficulty, that's a bit hard to say. If you at ease with editing configuration fiels by ahnd then you shouldn't have too much problems. The syntax is simple and all normal themeing stuff is pretty easy to learn. On the other hand, GTK and Gnome has many things that could be controlled individually but actually getting the required information can be quite hard unless you are a programmer familiar with GTK.

Icon themes are fairly simple and straightforward, but actually creating huge numbers of good looking icon images takes a lot of work. But the same advice applies here, take one existing theme and start changing the icon files one by one.
