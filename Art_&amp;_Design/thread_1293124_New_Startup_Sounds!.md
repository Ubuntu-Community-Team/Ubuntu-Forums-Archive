### Post by daas88 on 2009-11-01
Yes, that dialog is gone, besides some others (services for example). You must replace the current login sound. Do this in terminal:

```
sudo mv /usr/share/sounds/ubuntu/stereo/desktop-login.ogg /usr/share/sounds/ubuntu/stereo/desktop-login-backup.ogg 
```

Let's suppose you have the sound you liked in the desktop, and its name is owl.ogg:
```
sudo mv /home/USER_NAME/Desktop/owl.ogg /usr/share/sounds/ubuntu/stereo/desktop-login.ogg
```

You could do it too by running nautilus with root permissions and copying your sound to this folder /usr/share/sounds/ubuntu/stereo/ with this name "destop-login.ogg".
