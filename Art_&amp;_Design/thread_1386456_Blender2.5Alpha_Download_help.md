### Post by thatguruguy on 2010-01-20
It may be in your best interest to forego installing the alpha of 2.5. Wait until the stable release comes out; they'll give you a .deb file for it when it's ready, which will make it really easy to install.  

In the meantime, blender 2.49b is pretty good.  Get it here: [http://www.blender.org/download/get-blender/](http://www.blender.org/download/get-blender/)

You'll notice (if you look) that there's an ubuntu-centric .deb file listed on the page with the other Linux downloads.  Again, it's helpful to know if you're running 64-bit or 32-bit. However, it's easy enough to find out when you download the file.  The 64-bit version won't install if you're running 32-bit ubuntu, and vice-versa.
