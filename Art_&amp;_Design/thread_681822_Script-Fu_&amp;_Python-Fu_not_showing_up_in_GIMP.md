### Post by diaa on 2008-01-29
As far as I can tell, Starting with GIMP 2.4 they removed the Script-Fu and Python-Fu menus for better menu structure, so unless you have a custom script that requires to be shown in Python-Fu or Script-Fu menus you won't see any of them, the functions that used to be accessible through these menus are now in their logical place

quoted from [GIMP 2.4 Preview]("http://www.redhatmagazine.com/2007/10/23/gimp-24-preview/")
> 
... the menu is reorganized with a better structure (the cryptic Script-Fu and Python-Fu have disappeared, with the functions moved to a more logical place), and the dialogs now better follow the GNOME HIG.
