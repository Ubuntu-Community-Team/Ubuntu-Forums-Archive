### Post by AdotB on 2010-03-20
If you download fonts in the common formats like ttf or otf, then you should be able to install them in both windows and ubuntu by placing them in the right places

Ubuntu:
user:   $HOME/.fonts
system: /usr/share/fonts/truetype/
or use synaptic/apt if there is a package

Windows:
drag the fonts into:
    Control Panel -> Fonts
or: 
    %windir%\fonts
(I think those are right)

As for where to get them, as mention above, dafont.com is probably a good place. (Pay attention to the licensing if using them professionally of course)
Here are some that I have used:
[http://www.droidfonts.com/droidfonts/](http://www.droidfonts.com/droidfonts/) ([http://android.git.kernel.org/?p=platform/frameworks/base.git;a=tree;f=data/fonts;hb=HEAD](http://android.git.kernel.org/?p=platform/frameworks/base.git;a=tree;f=data/fonts;hb=HEAD))
[https://fedorahosted.org/liberation-fonts/](https://fedorahosted.org/liberation-fonts/)
[http://www.josbuivenga.demon.nl/index.html](http://www.josbuivenga.demon.nl/index.html)
