### Post by psyopper on 2007-07-25
I hope I'm not breaking any license agreements here. Presumably since it's on Gnome-look it's under some form of the GPL, so I took this compiz-fusion icon idea set (that unfortunately didn't make it) and made them into icon-ish sized (120x120 pixel) png's with transparency. If you need to adjust their size to match other icons on your desktop you can do so in your favorite image editor. If you use them in the Gnome toolbar it will automatically scale them for you. 

As required by GPL, the original artwork can be viewed at [http://gnome-look.org/content/show.php/Compiz+Fusion+logo+Idea+2?content=61266]("http://gnome-look.org/content/show.php/Compiz+Fusion+logo+Idea+2?content=61266")

I will ,of course, remove these at the request of either the original artist or gnome-look.org.

Also attached is a screenshot of my Gnome toolbar so you can see what they look like up (or down) there. These make a great accompaniment to the basic but quite functional [switch compiz/metacity script on this forum]("http://ubuntuforums.org/showthread.php?t=489341").

Make sure you click on the image to view the larger version to save from, these are the right size but some how the color isn't quite right.
