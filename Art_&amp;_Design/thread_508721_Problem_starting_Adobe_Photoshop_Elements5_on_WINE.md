### Post by hikaricore on 2007-07-25
Well I don't believe PSE5 works using WINE, so that would be why it won't start.

It looks like someone has gotten version 2 working (somewhat) but version 4 does nothing but crash.
Version 5 is not listed, but I'd imagine that it has about the same luck running at version 4. [http://appdb.winehq.org/appview.php?iAppId=1794](http://appdb.winehq.org/appview.php?iAppId=1794)

You're probably better off installing Qemu or Virtualbox and running it on a virtual machine.
