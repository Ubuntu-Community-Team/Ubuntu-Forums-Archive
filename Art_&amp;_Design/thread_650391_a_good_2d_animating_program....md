### Post by stephantom on 2007-12-27
The problem is that Adobe's SWF is a vendor-specific format. You can run Adobe Flash using Wine (there are some howtos) quite okay I guess.
There are a few libraries for Linux that can write to SWF files but I don't know about a decent GUI. Synfig is very nice (because it is - like Flash - a vector animation application), but it has no export for SWF, too.
