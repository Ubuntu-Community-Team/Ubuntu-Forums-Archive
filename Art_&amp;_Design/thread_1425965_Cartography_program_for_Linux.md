### Post by Choragos on 2010-03-10
If you have digital elevation models (which you can download from seamless.usgs.gov), you can plot these with a phenomenal program called Generic Mapping Tools (GMT) made by the University of Hawaii.  If you google GMT it should be easy to find.

The software is cumbersome, but if your end result is to produce gorgeous plots with data and such, it can't be beat.  It's also good for plotting any 2-D surface--it's how I generate all my scientific plots for publication.

This is an example, but I had to downsample and make it a jpeg so the quality isn't spectacular.  You can define whatever DPI you want for output though.
