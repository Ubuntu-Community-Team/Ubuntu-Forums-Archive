### Post by fabiopimenta on 2009-06-17
Hello Friends!
I get to install human-theme on debian lenny.
Follow steps:
1) get package human-theme_0.18_all.deb on: [http://packages.ubuntu.com/pt/hardy/all/human-theme/download](http://packages.ubuntu.com/pt/hardy/all/human-theme/download)
2) sudo dpkg -i human-theme_0.18_all.deb
You will get a error message of dependencies(  gtk2-engines-murrine gtk2-engines-ubuntulooks murrine-themes)
to fix this you must type:
sudo apt-get -f install
Good Luck!
