### Post by tretle on 2008-09-20
Anjuta is an integrated development environment which is currently in need of some love from the art community, tango icons splash screens etc are all welcome. It is part of core gnome and offers support for java, c++, c, vala and python.

Currently there is work being done on creating a start page for anjuta. There is some code done but hopefully it will be migrated to a webkit backend soon. The content of this start page is still in discussion but the idea is to give links to useful functions like creating new projects etc. More imaginative integration with gnome docs, launchpad etc would also be a great idea. 

The current mockup for anjuta start page can be seen here, this is a very basic first attempt. 

[http://bugzilla.gnome.org/attachment.cgi?id=110805&action=view](http://bugzilla.gnome.org/attachment.cgi?id=110805&action=view)

If anyone would like to take a stab at creating a mockup and posting it here I could pass it onto the devs. 
There are no restrictions on color pallets, and you are encouraged to not use as much purple as shown in the mockup above.
