### Post by satya164 on 2012-06-03
I was thinking to make a script which can ask the user for his preferred colors and automatically create a theme accordingly. Why? Some of my themes, like Minty were popular, but users wanted more colors. But it was not possible to make the theme with each and every color suggested. And it would rather be better if anyone could use his preferred color.

Then I came across an article in [OMG! Ubuntu!]("http://www.omgubuntu.co.uk/2012/05/chameleonic-ambiance-script-matches-gtk-theme-to-your-wallpaper"). This gave me some inspiration. Then I thought of making a Chameleon Gnome Shell theme. Then I made one for Gnome Shell and one for Cinnamon and I hope now everyone will get to enjoy some Chameleon goodness.

The themes can change colors according to the current GTK theme, current wallpaper or a user defined color. If it fails to get the color from the GTK theme or wallpaper, the user defined color will be used instead. After installing, start the background process from the menu just once. The theme is added to startup items and will start at login. Whenever it detects a wallpaper or GTK theme change, it will change the color according to your preferences.
