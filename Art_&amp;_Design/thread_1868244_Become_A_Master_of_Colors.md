### Post by dhor on 2011-10-24
... with a little help of new application Delaboratory 0.5:

[IMG]http://404.g-net.pl/wp-content/uploads/2011/10/delab_after-300x183.jpg[/IMG]

This neat tool allows you:

- floating point precision (32 bits per channel) 
- native support for 9 color spaces (RGB, BW, CMYK, CMY, LAB, LCH, XYZ, HSL, HSV) 
- fully non-destructive editing (layers) 
- input/output - 16-bit TIFF (or 8-bit JPEG)  
- curves 
- channel mixer 
- advanced blending (normal, multiply, screen, overlay, overlay inv, add, add inv, sub, difference, darken, lighten) with masks 
- apply luminance / apply color in RGB colorspace 
- blur, unsharp mask 
- color samplers, histograms

Source can be found at: [http://code.google.com/p/delaboratory/](http://code.google.com/p/delaboratory/)

PPA can be found at: [http://launchpad.net/~dhor/+archive/myway]("http://launchpad.net/%7Edhor/+archive/myway") (Lucid/Maverick/Natty/Oneiric)

Simply: 

```
sudo add-apt-repository ppa:dhor/myway
sudo apt-get update
sudo apt-get install delaboratory

```
