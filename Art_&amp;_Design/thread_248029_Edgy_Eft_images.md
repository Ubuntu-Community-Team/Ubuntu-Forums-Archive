### Post by CarpKing on 2006-09-01
Nice pictures.  When I heard that the next release was to be named after the eft, I was excited.  I've always found the newt life cycle to be interesting. Most amphibians only have an aquatic larval stage and a semi-terrestrial adult stage.  Some North American newt species, however, develop from the larval stage into an eft, which is terrestrial.  When the eft matures, it returns to the water and transforms again into and adult newt, which is almost completely aquatic.  So while the name "Eft" indicates that Ubuntu is metamophosing, it also means that it has a long future ahead!
