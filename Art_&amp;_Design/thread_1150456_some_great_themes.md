### Post by richardjennings on 2009-05-07
> Sorta a noob here. How do I install this on my Ibex?
Add the repository to your list + add the key:


open up a terminal

copy this & paste into terminal (edit/paste)

```
sudo apt-key adv --recv-keys --keyserver keyserver.ubuntu.com 0x1781bd45c4c3275a34bb6aec6e871c4a881574de 
```

asks for password, type it in.

then type this into the terminal
```
sudo gedit /etc/apt/sources.list
```

add this to the bottom of the file...

##Themes du ZgegBlog
deb [http://debian.vogelweith.com/](http://debian.vogelweith.com/) intrepid zgegthemes
deb-src [http://debian.vogelweith.com/](http://debian.vogelweith.com/) intrepid zgegthemes


click save in gedit / close


type ```
sudo apt-get update
```

then you can install the themes by name like, ```
sudo apt-get install exotic-theme
```

### Post by richardjennings on 2009-05-07
the file /etc/apt/sources.list contains the details of all the repositories you are using (places you trust to install programs from)

the instructions above first detail adding the repository key, then explain how to open up gedit, add the new repository details, save it & update apt (the program that does the download / installing). Once apt is updated with the new repository packages, you can install with apt-get install or use Synaptic.


It sounds a lot more difficult than it is, probably my fault :p
