### Post by pt123 on 2011-02-08
Maybe be I had posted this in the wrong section (i.e. the programming and dev), so I am posting it here as there was no reaction there. Hoping there is more appreciation for the look of Ubuntu here.
I have never liked the way most websites look on a default installation or Live CD, it doesn't do anything to impress the new user. I feel this is largely due the fonts mess in Linux. 

On [my blog]("http://freeparticles101.blogspot.com/2011/02/getting-most-out-of-ubunt-font.html") I have a detailed explanation of getting the Ubuntu font to be the default font used by web browsers to display websites. The experimentation was carried out on Maverick. 

I was blown away by how beautiful websites looked when they used the Ubuntu font. The font was fuller and smoother.
[In this gallery ]("https://picasaweb.google.com/109468742935450671983/UbuntuFont?feat=directlink") I have comparisons.

Canonical  should change the default font settings to allow Ubuntu to be the font used by Firefox, after investing resources in developing the font they need to make the most of it. It will impress new users testing out Natty. 

Although my solution was a hack someone might be able to come up with something more elegant.
