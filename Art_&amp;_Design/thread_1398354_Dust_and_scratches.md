### Post by mcduck on 2010-02-07
> **rwltrz4 said:**
> I use to use photoshop and loved the dust and scratch removal for scans, what can I use in ubuntu that is similiar? I do not see this feature in the gimp
Are you talking about the Healing Brush found in latest Photoshop versions? or some filter?

If it's the brush, then the usual way of doing tat kind of repairs is to use Cloning Tool. Actually the Healing Brush in new PS versions is just an automatic cloning brush, the only difference is that when using Cloning Tool for the task is that you have to choose yourself where to copy image data to the area you are repairing. (exactly the same way you'd have done this kind of job with any older PS versions). Anyway, any new Gimp version should also have a Healing Tool. :)

If it's the filter you are after, you can find some repairing filters in Filters/Enhance, but there's no specific tool for removing dust and scratches. Despecle might help a bit but you'll probably have  to also use the mahual tools to get good results.

### Post by LoriQuaid on 2010-02-09
Sometimes when scanning in a photo, a white, straight line is introduced which stretches across the length of the photo.  Years ago when using Paintshop Pro, I used a built-in tool that would allow me to sweep out a long, narrow rectangle around such a scratch, and then apply the scratch removal filter.  It was extremely easy to use.

I have heard the equivalent tool in Gimp is the healing brush.  This is much harder to use because I have to follow the long, straight scratch in a freehand movement, and my hand just isn't steady enough to follow such a long line without jitters.
