### Post by IYY on 2007-06-17
I've went and cloned the Windows iTunes theme for IceWM. Here's the result:

[[IMG]http://img398.imageshack.us/img398/7851/icetuneskv9.th.png[/IMG]](http://img398.imageshack.us/my.php?image=icetuneskv9.png)

If you want to use it, make sure that you set the colors of your theme appropriately. 

Window background: #C4C4C3
Highlight: #9FAAC1
Input Box : #DEDEDE

(this is easy with themes like ClearLooks that allow you to change colors in the theme manager). 

I'm uploading this here before sending it to FreshMeat to see if there are any problems. Please, test and send feedback. :)
