### Post by knn on 2006-06-22
[FBSplash]("http://ubuntuforums.org/showthread.php?t=178439") is now available for Ubuntu. I've made a howto based on scripts ported to Debian and a theme based on the gdm background image for the silent splash and the Breezy default wallpaper for the console background. This theme is very basic, so it would be great if someone could make a new one. Download my Ubuntu theme to see how it works
There's [some info]("http://www.bootsplash.org/config.html") about making bootsplash themes at [bootsplash.org]("http://www.bootsplash.org"), and these themes can be ported to fbsplash.
You could also download some Suse themes and see how they work.
Animations don't work (they don't work on Gentoo either), but I can make them work. Instead of displaying for example Configuring network devices like usplash, there could be an animation. Also, each tty can have a separate background (e.g. displaying the number of the console somewhere).
Of course this theme should blend in well with the rest of Ubuntu.
So let's make Ubuntu's splashscreen the best looking splash in the Linux world!
