### Post by Spark* on 2005-04-08
As you might remember, during the preview there shortly was a version of the Clearlooks Metacity theme with squared corners. The new clearlooks release has squared bottom corners and squared utility windows, so just for fun I also made a totally squared version of it. In case you are crazy enough to like that ;) you can get it from here:
[http://213.133.111.182/clearlooks/clearlooks-metacity-0.5.2_squared.tar.gz](http://213.133.111.182/clearlooks/clearlooks-metacity-0.5.2_squared.tar.gz)

Of course if you prefer the rounded edges, you can always get the latest release from gnome-look:
[http://gnome-look.org/content/show.php?content=21237](http://gnome-look.org/content/show.php?content=21237)
The latest release has some changes to the one included in Ubuntu, mainly the squared edges and a softer titlebar for unfocused windows.
