### Post by eilu on 2007-12-24
it depends. A lawyer would call it a "gray area" :lol:
[LIST]
[*]if the work is a parody, it is ok under free speech
[*]if it uses a similar style but is easily distinguishable from the copyrighted work then it is probably ok. I say probably, because "easily distinguishable" is a little subjective.
[*]if it is exactly the same (i.e., a total knock-off) then it is not ok. Also, if it can be mistaken for the real thing then the copyright holder may take issue and demand for it to be taken down.
[*]this goes without saying that passing it off as the real thing is a big no-no
[/LIST]
