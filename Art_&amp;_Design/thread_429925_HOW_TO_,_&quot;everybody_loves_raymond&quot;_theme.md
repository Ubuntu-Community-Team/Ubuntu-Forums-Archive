Ubuntu users, to use icons ,exract the [theme which you downloaded]("http://blog.shevin.info/2007/05/your-mind-is-like-donkey.html") f , and open a terminal , and go to the directory and then enter this command :

    sudo cp *.ico /usr/share/pixmaps/

and then right click on the program (or application launcher) which you want to use the icon for , and click on properties and choose the icon .

and for using Frank's voice for shut down, in Main Menu go to , System / Preferences / Sound / (then go to Sound tab and change the Log out sound to the Frank's wav file)
