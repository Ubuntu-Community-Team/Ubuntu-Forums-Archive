### Post by billstei on 2008-09-28
Go to System->Preferences->Windows->Movement Key and if it is set to "Control" then set it to one of the other choices.  Probably setting it to "Super (or windows logo)" will produce the least interference with Gimp and other programs that rely on the modifier keys Ctrl and Alt.
