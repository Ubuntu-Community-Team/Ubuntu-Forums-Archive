### Post by renzokuken on 2006-04-29
in the end, i simply copied the directory for the gnome icon theme located at /usr/share/icons/gnome ("gnome" being the name of the actual theme) to somewhere safe like my home directory (u can use any icon theme really, i wasn't gonna replace all the icons, just a few of the common ones and i like most of the "gnome" theme)

once there, i simply used gimp to resize the icons i acquired from various sites, and dropped them in the appropriate subdirectory with the correct/relevant name. then alter the "index.theme" file and change it to whatever u want to call your theme. when i'd finished, i compressed the folder with ArchiveManager  into a .tar.gz file.

then i simply went to System - Prefernces - Theme and the "install new theme" ..... selected the .tar.gz file i just created and hey presto, all done !!

i would post my custom theme but i didnt create any of the icons and so dont wanna be cheeky to the people who spent ages actually making them
