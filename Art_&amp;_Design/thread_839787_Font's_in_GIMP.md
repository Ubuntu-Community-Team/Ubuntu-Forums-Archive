### Post by Windsurfer619 on 2008-06-26
Make sure the checkbox "Antialiasing" is enabled for the text tool. Also make sure you're zoomed to 100% so you get a pixel-for-pixel display. Also make sure that under System > Appearance > Fonts you have some sort of font smoothing enabled.
