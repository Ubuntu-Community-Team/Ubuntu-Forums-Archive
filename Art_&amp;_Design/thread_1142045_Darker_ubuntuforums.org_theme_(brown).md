Someone had made a comment about my custom ubuntuforums.org theme they saw in a [screenshot I posted in another thread](http://ubuntuforums.org/showpost.php?p=7161618&postcount=11) so I decided to put a little more time into it and release it for everyone's enjoyment.

As I noted on the user style page, there aren't some classes or id's where you'd expect them to be and some things are just designed weird, so I had to do some CSS2 select tricks to wrangle up the code.  I'll be sorting them out and refining the color scheme in the future, so stay posted.

### Post by detrate on 2009-04-28
The serif fonts were just the ubuntu default for me.  Perhaps I'm missing some font packs.

add this to the top of the script and you're good, I'll put it in the next version as well. >>

```
* { font-family: verdana, arial, sans-serif !important; }
```

Optionally add
```
pre {
   font-family:monospace, courier !important;
   font-weight:bold !important;
}
```
for code blocks to have a fixed width font.
