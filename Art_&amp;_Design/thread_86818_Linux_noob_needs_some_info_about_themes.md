### Post by Xian on 2005-11-06
Metacity is the window manager, therefore metacity themes will change the way the window decoration looks. This is the area around the application where the close, maximize, minimize, and similar buttons are located.

GTK is a toolkit and gtk themes change the way the applications themselves look. Here we are talking about what is in the actual window, which would include things like toolbars, menus and scrollbars.
