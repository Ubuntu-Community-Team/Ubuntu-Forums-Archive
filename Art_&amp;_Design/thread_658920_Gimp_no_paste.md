Ok, this is going to sound weird, but I just tried it and it worked.  My directions will start with the original graphic.  Feel free to skip any things already done.

1.  Open graphic to be edited.
2.  Increase canvas size and center or position as appropriate.
3.  Create New Layer.  This will create a layer of the size of the re-sized canvas.
4.  Change order of the layers so that the original layer (Background) is above the new layer.  Technically, this may not be necessary, but I'm careful.
5.  Merge original layer down (onto New Layer) and now you can work with the whole canvas size size as you choose.

At this point, you can rename the new layer whatever you wish.

Hope this helps  .:)
