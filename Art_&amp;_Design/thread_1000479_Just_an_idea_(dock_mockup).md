I'm just sharing an odd idea that I had. I don't expect anything from this to be adopted into anything. 

The basic idea:
[LIST]
[*]The top toolbar is still there (I was just lazy)
[*]The "start" menu is displayed by default right over the desktop
[*]The arrows represent the direction that the OSX style dock opens up in (with a smooth animation)
[*]On the right, though I did not put this in would be a similar button for changing workspaces.
[*]When Windows are open, they completely cover the button but upon pressing the start menu button, both buttons appear over top of the windows (transparency and blur still apply)
[/LIST]

I focused more on the visual aspect, but the idea is to have a dock that is in-obtrusive, and yet extremely easy to access when needed, along with a workspace switcher (though not visible) that matches it.

Thoughts? (if you want to mess with it, I can supply the svg file.)
