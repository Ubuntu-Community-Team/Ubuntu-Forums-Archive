I was very bored and needed to sharpen my photoshop skills :( Took the best(most creative) features of all the mockups I remember and made them my own. I suppose I'l work on a metacity / gdm theme next :)

Description
----------------------
The text on the desktop (or I thought it would be cool to name it plate, because when its free of clutter, you could say I have a clean plate :D ) is just for your own convenience (say your copying and pasting multiple things, ect..). Like sticky notes in ubuntu, the text wont go away until you remove or edit it. The bottom left is where the currently open applications show (the vista/ubuntu icon is just a placeholder for the icon of a currently open program) and the shelf name ("eez") is like a start menu button. Appspin is just a program launcher and I suppose nautilus, firefox, and rhythmbox would be there on default for easy access. You could right click the desktop or click the shelf button ("eez") to bring up the main menu 

Lol, I'm pretty sorry to say but the chances are, this will never be available. It is just a mockup of what I think would be a cool Desktop Environment. It is just purely for designing/concept fun :) Tho, I suppose I could edit docky's theme source to look just like this, but then it would be a 3rd party program and I doubt anyone would want to use it. Lol, if anyone would consider testing or using please tell me :) Tho, I'm going to make a gdm/metacity theme first ^-^
