Largely true, but to say the layout is similar may be stretching it a bit, to say the least.  For a start, GIMP uses 3 separate windows, one for the graphic itself, one for the toolbox and the third for the layers, channels & paths, etc etc, instead of the one window for PS. ### Post by mcduck on 2010-11-15
> **ajgreeny said:**

Apparently some of the menu items are in different places too, but having never used PS, I can not comment further about that.

However, everything is there, and if you approach it with an open mind and are prepared to give it a chance, I think you will find that it is an extremely powerful package that can do just about anything with graphics files of almost any file type.

Well, it could be said that the layout is similar to Photoshop on OSX (as only the Windows version of PS uses MDI, mainly due to Windows' poor handling of SDI windows). ;)

But yes, it's a different program and obviously the interface isn't *identical* to Photoshop. Only *similar*, definitely close enough that you shouldn't have any issues if you know the basic photoediting stuff.

Don't expect it to work exactly like Photoshop, though. It's able to do all the same things, but might do them in a slightly different way. This shouldn't be a problem if you actually understand what you are doing with Photoshop, instead of just knowing the tools (as in "knowing how to edit images" VS "knowing how to use Photoshop"). Gimp also doesn't have quite as many automatic tools as latest Photoshop versions have, so instead of relying on Extract tool or Red-eye removal tool, for example, you need to know how to do the same thing by hand.

### Post by emilywind on 2010-11-15
GIMP is wonderful for general drawing/painting and image manipulation. If you want to do some 3D stuff, you can install Blender, MyPaint is great for simply painting, and Inkscape is the go-to for vector graphics.

There's quite good stuff all around. :)
