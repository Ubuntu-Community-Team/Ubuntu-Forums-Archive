Ok so I have a picture that I want to use as a skin for a 3d model and I was wondering is there any way that is like the masking in GIMP to select the parts of the picture that I wish to scale? 
I am kinda new to inkscape and have been reading the tutorials but I haven't really found a way to do this yet. Any help would be greatly appreciated.

### Post by smartboyathome on 2008-01-04
You can cut out the portion you want to scale and save it as a different image. Then open it in Inkscape, select it, go to the Paths menu, and select trace bitmap. Select one of the options to trace it with (I would go with color first if it is color), and then click ok. When it is done, save it.

That is the only way as far as I know.
