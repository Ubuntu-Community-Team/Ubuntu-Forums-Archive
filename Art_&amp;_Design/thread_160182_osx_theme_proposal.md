### Post by slackcor on 2006-04-16
Its easy to create your own compatible theme. First thing is that you must have rar (non-free) installed. Then:

-Extract the jar file in a folder.
-Open the file install.rdf
-Change the line <em:maxVersion>1.6</em:maxVersion> in <em:maxVersion>1.5</em:maxVersion> and save the file.
-Now select all the unpacked files in nautilus than <right click> and choose create archive. Change the tab <tar.gz> to <jar> and create a new jar.file 
-Install the new created theme ;)
