### Post by smartboyathome on 2008-01-17
My way of how I would do it would be to make the mirror using a selection, and save the selection as a path with Dialogs > Path > Selection to Path. Then, you can edit it and when you are ready to put the man in, use the same dialog, select the path that was made, and push the "path to selection" button. Now, go to edit > paste into to paste it into the selection. You can move it around with the move tool to where you like. When you are done, just do control+h to anchor the layer.
