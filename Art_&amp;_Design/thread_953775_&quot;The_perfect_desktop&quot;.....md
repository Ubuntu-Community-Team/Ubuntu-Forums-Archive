The swirly thing? That is the epiphany of making Ubuntu "presentable" to the rest of the worls. Its called....Ubuntu Studio (not too creative naming). Its an officially supported derivative of Ubuntu by Canonical. Basically, its Ubuntu with a completely revamped UI and kernel (ubuntu uses the -generic kernel, whereas sStudio uses the -rt or real time kernel) It also ships with a plethora of Audio, Video, and Image manipulation programs on the *DVD*. Oh yeah, a DVD of Sexy Looking Ubuntu Goodness. Hence the name Ubuntu *Studio*. Its a studio for creativity...kinda. You can even upgrade to it directly from your original Ubuntu instalation, all it does is replace and add some extra stuff. Check out Synaptic, but first read the wiki.

Check it out here [http://ubuntustudio.org/](http://ubuntustudio.org/) (its not updated that often)
or here [https://wiki.ubuntu.com/UbuntuStudio](https://wiki.ubuntu.com/UbuntuStudio) (the official wiki, more info for you)

As for the  "lovely Mac OS X-like thing on the bottom" its called Avant-Window-Navigator (AWN). Its still kinda buggy, but I use it all the time. You can get it from several places, but i'd start with Synaptic.

1. your desktop has to be practical
2. your desktop has to be beautiful (at least to you)

talked about on this page: [http://blog.xfce.org/?p=177](http://blog.xfce.org/?p=177)


> How do you change the background of the panels?

You use a .gtkrc-2.0 for changing colors of background or text colors. You can also use an image. This is a simple version of a .gtkrc-2.0:

```

style "panel"
{
  bg[NORMAL] = "#181818"
  bg_pixmap[NORMAL] = "image.png"
  fg[NORMAL] = "#FFFFFF"
  }

widget_class "*Panel*"      style "panel"
widget "*Panel*"            style "panel"
class "*Panel*"             style "panel"

```

..... you would just place the image and the .gtkrc-2.0 file in your home directory like: ~/.gtkrc-2.0 


> What is the hotkey for a terminal?


You need to make one at Xfce4 Menu-->--Settings-->--Keyboard Settings

then goto Shortcuts tab.... and press "Add". Now you can add any hotkey binding to open anything. I use to have hotkeys for all of my favorite apps and even could open Firefox up to my 10 different favorite pages using the exo-open command.
