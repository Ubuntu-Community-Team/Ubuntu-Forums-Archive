There are several color schemes over on [kde-look.org/]("http://www.kde-look.org/").  Just type "human" in the search box.

The icons used in the default Ubuntu Human theme are (oddly enough) the default GNOME icons (there is a separate "Human" icon theme installed, however).

So what you want are icons like the default GNOME ones.  The only thing I came across was based on the recent "gPerfection" theme (which is an update of the GNOME theme).  I actually prefer it to the default GNOME anyway).
You might too.

Take a look [here]("http://kde-look.org/content/show.php?content=27788")

As far as Window Decorations go, I'm sure there is something on that site which might be suitable.  I've only come across just a few KDE window decorations that are similar to anything in GNOME.
