### Post by tituomin on 2013-12-02
I ran in to the same problem. I'm not sure if this is the only solution, because it's really awkward. But at least it works:
[LIST=1]
[*]With Scribus open, close all documents.
[*]Change the color palette from Edit > Colors
[*]Create a new document, save it, close it.
[*]Open your previous document with the wrong color palette.
[*]Choose Edit > Colors
[*]Click Import and find the file you saved in step 3.
[/LIST]
The colors you chose in step 2 will be imported to your new file. This is really difficult, maybe there's another way?
