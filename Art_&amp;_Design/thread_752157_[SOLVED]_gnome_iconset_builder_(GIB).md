### Post by smartboyathome on 2008-04-11
I don't know about another project, but creating one is pretty easy as it is. Take your favorite icon theme (if you installed it via Appearance, then it is in .icons in your home folder, which is hidden, if it was preinstalled/installed by the package manager, it is in /usr/share/icons/) and customize all the icons to be what you want them to be.

### Post by suchawato on 2008-04-13
most icons are either an .svg file or a png set of different  resolutions.
You can make either using Inscape, Gimp, or Phootoshop, Illustrator, or Fireworks.
Look in your system files for varrious different icon folders to see what you need to make for different projects.
Desktop icons are .png images,
application launcher icons are usually .svg images.
that's because the application bar(s) can be scaled to any size the user wants so a stand alone rastar image would not be the best for those.
For the desktop .png rastar images are fine.
.png's need to be saved with a transparent alpha channel bacgrond if you want them to stand out as somthing other than a box.

I think the standard sizes the last time I made a set was 8x8pixels, 16x16, 22x22, 48x48, 64x64, and 128x128
I think. some people include 36x36, and 256x256.
Be sure to save a large version for working, and a .psd or a GIMP .xcf file for working.
Remember, Gimp can read and write a psd file but Photoshop cannot read a .xcf file

-Sara
