### Post by jem7v on 2007-03-13
That's part of whatever GTK2 theme you're using.  Under System -> Preferences, you will find a Theme option... If you click the Theme Details button you'll get a window with 3 tabs: Controls, Window Border, and Icons.  Controls = GTK2 themes (aka Application themes at Gnome Art).
You can find more GTK2 themes at [http://art.gnome.org/themes/gtk2/](http://art.gnome.org/themes/gtk2/) and [http://www.gnome-look.org/index.php?xcontentmode=100](http://www.gnome-look.org/index.php?xcontentmode=100) and to install them, simply drag the themename.tar.gz files onto the Theme chooser and they'll be installed automatically.

If you just want to change the text and nothing else, though? Not sure.
