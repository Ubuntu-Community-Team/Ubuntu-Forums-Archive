..... with Inkscape and an .svg image you can modify icons very easily. Then to transform them into .png form I use this command:

```
for i in *.svg; do inkscape -f "$i" -e "$i.png" -w48 -h48; done
```


just change the -w and -h (width and height) to any pixel size.

You should install the app called "Inkscape" to make icons (it should be in synaptic). Then take the Inkscape tutorial that it comes with (super easy to learn what you need to know).


Then download a ".svg" image for the icon you want. I found this way works for me:


First, google search for the logo: "google earth logo .svg"

This should give you a wiki page here: [http://en.wikipedia.org/wiki/File:Google_Earth.svg](http://en.wikipedia.org/wiki/File:Google_Earth.svg)

Secondly, right click the Google_Earth.svg image and select "Copy Link Location" from the drop down menu.

Next open a Terminal and download that .svg using the wget command:

```
wget http://upload.wikimedia.org/wikipedia/en/e/e9/Google_Earth.svg
```

Now you can open the Google_Earth.svg using "Inkscape" and you'll find it easy to change the appearance of most .svg images. 



I usually modify all of my icon packs by redoing the scalable icons and then transforming them into .png sizes for 48, 32, 24, 22, and 16 pixel sizes.
