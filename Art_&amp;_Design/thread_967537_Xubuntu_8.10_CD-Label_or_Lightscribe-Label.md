### Post by kernst on 2008-12-24
It's not for Xubuntu, but since this is the only forum thread that comes up when you search for 'intrepid' and 'lightscribe', I'm going to post this here for now, and possibly start another thread later.

I started with [the Edgy CD label created by Aelfric5578]("http://ubuntuforums.org/showthread.php?t=285051") and remixed it a bit to make a very attractive (if I do say so myself :)) LightScribe-ready label for Intrepid. I did not include any alternate branding logos (Xubuntu, Kubuntu, etc.), so it's fairly generic, but functional. It is entirely monochrome with no funny transparency effects, resulting in very small file sizes, and very good results on a LightScribed disc at maximum contrast. I'll post a scanned-in sample of my disc in a few minutes.

A .ZIP file containing an 800x800 px PNG (as seen in the attachment), layered Gimp .XCF file (with editable text that you can change to whatever you want), as well as a template ready for use with the [Windows LightScribe Template Labeler]("http://www.lightscribe.com/downloadSection/windows/index.aspx?id=1524") (available from [lightscribe.com's download page]("http://www.lightscribe.com/downloadSection/index.aspx")) will be hosted on GIMPStuff.org. (*I'll edit this post once I get it uploaded.*)

I give [Aelfric5578]("http://ubuntuforums.org/member.php?u=112623") all the credit for this work, as it is essentially his, but he stated in [his original post]("http://ubuntuforums.org/showthread.php?t=285051") that the Edgy CD label was free for all to use so that extends to my LightScribe-able version as well.

**EDIT:** ZIP file of PNG, XCF, and LightScribe Template Labeler template versions is now available [at GIMPStuff.org]("http://gimpstuff.org/content/show.php?content=95781").
