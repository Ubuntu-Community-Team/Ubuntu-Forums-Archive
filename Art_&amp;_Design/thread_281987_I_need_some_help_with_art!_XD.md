### Post by smartalecks on 2006-10-22
for the background use the gradient tool.
for the wavy effect thing, 

1. brush some colors thickly with a solid brush
2. motion blur until you get the desired length etc.
3. mess with the layer type to see it there is something you like.
4. use the smudge tool to make it slightly wavy

Just mess around with it.
