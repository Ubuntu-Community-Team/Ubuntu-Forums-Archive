### Post by airtonix on 2008-08-25
A few suggestions, the 'recent comments' section. Personally I would move this down to the bottom of the page. 

In your main menu, you have an item "your_ip"...this really doesnt belong as a main menu item...it's more of a site-footer item.

```

[                          site header ]
[                              nav-bar ]
[ post-1                    ][  pages  ]
[ ......                    ][  .....  ]
[ ......                    ][  .....  ]
[ ......                    ][  .....  ]
[ post-2                    ][  .....  ]
[ ......                    ][  cate.  ]
[ ......                    ][  .....  ]
[ ......                    ][  .....  ]
[ post-3                    ][  .....  ]
[ ......                    ][  .....  ]
[ ......                    ][  .....  ]
[ ......                    ][  .....  ]
[  Recent Comments  ][  Related Posts  ]
[  ...              ][  ...            ]
[  ...              ][  ...            ]
[  ...              ][  ...            ]
[  ...              ][  ...            ]
[  Site Footer                         ]

```

Reasoning behind this is that comments are not what visitors would be coming to read, these are something that is more of an after thought...something to be digested after considering the main content.
Putting them down the bottom there also gives them the space they deserve to be fully appreciated as readable paragraphs rather than squashed sardines.

if you get what I mean, everything has a place and deserves the proper layout to fully honour its purpose.

I recommend you keep alistapart.com bookmarked and make it one of your daily readings.
