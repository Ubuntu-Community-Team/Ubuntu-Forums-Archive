### Post by SuperSonic4 on 2009-04-25
I'd use the path tool to draw around the edge of the circle and outward(you can use small links between points to make it more circular and press shift to start a new selection)

From then you can pick select from path and cut it out

---

### Post by Sand &amp; Mercury on 2009-04-25
Use the circular selection tool to select a circle around what you want to crop.
Select > Invert
Hit the delete key.
Image > Autocrop image.
