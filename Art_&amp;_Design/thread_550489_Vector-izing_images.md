### Post by amoser on 2007-09-14
No I don't think there is any software that does that, it goes down to the basic differences in the images.  GIMP images are Raster, so they use little dots called pixels.  The pixels are pre-drawn, and the data for each pixel is stored.  On the other hand vector graphics are really instructions on how to "draw the image at run time".  So there is no program that I know of that can do that.

~Alan

### Post by ghevan on 2007-09-14
On inkscape:

Load an image, that will make it an object, then go to "path> trace bitmap" tune your settings, then tune your vector.

There is no "magic" way for getting excellent looking results. As for your example, to get those results, may be better to trace it by hand.
