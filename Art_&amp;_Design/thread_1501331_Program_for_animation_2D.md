### Post by digitalis_vulgaris on 2010-06-04
It's sounds funny but the best free software for 2d is a 3d software - Blender. You can put images as plane textures and create South Park kind of animations. You can deform part of image using shape animation or armature. You have video editing tool for compositing animation layers. Blender is a dream tool for 2d animation...

It's not quite user frendly, but it's a best toy I ever had
