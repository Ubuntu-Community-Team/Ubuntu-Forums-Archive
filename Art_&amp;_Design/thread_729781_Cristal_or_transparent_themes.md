### Post by smartboyathome on 2008-03-20
If your using Ubuntu 7.10 Gutsy or later, then it is installed by default. Try going to the System > Preferences menu, is there an Appearance menu? If so, then you are running gutsy. Click on Appearance, and click on the visual effects tab. Is it set to "Normal" or "No visual effects"? If the latter, then compiz isn't installed on your computer.

Also, if you are using Gutsy, there aren't any "transparent" gtk2 themes, that only came recently to the murrine in hardy, and it is still very experimental. Check GNOME Look (as said above) for a lot of great themes.
