### Post by ubuntu_demon on 2005-12-19
I moved this thread. Please don't post questions in the customization section. I moved this thread.

You can customize your theme to make it look like a Mac theme.

Install this to get some themes :

$sudo apt-get gnome-themes-extras  

And to be able to handpick a lot of nice themes do this :

$sudo apt-get install gnome-art

go to system->preferences->art manager to install  all kind of themes

To customize your theme go to system->preferences->theme 

You can get inspiration here :
[http://www.ubuntuforums.org/gallery/](http://www.ubuntuforums.org/gallery/)

Look around in the art forum to find more interesting customization stuff :
[http://www.ubuntuforums.org/forumdisplay.php?f=16](http://www.ubuntuforums.org/forumdisplay.php?f=16)

good luck!
