### Post by sp-1070 on 2010-12-06
Ok so first of all this is my first thread so i hope its usefull to some of you guys.

I really love the way crunchbang looks and works but still ubuntu is working  more stable and that makes it just plain better.
Now i have only done this on ubuntu 10.10 but i suppose it should be the same on previous versions.

We are going to make them look and feel the same.

so first were going to start of with the terminal stuff.

start a terminal and go to edit

Go to background and click on the 
> Use Background settings from system themeThat enables you to start messing around

next thing you wanna do is click 
> Background imageAfter that 
> Transparent or image backgroundPut it at a 100% first and work youre way from there till you find what suits you best.

Depending on youre back ground you might wanna change youre colors.

Just go to the colors tab.
once again clik on
> use colors from system theme> Built-in schemes: Black or white(i use my own config as a preview of course feel free to change it in ways whatsoever)

So you now have a transparant terminal wich isn't all that special certanly not if youre doing this from a clean install its butt ugly.

Time to do something about that.

Close the terminal.
Now right clik annywhere on youre screen and go to
> Change Desktop Backgroundchoose something you like and go to theme,
now if youre on Ambiance its all ugly just bad.
(again i here state my personal taste)
Put it on the theme you like, i put it on Radiance.

Also you might wanna take a look at the fonts tab,
play with that.

Also just for fun put youre visual Effects on Extra.
Close it when youre done

So now you already have a relativly nice Desktop,
but its not quite Crunchbang yet is it.

go to 
> System-->Preferences-->Keyboard shortcutsI changed:

> Run a Terminal   to Windows key and T> Show the Panels main menu   to Windows key and SSo what these bassicly do is wel edit youre shortcuts there is a lot in there so look around a bit.

Done? Close it.
Go up to youre taskbar right clik right behind places and then click delete this panel and away it is.

So now its looking alot like crunchbang.
youre  panel shows up every where you want it to and such.

Missing something ? yes you are its called conky,
just instal it.

```
sudo apt-get install conky
```Now conky has a lot of diffrent setups its fully conifgurable and i won't post my own here just google it conky.conf theres so much out there youl be thrown dead.

Now you have a Crunchbang like Ubuntu system.
Just one more thing.
Go to 
> system-->preferences-->Startup applicationsHere you can all sorts of stuff to startup at a boot.

Also the best way to switch trough different workspaces is to make a shortcut for them i have.

Windows key +1 for workspace 1 2 for 2 and so goes on.

Well if you read it and liked it and it worked out for you.

Congratulations to you having a Crunch-bang look alike and to me this being my first howto please leave some feedback.
