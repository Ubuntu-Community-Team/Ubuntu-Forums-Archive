### Post by -yay- on 2008-12-14
That depends on what kind of site you want and your knowledge level.

If it's very simple site with static pages and you know XHTML and CSS then just hand code it in any notepad like program like gedit. (or use Bluefish, kompozer, geany to help you structure your code)

If you don't know XHTML and CSS but want to learn check this out:
[http://html.net/tutorials/html/](http://html.net/tutorials/html/)

If it's more advanced, like with dynamic pages, and you're not to comfortable with PHP, then use a Content Managing System to do it for you, many of these exist, like Wordpress (specialized for blogs), Joomla (allrounder but more difficult), phpbb3 (for forums). These don't require any knowledge of HTML, CSS or PHP so they are definitively the easiest way to go. Most of these come with the ability to install themes.
