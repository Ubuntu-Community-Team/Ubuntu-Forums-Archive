### Post by Famicommie on 2007-04-08
There is an OSX icon theme here: [http://www.gnome-look.org/content/show.php/OSX?content=31618](http://www.gnome-look.org/content/show.php/OSX?content=31618)
To install an icon theme, go to System->Preferences->Theme->Theme Details->Icons and drag the .tar.gz file into the Icon window. You can delete the tar.gz file from the desktop when you are done.

There is an OSX metacity theme available here: [http://www.gnome-look.org/content/show.php?content=35753](http://www.gnome-look.org/content/show.php?content=35753)
To install that, select Window Border from the Theme Details window and drop the tar.gz file into it.

For wallpapers, I suggest that you browse around here: [http://www.gnome-look.org/index.php?xcontentmode=170x171x172x173x174](http://www.gnome-look.org/index.php?xcontentmode=170x171x172x173x174)
