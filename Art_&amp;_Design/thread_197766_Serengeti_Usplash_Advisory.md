### Post by j_baer on 2006-06-16
The lastest updates _may_ modify your menu.lst file causing the boot splash image not to display.

There are two remedies.

1. Manually edit the menu.lst file and add back the vga=xxx argument to the kernel line.

or

2. Re-install Serengeti Usplash

I noticed this primarily on systems running the enhanced usplash image supporting nvidia graphics cards.

Cheers !
