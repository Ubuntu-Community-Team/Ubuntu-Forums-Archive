### Post by Sirron on 2009-11-22
Don't discount Inkscape for putting it all together. Although not technically a desktop publishing application, it can create and edit PDF files and supports text in boxes (select the text tool, then drag a box). I don't think you can make text automatically flow between boxes, though.

If you're using bitmaps in inkscape, resize them with inkscape's tools temporarily, but make sure you resize them with Gimp or something like [SmillaEnlarger]("http://sourceforge.net/projects/imageenlarger/") before calling it finished, because inkscape can't really upscale bitmaps properly.

As a testament to its awesomeness, I use inkscape every day at work, for everything from UI icons to pre-CAD sketches.
