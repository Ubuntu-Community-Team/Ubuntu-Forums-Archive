### Post by andynatt on 2009-11-25
There is a workaround to this.

```
 unset GTK_IM_MODULE && inkscape
```

This should do the trick until the bug is resolved

Bug:: [https://bugs.launchpad.net/inkscape/+bug/406036]("https://bugs.launchpad.net/inkscape/+bug/406036")
